# GOD Engine Framework Canon v2

Date: 2026-02-17

## Scope

This is a curated, foundational framework stack for building an AI-native content and growth operating system. Grounded in processed video content and validated business claims.

This is intentionally not a broad idea dump.

## Curation Method (Why This Is Not Random)

Inclusion gates:

1. Explicit framework structure in source
2. Trust quality gate (trust score >= 72)
3. Actionability to one or more GOD Engine layers
4. Claim support (prefer high confidence)
5. Multi-source triangulation for critical layers (constraint, planning, feedback)

Exclusions:

- Lifestyle/low-relevance videos even if keyword match was high
- Broad "content" advice without an operational model
- Low-trust sources with weak claims

## Framework Dossiers

### F1) Positioning Components + Niche Wedge

**Author:** April Dunford
**Core model:**
1. Competitive alternatives
2. Differentiated capabilities
3. Differentiated value
4. Best-fit customer segment
5. Category framing
6. Wedge strategy (dominate niche, then expand)

**Why it matters:**
- This should be the brand strategy root object, not a side note in docs.
- It gives deterministic inputs to content pillars, campaigns, and outbound narratives.

**Source:** [The Marketing Expert: Sell Anything with this Trick](https://www.youtube.com/watch?v=vM_1G1LCotU)

**Claims evidence:**
- Positioning is foundational input for marketing/sales.
- Weak positioning is visible in early calls (confusion, wrong comparisons, no pay reason).

### F2) Structural Churn + ICP Realignment

**Author:** Alex Hormozi
**Core model:**
1. Diagnose structural churn (customer instability, not delivery issue)
2. Run profitability analysis by customer type
3. Define top-20% ICP
4. Realign offer, pricing, proof, onboarding
5. Enforce qualification; phase out bad-fit intake

**Why it matters:**
- Prevents strategy from optimizing demand for low-value segments.
- Drives per-brand qualification rules and channel targeting constraints.

**Source:** [Business is hard until you do this](https://www.youtube.com/watch?v=CoPs-Bk8M9Y)

**Claims evidence:**
- Tactical path: customer profitability -> top 20% ICP -> full GTM realignment.
- Persistent churn can be structural due to customer volatility.

### F3) Constraint-First Strategy (Theory of Constraints)

**Author:** Alex Hormozi
**Core model:**
1. Business grows only to its current constraint
2. Improve the biggest bottleneck first
3. Re-test system; locate next bottleneck
4. Repeat continuously

**Why it matters:**
- Stops "do more content" behavior when bottleneck is ICP, conversion, onboarding, or retention.
- Should be first stage in weekly strategy decisions.

**Sources:**
- [3 Simple Fixes that Grow Any Business](https://www.youtube.com/watch?v=KQuyQpFANpA)
- [The Top 1% of Business Owners Aren't Smarter Than You](https://www.youtube.com/watch?v=q9OmYf_IlQ0)

**Claims evidence:**
- Improving the largest bottleneck gives outsized growth.
- Correct constraint can shift from customer acquisition to talent/product/retention.

### F4) JTBD Segment Sequencing (Multi-Vertical GTM)

**Author:** Asia Orangio
**Core model:**
1. Segment by job context, not industry label only
2. If buying context differs materially, run one segment motion at a time
3. Rank segments by market opportunity + PMF strength + supportability
4. Commit long enough to produce valid signal

**Why it matters:**
- Needed for multi-brand, multi-channel calendar sanity.
- Defines how segment-specific campaigns and pages should be sequenced.

**Source:** [How to Go To Market Across Many Verticals](https://www.youtube.com/watch?v=xtr92VAqop8)

**Claims evidence:**
- Different buying motions imply one-segment-at-a-time GTM.
- Segment expansion often needs vertical pages/content before full homepage rewrite.

### F5) PEACE Messaging + 3-Phase Campaign

**Author:** Donald Miller
**Core model:**
- PEACE narrative blocks: Problem, Empathy, Answer, Change, End Result
- Campaign phases: Curiosity -> Enlightenment -> Commitment

**Why it matters:**
- Gives a repeatable messaging scaffold per campaign and channel.
- Maps directly to calendar stages and approval criteria.

**Source:** [How To Sell Anything With The StoryBrand Soundbite Strategy](https://www.youtube.com/watch?v=5OJT1ph-yL4)

**Claims evidence:**
- Lower cognitive load in messaging improves response.
- Consistent PEACE language can materially change campaign performance.

### F6) Hub-and-Spoke Content Operations

**Author:** Tim Cameron-Kitchen
**Core model:**
1. Audience definition
2. Topic validation
3. Research
4. Production
5. Repurpose one hub asset into channel-native spokes

**Why it matters:**
- Core model for planning/execution glue across channels.
- Converts one high-quality research stream into many outputs without random duplication.

**Source:** [How CONTENT MARKETING works in 2023](https://www.youtube.com/watch?v=lUTXIs61_V0)

**Claims evidence:**
- Hub-and-spoke leverage reduces production waste.
- Channel output should be native format, not copy-paste cross-post.

### F7) Content Maturity Phasing (1 -> 5)

**Author:** Alex Hormozi
**Core model:**
1. Post once
2. Post consistently
3. Post reliably across channels
4. Maximize volume by channel
5. Build capture + purpose-built repurposing system

**Why it matters:**
- Prevents overbuilding too early.
- Gives explicit maturity state per brand/channel and realistic cadence targets.

**Source:** [1.2M Followers in 6 Months... My Content Marketing Strategy REVEALED](https://www.youtube.com/watch?v=MD5-HByRxoA)

**Claims evidence:**
- Explicit 5-phase model captured with high claim confidence.
- Early-stage content should often prioritize nurture/sales enablement before full acquisition dependence.

### F8) Content-as-Targeting (Interest Media)

**Author:** Alex Hormozi
**Core model:**
1. Social distribution behaves as interest media
2. Content itself is targeting
3. Optimize for buyer-fit quality, not total impressions

**Why it matters:**
- Critical for KPI design in dashboard analytics.
- Avoids shipping strategy around vanity metrics.

**Source:** [You're not ready for the next phase of social media.](https://www.youtube.com/watch?v=Oc9zSbi6uaw)

**Claims evidence:**
- Viral reach can lower commercial quality.
- Small highly-targeted audiences can outperform large unfocused ones for revenue.

### F9) DRI Scorecard + Weekly Operating Cadence

**Author:** Dan Martell
**Core model:**
1. Every metric has a DRI owner
2. Weekly sync cadence
3. R/Y/G status with trend visibility
4. Coach against outcomes, not opinions

**Why it matters:**
- This is the operating heartbeat for the unified UI.
- Ties strategy and execution to accountable owners by metric.

**Source:** [The Only Scorecard You Need to Build a $10M Business](https://www.youtube.com/watch?v=0hsm_iVrblg)

**Claims evidence:**
- Unowned metrics are a root cause of repeated issues.
- Weekly review structure improves alignment and self-correction.

### F10) Measurement-First Growth + Retention Loop

**Authors:** Alex Hormozi, Amir Movafaghi (Mixpanel)

**Core model:**
1. Instrument full funnel: click-to-close and close-to-resell
2. Use leading indicators (activation / NPS proxies)
3. Improve onboarding/retention before scaling acquisition
4. Scorecard red metrics drive experiment queue

**Why it matters:**
- Defines feedback architecture and integration priorities.
- Ensures performance data changes planning and scoring, not just reporting.

**Sources:**
- [You're WASTING Your Chance to Make INSANE Money](https://www.youtube.com/watch?v=QTZsh3BgOwY)
- [Building the Most Loved Analytics Tool...](https://www.youtube.com/watch?v=249tl3G2WIM)

**Claims evidence:**
- Measurement itself changes behavior and outcomes.
- Retention is lagging; operational leading indicators are required.
- Scorecard mapped to growth engine highlights the real optimization point.

## 90-Day Launch Order (Pragmatic)

1. **Weeks 1-2:** Layer 0 foundations
   - Implement F1 + F2 for your brand records
   - Add disqualification rules and wedge segment per brand

2. **Weeks 3-4:** Layer 1-2 discipline
   - Add constraint diagnosis workflow (F3)
   - Lock one active segment per brand (F4)
   - Add PEACE campaign fields in planner (F5)

3. **Weeks 5-8:** Layer 3-4 execution system
   - Enforce maturity phase + hub asset linking (F6 + F7)
   - Enforce DRI ownership for calendar and approvals (F9)

4. **Weeks 9-12:** Layer 5-6 learning loop
   - Add targeting quality KPI (F8)
   - Wire measurement + retention loop to planning scores (F10)

## Notes on Hero/Hub/Hygiene

Hero/Hub/Hygiene remains valid as a planning taxonomy, but the stronger operational evidence is from:

- Hub-and-spoke production mechanics (how to run the system)
- Maturity phasing (when to scale each motion)
- Targeting-quality distribution logic (how to judge output)

For launch reliability, use Hero/Hub/Hygiene as a labeling layer on top of F6/F7/F8, not as the only operating framework.
