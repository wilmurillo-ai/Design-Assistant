### G2.7 Blog Publishing

| Field | Value |
|-------|-------|
| **Trigger** | Blog content approved in Content QC & Approval (G2.5) |
| **Frequency** | Per-event |
| **Input** | Approved blog post with SEO metadata from G2.5, brand config for the publishing brand |
| **Owner** | Founder |
| **Executor** | Blog Publisher - fully automated |
| **Output** | Blog post live on HubSpot CMS, WordPress, Medium, Substack; triggers Style Corrections Extraction (G2.8) and Content Performance Review (G2.12) |
| **Test** | Published to all 4 platforms within 1 hour; SEO metadata correct; internal links verified by SEO Linker |
| **Systems** | publishing-engine, Blog Publisher, CMS Publisher, HubSpot CMS, WordPress, Medium, Substack, SEO Linker |

**Steps:**
1. Approved blog post received from G2.5 Content QC & Approval (automated)
2. Blog Publisher publishes to HubSpot CMS (automated)
3. Blog Publisher publishes to WordPress (automated)
4. Blog Publisher publishes to Medium (automated)
5. Blog Publisher publishes to Substack (automated)
6. SEO Linker runs post-publish analysis (automated)

**Target automation:**
- Auto-trigger from Content Approval UI (G2.5) without manual handoff
- Post-publish SEO verification and internal linking suggestions auto-applied
- Publication status feeds into Content Performance Review (G2.12)

**Blockers:** Minor - needs brand config for each brand's blog publishing to be configured.
