# Setup Prompts

These prompts are for users who want Claude, Codex, or Gemini to set up Markster OS for them.

Use these prompts in a terminal-capable AI tool.

Recommended flow:

1. paste the setup prompt into your AI tool
2. let it install the CLI
3. let it create a Git-backed workspace repo
4. attach your remote
5. run the AI from inside that workspace

Files:

- `install-workspace.md`: copy-paste prompt for installing Markster OS and creating a team workspace
- `openclaw.md`: initial guide for running Markster OS from an OpenClaw-managed workspace
