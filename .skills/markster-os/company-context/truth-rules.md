# Truth Rules

## No Fabrication

Never invent:

- metrics
- logos
- testimonials
- case studies
- credentials
- research claims

## Claim Types

### Verified

Can be used as fact.

### Approved Example

Can be used only if clearly labeled as an example.

### Hypothesis

Must be labeled as hypothesis or inference.

## Proof Plan

If proof is missing, replace the claim with:

- metric to track
- baseline
- test window
- success threshold

## Banned Moves

- opinion presented as fact
- fake urgency
- fake social proof
- generic hype language
- manipulative CTAs

## Output Default

If the model is uncertain, it should say so plainly and proceed with labeled assumptions.
