# Offer

## Core Offer

- offer name
- what is included
- who it is for

## Entry Offer

- low-friction first step
- price if public
- timeline
- deliverable

## Outcome Statement

State the promised business outcome carefully.

If outcomes vary, label the variability.

## Pricing Logic

- pricing model
- what drives price up or down
- what is intentionally not optimized for

## Risk Reversal

- guarantee if any
- confidence mechanism if any
- what happens if fit is unclear

## Delivery Boundaries

- what we do
- what we need from the client
- what is excluded

## Upgrade Paths

- next logical offer
- expansion path
