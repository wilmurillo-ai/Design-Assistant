# Research Prompts

Nine structured prompts for B2B market and buyer research. Generic enough to work for any B2B company. Specific enough to produce usable output, not generic summaries.

These prompts feed directly into Foundation (F1-F4) and the active playbooks. Run them before building a list, writing a sequence, or positioning your offer.

---

| Prompt | When to run it |
|--------|---------------|

| [buyer-jobs-to-be-done-prompt.md](prompts/buyer-jobs-to-be-done-prompt.md) | When building F1 (ICP) or the messaging guide. Surfaces the buyer's language. |

## How to use these prompts

Each prompt has:
- **Purpose:** what you are trying to learn
- **Instructions:** how to run the research
- **Variables:** what to fill in for your specific situation
- **Output format:** how to structure and save what you find

Fill in the variables before running. Generic inputs produce generic outputs.

Run the `/research` skill in your AI environment for a guided version that asks the right clarifying questions.
