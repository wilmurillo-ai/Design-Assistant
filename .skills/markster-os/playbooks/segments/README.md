## Not covered here

Biotech, medtech, deep tech, cleantech, fintech (lending/payments), gaming.

These require regulatory approval, grant funding, or platform-level network effects before GTM applies in a standard way. The GOD Engine runs after you have product-market fit and a buyer who can say yes. For these archetypes, that gate is different. The methodology still applies once you clear it.

## How to use these files

1. Find your type above.
2. Read the file. Note which GOD bricks it flags as Stage 1 priority.
3. Run the scorecard at `methodology/assessment/scorecard.md` through that lens.
4. Start with the playbook your archetype file points to first.
