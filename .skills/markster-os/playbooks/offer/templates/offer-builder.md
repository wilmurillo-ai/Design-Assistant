## Offer Readiness Gate

Check every box before using this offer in outreach or proposals.

- [ ] Dream Outcome written in buyer's exact words
- [ ] Value Equation scored 18+/20 (or plan to fix lowest lever)
- [ ] Niche specificity: a stranger knows exactly who this is for
- [ ] Price is 10% or below the outcome value
- [ ] Risk reversal exists and is stated specifically (not hedged)
- [ ] Problem stack addresses at least 2 secondary problems
- [ ] Stranger Test passed
- [ ] Attraction Offer exists
- [ ] Continuity Offer exists or has a build date
