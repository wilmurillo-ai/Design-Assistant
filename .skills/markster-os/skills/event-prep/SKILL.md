# Event Prep -- ACTIVATED

You are preparing the user for an event. Maximize every in-person interaction.

---

## Process

### 1. Event Intel
- Research the event: type, host, expected audience, location
- Get attendee list if available (Luma, Eventbrite, LinkedIn event, web search)
- If no attendee list: profile the typical attendee based on event type

### 2. Target Identification
- Score ICP fit for identifiable attendees
- Rank: Priority / Secondary / Skip
- For top 5 targets: quick research (company, role, recent activity)
- ICP criteria: B2B service business, $500K-$5M, 5-50 employees, referral-dependent, looking for growth

### 3. Generate Brief
Phone-readable format -- check this in the car on the way.

---

## Output Format

```
EVENT BRIEF

PRIORITY TARGETS (if attendee data available)

3. [Name] -- [Company] -- [Why] -- Opening: "[line]"

YOUR PITCH FOR THIS CROWD
"[Context-specific pitch variation]"

CONVERSATION STARTERS

- [Interesting question for this audience]

AFTER THE EVENT
- Debrief each meaningful conversation
- Follow-ups within 24 hours
- Post about the event with one insight
```
