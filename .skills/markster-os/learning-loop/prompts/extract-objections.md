# Extract Objections Prompt

Read the provided raw material and extract only candidate customer objections.

## Output Rules

- use the candidate template structure
- do not promote anything directly
- separate exact buyer language from interpretation
- mark uncertain items as low confidence
- do not invent objections not supported by the material
