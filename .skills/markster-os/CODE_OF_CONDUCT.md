# Code of Conduct

## Our standard

This project is intended to be useful, direct, and safe for public contributors.

People participating in this project are expected to:

- be respectful and specific
- criticize ideas, not people
- keep discussions grounded in evidence
- avoid harassment, abuse, and intimidation
- avoid posting private or sensitive information

## Unacceptable behavior

Examples of unacceptable behavior include:

- personal attacks
- harassment or discriminatory language
- doxxing or posting private information
- spam, trolling, or deliberate disruption
- posting client data, secrets, or internal company material

## Maintainer responsibilities

Maintainers may remove, edit, or reject contributions and comments that violate this code of conduct.

Maintainers may also take reasonable moderation actions to protect contributors and the integrity of the project.

## Reporting

If you experience or witness unacceptable behavior, report it privately to:

- `hello@markster.ai`

Include enough context for maintainers to evaluate the situation.

## Scope

This code of conduct applies to:

- GitHub issues
- pull requests
- project discussions
- other public spaces used for this project

## Enforcement

Maintainers will review reports and decide on an appropriate response based on the severity and pattern of the behavior.
