# 技术分析完整指南

> 用于判断买卖时机、趋势确认、支撑阻力位识别

---

## 一、趋势分析系统

### 移动平均线(MA)

| 周期 | 名称 | 用途 |
|:----:|:-----|:-----|
| MA5 | 短期趋势线 | 日内波段参考 |
| MA10 | 超短期线 | 波段止损位 |
| MA20 | 中短期趋势线 | **主力参考线**（机构常用）|
| MA50 | 中期趋势线 | 牛熊分界辅助线 |
| MA120 | 半年线 | 中期多空分界 |
| MA200 | **牛熊分界线** | 长期趋势决定者 |

**关键形态**：
- 🟢 **金叉**：短期MA上穿长期MA → 买入信号（尤其是MA50上穿MA200=黄金交叉，最强做多信号）
- 🔴 **死叉**：短期MA下穿长期MA → 卖出信号（尤其是MA50下穿MA200=死亡交叉）
- 价格在MA上方 = 多头排列 = 上升趋势
- 价格在MA下方 = 空头排列 = 下降趋势

### MACD（指数平滑异同移动平均线）

```
MACD线 = EMA12 - EMA26
Signal线 = MACD的EMA9
柱状图(Histogram) = MACD - Signal
```

| 信号 | 图形特征 | 可靠性 |
|:-----|:--------:|:------:|
| 金叉 | MACD上穿Signal线 | ⭐⭐⭐ |
| 死叉 | MACD下穿Signal线 | ⭐⭐⭐ |
| **多头背离** | 价格创新低但MACD没有 | ⭐⭐⭐⭐⭐ **最强信号！** |
| **空头背离** | 价格创新高但MACD没有 | ⭐⭐⭐⭐⭐ **最强信号！** |
| 零轴上方金叉 | MACD从零轴上方金叉 | ⭐⭐⭐⭐ |
| 零轴下方死叉 | MACD在零轴下方死叉 | ⭐⭐⭐⭐ |

**背离详解**（最重要！）：
- 底部背离：价格创阶段新低 → RSI/MACD/指标未创新低 → 强烈看多信号
- 顶部背离：价格创阶段新高 → RSI/MACD/指标未创新高 → 强烈看空信号
- 背离出现后，反转概率>70%，但时间不确定（可以提前1-4周）

## 二、动量指标

### RSI（相对强弱指数）

```
RSI = 100 - 100/(1 + RS)
RS = N日涨幅均值/N日跌幅均值
```
默认参数：N=14

| 区域 | RSI值 | 含义 | 操作建议 |
|:-----|:-----:|:-----|:---------|
| 超卖区 | <30 | 卖压过度，可能反弹 | 寻找买入机会 |
| 弱势区 | 30-50 | 空头主导 | 观望或轻仓 |
| 中性区 | 50 | 多空均衡 | 方向待定 |
| 强势区 | 50-70 | 多头主导 | 持有为主 |
| 超买区 | >70 | 买盘过度，可能回调 | 考虑止盈 |

**RSI最强用法：背离！（见上文MACD背离）**

### KDJ（随机指标）

| K/D/J值 | 区域 | 含义 |
|:-------:|:----:|:-----|
| >80 | 超买 | 可能回调 |
| <20 | 超卖 | 可能反弹 |

特点：比RSS更敏感（反应更快），适合短线操作。J线方向变化领先于K/D线。

## 三、 volatility指标

### 布林带(Bollinger Bands)

```
中轨 = MA(20)
上轨 = MA(20) + 2σ
下轨 = MA(20) - 2σ
```

| 形态 | 特征 | 操作意义 |
|:-----|:-----|:---------|
| **收口(Squeeze)** | 带宽急剧收窄 | 变盘在即！即将选择突破方向 |
| **开口(Expansion)** | 带宽突然放大 | 方向性运动已经开始 |
| **触及下轨反弹** | 价格到下轨后回升 | 支撑有效，可买入 |
| **触及上轨回落** | 价格到上轨后下跌 | 压力有效，可卖出 |
| **穿越上轨** | 价格突破上轨运行 | 极强势，可能延续 |
| **跌破下轨** | 价格跌破下轨运行 | 极弱势，可能延续 |

**布林带宽(BBW) = (上轨-下轨)/中轨**：BBW处于历史低位 = 即将大波动！

### ATR（真实波幅）
- 衡量波动率的绝对值
- 主要用途：设置止损位（如：ATR×2 作为止损距离）

## 四、量价关系（最基础却最有效）

### 核心原则：**量是价的先行指标**

| 价格走势 | 量能表现 | 含义 | 可靠性 |
|:---------|:-------:|:-----|:------:|
| 涨 | 放量 | 真上涨，资金进场 | ⭐⭐⭐⭐⭐ |
| 涨 | 缩量 | 动能衰减，可能见顶 | ⭐⭐⭐ |
| 跌 | 放量 | 真跌，资金出逃 | ⭐⭐⭐⭐⭐ |
| 跌 | 缩量 | 抛压减轻，可能企稳 | ⭐⭐⭐⭐ |
| 横盘 | 逐步放量 | 筹码收集中 | ⭐⭐⭐⭐ |
| 横盘 | 地量 | 筹码锁定，变盘在即 | ⭐⭐⭐⭐ |

### OBV（能量潮/On-Balance Volume）
```
今日OBV = 昨日OBV + sign(今日收盘-昨日收盘) × 今日成交量
sign() = 若收盘上涨则为正，下跌则为负，平盘为零
```

| OBV vs 价格 | 含义 |
|:-----------:|:-----|
| OBV↑ + 价格↑ | 多头强势 ✅ |
| OBV↓ + 价格↓ | 空头强势 ❌ |
| **OBV走平 + 价格创新高** | **顶部背离！警惕！** ⚠️ |
| **OBV走平 + 价格创新低** | **底部背离！机会！** 💚 |

## 五、经典图表形态

### 反转形态（预示趋势改变）

| 形态 | 图形描述 | 可靠性 | 确认条件 |
|:-----|:---------|:------:|:---------|
| **头肩顶** | 左肩-头部-右肩+颈线 | ⭐⭐⭐⭐⭐ | 跌破颈线+放量 |
| **头肩底** | 倒转的头肩顶 | ⭐⭐⭐⭐⭐ | 突破颈线+放量 |
| **双顶(M顶)** | 两峰相近高度 | ⭐⭐⭐⭐ | 跌破颈线 |
| **双底(W底)** | 两谷相近高度 | ⭐⭐⭐⭐ | 突破颈线 |
| **圆弧顶** | 缓慢弧形向下 | ⭐⭐⭐ | 跌破弧形切线 |
| **圆弧底** | 缓慢弧形向上 | ⭐⭐⭐ | 突破弧形切线 |

### 持续形态（预示趋势继续）

| 形态 | 图形描述 | 方向 | 确认条件 |
|:-----|:---------|:-----|:---------|
| **旗形** | 短矩形倾斜于原趋势 | 沿原方向继续 | 突破旗形边界 |
| **三角旗形** | 小三角形收敛 | 沿原方向继续 | 向上/下突破 |
| **上升三角形** | 水平顶+斜向上升底 | **大概率向上突破** | 突破水平顶线 |
| **下降三角形** | 水平底+斜向下降顶 | **大概率向下突破** | 跌破水平底线 |
| **对称三角形** | 两边对称收敛 | 待方向选择 | 向任一方向突破+放量 |
| **箱体震荡** | 水平区间来回 | 直到突破 | 突破箱体上下沿+放量 |

### 缺口理论

| 类型 | 位置 | 含义 |
|:-----|:-----|:-----|
| **普通缺口** | 横盘中 | 无特殊意义，很快补回 |
| **突破缺口** | 形态突破时 | 强势信号，不轻易补回 |
| **中继缺口** | 趋势中途 | 趋势加速，量能配合 |
| **衰竭缺口** | 趋势末端 | 力竭信号，即将反转 |

## 六、支撑与阻力

### 如何识别关键价位

**按可靠性排序**：
1. 前期高点/低点（多次触碰过的更强）
2. 整数关口（心理价位：如沪指4000点、创业板3000点）
3. 重要均线（MA200 > MA50 > MA20）
4. 斐波那契回撤位：23.6% / 38.2% / 50% / 61.8% / 78.6%
5. 布林带上/下轨
6. 前期缺口位置

### 支阻转换原则
> "阻力一旦被突破，就变成了支撑。支撑一旦被跌破，就变成了阻力。"

这是技术分析中最可靠的原理之一！

## 七、综合分析框架

### 分析流程（每次必做）

```
Step 1: 确定趋势（MA系统）
  ├── 价格在MA200上方？→ 大趋势偏多
  ├── MA50与MA200关系？→ 黄金/死叉
  └── MA排列方式？→ 多头/空头/纠缠

Step 2: 确定位置（支撑阻力）
  ├── 距离最近支撑/阻力多远？
  ├── 是否在关键整数关口附近？
  └── 斐波那契位置？

Step 3: 确认动能（动量指标）
  ├── RSI在哪个区域？超买/超卖？
  ├── MACD金叉/死叉？有无背离？
  └── 布林带位置？收口还是开口？

Step 4: 验证量价
  ├── 当前趋势是否有量能配合？
  ├── OBV方向与价格一致吗？
  └── 近期成交量趋势如何？

Step 5: 查找形态
  ├── 是否形成经典反转/持续形态？
  ├── 有无缺口？什么类型？
  └── 形态目标价位测算

Step 6: 综合判断
  ├── 多空因素加权
  ├── 风险/收益比评估
  └── 给出明确结论 + 止损位
```

## 八、常见陷阱与避坑

| 陷阱 | 为什么错 | 正确做法 |
|:-----|:--------:|:---------|
| **单一指标决策** | 没有任何指标100%可靠 | 多指标共振确认 |
| **过度优化参数** | 历史拟合 ≠ 未来有效 | 使用通用参数（MA20/50/200, RSI14）|
| **忽视大环境** | 个股技术面再好也扛不过大盘暴跌 | 先看大盘再看个股 |
| **逆势抄底** | "这次不一样"是最贵的话 | 等趋势企稳再入场 |
| **频繁交易** | 每次交易都有成本+心理损耗 | 减少操作次数，提高单次质量 |
| **事后确认偏差** | 图表上看什么都准 | 在当下做决策，记录预判并验证 |
