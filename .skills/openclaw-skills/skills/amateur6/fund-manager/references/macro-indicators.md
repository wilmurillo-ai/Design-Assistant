# 宏观经济指标手册

> 用于解读每日宏观数据、研判经济周期、理解政策动向

---

## 一、通胀指标

### CPI（消费者物价指数）

💡 **定义**：衡量居民消费品和服务价格变化的指标，反映普通人生活成本。

**为什么重要**：
- 央行制定货币政策的核心参考
- CPI过高 → 通胀压力 → 央行可能加息 → 利空股市（分母端变大）
- CPI过低甚至负值 → 通缩风险 → 央行可能降息 → 利好股市

**怎么看**：
| 数值 | 含义 | 政策倾向 |
|:----:|:-----|:---------:|
| <1% | 内需疲软，通缩风险 | 降息降准刺激 |
| ~2% | 健康（大多数央行目标） | 维持现状 |
| 2-3% | 温和通胀 | 开始关注 |
| >3% | 通胀压力较大 | 加息预期升温 |
| >5% | 高通胀（恶性风险） | 大幅加息 |

**对A股影响链**：
CPI ↑ → 通胀压力 → LPR/MLF可能上调 → 企业融资成本↑ → 利润承压 → 利空
但：温和CPI↑(1-3%)说明消费复苏 → 对消费板块利好

### PPI（生产者物价指数）

💡 **定义**：衡量工业企业产品出厂价格变化的指数。

**关键特性**：
- CPI的**先行指标**（PPI上涨约3-6个月传导至CPI）
- 反映企业生产成本和工业景气度

**解读**：
| PPI状态 | 含义 | 对股市影响 |
|:-------:|:-----|:-----------|
| 持续为负 | 工业通缩，企业利润承压 | 利空工业企业 |
| 由负转正 | 工业景气度改善，拐点信号 | **利好！**（如2026年4月中国PPI结束41连降转正）|
| 快速上涨 | 成本推动型通胀 | 上游原材料利好，下游承压 |

## 二、经济景气指标

### PMI（采购经理指数）

💡 **定义**：通过对采购经理问卷调查得到的综合景气指数。

**为什么重要**：经济的"温度计"，**每月最先发布的经济数据**。

**核心规则**：**50 是生死线**
- PMI > 50：制造业扩张（订单增加、生产活跃）
- PMI < 50：制造业收缩（订单减少、生产萎缩）

**子指标解读**：

| 子指标 | 权重 | 含义 |
|:-------|:----:|:-----|
| 新订单指数 | 最高 | 需求端，最前瞻 |
| 生产指数 | 高 | 供给端，同步指标 |
| 新出口订单 | 中 | 外需情况 |
| 从业人员 | 中 | 就业情况（滞后）|
| 原材料库存 | 低 | 企业备货意愿 |
| 供应商配送时间 | 低 | 物流效率（反向指标）|

**对A股影响**：
- 制造业PMI > 50 且上升 → 利好工业/制造/周期板块
- PMI持续低于50且下降 → 经济下行压力大 → 政策宽松预期升温

### GDP（国内生产总值）

💡 **定义**：一国在一定时期内生产的所有最终产品和服务的市场价值。

**怎么看**：
| 增速 | 中国 | 发达国家 | 含义 |
|:----:|:----:|:--------:|:-----|
| >7% | 高速增长 | - | 繁荣期，可能过热 |
| 5-6% | **健康区间** | - | 中国新常态下的理想增速 |
| 3-5% | 中速 | 健康 | 温和增长 |
| 0-3% | 低速 | 较弱 | 增长乏力 |
| <0% | - | 衰退 | 经济危机 |

**GDP发布节奏**：季度发布（初步核算->核实->最终），对市场影响较大。

## 三、就业指标

### 失业率

💡 **定义**：失业人口占劳动人口的百分比。

| 数值 | 状态 | 政策含义 |
|:----:|:-----|:---------|
| <3.5% | 劳动力过热 | 工资通胀风险，可能加息 |
| 4-5% | 充分就业 | 健康状态 |
| >6% | 就业疲软 | 刺激就业政策预期 |

**对A股影响间接但深远**：就业→消费→企业营收

## 四、货币政策指标

### LPR（贷款市场报价利率）

💡 **定义**：银行对最优质客户执行的贷款利率基准。

**直接影响**：
- LPR下调 → 房贷月供减少 → 居民可支配收入↑ → 消费↑
- LPR下调 → 企业贷款成本↓ → 扩产意愿↑ → 投资↑
- **LPR下调 = 全面利好A股！**（降低全市场折现率）

**LPR调整时机跟踪**：每月20日公布（遇节假日顺延）。MLF利率是LPR的定价基础。

### MLF（中期借贷便利）

💡 **定义**：央行向商业银行提供中期基础货币的工具。

**操作逻辑**：
- MLF利率下调 → LPR大概率跟随下调
- MLF投放量增加 → 央行在释放流动性 → 市场资金宽裕
- MLF缩量/利率不变 → 货币政策维持稳健

### 存准率(RRR)

💡 **定义**：商业银行必须在央行存放的准备金占存款的比例。

- **降准**：银行可贷资金增加 → 释放流动性 → 利好股市楼市
- **升准**：回收流动性 → 利空

### 社融 & M2

| 指标 | 定义 | 含义 |
|:-----|:-----|:-----|
| **社融** | 实体经济从金融体系获得的资金总额 | 实体融资需求，经济活力的先行指标 |
| **M2** | 广义货币供应量 | 市场流动性总量 |

**关键关系**：社融 > M2 = 实体需求旺盛，资金流入实体经济（好）
社融 < M2 = 资金在金融体系空转，实体需求不足（差）

## 五、市场指标

### 十年期国债收益率

💡 **定义**：政府发行的10年期债券的年化收益率 = 无风险利率基准。

**对股市的影响（非常重要！）**：

```
股票估值模型：股价 = 未来现金流 / (无风险利率 + 风险溢价)

国债收益率↑ → 分母变大 → 所有股票估值↓ → 尤其伤害高估值成长股
国债收益率↓ → 分母变小 → 所有股票估值↑ → 尤其利好成长股
```

| 收益率状态 | 含义 | 对股市影响 |
|:----------:|:-----|:-----------|
| 快速上行 | 通胀预期/紧缩预期 | **重大利空！**估值承压 |
| 稳定低位 | 宽松环境 | 利好 |
| **曲线倒挂**（短>长）| **衰退预警信号！** | 风险极高 |

### 北向资金

💡 **定义**：通过沪深港通进入A股的外资（"聪明钱"风向标）。

| 单日净流向 | 含义 |
|:----------:|:-----|
| >100亿 | 强力做多，外资看好A股 |
| 50-100亿 | 温和流入 |
| 0附近 | 观望 |
| <-50亿 | 温和流出 |
| <-100亿 | 大幅撤离，注意风险 |

### 两融余额

💡 **定义**：投资者通过融资（借钱炒股）+融券（借券做空）的资金余额。

- 两融余额快速↑ = 杠杆资金活跃 = 市场情绪高涨 = **但也意味着风险积累**
- 两融余额快速↓ = 去杠杆进行中 = **可能触发连锁平仓**

## 六、美林时钟框架

```
                    GDP增速 ↑
                       │
    ┌─────────────────┼─────────────────┐
    │    过热期        │      滞胀期      │
    │ (商品 > 股 > 债)│ (现金 >一切)     │
    │ 通胀↑ 利率↑     │ 通胀↑ GDP↓       │
    ├─────────────────┼─────────────────┤
    │    复苏期        │      衰退期      │
    │ (股 > 债 > 现)  │ (债 > 现 > 股)  │
    │ 通胀↓ 利率↓     │ 通胀↓ GDP↓       │
    └─────────────────┼─────────────────┘
                    GDP增速 ↓
                    
                      通胀 ↑
```

| 阶段 | 最佳资产 | 最差资产 | 特征 |
|:-----|:--------:|:--------:|:-----|
| **复苏** | 股票 > 债券 > 现金 > 商品 | 商品 | GDP加速，通胀下降，利率低位，最佳投资窗口 |
| **过热** | 商品 > 股票 > 现金 > 债券 | 债券 | GDP高位，通胀上升，开始加息，获利了结 |
| **滞胀** | 现金 > 商品 > 债券 > 股票 | **股票** | GDP放缓，通胀高企，最差环境 |
| **衰退** | 债券 > 现金 > 股票 > 商品 | 商品 | GDP下行，通胀下降，开始降息，防守为主 |

**2026Q2 中国所处位置判断**：**复苏初期** → 股票优于其他资产 ✅
依据：PMI回到扩张区、PPI转正、政策持续宽松、LPR低位

|:-----|:-----|:-----------|
| **美联储利率决议** | 美联储 | 全球流动性锚点，影响外资流向 |
