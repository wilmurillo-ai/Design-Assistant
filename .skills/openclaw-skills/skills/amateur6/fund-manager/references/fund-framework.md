## 一、选基五维评分模型

### 维度1：历史业绩（权重25%）

| 子维度 | 评分标准 |
|:-------|:--------|
| 近1年排名 | 同类前1/3=5分，前1/2=3分，后1/2=1分 |
| 近3年排名 | 同类前1/4=5分，前1/3=3分 |
| 最大回撤 | <10%=5分，10-20%=3分，>30%=1分 |
| 夏普比率 | >2=5分，1-2=3分，<1=1分 |
| 牛熊表现 | 牛市跟上+熊市少跌=5分 |

### 维度2：基金经理（权重25%）

| 子维度 | 评分标准 |
|:-------|:--------|
| 从业年限 | >10年=5分，5-10年=3分，<5年=1分 |
| 管理风格一致性 | 长期坚持同一策略=5分，经常漂移=1分 |
| 回撤控制能力 | 历史最大回撤<同类平均=5分 |
| 任职期间收益 | 年化>15%=5分，10-15%=3分 |
| 持仓透明度 | 定期公开持仓=5分，从不公开=1分 |

### 维度3：基金公司（权重15%）

| 子维度 | 评分标准 |
|:-------|:--------|
| 研究团队实力 | 明星研究员数量、覆盖广度 |
| 产品线完整性 | 提供多种策略选项=好 |
| 风控体系 | 有完善风控流程=加分 |
| 品牌口碑 | 行业声誉良好=加分 |

### 维度4：持仓质量（权重20%）

| 子维度 | 评分标准 |
|:-------|:--------|
| 重仓股基本面 | 各重仓股ROE>15%、护城河明显 |
| 行业分散度 | 前10重仓不超过3个行业=好 |
| 换手率 | <200%/年=5分（低换手=长期主义）|
| 规模适配 | 规模在基金经理能力圈内 |

### 维度5：费用效率（权重15%）

| 费用项 | 合理范围 |
|:-------|:--------|
| 管理费 | 股票型1-1.5%，债券型0.5-1% |
| 托管费 | 0.1-0.25%/年 |
| 申购费 | 通常1折后0.12-0.15% |
| 赎回费 | 持有>30天免赎回费（C类通常无）|
| 销售服务费 | C类0.4-0.8%/年（A类无）|

### A/C类选择指南
| 场景 | 选择 |
|:-----|:-----|
| 长期持有(>1年) | **A类**（一次性收取申购费）|
| 短期/波段操作 | **C类**（销售服务费按天计提，无申购费）|
| 不确定持有期 | C类更灵活 |

### 理想仓位模型（目标配置）

| 赛道 | 当前 | 目标范围 | 操作建议 |

| AI合计(025493+008586+科创AI) | 55% | 40-50% | 偏高，止盈后降至目标 |

| 新仓合计(低空+创新药) | 7% | 10-15% | 可继续观察加仓 |
| 现金(可用) | ~0 | 10-15% | 需要积累现金储备 |

### 加仓/减仓节奏

**加仓原则**：
1. 分批入场（不要一次性all-in）
2. 跌幅达到阈值再加（如-5%加一笔，-10%再加一笔）
3. 只在明确信号时加仓（不因"便宜"而加仓）

**减仓原则**：
1. 达到止盈线时分批减（先减30%，观察后再减）
2. 赛道逻辑发生变化时果断减仓
3. 发现更好的机会时，从弱势仓位转移

## 四、基金诊断清单

当你需要评估某只基金时，按此顺序检查：

### 第一步：基本信息
- [ ] 基金名称、代码、类型（主动/被动）
- [ ] 成立日期、规模（太小<5000万或太大>500亿都有问题）
- [ ] 基金经理（从业年限、任职时长、管理其他基金情况）
- [ ] 费率结构（管理费/托管费/申购赎回费）

### 第二步：业绩表现
- [ ] 近1/3/5年收益率及排名
- [ ] 最大回撤及恢复时间
- [ ] 与业绩基准的比较（跑赢/跑输多少）
- [ ] 牛市和熊市分别表现如何

### 第三步：持仓分析
- [ ] 最新持仓（前10重仓股是什么）
- [ ] 行业分布（集中在哪些行业）
- [ ] 持仓集中度（前10占比多少）
- [ ] 换手率（频繁调仓还是长期持有）
- [ ] 重仓股基本面（这些公司好不好）

### 第四步：风险评估
- [ ] 波动率（标准差）
- [ ] 夏普比率（风险调整后收益）
- [ ] 与现有持仓的相关性（是否分散）
- [ ] 最大连续亏损天数

### 第五步：综合判断
- [ ] 这只基金在我们的能力圈内吗？
- [ ] 它解决了我们组合中的什么问题？
- [ ] 它的风险我们能承受吗？
- [ ] 它的费用合理吗？

## 五、常见基金陷阱

| 陷阱 | 表现 | 如何规避 |
|:-----|:-----|:---------|
| **冠军魔咒** | 今年第一明年垫底 | 看3-5年业绩而非仅看今年 |
| **规模陷阱** | 太小易清盘，太大规模难操作 | 选择5-100亿规模的基金 |
| **风格漂移** | 名为"价值"其实追热点 | 查看持仓是否与宣称一致 |
| **明星基金经理陷阱** | 基金经理离职/跳槽后业绩大跌 | 关注基金经理稳定性+后备团队 |
| **高费用侵蚀收益** | 各种费用加起来吃掉2-3%年收益 | 优先选C类(短期)或打1折申购(A类)|
| **追热点发产品** | 公司在最热门的时候发新产品 | 冷门时发行的基金往往更好 |
| **分红幻觉** | 分红只是左口袋到右口袋（净值扣除）| 不要因为高分红就买入 |

## 六、定投策略补充

虽然我们目前以一次性投入为主，但定投知识仍重要：

### 定投优势
- 摊平成本（自动实现"低买多买，高买少买"）
- 克服人性弱点（不需要择时）
- 降低一次性投入风险
- 适合工薪族（每月固定金额）

### 定投策略选择

| 策略 | 做法 | 适用场景 |
|:-----|:-----|:---------|
| **普通定投** | 固定日期固定金额 | 最简单，适合新手 |
| **估值定投** | 低估值多投，高估值少投/停止 | 进阶策略，效果更好 |
| **均线定投** | 指数偏离均线越多投入越多 | 自动化的估值策略 |
| **目标止盈定投** | 达到目标收益率后止盈重启 | 解决"何时卖出"问题 |

### 我们的定投标的（如有新增资金）
推荐：创业板AI(025493) + 华夏AI(008586) — 核心主线，长期看好
- 定投频率：每周二/四（避开周一开盘效应和周五周末效应）
- 单笔金额：50-100元/次
- 止盈条件：单只累计盈利>30%时可暂停定投并考虑部分止盈
