# 口碑圈功能操作详解

## 01. 店铺互通

### 店铺导航直达
**配置路径**：店铺装修→导航设置
**操作步骤**：
1. 登录有赞商家后台
2. 进入店铺装修 → 导航设置
3. 添加"口碑圈"导航项
4. 配置跳转链接（圈子 URL）
5. 保存并发布

### 店铺积分互通
**配置路径**：口碑圈后台→积分设置
**操作步骤**：
1. 开启"与店铺积分互通"
2. 设置积分兑换比例
3. 配置积分获取规则（发帖/点赞/评论等）

### 社区优惠券
**配置路径**：营销中心→优惠券
**操作步骤**：
1. 创建店铺优惠券
2. 选择"在社区发放"
3. 设置领取条件（如：发帖后领取）
4. 配置优惠券面额和使用规则

### 社区商品贴
**操作步骤**：
1. 发帖时点击"添加商品"
2. 从店铺商品库选择商品
3. 支持添加多个商品
4. 商品链接自动嵌入帖子

### 同步店铺会员
**配置路径**：用户管理→会员同步
**操作步骤**：
1. 开启会员同步功能
2. 选择同步的会员等级
3. 配置会员权益映射

### 同步商品分组
**配置路径**：商品管理→分组同步
**操作步骤**：
1. 选择需要同步的商品分组
2. 开启自动同步
3. 设置同步频率

2. 配置排序规则（最新/最热/推荐）

### 自定义导航
**配置路径**：首页设置→导航配置
**操作步骤**：
1. 添加导航项
2. 配置导航名称和图标
3. 设置跳转链接

**配置路径**：互动功能→打卡
**操作步骤**：
1. 开启打卡功能
2. 配置打卡奖励
3. 设置打卡周期（每日/每周）

1. 点击"发布帖子"
2. 选择内容类型（图文/视频）
3. 上传内容
4. 添加话题标签
5. 关联商品（可选）
6. 发布

### 内容互动

**配置路径**：互动设置

### 内容管理
**配置路径**：内容管理

### 内容搜索

2. 配置搜索权重

### 话题分类

2. 设置话题图标和描述
3. 关联帖子

### 连续阅读
**配置路径**：阅读设置→连续阅读
**操作步骤**：
1. 开启连续阅读模式
2. 配置加载方式（自动/手动）

### 图片水印
**配置路径**：设置→水印管理
