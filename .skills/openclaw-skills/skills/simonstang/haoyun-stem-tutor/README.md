## ❓ Why It Matters | 为什么重要

### 📉 传统理科学习的痛点

```
┌─────────────────────────────────────────────────────────┐
│  传统方式                    浩云学长方式                  │
│                                                         │
│  知识点孤立 ───────→  知识网络串联                       │
│  只见树木 ────────→  既能见树木，也能见森林               │
│  死记硬背 ────────→  已知推未知，逻辑推导                 │
│  缺什么不知道 ─────→  前置知识自动检测+补救               │
│  学完就忘 ────────→  跨学科关联加深理解                   │
│                                                         │
│  提分效率 ████████░░  →  ████████████  (+50%+)          │
└─────────────────────────────────────────────────────────┘
```

- 🚫 **知识点孤立** — 学了二次函数，忘了它与一次函数的关系
- 🚫 **只见树木不见森林** — 知道公式，不知道为什么是这个公式
- 🚫 **缺什么不知道** — 前置知识有缺口，越学越吃力
- 🚫 **跨学科割裂** — 数学课上学的函数，物理课上不会用

### ✅ 浩云学长的解决方案

1. **知识网络导航** — 每个知识点都有前置知识和后继知识
2. **前置知识检测** — 学新知识前，先确认基础是否牢固
3. **已知推未知** — 用你已掌握的知识做桥梁，推导新知识
4. **跨学科关联** — 函数不只是数学，也是物理和化学的语言

### 📈 效果：提分效率提升50%+

> 💬 *学生反馈："以前觉得二次函数和物理运动完全没关系，浩云学长一讲，原来就是一回事！"*

## 🚀 快速开始 | Quick Start

### 在 OpenClaw 中使用

```bash
# 直接对浩云学长说：
@浩云学长 我二次函数不会

# 浩云会自动：
# 📌 Step 1: 检测前置知识 — 你的一次函数掌握得怎么样？
# 📌 Step 2: 如有缺口先补基础 — 一次函数 y=kx+b 复习
# 📌 Step 3: 用已知推导未知 — 在一次函数基础上加一个二次项
# 📌 Step 4: 触发跨学科关联 — 二次函数 = 抛体运动的数学描述
```

```bash

# 在 OpenClaw 中加载技能
# 将 SKILL.md 导入 OpenClaw Skill 系统
# knowledge_base.md 和 output_templates.md 放入 references/ 目录
```
