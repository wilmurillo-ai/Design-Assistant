# 波龙选股系统 V2.1 优化文档

## 📋 问题分析

唐总反馈的核心问题：**"追涨杀跌"**，选的股票都是已经涨了2-3倍的高位股（招商轮船、中远海能）。

### 风险点
- 招商轮船：涨幅120%，RPS=95，PE=45
- 中远海能：涨幅140%，RPS=97，PE=52

---

## 🔧 优化方案

| 优化项 | 旧策略问题 | 新策略方案 |
|--------|------------|------------|
| **涨幅过滤** | 无 | 近3月≤50%，近1月≤25% |
| **RPS优化** | 无 | RPS 70-85最佳区间，排除>90 |
| **估值筛选** | PE<60 | **PE 5-30**，PB<5 |
| **位置判断** | 无 | 排除距高点<5%的高位股 |
| **趋势过滤** | 无 | 排除涨停股，优先中低涨幅 |

## 📌 核心改进

**投资理念**：从"追涨杀跌"转变为"价值投资"
- 不追高，选择刚启动的股票
- 低估值优先（PE<30）
- RPS 70-85 最佳区间（不是越高越好）
- 排除高位震荡股
