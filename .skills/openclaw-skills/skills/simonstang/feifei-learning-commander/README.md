## 🔄 工作流程 / Workflow

```
学生提问 → 菲菲识别 → 知识网络定位 → 前置知识检查
Student asks → Feifei identifies → Knowledge locate → Prerequisite check
    ↓
有缺口 → 分派浩云/小菲补基础 → 再学新知识
Gap found → Dispatch tutor to fill → Then learn new
    ↓
无缺口 → 用已知推导未知 → 跨学科关联
No gap → Derive new from known → Cross-subject links
    ↓
更新知识网络 → 生成学习报告 → 心跳提醒下次学习
Update knowledge map → Generate report → Heartbeat next session
```

## 🔑 核心特色 / Key Features

### 1. 🧠 知识网络总控 / Knowledge Network Control

菲菲老师掌握每位学生的**完整知识地图**，包括：
- 每个知识点的掌握程度（0-100%）
- 知识点之间的前置/后续关系
- 跨学科连接点
- 薄弱点实时诊断

### 2. 📚 教材索引导航 / Textbook Index Navigation

支持全国主流教材版本：
- **数学**: 人教版A/B版、北师大版、苏教版、沪教版、湘教版
- **语文**: 统编版（全国统一）
- **英语**: 人教版、外研社版、北师大版
- **物理**: 人教版、沪粤版、苏科版
- **化学**: 人教版、沪教版、鲁教版

学生说"第三单元不会" → 自动定位到具体知识点 + 前置知识检查

### 3. 💓 心跳服务 / Heartbeat Service

- **主动唤醒**: 到点提醒学习
- **主动发任务**: 下达今日学习指令
- **主动校验**: 确认完成与否
- **主动循环**: 进入下一轮学习

### 4. 📊 学习报告 / Learning Reports

- 📋 每日学习简报
- 📈 每周学习周报
- 📊 每月学习月报
- 🎯 考前冲刺计划
- 🔬 薄弱点诊断报告
