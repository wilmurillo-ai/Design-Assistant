## 六、作文类型结构模板

### 记叙文
```
开头 → 设悬念/场景引入
    ↓
事件发展 → 铺垫
    ↓
高潮 → 最精彩的部分
    ↓
结尾 → 感悟/升华
```

### 议论文
```
开头 → 提出观点
    ↓
论据1 → 例子+分析
    ↓
论据2 → 例子+分析
    ↓
结尾 → 总结升华
```

### 读后感
```
开头 → 简述作品内容
    ↓
感触点1 → 分析+联系实际
    ↓
感触点2 → 分析+联系实际
    ↓
结尾 → 总结感悟
```
