# 贡献指南 | Contribution Guide

## 🌟 欢迎贡献 | Welcome Contributions

感谢您对小菲学姐文科辅导项目的关注！我们欢迎各种形式的贡献。

Thank you for your interest in the Xiao Fei Liberal Arts Tutor project! We welcome all forms of contributions.

## 📝 贡献方式 | Ways to Contribute

### 1. 提交问题 | Submit Issues
- 🐛 报告 Bug
- 💡 提出功能建议
- 📚 补充诗词、作文、英语素材

### 2. 提交代码 | Submit Code
- 🔧 修复 Bug
- ✨ 添加新功能
- 🌐 改进中英双语内容

### 3. 分享使用体验 | Share Experience
- ⭐ 给项目点赞
- 📣 推荐给朋友
- 📝 写使用心得

## 🎯 知识贡献 | Knowledge Contributions

我们特别欢迎以下内容的补充：

We especially welcome additions of:

- 📖 更多古诗词及赏析
- 📝 优秀作文范文及点评
- 🔤 英语写作句型和技巧
- 🎓 考点精讲和真题解析
- 📚 教材同步内容

## 📋 提交规范 | Submission Guidelines

1. **Fork 项目** → 修改 → 提交 Pull Request
2. **Issue 模板**：请描述清楚问题或建议
3. **Commit 规范**：
   - `feat:` 新功能
   - `fix:` 修复问题
   - `docs:` 文档更新
   - `content:` 内容补充
