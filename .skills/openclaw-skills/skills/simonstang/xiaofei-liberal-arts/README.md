### 示例：李白"长风破浪"的知识网络 | Example: Li Bai's "Riding the Wind"

```
📖 原诗词 | Original Poetry
"长风破浪会有时，直挂云帆济沧海" — 李白《行路难》
"A time will come to ride the wind and cleave the waves; 
I'll set my cloud-white sail and cross the sea which raves"

         │
         ▼
┌─────────────────────────────────────────────────────┐
│ 🎨 意象提取 | Imagery Extraction                     │
│ 长风破浪 = 勇往直前、不畏艰险                        │
│ 云帆沧海 = 远大志向、广阔前程                        │
└─────────────────────────────────────────────────────┘
         │
         ▼
┌─────────────────────────────────────────────────────┐
│ 📝 作文应用 | Essay Application                      │
│ 主题：坚持、乐观、面对挫折                            │
│ Theme: Perseverance, Optimism, Facing Setbacks       │
│                                                     │
│ 引用示范：                                            │
│ "正如李白所吟'长风破浪会有时'，面对学习中的困难，    │
│  我也应当保持乐观，相信终有成功的一天"               │
└─────────────────────────────────────────────────────┘
         │
         ▼
┌─────────────────────────────────────────────────────┐
│ 🔤 英语迁移 | English Transfer                       │
│ 话题：Perseverance, Overcoming Challenges            │
│                                                     │
│ 英语表达：                                            │
│ "As Li Bai wrote, 'A time will come to ride the wind │
│  and cleave the waves.' This spirit of perseverance  │
│  inspires me to face challenges bravely."            │
└─────────────────────────────────────────────────────┘
```

### 典型对话示例 | Example Conversations

**场景1：诗词学习 | Poetry Learning**
```
用户：帮我讲解《登高》
小菲：🌙 这首诗是杜甫晚年代表作...
      📖 先来了解创作背景...
      🎨 意象分析：急风、高天、哀猿...
      📝 这些意象可以作文中怎么用...
```

**场景2：作文辅导 | Essay Coaching**
```
用户：我要写"坚持"主题的作文
小菲：🎯 推荐几个好素材：
      1. 李白"长风破浪会有时"——乐观坚持
      2. 荀子"锲而不舍，金石可镂"——持之以恒
      3. 袁隆平研究杂交水稻——科学坚持
      
      你想用哪个？我来帮你展开...
```

**场景3：英语学习 | English Learning**
```
用户：怎么用英语表达"思乡"？
小菲：🌙 中文有"举头望明月，低头思故乡"
      🔤 英语可以说：
      - Nostalgia for one's homeland
      - Homesickness creeps in when...
      - The moon reminds me of home...
      
      试试用这个写一段？
```
