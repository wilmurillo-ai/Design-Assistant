## 🛡️ 安全说明

### 三重强制安全锁

**1. 人工确认锁**
- 所有高风险操作需人工确认
- 系统弹出确认对话框
- 确认后才会执行

**2. 备份兜底锁**
- 每日自动备份（凌晨 1 点）
- 操作前自动备份
- 支持一键回滚

**3. 异常终止锁**
- 检测到异常立即终止
- 自动回滚到操作前状态
- 显示异常报告

## 📞 故障排查

### 问题 1：模块未找到

**解决：**
```bash
pip install -r requirements.txt
```

### 问题 2：API 调用失败

**解决：**
- 检查 `.env` 文件中 API Key 配置
- 检查网络连接
- 确认 API 权限是否有效

### 问题 3：选品数据不准确

**解决：**
- 检查淘宝 API 权限
- 确认数据源是否最新
- 手动核实关键数据
