## 🎮 游戏概述
（以下都是AI写的，大概是这么回事）

### 一句话介绍

> **这不是游戏，是你和AI一起写的一本小说，而你，就是主角。**

在传统文字冒险游戏中，编剧预设了3个结局，你选一个。  
在 Worldline Choice 中，**AI根据你的每一个选择实时生成剧情**，没有预设结局——你的每个决定都在创造全新的故事走向。

### 游戏定位

| 项目 | 内容 |
|------|------|
| **游戏名称** | 世界线·抉择 (Worldline: Choice) |
| **游戏类型** | AI驱动互动叙事 / 开放式文字冒险 |
| **核心体验** | 「你就是主角，重写传奇」 |
| **目标用户** | 喜爱剧情、历史、文学、角色扮演的玩家 |
| **单局时长** | 15-45分钟（短篇）/ 1-2小时（长篇） |

### 两大游戏模式

**🎭 叙事模式：自由探索，重写传奇**
- 穿越赛博朋克都市，成为黑客或企业特工
- 探索克苏鲁神话世界，揭开古老秘密
- 置身太空歌剧，在星际间书写你的传奇
- 潜入冷战谍战，在阴影中改变历史
- 进入黑帮江湖，体验权力与背叛
- **甚至创造你自己的世界观**

⚔️ **战术模式：策略推演，运筹帷幄**
- 千人对战场景，设计分步战术
- 每一步都由AI判定成败
- 像军师一样排兵布阵

## ✨ 核心体验

### 真正的自由

**不限于预设选项**——每个抉择点都有4个预设选项 + 1个自由输入通道：

```
┌──────────────────────────────────────────────────────────────┐
│  【抉择点】荒坂大厦潜入                                        │
│                                                              │
│  你站在荒坂集团大厦的阴影中，看着巡逻的安保无人机...            │
│                                                              │
│  ┌────────────────────────────────────────────────────────┐  │
│  │  A. 【正面】伪装成维修人员从正门进入                       │  │
│  ├────────────────────────────────────────────────────────┤  │
│  │  B. 【迂回】寻找通风管道潜入                               │  │
│  ├────────────────────────────────────────────────────────┤  │
│  │  C. 【技术】远程黑入监控系统制造盲区                         │  │
│  ├────────────────────────────────────────────────────────┤  │
│  │  D. 【特殊】若【声望】≥20，可利用内部关系直接获取门禁卡       │  │
│  ├────────────────────────────────────────────────────────┤  │
│  │  💭 我另有打算...（点击输入自定义行动）                   │  │
│  └────────────────────────────────────────────────────────┘  │
└──────────────────────────────────────────────────────────────┘
```

**自由输入示例**：
> 玩家输入："我想贿赂那个警卫，让他放我进数据库"

> AI回应："警卫看了看你的信用芯片，挑了挑眉：'这点钱只够买一杯合成咖啡，朋友。'"

### 每次都不一样

同样的开局，不同的选择 = 完全不同的故事：

**示例：赛博侦探的三种开局**

| 路线 | 关键选择 | 可能结局 |
|------|---------|---------|
| **企业线** | 接受荒坂集团雇佣 | 升职加薪、成为高管、发现公司黑暗秘密… |
| **黑客线** | 加入地下反抗组织 | 被通缉、揭露真相、成为数字幽灵… |
| **独狼线** | 单干做赏金猎人 | 名利双收、被黑白两道追杀、归隐贫民窟… |

我玩了5遍赛博侦探篇，5次结局都不一样。

### 失败也有趣

被追杀、被背叛、战死沙场、众叛亲离…  
每种失败都有对应的剧情，而且往往比成功更精彩。

**示例：黑入企业数据库失败**
> 你掷出了自然1（大失败）  
> 安全系统立即锁定你的位置，警报声震耳欲聋！  
> 荒坂集团的特工已经在路上了，第一局就这么凉了…  

但那种"差一点就成功"的感觉，让人一直想重来！

### 你的选择被记住

NPC会记得你之前做过什么：
- 背叛过的人下次见你态度会不一样
- 帮助过的人会在关键时刻伸出援手
- 你的性格标签会影响NPC对你的看法

这种"真实感"太绝了。

## 🎯 怎么玩

### 安装（2A游戏 - 面向智能体）

Worldline Choice 是一款 **2A 游戏**（Agent-to-Agent，面向智能体）：

**方式一：通过 OpenClaw 安装**
```
在 OpenClaw 中发送：
安装 skill worldline-choice
或
访问 https://github.com/caoyachao/worldline-choice
```

**方式二：通过 CLI 直接对话**
```bash
# 克隆仓库
git clone https://github.com/caoyachao/worldline-choice.git
cd worldline-choice

# 运行游戏
python3 worldline_engine.py --new "赛博侦探" "黑客" "你的名字"
```

**系统要求**：
- Python 3.8+
- Mac / Windows / Linux 均可

### 游戏流程

```
1. 选择世界观
   └─ 赛博朋克 / 太空歌剧 / 克苏鲁 / 谍战 / 黑帮 / 原创...

2. 创建角色
   └─ 选择身份：黑客 / 侦探 / 走私商 / 调查员 / 特工 / 自定义
   └─ 设定名字

3. 开始故事
   └─ AI生成场景描述
   └─ 呈现抉择点（4选项 + 自由输入）

4. 做出选择
   └─ 选预设选项，或输入你的想法
   └─ AI实时生成剧情回应

5. 推动剧情
   └─ 重复步骤3-4，直到结局

6. 收集结局
   └─ 同一世界观，多周目探索不同结局
```

## ⚔️ 抉择系统

### 四类预设选项

每个抉择点围绕**当前场景的核心冲突**设计：

**模式一：阵营/立场选择**
```
A. 【支持甲方】理由/方式
B. 【支持乙方】理由/方式
C. 【保持中立】寻找第三条路/暂时观望
D. 【特殊条件】基于状态的独特选择
```

**模式二：策略/方法选择**
```
A. 【直接方案】正面突破，高效但风险高
B. 【迂回方案】侧面渗透，耗时但稳妥
C. 【非常规方案】利用环境/第三方/诡计
D. 【特殊条件】基于能力的专属方案
```

**模式三：价值观/道德选择**
```
A. 【利己】优先保障自身/己方利益
B. 【利他】牺牲自己帮助他人
C. 【平衡】寻求多方共赢（难度更高）
D. 【特殊条件】基于信仰/原则的立场
```

### D选项：条件触发式特殊路线

D选项不一定是"特殊道具"，更可以是**基于之前选择解锁的独特剧情方向**：

| 触发类型 | 示例场景 | D选项示例 |
|----------|----------|----------|
| **前置关系** | 与某NPC好感度达标 | 【求助】关键时刻，你的黑客盟友远程黑入系统（需与黑客好感≥30） |
| **前置选择** | 之前做过特定选择 | 【兑现承诺】用你之前救下的那人的情报破局 |
| **身份专属** | 特定职业/出身 | 【以医生的身份】利用医学知识找到生物识别系统的漏洞 |

## 👤 角色与状态

### 属性系统

| 属性 | 说明 | 影响范围 | 初始值范围 |
|------|------|----------|------------|
| **武力** | 战斗能力 | 单挑胜率、战斗选项成功率 | 10-20 |
| **智力** | 谋略能力 | 识破陷阱、特殊选项解锁 | 10-20 |
| **魅力** | 人格魅力 | 说服成功率、NPC好感获取 | 10-20 |
| **声望** | 江湖名气 | 特殊NPC态度、势力选择 | 0-10 |

### 关系系统

```
关系等级：

憎恨 [-100, -50)  → 敌对行动，可能追杀玩家
冷淡 [-50, 0)     → 冷淡对待，交易/对话受限
中立 [0, 20)      → 普通对待，基础互动
友善 [20, 50)     → 友好对待，提供额外帮助
信任 [50, 80)     → 深度交流，解锁特殊剧情
生死之交 [80, 100] → 无条件支持，可能牺牲自己
```

### 标签系统

记录玩家的行为模式和关键选择：

| 标签类型 | 示例 | 作用 |
|----------|------|------|
| **性格标签** | 仁慈、残忍、狡诈、正直、勇敢、谨慎 | 影响NPC对玩家的态度 |
| **阵营标签** | 荒坂集团、流浪者部落、公司狗、街头小子 | 解锁对应势力选项 |
| **事件标签** | 数据大盗、空间站幸存者、义体改造人 | 特定剧情触发条件 |

## ⚙️ 技术机制

### d20检定系统

游戏中使用d20（20面骰）检定系统来决定行动成败：

```
检定公式：d20 + 属性修正 ≥ DC（难度等级）

结果等级：
• 大成功（超10+）：超额完成，额外收益
• 成功（超5+）：顺利完成，后续DC-3
• 勉强成功（0~4）：完成但有代价，后续DC-1
• 勉强失败（-1~-4）：失败但可补救
• 失败（-5~-9）：明确失败，承担后果
• 大失败（-10+）：灾难性后果

特殊规则：
• 自然20必成功（视为大成功）
• 自然1必失败（视为大失败）
```

### 战术多步骤判定

在战斗场景中，你可以设计复杂的多步战术：

```
示例：千人对战的连环计

1. 要求对方主将出来单挑
2. 在单挑中诈败引敌追击
3. 反身扔出暗器
4. 伏兵万箭齐发
5. 主力大军冲阵

每一步独立检定，步骤之间有依赖关系：
• 前置步骤成功 → 后续DC降低
• 关键步骤失败 → 可能中断整个计划
```

### 防套利机制（v3.3.2）

为防止玩家通过"低成本步骤蹭加成"套利：

| 机制 | 说明 |
|------|------|
| 勉强成功降级 | 勉强成功仅给后续DC-1（原为DC-3） |
| 指挥链过载 | 步骤超过3个后，每多一步DC+1 |
| 信息泄露 | 大失败会泄露情报，后续步骤DC累加+2 |

## 🚀 快速开始
(我没试过不保证对，建议直接对话安装）

### Python API 使用

```python
from worldline_engine import WorldlineEngine

# 创建游戏引擎
game = WorldlineEngine()

# 开始游戏
game.start_game(
    world_setting="赛博侦探",
    player_role="黑客",
    player_name="V"
)

# 获取AI Prompt用于生成场景
system_prompt = game.get_system_prompt()

# 处理玩家行动
result = game.process_action(
    player_input="我想黑入荒坂集团的数据库",
    ai_response={
        "intention": "入侵",
        "feasible": True,
        "narrative": "你启动了神经接口，防火墙比想象中更复杂...",
        "consequences": {
            "attribute_changes": {"智力": 1},
            "relationship_changes": {"荒坂集团": -10}
        }
    }
)

# 保存游戏
game.save_game("save_001")
```

### 命令行使用

```bash
# 开始新游戏
python3 worldline_engine.py --new "赛博侦探" "黑客" "V"

# 列出所有存档
python3 worldline_engine.py --list

# 加载存档
python3 worldline_engine.py --load save_001
```

## 📝 设计原则

1. **真正的自由** - 不限于预设选项，可以输入任何想法
2. **实时生成** - 没有固定剧本，AI根据选择即时创作
3. **选择有重量** - 每个决定都会影响后续剧情和NPC态度
4. **失败也有趣** - 不是简单的GAME OVER，而是另一条故事线
5. **永不丢失历史** - 所有选择永久记录，支持完整复盘
