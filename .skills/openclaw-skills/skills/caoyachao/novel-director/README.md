# Novel Director - AI互动小说导演板

**版本**: v2.0.0  
**更新日期**: 2025-04-13

AI互动小说创作工具 - 导演板模式（支持长篇连载）。

## 简介

用户作为导演设定场景、人物、目标，AI作为演员分镜演绎，在关键决策点停下等待导演选择。

**v2.0.0 新增功能**：
- ✅ 长篇小说上下文管理（三层架构）
- ✅ 智能检索（关键词+语义）
- ✅ 跨Session续写
- ✅ 角色档案与世界知识库
- ✅ 章节索引与自动摘要
- ✅ 用户指令触发的保存机制

## 核心工作流程

```
【导演指令】→ AI演绎一段 → 【决策点】→ 用户选择 → AI继续演绎 → 【决策点】...
```

## 快速开始

### 1. 新建小说

```
用户：新建小说

```


## 导演控制台

### 基础指令

| 指令 | 作用 |
|------|------|
| **"重来"** | 回到上一个决策点，重新选择 |
| **"改戏"** | 修改刚刚演绎的片段 |
| **"加人"** | 增加新角色进入场景 |
| **"换景"** | 切换到另一个并行场景 |
| **"快进到..."** | 跳过当前场景，直接到某个结果 |
| **"幕后"** | 查看当前场景的人物状态、隐藏信息 |
| **"这场戏过了"** | 结束当前场景，输出成片 |

### 长篇连载专用指令

| 指令 | 作用 |
|------|------|
| **"新建小说"** | 创建新小说项目 |
| **"继续写 [小说名]"** | 加载已有小说，恢复创作状态 |
| **"保存"** | 保存当前进度到文件 |
| **"导出"** | 导出完整小说为 Markdown 文件 |
| **"角色档案"** | 查看/编辑角色信息 |
| **"时间线"** | 查看故事时间线 |
| **"查找 [关键词]"** | 检索相关内容所在的章节 |
| **"上一章"** | 回顾上一章内容 |
| **"完成本章"** | 标记当前章节完成，更新索引 |

## 保存与加载机制

### 保存指令

用户说以下指令时，AI **必须** 保存当前进度：

| 用户指令 | AI 动作 |
|---------|---------|
| **"保存"** | 保存当前草稿到 `drafts/chapter_XX_draft.md` |
| **"保存并退出"** | 保存草稿并提示可以安全关闭 |
| **"完成本章"** | 保存完整章节到 `chapters/chapter_XX.md`，自动生成摘要，更新索引 |

### 加载指令

用户说以下指令时，AI **必须** 加载已有项目：

| 用户指令 | AI 动作 |
|---------|---------|
| **"继续写 [小说名]"** | 1. 读取 `world_knowledge.json` → 了解世界观<br>2. 读取 `chapter_index.json` → 了解章节概要<br>3. 读取最后一章完整内容 → 了解最新剧情<br>4. 加载当前草稿（如有）→ 恢复创作状态 |
| **"加载 [小说名]"** | 同上 |
| **"打开 [小说名]"** | 同上 |

### 自动加载（新 Session）

当用户在新 Session 中说"继续写"、"继续创作"或"恢复小说"时，AI 会自动检测最近的项目并提示选择。

## 导演指令格式

```
【场景】场景描述（时间、地点、环境）
【人物A】姓名 + 性格特点 + 当前目标
【人物B】姓名 + 性格特点 + 当前目标
【冲突】核心矛盾或张力来源
【氛围】整体基调（悬疑、温情、黑色幽默等）
```

## 文件结构

```
novel-director/
├── SKILL.md                    # Skill 定义文件
├── README.md                   # 项目说明
├── templates/                  # 模板文件
│   ├── world_knowledge.json    # 世界知识库模板
│   └── chapter_index.json      # 章节索引模板
└── projects/                   # 小说项目目录
    └── 你的小说/
        ├── world_knowledge.json    # 世界知识库
        ├── chapter_index.json      # 章节索引
        ├── chapters/               # 完整章节
        │   ├── chapter_01.md
        │   └── ...
        └── drafts/                 # 当前草稿
            └── chapter_XX_draft.md
```

## 三层上下文架构

```
┌──────────────────────────────────────────────┐
│  Layer 3: 世界知识库 (world_knowledge.json)    │
│  - 世界观设定、角色档案、关键事件时间线        │
│  - 核心设定，长期不变                          │
├──────────────────────────────────────────────┤
│  Layer 2: 章节索引 (chapter_index.json)        │
│  - 每章的元数据：标题、摘要、关键词            │
│  - 快速检索，无需读取全文                      │
├──────────────────────────────────────────────┤
│  Layer 1: 工作上下文 (Working Context)         │
│  - 当前章节原文（最近1-2章）                   │
│  - 当前场景的创作草稿                          │
└──────────────────────────────────────────────┘
```
