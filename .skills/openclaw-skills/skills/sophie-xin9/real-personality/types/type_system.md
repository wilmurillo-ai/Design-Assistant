# REAL 人格类型体系

> REAL = Reading Everyone's Actual Life
> SBTI 问你"你觉得你是谁"，REAL 看你的数据说你是谁。

---

## 分类维度（4 维度，每维度 2 极）

从社交数据中提取 4 个核心维度，每个维度两极，组合成类型代码：

### 维度 1：内容倾向 — 创（C）vs 吃（E）
- **C = Create**：原创内容 > 转发/消费内容。你在输出。
- **E = Eat**：收藏/点赞/转发 > 原创。你在吃别人的内容。
- **判定依据**：原创帖数 / 总互动数（收藏+点赞+转发）

### 维度 2：社交温度 — 热（H）vs 冷（L）
- **H = Hot**：频繁互动，回复评论，跟人对线。社交温度高。
- **L = Low**：几乎不互动，发了就走，评论区空无一人。
- **判定依据**：评论回复率、互动频率、@别人次数

### 维度 3：兴趣广度 — 杂（W）vs 专（D）
- **W = Wide**：什么都看，什么都收藏，兴趣标签 > 5 个品类。
- **D = Deep**：只关注 1-2 个领域，深度钻研型。
- **判定依据**：内容标签/主题的分散度（熵值）

### 维度 4：表里一致 — 真（R）vs 演（P）
- **R = Real**：发的内容和收藏/点赞的内容高度一致。表里如一。
- **P = Play**：发的是一套，收藏的是另一套。你在演。
- **判定依据**：公开内容主题 vs 私密行为（收藏/点赞）主题的重叠度

## 类型描述风格

每个类型的描述包含：
1. **一句话定义**（标签感）
2. **数据证据**（"你的收藏比发帖多 47 倍"）
3. **扎心解读**（好笑的、自嘲型的）
4. **隐藏优势**（每个类型都有正面解读）
5. **代表行为**（"你一定做过这些事"）

### 描述示例：ELWP「幽灵」👻

```
你是 ELWP —— 幽灵 👻

数据说：
📊 原创内容：23 条 | 收藏：1,847 条 | 点赞：4,329 条
💬 评论回复率：2%（基本不回）
🏷️ 公开内容标签：生活/美食/旅行（5个品类）
🔒 收藏内容标签：副业/理财/脱发治疗/相亲攻略

扎心解读：
你在互联网上几乎不存在。不发帖、不评论、不互动。
但你的收藏夹比任何人都丰富——而且跟你为数不多的公开内容完全不一样。
你发的是岁月静好，收藏的是焦虑人生。
你是互联网的幽灵——无处不在，但没人知道你在。

隐藏优势：
你是最好的观察者。你看得多、记得多、判断力强。
你不发言不代表没想法——你只是不觉得需要告诉别人。
在信息过载的时代，沉默是一种超能力。

你一定做过这些事：
✅ 收藏了一个"30天健身计划"然后再也没打开过
✅ 给别人的旅行照点赞但自己从不发
✅ 半夜刷到一篇深度文章收藏了，第二天忘了
✅ 注册了5个平台，每个都只用来看不用来发
```

## 分类算法逻辑

### 维度 1：C vs E
```
原创占比 = 原创帖数 / (原创帖数 + 转发数 + 收藏数 + 点赞数)
> 0.3 → C（你在创造）
≤ 0.3 → E（你在消费）
```

### 维度 2：H vs L
```
互动率 = (回复评论数 + @提及数 + 互动帖数) / 总帖数
> 0.2 → H（社交热）
≤ 0.2 → L（社交冷）
备选：评论区回复率、日均互动次数
```

### 维度 3：W vs D
```
兴趣熵值 = -Σ(p_i × log(p_i))，其中 p_i 为各主题占比
> 1.5 → W（兴趣广）
≤ 1.5 → D（兴趣深）
备选：TOP 1 主题占比 > 50% → D，否则 W
```

### 维度 4：R vs P
```
表里一致度 = 公开内容标签 ∩ 私密行为标签 / 公开内容标签 ∪ 私密行为标签
> 0.5 → R（真实）
≤ 0.5 → P（在演）
注意：如果只有公开数据（没有收藏/点赞），该维度标注"数据不足"并默认为 R
```

### 匹配度计算
每个维度不是非黑即白——计算距离阈值的偏离程度：
- 偏离越大 → 匹配度越高（"你是一个非常典型的 ELWP"）
- 接近阈值 → 匹配度较低（"你在 C 和 E 之间摇摆"）

最终匹配度 = 4 个维度匹配度的加权平均（0-100%）
