# 分析类文章模板

## 文章结构

### 【开头】- 强烈对比切入
- **场景对比**: `[强烈对比的场景A]` vs `[强烈对比的场景B]`
- **核心问题**: `[提出核心矛盾或问题]`
- **文章预告**: 这篇文章要做三件事：
  1. `[第一件事: 定义概念]`
  2. `[第二件事: 分析框架]`
  3. `[第三件事: 解决方案]`

### 【主体】- 层层深入

#### 1. 概念定义
- **创造核心概念**: `[新概念名称]`
- **概念解释**: `[用简单语言解释这个概念]`
- **为什么需要**: 因为`[现有概念的不足]`

#### 2. 分类讨论
- **建立坐标系**: `[X轴]` vs `[Y轴]`
- **四种类型**: 
  1. `[类型A]: [特征]`
  2. `[类型B]: [特征]`
  3. `[类型C]: [特征]`
  4. `[类型D]: [特征]`

#### 3. 逐层深入
- **表面现象**: `[大多数人看到的现象]`
- **中层原因**: `[结构性原因]`
- **深层本质**: `[人性/制度/文化本质]`

#### 4. 数据案例
- **关键数据**: `[具体数字，如"70%的用户..."]`
- **典型案例**: `[真实或代表性的案例]`
- **对比分析**: `[国内 vs 国外]` / `[过去 vs 现在]`

#### 5. 解决方案
- **短期建议**: `[可立即实施的建议]`
- **中期策略**: `[需要时间推进的策略]`
- **长期愿景**: `[理想状态描述]`

### 【结尾】- 升华收束
- **引用经典**: `[引用古籍/名言，如"《道德经》有云..."]`
- **回到开头**: 回到`[开头意象]`，但有了新的理解
- **开放式收束**: `[引发思考的问题或展望]`

## 语言风格要点

### 句式模板
1. **对比句式**: "你觉得这公平吗？大多数人觉得不公平。但大多数人也觉得没办法。"
2. **排比句式**: "不是...而是..." / "过去是...现在是..."
3. **设问句式**: "在没有路灯的夜晚...现在灯亮了，他还攥着不放。"

### 节奏控制
- **短句为主**: 每句不超过20字
- **段落简短**: 每段3-5句话
- **节奏变化**: 长短句交替，避免单调

### 情感基调
- **理性批判**: 直指问题，但不情绪化
- **悲悯关怀**: 理解个体困境
- **希望导向**: 批判后给出建设性方向

## 使用说明

1. **替换占位符**: 将`[ ]`中的内容替换为具体内容
2. **保持风格**: 确保语言符合上述风格要点
3. **逻辑连贯**: 各部分之间要有逻辑连接
4. **数据支撑**: 尽量使用具体数据和案例

## 示例主题
- 数字化转型中的中年职场困境
- 短视频时代的知识焦虑
- 城市化进程中的乡土情结
- AI时代的人类价值重估
