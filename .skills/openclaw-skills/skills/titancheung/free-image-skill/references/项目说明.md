# 图片解决方案项目

基于OpenClaw的AI生成与免费图库混合解决方案

## 项目概述

为解决文字配图难题，本项目提供了一套完整的图片解决方案：
- **AI图片生成**：重要图片使用OpenAI DALL-E 3生成
- **免费图库整合**：普通图片从免费图库获取
- **自动化处理**：批量调整、优化、索引创建

## 快速开始

### 1. 环境配置

```bash
# 设置OpenAI API密钥
export OPENAI_API_KEY="你的API密钥"

# 或使用配置文件
source /Users/titan/.openclaw/workspace/openai_env.sh
```

### 2. 安装依赖

```bash
# 安装Python依赖
pip3 install openai --break-system-packages

# 安装ImageMagick (macOS)
brew install imagemagick
```

### 3. 使用图片工作流

```bash
# 进入项目目录
cd /Users/titan/.openclaw/workspace/image-solution

# 查看工作流计划
python3 image_workflow.py --plan-only

# 生成商业类图片
python3 image_workflow.py --category business --count 5

# 根据文本生成图片
python3 image_workflow.py --text "你的文本内容" --count 3
```

### 4. 使用图片处理工具

```bash
# 批量调整尺寸
./image_tools.sh resize ./input ./output 800 600

# 批量转换格式
./image_tools.sh convert ./photos ./webp webp

# 批量优化质量
./image_tools.sh optimize ./images ./optimized 80

# 创建图片索引
./image_tools.sh index ./gallery ./IMAGE_INDEX.md
```

## 解决方案架构

### 核心组件

1. **AI图片生成器** (`image_workflow.py`)
   - 基于OpenAI DALL-E 3
   - 支持多种图片类别
   - 可定制提示词和参数

2. **图片处理工具** (`image_tools.sh`)
   - 批量尺寸调整
   - 格式转换
   - 质量优化
   - 索引创建

3. **资源管理器**
   - 免费图库资源列表
   - 图片分类系统
   - 工作流计划

### 工作流程

```
需求分析 → AI图片生成 → 免费图库搜索 → 后期处理 → 索引创建
```

## 免费图片资源

### 推荐图库

| 网站 | 特点 | 许可 |
|------|------|------|
| [Unsplash](https://unsplash.com) | 高质量摄影 | 商业免费 |
| [Pexels](https://www.pexels.com) | 照片和视频 | 商业免费 |
| [Pixabay](https://pixabay.com) | 矢量图和插画 | 商业免费 |
| [Freepik](https://www.freepik.com) | 矢量图和PSD | 需署名 |

### 搜索技巧

1. **使用具体关键词**：避免通用词汇
2. **筛选许可类型**：选择"商业免费"
3. **检查分辨率**：确保图片质量
4. **预览水印**：确认无水印版本

## 成本控制

### AI生成成本估算

| 模型 | 每张成本 | 建议用途 |
|------|----------|----------|
| DALL-E 3 (标准) | $0.04 | 普通配图 |
| DALL-E 3 (HD) | $0.08 | 重要图片 |
| GPT Image | $0.02 | 背景图片 |

### 预算建议

1. **重要内容**：10-20%预算用于AI生成
2. **普通配图**：80-90%使用免费资源
3. **批量处理**：减少重复生成

## 故障排除

### 常见问题

1. **API密钥错误**
   ```bash
   export OPENAI_API_KEY="正确密钥"
   source openai_env.sh
   ```

2. **依赖安装失败**
   ```bash
   pip3 install openai --break-system-packages
   brew install imagemagick
   ```

3. **图片生成失败**
   - 检查网络连接
   - 验证API配额
   - 简化提示词

### 性能优化

1. **批量操作**：减少API调用次数
2. **缓存结果**：重复使用已生成图片
3. **离线处理**：先下载后处理
