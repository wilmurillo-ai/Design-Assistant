### 4. 自动化工作流

创建 `auto_workflow.sh`:

```bash
#!/bin/bash
# 自动化工作流

# 1. 生成AI图片
python3 image_workflow.py --category "$1" --count "$2"

# 2. 处理图片
./image_tools.sh resize ./generated_images ./processed 1200 800

# 3. 创建索引
./image_tools.sh index ./processed ./IMAGE_INDEX.md

./generate_report.sh
```

## 故障排除示例

### 问题：API调用失败

```bash
# 检查API密钥
echo $OPENAI_API_KEY

# 测试连接
python3 -c "from openai import OpenAI; client = OpenAI(); print(client.models.list())"
```

### 问题：图片质量差

解决方案：
1. 使用更详细的提示词
2. 提高生成质量参数
3. 使用DALL-E 3 HD模式
4. 后期使用ImageMagick优化

### 问题：免费图库找不到合适图片

解决方案：
1. 使用更多关键词组合
2. 尝试不同图库
3. 考虑AI生成替代
4. 使用抽象或概念图
