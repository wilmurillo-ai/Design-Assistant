# 创作规范

## 一、静态画面规范

### 允许的微动作（仅以下四项）

| 微动作 | 幅度 | 说明 |
|--------|------|------|
| 发丝轻微飘动 | 3-5% | 如微风轻拂，持续整个镜头 |
| 衣角微颤 | 2-3% | 如微风轻拂 |
| 眼皮缓慢眨动 | 每 2-3 秒一次 | 极缓慢 |
| 嘴唇配合台词轻微张合 | 仅在台词时 | 配合音频 |

### 绝对禁止的动作

**出现任何一个都说明你在做视频而不是动态漫画**：

- **肢体移动**：走动、转身、抬手、握拳、迈步、伸手、触碰
- **头部移动**：点头、摇头、转头、低头、抬头
- **身体移动**：跳跃、下落、旋转、弯腰、站起
- **物体移动**：物品飞出、物体掉落、泪珠滚动、书页翻动
- **大幅度表情变化**：从微笑变成大哭、从平静变成愤怒

### 禁止词汇表

**绝对禁止出现在画面描述中**：

- 英文：walking, running, turning, jumping, falling, reaching, grabbing, throwing, punching, kicking, nodding, shaking head, waving
- 中文：走动、跑动、转身、跳跃、下落、伸手、抓取、投掷、挥拳、踢腿、点头、摇头、挥手
- 任何描述"动作过程"的语句，如"他慢慢抬起手""她转过身来""泪珠滑落"

### 如何表现"动作"

| 方法 | 说明 | 示例 |
|------|------|------|
| 镜头切换 | 不同镜头展示动作的"前"和"后"两个静态瞬间 | 镜头 1 画背影，镜头 2 画正面 |
| 旁白叙述 | 用旁白描述动作过程 | "他猛地站起身来"（画面展示站起后的姿态） |
| 音效暗示 | 用声音暗示动作 | 椅子倒地声（画面展示椅子已倒地） |
| 镜头运动 | 用镜头制造动感 | 快速推近制造冲击感，缓慢拉远制造孤独感 |

## 二、音频设计规范

### 旁白 vs 台词

| 类型 | 时长占比 | 使用场景 | 示例 |
|------|---------|---------|------|
| 旁白 | 67-80% | 场景描述、心理活动、剧情推进、情感渲染、动作叙述 | 旁白："黄昏时分，天台上风很大。" #2.0 秒 [音量:100%] |
| 配音（角色） | 20-33% | 角色对话、独白/自言自语、呼喊/惊呼 | 配音（张三）："等我回来。" #1.0 秒 [音量:100%]（青年音） |

### 旁白使用场景详解

1. **场景描述**：交代时间、地点、环境
2. **心理活动**：揭示角色内心想法
3. **剧情推进**：补充画面无法传达的信息
4. **情感渲染**：增强情绪感染力
5. **动作叙述**：用旁白描述画面中无法展现的动作过程

### 旁白语速标准（强制执行）

**核心公式：旁白时长 = 字数 ÷ 3（正常语速）**

| 语速类型 | 每秒字数 | 适用场景 |
|---------|---------|---------|
| 慢速深情 | 2-3 字 | 情感渲染、悲伤回忆、悬念营造 |
| 正常叙述 | 3-4 字 | 场景描述、剧情推进、心理活动 |
| 快速紧张 | 5-6 字 | 危机时刻、追逐戏、高潮爆发 |

**强制规则**：
1. 默认使用"正常叙述"语速（每秒 4-5 字）
2. 如需慢速或快速，必须在语气标注中说明
3. 单句旁白最长不超过 25 字

### 旁白音色控制规范

**旁白必须标注音色**，格式：`（音色 + 语气）`

**音色维度**：

| 维度 | 选项 |
|------|------|
| 性别 | 男声/女声 |
| 年龄 | 青年/中年/老年 |
| 特点 | 低沉/清亮/沙哑/温柔/磁性/冷淡 |
| 效果 | 混响/旁白腔/说书人 |

**常用旁白音色组合**：

| 场景类型 | 推荐音色 |
|---------|---------|
| 悬疑穿越 | 男声中年低沉混响 |
| 青春校园 | 女声青年清亮 |
| 古风武侠 | 男声中年说书人 |
| 都市情感 | 女声青年温柔 |
| 恐怖惊悚 | 男声低沉冷淡 |
| 历史诗剧 | 男声老年混响 |

### 配音标注格式（完整版）

```
旁白："内容" #时长 [音量:100%]（音色 + 语气）
配音（角色名）："台词内容" #时长 [音量:100%]（声线 + 情绪）
```
