# 人物刻画规范

## 五维人物刻画法

每个角色从五个维度立体刻画：

### 1. 外貌维度

**必填要素**：身材体型、发色发型、面部特征、服装款式、配饰特征

**描写原则**：
- 每个特征有目的（体现性格/身份/背景）
- 避免模糊词（好看、美丽），用具体细节
- 服装与身份匹配（战士→铠甲、刺客→紧身衣）
- 质感：皮肤纹理、材质光泽、光影折射

### 2. 性格维度

通过动作/台词/表情展现性格：
- 动作：急躁→动作快、冷静→动作稳
- 台词：话唠→长句、沉默者→短句

### 3. 动作维度

姿态特征、移动方式、战斗风格、习惯性动作
- 快节奏：战斗/追逐/危机
- 慢节奏：对峙/思考/情感
- 动静结合：慢动作→快动作→定格

### 4. 神态维度

眼神特点、微表情（嘴角/瞳孔）、气场、情绪变化

### 5. 气质维度

整体印象：英雄气质/反派气质/仙气飘渺/霸气侧漏/江湖气息

## 角色卡模板

```
【角色卡】
姓名：[角色名称]
身份：[身份定位]

【外貌】身材 + 发型 + 面部 + 服装 + 配饰
【性格】核心性格 + 行为习惯 + 情绪表达
【动作】站姿 + 移动 + 战斗风格 + 细节
【神态】眼神 + 表情 + 气场
【气质】整体印象 + 他人感受
```

## 一致性锚点

| 类型 | 说明 | 示例 |
|------|------|------|
| 服装锚点 | 固定服装特征 | 红色围巾、金色手镯 |
| 发型锚点 | 固定发型特征 | 披肩长发、马尾边发卡 |
| 动作锚点 | 习惯性动作 | 撩头发、托腮 |
| 表情锚点 | 标志性表情 | 微微皱眉、嘴角上扬 |
