## 四阶段工作流程

### 阶段1：故事输入 + 风格推荐

**📖 必读文档**: [人物刻画规范](人物刻画规范.md) - 角色刻画方法

**步骤**:
1. 用户描述视频内容
2. 确认素材（图片/视频）
3. 选择时长（5s/10s/15s/30s）
4. 选择类型（剧情/战斗/展示等）
5. 确认制作参数（Seedance 2.0）
6. 古诗词配置（如需要）

**输出**: 故事梗概 + 制作参数

---

### 阶段2：AI 智能分析 + 个性化风格推荐

**📖 必读文档**: [导演思维与节奏规范](导演思维与节奏规范.md) - 节奏设计

**执行**:
1. 分析故事类型和情感基调
2. 推荐 2-3 种视觉风格方案
3. 说明每种风格的特点和适用场景

**输出格式**:
```
🎨 深度视觉分析
- 核心色彩：[...]
- 光影氛围：[...]
- 镜头语言：[...]
- 特效风格：[...]

📝 个性化视觉方案
【方案 A：XXX】特点/适用场景
【方案 B：XXX】特点/适用场景
```

---

### 阶段3：AI 深度分析 + 个性化视觉方案

**📖 必读文档**: 
- [Prompt 技术规范](Prompt 技术规范.md) - 提示词结构
- [台词与音效规范](台词与音效规范.md) - 台词设计

**执行**:
1. 根据选择的风格生成详细视觉方案
2. 设计角色一致性锚点
3. 规划镜头数量和节奏
4. 设计台词和音效

**输出**: 完整视觉方案 + 角色卡

---

### 阶段4：确认画面比例 → 生成 prompt → 交付

**📖 必读文档**: [导演思维与节奏规范](导演思维与节奏规范.md) - 镜头组合

**执行**:
1. 确认画面比例（16:9/9:16/1:1/2.35:1）
2. 生成完整分镜脚本
3. 输出最终 prompt

**输出格式**:
```plaintext
（时长 X 秒）【标题】（风格）（色调）

【剧作要求】...
【剧情梗概】...
【人物重点】...
【场景重点】...

[镜头 1-首帧] X 秒 景别 + 运镜 人物描述 动作（场景）
【角色】:"台词" #Xs
（音效）音效 #Xs
（音乐）音乐描述 #Xs
...
[镜头 N-尾帧] X 秒 ... **转场处理**：最后 0.3 秒快速切黑
```

## 核心规范速查

### 镜头数量
- 5 秒：3-4 镜头
- 10 秒：5-7 镜头
- 15 秒：7-10 镜头
- 30 秒：12-18 镜头

### 台词规范
- 每句 2-6 字
- 爽感语速：每秒 3-5 字
- 口语化、个性化

### 尾帧处理
- 最后 0.3 秒快速切黑
- 音乐完整收尾
