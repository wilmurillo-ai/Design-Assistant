# Prompt 技术规范

## 基础规范

**可控三件套**：
1. 主体：关键特征（年龄气质、服装配饰、材质形态）
2. 动作：动作节奏与连续性（慢、连贯、自然）
3. 镜头：景别 + 运镜 + 稳定性

**万能结构**：
```
主体 + 动作 + 场景 + 光影 + 镜头/运镜 + 风格 + 画质 + 约束
```

**动作技巧**：
- ✅ 连续动作链：A→B→C（每步慢一点）
- ✅ 身体部位 + 小幅动作：眼神、手指、肩颈
- ✅ 节奏词：缓慢、轻柔、连贯、自然
- ❌ 避免高速/剧烈：狂奔、激烈打斗
- ❌ 避免大幅形变：身体扭曲、脸部快速变化

**镜头规范**：
- 景别：特写、近景、中景、全景、远景
- 运镜：缓慢推近、轻微拉远、平稳平移、稳定跟拍
- 稳定：固定/云台稳定、无抖动无果冻、丝滑

**角色稳定**：
```
角色卡："黑色齐肩发、浅色针织衫、银色耳环、五官清晰、服装发型一致"
约束：五官清晰、面部稳定不变形、人体结构正常、服装发型不变
```

**避雷**：
- ⚠️ 避免模糊词：好看、非常、极其
- ⚠️ 避免自相矛盾：超高速 + 极度稳定
- ⚠️ 避免角色过载：多人复杂互动
- ⚠️ 严禁高风险内容：色情、暴力、危险动作、版权人物
