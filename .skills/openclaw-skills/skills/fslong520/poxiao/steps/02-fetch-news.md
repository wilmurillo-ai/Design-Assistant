# Step 1: 新闻抓取（通用模块）

## 🎯 目标

并行搜索各新闻源，时效性验证，数据清洗。

---

## 🚨 铁律

**所有搜索关键词必须包含今日日期！**

```
❌ 错误：搜索 "AI 大模型 最新动态"
✅ 正确：搜索 "AI 大模型 最新动态 2026 年 3 月 13 日"
```

---

## 🔧 执行步骤

### 1. 确定搜索领域

根据报告类型确定搜索领域：

| 报告类型 | 必搜领域 | 可选领域 |
|----------|----------|----------|
| 🌅 早报 | 隔夜外盘、宏观政策、行业动态 | 用户自定义 |
| 🌞 午评 | 上午行情、板块异动、突发消息 | 个股公告 |
| 🌙 收盘报告 | 全天行情、龙虎榜、资金流向 | 晚间公告 |

---

### 2. 搜索关键词模板

**日期占位符**：`{今日日期}` 必须替换为实际日期

#### 宏观/政策
```
- "A 股 最新消息 {今日日期}"
- "央行 货币政策 {今日日期}"
- "证监会 监管 {今日日期}"
- "GDP 经济数据 {今日日期}"
```

#### 行业/板块
```
- "AI 科技 半导体 动态 {今日日期}"
- "新能源汽车 销量 {今日日期}"
- "医药生物 创新药 {今日日期}"
- "消费 零售 数据 {今日日期}"
```

#### 外盘/国际
```
- "美股 纳斯达克 道琼斯 {今日日期}"
- "美联储 利率 鲍威尔 {今日日期}"
- "原油 黄金 大宗商品 {今日日期}"
- "美股收盘 {英文日期}"  # 隔夜新闻用英文搜
```

#### 个股/公告
```
- "{股票代码} 公告 {今日日期}"
- "龙虎榜 机构买入 {今日日期}"
- "北向资金 外资 流入 {今日日期}"
```

### 3. 新闻源优先级

| 优先级 | 新闻源 | 特点 |
|--------|--------|------|
| ⭐⭐⭐⭐⭐ | 东方财富、同花顺 | 专业财经，数据准确 |

| ⭐⭐⭐ | 新浪财经、腾讯财经 | 综合新闻，速度快 |
| ⭐⭐⭐ | 华尔街见闻 | 国际视野，宏观分析 |

---

### 4. 时效性验证（24 小时铁律）

| 新闻时间 | 处理方式 |
|---------|---------|
| "X 小时前"（1-24 小时） | ✅ 通过 |
| "今日" / "今天" | ✅ 通过 |
| "昨夜" / "昨晚" | ✅ 通过（早报场景） |
| "昨天" | ⚠️ 仅限昨夜 20:00 后 |
| "1 天前" 超过 24 小时 | ❌ 丢弃，重搜 |
| "2 天前" 及更早 | ❌ 丢弃 |

**验证方法**：
1. 提取新闻发布时间
2. 计算与当前时间差
3. 超过 24 小时 → 标记为旧闻，重新搜索

---

### 5. 数据清洗

**必须提取的字段**：
```json
{
  "title": "新闻标题",
  "summary": "摘要（100 字内）",
  "source": "来源媒体",
  "publish_time": "发布时间",
  "url": "完整文章链接（必须含文章 ID）",
  "category": "分类（宏观/行业/个股/外盘）",
  "sentiment": "情感倾向（利好/利空/中性）"
}
```

**链接有效性检查**：
```
✅ 有效：https://36kr.com/p/1234567890
✅ 有效：https://finance.eastmoney.com/a/20260313123456.html
❌ 无效：https://36kr.com/newsflashes（主页）
❌ 无效：https://finance.eastmoney.com（主页）
```

## ⚠️ 常见问题

### Q: 搜索结果都是旧新闻？

**A**: 检查关键词是否包含日期！添加"今日"、"最新"等限定词。

### Q: 某个领域搜不到新闻？

**A**: 
1. 调整关键词重试（最多 3 次）
2. 尝试英文搜索
3. 标注"该领域今日暂无重要动态"

### Q: 链接是主页？

**A**: 这不是有效新闻链接，必须包含文章 ID，重新搜索！
