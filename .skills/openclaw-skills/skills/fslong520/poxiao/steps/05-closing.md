## 🔧 内容要求

### 1. 全天行情（必填）

**⚠️ 数据来源：必须通过浏览器打开东方财富网页实时抓取！禁止使用 API 或搜索二手信息！**

**必须包含**：
- 上证指数、深证成指、创业板指收盘价及涨跌幅
- 沪深两市全天成交额
- 涨跌家数比
- 涨停/跌停家数
- 市场情绪指标（如连板高度、炸板率）

**获取方式**：
1. 打开 `https://quote.eastmoney.com/center/gridlist.html#hs_a_board`
2. 等待页面加载完成（wait_for 3秒）
3. snapshot 提取指数数据、涨跌家数
4. 详见 `steps/01-realtime-data.md`

---

### 2. 大盘星图（必填）

**必须包含**：
- 全市场上涨/下跌/平盘家数
- 涨停/跌停家数
- 涨跌比判断（赚钱效应）
- 热门板块集中度分析
- 大小盘分化情况

**获取方式**：
- 读取 `steps/02-5-heatmap.md`
- ⚠️ **首选浏览器抓取**：打开 `https://quote.eastmoney.com/stockhotmap/`
- ⚠️ 如遇滑块验证，从行情中心页面推算，标注"热力图遇反爬"
- ❌ **禁止使用东方财富 API**（push2.eastmoney.com 经常返回空或报错）

---

### 3. 板块复盘（必填）

**⚠️ 数据来源：必须通过浏览器打开东方财富板块排行页面抓取！**

**必须包含**：
- 涨幅前 5 板块 + 领涨股 + 涨停家数
- 跌幅前 5 板块 + 领跌股
- 板块轮动分析

**获取方式**：
1. 打开 `https://quote.eastmoney.com/center/boardrank.html` → 板块涨跌排行
2. 等待页面加载完成 → snapshot 提取
3. 详见 `steps/01-realtime-data.md`

---

### 4. 重要消息（必填）

**必须包含**：
- 盘中重大新闻（2-3 条）
- 盘后重要公告（2-3 条）
- 政策动态（如有）

**搜索关键词**：
```
- "今日 重大 新闻 {今日日期}"
- "盘后 公告 利空 利好 {今日日期}"
- "政策 央行 证监会 {今日日期}"
```

---

### 5. 龙虎榜/资金（选填）

**如有数据则包含**：
- 龙虎榜：机构/游资买入卖出 Top5
- 北向资金：全天净流入/流出
- 融资融券：余额变化

**搜索关键词**：
```
- "龙虎榜 机构 游资 {今日日期}"
- "北向资金 外资 全天 {今日日期}"
- "融资融券 余额 {今日日期}"
```

---

### 6. 明日展望（必填）

**内容**：
- 大盘预判：看多/看空/震荡 + 理由
- 技术面分析：支撑位、压力位
- 关注方向：2-3 个板块
- 操作策略：仓位建议、买卖点
