# Step 3: 午评生成（11:30-13:00）

## 🎯 目标

生成午间行情点评，总结上午走势，展望下午行情。

---

## 📊 午评结构

```
🌞 破晓午评 · {日期} · {星期}
━━━━━━━━━━━━━━━━━━━━━━━━━━

📊 【上午行情】
- 三大指数涨跌
- 成交量情况
- 涨跌家数比

🌟 【大盘星图】
- 全市场涨跌分布
- 涨跌停家数
- 热门板块聚集
- 大小盘分化

🔥 【板块异动】
- 涨幅前列板块
- 跌幅前列板块
- 资金流向

📰 【突发消息】
- 午间突发新闻
- 盘中公告

 【下午展望】
- 大盘预判
- 关注方向
- 操作建议

━━━━━━━━━━━━━━━━━━━━━━━━━━
```

---

## 🔧 内容要求

### 1. 上午行情（必填）

**⚠️ 数据来源：必须通过浏览器打开东方财富网页实时抓取！禁止使用 API 或搜索二手信息！**

**必须包含**：
- 上证指数、深证成指、创业板指午盘点位及涨跌幅
- 沪深两市成交额（对比昨日同时段）
- 涨跌家数比
- 涨停/跌停家数

**获取方式**：
1. 打开 `https://quote.eastmoney.com/center/gridlist.html#hs_a_board`
2. 等待页面加载完成（wait_for 3秒）
3. snapshot 提取指数数据、涨跌家数
4. 详见 `steps/01-realtime-data.md`

---

### 2. 大盘星图（必填）

**必须包含**：
- 全市场上涨/下跌/平盘家数
- 涨停/跌停家数
- 涨跌比判断（赚钱效应）
- 热门板块集中度分析

**获取方式**：
- 读取 `steps/02-5-heatmap.md`
- ⚠️ **首选浏览器抓取**：打开 `https://quote.eastmoney.com/stockhotmap/`
- ⚠️ 如遇滑块验证，从行情中心页面推算，标注"热力图遇反爬"
- ❌ **禁止使用东方财富 API**（push2.eastmoney.com 经常返回空或报错）

---

### 3. 板块异动（必填）

**⚠️ 数据来源：必须通过浏览器打开东方财富板块排行页面抓取！**

**必须包含**：
- 涨幅前 3 板块 + 领涨股
- 跌幅前 3 板块 + 领跌股
- 北向资金流向（如有数据）

**获取方式**：
1. 打开 `https://quote.eastmoney.com/center/boardrank.html` → 板块涨跌排行
2. 打开 `https://data.eastmoney.com/zjlx/dpzjlx.html` → 资金流向
3. 等待页面加载完成 → snapshot 提取
4. 详见 `steps/01-realtime-data.md`

---

### 4. 突发消息（选填）

**搜索关键词**：
```
- "突发 利好 利空 {今日日期}"
- "盘中 公告 停牌 {今日日期}"
- "龙虎榜 机构 游资 {今日日期}"
```

---

### 5. 下午展望（必填）

**内容**：
- 大盘预判：延续/反转/震荡
- 关注方向：下午可能活跃的板块
- 操作建议：持仓/减仓/观望

---
