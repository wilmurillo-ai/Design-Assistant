# Step 1.5: 大盘星图（东方财富热力图）

## 🎯 目标

获取东方财富大盘星图（个股热力图），直观展示全市场涨跌分布。

---

## 🔧 获取方式

### 0. 先调用 edgeuse 技能（必须！）

**⚠️ 所有网页操作必须先调用 edgeuse 技能建立 CDP 连接！禁止直接调用 browser_use 的 start/open！**

```
调用 edgeuse 技能 → 等待 CDP 连接建立完成 → 再进行后续操作
```

### 1. 访问页面（首选方式）

**URL**：`https://quote.eastmoney.com/stockhotmap/`

**使用 edgeuse 技能或 browser_use 访问**：
```
browser_use action="open" url="https://quote.eastmoney.com/stockhotmap/"
browser_use action="wait_for" wait_time=3  // 等待页面加载
browser_use action="snapshot" → 获取热力图数据
```

### 2. 数据解析

从页面中提取以下信息：

| 数据项 | 说明 |
|--------|------|
| 上涨家数 | 全市场上涨个股数量 |
| 下跌家数 | 全市场下跌个股数量 |
| 平盘家数 | 价格不变的个股数量 |
| 涨停家数 | 涨停板个股数量 |
| 跌停家数 | 跌停板个股数量 |
| 热门板块 | 热力图中颜色最深/最亮的板块 |
| 热门个股 | 热力图中涨幅最大的个股 |

### 3. 备用方案（如遇滑块验证）

**大盘星图页面可能有滑块验证，如遇到：**
1. 不要硬闯，不要尝试自动化通过验证
2. 从行情中心页面（`gridlist.html`）的涨跌家数推算
3. 在报告中标注："热力图页面遇反爬验证，以下基于行情中心数据推算"

**行情中心推算方法**：
- 从 `gridlist.html` snapshot 中提取涨跌家数
- 从东方财富首页快照中提取热门板块标签
- 如无法获取准确数据，宁缺毋滥，不要编造

## 📊 星图分析要点

在报告中解读星图时，重点关注：

1. **涨跌比**：上涨/下跌家数比值，判断市场赚钱效应
   - 涨:跌 > 3:1 → 极强
   - 涨:跌 > 2:1 → 较强
   - 涨:跌 ≈ 1:1 → 分化
   - 跌:涨 > 2:1 → 偏弱
   - 跌:涨 > 3:1 → 极弱

2. **涨跌停比**：涨停/跌停家数比，判断游资活跃度
   - 涨停 > 50 家 → 短线活跃
   - 涨停 < 10 家 → 情绪低迷

3. **热门板块集中度**：热力图中是否有明显的热点聚集
   - 热点集中 → 资金有主线
   - 热点分散 → 轮动快，无主线

4. **大小盘分化**：对比沪深两市涨跌分布
   - 深强沪弱 → 中小盘活跃
   - 沪强深弱 → 权重股护盘

---

## ⚠️ 注意事项

1. ⚠️ **数据必须是盘中/盘后的实时数据，禁止用旧数据或估计值**
2. 涨跌停统计注意科创板/创业板是 20%，北交所是 30%
3. ⚠️ **禁止使用东方财富 API**（push2.eastmoney.com 经常返回空或 422 错误）
4. 如遇滑块验证，跳过并标注，宁缺毋滥
5. 星图数据要和行情中心数据交叉验证，确保一致性
6. 热门板块分析要具体，不要写"板块轮动快"这种空话
