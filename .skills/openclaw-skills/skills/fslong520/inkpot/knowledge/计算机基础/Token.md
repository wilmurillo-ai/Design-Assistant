# Token（令牌/标记）

## 一句话解释
Token 是在不同技术场景中有不同含义的核心概念，通常指"最小单元"或"访问凭证"。

## 详细解释

Token 在多个领域有不同含义：

### 1. 编程/编译器领域
词法分析单元：源代码被编译器拆分的最小单位。
例如 `int x = 5;` 被拆分为：`int`、`x`、`=`、`5`、`;` 五个 token。

### 2. AI/大模型领域
文本切分单位：模型处理文本时的最小单元。
- 一个 token 可能是一个字、一个词，或词的一部分
- 英文约 4 字符 = 1 token
- 中文约 1-2 字 = 1 token

### 3. 身份认证领域
访问令牌：用于验证身份和授权访问的凭证。
- JWT (JSON Web Token)
- OAuth Token
- 用户登录后获得 token，后续请求携带它来证明身份

### 4. 区块链领域
代币：在区块链上发行的数字资产。
- ERC-20 Token
- NFT（非同质化代币）

## 关键要点
- Token 本质是"最小单元"或"凭证"的概念
- 不同领域含义不同，需要结合上下文理解
- AI 领域的 token 直接影响模型的输入输出成本
