# 树状数组

## 一句话解释
一种支持单点修改和区间查询的数据结构，时间复杂度均为 O(log n)，代码简洁、常数小。

## 详细解释

树状数组(Binary Indexed Tree，简称BIT)利用二进制性质，将数组组织成树形结构。每个节点存储一段区间的和，节点管辖的区间长度由最低位1决定(lowbit)。

核心是lowbit运算：`lowbit(x) = x & (-x)`，取出x的最低位1及其后面的0。例如 `lowbit(6) = 2`，因为6的二进制是`110`。

### 操作原理

**向上更新**：修改时，从当前节点向父节点跳，路径是 `x += lowbit(x)`
**向下查询**：查询时，从当前节点向前跳，路径是 `x -= lowbit(x)`

## 关键要点

- 单点修改、前缀查询都是O(log n)
- 区间查询通过两次前缀查询实现：`sum(l,r) = query(r) - query(l-1)`
- 代码量小，常数比线段树小
- 无法直接处理区间修改（需差分变种）

## 代码示例

```cpp
int tree[N], n;

int lowbit(int x) {
    return x & (-x);
}

void update(int x, int v) {
    for (; x <= n; x += lowbit(x))
        tree[x] += v;
}

int query(int x) {
    int sum = 0;
    for (; x > 0; x -= lowbit(x))
        sum += tree[x];
    return sum;
}

int rangeQuery(int l, int r) {
    return query(r) - query(l - 1);
}
```
