# 前缀和

## 一句话解释

前缀和是一种**预处理技巧**，用 O(n) 时间预处理后，可以**O(1) 查询任意区间和**。

## 详细解释

### 定义

给定数组 `a[1..n]`，定义前缀和数组 `s[1..n]`：

```
s[i] = a[1] + a[2] + ... + a[i]
```

特别地，`s[0] = 0`。

### 核心性质

**区间和公式**：

```
区间 [l, r] 的和 = s[r] - s[l-1]
```

**推导**：

```
s[r]     = a[1] + a[2] + ... + a[l-1] + a[l] + ... + a[r]
s[l-1]   = a[1] + a[2] + ... + a[l-1]
s[r]-s[l-1] =                    a[l] + ... + a[r]  ✓
```

### 复杂度分析

| 操作 | 时间复杂度 | 说明 |
|------|------------|------|
| 预处理 | O(n) | 只需一次 |
| 单次查询 | O(1) | 任意区间 |
| 空间 | O(n) | 额外数组 |

## 关键要点

- **适用场景**：多次询问区间和（预处理一次，查询多次）
- **不适用场景**：需要动态修改数组元素 → 用树状数组/线段树
- **配合技巧**：前缀和 + 二分、前缀和 + 双指针
- **边界处理**：`s[0] = 0`，避免越界

## 代码示例

```cpp
const int MAXN = 100005;
int n;
int a[MAXN], s[MAXN];

// 预处理前缀和
void build() {
    s[0] = 0;
    for (int i = 1; i <= n; i++) {
        s[i] = s[i - 1] + a[i];
    }
}

// O(1) 查询区间和
int query(int l, int r) {
    return s[r] - s[l - 1];
}
```

## 示例演示

原数组：`[1, 3, 5, 7, 9]`

前缀和数组：`[1, 4, 9, 16, 25]`

查询 `[2, 4]` 的和：
```
s[4] - s[1] = 16 - 1 = 15
验证：3 + 5 + 7 = 15 ✓
```

## 进阶应用

### 1. 二维前缀和

矩阵区域求和，O(1) 查询任意子矩阵：

```cpp
s[i][j] = a[i][j] + s[i-1][j] + s[i][j-1] - s[i-1][j-1];

// 查询 (x1,y1) 到 (x2,y2) 的子矩阵和
int query(int x1, int y1, int x2, int y2) {
    return s[x2][y2] - s[x1-1][y2] - s[x2][y1-1] + s[x1-1][y1-1];
}
```

### 2. 前缀和 + 二分

求满足条件的最小区间（如和 ≥ k 的最短区间）

### 3. 前缀和 + 哈希

求和为 k 的子数组个数（LeetCode 560）
