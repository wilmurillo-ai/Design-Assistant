# 智能推荐系统

图片姬的核心能力：AI基于内容特征实时推理最佳视觉方案，不依赖预设映射表。

---

## AI自主分析维度

| 维度 | 分析内容 | 输出结果 |
|------|----------|----------|
| **主题深度理解** | 本质特征、情感基调、目标受众、文化背景 | 主题类型判断 |
| **视觉匹配推理** | 内容属性分析 | 最佳艺术风格 |
| **构图自适应** | 信息密度、重点分布 | 有效布局方式 |
| **色彩情绪联动** | 氛围需求分析 | 配色方案 |
| **风格创新融合** | 跨风格可能性 | 独特视觉效果 |

---

## AI自主推理能力

### 风格选择

分析主题类型，自主判断最适合的视觉风格：

| 主题类型 | 推荐风格方向 |
|---------|-------------|
| 技术/编程 | 极简线条、代码风格、暗色系 |
| 教育/教学 | 清晰信息图、扁平化、高对比度 |
| 艺术/文化 | 水墨风格、古典元素、暖色调 |
| 商业/营销 | 专业图标、渐变配色、现代感 |
| 科普/知识 | Bento Box布局、模块化、知识卡片 |

### 配色推荐

根据情感基调智能生成配色：

| 情感基调 | 配色倾向 |
|---------|---------|
| 冷静/专业 | 蓝色系、灰白搭配 |
| 温暖/亲和 | 橙黄色系、暖白底色 |
| 活力/动感 | 纅橙系、高饱和度 |
| 典雅/传统 | 金棕系、古典配色 |
| 科技/未来 | 蓝紫渐变、霓虹感 |

### 构图布局

基于内容密度动态选择：

| 内容密度 | 构图方式 |
|---------|---------|
| 信息密集 | Bento Box模块化、网格布局 |
| 重点单一 | 中心构图、大面积留白 |
| 流程叙事 | 线性布局、箭头引导 |
| 对比展示 | 左右分栏、对称构图 |

---

## 重要原则

1. **不预设固定映射**：保持灵活性，根据具体内容实时推理
2. **支持创新组合**：允许跨风格混搭创造独特效果
3. **基于世界知识**：AI利用通用知识做审美判断
4. **方案差异化**：多个推荐方案应有明显区别
