## Bento Box模块化布局模板

适用于：知识可视化、算法教学、概念解释

```
[主题] 知识可视化图解

**布局**：Bento Box模块化网格布局
- 主模块（大）：核心概念可视化，占画面40%
- 辅助模块（中）：关键要素分解，各占15-20%
- 连接模块（小）：关系箭头、流程线，占10%
- 文字模块：标题、标签、简要说明嵌入各模块

**风格**：扁平化极简/现代信息图表风格
**配色**：[根据主题选择，如科技用蓝白、教育用暖色]

**内容要素**：
- 标题：[标题文字]
- 核心概念：[概念名称]
- 分解要素：[要素1]、[要素2]、[要素3]
- 关系说明：[如何连接或关联]

**文字处理**：
- 标题字体：粗体，居上
- 标签字体：中等大小，嵌入模块
- 说明字体：较小，居模块下方
```

## 技术概念图模板

适用于：编程概念、数据结构、算法流程

```
[技术概念] 可视化图解

**风格**：极简技术风格
- 线条：干净利落，有几何感
- 元素：抽象图形，代码风格
- 背景：深色或浅色纯色

**构图**：
- 核心元素：[概念的核心表达]
- 关联元素：[相关概念展示]
- 流程方向：[如有流程，箭头指示]

**配色**：
- 背景：深蓝/深灰/纯白
- 元素：代码高亮色系（如蓝/绿/橙）
- 文字：与背景高对比度

**文字内容**：
- 标题：[概念名称]
- 标签：[关键术语]
- 代码片段：[如有，简短示例]

**字体风格**：
- 标题：粗体，代码风格字体
- 标签：中等，等宽字体
- 代码：代码字体，保持格式
```
