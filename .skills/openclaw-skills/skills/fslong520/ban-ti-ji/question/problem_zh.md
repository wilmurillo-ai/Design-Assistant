# Personnel Change

<div class="water">

高桥工作的公司有 $N$ 名员工，每名员工被分配一个员工编号 $1, 2, \ldots, N$。公司有 $M$ 个部门，称为部门 $1, 2, \ldots, M$。

员工 $i$ 本学期属于部门 $A_i$，下学期将属于部门 $B_i$。

对于每个部门 $1, 2, \ldots, M$，求下学期成员数减去本学期成员数的差值。

## 输入格式

第一行两个整数 $N$ 和 $M$。

接下来 $N$ 行，每行两个整数 $A_i$ 和 $B_i$，表示员工 $i$ 本学期和下学期所在的部门。

## 输出格式

输出 $M$ 行，第 $j$ 行输出部门 $j$ 的答案。

## 样例

### 样例输入 1
```
5 4
1 2
2 1
3 1
2 2
2 4
```

### 样例输出 1
```
1
-1
-1
1
```

### 样例解释 1

- 部门 1：本学期 1 人（员工 1），下学期 2 人（员工 2, 3），差值 $+1$
- 部门 2：本学期 3 人（员工 2, 4, 5），下学期 2 人（员工 1, 4），差值 $-1$
- 部门 3：本学期 1 人（员工 3），下学期 0 人，差值 $-1$
- 部门 4：本学期 0 人，下学期 1 人（员工 5），差值 $+1$

### 样例输入 2
```
10 5
3 2
3 4
1 2
2 2
4 4
3 1
3 4
4 2
3 3
3 2
```

### 样例输出 2
```
0
4
-5
1
0
```

## 数据范围

- $1 \le N \le 100$
- $1 \le M \le 100$
- $1 \le A_i \le M$
- $1 \le B_i \le M$
