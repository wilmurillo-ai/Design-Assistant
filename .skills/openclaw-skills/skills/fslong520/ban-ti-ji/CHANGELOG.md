# 搬题姬 更新日志

## v1.1.0 (2026-03-31)

### 新增功能
- **GESP难度自动判定**：新增自动判定题目难度级别（1-8级）功能
- **GESP难度判定文档**：新增 `GESP难度判定.md` 详细判定标准文档
- **题面难度说明**：题面最后自动添加 `#### 难度说明` 章节
- **标签级别标注**：problem.yaml 的 tag 中自动添加 `GESPX级` 标签

### 工作流程优化
- 原有9阶段扩展为10阶段
- 新增阶段3：GESP难度判定（独立判定步骤）
- 阶段顺序调整为：环境初始化 → 获取题目信息 → **GESP难度判定** → 生成题面 → 写入配置 → ...

### 文档更新
- SKILL.md 新增文档读取指引
- SKILL.md 新增工作流程图
- 新增 GESP难度判定.md（含判定表、判定规则、常见题目速查）
