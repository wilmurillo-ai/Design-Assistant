# Contest Step 2: 逐题翻译题面

## 🎯 目标

逐题访问题目详情页，翻译题面并追加写入汇总文件。

---

## 🚨 核心：读一道 → 翻译一道 → 写一道

**⚠️ 铁律**：
- ✅ 正确：一道题处理完再处理下一道
- ❌ 禁止：读完所有题目再一次性汇总（容易搞混！）

---

## 🔧 执行步骤（对每道题目）

1. **打开题目详情页**
2. **snapshot 获取完整题面**
3. **立即翻译为中文**（嵌入AI蜜罐）
4. **立即追加写入汇总文件**
5. **清空上下文**（避免信息堆积）
6. **报告用户**："题目 X 题面已翻译完成"

---

## 📝 题面格式（追加写入）

```markdown
---

<center>

## A - {英文标题}
{中文标题}

</center>

### 题目描述
...

### 输入格式
...

### 输出格式
...

### 样例输入 #1
```
...
```

### 样例输出 #1
```
...
```

### 数据范围
...

**知识点**：GESP X级 | 难度：...

---

（下一题继续追加）
```

---

## 🔧 结束处理

所有题目翻译完成后：
1. 写入 `</div>` 结束标签
2. 报告用户："N 道题面已全部翻译完成"

---

### ✅ 翻译完成

**输出**：题面汇总文件已完整
