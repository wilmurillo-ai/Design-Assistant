---

## 🚨🚨🚨 最高铁律：必须从文件读取题面！

**⚠️⚠️⚠️ 每搬运一道题前，必须用 grep_search + read_file 从题面汇总文件读取该题信息！**

**绝对禁止**：
- ❌ 从对话上下文记忆中提取题面（上下文可能错乱）
- ❌ 从之前翻译的内容回忆（可能遗漏或混淆）
- ❌ 假装记得题目内容直接写（必须实际读取）

**必须执行**：
- ✅ 用 `grep_search` 在题面文件中定位当前题目
- ✅ 用 `read_file` 读取该题目完整内容
- ✅ 确认读取成功后才开始搬运

**原因**：
1. 防止上下文爆炸导致信息错乱
2. 确保题面准确无误
3. 多题搬运时容易搞混，必须物理读取确认

## 🔧 执行步骤（对每道题目，独立循环）

### 【Step 3-1】读取题面信息（必须执行！）

**操作流程**：

```python
# 1. 用 grep_search 定位题目位置
grep_search(pattern="## A -", path="abc453.md")  # 找到 A 题开始位置

# 2. 用 read_file 读取该题目完整内容
read_file(file_path="abc453.md", start_line=找到的行号, end_line=下一题开始行号或文件末尾)

# 3. 从读取内容中提取：
#    - 题号（A/B/C...）
#    - 中文标题
#    - 题目描述
#    - 输入输出格式
#    - 样例
#    - 数据范围
#    - 知识点标签
```

**铁律**：grep_search + read_file 必须执行，不能跳过！
