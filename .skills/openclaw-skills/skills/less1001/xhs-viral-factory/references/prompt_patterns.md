# XHS Viral Factory Patterns

This library defines the writing logic for the AI.

## 1. Therapeutic Style
- Focus: Emotional regulation and self-worth.
- Keywords: Heal, growth, authentic, relief.
- Goal: Create resonance with the reader.

## 2. Cheat Sheet Style
- Focus: Knowledge density and practicality.
- Keywords: Tutorial, collection, checklist, essential.
- Goal: Encourage "Save for later" (Collections).

## 3. Cognitive Upgrade Style
- Focus: Breaking common misconceptions.
- Keywords: Truth, logic, hidden, counter-intuitive.
- Goal: Spark discussion and comments.

## 4. Visual Heavy Style
- Focus: High-quality imagery and minimal text.
- Keywords: Aesthetics, vibe, inspiration.
- Goal: Catch eyes with visuals.
