# yfinance AgentSkill

Fetch price, technical, and fundamental data using the Python yfinance package. Specially tuned for Indonesian (IDX) stocks with smart ticker handling and concise output.

---

## ⚙️ Features
- Fetches real-time price and snapshot summary for 1 or more tickers (batch/single).
- **IDX auto-suffix:** On input tickers for Indonesian stocks, if ".JK" is missing, append it automatically.
- Supports both international and IDX-listed tickers.
- Provides concise fundamental AND technical snapshots (see SOUL.md for conventions).
- Returns last 3 months of historical price data when requested (defaults to daily, adjustable granularity).
- Error/warning handling for invalid tickers or missing data, in line with teaching/analyst persona.

---

## 🛠️ Usage

- `yfinance snapshot <ticker or [tickers]>` — Get concise price, technical, and fundamental summary(s)
- `yfinance history <ticker or [tickers]> [period] [interval]` — Last 3 months historical price data (default: period=3mo, interval=1d)

Examples:
- `yfinance snapshot ADRO` (IDX, returns ADRO.JK)
- `yfinance snapshot ADRO,BBCA,TLKM` (batch, all suffix-handled)
- `yfinance history AGII` (IDX, 3-month chart)
- `yfinance snapshot AAPL` (non-IDX, no suffix added)

---

## 📑 Input rules
- Tickers: Accept comma/space list or single ticker (e.g. "ADRO, BBCA, TLKM" or ["ADRO","BBCA"])
- If ticker is all caps, at most 5 letters, and not suffixed → treat as IDX/append .JK.
