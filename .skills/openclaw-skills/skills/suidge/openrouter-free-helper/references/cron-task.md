# OpenRouter Free Model Monitor Cron Task

目标：检查 OpenRouter 当前可用的免费模型状态，并输出简短摘要。

执行要求：
1. 在工作区内运行本地检查脚本 `/Users/neoshi/.openclaw/workspace/skills/openrouter-free-helper/scripts/check-models.py --no-notify`
2. 不要依赖浏览器，不要访问无关网站
3. **禁止输出任何中间态消息、解释、过程说明、工具结果转述或“已推送/已通知”类状态描述**
4. 如果脚本成功运行，只基于脚本最终结果给出 **一条** 纯文本摘要
5. 摘要只允许说明以下之一：
   - 有新的免费模型
   - 有免费模型到期/移除
   - 没有变化
   - 脚本执行失败，并简述原因
6. **不要自行发送消息**，不要说“详情已推送至飞书通知”“通知已通过飞书推送”等，因为 cron delivery 会自动发送最终摘要
7. 直接把最终摘要作为唯一输出，交给 cron delivery 发送

注意：
- 仅执行本地脚本与必要的本地读取
- 保持任务自包含，不携带无关对话上下文
- 输出必须简短、明确、可直接发给主人
- 输出必须是一次性最终结果，不得先发脚本状态，再发汇总
