# MEDDIC 校准规则

## 用途

本文件用于在六步法主赢率基础上，判断当前赢率是否需要保持、折扣或阶段回退。

## 核心原则

- 六步法决定主赢率
- MEDDIC 不替代阶段赢率，而是做校验与校准
- 先判断阶段是否真实，再判断 MEDDIC 缺口

## 三种结果

### 1. 保持主赢率
适用：
- 当前阶段 gate 真实成立
- MEDDIC 关键项较完整
- 没有明显致命缺口

### 2. 风险折扣
适用：
- 当前阶段基本成立
- 但 MEDDIC 仍有明显缺口
- 例如：EB 未直接触达、Decision Process 模糊、Champion 偏弱

### 3. 阶段回退
适用：
- 当前阶段判断本身站不住
- MEDDIC 缺口说明该机会其实还没跨过当前 gate

## 典型致命缺口

以下缺口长期存在时，应优先考虑回退阶段，而不是只做小幅折扣：
- EB 不可触达
- 没有真实 Champion
- 痛点不成立或不紧急
- 决策流程不清晰
- 预算逻辑不存在
