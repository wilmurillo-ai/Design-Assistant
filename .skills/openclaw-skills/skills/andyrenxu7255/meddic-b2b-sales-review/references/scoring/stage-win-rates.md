# 阶段赢率主表

## 用途

本文件定义销售六步法的主赢率骨架。复盘时，先判断机会当前真实所处阶段，再用本表作为基础赢率。

## 主赢率

- **Discovery 通过** → **10%**
- **Scope 通过** → **50%**
- **Go/No-Go 通过** → **90%**
- **Validation Solution** → **95%**
- **Business Case** → **97%**
- **Negotiate & Close** → **99%**

## 使用原则

- 这里的赢率建立在“该阶段 gate 真正通过”的前提上
- 不是销售主观觉得到了这个阶段，就直接套这个赢率
- 六步法阶段赢率是主骨架，后续再用 MEDDIC 做校验和校准

## 输出要求

输出赢率时，要写明：
- 当前阶段
- 对应主赢率
- 该阶段是否真实成立
- 是否需要 MEDDIC 风险折扣或阶段回退
