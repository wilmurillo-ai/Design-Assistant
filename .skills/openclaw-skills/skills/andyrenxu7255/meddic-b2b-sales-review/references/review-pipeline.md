# Pipeline Review / 周会复盘 / Forecast 校准

## 目标

用于销售周会、月会、季度 review 中，对一组机会做**批量复盘和预测校准**。

## 建议读取顺序

1. 先读 `methods/go-no-go.md` 做预测/去留判断
2. 需要校准阶段真实性时，读 `methods/six-step.md`
3. 需要找关键短板时，读 `methods/meddic.md`
4. 输出表格和结论前，读 `templates/output-format.md`

## 管理者关注点

### 1. 阶段是否真实
- 销售说在 Validation，有没有客户验证行为？
- 销售说能签，有没有 EB / 采购 / 时间表证据？

### 2. Forecast 是否虚高
- Commit 必须有硬证据
- Best Case 不能包装成 Commit
- 还在探索的机会只能放 Pipeline

### 3. 资源该投哪里
- 哪些机会值得老板介入
- 哪些机会需要跨部门支持
- 哪些机会该降预测甚至放弃

## 输出重点

输出应该优先包括：
- 上调预测的机会
- 维持预测的机会
- 下调预测的机会
- 需要管理者介入的机会
