# 跟进优先级蒸馏规则

## 用途

当复盘后需要判断接下来先做什么、怎么排优先级时读取本文件。

## 优先级原则

1. 先做过 gate 的关键动作
2. 先做能提升机会质量的动作
3. 先做能推动下一个客户承诺的动作
4. 不要先做看起来忙、但不改变结果的动作

## 典型高优先级动作

- 确认 EB 身份与会面可能性
- 发展 Champion 并拿到真实推动动作
- 明确决策流程与时间表
- 量化业务价值与不做代价
- 为关键会议准备对客户有意义的材料

## 低优先级动作示例

- 在痛点不清时就反复优化方案 PPT
- 在关键人不清时就深入谈价格
- 在决策流程不清时就假设快签单
