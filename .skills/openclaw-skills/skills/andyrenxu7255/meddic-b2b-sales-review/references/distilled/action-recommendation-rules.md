# 动作推荐蒸馏规则

## 用途

当复盘目标不是停在判断，而是要生成下一步动作建议时读取本文件。

## 推荐逻辑

### 如果缺 Economic Buyer
推荐动作：
- 明确 EB 身份与角色
- 借 Champion 或管理层创造 EB 会面
- 在没有 EB 之前，不把机会虚高判断为后期

### 如果缺 Champion
推荐动作：
- 识别更有影响力的内部支持者
- 让潜在支持者做出真实推动动作
- 没有真实 Champion 时，减少方案堆砌

### 如果缺 Decision Process
推荐动作：
- 问清谁决策、谁审批、谁否决、什么时候决策
- 把时间表反推到下一个关键动作

### 如果痛点弱
推荐动作：
- 回到业务问题本身，重做痛点澄清
- 量化不做的代价
- 不急着推进商务

## 原则

- 推荐的不是“泛动作”，而是当前阶段最关键的过 gate 动作
- 动作必须能直接改变机会质量，而不是只是制造热闹
