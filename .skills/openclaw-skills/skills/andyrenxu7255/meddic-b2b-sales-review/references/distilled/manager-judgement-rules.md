# 管理判断蒸馏规则

## 用途

当任务需要站在销售管理者角度做判断，而不是单纯复述销售说法时读取本文件。

## 核心判断原则

- 先判断机会质量，再判断销售动作
- 先判断 Go/No-Go，再谈怎么赢
- 没有关键证据时，不做乐观预测
- 管理复盘不是监督记录，而是帮助团队做对方向判断

## 重点管理判断

### 这是不是一个值得继续投资源的机会？
看：
- EB 是否可触达
- Champion 是否真实
- 痛点是否紧急
- 决策流程是否清晰
- 客户有没有明确下一步承诺

### 当前最该补什么？
不是平均补，而是找最大短板：
- 关键人缺失
- 业务价值缺失
- 流程缺失
- 策略错位

### 管理者什么时候该介入？
- 销售迟迟进不去 EB
- 需要高层会面
- 客户进入采购 / 法务关键期
- forecast 与事实明显不一致

## 输出要求

必须输出：
- 方向判断
- 资源建议
- 管理动作
