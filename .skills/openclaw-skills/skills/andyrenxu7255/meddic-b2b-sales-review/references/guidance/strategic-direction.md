# 策略方向判断

## 用途

当复盘目标是判断“接下来应继续推进、转向、降温还是止损”时读取本文件。

## 四类方向

### 1. 继续推进
适用：
- 方向正确，关键短板可补
- 客户承诺明确
- 机会质量中高

### 2. 调整策略
适用：
- 当前切入点不对
- 销售动作和阶段错位
- 需要换关键人、换话题、换推进节奏

### 3. 降温观察
适用：
- 客户意愿不稳定
- 关键证据不足
- 不值得重资源持续投入

### 4. 暂停 / 止损
适用：
- 长期没有 EB / Champion / 预算 / 流程
- 机会本身不成立
- 投入产出明显失衡

## 输出要求

必须明确写出：
- 当前建议方向是哪一种
- 为什么不是另外三种
