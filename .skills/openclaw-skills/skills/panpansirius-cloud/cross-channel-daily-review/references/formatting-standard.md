# Formatting Standard

## Hard rules
- Use clear titled sections.
- Keep one major numbered point per paragraph block.
- Within each major block, use smaller sub-sections when needed.
- Do not send a dense wall of text.
- Do not wrap the whole response in a code block.
- Put the final action checklist in a standalone `text` code block when a checklist is required.

## Preferred management summary shape
- Title
- 1️⃣ 2️⃣ 3️⃣ major blocks
- Each block can include: `📊 现象`, `🔍 深度归因与思考`, `🎯 结合实际情况`, `✅ 专属策略`
- Final strategy sheet in one `text` block
