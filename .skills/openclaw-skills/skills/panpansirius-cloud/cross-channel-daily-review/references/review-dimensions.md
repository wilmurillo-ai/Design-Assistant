# Review Dimensions

Every synthesized report should cover these 5 dimensions:

1. **❌ 做错了什么**
   - concrete mistakes
   - premature claims
   - wrong assumptions

2. **⚠️ 没做好的地方**
   - weak execution
   - unclear communication
   - incomplete follow-through

3. **✅ 做得好的地方**
   - strong diagnosis
   - correct recovery
   - useful structure or process

4. **📝 遗漏事项**
   - missed asks
   - missing verifications
   - skipped edge cases

5. **💡 改进建议**
   - specific, executable next steps
   - process fixes
   - reliability improvements

## Writing rule
Use concrete evidence from raw files. Avoid vague praise or generic criticism.
