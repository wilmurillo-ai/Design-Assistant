# 变更日志 (Changelog)

所有重要的项目变更将记录在此文件中。

## [1.0.0] - 2026-03-12

### Added
- ✨ 初始版本发布
- 📜 `psctl.sh` 主控制脚本
  - 支持命令和脚本文件两种模式
  - 自动检测 Windows PowerShell 和 PowerShell Core
  - WSL 路径自动转换功能
  - 调试模式和详细输出模式
  - 彩色日志输出
  - 完整的帮助和版本信息
- 📖 完整的 README.md 文档
- 📄 Apache-2.0 许可证
- 🚫 .gitignore 文件
- 📝 变更日志模板

### Features
- 跨系统控制：从 WSL 调用 Windows PowerShell
- 路径转换：自动处理 WSL ↔ Windows 路径
- 错误处理：完善的错误检测和报告
- 灵活配置：支持环境变量和命令行选项

### Examples
- 系统信息查询
- 文件操作
- 进程管理
- 网络操作
