## 🔍 平台识别规则

### 自动识别
系统会自动识别 URL 所属平台：
```javascript
detectPlatform('https://a.feizhu.com/0HeTSg') 
// 返回：'fliggy'

detectPlatform('https://m.mafengwo.cn/p/123456')
// 返回：'mafengwo'
```

### 支持格式
- ✅ 完整 URL: `https://www.ctrip.com/p/123456`
- ✅ 短链接：`https://a.feizhu.com/abc123`
- ✅ 移动端：`https://m.qyer.com/deal/xxx`
- ✅ 无协议：`ctrip.com/p/123456`

## ⚠️ 注意事项

### 价格波动
- 不同平台价格可能有差异
- 建议跨平台对比
- 注意是否含机票/签证

### 退改政策
- 各平台退改政策不同
- 携程/飞猪相对灵活
- 特价产品可能不可退改

### 服务质量
- 携程：客服响应快
- 飞猪：平台 + 商家模式
- 其他：视具体商家而定
