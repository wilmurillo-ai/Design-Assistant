# SEO Optimizer Skill

## Overview
Search engine optimization for content and websites.

## Capabilities

### 1. Keyword Research
- Find primary keywords
- Long-tail opportunities
- Competitor keywords
- Search volume estimates

### 2. On-Page SEO
- Title tag optimization
- Meta descriptions
- Header structure
- Internal linking suggestions

### 3. Content Optimization
- Keyword density
- Readability score
- Semantic variations
- LSI keywords

### 4. Technical SEO
- Site speed suggestions
- Mobile optimization
- Schema markup
- Core Web Vitals

## Usage

### Commands
- `optimize for [keyword]`
- `suggest keywords for [topic]`
- `improve SEO for [URL/content]`
- `write meta description for [page]`
- `check readability of [text]`

## SEO Checklist

### Title Tags (50-60 chars)
- Include primary keyword
- Brand at end
- Unique per page

### Meta Descriptions (150-160 chars)
- Compelling summary
- Include CTA
- Unique per page

### Headers
- One H1 per page
- Include keywords
- Logical hierarchy

### Content
- 1-2% keyword density
- 300+ words minimum
- Include related keywords
- Original content
