# 微博热点日报

## 概述

获取微博热搜榜单，智能分类热点话题，生成分段式结构化日报。

**核心功能**：
- 从微博热搜源获取最新榜单
- 智能分类热点（财经科技/社会民生/文体娱乐/网络热点）
- 分段展示不同类型热点

## 使用场景

- 查看当日微博热搜
- 分析热点趋势分布
- 获取分类热点概览

## 数据源

- **主源**：https://weibo.g.renyuzhuo.cn/

## 日报格式

```
【微博热点日报 | YYYY-MM-DD】

📊 财经科技
- 热点1：简要说明
- 热点2：简要说明

📰 社会民生
- 热点1：简要说明
- 热点2：简要说明

🎬 文体娱乐
- 热点1：简要说明
- 热点2：简要说明

🔥 网络热点
- 热点1：简要说明
- 热点2：简要说明

---
by OpenClaw
```

## 热点分类

### 财经科技
股市行情、经济动态、科技新闻、企业动态、AI发展、新能源汽车等

### 社会民生
社会事件、民生话题、公共安全、教育话题、医疗健康等

### 文体娱乐
影视综艺、音乐动态、体育赛事、明星新闻、游戏动漫等

### 网络热点
网络流行、梗文化、网友讨论、生活趣事、情感话题等

## 使用方法

```
用户：微博热点日报
用户：今天的微博热搜
用户：微博热搜榜单
用户：生成微博热点日报
```

## 工作流程

1. **获取数据**：访问微博热搜源，解析榜单
2. **智能分类**：按财经科技/社会民生/文体娱乐/网络热点分类
3. **分段展示**：每个分类独立段落，清晰展示
4. **生成日报**：按格式输出
