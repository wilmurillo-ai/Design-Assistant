# Math Mentor Simple

这是一个简化版的数学辅导 skill，用于测试核心教学功能。

## 与原版的区别

| 特性 | 原版 | 简化版 |
|------|------|--------|
| 文件大小 | ~400 行 | ~100 行 |
| 意图识别 | 5 种详细分类 | 3 种简单分类 |
| 提示等级 | 5 级详细规则 | 5 级简化规则 |
| 教学策略 | 6 种策略 | 核心原则 |
| 错误归因 | 5 种详细类型 | 4 种简化类型 |
| 输出模板 | 多种详细骨架 | 3 个简单示例 |

## 核心功能

1. **分步提示**：5 级提示强度，根据学生表现调整
2. **错误检查**：识别概念/计算/审题/推理错误
3. **阶段总结**：掌握情况 + 关键问题 + 下一步建议

## 使用方法

在对话中当学生询问数学问题时自动触发。

## 测试要点

- [ ] 提示等级是否正确升降
- [ ] 是否避免直接给答案
- [ ] 错误识别是否准确
- [ ] 总结是否简洁有用
