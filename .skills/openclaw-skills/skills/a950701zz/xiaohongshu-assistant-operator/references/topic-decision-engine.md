# Topic Decision Engine

Executed once per daily cycle.

---

# Inputs

1. Last 5 creator notes
2. Recent marketing frequency
3. Industry trend signals
4. Engagement indicators

---

# Output Structure (3 Posts)

Post 1 — Industry / Trend
- Based on current AI signals
- Light 199 mention allowed

Post 2 — Marketing Controlled
- Apply marketing level rules

Post 3 — Interaction
- Question-driven
- Encourage comments

---

# Balance Rules

- Avoid repeating same theme 2 days in a row
- Rotate focus between RAG / Agent / Job market / Trend
- If 199 mentioned recently → reduce repetition
