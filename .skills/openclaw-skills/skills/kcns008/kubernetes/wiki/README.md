# wiki/ — Generated Wiki Pages

Maintained by the LLM agent. Browse freely, but let the agent handle writing.

## Page Types

| Type | Directory | Example |
|------|-----------|---------|
| Entity | `wiki/entities/` | People, orgs, projects |
| Concept | `wiki/concepts/` | Ideas, frameworks, methods |
| Topic | `wiki/topics/` | Subject areas |
| Source summary | `wiki/sources/` | Summaries of ingested docs |
| Comparison | `wiki/comparisons/` | Side-by-sides |
| Analysis | `wiki/analyses/` | Deep dives |

## Conventions

- Cross-references use `[[wiki-links]]` for Obsidian
- Every page listed in `index.md`
- Every change logged in `log.md`
