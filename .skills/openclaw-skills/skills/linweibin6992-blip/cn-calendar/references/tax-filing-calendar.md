# 中国企业常规税种纳税申报日历

来源：国家税务总局（https://www.chinatax.gov.cn/）

## 核心规则

1. **申报期限通用规则**：法定申报截止日为**每月15日**（或每季度终了后次月15日），遇节假日、休息日顺延至下一工作日。
2. **顺延原则**：若截止日恰逢法定节假日或周末，自动顺延到最近的工作日（不需另行通知）。
3. **具体顺延日期**：每年由国家税务总局官方公告，建议通过官网确认：https://www.chinatax.gov.cn/

| 税种 | 申报频率 | 法定截止日 | 备注 |

| 企业所得税（预缴） | 按月/季 | 次月/次季15日 | 大多数企业按季 |
| 个人所得税（代扣代缴） | 按月 | 次月15日 | |
| 消费税 | 按月 | 次月15日 | |

| 教育费附加 | 随增值税/消费税 | 同上 | |
| 地方教育附加 | 随增值税/消费税 | 同上 | |

| 税种 | 汇算期 | 截止日 |
|------|--------|--------|

## 各季度申报截止日参考（以2026年为例）

| 季度 | 预缴申报截止 | 顺延后截止（参考） |
|------|-------------|-------------------|
| 2026年Q1（1-3月） | 2026年4月15日 | 4月15日（周三，无需顺延） |
| 2026年Q2（4-6月） | 2026年7月15日 | 7月15日（周三，无需顺延） |
| 2026年Q3（7-9月） | 2026年10月15日 | 10月15日（周四，无需顺延） |

## 重要年度节点

| 月份 | 申报/操作 |
|------|----------|
| 1月 | 上年12月申报期；元旦假期顺延注意 |
| 2月 | 上年全年个人所得税汇算开始（3月1日） |
| 4月 | **企业所得税年度汇算清缴截止（4月30日）** |
| 5月 | 春节期间顺延关注 |
| 6月 | **个人所得税综合所得汇算截止（6月30日）** |
| 10月 | 国庆假期顺延注意 |
