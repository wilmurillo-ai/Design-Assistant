## 概览
本模板提供各类小说题材的特征、常见元素和写作建议。参考这些模板可以帮助保持题材特色，避免偏离预期风格。

## 玄幻题材

### 核心特征
- **力量体系**: 等级化的修炼力量（斗气、魔法、灵力等）
- **世界观**: 独特的魔法世界、异界、位面
- **主角**: 从弱到强的成长历程
- **冲突**: 争夺资源、种族战争、位面争霸

### 常见元素
- **力量等级**: 一阶至九阶、圣阶、神阶等
- **势力结构**: 家族、宗门、帝国、联盟
- **宝物**: 神器、圣药、功法、秘境
- **种族**: 人族、兽族、精灵族、魔族等

### 写作要点
1. **力量体系清晰**: 明确定义各等级实力差距和晋升条件
2. **升级节奏合理**: 不要让主角升级过快或过慢
3. **世界观宏大**: 构建独特且有逻辑的世界设定
4. **对手层次分明**: 逐步挑战更强的敌人
5. **爽点设置**: 打脸、升级、获得宝物等

### 避免事项
- 力量体系前后矛盾
- 主角过于无敌导致失去挑战性
- 世界观设定不清晰导致读者困惑

## 仙侠题材

### 核心特征
- **修行理念**: 道法自然、追求长生、逆天而行
- **力量体系**: 炼气、筑基、金丹、元婴等传统体系
- **世界观**: 修真界、仙界、神界
- **主角**: 寻求道、追求长生、渡劫飞升

### 常见元素
- **修炼境界**: 练气、筑基、金丹、元婴、化神、炼虚、合体、大乘
- **宗门**: 正道宗门、魔道宗门、散修联盟
- **法宝**: 飞剑、法宝、灵兽、丹药
- **天劫**: 心魔劫、雷劫、天人五衰

### 写作要点
1. **修行哲学**: 融入道家思想和修行理念
2. **境界描述**: 用诗意或哲学语言描述境界差距
3. **宗门冲突**: 正魔之争、资源争夺、理念冲突
4. **情感纠葛**: 师徒之情、道侣之情、因果情缘
5. **天道规则**: 飞升条件、天劫规律、因果报应

### 避免事项
- 功法效果描述不清
- 境界突破过于随意
- 因果逻辑混乱

## 都市题材

### 核心特征
- **现实背景**: 现代城市、职场、校园
- **社会关系**: 亲情、友情、爱情、职场关系
- **成长主题**: 青春成长、职场奋斗、人生抉择
- **现实主义**: 反映社会问题和人生困境

### 常见元素
- **职场**: 公司、行业、职业发展
- **校园**: 学校、社团、青春友情
- **家庭**: 家庭关系、代际冲突
- **爱情**: 初恋、暗恋、恋爱、婚姻

### 写作要点
1. **真实感**: 细节真实可信，反映现实
2. **人物立体**: 避免脸谱化，塑造复杂角色
3. **情感真挚**: 情感描写要真实自然
4. **节奏把握**: 根据内容调整叙述节奏
5. **社会观察**: 反映社会现象和问题

### 避免事项
- 过于理想化或过于阴暗
- 人物行为脱离现实
- 情节发展不合逻辑

## 科幻题材

### 核心特征
- **科技设定**: 未来科技、太空探索、人工智能
- **科学基础**: 基于科学原理的合理推演
- **未来想象**: 对未来社会的想象和反思
- **哲学思考**: 科技发展对人类的影响

### 常见元素
- **技术**: 人工智能、基因工程、星际旅行、时间旅行
- **社会**: 未来社会形态、外星文明、后人类时代
- **冲突**: 人机冲突、星际战争、资源争夺
- **主题**: 存在主义、科技伦理、人性探讨

### 写作要点
1. **科学逻辑**: 设定要有科学依据和内在逻辑
2. **技术细节**: 适当的技术细节增加真实感
3. **人文关怀**: 科技服务于人性探讨
4. **世界观独特**: 构建独特且一致的未来世界
5. **节奏掌控**: 信息量适中，避免说教

### 避免事项
- 科学设定前后矛盾
- 过度堆砌技术术语
- 忽视人物和情节

## 恐怖题材

### 核心特征
- **氛围营造**: 恐怖、紧张、不安的氛围
- **未知恐惧**: 对未知的恐惧、超自然现象
- **心理恐惧**: 深层心理恐惧、创伤、妄想
- **悬疑解谜**: 逐步揭示真相，增加紧张感

### 常见元素
- **场景**: 废弃建筑、医院、墓地、偏远地区
- **超自然**: 鬼魂、恶魔、诅咒、预言
- **心理**: 创伤、幻觉、疯狂、死亡
- **情节**: 逐步揭示、反转、意外

### 写作要点
1. **氛围营造**: 通过环境描写营造恐怖氛围
2. **悬念设置**: 逐步释放信息，保持悬念
3. **心理描写**: 深入描写角色的恐惧心理
4. **感官刺激**: 调动读者的感官恐惧
5. **节奏控制**: 张弛有度，避免持续紧张

### 避免事项
- 过度依赖血腥暴力
- 恐怖效果单一
- 逻辑漏洞过多

## 其他题材

### 恋爱题材
- **核心**: 情感发展、爱情故事
- **元素**: 相遇、相处、冲突、和解
- **要点**: 情感真实、节奏合理、有新意

### 历史题材
- **核心**: 历史背景、历史人物
- **元素**: 历史事件、文化背景、时代特色
- **要点**: 史实准确、时代氛围、人物可信

### 武侠题材
- **核心**: 武功、江湖、侠义
- **元素**: 武功秘籍、江湖恩怨、侠义精神
- **要点**: 武功体系清晰、江湖规矩明确、侠义精神体现

### 悬疑题材
- **核心**: 谜题、推理、解密
- **元素**: 线索、红鲱鱼、反转
- **要点**: 线索合理、逻辑严密、出人意料
