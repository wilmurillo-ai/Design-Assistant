# 章节模板

## 标题格式
```
第 X 章 [章节标题]
```

## 标准结构

### 开头（钩子）
立即抓住读者注意力的开场，可以是：
- 悬念或疑问
- 突发事件
- 角色面临的困境
- 对话或行动

示例：
```
林风站在悬崖边，手中紧握着那块古老的玉佩，远处是滔滔云海，而身后是追兵的喊杀声。他只剩下两个选择：跳下去，或者被抓住。
```

### 发展（展开）
推进情节，包含：
- 角色行动
- 对话交流
- 场景描写
- 情感变化
- 冲突升级

示例：
```
"你真的要这么做吗？"身旁的老者低声问道，眼中满是担忧。

林风转过身，看着这位陪伴自己十年的师父，苦笑道："师父，我没有选择。那件事必须有人去做，而我是唯一能做的人。"

老者沉默了片刻，从怀中掏出一本泛黄的古籍："如果你坚持，就把这个带上。这是你父亲留下的。"
```

### 高潮（转折）
情节的转折点：
- 关键抉择
- 意外发现
- 冲突爆发
- 真相揭露

示例：
```
就在林风准备跳下悬崖时，一道金光突然从玉佩中爆发，将他整个人包裹。他感到一股强大的力量涌入体内，那是他从未体验过的能量。

"这是...上古血脉之力？"老者震惊地看着这一幕，"原来你父亲的血脉是真的！"
```

### 结尾（收束）
解决本章问题，为下一章铺垫：
- 解决本章核心问题
- 揭示新信息
- 埋下伏笔
- 创造期待

示例：
```
金光消散后，林风发现自己并没有跳下悬崖，而是悬浮在半空中。他看着自己的双手，能够感受到那股新的力量。

"看来，命运的齿轮开始转动了。"林风低声说道，望向远方的天际。

（未完待续）
```

## 完整示例

```
第 1 章 觉醒

林风站在悬崖边，手中紧握着那块古老的玉佩，远处是滔滔云海，而身后是追兵的喊杀声。他只剩下两个选择：跳下去，或者被抓住。

"你真的要这么做吗？"身旁的老者低声问道，眼中满是担忧。

林风转过身，看着这位陪伴自己十年的师父，苦笑道："师父，我没有选择。那件事必须有人去做，而我是唯一能做的人。"

老者沉默了片刻，从怀中掏出一本泛黄的古籍："如果你坚持，就把这个带上。这是你父亲留下的。"

林风接过古籍，翻开第一页，上面密密麻麻地写满了古老的文字。虽然他不认识这些文字，但当他触摸到书页时，那些文字仿佛活了过来，闪烁着微弱的光芒。

"这是什么？"林风惊讶地问道。

"上古修仙秘籍，你父亲用生命守护的东西。"老者的声音变得严肃起来，"记住，这本书不仅能让你修炼，还能改变整个世界的命运。"

远处传来了追兵的声音，越来越近。

"快走吧！"老者催促道。

林风深吸一口气，转身面向悬崖。他知道，跳下去可能意味着死亡，但留下来必死无疑。

"师父，如果我能活下来，一定会回来找你。"

"活下去！"老者的声音有些颤抖。

林风闭上眼睛，纵身一跃。

风声呼啸，下坠的感觉让他心脏狂跳。就在他以为自己要坠落深渊时，一道金光突然从玉佩中爆发，将他整个人包裹。

他感到一股强大的力量涌入体内，那是他从未体验过的能量。他的身体不再下坠，而是悬浮在半空中，缓缓向上升去。

"这是...上古血脉之力？"老者震惊地看着这一幕，"原来你父亲的血脉是真的！"

金光消散后，林风发现自己回到了悬崖边，但他的身上散发着淡淡的光芒。他看着自己的双手，能够感受到那股新的力量。

"看来，命运的齿轮开始转动了。"林风低声说道，望向远方的天际。

追兵已经到了悬崖边，看到悬浮在空中的林风，都愣住了。

"妖人！"其中一人大喊道。

林风转过身，看着这些追兵，心中第一次有了信心。他不再是那个无力的少年，而是拥有力量的战士。

"你们来得正好，"林风说道，"正好试试我的新力量。"

（本章完，共 1256 字）
```

## 写作要点

1. **字数控制**
   - 标准章节：2000-4000 字
   - 避免过短（少于 1000 字）
   - 避免过长（超过 6000 字）

2. **结构比例**
   - 开头：10-15%
   - 发展：50-60%
   - 高潮：20-25%
   - 结尾：5-10%

3. **节奏把握**
   - 开头快速引入
   - 发展层次分明
   - 高潮紧张激烈
   - 结尾留有余味

4. **伏笔设置**
   - 在结尾设置悬念
   - 不要解决所有问题
   - 为下一章做好铺垫

5. **情感投入**
   - 让读者关心主角
   - 创造情感共鸣
   - 保持角色情感连贯
