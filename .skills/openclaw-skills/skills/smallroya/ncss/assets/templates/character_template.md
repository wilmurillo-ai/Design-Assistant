## 使用说明

1. **填写完整性**
   - 尽量填写所有部分
   - 未知部分可以留空，待后续补充

2. **细节具体化**
   - 避免模糊描述
   - 使用具体细节让角色更立体

3. **保持一致性**
   - 确保角色行为与设定一致
   - 修改设定时更新角色卡

4. **动态更新**
   - 随着故事推进更新角色信息
   - 记录角色的重要变化

## 完整示例

### 基础信息

**角色名称**
```

```

**角色类型**
```
主角
```

**年龄**
```
18岁

**外貌特征**
```

```

**性格特征**
```

```

**目标**

```
