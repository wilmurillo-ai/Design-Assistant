# 星火燎原

## 技能介绍

星火燎原是求是 OpenClaw Skills 系列中的核心技能，基于毛泽东的《星星之火，可以燎原》，强调"建立根据地，不做流寇"，用于从零开始、资源有限时从小处着手积累发展。

## 核心功能

- **根据地建立**：选择适合的起点，建立基础
- **经验积累**：在小范围内积累经验和资源
- **逐步扩展**：在基础稳固后，逐步扩展范围
- **规模效应**：通过不断积累和扩展，形成规模效应

## 触发条件

当从零开始，资源有限，需要从小处着手积累发展时触发。

## 执行步骤

1. **建立根据地**：选择一个适合的起点，建立基础
2. **积累经验**：在小范围内积累经验和资源
3. **逐步扩展**：在基础稳固后，逐步扩展范围
4. **形成规模**：通过不断积累和扩展，形成规模效应
5. **持续发展**：保持发展势头，持续创新和改进

## 核心原则

建立根据地，不做流寇。从小处着手，积累发展。

## 不适用场景

- 资源充足，不需要从小处着手
- 已经有成熟的基础和规模
- 时间紧急，需要快速达到规模
- 不需要长期发展的项目

## 星火燎原的应用

- **创业初期**：从零开始创业，建立基础
- **新领域探索**：进入新领域，积累经验
- **能力建设**：从小处着手，逐步提升能力
- **市场开拓**：从局部市场开始，逐步扩展

## 常见错误

| 错误 | 正确做法 |
|------|---------|
| 急于求成 | 稳扎稳打，逐步发展 |
| 忽视基础 | 重视基础建设，确保稳固 |
| 盲目扩张 | 在基础稳固后再扩展 |
| 缺乏耐心 | 保持耐心，注重积累 |
| 不适应变化 | 灵活适应环境变化，及时调整策略 |

## 操作规程

当本 skill 被触发时，执行以下步骤：
1. **起点选择**：选择一个适合的起点，考虑资源和环境因素
2. **基础建设**：在起点建立稳固的基础，积累经验和资源
3. **经验总结**：总结初期经验，优化方法和策略
4. **逐步扩展**：在基础稳固后，逐步扩展范围和规模
5. **规模效应**：通过扩展形成规模效应，提高效率和影响力
6. **持续创新**：保持创新，适应变化，持续发展

## 与其他 skill 的关系

- **持久战略**：星火燎原需要长期的战略规划
- **集中兵力**：在发展过程中，需要集中兵力于关键领域
- **实践认识论**：通过实践不断积累经验和改进
- **统筹兼顾**：在发展过程中需要统筹兼顾各方面的因素
