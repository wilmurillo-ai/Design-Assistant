# 持久战略

## 技能介绍

持久战略是求是 OpenClaw Skills 系列中的核心技能，基于毛泽东的《论持久战》，强调"战略上藐视，战术上重视"，用于面对长期复杂任务时制定长期计划。

## 核心功能

- **战略分析**：分析任务的性质、特点和挑战
- **阶段划分**：将长期任务划分为多个阶段
- **计划制定**：为每个阶段制定具体的计划和目标
- **资源分配**：合理分配资源，确保各阶段的顺利进行
- **执行监控**：监控执行情况，及时调整计划

## 触发条件

当面对长期复杂任务，需要制定长期计划时触发。

## 执行步骤

1. **战略分析**：分析任务的性质、特点和挑战
2. **阶段划分**：将长期任务划分为多个阶段
3. **制定计划**：为每个阶段制定具体的计划和目标
4. **资源分配**：合理分配资源，确保各阶段的顺利进行
5. **执行监控**：监控执行情况，及时调整计划
6. **阶段评估**：对每个阶段的成果进行评估
7. **持续推进**：保持动力，持续推进任务的完成

## 核心原则

战略上藐视，战术上重视。不急于求成，也不畏难放弃。

## 不适用场景

- 短期简单任务
- 已经有明确的解决方案
- 时间紧急且需要快速完成的任务
- 不需要长期规划的任务

## 持久战略的应用

- **长期项目**：制定长期项目的阶段性计划
- **职业发展**：规划个人或组织的长期发展
- **问题解决**：解决复杂的长期问题
- **能力建设**：长期能力建设和培养

## 常见错误

| 错误 | 正确做法 |
|------|---------|
| 急于求成 | 制定合理的时间表，稳步推进 |
| 畏难放弃 | 保持信心，坚持到底 |
| 缺乏规划 | 制定详细的阶段计划 |
| 资源分配不当 | 合理分配资源，确保各阶段的需求 |
| 监控不力 | 定期监控执行情况，及时调整 |

## 操作规程

当本 skill 被触发时，执行以下步骤：
1. **任务分析**：分析任务的性质、难度和时间要求
2. **阶段划分**：将任务划分为多个合理的阶段
3. **目标设定**：为每个阶段设定具体的目标和指标
4. **计划制定**：制定每个阶段的详细计划和行动方案
5. **资源规划**：规划所需的资源和支持
6. **执行实施**：按照计划执行，定期检查进度
7. **评估调整**：对每个阶段进行评估，根据情况调整计划
8. **持续推进**：保持动力，直至任务完成

## 与其他 skill 的关系

- **矛盾分析法**：持久战略需要分析长期任务中的矛盾
- **实践认识论**：持久战略需要通过实践不断调整
- **集中兵力**：在持久战略中，需要在关键阶段集中兵力
- **统筹兼顾**：持久战略需要统筹兼顾各方面的因素
