# Suno Poetry Music Creator

增强版 Suno 歌曲创作工具，支持参考歌曲分析和智能歌词优化。

## 功能特点

- 🎧 **参考歌曲分析** - 分析用户提供的参考歌曲，提取风格 DNA
- 🔄 **智能歌词优化** - 基于 Suno 生成反馈迭代优化歌词
- 🏮 **中国古典风格** - 支持平仄韵律和《中华新韵》14 韵部
- ✨ **诗意提升** - 将直白描述转化为诗意意象

## 使用场景

- 创作类似某首歌曲风格的新歌
- 优化 Suno 生成的歌词
- 将中国古典诗词改编成歌曲
- 匹配特定的音乐风格

## 工作流程

1. **发现** - 理解主题，收集参考歌曲
2. **分析** - 提取风格特征，研究韵律
3. **创作** - 生成初版歌词和风格标签
4. **提升** - 诗意化改写，增强意象
5. **优化** - 根据反馈迭代改进
