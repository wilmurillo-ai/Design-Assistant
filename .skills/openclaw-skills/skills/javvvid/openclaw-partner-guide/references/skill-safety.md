# Skill Safety

## Review Checklist

- Confirm source and author.
- Read all files.
- Check writes, commands, file access, and network calls.
- Escalate credential requests, sensitive-file access, obfuscation, or broad privilege needs.

## Preferred Skills

- Local-first
- Auditable
- Narrow scope
- Honest about what they actually do

## Publishing Checklist

- Remove secrets and local-path assumptions.
- Keep metadata concise and trigger-friendly.
- Test representative scripts.
