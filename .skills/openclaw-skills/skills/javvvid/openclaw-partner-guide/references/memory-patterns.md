# Memory Patterns

## Retrieval Order

1. Run semantic search for the topic.
2. Read only the relevant snippets.
3. Answer from retrieved evidence.
4. State uncertainty when retrieval is weak.

## Storage Split

- Long-term memory: durable preferences, stable decisions, recurring context.
- Daily logs: raw notes, progress, temporary state, lessons learned.

## Context Compression

When summarizing a long thread, preserve:

- Goal
- Current status
- Key decisions
- Blockers
- Next step
