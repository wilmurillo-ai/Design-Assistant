## 🚀 快速开始

### 1. 安装依赖

```bash
pip3 install akshare pandas numpy pyyaml
```

### 2. 配置系统

```bash
# 复制配置文件
cp /tmp/config_v22.yaml ./config.yaml

# 编辑配置（可选）
vim config.yaml
```

### 3. 运行回测

```bash
# 复制文件
cp /tmp/run_backtest_v22_simple.py .

# 运行回测
python3 run_backtest_v22_simple.py
```

### 4. 实时分析

```python
from pipeline import StockAnalysisPipeline

# 创建流水线
pipeline = StockAnalysisPipeline(
    config={
        'region': 'cn',
        'realtime': {'enabled': True},
        'market_environment': {'enabled': True},
    }
)

# 分析股票
results = pipeline.analyze_stocks(['600519', '000858'])
```

## 🎯 技术亮点

### 1. 模块化设计

每个模块独立运行，职责清晰：

```python
# 独立使用各模块
from data_manager import DataFetcherManager
from market_analyzer import MarketAnalyzer
from email_notifier import EmailNotifier

# 或使用完整流水线
from pipeline import StockAnalysisPipeline
```

### 2. 配置化驱动

所有参数通过YAML配置：

```yaml
signal:
  factors:
    momentum: 0.25
    ma_deviation: 0.20
    # ... 可灵活调整

strategy:
  stop_loss: -0.05
  take_profit: 0.10
```

### 3. 多数据源容错

自动故障切换，提升稳定性：

```python
# 优先级: Efinance > AKShare > Tushare
# 失败自动切换下一个
# 熔断机制保护
```

### 4. 动态环境适应

根据市场环境自动调整策略：

```python
强势市场: 买入阈值 8.5（正常交易）
震荡市场: 买入阈值 9.0（提高门槛）
弱势市场: 买入阈值 9.5（谨慎观望）
```

### 5. 完整通知体系

多类型消息推送：

```python
买入信号 → 邮件通知
卖出信号 → 邮件通知
风险警报 → 邮件通知
每日报告 → 邮件通知
大盘复盘 → 邮件通知
```
