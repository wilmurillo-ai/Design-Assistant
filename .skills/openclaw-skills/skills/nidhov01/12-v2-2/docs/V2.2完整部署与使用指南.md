## 📊 功能模块详解

### 1. 数据管理器（data_manager.py）

**功能**: 多数据源自动故障切换

```python
from data_manager import DataFetcherManager

manager = DataFetcherManager()

# 获取日线数据
df = manager.get_daily_data("600519", days=30)

# 获取实时行情
quote = manager.get_realtime_quote("600519")
print(f"量比: {quote.volume_ratio}")
print(f"换手率: {quote.turnover_rate}%")
```

**数据源优先级**:
1. Efinance（最快）
2. AKShare（稳定）
3. Tushare（备用）

**熔断机制**:
- 连续失败3次 → 熔断5分钟
- 自动恢复 → 半开状态

### 2. 大盘分析器（market_analyzer.py）

**功能**: 判断市场环境，动态调整阈值

```python
from market_analyzer import MarketAnalyzer

analyzer = MarketAnalyzer(region='cn')

# 获取市场概览
overview = analyzer.get_market_overview()
analyzer.print_overview(overview)

# 判断市场环境
env = analyzer.analyze_environment(overview)
threshold = analyzer.get_buy_threshold(env)

print(f"市场环境: {env.value}")
print(f"买入阈值: {threshold}")
```

**环境分类**:
- 强势（STRONG）: 指数多头 + 涨跌比>2 → 阈值8.5
- 震荡（NEUTRAL）: 其他情况 → 阈值9.0
- 弱势（WEAK）: 指数空头 + 涨跌比<0.5 → 阈值9.5

### 3. 邮件通知器（email_notifier.py）

**功能**: 推送交易信号和警报

```python
from email_notifier import EmailNotifier, NotificationConfig

config = NotificationConfig(
    enabled=True,
    sender="your_email@qq.com",
    password="auth_code",
    receivers=["receiver@example.com"],
)

notifier = EmailNotifier(config)

# 发送买入信号
notifier.send_buy_signal(
    code="600519",
    name="贵州茅台",
    price=1680.00,
    signal_score=8.8

# 发送卖出信号
notifier.send_sell_signal(
    code="600519",
    name="贵州茅台",
    price=1750.00,
    reason="止盈",
    buy_price=1680.00,
    holding_days=15
)

# 发送风险警报
notifier.send_risk_alert(
    code="600519",
    name="贵州茅台",
    current_price=1600.00,
    drawdown=-0.08,
    max_drawdown=-0.10
)

# 发送每日报告
notifier.send_daily_report(
    positions=positions_list,
    total_pnl=0.017,
    win_rate=0.67,
    market_env="neutral"
)
```

### 4. 分析流水线（pipeline.py）

**功能**: 整合所有模块，完整分析流程

```python
from pipeline import StockAnalysisPipeline

pipeline = StockAnalysisPipeline(
    config={
        'region': 'cn',
        'max_workers': 4,
        'realtime': {'enabled': True},
        'chip_distribution': {'enabled': True},
        'market_environment': {'enabled': True},
    }
)

# 批量分析
results = pipeline.analyze_stocks(
    stock_list=['600519', '000858', '600036'],
    buy_threshold=9.0,
    send_notification=True
)

# 查看结果
for code, result in results.items():
    print(f"\n{result.name}({code}):")
    print(f"  信号: {result.signal_score:.2f}/10")
    print(f"  建议: {result.recommendation}")
    print(f"  原因: {result.reason}")
    print(f"  风险: {result.risk_level}")
```

**分析流程**:
```
数据获取 → 实时行情 → 筹码分析 → 大盘环境 →
信号计算 → 决策生成 → 风险评估 → 通知推送
```

## ⚠️ 注意事项

### 数据质量

1. **AKShare连接**: 可能遇到网络波动，属于正常现象
2. **数据缺失**: 某些股票可能数据不足60天
3. **异常值**: 系统会自动处理，但需关注日志

### 性能考虑

1. **回测时间**: V2.2计算量略大于V2.1
2. **内存占用**: 同时分析多只股票会占用较多内存
3. **并发控制**: 可通过`max_workers`调整并发数

### 邮件配置

1. **授权码**: QQ邮箱需要授权码，不是登录密码
2. **SMTP限制**: 部分邮箱有发送频率限制
3. **通知时机**: 避免过度通知造成干扰

### API密钥

1. **图片识别**: 需要Gemini/OpenAI API Key
2. **AI回测**: 需要配置相关API
3. **费用**: 注意API调用的费用限制

## 🛠️ 故障排查

### 问题1: AKShare连接失败

```
Connection aborted: RemoteDisconnected
```

**解决方案**:
1. 检查网络连接
2. 等待一段时间后重试
3. 启用efinance备用数据源

### 问题2: 邮件发送失败

```
SMTP authentication error
```

**解决方案**:
1. 确认使用授权码而非登录密码
2. 检查SMTP服务器地址和端口
3. 部分邮箱需要开启"IMAP/SMTP服务"

### 问题3: 数据不足

```
Skipping XXX: 数据不足
```

**解决方案**:
1. 确认股票代码正确
2. 新股上市时间不足60天会跳过
3. 可调低`calculation_days`参数

### 问题4: 内存不足

```
MemoryError
```

**解决方案**:
1. 降低`max_workers`并发数
2. 分批分析股票
3. 增加`chunk_size`参数
