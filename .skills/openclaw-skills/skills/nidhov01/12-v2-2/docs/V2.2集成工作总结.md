## 📊 V2.2核心改进

### 信号系统升级

| 维度 | V2.1 | V2.2 | 改进 |
|-----|------|------|------|
| **因子数量** | 6个 | 8个 | +2个 |
| **因子权重** | 总和100% | 总和100% | 重新分配 |
| **新增因子** | - | 量比(5%) + 筹码(5%) | - |
| **动量指标** | 30% | 25% | -5% |
| **均线偏离** | 25% | 20% | -5% |

### 新增因子详解

#### 1. 量比因子 (5%)

**计算方法**:
```
量比 = 今日成交量 / 5日平均成交量
```

**评分标准**:
- 量比 1.0-2.0（正常）: 6分
- 量比 2.0-3.0（温和放量）: 8分
- 量比 3.0-5.0（明显放量）: 9分
- 量比 >5.0（巨量）: 5分（风险）
- 量比 <1.0（缩量）: 3分

#### 2. 筹码集中度因子 (5%)

**计算方法**:
```
使用价格变异系数(CV)作为筹码集中度代理指标
CV = (标准差 / 均值) × 100%
```

**评分标准**:
- CV <5%（高度集中）: 9分
- CV 5-10%（较集中）: 7分
- CV 10-15%（中等）: 5分
- CV 15-20%（较分散）: 4分
- CV >20%（高度分散）: 3分
