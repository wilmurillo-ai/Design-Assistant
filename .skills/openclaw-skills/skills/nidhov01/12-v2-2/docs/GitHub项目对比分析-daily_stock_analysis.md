## 📊 项目概览

### daily_stock_analysis
这是一个**基于AI大模型**的A股/港股/美股自选股智能分析系统,每日自动分析并推送「决策仪表盘」到多种通讯渠道。

**核心特点**:
- 🤖 AI驱动决策(一句话核心结论 + 精确买卖点位)
- 📱 多渠道推送(企业微信/飞书/Telegram/邮箱等)
- 🌍 全球市场支持(A股/港股/美股)
- 🎯 内置11种交易策略(均线金叉/缠论/波浪理论等)
- 🔄 GitHub Actions定时运行(无需服务器)
- 🖥️ Web界面管理
- 📈 AI回测验证功能

| 理念 | daily_stock_analysis | 您的V2.1系统 |
