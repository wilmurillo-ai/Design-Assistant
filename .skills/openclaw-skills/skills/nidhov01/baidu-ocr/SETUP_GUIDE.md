# 百度 OCR API 配置指南

## 当前状态

✅ **API Key 验证成功**
- Token 获取正常
- 应用认证通过

❌ **OCR 服务未开通**
- 错误：`No permission to access data`
- 原因：需要在百度 AI 平台开通 OCR 服务

---

## 开通步骤

### 1. 登录百度 AI 开放平台

访问：https://console.bce.baidu.com/ai/

使用百度账号登录。

### 2. 找到 OCR 服务

1. 点击左侧菜单 **"应用管理"**
2. 找到 **"文字识别 OCR"**
3. 点击 **"开通服务"**

### 3. 创建/选择应用

1. 如果已有应用，选择该应用
2. 如果没有，点击 **"创建应用"**
   - 应用名称：例如 `小飞象 OCR`
   - 应用类型：选择 `通用`
   - 描述：可选

### 4. 开通具体服务

在应用详情页面，找到以下服务并开通：

- ✅ **通用文字识别（标准版）** - 每天 500 次免费
- ✅ **通用文字识别（高精度版）** - 每天 50 次免费
- ✅ **表格文字识别** - 每月 500 次免费
- ✅ **公式识别** - 每月 500 次免费

### 5. 获取新的 API Key

开通服务后：

1. 进入应用详情页
2. 点击 **"概览"** 或 **"应用信息"**
3. 复制新的 **API Key** 和 **Secret Key**

## 常见问题

### Q: 提示 "No permission to access data"
**A**: OCR 服务未开通，按上述步骤开通即可。

### Q: 提示 "Invalid app"
**A**: API Key 或 Secret Key 错误，重新复制。

### Q: 提示 "QPS 超限"
**A**: 调用频率过高，稍等再试或升级套餐。
