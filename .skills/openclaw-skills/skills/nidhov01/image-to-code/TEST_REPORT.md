## 改进建议

### 短期优化
1. **图像预处理**: 增加对比度、去噪
2. **后处理**: 合并同一行的多个识别块
3. **上下文校正**: 根据前后文修正 OCR 错误

### 长期方案
1. **切换 OCR 引擎**: 使用百度/腾讯 OCR API（准确率 95%+）
2. **视觉 AI**: 使用 GPT-4V/Claude Vision（准确率 98%+）
3. **混合方案**: Tesseract + 视觉 AI 校验

## 结论

**当前状态**: 基本功能可用，但 OCR 精度有待提升

**推荐方案**: 
- 对精度要求不高 → 使用 Tesseract（免费、离线）
- 对精度要求高 → 使用百度/腾讯 OCR API（付费、高精度）
