## ✅ 核心优化

### 优化 1：智能下标规则

**v2.1.0 问题**：
```python
# 过于激进 - 所有字母 + 数字都转下标
re.sub(r'([a-zA-Z])([0-9]+)', r'\1_\2', text)

# 错误示例：
×0.023 → \times_0.023  ❌
×994 → \times_994      ❌
```

**v2.2.0 改进**：
```python
# 只处理特定变量的下标
re.sub(r'([pPtTnN])([0-9])(?![0-9.])', r'\1_\2', text)

# 正确示例：
p1 → p_1              ✅
t2 → t_2              ✅
×0.023 → \times 0.023 ✅（保持原样）
×994 → \times 994     ✅（保持原样）
```

### 优化 2：OCR 错误纠正字典

```python
# 新增常见错误纠正
'R41' → 'Re'                      # 雷诺数
'30u' → '3u^2'                    # 3 倍速度平方
'EAp' → '\Sigma\Delta p'          # 压力降求和
'd:P' → 'd\cdot u\cdot\rho'       # 直径·速度·密度
```

### 优化 3：标题识别优化

```python
# 长度检查
if len(text) < 3:
    return None

# 内容检查（连接词不是标题）
if any(word in text for word in ['所以', '因此', '则', '故', '即']):
    return None
```

### 修改的方法

1. **text_to_latex()**
   - 智能下标规则（只处理特定变量）
