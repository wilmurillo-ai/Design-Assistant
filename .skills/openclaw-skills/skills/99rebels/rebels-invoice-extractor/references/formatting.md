# Platform Formatting Examples

Adapt the confirmation output to the current channel's supported formatting.

---

## Slack

Use bold (`*text*`) and blockquote formatting:

```
📋 *Invoice Extracted*
*Vendor:* Amazon
*Date:* 2026-04-01
*Invoice #:* INV-2026-001
*Description:* Office supplies — keyboard and monitor
*Total:* €539.96 (incl. €100.97 tax)
*Category:* office (auto)

Add to ledger? (yes/edit/skip)
```

### Slack Expense Summary

```
📊 *Expense Summary — April 2026*
• software: €312.50
• travel: €189.00
• office: €539.96
• food: €67.40
━━━━━━━━━━━━━━━
*Total: €1,108.86*
```
