# 小红书发布助手 (xhs-publish)

小红书图文/视频笔记发布助手，支持文案生成、封面制作、一键发布。

## 功能

- 📷 **图文笔记**：标题生成 → 正文创作 → 封面制作 → 发布
- 🎬 **视频笔记**：标题生成 → 正文创作 → 视频准备 → 发布
- 🤖 **AI 生图**：支持混元、豆包等模型
- 🎬 **AI 生成视频**：支持豆包 Seedance

## 使用方法

当用户要求发小红书、写小红书笔记时自动触发。

## 依赖

- ImageMagick (`convert`)
- curl

## 安装

```bash
clawhub install xhs-publish
```
