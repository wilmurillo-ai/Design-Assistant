## 🎯 核心功能：自然语言控制

### 什么是自然语言控制？

用户可以用日常语言直接控制桌面，无需记忆具体命令。

**示例**:
```
❌ 旧方式：bash scripts/screenshot.sh
✅ 新方式："帮我截个屏"

❌ 旧方式：bash scripts/window_snap.sh left Safari
✅ 新方式："把 Safari 放到左边"

❌ 旧方式：bash scripts/app_control.sh open "Google Chrome"
✅ 新方式："打开 Chrome 浏览器"
```

## 🧠 技术实现

### 架构

```
用户输入
  ↓
自然语言解析 (parse_command)
  ↓
动作识别 + 参数提取
  ↓
置信度评估
  ↓
执行命令 (execute_command)
  ↓
反馈结果
```

### 解析流程

**示例**: "帮我截个屏"

1. **文本预处理**: `帮我截个屏`
2. **关键词匹配**: `截个屏` → 截屏类
3. **参数提取**: `delay=0`
4. **置信度评估**: `0.95` (95%)
5. **执行命令**: `screenshot`
6. **反馈**: `✅ 截屏成功！`

---

## 📈 智能特性

### 1. 模糊匹配

支持多种说法：
- "截屏"、"截图"、"截个屏"、"拍个照" → 都识别为截屏

### 2. 参数提取

自动提取参数：
- "延迟 5 秒截屏" → `delay=5`
- "把 Chrome 放右边" → `app=Chrome, position=right`

### 3. 置信度评估

- **高置信度** (≥0.8): 直接执行
- **中置信度** (0.6-0.8): 执行并提示
- **低置信度** (<0.6): 提示用户重新表述

### 4. 上下文理解

识别应用名称：
- Safari、Chrome、Firefox
- VS Code、Terminal、iTerm
- QQ、微信、钉钉、企业微信
