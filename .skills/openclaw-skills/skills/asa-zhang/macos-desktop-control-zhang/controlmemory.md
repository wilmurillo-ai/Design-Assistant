## 📖 使用指南

### 质量评分系统

**评分规则**:
```
质量评分 = 借鉴次数 - 2 × 失败次数
```

**示例**:
```
操作 A: 👁️ 100 次 ❌ 5 次 → 评分 = 100 - 2×5 = 90
操作 B: 👁️ 80 次  ❌ 2 次 → 评分 = 80 - 2×2 = 76
操作 C: 👁️ 120 次 ❌ 30 次 → 评分 = 120 - 2×30 = 60

排序：A(90) > B(76) > C(60)
```

### 优先规则

同一应用的多个操作：
1. **质量评分高的优先**（真实可靠性）
2. 借鉴次数多的优先
3. 成功率高的优先

**示例**:
```
打开浏览器：
1. 打开 Safari 👁️ 150 ❌ 5  评分 140 ✅ (优先)
2. 打开 Chrome 👁️ 100 ❌ 2  评分 96  ✅
3. 打开 Firefox 👁️ 50  ❌ 1  评分 48  ✅
```

### 成功率计算

```
实际成功率 = 借鉴次数 / (借鉴次数 + 失败次数) × 100%

示例：
- 👁️ 100 ❌ 0 → 100% (完美)
- 👁️ 100 ❌ 10 → 91% (优秀)
- 👁️ 100 ❌ 50 → 67% (一般)
- 👁️ 100 ❌ 100 → 50% (较差)
```
