## 🎯 核心功能：Vosk 离线语音控制

### 什么是 Vosk 语音控制？

完全离线、免费、隐私保护的语音识别方案，无需 API Key，无需网络。

**特点**:
- ✅ **完全免费** - 无隐藏费用
- ✅ **离线运行** - 无需网络
- ✅ **隐私保护** - 数据不上云
- ✅ **中文支持** - 普通话识别
- ✅ **易于集成** - 与自然语言控制无缝对接

## 📦 安装步骤

### Step 1: 安装 Vosk（已完成 ✅）

```bash
pip3 install --user --break-system-packages vosk
```

**状态**: ✅ 已安装 (v0.3.44)

---

### Step 2: 下载语音模型

**模型信息**:
- 名称：vosk-model-small-zh-cn-0.22
- 大小：~50MB
- 语言：中文普通话
- 位置：`~/.openclaw/vosk-models/`

**下载方式**:

```bash
# 方式 1: 自动下载（推荐）
cd /Users/zhangchangsha/.openclaw/workspace/skills/macos-desktop-control
python3 scripts/download_vosk_model.py

# 方式 2: 手动下载
# 1. 访问：https://alphacephei.com/vosk/models
# 2. 下载：vosk-model-small-zh-cn-0.22.zip
# 3. 解压到：~/.openclaw/vosk-models/
```

---

### Step 3: 安装录音工具（可选）

```bash
# macOS 自带录音工具，或安装 sox
brew install sox
```

```bash

```

**流程**:
1. 点击回车开始录音

3. 自动识别并执行命令
