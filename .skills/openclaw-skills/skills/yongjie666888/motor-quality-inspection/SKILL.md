# 军用电机质量检验助手

根据 GJB 军工体系标准，生成军用电机产品质量检验规程，覆盖来料、过程、成品、出厂全流程。

## 检验标准体系

```
GJB9001C-2017   质量管理体系要求（基础）
GJB179A         计数抽样检验程序
GJB360B         电子电气元件试验方法
GJB1184         永磁同步电动机通用规范
GJB2153         军用电机通用规范
GJB423          装备环境试验通用要求
```

## 检验流程总览

```
来料检验（IQC）
    ↓
过程检验（PQC）
    ↓
成品检验（FQC）
    ↓
出厂检验（OQC）
    ↓
质量文件归档
```

## 一、来料检验（IQC）

### 1.1 磁性材料检验

| 检验项目 | 标准 | 方法 | AQL |
|---------|------|------|-----|
| 硅钢片厚度 | 0.35±0.02mm | 千分尺 | II级 |
| 硅钢片表面涂层 | 均匀无脱落 | 目测 | I级 |
| 磁通密度 | ≥1.5T @ 5000A/m | 爱泼斯坦仪 | S-4 |
| 磁钢牌号 Br | 符合图样 | 退磁曲线测试 | S-3 |
| 磁钢尺寸 | ±0.05mm | 卡尺 | II级 |

### 1.2 绝缘材料检验

| 检验项目 | 标准 | 方法 | AQL |
|---------|------|------|-----|
| 槽绝缘厚度 | 0.25±0.02mm | 千分尺 | II级 |
| 耐温等级 | F级(155℃) | 热老化试验 | S-3 |
| 击穿电压 | ≥6kV/mm | 耐压仪 | S-4 |

### 1.3 轴承检验

| 检验项目 | 标准 | 方法 |
|---------|------|------|
| 外观 | 无划伤、无锈蚀 | 目测 |
| 游隙 | 符合C3组 | 塞规 |
| 旋转灵活性 | 无异响 | 手动旋转 |
| 防锈油脂 | 适量、无变色 | 目测 |

### 1.4 电磁线检验

| 检验项目 | 标准 | 方法 |
|---------|------|------|
| 线径 | ±0.01mm | 微千分尺 |
| 绝缘厚度 | 符合QZ-2标准 | 剥除外被测量 |
| 耐电压 | 2000V/1min不击穿 | 耐压仪 |
| 软化延伸率 | ≥20% | 延伸试验 |

## 二、过程检验（PQC）

### 2.1 定子检验

| 检验项目 | 要求 | 检验时机 |
|---------|------|---------|
| 槽满率 | 45%~55% | 嵌线后 |
| 直流电阻 | 三相不平衡≤2% | 嵌线后 |
| 绝缘电阻 | ≥100MΩ | 浸漆前/后 |
| 匝间绝缘 | 无击穿 | 浸漆前 |
| 相间绝缘 | 无短路 | 浸漆前 |

### 2.2 转子检验

| 检验项目 | 要求 | 检验时机 |
|---------|------|---------|
| 动平衡 | G2.5级 | 磁钢安装后 |
| 轴向跳动 | ≤0.02mm | 压装后 |
| 径向跳动 | ≤0.03mm | 压装后 |

### 2.3 整机装配检验

| 检验项目 | 要求 | 方法 |
|---------|------|------|
| 气隙均匀性 | 偏差≤10%均值 | 气隙规 |
| 端盖同轴度 | ≤0.03mm | 同轴度仪 |
| 螺栓力矩 | 符合工艺文件 | 力矩扳手 |
| 轴承游隙 | 转动灵活无卡滞 | 手动 |

## 三、成品检验（FQC）

### 3.1 电气性能检验

| 检验项目 | 标准要求 | 检验方法 |
|---------|---------|---------|
| 空载电流 | ≤额定电流30% | 功率分析仪 |
| 空载转速 | 额定转速±5% | 光电转速仪 |
| 绝缘电阻 | ≥100MΩ | 500V兆欧表 |
| 耐电压 | 1500V/1mA/1min | 耐压仪 |
| 接地电阻 | ≤0.1Ω | 接地电阻仪 |
| 振动 | ≤2.8mm/s | 振动仪 |
| 噪音 | ≤65dB(A) | 声级计 |

### 3.2 环境试验（按GJB423）

| 试验项目 | 条件 | 检验时机 |
|---------|------|---------|
| 高温试验 | 85℃×2h | 首批/定期 |
| 低温试验 | -40℃×2h | 首批/定期 |
| 湿热试验 | 40℃×95%RH×48h | 首批/定期 |
| 振动试验 | GJB360B方法 | 首批 |
| 冲击试验 | 50g/11ms | 首批 |

## 四、出厂检验（OQC）

### 4.1 检验项目清单

```
□ 外观检查：无损伤、无锈蚀、标识清晰
□ 型号规格核对：与合同/技术协议一致
□ 绝缘电阻：≥100MΩ
□ 耐压：1500V/1mA/1min无击穿
□ 空载运行：30min，无异响、无振动
□ 负载运行：额定负载30min，温升正常
□ 噪声：≤65dB(A)
□ 振动：≤2.8mm/s
□ 随机文件：检验报告、合格证、说明书
```

### 4.2 抽样方案（按GJB179A）

| 检验级别 | 批量范围 | 样本量 | 判定数组 |
|---------|---------|--------|---------|
| S-3 | 2~50 | 3 | [0,1] |
| S-4 | 51~500 | 13 | [0,1] |
| II | 501~3200 | 50 | [1,2] |
| III | 3201~35000 | 125 | [2,3] |

## 六、不合格品处理

| 不合格等级 | 处理方式 |
|-----------|---------|
| 致命不合格（安全问题）| 返工、报废，禁止出厂 |
| 严重不合格（功能失效）| 返工，重新检验 |
| 一般不合格（外观等）| 返工或降级使用 |
| 轻微不合格（轻微缺陷）| 让步接收或返工 |

## 检验报告生成脚本

```bash
# 列出所有检验类型
python scripts/inspection_report.py --list

# 来料检验报告
python scripts/inspection_report.py --type iqc --supplier "某某磁材" --batch "B2024001"

# 过程检验报告
python scripts/inspection_report.py --type pqc --model "PMSM-1kW" --sn "20240015"

# 成品检验报告
python scripts/inspection_report.py --type fqc --model "BLDC-200W" --sn "20240020"

# 出厂检验报告（最常用）
python scripts/inspection_report.py --type oqc --model "PMSM-1kW" --sn "20240020" \
    --output oqc_report_20240020.txt
```

### 支持的检验类型

| 类型 | 标题 | 适用场景 |
|------|------|---------|
| iqc | 来料检验报告 | 磁性材料、绝缘材料、轴承、电磁线入料 |
| pqc | 过程检验报告 | 定子、转子、装配过程 |
| fqc | 成品检验报告 | 电气性能、机械性能、整机性能 |
| oqc | 出厂检验报告 | 出货前最终检查 |
