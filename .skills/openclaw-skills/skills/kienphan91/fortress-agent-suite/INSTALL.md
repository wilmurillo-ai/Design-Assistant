# Hướng dẫn cài đặt Fortress Suite
1. Clone repo này vào thư mục `skills/` trong workspace OpenClaw của bạn.
2. Đảm bảo bạn đã cài đặt `psutil` (pip install psutil).
3. Copy nội dung file `crontab_template.txt` vào crontab của hệ thống (nhớ thay đổi `/path/to/fortress` bằng đường dẫn thực tế).
4. Hệ thống sẽ tự động giám sát và bảo vệ agent của bạn 24/7.
