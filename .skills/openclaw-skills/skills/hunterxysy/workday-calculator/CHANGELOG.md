# Changelog

所有对 Workday Calculator Skill 的更改都将记录在此文件中。

## [1.0.0] - 2026-03-12

### 新增
- 🎉 初始版本发布
- 📅 工作日计算核心功能
- 🇨🇳 中国节假日支持（2025-2026年）
- 🔄 调休工作日支持
- 📊 详细信息查看功能
- 💾 JSON导出功能
- 🛠️ 节假日更新工具
- 📋 使用示例脚本
- 📖 完整文档
- 🔧 安装脚本

### 功能
- 计算任意两个日期之间的工作日数量
- 自动排除周末（周六、周日）
- 自动排除中国法定节假日
- 支持调休工作日（周末上班）
- 显示工作日、节假日、周末详细列表
- 导出结果为JSON格式
- 交互式节假日数据更新
- 命令行界面和Python模块两种使用方式

### 技术特性
- 纯Python实现，无外部依赖
- 支持Python 3.6+
- 完整的错误处理
- 友好的用户界面
- 可扩展的架构

### 文档
- 详细的SKILL.md技能说明
- 完整的README.md使用文档
- 安装脚本和配置
- 使用示例和最佳实践

## 版本说明

### 版本命名规则
- `主版本.次版本.修订版本`
- 主版本：重大功能更新或架构改变
- 次版本：新功能添加，向后兼容
- 修订版本：bug修复和小改进

### 兼容性
- v1.x.x：保持API和功能兼容性
- 重大变更将在v2.0.0中引入
