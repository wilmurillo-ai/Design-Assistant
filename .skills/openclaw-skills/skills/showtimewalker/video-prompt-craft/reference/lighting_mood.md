# 光影氛围词汇库

## 光线类型

| 关键词 | 效果 | 适用场景 |
|--------|------|---------|
| `golden hour light` | 暖金色柔光，温馨浪漫 | 日出日落、户外人像、旅行 |
| `blue hour` | 冷蓝色调，宁静 | 城市夜景、黄昏海景 |
| `backlight` | 逆光轮廓，戏剧感 | 人像剪影、烟雾/水雾场景 |
| `sidelight` | 侧光，立体感强 | 建筑雕塑、人物肖像 |
| `overcast / diffused` | 阴天散射光，柔和均匀 | 自然生态、室内场景 |
| `neon / neon-lit` | 霓虹灯光，赛博朋克 | 城市夜景、科技感 |
| `volumetric lighting` | 体积光/丁达尔效应 | 森林、教堂、舞台 |
| `rim light` | 边缘光，勾勒轮廓 | 人物、产品 |
| `practical lighting` | 场景内光源（台灯、蜡烛等） | 室内叙事、情绪场景 |

## 电影质感关键词

```
cinematic 35mm film          — 35mm 胶片质感
anamorphic lens              — 变形宽银幕镜头
lens flare                   — 镜头光晕
dreamy bokeh                 — 梦幻散景
high dynamic range / HDR     — 高动态范围
color graded                 — 调色后质感
film grain                   — 胶片颗粒感
soft focus                   — 柔焦
motion blur                  — 动态模糊
depth of field / DoF         — 景深效果
```

## 抽象情绪 → 可视化描述

AI 无法渲染抽象情绪，必须转化为具体画面：

| 抽象 | 可视化 |
|------|--------|
| 悲伤 | `a person with tears streaming down their face, looking down, rain falling on their shoulders` |
| 孤独 | `a single figure standing in an empty street, long shadow stretching behind them` |
| 希望 | `sunlight breaking through dark clouds, a hand reaching towards the light` |
| 紧张 | `shallow depth of field, a character glancing nervously over their shoulder` |
| 温暖 | `soft golden light, two people sharing a laugh, steam rising from coffee cups` |
| 神秘 | `low-key lighting, shadows obscuring half the face, fog swirling around the scene` |
