# Changelog

所有重要的项目变更都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [2.0.0] - 2026-03-16

### ✨ 新增功能
- **进度条显示**: 添加 tqdm 进度条，实时显示处理进度
- **并发处理**: 支持多线程并发处理 Issue，速度提升 3-5 倍
- **智能重试**: 网络请求失败自动重试，最多 3 次
- **结果缓存**: 缓存已处理的 Issue，避免重复处理
- **增强日志**: 结构化日志输出，支持 JSON 格式
- **性能报告**: 生成详细的性能分析报告

### 🐛 修复
- 修复 LLM API 调用超时问题，增加超时设置
- 修复重复检测准确率低的问题，优化相似度算法
- 修复标签添加失败时没有错误提示的问题
- 修复并发处理时的竞态条件

### ⚡ 性能优化
- 优化 GitHub API 调用，减少不必要的请求
- 使用连接池复用 HTTP 连接
- 批量操作替代单个操作，减少 API 调用次数
- 内存优化，支持处理大量 Issue

### 📚 文档
- 添加详细的 API 文档
- 添加更多使用示例
- 添加故障排查指南
- 添加性能调优建议

### 🧪 测试
- 添加单元测试（覆盖率 > 80%）
- 添加集成测试
- 添加性能基准测试
- 添加 Mock 测试

### 🔒 安全
- 增强 Token 安全管理
- 添加敏感信息脱敏
- 支持只读模式
