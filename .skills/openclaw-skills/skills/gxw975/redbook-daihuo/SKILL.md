# 小红书宠物带货笔记

## 技能功能

本技能用于批量生成小红书宠物带货笔记，适用于宠物用品商家、宠物博主等需要大量高质量内容的场景。

## 使用流程

### 1. 上传产品知识库
- 上传产品基本信息文档，包含：
  - 产品成分
  - 原料来源
  - 成分功效
  - 原料纯度

### 2. 输入变量信息

**人称视角：**
- 第一人称
- 第二人称
- 第三人称
- 混合视角

**产品信息：**
- 产品功效（包括每个功效对应的产品成分，因缺乏该成分会出现的症状）
- 产品使用方法
- 产品适口性
- 包装特点
- 适用的宠物种类

**宠物主信息：**
- 性别（男或女）
- 年龄（18岁、19岁、20岁、21岁、22岁、23岁、24岁、25岁、26岁、27岁、28岁、29岁、30岁、31岁、32岁、33岁、34岁、35岁、36岁、37岁、38岁、39岁，每个年龄段作为单独变量）

**宠物信息：**
- 品种（猫或狗）
- 问题和症状（每个问题和症状作为单独变量）

**场景信息：**
- 时间（早上、中午、晚上、昨天、前几天、几个月前）
- 地点（家、宿舍、公园、商场、小区楼下）
- 宠物主的情绪（恐惧、焦虑、不知所措、愤怒、讨厌、满足、放松）

**使用效果：**
- 宠物问题和症状的改善情况
- 宠物健康状态
- 宠物主情绪变化

**活动变量：**
- 近期促销活动内容
- 免费提供的资料

### 3. 生成文案
- 系统将上述所有变量进行随机组合，包括人称视角、时间地点、宠物主情绪、年龄性别等
- 每一组变量组合生成一条笔记
- 每条笔记内容整体完整，前后逻辑自洽，表达流畅自然，符合小红书带货笔记的要求
- 每篇文案包含以下结构：
  1. **吸引注意**：通过养宠故事，对比使用前后状态
  2. **传递价值**：从成分功能入手，展示核心卖点
  3. **引导转化**：明确行动号召，增强社交证明
- 支持多种人称视角创作：第一人称、第二人称、第三人称、混合视角
- 时间变量随机组合当天时间段与日期时间段
- 地点变量随机选择不同场景
- 宠物问题和症状作为单独变量进行组合
- 宠物主情绪作为单独变量进行组合
- 年龄与性别结合形成与宠物主身份特征一致的笔记文字风格

### 4. 生成图片
- 将文案分割成5-10段
- 每段提取核心关键语句
- 生成对应图片，保持人物、宠物和风格的一致性
- 确保图片真实感强，贴近生活
