## Search Strategies

### 1. Smart Judgment on Whether to Search

**Scenarios requiring search:**
- Technical, professional topics
- Viewpoints needing data support
- Current hot topics and latest trends
- Content needing authority enhancement
- Unfamiliar fields

**Scenarios not requiring search:**
- Pure personal experience and reflections
- Creative writing and imaginative content
- User explicitly states no search needed
- Extremely simple and common-sense content

### 2. Keyword Extraction

Extract from user input:
```
Core concept + Domain qualifiers + Time qualifiers

For example:
"WebAssembly development prospects"
→ Core concept: WebAssembly
→ Domain qualifier: Web development
→ Time qualifier: 2024/2025 trends
```

### 3. Search Query Construction

**Multi-angle search:**

| Angle | Query Example |
|-------|--------------|
| Basic information | "What is [topic]" |
| Latest trends | "[topic] 2024/2025 trends" |
| Data research | "[topic] research/study" |
| Case practice | "[topic] case study/examples" |
| Problem solving | "how to [verb] [topic]" |
| Comparison analysis | "[topic A] vs [topic B]" |

### 4. Source Screening Criteria

**High-quality source features:**
- ✓ Authoritative institution release
- ✓ Clear author
- ✓ Provides data/citation sources
- ✓ Deep and professional content
- ✓ Recent publication time

**Low-quality source features:**
- ✗ Content farms
- ✗ No author information
- ✗ Pure advertising content
- ✗ Outdated information
- ✗ Clickbait titles

### 5. Information Integration Methods

**Data citation:**
```
According to [Source]'s research, [specific data].

For example: According to McKinsey's 2024 report, AI can improve enterprise efficiency by 40%.
```

**Case supplement:**
```
[Company/Person]'s case illustrates [viewpoint].

For example: Netflix's recommendation system case illustrates machine learning application in user experience.
```

**Comparison analysis:**
```
[Viewpoint A], but [Viewpoint B] provides a different perspective.

For example: Some believe AI will replace jobs, but World Bank research shows AI mostly enhances rather than replaces.
```

**Trend citation:**
```
Industry trends show, [trend description].

For example: Gartner predicts that by 2025, 50% of enterprises will adopt cloud-native architecture.
```

## Integrating Information into Posts

### Integration Strategy

1. **Opening citation**: Use shocking data/research at the beginning to attract attention
2. **Viewpoint support**: Use research data to support your views
3. **Case enrichment**: Use real cases to enhance persuasiveness
4. **Trend outlook**: Combine industry trends to enhance foresight

### Reference Material Format

Attach reference materials at the end of the post:

```markdown
---
**Reference Materials:**

1. [Paper/Report Title], [Author/Organization], [Year]
2. [Article Title], [Media Name], [Date]
3. [Document Name], [Official Source]
```

### Citation Considerations

- Don't over-cite, moderate amount is sufficient
- Citations should be accurate, don't distort original meaning
- Clearly mark sources
- Prioritize latest sources (within 1-2 years)
- Diversify sources, don't just cite one
