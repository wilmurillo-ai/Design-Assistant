# Post Structure Guide

This document provides detailed explanations of post structures and writing techniques for different platforms.

---

## Universal Post Structure

Basic elements that all platform posts should include:

```
┌─────────────────────────────────────┐
│           Title                     │  ← Engaging title
├─────────────────────────────────────┤
│           Hook                      │  ← Attention-grabbing opening
├─────────────────────────────────────┤
│                                     │
│          Body                       │  ← Core content, expand by framework
│                                     │
├─────────────────────────────────────┤
│         CTA/Conclusion              │  ← Prompt action or summary
├─────────────────────────────────────┤
│           Tags                      │  ← Relevant topic tags
└─────────────────────────────────────┘
```

## WeChat Official Account

### Platform Characteristics

- **Word count limit**: 2000+ words, in-depth long articles
- **Reading scenario**: Immersive reading, high focus
- **Reader expectations**: Depth, practical value, clear structure
- **Distribution method**: Moments sharing, community sharing

### Structure Template

```markdown
# [Title] Main Title | Subtitle

**Summary/Lead**: One sentence to summarize core viewpoint, spark reading interest

---

## 1. Engaging Opening

Use a story, data point, or question to grab readers

> "Did you know, 90% of people..."

## 2. Core Viewpoint

### 2.1 First Point

**Point Explanation**

Specific explanation, supported by data or cases

【Case/Data】Specific case demonstration

### 2.2 Second Point

...

## 3. In-depth Analysis

### 3.1 Analysis Dimension One

...

### 3.2 Analysis Dimension Two

...

## 4. Practical Advice/Methodology

### Step One: xxxx

Specific operation method

### Step Two: xxxx

...

## 5. Summary and Outlook

Review core viewpoint, look to future

---

**【Call to Action】**

If you found this helpful, welcome to like and forward

Click the link below to learn more: [link]

---

## Recommended Reading

- [Related Article 1](link)
- [Related Article 2](link)
```

### Writing Techniques

1. **Title techniques**
   - Number method: "7 xxx Tips"
   - Question method: "Why xxx?"
   - Contrast method: "From xxx to xxx"
   - Golden sentence method: "xxx: One Sentence Summary"

2. **Formatting techniques**
   - Use multiple heading levels (# ## ###)
   - Bold important content
   - Use quote blocks (>) for golden sentences
   - Use horizontal rules (---) appropriately

3. **Image suggestions**
   - One attractive cover image at the beginning
   - Relevant illustration for each section
   - Use charts for data information
   - Call to action image at the end

4. **Structure techniques**
   - One core viewpoint throughout
   - Subheading for each part
   - No more than 5 lines per paragraph
   - Use "introduction-body-conclusion" structure

## Xiaohongshu

### Platform Characteristics

- **Word count limit**: 500-1000 words, practical content
- **Reading scenario**: Quick browsing, fragmented reading
- **Reader expectations**: Practical, actionable, valuable
- **Distribution method**: Algorithm recommendation, search discovery

### Structure Template

```markdown
【Title】🔥Shock/Must-See/Dry Goods! N Methods for xxx

One eye-catching cover image

---

📌 Intro: One sentence pain point + solution

Have you also encountered xxx situation? Today I'm sharing my xxx method, personally tested and effective!

---

✨ Method One: xxx

👉 Specific operation:
1️⃣ First step
2️⃣ Second step
3️⃣ Third step

💡 Tip: Pay attention to xxxx

---

✨ Method Two: xxx

👉 Specific operation:
...

---

📊 Results display:

Before using: xxx
After using: xxx

---

💡 Summary/Core Points

1. Point one
2. Point two
3. Point three

---

🔗 Related recommendations

#topic1 #topic2 #topic3

💬 See you in the comments! Questions welcome~

---

【Save for later】
【Like to help more people see】
```

### Writing Techniques

1. **Title techniques**
   - Add emoji to attract attention 🔥
   - Use words like "shock", "must-see"
   - Emphasize practical value
   - Numbers attract: "5 xxx Tips"

2. **Emoji usage**
   - ✨ Methods/tips
   - 👉 Operation steps
   - 💡 Tips/key points
   - 📊 Data/comparison
   - 🔥 Hot/important
   - 💬 Interaction guidance

3. **Formatting techniques**
   - Separate methods with horizontal rules
   - Bold keywords
   - Use numbered lists for steps
   - Ample white space, not dense

4. **Content points**
   - Give methods directly, less fluff
   - Strong actionability
   - Include before/after comparison images
   - Emphasize results

5. **Tag strategy**
   - 3-5 relevant topic tags
   - Include hot tags
   - Include specific tags
   - Place tags at the end

## Weibo/Twitter

### Platform Characteristics

- **Word count limit**: 140-280 characters, extremely concise
- **Reading scenario**: Quick scroll, extremely short attention
- **Reader expectations**: Quick information, one-sentence essence
- **Distribution method**: Retweets, likes, topic aggregation

### Structure Template

```
【Golden sentence type】
One-sentence core viewpoint + Short explanation + Topic tags

【Data type】
Shocking data + One-sentence interpretation + Topic tags

【Viewpoint type】
Viewpoint + Reason + Conclusion + Topic tags

【Question type】
Question + Reflection + Open ending + Topic tags

【Action type】
Pain point + Solution + Call to action + Topic tags
```

### Specific Examples

```markdown
# Golden sentence type

90% of people don't know the efficiency secret: Not doing more, but doing less.
Focus on the critical 20%, produce 80% of value.
#efficiency #productivity

---

# Data type

Shocking! Research found: People who work 4 hours a day produce 30% more than those who work 8 hours.
Quality over quantity.
#workEfficiency #research

---

# Viewpoint type

I believe remote work is the future trend.
Reason: Lower costs, higher efficiency, larger talent pool.
Suggestion: Companies should embrace, not resist.
#remoteWork #workStyle

---

# Question type

Why does more effort equal more anxiety?
Perhaps we're trying in the wrong direction.
What do you think?
#thinking #life

---

# Action type

Still learning inefficiently?
Try Feynman technique: Teaching others = Learning yourself.
Start today!
#learningMethod #feynmanTechnique
```

### Writing Techniques

1. **Opening grab attention**
   - Start with numbers
   - Start with questions
   - Start with counter-intuitive viewpoints
   - Start with pain points

2. **One-sentence core**
   - One core viewpoint per tweet
   - One sentence if possible, not two
   - Delete all redundant words

3. **Golden sentence expression**
   - Contrast: From xxx to xxx
   - Parallelism: xxx, xxx, xxx
   - Metaphor: xxx is like xxx
   - Aphorism: xxx is not xxx, but xxx

4. **Topic tags**
   - 1-3 relevant tags
   - Tags at the end
   - Include hot topics
   - Include niche topics

5. **Image strategy**
   - One image worth a thousand words
   - Infographics for data
   - Golden sentence cards
   - Before/after comparison images

## LinkedIn/Maimai

### Platform Characteristics

- **Word count limit**: 1000-1500 words, professional workplace
- **Reading scenario**: Professional development time, relatively focused
- **Reader expectations**: Professional insights, industry perspectives, workplace experience
- **Distribution method**: Professional network sharing, algorithm recommendation

### Structure Template

```markdown
【Title】Professional, concise title

One professional image (optional)

---

## Opening: Professional background/Industry status

Briefly describe current industry situation or personal career experience

> "Working in xxx industry for 10 years, I found..."

## Core Viewpoint/Insight

### First Point

**Viewpoint elaboration**

Explain viewpoint, provide professional insights

【Case】Relevant professional case or industry case

### Second Point

...

## Practical Advice/Methodology

Specific advice for workplace professionals

1. Advice 1: Specific explanation
2. Advice 2: Specific explanation
3. Advice 3: Specific explanation

## Personal Reflection/Thinking

Share personal career reflections

> "Looking back at my career path, I think..."

## Conclusion + Call to Action

Summarize points, invite discussion

"What do you think about this? Welcome to share your views in the comments"

---

#industryTopic #careerDevelopment #professionalField
```

### Writing Techniques

1. **Title techniques**
   - Professional terminology: Like "xxx Best Practices"
   - Industry hot topics: Combine current industry trends
   - Experience summary: "What I Learned in xxx Years"
   - Method sharing: "How to xxx"

2. **Professional expression**
   - Use industry terminology
   - Cite data and reports
   - Mention well-known companies/cases
   - Show professional thinking

3. **Clear structure**
   - Use subheadings to separate content
   - Use numbered lists for points
   - Use quote blocks for viewpoints
   - Tight logic

4. **Workplace related**
   - Combine career experience
   - Share industry insights
   - Provide workplace advice
   - Trigger professional thinking

5. **Interaction guidance**
   - End with questions to spark discussion
   - Encourage sharing experiences
   - Solicit opinions and suggestions
   - Build professional connections

## Cross-Platform Adaptation Strategy

### Same Content, Multi-Platform Publishing

```
WeChat OA (Long article)
      ↓ Condense
   Xiaohongshu (Practical version)
      ↓ Extract
   Weibo (Golden sentence version)
      ↓ Adapt
 LinkedIn (Professional version)
```

### Adaptation Principles

| Original Content | WeChat OA | Xiaohongshu | Weibo | LinkedIn |
|-----------------|-----------|-------------|-------|----------|
| Deep analysis | ✓Complete | ✓Condensed | ✗Delete | ✓Keep |
| Case stories | ✓Complete | ✓Brief | ✗Delete | ✓Keep |
| Data citations | ✓Detailed | ✓Brief | ✓One sentence | ✓Cite |
| Operation steps | ✓Detailed | ✓Complete | ✗Delete | ✓Keep |
| Personal views | ✓Complete | ✓Keep | ✓Core | ✓Professional |
| Golden sentences | ✓Keep | ✓Keep | ✓Main focus | ✓Keep |

### Content Layering

```
First layer: Core viewpoint (all platforms)
    ↓
Second layer: Supporting reasons (WeChat OA, LinkedIn, Xiaohongshu)
    ↓
Third layer: Detailed cases (WeChat OA, LinkedIn)
    ↓
Fourth layer: Operation methods (WeChat OA, Xiaohongshu)
```

## Universal Writing Techniques

### Title Formulas

1. **Number method**: "N xxx Tips"
2. **Question method**: "How to xxx?" "Why xxx?"
3. **Contrast method**: "From xxx to xxx" "xxx vs xxx"
4. **Secret method**: "xxx's Secret" "Little-known xxx"
5. **Warning method**: "Beware of xxx" "xxx Misconceptions"
6. **Promise method**: "Master xxx in 30 Days" "Let You xxx"

### Opening Techniques

1. **Story opening**: "I encountered a problem..."
2. **Data opening**: "90% of people..."
3. **Question opening**: "Have you ever..."
4. **Golden sentence opening**: "xxx is not xxx, but xxx..."
5. **Pain point opening**: "Are you also..."

### Ending Techniques

1. **Summary review**: "In summary..."
2. **Call to action**: "Do it now..."
3. **Open question**: "What do you think..."
4. **Future outlook**: "In the future..."
5. **Golden sentence elevation**: "Remember, xxx..."

### Formatting Principles

1. **Ample white space**: Don't stack densely
2. **Clear hierarchy**: Separate with headings
3. **Highlight key points**: Bold/quotes
4. **Visual guidance**: Use symbols/emoji
5. **Short paragraphs**: 3-5 lines per paragraph
