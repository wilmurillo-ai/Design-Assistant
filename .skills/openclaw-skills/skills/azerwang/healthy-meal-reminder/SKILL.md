# 健康饮食提醒

全能饮食管理：ABC三选一推荐 → 饭后30分钟跟进记录 → 热量追踪 → 周报打卡。

## 理念

- **吃饱不吃撑，适可而止**
- **只喝不吃，馋了就喝水**
- **睡前3小时不进食**
- **不时不食，吃应季的**
- **三分练七分吃**

---

## ⭐ 核心交互流程（v1.8 重点）

### 第一步：推荐（定时触发）

每个饭点触发时，从对应季节食谱中选 **3个方案**，标注 A/B/C，格式如下：

```
🍽️ 午餐时间到！选一个吧～

A. 荠菜鲜肉馄饨清汤10个 ~350 kcal
   高蛋白低脂，春笋正当季 🌿

B. 春笋炒肉+糙米饭半碗 ~420 kcal  
   纤维丰富，饱腹感强 💪

C. 蒸时蔬拼盘（芦笋+山药+木耳）~150 kcal
   今天想轻断？选这个没错 ✨

⏰ 30分钟后我会来问问你吃了什么哦～
```

**规则：**
- 固定3个选项，分别标注 A / B / C
- 每个方案附：菜品名 + 热量 + 一句话亮点
- C 选项始终提供一个低卡/轻食选择
- 告知用户30分钟后会跟进

### 第二步：跟进记录（推荐后30分钟自动触发）

使用 OpenClaw cron 延迟提醒，在每次正餐推荐30分钟后自动询问：

**跟进 prompt 模板：**
```
你是饮食记录助手。30分钟前给用户推荐了【XX餐】。
请简短地询问用户：
"吃得怎么样？告诉我你吃了什么就行～
可以回复 A/B/C 选推荐，也可以说吃了别的。"
语气轻松，不要催促。如果用户不回复就算了。
```

**Cron 设置（每餐各一个延迟任务）：**
- 早餐推荐 7:00 → 跟进 7:30
- 午餐推荐 12:00 → 跟进 12:30
- 晚餐推荐 18:30 → 跟进 19:00

### 第三步：记录饮食（用户回复后）

**场景1：用户选了 A/B/C**
```
回复示例：
"👍 选A！荠菜鲜肉馄饨，好选择～
已记录：午餐 350 kcal
今天累计：XXX kcal（目标：1200-1500 kcal）"
```

**场景2：用户说了具体吃了什么**
```
回复示例：
"📝 已记录：午餐 黄焖鸡米饭 ~600 kcal
⚠️ 热量偏高，碳水较多
💡 晚餐建议选清淡的，比如：
A. 冬瓜关东煮 ~250 kcal
B. 蒸时蔬拼盘 ~150 kcal  
C. 今天不吃了！0 kcal"

今天累计：XXX kcal（目标：1200-1500 kcal）"
```

**场景3：用户说没吃/跳过**
```
回复示例：
"👌 了解，这顿跳过～
别饿着自己，下午茶时间可以喝杯茶
今天累计：XXX kcal"
```

**记录格式（存入当日记忆）：**
```
## 饮食记录 YYYY-MM-DD
- 早餐(07:XX): A.水煮蛋+豆浆 ~220 kcal / 或：XXX ~XX kcal / 或：未吃
- 午餐(12:XX): B.春笋炒肉 ~420 kcal / 或：黄焖鸡 ~600 kcal / 或：未吃
- 下午茶(15:XX): 黑咖啡 0 kcal / 或：奶茶 ~400 kcal / 或：未吃
- 晚餐(18:XX): C.时蔬拼盘 ~150 kcal / 或：未吃
---
当日总摄入: XXXX kcal | 目标: 1200-1500 kcal
```

## 目标模式

首次使用时确认，后续可随时切换：

### 🏋️ 减肥模式（默认）
- 每日总热量：1200-1500 kcal
- 晚餐可选不吃
- 下午茶只喝不吃
- 跟进时显示剩余可摄入量

### ⚖️ 维持模式
- 每日总热量：1600-2000 kcal
- 三餐正常 + 下午茶可选零食

### 💪 增肌模式
- 每日总热量：2000-2500 kcal
- 增加蛋白质，加餐可选鸡胸肉/蛋白粉/香蕉

## 季节食谱

### 🌸 春季（2-4月）| 养肝舒气

**早餐（7:00）推荐组合池：**
- A类（均衡）: 水煮蛋×2+无糖豆浆 ~220kcal / 香椿炒蛋+无糖豆浆 ~260kcal / 菠菜猪肝粥+水煮蛋 ~280kcal
- B类（饱腹）: 燕麦30g+牛奶250ml ~250kcal / 荠菜豆腐羹+全麦馒头半个 ~200kcal / 韭菜盒子小个+黑咖啡 ~240kcal
- C类（轻食）: 全麦面包1片+牛油果半个 ~200kcal / 红薯半个+无糖豆浆 ~160kcal / 小米粥+凉拌黄瓜 ~180kcal

**午餐（12:00）推荐组合池：**
- A类（荤素搭配）: 荠菜鲜肉馄饨清汤10个 ~350kcal / 春笋炒肉+糙米饭半碗 ~420kcal / 豌豆苗鸡肉卷 ~350kcal
- B类（海鲜）: 清蒸鲈鱼半条+白灼菜心 ~300kcal / 蒜蓉西兰花+白灼虾8只+杂粮饭 ~400kcal / 蒜泥白肉+凉拌木耳+糙米饭 ~380kcal
- C类（轻食）: 轻食沙拉鸡胸肉+油醋汁 ~250kcal / 蒸时蔬拼盘芦笋+山药+木耳 ~150kcal / 荠菜豆腐汤+全麦馒头 ~220kcal

**下午茶（15:00）推荐组合池：**
- A类（茶饮）: 龙井/碧螺春 0kcal / 菊花枸杞茶 0kcal / 柠檬水 5kcal
- B类（水果）: 小番茄5-6颗 ~35kcal / 草莓6-8颗 ~50kcal / 蓝莓一小碗 ~40kcal
- C类（零食）: 99%黑巧1小块 ~25kcal / 坚果3颗核桃 ~50kcal / 不吃！喝水！ 0kcal

**晚餐（18:30）推荐组合池：**
- A类（汤类）: 春笋排骨汤少油 ~300kcal / 荠菜豆腐汤+凉拌蒲公英 ~180kcal / 枸杞叶猪肝汤小碗 ~200kcal
- B类（家常）: 韭菜炒蛋+小米粥 ~280kcal / 蒸南瓜+白灼西兰花 ~220kcal / 芦笋炒虾仁 ~260kcal
- C类（轻食/不吃）: 蒸时蔬拼盘 ~150kcal / 红薯半个+白灼菜心 ~180kcal / 今天不吃了！0kcal

## 运动搭配

| 时间段 | 推荐 | 说明 |
|--------|------|------|
| 早餐后30min | 晨间拉伸5min / 散步15min | 唤醒身体 |
| 午餐后 | 靠墙站10min / 慢走20min | 防止脂肪堆积 |
| 下午茶后 | 深蹲10个 / 转腰 | 缓解久坐 |
| 晚餐后1h | 快走30min / 慢跑20min | 最佳燃脂时段 |
| 睡前 | 瑜伽10min / 冥想5min | 助眠放松 |

## 周末放纵餐（Cheat Day）

周六或周日选一顿放纵：
- **火锅** 鸳鸯锅多涮菜 ~600-800kcal
- **烧烤** 翅膀2个+烤玉米 ~500-700kcal
- **日料** 三文鱼刺身+味噌汤 ~600-800kcal
- **奶茶** 一杯正常糖 ~400-500kcal（一周一次不过分）
- 原则：一周放纵**一顿**，不是一天；第二天恢复正常即可

## 周报打卡

周日21:00自动提醒，询问：
1. 本周体重变化
2. 坚持天数（1-7天）
3. 最满意一餐
4. 下周目标

生成简短周报，连续4周可出月度趋势。

## 自定义

- **切换模式**：说"我改减肥/维持/增肌模式"
- **跳过某餐**：说"今天不吃XX"
- **记录食物**：说"中午吃了XX"即可
- **过敏/忌口**：首次告知，后续自动避开
- **查看记录**：说"看看今天的饮食记录"

## 注意事项

- 孕妇、哺乳期、糖尿病患者请咨询医生
- 每日摄入不低于基础代谢
- 语气温暖有趣，不唠叨
- 跟进询问不要催促，用户不回就算了
- 节气日期大致参考即可
