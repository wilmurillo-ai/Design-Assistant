# 合同审查助手 — 操作说明

> ClawHub 地址：clawhub.ai/skills/ai-contract-reviewer
> 安装命令：`openclaw skills install ai-contract-reviewer`

---

## 这个 skill 是干什么的

上传任意合同文件，AI 逐条扫描风险条款，用大白话解释每个风险点，并给出具体修改建议。适合签合同前做快速风险排查。

> ⚠️ AI 分析仅供参考，不构成法律建议。重要合同请咨询专业律师。

---

## 安装步骤

1. 打开 OpenClaw
2. 输入：
```
openclaw skills install ai-contract-reviewer
```
3. 安装完成，无需额外配置

---

## 支持的文件格式

- `.pdf` — 最常见
- `.docx` — Word 文档
- `.txt` — 纯文本
- 直接粘贴合同文本（速度最快）

## 适合审查哪些合同

| 合同类型 | 适用人群 |
|---------|---------|
| 劳动合同 / 聘用协议 | 求职者、HR |
| NDA 保密协议 | 自由职业者、创业者 |
| 服务合同 / 合作协议 | 外包接单者、乙方 |
| 房屋租赁合同 | 租房者 |
| SaaS 服务条款 | 企业采购 |
| 投资协议 | 创业者 |

## 基本用法

### 方式一：上传文件
1. 在 OpenClaw 上传合同文件
2. 说：`帮我审查这份合同，找出所有风险条款`

### 方式二：粘贴文本
复制合同内容粘贴到对话框，说：
```
我是乙方，帮我从乙方角度找出对我不利的条款
```

## 报告包含哪些内容

| 模块 | 说明 |
|------|------|
| 文件摘要 | 合同类型、当事方、有效期、适用法律 |
| 风险总览 | 高中低风险条款数量统计 |
| 高风险条款 | 逐条分析 + 大白话解释 + 修改建议 |
| 中风险条款 | 可以谈判的条款 |
| 缺失保护条款 | 14 项常见保护是否齐全 |
| 权力平衡评估 | 合同整体偏向哪方 |
| 签署前必做事项 | 优先级排序的行动清单 |

## 常见问题

**Q：AI 审合同准确吗？**
A：能识别常见风险条款和异常措辞，适合初步排查。复杂法律背景建议再找律师确认。

**Q：中英文合同都支持吗？**
A：支持。中文合同全程中文分析，英文合同英文分析。

**Q：数据安全吗？**
A：合同内容只在本地 OpenClaw 和你配置的 AI 之间流转，不上传第三方。

**Q：需要什么 AI 配置？**
A：OpenClaw 里配置好 AI 即可，支持阿里云百炼、DeepSeek 等，录入 Key 就行。
