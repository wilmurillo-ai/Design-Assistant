# V2EX 推广文案 — 合同审查助手
# 发布节点：分享创造

---

**做了一个合同风险扫描的 OpenClaw Skill，自由职业者用的上**

起因：自己接外包单子，每次收到甲方合同都要花很长时间逐条看，重复的检查项目其实都一样（竞业限制、IP归属、付款条件、违约责任）。

把这个流程做成了 OpenClaw Skill：上传 PDF/DOCX 或粘贴文本 → 自动识别合同类型和当事方 → 逐条标注风险等级（HIGH/MEDIUM/LOW）→ 用白话解释每条风险 → 检查 14 项常见保护条款是否缺失 → 给出修改建议。

中文合同全程中文输出，英文合同英文输出。

加了个 power balance 评估，会告诉你合同整体偏向哪一方，以及具体原因。

免责声明在最前面强制输出，不给法律建议，只做风险识别。

```
openclaw skills install ai-contract-reviewer
```
clawhub.ai/skills/ai-contract-reviewer

有没有做过类似法律文档处理工具的，那 14 项常见保护条款的检查清单来自实践，欢迎补充。
