# 知乎推广文案 — 合同审查助手
# 适合回答的问题：「自由职业者怎么保护自己？」「签合同前需要注意什么？」

---

做了几年外包，踩过一些合同上的坑，现在签合同前必做一件事：先用 AI 扫一遍风险。

**为什么需要合同审查**

大多数人签合同的方式是：快速翻一遍，觉得没什么大问题，签了。问题在于，真正的坑往往藏在措辞里：「甲方有权随时终止合同」和「甲方提前 30 天书面通知可终止合同」，后者对乙方的保护差距很大，但快速浏览很容易忽略。

**我现在的做法**

把合同文本粘贴到 OpenClaw，触发「合同审查助手」这个 skill，说「帮我从乙方角度找高风险条款」。

几分钟出一份结构化报告：
- 每条风险按 HIGH / MEDIUM / LOW 分级
- 用白话解释每条的实际含义（不是法律黑话）
- 列出缺失的保护条款（比如没有争议解决机制、没有责任上限）
- 整体权力平衡评估

**最实用的追问**

「帮我起草一封邮件，要求甲方修改高风险条款 1 和 2」——直接出可以发送的邮件，说清楚你的诉求和替代方案。对不擅长谈判的人很有用。

**局限性**

AI 不是律师，复杂的商业合同、投资协议，最终还是要找专业律师看。这个工具适合日常的服务合同、NDA、劳动合同的初步排查，帮你快速识别明显的问题条款。

```
openclaw skills install ai-contract-reviewer
```
