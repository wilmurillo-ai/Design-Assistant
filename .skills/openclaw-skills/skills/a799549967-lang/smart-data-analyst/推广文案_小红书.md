# 小红书推广文案 — 数据分析助手

---

**上传一个 Excel，AI 帮你把数据分析报告写完**

老板让你分析这个月的销售数据，你打开 Excel 傻眼了——100多行，不知道从哪下手？

我之前也是这样。后来用了一个叫「数据分析助手」的工具，直接把文件拖进去，说一句「帮我分析这个数据，找出销售下滑原因」，两分钟出一份完整报告。

报告里有什么：
- 数据概览（有没有缺失值、异常值）
- 关键趋势分析（哪个月涨了，哪个月跌了）
- 异常数据标红解释
- 最后还有「建议下一步做什么」

最有用的是可以追问——「帮我生成一份 200 字的摘要，给老板看」，直接复制粘贴发过去。

它跑在 OpenClaw 上，配好 AI（阿里云百炼或者 DeepSeek 都行）就能用，数据不会上传到任何第三方，安全。

装的方式：OpenClaw 里输
`openclaw skills install smart-data-analyst`

详细介绍：clawhub.ai/skills/smart-data-analyst
