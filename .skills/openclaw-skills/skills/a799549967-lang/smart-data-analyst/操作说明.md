# 数据分析助手 — 操作说明

> ClawHub 地址：clawhub.ai/skills/smart-data-analyst
> 安装命令：`openclaw skills install smart-data-analyst`

---

## 这个 skill 是干什么的

上传任意数据文件（Excel、CSV、JSON），AI 自动生成完整分析报告，包含关键发现、异常检测、可执行建议。不需要会写公式，不需要会 Python，说人话就行。

---

## 安装步骤

1. 打开 OpenClaw
2. 输入：
```
openclaw skills install smart-data-analyst
```
3. 安装完成后即可使用，无需额外配置

---

## 支持的文件格式

- `.csv` — 最常用，直接拖进来
- `.xlsx` / `.xls` — Excel 文件
- `.json` — 接口导出的数据
- `.tsv` — 制表符分隔
- 直接粘贴表格内容也行

---

## 基本用法

### 方式一：上传文件
1. 在 OpenClaw 对话框点附件按钮，上传文件
2. 说：`帮我分析这个数据`
3. 等待报告生成（通常 10-30 秒）

### 方式二：粘贴数据
直接把表格内容粘贴到对话框，然后说：
```
帮我分析这个数据，重点看销售趋势
```


---

## 报告包含哪些内容

| 模块 | 说明 |
|------|------|
| 数据概览 | 行列数、字段类型、缺失值统计 |
| 描述性统计 | 均值、中位数、异常值 |
| 相关性分析 | 字段之间的关联关系 |
| 时序趋势 | 有日期列时自动检测趋势 |
| 异常检测 | 超出正常范围的数据点 |
| 数据质量评分 | 完整度百分比 |
| 可执行建议 | 具体下一步行动 |

## 常见问题

**Q：数据安全吗？**
A：数据只在本地 OpenClaw 和你配置的 AI 之间流转，不上传到第三方服务器。

**Q：文件多大没问题？**
A：建议 10MB 以内。超大数据集会自动采样分析，并说明采样量。

**Q：中文数据支持吗？**
A：完全支持，中文字段名、中文数据，报告全程中文输出。

**Q：需要什么 AI 配置？**
A：在 OpenClaw 里配置任意 AI 即可，支持阿里云百炼、DeepSeek 等国内模型，录入 Key 就行。
