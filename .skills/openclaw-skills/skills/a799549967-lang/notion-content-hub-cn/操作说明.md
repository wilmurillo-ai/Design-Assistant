# 飞书内容管理中枢 — 操作说明

> ClawHub 地址：clawhub.ai/skills/notion-content-hub-cn
> 安装命令：`openclaw skills install notion-content-hub-cn`

---

## 这个 skill 是干什么的

把飞书多维表格变成你的自媒体内容大脑。接入实时热榜数据（知乎/微博/百度/头条/抖音），AI 自动生成有流量基础的选题、追踪发布数据、月底自动复盘分析爆款规律。

**飞书多维表格是什么？** 字节跳动出品的免费在线表格，功能比 Excel 强，可以做看板、日历视图、自动提醒。国内直连，不需要翻墙。

---

## 安装步骤

### 第一步：安装 skill
```
openclaw skills install notion-content-hub-cn
```

### 第二步：准备飞书多维表格
1. 打开飞书 App 或网页版（feishu.cn）
2. 新建「多维表格」，名字叫「内容管理」
3. **表格字段不需要手动建**，第一次使用时 skill 自动创建所有字段

### 第三步：获取飞书 API 授权
1. 打开 [open.feishu.cn](https://open.feishu.cn)
2. 「创建企业自建应用」→ 名称填「内容助手」→ 提交
3. 进入应用 → 「凭证与基础信息」→ 复制 **App ID**（`cli_` 开头）和 **App Secret**
4. 「权限管理」里搜索并开通：`bitable:app`
5. 打开你的多维表格 → 右上角「···」→「添加应用」→ 选「内容助手」

### 第四步：配置到 OpenClaw
在 OpenClaw 里说：
```
我的飞书 App ID 是 cli_xxx，App Secret 是 xxx，多维表格链接是 https://feishu.cn/base/xxx
```

## 核心功能和用法

### 功能一：热榜选题生成（最核心）

接入 5 个平台实时热榜，选题有数据支撑：

| 热榜 | 适合平台 |
|------|---------|
| 知乎热榜 | 公众号、知乎 |
| 微博热搜 | 小红书、微博 |
| 百度热搜 | 公众号、头条 |
| 今日头条热榜 | 头条号 |
| 抖音热点 | 抖音、小红书 |

**使用方式：**
```
今天知乎和微博有什么热点，帮我选 3 个适合我的选题
帮我出本周内容计划，参考热榜数据，5 个选题，方向是 AI 工具和副业
```

**生成结果示例：**
```
📌 选题 1
标题：《普通人怎么用 AI 工具月入过万？我试了 3 个月》
热点来源：知乎热榜「AI 副业」（热度 82万）
为什么会点：结合热点 + 利益驱动标题
建议发布：周三晚 8-10 点
```
→ 自动写入飞书，状态标注「选题中」

---

### 功能二：发布数据追踪

发完内容，一句话更新，无需手动填表：
```
昨天发的「DeepSeek 替代 ChatGPT」，公众号阅读 12000，点赞 450，涨粉 80
```
→ 自动找到对应文章，更新所有字段，并告诉你这篇的历史排名

---

### 功能三：月度复盘

```
帮我做 3 月内容复盘
```
→ 自动读取飞书全部数据，生成报告：平台对比、爆款规律、最佳发布时间、下月建议

---

### 功能四：拖稿提醒

```
我有哪些草稿超过一周没动了
```
→ 列出所有超期草稿，问你要继续写还是归档

---

### 功能五：内容日历

```
展示本周内容计划
这个月还差几篇没写
```

## 常见问题

**Q：飞书要付费吗？**
A：免费版完全够用，API 也免费。

**Q：热榜数据是实时的吗？**
A：是，每次调用都拉取最新数据，5 个平台全覆盖。

**Q：新账号没有历史数据怎么办？**
A：完全没问题，直接基于热榜+你说的方向生成选题。

**Q：手机上能看数据吗？**
A：可以，飞书 App 实时同步多维表格。

**Q：需要什么 AI 配置？**
A：OpenClaw 配置好 AI 即可，支持阿里云百炼、DeepSeek 等，录入 Key 就行。
