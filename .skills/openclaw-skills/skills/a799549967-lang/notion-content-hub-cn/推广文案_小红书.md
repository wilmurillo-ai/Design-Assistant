# 小红书推广文案 — 飞书内容管理中枢

---

**接入实时热榜生成选题，发布数据自动存飞书，月底 AI 帮你复盘**

做自媒体最难的三件事：不知道写什么、忘了记数据、不知道哪类内容有效。

这个叫「飞书内容管理中枢」的工具把这三件事全解决了，而且接了实时热榜，选题不靠拍脑袋。

**具体怎么用：**

一、选题的时候
```
今天知乎和微博有什么热点，帮我出 3 个公众号选题，方向是 AI 工具
```
它会拉取实时热榜数据，筛选和你账号方向相关的热点，生成选题，每个附带「为什么会点」「建议发布时间」，然后自动存到飞书。

二、发完内容
```
昨天那篇阅读 8000，点赞 320，涨粉 45
```
自动找到那篇文章，更新所有数据字段，还告诉你这篇在你历史里排第几。

三、月底复盘
```
帮我做本月内容复盘
```
读取飞书全部数据，输出：哪类选题效果最好、最佳发布时间、下月建议。以前这些数据都是散的，现在全汇总好给你看。

飞书是字节的免费产品，不需要翻墙，手机也能实时查看数据。
