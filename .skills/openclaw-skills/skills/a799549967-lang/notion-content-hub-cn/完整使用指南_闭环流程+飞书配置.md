# 飞书内容管理中枢 — 完整闭环使用指南

> 这份文档解决两个问题：
> 1. 飞书怎么配置（图文步骤）
> 2. AI + 飞书 + 发布 的完整工作流是什么样的

---

## 先说清楚这套工具是什么

**飞书多维表格** = 你的内容数据库（可视化界面，手机电脑都能看）
**OpenClaw + 飞书内容管理中枢 skill** = 你的 AI 助理（帮你生成选题、存数据、做分析）
**一稿多发助手 skill** = 你的内容生产工厂（一个选题 → 自动生成多平台版本）

三者组合成一条完整流水线，你只需要做两件事：
- 决策（这个选题要不要做）
- 发布（把生成的内容复制到各平台）

其他全部自动化。

---

## 第一部分：飞书配置（一次性，约 20 分钟）

### 步骤 1：创建飞书多维表格

1. 打开飞书（feishu.cn 或手机 App）
2. 左侧点「+」新建 → 选「多维表格」
3. 名称填：`自媒体内容管理`
4. 点击创建

> 💡 表格字段不需要手动建，第一次对话时 AI 自动创建所有字段

---

### 步骤 2：获取飞书 API 凭证

**进入飞书开放平台：**
1. 浏览器打开 [open.feishu.cn](https://open.feishu.cn)
2. 右上角用你的飞书账号登录

**创建应用：**
1. 点击「开发者后台」→「创建企业自建应用」
2. 填写：
   - 应用名称：`内容助手`（随意）
   - 应用描述：随意填
3. 点击「确定创建」

**获取 App ID 和 App Secret：**
1. 进入刚创建的应用
2. 左侧点「凭证与基础信息」
3. 复制 **App ID**（格式：`cli_xxxxxxxxxx`）
4. 复制 **App Secret**（点击「查看」后复制）

> ⚠️ App Secret 只显示一次，立刻复制保存

**开通权限：**
1. 左侧点「权限管理」
2. 搜索 `bitable`
3. 开通：`bitable:app`（多维表格读写）
4. 点击「申请权限」→「确认」

**发布应用（重要，不然 API 不生效）：**
1. 左侧点「版本管理与发布」
2. 点「创建版本」→ 填版本号 `1.0.0` → 提交
3. 点「申请线上发布」→ 选「无需审核直接发布」

---

### 步骤 3：把应用连接到你的多维表格

1. 打开你刚创建的「自媒体内容管理」多维表格
2. 右上角点「···」（更多）
3. 点「添加应用」
4. 找到「内容助手」→ 点「添加」

---

### 步骤 4：获取多维表格链接

1. 在多维表格页面，复制浏览器地址栏的完整链接
2. 格式类似：`https://feishu.cn/base/xxxxxxxxxx?table=tblxxxxxx`

---

### 步骤 5：告诉 OpenClaw 你的配置

打开 OpenClaw，说：

```
我的飞书 App ID 是 cli_xxx
App Secret 是 xxx
多维表格链接是 https://feishu.cn/base/xxx
```

AI 会自动验证连接，并在表格里创建所有字段。

**验证成功提示：**
```
✅ 已连接飞书多维表格「自媒体内容管理」
已自动创建 11 个字段：标题、状态、平台、发布日期、阅读量、点赞、涨粉、热点来源、选题来源、关键词、备注
当前共 0 条记录，可以开始使用了
```

## 第二部分：飞书可视化界面怎么用

配置好之后，飞书里有 4 种视图，切换方式：点左上角的视图名称旁边的「+」

---

### 视图 1：表格视图（默认）
看所有文章的状态和数据，像 Excel

```
标题              状态    平台        发布日期    阅读量  点赞  涨粉
AI副业完整指南    已发布  小红书      3月15日     8,200  320   45
ChatGPT使用教程  写作中  公众号      —           —      —     —
副业第一桶金      选题中  —          —           —      —     —
2026热门技能      选题中  知乎/公众号  —          —      —     —
```

**用途：** 全局查看所有内容的状态，手动填数据也很方便（直接点格子输入）

---

### 视图 2：看板视图
按「状态」字段分组，像便利贴墙

```
┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐
│  选题中   │  │  写作中   │  │  已发布   │  │  已归档   │
├──────────┤  ├──────────┤  ├──────────┤  ├──────────┤
│副业第一桶金│  │ChatGPT教程│  │AI副业指南 │  │旧文章...  │
│2026热门技能│  │          │  │效率工具盘点│  │          │
│DeepSeek对比│  │          │  │          │  │          │
└──────────┘  └──────────┘  └──────────┘  └──────────┘
```

**用途：** 直观看内容处于哪个阶段，拖拽卡片就能改状态

---

### 视图 3：日历视图
按「发布日期」展示，像日程表

```
        周一    周二    周三    周四    周五    周六    周日
第1周           AI副业          ChatGPT         副业桶金
第2周   效率工具         热点追踪
```

**用途：** 看本周/本月发布节奏，避免断更或扎堆发布

---

### 视图 4：图表视图（数据看板）
把数字变成可视化图表，需要手动配置一次：

**配置步骤：**
1. 点击左上角「+新增视图」→ 选「统计」
2. 添加图表：
   - **柱状图**：X轴=平台，Y轴=阅读量均值 → 对比各平台表现
   - **折线图**：X轴=发布日期，Y轴=阅读量 → 看趋势
   - **数字卡片**：显示本月发布篇数、总涨粉
   - **排行榜**：按阅读量排序，看 TOP 文章

配置完之后，每次数据更新，图表自动刷新。

## 第三部分：完整工作流闭环

### 每周工作流（全程约 30 分钟）

---

#### 🔵 周一：选题策划（10 分钟）

**你在 OpenClaw 说：**
```
今天知乎和微博有什么热点，帮我出本周 5 个内容选题
我的账号方向是 AI 工具和副业，主要发小红书和公众号
```

**AI 做的事：**
1. 实时拉取知乎热榜 + 微博热搜
2. 筛选与你方向相关的热点
3. 生成 5 个选题，每个包含：
   - 标题
   - 热点来源（知乎热榜·热度82万）
   - 为什么适合你的账号
   - 目标读者
   - 核心写作角度
   - 建议发布平台和时间
4. 全部写入飞书，状态=「选题中」

**你在飞书看到：**
```
标题                    状态    热点来源
普通人用AI月入过万      选题中  知乎热榜·82万
副业第一桶金怎么来的    选题中  微博热搜·45万
DeepSeek比GPT好在哪     选题中  百度热搜·120万
...
```

**你决策：**
- 第 1、3 个做 ✅
- 第 2 个不想写，直接在飞书里改状态为「已归档」

---

#### 🟡 周一/周二：内容生产（5 分钟/篇）

**针对选定的选题，在 OpenClaw 说：**
```
帮我给「普通人用AI月入过万」生成内容
小红书 + 公众号两个版本
```

**AI 做的事（一稿多发助手）：**
- 小红书版：口语化、第一人称、带话题标签、200-400字
- 公众号版：有大标题、分节、引导转发、800-1500字

**你做的事：**
- 检查内容是否符合预期
- 追问调整：「小红书那版改得更接地气一点」
- 复制粘贴到各平台发布

---

#### 🟠 发布后：数据录入（1 分钟/篇）

发布后去平台后台看一眼数据，回到 OpenClaw 说：
```
「普通人用AI月入过万」小红书发了，点赞 180，收藏 320，涨粉 12
```

**AI 自动：**
- 在飞书里找到这篇文章
- 更新：状态=已发布，发布日期，点赞，收藏，涨粉
- 顺带告诉你这篇在历史里排第几

**或者你直接在飞书表格里填数字**（更快，就像填 Excel）：
- 打开飞书 → 找到这行 → 点格子直接输入数字

两种方式都行，看你当时方便哪种。

---

#### 🔴 月底：复盘分析（5 分钟）

**在 OpenClaw 说：**
```
帮我做本月内容复盘
```

**AI 读取飞书全部数据，生成报告：**

```
## 3 月内容复盘

📊 发布 14 篇 | 平均点赞 186 | 总涨粉 312

### 平台对比
小红书  均值点赞 280  ✅ 互动最好
公众号  均值阅读 6200  ✅ 阅读最高
知乎    均值阅读 1800  ⚠️ 性价比低

### 爆款规律（点赞 >300 的共同特征）
- 话题：AI工具 > 副业方法
- 标题：带数字的文章占爆款 80%
- 发布时间：周三晚 8-10 点最好

### 下月建议
1. 小红书多发「AI工具实测」类，带数字标题
2. 公众号主攻「副业方法论」，适合深度长文
3. 知乎减少投入，时间产出比低
4. 热点追踪要更快，热点 48 小时内发效果最好
```

## 第四部分：常见问题

**Q：10 个选题都存到飞书了，怎么选第 3 个生成内容？**

A：两种方法：
- 直接在飞书看板视图里找到第 3 个，复制标题
- 告诉 AI：「帮我列出飞书里「选题中」的所有文章」→ AI 列出来 → 你说「生成第 3 个的内容」

---

**Q：不同平台的数据要分别录吗？**

A：同一篇文章发了多个平台，可以这样说：
```
「AI月入过万」这篇：
小红书 点赞280 收藏420 涨粉18
公众号 阅读6800 点赞95 涨粉32
```
AI 会分别更新两条记录（飞书里这篇文章有两行，每行对应一个平台）

---

**Q：每天都要打开飞书看吗？**

A：不需要每天看。你的工作模式是：
- 创作时：用 OpenClaw 生成
- 发布后：顺手说一句数据（或者飞书里填一下）
- 周一：用 OpenClaw 出本周计划
- 月底：用 OpenClaw 做复盘

飞书是数据的「存放地」和「可视化界面」，需要看数据的时候打开，不需要频繁刷新。

---

**Q：手机上能用吗？**

A：可以。飞书手机 App 可以看所有视图，在外面也能查看内容状态和数据。OpenClaw 目前主要在电脑上用。

---

**Q：飞书要收费吗？**

A：免费版完全够用：
- 多维表格：免费，不限行数
- API 调用：免费
- 手机 App：免费
