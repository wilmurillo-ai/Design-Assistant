# 一稿多发助手 — 操作说明

> ClawHub 地址：clawhub.ai/skills/content-repurposer-cn
> 安装命令：`openclaw skills install content-repurposer-cn`

---

## 这个 skill 是干什么的

一篇文章或一个主题，自动生成适配 7 个平台的版本：公众号、知乎、小红书、抖音口播脚本、头条号、Twitter Thread、LinkedIn。每个版本都是按平台调性重新写的，不是简单复制粘贴改格式。

---

## 安装步骤

1. 打开 OpenClaw
2. 输入：
```
openclaw skills install content-repurposer-cn
```
3. 安装完成，无需额外配置

---

## 支持的输入方式

- 粘贴已有文章全文
- 给一个主题或标题
- 给几个关键词或大纲
- 上传文档文件

---

## 支持的输出平台

| 平台 | 格式 | 字数/时长 |
|------|------|---------|
| 微信公众号 | 长文，含标题+摘要+分节 | 800-1500字 |
| 知乎 | 问答格式，含自动生成的问题 | 500-1000字 |
| 小红书 | 口语化，含话题标签 | 200-500字 |
| 抖音/TikTok | 口播脚本，含镜头提示 | 30-60秒 |
| 头条号 | 短段落，移动端友好 | 600-1200字 |
| Twitter Thread | 5-8条推文，英文 | 280字/条 |
| LinkedIn | 专业风格，英文 | 150-300字 |

---

## 基本用法

### 方式一：基于已有内容改写
把文章粘贴进来，说：
```
帮我把这段内容一稿多发，适配各个平台
```

### 方式二：指定平台
```
只需要微信公众号和小红书版本
帮我把这篇改成知乎回答和抖音脚本
```

### 方式三：从主题开始
```
我有个主题：「年轻人为什么越来越不愿意加班」，帮我生成各平台内容
```

## 各平台版本特点

**公众号版**：有标题、摘要、分节标题，结尾有引导关注语句
**知乎版**：自动生成配套问题，结构化数据驱动，以专业口吻回答
**小红书版**：第一人称、口语化、带 emoji 标题、结尾有互动话题标签
**抖音脚本**：前 3 秒有钩子，三段式结构，口语化，含`[镜头切换]`等舞台提示
**头条版**：移动端友好的短段落，标题略带悬念
**Twitter Thread**：英文，每条独立成句，末条有 CTA
**LinkedIn**：英文，专业但不死板，结尾有问题引发评论

## 追问技巧

生成后可以继续调整：
```
小红书那版改得更接地气，像普通女生写的
抖音脚本加上字幕格式
公众号标题帮我想 5 个备选，A/B 测试用
知乎那版加入更多数据和案例
```

## 常见问题

**Q：可以从零主题开始吗？**
A：可以，给主题或关键词，skill 自动展开内容。

**Q：只需要某几个平台？**
A：告诉它你需要哪些平台，只生成指定的版本。

**Q：中英文都能做吗？**
A：支持。中文内容生成中文平台版本，同时自动翻译适配 Twitter / LinkedIn 的英文版本。

**Q：需要什么 AI 配置？**
A：OpenClaw 里配置好 AI 即可，支持阿里云百炼、DeepSeek 等，录入 Key 就行。
