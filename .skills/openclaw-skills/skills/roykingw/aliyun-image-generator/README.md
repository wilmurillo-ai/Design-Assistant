# 🎨 阿里云百炼智能生图助手 (Aliyun Image Generator)

这是一个纯 Python 实现、开箱即用的 Agent SKILL。
只需输入专属指令[千问画个图：]，Agent 就能自动将你简单的想法扩写为丰富细腻的画面描述，并调用阿里云百炼大模型（通义万相）为你生成精美图片，直接保存到你的电脑上！

## ✨ 核心亮点
- **专属指令防冲突**：采用特定指令触发，完美兼容 QClaw/OpenClaw 等平台，不会与系统自带的画图工具打架。
- **极简画图**：你只需说“画个程序员”，Agent 会自动帮你扩写成 100 字的专业级生图提示词。
- **零环境配置**：无需安装复杂的依赖，脚本会自动检测并安装所需的 Python 库 (`requests`)。
- **对话式配置**：无需在复杂的 UI 界面里找配置项，直接在聊天窗口把 API Key 发给 Agent 即可自动保存！

## 🚀 快速开始

### 1. 获取阿里云百炼 API Key (免费)
这个 SKILL 依赖阿里云的“通义万相”大模型。新用户通常有免费额度。
1. 访问 [阿里云百炼控制台](https://bailian.console.aliyun.com/) 并登录。
2. 在左侧导航栏找到 **“API-KEY 管理”**。
3. 点击“创建新的 API-KEY”，你会得到一串以 `sk-` 开头的密钥。**请复制并保存好它。**

### 2. 安装 SKILL
将整个 `aliyun-image-gen` 文件夹放入你的 Agent 框架的 `skills/` 目录下，并重启/刷新你的 Agent。  
或者从ClawHub获取

### 3. 开始使用 (专属触发指令)
为了防止与其他画图插件冲突，**你必须使用 `千问画个图：` 作为开头**来呼叫这个专属画师。

> **你**：千问画个图：赛博朋克风格的猫咪。
> 
> **Agent**：好的！不过我发现您还没配置阿里云的 API Key。请直接把您的 Key（sk-开头）发给我，我帮您保存下来。
> 
> **你**：sk-123456789abcdefg...
> 
> **Agent**：收到！Key 已安全保存到本地。正在为您构思画面并作画...
> 
> *(等待几秒钟)*
> 
> **Agent**：画好了！图片已保存到您的电脑：`downloads/generated_img_1700000000.png`。我为您设计的画面是：一只戴着霓虹护目镜的机械黑猫，正蹲在雨夜的霓虹灯牌上...

**💡 提示**：API Key 只需在第一次使用时提供，脚本会将其安全地保存在本地的 `.aliyun_key` 隐藏文件中。以后你就可以直接让它画图了！
