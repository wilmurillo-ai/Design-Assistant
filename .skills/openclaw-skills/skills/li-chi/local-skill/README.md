# local-skill

Local skill repository to be released, including publish/check scripts, package metadata, source files, and documentation required for pre-publish inspection and publishing.

## Usage

- `bash scripts/check.sh`
- `bash scripts/publish.sh --dry-run`

## Release checklist

1. Run checks
2. Review package metadata
3. Run a dry publish
4. Publish

## Repository contents

- `src/`: implementation
- `scripts/`: publish and validation helpers
- `docs/`: release notes and process docs
