## 发票类型与代码规范

### 常见发票类型

| 发票类型 | 编码 | 特征描述 |
|---------|------|---------|
| 增值税专用发票 | 1 | 购买方和销售方均有税号，有税额信息 |
| 增值税普通发票 | 2 | 购买方和销售方均有税号，有税额信息 |
| 区块链电子发票 | 1009 | 发票代码20位，支持二维码扫描 |
| 医疗电子票据(住院) | 1011 | 医院开具的住院费用票据 |
| 医疗电子票据(门诊) | 1012 | 医院开具的门诊费用票据 |

### 发票代码规则

**增值税发票代码**（12位）：

| 位置 | 含义 | 取值范围 |
|-----|------|---------|
| 第1位 | 发票类型代码 | 0=增值税专用发票，1=增值税普通发票 |
| 第2-5位 | 地区代码 | 行政区划代码 |
| 第6-7位 | 发票种类代码 | 01=专用发票，03=普通发票 |
| 第8-12位 | 年度月份代码 | 如2401表示2024年1月 |

**示例**：`144031900110`

- 第1位：`1` → 增值税普通发票
- 第2-5位：`4403` → 广东省广州市
- 第6-7位：`19` → 通用机打发票
- 第8-12位：`00110` → 批次号

**区块链电子发票代码**（20位）：

- 以"1"开头
- 包含区域、行业、时间等信息
- 支持扫码枪扫描

## 关键字段格式定义

### 发票代码

- **格式**：10位或12位数字
- **规则**：
  - 纯数字组成
  - 不能以0开头（第1位为1-9）
  - 特定位置有特定含义

### 发票号码

- **格式**：8位数字
- **规则**：
  - 纯数字组成
  - 通常印刷在发票右上角

### 校验码

- **格式**：20位或32位数字
- **规则**：
  - 增值税专用发票：32位
  - 增值税普通发票：20位
  - 位于发票右下角

### 纳税人识别号（税号）

- **格式**：15/17/18/20位
- **规则**：
  - 统一社会信用代码（18位）
  - 或旧版税号（15位）
  - 18位统一代码：统一社会信用代码的标准格式
  - 第9-17位为核心主体标识码

**18位统一代码结构**：

| 段位 | 含义 | 说明 |
|-----|------|------|
| 第1位 | 注册类型区 | 1=企业,2=个体工商户,3=机关事业单位等 |
| 第2-5位 | 登记机关区 | 行政区划代码 |
| 第6-17位 | 企业序号区 | 顺序码+校验码 |
| 第18位 | 校验码 | 通过特定算法得出 |

### 开票日期

- **格式**：`YYYY-MM-DD`
- **规则**：
  - 4位年份
  - 2位月份（01-12）
  - 2位日期（01-31）
  - 日期需为有效日期

### 金额字段

| 字段 | 精度要求 | 说明 |
|-----|---------|------|
| 单价 | 2位小数 | 商品或服务的单位价格 |
| 数量 | 整数或2位小数 | 商品或服务的数量 |
| 金额 | 2位小数 | 单价 × 数量 |
| 税率 | 百分比 | 如0.13表示13% |
| 税额 | 2位小数 | 金额 × 税率 |
| 不含税金额 | 2位小数 | 合计金额（税前） |
| 价税合计 | 2位小数 | 不含税金额 + 税额 |

## 金额计算规则

### 基本公式

```
价税合计 = 不含税金额 + 税额
```

### 单项计算

```
税额 = 金额 × 税率
价税合计 = 金额 × (1 + 税率)
```

### 明细汇总

```
总不含税金额 = Σ(每项不含税金额)
总税额 = Σ(每项税额)
总价税合计 = 总不含税金额 + 总税额
```

### 有效税率

| 税率类型 | 有效税率 |
|---------|---------|
| 一般纳税人 | 13%、9%、6% |
| 小规模纳税人 | 3%、5% |
| 免税 | 0% |

### 金额逻辑校验规则

1. **一致性校验**：同一发票中，各项金额的价税合计应等于明细汇总
2. **精度校验**：所有金额字段保留2位小数
3. **非负校验**：金额、税率、税额均需大于等于0
4. **范围校验**：税率必须在有效范围内（如0.03、0.06、0.09、0.13等）

## 格式校验示例

### 示例1：正确的增值税普通发票

```
发票代码：144031900110
发票号码：12345678
开票日期：2024-01-15
购买方税号：91440300XXXXXXXXX
销售方税号：91440300YYYYYYYYY
不含税金额：100.00
税率：0.13
税额：13.00
价税合计：113.00
```

**校验结果**：
- 发票代码：12位数字 ✓
- 发票号码：8位数字 ✓
- 日期格式：YYYY-MM-DD ✓
- 税额计算：100.00 × 0.13 = 13.00 ✓
- 价税合计：100.00 + 13.00 = 113.00 ✓

### 示例2：格式错误示例

**错误1：税号位数不对**

```
纳税人识别号：91440300XX
```

- 期望：15/17/18/20位
- 实际：12位
- 错误类型：格式错误

**错误2：金额逻辑错误**

```
不含税金额：100.00
税率：0.13
税额：13.01
价税合计：113.00
```

- 期望税额：100.00 × 0.13 = 13.00
- 实际税额：13.01
- 错误类型：税额计算错误

**错误3：校验码格式错误**

```
校验码：1234567890
```

- 期望：20位（普通发票）或32位（专用发票）
- 实际：10位
- 错误类型：校验码长度不符合规范
