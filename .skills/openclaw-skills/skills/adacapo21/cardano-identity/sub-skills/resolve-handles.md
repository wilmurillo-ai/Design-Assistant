# Resolve ADAHandles

## Steps

1. Call `get_adahandles` to retrieve handles owned by the wallet.
2. Display each handle with a `$` prefix.
3. If the user wants their address for a handle, combine with `get_addresses`.

## Example prompt

> "What are my ADAHandles?"

## Example response

You own the following ADAHandles:
- **$alice**
- **$alice_secondary**
