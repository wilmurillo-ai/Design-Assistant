# Active Imagination: Protocols and Practice

## What is Active Imagination?

Active imagination is Jung's central technique for dialoguing with the unconscious. Unlike passive fantasy or daydreaming, it involves conscious participation with autonomous inner figures while maintaining ego awareness.

> "Active imagination is the process of dialoguing with aspects of one's unconscious. During this reflective process, emotions, symbols, images, and subpersonalities are brought into consciousness through one's imagination."

Jung discovered this method between 1913-1916 during his "confrontation with the unconscious"—a period of intense inner work documented in The Red Book. It became his primary tool for self-healing and integration.

```

│  ✗ History of psychosis or dissociative disorders      │
│  ✗ Active substance use (including cannabis)           │
```

## The Dangers: Understanding Risk

### 1. Psychosis Risk

```
PSYCHOSIS RISK FACTORS
════════════════════════════════════════════════════════════

The most serious danger. Occurs when:
┌─────────────────────────────────────────────────────────┐
│  • Ego is overwhelmed by unconscious content           │
│  • Person loses ability to distinguish inner/outer     │
│  • Reality testing becomes impaired                    │
│  • Boundary between psyche and world dissolves         │
└─────────────────────────────────────────────────────────┘

Risk increases with:
├── Personal or family history of psychosis
├── Substance use during practice
├── Weak ego structure
├── Lack of professional guidance
└── Pushing too far too fast
```

### 2. Ego Inflation

```
INFLATION DYNAMICS
════════════════════════════════════════════════════════════

What happens:
          ┌───────────────────────────────────┐
          │   EGO ENCOUNTERS ARCHETYPE        │
          │   (e.g., the Wise Old Man)        │
          └─────────────────┬─────────────────┘
                            │
                            ▼
    ┌───────────────────────────────────────────────────┐
    │          HEALTHY RESPONSE:                        │
    │   "The wise old man archetype is active in me;    │
    │    I can access wisdom beyond my ego."            │
    └───────────────────────────────────────────────────┘
                            │
                            ▼
    ┌───────────────────────────────────────────────────┐
    │          INFLATION (dangerous):                   │
    │   "I AM the Wise Old Man. I have special          │
    │    knowledge that others don't have."             │
    └───────────────────────────────────────────────────┘

Signs of inflation:
• Grandiosity, feeling "chosen" or special
• Know-it-all attitude
• Believing you have special powers
• Dismissing others' perspectives
• Messianic feelings
```

### 3. Archetypal Possession

```
POSSESSION DYNAMICS
════════════════════════════════════════════════════════════

The ego doesn't just identify with the archetype—
it gets TAKEN OVER by it:

     STRONG EGO                    WEAK EGO
     ──────────                    ────────
         │                             │
         ▼                             ▼
  ┌─────────────────┐          ┌─────────────────┐
  │   ARCHETYPE     │          │   ARCHETYPE     │
  │   ┌───────┐     │          │                 │
  │   │ EGO   │     │          │    (ego is      │
  │   │ stays │     │          │    submerged)   │
  │   │ aware │     │          │                 │
  └───┴───────┴─────┘          └─────────────────┘
         │                             │
         ▼                             ▼
   INTEGRATION                    POSSESSION

"The danger of identifying with archetypes is that
our ego simply can't contain them."
```

### 4. Dissociation

```
DISSOCIATION RISK
════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────┐
│  WARNING SIGNS:                                        │
│                                                         │
│  • Feeling "spacey" or unreal after sessions           │
│  • Difficulty returning to normal awareness            │
│  • Inner figures feel more real than outer life        │
│  • Time distortion or memory gaps                      │
│  • Depersonalization or derealization                  │
│  • Trouble maintaining daily responsibilities          │
└─────────────────────────────────────────────────────────┘

"Experimenting with this contact regularly leads
either to the so-called communications becoming
more and more stupid or to a dangerous dissociation
of consciousness."
                                        — C.G. Jung
```

## Robert Johnson's Four-Step Protocol

Based on *Inner Work: Using Dreams and Active Imagination for Personal Growth*

### Overview

```
THE FOUR STEPS
════════════════════════════════════════════════════════════

┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│ 1. INVITE   │───►│ 2. DIALOGUE │───►│ 3. VALUES   │───►│ 4. RITUAL   │
│             │    │             │    │             │    │             │
│ Set stage   │    │ Interact    │    │ Ethical     │    │ Ground in   │
│ Open door   │    │ with figure │    │ discernment │    │ real life   │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
```

### Step 1: The Invitation

```
STEP 1: SETTING THE STAGE
════════════════════════════════════════════════════════════

PREPARATION:
┌─────────────────────────────────────────────────────────┐
│                                                         │
│   □ Find a quiet, private space                        │
│   □ Allow at least 30-60 minutes uninterrupted         │
│   □ Have recording materials (journal, voice recorder) │
│   □ Be sober (no substances of any kind)              │
│   □ Be in reasonable psychological state               │
│   □ Have a specific prompt (dream, mood, feeling)      │
│                                                         │
└─────────────────────────────────────────────────────────┘

ENTERING THE SPACE:
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  1. RELAX the body completely                          │
│  2. FOCUS attention inward                             │
│  3. RECALL the prompt (dream image, mood, feeling)     │
│  4. CONCENTRATE on it—hold it in awareness             │
│  5. WATCH for it to begin moving/changing              │
│                                                         │
│  "You begin with any image... concentrate on it by     │
│  simply catching hold of it and looking at it."        │
│                                        — C.G. Jung     │
│                                                         │
└─────────────────────────────────────────────────────────┘

STARTING POINTS:
├── A dream image that stays with you
├── A strong mood (examine what image it produces)
├── A persistent emotional state
├── A recurring thought or concern
├── A physical sensation without clear cause
└── NEVER from mere curiosity or boredom
```

### Step 2: The Dialogue

```
STEP 2: THE DIALOGUE
════════════════════════════════════════════════════════════

This is the HEART of active imagination.

THE PROCESS:
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  1. The image begins to move/change on its own         │
│     (you don't direct this—you WATCH)                  │
│                                                         │
│  2. A figure may emerge or crystallize                 │
│                                                         │
│  3. You BEGIN TO SPEAK to the figure                   │
│     (either aloud or in imagination)                   │
│                                                         │
│  4. The figure RESPONDS                                │
│     (you'll be startled—it says things you didn't know)│
│                                                         │
│  5. True DIALOGUE unfolds                              │
│     (back and forth, genuine exchange)                 │
│                                                         │
└─────────────────────────────────────────────────────────┘

CRITICAL PRINCIPLES:
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  ★ Let the figure be AUTONOMOUS                        │
│    (it has its own perspective, opinions, knowledge)   │
│                                                         │
│  ★ Don't DOMINATE or manipulate                       │
│    (stay curious and receptive)                        │
│                                                         │
│  ★ Don't SCRIPT the responses                         │
│    (this isn't creative writing)                       │
│                                                         │
│  ★ RECORD everything                                   │
│    (write or speak as it happens)                      │
│                                                         │
│  ★ Stay KIND and respectful                           │
│    (these are parts of you)                            │
│                                                         │
│  ★ Keep RETURNING to the figure                       │
│    (don't let mind wander to other topics)             │
│                                                         │
└─────────────────────────────────────────────────────────┘

WHAT TO SAY:
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  "Who are you?"                                        │
│  "What do you want?"                                   │
│  "What do you have to tell me?"                        │
│  "Why are you appearing now?"                          │
│  "What do you need from me?"                           │
│  "What are you trying to protect?"                     │
│  "What would you have me understand?"                  │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### Step 3: The Values (Ethical Discernment)

```
STEP 3: ETHICAL CONFRONTATION
════════════════════════════════════════════════════════════

The archetypes are AMORAL—they don't share human ethics.
The ego must bring ethical discernment.

┌─────────────────────────────────────────────────────────┐
│                                                         │
│   UNCONSCIOUS FIGURE:                                  │
│   "Destroy your enemies"                               │
│   "Leave your responsibilities"                        │
│   "You are above normal rules"                         │
│   "Act on this impulse now"                           │
│                                                         │
│            ┌───────────────────────┐                   │
│            │     EGO's ROLE:       │                   │
│            │                       │                   │
│            │  ★ Listen fully       │                   │
│            │  ★ Understand meaning │                   │
│            │  ★ Apply ethics       │                   │
│            │  ★ Negotiate          │                   │
│            │  ★ Integrate wisely   │                   │
│            └───────────────────────┘                   │
│                                                         │
│   NOT: "Yes, I'll do exactly as you say"              │
│   NOT: "No, you're evil, go away"                     │
│                                                         │
│   INSTEAD: "I hear you. What is the deeper need       │
│   here? How can I honor that within my human          │
│   values and responsibilities?"                        │
│                                                         │
└─────────────────────────────────────────────────────────┘

THIS IS NOT A DEMOCRACY:
The ego has final say about actions in the outer world.
But it must genuinely engage with what it hears.
```

### Step 4: The Ritual (Integration)

```
STEP 4: GROUNDING AND INTEGRATION
════════════════════════════════════════════════════════════

Active imagination that doesn't connect to real life is
useless or dangerous. Integration is essential.

GROUNDING PRACTICES:
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  IMMEDIATELY AFTER:                                    │
│  • Stand up, feel your feet on the ground             │
│  • Touch something physical                            │
│  • State your name, the date, where you are           │
│  • Have a drink of water or tea                        │
│  • Look around the room, name what you see             │
│                                                         │
│  RITUAL ACTION:                                        │
│  Choose a CONCRETE symbolic act to honor the session:  │
│                                                         │
│  • Light a candle briefly                              │
│  • Write a one-sentence summary                        │
│  • Draw a simple image                                 │
│  • Step outside and touch a tree                       │
│  • Do a brief physical action (stretches, walk)       │
│                                                         │
│  This tells the psyche: "I took this seriously,       │
│  AND I am returning to ordinary reality."              │
│                                                         │
└─────────────────────────────────────────────────────────┘

LIFE APPLICATION:
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  Ask: "How does this apply to my actual life?"         │
│                                                         │
│  • What attitude might I shift?                        │
│  • What relationship pattern is being shown?           │
│  • What unlived life is being pointed to?              │
│  • What action (however small) could I take?           │
│                                                         │
│  "He did not feel at ease until he took 'the most     │
│  important step of all': finding their place and       │
│  purpose in his own actual outer life."                │
│                            — Barbara Hannah on Jung    │
│                                                         │
└─────────────────────────────────────────────────────────┘

DO NOT:
• Project inner figures onto real people
• Abandon real responsibilities for inner adventures
• Treat active imagination as "more real" than life
• Stop at the experience without integration
```

## Different Modalities

Active imagination can be done through multiple channels:

```
MODALITIES FOR ACTIVE IMAGINATION
════════════════════════════════════════════════════════════

VISUAL (Jung's primary method):
┌─────────────────────────────────────────────────────────┐
│  Internal visualization of images and dialogue          │
│  Similar to shamanic journeying                         │
│  Requires strong imaginative capacity                   │
└─────────────────────────────────────────────────────────┘

WRITING (Automatic/Dialogic Writing):
┌─────────────────────────────────────────────────────────┐
│  Write dialogue as it emerges                          │
│  Let hand move without conscious direction             │
│  Good for those who struggle with visualization        │
└─────────────────────────────────────────────────────────┘

ARTISTIC:
┌─────────────────────────────────────────────────────────┐
│  Painting, drawing, sculpting                          │
│  Let the image emerge through the medium               │
│  Jung: "Often the hands know how to solve a riddle    │
│         with which the intellect has wrestled in vain" │
└─────────────────────────────────────────────────────────┘

MOVEMENT (Dance/Body):
┌─────────────────────────────────────────────────────────┐
│  Let the body express the unconscious content          │
│  Authentic movement practice                           │
│  Good for those more kinesthetically oriented          │
└─────────────────────────────────────────────────────────┘

MUSIC/SOUND:
┌─────────────────────────────────────────────────────────┐
│  Improvise with voice or instruments                   │
│  Let sounds emerge from the mood or image              │
│  Particularly effective for pre-verbal material        │
└─────────────────────────────────────────────────────────┘

SANDPLAY (Related):
┌─────────────────────────────────────────────────────────┐
│  Three-dimensional arrangement of figures in sand      │
│  Usually done with a trained therapist present         │
│  Creates physical manifestation of inner world         │
└─────────────────────────────────────────────────────────┘
```

## Maintaining Critical Distance

Jung emphasized maintaining ego awareness throughout:

```
THE TWO-CHAIR MODEL
════════════════════════════════════════════════════════════

During active imagination, you occupy two positions:

    ┌──────────────────────┐
    │   OBSERVING EGO      │
    │                      │
    │   "I am watching     │
    │   this unfold"       │
    │                      │
    │   Maintains reality  │
    │   testing            │
    └──────────┬───────────┘
               │
               │ BOTH present simultaneously
               │
    ┌──────────▼───────────┐
    │  PARTICIPATING EGO   │
    │                      │
    │  "I am in dialogue   │
    │   with this figure"  │
    │                      │
    │   Engages genuinely  │
    └──────────────────────┘

BALANCE IS KEY:
• Too much observer → session stays superficial
• Too little observer → inflation/possession risk
```

## Common Inner Figures

```
TYPICAL FIGURES ENCOUNTERED
════════════════════════════════════════════════════════════

THE SHADOW:
Same-sex figure embodying rejected qualities
• Often appears threatening or inferior at first
• Contains both "dark" and "golden" material
• Key question: "What do I deny in myself?"

THE ANIMA/ANIMUS:
Contrasexual figure bridging to deeper Self
• May be seductive, critical, or guide-like
• Changes form at different developmental stages
• Key question: "How do I relate to the other within?"

THE WISE OLD MAN/WOMAN:
Archetypal sage figure
• Offers guidance, perspective, meaning
• Risk: Inflation if identified with
• Key question: "What wisdom is being offered?"

THE INNER CHILD:
Younger version of self
• Carries wounds and unlived potential
• May need protection, nurturing, play
• Key question: "What was lost that wants reclaiming?"

THE INNER CRITIC:
Attacking or judging voice
• Often internalized parental/social voice
• May protect against deeper vulnerability
• Key question: "What is this protecting?"

THE TRICKSTER:
Disruptive, playful energy
• Challenges rigidity and pretension
• Can lead astray or to necessary chaos
• Key question: "What needs shaking up?"

ANIMAL GUIDES:
Instinctual energies in animal form
• Snake, bear, wolf, bird, etc.
• Connect to embodied/instinctual wisdom
• Key question: "What instinct is speaking?"
```

## Safety Protocols

### Before Every Session

```
PRE-SESSION CHECKLIST
════════════════════════════════════════════════════════════

□ I am sober (no alcohol, cannabis, or other substances)
□ I am in a reasonably stable emotional state
□ I have a genuine prompt from the unconscious
□ I have adequate time and privacy
□ I know how to ground myself afterward
□ I have contact info for my therapist if needed
□ I am not doing this to escape reality
□ I remember: I maintain ethical discernment
```

### During Sessions

```
SESSION SAFETY
════════════════════════════════════════════════════════════

IF OVERWHELMED:
┌─────────────────────────────────────────────────────────┐
│  1. Open your eyes immediately                         │
│  2. Look around the room, name 5 things you see        │
│  3. Feel your feet on the ground                       │
│  4. Say your name, today's date, where you are         │
│  5. Stand up and move your body                        │
│  6. The session is over—do not continue today          │
│  7. Contact your therapist or a trusted person         │
└─────────────────────────────────────────────────────────┘

RED FLAGS (stop immediately):
• Losing sense of where you physically are
• Voices telling you to harm yourself or others
• Inability to "come back" when you choose
• Intense terror that won't subside
• Feeling "taken over" and unable to stop
• Confusion about what's real
```

### When to Seek Professional Support

```
SEEK A JUNGIAN ANALYST WHEN:
════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────┐
│  • Material becomes overwhelming                       │
│  • Difficulty returning to normal awareness            │
│  • Signs of inflation or grandiosity                   │
│  • Persistent disturbing imagery                       │
│  • Life functioning is affected                        │
│  • Dissociative symptoms appear                        │
│  • You want a relational container for the work        │
│  • Complex or traumatic material emerges               │
│  • You're uncertain about safety                       │
└─────────────────────────────────────────────────────────┘

"Working with a trained professional can lend us that
strong ego and protect us from inflation."
                                        — Rafael Krüger

RESOURCES:
├── IAAP (International Association for Analytical Psychology)
├── C.G. Jung Institute (various cities)
└── Local Jungian-oriented therapists
```

## Integration with Other Practices

### Active Imagination and Dreams

```
DREAMS → ACTIVE IMAGINATION
════════════════════════════════════════════════════════════

A dream provides the best starting material:

1. Take a striking image from a dream
2. Return to it in active imagination
3. Let the figure speak what it couldn't in the dream
4. Ask the questions you wish you'd asked
5. Watch for the dream to continue/transform

"The dream is a fragment of involuntary psychic activity...
Active imagination consciously continues the dream."
```

### Active Imagination in Addiction Recovery

```
RECOVERY APPLICATION
════════════════════════════════════════════════════════════

Can be used to dialogue with:

THE CRAVING:
┌─────────────────────────────────────────────────────────┐
│  "What do you really want?"                            │
│  "What are you trying to give me?"                     │
│  "What emptiness do you speak to?"                     │
└─────────────────────────────────────────────────────────┘

THE ADDICT PART:
┌─────────────────────────────────────────────────────────┐
│  "What do you need that you're trying to get this way?"│
│  "What wound are you protecting?"                      │
│  "What would happen if you couldn't use?"              │
└─────────────────────────────────────────────────────────┘

THE HEALTHY SELF (or Higher Power):
┌─────────────────────────────────────────────────────────┐
│  "What guidance do you offer?"                         │
│  "How do I find what I seek another way?"              │
│  "What does true nourishment look like?"               │
└─────────────────────────────────────────────────────────┘

CAUTION: In early recovery, this should be done
WITH professional support, not alone.
```

## Remember

> "The essential thing is to differentiate oneself from these unconscious contents by personifying them, and at the same time to bring them into relationship with consciousness. That is the technique for stripping them of their power."
> — C.G. Jung

Active imagination is powerful medicine. Used wisely with proper containment, it enables genuine dialogue with the depths. Used carelessly, it risks inflation, possession, or dissociation.

**The goal is not to escape into the inner world, but to bring its riches back to serve life.**
