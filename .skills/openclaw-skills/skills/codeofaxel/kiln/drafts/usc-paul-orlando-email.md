I'm Adam Arreola, a USC MSEI alum (Cohort 8). I recently launched an open-source project called **Kiln** — infrastructure that lets AI agents (Claude, GPT, OpenClaw, etc.) autonomously control physical manufacturing equipment. It's live at kiln3d.com and on GitHub.

Kiln currently supports 3D printers across OctoPrint, Klipper, Bambu Lab, Prusa Connect, and Elegoo. Agents can pull models from user files or download from Thingiverse, MyMiniFactory, and Cults3D; generate new models from text prompts using Meshy AI and Google DeepMind tools; print locally; monitor jobs via webcam; and route to remote fulfillment through Craftcloud's network of 150+ print shops when no local printer is available. Nothing like this existed before — it's the first open-source AI manufacturing runtime.

The architecture is designed to generalize beyond 3D printing to laser cutters, CNC mills, and SLA printers. I've already built the adapter layer for those device types — what I need now is physical hardware access to validate them.
