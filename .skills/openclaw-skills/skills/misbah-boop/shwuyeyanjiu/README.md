# 上海物业研究 Skill

## 📋 简介

这个技能用于研究**具体某个物业项目**的详细信息，包括招标公告、中标公告、饱和收入等。

**定位**：项目级研究（微观视角）

## 🎯 核心功能

### 1. 项目研究
- 输入：项目名称（如"静安区武泰公寓"）
- 输出：项目的详细信息

### 2. 信息内容
- ✅ 招标公告：合同期限、物业费标准
- ✅ 中标公告：中标日期、中标企业
- ✅ 评标结果公告：评标日期、候选企业
- ✅ 饱和收入：年度收入计算

### 3. 数据源
- **上海市物业招标公告网站**：https://962121.fgj.sh.gov.cn/wyweb/web/front/common/greenmorelist.jsp

## 🚀 快速开始

### 安装依赖
```bash
cd ~/.openclaw/workspace/skills/shwuyeyanjiu/scripts
uv sync
```

### 运行示例
```bash
# 研究具体项目
uv run python research_project.py --project "静安区武泰公寓"

# 结果示例
项目名称: 静安区武泰公寓
合同期限: 3年
中标日期: 2025-08-19
合同到期: 2028-08-31
饱和收入: 831,526 元/年
```

## ⚠️ 注意事项

1. **项目名称**：需要提供准确的项目名称
2. **数据完整性**：部分项目可能缺少某些公告
3. **OCR识别**：PDF文件是扫描件，需要OCR识别
