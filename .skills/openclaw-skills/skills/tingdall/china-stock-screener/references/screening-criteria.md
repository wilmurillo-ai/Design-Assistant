## 技术面筛选标准

### 均线多头排列
```
条件：MA5 > MA10 > MA20 > MA60
评分：+10分
```

### MACD金叉
```
条件：DIF上穿DEA，且在零轴上方
评分：+10分
```

### KDJ金叉
```
条件：K值上穿D值，J值<100
评分：+5分
```

### 放量上涨
```
条件：成交量 > 20日均量 × 1.5
评分：+5分
```

### 突破新高
```
条件：收盘价创20日新高
评分：+5分
```

### 站上年线
```
条件：收盘价 > MA250
评分：+5分
```

## 基本面筛选标准

### PE估值
```
低估：PE < 10 或 历史分位 < 20%
合理：PE 10-25
高估：PE > 30
```

### PB估值
```
低估：PB < 1.5
合理：PB 1.5-3
高估：PB > 5
```

### ROE（净资产收益率）
```
优秀：ROE > 20%
良好：ROE 15-20%
一般：ROE 10-15%
较差：ROE < 10%
```

### 股息率
```
高息：股息率 > 4%
中等：股息率 2-4%
低息：股息率 < 2%
```

### 净利润增速
```
高增长：净利润同比 > 30%
稳定增长：净利润同比 15-30%
低速增长：净利润同比 0-15%
负增长：净利润同比 < 0
```
