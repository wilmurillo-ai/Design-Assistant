# 三维投资分析框架

基于宏观-行业-公司三维认知的系统性投资分析方法论。

## 核心框架

### 一、三维认知体系

#### 1. 宏观维度 - 理解经济周期
- **经济周期四阶段**：繁荣 → 衰退 → 萧条 → 复苏
- **股市领先性**：股市是经济的提前反应（通常提前3-6个月）
- **关键指标**：
  - GDP增速、PMI、CPI/PPI
  - 货币政策（利率、存款准备金率）
  - 财政政策（基建投资、减税）
  - 国际环境（汇率、地缘政治）

#### 2. 行业维度 - 理解赛道生命周期
- **生命周期四阶段**：
  - 初创期：高风险高回报，技术驱动
  - 成长期：渗透率快速提升，龙头崛起
  - 成熟期：竞争格局稳定，现金流为王
  - 衰退期：需求萎缩，避免参与
- **选股先选行业**：好行业 > 好公司 > 好价格

#### 3. 公司维度 - 理解企业质地
- **核心三问**：
  - 靠什么赚钱？（商业模式）
  - 能否持续赚钱？（护城河）
  - 当前估值是否合理？（安全边际）

### 二、四大核心体系

#### 1. 选股体系
- **坚守能力圈**：只投看得懂的标的
- **能力圈拓展**：通过深度研究逐步扩大
- **负面清单**：
  - 不碰看不懂的业务模式
  - 不碰财务造假嫌疑公司
  - 不碰管理层诚信存疑公司

#### 2. 交易体系
- **买入有理由**：每笔买入都有明确的逻辑支撑
- **卖出有条件**：预设止盈止损条件
- **应对而非预测**：市场不可预测，但可应对
- **顺势而为**：趋势形成后不易改变

#### 3. 仓位体系
- **管理比选股更重要**：仓位决定心态
- **动态调整**：根据市场环境调整仓位
- **核心原则**：
  - 牛市重仓（70-90%）
  - 震荡市中仓（50-70%）
  - 熊市轻仓（30-50%）

#### 4. 风控体系
- **设定底线**：单笔亏损不超过总资金5%
- **活下来优先**：保本第一，赚钱第二
- **止损纪律**：
  - 技术止损：跌破关键支撑位
  - 时间止损：买入后不达预期
  - 逻辑止损：买入逻辑被破坏

### 三、盈利三层次

#### 第一层次：保障资本
- **仓位管理**：不满仓，保留现金
- **分散投资**：单票不超过20%
- **止损纪律**：坚决执行，不抱幻想

#### 第二层次：稳健盈利
- **交易系统**：建立可重复的交易规则
- **一致性执行**：克服情绪干扰
- **复利效应**：追求稳定收益而非暴利

#### 第三层次：追求卓越
- **持续学习**：不断提升认知
- **耐心等待**：好机会是等出来的
- **长期主义**：时间是优秀企业的朋友

## 分析工作流程

### Step 1: 宏观环境研判
1. 判断当前经济周期阶段
2. 分析货币政策取向
3. 评估市场风险偏好
4. 确定整体仓位策略

### Step 2: 行业赛道筛选
1. 识别高景气度行业
2. 评估行业生命周期位置
3. 分析竞争格局
4. 确定重点跟踪行业

### Step 3: 公司深度分析
1. 商业模式分析
2. 财务健康度评估
3. 护城河识别
4. 估值合理性判断

### Step 4: 交易计划制定
1. 确定买入理由
2. 设定目标价位
3. 制定止损策略
4. 规划仓位分配

### Step 5: 持仓管理与复盘
1. 定期检视持仓逻辑
2. 跟踪关键变量变化
3. 执行交易纪律
4. 总结得失经验

## 关键认知要点

### 股市本质
- **短期零和博弈**：交易是财富的转移
- **长期正和游戏**：优秀企业创造价值
- **指数投资**：分享经济增长红利的最简单方式

### 适投人群特质
- **耐心**：愿意等待好机会
- **纪律**：严格执行交易计划
- **自省**：持续反思和改进
- **抗波动**：能承受短期回撤

### 核心要诀
- **知识需实战转化**：拒绝"意念炒股"
- **顺势而为**：不与趋势作对
- **独立思考**：不盲目跟风
- **控制情绪**：贪婪与恐惧是最大敌人

## 使用指南

### 何时使用此Skill

1. **个股分析**：需要深度分析某只股票的投资价值
2. **组合诊断**：需要评估现有持仓的合理性
3. **行业研究**：需要了解某个行业的发展前景
4. **交易决策**：需要制定买入/卖出计划
5. **投资教育**：需要学习系统性的投资方法论

### 如何结合其他工具

- **结合WebSearch**：获取最新的宏观数据、行业动态、公司公告
- **结合Stock Analysis**：获取实时行情和技术指标
- **结合Buffett Analysis**：进行深度价值投资分析
