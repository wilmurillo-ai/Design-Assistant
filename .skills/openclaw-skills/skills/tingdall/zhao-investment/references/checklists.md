# 投资分析检查清单

快速参考的检查清单，确保分析不遗漏关键要素。

---

## 一、买入前检查清单

### 宏观环境（权重20%）
- [ ] 当前经济周期阶段明确
- [ ] 货币政策取向清晰
- [ ] 无重大系统性风险
- [ ] 市场风险偏好适中

### 行业分析（权重25%）
- [ ] 行业处于成长期或成熟期早期
- [ ] 行业景气度向上
- [ ] 竞争格局良好（非红海）
- [ ] 政策环境支持或中性
- [ ] 行业空间足够大（TAM > 1000亿）

### 公司质地（权重35%）
- [ ] 商业模式清晰可持续
- [ ] 具有明确的护城河
- [ ] 管理层诚信可靠
- [ ] 财务健康（负债率<60%，现金流为正）
- [ ] ROE连续3年>15%
- [ ] 营收/净利润连续3年正增长

### 估值水平（权重15%）
- [ ] PE处于历史30%分位以下，或
- [ ] PB处于历史30%分位以下，或
- [ ] PEG < 1.5
- [ ] 相对行业有估值优势

### 技术面（权重5%）
- [ ] 处于上升趋势，或
- [ ] 在关键支撑位附近，或
- [ ] 底部区域企稳

**评分标准：**
- 90-100分：强烈推荐买入
- 80-89分：建议买入
- 70-79分：可小仓位试探
- 60-69分：观望等待
- <60分：不建议买入

## 二、持仓定期检查清单

### 每周检查
- [ ] 持仓个股是否有重大公告
- [ ] 是否触及止损线
- [ ] 是否达到目标价位
- [ ] 行业是否有政策变化
- [ ] 大盘是否出现系统性风险

### 每月检查
- [ ] 财报数据是否恶化
- [ ] 买入逻辑是否仍然成立
- [ ] 估值是否变得过高
- [ ] 是否有更好的替代标的
- [ ] 仓位配置是否合理

### 季度深度检查
- [ ] 季度业绩是否达预期
- [ ] 行业竞争格局是否变化
- [ ] 护城河是否被侵蚀
- [ ] 管理层是否有变动
- [ ] 战略方向是否调整

---

## 三、卖出决策检查清单

### 必须卖出的信号
- [ ] 触及止损线（无条件执行）
- [ ] 买入逻辑被破坏
- [ ] 发现财务造假证据
- [ ] 管理层重大诚信问题
- [ ] 行业政策重大不利变化

### 考虑卖出的信号
- [ ] 估值严重高估（PE>历史90%分位）
- [ ] 基本面明显恶化
- [ ] 发现明显更好的替代标的
- [ ] 持仓比例过高需要平衡
- [ ] 达到目标价位

### 继续持有检查
- [ ] 短期波动不影响长期逻辑
- [ ] 基本面依然健康
- [ ] 估值仍然合理
- [ ] 无更好替代标的

---

## 四、风险控制检查清单

### 仓位管理
- [ ] 单票仓位不超过20%
- [ ] 单一行业不超过30%
- [ ] 总仓位与大盘环境匹配
- [ ] 保留一定现金储备（至少10%）

### 止损管理
- [ ] 每笔交易都有明确止损位
- [ ] 单笔亏损不超过总资金5%
- [ ] 连续亏损3笔后强制休息
- [ ] 月度亏损超过15%减仓

### 情绪管理
- [ ] 不因恐慌而盲目卖出
- [ ] 不因贪婪而追高买入
- [ ] 不因FOMO而随意交易
- [ ] 保持独立思考，不盲从

## 五、财务分析速查表

### 关键财务指标健康标准

| 指标 | 优秀 | 良好 | 一般 | 较差 |
|------|------|------|------|------|
| ROE | >20% | 15-20% | 10-15% | <10% |
| 毛利率 | >40% | 30-40% | 20-30% | <20% |
| 净利率 | >15% | 10-15% | 5-10% | <5% |
| 营收增速 | >30% | 20-30% | 10-20% | <10% |
| 净利润增速 | >30% | 20-30% | 10-20% | <10% |
| 资产负债率 | <40% | 40-60% | 60-70% | >70% |
| 流动比率 | >2 | 1.5-2 | 1-1.5 | <1 |
| 经营现金流/净利润 | >1 | 0.8-1 | 0.5-0.8 | <0.5 |

### 财务风险信号
- [ ] 应收账款增速 > 营收增速
- [ ] 存货增速 > 营收增速
- [ ] 经营现金流持续为负
- [ ] 自由现金流持续为负
- [ ] 商誉占净资产比例 > 30%
- [ ] 有息负债率快速上升
- [ ] 大股东高比例质押
- [ ] 关联交易占比过高

## 六、估值速查表

### 不同行业合理PE区间

| 行业类型 | 低估值 | 合理估值 | 高估值 |
|----------|--------|----------|--------|
| 银行 | <5 | 5-8 | >8 |
| 保险 | <8 | 8-12 | >12 |
| 地产 | <6 | 6-10 | >10 |
| 公用事业 | <12 | 12-18 | >18 |
| 消费（成熟） | <20 | 20-30 | >30 |
| 消费（成长） | <30 | 30-50 | >50 |
| 医药 | <25 | 25-40 | >40 |
| 科技（成熟） | <20 | 20-35 | >35 |
| 科技（成长） | <40 | 40-80 | >80 |

### 估值方法选择

| 行业特征 | 首选估值法 | 次选估值法 |
|----------|------------|------------|
| 盈利稳定 | PE | PEG |
| 盈利波动大 | PB | PS |
| 重资产 | PB | EV/EBITDA |
| 轻资产/互联网 | PS | P/MAU |
| 高成长 | PEG | PS |
| 金融 | PB | PE |
| 周期股 | PB | PS |

## 七、交易纪律检查清单

### 买入纪律
- [ ] 有明确的买入理由
- [ ] 已设定目标价位
- [ ] 已设定止损价位
- [ ] 已规划仓位比例
- [ ] 不在盘中冲动买入

### 卖出纪律
- [ ] 触及止损无条件执行
- [ ] 达到目标分批止盈
- [ ] 逻辑破坏立即卖出
- [ ] 不因小幅波动频繁交易
- [ ] 不在恐慌中盲目割肉

### 持仓纪律
- [ ] 定期检查持仓逻辑
- [ ] 不因短期波动焦虑
- [ ] 盈利仓位让利润奔跑
- [ ] 亏损仓位及时止损
- [ ] 保持组合平衡

---

## 八、复盘检查清单

### 交易复盘
- [ ] 记录每笔交易的买入理由
- [ ] 记录实际执行与计划的差异
- [ ] 分析盈利交易的成功因素
- [ ] 分析亏损交易的失败原因
- [ ] 总结可复用的经验

### 定期复盘
- [ ] 月度收益率统计
- [ ] 胜率与盈亏比计算
- [ ] 最大回撤分析
- [ ] 交易频率评估
- [ ] 策略有效性检验

### 体系优化
- [ ] 识别分析框架的盲点
- [ ] 更新行业认知
- [ ] 完善检查清单
- [ ] 优化仓位管理规则
- [ ] 改进情绪控制方法
