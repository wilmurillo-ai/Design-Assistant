## 宏观分析工作流

### 第一步：经济周期定位

**关键指标检查清单：**
- [ ] GDP增速（同比/环比）
- [ ] PMI指数（制造业/非制造业）
- [ ] 工业增加值
- [ ] 社会消费品零售总额
- [ ] 固定资产投资增速

**周期判断标准：**
| 阶段 | GDP趋势 | PMI | 通胀 | 政策取向 |
|------|---------|-----|------|----------|
| 复苏期 | 触底回升 | >50且上升 | 低位 | 宽松 |
| 繁荣期 | 高位运行 | >50 | 上升 | 中性偏紧 |
| 衰退期 | 见顶回落 | 跌破50 | 高位 | 观望 |
| 萧条期 | 低位徘徊 | <50 | 下降 | 宽松 |

### 第二步：货币政策分析

**央行政策工具：**
- 存款准备金率调整
- 基准利率/LPR调整
- 公开市场操作（MLF/逆回购）
- 再贷款再贴现政策

**流动性判断：**
- 银行间市场利率（DR007）
- M2增速与社融规模
- 信用利差变化

### 第三步：财政政策评估

**政策力度：**
- 财政赤字率
- 专项债发行规模
- 减税降费力度
- 基建投资增速

### 第四步：国际环境扫描

**外部因素：**
- 美联储政策动向
- 美元指数走势
- 人民币汇率
- 地缘政治风险

## 行业分析工作流

### 第一步：行业生命周期判断

**判断标准：**

| 阶段 | 收入增速 | 利润率 | 竞争格局 | 资本开支 |
|------|----------|--------|----------|----------|
| 初创期 | 极高(>50%) | 负或极低 | 分散 | 高 |
| 成长期 | 高(20-50%) | 快速提升 | 集中度提升 | 高 |
| 成熟期 | 中低(<20%) | 稳定 | 寡头垄断 | 低 |
| 衰退期 | 负增长 | 下滑 | 出清 | 极低 |

### 第二步：行业空间测算

**TAM/SAM/SOM分析：**
- TAM（总可及市场）：理论市场空间
- SAM（可服务市场）：实际可达市场
- SOM（可获得市场）：企业可获得份额

**增长驱动因素：**
- 渗透率提升空间
- 单价提升潜力
- 使用频次增加
- 新产品/新场景

### 第三步：竞争格局分析

**波特五力模型：**
1. 现有竞争者竞争强度
2. 潜在进入者威胁
3. 替代品威胁
4. 上游供应商议价能力
5. 下游客户议价能力

**竞争要素：**
- 成本优势
- 差异化能力
- 规模效应
- 转换成本
- 网络效应

### 第四步：政策环境评估

**政策类型：**
- 鼓励类：补贴、税收优惠、准入放宽
- 限制类：环保要求、能耗限制、产能管控
- 中性类：行业自律、标准制定

## 公司分析工作流

### 第一步：商业模式理解

**核心问题：**
- 公司卖什么？（产品/服务）
- 卖给谁？（客户画像）
- 怎么卖？（渠道/模式）
- 怎么赚钱？（盈利模式）

**商业模式画布：**
1. 价值主张
2. 客户细分
3. 渠道通路
4. 客户关系
5. 收入来源
6. 核心资源
7. 关键业务
8. 重要伙伴
9. 成本结构

### 第二步：护城河识别

**护城河类型：**

| 类型 | 特征 | 可持续性 | 典型案例 |
|------|------|----------|----------|
| 品牌护城河 | 强品牌认知 | 高 | 茅台、可口可乐 |
| 成本护城河 | 规模效应/资源优势 | 中高 | 隆基、福耀 |
| 技术护城河 | 专利/研发优势 | 中 | 华为、恒瑞 |
| 网络护城河 | 用户网络效应 | 高 | 腾讯、阿里 |
| 牌照护城河 | 准入壁垒 | 高 | 银行、保险 |
| 转换成本 | 高切换成本 | 中 | 软件、数据库 |

### 第三步：财务分析

**盈利能力：**
- [ ] 毛利率（行业对比）
- [ ] 净利率（趋势分析）
- [ ] ROE（杜邦分析）
- [ ] ROIC（投入资本回报率）

**成长能力：**
- [ ] 营收增速（3年复合）
- [ ] 净利润增速（3年复合）
- [ ] 现金流增速

**运营效率：**
- [ ] 资产周转率
- [ ] 存货周转天数
- [ ] 应收账款周转天数
- [ ] 现金周期

**财务安全：**
- [ ] 资产负债率
- [ ] 流动比率/速动比率
- [ ] 利息保障倍数
- [ ] 自由现金流

**现金流质量：**
- [ ] 经营现金流/净利润 > 1
- [ ] 资本开支占经营现金流比例
- [ ] 分红率与股息率

### 第四步：估值分析

**绝对估值法：**
- DCF（现金流折现）
- DDM（股利折现）

**相对估值法：**
- PE（市盈率）
- PB（市净率）
- PS（市销率）
- EV/EBITDA
- PEG（PE/增长率）

**估值对比：**
- 历史分位数
- 行业对比
- 国际对比

## 交易计划工作流

### 第一步：买入理由明确

**买入 checklist：**
- [ ] 宏观环境支持
- [ ] 行业处于景气周期
- [ ] 公司质地优良
- [ ] 估值合理或低估
- [ ] 技术面有支撑
- [ ] 风险收益比合适

### 第二步：目标价位设定

**目标价测算：**
- 基于估值修复：历史均值/行业均值
- 基于业绩增长：盈利预测 × 目标PE
- 基于DCF：内在价值

### 第三步：止损策略制定

**止损类型：**
1. **技术止损**：跌破关键支撑位（如MA60、前期低点）
2. **比例止损**：亏损达到总资金X%（建议5-8%）
3. **时间止损**：买入后Y天不达预期
4. **逻辑止损**：买入逻辑被破坏

### 第四步：仓位规划

**仓位分配原则：**
- 单票不超过总仓位20%
- 行业不超过总仓位30%
- 根据确定性调整仓位（确定性高→仓位重）

**分批建仓策略：**
- 首次建仓：计划仓位的30-50%
- 二次加仓：确认趋势后加30%
- 三次加仓：突破关键位置后加剩余

## 持仓管理工作流

### 每日检查清单

- [ ] 持仓个股是否有重大公告
- [ ] 行业是否有政策变化
- [ ] 大盘是否出现系统性风险信号
- [ ] 个股是否触及止损线
- [ ] 个股是否达到目标价位

### 每周复盘清单

- [ ] 本周交易执行是否符合计划
- [ ] 持仓逻辑是否仍然成立
- [ ] 仓位配置是否需要调整
- [ ] 新增关注标的筛选
- [ ] 下周交易计划制定

### 每月深度复盘

- [ ] 月度收益率统计
- [ ] 胜率/盈亏比分析
- [ ] 最大回撤评估
- [ ] 交易纪律执行情况
- [ ] 投资体系优化方向

### 调仓决策树

```
是否触及止损线？
├── 是 → 坚决止损，不问理由
└── 否 → 买入逻辑是否破坏？
    ├── 是 → 卖出
    └── 否 → 估值是否严重高估？
        ├── 是 → 减仓
        └── 否 → 是否有更好标的？
            ├── 是 → 换股
            └── 否 → 继续持有
```

## 常见错误清单

### 分析阶段常见错误
- [ ] 忽视宏观环境变化
- [ ] 过度乐观估计行业空间
- [ ] 对护城河判断过于主观
- [ ] 财务数据理解不深入
- [ ] 估值方法选择不当

### 交易阶段常见错误
- [ ] 没有明确的买入理由就买入
- [ ] 不设止损或止损不执行
- [ ] 满仓单吊一只股票
- [ ] 追涨杀跌
- [ ] 频繁交易

### 持仓阶段常见错误
- [ ] 忽视持仓逻辑变化
- [ ] 盈利时过早止盈
- [ ] 亏损时死扛不止损
- [ ] 情绪化交易
- [ ] 不复盘不总结

## 工具与资源

- 宏观经济：国家统计局、央行、Wind
- 行业数据：行业协会、券商研报
- 公司数据：财报、公告、调研
- 市场数据：交易所、行情软件

### 分析工具
- 财务分析：Excel、Python
- 估值模型：DCF模板、可比公司表
- 技术分析：行情软件
- 信息聚合：RSS、投研平台
