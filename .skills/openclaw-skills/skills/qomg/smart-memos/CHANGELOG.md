# Changelog

所有项目的显著变更都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.2.0] - 2024-03-18

### 新增
- ✏️ 编辑修改功能 - 支持修改标题、内容、分类、标签
- 🔖 Markdown 标签导入 - 自动识别 `#标签名` 格式
- 📋 显示 ID - list/search 命令显示备忘录 ID
- 🏷️ 新增分类 - 健康、美食

### 优化
- 🏷️ 分类算法 - 增加权重机制，12 个分类更准确
- 🔍 搜索体验 - 显示 ID 方便后续操作

## [1.1.0] - 2024-03-18

### 新增
- 📥 PDF 导入支持
- 📥 Markdown 导入支持
- 📥 HTML/TXT/JSON 导入支持

### 优化
- 📄 PDF 文本提取 - 支持 pdfplumber/PyPDF2/pdftotext
- 📝 Markdown 解析 - 支持按标题和列表分割

## [1.0.0] - 2024-03-18

### 新增
- 📝 添加备忘录
- 🔍 搜索备忘录
- 📋 列出备忘录
- 🏷️ 智能自动分类
- 🔤 语义搜索
- 📊 统计分析
- 🗑️ 删除/归档
