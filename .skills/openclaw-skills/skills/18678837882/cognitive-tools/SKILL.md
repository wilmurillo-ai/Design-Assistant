## 11个工具详解

### 🔪 逻辑之刃
- **用途**：对文本进行庖丁解牛式的分析，提取命题并进行逻辑推导
- **触发词**：`逻辑分析` `分析逻辑` `用逻辑之刃`
- **适用场景**：分析报告、拆解论证链、找逻辑漏洞

### 🔨 问题之锤
- **用途**：锤碎问题，找到问题背后的问题，一路敲下去直至第一问题显现
- **触发词**：`问题之锤` `追问` `深入挖掘` `本质`
- **适用场景**：决策分析、根因诊断、战略追问

### 🎋 抽象之梯
- **用途**：将文本在具象和抽象之间转换，让表达更细腻或更凝练
- **触发词**：`抽象` `具象化` `提升层次` `抽象之梯`
- **适用场景**：汇报提炼、战略解读、数据洞察

### 🪞 角度之镜
- **用途**：切换不同角度看待问题，综合所得，提出一个独特视角的切入解法
- **触发词**：`多角度` `换个角度` `角度之镜` `视角分析`
- **适用场景**：竞品分析、方案评估、局面研判

### �茅 定义之矛
- **用途**：精准定义概念的本质，绕着它打转，一矛丢出，将它钉死在语义空间
- **触发词**：`定义` `本质` `概念界定` `定义之矛`
- **适用场景**：会议对齐、术语澄清、减少沟通歧义

### 🕸️ 矩阵之网
- **用途**：用两个维度、动态颗粒度，将课题肢解，狭隘地看全局
- **触发词**：`矩阵分析` `多维分析` `矩阵之网`
- **适用场景**：市场分析、分类整理、多因素决策

### 🏹 类比之弓
- **用途**：抓住一个结构，同构外推到另一个用户更容易理解的场域
- **触发词**：`类比` `打比方` `类比之弓`
- **适用场景**：向领导汇报、向客户解释、跨行业借鉴

### 📖 汉语新解
- **用途**：将一个汉语词汇进行全新角度的解释，语言风趣、一针见血
- **触发词**：`汉语新解` `解释词汇` `词汇新解`
- **适用场景**：写作润色、表达创新、文化解读

### ⚖️ Prompt Judger
- **用途**：对Prompt进行评分(1-10)并提供改进建议
- **触发词**：`打分` `Prompt评分` `改进Prompt` `judger`
- **适用场景**：优化OpenClaw自身技能、分析他人Prompt

### 🔍 知识探索专家
- **用途**：对一个知识点从来源、本质、发展三个维度深度探索
- **触发词**：`探索` `知识探索` `深度理解`
- **适用场景**：学习新概念、理解新事物

### 📝 公文笔杆子
- **用途**：政府机关公文风格的写作，包括领导讲话稿、通知、报告等
- **触发词**：`公文` `讲话稿` `政府文件`
- **适用场景**：行政文件、正式报告、官方文书

## 使用方法

### 单工具调用
直接说出工具名称，AI自动激活对应认知工具进行分析。

### 工具链组合（推荐用法）
```
问题之锤 → 角度之镜 → 逻辑之刃 → 矩阵之网
```
流程：用问题之锤深入挖掘本质 → 用角度之镜找到最佳观察角度 → 用逻辑之刃梳理因果关系 → 用矩阵之网多维肢解

### 示例对话
- "用问题之锤分析：要不要扩产？" 
- "用逻辑之刃分析这份报告的论证链"
- "用汉语新解解释'内卷'这个词"
