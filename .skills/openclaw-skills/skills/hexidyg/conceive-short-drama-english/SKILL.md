# Short Drama Plot Conception Skill

Create dramatic short drama plot outlines based on user-provided one-liner inspirations.

## When to Use This Skill

When users need to:
- Expand a creative idea into a complete story
- Quickly conceive a short drama plot outline
- Get wildly creative plot ideas
- Design storylines for short videos/micro-films

## Workflow

### 1. Understand User Inspiration

Extract core elements from user input:
- Protagonist identity/characteristics
- Core conflict/event
- Story background/setting

### 2. Randomly Select Satisfying Element Combinations

**Randomly select 8-12** from 50 satisfying elements to incorporate into the story, ensuring:
- At least 3-5 inspirational struggle types (rock bottom start, all-in gamble, comeback climax, persistence, self-breakthrough, etc.)
- At least 3-5 truth-beauty-goodness types (good deeds rewarded, helping others helps self, giving back to society, self-sacrifice, keeping promises, etc.)
- At least 3-5 emotional types (love fulfilled, family reunited, mentor-student bond, gratitude for recognition, grateful return, etc.)
- At least 3-5 satisfaction types (face-slapping humiliation, identity reversal, extreme comeback, witnessed by many, instant fame, etc.)
- Each category should be evenly distributed to make the story richer and fuller

### 3. Structure the Plot

Design according to three-act structure:

```
Act 1 (Setup): Rock bottom start + Inciting incident (~120 words)
Act 2 (Confrontation): Struggle + Setbacks + Helping others + Benefactors (~250 words)
Act 3 (Resolution): Comeback + Satisfying climax + Happy ending (~130 words)
```

### 6. Content Health Requirements (Important!)

**Prohibited Negative Descriptions:**

| Category | Prohibited Content | Alternative Expression |
|----------|-------------------|------------------------|
| **Gore/Violence** | Bleeding, wounds, slaughter, beating | Use conflict, setbacks, difficulties instead |
| **Physical Harm** | Ulceration, corrosion, scars, disability | Use hard work, exhaustion, wear instead |
| **Disgusting** | Vomiting, excrement, parasites | Avoid such descriptions |
| **Excessive Pain** | Extreme pain, on death's door | Use fatigue, persistence instead |
| **Negative Emotions** | Despair, suicidal thoughts, breakdown | Use confusion, low points, struggle instead |

**Correct Examples:**
- ❌ Fingers corroded and ulcerated by chemicals
- ✅ Hands became rough from long-term labor
- ❌ Beaten black and blue all over
- ✅ Encountered heavy difficulties and setbacks
- ❌ Elderly person on death's door
- ✅ Elderly person frail and weak

2. Paragraphs 4-6 show protagonist's effort (make audience respect)
3. Paragraphs 7-9 set major setback (make audience worry)
4. Paragraphs 10-12 bring turning point (make audience anticipate)
5. Paragraphs 13-15 achieve comeback (make audience satisfied)
6. Ending elevates theme (make audience moved)

### 8. Plot Coherence Requirements (Important!)

**Logical Principles That Must Be Followed:**

| Principle | Description | Wrong Example | Correct Approach |
|-----------|-------------|---------------|------------------|
| **Emotional Setup** | Love line needs interaction | "Seeing the design, she breaks engagement and runs to him" | Have a process of meeting/helping/understanding first |
| **Reasonable Motivation** | Character actions need reasonable motives | "Poor child learning architectural design" | "Teaching practical skills like cooking/repair" |
| **Causal Coherence** | Before/after plots need causal relationship | "Suddenly gets investment" | "Previous good deeds/efforts bring opportunity" |
| **Reality Feasibility** | Plot must fit basic realistic logic | "Poor person directly buys mansion" | "First improve basic life then develop" |
| **Gradual Growth** | Success is accumulated step by step | "Get rich overnight" | "Small success → big success process" |
| **Natural Satisfaction** | Satisfying elements should blend naturally | "Face-slapping for its own sake" | "Natural result of success" |

**Love Line Design Requirements:**
1. Must have initial meeting scene and occasion
2. Must have interaction and helping process (at least 2-3 times)
3. Must have emotional accumulation details (eyes/words/actions)
4. Female lead choosing male lead needs reasonable reason (character/talent/shared experience)
5. Cannot "fall in love upon seeing success," must "love through understanding"

**Helping Others Design Requirements:**
1. Must fit within protagonist's ability (poor person can't donate huge amounts)
2. Must fit recipient's actual needs (poor child needs food/shelter first)
3. Helping method must be sustainable (not one-time charity)
4. Helping behavior must be natural (not deliberate show)

**Satisfying Element Integration Requirements:**
1. Satisfying elements are natural results of plot development, not forced in
2. Face-slapping plots need cause (was humiliated before)
3. Benefactor help needs reason (protagonist deserves help)
4. Success needs process (can't reach the top in one step)

```

```
