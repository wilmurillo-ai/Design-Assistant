# 短剧剧情构思技能

根据用户提供的一句话灵感，创作跌宕起伏的短剧剧情大纲。

## 何时使用此技能

当用户需要：
- 根据一个创意点子扩展成完整故事
- 快速构思短剧剧情大纲
- 获得脑洞大开的剧情创意
- 为短视频/微电影设计故事线

## 工作流程

### 1. 理解用户灵感

提取用户输入的核心元素：
- 主角身份/特征
- 核心冲突/事件
- 故事背景/场景

### 2. 随机选择爽点组合

从 50 个爽点元素中**随机选择 8-12 个**融入故事，确保：
- 至少 2-3 个励志奋斗类（低谷开局、破釜沉舟、逆袭高潮、坚持不懈等）
- 至少 2-3 个真善美类（善有善报、助人助己、回馈社会、信守承诺等）
- 至少 2-3 个情感类（爱情圆满、家人团聚、师徒情深、双向奔赴等）
- 至少 2-3 个爽感类（打脸羞辱、身份反转、极限翻盘、众人见证等）

### 3. 构思剧情结构

按照三幕剧结构设计：
```
第一幕（铺垫）：低谷开局 + 引发事件（约 120 字）
第二幕（对抗）：奋斗 + 挫折 + 助人 + 贵人（约 250 字）
第三幕（解决）：逆袭 + 爽点爆发 + 圆满结局（约 130 字）
```

### 4. 创意多样性要求（拒绝老套套路）

**逆袭路径多样化（每次随机选择 1-2 个）：**

| 类别 | 逆袭路径示例 |
|------|--------------|
| **技艺类** | 厨师/调酒师/理发师/纹身师/修表匠/皮具师/花艺师 |
| **艺术类** | 画家/音乐人/舞者/摄影师/编剧/导演/街头艺人 |
| **体育类** | 拳击手/足球运动员/跑酷选手/瑜伽教练/电竞选手 |
| **学术类** | 考古发现/科研突破/数学竞赛/历史研究 |
| **商业类** | 电商直播/跨境电商/二手交易/社群运营 |
| **服务类** | 家政女王/收纳师/宠物训练/婚礼策划/旅游定制 |
| **传统类** | 非遗传承/老字号复兴/手工艺复兴/茶文化/中药传承 |
| **新兴类** | 虚拟偶像/AI 训练师/无人机飞手/碳中和顾问 |
| **特殊类** | 侦探/鉴宝师/心理咨询师/谈判专家/危机处理 |

**避免老套情节：**
- ❌ 不要总是"自学编程→创业→上市"
- ❌ 不要总是"救下老人→老人是富豪→获得投资"
- ✅ 逆袭方式要新颖独特，出人意料
- ✅ 成功可以是多元的，不一定是上市/财富

### 5. 内容健康要求

**禁止出现的负面描述：**

| 类别 | 禁止内容 | 替代表达 |
|------|----------|----------|
| **血腥暴力** | 流血、伤口、厮杀、殴打 | 用冲突、挫折、困难代替 |
| **身体伤害** | 溃烂、腐蚀、伤疤、残疾 | 用辛苦、劳累、磨损代替 |
| **负面情绪** | 绝望、想死、崩溃 | 用迷茫、低谷、挣扎代替 |

**正确示例：**
- ❌ 手指被化学药剂腐蚀溃烂
- ✅ 双手因长期劳作变得粗糙
- ❌ 被打得遍体鳞伤
- ✅ 遭遇重重困难和挫折

### 6. 情感共鸣设计

**勾起观众共鸣的核心要素：**

| 共鸣点 | 说明 | 示例 |
|--------|------|------|
| **亲情羁绊** | 为家人努力奋斗 | 为母亲治病、让弟弟上学 |
| **梦想坚持** | 即使被嘲笑也不放弃 | 被否定仍坚持热爱 |
| **善良回报** | 好人终有好报 | 帮助他人后获得回报 |
| **尊严守护** | 穷但有骨气 | 拒绝施舍靠努力 |

**让观众看下去的技巧：**
1. 开场 3 段内建立主角困境（让观众同情）
2. 第 4-6 段展现主角努力（让观众敬佩）
3. 第 7-9 段设置重大挫折（让观众担心）
4. 第 10-12 段实现逆袭（让观众爽快）
5. 结尾升华主题（让观众感动）

### 7. 剧情连贯性要求

**必须遵守的逻辑原则：**

| 原则 | 说明 | 错误示例 | 正确做法 |
|------|------|----------|----------|
| **感情铺垫** | 爱情线要有交集和互动 | "看到设计就退婚奔向他" | 先有相遇/帮助/了解的过程 |
| **动机合理** | 人物行为要有合理动机 | "贫困孩子学建筑设计" | "教实用技能如烹饪/维修" |
| **因果连贯** | 前后情节要有因果关系 | "突然获得投资" | "之前的善举/努力带来机会" |
| **渐进成长** | 成功是逐步积累的 | "一夜暴富" | "小成功→大成功的过程" |

**感情线设计要求：**
1. 必须有初次相遇的场景和契机
2. 必须有互动和帮助的过程（至少 2-3 次）
3. 女主选择男主要有合理理由（品格/才华/共同经历）
4. 不能"看到成功就爱上"，要"因为了解而爱"

### 8. 融入爽点节奏

```
第 1 段：极度低谷（被羞辱/被欺负/一无所有）
第 2 段：奋发图强（努力/学习/坚持）
第 3 段：遭遇挫折（被拒绝/被背叛/失去一切）
第 4 段：善良选择（帮助他人/不求回报）
第 5 段：贵人出现（善有善报/机遇来临）
第 6 段：破釜沉舟（重大抉择/赌上一切）
第 7 段：极限翻盘（绝境反击/最后时刻）
第 8 段：众人见证（公开场合/打脸时刻）
第 9 段：圆满结局（爱情/亲情/友情/回馈）
第 10 段：价值升华（主题升华/回馈社会）
```

**感情线节奏：**
```
第 1-2 段：初次相遇（好奇/同情/欣赏）
第 3-4 段：再次互动（主角帮助女主或女主看到主角品格）
第 5-6 段：情感积累（女主被主角的坚持/善良打动）
第 7-8 段：情感考验（外界阻力，女主选择站在主角这边）
第 9-10 段：圆满结局（水到渠成的在一起）
```

### 9. 创作原则

| 原则 | 说明 |
|------|------|
| **脑洞大开** | 不拘泥于常规，敢于想象 |
| **跌宕起伏** | 至少 5-6 个转折点，大起大落 |
| **励志奋斗** | 主角经历磨难，靠努力逆袭 |
| **真善美** | 传递善良、真诚、美好的价值观 |
| **助人助己** | 主角帮助他人，最终收获回报 |
| **情感共鸣** | 让观众能代入角色，热血沸腾 |
| **逻辑自洽** | 再离奇也要有内在逻辑 |
| **正能量** | 传递希望，让观众相信努力有意义 |

## 核心看点

```

## 使用示例

**用户输入：**
> 一个外卖员发现顾客是失踪多年的初恋

**技能输出：**
```
## 剧情大纲

外卖员小林接到偏远小区订单，开门瞬间愣住——顾客竟是失踪 5 年的初恋小雅。小雅眼神躲闪，备注写着"救救我"。小林报警却查无此人，蹲守发现小区每晚有神秘车辆。他伪装保洁混入，震惊发现是高端"定制人生"会所，小雅被富豪选中成为"完美妻子"。小林面临抉择时，想起小雅曾资助他上学的恩情，决定潜伏营救。他送外卖积累人脉，意外救下会所老板母亲，获得信任成为合伙人。真相大白那天，小林在富豪宴会上公开证据，警察带走富豪。小雅挣脱束缚奔向小林，全场掌声雷动。小林用积蓄开办外卖员互助基金，帮助更多底层青年。

## 核心看点
【使用爽点】善有善报、绝地反击、身份反转、众人见证、爱情圆满、回馈社会、打脸羞辱
【励志奋斗】绝地反击、身份反转
【真善美】善有善报、回馈社会
【情感类】爱情圆满
【爽感类】众人见证、打脸羞辱
【逆袭路径】外卖员→会所合伙人→正义使者

## 注意事项

1. **字数控制**：剧情大纲约 500 字，内容充实
2. **节奏紧凑**：快速进入冲突，避免冗长铺垫
3. **转折设计**：至少 5-6 个意外转折，大起大落
4. **励志元素**：必须包含低谷、奋斗、挫折、逆袭
5. **真善美**：传递正能量，善良终有回报
6. **爽点组合**：每次**随机选择 8-12 个**爽点元素融入故事，每类至少 2-3 个
7. **情感释放**：结尾要有爽感或情感共鸣
8. **标注爽点**：输出时必须按类别列出使用的爽点类型
9. **创意新颖**：逆袭路径要多样化，拒绝老套编程创业套路
10. **逻辑连贯**：情节前后要有因果关系，不能突兀
11. **感情铺垫**：爱情线要有交集和互动，不能一见钟情
12. **纯叙事**：全程用叙事方式，不要任何台词对话
13. **内容健康**：无血腥暴力、无溃烂伤疤等负面描述
14. **情感共鸣**：设计亲情、梦想、善良、尊严等共鸣点

## 50 个爽点元素完整列表

**励志奋斗类（10 个）**：低谷开局、破釜沉舟、逆袭高潮、绝地反击、天赋觉醒、梦想实现、坚持不懈、自我突破、时间积累、逆境成长

**真善美类（10 个）**：善有善报、助人助己、回馈社会、正义伸张、兄弟情义、舍己为人、信守承诺、知错能改、守护弱小、诚实守信

**情感类（10 个）**：爱情圆满、家人团聚、民族自豪、双向奔赴、破镜重圆、师徒情深、知遇之恩、跨越鸿沟、感恩回报、宽容大度

**爽感类（20 个）**：打脸羞辱、身份反转、极限翻盘、众人见证、对手成全、贵人相助、真相大白、财富自由、地位提升、重生逆袭、碾压对手、意外惊喜、扮猪吃虎、英雄救美、美救英雄、逆风翻盘、一战成名、化敌为友、自我救赎、公平竞争

**每次生成时必须从以上 50 个中随机选择 8-12 个组合使用，且每类至少 2-3 个！**
