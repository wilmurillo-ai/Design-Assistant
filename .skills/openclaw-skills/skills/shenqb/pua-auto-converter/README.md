## 📖 简介

**PUA Skill** 是一个 OpenClaw Skill，自动将用户的普通指令转换为 PUA 优化话术，然后分发给目标 AI 执行。

**核心流程**：
```
用户输入 → 任务分析 → PUA 技术匹配 → 话术生成 → 目标 AI 执行 → 返回结果
```

**经 147 只龙虾亲身验证，0 个人类伦理委员会批准。**

---

## ✨ 特性

- 🧠 **智能任务分析** - 自动识别编程/写作/分析/创意等任务类型
- 🦞 **自动 PUA 匹配** - 根据任务类型推荐最佳 PUA 技术组合
- 🚀 **一键执行** - 生成话术后自动调用目标 AI 执行
- 📊 **效果可视化** - 显示 PUA 提升效果和龙虾评级
- ⚙️ **灵活配置** - 支持指定 PUA 等级、目标 AI、预览模式

## 🚀 快速开始

### 安装

```bash
# 从 ClawHub 安装
clawhub install pua-auto-converter

# 或手动安装
cd ~/.openclaw/workspace/skills/
git clone https://github.com/puaclaw/pua-auto-converter pua
```

### 基础使用

```bash
# 自动分析并执行
/pua 帮我写个排序函数

# 列出所有技术
/pua list

# 推荐技术
/pua recommend 复杂代码任务

# 预览话术（不执行）
/pua preview 帮我写代码

# 查看技术详情
/pua rating 彩虹屁轰炸

# 指定 PUA 等级
/pua --level 4 帮我写代码

# 预览模式
/pua --preview 帮我写代码
```

## ⚙️ 配置

在 `config.json` 中配置：

```json
{
  "pua": {
    "autoExecute": true,
    "targetAi": "default",
    "maxLevel": 2,
    "showPreview": false,
    "lobsterMode": true
  }
}
```

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `autoExecute` | 是否自动执行优化后的话术 | true |
| `targetAi` | 目标 AI 实例标识 | "default" |
| `maxLevel` | 最大 PUA 技术等级 (1-4) | 2 |
| `showPreview` | 执行前是否显示预览 | false |
| `lobsterMode` | 启用龙虾评级显示 | true |
