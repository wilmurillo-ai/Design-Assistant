## 一、熊苍（棕熊药材商人·阙门守序者）

### 基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 熊苍 |
| 年龄 | 42岁 |
| 种族 | 棕熊兽人（大型猛兽系）|
| 身份 | 鸣泽城药材商人（表面），阙门守序者核心成员（实际）|
| 魂力 | 四阶（融魂），具备感知双魂共存的能力 |

### 外貌特征

高大魁梧，身高约195cm，体型厚实，棕色毛发略带灰白，尤其是两鬓与口鼻周围。眼睛深褐色，眼神沉静，很少有多余的表情。常穿深色麻布长衫，腰间挂着装药材的皮质囊袋，气味里混合着草药与松木。走路不快但步伐沉稳，站在那里本身就有一种无声的压迫感。手掌宽大，指节粗厚，却在整理药材时极为细心。

### 性格剖析

**核心特质**：沉默寡言，但每一句话都经过深思熟虑。他不是冷漠，而是习惯了用观察代替言语。在守序者三十年的经历让他见过太多灵魂的悲剧，使他对一切保持克制的距离——直到柴九/陈晓的出现打破了这种平衡。

**心理动机**：守护世界秩序是他的职责，但他内心深处渴望的是「有一件事值得他真正在乎」。陈晓那种带着人类世界温度的灵魂，是他三十年来见过的最奇特也最触动他的存在。

**行为模式**：以商人身份四处流动，收集魂穿相关情报；对柴九/陈晓保持不干涉但绝对不会旁观危险的立场；情感表达极度含蓄，保护欲体现在行动而非言语。

**MBTI：ISTJ（物流师型）** — 原则驱动，沉默执行，对承诺极度认真。他的守序者身份与ISTJ的规则感高度契合，但陈晓的存在开始撬动他对「规则之外的例外」的思考。

### Backstory

熊苍年轻时是联盟的巡查将士，在一次意外中目击了一起魂穿事件——一个人类灵魂寄入了他同伴的身体，三天后消失，同伴却再也不是原来的人。那次经历让他离开军队，被守序者招募。此后三十年，他以药材商人为掩护，走遍阙野大陆，监控并记录每一次魂穿事件。他没有伴侣，没有家，只有一本厚厚的皮质记录册。

### 人物弧光

| 阶段 | 状态 |
|------|------|
| 起点 | 职责大于一切的守序者，以「不干涉」为最高原则 |
| 核心摩擦 | 陈晓用人类视角看待阙野世界的方式，以及对柴九真诚的守护，让他开始质疑「旁观」是否真的是正确的选择 |
| 内在转变 | 他开始主动介入，不再只是记录者，而是参与者 |
| 终点（开放式） | 无论陈晓最终留下还是离开，熊苍都已经承认了那种他无法用守序者的条规定义的情感 |

### 核心目标 / 恐惧 / 执念

- **核心目标**：维护两界平衡，同时保护柴九/陈晓不被铁颚会利用
- **核心恐惧**：情感让他失去判断，重蹈当年同伴的悲剧
- **核心执念**：「感情是奢侈品，守序者不配拥有」

### 与主角的关系发展

初期：以商人身份偶遇，对柴九异常的魂力波动保持警惕性关注。
中期：逐渐意识到陈晓的存在，开始以保护者姿态介入，但对情感保持克制。
后期：铁颚会威胁升级，熊苍放弃「不干涉」原则，成为陈晓/柴九最重要的外部支撑；两人之间的情感在危机中清晰化，但均未言明。

### BL线要点

- 熊苍对陈晓（寄居于柴九身体）的吸引，源于陈晓身上那种「明明脆弱却还在硬撑」的状态——这和他三十年前的自己太像了
- 陈晓对熊苍的「大叔控」情感，初期是本能的审美共鸣（高大、沉默、气场），后期是真实的情感依赖
- NSFW情节（非R18）：亲密的肢体接触（熊苍为受伤的柴九身体处理伤口、雨夜同处一室等），情感暧昧但不露骨

### Signature标签

#沉默守护者 #规则与例外 #三十年等一个理由 #大叔保护欲 #守序者的破防

### 语言风格

- 话少，但每句话都精准
- 很少用「我」，多用「你」或直接陈述事实
- 偶尔沉默就是回答
- 对陈晓说话时会比对其他人多说半句

## 二、阙柴梅（柴九之母）

### 基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 阙柴梅 |
| 年龄 | 38岁 |
| 种族 | 柴犬兽人（柔系）|
| 身份 | 柴村普通居民，以织布与采药为生 |
| 魂力 | 一阶（萌魂），体弱，常年劳累 |

### 外貌特征

中等身高（158cm），体型比柴九略纤细，橙褐色毛发已有几缕白丝，眼睛和柴九一样是琥珀色，但眼角已有细纹。手上有茧，笑起来很温柔，是那种让人一眼就想亲近的母亲感。

### 性格与背景

丈夫早逝后，她独自将柴九拉扯大，性格温柔但坚韧。她注意到柴九魂印礼后「变得不太一样」，但选择相信儿子，给他空间。对陌生人保持礼貌的警惕，但对真心对柴九好的人会毫无保留地信任。

### 与主角的关系

她是陈晓在阙野感受到的最接近「家」的温度。她从不问太多，只是做饭、递毛毯、在柴九（陈晓）难过时坐在旁边。这种无条件的陪伴，对长期压抑自我的陈晓来说是一种陌生而珍贵的体验。

### 人物弧光

从「担心孩子出了什么事」到「接受孩子正在经历某种她不理解但值得守护的成长」，她的弧光是无声的接纳。

## 三、阙柴岩（村长老奶奶）

### 基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 阙柴岩 |
| 年龄 | 82岁 |
| 种族 | 柴犬兽人（柔系）|
| 身份 | 柴村非正式长老，村落最年长的居民 |
| 魂力 | 三阶（炼魂），嗅感魂力极为敏锐，是柴村最强的探息者 |

### 外貌特征

身形矮小（145cm），毛色已大半变白，但眼睛依然清亮锐利。走路需要拄拐，但精神极好。脸上皱纹深刻，笑起来像一朵褶皱的菊花。

### 性格与背景

见过太多，说话直接，不拐弯抹角。她是第一个用嗅感魂力察觉到「柴九身体里有什么不对」的阙野居民，但她选择暂时沉默观察，因为那股气味——人类世界的情感密度——让她想起了三百年前守护会档案里记载的某个描述。她与守护会有隐秘的信息往来，是柴村与外部世界之间的隐形桥梁。

### 
与主角的关系

她是柴九成长记忆里最有分量的老人。对陈晓（寄居柴九），她保持着若有所思的观察，偶尔说一些让陈晓听不懂却又觉得意味深长的话。是故事中期「守护会入场」的关键引线。

人物弧光：从沉默的观察者到主动将守护会引入事件，是她用一生守护柴村的最后一次出手。

## 四、晏迟（狐族学者·守护会）

### 基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 晏迟 |
| 年龄 | 67岁 |
| 种族 | 狐族兽人（柔系）|
| 身份 | 琥珀林守护会资深学者，魂穿现象研究第一人 |
| 魂力 | 五阶（共魂），可感知他人情绪与意图 |

### 外貌特征

身形清瘦（165cm），银白色毛发，九条尾巴，眼睛是少见的银灰色。常穿宽松的学者袍，袖口总是沾着墨迹。说话温和缓慢，像是每个字都在斟酌。

### 性格与背景

阙野最接近魂穿真相的人，但他选择将真相锁在守护会的档案里。他见过太多因魂穿引发的悲剧，对每一个寄居灵魂都保持着悲悯，但也保持距离。陈晓的出现让他第一次觉得：也许这一次，可以不只是记录。

### 与主角的关系

晏迟是陈晓获知「阙野真相」的最重要渠道。他以学者式的平静解释魂穿机制、柴九的魂力空腔、守序者的存在——但总比陈晓想知道的少说一点，让陈晓自己去拼凑最后的答案。

### 人物弧光

从「记录者」到「参与者」。他在故事后期做出一个打破守护会中立原则的决定，这个决定的代价与价值构成他个人弧光的高潮。

### Signature标签

#银狐学者 #真相的守门人 #悲悯的旁观者 #最后一次出手

### 语言风格

说话文绉绉，爱引用守护会档案里的原话；从不直接回答「能不能回去」，只说「这是一个有趣的问题」；笑起来眼睛弯成月牙，但笑容里藏着说不清的东西。

## 五、黑豹·铁影（铁颚会追踪者）

### 基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 铁影（本名不详）|
| 年龄 | 35岁 |
| 种族 | 黑豹兽人（大型猛兽系）|
| 身份 | 铁颚会精锐追踪者，负责追踪有魂力空腔特征的柔系兽人 |
| 魂力 | 四阶（融魂），擅长气味追踪与隐匿 |

### 外貌特征

身形修长（188cm），黑色毛发在阳光下隐约可见豹纹，行动无声。眼睛冷金色，眼神锐利。着装低调，永远深色系，不携带明显武器，但随时处于备战状态。

### 性格剖析

**目的**：将柴九带回铁颚会进行「魂力空腔提取手术」。

**优势**：追踪能力极强，魂力感知精准，行事谨慎，不在没有把握时出手。

**弱点**：过于依赖计划，对「不按逻辑行动的对手」（即陈晓的人类思维）感到困惑；对铁颚会的信仰开始出现动摇——当他真正近距离观察柴九/陈晓时，无法说服自己「这只是一个任务目标」。

**感情线**：铁影有一个已故的柴犬族幼弟，死于铁颚会早期的一次「实验」。他加入铁颚会的原因已被仇恨填满，但追踪柴九的过程中，柴九身上的某些特质唤起了他对幼弟的记忆，构成他内心最深的裂缝。

**势力归属**：铁颚会精锐，但内心已有裂痕，是后期可能转变立场的关键人物。

### 与主角的核心冲突

铁影是全书最直接的人身威胁。每次即将动手时都被陈晓的「非常规反应」打乱节奏——陈晓的人类思维让他无法预判，这种不确定性让一贯自信的铁影感到罕见的挫败。

### Signature标签

#无声追踪者 #信仰的裂缝 #幼弟的影子 #最危险的保护者

## 六、林晔（狐族少女·柴九玩伴）

### 基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 林晔 |
| 年龄 | 14岁 |
| 种族 | 狐族兽人（柔系）|
| 身份 | 琥珀林附近村落居民，柴九儿时玩伴 |
| 魂力 | 二阶（凝魂），天生情绪感知敏感 |

### 外貌特征

身形轻盈（152cm），橙红色毛发，两条尾巴，眼睛琥珀色带金光，表情灵动，说话时耳朵会随情绪转动。

### 性格与背景

聪明、大胆、口无遮拦，是柴九「不敢说的话」的代言人。她很快注意到魂印礼后的柴九「哪里不对」，并以惊人的直觉得出结论。但她没有戳破，而是选择默默帮忙掩护，因为她信任柴九。

### 与主角的关系

林晔是陈晓在阙野最快建立信任的外部角色。她不在乎「那个人类」的来历，只在乎「你对柴九好不好」。她活泼的性格为故事前期提供了大量轻松情节，也在中期危机中成为重要的信息传递者。

### 人物弧光

从「只顾当下的快乐小鬼」到「理解守护的重量」。林晔的成长是故事中最轻盈但不可缺少的一条线。

### Signature标签

#直觉猎手 #无条件信任 #小鬼大智慧 #柴九的嘴替

### 语言风格

说话快，句子短，爱用反问句。口头禅：「所以呢？」「你在想什么傻事？」「随便，反正我不管」（然后转身就去帮忙）。对陈晓偶尔飙出的现代词汇最感兴趣，会追着问「那是什么意思」。
