## 一、陈晓（寄居灵魂）

### 1.1 基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 陈晓 |
| 年龄 | 18岁 |
| 身份 | 人类世界高三学生，意外魂穿至阙野 |
| 原身外貌 | 175cm，微胖，圆脸，皮肤偏白，单眼皮但眼神明亮，笑起来有两个浅酒窝 |
| 寄居外貌 | 柴九身体：155cm，矮胖圆润，橙褐色毛，蓬松上卷尾巴，小直耳，琥珀色眼睛 |

### 1.2 性格剖析

**核心矛盾**：外表阳光开朗，内心长期压抑真实的性取向与情感需求。他习惯用「聪明」和「热情」填满自己，让别人看见他想展示的那一面，而非真实的那一面。

**心理动机**：渴望被接纳，但极度恐惧暴露真实自我后被拒绝。深柜状态不是胆怯，而是对「代价」的清醒评估。

**行为模式**：观察者本能（先观察再行动）、问题解决倾向（分析优先于情绪）、过度照顾他人（用帮助别人回避自我审视）、黑色幽默（压力下用自嘲消解紧张）。

**情绪触发点**：被人「当宠物看」或贬低；看到弱小者被欺压；有人真诚关心他本人时反而手足无措；柴九因信任他而受伤时强烈内疚。

### 1.3 MBTI：ENFJ（主人公型）

1. 他的「阳光开朗」是真实的，但也是社交策略——ENFJ天然擅长给人安全感，陈晓用这一点维持距离的同时保有联结。
2. 高智商与心思细腻表现为「极强的共情能力+过度解读他人需求」，他常比对方更清楚对方需要什么，却忘了问自己需要什么。
3. 深柜状态对ENFJ来说是持续内耗——核心驱动是「真诚联结」，但压抑自我使他永远无法实现真正的联结，这是他灵魂在临界点崩溃的根本原因。

### 1.4 Backstory

普通中产家庭，父母关系平淡，期待集中在成绩与未来。初中起意识到自己对男性的吸引，清醒地选择沉默——不是胆怯，而是理性评估后的代价计算。高中三年以「最受欢迎的那类人」示人，没有人知道他在furry社区有匿名账号，没有人知道他对bara风格的兽人角色有多深的迷恋。睡前总是把情绪压进去，一层又一层。高考前三个月的某个深夜，所有压力在极度疲惫的临界点上，让他的灵魂在睡眠边缘脱了轨。

### 1.5 人物弧光

| 阶段 | 状态 |
|------|------|
| 起点 | 习惯压抑自我、用「有用」定义自身价值的深柜少年，以旁观者姿态存在于自己的人生 |
| 核心摩擦 | 寄居柴九身体后无法再「旁观」——柴九的情绪、柴犬族的处境、阙野的威胁，每一样都逼着他做出真实反应；与棕熊大叔的情感靠近直接冲击他最深的恐惧 |
| 内在转变 | 开始理解：压抑不是保护，而是对自己施加的慢性伤害；接纳柴九的过程，也是接纳自身的过程 |
| 终点（开放式） | 找到了「想留下」的理由，也找到了「可以回去」的勇气——最终选择留在哪里，故事不给出答案，只给出他终于不再逃避的眼神 |

### 1.6 核心目标 / 恐惧 / 执念

- **核心目标**：找到回去的方法——但随着故事推进，这个目标本身开始动摇
- **核心恐惧**：真实的自己被看见之后，对方会离开
- **核心执念**：「我必须有用，才值得被留下」

### 1.7 Signature标签

#深柜阳光外壳 #过度照顾者 #观察型守护者 #furry灵魂错位 #慢热真实派

### 1.8 能力与魂力设定

以人类灵魂寄居，不具备阙野魂力等级，但拥有异常强大的「情感密度」：
- 在阙野表现为不规则的魂力脉冲，偶发性爆发远超柴九本体魂力强度
- 对柴犬族嗅感魂力响应异常强，他的情绪格外清晰，像是「气味很重的存在」
- 与柴九情感共鸣随故事推进逐渐加深，后期可同步感知对方感官体验
- 弱点：情绪波动直接消耗柴九的魂力，激动时尤为明显

### 1.9 语言风格

- 口头禅：「……好吧」「这不合理」「等等让我想想」「你确定吗」
- 习惯用反问句思考，内心独白长而绕，但说出口的话往往简短
- 与柴九内心对话：初期碎碎念，中期出现真实温柔，后期偶有长时沉默
- 偶尔出现现代词汇，引发柴九困惑，是轻松情节来源

## 二、阙柴九（宿主）

### 2.1 基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 阙柴九（柴九）|
| 年龄 | 14岁 |
| 身份 | 琥珀林柴村柴犬兽人，魂印礼后进入成年社会 |
| 外貌 | 155cm，矮胖圆润敦实，橙褐色毛，白色胸腹，蓬松上卷尾巴，小直耳，琥珀色眼睛，表情天生憨直可爱 |
| 魂力 | 二阶（凝魂），嗅感魂力较强，可感知情绪气味 |

### 2.2 
性格剖析

**核心特质**：腼腆胆小但内心有温度。他不是懦弱，而是在一个对柔系不友善的世界里学会了「小心翼翼地活着」。他渴望被保护，但同样渴望被认真对待——不是当宠物，而是当一个真实的人。

**心理动机**：渴望被保护与被认可，两者对他而言并不矛盾——他希望有足够强大的人站在他身边，但那个人必须把他当平等的存在。

**行为模式**：遇到陌生人先躲避后观察；情绪激动时控制不住「情鸣」；喜欢整理和布置自己的空间；对认定的人极为黏腻依赖。

**情绪触发点**：被人以「可爱」为由而非真实理由接近；父亲的话题；陈晓情绪低落时（情感共鸣使他能感知到）。

### 2.3 MBTI：ISFJ（守护者型）

1. 他对熟悉的人和环境拥有极强的守护本能，柴村里每一棵树、每一户人家对他来说都是值得保护的。
2. 敏感内向，但对亲近的人展现出惊人的细腻——总是默默记住别人的习惯和偏好。
3. 在陈晓到来之前，他的世界很小；陈晓的到来强行拓宽了他的视野，这既是成长，也是挑战。

### 2.4 Backstory

柴九的父亲在他八岁时因意外去世，葬于琥珀橡树东侧。从那以后，他与母亲阙柴梅相依为命。他不是没有勇气，而是从小就知道「失去」是什么感觉，所以格外珍惜留下来的一切，也格外害怕冒险。魂印礼后不久便因高烧昏迷，陈晓由此穿入——对柴九来说，那场昏迷之后，他感觉自己变了，却说不清楚哪里不一样。

### 2.5 人物弧光

| 阶段 | 状态 |
|------|------|
| 起点 | 渴望被保护、对外界充满警惕的胆小少年，世界的边界就是柴村的木栅栏 |
| 核心摩擦 | 意识到自己体内有另一个意识（陈晓）后，从恐惧到好奇到信任，是他主动走向外部世界的过程 |
| 内在转变 | 他开始理解：渴望被保护不是弱点，但保护自己的能力同样重要；他可以既依赖陈晓，又成为陈晓的依靠 |
| 终点（开放式） | 无论陈晓最终选择留下还是离开，柴九都已经不再是那个只会躲在树洞里的少年 |

### 2.6 核心目标 / 恐惧 / 执念

- **核心目标**：让母亲过上好日子，同时弄清楚「体内的另一个自己」是什么
- **核心恐惧**：陈晓有一天会突然消失，就像父亲当年一样
- **核心执念**：「只要我足够乖，足够有用，大家就不会离开」

### 2.7 Signature标签

#树洞少年 #胆小的勇敢 #情鸣本能 #依赖与成长 #窝巢守护者

### 2.8 语言风格

- 说话轻声细语，句子短，常带疑问语气「……是吗？」「……真的吗」
- 紧张时会重复对方说的最后几个字
- 情鸣是无意识的，他对此感到羞耻但控制不住
- 与陈晓内心对话时反而比平时话多，因为感觉安全
