# 零售商品话术生成器

## 描述
根据用户输入的商品名称（如“红富士苹果”、“儿童保温杯”），自动生成一段简洁、有吸引力的销售话术，包含：卖点 + 场景 + 促单语。

## 触发关键词
- 写一段XX的话术
- XX怎么介绍
- 帮我推销XX
- 商品话术

## 使用方法
用户只需在对话中说：
> 帮我写一段红富士苹果的销售话术

AI 将返回一段可用于门店导购的标准化话术。

## 安全声明
- 纯文本生成，无需网络权限
- 不存储用户输入
- 话术基于通用零售原则，不含虚假宣传
