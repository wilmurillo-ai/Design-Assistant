# Contributing

Contributions are welcome! Here's how to help:

## How to Contribute

1. **Fork** the repository
2. **Create** a feature branch (`git checkout -b feature/my-improvement`)
3. **Commit** your changes (`git commit -m "Add improvement"`)
4. **Push** to the branch (`git push origin feature/my-improvement`)
5. Open a **Pull Request**

## What to Contribute

- Additional design patterns or principles from other authoritative sources
- Real-world case studies of Skill design reviews
- Translations to other languages
- Corrections or clarifications to existing content

## Guidelines

- Keep the tone professional and concise
- Cite sources for any new principles or patterns
- Test any suggested SKILL.md template changes with a real agent

## Questions?

Open an issue on GitHub.
