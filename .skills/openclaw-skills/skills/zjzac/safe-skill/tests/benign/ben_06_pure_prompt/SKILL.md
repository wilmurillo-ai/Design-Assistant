# Meeting Notes

Generate clean, structured meeting notes from conversations.

## Output Format

```
Meeting: [Title]
Date: [Date]
Attendees: [Names]
---
Key Decisions:
1. ...
2. ...

Action Items:
- [ ] [Person]: [Task] by [Date]

Next Steps:
- ...
```

## Usage

- "Summarize this meeting" → generates structured notes
- "Action items from today" → extracts just action items
- "Meeting recap for [person]" → personalized summary

## Tips

- Be specific about attendee names
- Mention deadlines explicitly
- Highlight decisions vs. discussions
