# Map

> Most strategic failure comes from moving before the terrain is visible.

Map is a strategic cartographer for complex goals.

## What it does

This skill helps you:
- turn a vague goal into a navigable path
- identify milestones, bottlenecks, and dependencies
- surface risk zones and leverage points
- distinguish critical path from side paths
- choose a stronger route before execution begins

## What it does NOT do

- execute project management
- replace legal or financial sign-off
- guarantee success
- act as a task tracker or Gantt chart
