# Meeting Pro Examples

## 1. Pre-Meeting Briefing (Preparation Gap)
**Prompt:** "Prepare for my 2 PM sync with the Product Team."
**Response:**
### 🔍 MEETING EXECUTION DIAGNOSIS
- **Type**: Status Sync
- **Primary Gap**: Preparation Gap (No agenda linked to calendar invite)
- **ROI Rating**: Medium (High attendee cost, low clarity)

### 📝 PRE-MEETING BRIEF
- **Stakeholders**: Sarah (PM), Tom (Lead Dev), Leo (Design).
- **Institutional Memory**: In the last sync (March 4th), Tom promised the API docs by today.
- **Success Criteria**:
  1. Confirm if API docs are live.
  2. Decide on the 'V2' release date.
  3. Resolve the bottleneck in the Design hand-off.

## 4. Recurring Issue Tracking (Repeat Meeting Syndrome)
**Prompt:** "Are there any recurring blockers in my Weekly Team Syncs?"
**Response:**
### 🔍 MEETING EXECUTION DIAGNOSIS
- **Alert**: **Repeat Meeting Syndrome Detected.**
- **Issue**: "The Cloud Budget approval" has been discussed in the last 3 syncs without a decision.
- **Diagnosis**: Lack of authority in the room or missing data from Finance.
- **Tactical Advice**: Cancel the next sync until Finance provides the Q2 projections, or escalate to the CFO.
