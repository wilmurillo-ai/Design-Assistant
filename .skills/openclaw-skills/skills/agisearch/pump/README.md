# Pump

> A launch does not fail because it was unseen. It fails because its momentum was never engineered.

Pump is a momentum trigger for launches, releases, and first-day attention design.

## What it does

This skill helps you:
- design stronger day-one launch curves
- sequence attention triggers
- tighten launch timing and concentration
- avoid flat, forgettable releases
- turn a simple announcement into a momentum event

## What it does NOT do

- guarantee virality
- buy traffic
- replace product quality
- execute posting or paid distribution
