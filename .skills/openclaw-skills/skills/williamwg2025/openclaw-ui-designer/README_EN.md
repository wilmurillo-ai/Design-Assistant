---

## 🚀 Installation

```bash
cd ~/.openclaw/workspace/skills
# Skill installed at: ~/.openclaw/workspace/skills/ui-designer
chmod +x ui-designer/scripts/*.py
```

**That's it! All scripts included, no external cloning needed.**

---

## 📖 Usage

```bash
# Basic usage
python3 ui-designer/scripts/*.py

# Check help
python3 ui-designer/scripts/*.py --help
```
