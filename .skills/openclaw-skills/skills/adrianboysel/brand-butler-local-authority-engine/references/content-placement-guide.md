# Content Placement Article Guide

How to write SEO/AEO-optimized articles for paid link placements (Copify, guest posts, sponsored content, niche edits).

## Article Structure

### Requirements
- **Minimum:** 750 words (aim for 1,000-1,200 for maximum value)
- **Links:** Maximum 2 links to client site (unless placement allows more)
- **Anchor text:** Use target keyword phrases, not generic text
- **Format:** Rich text with H2 subheadings, paragraphs, occasional bullet lists

### Title (H1)
- Include the primary target keyword
- Include the location/city name
- Make it sound editorial, not promotional
- Example: "Why [City] Homeowners Should Never Ignore Early Signs of AC Trouble"

### Body Structure
```
H1: Title with primary keyword + location

Intro paragraph (100-150 words)
- Hook the reader with a local angle (climate, regional challenges, etc.)
- Establish the problem or topic

H2: Context section (150-200 words)
- Local/regional details that make the article feel authentic
- Climate, demographics, housing stock, or industry-specific factors

H2: Main educational content (200-300 words)
- Core information the reader came for
- Lists of signs, steps, tips, or considerations
- **Place Link 1 here** — naturally within a paragraph about the topic
  Example: "Homeowners who notice these issues should contact a trusted provider of [City AC repair](https://clientsite.com) services."

H2: Deeper value section (150-200 words)
- Hidden costs, long-term consequences, or industry insights
- Data points (DOE estimates, industry averages, etc.)

H2: How to choose a provider (150-200 words)
- What to look for in a reputable company
- Certifications, licensing, reviews, warranties
- **Place Link 2 here** — naturally within the selection advice
  Example: "A reputable [City heating and air](https://clientsite.com) company will provide transparent pricing and stand behind their work."

H2: Preventive advice or call-to-action (100-150 words)
- Actionable tips the reader can implement
- Smart thermostat, maintenance schedules, etc.
- Closing paragraph with urgency but not salesy
```

## Anchor Text Strategy

### Primary Anchor (Link 1)
Use the main target keyword phrase:
- "[City] AC repair"
- "[City] plumbing services"
- "[City] personal injury lawyer"

### Secondary Anchor (Link 2)
Use a related keyword variation:
- "[City] heating and air"
- "[City] emergency plumber"
- "[City] accident attorney"

### Rules
- Never use "click here", "visit website", or "learn more" as anchor text
- Never use the raw URL as anchor text
- Anchor text should read naturally within the sentence
- Surrounding context should be relevant to the target page
- Don't place both links in the same paragraph

## AEO (Answer Engine Optimization) Elements

AI answer engines (Perplexity, ChatGPT, Google AI Overviews) pull from content that:

1. **Answers questions directly** — use H2 headers that match conversational queries
   - "What Are the Signs Your AC Needs Repair?"
   - "How Much Does AC Repair Cost in [City]?"
   - "How to Choose an HVAC Company in [City]"

2. **Uses structured lists** — bulleted/numbered lists are easier for AI to extract

3. **Includes specific data** — costs, percentages, timelines, temperature ranges

4. **Covers local context** — city names, regional climate, local utility programs, neighborhood references

5. **Demonstrates expertise** — mention certifications, industry standards, building codes

## Special Requirements Field

Always submit these instructions with the placement order:

```
Meta Title: [Primary Keyword] | [City, State] Guide
Meta Description: [150-160 chars naturally including primary + secondary keywords]

Please keep the anchor text links exactly as written in the article:
1. "[Anchor Text 1]" → [client URL]
2. "[Anchor Text 2]" → [client URL]

Do not add any additional outbound links to the article.
Do not modify the anchor text of the existing links.
Category: [Relevant category for the host site]
No images needed — content only placement.
```

## Host Site Matching

Before writing, always:
1. **Read 2-3 existing articles** on the host site to match their:
   - Tone (formal vs conversational)
   - Article length
   - Header style
   - Use of lists vs paragraphs
   - Whether they use first person, second person, or third person
2. **Check their content guidelines** if available
3. **Match their category structure** — place the article in the most relevant section
