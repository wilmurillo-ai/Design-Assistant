# 训练模板库

基于权威训练体系整理的模板，按经验和目标分类。

## 新手计划（经验 < 6个月）

### 全身训练 A/B（3天/周）

适合：刚入门，建立基础力量

**训练 A**
| 动作 | 组数 | 次数 | 休息 |
|------|------|------|------|
| 深蹲 | 3 | 10 | 90s |
| 卧推 | 3 | 10 | 90s |
| 划船 | 3 | 10 | 60s |
| 肩推 | 2 | 12 | 60s |
| 平板支撑 | 3 | 30s | 45s |

**训练 B**
| 动作 | 组数 | 次数 | 休息 |
|------|------|------|------|
| 硬拉 | 3 | 8 | 120s |
| 上斜卧推 | 3 | 10 | 90s |
| 引体向上/高位下拉 | 3 | 8-10 | 60s |
| 弯举 | 2 | 12 | 45s |
| 卷腹 | 3 | 15 | 45s |

**安排：** 周一 A / 周三 B / 周五 A，下周轮换

## 中级计划（经验 6-24个月）

### 推拉腿分化（PPL）（3-6天/周）

适合：有一定基础，追求肌肥大

**推日（胸/肩/三头）**
| 动作 | 组数 | 次数 | 休息 |
|------|------|------|------|
| 卧推 | 4 | 8-10 | 90s |
| 上斜哑铃推举 | 3 | 10-12 | 60s |
| 哑铃侧平举 | 3 | 12-15 | 45s |
| 绳索下压 | 3 | 12-15 | 45s |
| 俯卧撑（收尾） | 2 | 力竭 | 45s |

**拉日（背/二头）**
| 动作 | 组数 | 次数 | 休息 |
|------|------|------|------|
| 引体向上/高位下拉 | 4 | 8-10 | 90s |
| 杠铃划船 | 3 | 8-10 | 60s |
| 坐姿划船 | 3 | 10-12 | 60s |
| 哑铃弯举 | 3 | 10-12 | 45s |
| 锤式弯举 | 2 | 12 | 45s |

**腿日（股四/股二/臀/小腿）**
| 动作 | 组数 | 次数 | 休息 |
|------|------|------|------|
| 深蹲 | 4 | 8-10 | 90s |
| 罗马尼亚硬拉 | 3 | 10-12 | 60s |
| 腿举 | 3 | 10-12 | 60s |
| 腿弯举 | 3 | 12-15 | 45s |
| 提踵 | 3 | 15-20 | 45s |

**安排：**
- 3天/周：推-拉-腿（休息日穿插）
- 4天/周：推-拉-腿-推（下周拉-腿-推-拉）
- 6天/周：推拉腿推拉腿

## 居家徒手计划

### 全身徒手（3-4天/周）

适合：无器械，居家训练

**训练内容**
| 动作 | 组数 | 次数 | 休息 |
|------|------|------|------|
| 深蹲 | 4 | 15-20 | 45s |
| 俯卧撑 | 4 | 10-15 | 45s |
| 弓步蹲 | 3 | 10/腿 | 45s |
| 平板支撑 | 3 | 30-45s | 30s |
| 臀桥 | 3 | 15 | 45s |
| 登山跑 | 3 | 20 | 30s |

**进阶变式：**
- 俯卧撑 → 钻石俯卧撑 / 弹起俯卧撑
- 深蹲 → 跳蹲 / 单腿深蹲
- 平板支撑 → 侧平板 / 动态平板

## 有氧/跑步计划

### 跑步基础（3-4天/周）

适合：减脂、提升心肺

**周计划**
| 天 | 内容 | 时长 | 强度 |
|----|------|------|------|
| 1 | 轻松跑 | 30min | 对话可说话 |
| 2 | 休息/交叉训练 | - | - |
| 3 | 间歇跑 | 20min | 高强度 |
| 4 | 休息 | - | - |
| 5 | 长距离慢跑 | 40-60min | 低强度 |
| 6-7 | 休息/轻活动 | - | - |

**间歇跑示例：**
- 热身 5min
- 快跑 1min + 慢跑 1min × 8组
- 冷身 5min

## 减脂专项

### 高强度间歇（HIIT）（3天/周）

适合：减脂为主，时间紧张

**训练内容**
循环以下动作，每个 30s，休息 15s，完成 3-4 轮：

1. 开合跳
2. 深蹲跳
3. 俯卧撑
4. 高抬腿
5. 波比跳
6. 登山跑

总时长：约 20 分钟

## 模板选择逻辑

```
IF 经验 = 新手 AND 场地 = 健身房:
    → 全身训练 A/B（3天）

IF 经验 = 中级 AND 目标 = 增肌:
    → PPL 分化（4-6天）

IF 经验 = 中级 AND 目标 = 力量:
    → 上肢下肢分化（4天）

IF 场地 = 居家:
    → 全身徒手计划

IF 目标 = 减脂 AND 时间紧张:
    → HIIT（3天）

IF 目标 = 减脂 AND 有氧偏好:
    → 跑步基础
```
