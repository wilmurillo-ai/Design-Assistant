## 动作选择原则

### 按目标选择

- **增肌**：复合动作优先，8-12次/组
- **力量**：大重量复合动作，3-6次/组
- **耐力**：轻重量高次数，15-20次/组
- **减脂**：复合动作+HIIT组合

### 按场地选择

- **健身房**：优先杠铃/哑铃/器械
- **居家**：自重动作+哑铃
- **户外**：跑步+徒手动作

### 按伤病调整

- 肩部问题 → 避免过头推举，用侧平举替代
- 膝盖问题 → 避免深蹲，用腿举/弓步替代
- 腰部问题 → 避免硬拉/划船，用器械替代

## 替换建议

用户不喜欢某个动作时，优先从同类型中选择替代：

```
不喜欢卧推 → 上斜哑铃推举 或 俯卧撑
不喜欢引体向上 → 高位下拉
不喜欢深蹲 → 高脚杯深蹲 或 腿举
不喜欢硬拉 → 罗马尼亚硬拉（更易掌握）
```
