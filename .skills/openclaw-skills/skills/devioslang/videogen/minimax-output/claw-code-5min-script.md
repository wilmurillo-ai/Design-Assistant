**类型**：知识讲解 · 动画/图示
**画面**：系统架构图动画，展示agent-harness结构
**旁白**：
"这个项目名叫claw-code，作者是韩国工程师Sigrid Jin。他研究这个领域已经很久了，有一个专门的名字——harness engineering。什么意思？AI模型像一个强大的引擎，但引擎本身不会开车，你需要一个 harness——也就是工具串联系统——才能真正把能力输出出来。Claude Code本质上就是这样一个harness：它连接AI大脑和你的文件系统、终端、浏览器，让AI真正替你写代码、跑命令、管项目。"

**旁白**：
"但他没有直接用泄露的代码。他做了一次clean-room rewrite——干净 room重写。这是工程里的常用手法：看原理，但自己实现，不复制一行代码。结果是，一套和Claude Code架构几乎一模一样的Python系统诞生了，工具、命令、会话管理、权限控制——全部从零重建。更有意思的是，他现在正在用Rust重写第二版，目标是把20万行代码压缩成更小、更快、更安全的本地CLI。"
