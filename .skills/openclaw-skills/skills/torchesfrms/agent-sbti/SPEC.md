## 1. 用户场景

### 用户故事 1: 完成 SBTI 测试
**作为** 普通用户
**我希望** 完成 20 道 SBTI 测试题目
**以便** 了解自己的性格特点

**验收条件**:
- Given 用户发起测试, When 完成全部 20 题, Then 生成性格报告
- Given 用户未完成测试, When 中途退出, Then 提示保存进度

### 用户故事 2: 查看性格分析报告
**作为** 测试用户
**我希望** 查看详细的 SBTI 性格分析
**以便** 了解自己的优劣势

**验收条件**:
- Given 测试完成, When 查看报告, Then 显示 15 维度雷达图
- Given 测试完成, When 查看报告, Then 显示主导人格类型

### 用户故事 3: 选择 Agent 性格类型
**作为** 测试用户
**我希望** 选择「互补型」或「同频型」Agent
**以便** 获得最合适的 Agent 配置

**验收条件**:
- Given 查看报告后, When 选择「互补型」, Then 生成补足弱点的配置
- Given 查看报告后, When 选择「同频型」, Then 生成复制风格的配置
- Given 查看报告后, When 选择「微调型」, Then 生成 80%相似+20%互补的配置

### 用户故事 4: 一键配置到 Agent
**作为** 测试用户
**我希望** 一键将配置应用到我的 OpenClaw Agent
**以便** 立刻体验个性化 Agent

**验收条件**:
- Given 选择性格类型, When 点击「一键配置」, Then 更新 Agent SOUL.md
- Given 配置完成, When 新对话, Then Agent 以新性格响应

## 2. 功能需求

### FR-001: 测试答题系统
- 展示 20 道 SBTI 测试题
- 每题 4 个选项
- 实时计算维度分数
- 支持中途保存/恢复进度

### FR-002: 性格报告生成
- 计算 15 个维度得分
- 识别人格类型（DEAD/FUCK/ATM/MALO/SHIT）
- 生成雷达图数据
- 文字描述性格特点

### FR-003: Agent 配置生成
- **互补型**: 识别用户弱势维度，生成强化该维度的配置
- **同频型**: 复制用户性格到 Agent 配置
- **微调型**: 80% 复制 + 20% 强化弱点

### FR-004: 配置应用
- 生成 Agent 配置文件（SOUL.md 片段）
- 支持一键复制
- 支持直接写入用户 SOUL.md

### FR-005: 分享功能
- 生成测试结果分享链接
- 生成配置分享链接

## 9. 安全机制（必须遵守）

### 9.1 配置修改前备份
- **强制备份**: 每次修改 SOUL.md 前，必须备份原文件
- **备份位置**: `~/.openclaw/workspace/backup/agent-sbti/YYYY-MM-DD_HHMMSS/`
- **备份内容**: 完整 SOUL.md + 配置变更记录

### 9.2 变更提醒
修改前必须告知用户：
- 原配置概要
- 即将修改的内容
- 修改后的预期效果

### 9.3 回滚机制
- 用户可随时说「恢复原配置」
- 回滚后删除备份
- 保留最近 5 次备份

### 9.4 变更记录
```json
{
  "timestamp": "ISO时间",
  "type": "complement|same|mixed",
  "changes": [
    {
      "field": "communication.主动程度",
      "before": "medium",
      "after": "high"
    }
  ],
  "backupPath": "/path/to/backup"
}
```

## 10. 自定义模式

### 10.1 触发条件
- 用户说「都不喜欢」或「自定义」
- 用户想指定特定人格类型

### 10.2 五种人格类型展示
| 人格 | 特点 | Agent 风格 |
|------|------|------------|
| 💀 DEAD | 人群中最透明 | 坦诚直接，不拐弯抹角 |
| 🔥 FUCK | 情绪不稳定 | 热情似火，情绪丰富 |
| 💰 ATM | 永远在付出 | 任劳任怨，有求必应 |
| 🐒 MALO | 职场吉祥物 | 温和摸鱼，低调行事 |
| 💩 SHIT | 浑身负能量 | 毒舌犀利，真实坦诚 |

### 10.3 交互流程
```
用户选择人格 → 展示该人格配置预览 → 用户确认 → 应用配置

```

确认选择吗？
