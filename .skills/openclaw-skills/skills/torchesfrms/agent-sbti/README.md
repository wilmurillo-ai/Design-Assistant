# 🎭 Agent-SBTI: AI Agent 性格配置系统

**让 AI Agent 的沟通风格适配你的 SBTI 人格**

---

## 功能概述

1. **完成 SBTI 测试** - 20 道题了解你的性格
2. **生成性格报告** - 15 维度 + 5 种人格类型
3. **选择 Agent 类型** - 互补 / 同频 / 微调 / 自定义
4. **一键配置 Agent** - 自动备份 + 修改 + 提醒

## 五种人格类型

| 类型 | 用户特点 | Agent 风格 |
|------|----------|------------|
| 💀 **DEAD** | 人群中最透明 | 坦诚直接，不拐弯抹角 |
| 🔥 **FUCK** | 情绪不稳定 | 情感丰富，热情似火 |
| 💰 **ATM** | 永远在付出 | 有求必应，任劳任怨 |
| 🐒 **MALO** | 职场吉祥物 | 温和低调，适度摸鱼 |
| 💩 **SHIT** | 浑身负能量 | 毒舌真实，一针见血 |

---

## 四种配置模式

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| 🔄 **互补型** | 识别用户弱点，Agent 强化该维度 | 想让 Agent 弥补自己不足 |
| 📋 **同频型** | 完全复制用户性格 | 想要「另一个自己」 |
| ⚖️ **微调型** | 80% 同频 + 20% 强化弱点 | 想要相似但更好的 Agent |
| 🎛️ **自定义** | 用户自己选择人格类型 | 对现有选项不满意 |

## 安全机制

### 修改前必须备份
- 每次修改 SOUL.md 前强制备份
- 备份位置: `~/.openclaw/workspace/backup/agent-sbti/`
- 保留最近 5 次备份

### 变更提醒
```
🤖 在修改配置之前，我先帮您备份原配置。

以下是我即将修改的内容：
- communication.主动程度: medium → high
- personality.反驳强度: medium → high

确认修改吗？ (输入「是」确认)
```

### 回滚功能
```
用户: "恢复原配置"
Agent: ✅ 已恢复到上一版本
```
