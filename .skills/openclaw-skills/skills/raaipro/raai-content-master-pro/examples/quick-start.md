# Контент-мастер PRO — Quick Start
**Старт без воды: 3 команды, чтобы быстро получить нормальный результат.**

## Сначала ответьте себе на 4 вопроса
1. Что Вы продаёте?
2. Кому?
3. Где будете публиковать?
4. Что хотите от текста: охват, доверие или заявку?

Если на это нет ответа, хороший контент не соберётся.

## Промпт 1: понять, о чём вообще говорить
```text
стратегия контента для [ваша ниша].
Продукт: [что продаёте].
Кому продаёте: [кто эти люди].
Где публикуете: Telegram / Instagram / email.
Цель: [охват / доверие / заявка].
Пиши простым языком, без маркетингового жаргона.
```

**Что получите:**
- понятные темы;
- нормальный тон;
- идеи, которые не стыдно показывать клиенту.

## Промпт 2: собрать план на неделю, а не на всю жизнь
```text
сделай контент-план на 7 дней.
Продукт: [что продаёте].
Аудитория: [кто они].
Канал: Telegram.
Цель недели: [охват / доверие / заявка].
Пиши так, чтобы обычный человек всё понял с первого раза.
```

**Что получите:**
- 7 тем на неделю;
- цель каждого поста;
- понятный следующий шаг.

## Промпт 3: написать один живой пост
```text
напиши Telegram-пост.
Тема: [тема].
Продукт: [что продаёте].
Для кого: [кто читает].
Цель: [охват / доверие / заявка].
Tone: [прямой / спокойный / живой].
Пиши простым языком для обычного собственника. Без терминов, которые надо объяснять.
```

**Что получите:**
- один нормальный пост;
- одну понятную мысль;
- один CTA без каши.

## Что делать дальше
1. Если пост получился сильный → `рерайт под форматы: [текст]`
2. Если темы слабые → `сделай 5 заголовков для: [тема]`
3. Если посты есть, а заявок нет → `разложи путь клиента до заявки по контенту`
4. Если нужна доверительная серия → `сделай 3 поста на доверие под [продукт]`
5. Если нужна продажа → `сделай 3 поста на заявку под [продукт]`

## Главное правило
Если текст не поймёт человек вне маркетинга — переписывать.
