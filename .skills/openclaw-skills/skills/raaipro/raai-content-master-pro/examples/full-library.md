# Контент-мастер PRO — Полная библиотека примеров
**Все 10 режимов. Полные промпты и ожидаемый вывод.** --- ## Быстрая навигация | # | Режим | Триггер |
|---|---|---|
| 1 | Контент-стратегия | `стратегия контента для [ниша]` |
| 2 | Контент-план на месяц | `контент-план на месяц: [ниша + площадки]` |
| 3 | SEO-ядро | `SEO-ядро для [ниша]` |
| 4 | Посты 7 форматов | `напиши [формат] про [тема]` |
| 5 | Рерайт → 5 форматов | `рерайт под форматы: [текст]` |
| 6 | Анализ конкурентов | `анализ конкурентов в [ниша]` |
| 7 | A/B заголовки | `5 заголовков для: [тема]` |
| 8 | Контент-воронка | `воронка контента для [продукт]` |
| 9 | Отчёт по контенту | `отчёт контент за [месяц]` |
| 10 | Email-цепочки | `email-цепочка: welcome для [ниша]` | --- ## РЕЖИМ 1: КОНТЕНТ-СТРАТЕГИЯ ### Полный триггер ```
стратегия контента для [ниша]:
[Описание бизнеса 2-3 предложения. Продукт, аудитория, главное преимущество.]
Площадки: [TG / IG / email / YouTube / VK]
Цель: [рост подписчиков / продажи / узнаваемость / лиды]
``` ### Примеры по нишам **Фитнес-тренер (личный бренд):**
```
стратегия контента для фитнес-тренера:
Тренирую женщин 30-50 лет после беременности, специализация — диастаз и восстановление.
Веду онлайн-группы по 15 человек, курс 3 месяца, стоимость 35 000 руб.
Площадки: TG + IG
Цель: лиды на групповые курсы — 10 заявок в месяц
``` **B2B: юридическая компания:**
```
стратегия контента для юридической компании:
Специализируемся на защите бизнеса: трудовые споры, договоры, регуляторные риски.
Клиенты — малый и средний бизнес, выручка 10-300 млн/год, собственники.
Площадки: TG + email
Цель: позиционирование как экспертов, 5 тёплых лидов/мес из контента
``` ### Ключевые элементы вывода - Анализ ниши: аудитория, конкуренты в контенте, незанятые ниши
- Позиционирование: экспертная позиция, уникальный угол, главный тезис
- TOV: 3 прилагательных, обращение, запрещённые/разрешённые слова, примеры
- Контент-микс: образовательный 40% / продающий 20% / вовлекающий 25% / личный 15%
- Ротация форматов по дням недели
- KPI: охват, ER, лиды --- ## РЕЖИМ 2: КОНТЕНТ-ПЛАН НА МЕСЯЦ ### Полный триггер ```
контент-план на [месяц год] для [ниша]:
Площадки: [TG / IG / email]
Контент-цель: [цель месяца]
[опционально] Инфоповоды: [праздники / события / запуски в этом месяце]
``` ### Пример ```
контент-план на июнь 2026 для маркетингового агентства:
Площадки: TG + IG
Контент-цель: прогрев к запуску нового продукта AI-аудит сайта 20 июня
Инфоповоды: конференция iForum 3-4 июня, лето — сезон рекламных кампаний
``` ### Ключевые элементы вывода - Таблица 30 публикаций: №, дата, площадка, формат, тема, колонка, этап воронки, цель, метрика
- Распределение TOFU/MOFU/BOFU по неделям
- Привязка к инфоповодам
- Вечнозелёные посты для переиспользования --- ## РЕЖИМ 3: SEO-ЯДРО ### Полный триггер ```
SEO-ядро для [ниша]:
Регион: [Россия / СНГ / конкретный город]
Формат контента: [статьи / посты / лонгриды]
[опционально] Главный конкурент: [домен или @канал]
``` ### Пример ```
SEO-ядро для онлайн-школы английского языка:
Регион: Россия
Формат: статьи для блога + посты TG
``` ### Ключевые элементы вывода - ВЧ/СЧ/НЧ запросы с частотой и интентом
- Кластеризация: 15-20 кластеров с основными и вспомогательными ключами
- LSI-слова для естественного использования
- Правила интеграции ключей в TG/IG/email
- Топ-5 кластеров для старта --- ## РЕЖИМ 4: ПОСТЫ — 7 ФОРМАТОВ ### Триггеры по форматам ```
напиши сторителлинг про [момент/история]: [контекст]
напиши кейс [ниша клиента]: [результат] за [срок]
напиши экспертный пост про [утверждение]
напиши провокационный пост: [тема/мнение против тренда]
напиши гайд/инструкцию: как [действие] [результат]
напиши личный пост про [момент из жизни/бизнеса]
напиши продающий пост для [продукт]: [оффер]
``` ### Пример: Кейс ```
напиши кейс маркетолог → агентство: увеличил выручку клиента с 500К до 2.3 млн/мес за 4 месяца.
Что делали: перестроили контент-стратегию, запустили email, исправили воронку.
Площадка: Telegram
TOV: деловой, цифры важны, без лишних слов
``` --- ## РЕЖИМ 5: РЕРАЙТ → 5 ФОРМАТОВ ### Полный триггер ```
рерайт под форматы: [вставьте исходный текст]
Площадки: TG + карусель IG + Reels + Stories + Email
TOV: [ваш стиль]
``` ### Пример ```
рерайт под форматы:
"Каждый раз когда маркетолог говорит 'нам нужно больше контента' — я спрашиваю: а зачем? Контент без стратегии — это трата времени. 3 года назад мы публиковали 2 поста в день и получали 12 заявок/мес.
Сейчас 5 постов в неделю и 40 заявок/мес. Меньше — лучше, когда есть система." Площадки: TG + карусель IG + Reels + Stories + Email
TOV: прямой, экспертный, без воды
``` ### Что получите 5 адаптированных форматов:
- Пост для Telegram (до 1500 символов, жирный для акцентов)
- Карусель Instagram (8-10 слайдов с текстом на каждом)
- Reels-скрипт (60 секунд: крючок 3 сек + контент 45 сек + CTA 10 сек)
- Stories (5 карточек с дизайн-рекомендацией)
- Email (тема + прехедер + тело + CTA) --- ## РЕЖИМ 6: АНАЛИЗ КОНКУРЕНТОВ ### Полный триггер ```
анализ конкурентов в [ниша]:
Конкуренты: [@канал1, @канал2, @канал3]
[или] Найди топ-3 конкурентов в нише [ниша] и разбери их контент
``` ### Пример ```
анализ конкурентов в нише онлайн-образование для предпринимателей:
Конкуренты: @business_school_ru, @marketing_tg_best, @ceo_practice
``` --- ## РЕЖИМ 7: A/B ЗАГОЛОВКИ ### Полный триггер ```
5 заголовков для: [тема поста]
Площадка: [TG / IG / email]
Аудитория: [холодная / тёплая / смешанная]
``` ### Пример ```
5 заголовков для: почему маркетологи тратят время впустую
Площадка: TG
Аудитория: маркетологи и собственники малого бизнеса
``` ### Что получите 5 вариантов с типом (цифра/провокация/история/вопрос/результат) + CTR-прогноз + рекомендация по аудитории --- ## РЕЖИМ 8: КОНТЕНТ-ВОРОНКА ### Полный триггер ```
воронка контента для [продукт]:
Целевое действие BOFU: [покупка / заявка / звонок / регистрация]
Средний цикл сделки: [дни/недели]
``` ### Пример ```
воронка контента для онлайн-курса по финансовой грамотности:
Целевое действие: покупка курса за 12 000 руб
Средний цикл: 3-4 недели от знакомства до покупки
``` --- ## РЕЖИМ 9: ОТЧЁТ ПО КОНТЕНТУ ### Полный триггер ```
отчёт контент за [месяц год]:
Данные: [вставьте цифры из аналитики — охваты, ER, подписчики, заявки]
``` ### Пример ```
отчёт контент за апрель 2026:
Публикаций: 22 из 30 запланированных
Средний охват TG: 650 (план: 800)
ER TG: 9.2% (план: 8%)
Новых подписчиков TG: 180 (план: 200)
Заявок из контента: 18 (план: 20)
Топ-пост: 15 апреля, кейс "как за 2 месяца контент начал приносить заявки", охват 2100
Худший пост: 8 апреля, обучающий про SEO, охват 280
``` --- ## РЕЖИМ 10: EMAIL-ЦЕПОЧКИ ### Триггеры по типам ```

email-цепочка: реактивация для [ниша] # 3 письма, 7 дней
``` ### Пример: Welcome ```
email-цепочка: welcome для онлайн-школы кулинарии:
Лид-магнит: чек-лист "10 техник которые используют шеф-повара"

TOV: тёплый, как письмо от подруги-повара
