## Быстрые запросы без жаргона
### 1. Понять, что писать
```
стратегия контента для [ниша].
Продукт: [что продаёте].
Аудитория: [кто эти люди].
Пиши простым языком.
```

### 2. Собрать план на неделю или месяц
```
сделай контент-план на [7 дней / месяц].
Продукт: [что продаёте].
Канал: Telegram / Instagram / email.
Цель: [охват / доверие / заявка].
```

### 3. Написать один пост
```
напиши [экспертный / продающий / личный / кейс] пост про [тема].
Для кого: [кто читает].
Пиши без маркетинговых терминов.
```

### 4. Усилить слабый заход
```
сделай 5 заголовков для темы: [тема].
Нужно, чтобы человек понял смысл с первой строки.
```

### 5. Разложить один сильный текст на каналы
```
рерайт под форматы: [вставить текст].
Сделай Telegram, Instagram и email.
```

### 6. Понять, почему контент не приводит к заявкам
```
разложи путь клиента до заявки по контенту для [продукт].
Где прогрев, где доверие, где продажа?
```

### 7. Запустить welcome-письма
```
сделай welcome-цепочку для [ниша].
Что человек должен понять и сделать после подписки?
```
