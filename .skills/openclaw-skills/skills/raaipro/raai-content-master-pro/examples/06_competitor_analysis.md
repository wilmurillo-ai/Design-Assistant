# Пример 6: Анализ конкурентов — онлайн-школа дизайна интерьеров **Запрос:**
```
анализ конкурентов: онлайн-школа дизайна интерьеров, конкуренты: @design_school_pro, @interior_academy, @home_design_online
``` --- **Вывод агента:** ```
🔎 АНАЛИЗ КОНКУРЕНТОВ: Онлайн-школа дизайна интерьеров ═══ СРАВНИТЕЛЬНАЯ ТАБЛИЦА ═══ | Параметр | @design_school_pro | @interior_academy | @home_design_online | ВЫ (текущее) |
|-----------------------|-------------------|-------------------|---------------------|--------------|
| Подписчики TG | 12 400 | 8 200 | 5 600 | 3 800 |
| Подписчики IG | 34 000 | 18 500 | 9 200 | 6 100 |
| Частота постов/нед | 7 | 4 | 3 | 2-3 |
| ER Telegram | 7.2% | 11.5% | 8.8% | 5.4% |
| ER Instagram | 3.1% | 4.8% | 2.9% | 2.2% |
| Форматы контента | Reels, посты | Карусели, видео | Посты, Stories | Посты |
| Email-рассылка | Да (2/нед) | Да (1/нед) | Нет | Нет |
| TOV | Вдохновляющий | Экспертный | Дружеский | Нет единого |
| Контент-воронка | Есть (TOFU/MOFU) | Частично | Нет | Нет |
| Лид-магнит | "100 ошибок" | "Гайд по стилям" | Нет | Нет | ═══ ДЕТАЛЬНЫЙ АНАЛИЗ ═══ ▎@design_school_pro (лидер рынка)
Сильные стороны: • Высокая частота — 7 постов/нед, аудитория привыкла к ритму • Reels с интерьерами "до/после" — вирусный формат, средний reach 15K • Лид-магнит "100 ошибок в дизайне интерьера" — конвертирует в email-базу • Email 2 раза/нед: полезный + продающий (чередование)
Слабые стороны: • TOV размытый — то вдохновляющий, то продающий, нет единства • Карусели слабые — много текста, мало визуала • Нет SEO-контента — блог мёртвый, органика нулевая • Контент-воронка неполная — нет BOFU, продажа через прямой оффер ▎@interior_academy (нишевый эксперт)
Сильные стороны: • Высокий ER в TG (11.5%) — лояльная аудитория, активные комментарии • Экспертный контент — разборы проектов с чертежами и расчётами • Карусели-гайды — "5 правил планировки кухни", "Как выбрать диван" • Чёткий TOV — экспертный, без лишнего
Слабые стороны: • Низкая частота (4/нед) — теряют reach • Нет Reels — упускают вирусный формат • Продающий контент — только "запись открыта", без прогрева • Email раз в неделю — мало для nurture-цепочки ▎@home_design_online (нишевый новичок)
Сильные стороны: • Дружеский TOV — ощущение общения с подругой • Stories активны — вопросы, опросы, закулисье • Быстрый рост TG (+300/мес при базе 5600)
Слабые стороны: • Нет системы — темы случайные, нет плана • Нет email — теряет подписчиков после отписки • Нет воронки — контент "для вовлечения", продаж из него мало • Визуал слабый — стоковые фото вместо реальных проектов ═══ НЕЗАНЯТЫЕ НИШИ (ВАШИ ВОЗМОЖНОСТИ) ═══ 1. SEO-контент для блога — ни один конкурент не использует Потенциал: 2-5К органических визитов/мес на запросы "дизайн [комната]", "как расставить мебель", "стиль [название]" 2. Email-цепочки nurture — есть только у лидера, и те слабые Потенциал: welcome + nurture + реактивация = рост конверсии 3. Карусели с чертежами "до/после" — @interior_academy делает, но без визуала результата. Совместите чертёж + 3D + фото 4. Reels-разборы интерьеров — @design_school_pro делает "до/после", но не объясняет ПОЧЕМУ. Формат "разбор + обучение" свободен 5. Контент-воронка TOFU/MOFU/BOFU — ни у кого нет полной. Вы можете быть первыми с системным путём подписчик→студент ═══ РЕКОМЕНДАЦИИ ═══ Приоритет 1 (месяц 1): • Определить TOV и зафиксировать в стратегии • Запустить контент-план 5 постов/нед • Создать лид-магнит (конкурировать с "100 ошибок") • Запустить welcome email-цепочку Приоритет 2 (месяц 2): • Запустить Reels с разборами (формат "почему это работает") • SEO-контент: 4 статьи по приоритетным кластерам • Карусели: чертёж + 3D + фото результата Приоритет 3 (месяц 3): • Воронка TOFU/MOFU/BOFU — полная интеграция • A/B тесты заголовков • Nurture email-цепочка
``` --- **Признак сильного выхода:** Анализ содержит сравнительную таблицу по 10+ параметрам для 3 конкурентов + текущее состояние клиента. Каждый конкурент разобран: сильные/слабые стороны с конкретикой. Выделены 5 незанятых ниш. Рекомендации приоритизированы по месяцам.
