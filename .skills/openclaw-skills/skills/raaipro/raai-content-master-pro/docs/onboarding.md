# Путь внедрения Контент-мастер PRO ## Общий принцип Внедрение идёт по спирали: **стратегия → производство → аналитика → оптимизация**. Не нужно внедрять всё сразу — каждый этап даёт самостоятельный результат. --- ## День 1: Фундамент ### Шаг 1. Заполнить config.yaml
- Ниша, продукт, цены, аудитория
- Каналы: какие площадки ведёте, с какими KPI
- TOV: как разговариваете с аудиторией
- Конкуренты: 3 основных **Без config.yaml система работает "в общем". С ним — под ваш бизнес.** ### Шаг 2. Контент-стратегия
```
стратегия контента для [ниша]: [описание бизнеса 2-3 предложения]
```
Получаете:
- Анализ ниши и конкурентов
- Позиционирование в контенте
- Tone of Voice (голос бренда)
- Контент-микс с пропорциями
- KPI по каналам ### Шаг 3. SEO-ядро
```
SEO-ядро для [ниша]
```
Получаете:
- Ключи с частотностью и конкуренцией
- Кластеры с приоритетами
- LSI-ключи для каждого кластера
- План интеграции в контент ### Шаг 4. Первый пост по шаблону
```
напиши кейс про [ваш результат]
```
Увидеть формат, качество и скорость работы. **Результат дня 1:** стратегия + TOV + SEO-ядро + первый пост. Это 80% работы, которую агентство делает за 2-4 недели. --- ## Неделя 1: Запуск производства ### День 2-3. Контент-план на месяц
```
контент-план на [месяц] для [ниша], площадки: TG + Instagram + email
```
30 постов с темами, форматами, привязкой к воронке и KPI. ### День 3-4. Набить руку на шаблонах
Написать 5-7 постов по разным форматам:
- `напиши сторителлинг про [тема]`
- `напиши экспертный про [тема]`
- `напиши продающий про [оффер]`
- `напиши кейс про [результат]` Каждый пост — 20-30 минут вместо 2 часов. ### День 4-5. Анализ конкурентов
```
анализ конкурентов: [ниша или @ник, @ник, @ник]
```
Сравнительная таблица + незанятые ниши + рекомендации. ### День 5-6. Первый рерайт ×5
```
рерайт под форматы: [лучший пост недели]
```
Один текст → TG + карусель + Reels + Story + email. 15 минут — и 5 единиц контента. ### День 7. Welcome email-цепочка
```
email-цепочка: welcome для [ниша], лид-магнит: [название]
```
5 писем для новых подписчиков: знакомство → ценность → доверие → оффер. **Результат недели 1:** план на месяц + 5-7 постов + конкуренты + рерайт + email-welcome. Контент-машина запущена. --- ## Месяц 1: Система в работе ### Недели 2-3. Производство по плану
- Публикация 5 постов/нед по контент-плану
- Рерайт каждого поста в 5 форматов
- Welcome-цепочка работает для новых подписчиков
- A/B тесты заголовков (2-3 теста/нед):
```
5 заголовков для: [тема поста]
``` ### Неделя 3. Контент-воронка
```
воронка контента для [ваш продукт]
```
TOFU/MOFU/BOFU с темами, форматами и CTA. Каждый пост получает роль в воронке. ### Неделя 4. Первый отчёт
```
отчёт контент за [месяц]: [данные по каналам]
```
План/факт, ER по каналам, топ-посты, антитоп, рекомендации. ### Неделя 4. Nurture email-цепочка
```
email-цепочка: nurture для [ниша]
```
5 писем прогрева: экспертность → кейсы → доверие → мягкий оффер. **Результат месяца 1:**
- 30 постов вышли по плану и стратегии
- Воронка TOFU/MOFU/BOFU запущена
- Email-цепочки (welcome + nurture) работают
- Первый отчёт с цифрами и рекомендациями
- Время на контент: 15 ч/нед → 7 ч/нед --- ## Квартал 1: Оптимизация и масштабирование ### Месяц 2. A/B оптимизация
- Тестировать заголовки каждую неделю
- Определить лучшие форматы по ER и конверсии
- Оптимизировать контент-микс по данным отчёта
- Запустить продающую email-цепочку для нового продукта ### Месяц 2. SEO-контент
- 4 статьи по приоритетным кластерам из SEO-ядра
- Рерайт каждой статьи в посты для TG и IG
- Внутренняя перелинковка между статьями
- Первые результаты: рост позиций по целевым запросам ### Месяц 3. Масштабирование
- Реактивация email-базы:
```
email-цепочка: реактивация для [ниша]
```
- Масштабирование контента: помощник/копирайтер с TOV-гайдом
- Квартальный отчёт: динамика ER, подписчиков, лидов, выручки
- Корректировка стратегии на Q2 **Результат квартала 1:**
- Контент-машина работает автономно
- SEO-контент даёт органику
- Email-цепочки прогревают и конвертируют
- A/B тесты оптимизировали заголовки и CTA
- Контент-воронка генерирует лиды предсказуемо
- Время на контент: 5 ч/нед (с помощником — 2-3 ч/нед) --- ## Чек-лист внедрения ### День 1
