# AI-Аналитик бизнеса PRO — Quick Start (3 первых промпта) Скопируйте любой из трёх сценариев и отправьте боту/агенту. Замените [квадратные скобки] на ваши данные. --- ## Сценарий 1 — Дашборд за 30 секунд ```
выручка: [сумма] руб, расходы: [сумма] руб, клиентов: [N], средний чек: [сумма] руб,
новых клиентов: [N], ушло клиентов: [N], CAC: [сумма] руб, реклама: [сумма] руб дашборд
``` **Что получите:** дерево метрик Revenue → ARPU × Users, светофор KPI, аномалии, 1 топ-действие. --- ## Сценарий 2 — Unit-экономика за 2 минуты ```
Период: [месяц]
MRR: [сумма] руб
Новых клиентов: [N], ушло: [N]
Активных клиентов: [N]
Бюджет на рекламу: [сумма] руб unit экономика
``` **Что получите:** LTV, CAC, LTV/CAC с диагнозом (✅/⚠️/❌), Payback Period, Churn Rate, рекомендации. --- ## Сценарий 3 — Финансовая модель (P&L) ```
Выручка за [месяц]: [сумма] руб
Себестоимость: [сумма] руб
Маркетинг: [сумма] руб
Зарплаты: [сумма] руб
Серверы: [сумма] руб
Прочие: [сумма] руб финмодель
``` **Что получите:** полный P&L с Gross Margin / EBITDA / Net Margin, сравнение с прошлым периодом, ключевые коэффициенты. --- ## Следующий шаг После первого запуска читайте `examples/full-library.md` — там все 12 режимов с примерами данных.
