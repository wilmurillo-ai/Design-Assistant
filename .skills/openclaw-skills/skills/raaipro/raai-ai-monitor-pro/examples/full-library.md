``` --- ## Советы по работе - **Строй + ИТ одновременно**: можно мониторить и объект, и инфру в одном чате.
- **Чем больше данных — тем точнее ответ**: давайте реальные показатели, не «что-то высокое».
- **Алёрты настраивайте в config.yaml** — тогда система сама скажет когда превышен порог.
- **Post-mortem после каждого CRITICAL** — через 48 часов. Discipline = prevention.
