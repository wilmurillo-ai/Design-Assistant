# AI-Поддержка PRO — полная библиотека промптов Все 10 режимов работы с готовыми промптами под разные ниши. --- ## РЕЖИМ 1 — Категоризация обращений ### E-commerce (возврат + злой клиент)
```
Обработай обращение клиента по правилам ai-support-pro: Клиент: Марина Соколова, постоянный клиент, 3 заказа, LTV 12 500 руб
Канал: чат на сайте
Текст: "Мне пришёл сломанный телефон. Требую НЕМЕДЛЕННО вернуть деньги или заменить товар. Фото прилагаю. Если сегодня не решите — пишу в Роспотребнадзор и на всех отзовиках." Определи: тип, приоритет, sentiment, маршрут, SLA-дедлайн.
Дай готовый первый ответ клиенту.
``` ### SaaS (технический инцидент)
```
Обработай обращение по правилам ai-support-pro: Клиент: ООО АльфаЛогистика, B2B, LTV 85 000 руб
Тип: VIP корпоративный клиент
Канал: email
Текст: "У нас не работает API интеграция с вашей платформой с 9:00 утра.
Ошибка 500 на всех запросах. Это блокирует нашу работу." Категоризируй, расставь приоритет, определи маршрут.
``` ### Услуги (жалоба на сотрудника)
```
Обработай обращение по правилам ai-support-pro: Канал: Telegram
Текст: "Ваш мастер Дмитрий был сегодня у нас. Нагрубил, опоздал на час
без предупреждения и оставил мусор после работы. Это недопустимо." Все шаги категоризации + готовый ответ.
``` --- ## РЕЖИМ 2 — SLA-трекинг ### Дневной дашборд
```

P2: 8 тикетов (3 открыты >6ч, SLA 8ч)
P3: 24 тикета (7 открыты >20ч, SLA 24ч)
P4: 13 тикетов Дай: статус, горящие, риски, рекомендации по приоритизации на вечер.
``` ### Еженедельный отчёт
```

``` --- ## РЕЖИМ 3 — Эскалация ### Ручная эскалация на L2

```
