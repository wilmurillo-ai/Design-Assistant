# Шаблон клиентского отзыва — AI-Продажник PRO > Шаблон для получения отзыва от РОПа или собственника через 30-90 дней использования. Кирилл может использовать как есть или адаптировать. --- ## Когда запрашивать | Точка | Что спрашиваем |
|---|---|
| **День 7** (первая неделя) | Первое впечатление, что уже заработало, что не получилось |
| **День 30** (первый месячный отзыв) | Конкретные цифры: сколько сделок добавилось, насколько вырос follow-up return rate |
| **День 90** (полный квартал) | Финальный отзыв с рублёвыми эффектами для proof/case | --- ## Вопросник (отправить в Telegram/Email) ```

Менеджеры реально пользуются скриптами или только вы? [ ] Только я (РОП/собственник) [ ] Команда использует скрипты звонков [ ] Команда + я используем несколько режимов 3. Как изменилась конверсия лид→сделка за этот период? [ ] Выросла (на ___%) [ ] Не изменилась [ ] Пока сложно оценить 4. Сколько «тихих» лидов (не ответивших на КП) вернула follow-up цепочка? ``` --- ## Шаблон кейса на основе отзыва ```markdown
# Proof 0X — [Ситуация одним предложением] ## Ситуация до РОП [отрасль] (команда [N] менеджеров, выручка [X] млн ₽/мес) столкнулся с [проблема]. [2-3 абзаца деталей] ## Что внедрили - Режим [X] для [задача]
- Режим [Y] для [задача]
- Настройка config.yaml под нишу [отрасль] ## Ситуация после Через [N] дней:
- [Изменение 1 с цифрами]
- [Изменение 2 с цифрами] ## Эффект - [Главное достижение]
- [Второе достижение]
- За [период]: [рублёвый эффект] ## ROI в рублях **Исходные данные:** выручка [X] млн ₽/мес, средний чек [Y] 000 ₽. | Источник | Расчёт | Сумма |
|---|---|---|
| Дополнительные сделки | +[N] сделок × [Y] ₽ = | **[сумма] ₽/мес** |
| Сокращение цикла | [старый] → [новый] дней, -[%] = | **[сумма] ₽/мес** |
| Follow-up возврат | [N] из [M] тихих лидов вернулись = | **[сумма] ₽/мес** | **Итого за месяц:** [сумма] ₽.

``` --- ## Этические правила 1. Письменное согласие клиента до публикации имени/компании.
2. Анонимно — «РОП IT-агентства с выручкой 8 млн ₽/мес».
3. Не редактировать цифры — если клиент сказал «+12%», не писать «+20%».
4. Не выдумывать отзывы. --- ## Хранение отзывов ```
