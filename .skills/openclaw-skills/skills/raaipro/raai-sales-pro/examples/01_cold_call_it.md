# Пример 1: Холодный звонок — IT-автоматизация для стоматологий **Запрос:**
```
скрипт звонка: холодный, продаю AI-бот для записи клиентов, ниша — стоматологии, цена 12 000 руб/мес
``` --- **Вывод агента:** ```
📞 СКРИПТ: Холодный звонок — AI-бот для записи, стоматология
Цена: 12 000 руб/мес ПОДГОТОВКА:
• Сайт клиники: проверить есть ли онлайн-запись, сколько врачей
• Триггер: клиника без онлайн-записи или с неудобной формой
• Цель: назначить 20-минутное демо ОТКРЫТИЕ:
«Марина Сергеевна, добрый день! Это Алексей из RAAI.
Вижу, что у вас активная клиника — нашёл через поиск по [город].
Звоню потому что мы автоматизируем запись пациентов для стоматологий:
бот принимает звонки и сообщения 24/7, записывает без участия администратора.
Клиники экономят 3-4 часа работы ресепшн в день.
Удобно сейчас буквально 2 минуты?» → ДА: перейти к SPIN-блоку SPIN-БЛОК:
S: «Как сейчас организована запись — только по телефону или есть онлайн?»
P: «Что чаще всего не успевает администратор в часы пик?»
I: «Случается что пациент не дозвонился и ушёл к конкурентам?»
N: «Если бы запись работала автоматически 24/7 — сколько пациентов в день это даёт дополнительно?» ПРЕЗЕНТАЦИЯ:
«Понял. Именно для этого наш бот — он отвечает на звонки и в WhatsApp одновременно,
записывает в вашу МИС, отправляет напоминания. Без участия администратора.
Клиника "ДентаПлюс" в Казани за первый месяц получила +23 дополнительных записи.
Покажу как это выглядит — демо 20 минут, удобно завтра после 14:00?» → ДОРОГО (12 000):
«Понял. 12 000 / 30 дней = 400 руб в день. Одна дополнительная запись у вас сколько стоит — 3 000?
При +23 записях в месяц — ROI 17x. Стоит попробовать?» → УЖЕ ЕСТЬ ЗАПИСЬ ОНЛАЙН:
«Отлично! А в WhatsApp и по телефону в нерабочее время кто отвечает?»
``` --- **Признак сильного выхода:** Скрипт содержит полный SPIN-блок (4 вопроса с переходами), минимум 2 проработанные ветви возражений с ROI-аргументом, конкретный следующий шаг (демо) с альтернативным закрытием по времени.
