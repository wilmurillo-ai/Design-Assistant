# AI-Продажник PRO — Quick Start (3 промпта для первого запуска) > Эти 3 запроса дадут результат за первые 15 минут. Без настройки — сразу после установки. --- ## Промпт 1 — первый скрипт звонка Скопируйте и замените `[ваш продукт]` на название вашего продукта или услуги. ```
скрипт звонка: [ваш продукт] Ниша: [укажите — IT-услуги / юридические услуги / производство / агентство]
Целевой клиент: [собственник / генеральный директор / РОП]
Текущая боль клиента: [например — теряет клиентов, нет системы, конкуренты обгоняют]
Средний чек: [укажите — например, 60 000 руб]
``` **Что получите:** структурированный скрипт звонка с 8 ветвлениями под каждое типичное возражение. Готов к использованию менеджерами сразу. --- ## Промпт 2 — квалификация первого лида ```
квалифицируй: [краткое описание лида] Компания: [название / описание]
Что говорит: [цитата или резюме разговора]
Откуда: [источник — сайт / холодный звонок / рекомендация]
Бюджет: [что известно]
Срочность: [есть / нет / неизвестно]
``` **Что получите:** BANT+MEDDIC скоринг лида по 8 критериям с оценкой от 0 до 10 и рекомендацией — работать активно, переместить в nurture или дисквалифицировать. --- ## Промпт 3 — обработка типичного возражения ```
возражение: [текст возражения клиента] Контекст: [стадия сделки — первый контакт / после КП / при дожиме]
Что предлагаете: [1 строка — ваш продукт]
Что уже сказали клиенту: [2-3 предложения]
``` **Примеры возражений для старта:**
- `возражение: дорого`
- `возражение: нам это не нужно`
- `возражение: подумаем, перезвоним`
- `возражение: у нас уже есть CRM / решение`
- `возражение: сейчас не время, после нового года` **Что получите:** диагноз типа возражения + 3 варианта ответа + ветвление что делать если каждый не сработал. --- ## Следующий шаг после Quick Start Заполните `config.yaml` — особенно:
- `business.product` — ваш продукт
- `business.niche` — ваша ниша
- `sales_metrics.monthly_leads` — сколько лидов в месяц
- `unit_economics.avg_deal_size` — средний чек После этого все режимы будут работать с вашим контекстом, а не с примерами. **Полная библиотека режимов** → `examples/full-library.md`
