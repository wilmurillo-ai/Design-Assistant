# AI-Сметчик PRO — Ограничения и честные границы Что продукт НЕ делает. Где он НЕ поможет. Что нужно на вход.
В строительстве ошибка стоит деньги. Честность про ограничения = защита от ошибки. --- ## Где AI-Сметчик PRO НЕ поможет ### 1. Не заменяет замеры и техобследование Смета — функция от объёмов. Объёмы — функция от замеров.
Если замеры не сделаны или сделаны криво — смета будет неверной независимо от методологии. AI-Сметчик PRO считает правильно при правильных вводных.
Правильные вводные = замеры, техническое задание, подтверждённый состав работ. **Аналогия:** навигатор не поможет, если стартовая точка указана неверно. ### 2. Не работает без состава работ Без понимания, что именно надо сделать — невозможно посчитать сколько это стоит. Минимум для начала расчёта:
- тип объекта (квартира, офис, дом)
- площадь и основные помещения
- состав работ (хотя бы укрупнённо: отделка / электрика / сантехника)
- регион Без этого система предложит структуру, но не считает конкретику. ### 3. Не спасает при хаосе изменений без фиксации Самая частая причина разногласий на объекте: одна сторона помнит «договорились так», другая — «договорились иначе». Все изменения состава работ должны:
- фиксироваться письменно (доп. соглашение или акт)
- отражаться в корректировке сметы Без фиксации изменений — смета устаревает в день подписания. ### 4. Не вычислит объёмы из описания «на глаз» «Большая квартира», «нормальный офис», «примерно 100 квадратов» — не вводные для сметы. Нужны конкретные данные:
- площадь по помещениям
- высота потолков
- периметры стен
- количество точек (электрика, сантехника) Если нет замеров — начните с объёмной ведомости: `объёмы` ### 5. Не гарантирует выигрыш тендера Хорошая смета с обоснованием — сильный аргумент. Но заказчик принимает решение по многим критериям: репутация, сроки, гарантии, личное доверие. AI-Сметчик PRO усиливает экономическую аргументацию. Остальное — за компанией. ### 6. Не заменяет опытного сметчика в сложных объектах Для стандартных работ (отделка, электрика, сантехника в жилье и офисах) — система покрывает 90%+ задач. Для сложных объектов (промышленность, специальные конструкции, уникальные технологии) — нужен профессиональный сметчик с экспертизой в нише. --- ## Garbage in → Garbage out Самый опасный паттерн: ```
Шаг 1: Ввели состав работ «примерно» и объёмы «на глаз»
Шаг 2: Получили смету
Шаг 3: Подписали контракт
Шаг 4: По ходу выяснилось, что объёмов в 1.5 раза больше
Шаг 5: Конфликт, переделки, потери маржи
``` Система посчитала правильно. Вводные были неверными. **Правило:** лучше потратить 2 часа на замеры и получить точную смету, чем 2 минуты на приблизительную и потерять 200К на объекте. --- ## Что нужно на вход ### Для сметы: | Вводные | Обязательно? |
|---|---|
| Тип объекта и его площадь | Да |
| Состав работ (хотя бы укрупнённо) | Да |
| Регион и ценовая база | Да |
| Объёмы по помещениям (замеры) | Желательно, для точности |
| Высоты, периметры, количество точек | Для специфических позиций | ### Для аудита: | Вводные | Обязательно? |
|---|---|
| Смета в текстовом/табличном формате | Да |
| Регион объекта | Да |
| Тип работ | Да | ### Для КС-2/КС-3: | Вводные | Обязательно? |
|---|---|
| Подписанная смета или её основа | Да |
| Фактически выполненные работы и объёмы | Да |
| Даты выполнения | Да | --- ## Антипаттерны в строительных сметах ### «Считаем от «цены за квадрат»» Ориентировочная цена за квадрат — инструмент на этапе «брать или не брать объект».
Смета для договора и контроля — другой уровень детализации.
Путать одно с другим → либо потерять тендер, либо работать в убыток. ### «НР применяем ко всей сметной стоимости» Стандартная ошибка: накладные расходы и сметная прибыль начисляются на ФОТ рабочих и механизаторов (МДС 81-33.2004), а не на всю прямую стоимость.
Ошибка даёт завышение НР в 2.5–3 раза от правильного значения. ### «Документы делаем по завершении» КС-2 задним числом — это юридический риск.
Акты скрытых работ после сдачи — невозможно: скрытые работы уже закрыты.
Дефектовка после приёмки — у заказчика теперь 12 замечаний. Документация ведётся по ходу объекта, а не в конце. ### «Аудируем только большие сметы» Завышения есть в сметах любого размера.
На объекте 500К найти 5% завышение = 25 000 руб = 83% стоимости продукта.
Аудитировать стоит всё, что >300 000 руб. --- ## Честные границы по каждому режиму | Режим | Что делает хорошо | Где ограничен |
|---|---|---|
| Локальная смета | Структура ПЗ → НР → СП → НДС | Нужны объёмы и состав работ |
| Материалы | Расход с потерями и запасом | Нужны конкретные объёмы |
| Аудит | Находит системные завышения | Нужна смета в структурированном виде |
| Объёмная ведомость | Считает геометрию | Нужны замеры |
| КС-2 / КС-3 | Шаблоны по унифицированной форме | Нужны факт работы и даты |
| Тендер / КП | Структура и обоснование | Нужны данные объекта и расценки |
| Дефектовка | Стандартные позиции по НР | Нужно физическое обследование | --- ## Резюме: формула успеха ```
Результат = Качество вводных × Регулярность аудита × Применение документов Где: Качество вводных = замеры + состав работ + регион Регулярность аудита = каждая смета >300К проверяется Применение документов = КС-2, дефектовка, объёмная ведомость ведутся ПО ХОДУ Если хотя бы один множитель ≈ 0, результат ≈ 0.
``` AI-Сметчик PRO — честный инструмент.
Он не обещает идеальную смету из воздуха. Он обещает правильную логику при правильных вводных. --- *AI-Сметчик PRO | RAAI | Честные границы продукта*
