# AI-Сметчик PRO — Quick Start **3 промпта для первого запуска за 5 минут.** --- ## Промпт 1 — Быстрая локальная смета ```
смета: косметический ремонт квартиры 52м², Москва.
Состав: штукатурка стен 140м², стяжка 52м², плитка ванная 18м²,
ламинат 34м², покраска потолков 52м², обои 88м², электроточки 12шт.
Бюджет ориентировочно 1.2 млн. Тип заказчика: физлицо.
``` Получите: локальный сметный расчёт с ПЗ → НР → СП → НДС, шифры ФЕР по каждой строке, итог к оплате. --- ## Промпт 2 — Аудит сметы подрядчика ```
проверь смету: подрядчик прислал смету на ремонт офиса 90м². | Работа | Ед. | Кол. | Цена | Сумма |
|--------|-----|------|------|-------|
| Штукатурка стен | м² | 240 | 1100 | 264 000 |
| Шпаклёвка | м² | 240 | 600 | 144 000 |
| НР 145% от ФОТ | — | — | — | 186 000 |
| СП 90% от ФОТ | — | — | — | 115 000 | Регион: Москва. Заказчик: ООО (НДС нужен).
Хочу понять где завышено и на сколько.
``` Получите: сравнение расценок с ФЕР/рыночными ценами, выявленное завышение в рублях, рекомендацию — принять / торговаться / отклонить. --- ## Промпт 3 — Расчёт материалов ```
материалы: штукатурка стен гипсовой смесью, слой 15мм, площадь 180м².
Регион: Екатеринбург. Дать три варианта: эконом / стандарт / премиум.
``` Получите: таблицу расхода материалов с нормами ГЭСН, запасом на потери, ценами по трём уровням, итогом в рублях. --- ## Что делать после первого запуска 1. Откройте `config.yaml` — заполните регион, тип объектов, контрольные расценки под ваш рынок.
2. Прогоните режим `дефектовка` на реальном объекте.
3. Запустите `тендер` для подготовки первого КП по нормам. Подробная библиотека: `examples/full-library.md`
