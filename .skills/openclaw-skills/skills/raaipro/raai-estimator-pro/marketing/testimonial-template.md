# AI-Сметчик PRO — Шаблоны отзывов (для сбора и оформления) **Использование:** после передачи коробки клиенту — через 2-4 недели попросить отзыв по этому шаблону. Готовый отзыв размещается в `proof/` и используется в маркетинге. --- ## Шаблон 1 — Прораб / Сметчик (краткий, 3-5 строк) ```
[Должность, город, тип объектов] До: [проблема — время сметы / завышения / документы]
После: [результат с цифрами — часы, рубли, объекты]
Конкретно: [самый полезный режим]
Рекомендую: [да/нет + кому]
``` **Пример заполнения:**
> Прораб, Екатеринбург, жилые дома.
> До: смета на секцию — 2 дня, считал вручную в Excel.
> После: за 4 часа — локальная смета, объёмная ведомость, КП для заказчика.
> Конкретно: режим «объёмы» убрал спор с заказчиком — всё в числах.
> Рекомендую: да — для тех, кто тратит на сметы больше дня в неделю. --- ## Шаблон 2 — Директор / Заказчик (результат в деньгах) ```
[Должность, компания, оборот/год] Задача: [зачем взяли продукт]
Самое важное: [какой режим / что нашли]
Деньги: [сколько сохранили / нашли завышение]
Выводы: [как изменился процесс]
``` **Пример заполнения:**
> Директор ООО, Москва, ремонт коммерческой недвижимости, ~50 млн/год.
> Задача: проверить смету подрядчика на реконструкцию офиса 400м².
> Самое важное: режим «проверь смету» нашёл завышение НР на 145% вместо 105% + дубль раздела демонтажа.
> Деньги: сохранено 220 000 руб на одном объекте — коробка окупилась 8-кратно.
> Выводы: теперь каждая смета от подрядчика проходит через AI-аудит до подписания договора. --- ## Шаблон 3 — Генподрядчик / Тендерный отдел ```
[Должность, тип тендеров, годовой объём] Проблема: [как готовили КП раньше]
Решение: [что изменил AI-Сметчик PRO]
Результат: [выигранные тендеры / скорость / качество КП]
Совет новым пользователям: [что настроить сразу]
``` **Пример заполнения:**
> Руководитель тендерного отдела, Новосибирск, госзаказ + коммерция, 200 млн/год.
> Проблема: КП собирали 3-5 дней, часто опаздывали на дедлайн подачи.
> Решение: режим «тендер» даёт 3 пакета за 2 часа с обоснованием по ТЕР — копируем в форму.
> Результат: подали 3 тендера, 2 выиграли — цена обоснована нормами, а не «из головы».
> Совет: сразу заполнить config.yaml — регион, типовые объекты, контрольные расценки. Потом работает быстро. --- ## Шаблон 4 — Короткий для соцсетей / Telegram ```

"[цитата 1-2 предложения с конкретным числом]"
#AIСметчик #стройка #смета
``` **Примеры:**

> #AIСметчик #смета #ФЕР --- ## Инструкция по сбору отзывов **Когда просить:**
- Через 14-21 день после передачи коробки.
- После первого аудита сметы с найденным завышением — горячий момент.
- После выигранного тендера. **Как просить (скрипт для Telegram):**
```
Привет! Прошло 2 недели — как AI-Сметчик PRO?
Есть пара минут написать короткий отзыв по шаблону? Он поможет другим прорабам решить стоит ли брать коробку. Шаблон: [вставить нужный шаблон из этого файла] Если есть конкретная цифра — сохранённые деньги, часы — будет идеально.
``` **Куда класть готовый отзыв:**
- `proof/case-0N-[имя-ситуации].md` — развёрнутый кейс
- `marketing/testimonials.md` — накопительный файл коротких цитат
- При публикации на сайте — уточнить разрешение на имя/компанию --- ## Пометки по формату кейсов - Цифры в рублях или часах — обязательно
- Тип объекта + регион — даёт доверие
- До/После — структура, которая продаёт
- Без цифр — 🟡 гипотеза, публиковать как «похожая ситуация»
