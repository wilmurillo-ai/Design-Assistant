# Пример 09: Исполнительная документация — что сдать заказчику ## Запрос
```
исполнительная документация: квартира под ключ 55м², заканчиваем через 2 недели. Что собрать для финальной сдачи?
``` ## Ответ AI-Прораб PRO **КОМПЛЕКТ ИСПОЛНИТЕЛЬНОЙ ДОКУМЕНТАЦИИ (ИД) — СДАЧА КВАРТИРЫ ПОД КЛЮЧ**
Нормативная основа: РД 11-02-2006, СП 48.13330.2019 **СРОЧНО НАЧАТЬ СОБИРАТЬ (за 2 недели до сдачи):** ### РЕЕСТР ИД — ЧТО ДОЛЖНО БЫТЬ ГОТОВО ```
РЕЕСТР ИСПОЛНИТЕЛЬНОЙ ДОКУМЕНТАЦИИ
Объект: Кв. №___, [адрес]
Прораб: [ФИО] | Дата составления: ___________ СЕКЦИЯ I — ЖУРНАЛЫ:
[ ] 1. Общий журнал работ (КС-6а) — заполнен от первого до последнего дня
[ ] 2. Журнал входного контроля материалов — все поставки записаны
[ ] 3. Журнал инструктажей по ТБ — все подписи есть СЕКЦИЯ II — АКТЫ СКРЫТЫХ РАБОТ:
[ ] 4. Электрика в полу/стенах (кабели черновые) № ___
[ ] 5. Гидроизоляция ванной и с/у № ___
[ ] 6. Трубы водоснабжения в полу/стенах № ___
[ ] 7. Трубы канализации (скрытые участки) № ___
[ ] 8. Теплоизоляция/звукоизоляция (если была) № ___
[ ] 9. Подготовка основания под стяжку № ___ СЕКЦИЯ III — ПРОТОКОЛЫ ИСПЫТАНИЙ:
[ ] 10. Протокол измерения сопротивления изоляции кабелей Результат: R ≥ 0.5 МОм по каждой линии (ПУЭ 7)
[ ] 11. Протокол измерения сопротивления заземления Результат: R ≤ 4 Ом
[ ] 12. Протокол гидравлического испытания труб Давление: ___ бар, время: 30 мин
[ ] 13. Протокол испытания гидроизоляции (пол залит водой 24ч) СЕКЦИЯ IV — СЕРТИФИКАТЫ И ПАСПОРТА:
[ ] 14. Кабель ВВГнг-LS — сертификат соответствия ГОСТ
[ ] 15. Трубы ПП/металлопласт — сертификат + санитарное заключение
[ ] 16. Плитка/керамогранит — декларация о соответствии
[ ] 17. Гидроизоляция — технический паспорт материала
[ ] 18. Сухие смеси (штукатурка, клей) — декларации
[ ] 19. Оконные блоки (если менялись) — паспорта изделий СЕКЦИЯ V — ИСПОЛНИТЕЛЬНЫЕ СХЕМЫ (AS-BUILT):
[ ] 20. Схема электрики: трассы кабелей, розетки, выключатели (план)
[ ] 21. Схема водоснабжения: трассы труб, диаметры, запорная арматура
[ ] 22. Схема канализации: трассы, уклоны, ревизии
[ ] 23. Схема тёплого пола (если был): зоны, терморегуляторы СЕКЦИЯ VI — АКТЫ ПРИЁМКИ:
[ ] 24. Акты промежуточной приёмки по этапам (со своих слов)
[ ] 25. Финальный акт приёмки — подписывается В ДЕНЬ СДАЧИ ФОТОАРХИВ (на флешке или в облаке):
[ ] Все скрытые работы до закрытия
[ ] Испытания (залитый пол, манометр)
[ ] Финальное состояние по каждому помещению
``` ### ПЛАН НА ДВЕ НЕДЕЛИ ДО СДАЧИ: | День | Задача |
|------|--------|
| Сегодня | Собрать все уже оформленные акты, найти пробелы |
| Д-14 | Запросить сертификаты у поставщиков (нет = плохой поставщик) |
| Д-7 | Испытания электрики (вызвать электролаборатоирю) |
| Д-5 | Испытания труб (вместе с сантехником) |
| Д-3 | Нарисовать исполнительные схемы (фото + схема от руки = работает) |
| Д-1 | Проверить КС-6а: все дни заполнены? |
| День сдачи | Финальный акт приёмки, передать ИД под подпись | **МИНИМАЛЬНЫЙ КОМПЛЕКТ** (если не всё успели):
Без актов скрытых работ и протоколов испытаний — заказчик вправе не принять объект или заявить о дефектах позже. Каждый пропущенный акт = потенциальный спор без доказательств с вашей стороны.
