# AI-Прораб PRO — Quick Start (3 первых запуска) Установили коробку. Теперь три запуска за 15 минут — убедитесь что всё работает. --- ## Запуск 1: План дня (3 минуты) Скопируйте и отправьте агенту: ```
план дня: объект Квартира ул.Ленина 12 кв.45, 65м², сегодня 20.04
Бригада: Иванов (штукатур), Петров (плиточник), Сидоров (разнорабочий)
Задачи: штукатурка стен в кухне 18м², укладка плитки в санузле 6м²
Поставка цемента М500 в 10:00 (5 мешков)
``` **Ожидаемый ответ:** наряд-задание с распределением задач, нормами выработки, ТБ-инструктажем и планом входного контроля для цемента. --- ## Запуск 2: Приёмка штукатурки (5 минут) ```
приёмка: штукатурка, стандарт улучшенная, площадь 18м², исполнитель Иванов
Дата: сегодня. Объект: Квартира ул.Ленина 12 кв.45
``` **Ожидаемый ответ:** чек-лист приёмки с допусками по СП 71.13330.2017 (отклонение ≤2 мм/м для улучшенной), инструменты для проверки, шаблон акта приёмки. --- ## Запуск 3: Запись в журнал КС-6а (3 минуты) ```
КС-6а: запись, объект Квартира ул.Ленина 12 кв.45
Работа: устройство цементно-песчаной стяжки пола, 25м², толщина 50мм
Бригадир: Петров П.П., состав: 2 чел
Материал: цемент М500 3 мешка, песок 0.2м³
Результат: принято, отклонений нет
``` **Ожидаемый ответ:** готовая запись для журнала производства работ КС-6а в формате РД 11-02-2006 с подписными строками. --- ## Что делать после 1. Заполните OBLIGATORY поля в `config.yaml` под свой объект
2. Попробуйте `бригада: [наряд на вашу реальную задачу сегодня]`
3. Сохраните триггеры в избранное Telegram (если используете TG-бот)
4. Полная библиотека примеров: `examples/full-library.md` --- ## Если что-то не работает - `docs/anti-fail.md` — типовые проблемы и фиксы
