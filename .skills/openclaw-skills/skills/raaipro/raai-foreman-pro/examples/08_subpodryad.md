# Пример 08: Управление субподрядчиком — электрик ## Запрос
```
субподрядчик: электрик хочет начать работы на объекте, как его завести, что подписать
``` ## Ответ AI-Прораб PRO **ВВОД СУБПОДРЯДЧИКА НА ОБЪЕКТ — ЧЕК-ЛИСТ** ### ШАГ 1: ДО ПРИХОДА НА ОБЪЕКТ (оформить заранее) - [ ] Договор субподряда подписан с обеих сторон
- [ ] Смета / ТЗ: объём работ, материалы (чьи?), стоимость, срок — в тексте договора
- [ ] Схема электрики согласована и передана исполнителю
- [ ] Марки и сечения кабелей согласованы (прораб утвердил)
- [ ] Применяемые материалы в договоре: ВВГнг-LS ГОСТ 31996, конкретный производитель
- [ ] Гарантийный срок: в договоре (минимум 2 года по ГК РФ ст.756)
- [ ] Штрафы за просрочку и брак: прописаны ### ШАГ 2: ПЕРВЫЙ ДЕНЬ НА ОБЪЕКТЕ - [ ] Паспорт ответственного: проверить и скопировать
- [ ] Удостоверение электрика (группа допуска ≥ III): проверить срок
- [ ] Вводный инструктаж по ТБ объекта: провести, подпись в журнале
- [ ] Предупредить о запретах: курение, мусор, хождение по чужим работам
- [ ] Показать: где щиток, где перекрыть воду, аптечка, эвакуация ### ШАГ 3: КОНТРОЛЬ В ПРОЦЕССЕ РАБОТ | Момент | Что проверить | Документ |
|--------|--------------|---------|
| Начало кабельных работ | Марка кабеля, сечение (микрометр) | Журнал входного контроля |
| Укладка в штробы | Глубина ≥ 20мм, кабель в гофре | Фото |
| До штукатурки | Трассировка по схеме, испытание изоляции ≥ 0.5МОм | Акт скрытых работ |
| Монтаж щитка | Автоматы по схеме, маркировка | Осмотр |
| Финал | Испытания R изоляции + R заземления | Протоколы | ### ДОГОВОР — КЛЮЧЕВЫЕ ПУНКТЫ ```
ДОГОВОР СУБПОДРЯДА — ЭЛЕКТРОМОНТАЖНЫЕ РАБОТЫ Предмет:
- Черновая электрика: штробление, кабель, коробки — __ м²
- Сборка щита, чистовая электрика: __ точек
- Испытания и протоколы: в объёме работ Стоимость: ___________ руб.
Аванс: 30% при подписании = ___________ руб.
Итог: 70% после подписания актов и протоколов испытаний Материалы: Заказчик / Исполнитель (нужное указать)
При материалах Исполнителя: только ГОСТ, согласованные марки Срок: черновая до _______, итог до _______ Гарантия: 3 года Штраф за просрочку: 0.3% от суммы за каждый день
Штраф за брак (R изоляции < нормы): стоимость переделки × 1.5 Приёмка: Прораб [ФИО] принимает каждый этап.
Акты: подписываются при фактической готовности, не по словам.
``` ### КРАСНЫЕ ФЛАГИ — ОСТАНОВИТЬ РАБОТЫ: - Использует кабель ТУ (не ГОСТ) без согласования
- Штробы по диагонали
- Не делает акты скрытых работ перед штукатуркой
- Просит аванс больше 30% без выполненных работ
- Отказывается от испытаний изоляции
