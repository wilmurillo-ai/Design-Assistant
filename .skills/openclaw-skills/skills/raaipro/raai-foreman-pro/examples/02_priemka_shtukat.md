# Пример 02: Приёмка штукатурки ## Запрос
```
приёмка: штукатурка стен, улучшенное качество, квартира, спальня 18м²
``` ## Ответ AI-Прораб PRO **ПРИЁМКА ШТУКАТУРКИ — УЛУЧШЕННОЕ КАЧЕСТВО**
Норма: СП 71.13330.2017, таблица 7.4 **Инструменты:** правило 2м, уровень 2м, угольник 90°, фонарик, шпатель. | Параметр | Норма (улучш.) | Как проверять | Факт | Оценка |
|----------|---------------|---------------|------|--------|
| Отклонение от вертикали | ≤ 2 мм/м | Уровень + правило 2м, замер в 3 точках | ___ | |
| Отклонение от горизонтали | ≤ 2 мм/м | Уровень 2м, замер по всей плоскости | ___ | |
| Неровности | ≤ 3мм, не более 2 шт на 2м | Правило 2м + щуп, свет сбоку | ___ | |
| Отклонение откосов | ≤ 2 мм | Уровень + угольник | ___ | |
| Углы (прямые) | 90° ± 2мм | Угольник | ___ | |
| Сцепление со стеной | Нет пустот | Простукивание молотком по всей площади | ___ | |
| Трещины | Отсутствуют | Визуальный осмотр при боковом свете | ___ | | **Порядок проверки:**
1. Осмотр при боковом свете (фонарь под углом 15-20°) — видны все волны и неровности
2. Правило 2м приложить в 5-7 местах: вертикально, горизонтально, диагонально
3. Простукать всю поверхность шпателем — «бухтение» = пустота, переделка
4. Проверить углы: откосы с уровнем, примыкания с угольником **Решение:**
- Все параметры в норме → Принять, подписать акт, внести в КС-6а
- Есть дефекты → Указать точно: "откос оконный, отклонение 4мм — не принят, переделать до [дата]" **ВНИМАНИЕ:** Штукатурка должна полностью высохнуть перед шпаклёвкой (1мм ≈ 1 день при +20°С). Средний слой 12мм → минимум 12 суток.
