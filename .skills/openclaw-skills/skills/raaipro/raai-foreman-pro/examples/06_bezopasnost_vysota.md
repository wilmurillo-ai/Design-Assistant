# Пример 06: Техника безопасности — работы на балконе 9 этаж ## Запрос
```
безопасность: завтра начинаем работы на балконе 9 этаж — монтаж ограждения и покраска парапета. Что нужно подготовить?
``` ## Ответ AI-Прораб PRO **ТБ: РАБОТЫ НА ВЫСОТЕ — БАЛКОН 9 ЭТАЖ**
Нормативная база: СНиП 12-03-2001 (п.6), ГОСТ 12.0.004-2015 **РАБОТЫ НА ВЫСОТЕ БОЛЕЕ 1.3М — ТРЕБОВАНИЯ ОБЯЗАТЕЛЬНЫ** ### ДОКУМЕНТЫ ДО НАЧАЛА РАБОТ - [ ] Наряд-допуск на работы на высоте (если организация, не физлицо)
- [ ] Целевой инструктаж всех работников с подписью в журнале ТБ
- [ ] Проверка страховочных привязей: маркировка, дата испытания (< 1 года) ### СИЗ ДЛЯ КАЖДОГО РАБОТНИКА | СИЗ | Требование | Проверить |
|-----|-----------|---------|
| Каска строительная | Обязательно | Нет трещин, ремень затянут |
| Страховочная привязь | Обязательно (высота > 1.8м) | Дата испытания, крепёж |
| Строп с амортизатором | Обязательно | Целостность, крюк с замком |
| Нескользящая обувь | Обязательно | Рантовая подошва |
| Перчатки | Обязательно | — |
| Очки (при покраске) | Обязательно | — |
| Респиратор FFP2 (при покраске) | Обязательно | — | ### ОРГАНИЗАЦИЯ РАБОЧЕГО МЕСТА - [ ] Точки крепления стропа: проверить прочность (несущая > 15 кН на точку) — Куда крепить: анкеры в стену, перекрытие, но НЕ трубы отопления/канализации
- [ ] Зона под балконом: оградить, запретить проход людей — Радиус опасной зоны: высота 27м + 7м = 34м минимум → ограждение
- [ ] Инструмент: привязать на шнурки (падение с 9 этажа = смертельная опасность) — Щётки, кисти, молотки — шнуровка на запястье
- [ ] Материалы: не перегружать балкон (< 150 кг/м² жилой балкон)
- [ ] Леса/вышка (если нужна снаружи): ГОСТ 27321, оформить акт ### ПОГОДА (ЗАПРЕТ РАБОТ): | Условие | Ограничение |
|---------|-------------|
| Ветер | > 15 м/с — СТОП |
| Гроза | СТОП, до конца |
| Гололёд/иней | СТОП, до уборки |
| Снег | СТОП при видимости < 20м |
| Дождь | СТОП при работах с ЛКМ |
| Температура (покраска) | < +5°С — большинство красок не сохнут | ### ПОРЯДОК РАБОТ ЗАВТРА 07:00 — Целевой инструктаж, подпись в журнале
07:30 — Проверка всех СИЗ, строп прикреплён до выхода на балкон
08:00 — Начало монтажа ограждения (сначала — потом покраска)
После монтажа — ограждение проверено: нагрузка 0.5кН/м по горизонтали (СП 20.13330)
После ограждения — покраска парапета в проветриваемом месте (маска!) **АПТЕЧКА:** на месте работ, доступна. Знать ближайший травмпункт.
**СВЯЗЬ:** прораб должен быть доступен по телефону весь день.
