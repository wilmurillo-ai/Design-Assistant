# AI-Прораб PRO — сравнение альтернатив ## Альтернатива 1: Нанять дополнительного прораба-линейника | Параметр | Линейный прораб | AI-Прораб PRO |
|---|---|---|
| Стоимость | 80-150К ₽/мес | 30К ₽ разово |
| Нормативная база | Зависит от опыта | 12 СП/ГОСТ/ПУЭ всегда актуальны |
| Документация | «Потом заполним» | Шаблон в момент работы |
| Доступность | 8 ч/день × 5 дней | 24/7 |
| Масштабирование | +80-150К на каждый объект | Один ключ — все объекты |
| Болезни/отпуска | Срывает сроки | Не болеет | **Вывод:** AI-Прораб PRO не заменяет прораба на объекте физически, но берёт 40-50% его рутины (планирование, документы, отчёты). Стоит в 50-90 раз дешевле. --- ## Альтернатива 2: Excel + Word шаблоны | Параметр | Excel/Word | AI-Прораб PRO |
|---|---|---|
| Составить наряд | 30-40 мин (поиск файла + заполнение) | 3 мин (триггер → готовый шаблон) |
| Найти допуск по СП | Отдельный поиск + риск устаревания | Встроено в шаблон приёмки |
| Отправить бригаде | Скопировать в WA вручную | TG-бот по триггеру |
| Отчёт прораба | 1-2 ч/нед на компиляцию | 10 мин: структурированный вывод |
| Хранение фото | Папки без структуры | Google Drive + теги по этапам |
| Поиск документа | «Где тот акт от прошлого месяца?» | Структура в журнале КС-6а | **Вывод:** Excel/Word бесплатны но стоят 5-7 ч/нед прорабского времени. AI-Прораб PRO окупается за 1-2 недели. --- ## Альтернатива 3: Общие AI-ассистенты (ChatGPT / Claude без коробки) | Параметр | Общий AI | AI-Прораб PRO |
|---|---|---|
| Нормы РФ (СП/ГОСТ) | Ненадёжно, может выдумать | Встроены, проверены |
| Шаблоны документов | Нужно описывать каждый раз | Готовые, вызываются триггером |
| Профиль объекта | Нужно вводить заново каждый раз | config.yaml — один раз |
| Триггеры | Длинные промпты | Короткий триггер: «приёмка: штукатурка» |
| TG-интеграция | Нет | TELEGRAM_BOT_TOKEN + шаблоны |
| Обучение команды | Каждый прораб промптует по-своему | Единый стандарт для всей компании | **Вывод:** ChatGPT знает строительство в общем, но не знает ваш объект, ваши нормы и ваш формат документов. AI-Прораб PRO — это настроенный инструмент, не универсальный ответчик. --- ## Итоговая матрица выбора ```
Сколько объектов? 1-2 → Коробка (30К) — окупится за месяц 3-10 → Агент (60К) — TG-бот + настройка под компанию 10+ → Приложение client-owned foreman app (150K+) — operations system
```
