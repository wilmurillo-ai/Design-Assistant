- Fixed scope + deadline → Waterfall
- Evolving requirements → Agile (Scrum/Kanban)
- Fixed milestones + flexible features → Hybrid
- Solo/2-person → Kanban **Three-Point Estimation (PERT):**
Expected = (Optimistic + 4×Likely + Pessimistic) / 6 **Risk Scoring Matrix:** Probability(1-3) × Impact(1-3) = 1-9 → Accept/Watch/Act/ESCALATE **Project Health Score (0-100):**
85-100 🟢 Excellent / 70-84 🟢 Good / 55-69 🟡 At Risk / 40-54 🔴 Troubled / 0-39 🔴 Critical ## Триггерные команды (12 штук) "New project [name]" / "Break down [deliverable]" / "Estimate [task]" / "Status report" / "Risk check" / "Health score" / "Change request [description]" / "Sprint plan" / "Retro" / "Close project" / "What's at risk?" / "Compare plan vs actual" ## Ограничения - Только методология — не интегрируется с Jira/Asana/MS Project
