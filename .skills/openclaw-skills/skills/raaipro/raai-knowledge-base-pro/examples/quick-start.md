# Quick Start — 3 промпта для первого запуска **AI-База знаний PRO.0**
Запустите эти 3 промпта сразу после установки — убедитесь что коробка работает. --- ## Промпт 1 — Первый поиск Скопируйте и отправьте агенту: ```
найди: как оформить возврат товара
``` **Что должен вернуть агент:**
- Прямой ответ на вопрос (или «в базе нет, добавить?»)
- Источник: категория + название записи + версия + дата
- Связанные темы (2-3 варианта) --- ## Промпт 2 — Добавить первую запись FAQ ```
добавь в базу FAQ:
В: Как связаться с поддержкой?
О: Пишите в Telegram @support или на email support@company.ru. Отвечаем в рабочие часы 9-18 МСК в течение 2 часов.
``` **Что должен вернуть агент:**
- Созданная запись с ID (KB-YYYY-NNNN)
- Автоматически заполненные теги и метаданные
- Предложение добавить варианты вопроса --- ## Промпт 3 — Запустить онбординг ```
онбординг менеджер по продажам
``` **Что должен вернуть агент:**
- Структурированный план: День 1 → Неделя 1 → Месяц 1
- Чек-листы задач с привязкой к KB-записям
- Список ключевых документов для изучения
- Тест по итогам недели 1 --- ## Если что-то не работает Смотрите: [docs/anti-fail.md](../docs/anti-fail.md) Или запустите диагностику:
```
bash test/smoke-test.sh
``` Все проверки PASS? Коробка готова к работе.
