# Пример 12 — Расширенный поиск с фильтрами ## Поиск по дате и автору
```
найди: регламент возврат фильтр:дата=>2026-01-01 фильтр:автор=Иванова
``` ```
ОТВЕТ НА ЗАПРОС: "регламент возврат" [фильтры: дата > 2026-01-01, автор = Иванова] Найдено 2 записи: 1. KB-2026-0012: "Регламент возврата товара" — релевантность 97% Автор: Иванова М. | Обновлено: 2026-03-15 Краткое резюме: Пошаговый процесс оформления возврата для сотрудников поддержки 2. KB-2026-0013: "Регламент обмена товара" — релевантность 74% Автор: Иванова М. | Обновлено: 2026-02-10 Краткое резюме: Отличия обмена от возврата, особые случаи
``` ## Поиск по типу контента и отделу
```
найди: обработка заявки фильтр:тип=sop фильтр:отдел=продажи фильтр:статус=актуально
``` ```
ОТВЕТ НА ЗАПРОС: "обработка заявки" [SOPs, продажи, актуальные] Найдена 1 запись: 1. SOP-2026-0001: "Обработка входящей заявки от клиента" — релевантность 99% Статус: Актуально | Версия: v1.0.0 | Обновлено: 2026-04-16 Владелец: Руководитель отдела продаж [Полное содержание SOP...]
``` ## Поиск по уровню доступа (руководство)
```
найди: финансовые показатели фильтр:доступ=3
``` ## Поиск без ответа — что происходит
```
найди: интеграция с 1С пошаговая инструкция
``` ```
ОТВЕТ НА ЗАПРОС: "интеграция с 1С пошаговая инструкция" В базе знаний нет записи по этой теме. Этот запрос задавали 18 раз за последние 30 дней.
Рекомендую добавить запись — это явный gap в базе.
