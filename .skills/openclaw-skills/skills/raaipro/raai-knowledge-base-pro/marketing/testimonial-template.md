Привет! Прошло [2-4] недели с момента установки AI-Базы знаний PRO.
Буду благодарен если ответите на 3 вопроса (2 минуты): 1. Какой режим используете чаще всего? (FAQ / онбординг / поиск / регламенты / аудит) 2. Что изменилось по сравнению с тем как было до? (напишите свободно, 2-3 предложения) 3. Есть конкретная цифра экономии или ускорения? (не обязательно — только если считали) Спасибо! Отзыв буду использовать анонимно, если не против.
``` --- ## Правила работы с отзывами - Собирать через 2-4 недели, не раньше — за 3 дня нет ещё эффекта
- Рублёвые цифры = золото, добиваться конкретики
- Сохранять в `proof/` с именем `feedback-[компания]-[дата].md`
- При публикации: всегда получать явное разрешение на имя/компанию
- Анонимные отзывы = допустимо, но уточнять отрасль и размер компании
