- Команда говорит по-русски
- Нужен готовый онбординг по ролям (не писать шаблоны с нуля)
- Хотите FAQ + регламенты + SOP + аудит в одном месте
- Нет DevOps для настройки LightRAG-сервера
- Нет бюджета на Guru SaaS ($10-20/user/мес = 100-200К/год для 50ч)
- Нужно за 15 мин, не за 3 дня ### Выбрать LightRAG если:
- У вас есть DevOps и вы хотите RAG по >1000 документам
- Работаете на English
- Готовы настроить сервер ### Выбрать Guru MCP если:
- Уже платите за Guru SaaS
- Работаете на English
- Нужна только интеграция существующей Guru-базы в агента --- ## Ценовая аргументация **«Почему не бесплатные альтернативы?»** - LightRAG — бесплатный, но требует: Linux-сервер (~2-5К/мес VPS) + DevOps-настройка (8-16ч × 2000 ₽/ч = 16-32К) + нет русского + нет онбординга-шаблонов. Итого: реальная стоимость запуска 20-40К + ежемесячно 2-5К.
- Guru MCP — бесплатный скилл, но Guru SaaS: $10-20/user/мес × 20 сотрудников = $200-400/мес = **24-48К ₽/год**.
- knowledge-base-pro: **30К ₽ один раз** + работает без дополнительных расходов. **Вывод:** для компании 15-50 человек knowledge-base-pro = самый дешёвый путь к работающей KB-системе на русском языке.
