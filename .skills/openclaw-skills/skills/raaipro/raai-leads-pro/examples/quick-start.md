# AI-Лиды PRO — Quick Start (3 готовых промпта) > Три промпта для первого запуска. Скопируйте в Claude/OpenClaw и подставьте свои данные. --- ## Промпт 1 — Аватар клиента (ICP) Для кого: маркетолог, собственник, РОП — кто угодно, кто хочет понять «кто мой идеальный покупатель». ```
аватар: [ваша ниша] Пример: аватар: B2B-консалтинг по автоматизации, клиент — производственные компании 50-500 чел
Пример: аватар: онлайн-курсы по инвестициям, клиент — наёмный менеджер 35-45 лет
Пример: аватар: строительная компания ищет подрядчиков на монолит
``` **Что получите:**
- Демография + психография (боли, страхи, желания, возражения)
- Поведение онлайн: где сидит, что гуглит, кого читает
- Путь к покупке: 5 этапов Customer Journey
- Ценовой профиль: комфортный/принимаемый/болевой чек
- Ключевое сообщение + CTA --- ## Промпт 2 — BANT-квалификация лида Для кого: менеджер по продажам или РОП, который хочет понять «стоит ли тратить время на этот лид». ```
квалифицируй: [описание лида] Пример: квалифицируй: Василий Петров, директор торговой компании 20 чел, написал в TG "хотим автоматизировать продажи", бюджет не называл, планирует "в этом квартале" Пример: квалифицируй: входящий с сайта, Ольга, HR-директор компании 500 чел, спрашивает про онбординг-боты, "нужно согласовать с CEO", бюджет 50-100К заложен в годовом плане
``` **Что получите:**
- BANT-скоринг (Budget/Authority/Need/Timeline)
- Итоговый балл 0-100 с расшифровкой
- Решение: «Горячий → передать», «Тёплый → нутрить», «Холодный → парковать»
- Следующий шаг: конкретное действие для менеджера --- ## Промпт 3 — Отчёт РОПа по воронке входа Для кого: РОП или собственник, кому нужен быстрый отчёт по лидогенерации. ```
отчёт лиды (AI спросит: подставьте свои цифры за период — лиды, квалифицированные, встречи, сделки, бюджет на лидген) Пример полных данных:
Период: апрель 2026
Лиды всего: 85
Из них квалифицированных (BANT ≥60): 31
Встречи проведены: 18
Коммерческие предложения: 9
Сделки закрыты: 3
Средний чек: 120 000 руб
Бюджет на лидген: 45 000 руб
Каналы: контекст (30 лидов), таргет VK (25), холодный TG (20), реферал (10)
``` **Что получите:**
- Воронка с конверсиями на каждом этапе
- CAC по каналам (стоимость привлечения клиента)
- LTV и ROI по каналам
- Прогноз выручки на следующий месяц
- Рекомендации: где усилить, где сократить бюджет --- ## Следующий шаг Полная библиотека всех 13 режимов — в [full-library.md](full-library.md).
