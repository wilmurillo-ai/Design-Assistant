# French Business Assistant - OpenClaw Skill

Agent assistant pour entrepreneurs et freelances francophones.

## Fonctionnalites

- **Emails professionnels** : redaction, reponses, ton adapte au contexte
- **Devis et factures** : conformes au droit francais (mentions legales, TVA, SIRET)
- **Relances clients** : escalade progressive (amical -> ferme -> mise en demeure)
- **Briefs clients** : preparation de RDV, historique, points a aborder
- **Prospection** : emails froids personnalises, messages LinkedIn

## Installation

```bash
clawhub install french-business-assistant
```

## Configuration

Ajoutez dans votre `USER.md` ou `MEMORY.md` :
- Nom / Raison sociale
- SIRET
- Adresse
- Numero TVA intracommunautaire
- IBAN
- Signature email
- Tarif horaire / jour
