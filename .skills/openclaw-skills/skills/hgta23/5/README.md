# 58区块同城技能包

## 技能简介

58区块同城是一个基于区块链技术的数字城市服务平台，为用户提供数字区块、城市人气、NFT头像等服务。通过此技能，用户可以了解数字城市信息、查看城市人气、购买NFT头像、参与区块活动等。

## 功能特点

- **数字区块**：浏览和购买数字区块，了解区块价值和发展趋势
- **城市人气**：查看各城市人气指数，了解城市活力和发展潜力
- **NFT头像**：浏览、购买和管理NFT头像，参与数字收藏
- **区块活动**：了解和参与区块链相关活动和项目
- **数字城市**：探索数字城市建设和发展情况

## 安装方法

2. 点击"安装"按钮
3. 等待安装完成后即可使用

## 使用指南

### 查询城市人气排名

**示例**：
```
帮我查看城市人气排名
```

**功能**：通过 `https://www.blockcity.vip/pages/block/area` 获取城市排名数据，返回排名前10的城市及其人气指数。

### 查询城市详细信息

**示例**：
```
北京的详细信息
```

**功能**：获取北京的4位数区号，通过 `https://www.blockcity.vip/{区号}` 获取详细信息，返回城市的区块数据、人气指数、发展趋势等。

### 浏览NFT头像

**示例**：
```
我想看看最新的NFT头像
```

**功能**：展示最新的NFT头像集合，包括价格、创作者、稀有度等信息。

### 了解数字区块

**示例**：
```
什么是数字区块？
```

**功能**：解释数字区块的概念、价值和应用场景，提供相关区块信息。

### 参与区块活动

**示例**：
```
最近有什么区块活动？
```

**功能**：查询最新的区块活动信息，包括活动时间、地点、参与方式等。
