# h - Skill 发现中心

## 🚀 简介

**h** 是一款专为 ClawHub.ai 设计的 **Skill 搜索和发现工具**，让你比 find-skill 更高效地找到喜欢的技能包！

- **H**ub - Skill 发现中心，一站式技能搜索和管理平台
- **H**unt - 智能搜索，快速定位你需要的技能包
- **H**ighlight - 精选推荐，发现热门和优质技能
- **H**istory - 使用历史，快速回顾和重新发现
- **H**elp - 智能匹配，根据你的使用场景推荐技能

## ✨ 核心功能

### 🔍 智能搜索
- 关键词模糊匹配
- 标签和分类筛选
- 作者和热门度排序
- 自然语言搜索理解

### 🌟 精选推荐
- 每日热门技能榜
- 新技能首发推荐
- 相似技能关联推荐
- 个性化智能推荐

### 📚 技能管理
- 收藏喜爱的技能
- 使用历史记录
- 技能评分和评价
- 安装/卸载快速操作

### 🎯 场景匹配
- 根据任务推荐技能
- 工作流智能组合
- 技能使用教程
- 最佳实践分享

## 📦 安装

在 ClawHub.ai 上搜索 "h"，点击安装即可使用！

## 🎯 使用场景

### 1. 快速找技能
```
/h search python - 搜索 Python 相关技能
/h hot - 查看今日热门技能
/h recommend - 获取个性化推荐
```

### 2. 技能管理
```
/h favorites - 查看收藏的技能
/h history - 查看使用历史
/h install [skill-name] - 快速安装技能
```

### 3. 发现新技能
```
/h new - 查看最新上线技能
/h trending - 查看上升最快技能
/h similar [skill-name] - 找相似技能
```

## 📚 进阶技巧

### 组合搜索
使用多个关键词组合搜索更精准：
```
/h search 可视化 react - 搜索同时包含可视化和 react 的技能
```

### 标签筛选
通过标签快速分类浏览：
```
/h tags 数据分析 - 查看数据分析标签下的所有技能
```

### 智能推荐
让 h 了解你的使用习惯，获得更好的推荐：
```
1. 经常使用 /h search 搜索你需要的技能
2. 用 /h favorites 收藏喜欢的技能
3. h 会根据历史自动推荐相关技能
```
