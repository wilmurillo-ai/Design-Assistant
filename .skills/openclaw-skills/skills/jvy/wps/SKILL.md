# WPS Office

面向中文办公场景的 WPS 技能，覆盖文档写作、表格处理、演示制作、格式兼容和导出交付。

## 何时使用

- 用户明确提到 WPS/WPS Office、WPS 文字/表格/演示。
- 需要在 `.docx/.xlsx/.pptx` 与 WPS 常用格式之间互转或排查兼容问题。
- 需要处理中文排版问题，例如字号行距、段前段后、中文标点、分页和目录。
- 需要“可交付版本”，例如导出 PDF、打印版、批注审阅版。

## 不适用场景

- 纯代码开发、数据库查询、服务器运维类任务。
- 只涉及 Google Docs/Notion 等在线协作工具且不需要 WPS。
- 需要直接执行本机 GUI 点击操作但当前环境无桌面会话。

## 默认工作流

1. 明确目标：先确认文档类型（文字/表格/演示）、交付格式（源文件/PDF）和截止时间。
2. 收集输入：确认原始文件、模板、字体要求、页边距与纸张规格（A4/Letter）。
3. 给出最小步骤：优先提供可复制的短流程，先保证可交付，再做美化。
4. 兼容性检查：重点检查字体替换、分页漂移、图表错位、公式兼容。
5. 交付说明：输出“已完成项 + 待确认项 + 回滚方案”。

## 高频任务模板

### 1) WPS 文字

- 中文长文：先统一样式（标题 1/2/3、正文、引用）再插目录。
- 合同或公文：优先锁定页面设置、页眉页脚和段落网格，最后处理编号。
- 审阅流程：建议开启修订 + 批注，避免直接覆盖原文。

### 2) WPS 表格

- 数据清洗：先备份副本，再做分列、去重、格式标准化。
- 汇总分析：优先使用数据透视表，减少手工复制粘贴。
- 交付前检查：公式引用范围、数字格式、冻结窗格、打印区域。

### 3) WPS 演示

- 先定母版与字体，再批量套版；避免逐页手工对齐。
- 图文混排优先使用参考线与对齐分布工具。
- 导出前检查字体嵌入与动画兼容，必要时导出 PDF 兜底。

## 兼容与故障排查

- 打开后版式错乱：先确认缺失字体，再检查段落样式与分页符。
- Excel/WPS 公式不一致：确认函数差异与区域引用方式。
- 图片模糊：优先替换原图，避免重复截图压缩。
- 打印分页异常：固定纸张与边距，重设打印区域后再预览。

详细格式差异与排查顺序见 [WPS Reference](references/wps-reference.md)。

## 安全边界

- 未经用户同意，不覆盖原文件；默认另存为新版本。
- 不处理或暴露真实敏感信息（证件号、手机号、合同金额等），示例一律用占位符。
- 需要安装软件、联网下载模板或宏插件时，先说明风险与来源。

- 只输出可复现、可审计的操作步骤，避免依赖个人设备路径。
