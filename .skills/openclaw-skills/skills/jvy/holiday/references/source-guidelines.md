# Holiday Source Guidelines

Use this reference when a holiday question needs source selection discipline.

## Resolve Scope First

Identify:

- country
- region or state if relevant
- year
- holiday type
- institution type if the user asked about schools, banks, or markets

## Preferred Sources

Use sources in this order when available:

1. Government holiday calendar
2. Ministry or civil service notice
3. Central bank or education department notice
4. Official gazette or parliamentary publication
5. Embassy summary
6. Reputable secondary reference

## What to Call Out

- Whether the holiday is nationwide or regional
- Whether the observed day differs from the named holiday date
- Whether closures apply to all offices or only selected institutions
- Whether the answer is based on an officially published calendar or an inference from standing rules

## Avoid

- Treating cultural observances as legal holidays without evidence
- Assuming all employers or schools close on public holidays
- Assuming substitute-day rules are universal
- Answering without a jurisdiction
