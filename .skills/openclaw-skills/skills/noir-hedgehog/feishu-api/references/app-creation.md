# 创建飞书应用

## Step 1: 注册开发者账号

1. 访问 [飞书开放平台](https://open.feishu.cn/)
2. 登录后进入「开发者后台」
3. 选择企业或自建应用

## Step 2: 创建应用

1. 点击「创建应用」
2. 选择「企业自建应用」
3. 填写应用信息：
   - 应用名称
   - 应用描述
   - 应用图标（可选）

## Step 3: 获取凭证

在「凭证与基础信息」页面获取：
- `App ID`（格式：`cli_xxx`）
- `App Secret`

**重要**：妥善保管 App Secret，不要泄露。

## Step 4: 配置权限

1. 进入「权限管理」
2. 添加所需权限：

常用权限：
- `bitable:app` - 多维表格
- `docx:document` - 文档读写
- `drive:drive` - 云文档
- `contact:user.id:readonly` - 用户 ID

## Step 5: 配置应用功能

### 启用「机器人」功能

如需使用机器人：
1. 进入「应用功能」
2. 开启「机器人」

### 配置网页

如需 OAuth 回调：
1. 进入「应用功能」→「网页」
2. 添加网页地址（需 HTTPS）

## Step 6: 发布应用

1. 进入「版本管理与发布」
2. 创建版本
3. 申请发布（企业管理员审批）

## OAuth 配置

### 设置重定向 URL

1. 进入「安全设置」
2. 「重定向 URL」中添加授权回调地址
3. 必须使用 HTTPS

### 权限范围

在 OAuth 配置页面设置应用申请的权限范围。

## 获取用户授权

应用已创建并发布后，用户访问授权 URL 完成授权。

## 注意事项

1. **权限最小化**：只申请必要的权限
2. **HTTPS 必需**：生产环境回调地址必须 HTTPS
3. **企业管理员审批**：自建应用需要管理员审批才能生效
