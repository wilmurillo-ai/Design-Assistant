### 一、K线形态识别（61种）

InStock 基于 TA-Lib 精准识别61种形态，结果含义：
- **正值**：买入信号
- **负值**：卖出信号
- **0**：未出现该形态

**重点关注的形态（实战优先级高）**：

| 形态 | 含义 | 实战用途 |
|------|------|----------|
| 射击之星 | 上影线长、实体小，高位出现 | 见顶信号，配合巨量更可信 |
| 乌云盖顶 | 高开低走覆盖前日阳线 | 短期顶部反转 |
| 吞噬模式 | 大阴线完全包住前日阳线 | 强烈卖出信号 |
| 暮星/黄昏之星 | 三K线顶部反转组合 | 中期见顶确认 |
| 晨星/曙光初现 | 三K线底部反转组合 | 底部买入参考 |
| 锤头 | 下影线长、实体小，低位出现 | 底部反转，配合放量 |
| 倒锤头 | 上影线长，低位出现 | 潜在底部反转 |
| 十字星 | 开收基本相等 | 多空分歧，变盘信号 |
| 上升/下降三法 | 五K线持续形态 | 趋势延续确认 |

**InStock查询方式**：进入 http://localhost:9988/ → K线形态 → 选择形态 → 查看识别结果

### 二、11种选股策略（策略逻辑摘要）

可在 InStock 直接调用回测验证，以下是核心策略逻辑：

**1. 放量上涨**
- 当日成交量/5日均量 ≥ 2
- 成交额不低于2亿
- 当日上涨（收盘>开盘）
→ 主力资金参与的启动信号

**2. 停机坪**（强势品种特有）
- 近15日有涨幅大于9.5%的放量大阳
- 之后连续2~3日高开微涨（每日涨幅在5%以内）
- 量能持续收缩
→ 主力洗盘控盘的典型形态，洗盘结束后往往二次拉升

**3. 回踩年线**
- 前段从MA250以下向上突破
- 后段在MA250上方运行
- 回踩伴随缩量，回踩幅度不超过20%
- 最高价到最低价间距10~50日
→ 年线支撑买点，风险收益比优

**4. 突破平台**
- 60日内股价在MA60附近（-5%~+20%）横盘整理
- 某日放量突破（收盘≥MA60，且成交量/5日均量≥2）
→ 平台蓄势后的方向选择，突破有效性高

**5. 海龟交易法则**
- 当日收盘价 = 近60日最高收盘价
→ 趋势追踪，动量策略，适合强势品种

**6. 均线多头**
- MA30连续上行（30日前<20日前<10日前<当日）
- 当日MA30 / 30日前MA30 > 1.2
→ 中期趋势确认，适合持有

**7. 高而窄的旗形**
- 24~10日内最低价到当日收盘涨幅 ≥ 90%
- 期间有连续两天涨幅 ≥ 9.5%
→ 强势股短期旗形整理后的继续拉升

**8. 低ATR成长**
- 近10日最高收盘 / 最低收盘 > 1.1
- 波动率（ATR）低
→ 稳健上涨，非投机性拉升，机构建仓形态

**9. 无大幅回撤**
- 60日内无单日跌幅超7%
- 无两日累计跌幅超10%
→ 筹码稳定，适合中线持有

### 三、扩展指标参考（KDJ/CCI/CR/WR）

InStock 计算的额外指标，补充 MACD+RSI 体系：

**KDJ**
- K>80, D>70, J>90 → 超买，谨慎
- K<20, D<30 → 超卖，可能反弹
- 金叉（K上穿D）低位出现 → 买入参考

**CCI**
- CCI > +100 → 超买区，关注回调
- CCI < -100 → 超卖区，逢低吸纳参考

**CR（能量线）**
- CR < 40 → 建仓良机
- CR从低位上升超160 → 短线获利考虑

**WR（威廉指标）**
- WR达到20 → 超买，可能见顶
- WR达到80 → 超卖，可能见底

**VR（成交量比率）**
- VR 40~70 → 低价区，可买进
- VR 160~450 → 获利区，考虑了结

### 四、筹码分布（成本分布）

InStock 默认计算210日筹码分布，与东方财富一致。

**实战解读**：
- 筹码高度集中在当前价附近 → 机构控盘，不容易大跌
- 筹码大量集中在上方（套牢盘） → 反弹压力大
- 筹码向下集中（获利盘在下） → 持仓成本低，上涨空间大
- 筹码从上方向下方转移 → 换手充分，可能见底

配合千股千评"控盘度"指标交叉验证。

### 五、综合选股维度（200+）

InStock 综合选股支持以下维度自由组合：

| 大类 | 内容 |
|------|------|
| 股票范围 | 市场、行业、地区、概念、指数成份 |
| 基本面 | 市盈率、市净率、ROE、净利润增长 |
| 技术面 | 61种K线+11种策略+各指标条件 |
| 消息面 | 公告、机构持股家数/比例 |
| 人气指标 | 股吧排名、新晋粉丝、7日关注 |
| 行情数据 | 资金流向、沪深股通、龙虎榜 |

**推荐组合（快速筛优质标的）**：
1. MA多头排列 + OBV上行 + 不对称比<1.2 + 成交额>5000万
2. 突破平台策略 + 机构持股>3家 + 市盈率<50
3. 放量上涨 + RSI在50~70 + 股价在MA60上方

### 六、InStock 本地操作指南

```bash
# 启动容器（如未运行）
open -a Docker  # 先启动Docker daemon
docker start instock

# 查看运行状态
docker ps | grep instock

# 手动触发数据更新（收盘后）
docker exec -it instock bash -c "cd /data/InStock && python instock/bin/execute_daily_job.py"

# 查看日志
docker exec -it instock cat InStock/instock/log/stock_execute_job.log | tail -50

# 只更新基础数据（快，约1秒）
docker exec -it instock bash -c "cd /data/InStock && python instock/bin/basic_data_daily_job.py"
```

访问界面：http://localhost:9988/

**注意**：容器每天17:30自动执行全量数据抓取，无需手动触发。

### 七、东方财富 Cookie 配置（提升API稳定性）

如果东方财富API频繁被限制，可注入Cookie：

1. 浏览器打开 https://quote.eastmoney.com/center/gridlist.html#hs_a_board
2. 开发者工具 → Network → 找任意 push2.eastmoney.com 的请求
3. 复制 Request Headers 中的 Cookie 值
4. 注入方式：
```bash
# curl请求加Cookie
curl -s --ipv4 -H "Cookie: 你的cookie值" -H "User-Agent: Mozilla/5.0" "https://push2.eastmoney.com/..."
```

Cookie有效期通常数天至数周，过期重新获取。
**安全提示**：Cookie属于短效个人凭证，勿硬编码到脚本，勿提交到代码仓库，勿在共享环境中使用。不配置Cookie时，核心分析功能完全正常，Cookie仅用于提升高频请求的稳定性。
