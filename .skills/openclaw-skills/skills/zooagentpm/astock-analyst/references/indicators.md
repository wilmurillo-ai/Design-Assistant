# 技术指标判断手册

## OBV（能量潮）

**原理**：涨日加量，跌日减量，累计反映资金净流向。

**判断**：
- OBV持续上行 → 资金净流入，趋势健康
- OBV下行 + 股价上涨 → 背离，主力出货，危险
- OBV下行 + 股价下跌 → 正常下跌，无异常
- 比较近期OBV vs 10日前OBV，简单判断方向

**陷阱**：OBV单次大量会造成短期扭曲，结合涨跌量不对称比交叉验证。

---

## 涨跌量不对称比

**公式**：`跌日总成交量 ÷ 涨日总成交量`（近60日）

| 数值 | 含义 |
|------|------|
| < 1.0 | 涨日量更大，买盘强势 |
| 1.0 ~ 1.2 | 正常范围 |
| > 1.2 | 跌日量大，出货特征 ⚠️ |
| > 1.5 | 强烈出货信号 ❌ |

**配合使用**：不对称比高 + OBV下行 = 主力系统性出货，坚决不进。

## MACD

**参数**：EMA(12), EMA(26), DEA=EMA(DIF, 9)

**判断**：
- DIF > DEA → 金叉（做多信号）
- DIF < DEA → 死叉（做空信号）
- DIF在零轴上方金叉 > DIF在零轴下方金叉（强弱之分）
- MACD柱由负转正 → 动能转换
- DIF持续下行即使金叉 → 弱金叉，不可盲信

**实战**：MACD金叉是加分项，不是必要条件；死叉是减分，配合其他指标综合判断。

## RSI（14日）

| 数值 | 含义 |
|------|------|
| < 30 | 超卖，可能反弹 |
| 30 ~ 40 | 偏弱，谨慎 |
| 40 ~ 70 | 健康区间 ✅ |
| 70 ~ 80 | 偏强，注意回调 |
| > 80 | 超买，高风险 ❌ |

## 布林带

**参数**：中轨=MA20，上下轨=MA20 ± 2σ

**判断**：
- 价格突破上轨 → 超买，两种走法：强势横盘消化 or 回踩中轨(-14%)
- 价格在中轨附近 → 位置合理，均衡
- 价格跌破下轨 → 超卖，可能反弹，也可能继续跌
- 布林带收窄 → 即将变盘，方向不明

**计算**（需至少20日数据）：
```python
import statistics
window = closes[-20:]
mid = sum(window) / 20
std = statistics.stdev(window)
upper = mid + 2 * std
lower = mid - 2 * std
pct = (closes[-1] - lower) / (upper - lower) * 100  # 布林带位置%
```

## 均线体系

| 均线 | 含义 | 用途 |
|------|------|------|
| MA5 | 超短期 | 日内/短线方向 |
| MA10 | 短期 | 短线支撑压力 |
| MA20 | 中短期 | 关键支撑，跌破警惕 |
| MA60 | 中期 | 趋势判断，必须在上方才入场 |
| MA120/250 | 长期 | 牛熊分界（参考） |

**实战规则**：
- 入场必须在MA60上方
- MA20是持仓最后防线，跌破考虑止损
- 多头排列（MA5>MA10>MA20>MA60）= 趋势最健康
- 股价连续跌破多条均线 = 趋势破坏，不摸底

## 期望值公式

```
EV = P(涨) × 平均涨幅 + P(跌) × 平均跌幅
```

- 用近60日日线计算涨跌概率和平均幅度
- EV > 0：赔率合理，可以考虑入场
- EV < 0：赔率不利，不操作
- EV只是历史统计，不是预测；配合当前趋势综合判断

## 量能分析

| 信号 | 判断 |
|------|------|
| 放量上涨（>1.5x均量）| 主力参与，趋势确认 |
| 缩量上涨 | 浮筹少，可能持续但注意随时变脸 |
| 巨量（>3x）高开低走 | 短期见顶，主力出货 |
| 放量下跌 | 恐慌性抛售或主力砸盘 |
| 缩量下跌 | 无量阴跌，等量能回来再判断 |
| 量能持续萎缩 | 流动性差，关注度下降，谨慎 |

均量计算：近20日（排除最近2日异常）的平均成交量。

## 综合判断框架

**入场检查清单**（按顺序，一票否决）：
1. MA60上方？否 → 不入
2. OBV上行？否 → 不入
3. 不对称比<1.2？否 → 不入
4. 近3月涨幅>0%？否 → 不入
5. 期望值>0？否 → 不入
6. MACD金叉/RSI健康/放量？加分
7. 风险点能否推翻自己？写出来

**止损原则**：
- 设好止损再入场，不入场不设止损
- 跌破MA20考虑减仓，跌破止损线无条件出
- 止损后不追悔，记录原因，下次改进
