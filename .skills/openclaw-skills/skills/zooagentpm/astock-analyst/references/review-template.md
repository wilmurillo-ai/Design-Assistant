# 每日复盘模板

收盘后16:00执行，写入 `memory/daily-stock-analysis.md`，教训写入 `memory/learnings.md`。

---

## 复盘结构

### 一、昨日判断自我校验

对每只股票，逐一核对：

```
【股票名】
- 昨日判断：[涨/跌/持有/止损/观望]
- 今日实际：[涨X% / 跌X% / 触发止损 / ...]
- 判断结果：[正确 / 错误]
- 对了为什么？（哪个指标给了正确信号）
- 错了为什么？（哪个指标误判/忽略了什么）
- 下次怎么改？
```

错误必须写入 `memory/learnings.md`，格式：
```
## YYYY-MM-DD 错误记录
- 股票：XXX
- 错误：[描述]
- 原因：[分析]
- 改进：[具体行动]
```

### 二、持仓更新

| 股票 | 股数 | 成本 | 今收 | 浮盈亏 | 止损线 | 止盈线 |
|------|------|------|------|--------|--------|--------|
| ... | ... | ... | ... | ... | ... | ... |

更新 MEMORY.md 持仓记录。

### 三、关注股扫描

对关注股票池每只跑一遍快速数据：
- 当日涨跌幅
- 是否触及入场价位
- 是否出现异动（量能、形态）
- 明日是否需要关注

### 四、明日操作计划

```
【持仓股】
- 瑞芯微：继续持有 / 止损价XXX
- 杭电股份：条件单挂着，关注是否触发

【关注股】
- 深水海纳：关注回调至14.8~15.2区间
- 青鸟消防：等MACD企稳+回调至11.0~11.2

【大盘判断】
- 明日情绪预判：乐观/中性/谨慎
- 理由：
```

### 五、今日市场总结（可选）

- 大盘今日表现
- 板块热点
- 对持仓股的影响

## 荐股复盘（如有新推荐）

每次推荐后，下一个复盘必须回顾：
- 推荐时的判断依据
- 推荐后的实际走势
- 是否验证了判断逻辑
- 如果跑偏了，哪个假设错了

---

## learnings.md 写入规范

只写有价值的内容，不写流水账：

**值得写入的**：
- 发现新的指标组合或规律
- 踩了坑（数据错误、判断失误）
- 验证了某个方法论
- 老大说了有价值的交易经验

**不需要写的**：
- 每天正常持有、无操作
- 重复已知的内容
- 没有新信息的观望

格式：
```markdown
## [日期] [主题]

**背景**：[一句话说清楚场景]
**发现/错误**：[具体内容]
**结论**：[可复用的规律或改进点]
```
