# 喜剧类型速查

> 根据素材特征推荐最适合的喜剧类型和手法组合。

## 类型总览

| 类型 | 时长 | 核心机制 | 适合素材 | 主要手法 |
|------|------|---------|---------|---------|
| 脱口秀 | 5-20分钟 | 个人叙事+观点反转 | 个人经历、社会观察 | 铺垫反转、call back |
| 小品 | 10-30分钟 | 角色冲突+情境升级 | 有鲜明角色的故事 | 弹簧魔鬼、雪球效应 |
| 漫才 | 5-15分钟 | 搭档配合+节奏堆叠 | 对话素材、误解场景 | 正反打、连续否认 |
| 剧本 | 60-120分钟 | 人物弧线+结构完整 | 需要完整故事的内容 | 性格喜剧、多线干扰 |
| 讽刺文 | 不限 | 换位+反讽 | 严肃议题、权威文本 | 庄→俗换位、反转 |
| 仿讽 | 不限 | 模仿+颠覆 | 有既定风格的文本 | 仿体+恶搞内核 |
| 荒诞剧 | 不限 | 系列干扰极致化 | 揭示无意义的主题 | 无因果链条、循环 |

---

## 脱口秀（Stand-up Comedy）

### 核心节奏

```
铺垫（建立世界观）→ 细节填充（让观众相信）→ 转折（笑点）→ call back（回顾）
```

### 结构特点

- **单一视角**：第一人称叙述者
- **核心单位**：一个笑话 = 铺垫+笑点
- **密度**：每30-60秒一个笑点
- **情感线**：通常有脆弱时刻（共鸣）+ 强势笑点（掌声）

### 适合手法

- 铺垫-反转（Expectation vs Reality）
- Call back（前面埋的笑点后面再抖）
- Callback to self（用自己打脸自己）
- 身份错位（我是XXX，但我/他们做了YYY）

### 创作检查

- [ ] 每个段落有一个核心笑点
- [ ] 铺垫足够长让观众相信
- [ ] 转折足够意外但不突兀
- [ ] 有call back连接全篇

## 小品（Sketch Comedy）

### 核心节奏

```
建立情境 → 引入冲突 → 冲突升级（弹簧/雪球） → 极端爆发 → 收尾
```

### 结构特点

- **多角色**：2-5人
- **核心单位**：一个场景 = 一个情境
- **密度**：每个场景2-3个笑点
- **情感线**：可以有温情但点到为止

### 适合手法

- 弹簧魔鬼（冲突反复被打压又弹起）
- 雪球效应（一个小动作引发大灾难）
- 角色执念（一个人物坚持某种荒谬行为）

### 创作检查

- [ ] 情境建立够快（前30秒）
- [ ] 冲突升级有层次（至少3层）
- [ ] 每个角色有一个鲜明的执念
- [ ] 结尾要么爆发要么反转

## 漫才（Manzai）

### 核心节奏

```
正论 → 否定 → 更荒谬的正论 → 再次否定 → 极致荒谬
```

### 结构特点（正反打）

```
漫才演员A（上手）：提出正经/荒谬的观点
漫才演员B（下手）："不对！"
A：坚持原观点，但理由更荒谬
B："还是不对！"
A：理由极度荒谬
B："不是，你听我说……"（放弃反驳）
→ 笑
```

### 适合手法

- 连续否定（连续三次反驳，每次理由更荒谬）
- 极端化（话题逐级推向荒谬极端）
- 逻辑滑坡（从一个正常前提推到荒谬结论）

### 与相声的区别

- 相声：捧逗有信息差（一个装傻一个精明）
- 漫才：两人都在装傻，配合节奏

### 创作检查

- [ ] 有明确的"上手"和"下手"
- [ ] 否定的升级有节奏感
- [ ] 最后的荒谬有冲击力

## 讽刺文（Satire）

### 核心节奏

```
严肃陈述（建立权威感）→ 逐渐揭示荒谬 → 完全反转 → 余韵
```

### 结构特点

- **无角色**：作者直接发言
- **核心单位**：一个讽刺段落 = 庄重外表+荒谬内核
- **情感线**：表面义正言辞，内核辛辣

### 适合手法

- 换位（庄→俗）：用正式语言描述琐事
- 反讽（Irony）：说的和实际相反
- 归谬法：从对方逻辑推出荒谬结论
- 夸张放大：把缺点/荒谬放大到荒谬程度

### 创作检查

- [ ] 表面足够严肃
- [ ] 笑点来自对严肃的反转
- [ ] 讽刺的是行为/系统，不是个人
- [ ] 有明确立场但不直接骂人

---

## 仿讽（Parody）

### 核心节奏

```
模仿原文风格 → 替换内核 → 产生反差笑果
```

### 结构特点

- **有原文**：必须熟悉被仿讽的文本/风格
- **核心单位**：一次成功的仿讽 = 精准模仿+意外内核

### 适合手法

- 风格模仿（语言、结构、节奏）
- 内核替换（用荒诞内容替换原正经内容）
- 结构复用（旧瓶装新酒）

### 创作检查

- [ ] 足够熟悉原文风格
- [ ] 模仿足够精准让观众认出
- [ ] 替换的内核产生反差
- [ ] 不侵权（仿讽是合理使用）

---

## 荒诞剧（Absurdist Theatre）

### 核心节奏

```
无原因开始 → 无逻辑发展 → 无意义持续 → 也许有隐喻的结尾
```

### 结构特点

- **无意义**：事件之间无因果关系
- **重复循环**：同样的对话反复出现
- **哲学内核**：无意义背后的意义

### 适合手法

- 系列干扰的极致化（两条线索都不通顺）
- 机械重复（没有任何进展）
- 反高潮（预期的大事件变成虚无）

### 创作检查

- [ ] 有意打破因果逻辑
- [ ] 重复有变化（不是简单的原地重复）
- [ ] 荒谬背后有可解读的隐喻
