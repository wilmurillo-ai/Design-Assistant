# 喜剧创作核心原理

## 核心公式

**喜剧 = 镶嵌在活物上的机械装置**

当人的行为表现出机械性、僵硬性、固执性时，喜剧效果便产生。

## 笑的三个条件

1. **仅存在于人类领域** — 风景不笑，动物可笑只因像人
2. **需要不动情的旁观** — 情感是笑的敌人，冷眼旁观才能发笑
3. **需要社会回声** — 笑是一群人的笑，需要共鸣和同谋感

## 笑的社会功能

笑是一种社会性示意：
- 纠正偏离社会的僵硬行为
- 使人努力表现出"应该的样子"
- 喜剧 = 社会的弹性装置，恢复社会的柔软

## 喜剧手法体系

### 情境喜剧三大基本手法

从"生命 vs 机械"的对立反推：

| 生命特征 | 反转 = 喜剧手法 | 说明 |
|---------|----------------|------|
| 持续变化 | **重复** | 同一场景反复出现，逐级升级 |
| 不可逆性 | **反转** | 角色互换，迫害者自作自受 |
| 个体封闭性 | **系列干扰** | 两个独立事件系列交叉碰撞 |

### 三大具体手法

**弹簧魔鬼**
- 被压制的力量反复弹起，每次更猛烈
- 应用：反复被打断又重新开口；一个执念被压制又冒出来

**雪球效应**
- 微小起因导致巨大后果，效应自我增强
- 应用：连锁反应、蝴蝶效应、误会叠加

**系列干扰**
- 两条独立事件线索交叉，产生误会
- 应用：信息差、认知错位、误会导致的荒诞后果

### 语言喜剧手法

**换位**
- 庄→俗：滑稽模仿，用市井语言描述高贵事物
- 俗→庄：幽默，用正经语气描述卑鄙行为

**反转**
- 主宾对调，让对方自食其果

**现成句式+荒谬内容**
- 刻板语言表达荒谬内容

**字面理解比喻**
- 假装以字面意思理解比喻，暴露物质性

### 性格喜剧三条件

1. **不合群性** — 不愿与他人接触，自我封闭
2. **无动于衷** — 观众不产生情感共鸣
3. **自动性** — 角色越不自觉，越可笑

> 喜剧人物通常是一个心不在焉的人

**喜剧 vs 悲剧的本质区别：**
- 喜剧 → 类型、普遍性、可重复
- 悲剧 → 个体、独特性、不可再现

## 喜剧创作黄金八法则

1. **机械化法则** — 让活人表现出机器般的僵硬
2. **心不在焉法则** — 角色越不自觉，越可笑
3. **重复法则** — 重复同一动作/台词增强喜剧效果
4. **升级法则** — 喜剧效果逐级递增
5. **反差法则** — 庄严与琐碎的对立
6. **误判法则** — 角色对处境的错误判断（而观众知道真相）
7. **无恶意法则** — 保持不动情，避免观众同情
8. **社会性法则** — 喜剧针对不合群/僵硬的行为
