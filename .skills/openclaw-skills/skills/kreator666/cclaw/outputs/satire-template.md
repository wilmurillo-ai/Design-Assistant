# 讽刺文输出模板

> 适用：无角色散文讽刺，篇幅不限。核心是用庄重语言揭示荒谬。

## 讽刺文 vs 其他喜剧类型的区别

| 类型 | 是否有角色 | 核心机制 |
|------|-----------|---------|
| 讽刺文 | 无（作者直接发言） | 换位+反讽 |
| 仿讽 | 有（模仿特定对象） | 模仿+替换 |
| 小品 | 有（角色扮演） | 情境冲突升级 |
| 脱口秀 | 有（第一人称叙述） | 铺垫-反转 |

## 输出规格

```
【讽刺文标题】

【开篇：庄严切入】
用最正式、最学术、最官方的语气开篇
建立权威感，让读者以为在看正经文章

【中段：逐层揭示荒谬】
每段揭示一个荒谬层面
从宏观到微观，从系统到个人
语言始终保持庄重（反差产生笑）

【高潮：荒谬汇聚】
把所有荒谬串联起来
用正式语气说出最荒谬的结论

【收尾：余韵或反转】
一个戛然而止的荒谬结尾
或一句看似正确实则讽刺到极致的话
```

## 创作步骤

### Step 1：找准讽刺对象

讽刺对象 = 一种系统性的荒谬（不是针对个人）

### Step 2：梳理荒谬的层次

从浅到深列出荒谬的3-5个层面。

### Step 3：设计语言策略

| 讽刺强度 | 语言策略 |
|---------|---------|
| 温和讽刺 | 正话反说，让事实说话 |
| 中度讽刺 | 正式语言+荒谬内容 |
| 辛辣讽刺 | 归谬法——把对方逻辑推到极致 |

### Step 4：保持不动情

讽刺文最大的敌人是愤怒。一旦愤怒，讽刺就变成了控诉。
保持冷眼旁观的姿态——这是"社会性示意"，不是控诉。

## 手法清单（讽刺文优先）

| 手法 | 说明 |
|------|------|
| 庄→俗换位 | 用正式语言描述琐碎/荒谬 |
| 归谬法 | 从对方逻辑推出荒谬结论 |
| 数据冷幽默 | 用真实数据制造反差 |
| 反讽（Irony） | 说的和实际意思相反 |
| 反复强调 | 强调荒谬到让荒谬自己暴露 |
