# 小品输出模板

> 适用：2-5人场景喜剧，10-30分钟。核心是角色冲突 + 情境升级。

## 输出规格

### 结构规范

```
【小品标题】

【人物表】
角色名：特征描述

【第一幕：情境建立】（约2分钟）
- 快速介绍场景和人物
- 每人展示一个核心执念/习惯
- 建立基本冲突

【第二幕：冲突升级】（约5-15分钟，分2-3个场景）
- 弹簧魔鬼：冲突被打压→弹起→更强打压→完全爆发
- 或：雪球效应：一个小动作引发连锁反应
- 角色执念逐渐暴露，越来越荒谬

【第三幕：极端爆发 + 收尾】（约1-2分钟）
- 所有执念同时发作到极致
- 结局荒谬但有逻辑
```

## 创作步骤

### Step 1：设计 2-3 个核心执念

每个主要角色有一个执念。执念 = 机械僵硬的核心。

### Step 2：设计冲突类型

| 冲突类型 | 适合场景 | 核心手法 |
|---------|---------|---------|
| 执念 vs 现实 | 一个固执的人遇到变化 | 弹簧魔鬼 |
| 执念 vs 执念 | 两个人互不相让 | 弹簧+雪球 |
| 信息差冲突 | 双方认知不一致 | 系列干扰 |

### Step 3：设计升级路径

```
第一层：执念只影响小事
第二层：执念开始影响核心冲突
第三层：执念导致完全荒谬的后果
```

### Step 4：安排肢体喜剧

小品有可视化的肢体动作。设计：
- 每个角色的标志性动作（重复出现）
- 场景中的道具喜剧
- 走位调度

## 手法清单（小品优先）

| 手法 | 说明 |
|------|------|
| 弹簧魔鬼 | 冲突被打压又弹起 |
| 雪球效应 | 小因→大果连锁反应 |
| 角色执念 | 每个人坚持荒谬行为 |
| 无意识暴露 | 角色不知道自己在干什么 |
| 重复+升级 | 同一动作/台词逐次升级 |
