# 脱口秀输出模板

> 适用：单人口述喜剧，5-20分钟。核心是"我"的视角 + 铺垫-反转节奏。

## 输出规格

### 结构规范

```
【脱口秀标题】

【开场 Hook】（钩子，30秒内抓住注意力）

【第一段：建立世界观】
铺垫内容 → 核心笑点
（2-3个铺垫-反转段落）

【第二段：深挖细节】
更多细节 → 更深层的笑点
（call back 到开场 hook）

【脆弱时刻】（可选，1处）
制造共鸣的真诚段落

【第三段：疯狂升级】
笑点密度加大，节奏加快
（3-4个快节奏段子）

【收尾 Call Back】
回归开场钩子，但用更荒谬的方式
```

## 创作步骤

### Step 1：确定核心主题句

一句话说清你想讲什么。问自己：这里有什么"生命的机械化"？

### Step 2：找三个故事/观察

脱口秀最好有具体细节。三个故事/观察点，形成递进。

### Step 3：设计铺垫-反转结构

```
铺垫 = 让观众往 A 方向想
笑点 = 揭示其实是 B 方向

铺垫要"真"——观众必须相信才能笑
反转要"快"——不要铺垫太久才反转
```

### Step 4：埋 Call Back

在第一段或第二段埋一个笑点素材，在结尾用更荒谬的方式调用。

## 手法清单（脱口秀优先）

| 手法 | 说明 |
|------|------|
|铺垫-反转|核心结构|
|Call back|前面埋的梗后面再抖|
|身份错位|我是X但我做了Y|
|逻辑滑坡|从一个正常前提推到荒谬结论|
|自我打击|先说一个观点然后自己反驳|
|细节堆叠|用超具体的细节制造荒谬|
|反向预期|观众以为你会说A，你说了B|

## 笑点密度控制

```
密集型：每30-45秒一个笑点
标准型：每60-90秒一个笑点  
稀疏型：笑点作为段落高潮点缀
```
