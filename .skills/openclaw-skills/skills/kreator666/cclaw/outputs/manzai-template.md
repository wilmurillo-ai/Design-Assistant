# 漫才输出模板

> 适用：2人对口喜剧，5-15分钟。核心是搭档配合 + 节奏堆叠。源自日本，但结构通用。

## 漫才 vs 相声的核心区别

| | 相声 | 漫才 |
|--|------|------|
| 结构 | 捧逗有信息差 | 两人都在装傻，配合节奏 |
| 重点 | 包袱的质量（三番四抖） | 节奏的密度（连续否认） |
| 笑点位置 | 通常在捧哏的翻/捧 | 通常在上手下放弃反驳的瞬间 |

## 输出规格

### 结构规范（标准漫才）

```
【漫才标题】

【人物】
漫才演员A（上手/先辈）：提出话题 + 制造逻辑
漫才演员B（下手/後輩）：连续否定 + 追问

【第一段：建立话题】（约1分钟）
A 提出一个话题/观点
B 追问或质疑

【第二段：连续否定】（约3-8分钟）
B："不对！"
A：坚持观点，理由更荒谬
B："还是不对！"
A：理由更更荒谬
B："完全不是！"
A：理由极度荒谬
B："不是……算了，你继续吧"
→ 笑

【第三段：切换话题或极致升级】（约1-3分钟）
可以切换新话题重复同样节奏
或在一个话题上极致升级到荒谬

【收尾】
两人突然恢复正常，礼貌告别
（反差收尾）
```

## 创作步骤

### Step 1：确定话题 +荒谬方向

话题 = 一个日常现象/社会观察
荒谬方向 = 从哪个角度把这个话题推向荒谬

### Step 2：设计三次否定

```
第一次否定（B）：轻微质疑
→ A 的回应：稍微升级
第二次否定（B）：强烈质疑
→ A 的回应：更荒谬的理由
第三次否定（B）：彻底放弃
→ A 的理由已经荒谬到极致
```

### Step 3：节奏设计

漫才的笑点来自**节奏密度**，不是单点爆发。
每30-60秒一个笑点，用语速和肢体配合。

### Step 4：收尾设计

标准收尾：突然恢复正常礼貌对话（反差萌）

## 手法清单（漫才优先）

| 手法 | 说明 |
|------|------|
| 连续否认 | 三次以上否定，每次理由更荒谬 |
| 极致化 | 话题推向逻辑终点 |
| 逻辑滑坡 | 从正常前提推到荒谬结论 |
| 装傻配合 | 两人一起装傻 |
| 突然正常 | 极度荒谬后突然清醒收尾 |

## 格式注意

漫才脚本用**对话格式**，标注"（上）"和"（下）"：

```
上：XXX
下：XXX
上：XXX
下：XXX
```
