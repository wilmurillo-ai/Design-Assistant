# 剧本输出模板

> 适用：60-120分钟完整喜剧剧本。核心是人物弧线 + 完整结构 + 性格喜剧。

## 输出规格

```
【剧本标题】

【人物表】
角色名：执念/缺点 · 人物小传（2-3句）

【第一幕：建置】
介绍主要人物和世界
展示每个人的核心执念
建立基本冲突/目标

【第二幕：对抗】
主角追求目标过程中
执念不断阻碍
冲突逐步升级
中点转折

【第三幕：危机 + 高潮】
执念导致的最终爆发
冲突达到顶点
解决/荒谬收尾

【幕间】（可选）
核心笑点串联的过场
```

## 创作步骤

### Step 1：设计 3-5 个核心执念

每个主要角色有一个执念。执念 = 人格中被机械化的部分。

### Step 2：设计执念如何碰撞

每个场景 = 执念A vs 执念B
碰撞越多，笑点越多。

### Step 3：设计人物弧线

```
开场：角色在执念控制下
中点：执念导致严重后果
结尾：角色有了一丝觉醒（或完全没觉醒，更可笑）
```

### Step 4：安排多线干扰

如果场景够复杂，可以安排两条以上事件线交叉。

## 喜剧剧本 vs 普通剧本

| 要素 | 普通剧本 | 喜剧剧本 |
|------|---------|---------|
| 人物深度 | 必须有成长弧线 | 可以只有类型化人物 |
| 情感 | 有深度情感段落 | 情感点到为止，避免过度煽情 |
| 笑点 | 主要靠对白 | 笑点可以来自情境、动作、语言 |
| 结尾 | 通常有情感解决 | 可以是荒谬结局 |

## 手法清单（剧本优先）

| 手法 | 应用位置 |
|------|---------|
| 性格喜剧 | 每个角色围绕执念设计 |
| 弹簧魔鬼 | 核心冲突反复被打压又弹起 |
| 雪球效应 | 一条故事线引发的连锁反应 |
| 系列干扰 | 两条故事线交叉产生误会 |
| 换位 | 用正式语言说荒谬内容 |
| Call back | 主要笑点在中后段回呼 |

## 格式规范

```
【第一幕·第一场】

内景 咖啡馆 - 日

李老师正在看报纸。桌上摆满图表和数据。

李老师：（看表）下午三点十五分……（翻一页）三点十六分……
         PPI同比持平。M2同比增长12.3%。
         （叹气）又是一个不需要消费的日子。

……（继续）
```
