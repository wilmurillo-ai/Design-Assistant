# 仿讽输出模板

> 适用：模仿特定文本/风格进行喜剧创作。核心是精准模仿 + 意外内核。

## 仿讽 vs 讽刺的区别

| | 讽刺 | 仿讽 |
|--|------|------|
| 对象 | 一种荒谬现象/系统 | 一个特定的文本/风格 |
| 手法 | 换位、反讽、归谬 | 风格模仿 + 内核替换 |
| 效果 | 让荒谬暴露 | 让被模仿对象"说出真相" |

## 输出规格

```
【仿讽标题】——（注明仿对象）

【开头：完全复刻风格】
用最接近原文的语言、结构、节奏开头
让读者/观众完全沉浸在模仿氛围中

【中段：逐渐植入荒谬】
开始用原文风格讨论完全不同的主题
或开始揭示原文不愿意面对的真相

【高潮：完全颠覆】
用原文风格说出完全颠覆原文立场的观点
或用原文语言结构得出荒谬结论

【收尾：回呼开头】
最后一个段落完全呼应开头，但内核完全反转
```

## 创作步骤

### Step 1：彻底研究原文

仿讽的前提是模仿者比原文更懂原文。
必须完全熟悉：
- 原文的语言风格（用词、句式、节奏）
- 原文的结构（开头、发展、高潮、结尾）
- 原文的核心论点

### Step 2：找准替换角度

常见替换角度：
- 原文说的是A，仿讽说反A
- 原文说的是A，仿讽把A用到荒谬场景
- 原文忽略B，仿讽专谈B

### Step 3：保持形式、替换内核

最有效的仿讽 = 形式完全不变，内核彻底替换。
读者/观众在"形式熟悉"的安全感下，突然被"内核陌生"击中。

### Step 4：控制颠覆程度

- 太保守：没有笑果
- 太颠覆：读者认不出原文
- 最佳点：既认得出，又完全没想到

## 仿讽类型

| 类型 | 说明 | 例子 |
|------|------|------|
| 风格仿讽 | 模仿某作者/类型的语言风格 | "假如鲁迅写淘宝好评……" |
| 结构仿讽 | 模仿某经典作品的叙事结构 | "假如《黑客帝国》发生在办公室……" |
| 体裁仿讽 | 模仿公文/报告/论文的格式 | 用学术论文格式写"养猫的十大好处" |
| 混搭仿讽 | 同时模仿多个来源的风格 | "用新闻联播风格报道家里断网" |

## 手法清单（仿讽优先）

| 手法 | 说明 |
|------|------|
| 形式复刻 | 精准复制原文的语言节奏 |
| 内核替换 | 替换成荒谬/相反的内容 |
| 关键词保留 | 保留原文关键词，但改变语境 |
| 结构复用 | 沿用原文结构框架 |
| 反转揭露 | 在仿讽最后揭示原文的盲点 |

## 法律提示

仿讽属于合理使用，但注意：
- 不用于商业诋毁
- 不冒充原文作者
- 足够让受众识别出是仿讽而非原文
