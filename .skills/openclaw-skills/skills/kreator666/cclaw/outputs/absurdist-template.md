# 荒诞剧输出模板

> 适用：无因果、无意义的荒诞喜剧。核心是揭示无意义背后的意义。

## 荒诞剧 vs 其他喜剧的区别

| 类型 | 逻辑 | 因果 | 意义 |
|------|------|------|------|
| 小品 | 有 | 情境因果链 | 有明确主题 |
| 讽刺文 | 有 | 逻辑推理 | 有明确立场 |
| 荒诞剧 | 无 | 事件无关联 | 无意义 = 意义本身 |

## 输出规格

```
【荒诞剧标题】

【人物】（通常是扁平、无背景的符号化人物）
人物A：（可以是职位、编号、或无意义的词）
人物B：

【第一场：无意义的开始】
无原因地开始
无意义的对白
建立"这个世界的规则"

【第二场：无逻辑的发展】
事件之间没有因果关联
对话循环重复
角色在重复中逐渐变化（或完全不变）

【第三场：无结果的等待】
某种"等待"驱动全剧
等待的东西永远不会来
或者来了也没有意义

【收尾：循环回归】
回到开场状态
暗示一切可以重新开始
或者一切从未真正开始
```

## 创作步骤

### Step 1：确定"荒诞的核心主题"

荒诞剧不是无意义，而是把某种无意义**揭示**出来。
常见的荒诞核心：
- 沟通的不可能性
- 等待的徒劳
- 身份的丧失
- 意义的消解

### Step 2：设计"无意义的重复模式"

荒诞剧的核心节奏 = 重复 + 微小变化
找到一个核心动作/台词，在整个剧中反复出现，每次有微小变化。

### Step 3：打破因果逻辑

让事件之间**没有因果关系**，但有**时间上的承接**。
A发生了，然后B发生了——不是因为A，而是恰好在A之后。

### Step 4：保持克制的语调

越严肃地讲无意义，笑点越强。
不要用"荒谬"的语气讲荒谬，要用"日常"语气讲荒谬。

## 核心机制对照

| 喜剧手法 | 荒诞剧应用 |
|---------|----------|
| 反复压制与弹起 | 同样的对话反复弹起 |
| 无因连锁 | 无因的连锁，积累荒谬感 |
| 自动性 | 角色对彼此完全不了解 |
| 无社会性 | 角色之间无法形成互动 |

## 手法清单（荒诞剧优先）

| 手法 | 说明 |
|------|------|
| 无因果链 | 事件没有逻辑关联 |
| 循环重复 | 同一场景/对话重复出现 |
| 幽灵物体 | 一直在谈论但从未出现的东西 |
| 反高潮 | 预期的大事件变成虚无 |
| 语言解构 | 语言无法真正沟通 |

## 创作提示

荒诞剧的核心：生命在无意义的重复中机械运转，
但角色完全不觉察这种无意义——
这种**不自觉**本身，就是最大的喜剧来源。
