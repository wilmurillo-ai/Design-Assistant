# 用户命令配置

> 本文件记录 comedy-writer 技能支持的所有命令模式。
> 增删命令 → 修改本文件，SKILL.md 自动引用。

## 命令列表

| 命令关键词 | 输出类型 | 优先级 |
|-----------|---------|--------|
| 脱口秀 / standup / 单口喜剧 | 脱口秀 | 高 |
| 小品 / sketch / 情景喜剧 / 素描喜剧 | 小品剧本 | 高 |
| 漫才 / manzai / 日本漫才 | 漫才对口 | 高 |
| 剧本 / 喜剧剧本 / 独幕剧 | 完整剧本 | 高 |
| 讽刺 / satire / 讽刺文章 | 讽刺文 | 中 |
| 仿讽 / 戏仿 / 恶搞 / parody | 仿讽作品 | 中 |
| 荒诞 / 荒诞剧 / 荒诞喜剧 | 荒诞剧 | 中 |
| 段子 / 笑话 / joke / 搞笑段子 | 独立笑点 | 中 |
| 改编 / 改成喜剧 / 喜剧化 | 原文喜剧改编 | 高 |

## 默认命令

无指定类型时 → 根据素材判断最适合的类型，或询问用户。

## 命令扩展规则

新增命令格式：
```
| 关键词 / 同义词 | 输出类型 | 备注 |
```

扩展时直接追加行，无需修改 SKILL.md。
