# 常用城市 / 机场三字码（辅助）

`flyai search-flight` 通常接受 **中文城市名**；用户只说代码时，可对照下表填入 `--origin` / `--destination`（**若 CLI 不接受代码则改用中文名**）。

| 城市 | IATA 城市码（常见） |
|------|---------------------|
| 北京 | BJS |
| 上海 | SHA |
| 广州 | CAN |
| 深圳 | SZX |
| 杭州 | HGH |
| 成都 | CTU |
| 三亚 | SYX |
| 西安 | SIA |
| 昆明 | KMG |
| 重庆 | CKG |
| 南京 | NKG |
| 武汉 | WUH |
| 青岛 | TAO |
| 厦门 | XMN |
| 乌鲁木齐 | URC |

国际线以用户目的地为准；不确定时 **优先让用户确认城市中文名** 再查。
