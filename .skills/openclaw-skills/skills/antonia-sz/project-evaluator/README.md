# AI 项目评估助手

> 描述项目想法，AI 从市场/技术/商业/风险四维评估，输出报告 + 竞品速查 + MVP建议。

## ✨ 功能
- 四维评分（各10分）+ 综合评分
- 竞品速查（自动列出3-5个竞品）
- MVP 建议（最小可行版本）
- 明确结论：值得做 / 谨慎 / 不建议

## 🚀 快速开始
```bash
export DEEPSEEK_API_KEY=your_key
python3 scripts/evaluate_project.py --idea "做一个帮用户管理微信好友的工具"
python3 scripts/evaluate_project.py --idea "..." --output ~/report.md
```
