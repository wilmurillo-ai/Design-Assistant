# A3 PDF智能切分工具

这是一个专门用于将A3格式试卷PDF智能切分为A4格式的工具，自动识别试卷中间的最佳空白位置进行切割，避免切到文字内容。

## 功能特点
- ✅ 智能识别最佳切割位置，自动避开文字
- ✅ 支持横向A3（左右切分）和纵向A3（上下切分）
- ✅ 400DPI高分辨率输出，打印效果清晰
- ✅ 自动保留原PDF页面顺序
- ✅ 简单易用，只需提供输入输出路径

## 安装依赖
```bash
pip install pypdfium2 pillow
```

## 使用方法
### 命令行使用
```bash
python scripts/split_a3_smart.py "输入A3试卷.pdf" "输出A4版本.pdf"
```

### 在Skill中使用
当用户需要切分A3试卷时，系统会自动调用此工具。你只需要提供：
1. 输入PDF文件路径
2. 输出PDF文件路径

## 示例
```bash
python scripts/split_a3_smart.py "历史一、二单元4.3.pdf" "历史一、二单元4.3_A4切分版.pdf"
```

## 输出示例
```
第1页最佳切割位置：x=2480 (总宽度4960)
第2页最佳切割位置：x=2478 (总宽度4960)

处理完成！输出文件：历史一、二单元4.3_A4切分版.pdf
原始页数：2，切分后页数：4
```

## 适用场景
- 学校试卷打印
- 复习资料整理
- 办公文档格式转换
- 任何需要将A3 PDF转换为A4的场景

## 注意事项
- 输入文件必须是PDF格式
- 确保输出路径有写入权限
- 大文件处理可能需要较长时间，请耐心等待
- 建议输出文件名包含"_A4切分版"等标识方便区分
