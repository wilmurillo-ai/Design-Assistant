# prompt-buttons

Short, tappable button menus for OpenClaw agents.

## Install

```bash
clawhub install prompt-buttons
```

## Publish (maintainer)

```bash
# 1. Clone or navigate to this skill folder
cd skills/prompt-buttons

# 2. Login to ClawHub (one-time, opens browser)
clawhub login

# 3. Publish
clawhub publish . --slug prompt-buttons --name "Prompt Buttons" --version 1.0.0 --changelog "Initial release"
```
