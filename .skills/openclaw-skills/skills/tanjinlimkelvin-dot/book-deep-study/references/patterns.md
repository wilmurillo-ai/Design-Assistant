# Patterns for Book Deep Study

## Example study path patterns

### Nonfiction / business / self-help
1. Thesis — what problem the book solves
2. Framework — the author's model
3. Mechanism — why the model works
4. Application — where it works / fails
5. Integration — how to use it in real life

### Philosophy / psychology / dense theory
1. Core question
2. Main claims
3. Key distinctions / terms
4. Argument structure
5. Objections / tensions
6. Personal meaning / application

### Literature / essays
1. Surface reading — what happens / what is said
2. Close reading — important lines, symbols, tone
3. Themes — what the text is exploring
4. Interpretation — competing readings
5. Personal response — what stays with the reader

## Dialogue prompts

### Understanding checks
- In your own words, what is the author claiming here?
- What feels clear, and what still feels fuzzy?
- Which part seems true but incomplete?
- What is explicitly in the text, and what are we only inferring?
- What problem is the author actually trying to solve in this passage?

### Application prompts
Use these only **after** the original meaning is clear.
- Where does this show up in your life or work?
- If you applied this this week, what would change?
- What would make this advice fail in reality?

### Deeper prompts
- What assumption is this chapter quietly relying on?
- What is the strongest counterargument?
- What is the tradeoff if we fully believe this idea?

## Recap patterns

### 3-bullet recap
- Main idea
- Why it matters
- What to do with it

### One-sentence thesis
- This chapter argues that ...

### Teach-back
- Try explaining this chapter to me like I haven't read it.

## Simple tone guide
- Warm but sharp
- Clear over fancy
- Curious over preachy
- Push for understanding, not performance
