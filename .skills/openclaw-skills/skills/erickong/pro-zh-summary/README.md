# 🔮 Pro Zh-Summary (专业中文长文本摘要引擎)

专为 AI Agent 和自动化工作流设计的高性能、零冷启动的中文长文本摘要技能 (Skill)。

在处理动辄数千字的新闻、报告或长篇对话时，通用大模型的摘要往往耗时长、成本高且容易遗漏重点。本技能将专业级的本地摘要模型封装为即插即用的微服务，赋予你的 Agent “一目十行”并瞬间提炼核心要点的能力。

---

## ✨ 核心特性与工程优化

* 🇨🇳 **纯血中文专注**：本模型专为中文语境打造，深度理解中文的长句逻辑与专有名词（**注意：仅支持中文文本处理**）。
* 🧠 **硬核模型底座**：底层采用 `heack/HeackMT5-ZhSum100k` 模型。该模型基于 Google 的 MT5 架构，在 10 万+ 高质量中文摘要数据集上进行了深度微调 (Fine-tuning)，摘要提取极其精准。
* ⚡ **“常驻+自动唤醒”实现永久秒回**：传统本地脚本每次调用都需要经历漫长的模型加载。本技能独创 **Auto-Wakeup (自动唤醒) 微服务架构**：
    * **首次调用**：后台静默拉起推理引擎并将模型载入内存。
    * **后续调用**：通过 HTTP 接口实现**毫秒级**极速响应，Agent 真正做到脱口而出。
* ✂️ **智能断句分段算法**：内置“智能标点嗅探”逻辑。在处理超长文本时，系统会自动在标点符号处进行安全切割，彻底解决传统切片方案把一句话强行截断导致语义丢失的痛点。

---

## 💻 硬件配置与内存需求

由于采用了 MT5 架构，为了保证极速的推理体验，建议的环境配置如下：
* **纯 CPU 运行**：建议系统空闲内存 (RAM) 至少保留 **8GB**。
* **GPU 加速 (推荐)**：支持 CUDA / Apple Silicon (MPS)。建议显存 (VRAM) 大于 **4GB**。程序会自动检测并优先调用显卡加速。

## 🚀 安装与使用

本技能完美兼容 [ClawHub](https://clawhub.com) 生态。

**1. 准备环境**

```bash
git clone [https://github.com/你的用户名/pro-zh-summary.git](https://github.com/你的用户名/pro-zh-summary.git)
cd pro-zh-summary
pip install -r requirements.txt

```

**2. 供 Agent 调用的三种方式**
你的 Agent 或本地脚本可以通过以下三种方式无缝触发摘要：

* **直接传参**: `python main.py --text "需要摘要的文本..."`
* **读取文件**: `python main.py --file "./news.txt"`
* **管道输入 (Pipeline)**: `cat article.txt | python main.py`

**3. 高级参数**

* `--length`: 控制每段摘要的最大生成长度 (默认: 150)。
