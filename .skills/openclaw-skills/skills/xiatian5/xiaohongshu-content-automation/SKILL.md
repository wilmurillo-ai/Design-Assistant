# 小红书内容自动化工作流 📝✨

一站式小红书内容创作自动化工作流，从选题、撰写、排版到发布，帮助你快速产出高质量小红书内容。整合内容生成、标题优化、话题推荐、排版格式化全流程。

## 触发词

当用户说这些话时，调用这个技能：
- "帮我写小红书文案"
- "小红书内容自动化"
- "生成小红书帖子"
- "小红书选题推荐"
- "优化小红书标题"
- "自动发布小红书"

## 功能特性

| 功能 | 说明 |
|------|------|
| **智能选题** | 根据热点和关键词推荐热门选题方向 |
| **内容生成** | 生成符合小红书风格的文案内容 |
| **标题优化** | 提供多个标题选项，优化点击率 |
| **话题推荐** | 自动推荐相关热门话题 |
| **排版格式化** | 自动添加表情、分段、符合小红书风格 |
| **发布准备** | 生成可直接复制粘贴的最终内容 |

## 工作流程

### 标准工作流

```
1. 需求收集 → 主题/关键词/风格
2. 选题推荐 → 给出3-5个选题方向供选择
3. 内容生成 → 根据选题生成正文
4. 标题优化 → 生成5个候选标题
5. 话题推荐 → 推荐5-8个相关话题
6. 排版输出 → 输出格式化好的最终内容
```

## 内容风格指南

小红书内容特点：
- **开头吸引眼球** - 用表情+钩子式开头
- **短句分段** - 短段落，多用换行
- **表情点缀** - 适当使用emoji增加亲和力
- **干货要点** - 使用✅、💡、❗等符号突出重点
- **结尾互动** - 提问引导评论
- **话题标签** - 5-8个相关话题，包含大中小流量词

## 话题选择策略

- **大流量词** (1-2个)：#护肤 #健身 #美食 #旅行
- **中流量词** (2-3个)：#新手护肤 #健身入门 #家常菜
- **长尾词** (2-3个)：#混油痘肌护肤 #学生党健身攻略

## 依赖

- 需要 `xiaohongshu-cn` 基础技能
- 内容生成依赖大语言模型
- 自动发布需要配合浏览器自动化 (`playwright-browser-automation`)

## 最佳实践

1. **明确受众** - 确定是给新手还是进阶用户
2. **控制长度** - 小红书正文建议300-500字效果最佳
3. **加入个人体验** - 小红书用户喜欢真实体验分享
4. **Always 结尾提问** - 提问能提高互动率，增加流量
