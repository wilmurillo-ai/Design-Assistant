# 中文写作润色助手 ✍️🇨🇳

专业的中文写作润色工具，优化表达、修正语法错误、提升文采流畅度。支持多种文体，从日常邮件到正式文章，帮你写出更专业、更流畅的中文内容。

## 功能特性

| 功能 | 说明 |
|------|------|
| **语法修正** | 自动发现并修正错别字、语病、标点错误 |
| **表达优化** | 优化句子结构，让表达更流畅自然 |
| **文采提升** | 在保持原意的基础上提升语言质感 |
| **风格适配** | 支持正式/非正式/学术/日常等不同风格 |
| **多文体支持** | 文章、简历、邮件、社交媒体、议论文 |

## 润色维度

1. **准确性** - 修正错别字、语法错误、标点误用
2. **流畅性** - 优化句子连接，去除冗余表达
3. **准确性** - 用词更贴切，符合语境
4. **风格统一** - 保持整体风格一致
5. **逻辑连贯** - 优化段落和句子之间的衔接

## 润色示例

### 原文：
```
我今天早上来到公司之后，就开始了我的工作，然后我发现我昨天忘记把文件保存了，所以我不得不重新做一遍，这让我感觉非常的不开心。
```

### 润色后：
```
今天早上到公司后，我开始工作时发现昨天忘记保存文件，不得不重新做一遍，这让我很不开心。
```

### 修改说明：
- 去除冗余连接词（"之后"、"然后"、"这让我感觉"）
- "非常的不开心" → "很不开心"（副词"地"使用修正，同时更简洁）
- 整体句子更紧凑流畅

## 简历润色要点

- 用动词开头增强表现力
- 量化成果（"提升转化率15%" 比 "提升转化率" 更好）
- 删除空话套话
- 使用行业专业术语
- 保持简洁，一页简历控制在一千字以内

## 邮件润色要点

- 主题清晰明了
- 开头礼貌问候
- 段落简短，重点突出
- 结尾明确行动要求
- 落款规范

## 社交媒体文案润色要点

- 开头吸引眼球
- 短句分段，易于手机阅读
- 适当使用表情符号增加亲和力
- 结尾互动引导

## 最佳实践

1. **保持原意** - 润色不改变原文核心意思，只优化表达
2. **适度优化** - 不过度润色，保留作者个人风格
3. **给出说明** - 列出主要修改点，让用户明白为什么这么改
4. **保留选项** - 对于重要修改，可以提供多个版本供选择
