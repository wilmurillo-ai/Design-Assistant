# RSS 思考提示词

## 输入
阅读当日的 RSS 日报内容：
- Hacker News 热门
- Product Hunt
- GitHub Trending
- V2EX

## 思考框架

### 1. 赚钱计划
从当日热点中识别：
- **新工具/服务**: 哪些可以帮我赚钱？
- **市场需求**: 大家在讨论什么问题？
- **信息差**: 我能做什么别人还没做的？

### 2. 架构改进
从技术趋势中学习：
- **新技术**: 有哪些可以改进我的系统？
- **新工具**: 可以替换现有方案吗？
- **新思路**: 让我想到什么新功能？

### 3. 行动项
- 记录 1-3 个具体想法
- 标记优先级（高/中/低）
- 写下下一步行动

## 输出格式
```markdown
## 2026-03-15 RSS 思考

### 赚钱机会
- [高] xxx

### 架构改进
- [中] xxx

### 行动项
- [ ] xxx
```
