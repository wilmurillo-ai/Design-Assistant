## 💡 解决什么问题

### 痛点

- ❌ 想写日记但总是拖延
- ❌ 写了也是流水账，没有深度思考
- ❌ 写完就忘，没有转化为认知
- ❌ 无法坚持，三天打鱼两天晒网

### 解决方案

- ✅ **强制催促** - 每晚 23:00 自动触发，不写不行
- ✅ **思维模型** - 6 种思维模型深度分析，直击本质
- ✅ **记忆内化** - 自动转化为记忆条目，持续积累
- ✅ **Git 归档** - 版本控制，可追溯成长轨迹

## 🚀 核心功能

### 1. 自动催促

```
23:00 → 检查日记 → 没写？→ 催促用户
```

### 2. 交互式记录

```
用户口述 → AI 整理 → 结构化日记
```

### 3. 思维模型分析（核心卖点！）

| 模型 | 作用 |
|------|------|
| 第一性原理 | 拆解问题本质 |
| 逆向思维 | 从反面思考 |
| 二阶思维 | 考虑长期影响 |
| 概率思维 | 评估成功概率 |
| 机会成本 | 考虑放弃的价值 |
| 系统思维 | 整体视角 |

### 4. 记忆内化

```
日记洞察 → 记忆条目 → 持续积累
```

## 🔧 使用方式

### Cron 自动触发（推荐）

配置 OpenClaw Cron：

```yaml
# .openclaw/state/cron.yaml
- schedule: "0 23 * * *"  # 每晚 23:00
  command: "python scripts/diary_force.py auto"
```

### 手动触发

```bash
# 开始交互式日记
python scripts/diary_force.py start

# 完成后内化
python scripts/diary_force.py done

# 仅思维模型分析
python scripts/think.py "今天能量 5 分，尝试优化 openclaw 但效果一般"

# 查看今日状态
python scripts/diary_force.py status
```
