# ⚡ GitPulse · Git 提交日报生成助手

> 每天工作结束后，一句话触发，自动扫描当日所有 Git 提交记录，生成结构化日报。

---

## 简介

GitPulse 是一个零依赖的 Python 脚本，通过调用本地 `git log` 命令，扫描指定目录下的所有 Git 仓库，汇总**当天 06:00 ～ 次日 05:59** 时间窗口内当前用户的 commit 记录，输出格式化日报。

**适用场景**：
- 每日工作复盘
- 快速整理今天"干了什么"
- 多仓库并行开发时的提交汇总

---

## 特性

- ✅ **零额外依赖** —— 仅使用 Python 标准库
- ✅ **多仓库扫描** —— 自动发现子目录中的 Git 仓库（最深 3 层）
- ✅ **只看自己的提交** —— 自动读取 `git config user.name` 过滤当前用户
- ✅ **跨午夜统计** —— 凌晨还在敲代码也算"今天"（06:00 为日期分界）
- ✅ **支持历史日期** —— `--date` 参数查任意一天的记录

---

## 快速使用

```bash
# 扫描 e:\py 目录下所有仓库，生成今日日报
python scripts/gitpulse.py --root e:\py

# 查昨天的记录
python scripts/gitpulse.py --root e:\py --date 2026-04-09

# 输出 JSON 格式（供程序解析）
python scripts/gitpulse.py --root e:\py --format json
```

在 AI 助手中，直接输入以下任意关键词即可触发：

> `git日报` · `今日提交` · `今天写了什么` · `gitpulse`

## 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `--root` | 当前目录 | Git 仓库根目录 |
| `--date` | 今天 | 指定统计日期，格式 `YYYY-MM-DD` |
| `--format` | `text` | 输出格式：`text` 或 `json` |
| `--depth` | `3` | 子目录最大搜索深度 |
