# Reply Generation Guide

当 `pendingTweets` 中存在待处理推文时：

1. 读取第一条待处理推文
2. 基于其 `text`、`author`、`url` 生成 3 条中文回复候选
3. 默认输出这三种风格：
   - 朋友互动版
   - 技术点评版
   - 高情商简洁版
4. 候选回复要求：
   - 不要过长，优先 1~3 句
   - 不要像机器人复读原文
   - 尽量像真人评论，不要过度谄媚
   - 避免空泛夸夸，尽量带一点具体判断
   - 如果原推明显是 reply / 问答 / 招呼语，可改成轻互动风格
5. 输出格式建议：

```markdown
发现新推：
- 作者：@xxx
- 链接：https://x.com/xxx/status/...
- 内容：...

候选回复：
1. ...
2. ...
3. ...
```

只有当用户明确选择其中一条或要求修改后，才进入浏览器发送步骤。
