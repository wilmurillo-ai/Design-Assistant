# Agent 管理专家 (Agent Manager Skill)

这是一个专为 OpenClaw 打造的核心技能（Skill），旨在赋予 AI 助手**自我繁衍和管理**的能力。通过这个技能，你可以直接在聊天框里用自然语言让 OpenClaw 帮你创建新的子 Agent、配置工作区，甚至全自动完成飞书机器人的注册与绑定。

## ✨ 核心特性

- **一句话建机器人**：只需告诉 AI “帮我建一个负责处理报销的助理”，它会自动推断 ID、撰写人设（SOUL.md）并初始化干净的独立工作区。
- **飞书无缝集成**：告别繁琐的开发者后台配置！直接在聊天框发送飞书原生交互卡片，一键唤起侧边栏授权，自动获取凭证并绑定。
- **安全的权限控制**：默认创建全员可用的飞书机器人，但你可以随时要求开启“身份认证（Pairing）”来保护私密 Agent。
- **优雅的防冲突机制**：自动清理新 Agent 工作区中可能导致 Git 嵌套冲突的冗余文件，并在重启网关前主动向你请示。

## 📂 目录结构

```text
agent-manager/
├── SKILL.md                  # 技能说明与提示词（大模型看的文档）
├── README.md                 # 技能介绍（你正在看的文档）
└── scripts/
    ├── manage_agent.py       # 核心调度脚本（创建Agent、写配置、清理工作区）
    ├── feishu_auth.py        # 飞书 OAuth 授权流与轮询脚本
    └── verify_bindings.py    # 飞书路由绑定关系检查脚本
```

## 🚀 如何使用？

无需任何前置配置。在安装了此技能的 OpenClaw 会话中，你可以直接对大模型说：

> *"我想建一个新助手，专门负责查股票，需要绑定飞书，并且仅我个人可用（需要认证）。"*

大模型会按照以下标准工作流响应你：

1. **信息确认**：向你发送一份详细的《创建计划确认》表单。
2. **飞书授权**（若需要）：给你发送一张精美的飞书交互卡片，点击即可在端内完成扫码授权。
3. **自动执行**：当你回复“好了”之后，它会在后台飞速建好 Agent、写好 JSON 配置，并清理环境。
4. **请示重启**：一切就绪后，它会询问你是否现在重启网关让新机器人上线。

## 🛠️ 手动调用脚本（高级用户）

如果你更喜欢在终端（Terminal）里操作，这些内置脚本也完全支持独立运行。

**创建一个不绑飞书的本地 Agent：**
```bash
python3 scripts/manage_agent.py create \
  --name "数据分析师" \
  --id "data-analyst" \
  --description "负责处理 Excel 和生成图表"
```

**检查当前的飞书路由绑定状态：**
```bash
python3 scripts/verify_bindings.py
```
