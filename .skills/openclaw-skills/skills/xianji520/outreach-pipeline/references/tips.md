# 送达与账号配置提示

- 域名与身份
  - 使用独立业务域名；配置 SPF、DKIM、DMARC 提升送达
  - From 和 Reply-To 一致性，签名中包含公司/地址（合规）
- 速率与预热
  - 新域名/新 IP 从低速开始（如 5–10 封/分钟，逐步提升）
  - 控制每日总量（50–200 封/邮箱起步，逐渐增加）
- Gmail/Outlook
  - 推荐 App Password 或 OAuth；避免明文密码存文件
  - 注意免费帐号的发送阈值与限制
- 退订与合规
  - 在模板底部提供退订说明/链接（即使是 B2B 也建议提供）
  - 不发送给未经过任何同意或缺乏正当利益的个人邮箱
