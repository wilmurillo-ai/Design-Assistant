# HexaLotto 六爻奇门测彩 V2.0

> 基于传统六爻排盘逻辑的多彩种彩票推演娱乐 Skill

**支持三大彩种**：双色球 (SSQ) · 大乐透 (DLT) · 六合彩 (Mark Six)

**同时推算**：主号（红球/前区/平码）+ 特殊球（蓝球/后区/特码特肖）

## 功能亮点

- 🎯 **全号段覆盖** — 每个彩种同时推荐主号和特殊球，不只算一半
- 🔮 **完整六爻引擎** — 纳甲装卦、旺衰评估、回头生克、病药应期，7条规则 + fallback
- 📡 **自动获取号码** — 通过 web_search + web_fetch 自动搜索最新开奖结果，无需手动输入
- 🐏 **六合彩动态生肖表** — 按年自动偏移，内置 2025/2026 预置表
- 🧮 **大乐透智能补位** — 5 个前区号码自动补第 6 爻（求和法 / 干支借位法）

## 快速开始

```bash
# 安装
clawhub install hexalotto

# 使用 — 直接对 OpenClaw 说：
"帮我算一卦下期双色球"
"预测六合彩下期特码"
"大乐透起一卦"
```

## 输出示例

```
双色球：红球: 01 06 07 08 09 13 / 蓝球: 07
大乐透：前区: 04 06 08 09 10 / 后区: 08 10
六合彩：平码: 03 05 06 11 12 15 / 特肖:【羊】-> 12 24 36 48
```

## 技术栈

- 纯 Python，零外部依赖
- 干支历法推算（无需 lunar-python）
- 加权评分系统（动爻 3 分 / 静爻 1 分 + 特殊状态加权）
- 号码获取：OpenClaw web_search + web_fetch 自动抓取最新开奖数据
