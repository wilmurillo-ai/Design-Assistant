## 已知问题

### v1.0.0

1. **全黑图片识别率低**
   - 影响：某些暗色主题网站识别困难
   -  workaround：使用 `--debug` 模式调整参数
   
2. **新型验证码支持有限**
   - 影响：未见过验证码类型可能无法识别
   - 解决：使用 debug 模式分析，提交 Issue 反馈

3. **headless 模式容易被检测**
   - 影响：使用 headless 浏览器可能验证失败
   - 解决：使用真实浏览器（非 headless）
