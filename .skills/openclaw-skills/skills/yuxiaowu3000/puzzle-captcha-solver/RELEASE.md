### 2. 详细错误提示
```
⚠️  未检测到验证码弹窗
可能原因：
  1. 截图时机不对，验证码未加载完成
  2. 使用了全屏截图，验证码区域太小
  3. 验证码样式与已知模式不匹配
建议：
  - 增加页面等待时间：agent-browser wait 3000
  - 使用全屏截图：agent-browser screenshot --full captcha.png
  - 使用 debug 模式查看详细检测结果：--debug
```
