# 安全等级详解

## 四级安全等级

| 等级 | 类型 | 操作范围 | 门控 |
|------|------|---------|------|
| 🟢 L1 | 知识层 | MEMORY.md、日志、笔记 | 自动执行 |
| 🟡 L2 | 技能层 | SKILL.md（纯文档）、纯文档skill模板 | 自动执行+记录 |
| 🟠 L3 | 脚本层 | scripts/、工具脚本、所有涉及代码/prompt/tool参数的修改 | **@董事长审批** |
| 🔴 L4 | 系统层 | AGENTS.md、SOUL.md、openclaw.json | **@董事长审批+备份** |

## L2/L3 边界修正

任何涉及以下内容的修改一律升为 L3 需审批：
- 代码逻辑
- prompt 逻辑
- tool 调用参数

## 绝对禁区（不进化）

```
❌ credentials/ — 任何凭证文件
❌ .env — 环境变量
❌ crontab — 定时任务配置
❌ 他人隐私文件 — 未经授权不得访问/修改
```

## 递归腐化熔断机制

如果进化过程本身出错（≥2次连续失败），进入 **Lockdown 状态**：

```
Lockdown状态：
- 禁止任何自动进化（L1/L2也暂停）
- 所有改进必须走 @董事长审批
- 必须由猫经理确认问题修复后才能解除
- 解除方式：在 GENES.md 中注明"Lockdown已解除，原因：[说明]"
```

**触发 Lockdown 的条件：**
- 连续2次进化引入错误或回归
- 进化脚本执行失败
- GENES.md 写入失败
