## 常见问题

**Q：需要什么 API Key？**
A：核心需要 SerpAPI（https://serpapi.com），用于 Yandex/Google 俄语搜索。可选 TGStat（用于Telegram社群分析）。

**Q：数据多久更新一次？**
A：报告数据截止当天。所有新闻/政策数据均为实时采集，患者统计数据使用IDF最新版本（当前为2025版，统计至2024年）。

**Q：报告语言是什么？**
A：中文正文，附俄文原文术语（如 глюкометр = 血糖仪），数据来源标注原文名称。

**Q：可以生成其他国家报告吗？**
A：可以，只需将「俄罗斯」替换为其他国家（如哈萨克斯坦、白俄罗斯），数据源矩阵自动适配。
