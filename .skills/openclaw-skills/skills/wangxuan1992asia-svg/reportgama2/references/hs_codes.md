## 七、查询方式

### UN Comtrade 查询
1. 访问 https://comtrade.un.org
2. 注册免费账户
3. 选择 Reporter Country → Russia
4. 输入 HS Code → 901890
5. 选择年份 → Download

### OEC (经济复杂度数据库)
- 访问 https://oec.world
- 选择国家 → Russia
- 选择产品类别 → Medical Devices

### 俄罗斯联邦海关 (ФТС)
- http://www.customs.ru
- 进出口统计数据 → 按商品编码查询
