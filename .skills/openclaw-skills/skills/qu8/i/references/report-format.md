# 统计报告输出格式

生成报告时严格按以下 Markdown 格式输出。

## 报告模板

```markdown
# 工作记录导出

**导出时间**：{yyyy年MM月dd日 HH:mm}
**记录期间**：{start_date} - {end_date}
**记录总数**：{N}条

---

## 记录明细

### 1. {work_date} {time_info（非"未标注"时显示）}
| 字段 | 内容 |
|------|------|
| **工作内容** | {content} |
| **工作类型** | {work_type} |
| **事务性质** | {planning} |
| **重要性** | {importance（未标注时不显示此行）} |
| **紧迫性** | {urgency（未标注时不显示此行）} |
| **完成质量** | {quality（未标注时不显示此行）} |
| **对接人员** | {contacts 逗号连接，空时写"无"} |
| **对接人数** | {contact_count}人 |

---

（每条记录重复上述格式）

## 统计汇总

| 统计维度 | 数量/占比 |
|----------|-----------|
| **总记录数** | {N}条 |
| **计划内事务** | {n}条（{%}%） |
| **临时事务** | {n}条（{%}%） |
| **总对接人数** | {sum(contact_count)}人 |

### 工作类型分布
| 工作类型 | 数量 | 占比 |
|----------|------|------|
| {类型} | {n}条 | {%}% |

---

## 待办事项

（仅在存在 is_todo=true 且 todo_done=false 的记录时显示此节）

- [ ] {content}

---

*本报告由工作记录助手自动生成*
```

## 时间段解析规则

| 用户输入 | 解析方式 |
|---------|---------|
| 本周 | 本周一 00:00 ~ 本周日 23:59 |
| 上周 | 上周一 00:00 ~ 上周日 23:59 |
| 本月 | 本月 1 日 ~ 本月最后一天 |
| 上月 | 上月 1 日 ~ 上月最后一天 |
| 今天 | 当天 00:00 ~ 23:59 |
| 2026-02-26 至 2026-02-28 | 自定义范围 |
| 2月 | 2026-02-01 ~ 2026-02-28 |
