# 如何贡献 Pacer

## 最欢迎的贡献类型

1. **新行业的示范对话** — 医疗、制造、教育等垂直场景
2. **提示词优化** — 让Pacer的回复更准确、更有温度
3. **多语言支持** — 目前支持中文和英文
4. **真实用户案例** — 脱敏后的真实使用故事

## 贡献流程

1. Fork 本仓库
2. 创建你的分支：`git checkout -b feature/your-feature`
3. 提交改动：`git commit -m 'Add: 描述你的改动'`
4. Push：`git push origin feature/your-feature`
5. 开一个 Pull Request

## 提示词贡献规范

- 改动提示词时，请附上改动前后的对话对比示例
- 说明改动的理由
- 在 `examples/` 目录下提供至少一个测试对话
