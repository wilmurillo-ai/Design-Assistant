## 它能做什么

Pacer 是一个四阶段的职业陪跑系统，每一步都有可视化图表输出：

| 阶段 | 触发方式 | 输出 |
|------|----------|------|
| 📄 **扫描现状** | 上传CV / 文字描述自己 | 现状快照卡（6维度） |
| 🗺️ **规划路径** | 有目标→深度规划，没目标→多方向对比 | 技能雷达图 + 阶段甘特图 |
| 🔮 **模拟未来** | "如果我选择..." / "模拟一下X个月后" | 双轨交互式时间轴 |
| 📍 **追踪进度** | 确认路径后启动，每周自动 Heartbeat | 进度环形图 + 里程碑甘特图 + 打卡热力图 |

## 快速安装

**方式一：直接告诉你的 OpenClaw 智能体**
```
Install this skill: https://github.com/hectormeta/pacer-skill
```

**方式二：一句话安装（需已登录 ClawHub）**
```bash
npx clawhub@latest install pacer
```

**方式三：全局安装 CLI（推荐，避免限速）**
```bash
npm install -g clawhub
clawhub login
clawhub install pacer
```

> 注：`npx` 方式以匿名身份请求，偶发限速报错时请改用方式三。

## 开始使用

安装后，发送以下任意一条消息即可启动：

```
帮我用Pacer规划一下我的下一步
```
```
我现在很迷茫，不知道该做什么
```
```
[直接上传你的CV文件（PDF / Word / TXT）]
```

任意触发词也会自动激活：`职业规划` `career` `我该怎么办` `下一步` `创业方向` `迷茫` `规划` `pacer`

## 示范对话

### 场景一：文字描述 → 多方向对比（创业规划）

```
用户：我想创业，但不知道该做什么。

Pacer：好，先帮你摸清底牌。你现在最擅长的3件事是什么？

用户：写内容、用户运营、数据分析。

[经过5个问题的扫描后...]

Pacer：
╔══════════════════════════════════════╗
║         🏃 你的现状快照              ║
╠══════════════════════════════════════╣
║ 🛠️  核心技能   │ 内容创作 / 用户运营 / 数据分析 ║
║ ⏰  可用时间   │ 每周约 15 小时              ║
║ 💰  经济储备   │ 约 6 个月                  ║
║ 🤝  人脉优势   │ 互联网（电商、SaaS圈）       ║
║ 🔥  核心动力   │ 拒绝重复，需要持续成长        ║
║ 📌  当前状态   │ 在职                        ║
╚══════════════════════════════════════╝

你现在有没有一个大致想去的方向？

用户：想做内容创业，不确定是自媒体还是帮别人做内容服务。

Pacer：基于你的情况，有 3 条路可以走——
方向A — 内容咨询顾问：溢价空间大，第一步：本周联系1个电商朋友问他们内容痛点。
方向B — 垂直领域自媒体：冷启动快，第一步：今天定下细分话题写下来。
方向C — SaaS内容增长服务：人脉直接对接客户，第一步：列出3家认识的SaaS公司。
[输出多方向对比图表]
```

### 场景二：CV上传 → 单目标深度规划（职场转型）

```
用户：[上传 resume.pdf]

Pacer：我从你的CV里看到——5年消费品行业，高级市场专员，
核心能力是品牌传播和用户调研。这些准确吗？

用户：准确，我想转产品经理。

Pacer：
🎯 目标：产品经理（消费品/电商方向）

✅ 你的优势匹配
  • 5年用户调研经验 — 很多技术出身PM最薄弱的地方，你有先天优势
  • 品牌传播背景 — 理解用户心智，产品定位会更敏锐

⚠️ 需要补齐的差距
  • 没有PRD经验 — 需要从写需求文档开始
  • 对技术实现缺乏感知 — 和工程师沟通会有障碍

⏱️ 预计见到结果：4-6个月
🚧 最大风险：初期可能需要接受薪资平移而非增长

[输出技能雷达图 + 三阶段甘特图]
```

完整示范见 [`examples/`](examples/) 目录。

## 核心能力说明

### CV 解析
支持上传 **PDF / Word / TXT** 格式简历，自动提取：
- 工作年限与职级判断
- 核心技能标签
- 行业和人脉背景
- 已知信息不重复追问

### 可视化图表（Chart.js 4.x，WebChat 渲染）
- **技能雷达图** — 当前能力 vs 目标职位所需，差距一目了然
- **多方向对比图** — 4个维度（适配度/收入潜力/上手难度/风险）横向对比
- **双轨时间轴** — 选择新路径 vs 维持现状，看清6-12个月的两种轨迹
- **进度环形图** — 整体完成度实时更新
- **里程碑甘特图** — 各阶段状态（进行中/已完成/逾期）颜色区分
- **打卡热力图** — GitHub 风格的每日行动记录

### 每周 Heartbeat
确认路径后自动启动，每周主动问进展，更新所有图表数据，记录成长轨迹。
