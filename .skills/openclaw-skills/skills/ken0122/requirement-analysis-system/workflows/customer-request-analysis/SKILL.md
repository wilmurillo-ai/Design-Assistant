## 📋 技能描述

**面向场景**: B2B 场景的客户需求快速分析  
**目标用户**: 售前工程师、客户经理、解决方案架构师  
**核心价值**: 快速响应客户需求，转化为可执行的解决方案

---

## 🎯 工作流阶段

### 阶段 1: 客户调研 (30 分钟)
1. 运行 `company-research` — 了解客户
2. 识别决策链和痛点
3. **输出**: 客户调研报告

### 阶段 2: 问题定义 (20 分钟)
1. 运行 `problem-statement` — 明确问题
2. 运行 `proto-persona` — 定义用户
3. **输出**: 问题定义

### 阶段 3: 方案匹配 (30 分钟)
1. 基于现有产品/能力匹配需求
2. 运行 `user-story` — 定义需求
3. **输出**: 解决方案草案

### 阶段 4: 价值量化 (20 分钟)
1. 运行 `feature-investment-advisor` — ROI 评估
2. 量化客户价值
3. **输出**: 价值主张

### 阶段 5: 提案生成 (30 分钟)
1. 整合所有内容
2. 生成客户提案
3. **输出**: 完整提案文档
