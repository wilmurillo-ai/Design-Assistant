# 角色技能包手册

## 使用方法
- 先使用 [行业分类词库](./industry-taxonomy.md) 做行业归类，再选择行业模板。
- 先根据公司画像选择行业模板。
- 每个模板至少覆盖 3 个角色：业务、交付、增长。
- 若 claw.ai / ClawHub 搜索不到完全匹配技能，选择职责最接近的通用 skill。

## 行业模板

### ToB SaaS
- 业务角色：销售开发、售前顾问、客户成功
- 交付角色：产品经理、全栈工程师、测试/实施
- 增长角色：内容营销、增长运营、数据分析

### 电商与零售
- 业务角色：选品采购、店铺运营、客服主管
- 交付角色：商品运营、自动化工程、供应链协同
- 增长角色：投放优化、活动策划、私域运营

### 本地生活与服务业
- 业务角色：BD 拓展、门店运营、商家成功
- 交付角色：流程运营、客服质检、排班协同
- 增长角色：活动拉新、用户运营、数据复盘

### AI/软件外包与交付团队
- 业务角色：项目销售、方案顾问、客户经理
- 交付角色：项目经理、研发工程、交付测试
- 增长角色：品牌内容、案例营销、线索运营

## 一键安装包输出规范
- 包名：`龙虾直聘-{行业/公司关键词}-v1`
- 结构：`基础包` + `增强包`
- 每个 skill 都给出：
`名称 | 角色 | 解决的问题 | 适配理由 | 安装标识`

## 安装确认口令
- 必须等待用户输入以下之一再执行安装：
`确认安装`
`开始安装`

## 安装后回执格式
- `安装成功`：列出已安装 skill 名称
- `安装失败`：列出失败项 + 失败原因
- `建议动作`：补装顺序或替代 skill
