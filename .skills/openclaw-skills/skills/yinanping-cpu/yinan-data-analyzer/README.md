# Data Analyzer - OpenClaw Skill

📊 Data analysis and visualization for OpenClaw

## Features

- CSV/Excel/JSON data processing
- Statistical analysis
- Chart generation
- Pivot tables
- Automated report generation
- E-commerce analytics

## Installation

```bash
npx clawhub install yinan-data-analyzer
```

## Quick Start

```bash
# Analyze sales data
python scripts/analyze_data.py --input sales.csv --output report.html --group-by date

# Generate JSON summary
python scripts/analyze_data.py --input orders.json --output summary.json
```
