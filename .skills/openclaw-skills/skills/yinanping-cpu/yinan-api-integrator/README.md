# API Integrator - OpenClaw Skill

🔌 API integration and automation for OpenClaw

## Features

- REST/GraphQL API integration
- OAuth 2.0, Bearer, Basic auth support
- Webhook handling
- Rate limit management
- Auto-retry on failures
- API client code generation

## Installation

```bash
npx clawhub install yinan-api-integrator
```

## Quick Start

```bash
# Test API endpoint
python scripts/api_client.py --action test --url https://api.example.com/users --method GET

# Generate API client
python scripts/api_client.py --action create-client --name myapi --base-url https://api.example.com
```
