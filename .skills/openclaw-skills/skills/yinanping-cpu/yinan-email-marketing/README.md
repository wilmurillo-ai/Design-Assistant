# Email Marketing - OpenClaw Skill

📧 Email marketing automation for OpenClaw

## Features

- Campaign creation and management
- Email templates (promotional, welcome, newsletter)
- Subscriber list management
- Drip campaign automation
- E-commerce integration (Taobao/Douyin)
- Analytics and reporting

## Installation

```bash
npx clawhub install yinan-email-marketing
```

## Quick Start

```bash
# Create a campaign
python scripts/email_campaign.py --action create --name "Spring Sale" --template promotional --list customers

# Send campaign
python scripts/email_campaign.py --action send --name "Spring Sale"
```

## Templates

- `promotional.txt` - Sales and promotions
- `welcome.txt` - Welcome series for new subscribers
