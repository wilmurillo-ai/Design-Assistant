# 封面图生成指南

## 各平台封面风格

| 平台 | 尺寸比例 | 风格 | 色调 | 文字要求 |
|------|---------|------|------|---------|
| 小红书 | 3:4（竖图）| 精致种草风、质感生活感 | 明亮、暖色调优先 | 可叠加封面文案，字大醒目 |
| 抖音 | 9:16（竖图）| 强视觉冲击、情绪感 | 高对比度、饱和度高 | 前3字必须抓眼球 |
| B站 | 16:9（横图）| 真实记录感、测评风 | 自然、中性 | 可加【】标题注释 |
| 公众号 | 2.35:1 或 1:1 | 媒体感、深度感 | 沉稳、黑白或深色系 | 标题简洁为主 |

## 封面图生成提示词模板

### 📕 小红书封面

**模板：**
```
[主体物品/场景]，精致种草风格，浅色背景，暖色调，
高清摄影，Instagram风格，超清质感，8K分辨率
+ （可选叠加文字：[封面文案]）
```

**示例（火锅场景）：**
```
热气腾腾的麻辣火锅，特写镜头，精致种草风格，
浅色背景，暖色调，高清摄影，Instagram风格，
超清质感，8K分辨率
```

**示例（VS Code 插件场景）：**
```
代码编辑器界面截图，带彩色语法高亮，
右上角有报错红色高亮提示，精致科技感，
浅色背景，高清截图风格，超清质感，8K分辨率
```

## 封面图生成调用示例

启用 `--with-cover` 后，每个平台额外输出如下字段：

```json
{
  "platform": "小红书",
  "cover_image_prompt": "热气腾腾的麻辣火锅，特写镜头，精致种草风格，浅色背景，暖色调，高清摄影，Instagram风格，超清质感，8K分辨率，叠加文字「绝了」"
}
```

将此 prompt 传入 `image_generate` 工具（如 DALL-E、Stable Diffusion 等）即可生成封面图。
