# Troubleshooting

## Common Issues

### 1. Browser Fails to Launch
- **Cause:** Chrome binary missing or inaccessible.
- **Solution:** Verify Chrome is at `/usr/bin/google-chrome` and executable.

---
### 2. `401 Unauthorized` API Errors
- **Cause:** Bearer token is missing or invalid.
- **Solution:** Ensure the `Authorization` header contains the correct token.

---
### 3. Unable to Click Elements
- **Cause:** Incorrect CSS selector.
- **Solution:** Verify the selector exists in the page you navigated to.

---
