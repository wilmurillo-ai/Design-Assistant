## 页面定位要点总结

1. **优先使用文本匹配**：按钮文案是最可靠的定位方式，如 `"赞同"`、`"收藏"`、`"发布"`
2. **多同类按钮时用邻近关系**：找不到唯一匹配时，找与目标内容（标题/文章）位置最近的那个
3. **关注状态字段**：`[disabled]` 表示按钮当前不可用
4. **动态内容**：数字会变化，如 `"赞同 161"` → 需要模糊匹配或取前缀 `"赞同"`
