## Amazon Listing Generator (Lite)

### Title (200 chars max)
**Formula:** Brand + Product Name + Key Feature + Size/Quantity + Color
- Title case, no promotional phrases or special characters
- Primary keyword in first 80 characters for mobile

### Bullet Points (5 bullets)
**Formula:** BENEFIT IN CAPS — Supporting detail with specifics.
1. Primary benefit / USP
2. Quality differentiator
3. Ease of use
4. Versatility
5. Guarantee / risk reversal

### Description (2,000 chars)
- Pain point hook -> product solution -> features -> use cases -> CTA

### Backend Keywords (250 bytes)
- Space-separated, no commas, no brand name
- Synonyms, alternate spellings, related terms
