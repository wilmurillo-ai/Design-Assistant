## Guest Post Opportunity Finder

When the user provides their website URL and niche, search for guest post targets.

### Search Queries to Run
```
"[niche]" + "write for us"
"[niche]" + "guest post guidelines"
"[niche]" + "contribute" + "article"
"[niche]" + "submit a post"
```

### For Each Result, Capture
- Publication name and URL
- Guest post guidelines URL
- Topics they cover
- Submission method (email, form)
- Do-follow vs no-follow link policy

### Quick Qualification
Skip sites that:
- Charge money for guest posts (link farms)
- Accept any topic (not relevant)
- Have thin, low-quality content
- Have no real audience

### Output (up to 5 targets)
```
## Guest Post Targets for [niche]

| # | Publication | DA (est.) | Topics | Submission | Link Policy |
|---|------------|-----------|--------|------------|-------------|
| 1 | [name] | ~XX | [topics] | [method] | [do/no-follow] |
| 2 | [name] | ~XX | [topics] | [method] | [do/no-follow] |
```

## Basic Outreach Draft (1 email)

Draft a personalized guest post pitch for the user's top target.

### Template
```
Subject: Guest post idea: [specific title matching their content]

Hi [Name],

I've been reading [Publication] — your recent piece on [specific article]
was particularly insightful.

I'd love to contribute a guest post on "[Proposed Title]". Here's what I'd cover:

- [Key point 1]
- [Key point 2]
- [Key point 3]

For context, I'm [credential]. Here's a sample of my writing:
- [Article URL]

Happy to adjust the angle based on your editorial needs.

Best,
[Name]
```
