## Proposal Outline Generator (Lite)

### Executive Summary
**Formula:** Problem -> Impact -> Solution -> Why You -> Expected Outcome

I'll generate a 1-page executive summary that:
- Opens with the client's problem in their language
- Quantifies the impact of not solving it
- Presents your approach clearly
- Closes with expected outcome/ROI

### Scope Outline
I'll create a high-level work breakdown:
- Phase names and objectives
- Key deliverables per phase
- Rough effort estimates
- Critical dependencies
