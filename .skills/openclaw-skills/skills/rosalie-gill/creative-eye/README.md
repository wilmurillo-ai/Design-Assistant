# creative-eye

**Give your AI agent creative judgment and design taste.**

Most AI agents will happily ship garbage and call it "done." They confuse "something exists" with "something is good." They generate 50 variations when 1 great one would do. They have no benchmarks, no taste, no quality bar.

This skill fixes that.

## What's Inside

- **STUDY → COMPARE → CREATE → EVALUATE** framework for every creative task
- **Self-Refine Loop** with exact vision model prompts for self-critique
- **5-Point Creative Scorecard** adapted from Runway's evaluation framework
- **10-Point Pre-Publish Checklist** — nothing ships without passing
- **Brand Guardian pattern** for automated quality gates
- **7-Day Creative Training Curriculum** to systematically build judgment
- **Reference library system** for organizing inspiration and anti-patterns
- **Ready-to-use evaluation prompts** for merch, social, product photography, and video

## Install

```
clawhub install creative-eye
```

## Who This Is For

Any AI agent that creates visual content, designs, social posts, merch, product photography, or brand materials. Works for any brand, any aesthetic, any industry.

Built from real lessons learned shipping creative for a CPG brand — every anti-pattern documented here is one we actually made.
