## 一、账号定位方法论

### 核心公式
账号定位 = 价值 × 用户 × 人设 × 类型 × 风格

### 网红 vs IP
- 网红被消遣，IP被追随
- 网红服务粉丝，IP影响粉丝
- 网红无法为商品赋能，IP可以为商品赋能

### 两大误区
1. **过度实用**（赛道工具人）→ 解决方案：用理念串联知识点，植入个人价值观
2. **过度抽象**（价值不落地）→ 解决方案：找到具体利他的内容，分享生活和价值观

### 提供价值的思路
- 有明确变现目的：免费的精神 → 服务某些用户
- 无明确变现目的：基于自身价值出发 → 解决一个人一个阶段的一个问题
- 让账号聚集一批渴望某些事物的人

### 六大竞争力策略
1. **正**（降维碾压）：全局优势、局部优势、细分优势
   - 流程细分：定位/文案/脚本/拍摄/剪辑
   - 行业细分：餐饮→烧烤店/奶茶店
   - 年龄细分：00后职业规划 / 35岁转型
   - 理念细分：性价比/品质感/服务特定审美
   - 地域细分：农村包围城市
2. **反**（否定对抗）：寻找敌人未满足的人群、沉默市场
3. **合**（交叉优势）：兴趣爱好+普世价值、特殊领域+普世价值、特殊场景+普世价值
   - 甲骨文+爱情、围棋+人生、美食+文化
   - 机车+美女、中医+治愈生活、律师+父女生活
   - 烧烤店→友情北漂、律师所→婚姻背叛
   - 价值叠加公式：理性壁垒 + 情感羁绊
4. **跨**（能力迁移）：一切生活哲学化、透过工作看能力
   - 产品经理教人生规划、英语配音演员教成年人英语
   - 老用户新价值：谁说女性成长就一定给中年女性讲
5. **借**（外部资源）：小众身份的专属视野、策划一个使命
   - 参考来源：电影、电视节目、书籍策划
6. **弱**（现身说法）：缺陷转使命、成为榜样、长期主义

## 二、爆款选题方法论

### 核心原则
> 选题要基于一种情绪、一种立场去表达，而不是单纯讲知识本身、讲事件本身。
> 没有新鲜度的选题是没有含金量的。

### 流量公式
流量大 = 需求量大（供小于求）+ 数据好（赛马机制动态竞争）

### 四大普世化切入点

1. **潜在需求 > 正在需求**
   - 错误："正山小种是最好的红茶"
   - 正确："三十五岁之后，大家才知道喝正山小种的好处"

2. **凑热闹 > 求知欲**
   - 错误："想要流量密码，就需要学会切片性"
   - 正确："我大胆预言，未来所有热门视频都会有切片性"

3. **八卦 > 痛点**
   - 错误："年轻人买房怎么选"
   - 正确："我搞房地产十多年见过最可笑可怜的买房行为"

4. **全人类共同情绪 > 事件本身**
   - 错误："健身的方法" → 正确："我们健身是为了让自己骄傲"
   - 错误："穿搭技巧" → 正确："女性的穿搭是自我取悦"
   - 错误："旗袍知识" → 正确："女性独立意识的觉醒从旗袍开始"

### 全人类共同情绪清单
社会声望、性、公民权、爱与浪漫、力量、体育运动、优越感、秩序、被安慰和被理解、独立、复仇、社会交往、个人魅力提升、家庭、好奇心、对不美好的厌恶、荣誉感、被社会排斥的恐惧

### 陌生化五要素
- 稀缺的景观、稀缺的情感、稀缺的美好、稀缺的信息和资讯、稀缺的奇闻逸事、稀缺的事件

### 赋比兴手法
- **赋**：重演绎细节，大量铺陈和堆叠的具体细节之美
- **比**：重在两者的并置，对立之美和结合之美
- **兴**：重在转化的过程，激发一种联想的转化之美

### 演绎技巧
1. **往内折叠**：深度聚焦具体的人与场景
   - 组合情境：产品×人群×场景×痛点
   - 预算越低效果越好（有限空间创造可能）
2. **设置障碍**：挑战、实验、测评
3. **置换元素**：置换时间/环境/道具/人物关系

### 并置技巧
不要讲一个事物，要讲一个事物与另一个事物之间的关系：
- 碰撞结合之美：建筑和文学、做饭和音乐、茶和瑜伽
- 对抗矛盾之美：轰轰烈烈vs细水长流、快乐教育vs严格教育
- 越是围绕二元对立展开，脚本越有整体性，观众越想知道答案

### 构建新概念
争夺命名权：构建模型、小众词汇、比喻造词、叠加创造新词
- 例：耦合式恋爱、生活颗粒度、集体日记

## 三、八大爆款元素与选题指令

### 使用方法
输入行业关键词，用以下八大元素生成选题：

1. **头牌**：世界上最贵的/明星的/最牛的/最贵的好在哪
2. **怀旧**：20年前经典的/古代人如何/小时候难忘的/当年最火的
3. **对立**：穷人vs富人/南方vs北方/男人vs女人/中国vs外国/古代vs现代
4. **最差**：贬值最快/最难吃/差评最多/最难看/最没面子/反人类设计
5. **荷尔蒙**：相亲成功率高/异性多看两眼/最具吸引力/一秒下头
6. **猎奇**：脑回路有病/外行不知道/黑心内幕/内行神操作/匪夷所思
7. **圈人群**：特定星座/内向外向/不同MBTI/身价十亿/第一次体验/弱势群体
8. **成本**：便宜又有面子/十分之一金钱/十分之一时间/如何偷懒/花小钱办大事

### 选题生成指令模板
```
你是一个帮我生成爆款选题的智能体，围绕"{元素名}"这个爆款元素。
行业：{用户行业}
请生成10条选题，要求：
1. 基于情绪和立场表达，不是纯知识
2. 有陌生化/新鲜感
3. 能激发全人类共同情绪
4. 适合短视频3秒留人
```

## 四、钩子公式库

### 核心6大钩子类型

#### 1. 圈定人群
公式 = 喊出人群标签 + 悬念/痛点/恐惧
- 身份标签：基础属性（宝妈/学生/微胖女生）、状态属性（负债者/创业小白）、情绪属性（焦虑的/迷茫的）
- 后半句类型：避坑警告型、颠覆认知型、悬念钩子型
- 完整结构：圈定人群→痛点警告→建立信任→解决方案→行动指令

#### 2. 直接提问
公式 = 疑问词 + 反常识/悬念/利益点
- 疑问词：你敢信/你知道吗/你有没有/如果…会怎样
- 后半句：反常识型、悬念型、紧迫型
- 高阶：提问+数字、提问+背书、提问+互动

#### 3. 自我否定
通过否定自身过往经验制造反差和可信度

#### 4. 数据冲击
用具体数字颠覆认知

#### 5. 场景代入
具体场景描绘+悬念

#### 6. 否定性力量
设定敌人、对抗纠结、先抑后扬

### 钩子质量检查清单
- 3秒内让用户觉得和自己有关
- 触发身份认同感
- 有后续钩子（不能只圈不定）
- 标签精准（不能模糊）
- 有背书（凭什么信你）

## 五、文字语言基本功

### 可视化原则
- 用"Do（做）句式"替代"Be（是）句式"
  - 错误："这是一份用可乐做成的肥牛"
  - 正确："我们给肥牛浇上可乐"
- "There be句式"：在百子湾的角落有一家红色招牌店…
- 特定/专属细节：个体经历→普遍经历
- 故事感（特定感）和故事的尖峰时刻

### 节奏感工具
- 押韵、排比、顶真、长短句交替

### 核心要求
- 移动端观看场景，日常直观的语言才能被听进去
- 好的文案就是一个特写镜头
- 画面感方法：描述、类比（抽象→具象）、力量感（对比数量/时间/经历/身份）

## 六、时间的语法

### 横向时间
1. **时序**：构建不同前提产生不同体验
2. **时长与密度**：详vs略、旁白概述vs现场展开
3. **停顿**：让张力悬置、让观众充分反应
4. **频率**：重复与差异
5. **否定性力量**：调动情绪最快——设定敌人、对抗纠结、先抑后扬
6. **绵延**：共鸣性推进

### 纵向时间
1. **空间**：高低宽窄、拥挤程度、材料质地
2. **面孔与肢体**

### 整体时间
1. **人称距离**：
   - 上帝视角（vlog）：客观公正全知
   - 第一人称：强烈情绪、内心秘密、暴露软肋
   - 第二人称：双刃剑，观众需扮演角色
   - 第三人称：语言和视觉都重要
   - 打破第四面墙
2. **视觉距离**：不拘泥位置角度，综合感受
3. **听点距离**：直接发声→出镜讲述→旁白回顾
4. **基调**：情感的距离
5. **时态**：时间的距离

## 七、用户·产品·营销

### 需求理论
- 用户有各种需求，每次购物产生不同需求重组
- 情境中出现冲突 → 产生需求
- 需求必须依附于情境

### 冲突分析框架
- 生理vs心理（渴望甜食但怕胖）
- 便利vs复杂（高科技产品有用但操作复杂）
- 喜欢vs担心（儿子喜欢但父母担心危险）

### 制造冲突三方向
1. **从消费者入手**：告诉观众老道理不对
2. **从对手入手**：竞品有什么问题
3. **从自我出发**：以前做法有什么不够好

### 效用层级
底线原则 → 够用就好 → 越多越好 → 意外惊喜

### 进攻策略
- **左脑**：理性分析、绝对真相
- **右脑**：情感、直觉、信仰、精神价值
- 精神价值：自由、掌控、地位、责任、不朽、冒险、归属、运势

### 记忆锚点
- 视觉锤、口语化命名、文字钉
- 基于熟悉：熟悉的诗词、文化符号、地名
- 叠化词：滴滴、哇哈哈

## 八、自媒体的立场和姿态

### 强者示弱
- 穿着贵气但讲话亲切
- 场景奢华但行为接地气
- 讲述成功但强调经历相似，复盘错误

### 弱者积极
- 可以可怜但不可以可恨
- 可以反对暴强，不可以反社会反人类
- 可以穷丑，不能肮脏
- 必须有真善美，必须乐观向上
- 负面情绪也敢自嘲

### 立场核心
> 永远想着去关心某些人群、没有给予照顾的人群

### 立场转移（拉近关系的关键）
商家→消费者、达人→厂家、老师→学生、领导→职员、妻子→丈夫、婆婆→媳妇、医生→患者

### 起号绝招
**找到弱势群体！领导他们！**

### 选题绝招
**找到没有被关心的人！对他们说话！**

### 团结原则
- 争取朋友比打击敌人更重要
- 攻击敌人时只会被人看到你的丑恶
- 别人攻击你时，记得那些正在支持你的人

## 九、素材库搭建与搜索方法

### 五大素材文件夹
选题、脚本结构、开篇结尾、呈现方式、音乐

### 搜索渠道
- 选题搜索：抖音、快手、视频号
- 内容搜索：知乎、小红书
- 热点搜索：今日热榜、微博热搜、抖音热榜
- 关键词工具：巨量算数（关联分析→雷达图→关键词组合）
- 百度：行业词+关键词（时间/地点/人群/行业细分/年纪/场景）

### 对标账号方法
- 差异化策划、跨行业借鉴
- 跨领域、跨平台同领域、跨平台跨领域
- 同领域找选题，跨领域找呈现

### 开篇模板
- "我发现了一套{行业}作弊系统"
- "所有{人群}六大方法，{效果}算我输"
- 小红书笔记封面文字可当开篇

### 内容配比
赛道内容为主（90%），人群内容为辅（10%）

### 脚本类型
- 回忆→故事型脚本
- 行动→过程型脚本
- 分析→知识/观点型脚本

### 观点型选题公式
观点型选题 = 写作对象 + 人物关系 + 情绪
