# 商品上新与文案手册

## 1. 标题公式

常用标题结构：
- 品牌 + 型号 + 成色 + 核心卖点
- 品类 + 规格 + 配件情况 + 价格吸引点
- 热搜词 + 商品名 + 状态说明

示例：
- Apple iPhone 13 128G 国行在保 成色不错支持验机
- 戴森吹风机 HD08 原装配件齐全 正常使用无暗病
- 宜家书桌 9成新 同城自提优先

## 2. 正文结构

推荐顺序：
1. 一句话概述
2. 成色 / 使用时间
3. 功能是否正常
4. 瑕疵说明
5. 配件说明
6. 交易方式
7. 价格态度

模板：
- 这个是 [商品名]，平时自用为主。
- 成色 [几成新]，已使用 [时间]，功能一切正常。
- 需要说明的地方： [瑕疵]
- 配件包含： [配件列表]
- 支持 [邮寄/自提]，价格 [可小刀/不刀/诚心可聊]。

## 3. 定价建议

建议同时给出三层价格：
- 挂价：留出轻微议价空间
- 心理成交价：最希望成交的价格
- 底线价：不建议再低于此价

## 4. 提升转化的小技巧

- 瑕疵要提前说，减少售后风险
- 配件说全，降低买家犹豫
- 不夸张，不写“99新”除非真的接近
- 标题优先信息密度，不要太花哨
- 正文尽量让买家少追问

## 5. 常见预设问答

- Q: 还在吗？
  A: 在的，目前还没出。
- Q: 最低多少？
  A: 价格已经比较实在了，诚心要的话可以小聊。
- Q: 有暗病吗？
  A: 目前正常使用没发现问题，具体情况我都按实际写在描述里了。
- Q: 包邮吗？
  A: 具体看地区，合适的话我可以帮你协调一下。
