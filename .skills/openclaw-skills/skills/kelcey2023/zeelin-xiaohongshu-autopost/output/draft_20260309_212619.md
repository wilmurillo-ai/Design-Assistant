# 浙江周末游总踩雷？本地人私藏路线来了

上周末又去了趟西湖，结果人挤人拍不出照片，吃饭排队2小时，回来累得不想说话。明明是放松的，怎么比上班还累？后来跟浙江本地朋友聊才发现，我们外地人去的都是"游客专线"。

问题出在哪？大部分人规划行程都是搜"浙江必去景点"，结果全是西湖、乌镇、普陀山这些热门地。旺季去就是花钱买罪受，淡季去又觉得冷清没氛围。其实浙江好玩的地方太多了，只是没人告诉你怎么避开人潮。

本地朋友给我支了三招，亲测有效：

第一，错峰选小众目的地。比如安吉的云上草原、仙居的公盂村、松阳的古村落群，景色不输热门景点，但游客少一半。周五下午出发，周日下午回，避开周六的人流高峰。

第二，用对工具做攻略。现在有些旅行平台会根据实时数据推荐人少景美的路线，还能看到景区的实时人流量。出发前查一下，能省不少冤枉路。

第三，住民宿别住酒店。浙江很多古村都有改造过的民宿，老板本身就是本地人，会告诉你哪些地方好玩、哪家店好吃。价格比景区酒店便宜，体验还更地道。

上个月按这个思路去了趟松阳，三天两晚人均不到800，拍了一堆出片的照片，吃到了正宗的土菜，回来感觉真的放松了。
