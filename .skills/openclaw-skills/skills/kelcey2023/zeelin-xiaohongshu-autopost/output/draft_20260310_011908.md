# 春游拍照总是路人甲？我找到了原因

上周末约了闺蜜去公园赏花，拍了一下午照片，回来一看全是游客照。别人发的春游照片氛围感拉满，我的就是"到此一游"。明明去的是同一个地方，差距怎么这么大？

问题出在三个地方：时间没选对、角度没找对、道具没带对。大部分人春游都是周末中午去，那会儿光线最硬、人最多，拍出来的照片又平又挤。看着别人的照片羡慕，但从来不研究人家是怎么拍的。

后来我跟一个摄影博主学了几招，再去拍效果完全不一样：

第一，选对时间和天气。早上8-10点或者下午4-6点去，光线柔和，人也少。阴天比大晴天更适合拍照，光线均匀不会有阴影。如果只能周末去，那就早点出门，9点前到景区。

第二，找对角度和构图。别站在花前面正面拍，试试侧身、回头、低头闻花这些动作。用花枝做前景虚化，或者找树荫下的光影。手机开人像模式，背景自然就虚化了。

第三，准备简单的道具。带个草帽、野餐垫、复古相机，或者一本书。这些小道具能让照片更有故事感，也不会显得刻意摆拍。现在有些旅行博主会分享拍照道具清单，出发前看一眼就知道该带什么。

上周按这个思路重新去拍了一次，朋友都问我是不是请了摄影师。其实就是换了个思路，照片质量直接翻倍。
