# 北京旅游总踩雷？本地人私藏路线来了

上个月朋友来北京玩，第一天去故宫排队3小时，第二天爬长城累到怀疑人生，第三天在南锣鼓巷被宰了一顿。回去后跟我吐槽：北京除了人多就是贵，一点都不好玩。我听完直摇头，这不是北京的问题，是你的打开方式不对。

大部分外地游客来北京都是奔着"必打卡景点"去的，结果就是花最多的钱、排最长的队、看最挤的人头。其实北京好玩的地方太多了，只是那些真正有意思的地方，游客攻略上根本不会写。

后来我带朋友重新玩了一遍，完全是另一种体验：

第一，避开热门时段和景点。故宫可以换成国子监、雍和宫，长城可以去箭扣、黄花城水长城。周一到周四去，人少一半，拍照也不用排队。早上8点前或者下午4点后进景区，光线好还清净。

第二，用对工具做攻略。现在有些旅行平台会根据实时数据推荐人少景美的路线，还能看到景区的实时人流量和天气。出发前查一下，能省不少冤枉路和冤枉钱。

第三，吃本地人吃的馆子。别去景区附近的网红店，去胡同里找老字号和苍蝇馆子。护国寺小吃、白魁老号、爆肚冯，人均50就能吃到正宗的北京味儿。

按这个思路重新玩了三天，朋友临走前跟我说：早知道北京这么好玩，应该多待几天。
