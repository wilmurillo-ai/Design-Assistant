# 自动化网络内容运营 (Auto Content Ops)

## 技能概述

本技能将薛辉自媒体方法论体系化为一套可执行的自动化内容生产流水线。
核心流程：**抓热文 → 生热梗 → 结合产品 → 输出发布文案**。

在执行任何步骤前，先阅读 `references/methodology.md` 获取完整方法论知识库。

---

## 一、全流程 Pipeline

### Phase 1：热点抓取（抓热文）

用 web_search 工具执行以下搜索策略：

1. **热榜扫描**：搜索"抖音热搜今日"、"微博热搜榜"、"今日热榜"
2. **行业热点**：搜索"{用户行业} 最新热点 2026"、"{行业} 抖音爆款"
3. **竞品监控**：搜索"{对标账号/竞品} 最新内容"
4. **平台趋势**：搜索"抖音/小红书 近期流行趋势"

**输出格式**：
```
热点清单：
1. [热点标题] | 热度指数 | 情绪标签 | 可结合方向
2. ...
```

**筛选原则**（参考方法论）：
- 必须是目标用户群体关心的话题
- 优先选择能激发"全人类共同情绪"的热点
- 判断热点生命周期：是否还有追的价值

---

### Phase 2：热梗生成（生热梗）

基于抓取的热点，运用八大爆款元素生成内容梗：

**八大爆款元素**：成本、人群、反差、奇葩、最差、怀旧、头牌、荷尔蒙

**生成步骤**：
1. 从热点中提取核心冲突（参见方法论"冲突分析"）
2. 用八大元素逐一匹配，生成候选梗
3. 运用"陌生化"原则优化——稀缺景观、稀缺情感、稀缺信息
4. 用"赋比兴"手法丰富表达

**梗的质量检查**：
- 是否有情绪驱动（不是纯知识/纯事件）
- 是否有立场（关心谁？为谁说话？）
- 是否有陌生感（新鲜度）
- 是否能激发"凑热闹"心理（资讯感 > 知识感）

---

### Phase 3：产品结合（结合产品）

将热梗与用户的产品/服务/账号价值绑定：

**结合策略**（参考"用户·产品·营销"方法论）：

1. **冲突嫁接**：找到产品能解决的冲突点
   - 生理vs心理冲突（如：想吃甜食但怕胖 → 低卡产品）
   - 需求vs现状冲突（如：想旅行但没时间 → 浙江周边游）

2. **价值叠加**：普世情绪 + 产品价值
   - "醉翁之意不在酒"——美食的关键不在食物，旅行的关键不在景点
   - 理性壁垒 + 情感羁绊

3. **进攻左脑+右脑**：
   - 左脑：数据、对比、绝对真相
   - 右脑：情感、直觉、精神价值（自由、归属、地位、冒险…）

4. **用户成本最小化**：降低货币/时间/体力/心力/风险成本的感知

**浙江文旅特别策略**：
- 结合浙江地域文化符号（西湖、乌镇、千岛湖、南宋文化…）
- 地域细分：农村包围城市策略，发掘小众目的地
- 跨界并置：古镇+美食、山水+治愈、非遗+体验
- 季节/节气/节日热点绑定

---

### Phase 4：文案输出（自动发布）

生成完整的发布文案包，包含以下组件：

#### 4.1 钩子开头（3秒留人）

运用36种钩子模板，核心6类：

| 钩子类型 | 公式 | 示例 |
|---------|------|------|
| 圈定人群 | 人群标签 + 悬念/痛点 | "想来浙江旅游的人注意了！" |
| 直接提问 | 疑问词 + 反常识/悬念 | "你知道杭州本地人周末都去哪吗？" |
| 自我否定 | 否定自身经验 + 转折 | "做了5年旅游博主，我居然不知道…" |
| 数据冲击 | 具体数字 + 颠覆认知 | "这个古镇99%的游客都走错了路线" |
| 场景代入 | 具体场景描绘 + 悬念 | "凌晨5点的西湖边，我看到了…" |
| 否定性力量 | 设定敌人/对抗 | "别再被网红打卡点骗了！" |

#### 4.2 正文结构

根据内容类型选择脚本结构：

- **观点型**：钩子 → 观点亮出 → 论据铺陈（赋） → 金句收尾
- **故事型**：钩子 → 情境设置 → 冲突升级 → 反转/解决 → 升华
- **知识型**：钩子 → 痛点共鸣 → 方法拆解 → 案例验证 → 行动指令
- **种草型**：钩子 → 场景需求 → 产品亮相 → 效果展示 → 促单

#### 4.3 文案语言要求

参考"文字语言基本功"：
- **可视化**：用"Do句式"替代"Be句式"，用动作和场面替代抽象描述
- **节奏感**：押韵、排比、顶真、长短句交替
- **特定感**：个体经历引发普遍共鸣，故事的尖峰时刻
- **口语化**：移动端观看场景，日常直观的语言

#### 4.4 时间语法

参考"时间的语法"控制视频节奏：
- **时序**：构建不同前提产生不同体验
- **密度**：详略得当，旁白概述vs现场展开
- **停顿**：让张力悬置，让观众充分反应
- **频率**：重复与差异的节奏
- **否定性力量**：调动情绪最快的方式

#### 4.5 立场与姿态

参考"自媒体的立场和姿态"：
- **强者示弱**：穿着贵气但讲话亲切，讲述成功但强调经历相似
- **弱者积极**：可以可怜但不可恨，必须有真善美
- **立场转移**：站在弱势群体一边，为没有被关心的人说话
- **团结力量**：争取朋友比打击敌人更重要

## 三、账号定位诊断（可选模块）

如果用户还没有明确的账号定位，先执行定位诊断：

1. **确认公式**：账号定位 = 价值 × 用户 × 人设 × 类型 × 风格
2. **避免两大误区**：过度实用（赛道工具人）、过度抽象（价值不落地）
3. **竞争力分析**：正（降维）、反（否定对抗）、合（交叉优势）、跨（能力迁移）、借（外部资源）、弱（现身说法）
4. **找到弱势群体**：起号绝招——找到没有被关心的人，领导他们，对他们说话

## 四、浙江文旅专项模块

针对浙江文旅账号的特殊策略：

### 4.1 内容矩阵
- **文化线**：宋韵文化、丝绸之路、茶文化、非遗手艺
- **自然线**：千岛湖、莫干山、雁荡山、楠溪江、海岛
- **城市线**：杭州新消费、义乌商贸、温州创业、绍兴黄酒
- **生活线**：江南水乡生活、浙菜美食、民宿体验、乡村振兴
- **情感线**：诗画江南、白蛇传说、梁祝故里、南宋风雅

### 4.2 选题公式
- 浙江文旅 + 八大爆款元素：
  - 成本：500块玩转杭州3天2夜
  - 人群：带父母游浙江的正确打开方式
  - 反差：本地人vs游客的浙江玩法
  - 头牌：浙江最贵的民宿到底值不值
  - 怀旧：90年代的西湖长什么样
  - 荷尔蒙：浙江最适合约会的10个地方
  - 猎奇：浙江人自己都不知道的隐藏景点
  - 最差：浙江最不值得去的网红打卡点

### 4.3 并置创新
- 古镇 + 咖啡文化
- 山水 + 心理疗愈
- 非遗 + Z世代体验
- 美食 + 地域性格
- 古诗词 + 实景打卡

## 五、执行注意事项

1. **每次生成前确认**：用户行业、目标人群、产品/服务、账号调性
2. **陌生化检查**：生成的内容是否有新鲜感，是否避免了"老生常谈"
3. **情绪检查**：是否基于情绪和立场表达，而非纯知识讲述
4. **冲突检查**：是否制造了有效冲突，冲突是否与产品相关
5. **成本检查**：用户看完是否能低成本行动
6. **立场检查**：是否站在了弱势一方，是否在关心没被关心的人

> 记住核心原则：选题要基于一种情绪、一种立场去表达，而不是单纯讲知识本身、讲事件本身。

## 六、网页发布SOP（2026-03 同步）

当执行自动发布时，网页流程以脚本为准，固定步骤如下：

1. 进入创作首页后点击「发布笔记」。
2. 在弹层点击「写长文」。
3. 若弹出桌面素材选择框，先点「取消」。
4. 点击「新的创作」进入编辑器。
5. 填写标题与正文后点击「一键排版」，等待排版完成。
6. 在模板区选择「基础简约」（若未出现则选择首个可用模板）。
7. 点击左下角「下一步」。
8. 在确认页点击红色「发布」。

脚本入口：
- `scripts/auto_topic_post_xiaohongshu.sh`：主题到发布一体化。
- `scripts/post_xiaohongshu.sh`：仅执行发布流程（接收标题与正文）。

发布成功判定关键字：`发布成功`、`审核中`、`已发布`、`内容已提交`。
