# 🚀 快速开始指南

## 5 分钟上手盲盒游玩方案生成器

### 第一步：确认安装（1 分钟）

技能已经创建在以下位置：
```bash
~/.qoderwork/skills/travel-blind-box/
```

确认文件存在：
```bash
ls ~/.qoderwork/skills/travel-blind-box/SKILL.md
```

如果看到 `SKILL.md`，说明安装成功！✅

---

### 第二步：测试预算脚本（1 分钟）

运行一个简单的测试：
```bash
python3 ~/.qoderwork/skills/travel-blind-box/scripts/budget_allocator.py 2000 3
```

你应该看到类似输出：
```
============================================================
🎁 盲盒旅行预算分配方案
============================================================
总预算：¥2000.00
游玩天数：3 天 2 晚
日均预算：¥666.67/天
...
```

---

### 第三步：在 QoderWork 中使用（3 分钟）

1. **打开 QoderWork**

2. **输入你的第一个请求**：
   ```
   我想去杭州玩 3 天，预算 2000 元
   ```

3. **等待 QoderWork 响应**：
   - 它会自动识别你的需求
   - 调用 travel-blind-box skill
   - 可能会追问人数等信息
   - 调用 flyai skill 查询实时信息
   - 生成完整的盲盒方案

4. **查看生成的方案**：
   - 点击链接查看完整文档
   - 检查预算分配是否合理
   - 查看每日行程安排

## ⚠️ 常见问题

### Q: 技能没有自动触发怎么办？

**A**: 尝试明确提及技能名称：
```
请用盲盒游玩方案生成器帮我规划杭州 3 天行程
```

### Q: 预算分配不合理怎么办？

**A**: 可以手动调整：
```
住宿预算能提高一点吗？
娱乐预算太少了，能增加一些吗？
```

### Q: 对某个景点不感兴趣怎么办？

**A**: 要求替换：
```
能把西湖换成灵隐寺吗？
这个博物馆不太想去，有替代的吗？
```

### Q: 想完全重新生成怎么办？

**A**: 直接说：
```
整体风格不太喜欢，能完全重新生成吗？
```

## 📱 最佳实践

### ✅ 建议这样做

1. **提供清晰的信息**：
   - 城市：明确目的地
   - 天数：具体数字
   - 预算：总金额

2. **补充额外信息**（可选）：
   - 人数：几个人出行
   - 日期：什么时候去
   - 偏好：有什么特别想体验的

3. **积极互动**：
   - 不满意就说出来
   - 要求调整特定部分
   - 提出具体问题

### ❌ 避免这样做

1. **信息过于模糊**：
   - ❌ "帮我规划旅行"
   - ✅ "我想去杭州玩 3 天，预算 2000 元"

2. **预算不切实际**：
   - ❌ "500 元玩 7 天"
   - ✅ "1000 元玩 2 天" 或 "500 元玩 1 天"

3. **期望完美方案**：
   - ❌ "必须完全符合我的要求"
   - ✅ "先看看方案，不合适再调整"
