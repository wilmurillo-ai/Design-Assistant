# 🎁 盲盒游玩方案生成器 (travel-blind-box) - 放松版

一个为你生成随机盲盒旅行计划的智能助手。**只需告诉它出发日期、目的地、玩几天、有多少预算**，就能自动生成一套轻松随意的完整游玩方案！

## ✨ 核心特性

### 🎲 真正的盲盒体验
- **随机景点推荐**：从当地热门景点中随机选择
- **随机行程安排**：每日活动时间和内容完全随机
- **随机预算分配**：自动分配住宿、餐饮、交通、娱乐比例
- **随机主题风格**：文化探索、美食之旅、冒险体验、休闲度假等

### 🌿 放松式旅行理念 ⭐ NEW
- **每天只安排 1-2 个核心活动**，不赶场
- **早上 10 点后再出门**，睡到自然醒
- **下午保留自由时间**，可休息可继续游玩
- **晚上主要推荐美食**，不强制安排活动

### 🤖 智能功能
- **必须收集出发日期**：用于查询天气、节假日、交通时刻
- **动态追问**：根据需要自动补充询问人数、偏好等信息
- **实时查询**：调用 flyai skill 获取最新景点、酒店、美食信息
- **预算优化**：智能分配预算，确保在预算范围内玩得开心
- **一键预订**：提供第三方平台链接，方便直接预订

### 📋 行程单自动生成 ⭐ NEW
- **预订后自动生程表**：每次完成预订后自动生成详细行程表
- **包含关键信息**：酒店地址、确认号、航班/高铁时间等
- **支持多种格式**：Markdown（可打印）、纯文本（适合微信发送）
- **实时更新**：每次新预订后自动更新并重新发送

**支持格式**：
- **Markdown**：可打印的精美文档
- **纯文本**：适合微信/短信发送

[查看详细示例](example_outputs/itinerary_example.md)

技能使用 `scripts/budget_allocator.py` 脚本进行预算分配：

```bash
# 手动运行预算分配
python scripts/budget_allocator.py 2000 3

# 输出 JSON 格式
python scripts/budget_allocator.py 2000 3 json
```

#### 默认分配比例

| 类别 | 比例范围 | 说明 |
|------|----------|------|
| 住宿 | 35-45% | 按 (天数 -1) 晚计算 |
| 餐饮 | 20-30% | 含三餐和小吃 |
| 交通 | 15-25% | 市内交通为主 |
| 娱乐 | 10-20% | 景点门票和活动 |

```
"住宿预算能提高一点吗？"

# 完全重新生成
"整体风格不太喜欢，能完全重新生成吗？"
```

## 🎯 适用场景

### ✅ 适合的场景
- 一个人或两个人的随心旅行
- 周末短途游（1-3 天）

- 说走就走的旅行

## 🐛 常见问题

### Q1: 预算太低能生成吗？

**A**: 可以生成，但会给出提示。建议：
- 500 元以内：建议缩短天数或增加预算
- 500-1000 元：适合 1-2 天周边游
- 1000-3000 元：适合 3-5 天省内游
- 3000 元以上：可考虑跨省游

### Q2: 生成的方案能保证预订成功吗？

**A**: 方案提供第三方平台链接，实际预订需跳转到飞猪、携程等平台完成。价格和 availability 以平台为准。

### Q3: 可以指定不去某些地方吗？

**A**: 可以在生成前说明，如"不要去太商业化的地方"。生成后也可以要求替换。

### Q4: 支持国外城市吗？

**A**: 当前版本主要支持国内城市。如需支持国外城市，需要 flyai skill 支持国际旅行信息查询。
