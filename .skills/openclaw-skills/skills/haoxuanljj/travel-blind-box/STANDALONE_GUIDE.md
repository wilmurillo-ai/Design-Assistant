## 🔧 功能特点

### ✅ 已实现功能

1. **智能信息收集**
   - 交互式问答收集当前位置、日期、天数、预算
   - 自动验证输入有效性
   - 友好的错误提示

2. **历史记录管理**
   - 自动查询去过的城市
   - 严格过滤，永不重复推荐
   - 旅行结束后自动记录

3. **盲盒推荐系统**
   - 根据当前位置推荐周边城市
   - 考虑交通便利性、预算匹配度
   - 随机选择目的地，保持惊喜感

4. **行程生成**
   - 轻松的行程安排（每天 1-2 个活动）
   - 自动预算分配
   - 包含住宿、餐饮、交通建议

5. **预订流程**
   - 模拟酒店预订
   - 模拟交通预订
   - 自动生成预订总结

## 🎯 使用建议

### 最佳体验方式

1. **找个安静的时间** - 完整流程约 5-10 分钟
2. **准备好真实信息** - 当作真的要旅行一样回答
3. **保持开放心态** - 让盲盒给你惊喜！
4. **多次体验** - 每次都会去不同的地方

### 输入技巧

- **城市名称**：输入省会或大城市（如"上海"、"北京"）
- **日期格式**：支持多种格式
  - `4.27` 或 `4.27 号`
  - `4 月 27 号` 或 `四月二十七`
  - `下周六` 或 `这个周末`
- **天数**：直接输入数字，如 `3`
- **预算**：可以带单位或不带
  - `5000` 或 `5000 元`
  - `5k` 或 `5000 左右`

## 🐛 常见问题

### Q: 为什么推荐的城市有限？

**A**: 独立版使用简化的城市数据库，只包含部分热门城市。QoderWork 版本会调用 flyai skill 查询更多城市。

### Q: 可以自定义推荐的城市吗？

**A**: 当前版本暂不支持。如需自定义，可以编辑 `blind_box_travel.py` 中的 `city_database` 字典。

### Q: 历史记录存储在哪里？

**A**: 存储在 `~/.qoderwork/travel-blind-box/history.json` 文件中。

### Q: 如何清空历史记录？

**A**: 删除历史文件即可：
```bash
rm ~/.qoderwork/travel-blind-box/history.json
```

### Q: 预订是真实的吗？

**A**: 独立版的预订功能是模拟演示，不会真实预订。QoderWork 版本会提供真实的预订链接。
