## 关键点总结

### 真正的盲盒体验

1. **用户只需提供**：
   - 当前位置
   - 出发日期
   - 游玩天数
   - 预算

2. **系统自动完成**：
   - 查询历史记录（避免重复）
   - 筛选候选城市
   - 随机选择目的地
   - 生成完整行程
   - 记录新的历史

3. **历史不重复**：
   - 每次都会查询去过的城市
   - 严格过滤已去过的地方
   - 确保每次都去新地方

4. **新奇体验**：
   - 用户不知道会去哪里
   - 保持期待感和惊喜感
   - 每次都解锁新目的地
