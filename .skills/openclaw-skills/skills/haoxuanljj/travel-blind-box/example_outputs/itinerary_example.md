## 使用说明

### 何时生成行程表

1. **第一版**：用户完成第一个预订（通常是酒店）后立即生成
2. **更新版**：每次用户完成新的预订后都重新生成
3. **最终版**：所有重要项目都预订完成后生成完整版

### 如何发送给用户

1. **Markdown 文件**：保存为 PDF 或 Markdown 文件，提供下载链接
2. **纯文本版**：通过微信/短信发送给用户
3. **打印版**：优化格式，方便用户打印携带

### 行程表特点

✅ **已确认预订一目了然**：表格形式展示所有已预订信息  
✅ **时间安排宽松**：每天只安排 1-2 个核心活动  
✅ **关键信息突出**：确认号、地址、电话等必备信息  
✅ **灵活可调整**：标注自由时间，用户可自行决定  
✅ **实用性强**：包含紧急联系方式和温馨提示
