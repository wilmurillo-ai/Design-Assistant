## ⚠️ 注意事项

### 📋 行前准备
1. **证件**：身份证（入住酒店和部分景点需要）
2. **天气**：查看天气预报，准备雨具（杭州多雨）
3. **衣物**：舒适步行鞋，春秋带外套
4. **药品**：常用药（感冒药、肠胃药、创可贴）

### 🎫 预约提醒
1. **浙江省博物馆**：需提前 1-3 天在官网预约
2. **热门餐厅**：建议提前电话预订座位
3. **酒店**：旅游旺季提前预订价格更优

### 💳 交通建议
1. **地铁**：下载"杭州地铁"APP 或使用支付宝乘车码
2. **公交**：支付宝搜索"杭州通"领取电子卡
3. **打车**：滴滴出行或高德打车
4. **共享单车**：美团单车/哈啰单车适合短途

### 🍜 美食提示
1. **口味**：杭帮菜偏甜，如不习惯可提前说明
2. **排队**：热门餐厅饭点需排队，建议错峰就餐
3. **小吃**：河坊街小吃价格略高，浅尝即可

### 💰 省钱技巧
1. **景点联票**：部分景点有联票优惠
2. **学生证**：如有学生证可享景点半价
3. **团购券**：美团/大众点评有餐饮优惠券
4. **免费景点**：西湖、河坊街、博物馆都免费
