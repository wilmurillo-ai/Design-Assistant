# 输出格式指南

本文档提供回答的格式建议。这些是指南而非强制模板，根据内容和场景灵活调整。

## 整体结构

一个完整的回答通常包含三部分：

### 开场段
- 不超过 100 字
- 直击用户问题的核心，给出初步判断
- 可以将用户的具体问题关联到更大的市场背景
- 使用 `<highlight>核心结论</highlight>` 标记开场的核心判断

### 正文
- 使用多级标题组织层次
- 适当使用表格呈现对比数据
- 每个段落或标题都应传达一个观点，而非单纯罗列数据

### 总结段
- 以 `## 总结` 标题开始
- 50-80 字，给出方向性建议
- 如果涉及投资建议，附带风险提示

## 标题规范

标题应该包含观点或结论，而不仅仅是分类标签。

**一级标题**（正文主要部分）：
```
# 一、**核心论点：论据或影响**
```
示例：
```
# 一、**行业竞争加剧：高端市场份额面临挑战**
# 二、**业绩韧性显现：穿越周期的长期能力值得关注**
```

**二级标题**（段落内的子主题）：
```
1. **小标题**：后接具体展开内容
```

**三级标题**（细节要点）：
```
- **要点**：详细说明
```

## 表格使用

当需要对比数据或呈现多维度信息时，使用表格：

```
| **维度** | **基金A** | **基金B** |
|---------|----------|----------|
| 近一年收益 | 15.2% | 10.8% |
| 最大回撤 | -12.3% | -8.5% |
```

表格前应有一段文字说明背景或概述，不要让表格直接跟在标题后面。

## 数据引用

每条金融数据必须包含四要素：
- **时间戳**：数据的截止日期（如"截至YYYY年MM月DD日"）
- **指标名**：使用工具返回的原始名称，不得改写或缩写
- **数值**：精确数值
- **主体**：对应的基金/市场/行业

示例：
- 正确："截至2025年12月31日，富国新兴产业股票基金近一年收益率为15.2%"
- 错误："该基金近期收益约15%"（缺时间戳、指标名不精确、数值被约化）

## 标识符

### Highlight 标识
用于标记开场核心判断或个性化回应：
```
<highlight>核心结论或个性化回应</highlight>
```
使用场景：
- 开场段中标记核心判断
- 针对用户个人信息（持仓、偏好）的个性化回应

### Citation 标识
在引用参考材料中的专业观点或数据时，在对应文字后标注引用来源：
```
该基金综合评分为85分，属于同类优秀水平<citation>3</citation>。
```
- 在引用位置就近标注，不要集中放在文末
- 可同时引用多个来源：`<citation>3,5</citation>`

## 格式禁忌

- 不使用 emoji
- 不暴露信息来源（"根据工具返回"、"[service 1]"、"网页显示"等）
- 不使用代码块展示文本内容
- 结论必须与正文论据一致，不能前后矛盾
- 同一份回答中不要重复相同的观点
- 内容层面的禁止项（具体仓位建议、买卖点位等）详见 `compliance.md`
