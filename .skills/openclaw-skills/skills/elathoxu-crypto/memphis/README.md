## ⚡ Quick Start (5 minutes)

```bash
# 1. Install
clawhub install memphis

# 2. Initialize
memphis init

# 3. First memory
memphis journal "Hello Memphis!" --tags hello

# 4. Search
memphis ask "What is my first memory?"

# 5. Decide
memphis decide "Framework" "React" -r "Best ecosystem"

# Done! 🎉
```
