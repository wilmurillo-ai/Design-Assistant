# Memphis Quick Start

**Get Memphis running in 5 minutes! 🚀

---

## ⚡ SUPER QUICK (3 minutes)

### 1. Install Memphis (1 min)

```bash
# One command:
curl -fsSL https://raw.githubusercontent.com/elathoxu-crypto/memphis/main/install.sh | bash
```

✅ **Done!** Memphis is installed!

### 2. Test (10 seconds)

```bash
memphis status
```

✅ **Ready!** Memphis CLI is working!

---

## 🎉 You're ready!

**What can you do now:**
- ✅ `memphis decide` - Record decisions
- ✅ `memphis journal` - Add notes
- ✅ `memphis ask` - Ask questions
- ✅ `memphis recall` - Search memory

**Advanced:**
- ✅ `memphis graph build` - Knowledge graph
- ✅ `memphis reflect` - Insights
- ✅ `memphis share-sync` - Multi-agent sync

---

## 💡 Need help?

```bash
# Show all commands
memphis --help

# Ask specific question
memphis ask "How do I decide?"

# Full documentation
cat ~/memphis/docs/README.md
```
