# 一键多平台发布 Skill

**一句话需求 → 多平台同步发布**

## 支持平台

### 视频发布

| 平台 | 状态 | 发布方式 | 备注 |
|------|------|----------|------|
| 小红书 | ✅ 可用 | 浏览器自动化 | 自动生成标签 |
| 抖音 | ✅ 可用 | 浏览器自动化 | 需要标题+话题 |
| 快手 | ✅ 可用 | 浏览器自动化 | 最简单，一键发布 |
| B站 | ✅ 可用 | 浏览器自动化 | 自动使用视频第一帧作为封面 |
| 视频号 | ✅ 可用 | 浏览器自动化 | 微信扫码登录 |

## 使用流程

1. **用户给出需求** - 告诉我想发什么内容
2. **AI 生成文案** - 根据平台特点生成适配文案
3. **用户确认** - 确认各平台文案和配图
4. **一键发布** - 自动登录各平台并发布

## 平台适配规则

### 小红书
- 标题：≤20字，带 emoji
- 正文：≤1000字，分段清晰
- 配图：文字配图 AI 生成，科技/简约风格
- 话题：添加相关话题标签

### 视频号
- 标题：简洁有力，带关键词
- 正文：适合短视频描述
- 封面：16:9 横版或 9:16 竖版
- 标签：3-5 个精准标签

### 抖音
- 标题：抓眼球，带热门话题
- 正文：简短有力，引导互动
- 封面：竖版 9:16
- 话题：#热门话题 + 定位

## 注意事项

- 首次使用需要分别扫码登录各平台
- 浏览器需要配置 SSRF 策略
- 各平台有发布频率限制
- 建议内容差异化，避免被判定为重复内容

## 视频发布流程

### 1. 视频来源
- 即梦 AI 生成：https://jimeng.jianying.com
- 本地视频文件

### 2. 即梦视频生成设置
- **重要**：选择 **10秒** 模式（不是5秒！）
- 积分充足时选择"全能参考"模式（25积分）

### 3. 即梦视频下载
由于 headless 浏览器下载不生效，需要：
1. 打开即梦资产页面 https://jimeng.jianying.com/ai-tool/asset
2. 点击视频预览
3. 用 JavaScript 获取 video 元素的 src URL
4. 用 Invoke-WebRequest 下载到本地

### 4. 浏览器操作技巧
- 用**不同标签页**打开各平台，不要关闭已登录的页面
- 登录状态可以保持
- 首次登录需要扫码，后续保持登录

### 5. 各平台发布步骤

#### 小红书视频发布
1. 打开 https://creator.xiaohongshu.com/publish/publish?from=homepage&target=video
2. 上传视频文件
3. 填写标题和正文
4. 系统自动生成推荐标签
5. 点击发布

#### 抖音视频发布
1. 打开 https://creator.douyin.com/
2. 点击发布视频
3. 上传视频文件
4. 填写标题（简短有力）
5. 添加话题标签
6. 点击发布

#### 快手视频发布
1. 打开 https://cp.kuaishou.com/article/publish/video
2. 上传视频文件
3. 系统自动填充
4. 点击发布

#### B站视频发布
1. 打开 https://member.bilibili.com/platform/upload/video/frame
2. 上传视频文件
3. **注意**：B站现在自动使用视频第一帧作为封面，无需手动上传
4. 填写标题、分区、标签
5. 点击立即投稿

#### 视频号发布
1. 打开 https://channels.weixin.qq.com/platform
2. 用微信扫码登录
3. 点击"发表视频"
4. 上传视频文件
5. 填写视频描述（可添加话题标签）
6. 填写短标题（6-16字）
7. 点击"直接发表"或"声明原创"
