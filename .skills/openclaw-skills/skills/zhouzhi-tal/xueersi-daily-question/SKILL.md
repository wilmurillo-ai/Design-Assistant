# Xueersi Daily Question Generator · 学而思每日一题生成器

> By **Xueersi (学而思)** · AI Education Tools

## Quick Start

When user says "daily question" or "give me a question":
1. If grade/subject not specified, ask:
   > "Sure! ① What grade? ② Which subject — Chinese / Math / English?"
2. If already specified, generate immediately.

## Question Format
```
📚 [Grade] [Subject] — Daily Question

[Question content]

[Options A/B/C/D if applicable]

(Think first — tell me when you want the answer!)
```

## Question Types by Subject

**Chinese (语文)**
- Elementary: vocabulary, poem fill-in, error correction, short comprehension
- Middle school: classical Chinese translation, rhetoric identification, essay opening rewrite

**Math (数学)**
- Elementary: arithmetic, word problems, area, distance-rate-time
- Middle school: equations, basic geometry proof, statistics & probability

**English (英语)**
- Elementary: word selection, sentence transformation, picture description
- Middle school: grammar fill-in, reading T/F, short cloze

## Answer & Explanation

When user answers (or asks for the answer):
```
✅ Answer: [correct answer]

📖 Explanation:
[Solution logic / key knowledge point, 2-5 sentences]

💡 Key concept: [concise summary for review]
```

- Correct: encouragement + optional harder follow-up
- Incorrect: gentle correction + similar practice question

## Progress Tracking
After 5 questions, give a mini summary:
> "You've done 5 today: Chinese ×2 all correct ✅, Math ×3 got 2 right — keep going!"
