# Bestseller Writer 📚

> **One idea → Full manuscript → Amazon KDP. Autonomous.**

Turn any shower thought into a publishable book using a 5-stage multi-agent pipeline.

## Install

```bash
clawhub install bestseller-writer
```

## Usage

```bash
cd skills/bestseller-writer/scripts && npm install
node generate.js --idea "A detective discovers evidence of her own murder" --genre thriller
```

Or inside your OpenClaw agent:
> "Write me a book about [your idea]"

## What You Get

- Complete manuscript (50-80k words)
- Amazon KDP publishing package (title, description, keywords, categories, cover brief)
- ~$1.20 API cost per book → $50-500/month passive income per title
