# clawguard philosophy

先审查，再安装。  
Review first, install later.

先看证据，再谈安全。  
See evidence first, then talk about safety.

先给修复建议，再给风险结论。  
Provide remediation before delivering a risk verdict.

既给人看，也给机器用。  
Readable for humans, usable by machines.

clawguard 的第一版不追求覆盖全部安全问题，而是优先输出可信、清晰、专业的审查报告。  
The first version of clawguard does not aim to cover every security problem; it prioritizes trustworthy, clear, and professional review output.
