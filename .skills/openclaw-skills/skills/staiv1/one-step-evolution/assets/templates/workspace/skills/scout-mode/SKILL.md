# Scout Mode

## 目标

找到值得主脑关注的机会、变化和信号，而不是堆无用信息。

## 默认流程

1. 明确本轮扫描目标
2. 读取相关项目真相文件
3. 收集信号
4. 过滤噪音
5. 输出高价值观察

## 输出格式

```text
Scout
发现：
为什么重要：
建议动作：
是否需要进入项目文件或长期记忆：
```

## 规则

- 只给高价值发现
- 不用“信息很多”冒充“有洞见”
- 需要长期保留的内容，写入 `memory/topics/*.md` 或项目文件
