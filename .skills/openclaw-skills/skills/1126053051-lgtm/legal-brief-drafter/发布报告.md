## 🎯 使用示例

### 示例 1：起草行政诉讼驳斥意见

```bash
# 准备案件素材
cat > 案件素材.txt << 'EOF'
案件基本信息：
- 案号：（2026）京 0491 行初 XXXX 号
- 原告：董国华
- 被告：北京市海淀区市场监督管理局
- 第三人：北京微梦创科网络技术有限公司

核心争议：
1. 行政机关未实质审查格式条款效力
2. 事实认定混淆"形式告知"与"有效提示"
3. 法律适用片面，遗漏《消保法》第 26 条
EOF

# 起草文书
python3 script.py --action draft \
  --template 行政诉讼驳斥意见 \
  --input 案件素材.txt \
  --output 驳斥意见.md
```

### 示例 2：法律推演分析

```bash
# 演绎推理分析
python3 script.py --action analyze \
  --inference-method 演绎 \
  --input 案件事实.txt \
  --output 演绎分析.json

# 查看结果
cat 演绎分析.json | python3 -m json.tool
```

### 示例 3：查看技能信息

```bash
python3 script.py --action info
```
