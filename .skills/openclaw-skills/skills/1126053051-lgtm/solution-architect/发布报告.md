## 🚀 使用方式

### 1. 查看所有解决方案
```bash
python3 script.py list
```

### 2. 查看方案详情
```bash
python3 script.py show growth-marketing
python3 script.py show content-upgrade
python3 script.py show platform-growth
```

### 3. 根据问题找方案
```bash
python3 script.py solve --problem "内容团队效率低，想引入 AI"
```

### 4. 根据目标找路径
```bash
python3 script.py plan --goal "6 个月内将开发者数量从 1000 提升到 10 万"
```

### 5. 优化现有方案
```bash
python3 script.py enhance --my-plan 我的方案.md --reference growth-marketing
```

## 💡 核心设计理念

### 1. 方案驱动，而非格式驱动
- **不是**生成格式化的报告文档
- **而是**提供可执行的解决方案框架

### 2. 提炼方法论，而非复制内容
- **不是**复制原始报告的具体内容
- **而是**提炼核心框架和战术

### 3. 可迁移，而非生搬硬套
- **不是**固定场景的专用方案
- **而是**可跨场景迁移的方法论

### 4. 量化目标，而非空泛建议
- **不是**"提升品牌影响力"等空话
- **而是**"社交媒体声量提升 50%"等量化指标

## 📊 方案核心内容

### 增长营销方案（特斯拉）

**核心战术**:
- ✅ 视频内容矩阵（5 平台定位 + 内容主题规划）
- ✅ AI 内容生产线（创意/视觉/视频/音频/数据分析）
- ✅ 社群裂变（车主俱乐部+KOC 培养 + 引荐计划）
- ✅ 跨界项目（环岛直播/FSD 体验官/未来生活节）

**KPI 体系**:
- 社交媒体声量提升 50%
- 内容驱动线索转化率提升 30%
- 社群月活达 5 万
- 车主推荐参与率 15%

---

### 内容升级方案（36 氪）

**核心战术**:
- ✅ 三级产品矩阵（流量层/核心层/产品层）
- ✅ 财报分析 SOP（五步流程）
- ✅ AI 生产管线（AI 研究员 + 人类分析师）
- ✅ 专家网络（上市公司 IR/行业专家/券商分析师）
- ✅ 高价值社群运营

**岗位重塑**:
- 从"主编"到"首席产品官"
- 核心价值三角：价值发现者/预期管理者/效率提升者

---

### 平台增长方案（小米 MiMo）

**核心战术**:
- ✅ 产品力引擎（定价/API/解决方案/工具链）
- ✅ 运营力引擎（开发者全生命周期 5 阶段）
- ✅ 生态力引擎（内部融合 + 外部拓展）
- ✅ 增长飞轮设计（吸引→激发→网络效应→商业闭环）

**18 个月路线图**:
- 奠基期（0-6 个月）：夯实基础，引爆声量
- 扩张期（7-12 个月）：丰富供给，拓展生态
- 深化期（13-18 个月）：价值深化，商业闭环
