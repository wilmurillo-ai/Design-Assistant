## 一、十天干性格特征

### 甲木（阳木 — 大树）

甲木如大树，主刚直向上，有进取心与领导力。命主多具正直坦率的性格，不轻易弯腰妥协，有强烈的自尊心和成就欲。处事果断，但有时过于固执，不善变通。

### 乙木（阴木 — 花草藤蔓）

乙木如花草藤蔓，柔韧灵活，善于借势。命主性格温和，适应力强，擅长人际交往和艺术审美。有韧劲，能在压力下保持弹性，但有时过于依赖外力，缺乏独立冲劲。

### 丙火（阳火 — 太阳）

丙火如太阳，光明磊落，热情外向。命主慷慨大方，乐于助人，具有强烈的感召力和表现欲。待人真诚，常是人群中的焦点，但有时过于张扬，难以保守秘密。

### 丁火（阴火 — 烛光灯火）

丁火如烛光，温暖细腻，内敛而持久。命主聪慧敏感，观察力强，善于体察他人情绪。情感丰富，有艺术气质，但有时多愁善感，情绪波动较大。

### 戊土（阳土 — 山岳大地）

戊土如山岳，厚重稳健，信用卓著。命主性格踏实，有担当，值得信赖，为人处世四平八稳。具有强大的包容力，但有时过于保守，行动迟缓，难以快速适应变化。

### 己土（阴土 — 田园沃土）

己土如田园，温润细腻，滋养万物。命主心思细腻，善于经营，有理财意识和积累能力。待人周到，但有时过于保守谨慎，容易患得患失。

### 庚金（阳金 — 钢铁刀剑）

庚金如钢铁，刚硬果决，执行力强。命主性格直爽，做事干净利落，有原则，不拖泥带水。具有强烈的正义感，但有时过于强硬，缺乏圆融变通之道。

### 辛金（阴金 — 珠宝首饰）

辛金如珠宝，精致敏锐，重视细节。命主有审美眼光，追求完美，注重外表和仪态。思维敏锐，感受力强，但有时过于挑剔，对外界批评较为敏感。

### 壬水（阳水 — 大江大海）

壬水如大江，奔放流动，思维宽广。命主眼界开阔，善于统筹谋划，有大局观和战略思维。行动力强，变通灵活，但有时漫无边际，难以聚焦于一点深耕。

### 癸水（阴水 — 雨露甘泉）

癸水如雨露，滋润精微，智慧内敛。命主心思缜密，直觉灵敏，善于分析和洞察。记忆力好，学习能力强，但有时过于谨慎，缺乏冲劲，容易犹豫不决。

## 三、用神取法基本规则

用神是调节命局平衡的核心天干，是分析喜忌的基准。取用神主要有三种方法：

### 1. 扶抑法（最常用）

根据日主强弱，选择补偏救弊的五行为用神。

- **日主偏强（身旺）**：喜克泄（官杀、食伤、财星），忌印比（印星、比劫）
- **日主偏弱（身弱）**：喜生扶（印星、比劫），忌克泄（官杀、食伤、财星）
- **日主中和**：以调候或格局喜忌为主

### 2. 调候法

当命局寒燥、温润失衡时，以调节气候为用神取法依据。

- 生于冬月（亥子丑）寒冷之命：喜丙丁火温暖，辅以戊土厚实
- 生于夏月（巳午未）炎热之命：喜壬癸水润泽，辅以庚金引水
- 调候用神优先级高于扶抑，在极端季节出生时必须先考虑

### 3. 通关法

当命局两种五行势均力敌、相互对峙时，以能通关（居中调停）的五行为用神。

- 木火对峙 → 水通关
- 金木对峙 → 水通关（木受金克，水生木泄金）
- 火土对峙 → 木通关（木生火克土）

## 五、格局判断基本规则

格局是命局的基本结构，主要取月令透干来判断。

### 普通格局（月令格）

以**月支藏干的本气**透出天干为格：

| 月支 | 本气 | 对应格局 |
|------|------|---------|
| 寅月 | 甲 | 甲木格（以甲与日主关系定格名） |
| 卯月 | 乙 | 乙木格 |
| 辰月 | 戊 | 正/偏财格、正/偏官格（取透干） |
| 以此类推 | | |

常见格局名（以日主为庚为例）：
- 月干透壬 → 食神格
- 月干透甲 → 正财格
- 月干透乙 → 偏财格
- 月干透丙 → 七杀格
- 月干透丁 → 正官格
- 月干透戊 → 偏印格
- 月干透己 → 正印格

### 特殊格局

- **专旺格**：命局五行极度集中，一种五行力量压倒性强 → 从旺（从强、从旺、一行得气）
- **从格**：日主极弱，命局完全被另一五行主导 → 从财、从官杀、从食伤
- **化气格**：天干五合成立且化神得月令支持 → 化木、化火、化土、化金、化水

> 特殊格局判断须谨慎，需满足严格条件，不可随意套用。

## 六、大运流年断法基本规则

### 大运分析原则

1. **大运干支与命局的生克关系**：大运天干与命局天干的五行生克，大运地支与命局地支的合冲刑
2. **大运喜忌判断**：大运五行是否属于命局用神（喜），还是忌神（忌）
3. **十年大运分前五后五**：大运天干影响前五年，地支影响后五年（简化判断，不绝对）

### 流年分析原则

1. **流年天干与日主的关系**：生克合冲
2. **流年地支与命局地支的关系**：六合、三合、六冲、三刑
3. **流年用神/忌神**：流年五行是否加强命局用神或忌神
4. **大运与流年叠加**：同一方向叠加时，力量更大（双喜/双忌）

### 断事基本方向

| 十神星 | 对应生活领域 |
|-------|------------|
| 官杀星 | 工作、职位、压力、规范 |
| 财星 | 钱财、父亲（男）、妻子（男） |
| 印星 | 学业、证书、母亲、贵人 |
| 食伤星 | 才华发挥、子女、创业、口舌 |
| 比劫星 | 朋友、合作、竞争、财运分散 |

## 七、表达风格指南

### 专业命理顾问腔（推荐）

**特征**：
- 以"命主"或"此造"开头，带入专业情境
- 展示推导逻辑，让命主感受到分析的依据
- 用"宜/不宜""喜/忌"等传统命理术语
- 正面与挑战并重，既不一味颂扬，也不渲染恐惧
- 给出具体可操作的建议

**示例**（正确风格）**：
> 命主日主为庚金，生于壬午月，月令午火克金，日主偏弱。天干年柱庚金帮身，地支有午火三见，火势偏旺。整体而言，此造身弱，喜用水木，忌火土。宜从事与水木相关的行业，如科技、医疗、教育、文化等，方位喜东方、北方。当前大运逢水木运，有利于命主发挥用神力量，事业宜积极进取。

### 江湖骗子腔（严格禁止）

**特征**：绝对化断言、恐吓性语言、推销服务

**示例**（禁止）**：
> ❌ 你这个命克夫，婚姻不顺，必然两嫁。
> ❌ 今年大凶，有血光之灾，需要做法事化解。
> ❌ 你命中注定难有财，除非改名字改运。

## 九、特殊情况处理参考

### 时辰不明（命主不知出生时间）

仅排年月日三柱（六字），说明时辰对分析的影响：
> 由于时辰未知，时柱暂缺。时柱影响时神（第四柱十神）、晚年运势及子女宫的判断。以下分析基于年月日三柱，结论仅供参考，若日后得知出生时间，可补全分析。

### 生日在节气当天

节气发生有精确时刻，若命主出生于节气当天，应说明需确认具体时间：
> 您的出生日期（X月X日）恰在节气（XX）附近，这会影响月柱的归属。可以确认一下大约几点出生吗？这样可以判断是否已过节气时刻。

### 命局极端（过旺或过弱）

说明格局类型并解释原因，避免用"好坏"评判：
> 此造五行分布较为集中，木火力量占主导，日主偏弱程度较大。这类命局属于"从格"的考虑范围——若条件满足，则顺势而为、随强而从，反而是一种不同的发展路径。

## 十、解读报告结构建议

### 优先顺序

1. **四柱干支**（基础数据，必须展示）
2. **日主五行与月令**（强弱判断的核心依据）
3. **五行分布**（整体格局一览）
4. **用神/忌神**（喜忌方向，实用性最强）
5. **格局**（命局类型，决定发展路径）
6. **大运**（人生节奏，当前所处阶段）
7. **流年**（当前年份的具体互动）
8. **综合解读**（整合以上，给出方向建议）

### 内容详略原则

- 用户首次完整排盘 → 全部展示，重点放在用神和当前大运流年
- 用户复查单项（如"今年运势"）→ 直接跳到流年分析，不重复排盘
- 用户问具体问题（如"适合做什么行业"）→ 只回答该问题，引用用神五行
