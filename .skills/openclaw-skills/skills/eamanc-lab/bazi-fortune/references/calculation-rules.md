```
木 → 火 → 土 → 金 → 水 → 木（循环）
```

- 木生火，火生土，土生金，金生水，水生木

### 相克（隔一）

```
木 克 土，土 克 水，水 克 火，火 克 金，金 克 木
```

## 三、年柱推算

### 公式

```
天干序号 = (年份 - 3) % 10
地支序号 = (年份 - 3) % 12
```

**示例**：1990年
- 天干：(1990 - 3) % 10 = 1987 % 10 = 7 → 序号7 = 庚
- 地支：(1990 - 3) % 12 = 1987 % 12 = 7 → 序号7 = 午
- 年柱：庚午

### 立春为界规则

**重要**：年柱以当年**立春**为界，不以元旦或春节为界。

- 立春前出生（通常为2月3日～5日之前）→ 使用上一年的年柱
- 立春当日及之后出生 → 使用当年的年柱

**各年立春参考日期**（以公历为准，实际需精确到节气时刻）：

| 年份范围 | 立春通常日期 |
|---------|------------|
| 现代（1900-2100年） | 每年2月3日、4日或5日 |

> 精确操作：若出生日期为2月3～5日，需查阅当年立春具体时刻（可告知用户并允许其确认）。

## 五、日柱推算

### 基准日推算法

**基准日**：1900年1月1日 为甲戌日（天干序号0=甲，地支序号10=戌）

**推算步骤**：

1. 计算目标日期距1900年1月1日的天数差 N
2. 天干序号 = (0 + N) % 10
3. 地支序号 = (10 + N) % 12

**闰年规则**：
- 能被4整除但不能被100整除的年份为闰年
- 能被400整除的年份为闰年
- 平年365天，闰年366天，2月分别为28天和29天

**各月天数（平年）**：1月31、2月28、3月31、4月30、5月31、6月30、7月31、8月31、9月30、10月31、11月30、12月31

> 对于具体日期的推算，建议使用精确的天数差计算，不可凭印象跳步。

### 子时说明

子时跨日（23:00-00:59），有两种流派处理方式：

- **晚子时（23:00-23:59）**：属当日日柱（多数命理家采用此法）
- **早子时（00:00-00:59）**：属次日日柱（部分流派）

遇到子时出生时，需告知用户两种算法并确认，或两种都算出供参考。

## 十、大运排法

### 顺逆规则

| 性别 | 年干阴阳 | 大运方向 |
|------|---------|---------|
| 男 | 阳年干（甲丙戊庚壬） | 顺推（向后的节气） |
| 男 | 阴年干（乙丁己辛癸） | 逆推（向前的节气） |
| 女 | 阳年干 | 逆推 |
| 女 | 阴年干 | 顺推 |

**判断依据**：

- 阳男阴女 → 顺行
- 阴男阳女 → 逆行

### 起运年龄计算

1. 确定顺/逆方向后，从出生日起计算**到下一个（或上一个）节气的天数**
2. 每 3 天对应 1 岁，余 1 天约等于 4 个月，余 2 天约等于 8 个月
3. 起运年龄 = 天数 ÷ 3（四舍五入到月份精度）

**示例**：出生后到下一个节气共 12 天，起运年龄 = 12 ÷ 3 = 4 岁起运。

### 大运干支排列

以起运时的**月柱**为参考起点，按顺/逆方向，每步走一个干支：

- **顺行**：从月柱下一个干支开始，依次甲子→乙丑→丙寅……
- **逆行**：从月柱上一个干支开始，依次逆推

每步大运管 10 年，共列出 8 步（80 年跨度）。

## 十三、五行旺衰与月令

月令（月支）决定当月当令五行，是判断日主强弱的核心依据。

| 月支 | 当令五行 |
|------|---------|
| 寅、卯 | 木旺 |
| 巳、午 | 火旺 |
| 申、酉 | 金旺 |
| 亥、子 | 水旺 |
| 辰、戌、丑、未 | 土旺（各含其余五行余气） |

**日主强弱判断步骤**：

1. 日主五行是否得月令（当旺之月）
2. 日主五行在地支是否有根（地支中有同五行本气或中气）
3. 天干中有多少帮身（同五行）与生身（生助五行）

综合以上三点判断：身旺 / 中和 / 身弱。
