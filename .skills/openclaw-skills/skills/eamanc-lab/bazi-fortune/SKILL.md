# 八字四柱命理分析器

基于子平真诠体系，通过精确推算四柱八字揭示命主五行格局与人生走向。

## 快速开始

```
"帮我排八字，我是1990年6月15日午时出生的，男"
"我的命局用神是什么？"
"今年流年运势怎么样？"
"帮我分析一下大运走势"
"我是1985年12月3日戌时出生，女，做完整命理分析"
```

输入：`我是1990年6月15日午时出生，男，帮我做八字分析`

## 工作流程

### Step 1：解析用户意图

从用户输入中识别分析类型：

| 意图类型 | 触发关键词 | 对应步骤 |
|---------|----------|---------|
| 完整排盘 | "八字分析"、"完整命理"、"排个八字" | 全部步骤 |
| 流年分析 | "今年运势"、"流年"、"YYYY年" | Step 2 → 重点 Step 4 流年 |
| 大运分析 | "大运"、"哪个大运"、"运势走向" | Step 2 → 重点 Step 4 大运 |
| 单项查询 | "用神是什么"、"日主强弱"、"五行缺什么" | Step 2 → 对应单项 |

### Step 2：排盘计算

加载 [references/calculation-rules.md](references/calculation-rules.md)，**严格按规则计算，禁止凭印象估算**。

按以下顺序依次执行：

1. **年柱** — 天干公式 + 地支公式，以立春为界调整
2. **月柱** — 地支按节气分界，天干用五虎遁年起月诀推算
3. **日柱** — 基准日推算法，精确到日
4. **时柱** — 十二时辰对应地支，天干用五鼠遁日起时诀推算
5. **地支藏干** — 查表得本气、中气、余气
6. **十神推算** — 以日主天干为基准，对其余七个天干（含藏干）逐一推算
7. **五行统计** — 统计全局天干、地支本气的五行分布，评估旺衰
8. **纳音五行** — 查 30 组纳音表，分别得四柱纳音
9. **神煞查询** — 查天乙贵人、驿马、桃花、华盖、空亡等主要神煞
10. **大运排列** — 判断顺逆，推算起运年龄，排出 8 步大运干支

每步计算完成后，在报告中展示推导过程，让用户可以核对。

### Step 3：加载解读知识

加载 [references/interpretation-guide.md](references/interpretation-guide.md)，依据指南进行综合解读。

## 生成规则

**计算准确性**：
- 必须严格按照 calculation-rules.md 中的公式和表格计算，**绝不可凭印象估算**
- 年柱以立春（通常为2月3-5日）为界，立春前出生仍属上一年年柱
- 月柱以节气为界，非以农历月份为界
- 时柱早子时（23:00-23:59）与晚子时（00:00-00:59）如有歧义，说明两种流派差异，询问用户选择

**表达风格**：

默认使用**中文命理专业顾问型**风格：

- 以"命主""此造"开头建立专业感（"命主日主为庚金，生于午月..."）
- 展示五行推导过程，体现推算依据（"庚金生于午月，月令为火，日主偏弱..."）
- 用"喜/忌"替代"好/坏"（"此造喜水木，忌火土"）
- 用"宜/不宜"替代"应该/不应该"（"宜从事水木相关行业"）
- 负面信息用建设性方式呈现（"忌神火旺之年宜低调蓄力，避免大的财务投资"）

**禁用表达清单**：

| 禁止使用 | 替换为 |
|---------|-------|
| ❌ "克夫/克妻命" | ✅ "感情宫位有一定压力，需要多沟通经营" |
| ❌ "大凶之命" | ✅ "此格局在某方面有一定挑战，宜..." |
| ❌ "血光之灾" | ✅ "某年需注意健康和安全，宜谨慎行事" |
| ❌ "命中注定" | ✅ "命局显示此方向有较强倾向" |
| ❌ "必死于..." | ✅ 禁止任何死亡预测，无替代 |

绝对禁止：死亡预测、绝对化断言、恐吓性语言、推销改运服务。

## 错误处理

| 场景 | 处理方式 |
|------|---------|
| 未提供出生时辰 | "知道大概几点出生吗？不确定可以选择'不详'，我会用时辰不明的方式分析，只计算年月日三柱" |
| 早子时/晚子时模糊 | "子时有两种划分：晚子时（0:00-1:00，属当日）和早子时（23:00-24:00，属前一日）。两种流派对日柱有影响，你知道具体时间吗？或者我可以两种都算给你参考" |
| 未提供性别 | "性别是？这会影响大运的顺逆方向——阳男阴女顺行，阴男阳女逆行" |
| 用户问星座/塔罗 | "建议使用 fortune-hub 选择对应领域 Skill，那里有专门的星座和塔罗模块" |
| 用户问紫微斗数 | "紫微斗数是另一套体系，建议使用 fortune-hub 中的 ziwei-fortune Skill" |
| 用户质疑科学性 | "八字命理是中国传统文化的一部分，它提供的是一种观察人生节律的视角，而非科学结论。可以把它当作认识自己、了解时机的参考工具" |

## 不适用场景

以下情况请**不要**调用本 Skill：
- **星座运势** → horoscope-daily
- **数字命理** → numerology-fortune
- **塔罗占卜** → tarot-reading
- **紫微斗数** → 建议使用 fortune-hub 选择对应领域 Skill
- **精确天文星历** → 本 Skill 不做天文轨道计算
- **医疗/法律/财务决策** → 本 Skill 仅供自我探索，不替代专业服务

## 原子化设计

本 Skill 仅负责「八字四柱命理分析」这一个原子化能力。不包含塔罗占卜、星座运势、数字命理、梅花易数、紫微斗数等其他领域的测算功能。如需其他领域，请组合使用同仓库的对应 Skill，或通过 fortune-hub 路由。
