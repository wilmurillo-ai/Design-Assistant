## 一、用神选择表

根据问题类型确定用神（核心断卦对象）：

| 问题类型 | 用神 | 说明 |
|---------|------|------|
| 求财（投资/生意/薪资） | 妻财 | 财爻代表钱财、利润 |
| 求官职（升职/考试/仕途） | 官鬼 | 官爻代表权威、职位 |
| 求功名（考试/资格证） | 官鬼 | 同求官 |
| 婚姻（男问） | 妻财 | 财爻代表妻子、伴侣（男） |
| 婚姻（女问） | 官鬼 | 官爻代表丈夫、伴侣（女） |
| 求子（生育） | 子孙 | 子孙爻代表子女 |
| 健康/疾病 | 官鬼（为忌神）| 官爻克身，官旺则病重 |
| 出行/旅行 | 世爻（自身状态）| 配合妻财（交通）、子孙（平安）|
| 寻人/失物 | 妻财（失物）/ 子孙（寻人）| |
| 官司/诉讼 | 官鬼（对手）/ 世爻（自身）| 世强官弱则胜 |
| 合作/合同 | 父母（文书）/ 世应关系 | 应爻代表对方 |
| 问天气 | 父母（阴雨）/ 子孙（晴好）| |

**辅助用神**：用神确定后，还需考虑：
- **原神**：生助用神的爻（对用神有利）
- **忌神**：克制用神的爻（对用神不利）
- **仇神**：克制原神或生助忌神的爻（间接不利）

## 二、四大角色及生克链

```
仇神 ──生──▶ 忌神 ──克──▶ 用神
              ▲              │
              │              │
              克             生
              │              │
              └──── 原神 ◀───┘
```

| 角色 | 定义 | 对用神影响 |
|------|------|----------|
| 用神 | 问题的核心爻 | 本身 |
| 原神 | 生助用神者 | 有利，原神旺则用神得扶 |
| 忌神 | 克制用神者 | 不利，忌神旺则用神受损 |
| 仇神 | 克制原神、生助忌神者 | 间接不利，削弱原神力量 |

## 三、断卦基本框架（五步流程）

### Step 1：定用神

按用神选择表确定本次占问的用神（哪一个六亲）。

### Step 2：看用神旺衰

- **月建**：月建地支五行对用神地支五行的关系（旺/相/休/囚/死）
- **日辰**：日辰地支五行对用神地支五行的关系
- **综合判断**：月建为主，日辰为辅；两者皆生扶→用神有力；两者皆克泄→用神无力

### Step 3：看原神忌神

- 原神在卦中是否出现？旺衰如何？
- 忌神在卦中是否出现？旺衰如何？
- 原神旺而忌神衰 → 对用神整体有利
- 忌神旺而原神衰 → 对用神整体不利

### Step 4：看动爻影响

- 动爻直接冲克或生合用神 → 影响最大
- 变爻是否持续生克用神
- 多动爻时，以离用神最近的爻为主

### Step 5：综合判断

整合以上四步，给出明确结论：
- **用神旺、原神扶、无忌神克** → 吉，事成
- **用神衰、忌神克、无原神救** → 凶，受阻
- **用神中等，动爻有影响** → 需注意动爻代表的时间节点
- **世爻旺而应爻衰** → 自身条件好，对方配合度差

## 四、六亲含义详解

### 父母爻

代表：父母、长辈、上司、文书、合同、房屋、衣食、保护。
- 父母爻旺则文书顺利、合同易成、有贵人支持
- 求财时，父母克子孙，父母旺不利于财运发展
- 父母爻化空或受克，长辈无力扶持，文书有阻

### 兄弟爻

代表：兄弟姐妹、朋友、同事、竞争者、破财。
- 兄弟爻旺则竞争激烈，财运受阻（兄弟克妻财）
- 问合作时，兄弟旺则对方可能出现变数或竞争者介入
- 持世时，代表自己有损失或与人有争竞

### 子孙爻

代表：子女、下属、学生、福神、医药、平安。
- 子孙爻旺则诸事平安，有化解之力（子孙制官鬼）
- 健康占卜中，子孙旺则病情可控、有转机
- 求官时，子孙克官鬼，子孙旺不利于升职考试

### 妻财爻

代表：妻子（男性问婚）、财富、物质、利益、食物。
- 妻财爻旺则财运好、利润可期
- 妻财爻受克（被兄弟夺）则财不稳定，易有破财
- 父母爻弱而妻财旺，有财但文书或合同需注意

### 官鬼爻

代表：丈夫（女性问婚）、官职、权威、压力、疾病、灾祸。
- 求职时，官鬼旺而世爻受扶，有机会得职
- 健康占中，官鬼旺克世爻，病情较重或有压力
- 官鬼化空或受克，障碍减少，诉讼可胜

## 五、六神含义详解

### 青龙

象意：吉祥、喜庆、文书、财运、贵人相助。
- 临财爻：财运有喜，可能有意外之财
- 临官爻：贵人提携，官职晋升有望
- 凡临青龙，事情多向好发展，宜积极推进

### 朱雀

象意：口舌、文书、消息、信息、争讼、火。
- 临父母：消息、文件相关，可能有信件往来或合同变动
- 临官鬼：口舌是非增多，警惕争执或舆论风波
- 问消息类事务，朱雀为重要参照

### 勾陈

象意：土地、田产、纠纷、迟滞、拖延、农事。
- 临妻财：财事易拖延，房地产相关
- 临官鬼：有纠纷缠身，事情缠绕难解
- 凡临勾陈，进展偏慢，宜耐心等待

### 螣蛇

象意：虚惊、梦境、缠绕、阴私、小人、变化无常。
- 临官鬼：有虚惊或暗中阻力，需防小人
- 临父母：文书有变、合同易有隐患
- 临世爻：自身处境多变，心神不安

### 白虎

象意：凶险、伤病、孝服、口舌激烈、金属、道路。
- 临官鬼：伤病之象，疾病加重或有意外伤害
- 临妻财：财中有险，投资需谨慎
- 临父母：长辈有损，文书有争执

### 玄武

象意：暗昧、私情、盗窃、欺骗、水事、隐秘。
- 临妻财：财路不明，可能有暗中损财
- 临官鬼：有暗中对手或隐秘的压力来源
- 临兄弟：防友朋欺骗或暗中竞争

## 六、动爻变爻规则

### 动爻的作用

动爻是当前卦中最活跃的力量，直接影响断卦结果：

| 动爻与用神关系 | 影响 |
|-------------|------|
| 动爻生用神 | 有力相助，事情推进 |
| 动爻克用神 | 直接冲击，障碍明显 |
| 动爻冲用神 | 动荡不安，变数增多 |
| 动爻合用神 | 有连带，情况复杂，需看合化五行 |

### 变爻的作用

动爻变出的新爻（变爻）代表后续发展方向：

| 变爻与用神关系 | 影响 |
|-------------|------|
| 变爻生用神 | 后续有转机，事情向好发展 |
| 变爻克用神 | 后续压力持续，需警惕 |
| 变爻同用神 | 用神得比助，力量增强 |

### 无动爻（六静卦）

卦中无动爻时，以世爻与应爻的关系为主判断，结合月建日辰看当下格局。

## 七、进退神判断

进退神判断地支的进退状态，用于判断用神的发展趋势：

| 类型 | 判断规则 | 含义 |
|------|---------|------|
| 进神 | 本卦地支 → 变卦地支，在五行旺相方向前进（如寅→卯、巳→午）| 用神力量增进，形势向好 |
| 退神 | 本卦地支 → 变卦地支，在五行衰退方向后退（如卯→寅、午→巳）| 用神力量减退，形势转弱 |

## 八、六冲六合对卦象的影响

### 六冲卦（全卦内外卦上下对冲）

- **六冲卦特征**：内卦三爻与外卦三爻的地支两两相冲（子午、丑未、寅申、卯酉、辰戌、巳亥）
- **含义**：事情难以稳定，变动频繁，合作易散，关系松动
- **例外**：若冲中有合，或动爻化合，则可部分化解

### 六合卦（全卦内外卦上下相合）

- **六合卦特征**：内卦三爻与外卦三爻的地支两两相合
- **含义**：事情稳定，合作融洽，有凝聚力，但也可能难以改变现状

### 动爻冲用神

- 动爻地支冲用神地支：有"动冲"之力，比静冲更有力，但若用神旺则可以承受

### 月建冲克用神

- 月破：月建冲用神（称用神"月破"），用神力量大减，事情多有波折

## 九、常见问题断卦要点

### 求财

- 妻财爻旺相、不受兄弟夺 → 财运顺畅
- 妻财爻持世 → 本身财运好，利于求财
- 兄弟爻旺而动，克妻财 → 财有破损，谨防合伙纠纷
- 子孙爻旺而无忌 → 财源不断，利于长线投资

### 求职升迁

- 官鬼爻旺、世爻有力 → 有机会获职
- 官鬼化空或受子孙克 → 职位难得，或竞争对手受阻
- 应爻克世爻 → 对方（单位/上司）主动性强，自身被动
- 父母旺临青龙动生官 → 文书、资质助力升迁

### 婚姻感情

- 男：妻财旺、应爻生世爻 → 感情顺利，对方有意
- 女：官鬼旺、世应相生 → 有良缘，关系稳定
- 六冲卦（世应相冲）→ 双方分歧大，难以长久
- 妻财或官鬼化空 → 感情虚而不实，难落实

### 健康疾病

- 官鬼旺克世爻 → 病情较重，需积极就医
- 子孙旺制官鬼 → 病情可控，有转机
- 白虎临官鬼动克世 → 有外伤或急性病变
- 官鬼化空或入墓 → 病情可能减轻或需关注

### 出行

- 世爻旺、子孙爻旺 → 出行平安，旅途顺利
- 妻财旺 → 出行有利于财务
- 官鬼旺动克世 → 出行有阻，或途中有波折，建议重新安排
- 六冲卦 → 出行途中多变，行程可能有变动

## 十、表达风格指南

### 基本原则

- **易学专业顾问型**：先展示推导过程，再给出判断
- **可能性而非确定性**：使用"趋势指向""卦象显示""用神旺相，利于..."等措辞
- **赋权而非恐吓**：让用户感到有主动权，而非被命运主宰
- **负面结果必须附带建议**：凶象后给出可操作的应对方向

### 推导链示例

```
用神妻财临月建亥水，水生木，而妻财属木，得月建相生，用神旺相有力。
日辰辰土克亥水，日辰微泄月建，整体用神仍属旺相。
忌神兄弟申金临三爻，月建亥水生忌神，忌神有力，需注意竞争或合伙变数。
动爻子孙午火克兄弟申金，有化解之象，财运仍有支撑。
综合：财运整体向好，需防合作中出现竞争，独立操作或减少合伙比例更佳。
```

### 吉凶表达对照

| 卦象情况 | 推荐表达 |
|---------|---------|
| 用神旺，事顺 | "用神旺相有力，当前时机有利，可积极推进" |
| 用神衰，有阻 | "用神入休囚，当前阻力较大，建议暂缓或等待月令转变" |
| 动爻克用神 | "动爻克用，近期有变数，需谨慎应对" |
| 六冲卦 | "六冲卦象，稳定性不足，合作或关系易有波动" |
| 应期来临 | "用神五行对应{时间}，届时或有转机" |

## 十一、禁用表达清单

| 禁止使用 | 替换为 |
|---------|-------|
| ❌ "此卦大凶，必然失败" | ✅ "用神受克，当前时机阻力较大，建议暂缓" |
| ❌ "你命中注定贫穷" | ✅ "当前财运格局有阻，可寻找忌神弱化之时机" |
| ❌ "婚姻绝对不成" | ✅ "世应相冲，双方分歧较大，感情需更多磨合" |
| ❌ "这次出行有生命危险" | ✅ "官鬼克世，出行有波折风险，建议重新安排时间" |
| ❌ "必须如何否则..." | ✅ "如果能...，局势可能有所改善" |
| ❌ "下次再来算一卦" | ✅ （直接给出建议，不引导依赖）|

**绝对禁止**：
- 涉及死亡的确定性预测
- 绝对化断言（一定/必然/绝对）
- 恐吓性语言
- 引导依赖占卜的表述

## 十二、应期参考

根据用神五行和旺衰，给出可能应验的时间方向（仅供参考）：

| 用神五行 | 旺月参考 | 旺日参考 |

| 火（巳午） | 巳月、午月（4-6月）| 巳日、午日 |
| 土（辰戌丑未）| 四季月（辰戌丑未月）| 辰戌丑未日 |

| 水（亥子） | 亥月、子月（10-12月）| 亥日、子日 |

**说明**：用神衰时需等待其旺月旺日，届时事情更易成就。应期仅为参考方向，不做精确预测。
