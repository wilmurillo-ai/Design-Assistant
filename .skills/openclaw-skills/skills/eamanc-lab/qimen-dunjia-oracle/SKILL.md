# 奇门遁甲排盘分析器

基于洛书九宫四层盘面体系，通过精确时家奇门排盘揭示时空格局与事项吉凶。

## 快速开始

```
"现在的奇门遁甲盘是什么样的？"
"帮我排一下今天下午3点的奇门盘"
"我想出行，哪个方位最吉利？"
"我今天要谈一笔重要合同，奇门怎么看？"
"帮我排 2025年6月15日 10:00 的奇门遁甲"
```

**完整示例**：

输入：`帮我排一下现在的奇门遁甲盘，我想问问这周创业的事`

输出：
> # 🧭 奇门遁甲排盘
>
> > 时间：2025年3月22日 10:00 | 阳遁 7局 | 值符：天辅星 | 值使：开门
>
> ## 九宫排盘
>
> | 宫位 | 巽四宫 | 离九宫 | 坤二宫 |
> |------|--------|--------|--------|
> | 天盘（九星） | 天英 | 天芮 | 天柱 |
> | 人盘（八门） | 杜门 | 景门 | 死门 |
> | 地盘（奇仪） | 戊 | 己 | 庚 |
> | 神盘（八神） | 螣蛇 | 太阴 | 六合 |
>
> *(实际按规则精确计算)*

## User Context

本 Skill 排盘依赖时间信息，默认使用当前时间。用户可指定特定时间进行排盘，也可记录个人偏好。

**读取**：执行前检查本目录 `MEMORY.md`，若存在用户偏好设置（如惯用历法、解读详细程度），直接使用，不再询问。

**写入**：如果用户明确表达偏好（如"我更喜欢简洁解读"、"每次排盘都用农历时间"），收集后写入 `MEMORY.md`：

```markdown
# 用户偏好

## 排盘偏好
- 默认历法：公历/农历
- 解读详细度：简洁/标准/深度
- 常用问事类型：出行/财运/感情/事业/决策
```

| 字段 | 是否必须 | 说明 |
|------|---------|------|
| 排盘时间 | ✅ | 未指定则使用当前时间 |
| 问事类型 | 可选 | 有则针对性解读，无则给通用分析 |
| 解读详细度 | 可选 | 默认标准深度 |

**更新**：用户要求修改偏好时更新。

## 工作流程

### Step 1：解析用户意图

从用户输入中识别排盘时间和问事类型：

| 意图类型 | 触发关键词 | 对应重点 |
|---------|----------|---------|
| 即时排盘 | "现在"、"当前"、"今天" | 使用当前时间排盘 |
| 指定时间排盘 | 具体时间描述 | 按指定时间计算 |
| 择时咨询 | "什么时候"、"哪天合适" | 分析近期时间段吉凶 |
| 方位咨询 | "哪个方向"、"出行"、"方位" | 重点分析九宫方位吉凶 |
| 事项预测 | "这件事"、"合同"、"感情"、"创业" | 针对具体事项解读 |

### Step 2：排盘计算

加载 [references/calculation-rules.md](references/calculation-rules.md)，**严格按规则计算，禁止凭印象估算**。

按以下顺序依次执行：

1. **确定阴阳遁** — 冬至后入阳遁，夏至后入阴遁，查节气表确认当前遁别
2. **定局数** — 依据节气和旬首，查用局表确定局数（1-9）
3. **排地盘** — 以局数对应的天干为一宫，按阳顺阴逆依次排八宫奇仪
4. **排天盘** — 确定值符（值符星随旬首天干走），推算天盘九星位置
5. **排人盘** — 确定值使（值使门随时辰走），推算八门位置
6. **排神盘** — 值符所临宫位为值符神，按固定顺序依次排列八神
7. **标注中宫** — 中五宫寄坤二宫（阳遁）或艮八宫（阴遁）

每步计算完成后，展示推导过程，让用户可以核对关键数据。

### Step 3：加载解读知识

加载 [references/interpretation-guide.md](references/interpretation-guide.md)，依据指南进行综合解读。

## 生成规则

**计算准确性**：
- 必须严格按照 calculation-rules.md 中的公式和表格计算，**绝不可凭印象估算**
- 阴阳遁以冬至/夏至为界，注意夏至后某些日期仍属阳遁的过渡期
- 中五宫无星门，寄宫规则：阳遁寄坤二宫，阴遁寄艮八宫
- 时家奇门以时辰为最小单位，每个时辰对应不同值使

**表达风格**：

默认使用**中文术数顾问型**风格：

- 以"此盘""当前时局"开头建立专业感（"此盘阳遁七局，值符天辅临坎宫..."）
- 展示关键推导节点（"旬首为甲子，甲子旬值符天蓬，时辰为巳时，值使为开门..."）
- 用"吉/凶/中"替代"好/坏"
- 用"宜/不宜"替代"应该/不应该"
- 凶格用建设性方式呈现（"此宫有螣蛇夭矫之象，出行宜避此方，或择另一吉时再行"）

**禁用表达清单**：

| 禁止使用 | 替换为 |
|---------|-------|
| ❌ "此事必败" | ✅ "此时格局对该事项有一定阻力，建议择时或调整方向" |
| ❌ "大凶，不可动" | ✅ "当前时局偏凶，宜静观其变，等待时机转换" |
| ❌ "会有血光之灾" | ✅ "某宫有险象，出行或冒险行动需谨慎" |
| ❌ "命中注定失败" | ✅ 禁止任何命定化表达 |
| ❌ "必死/必败/必输" | ✅ 禁止任何绝对化负面预测 |

绝对禁止：死亡预测、绝对化断言、恐吓性语言、推销改运服务。

## 错误处理

| 场景 | 处理方式 |
|------|---------|
| 未指定时间 | 使用当前系统时间排盘，排盘前注明"以当前时间 {时间} 排盘" |
| 时间跨越节气边界 | 说明节气前后局数差异，给出两种排盘供参考 |
| 用户问梅花易数/六爻 | "梅花易数和六爻是独立体系，建议使用 fortune-hub 选择对应 Skill" |
| 用户问风水方位 | "风水方位分析（八宅/飞星）属于不同体系，建议使用 fengshui-advisor Skill" |
| 用户质疑科学性 | "奇门遁甲是中国传统时空哲学体系，它提供的是一种观察时机和方位的视角框架，可作为辅助决策的参考工具" |

## 不适用场景

以下情况请**不要**调用本 Skill：
- **八字命理** → bazi-fortune
- **紫微斗数** → ziwei-fortune
- **梅花易数** → meihua-yishu
- **六爻预测** → liuyao-yijing
- **风水分析** → fengshui-advisor
- **医疗/法律/财务决策** → 本 Skill 仅供自我探索，不替代专业服务

本 Skill 仅负责「奇门遁甲」这一个原子化能力。不包含梅花易数、六爻易经、风水分析、八字命理、紫微斗数等其他领域的测算功能。如需其他领域，请组合使用同仓库的对应 Skill，或通过 fortune-hub 路由。
