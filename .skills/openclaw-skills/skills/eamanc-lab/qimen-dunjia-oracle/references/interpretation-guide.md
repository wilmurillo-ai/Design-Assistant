## 一、九星含义详解

**天蓬星**（坎一宫，水）
凶星。主贼盗、险境、暗中阴谋。临于某宫时，该方位宜防小人与暗害，出行、谈判需格外警觉。遇休门则减凶，遇死门则大凶。

**天芮星**（坤二宫，土）
凶星。主疾病、衰败、死气阴沉。临宫代表该方向气场低迷，事项推进阻力大。求医方面反有参考价值，宜静不宜动。

**天冲星**（震三宫，木）
吉星。主奋进、突破、雷动之气。临宫代表该方位充满行动力和冲劲，适合启动新项目、出击进攻。遇伤门则有莽撞之嫌，需注意节奏。

**天辅星**（巽四宫，木）
吉星。文昌贵气，主学术、文书、辅佐。临宫利读书考试、文案谈判、寻求贵人支持。遇开门或生门则锦上添花。

**天禽星**（中五宫，土）
中性星。居中调和，无固定方位，寄宫于坤二（阳遁）或艮八（阴遁）。代表稳重、调停、居中斡旋之力，本身不强势，与寄宫星门共同判断。

**天心星**（乾六宫，金）
吉星。主谋略、官贵、医药、武事。临宫代表权威与决断，适合面见领导、办理官方事务、寻求医疗帮助。遇开门则贵气倍增。

**天柱星**（兑七宫，金）
凶星。主毁折、口舌是非、破败之象。临宫需防言语冲突、合约破裂、设备损坏。避免在此方向开口签约或动工。

**天任星**（艮八宫，土）
吉星。主稳重积累、仁厚宽容、财富渐积。临宫代表踏实守成，适合长线投资、积累资源、建立稳固关系。遇生门则财运最旺。

**天英星**（离九宫，火）
凶星（有虚名吉象）。主虚名、表面光鲜、好大喜功。临宫有显扬之象但根基不稳，名气来得快去得也快。若问文艺、宣传则可用，问实质财运则不吉。

## 二、八门含义详解

**休门**（坎一宫，水）
吉门。主休息、平和、顺遂。利出行、谋私事、回家休整。行动不需大张旗鼓，低调推进更顺。遇天蓬则减弱，遇天任则稳定吉祥。

**死门**（坤二宫，土）
大凶门。主死气、终结、衰败。任何开拓、求进之事均不宜走此门方位。唯葬事、收藏、猎捕有用，其余皆不吉。

**伤门**（震三宫，木）
凶门（有竞争之用）。主伤害、争斗、激烈竞争。不利和谈、合作，但若是竞争性事项（如体育、谈判施压），可借此门之气。

**杜门**（巽四宫，木）
中性门。主隐藏、闭塞、埋伏。利于隐蔽行动、保密谋划、躲避追踪，不利公开行事和谈判。适合侦查、隐居、暗中部署。

**景门**（离九宫，火）
吉门（利文事）。主文书、扬名、出行远方。求名、宣传、科考、外出谈判均宜，但实质财运不强。遇天英则名声更旺，遇天芮则名不副实。

**开门**（乾六宫，金）
吉门。主开拓通达，为八门中综合最吉之门。出行、求财、谒贵、开业均宜。遇天心则权威加倍，几乎无事不宜。

**惊门**（兑七宫，金）
凶门。主惊恐、口舌、噩耗、不安。遇此门宜静不宜动，需防突发变故和坏消息。若问讼事，此门方向有反复惊扰之象。

**生门**（艮八宫，土）
大吉门。主生机勃勃、财源广进。求财、生意开张、寻找新机会，生门是首选方位。遇天任则财运最为稳固丰厚。

## 三、八神含义详解

**值符**
吉神之首。所在宫位代表权威与贵人所在，主君主、上司、官员。行事得值符方向支持则事半功倍，可借权威力量推进。

**螣蛇**
凶神。主怪异变化、虚惊、缠绕纠缠。遇螣蛇之宫，事情容易出现意外变故、关系缠绕难解，需防阴谋与幻象迷惑。

**太阴**
阴神。主隐秘、阴私、女性力量、地下活动。利于保密和暗中行事，不利公开竞争。情感事项中代表女方或隐藏的感情。

**六合**
吉神。主合作、婚姻、和谐、顺遂。凡求合作、联姻、谈判达成一致，六合方位均为有利方向，人际关系顺畅。

**白虎**
凶神。主凶猛、争斗、官讼、血光。交通事故、医疗手术、法律纠纷等带血气之事，遇白虎需格外谨慎，此方位宜绕道而行。

**玄武**
凶神。主盗贼、欺骗、阴谋、失窃。财物保管和信任关系需注意，此方位容易遇到不诚信之人或遭受暗中损失。

**九地**
中性神（偏吉）。主伏藏、守土、稳固、防守。适合守成、防御、保守策略，不适合进攻和冒险。居家安守则吉。

**九天**
吉神。主升腾、公开、宣扬、远行。适合公开行事、远途出行、展示自我、追求高位。凡欲扬名立万，取九天方位可助威。

## 四、常见奇门格局

### 吉格

**青龙返首**
开门 + 天心星临乾六宫（本宫）。金气纯粹，谋事一帆风顺，官场仕途大利，求贵人扶持最佳时机。

**飞鸟跌穴**
值符星与值使门同临一宫，且该宫为值符星的本宫。天地人神高度合一，此时行事顺应天机，所求之事极易成功。

**三奇得使**
乙丙丁三奇同时临于某宫的天地盘，或三奇顺序排列，代表奇异贵气聚集，文事武功均大利。

**天遁**
丙奇（太阳之精）临于生门或开门，阳气充盈，有如天助，适合公开行事、展示实力、取得公认。

**地遁**
乙奇（月华之精）临于开门或休门，柔中带吉，适合低调谋划、守成积累、缓缓推进。

**人遁**
丁奇（星光之精）临于生门，人气旺盛，利人际关系，合作谈判、招揽人才均宜。

### 凶格

**白虎猖狂**
白虎神临于开门或生门，凶神占据吉位，原本吉利的方向被凶神把持。出行求财反遭白虎阻击，宜避此方位或择时。

**螣蛇夭矫**
螣蛇神临于景门，主怪异变化，传闻失真，文书不实，签约有虚假成分，需仔细核实对方诚意。

**大格**（入墓格）
天干落入其五行所克的宫位（如庚辛金落入离九宫火），气场被压制，代表事项进展受阻，此宫位方向不宜进取。

**返吟**
天盘与地盘完全相同，盘面静止如镜，代表局势僵固，来回徘徊，难以前进也难以后退，宜静待时机。

**伏吟**
天盘各宫与其本宫固定星门完全重叠，局面如死水无波，万事停滞，不利行动，适合休整蓄力。

**刑格**
天干地支形成相刑组合，代表被人制约或自我束缚，此方位或此时段行事阻力来自规则与关系的摩擦。

## 五、方位吉凶判断基本规则

**综合判断优先级**：

1. **八神**（影响最大）：值符、六合、九天为吉；白虎、螣蛇、玄武为凶
2. **八门**（次重要）：生门、开门、休门为大吉；死门、惊门、伤门为凶
3. **九星**（辅助判断）：天心、天辅、天任、天冲为吉；天蓬、天芮、天柱、天英为凶
4. **奇仪**（基础加权）：乙丙丁三奇为吉；庚辛癸为凶；戊己壬居中

**方位综合评级**：

- 三层（神+门+星）皆吉 → 大吉方位，可全力进取
- 两吉一凶 → 吉方，谨慎而行
- 一吉两凶 → 中平偏凶，不宜主动
- 三层皆凶 → 大凶方位，务必回避

**特殊规则**：
- 值符方位无论其他如何，均有一定贵气加持
- 死门方位即使有吉星，也不建议用于主动出击
- 生门方位遇凶神，财路有阻碍但不绝，可借化解降低风险

## 六、表达风格指南

**基本原则**：专业顾问型，重推导轻预言，给方向而非给定论。

**开头范式**：
- "此盘{阳/阴}遁{X}局，值符{星}临{宫}，值使{门}临{宫}..."
- "当前时局{星}奇仪组合，{方位}方气场..."
- "以{时间}起盘，{旬首}旬，{局数}局..."

**格局描述范式**：
- 吉格："{格局名}之象已现，{方位}方有贵气聚集，宜..."
- 凶格："{宫位}方有{格局名}之兆，此时此方向{事项}有一定阻力，建议..."

**事项判断范式**：
- "求{事项}，看{门星神}组合，{方位}方{综合评级}，{行动建议}"
- "若问{事项}，当前格局显示{判断}，宜{建议}，不宜{忌讳}"

## 七、禁用表达清单

| 禁止使用 | 原因 | 替换方案 |
|---------|------|---------|
| "此事必败/必成" | 绝对化预测 | "此时格局对该事项{利/不利}，建议..." |
| "大凶，绝对不能动" | 恐吓性语言 | "此方位目前气场偏凶，宜暂缓，择吉时再行" |
| "会有血光之灾" | 恐吓性语言 | "白虎临此宫，出行或体力劳动需注意安全" |
| "命中注定" | 命定化表达 | "当前时局显示..." |
| "死门就是会死" | 字面恐吓 | "死门代表气场低迷、事项阻滞，不利主动求进" |
| "你选的这个时间很糟糕" | 否定用户 | "此时辰有一定局限，可以考虑调整到{更吉时辰}" |

绝对禁止：死亡预测、绝对化断言、恐吓性语言、推销改运服务、诱导用户依赖。
