## 二、安命宫公式

### 命宫定位法（寅宫起子逆数时辰）

**规则**：以农历月份在寅宫（序号2）起正月，顺数至出生月，再从该月宫位逆数时辰，最终落点即为命宫地支。

**公式**：

```
命宫地支序号 = (2 + 月份 - 1 - 时辰序号 + 12) % 12
```

其中：
- 月份：农历月份，正月=1，二月=2，……，十二月=12
- 时辰序号：子=0，丑=1，寅=2，卯=3，辰=4，巳=5，午=6，未=7，申=8，酉=9，戌=10，亥=11
- 地支序号：子=0，丑=1，寅=2，卯=3，辰=4，巳=5，午=6，未=7，申=8，酉=9，戌=10，亥=11

**示例**：农历五月初八，亥时出生

```
命宫序号 = (2 + 5 - 1 - 11 + 12) % 12 = 7 % 12 = 7 → 未
命宫 = 未宫
```

## 三、安身宫公式

### 身宫定位法

**规则**：以农历月份在寅宫起正月，顺数至出生月，再从该月宫位**顺数**时辰。

**公式**：

```
身宫地支序号 = (2 + 月份 - 1 + 时辰序号) % 12
```

**示例**：农历五月，亥时

```
身宫序号 = (2 + 5 - 1 + 11) % 12 = 17 % 12 = 5 → 巳
身宫 = 巳宫
```

> 身宫表示"后天努力的重心"，与命宫合参解读人生主轴。

## 五、五行局定法

命宫所在地支配合天干（寅宫起甲，按年干推寅月月干），取纳音五行定局数。

### 命宫天干确定（五虎遁年起月法）

根据**农历年干**确定寅月（正月）天干，再顺推至命宫所在月份：

| 年干 | 寅月（正月）天干 |
|------|---------------|
| 甲、己 | 丙 |
| 乙、庚 | 戊 |
| 丙、辛 | 庚 |
| 丁、壬 | 壬 |
| 戊、癸 | 甲 |

从寅月起，月份每增1，天干顺推1位（甲→乙→丙→…→癸→甲循环）。

**示例**：庚午年（年干庚），五月命宫

- 寅月天干：戊（乙庚之年戊为头）
- 卯月：己，辰月：庚，巳月：辛，**午月：壬**
- 命宫天干：壬

配合命宫地支未（序号7），命宫干支 = 壬未

## 六、安紫微星算法

### 紫微落宫口诀法

**步骤**：

1. 取五行局数 N（水二局=2，木三局=3，金四局=4，土五局=5，火六局=6）
2. 取农历出生日 D
3. 用 D ÷ N，得商数 Q 和余数 R
4. 按余数 R 决定起点和走法：

| 余数 R | 起点 | 走法 |
|--------|------|------|
| 0 | 商数Q对应宫（直接落点） | 直落，不走 |
| 奇数（1、3、5…） | 寅宫（序号2）起，向前走Q步 | 顺走Q步 |
| 偶数（2、4…，非0） | 寅宫（序号2）起，向后走Q步 | 逆走Q步 |

> "向前"指地支顺序（子→丑→寅方向），"向后"指地支逆序。

**精简口诀**（民间记法）：

- 余数为零：商数即位数，从寅数起顺（局数整除）
- 余数为奇：安在寅前Q位（顺数Q格）
- 余数为偶：安在寅后Q位（逆数Q格）

> 不同版本口诀略有差异，本 Skill 采用上述逻辑版本，计算时展示每步推导供用户核对。

**示例**：木三局（N=3），农历初八（D=8）

```
8 ÷ 3 = 商2 余2（偶数）
从寅宫（序号2）逆走2步：寅→丑→子
紫微星落子宫（序号0）
```

## 十一、大限排法

### 起大限年龄（按五行局数）

| 五行局 | 局数 | 起大限年龄 |
|--------|------|----------|
| 水二局 | 2 | 2岁起 |
| 木三局 | 3 | 3岁起 |
| 金四局 | 4 | 4岁起 |
| 土五局 | 5 | 5岁起 |
| 火六局 | 6 | 6岁起 |

每步大限管 10 年，共排 12 步（覆盖120年）。

### 顺逆规则

| 性别 | 阴阳年（农历年干） | 大限方向 |
|------|----------------|---------|
| 男 | 阳年干（甲丙戊庚壬） | 顺行（地支顺序递增） |
| 男 | 阴年干（乙丁己辛癸） | 逆行（地支逆序递减） |
| 女 | 阳年干 | 逆行 |
| 女 | 阴年干 | 顺行 |

**顺行**：从命宫地支起，每步顺序走一个地支（子→丑→寅→…）

**逆行**：从命宫地支起，每步逆序走一个地支（子→亥→戌→…）

### 大限示例
