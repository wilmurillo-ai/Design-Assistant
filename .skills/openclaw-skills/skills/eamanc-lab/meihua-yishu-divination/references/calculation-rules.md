### 相生（有利）

```
金生水 → 水生木 → 木生火 → 火生土 → 土生金
```

```
金克木 → 木克土 → 土克水 → 水克火 → 火克金
```

## 三种起卦法

### 一、时间起卦法（最常用）

使用农历年月日时的数字起卦。

**所需数据**：
- 农历月份（数字，如四月 → 4）
- 农历日期（数字，如十一日 → 11）
- 当前时辰（地支序数，如巳时 → 6）

**计算公式**：

```
上卦数 = (农历月 + 农历日) % 8       余 0 取 8
下卦数 = (农历月 + 农历日 + 时辰序数) % 8   余 0 取 8
动爻数 = (农历月 + 农历日 + 时辰序数) % 6   余 0 取 6
```

**完整示例**：农历四月十一日巳时

```
上卦数 = (4 + 11) % 8 = 15 % 8 = 7 → 艮 ☶
下卦数 = (4 + 11 + 6) % 8 = 21 % 8 = 5 → 巽 ☴
动爻数 = (4 + 11 + 6) % 6 = 21 % 6 = 3 → 第三爻

本卦：山风蛊 ☶☴（上艮下巽）
```

> 农历日期换算：如果只知道公历日期，可直接用公历月和日代替（精度略低，但可用）。

---

### 二、数字起卦法

用户提供数字时使用。共三种情形：

#### 情形 A：用户提供两个数字

```
第一个数字为上卦，取余 8（余 0 取 8）
第二个数字为下卦，取余 8（余 0 取 8）
两数之和取余 6 为动爻（余 0 取 6）
```

示例：用户报 7 和 3

```
上卦数 = 7 % 8 = 7 → 艮 ☶
下卦数 = 3 % 8 = 3 → 离 ☲
动爻数 = (7 + 3) % 6 = 10 % 6 = 4 → 第四爻

本卦：山火贲 ☶☲（上艮下离）
```

#### 情形 B：用户提供一个多位数（如 "73"）

```
取前半段数字为上卦，后半段为下卦，全数取余 6 为动爻
```

示例：用户报 73

```
上卦数 = 7 % 8 = 7 → 艮 ☶
下卦数 = 3 % 8 = 3 → 离 ☲
动爻数 = 73 % 6 = 1 → 第一爻
```

#### 情形 C：用户提供三个数字

```
第一个数取余 8 为上卦
第二个数取余 8 为下卦
第三个数取余 6 为动爻
```

### 三、文字起卦法

用户提供字词时，按笔画数起卦。

**笔画取余规则**：

```
上卦数 = 第一个字笔画 % 8    余 0 取 8
下卦数 = 第二个字笔画 % 8    余 0 取 8
动爻数 = 所有字笔画总和 % 6  余 0 取 6
```

**各种情形处理**：

| 字数 | 上卦 | 下卦 | 动爻 |
|------|------|------|------|
| 单字 | 笔画 % 8 | 笔画 % 8（同字） | 笔画 % 6 |
| 两字 | 第一字笔画 % 8 | 第二字笔画 % 8 | (两字笔画之和) % 6 |
| 三字及以上 | 第一字笔画 % 8 | 第二字笔画 % 8 | (所有字笔画之和) % 6 |

**常用汉字笔画参考**：

## 本卦、互卦、变卦推导

### 本卦

由上卦和下卦直接组成，查六十四卦速查表得卦名。

### 互卦（中间过程）

```
互卦上卦 = 本卦第三、四、五爻（从下往上数）
互卦下卦 = 本卦第二、三、四爻（从下往上数）
```

> 爻序：第一爻（初爻）在最下方，第六爻（上爻）在最上方。

### 变卦（最终结果）

动爻所在卦中，将动爻的阴阳互换（阳变阴，阴变阳），得到变卦。

```
动爻在上卦（第四、五、六爻） → 上卦发生变化，下卦不变
动爻在下卦（第一、二、三爻） → 下卦发生变化，上卦不变
```

**阴阳互变规则**：阳爻（⚊）变阴爻（⚋），阴爻变阳爻。每个卦的三爻结构见先天八卦数表。

## 体用判断规则

```
动爻在上卦（第四、五、六爻）→ 上卦为用卦，下卦为体卦
动爻在下卦（第一、二、三爻）→ 下卦为用卦，上卦为体卦
```

**体卦**：代表问卦者自身，是判断的基准。
**用卦**：代表所问之事（对方、外部环境），与体卦产生生克关系。

## 旺衰判断表

体卦五行在不同季节的旺衰状态，影响体用生克的力度。

| 季节 | 当旺五行 | 相（次旺） | 休（平） | 囚（弱） | 死（极弱） |
|------|---------|----------|---------|---------|----------|
| 春（寅卯辰月） | 木旺 | 火相 | 水休 | 金囚 | 土死 |
| 夏（巳午未月） | 火旺 | 土相 | 木休 | 水囚 | 金死 |
| 秋（申酉戌月） | 金旺 | 水相 | 土休 | 火囚 | 木死 |
| 冬（亥子丑月） | 水旺 | 木相 | 金休 | 土囚 | 火死 |
| 四季末（辰戌丑未月） | 土旺 | 金相 | 火休 | 木囚 | 水死 |

**旺衰对判断的影响**：
- 体卦旺相 → 能承受用卦的克，凶的程度减轻
- 体卦休囚死 → 承受克的能力弱，凶的程度加重；得生的好处也更显著
- 用卦旺相 → 生体力度更强（更吉）或克体力度更强（更凶）
