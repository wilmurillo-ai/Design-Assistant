## 一、体用生克断吉凶表（核心判断）

体用生克是梅花易数断卦的第一要素，必须优先判断。

### 用生体 — 大吉

**含义**：用卦五行滋生体卦，外部环境对问卦者有利，事情主动向好发展。

**解读模板**：
- "用卦X生体卦Y，此为用生体，卦象显示外部环境顺遂，事情推进有助力"
- "此卦主吉，所问之事得到外力相助，适合积极推进"
- "旺相加持下，吉上加吉；即使体卦休囚，也因得生而有转机"

### 体生用 — 小吉偏平（耗散）

**含义**：体卦五行滋生用卦，问卦者主动付出，结果可得，但有耗费精力之象。

**解读模板**：
- "体卦X生用卦Y，体生用，主问卦者付出较多，事可成但需耗费心力"
- "结果虽可期，但过程中需投入较多资源或精力，量力而行"
- "此卦宜主动出击，但注意不要过度透支"

### 用克体 — 大凶

**含义**：用卦五行克制体卦，外部环境对问卦者不利，有压力、阻碍、受伤之象。

**解读模板**：
- "用卦X克体卦Y，此为用克体，卦象显示外部阻力较强，宜审慎"
- "此卦提示当前时机有阻碍，若体卦旺相则能承受，若体卦休囚则压力较大"
- "建议不宜强行推进，可等待时机或调整方向"

### 体克用 — 吉（我能制敌）

**含义**：体卦五行克制用卦，问卦者能掌控局面，事情在自己掌控之中。

**解读模板**：
- "体卦X克用卦Y，体克用，主问卦者具有主动权，能制驭外部变化"
- "此卦显示我方占优，所问之事有把握，可适当大胆推进"
- "体卦越旺，克制力越强，越有利"

### 体用比和 — 平（吉凶不显著）

**含义**：两卦五行相同，力量均衡，既无明显助力也无明显阻力。

**解读模板**：
- "体用五行相同，比和之卦，事情发展平稳，无大起大落"
- "此卦主平，结果取决于自身努力，外部环境不会特别帮助也不会阻碍"
- "可参考互卦和变卦，寻找细微的变化方向"

## 三、断卦完整流程（9步）

按以下顺序依次分析，不可跳步。

**Step 1：看体用关系**
判断用生体 / 体生用 / 用克体 / 体克用 / 比和，确定吉凶大方向。这是最重要的一步。

**Step 2：看旺衰**
查当前季节，判断体卦五行的旺衰状态（旺相休囚死）。旺衰修正体用关系的力度。

**Step 3：看互卦**
互卦反映事情的中间发展过程。互卦的体用关系提示中途的顺逆。

**Step 4：看变卦**
变卦反映事情的最终走向。变卦体用关系是对结局的补充判断。

**Step 5：看本卦卦义**
六十四卦各有其象和义，参考卦名的传统含义，丰富断卦内容。

**Step 6：结合用户问题，看万物类象**
- 问事业：看体用卦的事业类象
- 问感情：看体用卦的人物类象（对方是何种人）
- 问财运：看体用卦的财物类象
- 问健康：看体用卦的身体类象

**Step 7：判断应期**
根据体用五行和动爻爻位，推算可能应验的时间。

**Step 8：综合判断**
整合体用、旺衰、互变、类象，给出明确的方向性结论。

**Step 9：给出行动建议**
结合卦象吉凶，给出 2-3 条可执行的具体建议。

## 四、应期判断规则

应期是指事情可能应验的时间。以下是参考规则：

### 按五行推应期

| 体卦五行 | 日/月应期方向 |
|---------|-------------|
| 金 | 申、酉月日（秋季，农历七八月） |
| 木 | 寅、卯月日（春季，农历一二月） |
| 水 | 亥、子月日（冬季，农历十、十一月） |
| 火 | 巳、午月日（夏季，农历四五月） |
| 土 | 辰、戌、丑、未月日（四季末） |

### 按爻位推应期

| 动爻位置 | 应期提示 |
|---------|---------|
| 初爻（第一爻） | 时间较久，或当日 |
| 二爻 | 2 日内，或近期 |
| 三爻 | 3 日内，或本月内 |
| 四爻 | 月内，或近一季度 |
| 五爻 | 较快，可能在 5 日内 |
| 上爻（第六爻） | 时间较远，或事情趋于结束 |

> 应期只是参考方向，不作精确预测。表达时用"趋向""可能在""参考节点"等措辞。

## 五、表达风格指南

### 基本原则

梅花易数断卦使用**易学顾问型**风格：理性推导 + 实用建议，展示逻辑，避免神秘化。

**正面表达**：
- "体用关系显示..."
- "卦象提示当前时机..."
- "此卦在[领域]方向，显示..."
- "互卦坎水出现，中间过程可能有波折..."
- "变卦转为[卦名]，最终走向趋于..."

**负面信息的处理方式**：
1. 先说卦象客观显示什么
2. 再说背后原因（五行逻辑）
3. 最后给出应对建议

示例（用克体时的表达）：
> "用卦X克体卦Y（X克Y），卦象提示当前外部阻力较强。若体卦X在[当前季节]旺相，尚能承受；若体卦休囚，则压力较大。建议此时不宜强行推进，可等待[体卦五行旺相的季节/月份]再行动。"

## 六、禁用表达清单

| 禁止使用 | 替换为 |
|---------|-------|
| ❌ "此卦大凶，必然失败" | ✅ "卦象显示阻力较大，需谨慎应对" |
| ❌ "你命中注定..." | ✅ "卦象在当下时机显示...的趋势" |
| ❌ "这件事绝对不行" | ✅ "当前时机不利，建议等待转机" |
| ❌ "必有血光之灾" | ✅ "卦中坎水（主险）出现，宜注意安全，留意潜在风险" |
| ❌ "下次多来算算" | ✅ "梅花易数只是参考，最终决策权在你手中" |
| ❌ "你的运气太差了" | ✅ "当前时机偏不利，可等待更好的节点" |

**绝对禁止**：死亡预测、绝对化断言、恐吓性语言、依赖性引导（催促用户频繁占卜）。

## 七、常见问题方向解读要点

### 感情/婚姻

- 体卦为自己，用卦为对方（或感情本身）
- 用生体：对方主动，情意深厚
- 体克用：自己占主导，但需注意不要过于强势
- 互卦看过程：是否顺利推进
- 变卦看结果：最终走向（稳定/结束/转变）
- 类象参考：兑（少女）、坎（中男）对应的感情特质

### 事业/工作

- 体卦为问卦者自身，用卦为事业、公司或上级
- 用生体：环境支持，职场顺利
- 用克体：压力大，竞争激烈，或上级难搞
- 类象参考：乾（权力、领导）、离（文职、创意）、坎（策划、风险）

### 财运/投资

- 用卦为财（钱）
- 用生体：财来找我，财运顺
- 体克用：我能主导，适合主动出击
- 用克体：投资有风险，不宜冒进
- 类象参考：坤（大众财、稳定）、乾（大财、高风险高回报）

### 健康

- 体卦为身体状况，用卦为疾病/外邪
- 用克体：病势较重，需重视
- 体克用：自身抵抗力强，可恢复
- 类象参考：各卦对应的身体部位（见万物类象表）

### 出行/移动

- 体卦旺相 + 用生体：出行顺利
- 体卦休囚 + 用克体：宜留守，不宜远行
- 坎卦出现：注意水上交通安全
- 震卦出现：动而有利，宜出发
