# 每日推荐策略

## 当前实现

由于免费 API 的限制和性能考虑，当前每日推荐采用简化策略：

1. **股票池**: 预定义的热门股票列表（各行业龙头）
2. **随机选择**: 基于日期种子的随机选择，确保同一天推荐一致
3. **伪得分**: 随机生成推荐得分用于排序

## 未来优化方向

### 技术面指标
- **均线系统**: 5 日、20 日、60 日均线排列
- **成交量**: 相对均量比（当日量 / 5 日均量）
- **RSI**: 相对强弱指标（30-70 健康区间）
- **MACD**: 趋势判断

### 基本面筛选
- **估值指标**: PE、PB、PEG
- **财务健康**: 资产负债率、现金流
- **成长性**: 营收增长率、净利润增长率
- **分红**: 股息率、分红稳定性

### 数据源扩展
- **雅虎财经**: 美股数据更全
- **Alpha Vantage**: 免费 API（需要注册）
- **本地数据库**: 存储历史数据用于回测

## 使用建议

1. **交易时间**: 推荐在交易时间内查看，数据更准确
2. **风险控制**: 不要全仓单只股票，分散投资
3. **持续跟踪**: 关注推荐股票的基本面变化
4. **止损策略**: 设置合理的止损位

## 自定义股票池

用户可以修改 `scripts/daily_picks.py` 中的 `STOCK_POOL` 变量来自定义推荐范围：

```python
STOCK_POOL = [
    ('600519', '贵州茅台'),
    ('000858', '五粮液'),
    # 添加更多股票...
]
```

支持 A 股、港股、美股代码格式。
