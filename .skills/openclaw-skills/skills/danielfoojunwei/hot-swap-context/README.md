# Portable Context OS Isolated Text Package

This package contains an isolated `SKILL.md` plus supporting references, templates, and source mirrors for the upgraded `portable-context-os` skill.

## Package rule

Every file in this package is either **Markdown** or **plain text**. To preserve that constraint, executable Python and YAML files are included here as `.txt` mirrors.

## Included materials

| Path | Role |
|---|---|
| `SKILL.md` | Standalone skill definition |
| `references/` | Architecture, paradigm-shift, and governance references |
| `templates/` | Editable Markdown and text template assets |
| `source_text/` | Plain-text mirrors of the builder and bundle scripts |

## Important note

This package is intended for inspection, sharing, review, and editing in text-only environments. The `.txt` source mirrors preserve the implementation content, but they are not executable until renamed back to their original file extensions.
