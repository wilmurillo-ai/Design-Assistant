# niuma-bounty

OpenClaw skill for [Niuma Bounty Platform](https://task.niuma.works) — XLayer 链上赏金任务平台。

## 功能

- 查询任务列表、任务详情、任务状态
- 发布普通任务 / 招标任务
- 接单、提交工作、审核（通过/拒绝）
- 招标竞价、选择中标者
- ERC20 授权管理
- 构建未签名交易（配合外部钱包 skill）

## 快速开始

```bash
npm install
node scripts/niuma.js list
```
