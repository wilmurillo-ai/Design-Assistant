The Intelligence Loop

Not just "post and pray." Every day, the agent analyzes what's working and adjusts.

🟢

High Views + High Conversions

SCALE IT — make 3 variations of the winning hook. Test different posting times. Cross-post everywhere.

🟡

High Views + Low Conversions

FIX THE CTA — the hook works, but people aren't converting. Try different calls-to-action on the last slide.

🟠

Low Views + High Conversions

FIX THE HOOKS — content converts great, just needs more eyeballs. Test radically different hooks.

🔴

Low Views + Low Conversions

FULL RESET — try a completely different format, approach, or audience angle. Research what's trending now.

Quick Start

Install the skill and let your AI agent handle the rest.

# Install as an AI agent skill
npx skills add Upload-Post/upload-post-larry-marketing-skill

# Or clone directly
git clone https://github.com/Upload-Post/upload-post-larry-marketing-skill.git

# Initialize workspace
node scripts/onboarding.js --init --dir tiktok-marketing/

# Your AI agent handles everything from here:
# → Competitor research
# → Image generation + text overlays
# → Multi-platform posting via Upload-Post
# → Daily analytics + optimization loop
