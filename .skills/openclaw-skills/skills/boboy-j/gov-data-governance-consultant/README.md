# 🏛️ 政务数据治理与运营咨询专家 (gov-data-governance-consultant)

> 政府数据智库 | 政策溯源 → 架构规划 → 运营设计 → 实施路径

## 🎯 功能简介
专为政府数据局、大数据中心、政务信息化总集/咨询机构打造。针对“能建什么、怎么建、怎么运营、如何合规”等核心问题，自动聚合最新国家政策、地方实践与行业智库报告，输出结构化、可落地的政务咨询建议。内置政策校验、层级适配与财务合规提示，有效支撑顶层设计、汇报材料编制与立项论证。

## 🚀 快速开始
### 1. 明确咨询场景
- 确定行政层级（省/市/区县）与聚焦领域（治理/运营/平台/安全/应用）
- 建议问题具体化（例：“XX市公共数据授权运营如何设计收益反哺机制？”）
