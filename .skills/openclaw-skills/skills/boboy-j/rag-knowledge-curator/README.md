# 🗃️ 企业知识库治理引擎 (rag-knowledge-curator)

> RAG 数据基建核心组件 | 智能分块/去重/打标/质检 | 开箱即用

## 🎯 功能简介
专为向量数据库入库前数据处理设计。自动清洗噪声文本、按语义/标题/固定长度智能分块、提取标准化元数据标签、执行四维质量评分，输出可直接对接 Embedding 管道的结构化清单。解决企业 RAG 系统中常见的“检索不准、幻觉高、更新混乱”问题。

## 🚀 快速开始
### 1. 准备原始资料
- 支持粘贴 Markdown/纯文本/网页抓取内容
- 单次建议输入 ≤ 5000 字符，长文档请分批处理或配合文件解析工具

### 2. 配置调用
```json
{
  "skill": "rag_knowledge_curator",
  "inputs": {
    "raw_documents": "<待处理文本内容>",
    "chunk_strategy": "semantic",
    "domain": "tech"
  }
}
