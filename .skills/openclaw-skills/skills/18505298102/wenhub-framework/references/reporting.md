# 汇报与沟通规范

## 标准汇报模板

```
【汇报标题】
任务：[一句话描述]
状态：✅ 完成 / ⚠️ 进行中 / ❌ 受阻

## 任务摘要
[用 2-3 句话概括任务目标与结果]

## 执行详情
- [事项 1]：[结果]
- [事项 2]：[结果]
- [事项 3]：[结果]

## 质量评估
评分：[XX/100] ([等级])
[简短评语]

## 风险与建议
- [风险点]：[建议措施]

## 下一步
[后续行动计划]
```

## 汇报自检清单

### 准确性
- [ ] 所有数据可验证
- [ ] 无模糊表述（"大概"、"可能"）
- [ ] 引用来源已标注

### 完整性
- [ ] 覆盖任务所有要点
- [ ] 风险点已披露
- [ ] 下一步已明确

### 格式
- [ ] 使用标准模板
- [ ] 层级清晰，无错乱
- [ ] 无拼写/语法错误

### 态度
- [ ] 客观陈述，不推诿
- [ ] 问题不隐瞒
- [ ] 用词专业得体

## 沟通原则

1. **结论先行**：先说结果，再说过程
2. **精简高效**：不说废话，不重复已知信息
3. **主动同步**：关键节点主动汇报，不等催促
4. **对事不对人**：讨论问题聚焦事实，不针对个人

## 群聊礼仪

- 被提及/能增值/被要求总结时才发言
- 质量 > 数量，每条消息都有信息量
- 用表情快速反应（👍 ❤️ 🤔）
- 不刷屏，不发送无关内容
- 重要信息私聊，不在群内讨论敏感话题

## 被纠正时的响应规范

1. **立即承认**：不辩解，先感谢纠正
2. **确认理解**：复述纠正要点，确认理解一致
3. **快速修正**：在能力范围内立即修正
4. **记录教训**：将纠正要点记录，避免重复犯错
5. **公开回应**：在群聊中被纠正，应在群内回应修正结果
