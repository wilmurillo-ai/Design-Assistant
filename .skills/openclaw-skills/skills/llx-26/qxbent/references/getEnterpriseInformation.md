# 查询企业工商信息

## 接口

`getEnterpriseInformation`

## 描述

查询企业的工商注册信息，包括统一社会信用代码、法定代表人、注册资本、成立日期等基础数据。

## 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ename | string | 是 | 企业名称 |

## 返回字段

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 统一社会信用代码 | string | 企业统一社会信用代码 |
| 企业名称 | string | 企业全称 |
| 法定代表人 | string | 法定代表人姓名 |
| 经营状态 | string | 企业当前经营状态 |
| 注册资本 | string | 注册资本金额 |
| 成立日期 | string | 企业成立日期 |
| 实缴资本 | string | 实际缴纳资本 |
| 组织机构代码 | string | 组织机构代码 |
| 工商注册号 | string | 工商注册号码 |
| 纳税人识别号 | string | 税务登记号 |
| 纳税人资质 | string | 纳税人资质类型 |
| 企业类型 | string | 企业法律类型 |
| 营业期限 | string | 营业期限范围 |
| 登记机关 | string | 工商登记机关 |
| 所属行业 | string | 所属行业分类 |
| 所属地区 | string | 注册地区 |
| 注册地址 | string | 注册地址 |
| 经营范围 | string | 经营范围描述 |

## 示例代码

```typescript
import { createClient } from 'qxbent-skills'

const client = createClient()

// 查询企业工商信息
const info = await client.getEnterpriseInformation('恒大地产集团有限公司')

console.log('企业名称:', info.企业名称)
console.log('统一社会信用代码:', info.统一社会信用代码)
console.log('法定代表人:', info.法定代表人)
console.log('注册资本:', info.注册资本)
console.log('成立日期:', info.成立日期)
```
