# TTS 语音生成器

## 使用方式

### 示例命令

```
列出所有可用的声音
```

```
用第 2 个声音生成：你好，这是测试文本
```

```
用温柔女声生成：春眠不觉晓，处处闻啼鸟
```

```
上传这个音频文件作为参考声音：./sample.wav
```

```
查询任务状态：taskId_xxx
```

### 完整工作流程

1. **浏览声音**：先查看可用声音列表
2. **选择声音**：说"用第 X 个声音"或"用 [声音名称]"
3. **生成语音**：提供文本内容，使用选中的声音生成
4. **自定义声音**：上传自己的音频作为参考声音

## 触发方式

当用户请求：
- "生成语音" / "TTS" / "文本转语音"
- "列出声音" / "获取声音列表" / "有哪些声音"
- "上传音频" / "添加声音" / "自定义声音"
- "查询任务状态" / "检查 TTS 进度"
- "用第 X 个声音生成：[文本]"
- "用 [声音名称] 生成：[文本]"
- 提供文本内容要求转换成语音

## 配置要求

### 首次使用

需要在 `~/.openclaw/config.json` 中配置 API Key（与下载工具 Skill 共用）：

```json
{
    "datamass_api_key": "您的 API Key"
}
```

### 获取 API Key

1. 访问 https://www.datamass.cn
2. 注册并登录
3. 创建 API Key
4. 复制生成的 API Key 到配置文件中

## 功能特点

- 支持多种声音风格（影视娱乐、新闻热点、科普知识等）
- 支持声音克隆和定制
- 支持按序号选择声音（"用第 2 个声音"）
- 支持按名称选择声音（"用温柔女声"）
- 异步任务处理，无需等待
- 自动状态轮询
- 音频文件保存到 OSS，24 小时有效

## 核心功能

| 方法 | 说明 |
|------|------|
| `show_voice_list()` | 显示可用声音列表（格式化输出） |
| `list_voices()` | 获取可用声音列表（原始数据） |
| `select_voice(index)` | 根据序号选择声音 |
| `generate_speech()` | 提交 TTS 任务（支持 voice_index/voice_name/media_path） |
| `check_task_status()` | 查询任务状态 |
| `upload_audio()` | 上传音频文件 |
| `add_custom_voice()` | 添加定制声音 |
| `upload_and_add()` | 一键上传并添加为定制声音 |

## 注意事项

- TTS 服务按生成音频时长计费
- 最低余额门槛：5 积分
- 100 字以内：2 积分，超出部分每 100 字加 1 积分
- 需要充值积分后使用
