## 📖 简介

Adaptive Socratic Questioning 是一个基于苏格拉底式提问法的 AI 技能，专注于培养科研思维、批判性思维和独立解决问题的能力。

不同于直接给出答案，本技能通过精心设计的连续追问，引导用户：
- 🔍 发现知识盲点
- 🧩 构建逻辑链条  
- ✅ 验证假设合理性
- 💡 形成独立判断能力

## ✨ 核心特性

### 四阶段工作流程

1. **问题理解与澄清** - 明确研究边界和目标
2. **引导探索与发现** - 帮助发现解决方案
3. **深化思考与批判** - 促进深度反思
4. **行动与验证** - 制定可执行计划

### 多元追问策略

- **渐进深入策略** - 从简单到复杂逐步推进
- **逆向追问策略** - 从结论追溯推理过程
- **对比追问策略** - 通过对比深化理解
- **假设性追问策略** - 拓展思维边界

### 智能适应性

- 根据用户水平调整问题难度
- 根据问题类型调整追问重点
- 根据对话进展调整交互方式

## 🚀 使用方法

### 触发条件

当遇到以下场景时，可以请求加载此技能：

- 科研问题探讨
- 学术研究指导
- 方法论咨询
- 批判性思维训练
- 论文写作辅导

### 使用示例

```
用户：我想研究社交媒体对青少年心理健康的影响。

AI：这是一个很有意义的研究方向。让我通过几个问题帮助你理清思路：
    1. 这个研究问题具体关注哪个年龄段的青少年？
    
用户：12-18岁的中学生。

AI：好的。那么你打算研究社交媒体的哪些具体使用行为？
    是使用时长、使用频率，还是具体的使用方式？
    
用户：我想研究每天使用时长与焦虑水平的关系。

AI：理解了。那么心理健康指标中的"焦虑"，你打算如何测量？
    是否有特定的量表或评估工具？
```

## 💡 设计理念

### 苏格拉底式教学

苏格拉底认为，真正的知识来源于内心的觉醒，而非外在的灌输。通过不断追问"为什么"，帮助学生：

- 质疑习以为常的假设
- 发现思维中的矛盾
- 建立更稳固的知识体系

### 建构主义学习

知识不是被动接收的，而是主动建构的。本技能通过：

- 搭建"脚手架"支持学习
- 提供适度的认知挑战
- 促进最近发展区内的成长
