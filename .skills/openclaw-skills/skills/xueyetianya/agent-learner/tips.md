# Agent Learner — Tips

1. **Log mistakes immediately** — Don't wait. The context is freshest right after the error occurs.

2. **Include the "why"** — When logging mistakes, always note why it happened, not just what happened.

3. **Pair corrections with mistakes** — Link each correction back to the original mistake for complete learning loops.

4. **Run pattern analysis weekly** — Regular pattern checks reveal blind spots you didn't know you had.

5. **Use quiz mode honestly** — Don't peek at answers. The point is to test genuine recall and understanding.

6. **Review reports monthly** — Track whether your mistake rate is actually decreasing over time.

7. **Share best practices** — Export your learned best practices to share across agent instances.

8. **Reset selectively** — Instead of full reset, consider archiving old data first for historical reference.
