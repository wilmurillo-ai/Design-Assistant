# 💡 Data Visualizer Tips

1. **Consistent input format**: Use `Label:Value` pairs, comma-separated — e.g., `Q1:100,Q2:150,Q3:200`.

2. **Bar charts for comparison**: Best with 10 or fewer categories. More than that gets cluttered — consider grouping.

3. **Line charts for trends**: Time-series data is best shown as line charts — growth and decline patterns jump out immediately.

4. **Keep pie charts under 5 slices**: Too many slices become unreadable. More than 5 categories? Switch to a bar chart.

5. **SVG is editable**: Output SVG can be tweaked in any text editor — change colors, sizes, or labels after generation.

6. **Save and open in browser**: Save SVG output to a `.svg` file and double-click to view. Works in any modern browser.

7. **Dashboard for presentations**: Dashboards combine multiple metrics on one view — perfect for executive reports.

8. **Trend charts auto-annotate**: Trend mode marks high and low points automatically — great for spotting anomalies.
