# Task Planner — Tips

1. **Start with plan** — Always decompose before estimating. You can't estimate what you haven't broken down.

2. **Be honest with estimates** — Optimistic estimates cause more harm than conservative ones. Add buffer.

3. **Map dependencies first** — Before prioritizing, know what blocks what. Dependencies shape the critical path.

4. **Look for parallel opportunities** — Independent subtasks can run simultaneously, dramatically reducing total time.

5. **Assess risk per subtask** — Not all tasks carry equal risk. Focus mitigation on high-risk items.

6. **Update progress regularly** — Stale progress data leads to bad decisions. Keep it current.

7. **Always do retrospectives** — The biggest gains come from learning what worked and what didn't after completion.
