# Review Responder — Tips

## 使用建议

1. **好评回复 (positive)**: 真诚感谢+引导复购+推荐其他产品
2. **差评处理 (negative)**: 先道歉+再解决+最后跟进
3. **回复模板 (template)**: 避免千篇一律，个性化修改
4. **评价分析 (analysis)**: 定期分析评价趋势，发现问题
5. **改进建议 (improve)**: 将差评转化为产品改进方向
6. **批量回复 (batch)**: 提高效率但保持个性化

## 最佳实践

- 24小时内回复，越快越好
- 差评要私信+公开回复双管齐下
- 不要和客户争论，解决问题为先
- 好评回复要走心，别用万能模板
- 定期分析差评数据，找到根因
