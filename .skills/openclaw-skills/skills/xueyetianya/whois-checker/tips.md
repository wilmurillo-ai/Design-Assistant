## Quick Tips

1. **Check availability first** — Use `available` command for quick domain availability checks
2. **Monitor expiry dates** — Set up `monitor` with your domain portfolio file to track expirations
3. **Use warning thresholds** — Set `--warn-days 90` for enterprise domains needing advance renewal
4. **Batch for portfolios** — Put all your domains in a text file for efficient bulk lookups
5. **JSON for automation** — Use `--format json` to feed into domain management systems
6. **Raw for debugging** — Use `raw` command when parsed output doesn't capture all fields
7. **Check nameservers** — Verify nameservers match expected values after DNS migrations
8. **Status codes** — Domain status like `clientTransferProhibited` tells you about domain lock state
9. **Rate limits** — WHOIS servers have rate limits; add delays between batch queries
10. **CSV for spreadsheets** — Export as CSV for easy import into spreadsheet tools
