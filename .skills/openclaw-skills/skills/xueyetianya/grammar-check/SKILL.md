# grammar-check

语法检查助手。英语纠错、语法解释、难度分级、语法练习题、中英对比分析。Grammar checker with error correction, explanations, difficulty levels, practice exercises, Chinese-English comparison.

## 常见问题

**Q: 这个工具适合谁用？**
A: 任何需要grammar-check的人，无论是个人还是企业用户。

**Q: 输出格式是什么？**
A: 主要输出Markdown格式，方便复制和编辑。

```
  check           check

  compare         compare
```

## 专业建议

- 冠词遗漏**: ~~I am student~~ → I am **a** student
- 时态混乱**: ~~I go there yesterday~~ → I **went** there yesterday
- 主谓不一致**: ~~He don't~~ → He **doesn't**
- 可数/不可数**: ~~informations~~ → information
- 介词错误**: ~~arrive to~~ → arrive **at/in**
