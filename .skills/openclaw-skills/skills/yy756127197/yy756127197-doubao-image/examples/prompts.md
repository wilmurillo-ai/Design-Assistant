## Prompt 编写技巧总结

### 基本公式
```
主体描述 + 环境氛围 + 艺术风格 + 技术参数
```

### 质量关键词
- 4K, 8K, 超高清，杰作
- 精致细节，超精细
- 电影大片，国家地理

### 光影关键词
- 黄金时刻，蓝色时刻
- 丁达尔效应，体积光
- 光线追踪，全局光照

### 构图关键词
- 对称构图，三分法
- 广角，微距，鱼眼
- 鸟瞰，仰视，特写

### 风格关键词
- 写实，抽象，印象派
- 赛博朋克，蒸汽朋克
- 水彩，油画，素描
