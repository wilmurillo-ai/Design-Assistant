# 宏观数据源参考

## 一级数据源（权威机构）

### 国家统计局
- URL: https://www.stats.gov.cn
- 重点数据: GDP、行业增加值、固定资产投资、社零、人口结构
- 路径: 国家统计局 → 统计数据 → 分省年度/月度数据

### 中国人民银行
- URL: https://www.pbc.gov.cn
- 重点数据: M2、社融、LPR、信贷投向、外汇储备
- 路径: 人民银行 → 调查统计 → 金融统计数据报告

### 国家发改委
- URL: https://www.ndrc.gov.cn
- 重点数据: 产业政策、重大项目批复、区域规划
- 路径: 发改委 → 政策发布 → 产业发展

### 工信部
- URL: https://www.miit.gov.cn
- 重点数据: 工业增加值、高技术制造业、专精特新
- 路径: 工信部 → 运行监测协调 → 行业运行

### 国家医保局
- URL: https://www.nhsa.gov.cn
- 重点数据: DRG/DIP推进进度、集采结果、医保基金收支
- 路径: 医保局 → 政策法规 / 数据公开

### NMPA（国家药监局）
- URL: https://www.nmpa.gov.cn
- 重点数据: 创新药审评、医疗器械注册、绿色通道
- 路径: 药监局 → 法规 / 数据查询

### 证监会/交易所
- 科创板: https://www.sse.com.cn （18A规则、医药企业上市动态）
- 港交所: https://www.hkex.com.hk （18A上市规则）

## 二级行业数据源

- 中国医药创新促进会（PhRDA）
- 弗若斯特沙利文（Frost & Sullivan）行业报告
- 艾瑞咨询医疗健康行业报告
- 麦肯锡/BCG/贝恩医疗健康白皮书
- 清科研究中心（投资/融资数据）

## 境外数据源

- FDA: https://www.fda.gov
- EMA: https://www.ema.europa.eu
- ClinicalTrials.gov: https://clinicaltrials.gov （临床试验数据）
- WHO: https://www.who.int
- PitchBook / CB Insights （海外投融资）
- Crunchbase （全球创投数据）
