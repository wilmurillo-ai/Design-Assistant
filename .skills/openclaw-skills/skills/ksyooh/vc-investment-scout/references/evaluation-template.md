# 企业基本面评估模板

> 根据投资阶段自动选择对应评估维度和量化标准。

## 评估流程

1. 确认投资阶段（天使/VC/成长/并购）
2. 根据阶段加载对应评估维度
3. 对每个维度进行数据收集和评分
4. 汇总评分，给出投资建议

## 评估维度映射

### 天使投资（0到1）
- [ ] 技术可行性 — 是否完成PCC
- [ ] 安全风险 — 动物实验数据
- [ ] 伦理与合规 — NMPA/FDA路径
- [ ] 经济与运营 — 成本对比
- [ ] 团队能力 — 科研+商业背景
- [ ] 商业模式 — 支付闭环、HIS/LIS嵌入
- [ ] 政策匹配 — 国家战略方向

### VC投资（1到10）
- [ ] 技术创新性 — 发明专利
- [ ] 产业化程度 — 临床/市场进度
- [ ] 市场前景 — 市场规模、CAGR
- [ ] 团队实力 — 高管经验
- [ ] 发展规划 — 商业模式、风险预警

### 成长性投资（10到100）
- [ ] 核心技术 — 知识产权
- [ ] 团队互补性 — 科学家+经理人
- [ ] 市场需求 — 刚性需求
- [ ] 财务健康度 — PS/PE/rNPV
- [ ] 安全边际 — ROE

### 并购投资（100到N）
- [ ] 财务表现 — 毛利率、ROE、资产负债率
- [ ] 估值水平 — P/E、EBIT
- [ ] 技术壁垒 — FIC/BIC、专利
- [ ] 战略协同性 — 互补效应
- [ ] 人才保留 — 激励方案

## 评分标准

每个维度采用5分制：
- 5分：显著超出标准
- 4分：满足标准
- 3分：基本满足但有明显不足
- 2分：部分满足，需重大改善
- 1分：不满足

综合评级：
- 总分占比≥80%：强烈推荐
- 60-80%：推荐
- 40-60%：观望
- <40%：不推荐
