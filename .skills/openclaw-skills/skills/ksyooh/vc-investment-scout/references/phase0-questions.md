# Phase 0 用户偏好收集 - 交互式选择题

## 偏好检查流程

1. 检查 `memory/vc-user-prefs.md` 是否存在
2. **不存在（新用户）** → 发送引导消息 → 依次收集全部偏好
3. **存在（老用户）** → 简要确认偏好是否有变化

## 引导消息

新用户首次使用时发送：

```
你好，我是VC投资筛选助手。我可以帮你完成：

1. 完整投资筛选 — 从宏观政策→行业趋势→细分赛道→企业评估，四层递进式筛选
2. 企业评估 — 对某个具体企业做基本面打分和投资建议
3. 赛道追踪 — 定期更新你关注的赛道动态
4. 历史回顾 — 调取之前的分析报告，不用重跑

你可以直接说：
- "帮我做投资筛选" — 跑完整四层分析
- "评估一下英矽智能" — 单独评估某个企业
- "上次AI医疗的分析结果" — 查看历史报告

要开始吗？我会先了解你的投资偏好，然后开始分析。
```

## 问题组1：投资阶段

```
请选择你的主要投资阶段（可多选）：

A. 天使投资（0到1，源头创新，5-8年回收期）
B. 风险投资/VC（1到10，成长确定性，4-6年回收期）
C. 成长性投资（10到100，规模化，3-5年回收期）
D. 并购投资（100到N，生态整合，2-4年回收期）

请回复选项字母，如 A、AB、BCD
```

## 问题组2：关注行业

```
请选择你关注的行业方向（可多选）：

A. AI应用（大模型、AI Agent、垂直应用）
B. 机器人与自动化（工业机器人、服务机器人、具身智能）
C. 半导体（芯片设计、制造、设备、材料）
D. 医疗健康（创新药、医疗器械、数字医疗、诊断、康复）
E. 金融科技（支付、保险科技、财富管理）
F. 新能源（电池、光伏、储能、氢能）
G. 消费科技（智能硬件、新消费品牌）
H. 其他（请说明）

请回复选项字母
```

## 问题组3：医疗细分（仅当用户选了D时追问）

```
医疗健康方向中，你最关注的细分领域（可多选）：

A. 创新药与生物药（小分子、大分子、细胞基因治疗）
B. 高端医疗器械（影像设备、手术机器人、介入器械）
C. 数字健康与AI医疗（远程医疗、智能诊疗、健康大数据）
D. 体外诊断（IVD、分子诊断、POCT）
E. 康复医疗与智慧养老
F. 医药服务（CRO/CDMO、临床研究）

请回复选项字母
```

## 问题组4：查询原因

```
你这次想分析什么？

A. 看看当前哪些赛道值得关注
B. 有具体企业想评估
C. 跟踪某个行业的最新变化
D. 准备投委会材料，需要完整报告

请回复选项字母
```

## 偏好保存格式

```markdown
# VC投资偏好配置
- 更新时间: YYYY-MM-DD HH:MM
- 用户: [用户名]
- 投资阶段: [选择]
- 关注行业: [选择]
- 医疗细分: [选择或"未选"]
- 单笔规模: [选择或"未指定"]
- 本次查询原因: [选择]
```
