# 🎯 SWOT 分析

> 知己知彼，百战不殆

---

## 📖 什么是 SWOT 分析

SWOT 是一种**内外部综合分析**框架，涵盖四个维度：

- **S**trengths (优势) - 内部有利因素
- **W**eaknesses (劣势) - 内部不利因素
- **O**pportunities (机会) - 外部有利因素
- **T**hreats (威胁) - 外部不利因素

---

## 🎯 适用场景

- ✅ 竞争分析与战略制定
- ✅ 个人/组织自我评估
- ✅ 项目可行性分析
- ✅ 决策前的全面评估

---

## 🔧 分析框架

### 内部因素 (可控)
```
┌─────────────────┬─────────────────┐
│   优势 (S)      │   劣势 (W)      │
│   • 我们擅长什么？│   • 我们不足什么？│
│   • 独特资源？   │   • 资源缺口？   │
│   • 竞争优势？   │   • 需要改进？   │
└─────────────────┴─────────────────┘
```

### 外部因素 (不可控)
```
┌─────────────────┬─────────────────┐
│   机会 (O)      │   威胁 (T)      │
│   • 市场趋势？   │   • 竞争压力？   │
│   • 政策利好？   │   • 技术颠覆？   │
│   • 新兴需求？   │   • 经济风险？   │
└─────────────────┴─────────────────┘
```

## 📊 输出示例

### 某创业公司 SWOT 分析（摘要）

| 维度 | 因素 | 优先级 |
|------|------|--------|
| **S** | 技术领先、团队经验丰富 | 🔴 高 |
| **W** | 资金有限、品牌知名度低 | 🟡 中 |
| **O** | 市场快速增长、政策支持 | 🔴 高 |
| **T** | 巨头入场、经济下行 | 🟡 中 |

**SO 策略**：利用技术优势快速占领新兴市场
**WO 策略**：寻求融资弥补资金短板
**ST 策略**：建立技术壁垒应对巨头竞争
**WT 策略**：控制成本，聚焦核心业务

---

## ⚠️ 使用提示

1. **具体化** - 避免空泛描述，用数据和事实支撑
2. **优先级** - 不是所有因素都同等重要
3. **动态更新** - 定期重新评估（季度/半年）
4. **交叉分析** - SWOT 的价值在于交叉组合策略
