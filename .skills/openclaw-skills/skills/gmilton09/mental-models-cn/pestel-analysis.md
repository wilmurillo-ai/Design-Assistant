# 🌍 PESTEL 分析

> 宏观环境扫描的完整框架

---

## 📖 什么是 PESTEL 分析

PESTEL 是一种分析**宏观环境**的结构化框架，涵盖六个维度：

- **P**olitical（政治）
- **E**conomic（经济）
- **S**ocial（社会）
- **T**echnological（技术）
- **E**nvironmental（环境）
- **L**egal（法律）

---

## 🎯 适用场景

- ✅ 行业研究与市场分析
- ✅ 战略规划与投资决策
- ✅ 新市场进入评估
- ✅ 风险识别与机会发现

---

## 🔧 分析框架

### P - 政治因素 (Political)
```
- 政府政策与稳定性
- 贸易政策与关税
- 税收政策
- 劳动法
- 环境法规
- 政治风险
```

### E - 经济因素 (Economic)
```
- GDP 增长率
- 利率与汇率
- 通货膨胀率
- 失业率
- 可支配收入
- 经济周期阶段
```

### S - 社会因素 (Social)
```
- 人口结构与趋势
- 文化价值观
- 生活方式变化
- 教育水平
- 健康意识
- 消费习惯
```

### T - 技术因素 (Technological)
```
- 技术创新速度
- 研发投入
- 技术采用率
- 自动化趋势
- 数字化程度
- 技术壁垒
```

### E - 环境因素 (Environmental)
```
- 气候变化影响
- 碳排放法规
- 可持续发展趋势
- 资源稀缺性
- 废物管理
- 绿色技术
```

### L - 法律因素 (Legal)
```
- 行业法规
- 消费者保护法
- 知识产权法
- 数据保护法
- 竞争法
- 健康与安全法
```

## 📊 输出示例

### 电动汽车行业 PESTEL 分析（摘要）

| 维度 | 关键趋势 | 影响程度 |
|------|----------|----------|
| **P** | 各国补贴政策支持 | 🔴 高 |
| **E** | 电池成本下降 89% (2010-2020) | 🟢 高 |
| **S** | 环保意识提升 | 🟢 中 |
| **T** | 自动驾驶技术突破 | 🟢 高 |
| **E** | 碳中和目标 | 🟢 高 |
| **L** | 排放法规趋严 | 🔴 高 |

## ⚠️ 使用提示

1. **动态更新** - 宏观环境持续变化，定期更新分析
2. **权重区分** - 不同因素对不同行业影响程度不同
3. **交叉验证** - 结合 SWOT、波特五力等模型
4. **数据支撑** - 尽量使用量化数据而非定性描述
