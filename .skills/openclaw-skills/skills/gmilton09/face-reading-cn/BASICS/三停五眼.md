# 📐 三停五眼 - 面部比例基础

**面相学第一课**: 面部比例是面相分析的基础

---

## 🎯 什么是三停？

三停是将面部垂直分为三个部分，代表不同年龄段的运势。

```
        发际线
           │
    ═══════╪═══════  上停 (15-30 岁)
           │       早年运、父母、学业
    ═══════╪═══════  眉毛
           │
           │       中停 (31-50 岁)
           │       中年运、事业、财富
           │
    ═══════╪═══════  鼻尖
           │
           │       下停 (51 岁+)
           │       晚年运、子女、福气
           │
    ═══════╪═══════  下巴
```

---

## 📊 三停详解

### 上停 - 早年运势（15-30 岁）

**位置**: 发际线到眉毛

**代表**:
- 早年运势
- 父母关系
- 学业成就
- 智力发展

**吉凶判断**:
| 特征 | 含义 |
|------|------|
| 饱满圆润 | 聪明、学业好、父母助力 |
| 高广开阔 | 志向远大、早年顺利 |
| 低窄凹陷 | 早年辛苦、学业波折 |
| 有疤痕 | 学业或家庭有变故 |

---

### 中停 - 中年运势（31-50 岁）

**位置**: 眉毛到鼻尖

**代表**:
- 中年运势
- 事业发展
- 财富积累
- 个人能力

**吉凶判断**:
| 特征 | 含义 |
|------|------|
| 长而饱满 | 事业顺利、中年发达 |
| 鼻子高挺 | 财运好、有领导力 |
| 颧骨有肉 | 有权力、人缘好 |
| 过短或凹陷 | 中年运势波折 |

---

### 下停 - 晚年运势（51 岁+）

**位置**: 鼻尖到下巴

**代表**:
- 晚年运势
- 子女运
- 福气
- 稳定性

**吉凶判断**:
| 特征 | 含义 |
|------|------|
| 圆润饱满 | 晚年幸福、子女孝顺 |
| 下巴方圆 | 有福气、晚年稳定 |
| 过尖或后缩 | 晚年辛苦、子女缘薄 |
| 有双下巴 | 有福气、晚景好 |

---

## ⚖️ 三停比例分析

### 理想比例
**三停均等** - 1:1:1
- 一生运势平稳
- 各阶段发展均衡

### 常见比例

| 比例 | 特征 | 运势 |
|------|------|------|
| **上停长** | 额头高 | 早年顺利，智力高 |
| **中停长** | 鼻子长 | 中年发达，事业强 |
| **下停长** | 下巴长 | 晚年幸福，福气好 |
| **上停短** | 额头低 | 早年辛苦，需努力 |
| **中停短** | 鼻子短 | 中年平淡，守成为主 |
| **下停短** | 下巴短 | 晚年需规划，早做准备 |

## 👁️ 什么是五眼？

五眼是面部水平比例的测量标准。

```
    ← 1眼 →← 1眼 →← 1眼 →← 1眼 →← 1眼 →
    ════════════════════════════════════
    │      │      │      │      │      │
   太阳穴  眼尾   鼻梁   眼尾   太阳穴
```

### 标准五眼

1. **第一眼**: 左太阳穴到左眼外角
2. **第二眼**: 左眼宽度
3. **第三眼**: 两眼之间（鼻梁）
4. **第四眼**: 右眼宽度
5. **第五眼**: 右眼外角到右太阳穴

### 五眼分析

| 特征 | 含义 |
|------|------|
| 五眼均等 | 面部比例协调，运势平稳 |
| 眼距过宽 | 性格开放，但易轻信 |
| 眼距过窄 | 心思细腻，但易纠结 |
| 太阳穴凹陷 | 早年或晚年运势波动 |

## 🎯 实战应用

### 自我分析步骤

1. **照镜子** - 正面、光线充足
2. **观察三停** - 哪一停最长？哪一停最短？
3. **测量五眼** - 眼距是宽是窄？
4. **记录特征** - 在 progress.md 中记录

### 分析模板

```markdown
## 我的三停五眼分析

### 三停比例
- 上停：[长/中/短]
- 中停：[长/中/短]
- 下停：[长/中/短]
- 整体评价：

### 五眼比例
- 眼距：[宽/标准/窄]
- 太阳穴：[饱满/标准/凹陷]
- 整体评价：

### 综合解读
[根据上述特征进行解读]
```

## ⚠️ 注意事项

1. **综合判断** - 不可单凭三停五眼下结论
2. **动态观察** - 面相会随时间变化
3. **娱乐为主** - 不要过度迷信
4. **尊重差异** - 每个人都是独特的
