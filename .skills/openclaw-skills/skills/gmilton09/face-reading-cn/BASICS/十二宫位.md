## 📊 十二宫位详解

### 1️⃣ 命宫 - 核心运势

**位置**: 两眉之间（印堂）

**代表**: 整体运势、核心性格

| 特征 | 含义 |
|------|------|
| 饱满开阔 | 运势好、心胸开阔 |
| 狭窄凹陷 | 运势受阻、心思细腻 |
| 有痣或疤 | 人生有重大转折 |
| 红润有光 | 运势正旺 |

---

### 2️⃣ 兄弟宫 - 平辈关系

**位置**: 眉毛

**代表**: 兄弟姐妹、朋友、同事关系

| 特征 | 含义 |
|------|------|
| 眉毛浓密 | 兄弟朋友多、人缘好 |
| 眉毛稀疏 | 独立、朋友较少 |
| 眉形整齐 | 人际关系和谐 |
| 眉毛杂乱 | 人际关系复杂 |

---

### 3️⃣ 夫妻宫 - 婚姻感情

**位置**: 眼尾（奸门）

**代表**: 婚姻、感情、配偶

| 特征 | 含义 |
|------|------|
| 饱满光滑 | 婚姻幸福、配偶好 |
| 凹陷有纹 | 感情波折、婚姻不顺 |
| 有痣 | 感情有变数 |
| 鱼尾纹 | 感情丰富、人缘好 |

---

### 4️⃣ 子女宫 - 子女运势

**位置**: 眼下（泪堂）

**代表**: 子女、后代、创造力

| 特征 | 含义 |
|------|------|
| 饱满有肉 | 子女运好、有创造力 |
| 凹陷发黑 | 为子女操心、注意健康 |
| 有眼袋 | 子女缘深、但需注意健康 |
| 光滑平整 | 子女健康、发展好 |

---

### 5️⃣ 财帛宫 - 财富运势

**位置**: 鼻子

**代表**: 财富、收入、理财能力

| 特征 | 含义 |
|------|------|
| 鼻子高挺 | 财运好、有领导力 |
| 鼻头有肉 | 聚财、理财能力强 |
| 鼻翼饱满 | 守财、有积蓄 |
| 鼻子塌陷 | 财运一般、需努力 |

---

### 6️⃣ 疾厄宫 - 健康状况

**位置**: 鼻梁（年上、寿上）

**代表**: 健康、疾病、体质

| 特征 | 含义 |
|------|------|
| 鼻梁直 | 健康良好、体质强 |
| 鼻梁弯 | 注意脊椎、消化系统 |
| 有节或凸起 | 健康有波折 |
| 红润有光 | 气血充足、健康 |

---

### 7️⃣ 迁移宫 - 出行变动

**位置**: 额头两侧（天仓）

**代表**: 出行、搬迁、工作变动

| 特征 | 含义 |
|------|------|
| 饱满开阔 | 出行顺利、机遇多 |
| 凹陷 | 出行需谨慎、变动少 |
| 有痣 | 出行有变数 |
| 发暗 | 近期不宜远行 |

---

### 8️⃣ 奴仆宫 - 下属晚辈

**位置**: 下巴（地阁）

**代表**: 下属、晚辈、晚年运

| 特征 | 含义 |
|------|------|
| 方圆饱满 | 有领导力、晚年好 |
| 尖削 | 独立、晚年需规划 |
| 有双下巴 | 有福气、人缘好 |
| 后缩 | 晚年需注意健康 |

---

### 9️⃣ 官禄宫 - 事业运势

**位置**: 额头中央（司空、中正）

**代表**: 事业、职位、社会地位

| 特征 | 含义 |
|------|------|
| 高广饱满 | 事业顺利、有地位 |
| 低窄 | 事业需努力、大器晚成 |
| 有痣或疤 | 事业有波折 |
| 发际线整齐 | 事业发展稳定 |

---

### 🔟 田宅宫 - 房产家宅

**位置**: 眉眼之间（上眼睑）

**代表**: 房产、家宅、家庭

| 特征 | 含义 |
|------|------|
| 宽广 | 房产运好、家庭和睦 |
| 狭窄 | 房产运一般、需努力 |
| 有痣 | 房产有变数 |
| 下垂 | 家庭责任重 |

---

### 1️⃣1️⃣ 福德宫 - 福气品德

**位置**: 眉上方

**代表**: 福气、品德、精神生活

| 特征 | 含义 |
|------|------|
| 饱满 | 有福气、品德好 |
| 凹陷 | 需积德行善 |
| 有痣 | 福气有波折 |
| 光滑 | 精神生活丰富 |

---

### 1️⃣2️⃣ 父母宫 - 父母运势

**位置**: 额头两侧上方（日月角）

**代表**: 父母、长辈、祖业

| 特征 | 含义 |
|------|------|
| 饱满 | 父母健康、有助力 |
| 凹陷 | 父母需关心 |
| 有痣 | 父母健康需注意 |
| 发暗 | 长辈运势波动 |
