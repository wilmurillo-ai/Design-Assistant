## 🎯 使用技巧

### 观察要点

1. **光线充足** - 自然光下观察最佳
2. **素颜状态** - 不化妆时观察
3. **正面侧面** - 多角度观察
4. **动态气色** - 注意近期变化

### 分析原则

1. **综合判断** - 不可单特征论
2. **抓大放小** - 关注主要特征
3. **积极解读** - 多强调优势
4. **娱乐为主** - 不要过度迷信

### 记录建议

1. **拍照记录** - 记录分析时状态
2. **定期对比** - 观察变化
3. **验证反馈** - 记录准确性
4. **持续学习** - 不断改进
