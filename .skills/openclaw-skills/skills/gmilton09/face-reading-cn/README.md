## 📚 内容目录

### 1. 三停五眼 - 面部比例基础
**位置**: `BASICS/三停五眼.md`

**核心内容**:
- 上停：发际线到眉毛（早年运 15-30 岁）
- 中停：眉毛到鼻尖（中年运 31-50 岁）
- 下停：鼻尖到下巴（晚年运 51 岁+）
- 五眼：面部宽度的标准比例

**学习重点**: 理解面部比例与运势的关系

---

### 2. 十二宫位 - 面部区域划分
**位置**: `BASICS/十二宫位.md`

**核心内容**:
| 宫位 | 位置 | 代表 |
|------|------|------|
| 命宫 | 两眉之间 | 整体运势 |
| 兄弟宫 | 眉毛 | 兄弟姐妹、朋友 |
| 夫妻宫 | 眼尾 | 婚姻感情 |
| 子女宫 | 眼下 | 子女运 |
| 财帛宫 | 鼻子 | 财富 |
| 疾厄宫 | 鼻梁 | 健康 |
| 迁移宫 | 额头两侧 | 出行、变动 |
| 奴仆宫 | 下巴 | 下属、晚辈 |
| 官禄宫 | 额头中央 | 事业 |
| 田宅宫 | 眉眼之间 | 房产、家宅 |
| 福德宫 | 眉上方 | 福气、品德 |
| 父母宫 | 额头两侧上方 | 父母运 |

**学习重点**: 记住各宫位的位置和含义

---

### 3. 五行面相 - 金木水火土
**位置**: `BASICS/五行面相.md`

**核心内容**:

| 五行 | 脸型 | 特征 | 性格 |
|------|------|------|------|
| **金** | 方形 | 脸方、色白、骨突 | 果断、坚毅 |
| **木** | 长形 | 脸长、色青、身直 | 仁慈、上进 |
| **水** | 圆形 | 脸圆、色黑、肉多 | 智慧、灵活 |
| **火** | 尖形 | 脸尖、色红、发少 | 热情、急躁 |
| **土** | 厚重 | 脸厚、色黄、肉实 | 诚信、稳重 |

**学习重点**: 根据脸型判断五行属性

---

### 4. 气色理论 - 面色与运势
**位置**: `BASICS/气色理论.md`

**核心内容**:

| 气色 | 含义 | 运势 |
|------|------|------|
| **红润** | 气血充足 | 运势正旺 |
| **苍白** | 气血不足 | 运势低迷 |
| **发黄** | 脾胃问题 | 财运波动 |
| **发青** | 肝气郁结 | 情绪不稳 |
| **发黑** | 肾气不足 | 健康警示 |

**学习重点**: 观察面色变化，了解近期运势
