## 📖 技能说明

面相学是中国传统文化的重要组成部分，历史悠久，源远流长。本技能提供系统的面相学入门知识，帮助用户了解这一传统文化。

### 核心价值
- 📚 **文化传承** - 学习传统面相学知识
- 🔍 **快速查询** - 查询面部特征含义
- 🎯 **组合分析** - 多特征综合分析
- 🎭 **娱乐参考** - 趣味性的自我了解

## 🚀 使用方式

### 🆕 方式 1: 图像识别分析（v1.3.0 新增）

上传面部照片，AI自动分析面相：

```bash
# 综合分析（推荐）
python3 scripts/face_image_analysis.py "<图片URL>" combined

# 分析三停比例
python3 scripts/face_image_analysis.py "<图片URL>" three-stops

# 分析五眼比例
python3 scripts/face_image_analysis.py "<图片URL>" five-eyes

# 分析五官特征
python3 scripts/face_image_analysis.py "<图片URL>" features
```

**在对话中使用：**
- 发送图片并说"分析这张面相"
- "看看这个人的面相"
- "分析照片面相"

**图片要求：**
- 正面清晰照片效果最佳
- 图片需可公开访问（URL）
- 支持 JPG/PNG 格式
