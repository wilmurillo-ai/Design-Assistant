# 📉 损失厌恶 (Loss Aversion)

> "损失的痛苦是同等收益快乐的约 2-2.5 倍。" — 丹尼尔·卡尼曼

---

## 📖 什么是损失厌恶

**损失厌恶**是行为经济学的核心发现：人们对损失的敏感度远高于对收益的敏感度。

### 核心发现
| 情境 | 反应 |
|------|------|
| 获得 100 元 | 快乐程度: +1 |
| 损失 100 元 | 痛苦程度: -2 ~ -2.5 |

**关键洞察**: 损失的痛苦 ≈ 2-2.5 倍同等收益的快乐

---

## 🎯 核心概念

### 1. 禀赋效应
```
对自己拥有的东西估值更高
- 买入前: "这只值 100"
- 买入后: "这至少值 150"
```

### 2. 处置效应
```
过早卖出盈利资产
过晚卖出亏损资产
结果: 截断利润，让亏损奔跑
```

### 3. 现状偏见
```
倾向于维持当前状态
即使改变是理性的
```

### 4. 沉没成本谬误
```
已投入的成本不应影响未来决策
但人们常因"已经投入了"而继续错误
```

## 🔧 分析框架

### 投资决策检查清单

```markdown
## 损失厌恶自检

### 1. 持仓评估
□ 是否有亏损股票持有了超过 6 个月？
□ 是否有过盈利股票在小幅上涨后就卖出？
□ 是否经常"回本就想卖"？

### 2. 决策偏见
□ 是否对浮亏特别焦虑，但对浮盈没那么兴奋？
□ 是否更愿意持有亏损股票"等回本"？
□ 是否对"割肉"有强烈抗拒？

### 3. 锚定偏差
□ 是否总是以买入价作为决策参考？
□ 是否忽视当前基本面，只看盈亏状态？

### 4. 止损纪律
□ 是否有明确的止损规则？
□ 是否严格执行了止损？
□ 最近一次止损是什么时候？
```

## 📊 经典案例

### 案例 1：股票投资的处置效应

**场景**: 投资者持有两只股票
- 股票 A: 买入价 100，现价 120 (+20%)
- 股票 B: 买入价 100，现价 80 (-20%)

**损失厌恶行为**:
```
错误决策:
- 卖出 A "锁定利润" (太早)
- 持有 B "等回本" (太晚)

结果:
- A 后来涨到 150 (错失 30%)
- B 后来跌到 50 (扩大亏损)
```

**理性决策**:
```
只问: "现在持有 A/B，我会买入吗？"
- 不问历史成本
- 只看未来预期
```

### 案例 2：期货交易的止损困境

**场景**: 做多螺纹钢，入场 3800，止损设 3750

**损失厌恶心理**:
```
价格跌到 3755 时:
- "再等等，说不定会反弹"
- "马上就到支撑了"
- "亏 50 个点太心疼"

结果:
- 未执行止损
- 价格跌破 3700
- 亏损从 50 点扩大到 100+ 点
```

**反脆弱设计**:
```
- 预设止损，自动执行
- 将止损视为"保险费"
- 接受小损失是交易的一部分
```

### 案例 3：资产配置的再平衡困境

**场景**: 股债 60/40 配置，股市大跌后变成 40/60

**损失厌恶行为**:
```
"股票亏了这么多，现在减仓就是认亏"
→ 不执行再平衡
→ 风险敞口降低
→ 错过反弹
```

**理性行为**:
```
"低买高卖"机械执行
- 卖出债券 (高点)
- 买入股票 (低点)
- 恢复 60/40 配置
```

## ⚠️ 应对策略

### 1. 心理账户隔离
```
将投资视为"生意"
- 不问历史成本
- 只问当前价值
- 每笔决策独立
```

### 2. 自动化决策
```
- 预设止损/止盈
- 自动再平衡
- 减少人为干预
```

### 3. 重新锚定
```
不问: "我亏了多少？"
要问: "现在持有，未来预期如何？"

不问: "我成本是多少？"
要问: "当前价格是否合理？"
```

### 4. 损失预设
```
将损失纳入预期
- "每笔交易都可能亏损"
- "小损失是成功的一部分"
- "避免大损失才是目标"
```
