## [1.2.0] - 2026-03-14

### ✨ 新增（中期升级完成）

#### 模型推荐引擎
- 新增 `scripts/recommend.py` Python 脚本
- 基于关键词和场景智能推荐模型
- 支持命令行调用和 JSON 输出
- 包含 25 个模型的知识库
- 支持 11 个预定义场景匹配

**使用示例：**
```bash
python scripts/recommend.py "我要分析新能源汽车行业"
python scripts/recommend.py "如何评估这个投资机会"
python scripts/recommend.py --json "分析 Tesla 的竞争优势"
```

#### 实战模板生成
- 新增 `TEMPLATES.md` 文档
- 4 个完整分析模板：
  - **行业分析模板**: 波特五力 + PESTEL + 周期 + 临界质量
  - **投资决策模板**: 概率 + 贝叶斯 + 反脆弱 + 幂律
  - **竞争策略模板**: 波特五力 + 博弈论 + 比较优势 + 蓝海
  - **创业方向模板**: 蓝海 + 精益 + 网络 + 临界
- 每个模板包含完整的分析框架和填写指引
- 支持 AI 辅助填写

#### 学习进度追踪
- 新增 `progress.md` 文档
- 25 个模型的学习记录表
- 应用次数追踪
- 月度提醒系统
- 成就系统（学习/应用/深度三个维度）
- 学习心得汇总区

### 📝 改进

- 更新 SKILL.md 添加 3 种新使用方式
- 新增 3 个触发词：`生成模板 `、` 学习进度`
- 配套文档从 5 个增加到 9 个

### 📊 统计

- 思维模型总数：25 个（不变）

## [1.1.0] - 2026-03-14

### ✨ 新增

#### 快速查询功能
- 添加 `/mental-model` 命令式调用支持
- 新增 **快速查询卡片 (Quick Lookup)** 表格
  - 12 个常见场景的快速模型推荐
  - 包含问题类型、推荐模型、文档链接三列
  - 支持一键跳转到详细文档

#### 新增案例（4 个）
- **案例 9: OpenAI 的 AGI 战略**
  - 使用模型：第一性原理 + 长期主义 + 网络效应 + 临界质量
  - AI 时代的规模效应和网络效应分析

- **案例 10: 比特币周期投资**
  - 使用模型：周期理论 + 概率思维 + 反脆弱 + 幂律分布
  - 加密资产的周期性和反脆弱策略

- **案例 11: 宁德时代新能源战略**
  - 使用模型：蓝海战略 + 网络效应 + 波特五力 + 长期主义
  - 新能源行业的竞争分析和规模效应

- **案例 12: AI Agent 创业机会**
  - 使用模型：比较优势 + 机会成本 + 精益创业 + 蓝海战略
  - AI 时代的创业策略和垂直场景机会

#### 触发词扩展
- 新增触发词：`/mental-model`、`推荐模型`、`行业分析`、`投资决策`、`竞争策略`
- 现在用户可以通过更多方式触发技能
