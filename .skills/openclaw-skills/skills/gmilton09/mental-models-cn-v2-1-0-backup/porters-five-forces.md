# 🏭 波特五力分析

> 行业竞争结构的经典框架

---

## 📖 什么是波特五力

波特五力（Porter's Five Forces）是分析**行业竞争结构**的框架，由迈克尔·波特提出：

1. **现有竞争者的竞争程度**
2. **潜在进入者的威胁**
3. **替代品的威胁**
4. **供应商的议价能力**
5. **购买者的议价能力**

---

## 🎯 适用场景

- ✅ 行业吸引力评估
- ✅ 投资可行性分析
- ✅ 竞争战略制定
- ✅ 市场进入决策

---

## 🔧 分析框架

```
                    潜在进入者威胁
                         ↓
    供应商议价能力 → 现有竞争 ← 购买者议价能力
                         ↓
                    替代品威胁
```

### 1. 现有竞争者的竞争程度
```
考量因素：
• 竞争者数量与规模
• 行业增长率
• 产品差异化程度
• 退出壁垒
• 固定成本比例
```

### 2. 潜在进入者的威胁
```
考量因素：
• 进入壁垒 (资金/技术/法规)
• 规模经济
• 品牌忠诚度
• 转换成本
• 分销渠道获取难度
```

### 3. 替代品的威胁
```
考量因素：
• 替代品性能对比
• 替代品价格优势
• 客户转换意愿
• 替代品发展趋势
```

### 4. 供应商的议价能力
```
考量因素：
• 供应商集中度
• 供应商产品差异化
• 前向整合能力
• 转换成本
• 供应品重要性
```

### 5. 购买者的议价能力
```
考量因素：
• 购买者集中度
• 购买量占比
• 产品标准化程度
• 后向整合能力
• 价格敏感度
```

## 📊 输出示例

### 电动汽车行业五力分析（摘要）

| 力量 | 强度 | 关键因素 |
|------|------|----------|
| 现有竞争 | 🔴 高 | 特斯拉、比亚迪、传统车企涌入 |
| 潜在进入者 | 🟡 中 | 资金壁垒高，但科技公司跨界 |
| 替代品 | 🟢 低 | 燃油车逐步淘汰，公共交通有限替代 |
| 供应商 | 🟡 中 | 电池供应商集中，但多元化趋势 |
| 购买者 | 🟡 中 | 选择增多，但品牌忠诚度存在 |

**行业吸引力**：中等偏上（增长快但竞争激烈）

## ⚠️ 使用提示

1. **动态分析** - 五力会随时间变化，定期更新
2. **量化评分** - 尽量使用数据支撑判断
3. **结合其他模型** - 与 PESTEL、SWOT 配合使用
4. **细分行业** - 不同细分市场五力可能不同
