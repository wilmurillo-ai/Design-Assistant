# 🎯 思维模型组合包

**目的**: 针对常见场景预设模型组合，提升分析效率

---

## 📊 宏观分析包

### 适用场景
- 国家/地区经济分析
- 政策影响评估
- 市场进入决策

### 推荐组合（4 个模型）

| 模型 | 作用 | 使用顺序 |
|------|------|----------|
| **PESTEL 分析** | 宏观环境扫描 | 第 1 步 |
| **周期理论** | 判断周期位置 | 第 2 步 |
| **系统思维** | 理解相互作用 | 第 3 步 |
| **长期主义** | 长远视角判断 | 第 4 步 |

### 分析流程

```
1. PESTEL 扫描 → 识别关键因素
        ↓
2. 周期理论 → 判断当前阶段
        ↓
3. 系统思维 → 分析反馈回路
        ↓
4. 长期主义 → 制定长期策略
```

### 输出模板

```markdown
## 宏观分析结论

**周期位置**: 复苏/过热/滞胀/衰退
**关键驱动**: [PESTEL 中最重要的 2-3 个因素]
**系统风险**: [系统思维识别的风险点]
**长期趋势**: [长期主义视角的判断]
**行动建议**: [具体策略]
```

## 💰 投资决策包

### 适用场景
- 股票/基金投资
- 资产配置
- 风险评估

### 推荐组合（5 个模型）

| 模型 | 作用 | 使用顺序 |
|------|------|----------|
| **概率思维** | 期望值计算 | 第 1 步 |
| **贝叶斯定理** | 持续更新判断 | 第 2 步 |
| **反脆弱** | 风险收益设计 | 第 3 步 |
| **幂律分布** | 头部集中策略 | 第 4 步 |
| **机会成本** | 资源配置优化 | 第 5 步 |

### 分析流程

```
1. 概率思维 → 计算期望值
        ↓
2. 贝叶斯定理 → 根据新证据更新
        ↓
3. 反脆弱 → 设计风险收益结构
        ↓
4. 幂律分布 → 集中投资头部
        ↓
5. 机会成本 → 优化资产配置
```

### 输出模板

```markdown
## 投资决策结论

**期望值**: [概率×回报]
**成功概率**: [贝叶斯更新后]
**下行风险**: [最大损失]
**上行空间**: [最大收益]
**配置建议**: [仓位/分散度]
**止损条件**: [退出触发点]
```

## 🚀 创业方向包

- 市场进入策略

### 推荐组合（4 个模型）

|------|------|----------|

**价值创新**: [差异化 + 低成本]
**MVP 设计**: [最小可行产品]
**验证指标**: [关键指标]

## 🔍 风险评估包

### 适用场景
- 项目风险评估
- 投资决策风控
- 战略规划验证

### 推荐组合（4 个模型）

| 模型 | 作用 | 使用顺序 |
|------|------|----------|
| **逆向思维** | 反向思考风险 | 第 1 步 |
| **二阶效应** | 连锁反应推演 | 第 2 步 |
| **幸存者偏差** | 识别盲点 | 第 3 步 |
| **反脆弱** | 设计应对机制 | 第 4 步 |

### 分析流程

```
1. 逆向思维 → 思考失败原因
        ↓
2. 二阶效应 → 推演连锁反应
        ↓
3. 幸存者偏差 → 寻找沉默证据
        ↓
4. 反脆弱 → 设计应对方案
```

### 输出模板

```markdown
## 风险评估结论

**主要风险**: [3-5 个关键风险]
**触发条件**: [风险触发点]
**连锁反应**: [二阶/三阶效应]
**盲点识别**: [幸存者偏差]
**应对策略**: [对冲/规避/转移]
**监控指标**: [预警信号]
```
