# 🗺️ 思维模型选择流程图

**目的**: 根据问题类型快速选择合适的模型

---

## 🎯 快速选择树

```
开始：我要分析什么问题？
│
├─ 宏观环境/外部因素？
│   └─→ PESTEL 分析
│
├─ 行业竞争结构？
│   └─→ 波特五力
│
├─ 内外部综合分析？
│   └─→ SWOT 分析
│
├─ 复杂系统/相互关系？
│   └─→ 系统思维
│
├─ 需要创新/本质思考？
│   └─→ 第一性原理
│
├─ 问题拆解/结构化？
│   └─→ MECE 原则
│
├─ 决策/选择问题？
│   │
│   ├─ 不确定性高？
│   │   └─→ 概率思维 + 贝叶斯定理
│   │
│   ├─ 需要考虑后果？
│   │   └─→ 二阶效应
│   │
│   ├─ 简化选择？
│   │   └─→ 奥卡姆剃刀
│   │
│   ├─ 归因判断？
│   │   └─→ 汉隆剃刀
│   │
│   ├─ 反向思考？
│   │   └─→ 逆向思维
│   │
│   ├─ 策略互动？
│   │   └─→ 博弈论
│   │
│   ├─ 资源配置？
│   │   └─→ 机会成本 + 比较优势
│   │
│   └─ 创业/新产品？
│       └─→ 精益创业
│
├─ 趋势/周期判断？
│   │
│   ├─ 经济周期？
│   │   └─→ 周期理论
│   │
│   ├─ 增长拐点？
│   │   └─→ 临界质量
│   │
│   ├─ 长期趋势？
│   │   └─→ 长期主义
│   │
│   └─ 分布特征？
│       └─→ 幂律分布
│
├─ 风险/不确定性？
│   │
│   ├─ 风险评估？
│   │   └─→ 逆向思维 + 二阶效应
│   │
│   ├─ 盲点识别？
│   │   └─→ 幸存者偏差
│   │
│   └─ 风险收益设计？
│       └─→ 反脆弱
│
├─ 商业/市场策略？
│   │
│   ├─ 平台/社交产品？
│   │   └─→ 网络效应
│   │
│   ├─ 市场进入？
│   │   └─→ 蓝海战略
│   │
│   └─ 热点分析？
│       └─→ 热点分类
│
└─ 其他？
    └─→ 组合使用多个模型
```

## 🎯 单模型速查

### 何时使用这个模型？

| 模型 | 触发问题 | 典型场景 |
|------|----------|----------|
| **PESTEL** | "外部环境如何影响？" | 战略规划、市场进入 |
| **SWOT** | "我们的位置和方向？" | 竞争分析、自我评估 |
| **波特五力** | "行业吸引力如何？" | 投资分析、市场进入 |
| **MECE** | "分类是否完整独立？" | 问题拆解、报告撰写 |
| **系统思维** | "系统结构如何？" | 组织分析、生态分析 |
| **第一性原理** | "什么是基本事实？" | 创新、复杂问题 |
| **二阶效应** | "然后呢？" | 政策分析、决策评估 |
| **奥卡姆剃刀** | "这个必要吗？" | 假设验证、模型选择 |
| **汉隆剃刀** | "是否有更简单解释？" | 人际分析、组织行为 |
| **逆向思维** | "反过来会怎样？" | 决策、风险评估 |
| **博弈论** | "对方如何应对？" | 竞争策略、谈判 |
| **比较优势** | "我的相对优势？" | 职业选择、业务外包 |
| **机会成本** | "我放弃了什么？" | 资源配置、时间管理 |
| **精益创业** | "最小可行产品？" | 创业、新产品 |
| **周期理论** | "什么周期位置？" | 宏观经济、资产配置 |
| **幂律分布** | "分布类型？" | 投资、资源配置 |
| **反脆弱** | "如何从波动获益？" | 风险管理、系统设计 |
| **临界质量** | "临界点在哪？" | 平台冷启动、技术推广 |
| **长期主义** | "10 年后如何？" | 投资、职业规划 |
| **概率思维** | "概率和期望值？" | 不确定性决策 |
| **贝叶斯定理** | "新证据如何改变？" | 持续修正判断 |
| **幸存者偏差** | "失败者在哪？" | 数据分析、案例研究 |
| **网络效应** | "网络效应强度？" | 平台、社交产品 |
| **蓝海战略** | "如何无竞争？" | 市场进入、产品定位 |
| **热点分类** | "什么类型话题？" | 热点分析、内容分类 |

## 🔄 组合使用流程

### 标准分析流程

```
1. 问题定义
   ↓
2. 选择模型（参考本流程图）
   ↓
3. 收集信息
   ↓
4. 应用模型
   ↓
5. 得出结论
   ↓
6. 验证结论（可用其他模型交叉验证）
   ↓
7. 制定行动
```

### 交叉验证组合

| 主要模型 | 验证模型 | 目的 |
|----------|----------|------|
| SWOT | 波特五力 | 验证行业判断 |
| 第一性原理 | MECE | 验证拆解完整性 |
| 概率思维 | 贝叶斯定理 | 验证概率更新 |
| 蓝海战略 | 波特五力 | 验证竞争格局 |
| 逆向思维 | 二阶效应 | 验证风险推演 |
| 周期理论 | 长期主义 | 验证长期趋势 |

## 📱 快速决策卡片

### 30 秒决策流程

```
1. 这是什么类型的问题？（5 秒）
   ↓
2. 查表找到对应模型（10 秒）
   ↓
3. 应用核心问题（15 秒）
   ↓
4. 得出初步结论
```

### 5 分钟分析流程

```
1. 问题定义（1 分钟）
   ↓
2. 选择 2-3 个模型（1 分钟）
   ↓
3. 分别应用（2 分钟）
   ↓
4. 交叉验证（1 分钟）
   ↓
5. 得出结论
```
