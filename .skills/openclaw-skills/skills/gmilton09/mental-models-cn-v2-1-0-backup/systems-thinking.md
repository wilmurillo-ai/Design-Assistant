# 🕸️ 系统思维

> "系统思维是看见整体、看见相互关系、看见变化模式的修炼。" — 彼得·圣吉

---

## 📖 什么是系统思维

系统思维（Systems Thinking）是一种**整体性思考方式**，关注系统中各要素的相互关系和动态变化，而非孤立分析单个元素。

### 核心概念
- **系统** - 相互关联的元素组成的整体
- **反馈回路** - 输出影响输入的循环
- **延迟** - 原因与结果之间的时间差
- **涌现** - 整体具有部分没有的特性

## 🔧 核心工具

### 1. 因果回路图 (Causal Loop Diagram)
```
      ┌─────────────┐
      │   变量 A    │
      └──────┬──────┘
             │ (+)
             ↓
      ┌──────┴──────┐
      │   变量 B    │
      └──────┬──────┘
             │ (-)
             ↓
      ┌──────┴──────┐
      │   变量 C    │
      └─────────────┘
```

### 2. 反馈回路类型

**正反馈 (增强回路)**
```
人口增长 → 更多生育 → 人口进一步增长 → ...
```

**负反馈 (调节回路)**
```
体温升高 → 出汗散热 → 体温下降 → ...
```

### 3. 系统基模 (System Archetypes)
| 基模 | 特征 | 案例 |
|------|------|------|
| 增长上限 | 快速增长后停滞 | 创业公司扩张 |
| 舍本逐末 | 治标不治本 | 依赖止痛药 |
| 公地悲剧 | 个体理性导致集体非理性 | 过度捕捞 |
| 饮鸩止渴 | 短期缓解长期恶化 | 借债消费 |

## 📊 经典案例

### 案例：城市交通拥堵

```
         ┌─────────────────────────────────┐
         │                                 │
         │    (+)        (+)      (+)      │
    ┌────┴───┐      ┌────────┐  ┌────────┐ │
    │ 拥堵   │←─────│ 车辆数 │←─│ 购车意愿││

         │ (-)          │ (+)                │
```

**杠杆点分析**：
- 低杠杆：扩建道路（短期缓解，长期诱导更多需求）
- 中杠杆：提高停车费、拥堵费
- 高杠杆：改善公共交通 + 土地利用规划

## ⚠️ 分析要点

### 识别技巧
1. **寻找循环** - 因果不是线性的，是循环的
2. **识别延迟** - 结果可能很久后才显现
3. **关注关系** - 要素间的关系比要素本身重要
4. **寻找杠杆点** - 小干预产生大改变的位置

### 常见陷阱
| 陷阱 | 表现 | 避免方法 |
|------|------|----------|
| 线性思维 | A→B→C 单向思考 | 画因果回路图 |
| 忽视延迟 | 期望立竿见影 | 标注时间延迟 |
| 边界过窄 | 忽略外部因素 | 扩大系统边界 |
| 静态分析 | 忽略动态变化 | 考虑时间维度 |
