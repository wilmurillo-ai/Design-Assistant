### G2核心改进

**唯一改动**: 将量比>1.2从"加分项"升级为"入场硬门槛"

- 原版: 量比>1.2 → +10分
- G2: 量比<1.2 → 直接排除

这个简单改动将回测收益从16.6%提升到26.7%！

## 📊 v5.2评分体系

### 九维度综合评分

```
总分 = 位置(12%) + 量能(15%) + 趋势(10%) + 筹码(10%) + 
       业绩(12%) + 均线(8%) + 换手率(8%) + 情绪(5%) + 超跌反弹(20%)
```

### 各维度详解

1. **位置分析 (12%)**: 距60日均线涨跌幅，低位=安全边际
2. **量能分析 (15%)**: 量比反映资金入场程度
3. **趋势分析 (10%)**: MACD、KDJ等技术指标
4. **筹码分析 (10%)**: 底部缩量=主力建仓信号
5. **业绩分析 (12%)**: 净利润、营收增长
6. **均线分析 (8%)**: MA5>MA20>MA60多头排列
7. **换手率 (8%)**: 4%~12%为活跃区间
8. **情绪分析 (5%)**: 涨跌幅、市场氛围
9. **超跌反弹 (20%)**: 近期跌幅>30%为强势反弹信号

## ⚙️ 配置参数

### 选股条件

```json
{
  "min_score": 40,           // 评分≥40分
  "max_position_pct": 65,   // 位置<65%
  "vol_ratio_min": 1.2,     // 量比≥1.2 (G2核心)
  "change_pct_min": 2,      // 涨幅≥2%
  "change_pct_max": 6,      // 涨幅≤6%
  "price_above_ma5": true   // 价格>MA5
}
```

### 交易规则

```json
{
  "stop_loss_pct": 2.0,      // 止损-2%
  "take_profit_full_pct": 25.0  // 止盈+25%
}
```

### 仓位管理

```json
{
  "max_positions": 3,        // 最大3只持仓
  "position_size": 25        // 单只25%仓位
}
```

## 📅 每日交易流程

```
09:30-11:30  → 盘中监控（可选）
14:00        → 扫描全市场，计算v5.2评分
14:30        → 筛选符合条件股票
              ✓ 评分≥40
              ✓ 位置<65%
              ✓ 量比≥1.2  ← G2硬过滤
              ✓ 涨幅2%~6%
14:30-15:00 → 尾盘买入（使用收盘价）
              
持仓期间    → 监控止损/止盈
              亏损≥2% → 止损
              盈利≥25% → 止盈
```
