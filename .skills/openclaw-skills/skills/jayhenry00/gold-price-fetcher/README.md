# 金价查询工具 (Gold Price Fetcher)

通过京东金融 API 获取实时金价，返回带时间戳的完整信息。

## 功能特性

- ✅ 实时获取京东金融金价数据
- ✅ 自动缓存查询结果
- ✅ 返回带时间戳的格式化信息
- ✅ 支持金价变化对比

## 触发场景

当您需要查询实时金价时，可以说：
- "查询今日金价"
- "现在金价多少"
- "获取实时金价"
- "金价最新是多少"

## 输出格式

```
2026-04-04 19:14 金价：568.50 元/克
```

## 技术实现

### API 来源

- **接口**: 京东金融 - 民生积存金
- **URL**: https://ms.jr.jd.com/gw2/generic/CreatorSer/newh5/m/getFirstRelatedProductInfo
- **参数**:
  - circleId: 13245
  - invokeSource: 5
  - productId: 21001001000001

### 依赖

- Python 3.x
- requests 库

### 缓存机制

查询结果会缓存在 `cache/last_price.json` 文件中，包含：
- 金价数值
- 查询时间戳
- 格式化时间

可用于对比金价变化趋势。

## 手动使用

```bash
# 直接运行脚本获取金价
python scripts/fetch_gold_price.py
```

## 数据来源

数据来源于京东金融 - 民生积存金，仅供参考，实际交易价格以银行柜台为准。
