## 发布到 ClawHub

### 方式一：网页上传（推荐）

1. 访问：**https://clawhub.ai/import**

2. 填写表单：
   | 字段 | 值 |
   |------|-----|
   | **Slug** | `gold-price-fetcher` |
   | **Display name** | `金价查询工具` |
   | **Owner** | （留空，自动填充） |
   | **Version** | `1.0.0` |
   | **Tags** | `latest` |

3. 上传文件夹：
   - 点击 "Drop a folder" 或拖拽
   - 选择 `gold-price-fetcher-release` 文件夹

4. 填写 Changelog：
   ```
   Initial release: 支持京东金融实时金价查询，带缓存机制
   ```

5. 确认所有验证通过（绿色✅）

6. 点击 **"Publish skill"**

### 方式二：上传 ZIP 文件

如果网页支持 ZIP 上传，可以直接上传：
- `gold-price-fetcher-release.zip`
