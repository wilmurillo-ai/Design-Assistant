# jetson-info

NVIDIA Jetson 设备信息查询工具。

## 功能

- 设备型号识别
- JetPack 版本
- L4T 版本
- CUDA/cuDNN/TensorRT 版本
- 序列号查询
- 硬件规格

## 使用方法

```bash
# 查看完整设备信息
jetson-info

# 查看简略信息
jetson_release
```

## 依赖

- jetson_release (jetson-stats)

## 环境要求

- NVIDIA Jetson AGX Orin
- NVIDIA Jetson Xavier
- NVIDIA Jetson Nano/TX2
