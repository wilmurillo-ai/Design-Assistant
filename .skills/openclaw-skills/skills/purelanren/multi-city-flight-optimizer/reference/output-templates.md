## 可玩性建议模板

```
🗺️ 可玩性建议
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

如果选择 方案1（去巴塞罗那 → 回里斯本）：

  推荐路线：巴塞罗那(2天) → 马德里(2天) → 里斯本(2天)
  
  城际交通：
  · 巴塞罗那 → 马德里：高铁AVE 约2.5h ≈ ¥300-500/人
  · 马德里 → 里斯本：夜间大巴 约9h ≈ ¥200/人（省一晚住宿）
    或飞机 约1h ≈ ¥300-600/人
  
  💡 这条路线从东向西一路不走回头路，城际交通最省时。


如果选择 方案2（去马德里 → 回马德里）：

  推荐路线：马德里(1天) → 巴塞罗那(2天) → 马德里(1天) → 里斯本...
  
  ⚠️ 需要折返回马德里坐回程，会多一段城际交通。
  建议改为"去马德里→回里斯本"可省一段折返。


如果选择 方案3（去巴塞罗那 → 回巴塞罗那）：

  推荐路线：巴塞罗那(1天) → 马德里(2天) → 里斯本(2天) → 巴塞罗那(回程)
  
  ⚠️ 里斯本→巴塞罗那需额外飞回或坐长途大巴。
  里斯本→巴塞罗那飞约2h ≈ ¥400-800/人

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 特殊标注模板

### 红眼航班标注

```
🌙 提醒：去程为红眼航班
   出发时间 23:30，到达次日 06:15
   建议到达当天安排轻松行程（不要赶景点）
```

### 超长中转标注

```
⚠️ 提醒：回程在[中转城市]需中转 [X]小时
   中转时间较长，建议：
   · 提前确认是否需要过境签
   · 利用中转时间在机场休息/逛免税店
   · 如超过8小时，考虑是否接受
```

### 价格波动提示

```
💡 价格提醒：
   以上价格为当前实时搜索结果，国际机票价格波动较大。
   如果对方案满意，建议尽快预订锁定价格。
   通常提前30-60天预订长途机票性价比最高。
```

### 签证提醒

```
🛂 签证提醒：
   本行程涉及 [国家1]、[国家2]、[国家3]
   · 西班牙、葡萄牙均为申根区，持一个申根签证即可通行
   · 建议向停留时间最长的国家领事馆申请签证
   · 申根签证通常需提前15-30个工作日申请
```
