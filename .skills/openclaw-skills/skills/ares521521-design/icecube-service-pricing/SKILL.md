## 三、收款流程

### 标准流程

```
需求确认 → 报价发送 → 预付款 50% → 开发执行 → 交付验收 → 尾款 50%
```

### 收款方式

**微信支付：**
- 优点：即时到账，无手续费
- 缺点：单笔限额
- 适合：¥5000 以下

**支付宝：**
- 优点：即时到账，可开发票
- 缺点：需实名认证
- 适合：所有金额

**银行转账：**
- 优点：大额无限制
- 缺点：需 1-2 天到账
- 适合：企业客户，¥5000 以上

## 五、风险控制

### 预付款保护

- 必须收取预付款（至少 30%）
- 预付款确认后开始工作
- 预付款不退（已开始工作）

### 交付验收

- 明确验收标准
- 验收期限：交付后 3 天内
- 超期未验收视为验收通过

### 尾款收取

- 验收通过后 3 天内支付尾款
- 超期未付暂停支持
- 提供催款话术模板
