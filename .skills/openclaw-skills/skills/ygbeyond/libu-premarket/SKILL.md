## ⚙️ 进阶配置

编辑 `config.json` 可调整选股标准：

```json
{
  "filter_financial": {
    "min_netprofit_yoy": 0.30,   // 净利润增速 ≥30%
    "min_roe": 8                  // ROE ≥8%
  },
  "cache": {
    "root_dir": "./cache_data"
  }
}
```
