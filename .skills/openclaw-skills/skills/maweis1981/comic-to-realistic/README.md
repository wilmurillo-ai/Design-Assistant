# Comic to Realistic

Transform anime or comic-style images into realistic human portraits using AI.

## ✨ Features

- Convert 2D anime to realistic human faces
- Prompt-controlled output style
- High-quality image generation
- Async workflow execution

## 🚀 How It Works

1. Upload an anime/comic image URL
2. Provide a prompt describing the desired realism
3. AI enhances the prompt
4. Image is converted to realistic style
