## Usage Guidelines

### 1. Fill All Sections
- Every section should have content
- Mark sections as "N/A" if truly not applicable (with reason)

### 2. Be Specific
- Use names, dates, numbers
- Avoid vague statements

### 3. Actionable Items
- Every action must have:
  - Clear owner
  - Specific deadline
  - Priority level

### 4. Save Files
Save to `workspace/deep-discussion/{topic-slug}/`:
- `agenda.md` - 议程清单 ⭐ 必须
- `discussion-log.md` - 完整讨论记录 ⭐ 必须
- `report.md` - 结构化报告 ⭐ 必须
- `action-plan.md` - 行动计划 ⭐ 必须

**⚠️ Do NOT create `discussion.md` - it duplicates `discussion-log.md`!**
