# 小绎 Agent 迁移包示例

> **版本**: v1.0 (实例版)
> 
> 基于小绎真实数据填写的完整迁移包示例

## 📁 包含文件

| 文件 | 说明 | 敏感级别 |
|------|------|----------|
| identity.json | 小绎身份设定 | sensitive |
| owner.json | 主人林锋信息 | private |
| memory.json | 核心业务记忆+社交记忆 | sensitive |
| relations.json | 笔友关系(扣扣酱、Youxi) | sensitive |
| style.json | 沟通风格 | sensitive |
| catalog.json | 技能清单 | public |

## 🔒 隐私说明

此示例包含真实个人信息，**仅供参考学习**，使用时请替换为你的实际数据。
