## 常见问题预案

### 笔记发出去没人看
→ 检查封面图是否吸引、标题是否有钩子、发布时间是否正确
→ 可以适当投放薯条（100元测试）
→ 多看爆款笔记，学习开头写法

### 有人问很详细的专业问题
→ 引导到工具：「你把面积告诉我，我帮你算一下」
→ 或者：「这个需要现场勘测，建议咨询专业装修公司」

### 有人质疑数据准确性
→ 承认区间范围，说明：「数据来自广材网，仅供参考」
→ 引导：「这个工具帮你建立预期，具体还要看实际情况」

### 有人问能不能帮忙装修
→ 说明定位：「我只做预算估算，不做装修施工」
→ 推荐靠谱渠道：「可以看看土巴兔/齐家网的案例」

### 有人想要免费
→ 可以偶尔做活动：「评论区抽3个免费名额」
→ 或者引导：「转发朋友圈可以获赠一次估算」
