# 真实案例场景

## 案例1：刚需上车族

### 用户画像
```
姓名：张先生
年龄：28岁
职业：互联网公司产品经理
城市：上海
收入：月薪1.5万
家庭：单身，和父母同住

购房情况：外环外小两居，70㎡，准备结婚用
装修预算：有限，希望控制在20万以内
核心诉求：功能齐全、不超预算、质量过关
```

### 使用场景
```
张先生买了人生第一套房，想装修成婚房。
对装修完全不懂，去了两家装修公司，
第一家报价18万，第二家报价22万，
差距这么大，不知道该信谁。

他刷小红书时看到我们的工具，
试着输入：70㎡ + 精装 + 上海

结果出来：
• 经济型：14.2万
• 舒适型：17.8万
• 品质型：21.7万

他对比了两家装修公司的报价，
发现第一家18万在品质型区间，
但清单里有些项目没写清楚。

他拿着这份估算报告去找装修公司谈，
成功压价到16万含清包工。

最后实际花费：16.5万，在舒适型区间内。
```

### 用户的原话
> "知道市场价是多少之后，谈判底气都足了。"

## 案例2：改善型换房

### 用户画像
```
姓名：李女士
年龄：35岁
职业：外企财务经理
城市：杭州
收入：家庭年收入50万
家庭：一家三口，有个5岁孩子

购房情况：140㎡大三居，改善型住房
装修预算：希望控制在60-80万
核心诉求：环保、安全、收纳做足
```

### 使用场景
```
李女士准备装修第二套房子，这次是给全家人住。
她很谨慎，看了5家装修公司，
报价从55万到85万不等。

她用工具算了140㎡ + 豪装 + 杭州

结果出来：
• 经济型：68万
• 舒适型：94.5万
• 品质型：121万

她发现：
1. 报价55万的那家，明显偏低，可能有漏项
2. 报价85万的，有几个项目可以优化
3. 她的预算在舒适型区间内

最后她选择了报价72万的那家公司，
用了工具帮他核查报价单，
发现了两处漏项和一处重复计价。

最终花费：73万，在豪装经济型区间内。
```

### 用户的原话
> "终于知道钱花哪儿去了，连老公都夸我会买东西。"

## 案例3：投资型装修

### 用户画像
```
姓名：王先生
年龄：42岁
职业：投资人
城市：成都
收入：主要靠投资收入

购房情况：买了两套小户型（各50㎡）
装修目的：出租 + 以后给父母住
装修预算：越便宜越好，但不能太掉价

核心诉求：控制成本、租相好、好出租
```

### 使用场景
```
王先生买了两套小户型准备出租。
他找了装修公司，第一套报价8万，
他想控制在7万以内，但不知道怎么谈。

他用工具算了：50㎡ + 简装 + 成都

结果出来：
• 经济型：6.8万
• 舒适型：8.5万
• 品质型：10.3万

他拿着经济型的结果去谈，
成功把第一套装修压到6.5万。

第二套他直接用工具自己核查报价单，
发现装修公司漏报了拆除费用，
提前沟通避免了增项。

两套实际花费：
第一套：6.5万（比预算省了5000）
第二套：7.8万（装修公司首次报价9万）

总计节省：3.2万
```

### 用户的原话
> "省下来的钱都够我再投一套的首付了。"
