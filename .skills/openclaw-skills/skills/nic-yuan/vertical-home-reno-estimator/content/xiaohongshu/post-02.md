## 笔记正文

```
【标题】
问完这10个问题，装修公司不敢乱报价｜内部资料泄露

【正文】

做了X年造价师，
见过的坑比你吃过的盐还多。

今天把装修公司最怕被问的10个问题整理出来，
看完再去谈，至少省2万。

---

❓ 问题1：水电改造怎么算钱？
→ 按实际米数？点位？面积？
→ 装修公司：按米算，每米XX元
→ 你应该问：总价上限是多少？超出怎么算？
⚠️ 水电是增项重灾区！

❓ 问题2：哪些项目不在报价里？
→ 垃圾清运费、搬运费、成品保护费...
→ 装修公司：这些小钱到时候再说
→ 你应该问：清单里有没有包含所有项目？
⚠️ 这些"小钱"加起来可能5000+

❓ 问题3：材料品牌和规格是什么？
→ "装修用好材料"
→ 你应该问：具体品牌+型号写清楚
⚠️ 口说无凭，必须写在合同里

❓ 问题4：能看看你们在建工地吗？
→ 装修公司：样板间都是最好的
→ 你应该问：看正在施工的，不是样板间
⚠️ 真实工地才能看出工艺

❓ 问题5：工期延误怎么算违约金？
→ "放心，我们从不延误"
→ 你应该问：每天违约金多少？写进合同
⚠️ 没有违约金，工期就是没有deadline

❓ 问题6：售后保修多久？
→ "终身维修"
→ 你应该问：具体哪些项目保修？超出保修期怎么收费？
⚠️ 口头承诺等于没有承诺

❓ 问题7：增项超过多少需要我同意？
→ "不超过总价的5%"
→ 你应该问：所有增项必须书面确认，总价浮动不超过X%
⚠️ 增项不确认是最大的坑

❓ 问题8：设计师出了几套方案？
→ "一套就够"
→ 你应该问：能出2-3套方案对比吗？
⚠️ 设计是装修的灵魂，多方案才有选择

❓ 问题9：付款方式是怎样的？
→ "3331" or "先付60%"
→ 你应该问：尾款留多少？验收完才付清
⚠️ 钱付太快，主动权就没了

❓ 问题10：这份报价的有效期是多久？
→ "最近材料都在涨"
→ 你应该问：30天内不涨价的承诺写进合同
⚠️ 价格有效期很重要
```
