## 内容生产模板库

### 模板A：数字型爆款
```
公式：XX㎡装修要花多少钱？| XX城市真实案例

结构：
1. 开头：引发共鸣（我之前也纠结过）
2. 数据：直接给数字（总价+分项）
3. 细节：重点说1-2个踩坑点
4. 结尾：引导评论（你家花了多少？）

标签：#装修费用 #城市名装修 #装修日记
```

### 模板B：避坑型爆款
```
公式：装修公司绝对不会告诉你的X个秘密

结构：
1. 开头：行业背景（我做造价X年了）
2. 列表：X个避坑点（每个200字）
3. 技巧：给出具体操作建议
4. 结尾：引导工具（想知道你家要花多少？）

标签：#装修避坑 #装修公司 #装修干货
```

### 模板C：对比型爆款
```
公式：XX vs XX，装修选哪个更划算？

结构：
1. 开头：抛出选择题（简装vs精装选哪个）
2. 对比：列出各维度对比表
3. 结论：给建议（什么情况下选哪个）
4. 结尾：引导使用工具

标签：#装修预算 #装修选材 #装修对比
```

### 模板D：故事型爆款
```
公式：我是如何用AI省了X万装修费的

结构：
1. 背景：我的情况（面积/城市/预算）
2. 冲突：装修公司的报价vs我的预期
3. 转折：用了AI工具后发现...
4. 结果：成功省了X万
5. 结尾：工具使用方式

标签：#装修日记 #装修省钱 #AI工具
```

## 批量内容生产计划

### 批量生产流程（每月一次）

```
Step 1: 收集素材（2小时）
- 翻看用户评论，收集常见问题
- 翻看竞品爆款笔记，找灵感
- 整理本月行业热点

Step 2: 规划日历（1小时）
- 制定下月内容日历
- 确定每周主题
- 分配发布时间

Step 3: 批量写作（4小时）
- 集中写完一个月的笔记草稿
- 预留填空位置（城市/面积等）
- 保存到内容库

Step 4: 素材准备（2小时）
- 批量制作封面图（用模板）
- 批量准备配图
- 预写评论区话术

Step 5: 定时发布
- 使用小红书草稿箱定时功能
- 提前1天准备好所有内容
```

## 内容效率工具

### 效率工具推荐
```
• 写作：Notion（内容管理）
• 封面图：Canva（免费模板）
• 视频：剪映（手机剪辑）
• 排期：飞书文档（内容日历）
• 数据：千瓜数据（小红书分析）
```

### AI辅助写作
```
用 AI 帮助的环节：
• 生成标题备选（让AI给10个标题）
• 润色文案（让AI优化表达）
• 生成标签（让AI给相关标签）
• 翻译英文（发布到英文平台）

不需要AI的环节：
• 真实案例故事（必须自己写）
• 个人经验分享（必须真实）
• 情感共鸣内容（必须真诚）
```

## 内容质量标准

### 每篇笔记必须包含
```
□ 封面图：吸引眼球，清晰表达主题
□ 标题：30字以内，含数字/痛点词
□ 开头：3行内引发共鸣
□ 主体：信息密度高，干货充足
□ 结尾：有互动引导（评论/关注）
□ 标签：8-15个相关标签
```

### 每篇笔记禁止
```
□ 虚假数据（会被扒）
□ 过度承诺（"保证省XX万"）
□ 拉踩竞品（容易招黑）
□ 敏感词汇（平台违规词）
□ 水印太多（影响美观）
```
