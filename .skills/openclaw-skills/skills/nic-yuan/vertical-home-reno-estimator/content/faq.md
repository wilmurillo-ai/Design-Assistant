# FAQ 问题库

## 用户常见问题

### Q1：装修到底要花多少钱？
```
A：装修费用因城市、面积、档次差异很大。

简单参考：
• 简装：1000-1600元/㎡
• 精装：2200-3600元/㎡
• 豪装：5000-9000元/㎡

100㎡ 精装 пример：
• 北京/上海：35-50万
• 成都/杭州：25-35万
• 三四线城市：18-28万

想知道精确数字？评论区告诉我你的情况！
```

### Q2：装修公司报价多少合适？
```
A：没有统一答案，取决于你的需求。

判断报价是否合理：
1. 用工具算出市场参考价
2. 对比装修公司报价
3. 差距超过20%要谨慎
4. 差距超过30%建议换一家

注意：报价太低可能有漏项，报价太高可能宰客。
```

### Q3：半包还是全包好？
```
A：看情况。

半包（包工包辅料）：
✅ 业主：买主材（地板/瓷砖/橱柜）
✅ 优点：主材自己掌控
❌ 缺点：需要时间精力
适合：有时间、懂材料的人

全包（包工包全部）：
✅ 业主：什么都不用管
✅ 优点：省心
❌ 缺点：价格高，可能以次充好
适合：没时间、预算充足的人

建议：第一次装修选半包，有经验后可选全包。
```

### Q4：装修要准备多少钱？
```
A：通常建议准备预算的110-115%

预算分配建议：
• 硬装：50-55%
• 软装：25-30%
• 电器：15-20%
• 弹性空间：10-15%

重要：签订闭口合同，避免增项超支。
```

### Q5：如何判断装修公司靠不靠谱？
```
A：看这5点：

1. 口碑：问身边装修过的朋友
2. 工地：去他们在建工地看
3. 报价：清单详细，不漏项
4. 合同：闭口合同，不是开口合同
5. 付款：尾款留20-30%，验收完才付清

避坑口诀：先看工地再签约，闭口合同最保险。
```

### Q6：什么时候开始买材料？
```
A：时间线参考：

装修前1-2个月：
□ 确定装修风格
□ 确定橱柜方案
□ 确定全屋定制方案

装修前1个月：
□ 购买瓷砖
□ 购买地板
□ 购买门窗

装修前2周：
□ 购买洁具
□ 购买灯具
□ 购买开关插座

提示：材料到场要验收，防止货不对板。
```

### Q7：装修一般要多久？
```
A：100㎡左右的新房装修：

拆除（如果有）：3-7天
水电改造：7-14天
泥瓦工程：15-25天
木工工程：10-20天
油漆工程：15-20天
安装工程：10-15天
通风散味：1-2个月

总计：约3-4个月

老房翻新：再加1-2个月（拆除量大）
```

### Q8：如何验收水电改造？
```
A：重点验收这5项：

1. 水压测试
   → 关闭总阀，加压0.8MPa
   → 30分钟后压降不超过0.05MPa合格

2. 电路测试
   → 每个插座用相位仪测试
   → 确保左零右火上接地

3. 水路走向
   → 拍照留存，后期安装避免打到水管

4. 开关位置
   → 确认高度、位置符合需求

5. 防水测试
   → 卫生间蓄水48小时
   → 楼下检查是否渗漏

隐蔽工程一定要验收，不然后期返修成本极高！
```

### Q9：装修中有哪些坑要避开？
```
A：Top 10 避坑清单：

1. 水电改造增项超支
2. 防水没做好导致漏水
3. 瓷砖铺贴不平整
4. 木工偷工减料
5. 油漆色差大
6. 定制柜体尺寸出错
7. 材料被掉包
8. 工期无限拖延
9. 售后服务推诿
10. 合同条款有陷阱

建议：每个节点验收，签字确认后再付款。
```

### Q10：家具什么时候买？
```
A：建议分批购买：

硬装阶段：
□ 橱柜（装修前1个月下单）

木工阶段：
□ 门（测量后30-45天交货）

油漆完成后：
□ 地板
□ 踢脚线
□ 窗台石

通风期间：
□ 床垫/床架
□ 沙发
□ 餐桌椅

通风散味后（入住前1周）：
□ 窗帘
□ 绿植（除甲醛）
□ 小家电
□ 贴身日用品

注意：甲醛检测合格后再入住！
```

## 客服话术

### 主动引导话术
```
• "帮你算了一下，你家装修大约在XX-XX万之间"
• "这个价格在合理范围内，比报价单上那家便宜了XX万"
• "建议你拿着这个报告去和装修公司谈，更有底气"
• "完整报告（含详细分项）只需19.9，需要吗？"
```

### 应对价格争议
```
• "估算基于市场平均价格，实际会有10-15%浮动"
• "不同装修公司报价差异较大，建议多对比几家"
• "材料品牌、工艺要求不同会导致价格差"
```

### 应对专业问题
```
• "这个问题需要现场勘测，建议找专业装修公司"
• "我可以帮你估算总价，具体方案需要设计师出图"
• "隐蔽工程建议找专业人士验收，避免后期隐患"
```
