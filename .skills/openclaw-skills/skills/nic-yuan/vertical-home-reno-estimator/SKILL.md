# 住宅装修造价估算工具

快速生成住宅装修造价估算报告，帮助业主了解装修成本区间。

## 工作流程

装修造价估算分为以下步骤：

1. **收集基础信息**（必须）
2. **估算计算**（调用脚本）
3. **生成报告**（按模板输出）

### Step 1: 收集基础信息

必须获取以下三项信息：

| 项目 | 说明 | 有效值 |
|------|------|--------|
| 房屋面积 | 套内建筑面积 | 30-500 ㎡ |
| 装修档次 | 决定单位面积造价 | 简装 / 精装 / 豪装 |
| 所在城市 | 影响人工费系数 | 中国城市名 |

**如果用户没有提供完整信息**，按以下优先级处理：
1. 从上下文推断（如"北京的房子"）
2. 提示用户补充

**档次判断参考**：
- 简装：基础功能性装修，出租或过渡性居住
- 精装：自住标准，中等品牌材料
- 豪装：高端品牌，定制化需求多

### Step 2: 估算计算

调用估算脚本：

```bash
python scripts/estimate.py <面积> <档次> <城市>
```

示例：
```bash
python scripts/estimate.py 100 精装 杭州
```

脚本返回 JSON 格式数据，包含：
- 总价估算（经济型/舒适型/品质型三档）
- 分项估算（硬装/软装/电器）
- 硬装明细（水电/泥瓦/木工/油漆/主材）

### Step 3: 生成报告

按照以下模板输出完整报告：

## ⚠️ 不确定项提示

以下项目可能导致实际费用偏差较大：

1. **水电改造范围**
   - 老房翻新可能比新房高 50%-100%
   - 异形空间、多卫生间会增加成本

2. **定制柜体**
   - 橱柜、衣柜按投影面积计价
   - 五金件品牌差异大（进口 vs 国产）

3. **隐蔽工程**
   - 防水层、隔音处理等不可预见项
   - 建议预留 5%-10% 预算

4. **设计变更**
   - 施工中的方案调整会增加费用
   - 签订闭口合同可控制风险

### 处理边界情况

**情况1：用户问局部装修**
> "本工具适用于整装估算。局部装修（如仅做厨房、卫生间）建议咨询专业装修公司，因为涉及拆旧、成品保护等额外费用。"

**情况2：用户问商业空间**
> "本工具仅适用于住宅装修。商业空间（办公、餐饮、零售）有不同定额标准，建议咨询专业造价公司。"

**情况3：用户要精确到品牌**
