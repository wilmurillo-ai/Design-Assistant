## 🚀 产品演示

### 输入

```
面积：100 ㎡
档次：精装
城市：杭州
```

### 输出

```
经济型：25.0 万
舒适型：33.4 万
品质型：41.7 万

硬装 52% = 17.3 万
├── 水电改造 3.0 万（18%）
├── 泥瓦工程 3.9 万（22%）
├── 木工工程 3.0 万（18%）
├── 油漆工程 2.2 万（12%）
└── 主材费用 5.2 万（30%）

软装 27% = 8.0 万
电器 21% = 6.0 万
```

### 运行方式

```bash
python scripts/estimate.py 100 精装 杭州
```
