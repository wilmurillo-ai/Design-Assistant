## 上架步骤

### Step 1: 注册 ClawHub 账号
1. 访问 https://clawhub.ai
2. 使用 GitHub 账号登录
3. 完成开发者认证（如需要）

### Step 2: 创建 Skill 页面
1. 点击 "Publish Skill"
2. 填写以下信息：

| 字段 | 说明 | 示例 |
|------|------|------|
| Name | Skill 名称 | home-reno-estimator |
| Description | 简短描述 | 住宅装修造价估算工具 |
| Category | 分类 | productivity/business |
| Tags | 标签 | 装修,造价,估算,预算 |
| Price | 价格 | ¥19.9 或 Free |

### Step 3: 上传文件
1. 打包 .skill 文件（运行 package.py）
2. 或直接上传 skill 目录

### Step 4: 填写详细说明
在 README 或 Description 中包含：
- 功能介绍
- 使用方法
- 价格说明
- 数据来源
- 免责声明

### Step 5: 提交审核
1. 检查所有信息
2. 点击发布
3. 等待审核（通常 1-3 天）

## ClawHub 定价策略建议

### 方案A：免费引流
- 优点：快速获取用户，积累口碑
- 缺点：变现路径长
- 适合：MVP 阶段验证

### 方案B：付费下载
- 价格：¥9.9-29.9
- 优点：直接变现
- 缺点：转化率可能较低
- 适合：有明确价值主张的产品

### 方案C： Freemium
- 免费版：基础估算（3次/天）
- 付费版：无限次 + 详细报告
- 价格：¥49/年
- 优点：平衡用户体验和变现
- 缺点：需要开发两套逻辑

## 常见问题

### Q: 上架需要付费吗？
A: ClawHub 目前免费上架，分成比例 70%（开发者）/ 30%（平台）

### Q: 如何设置价格？
A: 上架页面有价格设置选项，支持一次性买断或免费

### Q: 收入如何结算？
A: 通常按月结算，需绑定支付账户（支付宝/微信）

### Q: 如何更新 Skill？
A: 在 ClawHub 后台上传新版本 .skill 文件即可

### Q: 审核需要多久？
A: 通常 1-3 个工作日，首次上架可能稍长
