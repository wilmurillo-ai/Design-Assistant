# Skill 开发笔记 - home-reno-estimator

## 项目概述

**名称**: home-reno-estimator (住宅装修造价估算器)

**定位**: 面向小白的住宅装修费用估算工具

**核心价值**: 
- 权威数据来源（广材网、造价通）
- 快速估算（3个参数：面积、档次、城市）
- 专业报告格式


## 数据来源

1. **广材网 (gldjc.com)** - 广联达旗下，市场价+信息价
2. **造价通 (zjtcn.com)** - 工程造价信息网
3. **各省市造价信息网** - 政府指导价

## 核心算法

```
总价 = 单位面积造价 × 面积 × 城市系数
```

- 单位面积造价：根据档次（简装/精装/豪装）查表
- 城市系数：一线城市1.4，新一线1.15，二线1.0，三四线0.85
