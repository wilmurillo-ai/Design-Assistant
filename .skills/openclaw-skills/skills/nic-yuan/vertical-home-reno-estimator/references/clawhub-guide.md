# ClawHub 上架详细指南

## ClawHub 是什么

ClawHub（clawhub.ai）是 OpenClaw 的官方 Skill 分发平台，类似于 npmjs.com 或者 GitHub Marketplace。

**为什么先上架 ClawHub：**
- ✅ 平台负责支付处理（不用自己对接微信/支付宝）
- ✅ 减少前期运营麻烦
- ✅ 可以积累下载量和评分
- ✅ 70% 收入归开发者（平台30%）

---

## Step by Step 上架流程

### Step 1: 访问 ClawHub

```
1. 打开浏览器
2. 访问 https://clawhub.ai
3. 点击右上角 "Sign in"
```

### Step 2: 登录账号

```
推荐：用 GitHub 账号登录
1. 点击 "Sign in with GitHub"
2. 授权 GitHub 账号
3. 完成登录

备选：用邮箱注册（如果没有 GitHub）
```

### Step 3: 找到发布入口

```
1. 登录后在顶部导航栏
2. 找到 "Publish Skill" 或 "发布 Skill" 链接
3. 点击进入发布页面
```

### Step 4: 填写 Skill 信息

#### 基本信息

| 字段 | 填写内容 | 示例 |
|------|----------|------|
| Name | Skill 名称（英文，字母+数字） | home-reno-estimator |
| Display Name | 显示名称（中文） | 住宅装修造价估算器 |
| Description | 简短描述（1-2句话） | 输入面积、城市、档次，3秒出专业装修造价报告 |
| Category | 分类 | productivity / business / tools |
| Tags | 标签（用逗号分隔） | 装修,造价,估算,预算,工程造价 |

#### 详细描述

在详细描述框中填写：

```
## 功能特点

- 3秒出专业装修造价报告
- 基于广材网、造价通权威数据
- 支持300+城市
- 三档价格可选（经济型/舒适型/品质型）
- 分项费用拆解

## 使用方法

1. 输入房屋面积（如：100）
2. 选择装修档次（简装/精装/豪装）
3. 选择城市
4. AI自动生成专业报告

## 数据来源

- 广材网(gldjc.com) - 市场价数据
- 造价通(zjtcn.com) - 工程造价信息
- 各省市造价信息网 - 政府指导价

## 适用人群

- 准备装修的业主
- 需要快速估算装修成本的从业者
- 希望了解装修市场价的小白
```

#### 定价设置

```
选项A：免费
→ 快速积累用户和评分
→ 适合 MVP 验证阶段

选项B：付费
→ Price: ¥19.9 或 Free
→ 建议第一版设为 Free，后续再收费
```

### Step 5: 上传文件

```
上传打包好的 .skill 文件：

1. 点击 "Upload" 或 "上传文件"
2. 选择文件：
   C:\Users\espre\.qclaw\workspace\skills\dist\home-reno-estimator.skill
3. 上传完成后会显示文件信息
```

### Step 6: 提交审核

```
1. 检查所有信息是否正确
2. 确认没有敏感内容
3. 点击 "Submit" 或 "发布"
4. 等待审核（通常1-3个工作日）
```

---

## 审核通过后

### 查看 Skill 页面

审核通过后，你的 Skill 会出现在：
- 搜索结果中
- 分类列表中
- 你的个人主页

页面 URL 类似：
```
https://clawhub.ai/skills/home-reno-estimator
```

### 管理后台

登录 ClawHub 后可以访问：
- 下载统计
- 用户评分和评论
- 收入报表
- 版本更新

### 更新 Skill

```
1. 修改本地文件
2. 重新打包（运行 package.py）
3. 在管理后台点击 "Update"
4. 上传新版本 .skill 文件
5. 提交更新
```

## 常见问题

### Q: 需要准备哪些材料？
```
A: 
- GitHub 账号（推荐）或邮箱
- Skill 名称和描述
- .skill 打包文件
- 可选：封面图（512×512 PNG）
```

### Q: 上架需要费用吗？
```
A: 完全免费
```

### Q: 收入怎么结算？
```
A:
- 平台抽成 30%
- 你拿 70%
- 按月结算
- 需要绑定收款方式（支付宝/微信）
```

### Q: 审核需要多久？
```
A:
- 通常 1-3 个工作日
- 首次上架可能稍长
- 如被拒绝，会收到拒绝原因
```

### Q: 审核被拒绝怎么办？
```
A: 
常见拒绝原因：
1. 描述不清 → 补充详细描述
2. 功能不完整 → 完善后再提交
3. 有安全风险 → 检查代码安全性

收到拒绝通知后，按要求修改，重新提交。
```

### Q: 可以设置下载限制吗？
```
A: 可以
- 免费：无限制
- 付费：设置价格，按购买次数
```

## 数据追踪

### 关键指标
```
- 下载量：有多少人下载了 Skill

- 评论：用户反馈
```

### 如何提高评分
```
1. 确保 Skill 功能稳定
2. 描述准确，不夸大
3. 及时回复用户反馈
4. 持续更新优化
5. 提供好的使用体验
```
