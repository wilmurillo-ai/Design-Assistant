## 二、补贴政策分析

### 补贴类型分类

```
1. 直接补贴（现金）
   - 无锡：50-500万
   - 常熟：600万
   - 合肥：1000万

2. 算力/资源补贴（券）
   - 合肥：1000万算力券 + 100万语料券 + 200万模型券

3. 贷款贴息
   - 合肥：最高50%贴息

4. 场景支持
   - 合肥：教育、医疗、制造、智慧城市等场景优先接入
```

---

### 补贴条件分析

| 条件 | 难度 | 说明 |
|------|------|------|
| 国家备案 | 中 | 垂直大模型需要通过国家备案 |
| 技术突破 | 高 | 具身智能机器人等需要关键技术突破 |
| 高层次人才 | 中 | 需要符合各级人才计划 |
| 开源框架/标准 | 高 | 需要牵头开发或制定行业标准 |
| 应用示范 | 中 | 需要获评标杆示范项目 |

## 三、根据补贴政策的创业方向推荐

