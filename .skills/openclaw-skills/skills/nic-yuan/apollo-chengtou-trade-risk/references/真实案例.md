# 真实案例测试示例

## 案例1：带回购条款的钢贸合同

**合同文本：**

```
钢材购销合同

甲方（买方）：某城投供应链公司
乙方（卖方）：江苏某民营钢贸公司

第一条 标的
甲方向乙方采购热轧卷板1000吨，单价4000元/吨，总价400万元。

第二条 付款方式
甲方在合同签订后5个工作日内支付合同总价的80%，即320万元作为预付款。

第三条 交货条款
乙方应在收到预付款后60天内完成交货。如乙方无法按时交货，应在5个工作日内退还甲方已支付的全部款项。

第四条 回购条款
如钢材市场价格下跌超过10%（以交货当日MySteel价格指数为准），甲方有权要求乙方按原合同价格回购甲方持有的剩余货物。

第五条 违约责任
任何一方违约，违约方应向守约方支付合同总价5%的违约金。
```

**AI 分析输出：**

```
## 合同风险审查报告

风险等级：🔴 高

### 风险条款清单

| 条款位置 | 条款内容 | 风险类型 | 风险等级 | 修改建议 |
|---------|---------|---------|---------|---------|
| 第四条 | 回购条款 | 融资性贸易特征 | 高 | 删除回购条款，或改为价格保护机制 |
| 第三条 | 退款条款 | 风险转移 | 中 | 明确货物灭失风险由谁承担 |

### 监管对应

- 十不准第一条：不准开展融资性贸易
  - 回购承诺使交易实质变为"借货融资"
- 十不准第三条：不准超越主业开展贸易
  - 需确认公司核准主业范围

### 修改建议

1. 删除第四条（回购条款），改为：
   - 建立价格波动时的协商机制
   - 约定最大库存量控制
2. 补充货权凭证要求：
   - 仓库仓单
   - 货权转移确认书
3. 增加银行监管账户
```

## 案例2：托盘贸易风险识别

**用户描述：**
"有一家民营贸易公司向我们提出托盘贸易合作，他们采购货物，我们先帮他们垫资采购，等他们销售出去再还款。这个业务模式有没有风险？"

**AI 分析输出：**

```
## 业务风险评估

业务名称：托盘贸易
风险等级：🔴 高

### 风险识别

| 风险类型 | 具体表现 | 风险等级 |
|---------|---------|---------|
| 融资性贸易 | 实质是为对方提供资金 | 高 |
| 信用风险 | 对方还款能力依赖销售情况 | 高 |
| 货权风险 | 垫资期间货物归属不清晰 | 中 |
| 合规风险 | 可能违反十不准第一条 | 高 |

### 十不准对应

- 第一条：不准开展融资性贸易
  - 垫资 = 提供融资，托盘贸易实质是融资
- 第七条：不准违规拆借资金
  - 资金通过货物"过桥"，实质仍是资金拆借

### 合作建议

**不建议直接开展此类业务**

如确需合作，建议：
1. 改为银行承兑汇票或保理
2. 要求对方提供足额抵押/质押担保
3. 控制单笔金额不超过注册资本的20%
4. 要求对方提供至少3个月的销售回款记录
5. 内部审批必须经过法务+风控+分管领导
```

## 案例3：尽职免责证据清单

**用户问：**
"如果万一出了风险，怎样才能'尽职免责'？"

**AI 分析输出：**

```
## 尽职免责指南

### 尽职免责的4个核心条件

| 条件 | 具体要求 |
|------|---------|
| 1. 尽调程序完整 | 已执行完整对手方尽调 |
| 2. 货权凭证真实 | 已取得真实完整的货权凭证 |
| 3. 档案记录完整 | 已建立完整的业务档案 |
| 4. 内部审批合规 | 已履行规定的内部审批程序 |

### 必须保留的6类证据材料

**一、对手方尽调材料**
- [ ] 工商登记信息查询截图
- [ ] 信用报告（第三方出具）
- [ ] 实地走访照片+记录
- [ ] 股东/实控人身份证验证
- [ ] 上下游客户名单+交易历史

**二、合同签署材料**
- [ ] 合同原件（各方盖章齐全）
- [ ] 合同审批流程记录
- [ ] 合同变更/补充协议（如有）

**三、货权凭证材料**
- [ ] 采购合同+发票
- [ ] 仓库入库单+仓单
- [ ] 货权转移确认书
- [ ] 物流单据（运输+签收）
- [ ] 库存盘点记录

**四、资金流转材料**
- [ ] 银行付款凭证
- [ ] 收款凭证
- [ ] 银行流水对账单

**五、内部审批材料**
- [ ] 业务立项申请
- [ ] 风控审查意见
- [ ] 法务审查意见
- [ ] 领导审批签字
- [ ] 党委会/办公会决议（如需）

**六、沟通记录**
- [ ] 与对手方的往来邮件
- [ ] 重要决策的聊天记录截图
- [ ] 会议纪要
```
