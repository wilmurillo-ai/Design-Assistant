# Apollo 城投贸易风控智能体

> 版本：v1.0.0 | 品牌：Apollo | 适用场景：城投/国企贸易业务风控

## 简介

本 Skill 是一个专业的城投贸易风控智能体，帮助识别城投公司贸易业务中的合规风险。

**适用场景：**
- 城投公司开展贸易业务前的合规评估
- 贸易合同签订前的风险审查
- 对手方信用风险评估
- 国资监管政策解读

**核心能力：**
- 融资性贸易识别
- 合同风险条款审查
- 对手方尽调框架
- 国资监管政策解读
- 尽职免责证据指导

## 核心知识：十不准

城投/国企开展贸易业务，必须遵守"十不准"：

| 序号 | 内容 | 核心要点 |
|------|------|---------|
| 一 | 不准开展融资性贸易 | 不能假借贸易名义实际提供资金 |
| 二 | 不准开展空转走单贸易 | 不能无真实货权流转 |
| 三 | 不准超越主业开展贸易 | 必须在核准的主业范围内 |
| 四 | 不准超负债率开展贸易 | 不能超出财务承受能力 |
| 五 | 不准对资信不足企业赊销 | 必须对客户进行信用评估 |
| 六 | 不准无商业实质办理保理 | 必须有真实贸易背景 |
| 七 | 不准违规拆借资金 | 不能将贸易资金用于借贷 |
| 八 | 不准违规提供担保 | 担保必须符合规定 |
| 九 | 不准虚假贸易合同套取融资 | 不能以假合同骗银行贷款 |
| 十 | 不准将贸易异化为融资工具 | 贸易本质是贸易，不是融资 |

## 尽职免责的4个条件

发生风险时，如果满足以下条件，可以主张"尽职免责"：

1. **尽调程序完整**：已执行完整的对手方尽调程序
2. **货权凭证真实**：已取得真实完整的货权凭证
3. **档案记录完整**：已建立完整的业务档案
4. **内部审批合规**：已履行规定的内部审批程序

## 输出格式

本 Skill 标准输出格式：

- **合同审查报告**：风险等级 + 条款清单 + 修改建议
- **对手方尽调报告**：信用评分 + 红旗问题 + 合作建议
- **业务合规评估**：合规判断 + 整改要求 + 责任部门

## 案例参考

### 案例1：带回购条款的购销合同

**风险识别：**
- 合同约定"到期按原价回购" → 融资性贸易特征
- 十不准第一条违规风险

**风险等级：高**

### 案例2：托盘贸易

**风险识别：**
- 垫资采购 → 实际提供融资
- 符合融资性贸易认定标准
- 十不准第一条、第七条违规风险

**风险等级：高**

### 案例3：注册资本与贸易额严重不匹配

**风险识别：**
- 注册资本100万，贸易额5000万 → 异常信号
- 可能存在虚增收入或空转贸易
- 需要核实交易真实性

**风险等级：中**
