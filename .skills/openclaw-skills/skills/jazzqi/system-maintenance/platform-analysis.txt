# 平台依赖分析

## macOS 特定项:
1. 进程检测: `ps aux | grep "openclaw-gateway"` (macOS/Unix)
2. 服务管理: `launchctl` (macOS)
3. 用户目录: `/Users/` (macOS)
4. 临时目录: `/tmp/` (macOS/Unix)

## Unix/Linux 通用项:
1. `crontab` 命令
2. `grep/awk/sed` 文本处理
3. `curl` HTTP 请求
4. `find/wc/tail/head` 文件操作

## 潜在跨平台问题:
1. 路径分隔符: `/` vs `\`
2. 环境变量: `$HOME` vs `%USERPROFILE%`
3. 命令可用性: 某些命令在不同系统不同
4. 权限系统: Unix vs Windows

## 建议解决方案:
1. 抽象平台检测函数
2. 使用条件分支处理平台差异
3. 提供配置选项
4. 添加文档说明
