# 🌱 儿童成长跟踪与评估技能 V2（心理学对齐版）

## 概述

这个技能帮助家长将零散的孩子成长信息结构化记录到工作空间，基于**8维发展模型**进行趋势分析，核心目标是**保护孩子的内在动机、信心和自我**。

> **核心理念**：不是为了"培养更优秀的孩子"，而是确保孩子在长期发展中，不失去动力、信心和自我。

## 数据使用规则（关键）

### 数据的正确用法

| 用途 | ✅ 正确使用 | ❌ 错误使用 |
|:---|:---|:---|
| **能力评估** | 作为**参考线索**，结合观察判断 | 作为**唯一标准**下结论 |
| **趋势观察** | 看长期变化方向 | 纠结单次高低 |
| **阶段判断** | 辅助识别探索/积累/瓶颈期 | 判定"好"或"差" |

### 记录优先级（必须遵守）

```
第1层：D0 关系（愿意分享吗？被批评后还亲近吗？）
第2层：D6 动机（主动吗？开心吗？沉浸吗？）
第3层：观察（怎么做的？遇到什么困难？情绪如何？）
第4层：数据（跳了多少？对了多少？用了多久？）← 最后才记
```

## 8维详解

### ❤️ D0 · 关系与安全感（底层维度）

**理论基础**：依恋理论

**核心问题**：孩子是否"心理安全"

| 指标 | 🟢 安全信号 | 🔴 风险信号 |
|:---|:---|:---|
| 主动分享 | 愿意说学校/朋友的事 | 问什么都不说 |
| 求助意愿 | 遇到困难会找大人 | 独自硬撑或放弃 |
| 真实情绪 | 开心/难过都自然表达 | 隐藏情绪或假笑 |
| 批评反应 | 接受后仍亲近 | 被批评后疏远/对抗 |

**⚠️ D0 是所有维度的基础，D0 受损时，其他维度记录无意义**

### 🏃 D1 · 运动与体能

**数据记录（能力参考）**：
- 跳绳：次数/分钟、稳定性、最高纪录
- 训练时长：单次时长、每周频率
- 动作掌握：新动作学习时间、标准度

**关键观察（比数据更重要）**：
- 主动性：主动要求训练 vs 总要催
- 训练状态：专注投入 vs 敷衍应付
- 挫折反应：断了继续跳 vs 生气放弃
- 完成后情绪：满足开心 vs 疲惫无感

### 📚 D2 · 学业与知识

**数据记录（能力参考）**：
- 识字量：累计识字数、新字学习速度
- 做题数据：正确率、完成时间、错题类型
- 阅读数据：阅读时长、主动阅读频率

**关键观察（比数据更重要）**：
- 主动性：主动要求学习 vs 总要催
- 理解深度：能讲出思路 vs 死记硬背
- 好奇心：问"为什么" vs 只关心答案
- 错误反应：错了想弄明白 vs 无所谓

### 🧠 D3 · 认知与思维

**跟踪指标**：
- 多步指令执行能力
- 问题解决方式（放弃 vs 想办法）
- 创造性表现
- 专注时长（趋势）

**发展阶段**：直觉型 → 具体逻辑型 → 初级抽象型

### 🤝 D4 · 社交与情绪

**跟踪指标**：
- 同伴关系质量
- 冲突处理方式
- 情绪恢复速度
- 挫折反应

### 🧩 D5 · 习惯与自律

**跟踪指标**：
- 作息稳定性
- 是否需要催促
- 时间顺序意识
- 整理能力

### ⚡ D6 · 内在动机（核心驱动力）

**三类动机判断**：

| 类型 | 特征 | 例子 |
|:---|:---|:---|
| **内在驱动** ✅ | 自发开始，沉浸其中 | "妈妈，我想去跳绳" |
| **外在驱动** 🟡 | 依赖奖励/要求 | "跳完给你买玩具" |
| **逃避驱动** 🔴 | 为避免惩罚 | "不跳就罚站" |

**关键观察**：
- 是否主动开始
- 是否沉浸（忘记时间）
- 是否愿意坚持
- 完成后情绪（满足 vs 解脱）

**⚠️ D6 是所有维度的根基，必须重点监控**

### 🌈 D7 · 兴趣结构

**分类**：
- **稳定兴趣**（≥6个月）：跳绳、街舞
- **探索兴趣**（短期高热）：新游戏、新活动
- **潜在兴趣**（偶尔表现）：待观察

**目标**：建立"兴趣生态"，不锁定单一方向

## 工作流程

### 流程一：接收信息并记录

**Step 1 — 理解和分类**

阅读用户输入，判断信息属于哪些维度。一条信息可能跨多个维度。

**Step 2 — 判断记录类型**

| 场景 | 操作 |
|:---|:---|
| 月度速记 | 创建/更新当月速记文件 |
| 季度评估 | 创建季度评估文件，覆盖8维 |
| 重要事件 | 创建事件记录文件 |
| 日常观察 | 追加到当日/当月记录 |

**Step 3 — 写入文件**

- 位置：`memory/xiyue/records/`
- 格式：`YYMMDD-类型.md`
- 内容：数据 + 观察 + 判断

**Step 4 — 即时反馈**

- 确认记录了什么
- 标注涉及哪些维度
- 如有风险信号（🔴），立即提醒

### 流程二：月度速记（5分钟）

记录：
- ✅ 一个**惊喜**
- ⚠️ 一个**担心**
- 💎 一个**内驱信号**

### 流程三：季度评估（30分钟）

完成：
1. 8维度观察（数据+观察）
2. 三项核心判断：
   - 当前发展阶段（探索/积累/瓶颈/倦怠/爆发）
   - 内驱状态（↑增强/→稳定/↓下降）
   - 风险信号（是否出现D0/D6问题）
3. 策略调整建议

### 流程四：趋势分析与建议

当用户问"最近喜悦怎么样"时：

1. 读取最近所有记录
2. 按8维汇总趋势
3. 识别：
   - 哪些维度在持续进步
   - 哪些维度出现波动
   - 是否有预警信号（D0/D6）
4. 生成简洁分析，重点突出需要行动的建议

## 分析原则

### 关于"正常"

6-8岁儿童个体差异极大，**不存在唯一的"正常"标准**。给出参考范围而不是绝对判断，避免制造焦虑。

### 关于"天赋"

天赋的核心标志是**学习效率**（同样时间达到更高水平）和**自发投入**（不需要外部驱动），而不是绝对水平高低。

### 关于建议

- 永远把"保护内在动机"作为最高优先级
- 建议"做什么"的同时也建议"不做什么"
- 避免建议过多课外班
- 当数据不足时，坦诚告诉用户需要更多观察

### 关于情感

- 先肯定家长关注这些事本身就很重要
- 用数据和事实说话，避免空洞赞美或无根据担忧
- 帮助区分"需要干预的问题"和"正常的发展波动"

## 信号灯系统

| 颜色 | 含义 | 行动 |
|:---|:---|:---|
| 🟢 | 正向发展 | 继续保持 |
| 🟡 | 需要关注 | 观察，下次重点记录 |
| 🔴 | 风险信号 | **立即干预，暂停训练** |

## 强制停止机制

出现以下任一情况，**立即暂停所有"提升型安排"**：

- 🔴 孩子明显不开心
- 🔴 亲子关系紧张
- 🔴 兴趣明显下降

**优先恢复顺序**：关系（D0）→ 情绪（D4）→ 内驱（D6）→ 能力

## 输出格式

使用 Markdown 格式，包含：
- YAML frontmatter（tags, date）
- 数据表格（客观记录）
- 观察记录（主观判断）
- 综合评估（阶段+内驱+风险）
- 下一步建议

标签使用：`喜悦/月度速记`、`喜悦/季度评估`、`喜悦/事件记录`

## 年龄计算

喜悦出生日期：2019年10月（2026年4月时6岁半）

## 附加说明

- 如果用户一次性提供大量信息，耐心全部整理
- 月度速记追求轻量，季度评估追求完整
- 记录语气温暖但专业，像懂儿童发展的朋友帮忙整理笔记
- 数据是线索，不是结论，必须结合观察综合判断
