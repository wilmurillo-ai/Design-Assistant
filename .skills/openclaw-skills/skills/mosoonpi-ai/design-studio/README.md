# 🎨 Design Studio — Professional Design Skill for OpenClaw

Turn your OpenClaw agent into a full design studio. Covers, banners, logos, mockups, GIFs — all from chat.

## Why Use This

| Without design-studio | With design-studio |
|---|---|
| 2 hours per cover in Canva | **30 seconds** per cover |
| Guessing if design looks good | **Auto score 1-10** (contrast, balance, palette) |
| One variant at a time | **4 A/B variants** compared instantly |
| 50 covers = 50 hours | **50 covers from CSV** in one command |
| Static banners only | **Animated GIF banners** (fade, slide, pulse) |
| Pay $20/mo for mockup tools | **Device mockups** free (laptop, phone, tablet) |

## What's Included

- **12 Python scripts** — banners, covers, avatars, mockups, GIFs, watermarks, batch generation
- **23 SVG elements** — icons, badges, shapes, frames
- **3 reference guides** — 20 color palettes, 15 font pairs, design rules
- **Auto quality checker** — scores your designs 1-10 and tells you what to fix
- **Full design pipeline** — generate → check → auto-improve → final output

## Requirements

- Python 3.10+ with Pillow, svgwrite, CairoSVG, fonttools
- ImageMagick 6.9+
- Inkscape (optional), GIMP (optional)
- Cyrillic fonts recommended (PT Sans, Roboto, Montserrat)

## Install

```bash
clawhub install design-studio
```
