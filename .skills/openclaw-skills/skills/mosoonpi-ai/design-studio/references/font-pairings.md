## Классические пары

### 1. Montserrat + PT Serif
> Для: портфолио, презентации, лендинги
- **Заголовок**: Montserrat Bold — геометричный, современный
- **Текст**: PT Serif Regular — элегантный, читабельный
- Контраст: гротеск + антиква = идеальный баланс
```
Заголовок: /usr/local/share/fonts/Montserrat-Bold.ttf
Текст:     /usr/share/fonts/truetype/paratype/PTF55F.ttf
```

### 2. PT Sans + PT Serif
> Для: деловые документы, отчёты, русскоязычные проекты
- **Заголовок**: PT Sans Bold — строгий, чистый
- **Текст**: PT Serif Regular — классический, комфортный
- Одно семейство ParaType: идеальная совместимость
```
Заголовок: /usr/share/fonts/truetype/paratype/PTS75F.ttf
Текст:     /usr/share/fonts/truetype/paratype/PTF55F.ttf
```

### 3. Roboto + PT Serif

```

Текст:     /usr/share/fonts/truetype/paratype/PTF55F.ttf
```

## Правила комбинирования

1. **Контраст обязателен**: антиква + гротеск, или разные пропорции одного класса
2. **Не сочетай похожие**: два похожих гротеска будут конфликтовать
3. **Один источник**: пары из одного семейства (PT Sans + PT Serif) всегда безопасны
4. **Тестируй кириллицу**: не все шрифты одинаково хороши для русского текста
5. **Размер решает**: заголовочный шрифт может быть необычным, текстовый — всегда нейтральный
