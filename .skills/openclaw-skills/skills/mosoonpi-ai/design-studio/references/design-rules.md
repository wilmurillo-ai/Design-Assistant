# Правила профессионального дизайна

> Уровень студии Лебедева. Каждое правило — результат практики, а не теории.

---

## 1. Типографика

### Иерархия
- **Заголовок (H1)**: самый крупный, жирный. Привлекает внимание первым.
- **Подзаголовок (H2)**: поддерживает заголовок, средний вес.
- **Основной текст**: Regular начертание, комфортный для чтения размер.
- **Подписи/мета**: мельче основного, приглушённый цвет.

### Размеры
- Кегль **кратен 4px**: 12, 16, 20, 24, 28, 32, 36, 40, 48, 56, 64, 72, 80, 96
- Минимум для основного текста: **16px** (экран), **14px** (мелкий текст)
- Заголовок баннера: **48–96px** в зависимости от размера холста

### Межстрочный интервал (line-height)
- Основной текст: **1.4–1.6** от кегля
- Заголовки: **1.1–1.3** (крупный текст нуждается в меньшем интервале)
- Подписи: **1.3–1.5**

### Трекинг (letter-spacing)
- Заголовки капсом: **+2–5%** (UPPERCASE нуждается в разрядке)
- Основной текст: **0** (не трогай)
- Мелкий текст: **+1–2%** для читаемости

### Правила
- Не более **2 гарнитур** на проект (заголовок + текст)
- Контраст начертаний: Bold заголовок + Regular текст
- Никогда не растягивай и не сжимай шрифт
- Кириллица: используй шрифты с нативной кириллицей (PT Sans, Roboto, не фейковые)

## 2. Сетка и отступы

### Модульная сетка
- **12 колонок** — стандарт для web и баннеров
- Gutter (промежуток): **16–24px**
- Для баннеров можно упростить до **4–6 колонок**

### Отступы (spacing)
- Все отступы **кратны 8px**: 8, 16, 24, 32, 40, 48, 56, 64, 80, 96
- Padding контейнера: минимум **24px** от края
- Между блоками: **24–48px**
- Между заголовком и текстом: **8–16px**

### Выравнивание
- Элементы **всегда на сетке** — никаких «на глазок»
- Текст: лево или центр (право — только для чисел и дат)
- Если 2+ элемента рядом — они должны быть выровнены по базовой линии

## 3. Цвет

### Палитра
- **Ограниченная палитра**: 1–2 акцентных + 2–3 нейтральных
- Один цвет = один смысл (не используй синий и для ссылок, и для ошибок)
- Фон: нейтральный (белый, светло-серый, тёмный)

### Контраст (WCAG)
- Текст на фоне: минимум **4.5:1** (AA стандарт)
- Крупный текст (≥24px или ≥18.5px bold): минимум **3:1**
- Декоративный текст: ограничений нет, но читаемость — приоритет
- Проверка: `scripts/check_design.py` автоматически считает контраст

### Формула контраста
```
contrast_ratio = (L1 + 0.05) / (L2 + 0.05)
L = относительная яркость цвета (0 = чёрный, 1 = белый)
```

### Правила
- Не более **5 цветов** в одном дизайне
- Акцентный цвет — не более **10–20%** площади
- Градиенты: максимум 2 цвета, угол 135° или 180°
- Тени: цвет тени = основной цвет с opacity 15–25%

## 4. Композиция

### Правило третей
- Раздели холст на 9 равных частей (3×3)
- Ключевые элементы — на **пересечениях линий**
- Текст — в верхней или нижней трети

### Визуальная иерархия (порядок просмотра)
1. Самый крупный/яркий элемент (заголовок или фокус-изображение)
2. Поддерживающий элемент (подзаголовок, иллюстрация)
3. Детали (мелкий текст, лого, CTA)

### Воздух (whitespace)
- Вокруг каждого элемента должен быть **воздух**
- Чем важнее элемент — тем больше воздуха вокруг
- Пустое пространство — это не «пустая трата места», это инструмент
- Минимум **15–20%** площади должно быть пустым

### Баланс
- Симметричный: формальный, стабильный (для бизнеса)
- Асимметричный: динамичный, современный (для креатива)
- Визуальный вес: тёмное тяжелее светлого, крупное тяжелее мелкого

## 5. Логотипы

### Принципы
- **Простота**: узнаваем за 3 секунды
- **Масштабируемость**: читаем от 16×16px (favicon) до билборда
- **Ч/Б тест**: логотип должен работать без цвета
- **Фон-тест**: работает и на светлом, и на тёмном фоне

### Типы
- **Текстовый (wordmark)**: название = лого (Google, Coca-Cola)
- **Буквенный (lettermark)**: инициалы (IBM, HP)
- **Иконка + текст**: символ рядом с названием
- **Абстрактный**: геометрическая форма

### Технические требования
- Формат: SVG (вектор) + PNG (растр, прозрачный фон)
- Размеры PNG: 512×512, 256×256, 128×128, 64×64, 32×32
- Зона безопасности: минимум **10% от размера** вокруг логотипа

## 6. Баннеры

### Размеры (стандартные)
| Назначение | Размер (px) |
|---|---|
| GitHub/LinkedIn обложка | 1584×396 |
| YouTube обложка | 2560×1440 |
| Twitter/X заголовок | 1500×500 |
| Fiverr обложка гига | 1280×769 |
| Kwork обложка | 920×520 |
| Instagram пост | 1080×1080 |
| Телеграм-канал | 1280×720 |
| Общий web-баннер | 1920×600 |

### Правила баннеров
- **Focal point**: один главный фокус внимания
- **CTA-кнопка**: если есть — контрастная, с воздухом вокруг, скруглённые углы 8–12px
- **Минимум текста**: заголовок + 1 строка подзаголовка, не больше
- **Безопасная зона**: ключевой контент не ближе 10% от краёв
- **Мобильность**: центральная часть должна работать при кропе

## 7. Обложки фриланс-профилей

### Что работает на Fiverr/Kwork/Freelance.ru
- **Показывай результат**, а не процесс: скриншот, мокап, пример работы
- **Минимум текста**: 3–5 слов максимум
- **Лицо/персонаж**: увеличивает конверсию на 20–30%
- **Яркий фон**: выделяйся в ленте среди конкурентов
- **Чёткий оффер**: что получит клиент, а не кто ты

### Чего избегать
- Стоковые фото без обработки
- Мелкий нечитаемый текст
- Больше 3 цветов
- Клипарт и устаревший стиль
- Текст на сложном фоне без подложки

## 8. Чеклист перед сдачей

- [ ] Текст читаем на расстоянии вытянутой руки (баннер) / на маленьком экране (аватарка)
- [ ] Контраст текста ≥ 4.5:1
- [ ] Не более 2 шрифтов
- [ ] Не более 5 цветов
- [ ] Есть воздух вокруг ключевых элементов
- [ ] Логотип читаем при уменьшении до 32px
- [ ] Работает в ч/б (для логотипов)
- [ ] Ключевой контент в безопасной зоне
- [ ] Формат и размер соответствуют площадке
- [ ] `scripts/check_design.py` выдал оценку ≥ 7
