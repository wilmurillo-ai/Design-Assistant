## Как использовать

```python
# Пример: загрузка палитры в код
PALETTE = {
    "bg": "#0D1117",
    "surface": "#161B22",
    "text": "#E6EDF3",
    "accent1": "#58A6FF",
    "accent2": "#3FB950",
}
```

### Правила применения
1. **Фон** — основная заливка (60–70% площади)
2. **Поверхность** — карточки, блоки, выделения (15–25%)
3. **Текст** — основной текст и заголовки
4. **Акцент 1** — главный акцент: кнопки, ссылки, ключевые элементы (5–10%)
5. **Акцент 2** — вторичный акцент: иконки, разделители, hover (3–5%)
