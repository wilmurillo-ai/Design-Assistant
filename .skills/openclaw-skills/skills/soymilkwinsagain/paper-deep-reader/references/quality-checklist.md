# Quality Checklist

This file is kept for backward compatibility.

Use the new checklist structure instead:

1. Run the general checklist:
   - `{baseDir}/references/checklists/general.md`
2. Then run the routed domain / type checklist that matches the paper:
   - theory / mathematics / statistics
   - methods / ML / applied statistics
   - empirical / economics / social science
   - systems
   - physics
   - quantitative finance

If the paper is mixed, run more than one domain checklist.
