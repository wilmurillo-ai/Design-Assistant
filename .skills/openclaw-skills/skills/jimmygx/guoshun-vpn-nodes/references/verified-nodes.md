## 🔧 各客户端导入方法

### 电脑 - Clash for Windows
1. 打开 Clash for Windows
2. 点击左侧「配置(Profiles)」
3. 点击「新建配置( New Config)」
4. 粘贴 **Clash订阅地址**
5. 点击「下载配置」
6. 选择节点 → 点击「代理(Proxies)」启用

### 手机 - V2RayNG（安卓）
1. 打开 V2RayNG
2. 点击右上角「订阅( Subscription)」
3. 点击「添加订阅」
4. 粘贴 **V2Ray订阅地址**
5. 点击右上角「↻更新」
6. 选择节点 → 点击底部V图标连接

### 手机 - Shadowrocket（iOS）
1. 打开 Shadowrocket
2. 点击右上角「+」
3. 类型选「Subscribe」
4. 粘贴 **V2Ray订阅地址**
5. 点击完成
6. 选择节点，开启开关
