# 招标文件智能分析技能

## 功能概述

分析招标文件（PDF格式），生成结构化的投标决策支持报告，包含：
- 招标概览（5秒读懂项目）
- 资质匹配度检测
- 评分标准拆解与高分策略
- 风险预警（废标条款、霸王条款）
- 投标策略建议

## 输入

- PDF文件（上传到会话）
- 或招标文件关键章节文字
- 或招标公告链接（需要抓取内容）

## 输出

飞书卡片消息格式，包含以下模块：

```
📋 项目概览
- 项目名称、建设单位、地点
- 合同估算价、工期、质量标准
- 资质要求、建造师要求
- 关键时间节点

✅ 资质匹配度
- 资质要求 vs 我们的情况
- 建造师 vs 我们的情况
- 业绩要求 vs 我们的情况

🎯 评分标准拆解（如能从文件读取）
- 商务标/技术标权重
- 各评分项得分要点

⚠️ 风险预警
- 废标条款（必须规避）
- 不合理条款（霸王条款）

💡 投标策略建议
- 技术方案重点
- 商务报价策略
- 需提前准备的材料

📅 时间节点
- 招标文件获取截止
- 投标截止
- 开标时间
```

## 合同条款分析（重点！）

投标前必须评估的合同风险点：

### 付款条款
| 条款 | 分析要点 |
|-----|---------|
| **预付款** | 是否有预付款？比例多少？通常10-20% |
| **进度款** | 按月计量还是按节点支付？付款周期多长？ |
| **结算款** | 审计结算周期多长？是否留质保金？ |
| **财政资金** | ⚠️ 政府项目以财政拨付为准，可能延迟 |

### 保证金条款
| 条款 | 分析要点 |
|-----|---------|
| **投标保证金** | 金额多少？30-50万？可用信用承诺替代？ |
| **履约保证金** | 通常合同价10%，可用银行保函替代 |
| **质保金** | 通常3-5%，缺陷责任期后返还 |

### 工期与违约金
| 条款 | 分析要点 |
|-----|---------|
| **工期要求** | 日历天数，是否合理？ |
| **逾期竣工违约金** | ‰/天？ 有无上限？**无上限风险最大！** |
| **关键节点延误** | 单独列项的违约金？ |
| **误工责任** | 能否索赔工期？ |

### 变更与索赔
| 条款 | 分析要点 |
|-----|---------|
| **变更程序** | 变更必须及时签证，否则无法结算 |
| **价格调整** | 人工材料波动是否可调？ |
| **不平衡报价** | 单价超出基准±30%会被审查 |

### 风险识别重点
1. ⚠️ **财政资金项目**：付款以财政拨付为准，存在延迟风险
2. ⚠️ **无上限违约金**：逾期竣工无上限的慎重承接
3. ⚠️ **低价中标风险**：成本以下报价可能被废标
4. ⚠️ **农民工工资**：必须专户发放，有专项检查

## 分析流程

### Step 1: 提取项目基本信息

从招标文件公告章节提取：

| 字段 | 来源位置 |
|------|---------|
| 项目名称 | 招标公告/封面 |
| 标段编号 | 封面 |
| 招标单位 | 招标公告 |
| 代理机构 | 封面 |
| 建设地点 | 工程概况 |
| 建设规模 | 工程概况（面积/层数） |
| 合同估算价 | 工程概况（万元） |
| 工期要求 | 工程概况（日历天） |
| 质量标准 | 工程概况 |
| 招标范围 | 招标范围章节 |
| 资质要求 | 投标人资格要求 |
| 建造师要求 | 投标人资格要求 |
| 业绩要求 | 投标人资格要求 |
| 评标方法 | 评标办法章节 |
| 投标截止时间 | 招标公告/投标人须知 |

### Step 2: 资质匹配度分析

基于以下维度评估：

1. **企业资质** - 对比要求等级 vs 我们持有的资质
2. **项目负责人** - 建造师专业+等级+其他要求
3. **类似业绩** - 时间要求、金额要求、类型要求
4. **人员要求** - 安全员、质检员等配套人员
5. **信用要求** - 信用管理手册、失信被执行人等

### Step 3: 评分标准拆解

从评标办法章节提取：
- 综合评估法/合理低价法
- 商务标权重 vs 技术标权重
- 各评分项的评审标准

### Step 4: 风险识别

重点识别：
- **废标条款**：资质不符、保证金未交、签字盖章问题、联合体问题等
- **霸王条款**：超低保证金、过长工期扣款、不合理付款条件
- **隐藏风险**：两阶段评标、低价中标风险、变更限制等

### Step 5: 投标策略建议

基于以上分析生成：
- 技术方案应重点突出的内容
- 商务报价策略建议
- 需要提前准备的证明材料
- 资格预审/后审注意事项

## 招标文件结构识别

标准工程招标文件结构（徐州市公共资源交易中心格式）：

```
第一章 招标公告
第二章 投标人须知
  - 投标人须知前附表（关键参数汇总）
  - 投标人须知正文
第三章 评标办法（综合评估法/合理低价法）
  - 评标办法前附表（评分权重）
第四章 合同条款及格式
第五章 工程量清单
第六章 图纸
第七章 技术标准和要求
第八章 投标文件格式
```

重点读取章节：
1. 招标公告 → 基本信息+时间节点
2. 投标人须知前附表 → 关键参数一览
3. 第三章评标办法 → 评分标准
4. 第四章合同条款 → 合同风险

## 评标标准分析模式

### 两阶段评标标准结构（智能化工程典型）

**第一阶段：商务技术评审（12分）**
| 评审因素 | 分值 | 说明 |
|---------|------|------|
| 施工组织设计 | 10分 | 暗标，含BIM加分2.5分 |
| 投标人业绩 | 2分 | 每具备1个≥1550万类似工程0.5分 |

**第二阶段：报价评审（88分）**
| 评审因素 | 分值 | 说明 |
|---------|------|------|
| 投标报价 | 87分 | ABC合成法基准价 |
| 报价合理性 | 1分 | 控制价下浮10%对比 |

**报价得分规律：**
- 低于基准价：每低1%扣0.6分
- 高于基准价：每高1%扣0.9分（更严格）
- 最佳策略：略低于基准价

### 常见评标方法

1. **综合评估法（两阶段）**：商务技术入围 → 报价决胜（智能化工程常见）
2. **合理低价法**：报价最低者中标（最低价中标风险大）
3. **综合评估法（一次评审）**：技术+报价同时评审

## 注意事项

- PDF读取使用 pdfplumber 或 pdftotext
- 表格内容优先用 pdfplumber 提取
- 资质要求必须精确到等级（壹级/贰级/叁级）
- 建造师专业必须精确（建筑工程/机电工程）
- 业绩金额注意是"单项合同"还是"累计"
- 两阶段评标需重点关注第二阶段技术方案
- 联合体投标条款需单独分析

## 调用方式

用户上传PDF文件后，自动触发本技能进行分析。
