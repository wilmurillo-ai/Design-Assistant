## 关键发现

### 智能化工程评分规律
1. **两阶段评标是标准模式**：商务技术入围 → 报价决胜
2. **报价为王**：87-88分权重，技术标仅12分
3. **BIM是差异化得分点**：2.5分，可作为技术方案亮点
4. **建造师答辩是现场考核**：需提前准备，不能只靠书面材料
5. **业绩门槛清晰**：≥1550万建筑智能化（不含住宅类）

### 资质门槛规律
- 智能化工程普遍要求：**电子与智能化壹级**
- 建造师：**机电工程壹级**
- 联合体普遍接受（最多2家，牵头必须壹级）

### 合同风险规律
1. **工期违约金差异大**：有的是0.2‰/天有的是0.5‰/天，有的有上限有的无上限
2. **财政资金项目风险**：付款以财政拨付为准，可能存在延迟
3. **预付款不是都有**：有的10%预付款，有的没有
4. **农民工工资必须重视**：都有专项条例要求
5. **变更必须及时签证**：否则可能无法结算
