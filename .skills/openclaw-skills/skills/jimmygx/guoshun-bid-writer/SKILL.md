# 投标助手（技术方案生成器）— guoshun-bid-writer

## 技能定位

- **名称**：guoshun-bid-writer
- **版本**：1.0.0
- **适用公司**：江苏国顺智能科技有限公司
- **描述**：投标技术方案 AI 生成助手，支持全行业全专业，输入招标文件后自动生成技术方案初稿、技术偏离表、项目组织架构等投标文件核心内容。
- **触发关键词**：「投标」「技术方案」「施工组织设计」「招标」「标书」「偏离表」「资质自检」「编标」「投标助手」

---

## 核心功能清单

| # | 功能 | 说明 |
|---|------|------|
| 1 | 招标文件 AI 解析 | 解析 PDF，提取项目信息、招标要求、评标办法、时间节点 |
| 2 | 技术方案初稿生成 | 全行业全专业施工组织设计框架生成 |
| 3 | 投标资质自检 | 自动检查门槛条件，输出自检表及补救建议 |
| 4 | 技术偏离表生成 | 标准格式偏离表，正偏离绿色 / 负偏离黄色标注 |
| 5 | 项目组织架构生成 | 根据规模自动生成岗位配置（文字版） |
| 6 | 业绩证明材料生成 | 匹配历史业绩，生成符合招标文件格式的业绩描述 |

## 输入说明

### 支持的输入方式

**方式一：上传招标文件（推荐）**
- 用户上传招标文件的 PDF 文件
- AI 自动解析并提取关键信息
- 支持多文件上传（主招标文件 + 澄清补遗文件）

**方式二：粘贴招标文件正文**
- 用户直接粘贴招标公告 / 招标文件的文字内容
- 适用于无法上传 PDF 的场景

**方式三：提供关键信息**
- 项目名称、工程规模、工期要求、技术要求等关键字段
- AI 基于提供信息直接生成方案框架

### 用户需提供的基本信息（如上传文件无法解析）

- 项目名称
- 招标单位
- 工程地址
- 工程规模 / 建筑面积
- 计划工期
- 招标控制价（如有）
- 资质要求（等级 / 专业）
- 业绩要求（规模 / 数量 / 年限）
- 技术要求摘要

### 功能 2：技术方案初稿生成

#### 输入
- 招标文件（PDF / 文本）或结构化项目信息
- 工程类型（智能化/弱电/消防/机电/装饰/房建/市政等）
- 工程规模

#### 处理流程
1. 分析招标文件中的技术要求条款
2. 识别"必须响应条款"（标注 ⚠️ 必须响应）
3. 根据行业类型选择对应章节框架
4. 自动引用相关国家标准规范（GB/GB/T、JGJ、CJJ 等）
5. 生成完整施工组织设计初稿

#### 输出：施工组织设计（Markdown）

完整框架结构如下（通用版，AI 会根据招标技术要求自动调整侧重点）：

```markdown
# [项目名称] 施工组织设计

> **适用范围**：本方案为技术标初稿，标注 ⚠️ 的内容需根据实际情况调整或补充。
> **编制依据**：《建设工程项目管理规范》GB/T 50326、《建筑工程施工质量验收统一标准》GB 50300 等。
```

5. **业绩匹配**：业绩证明材料中的项目需确保真实存在并可提供证明文件。
6. **内容保密**：投标文件内容涉及商业机密，请勿对外传播。

## 常见问题

**Q：上传 PDF 无法解析怎么办？**
A：尝试将 PDF 文字内容复制粘贴，或提供项目关键信息的结构化描述（项目名称/规模/工期/技术要求等）。

**Q：方案内容太通用怎么办？**
A：提供更多项目具体信息（如工程特点、技术难点、施工环境），AI 会针对性调整方案侧重点。

**Q：偏离表中的负偏离如何处理？**
A：优先寻找替代方案消除负偏离；如无法消除，使用应答策略话术降低负面影响。

**Q：需要生成其他格式的输出吗？**
A：支持生成 Word 格式（Markdown 复制后用 Word 打开即可）、飞书文档直接写入。如需其他格式请告知。
