# 技术规格偏离表（模板）

> 本模板供生成偏离表时参考使用，实际使用时 AI 自动根据招标文件内容填写。

## 偏离表填写说明

- **✅ 正偏离**：我方技术参数优于招标文件要求，视为我方优势
- **○ 满足**：我方技术参数完全满足招标文件要求
- **⚠️ 负偏离**：我方技术参数低于招标文件要求，需给出应答策略
