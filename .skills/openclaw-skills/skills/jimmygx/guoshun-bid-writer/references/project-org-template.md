# 项目组织架构模板

> 根据工程规模自动匹配的岗位配置参考模板。标注 ⚠️ 的岗位需根据实际情况填写。

---

## 小型项目组织架构（合同金额 < 500万元）

**适用场景**：工期紧、人员少、单一专业施工

```
                    项目负责人（⚠️姓名）
                         │
          ┌──────────────┼──────────────┐
          │              │              │
      技术负责人      兼职安全员       施工班组
      （⚠️姓名）     （⚠️姓名）      （⚠️配置）
      （可兼职）      （可兼职）
```

**岗位配置**：

| 岗位 | 人数 | 资质要求 | 备注 |
|------|------|----------|------|
| 项目负责人 | 1 | 中级职称以上 | 可兼任技术负责人 |
| 技术负责人 | 1 | 大专以上学历 / 初级职称 | 可兼职施工员 |
| 兼职安全员 | 1 | 安全员C证 | 建议由技术负责人兼任 |
| 施工员 | 1 | 有效上岗证 | 负责现场管理 |
| 资料员 | 1 | 持有资料员证 | 可兼职 |

## 中型项目组织架构（合同金额 500万 - 2000万元）

**适用场景**：多专业交叉施工、有一定技术难度

```
                  项目经理（⚠️姓名）
                       │
     ┌─────────────────┼─────────────────┐
     │                 │                 │
  项目副经理        技术负责人         安全主管
  （⚠️姓名）        （⚠️姓名）         （⚠️姓名）
     │                 │                 │
  ┌──┴──┐         ┌───┴───┐         专职安全员
  │     │         │       │         （⚠️姓名）
 材料员 │    专业技术员    │              │
（⚠️）  │      （⚠️）      │              │
  │     │         │       │              │
  └─────┴─────────┴───────┴──────────────┘
                         │
              ┌──────────┼──────────┐
              │          │          │
           质量员     施工员     资料员
          （⚠️）     （⚠️×2）    （⚠️）
```

**岗位配置**：

| 岗位 | 人数 | 资质要求 | 备注 |
|------|------|----------|------|
| 项目经理 | 1 | 二级建造师以上 | 全职 |
| 技术负责人 | 1 | 大专 / 中级以上职称 | 全职 |
| 安全主管 | 1 | 安全员C证 | 全职 |
| 专职安全员 | 1 | 安全员C证 | 全职 |
| 质量员 | 1 | 质量员证 | 全职 |
| 施工员 | 2 | 有效上岗证 | 全职 |
| 材料员 | 1 | 材料员证 | 全职 |
| 资料员 | 1 | 资料员证 | 全职 |

## 大型项目组织架构（合同金额 > 2000万元）

**适用场景**：大型综合工程、技术难度高、多方协调

```
                       项目经理（⚠️姓名）
                            │
    ┌───────────┬───────────┼───────────┬───────────┐
    │           │           │           │           │
  项目副经理  技术负责人   安全主管    质量主管    材料主管
  （⚠️姓名）  （⚠️姓名）   （⚠️姓名）   （⚠️姓名）  （⚠️姓名）
    │           │           │           │           │
  ┌─┴─┐    ┌────┴────┐   ┌──┴──┐    ┌──┴──┐     ┌──┴──┐
  │    │    │         │   │     │    │     │     │     │
造价    深化设计  专职安全  专职质量  材料员  仓管
工程师  工程师    员×2     员×2     （⚠️）  （⚠️）
（⚠️） （⚠️）
                         │
                  ┌──────┼──────┐
                  │      │      │
               施工员  施工员  资料员
               ×3     ×3     （⚠️）
```

**岗位配置**：

| 岗位 | 人数 | 资质要求 | 备注 |
|------|------|----------|------|
| 项目经理 | 1 | 一级建造师 | 全职，应有同类项目经验 |
| 项目副经理 | 1 | 二级建造师以上 | 全职 |
| 技术负责人 | 1 | 大专 / 高级职称 | 全职 |
| 造价工程师 | 1 | 注册造价工程师 | 全职 |
| 深化设计工程师 | 1 | 中级以上职称 | 全职 |
| 安全主管 | 1 | 安全员C证 | 全职 |
| 专职安全员 | ≥2 | 安全员C证 | 全职 |
| 质量主管 | 1 | 质量员证 | 全职 |
| 专职质量员 | ≥2 | 质量员证 | 全职 |
| 施工员 | ≥3 | 有效上岗证 | 全职 |
| 材料员 | 1 | 材料员证 | 全职 |
| 仓管员 | 1 | — | 全职 |
| 资料员 | 1 | 资料员证 | 全职 |

## 智能化/弱电工程专项岗位补充

在通用架构基础上，智能化工程还需配置以下专业岗位：

| 专项岗位 | 人数 | 资质要求 | 主要职责 |
|----------|------|----------|----------|
| 系统集成工程师 | 1-2 | 中级职称以上 | 负责系统集成调试 |
| 网络工程师 | 1 |HCIP/CCNP以上 | 负责网络系统调试 |
| 调试工程师 | 按系统数量配置 | 厂家认证 | 负责各子系统调试 |
| 售后服务人员 | 1 | — | 负责后期运维 |

## 不同工程类型岗位配置差异

| 工程类型 | 特殊岗位 | 配置说明 |
|----------|----------|----------|
| 智能化工程 | 系统集成工程师、网络工程师 | 需配置各子系统调试人员 |
| 消防工程 | 消防工程师、注册消防工程师 | 需配置消防验收配合人员 |
| 机电安装 | 管道工程师、电气工程师 | 需配置专业工长 |
| 装饰装修 | 深化设计师、造价员 | 需配置效果图配合人员 |
| 房建工程 | 测量工程师、试验工程师 | 需配置标准养护室 |
| 市政工程 | 测量工程师、预算工程师 | 需配置现场造价控制 |
