# 国顺数字团队协作技能

> 为 OpenClaw 提供多智能体协作能力，模拟真实数字团队的分工与讨论机制

## 功能预览

```bash
# 辩论模式
./scripts/digital_team.sh debate "要不要上微服务架构" 3

# 角色池协作
./scripts/digital_team.sh roles "评审这个技术方案" "architect,developer,tester"

# 任务分发
./scripts/digital_team.sh assign "开发用户模块"
```

## 适用场景

- 📋 **技术方案评审** — 架构师 vs 程序员辩论最佳方案
- 🔍 **代码审查** — 多角色视角发现潜在问题
- 📊 **项目规划** — 拆解复杂任务并行执行
- 🎯 **需求讨论** — 产品、技术、测试多方视角
