# 技术偏离表模板

## 说明
- ✅ 正偏离：我司方案优于招标要求，明显优势
- ⚠️ 负偏离：我司方案不满足招标要求，需说明原因和替代方案
- ➖ 完全响应

| 序号 | 招标要求 | 我司应答 | 偏离类型 | 应答策略/说明 |
|-----|---------|---------|---------|--------------|
| 1 | [招标技术规格条款] | [我司提供的技术参数/方案] | ✅ 正偏离 / ⚠️ 负偏离 / ➖ 响应 | [说明为何偏离，如果是负偏离给出弥补措施] |
| 2 | [招标技术规格条款] | ... | ... | ... |
| 3 | [招标技术规格条款] | ... | ... | ... |

**填写要点**：
- 逐条响应招标文件中的技术规范
- 负偏离需保守处理，避免废标
- 正偏离要明确说明优势（性能更高/功能更多/服务更好）
