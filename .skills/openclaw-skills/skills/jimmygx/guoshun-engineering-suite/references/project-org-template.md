# 项目组织架构模板（根据规模自动匹配）

## 大型项目（合同额≥1000万）

```
                       项目经理（一级建造师）
                             |
         ┌─────────┬─────────┬─────────┐
    技术负责人    施工主管    质量主管    安全主管
  （高级工程师） （施工员）  （质量员）  （安全员）
         |           |          |          |
    各专业技术员  各班组长    质检员    安全员
    （弱电/消防等） （水电/安装等）
```

**总人数配置：15-30人**

## 中型项目（300万≤合同额<1000万）

```
                   项目经理（二级建造师）
                         |
            ┌────────┬────────┐
       技术负责人    施工员兼质量员    安全员
   （中级工程师）   （兼质量员）    （专职）
         |
     资料员（兼）
```

**总人数配置：8-15人**

## 小型项目（合同额<300万）

```
                   项目经理（可兼职）
                         |
                  施工员/技术负责人
                         |
                  质量员（兼职）
                         |
                  安全员（兼职）
```

**总人数配置：3-8人**

## 特殊专业岗位要求（智能化/消防工程）

- **弱电系统工程师**：负责综合布线、网络、安防、门禁等子系统对接
- **消防系统工程师**：负责火灾报警、灭火系统、防排烟
- **BIM工程师**（如业主要求）：负责深化设计、管线综合、4D模拟

**说明**：根据招标文件中对项目经理、技术负责人资格要求，匹配相应职称和证书。
