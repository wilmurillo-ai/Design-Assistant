# 样本文件目录

本目录用于存放招标文件和合同样本，供 AI 学习参考。

建议包含以下文件：

```
samples/
├── bid-templates/          # 招标文件样本（PDF）
│   ├── 智能化工程-招标文件示例.pdf
│   ├── 消防工程-招标文件示例.pdf
│   └── 施工总承包-招标文件示例.pdf
└── contract-templates/    # 合同样本（PDF/DOCX）
    ├── 施工分包合同-样本.pdf
    ├── 货物采购合同-样本.docx
    └── 智能化分包合同-样本.pdf
```

**说明**：
- 样本文件应脱敏处理，隐去真实公司名称、金额、地址等敏感信息
- 样本应覆盖典型工程合同条款（付款方式、工期、违约金、质保金等）
- 招标文件样本应包含评标办法、技术需求、资质要求等核心章节
