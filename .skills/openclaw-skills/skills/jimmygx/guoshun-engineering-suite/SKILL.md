本技能集成三大工程企业最高频的AI能力：

| 模块 | 核心功能 | 触发词示例 |
|------|---------|-----------|
| **📄 招标文件分析** | 解析招标文件，输出资质要求、评标标准、风险提示 | "分析招标文件"、"招标要求"、"评标办法"、"资质门槛" |

- 粘贴合同文本 → 触发"合同审核"

## 模块1：招标文件分析

### 功能描述
上传招标文件（PDF），AI自动解析并输出关键信息摘要，帮助商务人员快速掌握招标核心要求。

### 输入
- 招标文件 PDF（上传）
- 或招标文件关键页面的文字粘贴

### 输出
飞书卡片格式，包含：

```
📋 招标项目概览
- 项目名称、地址、规模、投资额
- 招标人、代理机构
- 投标截止、开标时间
- 工期要求、质量标准

🎯 资质门槛
- 企业资质要求（施工/设计/监理等级）
- 项目经理/关键人员资格要求
- 类似业绩要求（金额/数量）
- 安全生产许可证/ISO认证等

📊 评标办法
- 技术分权重（通常40-60%）
- 商务分权重（价格分占比）
- 具体评分标准摘要（如：施工方案20分、项目负责人10分、业绩15分等）
- 是否存在低价中标、综合评估法还是经评审的最低投标价法

⚠️ 风险提示
- 废标条款（必须响应项）
- 关键时间节点（澄清、踏勘、投标）
- 甲控材料、甲供工程
- 限制条件（联合体、分包、region lock）
```

### 典型使用场景
- 收到招标文件后，5分钟内获取核心信息
- 判断是否参与投标（资质门槛是否满足）
- 理解甲方关注点（技术分权重高的要重点准备）
- 避免废标（关注实质性响应条款）

- 招标文件 PDF（优先）或项目信息（项目名称/地址/规模/工期）

## 模块3：合同风险审核

### 功能描述
对工程合同（施工分包/货物采购/智能化分包等）进行全面风险扫描，输出结构化报告，帮助商务/法务快速识别高风险条款，指导谈判策略。

### 输入
- PDF/DOCX合同文件（上传）
- 或合同关键章节文字粘贴

### 输出
飞书卡片格式（详细内容同 guoshun-contract-analyzer V2.0）：

```
📋 合同概览
- 合同名称、甲乙双方、签约日期
- 合同类型、承包方式、合同总价

💰 付款条款分析
- 预付款比例（有/无）
- 进度款支付节点和比例
- 结算款支付节点
- 质保金比例和退还条件
- 🧮 合同价款核算器（自动计算实际到手金额、资金占用）

⏰ 工期与违约金评估
- 总工期、起止时间
- 工期类型（绝对工期/相对工期）
- 延期违约金（每天/每次/总额上限）

💎 合同闪光点识别
- A级（金牌条款）：预付款30%、有变更调价权、工期可顺延等
- B级（银牌条款）：进度款按月支付、质保金2%以内等
- C级（铜牌条款）：验收合格后7日内付款等

🎯 霸王条款谈判策略（高风险条款专项）
- 条款原文 + 风险等级 + 谈判话术 + 替代方案参考

🔍 关键风险识别
- 🔴 高风险条款（必须关注）
- 🟡 中风险条款（谨慎对待）
- 🟢 低风险条款（可接受）

📋 履约风险清单（按时间线）
- 签约阶段、施工阶段、竣工阶段、结算阶段、质保阶段各阶段风险点与应对措施

💡 承接建议
- 投标/承接决策参考
- 合同谈判要点
- 执行阶段注意事项

📊 综合评级
- 承接价值评分（1-5星）
- 风险系数评分（1-5星）
- 建议策略（承接/谨慎/拒绝）
```

## 注意事项

### 招标文件分析模块
- 支持PDF格式，建议上传清晰文字版PDF
- 图片扫描件需先用OCR识别
- 大型招标文件（>500页）可能解析不完整

### 投标助手模块
- 技术方案为初稿，需根据实际工程情况调整
- 资质自检基于公开信息，签约前需人工复核
- 偏离表回应需结合企业实际能力填写
- 业绩材料可根据实际情况补充或替换

### 合同审核模块
- 合同版本需完整（含所有附件和补充协议）
- 金额/日期等关键数据以合同原文为准
- 谈判话术仅供参考，具体策略需法务审核
- 本合同价款核算器仅做初步测算，真实成本需财务复核
