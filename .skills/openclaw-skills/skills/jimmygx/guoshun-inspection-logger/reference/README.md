# 参考标准文档

本目录存放江苏省建设工程施工/监理相关标准规范供参考。

## 主要标准

### 1. DGJ32/J 66-2015 建设工程施工现场标识标准
- 发布单位：江苏省住房和城乡建设厅
- 实施日期：2015年
- 主要内容：施工现场各类标识牌的设置要求、规格、颜色等

### 2. DGJ32/TJ 196-2015 建设工程监理规范
- 发布单位：江苏省住房和城乡建设厅
- 实施日期：2015年
- 主要内容：监理工作程序、监理资料管理、监理日志填写要求等

### 3. 江苏省建设工程施工阶段监理现场用表目录
- 包含表格类型：
  - FH-001 监理日志
  - FH-002 监理月报
  - SG-001 施工日志
  - AQ-001 安全巡检记录
  - YJ-001 隐蔽工程验收记录
  - WD-001 危大工程巡视记录

## 表格填写说明

### 监理日志（FH-001）
- 填写人：监理员
- 填写时间：每日
- 主要内容：施工部位、人员投入、设备投入、完成工程量、质量问题、安全文明施工情况等

### 施工日志（SG-001）
- 填写人：施工员
- 填写时间：每日
- 主要内容：与监理日志类似，以施工单位视角记录

### 安全巡检记录（AQ-001）
- 填写人：安全员
- 填写时间：每日巡检
- 主要内容：巡检区域、发现的安全隐患、整改要求、复查情况

### 隐蔽工程验收记录（YJ-001）
- 填写人：监理员、施工员
- 填写时间：隐蔽工程施工前
- 主要内容：隐蔽项目、施工内容、验收情况、质量问题、整改要求

### 危大工程巡视记录（WD-001）
- 填写人：监理员
- 填写时间：危大工程施工期间
- 主要内容：危大工程类型、施工状况、安全防护措施、存在问题、处理意见

## 注意事项

1. 所有记录应使用规范用语，避免口语化
2. 问题描述应具体：包含部位、现象、处理意见
3. 数字应使用阿拉伯数字
4. 日期格式：XXXX年XX月XX日
5. 签名栏不得为空，需由相关人员签字确认
