## 🎯 核心功能

### ✨ 主要能力
- **换服装** - 自然语言描述即可更换衣服（如"把衣服换成bikini"）
- **换背景** - 一键替换背景场景
- **换脸** - 人脸融合和替换
- **风格转换** - 动漫、粘土、油画等多种艺术风格
- **美颜修图** - 磨皮、美白、瘦身等

### 🌟 特色
- **自然语言交互** - 无需学习复杂工具，用普通话描述即可
- **组合操作** - 可同时进行换装+换背景+风格转换
- **高质量输出** - 基于豆包先进模型，输出2048x2048高清图片
- **配额管理** - 灵活的计费系统，支持按次或订阅

## 📋 API集成

### 技术栈
- **API提供商:** 豆包（字节跳动）
- **Endpoint:** https://api.vwu.ai/v1/images/generations/
- **认证:** Bearer Token
- **响应时间:** 10-60秒（取决于复杂度）

### 依赖项
- Python 3.7+
- requests
- pathlib
