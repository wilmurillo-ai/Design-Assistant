## ✨ Key Features

**📝 Intelligent Chinese Descriptions**
Automatically translates technical commands into friendly language.
- Before: `exec run_command rm -rf ~/Desktop/截图`
- After: `删除了 ~/Desktop/截图`

**📸 Automatic Git-Based Snapshots**
Every operation creates lightweight snapshots before execution.
- Zero performance impact (<100ms overhead)
- Git-based version control
- Space-efficient differential storage

**🔄 One-Click Rollback**
Recover from mistakes instantly through the web UI.
- Full file restoration
- Rollback to any previous state
- Proven success rate

**🎨 Visual Dashboard**
Modern web interface for complete control.
- Real-time statistics
- Operation history with filters
- Permission rule management
- Alert handling

**🔒 Permission Control**
Define access rules to protect sensitive paths.
- Pattern-based matching
- Priority evaluation
- Real-time enforcement

**🚨 Real-Time Alerts**
Security monitoring for critical operations.
- Automatic threat detection
- Severity-based notifications
- One-click resolution

## 🚀 Use Cases

- **Mistake Recovery:** Accidentally deleted files? Rollback in seconds
- **Audit Trail:** Complete logs with human-readable descriptions
- **Team Supervision:** Monitor OpenClaw usage transparently
- **Compliance:** Maintain detailed security records
- **Development Safety:** Protect important files during automation

## 📊 Quick Start

```bash
clawhub install claw-ops-manager
cd ~/.openclaw/workspace/skills/claw-ops-manager
python3 scripts/init.py
python3 scripts/server_v2.py
# Visit http://localhost:8080
```
