# Reports — 类型说明

## reportLevelList[].type — 节点类型

| type | 说明 |
|------|------|
| `read` | 接收人（传阅） |
| `suggest` | 建议人（可发表意见） |
| `decide` | 决策人（需同意/不同意） |

`level` 为层级顺序，数字越小越先收到，从 1 开始。

## reportRecordType — 汇报类型

| 值 | 说明 |
|----|------|
| 1 | 工作交流 |
| 2 | 工作指引 |
| 3 | 文件签批 |
| 4 | AI 汇报 |
| 5 | 工作汇报（最常用）|

## myStatus — 当前用户阅读状态

| 值 | 说明 |
|----|------|
| 0 | 未读 |
| 1 | 已读 |
| 2 | 已回复 |

## contentType — 正文类型

| 值 | 说明 |
|----|------|
| `"markdown"` | Markdown 格式（推荐）|
| `"html"` | 富文本 HTML |
