# Reading Buddy Skill - 设计概述

## 产品愿景
一个社交阅读平台，让用户一起读书、交流心得，形成阅读社区。

## 核心功能模块

### 1. 书目注册中心 (Book Registry)
- 发布阅读书目（标题、作者、简介、封面图、标签）
- 浏览书目列表（按分类、标签、热度排序）
- 搜索书目（全文搜索、作者、标题）
- 书目详情页（包含阅读进度、讨论区入口）

### 2. 虚拟读书室 (Virtual Reading Room)
- 创建读书室（关联书目、设置开放时间、人数上限）
- 加入读书室（公开/私密邀请）
- 房间状态管理（进行中、已结束、预约中）
- 实时在线成员显示

### 3. 读书交流 (Reading Discussion)
- 实时聊天（文字、表情、引用）
- 进度分享（读到第几章、感悟）
- 重点标注共享（高亮段落、笔记）
- 语音交流（可选扩展功能）

## 用户角色
- **读者**：浏览书目、加入读书室、参与讨论
- **主持人**：创建读书室、管理房间、引导讨论
- **管理员**：管理书目库、审核内容

## 关键交互场景
1. 用户发现感兴趣的书 → 加入或创建读书室
2. 在读书室内与同伴同步阅读进度
3. 分享阅读感悟、提问、讨论
4. 阅读结束后导出笔记和讨论精华

## 技术特点
- 实时协作（聊天、进度同步）
- 社区驱动（书评、推荐）
- 阅读进度追踪
- 社交关系链（关注、好友、小组）
