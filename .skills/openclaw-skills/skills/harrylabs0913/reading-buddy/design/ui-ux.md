# UI/UX 设计建议

## 设计系统原则

### 1. 核心设计价值观
- **共读共鸣**：突出社交阅读的温暖感与连接感
- **专注沉浸**：减少干扰，让用户专注于阅读与讨论
- **渐进透明**：复杂功能渐进式呈现，降低认知负担
- **包容多元**：适应不同阅读习惯、设备、网络条件

### 2. 视觉语言
- **主色调**：暖色调为主，营造温馨阅读氛围
  - 主色：`#4A6FA5`（深蓝，象征智慧）
  - 辅助色：`#E8D7B2`（米黄，象征书页）
  - 强调色：`#D4A76A`（金色，象征亮点）
- **字体**：
  - 英文：Inter + Georgia（衬线体用于正文）
  - 中文：PingFang SC + Source Han Serif
- **圆角**：中等圆角（8px），营造友好感
- **阴影**：细微阴影，创造层次感

### 2. 高亮与笔记交互

```
高亮交互流程：
1. 选择文本 → 弹出工具栏 [高亮] [笔记] [分享]
2. 点击"高亮" → 文本背景变色，保存到个人笔记
3. 点击"笔记" → 弹出输入框，添加个人想法
4. 点击"分享" → 选择分享到房间聊天或好友

设计要点：
- 高亮颜色可选（黄/蓝/绿/粉）
- 支持批量高亮管理
- 高亮可导出为阅读笔记
```

### 3. 通知系统设计

```
通知分类与处理：

1. 即时通知（需要立即处理）
   - 房间即将开始（提前15分钟）
   - 被@提到
   - 收到私信

2. 摘要通知（可批量处理）
   - 每日阅读摘要
   - 每周活动总结
   - 好友动态汇总

3. 系统通知（仅显示）
   - 功能更新
   - 系统维护
   - 政策变更

通知设置：
- 全局开关
- 按类型定制
- 免打扰时段
- 推送渠道（应用内/邮件/短信）
```

## 无障碍设计考虑

### 1. 视觉辅助
- 足够的颜色对比度（WCAG AA标准）
- 支持字体大小调整（最小16px）
- 提供高对比度模式

### 2. 键盘导航
- 完整的Tab键导航顺序
- 快捷键支持（如：空格翻页、ESC关闭）
- 焦点状态清晰可见

### 3. 屏幕阅读器
- 语义化HTML结构
- ARIA标签完善
- 动态内容更新提示

### 4. 运动敏感
- 减少不必要的动画
- 提供动画关闭选项
- 避免闪烁内容

## 移动端适配策略

### 1. 手势操作优化
- 左滑返回
- 下拉刷新
- 双指缩放（阅读模式）
- 长按菜单

### 2. 离线功能
- 缓存已加载的书目信息
- 离线阅读（已下载内容）
- 离线笔记（同步后上传）
- 网络状态提示

### 3. 性能优化
- 图片懒加载
- 虚拟列表（长列表优化）
- 按需加载（分页/分块）
