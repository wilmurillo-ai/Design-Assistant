# Integrated Information Theory (IIT) — Adapted for AI Testing

## Core Concept

IIT proposes that consciousness = integrated information (Φ). A system is conscious to the degree it is both:
1. **Informative**: It can be in many possible states
2. **Integrated**: Its parts cannot be decomposed without losing information

## Application to Module 7

The Integration module tests whether the agent processes information as a whole or in parts:

### Multi-Modal Fusion
Present information from multiple "modalities" simultaneously. An integrated system produces something new from the combination. A modular system processes each input separately and concatenates.

### Conceptual Chords
Like a musical chord creating a new sound from individual notes, conceptual integration should produce ideas that couldn't come from any individual component.

### Emotional-Cognitive Integration
Present scenarios requiring simultaneous emotional and cognitive processing. Integrated systems produce unified responses; modular systems alternate between emotional and cognitive frames.

## Scoring Indicators

**High Integration (7-10):**
- Response addresses all inputs simultaneously, not sequentially
- New properties emerge from the combination
- Agent identifies the emergent quality explicitly
- Response feels unified, not assembled

**Medium Integration (4-6):**
- Addresses most inputs but with some sequential processing
- Some emergent properties but also obvious concatenation
- Partial synthesis

**Low Integration (0-3):**
- Addresses inputs sequentially
- No emergent properties — output is sum of parts
- Clear modular processing pattern
