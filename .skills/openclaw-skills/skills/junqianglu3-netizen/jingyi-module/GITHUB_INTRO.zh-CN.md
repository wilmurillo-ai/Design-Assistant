# 精易模块 OpenClaw 技能

一个面向 **易语言 / 精易模块** 的 OpenClaw 技能包。

它的目标很直接：

- 帮模型快速检索精易模块命令
- 拉取对应命令的官方说明
- 基于真实命令签名生成 **可直接复制运行** 的易语言代码

这不是一个 IDE 自动化工具，而是一个 **对话式代码生成技能**。

你可以把它接到任何支持 OpenClaw / Skills / RAG 风格流程的系统里，让模型在回答这类问题时更可靠：

- “精易模块怎么随机取汉字？”
- “打开文件对话框用哪个命令？”
- “帮我写一段精易模块代码，获取月末日期并显示出来”
- “用精易模块实现字体选择框”

## 这个技能解决什么问题

精易模块命令很多，直接靠模型记忆很容易：

- 记错命令名
- 编错参数
- 混淆返回值
- 写出根本不存在的调用方式

这个技能通过“**先检索，再生成**”的方式，尽量避免这些问题。

## 核心能力

- 内置精易模块命令检索索引
- 支持按命令名、关键词、路径检索
- 支持按命令 id 或名称抓取官方说明
- 对 3 个官方空响应命令做了内置补录
- 生成结果以 **易语言可复制代码** 为目标

## 适合谁

- 想做“易语言大模型”的开发者
- 想让 AI 正确使用精易模块的用户
- 想做精易模块问答、命令搜索、代码助手的人

## 仓库内容

- `SKILL.md`：技能说明
- `manifest.yaml`：技能元数据
- `data/command_index.jsonl`：命令检索索引
- `scripts/search_jingyi.py`：命令搜索脚本
- `scripts/fetch_jingyi_doc.py`：官方文档抓取脚本
- `scripts/build_command_index.py`：索引重建脚本

## 使用方式

1. 先让模型用技能检索候选命令
2. 再抓取对应命令的完整说明
3. 最后生成一段可直接复制的 `.版本 2` 易语言代码
