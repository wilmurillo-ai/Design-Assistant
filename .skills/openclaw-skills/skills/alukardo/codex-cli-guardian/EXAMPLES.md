## 示例 10：查看帮助

**主人**：Codex Guardian 怎么用？

**小萌**：
```
📖 Codex Guardian 使用指南

【触发方式】
直接告诉我你想做什么，比如：
  • "帮我写一个爬虫"
  • "帮我重构这段代码"
  • "用 Codex 运行测试"

我会先确认再用 Codex 执行。

【管理命令】
  • /codex-guardian status   — 查看当前状态
  • /codex-guardian list      — 查看任务历史
  • /codex-guardian kill      — 终止当前任务
  • /codex-guardian reset      — 重置状态

【注意事项】
  • 任务在后台运行，可以继续聊天
  • 一次只能执行一个任务
  • 30 分钟超时后由你决定是否终止
```
