# 教学工作手册生成规则 v1.0

> **Tool Wrapper Pattern**: 当生成教学工作手册时加载此文档

---

## 📋 占位符清单

| 占位符 | 含义 | 示例 |
|--------|------|------|
| `$as$` | 学年 | `2025 - 2026` |
| `$ay$` | 学年（兼容） | `2025 - 2026` |
| `$sm$` | 学期 | `二` |
| `$g$` | 年级 | `23` |
| `$mj$` | 专业（C#模板） | `软工` |
| `$major$` | 专业（数据可视化模板） | `计算机` |
| `$g$$mj$` | 复合：年级+专业 | `23软工` |
| `$tot$` | 总人数 | `48` |
| `$v$` | 平均成绩 | `78.5` |
| `$y$` `$m$` `$d$` | 年月日 | `2026` `3` `28` |

## 📐 表格结构

### 作业表（学生作业登记表）
- **检测**: 表头包含"作业1"
- **结构**: 16列
  - cells[0]: 学号
  - cells[1]: 姓名
  - cells[2-14]: 作业1-13（填√或空）
- **分页**: 每表21行（1表头+20数据），不足20学生时留空行

### 成绩表（学生课程成绩登记表）
- **检测**: 表头包含"平时成绩"
- **结构**: 14列
  - cells[0]: 学号
  - cells[1]: 姓名
  - cells[8]: 平时成绩
  - cells[9]: 期中成绩
  - cells[10]: 期末成绩
  - cells[11]: 总评
- **分页**: 每表21行（1表头+20数据），不足20学生时留空行

## 🔧 文本替换规则

### 优先级
1. **复合占位符优先**: `$g$$mj$` 在 `$mj$` 和 `$g$` 之前匹配
2. **逐Run处理**: 保持原有下划线格式
3. **跨Run检测**: 如 `$` 和 `major$` 分开成两个Run，需合并后替换

### Run结构示例
```
模板: '$' + 'major$' → 占位符 '$major$'
替换: '$major$' → '软工'
```

## ✨ 单元格格式

### 居中对齐
- **水平居中**: `w:jc val="center"`
- **垂直居中**: `w:vAlign val="center"`
- **同时设置**: 单元格 tcPr 和段落 pPr

## 📊 分页逻辑

```
总学生数 = 48
每页行数 = 20

表1: 学生 1-20
表2: 学生 21-40
表3: 学生 41-48 (留空12行)
```

## ⚠️ 常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| 下划线消失 | 替换时清空了Run格式 | 逐Run替换，保持格式 |
| 占位符未替换 | Run被拆分 | 合并检测跨Run占位符 |
| 连续空行 | 分页计算错误 | 每表固定21行，不足留空 |
