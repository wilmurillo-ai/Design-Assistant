## Weekly Review Checklist

- [ ] Scan memory for repeated patterns (3+ occurrences)
- [ ] Review decisions >7 days old → follow up
- [ ] Identify 1-2 proactive builds that would delight human
- [ ] Check for automation opportunities
- [ ] Update this tracker

---

## Reverse Prompting Triggers

Ask these questions proactively:

1. "What are some interesting things I can do for you based on what I know?"
2. "What information would help me be more useful to you?"
3. "Is there anything repetitive you'd like me to automate?"

**Schedule:** Ask once per week, or when you notice frustration/repetition.
