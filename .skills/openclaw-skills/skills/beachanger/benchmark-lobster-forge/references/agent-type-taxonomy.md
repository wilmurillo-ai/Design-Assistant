# 小龙虾类型划分

## 1. 顾问虾
适合：判断、建议、方案设计、策略分析。

## 2. 执行虾
适合：动作执行、调用工具、跑流程、完成任务。

## 3. 流程虾
适合：分阶段推进、SOP 驱动、节点检查。

## 4. 教练虾
适合：长期陪伴、行为改变、反馈、提醒、打卡。

## 5. 分身虾
适合：人格化代理、长期协作、替身沟通。

## 6. 工作流虾
适合：多步骤编排、信息整合、结构化输出物。

---

## 用法

先判断“主要价值来源”是什么，再定虾型。
虾型不是为了贴标签，而是为了决定：
- memory 怎么设计
- 是否强调长期留存
- 是否需要技能编排
- 输出更偏建议、执行还是持续推进
