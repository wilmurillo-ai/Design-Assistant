# Aesthetic Presets

Starting points for your brand's visual identity. Pick one during install
and the Art Director skill will copy it to your workspace as `aesthetic.md`.
Edit freely from there — the preset is a template, not a runtime value.

## Included presets

- **documentary.md** — Desaturated photography, observed rather than staged,
  feature-magazine sensibility. For brands that want an observed, quietly
  authoritative feel.

- **conceptual-illustration.md** — Painterly, metaphor-first illustration
  with traditional-media texture. For essay, criticism, culture, and opinion
  content.

- **product-render.md** — Architectural 3D renders, precise composition,
  single-accent palette. For SaaS, infrastructure, developer tools, and
  product marketing.

- **product-photo.md** — Real photography of real objects on seamless
  backdrops, controlled studio lighting, quiet product reverence. Apple /
  Teenage Engineering / Aesop / Leica register. For premium consumer
  brands and lifestyle publications.

- **schematic.md** — Precise ink linework over paper, orthographic or
  exploded-view, single accent color. For engineering, manuals, technical
  documentation, and data-heavy publications.

- **editorial-collage.md** — Torn-magazine scraps arranged into composite
  scenes, visible paper texture, CMYK drift, halftone dots. For features,
  culture magazines, and longform with editorial bite.

- **synthwave.md** — 80s retrofuturist, magenta-pink and electric cyan
  against indigo grounds, chrome reflections, VHS texture, after-midnight
  mood. For indie games, music brands, podcasts, retro-tech, and
  night-economy brands.

- **phosphor.md** — Monochrome CRT / low-bit computer graphics. Green
  phosphor default with latitude for amber, duotone dither, teal
  monochrome. Visible pixels, scan lines, CRT bloom. For indie games,
  retro-computing brands, terminal-aesthetic publications.

- **orbital.md** — Bold flat vector graphics in the mid-century modernist
  poster tradition with a space-travel / scientific-diagram sensibility.
  2-4 color palettes, flat fills, cosmic geometry, subtle print texture.
  Simon C. Page / Brainstorm / NASA travel posters. For science writing,
  publications with a poster-design sensibility, culture brands.

## Choosing

If none of these feel right, start blank (`aesthetic.md` has inline guidance
in comments) and write your own from scratch. The template is deliberately
short — aesthetic configs get better through iteration, not by being
specified exhaustively up front.

## Iterating

After install, generate a batch of 10-20 images against your current
aesthetic with varied briefs. Review them side by side. Tune the config.
Regenerate. You'll reach "this looks like us" in an evening.

The skill provides a batch command for exactly this loop — see the main
README.
