## 📜 简介

**帝王会议**是一个基于毛泽东选集核心思想的 Multi-Agent 决策系统。通过模拟 5 位历史人物（毛主席 + 4 位帝王）的集体智慧，为用户提供战略分析、矛盾分析、组织设计等决策支持。

**核心理念：** 将毛选哲学（矛盾论、实践论、群众路线）与历史帝王治国智慧相结合，形成独特的决策分析框架。

---

## ✨ 特性

- 🎯 **5 位历史人物 Agent**：毛主席（会议主席）+ 秦始皇 + 汉武帝 + 唐太宗 + 朱元璋
- 🧠 **毛选思想核心**：矛盾论、实践论、群众路线、实事求是
- 📊 **结构化输出**：矛盾分析 → 多方观点 → 统一决策 → 执行计划
- 💬 **生动语言**：保持各帝王的历史语言风格
- 🔧 **可组合**：可与其他 Skills（如 Guilliman 战略顾问）组合使用

## 💡 使用场景

### ✅ 适合的问题

**战略规划类：**
- 公司战略方向选择
- 个人职业发展规划
- 产品路线图设计

**组织设计类：**
- 团队架构优化
- 跨部门协作问题
- 管理制度设计

**矛盾分析类：**
- 多方利益冲突
- 资源分配矛盾
- 短期 vs 长期平衡

**决策支持类：**
- 重大决策风险评估
- 多方案对比选择
- 执行计划制定

### ❌ 不适合的问题

- 简单事实查询（用搜索引擎）
- 技术问题调试（用 coding-agent）
- 情感安慰（用心理咨询 agent）
- 实时信息查询（用 search tool）

## 📖 示例

### 示例 1：公司管理问题

**提问：**
> "帝王会议，公司部门之间协作不畅，各自为政，如何改善？"

**输出要点：**
- 毛主席：主要矛盾（部门局部利益 vs 公司整体利益）
- 秦始皇：统一制度、设立 PMO
- 汉武帝：主动出击、建立互访机制
- 唐太宗：干部轮岗、跨部门委员会
- 朱元璋：发动员工、建立投诉机制
- 统一决策：改革考核机制 + 4 步执行计划

### 示例 2：个人职业发展

**提问：**
> "帝王会议，我是技术出身，想转管理，但担心失去技术优势，怎么办？"

**输出要点：**
- 毛主席：技术深度 vs 管理广度的矛盾
- 秦始皇：建立个人发展制度（70% 管理 +30% 技术）
- 汉武帝：主动承担管理职责、建立技术影响力
- 唐太宗：培养技术骨干、找导师指导
- 朱元璋：从团队中来，到团队中去
- 统一决策：辩证统一 + 3 阶段执行计划

## 🔧 技术实现

### Agent 编排
- **框架：** OpenClaw Multi-Agent 系统
- **协调机制：** 毛主席 Agent 作为会议主席
- **决策流程：** 矛盾分析 → 多方讨论 → 整合决策 → 执行计划

### 核心思想来源
- **《矛盾论》**：矛盾的普遍性与特殊性
- **《实践论》**：实践→认识→再实践→再认识
- **群众路线**：从群众中来，到群众中去
- **历史帝王智慧**：秦皇汉武唐宗宋祖（宋祖未加入）
