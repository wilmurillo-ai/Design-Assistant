## 配置文件说明

### config/accounts.json - 账号管理

```json
{
  "accounts": [
    {
      "id": "acc_001",
      "name": "账号名称",
      "status": "active",
      "lastLogin": "2026-04-02T10:00:00+08:00"
    }
  ],
  "activeAccountId": "acc_001"
}
```

### config/keyword_rules.json - 关键词规则

```json
{
  "keywordRules": [
    {
      "id": "kr_001",
      "enabled": true,
      "keywords": ["多少钱", "价格"],
      "matchType": "any",
      "reply": "欢迎咨询价格~",
      "priority": 1
    }
  ],
  "defaultAction": "ai_generate"
}
```

**匹配类型 (matchType)：**
- `any` - 任一关键词匹配即可
- `all` - 所有关键词都必须匹配
- `exact` - 精确匹配

### config/sensitive_words.json - 敏感词库

```json
{
  "sensitiveWords": {
    "severe": ["违禁词"],
    "warning": ["加微信", "加V"],
    "notice": ["竞品名"]
  },
  "actionRules": {
    "severe": { "action": "auto_delete", "notify": true },
    "warning": { "action": "mark_pending", "notify": true },
    "notice": { "action": "log_only", "notify": false }
  }
}
```

**敏感级别：**
- `severe` - 严重：自动跳过/删除
- `warning` - 警告：标记待人工审核
- `notice` - 提示：记录日志

### config/ai_config.json - AI模型配置

```json
{
  "defaultModel": "default",
  "models": {
    "deepseek": {
      "provider": "deepseek",
      "model": "deepseek-chat",
      "enabled": false,
      "apiKey": null
    }
  },
  "generationConfig": {
    "temperature": 0.7,
    "maxTokens": 150
  }
}
```

### config/monitor_state.json - 监控状态

```json
{
  "status": "running",
  "lastCheck": "2026-04-02T10:00:00+08:00",
  "processedComments": ["comment_id_1", "comment_id_2"],
  "stats": {
    "totalProcessed": 100,
    "totalReplied": 95
  }
}
```

### config/schedule_config.json - 定时开关

```json
{
  "enabled": true,
  "schedules": [
    {
      "name": "工作时间",
      "days": [1, 2, 3, 4, 5],  // 1=周一, 0=周日
      "timeSlots": [
        { "start": "09:00", "end": "12:00", "action": "auto_reply" },
        { "start": "14:00", "end": "18:00", "action": "auto_reply" }
      ]
    },
    {
      "name": "夜间静默",
      "days": [0, 1, 2, 3, 4, 5, 6],
      "timeSlots": [
        { "start": "23:00", "end": "08:00", "action": "log_only" }
      ]
    }
  ],
  "defaultAction": "log_only"
}
```

**动作类型：**
- `auto_reply` - 自动回复
- `log_only` - 仅记录不回复
- `pause` - 完全暂停监控

## 回复原则

对所有用户评论，智能分析后生成回复，遵循以下原则：

| 原则 | 说明 | 示例 |
|------|------|------|
| 积极 | 正面回应，传递正能量 | "感谢支持，会继续加油！" |
| 大方 | 不卑不亢，得体有度 | "感谢关注，欢迎常来互动~" |
| 热情 | 亲切友好，拉近距离 | "哈哈谢谢宝子，爱你哟❤️" |
| 合法 | 遵守法律法规和平台规范 | 不涉及敏感词、广告法违禁词 |

## ⚠️ 平台限制

1. **每条评论只能回复一次** - 抖音平台限制每条评论只能有一条作者回复
2. **回复要"一次到位"** - 由于只能回复一次，内容要完整有效

---

## 注意事项

1. **频率控制** - 避免过于频繁的刷新和回复，可能触发风控
2. **回复延迟** - 建议在收到消息后等待几秒再回复，模拟真人行为
3. **内容审核** - 回复内容需符合平台规范，避免敏感词
4. **登录状态** - 长时间运行可能需要重新登录
5. **一次回复原则** - 每条评论只有一次回复机会，内容要完整有效
6. **必须使用嵌套输入框** - 点击"回复"按钮后展开的输入框才是正确的

## 常见问题

### Q: 如何添加自定义关键词规则？

A: 对 Agent 说：`添加关键词规则：当评论包含"XXX"时，回复"YYY"`

### Q: 如何开启敏感词检测？

A: 默认已开启，可通过 `显示敏感词配置` 查看当前设置。

### Q: 评论被标记为待人工审核怎么办？

A: 在监控状态中查看 `pendingManualReview` 列表，手动处理这些评论。

### Q: 如何切换 AI 模型？

A: 编辑 `config/ai_config.json`，设置 `defaultModel` 和对应模型的 `enabled: true`。
