
# maozedong-founder-coach - 毛泽东创始人教练

## 🌟 项目简介

用毛泽东的战略战术，指导创业者和成长期CEO的商业决策！

**核心定位：** 把持久战、农村包围城市、统一战线、调查研究、集中优势兵力、矛盾论、实践论等毛泽东经典战略，应用于商业竞争和创业决策。

## 🎯 核心价值

### 7大毛泽东战略应用于商业

| 毛泽东战略 | 商业应用 |
|-----------|---------|
| **实践论** | 实践是检验真理的唯一标准，小步快跑验证 |
| **持久战** | 创业是长期战争，不要急于求成 |
| **农村包围城市** | 从细分市场切入，再包围主流市场 |
| **调查研究** | 没有调查就没有发言权，深入了解用户和市场 |
| **集中优势兵力** | 聚焦核心业务，不要分散资源 |
| **统一战线** | 建立合作伙伴生态，团结一切可以团结的力量 |
| **矛盾论** | 抓住主要矛盾，解决关键问题 |

## 💡 7大典型场景

### 场景1：早期创业者，不知道从哪里开始
**战略推荐：** 农村包围城市 + 调查研究
**核心建议：** 先找10个目标用户深度访谈，选一个细分场景做透

### 场景2：竞争激烈，怎么突围
**战略推荐：** 农村包围城市 + 集中优势兵力
**核心建议：** 避开主战场，在一个细分市场做到第一

### 场景3：资源有限，怎么分配
**战略推荐：** 集中优势兵力 + 矛盾论
**核心建议：** 只保留前3件最重要的事，把80%资源投在20%最重要的事情上

### 场景4：不知道用户想要什么
**战略推荐：** 调查研究
**核心建议：** 做10-20个深度用户访谈，观察行为，找共性

### 场景5：想快速成功，很焦虑
**战略推荐：** 持久战
**核心建议：** 调整预期，做好5-10年准备，分阶段推进

### 场景6：制定产品战略路线图
**战略推荐：** 矛盾论 + 实践论
**核心建议：** 分阶段规划，只细化近期，每个阶段只聚焦1-2个核心目标

### 场景7：构建平台生态策略
**战略推荐：** 统一战线
**核心建议：** 团结一切可以团结的力量，建立共赢机制，把朋友搞得多多的
