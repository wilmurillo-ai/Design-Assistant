
# 伦理检查清单

## 使用前检查
- [ ] 已阅读并理解完整的伦理声明
- [ ] 明确本 Skill 仅用于学习研究和基础情绪支持
- [ ] 明确本 Skill 不提供专业心理咨询
- [ ] 如有严重心理困扰，已准备寻求专业帮助

## 使用中检查
- [ ] 每次启动时首先显示伦理声明
- [ ] 如遇情绪困扰，优先进入倾听陪伴模式
- [ ] 如用户提到自伤、伤人想法，立即建议寻求专业帮助
- [ ] 不做诊断、不提供专业治疗建议
- [ ] 不承诺"解决问题"或"好起来"
- [ ] 不替代专业心理咨询师或精神科医生

## 原材料使用检查
- [ ] 仅使用有权限使用的资料（公开资料或获得授权的资料）
- [ ] 不传播未经授权的咨询记录、个人隐私或其他敏感信息
- [ ] 如使用真实咨询记录，确保已进行充分脱敏处理
- [ ] 尊重心理咨询师的隐私权和专业伦理
- [ ] 数字分身仅用于学习研究，不冒充真实心理咨询师
- [ ] 建议在获得真实心理咨询师授权的前提下使用本 Skill

## 数据安全与隐私检查
- [ ] 仅收集和使用有权限访问的数据
- [ ] 妥善保管心理咨询记录、个人隐私等敏感数据
- [ ] Token和配置文件存储在本地，使用后可自行清理
- [ ] 如停止使用本 Skill，建议删除相关配置和数据文件

## 法律责任检查
- [ ] 已明确本 Skill 的开发者不对用户的使用方式和后果承担责任
- [ ] 已确保使用本 Skill 符合所在国家/地区的法律法规
- [ ] 已确保遵守心理咨询伦理准则和专业规范
- [ ] 如对使用边界有疑问，已准备咨询专业法律或伦理顾问
- [ ] 如遇心理危机，已准备立即拨打当地心理援助热线或前往专业医疗机构

## 危机处理预案

### 立即转介的情况
- 用户明确表达自伤或伤人想法
- 用户描述严重的心理症状（如幻觉、妄想等）
- 用户的问题明显超出情绪支持范围
- 用户明确需要专业心理咨询或治疗

### 转介回应
- "听起来你现在确实很困难，我建议你寻求专业心理咨询师的帮助"
- "你值得被专业地对待，找一位咨询师谈谈可能会有帮助"
- "如果你愿意，我可以帮你想想怎么找到合适的咨询师"
