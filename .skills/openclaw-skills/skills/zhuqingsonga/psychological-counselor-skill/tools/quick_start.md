
# 快速上手指南

## 3 步创建你的第一个心理咨询师数字分身

### 第 1 步：基础信息（3 个问题）
1. **代号/花名（必填）** - 给这位咨询师起个名字
2. **基本信息（一句话）** - 姓名、流派、受训背景
3. **你想从这个 skill 获得什么？** - 学习风格？倾听陪伴？还是都要？

就这 3 个！其他都可以跳过。

---

### 第 2 步：原材料导入（3 种方式）

**方式 A：网页链接（推荐）**
- 直接给咨询师的个人主页、简介页、文章链接
- 自动抓取内容，快速生成

**方式 B：上传文件**
- PDF / Word / 图片 / 文本文件
- 支持多种格式

**方式 C：直接粘贴内容**
- 把咨询师的简介、受训背景、文章、访谈复制进来
- 快速、灵活

可以混用，也可以跳过（仅凭手动信息生成）。

---

### 第 3 步：确认 & 使用
- 自动分析、自动生成
- 确认后立即使用！

---

## 3 种使用模式（自动判断）

### 1️⃣ 咨询师风格模拟
- "如果是这位咨询师，他会怎么回应？"
- "这位咨询师的咨询风格是什么样的？"

### 2️⃣ 专业学习辅助
- "这位咨询师是怎么做个案概念化的？"
- "这位咨询师的理论取向是什么？"

### 3️⃣ 基础倾听陪伴
- "我现在心情不好，能陪我说说话吗？"
- "我有个困惑，想听听你的看法"

- `/counselor-rollback {slug} {version}` - 回滚到指定版本
- `/delete-counselor {slug}` - 删除指定咨询师
