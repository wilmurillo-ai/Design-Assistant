
# create-psychological-counselor - 心理咨询师.skill 创建器

## 🌟 项目简介

根据心理咨询师的教育背景、受训背景、咨询记录等材料，构建心理咨询师的数字分身，用于学习研究、风格模拟、基础倾听陪伴。

## 🎯 核心功能

- **咨询师风格模拟** - "如果是这位咨询师，他会怎么回应？"
- **专业学习辅助** - "这位咨询师的理论取向是什么？咨询风格是怎样的？"
- **基础倾听陪伴** - "我心情不好，能陪我说说话吗？"

## ⚠️ 重要伦理声明

⚠️ **使用边界声明**：
- 本 Skill 仅用于个人学习、研究、培训和情绪支持
- **不提供专业心理咨询、心理治疗或精神科诊断服务**
- 如遇严重心理困扰、自伤或伤人风险，请立即寻求专业医疗机构帮助

## 📝 使用流程

### Step 1: 基础信息录入（3个问题）

1. **代号/花名（必填）**
2. **基本信息（一句话）**：姓名、流派、受训背景、一句话描述
3. **你想从这个skill获得什么？**

### Step 2: 原材料导入（三种方式）

- **网页链接** - 咨询师的个人主页、简介页、文章链接
- **上传文件** - PDF / Word / 图片 / 文本文件
- **直接粘贴内容** - 把咨询师的简介、受训背景、文章、访谈复制进来

### Step 3: 自动生成 & 使用

自动分析、自动生成、立即使用！

## 💡 支持的原材料类型

- 教育背景（学校、专业、学位）
- 受训背景（长程培训、督导、个人体验）
- 咨询记录（脱敏后的咨询对话）
- 个案概念化（对个案的分析和理解）
- 个人访谈（媒体采访、视频、播客）
- 督导记录（接受督导的记录）
- 公开文章（发表的专业文章）
- 社交平台（微博、公众号、知乎等）

## 📁 文件结构

## 🔧 管理命令

- `/list-counselors` - 列出所有已生成的咨询师
- `/update-counselor {slug}` - 更新指定咨询师
- `/counselor-rollback {slug} {version}` - 回滚到指定版本
- `/delete-counselor {slug}` - 删除指定咨询师

## 🎨 设计理念

- **低门槛** - 只需要3个问题，给链接或文档就能用
- **伦理优先** - 严格遵守心理咨询伦理，边界清晰
- **智能判断** - 自动判断用户意图，无需选择模式
- **下载量过万设计** - 50+ 触发词，强引导分享
