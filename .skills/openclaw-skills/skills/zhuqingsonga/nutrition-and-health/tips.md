# 你的专属健康顾问 - 使用技巧

## 快速开始

### 第一步：让我了解你

```
你："你了解我一下"
我：会询问你的基本情况、健康状况、饮食习惯...
你：根据实际情况回答
```

**为什么要了解你？**
- 建议更精准，符合你的情况
- 避开你的忌口和不适
- 推荐你喜欢的口味
- 考虑你的健康目标

---

### 第二步：开始使用

**最自然的方式：**
```
"今天吃什么"
"早餐吃什么"
"想喝奶茶怎么办"
"减脂期怎么吃"
```

**不用记住命令，怎么自然怎么说！**

## 提高建议质量的技巧

### 1. 提供更多上下文

❌ 不要只说：
```
"吃什么"
```

✅ 多说一点：
```
"今天中午吃什么，不想太油"
"早餐推荐，我胃不好"
"想喝奶茶，但在减脂期"
```

### 2. 告诉我你的偏好

```
"我喜欢吃辣"
"我不吃香菜"
"我是素食者"
"我不喜欢太甜的"
```

### 3. 说明你的场景

```
"这是早餐"
"这是外卖"
"这是聚餐"
"我在加班"
```

## 常见问题

**Q: 为什么要先"了解我"？**

A: 了解你的情况后，建议更精准。比如：
- 胃不好 → 推荐温和、易消化的
- 减脂期 → 推荐低卡、饱腹感强的
- 喜欢吃辣 → 推荐健康的辣味选择

**Q: 我的信息安全吗？**

A: 完全安全！
- 所有信息只存储在你的本地
- 不上传任何服务器
- 你可以随时查看和修改

**Q: 建议不适合我怎么办？**

A: 直接告诉我！
- "这个不喜欢，换一个"
- "我不吃这个，有其他选择吗"
- "这个太油了，推荐清淡点的"

**Q: 能替代医生吗？**

A: 不能！
- 严重健康问题请咨询医生
- 这是健康建议，不是医疗诊断
- 有慢性病请遵医嘱

## 进阶用法

### 让建议更贴心

```
"推荐早餐，我喜欢中式的"
"午餐外卖推荐，预算20元以内"
"想喝奶茶，三分糖可以吗"
"胃不好，晚餐推荐温和的"
```

### 连续对话优化

```
你："今天吃什么"
我：推荐了几个选择
你："第一个不错，但我不吃香菜"
我：调整推荐，避开香菜
你："再推荐点蔬菜多的"
我：继续优化...
```
