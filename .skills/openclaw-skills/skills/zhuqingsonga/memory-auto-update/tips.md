# 记忆自动更新 - 使用技巧

## 快速开始

### 第一步：选择你的模式

```
你："设置更新模式"
我：展示三种模式供选择
你：选择最适合你的
```

**推荐：**
- 新手 → 主动模式（自动提示，不会忘）
- 老手 → 被动模式（完全控制，不打扰）
- 大多数人 → 智能混合模式（推荐，省心）

---

### 第二步：开始使用

**最简单的方式：**
```
"这个很重要，记下来"
"更新记忆"
"今天就这样"
```

**不用记住命令，怎么自然怎么说！**

## 模式选择指南

### 主动模式（推荐新手）

**特点：**
- 对话结束自动提示
- 不会忘记更新记忆
- 适合刚开始使用

**适合人群：**
- 经常忘记更新记忆
- 喜欢被提醒
- 刚开始使用这个技能

**触发示例：**
```
你："今天就这样"
我："好的！这是今天的重要内容..."
```

---

### 被动模式（推荐老手）

**特点：**
- 完全不主动打扰
- 你想更新时才更新
- 高效，不打断工作

**适合人群：**
- 喜欢自己掌控
- 工作时不想被打扰
- 熟练使用这个技能

**触发示例：**
```
你："更新记忆"
我："好的，马上提取！"
```

---

### 智能混合模式（默认推荐）

**特点：**
- 重要内容立即提醒
- 普通内容不打扰
- 平衡省心和高效

**适合人群：**
- 大多数人
- 想要省心但不想被频繁打扰
- 重要决策多，日常对话也多

**触发示例：**
```
（重要内容时）
我："这个很重要，要记下来吗？"

（对话结束时）
我："需要总结一下今天的内容吗？"
```

---

## 常见问题

**Q: 我总是忘记更新记忆怎么办？**

A: 选择"主动模式"，对话结束时会自动提醒你！

**Q: 不想被打扰怎么办？**

A: 选择"被动模式"，完全由你控制何时更新。

**Q: 我吐槽"你忘了吗"真的有用吗？**

A: 当然！这是专门设计的触发词，我会立即检查并更新记忆。

**Q: 识别的内容不对怎么办？**

A: 没关系！生成摘要后你可以修改、删除、补充，确认后再保存。

**Q: 会读取我的历史对话吗？**

A: 不会！只处理当前会话的内容，保护你的隐私。

**Q: 可以同时在多个对话中使用吗？**

A: 可以！每个会话独立处理，但保存到同一个记忆文件。

## 进阶用法

### 让识别更准确

**多说一点上下文：**

❌ 不要只说：
```
"这个很重要"
```

✅ 多说一点：
```
"这个很重要：下周三下午3点开会"
"记下来：预算是 10 万"
"别忘了：@小明 周五前完成"
```

### 连续对话优化

**对话中自然触发：**

```
你："我们决定用方案 A"
我："这个很重要，要记下来吗？"
你："要！"
我："好的，记下来了！"

（继续对话...）

你："今天就这样"
我："好的！这是今天的其他重要内容..."
```
