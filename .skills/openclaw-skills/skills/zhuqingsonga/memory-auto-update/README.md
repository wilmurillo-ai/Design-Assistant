# Memory Auto Update 记忆自动更新

> 智能识别重要内容，自动更新记忆，再也不用担心忘记了。

## ✨ 特色

- 🧠 **智能识别**：自动提取对话中的决策、待办、约定、事实
- 🎯 **吐槽触发**："你忘了吗"、"你怎么不记得" → 自动更新
- 🔄 **三种模式**：主动模式、被动模式、智能混合模式（推荐）
- ⏰ **五种频率**：实时、30分钟、1小时、手动、对话结束
- ⚙️ **用户可配置**：自由调整最适合的方式
- 🔒 **隐私第一**：本地存储，不上传任何数据
