# ICSD-3 / DSM-5 睡眠障碍分类映射

## ICSD-3（国际睡眠障碍分类第三版）

ICSD-3 由美国睡眠医学会（AASM）发布，是当前最权威的睡眠障碍分类系统。

---

### 主要分类概览

| 大类 | 代表疾病 | 核心特征 |
|------|----------|----------|
| 失眠障碍（Insomnia Disorder）| 慢性失眠 | 主观报告睡眠困难+日间功能障碍 |
| 睡眠呼吸障碍 | 阻塞性睡眠呼吸暂停（OSA）| 呼吸暂停+血氧下降 |
| 中枢性嗜睡症 | 嗜睡症（Narcolepsy）| 日间过度嗜睡，无法控制入睡 |
| 昼夜节律失调 | 睡眠时相延迟/提前 | 内源性生物钟与环境不同步 |
| 异态睡眠 | 睡行症、REM睡眠行为障碍 | 睡眠中异常行为 |
| 睡眠相关运动障碍 | 不宁腿综合征（RLS）| 夜间腿部不适+运动冲动 |

---

## 失眠障碍（Insomnia Disorder）

### DSM-5 定义

**核心标准**（需同时满足A、B、C）：

**A. 主要症状**（以下至少一项）：
- 入睡困难
- 维持睡眠困难（频繁觉醒或觉醒后难以再入睡）
- 比期望时间早醒且无法再入睡

**B. 日间功能障碍**（以下至少一项）：
- 疲劳/乏力
- 注意力、专注力或记忆力下降
- 社会/职业/学业功能受损
- 情绪不稳定/易激惹
- 日间嗜睡
- 动力/精力/主动性下降
- 错误/事故倾向增加
- 对睡眠担忧或不满

**C. 频率标准**：
- 症状出现频率：≥ 3夜/周
- 持续时间：≥ 3个月
- 单独出现或与其他睡眠障碍、精神障碍、躯体疾病或药物物质相关

**严重程度分级**：
| 分级 | 睡眠效率 | 日间功能影响 |
|------|----------|-------------|
| 轻度 | 85-90% | 轻微影响 |
| 中度 | 80-85% | 中度影响，某一领域明显受损 |
| 重度 | < 80% | 严重受损，多个领域受损 |

---

### ICSD-3 失眠亚型

| 亚型 | ICSD-3编码 | 特征 | 华为指标关联 |
|------|------------|------|-------------|
| 适应性失眠（短期失眠）| 急性失眠 | < 3个月，应激源消除可缓解 | 入睡困难为主 |
| 慢性失眠障碍 | 失眠障碍 | ≥ 3个月，每周≥3夜 | 多指标异常 |
| 其他失眠障碍 | 特指型 | 不符合上述分类 | 需综合评估 |

## 失眠的神经生理机制（华为指标映射）

### 过度觉醒模型（Hyperarousal Model）

**核心机制**：失眠患者24小时处于高觉醒状态，包括：
- 脑区激活：前额叶皮层、前扣带回、岛叶持续激活
- 神经递质：皮质醇、肾上腺素、去甲肾上腺素水平升高
- HPA轴：下丘脑-垂体-肾上腺轴过度活跃

**华为指标映射**：
| 神经指标 | 华为指标 | 异常表现 |
|----------|----------|----------|
| 高皮质醇 | 早醒（endTime<06:00）| 皮质醇早峰 |
| 肾上腺素↑ | 觉醒次数↑ | 交感神经激活 |
| 过度觉醒 | sleep_latency↑ | 入睡困难 |

---

### 3P模型（Spielman模型）

失眠发生的三大因素：

| 因素 | 说明 | 华为指标 |
|------|------|----------|
| Predisposing（易感因素）| 失眠倾向（年龄、性格特质）| sleep_latency基础水平 |
| Precipitating（促发因素）| 应激事件（压力、疾病）| 急性失眠起始 |
| Perpetuating（维持因素）| 使失眠持续的行为 | 睡眠效率↓，卧床时间过长 |

## 其他与华为数据相关的睡眠障碍

### 阻塞性睡眠呼吸暂停（OSA）

**ICSD-3 分类**：睡眠呼吸障碍 → 阻塞性睡眠呼吸暂停

**诊断标准**（多导睡眠图）：
- 呼吸暂停低通气指数（AHI）≥ 5次/小时
- AHI 5-15：轻度；15-30：中度；> 30：重度

**华为指标关联**：
| 症状 | 华为指标 | 触发转介阈值 |
|------|----------|-------------|
| 夜间低氧 | avg_spo2 < 94%，low_spo2_duration > 30min | SpO2 < 90% 反复出现 |
| 打鼾 | snoring_count | 打鼾次数显著增加 |
| 片段化睡眠 | wakeup_count↑（无明显原因）| 觉醒与呼吸事件相关 |

---

### 昼夜节律睡眠障碍

#### 睡眠时相延迟型（Delayed Sleep-Wake Phase）

**特征**：
- 入睡时间显著延迟（凌晨1-6点）
- 早上难以觉醒
- 日间嗜睡（如果被迫早起）

**华为指标映射**：
- sleep_latency：正常或偏短（一旦入睡后连续性尚可）
- 第一觉短但中后段睡眠尚可
- 周末与工作日睡眠时间差异大（> 2小时）

---

### 不宁腿综合征（RLS）

**ICSD-3 分类**：睡眠相关运动障碍 → 不宁腿综合征

**诊断标准**（4项必备）：
1. 有移动腿部的冲动，常伴不适感
2. 症状在休息时出现或加重
3. 症状在运动后部分或完全缓解
4. 症状在夜间加重

**华为指标映射**：
- 夜间频繁觉醒（wakeup_count↑，无法解释）
- 睡眠效率下降
- 无法从华为数据直接诊断，需用户主观描述

---

## DSM-5 与 ICSD-3 的对应关系

| DSM-5 诊断 | ICSD-3 对应分类 | 华为指标核心特征 |
|------------|----------------|----------------|
| 失眠障碍（Insomnia Disorder）| 失眠障碍 | sleep_latency↑, wakeup_count↑, 睡眠效率↓ |
| 嗜睡症（Hypersomnolence Disorder）| 中枢性嗜睡症 | 总睡眠时长↑但日间仍嗜睡 |
| 发作性睡病（Narcolepsy）| 嗜睡症1型 | REM侵入（日间突然跌倒入睡）|
| 阻塞性睡眠呼吸暂停 | 阻塞性睡眠呼吸暂停 | avg_spo2↓, snoring_count↑ |
| 昼夜节律睡眠障碍 | 昼夜节律失调性睡眠障碍 | 睡眠时相异常，与期望不符 |
| 快速眼动睡眠行为障碍 | 异态睡眠（REM相关）| REM期异常行为（需用户描述）|

## 严重度评分算法（用于排序）

```python
# 伪代码：基于华为指标计算各问题的严重度得分

def calculate_severity(record):
    scores = {}

    # 慢性失眠综合评分（基于ICSD-3诊断标准）
    insomnia_score = 0
    if record['wakeup_count'] >= 3:
        insomnia_score += (record['wakeup_count'] - 2) * 15
    if record['awake_time'] >= 30:
        insomnia_score += (record['awake_time'] - 30) * 0.5
    if record['sleep_efficiency'] < 85:
        insomnia_score += (85 - record['sleep_efficiency']) * 2
    scores['慢性失眠综合'] = min(100, insomnia_score)

    # 深睡不足
    if record['deep_sleep_rate'] < 15:
        scores['深睡不足'] = (15 - record['deep_sleep_rate']) * 5

    # REM不足
    if record['rem_sleep_rate'] < 20:
        scores['REM不足'] = (20 - record['rem_sleep_rate']) * 5

    # 入睡困难
    if record.get('sleep_latency', 0) > 30:
        scores['入睡困难'] = (record['sleep_latency'] - 30) * 2

    # 早醒
    if record['end_time'] < '06:00' and record['all_sleep_time'] < 360:
        end_mins = time_to_mins(record['end_time'])
        scores['早醒'] = (360 - end_mins) * 0.5

    # 血氧偏低（OSA筛查）
    spo2_score = 0
    if record['avg_spo2'] < 94:
        spo2_score += 30
    if record['low_spo2_duration'] > 30:
        spo2_score += 40
    scores['血氧偏低'] = min(100, spo2_score)

    return scores
```

## 专业转介决策树

```
开始
  ↓
检测到睡眠指标异常
  ↓
是否有转介触发条件？
  ├─ SpO2 < 90% 反复出现 → ⚠️ 立即转介（OSA筛查）
  ├─ 打鼾 + 夜间觉醒 + 血氧↓ → ⚠️ 转介（疑似OSA）
  ├─ 症状持续 > 3个月 + 多指标异常 → ⚠️ 建议就医（CBT-I）
  ├─ 白天严重嗜睡 + 无法控制入睡 → ⚠️ 转介（嗜睡症筛查）
  ├─ REM期异常行为/运动 → ⚠️ 转介（RBD评估）
  └─ 失眠 + 情绪低落/焦虑 → ⚠️ 建议心理/精神科评估
        ↓
    无转介指征
      ↓
    提供脑波音频辅助方案
```
