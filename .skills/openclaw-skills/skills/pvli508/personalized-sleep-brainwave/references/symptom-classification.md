## 1. DSM-5失眠诊断标准

**核心标准（满足以下至少一项）：**
- 入睡困难（儿童可表现为无看护者帮助难以入睡）
- 睡眠维持困难，频繁觉醒或觉醒后难以再入睡
- 早醒后无法再入睡

**功能损害：** 引起显著痛苦或社交、职业、学业、行为等功能损害

**频率：** 每周≥3晚

**持续时间：**
- 急性失眠：< 3个月
- 慢性失眠：≥ 3个月

**排除：** 不能更好地用其他睡眠障碍、物质效应或其他精神/躯体疾病解释

## 2. 症状类型详细判定

### SOI — 入睡困难
- **定义：** 卧床后超过30分钟仍无法入睡
- **用户描述关键词：** 睡不着、躺着翻来覆去、要很久才睡着、脑子不停转
- **评估问题：** "您通常躺下后大概多久能睡着？"
- **临界值：** >30 min = 存在；>60 min = 重度

### FNA — 睡眠维持困难（夜间频繁觉醒）
- **定义：** 夜间觉醒≥2次，或单次觉醒后清醒>30分钟
- **用户描述关键词：** 夜里总醒、睡眠很浅、一点声音就醒、醒了睡不着
- **评估问题：** "大概一晚上会醒几次？每次醒来大概多久能再睡着？"
- **临界值：** 觉醒≥2次 或 再入睡>30 min = 存在；觉醒≥4次 = 重度

### EMA — 早醒
- **定义：** 比预期或习惯起床时间提前≥30分钟醒来，且无法再入睡
- **用户描述关键词：** 醒得很早、天没亮就醒了、醒了就再也睡不着
- **评估问题：** "您平时几点起床？最近经常几点就醒了？"
- **临界值：** 提前≥30 min = 存在；提前≥90 min = 重度

### MIX — 混合型
- **定义：** 同时存在≥2种上述类型
- **处理：** 识别主要困扰类型（用户最在意哪个），以该类型为主，其余为辅
- **音频方案：** 全夜分段式组合（见 audio-parameters.md § 多类型组合）

## 3. 严重度评分

参照 ISI（失眠严重程度指数）简化版：

| 严重度 | 标准 | 对应处理 |
|-------|------|---------|
| 轻度 | 偶发（1–2天/周）；白天轻微影响 | 标准音频方案；4周改善计划 |
| 中度 | 频繁（3–4天/周）；白天明显疲倦/注意力下降 | 强化音频方案；配合睡眠卫生建议 |
| 重度 | 几乎每天（≥5天/周）；白天功能明显受损；持续≥3月 | 音频辅助 + 强烈建议专业评估（CBT-I或医生） |

**白天功能损害参考指标：**
- 疲倦/困倦感
- 注意力、记忆力下降
- 工作/学习效率降低
- 情绪烦躁、焦虑、低落
- 日间瞌睡但躺下反而睡不着

## 4. 多类型并发处理

### 优先级规则
1. **严重度优先：** 重度症状优先处理
2. **功能影响优先：** 对白天影响最大的优先
3. **用户诉求优先：** 用户最想解决哪个

### 常见并发组合

| 组合 | 推荐主要处理 | 音频时序 |
|------|-----------|---------|
| SOI + FNA | 以SOI为主（入睡是前提） | 入睡引导 → 整夜维持 |
| SOI + EMA | 以SOI为主 | 入睡引导 → 低频维持至晨 |
| FNA + EMA | 以FNA为主（觉醒更频繁） | 整夜维持（兼顾早醒段） |
| SOI + FNA + EMA | 全夜分段组合方案 | 见 audio-parameters.md |

### 向用户解释多问题的方式
```
我发现您有[N]个睡眠困扰——[简短列举]。
我们建议先重点改善「[主要问题]」，因为[1句理由]。
其他问题的音频方案我也会一并给您，但主要先聚焦这个。
```
