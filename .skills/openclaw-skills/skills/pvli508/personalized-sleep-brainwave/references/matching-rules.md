## 1. 匹配维度说明

音频库中每条音频均含以下标签，匹配时逐维度筛选：

| 维度 | 取值范围 | 来源步骤 |
|------|---------|---------|
| 症状类型 | SOI / FNA / EMA / MIX | Step 1 |
| 严重度 | 轻度 / 中度 / 重度 | Step 1 |
| 持续时长 | 急性（<3月）/ 慢性（≥3月）| Step 1 |
| 原因标签 | 见§2主匹配标签体系 | Step 3 |
| 用药状态 | 主要干预 / 辅助干预 | Step 2 |
| 年龄段 | 青年/青壮年/中青年/中年/初老/老年 | 用户信息 |
| 性别 | 男 / 女 / 未提供 | 用户信息 |

**匹配优先级：** 症状类型 → 原因标签 → 严重度 → 年龄性别 → 持续时长 → 用药状态

## 2. 主匹配标签体系

### SOI（入睡困难）

| 原因 | 音频库标签 | 说明 |
|------|----------|------|
| 认知觉醒/反刍思维 | `SOI·认知觉醒` | 脑子停不下来；放松阶段加权 |
| 情绪焦虑/低落 | `SOI·情绪舒缓` | 心情烦躁；情绪调节加权 |
| 节律紊乱（近期作息变化）| `SOI·节律重置-急` | 短期作息变乱；中速节律调整 |
| 节律紊乱（长期晚睡）| `SOI·节律重置-慢` | 习惯性晚睡；固定时间锚点 |
| 睡眠卫生不佳（屏幕）| `SOI·标准放松` | 屏幕蓝光；配套卫生建议 |
| 躯体不适 | `SOI·躯体放松` | 疼痛/不适；配就医建议 |
| 不确定/通用 | `SOI·通用` | 无明确原因；标准入睡引导 |

### FNA（睡眠维持困难）

| 原因 | 音频库标签 | 说明 |
|------|----------|------|
| 环境噪音/光线 | `FNA·觉醒阈值增强` | 提高睡眠稳定性 |
| 呼吸问题/打鼾 | `FNA·辅助舒缓` | ⚠️ 需就医；音频仅辅助 |
| 夜间如厕 | `FNA·深度维持` | 维持深度睡眠连续性 |
| 潮热/出汗（围绝经期）| `FNA·舒缓-围绝经` | 专项女性围绝经期 |
| 醒后认知觉醒 | `FNA·深度维持-认知` | 醒后无法停止思考 |
| 躯体疼痛 | `FNA·躯体放松` | 疼痛中断睡眠 |
| 睡不深/不确定 | `FNA·通用` | 标准睡眠维持方案 |

### EMA（早醒）

| 原因 | 音频库标签 | 说明 |
|------|----------|------|
| 情绪低落（抑郁信号）| `EMA·情绪支持` | ⚠️ 必须附抑郁筛查建议 |
| 压力/认知觉醒 | `EMA·维持深化+SOI前置` | 两段式：SOI引导+EMA维持 |
| 年龄相关（生理性）| `EMA·年龄适配` | 老年睡眠时相前移；减轻焦虑 |
| 环境唤醒 | `EMA·觉醒阈值增强` | 提高觉醒阈值 |
| 不确定/通用 | `EMA·通用` | 标准早醒方案 |

## 3. 多维度优先级规则

### 严重度对匹配的影响

| 严重度 | 匹配调整 |
|-------|---------|
| 轻度（偶发，1–2天/周）| 标准版音频；4周计划 |
| 中度（频繁，3–4天/周）| 强化版音频；配套行为建议强化 |
| 重度（几乎每天，≥5天/周，白天功能受损）| 强化版音频 + 建议专业评估（CBT-I / 医生）|
| 慢性（≥3月）| 优先匹配慢性专项音频；改善周期延长 |

### 用药状态对匹配的影响

| 状态 | 匹配调整 |
|------|---------|
| 正在用药（主要依赖药物）| 匹配"辅助版"音频（强度较标准版低）；明确定位为辅助 |
| 曾用药已停（<1个月）| 匹配戒断期辅助版；侧重焦虑/觉醒缓解 |
| 曾用药已停（>1个月）| 正常匹配，无特殊调整 |
| 未用药 | 正常匹配，完整强度 |

### 年龄性别对匹配的影响

| 情况 | 匹配调整 |
|------|---------|
| 65岁以上 | 优先`老年适配`标签；EMA接受度教育为主 |
| 46–55岁女性 + FNA | 优先`FNA·舒缓-围绝经`标签 |
| 18–25岁 + SOI/节律 | 优先`SOI·节律重置`标签；改善周期较短（1–2周） |
| 男性 + FNA + 打鼾/憋醒 | 强烈就医建议；仅辅助匹配 |

> 各年龄段详细睡眠基线见 `references/age-gender-profiles.md`

## 4. 混合型（MIX）匹配策略

### 确定主要困扰

向用户确认（在原因收集结尾自然询问）：
```
您刚才提到几个睡眠困扰，如果只能先改善一个，
您最想先解决哪个？是"睡不着"、"夜里总醒"，还是"醒得太早"？
```

### 常见组合匹配方案

| 组合 | 主匹配 | 辅助匹配 | 时序说明 |
|------|-------|---------|---------|
| SOI + FNA | SOI标签音频（主）| FNA标签音频（整夜辅助）| 入睡引导 → 整夜维持 |
| SOI + EMA | SOI标签音频（主）| EMA标签音频（持续至晨）| 入睡引导 → 低频维持 |
| FNA + EMA | FNA标签音频（主）| EMA标签辅助 | 整夜维持（兼顾后半夜）|
| SOI + FNA + EMA | 用户优先问题为主 | 组合时序方案 | 全夜分段 |

向用户说明组合方案时：
```
我们为您匹配了两个音频，配合使用效果更好：
一个帮您更快入睡，另一个在您睡着后整夜帮您保持深度睡眠。
```

## 5. 特殊情况处理

### ⚠️ 打鼾/憋醒（疑似OSA）
- **不作为主要匹配方向**
- 输出音频仅作辅助（`FNA·辅助舒缓`）
- **必须在温馨提示中写明：** 建议就医排查睡眠呼吸暂停综合征

### ⚠️ 早醒 + 情绪低落（疑似抑郁）
- 正常给出 `EMA·情绪支持` 标签音频
- **必须在温馨提示中写明：** 早醒伴随情绪低落与抑郁高度相关，强烈建议寻求心理科/精神科专业支持

### ⚠️ 重度失眠（≥5天/周 + 白天功能严重受损）
- 给出强化版音频方案
- **必须建议：** 考虑CBT-I（失眠认知行为治疗）或就医评估

### ⚠️ 躯体疼痛/不适为主原因
- 给出躯体放松型辅助音频
- **必须附加：** 建议就医排查躯体原因，音频仅作辅助

## 6. 匹配结果输出格式（内部记录，用于生成Step 5报告）

```
匹配结果：
- 主音频：[音频ID] [音频名称]
- 辅助音频（如有）：[音频ID] [音频名称]
- 使用时序：[什么时候用 / 几点 / 睡前多少分钟]
- 收听时长：[分钟]
- 收听方式：[耳机必须 / 音箱耳机均可]
- 音量建议：[低 / 中]
- 配套建议：[行为/环境建议，简短]
- 特殊提示标记：[就医 / 抑郁筛查 / 辅助定位 / 无]
- 匹配标签组合：[列出用到的所有标签]
```
