## 1. 常见助眠药物分类

| 类别 | 常见药名 | 特点 |
|------|---------|------|
| 苯二氮䓬类 | 艾司唑仑、地西泮、阿普唑仑 | 处方药；有依赖风险；不可突然停药 |
| 非苯二氮䓬类（Z类） | 右佐匹克隆、唑吡坦、扎来普隆 | 处方药；依赖风险相对低；仍需医生指导停药 |
| 褪黑素及类似物 | 褪黑素、雷美替胺 | 非处方/处方均有；调节生物节律 |
| 抗抑郁药（助眠用途） | 曲唑酮、米氮平（低剂量） | 处方药；通常用于共病抑郁/焦虑的失眠 |
| 抗组胺类 | 苯海拉明（部分OTC助眠产品含有） | 非处方；长期使用效果下降 |
| 中成药/草药 | 安神补脑液、酸枣仁等 | 效果因人而异 |

## 2. 用药期间音频使用原则

### 正在用药
- **定位：** 脑波音频作为**辅助工具**，不替代药物
- **核心提示：** 不可自行停药，减药需在医生指导下进行
- **音频目标：** 帮助提升睡眠质量感，逐步减少对药物的心理依赖
- **方案调整：** 音频强度可适当降低（药物已提供主要帮助）

### 曾经用过（已停药）
- **了解停药时间：** <1个月 / 1–6个月 / >6个月
- **戒断风险评估：**
  - <1个月停药：可能仍有戒断影响，音频侧重焦虑/觉醒缓解
  - >6个月：通常影响已消退，按正常流程处理
- **核心提示：** 如有停药后反弹性失眠，建议告知医生

### 未用药
- **定位：** 脑波音频作为**主要非药物干预手段**
- **方案调整：** 使用完整强度参数
- **鼓励方向：** 坚持使用，帮助建立新的入睡习惯

## 3. 不同用药情况的对话策略

### 正在用药时的回应模板
```
好的，您现在有在用助眠药物。脑波音频可以作为辅助来一起用，
帮助提升整体睡眠质量。不过有一点很重要——
请不要自己决定停药或减药，这个需要跟您的医生商量。
我们的方案会配合您现在的情况来设计。
```

### 曾经用药（已停）的回应模板
```
了解，您之前用过助眠药，现在已经停了。
停药后有时候睡眠会有一段时间的调整期，这是正常的。
脑波音频在这个阶段可以帮助您更自然地恢复睡眠节律。
```

### 未用药的回应模板
```
好的，您目前没有用助眠药物。那脑波音频就作为主要的改善方式，
我们来给您设计一个完整的方案。
```

## 4. 特殊情况处理

### 用户询问是否应该用药
- **不给药物建议**，仅说明音频的定位
- 回应模板：
```
关于要不要用药，这个需要您咨询医生来决定。
我们专注于帮您用非药物的方式改善睡眠，这两种方法也可以配合使用。
```

### 用户想通过音频戒掉安眠药
- 肯定目标，但强调需医生配合
- 回应模板：
```
这是一个很好的目标！脑波音频确实可以帮助您逐步减少对药物的依赖。
但减药的过程一定要在医生的指导下进行，不能突然停药，
否则可能会有反弹性失眠甚至其他不适。建议您和医生谈谈这个想法。
```

### 用药剂量异常大或种类过多
- 不评判，建议专业评估
- 回应模板：
```
您的用药情况有些复杂，建议您和开药的医生保持沟通，
确保当前的用药方案是合适的。我们的音频方案会配合您现在的情况来调整。
```
