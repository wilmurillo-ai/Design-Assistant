## 🏁 部署步骤

### 方式一：复制到 OpenClaw Skills 目录

```bash
# 复制整个 Skill 目录到 OpenClaw 的 skills 目录
cp -r brainwave-chronic-care /path/to/openclaw/skills/
```

### 方式二：配置加载本地 Skills

在 OpenClaw 配置文件中添加：

```json
{
  "plugins": {
    "load": {
      "paths": ["C:\\Users\\龚文瀚\\.openclaw\\workspace\\skills\\brainwave-chronic-care"]
    }
  }
}
```

### 启动验证

```bash
openclaw gateway restart
```

## ⚠️ 注意事项

1. **合规声明**：所有音频播放必须包含"本音频为非药物健康辅助，不替代医疗诊断、治疗及用药"

2. **音频资源**：当前测试使用模拟音频 URL，生产环境需要对接真实音频 CDN

3. **多租户隔离**：测试环境仅开放 `tenant_001`，其他租户无法访问

4. **网络要求**：设备需要联网以获取音频资源
