# AI Tattoo Prompt Builder

`ai-tattoo-prompt-builder` is a lightweight skill package for turning vague tattoo ideas into clearer AI-ready prompts.

It is designed for cases where the user already has a motif in mind but still needs help defining style, placement, scale, and mood before generating images or discussing the idea with a tattoo artist.

## What It Does

- turns a rough tattoo idea into one stronger base prompt
- returns alternative prompt directions for comparison
- keeps output visual instead of overly symbolic
- works well for image generation, concept exploration, and pre-artist handoff

## Included Files

- `SKILL.md` - main skill instructions
- `agents/openai.yaml` - UI metadata for compatible skill runtimes

## Example Prompt

`dragon tattoo, japanese style, forearm placement, vertical medium-size composition, powerful flowing movement`

## Intended Use

Use this skill when you want a model to improve tattoo prompts by combining:

- motif
- style
- placement
- scale
- mood

The goal is not poetic copy. The goal is a more renderable prompt.
