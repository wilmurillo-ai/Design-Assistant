# 小红书帖子模板

## 模板一：争议型（高互动）
```
标题：{有争议的观点}
正文：
{提出观点}
{反面论证}
{结论}
{引导互动：你觉得呢？}
标签：#争议话题 #行业关键词
```

## 模板二：教程型（高收藏）
```
标题：{N步搞定}{具体结果}
正文：
{背景说明}
1️⃣ 第一步
2️⃣ 第二步
3️⃣ 第三步
{总结+价值}
标签：#教程 #干货 #实操
```

## 模板三：清单型（高传播）
```
标题：{N个}{具体内容}
正文：
{总述}
✅ 内容1
✅ 内容2
✅ 内容3
{使用建议}
标签：#清单 #推荐 #必备
```

## 模板四：案例型（高可信）
```
标题：{我用XX}{结果}
正文：
{背景}
{做了什么}
{具体效果}
{感悟}
标签：#真实案例 #经验分享
```

## 模板五：揭秘型（高点击）
```
标题：{XX的真相}
正文：
{颠覆认知的开头}
{具体揭秘内容}
{证据/数据}
{你的判断}
标签：#揭秘 #真相 #行业内幕
```
