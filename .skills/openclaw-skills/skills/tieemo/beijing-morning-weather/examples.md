## Examples

### Example 1 (rainy + cold)

User: 早安

Assistant:

#### 今日北京天气
- 概况：阴有小雨，体感偏冷
- 温度：现在 6°C；最高 9°C / 最低 3°C
- 降水/风：上午有雨，风偏大，建议带伞并注意保暖
- 空气质量（如有）：一般（适合通勤，减少长时间户外停留）

#### 穿衣建议
- 建议叠穿：打底 + 毛衣/抓绒 + 防风外套
- 鞋子选防水/耐滑；出门带雨伞或轻便雨衣

#### 饮食建议
- 早餐：热粥/热豆浆 + 鸡蛋；或一碗热汤面
- 今天偏冷有风：午晚餐可选炖菜/汤类，少冰饮
- 避免：空腹喝冷饮、过多辛辣油炸

### Example 2 (sunny + big temperature swing)

User: 早上好

Assistant:

#### 今日北京天气
- 概况：晴到多云，早晚凉、中午暖
- 温度：现在 8°C；最高 19°C / 最低 6°C
- 降水/风：无明显降水，风不大

#### 穿衣建议
- 早晚加外套，中午可减一层（薄外套/针织衫即可）
- 温差大：优先选择可增减的分层穿搭

#### 饮食建议
- 早餐：燕麦/全麦面包 + 酸奶/豆浆 + 水果
- 中午偏暖：清淡为主，多喝水、蔬菜比例加大
- 避免：过咸、过油导致口渴
