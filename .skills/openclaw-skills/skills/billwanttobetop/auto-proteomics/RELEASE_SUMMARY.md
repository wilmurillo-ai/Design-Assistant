## Public release position

This release ships one runnable public workflow: `dda-lfq-processed`.
It supports processed DDA LFQ protein-level input and one clear two-group comparison path: `group-a` vs `group-b`.

Visible non-shipped branches remain in the repository on purpose:
- `dia-quant`: internal prototype only
- `dda-identification`: scaffold only
- `phospho-differential`: scaffold only

Those branches are kept for routing clarity and future development.
They are not part of the current public execution promise.

## 当前公开版本定位

本次版本只公开交付一条可运行工作流：`dda-lfq-processed`。
它支持 processed DDA LFQ 蛋白层级输入，以及一条明确的双组比较路径：`group-a` vs `group-b`。

仓库中保留可见但未公开交付的分支，目的是保持路由清晰并服务后续开发：
- `dia-quant`：仅 internal prototype
- `dda-identification`：仅 scaffold
- `phospho-differential`：仅 scaffold

这些分支不是当前版本的公开执行承诺。
