# Package Description

Writers Room Story Engine is a modular story-development package designed to help an agent generate, structure, draft, and revise compelling standalone stories.

This package synthesizes story development across Foundation, Character, Structure, Scene Logic, and Revision — drawing on the Pixar 22 Rules, Story Spine, Star Wars-style character-arc design, Hero’s Journey analysis, South Park’s therefore/but causality, and agent skill design best practices.

It is intended for:
- short fiction
- story concept generation
- screenplay-style narrative development
- world-supported stories
- revision and story diagnosis
- modular skill orchestration by a boss or supervisor agent

The package is organized into modular skills so an orchestration layer can route to the right phase instead of skipping straight into prose.

Core components:
- premise and story-core generation
- protagonist engine design
- Story Spine and beat structure
- causal plotting through therefore/but logic
- worldbuilding in service of pressure and consequence
- scene writing through objective, obstacle, subtext, and turn
- revision diagnostics that fix the highest-level failure first

Included files:
- core orchestration instructions
- modular story-suite skills
- a mega-skill for one-file testing or fallback deployment
- test prompts
- workflow guidance
- source notes
