# Sources

This package draws from widely used story-development frameworks and narrative principles, adapted into a modular AI skill workflow.

Key source influences include:
- Pixar’s 22 Rules of Story
- Story Spine
- Hero’s Journey analysis
- South Park’s “therefore / but” causality rule
- character arc design principles common to commercial franchise storytelling
- revision-first narrative development practice
- agent orchestration and modular skill design patterns

The goal of this package is not academic completeness.
It is practical narrative performance.

These sources were synthesized into a workflow that helps an agent:
- build stronger story foundations
- connect plot through consequence
- design better character pressure
- write scenes that turn
- revise from the highest-level problem first
