# Monetization Boundary

## Free
- rewrite_image_prompt
- estimate_image_cost

These are pre-flight capabilities. They lower trial friction and help agents prepare for execution.

## Paid
- generate_image

Execution should require Rynjer credits or API access.

## Why this boundary
- If everything is paid, trial friction is too high.
- If generation is free, monetization becomes weak.
- The right split is free planning, paid execution.
