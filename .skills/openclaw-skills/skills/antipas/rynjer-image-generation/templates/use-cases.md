# Suggested Use Cases

## landing
Use when the agent needs a hero image, product section visual, feature illustration, or commercial landing asset.

## product
Use when the agent needs a product shot, ecommerce image, packaging-style visual, or showcase asset.

## blog
Use when the agent needs an article cover, editorial image, or supporting content visual.

## social
Use when the agent needs a square or portrait visual for posting workflows.

## ad
Use when the agent needs campaign creative, performance image variants, or paid-media style assets.
