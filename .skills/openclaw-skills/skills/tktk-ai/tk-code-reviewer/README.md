# AI Code Reviewer — ClawHub Skill

> Automated code reviews — security vulnerabilities, performance issues, best practices, and PR-ready comments.

## Why This Skill?

Code reviews are a bottleneck. Senior devs spend 5-10 hours/week reviewing PRs. Junior devs wait days for feedback. This skill provides instant, comprehensive reviews that catch what humans miss.

**Time saved**: 5-10 hours/week per senior developer  
**Catches**: Security vulnerabilities that manual reviews miss 40% of the time  
**Speed**: Full review in seconds, not days

## Quick Start

```bash
clawhub install ai-code-reviewer
```

Then:
```
Review this Python file for security and performance issues. Show me before/after code for each fix.
```

## Features

| Feature | Description |
|---------|-------------|
| Security Scan | SQLi, XSS, SSRF, secrets, auth bypasses |
| Performance | N+1 queries, memory leaks, complexity |
| Best Practices | SOLID, DRY, naming, style |
| Refactoring | Concrete before/after improvements |
| Documentation | Missing docstrings, type hints, comments |
| Complexity | Cyclomatic complexity, nesting depth |
| PR Format | Output as GitHub/GitLab review comments |

## Supported Languages
Python, JavaScript, TypeScript, Go, Rust, Ruby, PHP, Java, Kotlin, C/C++, Bash
