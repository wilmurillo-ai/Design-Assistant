# Content Engine — ClawHub Skill

> Turn one idea into 30+ pieces of platform-optimized content with AI-powered brand voice consistency.

## The Problem

Creating content for multiple platforms is a time sink. One blog post takes 2-3 hours. Repurposing it into tweets, LinkedIn posts, and newsletters takes another 2-3 hours. Multiply by 5 topics per week and you're looking at a full-time job just on content.

## The Solution

Content Engine automates the production pipeline:

**Input**: One topic brief + your brand voice  
**Output**: Blog post + tweet threads + social posts + newsletter snippets  
**Time**: Minutes instead of hours

## Quick Start

```bash
clawhub install content-engine
```

Then:
```
Create a content pyramid on "remote work productivity tips"
Voice: Casual, direct, uses real examples
Audience: Tech workers and founders
```
