# Content Engine Skill

Transform one idea into 30+ pieces of platform-optimized content with consistent brand voice.

## What It Does

1. **Content Pyramid** — Takes a single topic/brief and produces a full content pyramid:
   - 1 long-form piece (blog/newsletter, 1,500-3,000 words)
   - 5 tweet threads (8-10 tweets each)
   - 15 standalone social posts (X, LinkedIn, Instagram captions)
   - 5 email/newsletter snippets

2. **Brand Voice Training** — Learns your writing style from samples and maintains it across all output

3. **SEO Optimization** — Keyword targeting, meta descriptions, header structure for blog content

4. **Multi-Platform Formatting** — Each piece is formatted for its target platform (character limits, hashtags, tone)

5. **Batch Processing** — Queue multiple briefs and produce in bulk

## Usage

### Quick Content Pyramid
```
Create a content pyramid from this topic:
Topic: "Why AI automation is replacing SaaS tools in 2026"
Brand voice: Direct, data-driven, slightly contrarian
Target audience: Solo founders and agency owners
Platforms: X/Twitter, LinkedIn, Blog
```

### Brand Voice Training
```
Learn my brand voice from these samples:
[Paste 10-20 writing samples]

Then write a blog post on [TOPIC] in my voice.
```

### Batch Production
```
Produce content for these 5 topics:
1. [Topic A]
2. [Topic B]
3. [Topic C]
4. [Topic D]
5. [Topic E]

For each: 1 blog post + 3 tweet threads + 10 social posts
Brand: [Brand name]
Voice: [Voice description]
```

### Repurpose Existing Content
```
Take this blog post and repurpose it:
[Paste blog post]

Create:
- 3 tweet threads (different angles)
- 5 LinkedIn posts
- 1 newsletter intro
- 10 standalone tweets
- 3 Instagram captions
```

## Output Format

### Blog Post
```
## [Title] — [Target Keyword]

**Meta Description**: [155 chars max]
**Target Keyword**: [primary keyword]
**Secondary Keywords**: [2-3 related terms]

[Full blog post with H2/H3 structure, internal link suggestions, CTA]
```

### Tweet Thread
```
### Thread: [Hook Summary]
Tweet 1: [Hook — max 280 chars]
Tweet 2-8: [Body tweets]
Tweet 9: [CTA]

Posting time: [Recommended]
```

### Social Posts
```
### [Platform] — Post [#]
[Content formatted for platform]
Hashtags: [if applicable]
Best time: [Recommended posting time]
```

## Best Practices

- Provide **brand voice samples** for best results (minimum 10 samples)
- Include **target audience description** — the more specific, the better
- One topic per pyramid produces higher quality than rushing 5 topics
- Always **review and add personal touches** before publishing
- Use the content calendar in references/ for optimal posting schedules
