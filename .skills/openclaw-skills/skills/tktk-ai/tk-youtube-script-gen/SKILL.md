# YouTube Script Generator

Complete video scripts optimized for retention — hooks, intros, body, CTAs, descriptions, tags, and thumbnail text.

## What It Does

1. **Script Generation** — Full video scripts with timestamps and visual cues
2. **Hook Writing** — First 30 seconds optimized for retention (the make-or-break zone)
3. **Retention Loops** — Open loops and pattern interrupts placed throughout
4. **SEO Optimization** — Title, description, tags optimized for YouTube search
5. **Thumbnail Copy** — High-CTR text overlay suggestions
6. **Multi-Format** — Educational, review, story, listicle, comparison templates

## Usage

### Full Video Script
```
Create a YouTube video script:

Topic: [TOPIC]
Video length: [TARGET MINUTES]
Format: [educational / review / story / listicle / comparison]
Target audience: [WHO WATCHES THIS]
Channel voice: [TONE — casual, authoritative, energetic, etc.]
CTA goal: [Subscribe / product link / lead magnet / next video]

Include:
- Hook (first 30 seconds — must stop the scroll)
- Intro with channel branding
- Body with retention loops every 2-3 minutes
- B-roll/visual suggestions in brackets
- CTA placement (mid-roll + end)
- Outro
- Full video description with SEO
- 15 tags
- 3 thumbnail text options
```

### Hook-Only Generator
```
Write 5 different hooks for a YouTube video about [TOPIC].

Each hook should:
- Be under 30 seconds when read aloud
- Create curiosity or urgency
- NOT start with "Hey guys" or "What's up"
- Use one of these formats: shocking stat, bold claim, question, story opener, demonstration

Rank them by predicted retention.
```

### Video Description + SEO
```
Write a YouTube video description for:
Title: [VIDEO TITLE]
Topic: [WHAT THE VIDEO COVERS]
Target keyword: [PRIMARY KEYWORD]

Include:
- Hook paragraph (above the fold)
- Timestamps/chapters
- Links section (affiliate, social, resources)
- Keywords naturally embedded
- Hashtags (3 max)
- 15 video tags (mix of broad and long-tail)
```

### Series Planner
```
Plan a 10-video YouTube series on [TOPIC]:

For each video:
1. Title (SEO-optimized, high CTR)
2. Thumbnail concept
3. Hook (first line)
4. Key points covered
5. CTA (what to watch next)
6. Internal linking (how videos reference each other)

Order by: viewer journey (beginner → advanced)
```

## Retention Techniques Built In

- **Cold open** — Skip pleasantries, start with value or shock
- **Open loops** — Tease upcoming content to prevent click-away
- **Pattern interrupts** — Visual/audio/energy changes every 60-90 seconds
- **Mid-roll CTA** — Ask for sub at the engagement peak, not the start
- **Curiosity gaps** — "Most people don't know this..." → deliver payoff later
- **Numbered structure** — "5 things..." gives viewers a progress bar mentally
