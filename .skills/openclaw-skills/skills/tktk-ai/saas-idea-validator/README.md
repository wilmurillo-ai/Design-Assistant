# SaaS Idea Validator — ClawHub Skill

> Validate product ideas before writing code. Market sizing, competitor mapping, MVP scope, pricing, and a data-backed go/no-go score.

## Why This Skill?

90% of startups fail. Most because they build something nobody wants. Validation agencies charge $5,000-15,000 for market research. This skill gives you the same analysis in minutes.

**Saves**: $5,000-15,000 per validation engagement  
**Prevents**: Months of building the wrong thing  
**Delivers**: Go/no-go recommendation with evidence

## Quick Start

```bash
clawhub install saas-idea-validator
```

Then:
```
Validate this idea: An AI tool that turns Reddit complaints into SaaS product specs with market sizing and MVP definitions.
```

## Features

| Feature | Description |
|---------|-------------|
| Problem Validation | Is this a real problem people pay to solve? |
| Market Sizing | TAM/SAM/SOM with methodology |
| Competitor Mapping | Who exists, where they fail |
| Pricing Analysis | What to charge, what model |
| MVP Definition | Build this first, cut that |
| Distribution Plan | How to reach first 100 users |
| Financial Model | 12-month revenue projection |
| Go/No-Go Score | Data-backed recommendation (0-50) |
