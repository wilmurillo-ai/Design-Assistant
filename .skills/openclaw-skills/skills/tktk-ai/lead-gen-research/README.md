# Lead Gen Research — ClawHub Skill

> Qualify sales prospects with AI-powered social profiling, trigger event detection, and personalized outreach angles.

## Why This Skill?

Manual prospect research takes 15-30 minutes per lead. This skill does it in seconds — with better coverage.

**Time saved**: 10+ hours/week for a team doing 50+ outreach per week  
**Quality boost**: Catches trigger events humans miss (funding rounds, job postings, product launches)  
**Conversion lift**: Personalized openers get 2-3x higher reply rates vs generic templates

## Quick Start

Install via ClawHub:
```bash
clawhub install lead-gen-research
```

Then ask your agent:
```
Research these 10 companies for outreach. Our service is AI content production. Score and rank them.
```

## Features

| Feature | Description |
|---------|-------------|
| Social Profiling | Public data from LinkedIn, X, company sites |

| Outreach Angles | 2-3 personalized openers per prospect |
