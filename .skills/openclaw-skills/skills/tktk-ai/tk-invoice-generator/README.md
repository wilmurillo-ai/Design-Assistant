# Invoice Generator — ClawHub Skill

> Professional invoices from simple inputs — auto-calculation, payment tracking, and overdue reminders.

## Why This Skill?

Freelancers waste time on invoicing. Dedicated tools cost $10-30/month. This skill generates invoices instantly from project data, tracks what's outstanding, and drafts payment reminders.

## Quick Start

```bash
clawhub install invoice-generator
```

Then:
```
Create an invoice for Client A: 20 hours of content writing at $75/hour, Net 30 payment terms.
```

## Ideal For

- **Freelancers** — Professional invoices without subscription tools
- **Agencies** — Batch invoicing for multiple clients
- **Consultants** — Hourly billing with expense tracking
- **Anyone** who bills clients and hates invoicing software
