# Landing Page Copy Generator

High-converting landing page copy — from hero section to footer CTA. Built on proven conversion frameworks.

## What It Does

1. **Full Page Copy** — Hero, features, benefits, social proof, FAQ, CTA — complete page
2. **Framework Selection** — PAS, AIDA, StoryBrand, or custom structure
3. **Audience Targeting** — Copy tuned to specific buyer personas
4. **A/B Variants** — Multiple headline and CTA options for testing
5. **Objection Handling** — FAQ section that overcomes buying hesitations
6. **Social Proof Blocks** — Testimonial formats, stat callouts, trust badges

## Usage

### Full Landing Page
```
Write landing page copy for:

Product: [NAME]
Type: [SaaS / physical product / service / lead magnet / event]
Target audience: [WHO]
Main benefit: [ONE SENTENCE — what transformation do they get?]
Price: [PRICE or "free" for lead gen]
Social proof: [Any testimonials, stats, or logos you have]
Competitor differentiation: [Why you, not them?]

Generate complete page copy:
1. Hero section (headline + subhead + CTA)
2. Problem section (agitate their pain)
3. Solution section (your product as the answer)
4. Features → Benefits (3-5 features translated to outcomes)
5. Social proof block (testimonials, stats, logos)
6. How it works (3-step simplicity)
7. Pricing section (if applicable)
8. FAQ (5-7 objection-handling questions)
9. Final CTA section (urgency + clear action)

Give me 3 headline variants and 2 CTA variants for A/B testing.
```

### Hero Section Only
```
Write 5 hero section variants for [PRODUCT]:

For each:
- Headline (under 10 words, benefit-driven)
- Subheadline (1-2 sentences expanding on the promise)
- CTA button text (action-oriented, specific)
- Supporting visual suggestion

Target: [AUDIENCE]
Main benefit: [BENEFIT]
```

### SaaS Pricing Page
```
Write copy for a SaaS pricing page:

Product: [NAME]
Tiers:
- Free: [features]
- Pro ($X/mo): [features]
- Team ($X/mo): [features]

For each tier:
- Tier name and tagline
- Feature list with benefit framing
- Who it's best for
- CTA button text

Plus:
- FAQ section (billing questions, comparisons)
- Money-back guarantee copy
- Annual vs monthly toggle copy
```

### Sales Email Landing Page
```
Write a landing page for email traffic:

Context: [Describe the email that sent them here]
Offer: [What they get]
Urgency: [Deadline / limited spots / bonus expiring]
Audience: [Who clicked the email]

The page should continue the email's narrative and close the deal.
Keep it short — they're already warm from the email.
```

## Conversion Principles

- **One page, one goal** — Every element drives toward the CTA
- **Benefits > Features** — People buy outcomes, not specifications
- **Social proof near CTAs** — Reduces friction at the decision point
- **FAQ = objection handler** — Address hesitations before they become dealbreakers
- **Urgency must be real** — Fake scarcity destroys trust
