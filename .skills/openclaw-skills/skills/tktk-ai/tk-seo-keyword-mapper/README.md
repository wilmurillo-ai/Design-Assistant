# SEO Keyword Mapper — ClawHub Skill

> Turn seed keywords into complete content strategies with clusters, intent mapping, difficulty estimates, and content briefs.

## Why This Skill?

Most SEO tools cost $100-300/month. They give you data but not strategy. This skill gives you the strategy — keyword clusters, content briefs, internal linking maps, and publication calendars — from a few seed keywords.

**Replaces**: Ahrefs keyword explorer ($99/mo), Semrush keyword magic tool ($129/mo), manual keyword research (10+ hours/month)

## Quick Start

```bash
clawhub install seo-keyword-mapper
```

| Feature | Description |
