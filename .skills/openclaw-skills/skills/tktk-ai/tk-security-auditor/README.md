# Security Auditor — ClawHub Skill

> AI-powered security auditing for Linux servers, web apps, and cloud infrastructure. Get actionable reports with copy-paste fix commands.

## Why This Skill?

Security audits are expensive ($500-2,000+ per engagement) and most small teams skip them entirely. This skill runs comprehensive checks and produces professional-grade reports with prioritized findings and exact fix commands.

**Cost savings**: $500-2,000 per audit vs manual consulting  
**Speed**: Full server audit in minutes vs days  
**Actionable**: Every finding includes the exact command to fix it

## Quick Start

```bash
clawhub install security-auditor
```

Then:
```
Run a full security audit on this server. Prioritize findings and give me fix commands.
```

## Features

| Feature | Description |
|---------|-------------|
| SSH Hardening | 12-point SSH configuration audit |
| Firewall Analysis | UFW/iptables rules and policy check |
| Port Scanning | Open ports, unnecessary service exposure |
| SSL/TLS Audit | Certificates, protocols, cipher suites |
| File Permissions | SUID, world-readable, sensitive files |
| Update Status | Pending patches, EOL software |
| Web Security | Headers, CORS, cookies, disclosure |
| Fix Commands | Copy-paste remediation for every finding |
