# Post Templates by Type

## 1. Single Punchy Tweet
```
[Bold claim or hot take in first line]

[2-3 lines expanding the point]

[Engagement hook: "Agree or disagree?" / "Repost if you've seen this 🔄"]
```

## 2. Short Thread (3-4 tweets)
```
Tweet 1 (Hook): [Bold claim + "Here's why:"]
Tweet 2 (Evidence): [Supporting point or example]
Tweet 3 (Insight): [Non-obvious takeaway]
Tweet 4 (CTA): [Call to action or link + engagement hook]
```

## 3. Question Tweet
```
[Genuine question that forces a side]

[1-2 lines of context or your own answer]

[Open-ended closer: "What's yours?" / "Tell me why 👇"]
```

## 4. Image Post
```
[Statement that the image reinforces]

[1-2 lines of context]

[Engagement hook: "Save this 🔖" / "Tag someone who needs this"]
```
Image: use quote cards, before/after screenshots, or data visualizations.

## 5. Build-in-Public
```
[Milestone or lesson: "Day X of building [thing]:"]

[What happened — be specific with numbers]

[What you learned]

[Question for audience: "Have you hit this wall?" / "What would you do?"]
```

## Medium-Polarizing Style Guide
- Take a clear stance (no "it depends" hedging)
- Use "most people" or "90% of" to create contrast
- Light sass is good, attacks are bad
- End with repost-friendly hooks
- Never cross into controversy or personal attacks
