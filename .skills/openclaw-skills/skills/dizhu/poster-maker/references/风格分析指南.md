# 风格分析指南

## 分析参考海报的 7 个维度

拿到参考海报后，按以下维度逐一分析，输出"风格分析卡"。

### 1. 构图

- 产品在画面中的位置：居中 / 偏左 / 偏右 / 底部
- 产品占画面的比例：20% / 40% / 60%
- 文字区域在哪里：上方 / 下方 / 左侧 / 右侧 / 叠在产品上
- 留白分布：上下对称 / 左重右轻 / 大面积顶部留白

### 2. 色调

提取 3-4 个核心色值：
- 背景色：[hex]
- 主色（标题/强调）：[hex]
- 辅色（装饰/卖点）：[hex]
- 文字色：[hex]

色调关键词：暖调 / 冷调 / 中性 / 高饱和 / 低饱和 / 莫兰迪 / 马卡龙

### 3. 字体风格

- 标题字体：衬线体 / 无衬线粗体 / 手写体 / 书法体
- 标题字号层级：特大 / 大 / 中
- 正文字体：和标题是否同一字体
- 特殊处理：描边 / 阴影 / 渐变色 / 竖排

### 4. 视觉元素

- 光效：柔光 / 硬光 / 无
- 装饰：花瓣 / 水滴 / 几何图形 / 粒子 / 线条 / 无
- 纹理：大理石 / 木纹 / 布纹 / 渐变 / 纯色
- 边框：有 / 无 / 圆角卡片

### 5. 排版

- 文字方向：横排 / 竖排 / 混合
- 对齐方式：居中 / 左对齐 / 右对齐
- 信息层级（从上到下或从大到小）：
  - 第一层：主标题
  - 第二层：副标题/卖点
  - 第三层：品牌/价格/活动信息

### 6. 信息密度

- 极简（只有产品 + 一句话）
- 标准（产品 + 标题 + 2-3 个卖点）
- 密集（产品 + 标题 + 卖点 + 价格 + 活动 + 二维码）

### 7. 整体气质（选择最接近的 1-2 个）

高端奢华 / 清新自然 / 可爱少女 / 科技未来 / 国潮复古 / 极简现代 / 活力运动 / 温暖治愈 / 专业医美 / 文艺杂志

## 输出模板：风格分析卡

参考海报风格分析：

- 构图：[产品居中偏下，占画面 45%，标题在上方 1/3 处]
- 色调：背景 #F5E6D3（暖米色）/ 标题 #2C1810（深棕）/ 强调 #D4956A（玫瑰金）
- 字体：标题用无衬线粗体白色，卖点用细体深棕色
- 装饰：柔光效果 + 少许花瓣飘散 + 底部轻微阴影
- 排版：横排居中，信息层级 3 层
- 信息密度：标准
- 气质：高端清新
