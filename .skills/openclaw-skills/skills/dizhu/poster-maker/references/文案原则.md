# 海报文案原则

## 写作规则

### 卖点 > 情怀
- ✅ "28 天淡化细纹，看得见的变化"
- ❌ "焕然新生，遇见更美的自己"

### 具体 > 抽象
- ✅ "含 5% 烟酰胺 + 2% 透明质酸"
- ❌ "多重精华成分"

### 数据可信
- 有实验数据就写数据（"临床测试 28 天有效率 89%"）
- 没有数据就写体感（"使用后皮肤更柔滑"），不编数据

### 短 > 长
- 主标题：6-12 字，一句话抓核心
- 副标题：12-20 字，补充说明
- 卖点 bullet：每条 8-15 字

## 禁用词

以下词汇在海报文案中禁止使用：

### 违规风险词（广告法）
- 最好、最佳、第一、NO.1、唯一、首选
- 国家级、世界级、全球领先
- 100% 有效、零风险、无副作用

### 空洞营销词
- 颠覆、革命、震撼、引爆、秒杀
- 完美、一流、精心、匠心
- 赋能、打造、缔造

### 替代原则
| 不要 | 改成 |
|------|------|
| "最好的护肤品" | "皮肤科医生推荐" |
| "完美无瑕" | "妆感自然，全天持妆 12 小时" |
| "颠覆传统" | "和传统方式相比，效率提升 3 倍" |
| "打造完美肌肤" | "28 天看见细纹淡化" |

## 文案结构模板

### 模板 A：问题 → 解决
主标题：[目标人群的痛点]
副标题：[产品名] + [核心卖点]

### 模板 B：数据 → 信任
主标题：[关键数据]
副标题：[数据的含义]

### 模板 C：场景 → 代入
主标题：[使用场景描述]
副标题：[产品名] + [场景中的价值]
