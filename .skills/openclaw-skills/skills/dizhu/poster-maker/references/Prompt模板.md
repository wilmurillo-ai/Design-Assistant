# 海报生成 Prompt 模板

## 模式 1：风格复刻 Prompt

    你是一名顶级平面设计师。请生成一张产品海报图片。

    【风格参考】
    请严格复刻以下风格（来自参考海报的分析）：
    - 构图：[从风格分析卡复制]
    - 色调：背景 [hex] / 标题 [hex] / 强调 [hex]
    - 字体风格：[标题用XX体，卖点用XX体]
    - 装饰：[光效/花瓣/几何/无]
    - 气质：[高端清新/科技极简/...]

    【产品】
    - 名称：[产品名]
    - 外观（详细描述）：[形状/颜色/材质/包装/标签文字]
    - 在画面中的位置：[居中/偏左]，占画面约 [X]%

    【文案】（所有中文必须清晰可读，不能模糊）
    - 主标题（最大最醒目）："[精确文字]"
    - 副标题："[精确文字]"
    - 卖点 1："[精确文字]"
    - 卖点 2："[精确文字]"
    - 品牌名（[位置]小字）："[品牌名]"
    - [其他信息如价格/活动时间]

    【尺寸】[宽]×[高]（如 1080×1440）

    **CRITICAL**: 所有中文文字必须清晰可读。
    请只生成一张图片。

## 模式 2：创意生成 Prompt

    你是一名顶级平面设计师和品牌创意总监。
    请为以下产品设计一张营销海报。

    【产品信息】
    - 名称：[产品名]
    - 品类：[护肤品/食品/服装/...]
    - 外观描述：[详细描述]
    - 核心卖点：[3-5 个卖点]
    - 目标人群：[25-35 岁都市女性/...]

    【文案】（已确认的终稿文案，所有中文必须清晰）
    - 主标题："[精确文字]"
    - 副标题："[精确文字]"
    - 卖点文字："[精确文字]"

    【设计方向】
    - 构图：[产品居中型/场景代入型/极简留白型/...]
    - 色调：[暖调米色系/冷调蓝白系/...]，具体色值 [hex]
    - 气质：[参考XX品牌的视觉风格，但不要直接抄袭]
    - 装饰：[柔光/水滴/花瓣/无]

    【尺寸】[宽]×[高]

    **CRITICAL**: 文案一字不差，中文清晰可读。
    请只生成一张图片。

## Prompt 铁律

1. **产品外观用文字描述**——"圆柱形白色磨砂玻璃瓶，高 15cm，金色压泵头，正面居中印有红色品牌 logo 'XXXXX'"，比"一瓶精华液"有用 100 倍
2. **文案原文粘贴**——不说"写一句有吸引力的标题"，直接给精确文字
3. **颜色用 hex**——不用"粉色"说 "#F5C6C6"
4. **位置要具体**——不说"标题在上面"，说"标题在画面上方 1/4 处水平居中"
5. **第一句不要写样式关键词**——避免 prompt 泄漏
6. **中文加 CRITICAL**——尤其是品牌名和价格

## 中文渲染 Fallback

如果 Nano Banana 生成的中文乱码：

**方案 A**：在 prompt 里加更强的 CRITICAL 指令重跑

**方案 B**：分两步生成
1. 先生成"无文字版"海报（只有产品和背景，不含任何文字）
2. 用 HTML + CSS 渲染文字层（Playwright 截图为透明 PNG）
3. 用 Pillow 将文字层叠加到产品背景上

方案 B 保证中文 100% 清晰，但需要额外的合成步骤。

## 常见迭代修复

| 问题 | 在 prompt 里加什么 |
|------|-----------------|
| 中文乱码 | CRITICAL: 所有中文文字必须完全清晰可读 |
| 产品外观不对 | 更详细的外观描述（尺寸/形状/颜色/材质/印刷内容）|
| 颜色偏了 | 用 hex 值替代颜色名 |
| 文字太小 | "主标题必须是画面中最大最醒目的元素" |
| 装饰太多 | "保持极简，不加花瓣/光效/几何装饰" |
| 排版乱了 | 精确描述每个元素的位置和大小比例 |
