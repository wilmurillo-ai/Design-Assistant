# 📚 学习积分小助手 - 使用指南

## 🎯 快速开始

在飞书中直接发送消息即可使用！

### 1️⃣ 记录积分

**格式**：以"学习积分"开头

**示例**：
```
学习积分 今天完成了汉字抄写 2 课，默写对了 5 个，口算题卡 2 篇全对，跳绳 350 个，自己洗澡了
```

**AI 会回复**：
```
✅ 今日积分已记录!

📊 得分详情:
• 汉字抄写：2 分 (2 课 × 1 分)
• 汉字默写：5 分 (5 个 × 1 分)
• 口算题卡：4 分 (2 篇 × 1 分 + 全对 2 分)
• 跳绳：1 分 (350 个:✓)
• 自主洗澡：1 分 (完成)

💰 总计：+13 分
```

### 2️⃣ 记录消费

**格式**：以"积分消费"开头

**示例**：
```
积分消费 买零食花了 20 分
```

**AI 会回复**：
```
✅ 消费已记录!

💸 支出：20 分
📝 用途：买零食花了
```

### 3️⃣ 查询积分

**示例**：
```
今日积分
```

**AI 会回复**：今日的积分明细

### 4️⃣ 发送学习图片

直接发送学习相关的图片（如口算题卡、作业照片），AI 会自动：
- 按日期重命名
- 存档到 `kids-points/archive/` 目录

## 💡 使用技巧

### 自然语言识别

AI 能理解多种表达方式：

✅ **这些都有效**：
- "学习积分 今天抄写了 2 课汉字"
- "学习积分 汉字抄写 2 课"
- "学习积分 完成了口算题卡 2 篇，都对了"
- "学习积分 跳绳跳了 350 个，1 分钟跳了 110 个"
- "学习积分 自己洗澡了，整理书包了"

### ABC Reading 自主完成

如果孩子**主动**完成 ABC Reading，记得说"自主"或"主动"：

✅ "学习积分 ABC Reading 读了 2 篇，是自主完成的" → **6 分**
❌ "学习积分 ABC Reading 读了 2 篇" → **3 分**

### 跳绳叠加

跳绳三项可以叠加：

✅ "学习积分 跳绳 350 个，1 分钟跳了 117 个" → **2 分** (350 个 +1 分，117 个 +1 分)
✅ "学习积分 跳绳完成 350 个，1 分钟 110 个，1 分钟 117 个" → **3 分** (全满)

## ❓ 常见问题

**Q: 积分有上限吗？**
A: 每月上限 400 分，达到后不再累计。

**Q: 积分可以存多久？**
A: 随时得分随时花，月底不清零。

**Q: 1 分等于多少钱？**
A: 1 分 = 1 元，可以直接兑换零花钱。

**Q: 可以补记之前的积分吗？**
A: 可以，但建议当天记录更准确哦。
