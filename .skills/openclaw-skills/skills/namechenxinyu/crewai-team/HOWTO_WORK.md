## 🔧 自定义配置

### 修改团队成员

编辑 `team_config_multi_model.py`：

```python
# 添加新角色
legal_expert = Agent(
    role="法律合规专家",
    goal="确保产品符合法律法规",
    ...
)

# 添加到团队
crew = Crew(
    agents=[..., legal_expert],
    tasks=[...]
)
```

### 修改工作流程

```python
# 并行执行（更快，但可能降低质量）
crew = Crew(
    ...,
    process=Process.hierarchical
)

# 顺序执行（默认，质量更好）
crew = Crew(
    ...,
    process=Process.sequential
)
```

### 添加自定义工具

```python
from crewai_tools import ScrapeWebsiteTool

# 添加网站抓取工具
scrape_tool = ScrapeWebsiteTool()

# 添加到 Agent
market_analyst = Agent(
    ...,
    tools=[scrape_tool]
)
```
