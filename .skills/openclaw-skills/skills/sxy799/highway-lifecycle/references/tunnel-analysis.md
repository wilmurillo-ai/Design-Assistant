# 隧道地质分析方法

## 一、分析流程

```
[隧道截面图像] 
    ↓
[图像预处理] → 去噪、增强、分割
    ↓
[特征提取] → 纹理、颜色、几何特征
    ↓
[参数推断] → 多模态模型推理
    ↓
[结果输出] → 结构化地质描述
```

## 二、特征类别详解

### 2.1 视觉感知类（可自动化）

**节理裂隙发育程度**：
| 等级 | 描述 | 视觉特征 | 模型表现 |
|-----|-----|---------|---------|
| 不发育 | 裂隙稀疏 | 连续完整岩体 | 良好 |
| 较发育 | 裂隙中等 | 部分割裂 | 良好 |
| 发育 | 裂隙密集 | 明显割裂 | 良好 |
| 很发育 | 裂隙极密集 | 破碎严重 | 良好 |

**岩石完整性**：
| 等级 | 描述 | 完整性指数Kv | 模型表现 |
|-----|-----|------------|---------|
| 完整 | 整体性好 | >0.75 | 良好 |
| 较完整 | 少量裂隙 | 0.55-0.75 | 良好 |
| 较破碎 | 裂隙较多 | 0.35-0.55 | 良好 |
| 破碎 | 裂隙密集 | 0.15-0.35 | 良好 |
| 极破碎 | 碎块状 | <0.15 | 良好 |

**地下水状态**：
| 等级 | 描述 | 流量特征 |
|-----|-----|---------|
| 干燥 | 无渗水 | 无可见水迹 |
| 潮湿 | 轻微渗水 | 表面湿润 |
| 滴水 | 明显渗水 | 点状滴水 |
| 线流 | 连续出水 | 线状水流 |
| 股流 | 大量出水 | 股状涌水 |

### 2.2 地质推理类（需大模型+Think）

**岩石坚硬程度**：
| 等级 | 单轴抗压强度Rc(MPa) | 推断依据 |
|-----|-------------------|---------|
| 坚硬岩 | >60 | 纹理致密、敲击声清脆 |
| 较坚硬岩 | 30-60 | 纹理较致密 |
| 较软岩 | 15-30 | 纹理疏松、易碎 |
| 软岩 | 5-15 | 手可掰断 |
| 极软岩 | <5 | 手可捏碎 |

**结构面结合程度**：
| 等级 | 特征描述 |
|-----|---------|
| 结合好 | 闭合、无充填 |
| 结合一般 | 微张、少量充填 |
| 结合差 | 张开、泥质充填 |
| 结合很差 | 宽张、软弱充填 |

**围岩等级**：
| 等级 | 综合特征 | 支护类型 |
|-----|---------|---------|
| Ⅰ | 坚硬岩，完整 | 局部锚杆或喷层 |
| Ⅱ | 较坚硬岩，较完整 | 锚杆+局部网 |
| Ⅲ | 较软岩，完整性一般 | 锚杆+网+喷层 |
| Ⅳ | 软岩，破碎 | 复合衬砌 |
| Ⅴ | 极软岩，极破碎 | 加强复合衬砌 |

### 2.3 专业参数类（需人工/仪器）

**不建议视觉推断的参数**：

| 参数 | 获取方式 | 原因 |
|-----|---------|-----|
| 地应力状态 | 地应力测试 | 视觉无法获取 |
| Rc值 | 单轴压缩试验 | 需物理测试 |
| Kv值 | 波速测试 | 需仪器测量 |
| 岩体结构类型 | 现场详细调查 | 需综合判断 |

**建议处理方式**：
```json
{
  "地应力状态": "需根据隧道埋深计算或现场测试",
  "Rc值": "需实验室测试或回弹仪现场测试",
  "Kv值": "需声波测试",
  "岩体结构类型": "需详细地质调查"
}
```

## 三、模型选型建议

### 3.1 任务适配

| 任务类型 | 推荐模型 | 配置 |

### 3.2 Think模式使用

**适用场景**：
- 围岩等级判断
- 岩石坚硬程度推断
- 结构面结合程度分析

**不适用场景**：
- 简单计数（裂隙数量）
- 直接观测（颜色、纹理）

## 四、标注规范

### 4.1 标注一致性要求

**问题：程度描述不统一**

错误示例：
```
标注1: "结合一般"
标注2: "结合一般或结合差"  // 不确定
标注3: "结合中等"  // 同义不同词
```

正确做法：
```markdown
## 结构面结合程度（穷举）
- 结合好
- 结合一般
- 结合差
- 结合很差

禁止使用：中等、较好、较差等非标准表述
```

### 4.2 标注模板

```json
{
  "image_id": "tunnel_001",
  "annotations": {
```

## 五、分析Prompt模板

### 5.1 快速分析模板

```
分析这张隧道施工截面图，回答以下问题：

1. 节理裂隙发育程度：[选择：不发育/较发育/发育/很发育]
2. 岩石完整性：[选择：完整/较完整/较破碎/破碎/极破碎]
3. 地下水状态：[选择：干燥/潮湿/滴水/线流/股流]

直接输出选项，无需解释。
```

### 5.2 详细分析模板

```
作为隧道地质工程师，详细分析这张隧道施工截面图。

分析内容：
1. 节理裂隙发育程度
   - 描述裂隙分布特征
   - 估算裂隙间距
   - 判断发育等级

2. 岩石完整性
   - 描述岩体割裂程度
   - 估算块体大小
   - 判断完整等级

3. 岩石坚硬程度推断
   - 分析纹理特征
   - 参考颜色和风化程度
   - 推断强度等级

4. 围岩等级综合判断
   - 综合上述分析
   - 给出围岩等级建议
   - 说明支护建议

输出格式：结构化分析报告
```

## 六、应用场景

### 6.1 施工阶段应用

**实时围岩分级**：
- 摄像头采集 → 模型分析 → 围岩等级
- 辅助施工决策
- 动态调整支护方案

**风险预警**：
- 检测异常地质特征
- 预警塌方风险
- 指导安全施工

### 6.2 设计优化

**参数校核**：
- 对比设计参数与实际
- 优化支护设计
- 控制工程成本

### 6.3 数据积累

**岩芯编录系统**：
- 自动化岩芯描述
- 建立地质数据库
- 以图搜图检索
