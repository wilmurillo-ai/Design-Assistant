## 三、数据集规范

### 3.1 高速事件检测

**数据格式**：
- 输入：10秒视频，分辨率 1920×1080 → 1024×576
- 抽帧：32帧
- 标签：9类二分类标签

**标签体系**：
```json
{
  "is_normal_status": false,
  "is_congestion": true,
  "is_abnormal_stop": false,
  "is_accident": false,
  "is_fire": false,
  "is_spillage": false,
  "is_pedestrian": false,
  "is_illegal_vehicle": true,
  "is_construction_rescue": false
}
```

### 3.2 隧道特征问答

**数据格式**：
- 输入：隧道施工截面图，分辨率 1102×705 → 1024×655
- 输出：参数文字描述

**特征类别**：
| 类别 | 类型 | 难度 |
|-----|-----|-----|
| 节理裂隙发育程度 | 视觉感知 | 低 |
| 岩石完整性 | 视觉感知 | 低 |
| 岩石坚硬程度 | 地质推理 | 中 |
| 结构面结合程度 | 地质推理 | 中 |
| 围岩等级 | 地质推理 | 高 |
| 地应力状态 | 专业参数 | 极高 |

### 3.3 道路病害检测

**数据格式**：
- 输入：现场图片，分辨率 512×512 → 1024×1024
- 标签：RDD标准代码

**病害代码**：
| 代码 | 名称 | 特征 |
|-----|-----|-----|
| D00 | 纵向裂缝 | 平行于车道 |
| D10 | 横向裂缝 | 垂直于车道 |
| D20 | 龟裂 | 网状纹理 |
| D40 | 坑槽 | 3D凹陷 |
| Repair | 修补 | 色块差异 |

## 四、评价指标

### 4.1 分类任务

**Macro F1**：
```
F1 = 2 × (Precision × Recall) / (Precision + Recall)
Macro F1 = average(F1_i) for all classes i
```

**混淆矩阵分析**：
- 误判模式识别
- 类别不平衡处理
- 模型偏见分析

### 4.2 问答任务

**LLM-as-judge**：
```python
prompt = """
判断模型回答是否正确。

标准答案：{ground_truth}
模型回答：{prediction}

输出：1（正确）或 0（错误）
"""
accuracy = correct_count / total_count
```

## 五、评测结果分析

### 5.1 Scaling Law效应

**关键发现**：
1. 参数量与性能正相关
2. 存在门槛效应（100B为分水岭）
3. 小模型可能存在能力坍缩

**示例数据**：
| 模型 | 高速F1 | 隧道ACC | 病害F1 |
|-----|--------|--------|--------|
| Qwen-30B | 0.55 | 0.25 | 0.36 |
| Qwen-235B | 0.62 | 0.32 | 0.46 |

### 5.2 Think模式影响

| 场景 | Think效果 | 原因 |
|-----|----------|-----|
| 高速事件 | ❌ 下降 | 过度推理导致幻觉 |
| 隧道分析 | ✅ 提升 | 推理增强准确性 |
| 病害检测 | ✅ 提升 | 自我辩论减少误判 |

### 5.3 模型选型建议

**事件检测首选**：
- Qwen3-VL-30B：性价比高，Recall优秀
- Qwen3-VL-235B：细粒度检测最优

**隧道分析首选**：
- Qwen3-VL-235B-Think：推理能力强
- GLM-4.5V-106B：地质知识丰富

**病害检测首选**：
- Qwen3-VL-235B：综合最优
- InternVL-241B：修补识别好

## 六、评测Prompt模板

### 6.1 事件检测

```
分析这段高速公路监控视频，判断是否存在以下事件：

事件类别：
1. 拥堵（is_congestion）
2. 异常停车（is_abnormal_stop）
3. 事故（is_accident）
4. 火灾（is_fire）
5. 抛洒物（is_spillage）
6. 行人（is_pedestrian）
7. 非法车辆（is_illegal_vehicle）
8. 施工/救援（is_construction_rescue）

输出格式：JSON，包含每个事件类别的布尔值
```

### 6.2 隧道分析

```
分析这张隧道施工截面图，描述以下地质特征：

1. 节理裂隙发育程度：[不发育/较发育/发育/很发育]
2. 岩石完整性：[完整/较完整/较破碎/破碎/极破碎]
3. 岩石坚硬程度：[坚硬岩/较坚硬岩/较软岩/软岩/极软岩]
4. 围岩等级：[Ⅰ/Ⅱ/Ⅲ/Ⅳ/Ⅴ级]

输出格式：结构化描述，包含判断依据
```

### 6.3 病害检测

```
分析这张道路图片，检测是否存在以下病害：

病害类型：
- D00：纵向裂缝
- D10：横向裂缝
- D20：龟裂
- D40：坑槽
- Repair：修补痕迹

输出格式：JSON，包含检测到的病害类型列表
```
