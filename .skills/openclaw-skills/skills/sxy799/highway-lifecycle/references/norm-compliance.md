## 三、审查要点

### 3.1 混凝土工程

**水胶比限值**：
| 混凝土等级 | 最大水胶比 |
|-----------|-----------|
| C30 | 0.55 |
| C40 | 0.45 |
| C50 | 0.36 |

**粗骨料要求**：
- 最大粒径 ≤ 钢筋最小净距的 3/4
- 最大粒径 ≤ 混凝土保护层厚度的 2/3

### 3.2 钢筋工程

**保护层厚度**（100年设计寿命，I-A类环境）：
- 板：30mm
- 梁：35mm
- 柱：40mm

**裂缝宽度限值**（JTG 3362-2018 第6.4.2条）：
| 环境类别 | 钢筋混凝土 | 预应力混凝土 |
|---------|-----------|-------------|
| Ⅰ类 | 0.20mm | 0.10mm |
| Ⅱ类 | 0.15mm | 不允许 |

### 3.3 预应力工程

**钢绞线强度**：
- 标准：1860MPa
- 常见错误：1395MPa（设计值误写）

**张拉参数**：
| 参数 | 符号 | 典型范围 |
|-----|-----|---------|
| 摩擦系数 | μ | 0.20-0.30 |
| 偏差系数 | k | 0.0015-0.0030 |
| 张拉应力 | - | 0.75fpk |

### 3.4 钢结构工程

**除锈等级适用**：
| 等级 | 方法 | 适用场景 |
|-----|-----|---------|
| Sa3 | 喷砂 | 工厂处理 |
| Sa2.5 | 喷砂 | 工厂处理 |
| St3 | 手工/电动 | 工地连接 |

**防腐涂装**：
- JC2环境（乡村）：配套涂装体系
- JC3环境（工业）：加强防腐体系

## 四、问题分级标准

| 严重程度 | 定义 | 示例 |
|---------|-----|-----|
| 高 | 影响安全/违反强制条款 | 保护层不足、强度不达标 |
| 中 | 影响耐久性/违反推荐条款 | 裂缝宽度超限、水胶比偏大 |
| 低 | 文档规范性问题 | 规范版本过时、名称不规范 |

## 五、常见问题清单

### 5.1 规范引用问题

```
问题：引用过时规范版本
检测：对比当前版本号
修复：更新为最新版本引用
```

```
问题：规范名称不规范
检测：校验标准全称
修复：使用规范全称
```

### 5.2 参数冲突问题

```
问题：同一参数多处不一致
检测：跨章节参数比对
修复：统一参数值，标注来源
```

```
问题：设计值与规范限值冲突
检测：数值范围验证
修复：调整设计或说明依据
```

## 六、审查Prompt模板

### 规范符合性审查

```
你是桥梁工程审图专家，具有20+年施工图审查经验。

审查依据：知识库中的规范条款

审查任务：
1. 检查规范名称、编号、版本是否正确
2. 检查技术指标是否满足规范最低要求
3. 检查检验频率、允许偏差是否符合规定
4. 检查材料性能、施工工艺是否满足要求

输出格式：JSON结构化结果
```

### 上下文一致性审查

```
你是桥梁设计审查专家，具备施工图审查经验。

审查任务：识别不同章节之间的冲突和不一致

重点检查：
1. 材料一致性（钢材牌号、混凝土等级）
2. 设计参数一致性（跨度、荷载、尺寸）
3. 预应力参数一致性（摩擦系数、张拉应力）
4. 同一结构描述一致性

判定规则：
- 数值不同 → 不一致
- 材料等级不同 → 不一致
- 同一对象多种描述 → 冲突
- 数值满足范围 → 一致

输出格式：JSON结构化结果
```
