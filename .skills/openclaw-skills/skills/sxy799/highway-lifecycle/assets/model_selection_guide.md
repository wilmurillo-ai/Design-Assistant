# 模型选型决策树

## 1. 确定任务类型

```
是什么任务？
├─ 视频事件检测 → 高速事件检测
├─ 隧道图像分析 → 隧道特征问答
└─ 路面图片检测 → 道路病害检测
```

## 2. 确定优先级

```
最看重什么？
├─ 速度/性价比 → 选择30B模型
├─ 准确度 → 选择235B模型
└─ 平衡 → 双模型方案
```

## 3. 确定资源

```
GPU资源？
├─ 1-2张A800 → 仅能部署30B模型
├─ 4张A800 → 可部署GLM-106B
└─ 8张A800 → 可部署235B模型
```

## 4. 确定Think模式

```
任务特性？
├─ 直觉感知（事件检测）→ 禁用Think
├─ 推理判断（隧道分析）→ 启用Think
└─ 模糊判断（病害检测）→ 启用Think
```
