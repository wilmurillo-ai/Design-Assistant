# Daily Log Template

> L2 每日归档模板

---

```markdown
# 每日归档 - YYYY-MM-DD

> 归档时间：23:05

---

## 活跃任务
- [ ] 任务描述
- [ ] 任务描述

## 关键决策
- 决策内容

## 洞察发现
- 洞察内容

## 完成项
- [x] 完成的任务

## 对话摘要
- 当日主要讨论内容摘要（200-500 字）

## 待办事项
- 未完成的任务，需要跟进
```

---

## 字段说明

| 字段 | 来源 | 用途 |
|------|------|------|
| 活跃任务 | L1 提炼 | 归档当日活跃任务 |
| 关键决策 | L1 提炼 | 归档当日关键决策 |
| 洞察发现 | L1 提炼 | 归档当日洞察 |
| 完成项 | L1 提炼 | 归档当日完成项 |
| 对话摘要 | L2 归档时生成 | 当日讨论内容摘要 |
| 待办事项 | L1 提炼 | 未完成的任务 |

---

## 命名规则

文件名：`YYYY-MM-DD.md`

示例：
- `2026-04-12.md`
- `2026-04-13.md`

**注意**：日期由 `date +%Y-%m-%d` 动态获取，禁止硬编码。
