# SESSION-STATE.md Template

> L1 当前任务状态模板

---

```markdown
[Last_Extracted_Time: YYYY-MM-DD HH:MM:SS]

## 活跃任务 (Active Tasks)
- [ ] 任务描述（优先级 P0/P1/P2）
- [ ] 任务描述（优先级 P0/P1/P2）

## 关键决策 (Key Decisions)
- 决策内容（时间戳）

## 洞察发现 (Insights)
- 洞察内容（时间戳）

## 完成项 (Completed)
- [x] 完成的任务（时间戳）

## 待归档上下文 (Pending Context)
- 需要保留到 L2 的对话摘要（200 字以内）
```

---

## 字段说明

| 字段 | 用途 | 更新频率 |
|------|------|---------|
| `Last_Extracted_Time` | 游标，防重复提炼 | 每次提炼后更新 |
| `活跃任务` | 当前待办事项 | 每小时 L1 提炼更新 |
| `关键决策` | 用户做出的决定 | 每小时 L1 提炼更新 |
| `洞察发现` | 用户的深度思考 | 每小时 L1 提炼更新 |
| `完成项` | 已完成的任务 | 每小时 L1 提炼更新 |
| `待归档上下文` | 跨会话需要的上下文 | 23:05 归档到 L2 后清空 |
