# TODO.md 示例

```markdown
# TODO

## 进行中

- [ ] 调研竞品项目X · 了解行业最佳实践 · 2026-03-20 @agent-01

## 待做

- [ ] 编写技能文档 · 需要给团队复用 · 2026-03-20 @agent-02
- [ ] 优化图片生成流程 · 当前速度太慢 · 2026-03-19 @agent-03
```

# DONE.md 示例

```markdown
# DONE

## 已完成

- [x] 搭建开发环境 · 用docker-compose一键启动 · 2026-03-20 @agent-01
- [x] 修复登录bug · 原因是token过期未刷新 · 2026-03-19 @agent-02
```

## 文件路径速查

| 文件 | 路径 |
|------|------|
| 待办 | `memory/todo/TODO.md` |
| 已完成 | `memory/todo/DONE.md` |
| 技能说明 | `skills/todo-list/SKILL.md` |
| 详细规范 | `skills/todo-list/references/spec.md` |
| 本示例 | `skills/todo-list/references/example-todo.md` |

## 最小记录要素

每个 TODO 必须包含：
1. **任务** — 要做什么（简洁）
2. **原因** — 为什么做
3. **日期** — 添加时间
4. **负责人** — 谁来做

每个 DONE 必须包含：
1. **任务** — 做了什么
2. **如何完成** — 简单说明方法
3. **日期** — 完成时间
4. **负责人** — 谁做的
