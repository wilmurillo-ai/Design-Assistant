# 子弹笔记生成器 - 项目概述

## 🎯 项目简介

子弹笔记生成器（Bullet Journal Generator）是一个智能笔记处理工具，能够将零散的自然语言输入转换为规范的子弹笔记格式，支持多种输出格式和Obsidian知识库集成。

### 核心价值

✅ **提高效率**：自动分类、润色、格式化，省去手动整理时间
✅ **统一规范**：使用标准符号系统，保持笔记一致性
✅ **多格式输出**：文本、Obsidian、HTML、PDF，满足不同需求
✅ **智能识别**：自动区分工作学习内容，便于知识管理
✅ **可视化展示**：精美HTML卡片，直观展示笔记内容

## 🎨 符号系统

| 符号 | 类型 | 说明 | 示例 |
|------|------|------|------|
| ● | 任务 | 待办事项 | ● 整理有机逆合成逻辑 |
| ○ | 事件 | 已发生或计划的活动 | ○ 09:17 紧急处理立项书 |
| – | 笔记 | 想法、灵感、关键信息 | – 股票市场波动较大 |
| > | 迁移 | 未完成推迟的任务 | > 购买卫生纸（推迟至明日） |
| < | 计划 | 未来安排 | < 明天 家家悦购买鸡蛋 |
| ☆ | 优先 | 高优先级任务 | ☆ 有机逆合成自动化整理 |

## 🔧 核心功能

### 1. 智能处理

#### 时间提取
```
"09:00" → 09:00
"09时00分" → 09:00
"明天" → 标记为计划
```

#### 语言润色
```
"要把" → "整理/处理/完成"
"突然" → 紧急标注
"最好" → 目标补充
"想一想" → "思考/研究"
```

#### 内容分类
```
动作动词 → 任务（●）
有时间标记 → 事件（○）
观察类 → 笔记（–）
"明天"/"下周" → 计划（<）
"紧急"/"重要" → 优先级（☆）
```

### 2. Obsidian集成

#### Frontmatter
```yaml
---
date: 2025-03-16
tags: ['2025-03', 'observe', 'plan', 'study', 'think', 'work']
type: daily_log
---
```

#### 双链
```markdown
- [[有机逆合成/自动化]] 整理有机逆合成逻辑
- [[立项书/政府项目]] 处理立项书
- [[股票分析/市场波动]] 市场波动分析
```

#### 结构化内容
```markdown
## 优先任务
## 任务
## 事件
## 思考与观察
## 下一步行动
```

### 3. 可视化展示

#### HTML卡片特性
- 🎨 手写风格设计
- 📱 响应式布局
- 🌡️ 天气图标和温度
- 🎯 按符号分类展示
- ✨ 精美的视觉效果

#### 打印版本特性
- 📄 A4纸张格式优化
- 🖨️ 高分辨率打印
- ✍️ 留有手写补充空间
- 📋 包含"打印"按钮

## 💡 使用场景

### 场景1：日常记录

**输入**：
```
今天要完成周报，下午3点开会。晚上去健身房。
明天记得给客户打电话。
```

**输出**：
```
● 完成周报
○ 15:00 团队会议
○ 晚间 健身房训练
< 明天 给客户打电话
```

### 场景2：工作学习

**输入**：
```
最近股票波动大，要试试新方法。老板要项目摘要，我先写一下。
```

**Obsidian日志**：
```markdown
---
date: 2025-03-16
tags: [工作, 研究]
type: daily_log
---

## 任务
- [[立项书/项目摘要]] 准备项目摘要 ⭐

## 思考与观察
[[股票分析/市场波动]] 市场波动大，研究新方法
```

### 场景3：混合内容

系统会自动区分：
- 工作学习 → Obsidian日志
- 生活内容 → 仅子弹笔记
- 所有内容 → HTML卡片和打印版本
