# 文档模板结构指南

> 本文档定义各类后方业务文档的标准结构与写作规范，AI 生成文档时应严格遵循对应模板结构。

---

## 一、制度类文档模板

### 模板01：公司规章制度（通用）

```markdown
【公司名称】[制度名称]

版本号：V1.0
生效日期：XXXX年XX月XX日
发布部门：人力资源部/行政部
适用范围：全体员工 / [指定部门]

---

第一章 总则

第一条 目的
[说明制定本制度的目的和意义，1-2句话]

第二条 适用范围
[明确适用对象：全体在职员工/管理层/特定岗位]

第三条 定义
[对制度中的关键术语进行定义，如适用]

---

第二章 [核心内容章节名]

第X条 [条款标题]
[具体规定内容，要求可操作、有明确标准]

（按需增加条款）

---

第三章 [执行与监督]

第X条 执行主体
[明确谁负责执行本制度]

第X条 监督与检查
[说明如何监督制度执行情况]

第X条 违规处理
[列明违规情形及对应处理方式]

---

第四章 附则

第X条 解释权
本制度由[人力资源部/行政部]负责解释。

第X条 生效与修订
本制度自[生效日期]起执行，如有修订，以最新版本为准。

---

附件：[相关表格/流程图，如有]
```

### 模板02：员工手册（章节结构）

```
第一章 公司简介与企业文化
  - 公司背景与发展历程
  - 使命/愿景/价值观
  - 组织架构概览

第二章 入职须知
  - 入职流程与所需材料
  - 试用期管理
  - 转正条件与流程

第三章 工作制度
  - 工作时间与考勤
  - 假期管理（各类假期规定）
  - 远程/弹性工作政策（如有）

第四章 薪酬与福利
  - 薪酬结构说明
  - 发薪日期与方式
  - 社保公积金说明
  - 员工福利清单

第五章 绩效管理
  - 绩效评估体系
  - 评估周期与流程
  - 结果运用说明

第六章 行为准则
  - 职业行为规范
  - 保密与知识产权
  - 利益冲突政策
  - 反骚扰/反歧视政策

第七章 离职管理
  - 离职申请流程
  - 工作交接要求
  - 离职证明与竞业限制

附录
  - 常用表格列表
  - 重要联系方式
  - 修订记录
```

## 二、通知公告类模板

### 模板03：全员通知

```markdown
标题：关于[主题]的通知

各部门、全体员工：

[开门见山说明通知目的，1句话]

一、[主要内容/政策说明]
[具体内容，分条列出]

二、[执行要求/注意事项]
[明确时间要求和执行规范]

三、[联系方式（如需要）]
如有疑问，请联系[部门/联系人]（邮箱/电话）。

特此通知。

[发文部门]
[日期]
```

### 模板04：放假通知

```markdown
标题：[节日名称]放假通知

全体员工：

根据国家法定节假日安排，结合公司实际情况，现将[节日]放假安排通知如下：

一、放假时间
[日期] 至 [日期]，共 X 天。

二、调休安排（如有）
[日期（周X）] 正常上班，作为调休补班。

三、值班安排
放假期间，[部门]安排专人值班，联系方式：[电话]。

四、温馨提示
- 请各部门在放假前做好工作交接
- [其他注意事项]

祝大家节日快乐！

[公司名称] 行政部
[日期]
```

## 六、文档质量检查清单

生成文档后，应检查以下项目：

**内容完整性**
- [ ] 标题、日期、版本号齐全
- [ ] 适用范围明确
- [ ] 关键条款有可操作的细则（数字/时限/负责人）
- [ ] 例外情形有处理路径

**合规性**
- [ ] 涉及劳动法的内容符合法律基线（参见 hr_policies.md）
- [ ] 含权益影响的条款已标注"建议法律审查"
- [ ] 无歧视性表述

**可读性**
- [ ] 语言正式、准确
- [ ] 无歧义的模糊表述
- [ ] 表格/列表格式整齐
