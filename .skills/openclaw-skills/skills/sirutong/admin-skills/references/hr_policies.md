# HR 制度要点与法律合规基线

> 本文档为 AI 参考资料，涉及法律条款时应提示用户进行专业法律审查。
> 基准：《劳动合同法》《劳动法》《社会保险法》及相关司法解释（截至2024年）

---

## 一、招聘与用工合规

### 1.1 劳动合同要点
- **签订时限**：入职起1个月内必须签订书面劳动合同，超期需支付双倍工资（最长11个月）
- **必备条款**：用人单位信息、劳动者信息、合同期限、工作内容与地点、工作时间与休息、劳动报酬、社保、劳动保护条件
- **试用期规定**：
  - 3个月以下合同：不得约定试用期
  - 3个月至1年合同：试用期≤1个月
  - 1年至3年合同：试用期≤2个月
  - 3年以上/无固定期限：试用期≤6个月
  - 试用期工资不低于转正工资80%，且不低于最低工资标准

### 1.2 招聘合规红线
- 禁止在招聘中设置性别、婚育状况、年龄等歧视性要求（高度风险）
- 禁止收取押金、扣押证件
- 背景调查须获得候选人书面授权
- 录用通知（Offer）发出后无故撤销可能承担缔约过失责任

### 1.3 劳务派遣与外包
- 劳务派遣岗位限于"临时性、辅助性、替代性"
- 派遣员工比例不超过用工总量10%
- 同工同酬原则适用

## 二、薪酬与福利

### 2.1 薪酬结构常见设计
```
月薪 = 基本工资 + 绩效工资 + 津贴补贴
年薪 = 月薪 × 12 + 年终奖
```

### 2.2 加班费标准（法定）
- 工作日加班：工资的150%
- 休息日加班（不能补休）：工资的200%
- 法定节假日加班：工资的300%（不可用调休替代）

### 2.3 社保公积金
- 五险：养老、医疗、失业、工伤、生育（各地缴费比例不同）
- 一金：住房公积金（各地比例5%-12%）
- **入职即缴**：应在员工入职当月完成社保登记

### 2.4 特殊薪酬场景
- 病假工资：连续病假≤6个月，按工龄系数（60%-100%）× 本人工资
- 婚假/丧假/年假：按国家和地方规定执行，不低于法定标准
- 年休假天数：工龄1-10年（5天）、10-20年（10天）、20年以上（15天）

## 三、绩效管理

### 3.1 绩效体系常见模式
| 模式 | 适用场景 | 核心逻辑 |
|------|---------|---------|
| KPI | 目标明确、可量化岗位 | 关键指标达成率打分 |
| OKR | 创新型、战略执行岗位 | 目标+关键结果，不直接关联薪酬 |
| 360度评估 | 管理岗/综合类岗位 | 上级+下级+同级+自评 |
| MBO | 中高层管理岗 | 目标管理法 |

### 3.2 绩效改进计划（PIP）合规要点
- PIP 须有明确的改进目标、时间节点、支持措施
- PIP 期间不得随意降薪（合同有约定除外）
- PIP 后解除劳动合同须有充分证据证明"不胜任工作"
- 解除前须提前30天通知或支付代通知金

### 3.3 末位淘汰注意事项
- 纯粹末位≠不胜任，直接解除合同具有法律风险
- 应结合绩效制度，明确"不胜任"的认定标准
- 需经过培训或岗位调整步骤后方可解除

## 四、员工关系

### 4.1 劳动合同解除

**员工主动离职（辞职）**
- 试用期：提前3天书面通知
- 转正后：提前30天书面通知
- 违反通知期可能承担赔偿责任

**公司主动解除（辞退）**
- 过失性解除（第39条）：无需补偿，需有充分证据
- 非过失性解除（第40条）：提前30天通知或支付1个月代通知金 + N个月经济补偿
- 经济性裁员（第41条）：20人以上或比例≥10%，需提前30天向劳动行政部门报告

**经济补偿金（N）计算**
```
N = 工作年限（每满1年计1个月）× 离职前12个月平均工资
工作不满1年：按比例计算（满6个月计0.5）
封顶：月工资不超过当地社平工资3倍，年限最多12年
```

### 4.2 劳动争议处理
- 仲裁时效：1年（知道或应当知道权益受侵害之日起）
- 流程：协商 → 调解 → 劳动仲裁 → 诉讼
- **高风险操作**：口头解雇、不出具离职证明、拖欠工资、违法竞业限制

### 4.3 竞业限制
- 仅适用于：高级管理人员、高级技术人员、知悉商业秘密的人员
- 期限不超过2年
- 必须支付竞业限制补偿金（离职后按月支付，不低于月工资25-30%）
- 不支付补偿金超过3个月，员工可解除竞业协议

## 五、行政管理

### 5.1 固定资产管理要点
- 建立资产台账（资产编号、名称、规格、购置日期、价值、使用人）
- 资产领用须有签收记录
- 离职交接时需完成资产归还清单
- 报废/处置需经审批，防止国有资产流失（适用于国企/上市公司）

### 5.2 供应商管理
- 供应商准入：资质审核、报价比较（≥3家询价）、合同签订
- 付款审批：明确金额阈值对应审批层级
- 定期评估：至少每年一次供应商评级
- 关联交易：须在审批表中披露关联关系

### 5.3 差旅与费用报销
- 差旅标准应按岗位级别制定（住宿、交通、餐补标准）
- 报销凭证要求：原始发票、出差审批单、行程证明
- 时效：出差结束后5-10个工作日内提交报销
- 超标处理：须经上一级审批并说明原因

## 六、数据隐私与信息安全（HR视角）

- 员工个人信息收集须遵循"最小必要原则"
- 人事档案不得对外泄露（包括薪资、绩效、医疗信息）
- 离职员工数据保存期限建议：劳动合同到期后至少2年（仲裁时效+缓冲）
- 背景调查、健康检查结果的使用须在劳动合同或隐私政策中说明

## 七、常见违规风险清单

| 风险场景 | 风险等级 | 处理建议 |
|---------|---------|---------|
| 超期不签劳动合同 | 🔴 高 | 立即补签，支付双倍工资 |
| 强制购买"内部产品" | 🔴 高 | 立即停止，可能构成违法克扣 |
| 口头解除劳动关系 | 🔴 高 | 必须书面通知，留存证据 |
| 试用期工资低于转正80% | 🔴 高 | 补发差额，修改制度 |
| 不缴或少缴社保 | 🔴 高 | 补缴，承担滞纳金 |
| 竞业协议不支付补偿 | 🟡 中 | 补付或协商解除协议 |
| 年休假未安排且未补偿 | 🟡 中 | 补发未休年假工资 |
| 绩效结果未书面告知员工 | 🟡 中 | 完善绩效反馈流程 |
| 采购未询价直接定标 | 🟡 中 | 补充比价记录，完善制度 |
