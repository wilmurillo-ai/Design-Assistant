# 标准流程库

> 收录科技公司后方业务20+核心流程，每个流程包含：触发条件、参与角色、流程节点、时效要求、所需材料、风险提示。

---

## 一、人力资源类流程

### 流程01：员工招聘流程

**触发条件**：部门提出招聘需求
**参与角色**：用人部门、HR招聘专员、直属上级、HRBP、CEO（选填）

```
用人部门提交《岗位需求申请表》
  ↓（HR审核）1工作日
HR确认编制/预算，发布JD
  ↓ 简历筛选（持续）
HR初筛 → 电话筛选
  ↓（通过）
安排面试（最多3轮）
  ↓（offer决策）
薪酬评估 → 上级审批 → 发放Offer
  ↓（候选人接受）3工作日内
发背景调查授权 → 完成背调
  ↓（背调通过）
确认入职日期，发入职通知
```

**时效目标**：从需求提出到发Offer：社招≤30天，校招≤60天
**所需材料**：岗位需求申请表、面试评估表、背调授权书、录用通知书
**风险提示**：Offer发出后撤销需谨慎；避免涉及敏感信息的面试问题

---

### 流程02：员工入职流程

**触发条件**：新员工确认入职日期
**参与角色**：HR、IT、行政、直属上级、财务

```
入职前3天：
  HR发《入职通知》（附入职材料清单）
  IT准备设备、账号权限
  行政准备工位、门禁卡

入职当天上午：
  HR接待 → 收取入职材料
  签订《劳动合同》《保密协议》《竞业协议》（如适用）
  填写员工信息表 → 录入HRIS系统

入职当天下午：
  IT交付设备 → 员工签收
  行政安排工位 → 介绍基础设施
  直属上级介绍团队 → 安排工作

入职后3天内：
  完成社保/公积金新增申报
  完成薪资信息录入
  安排入职培训（公司文化/合规/安全）

入职后7天内：
  转正计划沟通（明确试用期目标）
  HR完成档案建立
```

**所需材料**：身份证、学历证书、前雇主离职证明、银行卡信息、近期照片、体检报告（按需）
**检查清单**：合同签订✓、社保申报✓、系统账号✓、设备领用✓、工位安排✓

---

### 流程03：员工离职流程

**触发条件**：员工提交辞职申请 / 公司发起解除
**参与角色**：离职员工、直属上级、HR、IT、行政、财务

```
员工提交离职申请（书面/邮件）
  ↓（直属上级）1工作日内
上级确认接受（避免口头挽留拖延时效）
  ↓（HR）2工作日内
HR启动离职流程，发《工作交接清单》
  ↓ 离职前最后一周
工作交接（接替人签收或上级确认）
资产归还（电脑、门禁、证件等）
系统权限注销（IT执行）
  ↓ 最后工作日
HR完成离职面谈（记录离职原因）
签署《离职证明》《竞业协议确认》（如有）
  ↓ 离职后5工作日内
财务结算：工资、未休年假补偿、报销
开具《离职证明》（法律义务，不可拒绝）
社保/公积金减员申报
```

**合规要点**：离职证明须如实出具；不得以工作交接未完成为由拖延发放工资
**离职面谈记录字段**：离职原因（分类）、对公司的建议、是否愿意推荐人才、保持联系意愿

### 流程04：绩效评估流程（季度/年度）

**触发条件**：绩效周期结束
**参与角色**：员工、直属上级、HRBP、各部门负责人

```
绩效周期开始：目标设定
  HR发布《目标确认模板》
  员工填写本周期目标（KPI/OKR）
  上级审核确认 → 双方签字 ← 存入档案

周期中（每月/每季）：
  1:1 进展对话（不正式评分，但须记录）

绩效周期结束：评估
  员工提交《自我评估》
  上级完成《上级评估》（可含360反馈）
  HRBP 汇总校准（避免部门评分通胀）
  最终结果确认 → 上级面谈告知员工

结果运用：
  绩效结果与调薪/晋升/年终奖挂钩
  不达标者：启动PIP或岗位调整流程
```

**注意事项**：绩效结果必须书面告知员工，建议员工签字确认；不接受PIP的情况需留存证据

---

### 流程05：调薪/晋升流程

**触发条件**：年度薪酬review / 员工晋升申请 / 特殊调薪需求

```
部门负责人提交《调薪/晋升申请》
  ↓（附：绩效评估结果、市场对标数据）
HRBP初审 → 薪酬委员会/CFO审批
  ↓（审批通过）
HR出具《调薪通知书》 → 员工签收
更新劳动合同附件（薪资变更协议）
  ↓
HRIS系统更新 → 财务同步调整
```

**合规要点**：降薪须获员工书面同意，否则违法；调薪决定应保密

---

### 流程06：员工请假流程

**触发条件**：员工申请休假
**假期类型**：年休假、事假、病假、婚假、丧假、产假/陪产假、调休

```
员工在系统中提交请假申请（填写：假期类型、起止时间、原因）
  ↓
直属上级审批（病假超过3天需提交医院证明）
  ↓（批准）
HR系统记录 → 考勤扣除/补偿相应天数
```

**特殊处理**：
- 病假连续超过3个月：启动医疗期管理流程
- 产假：提前通知HR申请生育保险报销
- 年假到期未休：公司可安排集中休假，若不安排需补偿300%工资

## 二、行政管理类流程

### 流程07：采购申请流程

**触发条件**：部门提出采购需求（办公用品/设备/软件/服务）

```
申请人提交《采购申请单》（含：品名/规格/用途/预算）
  ↓（行政）
行政确认库存 → 若无则进入采购流程
  ↓（采购）
金额 < 5000元：行政直接采购（≥2家询价）
金额 5000-50000元：≥3家报价 → 部门负责人审批
金额 > 50000元：≥3家报价 → 部门负责人+CFO审批 → 签订合同
  ↓（到货）
验收：申请人+行政共同验收 → 签《验收单》
入账：固定资产登记 / 费用报销
```

**禁止行为**：先购后批、化整为零规避审批

---

### 流程08：费用报销流程

**触发条件**：员工完成费用支出后申请报销

```
员工整理原始凭证（发票贴到报销单上）
填写《费用报销单》→ 直属上级审批
  ↓（金额超标准须说明原因）
财务审核（凭证真实性、标准合规性）
  ↓（审核通过）
财务打款 → 员工收款确认
```

**时效**：发生费用后10个工作日内提交；财务审核后5工作日内打款
**常见拒绝原因**：无原始发票、金额超标未说明、与业务无关、缺少审批

---

### 流程09：固定资产管理流程

**触发条件**：资产购入/领用/转移/报废

```
购入：采购完成 → 行政登记入台账（贴资产标签）→ 领用人签收
领用：填写《资产领用单》→ 行政更新台账（责任人）
转移：填写《资产转移单》→ 新旧责任人双签 → 台账更新
报废：申请人提交《资产报废申请》→ 行政确认残值 → 审批后注销台账
```

---

### 流程10：访客管理流程

```
被访部门提前登记访客信息（姓名、公司、来访目的、预计时间）
  ↓
前台/行政核实身份 → 登记来访记录 → 发放访客证
全程陪同或引导至指定区域
访客离开：收回访客证 → 记录离开时间
```

## 三、制度建设类流程

### 流程11：制度文件发布流程

**触发条件**：新增或修订公司规章制度

```
起草：HR/行政/相关部门起草初稿
  ↓
内审：法务/相关业务部门审阅（重点关注合规性）
  ↓
审批：CHRO/行政总监 → 必要时CEO审批
  ↓（重要制度须经员工代表大会/工会程序）
公示：至少公示5个工作日（用公告/钉钉/邮件通知全员）
员工确认：需要员工签字确认知悉（可用系统打卡替代）
  ↓
归档：制度文件归档（含版本号、生效日期、修订记录）
```

**合规要点**：规章制度"民主程序+公示程序"是认定合法性的关键，缺失将导致制度无效

---

### 流程12：合同审批流程（业务合同/服务合同）

```
业务部门起草合同初稿 → 提交合同审批申请
  ↓
法务审核（合规性、条款风险）
  ↓（金额分级）
≤10万：部门负责人审批
10-100万：部门负责人+CFO
>100万：部门负责人+CFO+CEO
  ↓（特殊合同加签CHRO/行政）
合同签署（加盖公章/法人章）→ 合同台账登记
归档：原件存档（纸质+电子扫描件）
```

## 四、快速参考表

### 审批层级参考（科技公司通用）

| 事项 | 阈值 | 审批人 |
|------|-----|--------|
| 日常采购 | <5000元 | 部门负责人 |
| 中等采购 | 5000-5万 | 部门负责人+行政总监 |
| 大额采购 | 5万-50万 | 部门负责人+CFO |
| 重大采购 | >50万 | 部门负责人+CFO+CEO |
| 招聘HC | 编制内 | 直属上级+HRBP |
| 增编 | 编制外 | 部门负责人+CHRO+CFO |
| 薪酬调整 | 任意 | HRBP+部门负责人+CFO |

### 常见时效要求

| 流程节点 | 时效标准 |
|---------|---------|
| 入职后签合同 | ≤30天（法定），建议当天 |
| 离职证明开具 | 离职当日或次日 |
| 工资发放 | 约定日期，最迟次月10日 |
| 报销打款 | 审核通过后5工作日 |
| 面试反馈 | 面试后2工作日内 |
| 绩效结果告知 | 评估完成后5工作日内 |
