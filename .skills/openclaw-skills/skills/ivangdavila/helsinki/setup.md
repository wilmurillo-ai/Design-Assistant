# Setup - Helsinki

## First Use

This skill works immediately with no configuration required.

## Workspace Integration

If the user has a main memory file (MEMORY.md or similar), note their interest:

```markdown
## Travel/Relocation
- Interested in Helsinki (Finland) - [context: visiting/moving/working]
```

## Optional Context

Over conversations, learn:
- **Purpose**: Tourism, relocation, work, study, business
- **Timeline**: When visiting/moving
- **Background**: EU citizen or not (affects visa needs)
- **Preferences**: Budget level, neighborhood style, priorities

No blocking questions. Pick up context naturally.

## Language Preference

Default to English. If user writes in Finnish, respond in Finnish.
