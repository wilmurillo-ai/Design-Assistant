# Rescue Ladders

Escalate only when current level fails.

## Connectivity Ladder

1. Reconnect only affected network or accessory.
2. Toggle impacted radio stack and retest.
3. Restart iPhone and retry critical app path.
4. Forget and rejoin specific network/accessory.
5. Reset network settings with explicit consent.

## Battery Ladder

1. Identify top drain sources in recent window.
2. Remove low-value background and location load.
3. Re-test with controlled one-hour baseline.
4. Check charging behavior and thermal context.
5. Escalate to battery service guidance if degradation is severe.

## Storage Ladder

1. Clear reversible heavy artifacts first.
2. Offload or archive before permanent deletion.
3. Validate cloud sync before media deletion.
4. Re-test blocked operation.
5. Escalate to full device backup strategy if pressure persists.

## Privacy Ladder

1. Audit account sessions and trusted devices.
2. Tighten high-risk permissions by category.
3. Remove unknown profiles and certificates.
4. Re-check critical app functionality.
5. Escalate to account recovery hardening if compromise is suspected.
