# Photo Protocol

## Before Taking Comparison Photos

- Ask whether photo tracking is actually useful for this case.
- Confirm the user is comfortable storing photo metadata locally.
- Do not ask for intimate-area or minor photos unless a lawful caregiver context is already clear and the user explicitly wants tracking.

## Standard Capture Rules

Use the same setup whenever possible:
- same camera
- same room or lighting
- same distance and angle
- same body position
- no beauty mode, filters, flash changes, or auto-enhancement

Capture three views when useful:
- one context shot for body location
- one medium shot for shape and distribution
- one close shot with a neutral scale reference

## Naming Convention

Use a file label like:

```text
YYYY-MM-DD_body-site_view_distance_note
```

Examples:
- `2026-03-08_left-forearm_overview_60cm_baseline`
- `2026-03-15_left-forearm_close_20cm_after-week-1`

## Comparison Rules

- Compare only photos with reasonably similar conditions.
- If lighting or distance changed too much, describe the comparison as limited.
- Separate visible change from user-reported symptoms such as itch or pain.
- Never convert a photo difference into a diagnosis.

## What to Log in `photos.md`

- date
- file or source
- body site
- view type
- capture conditions
- comparison caveats

If the visual evidence is weak, say so plainly and rely more on the dated timeline.
