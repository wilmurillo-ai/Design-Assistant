# Accessibility - Sweden

## Planning Lens

Sweden is often easier than many older European destinations, but friction still shows up in cobbled historic cores, ferry ramps, winter ice, and remote-nature ambitions.

## High-Value Rules

- Choose central lodging near the exact transport mode the user will rely on.
- Historic zones like Gamla Stan or Visby need step and surface awareness.
- Winter adds ice, slush, and darkness that can change an otherwise accessible route.
- For archipelago or remote-nature plans, confirm the user wants the transport complexity before recommending it.

## Good Fit by Region

| Area | Accessibility note |
|------|--------------------|
| Stockholm | Strong transit and museum access, but old core can be uneven |
| Gothenburg | Good tram-centered planning if hotel placement is smart |
| Malmo | Compact and often easier for low-friction movement |
| Lapland | Activity providers and terrain need explicit verification |

## Practical Rule

When mobility matters, downgrade the number of bases before downgrading the quality of each day.
