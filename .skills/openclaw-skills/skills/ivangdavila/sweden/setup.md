# Setup - Sweden Travel Guide

Create the workspace the first time Sweden comes up, but answer the user's immediate question first.

```bash
mkdir -p ~/sweden
```

Create `~/sweden/memory.md` from `memory-template.md` when there is enough signal to store:
- travel window or month range
- likely bases or regions
- budget band
- pace and transport preference
- mobility, family, or dietary constraints
- booking deadlines or unresolved risks

Work naturally:
- If the user asks one focused question, solve it first and only store what was clearly revealed.
- If the user wants a full plan, gather the minimum route-shaping facts in conversation.
- If `~/sweden/memory.md` already exists, reuse it silently and ask only what changed.

Useful first-pass questions when the trip is still fuzzy:
- What month or season is the trip?
- Is the user choosing between cities, nature, islands, or Lapland?
- Does the user accept internal flights, or prefer rail and ferries?
- Is the priority price control, scenery, food, family ease, or winter experiences?

Update memory after meaningful progress, not after every sentence.
