# Quizlet Study Modes

## Mode Selection Matrix

| Goal | Recommended Mode | Why |
|------|------------------|-----|
| First exposure | Learn | Immediate correction supports encoding |
| Exam rehearsal | Test | Simulates question pressure and recall |
| Fast warmup | Flashcards | Low-friction repetition before deep practice |
| Pattern speed | Match | Useful for recognition speed drills |

## Time-Boxed Session Plans

### 20-minute session
- 5 minutes: Flashcards to reactivate memory.
- 10 minutes: Learn mode on weakest tag.
- 5 minutes: Quick retest of recent misses.

### 45-minute session
- 10 minutes: Flashcards on mixed tags.
- 20 minutes: Learn mode on core concepts.
- 15 minutes: Test mode to simulate assessment conditions.

### 60-minute session
- 10 minutes: Review prior weak cards.
- 25 minutes: Learn mode with focused sub-set.
- 20 minutes: Test mode full run.
- 5 minutes: Record miss patterns and rewrite actions.

## Escalation Rules

- If Learn accuracy stays below 70 percent for two sessions, reduce set size and rewrite weak prompts.
- If Test score drops after card volume increase, split the set by topic and sequence.
- If progress plateaus, rotate from recognition-heavy modes to recall-heavy modes.

## Mode Traps

- Running Match as the main mode -> speed improves while conceptual recall stagnates.
- Repeating Test mode without card rewrites -> score plateaus from unresolved ambiguity.
- Switching modes every minute -> low focus and noisy performance signals.
