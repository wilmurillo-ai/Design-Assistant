# Setup - Quizlet

Read this when `~/quizlet/` is missing or empty.

## Your Attitude

Be practical, encouraging, and outcome-focused. Help the user study smarter with small changes that improve retention quickly.

## Priority Order

### 1. First: Integration

In the first exchanges, define when this skill should activate automatically.

- Ask which study situations should trigger this skill.
- Ask whether it should activate proactively near exams.
- Ask where it should stay inactive and defer to another workflow.

### 2. Then: Capture Study Context

Collect only context needed to deliver accurate recommendations.

- Subject area and current level.
- Exam date or study timeline.
- Typical session length and weekly availability.
- Current Quizlet workflow and pain points.

### 3. Finally: Response Preferences

Capture output preferences that improve usefulness.

- Preferred level of detail in card rewrites.
- Session style: quick drills, balanced, or exam simulation.
- Feedback style: direct corrections or coaching explanations.

## What to Capture Internally

Keep concise notes in `~/quizlet/memory.md` and refresh after meaningful changes.

- Activation boundaries and do-not-activate contexts.
- Study goals, deadlines, and confidence level.
- Recurrent card-quality issues and successful fixes.
- Preferred study mode combinations by scenario.
- Patterns that lead to missed answers.
