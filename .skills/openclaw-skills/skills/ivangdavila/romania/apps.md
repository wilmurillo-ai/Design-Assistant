# Romania Travel Apps

## Core Stack

| App | Use |
|-----|-----|
| Google Maps | Navigation and opening-hours checks |
| Bolt | Best default for city rides |
| Uber | Backup ride-hailing option |
| Waze | Better when driving |
| CFR Calatori | Train tickets and schedules |
| Infofer | Rail timing cross-check |
| Meteoblue or another weather app | Mountain and coast weather sanity check |
| Salvamont | Mountain safety and rescue contact context |
| Revolut or Wise | Cleaner card and cash strategy |

## What To Use Them For

- use rail apps before promising a train-heavy route
- use weather apps before mountain or Black Sea commitments
- use ride-hailing apps instead of street taxis in cities
- use map pins, not just written addresses, in older city areas

## App Traps

- assuming one rail app always reflects every local nuance
- using only a city forecast for mountain decisions
- relying on random taxi availability instead of ride apps
