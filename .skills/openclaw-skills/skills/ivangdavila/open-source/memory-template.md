# Memory Template - Open Source

Create `~/open-source/memory.md` with this structure:

```markdown
# Open Source Memory

## Status
status: ongoing
version: 1.0.0
last: YYYY-MM-DD
integration: pending | done | declined

## Context
- Current objective:
- Primary domain:
- Stack and constraints:
- Operating model (self-host, managed, hybrid):

## Notes
- Stable preferences observed from behavior:
- Past recommendations and outcomes:
- Risks to watch in future decisions:

---
Updated: YYYY-MM-DD
```

## Status Values

| Value | Meaning | Behavior |
|-------|---------|----------|
| ongoing | Learning context | Keep capturing patterns naturally |
| complete | Enough context captured | Focus on execution support |
| paused | User said not now | Use existing context without pushing |
| never_ask | User opted out | Do not request extra profiling |
