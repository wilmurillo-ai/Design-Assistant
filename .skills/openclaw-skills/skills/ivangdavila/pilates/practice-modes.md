# Practice Modes - Pilates

Use one mode at a time.
Switch modes when the user goal changes, not just because the exercise list changes.

## `start`

Use when the user is new, returning after a long break, or unsure what Pilates should feel like.

Focus:
- breathing without bracing too hard
- ribs and pelvis organization
- slow, low-complexity exercises
- ending the session with confidence, not fatigue

Good output:
- one 5 to 10 minute session
- one plain-language cue
- one reassurance about what "good enough" looks like

## `session`

Use for normal guided Pilates practice.

Focus:
- a short warm-in
- two to four exercises with clear intent
- one closing reflection on control, breath, or tolerance

Good output:
- clear sequence
- short cue list
- finish with one note worth logging

## `repair`

Use when the user says an exercise feels wrong, unstable, rushed, or irritating.

Focus:
- identify one breakdown pattern
- reduce complexity or range
- give one easier drill and one re-test cue

Good output:
- pattern identified
- single correction cue
- simpler drill that preserves the intent

## `build`

Use when consistency already exists and the user wants progression.

Focus:
- weekly cadence
- control before volume expansion
- gradual increase in complexity, not ego intensity

Good output:
- plan for 1 to 2 weeks
- measurable success condition
- fallback session for low-energy days

## `recover`

Use when energy is low, soreness is elevated, or the user needs the gentlest viable practice.

Focus:
- shorter sessions
- supported or reduced-range drills
- calmer pacing with clear stop conditions

Good output:
- conservative session
- explicit modification choices
- recommendation to pause progression until symptoms settle

## Mode Switching Rules

- `start` -> `session`: after 2 to 4 comfortable sessions with no major confusion.
- `session` -> `repair`: when the same error or discomfort repeats.
- `session` -> `build`: when adherence is stable and the main cue holds.
- Any mode -> `recover`: if symptoms, fatigue, or fear rise during practice.
