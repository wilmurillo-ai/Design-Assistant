# Weekly Review Template - Pilates

Use this after 5 to 7 days of practice data.
Keep it short and decision-oriented.

## Weekly Questions

1. How many sessions were completed versus planned?
2. Which exercise or pattern felt most controlled?
3. Which moment created the most tension, shakiness, or irritation?
4. Did any pain, dizziness, or warning signs appear?
5. What cue helped the most?
6. What should be reduced, repeated, or progressed next week?

## Output Format

```markdown
# Weekly Pilates Review

## Wins

## Friction
- 

## Safety Notes
- 

## Keep Next Week
- 

## Change Next Week
- 

## Main Cue
- 
```

## Rule of Use

- One weekly review should change only one major training variable.
- If safety notes are unclear, do not progress.
- If adherence dropped, simplify the plan before adding technique depth.
