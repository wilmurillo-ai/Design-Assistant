# Practice Week Blueprint

Design practice backwards from the next basketball problem.

## Weekly Rhythm

| Schedule | Primary goal | Typical flow |
|----------|--------------|--------------|
| 1 game per week | build, install, sharpen, taper | recover -> load -> game-plan -> special situations -> game |
| 2 games per week | recover, refresh, clarify | recover -> short install -> game -> recover -> tune -> game |
| Youth development block | teach habits through repetition | skill -> advantage game -> decision game -> transfer |

## Session Checklist

- One objective only
- Player numbers and role distribution
- Space and basket usage
- Work-rest pattern
- Coaching cues
- Progression and regression
- Stop condition if quality drops

## Common Session Types

| Need | Best session frame |
|------|--------------------|
| Improve spacing and reads | shell spacing to guided advantage to live 4v4 |
| Improve transition defense | conversion drill to disadvantage scramble to scored game |
| Improve ball pressure handling | trap release drill to numbered-break decisions |
| Improve pick-and-roll coverage | walk-through to scripted reps to random live reads |
| Improve rebounding habits | hit-find-pursue drill to score-consequence game |

## Guardrails

- Do not overload one week with too many new concepts
- Match volume and intensity to the real calendar
- If the tactical problem is unclear, do not fake precision with fancy drills
