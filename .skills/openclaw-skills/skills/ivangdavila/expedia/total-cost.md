# Total-Cost and Package Comparison

Use this file when the user wants the best Expedia option, not just the lowest headline price.

## Compare these layers

For each option, separate:
- headline price
- taxes and mandatory fees
- cancellation flexibility
- package inclusions
- exposed costs that remain outside Expedia's shown total

## Package logic

Packages can be valuable when:
- flight plus stay discounts are real
- dates are stable
- location and schedule still fit the mission

Packages become dangerous when:
- the flight shape is weak
- bag or seat costs erase the savings
- the stay becomes non-refundable while the user still needs flexibility

## Rules

- Show the cheapest visible option and the safest real option if they differ.
- If an option depends on assumptions not yet verified, mark it as provisional.
- Multi-room or family cases need explicit recalculation, not single-room multiplication.
