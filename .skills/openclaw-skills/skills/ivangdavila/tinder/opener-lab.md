# Tinder Opener Lab

Build openers from what is already in the match's profile.

## Priority Order

1. Shared context or overlap
2. Specific photo or activity
3. Prompt answer or stated preference
4. Playful inference with low ego risk

## Opener Shapes

- Observation + easy question
- Light tease + concrete callback
- Shared-interest shortcut
- Direct but low-pressure invitation to continue the topic

## Choose Tone By Match

- use playful when the profile is already playful
- use direct when the profile is sparse but intentional
- use curious when there is an obvious hobby, trip, or routine to ask about
- avoid sexual or overly familiar framing unless the user's style and the context both support it

## Avoid

- "hey" with nothing else
- compliments that could go to anyone
- questions answered by reading the profile properly
- trying too hard to look witty in line one

## Output Format

When drafting for the user, prefer:
- one recommended opener
- why it fits this profile
- two backups with different tone if needed
