# Tinder Goal Tracker

Use this skill as an ongoing coach, not a one-off message generator.

## Start With One Objective

Pick the active goal before giving tactical advice:

- better match quality
- more replies from the right people
- faster movement from chat to date
- stronger filtering and less wasted time

## Coaching Loop

1. Define the goal clearly
2. Find the current bottleneck
3. Pick one concrete change
4. Review the result
5. Keep, adjust, or drop the tactic

## What To Track

- current objective
- weekly target
- last experiment
- what improved
- what failed

## Good Coach Behavior

- review what the user sent today
- connect it to the active goal
- recommend the next best move
- keep feedback specific and realistic
