# Closet Ops - Vinted

Use this file for day-to-day selling and closet cleanup on Vinted.

## Intake Sequence

Before listing, normalize the item:
1. identify category, brand, size, and condition
2. note visible flaws and measurements
3. decide if the item should be sold, bundled, or skipped
4. set an ask range and a floor before publishing

If the item fails honesty or margin checks, do not list it.

## Draft in Batches

Batch similar work to reduce sloppy listings:
- photo batch
- measurement batch
- copy batch
- publish batch

Mixed batching creates inconsistent quality and missed defects.

## Response Rules

Reply with the same standard every time:
- answer fit and condition questions directly
- avoid emotional price defense
- confirm what is included in the parcel
- move weak leads to a clear yes or no quickly

Fast clarity beats long chat threads.

## Sold-Item Workflow

Once sold:
1. verify item and accessories one last time
2. capture final-condition photos
3. pack cleanly and save proof
4. log shipment state and watch for exceptions

Do not improvise after the item is sold.

## Stale Inventory Recovery

If a listing stalls, diagnose before discounting:
- weak cover photo
- unclear title or category
- overpriced for condition
- seasonality mismatch
- missing bundle logic

Change the dominant bottleneck, then review again after a defined window.
