# Pro Ops - Vinted

Use this file only when the user sells as a business or has access to Vinted Pro features.

## Activation Rule

Start in Pro mode only when the user explicitly operates as a business.
Do not assume Pro workflows for casual closet cleanup.

## Operating Standard

Pro mode needs tighter discipline:
- faster response windows
- cleaner catalog consistency
- documented service rules
- batch intake and shipping routines

What feels optional for a casual seller becomes required for repeatable business output.

## Catalog Segmentation

Group stock by:
- category
- season
- price band
- condition tier

Segmentation prevents premium pieces from being managed with bargain-bin logic.

## Margin and Service Rules

Before scaling activity, define:
- target margin by category
- maximum prep time per item
- acceptable return or issue rate
- minimum proof standard per shipment

If service quality drops as volume rises, slow down before adding more stock.

## Market and Compliance Notes

Vinted Pro availability and business requirements vary by market and account status.
Verify local tax, invoicing, and policy obligations before building volume-dependent workflows.
