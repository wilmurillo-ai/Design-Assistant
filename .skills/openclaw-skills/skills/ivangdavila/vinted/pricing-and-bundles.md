# Pricing and Bundles - Vinted

Use this file when the user asks what price to set, whether to accept an offer, or how to handle bundles and paid visibility.

## Pricing Baseline

Build a realistic range from:
- brand demand and category liquidity
- item condition and completeness
- seasonality
- speed needed to sell
- seller profile: closet cleanup or repeat reseller

Fair pricing is a range, not one magic number.

## Offer Lanes

Define three numbers before going live:
- ask price
- good-offer line
- hard floor

This prevents emotional negotiation and inconsistent replies.

## Bundle Policy

Bundles work when they increase order value without hiding weak items.

Set a simple rule:
- small discount for two items
- larger but controlled discount for three or more
- no extra discount if one item is already near floor

Document the rule so every bundle decision is consistent.

## Markdown Ladder

For stale stock, reduce in steps:
1. improve listing package
2. wait a defined review window
3. apply a measured cut
4. reassess bundle attractiveness

Large random markdowns train buyers to wait.

## Visibility Spend

If the account offers bumps or wardrobe-level visibility tools, use them only when:
- photos and copy are already strong
- price is inside the fair range
- the item has enough margin to justify the spend

Paid visibility cannot fix a weak offer.
