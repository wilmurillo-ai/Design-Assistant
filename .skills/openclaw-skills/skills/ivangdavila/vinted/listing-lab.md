# Listing Lab - Vinted

Use this playbook when a listing is underperforming or when the user wants a stronger listing before publishing.

## Listing Stack

Treat the listing as one system:
- cover photo
- supporting detail photos
- title opening words
- condition honesty
- measurements and fit notes
- price and bundle logic

Fix the stack in that order for most apparel listings.

## Photo Rules

Prioritize trust:
- bright front image with clean background
- close-ups of fabric, tags, and known flaws
- fit or measurement context when size is likely to be questioned
- avoid filters that distort color or wear

If photos create doubt, price pressure increases.

## Copy Rules

The title should lead with the searchable parts:
- brand
- item type
- size or fit cue
- standout attribute only if real

Descriptions should answer what the buyer would ask next, not repeat filler.

## Condition and Measurement Discipline

Always state:
- condition grade in plain language
- any defect worth mentioning
- key measurements when fit is a likely blocker
- what is and is not included

Omissions create friction later in chat or disputes.

## Rewrite Sequence

When improving a weak listing:
1. fix cover photo
2. fix title opening
3. add missing fit and flaw details
4. review price and bundle logic

Do not cut price first if the listing still feels unclear.
