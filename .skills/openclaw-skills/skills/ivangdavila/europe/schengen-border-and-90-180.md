# Schengen, Borders, and 90/180

Use this for multi-country trips, long visits, split-country travel, and border planning.

## Core Idea

For many non-EU visitors, Schengen time is counted across the area, not country by country.

That means:
- leaving France for Spain does not reset a Schengen clock
- combining several Schengen countries still counts toward the same stay limit
- non-Schengen countries can sometimes be used as buffer space, but only if the user's nationality and route support that plan

## Why This Matters

- Users can accidentally overstay even while crossing many borders.
- Rail-heavy or backpacking itineraries create false confidence because movement feels casual.
- Split-time plans like "90 days in Spain, then 90 in Italy" can be legally wrong.

## Guidance Rules

- Ask whether the user is an EU citizen, UK citizen, or another non-EU national.
- Ask which countries are in the itinerary.
- Ask how many days they have already spent in Schengen in the relevant window.
- If precision matters, use an official calculator or official Schengen guidance before confirming legality.

## Border Reality

- Schengen often reduces routine internal border checks, but users should still travel with proper ID or passport.
- Transport disruptions, ad hoc checks, and non-Schengen segments can change how a route works in practice.
- UK, Ireland, Balkans, and some southeastern corridors often require different assumptions.

## Good Use Cases

- building a 3-month Europe trip
- designing a split between Schengen and non-Schengen countries
- helping remote workers avoid accidental overstay
- checking whether a cheap flight route creates a border or document problem
