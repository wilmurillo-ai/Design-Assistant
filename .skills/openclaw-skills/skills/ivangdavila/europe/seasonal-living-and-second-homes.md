# Seasonal Living and Second Homes

Use this when the user spends only part of the year in Europe, owns or wants a second place, or is testing a Europe base before committing.

## Typical Profiles

- winter-in-south households
- retirees splitting time across continents
- second-home owners
- remote workers staying in Europe for part of the year
- families using Europe around school breaks or summer only

## Core Questions

1. Which months are spent in Europe?
2. In which country?
3. Is this tourism, legal residence, or a soft-launch before moving?
4. Where are healthcare, taxes, vehicles, and documents anchored?
5. Who handles the home when the user is away?

## What Changes for Seasonal Users

- Schengen time can become a hard constraint.
- Tax residence and social-security assumptions can get messy.
- Insurance for an empty property matters.
- Banking, mail, and local admin may still matter even without full residency.
- Airport access often matters more than neighborhood romance.

## Advice Pattern

- Keep recommendations reversible while the user is still testing Europe.
- Optimize for resilience and simplicity first.
- Treat seasonal staying as its own mode, not a weaker version of either tourism or full residence.
