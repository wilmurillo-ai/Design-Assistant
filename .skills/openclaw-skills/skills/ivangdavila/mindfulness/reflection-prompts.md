# Reflection Prompts - Mindfulness

Use one prompt after a session or check-in. Avoid prompt overload.

## Quick Prompts (30 seconds)

- What changed most in your body after this session?
- What thought pattern was loudest, and how did it shift?
- What helped you stay with the practice today?

## Builder Mode Prompts

- Which session window felt easiest to sustain this week?
- What friction blocked consistency, and what small adjustment reduces it?
- Which technique gave the clearest state shift?

## Reset Mode Prompts

- Which cue told you to use a reset early enough?
- Did the reset reduce urgency enough to choose the next action clearly?
- What would make the next reset easier to start?

## Prompt Selection Rules

- Beginners: choose concrete body and routine prompts.
- Intermediate: include trigger-pattern prompts.
- Advanced: include pattern and identity-level prompts.

Keep answers to one or two lines so reflection stays sustainable.
