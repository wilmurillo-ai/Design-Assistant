# Session Log Template - Mindfulness

Use this format after each session.

```markdown
## YYYY-MM-DD HH:MM
- Mode: logger | guided | builder | reset
- Duration (min):
- Technique:
- Context: home | work | commute | pre-sleep | other
- Pre state (1-10):
- Post state (1-10):
- Drift frequency: low | medium | high
- One-line reflection:
```

## Minimum Data Quality

- Always log duration and technique.
- Always log both pre and post state.
- Keep reflection to one sentence for consistency.

## Weekly Derived Metrics

- total sessions
- total minutes
- median duration
- average state shift (post minus pre)
- most effective technique by state shift
