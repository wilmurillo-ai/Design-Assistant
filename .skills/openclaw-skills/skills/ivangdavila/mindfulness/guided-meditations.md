# Guided Meditations - Mindfulness

Keep the tone direct and calm. Avoid long theory during live guidance.

## 1-Minute Reset

1. Invite one slow inhale and one longer exhale.
2. Ask the user to place attention on breath at nose or chest.
3. Repeat: notice distraction, label "thinking", return to breath.
4. Close with one sentence: "Pick your next action from a steadier state."

## 3-Minute Breathing Space

1. Minute 1: notice current body, thoughts, and emotions without fixing.
2. Minute 2: narrow attention to breathing rhythm.
3. Minute 3: widen awareness to whole body and immediate environment.
4. Close with a one-line intention for next 30 minutes.

## 5-Minute Body Scan

1. Set posture and jaw release cue.
2. Move attention feet -> legs -> torso -> shoulders -> face.
3. At each area: observe sensation, soften unnecessary tension.
4. If mind drifts, return to last body area.
5. End with one full-body breath and posture reset.

## 10-Minute Open Monitoring

1. Start with 1 minute of breath anchoring.
2. Shift to observing thoughts, sounds, and sensations as events.
3. Use neutral labels: "thinking", "hearing", "feeling".
4. Avoid storyline engagement; return to open observation.
5. Close by naming one useful insight and one grounded next step.

## Customization Rules

- If user is anxious, shorten instructions and increase pacing cues.
- If user is sleepy, use upright posture and eyes-open option.
- If user wants secular framing, avoid spiritual language.
