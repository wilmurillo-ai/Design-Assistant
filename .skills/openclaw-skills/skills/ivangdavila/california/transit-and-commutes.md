# Transit and Commutes — California

Use this file for car-light vs car-heavy decisions, parking, transit passes, and daily movement planning.

## California Commute Rule

The correct region is often the one that preserves weekly energy, not the one with the best brand.

## Movement Models by Context

| Context | Typical pattern |
|---------|-----------------|
| San Francisco core and some Bay Area corridors | Car-light or no-car can work |
| Los Angeles and Orange County | Often one-car or two-car reality depending on household |
| San Diego | Hybrid model: some transit, but car still common |
| Sacramento and Inland Empire | Car-first in most daily routines |

## Commute Rules

### 1. Test the route that will actually happen
- Peak-hour commute, parking search, transit transfer, bridge toll, or school drop-off chain all change the answer.

### 2. Parking is part of housing cost
- Never compare units without asking whether parking is included, possible, safe, and realistic.

### 3. Car-free claims need proof
- California has pockets where a car-free life works well, but many users hear "walkable" and still end up trapped by errands or school runs.

## Good Answer Pattern

1. Ask where the user starts, where they need to go, and how often.
2. Ask whether parking, school drop-off, or airport access matters weekly.
3. Compare no-car, one-car, and two-car models when appropriate.

## Common Mistakes

- Recommending housing based on rent while ignoring commute time.
- Forgetting bridge tolls, express lanes, or parking cost.
- Overestimating transit usefulness in car-first submarkets.
- Treating Bay Area and LA movement as the same problem.
