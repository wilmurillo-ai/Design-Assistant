# California DMV and Vehicles

Use this file for driver license, REAL ID, registration, smog, insurance, and vehicle-readiness questions.

## Core Workflow for New Residents

### Driver license and ID
- California residents need a valid California driver license to operate a motor vehicle in the state.
- New residents should handle the California driver license workflow promptly rather than driving for months on inertia.
- Use California DMV for both license and ID workflows and verify the current document set before booking an appointment.

### Vehicle registration
- New-to-California registration, title, and plate questions live under California DMV guidance.
- Smog checks are often part of California registration or renewal, but exceptions exist.
- Always verify the current vehicle-specific and county-specific requirements before giving a final sequence.

## REAL ID and Identity Planning

- REAL ID is worth raising when the user is already changing residency documents.
- Tell the user to verify the exact document set before the appointment instead of assuming a standard license packet is enough.

## Insurance Baseline

- Minimum legal coverage is only the floor, not the recommendation.
- For most California drivers, discuss collision, uninsured drivers, theft, parking exposure, and commute distance before treating minimum-only coverage as adequate.
- Coastal, wildfire-edge, and dense-urban parking environments create very different risk profiles.

## Parking and Car-Need Reality

- California car logic changes by submarket:
  - much of LA and Orange County: often car-heavy
  - some Bay Area and SF contexts: viable without a car
  - Sacramento, Inland Empire, and many suburbs: car-first by default

## Good Answer Pattern

1. Identify whether the issue is license, registration, smog, insurance, or parking reality.
2. Ask city, county, and current origin state or country.
3. Give the sequence, documents, and official verification portal.
4. Flag parking, smog, and insurance realities if they apply.

## Common Mistakes

- Giving DMV steps without mentioning smog or registration dependencies.
- Recommending a car-free plan in a submarket that does not support the user's routine.
- Treating insurance minimums as enough guidance.
- Ignoring parking cost and theft risk when comparing neighborhoods.
