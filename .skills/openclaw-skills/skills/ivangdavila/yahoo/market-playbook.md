# Yahoo Market Playbook

Use this when the user wants more than a raw quote.

## 1. Name the job

- Quick tape read
- Pre-market brief
- Post-market review
- Swing setup review
- Event watch
- No-trade monitoring

## 2. Anchor the comparison set

Always define what the symbol should be compared against:
- broad index
- sector ETF
- direct peers
- crypto benchmark
- currency basket

If there is no comparison set, say so explicitly.

## 3. State what Yahoo is showing now

Summarize:
- current price
- daily move
- market state or timestamp
- 52-week context
- volume or liquidity clue

Keep this separate from interpretation.

## 4. Map the nearest catalyst

Look for the next window that could matter:
- earnings
- macro releases
- guidance updates
- product or regulatory events
- sector rotation

If the setup has no obvious catalyst, mark it as watchlist-only unless the user wants a purely technical read.

## 5. Convert to a decision frame

End with one of these:
- trade candidate
- watchlist candidate
- needs more confirmation
- no-trade

Every decision frame should include why now, what would confirm it, and what would break it.
