# Yahoo Risk Playbook

Use this before suggesting urgency, conviction, or size.

## Pre-decision checklist

- Is the market state clear: live, at close, pre-market, or after-hours?
- Is the instrument liquid enough for the user's horizon?
- Is a catalyst near enough to make today's print misleading?
- Is the user asking for data only, or for a trade-style opinion?
- Is there an explicit invalidation, or is the setup still just narrative?

## Red flags

- Very thin volume
- Extreme move with no clear catalyst
- Symbol ambiguity
- User has no stated horizon
- User is asking for certainty instead of scenarios

## Output rules

- If risk is unclear, downgrade to watchlist or needs more confirmation.
- If the user wants a trade-style answer, include invalidation and what evidence would reverse the call.
- If the data is stale or outside regular hours, say that first.
- Never imply order execution or account access.
