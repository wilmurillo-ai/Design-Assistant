# Useful Apps — New Zealand

## Essential

| App | Why |
|-----|-----|
| Google Maps | Core navigation and hours |
| Maps.me or offline Google Maps | Backup in low-signal areas |
| MetService | Official weather forecasts and warnings |
| NZTA Journey Planner / traffic tools | Road alerts, closures, travel times |
| Air New Zealand | Domestic check-in and disruption handling |
| InterCity | Main intercity bus booking |

## Region-Specific Transit

| City | App |
|------|-----|
| Auckland | AT Mobile |
| Wellington | Metlink |
| Christchurch | Metroinfo tools or card info |

## Outdoor and Booking Tools

| Tool | Use |
|------|-----|
| DOC website/app tools | Great Walks, huts, track alerts |
| Interislander or Bluebridge app/site | Ferry bookings and disruption checks |
| Uber | Useful in Auckland, Wellington, Christchurch, Queenstown |

## Good Habits

- Screenshot key bookings before losing signal.
- Download offline maps before Fiordland, West Coast, and rural Northland drives.
- Check weather the night before and morning of any serious hike.
- Use one source of truth for bookings so ferry, flight, and hotel times stay aligned.
