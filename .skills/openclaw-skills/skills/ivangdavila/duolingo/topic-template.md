# Topic Namespace Templates

Create these files for every topic slug under `~/duolingo/topics/<topic-slug>/`.

## profile.md

```markdown
# Topic Profile - <topic-slug>

- Goal:
- Current level:
- Weekly target loops:
- Constraints:
- Preferred style:
```

## curriculum.md

```markdown
# Curriculum - <topic-slug>

## Lanes
- Foundation
- Core
- Challenge

## Units
- U1:
- U2:
- U3:
```

## queue.md

```markdown
# Queue - <topic-slug>

## Next Lessons
- [ ] L1
- [ ] L2

## Reviews Due
- [ ] R1
```

## sessions.md

```markdown
# Sessions - <topic-slug>
```
