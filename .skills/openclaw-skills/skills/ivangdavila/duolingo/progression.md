# Progression Economy

Use one transparent economy across all topics.

## Shared Signals

| Signal | Purpose | Default |
|--------|---------|---------|
| XP | Learning momentum | 10 XP per clean loop |
| Hearts | Mistake budget | 4 hearts per topic session |
| Streak | Daily consistency | counts day with >=1 completed loop |
| Mastery tier | Competence state | novice -> active -> solid -> fluent |

## Multi-Topic Rules

- XP can be global and per-topic.
- Streak is global, but track completion is per-topic.
- Hearts reset per topic session, not globally.

## Reward Logic

- Full XP for clean pass.
- Reduced XP for pass after hints.
- Bonus XP for completed review queue item.
- No XP for skipped attempts.

## Recovery Rules

- One grace miss per week for global streak.
- After streak break, offer comeback loop with reduced difficulty.
- Do not punish returners with overloaded queues.

## Anti-Patterns

- Inflating XP without mastery evidence.
- Streak rules that delete motivation after one miss.
- Shared hearts across topics causing unnecessary frustration.
