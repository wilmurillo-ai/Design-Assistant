# Family Templates

Use these templates to create useful family notes quickly.

## Weekly Plan

```markdown
# Weekly Plan

## Priority Events
| Date | Event | Owner | Prep | Fallback |
|------|-------|-------|------|----------|
|      |       |       |      |          |

## Family Tasks
- [ ] task / owner / due

## Shopping Trigger
- item / why / store

## Risks
- collision, missing form, transport gap, low supplies
```

## Handoff Note

```markdown
# Handoff

- Next action:
- Deadline:
- Risk:
- Fallback:
```
