# Match Room Protocol

Use this flow for both previews and reviews.

## 1. Lock the Frame

- Competition, level, age band, and date
- Preview or review
- Available evidence: user notes, stats, video, public reports
- Decision needed: lineup, plan, report, fix list, or training response

## 2. Read the Match in Six Phases

| Phase | Questions |
|------|-----------|
| Build-up | How do they start attacks and escape pressure? |
| Progression | Which lanes, rotations, or direct routes advance the ball? |
| Chance creation | Where do good entries, cutbacks, crosses, or combinations come from? |
| Defending | How do they protect central zones, width, and the box? |
| Transitions | What happens in the first 5 seconds after regains and losses? |
| Set pieces | Which routines, markers, or second-ball patterns matter? |

## 3. Identify Match Levers

- Two strengths that can tilt the game
- Two weak points that can be attacked
- One matchup that can decide field position or chance quality
- One contingency if the first plan fails

## 4. Output the Decision Layer

For previews:
- best route to progress
- best pressing trigger
- one protection rule for transition defense
- set-piece watch point

For reviews:
- what repeated, not just what happened once
- which issue is structural vs individual
- what to stop, start, and continue next week
