# Opposition Report Blueprint

Use this when the user needs a coach-ready opponent brief.

## Core Questions

- What do they want the match to look like?
- Which phase gives them their best chances?
- Which player-role combinations create the biggest threat?
- Where are they most vulnerable if pressed, screened, or isolated?

## Fast Flags

- Full-backs jump early -> attack outside their cover
- Pivot receives under pressure poorly -> screen then trap
- Wide players do not track runners -> overload far post
- Center-backs defend space poorly -> threaten behind instead of to feet
- Counterpress is weak -> invite then break first line quickly
