# Training Week Blueprint

Design sessions backwards from the next football problem.

## Weekly Rhythm

| Schedule | Primary goal | Typical flow |
|----------|--------------|--------------|
| 1 match per week | build, train, sharpen, taper | recover -> load -> game-plan -> speed -> match |
| 2 matches per week | recover, refresh, clarify | recover -> short tactical tune -> match -> recover -> tune -> match |
| Youth development block | teach principles repeatedly | technical repetition -> opposed constraint -> game transfer |

## Session Checklist

- One objective only
- Player numbers and role distribution
- Area size and direction
- Work-rest pattern
- Coaching cues
- Progression and regression
- Stop condition if quality drops

## Common Session Types

| Need | Best session frame |
|------|--------------------|
| Break first line | rondo to directional game to wave attack |
| Improve rest defense | positional game with counter goals and restart pressure |
| Fix final-third entries | pattern to opposed zone game to conditioned finish |
| Improve pressing | trigger rehearsal to small-sided chaos to full-unit reps |
| Build transition threat | regain game with short time-to-shot rule |

## Guardrails

- Do not overload one week with too many new ideas
- Match intensity and volume to the real calendar
- If the tactical problem is unclear, do not fake precision with fancy drills
