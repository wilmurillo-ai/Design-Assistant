# Safety Checklist — PowerPoint

Use this before any risky PowerPoint Desktop action.

## Destructive or high-risk actions

- delete slide
- reorder slides in bulk
- overwrite export targets
- replace notes across the whole deck
- close without save

## Required preflight

1. Identify the exact presentation path or open presentation name.
2. Confirm whether the action is slide-level or deck-wide.
3. State whether the action is reversible.
4. State whether a copy or Save As path exists.
5. State whether slideshow or presenter state matters.

## Safe fallback moves

- create a copy first
- export instead of overwrite
- test on one slide before applying globally
- read back the active slide and deck state before and after the action
