# Substitutions Playbook — Walmart

Substitutions are a policy decision, not a cleanup step.

## Block by Default

Never auto-approve substitutions for:
- allergy-sensitive foods
- baby formula, infant food, or medically sensitive products
- pet food when the animal has a known brand or ingredient restriction
- pharmacy items or anything the user treats as health-critical

## Usually Flexible

These often allow bounded swaps if the user agrees:
- produce within the same cooking role
- pantry staples with same size tier and ingredient profile
- paper goods and cleaning items with equivalent count or concentration
- frozen vegetables with same cut and no sauce differences

## Value Guardrails

When proposing a substitute, check:
- unit value versus original
- pack size inflation
- flavor or dietary mismatch
- waste risk from oversized replacements

## Good Output Pattern

For each replaceable item, show:
- primary choice
- acceptable substitute
- cheaper trade-down if relevant
- when to reject all substitutions and leave the item unfilled

## Escalate Immediately

Escalate instead of guessing when:
- the item touches allergy, baby, pet, or pharmacy rules
- the user gave a strict brand or ingredient requirement
- the best substitute changes the use case, not just the label
