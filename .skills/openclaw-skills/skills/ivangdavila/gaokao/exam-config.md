# Gaokao Exam Configuration

## Exam Structure (2024+)

### Core Subjects (必考)
| Subject | Points | Duration | Format |
|---------|--------|----------|--------|
| 语文 (Chinese) | 150 | 150 min | Essay + reading + classical Chinese |
| 数学 (Math) | 150 | 120 min | Multiple choice + fill-in + problems |
| 英语 (English) | 150 | 120 min | Listening + reading + writing |

### Elective System

**3+1+2 Model (most provinces):**
- Choose 1 from: 物理 (Physics) OR 历史 (History) — 100 points raw
- Choose 2 from: 化学, 生物, 政治, 地理 — 100 points each (scaled)
- Total: 750 points

**3+3 Model (Shanghai, Zhejiang, etc.):**
- Choose 3 from all electives — 100 points each (scaled)
- Total: 750 points

### Score Scaling (等级赋分)

For scaled subjects, raw scores convert to ranks:
| Rank | Percentage | Scaled Score Range |
|------|------------|-------------------|
| A | Top 3% | 91-100 |
| B+ | 3-10% | 81-90 |
| B | 10-26% | 71-80 |
| C+ | 26-50% | 61-70 |
| C | 50-74% | 51-60 |
| D | 74-90% | 41-50 |
| E | Bottom 10% | 30-40 |

## Provincial Variations

Major differences by province:
- **Beijing/Shanghai**: Lower competition, higher cutoffs
- **Henan/Shandong**: Extreme competition, 1M+ test takers
- **Jiangsu**: Historically harder math papers
- **Zhejiang**: 3+3 system, multiple English attempts
