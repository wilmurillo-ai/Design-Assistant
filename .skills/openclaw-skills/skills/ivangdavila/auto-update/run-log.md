# Run Log Contract - Auto-Update

This document defines the compact run history.

## Every Entry Needs

- timestamp
- trigger source
- OpenClaw result
- skills updated or skipped
- backup references
- migration notes
- verification status
- next action

## Keep It Compact

The run log is for quick scanning before the next update pass.
Do not paste full terminal logs here. Save only the facts needed to understand what happened and what to do next.
