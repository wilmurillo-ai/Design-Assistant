# Telecoms and Apps — Switzerland

## Connectivity Reality

- Switzerland often breaks EU roaming assumptions, so verify the user's phone plan before the trip.
- If the route is rail-heavy, mobile data matters for live platform checks and weather shifts.
- Border-heavy itineraries can trigger extra billing surprises if the user is careless.

## Essential Tools

- SBB Mobile for timetables, platforms, and live rail planning
- MeteoSwiss for weather and warnings
- swisstopo or the official map stack for outdoor orientation
- Rega app or official rescue guidance for serious alpine users

## Good Default Advice

- Confirm roaming before arrival instead of solving it on the platform.
- Download key routes and bookings in advance.
- If the user is hiking, do not rely on one battery and one signal source.

## Low-Signal Trap

The most common telecom mistake is not technical failure. It is assuming Switzerland behaves like any EU stop for pricing and coverage decisions.
