# How to Choose a Macau Base

## Start With This Question

Which version of Macau matters most?

- Heritage and old streets -> historic peninsula
- Balanced base -> Taipa
- Luxury resort weekend -> Cotai
- Quiet and green -> Coloane
- Ferry and business convenience -> NAPE / Outer Harbour

## Best Match Table

| Priority | Best Base | Why |
|---------|-----------|-----|
| First-time sightseeing | Historic peninsula or Taipa | Best balance of access and character |
| Best hotel product | Cotai | Resort scale and amenities |
| Food-led trip | Taipa or historic peninsula | Strongest everyday dining |
| Family comfort | Taipa | More livable rhythm |
| Quiet | Coloane | Least intense district |
| Fast border logistics | NAPE / Outer Harbour | Practical positioning |

## The Wrong Reasons to Choose

- Choosing Cotai because it is "central"
- Choosing old-town streets if the user really wants pool-and-spa comfort
- Choosing Coloane on a rushed overnight visit

## Default Recommendation

If the user is undecided:
- 1 night -> Taipa or Cotai
- 2-3 nights -> Taipa
- heritage purist -> peninsula
- family or slower traveler -> Taipa with a Coloane half-day
