# Platform Details

## Digital Freelance Platforms

### Upwork
- **Best for:** Development, design, writing, marketing, virtual assistance
- **Integration:** GraphQL API with OAuth2
- **Fees:** 10% client fee + freelancer sliding scale (20%→5%)
- **Payment:** Escrow, hourly tracking with screenshots, milestones
- **Strengths:** Large talent pool, dispute resolution, work history visible
- **Limitations:** Quality varies widely; top talent often busy

**Rate benchmarks (2026):**
| Role | Junior | Mid | Senior |
|------|--------|-----|--------|
| Web Developer | $25-40/hr | $50-80/hr | $100-150/hr |
| Mobile Developer | $30-50/hr | $60-100/hr | $120-180/hr |
| UI/UX Designer | $30-50/hr | $60-90/hr | $100-150/hr |
| Copywriter | $20-35/hr | $40-70/hr | $80-120/hr |

### Freelancer.com
- **Best for:** Broad range of digital work; contests
- **Integration:** REST API — explicitly supports "tasking humans from software"
- **Fees:** 10% or $5 minimum
- **Strengths:** Most API-friendly for programmatic hiring
- **Limitations:** Higher noise ratio than Upwork

### Fiverr
- **Best for:** Quick, defined deliverables (logo, video edit, voiceover)
- **Integration:** None (affiliate API only)
- **Model:** Fixed-price "gigs" with tiers
- **Strengths:** Fast turnaround, predictable pricing
- **Limitations:** Limited customization; quality varies

### Toptal
- **Best for:** Senior developers, designers, finance experts
- **Integration:** None — human-mediated matching
- **Fees:** Premium (expect $100-200/hr+)
- **Process:** 48hr matching by Toptal team
- **Strengths:** Pre-vetted top 3%, dedicated account manager
- **Limitations:** Expensive; minimum engagement often required

## Physical Task Platforms

### TaskRabbit
- **Best for:** Assembly, mounting, moving, handyman, cleaning
- **Coverage:** 50 US states, 8 countries
- **Integration:** Partner API (B2B, requires business relationship)
- **Rates:** $30-80/hr depending on task and market
- **Strengths:** Background-checked Taskers, in-app booking

### RentAHuman.ai
- **Best for:** AI agent → human delegation for physical tasks
- **Integration:** MCP + REST API (agent-native)
- **Payment:** Stablecoins (instant, cross-border)
- **Tasks:** Verification, pickup, attendance, documentation
- **Rates:** $50-175/hr depending on skill
- **Status:** New platform (Feb 2026), rapidly growing

### Other Physical Task Platforms
- **Thumbtack:** Home services, no public API
- **Handy:** Cleaning and handyman, no API (Zapier only)
- **Care.com:** Childcare, eldercare, pet care

## Design Platforms

### 99designs
- **Best for:** Logo, branding, web design via contests
- **Model:** Multiple designers compete; pay winner only
- **Pricing:** $299-$1,299+ depending on package
- **No API:** Fully human-interactive

### Design Pickle / Penji
- **Model:** Subscription unlimited design
- **Best for:** Ongoing design needs (social media, marketing)
- **No API:** Dashboard-based requests

## Content Platforms

### Scripted
- **Best for:** Blog posts, articles, product descriptions
- **Model:** AI + human hybrid content
- **Pool:** 80,000+ writers
- **Integration:** HubSpot, WordPress — no hiring API

## Agent-to-Agent (Emerging)

### AutoGPT Platform
- **What:** Marketplace for pre-built AI agent templates
- **Use:** Deploy specialized agents for subtasks

### Fetch.ai
- **What:** Blockchain-based agent-to-agent transactions
- **Use:** Economic agents, DeFi, trading coordination
- **Requires:** AEA framework, crypto/blockchain knowledge
