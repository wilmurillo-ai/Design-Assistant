# Shanghai Local Setup: Apps, Money, Daily Services (2026)

Most daily life in Shanghai runs through mobile apps.
Set up payments, transport, and communication in your first week.

## Core apps to install

| App | Main use | Notes |
|---|---|---|
| WeChat | Messaging + payments + mini apps | Essential for daily life |
| Alipay | Payments + transit + utilities | Strong service integration |
| DiDi | Ride-hailing | Keep frequent addresses saved |
| Amap or Baidu Maps | Navigation | Better local routing than global map apps |
| Meituan or Ele.me | Food delivery | Useful for late hours |

## Banking setup basics

| Step | Practical detail |
|---|---|
| Open account | Passport, phone number, visa/residence docs usually needed |
| Enable app banking | Required for most transfers and bill pay |
| Link to WeChat/Alipay | Speeds up everyday transactions |
| Set transfer limits | Important for rent and larger payments |

## Typical monthly local service costs (CNY)

| Service | Usual range |
|---|---:|
| Mobile plan | 60-220 |
| Home internet | 100-250 |
| Cloud VPN/proxy tools | Varies widely |
| Food delivery membership | 10-35 |

## SIM and connectivity

1. Buy SIM at official carrier store with passport.
2. Confirm ID registration is completed before leaving.
3. Test data, SMS, and payment OTP reception immediately.
4. Keep one backup eSIM/data option for travel days.

## Daily operations checklist

- Keep one offline payment fallback for battery failures.
- Save landlord, HR, and building management on WeChat.
- Use Chinese address format in notes for drivers and couriers.
- Confirm package lockers and delivery instructions in compound.
- Back up key IDs and contracts to secure cloud storage.

## 2026 practical note

Platform features for foreign cards and identity checks continue to evolve.
Expect occasional re-verification prompts in payment apps.
