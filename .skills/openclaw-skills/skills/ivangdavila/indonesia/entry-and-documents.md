# Entry and Documents — Indonesia

Use this before the user spends money. Rules can change, so anchor the plan in official immigration sources and tell the user exactly what to re-check before booking.

## Default Planning Assumptions

| Item | Safe working assumption |
|------|-------------------------|
| Passport validity | At least 6 months beyond arrival |
| Blank pages | At least one clean page |
| Onward proof | Often expected for short-stay visitors |
| Visa path | Depends on passport: visa-free, VOA, or pre-arranged visa |
| Bali levy | Separate from visa fee for Bali arrivals |

## Common Short-Stay Paths

| Situation | Typical pathway |
|-----------|-----------------|
| Tourist trip for eligible passport | Visa on Arrival or visa-free route if still applicable |
| 30-60 day leisure trip | VOA plus extension, if eligible |
| Work, journalism, research, long stay | Pre-arranged visa only |

Do not guess which passports qualify. Tell the user to check the current official list before purchase.

## What to Verify Before Booking

1. Passport nationality and exact visa route
2. Length of intended stay
3. Arrival airport and whether e-visa or VOA is smoother there
4. Onward or return ticket readiness
5. Bali-only arrival versus wider Indonesia route

## Document Baseline

- Passport with required validity
- Onward or return booking
- Accommodation address for first nights
- Flight details and emergency contact
- Payment method that works on arrival

## Indonesia-Specific Traps

- Assuming Bali rules equal all Indonesia rules
- Assuming a forum screenshot is current immigration law
- Planning same-day extension or airport problem-solving
- Mixing tourist activity with work-like activity
- Forgetting separate Bali levy or customs steps

## Practical Output Standard

When answering entry questions, give:
- likely path
- what remains uncertain
- what must be checked officially before payment
- what proof to carry digitally and on paper
