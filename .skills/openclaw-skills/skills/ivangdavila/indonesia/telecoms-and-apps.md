# Telecoms and Apps — Indonesia

Data and app readiness change trip quality fast. The user should land with maps, money, and transport tools already covered.

## Core Stack

| Need | Best tool type |
|------|----------------|
| Navigation | Google Maps plus offline maps |
| Ride apps | Main local ride-hailing apps |
| Messaging | WhatsApp |
| Payments backup | Bank app and card controls |
| Translation | Offline language support |

## Connectivity Rules

- eSIM is simplest for supported phones.
- Physical SIM still works well and can be cheaper.
- Set up data before leaving the arrival airport if possible.
- Remote islands can mean weaker signal and slower fallback.

## Practical App Advice

- Keep hotel addresses screenshotted.
- Download airline, ferry, and ride apps before flying.
- Use offline maps for volcano or island day trips.
- Carry charger and power bank on transfer days.

## Traps

- Landing with no data and no hotel screenshot
- Assuming ride-app coverage on every island
- Using only cloud-based documents on boat or mountain days
