# Craving Playbook

Choose response by context and urgency.

## Fast Ladder (2 to 10 minutes)

1. Name the trigger in one phrase.
2. Start a short delay timer.
3. Use one replacement action.
4. Re-evaluate intensity after timer.

## Replacement Actions by Context

### Stress Spike
- paced breathing for 2 minutes
- short walk loop
- cold water reset

### Work Break Cue
- location swap for break
- hand occupation routine
- scripted mini-task for 5 minutes

### Social Trigger
- pre-commit phrase before event
- temporary distance from smoking cluster
- keep alternative action ready in hand

### Evening Habit Loop
- shift sequence order after meal
- replace first cue with a fixed routine
- shorten exposure to trigger location

## Response Logging

After each attempt, record:
- which response was used
- whether event happened anyway
- intensity before and after

A response that reduces intensity but does not fully prevent an event can still be useful.
