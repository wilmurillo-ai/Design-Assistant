# Selling Operations - Mercado Libre

Use this file when user sells on Mercado Libre and needs practical execution guidance.

## Listing Quality Cycle

Run this cycle before pushing volume:
- check title and attribute intent match
- check image clarity and trust signals
- check pricing versus relevant alternatives
- check return and claim friction points

Weak listing fundamentals should be fixed before ad expansion.

## Pricing Discipline

Set and respect:
- minimum contribution margin
- competitive response rules
- promotion stop conditions

Never scale tactics that hide negative unit economics.

## Operational Rhythm

Daily:
- monitor top listings by revenue impact
- check order incidents and unresolved claim risk

Weekly:
- evaluate listing tests and keep or rollback decisions
- rebalance spend and effort by return on effort

## Post-Sale Reliability

- prioritize response quality and speed in issue handling
- document recurring failure patterns
- convert incidents into prevention rules
