# Entry and Documents — Finland

## Fast Decision Tree

1. Confirm whether the traveler is visa-free for short Schengen stays or needs a visitor visa.
2. Check passport validity and remaining Schengen-day allowance before booking non-refundables.
3. If the user plans onward Schengen travel, count Finland inside the same 90-in-180 logic.
4. Verify current EES and ETIAS rollout status close to departure instead of relying on old articles or memory.

## Visitor Pathways

| Path | Typical use | Core limit |
|------|-------------|------------|
| Visa-free Schengen short stay | Tourism for eligible passports | Usually up to 90 days in 180 |
| Short-stay visa | Tourism for passports that need a visa | Usually up to 90 days in 180 |

## Practical Notes

- Finland follows Schengen rules, so the user must frame Finland together with the rest of their Schengen trip, not as a separate allowance.
- EES and ETIAS details are evolving and should be checked again before departure using current EU and Finland sources.
- If the route depends on crossing the eastern border, verify current restrictions first because overland assumptions can be wrong.
- Users combining Finland with the Baltics or Sweden should still keep an eye on entry sequence, ferry bookings, and passport access during travel.

## Before Locking Non-Refundables

- Correct visa-free or visa pathway confirmed
- Passport validity checked for the whole trip
- Schengen day count understood for the full Europe route
- EES and ETIAS status rechecked near departure
- Border assumptions verified for any nonstandard overland plan
