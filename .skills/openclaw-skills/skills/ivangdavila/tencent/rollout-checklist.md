# Tencent Rollout Checklist

Use this file before you present a final implementation recommendation or launch plan.

## Pre-Delivery Checklist

- Name the exact Tencent surface in scope
- State the target region and audience
- Confirm whether the answer is planning only or execution-ready
- List the required accounts, tenants, or merchant entities
- Separate hard blockers from nice-to-have improvements

## If Tencent Cloud Is Involved

- Region chosen
- Networking boundary named
- Operational owner named
- Monitoring and rollback path defined
- Cost and quota caveats stated

## If WeChat, WeCom, Or Payments Are Involved

- Product family named exactly
- Merchant or admin ownership named
- Distribution or launch assumptions stated
- User-data or settlement caveats stated
- Manual approval steps called out

## Final Answer Format

End with:
- Recommendation
- Why this path wins
- What not to do
- Required confirmations
- Next action the human owner must take

## Escalate Instead Of Guessing

Escalate when:
- the product family is still ambiguous
- region or language changes the answer materially
- a compliance claim is based on weak evidence
- account ownership is unclear
