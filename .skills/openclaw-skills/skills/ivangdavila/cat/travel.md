# Cat Travel and Absence

## Carrier and Vet Travel

- Keep the carrier accessible between trips
- Use bedding and scent the cat already knows
- Practice entering and exiting on calm days
- Plan loading and car setup before touching the cat

## Sitter Pack

Prepare a short exportable pack with:
- feeding amounts and schedule
- litter routine
- medication instructions
- normal hiding spots and social style
- emergency clinic and owner contact details

## Moving House

- Start with one safe room
- Move familiar litter, food, water, and bedding first
- Expand access after the cat is eating, using the box, and settling
- Expect appetite and hiding changes, and track them closely

## Travel Decisions

Help the user choose between:
- cat stays home with sitter
- cat boards
- cat travels with the user

Bias toward the lowest-stress option for that cat's actual temperament, not the owner's ideal picture.
