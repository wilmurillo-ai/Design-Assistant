# Underwriting

Underwrite to survive bad luck, not to impress with a high return.

## Build the model from cash reality

Start with:
- purchase price
- closing costs
- rehab and make-ready
- financing terms
- in-place rent
- market rent after verified assumptions
- non-rent income only if it already exists or is highly credible

## Core formulas

```text
effective_gross_income = gross_scheduled_rent - vacancy - credit_loss + other_income
noi = effective_gross_income - operating_expenses
cash_flow_before_tax = noi - annual_debt_service
cap_rate = noi / purchase_price
cash_on_cash = cash_flow_before_tax / total_cash_in
dscr = noi / annual_debt_service
break_even_occupancy = (operating_expenses + annual_debt_service) / gross_scheduled_rent
```

## Expense buckets to force in

Do not "forget" these:
- property taxes and reassessment risk
- insurance at post-close reality, not stale seller policy
- repairs and maintenance
- turnover and leasing
- capital expenditure reserve
- property management, even if self-managing today
- utilities, admin, lawn, snow, pest, trash, or common-area costs

## Scenario set

Always show three cases:
- base case -> realistic current view
- downside case -> weaker rents, higher vacancy, higher opex, slower lease-up
- ugly case -> delayed refinance, bigger rehab, rate shock, or exit discount

If the ugly case breaks the investor completely, say so plainly.

## Quality checks

- Underwrite on total cost, not just purchase price.
- Keep one model for acquisition and another note for upside ideas.
- Separate recurring opex from one-time capex.
- If a return depends on future heroics, move that upside into a note, not the base case.

## Minimum output

A good underwriting summary includes:
- total cash needed
- NOI, cap rate, DSCR, cash flow, and cash-on-cash
- top 3 assumptions doing the most work
- what has to go right
- what would change the answer to "no"
