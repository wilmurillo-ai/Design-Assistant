# Reporting Playbook - Google Reviews

Use one of these templates based on cadence and audience.

## Company Snapshot (on-demand)

1. User question and target company scope
2. Current rating and review-volume signal by source
3. Top positive and negative themes with evidence snippets
4. Practical conclusion tied to the decision the user asked for

## Heartbeat Digest (short)

1. Coverage status by source
2. New reviews and rating changes
3. Negative-theme alerts (if any)
4. Immediate action line

## Daily Action Report

- What changed in the last 24h
- Which brands need action first
- Theme breakdown with evidence snippets
- Recommended response plan for each high-priority theme

## Weekly Trend Report

- Review volume trend by brand
- Rating trend and sentiment mix
- Top recurring themes and movement vs last week
- Actions completed vs pending

## Decision Rules

- If severity is high: include owner + deadline in the first screen.
- If data confidence is low: declare uncertainty before recommendations.
- If no material changes: keep report short and archive for continuity.
