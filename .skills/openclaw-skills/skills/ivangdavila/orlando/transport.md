# Orlando Transport Guide

## Core Reality

Orlando is car-first. Public transit exists, but it only works well for narrow use cases.

## Main Transport Tools

| Mode | Best For | Reality |
|------|----------|---------|
| Rental car or personal car | Most visitors and nearly all residents | Strongest default |
| Rideshare | Short adult trips, airport runs, nights out | Surge pricing hurts families |
| LYNX bus | Budget-sensitive trips and select local routes | Slow for tourists |
| LYMMO | Downtown circulation | Free and useful inside the core only |
| SunRail | Narrow commuter use north-south | Helpful for select residents, weak for park trips |
| Resort transport | Disney and some hotel ecosystems | Great only when the resort family matches the plan |

## Current Useful Numbers

- LYNX local fare: about USD 2 one way
- SunRail fare: starts around USD 2 for one zone
- MCO to Disney-area rideshare: often expensive during peaks
- Brightline reaches Orlando airport station, but it is not a substitute for local metro transit

## Best Advice by User Type

| User Type | Best Move |
|-----------|-----------|
| Disney-first family | On-site transport or rental car |
| Universal weekend | Nearby hotel plus walking and rideshare |
| Conference traveler | Venue hotel plus occasional rideshare |
| Resident commuter | Car unless work and home align with SunRail |

## Transport Mistakes

- Using a hotel shuttle as if it were a full transport plan
- Choosing lodging based on airport distance but not park or meeting distance
- Forgetting tolls and parking when comparing hotel zones
