# Orlando Local Services

## First Useful Services

| Need | Typical Move |
|------|--------------|
| Power and water | OUC in much of the city, other utilities in outer zones |
| Internet | Spectrum, AT&T Fiber, and area-specific options |
| Cell service | Major U.S. carriers generally work well, but building quality varies |
| Tolls | Set up SunPass or rental toll coverage |
| Airport runs | MCO is easy to use but traffic timing matters |

## Errand Logic

- Big-box convenience dominates many daily routines.
- Winter Park and the urban core have better local-shop texture.
- Southwest and tourism-heavy areas often feel easier for chains than for community errands.

## High-Leverage Local Moves

1. Test the grocery, pharmacy, school, and airport run before finalizing a lease.
2. Ask about internet speed if remote work matters.
3. Verify whether the address sits in city or county utility patterns.
