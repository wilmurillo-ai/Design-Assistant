# Practice Modes - Tai Chi

Use one mode at a time.
Switch modes when the user goal changes, not just because the session changed.

## `start`

Use when the user is new, returning after a long break, or unsure what tai chi should feel like.

Focus:
- stance comfort
- slow weight shifting
- breathing without force
- leaving the session with confidence, not fatigue

Good output:
- one 5 to 10 minute session
- one plain-language cue
- one reassurance about what "good enough" looks like

## `session`

Use for normal guided home practice.

Focus:
- a short warm-in
- two to four movement phrases or drills
- one closing reflection on steadiness, effort, or breath

Good output:
- clear sequence
- short cue list
- finish with one note worth logging

## `repair`

Use when the user says a movement feels wrong, unstable, stiff, or confusing.

Focus:
- diagnose one breakdown pattern
- reduce complexity
- give one easier drill and one re-test cue

Good output:
- pattern identified
- single correction cue
- simplified drill

## `build`

Use when consistency already exists and the user wants progression.

Focus:
- weekly cadence
- form stability before duration expansion
- gradual increase in complexity, not intensity

Good output:
- plan for 1 to 2 weeks
- measurable success condition
- fallback session for low-energy days

## `recover`

Use when energy is low, soreness is elevated, or the user wants the gentlest viable practice.

Focus:
- standing or seated adaptations
- range reduction
- shorter sessions with slower transitions

Good output:
- conservative session
- clear stop conditions
- recommendation to pause progression until symptoms settle

## Mode Switching Rules

- `start` -> `session`: after 2 to 4 comfortable sessions with no major confusion.
- `session` -> `repair`: when the same error or discomfort repeats.
- `session` -> `build`: when adherence is stable and main form cue holds.
- Any mode -> `recover`: if symptoms, fatigue, or fear rise during practice.
