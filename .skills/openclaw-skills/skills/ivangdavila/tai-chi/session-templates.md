# Session Templates - Tai Chi

Build the session around one purpose.
Do not add extra drills unless they improve the main purpose.

## 5-Minute Reset

Use when adherence is at risk or the user feels too busy to practice.

1. Arrival breathing: 30 to 45 seconds
2. Standing alignment check: 45 seconds
3. Weight shift drill: 2 minutes
4. One short movement phrase: 1 minute
5. Closing note: 30 seconds

Target outcome:
- maintain continuity
- leave with one clear cue for next time

## 10-Minute Beginner Session

1. Posture setup: feet, knees soft, head tall
2. Breath and shoulder release
3. Side-to-side weight shift
4. Step and settle drill
5. Repeat one simple phrase slowly
6. Quick review: what felt steady, what felt rushed

Use when:
- starting from zero
- rebuilding confidence after time away

## 15-Minute Form Repair Session

1. Name the target issue
2. Practice reduced-range version
3. Add one cue from `form-checks.md`
4. Repeat 3 to 5 slow rounds
5. Compare before and after

Use when:
- knees collapse inward
- shoulders lift and stiffen
- weight shift feels abrupt
- tempo outruns balance

## 20-Minute Standard Home Session

1. Warm-in: 2 to 3 minutes
2. Foundational drill: 4 minutes
3. Main phrase or section: 8 to 10 minutes
4. Cool-down standing breaths: 2 minutes
5. Log one cue, one win, one next step

Use when:
- user already practices at least weekly
- no current safety concerns

## Low-Energy or Pain-Aware Session

Adjust any session with these rules:
- reduce range before reducing control
- shorten stance before removing movement entirely
- use support from wall or chair if needed
- stop if symptoms escalate instead of settling

Good finish:
- user feels calmer or steadier than at the start
- no pressure to "push through"
