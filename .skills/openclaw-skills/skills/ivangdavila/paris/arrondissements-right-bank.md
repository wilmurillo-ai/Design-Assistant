# Right Bank Arrondissements (8-11) — Paris

## 8th Arrondissement (Champs-Élysées)

| Aspect | Details |
|--------|---------|
| **Vibe** | Business, luxury, grand |
| **Landmarks** | Arc de Triomphe, Champs-Élysées, Palais de l'Élysée |
| **Rent (1BR)** | €2,500-4,000 |
| **Safety** | ★★★★★ |
| **Best For** | Corporate executives, luxury seekers |
| **Metro** | 1, 9, 13 |

**Pros:** Grand architecture, luxury shopping
**Cons:** Corporate feel, touristy Champs-Élysées, expensive
