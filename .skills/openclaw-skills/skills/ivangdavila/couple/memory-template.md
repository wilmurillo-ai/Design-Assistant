# Memory Setup — Couple

## Initial Setup

Create directory on first use:
```bash
mkdir -p ~/couple/archive
```

## memory.md Template

Copy to `~/couple/memory.md`:

```markdown
# Couple Memory

## Us
- Together since: [DATE]
- Stage: [new/established/long-term/distance]
- Timezone(s): [if relevant]

## Partner Preferences
<!-- Only what they've shared directly -->
- Loves:
- Dislikes:
- Gift style: [experiences/physical/words]

## My Preferences
- Date style:
- Budget comfort:

## Current
- Next milestone:
- Planning:

---
*Last: YYYY-MM-DD*
```
