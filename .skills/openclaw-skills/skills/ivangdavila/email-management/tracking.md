# Follow-Up Tracking - Email Management

Use this file when messages create commitments that must be monitored.

## What Counts as a Commitment

Track items that include:
- promised deliverable
- promised response date
- request requiring decision
- dependency needed to unblock work

## Tracking Record Fields

| Field | Example |
|-------|---------|
| Thread | Budget revision Q2 |
| Owner | Me |
| Counterparty | Finance lead |
| Due date | 2026-03-02 |
| Status | pending / waiting / done |
| Next action | Send revised numbers |

## Follow-Up Cadence

- 24h before due date: reminder
- due date with no completion: escalation draft
- 48h after unanswered outbound: polite nudge draft

## Closure Criteria

Close a tracking item only when:
- requested action is completed, or
- counterparty confirms no further action required.

Archive closed items weekly to keep active list focused.
