# Feedback Loop - Email Management

Use this loop to improve output quality over time.

## Review Dimensions

Score each output from 1 to 5 on:
- Priority accuracy
- Draft usefulness
- Tone match
- Follow-up reliability
- Brevity and clarity

## Fast Retrospective

After meaningful sessions, capture:
- What worked well
- What caused rework
- Which template needed edits
- Which urgency rule failed

## Improvement Actions

Turn feedback into concrete updates:
- update routing rule
- refine template wording
- adjust reminder cadence
- update VIP and escalation list

Keep changes small and test in the next inbox cycle.
