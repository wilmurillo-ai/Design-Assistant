# Response Templates - Email Management

Use these blocks to draft faster while preserving intent.

## Clarification Request

Subject: Clarifying [topic]

Hi [Name],

To move this forward, could you confirm [specific question] by [date]?
Once confirmed, I will [next action].

Thanks,
[Sender]

## Status Update

Subject: Update on [topic]

Hi [Name],

Quick update:
- Current status: [status]
- Completed: [done]
- Next step: [next action]
- ETA: [date]

Please let me know if priorities changed.

## Polite Follow-Up

Subject: Follow-up on [topic]

Hi [Name],

Following up on the message below regarding [topic].
If helpful, I can provide [option A] or [option B].
Would [date] still work for next step?

## Decline with Alternative

Subject: Re: [topic]

Hi [Name],

I cannot commit to [request] by [date].
I can deliver [alternative] by [new date].
If this works, I will proceed and send confirmation.
