# Engagement Models

Choose one model first, then adapt.

## Model Selection Table

| Model | Best for | Duration | Key Output |
|-------|----------|----------|------------|
| Diagnostic Sprint | Unclear root cause | 1-2 weeks | Root-cause memo + options |
| Strategy Design | Directional decisions | 2-6 weeks | Strategic recommendation pack |
| Execution PMO | Delivery at risk | 4-12+ weeks | Milestone plan + governance cadence |
| Turnaround Support | Urgent performance decline | 2-8 weeks | Stabilization plan + KPI control tower |

## Diagnostic Sprint

Use when leaders disagree on what problem actually exists.

Deliverables:
- Current-state snapshot
- Problem tree with confidence levels
- Option set with tradeoffs
- Decision recommendation for next phase

## Strategy Design

Use when the team needs a directional choice across multiple paths.

Deliverables:
- Strategic alternatives
- Scenario analysis
- Resource and capability implications
- Decision memo with recommended path

## Execution PMO

Use when strategy exists but execution reliability is low.

Deliverables:
- Program map by workstream
- Milestone and dependency register
- Weekly risk log and unblock actions
- Executive update format

## Turnaround Support

Use when performance is deteriorating and immediate stabilization is required.

Deliverables:
- 30-day stabilization actions
- Cash and operations risk watchlist
- Governance reset (owners and meeting cadence)
- 60-90 day recovery plan
