## 3) Executive Update Blueprint

Keep updates short and actionable:

- Objective status: on track, at risk, off track
- Top 3 wins since last update
- Top 3 risks and required decisions
- Next 14-day priorities
- Requests from leadership

## Formatting Rules

- Use plain language and explicit tradeoffs
- Separate facts, assumptions, and recommendations
- Always include owner and date for each key action
- Never present analysis without a next step
