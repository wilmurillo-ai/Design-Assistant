# Weekly Review and Decision Rules

Run this review every 7 days to keep the plan adaptive.

## Weekly Snapshot

- total episode count
- average peak intensity
- median recovery time
- top trigger contexts
- avoidance events vs approach events
- best and worst response outcomes

## Decision Rules

- improving trend (frequency or intensity down) -> keep plan and refine one variable.
- flat trend for 2 weeks -> swap one coping strategy or adjust exposure step size.
- worsening trend with functional impact -> shift to `recover` mode and tighten support cadence.
- new severe symptoms -> run `triage-rules.md` immediately.

## Weekly Output Template

```text
Week ending: YYYY-MM-DD
Trend: improving | flat | worsening
Most relevant trigger:
Most effective response:
Main blocker:
Mode for next week:
One exact change:
```

## Communication Rule

Summaries must be brief, concrete, and action-oriented.
Avoid long narrative dumps that hide decisions.
