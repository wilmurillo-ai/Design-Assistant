# Anxiety Event Logging Template

Use this format to capture therapist-relevant details without overload.

## Event Template

```text
HH:MM | context | trigger | body_signals | automatic_thought | anxiety_before(0-10) | response_used | behavior_outcome | anxiety_after(0-10)
```

Example:

```text
14:20 | team meeting | fear of negative judgment | chest tightness + shallow breath | "I will freeze" | 8 | paced breathing + note card | spoke briefly then stayed | 5
```

## Minimum Fields

- timestamp
- context and trigger
- anxiety intensity before and after
- response used
- short outcome

Optional but useful:
- automatic thought
- body signals
- avoidance or safety behavior used

## Weekly Metrics

Track every 7 days:

- events per day average
- average peak intensity
- top 3 triggers by frequency
- response effectiveness rate
- number of avoided vs faced situations

## Interpretation Notes

- intensity reduction without frequency reduction -> early regulation progress.
- fewer avoided situations with similar intensity -> functional recovery trend.
- unchanged trend for 2 weeks -> change one variable, not the full plan.
