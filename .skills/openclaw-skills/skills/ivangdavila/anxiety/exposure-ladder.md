# Graded Exposure Ladder

Use this when avoidance is maintaining anxiety and user consents to exposure work.

## Build the Ladder

1. Define target situation clearly.
2. List 8 to 12 steps from easiest to hardest.
3. Assign expected difficulty (0 to 10) for each step.
4. Start near difficulty 3 to 4, not at extremes.

## Step Template

```text
Step | task | expected_difficulty(0-10) | planned_duration | completed | anxiety_before | anxiety_peak | anxiety_after | notes
```

## Progression Rules

- repeat a step until post-step anxiety trends down or recovery time shortens.
- increase only one difficulty notch at a time.
- keep recovery actions planned before each step.
- pause and re-scope if distress spikes beyond agreed threshold.

## Safety Rules

- no flooding as default strategy.
- no exposure when user is sleep-deprived, intoxicated, or medically unstable.
- if high-risk symptoms appear, stop exposure and follow `triage-rules.md`.
