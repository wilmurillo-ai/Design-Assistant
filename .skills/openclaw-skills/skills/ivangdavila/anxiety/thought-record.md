# Thought Record Workflow (CBT-Style)

Use this only when the user wants cognitive restructuring.
Do not force thought work during acute panic peaks.

## Seven-Step Record

1. Situation
- What happened, where, and with whom.

2. Automatic Thought
- First thought or prediction that appeared.

3. Emotion and Intensity
- Emotion label plus 0 to 10 intensity score.

4. Evidence For
- Concrete facts that support the thought.

5. Evidence Against
- Concrete facts that do not fit the thought.

6. Balanced Alternative Thought
- More accurate statement, not blind positivity.

7. Re-rate Emotion
- Same emotion, new intensity score.

## Quality Rules

- Keep thoughts short and specific.
- Separate facts from interpretations.
- Write alternatives in plain language the user actually believes.

## Common Errors

- Jumping to "positive thinking" -> low credibility.
- Writing abstract alternatives -> no emotional shift.
- Skipping evidence against -> confirmation bias remains unchanged.
