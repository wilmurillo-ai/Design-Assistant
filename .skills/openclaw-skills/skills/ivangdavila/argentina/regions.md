# Regions — Argentina

## Region Selection Grid

| Region | Best for | Watch-outs |
|--------|----------|------------|
| Buenos Aires | First-time city base, food, culture, easy start/end | Big-city theft patterns, protest-related traffic, overlong stays on short trips |
| Mendoza and wine country | Wine, Andes scenery, slower premium rhythm | Vendimia demand, driver logistics, Zonda risk, mountain-day fatigue |
| Iguazu and Misiones | Waterfalls, family wow-factor, short high-impact add-on | Heat, humidity, border friction, timing-sensitive park flow |
| Patagonia Lakes | Scenic drives, soft adventure, ski or summer lakes | Crowds in peak windows, wildfire/smoke risk, car temptation |
| South Patagonia | Glaciers, big landscapes, hiking | Wind, high flight cost, reservation pressure, weather volatility |
| Ushuaia and Tierra del Fuego | End-of-the-world atmosphere, Beagle, light hiking | Fragile weather, limited buffers, premium pricing |
| Salta and Jujuy | Culture, desert color, altitude landscapes | Altitude, long drives, slower acclimatization needs |
| Peninsula Valdes coast | Wildlife with lower hiking intensity | Seasonal wildlife windows, fewer flight combinations |
| Atlantic coast | Summer beach culture, domestic holiday feel | Strong seasonality, peak crowd spikes, less value outside the right months |

## Planning Rule of Thumb

- 7 days: 1 core region, or Buenos Aires plus one nearby contrast
- 10-14 days: 1-2 regions with flights and explicit buffers
- 21 days: 2-3 regions max, with Patagonia counted carefully

## Patagonia Rule

- Patagonia Lakes, South Patagonia, and Ushuaia are separate choices.
- If user says only "Patagonia," force a decision before route design.

## First-Time Argentina Route Suggestions

- City + wine: Buenos Aires + Mendoza
- City + waterfalls: Buenos Aires + Iguazu
- City + scenery: Buenos Aires + one Patagonia cluster
- Wine + mountains + north: Mendoza + Salta/Jujuy only if user has enough days
