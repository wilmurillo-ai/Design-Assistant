# Entry and Documents — Argentina

## Fast Decision Tree

1. Confirm nationality and whether tourist entry is visa-free or visa-required.
2. If visa-free, verify the stay window and whether any border hop changes the plan.
3. If visa-required, route to consular workflow before buying non-refundable flights.
4. If trip includes Brazil or Chile side trips, re-check passport and onward requirements for each country.

## Tourist Entry Basics

- Migraciones defines tourist entry as a stay of up to 3 months, extendable for another similar period.
- Do not promise the extension automatically; tell user to verify conditions and timing with Migraciones.
- Passport should be valid, in good condition, and ready for any onward-travel questions.

## Visa Workflow Notes

| Path | Typical use | Practical note |
|------|-------------|----------------|
| Visa-free tourist entry | Many common tourism nationalities | Still verify current nationality-specific rules before booking |
| Tourist visa | Nationalities without waiver | Start early and use official consular guidance only |

## Family and Minor Notes

- If minors are involved, double-check entry and exit paperwork early, especially for binational families or Argentina residents.
- Mixed-custody, surname mismatch, or one-parent travel cases should be treated as document-sensitive.

## Checklist Before Locking Flights

- Entry path confirmed
- Passport validity checked
- Stay length compatible with plan
- Any Brazil/Chile side-trip requirements verified
- Family/minor paperwork checked if relevant
