# Data Sources by Category

Use these to research prices and validate deals.

## Price History

| Tool | What it does | Best for |
|------|--------------|----------|
| **CamelCamelCamel** | Amazon price history, alerts | Amazon purchases |
| **Keepa** | Multi-retailer price tracking | Electronics |
| **eBay Sold Listings** | Actual sale prices (not asks) | Used items, market value |
| **Google Shopping** | Cross-retailer comparison | General retail |

## Used Goods Market Value

- **eBay Completed Listings** — Filter by "Sold" for real prices
- **Facebook Marketplace** — Local pricing varies, search your area
- **Swappa** — Electronics (phones, laptops)
- **Kelley Blue Book / Edmunds** — Vehicles

## B2B / SaaS

- **G2, Capterra** — User reviews, feature comparison
- **Vendor website pricing** — Often hidden, ask for quote
- **Reddit r/sysadmin, r/devops** — Real user experiences
- **Contracts subreddit** — What others negotiated

## Reviews (Filter for Signal)

| Source | Trust level | Watch for |
|--------|-------------|-----------|
| Reddit | High | Real experiences, search "[product] problems" |
| Wirecutter | High | Tested recommendations |
| Amazon reviews | Medium | Filter fakes: skip 5-star with photos, read 2-3 star |
| YouTube | Medium | Sponsored content bias |

## Scam Detection

- **BBB / Trustpilot** — Seller reputation
- **Whois** — Domain age (new = risky)
- **Reverse image search** — Stolen photos
- **Too-good-to-be-true** — If 30%+ below market, investigate
