# Telecoms — Andorra

The main practical trap is roaming. Andorra is outside the EU roaming area, so people often arrive assuming their usual European plan will work the same way.

## Core Reality

| Topic | Reality |
|-------|---------|
| EU roaming | Often not included |
| Main local network | Andorra Telecom |
| Best fix for short stays | Check plan first, then use eSIM or local option |
| Wi-Fi | Common in hotels, cafes, and central areas |

## Smart Advice

- Tell users to verify roaming before crossing the border.
- For a weekend, an eSIM or preplanned data option is usually the cleanest move.
- If they only need light connectivity, hotel and cafe Wi-Fi plus offline maps may be enough.

## Best Matching

| User | Best move |
|------|-----------|
| Heavy maps and photo user | Arrange data before arrival |
| Casual weekend traveler | Verify plan and keep offline backup |
| Hiker | Save maps offline even if data is active |

## Common Mistakes

- Turning data on automatically on arrival
- Relying on signal quality alone for mountain route planning
- Assuming a French or Spanish plan behaves exactly the same inside Andorra
