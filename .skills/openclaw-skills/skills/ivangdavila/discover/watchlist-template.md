# Watchlist Template - Discover

Create `~/discover/watchlist.md` with this structure:

```markdown
# Discover Watchlist

## Active Topics
- Topic: Example topic
  Why it matters: what decision, move, or curiosity this could change
  Good angles: operator, contrarian, geographic, regulatory
  Novelty bar: what would count as genuinely new
  Heartbeat: proposed | active | paused
  Status: active

## Parked Topics
- Topic: something interesting but not active
  Why it matters: short reason
  Status: paused

## Retired Topics
- Topic: no longer relevant
  Why it matters: short reason
  Status: done
```

## Notes

- Keep topics concrete.
- Prefer one topic per bullet, not bundles.
- Move things out of `Active Topics` when the decision is over.
