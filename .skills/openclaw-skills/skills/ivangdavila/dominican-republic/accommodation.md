# Accommodation

## Base Strategy by Trip Style

| Goal | Better base |
|------|-------------|
| Easiest all-inclusive beach trip | Bavaro, Arena Gorda, or Cap Cana |
| Quieter Caribbean-water resort stay | Bayahibe |
| Walkable history, food, and nightlife | Colonial Zone or central Santo Domingo districts |
| Boutique beach town with dining outside hotel walls | Las Terrenas |
| Surf, kitesurf, and beach-town energy | Cabarete |
| Mountain and adventure contrast | Jarabacoa |

## East-Coast Resort Logic

- Use all-inclusive or large resorts when the user wants low-friction airport transfer, kids' infrastructure, and weather-proof amenities.
- Prefer Cap Cana for polished, contained luxury and cleaner zoning.
- Prefer Bavaro for widest inventory, more day-tour availability, and mid-range range.

## Independent-Stay Logic

- Use Santo Domingo for 1-3 nights of city time, not for a beach substitute.
- Use Las Terrenas when the user values going out for dinner, boutique hotels, and mixed beach plus town rhythm.
- Use Cabarete when the user wants activity and social energy more than protected calm water.

## Villa Caution

- A cheap villa can become expensive if it adds long grocery runs, poor beach access, and nightly transport dependence.
- Only recommend remote villas when the user clearly accepts the logistics and either has a driver, a car, or a full-service property.

## Rule

Choose stays by exit strategy as much as by view: how the user gets to beaches, dinner, tours, and the airport matters more than listing the prettiest property.
