# Entry and Documents

## What To Confirm First

- Whether the traveler is visa-free or needs a visa from Dominican foreign affairs.
- Whether arrival is by commercial flight, because the electronic entry-and-exit form applies there.
- Passport-validity rule for the traveler's nationality and trip type.
- Onward or return travel proof if airline staff ask for it.

## Core Rules

- Commercial airline passengers should complete the official free e-ticket before both inbound and outbound travel.
- For many tourist nationalities, the tourist card cost is already bundled into airline tickets on scheduled flights.
- Visa rules are nationality-specific. If there is any doubt, check the Dominican foreign-ministry path before buying non-refundable travel.
- Do not answer passport-validity questions from memory if the traveler's nationality falls outside the common U.S., Canada, EU, U.K., or Latin America tourist pathways.

## Practical Planning Advice

- Save a PDF or screenshot of the e-ticket QR before reaching the airport.
- Keep passport, hotel address, and outbound-flight details easy to show offline.
- If the user has a complex routing pattern, frame the entry step as a travel-day task, not an afterthought.

## Safe Output Pattern

For entry questions, answer in this order:
1. Whether this looks like a standard tourist pathway or a visa-check case
2. E-ticket requirement for air travel
3. Passport and onward-travel reminders
4. Official source to verify before payment
