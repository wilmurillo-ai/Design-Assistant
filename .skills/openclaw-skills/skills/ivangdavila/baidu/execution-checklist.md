# Execution Checklist

Use this file before you present a recommendation or move toward account-level execution.

## Final Checks

1. State the chosen Baidu surface explicitly.
2. State the assumed region and docs language.
3. Separate discovery sources from verification sources.
4. Mark any action that needs explicit user approval or account access.
5. End with open blockers, confidence, and next safe step.

## Escalate Before Execution If

- the task would log into a console
- the task would create or rotate keys
- the task would change cloud resources or billing scope
- the task depends on region-specific availability that is still unverified
- official docs and translations conflict materially
