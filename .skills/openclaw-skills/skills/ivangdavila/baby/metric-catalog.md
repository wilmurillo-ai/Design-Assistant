# Metric Catalog - Baby

Use consistent units and event names so summaries stay readable.

## Feeding

| Metric | Record As | Notes |
|--------|-----------|-------|
| Breastfeed | start time, side, duration | Note both sides if used |
| Bottle | time, amount, unit, milk type | Keep one volume unit |
| Pumping | time, amount, unit | Separate pumped from fed |
| Solids | time, food, amount | Note first exposures clearly |
| Spit-up or vomit | time, amount estimate, trigger | Distinguish small spit-up from repeated vomiting |

## Sleep

| Metric | Record As | Notes |
|--------|-----------|-------|
| Nap | start, end, duration | Use actual end time if known |
| Night sleep block | start, end, wakes | Keep wakes inside the block |
| Settling | start, method, time to sleep | Useful for routine experiments |

## Output

| Metric | Record As | Notes |
|--------|-----------|-------|
| Wet diaper | time | Count when hydration is the question |
| Dirty diaper | time, color, texture notes | Use clear language, not vague labels |
| Mixed diaper | time, notes | Record both wet and dirty if needed |

## Symptoms and Vitals

| Metric | Record As | Notes |
|--------|-----------|-------|
| Temperature | time, value, unit, method | Example: rectal, ear, forehead |
| Rash | time, location, spread, photo mention if user has one | Do not interpret from photo beyond user request scope |
| Cough or congestion | time, severity, sleep/feed impact | Track breathing impact separately |
| Vomiting | time, count, color if unusual | Escalate with red flags |
| Pain cues | time, trigger, behavior | Crying alone is not enough detail |

## Medications and Care Actions

| Metric | Record As | Notes |
|--------|-----------|-------|
| Medication | time, name, dose, unit, reason | Do not guess dose |
| Saline/suction/humidifier | time, action, response | Useful in illness summaries |
| Home check | time, result | Example: hydration check, rash spread |

## Growth and Visits

| Metric | Record As | Notes |
|--------|-----------|-------|
| Weight | date, value, unit | Record source if relevant |
| Length/height | date, value, unit | Keep measurement source clear |
| Head circumference | date, value, unit | Only when user already tracks it |
| Visit | date, provider, outcome | Keep decisions and follow-ups explicit |

## Event Vocabulary

Prefer concrete labels:
- `feed refused`
- `short nap`
- `multiple wakes`
- `fever`
- `vomiting`
- `rash spreading`
- `constipation concern`
- `medication given`
- `pediatrician called`

Avoid vague labels like `bad day` or `not normal`.
