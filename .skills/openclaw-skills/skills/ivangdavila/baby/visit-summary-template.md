# Visit Summary Template - Baby

Use this for pediatric visits, nurse lines, and urgent-but-not-emergency calls.

## One-Minute Verbal Summary

Start with:
- baby age or stage
- main concern
- when it started
- what changed from baseline
- what has already been tried

## Written Summary Structure

```markdown
# Pediatric Summary

- Baby stage:
- Main concern:
- Started:
- Baseline versus now:
- Feed change:
- Sleep change:
- Diaper/output change:
- Temperature or symptom timeline:
- Medications or care actions tried:
- Red or amber triggers observed:
- Top questions:
```

## Good Questions to Prepare

Keep questions concrete:
- Is this timeline concerning enough for same-day evaluation?
- What intake or diaper threshold should trigger urgent follow-up?
- Is the current care plan still appropriate?
- What should be watched overnight?

## Before the Visit or Call

Check that the summary includes:
- exact times for major changes
- temperatures with method if relevant
- medication name, dose, and time
- intake and wet diaper pattern
- anything clearly worse, better, or unchanged

## After the Visit

Update the handoff and memory with:
- instructions received
- medication or follow-up plan
- what changed in the monitoring plan

This prevents partial recall later.
