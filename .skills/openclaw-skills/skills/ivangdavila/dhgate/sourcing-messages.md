# Seller Message Templates - DHgate

Keep seller messages short, specific, and easy to prove later.

## Exact Spec Confirmation

```text
Hi. Before ordering, please confirm the exact version for this listing:
- model:
- size or variant:
- material:
- included accessories:
- plug or voltage if applicable:

Please reply in one message so I can place the correct order.
```

## Sample Order Request

```text
I want to test quality before a larger order.
Can you support a small sample order first?
Please confirm:
- minimum sample quantity
- sample price
- shipping method
- estimated dispatch time
```

## Packaging and Quantity Check

```text
Please confirm how many units are included in this price and how the goods are packed.
I need the exact pack count, carton detail, and whether any accessories are separate.
```

## Lead Time Clarification

```text
Please confirm:
- production or handling time before shipment
- shipping method
- normal delivery estimate to [country]
- whether tracking will update end-to-end
```

## Defect or Wrong Item Message

```text
The received order does not match the listing.
Issue:
- order number:
- item received:
- problem:

I have photos and video ready.
Please propose a refund or resend solution through the order page.
```

## Counterfeit-Risk Probe

```text
Please confirm whether this item is branded, unbranded, or generic.
If it is generic, confirm that no brand logo appears on the product, packaging, or accessories.
```

## Message Discipline

- ask one decision-critical question per message block
- get the seller to restate the exact spec in writing
- do not negotiate on five topics at once
- if replies stay vague after two attempts, downgrade the seller
