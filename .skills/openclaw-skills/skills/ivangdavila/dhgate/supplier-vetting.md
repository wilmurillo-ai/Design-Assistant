# Supplier Vetting - DHgate

Use this scorecard before recommending any DHgate listing for more than a one-off low-risk buy.

## Pass or Fail Scorecard

Score each area `0`, `1`, or `2`.

| Area | 0 | 1 | 2 |
|------|---|---|---|
| Store maturity | brand-new or unclear history | some history but limited proof | multi-year store with visible trading history |
| Listing clarity | vague specs, thin photos, contradictions | mostly clear with some missing details | exact specs, dimensions, variants, and useful photos |
| Review realism | generic praise only | mixed quality, limited evidence | detailed reviews with real-world photos and specifics |
| Price sanity | far below comparable offers | slightly below market without explanation | price fits the range after quantity and shipping are considered |
| Seller replies | evasive, copy-paste, or slow | answers some questions | direct, specific, and consistent answers |
| Sample readiness | pushes large order only | flexible but vague | accepts sample or small trial order with clear conditions |

## Decision Rule

- `10-12` -> acceptable shortlist candidate
- `8-9` -> possible sample order only
- `0-7` -> reject unless the user explicitly wants to gamble

One hard-stop red flag overrides any score.

## Hard-Stop Red Flags

- Brand-name goods at implausible discounts
- Seller asks to move payment off-platform
- Specs change between listing, chat, and invoice
- Shipping promise is aggressive but unsupported
- Identical catalog reused across many stores with no differentiator
- Seller refuses to confirm exact variant, size, material, or plug type

## Listing Checks That Matter Most

### 1. Match the Exact Product
- confirm model, size, color, material, pack quantity, and included accessories
- verify photos match the promised variant, not just a hero image
- for electronics, check voltage, plug standard, and certification claims

### 2. Compare the Offer, Not the Thumbnail
- compare at least three DHgate listings for the same item or same factory photo set
- normalize for quantity, shipping method, and delivery estimate
- note whether the cheapest offer quietly removes accessories or quality features

### 3. Message Test Before Trust
- send one focused pre-purchase message
- ask for the exact spec that would matter in a dispute later
- vague or contradictory replies now become expensive after payment

## Sample Order Plan

Before recommending a larger buy:
- order the smallest sensible batch
- document promised specs in chat
- track real days to ship and real days to arrive
- inspect quality, packaging, defects, and missing units
- only then discuss scale
