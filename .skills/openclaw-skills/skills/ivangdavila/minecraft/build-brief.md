# Build Brief - Minecraft

Use this when the user wants a structure, base, room layout, or themed area.

## Build Brief Template

```text
Goal:
Edition/version:
World mode: survival | creative
Biome or theme:
Footprint target:
Height target:
Material budget:
Functional needs:
Lag limits:
Must-keep terrain or landmarks:
```

## Output Shape

1. Footprint and height
2. Layer-by-layer shell plan
3. Material estimate by block family
4. Functional zones and path width
5. Lighting or spawn-proofing notes
6. What to mock in cheap blocks first

## Default Heuristics

- Start from the footprint, then chunk alignment, then facade.
- For survival builds, propose a cheap proof-of-concept shell before decorative detailing.
- Keep walkways at least two blocks wide unless the style depends on tighter movement.
- Name where vertical circulation lives early: ladder, stairs, elevator, bubble column.
- If the build includes farms or villagers, reserve expansion space from the start.

## Fast Example

For a survival starter base:
- 17x17 interior
- 3-block wall height before roof pitch
- separate storage, smelting, sleep, and crop access
- perimeter lighting before decoration
