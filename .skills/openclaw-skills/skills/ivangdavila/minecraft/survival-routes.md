# Survival Routes - Minecraft

Use this file when the user wants progression help rather than one isolated mechanic.

## Route by Goal

| Goal | Early focus | Midgame unlock | Late step |
|------|-------------|----------------|-----------|
| Fast iron and villagers | food, beds, village safety | iron farm or trading hall | curing, enchant cycle |
| Nether prep | shield, food, arrows, gold | portal safety box and coordinates | blaze rods, fortress route |
| Better tools fast | branch mining or caves | enchant setup and diamond reserve | netherite route |
| Boss progression | armor, bow, beds, food | blaze rods, pearls, brewing | dragon prep and exit plan |
| Long-term base | temp shelter and storage | renewable food and iron | transport, mega storage, specialty farms |

## Planning Heuristics

- Tie every route to the user's current gear and risk tolerance.
- Prefer renewable unlocks over one-time loot when the user wants stability.
- In survival, logistics often matter more than perfect aesthetics.
- If the user plays with friends, split roles instead of assuming solo throughput.

## Minimal Plan Shape

1. Current state
2. Next three unlocks
3. Resource checklist
4. Main risk
5. Recovery plan if the run goes wrong
