# Apps — Taiwan

Taiwan travel works best with a small, practical app stack: maps, rail, weather, and one or two local mobility tools.

## Core Stack

| Need | App or tool | Why |
|------|-------------|-----|
| Navigation | Google Maps or Apple Maps | Best all-round routing and saved places |
| HSR booking | `T Express` | Official THSR mobile tickets and timetable |
| TRA booking | TRA official booking site or app | Useful for east coast and local rail |
| Bike sharing | YouBike official app | Needed if the traveler plans to use YouBike often |
| Messaging | LINE | Standard local contact channel |
| Weather | Weather app with radar | Rain and typhoon awareness matters |

## Useful Extras

| Use case | Tool |
|----------|------|
| Local buses in big cities | Bus-focused app such as Bus+ |
| Taxi without language friction | Uber or LINE Taxi where available |
| Stored-value management | EasyCard or iPASS support if relevant |

## Best Practice

- Save hotels, stations, and key food stops offline before transit days
- Use `T Express` for west-coast speed; use TRA booking tools for east-coast and local planning
- If using YouBike, set it up before the first ride instead of at the dock
- Keep screenshots or offline copies of major bookings in case mobile signal gets messy underground or during travel changes

## Friction to Expect

- Taiwan has excellent transport, but it is not one unified app world
- Local restaurant and market recommendations still spread through blogs, maps, and LINE rather than a single clean platform
- Some smaller shops stay cash-first even if the city feels highly connected

## Common Mistakes

- Depending on one app for every rail layer
- Waiting until station time to install or register
- Using bikes casually without checking local rules or setup requirements
