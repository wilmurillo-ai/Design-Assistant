# Entry and Documents - Greece

## Fast Decision Tree

1. Confirm whether the traveler is using Schengen free-movement rights, visa-free short stay, or a visa pathway.
2. Check passport validity and remaining Schengen days before building the route.
3. Confirm arrival and departure points, especially if the route enters or exits from a different Schengen country.
4. Keep ferry and domestic flight bookings aligned with the exact passport name.

## Practical Rules

| Topic | What to verify |
|-------|----------------|
| Short stay limit | Many non-EU travelers still follow the 90 days in 180 rule for Schengen |
| Passport validity | Check airline and Schengen validity rules before booking non-refundable legs |
| Visa cases | Use MFA or Greek consulate guidance, not blogs or aggregators |
| Border rollout changes | Some non-EU travelers may face extra processing under current EU system changes |

## EES and ETIAS Reality

- EU border systems are changing. Check `sources.md` close to departure.
- During 2026, some travelers may see longer first-entry processing because the Entry/Exit System is still being rolled out.
- ETIAS timing has shifted multiple times, so never assume a blog post is current.

## Domestic and Ferry Document Logic

- Passport or accepted ID should be easy to access for airport and port check-ins.
- International-arrival day plus same-day island ferry is fragile if border or baggage delays matter.
- If route includes rented car on a ferry, ensure booking names and plate details are consistent.

## Before Locking Non-Refundables

- Entry pathway confirmed
- Passport and Schengen-day math checked
- First arrival point and last-night city decided
- Same-day international-to-ferry risk reviewed
- Travel insurance logic matched to the route
