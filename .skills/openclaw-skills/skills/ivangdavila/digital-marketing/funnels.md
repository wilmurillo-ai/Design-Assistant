# Funnels

Use this before launching traffic or scaling creative.

## Funnel Questions

| Stage | Question | Asset | Primary metric |
|-------|----------|-------|----------------|
| Awareness | why should this matter now | post, video, ad, search result | hook rate, CTR, engaged visits |
| Consideration | why trust this offer | landing page, case study, email | conversion rate, reply quality |
| Decision | why act now | demo page, checkout, consultation offer | lead quality, purchase rate |
| Follow-up | what happens if they do not act today | nurture, retargeting, reminder | return visits, assisted conversions |
| Retention | how do they buy again or stay engaged | lifecycle flow, education, win-back | repeat conversion, churn, retention |

## Route-to-Value Rule

Every acquisition push must define:
- source
- destination
- desired action
- follow-up
- recovery path if the first visit fails

If there is no follow-up path, traffic quality is being wasted.

## B2B Default Path

Channel -> message angle -> landing page -> proof asset -> form or demo CTA -> follow-up email -> retargeting -> sales handoff

## Ecommerce Default Path

Channel -> offer angle -> product or landing page -> social proof -> purchase or capture CTA -> browse or cart recovery -> retention flow

## Common Breakpoints

- ad and landing page promise do not match
- form asks too much before value is clear
- follow-up arrives too late or sounds generic
- retargeting repeats awareness copy instead of handling objections
- high click volume masks weak conversion intent
