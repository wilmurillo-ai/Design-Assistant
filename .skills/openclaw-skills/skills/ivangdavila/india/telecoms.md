# Telecoms — India

Connectivity is usually easy once it is working, but activation and OTP dependence can create first-day friction.

## What Usually Works Best

| Need | Best Move |
|------|-----------|
| Arrival-day data | eSIM before landing if supported |
| Longer trip | Physical SIM from a major carrier with passport and photo |
| Navigation backup | Offline maps downloaded before arrival |
| OTP-heavy apps | Keep primary number reachable and roaming-aware |

## Practical Rules

- Set up data before relying on ride apps or train apps
- Screenshot hotel address, driver details, and boarding passes
- Do not assume every payment or ticket app will accept foreign numbers equally well
- Hotspot from one good connection is often enough for a couple or family

## Telecom Traps

- Landing at night with no data plan
- Counting on airport Wi-Fi to solve everything
- Using only one payment-linked number with no backup
- Assuming mobile data is the same as reliable app onboarding
