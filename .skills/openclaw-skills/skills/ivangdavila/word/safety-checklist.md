# Safety Checklist — Word

Use this before any risky Word Desktop action.

## Destructive or high-risk actions

- accept all changes
- reject all changes
- delete comments in bulk
- close without save
- overwrite export targets
- replace text across the whole document

## Required preflight

1. Identify the exact document path or open document name.
2. Confirm whether the action is selection-level, section-level, or document-wide.
3. State whether the action is reversible.
4. State whether a copy or Save As path exists.
5. State whether track changes or document protection is active.

## Safe fallback moves

- create a copy first
- export instead of overwrite
- test on one selection before applying globally
- read back the review state before and after the action
