# Itineraries - Turkey

## 5 Days

- **Istanbul deep dive:** 4 nights in one base, split historic core and Bosphorus neighborhoods.
- **Cappadocia short break:** 3 nights with one balloon window and one flexible weather buffer.

## 7-8 Days

- **First-time classic:** 4 nights Istanbul + 3 nights Cappadocia
- **History and coast:** 3 nights Izmir/Selcuk + 4 nights Bodrum or Antalya
- **Food and old cities:** 3 nights Istanbul + 2 nights Gaziantep + 2 nights Mardin

## 10-12 Days

- **Balanced first trip:** 4 nights Istanbul + 3 nights Cappadocia + 3-4 nights Aegean or Mediterranean coast
- **West coast route:** Izmir/Selcuk + Bodrum/Cesme + Istanbul finish
- **East-focused route:** Istanbul entry + Sanliurfa/Mardin + Cappadocia or Kars

## 14+ Days

- **West Turkey arc:** Istanbul -> Cappadocia -> Aegean west coast -> Istanbul out
- **Coast-heavy route:** Istanbul -> Antalya/Kas/Fethiye corridor -> Aegean finish
- **Deep-history route:** Istanbul -> Cappadocia -> Gaziantep/Sanliurfa/Mardin

## Route Design Rules

- Keep one transfer-light base for every 3-4 active sightseeing days.
- Protect the last night before the international flight in the departure city or a same-airport base.
- If a route depends on one dawn activity, keep one flexible half-day nearby.
- If user wants both beach and archaeology, choose one coast and one inland site cluster, not every famous stop.
