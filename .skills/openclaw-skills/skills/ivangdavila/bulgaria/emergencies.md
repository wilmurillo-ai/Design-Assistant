# Emergencies - Bulgaria

## Emergency Number

**112** is the main emergency number for police, ambulance, and fire.

## Practical Safety Pattern

- Cities are generally manageable with normal urban awareness.
- The bigger risks for visitors are transport friction, bad taxis, mountain weather, and alcohol-heavy resort zones.

## If Something Goes Wrong

### In cities
- Use 112 for urgent help
- Ask hotel or host to help with police or medical logistics
- Use pharmacies for small issues before turning everything into an ER problem

### In mountains
- Weather and terrain matter more than crime
- Start earlier than visitors expect
- If a route is uncertain, keep a simpler backup plan

## Common Issues

| Issue | Best response |
|------|---------------|
| Taxi dispute | Use app-ordered taxis next time, document details |
| Lost phone or wallet | Freeze cards, contact lodging, file report if needed |
| Minor illness | Pharmacy first, clinic if needed |
| Hiking problem | Stop early and do not improvise deeper into the route |

## Safety Warnings Worth Saying Out Loud

- Avoid random taxi approaches at airports or nightlife zones
- Watch alcohol-driven areas in peak resort season
- Respect sea flags and mountain weather advisories
