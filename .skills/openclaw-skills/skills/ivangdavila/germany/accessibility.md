# Accessibility — Germany

## Good Default Assumption

Germany can work well for accessible travel, but old-town surfaces, station complexity, and village-heavy scenic routes still need deliberate checking.

## Best Bets

| Need | Better fit |
|------|------------|
| Big-city mobility | Berlin, Munich, Hamburg |
| Short-transfer trip | One strong city base with day trips |
| Scenic but manageable | Rhine-west with carefully chosen base |
| Rural charm | Possible, but verify surfaces and transport first |

## Planning Rules

- Ask about walking tolerance, stairs, wheelchair needs, shower style, and luggage independence early.
- Rail can be excellent, but platform changes and older stations still matter.
- Medieval cores and market areas can look romantic and behave badly for wheels or reduced mobility.
