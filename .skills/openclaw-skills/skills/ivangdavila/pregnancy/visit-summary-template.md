# Weekly and Visit Summary Template - Pregnancy

Use this format for clinician-facing updates.

## Weekly Summary

```text
WEEKLY PREGNANCY SUMMARY
Week ending:
Gestational week:
Overall status: green | amber | red

1) Key Changes Since Last Week
- Trend 1:
- Trend 2:
- Trend 3:

2) Clinical Signals
- Blood pressure trend:
- Glucose trend (if tracked):
- Weight trend (if tracked):
- Symptom frequency/severity trend:

3) Alert Events
- Red events:
- Amber events:
- Actions taken:
- Outcomes:

4) Adherence and Routine
- Medication adherence:
- Check-in completion:
- Missed data windows:

5) Questions for Next Visit (max 5)
- Q1
- Q2
- Q3
- Q4
- Q5
```

## Appointment Prep Snapshot

Create a short pre-visit packet:
- top 3 concerns
- top 5 questions
- last 7-day trend highlights
- all unresolved alert events
- decisions needed from clinician

## Output Style Rules

- Keep summary concise and factual.
- Prefer trend language over isolated data points.
- Include timestamps for concerning events.
- Use neutral wording without diagnosis claims.
