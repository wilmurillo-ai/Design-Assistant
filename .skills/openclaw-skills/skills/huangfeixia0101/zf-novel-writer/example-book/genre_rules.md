## 3. Character Rules / 角色规则

### Protagonist Rules / 主角规则
- [ ] Consistent personality traits / 性格特征一致
- [ ] Clear motivation and goals / 明确的动机和目标
- [ ] Gradual growth, not instant power-ups / 渐进式成长

### Antagonist Rules / 反派规则
- [ ] Has real motivations, not cartoonishly evil / 有真实动机
- [ ] Competent and threatening / 有能力、有威胁
- [ ] Not NPC-ified / 不被工具人化

---

## 4. Dialogue Rules / 对话规则

- Multi-layered dialogue (pauses, tone, expressions, actions) / 有层次（停顿、语气、表情、动作）
- Avoid flat, exposition-heavy dialogue / 避免直白、说教式对话
- Each character has a distinct voice / 每个角色有独特声音

---

## 5. Sensory Detail / 感官描写

- Minimum 3 senses per major scene / 每个主要场景至少3种感官
- Visual, auditory, tactile, olfactory, gustatory / 视觉、听觉、触觉、嗅觉、味觉

---

## 6. Forbidden Elements / 禁止元素

- No cliché golden lines (e.g., "三十年河东三十年河西") / 避免俗套金句
- No sudden emotional shifts without transition / 不允许无过渡的情感突变
- No deus ex machina / 不允许机械降神
