# Level System Example / 等级规则示例

> This is a template for level-based power systems. Adapt to your novel.
> 这是等级力量体系的模板。根据你的小说调整。

---

## Level Overview / 等级概览

| Level | Name / 名称 | Multiplier / 倍率 | Daily Limit / 每日限额 | Description / 描述 |
|-------|-----------|-------------------|---------------------|-------------------|
| 1 | Bronze / 青铜 | 1-2x | 100,000 | Entry level / 入门级 |
| 2 | Silver / 白银 | 3-5x | 500,000 | Intermediate / 中级 |
| 3 | Gold / 黄金 | 6-10x | 2,000,000 | Advanced / 高级 |
| 4 | Platinum / 铂金 | 11-20x | 10,000,000 | Expert / 专家级 |
| 5 | Diamond / 钻石 | 21-50x | Unlimited | Master / 大师级 |

---

## Level Progression / 等级进阶

- Each level requires specific conditions to advance
- Power increases are gradual, not sudden
- Unlocking new abilities should be paced with story beats
