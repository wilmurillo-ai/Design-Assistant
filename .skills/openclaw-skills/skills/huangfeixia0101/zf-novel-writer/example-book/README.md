# Example Book / 示例书籍项目

This is a minimal template for a new book project. Copy this directory and customize it.

这是一个最小化的书籍项目模板。复制此目录并自定义。

## Quick Setup / 快速设置

```bash
cp -r skills/ZF-novel-writer/example-book books/{YOUR_BOOK_NAME}
```

## Files to Customize / 需要自定义的文件

| File | Purpose |
|------|---------|
| `meta/canon_bible.json` | Replace `{BOOK_NAME}`, `{PROTAGONIST}`, `{HEROINE}` with your characters |
| `genre_rules.md` | Set your genre, style, and rules |
| `WORLD_SETTING.md` | Describe your world |
| `LEVEL_BRONZE.md` | Set up your power/level system (if applicable) |
| `story_outline.xlsx` | Create your chapter-by-chapter outline |

## story_outline.xlsx Format / 大纲格式

Create an Excel file with these columns:

| Column | Description |
|--------|-------------|
| 章节 | Chapter number (1, 2, 3...) |
| 标题 | Chapter title |
| 主要事件 | Core events of the chapter |
| 地点 | Scene location |
| 女主 | Appearing heroines |
| 冲突类型 | Type of conflict |
| 系统提示 | System feedback (if applicable) |
| 系统等级 | System level (if applicable) |
| 爽点 | Expected payoff points |
| 悬念钩子 | End-of-chapter hook |

## Directory Purposes / 目录说明

- `chapters/` — Archived final chapter text files
- `summaries_json/` — Per-chapter summary JSON files
- `temp_chapters/` — Temporary output from Writer Agent (cleaned after archive)
- `plans/` — Chapter plans generated by Orchestrator
- `meta/` — Core metadata (canon_bible.json, etc.)
