# 微信知识库字段设计参考

## 多维表格字段设计

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 姓名/话题 | 文本 | 客户名称或讨论主题 |
| 分类 | 单选 | 客户/供应商/团队/其他 |
| 微信ID | 文本 | 微信原始ID |
| 关键需求 | 多行文本 | 主要需求和痛点 |
| 沟通风格 | 单选 | 专业/随和/谨慎/果断 |
| 常用话术 | 多行文本 | 高频表达方式 |
| 消息数量 | 数字 | 统计消息总数 |
| 最后跟进 | 日期 | 上次联系时间 |
| 跟进状态 | 单选 | 待联系/跟进中/已成交/已结束 |
| 知识卡片 | 富文本 | 完整知识卡片内容 |
| 关联文档 | 附件 | 报价表/合同等 |

## 视图建议

1. **表格视图** - 默认视图，快速浏览
2. **看板视图** - 按跟进状态分类
3. **画廊视图** - 按分类展示
4. **分组视图** - 按沟通风格分组

## 示例数据

```json
{
  "姓名/话题": "张总",
  "分类": "客户",
  "关键需求": "采购50套无人机配件",
  "沟通风格": "专业",
  "常用话术": "性价比、质量认证、交货周期",
  "跟进状态": "跟进中"
}
```
