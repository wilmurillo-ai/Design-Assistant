# Tier1 VC Fundraising Agent

An AI agent skill designed to help startups raise venture capital.

## Features

- Evaluate venture-scale potential
- Generate VC-grade pitch narratives
- Discover and rank venture investors
- Execute structured outreach
- Improve fundraising efficiency

## Use Cases

- AI startups raising seed or Series A
- Developer tool companies
- Frontier technology companies
- Crypto infrastructure startups

## Installation

Install via ClawHub:

clawhub install tier1-vc-fundraising-agent
