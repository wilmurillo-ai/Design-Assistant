# E-commerce Product Copywriter

## name
ecommerce-product-copywriter

## description
Create compelling, SEO-optimized product descriptions for e-commerce platforms like Amazon, Shopify, Etsy, eBay, and more. Generate descriptions that convert browsers into buyers with persuasive copy, keyword integration, and multi-language support.

## instructions
### Overview
This skill helps you write professional product descriptions that sell. Whether you're listing on Amazon, your own Shopify store, Etsy, eBay, or any other marketplace, this skill creates compelling copy that highlights benefits, addresses buyer concerns, and incorporates SEO keywords for better visibility.

### When to Use
- Creating new product listings on any e-commerce platform
- Optimizing existing product pages for better conversions
- Translating product descriptions for international markets
- Writing descriptions for multiple similar products
- Creating Amazon Enhanced Brand Content (A+ Content)
- Writing product descriptions for catalogs and print
- Revamping descriptions that aren't converting

### Input Requirements
Provide as much information as available:

**Required:**
- Product name/title
- Product category
- Key features and specifications

**Recommended:**
- Target audience
- Key benefits (why customers want this)
- Price point/positioning
- Main competitors or alternatives
- Unique selling proposition (USP)

**Optional:**
- Brand guidelines
- Keywords to include/exclude
- Word count preferences
- Platform-specific requirements
- Existing descriptions (for rewriting)

### Supported Platforms
- **Amazon**: Standard descriptions, A+ Content, Amazon Vine
- **Shopify**: Product descriptions, metafields
- **Etsy**: Shop sections, listing descriptions
- **eBay**: Item specifics, description templates
- **WooCommerce**: Product pages, short descriptions
- **BigCommerce**: Product details, Google Shopping
- **General**: Any website or marketing material

### Description Frameworks

#### Framework 1: Feature-Benefit-Value
```
[Hook/Headline] → [Key Features] → [Benefits] → [Social Proof] → [CTA]
```

#### Framework 2: Problem-Solution-Result
```
[Customer Pain Point] → [Product Solution] → [Transformation] → [Guarantee]
```

#### Framework 3: Story-Driven
```
[Relatable Scenario] → [Challenge] → [Discovery] → [Solution] → [Invitation]
```

#### Framework 4: Ultra-Concise (for ads/short formats)
```
[Benefit-Focused Headline] + [3 Key Features] + [Trust Signal]
```

### Output Options
Choose your output format:

1. **Standard Description**: 100-200 words, balanced SEO and readability
2. **Long-Form**: 300-500 words with detailed benefits and social proof
3. **Short-Form**: 50-100 words for ads and featured snippets
4. **Platform-Specific**: Tailored for specific marketplace requirements
5. **Multi-Language**: Generate in multiple target languages

### Best Practices
1. Lead with benefits, not features
2. Use sensory words (hear, feel, see the difference)
3. Include social proof when possible
4. Address common objections preemptively
5. Match tone to your brand and audience
6. Include specific numbers and data
7. End with a clear call-to-action
