# 简历优化助手

## 功能说明
将您的简历打造成HR眼中脱颖而出的求职利器。优化ATS（人才管理系统）友好格式、专业措辞、量化成果，帮您获得更多面试机会。

## 使用方法
1. 提供您的现有简历（或关键信息）
2. 说明目标岗位/行业
3. 突出您的核心成就
4. 提及目标公司或职位
5. 获取专业优化的简历

## 示例

**输入**：
我有一个3年工作经验的运营简历，想应聘产品经理岗位，简历内容如下：
- 负责公司公众号运营，粉丝增长
- 参与产品需求讨论
- 用户反馈收集

**输出**：
优化后的简历片段：
- 主导公司公众号运营策略制定，**粉丝量从2万增长至8万，增长率300%**，单篇爆款阅读量突破10万+
- **深度参与5个核心产品功能的需求调研与迭代**，协同产品团队完成用户需求文档撰写
- 建立用户反馈闭环机制，**累计收集分析用户反馈2000+条**，推动产品优化20余项
