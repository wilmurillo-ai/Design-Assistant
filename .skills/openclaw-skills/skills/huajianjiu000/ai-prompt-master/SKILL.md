# AI Prompt Optimizer

## What It Does
Transform your rough ideas into powerful, precise AI prompts that deliver exactly the results you need. Whether you're using ChatGPT, Claude, Midjourney, Stable Diffusion, or any other AI tool, this skill helps you craft prompts that work.

## How to Use
1. Share your original idea or rough prompt
2. Specify the AI tool you're targeting (ChatGPT, Midjourney, etc.)
3. Describe your desired outcome
4. Get an optimized, professional prompt

## Features
- **Context Enhancement**: Add relevant context to make prompts more effective
- **Role Definition**: Define the AI persona for better responses
- **Output Format**: Specify exactly how you want results formatted
- **Constraint Setting**: Add limitations and requirements
- **Example Integration**: Include examples for few-shot learning

## Example

**Original Idea**: "Write something about marketing"

**Optimized Prompt**:
```
You are an expert digital marketing strategist with 10+ years of experience.

Task: Write a comprehensive product launch email sequence.

Requirements:
- 3 emails: Announcement, Benefits Focus, Urgency/CTA
- Each email should be 150-200 words
- Include subject line for each email
- Tone: Professional but approachable
- End each email with a clear call-to-action

Target Audience: Small business owners, 35-55 years old
Product: Project management software, $29/month
```

## Best Practices
- Be specific about the output format you want
- Include context about your audience
- Define constraints and limitations clearly
- Add examples when possible
