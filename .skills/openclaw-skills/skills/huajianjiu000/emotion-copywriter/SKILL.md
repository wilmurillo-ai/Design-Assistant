# 情绪文案生成器

## 基本信息

- **名称**: emotion-copywriter
- **描述**: 精准触达用户情绪的文案生成工具，通过情绪心理学原理创作引发强烈共鸣的社交媒体文案
- **版本**: 1.0.0
- **适用场景**: 微博、朋友圈、小红书、公众号、INS等社交平台情绪化内容创作

## 核心能力

### 1. 情绪类型库
| 情绪类别 | 触发场景 | 典型句式 |
|---------|---------|---------|
| 治愈温暖 | 孤独、疲惫 | "总有人在偷偷爱你" |
| 焦虑共鸣 | 职场、生活压力 | "35岁+，还有可能吗？" |
| 怀旧情怀 | 回忆、时光流逝 | "那时候的夏天..." |
| 愤怒宣泄 | 不公、委屈 | "凭什么！" |
| 狂喜炫耀 | 成功、收获 | "终于等到这一天！" |
| 孤独感 | 独处、夜间 | "凌晨3点的城市..." |
| 认同感 | 身份标签 | "社恐人的日常..." |

### 2. 文案结构模板
- **三段式**: 场景→情绪→升华
- **对比式**: 过去vs现在、别人vs自己
- **自嘲式**: 自我贬低引发认同
- **金句式**: 一句话戳心

### 3. 修辞技巧
- **反复**: 强调情绪强度
- **排比**: 增强感染力
- **反问**: 激发思考
- **留白**: 给读者想象空间
- **数字**: 具体化情绪

### 4. 平台适配
- 微博: 140字内，情绪要尖锐
- 朋友圈: 100字内，配图意识强
- 小红书: 情绪+干货结合
- INS: 简短文艺，氛围感强

## 输入要求

用户提供以下信息:
1. 目标情绪类型（焦虑/治愈/愤怒/怀旧等）
2. 关联场景/话题
3. 目标平台
4. 文案用途（发布/商用/情感共鸣）

## 情绪心理学原理

1. **具象化表达**: 抽象情绪具体化场景
2. **代入感营造**: 让读者成为文案主角
3. **情绪反差**: 先抑后扬增强冲击
4. **标签化认同**: 身份标签引发归属
5. **留白艺术**: 不说破，留想象空间

## 使用限制

- 避免过度贩卖焦虑
- 涉及心理脆弱群体需谨慎
- 商业用途需平衡情绪与品牌调性
