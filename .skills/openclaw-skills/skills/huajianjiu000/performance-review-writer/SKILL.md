# Performance Review Writer

## What This Skill Does

Performance reviews shape careers. I help create balanced, constructive reviews that accurately assess performance, acknowledge achievements, and set clear expectations for growth.

## Features

- **Balanced Feedback**: Strengths and growth areas
- **Behavior-Based**: Facts, not feelings
- **Specific Examples**: Evidence-backed observations
- **Development Focus**: Clear path forward
- **Rating Guidance**: Fair evaluation framework

## Review Components

### 1. Core Competencies Assessment
```
Job Knowledge: [Level]
- [Specific observation with example]

Quality of Work: [Level]
- [Specific observation with example]

Productivity: [Level]
- [Specific observation with example]

Communication: [Level]
- [Specific observation with example]

Teamwork: [Level]
- [Specific observation with example]

Initiative: [Level]
- [Specific observation with example]
```

### 2. Strengths Examples
```
"[Employee] consistently delivered [specific result]. 
For example, [concrete example with metrics if possible]. 
This demonstrates [competency/skill] at a [high/exceptional] level."

"[Employee] has shown particular strength in [area]. 
Their contribution to [project/initiative] resulted in 
[positive outcome]."
```

### 3. Growth Areas (Constructive)
```
"[Employee] could benefit from developing [specific skill]. 
[Observation showing opportunity for growth]. 

Recommendation: [Specific action item or support]."

"In [situation], there was an opportunity to [alternative 
approach]. [Employee] is receptive to feedback and committed 
to improvement."
```

### 4. Goals for Next Period
```
Short-term Goals:
1. [Specific, measurable goal] - by [date]
2. [Specific, measurable goal] - by [date]

Long-term Goals:
1. [Development-oriented goal]
2. [Career path aligned goal]

Support needed:
- [Training/mentoring/resource]
```

## Rating Scale Reference

| Rating | Description |
|--------|-------------|
| 5 - Exceptional | Consistently exceeds expectations |
| 4 - Exceeds | Frequently exceeds expectations |
| 3 - Meets | Consistently meets expectations |
| 2 - Developing | Sometimes meets, needs improvement |
| 1 - Needs Improvement | Does not meet expectations |

## Review Phrases

### Exceeds Expectations
- "Consistently goes above and beyond..."
- "Delivers exceptional results in..."
- "Role model for the team in..."
- "Identified and solved problems before..."

### Meets Expectations
- "Reliably completes work to standard..."
- "Consistently demonstrates..."
- "Good at balancing multiple priorities..."
- "Maintains positive relationships with..."

### Needs Improvement
- "Has shown progress in... but..."
- "Could benefit from more focus on..."
- "Opportunity to strengthen..."
- "Needs support to improve..."
