# 学习计划模板库

## 编程语言学习模板

### Python学习计划（12周）

```
# Python学习计划

## 学习目标
- 掌握Python基础语法
- 能够独立完成小项目
- 为数据分析/机器学习打基础

## 每日学习时间
- 每天2小时（工作日）+ 4小时（周末）

---

## 第一阶段：入门（第1-2周）

### 第1周：环境搭建和基础语法
| 日期 | 学习内容 | 练习 |
|------|----------|------|
| 周一 | 安装Python/Jupyter | 配置环境 |
| 周二 | 变量和数据类型 | 基础运算 |
| 周三 | 字符串操作 | 字符串处理练习 |
| 周四 | 条件语句 | if-else练习 |
| 周五 | 循环语句 | for/while循环 |
| 周六 | 总结+实战 | 小计算器 |
| 周日 | 复习 | 知识梳理 |

### 第2周：数据结构
| 主题 | 内容 |
|------|------|
| 列表 | 创建、切片、常用方法 |
| 字典 | 键值对、操作方法 |
| 元组 | 不可变序列 |
| 集合 | 去重、交集、并集 |

---

## 第二阶段：基础（第3-6周）

### 第3-4周：函数和模块
- 函数定义和调用
- 参数传递（位置/关键字）
- 返回值
- 匿名函数lambda
- 模块导入和使用

### 第5-6周：面向对象
- 类和对象
- 继承和多态
- 特殊方法
- 魔术方法

---

## 第三阶段：进阶（第7-10周）

### 第7-8周：文件操作和异常
- 文件读写
- 异常处理
- 上下文管理器
- 日志记录

### 第9-10周：常用库
- os/sys文件操作
- datetime时间处理
- json数据解析
- re正则表达式

---

## 第四阶段：实战（第11-12周）

### 项目实践
1. 命令行TODO应用
2. 数据分析小项目
3. Web爬虫基础
4. API接口调用

---

## 学习资源推荐

### 免费资源
- 官方文档：docs.python.org
- 菜鸟教程：runoob.com/python3
- B站视频教程

### 付费资源
- 极客时间Python课程
- 慕课网实战课程

---

## 评估方式
- 每周自测题
- 阶段项目考核
- 最终综合项目
```

## 考证备考模板

### PMP备考计划（3个月）

```
# PMP备考计划

## 目标
通过PMP认证考试（3A成绩）

## 学习周期
12周，每天2小时

---

## 第一阶段：预热（第1-2周）

### 目标
了解PMBOK框架，建立项目管理思维

### 任务
- [ ] 通读PMBOK第六版
- [ ] 观看基础视频课程
- [ ] 完成每章思维导图

---

## 第二阶段：精读（第3-6周）

### 目标
深入理解五大过程组、十大知识领域

### 任务
- [ ] 精读PMBOK每个知识领域
- [ ] 完成每个ITTO记忆
- [ ] 观看强化视频课程
- [ ] 开始刷题（每日30题）

---

## 第三阶段：冲刺（第7-10周）

### 目标
巩固知识点，提高答题正确率

### 任务
- [ ] 二刷PMBOK重点章节
- [ ] 错题回顾分析
- [ ] 模拟考试（每周1次）
- [ ] 参加学习小组讨论

---

## 第四阶段：考前（第11-12周）

### 任务
- [ ] 全真模拟考试3次
- [ ] 复习错题本
- [ ] 背诵关键公式
- [ ] 调整作息，保持状态

---

## 评估标准
- 模拟题正确率 > 85%
- ITTO能够默写
- 敏捷知识点掌握
```
