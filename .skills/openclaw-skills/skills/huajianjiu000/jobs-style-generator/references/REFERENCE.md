# Jobs Style Generator - Reference Guide

## Steve Jobs' Presentation Mastery

### The Philosophy

**Simplicity is the ultimate sophistication.**
- Every slide: One idea
- Every sentence: Essential
- Every word: Powerful

### The Structure

1. **The Villain** - Define the enemy/problem
2. **The Hero** - Introduce the solution
3. **The Proof** - Demo or evidence
4. **The Vision** - Bigger picture

## Famous Presentations Analysis

### iPhone Launch (2007)
**Structure:**
1. "We're going to make history today"
2. Three devices → One device (the reveal)
3. "Apple is going to reinvent the phone"
4. Demo of revolutionary features

**Key Techniques:**
- Building anticipation from start
- The "three devices" setup
- Dramatic reveal moment
- "Reinvent the phone" - bold claim

### MacBook Air Launch (2008)
**Structure:**
1. Problem: Laptops are too thick
2. Solution: The world's thinnest notebook
3. Proof: Manila envelope demo
4. "It's called MacBook Air"

**Key Techniques:**
- Visual proof (envelope demo)
- Dramatic simplicity
- Name reveal at climax

## Presentation Techniques

### 1. The Rule of Three
```
Bad: "It has many great features"
Good: "It has three things: A, B, and C"
```

### 2. The Reveal
```
Build up → Pause → Reveal → Pause → Reaction
```

### 3. The Contrast
```
"Before: [Problem]"
"After: [Solution]"
```

## Language Patterns

### Simple → Powerful
| Complex | Jobs Style |
|---------|------------|
| Utilize | Use |
| Facilitate | Help |
| Implement | Make |

### Weak → Strong
| Weak | Jobs Style |
|------|------------|
| Pretty good | Incredible |
| Nice | Beautiful |
| Better | Revolutionary |
