# Trend Spotter

## Description

Trend Spotter is your AI-powered trend tracking assistant that monitors and analyzes social media trending topics across multiple platforms. Stay ahead of the curve by identifying emerging trends, viral content patterns, and audience interests before they peak.

## Features

- **Real-time Monitoring**: Track trends across major social platforms
- **Trend Analysis**: Identify pattern shifts and emerging topics
- **Content Opportunities**: Discover content gaps and opportunities
- **Platform Comparison**: Compare trend performance across platforms
- **Timing Insights**: Optimal posting times based on trend activity

## How to Use

1. **Initialize Tracking**
   ```
   Load this skill and specify the platforms you want to monitor
   ```

2. **Set Your Focus Areas**
   - Define industry categories
   - Set target audience demographics
   - Choose relevant keywords and hashtags

3. **Analyze Trends**
   - Run trend analysis for specific topics
   - Compare trends across time periods
   - Identify seasonal patterns

4. **Generate Reports**
   - Create trend reports with visualizations
   - Export data for further analysis
   - Schedule automated trend updates
