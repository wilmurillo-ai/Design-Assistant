# YouTube Thumbnail Analyzer

## name
youtube-thumbnail-analyzer

## description
Analyze YouTube thumbnail designs to maximize click-through rates. Get professional feedback on visual hierarchy, color psychology, text placement, face emotions, and comparison with top-performing thumbnails in your niche. Essential for YouTubers, video marketers, and content creators.

## instructions
### Overview
This skill provides comprehensive analysis of YouTube thumbnails, helping you understand why certain thumbnails perform better and how to optimize your own for maximum clicks. Based on proven visual marketing principles and YouTube algorithm preferences.

### When to Use
- Before uploading a new video to optimize thumbnail appeal
- Redesigning underperforming video thumbnails
- Learning best practices for thumbnail creation
- A/B testing different thumbnail concepts
- Analyzing competitor thumbnails for insights
- Creating thumbnail templates for your brand

### Input Requirements
Provide the following for thumbnail analysis:

1. **Thumbnail Image**: Upload or provide URL of the thumbnail
2. **Video Topic/Title**: What is the video about? (for context)
3. **Target Audience**: Who should click on this video?
4. **Niche/Category**: Content category (tech, lifestyle, gaming, education, etc.)
5. **Current CTR** (Optional): If known, the video's current click-through rate
6. **Comparison Thumbnails** (Optional): 2-3 competitor thumbnails for benchmarking

### Analysis Framework

#### 1. Visual Hierarchy Assessment
Evaluate what draws the eye first:
- Primary focal point identification
- Eye flow path analysis
- Element size and prominence
- Contrast and readability

#### 2. Color Psychology Review
- Color palette effectiveness for the content type
- Contrast ratios for visibility
- Color harmony and brand consistency
- Platform appearance across devices

#### 3. Text Optimization Check
- Text readability at small sizes
- Font choice and emotional impact
- Text placement and hierarchy
- Spelling and grammar accuracy
- Number of words (ideal: 3-5 max)

#### 4. Emotional Impact Evaluation
- Face expressions and emotions conveyed
- Emotional triggers used
- Viewer curiosity factors
- Urgency or FOMO elements

#### 5. Design Best Practices
- Rule of thirds application
- Safe zone compliance (no text/icons cut off)
- Branding elements presence
- Visual clutter assessment
- Aspect ratio correctness

### Example Analysis

**Input Thumbnail**: Tech review video thumbnail
- Shows smartphone with text "BEST Phone 2025?"
- Blue/white background
- Creator face with neutral expression

**Output:**
```
---
# YouTube Thumbnail Analysis Report

## 📊 Overall Score: 7.5/10

## 🎯 Visual Hierarchy
**Strengths:**
- Clear focal point on the phone
- Good contrast between device and background

**Areas for Improvement:**
- Face is small and gets lost
- Text competes with the main subject

**Recommendation:** Enlarge face and move to upper corner

## 🎨 Color Analysis
**Strengths:**
- Clean, professional palette
- Good for tech content

**Areas for Improvement:**
- Lacks visual punch
- Doesn't stand out in search results

**Recommendation:** Add 1-2 accent colors for emphasis

## ✍️ Text Assessment
**Strengths:**
- Question format creates curiosity
- Readable font choice

**Issues Found:**
- "2025" is redundant (if video is current)
- Too many elements competing

**Recommendation:** Simplify to "BEST Phone?" or "WORST Choice"

## 😊 Emotional Impact
**Current Emotional Appeal:** Curious, informational

**What Works:**
- Question format engages curiosity

**Missing Elements:**
- Surprise/shock expression
- Personal opinion indicator (thumbs up/down)

**Recommendation:** Add reaction face showing verdict

## 📝 Summary & Action Items
1. **[High Priority]** Replace face with stronger reaction shot
2. **[Medium Priority]** Simplify text to single compelling phrase
3. **[Low Priority]** Add accent color or glow effect

## 💡 Quick Wins
- Increase face size by 30%
- Add subtle shadow/outline to text
- Add arrows or circles to highlight features
---
```

### Best Practices Reference
- Use faces with expressive emotions (surprise, excitement, confusion)
- Keep text to 3-5 words maximum
- High contrast colors for visibility
- Test thumbnails with 50% saturation
- Use the Rule of Thirds for composition
- Always include safe zones for mobile viewers
- A/B test variations when possible
