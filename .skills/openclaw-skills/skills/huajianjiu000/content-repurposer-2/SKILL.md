# Content Repurposer

## Description
Maximize the value of your existing content by transforming it into multiple formats. Given a piece of content (article, video transcript, podcast notes), generates adaptations for blog posts, social media (Twitter/X, LinkedIn, Instagram), email newsletters, video scripts, and more.

## How to Use
1. Input your original content (text, key points, or summary)
2. Specify target formats needed
3. Define target audience and platform nuances
4. Receive adapted content for each format

## Example

**Input:**
```
Original Content: Blog post about "5 Tips for Remote Work Productivity"
Target Formats: LinkedIn post, Twitter thread, Email newsletter, Video script
Target Audience: Remote workers, team managers
```
