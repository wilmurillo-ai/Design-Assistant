# 面试题库参考

## 通用面试题库

### 开场暖场题
1. 请简单介绍一下你自己
2. 您是通过什么渠道了解到我们公司的？
3. 您目前的状态是？

### 职业规划类
1. 您未来3-5年的职业规划是什么？
2. 您为什么对这个岗位感兴趣？
3. 您期望在下一份工作中获得什么？

### 离职原因类（慎问）
1. 您为什么考虑离开现在的公司？
2. 您对上一份工作最满意和最不满意的是什么？
3. 您希望在新公司避免什么？

### 优劣势类
1. 您觉得自己最大的优势是什么？
2. 您觉得自己需要改进的地方是什么？
3. 您的同事/领导通常如何评价您？

## 技术面试题库

### Java开发
```
【基础】
1. Java和C++的区别？
2. 什么是多态？Java如何实现多态？
3. String、StringBuilder、StringBuffer的区别？

【进阶】
4. HashMap的底层实现原理？
5. ConcurrentHashMap如何保证线程安全？
6. JVM的内存模型和垃圾回收机制？

【高级】
7. 如何排查线上OOM问题？
8. 分布式系统中如何保证数据一致性？
9. Redis的持久化机制和选择？
```

### Python开发
```
【基础】
1. Python中的装饰器是什么？如何使用？
2. 列表、元组、字典的区别？
3. Python中的GIL是什么？

【进阶】
4. 如何理解Python中的生成器和迭代器？
5. Flask和Django的区别？
6. Python中的异步编程如何实现？

【高级】
7. 如何设计一个高可用的爬虫系统？
8. Python中的内存泄漏如何排查？
9. asyncio的使用场景和注意事项？
```

### 测试工程师
```
【基础】
1. 什么是黑盒测试和白盒测试？
2. 测试用例设计方法有哪些？
3. 如何判断测试用例的覆盖率？

【进阶】
4. 接口测试如何做？
5. 如何定位和分析bug？
6. 自动化测试框架如何选型？

【高级】
7. 如何搭建持续集成测试体系？
8. 性能测试的指标和工具？
9. 如何评估测试的充分性？
```

## 产品经理面试题库

### 产品感觉类
1. 您最喜欢的产品是什么？为什么？
2. 如果让您优化XX产品，您会怎么做？
3. 您认为好产品和普通产品的区别是什么？

### 需求分析类
1. 遇到过最难处理的需求是什么？怎么解决的？
2. 如何判断一个需求该不该做？
3. 需求和用户反馈冲突时怎么处理？

### 数据分析类
1. 如何定义产品的核心指标？
2. 发现数据下降如何排查？
3. 如何通过数据验证需求效果？

### 项目管理类
1. 项目延期怎么处理？
2. 如何协调技术和产品的矛盾？
3. 同时多个项目如何管理优先级？
