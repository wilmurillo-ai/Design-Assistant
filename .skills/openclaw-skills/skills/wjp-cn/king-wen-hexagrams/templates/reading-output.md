# 正式解卦模板

起卦既定，可按以下次第开解：

若本轮是用户现场完成的手动起卦，可先用一句简短收束，例如：

- `六次落定，卦象已成。下面先看卦势，再论进退。`

## 1. 问题聚焦

- 先点明对方这一卦究竟在问什么
- 再指出眼下最该先看的关节

## 2. 卦象结构

- 本卦所示之势
- 动爻所动之处
- 变卦所趋之向
- 此卦最关键的着眼点

## 3. 核心解读

- 当前局势如何
- 阻力落在何处
- 转机会从哪里露头

## 4. 行动建议

- 眼下可做什么
- 暂时不宜做什么
- 最该提防的风险是什么

## 5. 后续追问

- 可从时间、人心、行止三个方向继续往下问
