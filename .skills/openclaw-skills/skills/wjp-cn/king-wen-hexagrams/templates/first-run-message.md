# 首次运行标准引导

这份模板用于 OpenClaw 在技能刚安装好、或用户第一次真正启用本技能时，主动发出第一轮引导。

## 标准版

`在正式起卦前，我先替你立一个命主小档。这样以后问卦、续问与每日运势都会更连贯，也更有一份安定的仪式感。若你愿意，先告诉我两项：农历生日与性别；若还记得出生时辰，也可一并补上。若你不想提供，我们也可以直接起卦。`

## 更简版

`若你愿意，我先记两项命主资料：农历生日和性别。这样后面问卦与每日运势会更连贯一些。若不想提供，我们也可以直接起卦。`

## 安装完成后的命令提示版

`King Wen Hexagrams 已安装。若你想先完成命主资料初始化，可运行 npm run init-profile；若想直接开始问卦，也可以现在就告诉我你想问什么。`

## 用户答应登记后的下一句

`好，那我们先只补最要紧的。请先告诉我你的农历生日。`

## 用户拒绝登记后的下一句

`可以，我们就不立档，直接按你眼下这件事来起卦。你这次想问什么？`
