# 文王六十四卦详解

> 本文档整理自《周易》原文，收录文王六十四卦的完整卦辞、彖传、象传及六爻解读。
> 按文王后天卦序（伏羲先天序）编排。

## 一、乾为天（111111）

**卦辞：** 元亨利贞。

**彖曰：** 大哉乾元，万物资始，乃统天。云行雨施，品物流形。大明终始，六位时成，时乘六龙以御天。乾道变化，各正性命，保合大和，乃利贞。

**象曰：** 天行健，君子以自强不息。

### 六爻

| 爻 | 辞 | 要义 |
|----|------|------|
| 初九 | 潜龙勿用 | 阳在地下，君子隐居，时机未到 |
| 九二 | 见龙在田，利见大人 | 阳现地上，贤人出现 |
| 九三 | 君子终日乾乾，夕惕若厉，无咎 | 勤勉警惕，夕惕无咎 |
| 九四 | 或跃在渊，无咎 | 或进或退，审时无灾 |
| 九五 | 飞龙在天，利见大人 | 圣人得位，大展宏图 |
| 上九 | 亢龙有悔 | 阳气过盛，知进忘退，必有悔 |
| 用九 | 群龙无首，吉 | 群龙腾跃不以首领自居，大吉 |

## 二、坤为地（000000）

| 六三 | 含章可贞。或从王事，无成有终 | 含美守正，善终 |
| 六四 | 括囊，无咎无誉 | 谨言慎行 |
| 六五 | 黄裳，元吉 | 黄色衣裳，大吉 |
| 上六 | 龙战于野，其血玄黄 | 阴阳交战 |
| 用六 | 利永贞 | 永久守正 |
