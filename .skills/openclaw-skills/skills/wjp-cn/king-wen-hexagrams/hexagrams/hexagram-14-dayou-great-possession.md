# 第 14 卦 大有

- `全名`：`火天大有`
- `代号`：`dayou`
- `二进制`：`111101`
- `上卦`：`离` / `火` / `明`
- `下卦`：`乾` / `天` / `健`
- `主题`：`丰足与掌控`

## 核心摘要

适合看资源充足时如何用好优势。

## 卦义展开

- 这卦的主轴是`丰足与掌控`，重点不是追求抽象吉凶，而是看你现在所处的结构是否与这个主题吻合。
- 更贴近经传语义的读法是：大有主丰足与掌控，说明手里已有资源与优势，关键变成能否守成、善用、知止。
- 从卦象来看，它还在提醒你：火在天上，光明显著，象征所有权、影响力与可见度都在上升。
- 真正需要小心的地方也很明确：注意盛时易生自满。

## 原典摘录

- `卦辞`：元亨。
- `象曰`：火在天上，大有。君子以遏恶扬善，顺天休命。

## 结构理解

- `内在动因`：下卦为`乾`，更像是在处理“主动开创、定方向、立原则”这一层的内部问题。
- `外部表现`：上卦为`离`，说明外部局势最终会把重点推向“看清事实、辨明重点、让信息显形”。
- `组合理解`：先把内部的`健`这一层站稳，再去面对外部的`明`，所以整体主题才会落到`丰足与掌控`。

- 如果问事业、项目或执行，这卦更强调`丰足与掌控`：事业上适合处理资源高位期的配置问题，重点不是再抢，而是怎么用好已有优势。
- 更稳的做法通常是先处理`乾`所代表的内部节奏，再应对`离`对应的外部变化。

- 落到现实里，仍要回看双方能不能一起承接`丰足与掌控`，而不是只凭一时情绪判断。

- `初爻动`：重点看`起点与试探`。事情刚起头，先看第一步是否对路，动作是否过急。更贴近本卦原典语义时，可读成：`初九`曰“无交害，匪咎，艰则无咎”；可把要点抓成：无交害，艰苦则无咎。

- `上爻动`：重点看`收束与过度`。这一层多看是否已经走到尾声，或是否出现过头迹象。更贴近本卦原典语义时，可读成：`上九`曰“自天祐之，吉，无不利”；可把要点抓成：得天保佑，吉祥。
