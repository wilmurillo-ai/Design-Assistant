# 第 64 卦 未济

- `全名`：`火水未济`
- `代号`：`weiji`
- `二进制`：`010101`
- `上卦`：`离` / `火` / `明`
- `下卦`：`坎` / `水` / `险`
- `主题`：`将成未成`

## 核心摘要

适合看临门一脚前的整理和校准。

## 卦义展开

- 这卦的主轴是`将成未成`，重点不是追求抽象吉凶，而是看你现在所处的结构是否与这个主题吻合。
- 更贴近经传语义的读法是：未济主将成未成，说明离成功只差最后整理，但越到这里越不能急躁抢线。
- 从卦象来看，它还在提醒你：火在水上，明在险前，象征已经看见终点，却仍要一项项校准、逐步过河。
- 真正需要小心的地方也很明确：注意差最后一步时最容易急。

## 原典摘录

- `卦辞`：亨。小狐汔济，濡其尾，无攸利。
- `象曰`：火在水上，未济。君子以慎辨物居方。

- `内在动因`：下卦为`坎`，更像是在处理“风险压力、反复测试、在险中求稳”这一层的内部问题。

- `组合理解`：先把内部的`险`这一层站稳，再去面对外部的`明`，所以整体主题才会落到`将成未成`。
