# 第 38 卦 睽

- `全名`：`火泽睽`
- `代号`：`kui`
- `二进制`：`110101`

- `下卦`：`兑` / `泽` / `悦`
- `主题`：`相异与背离`

## 核心摘要

适合判断观点不一、步调不齐的局面。

## 卦义展开

- 这卦的主轴是`相异与背离`，重点不是追求抽象吉凶，而是看你现在所处的结构是否与这个主题吻合。
- 更贴近经传语义的读法是：睽主相异与背离，说明差异已显，关键不是假装一致，而是学会在不一致中找最低限度的可合作点。
- 从卦象来看，它还在提醒你：上火下泽，二气不交，象征看法不同、立场不同，但小事尚可图吉。
- 真正需要小心的地方也很明确：注意差异扩大后更难回头。

## 原典摘录

- `卦辞`：小事吉。
- `象曰`：上火下泽，睽。君子以同而异。

- `组合理解`：先把内部的`悦`这一层站稳，再去面对外部的`明`，所以整体主题才会落到`相异与背离`。

- 落到现实里，仍要回看双方能不能一起承接`相异与背离`，而不是只凭一时情绪判断。
