# 第 18 卦 蛊

- `全名`：`山风蛊`
- `代号`：`gu`
- `二进制`：`011001`
- `上卦`：`艮` / `山` / `止`
- `下卦`：`巽` / `风` / `入`
- `主题`：`整顿积弊`

## 核心摘要

适合看旧问题、烂尾和历史包袱。

## 卦义展开

- 这卦的主轴是`整顿积弊`，重点不是追求抽象吉凶，而是看你现在所处的结构是否与这个主题吻合。
- 更贴近经传语义的读法是：蛊主整顿积弊，说明旧问题已经不能再拖，必须下决心修根本，而非只修表面。
- 从卦象来看，它还在提醒你：山下有风，象征久积之弊在内部发酵，整理虽费力，却是转败为治的起点。
- 真正需要小心的地方也很明确：注意只修表面难以止损。

## 原典摘录

- `卦辞`：元亨。利涉大川。先甲三日，后甲三日。
- `象曰`：山下有风，蛊。君子以振民育德。

## 结构理解

- `内在动因`：下卦为`巽`，更像是在处理“渐进渗透、协商进入、柔性推进”这一层的内部问题。
- `外部表现`：上卦为`艮`，说明外部局势最终会把重点推向“止损收束、设边界、先稳住脚跟”。
- `组合理解`：先把内部的`入`这一层站稳，再去面对外部的`止`，所以整体主题才会落到`整顿积弊`。

## 常见问事场景

### 事业与项目

- 如果问事业、项目或执行，这卦更强调`整顿积弊`：事业上适合处理烂尾、历史遗留、制度失灵和长期积压的问题。
- 更稳的做法通常是先处理`巽`所代表的内部节奏，再应对`艮`对应的外部变化。

### 合作与关系

- 如果问合作、团队或关系，这卦更贴近的提醒是：关系里常见旧伤、旧账、旧误会反复浮现，若不正视，就会一直发酵。
- 落到现实里，仍要回看双方能不能一起承接`整顿积弊`，而不是只凭一时情绪判断。

### 时机与去留

- 如果问现在该不该推进，这卦更常给出的节奏判断是：时机偏向宜治旧病，不宜只做新动作；先把烂根清掉，再谈新增长。
- 可以先看下卦的`入`是否稳住，再看上卦的`止`是否已经给出更明确的外部信号。

## 实务建议

- `宜`：事业上适合处理烂尾、历史遗留、制度失灵和长期积压的问题。
