# 第 33 卦 遁

- `全名`：`天山遁`
- `代号`：`dun`
- `二进制`：`001111`
- `上卦`：`乾` / `天` / `健`
- `下卦`：`艮` / `山` / `止`
- `主题`：`退避与抽身`

## 核心摘要

适合看何时该收、该退、该避锋芒。

## 卦义展开

- 这卦的主轴是`退避与抽身`，重点不是追求抽象吉凶，而是看你现在所处的结构是否与这个主题吻合。
- 更贴近经传语义的读法是：遁不是失败退场，而是看懂局势后主动抽身、保存实力、避其锋芒。
- 从卦象来看，它还在提醒你：天下有山，山势渐逼，象征小人渐长或风险渐近，此时高明之处在于退得及时。
- 真正需要小心的地方也很明确：注意退得太慢会失去主动。

## 原典摘录

- `卦辞`：亨，小利贞。
- `象曰`：天下有山，遁。君子以远小人，不恶而严。

## 结构理解

- `内在动因`：下卦为`艮`，更像是在处理“止损收束、设边界、先稳住脚跟”这一层的内部问题。
- `外部表现`：上卦为`乾`，说明外部局势最终会把重点推向“主动开创、定方向、立原则”。
- `组合理解`：先把内部的`止`这一层站稳，再去面对外部的`健`，所以整体主题才会落到`退避与抽身`。

## 常见问事场景

### 事业与项目

- 如果问事业、项目或执行，这卦更强调`退避与抽身`：事业上适合处理撤退、降杠杆、退出消耗局和避开不必要的正面冲撞。
- 更稳的做法通常是先处理`艮`所代表的内部节奏，再应对`乾`对应的外部变化。

### 合作与关系

- 如果问合作、团队或关系，这卦更贴近的提醒是：关系里它提示适度拉开距离、停止纠缠，不要在不对的局里越陷越深。
- 落到现实里，仍要回看双方能不能一起承接`退避与抽身`，而不是只凭一时情绪判断。

### 时机与去留

- 如果问现在该不该推进，这卦更常给出的节奏判断是：时机偏向宜退不宜争，先保全自己，再等待新的更好位置。
- 可以先看下卦的`止`是否稳住，再看上卦的`健`是否已经给出更明确的外部信号。

## 实务建议

- `宜`：事业上适合处理撤退、降杠杆、退出消耗局和避开不必要的正面冲撞。
