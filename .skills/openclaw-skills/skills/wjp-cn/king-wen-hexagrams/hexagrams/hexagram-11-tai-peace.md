# 第 11 卦 泰

- `全名`：`地天泰`
- `代号`：`tai`
- `二进制`：`111000`
- `上卦`：`坤` / `地` / `顺`
- `下卦`：`乾` / `天` / `健`
- `主题`：`通达顺行`

## 核心摘要

适合看内外协调、上下通气的阶段。

## 卦义展开

- 这卦的主轴是`通达顺行`，重点不是追求抽象吉凶，而是看你现在所处的结构是否与这个主题吻合。
- 更贴近经传语义的读法是：泰主上下交而其志同，说明局面处在通达顺行的阶段，但越顺越要防后续松弛。
- 从卦象来看，它还在提醒你：天地交而万物通，真正的泰不是安逸，而是内外通气、上下相接、循环畅达。
- 真正需要小心的地方也很明确：注意顺时更要防松懈。

## 原典摘录

- `卦辞`：小往大来，吉，亨。
- `象曰`：天地交，泰。后以财成天地之道，辅相天地之宜。

## 结构理解

- `内在动因`：下卦为`乾`，更像是在处理“主动开创、定方向、立原则”这一层的内部问题。
- `外部表现`：上卦为`坤`，说明外部局势最终会把重点推向“承接配合、整合资源、厚实基础”。
- `组合理解`：先把内部的`健`这一层站稳，再去面对外部的`顺`，所以整体主题才会落到`通达顺行`。

### 事业与项目

- 如果问事业、项目或执行，这卦更强调`通达顺行`：事业上多见于团队协作顺畅、资源调配通达、上下目标一致的时候。
- 更稳的做法通常是先处理`乾`所代表的内部节奏，再应对`坤`对应的外部变化。

### 合作与关系

- 如果问合作、团队或关系，这卦更贴近的提醒是：关系里它偏向彼此理解加深、互动顺畅、愿意互相成就。
- 落到现实里，仍要回看双方能不能一起承接`通达顺行`，而不是只凭一时情绪判断。

- 如果问现在该不该推进，这卦更常给出的节奏判断是：时机偏向可进可成，但越在顺势中，越要提前防范由盛转懈。
- 可以先看下卦的`健`是否稳住，再看上卦的`顺`是否已经给出更明确的外部信号。
