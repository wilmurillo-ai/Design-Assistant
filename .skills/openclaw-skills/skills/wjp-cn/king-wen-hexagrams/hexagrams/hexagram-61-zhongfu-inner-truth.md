# 第 61 卦 中孚

- `全名`：`风泽中孚`
- `代号`：`zhongfu`
- `二进制`：`110011`
- `上卦`：`巽` / `风` / `入`
- `下卦`：`兑` / `泽` / `悦`
- `主题`：`诚信与内实`

## 核心摘要

适合看信任、共识与内在真实意愿。

## 卦义展开

- 这卦的主轴是`诚信与内实`，重点不是追求抽象吉凶，而是看你现在所处的结构是否与这个主题吻合。
- 更贴近经传语义的读法是：中孚重在诚信与内实，真正决定关系与合作走向的，不是话说得多好，而是心里是否真。
- 从卦象来看，它还在提醒你：泽上有风，内虚而中实，象征以真实之心感动外界，故能涉大川。
- 真正需要小心的地方也很明确：注意空谈真诚而缺少兑现。

## 原典摘录

- `卦辞`：豚鱼吉。利涉大川。利贞。

- `内在动因`：下卦为`兑`，更像是在处理“沟通表达、协商互动、反馈回响”这一层的内部问题。

- `组合理解`：先把内部的`悦`这一层站稳，再去面对外部的`入`，所以整体主题才会落到`诚信与内实`。

## 常见问事场景

### 事业与项目

- 如果问事业、项目或执行，这卦更强调`诚信与内实`：事业上适合建立信任、形成共识、做长期合作和靠信誉取胜的事务。
- 更稳的做法通常是先处理`兑`所代表的内部节奏，再应对`巽`对应的外部变化。

### 合作与关系

- 如果问合作、团队或关系，这卦更贴近的提醒是：关系里它偏向真心、信任、愿意袒露内在，但也要求兑现与一致。
- 落到现实里，仍要回看双方能不能一起承接`诚信与内实`，而不是只凭一时情绪判断。

### 时机与去留

- 如果问现在该不该推进，这卦更常给出的节奏判断是：时机偏向宜以诚取人、以实立信，虚张声势反而会坏事。
- 可以先看下卦的`悦`是否稳住，再看上卦的`入`是否已经给出更明确的外部信号。

## 实务建议
