# 第 04 卦 蒙

- `全名`：`山水蒙`
- `代号`：`meng`
- `二进制`：`010001`
- `上卦`：`艮` / `山` / `止`
- `下卦`：`坎` / `水` / `险`
- `主题`：`启蒙与求教`

## 核心摘要

适合处理经验不足、信息不明的局面。

## 卦义展开

- 这卦的主轴是`启蒙与求教`，重点不是追求抽象吉凶，而是看你现在所处的结构是否与这个主题吻合。
- 更贴近经传语义的读法是：蒙主启蒙与求教，提醒人承认未知、先受教、再判断，不宜半懂不懂却急着定论。
- 从卦象来看，它还在提醒你：山下出泉，意在由遮蔽走向开窍，所以重在问对问题、找对老师、守住学习次序。
- 真正需要小心的地方也很明确：注意自以为懂会走偏。

## 原典摘录

- `卦辞`：亨。匪我求童蒙，童蒙求我。初噬告，以次勿问，烦乃不讼。利贞。
- `象曰`：山下出泉，蒙。君子以果行育德。

## 结构理解

- `内在动因`：下卦为`坎`，更像是在处理“风险压力、反复测试、在险中求稳”这一层的内部问题。
- `外部表现`：上卦为`艮`，说明外部局势最终会把重点推向“止损收束、设边界、先稳住脚跟”。
- `组合理解`：先把内部的`险`这一层站稳，再去面对外部的`止`，所以整体主题才会落到`启蒙与求教`。

## 常见问事场景

### 事业与项目

- 如果问事业、项目或执行，这卦更强调`启蒙与求教`：事业上多见于经验不足、信息不全、团队尚未成熟的阶段，宜先补认知再谈动作。
- 更稳的做法通常是先处理`坎`所代表的内部节奏，再应对`艮`对应的外部变化。

### 合作与关系

- 如果问合作、团队或关系，这卦更贴近的提醒是：关系里容易把想象当现实，若不澄清误会和期待，越投入越容易走偏。
- 落到现实里，仍要回看双方能不能一起承接`启蒙与求教`，而不是只凭一时情绪判断。

### 时机与去留

- 如果问现在该不该推进，这卦更常给出的节奏判断是：时机上宜先问清、先学会、先看懂，不宜在认知模糊时就强行推进。
- 可以先看下卦的`险`是否稳住，再看上卦的`止`是否已经给出更明确的外部信号。

- `初爻动`：重点看`起点与试探`。事情刚起头，先看第一步是否对路，动作是否过急。更贴近本卦原典语义时，可读成：`初六`曰“发蒙，利用刑人，用说桎梏以往吝”；可把要点抓成：启发蒙昧，宜教育人。
