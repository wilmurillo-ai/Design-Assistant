# Brunson Academy

## Description
Hybrid skill combining Russell Brunson's trilogy (Expert Secrets, DotCom Secrets, Traffic Secrets) into actionable frameworks for high-ticket offers. 80% quick commands for immediate results, 20% interactive mentor mode for learning.

## Purpose
Transform Brunson's marketing frameworks into operational tools for the Tio Huli ecosystem, increasing conversion rates, average ticket, and traffic quality for Master Business (R$ 15-25k investment program).

## Architecture

### Quick Commands (Core):
- `/brunson value-ladder [product]` - Complete Value Ladder
- `/brunson script [product]` - Epiphany Bridge script
- `/brunson traffic-plan [niche]` - Dream 100 + traffic strategy
- `/brunson webinar [product]` - Perfect Webinar script
- `/brunson analyze [URL/copy]` - Automatic diagnosis
- `/brunson email-seq [product]` - 7-day email sequence

### Mentor Mode (Optional):
- `/brunson coach` - Activates Coach Brunson AI
- `/brunson academy` - Educational modules
- `/brunson feedback [text]` - Personalized feedback

### Integrations:
- `/brunson joaquim [product]` - Copy via Joaquim + Brunson
- `/brunson luiz-audit [copy]` - Audit with Brunson metrics
