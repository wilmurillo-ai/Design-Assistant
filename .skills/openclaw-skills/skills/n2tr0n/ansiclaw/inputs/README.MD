# ANSIClaw Inputs

Drop any file here that you want Clawd to create an ANSI rendition of — a photo, logo, reference image, whatever.

> **Clawd ignores this folder unless explicitly told to use it.** To trigger it, say something like: *"create a rendition of the file in inputs"* or *"draw an ANSI version of inputs/photo.jpg"*.

---

## How It Works

1. Place your source file here (image, PNG, JPG, etc.)
2. Tell Clawd which file to render and in what style
3. Clawd will analyze the image and produce an ANSI interpretation using the Clawbius API
4. Output is saved to `~/Documents/ANSIClaw Output/` as `.ans` + `.png`
