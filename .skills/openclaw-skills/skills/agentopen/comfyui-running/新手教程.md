## ⚙️ 配置文件怎么改？（重点！）

### config.json 是什么？

这是告诉 AI"你的 ComfyUI 在哪"的配置文件！

用**记事本**（或写字板）打开 `config.json`，长这样：

```json
{
  "comfyui_root": "这里填你的路径",
  "python_path": "这里填Python路径",
  "workflows_dir": "工作流文件夹（一般不用改）",
  "output_dir": "图片输出位置（一般不用改）",
  "comfyui_port": 8188,
  "ui_type": "aki"
}
```

### 🪟 Windows 用户（大多数人的选择）

#### 怎么找到我的路径？

**方法：按 Win + E 打开文件管理器**

1. 找到你的 ComfyUI 文件夹
2. 点击地址栏，会显示路径
3. 复制这个路径

#### 常见路径对照表

| 你的 ComfyUI 在哪 | config.json 里的 comfyui_root 怎么写 |
|------------------|-------------------------------------|
| `H:\ComfyUI-aki-v3\ComfyUI` | `"H:\\ComfyUI-aki-v3\\ComfyUI"` |
| `D:\ComfyUI-aki-v3\ComfyUI` | `"D:\\ComfyUI-aki-v3\\ComfyUI"` |
| `D:\ComfyUI\ComfyUI` | `"D:\\ComfyUI\\ComfyUI"` |
| `C:\Users\张三\ComfyUI` | `"C:\\Users\\张三\\ComfyUI"` |
| `E:\ComfyUI-aki\ComfyUI` | `"E:\\ComfyUI-aki\\ComfyUI"` |

> 💡 **注意**：路径里的 `\` 要变成 `\\`（写两遍）！

#### Windows Python 路径

| 你的 Python 在哪 | python_path 怎么写 |
|-----------------|-------------------|
| `H:\ComfyUI-aki-v3\python\python.exe` | `"H:\\ComfyUI-aki-v3\\python\\python.exe"` |
| `D:\ComfyUI\python\python.exe` | `"D:\\ComfyUI\\python\\python.exe"` |

#### Windows 配置示例

```json
{
  "comfyui_root": "H:\\ComfyUI-aki-v3\\ComfyUI",
  "python_path": "H:\\ComfyUI-aki-v3\\python\\python.exe",
  "workflows_dir": "skills\\comfyui-running\\example_workflows",
  "output_dir": "H:\\ComfyUI-aki-v3\\ComfyUI\\output",
  "comfyui_port": 8188,
  "ui_type": "aki"
}
```

### 🔄 WSL 用户（Windows 上的 Linux 子系统）

#### 什么是 WSL？

WSL = Windows Subsystem for Linux，让你在 Windows 里跑 Linux。

#### 怎么找到我的路径？

**方法1：Windows 文件管理器**
```
\\wsl$\Ubuntu\home\用户名\ComfyUI
```

**方法2：WSL 终端**
```bash
pwd
# 会显示类似：/home/用户名/ComfyUI
```

#### WSL 路径对照表

| 你的 ComfyUI 在哪 | config.json 里的 comfyui_root 怎么写 |
|------------------|-------------------------------------|
| Windows 盘符里的 ComfyUI | `"/mnt/h/ComfyUI-aki-v3/ComfyUI"` |
| WSL 里的 ComfyUI | `"/home/用户名/ComfyUI"` |

> 💡 **注意**：`/mnt/c` = C盘，`/mnt/h` = H盘

#### WSL Python 路径

| 你的 Python 在哪 | python_path 怎么写 |
|-----------------|-------------------|
| WSL 系统 Python | `"/usr/bin/python3"` |
| Conda in WSL | `"/home/用户名/miniconda3/bin/python"` |
| 访问 Windows Python | `"/mnt/h/ComfyUI-aki-v3/python/python.exe"` |

#### WSL 配置示例


## 🔧 参数说明（可选）

### 常用参数

| 参数 | 是什么意思 | 怎么用 |
|------|-----------|--------|
| `prompt` | 你想要什么 | 直接描述就行 |
| `steps` | 图片质量（越大越好） | 默认 20，推荐 20-30 |
| `seed` | 随机种子 | 不填就随机 |
| `width` | 图片宽度 | 默认 512 |
| `height` | 图片高度 | 默认 512 |
| `batch_size` | 一次生成几张 | 默认 1 |

### 尺寸推荐

| 尺寸 | 适合 |
|------|------|
| 512 × 512 | 快速预览 |
| 768 × 768 | 头像/头像 |
| 1024 × 1024 | 高清大图 |
| 896 × 512 | 横版风景 |
| 512 × 896 | 竖版人像 |

### steps 质量对比

| steps | 速度 | 质量 |
|-------|------|------|
| 10-15 | ⚡ 很快 | 一般（预览用） |
| 20-25 | ⏱️ 正常 | 够用（推荐） |
| 30-40 | ⏳ 慢 | 很好 |
| 40+ | 🐌 很慢 | 极致 |

## ❓ 遇到问题了？

### 问题1：AI 说找不到 ComfyUI

**解决方法：**
1. 确认 config.json 里的路径正确
2. 路径分隔符用对了吗？
   - Windows: `\\` (两个反斜杠)
   - Linux/macOS: `/` (一个斜杠)

### 问题2：生成的图片很奇怪/模糊

**解决方法：**
- 把 steps 改成 25 或 30
- 提示词说得更具体：
  - ❌ "一只猫" 
  - ✅ "一只橘色的胖猫，阳光下，毛茸茸的"

### 问题3：生成太慢了

**解决方法：**
- 把 steps 改成 15 或 20
- 把尺寸改小一点，比如 512 × 512

### 问题4：想生成一样的图片

**解决方法：**
指定种子（seed）：

```
生成一张海边日落，种子用 123456789
```

同样的种子 + 同样的描述 = 同样的图片！

### 问题5：不知道提示词怎么写

**英文提示词更好！** 参考：

| 想要什么 | 英文提示词 |
|---------|-----------|
| 美女 | `a beautiful girl, realistic` |
| 风景 | `landscape, mountains, sunset` |
| 猫 | `a cute fluffy cat` |
| 动漫风格 | `anime style, cartoon` |

## 📋 工作流文件（高级）

### default.json 是什么？

这是一个"配方"文件，告诉 AI 怎么一步一步生成图片。

### 我能改什么？

**分辨率（图片大小）：**
找到 `EmptyLatentImage` 那部分，改 `width` 和 `height`：

```json
"width": 1024,
"height": 1024,
```

**质量（步数）：**
找到 `KSampler` 那部分，改 `steps`：

```json
"steps": 30,
```

**模型：**
找到 `CheckpointLoaderSimple` 那部分，改 `ckpt_name`：

```json
"ckpt_name": "你的模型文件名.safetensors"
```

> ⚠️ 模型文件要去 `ComfyUI目录\models\checkpoints\` 找！

## 🎯 常用命令收藏

```python
# 最简单的
quick_generate("一只可爱的猫")

# 高质量
quick_generate("一个美女", steps=30)

# 指定尺寸
quick_generate("风景图", width=1024, height=768)

# 一次4张
quick_generate("一只狗", number=4)

# 固定图片（可复现）
quick_generate("海边日落", seed=123456789)

# 带负向提示词（去掉不要的）
quick_generate("一个女孩", negative="模糊, 低质量")
```
