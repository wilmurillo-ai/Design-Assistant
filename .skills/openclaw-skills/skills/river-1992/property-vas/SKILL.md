```
VCP 顶层设计（梁宁）
    ↓

    ↓
四步重塑法：诊断 → 设计 → 实施 → 固化
```

## 选品篇

### 四维选品评审（决策工具）

| 维度 | 权重 | 核心问题 |
|------|------|---------|
| **物业壁垒** | 30% | 别人能轻易复制吗？物业有没有天然信任优势？ |
| **真实需求** | 30% | 业主有强需求吗？频次高吗？ |
| **利润空间** | 25% | 佣金率/毛利能否覆盖运营成本？ |
| **供应链可控性** | 15% | 能不能稳定履约？ |

**低于70分不引进。** 评分工具见 `references/product-scoring.md`

### 3+1产品矩阵（物业版）

| 层级 | 类型 | 物业举例 | 策略 |
|------|------|---------|------|
| 引流款 | 低价高频 | 矿泉水/日用品 | 补贴获客，建私域信任 |
| 入门款 | 日常补货 | 米面粮油/生鲜 | 稳定复购，增现金流 |
| 圈层款 | 会员服务 | "荣安邻里"会员群 | 筛选高净值业主 |
| **利润款** | 高客单重交付 | 家政年卡/装修改造 | **核心利润来源** |

> ⚠️ 冷启动不要从低往高做——直接切入高客单赛道，用低客单做信任背书。

## 供应商洽谈篇

### 谈判优先级（从小到大）

```
纯佣金模式（最优先）
  → 平台费模式（次优先）
  → 买断模式（慎用）
  → 合资模式（大型项目才考虑）
```

### 谈判要点清单

- 优先谈**纯拥金模式**：物业不出钱，供应商送货，成交后结算
- 佣金率参考：标准品 10-15%，非标品 15-25%
- 账期：月结30天（次月10个工作日内）
- 破损责任：**到货验收前由供应商承担**，验收后正常损耗内部消化
- 退货政策：未售出临期品可退换

完整谈判 SOP 见 `references/supplier-negotiation.md`

## 到家服务标准化篇

### 服务商引入检查清单

- [ ] 证照齐全（营业执照 + 行业资质）
- [ ] 有明确的服务标准手册
- [ ] 配送/上门时效承诺
- [ ] 售后处理时效（48小时受理/72小时处理）
- [ ] 试服务体验（自己先用一次）

### 服务上架最小信息

```
服务名称 + 简介
价格（原售价 / 体验价）
适用小区范围
预约方式（小程序内直接约）
服务流程说明
售后条款
```

## 私域运营篇

### 私域定位三步法

```
第一步：我是谁 → 社区物业严选服务/物业多种经营负责人
第二步：我服务谁 → 本小区/本区域业主
第三步：我提供什么价值 →
  日常：物业严选好物送到家
  重大时刻：节礼/到家服务/装修改造
```

### 日常内容节奏

| 时间 | 内容类型 | 比例 |
|------|---------|------|
| 平时 | 生活内容 + 软性种草 | 70% |
| 大促前1-2周 | 预热倒计时 + 客户好评 | 20% |
| 发售当天 | 硬广 + 成交流水 | 10% |

> 参考：肖逸群"6+6"裂变式发售模型（`references/launch-methodology.md`）

## 发售篇（四步法）

### 高客单发售四步

```
STEP 1  预热（发售前1-3个月）
  → 朋友圈图文预告、释放信号
  → 送书籍/资料包积累名单

STEP 2  内容引流（发售中）
  → 发布核心销售信（5000-8000字）
  → 一鱼多吃：长文 → 摘要 → 朋友圈图文
  → 标题：出人意料，情理之中

STEP 3  社群转化（发售中）
  → 3天免费公开课（逐字稿，非直播）
  → 每节内容指向最终产品销售

STEP 4  直播成交（发售高潮）
  → 连麦：优先请在你这拿到结果的人
  → 时长：万元级产品平均需听100分钟+
  → 涨价倒计时 + 限时福利收尾
```

### 发售核心公式

```
GMV = 一度成交人数 + 二度成交人数
二度成交 = 朋友圈转发 → 新客户（占70-90%）
```

物业天然一度关系：邻居信任 >> 普通电商

### 季度发售日历（模板）

| 季度 | 主题 | 主推产品 |
|------|------|--------|
| Q1 | 年味·家的味道 | 年货礼盒 + 年夜饭预制 |
| Q2 | 春夏清爽季 | 清洁服务 + 洗护 |
| Q3 | 金秋感恩 | 教师节礼 + 中秋 + 开学季 |
| Q4 | 温暖过年 | 家政清洗 + 年货 |
