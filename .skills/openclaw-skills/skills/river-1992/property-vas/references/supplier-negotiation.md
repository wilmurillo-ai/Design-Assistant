# 供应商洽谈 SOP

> 目标：谈下纯佣金合作，零资金占用，零库存风险

---

## 合作模式优先级

| 优先级 | 模式 | 说明 |
|--------|------|------|
| 🥇 最高 | 纯佣金 | 成交后结算，不成交不付费，零风险 |
| 🥈 次高 | 平台费 + 佣金 | 收固定平台费 + 成交佣金 |
| 🥉 备选 | 买断制 | 一次性买断，库存风险自担，慎用 |
| ❌ 不用 | 合资 | 复杂，适合大项目，不适合社区零售 |

## 洽谈话术模板

### 开场：介绍自己
> "我们是社区物业严选服务，专注宁波本地社区的物业严选平台。目前服务[X]个小区，超过[X]户业主。我们的定位是'物业严选好物送到家'，不做低质低价，主打信任和服务。"

### 谈佣金：直接报价
> "我们的合作模式是纯佣金——您供货，我们卖，卖出多少按比例结算，卖不出的退货处理，全程您零资金占用、零库存风险。按照行业标准，我们建议佣金比例在[ X ]%，您看这个比例能合作吗？"

### 应对压价
供应商说"你们量不大，要降低佣金"：
> "目前我们处于快速扩张期，单小区体量可能有限，但我们有[ X ]个小区的联动资源。如果您愿意给一个好的开局价格，我们承诺首批铺货[X]个小区，给您一个基础的量保证。"

### 谈账期
> "我们财务流程是每月结算一次，次月10个工作日内付款。账期不超过30天，对您来说资金压力也不大。"

## 合同必备条款（最低要求）

```
1. 合作模式：纯佣金，月结算
2. 佣金比例：____%
3. 结算周期：每月1日~月末，次月10个工作日内付款
4. 退货政策：未售出临期品（到期前30天）可退换
5. 破损责任：到货验收前由供应商承担；验收后正常损耗物业内部消化
6. 售后时效：48小时内受理投诉，72小时内给出处理方案
7. 最低定价：供应商须保证平台价格不高于同期市场价
8. 排他条款：（可选）本小区/本区域内独家合作
```
