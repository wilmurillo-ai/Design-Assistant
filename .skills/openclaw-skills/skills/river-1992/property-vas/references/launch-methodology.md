# 高客单发售方法论（肖逸群模型·物业版）

> 核心理念：流量的终点是私域，私域的终点是发售

---

## 物业发售的独特优势

普通电商：一度客户 → 朋友圈转发 → 二度成交（难，信任弱）
物业私域：邻居关系 → 天然一度信任 → 朋友圈裂变（易，转化高）

**一度关系（邻居信任）是物业做发售最大的护城河。**

---

## 6+6 裂变式发售模型

### 六大私域场景

| 场景 | 物业版对应 |
|------|-----------|
| 朋友圈图文 | 管家号/物业号朋友圈：倒计时+生活+好评 |
| 公众号销售信 | **核心武器**：物业故事（5000-8000字长文） |
| 视频号视频 | 短视频展示服务过程/产品 |
| 社群群容器 | 业主群/会员群：承接精准流量 |
| 直播间 | 物业直播间：服务展示+答疑+成交 |
| 私聊1对1 | 管家私聊：高意向客户逼单 |

### 六大核心要素

```
① 销售信（Story-WHY-How-What）
② 子弹好友激活（种子用户）
③ 朋友圈裂变（转发机制）
④ 社群预热（3天公开课）
⑤ 直播成交（100分钟+）
⑥ 涨价倒计时（限时优惠收尾）
```

## 物业版销售信写法

### 标题原则
- 出人意料，又在情理之中
- 引发好奇，而非标题党

**物业爆款标题方向：**
- "我在小区做了X次洗车，发现一个惊人规律……"
- "邻居推荐的这款水，竟然治好了我的选择困难症"
- "我们小区的物业，开始卖XXX了（附业主真实反馈）"

### 正文结构（Story-WHY-How-What）

```
【Story 故事】用一个真实业主故事开场（500字）
  → 建立情感连接，让读者代入

【Why 为什么】解释为什么我们来做这件事（500字）
  → 我们是谁？为什么可信？
  → 物业背书+员工故事+初心

【How 怎么做】我们是如何选品/服务的（800字）
  → 供应商筛选标准
  → 服务标准承诺
  → 和其他平台的区别

【What 卖什么】本次发售的产品+优惠（收尾）
  → 限量XX份
  → 限时优惠价
  → 如何下单
```

### 一鱼三吃策略

将一篇销售信拆成3种形式发布：

| 形式 | 字数 | 渠道 | 目的 |
|------|------|------|------|
| 完整版 | 5000-8000字 | 业主群/公众号 | 建立深度信任 |
| 摘要版 | 1000字 | 朋友圈 | 引发好奇，引导进群 |
| 截图版 | 3-5张图 | 朋友圈/群 | 降低阅读门槛 |

## 社群运营 SOP

### 3天免费公开课设计

| 时间 | 内容主题 | 目的 |
|------|---------|------|
| 第1天 | 痛点挖掘：你们有没有遇到过XX问题？ | 引发共鸣 |
| 第2天 | 解决方案：我们是怎么解决的（产品介绍） | 建立信任 |
| 第3天 | 成交主张：这次我们带来了什么优惠？ | 发起成交 |

> ⚠️ 内容必须是逐字稿，不是直播。
> 原因：减轻主讲的直播压力，用户可以爬楼回看，重复吸收。

## 直播发售 SOP

### 直播前准备

- [ ] 确定直播平台（视频号/微信群直播）
- [ ] 预约直播：提前3天在业主群预告
- [ ] 整理答疑 Q&A（20个常见问题）
- [ ] 准备好涨价倒计时PPT
- [ ] 邀请2-3位"种子用户"连麦分享

### 直播结构（100分钟模板）

```
0-10min   开场：自我介绍 + 今天为什么做这场直播
10-30min  痛点拆解：业主的XX烦恼，你有没有？
30-60min  产品/服务展示：详细介绍+现场演示+用户见证
60-80min  答疑互动：连麦解答 + 种子用户分享
80-95min  成交说明：限时优惠 + 涨价倒计时
95-100min 收尾：再次强调价值 + 告知购买方式
```

### 直播后跟进

- [ ] 2小时内私聊所有观看未购买用户
- [ ] 群里发购买截图（制造紧迫感）
- [ ] 24小时内跟进未付款订单
- [ ] 48小时内回访已购买用户体验
