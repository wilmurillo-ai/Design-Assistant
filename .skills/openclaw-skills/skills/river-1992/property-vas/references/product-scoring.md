# 四维选品评审工具

> 每个拟引进品类，必须先完成此评审表。低于70分建议不引进或暂缓。

---

## 评审维度说明

### 维度一：物业壁垒（30%）
物业做这件事，别人做不了什么？差距越大，壁垒越高。

加分项：
- 物业员工可上门服务（+10）
- 管家/物业有信任背书（+5）
- 小区内前置仓/自提（+5）
- 业主专属优惠协议（+5）

扣分项：
- 电商价格更低（-10）
- 无差异化卖点（-15）

### 维度二：真实需求（30%）
目标用户（小区业主）是否有强需求？频次如何？

强需求信号：
- 业主在群里问过、投诉过、讨论过
- 是生活必需品或高频服务
- 有明确使用场景

### 维度三：利润空间（25%）
能不能赚钱？佣金率/毛利扣除运营成本后是否有盈余？

参考佣金率：
| 类型 | 佣金率区间 |
|------|-----------|
| 矿泉水/饮料 | 8-12% |
| 米面粮油 | 8-10% |
| 社区零售综合 | 10-15% |
| 家政/清洗服务 | 15-25% |
| 洗车 | 15-20% |
| 干洗 | 12-18% |
| 节礼/高客单 | 15-25% |
| 装修改造 | 差异化定价 |

### 维度四：供应链可控性（15%）
能不能稳定履约？破损/售后谁来处理？

关键问题：
- 供应商响应时效是否达标（48小时）？
- 是否有清晰的退换货政策？
- 配送是否覆盖目标小区？
- 是否需要特殊存储条件？

## 决策标准

| 总分 | 结论 | 行动 |
|------|------|------|
| 85+ | 优质品类 | 立即引进，重点发展 |
| 70-84 | 可引进 | 试点小区试水 |
| 60-69 | 暂缓 | 补充短板后再评审 |
| <60 | 不引进 | 放弃或彻底重构模式 |

## 品类举例：矿泉水 vs 家政年卡

| 维度 | 矿泉水 | 家政年卡 |
|------|--------|---------|
| 物业壁垒 | +5（前置仓物业配送） | +20（员工上门，物业背书） |
| 真实需求 | +20（日常刚需） | +15（频率低但客单高） |
| 利润空间 | +10（低毛利） | +25（高佣金/高客单） |
| 供应链可控性 | +10 | +15（服务标准化后可控） |
| **总分** | **~65 暂缓** | **~85 优质** |

> 结论：矿泉水适合作为引流品吸引流量，家政年卡才是真正的利润款。
