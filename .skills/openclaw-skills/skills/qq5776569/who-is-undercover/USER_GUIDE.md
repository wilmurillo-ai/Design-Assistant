# 谁是卧底 - 用户指南

## 游戏简介
"谁是卧底"是一款经典的社交推理游戏，玩家分为平民和卧底两个阵营。平民获得相同的词语，而卧底获得一个相关但不同的词语。通过轮流描述自己的词语，玩家们需要找出卧底，而卧底则要隐藏身份并混淆视听。

## 如何开始游戏

### 基本命令
```
/skill who-is-undercover start [玩家总数] [人类玩家数]
```

**参数说明：**
- `玩家总数`：4-10之间的数字（默认6人）
- `人类玩家数`：1-玩家总数之间的数字（默认1人）

**示例：**
- `/skill who-is-undercover start` - 开始6人游戏，1个人类玩家
- `/skill who-is-undercover start 8 2` - 开始8人游戏，2个人类玩家

### 游戏流程

#### 1. 角色分配阶段
游戏开始后，系统会自动分配角色：
- **平民**：获得相同的词语
- **卧底**：获得相关的不同词语

你会收到私密消息告知你的身份和词语。

#### 2. 描述阶段
每轮游戏中，所有玩家需要提交对自己词语的描述：
```
/skill who-is-undercover describe "你的描述内容"
```

**描述技巧：**
- 不要直接说出你的词语
- 描述要模糊但有信息量
- 卧底需要模仿平民的描述风格
- 平民要注意发现异常描述

#### 3. 投票阶段
所有描述提交后，进入投票阶段：
```
/skill who-is-undercover vote [玩家编号]
```

**投票策略：**
- 根据描述的合理性判断
- 注意过于模糊或过于具体的描述
- 观察玩家的行为模式

#### 4. 结果公布
得票最多的玩家被淘汰，游戏继续直到分出胜负。

## 查看游戏状态
随时可以查看当前游戏状态：
```
/skill who-is-undercover status
```

## 结束游戏
如果需要提前结束游戏：
```
/skill who-is-undercover end
```

## 胜利条件

### 平民胜利
- 所有卧底都被淘汰

### 卧底胜利
- 卧底人数大于或等于平民人数

## AI玩家行为说明
AI玩家会：
- 生成合理的词语描述
- 分析其他玩家的描述进行投票
- 模拟真实人类的推理过程

## 常见问题

**Q: 我可以和其他真实玩家一起玩吗？**
A: 目前版本主要支持单人对战AI，多人协作功能正在开发中。

**Q: 词语库可以自定义吗？**
A: 当前使用内置词语库，未来版本将支持自定义词语。

**Q: 游戏难度如何调整？**
A: 通过调整卧底数量（玩家总数决定）来控制难度。
