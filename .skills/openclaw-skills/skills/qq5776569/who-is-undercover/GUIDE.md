# Who is Undercover - ClawHub 用户指南

## 🎮 游戏简介
"谁是卧底"是一款经典的社交推理游戏，现在通过AI技术实现了完整的自动化版本。玩家需要通过描述词语来找出隐藏在平民中的卧底，或者作为卧底成功隐藏身份。

## 🚀 快速开始

### 1. 安装技能包
```bash
clawhub install who-is-undercover
```

### 2. 开始游戏
```bash
/skill who-is-undercover start [玩家数量] [人类玩家数]
```
- 玩家数量：4-10人（默认6人）
- 人类玩家数：1-总玩家数（默认1人）

### 3. 游戏流程
1. **角色分配**：系统自动分配平民和卧底身份
2. **描述阶段**：每个玩家提交对自己词语的描述
3. **投票阶段**：玩家投票选出怀疑的卧底
4. **淘汰阶段**：得票最多的玩家被淘汰
5. **重复**：继续进行直到决出胜负

## 💡 游戏技巧

### 平民策略
- 描述要具体但不要过于明显
- 观察其他玩家的描述模式
- 投票时考虑描述的合理性

### 卧底策略  
- 描述要模糊但合理
- 避免使用可能暴露身份的词汇
- 观察平民的描述风格并模仿

## 🛠️ 常见问题

### Q: 游戏支持多人同时参与吗？
A: 当前版本主要针对单个人类玩家与AI对手游戏。多人游戏支持正在开发中。

### Q: AI对手智能吗？
A: AI能够生成合理的描述和投票，但仍有改进空间。我们正在持续优化AI智能度。
