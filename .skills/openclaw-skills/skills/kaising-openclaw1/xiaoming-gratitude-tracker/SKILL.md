# Gratitude Tracker

感恩追踪工具，帮你培养积极心态。

## 功能

- ✅ 感恩日记
- ✅ 积极心态
- ✅ 幸福感提升
- ✅ 回顾功能
- ✅ 统计报告

## 使用

```bash
# 写感恩
clawhub gratitude add --text "今天天气很好"

# 心情记录
clawhub gratitude mood --rating 8

# 回顾
clawhub gratitude review --days 7

# 导出
clawhub gratitude export --format pdf
```
