# Blood Pressure Tracker

血压追踪工具，监测你的血压健康。

## 功能

- ✅ 血压记录
- ✅ 趋势分析
- ✅ 健康建议
- ✅ 异常提醒
- ✅ 统计报告

## 使用

```bash
# 记录血压
clawhub bp log --systolic 120 --diastolic 80

# 查看趋势
clawhub bp trend --days 30

# 健康建议
clawhub bp tips

# 异常提醒
clawhub bp alert --high 140 --low 90
```
