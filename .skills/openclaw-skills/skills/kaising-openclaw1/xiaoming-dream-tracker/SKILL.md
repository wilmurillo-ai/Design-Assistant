# Dream Tracker

梦境追踪工具，帮你探索潜意识世界。

## 功能

- ✅ 梦境记录
- ✅ 梦境解析
- ✅ 统计分析
- ✅ 标签分类
- ✅ 导出分享

## 使用

```bash
# 记录梦境
clawhub dream add --text "梦见飞翔"

# 添加标签
clawhub dream tag --id 1 --tags "飞行，快乐"

# 梦境解析
clawhub dream analyze --id 1

# 统计
clawhub dream stats --month 2026-04
```
