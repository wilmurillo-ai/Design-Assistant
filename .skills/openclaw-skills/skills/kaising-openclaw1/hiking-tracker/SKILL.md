# Hiking Tracker

登山追踪工具，记录你的登山旅程。

## 功能

- ✅ 路线记录
- ✅ 海拔分析
- ✅ 安全提醒
- ✅ 天气预警
- ✅ 统计报告

## 使用

```bash
# 记录登山
clawhub hike log --trail "华山" --distance 10 --elevation 1000

# 海拔分析
clawhub hike elevation --trail "华山"

# 安全提醒
clawhub hike safety --trail "华山"

# 天气预警
clawhub hike weather --trail "华山"
```
