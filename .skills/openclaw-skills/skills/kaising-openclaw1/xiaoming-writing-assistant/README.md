# AI Writing Assistant ✍️

AI 驱动的写作助手，让写作效率提升 10 倍。

## 安装

```bash
npx clawhub@latest install ai-writing-assistant
```

## 功能

| 功能 | 说明 | 示例 |
|------|------|------|
| 大纲生成 | 根据主题生成文章大纲 | `clawhub writing outline --topic "xxx"` |
| 段落扩写 | 将短句扩展成长文 | `clawhub writing expand --text "xxx"` |
| 内容润色 | 优化文字表达 | `clawhub writing polish --text "xxx"` |
| 标题优化 | 生成吸引人的标题 | `clawhub writing title --content "xxx"` |
