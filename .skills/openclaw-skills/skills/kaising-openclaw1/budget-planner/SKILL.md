# Budget Planner

预算规划工具，帮你合理花钱。

## 功能

- ✅ 收支预算
- ✅ 分类管理
- ✅ 超支提醒
- ✅ 月度报告
- ✅ 储蓄目标

## 使用

```bash
# 设置预算
clawhub budget set --category "dining" --amount 2000 --month "2026-04"

# 记录支出
clawhub budget expense --category "dining" --amount 100

# 查看进度
clawhub budget progress --month "2026-04"

# 超支提醒
clawhub budget alert --threshold 90
```
