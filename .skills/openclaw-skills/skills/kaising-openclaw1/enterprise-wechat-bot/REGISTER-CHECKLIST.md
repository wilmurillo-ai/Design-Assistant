## 📋 注册前准备

### 必需信息
- [ ] 手机号码 (接收验证码)
- [ ] 邮箱地址
- [ ] 企业名称 (可用个人名义)
- [ ] 管理员姓名
- [ ] 身份证信息 (企业认证需要)

### 可选信息
- [ ] 营业执照 (企业认证)
- [ ] 组织机构代码 (企业认证)

---

## 🚀 注册流程

### 步骤 1: 访问官网
- [ ] 网址：https://work.weixin.qq.com
- [ ] 点击"立即注册"

### 步骤 2: 选择类型
- [ ] 选择"企业创建"
- [ ] 个人使用可选"个人企业"

### 步骤 3: 填写信息
- [ ] 企业名称
- [ ] 管理员姓名
- [ ] 手机号
- [ ] 验证码

### 步骤 4: 完成注册
- [ ] 设置管理员密码
- [ ] 登录管理后台

### 步骤 5: 创建应用
- [ ] 管理后台 → 应用管理
- [ ] 创建自建应用
- [ ] 填写应用信息

### 步骤 6: 添加机器人
- [ ] 进入应用 → 机器人
- [ ] 添加机器人
- [ ] 获取 Webhook Key

## 📝 信息记录

创建 `.env` 文件保存认证信息:

```bash
# 企业微信配置
WECHAT_CORPID=
WECHAT_CORPSECRET=
WECHAT_AGENT_ID=
WECHAT_WEBHOOK_KEY=
```

**⚠️ 安全提示**:
- 不要将 `.env` 提交到 Git
- 不要泄露 Secret 和 Key
- 定期更换密钥
