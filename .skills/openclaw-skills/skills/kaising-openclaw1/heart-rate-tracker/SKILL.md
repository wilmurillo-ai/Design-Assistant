# Heart Rate Tracker

心率追踪工具，监测你的心脏健康。

## 功能

- ✅ 心率记录
- ✅ 运动心率
- ✅ 健康分析
- ✅ 异常提醒
- ✅ 统计报告

## 使用

```bash
# 记录心率
clawhub hr log --bpm 75

# 运动心率
clawhub hr exercise --type "running" --bpm 150

# 健康分析
clawhub hr analyze

# 异常提醒
clawhub hr alert --high 100 --low 60
```
