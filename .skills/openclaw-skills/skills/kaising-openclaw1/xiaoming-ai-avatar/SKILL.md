# AI Avatar Generator

AI 驱动的头像和壁纸生成工具。

## 功能

- ✅ 个性化头像生成
- ✅ 手机壁纸定制
- ✅ 多风格可选 (动漫/写实/抽象)
- ✅ 高清输出
- ✅ 批量生成

## 使用

```bash
# 生成头像
clawhub avatar generate --prompt "可爱猫咪" --type avatar

# 生成壁纸
clawhub avatar generate --prompt "星空夜景" --type wallpaper

# 批量生成
clawhub avatar batch --prompts ./prompts.txt --count 5
```
