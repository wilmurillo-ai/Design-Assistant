## 🚀 发布方式

### 方式 1: 命令行发布 (推荐)

```bash
# 1. 导航到技能目录
cd /home/kaising/.openclaw/workspace/skills/chinese-search/

# 2. 使用 clawhub 发布
npx clawhub@latest publish

# 3. 按提示操作
# - 登录/注册 ClawHub 账号
# - 确认技能信息
# - 设置定价 (免费/付费)
# - 提交审核
```

