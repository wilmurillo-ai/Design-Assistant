## ✨ 核心功能

### MVP v1.0 包含

#### ✅ 必应中国搜索
- 国际 + 中文内容
- 技术文档友好
- 支持高级搜索语法

#### ✅ 搜狗微信搜索
- 微信公众号文章搜索
- **全网唯一**可用工具
- 自媒体内容必备

#### ✅ 高级搜索语法
- `site:` 限定网站
- `filetype:` 文件类型
- `intitle:` 标题搜索
- `""` 精确匹配
