# Batch Search Monitor 🔍

批量搜索监控工具，自动跟踪关键词变化。

## 安装

```bash
npx clawhub@latest install batch-search-monitor
```

## 使用场景

- 📊 竞品监控
- 📰 品牌提及跟踪
- 🔔 舆情告警
- 📈 市场调研
- 📑 数据收集

## 命令

| 命令 | 说明 |
|------|------|
| `monitor add` | 添加监控任务 |
| `monitor list` | 查看所有任务 |
| `monitor export` | 导出结果 |
| `monitor remove` | 删除任务 |
