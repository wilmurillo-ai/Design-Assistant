# Cycling Tracker Pro

骑行追踪专业版工具，帮你记录骑行旅程。

## 功能

- ✅ 骑行记录
- ✅ 速度分析
- ✅ 路线规划
- ✅ 训练计划
- ✅ 统计报告

## 使用

```bash
# 记录骑行
clawhub cycle log --distance 20 --time 60

# 速度分析
clawhub cycle speed --distance 20 --time 60

# 路线规划
clawhub cycle route --distance 30 --type "scenic"

# 查看统计
clawhub cycle stats --month 2026-04
```
