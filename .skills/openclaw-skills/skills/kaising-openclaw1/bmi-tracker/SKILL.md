# BMI Tracker

BMI 追踪工具，帮你管理健康体重。

## 功能

- ✅ BMI 计算
- ✅ 健康评估
- ✅ 改善建议
- ✅ 历史记录
- ✅ 目标设定

## 使用

```bash
# 计算 BMI
clawhub bmi calc --height 175 --weight 70

# 健康评估
clawhub bmi assess --bmi 22.8

# 查看历史
clawhub bmi history

# 目标设定
clawhub bmi goal --target 22
```
