# Mood Tracker

情绪追踪工具，帮你了解和管理情绪。

## 功能

- ✅ 情绪记录
- ✅ 模式分析
- ✅ 改善建议
- ✅ 触发因素
- ✅ 统计报告

## 使用

```bash
# 记录情绪
clawhub mood log --mood "happy" --note "完成了一个项目"

# 模式分析
clawhub mood pattern --days 30

# 改善建议
clawhub mood tips

# 触发因素
clawhub mood trigger --mood "anxious"
```
