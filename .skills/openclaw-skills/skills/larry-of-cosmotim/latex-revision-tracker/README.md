# LaTeX Revision Tracker

A comprehensive workflow skill for managing revisions to large LaTeX academic papers.

## What It Does

Provides systematic guidance for:

- **Version control** — file naming, change documentation, diff tracking
- **Content integration** — inserting new sections, figures, and citations into existing manuscripts
- **LaTeX compilation** — error handling, unicode issues, package conflicts
- **Bibliography management** — citation consistency, BibTeX workflows
- **Quality assurance** — pre-submission checklists, cross-reference validation
- **Collaboration** — multi-author workflows, reviewer response tracking

## Who It's For

- Graduate students managing thesis/dissertation revisions
- Researchers preparing journal submissions with multiple revision rounds
- Collaborative writing teams on multi-author papers
- Anyone working with LaTeX documents over 10 pages

## Installation

### ClawHub
```bash
clawhub install latex-revision-tracker
```

### Manual
Copy `SKILL.md` into your agent's `skills/latex-revision-tracker/` directory.

## Usage

Ask your agent to help with LaTeX revision tasks:
- "Help me integrate this new section into my paper"
- "Track the changes between v2 and v3 of my manuscript"
- "Run through the pre-submission checklist"
- "Set up version control for my thesis"
