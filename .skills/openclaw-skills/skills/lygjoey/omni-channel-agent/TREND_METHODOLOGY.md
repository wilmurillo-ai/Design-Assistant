# 社媒趋势识别方法论

> 来源: Lark 文档《Identifying Global Social Media Trends for AI Agent Development》

## 为什么做这件事

开发视觉AI Agent（滤镜/换装/舞蹈/角色扮演）需要紧跟全球社媒趋势。我们的方法**优先依赖人类直觉**而非纯数据工具，因为：

1. **滞后问题** — 数据工具依赖大量历史数据触发"趋势"警报，等到数据看板上显示时趋势已过峰值
2. **无法理解"氛围"** — 工具能告诉你音频增长率500%，但无法告诉你视频里人们在做什么
3. **垃圾数据** — 热门仪表板充斥无关内容（政治新闻、随机喜剧、机器人垃圾）
4. **利基社区** — 最好的AI滤镜来自创意亚文化（动漫粉、地下舞者、特效化妆师），这些利基数据量不够上数据看板

## Phase 1: 环境设置（克服地理围栏）

- **删除本地SIM卡** — TikTok用SIM确定位置，使用无SIM专用研究手机或目标市场国际SIM（如US SIM）
- **网络环境** — 使用可靠专用IP/代理连接目标地区Wi-Fi，避免频繁切换地区（触发安全过滤）
- **设备设置** — 系统语言改为英语（或目标语言），时区匹配代理IP
- **创建账号** — 使用目标地区网络环境注册新账户

## Phase 2: 趋势发现与跟踪方法论

### 1. 整理"趋势指标"观察列表
- 不只关注最大名人，**优先粉丝10k-100k的创作者**（更敏锐、行动更快）
- 持续更新列表，删除不活跃账号

### 2. 音频驱动的趋势跟踪
- **音乐和音频决定短视频趋势**，病毒式声音是舞蹈/角色扮演过渡的支柱
- 发现趋势后→点击视频底部音频图标→查看使用该音频的所有视频
- 如果数百个最近视频使用相同舞蹈/视觉效果 = 已验证趋势

### 3. "关注列表"逆向工程
- 查看早期采用者**正在关注谁** → 发现地下创作者（趋势真正来源）

## Phase 3: 算法训练（自动化发现）

训练平台推荐算法为你工作：
- **完整观看** — 角色扮演/舞蹈/视觉效果视频全程播放或循环2次
- **互动** — 点赞、保存、偶尔分享评论
- **忽略无关内容** — 本地新闻/随机喜剧立即滑过
- **结果** — 3-5天有序互动后，Feed成为高度定制的新兴趋势流

## Phase 4: 趋势评估（是否适合AI？）

**核心三问：**

1. ✅ **视觉上有区别吗？** — 是否有特定服装/独特妆容/高辨识度舞蹈？日常内容 = 不适合
2. ✅ **AI可以访问吗？** — 技术上能否轻松复制？面部/身体替换是否会严重扭曲？
3. ✅ **引发虚荣心吗？** — 用户使用后看起来更酷/漂亮/性感/有趣？答案是 → 用户会分享

## Phase 5: 数据管理与可操作输出

- **核心名单** — 每月更新"趋势指标"账户电子表格，删除不活跃+添加新发现
- **趋势记录** — 移交经过验证趋势时记录：
  1. 原始/最佳示例视频链接
  2. 趋势音频链接
  3. 视觉需求简短描述（如："用户需要在节拍下降时穿上赛博朋克服装"）
