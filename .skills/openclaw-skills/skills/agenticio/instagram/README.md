# Instagram — The Attention Sculptor (v2.1.1)

> Visuals stop the scroll. Architecture earns the action.

Instagram 2.1.1 is a **local-first content system** for hooks, captions, content structure, and optional draft storage.

It combines:
- **Attention Sculptor logic** for stronger hooks, retention, and CTA flow
- **Local-first safety** with no API connection, no auto-posting, and no automation
- **Optional persistence** through a simple local draft-saving script

## What it is for

This skill helps with two types of work:

### 1. Content Optimization
Use it to improve one specific piece of content:
- Reel hook
- Carousel opening
- Caption
- CTA
- Story bridge

### 2. Account Strategy
Use it to shape the system behind the account:
- funnel position
- content pillars
- identity consistency
- CTA route design

## Local-first workflow

- Generate or optimize content in chat
- Review the rewritten version
- Optionally save it locally with `scripts/write_caption.py`
- Publish manually

## Boundaries

This skill does **not**:
- auto-post
- connect to Instagram
- bulk-send DMs
- guarantee virality
- function as a full analytics dashboard

It provides strategy, structure, and optional local draft storage.
