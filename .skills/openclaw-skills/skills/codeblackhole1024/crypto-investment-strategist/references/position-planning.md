# Position Planning

Translate analysis into execution.

## Staged entry model

Use a staged entry instead of one all-in order whenever possible.

- **Entry 1**: initial position at acceptable value zone
- **Entry 2**: add only on support retest, successful breakout retest, or planned pullback
- **Entry 3**: optional confirmation add after thesis strengthens

## Position sizing rules

- High confidence spot setup: moderate size
- Medium confidence setup: small starter position
- Low confidence setup: wait or very small probe
- High volatility: cut size first, do not widen risk casually

## Invalidation

Every plan needs a clear invalidation level.

Examples:
- Loss of key support
- Lower low after a supposed bullish reversal
- Breakdown below range floor
- Failed breakout that closes back into range

## Take-profit ladder

Prefer multiple targets:
- TP1: de-risk
- TP2: scale out into strength
- TP3: leave runner only if trend stays healthy

## Practical rule

If you cannot define entry, invalidation, and size in one minute, the plan is not ready.
