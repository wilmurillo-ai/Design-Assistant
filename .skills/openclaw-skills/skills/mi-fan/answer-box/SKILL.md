# Answer Box · 答案盒子
**默念问题，一键获取专属温暖正向建议**
一款专为日常困惑、情绪内耗、轻量决策设计的治愈系Skill，无需复杂操作，心中想好问题调用技能，即可随机获得一句贴合场景的正向回复，温柔化解纠结、舒缓焦虑。

## 🌟 核心特性
✅ **6大分类精准适配**：覆盖「肯定鼓励、理性谨慎、行动推动、心态调节、轻松幽默、中性指引」，贴合日常90%以上的使用场景
✅ **纯正向无负面**：所有回复均为暖心设计，拒绝消极导向，兼顾理性建议与情绪安慰
✅ **极简操作**：无需输入/配置，默念问题后调用技能即可，一秒获取答案
✅ **无重复高实用**：60条原创回复，重复调用低概率撞款，适配多次使用需求

## 🎯 适用场景
### 日常个人使用
1. **职场解压**：加班焦虑、工作纠结时，默念「如何调整状态」获安慰
2. **轻量决策**：纠结「要不要尝试新方案」「是否开始学新技能」获指引
3. **情绪舒缓**：emo、内耗时，默念「怎么缓解不开心」获暖心回复
4. **日常鼓励**：迷茫、没动力时，随手调用获正向心理暗示

### 团队/培训使用
1. **培训演示**：作为正念/情绪管理培训的轻量互动工具，提升参与感
2. **团队破冰**：团建/小组活动时使用，化解尴尬，增加轻松氛围
3. **办公小工具**：团队成员纠结小问题时，用它获得随机正向建议

## 🚀 技能使用方式
### 基础调用（平台通用）
1. 找一个安静的环境，**心中清晰默念你的问题**（如：如何缓解焦虑？要不要接受新机会？）
2. 在Skill平台**语音/文字调用「answer-box」或「答案盒子」**
3. 系统自动返回**1条随机定制化暖心建议**，重复调用可获取不同答案

### 适配说明
- 支持重复无限次调用，每次随机匹配60条回复中的1条
- 无需输入问题，纯默念即可，适配所有需要轻量建议的场景
- 无设备/环境限制，平台支持的场景均可调用

## 📝 技能说明
1. 本技能所有回复均为原创正向内容，无任何消极/负面导向
2. 回复仅作**轻量参考/情绪安慰**，不构成专业决策建议
