# 卡路里计算参考

## 运动消耗计算公式

### 基础公式

```
卡路里 = MET × 体重(kg) × 时间(小时)
```

### 常见运动的MET值

| 运动项目 | MET值 | 说明 |
|----------|-------|------|
| 静坐 | 1.0 | 基础代谢 |
| 散步（慢走） | 2.5 | <5km/h |
| 快走 | 4.5 | 5-6km/h |
| 慢跑 | 7.0 | 8km/h |
| 快跑 | 10.0 | >10km/h |
| 游泳（休闲） | 6.0 | 自由泳/蛙泳 |
| 骑自行车（休闲） | 4.0 | <16km/h |
| 跳绳 | 8.0-12.0 | 视速度 |
| 瑜伽 | 3.0 | 温和瑜伽 |
| 力量训练 | 5.0 | 器械训练 |

## 步数消耗估算

### 每步消耗

```
卡路里/步 = 0.04 ~ 0.06
```

### 估算公式

```python
基础消耗 = 步数 × 0.04  # 日常步行
快走消耗 = 步数 × 0.06  # 快走
```

## 有效减脂心率

### 计算公式

```
目标心率 = (220 - 年龄) × 60% ~ 70%
```

### 示例（34岁）

```
最小： (220 - 34) × 0.60 = 112 次/分
最大： (220 - 34) × 0.70 = 130 次/分
目标区间：112 ~ 130 次/分
```

## 减脂建议

1. **每天消耗300-500卡** - 每周可减0.5-1斤
2. **优先有氧运动** - 脂肪供能比例最高
3. **保持心率在减脂区** - 效果最佳
4. **不要过度节食** - 基础代谢会下降
