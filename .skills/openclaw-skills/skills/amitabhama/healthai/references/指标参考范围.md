# 健康指标参考范围

## 🏥 肝功能指标

| 指标 | 参考范围 | 异常提示 |
|------|----------|----------|
| ALT（谷丙转氨酶） | 9-50 U/L | >50：肝细胞损伤，常见于脂肪肝/肝炎 |
| AST（谷草转氨酶） | 15-40 U/L | 与ALT联合判断 |
| AST/ALT比值 | 1.0左右 | <1：提示非酒精性脂肪肝；>2：提示酒精性肝病 |
| GGT（谷氨酰转肽酶） | 10-60 U/L | 升高：酒精肝/胆道疾病 |
| ALP（碱性磷酸酶） | 45-125 U/L | 升高：胆道梗阻/骨骼疾病 |

### 脂肪肝风险评估

| ALT水平 | AST/ALT | 可能情况 |
|---------|---------|----------|
| 轻度升高（50-100） | <1 | 非酒精性脂肪肝（NAFLD） |
| 中度升高（100-300） | <1 | 非酒精性脂肪性肝炎（NASH） |
| 升高 | >2 | 酒精性肝病 |
| 显著升高（>300） | >1 | 急性肝炎或其他严重肝病 |

## ❤️ 心血管指标

| 指标 | 参考范围 | 异常提示 |
|------|----------|----------|
| 血压-收缩压 | 90-140 mmHg | ≥140为高血压 |
| 血压-舒张压 | 60-90 mmHg | ≥90为高血压 |
| 总胆固醇（TC） | <5.2 mmol/L | ≥6.2为高胆固醇 |
| 甘油三酯（TG） | <1.7 mmol/L | ≥2.3为高血脂 |

### 高血压分级

| 分类 | 收缩压 | 舒张压 |
|------|--------|--------|
| 正常 | <120 | <80 |
| 正常高值 | 120-139 | 80-89 |
| 高血压 | ≥140 | ≥90 |

## 🩸 血糖指标

| 指标 | 参考范围 | 异常提示 |
|------|----------|----------|
| 空腹血糖 | 3.9-6.1 mmol/L | >6.1：糖尿病前期；>7.0：糖尿病 |
| 糖化血红蛋白（HbA1c） | <5.7% | 5.7%-6.4%：糖尿病前期；≥6.5%：糖尿病 |
| 餐后2h血糖 | <7.8 mmol/L | >11.1：糖尿病 |

### 糖尿病前期

| 状态 | 空腹血糖 | 餐后2h血糖 |
|------|----------|------------|
| 正常 | <6.1 | <7.8 |
| 空腹血糖受损 | 6.1-7.0 | <7.8 |
| 糖耐量异常 | <7.0 | 7.8-11.1 |
| 糖尿病 | ≥7.0 | ≥11.1 |

## 🩺 血常规指标

| 指标 | 参考范围 | 异常提示 |
|------|----------|----------|
| 白细胞（WBC） | 4-10 ×10⁹/L | 升高：感染/炎症；降低：免疫力下降 |
| 红细胞（RBC） | 4-5.5 ×10¹²/L | 升高：脱水/高原反应；降低：贫血 |
| 血红蛋白（Hb） | 120-170 g/L | 降低：贫血 |
| 血小板（PLT） | 100-300 ×10⁹/L | 降低：出血风险；升高：血栓风险 |
| MCV（平均红细胞体积） | 80-100 fl | <80：小细胞性贫血；>100：大细胞性贫血 |

### 贫血类型（按MCV）

| 类型 | MCV | 常见原因 |
|------|-----|----------|
| 小细胞性 | <80 fl | 缺铁性贫血 |
| 正细胞性 | 80-100 fl | 慢性病贫血、失血性贫血 |
| 大细胞性 | >100 fl | 叶酸/B12缺乏、骨髓增生异常 |

## 📋 体检报告解读原则

1. **综合评估**：单一指标异常意义有限，需结合多项指标综合判断
2. **动态观察**：指标变化趋势比单次数值更有参考价值
3. **个体差异**：参考范围基于大部分人群，个体最佳值可能不同
4. **生活习惯**：饮食、运动、饮酒、熬夜等都会影响指标
5. **复查确认**：初次异常建议复查排除暂时性因素

## 🚨 需要立即就医的情况

- 血压持续 ≥180/120 mmHg
- 空腹血糖 ≥16.7 mmol/L
- ALT/AST ≥300 U/L
- 胸痛、呼吸困难、剧烈头痛
- 任何指标的急剧变化
