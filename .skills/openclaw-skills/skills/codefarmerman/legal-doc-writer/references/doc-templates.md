## 一、税务筹划报告框架

```
【×××税务筹划报告】

委托方：
项目名称：
报告日期：
编制人：
密级：[保密]

一、项目概述
  1.1 背景
  1.2 筹划目标
  1.3 本报告说明

二、现行税务状况分析
  2.1 业务结构及交易描述
  2.2 现行纳税情况
  2.3 存在的税务问题

三、筹划方案
  3.1 方案一：[名称]
    - 方案内容
    - 法律依据
    - 预计税务效果
    - 风险评估
  3.2 方案二：[名称]（如有）
    ...
  3.3 方案比较与推荐

四、实施建议
  4.1 实施步骤
  4.2 配套文件
  4.3 注意事项

五、风险提示
  5.1 政策变化风险
  5.2 执行层面风险
  5.3 其他风险

六、声明
本报告依据委托方提供的资料及截至报告日有效的法律法规出具，
仅供委托方参考使用，不构成正式法律意见。
```

```

  4.1 [问题一分析]
    - 事实认定
    - 法律适用
    - 结论
  4.2 [问题二分析]
    ...

  5.1 主要结论
  5.2 具体建议

[日期]
```

## 五、合同税务条款示例

### 税费承担条款
```
第×条 税费承担

一、本合同项下因交易产生的所有税费，由各方依照中国现行法律法规的
    规定各自承担其应缴纳的税费。

二、[卖方/甲方]应缴纳的税费包括但不限于：增值税及附加、所得税等。

三、[买方/乙方]应缴纳的税费包括但不限于：契税、印花税等。

四、如因国家税收政策调整导致税负变化，双方应就税费承担问题另行
    协商。
```

### 发票条款
```
第×条 发票

一、[供应方]应于收到货款/服务费后×个工作日内，向[采购方]开具
    符合税法规定的增值税[专用/普通]发票。

二、发票信息如下：
    名称：
    税号：
    地址电话：
    开户行及账号：

三、如因[供应方]原因导致发票开具不符合规定，由此造成[采购方]
    无法抵扣进项税额的损失，由[供应方]承担赔偿责任。
```
