# 中国税务计算工具

## 核心功能

提供中国主要税种的精确计算，支持参数调整和多方案对比，为税务筹划提供数据支撑。

## 使用方式

运行 `scripts/tax_calc.py` 进行交互式计算，或直接描述计算需求由助手按公式计算。

## 计算模块

### 1. 增值税计算

**一般纳税人**
- 应纳增值税 = 销项税额 - 进项税额
- 销项税额 = 不含税销售额 × 适用税率
- 含税价换算：不含税价 = 含税价 ÷ (1 + 税率)

**小规模纳税人**
- 应纳增值税 = 不含税销售额 × 征收率

**附加税费**
- 城建税 = 应纳增值税 × 7%(市区)/5%(县镇)/1%(农村)
- 教育费附加 = 应纳增值税 × 3%
- 地方教育附加 = 应纳增值税 × 2%

### 2. 企业所得税计算

**基本公式**
- 应纳税所得额 = 收入总额 - 不征税收入 - 免税收入 - 各项扣除 - 弥补亏损
- 应纳所得税 = 应纳税所得额 × 适用税率 - 减免税额 - 抵免税额

**常用优惠计算**
- 高新技术企业税收优惠 = 应纳税所得额 × (25%-15%) = 节税额
- 研发加计扣除节税 = 研发费用 × 100% × 适用税率
- 小微企业税收优惠（≤300万）= 应纳税所得额 × (25%-5%)

### 3. 个人所得税计算

**综合所得（工资薪金为主）**
- 全年应纳税所得额 = 全年综合所得 - 60000元 - 专项扣除 - 专项附加扣除 - 其他扣除
- 应纳税额 = 全年应纳税所得额 × 适用税率 - 速算扣除数

**月度预扣预缴（工资薪金）**
- 累计预扣预缴应纳税所得额 = 累计收入 - 累计免税收入 - 累计减除费用(5000×月数) - 累计专项扣除 - 累计专项附加扣除
- 本月预扣税额 = 累计应纳税额 - 已预扣税额

**股权转让**
- 应纳税所得额 = 股权转让收入 - 股权原值 - 合理费用
- 应纳税额 = 应纳税所得额 × 20%

### 4. 土地增值税计算

**四步法**
1. 计算收入额（含价外费用）
2. 计算扣除项目金额（取得土地成本+开发成本+开发费用+税金+加计扣除）
3. 计算增值额 = 收入额 - 扣除项目金额
4. 计算增值率 = 增值额 ÷ 扣除项目金额
5. 按增值率确定税率，计算税额

**加计扣除（房地产开发）**
- 加计扣除 = (取得土地使用权支付金额 + 房地产开发成本) × 20%

### 5. 印花税计算

- 应纳税额 = 计税依据 × 适用税率

```

方案一   | 企业所得税 | XXX万 | 25% | XXX万 | | 
方案二   | ... | ... | ... | ... | XXX万 | X.X%

节税效果：方案二相比方案一节税 XXX万元（降低税负 X.X%）
```

## 计算注意事项

- 所有计算结果仅供参考，实际税负受具体业务细节影响
- 税率以 `china-tax-law` skill 中的最新税率表为准
- 重大交易的税务计算建议与主管税务机关确认
- 计算结果可导出至Excel进行详细分析（使用 `xlsx` skill）
