# 税务合规检查清单

## 一、企业日常税务合规

### 增值税合规
- [ ] 按时申报（月报/季报）
- [ ] 进项发票认证及时（360天内）
- [ ] 销项发票开具规范（内容完整、金额准确）
- [ ] 专用发票管理（防止丢失、作废规范）
- [ ] 出口退税申报（90天内）
- [ ] 留抵税额合理处理
- [ ] 关联交易定价符合独立交易原则

### 企业所得税合规
- [ ] 季度预缴（1月/4月/7月/10月各月15日前）
- [ ] 汇算清缴（次年5月31日前）
- [ ] 研发费用加计扣除材料归集
- [ ] 高新技术企业资质维护（每3年认定）
- [ ] 关联交易同期资料准备（次年6月30日前）
- [ ] 境外所得申报
- [ ] 资产损失申报备案

### 个人所得税合规
- [ ] 代扣代缴申报（次月15日前）
- [ ] 年度综合所得汇算清缴（次年3-6月）
- [ ] 股权激励、分红所申报规范
- [ ] 高管、外籍员工特殊处理

## 二、重大交易税务尽职调查清单

### 股权转让
- [ ] 确认转让方纳税义务（居民/非居民）
- [ ] 评估股权计税基础
- [ ] 核查是否适用特殊税务处理（重组）
- [ ] 关注间接转让反避税规则（698号文）
- [ ] 审查历史分红与留存收益
- [ ] 确认印花税处理

### 资产收购
- [ ] 区分一般性/特殊性税务处理
- [ ] 评估各类资产计税基础
- [ ] 核查或有税务负债
- [ ] 土地增值税清算情况
- [ ] 增值税进项留抵处理

### 企业重组
- [ ] 判断是否满足特殊性税务处理条件
- [ ] 股权支付比例（≥85%）
- [ ] 12个月不改变业务连续性要求
- [ ] 报备手续完整性

## 三、税务风险提示要点

### 高风险领域
1. **虚开发票**：刑事风险，重点核查
2. **关联交易定价**：转让定价调整风险
3. **股权架构设计**：防止被认定为滥用安排
4. **境外架构**：受控外国公司（CFC）规则、一般反避税（GAAR）
5. **税收协定滥用**："受益所有人"认定
6. **房地产开发**：土地增值税清算时机

### 常见税务争议类型
- 计税依据争议
- 税收优惠认定争议
- 纳税义务发生时间争议
- 关联交易定价争议
- 非居民企业税收争议

## 四、重要申报时限汇总

| 税种 | 申报期限 |
|-----|---------|
| 增值税（月报） | 次月15日 |
| 增值税（季报） | 季度末后15日 |
| 企业所得税预缴 | 季度末后15日 |
| 企业所得税汇算清缴 | 次年5月31日 |
| 个人所得税代扣代缴 | 次月15日 |
| 个人所得税汇算清缴 | 次年3月1日-6月30日 |
| 印花税 | 合同签订当月 |
| 契税 | 办理权属登记前 |
| 土地增值税 | 项目清算后 |
| 关联交易同期资料 | 次年6月30日前 |
