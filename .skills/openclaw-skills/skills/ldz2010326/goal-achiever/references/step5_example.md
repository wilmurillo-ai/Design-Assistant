# 标准示例

## 场景
核心目标：完成一次知乎回答类型的发布。
当前已完成任务：
- 选题与问题池筛选（done）
- 信息搜集与观点结构化（done）
- 撰写高质量回答（done）
- 优化发布与互动引导（done）

## 核心目标校验结论
仍缺少“实际发布动作/发布后互动跟进”相关任务，无法确保完成“发布”目标。

## 欠缺任务（需回补到 Step2 任务拆解列表）
新增任务：
1. task: 发布知乎回答
   - task_description: 在知乎完成回答发布动作，确认发布成功。
   - task_reason: 未发布则核心目标无法达成。
   - task_state: pending
   - task_result: null

2. task: 发布后互动跟进
   - task_description: 发布后2小时内检查评论并进行至少1次互动回复。
   - task_reason: 互动是高质量发布目标的一部分。
   - task_state: pending
   - task_result: null

## 回到 Step2 继续运行
将上述欠缺任务追加到任务 JSON，并重新进入 Step2→Step3 执行流程。
