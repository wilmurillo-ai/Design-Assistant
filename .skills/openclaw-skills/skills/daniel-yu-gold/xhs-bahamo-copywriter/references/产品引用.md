# 产品引用规则

## 核心原则

产品引用字段**直接使用**产品更新文档里的产品名称，**不要添加任何额外文字或说明**。

## 判断流程

```
生成器是否有产品更新文档？
├── 有 → 文案中推广产品，排期表"引用产品"字段填写产品名称
└── 没有 → 排期表"引用产品"字段留空
```

## 示例

### 正确示例

**产品文档中的名称**: `奥体万科A座 1803室`

**文案排期表填写**: `奥体万科A座 1803室`

### 错误示例

❌ `奥体万科A座-1803-直播` （添加了额外描述）
❌ `奥体万科A座 1803室（直播专用）` （添加了括号说明）
❌ 空着（有产品但没填）

## 常见产品名称格式

根据实际产品更新文档，产品名称通常为：
- `华业发展中心 XXX室`
- `奥体万科A座 XXX室`

直接使用文档中的原始名称即可。
