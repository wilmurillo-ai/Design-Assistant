# 小红书爆款文案生成技能

## 核心功能

- 根据项目信息，生成7天×N个账号的文案排期表
- 随机调用框架生成多样化爆款文案
- 自动写入飞书多维表格（文案排期表）

## 工作流程

### 1. 定位项目

根据用户输入的项目名称，前往飞书运营项目文件夹找到对应项目目录。

**运营项目文件夹**: `DykhftKeCliY4cdbj2Ccxa5xnCb`

### 2. 读取项目资源

在项目目录下找到以下内容：

| 资源 | 说明 | 位置 |
|------|------|------|
| 生成器文件夹 | 包含框架和关键词 | 项目根目录下 |
| 爆款框架 | 标题模板、内容模板、框架指引 | 生成器/爆款框架 |
| 关键词 | 随机变换的关键词 | 生成器/关键词 |
| 产品更新 | 需要推广的产品（如有） | 生成器/产品更新 |
| 文案排期表 | 写入生成的文案 | 项目根目录下的多维表格 |

### 3. 生成文案

**文案命名格式**: `账号名-框架类型-发布日期-周几`

**内容要求**:
- 参考框架，尤其是标题和封面要尽量一致
- 结合关键词文档进行提取和组合
- **禁止出现小红书敏感词**（详见 references/敏感词.md）

**产品引用规则**:
- 生成器**包含**产品更新文档 → 文案中推广产品，排期表填写产品名称
- 生成器**没有**产品更新文档 → 排期表"引用产品"字段**留空**
- 产品名称直接使用文档中的名称，**不添加额外文字**

**违禁词检查**:
- 价格、多少钱、费用、报价、租金、月租、年租
- 元/㎡、元/月、每平方
- 投资回报、升值、稳赚、保本、零风险

### 4. 写入排期表

将生成的文案批量写入对应项目的文案排期表多维表格。

**排序规则（写入顺序至关重要）**:
- 写入顺序：按发布日期**倒序**写入（最新日期在前、最旧日期在后）
- 例如：日期范围是 4/8→4/20，则写入顺序为 4/20→4/19→…→4/8
- 这样表格打开后，最新日期自动出现在最上方，符合阅读习惯
- 同一日期内：账号01→账号05 顺序写入
- ⚠️ 常见错误：按日期正序（4/8→4/20）写入，导致表格打开后最新日期被挤在最下方

**⚠️ 冲突检查规则（重要）**:
- 写入前先查询排期表，筛选「发布日期+账号」与待写入记录冲突的已有记录
- 冲突记录处理：先删除冲突记录，再写入新记录
- 绝不跳过冲突检查，禁止出现同一账号同一日期重复写入的情况
- 此规则适用于所有文案排期表写入操作，无需用户特别提醒

## 封面生成工作流

### Step 1：分析框架类型

读取对应框架文档，理解该框架的视觉调性：

| 框架 | 封面调性 | 色彩 | 氛围 |
|------|---------|------|------|
| 警告避坑型 | 揭露问题+给出方案，对比强烈 | 红色警示+白色，留意字 | 严肃但有温度 |
| 活动招募型 | 邀请感强，突出现场感 | 黄色为主（呼应素材主色）+白色 | 温暖活力 |
| 社群展示型 | 展示群体氛围，传递归属感 | 明黄+白色，温暖活泼 | 温馨有感染力 |
| 现场回顾型 | 回顾活动精彩瞬间 | 黄色+绿色点缀 | 专业+温暖 |

### Step 2：生成四宫格封面

使用 `image_generate` 工具生成封面。

**四宫格布局**：2×2拼图，4个格子各有内容，底部可加框架色块标题栏。

**封面生成 Prompt 公式**：

```
小红书爆款文案封面四宫格，主题：[框架名称]，[主题描述]。

布局：2列×2行四宫格拼图，白色分隔线。
风格：现代简约，小红书封面比例（3:4），文字清晰。

第一格：[格1内容，对应素材特点，如：专业讲师在讲台讲解PPT场景]
第二格：[格2内容，如：女性社群活动，十余人举杯共饮，温馨氛围]
第三格：[格3内容，如：精致活动现场，明黄色主题布置]
第四格：[格4内容，如：女性手持麦克风分享，干练自信]

底部标题栏：[框架主题色背景，白色文字标题，字号大]

防查重要求：对素材场景进行微调，如改变背景色温、调整人物姿态、改变室内陈设细节，避免与原图完全一致。
```

**示例 Prompt（警告避坑型）**：
```
小红书爆款文案封面四宫格，主题：警告避坑型，别再乱选！杭州靠谱小红书获客沙龙长这样。

布局：2列×2行四宫格拼图，白色分隔线，3:4比例。
风格：现代简约，现代感，警示氛围但不失温度。

第一格：专业讲师在讲台讲解PPT场景，讲台明亮，听众专注
第二格：精致活动现场布置，明黄色主题，长桌配有鲜花装饰
第三格：女性手持麦克风在活动现场分享，神态自信，内容充实
第四格：活动现场温馨氛围，参与者认真聆听，黄色主题布置

底部红色标题栏：白色大字"别乱选！"，副标题"杭州靠谱沙龙避雷指南"
对素材场景进行微调：调整灯光色温、改变人物相对位置、微调室内陈设，避免查重。
```

### Step 3：保存封面

- 封面图保存到 `/tmp/xhs-cover/` 目录
- 文件命名：`[框架名]-[日期]-[版本].jpg`，如 `警告避坑型-2026-04-20-v1.jpg`
- 提示用户下载或上传至飞书

## 触发方式

封面生成作为文案生成的配套环节自动进行：
- 用户说「生成文案+封面」→ 同时生成文案和封面
- 用户说「只生成封面」→ 仅生成封面（需指定框架和主题）
- 用户说「生成某框架封面」→ 按指定框架生成封面
