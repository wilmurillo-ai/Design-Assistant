# 打光生图器（Relight Image Generator）

## 定义

在不改变主体内容的前提下，重塑图像的光线表现（方向 / 强度 / 色温 / 阴影）。

属于**图像后处理模块**，通常位于生图之后、视频生成之前。

## 输入 / 输出

- **输入**：Image（单张，必选）
- **输出**：Image（光照调整后）

## 适用场景

- 调整光源方向（左/右/上/下）
- 调整光照强度与对比度
- 调整色温（冷光/暖光）
- 视觉氛围调整（日光→夜晚、柔光→硬光）

## 编排规则

```text
已有图像 + 需要改光 → 使用该模块
需要改主体内容 → 不使用
```

## 关键原则

- 不改变主体结构、构图布局、物体内容，只改光照
- 仅支持单图输入，不适用于多图融合或分镜列表
- 可叠加使用（Image → Relight → Relight → Image）

## 常见错误

- ❌ 用于生成新画面 → 该模块不改变主体内容
- ❌ 期待改变人物/场景 → 只能调整光照
- ❌ 与生图模块混用 → 本模块是后处理，不负责生成
