# 视频修改生视频器（Video Modify Generator）

## 定义

在保留原视频结构与时间轴的前提下，对内容进行局部或整体修改（主体 / 风格 / 细节）。本质是"编辑"而非"生成"。

## 输入 / 输出

- **输入**：Video（原始视频，必选）+ Image（主体参考，可选）+ Image（风格参考，可选）+ Text（修改指令，可选）
- **输出**：Video（修改后）

## 适用场景

- 人物外观修改 / 风格迁移（视频级）
- 主体替换（保持动作一致）
- 局部修改（Shot-level 微调）

## 核心机制

```text
原视频（结构 / 节奏 / 时长）+ 修改指令 → 新视频
```

三个保持原则：
- **时间轴保持**：视频长度和镜头节奏不变
- **动作保持**：原有动作轨迹保留
- **内容替换**：主体 / 风格 / 元素可修改

## 编排规则

```text
已有视频 + 需要改内容 → 使用该模块
无视频 → 不使用（需先生成视频）
需要结构变化（拆分镜 / 拼接） → 使用分镜模块
```

## 关键原则

- Video 是绝对主输入，决定结构/节奏/时长；其他输入只做覆盖或修改
- 不会改变视频结构，不拆分镜，不拼接镜头

## 常见错误

- ❌ 用于从零生成视频 → 必须有 Video 输入
- ❌ 期待改变结构 → 该模块只做修改，不做重构
- ❌ 与生成模块混用 → 该模块是编辑层能力
