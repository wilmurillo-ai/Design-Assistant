# 图生视频器（Image → Video Generator）

## 定义

将单张静态图像转化为单段短视频（通常 ≤15 秒）。是视频生成的基础执行模块。

## 输入 / 输出

- **输入**：Image（单张，必选）+ Text（结构化描述，可选）
- **输出**：Video（单段）

## 适用场景

- 人物图→动作视频（表情 / 动作 / 简单互动）
- 商品图→动态展示（轻运动 / 镜头移动）
- 快速验证创意、初版广告素材、UGC 内容

## 编排规则

```text
有图片 + 需要视频 → 使用该模块
无图片 → 不使用（需先生成图像）
多镜头 → 使用分镜视频模块（参考图分镜生视频器 / 故事板分镜生视频器）
```

## 关键原则

- 单图→单视频，不涉及列表态扩展
- 适用于短视频（≤15s），单表达单逻辑
- 不负责分镜、多段逻辑拆解，这些应在上游完成
- 图片可来自用户上传或上游生成结果

## 常见错误

- ❌ 用于长视频生成 → 应使用分镜视频模块
- ❌ 输入多图 → 本模块仅支持单图
- ❌ 期待复杂剧情 → 仅支持单段表达
