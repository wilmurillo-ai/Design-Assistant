# 换角度生图器（Angle Control Image Generator）

## 定义

在保持主体与语义一致的前提下，改变相机视角（水平/俯仰/远近），得到新的构图与透视效果。

属于**图像后处理模块**，通常位于生图之后。

## 输入 / 输出

- **输入**：Image（单张，必选）
- **输出**：Image（视角变化后）

## 适用场景

- 机位变化：正面→侧面/45°/俯视/仰视
- 商品多角度展示（电商正/侧/背/细节）
- 人物视角变化（不同机位的同一人物）

## 编排规则

```text
已有图像 + 需要换角度 → 使用该模块
需要改变内容/风格 → 不使用
```

## 关键原则

- 不改内容，只改视角（水平角、垂直角、Zoom）
- 仅支持单图输入，不适用于多图融合或列表态
- 可对同一输入多次调用，得到多角度结果集
- 从小角度（15°–45°）起步更稳定，角度+变焦过大可能失真

## 常见错误

- ❌ 期待生成新主体 → 本模块不负责内容生成
- ❌ 与多图融合混用 → 多图场景应使用图片参考生图器
- ❌ 与分镜列表混用 → 列表态应使用故事板分镜生图器
