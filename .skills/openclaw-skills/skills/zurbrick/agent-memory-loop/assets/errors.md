# Errors

> Format: `[YYYY-MM-DD] id:ERR-YYYYMMDD-NNN | COMMAND | what failed | fix | count:N | prevented:N | severity:medium | source:agent`
> All fields after fix are optional. Existing one-line entries still valid.
> Dedup: check ID first, then keyword grep. Increment count if exists.
> Review queue: count:3+ OR severity:critical → promotion-queue.md
