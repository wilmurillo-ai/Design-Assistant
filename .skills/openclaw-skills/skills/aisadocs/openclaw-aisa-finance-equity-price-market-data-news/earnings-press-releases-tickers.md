## Usage

```bash
curl "https://api.aisa.one/apis/v1/financial/earnings/press-releases?ticker=NVDA" \
  -H "Authorization: Bearer $AISA_API_KEY"
```

Passing a ticker outside this list returns `{"error":"Invalid ticker"}` — the endpoint's coverage is narrower than other financial endpoints. If you need earnings data for a ticker not listed here, use `/financial/analyst-estimates` or `/financial/financials/income-statements` instead.
