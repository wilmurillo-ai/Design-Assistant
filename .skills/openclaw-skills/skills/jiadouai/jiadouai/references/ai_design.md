## 模特试穿试戴

### model\_try\_clothes

**功能说明**：让模特试穿指定的衣服、裤子、衣服+裤子（套装）。

**使用场景**：

- 让模特穿上指定的衣服、裤子、套装
- 在调用时需要上传衣服、裤子、套装的图片

**请求参数**：

| 参数         | 类型     | 必填 | 默认值 | 说明                                            |
| ---------- | ------ | -- | --- | --------------------------------------------- |
| `url`      | string | 是  | -   | 公网URL或本地**完整路径**                              |
| `gender`   | string | 否  | 随机  | 模特性别，可选值：`男`、`女`                              |
| `category` | string | 否  | 随机  | 拍摄场景分类，可选值：`室内商拍`、`稚趣童装`、`女生街拍`、`户外行摄`、`男士街拍` |
| `feature`  | object | 否  | -   | 人物特征配置，详见下方 [feature 字段说明](#feature-字段说明)     |

#### feature 字段说明

所有字段均为非必填，值直接传入中文描述：

| 字段           | 类型     | 说明 | 示例值         |
| ------------ | ------ | -- | ----------- |
| `age`        | string | 年龄 | 儿童、青年、中年、老年 |
| `hair_style` | string | 发型 | 短发、长发、卷发、寸头 |
| `hair_color` | string | 发色 | 黑色、金色、棕色、白色 |
| `color`      | string | 肤色 | 白皙、自然、小麦色   |
| `figure`     | string | 身材 | 苗条、健壮、匀称    |
| `expression` | string | 表情 | 微笑、严肃、自然    |

**调用示例**：

示例1：基础调用（仅传入衣服图片）

```json
{
  "url": "https://example.com/clothes.jpg"
}
```

示例2：指定性别和场景分类

```json
{
  "url": "https://example.com/clothes.jpg",
  "gender": "女",
  "category": "女生街拍"
}
```

示例3：完整参数（含人物特征配置）

```json
{
  "url": "https://example.com/clothes.jpg",
  "gender": "女",
  "category": "室内商拍",
  "feature": {
    "age": "青年",
    "hair_style": "长发",
    "hair_color": "黑色",
    "color": "白皙",
    "figure": "苗条",
    "expression": "微笑"
  }
}
```

**返回值说明**：

成功返回：

```json
{
  "code": 0,
  "msg": "ok",
  "data": {
    "_id": "调用链追踪ID"
  }
}
```

失败返回：

```json
{
  "code": 1001,
  "msg": "图片URL无效",
  "data": null
}
```

> **注意**：
>
> - 返回结果中 `code!=0` 时，表示调用失败，需要检查参数是否正确
> - 任务提交成功后需要使用`get_job_status`查询生成进度和结果

## 商品场景图

### business\_product\_scene

**功能说明**：将商品图片放入指定场景生成场景化效果图。支持家电、家具、美妆等商品在不同场景展示。

**使用场景**：

- 在家电、家具商品图片生成场景化展示效果图
- 在美妆商品图片生成手持或户外场景展示效果图

**请求参数**：

| 参数      | 类型     | 必填 | 默认值 | 说明                                                      |
| ------- | ------ | -- | --- | ------------------------------------------------------- |
| `url`   | string | 是  | -   | 商品图片公网URL或本地**完整路径**                                    |
| `type`  | string | 否  | -   | 商品类型，可选值：`家电`、`家具`、`美妆`                                 |
| `scene` | string | 否  | -   | 商品场景，需与 `type` 匹配，详见下方 [scene 参数映射](#scene-参数映射)        |
| `ratio` | string | 否  | 3:4 | 生成图比例，可选值：`21:9`、`16:9`、`4:3`、`1:1`、`3:4`、`9:16`、`9:21` |

#### scene 参数映射

不同商品类型支持的场景选项不同：

| 商品类型 (`type`) | 支持的场景 (`scene`) |
| ------------- | --------------- |
| 家电            | `厨房`、`阳台`、`客厅`  |
| 家具            | `厨房`、`阳台`、`客厅`  |
| 美妆            | `手持`、`户外`       |

**调用示例**：

示例1：家电商品放入厨房场景

```json
{
  "url": "https://example.com/refrigerator.jpg",
  "type": "家电",
  "scene": "厨房",
  "ratio": "3:4"
}
```

示例2：美妆商品手持展示

```json
{
  "url": "https://example.com/lipstick.jpg",
  "type": "美妆",
  "scene": "手持"
}
```

**返回值说明**：

成功返回：

```json
{
  "code": 0,
  "msg": "ok",
  "data": {
    "_id": "调用链追踪ID"
  }
}
```

失败返回：

```json
{
  "code": 1001,
  "msg": "图片URL无效",
  "data": null
}
```

> **注意**：
>
> - 返回结果中 `code!=0` 时，表示调用失败，需要检查参数是否正确
> - 任务提交成功后需要使用`get_job_status`查询生成进度和结果
> - `scene` 参数需与 `type` 匹配，否则可能调用失败
