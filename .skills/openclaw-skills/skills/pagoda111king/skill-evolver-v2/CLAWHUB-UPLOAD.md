## ❓ FAQ（常见问题）

**Q1: skill-evolver 如何收集技能使用数据？**
A: 通过事件驱动架构自动捕获技能调用事件（调用时间、参数、耗时、结果、用户反馈），无需手动埋点。支持手动导入历史数据。

**Q2: 优化策略有哪些类型？**
A: 4 种策略：1) 性能优化（降低延迟、提升吞吐量）2) 体验优化（简化流程、提升易用性）3) 功能扩展（新增能力、填补空白）4) 稳定性优化（错误处理、容错机制）。

**Q3: 如何评估优化效果？**
A: 提供优化前后对比报告，包含关键指标变化（成功率、满意度、耗时、调用量），支持 A/B 测试框架验证优化效果。

**Q4: 支持哪些技能类型？**
A: 支持所有遵循标准接口规范的技能，包括内置技能（weather、pdf、xlsx 等）和第三方技能。自定义技能需实现标准事件接口。

**Q5: 数据隐私如何保障？**
A: 所有数据本地存储，不上传云端。企业版支持数据加密和访问控制。符合 GDPR 和中国数据安全法规。

## 📋 核心功能

### 1. 六维评估体系

| 维度 | 说明 | 权重 |
|------|------|------|
| **使用频率** | 单位时间内调用次数 | 15% |
| **成功率** | 成功完成的比例 | 25% |
| **用户满意度** | 用户评分（1-5 星） | 20% |
| **响应速度** | 平均响应时间 | 15% |
| **功能完整性** | 覆盖需求的比例 | 15% |
| **稳定性** | 无故障运行时间 | 10% |

**综合得分计算**:
```
综合得分 = Σ(维度得分 × 权重)
得分等级：
- 90-100: 优秀（S 级）
- 80-89: 良好（A 级）
- 70-79: 中等（B 级）
- 60-69: 及格（C 级）
- <60: 需改进（D 级）
```

### 2. 短板识别

**识别方法**:
- 六维得分最低的前 3 项
- 用户反馈中提及最多的问题
- 错误率最高的功能模块
- 响应时间最长的操作

**输出格式**:
```markdown
## 短板分析：[技能名]

### Top 3 短板
1. [短板 1] - 得分：XX - 影响：XX%
2. [短板 2] - 得分：XX - 影响：XX%
3. [短板 3] - 得分：XX - 影响：XX%

### 用户反馈热点
- 「希望增加 XX 功能」- 提及 23 次
- 「XX 操作太慢」- 提及 18 次
- 「XX 场景不支持」- 提及 15 次
```

### 3. 优化策略引擎（v0.2.0 新增）

**4 种优化策略**:

| 策略 | 适用场景 | 典型改进 |
|------|----------|----------|
| **性能优化** | 响应慢、吞吐低 | 缓存、异步、并行 |
| **体验优化** | 易用性差、学习成本高 | 简化流程、智能提示 |
| **功能扩展** | 需求覆盖不足 | 新增功能、填补空白 |
| **稳定性优化** | 错误率高、易崩溃 | 容错、降级、监控 |

**策略选择算法**:
```javascript
function selectStrategy(skillData) {
  if (skillData.avgResponseTime > 5000) {
    return 'PERFORMANCE'; // 性能优化
  }
  if (skillData.userSatisfaction < 3.5) {
    return 'EXPERIENCE'; // 体验优化
  }
  if (skillData.featureCoverage < 0.7) {
    return 'FEATURE'; // 功能扩展
  }
  if (skillData.errorRate > 0.05) {
    return 'STABILITY'; // 稳定性优化
  }
  return 'BALANCED'; // 综合优化
}
```

## 🔌 事件驱动架构（v0.2.0）

### 事件类型

```javascript
const SkillEvents = {
  // 技能生命周期事件
  CALLED: 'skill-called',        // 技能被调用
  COMPLETED: 'skill-completed',  // 技能完成
  FAILED: 'skill-failed',        // 技能失败
  TIMEOUT: 'skill-timeout',      // 技能超时
  
  // 用户反馈事件
  RATED: 'skill-rated',          // 用户评分
  FEEDBACK: 'skill-feedback',    // 用户反馈
  
  // 系统事件
  UPDATED: 'skill-updated',      // 技能更新
  DEPRECATED: 'skill-deprecated' // 技能废弃
};
```

### 事件订阅

```javascript
const { EventBus, SkillEvents } = require('./src/event-bus');

const eventBus = new EventBus();

// 订阅技能调用事件
eventBus.on(SkillEvents.CALLED, (event) => {
  console.log(`[${event.timestamp}] ${event.skillName} 被调用`);
  console.log('参数:', event.params);
});

// 订阅技能完成事件
eventBus.on(SkillEvents.COMPLETED, (event) => {
  console.log(`[${event.timestamp}] ${event.skillName} 完成`);
  console.log('耗时:', event.duration, 'ms');
  console.log('结果:', event.result);
});

// 订阅用户评分事件
eventBus.on(SkillEvents.RATED, (event) => {
  console.log(`用户评分：${event.rating}/5`);
  console.log('评价:', event.comment);
});
```

### 事件过滤

```javascript
// 只订阅特定技能的事件
eventBus.on(SkillEvents.CALLED, (event) => {
  // 处理所有技能调用
}, { skillName: 'meta-skill-weaver' });

// 只订阅特定用户的事件
eventBus.on(SkillEvents.RATED, (event) => {
  // 处理特定用户的评分
}, { userId: 'user-123' });
```
