## v0.3.0 (2026-04-13) - 重大升级

**升级类型**：重大升级 (B 选项 - 技能创造任务)

**核心改进**：
1. ✅ 新增状态机引擎 - 管理进化流程状态（8 状态完整状态机）
2. ✅ 新增持久化层 - 进化日志、版本历史、状态存储
3. ✅ 新增进化流水线 - 6 阶段自动化进化流程
4. ✅ 支持中断恢复 - 从文件读取状态继续执行
5. ✅ 增强事件系统 - 状态变更、流水线事件

**应用的设计模式**：
- 状态机模式 (State Machine Pattern) - 进化流程状态管理
- 状态外置原则 (State Externalization) - 文件>内存，支持中断恢复
- 流水线模式 (Pipeline Pattern) - 阶段化执行
- 事件驱动架构 (Event-Driven Architecture) - 增强现有事件系统

**新增文件**：
- `src/state-machine/evolution-state-machine.js` - 状态机核心
- `src/persistence/file-store.js` - 文件存储层
- `src/pipeline/evolution-pipeline.js` - 进化流水线
- `data/` - 数据目录（evolution-log, version-history, state-store）
