## ❓ FAQ（常见问题）

**Q1: workflow-optimizer 如何分析我的工作流程？**
A: 支持 3 种方式：1) 手动描述流程步骤 2) 导入任务管理工具数据（如 Trello、Jira）3) 自动追踪技能调用日志。分析后输出流程图、耗时分布、瓶颈识别报告。

**Q2: 自动化推荐是如何生成的？**
A: 基于规则引擎识别可自动化任务：重复性高、规则明确、低创意需求、高耗时。推荐方案包括脚本自动化、RPA、AI 代理等，附带实施难度和预期收益评估。

**Q3: 如何量化优化效果？**
A: 提供优化前后对比仪表盘，包含关键指标变化：总耗时、平均步骤数、错误率、人工操作比例、自动化率。支持设置基线和目标值，追踪达成进度。

**Q4: 支持哪些类型的流程？**
A: 支持任何可分解为步骤的流程：技能开发、内容创作、数据分析、客户支持、项目管理、日常行政等。流程步骤越多、重复性越高，优化空间越大。

**Q5: 团队版如何协作？**
A: 商业版和企业版支持团队流程库，成员可共享流程模板、优化方案、最佳实践。支持流程版本管理和变更审批，确保团队流程一致性。

### 1. 流程分析

```markdown
| 6 | 部署上线 | 30min | 5 | 运维 |

1 → 2 → 3 → 4 → 5 → 6 (总耗时 450min)

### 瓶颈识别

- 步骤 4（代码审查）等待时间长，平均等待 2 小时
```

### 2. 瓶颈识别

**识别维度**:
- **耗时瓶颈**: 单步骤耗时超过平均值 2 倍
- **等待瓶颈**: 步骤间等待时间过长
- **资源瓶颈**: 某角色/工具负载过高
- **质量瓶颈**: 某步骤返工率高

**瓶颈评分**:
```
瓶颈分数 = (耗时权重 × 0.4) + (等待权重 × 0.3) + (返工权重 × 0.3)

分数解释:
- 80-100: 严重瓶颈，优先优化
- 60-79: 中等瓶颈，计划优化
- 40-59: 轻微瓶颈，可选优化
- <40: 非瓶颈，无需关注
```

### 3. 优化建议

**建议类型**:
- **流程重组**: 调整步骤顺序、合并/拆分步骤
- **并行化**: 将串行步骤改为并行执行
- **自动化**: 用脚本/工具替代人工操作
- **标准化**: 建立模板、检查清单、SOP
- **工具升级**: 引入更高效的工具

**建议格式**:
```markdown
## 优化建议：[流程名]

### 当前问题
- 问题 1: [描述] - 影响：[量化]
- 问题 2: [描述] - 影响：[量化]

### 改进建议

#### 建议 1: [名称]
**类型**: 自动化
**描述**: [具体方案]
**预期收益**: 节省 XX 分钟/次，错误率降低 XX%
**实施难度**: 低/中/高
**预计时间**: X 天

#### 建议 2: [名称]
**类型**: 流程重组
**描述**: [具体方案]
**预期收益**: 总耗时减少 XX%
**实施难度**: 低/中/高
**预计时间**: X 天

### 优先级排序
1. 建议 1（收益高、难度低）
2. 建议 2（收益中、难度中）
3. 建议 3（收益低、难度高）
```

### 4. 自动化推荐

**可自动化判定规则**:
```
可自动化分数 = (重复性 × 0.4) + (规则明确性 × 0.3) + (创意需求 × -0.3)

判定标准:
- 分数 ≥ 70: 强烈推荐自动化
- 分数 50-69: 建议自动化
- 分数 30-49: 可考虑自动化
- 分数 <30: 不建议自动化
```

**自动化方案类型**:
| 类型 | 适用场景 | 示例 |
|------|----------|------|
| **脚本自动化** | 文件处理、数据转换 | Python 脚本批量处理 Excel |
| **RPA** | 跨系统操作、UI 交互 | 自动填写表单、数据录入 |
| **AI 代理** | 内容生成、智能决策 | AI 写报告、智能分类 |
| **API 集成** | 系统间数据同步 | Webhook 自动触发 |
| **定时任务** | 周期性重复工作 | 每日数据备份、周报生成 |

### API 调用（企业版）

```javascript
const WorkflowOptimizer = require('workflow-optimizer');

const optimizer = new WorkflowOptimizer();

// 分析流程
const analysis = await optimizer.analyze({
  name: '技能开发流程',
  steps: [
    { name: '需求分析', duration: 30 },
    { name: '技术方案', duration: 60 },
    { name: '代码开发', duration: 240 },
  ]
});

// 获取优化建议
const suggestions = await optimizer.suggest(analysis);

// 追踪效果
const effect = await optimizer.trackEffect({
  before: { duration: 450, errors: 8 },
  after: { duration: 215, errors: 2 }
});
```
