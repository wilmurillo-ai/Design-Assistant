## 📝 技能描述（200 字）

**context-booster（上下文增强器）** 是一款智能上下文管理引擎，解决 AI 对话中的 context window 限制问题。通过智能压缩、关键摘要、上下文增强、记忆检索、上下文预测五大核心功能，自动识别并压缩低价值内容，提取关键决策和行动项，在需要时补充相关背景信息。核心价值：提升长程任务表现，减少信息丢失，增强 AI"记忆力"，让 AI 在超长对话中保持上下文连贯，是复杂项目、多轮对话、知识管理场景的必备工具。

## ❓ FAQ（常见问题）

**Q1: context-booster 如何判断哪些内容可以压缩？**
A: 基于内容价值评分：决策、行动项、关键洞察评分高（保留）；重复内容、中间推理、已完成任务评分中（压缩）；闲聊、错误尝试、临时信息评分低（删除）。

**Q2: 记忆检索的准确率如何？**
A: 采用语义检索 + 关键词匹配双引擎，准确率约 92%。支持模糊查询（「那个关于 XX 的讨论」）和精确查询（「2026-03-15 的技术方案」）。

**Q3: 上下文压缩会丢失信息吗？**
A: 压缩采用「摘要 + 原文引用」双存储模式。摘要用于快速浏览，原文可随时回溯。关键信息 100% 保留，仅压缩冗余内容。

**Q4: 支持多少条记忆存储？**
A: 个人版 1000 条，商业版 10000 条，企业版无限。每条记忆平均 200-500 字。支持记忆归档和批量导出。

**Q5: 如何保护隐私数据？**
A: 所有数据本地加密存储，支持敏感信息自动识别和脱敏。企业版支持私有部署和数据访问审计。符合 GDPR 和中国数据安全法规。

## 📋 核心功能

### 1. 智能压缩

**触发条件**: Context 使用率 > 70%

**压缩策略**:
- 保留：决策、行动项、关键洞察、用户偏好
- 压缩：重复内容、中间推理过程、已完成的子任务
- 删除：无关闲聊、错误尝试、临时信息

**压缩输出**:
```markdown
## 压缩摘要 (YYYY-MM-DD HH:MM)

### 关键决策
- 决策 1
- 决策 2

### 行动项
- [ ] 任务 1
- [ ] 任务 2

### 关键洞察
- 洞察 1
- 洞察 2
```

**压缩示例**:
```
原始对话：5000 字
压缩后：800 字（保留 85% 关键信息）
压缩比：84%
```

### 2. 关键摘要

**摘要提取规则**:
- 识别「我们决定...」「下一步...」「重要的是...」等关键句式
- 提取带复选框的行动项（[ ]、[x]）
- 捕捉数字、日期、名称等实体信息
- 记录用户明确表达的偏好

**摘要格式**:
```markdown
## 对话摘要 (2026-04-01 14:00-15:30)

**参与方**: 用户、AI 助理

**讨论主题**: M 维度 ClawHub 上架材料准备

**关键决策**:
1. 选择 5 个核心技能进行上架
2. 每个技能需准备 CLAWHUB-UPLOAD.md 文件
3. 截止时间：次日 01:00

**行动项**:
- [x] 创建执行计划文件
- [x] 检查 5 个技能是否存在
- [ ] 为每个技能编写上架材料
- [ ] 生成检查清单
- [ ] 输出汇总报告

**关键洞察**:
- 5 个技能均存在于 ~/.openclaw/skills/ 目录
- 每个技能需要 200+ 行 README
- 需要包含使用示例≥3 个、FAQ≥3 个
```

### 3. 上下文增强

**增强模式**:
- **背景补充**: 自动补充项目历史、相关决策
- **知识链接**: 链接到相关记忆、文档、技能
- **状态同步**: 同步任务进度、待办事项

**增强示例**:
```

- 目标：完成 5 个核心技能的上架材料
- 当前进度：已完成 2/5 个技能（meta-skill-weaver、first-principle-analyzer）
- 待完成：skill-evolver、context-booster、workflow-optimizer

【上下文增强 - 关键决策】
- 输出文件位置：~/workspace/skills/[skill-name]/CLAWHUB-UPLOAD.md
- 质量标准：README≥200 行、示例≥3 个、FAQ≥3 个

```

### 4. 记忆检索

**检索方式**:
- **关键词检索**: 匹配记忆中的关键词
- **语义检索**: 理解查询意图，匹配相似内容
- **时间范围**: 限定检索时间范围
- **标签过滤**: 按标签筛选记忆

**检索示例**:
```

### 5. 上下文预测

**预测逻辑**:

- 匹配历史相似场景
- 预测下一步可能需要的信息
- 主动预加载相关上下文

**预测示例**:
```
