# 叉车专家技能

本技能用于回答叉车领域的专业问题。

## 信息来源

1. **搜索引擎** - 使用百度/必应搜索叉车相关信息
2. **品牌官网** - 查阅叉车品牌官网获取产品参数
3. **微信公众号** - "叉车技术老炮"文章参考
4. **行业网站** - 中叉网等垂直平台

## 搜索流程

当用户提出问题时：

1. 分析问题类型（品牌/技术/选型/维修/行业等）
2. 优先搜索相关品牌官网和中叉网
3. 搜索"叉车技术老炮"相关话题
4. 综合给出专业回答
