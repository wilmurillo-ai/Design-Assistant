# Founder GTM System — Router

This skill routes to platform-specific strategies. For detailed playbooks, use the dedicated skills.

## Quick Router

| User Wants | Use This Skill | Command |
|------------|---------------|---------|
| Twitter/X content or strategy | twitter-x-gtm | Direct user to ask "Twitter strategy" |
| LinkedIn content or strategy | linkedin-gtm | Direct user to ask "LinkedIn strategy" |
| Conference/event prep | event-gtm | Direct user to ask "CES planning" |
| Content creation/repurposing | founder-content | Direct user to ask "write a post" |
| Investor communication | founder-fundraising | Direct user to ask "investor update" |

## When to Use This Skill

Use founder-gtm ONLY when:
1. User asks general "what platform should I use?" questions
2. User wants cross-platform strategy comparison
3. User asks about overall GTM without specifying a platform

## Cross-Platform Quick Reference

### Posting Cadence

| Platform | Frequency | Best Times |
|----------|-----------|------------|
| Twitter/X | 3-5/day | 9-10 AM, 1-3 PM EST |
| LinkedIn | 3-4/week | 8-11 AM Tue-Thu |

### Algorithm Commonalities (2025)

All platforms share these signals:
- **Comments >> Likes** — engagement depth matters most
- **First hour critical** — velocity determines distribution
- **Saves = strongest signal** — indicates true value
- **AI content penalized** — LinkedIn -30% reach, others similar

### Content Adaptation Pattern

Same insight, different rendering:

| Platform | Angle | Format |
|----------|-------|--------|
| Twitter/X | Quick insight, spicy take | <280 chars, thread for depth |
| LinkedIn | Framework, professional lesson | 8-20 lines, spaced, carousel |

### Investor Relationship Timeline (All Platforms)

| Phase | Timeline | Activities |
|-------|----------|------------|
| Lurk | 12-6 months before raising | Follow, observe, occasional engagement |
| Engage | 6-3 months | Regular engagement, demonstrate expertise |
| Value-Add | 3-1 months | Direct value-adds (intros, useful data) |
| Ask | Raising | Warm outreach referencing relationship |

## Examples

### Example 1: User asks "which platform should I focus on?"

**Response pattern:**
1. Ask about their goal (brand awareness, investor reach, customer acquisition)
2. Ask about their capacity (hours per week)
3. Recommend based on:
   - Limited time → Pick ONE platform, go deep
   - DTC focus → Twitter first
   - Investor focus → LinkedIn first
