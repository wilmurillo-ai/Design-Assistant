## 模块关系总览

```
┌─────────────┐    ┌──────────────┐    ┌─────────────────┐
│ 主代理      │───►│ 子代理 A/B   │───►│ 封包代理        │
│ (Orchestrator)  │  (sessions_  │    │ (Packager)      │
│              │    │  spawn)     │    │                 │
└──────┬──────┘    └──────────────┘    └────────┬────────┘
       │                                        │
       │         ┌──────────────┐                │
       │         │ memory/     │                │
       │         │ history.json│                │
       │         │ patterns.log│                │
       │         └──────┬───────┘                │
       │                │                        │
```
