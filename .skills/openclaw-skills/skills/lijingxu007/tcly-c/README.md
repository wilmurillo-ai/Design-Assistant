# 🌍 旅游定制助手 (Travel Customizer)

这是一个专为旅游从业者设计的 OpenClaw 技能插件。它能通过自然对话收集客户的旅游需求，并自动结构化存储到**飞书多维表格**中，极大提高需求收集效率。

## ✨ 功能特点
- **智能对话**: 自动引导用户补充缺失的关键信息（如人数、预算）。
- **二次确认**: 提交前强制要求用户确认信息，避免错误录入。
- **无缝集成**: 数据直达飞书多维表格，方便团队协作跟进。
- **隐私保护**: 对话中自动脱敏敏感信息。

## 🚀 安装与配置

### 前置准备
1. **创建飞书多维表格**:
   - 新建表格，包含以下列名（必须完全一致）：`姓名`, `联系电话`, `意向目的地`, `出行人数`, `预计出发日期`, `人均预算`, `行程偏好`, `特殊需求`。
   - 获取表格的 `Base Token` 和 `Table ID`。

2. **创建飞书自建应用**:
   - 访问 [飞书开放平台](https://open.feishu.cn/)。
   - 创建应用，开通 `bitable:app` 和 `base:record:create` 权限。
   - 获取 `App ID` 和 `App Secret`。

### 在 ClawHub 中安装
1. 在 ClawHub 搜索 "旅游定制助手" 并点击安装。
2. 在配置页面填入以下信息：
   - **飞书 App ID**: `cli_xxxxx`
   - **飞书 App Secret**: `xxxxx`
   - **飞书 Base Token**: `bascn_xxxxx`
   - **飞书 Table ID**: `tbl_xxxxx`
3. 保存配置即可开始使用。

## 💬 使用示例
> **用户**: 我想去日本玩，两个人。
> **助手**: 好的！请问您大概什么时候出发？人均预算是多少呢？还有您的称呼是？
> ...
> **助手**: 收到，帮您总结一下：张三先生，2人去日本，预算人均1万，下月出发。确认提交给定制团队吗？
> **用户**: 确认。
> **助手**: ✅ 提交成功！顾问稍后联系您。
