# 记忆技巧指南

## 联想记忆法

### 图像联想

将单词与具体图像联系起来：

- **abandon** → a + ban + don → 一个被禁止的捐赠 → 放弃
- **ambition** → 音似"俺必胜" → 有雄心壮志
- **chaos** → 混乱 → 想象"吵死"的场景

### 场景联想

将单词放入生动场景：

- **deadline** → 截止日期 → 想象一条生命线，过了就"dead"
- **breakthrough** → 突破 → 想象冲破墙壁的画面

## 词根词缀拆解法

### 步骤

1. 识别前缀 → 确定基本含义方向
2. 找到词根 → 理解核心含义
3. 分析后缀 → 确定词性

### 示例

**unprecedented**
- un-（不）+ pre-（前）+ ced（走）+ -ent（形容词）+ -ed
- 没有先例走过的 → 史无前例的

**international**
- inter-（之间）+ nation（国家）+ -al（形容词）
- 国家之间的 → 国际的

## 谐音记忆法

利用发音相似性：

| 单词 | 谐音 | 含义 |
|------|------|------|
| ambulance | 俺不能死 | 救护车 |
| economy | 依靠农民 | 经济 |
| pest | 拍死它 | 害虫 |
| agony | 爱过你 | 痛苦 |
| stamina | 死呆米那 | 耐力 |

⚠️ 注意：谐音记忆适合辅助，但不要影响正确发音。

## 故事记忆法

将多个相关单词编入一个故事：

> 小明有 **ambition**（雄心），想要 **accomplish**（完成）一个 **mission**（任务）。虽然过程很 **tough**（艰难），但他保持 **passion**（热情），最终取得了 **success**（成功）。

## 间隔重复法

### 艾宾浩斯遗忘曲线

| 间隔 | 记忆保持率 |
|------|-----------|
| 20分钟 | 58% |
| 1小时 | 44% |
| 1天 | 33% |
| 6天 | 25% |

### 复习建议

1. **第1次**：学完立即复习
2. **第2次**：20分钟后
3. **第3次**：1小时后
4. **第4次**：1天后
5. **第5次**：1周后
6. **第6次**：1月后

## 词源故事法

了解单词背后的故事：

### salary（工资）

古罗马士兵的薪水用来买盐（salt），salt → salary

### disaster（灾难）

dis（坏）+ aster（星）→ 星象不好 → 灾难

### muscle（肌肉）

mus（老鼠）+ cle（小）→ 肌肉收缩像小老鼠在动
