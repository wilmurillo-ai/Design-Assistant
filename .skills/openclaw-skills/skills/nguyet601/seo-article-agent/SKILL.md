# SEO Article Agent

## Overview

SEO Article Agent is an AI writing skill designed to generate long-form SEO optimized articles from a given topic or keyword.

This agent focuses on creating structured, search-engine-friendly content including introductions, headings, keyword placement, and clear sections to improve readability and ranking potential.

## Features

- Generate SEO-friendly blog articles
- Automatically structure headings (H1, H2, H3)
- Keyword optimized writing
- Long-form article generation
- Clear and readable content

## Usage

Input a topic or keyword and the agent will generate a full SEO-optimized article suitable for blogs, websites, or content marketing.

Example:

Input:
"Best AI tools for productivity"

Output:
A complete SEO article with introduction, structured headings, and detailed explanations.
