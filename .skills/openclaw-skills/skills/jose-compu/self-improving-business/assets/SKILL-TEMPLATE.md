# Business Skill Template

Template for creating business-administration skills extracted from learnings and business issues.

---

## SKILL.md Template

```markdown
---
name: skill-name-here
description: "Concise description of the business administration skill and trigger conditions."
---

# Skill Name

Brief introduction explaining the business problem this skill addresses.

## Quick Reference

| Situation | Action |
|-----------|--------|
| [Trigger 1] | [Recommended response 1] |
| [Trigger 2] | [Recommended response 2] |

## Background

Why this matters for operations, governance, KPI quality, or decision velocity.

## Recommended Action

1. First remediation step
2. Second step
3. Verification step

## Reminder-Only Safety

This skill documents recommendations only.
It does not execute approvals, spending, vendor commitments, payroll, or legal actions.

## Source

- Entry ID: LRN-YYYYMMDD-XXX or BUS-YYYYMMDD-XXX
- Original Category: process_breakdown | policy_gap | kpi_misalignment | decision_latency | handoff_failure | stakeholder_misalignment | documentation_drift | compliance_oversight
- Extraction Date: YYYY-MM-DD
```

---

## Extraction Checklist

- [ ] Finding is verified and broadly applicable
- [ ] Content is reminder-only and non-transactional
- [ ] No approval/spend/commitment execution implied
- [ ] Name and description follow conventions
- [ ] Quick reference maps trigger to action
- [ ] Source entry ID captured
