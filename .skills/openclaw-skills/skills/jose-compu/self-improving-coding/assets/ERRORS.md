# Bug Patterns Log

Recurring bugs, common mistakes, lint errors, type mismatches, and runtime exceptions captured during development.

**Triggers**: lint_error | type_checker | runtime_exception | test_failure | code_review | static_analysis | profiler
**Areas**: syntax | logic | data_structures | algorithms | error_handling | testing | tooling

## Status Definitions

| Status | Meaning |
|--------|---------|
| `pending` | Not yet addressed |
| `in_progress` | Actively being fixed |
| `resolved` | Bug fixed, prevention added |
| `wont_fix` | Decided not to address (reason in Resolution) |
| `promoted` | Elevated to lint rule, style guide, or debug playbook |
| `promoted_to_skill` | Extracted as a reusable skill |
