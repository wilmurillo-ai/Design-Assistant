# Support Feature Requests

Support tooling, automation capabilities, and workflow improvements requested or identified during support operations.

**Areas**: triage | diagnosis | resolution | follow_up | documentation | escalation
**Statuses**: pending | in_progress | resolved | wont_fix

## Common Request Categories

| Category | Examples |
|----------|----------|
| Triage Automation | Auto-categorization, priority prediction, duplicate detection |
| Diagnosis Tools | Troubleshooting tree builder, log correlation, health check scripts |
| Resolution Aids | Canned response library, KB search improvements, solution templates |
| SLA Management | Timer dashboards, breach alerting, auto-escalation on approaching SLA |
| Customer Insights | Churn risk scoring, CSAT trend analysis, sentiment detection |
| Knowledge Base | Article gap detection, auto-suggest during ticket creation, freshness alerts |
