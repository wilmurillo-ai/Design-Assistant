# 刘晨明乖离率方法

## 原文来源

广发策略首席刘晨明在《【广发策略】如何区分主线是调整还是终结？》一文中提出。

## 核心目的

追踪主线并判断其属于**调整**还是**终结**。

## 计算公式

```
均线偏离度 = (对数收盘价 / 20 日 EMA) - 1
```

**注意：** 使用**自然对数 (Ln)**

## 交易阈值

| 偏离度 | 操作建议 |
|--------|---------|
| 0.6% - 1.8% | ✅ 最佳入场时机 |
| > 2% | ⚠️ 容易吃到回调 |
| < 0.6% 或负值 | ⚠️ 趋势可能转弱 |

## 五点心得体会

### ❶ 不上 5% 不够强
强势股的乖离率需要达到一定水平才能确认趋势强度。

### ❷ 关注回抽两步走
回调后的二次入场机会往往更安全。

### ❸ 品种之间择强辅助
对比多个标的，选择乖离率最强势的品种。

### ❹ 止损后不轻易入场
止损后等待新的乖离率信号，不要急于翻本。

### ❺ 逆练真经也有效
反向思维：乖离率过高时考虑止盈，过低时考虑抄底。

## 实战应用

### 入场条件
1. 乖离率进入 0.6%-1.8% 区间
2. 配合成交量放大
3. 均线多头排列
4. 其他指标共振

### 出场条件
1. 乖离率>2%，考虑止盈
2. 乖离率转负，趋势转弱
3. 跌破关键支撑位

## 注意事项

1. **适用场景：** 主线行情、趋势明确的股票
2. **不适用：** 震荡市、无明显趋势的标的
3. **配合使用：** 需结合其他指标综合判断
4. **动态调整：** 不同板块、不同市值股票阈值可微调
