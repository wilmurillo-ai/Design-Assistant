# WeryAI Account

Use this package when the user wants WeryAI account credits or balance information rather than generation or editing.

Preferred entry point:

- `node {baseDir}/scripts/account.js balance`

Route intents this way:

- credits balance
- account balance
- how many credits are left
- check my WeryAI credits

Read `SKILL.md` first for trigger language, API key guidance, and output expectations.
Read `references/account-api.md` when you need the exact official response contract.
