# OSS Invoice With Validation

> Read [fakturownia-invoices](../../subskills/invoices/SKILL.md) first.

Use this when an invoice should be marked as OSS and you want the server to validate the country rules before doing so.

## Command

```bash
fakturownia invoice create \
  --identify-oss \
  --input '{
    "kind":"vat",
    "seller_name":"Wystawca Sp. z o.o.",
    "seller_country":"PL",
    "buyer_name":"Klient1 Sp. z o.o.",
    "buyer_country":"FR",
    "use_oss":true,
    "positions":[{"name":"Produkt A1","tax":20,"total_price_gross":50,"quantity":3}]
  }' \
  --json
```
