# Download KSeF XML And UPO

> Read [fakturownia-invoices](../../subskills/invoices/SKILL.md) first.

Use this when you need the KSeF XML or UPO XML that the API exposes as invoice attachments.

## Download The KSeF Invoice XML

```bash
fakturownia invoice download-attachment --id 111 --kind gov --dir ./attachments --json
```

## Download The KSeF UPO XML

```bash
fakturownia invoice download-attachment --id 111 --kind gov_upo --dir ./attachments --json
```

## Notes

- `kind=gov` means the KSeF invoice XML.
- `kind=gov_upo` means the KSeF UPO XML.
- The CLI prefers the upstream filename from response headers when one is provided.
