# Similar Invoice From Another Document

> Read [fakturownia-invoices](../../subskills/invoices/SKILL.md) first.

Use this when the new document should inherit positions and core fields from an existing source document.

## Command

```bash
fakturownia invoice create --input '{"copy_invoice_from":42,"kind":"vat"}' --json
```

## Variants

- Advance invoice from an order: set `kind` to `advance`, plus `advance_creation_mode`, `advance_value`, and `position_name`.
- Final invoice from an order and advances: set `kind` to `final` and include `invoice_ids`.
