# Minimal Invoice From Existing IDs

> Read [fakturownia-invoices](../../subskills/invoices/SKILL.md) first.

Use this when you already know the `client_id`, `product_id`, and optionally the seller department/recipient IDs.

## Command

```bash
fakturownia invoice create --input '{"payment_to_kind":5,"client_id":1,"positions":[{"product_id":1,"quantity":2}]}' --json
```

## Notes

- This mirrors the README's minimal invoice example.
- The upstream API will default the issue date to today and the payment term to the `payment_to_kind` day count.
- Inspect `fakturownia schema invoice create --json` before building automated payloads.
