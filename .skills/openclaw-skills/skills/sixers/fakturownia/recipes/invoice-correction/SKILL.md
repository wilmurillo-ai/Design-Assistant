# Correction Invoice

> Read [fakturownia-invoices](../../subskills/invoices/SKILL.md) first.

Use this when you need a correction invoice or when you need the extra correction fields from an existing correction.

## Inspect an Existing Correction

```bash
fakturownia invoice get --id 2432393 \
  --additional-field corrected_content_before \
  --additional-field corrected_content_after \
  --correction-positions full \
  --json
```

## Create a Correction

```bash
fakturownia invoice create --input '{
  "kind":"correction",
  "correction_reason":"Zła ilość",
  "invoice_id":2432393,
  "from_invoice_id":2432393,
  "client_id":1,
  "positions":[{
    "name":"Product A1",
    "quantity":-1,
    "total_price_gross":"-10",
    "tax":"23",
    "kind":"correction",
    "correction_before_attributes":{"name":"Product A1","quantity":"2","total_price_gross":"20","tax":"23","kind":"correction_before"},
    "correction_after_attributes":{"name":"Product A2","quantity":"1","total_price_gross":"10","tax":"23","kind":"correction_after"}
  }]
}' --json
```
