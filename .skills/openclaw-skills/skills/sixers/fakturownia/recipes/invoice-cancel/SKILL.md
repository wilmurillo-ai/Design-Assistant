# Cancel Invoice

> Read [fakturownia-invoices](../../subskills/invoices/SKILL.md) first.

Use this when the invoice should be cancelled rather than edited.

## Cancel

```bash
fakturownia invoice cancel --id 111 --yes --reason 'Powód anulowania' --json
```

## Read Back the Stored Reason

```bash
fakturownia invoice get --id 111 --additional-field cancel_reason --json
```
