# Create Invoice From Receipt Or Link Existing Invoice

> Read [fakturownia-invoices](../../subskills/invoices/SKILL.md) first.

Use this when invoices and receipts need to be linked explicitly.

## Link an Existing Invoice to a Receipt

```bash
fakturownia invoice update --id 111 --input '{
  "from_invoice_id":222,
  "invoice_id":222,
  "exclude_from_stock_level":true
}' --json
```

## Create a New Invoice From a Receipt

```bash
fakturownia invoice create --input '{
  "from_invoice_id":222,
  "additional_params":"for_receipt",
  "exclude_from_stock_level":true,
  "buyer_name":"Klient1 Sp. z o.o.",
  "buyer_tax_no":"6272616681",
  "positions":[{"name":"Produkt A1","quantity":"1","tax":"23","total_price_gross":"10,23"}]
}' --json
```
