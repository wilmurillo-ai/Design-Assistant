# 长片完全指南（McKee价值/Harmon圆环/Snyder节拍）

> 75-120分钟的电影叙事

---

## 一、长片的独特挑战

### 1.1 长度意味着什么

90分钟不是9个10分钟故事的拼接，而是一个完整的旅程。

**Snyder观点：**
- 长片是"节奏机器"
- 每个节拍有精确的分钟定位
- 像过山车，有高有低

**McKee观点：**
- 长片是"价值转变的旅程"
- 每个场景必须有价值转变
- 冲突必须内外同步升级

**Harmon观点：**
- 长片是"角色变形记"
- 从A到B的完整变化
- 圆环结构天然完整

### 1.2 长片vs短片

| 方面 | 短片 | 长片 |
|------|------|------|
| 时长 | 5-10分钟 | 75-120分钟 |
| 弧光 | 一次性 | 多次伪变化 |
| 节拍 | Snyder简化版 | Snyder完整15节拍 |
| 价值 | 可选 | 每场景必须有 |
| 角色 | 1-2个 | 主角+配角+群像 |

## 二、McKee价值转变体系

### 2.1 核心概念

每个场景必须有价值转变：

```
场景开始：价值A（正/负）
场景结束：价值B（负/正）
如果没有转变 → 删除这个场景
```

### 2.2 "鸿沟"概念（McKee核心）

- 角色采取行动，期望得到结果A
- 实际得到结果B
- 这个落差就是戏剧张力
- 好的剧本让鸿沟层层加深

### 2.3 价值类型

| 价值 | 正 | 负 |
|------|-----|-----|
| 爱 | 爱 | 恨 |
| 自由 | 自由 | 束缚 |
| 真相 | 真相 | 谎言 |
| 生命 | 生 | 死 |
| 秩序 | 秩序 | 混乱 |

- 主角做出最终选择
- 这个选择定义了故事
- 改变观众对整个故事的理解

## 四、Harmon 故事圆环

### 4.1 八步圆环

```
1. YOU — 舒适区（建立常态）
   - 观众理解"这个人平时什么样"
   - 舒适区可能是痛苦的，但是熟悉的

2. NEED — 产生欲望
   - 角色意识到（或被强迫面对）缺少什么
   - 可以是外在任务或内在渴望

3. GO — 进入新世界
   - 跨过阈值，离开舒适区
   - 不可逆的决定

4. SEARCH — 寻找/适应/挣扎
   - 在新环境中试错
   - 用旧方法解决新问题

5. FIND — 得到想要的（Midpoint）
   - 达成表面目标
   - 但"得到"带来代价

6. TAKE — 付出代价
   - 代价开始显现
   - 最低点，内外压力最大

7. RETURN — 回到起点
   - 物理或心理回到起点
   - 但角色已经变了

8. CHANGE — 已改变
   - 展示变化后的新常态
   - 与Step 1对比
```

### 4.2 Snyder vs Harmon

| Snyder | Harmon | 适合类型 |
|--------|--------|---------|
| 15节拍精确分钟 | 8步灵活 | 商业片 vs 成长故事 |
| "最后一幕转折" | 圆环对称 | 类型驱动 vs 角色驱动 |
| 节拍机器 | 旅程变形 | 动作/惊悚 vs 文艺/成长 |

## 五、人物设计（McKee+Harmon）

### 5.1 McKee完整人物链条

```
Ghost（创伤） → Lie（谎言） → Flaw（缺陷） → Want（错误追求）
                                                          ↓
                                                    一系列考验
                                                          ↓
                                                    面对Need
                                                          ↓
                                                    关键选择
                                                          ↓
                                                    新常态
```

**Ghost**：过去的创伤事件，塑造当前世界观
**Lie**：因Ghost形成的错误信念
**Flaw**：Lie在行为层面的外化

### 5.2 Harmon人物构建

每个角色需要：
- 明确的Want
- 阻碍Want的外部力量
- 内心深处的Need
- 从舒适区到改变的旅程

### 5.3 Snyder人物类型

Snyder的15种故事形态中的角色原型：
- 受害者
- 学者
- 军人
- 情人
- 小丑
- 等等

## 六、次要角色设计

### 6.1 次要角色功能

1. **镜像角色**：映射主角的某个侧面
2. **催化角色**：推动主角变化
3. **主题角色**：具象化主题
4. **节奏角色**：提供调剂

### 6.2 Snyder配角要求

每个配角必须：
- 有自己的Want
- 与主角有明确关系
- 在某个节拍出现并发挥作用

---

## 七、伏笔系统（McKee核心）

### 7.1 伏笔类型

- **道具伏笔**：物件首次日常，再次关键
- **台词伏笔**：闲聊变关键
- **行为伏笔**：习惯暗示性格
- **结构伏笔**：无关场景变关键

### 7.2 McKee回扣原则

- 回扣必须改变观众对前面的理解
- "原来如此"时刻
- 每个伏笔必须回扣
- 长片至少3-5对伏笔

---

## 八、世界观设计（诺兰式分层）

### 8.1 三层结构

**第一层：表层规则**
- 观众最初看到的世界
- 基本运行方式
- 前30分钟

**第二层：运行逻辑**
- 表层背后的机制
- 为什么世界是这样
- 第二幕

**第三层：底层真相**
- 整个世界观的终极真相
- 改变对前面的理解
- 高潮段

### 8.2 诺兰式世界观

《盗梦空间》示例：
- 表层：梦境可以进入
- 运行：梦境有层级
- 真相：现实也可能是梦

## 九、节奏设计

### 9.1 Snyder精确控制

每个节拍有精确分钟，必须对齐：
- Opening Image：第1分钟
- Theme Stated：第5分钟
- Catalyst：第12分钟
- Midpoint：第55分钟

### 9.2 McKee节奏诊断

- 不能连续3场同一节奏
- 高潮后需要"呼吸"
- 情感重场之间需要"休息"

### 9.3 双轨节奏

**外部情节**：事件密度
- 紧：快节奏、高密度
- 松：日常、过渡

**内在情感**：角色情感
- 重：告白、失去
- 轻：调侃、闲聊
