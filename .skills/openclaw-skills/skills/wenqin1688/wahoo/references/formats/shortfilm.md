# 叙事短片完全指南（Snyder节奏/Sorkin对话）

> 5-10分钟的完整叙事

---

## 一、短片的独特魅力

### 1.1 短片是什么

短片是在有限时长内讲述一个完整故事的电影形式。5-10分钟意味着：

- 没有时间浪费
- 每一个场景都必须有用
- 情节密度必须高
- 情感冲击必须集中

### 1.2 Snyder 对短片的定义

短片是"压缩的戏剧"：
- 必须有明确的"最后一幕的转折"
- 节奏比长片更快
- 信息密度更高

### 1.3 Sorkin 对短片的看法

短片是展示"声音"的最佳形式：
- 5-10分钟足够建立角色声音
- 让角色用独特的说话方式定义自己

---

## 二、Snyder 故事形态

### 2.1 短片的故事形态（Snyder 15节拍压缩版）

Snyder的"故事形态"理论同样适用于短片：

| 节拍 | 短片版本 | 位置 |
|------|---------|------|
| Opening Image | 开场画面 | 0-15秒 |
| Theme Stated | 主题陈述 | 30秒 |
| Set-Up | 建立 | 1分钟 |
| Catalyst | 催化剂 | 1.5分钟 |
| Debate | 争论 | 2分钟 |
| Break into Two | 进入 | 2.5分钟 |
| Fun and Games | 游戏时间 | 3-5分钟 |
| Midpoint | 中点 | 5分钟 |
| Bad Guys Close In | 逼近 | 6分钟 |
| All Is Lost | 尽失 | 6.5分钟 |
| Dark Night | 至暗 | 7分钟 |
| Break into Three | 突破 | 7.5分钟 |
| Finale | 高潮 | 8分钟 |
| Final Image | 终场 | 8分钟 |

### 2.2 短片特有形态（Snyder）

**"倒计时"形态：**
- 明确的时间限制
- 压力递增
- 最后10秒决定一切

**"单点"形态：**
- 一个场景，一个事件
- 角色在其中被改变
- 《鱿鱼游戏》单集式

## 三、Sorkin 对话设计

### 3.1 对话的四重功能

Sorkin认为对话必须同时完成：

1. **揭示角色**
   - 角色通过说话展现自己
   - 不是"说什么"，而是"怎么说"

2. **推进情节**
   - 对话本身就是行动
   - 每个对话场景有戏剧目的

3. **传递信息**
   - 但不是"解释"
   - 信息通过冲突传递

4. **建立节奏**
   - 对话有快有慢
   - 像爵士乐有节奏感

### 3.2 Sorkin "厨房sink"原则

角色边做事边聊天：
- 做饭、开车、走路时聊天
- 动作与对话并行
- 看起来日常但信息丰富

### 3.3 角色"声音"设计（Sorkin核心）

每个角色必须有独特"声音"：

**用词偏好：**
- 律师角色用法律术语
- 军人角色用简洁命令
- 知识分子用复杂句式

**句式特点：**
- 角色A：短句、快节奏
- 角色B：长句、犹豫
- 角色C：爱用问句

**口头禅：**
- 每个角色有标志用语
- 观众听到就知道是谁

## 四、故事设计

### 4.1 一个核心事件

短片的故事必须围绕一个核心事件展开。

**Snyder检验：**
- 能用一句话说清这个故事吗？
- 这个目标足够紧迫吗？
- 这个阻碍足够强大吗？

### 4.2 戏剧动作

**定义**：戏剧动作 = 目标 + 阻碍 + 代价

好的短片只有一个主要的戏剧动作，所有场景都服务于它。

### 4.3 弧光设计

Snyder强调：短片必须有"变化"。

**弧光类型：**
- 外在弧光：处境改变
- 内在弧光：态度改变
- 关系弧光：关系改变

## 五、结构设计

### 5.1 Snyder压缩版四段式

**第一幕：建立（15%）**
- 展示角色常态
- 15秒内建立"问题"
- **开场钩子**：第一个画面抓住观众

**第二幕：对抗（50%）**
- 追求目标
- 遇到阻碍
- 阻碍升级
- **中点**：伪胜利或伪失败
- **"一切尽失"时刻**

**第三幕：解决（25%）**
- 关键选择
- 行动高潮
- 故事反转

**第四幕：结局（10%）**
- 展示"最后一幕的转折"
- Snyder：结局必须改变对前面的理解
- 不解释，让画面说话

### 5.2 时长分配

```
8分钟短片：
建立：1分钟
对抗：4分钟
解决：2分钟
结局：1分钟
```

## 六、场景设计

### 6.1 场景数量

5-10分钟的短片，通常有4-6个场景。

### 6.2 Snyder场景要求

每个场景必须有：
- 明确的戏剧动作（目标→障碍→结果）
- 结束时状态有变化
- 推进情节或揭示角色

### 6.3 场景标注

每个场景标注：
- 场景号
- 地点和时间
- 预估时长
- **Snyder节奏标签**：快/中/慢
- 情感重量：轻/中/重

---

## 七、写作技巧

### 7.1 视觉写作

- 只写能被摄影机拍到的
- 禁止心理描写
- 禁止括号解释

### 7.2 Sorkin对话技巧

- 口语化但有节奏
- 每个角色有独特声音
- 不要说双方都知道的信息
- 用动作代替解释

### 7.3 节奏控制

Snyder核心：短片节奏比长片快30%。

- 减少过渡场景
- 每个场景直接进入冲突
- 减少长对话，用动作代替
