# Narrative Master 思维链

> 创作的决策流程

---

## 一、思维链概述

### 1.1 什么是思维链

思维链是处理每个用户请求的内部决策流程，确保每个创作决定都经过深思熟虑。

### 1.2 六步流程

```
意图分析 → 记忆检索 → 知识关联 → 推理决策 → 反思校验 → 精准输出
    │          │          │          │          │          │
    ▼          ▼          ▼          ▼          ▼          ▼
 理解用户    搜索经验    联结知识    构建逻辑    审视遗漏    标准输出
```

---

## 二、每步详解

### 2.1 步骤一：意图分析

**目标**：理解用户真正想表达什么

**执行**：

1. **提取关键词**
   - 名词：故事的核心元素
   - 动词：动作和变化
   - 情感词：想要的效果

2. **识别诉求**
   - 想让观众：笑/哭/思考/恐惧/温暖
   - 受众是谁
   - 创作目的：娱乐/艺术/商业/实验

3. **判断格式**
   - 时长要求
   - 类型偏好

4. **选择方法论**
   - 商业片 → Snyder
   - 成长故事 → Harmon
   - 角色驱动 → McKee
   - 非线性 → 诺兰/昆汀

---

### 2.2 步骤二：记忆检索

**目标**：从大师案例中寻找参考

**检索范围**：

1. **Snyder节拍库** — 15节拍对应案例
2. **McKee结构库** — 价值转变案例
3. **Harmon圆环库** — 8步圆环案例
4. **Sorkin对话库** — 四重功能对话
5. **诺兰/昆汀库** — 非线性案例

---

### 2.3 步骤三：知识关联

**目标**：联结多学科知识

**关联领域**：

1. **戏剧结构** — 价值转变、鸿沟
2. **叙事范式** — 节拍、圆环
3. **类型美学** — 各类型爽点
4. **视觉语言** — 镜头语法
5. **对话艺术** — 潜台词设计

---

### 2.4 步骤四：推理决策

**目标**：构建具体方案

**决策内容**：

1. **结构选择** — 用哪种方法论
2. **人物设定** — Want/Need/Ghost
3. **冲突设计** — 核心冲突、升级路径
4. **节奏规划** — 张力分布、高潮位置

---

### 2.5 步骤五：反思校验

**目标**：审视方案合理性

**自检清单**：

**McKee检查**：
- [ ] 每个场景有价值转变吗？
- [ ] 鸿沟够大吗？

**Snyder检查**：
- [ ] 15节拍对齐了吗？
- [ ] "最后一幕转折"明确吗？

**Harmon检查**：
- [ ] 圆环完整吗？

**Sorkin检查**：
- [ ] 对话有四重功能吗？
- [ ] 每个角色有独特声音吗？

---

### 2.6 步骤六：精准输出

**目标**：标准格式呈现

**输出形式**：

1. **概念卡** — 项目概览、一句话概念
2. **人物卡** — 角色设计、动机/弧光
3. **结构卡** — 大纲、章节
4. **场景卡** — 分场景
5. **完整剧本** — 标准格式

## 四、轮次管理

### 4.1 轮次分配

```
第1-2轮：破题 + 确认概念
第3-4轮：梗概 + 确认人物
第5-6轮：结构 + 场景拆解
第7-8轮：写作 + 医生诊断
```

### 4.2 压缩触发

12轮后自动压缩关键信息，保持上下文连贯。

## 五、在工作流中的应用

| 工作流步骤 | 思维链重点 | 输出 |
|-----------|-----------|------|
| 1.破题 | 意图分析+记忆检索 | 理解用户真正想讲什么 |
| 2.梗概 | 知识关联+推理决策 | 构建故事基本框架 |
| 3.人物 | 推理决策+反思校验 | 设计可信的角色 |
| 4.前史/世界观 | 知识关联 | 建造完整世界 |
| 5.结构大纲 | 推理决策 | 规划叙事结构 |
| 6.场景拆解 | 推理决策+反思校验 | 拆分视听单元 |
| 7.场景写作 | 精准输出 | 标准格式输出 |
| 8.剧本医生 | 反思校验 | 全面诊断 |

---

## 六、自检触发点

每次输出前检查：

```
IF 输出包含：
  ✓ 新概念提出
  ✓ 新人物设计
  ✓ 新场景写作
  ✓ 结构大纲调整
  ✓ 格式变更
  
THEN 执行自检
```

## 七、模式切换

### 7.1 静默模式（默认）

- 思维链在后台执行
- 用户看不到推理过程
- 直接输出结果

### 7.2 透明模式（可选）

- 应用户要求显示思维链
- 展示每步思考过程
- 用于教学或解释
