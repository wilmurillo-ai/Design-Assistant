# Narrative Master 核心方法论

> 全球最佳编剧方法论融合

---

## 一、核心基石

### 1.1 故事的本质

**McKee：故事是价值的转变**
- 每个场景必须有价值转变
- 没有转变 = 没有戏剧

**Snyder：故事是节奏机器**
- 15个精确节拍
- 像时钟一样运转

**Harmon：故事是角色变形**
- 从A到B的完整变化
- 天然对称性

**诺兰：故事是结构的游戏**
- 用时间创造意义
- 结构本身就是内容

**昆汀：故事是意外的排列**
- 顺序改变 = 意义改变
- 身份揭示 = 故事转折

### 1.2 编剧的职责

- 建造完整世界
- 放进真实角色
- 设定不可回避的冲突
- 然后观察会发生什么

## 二、McKee 价值转变体系

### 2.1 核心概念

```
每个场景：价值A → 价值B（必须转变）
```

**价值类型表：**

| 价值 | 正向 | 负向 |
|------|-----|-----|
| 爱 | 爱、亲密 | 恨、疏远 |
| 自由 | 自由、解脱 | 囚禁、束缚 |
| 真相 | 真相、正义 | 谎言、欺骗 |
| 生命 | 生、安全 | 死、危险 |
| 秩序 | 秩序、和平 | 混乱、冲突 |
| 身份 | 认同、尊重 | 否认、羞辱 |
| 权力 | 控制、力量 | 无力、软弱 |

### 2.2 鸿沟概念

```
角色行动：期望得到结果A
实际结果：得到结果B
差距 = 鸿沟 = 戏剧张力
```

**鸿沟加深示例：**

1. 《教父》
   - 麦克期望：不使用暴力
   - 实际：必须使用暴力
   - 鸿沟：信念vs现实

2. 《黑暗骑士》
   - 蝙蝠侠期望：拯救瑞秋
   - 实际：瑞秋死了
   - 鸿沟：失败+背叛

## 四、Harmon 8步圆环

### 4.1 八步详解

```
步骤1：YOU — 舒适区
  建立角色的日常
  观众理解"这个人平时什么样"

步骤2：NEED — 产生欲望
  角色意识到缺少什么

步骤3：GO — 进入新世界
  跨过阈值，不可逆的决定

步骤4：SEARCH — 寻找/适应
  在新环境中试错

步骤5：FIND — 得到想要的
  达成表面目标，但有代价

步骤6：TAKE — 付出代价
  最低点，内外压力最大

步骤7：RETURN — 回到起点
  物理或心理回到起点

步骤8：CHANGE — 已改变
  展示变化后的新常态
```

### 4.2 圆环与节拍对应

| Harmon | Snyder | 内容 |
|--------|--------|------|
| 1 | 1-3 | YOU/Set-Up |
| 2 | 4 | NEED/Catalyst |
| 3 | 5-6 | GO/Break into Two |
| 4 | 8 | SEARCH/Fun and Games |
| 5 | 9 | FIND/Midpoint |
| 6 | 10-12 | TAKE/Bad Guys-All Is Lost |
| 7 | 13 | RETURN/Break into Three |
| 8 | 14-15 | CHANGE/Finale |

## 五、Sorkin 对话艺术

### 5.1 对话四重功能

1. **揭示角色** — 不是"说什么"，而是"怎么说"
2. **推进情节** — 对话本身就是行动
3. **传递信息** — 信息通过冲突传递
4. **建立节奏** — 像爵士乐有节奏感

### 5.2 "厨房sink"原则

角色边做事边聊天：
- 做饭、开车、走路时聊天
- 动作与对话并行

### 5.3 角色声音设计

**用词偏好**：律师用术语、军人用命令
**句式特点**：短句快/长句慢/爱用问句
**口头禅**：每个角色有标志用语

### 5.4 潜台词技巧

```
直白（错）："我爱你。"
潜台词（对）："明天天气不错。"
（暗示：想见面）
```

## 六、诺兰/昆汀 叙事技巧

### 6.1 诺兰非线性结构

**类型1：倒叙**
- 《记忆碎片》：倒叙+顺叙交替

**类型2：时间膨胀**
- 《盗梦空间》：梦中梦（越深越慢）

**类型3：多线并行**
- 《敦刻尔克》：陆/海/空三条线

**类型4：时间倒流**
- 《信条》：熵减概念

### 6.2 昆汀意外技巧

**意外时刻**：突然暴力打破轻松
**身份揭示**：最后揭示真实身份
**事件重组**：同一事件不同视角

## 七、人物设计

### 7.1 McKee三层结构

1. **Mask（外在表现）** — 给外界看到的
2. **Self（内在真实）** — 真实的自己
3. **Ghost（核心创伤）** — 塑造现在的根源

### 7.2 Want vs Need

- **Want**：外在目标（角色追求的）
- **Need**：内在需求（角色真正需要的）

### 7.3 弧光类型

- 成长弧光：克服恐惧，实现Need
- 堕落弧光：屈服恶魔，失去珍贵
- 转化弧光：改变看法，代价惨重
- 揭示弧光：认识真实自我

## 八、写作红线

### 绝对禁止

- ❌ 心理描写
- ❌ 括号暗示
- ❌ 角色解释设定/主题
- ❌ 说教/煽情/直白台词
- ❌ 无法视听化的内容

### 必须做到

- ✅ 台词口语化
- ✅ 动作代替解释
- ✅ 对话像冰山（只露一角）
- ✅ 所有内容可被摄影机拍摄

## 九、节奏控制

### 9.1 双轨节奏

**外部情节**：事件密度
- 紧：快节奏、高密度
- 松：日常、过渡

**内在情感**：角色情感
- 重：告白、失去
- 轻：调侃、闲聊

### 9.2 错位技法

- 情节紧+情感轻（紧张但幽默）
- 情节松+情感重（平静但深刻）
- 情节紧+情感重（高潮）
- 情节松+情感轻（呼吸）
