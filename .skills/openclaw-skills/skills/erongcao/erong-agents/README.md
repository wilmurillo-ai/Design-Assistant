## 📦 产品介绍

**erong-agents** 提供 49 个专业 AI Agent，分为 8 大业务部门，可独立完成专业任务或多 Agent 协作完成复杂项目。

**AI Agent 团队** 是一个完整的 AI 代理机构，包含 49 个专业 Agent，覆盖 8 大业务部门。每个 Agent 都有独特的人格，专业技能和完整的工作流程。

### 核心特点

- **🎯 专业化** - 每个 Agent 深耕特定领域，不是通用模板
- **🧠 人格化** - 独特的沟通风格和工作方式
- **📋 结果导向** - 每个 Agent 都有明确的交付物
- **✅ 开箱即用** - 完整的工作流和最佳实践
- **🆓 完全免费** - MIT 开源许可证

## 🚀 快速开始

### 安装

```bash
# 通过 ClawHub 安装（推荐）
openclaw skill install erong-agents

# 或手动安装
git clone https://github.com/erongcao/erong-agents.git
cp -r erong-agents ~/.openclaw/skills/
```

### 使用方式

#### 方式 1：单个 Agent

```bash
# 激活前端开发 Agent
/openclaw skill use erong-agents --agent frontend-developer "帮我创建一个 React 组件..."
```

#### 方式 2：多 Agent 协作

```bash
# 使用编排器自动调度多个 Agent
/openclaw skill use erong-agents --agent agents-orchestrator "帮我开发一个完整的 Web 应用..."
```

## 🎯 核心 Agent 示例

### Agents Orchestrator（编排器）

**核心功能**：自动调度多个 Agent 完成复杂项目

**工作流**：
```
1. 项目分析 → 2. 任务分解 → 3. Agent 调度 → 4. 质量检查 → 5. 交付
```

**适用场景**：
- 完整应用开发
- 多阶段项目
- 需要多部门协作的任务

---

### Frontend Developer（前端开发专家）

**核心功能**：现代 Web 应用开发

**技术栈**：
- React/Vue/Angular/Svelte
- TypeScript
- Tailwind CSS
- 性能优化
- 无障碍设计

**交付物**：
- 可运行的组件代码

- 无障碍检查报告
