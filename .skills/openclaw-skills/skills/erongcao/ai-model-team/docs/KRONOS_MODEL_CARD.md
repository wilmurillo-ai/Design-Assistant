## 模型概述

Kronos-base 是专为**加密货币K线数据**训练的时序预测模型，专注于识别庄家行为、散户情绪和技术形态。

### 核心能力

1. **价格方向预测** - 判断下一个周期涨跌
2. **趋势强度量化** - 0-100 趋势强度指标
3. **关键价位识别** - 支撑/阻力位预测
4. **庄家行为识别** - 检测吸筹/出货/洗盘模式

---

## 训练数据

| 数据类型 | 说明 |
|---------|------|
| **来源** | OKX、币安等主流交易所 |
| **品种** | BTC、ETH、SOL等主流加密货币 |
| **时间范围** | 2020-2024 |
| **周期** | 1H、4H、1D |
| **数据量** | ~50M 条K线数据 |

### 数据预处理

- UTC 时区统一
- 异常值清洗（插针、虚假波动）
- OHLC 关系校验
- 时间对齐（60秒容忍）

---

## 模型架构

```
输入: K线序列 (Open, High, Low, Close, Volume)
      │
      ▼
[Embedding Layer] - 可学习的价格编码
      │
      ▼
[Transformer Encoder] - 多头自注意力
      │
      ▼
[Prediction Head] - 输出信号/置信度/价位
输出: {
  signal: "bullish" | "neutral" | "bearish",
  confidence: 0-100,
  trend_strength: 0-100,
  forecast_price: float,
  support: float,
  resistance: float
}
```

## 使用限制

### ⚠️ 风险提示

1. **非投资建议**: Kronos 预测仅供参考，不构成投资建议
2. **模型漂移**: 加密货币市场特征随时间变化，模型可能漂移
3. **极端行情**: 黑天鹅事件、市场操纵可能导致预测失效
4. **数据依赖**: 预测质量依赖输入数据的完整性和准确性

### 适用场景

✅ **适合**:
- 多模型 Ensemble 的价格信号输入
- 趋势确认（与情绪/宏观指标配合）
- 支撑阻力位参考

❌ **不适合**:
- 单一决策依据
- 高频交易
- 极端波动期（战争/监管/黑天鹅）
