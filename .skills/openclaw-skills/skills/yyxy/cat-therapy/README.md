# 🐱 Cat Therapy - 治愈系撸猫技能

**工作累了？压力大？让可爱的小猫咪治愈你吧～**

---

## 💬 如何使用

### 触发词（说这些就能撸猫）

**中文：**
- 休息一下
- 累了
- 想撸猫
- 看看猫
- 压力大
- 放松一下
- 治愈我
- 喵～

**英文：**
- tired
- need a break
- show me cats
- cat pics
- stress
- relax
- meow

**示例：**
> 用户：休息一下
> 
> 🤖：*发送可爱猫咪图片 + 猫叫声 + 治愈语录*

---

## 🎨 自定义功能

### 上传你家主子的照片

1. 发送猫咪图片给我
2. 说："这是我的猫" 或 "保存这只猫"
3. 下次治愈时会优先使用你家主子的照片！

### 上传自定义猫叫声

发送语音或文字描述你家猫的叫声，例如：
> "这是我家猫的声音：喵～咪～"

### 恢复默认

说"重置猫咪"即可恢复随机猫咪模式。
