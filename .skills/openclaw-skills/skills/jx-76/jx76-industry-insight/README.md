# 行业洞察分析助手 (Industry Insight)

使用麦肯锡工作法，帮助产品经理快速了解任意行业。

## 功能特点

- 🔍 **关键词定义**：搜索行业上下游关键词，20-50个
- 📊 **报告推荐**：2024-2026年最新行业报告
- 📚 **书籍推荐**：入门到高级经典著作
- 🗺️ **思维导图**：Markdown格式体系化框架
- 💡 **洞察提炼**：3-5个战略级关键洞察
- 📦 **PM特供**：竞品分析、商业模式、增长机会

## 触发词

行业分析、行业洞察、行业研究、市场分析、行业调研、分析[行业名]

## 使用方法

当用户请求分析某个行业时，激活此skill，按流程执行：
1. 引导确认（ Initialization）
2. 关键词定义（Keywords）
3. 行业报告搜索（Reports）
4. 经典书籍推荐（Books）
5. 思维导图设计（Mind Map）
6. 关键洞察输出（Insights）

## 工具调用

- web_search：搜索关键词、报告、书籍
- web_fetch：获取报告详细内容
- browser：访问行业网站

## 输出示例

```
# 智能汽车行业分析报告

## 1. 行业关键词库
## 2. 推荐行业报告（2024-2026）
## 3. 经典书籍推荐
## 4. 行业分析思维导图
## 5. 关键洞察与战略建议
## 6. 产品经理视角分析（可选）
```

## 适用场景

- 产品经理准备新行业/新赛道分析
- 求职者快速了解目标行业
- 投资决策前的行业调研
- 职业规划/转型前的行业了解
