## 🔐 安全和隐私

### 认证要求

本技能需要 Digilifeform 账户的 API Key。

**获取方式：**
1. 访问 https://datu.digilifeform.com 注册账户
2. 进入 API 设置获取 API Key
3. 在 ClawHub 中配置环境变量 `DATU_API_KEY`

**新用户福利：** 注册赠送 20 积分

### 数据隐私

- ✅ 用户数据仅用于生成任务
- ✅ 生成的海报 30 天后自动删除
- ✅ 数据不用于训练模型
- ✅ 数据不与第三方共享

**建议：** 先用非敏感数据测试，确认工作流程后再处理机密文件。

详见 [SECURITY.md](SECURITY.md) 和 [隐私政策](https://datu.digilifeform.com/privacy)
