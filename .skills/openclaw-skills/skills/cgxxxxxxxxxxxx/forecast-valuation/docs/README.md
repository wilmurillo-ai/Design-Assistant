# Forecast & Valuation Skill

专业财务预测与估值模型生成器

## 快速开始

### 1. 配置（可选）

```bash
python3 scripts/configure.py
```

配置数据源（Gangtise/Tushare）和默认估值参数。

### 2. 生成模型

```bash
# 基本用法
python3 scripts/build_forecast.py "福耀玻璃" "600660.SH"

# 指定输出路径
python3 scripts/build_forecast.py "福耀玻璃" "600660.SH" \
  --output "福耀玻璃_估值模型.xlsx"

# 指定行业（用于自动筛选可比公司）
python3 scripts/build_forecast.py "福耀玻璃" "600660.SH" \
  --industry "汽车零部件"
```

## 输出内容

生成的 Excel 模型包含 10 个工作表：

| 序号 | 工作表 | 说明 |
|------|--------|------|
| 1 | 封面 | 公司基本信息、报告日期 |
| 2 | 历史报表 (简化) | 5 年历史财务数据 |
| 3 | 基本假设 | 收入驱动、利润率、资本配置假设 |
| 4 | 利润表预测 | 5 年利润表预测 |
| 5 | DCF 估值 | WACC 计算、自由现金流折现 |
| 6 | 可比公司估值 | 8-12 家同业对比 |
| 7 | 敏感性分析 | WACC vs g 敏感性矩阵 |
| 8 | Football Field | 多方法估值区间可视化 |
| 9 | 预测合理性检验 | 自动质量检验报告 |

## 模型特点

- ✅ **专业标准** - 融合高盛 DCF 框架 + Wind Evaluator 完整性
- ✅ **自动配平** - 资产负债表自动平衡检查
- ✅ **智能筛选** - 自动筛选可比公司
- ✅ **质量检验** - 预测合理性自动检验
