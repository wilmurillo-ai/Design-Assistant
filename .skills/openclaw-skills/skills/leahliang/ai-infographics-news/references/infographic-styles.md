# 信息图风格参考

> 当用户指定特定视觉风格时，用对应模板替换 SKILL.md 第四步的 DESIGN DETAILS 段落。

## 1. 暗色科技风（默认）
```
Background: deep navy (#0a0e1a) to black gradient
Cards: dark charcoal (#1a1f2e), colored left border accent
Colors: cyan #00f5ff, electric blue #0066ff, purple #8b5cf6
Keywords: "tech dashboard, data viz aesthetic, subtle grid lines, 
  neon glow, minimalist icons, professional editorial"
```

## 2. 亮色简洁风
触发词：`"简洁"` / `"白色"` / `"light"` / `"clean"`
```
Background: white to light gray (#f8fafc)
Cards: white with subtle drop shadow, rounded corners
Colors: deep blue #1d4ed8, emerald #059669, amber #d97706
Keywords: "clean minimal, Material Design style, flat icons, 
  generous whitespace, professional business infographic"
```

## 3. 报纸复古风
触发词：`"报纸"` / `"复古"` / `"newspaper"` / `"editorial"`
```
Background: aged cream (#faf7f0), subtle paper texture
Cards: white with thin black borders, column layout
Colors: black, dark red #991b1b, gold #b45309
Keywords: "vintage newspaper, editorial print design, ink texture, 
  serif headlines, masthead title, breaking news stamps, column grid"
```

## 4. 赛博朋克霓虹风
触发词：`"赛博朋克"` / `"霓虹"` / `"cyberpunk"` / `"neon"`
```
Background: pure black (#000000)
Cards: transparent with glowing neon borders
Colors: hot pink #ff0080, electric yellow #ffff00, toxic green #00ff41
Keywords: "cyberpunk, matrix glitch, neon glow borders, scanline overlay,
  dystopian tech poster, blade runner aesthetic, uppercase monospace font"
```

## 5. 柔和马卡龙风
触发词：`"可爱"` / `"柔和"` / `"pastel"` / `"粉色"`
```
Background: soft lavender (#f3f0ff) or mint (#f0fdf4)
Cards: pastel colored (pink, yellow, blue, green — muted tones)
Colors: coral #fb7185, sky blue #38bdf8, mint #34d399
Keywords: "cute pastel infographic, rounded bubble cards, playful icons,
  Gen-Z social media style, kawaii-inspired, soft gradients, friendly"
```

## 尺寸建议

| 场景 | 比例 | 说明 |
|------|------|------|
| 手机/Stories/微信 | 9:16 | 竖版，优先推荐 |
| Twitter/桌面宽图 | 16:9 | 横版 |
| 通用方图 | 1:1 | 所有平台通用 |
| Pinterest/详细长图 | 2:3 | 条目多时使用 |

## 条目数量建议

- **4–6 条**：内容宽松，文字大，可读性最佳
- **7–9 条**：标准密度，适合"本周速览"
- **10+ 条**：改用三栏布局，每张卡只显示标题+日期
