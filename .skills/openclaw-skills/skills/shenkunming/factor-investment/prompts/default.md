# 因子投资量化选股系统

## 系统概述

基于《因子投资：方法与实践》的专业量化选股系统，融合经典Fama-French多因子模型与中国A股市场实证研究，提供完整的选股、评分、复盘解决方案。

## 一、核心因子体系

### 1.1 Fama-French五因子模型

| 因子 | 定义 | A股实证效果 |
|------|------|-------------|
| **市场因子 (MKT)** | Rm - Rf | 核心风险溢价 |
| **规模因子 (SMB)** | 小市值 - 大市值 | 显著正收益 |
| **价值因子 (HML)** | 高账面市值比 - 低账面市值比 | 弱正效应 |
| **盈利因子 (RMW)** | 高盈利 - 低盈利 | 显著正收益 |
| **投资因子 (CMA)** | 低投资 - 高投资 | 显著正收益 |

### 1.2 技术面核心因子

| 因子类别 | 具体指标 | 权重 |
|----------|----------|------|
| **均线系统** | MA5 > MA10 > MA20 > MA60 | 核心条件 |
| **动量指标** | RSI (40-75健康区间) | 核心条件 |
| **量能因子** | 成交量 > 1.5×MA5(VOL) | 核心条件 |
| **趋势指标** | 价格 > MA5, 价格 > MA60 | 核心条件 |

## 二、量化选股5大核心条件

### 条件A：均线多头排列
```
MA5 > MA10 > MA20
```
**含义**：短期均线向上发散，表明股价处于上升趋势

### 条件B：突破MA60
```
收盘价 > MA60
```
**含义**：股价突破中长期均线，中期趋势转强

### 条件C：放量确认
```
成交量 > 1.5 × MA5(VOL)
```
**含义**：量能突破1.5倍短期均量，确认趋势有效性

### 条件D：趋势加速
```
收盘价 > MA5
```
**含义**：价格运行在短期均线上方，趋势加速中

### 条件E：RSI健康区间
```
40 ≤ RSI ≤ 75
```
**含义**：RSI处于健康区间，既不过度超买也不超卖

## 四、集合竞价选股因子

### 4.1 高频量能因子 (OCVP)

```python
OCVP = 开盘竞价成交量 / 全日预计成交量
```

**实战应用**：
- OCVP > 0.1：竞价活跃
- OCVP > 0.15：竞价爆量

### 4.2 竞价涨幅因子

```python
竞价涨幅 = (竞价价格 - 前收盘价) / 前收盘价 × 100%
```

**筛选标准**：
- 最佳区间：2% - 5%
- 过高风险：>7% 可能高开低走
- 过低无效：<1% 动能不足

### 4.3 集合竞价选股条件

1. **竞价涨幅**：2% - 5%
2. **竞价爆量**：成交量 > 昨日均量 × 1.5
3. **30日趋势**：均线多头排列
4. **板块过滤**：主板为主，规避ST

## 五、盘中实时监控因子

### 5.1 分时量价背离
```python
背离信号 = 价格创新高 AND 成交量未创新高
```

### 5.2 RSI动态监控
- RSI > 80：超买警戒，考虑止盈
- RSI < 30：超卖区域，关注反弹机会
- RSI > 75：进入止盈观察区

### 5.3 均线偏离度
```python
偏离度 = (MA5 - MA20) / MA20 × 100%
```
- 偏离度 > 5%：乖离过大，注意回调风险
- 偏离度 2-5%：健康发散
- 偏离度 < 2%：动能减弱
