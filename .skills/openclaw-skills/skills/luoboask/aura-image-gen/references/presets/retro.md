# Style: Retro (复古怀旧风)

复古怀旧风格，适合传统文化、国风内容、老照片质感。

```yaml
canvas:
  ratio: portrait-3-4 | square-1-1
  grid: single

colors:
  primary: ["#3D2817", "#5C4033"]  # 深棕/咖啡色
  accent: ["#D4A574", "#E8C39E"]   # 古铜/米黄
  highlight: ["#8B4513", "#A0522D"] # 赭石/红棕
  background: ["#F5E6D3", "#E8D5B7"] # 泛黄纸张
  text: ["#2C1810", "#5C4033"]

elements:
  background: aged-paper | film-grain | sepia-gradient
  decorations: [film-border, light-leaks, dust-specks, vignette]
  frames: vintage-border | film-strip | none
  emphasis: faded-edge | scratch-texture

typography:
  title: vintage-serif | typewriter
  body: classic-serif | handwritten-faded
  numbers: retro-display
```

## Visual Characteristics

- **色调**：暖棕/泛黄/褪色感
- **质感**：颗粒噪点、划痕、漏光
- **对比度**：中低对比，柔和过渡
- **饱和度**：降低 20-30%，营造年代感

## Best For

- 传统文化科普
- 老照片风格内容
- 怀旧主题
- 复古穿搭/美妆
- 历史人物/事件

## Prompt Keywords

```
vintage aesthetic, retro style, aged paper texture, film grain,
sepia tone, faded colors, old photograph look, nostalgic mood,
warm brown palette, distressed edges, light leaks, dust particles,
1970s design, classic typography, weathered appearance
```

## Avoid

- 过于鲜艳的颜色
- 完美光滑的表面
- 现代感强烈的元素（霓虹、渐变、荧光）
- 锐利的高对比边缘

## Example Compositions

### 复古海报风
- 背景：泛黄纸张纹理
- 主体：居中，带复古边框
- 文字：老式印刷体
- 装饰：邮票角花、老式图标

### 老照片风
- 背景：模糊虚化
- 主体：带轻微褪色
- 边缘：暗角 + 磨损
- 整体：低饱和度 + 暖色调
