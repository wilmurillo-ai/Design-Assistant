# Tarot Style (塔罗艺术风)

Art Nouveau borders, jewel tones, symbolic imagery.

```yaml
canvas:
  ratio: portrait-3-4
  grid: single

colors:
  primary: ["#1B0A2E", "#2D1B4E"]
  accent: ["#D4AF37", "#B8860B"]
  highlight: ["#8B0000", "#4A0080"]
  text: ["#F5E6CC", "#FFFFFF"]

elements:
  background: velvet-dark | ornate-pattern
  decorations: [art-nouveau-border, celestial-symbols, card-frame]
  frames: tarot-border | ornate-gold
  emphasis: gold-foil | emboss

typography:
  title: art-nouveau | display-serif
  body: elegant-serif
  card-name: all-caps-spaced
```

Best layout pairings: card (single card), spread (牌阵), sparse (daily card)
