# Bold Style (高冲击力风)

```yaml
canvas:
  ratio: portrait-3-4

colors:
  primary: ["#FF0000", "#FFD700", "#000000"]
  background: ["#FFFFFF", "#000000"]
  text: ["#FFFFFF", "#FF0000"]

elements:
  background: solid-contrast | diagonal-split
  decorations: [exclamation-marks, warning-icons, arrows-bold]
  frames: thick-border | none
  emphasis: highlight-box | underline-heavy

typography:
  title: ultra-bold-sans
  body: bold-sans
  numbers: oversized-display
```

Best layout pairings: list (warnings), comparison (vs), sparse (impact covers)
