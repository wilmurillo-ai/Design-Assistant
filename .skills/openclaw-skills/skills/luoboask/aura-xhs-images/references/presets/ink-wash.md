# Ink-wash Style (水墨国风)

Chinese ink wash painting aesthetic. For traditional culture, star mansion beasts.

```yaml
canvas:
  ratio: portrait-3-4

colors:
  primary: ["#1A1A1A", "#4A4A4A"]
  accent: ["#8B0000", "#C41E3A"]
  background: ["#F5F0E8", "#E8DCC8"]
  text: ["#2C2C2C", "#8B0000"]

elements:
  background: rice-paper-texture | ink-splash
  decorations: [brush-strokes, seal-stamp, ink-dots]
  frames: scroll-border | none
  emphasis: red-seal | brush-circle

typography:
  title: brush-calligraphy
  body: kai-style
```

Best layout pairings: card (beast cards), balanced (cultural content), sparse (quotes)
