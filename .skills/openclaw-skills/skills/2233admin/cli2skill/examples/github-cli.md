# github-cli

Work seamlessly with GitHub from the command line.

## Commands

```bash
gh auth
gh browse
gh codespace
gh gist
gh issue
gh org
gh pr
gh project
gh release
gh repo
gh cache
gh run
gh workflow
gh co
gh agent-task
gh alias
gh api
gh attestation
gh completion
gh config
gh copilot
gh extension
gh gpg-key
gh label
gh licenses
gh preview
gh ruleset
gh search
gh secret
gh ssh-key
gh status
gh variable
```

### auth
Authenticate gh and git with GitHub

### browse
Open repositories, issues, pull requests, and more in the browser

### codespace
Connect to and manage codespaces

### gist
Manage gists

### issue
Manage issues

### org
Manage organizations

### pr
Manage pull requests

### project
Work with GitHub Projects.

### release
Manage releases

### repo
Manage repositories

### cache
Manage GitHub Actions caches

### run
View details about workflow runs

### workflow
View details about GitHub Actions workflows

### co
Alias for "pr checkout"

### agent-task
Work with agent tasks (preview)

### alias
Create command shortcuts

### api
Make an authenticated GitHub API request

### attestation
Work with artifact attestations

### completion
Generate shell completion scripts

### config
Manage configuration for gh

### copilot
Run the GitHub Copilot CLI (preview)

### extension
Manage gh extensions

### gpg-key
Manage GPG keys

### label
Manage labels

### licenses
View third-party license information

### preview
Execute previews for gh features

### ruleset
View info about repo rulesets

### search
Search for repositories, issues, and pull requests

### secret
Manage GitHub secrets

### ssh-key
Manage SSH keys

### status
Print information about relevant issues, pull requests, and notifications across repositories

### variable
Manage GitHub Actions variables

## When to use
- Work seamlessly with GitHub from the command line.
- Available commands: `auth`, `browse`, `codespace`, `gist`, `issue`
