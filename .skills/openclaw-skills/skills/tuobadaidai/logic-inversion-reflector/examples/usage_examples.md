## 使用建议

### 最佳实践
1. **在重大决策前使用** - 不要用于日常小事
2. **保持开放心态** - 被挑战时不要防御
3. **记录反思** - 将 LIR 的输出记录到知识库
4. **团队共用** - 让整个团队习惯被挑战

### 避免的陷阱
1. **分析瘫痪** - 不要无限期地质疑
2. **为反而反** - LIR 不是抬杠，是深度思考
3. **忽视直觉** - 系统 2 重要，系统 1 也有价值
4. **过度使用** - 每天 1-2 次足矣
