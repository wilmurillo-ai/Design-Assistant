# Changelog

## [1.0.0] - 2026-02-19

### Added
- Docker operations: container status, log tailing with grep, health summary, Compose status
- Process management: list by CPU/memory, port inspection, zombie detection
- Log analysis: pattern matching, tail with highlighting, frequency analysis
- Health monitoring: HTTP endpoint checks, port scanning, system resource health
- Quick diagnostics (`diag`): one-command full system health report
- Unicode box-drawing terminal output with ANSI colours
- Graceful degradation when Docker or system utilities are unavailable
