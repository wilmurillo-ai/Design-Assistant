# CacheForge Ops

Operational commands for active CacheForge tenants:

- Balance and billing status
- Top-up links (`stripe` or `crypto`)
- Auto top-up configuration
- Upstream provider management
- Tenant API key lifecycle

Use this skill for day-2 account operations after onboarding.

## What It Does

This skill exposes account operations so agents can manage CacheForge tenants without leaving the terminal workflow.
