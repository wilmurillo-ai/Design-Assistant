# 双色球选号系统

本目录用于给 Jaben 自用的双色球选号分析与出号。

## 目录结构

- `config.json`：系统参数
- `data/ssq_history.csv`：历史开奖数据
- `scripts/update_ssq_history.py`：更新双色球历史开奖数据
- `scripts/generate_ssq.py`：选号脚本
- `outputs/`：每次生成的结果
- `docs/strategy.md`：选号策略说明

## 当前定位

- 仅做**选号辅助**
- 不承诺中奖
- 强调：固定规则、可复盘、可持续优化

## 当前进度

- 已接入双色球真实历史开奖数据
- 当前历史数据范围：`2003-02-23` 至 `2026-03-17`（共 3426 期）
- 选号系统已升级至 **v2**：
  - 动态冷热周期（10/20/30 期加权）
  - AC 值、极距、尾数和形态筛选
  - 蓝球细化（分区 + 奇偶 + 遗漏）
  - 多策略支持（稳健/均衡/激进）
  - 近期走势动态加权
  - 候选池扩大至 3000 注
- 当前可本地生成 **2 注主推 + 3 注备选**（共 5 注）
- 已新增基础回测脚本，可测试规则在历史数据上的表现
- 回测结果（滑动窗口 50 期，共 3376 期测试）：
  - 红球≥3 匹配：21.9%
  - 红球≥4 匹配：2.2%
  - 红球≥5 匹配：0.09%
  - 蓝球命中：11.8%

## 使用方法

### 1. 更新历史开奖数据

```bash
python3 lottery-ssq/scripts/update_ssq_history.py
```

### 2. 生成当期选号结果

```bash
python3 lottery-ssq/scripts/generate_ssq.py
```

生成结果会保存到：
- `lottery-ssq/outputs/`
