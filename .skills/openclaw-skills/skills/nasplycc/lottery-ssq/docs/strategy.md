# 双色球选号策略 v2

## 目标

建立一套稳定、可复盘、可调参的双色球选号辅助系统。

---

## v2 核心改进

### 1. 动态冷热周期
- **热号**：近 10 期（权重×3）
- **温号**：近 11-20 期（权重×2）
- **冷号**：近 21-30 期（权重×1）
- **综合频率** = 近期×3 + 中期×2 + 远期×1

### 2. 形态筛选增强
| 指标 | 范围 | 说明 |
|------|------|------|
| **AC 值** | 4-10 | 不同差值个数 - 5，反映复杂度 |
| **极距** | 18-30 | 最大号 - 最小号 |
| **尾数和** | 10-25 | 6 个红球个位数之和 |

### 3. 蓝球细化
- **分区**：1 区 (1-8) / 2 区 (9-16)
- **奇偶**：单独评分
- **遗漏**：≥10 期有回补加分

### 4. 多策略支持
| 策略 | 特点 | 适合场景 |
|------|------|----------|
| **stable (稳健)** | 热号×1.1, 温号×1.2, 冷号×0.95 | 追求稳定结构 |
| **balanced (均衡)** | 热号×1.0, 温号×1.15, 冷号×1.05 | 默认策略 |
| **aggressive (激进)** | 热号×0.95, 温号×1.0, 冷号×1.2 | 博冷号回补 |

### 5. 近期走势动态加权
- 分析近 10 期连号/重号趋势
- 如果近期连号多 → 适当青睐连号组合
- 如果近期重号多 → 适当青睐重号组合

### 6. 扩大候选池
- 800 注 → **3000 注**
- 高分注质量更高，多样性更好

## 评分公式（v2）

从 100 分起步，连乘调整：

| 维度 | 条件 | 系数 |
|------|------|------|
| **奇偶比** | 3:3, 4:2, 2:4 | ×1.2~1.25 |
| **分区分布** | 2-2-2, 1-2-3 等 | ×1.2~1.25 |
| **和值** | 90-120 | ×1.08 |
| **极距** | 18-30 | ×1.05 |
| **AC 值** | 4-10 | ×1.05 |
| **尾数和** | 10-25 | ×1.03 |
| **连号** | >2 组 | ×0.85~0.95 |
| **温号** | 每个 | ×1.0~1.2 |
| **冷号** | 每个 | ×0.95~1.2 |
| **遗漏≥10 期** | 每个 | ×1.05~1.25 |
| **蓝球-温** | 温蓝 | ×1.06 |
| **蓝球-遗漏≥10** | 是 | ×1.08 |

---

## 输出说明

每期默认输出：
- **2 注主推**（评分最高）
- **3 注备选**（评分次高）

每注包含：
- 红球 6 码 + 蓝球 1 码
- 评分
- 奇偶比、分区、和值
- 极距、AC 值

---

## 注意

彩票是随机事件，本系统是纪律化选号工具，不是预测工具。

所有改进仅基于历史统计规律，不保证中奖。
