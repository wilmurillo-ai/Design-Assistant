# DiagForge Bootstrap Skill

This folder is the ClawHub publishing package for a lightweight DiagForge bootstrap skill.

It intentionally contains only guidance files.
The full DiagForge system remains in the GitHub repository:

`https://github.com/qweadzchn/DiagForge`

## Publish intent

This skill is meant to:

- point agents to the correct repository
- reduce onboarding confusion
- teach the first safe validation path
- avoid copying the full project into ClawHub

## Files

- `SKILL.md`
  - the actual ClawHub skill entrypoint
- `README.md`
  - short package explanation
- `QUICKSTART.md`
  - short publish and usage notes
