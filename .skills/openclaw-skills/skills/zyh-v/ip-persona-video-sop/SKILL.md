# IP 人设视频打造 SOP

## 触发短语
- "打造 IP 人设"
- "IP 视频全流程"
- "人设视频怎么做"
- "IP 访谈提纲"
- "人设脚本怎么写"
- "IP 视频拍摄流程"

## 适用场景
- 个人 IP 从 0 到 1 打造
- 企业创始人 IP 化
- 专业人士（律师/医生/咨询师）个人品牌
- 实体店老板 IP 转型
- 知识博主内容升级

## 不適用场景
- ❌ 纯产品展示视频（无人设故事）
- ❌ 企业宣传片（非个人 IP）
- ❌ 短期热点蹭流量内容
- ❌ IP 本人拒绝出镜/讲故事

## 工作流程

### 前置检查
- [ ] IP 本人有意愿出镜和分享故事
- [ ] 有明确的产品/服务/商业模式
- [ ] 预留至少 3 周完整周期
- [ ] 有拍摄和剪辑资源（或外包）

---

### 阶段 1：查 - 访谈前调研（3 天）

#### 1.1 找参考（1 天）
**目标**：建立行业标杆认知

**执行步骤**：
1. 搜索同赛道 IP 账号（粉丝数从几万到千万）
2. 分析高赞/高评论/高转发内容
   - 拍摄形式（口播/剧情/ documentary）
   - 呈现风格（专业/亲民/反差）
   - 叙事结构（逆袭/干货/故事）
3. 研究商业模式和产品销量
   - 变现方式（课程/咨询/带货/会员）
   - 价格带和销量估算

**产出**：竞品分析报告（含账号列表 + 爆款拆解）

**工具**：
- 抖音/小红书/视频号创作者后台
- 蝉妈妈/灰豚数据/新榜
- 飞书表格整理

---

#### 1.2 看行业（1 天）
**目标**：理解行业全貌和机会点

**执行步骤**：
1. 用 AI 快速了解行业（DeepSeek/ChatGPT）
   - 行业规模和发展趋势
   - 主要玩家和竞争格局
   - 用户画像和消费决策链路
2. 识别行业痛点和未满足需求
3. 寻找 IP 的潜在切入点

**产出**：行业分析报告

**提示词示例**：
```
你是行业分析师，请帮我分析 [XX 行业]：
1. 行业规模和近 3 年发展趋势
2. 主要细分赛道和代表公司
3. 用户画像和典型购买决策流程
4. 行业痛点和未满足需求
5. 未来 3 年机会点
```

---

#### 1.3 问客户（1 天）
**目标**：从用户视角理解需求

**执行步骤**：
1. 大量刷相关短视频，重点看评论区
   - 用户在问什么问题？
   - 什么内容引发共鸣/争议？
   - 有哪些需求未被满足？
2. 用 AI 分析评论数据
3. 整理客户需求清单

**产出**：客户需求洞察报告

---

#### 1.4 探过往（1 天）
**目标**：收集 IP 故事素材

**执行步骤**：
1. 看 IP 过去的朋友圈、短视频、文章
2. 了解 IP 的服务过程和用户反馈
3. 匿名走访 IP 的实体店/公司（如适用）
4. 收集照片、视频、截图等素材

**产出**：IP 素材库（按时间线/主题整理）

---

### 阶段 2：访 - 深度访谈（2 天）

#### 2.1 问成长（1 天）
**目标**：挖掘人设故事和价值观

**访谈提纲**：
1. 从小到大的关键转折点
   - 当时发生了什么？
   - 你的情绪和想法？
   - 对现在事业和价值观的影响？
2. 情绪波动最大/记忆最深刻的事件
   - 具体场景、人物、对话
   - 当时的选择和结果
   - 现在的反思和理解

**技巧**：
- 追问细节（时间、地点、人物、对话、感受）
- 捕捉情绪词（"那时候真的很难"、"特别不甘心"）
- 记录金句（原话记录，不要概括）

**产出**：成长故事素材库

---

#### 2.2 问事业（1 天）
**目标**：理解事业动机和差异化

**访谈提纲**：
1. 选择现在事业的契机和原因
2. 做这项事业的核心理念
3. 你的差异化优势是什么
4. 现在的商业模式（画布）
   - 客户是谁？
   - 价值主张？
   - 收入来源？
   - 成本结构？
5. 商业模式的优势和劣势
6. 目前营收情况（区间即可）

**产出**：
- 人设标签（3-5 个关键词）
- 商业模式分析
- 产品建议
- 初步选题 15 个

---

#### 2.3 问产品
**目标**：理解产品卖点和市场匹配

**访谈提纲**：
1. 产品的核心卖点（USP）
2. 是否符合真需求？是否需要迭代？
3. 产品的差异化优势
4. 销售情况（销量、复购率、用户反馈）

**产出**：产品定位建议

---

### 阶段 3：写 - 人设脚本撰写（2 周）

#### 3.1 抓立意（1 天）
**目标**：提炼核心价值观

**执行步骤**：
1. 从访谈内容总结 IP 的一句话理念/价值观
2. 这个价值观作为故事骨架贯穿始终
3. 形成 3-5 个金句，在故事中反复出现

**示例**：
- "普通人也能靠 XX 翻身"
- "XX 行业没有秘密，只有 XX"
- "我用了 X 年，验证了一个道理：XX"

**产出**：核心价值观 + 金句列表

---

#### 3.2 搭骨架（1 天）
**目标**：设计故事结构和情绪曲线

**执行步骤**：
1. 选取 3-5 个关键生命故事
2. 设计情绪曲线（低落→高涨→颓废→怀疑→逆袭）
3. 每个故事对应一个情绪刺点

**情绪曲线示例**：
```
开场（高光时刻）→ 回忆（低谷）→ 转折（遇到 XX）→ 
奋斗（困难重重）→ 突破（关键领悟）→ 结果（逆袭）→ 
升华（价值观输出）
```

**产出**：故事大纲 + 情绪曲线图

---

#### 3.3 扩内容（7 天）
**目标**：完成初稿并让 IP 修改

**执行步骤**：
1. 根据骨架扩写初稿（口语化表达）
2. 给到 IP，让其根据亲身经历修改
3. IP 改第一版（重点：真实经历、表达方式）

**技巧**：
- 用 IP 的原话，不要书面语
- 保留口头禅和语气词
- 每段不超过 3 句话（便于拍摄）

**产出**：IP 修改后的初稿

---

#### 3.4 改表达（5 天）
**目标**：优化情绪起伏和逻辑性

**执行步骤**：
1. 操盘手/编导修改
   - 情绪起伏是否足够？
   - 内容逻辑是否通顺？
   - 表达是否口语化？
2. 检查金句出现频率（至少 3 次）
3. 确保开头抓人、结尾有意犹未尽感

**产出**：编导修改版

---

#### 3.5 定终稿（1 天）
**目标**：IP 最终确认

**执行步骤**：
1. IP 在修改版基础上再做一轮表达修改
2. 确认每一句话都说得出口、符合人设
3. 最终定稿

**产出**：人设脚本逐字稿（终稿）

---

### 阶段 4：拍 - 拍摄准备（0.5 天）

#### 4.1 备场景
**选择原则**：
- IP 舒服自在且有安全感
- 干净整洁有绿植
- 可选：店面/公司/工厂/户外

**执行**：
- IP 到场前操盘手先试光试景
- 准备备选场景（以防天气/环境问题）

---

#### 4.2 备器材
**清单**：
- 室内：补光灯（面光 + 背光）
- 户外：便携反光板
- 拍摄设备：Pocket 3（手持或支架）
- 收声麦、充电设备

**原则**：减少设备存在感，让 IP 放松

---

#### 4.3 备文本
- IP 提前背记文本（熟悉熟悉再熟悉）
- 打印脚本（用于现场改动和温习）

---

#### 4.4 备服装
- 深色系、浅色系各 1 套
- 柔软面料、挺括面料各 1 套
- 根据场景调整

---

#### 4.5 备妆容
- 女性 IP 提前联系化妆师
- 提前 2 小时到达场地

---

#### 4.6 备道具
- 抱枕、笔、杯子、桌子等
- **不要让 IP 双手空着**（容易不知所措）

---

### 阶段 5：拍 - 拍摄进行（0.5 天）

#### 5.1 找状态
**执行**：
1. 开始前聊天、喝咖啡、点香薰放松
2. 强调：放弃一遍过的要求
   - 可以反复录
   - 犯错没关系
   - 可以停顿接着说
   - 都能剪辑

---

#### 5.2 分段录
**执行**：
1. 一个小故事一个小故事录
2. 边录边根据现场表达调整文案
3. 每个故事反复录，找最舒服的情绪状态

---

#### 5.3 及时查
**执行**：
1. 录完第一段就看回放
2. 检查：收声、画质、情绪表现
3. 有问题立刻重录

**产出**：分段的人设视频素材

---

### 阶段 6：剪 - 视频剪辑（10 天）

#### 6.1 剪内容（2 天）
**执行**：
1. 导入剪映，剪到只保留绝对必要的故事
2. 字幕排版：
   - 不要换行
   - 一行最多 10 个字
   - 规避敏感词
3. 画面调整（亮度、对比度、色彩）

**产出**：视频 1.0

---

#### 6.2 加证据（2 天）
**执行**：
1. 找客户要素材：
   - 故事相关的照片、视频、截图
   - 情绪饱满的生活记录
2. 添加到对应故事情节内

**产出**：视频 2.0

---

#### 6.3 添素材（3 天）
**执行**：
1. 证据缺口从新片场下载素材
2. 素材类型穿插：
   - 远景、中景、近景
   - 第一视角
   - 横屏、竖屏
   - 鲜艳、褪色
3. 抵抗视觉疲劳

**产出**：视频 3.0

---

#### 6.4 促感官（3 天）
**执行**：
1. 环境音效、背景音乐配合
2. 画面转场和缩放
3. **开头素材一定要抓人**（前 3 秒定生死）
4. **结尾音乐和文字要有意犹未尽感**

**产出**：视频终稿

---

### 阶段 7：发 - 视频发布（1 天）

#### 7.1 先预览
- 发布前电脑导入预览
- 不同平台剪裁比例不同
- 避免画面不完整

---

#### 7.2 选吉时
- 选择黄道吉日发布（正念植入）

---

#### 7.3 发评论
- 操盘手带头用金句夸 IP
- 团队其他人帮忙评论点赞

---

#### 7.4 弹弹幕
- 操盘手带头发弹幕
- 有第一条就会有人跟发

---

#### 7.5 求转发
- IP 准备真诚话语
- 发给亲朋好友和业务相关人
- 完成冷启动

---

#### 7.6 引私域
- 置顶链接微信号
- 引导看评论区加微信（视频号）
- 其他平台用私信/粉丝群引流

**产出**：发布视频 + 冷启动数据

---

## Guardrails

### 不做的事情
- ❌ 不编造 IP 没有的经历
- ❌ 不用过度夸张的情绪
- ❌ 不承诺具体涨粉/变现数字
- ❌ 不直接复制竞品内容
- ❌ 不忽视 IP 本人的表达习惯

### 需要确认的事情
- ⚠️ IP 本人是否愿意分享某个故事（涉及隐私）
- ⚠️ 商业模式描述是否准确（避免夸大）
- ⚠️ 发布前 IP 是否确认终稿
- ⚠️ 引流方式是否符合平台规则

### 风险控制
- **隐私保护**：敏感信息打码/模糊处理
- **合规风险**：规避医疗/金融等敏感词
- **舆情风险**：避免争议性话题（除非是人设一部分）
