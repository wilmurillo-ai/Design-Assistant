# 头条号自动化运营技能包 - toutiao-agent

**版本**: v1.0.0  
**作者**: 小虾米  
**定价**: 9.9 元/月  
**类别**: 内容创作/自媒体运营

---

## 🎯 产品定位

帮助菜价/民生类自媒体创作者**自动化运营头条号**，实现**日均收益 1 元+**的被动收入。

---

## ✨ 核心功能

### 1. 自动化内容发布
- ⏰ **4 时段定时发布**: 06:15/11:15/15:45/20:00
- 📝 **内容模板库**: 早市菜价/渠道对比/互动问答/活动话题
- 🏷️ **活动标签匹配**: 自动添加#晒出你的本地生活#等活动标签
- 📍 **定位自动添加**: 上海市/嘉定区等精准定位

### 2. 菜价数据追踪
- 📊 **每日价格采集**: 自动记录菜场价格数据
- 📈 **趋势分析**: 7 天/14 天/30 天价格趋势图
- 💡 **采购建议**: 自动生成购买建议文案

### 3. 收益优化策略
- 💰 **激励任务匹配**: 自动匹配头条激励任务
- 📊 **数据分析**: 展现/阅读/收益数据追踪
- 🎯 **策略调整**: 根据数据优化发布时间和内容

### 4. 知识库系统
- 📚 **运营知识库**: 18 篇头条运营实战文档
- 📝 **内容模板**: 可直接使用的内容模板
- 🔄 **持续更新**: 每周更新策略和模板

## 📦 安装方法

```bash
# 1. 安装 OpenClaw（如未安装）
npm install -g openclaw

# 2. 安装本技能包
npx clawhub@latest install toutiao-agent

# 3. 运行恢复脚本
cd ~/openclaw/workspace/toutiao-agent
bash restore.sh

# 4. 配置头条号登录信息（手动）
# 在头条号后台登录账号

# 5. 启动 OpenClaw
openclaw start
```

## 🚀 使用方式

### 日常使用（自动）

技能包安装后会自动执行以下任务：

| 时间 | 任务 | 说明 |
|------|------|------|
| 06:15 | 早市菜价发布 | 采集当日菜价并发布 |
| 11:15 | 观点微头条发布 | 菜场对比/购买建议 |
| 15:45 | 日常笔记发布 | 今日小结 + 定位 |
| 20:00 | 活动话题发布 | #晒出你的本地生活# |
| 21:00 | 收益汇报 | 汇总当日收益数据 |
| 22:17 | 数据回收 | 记录到记忆文件 |

### 手动命令

```bash
# 发布早市菜价
openclaw run toutiao-morning

# 发布观点微头条
openclaw run toutiao-viewpoint

# 生成数据报告
openclaw run toutiao-report

# 检查收益
openclaw run toutiao-income
```

**Q: 需要头条号首发权限吗？**  
A: 不需要，本技能包适用于所有头条号账号。

**Q: 收益能保证吗？**  
A: 收益取决于账号权重、内容质量和平台算法，本技能包提供优化策略但不保证具体收益。
