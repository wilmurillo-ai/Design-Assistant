# 每日知识库学习流程

## 目的
在每日首个定时任务（07:00 早市菜价发布）前，自动读取和学习近一周的运营知识库，确保发布内容符合最新策略。

## 执行时间
每日 06:00（早于首个发布任务 1 小时）

## 学习范围

### 1. 知识库文件（knowledge/ 目录）
按修改时间排序，读取最近 7 天内更新的文件：
- `toutiao-growth.md` - 头条号涨粉全策略
- `toutiao-publishing-guide.md` - 发布流程指南
- `toutiao-timing-optimization.md` - 科学发布时间表
- `toutiao-user-demographics.md` - 用户画像分析
- `toutiao-traffic-sources.md` - 流量来源分析
- `price-calculation-guide.md` - 价格计算指南
- 其他近期更新的策略文件

### 2. 记忆文件（memory/ 目录）
读取最近 3 天的记忆文件：
- `YYYY-MM-DD.md`（今日 + 昨日 + 前日）
- 重点关注：待办事项、策略调整、问题诊断

### 3. 临时文件（temp/ 目录）
检查是否有未发布的草稿内容

## 学习输出

### 生成文件：`temp/daily-knowledge-summary.md`

包含以下内容：
1. **当前策略摘要**（300 字内）
   - 目标人群画像
   - 地域聚焦策略
   - 发布时间规范
   - 内容方向

2. **今日发布检查清单**
   - [ ] 标题含地域词（上海/嘉定）
   - [ ] 含活动标签 #晒出你的本地生活#
   - [ ] 字数 150-600 字
   - [ ] 标签 5-7 个
   - [ ] 未勾选首发（3/29 前冻结）

3. **近期问题提醒**
   - 自动化发布故障状态
   - Token 余额状态
   - 账号权重恢复进度

4. **数据基准**
   - 昨日展现量/阅读量
   - 当前粉丝数
   - 目标达成率

## 执行流程

```
06:00 触发知识库学习任务
  ↓
扫描 knowledge/ 目录（最近 7 天）
  ↓
扫描 memory/ 目录（最近 3 天）
  ↓
提取关键策略和待办事项
  ↓
生成 temp/daily-knowledge-summary.md
  ↓
输出学习摘要到聊天（供用户确认）
  ↓
07:00 早市菜价发布任务读取此摘要
```

## 更新机制

- 每日 06:00 自动生成新的摘要
- 覆盖前一日的摘要文件
- 历史摘要保留 7 天（用于回溯）
