## 🎯 四点设计原则（必须遵守）

### 1️⃣ 地域属性强
| 要求 | 检查项 | 示例 |
|------|--------|------|
| 标题有地域 | □ 标题有「上海」「嘉定」等 | ✅「上海人买菜...」 |
| 内容有本地 | □ 提到本地菜场/地名 | ✅「环镇北路」「江湾」 |
| 话题标签 | □ 有#上海# #嘉定# | ✅ 已添加 |

### 2️⃣ 转化导向
| 要求 | 检查项 | 示例 |
|------|--------|------|
| 关注理由 | □ 有实用价值说明 | ✅「每天看上海菜价」 |
| 福利承诺 | □ 有粉丝福利 | ✅「粉丝可领价格地图」 |
| 人设信任 | □ 有身份背书 | ✅「嘉定本地人，每天跑菜场」 |

### 3️⃣ 低门槛参与（网购/手机党友好）
| 要求 | 检查项 | 示例 |
|------|--------|------|
| 认可网购 | □ 不贬低 APP 买菜 | ✅「APP（方便省力）」 |
| 互动简单 | □ 无需出门/拍照 | ✅「选 A/B/C 即可」 |
| 手机友好 | □ 刷到就能评论 | ✅ 评论区选 A/B/C |
| 公平对比 | □ 菜场和 APP 都有优势 | ✅ 菜场新鲜/APP 方便 |

### 4️⃣ 老年男性偏好
| 要求 | 检查项 | 示例 |
|------|--------|------|
| 简单直接 | □ 一眼看懂，无花哨 | ✅ 价格对比表 |
| 实用主义 | □ 省钱、比价、性价比 | ✅「价格差一倍」 |
| 字体清晰 | □ 大字体、少 emoji | ✅ 每段独立成行 |
| 理性选择 | □ 让读者自己判断 | ✅ A/B/C 选项 |
| 不玩梗 | □ 无年轻向表达 | ✅ 无网络用语 |

## 🎯 爆款标题公式（基于 2.7 万展现验证）

### 2.7 万展现标题分析
**原标题**: 「上海 vs 浙江，哪里的春笋更便宜？」

**成功要素**:
1. ✅ 地域标签（上海）
2. ✅ 对比结构（vs）
3. ✅ 价格敏感（便宜）
4. ✅ 疑问句（引发好奇）
5. ✅ 简单直接（一眼看懂）

### 标题公式
```
[地域] + [对比/数字] + [价格/省钱] + [疑问/结论]

示例:
- 「上海 vs 浙江，哪里的春笋更便宜？」✅ 2.7 万展现
- 「嘉定 5 个菜场实测，这家最便宜」✅ 1387 展现
- 「盒马 vs 菜场，价格差多少？」待测试
```

## ⚠️ 避坑指南

### ❌ 不要做的
| 错误 | 原因 | 修正 |
|------|------|------|
| 标题无地域标签 | 吸引非目标用户 | 加「上海」「嘉定」 |
| 互动门槛高 | 用户不参与 | 改为选 A/B/C |

| 内容太年轻 | 老年用户看不懂 | 简单直接，不玩梗 |
| 字体太小 | 老年用户看不清 | 图片用大字体 |
