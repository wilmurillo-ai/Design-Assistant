# 价格换算标准公式

**创建时间**: 2026-03-26  
**用途**: APP 价格采集统一换算标准

---

## 📐 标准换算公式（两步法）

### 基础单位
- **1 斤 = 500 克 (g)**
- **1 kg = 1000 克 = 2 斤**

### 换算步骤

**第 1 步：把 g 换算成斤**
```
斤数 = 规格 (g) ÷ 500
```

**第 2 步：计算单价**
```
单价 (元/斤) = 总价 (元) ÷ 斤数
```

### 合并公式

```
单价 (元/斤) = 总价 (元) ÷ (规格 (g) ÷ 500)
```

---

## 📝 计算示例（两步法）

### 示例 1：500g 规格
```
商品：美团买菜 新鲜雷竹笋
规格：500g
价格：¥11.9

第 1 步：500g ÷ 500 = 1 斤
第 2 步：11.9 元 ÷ 1 斤 = 11.9 元/斤
```

### 示例 2：小规格（150g）
```
商品：美团买菜 去皮雷笋
规格：150g
价格：¥9.9

第 1 步：150g ÷ 500 = 0.3 斤
第 2 步：9.9 元 ÷ 0.3 斤 = 33.0 元/斤
```

### 示例 3：kg 规格
```
商品：奥乐齐 超值春笋
规格：1kg (1000g)
价格：¥9.09

第 1 步：1000g ÷ 500 = 2 斤
第 2 步：9.09 元 ÷ 2 斤 = 4.545 元/斤 ≈ 4.55 元/斤
```

### 示例 4：带壳 vs 剥壳
```
商品：盒马 手剥雷笋尖
规格：230g（剥壳笋肉）
价格：¥12.9

第 1 步：230g ÷ 500 = 0.46 斤
第 2 步：12.9 元 ÷ 0.46 斤 = 28.04 元/斤（笋肉）

⚠️ 注意：这是笋肉价格，不能直接和带壳笋对比！
带壳剥壳损耗约 50%，所以：
带壳等价 = 28.04 × 50% ≈ 14.02 元/斤（带壳等价）
```

## ⚠️ 注意事项

### 1. 带壳 vs 笋肉
- **带壳笋**：需要剥壳，损耗约 50%
- **笋肉**：已剥好，可直接食用
- **对比时**：必须分开对比，或换算后对比

### 2. 换算基准
```
带壳笋 1 斤 → 剥壳后约 0.5 斤笋肉
笋肉 1 斤 → 约需带壳笋 2 斤
```

### 3. 价格对比原则
- 带壳 vs 带壳
- 笋肉 vs 笋肉
- 或统一换算成笋肉对比
