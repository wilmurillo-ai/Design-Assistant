## ✅ 内容创作规范

### 优质观点微头条（0.4 元）
- **字数**: 30 字以上
- **结构**: 观点 + 论据 + 互动提问
- **示例**:
```
【菜场 vs 超市，哪里买菜更划算？】
跑了 7 天菜场，我的结论：
菜场胜在价格和新鲜，超市胜在环境和便利
追求性价比选菜场，图省事选超市
你们平时在哪买？来聊聊！
#上海菜价 #买菜攻略
```

### 日常微头条笔记（0.3 元）
- **字数**: >15 字
- **必需**: 图片 + 定位
- **示例**:
```
📍上海·嘉定
今天又跑了 3 个菜场
叶城菜场春笋最便宜！
老师傅们收好这张价格表～
```

### 带图发文（0.1 元）
- **必需**: 至少 1 张图片
- **建议**: 实拍图 > 网图
- **配图**: 菜场实拍、价格对比图
