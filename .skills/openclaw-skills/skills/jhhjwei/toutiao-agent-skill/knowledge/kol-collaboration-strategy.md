## 📝 合作 SOP（标准作业流程）

### 阶段 1：识别 KOL（每日）
- [ ] 浏览评论区，发现活跃用户
- [ ] 查看主页，确认粉丝数>1 万
- [ ] 检查内容垂直度（历史怀旧/民生）
- [ ] 确认粉丝群体重合（中老年男性）
- [ ] 记录到 KOL 清单

### 阶段 2：建立联系（1-2 天）
- [ ] 在对方最新内容下评论
- [ ] 发布内容@对方（投稿类）
- [ ] 评论区深度互动（至少 2 轮）
- [ ] 观察对方回应意愿

### 阶段 3：内容合作（2-3 天）
- [ ] 发布对方投稿内容
- [ ] @对方增加曝光
- [ ] 制造话题（地域对比/价格对比）
- [ ] 引导更多互动（评论/点赞）

### 阶段 4：深度绑定（1 周+）
- [ ] 互相关注
- [ ] 定期互相@（每周 2-3 次）
- [ ] 联合内容创作（地域 PK）
- [ ] 粉丝群互通（可选）

## 💡 内容合作模式

### 模式 1：投稿互动 ⭐⭐⭐⭐⭐
**流程**:
```
KOL 投稿 → 发布并@→ 评论区互动 → 制造话题
```

**案例**: 桃之 11zz 投稿春笋价格
- KOL 投稿：安徽马鞍山 4.58 元/斤
- 我方发布：@桃之 11zz，带活动标签
- 评论区：询问地点，引发讨论
- 制造话题：上海 vs 安徽价格对比

**效果**: 阅读率 63.6%，引发地域讨论

### 模式 2：地域 PK ⭐⭐⭐⭐
**流程**:
```
A 地 KOL → B 地 KOL → 价格对比 → 粉丝讨论
```

**案例**: 上海 vs 安徽春笋价格
- 上海：4 元/斤（带壳）= 8 元/斤（笋肉）
- 安徽：4.58 元/斤（剥壳笋肉）
- 结论：安徽便宜 43%
- 话题：上海老师傅羡慕吗？

**效果**: 引发地域讨论，增加互动

### 模式 3：互相@ ⭐⭐⭐⭐
**流程**:
```
我方内容 → @KOL → KOL 回复 → 互相引流
```

**案例**: 定期互相@
- 我方发布：@桃之 11zz 老师傅投稿
- KOL 回复：评论区互动
- 效果：粉丝互相流动

### 模式 4：联合创作 ⭐⭐⭐⭐⭐
**流程**:
```
策划主题 → 各自发布 → 互相@→ 粉丝 PK
```

**案例**: 全国春笋价格地图
- 我方：上海价格
- KOL：安徽价格
- 联合：全国价格地图
- 效果：粉丝大 PK

## ⚠️ 注意事项

### 避免事项
| 事项 | 原因 | 替代方案 |
|------|------|----------|
| **硬广推广** | 引起反感 | 软性植入 |
| **过度@** | 被视为骚扰 | 每周 2-3 次 |
| **单向索取** | 不可持续 | 互利共赢 |
| **虚假数据** | 失去信任 | 真实透明 |

### 风险控制
| 风险 | 概率 | 应对方案 |
|------|------|----------|
| **KOL 不回应** | 中 | 准备备选 KOL |
| **粉丝反感** | 低 | 控制频次 |
| **内容争议** | 低 | 避免敏感话题 |
| **合作中断** | 中 | 多元化合作 |

### 合规要求
- [ ] 内容真实，不夸大
- [ ] 价格准确，有依据
- [ ] 尊重 KOL，不强迫
- [ ] 遵守平台规则
