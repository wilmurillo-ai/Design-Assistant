# 用词规范（2026-03-27 更新）

## ✅ 推荐用词（亲切、尊重）

| 场景 | 推荐 | 说明 |
|------|------|------|
| 称呼女性 | 妈妈、家人、您 | 亲切、尊重 |
| 称呼男性 | 爸爸、先生、您 | 亲切、尊重 |
| 称呼老人 | 叔叔、阿姨、老师傅 | 尊重、亲切 |
| 称呼年轻人 | 小朋友、年轻人 | 友好 |
| 称呼读者 | 您、大家、各位 | 通用尊重 |

## ❌ 避免用词（刻板、不尊重）

| 避免 | 原因 | 替代 |
|------|------|------|
| 主妇 | 刻板印象 | 妈妈、家人 |
| 大妈 | 年龄歧视 | 阿姨、老师傅 |
| 老头 | 不尊重 | 叔叔、老师傅 |
| 穷人 | 冒犯 | 价格敏感人群 |
| 低端 | 冒犯 | 经济实惠型 |

## 📝 标题示例

### ✅ 推荐标题
- 《妈妈收藏！春笋保存 3 法》
- 《家人爱吃笋，这样挑最新鲜》
- 《老师傅教你挑笋，老板不敢坑》
- 《您那边的春笋多少钱？》

### ❌ 避免标题
- 《主妇必看！这样买春笋省一半钱》
- 《大妈都在抢的便宜菜》
- 《穷人买菜攻略》

## 🎯 内容语气

**亲切友好**：
- ✅ "帮您比价，买菜不花冤枉钱"
- ✅ "老师傅们，你们那边什么价？"
- ✅ "欢迎大家分享自家附近的菜价"

**避免说教**：
- ❌ "你必须这样做"
- ❌ "不懂这个就亏了"
- ✅ "建议这样买，更新鲜划算"
