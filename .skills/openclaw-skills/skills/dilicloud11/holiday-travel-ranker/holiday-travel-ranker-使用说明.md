# Holiday Travel Ranker - 假期旅行目的地推荐 Skill

## 使用说明

---

## 一、功能介绍

**Holiday Travel Ranker** 是一个假期旅行目的地综合评分推荐工具。它能够根据用户的出发城市、目标假期和个人偏好，自动收集多维度数据并进行评分排名，最终输出一份包含**国内 Top 10** 和**国外 Top 10** 最值得去的旅行目的地推荐报告。

### 核心能力

- **8 维度综合评分**：机票价格、酒店价格、当地物价、签证便利性、旅游好评度、当季气候适宜度、安全性、体验丰富度
- **智能数据采集**：自动通过网络搜索采集各目的地在特定假期时段的价格、签证、气候等数据
- **双榜排名**：国内和国外分别独立评分排名 Top 10
- **分维度排序**：除综合排名外，还提供按单维度（如机票最便宜、签证最方便等）的排行榜
- **个性化权重**：用户可选择特别在意的维度，系统自动调整权重
- **双格式输出**：同时生成 Markdown 报告和可交互的 HTML 页面

### 解决的痛点

每到长假前，很多人面临"去哪儿"的选择困难。这个 Skill 帮你把散落在各处的信息（机票价格、酒店价格、签证政策、天气、安全性等）汇总到一张打分表里，用数据说话，减少决策焦虑。

## 二、使用步骤

### 前置条件

- 已安装 CodeBuddy / WorkBuddy
- Python 3.8+ 环境（用于运行报告生成脚本）

### 安装 Skill

1. 解压 `holiday-travel-ranker.zip`
2. 将解压后的 `holiday-travel-ranker` 文件夹复制到 Skill 目录：
   - **WorkBuddy**：`~/.workbuddy/skills/holiday-travel-ranker/`
   - **CodeBuddy**：`~/.codebuddy/skills/holiday-travel-ranker/`
3. 重新加载/重启 WorkBuddy 或 CodeBuddy

### 使用方式

安装后，在对话中使用以下任一方式触发：

```
假期去哪里旅行最值？
帮我推荐五一去哪玩
国庆出行目的地推荐
春节旅游推荐排名
从上海出发，端午节去哪最划算？
```

或直接 `@holiday-travel-ranker` 调用。

### 工作流程

**Step 1 - 确认偏好（唯一需要交互的步骤）**

Skill 会询问你：
- 出发城市（默认上海）
- 目标假期（元旦/春节/清明/五一/端午/中秋/国庆）
- 特别在意的维度（可选，如"机票价格"、"签证便利性"）
- 预算区间（可选）

**Step 2 - 自动数据收集**

确认后自动开始，无需干预。AI 会通过网络搜索采集以下信息：
- 该假期时段从你的出发城市到各候选目的地的机票历史均价
- 中端酒店价格
- 当地旅行日均消费
- 签证政策（国外目的地）
- 旅游好评度和热度
- 该时段的气候数据
- 安全指数
- 景点/美食/文化丰富度

**Step 3 - 自动评分排名**

每个维度打 1-10 分，按权重计算综合分，国内/国外分别排名。

**Step 4 - 输出报告**

自动生成两份报告：
- **Markdown 报告** — 包含排名表、目的地详细卡片、分维度排行榜
- **HTML 交互页面** — 支持国内/国外 Tab 切换、按任意维度排序、展开目的地详情

## 三、评分维度说明

| 维度 | 默认权重 | 评分逻辑 |
|------|---------|---------|
| 机票价格 | 20% | 越便宜分越高；排除红眼航班和不合理转机 |
| 酒店价格 | 15% | 中端酒店（4星/精品）每晚均价越低分越高 |
| 当地物价 | 10% | 日均餐饮+交通+门票消费越低分越高 |
| 签证便利性 | 15% | 免签 > 落地签 > 电子签 > 贴纸签；国内默认满分 |
| 旅游好评度 | 15% | 主流平台综合评分和游客满意度 |
| 当季气候适宜度 | 10% | 该假期时段是否为最佳旅行季节 |
| 安全性 | 10% | 全球和平指数 + 外交部提醒等级 |
| 体验丰富度 | 5% | 景点密度、文化/美食多样性、可玩天数 |

**权重个性化调整**：如果你选择了"特别在意"的维度，该维度权重 +5%，从最低权重维度扣减，总权重保持 100%。

## 五、输出报告说明

### Markdown 报告包含

1. **评分说明** — 维度定义和权重
2. **国内 Top 10 综合排名表** — 含各维度分数
3. **国外 Top 10 综合排名表** — 含各维度分数
4. **目的地详细卡片** — 核心优势、重要劣势、最佳时段、预估花费、签证信息、参考机票/酒店价格
5. **分维度排行榜** — 按每个维度单独排序
6. **个性化建议** — 基于你的偏好给出推荐

### HTML 交互页面功能

- 国内/国外 Tab 一键切换
- 点击任意维度按钮即可重新排序
- 点击目的地卡片展开详细信息
- 彩色进度条直观展示各维度得分
- 响应式布局，手机端也能流畅使用

## 七、注意事项

1. **价格数据时效性**：机票和酒店价格基于历史同期数据或近期公开参考价，仅用于横向对比，不代表你实际预订时的价格。建议在出行前通过携程/去哪儿等平台查询实时报价。

2. **签证政策变动**：各国签证政策可能随时调整（尤其是免签/落地签政策），报告中的签证信息仅供参考，出行前请务必在中国领事服务网或目标国使馆官网确认最新政策。

3. **红眼航班排除**：所有机票价格评估均排除了 23:00-06:00 出发的航班和总旅行时长超过直达 2 倍的转机航班，确保推荐的是正常出行体验的价格。

4. **评分主观性**：虽然尽可能使用客观数据，但部分维度（如体验丰富度、旅游好评度）仍包含一定主观判断，建议结合个人兴趣做最终决策。

5. **假期人流量**：报告评分未单独设置"拥挤度"维度，但会在目的地卡片的"重要劣势"中提示假期期间可能的拥挤情况。

6. **数据采集耗时**：由于需要搜索大量信息，完整的数据采集和报告生成过程可能需要 5-15 分钟，请耐心等待。

## 九、常见问题

**Q: 可以换出发城市吗？**
A: 可以。在 Step 1 偏好确认时选择你的出发城市即可，默认上海。

**Q: 可以同时看多个假期的推荐吗？**
A: 可以。在目标假期选择时可以选择多个假期，会分别生成对应的排名。

**Q: 我只关心某一两个维度怎么办？**
A: 在"特别在意的维度"中选择，这些维度权重会自动上浮。同时报告中也提供了分维度排行榜，你可以直接看单维度排名。

**Q: 候选目的地可以自定义吗？**
A: 当前版本使用预设候选池（国内约 20 个、国外约 20 个），如有特别想加入的目的地，可以在对话中告诉 AI，它会补充进去。

**Q: 报告中的价格准确吗？**
A: 价格数据来自公开参考和历史均价，用于目的地之间的横向对比，不等于实际购买价格。请以实际预订平台为准。
