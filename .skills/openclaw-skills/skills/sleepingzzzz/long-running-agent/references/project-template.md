## 定向协议

**每个会话开始时，执行以下步骤**：

```
1. 读取 CHANGELOG.md 的"当前状态"和"下一步"
2. 确认没有回归（运行快速测试）
3. 从优先列表中选择最高影响的任务
4. 开始工作
```

**不要做的事**：
- ❌ 不要跳过定向协议
- ❌ 不要重新尝试已记录为"失败"的方法
- ❌ 不要在测试失败时提交代码

---
