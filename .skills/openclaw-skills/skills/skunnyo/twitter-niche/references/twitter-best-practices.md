# Twitter (X) Best Practices

## Posting
- Frequency: 3-7x/week for niches, threads 1-2x/week
- Timing: Analyze audience (Canada: evenings 6-9PM CST)
- Format: 80/20 value/entertainment; hooks first 3 words
- Hashtags: 1-3 targeted (#AIagents, not #AI)
- Visuals: Images/videos boost 2-3x engagement

## Growth
- Reply to all comments (24h window)
- Engage 10+ influencers/day
- Repurpose: Threads → newsletter → skill content
- Tools: Use browser for live search; web_search for benchmarks

## Metrics to Track
- Impressions/engagement rate &gt;2%
- Follower growth &gt;5%/month
- Viral threshold: 10% like/RT rate on posts

Tailor to niche: Hockey - game nights; AI - announcements.
