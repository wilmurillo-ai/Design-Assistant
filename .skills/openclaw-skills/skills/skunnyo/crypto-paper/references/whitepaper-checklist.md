# Crypto Whitepaper Checklist

## Core Sections Review
**Abstract/Intro**: Goals, hook, problem size?
**Problem**: Market gap validated? TAM?
**Solution/Tech**: Novelty? Scalability? Code open-source?
**Architecture**: Consensus? Interop?

## Tokenomics Deep Dive
| Metric | Questions | Red Flags |
|--------|-----------|-----------|
| Max Supply | Fixed/inflationary? | Unlimited mint |
| Allocation | % Team/VC/Ecosystem/Public | Team >25% unlocked |
| Vesting | Team/VC cliffs/periods | None/short |
| Utility | Governance/Staking/Payments | Speculative only |
| Burns/Mints | Mechanisms? | High inflation |
| Liquidity | Initial LP? Locks? | Unlocked dumps |

## Team & Execution
- Founders: Prior exits? Blockchain exp?
- Advisors: Reputable?
- Progress: MVP/testnet live?

## Roadmap & Risks
- Milestones: Achievable? Past delivery?
- Risks: Listed/mitigated?
- Audits: Firms? Recent?
- Governance: On-chain?

## Scoring Framework (1-5 per category)
- Innovation: 5=Breakthrough, 1=Copycat
- Tokenomics: 5=Aligned incentives, 1=Dump setup
- Team: 5=Proven, 1=Anon
- Execution: 5=Shipped, 1=Vaporware
**Overall: Weighted average. >4=Strong buy signal**
