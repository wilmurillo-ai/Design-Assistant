**你的专属 AI 面试考官，帮你高效准备面试.v3。**

基于简历或职位描述，生成**难度可控、题量灵活**的针对性面试题，并提供冲刺计划、就绪度评估与高分表达改写，帮助你更快接近 offer。

## 效果示例

### 基于简历生成（自动推断难度和题量）

```
用户      ❯ 你好考官，这是我的简历：resume.pdf

考官      ❯ 好的，我已阅读你的简历。检测到你有3年经验，
            申请前端工程师职位。我为你准备了【中级难度·10道题】：
            
            第1题：请解释 React 的 Virtual DOM 工作原理，
                   以及 diff 算法的优化策略？
            
            参考答案见文档末尾。
```

### 基于职位 + 指定难度和题量

```
用户      ❯ 来5道初级难度的 Java 面试题，我是应届生

考官      ❯ 明白。针对应届生 Java 面试，准备【初级难度·5道题】：
            
            第1题：请简述 Java 中接口和抽象类的区别？
            第2题：什么是 Java 的垃圾回收机制？
            ...
            
            建议用时：15分钟
```

## 使用方法

像和面试官对话一样自然，系统会自动识别你的意图：

### 基础用法

| 场景 | 你可以这样说 |
|------|-------------|
| **提供简历** | "你好考官，这是我的简历：path/to/resume.pdf" |
| **描述背景** | "你好考官，我是3年经验的前端工程师，帮我准备面试" |
| **指定职位** | "我想面试 AI 算法工程师" |
| **直接提问** | "准备一下 Java 后端面试题" |

### 指定难度和题量

| 场景 | 你可以这样说 | 效果 |
|------|-------------|------|
| **应届生** | "你好考官，我是应届生准备面试" | 初级难度，10题 |
| **快速练习** | "来5道简单的题热热身" | 初级难度，5题 |
| **深度准备** | "我是资深开发，要20道高级难度的题" | 高级难度，20题 |
| **系统复习** | "准备15道中级的后端题" | 中级难度，15题 |
| **任意数量** | "生成8道面试题" | 自动难度，8题 |

### 模拟面试模式

想体验真实的面试场景？使用模拟面试模式：

| 场景 | 你可以这样说 |
|------|-------------|
| **开始模拟面试** | "开始模拟面试" / "mock interview" |
| **逐题练习** | "一道一道来" / "逐题提问" |
| **实战演练** | "来面试我" / "面试模式" |

**模拟面试流程：**
1. 配置面试参数（难度、题数、题型）
2. 逐题提问，你回答后获得即时反馈
3. 面试结束后生成总结报告（得分、优缺点、改进建议）

### 查看历史记录与进步趋势

追踪你的练习历程，持续改进：

| 场景 | 你可以这样说 |
|------|-------------|
| **查看历史** | "查看历史" / "我的练习记录" |
| **能力雷达图** | "能力分析" / "雷达图" / "我的能力分布" |
| **进步趋势** | "进步情况" / "成绩趋势" |
| **上次成绩** | "上次面试怎么样" / "上次得分" |
| **个性化推荐** | "推荐练习" / "今天练什么" |

**历史记录功能：**

- 📈 追踪各维度得分变化趋势
- 🎯 生成可视化能力雷达图
- 💡 基于历史表现智能推荐下一步练习

**示例输出：**
```

系统设计    ██████████░░░░░░░░░░ 50分  ← 建议加强

## 推荐练习
基于你的进步曲线，建议今日：
- 难度: 高级

- 题数: 10题
```

### Offer 导向强化功能

不仅“练题”，更关注“拿 offer 的转化”：

| 场景 | 你可以这样说 | 系统输出 |
|------|-------------|----------|
| **7天冲刺计划** | "给我一个7天冲刺计划" / "一周面试计划" | 每日任务、题型配比、验收标准 |
| **岗位就绪度评分** | "我现在能面试吗" / "做个就绪度评估" | 100分制评估、风险项、两周提升路线 |
| **高分话术改写** | "帮我润色这个答案" / "改成高分表达" | 30秒版+2分钟版+追问应对 |

**典型收益：**
- ✅ 不再纠结“今天该练什么”
- ✅ 明确知道“现在能不能投递”
- ✅ 把“我知道”变成“我会说”

### 难度和题量会自动推断

- 从工作年限推断难度（应届生→初级，3年→中级，5年+→高级）
- 从描述推断题量（"热热身"→5题，"系统准备"→15/20题）
- 也可在对话中随时调整（"太难了，简单一点"或"再来5道"）

## 面试题设计

### 难度分级

| 难度 | 适用对象 | 题目特点 |
|------|----------|----------|
| **初级** | 应届生、1-2年经验 | 基础概念、常见场景、标准八股 |
| **中级** | 3-5年经验 | 原理深挖、方案对比、实际应用 |
| **高级** | 5年以上/专家 | 架构设计、源码分析、疑难排查 |

系统会根据你的工作经验自动推荐难度，也可手动指定。

### 题量选择

| 模式 | 题数 | 建议用时 | 适用场景 |
|------|------|----------|----------|
| **快速** | 5题 | 15分钟 | 碎片时间自测 |
| **标准** | 10题 | 30分钟 | 常规练习 |
| **深度** | 15题 | 45分钟 | 系统性复习 |
| **全面** | 20题 | 60分钟 | 模拟真实面试 |

### 题目分布（以10题为例）

| 维度 | 题数 | 说明 |
|------|------|------|
| **技术知识** | 2-3题 | 考察核心技术栈深度 |
| **项目经验** | 2-3题 | 基于简历项目或职位要求 |
| **问题解决** | 2题 | 场景分析和方案设计 |
| **行为面试** | 2-3题 | 团队合作、沟通能力等 |

### 题型说明

系统根据职位类型自动分配题型比例：

| 题型 | 适用职位 | 说明 |
|------|----------|------|
| **问答型** | 所有技术岗 | 考察技术原理深度 |
| **编程题** | 开发/算法 | LeetCode风格，含复杂度要求 |
| **系统设计** | 后端/架构 | 架构图+组件设计+扩展性 |
| **案例分析** | 产品/运营 | 业务场景分析+解决方案 |
| **行为面试** | 所有职位 | STAR法则回答，软技能 |

**也可指定题型：**
- "来点算法题" → 增加编程题比例
- "要系统设计题" → 增加架构题比例
- "行为面试" → 增加BQ比例

### 历史记录与进步追踪

每次练习后，系统会：

1. **保存记录** - 难度、题数、得分、用时、各题型表现
2. **分析趋势** - 计算平均分变化、进步速度
3. **生成雷达图** - 可视化展示5维能力分布
4. **智能推荐** - 基于薄弱点推荐下一步练习

**能力雷达图示例：**
```

         ╲    ╱        ╲    ╱

           ████████████████
          行为面试    案例分析
```

## 功能特性

- 📄 **简历分析** - 自动读取 PDF、Word、Markdown 格式的简历文件
- 🎯 **职位定制** - 根据指定职位生成针对性面试题
- 📊 **难度分级** - 支持初级、中级、高级三档难度，自动匹配你的经验水平
- 🔢 **灵活题量** - 可选 5/10/15/20 题，适应碎片时间或系统复习
- 🎨 **题型多样** - 5种题型覆盖真实面试场景：问答型、编程题、系统设计、案例分析、行为面试
- 🎭 **模拟面试** - 支持一对一模拟面试模式，逐题提问、即时反馈、总结报告
- 📈 **历史记录** - 保存每次练习记录，追踪进步趋势，生成能力雷达图
- 💡 **智能推荐** - 基于历史表现推荐个性化练习配置，突破薄弱环节
- 🚀 **7天冲刺计划** - 自动生成可执行的一周计划，解决“不会规划”问题
- 🎯 **岗位就绪度** - 100分制评估当前可投递程度，并给出两周提升路径
- 🗣️ **高分话术改写** - 将你的原始答案改写为30秒版/2分钟版，附追问应对
- 📝 **智能出题** - 基于简历内容、职位要求、难度级别和题型偏好生成高质量面试题
- ✅ **分级答案** - 每道题附带与难度匹配的参考答案和答题建议
- ✍️ **练习友好** - 每道题后预留空白供你手写答案，编程题提供代码区
