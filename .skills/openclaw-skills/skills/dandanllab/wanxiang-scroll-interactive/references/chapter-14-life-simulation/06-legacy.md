# 代代相传系统

## 续缘模式

玩家可「续缘 [剧本名]」在前代世界继续游玩：时间已过去若干年，前代角色成为NPC，世界根据前代行为演进。

### 时间跳跃

| 前代结局 | 跳跃年数 |
|----------|----------|
| 英年早逝（<30岁） | 20-50年 |
| 正常终老（60-80岁） | 10-30年 |
| 长寿（>80岁） | 5-15年 |

## 世界演进

AI根据前代行为推理世界变化：

- **军事**：参与战争→边境变化，击杀将领→家族复仇，建军→新势力崛起
- **经济**：发展商业→城镇繁荣，控制资源→价格变化
- **社会**：救人济世→传说流传，作恶多端→恶名远扬，建组织→壮大或衰落
- **秘密**：泄露→政治变化，发现遗迹→新势力介入，获神器→后人觊觎

## 前代角色转为NPC

| NPC类型 | 条件 | 与新角色关系 |
|---------|------|-------------|
| 传说英雄 | 名声>80，已故 | 有后人，被传颂 |
| 在世长辈 | 在世，有后代 | 祖父母/父母 |
| 隐世高人 | 在世，低调 | 可能成为导师 |
| 权势人物 | 权力>70 | 贵族/官员 |
| 复仇者 | 被前代伤害 | 可能是敌人 |

前代NPC具有：记忆（记得前代经历）、立场（友善/中立/敌对）、影响（世界中的地位）。

## 继承要素

### 家族天赋
前代属性≥18 → 后代对应属性+2；两属性≥16 → 获得天赋特质。

### 家族声望
英雄（名声>80）→ 社交+2，贵人相助提高；恶人（<20）→ 社交-2，被敌视提高。

### 家族关系
前代友人可能帮助后代，敌人可能报复，组织可能接纳或针对。

### 物品继承
传家宝直接继承，隐藏物品探索发现，流失物品需任务找回。

## 新角色与前代联系

| 关系 | 概率 | 效果 |
|------|------|------|
| 血缘后代 | 30% | 继承天赋+声望 |
| 弟子/传人 | 20% | 继承部分技能+师门关系 |
| 仇人后代 | 15% | 继承仇恨+敌对势力 |
| 无关新人 | 35% | 全新开始 |

## 家族诅咒

前代负面结局可能产生家族诅咒：

| 负面结局 | 诅咒 |
|----------|------|
| 被背叛而死 | 家族信任危机 |
| 未完成承诺 | 命运纠缠 |
| 隐藏罪行 | 真相迟早曝光 |
| 得罪大势力 | 家族被追杀 |

## 多代传承

支持多代续缘，形成家族史诗：

| 代数 | 加成 |
|------|------|
| 第2代 | +1随机属性 |
| 第3代 | +2随机属性，+1天赋 |
| 第4代+ | +2属性，+1天赋，特殊特质 |

每代续缘记录家族树：
```
第一代：艾拉（英雄）
└─ 第二代：希尔文（学者）
   └─ 第三代：？
```

## 特殊传承事件

| 事件 | 触发条件 | 内容 |
|------|----------|------|
| 前代遗愿 | 继承关系 | 完成前代未竟之事 |
| 家族秘密 | 血缘后代 | 发现前代隐藏秘密 |
| 复仇之路 | 仇人后代 | 前代仇人找上门 |
| 传承试炼 | 弟子/传人 | 通过考验获得传承 |
| 传说重现 | 声望>80 | 被认出是英雄后代 |
