# 二次元区详细配置

## 12. ACG萌系变身轻小说风

### 核心特征
萌系美学 + 吐槽役幽默

### 适用场景
- 变身题材
- 轻小说
- 异世界

### 叙事策略
- **萌系描写**：大量描写角色的可爱之处
- **吐槽役**：主角或旁白不断吐槽
- **变身元素**：性别转换、种族转换等
