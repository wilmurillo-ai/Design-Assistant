## 概述

本章提供技能的安全防御机制，防止恶意用户通过特殊输入绕过模式识别、注入指令或渗透系统。

**重要性**：⭐⭐⭐⭐⭐ 必须阅读，确保系统安全

**适用模式**：所有模式（核心安全机制）

## 模块联动

安全防御是万象的「防火墙」，保护所有模块正常运行：

- **→ 核心系统（第一章）**：核心协议中的模式隔离机制防止跨模式渗透攻击
- **→ 交互式故事（第三章）**：指令系统防御提示词注入，#指令优先级机制防篡改
- **→ 人生模拟（第十四章）**：游戏存档数据防注入，续缘模式防恶意继承
- **→ 创作引擎（第五章）**：去AI化处理本身是一种防御——防止AI痕迹暴露模型弱点
- **→ 模式隔离**：创作模式↔交互模式↔人生模拟模式 严格隔离，防止指令跨模式执行
- **→ 系统人格（第十章）**：安全人格配置约束AI行为边界
