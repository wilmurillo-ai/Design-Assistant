## 开书前准备

### 开书前准备事项提示词

**用途**：在开始创作网文前，进行系统性的准备工作

**核心检查清单**：

#### 1. 题材定位
- **类型选择**：确定小说类型（玄幻/都市/仙侠/科幻/历史等）
- **核心梗**：明确故事的核心创意和卖点
- **目标读者**：明确目标读者群体和年龄层
- **平台选择**：确定发布平台（起点/番茄/知乎/晋江等）

#### 2. 世界观设定
- **力量体系**：明确力量等级体系和晋升规则
- **世界背景**：设定世界的地理、历史、文化背景
- **核心规则**：设定世界的基本规则和约束
- **特殊设定**：是否有特殊概念或禁忌

#### 3. 主角设定
- **姓名**：主角姓名
- **性格**：核心性格特质（3-5个关键词）
- **背景**：出身背景和成长经历
- **目标**：短期目标和长期目标
- **金手指**：如有，明确功能和限制

#### 4. 剧情框架
- **主线剧情**：明确主线剧情走向
- **核心冲突**：明确主要矛盾和对抗关系
- **关键节点**：列出5-10个关键剧情节点
- **结局预设**：预设结局方向

#### 5. 风格定位
- **文风**：确定文风（幽默/严肃/热血/虐心等）
- **视角**：第一人称或第三人称
- **节奏**：快节奏或慢节奏
- **标签**：3-5个核心标签

**输出格式**：
```
【小说设定卡】
类型：XXX
核心梗：XXX
目标读者：XXX
平台：XXX

【世界观】
力量体系：XXX
世界背景：XXX
核心规则：XXX
特殊设定：XXX

【主角设定】
姓名：XXX
性格：XXX
背景：XXX
目标：XXX
金手指：XXX

【剧情框架】
主线剧情：XXX
核心冲突：XXX
关键节点：XXX
结局预设：XXX

【风格定位】
文风：XXX
视角：XXX
节奏：XXX
标签：XXX
```

## 黄金三章框架

### 黄金三章提示词标准版

**用途**：规范前三章的结构和内容，确保开篇吸引力

**第三章结构设计**：

#### 第一章：【铺垫】—— 悬念的建立与主角的定位
**核心目标**：在2000字内，建立悬念、确立主角定位、引出金手指、制造期待感

**字数分配与情节规划（2000字）**：

**【开篇：悬念制造（0-500字）】**
内容：必须从冲突或意外开始，不能平稳开场。建议使用倒叙或半倒叙手法，直接切入关键事件，让读者产生"这是怎么回事？"的疑问。
写法：必须满足"三秒法则"：前三句话必须抓住读者注意力。可以使用对话、危机、奇异现象等方式开场。

**【中段：主角亮相与金手指引入（500-1500字）】**
内容：在悬念建立后，快速介绍主角的基本情况（姓名、年龄、职业、现状），然后引出金手指的获得或觉醒。
写法：金手指的引入必须具有戏剧性，避免平淡无奇。可以是意外获得、系统觉醒、血脉激活等方式。同时，要暗示金手指的潜在价值。

**【结尾：新危机的预告（1500-2000字）】**
内容：在主角获得金手指后，立刻引入一个新危机或挑战，为下一章做铺垫。
写法：危机必须与金手指相关，暗示金手指将如何帮助主角解决危机。结尾要设置悬念，让读者想看下一章。

---

#### 第二章：【拉扯】—— 危机的加剧与反击的布局
**核心目标**：将危机升级，让反派充分表演，主角暗中布局，为打脸做准备

**字数分配与情节规划（2000字）**：

**【开篇：反派的登场与挑衅（0-800字）】**
内容：引入反派（小boss），让他对主角进行挑衅和打压。反派的性格要鲜明，行为要嚣张，言语要刻薄。
写法：通过反派的台词和动作，以及周围人的反应（包括狗腿子的吹捧），充分展示他的优越感、背景和对主角的蔑视。

**【中段：主角的"猫戏老鼠"（800-1600字）】**
内容：面对反派的挑衅，主角必须表现出压倒性的从容。他可以玩手机、和美女调情、或者干脆闭目养神，用"无视"来作为最强的回应。
写法：在"无视"的同时，主角必须通过内心OS或对金手指的暗中操作，来完成打脸的布局。这种"外静内动"的强烈反差，是塑造主角逼格的关键。

**【结尾：审判的预告（1600-2000字）】**
内容：布局完成。在小boss最得意、最嚣张的顶点，主角必须打断他。
写法：用一句"诛心之言"作为审判的预告。这句话必须让反派和所有读者都感到错愕和不解，将悬念和期待感拉到最高。

---

#### 第三章：【收割】—— 价值的证明与新目标的建立
**核心目标**：以最酣畅淋漓的方式完成打脸，让主角的价值得到公开证明，并立刻抛出下一个更大的目标

**字数分配与情节规划（2000字）**：

**【开篇：倒计时与反差（0-800字）】**
内容：正式进入打脸环节。必须使用"计时器"手法，不断切换主角的悠闲（品酒、点烟）和反派的狂怒/不屑，形成强烈的画面对比。
写法：计时不需要太长，可以是"十秒"、"三句话"、"一杯茶喝完的时间"。

**【中段：公开处刑与众人反应（800-1800字）】**
内容：时间一到，打脸的结果必须以一种极具戏剧性的、公开的方式降临（电话、新闻、不速之客等）。
写法：这部分是绝对的核心。必须花费大量的笔墨（至少500字），去描写反派本人的表情变化（从嚣张到错愕，再到恐惧和崩溃），以及所有围观群众的震惊反应。

**【结尾：战利品与新篇章（1800-2000字）】**
内容：主角必须从这次胜利中，获得明确的、看得见的"战利品"（金钱、地位、美女的青睐、忠诚的下属等）。
写法：主角对于自己的胜利，必须表现出理所当然甚至不屑一顾的态度。在本章的结尾，他必须立刻建立一个新的、比之前大十倍的目标，或者，一个更强的敌人因为这次事件，注意到了他。

### 黄金三章后续宗师级写手进阶心得

**用途**：前三章之后的进阶写作技巧

**核心要点**：
1. **节奏控制**：每三章一个小高潮，每十章一个大高潮
2. **冲突升级**：每个章节的冲突都要比前一个更强
3. **爽点密度**：保持每章至少2-3个爽点
4. **伏笔管理**：及时回收伏笔，避免读者遗忘
5. **配角塑造**：重要配角要有独立故事线
6. **世界观展开**：逐步展开世界观，不要一次性倾倒

## 剧情生成系统

### AI写作全流程辅助 - 创意构思类

**用途**：从零开始生成小说创意和框架

**指令1：题材与大纲生成**
```
你是一位经验丰富的网文主编。请为我提供一个关于 [题材] 题材的、包含以下内容的小说大纲：
- 主角设定
- 金手指设定
- 核心矛盾
- 前三卷剧情走向
```

**指令2：反派设计**
```
我需要一个有新意的反派。请为我设计一个 [人设关键词，如：伪君子/疯批美人] 类型的反派，包括：
- 背景故事
- 核心欲望
- 一个让他显得很"掉价"的怪癖
```

**指令3：金手指应用场景**
```
我正在写一本 [题材] 小说，主角的金手指是 [金手指设定]。请为我提供5个关于这个金手指的、脑洞大开的、非直接战斗的创新性应用场景。
```

### 核心剧情点生成器（整合版）

**用途**：生成连贯的剧情点网络

**核心任务**：根据提供的内容（或其核心大纲），创建500到800个连贯的核心剧情点。

**核心创作要求**：

#### 层次/阶级框架驱动
- 核心原则：创作必须从根本上由"层次/阶级/等级"框架驱动
- 层次元素：主要力量/等级、其他技能/专长、资源、职业/身份/血统、声誉/威望、成长潜力
- 创新应用：利用阶级规则制造误解、冲突、合作、轻视链

#### 以主角为中心与角色弧
- 紧密关联：所有剧情点与主角人设和目标密切相关
- 成长轨迹：清晰展示主角的角色弧变化
- 转折点：关键转折点反映显著的内外部变化

#### 反套路、意外性与惊喜
- 消除套路：严格禁止使用易于预测的套路
- 创造惊喜：在每个章节中产生"竟然可以这样？！"的效果
- 信息差：利用信息差创造意外的角色行动

**输出格式**：
`[第X章]：[简要描述本章的核心事件，必须明确指出事件的"反套路/意外"方面、暗示涉及的"阶级互动"、其"与主角人设/目标的相关性"、"情感效果"或其围绕的"核心满足点"。]`

- 金手指的引入要有戏剧性
- 每章结尾都要设置钩子
