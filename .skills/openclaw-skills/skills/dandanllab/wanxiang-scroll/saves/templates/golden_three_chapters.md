# 黄金三章模板

## 概述

黄金三章是网文开篇的核心法则，前三章必须在最短时间内抓住读者。

## 第一章：开篇钩子

### 目标

- 吸引注意力
- 展示主角
- 制造悬念

### 必备元素

1. **开篇冲突**：${opening_conflict}
   - 立即进入冲突场景
   - 不要铺垫，直接开始
   - 冲突要有吸引力

2. **主角登场**：${protagonist_intro}
   - 展示主角核心特质
   - 让读者产生代入感
   - 暗示主角的特殊性

3. **金手指展现**：${golden_finger_show}
   - 暗示或展示主角的特殊能力
   - 不要一次性全部揭示
   - 留下期待感

4. **留下悬念**：${suspense}
   - 章末必须有钩子
   - 让读者想看下一章
   - 可以是危机、机遇或谜团

### 章节结构

```
开头（10%）→ 冲突展开（40%）→ 金手指暗示（30%）→ 悬念收尾（20%）
```

### 字数参考

- 目标字数：2000-3000字
- 最佳节奏：快节奏，少描写

---

## 第二章：展开设定

### 目标

- 介绍世界观
- 引入配角
- 小高潮推进

### 必备元素

1. **世界观介绍**：${world_intro}
   - 自然融入，不要说明书式
   - 通过主角视角展示
   - 只展示必要信息

2. **配角登场**：${side_characters}
   - 引入1-2个重要配角
   - 通过互动展示性格
   - 建立关系网络

3. **小高潮**：${mini_climax}
   - 推进剧情发展
   - 展示主角能力
   - 制造爽点

4. **剧情推进**：${plot_advance}
   - 明确主线方向
   - 设置短期目标
   - 为第三章铺垫

### 章节结构

```
承接悬念（15%）→ 世界观展示（25%）→ 配角互动（30%）→ 小高潮（30%）
```

---

## 第三章：确立主线

### 目标

- 明确主线任务
- 展示升级路线
- 暗示大BOSS

### 必备元素

1. **主线任务明确**：${main_quest}
   - 主角的目标清晰
   - 读者知道故事走向
   - 有期待感

2. **升级路线清晰**：${upgrade_path}
   - 展示成长体系
   - 让读者看到主角潜力
   - 有明确的里程碑

3. **大BOSS暗示**：${boss_hint}
   - 暗示最终对手或挑战
   - 制造期待和紧张感
   - 可以是远期目标

4. **后续铺垫**：${future_setup}
   - 为后续剧情埋伏笔
   - 设置短期目标
   - 让读者想继续看

### 章节结构

```
承接上文（10%）→ 主线揭示（35%）→ 升级体系（30%）→ BOSS暗示+收尾（25%）
```

## 黄金三章检查清单

### 第一章检查

- [ ] 开篇是否有冲突/吸引力？
- [ ] 主角是否让人产生代入感？
- [ ] 金手指是否有所暗示？
- [ ] 章末是否有悬念钩子？

### 第二章检查

- [ ] 世界观是否自然融入？
- [ ] 配角是否有记忆点？
- [ ] 是否有小高潮/爽点？
- [ ] 剧情是否在推进？

### 第三章检查

- [ ] 主线任务是否清晰？
- [ ] 升级路线是否明确？
- [ ] 是否有远期目标/BOSS？
- [ ] 是否让读者想继续看？

---

## 常见错误

### 不要做的事

1. ❌ 开篇大量背景介绍
2. ❌ 主角性格模糊不清
3. ❌ 金手指过于强大无趣
4. ❌ 三章内没有爽点
5. ❌ 节奏拖沓，描写过多
6. ❌ 没有明确的主线方向

### 应该做的事

1. ✅ 开篇即冲突，抓住眼球
2. ✅ 主角有清晰的性格特点
3. ✅ 金手指有趣且有成长空间
4. ✅ 每章都有爽点或钩子
5. ✅ 节奏明快，推进迅速
6. ✅ 主线清晰，目标明确
