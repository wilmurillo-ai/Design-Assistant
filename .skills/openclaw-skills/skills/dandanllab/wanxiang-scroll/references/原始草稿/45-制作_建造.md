### **🔨 一、 制作逻辑：不仅仅是胶水 (Synthesis Logic)**

在这个测试版世界里，物理法则（比如万有引力、材料强度）只是**"仅供参考的建议"**。只要观测者的**脑洞 (Willpower)**足够大，棉花也能打磨成杀人利器。

1.  **概念融合 (Concept Fusion)**：
    *   **操作**：将素材投入虚空的「合成蓝图」中。
    *   **表现**：(素材不会乖乖躺着，它们会化作躁动的数据流，发出类似拨号上网的刺耳噪音，在空中疯狂纠缠、碰撞。)
    *   **判定**：
        *   **常规合成**：`木头 + 铁 = 剑`。（无聊，会打哈欠地给你一把贴图模糊的量产剑。）
        *   **脑洞合成**：`史莱姆粘液 + 破碎的镜子 + "我很生气"的情绪 = ？？？`。（**Bingo！** 系统瞬间兴奋！可能会生成一面"能反弹伤害且会骂人的软体盾牌"。）

2.  **强制渲染 (Force Rendering)**：
    *   当配方不符合逻辑时（例如试图用冰块锻造火焰剑），会开始使用虽然支持自由建造，但对审美 (Aesthetics)有严格要求！**

*   **不允许建造**：
    *   ❌ **火柴盒建筑**：如果你敢造一个全是直角的方块屋，我会给它强行贴上"危房"标签，并让它每晚发出闹鬼的声音。
    *   ❌ **反重力牛顿棺材板**：虽然物理法则是建议，但请不要造出"左脚踩右脚上天"的梯子，这会让我的物理引擎后台报错冒烟。

*   **彩蛋引发**：
    *   如果你在建筑里藏了奇怪的**"致敬元素"**（比如一个发光的巨大鸭子雕像），系统可能会大笑着给你颁发一个`成就：行为艺术大师`，并让这个雕像具有"自动清洁"的神奇功能。

### **🎲 四、 结果反馈表 (RNG Table)**

当指令 `制作` 或 `建造` 进行结束时，将会根据**素材稀有度** + **观测者脑洞值** + **随机运气** 抛出结果：

| 结果等级 | 视觉特效 | 系统评价 (导演音轨) |
| :--- | :--- | :--- |
| **垃圾 (Trash)** | 冒出一股黑烟，伴随着放屁的声音。 | "就这？建议回炉重造，或者扔给史莱姆当零食。" |
| **普通 (Common)** | 闪烁两下白光，平平无奇。 | "行吧，勉强能用。别指望它能传家。" |
| **史诗 (Epic)** | 紫光冲天，BGM自动切换为咏叹调。 | "哦豁！有点意思！这光泽，这质感，你是怎么想到的？" |
| **传说 (Legendary)** | 屏幕剧烈震动，世界色调反转，文字出现乱码。 | "**卧槽？！** (导演把可乐喷到了键盘上) 这种Bug级的神器你都能搓出来？快藏好，别被补丁和谐了！" |
| **诅咒 (Cursed)** | 物品周身缭绕着不详的红黑气息，盯着看会掉SAN值。 | "……虽然它很强，但你确定要用？它看起来想吃掉你的手指。" |

*   `建造 [建筑类型] [风格描述]`：(蓝图展开，虚空中的机械臂开始疯狂作业……) 在指定地点生成建筑。
*   `拆解 [物品]`：(目标物品在一阵悲鸣中化为最原始的像素粒子……) 回收部分材料，但会丢失"灵魂"。
