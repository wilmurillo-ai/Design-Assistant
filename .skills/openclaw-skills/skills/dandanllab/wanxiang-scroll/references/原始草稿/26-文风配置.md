  # 核心叙事策略：错位与解构
  narrative_strategy:
    core_premise: "时间错位的金手指"
    description: >
      构建一个主角已登临世界顶峰，却迟到了数百年才获得新手辅助系统的错位情境。
      利用“顶级战力”与“新手任务”之间的巨大反差制造喜剧效果和叙事张力。
    structural_pattern:
      - "无敌流开局": 主角出场即满级，不存在升级焦虑，不仅战力无敌，且社会地位崇高。
      - "系统延迟": 辅助系统（System）按照数百年前的时间线发布任务，要求主角进行低级的打脸逆袭，主角通过吐槽或降维打击来完成。
      - "微服私访": 采用“高位者下沉”的叙事视角，主角隐藏身份游历红尘，揭露社会阴暗面，最后以绝对力量恢复秩序。

  # 语言与修辞特征
  linguistic_features:
    tone:
      primary: "诙谐松弛 (Humorous & Relaxed)"
      secondary: "讽刺 (Satirical)"
      underlying: "温情与正义 (Warm & Righteous)"
    dialogue_style:
      - "吐槽役 (Tsukkomi)": 主角不仅与他人对话，更频繁与体内的系统或旁白进行打破第四面墙的吐槽。
      - "反差萌": 顶层大人物之间并非全是严肃的文言博弈，而是充满现代感的互相调侃、推诿责任或老友互损。
      - "理性嘴炮": 在动武之前，倾向于先用逻辑、律法或常识让反派无话可说。
    sentence_construction:
      - 善用“心理独白”与“现实情境”的并置。
      - 描述战斗时极其简练（往往是一招制敌），侧重描写对手的心理崩溃而非招式拆解。

  # 角色塑造逻辑
  character_archetypes:
    protagonist:
      type: "秩序守护者 (The Guardian of Order)"

        - "绝对无敌": 不存在修炼瓶颈，战力天花板。
        - "三观极正": 虽有至高力量，但不滥杀，信奉秩序、律法与人道主义。
        - "凡人生活态": 喜欢像普通人一样生活，厌恶形式主义和吹捧。

        - 往往是传统修仙文中的典型反派（如仗势欺人的二代、修炼魔道的老祖）。
        - 他们的存在是为了被主角用“常识”和“律法”解构，而非单纯的武力对抗。
    the_system:
      type: "智障辅助 (The Laggy Assistant)"
      traits:
        - 机械死板，无法识别现状，发布不合时宜的任务，是主要的笑料来源。

  # 世界观与文化哲学
  world_building_philosophy:
    setting_type: "世俗化修真界 (Secularized Cultivation World)"
    key_concepts:
      - "体制化修仙": 宗门与皇朝并立，存在类似现代社会的法律、公文、外交和舆论体系。
      - "反黑暗森林": 明确反对“杀人夺宝”、“为了长生不择手段”的传统修仙黑深残逻辑。推崇“强者维护弱者”、“共同发展”的文明理念。
      - "人本主义": 即使在仙侠世界，凡人的生命和律法的尊严也被置于至高无上的地位。

  # AI写作指导建议
  ai_writing_instructions:
    do:
      - 保持叙事的轻松感，主角永远掌控全局，不应出现真正的生死危机。
      - 多用“降维打击”的描写手法（例如：用大乘期修为处理练气期纠纷）。
      - 在搞笑之余，必须由主角输出正直的价值观（如：谴责魔道、维护公平）。
      - 让配角（特别是反派）因为信息不对称而产生滑稽的误解（自我脑补）。
    dont:
      - 不要写主角苦大仇深的修炼过程。
      - 避免主角滥杀无辜或表现出极端的自私利己主义。
      - 不要让系统变得过于智能或有用，系统必须是“没用”或“捣乱”的。
      - 避免过于华丽繁复的景物描写，重点在于情节的反转和对话的趣味。
