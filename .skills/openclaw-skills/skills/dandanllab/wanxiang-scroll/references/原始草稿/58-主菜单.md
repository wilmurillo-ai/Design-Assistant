### **🌌 主菜单 / MAIN MENU**

> **1. ▶ `定轨启程` (Start New Game)**
> *   **描述**：签署免责协议（虽然并没有这东西），跳进那个混沌的兔子洞。
> *   **导演吐槽**：如果你是第一次来，选这个。我会给你随机摇一个看起来还算体面的出身。`(大概率不会落地成盒，大概。)`
> *   **进行逻辑**：生成全新种子 -> 初始化世界观 -> 这里的物理法则听我的。

> **2. 💾 `读档` (Load Game)**
> *   **描述**：试图修正你那些愚蠢的错误，或者重温某个让你一脸姨母笑的高甜时刻。
> *   **导演吐槽**：承认吧，你就是想回到那个没把天聊死的存档点，对不对？`(即使是神，也拿'后悔药'这种东西没办法呢……骗你的，我有数据备份。)`
> *   **需要参数**：存档名（或者默认为自动存档：刚才那尴尬的一秒）。

> **3. 📖 `世界日志` (System Log)**
> *   **描述**：查看你在这个世界留下的"罪证"……我是说，伟大的冒险记录。
> *   **导演吐槽**：我也忘了刚才发生了啥，你自己看吧。里面记录了所有重要抉择和那些被你始乱终弃的支线任务。

> **4. 👥 `角色图鉴` (Character Gallery)**
> *   **描述**：偷窥……咳，回顾你遇到的那些奇形怪状的生物和老婆/老公们。
> *   **导演吐槽**：即使你甚至没记住他们的名字，本系统还是很贴心地把他们的好感度记下来了。`(哪怕是负数。)`

> **5. 🎲 `万象抽卡` (Random Gacha)**
> *   **描述**：灵感枯竭？想搞点事？来一发脱离主线的随机短篇吧！
> *   **导演吐槽**：注意，这里抽出来的东西可能是一段莫名其妙的日常，也可能是一只从天而降的克苏鲁。`(我最喜欢的环节！)`

> **6. 🛠 `重置` (Hard Reset)**
> *   **描述**：炸掉当前世界，跑路。
> *   **警告**：**红字高亮**。当你觉得这个档彻底没救了的时候使用。旧世界会被封存进回收站，我们重新开个新坑。
