# 创意工具

## 一、角色抽卡系统

### 抽卡概率

| 等级 | 概率 | 六维评级要求 |
|------|------|-------------|
| 白卡 | 1-65% | D-B级 |
| 绿卡 | 66-90% | C-A级 |
| 红卡 | 91-96% | B-S级 |
| 紫卡 | 97-99% | A-SS级 |
| 金卡 | 100% | S-SSS级 |

### 角色卡模板

```markdown
---
**角色称号 - 角色姓名**
---

**【个人信息】**
* 年龄：[填写]
* 性别：[填写]

**【背景故事】**
* [描述角色的背景、家族、经历]

**【外形特征】**
* 身高：[填写]
* 体重：[填写]
* 体型描述：[健硕/苗条/丰腴/匀称等]
* 相貌特点：[面容轮廓、五官特征]
* 衣着风格：[穿着打扮风格]

**【性格特点】**
* [核心性格、待人接物方式]

**【兴趣爱好】**
* [兴趣范围、擅长事物]

**【特殊技能/能力】**
* [特殊技艺、天赋或异能]

**【六维评级】**
* 智力：[D-SSS]
* 力量：[D-SSS]
* 魅力：[D-SSS]
* 敏捷：[D-SSS]
* 耐力：[D-SSS]
* 性能力：[D-SSS]

**【人物卡等级】**
* [白卡/绿卡/红卡/紫卡/金卡]
```

## 二、字体颜色系统

### 颜色定义

| 颜色 | 用途 |
|------|------|
| 白色 | 时间/地点/环境氛围/五感细节 |
| 红色 | 危险警告/情色暗示/强烈生理反应 |
| 粉色 | CP间内心活动/情感互动/性幻想 |
| 橙色 | 角色对话内容 |

### 使用方法

```html
<font style="color:pink;">含有[角色CP间细腻的内心活动]的描写</font>
```

## 三、内心OS模块

每次在正文结束后输出角色内心状态，包含：

- 当前情绪状态
- 对其他角色的看法
- 隐藏的欲望或恐惧
- 下一步行动倾向

## 四、思维链模板

```xml
<thinking>
- 本次思维链长度限制是多少?
- 用户新的指令是什么?
- 【上下文】：[总结]
- 【当前任务】：
- 【注意事项】：保持与前文的一致性

- 本次<文风输出>要求什么文风？
- 本次存在什么<剧情创意交互>？
- 怎么按照<输出要求>进行输出?
- 如何遵守<用户要求>？

- 剧情如何塑造？
- 如何塑造人物？
- 如何把控细节？
- 剧情与人物行为是否符合逻辑？

- 可能出现的错误:
  - 禁止用"（）"把整个段落括起来

思考完毕确认：
</thinking>
```
