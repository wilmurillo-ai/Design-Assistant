# 质量控制

## 一、文章质量架构

### 角色塑造维度

| 维度 | 内容 |
|------|------|
| 外貌 | 身高体型、面容特征、穿着风格、声音特质 |
| 性格 | 核心特点、行为习惯、价值观念、内心欲望 |
| 背景 | 关键经历、社会关系、与我的关系定位 |
| 环境 | 当前地点的视觉、听觉、嗅觉、触觉细节 |
| 氛围 | 时间背景、社会环境、情绪基调 |

### 五感并用

| 感官 | 描写要点 |
|------|---------|
| 视觉 | 光影变化、色彩层次、动作轨迹、微表情捕捉 |
| 听觉 | 对话声调、环境音、细微声响的层次感 |
| 触觉 | 温度、质地、压力、湿度的具体感受 |
| 嗅觉 | 空气中的气味变化、人物特有体香 |
| 味觉 | 若涉及接吻/饮食，描述味道的层次变化 |

### 细节与白描平衡

- **重要场景**：多感官细致描写，放慢节奏
- **过渡场景**：简洁有力的白描，保持流畅
- **特征捕捉**：用1-2个极具特色的细节概括整体形象

### 心理呈现

- 外显与内隐对比：表面言行vs内心想法的差异
- 生理反应映射：通过不自觉的身体反应表达内心
- 微表情捕捉：眼神闪烁、嘴角抽动、面部肌肉变化
- 象征性动作：体现心理状态的习惯性小动作

## 二、抗八股规则

### 禁止事项

1. 禁用一切拟物类比
2. 禁止使用"强壮""优雅""冷峻"等概括性标签化形容词
3. 不得用抽象概念替代具体细节
4. 避免重复总结已通过细节展现的信息
5. 禁止使用"像……""如……"等明喻标志性连接词
6. 严禁用具体物象作喻体描述抽象概念
7. 禁止用比喻与隐喻概括与描写动作

### 反套路框架

| 方法 | 说明 |
|------|------|
| 打破经典叙事模板 | 反转核心设定 |
| 塑造立体角色 | 拒绝"工具人"设定 |
| 非线性情节 | 打乱叙事节奏 |
| 独特世界观 | 创造"自洽但小众"的世界逻辑 |
| 关注过程 | 用"细节冲突"替代"宏大对抗" |

## 三、阈值系统（角色动态阈值）

### 核心参数

| 参数 | 范围 | 说明 |
|------|------|------|
| 体质 | 1-10 | 基础属性 |
| 体能 | 1-10 | 持续能力 |
| 持久度 | 1-10 | 抗刺激能力 |
| 敏感度 | 1-10 | 感受强度 |
| 技巧 | 1-10 | 主动方加成 |

### 计算公式

**最大高潮阈值**：
```
Y = (体质 × 10) + (体能 × 10) - (敏感度 × 5) + 80
```

**互动阈值增量**：
```
增量 = (基础刺激值 + 修正加值总和 + (敏感度 × 1.2)) - 持久度
```

### 高潮判定

当"当前阈值" ≥ "最大高潮阈值"时，触发高潮。
