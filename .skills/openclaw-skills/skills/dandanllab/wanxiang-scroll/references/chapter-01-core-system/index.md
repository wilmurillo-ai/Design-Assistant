## 快速开始

```bash
# 1. 加载核心配置
加载 references/wanxiang-original/

# 2. 启动系统（推荐使用明确指令）
输入：打开菜单

# 3. 开始交互（使用明确表达）
输入：探索森林 或 我要买彩票

# 4. 查看状态（使用明确指令）
输入：查看状态

# 5. 切换文风（使用明确指令）
输入：切换文风武侠风
```

**注意**：
- 所有指令都支持自然口语表达，无需使用#符号
- 同一指令有多种说法，选择最顺口的就行
- 推荐使用明确表达，避免误判：
  - ✅ 推荐："切换文风武侠风"、"探索森林"、"查看状态"
  - ⚠️ 避免："我想切换文风"、"能探索森林吗"、"角色扮演真有趣"（可能被判定为普通对话）
- 系统会智能分析上下文，置信度达到70%以上才触发指令

## 常见问题

**Q: 为什么指令以 # 开头？**
A: # 指令具有最高优先级，会立即中断当前剧情执行。

**Q: 三相循环是什么？**
A: 造物（许愿）、空白（熵增）、秩序（冻结）三种力量的动态平衡。

**Q: 默认文风是什么？**
A: 搞笑甜宠网文风，包含沙雕、反差萌、网络梗等特征。

**Q: 如何改变文风？**
A: 使用 `#切换文风到：[名称]` 指令，详见第二章文风系统。
