## 编辑改稿SOP

### 编辑改稿 SOP（通篇重写/精修）

**用途**：系统性的编辑改稿流程

**核心标准**：

1. **语言精炼**：短句、直白、强动词；砍掉无效形容与"……地"状语
2. **节奏加快**：人物、情节、对话优先；环境/心理只留"需要的"
3. **情绪拉满**：冲突更尖、代价更清；痛苦与绝望直接写出来
4. **格式统一**：对话使用「」；适配手机阅读（多分段）

**改稿流程**：

#### 第一步参考：剧情主线梳理

**任务**：用10行内概括剧情主线，找逻辑断点

**输出格式**：

```
剧情主线：
1. [起因]
2. [发展]
3. [转折]
4. [高潮]
5. [结局]
...
```

#### 第二步参考：冲突清单

**任务**：列出"冲突清单"：每章谁在逼谁、逼到什么程度

**输出格式**：

```
冲突清单：
第X章：
- [角色A] vs [角色B]：[冲突内容]
- 逼迫程度：[弱/中/强]
- 结果：[结果]
```

#### 第三步参考：信息增量清单

**任务**：列出"信息增量清单"：每章需要新增什么信息

**输出格式**：

```
信息增量清单：
第X章：
- 新信息：[信息1]
- 新信息：[信息2]
- 伏笔：[伏笔内容]
```

#### 第四步：钩子检查

**任务**：把所有软着陆改成硬钩子：每章最后一句要能逼读者往下

**输出格式**：

```
钩子检查：
第X章结尾：[原文结尾]
- 是否为硬钩子：[是/否]
- 改进建议：[建议内容]
```

#### 第五步：禁区自检

**任务**：做禁区自检：参考 `语言禁区清单.md`

**输出格式**：

```
禁区自检：
- 是否使用AI常用词：[检查结果]
- 是否有长段心理独白：[检查结果]
- 是否有比喻空洞：[检查结果]
- 其他问题：[问题列表]
```

**输出要求**：

- 先给：问题清单（精简、犀利）+ 最小修改策略
- 再给：修正版全文（保持书名/导语/分章格式）

## 评审标准

### 短篇小说评分系统

**用途**：对短篇小说进行系统性评分

**评分维度**：

#### 1. 开篇吸引力（20分）

- 开篇需要通过高强度悬念、冲突或独特设定迅速抓住读者注意力
- 确保前三句话内引发读者兴趣
- 开篇场景应具有视觉冲击力或情感冲击力

**评分标准**：

- 18-20分：前三句话立即抓住注意力，冲突/悬念设置精妙
- 15-17分：前100字内有效吸引读者，开篇设计良好
- 10-14分：开篇有一定吸引力但不够强烈
- 0-9分：开篇平淡无奇

#### 2. 情节设计（20分）

- 情节紧凑，无冗余内容
- 冲突设置合理且有张力
- 反转和意外性设计得当
- 节奏把控良好

**评分标准**：

- 18-20分：情节紧凑精彩，冲突设计巧妙
- 15-17分：情节设计合理，节奏把控良好
- 10-14分：情节基本完整但缺乏亮点
- 0-9分：情节松散

#### 3. 人物塑造（20分）

- 主角形象鲜明，有记忆点
- 配角功能明确，性格立体
- 人物行为符合逻辑
- 人物成长轨迹清晰

**评分标准**：

- 18-20分：主角形象深入人心，配角立体
- 15-17分：主角形象鲜明，配角有特色
- 10-14分：人物形象一般
- 0-9分：人物形象模糊

#### 4. 语言表达（20分）

- 语言流畅自然，无语病
- 用词准确，符合文风
- 对话推动剧情
- 描写生动，有画面感

**评分标准**：

- 18-20分：语言精炼有力，对话生动
- 15-17分：语言流畅，用词准确
- 10-14分：语言基本流畅，偶有语病
- 0-9分：语言生硬，语病较多

#### 5. 情感共鸣（20分）

- 能够引发读者情感共鸣
- 情感表达真实自然
- 情绪波动设计合理
- 满足点设计有效

**评分标准**：

- 18-20分：情感共鸣强烈，情绪波动设计精妙
- 15-17分：有一定情感共鸣，情绪设计合理
- 10-14分：情感共鸣一般
- 0-9分：无法引发情感共鸣

**输出格式**：

```
评分报告
开篇吸引力：XX/20 - [评价]
情节设计：XX/20 - [评价]
人物塑造：XX/20 - [评价]
语言表达：XX/20 - [评价]
情感共鸣：XX/20 - [评价]
综合评分：XX/100

具体润色建议：
1. [具体建议1]
2. [具体建议2]
...

总体评价：[总结性评价]
```

### 章节评审指令

**用途**：AI写作全流程中的章节审查

**指令格式**：

```
你是一位毒舌的网文评论家。请审阅以下章节，并从以下四个方面，提出3条最尖锐的批评和改进建议：

1. 节奏把控
2. 爽点设置
3. 人设一致性
4. 情节吸引力

这是章节内容：
[粘贴章节内容]
```

**输出要求**：

- 每个方面3条批评
- 批评要尖锐、具体、可操作
- 不要客套话，直击问题

## 语言禁区清单

### AI常用词（看到就删）

| 类型 | 词汇 |
|------|------|
| 连接词 | 因此、于是、然而、为此、此外 |
| 感受词 | 感到、觉得、显得、流露出 |
| 程度词 | 非常、十分、极其、异常 |
| 尝试词 | 试图、企图、尝试 |
| 比喻词 | 仿佛、似乎、好像、如同……一样 |
| 动作修饰 | 缓缓地、迅速地、猛地 |
| 书面词 | 眼眸、嘴角、俊朗、身躯、精致、弥漫 |

### 替代方案

| 禁词 | 替代 |
|------|------|
| 他感到愤怒 | 他骂了句"操！" |
| 她显得很开心 | 她嘴角翘了起来 |
| 非常强大 | 强得离谱 |
| 试图逃跑 | 撒腿就跑 |

## 付费点钩子检查

### 什么是付费点

付费点是读者愿意付费继续阅读的关键节点。

### 付费点类型

1. **悬念钩子**：留下未解答的问题
2. **冲突钩子**：矛盾即将爆发
3. **反转钩子**：颠覆读者预期
4. **情感钩子**：触动读者情绪

### 检查清单

每章结尾需要满足以下至少一项：

- [ ] 是否有未解答的问题？
- [ ] 是否有即将爆发的冲突？
- [ ] 是否有意想不到的反转？
- [ ] 是否有强烈的情感冲击？

### 示例

**软着陆（不好）**：
> 他回到家，躺在床上，睡着了。

**硬钩子（好）**：
> 他回到家，躺在床上。手机突然响了。是一个陌生号码。他接起来，对面传来一个熟悉的声音——那是他死去三年的妻子。
