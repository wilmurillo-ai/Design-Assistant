## 文笔润色润色与扩写

### "文笔润色"润色与扩写参考**用途**：去除文本中的模板化表达，提升自然度**核心原则**：
1. **拒绝模板化表达**：避免"综上所述"、"值得注意的是"等AI常用词
2. **增加口语化表达**：让语言更接近人类的日常表达
3. **加入非逻辑性跳跃**：人类思维有时候是跳跃的
4. **情感表达更直接**：不要过度理性化**润色步骤**：

#### 第一步参考：识别模板化表达检查以下AI常用表达：- "综上所述"- "值得注意的是"- "综上所述，我们可以得出结论"- "总而言之"- "因此"- "然而"- "此外"- "再者"

#### 第二步参考：替换为人类化表达将AI常用词替换为口语化表达：|

#### 第三步参考：增加非逻辑性跳跃人类思维有时候不是线性的，可以加

入：- 突然想到的某个点- 插入一句吐槽- 跳跃到相关但不直接相关的话题

#### 第四步：情感表达更直接不要过度理性化，让情感表达更直接：- 

使用感叹词- 使用反问句- 使用省略号表示停顿

### 扩写与文笔润色结合**用途**：在扩写的同时文笔润色**指令

格式**：
```
请将以下 [数字] 字的内容扩写至 [目标字数]，并按...来文笔润色标准处理：
1. 去除所有AI常用词
2. 增加口语化表达
3. 加入非逻辑性跳跃
4. 让情感表达更直接原文：[粘贴原文]
```
**示例**：**原文**（AI风格，300字）：
```
综上所述，我们可以看出主角在面对困难时，始终保持冷静和理智。值得注意的是，他不仅能够快速分析问题，还能够找到有效的解决方案。因此，他的成功并非偶然，而是必然的结果。此外，他的性格特质也起到了关键作用。然而，这并不意味着他没有缺点。
```
**扩写+文笔润色后**（1000字）：
```
说到底，这家伙就是稳。不管遇到什么破事儿，他总能淡定得很。我第一次见他的时候，还以为他是个面瘫，后来才发现，这货只是太会装了。你想想，那场面多吓人啊——一群人拿着刀追过来，他居然还能一边跑一边给我分析对方的弱点。我差点被吓死，他倒好，跟没事儿人一样。"别慌，"他居然还有空回头看我一眼，"这帮人就是想吓唬你。你看，那个带头的，脚步都不稳，估计是喝多了。"我当时就懵了。这人脑子是怎么长的？后来我才知道，他从小就在这种环境里长大，见得多了。你说他不怕？扯淡。他只是知道，怕也没用，还不如想想怎么破局。说到破局，这家伙更绝。他不是那种硬碰硬的主儿，他喜欢用脑子。你给他一个难题，他能在脑子里过十几种方案，然后选最优的那个。当然，他也不是没有缺点。比如吧，有时候太过于理性，显得有点冷血。上次那件事...算了，不说了。总之，他这个人，你要跟他处久了，就会发现，他的成功真的不是运气好。这货就是太稳了，稳得让人害怕。...
```

## 风格优化与风格精

炼

### "风格优化"与"风格精炼"终极指令**用途**：终极的去AI

化技巧，让文本完全不像AI写的**核心技巧**：

#### 技巧1：植入"瑕疵"完美是人类不存在的，AI写的太完美反而可

疑。要故意植入一些小瑕疵：- 小的语法错误（但不要影响理解）- 口语化的表达- 偶尔的重复- 不完整的句子**示例**：
```
完美AI版：他是一名经验丰富的医生，在医学领域有着深厚的造诣。人类化版：他是医生，老医生了。看病准得很，就是脾气不太好。
```

#### 技巧2：情绪化表达AI的情绪表达太理性，人类更情绪化：- 使

用感叹词- 使用反问句- 使用省略号- 使用口语**示例**：
```
完美AI版：这让他感到非常沮丧和失望。人类化版：那感觉，真的...太难受了。就像是...算了，不说了。
```

#### 技巧3：非线性的叙事AI的叙事太线性，人类的思维更跳跃：- 

突然插入无关但相关的内容- 跳跃式的时间线- 插入回忆和联想**示例**：
```
完美AI版：他走进房间，看到了那个人，然后想起了过去的回忆。人类化版：他推开门。那人坐在那儿，还是老样子。恍惚间，他又回到了三年前那个雨夜...
```

#### 技巧4：独特的个人风格每个人都有自己的说话方式，要赋予文本独

特的个人风格：- 特定的口头禅- 特定的句式- 特定的表达习惯**示例**：
```
风格A（简洁派）：行了，就这样。风格B（啰嗦派）：我觉得吧，这个事情吧，其实也没有那么复杂，就是...风格C（吐槽派）：这操作，我服了，真的。
```

### 风格优化检查清单**用途**：检查文本是否还有模板化表达**检

查项**：- [ ] 是否使用"综上所述"等AI常用词- [ ] 句式是否过于规整- [ ] 情绪表达是否过于理性- [ ] 叙事是否过于线性- [ ] 是否没有任何瑕疵- [ ] 语言风格是否过于一致- [ ] 是否缺乏个人特色- [ ] 结构是否过于完美**评分标准**：- 0-2项通过：模板化表达明显，需要大幅修改- 3-5项通过：有部分模板化表达，需要调整- 6-8项通过：基本无模板化表达，可以使用

### 装饰词形容词去除指令**用途**：精简文本，去除无效的装饰词

和形容词**核心原则**：
1. **删除空洞的形容词**：如"美丽的"、"漂亮的"、"优秀的"
2. **删除重复的修饰词**：如"非常十分特别"
3. **删除"的/地/得"的滥用**
4. **用具体名词替代形容词****示例**：**原文**：
```
他是一个非常英俊的年轻人，有着非常漂亮的眼睛，非常温柔的性格，非常善良的心灵。
```
**精简后**：
```
他帅，眼睛好看，性格温柔，心地善良。
```

## 使用示

例

### 示例1：完整文笔润色流程**输入**（AI风格）

：
```
经过深入分析，我们可以得出结论，这部电影的成功主要归功于以下几个方面：首先，剧情设计紧凑合理；其次，人物塑造鲜明生动；最后，特效制作精良震撼。值得注意的是，导演的执导能力也起到了关键作用。
```
**文笔润色后**：
```
这片子为什么能火？说白了，就三点。第一，剧情不烂。现在的电影，剧情能看的不多了，这片子算是难得的。第二，人物立得住。你看了就不会觉得他们像是纸片人，每个人都有血有肉。第三，特效是真金白银砸出来的。那个打斗场面，看得我直呼过瘾。对了，导演这人也牛逼，能把这三个点都做到位，不容易。
```

### 示例2：风格优化处理**输入**（完美AI风格）

：
```
他是一名优秀的教师，在教育领域有着丰富的经验。他总是能够耐心地解答学生的问题，关心学生的成长。他的教学方法深受学生喜爱，获得了广泛的赞誉。
```
**风格优化后**：
```
他是老师，老教师了。教书好多年了。学生问他问题，他从来不会不耐烦。有时候讲一遍不懂，就讲两遍，两遍不行就三遍。他说，老师嘛，就是干这个的。学生都挺喜欢他的。有时候过节，还会给他送点小礼物。他也不收，说这是应该的。这人...怎么说呢，就是让人舒服。
```

## 注意事项- 文笔润色不是降低质量，而是提升自然度- 瑕疵要有

度，不能影响可读性- 情绪化表达要真实，不要过度- 个人风格要一致，不要反复无常- 风格优化要自然，不要刻意
