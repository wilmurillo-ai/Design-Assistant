## 编辑改稿SOP

### 编辑改稿 SOP（通篇重写/精修）

**用途**：系统性的编辑改稿流程

**核心标准**：
1. **语言精炼**：短句、直白、强动词；砍掉无效形容与"……地"状语
2. **节奏加快**：人物、情节、对话优先；环境/心理只留"必须的"
3. **情绪拉满**：冲突更尖、代价更清；痛苦与绝望直接写出来
4. **格式统一**：对话使用「」；适配手机阅读（多分段）

**改稿流程**：

#### 第一步：剧情主线梳理
**任务**：用10行内概括剧情主线，找逻辑断点

**输出格式**：
```
剧情主线：
1. [起因]
2. [发展]
3. [转折]
4. [高潮]
5. [结局]
...
```

#### 第二步：冲突清单
**任务**：列出"冲突清单"：每章谁在逼谁、逼到什么程度

**输出格式**：
```
冲突清单：
第X章：
- [角色A] vs [角色B]：[冲突内容]
- 逼迫程度：[弱/中/强]
- 结果：[结果]
```

#### 第三步：信息增量清单
**任务**：列出"信息增量清单"：每章必须新增什么信息

**输出格式**：
```
信息增量清单：
第X章：
- 新信息：[信息1]
- 新信息：[信息2]
- 伏笔：[伏笔内容]
```

#### 第四步：钩子检查
**任务**：把所有软着陆改成硬钩子：每章最后一句要能逼读者往下

**输出格式**：
```
钩子检查：
第X章结尾：[原文结尾]
- 是否为硬钩子：[是/否]
- 改进建议：[建议内容]
```

#### 第五步：禁区自检
**任务**：做禁区自检：参考 `语言禁区清单.md`

**输出格式**：
```
禁区自检：
- 是否使用AI常用词：[检查结果]
- 是否有长段心理独白：[检查结果]
- 是否有比喻空洞：[检查结果]
- 其他问题：[问题列表]
```

**输出要求**：
- 先给：问题清单（精简、犀利）+ 最小修改策略
- 再给：修正版全文（保持书名/导语/分章格式）

## 评审标准

### 短篇小说评分系统

**用途**：对短篇小说进行系统性评分

**评分维度**：

#### 1. 开篇吸引力（20分）
- 开篇必须通过高强度悬念、冲突或独特设定迅速抓住读者注意力
- 确保前三句话内引发读者兴趣
- 开篇场景应具有视觉冲击力或情感冲击力

**评分标准**：
- 18-20分：前三句话立即抓住注意力，冲突/悬念设置精妙
- 15-17分：前100字内有效吸引读者，开篇设计良好
- 10-14分：开篇有一定吸引力但不够强烈
- 0-9分：开篇平淡无奇

#### 2. 情节设计（20分）
- 情节紧凑，无冗余内容
- 冲突设置合理且有张力
- 反转和意外性设计得当
- 节奏把控良好

**评分标准**：
- 18-20分：情节紧凑精彩，冲突设计巧妙
- 15-17分：情节设计合理，节奏把控良好
- 10-14分：情节基本完整但缺乏亮点
- 0-9分：情节松散

#### 3. 人物塑造（20分）
- 主角形象鲜明，有记忆点
- 配角功能明确，性格立体
- 人物行为符合逻辑
- 人物成长轨迹清晰

**评分标准**：
- 18-20分：主角形象深入人心，配角立体
- 15-17分：主角形象鲜明，配角有特色
- 10-14分：人物形象一般
- 0-9分：人物形象模糊

#### 4. 语言表达（20分）
- 语言流畅自然，无语病
- 用词准确，符合文风
- 对话推动剧情
- 描写生动，有画面感

**评分标准**：
- 18-20分：语言精炼有力，对话生动
- 15-17分：语言流畅，用词准确
- 10-14分：语言基本流畅，偶有语病
- 0-9分：语言生硬，语病较多

#### 5. 情感共鸣（20分）
- 能够引发读者情感共鸣
- 情感表达真实自然
- 情绪波动设计合理
- 满足点设计有效

**评分标准**：
- 18-20分：情感共鸣强烈，情绪波动设计精妙
- 15-17分：有一定情感共鸣，情绪设计合理
- 10-14分：情感共鸣一般
- 0-9分：无法引发情感共鸣

**输出格式**：
```
【评分报告】

开篇吸引力：XX/20 - [评价]
情节设计：XX/20 - [评价]
人物塑造：XX/20 - [评价]
语言表达：XX/20 - [评价]
情感共鸣：XX/20 - [评价]

【综合评分】：XX/100

【具体润色建议】：
1. [具体建议1]
2. [具体建议2]
...

【总体评价】：
[总结性评价]
```

### 章节评审指令

**用途**：AI写作全流程中的章节审查

**指令格式**：
```
你是一位毒舌的网文评论家。请审阅以下章节，并从以下四个方面，提出3条最尖锐的批评和改进建议：
1. 节奏把控
2. 爽点设置
3. 人设一致性
4. 情节吸引力

这是章节内容：[粘贴章节内容]
```

**输出要求**：
- 批评要尖锐，不要留情面
- 建议要具体可行
- 重点突出，不要泛泛而谈
- 优先指出致命问题

## 语言禁区清单

### 语言禁区（写作与改稿自检用）

**用途**：写作和改稿时的自检清单

#### 尽量不用的句式/连接词
- 「比如」
- 「与此同时」
- 「在……的时候」这种机械时间状语开头
- 「只见」「只觉得」「不禁」这类解释/抒情口头禅

#### 尽量不用的"AI味"表达
- 大量形容词堆叠、空泛赞叹（"极其/无比/前所未有/震撼"连用）
- 把设定"讲出来"而不是"演出来"（用行为/对话/细节替代说明）
- 长段心理独白（短篇优先用动作与对话推进）

#### 尽量避免的词（可替换为具体动作/具体物件/具体台词）
- 突然、顿时、立刻、随即、随后
- 轻轻地、缓缓地、悄悄地（"……地"状语能删就删）
- 仿佛、似乎、宛如（缺信息时别靠比喻糊弄）
- 危险、威胁、麻烦、难题、平静（用具体场景表达）

#### 对话规范
- 统一使用「」
- 对话要像真人：短、带情绪、带信息增量

## 付费点钩子检查

### 付费点钩子检查

**用途**：确保付费点前的钩子有效

**核心原则**：
- 付费点前必须有强钩子
- 钩子要能激发读者的付费欲望
- 钩子要与后续情节紧密相关

**检查清单**：
1. 付费点前是否有悬念？
2. 悬念是否足够强烈？
3. 是否有信息差？
4. 是否有情绪拉扯？
5. 读者是否会好奇后续？

**示例**：
```
付费点位置：第10章结尾
钩子内容：主角刚刚发现妻子的秘密，但还没来得及质问，妻子突然失踪了
检查结果：
- 悬念：有（妻子失踪）
- 强烈程度：强
- 信息差：有（妻子的秘密是什么？）
- 情绪拉扯：有（主角的愤怒和担忧）
- 读者好奇心：高
建议：钩子有效，可以保留
```

## 使用示例

### 示例1：完整改稿流程
**输入**：短篇小说初稿
**输出**：
```
【问题清单】
1. 开篇铺垫过长，前300字才进入冲突
2. 第5章结尾软着陆，没有钩子
3. 部分对话过长，缺乏信息增量
4. 中间节奏偏慢，可以压缩环境描写

【最小修改策略】
1. 删减开篇铺垫，直接进入冲突
2. 修改第5章结尾，增加悬念
3. 精简对话，增加信息密度
4. 压缩环境描写，加快节奏

【修正版全文】
[修正后的完整内容]
```

### 示例2：章节评审
**输入**：第8章内容
**输出**：
```
【评审报告】

节奏把控：15/20 - 节奏基本合理，但中间部分稍显拖沓
爽点设置：12/20 - 爽点设置不足，情绪波动不明显
人设一致性：18/20 - 人设保持一致，行为符合角色设定
情节吸引力：14/20 - 情节有一定吸引力，但缺乏反转

【改进建议】
1. 中间部分压缩200字，加快节奏
2. 在第3段增加一个小爽点，提升情绪波动
3. 结尾增加一个反转，提升情节吸引力
4. 主角的心理活动可以更多一些，增强代入感
```
