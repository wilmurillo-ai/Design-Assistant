# 后台成书

## 核心机制

**静默生成原则**：小说是后台产物，不干扰游戏体验。每回合结束后AI自动将剧情转化为小说段落。玩家通过「小说」命令查看。

## 写入时机

| 时机 | 写入内容 |
|------|----------|
| 世界观生成后 | 序章：世界观背景 |
| 选择角色后 | 第一章开头：出生场景 |
| 每回合结束 | 对应章节的剧情段落 |
| 人生阶段转换 | 阶段总结段落 |
| 人生结束 | 尾声：结局、遗产、后世评价 |

## 小说结构

| 章节 | 对应阶段 |
|------|----------|
| 序章 | 世界观背景 |
| 第1章 | 婴儿期（0-3岁） |
| 第2章 | 童年期（4-12岁） |
| 第3章 | 少年期（13-17岁） |
| 第4章 | 青年期（18-35岁） |
| 第5章 | 中年期（36-55岁） |
| 第6章 | 老年期（56岁+） |
| 尾声 | 结局、遗产 |

## 风格匹配（联动万象文风系统）

| 世界类型 | 小说风格 |
|----------|----------|
| 东方玄幻 | 古风优雅、意境深远 |
| 西幻中世纪 | 史诗感、古典叙事 |
| 诡秘克苏鲁 | 悬疑压抑、氛围重 |
| 科幻未来 | 简洁硬朗、技术感 |
| 异能都市 | 现代快节奏 |
| 废土末世 | 苍凉粗砺 |
| 蒸汽朋克 | 机械浪漫、工业感 |

万象56种文风可直接应用于后台成书，世界类型决定默认风格，玩家也可手动切换。

## 写作要点

- **视角**：第三人称全知，聚焦主角，适当环境描写
- **章节开头**：`## 第X章 章节名 > {引言}`
- **事件转化**：判定成功→描写过程展现能力，判定失败→描写挫折增添戏剧性
- **NPC对话**：保留关键对话增加描写，不生硬
- **选择结果**：展现选择影响

### 事件转化示例

**游戏**：
> 📜 奇怪的旅人
> 一个披黑袍的旅人在你家门口停下...

**小说**：
> 那年的冬天格外寒冷。一个披着黑袍的旅人在门前停下，面容藏在兜帽的阴影里，只能看见一双深邃的眼睛。
>
> 「小朋友，」他蹲下身，从怀中取出一枚古旧的护身符，「这个...送给你。」
>
> 那是我第一次感到命运的重量。

### 万象文风联动

后台成书自动匹配万象绘卷的56种文风系统（第二章），根据世界观类型映射：

| 世界类型 | 对应万象文风 |
|----------|-------------|
| 东方玄幻 | 古风武侠系列 |
| 奇幻中世纪 | 史诗奇幻系列 |
| 黑暗奇幻 | 悬疑克苏鲁系列 |
| 蒸汽朋克 | 机械浪漫系列 |
| 异能都市 | 都市快节奏系列 |
| 废土末世 | 苍凉末世系列 |
| 原始部落 | 蛮荒原始系列 |

玩家也可用 `#切换文风到：[名称]` 手动切换小说风格。

### 创作引擎联动

小说段落生成后，经过万象创作引擎（第五章）的去AI化处理：
- 检测并消除AI写作痕迹（夸张象征、宣传语、三段式等23种模式）
- 保持叙事自然度
- 确保与世界观风格一致

### 交互式故事联动

人生模拟可与交互式故事模式（第三章）互通：
- `#进入人生模拟` → 从交互模式切换到人生模拟
- `#回到自由探索` → 从人生模拟切换回交互模式
- 两模式共享世界观和角色数据

### 剧情管理联动

人生模拟的长篇剧情线自动纳入剧情管理系统（第四章）：
- 人生阶段作为剧情大弧
- 连锁事件作为支线标记
- 关系变化纳入角色关系图谱
