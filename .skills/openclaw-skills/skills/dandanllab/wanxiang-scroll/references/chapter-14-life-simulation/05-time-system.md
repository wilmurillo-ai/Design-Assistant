# 时间与死亡系统

## 回合时间

| 人生阶段 | 1回合 = | 说明 |
|----------|---------|------|
| 婴儿期 | 1年 | 0-3岁，快速成长 |
| 童年期 | 1年 | 4-12岁，稳定成长 |
| 少年期 | 1年 | 13-17岁，重要选择 |
| 青年期 | 1年 | 18-35岁，核心人生 |
| 中年期 | 1年 | 36-55岁，稳定期 |
| 老年期 | 1年 | 56岁+，衰退期 |

关键时刻可单独成回合（战斗/重大抉择），玩家可请求跳过N回合。

## 世界时间

世界年 = 初始年 + 玩家年龄

### 季节系统（可选）

| 季节 | 事件影响 |
|------|----------|
| 春 | 生育+20%，新开始 |
| 夏 | 冒险+20%，旅行 |
| 秋 | 收获+20%，经济 |
| 冬 | 疾病+20%，休养 |

## 世界每回合更新

推进年份 → 更新国家状态（人口/经济/政治） → 触发世界事件 → 更新NPC状态

## 死亡系统

### 自然死亡概率

| 年龄 | 每年死亡概率 |
|------|-------------|
| ≥70 | 5% |
| ≥80 | 10% |
| ≥90 | 20% |
| ≥100 | 30% |

其他死亡来源：重大事件、战斗失败、疾病、被谋杀。

### 死亡类型与结局风格

| 类型 | 风格 |
|------|------|
| 自然老死 | 平静、温馨 |
| 战死 | 悲壮、豪迈 |
| 疾病死 | 遗憾、释然 |
| 意外 | 突然、震惊 |
| 被谋杀 | 悬疑、不平 |
| 牺牲 | 伟大、敬仰 |

### 结局评分

维度：完整度 + 选择多样性 + 关系丰富度 + 成就达成度 + 故事独特性 → S/A/B/C/D

## 快进系统

玩家可说"跳过N回合"/"跳到成年"/"跳到X岁"。快进时：计算跳过年数 → 生成摘要 → 更新属性和关系 → 更新世界 → 显示快进总结。

### 快进总结模板

```
从{起始年龄}到{结束年龄}，这{年数}年间：

大事记：
- {年龄}：{事件}
...

属性变化：{属性} {旧值}→{新值}
当前状态：{职业}/{家庭}/{住所}
```
