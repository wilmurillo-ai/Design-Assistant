## 网文创作流程

### 第一步：开书前准备

#### 1.1 确定核心要素
- **题材**：玄幻、都市、言情、科幻、仙侠等
- **类型**：爽文、虐文、甜文、悬疑、喜剧等
- **目标读者**：男性向、女性向、全年龄向
- **核心卖点**：独特设定、爽点、情感线

#### 1.2 构建世界观
- **背景设定**：时代、地点、社会结构
- **力量体系**：等级、能力、限制
- **特殊设定**：金手指、系统、异世界等

#### 1.3 设计主角
- **身份背景**：出身、经历、性格
- **核心能力**：金手指、特长、成长路线
- **性格特点**：优点、缺点、成长弧光

### 第二步：黄金三章

#### 第一章：吸引注意
- 开篇冲突
- 主角登场
- 金手指展现
- 留下悬念

#### 第二章：展开设定
- 世界观介绍
- 配角登场
- 小高潮
- 推进剧情

#### 第三章：确立主线
- 主线任务明确
- 升级路线清晰
- 大BOSS暗示
- 为后续铺垫

### 第三步：细纲生成

#### 细纲结构
```markdown
## 第X章：[章节标题]

### 核心冲突
- 主要矛盾
- 次要矛盾

### 场景设置
- 时间
- 地点
- 氛围

### 角色行动
- 主角行动
- 配角行动
- 反派行动

### 关键情节
- 事件A
- 事件B
- 事件C

### 情感推进
- 角色情感变化
- 关系发展
- 伏笔埋设

### 结尾悬念
- 为下章铺垫
- 留下悬念
```

### 第四步：内容生成

#### 生成原则
1. **符合细纲**：严格按照细纲展开
2. **节奏控制**：张弛有度，节奏合理
3. **人物一致**：保持角色性格一致
4. **细节丰富**：增加感官描写和细节

#### 生成技巧
- 使用直接告知法（见第五章）
- 避免八股文表达
- 保持文风一致
- 增加互动感

### 第五步：平台适配

#### 起点风格
- **特点**：男性向、爽文、升级流
- **重点**：打脸、升级、装逼
- **节奏**：快节奏、密集爽点
- **文风**：简洁有力、爽感拉满

#### 番茄风格
- **特点**：脑洞大、更新快、爽文
- **重点**：金手指、系统、反套路
- **节奏**：超快节奏、短章节
- **文风**：通俗易懂、口语化

#### 知乎风格
- **特点**：故事感、真实性、思考性
- **重点**：人设、情感、共鸣
- **节奏**：适中、注重细节
- **文风**：文学性、深度

#### 晋江风格
- **特点**：女性向、情感细腻、文笔优美
- **重点**：情感线、人物塑造、氛围
- **节奏**：慢节奏、注重描写
- **文风**：优美细腻、情感丰富

## 内容生成技巧

### 1. 脑洞生成
- **跨界融合**：不同元素组合
- **反转套路**：经典套路创新
- **热点结合**：结合时事热点
- **用户需求**：满足读者期待

### 2. 细纲扩展
- **场景细化**：时间地点氛围
- **行动具体化**：角色具体行动
- **情感深化**：角色情感变化
- **冲突升级**：矛盾逐渐升级

### 3. 内容扩写
- **细节填充**：增加感官描写
- **对话丰富**：让对话更生动
- **心理描写**：深入角色内心
- **环境渲染**：营造氛围

### 4. 章节生成
- **开篇吸引**：开篇要有吸引力
- **中间推进**：情节稳步推进
- **结尾留悬念**：为下章铺垫
- **整体一致**：保持整体风格

## 注意事项

- 开书前准备要充分
- 黄金三章是关键
- 细纲要详细具体
- 内容生成要严格按细纲
- 平台风格要适配
- 保持更新节奏
- 重视读者反馈

## 模块联动

网文创作是万象的「生产流水线」，上下游紧密衔接：

- **→ 文风系统（第二章）**：平台适配（起点/番茄/知乎/晋江）调用对应文风库
- **→ 剧情管理（第四章）**：细纲生成→剧情大弧结构，节奏调节→三相循环节奏
- **→ 创作引擎（第五章）**：网文内容生成后经去AI化二次处理
- **→ 质量控制（第七章）**：网文评审标准（黄金三章/节奏/人设）把关
- **→ 辅助工具（第八章）**：书名生成器/简介生成器/角色设定工具直接用于网文
- **→ 拆书融合（第十一章）**：融合创作产出的网文可直接进入网文创作流程优化
- **→ 人生模拟（第十四章）**：后台成书的写作技巧可直接应用于网文创作
- **→ 李诞七步（第十五章）**：知识普及类网文（知乎短篇）适用李诞框架
