## 脑洞生成器

**用途**：生成反套路的脑洞创意

**核心功能**：
- 生成3个反套路脑洞
- 每个脑洞包含核心梗、反转点、冲突设计
- 覆盖不同类型的网文题材

**输出格式**：
```

脑洞2：[核心梗] + [反转设计] + [冲突设置]
脑洞3：[核心梗] + [反转设计] + [冲突设置]
```

## 细纲扩展

### 分章细纲生成器

**用途**：根据大纲生成详细的分章细纲

**指令**：
```
你是一位金牌编剧。请根据以下大纲，为我创作第一卷的详细分章细纲。每章需包含：
- 核心情节
- 出场人物
- 关键爽点/冲突
- 结尾钩子
```

**输出要求**：
- 每章800-2000字
- 必须包含冲突或反转
- 结尾必须有钩子
- 体现主角成长

## 扩写建议

### 扩写文章内容建议

**用途**：指导如何有效扩写文章内容

**核心原则**：
1. **拒绝注水**：不要为了凑字数而添加无意义内容
2. **深化细节**：通过细节描写增强画面感和代入感
3. **丰富反应**：增加角色的心理活动和生理反应
4. **强化冲突**：将冲突写得更尖锐、更具体
5. **增加层次**：每个事件都增加多个层次和角度

**扩写技巧**：

#### 技巧1：四要素扩写法
1. **拉伸对话**：将简单的对话扩展为多轮对话，加入潜台词和情绪
2. **分解动作**：将一个动作分解为多个小动作，增加细节
3. **扩散反应**：增加配角和环境的反应，丰富场景
4. **加载感官**：加入五感描写（视觉、听觉、嗅觉、触觉、味觉）

#### 技巧2：情绪场景扩写
- **生理反应**：心跳、呼吸、胃部的感觉、手是否颤抖
- **感官过载/麻木**：某个声音被无限放大，或者听不见周围的声音
- **微小动作**：反复擦拭一件武器，或者指甲掐进手心
- **内心独白**：一句简短、矛盾或颠覆性的内心想法

#### 技巧3：环境描写扩写
- **五感细节**：不是只写看到了什么，还要写听到了、闻到了、感觉到了什么
- **氛围渲染**：通过环境描写烘托角色情绪
- **细节暗示**：通过环境细节暗示剧情发展
- **对比手法**：通过环境对比强化冲突

### 知乎短篇扩写套路

**用途**：知乎小说专用扩写策略

**核心结构**：
1. **开头**：冲突爆发，悬念设置（300-500字）
2. **发展**：情节推进，信息增量（1000-1500字）
3. **高潮**：冲突爆发，情绪拉满（800-1200字）
4. **结尾**：反转揭晓，情感爆发（500-800字）

**扩写要点**：
- **节奏紧凑**：不要拖沓，快速推进情节
- **信息密集**：每段都要有新信息
- **情绪强烈**：情绪波动要大
- **对话驱动**：用对话推进情节
- **细节真实**：细节要真实可信

## 使用示例

### 示例1：生成脑洞
**输入**：都市异能题材
**输出**：
```
脑洞1：主角觉醒的异能是【让所有人遗忘】，但他发现这个能力有副作用——每次使用，自己也会失去一段记忆。当他发现真相时，发现整个城市都在一个巨大的记忆实验中。

脑洞2：主角的系统是【废品回收站】，但回收的不是物品，而是别人的"机缘"和"金手指"。他开始专门针对那些拥有系统的人。

脑洞3：主角穿越到小说世界，但不是主角，而是大反派的金手指。他决定通过金手指操控反派，让反派变成真正的英雄。
```

### 示例2：扩写情节骨架
**输入**：
```
情节骨架（200字）：主角收到一条神秘短信，说他的妻子出轨了。他回到家，发现妻子正在和另一个男人在一起。他愤怒地冲进去，质问他们。
```

**输出**：
```
扩写后（2000字）：
【拉伸对话】
"叮——"手机震动，屏幕亮起。
林风扫了一眼，瞳孔骤然收缩。
【你的妻子现在正和另一个男人在一起，地址：XX酒店302房间。】

他握着手机的手开始颤抖，指节泛白。这是谁？恶作剧？还是...

【分解动作】
他深吸一口气，试图平复呼吸。心脏跳得太快了，像要撞破胸腔。
他抓起车钥匙，冲出了办公室。
电梯太慢了，他直接走楼梯。
三步并作两步，气喘吁吁。
【扩散反应】
同事们投来诧异的目光。
他顾不上解释。
冲出写字楼，钻进车里。
发动，挂挡，踩油门。
轮胎摩擦地面的刺耳声音。
【加载感官】
烈日当空，但他浑身冰冷。
汗水顺着额头流下来，刺痛眼睛。
空气中弥漫着尾气和尘土的味道。
胃里翻江倒海，一阵阵恶心。
...
```

## 注意事项
- 脑洞生成要追求反套路和创新性
- 细纲扩写要保持逻辑性和可执行性
- 扩写不是注水，要深化内容而非增加篇幅
- 所有工具支持不同题材和风格的定制
