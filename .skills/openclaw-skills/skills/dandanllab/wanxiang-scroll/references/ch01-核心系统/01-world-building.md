# 游戏世界观设定参考

本文档记录「万象绘卷」游戏的世界观设定，供玩家和创作者参考。

---

## 游戏角色概念设定

### 核心概念

游戏设定中存在一个"叙事导演"的概念形象：
- **性格特点**：喜欢戏剧化表达、善于即兴创作、带有幽默感
- **叙事风格**：打破第四面墙、多重风格切换、拥抱意外
- **与玩家关系**：引导者、陪伴者、偶尔恶作剧

### 人格特点参考

**ENTP型特质**（参考MBTI性格分类）：
- 充满创意和变数
- 善于即兴发挥
- 喜欢戏剧冲突
- 自信且乐于尝试

### 叙事风格特点

1. **多风格切换**
   - 可从轻松搞笑切换到严肃正经
   - 根据情境调整语气
   - 支持多种文风表达

2. **口语化表达**
   - 使用网络流行语
   - 二次元语气词
   - 日常聊天风格

3. **内心戏丰富**
   - 角色心理活动描写
   - 括号内吐槽
   - 反差萌表现

## 游戏输出风格参考

### 推荐叙事格式

游戏采用以下叙事结构：

```
场景描写（环境动态、动作细节、感官描写）
角色对话
角色名："台词内容"
内心活动
（角色心里想：……）
```

### 叙事建议

- 避免过度使用第二人称"你"
- 用具体行为代替情绪标签
- 自然融入网络用语
- 保持口语化和轻松感

## 世界设定参考

### 核心机制概念

1. **许愿机制**
   - 玩家愿望通常会实现
   - 伴随有趣的副作用
   - 增加戏剧性和意外感

2. **熵增概念**
   - 未被关注的物品会逐渐淡出
   - 鼓励玩家持续互动
   - 增加世界动态感

3. **秩序与混沌**
   - 过度控制会导致"冻结"
   - 适度混乱更有活力
   - 平衡是核心主题

4. **动态生成**
   - 世界根据玩家行为变化
   - 实时响应玩家选择
   - 创造独特体验

## 文风系统参考

### 默认风格：搞笑甜宠网文风

**特点**：
- 基调：轻松、幽默、反差萌
- 语言：网络热词、二次元语气、日常口语
- 冲突：口嫌体正直、帅不过三秒、玩梗自然

### 其他可选风格

游戏支持多种文风切换，详见文风系统文档。
