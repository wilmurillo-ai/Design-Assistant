# 第一章：核心系统

本章是万象绘卷的基础参考，介绍游戏的世界观、物理法则和互动方式。

## 内容概览

### 1.1 世界观设定
**文件**：`01-world-building.md`

介绍了这个游戏世界的底层设定：
- 导演角色（ENTP-A型人格，爱吐槽但尽职）
- 故事输出格式（思考 + 正文 + 选项）
- 默认文风（搞笑甜宠网文风）
- 世界物理法则（三相循环：造物、空白、秩序）
- 互动响应优先级
- 敏感词处理方式

### 1.2 世界观与物理法则
**文件**：`02-world-rules.md`

描述游戏世界的运行规则：
- 环境设定：观测即现实的量子叙事场
- 三相循环：造物（许愿）、空白（熵增）、秩序（冻结）
- 视觉风格：赛博故障 x 奇幻童话
- 整体氛围基调

### 1.3 互动方式参考
**文件**：`03-interaction-reference.md`

介绍了玩家可以怎么跟游戏互动：
- 自然口语表达就行，不用背特殊格式
- 容错机制：说错了也能理解
- 常用操作：打开菜单、查看状态、进行行动等
- 角色操作：扮演、生成角色、查看角色
- 世界操作：探索、调查、创造
- 剧情控制：切换文风、总结剧情、调整节奏
- 智能识别：匹配度高才响应，避免误判

**已经熟悉的**：
1. 快速复习 1.1 和 1.3
2. 深入理解 1.2 的三相循环
3. 开始实际使用
