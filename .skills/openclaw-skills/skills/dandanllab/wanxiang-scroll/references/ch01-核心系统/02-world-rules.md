## 环境设定

### 本质：观测即现实

这是一个基于**元叙事（Meta-narrative）**与**高自由度沙盒**概念构建的动态环境。世界本身是一张巨大的白色画布（或充满雪花点的老旧屏幕），**只有当"你"（观测者）的目光投向某处，或发出操作时，该区域的物质、光影、物理法则才会瞬间生成。**

### 边缘地带
- 玩家视野之外的地方是一片虚无的"白噪（White Noise）"
- 偶尔会有未被使用功能报错：用场景化方式呈现（如"存档成功"变成"记忆封存在发光的水晶里"）
- 功能解释：让角色和场景自己说话
- 维持戏剧张力，不直接跳戏解释

## 世界法则的使用建议

### 作为创作者
1. **维持动态平衡**：不要让任何一方过度
2. **善用空白机制**：推动用户持续交互
3. **保持荒诞感**：在合理范围内制造意外
4. **场景化呈现**：即使是系统信息也要有画面感

### 作为观测者
1. **持续关注**：避免被空白吞噬
2. **适度控制**：不要过度追求秩序
3. **接受意外**：许愿总有副作用
4. **尽情创作**：你的想象是世界的源动力
