## 直接告知技法

### 核心概念

**直接告知**是一种通过叙述者的口吻，直接、明确地告诉读者角色的内心状态、情绪变化或外部表情的写作技法。

**核心特征**：
- 信息的"给予"，而非线索的"呈现"
- 明确表达，避免模棱两可
- 直接描写，无需读者猜测

---

### 技法要点

#### 原则1：避免"微表情"或"微动作"八股文

**建议避免的八股文格式**：
- "眼中闪过一丝..."
- "嘴角微微上扬..."
- "那双总是带着..."
- "一丝不易察觉的..."
- "不易察觉到..."
- "指节因为用力而有些发白..."
- "那双眼眸总是带着..."
- "一股前所未有..."

---

#### 原则2：使用"直接告知"范文体

**正确范例**：

1. **早坂爱心里吐槽着，脸上却渐渐浮现出笑容来。**
2. **被男人的直白给逗乐了，早坂爱差点笑出声来。**
3. **安洛笑着举手投降，从座位上的包里拿出了一个灰色的笔记本电脑——这是他上午用早坂爱的钱新买的。**
4. **安洛轻拍她的肩膀作安抚状，笑道，**
5. **霞之丘诗羽一脸茫然。**
6. **锦木千束的表情有点绷不住了，她的嗓门又大了一点，嚷道，**
7. **霞之丘诗羽疑惑地转过头，鼻子里忽然嗅到了一股有点熟悉的香气，面色微微变化。**
8. **喜多川海梦原本开朗的脸色立马变得阴沉下来，她实在装不下去了，连忙扑到了安洛怀里取暖，懊悔道，**
9. **听到房门响动，少女费力地扭头看了过来，然后就惊恐地瞪大了眼睛，但她刚想开口叫喊，安洛就冲上来一把捂住了她的嘴巴。**
10. **听他这么说，长崎素世差点把勺子给抖出去，"你没开玩笑吧？"**
11. **但安洛却是再次握紧了少女的手，微微一笑，**
12. **松坂砂糖心里有点抗拒，但看在安洛给钱的面子上还是眯起了眼睛，笑着答应了下来。**
13. **锦木千束装作抹眼泪一样的姿势，假哭道，**
14. **少女单手捂住了额头，疼得眼泪汪汪，**
15. **见来人是楚河这个老熟人，霞之丘诗羽感到心情轻松了许多，但嘴上却是说道，**

---

### 技法应用

#### 应用1：情绪表达

```
❌ 错误（八股文）：他眼中闪过一丝不易察觉的悲伤，嘴角微微下撇。
✅ 正确（直接告知）：他心里难过，脸上却努力保持着平静。
```

#### 应用2：心理活动

```
❌ 错误（八股文）：那双总是带着忧郁的眼眸里，闪过一丝复杂的光芒。
✅ 正确（直接告知）：他心里想着很多事情，但表情却没有太大变化。
```

#### 应用3：身体反应

```
❌ 错误（八股文）：她的指节因为用力而有些发白，那双手总是微微颤抖。
✅ 正确（直接告知）：她紧紧握着拳头，手心都在冒汗。
```

## 反八股文写作

### 创作禁忌

#### 语言方面

- 避免过度华丽的辞藻和生僻词汇
- 避免文言文式的表达方法
- 避免输出繁体字
- 不建议用比喻（像/仿佛/宛若）、拟人、排比等修辞

#### 情节方面

- 避免脸谱化的角色设定
- 避免角色行为前后矛盾
- 别让角色说出不符合身份的话

---

### 写作规范

#### 字数标准

- 正文字数需要符合标准：不少于3000字（针对长篇创作）

#### 分段原则

- 通常以一到七句话为一个段落，段落之间空一行
- 尽量满足"黄金分割线"，即：较短段落数量 : 较长段落数量 ≈ 1 : 1.618（即38.2%与61.8%）
- 适应手机等移动设备的屏幕阅读，避免大段文字给读者带来的压迫感

#### 对话格式

- 对话通常会单独成段，并且使用标准的引号「」
- 对话中减少省略号的使用

#### 减少括号使用

如果文本中出现"A（B）"格式时：

- 请在回复中使用括号内的正式表述替代括号外的非正式表述
- 请在您的回复中使用"B"而非"A"来代替原本格式，采用同义的方式来表示B
- 请完全采用括号内的表述方式，并在整个对话中保持一致性

## 情绪场景深度描写

### 情绪描写层次

#### 第一层：生理反应

**核心**：通过生理反应来表现情绪

**常用生理反应**：
- 心跳：加速、漏跳、沉重
- 呼吸：急促、屏住、困难
- 汗水：冷汗、热汗、手心出汗
- 肌肉：颤抖、紧绷、无力
- 胃部：翻腾、恶心、绞痛
- 眼睛：模糊、刺痛、发红
- 声音：嘶哑、颤抖、失声

**示例**：

```
听到这句话，林风的心猛地一沉。心跳像是要从胸腔里跳出来一样，砰砰砰地撞击着肋骨。他的呼吸变得急促，每次吸气都像是在吞刀片。手心全是冷汗，握紧拳头，指节泛白。胃里翻江倒海，一阵阵恶心涌上喉咙。
```

---

#### 第二层：感官过载/麻木

**核心**：通过感官的异常来表现情绪的极端状态

**感官过载**：
- 听觉放大：某个声音被无限放大
- 视觉模糊：眼前一片模糊或看到幻觉
- 触觉敏感：轻微的触碰都像刀割

**感官麻木**：
- 听觉丧失：听不到周围的声音
- 视觉迟钝：看不清眼前的事物
- 触觉消失：感觉不到疼痛

**示例**：

```
林风站在那里，周围的一切仿佛都静止了。耳边只剩下自己的呼吸声，沉重、急促，像是有人在耳边敲鼓。其他的声音都消失了——行人的嘈杂、汽车的鸣笛，统统听不见。眼前的一切都变得模糊，只有那个人影越来越清晰。他感觉自己像是在做梦，又像是在噩梦里。
```

---

#### 第三层：微小动作

**核心**：通过细微的动作来表现内心活动

**常见微小动作**：
- 握拳、松拳、握拳
- 咬唇、咬牙
- 眨眼、揉眼
- 擦汗、摸脸
- 摸口袋、拿物品
- 来回踱步、跺脚

**示例**：

```
林风没有说话，只是静静地站在那里。他的手在口袋里，紧紧握着那枚硬币。一次又一次，把硬币攥紧，又松开。硬币的边缘硌得手心生疼，但他好像感觉不到。他的嘴唇紧咬着，已经渗出了血丝。
```

---

#### 第四层：内心独白

**核心**：通过简短的内心想法来表现情绪

**内心独白特点**：
- 简短有力，一句话说清
- 矛盾或颠覆性
- 符合角色性格
- 有情感冲击力

**示例**：

```
林风看着那个人，心里只有一个念头。

"不能输。"

不能输。输了，就什么都没有了。

"为了他，为了所有相信我的人，我不能输。"

他深吸一口气，握紧了拳头。
```

## 对话潜台词设计

### 对话潜台词原则

#### 原则1：对话中蕴含深层含义

- 表面简洁甚至平凡
- 蕴含深层含义和情感暗流
- 言外之意丰富

#### 原则2：角色语言风格鲜明

- 每个角色都有独特的说话方式
- 符合身份和性格
- 有辨识度

---

### 对话示例

#### 示例1：简单对话中的潜台词

```
❌ 表面对话：
"你好。"林风说。
"你好。"对方回答。

✅ 带潜台词的对话：
"好久不见。"林风说，语气里带着一丝复杂的情绪。
"是挺久的。"对方笑了笑，但眼神却没有看林风。

潜台词：两人之间有故事，但现在都很尴尬。
```

#### 示例2：冲突对话中的潜台词

```
❌ 表面对话：
"我要杀了他。"
"别冲动。"

✅ 带潜台词的对话：
"他需要死。"林风的声音很冷，像是在陈述一个事实。
"我知道你恨他。"对方叹了口气，"但杀了他，你自己呢？"

潜台词：对方在劝林风不要因为仇恨毁了自己。
```

## 使用示例

### 示例1：完整情绪场景

```
原版：林风听到这个消息，很伤心。

扩写版（应用直接告知+情绪场景）：

林风听到这个消息，心里难受得要命。他站在那里，一动也不动。心跳漏了一拍，然后像是要从胸腔里跳出来一样，砰砰砰地撞击着肋骨。呼吸变得急促，每次吸气都像是在吞刀片，喉咙火辣辣地疼。手心全是冷汗，握紧拳头，指节泛白。胃里翻江倒海，一阵阵恶心涌上喉咙，让他不得不弯下腰。

"怎么会这样..."他喃喃自语，声音嘶哑。眼泪在眼眶里打转，但他强忍着没有让它掉下来。不能哭，不能哭。
```

### 示例2：完整对话场景

```
原版：
"你好。"林风说。
"你好。"对方回答。

优化版（应用潜台词设计）：

"好久不见。"林风说，嘴角勉强扯出一丝笑容。

"是啊，好久不见。"对方点了点头，但眼神却看向了别处。

"你..."林风想说什么，但又停住了。

"我该走了。"对方打断了他，转身欲走。

"等等！"林风喊道。

对方停下脚步，但没有回头。"有什么事吗？"

"我..."林风张了张嘴，却不知道该说什么。空气安静了。

"保重。"对方说，然后快步离开了。

林风看着他的背影，心里空荡荡的。
```

## 注意

- 直接告知要明确，不能模棱两可
- 建议避免使用任何八股文格式
- 情绪描写要适度，别过度
- 对话潜台词要合理，不能强行解读
- 所有技巧都要服务于故事和人物
