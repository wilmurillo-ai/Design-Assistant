## 导语生成器

### 万能导语生成器

**用途**：生成吸引人的小说导语（简介）

**核心要素**：
1. **主角设定**：主角的身份和特点
2. **金手指**：主角的特殊能力或优势
3. **核心冲突**：主角面临的主要挑战
4. **爽点预告**：暗示小说的爽点

**模板公式**：
```
[主角设定] + [意外事件/金手指觉醒] + [面临危机/挑战] + [反击/逆袭]
```

**输出格式**：
```
【导语】

[第一句]：主角的初始状态（简短有力）

[第二句]：意外事件或金手指觉醒

[第三句]：面临的危机或挑战

[第四句]：主角的反应或逆袭

[第五句]：爽点预告或悬念
```

---

### 不同类型导语模板

#### 都市爽文型
```
[主角]本是一个[平凡身份]，没想到[意外事件]。

从此，他觉醒了[金手指]。

面对[反派/挑战]，他不再忍耐。

"既然你不仁，就别怪我不义！"

且看主角如何[逆袭结果]，走上人生巅峰！
```

#### 玄幻修仙型
```
[主角]，原本是[废柴身份]，被所有人嘲笑。

直到那一天，[金手指觉醒]。

天地变色，乾坤逆转。

"三十年河东，三十年河西，莫欺少年穷！"

这一世，他要逆天改命，[目标]！
```

#### 悬疑推理型
```
[主角]本来过着平静的生活，直到[事件发生]。

一个巨大的阴谋浮出水面。

[线索/发现]

真相，远比想象中更可怕。

[悬念结尾]
```

#### 甜宠恋爱型
```
[主角]没想到，自己会爱上[男主/女主]。

从最初的[误会/冲突]，到后来的[心动/暗恋]。

"我愿意为你，放弃一切。"

这是一个关于[情感主题]的故事。

[甜宠预告]
```

## 书名生成器

### 万能书名生成器

**用途**：生成符合网文平台的书名

**核心公式**：

#### 公式1：身份+职业
```
[身份] + [职业] + （可选：之 + [特点]）
```
示例：
- 《魔道祖师》
- 《全职高手》
- 《盗墓笔记》

#### 公式2：冲突+金手指
```
[冲突] + ： + 金手指 + （可选：效果）
```
示例：
- 《废柴重生：开局获得无敌系统》
- 《都市之超级系统》
- 《重生之财源滚滚》

#### 公式3：动作+目标
```
[动作] + ： + 我要 + [目标]
```
示例：
- 《逆天改命：我要成神》
- 《重生之我要当首富》
- 《我要这天，再遮不住我眼》

#### 公式4：反差+悬念
```
[反差设定] + （可选：竟是 + [真相]）
```
示例：
- 《废柴竟是神之子》
- 《平凡人竟然是超级大佬》
- 《我的系统竟然是我的前世》

---

### 平台书名风格

#### 起点风格（长而详细）
- 特点：书名较长，信息量大，直接点明核心设定
- 示例：
  - 《我有一座恐怖屋》
  - 《大王饶命》
  - 《全职法师》

#### 番茄风格（简单直接）
- 特点：书名简短，爽点明确，一目了然
- 示例：
  - 《开局签到荒古圣体》
  - 《赘婿》
  - 《龙王殿》

#### 知乎风格（文艺有悬念）
- 特点：书名文艺，有悬念，引人好奇
- 示例：
  - 《那个夏天，我们一起追过的女孩》
  - 《如果时光可以倒流》
  - 《看不见的客人》

#### 晋江风格（唯美情感）
- 特点：书名唯美，情感浓郁，意境深远
- 示例：
  - 《何以笙箫默》
  - 《三生三世十里桃花》
  - 《微微一笑很倾城》

## 简介生成器

### 完整简介生成器

**用途**：生成完整的小说简介

**简介结构**：

#### 第一部分：核心设定（50-100字）
- 交代小说类型和核心设定
- 介绍主角的基本情况

#### 第二部分：剧情梗概（100-200字）
- 概括主线剧情
- 突出核心冲突

#### 第三部分：爽点预告（50-100字）
- 暗示小说的爽点
- 制造悬念

#### 第四部分：标签（可选）
- 列出3-5个核心标签

**输出格式**：
```
【小说简介】

[核心设定]

[剧情梗概]

[爽点预告]

【标签】[标签1] [标签2] [标签3]
```

---

### 简介示例

#### 示例1：都市爽文
```
【小说简介】

李阳，一个普通的快递员，在送餐途中意外捡到一个神秘包裹。

打开包裹后，他发现里面是一个超级系统。

从此，他的生活发生了翻天覆地的变化。

面对那些曾经看不起他的人，他不再忍耐。

"三十年河东，三十年河西，莫欺少年穷！"

且看他如何利用系统，走上人生巅峰，成为无人不知的传奇人物。

【标签】都市 系统 爽文 逆袭
```

#### 示例2：玄幻修仙
```
【小说简介】

林风，一个废柴少年，被所有人嘲笑。

直到有一天，他觉醒了上古血脉。

天地变色，乾坤逆转。

"既然你们不仁，就别怪我不义！"

这一世，他要逆天改命，成为无上仙帝！

【标签】玄幻 修仙 废柴流 热血
```

#### 示例3：悬疑推理
```
【小说简介】

一个普通的雨夜，一起离奇的凶杀案。

警察李阳奉命调查，却发现自己陷入了巨大的阴谋。

所有的线索都指向一个神秘的 organizations。

真相，远比想象中更可怕。

"有些事情，还是不知道的好。"

但已经晚了...

【标签】悬疑 推理 悬疑 惊悚
```

## 封面建议

### 封面设计要素

#### 起点风格（写实、宏大）
- 主题：主角形象或场景
- 风格：写实、精细
- 色调：根据小说类型调整（玄幻用金色/紫色，都市用蓝色/灰色）
- 元素：主角、武器、重要场景

#### 番茄风格（简单、醒目）
- 主题：核心设定或爽点
- 风格：简单、醒目
- 色调：鲜艳、对比强烈
- 元素：金手指、主角、关键物品

#### 知乎风格（文艺、意境）
- 主题：小说的核心情感或意境
- 风格：文艺、有意境
- 色调：柔和、温暖
- 元素：象征性物品、意境场景

#### 晋江风格（唯美、浪漫）
- 主题：人物关系或情感
- 风格：唯美、浪漫
- 色调：柔和、粉色系
- 元素：人物、花朵、浪漫场景

## 使用示例

### 示例1：生成完整导语
```
输入：
主角：李阳
身份：外卖员
金手指：万能外卖系统
冲突：被反派打压
爽点：逆袭成功

输出：
【导语】

李阳，一个普通的外卖员，每天都在城市的街头巷尾奔波。

直到有一天，他送外卖时意外获得了一个神奇的系统。

"恭喜宿主获得万能外卖系统！"

面对那些曾经看不起他的人，他不再忍耐。

"既然你不仁，就别怪我不义！"

且看他如何利用系统，走上人生巅峰，成为无人不知的传奇！
```

类型：都市爽文
核心设定：系统
爽点：逆袭

输出：
【书名选项】

1. 《都市之超级外卖系统》

## 注意事项
- 导语要简洁有力，抓住读者注意力
- 书名要符合平台风格，朗朗上口
- 简介要突出核心卖点和爽点
- 所有要素都要服务于小说类型和目标读者
