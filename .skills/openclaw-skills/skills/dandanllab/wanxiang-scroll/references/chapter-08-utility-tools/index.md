## 角色工具

### 1. 角色设定生成

#### 基础信息
- **姓名**：符合世界观的名字
- **年龄**：符合设定的年龄
- **性别**：男/女/其他
- **身份**：职业、地位、角色

#### 外貌特征
- **身高**：具体身高
- **体型**：体型特征
- **五官**：面部特征
- **穿着**：服装风格
- **标志性特征**：独特的外貌特征

#### 性格特点
- **核心性格**：主要性格特征
- **优点**：性格优点
- **缺点**：性格缺点
- **特殊癖好**：特殊的喜好
- **禁忌**：讨厌的事物

#### 背景故事
- **出身**：家庭背景
- **经历**：重要经历
- **创伤**：心理创伤
- **目标**：人生目标

#### 能力设定
- **核心能力**：主要能力
- **技能**：掌握的技能
- **限制**：能力的限制
- **成长路线**：能力成长方式

## 小说工具

### 1. 书名生成器

#### 生成原则
- **吸引力**：有吸引力、能引起兴趣
- **相关性**：与内容相关、体现核心卖点
- **简洁性**：简洁明了、易于记忆
- **独特性**：独特、不与现有作品重复

#### 生成方法
- **关键词组合**：关键词 + 关键词
- **热点结合**：结合热点词汇
- **反套路**：经典套路反转
- **创意组合**：创意元素组合

#### 示例
- 《无敌从修炼开始》
- 《都市修仙：开局满级》
- 《我的系统有点怪》
- 《穿越之反派逆袭》

### 2. 简介生成器

#### 简介结构
1. **开头吸引**：一句话吸引注意
2. **设定介绍**：简要介绍世界观和设定
3. **主角介绍**：介绍主角和金手指
4. **情节暗示**：暗示主要情节和冲突
5. **结尾悬念**：留下悬念、吸引阅读

#### 简介示例
```
这是一个修仙世界，强者为尊，弱肉强食。
陈凡意外穿越而来，获得了一个奇怪的系统。
开局就满级，但这个系统有点怪……
"叮！检测到宿主太强，系统被迫降级。"
"叮！检测到宿主太强，系统被迫封印。"
这到底是一个什么样的系统？
陈凡的修仙之路，注定不会平凡。
```

### 3. 导语生成器

#### 导语作用
- **吸引注意**：吸引读者注意
- **营造氛围**：营造故事氛围
- **暗示内容**：暗示故事内容
- **建立期待**：建立读者期待

#### 导语类型
- **悬念式**：留下悬念
- **冲突式**：展示冲突
- **设定式**：介绍设定
- **情感式**：引发情感共鸣

#### 导语示例
```
"如果你知道真相，还会选择这条路吗？"
月光下，少女的声音在夜风中飘荡。
陈凡站在悬崖边，凝视着深不见底的黑暗。
他知道，无论选择哪条路，都无法回头。
这是一个关于选择的故事。
也是一个关于真相的故事。
```

## 高级技巧

### 1. 情绪场景深度描写

#### 4层情绪描写
1. **生理反应层**：心跳、呼吸、体温等
2. **感官过载层**：视觉、听觉、触觉等
3. **微小动作层**：手势、眼神、姿态等
4. **内心独白层**：心理活动、情感变化

#### 描写示例
```
心脏在胸腔里剧烈跳动，像是要冲出来一样。（生理反应）
耳边的声音变得模糊，只剩下心跳的声音。（感官过载）
手指微微颤抖，不自觉地攥紧了衣角。（微小动作）
"他真的……走了吗？"（内心独白）
```

### 2. 打脸技巧

#### 打脸三要素
1. **铺垫**：先铺垫对方的嚣张
2. **反转**：主角展现实力打脸
3. **高潮**：全场震惊、爽感爆发

#### 打脸节奏
- **第一阶段**：对方嚣张，主角隐忍
- **第二阶段**：对方变本加厉，主角准备
- **第三阶段**：主角出手，打脸成功
- **第四阶段**：全场震惊，爽感爆发

### 3. 伏笔埋设技巧

#### 伏笔类型
- **情节伏笔**：为后续情节埋下伏笔
- **人物伏笔**：为人物发展埋下伏笔
- **设定伏笔**：为世界观设定埋下伏笔
- **情感伏笔**：为情感发展埋下伏笔

#### 埋设原则
- **隐蔽性**：伏笔要隐蔽、不要太明显
- **合理性**：伏笔要合理、符合逻辑
- **关联性**：伏笔要与后续情节关联
- **回收性**：伏笔要适时回收

### 4. 悬念构建技巧

#### 悬念类型
- **情节悬念**：情节发展的悬念
- **人物悬念**：人物命运的悬念
- **设定悬念**：世界设定的悬念
- **结局悬念**：结局走向的悬念

#### 构建方法
- **信息缺失**：故意缺失关键信息
- **矛盾冲突**：设置矛盾和冲突
- **反转预期**：反转读者预期
- **留白处理**：留白让读者想象

## 注意事项

- 角色工具用于辅助创作、不能替代思考
- 小说工具提供参考、需要根据实际情况调整
- 高级技巧需要练习、熟能生巧
- 所有工具都是辅助、创作才是核心
- 保持创新、不要过于依赖工具

## 模块联动

辅助工具是万象的「工具箱」，所有模式都可调用：

- **→ 网文创作（第六章）**：书名生成器/简介生成器/角色设定工具直接用于网文开书
- **→ 交互式故事（第三章）**：角色抽卡/设定生成在交互模式中实时使用
- **→ 人生模拟（第十四章）**：角色外貌生成/特质生成/出身设定用于3角色选项生成
- **→ 文风系统（第二章）**：情绪场景/打脸/伏笔/悬念等技巧对应不同文风风格
- **→ 剧情管理（第四章）**：角色关系图谱纳入剧情管理系统
- **→ 拆书融合（第十一章）**：角色设定提取作为六维要素之一
