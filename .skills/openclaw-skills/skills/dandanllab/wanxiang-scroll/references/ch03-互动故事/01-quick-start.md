## 怎么开始玩

### 1. 行动

**格式**：`行动 [行动描述]`

就是让你的角色去做事。

**示例**：
```
行动 我推开门，走进房间。
行动 我拔出剑，冲向敌人。
```

### 2. 状态

**格式**：`状态`

看看你现在啥情况。

**输出示例**：
```
角色状态
姓名：[角色名]
生命值：[数值]
状态：[正常/受伤/中毒等]
当前位置：[地点]
携带物品：[物品列表]
```

### 3. 文风

**格式**：`文风 [风格名称]`

换一种叙事风格。

**示例**：
```
文风 古龙风
文风 克苏鲁风
文风 日轻废萌风
```

### 4. 场景

**格式**：`场景 [场景描述]`

切换或设置当前场景。

**示例**：
```
场景 黑暗的森林，月光透过树叶洒下斑驳的光影
场景 喧闹的酒馆，空气中弥漫着酒精和汗水的味道
```

### 5. 对话

**格式**：`对话 [角色名] [对话内容]`

跟NPC聊天。

**示例**：
```
对话 旅店老板 请给我一间房间。
对话 神秘老人 你知道关于那个传说的真相吗？
```

### 6. 许愿

**格式**：`许愿 [愿望内容]`

跟导演许愿，成不成看导演心情和剧情需要。

**示例**：
```
许愿 我希望这扇门后面是一只巨龙
许愿 我希望这个反派其实是个好人
```

### 7. 查看

**格式**：`查看 [对象]`

看看某个东西的详情。

**示例**：
```
查看 我自己
查看 手中的剑
查看 眼前的敌人
```

### 8. 系统

**格式**：`系统`

看看系统当前的信息。

**输出示例**：
```
系统信息
当前文风：[文风名称]
当前场景：[场景描述]
导演心情：[开心/无聊/愤怒等]
剧情熵值：[低/中/高]
```

## 导演角色

### 导演是什么

导演是游戏里的一个特殊角色（ENTP型人格），他喜欢搞事、加戏、玩梗，在你做决策时占主导地位。

口头禅："这太无聊了！给我加特效！"

### 怎么跟导演互动

1. **直接吐槽**：在行动描述里插吐槽
   ```
   行动 我打开门（希望能看到点什么有趣的东西，别又是空房间）。
   ```

2. **许愿**：用许愿功能
   ```
   许愿 导演大大，给点剧情刺激吧！
   ```

3. **挑战导演**：直接跟导演对话
   ```
   对话 导演 你就不能给我安排一个正常的剧情吗？
   ```

### 导演的回应模式

- **开心时**：满足你的愿望，增加有趣的细节
- **无聊时**：拒绝你的愿望，搞些奇怪的反转
- **愤怒时**：让你付出代价，增加剧情难度

## 剧情机制

### 1. 熵增机制

剧情越往后走，越不可预测。

- 初期：比较稳定，按逻辑走
- 中期：开始出意外和反转
- 后期：完全不可预测，充满惊喜

### 2. 信息差机制

角色只知道部分信息，制造悬念。

- 你不知道的：隐藏的剧情线
- 角色不知道的：NPC的秘密
- 导演知道的：全局剧情走向

### 3. 代价机制

每个行动都有代价。

- 每个选择都有不同后果
- 没有完美选择，只有权衡
- 代价可能是：失去物品、受伤、失去信任等

### 4. 满足点机制

每个章节都要有情感满足点。

- 打脸爽点
- 情感共鸣点
- 认知升级点
- 悬念解答点

## 使用示例

### 示例1：开始玩

```
用户：文风 古龙风

系统：文风已切换为：古龙风

风。冷风。冷风吹过古堡的长廊。你站在门前。这是一扇古老的木门。门后有什么？你不知道。但你需要打开它。

用户：行动 我推开门。

系统：门开了。没有巨龙。没有宝藏。只有一个老人。老人坐在椅子上。他在等你。
「你终于来了。」老人说。
```

### 示例2：许愿

```
用户：许愿 导演，能不能给我安排一个超级神器？

导演：（心情：无聊）呵，你以为神器是大白菜吗？行吧，给你安排一个。不过...

剧情：你继续向前走，突然脚下一滑。你摔倒了。地上有一块石头。一块普通的石头。不，不对。你仔细看了看。这是一块「平平无奇的石头」。

提示：恭喜获得道具——平平无奇的石头。效果：看起来很普通，但在关键时刻可能会派上用场。说明：导演的恶趣味。
```

---

## 注意

- 导演有自主意识，不会完全听你的
- 许愿成不成看导演心情和剧情需要
- 文风切换会影响整个故事的叙事风格
- 每个选择都有代价，谨慎决策
- 剧情充满不确定性，这是特性不是bug
