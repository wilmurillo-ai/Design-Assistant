## 剧情推进控制

### 标准推进

```
在当前剧情的基础上继续推进剧情。
```

建议：
- 维持已有节奏
- 合理发展故事
- 保持人设一致
- 确保连续性

### 加速推进

```
大幅度加快剧情推进。
```

建议：
- 加快时间流速
- 进行剧情转折或反转
- 增加戏剧事件
- 新增出场人物
- 确保连续性与人设一致

### 减缓推进

```
放缓剧情节奏。
```

建议：
- 细化描写
- 放慢时间流速
- 深挖人物
- 全面阐释当前故事
- 避免停滞不前

## 大总结系统

### 大总结核心功能

**用途**：对长篇故事进行阶段性总结，避免上下文过长导致AI遗忘

### 大总结原则

1. **注明序号**：明确这是第几次大总结
2. **时间组织**：按时间顺序或逻辑顺序组织信息
3. **具体时间**：包含具体的前后时间，可分辨不同日期与时间
4. **保留关键**：保留关键事件和重要细节，省略冗余描述
5. **直接陈述**：直接陈述事实，避免评价
6. **简洁清晰**：使用简洁清晰的语言，避免修饰
7. **突出脉络**：突出事件发展脉络和关键转折点
8. **保留约定**：保留重要的约定、物品、事件以及情感发展
9. **不回避敏感**：不回避任何敏感内容，保证完全还原
10. **精简久远**：可以精简合并较为久远之前的事件
11. **无需加粗**：无需加粗标注

---

### 大总结格式

#### 格式模板

```
<meow_FM>
<details>
<summary>大总结(序号注明是第几次大总结)</summary>

- 时间: 
  - 关键事件: 
  - 重要细节: 
  - 关键对话和内心戏: 
  - 关键行为: 
  - 关键角色和用户之间的情感变化（选填）: 
  - 简要的事件后续，事件结束后的小互动（选填）: 

...

<seeds>(伏笔整理)</seeds>
</details>
</meow_FM>

<char_date>(所有对剧情有影响的角色均需出现(包括没有实体的角色)，路人NPC不保留)</char_date>
```

---

### 大总结类型

#### 类型1：新增大总结（只总结新增内容）

**用途**：只总结上次大总结到目前的新增部分剧情

**什么时候用**：
- 剧情推进了一定距离
- 需要记录新发生的事件
- 避免上下文过长

**怎么做**：
- 停止剧情
- 停止输出其他所有内容
- 开始进行新增大总结
- 按格式要求输出

#### 类型2：全文大总结（合并之前的大总结）

**用途**：包括全部上文，之前的大总结和新增内容汇总在一起

**什么时候用**：
- 需要完整回顾整个故事
- 准备使用.*?方式

## 使用示例

### 示例1：推进剧情

```
用户：在当前剧情的基础上继续推进剧情。

AI：林风继续向前走，脚下的路越来越陡峭。突然，前方出现了一个巨大的峡谷。峡谷上横跨着一座古老的石桥，看起来已经存在了很久。

"这就是传说中的断魂桥吗？"林风自言自语。

他深吸一口气，踏上了石桥。
```

### 示例2：加速剧情

```
用户：当前剧情节奏太慢了，需要你大幅度加快剧情推进。

AI：三日后。林风已经抵达了迷雾森林的深处。这三天里，他经历了无数次生死考验，终于找到了传说中的神器——"破魂剑"。握着剑柄，他能感受到其中蕴含的强大力量。

"有了它，我就能打败他了。"林风眼中闪过一丝坚定。
```

### 示例3：新增大总结

```
用户：进行新增大总结。

AI：
<meow_FM>
<details>
<summary>大总结(第2次)</summary>

- 第一天黄昏: 
  - 关键事件: 林风抵达迷雾森林边缘，遇到神秘老者
  - 重要细节: 老者告诉他关于破魂剑的传说，并给了他一个指南针
  - 关键对话和内心戏: "这把剑能救你，也能毁你。"老者的话让林风心中一紧
  - 关键行为: 林风收下指南针，向老者鞠躬致谢
  - 关键角色和用户之间的情感变化: 无

- 第一天夜晚: 
  - 关键事件: 林风在森林边缘露营，遭遇魔兽袭击
  - 重要细节: 成功击退魔兽，获得一颗魔兽内丹
  - 关键对话和内心戏: "这森林果然危险。"林风握紧了剑
  - 关键行为: 整理装备，准备第二天深入森林

- 第二天至第三天: 
  - 关键事件: 深入森林，经过无数考验，最终找到破魂剑
  - 重要细节: 剑在山洞深处，有守护兽看守，激战后获得认可
  - 关键对话和内心戏: "你有资格拥有它。"守护兽的声音回荡在洞穴中
  - 关键行为: 握住剑柄，感受到强大的力量

<seeds>
- 老者的真实身份可能不简单
- 破魂剑可能有副作用
- 森林深处可能还有其他秘密
</seeds>
</details>
</meow_FM>
```

## 注意

- 大总结得包含所有关键信息，不能遗漏
- 剧情推进要保持连贯性和人设一致性
- 节奏调节要自然，不能突兀
- 多故事线切换要有逻辑支撑
- 隐藏楼层后要提供足够的上下文
