# 认知偏差：为什么你总觉得自己是对的？> 李诞七步写作框架示例---## 第一步参考：开场故事（500字）周五晚上，咖啡馆。小明和小红在争论一个问题。小明说："我上周买股票，调研了三小时，分析了财报、行业趋势、宏观经济，最后决定买入。结果今天涨了15%。你看，研究是有用的。"小红说："那你之前买的那只呢？也研究了三个小时，怎么亏了30%？"小明愣了一下："那...那次是意外。市场突然出黑天鹅了，谁也预料不到。"小红笑了："所以你的逻辑是：涨了是我厉害，跌了是意外？"小明想反驳，但发现自己确实这么想。这就是我们今天的主题：**认知偏差**。不是知识不够，是你的大脑在骗你。---## 第二步参考：三种错误答案（各300字）### 错误答案一："我知道有偏差，所以我不会中招"这是最危险的错误。心理学家做过实验：给一群人讲解"确认偏误"（人只会看支持自己观点的证据），然后测试他们是否还会犯这个错误。结果？**学过的人犯得更多。**为什么？因为他们觉得自己"免疫"了，反而更自信，更不注意。**致命反例：** 医生学过"锚定效应"（最先看到的信息会影响判断），但在诊断时还是会过度依赖第一印象。知道自己有偏差，不等于能避免。### 错误答案二："认知偏差就是不聪明，聪明人不会"错了。智商160的人，偏差可能比普通人更严重。为什么？因为聪明人更擅长**为自己的偏差找理由**。普通人犯错了："我错了。"  聪明人犯错了："情况特殊""信息不全""对手太狡猾"...**致命反例：** 2008年金融危机前，最聪明的华尔街精英集体犯错。不是信息不够，是聪明人更会用复杂的模型，说服自己"这次不一样"。### 错误答案三："只要信息足够多，就能克服偏差"错。信息越多，偏差可能越严重。这叫"确认偏误的强化"：你带着偏见找信息，找到的全是支持你的。信息越多，你越坚信自己是对的。**致命反例：** 疫情期间的口罩争论。双方都能找到"科学证据"，信息越多，对立越严重。不是信息问题，是筛选信息的方式问题。---## 第三步参考：正确答案（800字）### 一句话定义**认知偏差：大脑为了节省能量，进化出来的思维捷径，但在现代社会经常失灵。**简单说：你的大脑是个懒鬼，能省则省，结果经常出错。### 为什么进化会这样？原始社会，你看到草丛动，最好的反应是"有狮子，跑！"而不是"先观察一下，收集数据，分析概率..."犹豫的原始人都被狮子吃了。剩下的，都是"反应过度"的。但现代社会，没有狮子。你的大脑还在用"防狮子"的模式，处理"股票投资""人际关系""政治观点"。 mismatch。### 核心特征1. **无意识**：你不知道自己正在犯错2. **普遍性**：所有人都有，智商高低不影响3. **系统性**：不是随机错误，是有规律的4. **难克服**：知道也没用，需要特定方法### 可验证标准怎么判断自己有没有认知偏差？问自己三个问题：1. **我有没有主动找过反对自己观点的证据？**   - 如果答案是没有 → 确认偏误警告2. **我的判断是否过度依赖第一印象/最先看到的信息？**   - 如果是 → 锚定效应警告3. **成功了我归因于能力，失败了我归因于运气/环境？**   - 如果是 → 自利性偏差警告---## 第四步：触类旁通（2500字）### 商业领域：为什么创业公司容易失败？创始人最常见的偏差：**过度自信+确认偏误**。他们坚信自己的点子能改变世界，只看好消息，忽略警告信号。投资人说不行？"他们不懂。"用户增长慢？"市场还没准备好。"直到钱花光。**真实案例：** 某知名共享经济公司，创始人看到了"共享"的趋势（确实对），但忽略了"高频刚需"的基本商业逻辑（确实错）。 confirmation bias 让他们只看到支持自己的案例（Airbnb、Uber），忽略了大量失败的共享项目。### 投资领域：为什么散户总亏钱？**损失厌恶 + 处置效应**。人讨厌损失的程度，是喜欢收益的两倍。所以：- 赚了钱的股票，早早卖掉（落袋为安）- 亏了的股票，死死抱住（不想承认失败）结果：好股票拿不住，坏股票抱到死。整体亏损。**数据：** 研究显示，散户卖出的盈利股票，后续平均还能涨15%；而持有的亏损股票，后续平均再跌20%。### 人际关系：为什么吵架时谁也说服不了谁？**基本归因错误**。别人做错事："他人品有问题。"  自己做错事："我那天心情不好/情况特殊/对方先惹我。"对别人用性格解释，对自己用情境解释。双标。**场景：** 情侣吵架。- 他迟到："他不重视我。"- 她迟到："路上太堵了/工作忙/手机没电。"同样的行为，不同的归因。吵不明白。### AI领域：为什么ChatGPT会"幻觉"？**模式匹配的过度应用**。AI和人一样，也会"脑补"。它看到"草...动..."，就像原始人脑补"狮子"一样，脑补出一个看似合理的答案。区别在于：- 原始人脑补错了，最多被狮子吃- AI脑补错了，可能生成虚假信息，被广泛传播**启示：** AI的"幻觉"不是bug，是它工作方式的副产品。就像人的认知偏差，不是bug，是进化遗产。### 日常生活：为什么你总觉得别人在针对你？**聚光灯效应 + 自我中心偏见**。你觉得自己是主角，所有人都在关注你。开会发言后，总觉得"刚才说错话了，大家肯定在笑我"。实际上？其他人都在想自己的事，根本没注意你。**实验：** 让受试者穿一件尴尬的T恤（比如印着一个过气明星）使用.*?方式。**数据：**- 原始人每天做约50个决策- 现代人每天做约35,000个决策（包括微小的）大脑的硬件没变，任务量翻了700倍。不出错才怪。### AI vs 人类**AI：** 没有认知偏差（理想情况下），纯理性计算，但缺乏常识和语境理解。**人类：** 充满认知偏差，但能理解语境，有常识，有创造力。**差距：** AI在需要精确计算的任务上碾压人类；但在需要"模糊判断""价值权衡"的任务上，人类仍然更强。**未来：** 不是AI取代人类，也不是人类胜过AI，而是**知道什么时候用AI，什么时候相信自己**。---## 第六步：结尾升华（300字）回到咖啡馆。小明沉默了很久，说："所以...我炒股亏钱，不是运气不好，是我的大脑在骗我？"小红点头："不只是你。所有人。包括我。""那怎么办？""没办法完全避免。但可以提醒自己：当你觉得'我肯定是对的'时，就是最需要怀疑自己的时候。"认知偏差不是bug，是feature的副作用。进化给我们的大脑安装了"快速反应"模式，让我们在草原上活了下来。只是，我们现在不在草原上了。**最后的洞察：** 知道自己有认知偏差，不会让你免疫。但会让你在犯错时，多一分警觉，少一分自信。而这，可能已经是人类能做到的最好的了。---## 第七步：延伸阅读（200字）### 🟢 入门（3本）1. 《思考，快与慢》- 丹尼尔·卡尼曼（偏差百科全书）2. 《黑天鹅》- 纳西姆·塔勒布（为什么你总被意外打脸）3. 《助推》- 理查德·塞勒（怎么用偏差改变行为）### 🟡 进阶（4本）4. 《错误的行为》-
