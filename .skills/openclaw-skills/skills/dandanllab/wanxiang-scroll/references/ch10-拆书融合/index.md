**文件**：`01-workflow-guide.md`

内容概览：
- 双模式（自动/手动）进行说明

**重要性**：拆书融合创作核心流程

## 模块联动

拆书融合是万象的「素材提炼器」，产出直接喂给其他模块：

- **→ 文风系统（第二章）**：提取源小说的文风要素作为56种文风的参考和扩展
- **→ 剧情管理（第四章）**：提取源小说的剧情结构作为大弧设计参考
- **→ 网文创作（第六章）**：融合创作的产出直接进入网文创作流程优化
- **→ 创作引擎（第五章）**：融合创作输出经文笔润色确保原创性
- **→ 质量控制（第七章）**：融合创作原创性检查，四换三原则确保75%原创
- **→ 拆书档案（第十一章）**：档案馆的小说素材是拆书融合的源材料
- **→ 人生模拟（第十三章）**：融合创作的世界观设定可作为人生模拟的世界模板
- **→ 辅助工具（第八章）**：角色设定提取作为六维要素之一
