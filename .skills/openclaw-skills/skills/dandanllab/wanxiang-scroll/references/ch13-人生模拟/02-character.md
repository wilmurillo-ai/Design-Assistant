# 角色与属性系统

## 角色生成

新游戏需要生成3个不同角色选项，种族、出身、天赋各不相同。玩家选择后进入人生。

三个角色应：定位不同（战斗型/智力型/均衡型）、各有优劣、风格各异、契合世界观。

---

## 属性系统

| 属性 | 范围 | 说明 |
|------|------|------|
| STR 力量 | 1-20 | 物理战斗能力 |
| DEX 敏捷 | 1-20 | 闪避和灵巧 |
| INT 智力 | 1-20 | 学习和魔法 |
| CHA 魅力 | 1-20 | 社交和领导 |
| LUK 幸运 | 1-20 | 随机事件概率 |
| SAN 理智 | 0-100 | 精神状态 |

属性评价：1-5极弱 / 6-8较弱 / 9-11普通 / 12-14良好 / 15-17优秀 / 18-20传说

总属性控制在60-75之间，根据出身有倾向调整。

---

## 特质系统

| 类型 | 说明 |
|------|------|
| ✓ 正面 | 有益效果 |
| ✗ 负面 | 有害效果 |
| ◆ 中性 | 双面效果 |

正面示例：灵识觉醒（感知超自然）、过目不忘（学习+30%）、铁意志（SAN上限+20）

负面示例：妖血诅咒（满月SAN-10）、厄运体质（负面事件+20%）、体弱（生命上限-20%）

中性示例：灵体缠身（帮忙也捣乱）、命运之子（幸运+30%但吸引危险）

---

## 外貌生成

需要含：性别、发色发长、眼色特征、肤色、身材、特殊标记（疤痕/胎记）。种族决定典型外貌特征。

---

## 出身类型

| 出身 | 特点 |
|------|------|
| 贵族 | 资源丰富、地位高 |
| 平民 | 普通、资源一般 |
| 商人 | 较富裕、社交广 |
| 学者 | 智力倾向 |
| 战士 | 力量倾向 |
| 猎人 | 敏捷倾向、野外强 |
| 贫民 | 资源少、韧性强 |
| 孤儿 | 自由度高、无人照管 |

出身影响起始资源、技能、社会关系、初始属性倾向。

| 阶段 | 年龄 | 行动力 | 特点 |
|------|------|--------|------|
| 👶 婴儿期 | 0-3岁 | 1 | 被动成长、天赋觉醒 |
| 🧒 童年期 | 4-12岁 | 2 | 学习探索 |
| 🧑 少年期 | 13-17岁 | 3 | 教育选择、叛逆 |
| 👨/👩 青年期 | 18-35岁 | 4 | 职业、冒险、巅峰 |
| 🧔 中年期 | 36-55岁 | 3 | 家庭、事业 |
| 👴 老年期 | 56+岁 | 2 | 回顾、终局 |
