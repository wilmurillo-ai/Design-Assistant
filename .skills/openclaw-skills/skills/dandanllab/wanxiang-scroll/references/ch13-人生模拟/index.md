# 第十三章：人生模拟

回合制异世界人生模拟引擎。玩家穿越到随机生成的异世界，从出生到死亡体验完整人生，支持代代相传、自由行动、后台静默成书。

## 模块联动

人生模拟是万象的「完整故事引擎」，与所有模块深度串联：

- **→ 文风系统（第二章）**：自动成书根据世界类型自动匹配文风（东方→古风、西幻→史诗等），可手动切换56种
- **→ 创作引擎（第五章）**：成书段落经文笔润色23种模式处理，确保自然
- **→ 交互式故事（第三章）**：两模式共享世界观+角色数据，`进入人生模拟`/`回到自由探索`无缝切换
- **→ 剧情管理（第四章）**：人生阶段→剧情大弧，连锁事件→支线标记，关系变化→角色图谱
- **→ 拆书融合（第十章）**：融合创作的世界观可直接作为人生模拟的世界模板
- **→ 辅助工具（第八章）**：角色设定生成/特质生成/外貌生成用于3角色选项
- **→ 质量控制（第七章）**：自动成书小说段落的质量检测
- **→ 李诞七步（第十二章）**：结局写作适用"结尾升华"步骤结构

## 游戏流程

```
新游戏 → 世界观生成 → 剧本命名 → 3角色选项 → 选择角色 → 第一回合 → ...
续缘 → 使用功能，每回合可自由选择做的事
```

## 数据结构

所有数据存储在 SQLite 数据库 (`data/wanxiang.db`) 和 JSON 文件中：

### 数据库表结构

```
核心故事系统:
├── stories          # 故事元信息
├── worlds           # 世界观配置
└── world_state      # 世界状态快照

角色系统:
├── characters           # 角色基础信息
├── character_attributes # 角色属性 (str/dex/int/cha/luk/san)
├── character_traits     # 角色特质
├── character_skills     # 角色技能
└── relationships        # 角色关系

事件系统:
├── events           # 事件记录
└── event_chains     # 事件链

剧情管理:
├── plot_arcs        # 剧情弧线
├── hidden_floors    # 隐藏楼层 (伏笔/备注)
└── summaries        # 阶段总结

小说生成:
├── novel_chapters   # 小说章节
└── novel_sections   # 章节段落

代代相传:
├── legacies         # 代际传承
└── family_tree      # 家族树

物品系统:
└── items            # 物品
```

### 文件结构

```
stories/[剧本名]/
├── novel.md           # 生成的小说
├── world.json         # 世界观配置
├── character.json     # 角色数据
└── events.json        # 事件记录
```
