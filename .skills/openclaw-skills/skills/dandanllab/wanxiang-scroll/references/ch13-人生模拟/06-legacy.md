# 代代相传

## 续缘模式

角色死亡后，可选择"续缘"——在同一个世界中创建新角色继续冒险。

---

## 前代NPC

前代角色死亡后变为NPC，保留：
- 姓名、种族、外貌
- 属性（按死亡时数值）
- 已获得特质
- 人际关系
- 物品（部分传承给后代）

前代NPC会根据其性格继续在世界中行动，可能成为：
- 导师：指导新角色
- 对手：与新角色冲突
- 盟友：协助新角色
- 传说：成为世界中的传奇人物

---

## 继承要素

### 可继承

| 要素 | 继承比例 |
|------|----------|
| 金币 | 30-50% |
| 物品 | 部分传家宝 |
| 技能 | 基础等级的20% |
| 人脉 | 部分关系继承 |
| 声望 | 50% |

### 不可继承

- 属性值
- 特质
- 个人任务
- 具体记忆

---

## 家族诅咒

某些负面特质可能以"家族诅咒"形式传承：

- 血脉诅咒：代代相传的负面特质
- 宿命纠缠：与前代未完成的事件相关
- 因果报应：前代行为的后果影响后代
- 灵魂羁绊：前代灵魂附体

---

## 多代传承

| 代数 | 特殊效果 |
|------|----------|
| 第2代 | 继承基础要素 |
| 第3代 | 家族特质觉醒 |
| 第5代 | 家族传说形成 |
| 第10代 | 家族成为世界势力 |

多代传承后，家族可能成为世界中的重要势力，影响世界走向。
