# 行动力与自由行动

## 行动力 (AP)

| 人生阶段 | 每回合AP |
|----------|----------|
| 婴儿期 | 1 |
| 童年期 | 2 |
| 少年期 | 3 |
| 青年期 | 4 |
| 中年期 | 3 |
| 老年期 | 2 |

---

## 行动类型

### 1. 基础行动（1AP）

- 休息：恢复少量HP/SAN
- 社交：与NPC互动
- 探索：发现新地点/事件
- 学习：提升技能/属性
- 工作：获得金币/资源

### 2. 进阶行动（2AP）

- 深入探索：发现隐藏区域/事件
- 训练：大幅提升技能
- 交易：买卖物品/装备
- 社交深化：建立/深化关系
- 调查：获取情报/线索

### 3. 特殊行动（3AP）

- 冒险：进入危险区域
- 创造：制作/建造物品
- 旅行：前往其他地区
- 修炼：大幅提升属性
- 求学：系统学习技能

---

## 行动规则

1. 每回合AP用完则回合结束
2. 未用完的AP不累积
3. 某些事件可额外消耗AP
4. 特殊状态可增减AP

---

## 自由行动

玩家可自由描述想做的事，系统根据描述消耗AP并生成结果。

**示例**：
- "我想去酒馆打听消息" → 社交行动（1AP）
- "我要在森林里寻找草药" → 探索行动（1AP）
- "我想挑战地下城" → 冒险行动（3AP）
- "我要向师父学习剑术" → 求学行动（3AP）
