# 自动成书

## 静默小说生成

游戏过程中，系统在后台自动将经历转化为小说章节。

---

## 章节结构

| 元素 | 说明 |
|------|------|
| 章节标题 | 根据核心事件生成 |
| 开头 | 场景描写，环境渲染 |
| 发展 | 事件经过，人物互动 |
| 高潮 | 关键冲突，选择时刻 |
| 结尾 | 结果，悬念钩子 |

---

## 风格匹配

根据世界类型自动匹配文风：

| 世界类型 | 匹配文风 |
|----------|----------|
| 东方玄幻 | 古风仙侠 |
| 西方奇幻 | 史诗奇幻 |
| 现代都市 | 都市现实 |
| 蒸汽朋克 | 机械美学 |
| 黑暗奇幻 | 暗黑哥特 |

可手动切换56种文风。

---

## 事件转化

| 游戏事件 | 小说表现 |
|----------|----------|
| 属性判定 | 动作描写 |
| 随机事件 | 意外转折 |
| 选择事件 | 内心挣扎 |
| 关系事件 | 情感互动 |
| 战斗事件 | 动作场景 |
| 世界事件 | 宏大叙事 |

---

## 质量控制

成书段落经文笔润色23种模式处理，确保自然。

关键检查点：
- 对话占比 ≥ 30%
- 禁用词检查
- 短句节奏
- 情感真实度
- 逻辑连贯性

---

## 查看小说

使用 `小说` 命令查看当前生成的小说内容。

小说存储在 `stories/[剧本名]/novel.md`。
