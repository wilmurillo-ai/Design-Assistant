# 世界观生成方法

## 生成流程

```
基础设定 → 魔法体系 → 种族设定 → 地理国家 → 历史背景 → 剧本命名
```

---

## 世界类型

| 类型 | 核心元素 | 世界名风格 |
|------|----------|-----------|
| 奇幻中世纪 | 剑与魔法、骑士、巨龙 | 艾尔德兰、瓦伦西亚 |
| 东方玄幻 | 修仙、灵气、宗门 | 九洲、玄天大陆 |
| 蒸汽朋克 | 机械、齿轮、蒸汽 | 齿轮城、蒸汽纪元 |
| 黑暗奇幻 | 克苏鲁、深渊、绝望 | 沉沦之地 |
| 异能都市 | 现代+超能力 | 都市隐秘 |
| 原始部落 | 远古、自然、图腾 | 荒野神灵 |
| 废土末世 | 文明崩塌、生存 | 废墟变异 |

### 隐藏世界（低概率引发）

| 设定 | 概率 | 描述 |
|------|------|------|
| 众神游戏 | 5% | 世界是神的游戏场 |
| 轮回世界 | 3% | 世界不断轮回 |
| 虚假世界 | 2% | 世界是幻象 |
| 混合世界 | 5% | 多个世界融合 |
| 倒计时世界 | 3% | 世界即将毁灭 |

## 魔法体系（1-3种共存）

| 类型 | 代表能力 |
|------|----------|
| 元素魔法 | 火球术、冰封、雷电 |
| 灵魂魔法 | 读心、附体、精神攻击 |
| 符文魔法 | 法阵、符文武器 |
| 自然魔法 | 治疗、变形、召唤 |
| 时空魔法 | 瞬移、时间停止 |
| 亡灵魔法 | 召唤亡灵、诅咒 |
| 星辰魔法 | 预言、星辰之力 |

### 魔法稀有度

| 稀有度 | 人口占比 | 能力上限 |
|--------|----------|----------|
| 无魔法者 | 60-80% | 无 |
| 微弱天赋 | 15-25% | 初级 |
| 普通法师 | 5-10% | 中级 |
| 强力法师 | 1-3% | 高级 |
| 大魔导师 | 0.1-0.5% | 禁咒级 |
| 传说级 | 0.01% | 神级 |

### 魔法来源

血脉 / 契约 / 学习 / 奇遇 / 神器

## 种族设定（随机3-6种）

| 种族 | 特点 | 寿命 |
|------|------|------|
| 人类 | 适应性强 | 80年 |
| 精灵 | 魔法亲和 | 800年 |
| 矮人 | 锻造大师 | 300年 |
| 兽人 | 强壮野性 | 60年 |

种族间设同盟、敌对、疏远、奴役关系。

## 历史背景（3-5个重大事件）

建国战争 / 魔法灾难 / 种族战争 / 神灵降临 / 英雄传说 / 王朝更替 / 瘟疫灾难 / 经济繁荣

生成当前纪元名称、年份、时代特点、核心矛盾。

## 用户偏好调整

| 用户说 | 调整方向 |
|--------|----------|
| "多点魔法" | 魔法稀有度降低 |
| "轻松一点" | 减少黑暗，增加喜剧 |
| "现实向" | 减少魔法，增加政治 |
| "冒险向" | 增加未知区域 |
| "战争向" | 增加国家冲突 |
