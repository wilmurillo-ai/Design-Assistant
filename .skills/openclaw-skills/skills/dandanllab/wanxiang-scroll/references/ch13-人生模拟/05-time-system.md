# 时间系统

## 回合制时间

每回合 = 1年（婴儿期/童年期）或 1季（少年期以后）

---

## 世界年份

起始年份由世界观生成决定，每回合推进对应时间。

---

## 季节系统

| 季节 | 特点 |
|------|------|
| 春 | 万物复苏，探索+10% |
| 夏 | 丰收季节，资源+10% |
| 秋 | 收获时节，交易+10% |
| 冬 | 严寒考验，生存难度+20% |

季节影响：事件类型、资源获取、行动效果、NPC行为。

---

## 死亡概率

每回合基础死亡概率极低，随年龄和状态变化：

| 阶段 | 基础死亡概率 |
|------|-------------|
| 婴儿期 | 2% |
| 童年期 | 0.5% |
| 少年期 | 0.3% |
| 青年期 | 0.2% |
| 中年期 | 0.5% |
| 老年期 | 3%+每年递增 |

死亡概率修正：
- HP低 → +2%
- SAN低 → +1%
- 负面特质 → +1%
- 正面特质 → -0.5%
- 幸运高 → -1%

---

## 快进

玩家可以说"快进X年"，系统自动生成期间事件摘要。

快进规则：
- 跳过期间只生成关键事件
- 属性/关系/资源自动演算
- 可能错过隐藏事件
- 快进后提供期间摘要
