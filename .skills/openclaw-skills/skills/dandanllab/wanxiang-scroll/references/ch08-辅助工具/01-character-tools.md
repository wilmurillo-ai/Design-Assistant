## 人设生成器

### 完整人设生成器**用途**：生成完整的角色设定**输入要素**：- 角色类型（主角/反派/配角）- 角色定位（身份/职业）- 核心特质（3-5个关键词）- 背景设定（可选）**输出格式**：
```
角色设定卡基本信息：- 姓名：[角色名]- 年龄：[年龄]- 性别：[性别]- 职业：[职业]外貌特征：- 身高：[身高]- 体型：[体型]- 发型：[发型]- 眼睛：[眼睛颜色和形状]- 特殊标记：[疤痕/纹身/胎记等]性格特质：- 核心性格：[3-5个关键词]- 优点：[2-3个优点]- 缺点：[2-3个缺点]- 怪癖：[1个让角色"掉价"的怪癖]背景故事：- 出身：[出身背景]- 成长经历：[关键成长事件]- 创伤：[心理创伤或阴影]- 目标：[短期和长期目标]能力设定：- 核心能力：[主要能力]- 技能列表：[技能1、技能2、技能3]- 特殊能力：[如有，详细描述]- 能力限制：[能力的限制和代价]人际关系：- 家人：[家人关系]- 朋友：[朋友关系]- 敌人：[敌人关系]- 爱慕对象：[如有]对话风格：- 语气：[语气特征]- 口头禅：[标志性台词]- 语言习惯：[说话特点]- 禁忌话题：[不想谈论的话题]行为模式：- 行为习惯：[日常习惯]- 决策方式：[如何做决策]- 应对压力：[压力下的反应]- 情绪表达：[如何表达情绪]成长弧：- 起点状态：[故事开始时的状态]- 转折点：[关键转折事件]- 终点状态：[故事结束时的状态]- 成长意义：[成长的主题]
```

### 反派人设生成器**用途**：生成有深度、有魅力的反派角色**核心原则**：- 反派不是纯粹的坏人，有自己的动机和理由- 反派要有魅力，让读者既恨又爱- 反派要有弱点，不是无敌的**输出要素**：
```
反派设定卡基本信息：- 姓名：[反派名]- 身份：[身份/职业]- 定位：[大反派/中级反派/小反派]核心动机：- 表面动机：[表面上想要什么]- 真实动机：[实际上想要什么]- 动机来源：[动机的根源]反派魅力：- 魅力点：[1-2个吸引人的特质]- 可理解性：[动机的合理性]- 悲剧性：[是否有悲剧色彩]反派实力：- 能力等级：[相对于主角的强度]- 智力水平：[智商/计谋]- 资源掌控：[拥有的资源]反派弱点：- 致命弱点：[最致命的弱点]- 心理弱点：[心理上的弱点]- 关系弱点：[人际关系上的弱点]反派人设关键词：[3-5个关键词，如：伪君子、疯批美人、冷血杀手、悲情英雄]
```

### 配角人设生成器**用途**：生成功能明确、性格鲜明的配角**核心原则**：- 配角要有独立的功能- 配角要有鲜明的性格- 配角要能反衬主角**输出要素**：
```
配角设定卡基本信息：- 姓名：[配角名]- 身份：[身份/职业]- 出场频率：[主要配角/次要配角/路人]角色功能：- 功能定位：[如：主角的导师、反派的助手、搞笑担当]- 对主角的作用：[对主角的影响]- 对剧情的作用：[对剧情的推动]性格特征：- 核心性格：[1-2个关键词]- 突出特点：[最突出的特点]- 反衬作用：[如何反衬主角]记忆点：- 外貌记忆点：[外貌上的特征]- 行为记忆点：[行为上的特征]- 语言记忆点：[语言上的特征]
```

## 角色关系图谱

### 关系类型定义
1. **正向关系**：朋友、恋人、家人、盟友
2. **负向关系**：敌人、竞争对手、仇人
3. **复杂关系**：亦敌亦友、爱恨交织、利益相关

### 关系强度等级
1. **强关系**：深层的情感连接，影响剧情走向
2. **中关系**：稳定的联系，影响角色发展
3. **弱关系**：表面的联系，偶尔影响剧情

### 关系图谱模板
```
角色关系图谱角色A（主角）├─ 强关系 → 角色B（恋人/朋友）├─ 中关系 → 角色C（盟友）├─ 弱关系 → 角色D（路人）├─ 负向关系 → 角色E（敌人）└─ 复杂关系 → 角色F（亦敌亦友）角色B├─ 强关系 → 角色A├─ 中关系 → 角色C└─ 负向关系 → 角色E[以此类推...]
```

## 使用示例

### 示例1：生成主角
```
可以说：角色类型：主角角色定位：现代都市，外卖员核心特质：坚韧、幽默、有正义感输出：主角设定卡基本信息：- 姓名：李阳- 年龄：24岁- 性别：男- 职业：外卖员性格特质：- 核心性格：坚韧、幽默、有正义感- 优点：从不放弃、乐观向上、乐于助人- 缺点：有时过于冲动、不善言辞- 怪癖：送餐时喜欢哼歌，但总是跑调背景故事：- 出身：普通家庭，父母是工人- 成长经历：高中毕业后开始打工，做过很多工作- 创伤：曾经因为没钱给家人治病，感到无能为力- 目标：短期目标是多赚钱，长期目标是创业能力设定：- 核心能力：超强的记忆力- 技能列表：熟悉城市道路、会做饭、修自行车- 特殊能力：无- 能力限制：记忆力超强，但只记有用的信息金手指：- 名称：万能外卖系统- 功能：接单时有特殊提示、能获得隐藏奖励- 限制：每天只能使用3次特殊功能
```

### 示例2：生成反派
```
可以说：角色类型：反派角色定位：企业高管核心特质：伪君子、冷血、有野心输出：反派设定卡基本信息：- 姓名：赵天- 年龄：35岁- 性别：男- 身份：某企业CEO核心动机：- 表面动机：推动企业发展- 真实动机：掌控一切，获得绝对权力- 动机来源：童年的不安全感反派魅力：- 魅力点：外表儒雅，谈吐得体- 可理解性：童年的创伤让他渴望掌控- 悲剧性：为了权力失去了亲人反派实力：- 能力等级：极高（财力、人脉、智谋）- 智力水平：极高- 资源掌控：掌控企业命脉反派弱点：- 致命弱点：不信任任何人- 心理弱点：害怕失控- 关系弱点：没有真正的朋友
```

## 注意事项- 角色设定要服务于剧情- 角色要有成长弧- 反派要有魅力，不能脸谱化- 配角要有独立的功能- 所有角色都要有弱点
