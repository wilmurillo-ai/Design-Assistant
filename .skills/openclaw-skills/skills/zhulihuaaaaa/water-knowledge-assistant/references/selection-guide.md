# 选型指南

## 液位计选型规则

### 1. 介质特性
- **腐蚀性介质** → 雷达液位计、聚四氟乙烯材质传感器
- **高温介质** → 高温型雷达液位计、热电偶液位计
- **易挥发介质** → 雷达液位计、隔爆型液位计
- **粘稠介质** → 音叉液位计、射频导纳液位计
- **含气泡介质** → 雷达液位计、压力式液位计

### 2. 环境条件
- **户外环境** → IP68防护等级设备
- **防爆环境** → 隔爆型、本质安全型设备
- **振动环境** → 抗震型设备
- **粉尘环境** → 防尘型设备

### 3. 测量要求
- **高精度要求** → 雷达液位计（±3mm）、激光液位计
- **大量程要求** → 雷达液位计（可达70m）
- **小量程要求** → 超声波液位计、投入式液位计
- **连续测量** → 雷达液位计、超声波液位计、投入式液位计
- **点式测量** → 浮球液位计、音叉液位计

### 4. 安装条件
- **顶部安装** → 雷达液位计、超声波液位计
- **侧面安装** → 磁翻板液位计、侧装式浮球液位计
- **底部安装** → 投入式液位计、压力式液位计
- **狭小空间** → 小型雷达液位计、微型液位计

## 水尺选型规则

### 1. 使用环境
- **淡水环境** → 搪瓷水尺、不锈钢水尺
- **海水环境** → 316L不锈钢水尺、玻璃钢水尺
- **酸碱环境** → 玻璃钢水尺、聚四氟乙烯水尺
- **高温环境** → 耐高温不锈钢水尺

### 2. 安装方式
- **直立安装** → 搪瓷水尺、不锈钢水尺
- **倾斜安装** → 倾斜式水尺
- **悬挂安装** → 悬挂式水尺
- **自动化监测** → 电子水尺

### 3. 精度要求
- **高精度要求** → 电子水尺（±1mm）
- **一般精度要求** → 搪瓷水尺、不锈钢水尺（±1cm）

## 流量计选型规则

### 1. 介质特性
- **导电液体** → 电磁流量计
- **腐蚀性液体** → 衬四氟电磁流量计、涡街流量计
- **粘性液体** → 容积式流量计、质量流量计
- **含颗粒液体** → 电磁流量计、超声波流量计
- **气体** → 涡街流量计、热式气体质量流量计

### 2. 测量要求
- **高精度要求** → 质量流量计、电磁流量计
- **大流量要求** → 涡街流量计、电磁流量计
- **小流量要求** → 涡轮流量计、椭圆齿轮流量计
- **低压力损失** → 电磁流量计、超声波流量计

### 3. 安装条件
- **满管要求** → 电磁流量计、涡轮流量计
- **直管段要求** → 涡街流量计、电磁流量计
- **脏污介质** → 电磁流量计（带自清洗功能）
