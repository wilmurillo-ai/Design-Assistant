# Popular TLDs Reference

| TLD | Country/Origin | Notes |
|-----|---------------|-------|
| `.com` | Commercial (global) | Most recognized; first choice for brands |
| `.net` | Network (global) | Strong alternative when .com is taken |
| `.org` | Organization (global) | Best for nonprofits, open-source, communities |
| `.io` | British Indian Ocean Territory | Extremely popular for tech startups and dev tools |
| `.ai` | Anguilla | The go-to TLD for AI/ML products and startups |
| `.co` | Colombia | Short, catchy; used by many startups as .com alternative |
| `.app` | Google Registry | Modern, great for mobile/web apps; requires HTTPS |
| `.dev` | Google Registry | Developer-focused; requires HTTPS |
| `.so` | Somalia | Short and trendy; used for productivity/social tools |
| `.xyz` | General | Cheap, popular with Gen Z brands and creative projects |
| `.tech` | Radix | Good for technology companies |
| `.me` | Montenegro | Personal brands, portfolios, identity-focused apps |
| `.gg` | Guernsey | Gaming community favorite |
| `.sh` | Saint Helena | Popular for shell scripts and developer CLI tools |
| `.run` | General | Great for automation, CI/CD, and action-oriented products |
| `.tools` | General | Descriptive; good for utility software |
| `.cloud` | General | Cloud services and SaaS products |
| `.store` | General | E-commerce and product-focused businesses |
| `.design` | General | Creative agencies and design tools |
| `.studio` | General | Creative studios, game studios, media |

## Notes for AI/tech startups

- `.ai` is currently the most premium TLD in the AI space — expect higher registration costs (~$70–100/yr)
- `.io` remains highly valued in the dev tooling space despite the political controversy around BIOT
- `.app` and `.dev` are Google-managed, HSTS-preloaded (HTTPS required), and increasingly trusted
- `.co` is short, memorable, and used by notable companies (AngelList uses angel.co)
