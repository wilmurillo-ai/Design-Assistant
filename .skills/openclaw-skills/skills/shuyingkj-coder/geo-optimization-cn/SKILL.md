description: 企业营销内容智能生成器。功能：1) 根据公司名称和行业生成搜索词前缀/后缀各5个；2) 生成10-15个行业问答词；3) 将Word资料转换为GEO优化的纯Q&A知识库（无分类、无关键词）；4) 从Word文档提取企业6大关键信息（纯文本、无表格）。

# 企业营销内容智能生成器

## 功能概览

| 功能 | 命令 | 说明 |
|------|------|------|
| 🔍 搜索词生成 | `python business_content_generator.py search <公司名> [行业]` | 生成前缀5个 + 后缀5个 + 组合示例 |
| 💬 问答词生成 | `python business_content_generator.py qa <公司名> [行业]` | 生成10-15个行业问答词 |
| 📚 QA知识库生成 | `python business_content_generator.py kb <word文件路径>` | Word → 纯Q&A知识库（无分类、无关键词） |
| 📊 企业信息提取 | `python business_content_generator.py extract <word文件路径>` | 提取6大信息（纯文本、无表格） |

---

## 安装依赖

```bash
# 安装 markitdown（用于读取Word）
uv tool install markitdown

# 安装 python-docx（用于生成Word）
pip install python-docx
```

## 功能3：QA知识库生成

### 使用方式
```bash
# 生成Markdown
python business_content_generator.py kb ./公司资料.docx

# 生成Word
python business_content_generator.py kb ./公司资料.docx --output ./output/QA知识库.docx
```

### 输出格式（纯Q&A，无分类、无关键词）
```markdown

粤腊煌（全称：江门市粤腊煌食品有限公司）是位于广东江门的专业广式腊肠生产企业，拥有30余年生产经验...

---

### 粤腊煌的主要产品有哪些？

粤腊煌核心产品线包括四大类广式腊味：①美味香肠；②切肉腊肠；③金牌纯肉腊肠；④广式腊肠...

---

...（继续其他QA）
```

**特点：**
- ❌ 无分类标题（如"## 分类一：企业概况"）
- ❌ 无问题编号（如"Q1:"、"Q2:"）
- ❌ 无关键词标注
- ❌ 无知识库名称、版本、日期、工具信息
- ✅ 纯问题 + 答案 + 分隔线

## 完整使用示例

```bash
# 1. 进入Skill目录
cd ~/.claude/skills/business-content-generator

# 2. 生成搜索词和问答词
python business_content_generator.py search "粤腊煌食品有限公司" "餐饮"
python business_content_generator.py qa "粤腊煌食品有限公司" "餐饮"

# 3. 准备Word资料（如：公司介绍.docx）

# 4. 生成QA知识库（Word格式）
python business_content_generator.py kb ./公司介绍.docx --output ./output/QA知识库.docx

# 5. 提取企业信息（Word格式）
python business_content_generator.py extract ./公司介绍.docx --output ./output/企业信息.docx
```

## 输出文件说明

### QA知识库文档
- **用途**：网站FAQ、智能客服训练、SEO文章
- **格式**：纯Q&A，每个QA之间用分隔线
- **字数**：每个答案100-200字

### 企业信息提取文档
- **用途**：企业简介、招商手册、宣传材料
- **格式**：7大章节 + 附录，纯文本无表格
- **字段**：公司名称、业务范围、产品特点、信任背书、客户案例、品牌故事、用户痛点

---

## 技术说明

- **文档读取**：`markitdown` 库（uvx运行，无需安装）
- **Word生成**：`python-docx` 库
- **AI分析**：Claude 进行内容理解和重组
- **GEO优化**：遵循 Answer-First 结构

---

## 注意事项

1. Word文档建议包含清晰的章节结构（标题层级）
2. 图片内容会被提取为描述文字
3. 表格内容会转换为文本描述
4. 生成的内容需要人工审核确保准确性
5. 所有输出**不包含**版本信息、生成日期、制作工具等元数据
