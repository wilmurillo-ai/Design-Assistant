# Gmail Briefings

## Workflow
1. Search inbox: unread, today, senders (clients/boss).
2. Summarize: Subject/sender/snippet/priority.
3. Draft replies for top 3.
4. read references/gog-sop.md for auth/config.

## Commands
- List: `gog g inbox unread`
- Read: `gog g read ID`
- Reply draft: Use templates in assets/.

refs/gmail-filters.md: Custom searches.
