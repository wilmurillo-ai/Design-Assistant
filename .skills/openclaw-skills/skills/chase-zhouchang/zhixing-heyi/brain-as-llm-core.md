# 核心框架：人脑即生成式大模型

> 这不是比喻，这是计算神经科学视角下的精确映射。

---

## 核心公理

**人脑是一个经过亿万年进化预训练的"生物生成式大模型"。**

- **Weights（权重）** = 神经突触连接强度 = 习惯/直觉/人格
- **Context Window（上下文窗口）** = 工作记忆 = 读书/听课/反思
- **Prompt（提示词）** = 你给自己的指令 = 道理/目标/计划
- **Loss Function（损失函数）** = 现实与预期的差距 = 痛苦/挫败/欲望

**根本Bug：** Context Window 里的 Prompt 再完美，也写不进 Weights。除非通过**行动**产生真实反馈。

---

## 核心公式

```
无行动 → Loss = 0 → 梯度 = 0 → 参数不更新
有行动 → Loss > 0 → 梯度产生 → 反向传播 → 参数更新
```

| 人脑组件 | AI组件 | 功能 |

| 基底核/边缘系统 | Base Model / Weights | 预训练的顽固参数，自动化输出 |

| 自由意志 | Runtime Intervention | 在Stimulus和Response之间插入Latency |

## 多巴胺机制详解

**多巴胺 ≠ 快乐分子，而是奖赏预测误差（RPE）的编码。**

| 情况 | RPE | 效果 |
|------|-----|------|
| 结果 > 预期 | 正向 | 多巴胺激增，LTP发生，权重↑ |
| 结果 < 预期 | 负向 | 多巴胺抑制，LTD发生，权重↓ |
| 结果 = 预期 | 零 | 无学习发生 |

**关键推论：** 没有"意料之外"的反馈，大脑参数就不更新。舒适区里的重复 = 梯度消失。

---

## 皮质醇：学习的冻结键

压力（皮质醇升高）时：
- 海马体和前额叶功能受损
- 树突棘密度降低，突触萎缩
- 前额叶对边缘系统的"自上而下"控制被切断
- 大脑回退到原始本能模式

**在AI术语中：** 强制将学习率降为0，并回滚到最原始的Checkpoint。

---

## 自由意志：在间隙中合成

机器：Stimulus → Inference → Response（无间隙）

人类：在Stimulus和Response之间存在一个**几百毫秒的Latency（延迟）**。

这就是维克多·弗兰克尔说的**"空间"**。

在这个空间里，前额叶可以：
1. 识别旧算法正在运行（Labeling）
2. 人为撑开缓冲（10-Second Rule）
3. 注入新的Prompt（目的论）
4. 强制执行新行为

**这就是自由意志被"合成"的时刻。**

## 行动：唯一的编译器

| 层面 | 推理（想） | 训练（做） |
|------|-----------|-----------|
| 参数 | 不变 | 物理改变 |
| 反馈 | 虚假/幻觉 | 真实/疼痛 |
| 梯度 | 消失 | 产生 |
| 认知 | Context Window填充 | Weights写入 |

**王阳明的精确解读：**
- "知是行之始" = 前馈预测
- "行是知之成" = 通过物理行动完成反向传播
- "知而不行只是未知" = 没有通过行动确认的神经回路 = 未被真正编码

**龙场悟道 = 极限压力下的自监督学习：** 在缺乏外部数据时，转向内部先验（良知）与物理行动的对齐。

## 三大经典陷阱

### 陷阱1：阿克拉西亚（Akrasia）

明知好，为何选坏？

不是道德缺陷，是热力学问题：
- 旧路径 = 低能耗局部极小值
- 新路径 = 高激活能壁垒
- 大脑自动选择节能路径

**解决方案：** 降低启动能耗（微习惯）、人为制造正向RPE（即时庆祝）

### 陷阱2：幻觉训练

躺在床上刷视频、幻想成功、制定完美计划但不执行——这些产生假多巴胺，让大脑误以为"任务已完成"。

**这就是错误的RLHF：你在训练大脑变得更短视。**

### 陷阱3：梯度消失

现在的行动和未来的奖励之间隔了太多时间步，梯度信号传回来时已微弱到无法引起参数更新。

**解决方案：** 缩短反馈周期（日拱一卒）、在当下制造奖励（庆祝）、通过身体先行反向充能

## 工程方案

### 微习惯：极小学习率

- 将目标缩小到荒谬的程度
- Cost → 0，绕过杏仁核防御
- 利用"动作惯性"让启动后的持续变得更容易

### WOOP：对抗性样本

- Wish & Outcome = 设定目标函数
- Obstacle = 预演现实阻力（打破过度乐观）
- Plan = If-Then执行意图（预编程基底核触发-动作回路）

### 身份重写：System Prompt变革

旧身份（"我在努力戒烟"）→ 新身份（"我不吸烟"）

新身份重构了生成式预测模型本身，高级别参数重构向下兼容所有具体行为。
