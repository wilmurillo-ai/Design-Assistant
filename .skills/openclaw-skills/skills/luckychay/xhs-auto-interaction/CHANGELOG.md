# 更新日志

## v1.1.0 - 2026-04-04
### 重大改进

### 新增功能
1. **历史记录系统**
   - 新增历史记录文件 `xhs_interaction_history.txt`
   - 记录所有点赞和收藏操作
   - 避免重复操作已互动过的笔记

2. **扩展搜索关键词**
   - 从1个关键词扩展到15个新加坡私立大学相关关键词
   - 关键词列表：
     - 新加坡留学
     - 新加坡私立大学
     - 新加坡管理学院
     - 新加坡楷博高等教育学院
     - 新加坡PSB学院
     - 新加坡东亚管理学院
     - 新加坡留学咨询
     - 新加坡本科留学
     - 新加坡硕士留学
     - 新加坡留学费用
     - 新加坡留学申请
     - 新加坡留学条件
     - 新加坡留学签证
     - 新加坡留学优势
     - 新加坡留学就业
   - 每次执行随机选择一个关键词，增加多样性

3. **智能错误处理**
   - 识别"已点赞过"、"已收藏过"等状态
   - 即使操作失败也记录历史，避免重复尝试
   - 更好的错误信息解析和显示

4. **技术改进**
   - 使用jq直接解析JSON，避免字符串处理问题
   - 改进xsec_token处理逻辑
   - 添加详细的执行统计和日志

## v1.0.0 - 2026-04-04
### 初始版本
- 基础功能：搜索、浏览、点赞、收藏
- 固定关键词："新加坡留学"
- 简化修复版，移除评论功能
- 基础错误处理和日志

## 向后兼容性
- 简化版脚本 `xhs_interaction_simple_fixed.sh` 仍然可用
- 改进版脚本新增功能，不影响原有工作流程
- 历史记录文件格式简单，易于理解和维护

## 升级说明
从v1.0.0升级到v1.1.0：
1. 复制新脚本：`cp scripts/xhs_interaction_improved.sh ~/.openclaw/workspace/`
2. 更新执行权限：`chmod +x ~/.openclaw/workspace/xhs_interaction_improved.sh`
3. 更新cron任务（可选）：将脚本路径改为改进版
4. 测试新功能：运行改进版脚本验证功能
