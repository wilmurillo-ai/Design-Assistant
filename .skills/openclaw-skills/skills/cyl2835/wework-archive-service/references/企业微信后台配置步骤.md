# 企业微信后台配置步骤

### ⚠️ 配置前强制检查项
1. ✅ 已准备好企业已备案的官方域名，所有回调URL将使用该域名
2. ✅ 若服务器无固定公网IP，已完成Cloudflare Tunnel配置，将8400端口映射到企业域名
3. ✅ 已获取服务器出口IP地址，准备配置到企业微信白名单
4. ✅ 已安装pycryptodome依赖：`pip3 install pycryptodome`

---
### 🔑 RSA密钥对生成（必须完成）
会话内容存档功能需要RSA密钥对进行数据加密传输，请运行以下脚本生成：
```python
python3 -c "
from Crypto.PublicKey import RSA
# 生成2048位RSA密钥对
key = RSA.generate(2048)
# 导出公钥并解码为UTF-8字符串
public_key = key.publickey().export_key().decode('utf-8')
# 导出私钥并解码为UTF-8字符串（完整输出）
private_key = key.export_key().decode('utf-8')

# 打印公钥（配置到企业微信）
print('公钥（配置到企业微信）:')
print(public_key)
# 打印完整私钥
print('\n私钥（保存好，不要泄露）:')
print(private_key)
"
```
#### 密钥配置说明：
1. 运行脚本后，将输出的**公钥**完整复制，粘贴到企业微信管理后台「会话内容存档」的公钥配置处
2. 将输出的**私钥**完整保存到技能的`config/private_key.pem`文件中，注意不要泄露给第三方
3. 企业微信会使用你上传的公钥加密会话内容，服务使用私钥解密，确保数据安全

## 1. 准备工作

### 1.1 企业微信账号
- 注册企业微信账号（https://work.weixin.qq.com）
- 完成企业认证（可选，但推荐）

### 1.2 获取企业信息
1. 登录企业微信管理后台
2. 进入"我的企业" > "企业信息"
3. 记录以下信息：
   - **企业ID (CorpID)**: 用于API调用
   - **企业名称**: 用于识别

## 2. 创建自建应用

### 2.1 创建应用
1. 进入"应用管理" > "自建"
2. 点击"创建应用"
3. 填写应用信息：
   - 应用名称: `企业微信存档服务`
   - 应用Logo: 上传合适的图标
   - 应用介绍: 描述应用功能
4. 点击"创建应用"

### 2.2 配置应用权限
1. 进入应用详情页
2. 配置"可见范围":
   - 选择需要使用该应用的成员
   - 或选择"全部成员"
3. 配置"应用权限":
   - 通讯录权限: 读取成员信息
   - 应用权限: 发送消息

## 3. 配置普通回调

### 3.1 获取回调配置信息
1. 在应用详情页，点击"接收消息"
2. 点击"设置API接收"
3. 系统会生成：
   - **Token**: 随机字符串，用于签名验证
   - **EncodingAESKey**: 43位随机字符串，用于消息加解密
4. 记录这些信息，填写到服务配置文件中

### 3.2 配置回调URL
1. 在"接收消息"设置页面：
   - URL: `https://你的域名/callback`
   - Token: 填写上一步获取的Token
   - EncodingAESKey: 填写上一步获取的EncodingAESKey
2. 选择"消息加密方式":
   - 推荐选择"加密方式"
3. 点击"保存"

### 3.3 验证回调配置
1. 点击"保存"后，企业微信会发送验证请求
2. 确保服务已启动并可以访问
3. 验证成功后显示"保存成功"

## 4. 配置会话内容存档

### 4.1 开启会话存档
1. 进入"管理工具" > "会话内容存档"
2. 点击"开启"
3. 阅读并同意《会话内容存档功能协议》
4. 点击"同意并开启"

### 4.2 配置存档范围
1. 选择需要存档的成员范围：
   - 可以按部门选择
   - 可以按成员选择
2. 点击"确认"

### 4.3 获取存档Secret
1. 开启后，系统会生成：
   - **Secret**: 用于存档回调验证
2. 记录这个Secret，填写到服务配置文件中

### 4.4 配置存档回调
1. 在会话存档设置页面：
   - 回调地址: `https://你的域名/archive/callback`
   - Token: 填写上一步获取的Secret
2. 点击"保存"

## 5. 配置IP白名单

### 5.1 获取服务器IP
1. 获取运行服务的服务器公网IP
2. 如果是动态IP，考虑使用域名

### 5.2 配置白名单
1. 进入"我的企业" > "安全与保密"
2. 点击"可信IP"
3. 添加服务器IP地址
4. 点击"保存"

**注意**: 企业微信只会向白名单中的IP发送回调

## 6. 测试配置

### 6.1 测试普通回调
1. 在应用详情页，点击"接收消息"
2. 点击"测试回调URL"
3. 选择测试事件类型
4. 点击"发送测试"
5. 检查服务日志，确认收到测试消息

### 6.2 测试消息发送
1. 在应用详情页，点击"发消息"
2. 选择接收成员
3. 发送测试消息
4. 检查是否成功接收

### 6.3 测试会话存档
1. 在已开启存档的成员间发送消息
2. 等待几分钟（企业微信异步推送）
3. 检查服务日志，确认收到存档数据

## 7. 常见问题解决

### 7.1 回调验证失败
**可能原因：**
1. Token或EncodingAESKey不匹配
2. 服务器时间不同步
3. 网络连接问题

**解决方案：**
1. 检查配置文件中的Token和EncodingAESKey
2. 同步服务器时间
3. 检查防火墙和网络配置

### 7.2 收不到回调
**可能原因：**
1. IP不在白名单中
2. 服务未启动
3. 域名解析问题

**解决方案：**
1. 检查IP白名单配置
2. 检查服务运行状态
3. 检查域名解析和HTTPS证书

### 7.3 消息解密失败
**可能原因：**
1. EncodingAESKey错误
2. 加密算法不匹配
3. 消息格式错误

**解决方案：**
1. 检查EncodingAESKey配置
2. 检查加解密代码实现
3. 查看原始消息日志
