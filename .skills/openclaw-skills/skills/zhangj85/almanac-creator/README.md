# 📅 黄历制作技能 (Almanac Creator)

> 按照 almanac-image-standard.md V2 标准生成每日黄历图片

---

## 🎯 功能特性

- ✅ **标准化制作**: 严格按照 V2 标准（1080x1400 像素）
- ✅ **三页输出**: 封面 + 宜忌 + 生肖、财神 + 养生、科普 + 故事
- ✅ **双层边框**: 外红内金，传统中国风
- ✅ **自动居中**: 智能文字居中排版
- ✅ **多平台适配**: 今日头条、抖音、小红书

---

## 📦 安装使用

### 前置要求

- Python 3.8+
- PIL (Pillow) 库

```bash
pip install Pillow
```

### 快速开始

```bash
# 生成今日黄历（3 页）
python scripts/generate_almanac.py --date 2026-04-08

# 生成指定页数
python scripts/generate_almanac.py --date 2026-04-08 --pages 1

# 指定输出目录
python scripts/generate_almanac.py --date 2026-04-08 --output ./output
```

### 参数说明

| 参数 | 说明 | 默认值 | 选项 |
|------|------|--------|------|
| `--date` | 日期（YYYY-MM-DD） | 今天 | 任意有效日期 |
| `--pages` | 生成页数 | 3 | 1/2/3 |
| `--output` | 输出目录 | reports/ | 任意目录路径 |

## 🎨 输出示例

### 第 1 页：封面 + 宜忌 + 生肖
- 每日黄历标题
- 公历 + 农历日期
- 节气信息
- 今日宜忌
- 红榜/黑榜生肖

### 第 2 页：财神 + 养生
- 财神方位（喜神/福神/财神/胎神）
- 季节养生（饮食/运动/作息）

### 第 3 页：科普 + 故事
- 黄历科普（天干地支/冲煞/吉神凶神）
- 传统故事（与节气或日期相关）

## 📊 技术标准

| 参数 | 值 |
|------|-----|
| **宽度** | 1080 像素 |
| **高度** | 1400 像素 |
| **格式** | PNG |
| **文件大小** | 150-250 KB/页 |
| **字体** | Windows 黑体 (simhei.ttf) |
| **颜色** | 中国红 (#8B0000)、金色 (#D4AF37) |
