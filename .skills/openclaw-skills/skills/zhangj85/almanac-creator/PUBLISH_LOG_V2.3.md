## 🚀 重大升级

### 1. 字体全面加大 ⭐⭐⭐⭐⭐

**调整详情**:

| 用途 | V2.2.0 | V2.3.0 | 变化 | 提升 |
|------|--------|--------|------|------|

| **故事/养生正文** | 40px | **44px** | +4px | +10% |

**影响页面**:
- ✅ 第 2 页（养生页）：财神、吉时、养生建议字体加大

## 🎯 代码修改

### 修改文件

```
skills/almanac-creator/scripts/generate_almanac.py
```

### 修改内容

```python
# V2.2.0
FONT_SIZES = {
    'title': 90,
    'subtitle': 65,
    'section': 55,        # 栏目题
    'content': 45,        # 正文
    'small': 38           # 小字
}

# V2.3.0
FONT_SIZES = {
    'title': 90,
    'subtitle': 65,
    'section': 60,        # 55 → 60 (+5px)
    'content': 48,        # 45 → 48 (+3px)
    'content_large': 44,  # 新增：故事/养生正文
    'small': 40           # 38 → 40 (+2px)
}

# 间距调整
SPACING = {
    'after_section': 28,   # 25 → 28 (+3px)
    'after_content': 20,   # 18 → 20 (+2px)
    'separator_after': 28, # 25 → 28 (+3px)
}
```
