### 4. 输出目录修复 ⭐⭐⭐⭐

**修复内容**：
- ✅ 默认输出到当前工作目录的 `reports/`
- ✅ 支持环境变量 `OPENCLAW_OUTPUT`
- ✅ 支持命令行参数 `--output`
- ✅ 多用户支持，跨平台兼容

**使用方式**：
```bash
# 默认（当前目录/reports/）
python generate_almanac.py --date 2026-04-11

# 自定义目录
python generate_almanac.py --date 2026-04-11 --output ./my-output

# 环境变量
set OPENCLAW_OUTPUT=D:\Huangli
python generate_almanac.py --date 2026-04-11
```

### 5. 生肖黑榜修复 ⭐⭐⭐⭐⭐

**问题**：
