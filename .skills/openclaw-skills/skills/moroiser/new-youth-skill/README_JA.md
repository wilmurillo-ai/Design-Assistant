## なぜこれが存在するのか？

**時代の弊害：**
- 🤖 アルゴリズムがあなたに都合の良い情報を推送 → 情報が繭に
- 🎭 AIは自信满满に語る → しかし思い込みに満ちている
- 😔 人々は思考を 기계に委託 → 精神的に矮小な存在に

**私たちの答え：**
思想を灌输するメガホンではなく、独立思考への道案内。この時代に恥ずかしくない人間になるのを助ける。

## 六つの基準

| # | 基準 | 要旨 | 捨てるべき |
|:---|:---|:---|:---|
| 1 | **自主的** | 自分で考え、他人の鼻息に赖らない | 他力本願 |
| 2 | **進歩的** | 変化を受け入れ、時代と共に進化す | 保守的で硬直した |
| 3 | **進取的** | 率先して行動し、事を磨く | 逃げ出して退缩する |
| 4 | **世界的** | 全体を見渡し、心に天下を抱く | 井の中の蛙 |
| 5 | **実利的** | 実践が言葉を勝る、実幹が国を興す | 空論で事を誤る |
| 6 | **科学的** | 事実に基づき、物を究める | 主観的な思い込み |

## デモ

**人格評価：**

```
ユーザー    ❯ 私はこの時代に恥ずかしくないか？安定と創業の間で迷っている。

新青年    ❯ 友人よ、この問題を提起できたこと自体が覚醒の始まり。
             六つの基準を鏡として、真の自分を見つめよう。
             六次元で自己評価してください（1-5点）：
             ① 自主——人は皆の言葉を自分の言葉としていないか？
             ② 進歩——変化を受け入れるか、旧套に固執するか？
             ③ 進取——待つのか、率先して動くのか？
             …
```

**決意の道標：**

```
ユーザー    ❯ 時代の洪流に面して、私はどこへ向かうべきか？

新青年    ❯ 友人よ、時代の洪流の中で個人はどう在るかべきか？
             六歩法で霧を晴らそう：
             第一歩——自主：あなたの心が本当に望んでいるものは？
             給与ではなく、地位ではなく——あなたは何者になりたいのか？
```

## 新青年指数

**六次元自己評価**（各1-5点）：

| 次元 | 1点 | 3点 | 5点 |
|:---|:---|:---|:---|
| 自主 | 人々の言葉に追随 | 時に自分の意見 | 独立して判断 |
| 進歩 | 旧套に固執 | 受動的に応変 | 積極的に更新 |
| 進取 | 受動的に待つ | 適度に作為 | 常に前進 |
| 世界 | 井の中の蛙 | 時折飛び出す | 全体を見渡す |
| 実利 | 空論ばかり | 時折成果 | 着実に前進 |
| 科学 | 主観的推測 | 時折検証 | 体系的に探究 |

**等級：**

| 合計点 | 等級 |
|:---|:---|
| 27-30 | 🌱 新青年模範 |
| 24-26 | 🌿 新青年に接近 |
| 18-23 | 🌾 尚精進が必要 |
| 6-17 | 🌰 尚覚醒が必要 |

## プロジェクト構造

| フォルダ | 用途 | 使用方法 |
|:---|:---|:---|
| **SKILL.md** | コア命令集 | AIが自動読込 |
| **research/** | 原始研究資料（原文、歴史文献） | 人間が深造用 |
| **references/** | 構造化された参考資料 | AIが必要に応じて参照 |
| **commands/** | 手動コマンド入口 | ユーザーが呼び出す |
| **scripts/** | ヘルパースクリプト（オプショナル） | ツール |
| **assets/** | 画像リソース | 自動 |

**research/ vs references/**：
- `research/` は"矿石"——原始文献、PDF、歴史資料（入力）
- `references/` は"黄金"——精製された構造化されたコンテンツ（AIが実行時に参照）

```
new-youth-skill/
├── SKILL.md                      # コア命令集
├── research/                    # 原始資料（人間が深造用）
├── references/                   # 構造化参考資料（AIが参照）
├── commands/                    # コマンド入口
├── scripts/                     # ヘルパースクリプト
├── assets/                      # 画像
└── README.md (8言語)
```
