## Зачем это нужно?

**Недуг эпохи:**
- 🤖 Алгоритмы угождают тебе → информационные пузыри
- 🎭 ИИ говорит с уверенностью → но полон галлюцинаций
- 😔 Люди отдают своё мышление машинам → духовные карлики

**Наш ответ:**
Не рупор для промывки мозгов, а проводник к самостоятельному мышлению. Помочь тебе стать достойным этой эпохи.

## Демо

**Оценка личности:**

```
Пользователь    ❯ Достойны ли мы этой эпохи? Я колеблюсь между стабильностью и предпринимательством.

Новая молодёжь  ❯ Друг, что ты задаёшь этот вопрос — уже начало пробуждения.
                Используем шесть стандартов как зеркало, чтобы увидеть истинное Я.
                Пожалуйста, оцени себя по шести измерениям (1-5):
                ① Автономность — Ты часто принимаешь чужие мнения за свои?
                ② Прогресс — Ты принимаешь изменения или цепляешься за старое?
                ③ Предприимчивость — Ты ждёшь или действуешь?
                …
```

**Направление решений:**

```
Пользователь    ❯ Перед лицом течений эпохи, куда мне идти?

Новая молодёжь  ❯ Друг, что может сделать индивид в потоках времени?
                Позволь шести шагам развеять туман:
                Первый шаг — Автономность: Чего ты действительно жаждешь?
                Не зарплата, не статус — кем ты хочешь стать?
```

## Индекс Новой молодёжи

**Шестимерная самооценка** (1-5 каждое):

| Измерение | 1 | 3 | 5 |
|:---|:---|:---|:---|
| Автономность | Следую за толпой | Иногда независим | Независимый мыслитель |
| Прогресс | Консервативен | Пассивно адаптируюсь | Проактивно обновляюсь |
| Предприимчивость | Пассивно жду | Умеренно действую | Всегда вперёд |
| Глобальность | Провинциален | Иногда прорываюсь | Панорамное видение |
| Прагматичность | Всё болтовня | Иногда результаты | Стабильный прогресс |
| Научность | Субъективные домыслы | Иногда проверяю | Систематическое исследование |

**Уровни:**

## Структура проекта

| Папка | Назначение | Как используется |
|:---|:---|:---|
| **SKILL.md** | Основные инструкции | ИИ загружает автоматически |
| **research/** | Исходные материалы (оригинальные тексты, исторические документы) | Для изучения человеком |
| **references/** | Структурированные материалы | ИИ ссылается по необходимости |
| **commands/** | Командный вход | Пользователь вызывает |
| **scripts/** | Вспомогательные скрипты | Инструменты |
| **assets/** | Изображения | Автоматически |

**research/ vs references/**:
- `research/` — это "руда" — оригинальные тексты, PDF, исторические источники (вход)
- `references/` — это "золото" — очищенный структурированный контент (ИИ ссылается во время выполнения)

```
new-youth-skill/
├── SKILL.md                      # Основные инструкции
├── research/                    # Исходные материалы
├── references/                   # Структурированные ссылки
├── commands/                    # Командный вход
├── scripts/                     # Вспомогательные скрипты
├── assets/                      # Изображения
└── README.md (8 языков)
```
