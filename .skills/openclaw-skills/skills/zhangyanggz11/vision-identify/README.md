# 📷 拍照识万物

**全能视觉识别助手** — 不仅识别物品，更分析场景、推断意图、给出精准选购方案。

[![Version](https://img.shields.io/badge/version-2.0.0-blue.svg)](https://github.com/wtmvw/vision-identify)
[![License](https://img.shields.io/badge/license-MIT--0-green.svg)](https://opensource.org/licenses/MIT-0)

---

## ✨ 功能特点

| 特点 | 说明 |
|------|------|
| 📸 **万物识别** | 动植物、商品、五金配件、电子产品、场景地点等 |
| 🔍 **场景分析** | 观察物品状态，推断用户真实意图 |
| 🛒 **精准选购** | 关联推荐防买错，线上线下双渠道 |
| ⚠️ **兼容提醒** | 容易买错的地方用 ⚠️ 标注 |

---

## 🚀 使用方法

### 触发方式

上传图片并说：
- "这是什么？"
- "帮我识别一下"
- "这是什么植物/动物/产品？"
- "帮我看看这张图"

### 示例场景

| 场景 | 输出 |
|------|------|
| 老旧三角阀 | 识别 + 分析可能漏水 + 选购方案（4分/6分别买错） |
| 台式机主板 | 识别 + 分析升级需求 + DDR4/DDR5兼容性说明 |
| 花卉植物 | 识别 + 养护建议 + 购买链接 |
| 景点建筑 | 识别 + 历史介绍 + 旅游攻略 |

---

## 🎯 识别三步法

```
第一步：识别物品 → 直给名称（如"三角阀"）
第二步：场景分析 → 观察状态推断意图
第三步：精准选购 → 关联推荐防买错
```
