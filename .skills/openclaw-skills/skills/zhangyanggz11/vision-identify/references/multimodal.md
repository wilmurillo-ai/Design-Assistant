# 拍照识万物 - 技术参考

## 多模态识别支持

### 内置支持
- WorkBuddy 当前使用 **腾讯混元 Turbo** 模型
- 支持图像输入和多模态识别

### 调用方式
用户上传图片后，系统会自动将图片作为上下文一起发送，模型直接返回识别结果。

## 识别增强策略

### 1. 单图识别
- 直接发送图片给模型
- 模型返回物体名称、特征描述

### 2. 多图对比识别
- 当需要对比多个物品时
- 按顺序发送多张图片
- 请求模型对比分析

### 3. 特定领域识别
- **植物**：结合叶片形态、花朵结构
- **动物**：结合体型特征、生活环境
- **商品**：结合品牌标识、外观特征
- **文字**：结合语言环境、上下文

## 识别置信度说明

| 置信度 | 等级 | 说明 |
|--------|------|------|
| 95%+ | 非常有把握 | 可以直接确认 |
| 80-95% | 比较确定 | 提供首选答案 |
| 60-80% | 不太确定 | 提供可能选项 |
| <60% | 无法确定 | 说明限制，建议其他方式 |

## 常见问题处理

### 图片模糊
→ 尝试识别主要特征，降低置信度标注

### 多物体混杂
→ 识别主要/主体物体，可补充说明背景物体

### 罕见物品
→ 提供最可能的类别，无法确定时诚实说明

## 扩展能力

### 可选功能
1. **OCR 文字提取** - 识别图片中的文字
2. **数学公式识别** - 识别并可进一步解答
3. **多语言翻译** - 识别并翻译外文内容
4. **相似物推荐** - 基于识别结果推荐相似物品
