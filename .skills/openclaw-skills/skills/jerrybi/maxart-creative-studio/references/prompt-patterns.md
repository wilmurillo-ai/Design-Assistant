# Prompt Patterns

Use these patterns as starting points, not rigid templates.

## 1. Product hero still

Use when the user wants a clean ecommerce hero image or launch visual.

Template:
- Subject: [product]
- Action/state: centered, pristine, premium presentation
- Environment: [studio / soft gradient / reflective surface / branded set]
- Composition: hero shot, high detail, clear silhouette, premium framing
- Lighting: soft key light, rim light, controlled highlights
- Style: commercial product photography, polished, modern, conversion-oriented

Example:
A premium wireless headset displayed as a hero product shot on a soft graphite gradient background, centered composition, subtle reflective surface, crisp silhouette, soft key light with clean rim highlights, commercial product photography, polished modern launch campaign aesthetic.

## 2. Lifestyle ad creative

Use when the user wants social ads, paid media, or website banners.

Template:
- Subject: [person using product / product in context]
- Action: [specific use moment]
- Environment: [home / office / city / gym / cafe]
- Composition: ad-friendly framing with space for headline or CTA
- Lighting: natural or cinematic depending on brand
- Style: believable commercial lifestyle imagery

Example:
A young creator using an AI design tool on a laptop in a sunlit studio, focused expression, modern creative workspace, medium shot with negative space for ad copy, soft natural morning light, believable premium lifestyle advertising, clean editorial-commercial style.

## 3. Stylized transformation

Use when the user wants to transform an existing image while keeping core identity.

Template:
- Preserve: [identity / product shape / pose / composition]
- Change: [style / palette / texture / setting]
- Target style: [editorial / anime / cinematic / clay / retro-futurist]
- Quality guardrails: keep facial structure, product geometry, and readable branding intact

Example:
Preserve the subject pose, facial structure, and jacket silhouette; transform the scene into a neon retro-futurist night market with reflective wet streets, magenta-cyan palette, cinematic haze, high-detail editorial finish, while keeping the person recognizable and proportions natural.

## 4. Image-to-video motion direction

Use when the user already has a strong keyframe and wants motion guidance.

Template:
- Opening frame: [describe the starting still]
- Camera move: [slow push-in / orbit / handheld drift / tilt-up]
- Subject motion: [blink / walk / hair movement / product rotation]
- Secondary motion: [fog / particles / reflections / crowd]
- Pacing: [slow elegant / energetic / suspenseful]
- End frame: [describe where the clip should land]

Example:
Open on a still hero portrait of a fashion model under neon signage, slow cinematic push-in with slight handheld drift, subtle hair and jacket movement, reflections flicker on the wet street behind, moody medium pacing, end on a confident close framing with the subject looking directly into camera.

## 5. Short-form video ad concept

Use when the user wants a 6-15 second marketing clip.

Structure:
- Hook shot
- Product reveal or key transformation
- Benefit moment
- CTA end frame

Example flow:
- Shot 1: chaotic messy desktop
- Shot 2: MaxArt-style instant visual transformation
- Shot 3: clean polished campaign asset appears
- Shot 4: end frame with product and CTA

## 6. Multi-variant testing pack

Use when the user wants several strong prompt angles fast.

Vary only 1-2 dimensions at a time:
- environment
- camera distance
- lighting mood
- style intensity
- color system
- realism vs stylization

Suggested matrix:
- Variant A: premium studio
- Variant B: lifestyle usage
- Variant C: bold high-contrast ad
- Variant D: cinematic narrative
- Variant E: playful stylized concept

## 7. Negative constraints

Use negative constraints to remove common generation failure modes.

Common examples:
- distorted hands
- extra fingers
- warped product geometry
- unreadable text
- duplicate objects
- asymmetrical eyes
- muddy lighting
- cluttered background
- oversaturated skin tones
- low-detail materials

## 8. Production notes

Always end with one short suggestion about what to test next.

Examples:
- Test the same prompt with a tighter crop for paid social.
- Keep the product fixed and swap only the environment.
- Increase motion in the background, not the main subject.
- If text rendering breaks, remove text from the generation and add it in post.
