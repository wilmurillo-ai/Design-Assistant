# Benchmark Analysis

## 核心原则

前 3 秒决定生死。

钩子 = 文案钩子 + 视觉钩子。

两条线缺一不可。

## 7 大拆解维度

### 1. Hook

必须同时看：

- 文案钩子类型
- 视觉钩子类型
- 第一帧能否单独吸引停留
- 声音 / 语气 / 字幕是否强化同一信息

常见文案钩子：

- 反常识
- 痛点
- 数据
- 场景
- 悬念
- 利益

常见视觉钩子：

- 视觉奇观
- 产品堆叠
- 画面突变
- 动作冲击
- 文字弹出
- 第一人称 POV

### 2. Storyboard

逐镜头拆：

- 时长
- 景别
- 画面内容
- 旁白 / 字幕
- 转场方式

### 3. Pace

看节奏属于：

- 快-慢-快
- 渐进加速
- 脉冲式

### 4. Visual System

看：

- 色调
- 字幕样式
- 贴纸与花字
- 产品展示方式

### 5. Conversion Design

看：

- CTA 在哪里出现
- 是软引导还是硬引导
- 是否有价格锚定
- 是否有紧迫感

### 6. Compliance Risk

看：

- 绝对化用语
- 功效承诺
- 虚假或无法验证数据
- 平台规则高风险点

### 7. Transferability

必须区分：

- 可复制：句式、结构、节奏、镜头逻辑
- 不可复制：达人势能、独特场景、原品牌资产
