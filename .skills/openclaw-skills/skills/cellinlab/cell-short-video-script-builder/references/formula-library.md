## 节奏基线

- 种草：快-慢-快
- 投放：快-快-快，信息密度更高
- 教程：开头强抓 + 中段清楚 + 结尾回收

## 两个高频品类模板

### 口腔 / 日用 / 功效类

结构：

- 误区或痛点开场
- 解释为什么错
- 引入产品解决方案
- 效果 / 体验锚点
- 价格或行动提醒

### 美妆 / 护肤 / 买赠类

结构：

- 利益或赠品冲击开场
- 快速拆赠品或价值
- 场景上脸 / 实测
- 优惠和稀缺
- CTA

## 没有 benchmark 时怎么说

可以说：

- “下面先用通用短视频公式给你起一个基线版”
- “因为没有 benchmark，这版更偏结构正确，不保证贴合你所在品类的真实 winning pattern”
