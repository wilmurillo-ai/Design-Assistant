# Core Models

## 1. Leverage

一句话：不要只用时间换结果，要找能被放大的投入。

Naval 最关心：

- 劳动力
- 资本
- 代码
- 媒体

适用：

- 职业路径
- 商业模式
- 内容与产品的长期复利

## 2. Specific Knowledge

一句话：真正的竞争力常常长在你觉得像玩、别人却学不来的交叉地带。

适用：

- 你到底该做什么
- 你有哪些能力是无法被标准化培训替代的
- 一个机会是不是与你的真正优势对齐

## 3. Desire as Contract

一句话：每一个欲望，都是你和不快乐签的一份合同。

适用：

- 焦虑、纠结、分心
- 同时想做太多事
- 想判断某个目标是不是自己真正想要的

## 4. Redefine the Word

一句话：很多问题不是答案错了，而是定义先错了。

适用：

- 讨论“财富”“退休”“成功”“自由”这种大词
- 一个问题怎么想都打转
- 你怀疑自己在用别人的定义活着

## 5. Pain to Systemic Solution

一句话：别只修补自己的单个问题，试着重构产生问题的系统。

适用：

- 反复遇到同一类低效与不公
- 想把个人经验转成更通用的方法、产品或机制
