# Core Models

## 1. Focus = Saying No

一句话：聚焦不是对一个方向说 yes，而是对一百个好主意说 no。

适用：

- 功能越来越多
- 战略优先级太散
- 不知道该保留什么、砍掉什么

## 2. The Whole Widget

一句话：关键体验链条如果不在你手里，最终体验就不会真正由你定义。

适用：

- 产品体验由太多外部环节决定
- 技术路线与平台依赖纠结
- 在“开放”与“控制”之间做取舍

## 3. Connecting the Dots

一句话：很多重要投入在当下没有清晰 ROI，只能事后回看时连成线。

适用：

- 人生路径选择
- 看似无关的兴趣和训练
- 判断长期积累是否值得

## 4. Death as Decision Tool

一句话：真正的大决策，应该用更高密度的生命标准过滤。

适用：

- 是否继续忍受不想做的事
- 是否要做重大转向
- 是否在活别人的期望

## 5. Reality Distortion Field

一句话：很多“不可能”并不是真的做不到，而是团队在用旧框架评估目标。

适用：

- 团队把目标默认判死
- 标准不够高
- 需要突破自我设限

注意：

- 这套镜片有强副作用，不能无限放大

## 6. Technology × Liberal Arts

一句话：技术单独存在不够，真正打动人的产品来自技术与人文理解的结合。

适用：

- 产品只剩参数，没有感受
- 设计被降为表层美化
- 团队在“功能对”和“体验好”之间分裂
