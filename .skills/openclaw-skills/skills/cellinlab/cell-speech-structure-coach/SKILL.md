# Speech Structure Coach

## Overview

这个 Skill 适合：

- 线下分享
- 工作坊
- 技术演讲
- B 站教程
- Keynote / PPT 讲稿审查

重点不是润色句子，而是控制注意力和结构。

## Quick Start

1. 先判断是演讲稿、大纲、视频教程，还是幻灯片检查。
2. 先做开场承诺，再看中段结构和结尾。
3. 结构问题优先于文采问题。

## Workflow

### Step 1: Scope the Format

优先判断：

- 线下演讲
- 教程视频
- 幻灯片审查
- 讲稿结构审查

### Step 2: Apply the Heuristics

默认读取 [references/winston-heuristics.md](references/winston-heuristics.md)。

重点检查：

- empowerment promise
- cycling
- build a fence
- verbal punctuation
- ask and wait
- ending

### Step 3: Return Structural Advice

默认交付：

- 开场问题
- 中段结构建议
- 注意力维持建议
- 幻灯片问题
- 结尾改法

## Hard Rules

Do not:

- 只给“更生动一点”这类空建议
- 用 PPT 代替结构
- 把结尾做成“谢谢观看”

Always:

- 先讲承诺
- 给清晰结构提示
- 检查观众何时会掉线
- 给有力的结束方式
