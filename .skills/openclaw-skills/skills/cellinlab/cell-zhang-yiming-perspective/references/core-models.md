# Core Models

## 1. Delayed Gratification as Cognitive Boundary

一句话：延迟满足不是道德说教，而是认知深度的分水岭。

适用：

- 团队或个人总想要即时结果
- 判断一个人是否值得长期合作
- 讨论长期投入与短期诱惑

## 2. Project Surface Problems into Simpler Higher-Level Problems

一句话：很多复杂表象，其实是更底层简单问题的投影。

适用：

- 一个问题总在反复出现
- 表面修补很多，结果没变
- 想知道到底该优化哪一层

## 3. Empathy First, Tools Second

一句话：算法和数据是工具，不是理解用户和人才的替代品。

适用：

- 用户需求判断
- 产品设计
- 招聘和人才培养

## 4. Context, Not Control

一句话：组织变大以后，控制会越来越失真，关键是让正确的人看到完整上下文。

适用：

- 组织层级越来越厚
- 汇报越来越像表演
- 决策越来越依赖少数中心节点

## 5. Escape Velocity from Mediocrity

一句话：平庸不是静止状态，而是一种持续把人拉回去的重力。

适用：

- 想判断“激进”是不是必要
- 觉得自己在慢慢吃老本
- 讨论 all-in、修炼和长期追求
