# Workflow

## 目标校准

这个 skill 不是为了产出“这本书主要讲了什么”的概览，而是为了回答：

- 这本书里哪些内容可以变成观点
- 哪些内容可以变成故事或案例
- 哪些表达可以被改写成金句
- 怎样把这些内容重组成一篇可继续深写的素材文章

如果你发现自己正在逐章复述，说明已经偏离目标。

## 输入模式判断

优先判断当前是哪种输入模式：

### 1. 书名驱动

用户只给书名或作者名。

处理建议：

- 先建立保守认知，不要假装自己拿到了全文
- 优先抓作者核心命题、目录骨架、公开摘要、可靠书评、作者访谈
- 把输出重点放在“可能成立的写作方向”与“已知高置信材料”
- 对覆盖不足处显式标注

### 2. 摘录驱动

用户给了若干段落、章节摘录或句子。

处理建议：

- 把摘录当成局部窗口，不要外推成全书结论
- 先识别摘录背后的问题意识和观点轴
- 可重点产出观点块和金句块

### 3. 笔记驱动

用户给的是读书笔记、划线、摘要或自己的拆解。

处理建议：

- 先区分“原书内容”和“用户自己的理解”
- 优先提炼用户已经感兴趣的部分
- 适合直接生成素材块和素材文章

### 4. 章节驱动

用户提供章节梗概、目录或部分章节全文。

处理建议：

- 先重建章节的因果关系和推进关系
- 识别哪些章节提供观点，哪些章节提供故事，哪些章节只是过渡
- 不要求每章都保留

### 5. 全书或高覆盖度输入

用户提供了完整内容或极高覆盖度材料。

处理建议：

- 可以做完整三轮
- 允许构建更稳的观点库、故事库和素材文章
- 仍然要克制，目标是可写素材，不是百科全书

## 三轮工作流

## 第一轮：主题与张力扫描

目标：把“书”压成“值得写的几个核心矛盾”。

至少提炼：

- 中心问题：作者到底在试图回答什么
- 核心回答：作者给出的最核心判断
- 关键张力：这本书最重要的矛盾、冲突、反差或悖论
- 概念骨架：高频概念与它们之间的关系
- 写作角度：哪些角度适合拿去写成内容

优先问自己：

- 这本书真正对抗的是什么误解
- 哪个判断最可能引发读者“停一下，我得想想”
- 哪组概念最适合变成文章中的主线结构

避免：

- 把目录改写一遍当作结构
- 只说“作者很有启发”
- 只说“大意如此”却没有清晰张力

## 第二轮：素材颗粒提炼

目标：把内容拆成能直接拿去写的颗粒。

### 观点提炼法

每条观点尽量包含：

- 观点母句
- 这条观点在反驳什么
- 为什么对今天的读者有价值
- 能展开成哪几类文章
- 证据基础或来源位置

### 故事提炼法

优先识别：

- 人物决定时刻
- 冲突升级过程
- 实验设计与结果
- 一个具体情境里的选择代价
- 能承载隐喻的场景

每个故事都尽量包含：

- 谁在什么场景里
- 遇到了什么矛盾
- 发生了什么转折
- 这个故事能证明什么
- 它适合写在开头、中段还是结尾

### 金句化提炼法

默认优先“金句化改写”，而不是照抄。

只有在以下情况才保留原句：

- 用户直接提供了原句
- 原句足够短
- 保留原句对分析有明显价值

每条金句都要说明：

- 是原句还是改写
- 适合用在开头、收束、过渡还是点题
- 它后面最适合接什么展开

### 素材块提炼法

素材块是为了减少未来写作时的拼装成本。

一个好素材块通常由 2-4 个颗粒组成，例如：

- 一个观点 + 一个故事
- 一个反常识判断 + 一个对比
- 一个概念解释 + 一个现实场景
- 一个结论 + 一个行动建议

## 第三轮：写作重组

目标：把颗粒变成可继续深写的素材文章。

先做两步：

### 1. 主线选择

从下面三种主线里选最稳的一种：

- 问题主线：围绕一个核心问题推进
- 观点主线：围绕一个中心判断展开
- 读者困境主线：围绕读者最常见的误区、困境或选择推进

### 2. 结构压缩

把前面提炼出的材料压成 3-5 个 section：

- 开头：问题、误解、冲突或场景
- 中段 1：核心判断与第一组证据
- 中段 2：反例、边界或更深一层机制
- 中段 3：迁移意义、现实含义或方法启发
- 结尾：给读者留下一个更强的问题、判断或行动方向

## 素材文章标准

素材文章应满足：

- 结构已经清楚，但还保留可继续深化空间
- 有论点，不只是信息拼盘
- 有素材密度，不是漂亮空话
- 有段落职责，不是流水账
- 可以直接交给 `$deep-writer` 做 `source-driven` 或 `draft-deepening`

## 何时停下来

出现以下情况时，不要强行补全：

- 用户只给了极薄材料
- 关键章节根本没有覆盖
- 重要故事只看到二手转述
- 书的中心问题还无法稳定判断

这时应改为：

- 明确已知范围
- 输出高置信素材
- 标记待补材料
- 只给一个保守版素材文章
