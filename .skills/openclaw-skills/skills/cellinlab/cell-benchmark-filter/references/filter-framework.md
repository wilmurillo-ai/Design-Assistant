# 五层筛选框架

## 1. Economic Signal

先看这是不是一个真实运转的生意或内容系统。

信号可以包括：

- 明确产品或服务
- 可见成交入口
- 稳定内容分发与承接关系
- 可推断的转化链

弱信号包括：

- 只有粉丝数
- 只有热闹评论
- 只有人设，没有产品

## 2. Model Legibility

判断你能不能大致看懂它怎么工作。

最低要求：

- 怎么被看见
- 为什么被信任
- 靠什么赚钱
- 如何交付

如果只看见结果，看不见机制，就还不适合深学。

## 3. Copyable Mechanism

把可复制部分和不可复制部分拆开。

较可复制：

- 选题结构
- 内容节奏
- 产品分层
- 交付路径
- 转化链条

较难复制：

- 十几年行业资历
- 名人背书
- 特殊历史窗口
- 独家渠道与关系

## 4. Stage Relevance

判断这个 benchmark 是否处在一个可比较的阶段。

如果对方已经是成熟公司或超级头部 IP，而用户还在最初期：

- 可以学局部机制
- 不要把整体状态拿来当直接对比对象

## 5. Ego-Noise Control

常见噪音：

- 我不喜欢这种表达
- 这不像我
- 这个看起来太土
- 这个太重复了

先问：

- 它是不值得学，还是只是“不够让我感觉高级”
- 真学不动，是机制不成立，还是 ego 在拒绝重复和笨功夫
