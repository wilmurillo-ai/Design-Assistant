# 12 类标题触发器速查

| 家族 | 编码前缀 | 核心机制 | 最适合 |
| --- | --- | --- | --- |
| 认知冲突 | C | 打破读者原有判断 | 观点、误区、反常识 |
| 好奇缺口 | G | 暗示你不知道的东西 | 内幕、揭秘、经验差 |
| 损失规避 | L | 激活“不看就亏” | 避坑、警告、纠错 |
| 身份代入 | I | 让目标人群觉得“在说我” | 精准圈人、垂直获客 |
| 数字锚定 | N | 用数字降低理解成本 | 清单、步骤、方法 |
| 结果承诺 | R | 给出可量化结果 | 教程、变现、效率 |
| 案例反转 | S | 用故事建立可信度 | 经验分享、案例复盘 |
| 争议挑衅 | D | 制造站队与讨论 | 观点表达、评论互动 |
| 场景条件 | M | 精准命中用户当前状态 | 解决方案、状态内容 |
| 行动号召 | A | 直接推进行动或放弃旧动作 | 态度、习惯、行动指南 |
| 权威借力 | U | 借他人或案例背书 | 方法论、认知迁移 |
| 互动测试 | T | 激活自测或互动冲动 | 测试、投票、评论 |

## 简化选择法

- 你在纠偏 -> `C` / `L`
- 你在吊起未知 -> `G`
- 你在精准找人 -> `I` / `M`
- 你在交付方法 -> `N` / `R`
- 你在讲自己或案例 -> `S`
- 你在制造讨论 -> `D` / `T`
- 你在借权威降低不确定性 -> `U`
- 你在推动人动起来 -> `A`
