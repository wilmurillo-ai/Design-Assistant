# 标题公式目录

以下是 Cell 版的小红书标题公式目录。

使用规则：

- 每次输出都要标注公式编号
- 公式逻辑可以换词，但不要换骨架
- 默认从不同家族里选 5-8 个，而不是在同一家族里堆满

## 一、认知冲突 C

- `C01` 为什么 [大家默认是对的事] 反而在伤害 [目标]  
  适合：反常识、误区纠偏
- `C02` 我为什么不再 [常见动作]  
  适合：个人立场、做法反转
- `C03` [几个普通动作] 的共同代价  
  适合：把零散现象并进一个新判断
- `C04` 为什么 [变化 / 新变量] 会改写 [旧做法]  
  适合：平台变化、趋势变化、规则变化

## 二、好奇缺口 G

- `G01` [一群人] 不会主动告诉你的 [建议 / 真相]  
  适合：内幕、经验差、行业信息差
- `G02` 关于 [话题]，大多数人太晚知道的 [数字] 件事  
  适合：教训、经验复盘
- `G03` [结果] 背后那个多数人没看见的关键  
  适合：解释类、方法类
- `G04` 看完这个，你对 [话题] 的看法会变  
  适合：认知升级、观点转换

## 三、损失规避 L

- `L01` [不想要的结果] 的最快方法  
  适合：避坑、反面提醒
- `L02` [结果做不成] 的根本原因  
  适合：失败归因、纠错
- `L03` [数字] 个正在让你 [努力 / 投入] 白费的动作  
  适合：清单式纠错
- `L04` 在 [平台 / 场景] 里，这 [数字] 件事千万别做  
  适合：平台规范、常见错误

## 四、身份代入 I

- `I01` 给每个 [身份 / 年龄层 / 状态人群] 的一个提醒  
  适合：精准圈人
- `I02` 为 [某类人] 量身定制的 [方案 / 路线]  
  适合：方案类、服务类
- `I03` [有某种特征] 的人，先看这条  
  适合：强筛选、首屏识别
- `I04` 给 [一群人] 的最好忠告  
  适合：经验型内容、关系型表达

## 五、数字锚定 N

- `N01` [数字] 个让你 [结果] 的方法  
  适合：清单、方法合集
- `N02` 搞懂 [话题] 的 [数字] 个步骤  
  适合：教程、路径拆解
- `N03` [结果] 的 [数字] 个关键动作  
  适合：执行类内容
- `N04` 关于 [话题] 的 [数字] 个最大谎言  
  适合：清单式反常识

## 六、结果承诺 R

- `R01` 如何不 [讨厌动作]，也能在 [时间] 内 [结果]  
  适合：低阻力承诺
- `R02` [数字] 天内做到 [结果] 的最短路径  
  适合：时间框架强的结果型内容
- `R03` 我如何在 [时间] 内做到 [结果]  
  适合：经验分享、案例验证
- `R04` 没有 [资源]，也能 [结果]  
  适合：降低门槛

## 七、案例反转 S

- `S01` 我是如何从 [坏状态] 到 [好结果]  
  适合：逆转故事
- `S02` 从 [经历] 里，我学到最重要的一课  
  适合：经验提炼
- `S03` [负面事件] 反而让我拿到了 [正面结果]  
  适合：反转叙事
- `S04` 一个真实案例，说明 [判断]  
  适合：案例驱动的观点表达

## 八、争议挑衅 D

- `D01` [动作 / 选择] 真的值得么？  
  适合：讨论、立场碰撞
- `D02` [X] 还是 [Y]，你到底该站哪边  
  适合：二选一争议
- `D03` 别再把 [旧说法] 当答案了  
  适合：打脸旧观点
- `D04` 你到底要不要 [动作 / 路线]  
  适合：选择焦虑、分叉判断

## 九、场景条件 M

- `M01` 如果你正在 [某个状态]，先做这件事  
  适合：状态命中
- `M02` 当你开始 [某件事]，前 [数字] 天最该做什么  
  适合：起步类内容
- `M03` 在 [人生阶段 / 业务阶段] 之后，你需要重做的事  
  适合：阶段切换
- `M04` 当 [某种情况出现]，你该怎么 [结果]  
  适合：条件式解决方案

## 十、行动号召 A

- `A01` 为什么你该停止 [旧动作]  
  适合：破旧立新
- `A02` 别再研究 [抽象东西]，先做 [具体动作]  
  适合：行动推动
- `A03` 戒掉 [普通习惯]，你会更容易 [结果]  
  适合：习惯类内容
- `A04` 如果你想要 [结果]，今天先做这一步  
  适合：首行动作

## 十一、权威借力 U

- `U01` [名人 / 机构 / 案例] 让我重新理解了 [话题]  
  适合：借权威引题
- `U02` 从 [名人 / 行业高手] 身上学到的 [关键能力]  
  适合：方法迁移
- `U03` 最好的 [一群人]，在 [动作] 上都做对了什么  
  适合：群体背书
- `U04` 一个经典案例，解释了为什么 [判断]  
  适合：案例解释观点

## 十二、互动测试 T

- `T01` 敢不敢测一测，你在 [话题] 上到底是哪一型  
  适合：测试、互动
- `T02` 你以为你在 [做对的事]，先对照这 [数字] 条  
  适合：自检型互动
- `T03` 关于 [话题]，你更同意哪一种说法  
  适合：评论互动
- `T04` 如果你也卡在 [问题]，看完来对答案  
  适合：同类人召集
