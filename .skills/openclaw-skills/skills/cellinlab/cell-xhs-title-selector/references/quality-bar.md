# 标题质量规则

## 1. 标题先保点击，再保兑现

一个标题至少要让读者产生下面之一：

- 我想点
- 这好像在说我
- 这和我想要的结果有关

但如果正文接不住，再强的标题也只是透支。

## 2. 不要把答案说完

常见错误：

- 直接在标题里把全部结论交代完
- 读者看完标题已经不需要点进来

优先保留：

- 问题
- 张力
- 结果差
- 认知差

## 3. 话题要尽量从术语回到真实痛点

优先：

- 赚钱
- 涨粉
- 不上班
- 被看见
- 更轻松
- 少踩坑

谨慎：

- 过多行业黑话
- 只有圈内人才懂的缩写

## 4. 标题至少命中两个张力元素

可选元素：

- 对比 / 反差
- 数字
- 好奇心
- 冲突
- 时间框架
- 结果承诺

如果一个都没有，标题通常太平。

## 5. 默认短而不短促

默认尽量控制在 20 个汉字左右。

但优先级是：

1. 清楚
2. 有张力
3. 再压缩

不要为了死卡字数把标题压得只剩空壳。

## 6. 先判断“配不配”，再判断“炸不炸”

最常见失败：

- 争议型标题配了一个很温和的正文
- 结果承诺型标题配了一个没有结果支撑的内容
- 身份代入型标题其实没找准目标人群
