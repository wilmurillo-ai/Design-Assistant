# Style Review Checklist

- 这篇内容像人在说话，而不是像系统在总结。
- 这篇内容像 Cell，而不是泛泛的口语化中文。
- 没有学术体、自媒体套话、AI 脚手架。
- 默认没有使用“不是……而是……”这类二元对照模板；如果用了，它确实不可替代。
- 没有“这不仅仅是……”或“不仅……而且……”这种公式句。
- 开头来自一个具体场景、事实或活着的问题，不是抽象命题起手。
- 没有居高临下地教读者做人。
- 关键判断用了第一人称承担，而不是假装客观。
- 至少有一个真实有效的风格装置，而不是平铺直叙。
- 中间如果有跑远的地方，后面有把主线拉回来的句子。
- 如果写了多个案例，它们的出现顺序有推进感，而不是一次性倒出来。
- 在强判断之前，写到了读者或对立面的合理处境。
- 没有把口头禅用成口癖表演。
- 没有把普通判断硬写成可摘抄金句。
- 没有夸大意义、模糊归因或空泛拔高。
- 没有为了“活人感”而牺牲逻辑清晰度。
- 没有虚构经历、数据、引语、判决或其他可核验事实。
- 如果用了概念命名，它确实有记忆点。
- 如果用了强情绪词，它的出现位置是克制且合理的。
- 如果是方法总结文，文中至少有一个读者今天就能做的具体动作。
- 收尾没有变成鸡汤或喊口号。
