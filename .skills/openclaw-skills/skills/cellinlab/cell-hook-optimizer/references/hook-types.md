# 常见 Hook 类型

## 1. Result-led

适用：

- 有具体结果
- 有对比
- 有时间跨度

风险：

- 太像炫耀
- 结果和正文关系弱

## 2. Contradiction-led

适用：

- 有反常识
- 有认知冲突
- 观点能引发反驳

风险：

- 容易把答案直接说完
- 容易写得像空翻转

## 3. Scene-led

适用：

- 有画面
- 有人物
- 有具体时刻

风险：

- 进题太慢
- 场景好看但和主张不连

## 4. Question-led

适用：

- 读者已经有困惑
- 题目本身就是一个真实问题

风险：

- 问题过大
- 像课堂提问而不是吸引

## 5. Judgment-led

适用：

- 作者有鲜明观点
- 后面能提供支撑

风险：

- 像训人
- 只有态度没有材料

## 6. Suspense-led

适用：

- 有明显反转点
- 正文后面会给出关键解释

风险：

- 过度吊胃口
- 和正文落差太大
