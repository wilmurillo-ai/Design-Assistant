# Authority-First Research

## Source Priority

按以下顺序优先选源：

1. 官方文档 / 官方课程 / 官方规范
2. 标准组织与监管机构资料
3. 项目维护者与一手发布渠道
4. 高质量实践复盘（带可验证过程）
5. 普通二手解读（仅作补充，不作核心依据）

## Minimum Evidence Rule

每个核心结论至少对应 1 个权威来源；
高风险结论（涉及合规、健康、财务）至少 2 个独立来源交叉验证。

## Source Card Fields

为每个关键来源记录：

- 标题
- URL
- 访问日期
- 来源类型
- 可信理由
- 可用于哪个学习阶段
- 风险备注（是否已过时/是否存在争议）

## Anti-Patterns

- 只看短视频二创，不回到原始材料
- 只做“观点摘抄”，不做来源追溯
- 把未经验证的经验当成通用结论
