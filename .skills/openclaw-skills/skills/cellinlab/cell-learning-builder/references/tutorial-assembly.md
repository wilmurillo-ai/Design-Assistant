# Tutorial Assembly Contract

## Core Principle

学习包必须同时回答三件事：

1. 学什么（知识边界）
2. 怎么学（阶段动作）
3. 学完如何验证（产出与验收）

## Required Sections

1. 学习目标与边界
2. 学习者画像摘要
3. 阶段化学习路径（建议 3-5 阶段）
4. 每阶段练习与验收标准
5. 常见误区与纠偏动作
6. 来源附录
7. 30 天复盘节奏

## Stage Definition

每个阶段至少包含：

- 阶段目标
- 推荐输入材料
- 关键动作（动词开头）
- 阶段产物
- 验收标准
- 预计时间

## Creator-Oriented Extension

若用户是创作者，默认加一个“输出转化层”阶段：

- 将学习内容转成文章提纲 / 视频脚本 / 分享讲义
- 列出 3 个可发布选题
- 给出下一步可交接给 `deep-writer` 或 `hook-optimizer` 的输入包
