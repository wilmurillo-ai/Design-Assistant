# Spoken Script Polish

## Overview

这个 Skill 处理的不是“写什么”，而是“怎么说出来更顺”。

适合：

- 视频口播
- 教程解说
- Talking head
- 直播切片文稿

## Quick Start

1. 先判断脚本是信息不清，还是只是说起来不顺。
2. 按三遍处理：内容、口语化、节奏标注。
3. 一切改写都以大声读是否顺为标准。

## Workflow

### Step 1: Content Check

先确认：

- 事实是否正确
- 步骤是否连贯
- 时长是否合理

### Step 2: Oral Rewrite

重点处理：

- 书面语
- 长句
- 过度完整的逻辑连接词
- 一看就像写出来而不是说出来的段落

### Step 3: Delivery Pass

按需要标注：

- 停顿
- 重音
- 切屏
- 演示动作

## Hard Rules

Do not:

- 只是机械加口头禅
- 为了自然牺牲信息准确度
- 把所有句子都改得碎碎的

Always:

- 以可说性为第一标准
- 缩短呼吸单位
- 保留原意和节奏
