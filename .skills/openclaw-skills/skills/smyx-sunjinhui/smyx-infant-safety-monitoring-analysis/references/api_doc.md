# 婴儿智能安全看护分析 API 文档

## 接口概述

本技能调用云端统一开放接口，基于视觉AI模型实现婴儿行为安全监测。

## 支持识别的高风险行为

| 风险行为类型 | 描述 | 风险等级 |
|--------------|------|----------|
| 口鼻遮挡 | 被褥、毛巾等物体遮挡婴儿口鼻 | 🔴 高危 |
| 趴着睡眠 | 婴儿长时间俯卧位睡眠姿势 | 🟠 中高危 |
| 爬出床栏 | 婴儿攀爬、试图翻越婴儿床围栏 | 🔴 高危 |
| 坠床风险 | 婴儿身体已接近床边，存在坠床可能 | 🟠 中高危 |
| 危险攀爬 | 攀爬家具、窗台等危险区域 | 🔴 高危 |
| 接触危险物 | 接触小物件、尖锐物品等危险品 | 🟠 中高危 |

## API 响应字段说明

### 基础信息

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 分析记录ID |
| data.analysis_time | string | 分析时间 |
| data.infant_detection.status | string | 婴儿检测状态 |
| data.infant_detection.quality_score | int | 画面质量评分 0-100 |

### 诊断结果

| 字段 | 类型 | 说明 |
|------|------|------|
| data.diagnosis.safety_score | int | 整体安全评分 0-100 |
| data.diagnosis.overall_safety | string | 整体安全状况：安全/轻度风险/中度风险/高度风险 |
| data.diagnosis.risk_behavior | object | 各风险行为检测结果 |
| data.diagnosis.body_position | object | 身体姿势评估结果 |
| data.diagnosis.environment | object | 环境安全评估结果 |

### 预警与建议

| 字段 | 类型 | 说明 |
|------|------|------|
| data.safety_warnings | array[string] | 安全警示信息列表 |
| data.care_suggestions | array[string] | 看护建议列表 |

## 错误码说明

| 错误码 | 说明 |
|--------|------|
| 200 | 请求成功 |
| 400 | 请求参数错误 |
| 401 | API 鉴权失败 |
| 413 | 文件大小超出限制 |
| 415 | 不支持的文件格式 |
| 500 | 服务器内部错误 |
| 503 | 服务繁忙，请稍后重试 |

## 注意事项

1. 本工具为辅助监测工具，不能替代家长/监护人的实时看护
2. 分析结果仅供参考，不构成专业护理建议
3. 视频清晰度、光线条件会影响识别准确率
4. 建议在光线充足、画面稳定的条件下拍摄
