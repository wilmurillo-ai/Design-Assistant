# 智能失禁状态提醒分析 API 文档

## 接口概述

本技能调用云端视觉AI接口，自动识别护理对象衣物潮湿和排泄异常状况，及时提醒看护人员，提升护理质量，减少压疮、红臀等并发症发生。

## 支持识别的异常状态

| 异常类型 | 描述 | 预警级别 |
|----------|------|----------|
| 衣物轻微潮湿 | 衣物局部潮湿，未渗透，可安排时间更换 | 🟡 需要关注 |
| 衣物中度潮湿 | 衣物明显潮湿，需要更换护理用品 | 🟠 需要处理 |
| 排泄异常 | 明显排泄，需要立即清洁更换 | 🔴 需要立即处理 |
| 正常干燥 | 衣物干燥，无需处理 | 🟢 正常 |

## 适用人群

| 对象类型 | 特点 |
|----------|------|
| 失能老人 | 长期卧床，无法自主表达 |
| 卧床病人 | 术后康复需要长期卧床 |
| 婴幼儿 | 无法自主表达，需要及时更换尿布 |
| 其他 | 其他需要护理的人群 |

## 检测模式

| 模式 | 适用场景 |
|------|----------|
| 实时监控 | 24小时持续监测，异常即时报警 |
| 定时巡查 | 按固定时间间隔拍照检测，节省资源 |

## API 响应字段说明

### 基础信息

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 分析记录ID |
| data.analysis_time | string | 分析时间 |
| data.person_detection.status | string | 对象检测状态 |
| data.person_detection.quality_score | int | 画面质量评分 0-100 |

### 诊断结果

| 字段 | 类型 | 说明 |
|------|------|------|
| data.diagnosis.health_score | int | 整体状况评分 0-100 |
| data.diagnosis.alert_level | string | 预警等级：normal/warning/alert/emergency |
| data.diagnosis.damp_clothing | boolean | 是否检测到衣物潮湿 |
| data.diagnosis.excretion_detected | boolean | 是否检测到排泄 |
| data.diagnosis.need_change | boolean | 是否需要更换护理用品 |
| data.diagnosis.clothing_condition | object | 各部位衣物状态 |
| data.diagnosis.detection_area | object | 各检测区域评估 |

### 警示与建议

| 字段 | 类型 | 说明 |
|------|------|------|
| data.care_warnings | array[string] | 护理警示信息列表 |
| data.care_suggestions | array[string] | 清洁护理建议列表 |

## 错误码说明

| 错误码 | 说明 |
|--------|------|
| 200 | 请求成功 |
| 400 | 请求参数错误 |
| 401 | API 鉴权失败 |
| 413 | 文件大小超出限制 |
| 415 | 不支持的文件格式 |
| 500 | 服务器内部错误 |
| 503 | 服务繁忙，请稍后重试 |

## 隐私与伦理提示

1. 本工具仅作为辅助提醒工具，不能替代人工检查和专业护理
2. 检测结果受光线、角度、衣物厚度等因素影响，可能存在误判
3. 涉及个人隐私敏感信息，请妥善保管检测记录，仅授权看护人员访问
4. 发现预警后请及时进行人工确认，避免延误护理
