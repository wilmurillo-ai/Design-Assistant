## 核心概念

### 1. 几何结构分析

几何结构分析基于K线形态的几何特征判断趋势方向和强度。

**主要形态类型：**

| 形态 | 方向 | 说明 |
|------|------|------|
| 上升趋势线 | 看涨 | 连接逐步升高的低点 |
| 下降趋势线 | 看跌 | 连接逐步降低的高点 |
| 头肩顶 | 看跌 | 左肩、头、右肩，跌破颈线 |
| 头肩底 | 看涨 | 左肩、头、右肩，突破颈线 |
| 双重顶 | 看跌 | 两个相近高点 |
| 双重底 | 看涨 | 两个相近低点 |
| 三角形整理 | 中性 | 收敛形态，突破后确认方向 |
| 旗形整理 | 延续 | 短期整理后延续原趋势 |

### 2. MACD风洞结构

MACD风洞结构是多周期共振的核心分析方法。

**风洞组成：**

1. **本周期风洞**
   - 上涨风洞：红柱放大→缩小到接近0/变绿→迅速变红/扩大
   - 下跌风洞：绿柱放大→缩小到接近0/变红→迅速变绿/扩大

2. **1/2.5周期信号**
   - 周线对应2日线
   - 日线对应90分钟
   - MACD零轴金叉/死叉

**共振判定：**
- 两部分同时满足 = 标准风洞
- 部分满足 = 模糊处理

### 3. 缩量短线

成交量和波动率同时收缩，预示后期将爆发。

**判断标准：**
- 成交量低于均量70%
- ATR/波动率低于均值70%
- 多周期同时收敛 = 强信号

## 分析方法详解

### 几何结构分析流程

```
1. 获取K线数据（至少60根）
2. 计算OHLC特征
3. 趋势线检测（线性回归斜率）
4. 形态识别（极值点检测）
5. 评分计算
```

### MACD风洞分析流程

```
1. 计算MACD指标（12,26,9）
2. 检测本周期风洞结构
3. 检测1/2.5周期零轴交叉
4. 多周期共振评估
5. 综合评分
```

### 缩量短线分析流程

```
1. 计算ATR和波动率
2. 分析成交量趋势
3. 检测波动率收敛
4. 多周期共振判断
5. 评分计算
```

## 评分标准

### 核心因子（70分）

| 因子 | 满分 | 评分要点 |
|------|------|----------|
| 几何结构 | 25 | 趋势线10分 + 形态15分 |
| MACD风洞共振 | 30 | 风洞15分 + 零轴15分 |
| 缩量短线 | 15 | 成交量趋势8分 + 波动率7分 |

### 辅助因子（30分）

| 因子 | 满分 | 评分要点 |
|------|------|----------|
| 基本面 | 10 | PE、营收、行业地位 |
| 情绪面 | 10 | 资金流向、板块热度 |
| AI预测 | 10 | 趋势外推、概率统计 |

### 综合评级

| 评分 | 评级 | 操作建议 |
|------|------|----------|
| 70-100 | 强烈看涨 | 可加仓 |
| 50-70 | 偏多 | 可买入 |
| 30-50 | 中性 | 观望 |
| 0-30 | 看跌 | 减仓/观望 |

## 注意事项

1. **数据质量**：确保K线数据完整准确
2. **周期选择**：建议使用月线、周线、日线组合
3. **风险控制**：评分仅供参考，需结合实际仓位管理
4. **持续更新**：定期重新分析，及时调整策略
