# Skills.sh / SkillsMP Index

This file makes QA Pilot discoverable on skills.sh, SkillsMP, and other agent skill directories.

## Installation

```bash
# Via skills.sh (Vercel)
npx skills add helal-muneer/qa-pilot

# Via ClawHub
clawhub install qa-pilot

# Manual
git clone https://github.com/helal-muneer/qa-pilot.git
```

## Compatibility

- ✅ OpenClaw (native)
- ✅ Claude Code (via SKILL.md)
- ✅ Cursor (via .cursorrules)
- ✅ Codex CLI (via SKILL.md)
- ✅ Any AI agent that reads markdown instructions

## What This Does

QA Pilot teaches AI agents to test their own work before reporting completion. It eliminates the "I'm done → user finds bugs → back and forth" loop that makes AI-assisted development frustrating.

[Learn more →](https://github.com/helal-muneer/qa-pilot)
