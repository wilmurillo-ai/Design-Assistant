    About TaskFlow

    TaskFlow is a modern, minimalist task management app designed to help you stay organized and productive.

    Built with simplicity in mind — no distractions, just your tasks.

    Features

    • Add, complete, and delete tasks
    • Priority levels (High, Medium, Low)
    • Dark mode support
    • Filter tasks by status
    • Local storage persistence
    • Responsive design for all devices
