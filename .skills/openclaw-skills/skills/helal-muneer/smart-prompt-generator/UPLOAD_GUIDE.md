## 🚀 خطوات الرفع على GitHub

### 1️⃣ إنشاء Repository على GitHub

```
1. اذهب إلى: https://github.com/new
2. الاسم: skill-prompt-generator
3. الوصف: AI Prompt Generator skill for OpenClaw with Flutter templates
4. نوعه: Public
5. لا تضف README (لدينا واحد)
6. اضغط "Create repository"
```

### 2️⃣ رفع الملفات

```bash
cd ~/.openclaw/workspace/skills/prompt-generator

# تهيئة Git
git init

# إضافة جميع الملفات
git add .

# إنشاء commit أولي
git commit -m "🎉 Initial release: AI Prompt Generator skill v1.0.0

Features:
- 30+ comprehensive templates for Flutter apps and games
- Professional documentation with examples
- GitHub integration (Actions, Issue templates, PR template)
- ClawHub and NPM ready
- Community-focused (Contributing, Code of Conduct, Security)

Categories:
- Flutter App Development (10 templates)
- Flutter Game Development (10 templates)
- Creative Writing
- Problem-Solving
- Educational
- Business & Marketing
- Programming

Examples:
- Flutter Authentication Feature
- Game Jump Mechanic

Documentation:
- Comprehensive README with badges
- Contributing guide
- Security policy
- Code of conduct
- Changelog

Time saved: 8-10 hours per project
Templates: 30+
Examples: 2 detailed examples
"

# تحديد الفرع الرئيسي
git branch -M main

# إضافة remote
git remote add origin https://github.com/YOUR_USERNAME/skill-prompt-generator.git

git push -u origin main
```

## 🎯 خطوات النشر على ClawHub

### 1️⃣ التسجيل في ClawHub

```
1. اذهب إلى: https://clawhub.com
2. سجل دخول باستخدام GitHub
3. اربط حسابك
```

### 2️⃣ إضافة المهارة

```
1. اضغط "Add Skill"
2. اختر repository من قائمة repos الخاصة بك
3. تأكد من وجود clawhub.json
4. اضغط "Publish"
```

### 3️⃣ التحقق

```
ClawHub سيقوم بـ:
- فحص SKILL.md
- التحقق من clawhub.json
- قراءة metadata
- نشر المهارة
```
