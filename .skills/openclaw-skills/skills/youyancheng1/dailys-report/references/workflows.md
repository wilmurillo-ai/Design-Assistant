# Daily Report Workflows

## Workflow 1: 快速日报

```
用户: 生成今日日报
贾维斯: [自动读取memory文件，生成日报]
```

**适合场景：** 每日收工时快速生成

---

## Workflow 2: 导出分享

```
用户: 生成日报导出到桌面
贾维斯: [生成日报，保存到桌面]
        ✅ 已保存到: ~/Desktop/日报_2026-03-17.txt
```

**适合场景：** 需要发给领导或存档

---

## Workflow 3: Word格式

```
用户: 生成Word日报
贾维斯: [生成Word文档]
        ✅ Word日报已生成: ~/Desktop/日报_2026-03-17.docx
```

**适合场景：** 正式汇报

---

## Workflow 4: 自定义日期

```
用户: 生成昨天的日报
贾维斯: [读取昨天的记忆文件，生成日报]
```

**适合场景：** 补写日报

---

## Workflow 5: 手动输入

```
用户: 帮我生成日报。今天完成了：1.效果图修改 2.客户沟通 3.文件整理
贾维斯: [根据用户输入生成日报]
```

**适合场景：** 没有记忆文件时
