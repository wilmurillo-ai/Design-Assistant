# upcoming-metal-concerts

OpenClaw skill to search for upcoming metal concerts and festivals worldwide using concerts-metal.com.

No API key required.

## Install

Copy this directory into `~/.openclaw/skills/upcoming-metal-concerts/` or your workspace `skills/` folder.

## Standalone usage

```bash
python3 events.py --country ES --city Valencia
python3 events.py --country DE --genre "death"
python3 events.py --country GB --band "Kreator"
python3 events.py --list-countries
```
