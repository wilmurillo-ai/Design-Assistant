# Personal Zalo Automation (Unofficial)

## Overview
- `zca-js` is an unofficial library that automates Zalo Web.
- It typically requires cookies and device identifiers.
- Some tools wrap Zalo Web via `zca-cli` for automation.

## Risks and constraints
- Unofficial automation can violate platform terms.
- Accounts may be limited or banned.
- Cookies and session state are sensitive and must be protected.

## When to consider
- Only for personal, low-risk workflows.
- Only with explicit user consent and acceptance of risk.

## Operational notes
- Store cookies securely and rotate often.
- Avoid running on shared hosts.
- Prefer official bot APIs when possible.
