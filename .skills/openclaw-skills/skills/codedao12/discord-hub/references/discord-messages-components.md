# Messages and Components

## 1) Messages
- Create, edit, delete messages via REST.
- Use `allowed_mentions` to prevent accidental pings.
- Prefer embeds for structured content.

## 2) Components
- Buttons for primary actions.
- Select menus for multi-choice actions.
- Keep `custom_id` stable and parseable.

## 3) Modals
- Use modals for text input.
- Validate input server-side.
