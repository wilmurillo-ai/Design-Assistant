# 股票查询助手

## 概述
根据用户提供的股票代码或名称，通过东方财富/新浪财经API获取该股票的实时行情数据，返回当前价格、涨跌幅、成交量等核心指标。适用于快速查看股价、判断涨跌场景。

## 工作流程
1. 从用户消息中提取股票标识（代码优先，其次是名称）
2. 判断市场：6位数字且60/00/30开头 → A股；数字.HK → 港股；字母组合 → 美股
3. 调用对应市场的实时行情API
4. 解析返回的JSON数据
5. 格式化为易读文本输出
