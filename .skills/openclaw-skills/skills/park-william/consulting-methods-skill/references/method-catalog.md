# 管理咨询方法速查

## 1) 问题定义与拆解

### 是什么-为什么-怎么做
- 用途：统一问题定义，避免直接跳方案。
- 产出：目标定义、必要性说明、资源与行动框架。

### 脑图
- 用途：目标分解与要素穷举。
- 典型场景：制定计划、寻找影响因素、梳理层次关系。

### 鱼骨图
- 用途：围绕既定维度（人机料法环等）追溯问题原因。
- 典型场景：质量异常、流程失效、销售困境。

### 树图/系统展开图
- 用途：将目标和问题分层展开，定位根因和分支路径。

## 2) 原因追踪与优先级

### 5W2H + 28问
- 用途：在 `人/时/地/因/事/法/费` 七维进行多层质询。
- 产出：定人、定时、定位、定原因、定事项、定方法、定耗费。

### 5WHY
- 用途：连续追问“为什么”直到触达制度或系统层根因。
- 提醒：第五层通常不是“个人问题”，多与机制和激励相关。

### ABC分析法（层别图）
- 用途：识别“关键少数”，聚焦最主要问题类别。
- 产出：不良率累计曲线、优先改善对象。

## 3) 计划与执行

### CPM / PERT（最佳路线分析）
- 用途：拆分工作包、识别关键路径、安排责任与节点。
- 产出：实施时序图、关键里程碑、责任人清单。

### PDCA
- 用途：将改善闭环固化到计划-执行-检查-评审。
- 产出：问题点、原因、主因、改善计划、行动、效果确认、标准化、遗留问题。

## 4) 战略与经营分析

### PESTEL
- 用途：评估政治/经济/社会/技术/环保/法律对行业影响。

### 波特五力
- 用途：判断行业竞争压力来源（新进入者、供应商、买方、替代品、现有竞争）。

### 麦肯锡7-S
- 用途：评估战略、结构、系统与人员、技能、作风、共同价值观的一致性。

### 内外部16要素
- 内部8：文化、决策、人力、产品、技术、组织、流程制度、硬件资源。
- 外部8：资本、营销、广宣、客户满意、信息系统、战略同盟、特殊资源、国际能力。

### 价值链
- 用途：识别增值环节与低效环节，连接战略与运营动作。

### 波士顿矩阵
- 用途：按市场增长率与相对份额划分明星/金牛/问题/瘦狗。

### GE矩阵
- 用途：基于市场吸引力与业务优势确定投资、选择、收缩策略。

### SWOT + BSC 组合
- 用途：把SWOT策略映射到BSC四维（财务、客户、内部流程、学习创新）。
- 产出：从战略语义到指标和行动项目的映射表。

## 5) 推荐组合打法

- 运营问题：`鱼骨图 -> 5WHY -> ABC -> PDCA`
- 战略复盘：`PESTEL -> 五力 -> SWOT -> BSC`
- 组织诊断：`7-S -> 内外部16要素 -> 价值链`
- 项目推进：`5W2H+28问 -> CPM -> PDCA`

## 6) 最小交付模板

1. 背景与目标（是什么）  
2. 原因机制（为什么）  
3. 方法链路（用哪些工具、为何这样组合）  
4. 关键发现（Top3）  
5. 90天行动计划（谁在何时做什么）  
6. 指标看板（财务/客户/流程/组织）
