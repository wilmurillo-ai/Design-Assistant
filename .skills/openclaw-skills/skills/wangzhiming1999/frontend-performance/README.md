# Frontend Performance（前端性能优化）

围绕加载性能与运行性能，给出可落地的优化方案与优先级。

## What It Does

- 分析维度：LCP/FCP/TTI、CLS/卡顿、JS 体积/图片/请求
- 现状量化与瓶颈定位（Network、Performance、bundle 分析）
- 优化方案表（收益、成本、优先级）
- 建议落地顺序与验证方式

## How to Use

当用户说「性能优化」「首屏慢」「页面卡顿」「打包体积大」「LCP/FCP 差」或提供 Lighthouse/Performance 数据时启用，输出《性能优化报告》。

## Requirements

- 无额外依赖；可选 Lighthouse、bundle analyzer 等工具辅助
