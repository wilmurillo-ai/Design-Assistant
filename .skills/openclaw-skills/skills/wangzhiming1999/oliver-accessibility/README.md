# Accessibility（无障碍 / a11y）

帮助实现符合 WCAG 的无障碍：语义化、ARIA、键盘与焦点、屏幕阅读器支持。

## What It Does

- 语义与结构、键盘与焦点、ARIA 与名称、视觉与对比 四维度检查
- 按优先级列问题并给出具体修改建议
- 输出可落地的《无障碍检查与改进报告》

## How to Use

当用户提到「无障碍」「可访问性」「a11y」「键盘导航」「读屏」「WCAG」或需要做 a11y 审计时启用。

## Requirements

- 无额外依赖；可选 axe DevTools、Lighthouse、WAVE 辅助验证
