# interactive-skills-platform-mvp

把“CLI skill 仅技术用户可用”的问题，转成“非技术用户可在 Web 上直接使用 skill”的 MVP 设计与实施方案。

## 适用场景

- 用户要做 skills 平台或 skill 网页化
- 用户要上传 `SKILL.md` 并自动生成交互界面
- 用户要对话式引导 + 动态 UI（进度条/链接/文本）

## 触发词（示例）

- `interactive skills platform`
- `skills 平台`
- `skill 网页化`
- `上传 SKILL.md 自动生成 UI`
- `对话式 skill 执行`

## 默认交付

1. 问题定义（WHO/SITUATION/PROBLEM/IMPACT）
2. MVP 架构设计（React + FastAPI + 文件存储 + Agent SDK）
3. API 草案（上传/分析/对话/执行/状态）
4. 分阶段 Checklist（Week 1-4）
5. 验证方案（至少 `felo-search` + `felo-slides`）
6. 后续迭代边界（安全、认证、市场）

- `SKILL.md`：可执行型工作流 + 完整设计稿参考
