# Frontend Testing（前端测试）

设计测试策略并编写单元/组件/集成/E2E 测试，处理 mock 与覆盖率。

## What It Does

- 测试分层与工具选型（Jest/Vitest、RTL、Playwright 等）
- 用例设计、Mock 策略、断言与可维护性
- 输出测试方案或示例代码

## How to Use

当用户提到「写测试」「单元测试」「组件测试」「E2E」「Jest」「Vitest」「Testing Library」或提高覆盖率时启用。

## Requirements

- 无额外依赖；按项目已有测试框架给出建议
