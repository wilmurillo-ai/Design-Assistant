# Bug Investigation（Bug 排查）

用可复现步骤、假设验证和工具链，快速定位并诊断前端问题根因。

## What It Does

- 明确现象与环境（操作路径、预期 vs 实际、浏览器/设备/网络）
- 复现与缩小范围（最小复现步骤、范围判断）
- 假设与验证（前端逻辑、接口、样式、时机、兼容性）
- 定位根因（具体文件与位置、一句话根因）
- 修复建议（最小改动方案、边界与测试建议）

## How to Use

在 Cursor/OpenClaw 中当用户说「排查 bug」「定位问题」「复现不了」「调试」或描述白屏、报错、样式错乱、请求失败等现象时，本 skill 会按流程输出结构化《Bug 排查报告》。

## Requirements

- 无额外依赖
- 适用于前端项目（React/Vue/Next 等）
