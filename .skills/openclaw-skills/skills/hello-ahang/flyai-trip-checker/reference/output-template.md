## 诊断状态标记

| 状态 | 符号 | 含义 |
|-----|------|------|
| 正常 | ✅ | 价格合理/安排合理 |
| 警告 | ⚠️🟡 | 存在风险/可优化 |
| 严重 | 🔴 | 必须修改 |
| 提示 | 🟢 | 无需担心 |

## 预订链接说明

FlyAI 返回数据中的 `jumpUrl` 字段是飞猪预订链接：

- **机票**：`search-flight` 返回的每个航班包含 `jumpUrl`
- **酒店**：`search-hotel` 返回的每个酒店包含 `jumpUrl`
- **景点**：`search-poi` 返回的每个景点包含 `jumpUrl`

输出格式：
```
👉 [立即预订机票](https://a.feizhu.com/xxxxx)
👉 [查看酒店详情](https://a.feizhu.com/xxxxx)
👉 [购买景点门票](https://a.feizhu.com/xxxxx)
```
