# 签证提醒规则

## 热门目的地签证办理时间

| 目的地 | 办理周期 | 备注 |
|--------|----------|------|
| 日本 | 5-7 个工作日 | 需提前准备材料 |
| 韩国 | 5 个工作日 | - |
| 泰国 | 落地签或提前办理 | 落地签可当场办理 |
| 欧洲申根国 | 10-15 个工作日 | 含预约面签时间 |
| 美国 | 1-3 个月 | 需面签 |
| 澳大利亚 | 10-20 个工作日 | 电子签 |

## 免签/落地签目的地（中国护照）

- **免签**: 新加坡、马来西亚、阿联酋、塞尔维亚等
- **落地签**: 泰国、马尔代夫、印度尼西亚等

## 提醒规则

当方案出发日期距现在不足签证办理时间时，标红提示：

> ⚠️ 签证办理约需X个工作日，该方案时间可能不足，建议优先考虑其他方案或加急办理。
