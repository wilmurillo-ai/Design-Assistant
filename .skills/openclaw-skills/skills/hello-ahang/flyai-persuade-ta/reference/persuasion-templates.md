# 说服模板

根据不同的说服对象和顾虑类型，选择对应的模板。

---

## 伴侣版模板

**语气**：温馨浪漫，带emoji
**重点**：性价比、特别体验、情感价值

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📩 一封给[老婆/老公/亲爱的]的旅行提案
   （可直接转发微信/截图发送）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

亲爱的[称呼]🌸

[特殊日期：结婚X周年/恋爱纪念日/生日]快乐！
我认真做了个功课，想带你去[目的地]，
你先别说[顾虑]——看完这份方案再决定👇

━━━━━━━━━━━━━━━

💰 关于「贵不贵」这件事

  你可能以为[目的地]要花[预期高价]？
  其实——

  ✈️ 机票：¥[价格]/人（往返含税！）
     [日期] [航班号] 直飞[时长]h
     👆 比国内飞三亚贵不了多少

  🏨 酒店：¥[价格]/晚 
     [酒店名]（[特色：泳池别墅/海景房]）
     👆 是的，[卖点描述]
        比国内很多酒店还便宜

  📍 景点：大部分门票不到¥[价格]
  🍜 餐饮：人均¥[金额]就能吃很好
  💆 [特色体验]：[价格描述]

  ┌────────────────────────────┐
  │ 💰 全部算下来：              │
  │ 两人总计 ¥[总价]            │
  │ 人均 ¥[人均]                │
  │                             │
  │ 👉 不到一个包的价格          │
  │    换一次[目的地]之旅        │
  │    我觉得很值！              │
  └────────────────────────────┘

━━━━━━━━━━━━━━━

🌴 关于「去了玩什么」

  D1 到达·[酒店/区域]泡泳池
  D2 [景点1]+[景点2]+[体验]
  D3 [活动描述]
  D4 [活动描述]
  D5 [特色体验] → 带着[收获]回家

  📍 已确认的景点（飞猪可查）：
  · [景点1] — [一句话亮点]
  · [景点2] — [一句话亮点]

━━━━━━━━━━━━━━━

📋 你可能还担心的：

  ❓ 安全吗？
  ✅ [安全说明]

  ❓ 签证麻烦吗？
  ✅ [签证说明，如：免签！护照直接飞！]

  ❓ 语言不通？
  ✅ [语言说明]

  ❓ [季节]去热/冷吗？
  ✅ [气候说明]

━━━━━━━━━━━━━━━

所以……[称呼]大人，批准吗？🥺

  [👍 批准！帮我们订吧]
  [🤔 再想想（但方案已保存）]
  [✏️ 我有其他顾虑（告诉AI继续完善）]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 老板版模板

**语气**：专业理性，简洁有力
**重点**：请假方案、工作交接、高效利用时间

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📋 请假申请说明
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[领导称呼]好：

计划于[日期区间]申请[N]天年假，现将行程安排和工作交接方案汇报如下：

━━━━━━━━━━━━━━━

📅 时间安排（最优拼假方案）

  ┌─────────────────────────────────┐
  │ 实际请假：[N]天                  │
  │ 加上周末：共[M]天自由时间         │
  │                                  │
  │ [周五] → 下班后出发               │
  │ [周六-周日] → 周末（不占年假）     │
  │ [周一-周二] → 请假2天             │
  │ [周三] → 正常上班                 │
  │                                  │
  │ 👉 只请2天假，玩满5天             │
  └─────────────────────────────────┘

━━━━━━━━━━━━━━━

✈️ 往返时间

  去程：[日期] [航班] [时间起飞]
       → 不影响周五下午工作
  
  回程：[日期] [航班] [时间到达]
       → [周三]正常到岗

━━━━━━━━━━━━━━━

📋 工作交接安排

  │ 工作内容          │ 交接人    │ 状态   │
  │─────────────────│──────────│───────│
  │ [项目/任务1]      │ [同事A]   │ 已沟通  │
  │ [项目/任务2]      │ [同事B]   │ 已沟通  │
  │ [日常工作]        │ [同事C]   │ 已沟通  │

  紧急联系：假期期间如有紧急事项，可通过微信联系
  
━━━━━━━━━━━━━━━

请假期间：
· 微信保持在线，重要消息当日回复
· 已完成手头紧急工作
· 邮件已设置自动回复

烦请批准，感谢理解！

[你的名字]
[日期]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 爸妈版模板

**语气**：温暖贴心，孝顺体贴
**重点**：省钱、安全、对身体好

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📩 给爸妈的旅行提案
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

亲爱的爸/妈：

想带你们去[目的地]玩几天，
这次我都安排好了，你们就负责玩就行～

━━━━━━━━━━━━━━━

💰 关于花钱这事儿

  我知道你们会担心花钱，但真的没多少：

  ✈️ 机票：¥[价格]/人
     → 比坐高铁也贵不了多少
     → 直飞[时长]小时就到，不累

  🏨 酒店：¥[价格]/晚
     → 都是正规大酒店，卫生有保障
     → 有电梯，不用爬楼梯

  ┌────────────────────────────┐
  │ 全程费用（我来出）：         │
  │ [人数]人合计 ¥[总价]         │
  │                             │
  │ 这钱我早就攒好了，            │
  │ 你们就当帮我个忙，一起出去散散心│
  └────────────────────────────┘

━━━━━━━━━━━━━━━

🌴 行程安排（都是轻松的）

  D1 到达·入住酒店休息
     → 不赶行程，先休息好
  
  D2 [景点]·轻松游览
     → 都是平路，不用爬山
     → 中午回酒店休息，下午再出去
  
  D3 [活动]
     → [对老人友好的描述]
  
  D4 自由活动·返程
     → 睡到自然醒，不着急

━━━━━━━━━━━━━━━

📋 你们可能担心的：

  ❓ 安全吗？
  ✅ [目的地]是[成熟度描述]，
     治安很好，很多中国游客

  ❓ 语言不通怎么办？
  ✅ 我全程陪着，而且[语言说明]

  ❓ 吃得惯吗？
  ✅ [饮食说明]，还有中餐厅

  ❓ 身体吃得消吗？
  ✅ 行程都安排得很松，随时可以回酒店休息
     [气候说明]，对身体有好处

━━━━━━━━━━━━━━━

我平时工作忙，没时间陪你们，
趁这次机会，咱们好好出去玩一趟～

等你们回复，定好日期就买票！

爱你们的[你的名字]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 朋友版模板

**语气**：轻松活泼，适度怂恿
**重点**：AA费用、FOMO制造

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🎉 [目的地]召集令！
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

@[朋友名字们] 看这里！！

[日期]，走不走？

━━━━━━━━━━━━━━━

💰 费用透明（AA下来超便宜）

  ✈️ 机票：¥[价格]/人
  🏨 酒店：¥[价格]/人（[晚数]晚）
  📍 门票+交通：约¥[价格]/人
  🍜 吃喝：约¥[价格]/人

  ┌────────────────────────────┐
  │ 📊 人均总花费：¥[总价]       │
  │                             │
  │ 👉 一顿火锅钱 = 一次[目的地]   │
  │    不去血亏！               │
  └────────────────────────────┘

━━━━━━━━━━━━━━━

🔥 这次必须去的理由

  1️⃣ [理由1：如价格便宜/季节好]
  2️⃣ [理由2：如特别体验]
  3️⃣ [理由3：如FOMO制造]

━━━━━━━━━━━━━━━

📅 行程安排（已做好功课）

  D1 [简要行程]
  D2 [简要行程]
  D3 [简要行程]

━━━━━━━━━━━━━━━

⏰ 决策DDL

  机票现在这个价，过几天可能就涨了
  [日期]之前确认，我统一订
  
  接龙：
  1️⃣ [发起人]：去！
  2️⃣ 
  3️⃣ 
  4️⃣ 

━━━━━━━━━━━━━━━

🤷‍♂️ 不去的理由我帮你想好了：

  ❌ "没时间" → 只要请[N]天假
  ❌ "没钱" → 人均才[金额]，少吃两顿火锅
  ❌ "下次吧" → 下次永远没有

所以，别废话了，去不去？

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

- 示例：`说服老婆-巴厘岛-提案.html`
