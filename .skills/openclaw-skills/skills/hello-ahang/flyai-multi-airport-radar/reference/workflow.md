# 工作流程

### 前置步骤：安装/升级 FlyAI CLI

在执行任何搜索之前，**必须先执行安装命令**（无论是否已安装，确保为最新版本）：

```bash
npm install -g @fly-ai/flyai-cli@latest --registry=https://registry.npmjs.org
```

> 💡 此命令会自动处理首次安装和版本升级，无需手动判断是否已安装。**安装后验证：**
```bash
flyai --help
```

**安装失败处理：**
| 情况 | 处理方式 |
|-----|---------|
| npm 未安装 | 提示用户先安装 Node.js (https://nodejs.org/) |
| 权限不足 | 建议使用 `sudo npm install -g @fly-ai/flyai-cli@latest --registry=https://registry.npmjs.org` 或使用 nvm 管理 Node |
| 网络问题 | 建议用户检查网络或使用国内镜像 `npm config set registry https://registry.npmmirror.com` |

**注意：** 此步骤只在首次使用时执行，后续调用会直接跳过已安装的情况。

### 第1步：收集用户信息

使用 `ask_user_question` 工具收集必要信息。

**必须收集的信息**：
1. **大致出发区域** - 如"上海"或"长三角"
2. **目的地** - 如"大阪"、"三亚"
3. **弹性日期范围** - 如"8月10日-20日之间出发"
4. **行程天数** - 如"5天"

**可选收集的信息**：
5. **同行人数** - 影响总省钱金额
6. **筛选条件** - 只看直飞、舱位偏好等

### 第2步：确定搜索范围

**2.1 扩展出发机场列表**

根据用户的出发区域，自动扩展到周边城市机场：

| 中心城市 | 扩展搜索机场 | 跨城交通参考 |
|---------|-------------|-------------|
| 上海 | 浦东、虹桥、杭州萧山、宁波栎社、南京禄口、无锡硕放 | 高铁 1-2h，¥50-150 |
| 北京 | 首都、大兴、天津滨海、石家庄正定 | 高铁 0.5-2h，¥50-200 |
| 广州 | 白云、深圳宝安、珠海金湾、香港、澳门 | 高铁/跨境巴士 1-2h |
| 成都 | 双流、天府、重庆江北 | 高铁 1.5h，¥150 |

**2.2 确定日期扫描范围**

根据弹性范围，生成出发日期列表：
- 固定日期：只搜该天
- 一周内：每天都搜
- 两周内：隔天采样，找到低价区间后细搜
- 一个月内：每周采样3天（周二、周四、周六），再细化

---

### 第3步：执行矩阵搜索

**3.1 批量调用机票搜索**

对每个 (出发机场 × 出发日期) 组合调用：

```bash
NODE_TLS_REJECT_UNAUTHORIZED=0 flyai search-flight \
  --origin "{出发机场}" \
  --destination "{目的地}" \
  --dep-date {出发日期} \
  --back-date {返回日期} \
  --sort-type 3
```

**参数说明**：
| 参数 | 说明 |
|------|------|
| `--origin` | 出发城市/机场 |
| `--destination` | 目的地城市 |
| `--dep-date` | 出发日期 (YYYY-MM-DD) |
| `--back-date` | 返回日期 (YYYY-MM-DD) |
| `--journey-type` | 1=只看直飞，不填=含中转 |
| `--sort-type` | 3=价格从低到高 |

**3.2 并行优化策略**

搜索量大时（>20次），采用分批策略：
1. 先搜索主要机场（如上海浦东、虹桥）
2. 找到大致价格区间
3. 对最可能出现低价的组合再细搜

---

### 第4步：计算综合成本

**4.1 跨城交通成本数据库**

| 出发地 | 到机场 | 交通方式 | 单程价格 | 单程时间 |
|-------|-------|---------|---------|---------|
| 上海 | 杭州萧山 | 高铁 | ¥73 | 1h |
| 上海 | 宁波栎社 | 高铁 | ¥71 | 1.5h |
| 上海 | 南京禄口 | 高铁 | ¥135 | 2h |
| 北京 | 天津滨海 | 高铁 | ¥55 | 0.5h |
| 北京 | 石家庄正定 | 高铁 | ¥128 | 1.5h |
| 广州 | 深圳宝安 | 高铁 | ¥75 | 0.5h |
| 成都 | 重庆江北 | 高铁 | ¥154 | 1.5h |

**4.2 综合成本计算**

```
综合成本 = 机票往返价格 + 跨城高铁往返 + (需提前一晚住宿 ? 住宿费 : 0)
```

**是否需要前一晚住宿判断**：
- 早班机（8:00前起飞）+ 跨城交通 > 2h → 需要前一晚住宿
- 住宿预估按 ¥200-300/晚计算
