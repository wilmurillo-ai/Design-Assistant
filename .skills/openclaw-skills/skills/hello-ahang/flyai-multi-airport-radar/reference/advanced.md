# 进阶功能

### 到达侧多机场比价

如果目的地也有多个机场（如东京成田/羽田），同时展示：

| 到达机场 | 最低价 | 机场优势 |
|---------|-------|---------|
| 大阪关西 | ¥1,380 | 直达市区便捷 |
| 大阪伊丹 | ¥1,520 | 近市区但航班少 |
| 神户 | ¥1,280 | 可能更便宜但需转车 |

### 与其他 Skill 联动

- **搜完机票** → 提示"要顺便看看酒店价格日历吗？"（调用 flyai-flight-calendar）
- **选定方案** → 提示"要生成完整行程吗？"（调用 flyai-travel-chat）
- **预算敏感** → 提示"要看看预算内还能去哪？"（调用 flyai-reverse-budget）
