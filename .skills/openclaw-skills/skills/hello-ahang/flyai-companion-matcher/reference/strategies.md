# 调和方案策略库

### 节奏差异调和

| 组合 | 调和方案 |
|-----|---------|
| 暴走 + 躺平 | "上午自由·下午同行"模式 — 早起的人上午独自打卡，下午汇合轻松逛 |
| 暴走 + 中等 | 每天核心景点2-3个，其余时间弹性 |
| 躺平 + 中等 | 下午开始活动，晚上可以有夜生活 |

### 住宿差异调和

| 组合 | 调和方案 |
|-----|---------|
| 便宜 + 格调 | 选设计感强但价格适中的酒店（如4星设计酒店）|
| 便宜 + 格调 | 部分天数住经济型 + 部分天数住体验型 |
| 适中 + 高端 | 选5星酒店的促销价或套餐 |

### 拍照差异调和

| 组合 | 调和方案 |
|-----|---------|
| 快拍 + 出片 | 提前标注"出片点"和"打卡点"，出片点预留拍照时间 |
| 快拍 + 出片 | 出片时间不拍照的人可以自由探索周边 |

### 消费差异调和

| 组合 | 调和方案 |
|-----|---------|
| 省钱 + 享受 | 各带独立预算，AA部分事先约定清楚 |
| 省钱 + 享受 | 餐厅轮流做主：Day1省钱派选、Day2享受派选 |
