# 对话示例

## 示例1: 从模糊到具体的完整流程

### 用户输入
> 8月想带女朋友去个不那么热的地方，最好有山有水，预算别太高

### AI 执行步骤
1. 解析：8月 + 情侣 + 凉爽 + 山水 + 性价比
2. 调用 fliggy-fast-search 搜索候选目的地
3. 输出3个方向推荐
4. 用户选择后，搜索机票、酒店、景点
5. 输出完整方案

### AI 输出（第1轮）
```
8月凉爽+有山有水+适合情侣，帮你锁定了3个方向：

🏔️ 方向A：贵州（荔波小七孔+黄果树）
   8月均温22-26°C | 山水绝美 | 人均3,000左右
   
🏔️ 方向B：云南大理+丽江
   8月均温18-25°C | 经典情侣线 | 人均3,500左右
   
🏔️ 方向C：四川甘孜（稻城亚丁）
   8月均温10-20°C | 震撼级风光 | 人均4,000左右
   ⚠️ 有高反风险，需要一定体力

你对哪个更感兴趣？
```

### 用户选择
> 贵州听起来不错，我们从上海出发，大概玩5天

```

   安顺1晚：黄果树瀑布旁xx酒店 ¥320/晚

   荔波小七孔 | 大七孔 | 西江千户苗寨 | 黄果树瀑布 | 天星桥
```
