# 工作流程

### 前置步骤：安装/升级 FlyAI CLI

在执行任何搜索之前，**必须先执行安装命令**（无论是否已安装，确保为最新版本）：

```bash
npm install -g @fly-ai/flyai-cli@latest --registry=https://registry.npmjs.org
```

> 💡 此命令会自动处理首次安装和版本升级，无需手动判断是否已安装。**安装后验证：**
```bash
flyai --help
```

**安装失败处理：**
| 情况 | 处理方式 |
|-----|---------|
| npm 未安装 | 提示用户先安装 Node.js (https://nodejs.org/) |
| 权限不足 | 建议使用 `sudo npm install -g @fly-ai/flyai-cli@latest --registry=https://registry.npmjs.org` 或使用 nvm 管理 Node |
| 网络问题 | 建议用户检查网络或使用国内镜像 `npm config set registry https://registry.npmmirror.com` |

**注意：** 此步骤只在首次使用时执行，后续调用会直接跳过已安装的情况。

### 第1步：使用 ask_user_question 收集用户信息

**必须使用 `ask_user_question` 工具**来交互式收集信息，让用户体验更好：

**必须收集的信息：**
1. **出发地** - 用户从哪个城市出发
2. **目的地** - 最终要去哪里
3. **出行日期** - 什么时候出发

**可选收集的信息：**
4. **偏好** - 最便宜 / 中转时间最有趣 / 最短总时长
5. **中转城市偏好** - 想在哪些城市中转

**提问示例：**

```
问题1: "你这次行程是从哪里出发，要去哪里？"
选项:
- "上海 → 伊斯坦布尔"
- "北京 → 巴黎"
- "广州 → 洛杉矶"
- "其他（请输入）"

问题2: "你什么时候出发？"
选项:
- "最近7天内"
- "本月"
- "我指定具体日期"

问题3: "你的优先偏好是什么？"
选项:
- "💰 最省钱 - 价格越低越好"
- "🎢 中转最有趣 - 能顺便逛个城市"
- "⏱ 总时长最短 - 尽快到达"
- "🎁 帮我综合推荐"

问题4: "有心仪的中转城市吗？"（可选）
选项:
- "东南亚（曼谷、新加坡、吉隆坡等）"
- "中东（迪拜、多哈、阿布扎比等）"
- "东亚（首尔、东京、香港等）"
- "都可以，帮我找最划算的"
```

### 第2步：调用 FlyAI 搜索航班

**重要：SSL 证书验证问题处理**
如果遇到 "SSL 证书验证失败" 或类似 `UNABLE_TO_VERIFY_LEAF_SIGNATURE` 错误，需要在命令前加上环境变量：
```bash
NODE_TLS_REJECT_UNAUTHORIZED=0 flyai <command>
```

**2.1 搜索直飞航班（作为对比基准）**
```bash
flyai search-flight --origin "[出发城市]" --destination "[目的地]" --dep-date [日期] --journey-type 1 --sort-type 3
```

**2.2 搜索中转航班**
```bash
flyai search-flight --origin "[出发城市]" --destination "[目的地]" --dep-date [日期] --journey-type 2 --sort-type 3
```

**2.3 筛选逻辑**
从返回的中转航班中，筛选出中转时间在 **6-15小时** 之间的航班：
- < 6小时：时间太紧，不推荐出机场
- 6-15小时：**黄金中转时间**，可以出机场逛半天
- > 15小时：可能需要住一晚，另外处理

### 第3步：搜索中转城市 POI

对每个符合条件的中转城市：
```bash
flyai search-poi --city-name "[中转城市]" --poi-level 5
```

如果5级景点较少，降级搜索4级景点：
```bash
flyai search-poi --city-name "[中转城市]" --poi-level 4
```

### 第4步：计算时间安全垫

**时间安全垫计算公式：**

```
可用时间 = 中转总时长 - 出关时间(45min) - 往返交通时间 - 入关安检时间(2h) - 缓冲时间(30min)
```

**各城市参考数据：**
| 中转城市 | 机场到市区 | 出关时间 | 建议提前到机场 |
|---------|-----------|---------|--------------|
| 曼谷(BKK) | 40min | 30-45min | 3h |
| 新加坡(SIN) | 30min | 20-30min | 2.5h |
| 吉隆坡(KUL) | 50min | 30-40min | 3h |
| 迪拜(DXB) | 25min | 20-30min | 2.5h |
| 多哈(DOH) | 20min | 15-25min | 2.5h |
| 首尔(ICN) | 60min | 30-40min | 3h |
| 东京(NRT) | 70min | 30-45min | 3h |
| 香港(HKG) | 30min | 20-30min | 2.5h |

**安全标准：**
- ✅ 安全余量 ≥ 3h：非常充裕
- ⚠️ 安全余量 2-3h：可以但要抓紧
- ❌ 安全余量 < 2h：风险较大，不推荐出机场

### 第6步：搜索目的地酒店和景点门票

为了提供完整的预订体验，需要同时搜索目的地的酒店和景点门票：

**6.1 搜索目的地酒店**
```bash
flyai search-hotel --dest-name "[目的地]" --check-in-date [入住日期] --check-out-date [离店日期] --sort rate_desc
```

**6.2 搜索目的地景点门票**
```bash
flyai keyword-search --query "[目的地]景点门票"
```

**6.3 搜索中转城市景点门票**
```bash
flyai keyword-search --query "[中转城市]景点门票"
```

**重要：** FlyAI返回的数据中包含 `jumpUrl` 或 `detailUrl` 字段，这是飞猪预订链接，必须在输出中展示为可点击的预订按钮。
