## 解析结果模板

当完成攻略内容解析后，输出以下格式：

```
📖 已解析攻略内容：
━━━━━━━━━━━━━━━━━━━━━━━━━━━
原作者：@[作者名]  发布于[发布时间]
标题：「[攻略标题]」

提取到的关键信息：
  📍 目的地：[目的地1] + [目的地2]
  📅 原行程：[N]天[M]晚
  🏨 提到的住宿：
     - [酒店名1]（[位置/特色描述]）
     - [酒店名2]（[位置/特色描述]）
  📍 提到的景点/活动：
     - [景点1]
     - [景点2]
     - [景点3]
     ...
  🍜 提到的餐厅：
     - [餐厅1]
     - [餐厅2]
  🚗 提到的交通方式：
     - [交通说明]

⚠️ 发现[N]个需要调整的地方：
  ① [差异1：如原攻略6天，你只有5天]
  ② [差异2：如原作者从成都出发，你从上海出发]
  ③ [差异3：如原攻略发布于5月，信息可能过时]
━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔄 正在用飞猪数据验证和填充...
```

## 验证状态标识

对每个信息节点使用以下状态标识：

| 标识 | 含义 | 使用场景 |
|------|------|----------|
| ✅ | 可用 | 找到匹配数据，可直接预订 |
| ⚠️ | 需确认 | 信息可能过时，建议用户自行确认 |
| ❌ | 不可用 | 酒店满房/景点关闭等，需要替代方案 |
| ℹ️ | 原文保留 | 飞猪暂无数据（如餐厅），保留原推荐 |

**使用示例**：

```
🏨 入住
┌──────────────────────────────────────────┐
│ 原攻略推荐：云隐山房（洱海无边泳池）        │
│                                           │
│ 🔍 飞猪搜索结果：                          │
│ ✅ 有房！¥580/晚 洱海大床房（含早）         │
│ ⭐ 4.8分 · 与原攻略描述一致                 │
│ 📍 位置：洱海边磻溪村，离双廊车程20min      │
│ 👉 [查看详情并预订]                         │
└──────────────────────────────────────────┘

📍 苍山感通索道
   → search-poi 验证结果：
   ✅ 苍山感通索道 · 门票+索道 ¥115/人
   ⏰ 开放时间：08:30-17:00
   💡 原攻略提醒：山上冷，带件外套
   ⚠️ 8月雨季可能云雾大，看运气

🍜 晚餐
┌──────────────────────────────────────────┐
│ 原攻略推荐：段公子·再回大理                 │
│ ℹ️ 飞猪暂无该餐厅数据                      │
│ 💡 保留原攻略推荐，建议到店前大众点评查排队  │
└──────────────────────────────────────────┘
```

│                                           │

│    → 实际：[实际情况]                       │
│    → 建议：[处理建议]                       │

## 费用总览模板

```
💰 费用总览（[X]人合计）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
┌─────────────┬──────────┬───────────────────┐
│ 项目         │ 费用     │ 备注               │
├─────────────┼──────────┼───────────────────┤
│ 机票往返×[N] │ ¥[金额]  │ [航班说明]         │
│ 住宿[N]晚    │ ¥[金额]  │ [酒店]¥[单价]/晚   │
│ 景点门票×[N] │ ¥[金额]  │ [景点列表]         │
│ 城际交通      │ ¥[金额]  │ [如：大理→丽江动车] │
│ 当地交通      │ ¥[金额]  │ 打车+租电瓶车等    │
│ 餐饮预估      │ ¥[金额]  │ [N]天             │
├─────────────┼──────────┼───────────────────┤
│ 📊 合计      │ ¥[总额]  │ 人均 ¥[人均]       │
└─────────────┴──────────┴───────────────────┘

[预算对比说明]
```

**预算对比说明示例**：

```
⚠️ 人均略超预算（¥5,000），可调整：
  → 住宿换替代：省 ¥450（大理换¥320那家+丽江换¥350那家）
  → 调整后人均：¥4,376 ✅ 预算内
  👉 [一键切换为省钱版住宿]

✅ 人均¥4,200，在预算¥5,000内，还有¥800余量
  💡 可以考虑升级酒店或多加一个景点
```

## 差异对比模板

```
📋 与原攻略的差异说明
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
┌───┬────────────────┬────────────────────────┐
│ # │ 原攻略          │ 你的版本               │
├───┼────────────────┼────────────────────────┤
│ 1 │ [原内容]        │ [调整内容]             │
│   │                 │ [调整原因]             │
├───┼────────────────┼────────────────────────┤
│ 2 │ [原内容]        │ [调整内容]             │
│   │                 │ [调整原因]             │
├───┼────────────────┼────────────────────────┤
│ 3 │ [原内容]        │ [调整内容]             │
│   │                 │ [调整原因]             │
└───┴────────────────┴────────────────────────┘
```

**常见差异类型**：

| 差异类型 | 原攻略 | 你的版本 |
|---------|-------|---------|
| 天数压缩 | 6天5晚 | 压缩为5天4晚<br>去掉了原Day3的"发呆日" |
| 出发城市 | 成都出发<br>往返¥600 | 改为上海出发<br>往返¥2,060 |
| 路线优化 | 洱海全程环游<br>（电瓶车一整天） | 改为半程（喜洲-双廊）<br>更现实，精华都覆盖了 |
| 信息补充 | 未提到预约问题 | 补充了雪山大索道<br>必须提前预约的信息 |
| 数据保留 | 推荐了3家餐厅 | 餐厅信息保留原推荐<br>（飞猪暂无餐饮数据） |

  [✈️ 预订回程机票 [航班号] ¥[价格]×[人数]]
  [🏨 预订[酒店名] [日期区间] [N]晚]

  [💰 帮我切换成省钱版]
