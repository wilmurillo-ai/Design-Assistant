# 使用示例

## 场景：一家四口成都 4 天游

**用户输入**：
> 我想带老婆、4岁女儿和65岁老妈去成都玩4天，帮我推荐适合的景点

**AI执行**：
1. 解析同行人画像：成人×2 + 幼儿4岁 + 老人65岁
2. 确定筛选原则：低体力消耗 | 幼儿友好 | 无障碍优先
3. 调用 search-poi 获取成都景点
4. 对每个景点做适配评分
5. 分类输出推荐 + 分日建议
