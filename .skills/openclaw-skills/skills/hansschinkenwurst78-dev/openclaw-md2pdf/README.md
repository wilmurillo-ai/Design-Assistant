# md2pdf

Markdown to PDF converter for OpenClaw agents. Uses Pandoc + Typst for professional output.

## Install

```bash
npx clawhub@latest install md2pdf
```

## Requirements

- Pandoc v3.2+ (`brew install pandoc`)
- Typst v0.12+ (`brew install typst`)

## Usage

```bash
bash ~/.openclaw/workspace/skills/md2pdf/scripts/md2pdf.sh input.md
```

PDF is created next to the source file.
