# Changelog

## [1.0.0] - 2026-03-09

### 初始发布
- 实现地址智能分类功能，支持四类分类：本省本市、本省外市、外省、地址不全
- 支持贵州省及全国其他省份地址识别
- 内置搜索补全功能，可识别不完整地址（如乡镇村名）
- 支持多种输出格式：TXT、CSV、JSON
- 提供完整的行政区划映射配置（config.yaml）
- 包含详细注释的生产级Python代码
- 完整的异常处理和日志系统
- 支持自定义目标省份和城市

### 功能特性
- 自动识别贵阳市主城区及下辖县市（开阳、息烽、修文、清镇）
- 自动识别贵州省内其他8个市州（毕节、遵义、安顺等）
- 智能补全5种常见不完整地址（西河镇、善广乡、宝龙镇等）
- 地址完整性检测机制
- 支持批量文件处理和单条地址分类
