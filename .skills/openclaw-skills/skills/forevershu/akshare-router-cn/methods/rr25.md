# Methods: RR25 (25-delta Risk Reversal) — MVP

RR25 定义（常见约定）：
- **RR25 = IV(Call, +0.25Δ) − IV(Put, −0.25Δ)**

一期实现策略：
- 不做连续插值/拟合 smile；用“最接近 ±0.25 delta 的合约”做近似。

## 输入
- underlying（例如 510050）
- 到期月份 trade_date（例如 202603）
- 当天 greeks/iv 数据：
  - 先通过 `ak.option_sse_codes_sina(看涨/看跌, trade_date, underlying)` 拿期权代码列表
  - 对每个代码调用 `ak.option_sse_greeks_sina(code)` 抽取 `delta` 与 `iv`

## 算法
1) 取看涨合约集合 C：保留 delta ∈ (0,1)
2) 取看跌合约集合 P：保留 delta ∈ (-1,0)
3) 找 C 中 |delta-0.25| 最小者，得到 iv_call_25
4) 找 P 中 |delta+0.25| 最小者，得到 iv_put_25
5) RR25 = iv_call_25 - iv_put_25

## 输出
- rr25
- 选中的 call/put 合约（sina code + trade_code + strike + delta + iv）

## 风险/局限
- delta 离散且受标的价格/利率/分红假设影响；不同源的 Delta 口径可能不同。
- 深度虚值时 IV/Delta 波动大，选取可能不稳。
