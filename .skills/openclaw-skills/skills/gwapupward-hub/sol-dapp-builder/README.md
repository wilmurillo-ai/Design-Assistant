# Solana dApp Builder

A ClawHub/OpenClaw skill for founders and developers building real Solana applications.

## What this skill does

This skill converts rough ideas into concrete build plans for Solana dApps. It helps define the product loop, decide what belongs on-chain, design accounts and PDAs, plan backend and indexing services, recommend practical tooling, and identify the exact keys and secrets needed to ship.

## What makes this version publishable

This package includes meaningful skill instructions instead of placeholder text. It contains:
- a detailed workflow for taking a dApp from idea to architecture
- Solana-specific modeling rules
- key and secret detection logic
- CLI and toolchain guidance
- security and cost checklists
- output structure that produces developer-ready deliverables
- examples and test prompts for validation before publishing

## Best use cases

- new Solana product architecture
- MVP scoping
- Anchor program design
- wallet and transaction UX planning
- backend/indexer design
- repo and monorepo planning
- security review and tradeoff analysis

## Included files

- `SKILL.md` — main skill definition and behavior
- `README.md` — overview and publishing context
- `examples.md` — concrete usage examples
- `test_prompts.md` — prompts to validate quality
- `publish_notes.md` — recommended marketplace listing details

## Publishing notes

If ClawHub rejects a skill as thin or templated, it usually means the main skill file is too generic, too short, or clearly placeholder content. This package fixes that by making the skill self-contained, specific, and operationally useful.
