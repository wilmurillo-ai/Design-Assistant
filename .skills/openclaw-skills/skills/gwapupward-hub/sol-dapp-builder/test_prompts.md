# Test prompts

Use these before publishing the skill.

## Product discovery quality
- I have an idea for a loyalty app on Solana. Tell me whether Solana is actually necessary and define the MVP.
- I want to build a social reputation product on Solana. Separate on-chain state from off-chain state.

## Architecture quality
- Design a Solana escrow app with PDAs, release logic, dispute handling, backend routes, and indexing.
- Compare a single-program design vs multi-program design for a naming service.

## Keys and secrets behavior
- Tell me exactly which wallets, API keys, and secrets I need for a Solana dApp that uses Helius webhooks and Postgres.
- For a local-only MVP, tell me which keys are not needed yet.

## Tooling behavior
- Recommend the best CLI setup for building a Solana dApp today.
- Give me a repo structure for Anchor + Next.js + NestJS + Postgres.

## Constraint handling
- Keep this Solana dApp as centralized as possible for MVP while preserving the core trust boundary.
- Tell me which parts of my design should not be on-chain.
