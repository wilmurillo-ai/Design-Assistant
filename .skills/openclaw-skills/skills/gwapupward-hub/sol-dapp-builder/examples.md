# Example prompts for Solana dApp Builder

## 1. Escrow marketplace
Build a Solana escrow marketplace for freelancers and clients. I need the on-chain program design, backend services, dispute flow, keys required, and a monorepo structure.

## 2. Naming protocol
Design a Solana naming service with namespaces, resolver records, reverse lookup, and renewals. Keep MVP lean and explain what should stay off-chain.

## 3. Trust scoring protocol
I want a GwapScore-style reputation protocol that reads wallet activity and selected app events, calculates scores off-chain, and exposes a public score lookup page. Tell me the exact keys and APIs required.

## 4. Existing repo review
I already have an Anchor repo and a Next.js frontend. Review the architecture, find the weak points, and tell me what to fix first without rewriting the whole project.

## 5. Repo scaffold
Scaffold an Anchor + Next.js + NestJS monorepo for a Solana dApp that uses Postgres and BullMQ. Include environment variables, local setup steps, and deployment notes.
