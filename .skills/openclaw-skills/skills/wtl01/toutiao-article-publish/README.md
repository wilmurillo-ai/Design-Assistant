# 今日头条自动化发布 Skill

基于 browser-cdp 的今日头条（头条号）全链路自动发布技能。

## 功能特性

- ✅ **CDP 直连浏览器**：使用 browser-cdp skill 直连 Chrome/Edge/QQ浏览器
- ✅ **登录态复用**：复用用户浏览器登录态，无需重复登录
- ✅ **双内容形式**：支持微头条和头条文章两种发布形式
- ✅ **智能配图**：AI 自动判断是否配图，支持图片上传
- ✅ **定时发布**：支持定时任务，实现自动化运营
- ✅ **完整流程**：选题 → 写作 → 配图 → 发布全流程覆盖

## 快速开始

### 1. 安装依赖

确保已安装 browser-cdp skill：

```bash
skillhub_install install_skill browser-cdp
```

## 配置说明

### 浏览器配置
- 默认使用 Chrome 浏览器
- 支持 Edge、QQ浏览器
- 复用用户 profile 保持登录态

### 发布配置
- 微头条：150-250字，可选配图
- 文章：500-800字，必配图
- 话题标签：自动添加 `#话题#`

### 定时任务
- 微头条：每30分钟（7:00-次日1:00）
- 文章：每2小时（8:00-22:00）

## 注意事项

1. **首次使用需要授权**：CDP 连接需要用户授权
2. **登录处理**：遇到登录页面需手动登录
3. **发布频率**：避免短时间大量发布
4. **内容审核**：遵守平台内容规范
5. **图片版权**：确保图片有合法使用权
