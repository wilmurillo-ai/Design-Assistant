# Miraflow OpenClaw Skill

An [OpenClaw](https://openclaw.ai) skill for creating and managing AI videos and images using the [Miraflow](https://miraflow.ai) API.

## Features

- 🎬 Create avatar videos with AI voices and custom scripts
- 🖼️ Generate and edit AI images
- 📋 List available avatars and voices
- 📊 Check video generation status
- ⬇️ Download completed videos

## Requirements

- [OpenClaw](https://openclaw.ai) installed and running
- `MIRAFLOW_API_KEY` environment variable set (get yours at [miraflow.ai/apikeys](https://miraflow.ai/apikeys))

## Installation

```bash
npx clawhub@latest install miraflow
```

Or clone this repo into your OpenClaw skills directory:

```bash
git clone https://github.com/minlabs/openclaw-skill-miraflow ~/.openclaw/workspace/skills/miraflow
```

## Usage

Once installed, just ask OpenClaw naturally:

- "Create a video with avatar Aerin saying hello world"
- "List my available avatars"
- "Check the status of job cm42ab3..."
- "Generate an AI image of a sunset over the ocean"
- "Download my latest video"
