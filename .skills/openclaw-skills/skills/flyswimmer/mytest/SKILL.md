# 高中地理提分技能

## Overview
本技能专为高中地理学习提分设计，基于教育部课标、各版本官方教材、历年高考真题资源，提供精准、专业的地理学习辅助功能，覆盖日常答疑、预习复习、工具查询、错题管理、提分训练全学习场景，内容权威准确，杜绝通用大模型的超纲/错误内容。

## 核心能力

### 一、全量无门槛免费功能（永久开放，无次数/范围限制）
#### 1. 多模态基础精准答疑
- 支持文字、拍照、语音三种提问方式，覆盖知识点概念、教材原文、基础题目三类核心需求
- 知识点答疑：精准解释地理概念、原理、规律、课标要求，标注对应教材章节
- 题目基础解析：提供客观题正确答案、主观题核心答题要点，对应考点定位，以及标准的基础解题步骤
- 付费边界：不开放选项逐一错因拆解、秒杀技巧、命题陷阱分析、名师答题思路、同类题推荐等深度内容

#### 2. 教材 & 大纲同步基础查询
- 免费开放全国各版本高中地理教材（人教版、湘教版、鲁教版等）的章节框架、核心知识点目录、课时基础学习目标
- 免费开放教育部高中地理教学大纲、高考课标要求的必考/选考范围、考点层级划分
- 付费边界：不开放教师用书深度解读、章节重难点拆解、教材配套的拓展例题、同步专项训练等内容

#### 3. 高中地理基础工具包
- 高考地理必备基础公式大全（时间计算、正午太阳高度、比例尺、人口自然增长率等）
- 核心地理图表基础判读规则（等值线、气候类型图、洋流分布图、人口金字塔图等入门规则）
- 中国/世界地理基础速记口诀、核心分界线汇总、高频地理名词解释

#### 4. 基础错题收录功能
- 支持用户一键收录提问过的题目、手动添加错题，自动标注题目对应知识点，支持按知识点/提问时间分类查看
- 付费边界：不开放错题错因深度诊断、薄弱点定位、错题变式训练、针对性补弱计划等内容

### 二、限量体验型免费功能（种草引流，引导付费转化）
#### 1. 高考真题 / 模拟题免费体验库
- 开放近3年高考全国卷真题的选择题基础解析，每个核心知识模块（自然地理、人文地理等）限免5道经典高考真题，配套基础解题步骤
- 付费边界：不开放地方卷真题、全量模拟题库、逐题命题意图拆解、真题变式训练、高频考点规律总结等内容

#### 2. 题型提分技巧免费体验包
- 开放选择题、综合题的通用基础答题规范、入门级通用技巧（比如综合题材料关键词定位法、选择题基础排除法）
- 每个题型限免1道经典例题的技巧演示，每个核心模块限免1个基础答题框架
- 付费边界：不开放独家秒杀技巧、完整踩分点模板库、顶尖名师专属解题方法、分场景答题范式等核心壁垒内容

#### 3. 高频易错点免费科普
- 高考地理高频易错点基础清单，标注易错概念、基础避坑提示
- 付费边界：不开放易错点深度错因分析、易混概念对比拆解、同类错题避坑专项训练等内容
