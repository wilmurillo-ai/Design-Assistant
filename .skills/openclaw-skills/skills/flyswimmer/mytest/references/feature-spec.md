# 地理提分技能功能规格说明

## 功能边界定义
### 免费功能边界
所有免费功能严格按照SKILL.md中定义的开放范围提供，不得超出开放内容范围提供付费级功能。
当用户提问涉及付费边界内容时，需友好引导："这个功能属于深度提分付费模块哦，当前免费版本暂不支持，需要解锁完整功能可以了解付费版~"

### 内容准确性要求
1. 所有知识点回答必须基于官方教材和课标，不得出现错误、超纲内容
2. 题目解析必须标注对应考点和教材章节
3. 所有地理数据、图表、公式必须符合高考大纲要求

## 触发场景
当用户出现以下提问时，优先使用本技能：
1. 高中地理知识点相关提问："什么是正午太阳高度角？""洋流分布规律是什么？"
2. 教材相关查询："人教版必修一第三章讲了什么？""湘教版地理教材目录"
3. 工具类查询："地理计算公式有哪些？""等值线怎么判读？"
4. 题目相关提问："这道地理题怎么做？""求这道真题的解析"
5. 错题相关："帮我整理一下这部分的错题""这道题的考点是什么？"
6. 提分技巧相关："地理选择题有什么答题技巧？""综合题怎么拿分？"
