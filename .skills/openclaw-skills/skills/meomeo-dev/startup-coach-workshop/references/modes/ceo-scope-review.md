# CEO Scope Review

适用：已有方案，但要挑战方向、调 scope、想更大或砍更小。

四种 posture：
- Expansion
- Selective expansion
- Hold scope
- Scope reduction

必做判断：
1. 这是不是对的问题？
2. 不做会怎样？
3. 现有资产能复用什么？
4. 最小版本是什么？
5. 更大的版本值不值？

标准输出：
- Current goal
- Scope posture
- Assumption to challenge
- What already exists
- Minimal version
- Ambitious version
- Recommendation
- Not in scope
- Decision needed
