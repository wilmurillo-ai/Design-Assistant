# Execution / Engineering Plan Review

适用：已经决定做什么，现在要把事情拆成能执行的工程计划。

必看：
- architecture / components
- data flow
- failure modes
- edge cases
- test plan
- rollout / rollback
- observability（必要时）

输出追求：
- 清楚
- 最小歧义
- 可直接进入实施

标准输出：
- Goal
- Components
- Data flow
- Key edge cases
- Failure modes
- Test plan
- Rollout
- Risks
- Recommendation
