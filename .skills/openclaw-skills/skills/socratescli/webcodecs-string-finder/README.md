# webcodecs-string-finder

Finds valid WebCodecs strings for video and audio by researching codec support tables and detailed specifications on webcodecsfundamentals.org.

## Installation

### gemini-cli

```bash
gemini skills install https://github.com/socratescli/webcodecs-string-finder-skill
```

After installation, reload your skills in the interactive session:

```bash
/skills reload
```
