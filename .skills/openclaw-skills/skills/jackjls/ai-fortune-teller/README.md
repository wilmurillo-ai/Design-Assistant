# AI Fortune Teller 🔮

AI玄学大师 - 基于 MiniMax API 的智能命理占卜工具

## 功能概览

| 模块 | 功能 | 技术 |
|------|------|------|
| 八字命理 | 四柱八字完整分析 | MiniMax Text + 传统典籍 |
| 塔罗占卜 | 韦特塔罗78牌全系 | MiniMax Text + 牌阵系统 |
| 每日运势 | 星象+八字日主综合运势 | MiniMax Text |
| 人生K线图 | 股票K线风格的运势可视化 | MiniMax Image |
| 前世画像 | 灵魂特质可视化 | MiniMax Image |
| 未来伴侣 | 理想伴侣可视化 | MiniMax Image |

## 安装

```bash
clawhub install ai-fortune-teller
```

## 技术栈

- **文本生成**：MiniMax API (text-01)
- **图片生成**：MiniMax Image Generation API (image-01)
- **命理框架**：穷通宝鉴、滴天髓、三命通会等典籍
- **塔罗体系**：韦特塔罗完整78牌
