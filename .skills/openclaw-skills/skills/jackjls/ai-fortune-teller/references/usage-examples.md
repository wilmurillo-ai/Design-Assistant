## 快速开始

发送以下格式即可开始使用：

**八字分析**：
```
【八字命理分析】
姓名：你的姓名
性别：男/女
出生地：省/市
公历出生日期：XXXX年X月X日
出生时间：X时
```

**塔罗占卜**：
```
【塔罗占卜】
问题：你的问题
牌阵：单张/三张/凯尔特十字
```

**运势查询**：
```
【今日/本周/本月运势】
出生年月日：XXXX年X月X日
关注领域：综合/事业/爱情/财运/健康
```
