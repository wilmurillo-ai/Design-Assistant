# Vis Moot国际商事仲裁模拟法庭指南

## 描述
Willem C. Vis Moot（维斯模拟仲裁庭）是全球规模最大、影响力最广的国际商事仲裁模拟法庭赛事。本Skill涵盖Vis Moot的赛事介绍、备赛流程、书状写作和庭辩训练等全方面内容。

## 适用场景
- 准备参加Vis Moot或Vis East赛事
- 参加贸仲杯等Vis预选赛
- 学习国际商事仲裁和CISG（国际货物销售合同公约）
- 模拟法庭书状写作和庭辩训练

## 赛事概览

### 赛事体系
- **Vis Moot（维也纳）** - 主赛事，每年4月在奥地利维也纳举办
- **Vis East（香港）** - 姐妹赛事，每年3月底在香港举办
- **预选赛（Pre-Moot）** - 全球各地的热身赛，包括：
  - 贸仲杯（CIETAC，中国内地最大选拔赛）
  - Moot Shanghai（上海邀请赛）
  - 北京邀请赛
  - 广仲杯
  - 韩国Seoul Pre-Moot、日本Vis Japan等

### 比赛结构
- 每队2名庭辩员：1位负责程序问题，1位负责实体问题
- 程序问题：依据当年选定的仲裁机构规则
- 实体问题：CISG框架下的争议，可能涉及PICC、IBA Guidelines等
- 仲裁员：资深仲裁员、学者和律师

### 时间线（以贸仲杯为节点）
| 时间 | 事项 |
|------|------|
| 10月初 | Vis Moot官网发布赛题 |
| 11月中旬 | 提交贸仲杯书状 |
| 11月下旬-12月上旬 | 贸仲杯庭辩 |
| 12月中旬 | 提交Vis Moot申请书 |
| 1月下旬 | 提交Vis East答辩状 |
| 3月底-4月初 | Vis East（香港） |
| 4月中下旬 | Vis Moot（维也纳） |

## 赛事特点

### 基本特征
1. **假案子真规则** - 虚构企业争端，但使用真实仲裁规则
   - 第33届（2025-2026）：SIAC（新加坡国际仲裁中心）规则
   - 第34届（2026-2027）：AAA-ICDR规则
   - 第35届（2027-2028）：CRCICA规则

2. **五花八门的预选赛** - 预选赛是热身赛，优胜者可能获得资助

3. **争议复杂、涵盖面广**
   - 4个核心争议点：2个程序 + 2个实体
   - 每年引入"特色"法规（如反腐败公约、GDPR等）
   - 当事人名称固定：申请人Medditeraneo国，被申请人Equatoriana国
   - 赛题发布后三周可申请事实澄清（PO2）

### 预选赛 vs 国际赛风格差异

#### 内地预选赛特点
- 仲裁员多为实务律师，更重视程序议题
- 提问角度务实，侧重实际争议解决
- 更关注交流感和表达流畅度，而非法律研究深度

#### 国际赛特点
- 提问角度更多元、更深入，常有意料之外的切入
- 更偏爱对法律本身有扎实研究的队伍
- 英美法仲裁员可能使用政策论证
- 论点会随比赛进程发生变化（预选赛到国际赛讨论重心会转移）

## 备赛流程

### 时间管理
- 以贸仲杯为前期备赛时间线
- 先写C方大纲，再细化书状，再写R方书状
- 建议C方文书完成后就开始庭辩训练
- **克服拖延症**：先输出，再迭代

### 队内角色划分
- 一般分为庭辩员和研究员
- 部分赛队：招新时即分赛道
- 部分赛队：前期不分配，所有队员参与书状和庭辩训练
- 研究员同样重要：记录、稿件调整、观察对手等

### 指导老师的作用
- 全程参与竞赛，阅读书状并提方向性意见
- 提供庭辩训练，联络专家答疑
- 赛前赛后情绪安抚，适时团建
- 没有教练也不意味着没有破局可能（靠团队团结和坚韧）

## 书状写作

### 评分机制（Vis Vienna）
- 首轮：每位评委收到4份书状并排序
- 约前20%进入第二轮评分
- 评分维度：实质论点、简洁表达、法律渊源、整体呈现、语言能力
- **重要**：正赛晋级几乎完全取决于庭辩表现，不受书状成绩影响（但需及时提交）

### 格式要求
- 核心成分：封面、目录、缩略语表、参考文献、事实陈述、论点总结、管辖权论点、实体论点、仲裁请求
- 正文不超过35页A4，字号不小于12
- 引用在正文，不使用脚注/尾注，完整信息在参考文献（Index of Authority）

### 赛题阅读方法
赛题按仲裁时间顺序排列：
1. 申请仲裁（Request for Arbitration）- 申请方视角+证据
2. 答辩书（Answer）- 被申请方反驳+证据
3. 新程序请求（如有）
4. 第一份程序令（PO1）- 归纳争议点+一致同意的事实
5. 第二份程序令（PO2）- 回复事实澄清问题

**技巧**：赛题中的"代理律师"已经给出初步法律论述，指明相关法律条款和思路！

### 法律渊源
- 以当事方约定为主（合同中的法律选择条款）
- 常出现两大焦点：
  1. 仲裁协议适用法（仲裁地法 vs 主合同适用法）
  2. 主合同适用法是否为CISG

## 庭辩训练

### Vis庭辩特点
- 氛围轻松融洽，对话感强，交流意味浓
- 目标：成为"讨喜"、专业、商业化的律师
- 没有"正义" vs "非正义"，只有复杂的商业权衡

### 庭辩心态
1. 向对方辩手保持礼貌友好
2. 面对仲裁员摆正心态：帮助仲裁员理清问题，不是辩论
3. 即便问题刁钻，也是给辩手机会发表意见
4. 温和、友好、专业是主基调

### 赛场流程
- 每队2名辩手，3位仲裁员（1位主裁+2位边裁）
- 申请人=Claimant，被申请人=Respondent
- 正赛期间穿插社交活动和讲座，建议积极参与

## 备赛建议

### 核心要点
1. **尽早开始** - 不要等研究清楚再动笔，先输出再迭代
2. **多参加预选赛** - 接触不同论证思路和风格，但注意平衡
3. **团队合作** - 研究员同样重要，互相帮助
4. **保持开放心态** - 国际赛会打破默会共识，灵活应变

### 首届参赛建议
- 挑战：不熟悉其他法域对手的能力和惯例
- 书状内容呈现顺序、法律渊源要求、庭辩身体语言、国际化演讲风格等
- 有前辈/教练帮助更好，但没有也可能成为黑马
