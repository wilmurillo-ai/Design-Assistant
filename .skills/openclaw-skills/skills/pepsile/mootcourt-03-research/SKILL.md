# mootcourt-03-research

## 描述

国际法模拟法庭书状写作全攻略，涵盖国际法渊源检索、书状结构设计、案例分析方法、语言风格优化等核心技能。

## 核心功能

### 1. 国际法渊源检索与适用

#### 条约法
- **条约解释规则**：遵循《维也纳条约法公约》第31-33条
- **检索资源**：UNTC（联合国条约集）、ILC（国际法委员会）文件

#### 习惯国际法
- **构成要件**：国家实践 + 法律确信
- **国家实践检索方向**：
  - 联合国《联合国立法丛书》
  - 《美国对外关系法第三次重述》
  - British and Foreign State Papers (1812-1968)
  - 《中国外交概览》《中国国际法实践与案例》
- **法律确信证明技巧**：参考 ICJ、国际法委员会等权威机构的认定

#### 一般法律原则
- **适用场景**：比例原则、必要原则、善意原则等常见原则可直接使用
- **证明方法**：从各国国内法、条约序言、ICJ 判例中拼凑
- **注意事项**：原则是兜底条款，条约和习惯法优先适用

### 2. Application 写作（案例分析）

#### 英美法案例思维
- **关键步骤**：通过案例建立和拆解联结（linkage）
- **案情分析要点**：
  - 发生了什么（What happened）
  - 有谁涉案（Who are involved）
  - 涉案人行为动机
  - 关键事实 vs 辅助事实
  - 原审法院认定

#### ICJ 案例检索
- **官网检索**：直接输入案例名称（如 Nicaragua v United States of America）
- **注意事项**：
  - 注意引用年份（同一案例可能有多个判决）
  - 阅读判决中的论证过程，不只是结论
  - 参考法官意见（separate/dissenting opinion）获取不同视角

### 3. 书状结构设计

#### IRAC 结构
- **Issue**：争议点
- **Rules**：法律规则（要有约束力的法源）
- **Application**：结合案情适用规则
- **Conclusion**：简短结论

#### 常见写作雷区
- ❌ 直接大段复制粘贴法条原文
- ❌ Rule 不够充足，只有一个
- ❌ 对 rule 没有解释说明
- ❌ 事实匹配要件时没有抓住重点
- ❌ 简单罗列事实，缺乏逻辑关系
- ❌ 对事实过度推测

#### 思维导览（Roadmap）
在每个大争议点开头加上摘要，提醒法官这部分要论述什么。

### 4. 语言风格优化

#### 简明法律英语
- ✅ 使用正式表述（do not, will not）
- ✅ 主语后面紧跟动词
- ✅ 少用被动语态
- ✅ 可以适当炫技，但不要影响阅读体验

#### 反例与修改
**不好**：
```
There are several procedures involved, which, if there is to be an increase in corporate income, must be carefully followed.
```

**好**：
```
Several procedures must be carefully followed to increase corporate income.
```

### 5. 事实声明写作

- **要点**：不直接给出结论，通过事实罗列给出倾向性暗示
- **技巧**：申请人方和被申请人方互拆，从不同立场解读事实

## 使用示例

### 示例 1：习惯国际法证明

```markdown
## Rules

根据国际法委员会《习惯国际法的识别》，习惯国际法需满足两个要件：国家实践和法律确信。

### 国家实践

国家实践包括：
1. 国家的行政行为、立法、法院判决和国际缔约活动
2. 联合国大会决议、国际司法机构判决
3. 各国国内法（如 Scotia 案中英国航行程序立法构成国际习惯基础）

检索资源可参考《联合国立法丛书》《美国对外关系法第三次重述》等。

### 法律确信

法律确信的证明难度较高，但书状写作中只需充分说理即可。可参考 S·普拉卡什·辛哈在《国际法上的庇护》一书中证明"外交庇护"不构成国际习惯法的论述方式。

## Application

在本案中，A 国对 R 国实施的经济制裁...
```

### 示例 2：案例联结建立与拆解

**A 方立场**：
> A 国可以引用尼加拉瓜诉美国案，说它里面也存在相似的经济制裁，且 ICJ 裁定不违反不干涉原则，所以自己这么做是合法的。

**R 方反驳**：
> 尽管 ICJ 裁定不违反干涉原则，但是裁定其违反了尼加拉瓜和美国之间的经济互助协定，本案也有一个互助协定，所以还是违法的。

**A 方再反驳**：
> 尼加拉瓜案中美国是切断了和尼加拉瓜的所有贸易往来，所以才违法，但本案只切断了一小部分贸易往来。

**R 方再反驳**：
> 违不违法的本质不是切断了多少贸易往来，而是这种手段是不是强迫的（coercion），以及这部分贸易对整个经济发展的影响。
