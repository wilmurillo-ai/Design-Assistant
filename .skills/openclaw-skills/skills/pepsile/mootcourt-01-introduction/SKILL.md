# 模拟法庭入门指南 Skill

## name: mootcourt-01-introduction

## description: 
国际模拟法庭入门基础指南，帮助初学者了解什么是国际模拟法庭、是否应该参加、如何组建/加入赛队，以及书状写作的基础知识。

---

## 核心功能

### 1. 国际模拟法庭基础认知
- **概念界定**：区分英文国际模拟法庭与其他法律竞赛
- **赛事分类**：按学科（国际公法/私法/经济法）和类型（法庭/仲裁庭）划分
- **含金量排序**：Jessup/Vis 为第一档，FDI/ICC/IHL 次之，各类专业赛事适合入门

### 2. 参赛决策评估
- **好处**：
  - 务实：保研加分、升学/求职简历背书、实习机会
  - 务虚：能力提升（法律思维、英文读写、庭辩）、开拓眼界、积累人脉
- **坏处**：
  - 时间成本高（大半年备赛）
  - 存在运气因素（匹配机制、评委专业性）
  - 需要长期主义投入

### 3. 赛队组织与建设
- **从零开始组队**：
  - 规模：6-8人为宜，宜少不宜多
  - 角色：研究员→庭辩员自然演化，无需过早划分
  - 目标：长期高远目标 + 短期具体目标
  - 资源：主动调动校内校外资源，用好"六度分割"理论

- **校内选拔技巧**：
  - 创造亮点：做好1-2个细节点比泛泛而谈更重要
  - 材料精炼：简历/附加材料宜短不宜长，有的放矢
  - 态度：自信且诚恳，Fake it until you make it
  - 加分项：提前协调时间、提交额外的案例分析/检索报告

- **团队协作**：
  - 沟通：及时、高质量、适度频率
  - 主人翁意识：积极主动，眼里要有活儿
  - 角色定位：教练/队长/研究员/庭辩员各司其职

### 4. 后进赛队超越策略
- **选材**：变被动为主动，定向寻找+熟人推荐
- **心态**：以学习为目的，享受过程
- **认知**：
  - 模拟法庭是教育过程，竞技只是手段
  - 承认差距，笨鸟先飞
  - 宏观把握赛题，挖掘"真问题"

## 使用示例

### 场景1：新手入门决策
```
问题：我是大一法学生，该不该参加模拟法庭？

回答：
1. 先评估你的目标（出国/律所/升学？）→ 如果是，模拟法庭很有价值
2. 评估时间投入→ 能否接受大半年备赛？
3. 可以从 ICC/IHL/航空法等入门赛事开始，再过渡到 Jessup/Vis
4. 关键：选择后把选择变成好选择，而非期待选择本身改变人生
```

### 场景2：从零开始组队
```
步骤：
1. 找到 3-4 个态度靠谱的核心队友（能长跑的，而非一开始跑得快的）
2. 设置长期目标（如2年内冲进国际赛）+ 短期目标（读完近5年最佳书状）
3. 主动找学校要政策支持（保研/经费/场地）
4. 厚脸皮找资源：老师、前辈、"法律竞赛"公众号
5. 定期团建，保持积极备赛情绪
```

### 场景3：准备校内选拔
```
建议：
1. 不要追求全面，挑 1-2 个赛题细节深入解读
2. 简历 1 页足够，附加材料可提交 1 页 ICJ 案例分析
3. 面试时自信展示已知，诚恳承认未知
4. 如没选上，可申请随队学习一年
5. 时间安排：提前协调选课，不参与过多社团
```

## 核心观点提炼

1. **道路通向场外**：模拟法庭本身只是道路，更重要的是它带给你的锻炼和思考
2. **长期主义**：Moot 是先苦苦苦后甜的项目，长期主义者走得更远
3. **选择之后**：问题不是如何做出好选择，而是选择后如何把它变成好选择
4. **Fake it until you make it**：面试时的自信可以"装"出来，装到成功为止
5. **Talk less, show me your paper**：少说多做，用实际行动证明
6. **眼里要有活儿**：主动积极，不要等着别人布置任务
