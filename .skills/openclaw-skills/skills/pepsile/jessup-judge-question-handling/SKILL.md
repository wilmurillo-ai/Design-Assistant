# Jessup模拟法庭法官质询应对技巧

## 描述
Jessup国际模拟法庭竞赛中法官质询的应对策略和技巧。包括如何识别法官关切、如何正面回应问题、以及如何巧妙处理法律与事实结合的论证。

## 适用场景
- 参加Jessup国际模拟法庭竞赛
- 准备模拟法庭庭辩和法官质询环节
- 学习国际法语境下的法律论证技巧
- 危机公关和质询应对训练

## 核心知识点

### 1. 识别法官的真实关切
法官的问题往往不是字面意思，而是背后隐藏的担忧。需要：
- 从问题中寻找法官最关心的点
- 针对关切点结合事实或法律正面回击
- 不要回避问题，即使法律依据不足

### 2. 法律与事实结合的论证
当法律规则不适用时，仍需正面回应：
- 解释法律不适用的原因
- 更重要的是解释法律规则背后的立法意图
- 说明本案中这些担忧并不存在或情形紧急
- 用事实论证来缓释法官的关切

**示例：** 2015年赛题条约解除问题
> VCLT 65-68条程序规定不是习惯国际法，但不能以此为由单方解除条约。需要解释：
> 1. VCLT不适用
> 2. 但VCLT起草时的考虑因素在本案中不存在
> 3. 或本案情形紧急，无法预先通知

### 3. 纠正法官认知错误的方法
当法官对国际法规则认识错误时：
- 先确认并直接回应法官的关切
- 循序渐进地解释现行国际法规则
- 不要直白地告诉法官"您知识匮乏"
- 用国家实践改变法官的常识认知

**示例：** 2017年赛题停水侵犯人权问题
> 法官认为A国停水侵犯人权。应对策略：
> 1. 坚定立场：我方没有侵犯人权（或克减有合理性）
> 2. 用国家实践类比：非洲很多国家供水定量配给
> 3. 事实上撇清关系：不是A国行为，而是客观事实造成饥荒
> 4. 最后说明：侵犯本国国民人权不意味着赔偿其他国家

### 4. 针锋相对的策略
- 直接回答问题，不要回避
- 以欢迎和开放的心态回应法官问题
- 庭辩是对话，不是单人秀
- 问题多的法官往往更愿意了解选手，会给出更有价值的建议

## 实操建议

### 庭辩心态
- 庭辩本身是与法官的对话，不是单人秀
- 有问题的法官往往是更愿意了解选手的
- 如果法官都不提问，反而容易提心吊胆
- 以欢迎和开放的心态回应问题

### 关键原则
1. **不要回避问题** - 即使法律依据不足，也要正面回应
2. **找到关切点** - 法官的问题背后都有真实担忧
3. **法律+事实** - 结合法律规则和案件事实论证
4. **温和专业** - 保持专业态度，不要对抗
