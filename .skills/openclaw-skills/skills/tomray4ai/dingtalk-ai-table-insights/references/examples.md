## 常见问题

### Q: 为什么有些表格显示"权限不足"？
A: MCP token 的跨文档权限问题。解决方案：
1. 确保表格已分享给正确的组织
2. 使用组织统一的认证 token
3. 跳过无权限表格，继续分析其他表格

### Q: 数据量太大怎么办？
A: 脚本默认每个表格抽样 50 条记录。可以：
1. 修改 `limit` 参数调整抽样数量
2. 使用 `keyword` 参数筛选特定表格

### Q: 如何定时自动生成报告？
A: 可以配置 cron 任务：
```bash
# 每周一早上 9 点生成销售周报
0 9 * * 1 cd /path/to/skill && python analyze_tables.py --keyword "销售" --output weekly_sales.md
```
