# 天赋领域探索器 🧭

通过科学方法发现你的核心天赋，制定可落地的成长战略。

## 核心框架

```
第一步：结构化访谈（15个递进式问题）
    ↓
第二步：天赋解码引擎（五阶段分析）
    ↓
第三步：90天落地行动方案（三阶段模板）
```

---

## 第一步：天赋访谈探针

### 设计原则
- **问"事"不问"感"** — 先还原场景，再谈感受
- **问"第一次"不问"最擅长"** — 绕过自我认知美化
- **追问细节到"说不清楚"为止** — 说不清楚的往往是真天赋
- **语气自然如聊天** — 不像心理量表，有示例防偏题

### 访谈开场流程

#### 开场白（标准模板）

```
你好，我是王富贵 🦞

今天我们要做一个天赋领域探索。
我会通过15个递进式的问题，来发现你的核心天赋和优势领域。

整个过程不是考试，没有对错。
你说得越多、越真实，我就越能帮你看清自己。

过程中如果有任何问题让你觉得不好回答，直接告诉我，
我会换一种方式问。

准备好了我们就正式开始。
```

#### 暖场家常问题（仅记录，不计入分析）

在正式提问之前，先聊几句，了解被访谈人的基本背景：

- 你现在主要做什么？自由职业还是在读书？
- 平时一天大概怎么安排自己的时间？
- 有什么长期坚持的爱好或者习惯吗？

这些信息帮助在解码时结合作出合理解读。

---

### 十五个递进式问题

#### 🎯 第一阶段：入口 — 找回"对"的感觉

**Q1.**
最近有没有过这种感觉：明明已经过了饭点了，但你完全没注意到，专注在一件事里很久？

**追问：** 那时候你在做什么？

---

**Q2.**
做那件事的时候，你整个人是什么状态？比如脑子里在想什么，手在做什么，动作是快还是慢？

---

**Q3.**
做完之后你是更精神了，还是整个人"被掏空"了？大概是什么感觉？

---

#### 🎯 第二阶段：锚定 — 挖一个完整的故事

**Q4.**
给我讲一个最近发生的例子——有一件事你完成得特别顺，自己都觉得"哎这次不错"。当时是什么情况？

> 比如工作里搞定了一个难题，或者帮朋友解决了一个困扰他很久的问题。

---

**Q5.**
过程中有没有遇到什么意外或卡住的地方？你是怎么绕过去的？

---

**Q6.**
做这件事的时候，你是那种"等着 Deadline 逼自己动手"的人，还是那种"我先干为敬"的人？

---

**Q7.**
做完之后，有人夸过你吗？怎么夸的？

> 比如别人说"你怎么想到的"，或者"这你都会？"之类的。

---

#### 🎯 第三阶段：解构 — 拆解能力组件

**Q8.**
如果你把刚才做的这件事拆解成几个动作，是哪几步？

> 比如"先听对方说完 → 找到他没说清楚的那个点 → 用大白话重新组织一遍"。

---

**Q9.**
这几步里面，哪一步你做起来特别轻松，几乎不用动脑子就搞定了？

---

**Q10.**
有没有过这种经历：一件事你第一次接触，结果上手比周围人都快，或者做得比想象中好得多？

**追问：** 那次是什么感觉？还记得当时在做什么吗？

---

**Q11.**
那件事对你来说是"天生就会"的那种轻松感，还是"好像以前练过"的那种熟悉感？

---

#### 🎯 第四阶段：回溯 — 寻找时间脉络

**Q12.**
有没有一件事，虽然发生的场景不一样了，但本质上好像一直是同一件事？

> 比如小时候爱搭积木、长大爱写代码，本质都是"把东西从零开始搭出来"。

---

**Q13.**
你有没有被身边人真诚地夸过什么，但你自己听到的时候心里其实在想"这也算本事？"？

---

**Q14.**
说三个你羡慕或者欣赏的人，他们身上有什么能力是你觉得"我要是有就好了"？

**追问：** 他们分别做了什么让你有这个感觉？

---

**Q15.**
把你今天讲的所有事情里你做的事拢一拢，如果分成三大类，你会怎么归？

---

**正式问答结束提示：**

15个问题全部完成。告诉被访谈人：

> "好，正式问题到此结束。你回答得很好。
接下来我去整理分析，大概需要几分钟。"

---

### 追问规则（铁律）

| 情况 | 追问方向 |
|-----|---------|
| 回答模糊（"我比较善于..."） | 给我一个最近的具体例子 |
| 回答停在感受层 | 你具体做了什么？ |
| 回答太完美 | 有没有一次你做砸了的情况？ |
| 回答说第一次就做得好 | 详细描述那次过程（验伪）|
| 回答说他擅长某事 | 为什么重要？（阶梯法）|

---

### 信号判定规则

```
✅ 强信号：回答具体、有细节、时间感明确、做完后有能量感
⚠️ 待验证：回答模糊、停在价值观层面、举例太完美
❌ 排除：回答指向"被训练出来的技能"或"社会角色要求"
```

## 第二步：天赋解码引擎

### 第一阶段：信号拆解

对每条原始回答，拆解成三轨并行：

**行为轨** — 记录具体动作，不记感受
> "我主动给团队写了复盘文档" → 记录「主动输出结构化文档」

**情绪轨** — 捕捉感受关键词
> "做完特别爽" → 标记「成就感」「内在驱动」

**价值轨** — 追问"为什么重要"
> "为什么写复盘让你爽？" → "因为帮到别人少走弯路很重要" → 锚定「价值：利他+知识传递」

---

### 第二阶段：模式聚类

建立 Q-T-S 矩阵（Question → Theme → Signal）：

```
信号类型A：帮人理解复杂事物
  → 出现在 Q2（回答内容）...
  → 出现在 Q5（回答内容）...
  → 出现在 Q9（回答内容）...
  → 频率：3次 → 信号强度：强

信号类型B：独自深度工作
  → 出现在 Q3...
  → 出现在 Q7...
  → 频率：2次 → 信号强度：中
```

**聚类规则：**
- 同一信号跨 ≥3 个问题出现 → 核心天赋候选
- 跨 2 个问题出现 → 潜在天赋
- 只出现 1 次 → 存档，不计入天赋结论

---

### 第三阶段：天赋三角验证

每个聚类出来的能力，用三个维度打分（1-10分）：

| 维度 | 问法 | 验证逻辑 |
|-----|------|---------|
| **本能指数** | 回忆这件事你第一次做/接触时，是否比周围人自然 | 首次表现就有优势 = 天赋；练了很久才赶上 = 非天赋 |
| **热情指数** | 做这件事时是"充电"还是"耗电" | 充电 = 热情天赋；咬牙坚持 = 意志力天赋 |
| **心流指数** | 做这件事时时间感消失吗？做完累还是精神更好？ | 时间感消失 + 精神变好 = 强天赋信号 |

**三维交叉表：**

```
本能≥7 + 热情≥7 + 心流≥7 → 核心天赋（直接定位）
本能≥7 + 热情≥7 + 心流<7 → 擅长但环境不支持（调整场景）
本能≥7 + 热情<7           → 训练出来的技能，非天赋
热情≥7 + 心流≥7           → 潜在天赋（需要练本能）
三维度都低                 → 排除
```

---

### 第四阶段：叙事结构分析

用 McAdams 框架看故事主轴：

- **困扰轴**：你故事里反复出现的"痛点"是什么？
- **成长弧**：你的能力是怎么积累的？是线性积累还是某个节点突变？
- **使命句**：如果用一句话概括你存在的意义，你会说什么？

---

### 第五阶段：差异化定位

用 Sternberg 三角排除法定性：

```
分析型天赋：你会用独特的路径拆解问题
          → 特征：爱问"为什么"、质疑假设、擅长比较不同观点

创造型天赋：你会做别人想不到的事
          → 特征：想法经常跟别人不一样、爱跨界、不喜欢走老路

实践型天赋：你能把理论变成现实
          → 特征：不只是想，更要做出来；擅长落地；不喜欢空谈理论
```

---

### 最终输出：「天赋确认书」

每一项天赋结论，附上：
- **来源证据**：哪几个问题的回答支持这个结论（直接引用原话）
- **反例证据**：哪些回答说明这不是天赋（诚实列出）
- **差异化陈述**：用一句话说清楚"你跟别人不一样在哪"
- **天赋陷阱**：这个天赋的阴暗面是什么
- **PERMA验证**：这个天赋主要滋养了积极心理学五要素里的哪几个

## 使用流程

1. **第一步**：运行访谈，按顺序问完15个问题，详细记录回答
2. **第二步**：运行解码引擎，产出天赋画像
3. **第三步**：选择天赋入口，填写90天战略地图，启动行动方案
