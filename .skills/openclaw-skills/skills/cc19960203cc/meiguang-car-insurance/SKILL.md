# 美光车险PDF转Excel v2.2.0

从车险保单 PDF 自动提取 14 个字段，输出 Excel。

## 支持的保险公司和险种

| 保险公司 | 险种 |
|---------|------|
| 中国太平洋财产保险股份有限公司 | 交强险、商业险、驾意险（非车险） |
| 中国人民财产保险股份有限公司 | 交强险、商业险 |
| 亚太财产保险有限公司 | 交强险、商业险、非车险 |
| 中国大地财产保险股份有限公司 | 交强险、商业险、驾意险、非车险（安行如意保） |
| 浙商财产保险股份有限公司 | 交强险、商业险 |
| 华海财产保险股份有限公司 | 交强险、商业险、非车险 |

## 提取字段（14个）

1. 签单时间
2. 保险公司名称（标准化全称）
3. 保单号
4. 保险起期
5. 车辆使用性质
6. 车架号（VIN，17位）
7. 车辆型号名称
8. 被保人姓名
9. 被保险人证件号
10. 被保险人手机号
11. 车牌号码
12. 险种名称原始
13. 实收保费
14. 车船税

## 使用方法

直接运行：
```bash
python scripts/run_extract.py
```

输入：`C:\Users\Administrator\Desktop\车险保单`
输出：`C:\Users\Administrator\Desktop\车险保单提取结果.xlsx`

## 配置

在 `scripts/run_extract.py` 顶部修改：
```python
PDF_FOLDER = r"C:\你的PDF文件夹路径"
OUTPUT_FILE = r"C:\输出文件路径.xlsx"
```
