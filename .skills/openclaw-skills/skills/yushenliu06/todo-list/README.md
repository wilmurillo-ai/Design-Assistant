# Todo List 待办事项管理技能

## 依赖要求

- **Python 3** - 运行脚本
- **OpenClaw CLI** - 创建定时提醒和发送消息

## 功能特性
- ✅ 添加待办事项，支持优先级和到期时间设置
- ✅ 查看待办列表，支持按状态和优先级筛选
- ✅ 标记任务完成
- ✅ 删除任务
- ✅ 查看任务详情
- ✅ 自动到期提醒（通过 OpenClaw cron）
- ✅ 标签系统
- ✅ 项目管理
- ✅ 附件功能（带安全限制）

## 安全说明

### 文件访问
- 只允许访问用户明确指定的文件
- 禁止访问系统敏感目录（/etc, /root, /var 等）
- 文件大小限制：最大 50MB
- 附件会被复制到隔离目录，不修改原文件

### Cron 任务
- 所有提醒通过 OpenClaw CLI 创建
- 用户可以通过 `openclaw cron list` 查看
- 提醒只发送到用户配置的频道

### 数据存储
所有数据存储在 `~/.openclaw/workspace/memory/` 目录
