# AI 图片提示词优化器 (最终版)

## Overview

根据用户上传的参考图片和需求描述，生成经过专业优化的 AI 图片生成提示词，并解答如何消除 AI 感、提升写实效果。

**核心能力：**
1. 自动识别图片数量，使用 "图1"、"图2" 等方式留出图片引用位置
2. 智能分析衣服面料特征，确保面料质感准确还原
3. 支持中英文和 JSON 格式输出
4. 针对 Nonbana Pro（Flux）、Nano Banana 等平台深度优化

**知识来源：**
- aouiho 实际案例库（产品图、模特图、场景合成）
- Nano Banana 39+ 热门提示词（3D建模、风格迁移、图像编辑）
- 面料识别与描述专业指南
- 消除 AI 感完整技术方案

## Workflow

### Step 0: 判断用户意图（新增）

首先判断用户是需要：

| 意图类型 | 用户行为 | 处理方式 |
|----------|----------|----------|
| **正向生成** | 上传参考图 + 描述需求 | 进入 Step 1-6，生成新提示词 |
| **逆向推导** | 只上传图片，要求"分析"、"反推"、"推导提示词"、"1:1还原" | 进入【逆向推导模式】 |

**逆向推导触发词：**
- "分析这张图片"
- "反推提示词"
- "推导这张图的提示词"
- "1:1还原"
- "复现这张图片"
- "这张图的提示词是什么"

### 【逆向推导模式】图片分析与提示词复原

当用户要求逆向推导时，执行以下步骤：

#### Step 0.1: 全面分析图片元素

对上传的图片进行逐元素分析，包括但不限于：

**1. 主体分析**
| 分析项 | 具体内容 |
|--------|----------|
| 主体类型 | 人物/产品/风景/建筑/静物/抽象图形 |
| 主体数量 | 单个/多个/群组 |
| 主体特征 | 外貌、颜色、材质、形状、尺寸 |

**2. 人物分析（如有人物）**
| 分析项 | 具体内容 |
|--------|----------|
| 人物数量 | 单人/双人/多人 |
| 人物特征 | 年龄、性别、种族、发型、发色 |
| 穿着 | 衣服类型、颜色、面料、配饰 |
| 表情 | 微笑/严肃/惊讶/自然/其他 |
| 姿势 | 站姿/坐姿/躺姿/动作/手势 |

**3. 场景与环境**
| 分析项 | 具体内容 |
|--------|----------|
| 场景类型 | 室内/室外/虚拟场景 |

| 环境元素 | 家具、植物、建筑、道具、装饰 |
| 空间感 | 前景/中景/背景层次 |

**4. 光线与色彩**
| 分析项 | 具体内容 |
|--------|----------|
| 光源类型 | 自然光/人工光/混合光 |

| 整体色调 | 高饱和/低饱和/黑白/复古/电影感 |

| 构图方式 | 中心构图/三分法/对称/引导线 |

| 景别 | 特写/近景/中景/全景/远景 |

**6. 质感与细节**
| 分析项 | 具体内容 |
|--------|----------|
| 表面质感 | 光滑/粗糙/金属/布料/木质/玻璃 |
| 纹理细节 | 皮肤毛孔/布料纹理/金属反光 |
| 特效 | 光晕/颗粒/模糊/锐化/ HDR |

**7. 风格与氛围**
| 分析项 | 具体内容 |
|--------|----------|
| 艺术风格 | 写实/卡通/油画/水彩/像素/赛博朋克 |

**8. 文字与图形元素**
| 分析项 | 具体内容 |
|--------|----------|
| 文字内容 | 所有可见文字（逐字记录） |

| 水印/签名 | 位置、内容、样式 |

### Step 1: 识别图片数量与内容（正向生成模式）

**关键：首先确认用户上传了多少张图片，并分析每张图的内容**

| 图片编号 | 分析要点 |
|----------|----------|
| 图1 | 主体类型、风格、颜色、材质、关键特征 |
| 图2 | 同上（如有） |
| 图3+ | 同上（如有） |

**图片引用规则：**
- 1 张图片 → 使用 "图1"
- 2 张图片 → 使用 "图1"、"图2"
- 3 张图片 → 使用 "图1"、"图2"、"图3"
- 以此类推...

### Step 2: 判断任务类型

根据用户需求，判断属于以下哪种任务类型：

| 任务类型 | 典型需求 | 参考模板 |
|----------|----------|----------|
| 人像摄影 | 韩国ins风格、日系胶片、纯欲风 | 模式 A |
| 模特换装 | 给模特换衣服、虚拟试穿 | 模式 B |
| 产品图 | 重新打光、场景合成、角度变换 | 模式 C |
| 材质替换 | 换面料、贴图案 | 模式 D |
| 图像编辑 | 改表情、换姿势、换背景 | 模式 E |
| 风格迁移 | 吉卜力风格、像素风、3D手办 | 模式 F |
| 海报设计 | 电商详情页、产品海报 | 模式 G |

### Step 3: 面料识别（衣服类必做）

**如果涉及衣服，必须进行面料分析：**

1. **视觉分析** - 观察光泽度、纹理、厚度、垂坠感
2. **类型判断** - 确定是棉、丝、牛仔、针织等
3. **描述生成** - 使用专业面料词汇描述

**面料不清时，主动询问用户：**
> "为了更好地还原衣服质感，能否告诉我这是什么面料？常见选项：纯棉、牛仔、丝绸、针织、皮革等。或者提供一张面料特写图？"

#### 模式 B：模特换装类（双图/多图）

**结构：**

```
图1的[人物描述]穿着图2的[衣服描述]，[面料描述]，[姿势要求]，[场景]，[画质要求]，保持图1的人物样貌不变
```

**示例：**

### Step 5: 消除 AI 感处理

**根据图片类型，添加去 AI 感关键词：**

**人像类加法：**
```
skin pores, realistic skin texture, natural skin imperfections,
subtle freckles, fine lines, skin details,
natural eye reflection, realistic iris, catchlight in eyes
```

**人像类减法（避免）：**
```
beautiful, perfect skin, flawless, doll-like,
overexposed, HDR, oversaturated
```

**产品类加法：**
```
product photography, studio lighting, soft shadows,
realistic material texture, detailed surface
```

**通用负面提示词：**
```
low quality, worst quality, bad anatomy, bad hands, text, error, 
missing fingers, extra digit, fewer digits, cropped, jpeg artifacts,
signature, watermark, blurry, deformed, ugly
```

## 面料识别与描述

### 常见面料类型

| 面料 | 视觉特征 | 描述词 |
|------|----------|--------|
| 纯棉 | 哑光、轻微纹理、自然褶皱 | 柔软棉质面料，自然纹理，透气舒适 |
| 牛仔 | 斜纹纹理、磨白效果、挺括 | 经典牛仔布料，水洗磨白效果，挺括有型 |
| 丝绸 | 柔和光泽、光滑、优雅垂坠 | 光滑丝绸面料，柔和光泽，优雅垂坠 |
| 针织 | 线圈结构、有弹性、柔软 | 细腻针织纹理，罗纹收口，柔软有弹性 |
| 皮革 | 自然纹理、哑光或光泽 | 哑光/光泽皮革质感，自然纹理，柔软耐用 |
| 亚麻 | 明显纹理、清爽透气 | 天然亚麻面料，明显纹理，清爽透气 |
| 羊毛 | 细腻绒毛、温暖质感 | 细腻羊毛面料，温暖质感，柔软有弹性 |
| 雪纺 | 轻薄透明、飘逸 | 轻薄雪纺面料，飘逸透明，微光泽 |
| 蕾丝 | 镂空花纹、半透明 | 精致蕾丝面料，镂空花纹，半透明 |

### 面料询问模板

```
为了更好地还原衣服质感，能否告诉我这是什么面料？
常见选项：
A. 纯棉（哑光、柔软、自然纹理）
B. 丝绸（有光泽、光滑、垂坠感好）
C. 牛仔（挺括、磨白效果）
D. 针织（有弹性、纹理明显）
E. 其他：________

或者提供一张面料特写图，我可以更准确地分析。
```

## 常见错误避免

### ❌ 错误示例

**错误 1：没有使用图引用**
```
韩国风格的小姐姐穿着这件衣服...
```

**错误 2：遗漏面料描述**
```
图1的模特穿着图2的连衣裙...（未描述面料）
```

**错误 3：图片编号混乱**
```
图2的模特穿着图1的衣服...（逻辑错误）
```

### ✅ 正确示例

**正确 1：明确图引用 + 面料描述**
```
图1的模特穿着图2的丝绸连衣裙，面料光滑带有柔和光泽感...
```

**正确 2：逻辑清晰的编号**
```
图1的女生穿着图2的上衣和图3的裤子...
```
