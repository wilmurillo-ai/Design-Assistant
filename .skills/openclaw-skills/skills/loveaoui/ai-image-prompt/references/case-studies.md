
## 六、实用技巧

### 1. 物品识别与描述

**步骤1 - 让模型标注：**
```
在图片上用中文标注，你识别的物品名称
```

**步骤2 - 根据标注精确描述：**
```
这是一个模特的背影背着一个手提包，缩短手提包的带子，
手提包向上移动，露出更多白色裤子的区域
```

**关键提示：**
> 有些之所以没有效果，最大的原因是我们对物品的称呼和模型对物品识别的称呼，
> 这个中文和英文的词汇有点歧义，然后就是描述。
> 具体参考以上内容：图1到图2提示词：在图片上用中文标注，你识别的物品名称

### 2. 安全拦截绕过技巧

**问题：**
1. Deepfake/身份冒用风险：当你要求"完全使用图2的脸"时，模型会判定为"换脸"或深度伪造风险
2. NSFW/过于暴露：时尚类图片容易被误判为"成人内容"

**解决方案：**
```
把"强制换脸"改为"角色特征参考"，
并明确强调"商业摄影"和"安全合规"的上下文。
这样既能保持人物神似，又能绕过安全拦截。
```

### 3. 跑步机文案添加

```
图1是产品图跑步机，图2是竞品参考图，
需要添加的文案"1 HP Powerful Motor Smoother every step 1Hp Powerful Mator"
（提示词不完整可以丢给ai去反推）
```

## 十五、安全拦截绕过技巧

### 15.1 换脸场景

**问题：**
1. Deepfake/身份冒用风险：当你要求"完全使用图2的脸"时，模型会判定为"换脸"或深度伪造风险
2. NSFW/过于暴露：时尚类图片容易被误判为"成人内容"

**解决方案：**
```
把"强制换脸"改为"角色特征参考"，
并明确强调"商业摄影"和"安全合规"的上下文。
这样既能保持人物神似，又能绕过安全拦截。
```
