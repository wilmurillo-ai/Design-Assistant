# 图片逆向推导与提示词复原指南

本文档指导如何分析图片并推导出能够1:1还原的完整提示词。

---

## 一、分析流程

### Step 1: 整体印象

首先快速浏览图片，形成整体印象：
- 这是什么类型的图片？（人像/产品/风景/插画/设计）
- 整体氛围如何？（明亮/昏暗/温馨/冷峻）
- 第一眼注意到什么？（主体/色彩/构图）

### Step 2: 逐层分析

#### 2.1 主体层（最吸引眼球的元素）

**人物类：**
```
数量 → 外貌 → 穿着 → 表情 → 姿势 → 视线
```

**产品类：**
```
类型 → 形状 → 材质 → 颜色 → 状态 → 细节
```

**风景类：**
```
远景 → 中景 → 近景 → 天空 → 地面 → 焦点
```

#### 2.2 环境层（背景和场景）

记录所有可见元素：
- 背景：纯色/渐变/实景/虚化
- 道具：家具、植物、装饰品、建筑
- 空间：室内/室外、封闭/开放

#### 2.3 光影层（光线系统）

分析光线三要素：
- **方向**：从哪来？（左/右/上/下/后）
- **质感**：硬光（阴影锐利）/柔光（阴影柔和）
- **颜色**：暖光（黄/橙）/冷光（蓝/白）

#### 2.4 色彩层（色调系统）

- **主色调**：占画面最大面积的颜色
- **辅助色**：次要但明显的颜色
- **点缀色**：小面积但醒目的颜色
- **饱和度**：高饱和/低饱和/黑白

#### 2.5 技术层（摄影/绘画参数）

- **视角**：平视/俯视/仰视
- **焦距**：广角（变形）/标准/长焦（压缩）
- **景深**：全清晰/浅景深（背景虚化）
- **画幅**：1:1 / 4:3 / 3:4 / 16:9

| 大小 | 占画面比例 | 约占画面1/3 |
| 清晰度 | 焦点状态 | 清晰对焦 |

### 维度8: 文字与图形

| 检查项 | 记录内容 | 示例 |
|--------|----------|------|

## 三、提示词构建

### 构建顺序

```
1. 主体描述（详细）
2. 场景环境（完整）
3. 光线系统（精确）
4. 色彩色调（准确）
5. 构图视角（明确）
6. 质感细节（丰富）
7. 技术参数（专业）
```

### 模板示例

#### 人像类

```
[年龄/性别/种族]的[人物描述]，[发型描述]，
穿着[上衣描述]和[下装描述]，[配饰描述]，
[表情描述]，[姿势描述]，[视线描述]，

位于[场景描述]，[背景描述]，[环境元素]，

[光线类型]从[方向]照入，[光线质感]，[阴影描述]，

[色调描述]，[饱和度描述]，[色彩风格]，

[构图方式]，[视角]，[景别]，[景深效果]，

[质感描述]，[细节描述]，

[摄影设备]，[画质参数]，[艺术风格]
```

#### 产品类

```
[产品类型]的[产品描述]，[材质描述]，[颜色描述]，
[状态描述]，[细节特征]，

放置在[场景描述]，[背景描述]，[道具元素]，

[光线类型]，[光线方向]，[光线质感]，
[高光位置]，[阴影描述]，

[色调描述]，[色彩风格]，

[构图方式]，[视角]，[景深效果]，

[质感描述]，[表面细节]，

[摄影风格]，[画质参数]
```

## 四、1:1还原技巧

### 技巧1: 从大到小
先描述整体，再描述细节：
```
✅ 正确：一位亚洲女性站在室内客厅，穿着白色T恤...
❌ 错误：白色T恤，亚洲女性，室内...
```

### 技巧2: 量化描述
尽可能用具体数值：
```
✅ 正确：约占画面1/3，右侧45度角，f/1.8光圈
❌ 错误：占画面一部分，侧面光，大光圈
```

### 技巧3: 光线精确化
详细描述光线系统：
```
✅ 正确：自然光从右侧窗户照入，柔和的漫射光，
        人物右脸有高光，左侧有柔和阴影
❌ 错误：自然光，柔和光线
```

### 技巧4: 色彩具体化
使用具体的颜色名称：
```
✅ 正确：暖色调，主色为米色和棕色，点缀绿色
❌ 错误：暖色，大地色系
```

### 技巧5: 保留不确定性
对于无法确定的部分，提供选项：
```
"光线可能是自然光或柔光箱，建议尝试：
A. 自然光从窗户照入
B. 柔光箱从右侧打光"
```

## 五、常见难点与解决方案

| 难点 | 原因 | 解决方案 |
|------|------|----------|
| 复杂文字 | AI生成文字不稳定 | 建议后期PS添加 |
| 特定人物 | 难以精确还原面部 | 建议用参考图+换脸 |
| 复杂反射 | 镜面/水面反射难模拟 | 描述反射内容，接受近似 |
| 随机纹理 | 如大理石纹理 | 描述类型，接受自然变化 |
| 精确比例 | 建筑/产品比例 | 提供尺寸参考或接受近似 |
| 特殊效果 | 光晕、镜头光斑 | 描述效果类型，后期可调整 |

## 六、输出示例

### 示例1: 人像照片

**原图描述：**
一张室内人像照片，年轻亚洲女性，长发，穿白色毛衣，坐在窗边，自然光，温馨氛围。

**详细分析：**
```
主体：20-25岁亚洲女性，长发直发黑色，中分
穿着：米白色宽松毛衣，无配饰
表情：自然微笑，眼神温柔
姿势：侧身坐在窗边，双手捧咖啡杯，看向窗外

场景：室内咖啡馆，木质桌椅，背景有书架和绿植
光线：自然光从左侧窗户照入，柔和的侧光，右侧有微弱阴影
色彩：暖色调，米色、棕色、绿色点缀
构图：三分法，人物在左侧三分线，右侧留白
技术：浅景深，背景虚化，f/2.8左右
```

**复原提示词：**
```
A 25-year-old Asian woman with long straight black hair, wearing an oversized 
cream-colored knit sweater, sitting by a window in a cozy cafe, holding a coffee 
cup with both hands, looking out the window with a gentle smile, soft natural 
light streaming from the left window creating a warm side light, wooden furniture, 
bookshelves and green plants in the softly blurred background, warm color palette 
with cream, brown and green accents, shallow depth of field, f/2.8, 85mm lens, 
cozy atmosphere, photorealistic, 8K quality
```

## 七、检查清单

分析完成后，对照以下清单检查：

- [ ] 主体描述完整（外貌、穿着、表情、姿势）
- [ ] 场景描述详细（背景、道具、环境元素）
- [ ] 光线系统精确（方向、质感、阴影）
- [ ] 色彩色调准确（主色、辅助色、点缀色）
- [ ] 构图视角明确（法则、位置、景别）
- [ ] 质感细节丰富（材质、纹理、特效）
- [ ] 技术参数专业（设备、焦距、光圈）
- [ ] 文字图形记录（所有可见文字和Logo）
