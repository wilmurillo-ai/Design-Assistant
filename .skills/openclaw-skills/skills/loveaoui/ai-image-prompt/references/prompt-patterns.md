# 提示词模式模板库 (aouiho 版)

本文档整理了从实际案例中学习到的各种提示词结构模式，可直接套用或参考。

**重要：所有模板中的 "图1"、"图2" 等引用，必须根据用户实际上传的图片数量自动调整。**

---

## 图片引用规范

### 基本规则

1. **自动识别图片数量**：首先确认用户上传了几张图片
2. **按顺序编号**：第一张=图1，第二张=图2，以此类推
3. **明确用途**：在提示词中说明每张图的用途
4. **不要遗漏**：使用所有用户上传的图片

### 引用格式

| 图片数量 | 引用方式 | 示例场景 |
|----------|----------|----------|
| 1 张 | 图1 | `穿着图1的衣服` |
| 2 张 | 图1、图2 | `图1的模特穿着图2的衣服` |
| 3 张 | 图1、图2、图3 | `图1的女生穿着图2的衣服和图3的裤子` |
| 4+ 张 | 图1、图2、图3、图4... | 按需引用 |

### 常见引用模式

**衣服类：**
- `穿着图1的衣服`
- `穿着图1的上衣和图2的裤子`
- `穿着图1的外套，搭配图2的配饰`

**产品类：**
- `对图1的产品进行...`
- `图1的产品放在图2的场景中`
- `图1使用图2的材质`

**人物类：**
- `图1的模特...`
- `图1的女生...`
- `保持图1的人物样貌不变`

**场景类：**
- `图1的背景更换为图2的场景`
- `图1放在图2的场景中`
