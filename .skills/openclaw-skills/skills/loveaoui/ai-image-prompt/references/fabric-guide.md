# 衣服面料识别与描述指南

本文档帮助识别衣服面料类型，并在提示词中准确描述面料特征。

---

## 一、面料识别流程

### Step 1: 视觉分析

观察图片中的以下特征：

| 观察项 | 判断要点 |
|--------|----------|
| **光泽度** | 哑光 / 微光 / 高光 / 镜面反光 |
| **纹理** | 平滑 / 粗糙 / 编织纹理 / 印花 |
| **厚度** | 轻薄 / 中等 / 厚重 |
| **垂坠感** | 挺括 / 自然垂落 / 飘逸 |
| **透明度** | 不透明 / 半透明 / 透明 |
| **弹性** | 无弹 / 微弹 / 高弹 |

### Step 2: 面料类型判断

根据观察结果，判断面料类型：

#### 天然纤维

| 面料 | 视觉特征 | 触感描述 |
|------|----------|----------|
| **纯棉** | 哑光、轻微纹理、自然褶皱 | 柔软、透气、吸湿 |
| **亚麻** | 明显纹理、轻微光泽、易皱 | 清爽、透气、有骨感 |
| **丝绸** | 柔和光泽、光滑表面、优雅垂坠 | 丝滑、凉爽、柔软 |
| **羊毛** | 细腻绒毛、温暖质感、微弹性 | 保暖、柔软、有弹性 |
| **羊绒** | 细腻柔软、轻微光泽、高级质感 | 极柔软、保暖、轻盈 |
| **真皮** | 自然纹理、哑光或光泽、挺括 | 柔软、耐用、有质感 |

#### 化学/合成纤维

| 面料 | 视觉特征 | 触感描述 |
|------|----------|----------|
| **聚酯纤维** | 光泽感强、平滑、不易皱 | 滑爽、快干、耐用 |
| **尼龙** | 光泽感、轻薄、耐磨 | 光滑、轻盈、有弹性 |
| **腈纶** | 类似羊毛、蓬松、保暖 | 柔软、保暖、轻盈 |
| **氨纶** | 高弹性、贴身、光泽 | 弹性极佳、贴身舒适 |

#### 混纺面料

| 面料 | 特点 | 常见用途 |
|------|------|----------|
| **棉涤混纺** | 结合棉的舒适和涤纶的耐用 | T恤、衬衫 |
| **毛涤混纺** | 结合羊毛的保暖和涤纶的易打理 | 西装、外套 |
| **丝棉混纺** | 结合丝绸的光泽和棉的舒适 | 高档衬衫、连衣裙 |

#### 特殊面料

| 面料 | 视觉特征 | 常见用途 |
|------|----------|----------|
| **牛仔布** | 斜纹纹理、靛蓝色、磨白效果 | 牛仔裤、外套 |
| **针织布** | 线圈结构、有弹性、柔软 | T恤、毛衣 |
| **蕾丝** | 镂空花纹、精致、半透明 | 女装、内衣 |
| **雪纺** | 轻薄透明、飘逸、微光泽 | 连衣裙、衬衫 |
| **灯芯绒** | 纵向条纹、绒面质感、复古 | 裤子、外套 |
| **天鹅绒** | 浓密绒毛、光泽感、奢华 | 礼服、家居服 |

## 三、面料细节询问模板

当图片中面料细节不清晰时，使用以下方式询问用户：

### 方式 1：直接询问面料类型
```
为了更好地还原衣服质感，能否告诉我这是什么面料？
常见选项：纯棉、牛仔、丝绸、针织、皮革、毛绒、亚麻、羊毛等
```

### 方式 2：请求面料特写图
```
为了更好地还原面料质感，能否提供一张面料的特写图？
特别是能看清纹理和光泽的部分，这样生成的效果会更真实
```

### 方式 3：提供选项让用户选择
```
根据图片，我初步判断这可能是以下面料之一：
A. 纯棉（哑光、柔软、自然纹理）
B. 丝绸（有光泽、光滑、垂坠感好）
C. 针织（有弹性、纹理明显、柔软）
D. 其他：________

请帮我确认一下，这样提示词会更准确
```

## 四、面料在提示词中的应用

### 示例 1：单件衣服

**用户上传：** 卫衣图片
**面料分析：** 棉质、柔软、微起球纹理
**生成提示词：**
```
韩国潮流风格的可爱亚洲小姐姐，穿着图1的棉质卫衣，
面料柔软带有自然纹理和轻微起球感，宽松版型，
在时尚街头摆着自然的姿势，4K超清，胶片质感
```

### 示例 2：模特换装

**用户上传：** 图1=模特，图2=丝绸连衣裙
**面料分析：** 丝绸、光滑、微光泽、优雅垂坠
**生成提示词：**
```
图1的模特穿着图2的丝绸连衣裙，面料光滑带有柔和光泽感，
优雅垂坠，贴身剪裁，姿势为专业模特姿势，
室内简约背景，柔和灯光，电商产品图风格，
保持图1模特的样貌和发型不变
```

### 示例 3：多件衣服组合

**用户上传：** 图1=模特，图2=牛仔外套，图3=棉质T恤
**面料分析：**
- 图2：牛仔布、挺括、磨白效果
- 图3：纯棉、柔软、自然纹理
**生成提示词：**
```
图1的模特穿着图2的牛仔外套和图3的棉质T恤，
外套为挺括的牛仔布料带有磨白效果，
内搭为柔软的纯棉面料带有自然纹理，
层次搭配，街头风格姿势，城市背景，4K超清
```

## 五、面料保持原则

### 必须做到：

1. **识别面料类型** - 准确判断是棉、丝、牛仔还是其他
2. **描述面料特征** - 光泽、纹理、厚度、垂坠感
3. **使用专业词汇** - 在提示词中加入面料描述词
4. **保持面料一致性** - 确保生成结果与参考图面料相符

### 避免的错误：

1. ❌ 忽略面料描述 - 只写"衣服"而不提面料
2. ❌ 面料判断错误 - 把丝绸说成棉质
3. ❌ 描述过于笼统 - "好看的衣服"而不是"柔软的棉质卫衣"
4. ❌ 不询问确认 - 图片不清时直接猜测而不问用户

## 六、常见面料问题排查

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 生成结果面料质感不对 | 面料描述不准确 | 重新分析图片，询问用户确认 |
| 面料光泽度不符 | 光泽描述缺失 | 加入"哑光"/"光泽"/"微光"等描述 |
| 纹理细节丢失 | 纹理描述不够具体 | 加入具体纹理词如"斜纹"/"罗纹"/"编织纹理" |
| 垂坠感不自然 | 未描述垂坠特性 | 加入"挺括"/"垂坠"/"飘逸"等词 |
