### 各术语实操详解

#### FOB（船上交货）- 最常用

**卖方责任：**
- 出口清关（报关、商检、产地证）
- 将货物装上买方指定船只
- 承担装船前一切费用和风险

**买方责任：**
- 租船订舱，支付运费
- 购买保险
- 承担货物装船后风险
- 进口清关

**报价构成：**
```
FOB Price = 
  出厂价
  + 国内运输费
  + 港口杂费
  + 报关费
  + 港杂费
  + 利润

= Ex-factory + Local charges + Export clearance
```

**使用场景：**
- 海运/内河运输
- 买方有优势运费资源
- 大宗货物，买方整船租赁
- 客户熟悉目的港操作

**风险提示：**
⚠️ 装船后风险转移，但建议卖方购买装船前保险
⚠️ 指定货代可能存在无单放货风险，需核实资质

---

#### CIF（成本加保险费运费）- 最常用

**卖方责任：**
- FOB全部责任
- 租船订舱，支付运费
- 购买最低险别海运保险（通常CIC平安险）

**买方责任：**
- 承担货物装船后风险（注意：风险转移点在装运港，非目的港）
- 进口清关

**报价构成：**
```
CIF Price = 
  FOB价格
  + 海运费
  + 保险费
  
海运费计算：根据体积/重量，取较大者（W/M）
保险费计算：CIF × 110% × 保险费率

常用保险费率参考：
• 平安险(FPA): 0.15-0.25%
• 水渍险(WPA): 0.25-0.35%
• 一切险(All Risks): 0.35-0.5%
```

**使用场景：**
- 客户不熟悉运输安排
- 卖方有优势运费资源
- 便于卖方控制船期

**风险提示：**
⚠️ 风险在装运港已转移，但卖方需付运费和保险
⚠️ 建议投保一切险，降低纠纷风险
⚠️ 保险索赔通常由买方提出，卖方协助

---

#### DDP（完税后交货）- 卖方责任最大

**卖方责任：**
- 全程运输和费用
- 出口和进口清关
- 支付关税、增值税等所有税费
- 承担全程风险至目的地

**买方责任：**
- 在目的地接收货物

**报价构成：**
```
DDP Price = 
  出厂价
  + 国内费用
  + 海运费
  + 保险费
  + 目的港费用
  + 进口清关费
  + 关税
  + 增值税
  + 目的国内陆运输
  + 利润和风险溢价
```

**使用场景：**
- 客户完全不想处理进口事务
- 卖方有目的国代理和清关能力
- 样品、小批量订单

**风险提示：**
⚠️ 需要熟悉目的国海关法规和税率
⚠️ 汇率波动影响关税计算
⚠️ 建议加收DDP风险溢价（3-5%）
⚠️ 部分国家限制DDP（如巴西）

### T/T（电汇）详解

#### 标准操作流程

```
【30%预付 + 70%见提单副本】流程：

Step 1: 签合同，PI确认
        ↓
Step 2: 买方支付30%定金（Down Payment）
        卖方确认收款后开始生产
        ↓
Step 3: 生产完成，验货通过
        ↓
Step 4: 卖方安排装船，获取提单(B/L)
        ↓
Step 5: 卖方发送提单副本给买方
        买方确认无误
        ↓
Step 6: 买方支付70%尾款
        ↓
Step 7: 卖方确认收款后，寄送全套单据
        或电放提单(Telex Release)
        ↓
Step 8: 买方凭单据提货
```

#### T/T注意事项

**收款信息准确性：**
```
必须确认：
✓ SWIFT/BIC代码（银行识别码）
✓ 银行名称和地址
✓ 账户名称（必须与合同一致）
✓ 账号/IBAN
✓ 中转行信息（如有）

常见错误：
× 账号数字错误（特别是IBAN国家）
× 账户名称拼写错误
× 漏写中转行导致延迟
```

**到账时间参考：**
```
• 同银行转账：当日
• 境内跨行：1-2工作日
• 亚洲区域内：2-3工作日
• 欧美到亚洲：3-5工作日
• 涉及中转行：+1-2工作日
• 非洲/南美：5-7工作日或更长
```

**防范欺诈要点：**
⚠️ 收款账号变更需电话+邮件双重确认
⚠️ 警惕"付款截图"造假，以银行到账为准
⚠️ 新客户建议分批付款验证真实性

### L/C（信用证）详解

#### 信用证基本流程

```
        买方(申请人)               卖方(受益人)
            │                          │
            │ ①申请开证               │
            ▼                          │
        开证行 ──────────────────► 通知行
            │     ②开立信用证          │
            │                          │
            │                          ▼
            │                    ③审核并准备单据
            │                          │
            │ ④交单                  ④交单
            │◄─────────────────────────│
            │                          │
            ▼                          │
        ⑤审单付款/承兑                │
            │                          │
            │ ⑥付款                   │
            └─────────────────────────►
```

#### 信用证关键条款审核清单

```
【必须审核的要点】

1. 基本信息
   □ 信用证类型：可撤销/不可撤销（必须不可撤销）
   □ 开证行资信是否可接受
   □ 有效期和交单期是否合理
   □ 到期地点：受益人所在国（最好）

2. 金额与货物
   □ 金额大小写一致
   □ 币种正确
   □ 数量溢短装条款（通常±5-10%）
   □ 价格术语确认

3. 单据要求
   □ 商业发票要求（份数、签署）
   □ 提单要求（全套/副本、收货人、通知方）
   □ 保险单（如CIF/CFR）
   □ 产地证（类型、签发机构）
   □ 质检证（标准、机构）
   □ 包装单、重量单等

4. 装运条款
   □ 装运港和目的港明确
   □ 分批装运允许/禁止
   □ 转运允许/禁止
   □ 最迟装运期

5. 软条款识别（必须删除或修改）
   ✗ 客检证作为议付单据
   ✗ 1/3提单直寄买方
   ✗ 货到目的港后付款
   ✗ 开证行单方面可随时撤销
   ✗ 单据需经开证申请人确认
```

#### 单据制作要求

**常见单据不符点：**
```
1. 商业发票
   ✗ 金额超证或低于证
   ✗ 货物描述与L/C不符
   ✗ 未显示价格术语
   ✗ 签署不符合要求

2. 提单
   ✗ 收货人(Consignee)不符合要求
   ✗ 通知方(Notify Party)错误
   ✗ 运费未按L/C显示"FREIGHT PREPAID/TO COLLECT"
   ✗ 装船批注日期晚于最迟装运期
   ✗ 提交副本而非全套正本

3. 保险单
   ✗ 币种与L/C不一致
   ✗ 保险金额不足110%发票金额
   ✗ 险别不符（如要求All Risks只保FPA）
   ✗ 出单日期晚于提单日期

4. 产地证
   ✗ 签发机构不符（如要求商会签发但由贸促会签发）
   ✗ 显示制造商与L/C不符
```

### 海运实操指南

#### 集装箱类型与装载

| 箱型 | 内尺寸(长×宽×高) | 容积 | 限重 | 适用货物 |
|-----|------------------|------|------|---------|
| **20'GP** | 5.9×2.35×2.39m | 33 CBM | 21.5吨 | 重货、机器 |
| **40'GP** | 12.03×2.35×2.39m | 67 CBM | 26.5吨 | 普通货物 |
| **40'HQ** | 12.03×2.35×2.69m | 76 CBM | 26.2吨 | 轻泡货 |
| **45'HQ** | 13.56×2.35×2.70m | 86 CBM | 27.5吨 | 大量轻货 |

**装柜计算：**
```
计算方法：
1. 计算单箱体积：长×宽×高（立方米）
2. 计算总数量可装：集装箱容积 ÷ 单箱体积
3. 考虑包装间隙（通常留10-15%空间）
4. 检查重量限制

示例：
产品包装：50×40×30cm = 0.06 CBM
40HQ容积：76 CBM
理论可装：76 ÷ 0.06 = 1266箱
考虑间隙：1266 × 0.85 = 1076箱
检查重量：1076箱 × 10kg/箱 = 10.76吨（OK）
```

#### 海运费用构成

```
海运费 = 
  基本运费(O/F) + 
  燃油附加费(BAF) +
  币值附加费(CAF) +
  旺季附加费(PSS) +
  港口拥挤附加费(PCS) +
  其他（根据航线/时期）

目的港费用：
  码头操作费(THC) +
  文件费(DOC) +
  清关费 +
  拆箱费(LCL) +
  内陆运输费
```

### 空运实操指南

#### 计费重量计算

```
空运按"计费重量"收费，取实际重量和体积重量较大者：

体积重量 = 长(cm) × 宽(cm) × 高(cm) ÷ 6000

示例：
实际重量：50kg
包装尺寸：80×60×50cm
体积重量：80×60×50÷6000 = 40kg

计费重量：50kg（实际重量更大）

另一例：
实际重量：30kg
包装尺寸：100×80×60cm
体积重量：100×80×60÷6000 = 80kg

计费重量：80kg（体积重量更大）

技巧：
对于轻泡货，尽量压缩包装尺寸可节省运费
```

## 第五部分：退税实务

### 中国出口退税基础

#### 退税率查询

```
查询渠道：
1. 国家税务总局官网
2. 出口退税咨询网
3. 各地税务局网站
4. 专业退税软件

注意：
• 退税率会调整，以最新政策为准
• 不同HS编码对应不同退税率
• 部分产品可能取消退税
```

#### 退税条件

```
基本条件：
✓ 增值税一般纳税人
✓ 实际出口并收汇
✓ 货物已报关离境
✓ 财务已做销售处理
✓ 在退税申报期内申报

退税申报期限：
• 一般货物：出口次年4月增值税申报期截止前
• 逾期未申报：视同内销征税
```

#### 退税计算

```
应退税额 = 出口货物不含税价格 × 退税率

或：
应退税额 = 增值税专用发票进项税额（以较小者为准）

示例：
出口金额：$100,000
汇率：7.0
FOB人民币价：700,000元
退税率：13%

应退税额 = 700,000 × 13% = 91,000元
```

### 退税操作SOP

```
【出口前准备】
1. 确认供货商可开增值税专用发票
2. 确认退税率
3. 确保报关单经营单位与退税单位一致

【出口时】
4. 正确填报报关单
   - 成交方式、价格准确
   - HS编码正确
   - 商品名称规范

【收汇后】
5. 外汇到账后结汇
6. 准备退税单证：
   - 增值税专用发票
   - 出口货物报关单（退税联）
   - 出口收汇核销单（如需要）
   - 出口发票

【申报】
7. 在退税系统中录入数据
8. 生成申报数据
9. 提交税务局审核
10. 等待退税款到账（通常1-3个月）
```

### 客户信用风险

#### 风险评估维度

```
【基础信息】
□ 公司注册信息（成立时间、注册资本）

【第三方验证】
□ 银行资信证明
□ 中信保资信报告
□ 行业口碑调查
□ 其他供应商反馈
```

#### 风险控制措施

| 风险等级 | 客户特征 | 建议付款方式 | 信用额度 |
|---------|---------|-------------|---------|
| **A级** | 知名大客户、多年合作 | O/A 30-90天 | 可高 |
| **B级** | 中等规模、合作良好 | L/C或T/T定金 | 中等 |
| **C级** | 新客户、小规模 | T/T预付或定金 | 低 |
| **D级** | 高风险地区、资信不明 | 100%预付或中信保 | 谨慎 |

### 汇率风险管理

```
【锁汇工具】
1. 远期外汇合约
   - 与银行约定未来汇率
   - 锁定成本，消除不确定性

2. 外汇期权
   - 支付期权费获得选择权
   - 汇率有利可放弃执行

3. 自然对冲
   - 用外币支付进口
   - 减少外汇敞口

【操作建议】
• 预测汇率走势，选择合适工具
• 分批锁汇，降低判断错误风险
• 避免过度投机，以保值为目的
```
