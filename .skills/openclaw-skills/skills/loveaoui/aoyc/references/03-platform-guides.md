### Listing优化黄金法则

#### 标题优化公式
```
[品牌名] + [核心关键词] + [材质/功能] + [规格] + [适用场景] + [型号]

示例：
Nike Air Zoom Running Shoes Men's Lightweight Breathable Athletic 
Sneakers Size 10 Black

限制：200字符（标题搜索优化权重最高）
```

**标题关键词布局优先级：**
1. 前80字符：核心关键词+品牌（移动端显示）
2. 80-130字符：次要关键词+材质功能
3. 130-200字符：长尾词+规格参数

#### 五点描述（Bullet Points）结构
```
1. 核心卖点/差异化价值（情感触发）
   示例：Stay cool and comfortable all day with our breathable 
   mesh upper that adapts to your foot shape

2. 材质/工艺/品质（理性支撑）
   示例：Premium synthetic leather with reinforced stitching for 
   durability that lasts 2x longer than standard shoes

3. 功能特性/使用场景（使用想象）
   示例：Perfect for running, gym workouts, or casual wear - 
   versatile design matches any outfit

4. 尺寸规格/包装内容（消除疑虑）
   示例：Available in sizes 6-15 including half sizes. Each pair 
   comes with extra laces and a storage bag

5. 售后保障/品牌承诺（降低风险）
   示例：30-day money-back guarantee. 1-year warranty against 
   manufacturing defects. 24/7 customer support
```

**Bullet Points最佳实践：**
- 每个bullet 200-250字符
- 首字母大写，句末不加句号
- 每点一个核心卖点
- 包含1-2个关键词
- 数字具体化（不用"high quality"，用"300-thread count cotton"）

#### 后台搜索词（Backend Keywords）
```
规则：
• 250 bytes限制（约50个单词）
• 无需重复标题和bullet中的词
• 不使用竞品品牌名
• 不使用ASIN
• 不使用临时性词汇（"new", "on sale"）
• 用空格分隔，不用逗号
• 包含同义词、拼写变体、使用场景词

示例：
running shoe trainer athletic footwear gym workout fitness 
sneaker jogging walking sports exercise men breathable lightweight
```

### 广告投放策略详解

#### 自动广告四个组别解析

| 组别 | 匹配逻辑 | 目的 | 建议竞价 |
|-----|---------|------|---------|
| **Close Match** | 与产品高度相关 | 精准流量，测试核心词 | 较高 |
| **Loose Match** | 宽泛相关 | 拓词，发现新机会 | 中等 |
| **Substitutes** | 同类竞品 | 竞品拦截 | 中等 |
| **Complements** | 互补产品 | 关联流量 | 较低 |

**自动广告操作SOP：**
1. 新品上架开启所有四个组别
2. 跑2周后下载搜索词报告
3. 高转化词加入手动广告
4. 无关词加入否定关键词
5. 调整竞价优化ACOS

#### 手动广告匹配方式

| 匹配类型 | 展示条件 | 流量质量 | 使用场景 |
|---------|---------|---------|---------|
| **Broad** | 相关变体、同义词 | 泛 | 拓词阶段 |
| **Phrase** | 短语前后加词 | 中 | 控制流量 |
| **Exact** | 完全一致 | 精准 | 推排名、保利润 |

**广告架构金字塔：**
```
Layer 1 (顶部): Exact匹配 - 高转化词 - 推排名
Layer 2 (中部): Phrase匹配 - 中等词 - 稳定流量
Layer 3 (底部): Broad/Auto - 拓词 - 发现机会
```

#### 广告指标基准

| 指标 | 健康值 | 优秀值 | 优化建议 |
|-----|-------|-------|---------|
| **ACOS** | <30% | <20% | 高于30%需优化 |
| **CTR** | >0.3% | >0.5% | 优化主图和标题 |
| **CVR** | >10% | >15% | 优化listing详情 |
| **Impressions** | 持续增长 | - | 提高竞价或拓词 |

### FBA库存管理

#### 关键指标管理

| 指标 | 计算公式 | 健康范围 | 优化建议 |
|-----|---------|---------|---------|
| **库存周转天数** | 平均库存 / 日均销量 | 30-60天 | 超90天需促销 |
| **售罄率** | 售出 / 入库 | 60-80% | 避免断货或积压 |
| **库存绩效指标(IPI)** | 亚马逊算法 | >400 | 低于400限制仓储 |

#### FBA费用结构

```
FBA总费用 = 
  配送费 (基于尺寸重量) +
  仓储费 (按月/立方英尺) +
  长期仓储费 (超过365天) +
  移除/弃置费 (如需) +
  退货处理费 (如发生)

尺寸分段：
• 小标准：≤15"×12"×0.75", ≤12oz
• 大标准：≤18"×14"×8", ≤20lbs
• 小号大件：≤60"×30", ≤70lbs
• 中号大件：≤108"×130", ≤150lbs
• 大号大件：>108"或>150lbs
```

### Review管理

**获取Review的合规方法：**
1. Amazon Vine计划（新品）
2. 订单后跟进邮件（请求反馈，非评价）
3. 包装内插页（引导到客服，非直接留评）
4. 优质产品自然获得

**处理差评：**
- 24小时内响应
- 专业礼貌，承认问题
- 提供解决方案
- 尝试联系客户（通过订单消息）
- 举报恶意差评（如有证据）

## Temu平台运营全攻略

### 全托管 vs 半托管对比

| 维度 | 全托管 | 半托管 |
|-----|-------|-------|
| **运营责任** | 平台负责 | 平台负责 |
| **定价权** | 平台主导 | 平台参考，可协商 |
| **供货方式** | 备货到平台仓 | 海外仓发货 |
| **物流责任** | 平台负责 | 卖家负责头程 |
| **售后客服** | 平台负责 | 平台负责 |
| **入驻门槛** | 低 | 需海外仓资源 |
| **利润水平** | 薄利 | 稍高 |
| **适合对象** | 工厂/供应链 | 有海外仓的卖家 |

### 核价策略

**影响核价通过的因素：**
1. **价格竞争力**：与竞品对比
2. **产品质量**：退货率指标
3. **供货稳定性**：库存深度
4. **响应速度**：发货时效
5. **合规性**：资质齐全

**提高核价通过率技巧：**
- 提供成本明细（材料、人工、包装）
- 展示工厂实力（产能、设备、认证）
- 说明产品差异化（材质、工艺、设计）
- 提供竞品对比分析

### 选品策略

**Temu偏好产品特征：**
- 低客单价（$5-20最佳）
- 轻小件（物流成本低）
- 非季节性（稳定销售）
- 标准化（少售后）
- 高频消费（复购率）

**避免品类：**
- 大件重货（物流成本高）
- 高售后率（服装尺码等）
- 侵权风险（知名品牌款）
- 资质门槛高（食品、化妆品）

### 供货与库存管理

```
供货流程：
1. 产品上架（等待核价）
2. 核价通过，创建首单
3. 备货至Temu仓库
4. 平台销售
5. 补货提醒

库存管理要点：
• 保持安全库存（7-14天销量）
• 及时响应补货提醒
• 关注滞销预警
• 控制退货率（影响评级）
```

## 合规与风险控制

### 账号安全

| 风险 | 预防措施 |
|-----|---------|
| **关联封号** | 独立IP、独立设备、独立资料 |
| **侵权投诉** | 自查IP、商标注册、授权链完整 |
| **绩效超标** | 监控ODR、准时发货率、有效追踪率 |
| **虚假评论** | 合规获取评价，不操纵评论 |

- **美国**：销售税（Nexus州需申报）
- **欧洲**：VAT注册和申报
- **英国**：UK VAT
- **记录保存**：完整交易记录、发票

### 产品合规
