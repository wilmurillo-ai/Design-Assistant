## 适用于任何运输方式的术语详解

### EXW (Ex Works) - 工厂交货

**定义：** 卖方在其所在地或其他指定地点（如工厂、仓库）将货物交给买方处置时，即完成交货。

**责任划分：**

| 责任 | 卖方 | 买方 |
|-----|------|------|
| 出口清关 | ❌ | ✅ |
| 装货 | ❌ | ✅ |
| 运输（出口国） | ❌ | ✅ |
| 出口报关 | ❌ | ✅ |
| 主运费 | ❌ | ✅ |
| 保险 | ❌ | ✅ |
| 进口报关 | ❌ | ✅ |
| 进口税费 | ❌ | ✅ |
| 运输（进口国） | ❌ | ✅ |

**适用场景：**
- 买方在出口国有强大的物流能力
- 买方希望完全控制运输过程
- 国内贸易或买方熟悉出口流程

**风险提示：**
- ⚠️ 买方承担最大责任和风险
- ⚠️ 卖方协助出口清关不是义务，只是协助
- ⚠️ 买方需确保有能力完成出口报关

---

### FCA (Free Carrier) - 货交承运人

**定义：** 卖方在指定地点将货物交给买方指定的承运人，即完成交货。

**两种交货地点：**

| 交货地点 | 卖方责任 | 适用情况 |
|---------|---------|---------|
| 卖方所在地 | 负责装货 | 工厂/仓库交货 |
| 其他地点 | 不负责卸货 | 港口/机场/物流园 |

**FCA vs FOB 的优势：**
- ✅ 适用于任何运输方式
- ✅ 卖方负责出口清关
- ✅ 风险转移点更明确
- ✅ 集装箱运输推荐使用FCA而非FOB

**适用场景：**
- 多式联运（海运+铁路+公路）
- 集装箱运输
- 卖方需控制出口报关

---

### CPT (Carriage Paid To) - 运费付至

**定义：** 卖方将货物交给承运人，并支付运至指定目的地的运费，风险在交货给承运人时转移。

**关键特点：**
- 卖方支付主运费
- 风险在出口国转移（与CIF/CFR不同）
- 适用于任何运输方式

**CPT vs CFR/CIF：**
| 对比项 | CPT | CFR/CIF |
|-------|-----|---------|
| 适用运输 | 任何方式 | 海运/内河 |
| 风险转移 | 交给承运人 | 装运港船上 |
| 运费支付 | 卖方付至目的地 | 卖方付至目的港 |

---

### CIP (Carriage and Insurance Paid To) - 运费保险费付至

**定义：** 与CPT相同，但卖方还需办理货物运输保险。

**保险要求（Incoterms® 2020更新）：**
- 最低险别：一切险（All Risks）
- 保险金额：合同价款的110%
- 币种：合同货币

**CIP vs CIF：**
- CIP适用于任何运输方式
- CIF仅适用于海运
- CIP保险为一切险，CIF为平安险

---

### DAP (Delivered at Place) - 目的地交货

**定义：** 卖方将货物运至指定目的地，在到达的运输工具上准备卸货时，即完成交货。

**卖方责任：**
- 出口清关 ✅
- 主运费 ✅
- 进口国运输 ✅
- 进口清关 ❌（买方负责）
- 进口税费 ❌（买方负责）

**适用场景：**
- 卖方有进口国物流能力
- 买方不愿处理进口手续
- 跨境电商海外仓模式

---

### DPU (Delivered at Place Unloaded) - 目的地卸货后交货

**定义：** 卖方在指定目的地卸货后将货物交给买方处置，即完成交货。

**DPU vs DAP：**
- DPU：卖方负责卸货
- DAP：卖方不负责卸货

**DPU vs DAT：**
- Incoterms® 2020将DAT改为DPU
- DPU范围更广，不限于"终端"

---

### DDP (Delivered Duty Paid) - 完税后交货

**定义：** 卖方承担将货物运至目的地的一切费用和风险，包括进口清关和税费。

**卖方责任：**
- 出口清关 ✅
- 主运费 ✅
- 进口清关 ✅
- 进口税费 ✅
- 进口国运输 ✅

**风险提示：**
- ⚠️ 卖方承担最大责任和风险
- ⚠️ 卖方需有能力完成进口清关
- ⚠️ 某些国家限制外国公司进口

**适用场景：**
- 卖方在进口国有分公司/代理
- 卖方非常熟悉进口国法规
- 买方希望"一站式"服务

## 适用于海运和内河运输的术语详解

### FAS (Free Alongside Ship) - 船边交货

**定义：** 卖方在指定装运港将货物放置于船边（如码头或驳船上），即完成交货。

**适用场景：**
- 大宗散货（矿石、粮食等）
- 船边直接装船
- 较少用于集装箱运输

**注意：**
- 卖方不负责装船费用
- 风险在船边转移

---

### FOB (Free On Board) - 装运港船上交货

**定义：** 卖方在指定装运港将货物装上买方指定的船舶，即完成交货。

**FOB费用划分：**

| 费用项目 | 卖方 | 买方 |
|---------|------|------|
| 货物成本 | ✅ | ❌ |
| 内陆运费 | ✅ | ❌ |
| 装港杂费 | ✅ | ❌ |
| 装船费 | ✅ | ❌ |
| 海运费 | ❌ | ✅ |
| 保险费 | ❌ | ✅ |
| 目的港杂费 | ❌ | ✅ |
| 进口税费 | ❌ | ✅ |

**FOB变形（装船费用划分）：**

| 变形 | 含义 | 适用 |
|-----|------|------|
| FOB Liner Terms | 班轮条件，卖方不付装船费 | 班轮运输 |
| FOB Under Tackle | 吊钩下交货，卖方付至吊钩下 | 传统 |
| FOB Stowed | 包括理舱费 | 杂货 |
| FOB Trimmed | 包括平舱费 | 散货 |
| FOBST | 包括理舱和平舱费 | 散货 |

**FOB注意事项：**
1. **船货衔接**：买方应及时派船，卖方应及时备货
2. **装船通知**：卖方装船后必须及时通知买方
3. **集装箱运输**：建议使用FCA而非FOB

**FOB实战案例：**
```
案例：FOB Shanghai
卖方责任：
- 将货物运至上海港
- 完成出口清关
- 将货物装上买方指定船舶
- 承担装船前的一切风险

买方责任：
- 租船订舱，通知卖方船名装期
- 支付海运费
- 购买保险
- 承担装船后的一切风险
```

---

### CFR (Cost and Freight) - 成本加运费

**定义：** 卖方将货物装上船，并支付运至目的港的运费，风险在装运港转移。

**CFR关键要点：**
- 卖方支付海运费
- 风险在装运港转移（与运费支付分离）
- 卖方必须及时发出装船通知

**装船通知的重要性：**
```
CFR术语下，装船通知至关重要！

原因：
- 风险在装运港转移给买方
- 买方需要及时购买保险
- 如果卖方未及时通知，货物损失由卖方承担

通知内容：
- 船名航次
- 装船日期
- 货物详情
- 预计到达时间
```

**CFR变形（卸货费用划分）：**

| 变形 | 含义 | 适用 |
|-----|------|------|
| CFR Liner Terms | 班轮条件，卖方不付卸货费 | 班轮 |
| CFR Landed | 包括卸货费 | 买方不便 |
| CFR Ex Tackle | 吊钩下交货 | 传统 |
| CFR Ex Ship's Hold | 舱底交货，买方付卸货费 | 散货 |

---

### CIF (Cost Insurance and Freight) - 成本保险费加运费

**定义：** 与CFR相同，但卖方还需购买最低限度的海运保险。

**CIF保险要求：**
- 险别：最低为平安险（FPA）
- 金额：合同价款的110%
- 币种：合同货币
- 可转让：保险单应可背书转让

**CIF注意事项：**
1. **象征性交货**：凭单交货、凭单付款
2. **保险最低险别**：如买方需要更高险别，应在合同中明确
3. **运费到付**：卖方付运费，但风险已转移

**CIF vs FOB 选择指南：**

| 考虑因素 | 选择FOB | 选择CIF |
|---------|--------|--------|
| 运输控制 | 买方希望控制运输 | 卖方希望控制运输 |
| 新客户 | - | 卖方控制货权，更安全 |
| 老客户 | 买方获得更好运费 | - |
| 运费波动 | 买方承担风险 | 卖方承担风险 |
| 货代关系 | 买方有优势货代 | 卖方有优势货代 |

## 术语选择决策框架

### 决策树

```
是否需要卖方控制运输？
├── 是 → 选择 CIF/CIP/DDP
└── 否 → 买方是否熟悉出口流程？
    ├── 是 → 选择 FOB/FCA/EXW
    └── 否 → 选择 FCA/CIF

是否需要门到门服务？
├── 是 → 选择 DAP/DPU/DDP
└── 否 → 标准海运术语

是否集装箱运输？
├── 是 → 优先选择 FCA/CPT/CIP
└── 否 → FOB/CFR/CIF 可用

新客户 vs 老客户？
├── 新客户 → CIF（卖方控制货权）
└── 老客户 → FOB/CIF 均可
```

## 报价中的价格构成

### FOB报价构成

```
FOB Price = 
  产品成本
  + 利润
  + 国内运费（工厂→港口）
  + 港口杂费（THC, DOC, ORC等）
  + 报关费
  + 银行费用
  + 其他杂费
```

### CIF报价构成

```
CIF Price = 
  FOB Price
  + 海运费（O/F）
  + 附加费（BAF, CAF, THC等）
  + 保险费
  
保险费计算：
Insurance = CIF × 110% × 保险费率
（保险费率：平安险约0.1-0.3%，一切险约0.3-0.5%）
```

### 价格换算公式

```
FOB → CIF：
CIF = (FOB + 运费) / (1 - 1.1 × 保险费率)

CFR → CIF：
CIF = CFR / (1 - 1.1 × 保险费率)

CIF → FOB：
FOB = CIF × (1 - 1.1 × 保险费率) - 运费
```

## 常见错误与误区

### 术语误用案例

**案例1：集装箱运输使用FOB**
```
❌ 错误：
"We quote FOB Shanghai for container shipment."

✅ 正确：
"We quote FCA Shanghai for container shipment."

原因：
FOB风险转移点是"装运港船上"，但集装箱在堆场就已交给承运人。
使用FOB会导致风险划分不清。
```

**案例2：目的港费用争议**
```
❌ 错误：
合同写CIF，但目的港THC费用争议

✅ 正确：
明确CIF Landed或CIF Ex Ship's Hold

原因：
CIF本身不规定卸货费，需用变形明确。
```

**案例3：DDP的进口清关**
```
❌ 错误：
中国卖方对德国买方报DDP，但无德国进口资质

✅ 正确：
改为DAP，或确保有德国进口代理

原因：
DDP要求卖方完成进口清关，某些国家对外国公司进口有限制。
```

### 术语使用禁忌

| 禁忌 | 原因 | 替代方案 |
|-----|------|---------|
| 集装箱用FOB | 风险点不清 | 使用FCA |
| 滚装船用FOB | 装船点不明 | 使用FCA |
| 空运用FOB/CIF | 不适用 | 使用FCA/CPT/CIP |
| 陆运用FOB/CIF | 不适用 | 使用FCA/CPT/CIP |
| 省略港口名称 | 地点不明 | FOB Shanghai, CIF Hamburg |

## 合同条款中的术语应用

### 标准报价条款模板

```
Price Terms: FOB/CIF/DDP [Port/Place]
Currency: USD/EUR
Payment Terms: 30% T/T deposit, 70% against B/L copy / L/C at sight
Delivery Time: XX days after deposit received
Shipping Port: [Port name]
Destination: [Port/Place name]
Insurance: To be covered by [Seller/Buyer] for 110% of invoice value
```

### 术语与付款方式搭配

| 术语 | 推荐付款方式 | 风险等级 |
|-----|-------------|---------|
| EXW | 100%预付 | 卖方安全 |
| FOB/FCA | 30%预付+70%见提单副本 | 平衡 |
| CFR/CIF | 30%预付+70%即期LC | 卖方较安全 |
| DAP/DPU | 即期LC或O/A | 需信用评估 |
| DDP | 100%预付或保兑LC | 卖方风险大 |

## 实操检查清单

### 报价前检查

- [ ] 术语选择是否符合运输方式？
- [ ] 港口/地点名称是否明确？
- [ ] 费用构成是否完整计算？
- [ ] 是否考虑了术语变形？
- [ ] 付款方式与术语是否匹配？

### 合同审查检查

- [ ] 术语版本是否明确（Incoterms® 2020）？
- [ ] 装卸费用划分是否清晰？
- [ ] 保险责任是否明确？
- [ ] 风险转移点是否双方理解一致？
- [ ] 进出口清关责任是否明确？

### 执行过程检查

- [ ] 是否按术语要求完成各环节？
- [ ] 装船通知/交货通知是否及时发出？
- [ ] 单据是否符合作价要求？
- [ ] 费用划分是否与术语一致？
- [ ] 争议发生时是否有合同依据？
