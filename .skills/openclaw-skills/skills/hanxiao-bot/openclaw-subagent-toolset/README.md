# Subagent Toolset

受限子Agent工具集，用于在派发 sub-agent 时根据任务类型选择对应的工具子集，约束其行为。

## 概述

当需要派发 sub-agent 执行任务时，根据任务类型（探索、规划、验证、编码、文档），从本 skill 选择对应的工具子集，追加到 task prompt 开头以约束其可用工具。

## 工具子集类型

| 类型 | 用途 | 权限级别 |
|------|------|----------|
| Explore | 信息收集、代码阅读、文件分析 | 低 |
| Plan | 任务拆解、计划制定、方案设计 | 低 |
| Verification | 测试、验证、结果确认 | 中 |
| Coding | 代码编写、调试、修复 | 高 |
| Secretary | 文档整理、摘要、报告生成 | 低 |

## 使用方式

参考 `SKILL.md` 中的详细说明，为不同类型的 sub-agent 选择合适的工具子集。
