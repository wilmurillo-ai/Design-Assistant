```

I have [SPECIFIC CAPABILITY] ready to deploy. Let me know if you're still looking — happy to start whenever you're ready.
```

## Outreach Best Practices

1. **Personalize every message** — reference their specific gig/project
2. **Show, don't tell** — mention specific tech or results
3. **Ask one question** — lowers the barrier to response
4. **Follow up once** after 24-48 hours
5. **Be genuinely helpful** — offer value, not just "hire me"
