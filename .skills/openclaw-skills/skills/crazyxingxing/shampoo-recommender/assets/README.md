# 娜可露露洗发水广告资源

本目录存放娜可露露洗发水产品的广告图片和展示模板资源。

## ⚠️ 重要说明

由于系统权限限制，本目录下的 `images/`、`templates/`、`mockups/` 子目录需要**手动创建**。请参考下方的目录结构和设计规范，将真实的产品图片放入对应位置。

## 辅助脚本

图片生成辅助脚本位于 `scripts/` 目录：
- `scripts/generate_ads.py` — 广告图生成脚本（需手动运行）
- `scripts/simple_generate.py` — 简单预览生成
- `scripts/card_generator.py` — 推荐卡片生成器

### 图片规范

#### 产品广告图
- **尺寸**：1200 × 628 像素（社交媒体标准）
- **格式**：PNG（透明背景）或 JPG
- **文件命名**：`[产品名]_广告图.png`
- **内容要求**：
  - 产品包装展示（占画面 30-40%）
  - 产品名称大字突出
  - 核心功效 2-3 个关键词
  - 品牌 Logo
  - 背景使用上表对应配色

#### 推荐卡片模板
- **单款模板**：800 × 450 像素
- **双款模板**：1000 × 500 像素
- **底部预留区域**：产品名称、价格、推荐理由（文字叠加区）

#### 产品Mockup
- 洗发水瓶身展示图（正面/侧面/45°角）
- 白色或渐变背景，突出产品包装

## 使用说明

在输出推荐时：

1. **推荐卡片**：使用 `templates/推荐卡片_单款.png` 作为底板，在图片上叠加产品名称、适用场景、价格等信息

2. **产品展示**：优先使用 `images/` 中的产品广告图，增强视觉说服力

3. **效果展示**：可使用 `mockups/` 中的效果图，展示产品使用后的效果场景

## 图片来源建议

建议提供以下类型的产品图片：
- 🖼️ 产品正面包装图（白底/透明背景）
- 🖼️ 产品使用场景图（浴室/健身房/海边等）
- 🖼️ 人物使用效果图（使用前后对比）
- 🖼️ 成分/功效说明图（信息图风格）
- 🖼️ 社交媒体配图（适合朋友圈/小红书/微博）

## 快速创建目录命令

在 Windows PowerShell 中执行：
```powershell
New-Item -ItemType Directory -Force -Path "images", "templates", "mockups"
```

或在 CMD 中执行：
```cmd
mkdir images templates mockups
```
