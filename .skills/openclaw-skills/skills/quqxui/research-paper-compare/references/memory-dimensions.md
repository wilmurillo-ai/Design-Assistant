# Memory Dimensions

This file is optional and should be read only when the compared papers are specifically about memory systems, retrieval systems, long-context methods, or agent memory.

## Additional Comparison Axes

- memory representation
- storage unit
- write policy
- retrieval trigger
- retrieval granularity
- summarization or compression strategy
- forgetting or pruning strategy
- latency and scalability implications
- memory-update reliability
- long-horizon consistency

## Usage Note

These dimensions are optional extensions.
The base `paper-compare` skill remains domain-general and should not assume a memory-specific lens unless the user asks for it or the paper set clearly requires it.
