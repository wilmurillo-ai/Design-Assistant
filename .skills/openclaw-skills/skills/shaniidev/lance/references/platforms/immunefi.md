# Immunefi Reporting Guidance

## Triage Mindset

Immunefi triage is exploitability and impact first.

## Must-Have Fields

- concise summary
- affected contracts/functions
- full exploit path (step-by-step)
- impact quantification
- severity rationale aligned to impact class

## Frequent Rejection Reasons

- theoretical only
- impact inflation
- no reproducible sequence
- out-of-scope class

## Verdict Labels

- `Accepted`
- `Needs More Info`
- `Rejected`
