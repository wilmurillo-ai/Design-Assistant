# 审计委员会监督要点

> 依据：《上海证券交易所上市公司自律监管指引第1号——规范运作》

## 6.1 组织架构

### 成员要求

| 要求 | 具体规定 |
|-----|---------|
| **成员数量** | 由**3名或以上**董事组成 |
| **独立董事比例** | **独立董事占多数** |
| **会计专业要求** | 至少一名独立董事是**会计专业人士** |
| **主任委员** | 由**会计专业人士**担任 |

### 成员任职资格

- 独立董事应当具备独立性
- 应当具备财务、会计、审计等专业知识
- 应当具备相关工作经验

## 6.2 职责范围

### 财务报告监督

```
财务报告监督职责：
├── 审议年度财务报告、半年度报告
├── 监督财务报告审计工作
├── 审核重大关联交易
├── 审核会计政策变更
├── 审核重大会计估计
└── 审核财务信息对外披露
```

### 内部控制监督

```
内部控制监督职责：
├── 监督内部控制制度的建立健全
├── 审核内部审计计划
├── 监督内部审计工作
├── 审核内部审计报告
├── 评估内控有效性
└── 监督内控缺陷整改
```

### 会计师事务所监督

```
会计师事务所监督职责：
├── 提议聘请或更换外部审计机构
├── 监督外部审计机构工作
├── 审核审计费用
├── 审核审计报告
├── 评估审计质量
└── 协调审计与管理层沟通
```

### 其他职责

1. 监督董事、高级管理人员履职行为
2. 监督募集资金使用
3. 监督对外担保
4. 监督重大投资
5. 监督承诺履行

## 6.3 会议要求

### 会议频次

- **每年至少召开4次会议**
- 原则上每季度召开1次
- 必要时可召开临时会议

### 会议通知

- 会议召开前**7日**通知全体委员
- 会议材料应提前**3日**送达

### 会议召开

- 应有**过半数**委员出席方可召开
- 决议须经**过半数**委员通过
- 委员应亲自出席或委托出席

### 会议记录

- 会议应有详细记录
- 记录内容包括：
  - 会议时间、地点
  - 出席委员名单
  - 审议事项
  - 讨论意见
  - 表决结果
- 会议记录保存期限不少于**10年**

## 6.4 与会计师事务所沟通

### 年度审计沟通

审计委员会应当在年度审计中：

1. **审计前沟通**
   - 与会计师事务所确定审计范围
   - 确定审计时间安排
   - 沟通审计重点领域

2. **审计中沟通**
   - 听取审计进展汇报
   - 解决审计中发现的问题
   - 跟踪审计进度

3. **审计后沟通**
   - 审阅审计报告
   - 评价审计质量
   - 决定是否续聘

### 沟通内容

```
审计沟通内容：
├── 审计范围和审计计划
├── 重大审计发现
├── 关键审计事项
├── 会计政策选择
├── 会计估计合理性
├── 审计调整事项
├── 持续经营假设
├── 审计意见类型
└── 审计收费
```

## 6.5 履职报告

### 年度履职报告内容

审计委员会应向年度股东会提交履职报告，内容包括：

1. 召开会议次数
2. 出席委员情况
3. 审议事项
4. 发表的审核意见
5. 监督工作成效
6. 存在的不足
7. 改进建议

## 6.6 对会计师事务所的监督评估

### 评估内容

```
会计师事务所评估要素：
├── 审计质量
│   ├── 审计报告质量
│   ├── 重大问题识别能力
│   └── 专业判断能力
├── 独立性
│   ├── 与上市公司的关系
│   ├── 审计收费占比
│   └── 非审计服务
├── 团队能力
│   ├── 项目团队专业胜任能力
│   ├── 行业经验
│   └── 人员稳定性
├── 工作效率
│   ├── 审计计划执行情况
│   ├── 按时完成能力
│   └── 沟通配合
└── 收费合理性
    ├── 审计费用水平
    ├── 费用构成
    └── 性价比
```

### 评估结论

1. 建议续聘
2. 建议更换
3. 有条件续聘（改进事项）

## 6.7 与内部审计协调

### 协调机制

1. 内部审计部门向审计委员会负责
2. 审计委员会指导内部审计工作
3. 内部审计报告应提交审计委员会审议

### 监督内容

1. 内部审计制度建立健全
2. 内部审计工作计划
3. 内部审计工作实施
4. 内部审计发现整改
5. 内部审计人员管理

## 6.8 关联交易审核

### 审核范围

1. 重大关联交易
2. 日常关联交易
3. 关联资金往来
4. 关联担保

### 审核重点

```
关联交易审核重点：
├── 必要性
│   └── 交易是否必要、合理
├── 公允性
│   └── 定价是否公允
├── 完整性
│   └── 披露是否完整
├── 程序合规性
│   └── 决策程序是否合规
└── 独立性影响
    └── 是否影响上市公司独立性
```

## 6.9 募集资金监督

### 监督要点

1. 募集资金存放
2. 募集资金使用
3. 变更募集资金用途
4. 募集资金投资项目
5. 闲置募集资金使用

## 6.10 合规建议

### 会议管理

1. 规范会议通知和材料送达
2. 详细记录会议内容
3. 及时形成会议决议
4. 妥善保存会议档案

### 沟通机制

1. 建立与会计师事务所的定期沟通机制
2. 建立与内部审计的协调机制
3. 建立与管理层的沟通机制

### 履职要求

1. 积极参加各项会议
2. 深入了解公司情况
3. 客观发表审核意见
4. 及时关注风险事项
