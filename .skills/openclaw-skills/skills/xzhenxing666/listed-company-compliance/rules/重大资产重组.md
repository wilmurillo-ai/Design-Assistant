# 重大资产重组合规要点

> 依据：《上市公司重大资产重组管理办法》《上海证券交易所上市公司自律监管指引第6号——重大资产重组》

## 3.1 重大资产重组标准

### 主板上市公司标准

| 指标 | 比例要求 |
|-----|---------|
| 资产总额 | 达到或超过最近一个会计年度期末资产总额的**50%** |
| 营业收入 | 达到或超过最近一个会计年度营业收入的**50%** |
| 净资产 | 达到或超过最近一个会计年度期末净资产额的**50%** |

> 注：以上三项指标达到任一项即构成重大资产重组

### 创业板/科创板标准

- 购买出售资产净额占创业板/科创板上市公司最近一个会计年度期末净资产额的比例达到50%以上
- 且超过5000万元人民币

## 3.2 重组类型

### 购买资产

- 现金购买资产
- 发行股份购买资产
- 发行可转换公司债券购买资产

### 出售资产

- 现金出售资产
- 资产置换

### 组合形式

- 现金+发行股份购买资产
- 重大资产置换加发行股份购买资产

## 3.3 重组方案要求

### 信息披露要求

#### 首次披露

- 可以披露**重组预案**或直接披露**重组报告书**
- 重组方案应符合《内容与格式准则第26号》要求

#### 进展披露

- 首次披露至股东会通知前，**至少每30日**披露一次进展公告

#### 重大进展包括

1. 签订重组服务协议
2. 签订或修订重组框架协议
3. 取得有权部门审批意见
4. 尽职调查、审计、评估取得进展
5. 出现终止风险

#### 重大变化包括

1. 更换、增加、减少交易标的
2. 更换独立财务顾问等中介机构
3. 配套融资方案、交易作价重大调整
4. 交易标的产业、行业发生重大变化
5. 交易标的经营财务状况重大变化
6. 涉及重大诉讼仲裁
7. 无法获得有关部门审批
8. 相关主体被立案调查

## 3.4 重组审议程序

### 董事会审议

```
董事会审议 → 披露重组预案/报告书 → 次一交易日披露
```

### 董事会决议后需提交文件

1. 上市公司与交易对方签订的附生效条件合同
2. 交易对方出具的承诺
3. 国家有权部门原则性批复
4. 交易标的完整权利证明文件
5. 审计保留意见的专项核查意见（如适用）
6. 交易进程备忘录

### 股东会审议

- 股东会对《重组办法》所列事项**逐项表决**
- 发行股份购买资产同时募集配套资金的，可以分拆为两项议案分别表决

### 发行股份购买资产特殊规定

- 首次董事会决议公告后**6个月**内未召开股东会的
- 需重新确定定价基准日

## 3.5 业绩承诺补偿

### 协议必备内容

| 内容 | 要求 |
|-----|------|
| 业绩承诺方 | 明确具体的承诺方 |
| 补偿方式 | 股份补偿或现金补偿 |
| 计算方法 | 具体的计算公式 |
| 补偿数量和金额 | 具体的数量和金额 |
| 触发条件 | 业绩承诺的触发条件 |
| 执行程序 | 补偿的执行流程 |
| 时间期限 | 补偿的时间安排 |
| 保障措施 | 保障补偿实现的措施 |
| 争议解决 | 争议解决方式 |

### 可行性论证要求

董事会和独立财务顾问应论证：

1. 补偿时间安排
2. 股份解除限售安排
3. 股份质押安排
4. 补偿股份表决权和股利分配权安排
5. 业绩补偿协议是否明确可行

### 风险提示

应充分提示：
- 补偿不足风险
- 补偿不及时风险
- 业绩承诺无法实现风险

## 3.6 内幕信息管理

### 内幕信息知情人登记

- 上市公司及有关各方应按规定**登记、报送**内幕信息知情人档案
- 应编制**交易进程备忘录**
- 应制定**保密措施**
- 应严格**履行保密义务**

### 禁止行为

- 禁止利用内幕信息买卖股票
- 禁止泄露内幕信息
- 禁止建议他人买卖股票

## 3.7 重组终止

### 终止披露要求

- 披露**终止重大资产重组公告**
- 说明**终止原因**
- 说明**终止过程**
- 履行相关审议程序
- 承诺自公告之日起**至少1个月**内不再筹划重大资产重组

### 终止情形

1. 股东大会否决
2. 交易对方放弃
3. 无法达成一致
4. 重大障碍
5. 市场环境变化

## 3.8 停复牌要求

### 停牌原则

- 审慎申请停牌
- 严格控制停牌时间
- 避免滥用停牌
- 避免无故拖延复牌
- 不得以申请停牌代替信息保密

### 复牌条件

- 重组方案确定后及时复牌
- 终止重组后及时复牌

## 3.9 独立财务顾问职责

### 基本要求

1. 遵守法律法规和中国证监会规定
2. 诚实守信、勤勉尽责
3. 审慎接受业务委托
4. 切实履行尽职调查义务
5. 认真核查披露和申请文件
6. 独立出具专业意见
7. 督促上市公司及时履行信息披露义务

### 核查要求

1. 核查股份变动是否适用股份支付准则
2. 核查重大资产重组是否合规
3. 核查业绩承诺是否可行
4. 核查信息披露是否真实准确完整

## 3.10 常见违规情形

1. 虚假记载重组信息
2. 重大遗漏
3. 误导性陈述
4. 延迟披露进展
5. 内幕交易
6. 违规停复牌
7. 业绩承诺不达标
8. 变更重组方案未披露
