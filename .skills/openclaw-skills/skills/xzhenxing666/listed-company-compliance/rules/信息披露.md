# 信息披露合规要点

> 依据：《上市公司信息披露管理办法》《上海证券交易所上市公司自律监管指引第2号——信息披露事务管理》

## 2.1 信息披露基本原则

### 五大基本原则

| 原则 | 要求 | 违规风险 |
|-----|------|---------|
| **真实性** | 披露的信息必须如实反映实际情况，不得有虚假记载 | 虚假记载 |
| **准确性** | 披露的信息必须客观，不得夸大其辞，不得存在歧义、误导性陈述 | 误导性陈述 |
| **完整性** | 披露的信息必须内容完整，不得存在重大遗漏 | 重大遗漏 |
| **及时性** | 必须在规定期限内披露法定信息 | 延迟披露 |
| **公平性** | 确保所有投资者可以平等地获取同一信息 | 信息泄露 |

### 信息披露义务人

- 上市公司
- 控股股东、实际控制人
- 董事、监事、高级管理人员
- 收购人
- 重大资产重组各方
- 为上市公司提供服务的中介机构

## 2.2 信息披露事务管理制度

### 基本要求

1. **明确负责机构**
   - 明确信息披露事务管理部门
   - 董事会秘书负责协调执行

2. **制度建设**
   - 制定信息披露事务管理制度
   - 经董事会审议通过并披露

3. **责任分工**
   - 董事长承担首要责任
   - 董事会秘书承担直接责任
   - 财务负责人配合财务信息披露

### 必备制度内容

```
信息披露事务管理制度应包括：
├── 信息传递审核披露流程
├── 内幕信息知情人登记管理
├── 未公开信息保密措施
├── 对外发布信息流程
├── 子公司信息报告制度
├── 股东信息通报制度
├── 信息披露暂缓豁免制度
└── 责任追究机制
```

## 2.3 定期报告披露要求

### 披露时间

| 报告类型 | 披露时间 | 审计要求 |
|---------|---------|---------|
| **年度报告** | 会计年度结束后**4个月内** | 需审计 |
| **半年度报告** | 上半年结束后**2个月内** | 不需审计 |
| **季度报告** | 季度结束后**1个月内** | 不需审计 |

### 年度报告内容要求

1. 公司基本情况
2. 会计数据和财务指标
3. 管理层讨论与分析
4. 公司治理
5. 重要事项
6. 股份变动及股东情况
7. 优先股相关情况
8. 债券相关情况
9. 财务报告

### 定期报告审议程序

```
董事会审议 → 监事会审核 → 独立董事意见 → 董事签署确认意见 → 披露
```

## 2.4 临时报告披露标准

### 需要披露的重大事项

1. **重大交易**
   - 交易金额达到净资产10%以上
   - 购买出售资产、对外投资、提供担保、赠与

2. **重大事项**
   - 重大资产重组
   - 重大诉讼、仲裁
   - 重大债务
   - 重大亏损

3. **业绩信息**
   - 业绩预告（亏损、扭亏为盈、变动50%以上）
   - 业绩快报

4. **权益变动**
   - 控股股东、实际控制人变更
   - 持股5%以上股东变动
   - 董事、监事、高级管理人员变动

5. **其他重大事项**
   - 变更募集资金用途
   - 变更会计政策
   - 股票终止上市或恢复上市
   - 分配预案

### 披露时限

- 重大事项发生后**2个工作日**内披露
- 业绩预告：年度结束后**1个月内**披露
- 业绩快报：年度结束后**2个月内**披露

## 2.5 内幕信息知情人登记

### 需要登记的事项

1. 重大资产重组
2. 高比例送转股份
3. 导致实际控制人变更的权益变动
4. 要约收购
5. 发行证券（IPO、增发、配股等）
6. 合并、分立、分拆上市
7. 回购股份
8. 中国证监会和交易所要求的其他事项

### 登记人员范围

```
内幕信息知情人包括：
├── 上市公司董事、监事、高级管理人员
├── 控股股东、实际控制人及其董事、监事、高级管理人员
├── 收购人重大资产交易方
├── 证券服务机构相关人员
├── 会计师、律师、评估师
├── 亲属（配偶、父母、子女等）
└── 其他知悉内幕信息人员
```

### 登记时限

- 内幕信息形成之日起至信息披露后**2个工作日**内
- 内幕信息知情人档案保存期限不少于**10年**

## 2.6 信息披露违规处罚

### 处罚类型

| 处罚主体 | 处罚措施 |
|---------|---------|
| **证监会** | 行政处罚、罚款、市场禁入 |
| **交易所** | 纪律处分、通报批评、公开谴责 |
| **自律组织** | 监管措施、警示函 |

### 常见违规情形

1. 虚假记载
2. 误导性陈述
3. 重大遗漏
4. 延迟披露
5. 选择性披露
6. 信息泄露

## 2.7 直通披露

### 直通披露时段

1. 交易日早间披露时段：7:30-8:30
2. 交易日午间披露时段：11:30-12:30
3. 交易日盘后披露时段：15:30-19:00
4. 非交易时段：13:00-17:00

### 直通披露要求

- 完整、准确选择公告类别
- 关注业务参数等信息录入
- 及时查看确认披露结果
- 发现异常立即向交易所报告

## 2.8 信息披露评价

### 评价内容

1. 信息披露的规范性（真实、准确、完整、及时、公平）
2. 信息披露的有效性
3. 投资者关系管理
4. 回报投资者情况
5. 履行社会责任
6. 被处罚处分情况

### 评价结果

| 等级 | 分数 | 说明 |
|-----|------|-----|
| A | 90分以上 | 优秀 |
| B | 80-90分 | 良好 |
| C | 60-80分 | 合格 |
| D | 60分以下 | 不合格 |

### 评价限制

不得为A的情形：
- 上市当年净利润下滑50%以上
- 盈利低于盈利预测80%
- 被采取行政监管措施
- 董事高管被行政处罚
- 董事会秘书空缺超过6个月
