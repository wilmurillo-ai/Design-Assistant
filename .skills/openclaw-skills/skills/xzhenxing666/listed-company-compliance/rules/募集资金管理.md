# 募集资金管理合规要点

> 依据：《上海证券交易所上市公司自律监管指引第1号——规范运作》

## 7.1 募集资金存放

### 基本要求

1. **专户存放**
   - 募集资金应当存放于**专项账户**
   - 不得存放于集团财务公司
   - 专户不得存放非募集资金

2. **专户数量**
   - 专户数量应与募投项目数量匹配
   - 一个项目对应一个专户
   - 不得将一个专户用于多个项目

### 账户管理

```
募集资金账户管理：
├── 开户
│   ├── 经董事会批准
│   ├── 与保荐机构、存放银行签订三方监管协议
│   └── 协议内容包括资金用途、监管措施等
├── 存放
│   ├── 仅存放募集资金
│   ├── 不得存放其他款项
│   └── 定期核对账目
└── 变更
    ├── 需经董事会审议
    └── 需与保荐机构、银行重新签订协议
```

## 7.2 募集资金使用

### 使用原则

1. **专款专用**
   - 募集资金应用于募投项目
   - 不得随意变更用途

2. **规范使用**
   - 严格按照承诺投资计划使用
   - 履行内部决策程序

### 禁止用途

```
募集资金不得用于：
├── 质押、委托贷款
├── 开展证券投资
├── 衍生品投资
├── 提供担保
├── 投资交易性金融资产
├── 投资高风险产品
├── 直接或间接投资于买卖股票
└── 法律法规禁止的其他用途
```

### 暂时闲置资金使用

暂时闲置的募集资金可以用于：

1. **现金管理**
   - 投资安全性高、流动性好的保本型产品
   - 投资产品期限不超过12个月

2. **补充流动资金**
   - 暂时补充流动资金
   - 期限不超过12个月
   - 需经董事会审议

3. **永久补充流动资金**
   - 募投项目完成后
   - 节余募集资金（占比小于10%）
   - 金额低于500万元

## 7.3 变更募集资金用途

### 变更程序

| 变更比例 | 程序要求 |
|---------|---------|
| **超过50%** | 需经股东会审议 |
| **未超过50%** | 需经董事会审议 |

### 变更条件

1. 募投项目不具备可行性
2. 市场环境发生重大变化
3. 募投项目效益不佳
4. 战略调整需要
5. 其他合理原因

### 变更要求

```
变更募集资金用途程序：
├── 董事会审议
│   ├── 分析变更原因
│   ├── 评估新项目可行性
│   ├── 说明新项目经济效益
│   └── 保荐机构意见
├── 独立董事意见
│   └── 对变更必要性、合理性发表意见
├── 保荐机构意见
│   └── 对变更合规性发表意见
└── 股东会审议（如超过50%）
    └── 对变更事项进行表决
```

## 7.4 募集资金投向

### 投向要求

1. **服务实体经济**
   - 符合国家产业政策
   - 主要投向主营业务
   - 不得用于财务性投资

2. **科创板特殊要求**
   - 主要投向科技创新领域
   - 符合科创板定位

### 投向变更限制

1. 不得变相改变募集资金用途
2. 不得将募集资金用于质押担保
3. 不得通过关联交易输送利益

## 7.5 募集资金使用管理

### 日常管理

1. **台账管理**
   - 建立募集资金使用台账
   - 记录每笔资金使用情况

2. **审批管理**
   - 逐级审批
   - 确保资金使用合规

3. **档案管理**
   - 保存相关审批文件
   - 保存资金使用凭证

### 监督机制

```
募集资金监督机制：
├── 内部监督
│   ├── 董事会定期检查
│   ├── 管理层日常监督
│   └── 内审部门定期审计
├── 中介机构监督
│   ├── 保荐机构持续督导
│   └── 会计师事务所年度审计
└── 信息披露监督
    ├── 年度报告披露使用情况
    └── 临时报告披露重大变化
```

## 7.6 信息披露要求

### 定期披露

1. **年度报告**
   - 募集资金存放与使用情况
   - 变更情况
   - 投资产品情况
   - 节余资金使用情况

2. **半年度报告**
   - 简要披露募集资金使用情况

### 临时披露

```
需要临时披露的情形：
├── 变更募集资金用途
├── 募集资金投资项目对外转让
├── 募集资金投资项目置换
├── 闲置募集资金补充流动资金
├── 闲置募集资金进行现金管理
├── 超过募集资金投资计划完成期限
├── 募集资金投资项目无法单独核算效益
└── 预计无法达到承诺效益
```

### 披露内容

1. 募集资金的存放金额
2. 募集资金的使用情况
3. 募集资金使用效益
4. 募集资金管理存在的问题
5. 改进措施

## 7.7 超募资金管理

### 超募资金认定

- 实际募集资金超过拟募集资金

### 超募资金使用

```
超募资金使用顺序：
├── 补充募投项目资金缺口
├── 扩大募投项目投资规模
├── 收购资产或股权
├── 补充流动资金
└── 其他合规用途
```

### 超募资金限制

1. 超募资金使用需经董事会审议
2. 超募资金用于永久补充流动资金需经股东会审议
3. 超募资金不得用于证券投资

## 7.8 违规处罚

### 常见违规情形

1. 募集资金存放不合规
2. 改变募集资金用途
3. 募集资金用于禁止用途
4. 变更程序不合规
5. 信息披露不完整
6. 超募资金使用不合规

### 处罚措施

1. 监管措施
2. 行政处罚
3. 纪律处分
4. 民事赔偿

## 7.9 合规建议

### 制度建设

1. 制定募集资金管理制度
2. 明确各部门职责
3. 规范使用流程
4. 建立监督机制

### 日常管理

1. 专户专人管理
2. 定期核对账目
3. 及时记录台账
4. 妥善保存档案

### 信息披露

1. 按时披露使用情况
2. 完整披露变更事项
3. 充分提示风险
4. 及时披露异常情况
