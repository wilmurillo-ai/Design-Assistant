# 股东减持合规要点

> 依据：《上市公司股东减持股份管理暂行办法》【第224号令】

## 4.1 减持方式与限制

### 集中竞价交易

| 股东类型 | 减持限制 |
|---------|---------|
| **大股东** | 任意连续90日内，减持股份总数不得超过公司股份总数的**1%** |
| **特定股东** | 任意连续90日内，减持股份总数不得超过公司股份总数的**1%** |

### 大宗交易

| 股东类型 | 减持限制 |
|---------|---------|
| **大股东** | 任意连续90日内，减持股份总数不得超过公司股份总数的**2%** |
| **特定股东** | 任意连续90日内，减持股份总数不得超过公司股份总数的**2%** |

> **重要**：大宗交易受让方在受让后**6个月内**不得转让所受让股份

### 协议转让

| 情形 | 要求 |
|-----|------|
| 单个受让方 | 受让股份比例不得低于**5%** |
| 转让后 | 受让方在受让后**6个月内**不得转让 |
| 转让比例 | 转让方持股比例降至5%以下的，履行信息披露义务 |

### 司法强制执行

- 通过司法强制执行方式减持股份的，参照上述规定执行

### 股权激励

- 限制性股票、股票期权等股权激励计划获得的股份减持，参照上述规定执行

## 4.2 大股东定义

> 大股东是指上市公司**控股股东**和**持股5%以上股东**

### 控股股东认定

- 持股比例超过50%
- 或持股比例虽不足50%但对股东大会决议产生重大影响
- 或通过协议或其他安排能够实际控制公司

## 4.3 特定股东定义

> 特定股东是指**减持特定股份**的股东

### 特定股份包括

1. 首次公开发行前股份
2. 上市公司非公开发行股份
3. 持有上述股份对应的表决权
4. 法律法规规定的其他特定股份

## 4.4 信息披露要求

### 预披露

- 上市公司大股东、董监高计划通过证券交易所集中竞价交易减持股份
- 应当在**首次卖出前15个交易日**向证券交易所报告并预先披露减持计划
- 减持计划内容应包括：拟减持股份数量、来源、减持时间区间、价格区间、原因

### 进展披露

- 减持时间过半时，应当披露减持进展
- 减持计划实施完毕后，应当披露减持结果

### 权益变动披露

| 情形 | 要求 |
|-----|------|
| 减持至5% | 应当披露权益变动报告 |
| 减持至5%以下 | 应当披露权益变动报告 |
| 变动达5% | 应当披露权益变动报告 |

## 4.5 窗口期限制

### 禁止减持期间

```
禁止减持的窗口期包括：
├── 定期报告窗口期
│   ├── 年度报告公告前30日内
│   ├── 半年度报告公告前15日内
│   └── 季度报告公告前10日内
├── 业绩预告、业绩快报窗口期
│   ├── 业绩预告公告前10日内
│   └── 业绩快报公告前10日内
├── 重大事项敏感期
│   ├── 重大资产重组筹划期间
│   ├── 股权激励计划筹划期间
│   └── 其他可能影响股价的重大事项
└── 法律、法规规定的其他禁止期间
```

### 董监高特殊规定

- 董监高在任期届满前离职的，应在其就任时确定的任期内和任期届满后6个月内：
  - 每年转让的股份不超过其所持有股份总数的25%
  - 离职后半年内不得转让股份

## 4.6 违规减持处罚

### 处罚措施

| 处罚主体 | 处罚措施 |
|---------|---------|
| **证监会** | 责令改正、没收违法所得、罚款、市场禁入 |
| **交易所** | 公开谴责、通报批评、限制交易 |

### 罚款标准

- 没收违法所得
- 并处买卖证券等值以下的罚款
- 情节严重的，处以更重罚款

### 违规情形

1. 超过规定比例减持
2. 在窗口期减持
3. 未履行预披露义务
4. 未履行权益变动披露
5. 大宗交易受让方违规转让
6. 利用内幕信息减持

## 4.7 特殊情形

### 创投基金差异化减持

- 符合条件的创投基金可以享受差异化减持政策
- 减持比例、锁定期等可适当放宽

### 员工持股计划

- 员工持股计划减持参照上述规定执行
- 具体按照员工持股计划方案执行

### 分类差异

| 股东类型 | 减持限制 | 信息披露 |
|---------|---------|---------|
| 控股股东 | 严格限制 | 预披露+进展+结果 |
| 持股5%以上 | 限制 | 预披露+进展+结果 |
| 特定股东 | 限制 | 权益变动披露 |
| 董监高 | 任职期间限制 | 任职期间变动披露 |
| 创投基金 | 差异化政策 | 按规定披露 |

## 4.8 合规建议

### 减持前

1. 确认股东身份（是否为大股东、特定股东）
2. 确认股份性质（是否属于特定股份）
3. 计算可减持数量
4. 确定减持方式
5. 制定减持计划
6. 履行预披露义务

### 减持中

1. 控制减持节奏
2. 避免窗口期减持
3. 及时披露进展
4. 记录交易情况

### 减持后

1. 披露减持结果
2. 更新股东名册
3. 履行税务义务
4. 保存交易记录
