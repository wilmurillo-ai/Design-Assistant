# 关联交易合规要点

> 依据：《上海证券交易所股票上市规则》《上市公司信息披露管理办法》

## 9.1 关联关系认定

### 关联方范围

```
关联方包括：
├── 关联法人
│   ├── 控股股东
│   ├── 实际控制人
│   ├── 控股股东控制的企业
│   ├── 实际控制人控制的企业
│   ├── 持股5%以上股东控制的企业
│   └── 关联自然人控制的企业
├── 关联自然人
│   ├── 控股股东、实际控制人
│   ├── 董事、监事、高级管理人员
│   ├── 持股5%以上自然人股东
│   └── 上述人员的近亲属
└── 其他关联关系
    ├── 产生重大影响的投资方
    ├── 产生重大影响的供应商
    └── 其他可能输送利益的主体
```

### 关联自然人近亲属

1. 配偶
2. 父母
3. 子女
4. 兄弟姐妹
5. 配偶的父母、兄弟姐妹
6. 子女的配偶
7. 配偶的兄弟姐妹

## 9.2 关联交易类型

### 常见关联交易

```
日常性关联交易：
├── 采购商品、接受劳务
├── 销售商品、提供劳务
├── 租入或租出资产
├── 资金往来（拆入、拆出）
├── 托管
└── 许可协议

非日常性关联交易：
├── 购买或出售资产
├── 对外投资
├── 提供担保
├── 赠与或受赠资产
├── 债权债务重组
├── 签订许可协议
├── 重大关联交易
└── 其他
```

## 9.3 关联交易金额标准

### 重大关联交易标准

| 指标 | 标准 |
|-----|------|
| 资产总额 | 成交金额占上市公司最近一期经审计资产总额**5%以上** |
| 营业收入 | 成交金额占上市公司最近一个会计年度营业收入**5%以上** |
| 净利润 | 成交金额占上市公司最近一个会计年度净利润**5%以上** |
| 成交金额 | 金额在**3000万元以上** |

> 注：以上指标计算中涉及的数据如为负值，取绝对值计算

### 需披露的关联交易

| 交易类型 | 披露标准 |
|---------|---------|
| 购买出售资产 | 金额达到净资产1%以上且超过300万元 |
| 对外投资 | 金额超过1000万元 |
| 提供担保 | 任何金额 |
| 赠与资产 | 任何金额 |
| 资金往来 | 单笔或累计超过1000万元 |

## 9.4 决策程序

### 董事会审议

```
关联交易董事会审议流程：
├── 部门汇报
│   └── 相关部门汇报关联交易情况
├── 独立董事意见
│   └── 重大关联交易需独立董事事前认可
├── 董事会讨论
│   └── 董事对关联交易进行讨论
├── 董事会表决
│   └── 关联董事回避表决
└── 会议记录
    └── 详细记录关联交易讨论和表决情况
```

### 回避表决

```
回避表决规则：
├── 关联董事回避
│   └── 关联董事不参与表决
├── 关联股东回避
│   └── 关联股东在股东会上回避表决
├── 独立董事例外
│   └── 独立董事可以发表独立意见
└── 表决通过条件
    └── 非关联董事/股东过半数通过
```

### 股东会审议

```
需提交股东会审议的关联交易：
├── 重大关联交易
├── 与关联人达成的资产转让、受让
├── 对外担保（另有规定除外）
├── 赠与资产
├── 其他重大关联交易
└── 董事会认为需要提交股东会的事项
```

## 9.5 定价公允性

### 定价原则

| 定价方法 | 适用情形 |
|---------|---------|
| **市场价** | 存在活跃市场的商品、劳务 |
| **协议价** | 无活跃市场，但有可比价格 |
| **成本加成** | 无法获取市场价格 |
| **评估价** | 资产转让、受让 |

### 定价依据

1. 关联交易价格应当符合市场公允价格
2. 如无市场价格，应当采用合理成本费用加合理利润
3. 关联交易定价方法应当披露

### 公允性核查

1. 独立董事对定价公允性发表意见
2. 聘请独立财务顾问发表意见
3. 聘请专业评估机构进行评估

## 9.6 信息披露

### 披露时间

| 交易类型 | 披露时间 |
|---------|---------|
| 日常性关联交易 | 年度预计框架协议披露 |
| 重大关联交易 | 董事会或股东会召开后2个工作日内 |
| 关联交易进展 | 及时披露进展 |

### 披露内容

```
关联交易披露内容：
├── 关联方基本情况
│   ├── 关联关系
│   ├── 关联方名称
│   └── 关联方情况
├── 关联交易内容
│   ├── 交易标的
│   ├── 交易金额
│   ├── 定价依据
│   └── 结算方式
├── 关联交易目的
│   ├── 交易必要性
│   ├── 交易持续性
│   └── 对上市公司的影响
├── 关联交易审议程序
│   ├── 董事会审议
│   ├── 独立董事意见
│   └── 股东会审议（如适用）
└── 关联交易承诺
    └── 承诺事项
```

### 持续性关联交易

```
年度日常关联交易披露：
├── 年度日常关联交易预计
│   ├── 关联交易类别
│   ├── 关联人
│   ├── 预计金额
│   └── 占同类业务比例
├── 实际执行情况
│   ├── 实际发生金额
│   └── 与预计金额差异
└── 超额处理
    └── 超额部分需重新审议
```

## 9.7 违规情形与处罚

### 常见违规情形

1. **程序违规**
   - 未履行审议程序
   - 未回避表决
   - 未披露

2. **定价违规**
   - 定价不公允
   - 利益输送
   - 损害上市公司利益

3. **披露违规**
   - 虚假记载
   - 重大遗漏
   - 延迟披露

### 处罚措施

| 处罚主体 | 处罚措施 |
|---------|---------|
| **证监会** | 行政处罚、罚款、市场禁入 |
| **交易所** | 纪律处分、通报批评、公开谴责 |
| **上市公司** | 追责、赔偿 |

## 9.8 关联交易管理

### 制度建设

1. **制定关联交易管理制度**
   - 明确关联方的范围和认定标准
   - 明确关联交易的决策程序
   - 明确关联交易的定价原则
   - 明确关联交易的信息披露

2. **建立关联方名单**
   - 定期更新
   - 全面覆盖
   - 动态管理

### 日常管理

```
关联交易日常管理：
├── 关联方识别
│   ├── 新增关联方识别
│   ├── 关联方关系变化跟踪
│   └── 关联方名单维护
├── 交易事前管理
│   ├── 交易计划申报
│   ├── 交易额度审批
│   └── 合同审批
├── 交易事中控制
│   ├── 交易执行监控
│   └── 定价执行监督
└── 交易事后管理
    ├── 交易核对
    └── 差异分析
```

## 9.9 合规建议

### 识别阶段

1. 全面梳理关联关系
2. 及时更新关联方名单
3. 识别潜在关联交易

### 决策阶段

1. 充分准备会议材料
2. 严格执行回避制度
3. 详细记录讨论过程

### 执行阶段

1. 按照批准方案执行
2. 控制交易金额
3. 保存交易凭证

### 披露阶段

1. 及时履行披露义务
2. 完整披露相关信息
3. 确保披露内容真实准确
