# 内部控制合规要点

> 依据：《上海证券交易所上市公司自律监管指引第1号——规范运作》《企业内部控制基本规范》

## 8.1 内部控制目标

### 基本目标

```
内部控制五大目标：
├── 合理保证企业经营管理合法合规
├── 资产安全
├── 财务报告真实完整
├── 提高经营效率和效果
└── 促进企业实现发展战略
```

## 8.2 内部控制原则

### 基本原则

| 原则 | 要求 |
|-----|------|
| **全面性** | 覆盖公司所有业务、部门、人员 |
| **重要性** | 重点关注重要业务和高风险领域 |
| **制衡性** | 相互制约、相互监督 |
| **适应性** | 与公司规模、业务、风险相适应 |
| **成本效益** | 权衡成本与效益 |

### 五要素框架

```
内部控制五要素：
├── 内部环境
│   ├── 公司治理结构
│   ├── 组织架构
│   ├── 权责分配
│   ├── 企业文化
│   └── 内部审计
├── 风险评估
│   ├── 目标设定
│   ├── 风险识别
│   ├── 风险分析
│   └── 风险应对
├── 控制活动
│   ├── 不相容职务分离
│   ├── 授权审批
│   ├── 会计系统
│   ├── 财产保护
│   └── 预算控制

```

## 8.4 组织架构

### 治理层

1. **董事会**
   - 负责内部控制的建立健全
   - 监督内部控制有效实施
   - 评价内部控制有效性

2. **监事会**
   - 监督董事会建立实施内部控制
   - 监督内部审计工作

3. **审计委员会**
   - 监督内部审计工作
   - 协调内部控制审计

### 管理层

1. **总经理**
   - 组织制定内部控制制度
   - 组织实施内部控制

2. **财务负责人**
   - 组织会计核算
   - 监督财务报告编制

### 执行层

1. **内部审计部门**
   - 开展内部控制审计
   - 评价内部控制有效性
   - 报告内部控制缺陷

2. **各职能部门**
   - 执行内部控制制度
   - 报告控制缺陷

## 8.6 内部控制评价

### 评价要求

1. **年度评价**
   - 每年至少进行一次内部控制评价
   - 评价期间为上一个会计年度

2. **全面评价**
   - 覆盖所有重要业务和事项
   - 覆盖所有重要单位

3. **评价方法**
   - 个别访谈
   - 专题讨论
   - 穿行测试
   - 抽样测试

### 评价报告

```
内部控制评价报告内容：
├── 董事会声明
├── 内部控制评价范围
├── 内部控制评价程序和方法
├── 内部控制重大缺陷认定
├── 内部控制缺陷整改
└── 内部控制有效性结论
```

### 缺陷认定

| 缺陷类型 | 定义 |
|---------|------|
| **重大缺陷** | 一个或多个控制缺陷的组合，可能导致严重偏离控制目标 |
| 重要缺陷 | 一个或多个控制缺陷的组合，可能导致中等偏离控制目标 |
| 一般缺陷 | 除重大缺陷、重要缺陷之外的其他缺陷 |

## 8.7 内部控制审计

### 审计要求

1. **审计范围**
   - 财务报告内部控制
   - 非财务报告内部控制

2. **审计意见**
   - 标准无保留意见
   - 保留意见
   - 否定意见
   - 无法表示意见

### 审计报告

```
内部控制审计报告内容：
├── 审计责任
├── 管理层责任
├── 审计意见
├── 重大缺陷说明
└── 非财务报告内部控制重大缺陷
```

## 8.8 责任追究

### 责任主体

1. 董事会
2. 监事会
3. 管理层
4. 各职能部门
5. 工作人员

### 追责情形

1. 未建立内部控制制度
2. 内部控制制度不健全
3. 未执行内部控制制度
4. 内部控制执行不到位
5. 隐瞒内部控制缺陷

### 追责措施

1. 行政处分
2. 经济处罚
3. 纪律处分
4. 法律责任

## 8.9 合规建议

### 制度建设

1. 制定内部控制制度
2. 明确各部门职责
3. 建立监督机制
4. 完善问责机制

### 执行监督

1. 定期开展内部控制评价
2. 及时整改内部控制缺陷
3. 加强内部审计监督
4. 接受外部审计监督

### 持续改进

1. 根据业务变化及时更新内控制度
2. 根据监管要求完善内控体系
3. 借鉴最佳实践提升内控水平
