# 业绩预告公告模板

## 模板一：年度业绩预告

```
________股份有限公司
____年度业绩预告公告

本公司董事会及全体董事保证本公告内容不存在任何虚假记载、误导性陈述或者重大遗漏，并对其内容的真实性、准确性和完整性承担个别及连带责任。

一、业绩预告期间
____年____月____日至____年____月____日

二、业绩预告情况
（一）归属于上市公司股东的净利润
1. 预计实现归属于上市公司股东的净利润：____万元
2. 与上年同期相比预计增加/减少：____%
3. 上年同期归属于上市公司股东的净利润：____万元

（二）归属于上市公司股东的扣除非经常性损益的净利润
1. 预计实现归属于上市公司股东的扣除非经常性损益的净利润：____万元
2. 与上年同期相比预计增加/减少：____%
3. 上年同期归属于上市公司股东的扣除非经常性损益的净利润：____万元

三、业绩变动的主要原因
________（详细说明业绩变动的主要原因，包括：主营业务变化、非经常性损益影响、市场环境变化等）

四、风险提示
1. 本次业绩预告是公司财务部门初步测算的结果，具体财务数据以公司正式披露的____年年度报告为准；
2. 如存在业绩预告需要更正的情形，公司将及时发布业绩预告更正公告；
3. 提醒投资者注意投资风险。

五、其他说明事项
（如有需要补充说明的事项）

特此公告。

________股份有限公司董事会
____年____月____日
```
