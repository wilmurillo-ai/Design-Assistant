# 上市公司合规检查清单

## 一、独立董事合规检查

### 1.1 任职资格检查
- [ ] 独立董事是否具备5年以上法律、经济、管理、会计、财务等工作经验
- [ ] 是否通过证券交易所组织的独立董事培训并取得资格证书
- [ ] 是否存在禁止任职的情形（上市公司任职、持股1%以上、亲属关系等）
- [ ] 是否具备独立董事独立性（不存在利益冲突）

### 1.2 人数比例检查
- [ ] 董事会成员中独立董事是否达到三分之一
- [ ] 审计委员会中独立董事是否占多数
- [ ] 提名委员会中独立董事是否占多数
- [ ] 薪酬与考核委员会中独立董事是否占多数
- [ ] 审计委员会至少有一名会计专业人士

### 1.3 职责履行检查
- [ ] 重大关联交易是否经独立董事事前认可
- [ ] 独立董事是否独立发表意见
- [ ] 独立董事是否亲自出席董事会会议
- [ ] 独立董事是否对定期报告签署书面确认意见
- [ ] 独立董事是否出席年度股东大会

### 1.4 会议记录检查
- [ ] 独立董事会议是否有详细记录
- [ ] 独立董事意见是否完整保存
- [ ] 独立董事履职报告是否完整

---

## 二、信息披露合规检查

### 2.1 制度建立检查
- [ ] 是否制定信息披露事务管理制度
- [ ] 是否明确董事会秘书职责
- [ ] 是否明确董事长首要责任
- [ ] 是否建立内幕信息知情人登记管理制度
- [ ] 是否建立信息披露暂缓豁免制度

### 2.2 定期报告检查
- [ ] 年度报告是否在会计年度结束后4个月内披露
- [ ] 半年度报告是否在上半年结束后2个月内披露
- [ ] 季度报告是否在季度结束后1个月内披露
- [ ] 定期报告是否经董事会审议
- [ ] 定期报告是否经监事会审核
- [ ] 董事、高级管理人员是否签署书面确认意见

### 2.3 临时报告检查
- [ ] 重大事项是否及时披露
- [ ] 重大合同是否及时披露
- [ ] 重大诉讼仲裁是否及时披露
- [ ] 业绩预告是否及时披露
- [ ] 股东权益变动是否及时披露
- [ ] 控股股东、实际控制人变更是否及时披露

### 2.4 信息质量检查
- [ ] 披露信息是否真实
- [ ] 披露信息是否准确
- [ ] 披露信息是否完整
- [ ] 披露信息是否及时
- [ ] 是否公平对待所有投资者

### 2.5 内幕信息管理检查
- [ ] 内幕信息知情人是否及时登记
- [ ] 内幕信息知情人档案是否完整
- [ ] 内幕信息是否保密
- [ ] 是否存在内幕交易

---

## 三、重大资产重组合规检查

### 3.1 重组标准检查
- [ ] 购买出售资产总额是否达到50%以上
- [ ] 购买出售资产营业收入是否达到50%以上
- [ ] 购买出售资产净额是否达到50%以上

### 3.2 前期准备检查
- [ ] 是否按规定登记内幕信息知情人
- [ ] 是否编制交易进程备忘录
- [ ] 是否制定保密措施
- [ ] 是否审慎筹划重组事项

### 3.3 信息披露检查
- [ ] 重组方案是否及时披露
- [ ] 是否每30日披露一次进展
- [ ] 重大变化是否及时披露
- [ ] 重组报告书内容是否完整
- [ ] 独立财务顾问报告是否完整

### 3.4 审议程序检查
- [ ] 重组方案是否经董事会审议
- [ ] 重组方案是否经独立董事意见
- [ ] 重组方案是否提交股东会审议
- [ ] 发行股份购买资产定价基准日是否合规

### 3.5 业绩承诺检查
- [ ] 业绩承诺协议条款是否完整
- [ ] 补偿方式是否明确
- [ ] 补偿数量金额是否明确
- [ ] 履约风险是否充分提示
- [ ] 业绩补偿是否具有可行性

### 3.6 重组终止检查
- [ ] 终止重组是否及时披露
- [ ] 终止原因是否说明
- [ ] 是否承诺一定期限内不再筹划

---

## 四、股东减持合规检查

### 4.1 减持方式检查
- [ ] 集中竞价交易是否超过90日内1%
- [ ] 大宗交易是否超过90日内2%
- [ ] 协议转让是否达到5%以上
- [ ] 司法强制执行、股权激励等是否合规

### 4.2 信息披露检查
- [ ] 是否提前15个交易日预披露
- [ ] 减持比例达到5%是否披露
- [ ] 减持至5%以下是否披露
- [ ] 减持完成后是否披露

### 4.3 窗口期检查
- [ ] 是否在定期报告窗口期减持
- [ ] 是否在业绩预告窗口期减持
- [ ] 是否在内幕信息敏感期减持

### 4.4 特定股东检查
- [ ] 控股股东、实际控制人减持是否合规
- [ ] 特定股东减持是否合规
- [ ] 高管持股变动是否合规

---

## 五、内部控制合规检查

### 5.1 制度建立检查
- [ ] 是否建立内部控制制度
- [ ] 是否明确内控职责分工
- [ ] 是否建立风险评估机制
- [ ] 是否建立内部审计制度

### 5.2 执行情况检查
- [ ] 内部控制是否有效执行
- [ ] 是否定期开展内控自评
- [ ] 是否聘请会计师事务所进行内控审计
- [ ] 内控缺陷是否及时整改

### 5.3 重点领域检查
- [ ] 募集资金管理是否合规
- [ ] 关联交易管理是否合规
- [ ] 对外担保管理是否合规
- [ ] 重大投资管理是否合规

---

## 六、审计委员会合规检查

### 6.1 组织架构检查
- [ ] 审计委员会成员是否为3名或以上
- [ ] 独立董事是否占多数
- [ ] 是否有会计专业人士
- [ ] 主任委员是否由会计专业人士担任

### 6.2 职责履行检查
- [ ] 是否审议年度财务报告
- [ ] 是否监督审计工作
- [ ] 是否审核关联交易
- [ ] 是否监督内部控制
- [ ] 是否评估聘请会计师事务所

### 6.3 会议运作检查
- [ ] 每年是否至少召开4次会议
- [ ] 会议是否有详细记录
- [ ] 委员是否亲自出席或委托出席

---

## 七、可持续发展报告（ESG）合规检查

### 7.1 披露主体检查
- [ ] 是否属于强制披露主体（上证180、科创50、境内外同时上市）
- [ ] 是否属于自愿披露主体

### 7.2 披露时间检查
- [ ] 是否在会计年度结束后4个月内披露
- [ ] 是否经董事会审议通过
- [ ] 是否不早于年度报告披露

### 7.3 披露内容检查
- [ ] 治理信息披露是否完整
- [ ] 战略信息披露是否完整
- [ ] 影响风险机遇管理披露是否完整
- [ ] 指标与目标披露是否完整
- [ ] 应对气候变化信息披露是否完整
- [ ] 温室气体排放核算是否完整

### 7.4 信息质量检查
- [ ] 信息披露是否客观真实
- [ ] 数据计量方法是否一致
- [ ] 重大影响是否充分识别
- [ ] 重要性评估是否合理

---

## 八、募集资金管理合规检查

### 8.1 存放检查
- [ ] 募集资金是否存放于专项账户
- [ ] 是否存放于集团财务公司（禁止）
- [ ] 专户数量是否与募投项目匹配

### 8.2 使用检查
- [ ] 募集资金是否用于募投项目
- [ ] 是否存在变更募集资金用途
- [ ] 变更程序是否合规
- [ ] 暂时闲置资金使用是否合规

### 8.3 监督检查
- [ ] 持续督导是否到位
- [ ] 募集资金使用是否定期披露
- [ ] 年度报告是否披露使用情况

---

## 九、财务报告合规检查

### 9.1 编制质量检查
- [ ] 会计政策选择是否恰当
- [ ] 会计估计是否合理
- [ ] 关联交易定价是否公允
- [ ] 重大资产计价是否准确

### 9.2 审计配合检查
- [ ] 是否及时提供审计所需资料
- [ ] 是否配合审计师工作
- [ ] 审计意见是否恰当

### 9.3 披露合规检查
- [ ] 年度报告内容是否完整
- [ ] 半年度报告内容是否完整
- [ ] 财务指标变动是否说明原因
- [ ] 重大事项是否充分披露

---

## 十、关联交易合规检查

### 10.1 关联关系识别
- [ ] 关联方名单是否完整
- [ ] 关联关系是否及时更新
- [ ] 潜在关联交易是否识别

### 10.2 决策程序检查
- [ ] 关联交易是否经董事会审议
- [ ] 重大关联交易是否经独立董事认可
- [ ] 重大关联交易是否经股东会审议
- [ ] 关联董事、股东是否回避表决

### 10.3 定价公允性检查
- [ ] 关联交易定价是否公允
- [ ] 是否参照市场价格
- [ ] 是否有利益输送

### 10.4 披露合规检查
- [ ] 关联交易是否及时披露
- [ ] 披露内容是否完整
- [ ] 持续性关联交易是否定期披露
