# 董事会会议通知模板

## 模板一：定期会议通知

```
________股份有限公司董事会会议通知

各位董事：

公司第__届董事会第__次会议定于____年____月____日（星期__）召开，现将有关事项通知如下：

一、会议时间
____年____月____日（星期__）上午/下午____时

二、会议地点
________股份有限公司会议室

三、会议召集人
____董事长

四、审议事项
1. 审议《公司____年度总经理工作报告》
2. 审议《公司____年度董事会工作报告》
3. 审议《公司____年度财务决算报告》
4. 审议《公司____年度利润分配预案》
5. 审议《公司____年度报告及摘要》
6. 审议《关于召开____年度股东大会的议案》
7. 其他事项

五、参会人员
全体董事、监事、高级管理人员

六、会议材料
会议材料已于____年____月____日通过电子邮件/专人送达各位董事

七、其他事项
1. 请各位董事准时参加，原则上不得委托非董事代为出席
2. 如有特殊情况不能参会，请提前____个工作日向董事会秘书请假
3. 会议期间请将手机调至静音模式

联系人：________
联系电话：________
电子邮箱：________

特此通知。

　　　　　　　　　　　　　　________股份有限公司董事会
　　　　　　　　　　　　　　　____年____月____日

（董事会章）
```

## 模板三：通讯表决通知

```
________股份有限公司董事会通讯表决通知

各位董事：

根据《公司章程》和《董事会议事规则》的有关规定，现以通讯方式就以下事项进行表决：

一、表决事项
1. 审议《关于________的议案》
2. 审议《关于________的议案》

二、表决要求
1. 请各位董事收到本通知后____日内（以发出日为准）将表决意见反馈至董事会秘书处
2. 表决意见分为：同意、反对、弃权
3. 如有异议，请在反馈表决意见时说明理由
4. 逾期未反馈视为同意

三、表决方式
请填写《董事会通讯表决票》并签字确认后，传真/扫描发送至董事会秘书处

四、联系人
联系人：________
联系电话：________
传真：________
电子邮箱：________

特此通知。

　　　　　　　　　　　　　　________股份有限公司董事会
　　　　　　　　　　　　　　　____年____月____日

（董事会章）
```

## 模板四：董事会决议公告

```
________股份有限公司
第__届董事会第__次会议决议公告

本公司董事会及全体董事保证本公告内容不存在任何虚假记载、误导性陈述或者重大遗漏，并对其内容的真实性、准确性和完整性承担个别及连带责任。

一、董事会会议召开情况
（一）会议召开时间：____年____月____日
（二）会议召开地点：公司会议室
（三）会议召集人：____董事长
（四）会议召开方式：现场/通讯
（五）会议应到董事____人，实到董事____人
（六）公司监事、高级管理人员列席会议
（七）会议符合《公司法》和《公司章程》的规定

二、董事会会议审议情况
会议以记名投票表决方式审议通过了以下议案：

（一）审议通过《公司____年度总经理工作报告》
表决结果：同意____票，反对____票，弃权____票

（二）审议通过《公司____年度董事会工作报告》
表决结果：同意____票，反对____票，弃权____票

（三）审议通过《公司____年度财务决算报告》
表决结果：同意____票，反对____票，弃权____票

（四）审议通过《公司____年度利润分配预案》
表决结果：同意____票，反对____票，弃权____票
（内容详见与本公告同时披露的____公告）

（五）审议通过《公司____年度报告及摘要》
表决结果：同意____票，反对____票，弃权____票

（此内容如涉及关联交易，需说明关联董事回避表决情况）

三、备查文件
1. 董事会会议决议
2. 独立董事独立意见（如有）
3. 相关议案文本

特此公告。

________股份有限公司董事会
____年____月____日
```
