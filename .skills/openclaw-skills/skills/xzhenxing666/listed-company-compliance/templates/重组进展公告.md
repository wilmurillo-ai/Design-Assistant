# 重大资产重组进展公告模板

## 模板一：重组进展公告

```
________股份有限公司
关于重大资产重组进展公告

本公司董事会及全体董事保证本公告内容不存在任何虚假记载、误导性陈述或者重大遗漏，并对其内容的真实性、准确性和完整性承担个别及连带责任。

一、重大资产重组基本情况
公司于____年____月____日披露了《重大资产重组预案》/《重大资产重组报告书》，公司拟________（简要说明重组方案，如：发行股份购买________公司100%股权）

二、重组进展情况
自上次披露（____年____月____日）以来，公司重大资产重组事项取得以下进展：

1. 尽职调查工作
公司及相关各方正在积极推进本次重大资产重组的尽职调查工作，________（说明进展情况）

2. 审计/评估工作
________（说明审计、评估工作进展）

3. 协议谈判
________（说明与交易对方谈判进展）

4. 审批事项
________（说明已取得或正在申请的审批）

三、风险提示
1. 本次重大资产重组尚需经公司董事会、股东大会审议通过，并需经中国证监会核准/注册，存在一定不确定性；
2. 重组方案可能根据审慎情况进行调整；
3. 公司将根据事项进展情况及时履行信息披露义务。

四、后续安排
1. 公司将根据相关规定，每30日披露一次重组进展公告；
2. 待相关工作完成后，公司将召开董事会审议重组报告书等文件；
3. 依据监管要求履行相应的审批程序。

五、备查文件
1. ________（其他备查文件）

特此公告。

________股份有限公司董事会
____年____月____日
```

1. ____年____月____日，公司与交易对方进行协商

## 模板四：重组提示性公告

```
________股份有限公司
关于筹划重大资产重组事项的提示性公告

本公司董事会及全体董事保证本公告内容不存在任何虚假记载、误导性陈述或者重大遗漏，并对其内容的真实性、准确性和完整性承担个别及连带责任。

一、筹划事项基本情况
公司正在筹划________（重大资产重组事项），主要情况如下：

1. 交易对方：________
2. 交易标的：________
3. 交易方式：________（发行股份购买资产/现金购买资产/资产置换等）
4. 标的资产估值：预计不超过____亿元（暂定）

二、交易背景及目的
________（说明本次交易的背景和目的）

三、风险提示
1. 该事项尚处于筹划阶段，存在重大不确定性；
2. 该事项需经公司董事会、股东大会审议，并需经中国证监会核准/注册；
3. 该事项可能存在因市场环境变化、审批未通过等因素导致终止的风险；
4. 公司股票将根据相关规定继续停牌。

四、后续工作安排
1. 公司将尽快确定独立财务顾问、审计机构、评估机构等中介机构；
2. 公司将按照《上市公司重大资产重组管理办法》等规定，组织相关中介机构开展尽职调查、审计、评估等工作；
3. 公司将在停牌期限届满前披露重组预案/报告书，并申请复牌。

五、特别提示
1. 投资者在作出投资决策前，请仔细阅读相关公告，注意投资风险；
2. 公司将根据事项进展情况及时履行信息披露义务。

特此公告。

________股份有限公司董事会
____年____月____日
```
