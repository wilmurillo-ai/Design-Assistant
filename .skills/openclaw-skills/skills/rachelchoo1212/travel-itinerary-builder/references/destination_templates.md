## Tokyo, Japan

**Best time to visit**: March-May (cherry blossoms), September-November (fall colors)

**Must-see attractions**:
- 東京塔 (Tokyo Tower / Tokyo Tower)
- 浅草寺 (Sensoji Temple / Asakusa Temple)
- 渋谷スクランブル交差点 (Shibuya Scramble Crossing)
- 明治神宮 (Meiji Shrine)

**Local food**:
- Ramen (拉面 / ラーメン)
- Sushi (寿司 / 寿司)
- Tempura (天妇罗 / 天ぷら)

**Transportation**: JR Pass, Suica card, metro

## Seoul, South Korea

**Best time to visit**: March-May (spring), September-November (fall)

**Must-see attractions**:
- 경복궁 (Gyeongbokgung Palace / 景福宫)
- N서울타워 (N Seoul Tower / N首尔塔)
- 명동 (Myeongdong / 明洞)

**Local food**:
- Korean BBQ (韩式烤肉 / 한국 바비큐)
- Bibimbap (拌饭 / 비빔밥)
- Kimchi (泡菜 / 김치)

**Transportation**: T-money card, metro, KTX
