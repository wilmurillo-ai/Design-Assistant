## 概述

OpenClaw Security Configurator 是一个专门为OpenClaw设计的企业级安全配置和监控技能。基于市场调研发现的安全痛点，本技能提供完整的解决方案，帮助用户安全部署和使用OpenClaw，特别适合金融机构和对安全要求高的企业环境。

## 快速开始

### 安装
```bash
# 克隆仓库
git clone https://github.com/your-repo/openclaw-security-skill.git
cd openclaw-security-skill

# 安装依赖
chmod +x scripts/*.sh

# 配置监控
./scripts/token-monitor.sh config
```

### 基础使用
```bash
# 执行安全检查
./scripts/security-check.sh

# 启动Token监控
./scripts/token-monitor.sh start

# 单次Token检查
./scripts/token-monitor.sh check
```

## 配置说明

### 配置文件
默认配置文件：`/etc/openclaw/token-monitor.conf`

主要配置项：
```bash
# 告警阈值
ALERT_THRESHOLD=10000      # 单次请求超过10000 tokens告警
DAILY_LIMIT=100000         # 每日限额100000 tokens

# 监控间隔
CHECK_INTERVAL=300         # 每5分钟检查一次

# 告警方式
ALERT_METHOD="log"         # log/email/webhook
```

### 日志文件
- 监控日志：`/var/log/openclaw-token-monitor.log`
- 每日统计：`/var/log/openclaw-token-daily-YYYY-MM-DD.log`
- 分析报告：`/tmp/openclaw-token-analysis-YYYYMMDD.txt`
