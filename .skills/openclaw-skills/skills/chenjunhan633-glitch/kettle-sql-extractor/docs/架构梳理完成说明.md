## 🔄 主要变更

### 1. **目录结构调整**
- **根目录**: 只保留 `SKILL.md` 和 `README.md`
- **scripts/**: 集中存放所有Python和Shell脚本
- **docs/**: 存放所有Markdown文档
- **tests/**: 存放测试脚本

### 2. **文件路径更新**
- **SKILL.md**: 更新所有脚本引用路径为 `scripts/`
- **README.md**: 更新文档引用路径为 `docs/`
- **所有脚本调用**: 使用 `python scripts/merge_kettle_sql.py` 格式

### 3. **文档整理**
- 将所有 `.md` 文件移动到 `docs/` 目录
- 保持文件分类清晰
- 更新所有内部链接

## 🛠️ 使用方式更新

### 旧方式
```bash
python merge_kettle_sql.py job.kjb --simple-output
python batch_extract_kettle_sql.py --dir /path/to/kettle/jobs
```

### 新方式
```bash
python scripts/merge_kettle_sql.py job.kjb --simple-output
python scripts/batch_extract_kettle_sql.py --dir /path/to/kettle/jobs
# 使用简化脚本
./scripts/extract_kettle_sql_simple.sh /path/to/kettle/jobs /path/to/output
scripts\extract_kettle_sql_simple.bat C:\path\to\kettle\jobs C:\path\to\output
```
