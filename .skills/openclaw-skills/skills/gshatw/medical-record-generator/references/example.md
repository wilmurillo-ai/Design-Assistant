# 病历生成器 - 使用示例

## 示例数据

```javascript
const MedicalRecordGenerator = require('./scripts/generate_record.js');
const generator = new MedicalRecordGenerator();

// 示例病历数据
const patientData = {
  name: "张无及",
  gender: "男性",
  age: "35岁",
  maritalStatus: "未婚",
  occupation: "职员",
  birthplace: "北京市",
  chiefComplaint: "发热、咳嗽3天",
  historyOfPresentIllness: "患者于3天前无明显诱因出现发热、咳嗽。发热为持续性，最高体温38.5℃，咳嗽为干咳，无痰；伴有乏力、全身酸痛，无胸痛、呼吸困难。发病后未在外院就诊。发病以来，精神稍差，睡眠正常，食欲正常，大小便正常，体重无明显变化。",
  preliminaryDiagnosis: "1. 急性上呼吸道感染\n2. 发热待查：感染性？"
};

// 生成病历
const record = generator.generateRecord(patientData);
console.log(record);

// 保存到文件
const filepath = generator.saveToFile(record);
console.log(`病历已保存到: ${filepath}`);
```
