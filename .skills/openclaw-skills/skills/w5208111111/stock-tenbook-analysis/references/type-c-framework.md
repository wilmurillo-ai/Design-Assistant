# C型（爆发型）产业趋势框架

## 何时使用C型框架

当公司满足以下条件时：
- 利润增速 > 100%
- PE > 50倍
- 处于全新的产业爆发期（AI、新能源、半导体等）

C型核心逻辑：**PE不是主要指标，关键是"利润能否持续指数级增长"。**

## C-1. 产业趋势评估（0-10分）

评分标准：
- 全新产业趋势（AI/新能源/半导体等）：+3
- 行业增速 > 50%：+2
- 下游资本开支加速：+2
- 技术壁垒高：+1
- 政策支持：+1
- 全球市场（非仅国内）：+1

## C-2. 竞争格局评估

关键问题：
- 全球/国内市占率是多少？
- 市占率趋势（提升/稳定/下降）？
- 毛利率趋势？（提升=护城河变宽）
- 第二名差距有多大？
- 新进入者威胁？

## C-3. PEG + 动态PE消化

PEG评估标准：
- PEG < 0.5 → 极度低估
- PEG 0.5-1.0 → 合理偏低
- PEG 1.0-1.5 → 合理
- PEG 1.5-2.0 → 偏贵
- PEG > 2.0 → 高估

**动态PE消化**：如果利润增速维持，1-2年后PE会降到多少？
- 当前PE 50倍，增速100% → 1年后PE 25倍 → 2年后PE 12.5倍

## C-4. 终局估值

计算步骤：
1. 估算全球/中国市场成熟期规模
2. 公司合理市占率
3. 成熟期合理净利率
4. 成熟期利润 = 市场规模 × 市占率 × 净利率
5. 合理PE（参考全球可比公司）
6. 终局市值 = 成熟期利润 × 合理PE
7. 比较当前市值 vs 终局市值

**全球可比公司PE参考：**
- AI基础设施龙头：30-60倍
- 制造业龙头：15-25倍
- 消费品龙头：20-30倍

## C-5. 十书风控（精简版）

只看三个维度：
1. **格雷厄姆**：PE是否极端过高（>200倍+增速放缓→泡沫信号）
2. **利弗莫尔**：走势是否过热（天量天价/冲高回落/涨幅>300%）
3. **马克斯**：情绪是否极端（所有人一致看好→警惕）
4. **唐朝**：财报是否有隐患（毛利率下降/现金流恶化）

## C-6. 关键验证指标（必须持续跟踪）

1. **毛利率趋势**（下降 = 危险信号）
2. **市占率变化**（下降 = 龙头地位动摇）
3. **下游资本开支**（下降 = 需求见顶）
4. **利润增速**（<30% = 从C型降级为B型）

## C型降级规则

以下任一条件触发，降级为B型（按十书原版重新评估）：
- 毛利率连续两个季度下降
- 市占率被竞争对手追赶
- 利润增速降到30%以下
- 下游客户缩减资本开支
