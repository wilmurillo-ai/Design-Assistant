# 十书详细分析要点

## 1.《投资中最简单的事》邱国鹭 — 三个基本问题

- **估值**：便宜吗？PE/PB在历史什么位置？同行对比？
- **品质**：有定价权吗？行业竞争格局？是不是"月亮"？
- **时机**：现在买合适吗？政策面、资金面、情绪面？
- 📌 建议：买入/观望/回避

## 2.《手把手教你读财报》唐朝 — 财报排雷

- 审计意见是否标准无保留？
- 经营现金流 vs 净利润（利润含金量）
- 应收账款增速 vs 营收增速
- 存货周转率趋势
- 现金流肖像（奶牛型？）
- 📌 建议：财报健康/需警惕/有雷

## 3.《聪明的投资者》格雷厄姆 — 安全边际

- 当前价格 vs 估算内在价值的折扣率
- 距离安全边际有多远？
- 属于防御型还是进取型投资？
- 📌 建议：有安全边际/合理/无安全边际

## 4.《彼得·林奇的成功投资》— 公司分类与PEG

- 六种公司类型：缓慢增长/稳定增长/快速增长/周期型/困境反转/隐蔽资产
- PEG = PE ÷ 盈利增长率
- 生活中能否感知到产品/服务？
- **周期股特殊法则**：在PE最低时（利润最高点）要小心
- 📌 建议：买入/等更好的价格/不适用

## 5.《怎样选择成长股》费舍 — 成长性评估

- 市场空间天花板够高吗？
- 研发投入有效吗？
- 利润率行业领先吗？
- 管理层诚信度如何？
- 有没有护城河？
- 📌 建议：值得长期持有/短期关注/不具备成长性

## 6.《股票作手回忆录》利弗莫尔 — 交易纪律

- 当前走势是试水建仓还是已经加速？
- 止损位应该设在哪？
- 仓位建议（试探/加仓/减仓）
- **天量天价+冲高回落=逃命信号**
- **涨幅>300%+涨停=不追**
- 📌 建议：右侧追入/等回调/不追高

## 7.《投资最重要的事》马克斯 — 风险与逆向

- **第二层次思维**：大家怎么看？反过来是什么？
- 当前市场情绪钟摆位置
- 最大的风险是什么？概率多大？
- **"这次不一样"是最危险的四个字**
- 📌 建议：逆向买入/等恐慌/过度乐观需谨慎

## 8.《股市真规则》多尔西 — 行业分析

- 所属行业用什么估值方法？
- 有没有护城河？哪种类型？（无形资产/规模经济/转换成本/网络效应）
- 同行对比排名？
- 📌 建议：行业龙头可买/行业中游观望/行业底部回避

## 9.《周期》马克斯 — 周期位置判断

- 处于经济周期什么位置？
- 信贷环境如何？
- 行业周期在什么阶段？
- **上行周期的高点往往伴随极端价格**
- 📌 建议：周期底部可买/周期中段持有/周期顶部回避

## 10.《漫步华尔街》马尔基尔 — 指数基准

- 主动买这只股票 vs 买行业ETF/宽基指数，哪个更划算？
- 这只股票值得花时间研究吗？
- 📌 建议：值得主动买入/不如买指数/不值得研究
