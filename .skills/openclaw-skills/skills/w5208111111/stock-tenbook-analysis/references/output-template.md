# 标准输出模板

## 输出结构

### 零、公司分类声明

```
> 📊 公司分类：A型（价值型）/ B型（成长型）/ C型（爆发型）/ D型（概念型）
> 分类依据：利润增速XX% + PE XX倍 + [产业趋势/毛利率趋势/其他]
```

### 一、基础数据

**行情数据表：**
| 项目 | 数据 |
|------|------|
| 最新价 | XX元 |
| 总市值 | XX亿 |
| PE(TTM) | XX倍 |
| PB | XX倍 |
| 52周最高 | XX |
| 52周最低 | XX |
| ROE | XX% |

**最新财报表**

**PEG计算表（B型/C型必填）**

### 二、十书/C型分析

每本书：📘 书名 + 作者，✅⚠️❌标记，📌建议

### 三、投票汇总表

**A/B型投票表：**
| 视角 | 建议 | 核心理由 |
|------|------|----------|
| 邱国鹭 | ✅/⚠️/❌ | 一句话 |
| ... | ... | ... |

**C型投票表：**
| 维度 | 评分/建议 | 核心理由 |
|------|----------|----------|
| 产业趋势（/10） | X分 | 一句话 |
| 竞争格局（/10） | X分 | 一句话 |
| PEG | ✅/⚠️/❌ | 数值+评价 |
| 终局估值 | ✅/⚠️/❌ | 上涨空间 |
| 毛利率趋势 | ✅/⚠️/❌ | 提升稳定或下降 |
| 利润增速 | ✅/⚠️/❌ | 增速及持续性 |
| 利弗莫尔（风控） | ✅/⚠️/❌ | 走势判断 |
| 马克斯（风控） | ✅/⚠️/❌ | 情绪判断 |
| 唐朝（风控） | ✅/⚠️/❌ | 财报判断 |

### 四、综合评级

**评级 + 综合判断（2-3段）**

**操作方案表：**
| 操作 | 价格区间 | 仓位 | 说明 |
|------|----------|------|------|
| 首批建仓 | XX元 | X% | 说明 |
| 加仓 | XX元 | X% | 说明 |
| 止损 | XX元 | — | 说明 |
| 目标止盈 | XX元 | — | 说明 |

**C型附加：降级信号 + 验证指标**

### 结尾

⚠️ 以上分析不构成投资建议，市场有风险，投资需谨慎。
