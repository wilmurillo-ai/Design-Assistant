# 小红书运营知识汇总

## 发帖时间建议

- **最佳时间：** 工作日 12:00-13:30、19:00-21:30
- **次优时间：** 周末 10:00-12:00、14:00-17:00
- **避免时间：** 凌晨 1:00-7:00（流量低谷）

## 标签使用技巧

- 每篇笔记加 5-10 个标签
- 标签要与内容高度相关
- 混合使用：大流量词 + 细分词 + 品牌词
- 参考同行的高赞笔记使用的标签

## 封面图规范

- 尺寸：3:4（竖版）最佳
- 文字要少、醒目、突出价值
- 颜色对比要强
- 避免过度滤镜

## 标题公式

- 数字法：「5 个技巧让你 xxx」
- 痛点法：「为什么你 xxx？这是因为...」
- 悬念法：「原来 xxx 这么简单！」
- 对比法：「xxx 和 xxx 的区别，绝大多数人不知道」

## 限流自检

被限流的常见原因：
1. 笔记含有敏感词（检查是否含违禁语）
2. 短时间内发帖过多（建议每天 ≤3 篇）
3. 原创度不够（搬运/洗稿易被限）
4. 频繁删改笔记

## 数据指标参考

| 指标 | 合格 | 优秀 |
|------|------|------|
| 点赞数/阅读量 | >3% | >8% |
| 收藏数/阅读量 | >5% | >10% |
| 评论数/阅读量 | >0.5% | >2% |

## 提升互动率技巧

1. 结尾提出问题，引导评论
2. 主动在评论区回复
3. 利用投票/问答功能
4. 发布时间配合热点节点
