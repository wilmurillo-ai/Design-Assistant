# 小红书文案模板库

直接告诉我主题，我会用以下模板生成内容。你也可以指定模板编号。

---

## 模板 1：好物推荐清单

**结构：** 开头痛点 → 3-5 个好物 → 使用心得 → 结尾引导

**示例主题：** 提升居家幸福感的智能家居产品

**标签：** #智能家居 #居家好物 #生活小技巧 #科技感生活 #家居博主

---

## 模板 2：干货教程型

**结构：** 痛点导入 → 解决方案（分步骤） → 效果展示 → 总结

**示例主题：** 3 步搞定英文邮件写作

**标签：** #英语学习 #职场技能 #干货分享 #英文写作 #效率工具

---

## 模板 3：个人成长/心得型

**结构：** 经历引入 → 转折点 → 具体做法 → 现在的改变

**示例主题：** 30 天养成早起习惯

**标签：** #自我提升 #早起打卡 #好习惯养成 #时间管理 #个人成长

---

## 模板 4：合集/盘点型

**结构：** 主题引入 → N 项盘点 → 每项简短点评 → 结尾互动

**示例主题：** 2024 年最值得买的 5 款面膜

**标签：** #护肤 #面膜推荐 #美妆合集 #护肤心得 #平价好物

---

**结构：** 提出问题 → 给出答案 → 引导讨论

**示例主题：** 月薪 3000 和 30000 的人差在哪里？

**标签：** #职场干货 #职业发展 #自我反思 #成长记录 #求职技巧

## 模板 6：场景化推荐

**示例主题：** 独居女孩的安全感神器

**标签：** #独居女孩 #安全感 #智能门锁 #安防 #独居好物

---

## 模板 7：避坑/揭秘型

**结构：** 先说常见误区 → 揭秘真相 → 正确做法

**示例主题：** 99% 的人都踩过的装修坑

**标签：** #装修日记 #家居装修 #装修避坑 #室内设计 #新房装修

## 模板 8：日常 vlog 文案型

**结构：** 时间线 → 做了什么 → 感想 → 明天计划

**示例主题：** 一个人的周末怎么过

**标签：** #日常 #独居生活 #周末日常 #生活记录 #慢生活

## 模板 9：产品对比型

**结构：** 引入对比需求 → A vs B 各有优劣 → 选购建议

**示例主题：** iPad vs 笔记本电脑，选哪个好？

**标签：** #平板推荐 #iPad #笔记本推荐 #数码对比 #学习设备
