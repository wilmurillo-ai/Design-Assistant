## 一句话简介

配置好后，对 AI 说「发小红书：xxx 主题」，自动生成文案并发布。

## 快速开始

### 第一步：安装 MCP 服务

参考 `references/setup.md` 完成配置。

### 第二步：测试服务

```bash
curl -s -X POST http://localhost:18060/mcp \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","method":"initialize","params":{},"id":1}'
```

### 第三步：授权登录

对 AI 说「小红书重新登录」，扫码授权。

### 第四步：开始发帖

对 AI 说：`发小红书：分享一个提升效率的 AI 工具`
