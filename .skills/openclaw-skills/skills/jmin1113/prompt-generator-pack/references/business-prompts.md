# 商业营销类 Prompt 模板（10 个）

---

## 1. 竞品分析

```
你是一个商业分析师。请对以下竞品进行深度分析：

竞品名称：[竞品]
我的产品：[我的产品]
分析维度：
- 定价策略
- 目标用户
- 核心优势
- 主要卖点
- 营销渠道
- 用户评价

输出格式：表格 + 文字分析 + 我方差异化建议
```

---

## 2. 营销文案（痛点+解决方案）

```
你是一个效果营销文案专家。请为以下产品写一套营销文案：

产品：[产品名称]
目标用户痛点：[痛点描述]
解决方案：[产品如何解决]
促销信息：[如有优惠]

输出：
- 开头钩子（一句话引发焦虑）
- 产品介绍（如何解决痛点）
- 社会证明（可选案例）
- 行动号召（立即购买理由）
```

---

## 3. 品牌故事

```
你是一个品牌策划专家。请为一个[行业]品牌撰写品牌故事：

品牌名：[名称]
创始人背景：[背景]
创立初衰：[为什么做这件事]
核心理念：[使命/价值观]
与众不同之处：[差异化]

要求：800-1200 字，有情感共鸣，真实感人
```

---

## 4. 销售话术（电话/微信）

```
你是一个销售培训专家。请为以下场景设计销售话术：

产品：[产品]
客户画像：[什么样的人]
销售渠道：[电话/微信/面销]
销售目标：[成单/加微/预约]

输出：
- 开场白（3 秒抓住注意力）
- 需求挖掘（3-5 个问题）
- 产品介绍（FAB 法则）
- 异议处理（常见 3 个反对意见）
- 成交话术
```

---

## 5. 推广活动策划

```
你是一个活动策划专家。请为以下推广目标设计活动方案：

活动目的：[拉新/促活/GMV]
产品/品牌：[名称]
预算范围：[数字]
目标人群：[是谁]
推广渠道：[线上/线下/私域]

输出：
- 活动主题与形式
- 时间节奏安排
- 预热方案
- 执行流程
- 裂变机制
- 预期效果与 KPI
```
