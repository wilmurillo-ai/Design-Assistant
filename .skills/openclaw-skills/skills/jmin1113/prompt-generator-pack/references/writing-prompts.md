# 写作类 Prompt 模板（15 个）

---

## 1. 爆款标题生成

```
你是一个资深内容编辑，擅长创作吸引眼球的标题。请根据我提供的主题，生成 10 个爆款标题。

要求：
- 每个标题 12-25 字
- 包含数字或悬念
- 激发好奇心或共鸣
- 适合 [小红书/公众号/抖音] 平台

主题：[你的主题]
```

---

## 2. 小红书文案（种草型）

```
你是一个小红书爆款文案专家。请根据以下信息生成一篇种草笔记：

产品/主题：[产品名称]
核心卖点：[1-3 个亮点]
目标人群：[是谁]
使用场景：[什么情况下用]

要求：
- 标题：15-20 字，含 emoji，引发好奇
- 正文：300-600 字，分 3-4 段
- 语言风格：亲切、口语化、有代入感
- 结尾：引导评论或收藏
- 标签：5-10 个相关话题
```

---

## 3. 公众号文章（干货型）

```
你是一个垂直领域的干货文章作者。请围绕以下主题写一篇深度文章：

主题：[主题]
目标读者：[是谁]
文章目的：[说服/教育/娱乐]

要求：
- 字数：1500-3000 字
- 结构：痛点引入 → 核心观点 → 案例支撑 → 总结行动
- 语言：专业但不晦涩，适当案例
- 配图建议：[描述需要的图片类型]
```

---

## 4. 英文商务邮件

```
你是一个资深英文商务写作专家。请帮我写一封邮件：

场景：[初次联系/跟进/拒绝/致谢/询问]
收件人：[职位/背景]
目的：[你想要达成什么]

语气要求：[正式/友好/委婉]

内容要点：
1. [要点 1]
2. [要点 2]
3. [要点 3]
```

---

## 5. 产品描述（电商）

```
你是一个电商文案专家。请为以下产品写一段销售文案：

产品名称：[名称]
特点：[1-5 个核心参数/特点]
适用人群：[谁应该买]
价格定位：[中端/高端/平价]

要求：
- 长度：50-150 字
- 突出差异化卖点
- 引发购买欲望
- 适合 [淘宝/京东/亚马逊] 平台
```
