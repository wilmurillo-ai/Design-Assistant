# Prompt Generator Pack（爆款 Prompt 套装）

## 这是什么

50+ 精选 Prompt 模板库，涵盖常用场景。复制到 ChatGPT / Claude / OpenClaw 等 AI 工具直接使用。

---

## 使用方法

直接说「用 xxx 模板」或「给我一个 xxx Prompt」，我来帮你生成并解释。
