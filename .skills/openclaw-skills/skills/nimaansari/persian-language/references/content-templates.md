# Persian Content Templates

Ready-to-use structural templates for common Persian content types. Adapt the register, tone, and details to the specific context.

---

## 1. Formal Business Email (ایمیل رسمی)

### Structure

```
موضوع: [موضوع ایمیل]

[نام و عنوان گیرنده] + عزیز،
(جناب آقای دکتر احمدی / سرکار خانم مهندس رضایی)

با سلام و احترام،

[پاراگراف اول: هدف ایمیل — چرا می‌نویسید]
بدین‌وسیله به استحضار می‌رساند که…
پیرو مکاتبه مورخ ۱۴۰۵/۰۱/۱۵…
با توجه به جلسه مورخ…

[پاراگراف دوم: جزئیات و توضیحات]

[پاراگراف سوم: درخواست یا اقدام مورد انتظار]
خواهشمند است دستور فرمایید…
ممنون می‌شوم در صورت امکان…
لطفاً نتیجه را به این آدرس ارسال فرمایید.

با تشکر و احترام،
[نام]
[سمت]
[اطلاعات تماس]
```

### Example

```
موضوع: درخواست هماهنگی جلسه بررسی پروژه

جناب آقای مهندس محمدی عزیز،

با سلام و احترام،

پیرو صحبت‌های انجام‌شده در جلسه مورخ ۱۴۰۵/۰۱/۱۰، بدین‌وسیله
درخواست می‌شود جلسه‌ای جهت بررسی وضعیت پروژه در هفته جاری
هماهنگ گردد.

پیشنهاد می‌شود جلسه روز سه‌شنبه ۱۴۰۵/۰۱/۲۴ ساعت ۱۰ صبح
برگزار شود. در صورتی که زمان دیگری مناسب‌تر است، لطفاً
اعلام فرمایید.

با تشکر و احترام،
علی رضایی
مدیر پروژه
a.rezaei@example.com
```

## 2. Semi-Formal Email (ایمیل نیمه‌رسمی)

### Structure

```
سلام [نام]،

امیدوارم حالت خوب باشه.

[متن اصلی — مستقیم و صمیمی ولی محترمانه]

ممنون می‌شم اگه [درخواست].

مرسی،
[نام]
```

### Example

```
سلام مریم جان،

امیدوارم حالت خوب باشه.

می‌خواستم بدونم فایل طراحی که قرار بود بفرستی آماده شده یا نه؟
اگه تا فردا بفرستی عالی می‌شه چون باید تا آخر هفته نهایی‌اش کنیم.

ممنون،
سارا
```

## 3. Social Media — Instagram Caption

### Structure

```
[خط اول: هوک — جمله‌ای جذاب که توجه جلب کند]

[متن اصلی: ۲-۴ خط، محاوره‌ای، مرتبط با تصویر/ویدیو]

[دعوت به اقدام: سؤال یا درخواست تعامل]

[هشتگ‌ها: ترکیبی از فارسی و انگلیسی]
```

### Example (Cafe/Lifestyle)

```
یه قهوه خوب می‌تونه کل روزت رو نجات بده ☕

امروز رفتم یه کافه جدید که دوستم معرفی کرده بود و واقعاً
غافلگیر شدم. فضاش عالیه و قهوه‌ترکیش بی‌نظیر بود.

شما کافه مورد علاقه‌تون کجاست؟ بگید تا یه سر بزنم 👇

#کافه_گردی #قهوه #تهران_گردی #کافه_تهران #coffeelover
```

### Example (Tech/Business)

```
یه اشتباه ساده که خیلی از استارتاپ‌ها مرتکب می‌شن 👇

فکر می‌کنن اول باید محصول کامل بسازن، بعد برن سراغ مشتری.
ولی واقعیت اینه که اول باید مشکل رو اعتبارسنجی کنید.
MVP یعنی همین: کمترین چیز برای بیشترین یادگیری.

تجربه شما چیه؟ اول محصول ساختید یا اول با مشتری حرف زدید؟

#استارتاپ #کارآفرینی #MVP #startup #بیزینس
```

## 4. Telegram Channel Post

### Structure

```
[عنوان با ایموجی]

[متن: کوتاه، مستقیم، ۳-۵ خط]

[لینک یا منبع اگر لازم است]

[دکمه/لینک دعوت به اقدام]
```

### Example

```
🔥 آپدیت جدید اپلیکیشن منتشر شد!

نسخه ۲.۵ با این تغییرات آماده شده:
• رابط کاربری کاملاً بازطراحی شده
• سرعت بارگذاری ۳ برابر بیشتر
• اضافه شدن حالت تاریک

همین الان آپدیت کنید 👇
[لینک دانلود]
```

## 5. Formal Announcement (اطلاعیه رسمی)

### Structure

```
بسم‌الله الرحمن الرحیم (optional — common in Iranian official docs)

اطلاعیه

[موضوع اطلاعیه]

به اطلاع [مخاطبان] می‌رساند که…

[جزئیات: زمان، مکان، شرایط، الزامات]

[اقدام مورد نیاز از مخاطب]

[اطلاعات تماس برای سؤالات]

[امضا / مهر سازمان]
تاریخ: [تاریخ شمسی]
```

### Example

```
اطلاعیه

به اطلاع کلیه کارکنان محترم می‌رساند:

جلسه عمومی سالانه شرکت روز چهارشنبه مورخ ۱۴۰۵/۰۱/۲۵
ساعت ۱۴:۰۰ در سالن اجتماعات طبقه پنجم برگزار خواهد شد.

حضور کلیه مدیران و سرپرستان الزامی است. سایر همکاران
نیز می‌توانند در صورت تمایل شرکت نمایند.

جهت هماهنگی با واحد اداری تماس حاصل فرمایید.
داخلی: ۲۳۴

مدیریت منابع انسانی
تاریخ: ۱۴۰۵/۰۱/۲۰
```

## 6. Product/Service Description (توضیحات محصول)

### Structure

```
[نام محصول/خدمت]

[یک جمله خلاصه: این محصول چیست و چه مشکلی حل می‌کند]

[ویژگی‌های کلیدی — بولت‌پوینت]
• ویژگی ۱
• ویژگی ۲
• ویژگی ۳

[مزیت رقابتی / چرا این محصول]

[قیمت اگر لازم است]

[دعوت به اقدام: خرید، ثبت‌نام، تماس]
```

### Example

```
اپلیکیشن مدیریت پروژه «تسکینو»

مدیریت تیم و پروژه‌هاتون رو ساده کنید — همه‌چیز در یک جا.

• ایجاد تسک و تعیین مسئول با یک کلیک
• نمودار گانت فارسی با تقویم شمسی
• گزارش‌گیری خودکار هفتگی
• اپ موبایل برای اندروید و iOS
• پشتیبانی ۲۴ ساعته به زبان فارسی

برخلاف ابزارهای خارجی، تسکینو از ابتدا برای تیم‌های ایرانی
طراحی شده و با تقویم شمسی، زبان فارسی و نیازهای بومی سازگاره.

۳۰ روز رایگان امتحان کنید — بدون نیاز به کارت بانکی.

taskino.ir/register
```

## 7. Formal Report Section (بخش گزارش رسمی)

### Structure

```
[عنوان بخش]

[مقدمه: هدف این بخش از گزارش]

[یافته‌ها / داده‌ها / تحلیل]

[جدول یا نمودار اگر لازم است]

[نتیجه‌گیری بخش]
```

### Example

```
۳. بررسی عملکرد فروش فصل زمستان ۱۴۰۴

در این بخش، عملکرد فروش شرکت در فصل زمستان ۱۴۰۴
(دی، بهمن و اسفند) مورد بررسی قرار می‌گیرد.

مجموع فروش فصل زمستان به ۱۲٫۵ میلیارد تومان رسید که
نسبت به فصل مشابه سال قبل ۱۸ درصد رشد نشان می‌دهد.

| ماه | فروش (میلیارد تومان) | رشد نسبت به سال قبل |
|---|---|---|
| دی ۱۴۰۴ | ۳٫۸ | ۱۵٪ |
| بهمن ۱۴۰۴ | ۴٫۲ | ۲۰٪ |
| اسفند ۱۴۰۴ | ۴٫۵ | ۱۹٪ |

بیشترین رشد مربوط به بهمن‌ماه بوده که عمدتاً به دلیل
کمپین تخفیف ویژه یلدا و افزایش تقاضای آنلاین است.
```

### Structure

```
[وضعیت: ⚠️ / ✅ / 🔧]

[چه اتفاقی افتاده / چه کاری در حال انجام است]

[زمان تخمینی رفع مشکل]

[اقدام جایگزین اگر وجود دارد]

[عذرخواهی + تشکر]
```

### Example

```
🔧 اطلاعیه تعمیرات

سامانه پرداخت به دلیل به‌روزرسانی زیرساخت، امروز
(۱۴۰۵/۰۱/۲۱) از ساعت ۲:۰۰ تا ۶:۰۰ بامداد در دسترس
نخواهد بود.

در این بازه زمانی می‌توانید از درگاه جایگزین به آدرس
pay2.example.ir استفاده کنید.

از صبر و شکیبایی شما سپاسگزاریم.
```

## Notes on Template Usage

- **Register consistency:** All text within a single piece should stay in one register. Don't mix formal body with casual sign-off.
- **Dates:** Always use Solar Hijri (شمسی) for Iranian audiences. Add Gregorian in parentheses only if the audience is international.
- **Numbers:** Follow the numeral rules from `writing-standards.md` — Persian numerals in prose, Western in technical/code contexts.
- **Half-spaces:** Apply throughout — templates show the correct form but double-check generated output.
- **Guillemets:** Use «» for any quoted text in formal and standard registers.
