# Causal Chain Mapping: From Event to Global Effect

## Purpose

Most analysts stop at the immediate consequence of an event. Geopolitics is about **second- and third-order effects**: the ripples that spread for months and years.

Causal chain mapping traces: **Initial event → 6-month effects → 12-month effects → 24-month effects**, with probability and confidence at each stage.

**Example chain:**
```
Ethiopia fills GERD reservoir
→ (6 months) Egypt water crisis
→ (12 months) Sudan pressure + Egypt domestic instability
→ (24 months) Regional war risk / US-Egypt alliance stress
```

## Causal Chain Mapping Principles

### 1. Probability Compounds
If each Tier has 70% probability → 3 tiers = 0.7 × 0.7 × 0.7 = 34% final probability

Early mistakes cascade. An error in Tier 1 makes all downstream tiers wrong.

### 2. Branching Paths
At each Tier, multiple outcomes possible. Chain splits:
```
Tier 1 → Outcome A (50%)
       ↘ Outcome B (50%)

Tier 2 → (if A) → Outcome A1 (60%) or A2 (40%)
       → (if B) → Outcome B1 (70%) or B2 (30%)

Tier 3 → (if A1) → specific Tier 3 outcome
       → (if A2) → different Tier 3 outcome
       → ...
```

This rapidly becomes a tree. Decision trees are useful here.

### 3. Feedback Loops
Some chains double back:
```
Military escalation (Tier 2)
→ Sanctions (Tier 2)
→ Economic crisis (Tier 3)
→ Domestic instability (Tier 3)
→ Leadership change (Tier 4)
→ New military leader (Tier 4)
→ Re-escalation (Tier 1 again — cycle)
```

Note feedback loops explicitly. They create spiral dynamics (escalation or de-escalation becoming self-sustaining).

### 4. Intervening Events
External shocks can break a chain:
```
Predicted Tier 3: Economic restructuring
Intervening event: Oil price collapse (unrelated to our event)
Actual Tier 3: Much more severe economic restructuring (cheaper oil enables evasion of energy sanctions)
```

Note: "What external events could derail this chain?"

**Triggered:** In Step 9 (Output), for FULL assessments when major events are analyzed

- Actor responses (Step 5)
- Historical patterns of similar events (Step 6)

**Output:**
- Causal chain visualization (4-tier structure)
- Key decision points (where does chain diverge?)
- Divergence probabilities (which path is most likely?)

**Use case:** Strategic planners who need to think 12-24 months ahead about how today's event could reshape the system.
