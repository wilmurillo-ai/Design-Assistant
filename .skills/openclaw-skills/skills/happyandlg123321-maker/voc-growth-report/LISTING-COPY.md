# ClawHub Listing Copy

## Name
voc-growth-report

## Tagline
Turn Xiaohongshu and social-comment CSV exports into VOC insight, commercial opportunity analysis, Feishu-ready workflows, and boss-ready HTML report links.

## Short description
A delivery-first skill for turning exported social media comments — especially Xiaohongshu / 社媒助手 CSVs — into actionable VOC and growth outputs. Instead of stopping at sentiment or keywords, it helps extract audience segments, satisfied / unsatisfied / latent needs, commercial opportunity, value propositions, Feishu-ready comment library structure, and boss-ready HTML report delivery prompts that return links instead of raw code.

## Why this skill exists
Most comment-analysis workflows fail in one of two ways:
1. they stop at shallow outputs like sentiment, keywords, or word clouds;
2. they generate HTML code, but do not complete the final delivery step.

This skill is built for the real operating path:
**社媒助手抓取 → CSV导出 → agent分析与生成报告 → 返回可访问链接 → 飞书沉淀与跟进**

That makes it useful not just for analysis, but for real business execution.

## Core value
This skill helps turn raw comments into:
- user segmentation,
- need discovery,
- commercial opportunity scoring,
- value proposition mapping,
- growth action recommendations,
- Feishu/Bitable operating structure,
- and executive-ready HTML report delivery.

## Best use cases
- Analyze Xiaohongshu / 小红书 comment CSV exports
- Turn 社媒助手 comment exports into VOC and growth insight
- Build boss-ready HTML decision reports from comment data
- Create Trae / Cursor / Claude Code / Codex prompts that return report links, not raw HTML
- Design a Feishu Bitable comment library for manual review and follow-up
- Upgrade comment analysis from sentiment-only to segmentation + needs + opportunity + action

## What makes it competitive
### 1. Delivery-first, not code-first
It does not stop at “generate HTML”. It pushes toward saving the report, previewing it, and returning a usable link/path.

### 2. VOC + growth, not just sentiment
It goes beyond emotion labels and keyword clouds to cover segmentation, needs, commercial intent, conversion blockers, and value propositions.

### 3. Built around a real workflow
It matches the workflow operators actually run:
**抓取评论 → 导出CSV → 交给代理生成报告 → 获取链接 → 飞书沉淀**

### 4. Useful after analysis, not only during analysis
It includes Feishu/Bitable structure guidance so the output can continue into operations, follow-up, and review.

## Ideal users
- founders and operators who want executive-ready comment insight
- social media teams analyzing Xiaohongshu comments
- marketing and growth teams turning comment exports into decisions
- AI operators using Trae / Cursor / Claude Code / Codex for delivery
- teams storing comment workflows in Feishu / Bitable
