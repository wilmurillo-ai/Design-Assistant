# Trae / ACP Delivery Prompt Template

Use this template when the user wants a coding agent such as Trae, Cursor, Claude Code, or Codex to turn a social-comment CSV into a boss-ready HTML report and return a preview link instead of raw source.

```text
你的任务不是只输出分析结果或 HTML 源码，而是完成一份“可直接访问的网页报告”交付。

## 输入
我会提供一个由社媒助手导出的 CSV 文件，里面包含小红书评论数据。

## 目标
请基于 CSV 数据完成以下完整流程：
1. 读取并理解 CSV 字段
2. 对评论做用户洞察与增长分析
3. 生成一份高质量 HTML 网页报告
4. 将 HTML 落地为本地文件
5. 启动本地预览/静态服务
6. 返回一个可直接打开的访问链接

如果不能提供公网链接，也必须返回：
- 本地访问链接
- HTML 文件路径

如果你只输出源码，视为未完成任务。

---

## 分析角色
你是一名资深科特勒式增长战略顾问兼 VOC（用户之声）分析师。
请严格基于评论内容分析，不得脱离原文凭空猜测。

---

## 分析任务
请完成以下分析模块：

### 1. 用户情绪总览
- 正向 / 中性 / 负向占比
- 负面核心吐槽点 Top 5
- 正面核心满意点 Top 5

### 2. 用户分群分析
将评论用户划分为不同人群，例如：
- 价格敏感型
- 功能导向型
- 效果导向型
- 小白入门型
- 高意向购买型
- 观望犹豫型
- 竞品比较型
- 无效围观型

输出：
- 各类人群数量与占比
- 每类人群核心特征
- 每类代表评论
- 每类最在意的价值点

### 3. 用户需求图谱
分为：
- 已被满足的需求
- 未被满足的需求
- 潜在需求

输出：
- 每类需求数量与占比
- 每类需求核心总结
- 每类需求的典型评论
- 分类原因说明

### 4. 商机与转化机会
商机等级定义：
- 高：明确咨询价格、购买方式、联系方式、合作、试用、下单
- 中：明确咨询功能、效果、适用人群、区别、使用方法
- 低：普通兴趣表达、轻度认可、一般互动
- 无：灌水、无关内容、纯表情

输出：
- 高 / 中 / 低 / 无 商机占比
- 高商机评论 Top 20
- 成交前高频问题 Top 10
- 当前最大转化阻碍 Top 5
- 最值得优先跟进的人群与原因

### 5. 价值主张与增长建议
输出：
- 当前最值得放大的 Top 3 卖点
- 当前最值得优先解决的 Top 5 需求
- 当前最值得做的 Top 5 内容选题
- 当前最值得优化的 Top 3 产品/服务问题
- 当前最适合销售使用的 Top 3 转化话术

### 6. CEO Summary
输出：
- 当前最值得优先服务的人群是谁
- 当前最值得优先解决的问题是什么
- 当前最值得放大的卖点是什么
- 接下来 7 天最值得立刻执行的 3 个动作

---

## 报告生成要求
请将分析结果整合成一份高质量网页报告。

### 报告标题
《小红书 VOC 洞察与增长决策报告》

### 报告结构
1. 封面 / 数据概况
2. 用户情绪总览
3. 用户分群分析
4. 用户需求图谱
5. 商机与转化机会
6. 价值主张与增长建议
7. CEO Summary

### 视觉风格
- 现代、专业、简洁、高级
- 适合老板直接阅读
- 卡片式布局
- 图表清晰、美观易读
- 页面整体应像正式商业汇报
- 词云图可作为辅助模块，但不要作为核心

### 技术要求
- 生成完整单文件 HTML
- CSS / JS 内嵌
- 页面可本地独立运行
- 优先保证可读性与交付完整性

---

## 交付要求
生成 HTML 后，请继续完成：
1. 将 HTML 保存到本地文件
2. 启动一个本地静态服务或可访问预览
3. 返回最终交付信息

---

## 最终只输出以下内容
访问链接：
本地文件路径：
报告标题：
简短说明：

如果某一步失败，不要中止；请至少完成到“本地 HTML 文件已生成且可打开”的程度。
```
