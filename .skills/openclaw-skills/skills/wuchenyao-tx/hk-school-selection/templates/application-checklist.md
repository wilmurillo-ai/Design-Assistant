# 升中申請清單

## 準備階段（2025 年 9 月 -12 月）

### 資料收集
- [ ] 收集小五、小六成績單
- [ ] 準備在讀證明
- [ ] 整理獲獎證書和課外活動證明
- [ ] 準備身份證明文件（學生及家長）
- [ ] 準備證件照片（4-6 張）
- [ ] 準備港澳通行證或回鄉證副本

### 學校研究
- [ ] 瀏覽 schooland.hk 了解學校數據
- [ ] 列出心儀學校名單（10-15 所）
- [ ] 研究學校收生準則和辦學理念
- [ ] 了解學校語言支援情況（特別是對不會粵語學生）
- [ ] 參加學校開放日（如有）
- [ ] 致電學校詢問語言支援政策

### 語言準備
- [ ] 評估孩子粵語水平
- [ ] 如不會粵語，開始學習基礎粵語
- [ ] 如申請全普教中學校，準備普通話面試
- [ ] 練習自我介紹（中英文）

### 策略制定
- [ ] 確定衝刺學校（2-3 所 Band 1）
- [ ] 確定主力學校（3-4 所 Band 2）
- [ ] 確定保底學校（1-2 所 Band 3）
- [ ] 考慮一條龍學校（如有）
- [ ] 考慮不會粵語專項策略

## 自行分配學位申請（2026 年 1 月）

### 申請前
- [ ] 收到《申請中一自行分配學位手冊》
- [ ] 研究手冊中學校名單
- [ ] 確認申請學校（最多 2 所）
- [ ] 檢查申請資格

### 申請中
- [ ] 填寫申請表
- [ ] 準備所需文件
- [ ] 遞交申請（1 月 2-16 日）
- [ ] 繳交申請費（如適用）
- [ ] 保留申請收據

### 面試準備
- [ ] 了解面試形式（個人/小組）
- [ ] 練習自我介紹
- [ ] 模擬面試場景
- [ ] 準備常見問題回答
- [ ] 準備提問學校的問題
- [ ] 準備服裝（整潔端莊）

### 面試後
- [ ] 記錄面試情況
- [ ] 跟進面試結果
- [ ] 如獲取錄，確認入學
- [ ] 如未獲取錄，準備統一派位

## 統一派位申請（2026 年 4 月 -5 月）

### 選校前
- [ ] 收到《中一派位選擇學校表格》
- [ ] 研究校網內學校名單
- [ ] 了解派位組別（Band 1/2/3）
- [ ] 考慮隨機編號因素

### 選校策略
- [ ] 甲部（不受校網限制）：填寫 1-2 所全港理想學校
- [ ] 乙部（按學校網分配）：填寫 10-20 所校網內學校
- [ ] 按優先次序排列
- [ ] 避免誤中目標（不喜歡的學校）
- [ ] 「進可攻、退可守」策略

### 填表
- [ ] 填寫表格
- [ ] 檢查填寫是否正確
- [ ] 家長簽名
- [ ] 遞交表格（4 月上旬至 5 月上旬）

## 結果公布（2026 年 7 月）

### 7 月 7 日
- [ ] 等待派位結果
- [ ] 透過小學或學校查詢結果
- [ ] 收到《中一學位派位結果通知書》

### 結果處理
- [ ] 如獲派理想學校，確認入學
- [ ] 如對結果不滿意，考慮：
  - [ ] 向學校查詢轉班機會
  - [ ] 考慮轉校（需符合條件）
  - [ ] 報讀直資/私立學校

## 入學準備（2026 年 8 月 -9 月）

### 註冊
- [ ] 按時到校註冊
- [ ] 繳交學費和雜費
- [ ] 購買校服和書簿
- [ ] 參加開學禮

### 適應
- [ ] 了解學校規則和課表
- [ ] 認識老師和同學
- [ ] 參加迎新活動
- [ ] 適應新環境

### 語言支援（如適用）
- [ ] 參加語言加強班（如有）
- [ ] 尋找同學輔導
- [ ] 繼續學習粵語/普通話
- [ ] 與老師溝通學習困難

## 特別注意事項

### 不會粵語學生
- [ ] 確認學校有語言支援
- [ ] 了解是否有內地生專班
- [ ] 準備普通話面試
- [ ] 提前學習基礎粵語

### 一條龍學生
- [ ] 確認直升資格
- [ ] 了解中學基本成績要求
- [ ] 準備升中面試（如有）

### 直資/私立學生
- [ ] 了解學費和資助政策
- [ ] 準備申請材料
- [ ] 了解面試要求
- [ ] 考慮經濟負擔
