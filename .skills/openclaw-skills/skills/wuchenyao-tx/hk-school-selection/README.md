## 📋 核心功能

### 1. 學校數據查詢
- **基础数据库**：已生成 `schools.json`，包含 37 所一条龙学校及常见名校（状态：待核实）。
- **分类筛选**：地区、Banding、教学语言、性别、宗教、办学团体。
- **学校详情**：学费、班级结构、教学语言、收生情况、升学表现（需人工核实）。

### 2. Banding 分析
- **划分原理**：非官方分类，参考中一新生源和中六毕业成绩。
- **分级**：Band 1A/1B/1C, Band 2A/2B/2C, Band 3A/3B/3C。
- **判断方法**：收生源頭、公开试成绩、英文班开设情况。

### 3. 升中派位指导
- **三回合机制**：自行分配 (30%)、统一甲部 (7%)、统一乙部 (59%)。
- **时间表**：1 月申请、4-5 月选校、7 月公布结果。
- **满意率**：整体 95%, 首志愿 85%。

### 4. 一條龍学校
- **定义**：小学和中学结龙，小六学生可直升。
- **数量**：38 所 (16 资助 + 15 直资 + 7 私立)。
- **优势**：稳定、减轻升中压力。

### 5. 特殊需求选校
- **不会粤语**：首选全普教中（培僑、漢華），次选有语言支援。
- **英语较弱**：优先选择中文中学，有数学拔尖班。
- **语言支援标注**：🗣️全普教中, 🗣️普通话辅助, 🌏内地生友好。

### 6. 学费比较
- **官立/资助**：免学费，杂费 HK$2,000-5,000/年。
- **直资**：HK$10,000-60,000/年。
- **私立/国际**：HK$50,000-200,000+/年。

### 7. 办学团体
- **主要团体**：基督教 147, 天主教 88, 佛教 23, 东华三院 18, 保良局 17, 圣公会 32。

## ⚠️ 注意事项

1. **Banding 非官方**: 坊间分类，仅供参考。
2. **数据时效性**: 学校收生情况年年不同，需核实最新信息。
3. **个性化建议**: 需结合孩子具体情况。
4. **地区因素**: 考虑交通、住宿、家庭位置。
5. **经济因素**: 学费、杂费、补习费预算。
