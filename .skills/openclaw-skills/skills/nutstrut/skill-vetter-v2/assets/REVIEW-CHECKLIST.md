# Review Checklist

## Read first
- SKILL.md
- README.md
- _meta.json
- all scripts
- all hooks
- references and assets

## Capability review
- file reads
- file writes
- command execution
- package installs
- network access
- data handling
- secrets or credential access

## Red flags
- obfuscation
- unexplained network calls
- privilege escalation
- persistence mechanisms
- broad access to memory or secret stores

## Decision
- safe
- caution
- unsafe
