# OpenClaw Turbo-Bundle: Groq, OpenRouter & Elite Bilingual TTS 🚀

The ultimate upgrade for your OpenClaw agent. Combines the lightning-fast speed of **Groq**, the flexibility of **OpenRouter**, and studio-quality **Voice AI**—all while staying **100% cost-free**.

## What's Inside:
1.  **Multi-Provider AI**: Pre-configured access to **Groq** and **OpenRouter** models.
2.  **Elite Voice AI (TTS)**: Professional Saudi Arabic and English voices.
3.  **Smart Cost Optimization**: Automatically uses free models (Free-Ride) to ensure zero usage costs.
4.  **Auto-Setup Scripts**: Get everything running in seconds.

## Quick Start

### 1. Skill Setup
Copy the `skills/groq-arabic-tts` folder into your OpenClaw `workspace/skills/` directory.

### 2. Dependency Automation
Run the configuration script to install Python libraries:
```bash
python configure.py
```

### 3. API Keys
Rename `ENV_EXAMPLE.txt` to `.env` (inside the folder) and add your `GROQ_API_KEY`.
Ensure your `OPENROUTER_API_KEY` is set in your environment variables.

### 4. Model Import
The `config/MODELS_JSON_EXAMPLE.txt` file contains the exact codes to copy/paste into your `models.json`.
