# OpenClaw Groq Ultimate Bundle 🚀🇸🇦🇺🇸

Ce pack tout-en-un permet d'ajouter la puissance de **Groq** et l'optimisation **Free-Ride** à votre agent OpenClaw.

## Inclus dans ce pack :
1.  **Skill TTS Multi-Voix** : Arabe (Saoudien) et Anglais avec des voix professionnelles (Fahad, Lulwa, Autumn, Troy, etc.).
2.  **Configuration Groq** : Modèles Llama 3 et Mixtral ultra-rapides.
3.  **Optimisation Free-Ride** : Utilisation automatique des modèles gratuits avec basculement intelligent.

## Installation Simple

### 1. Préparation
Copiez le dossier `skills/groq-arabic-tts` dans votre dossier `workspace/skills/` d'OpenClaw.

### 2. Dépendances & Automatisation
Lancez le script de configuration :
```bash
python configure.py
```

### 3. Clés API
- Renommez `.env.example` en `.env` dans le dossier de la skill et ajoutez votre `GROQ_API_KEY`.
- Assurez-vous d'avoir votre `OPENROUTER_API_KEY` dans vos variables d'environnement.

### 4. Configuration des Modèles
Copiez les modèles depuis `config/models.json.example` vers votre fichier `models.json` habituel.

## Utilisation
Demandez simplement à votre agent :
- *"Dis-moi 'Bonjour' en vocal saoudien avec la voix de Lulwa"*
- *"Say 'Welcome' in English with the voice of Autumn"*
