```

```

女频URL格式：https://nvsheng.qidian.com/rank/{榜单}/

## 常见用户需求 → 推荐策略

```
"爽文"     → 都市月票榜 + 玄幻月票榜，筛选简介含"系统""无敌""龙傲天"元素
"硬核"     → 科幻榜 + 奇幻榜，筛选大字数（>200万）
"轻松"     → 都市榜 + 轻小说榜，筛选字数适中（<200万）
"虐文"     → 女频古代言情 + 玄幻言情
"甜文"     → 女频现代言情，筛选简介含"甜""宠"元素
"完结"     → 直接用完本精品页，不用月票榜
"经典"     → 月票榜历史冠军：《诡秘之主》《大奉打更人》《遮天》等
```
