# Smart Skill Advisor

Turn a goal into the right skill stack safely.

## What it does

This skill helps you:
- Translate a user goal into the right skills
- Compare options instead of dumping search results
- Combine multiple skills into one practical workflow
- Reduce bad installs by calling out tradeoffs, trust, and risk

## Positioning

This is not just a skill finder.

It acts like a skill architect:
- Understand the goal
- Find candidate skills
- Filter and rank them
- Recommend the smallest useful stack
- Explain why each one is included
- Ask before giving install guidance

## Why it's different from find-skills

- Not just search -> decision support
- Not just one skill -> full stack recommendation
- Not just popularity -> fit, trust, and setup cost
- Not just install guidance -> workflow design with explicit approval boundaries

## Example

Input:
"I want to automate research and turn it into weekly reports"

Output:
- Recommended stack
- Step-by-step execution plan
- Reasons for each choice
- Alternatives considered
- Install guidance after user confirmation

## Perfect for

- Developers
- Builders
- AI beginners
- Users who do not know which skills to combine
- Teams that want a safer, clearer setup path
