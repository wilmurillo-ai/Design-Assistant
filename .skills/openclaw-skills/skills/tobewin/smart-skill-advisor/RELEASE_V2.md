# Smart Skill Advisor v2.0.0

## Marketplace Summary

Turn a user goal into the right skill stack safely.

Smart Skill Advisor does not just find skills. It recommends the smallest useful stack, explains why each skill is included, highlights tradeoffs and trust signals, and only gives install guidance after explicit user approval.

Best for:
- users who do not know which skill to choose
- workflows that need multiple skills working together
- safer decision-making before installation

## Suggested Changelog

Smart Skill Advisor v2.0.0

- Repositioned the skill from simple skill search to goal-to-skill-stack advising.
- Added a deeper recommendation workflow: goal clarification, capability breakdown, candidate filtering, stack design, and alternatives analysis.
- Improved safety boundaries by requiring explicit user approval before any install guidance.
- Removed assumptions about a specific CLI being available and made discovery method environment-aware.
- Added stronger trust and risk guidance so suspicious or low-signal skills are treated more cautiously.
- Upgraded marketplace messaging to better differentiate from basic search-oriented skills.
