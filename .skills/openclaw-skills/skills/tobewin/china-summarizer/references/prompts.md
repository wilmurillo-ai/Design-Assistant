# 总结提示词模板

根据内容类型选择对应模板，将提取的文本填入 [内容文本] 处。

---

## 通用总结（默认）

```
请对以下内容进行智能总结：

[内容文本]

要求：
1. 用中文输出
2. 用1-3句话概括核心观点
3. 列出3-7条主要内容要点
4. 保留所有具体数字、时间、人名、关键结论
5. 最后用一句话总结全文精髓
6. 总字数控制在300-500字
```

---

## 技术文章/教程

```
请对以下技术内容进行结构化总结：

[内容文本]

输出格式：
- 技术主题：
- 解决的问题：
- 核心方案/方法：（列出关键步骤或技术要点）
- 注意事项/踩坑点：
- 适用场景：
```

---

## 新闻/时事简报

```
请对以下新闻内容进行简报式总结：

[内容文本]

输出格式：
- 事件：（一句话说明发生了什么）
- 时间/地点：
- 关键人物/机构：
- 事件经过：（2-3句话）
- 影响/结果：
- 背景信息：（如有）
```

---

## 学术/研究报告

```
请对以下研究内容进行学术摘要：

[内容文本]

输出格式：
- 研究问题：
- 研究方法：
- 主要发现：
- 核心数据：（保留所有具体数字）
- 结论：
- 局限性：（如有提及）
```

---

## 长文分段总结

内容超过8000字时分两步处理。

### 第一步：分段摘要

```
以下是一篇长文的第[X]段（共[N]段），请生成本段摘要：

[分段文本]

要求：用3-5句话概括本段核心内容，保留所有关键数字和结论。
```

### 第二步：汇总

```
以下是一篇长文各段的摘要，请生成最终总结：

[各段摘要]

要求：
1. 整合所有段落核心内容
2. 梳理全文逻辑结构
3. 提炼最重要的3-5个观点
4. 总字数控制在500字以内
```
