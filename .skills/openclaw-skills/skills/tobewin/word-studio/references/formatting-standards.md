# Formatting Standards

## Page Setup (页面设置)

### Margins (页边距)
- 上: 2.54cm (1 inch)
- 下: 2.54cm (1 inch)
- 左: 3.17cm (1.25 inch)
- 右: 3.17cm (1.25 inch)

### Paper Size (纸张大小)
- A4: 21cm × 29.7cm
- Letter: 8.5" × 11"

## Font Standards (字体规范)

### Chinese Fonts (中文字体)
| 用途 | 推荐字体 | 备选 |
|------|----------|------|
| 正文 | 宋体 | 仿宋 |
| 标题 | 黑体 | 微软雅黑 |
| 引用 | 楷体 | 华文楷体 |
| 公文 | 仿宋 | 宋体 |

### English Fonts (英文字体)
| 用途 | 推荐字体 | 备选 |
|------|----------|------|
| 正文 | Times New Roman | Calibri |
| 标题 | Arial | Helvetica |
| 代码 | Consolas | Courier New |

### Font Sizes (字号)
| 级别 | 中文 | 英文 |
|------|------|------|
| 一级标题 | 小二 (18pt) | 18pt |
| 二级标题 | 小三 (15pt) | 15pt |
| 三级标题 | 四号 (14pt) | 14pt |
| 正文 | 小四 (12pt) | 12pt |
| 注释 | 五号 (10.5pt) | 10pt |

## Paragraph Settings (段落设置)

### Line Spacing (行距)
- 正文: 1.5倍行距
- 标题: 单倍行距
- 引用: 1.25倍行距

### Indentation (缩进)
- 首行缩进: 2字符 (0.74cm)
- 悬挂缩进: 0
- 左缩进: 0
- 右缩进: 0

### Spacing (间距)
- 段前: 0行
- 段后: 0行

## Title Standards (标题规范)

### Academic Paper (学术论文)
```
一级标题: 黑体 小二 居中
二级标题: 黑体 小三 左对齐
三级标题: 黑体 四号 左对齐
```

### Business Report (商务报告)
```
一级标题: 黑体 小二 加粗
二级标题: 黑体 小三 加粗
三级标题: 黑体 四号 加粗
```

### Government Document (公文)
```
标题: 二号小标宋 居中
正文: 三号仿宋
一级标题: 三号黑体
二级标题: 三号楷体
```

## Table Standards (表格规范)

### Table Style
- 边框: 1pt 黑色实线
- 表头: 浅灰色背景
- 对齐: 居中对齐
- 字体: 宋体 五号

### Table Caption
- 位置: 表格上方
- 格式: "表 X-X 表格标题"
- 对齐: 居中

## Image Standards (图片规范)

### Image Caption
- 位置: 图片下方
- 格式: "图 X-X 图片标题"
- 对齐: 居中

### Image Size
- 宽度: 不超过页面宽度
- 分辨率: 300dpi以上
- 格式: PNG/JPG

## Header/Footer (页眉页脚)

### Header
- 内容: 文档标题/公司名称
- 字体: 宋体 五号
- 对齐: 居中

### Footer
- 内容: 页码
- 格式: "第 X 页"
- 对齐: 居中

## Reference Format (参考文献)

### GB/T 7714-2015
```
[1] 作者. 题名[J]. 刊名, 年, 卷(期): 页码.
[2] 作者. 书名[M]. 出版地: 出版社, 年.
[3] 作者. 题名[D]. 保存地: 保存单位, 年.
```

### APA Format
```
Author, A. A. (Year). Title of article. Journal Name, Volume(Issue), Pages.
```
