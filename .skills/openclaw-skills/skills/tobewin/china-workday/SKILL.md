# 中国工作日查询 China Workday Checker

专为职场场景设计：判断上班/放假、补班提醒、月工作日统计、假期规划。
2025–2026年完整数据内置，来源国务院办公厅官方通知，零依赖，无需网络。

## 触发时机

- "明天要上班吗" / "X月X日是工作日吗"
- "这个周末要补班吗"
- "五一怎么放" / "春节放几天"
- "4月有几个工作日"
- "距离下个假期还有几天"
- "这周连续上几天班了"

---

## 判断优先级（必须严格按此顺序）

```
1. 在补班日期列表？→ 工作日（周末也要上班）
2. 在放假日期列表？→ 休息日（工作日也放假）
3. 是周六或周日？  → 普通休息日
4. 是周一至周五？  → 普通工作日
```
