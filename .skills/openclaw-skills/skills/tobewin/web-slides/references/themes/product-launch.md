# product-launch

## Positioning

面向新品发布、功能发布、路线图发布的舞台型主题。

## Visual DNA

- 大幅标题
- 强聚焦产品图或卖点
- 干净黑底或深蓝底
- 大留白配合局部高亮

## Best Scenes

- 新品发布会
- 功能发布
- 产品亮点演讲
- 路线图宣讲

## Design Tokens

- Background: `#05070b`
- Surface: `#10131a`
- Primary: `#ffffff`
- Accent: `#2dd4bf`
- Text: `#f8fafc`

## Layout Notes

- `cover`, `section-break`, `closing` 是情绪锚点
- `insight` 用于卖点表达
- `timeline` 用于 roadmap

## Mobile Note

单页文案要更少，保证发布会大标题在小屏下仍有冲击力。
