# cyber-grid

## Positioning

面向 AI、云计算、科技峰会、行业论坛的旗舰科技风主题。

## Visual DNA

- 深色底
- 冷蓝青光
- 网格、描边、发光线条
- 标题强、数字亮、信息板块清晰

## Best Scenes

- AI 产品发布
- 技术品牌介绍
- 云平台方案宣讲
- 行业趋势 keynote

## Design Tokens

- Background: `#07111f`
- Surface: `#0d1b2f`
- Primary: `#59e1ff`
- Accent: `#7af7c4`
- Text: `#eaf4ff`

## Layout Notes

- `cover` 强背景氛围，可用网格与径向光
- `metrics` 适合做 HUD 仪表盘感
- `chart-focus` 适合荧光描边图表

## Mobile Note

减少装饰线数量，优先保证标题和数字的对比度。
