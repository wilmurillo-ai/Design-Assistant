# executive-dark

## Positioning

用于董事会、高管汇报、战略复盘的高压场正式主题。

## Visual DNA

- 深墨色背景
- 低饱和中性色
- 强对比标题
- 卡片克制、边框精细

## Best Scenes

- 管理层汇报
- 战略复盘
- 年度经营总结
- 高级客户会议

## Design Tokens

- Background: `#111315`
- Surface: `#1b1f24`
- Primary: `#f3f4f6`
- Accent: `#c7a86a`
- Text: `#f8fafc`

## Layout Notes

- `insight` 与 `comparison` 是主力
- `closing` 要稳，不做夸张动效
- `metrics` 用少量金色强调关键数字

## Mobile Note

保持大标题与留白，避免深色底下文字行距过密。
