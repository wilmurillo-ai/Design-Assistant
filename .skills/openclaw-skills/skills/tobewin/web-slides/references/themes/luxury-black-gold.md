# luxury-black-gold

## Positioning

面向高端品牌、重要发布、仪式感场景的奢雅主题。

## Visual DNA

- 黑金配色
- 极低噪音背景
- 高级衬线或精致无衬线标题
- 节奏慢、气场强

## Best Scenes

- 高端品牌发布
- 重要管理层展示
- 高净值客户提案
- 高端活动开场

## Design Tokens

- Background: `#090909`
- Surface: `#141414`
- Primary: `#e8d3a1`
- Accent: `#f6e7c9`
- Text: `#f8f3ea`

## Layout Notes

- `cover` 必须有仪式感
- `quote` 页面非常适合
- `metrics` 只展示少量关键数字

## Mobile Note

控制金色使用面积，避免手机端发灰或显脏。
