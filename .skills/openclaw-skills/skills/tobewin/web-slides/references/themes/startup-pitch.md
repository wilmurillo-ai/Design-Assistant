# startup-pitch

## Positioning

面向融资路演、创业 BP、增长故事的进攻型主题。

## Visual DNA

- 高对比深浅搭配
- 明亮品牌色强调
- 数字和标题有冲劲
- 页面节奏快

## Best Scenes

- Seed / A 轮路演
- 增长故事
- 商业模式说明
- 招商演示

## Design Tokens

- Background: `#0f172a`
- Surface: `#ffffff`
- Primary: `#7c3aed`
- Accent: `#22c55e`
- Text: `#e2e8f0`

## Layout Notes

- `cover` 要强势
- `metrics` 和 `timeline` 很关键
- `comparison` 可用于竞品或方案优势

## Mobile Note

保证关键数字和融资亮点在首屏内即可被看到。
