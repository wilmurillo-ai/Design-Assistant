# dev-summit

## Positioning

面向架构分享、开发者大会、工程团队技术演示的工程型主题。

## Visual DNA

- 工程感强
- 字体清晰
- 代码与图示兼容
- 科技但不过度炫

## Best Scenes

- 架构演讲
- 技术方案分享
- 开发者大会
- 开源项目介绍

## Design Tokens

- Background: `#0b1020`
- Surface: `#121933`
- Primary: `#7dd3fc`
- Accent: `#a3e635`
- Text: `#dbeafe`

## Layout Notes

- `two-column` 和 `insight` 最常用
- `timeline` 适合讲技术演进
- `chart-focus` 适合性能数据展示

## Mobile Note

代码截图不应成为主角，优先提炼为结构图和结论卡片。
