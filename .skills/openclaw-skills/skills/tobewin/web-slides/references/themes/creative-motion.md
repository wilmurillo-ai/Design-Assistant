# creative-motion

## Positioning

面向创意提案、营销 campaign、品牌活动的高张力主题。

## Visual DNA

- 鲜明色块
- 有节奏的几何形状
- 动态构图
- 动效比其他主题更明显

## Best Scenes

- 活动方案
- 创意 campaign
- 品牌年轻化提案
- 宣传展示

## Design Tokens

- Background: `linear-gradient(135deg, #0f172a, #1d4ed8)`
- Surface: `rgba(255,255,255,0.08)`
- Primary: `#f97316`
- Accent: `#22d3ee`
- Text: `#f8fafc`

## Layout Notes

- `cover`、`section-break`、`quote` 更有表现力
- `timeline` 可以更动感
- 内容型页面仍要克制装饰

## Mobile Note

动画幅度要缩小，保证移动端不会显得凌乱。
