# neo-minimal

## Positioning

面向极简设计提案、作品陈述、品牌升级说明的高级极简主题。

## Visual DNA

- 大留白
- 少量颜色
- 版式精确
- 质感来自比例而非装饰

## Best Scenes

- 设计提案
- 品牌升级
- 极简商务汇报
- 作品集陈述

## Design Tokens

- Background: `#f8f8f6`
- Surface: `#ffffff`
- Primary: `#111111`
- Accent: `#b6b09f`
- Text: `#18181b`

## Layout Notes

- `cover`、`insight`、`closing` 很强
- `comparison` 适合做克制对照
- 不要堆过多图形元素

## Mobile Note

这是移动端表现最稳的主题之一，但要避免大段正文。
