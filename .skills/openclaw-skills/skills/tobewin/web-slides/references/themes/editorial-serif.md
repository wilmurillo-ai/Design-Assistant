# editorial-serif

## Positioning

面向白皮书、研究报告、教育内容、深度分析的编辑型主题。

## Visual DNA

- 杂志感排版
- 衬线标题
- 温和底色
- 知识型、耐读型

## Best Scenes

- 白皮书
- 研究报告
- 教学课程
- 学术与人文内容

## Design Tokens

- Background: `#f2efe8`
- Surface: `#fbfaf7`
- Primary: `#2f3a4d`
- Accent: `#8b5e3c`
- Text: `#1f2937`

## Layout Notes

- `quote`, `two-column`, `insight` 表现稳定
- `chart-focus` 应使用温和配色
- `closing` 适合知识型总结

## Mobile Note

正文行长要严格控制，避免衬线字体在手机上显得拥挤。
