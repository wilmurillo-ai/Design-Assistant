# executive-light

## Positioning

用于咨询方案、客户提案、项目汇报的高级明亮商务主题。

## Visual DNA

- 大量留白
- 浅灰与白色表面
- 深色标题
- 局部品牌蓝或青色点缀

## Best Scenes

- 项目提案
- 咨询报告
- 客户汇报
- 周报月报

## Design Tokens

- Background: `#f5f7fb`
- Surface: `#ffffff`
- Primary: `#1f3c88`
- Accent: `#22c1c3`
- Text: `#101828`

## Layout Notes

- `agenda`, `insight`, `two-column` 最稳
- `chart-focus` 要走清爽数据面板路线
- `comparison` 适合做方案优劣矩阵

## Mobile Note

这是最适合移动端阅读的主题之一，优先保留卡片和标题层次。
