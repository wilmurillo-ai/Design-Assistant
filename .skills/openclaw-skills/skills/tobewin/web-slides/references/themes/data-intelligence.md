# data-intelligence

## Positioning

面向经营分析、BI 报告、增长复盘的数据型主题。

## Visual DNA

- 冷静理性
- 规则网格
- 数据卡片清晰
- 图表优先服务结论

## Best Scenes

- 月度经营分析
- 年度复盘
- 用户增长汇报
- 业务看板演示

## Design Tokens

- Background: `#0c1220`
- Surface: `#121a2b`
- Primary: `#6aa9ff`
- Accent: `#4be3c2`
- Text: `#e5edf9`

## Layout Notes

- `metrics` 和 `chart-focus` 是绝对主力
- `timeline` 可用于季度规划
- `insight` 页面要结论先行

## Mobile Note

优先将图表简化为关键指标卡，不在手机端展示复杂细图。
