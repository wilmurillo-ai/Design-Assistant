# glass-future

## Positioning

用于创新、AIGC、概念展示的轻未来玻璃感主题。

## Visual DNA

- 透明玻璃卡片
- 柔和渐变背景
- 霓虹弱化，不刺眼
- 漂浮感与轻模糊

## Best Scenes

- AIGC 方案
- 创新实验室
- 未来产品概念
- 科技品牌介绍

## Design Tokens

- Background: `linear-gradient(135deg, #08131f, #1f2f56)`
- Surface: `rgba(255,255,255,0.10)`
- Primary: `#8bdcff`
- Accent: `#b69cff`
- Text: `#f3f8ff`

## Layout Notes

- `cover` 与 `quote` 最出彩
- `two-column` 用半透明玻璃卡片承载信息
- `closing` 可以做未来感收束

## Mobile Note

玻璃层数不要过多，避免小屏阅读发灰。
