# Runtime

## Positioning

本 skill 使用自定义单文件 HTML 演示运行时。

它不是某个模型专属方案，也不依赖特定推理厂商。
视觉效果来自：

- 预制主题系统
- 固定布局原型
- 结构化内容输入
- 本地脚本生成流程

## Core Principles

- 与模型无关
- 与 API Key 无关
- 与外部设计服务无关
- 优先保证效果和稳定性

## Runtime Behavior

当前输出运行时特征：

- 单文件 HTML
- CSS 内嵌
- 轻量 JS 翻页
- 键盘翻页
- 触摸滑动
- 移动端自动降级为纵向连续浏览

可选的本地视觉 QA：

- 可用 headless Chrome 导出 PNG 预览
- 可批量生成桌面与移动端预览图
- 这部分属于本地辅助验收能力，不是核心生成链的强依赖

## Why This Matters

这样设计的好处是：

- skill 可以被不同模型稳定调用
- 输出质量更多由模板系统保证，而不是靠提示词碰运气
- token 消耗更可控
- 用户不需要关心底层模型差异
