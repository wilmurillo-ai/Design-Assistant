# 小红书话题标签库

## 使用规则

```
组合公式：核心话题(1-2个) + 垂类话题(2-3个) + 流量话题(1-2个) = 5-8个
核心话题：与内容强相关，精准引流
垂类话题：该领域活跃标签，圈层内传播
流量话题：千万级大标签，扩大曝光但不能喧宾夺主
```

## 标签禁用规则

```
❌ 避免使用已被过度滥用的标签（权重极低）：
   #干货  #必看  #建议收藏  #学习打卡  #日常
   （单独使用这些标签流量近乎为零）

❌ 不要一次性堆8个以上，会触发平台降权

✅ 最佳实践：5-6个精准标签 > 8个泛标签
```
