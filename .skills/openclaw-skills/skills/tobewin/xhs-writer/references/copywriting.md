# 小红书爆款写作公式库

## [TECH] 科技/AI/工具类

### 标题公式

```
反差冲击：
  "花了XX小时，才发现AI其实不会做这件事🤯"
  "用了[工具名]一个月，我把[耗时任务]砍到了X分钟"
  "所有人都在夸[工具名]，但没人说它的这个缺点"

身份共鸣：
  "程序员才懂的[工具名]隐藏用法🔧"
  "用AI的人和不用AI的人，差距已经出来了"
  "每天用Claude的人，都在偷偷做这件事"

悬念缺口：
  "为什么[工具名]用了一周就弃坑？我找到原因了"
  "[工具名]到底好不好用？用了X个项目后说说真实感受"

具体数字：
  "3个提示词技巧，让Claude的输出质量提升80%"
  "我用了X款AI工具，只有这X个值得长期留着"

结果前置：
  "靠这个工作流，我一个人顶了半个团队📈"
  "学会这X个AI技巧，效率直接翻倍不夸张"
```

### 正文开场白公式

```
场景痛点型：
  "上周帮朋友[场景]，折腾了[时间]没搞定，后来发现..."
  "以前[耗时任务]要[时间]，现在[X分钟]就搞完..."

反常识型：
  "[普遍做法]其实不对，大多数人都忽略了一个细节..."
  "我见过太多人[错误做法]，这其实是在[负面结果]..."

身份代入型：
  "如果你也在用[工具/方法]，一定遇到过这个问题..."
  "做[职业/事情]的朋友看过来，这个对你绝对有用..."
```

## [KNOWLEDGE] 知识干货/方法论类

### 标题公式

```
反差冲击：
  "读了X本书没解决的问题，被这X句话说清楚了"
  "学了X年[领域]，才明白[反常识结论]"
  "大家都在学[流行方法]，但真正有效的其实是..."

身份共鸣：
  "想提升[能力]的人，先搞清楚这件事"
  "在[职业/阶段]卡了很久？可能是这X个原因"

悬念缺口：
  "[看似简单的问题]——90%的人答不上来"
  "为什么有人[努力但没结果]？根源在这里"

具体数字：
  "X个思维模型，覆盖了我90%的决策场景"
  "坚持做这X件事X个月，我的[能力/状态]变了"

结果前置：
  "用这个方法，我把[复杂任务]的时间缩短了X倍"
  "[具体成果]，只用了一个很简单的习惯"
```

### 正文开场白公式

```
共鸣型：
  "有没有一种感觉：[描述常见困境]，但不知道问题出在哪..."
  "很多人[付出努力]，结果却[不如预期]，其实是因为..."

故事型：
  "三年前我[描述过去状态]，现在[现在的变化]。中间发生了一件事..."
  "[具体场景]里，我第一次意识到[核心洞察]..."

对比型：
  "同样[努力程度]，有的人[好结果]，有的人[平庸结果]，差别只在..."
```

## [LIFESTYLE] 生活/穿搭/美食类

### 标题公式

```
反差冲击：
  "XX元的[物品]，比XX元的好用太多了😭"
  "买了X年[品类]，终于找到最适合[特点]的款"
  "之前踩的坑，帮你们避开了"

身份共鸣：
  "[身材特点/肤质/需求]姐妹看这里！"
  "住[地方]的朋友，这家不去后悔"

具体数字：
  "X块钱的[物品]，我用了X年还没坏"
  "试了X款[品类]，这X个值得回购"

结果前置：
  "找到了！[期待效果]的[品类]真的存在"
  "终于显瘦了，就靠这X件单品"
```

### 正文开场白公式

```
问题共鸣型：
  "一直在找[理想特点]的[品类]，试了很多要么[缺点A]要么[缺点B]..."
  "[场景]总是遇到[痛点]，一直没找到满意的解决方案，直到..."

打卡记录型：
  "在[地方]待了X天，把觉得值得去的地方整理了一下..."
  "上周路过[地方]随手进了一家店，没想到..."
```

## [TRAVEL] 旅行/探店/打卡类

### 标题公式

```
反差冲击：
  "避开了所有景点，[城市]这X个地方才是真正的宝藏"
  "人均XX的[餐厅类型]，好吃程度完全不输网红店"

悬念缺口：
  "[地方]最值得去的打卡点，本地人才知道"
  "为什么大家去[热门景点]都说失望？我找到原因了"

具体数字：
  "[城市] X天X夜，人均XX，完整攻略在这里"
  "踩点了X家[类型]，只推荐这X家"

结果前置：
  "找到了！[城市]最适合[需求]的住宿"
  "这顿饭值了，XX的[菜名]绝了"
```

### 正文开场白公式

```
反预期型：
  "本来没抱太大期望，没想到..."
  "被朋友强推了很久，终于去了，结果..."

氛围描写型：
  "[时间+地点]，[一句氛围描写]。这就是[地方]给我的第一印象..."
  "走进[地方]的瞬间，突然明白为什么大家都说[形容词]了..."
```

## [GENERAL] 通用公式

内容方向不明确时，优先使用以下万能结构：

### 万能标题结构

```
[emoji] + [具体场景/数字] + [意外结果/反转] + [emoji]

例：
  "📱 用了 3 款笔记软件，最后靠这个习惯解决了所有问题"
  "🌿 一个月不买新衣服，反而穿出了最满意的状态"
```

### 万能正文结构

```
第1段：场景共鸣（让读者说"这说的是我"）
第2-4段：核心内容（每段一个观点，emoji开头）
第5段：总结+互动引导
```
