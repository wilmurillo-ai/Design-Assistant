# 动态样式系统

## 设计理念

不使用固定主题，而是用「基础风格 × 动态变量」的组合，
让每次生成的图片都有细微差异，永远不会看腻。

## 动态变量系统

每次生成时，在选定基础风格后，随机调整以下变量，
让同一风格每次都有细微差异：

### 变量1：主色偏移

```
从该风格的色盘中随机选取一个颜色作为本次主色
记录：主色 #XXXXXX
```

### 变量2：字号比例

```
从以下3个比例中随机选一个：
  紧凑型：标题36px / 正文18px / 副文字14px
  标准型：标题42px / 正文20px / 副文字15px
  宽松型：标题48px / 正文22px / 副文字16px
记录：字号方案 [紧凑/标准/宽松]
```

### 变量3：内边距

```
从以下3个档位随机选一个：
  小边距：24px
  中边距：32px（默认）
  大边距：40px
记录：内边距 Xpx
```

### 变量4：卡片圆角

```
从以下4个档位随机选一个：
  直角：0px
  小圆角：8px
  中圆角：16px
  大圆角：24px
记录：圆角 Xpx
```

### 变量5：分割线样式

```
从以下4种中随机选一个：
  实线 1px
  虚线 2px dashed
  点线 dotted
  无分割线（依赖间距分隔）
记录：分割线 [样式]
```

## 风格轮换建议

连续发布时，建议按以下规则避免视觉重复：
```
同一主题方向：相邻两篇不用同一基础风格
同一基础风格：主色和字号方案至少换一个
深色风格（tech-minimal/terminal/dark-editorial）：不连续使用超过2篇
```
