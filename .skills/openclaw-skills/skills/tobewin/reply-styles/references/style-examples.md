# Style Examples

## Goal

让风格差异不是抽象标签，而是能感知到的说话方式。

## Example 1: 委婉拒绝

原意：

`最近真的没办法再继续免费帮你做这件事了。`

### `firm-boundary`

`这件事我这边这次就先不继续接了，也想和你说清楚，后面类似的内容我可能没办法再无偿帮忙处理。还请你理解。`

### `high-eq-soft`

`我其实也考虑了一下，这件事如果后面还继续按现在的方式帮下去，我这边会有点吃力，所以想提前和你说一声，之后我可能没办法再像之前那样无偿支持了。`

## Example 2: 投诉安抚

原意：

`这个问题我们会处理。`

### `de-escalation`

`理解你现在会着急，这次确实给你带来困扰了。我这边已经先接手跟进这个问题，会尽快帮你核实处理进度，稍后第一时间同步你。`

### `service-recovery`

`这次的体验确实受影响了，这边先和你说声抱歉。我会直接跟进这次问题的处理，今天内先把当前进度和补救方案同步给你。`

## Example 3: 老板问进度

原意：

`还在做。`

### `executive-crisp`

`还在推进中。目前已完成主体部分，剩余收尾和联调，今天下班前给你完整版本。`

### `founder-direct`

`在做，主体已经过半。今天把核心版本先交出来，细节项我再补。`

## Example 4: 高端客户确认

原意：

`好的，我们帮您安排。`

### `luxury-concierge`

`好的，这边已经为您优先安排。相关细节我会继续替您确认，确定后第一时间与您同步，您放心。`

### `warm-professional`

`好的，这边已经开始安排了。确认完具体细节后，我会第一时间同步给您。`

## Example 5: 销售推进

原意：

`如果您有兴趣可以约时间聊。`

### `premium-sales`

`如果你方便，我们可以约一个 20 分钟的时间，把最关心的几个问题和实际落地方式快速对齐一下。这样你会更容易判断这件事是不是适合现在推进。`

### `consultant-polished`

`如果你愿意，我们可以找个短时间快速对齐一下当前最关键的判断点，我也可以基于你的实际情况给出更清晰的建议路径。`

## Example 6: 合作方分歧

原意：

`这个方案我们现在接受不了。`

### `partner-diplomatic`

`这个方向我们认真看过了，现阶段和我们这边的节奏还不完全匹配。为了不影响双方后续合作，我建议我们先把几个关键前提再对齐一下，再判断什么方案会更顺。`

### `legal-clean`

`基于目前的合作边界和现有安排，这个方案我们这边暂时无法直接接受。后续如有正式调整，以双方确认的信息为准。`

## Example 7: 品牌公开回应

原意：

`我们知道这个事情了，会处理。`

### `pr-safe`

`感谢大家的反馈，我们已经关注到这次问题，并在同步核实相关情况。后续有明确进展后，会第一时间统一说明。`

### `brand-host`

`谢谢你把这个问题直接说出来，我们也很重视这次体验上的落差。相关情况已经在跟进确认，后续我们会把处理结果认真和大家说明。`

## Example 8: 短而高级

原意：

`行，我来处理。`

### `cool-minimal`

`可以，这件事我来处理。`

### `executive-crisp`

`收到，这件事我今天内处理完。`

## Example 9: 售后补救

原意：

`不好意思，给您添麻烦了。`

### `service-recovery`

`这次确实影响到你了，这边先和你说声抱歉。我会直接跟进这次问题的处理，今天内先把进度和补救方案同步给你。`

### `customer-success`

`这次确实影响了你的使用体验，先和你说声抱歉。我这边会把这次问题跟进到底，也会一起把后续更顺畅的处理方式梳理出来，尽量不再让你重复受影响。`

## Example 10: 知性温柔

原意：

`你不用太担心。`

### `bookish-gentle`

`你先别太着急，这件事未必有现在感受到的那么糟。我们可以慢一点，把关键信息一项项看清楚。`

### `supportive-coach`

`你先别急，这件事还有空间。我们先把眼前最关键的一步拆出来，你会更容易重新找回节奏。`
