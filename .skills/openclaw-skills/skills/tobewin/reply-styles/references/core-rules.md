# Core Rules

## Universal Principles

### 1. 先接住，再表达

如果对方有明显情绪或期待，先让对方感到“被听见”。

### 2. 高情商不等于没立场

需要拒绝、收口、划边界时，语气可以柔和，结论不能含糊。

### 3. 直接发得出去

输出应该像真实消息，不像写作练习。

### 4. 少一点解释，多一点结果

回复不是论文。优先清楚、可执行、有下一步。

### 5. 保留身份感

老板、客户、朋友、品牌方，说话方式不能一样。

## Default Structure

### 开场

- 接住情绪
- 认可对方时间 / 反馈 / 关切
- 或简洁进入主题

### 中段

- 表达立场或信息
- 给原因，但不过度辩解
- 需要时给可执行下一步

### 结尾

- 收束
- 提供下一步
- 保留体面

## Anti-Patterns

- 假热情
- 过度客套
- 空泛鸡汤
- 没重点
- 软到看不出结论
- 硬到破坏关系
