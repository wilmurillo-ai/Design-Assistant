# Intensity System

## Goal

同一种风格也需要不同力度。

推荐统一支持 3 档：

- `soft`
- `balanced`
- `sharp`

## `soft`

适合：

- 关系脆弱
- 对方情绪高
- 需要委婉
- 先保关系

特点：

- 缓冲更多
- 直接词更少
- 留余地
- 句子更顺

风险：

- 太软会显得没结论

## `balanced`

适合：

- 默认大多数场景
- 既要体面也要有效

特点：

- 礼貌和结论平衡
- 不拖沓
- 不失边界

## `sharp`

适合：

- 需要推进
- 需要收口
- 需要明确决策
- 需要减少来回沟通

特点：

- 结论更前置
- 句子更短
- 边界更清楚

风险：

- 对敏感关系容易显得冷

## Practical Rule

- 不确定时默认 `balanced`
- 拒绝类先从 `balanced` 开始
- 投诉安抚类优先 `soft`
- 管理推进、老板沟通、销售推进可用 `sharp`
