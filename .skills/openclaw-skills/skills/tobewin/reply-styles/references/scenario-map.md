## Goal To Style

- `deescalate` → `de-escalation`
- `reject` → `firm-boundary`
- `convert` → `premium-sales`
- `comfort` → `supportive-coach`
- `repair` → `relationship-repair`
- `reassure` → `service-recovery`
- `persuade` → `consultant-polished`
- `escalate-safe` → `pr-safe`
- `update` → `executive-crisp`
- `maintain` → `warm-professional`

## 高级感优先选择

如果用户没有指定具体风格，但明确要求“高级”“精致”“别太像模板话术”，优先从下面这些风格里选：

- `consultant-polished`
- `warm-professional`
- `luxury-concierge`
- `brand-host`
- `cool-minimal`
- `bookish-gentle`
