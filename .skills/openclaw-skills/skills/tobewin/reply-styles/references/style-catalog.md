# Style Catalog

## Usage

先根据场景和关系选风格，再写回复。

优先选择一个主风格，不要混太多。

## Quick Guidance

- 不确定时默认：`warm-professional`
- 对方情绪高时优先：`de-escalation`
- 需要拒绝但不撕破脸：`firm-boundary`
- 想显得更高级更贵：`luxury-concierge`
- 想显得更利落：`executive-crisp`
- 想拉近关系：`high-eq-soft` 或 `playful-friendly`
- 想更像顾问：`consultant-polished`
- 想更像品牌 / 主理人：`brand-host`
- 想更稳妥低风险：`pr-safe` 或 `legal-clean`

## Recommended Intensities

- 更稳妥：`soft`
- 默认通用：`balanced`
- 更利落：`sharp`
