# Effect Rubric

高情商回复的目标不是“更礼貌”，而是“更像真人高手说的话”。

## 过稿标准

一条回复至少要同时满足下面 6 条里的 5 条：

1. 像人说话
   - 读起来像真实的人会发出去的话
   - 不是拼接式模板，不是客服腔，不是鸡汤腔

2. 风格稳定
   - 同一条回复只保留一个主风格
   - 不要一半像朋友，一半像客服，一半又像公文

3. 关系拿捏对
   - 对老板不能太碎
   - 对客户不能太冷
   - 对敏感关系不能过硬
   - 对公开场合不能失控

4. 有结论有动作
   - 不要只是“情绪价值”
   - 要么给结论，要么给下一步，要么给边界

5. 高级但不端
   - 可以精致，但不能悬浮
   - 可以温柔，但不能做作
   - 可以坚定，但不能冒犯

6. 短而有密度
   - 少废话
   - 句子不要拖泥带水
   - 每一句都应该有作用

## 常见翻车点

- 太像模板：全是套话，像复制粘贴
- 太软没骨头：看起来礼貌，其实没表达清楚立场
- 太硬没情商：只剩结论，没有缓冲
- 太长：用户不可能真的发出去
- 太装：为了显高级，写得不自然

## 最后一眼检查

交付前快速问自己：

- 这句话像不像用户自己会发出去？
- 这句话是不是既保住体面，又达成目的？
- 如果去掉风格名，只看内容，依然成立吗？
