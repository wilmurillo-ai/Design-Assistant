# Example Scenarios

- 帮我高情商拒绝一个总来借人情但我不想继续帮忙的朋友
- 用高端一点的口吻回复一个 VIP 客户的预约确认
- 帮我写一条给老板的简短进度更新，要干练一点
- 对方在群里有点情绪化，帮我用社群运营口吻安抚一下
- 帮我把这句话改得更像高级销售，而不是推销

## 风格 × 强度 × 渠道示例

- `firm-boundary + soft + wechat`
  场景：礼貌拒绝继续无偿帮忙

- `de-escalation + soft + support-ticket`
  场景：用户投诉产品出错，先安抚再给处理动作

- `executive-crisp + sharp + wechat`
  场景：给老板回项目进度，短而清楚

- `luxury-concierge + balanced + email`
  场景：高端客户预约确认与细节说明

- `community-manager + balanced + group`
  场景：群公告、规则提醒、活动通知

## 风格 × 人格 × 模式 示例

- `consultant-polished + trusted-advisor + clear-reject`
  场景：体面拒绝一个不合适的合作方案，但仍保留合作空间

- `brand-host + premium-host + public-safe`
  场景：公开回应用户对品牌体验的不满，既有温度又稳妥

- `de-escalation + calm-operator + soothe-first`
  场景：用户投诉升级，先降火再给明确处理动作

- `executive-crisp + sharp-founder + concise-update`
  场景：给老板回一句短进度，结论先行
