# 中国法律场景解析工具

分析具体法律场景，提供法律解读和建议。

## 功能特点

- ⚖️ **场景解析**: 分析具体法律场景
- 📋 **法律依据**: 引用相关法律条文
- 🎯 **风险评估**: 识别潜在法律风险
- 💡 **建议方案**: 提供解决方案建议
- 📊 **案例参考**: 引用类似案例
- 🇨🇳 **中国法律**: 专注中国法律体系

## ⚠️ 免责声明

> **本工具仅供参考，不构成法律建议。**
> 不同AI模型能力不同，解析结果可能有差异。
> 重要法律事务请咨询专业律师。
> 法律条文以最新官方发布为准。

## 支持的场景

| 场景 | 示例 |
|------|------|
| 合同纠纷 | 合同违约、合同解除 |
| 劳动争议 | 辞退、加班、工伤 |
| 财产纠纷 | 房产、继承、债务 |
| 婚姻家庭 | 离婚、抚养权、财产分割 |
| 知识产权 | 专利侵权、商标纠纷 |
| 消费者权益 | 退换货、假货赔偿 |

## 使用方式

```
User: "我被公司无故辞退了，该怎么办"
Agent: 分析劳动法相关条文，提供维权建议

User: "签了合同对方不履行，怎么处理"
Agent: 分析合同法相关条文，提供解决方案

User: "邻居噪音扰民，有什么法律依据"
Agent: 分析相邻权相关法律，提供维权途径
```

## 分析流程

```
用户描述场景
    ↓
1. 识别法律领域
    ↓
2. 搜索相关法律条文
    ↓
3. 分析法律关系
    ↓
4. 评估法律风险
    ↓
5. 提供解决方案
```
