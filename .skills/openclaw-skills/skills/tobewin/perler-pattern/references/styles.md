# 图纸样式参数说明

## 网格尺寸选择

```
29×29：标准拼豆板（1块板）
  优点：正好一块板，无需拼接
  缺点：细节少，适合简单图案
  适合：文字、简单动物、符号

40×40：小红书最常见尺寸（本 skill 默认）
  优点：细节适中，效果好看，符合小红书审美
  缺点：需要稍微裁剪或拼接板子
  适合：宠物头像、卡通人物

48×48：中等精细
  优点：细节丰富，效果精美
  缺点：需要多块板拼接（约3块板）
  适合：人像、复杂图案、宠物全身

64×64：大尺寸精品
  优点：细节极丰富
  缺点：用豆多、拼接复杂（约5块板）
  适合：精品展示、送礼

96×96：超大作品
  适合：宫格类拼比、团队合作大作品
```

## 颜色数量建议

```
8-12 种：极简风格，适合文字、简单图标
13-18 种：入门级，细节适中，购买方便
19-25 种：标准级，本 skill 默认20种
26-35 种：精细级，色彩丰富，效果接近照片
35+种：大师级，需要大量不同颜色的豆
```

## 图片预处理建议

```
效果最好的图片类型：
  ✅ 纯色背景的宠物/人像照片
  ✅ 卡通角色（颜色本来就少）
  ✅ 游戏截图/像素风图片
  ✅ Logo/图标

效果一般的图片类型：
  ⚠️ 风景照（颜色太多太复杂）
  ⚠️ 背景复杂的照片（建议先抠图）
  ⚠️ 分辨率很低的图片

优化技巧：
  1. 提前用手机/PS 去除背景
  2. 适当增加图片饱和度和对比度
  3. 将主体放在图片中央
```

## SVG 图纸打印建议

```
A4 纸打印：
  29×29（cell=10px）→ 290×290px，刚好A4
  48×48（cell=6px）→ 288×288px，刚好A4

大尺寸打印：
  使用 SVG 无损放大，建议打印到 A3
  或分页打印（多张A4拼接）

打印设置：
  - 选择"实际大小"不要缩放
  - 关闭页眉页脚
  - 建议打印到厚纸（方便垫在豆板下参考）
```
