# 色板说明

## 已内置色板（SKILL.md 中完整内嵌）

| 品牌 | 参数值 | 颜色数 | 适用场景 |
|---|---|---|---|
| 哈马 Hama | hama | 35色 | 最流行，中国用户首选 |
| 欧小 Artkal | artkal | 30色 | 国产品牌，性价比高 |
| 拼豆 Perler | perler | 24色 | 北美品牌 |
| 通用 Universal | universal | 40色 | 不限品牌，颜色最全 |

## 品牌选择建议

```
用户说"哈马/Hama"         → hama
用户说"欧小/Artkal"       → artkal
用户说"拼豆/Perler"       → perler
用户说"通用/不限/我有的"  → universal
用户未指定                 → hama（默认，最流行）
用户说"颜色种类多一点"    → universal（40色最全）
```

## 特殊颜色说明

```
透明色（Perler P24）：
  用于需要镂空效果的设计
  提示用户透明豆作为占位使用

荧光色（Hama 32/33/34, Universal 38/39/40）：
  在黑光灯（UV灯）下会发光
  适合特殊场景装饰

金色/银色：
  金属质感，常用于装饰细节
  注意：有些品牌金属色需单独购买
```
