# 分镜设计指南

## 基本原则

```
1. 每个分镜只传递一个核心信息
2. 画面与解说词相辅相成，不要重复
3. 前3秒必须抓住眼球
4. 结尾必须有行动引导（CTA）
5. 图片 prompt 用英文，中文描述用于 TTS
```

## Prompt 设计技巧

### 画面 Prompt 结构
```
[主体描述] + [场景] + [风格] + [质量参数]

例：
"A premium skincare product bottle on marble surface,
 soft studio lighting, luxury brand photography style,
 high detail, 8k resolution, commercial photography"
```

### 常用风格关键词
```
商务/产品：commercial photography, studio lighting, white background, product shot
科技/AI：  futuristic, holographic, digital, neon lights, dark background
温暖/生活：lifestyle photography, natural light, warm tones, candid
科普/教育：infographic style, clean design, illustration, flat design
奢华/高端：luxury, elegant, gold accents, dark moody lighting
活泼/年轻：vibrant colors, dynamic, energetic, social media style
```

## 各类视频分镜模板

### 产品宣传（30秒，6个分镜）
```
分镜1（3s）开场震撼：产品大特写，神秘感揭示
分镜2（5s）痛点共鸣：目标用户的困境场景
分镜3（5s）产品亮相：产品正面展示，核心卖点
分镜4（5s）功能展示：1-2个核心功能的使用场景
分镜5（5s）用户反馈：真实使用场景，情感共鸣
分镜6（4s）行动引导：购买按钮/二维码/品牌logo
```

### 知识科普（60秒，12个分镜）
```
分镜1（3s）引子：抛出一个有趣问题
分镜2-4（各5s）：分三个角度解释核心概念
分镜5-7（各5s）：举具体例子/数据
分镜8-10（各5s）：深入延伸
分镜11（5s）总结：核心结论一句话
分镜12（3s）引导：关注/互动/更多内容预告
```

### 品牌故事（45秒，8个分镜）
```
分镜1（3s）情感开场：触动人心的场景
分镜2-3（各5s）：品牌起源/创始人故事
分镜4-5（各5s）：品牌发展历程
分镜6（5s）：产品/服务展示
分镜7（5s）：用户/社区
分镜8（3s）：品牌理念/口号
```

### 小红书种草（20秒，5个分镜）
```
分镜1（3s）：产品美图，高颜值展示
分镜2（4s）：使用前/痛点
分镜3（5s）：使用中/过程
分镜4（4s）：使用后/效果对比
分镜5（4s）：氛围感收尾，引导点赞收藏
```

## 时长与字数对照

```
TTS 朗读速度：约 4字/秒（中文，正常语速）
快语速（speed=1.2）：约 5字/秒
慢语速（speed=0.8）：约 3字/秒

分镜时长 → 解说词字数：
  3秒  → 约 12字
  4秒  → 约 16字
  5秒  → 约 20字
  8秒  → 约 32字
  10秒 → 约 40字
```

## 常见错误

```
❌ 解说词太长：画面5秒，解说词写了40字，TTS来不及读完
❌ 画面与音频不匹配：说"功能强大"但画面是风景
❌ 分镜太碎：每张图1-2秒，视觉太跳跃
❌ 分镜太少：30秒只有2张图，单调无聊
❌ 开场太平淡：前3秒没有吸引力，用户划走
✅ 解决方案：写完解说词先数字数，再决定分镜时长
```
