# Code Avatar & Icon Generator

用代码生成高质量头像、App 图标与基础品牌图形。

## What it does

This skill helps users:
- Generate avatars with code
- Generate app icons with code
- Create favicon-ready graphics
- Build editable SVG-based visual assets
- Export designs that stay clear at small sizes

## Why it is useful

- More controllable than random image generation
- Easier to iterate
- Easier to export in multiple sizes
- Better for minimal, geometric, and product-ready icon work

## Best for

- Personal avatars
- Product avatars
- App icons
- Favicon graphics
- Minimal logo marks
