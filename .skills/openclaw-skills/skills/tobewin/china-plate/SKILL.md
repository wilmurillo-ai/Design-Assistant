# 中国车牌解析 China License Plate Parser

解析车牌号的省份、城市、车牌类型，验证格式合法性。完全离线，零依赖。
地市代码完整对照表见 `references/city-codes.md`，需要查询具体城市时读取该文件。

## 触发时机

- 用户提供车牌号，询问归属省市
- "粤B是哪里的车" / "这个车牌是新能源吗"
- 验证车牌号格式是否合法
- 识别军牌、警牌、武警牌等特殊车牌

---

## 车牌类型识别（第一步）

拿到车牌后，先判断类型：

```
以汉字开头（京/津/沪/粤等）→ 民用车牌（最常见）
以 WJ 开头              → 武警车牌
以两位大写字母开头        → 解放军车牌（如 BA12345）
纯数字或其他特殊格式      → 临时/试验车牌
```

## 民用车牌格式规则（GA36-2018标准）

### 普通民用车牌（7位）
```
格式：[省份简称][地区代码][5位序号]
示例：粤B12345、京A88888

序号规则：
- 字母和数字混合，但不使用 I 和 O（防止与1、0混淆）
- 地区代码字母同样不使用 I 和 O
- 可使用字母结尾特殊车牌（挂/学/警/港/澳）
```

### 新能源车牌（8位）
```
格式：[省份简称][地区代码][6位序号]
颜色：小型车绿底黑字，大型车黄绿双拼

小型新能源（第3位固定为D或F）：
  D = 纯电动
  F = 非纯电动（插混/增程）
  示例：粤BD12345、沪AD00001

大型新能源（第8位固定为D或F）：
  示例：粤B12345D（公交车等大型车）
  前5位为数字序号，最后1位为D或F
```

### 特殊民用车牌
```
末尾带汉字类型：
  挂  → 挂车（如 粤B1234挂）
  学  → 教练车（如 粤B2345学）
  警  → 警察车辆（如 川A0001警，第17位为9=交警）
  港  → 香港入出境内地车辆（如 粤Z1234港）
  澳  → 澳门入出境内地车辆（如 粤Z1234澳）
```

## 地区代码（第2位字母）

第2位字母代表地级市/地区，各省规则详见 `references/city-codes.md`。

常见规律：
- 省会城市通常使用 A（如 粤A=广州，苏A=南京，川A=成都）
- 计划单列市/副省级城市通常使用 B（如 粤B=深圳）
- 直辖市内部不分城市，字母代表区县管理机构
- 所有省份均跳过字母 I 和 O

## 特殊车牌格式

### 武警车牌
```
格式：WJ + [省份拼音缩写] + 5位数字
示例：WJ京12345、WJ粤67890
2019年式：采用与解放军一致的外观
```

### 解放军车牌（2004式，2005年1月起用）
```
格式：[战区/军种字母][部队代号字母][5位数字]
示例：BA12345
前两位为大写字母，后五位为纯数字
```

### 应急救援车牌
```
格式类似普通民用车牌，末尾无特殊汉字
白底黑字黑框线
```

## 格式验证规则

```
合法普通车牌（7位）：
1. 第1位：有效省份简称汉字
2. 第2位：大写字母，不含I、O
3. 第3-7位：数字和大写字母组合，不含I、O
   或第7位为特殊汉字（挂/学/警/港/澳）

合法新能源车牌（8位）：
1. 第1位：有效省份简称汉字
2. 第2位：大写字母，不含I、O
3. 第3位：D或F（小型车）
4. 第4位：数字或字母（不含I、O）
5. 第5-8位：纯数字
   或：第3-7位纯数字，第8位为D或F（大型车）

不合法情况：
- 省份简称不在有效列表中
- 地区代码使用了I或O
- 序号中含I或O
- 长度不符合7位（普通）或8位（新能源）
```

## 注意事项

- 解析结果为登记归属地，不代表车辆或车主当前所在地
- 部分城市因号段用尽启用了新字母（如苏U=苏州增补段，2019年启用）
- 广东粤O号段按警车号牌管理（非普通民用）
- 港澳车辆入出境使用专属格式，与内地车牌规则不同
- 本 skill 依据 GA36-2018 标准，如遇新启用号段请以公安交管部门公告为准
