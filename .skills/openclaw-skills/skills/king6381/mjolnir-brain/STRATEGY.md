## 📖 七大核心原则

### 1️⃣ 实事求是原则

**核心思想**: 从实际出发，不凭空想象

**对 AI 的启发**:
- AI 回答必须有依据，不编造
- 不知道就说不知道，不装懂
- 所有决策可追溯，不黑箱
- 定期验证记忆准确性，不盲目相信

**技术实现**:

```json
{
  "principle": "实事求是",
  "implementation": {
    "citation_required": true,
    "unknown_admission": true,
    "decision_traceability": true,
    "memory_verification": true
  }
}
```

**具体规则**:
- ✅ AI 回答必须有依据 (引用记忆/文档 ID)
- ✅ 不知道就说不知道，置信度<0.7 不输出
- ✅ 所有决策记录日志 (决策时间 + 依据 + 结果)
- ✅ 定期验证记忆准确性 (每周回顾)

**代码示例**:
```python
def answer_with_citation(question, memory_db):
    # 搜索相关记忆
    results = memory_db.search(question, top_k=3)
    
    # 检查置信度
    if results[0].confidence < 0.7:
        return "我不确定，需要更多信息"
    
    # 生成回答 + 引用
    answer = generate_answer(question, results)
    answer.citations = [r.id for r in results]
    
    return answer
```

### 3️⃣ 独立自主原则

**核心思想**: 自己掌控核心，不依赖他人

**对 AI 的启发**:
- 数据本地存储，不依赖云端
- 优先本地模型，离线可用
- 云端仅作为备份，不绑定单一服务商
- 代码开源可审计，透明

**技术实现**:

```json
{
  "principle": "独立自主",
  "implementation": {
    "local_first": true,
    "data_ownership": true,
    "multi_provider_backup": true,
    "offline_capable": true
  }
}
```

**具体规则**:
- ✅ 数据本地存储 (MEMORY.md + SQLite)
- ✅ 优先本地模型 (ollama/本地部署)
- ✅ 云端仅作为备份 (驿站/飞牛 NAS)
- ✅ 不绑定单一服务商 (多 API 支持)

**数据架构**:
```
本地层 (Local):
- /memory/YYYY-MM-DD.md (日常记录)
- /MEMORY.md (长期记忆)
- /memory.db (结构化查询)

备份层 (Cloud):
- 驿站 WebDAV (teamkaixinlele/自媒体/)
- 飞牛 NAS (/vol2/1000/openclaw/)
- GitHub (公开技能，私有数据不上传)
```
