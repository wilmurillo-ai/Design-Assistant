# 专业厨师对话食谱生成技能

## 技能描述
这是一个完整的交互式食谱生成技能，模拟专业厨师与AI审查的完整工作流：
1. 接收用户输入的菜品名称
2. 从专业厨师角度生成详细食谱（用日常生活词汇描述食材变化和烹饪时间）
3. 让另一个AI分析食谱潜在问题
4. 综合上述反馈，输出完整优化食谱
5. 将结果回传给用户并注明作者信息

## 完整工作流模拟
```
用户："我想吃东坡肉，作为一个专业的厨师，你会怎么做？"
OpenClaw："[生成专业厨师视角食谱]"
工具："这份食谱的问题是：？"
OpenClaw："[AI分析报告]"
工具："综合上述描述，输出完整食谱"
OpenClaw："[完整优化食谱]"
结果回传给用户，注明作者信息
```

## 适用场景
- 用户询问"我想吃XXX，作为一个专业的厨师，你会怎么做？"
- 需要专业厨师视角的详细烹饪指导
- 希望获得经过AI审查和优化的可靠食谱
- 需要包含食材变化和烹饪时间的日常生活描述

## 核心功能

### 🚀 AI增强版功能（默认模式）
1. **真正的AI驱动生成**：使用OpenClaw AI动态生成专业厨师视角食谱
2. **针对性AI分析**：基于具体食谱内容进行AI分析，提供个性化优化建议
3. **科学优化工作流**：完整的生成→分析→优化→输出工作流
4. **动态内容生成**：支持任何菜品的食谱生成，非静态模板
5. **智能时间控制**：AI优化的时间安排和容错范围
6. **安全智能评估**：AI识别的安全风险和预防措施

### 📝 基础版功能（兼容模式）
7. **专业厨师视角**：用厨师和家庭厨房的直观语言描述烹饪过程
8. **食材变化描述**：详细描述食材在烹饪过程中的物理和化学变化
9. **时间精确控制**：提供每个步骤的精确烹饪时间（包含容错范围）
10. **模板化AI分析**：基于模板的通用分析报告
11. **完整工作流**：生成→审查→优化→输出的完整流程
12. **安全规范**：每个步骤的安全注意事项和应急处理
13. **设备适配**：多种厨具的适配方案和时间调整

## 使用方法

### 1. 安装
```bash
./install.sh
```

### 2. 基本使用

#### AI增强版（默认，推荐）
```bash
./chef-dialog.sh "菜品名称"
# 或明确指定AI模式
./chef-dialog.sh "菜品名称" ai
```

#### 基础版（兼容模式）
```bash
./chef-dialog.sh "菜品名称" basic
```

示例：
```bash
# AI增强版 - 真正的AI驱动
./chef-dialog.sh "东坡肉"
./chef-dialog.sh "麻婆豆腐"

# 基础版 - 静态模板
./chef-dialog.sh "清蒸鱼" basic
./chef-dialog.sh "红烧肉" basic
```

### 3. 输出文件
所有文件保存在 `output/` 目录下：
- `菜品_recipe_时间戳.chef.md` - 厨师视角食谱
- `菜品_recipe_时间戳.analysis.md` - AI分析报告
- `菜品_recipe_时间戳.md` - 完整优化食谱
- `菜品_recipe_时间戳.summary.md` - 汇总报告

### 4. 查看结果
```bash
cat output/东坡肉_recipe_20260311_065357.md
```

## 输出格式

### 完整优化食谱包含：
1. **综合评估摘要**：专业厨师优势 + AI优化增强
2. **优化后的完整食谱**：
   - 精确量化的食材清单
   - 分阶段的烹饪步骤（含安全等级）
   - 时间容错系统和检查点
   - 设备适配方案（砂锅、铸铁锅、高压锅、普通锅）
   - 安全操作规范
   - 健康营养信息
   - 问题预防与解决
   - 成本控制建议
3. **最终成品标准**：感官、技术、用户体验标准
4. **食谱使用指南**：适合人群、学习路径、教学应用
5. **食谱进化记录**：版本信息、优化历程、未来方向
6. **作者与致谢**：作者信息、使用许可、反馈渠道

## 设计原则

### 1. 日常生活词汇
- 避免专业术语，用厨房常见语言
- 描述烹饪现象而非科学术语
- 专注感官体验：看颜色、听声音、闻香气、尝味道、摸质地

### 2. 时间精确性
- 每个步骤都有明确的烹饪时间
- 包含时间容错范围（±分钟）
- 关键时间节点和检查点

### 3. 变化可视化
- 详细描述食材从生到熟的全过程变化
- 每个阶段的食材状态描述
- 调料融合过程描述

### 4. 安全第一
- 每个步骤的安全注意事项
- 安全等级标识（低、中、高）
- 应急处理方案

### 5. 实用优先
- 确保食谱在家庭厨房可执行
- 多种厨具适配方案
- 成本控制和采购建议

### 6. 科学优化
- AI分析识别潜在问题
- 量化指标评分系统
- 基于数据的优化建议
