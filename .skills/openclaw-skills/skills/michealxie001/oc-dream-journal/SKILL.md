# Dream Journal - 梦日记 🌙

## 核心特点

**动态学习，越用越聪明**

不像传统的静态词典，这个 skill 从一个极简的种子意象库开始，通过分析你的每一个梦境，自动学习新的意象词汇，持续丰富属于你自己的梦境语言体系。

## 功能

1. **自动识别梦境**：检测触发词，自动记录
2. **动态意象学习**：从梦境中提取新词汇，自动分类
3. **文学修饰**：为每个梦境生成诗意标题
4. **意象分析**：统计各类意象出现频率
5. **梦境搜索**：关键词、意象、日期搜索
6. **持续成长**：意象库随使用不断扩充

## 触发词

自动记录触发：
- "我做了个梦" / "昨晚梦见" / "梦里" / "我梦见" / "做梦梦到"

手动查询：
- "查看梦日记" / "分析梦境意象" / "搜索梦境 [关键词]"

## 8大意象类别

| 类别 | 说明 |
|------|------|
| 自然元素 | 水、火、风、山、天空等 |
| 动物 | 蛇、鸟、鱼、猫、狗等 |
| 建筑与空间 | 房子、门、路、桥、楼梯等 |
| 人物 | 母亲、父亲、陌生人等 |
| 身体与感知 | 手、眼睛、心、坠落、飞翔等 |
| 物品与象征 | 书、钥匙、镜子、车等 |
| 情绪与场景 | 逃跑、黑暗、光明等 |
| 抽象概念 | 其他难以归类的意象 |

## 动态学习机制

### 种子阶段
开始时只有约 30 个基础意象词汇作为"种子"。

### 学习阶段
每次记录梦境时：
1. 识别已知的意象
2. 从内容中提取新词汇
3. 通过智能规则自动分类
4. 保存到你的个人意象库

### 成长展示
```
自然元素: 7基础 + 23学习 = 30个
动物: 6基础 + 15学习 = 21个
...
当前意象库规模: 127个词汇
```

```
memory/

├── dream_imagery_db.json  # 意象数据库
├── user_imagery.json      # 用户学习到的意象
└── dream_stats.json       # 统计数据
```

## 使用示例

### 记录梦境
> 用户：昨晚我做了个梦，梦见自己在一个紫色的水晶洞穴里，周围漂浮着发光的蝴蝶...

系统识别后：
- 提取已知意象：光、蝴蝶
- 学习新意象：水晶、洞穴、紫色
- 生成标题：在水晶与蝴蝶之间
- 记录到 dreams.md

### 查看统计
> 用户：分析我的梦境意象

输出：
```
总共记录: 15 个梦境
意象库规模: 127 个词汇
常见意象:
  • 水: 8 次
  • 飞翔: 5 次
  • 紫色: 3 次
```

### 搜索梦境
> 用户：搜索梦境 飞翔

返回所有包含"飞翔"意象的梦境记录。
