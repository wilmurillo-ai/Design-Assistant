# AI签证证件助手 (ai-visa-documents-skill)

## 技能描述

AI签证证件助手提供全球各国签证政策查询、材料清单自动生成、办理流程详细指导、办理时间预估、免签/落地签信息查询、护照有效期提醒等一站式签证服务。帮助用户轻松应对签证办理的各种问题。

---

⚠️ 【重要约束】
- 签证政策信息需定期更新
- 必须标注政策更新日期
- 材料清单需根据最新要求
- 办理时间仅供参考，以使馆实际为准
- 提醒用户以官方信息为准

---

## 核心功能

### 1. 签证政策查询
- 免签国家查询
- 落地签国家查询
- 电子签国家查询
- 需提前办理签证国家查询
- 过境签政策查询

### 2. 材料清单生成
- 根据目的地自动生成材料清单
- 根据签证类型调整材料要求
- 材料模板下载链接
- 材料准备注意事项

### 3. 办理流程指导
-  step-by-step 办理流程
- 在线申请指导
- 面签预约指导
- 材料递交指导

### 4. 办理时间预估
- 正常办理时间
- 加急办理时间
- 建议提前申请时间
- 节假日影响提醒

### 5. 护照/证件提醒
- 护照有效期检查
- 空白页数量检查
- 证件照要求
- 其他证件提醒（驾照、身份证等）

1. **签证政策查询**

   - "办日本签证需要什么材料？"
   - "美国签证面签需要带什么？"
   - "护照照片有什么要求？"

3. **办理流程**
   - "日本签证怎么办理？"
   - "申根签证要去哪里办？"
   - "签证申请流程是什么？"

4. **时间规划**
   - "办日本签证要多久？"
   - "提前多久申请签证比较合适？"
   - "可以加急办理吗？"

   - "护照还有半年过期能出国吗？"

## 输入参数

| 参数 | 类型 | 必填 | 说明 |
|:---|:---|:---|:---|
| destination | string | 是 | 目的地国家 |
| nationality | string | 否 | 国籍，默认中国 |
| visa_type | string | 否 | 签证类型：tourist/business/transit/student |
| stay_duration | int | 否 | 计划停留天数 |
| passport_expiry | string | 否 | 护照过期日期 |

## API接口列表

| 接口名称 | 功能 | 参数 |
|:---|:---|:---|
| check_visa_requirement | 查询签证要求 | destination, nationality |
| get_document_checklist | 获取材料清单 | destination, visa_type |
| get_application_process | 获取办理流程 | destination |
| get_processing_time | 查询办理时间 | destination, visa_type |
| check_passport_validity | 检查护照有效性 | passport_expiry, destination |
| get_visa_free_countries | 查询免签国家 | nationality |
| get_embassy_info | 获取使馆信息 | destination, city |

## 数据来源

- 签证政策：各国外交部/移民局官网
- 中国领事服务网：cs.mfa.gov.cn
- 签证中心：VFS Global, TLScontact
- 更新时间：建议每月更新一次
