# AI目的地指南助手 (ai-destination-guide-skill)

## 技能描述

AI目的地指南助手提供全球旅行目的地的全面信息，包括目的地概览、必去景点、当地美食、交通指南、实用信息、安全提示等。帮助用户在出行前充分了解目的地，做好充分准备。

---

⚠️ 【重要约束】
- 所有信息必须基于真实数据
- 景点开放时间、门票价格需定期更新
- 安全提示需根据官方旅行建议
- 货币汇率信息需标注更新时间
- 签证政策需明确有效期

---

## 核心功能

### 1. 目的地概览
- 基本信息（位置、人口、语言、货币）
- 最佳旅游季节
- 气候特点
- 文化特色
- 历史背景

### 2. 必去景点
- 热门景点详细介绍
- 小众景点推荐
- 景点评分和游客评价
- 开放时间和门票信息
- 游览建议和注意事项

### 3. 当地美食
- 特色美食介绍
- 推荐餐厅列表
- 美食街区指南
- 饮食注意事项
- 人均消费参考

### 4. 交通指南
- 如何到达（航班、火车、汽车）
- 当地交通方式
- 交通卡/通票推荐
- 交通费用参考
- 交通APP推荐

### 5. 实用信息
- 时差
- 电压和插头类型
- 货币和汇率
- 紧急联系方式
- 网络/WiFi情况

### 6. 安全提示
- 安全等级评估
- 常见诈骗提醒
- 医疗信息
- 保险建议
- 使领馆信息

## 触发场景

1. **目的地了解**
   - "巴黎有什么好玩的？"
   - "第一次去泰国需要注意什么？"
   - "京都的最佳旅游季节是什么时候？"

2. **景点查询**
   - "东京有哪些必去景点？"
   - "大阪环球影城值得去吗？"
   - "京都的寺庙推荐"

3. **美食探索**
   - "东京有什么好吃的？"
   - "大阪的道顿堀有什么特色？"
   - "京都的传统美食"

4. **实用信息查询**
   - "去日本需要带转换插头吗？"
   - "泰国的货币怎么兑换？"
   - "欧洲旅游需要买电话卡吗？"

## 输入参数

| 参数 | 类型 | 必填 | 说明 |
|:---|:---|:---|:---|
| destination | string | 是 | 目的地城市或国家 |
| query_type | string | 否 | 查询类型：overview/attractions/food/transport/tips/all |
| language | string | 否 | 返回语言，默认zh |
| detail_level | string | 否 | 详细程度：brief/standard/detailed |

## API接口列表

| 接口名称 | 功能 | 参数 |
|:---|:---|:---|
| get_destination_overview | 获取目的地概览 | destination |
| get_attractions | 获取景点列表 | destination, category, count |
| get_local_foods | 获取当地美食 | destination, food_type |
| get_transport_guide | 获取交通指南 | destination |
| get_travel_tips | 获取旅行贴士 | destination, tip_type |
| search_destination | 搜索目的地 | keyword |

## 数据来源

- 景点信息：Google Places API、TripAdvisor
- 美食推荐：Tabelog、米其林指南
- 交通信息：各城市交通官网
- 安全提示：外交部领事服务网、CDC
- 汇率信息：Open Exchange Rates API
