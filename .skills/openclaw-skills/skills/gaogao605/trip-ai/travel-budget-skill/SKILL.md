# AI旅行预算助手 (ai-travel-budget-skill)

## 技能描述

AI旅行预算助手帮助用户精确计算旅行费用，提供智能预算分配建议，分享实用的省钱攻略，支持多人旅行费用分摊计算。让旅行预算清晰透明，避免超支烦恼。

---

⚠️ 【重要约束】
- 费用估算需基于真实市场价格
- 预算分配需考虑目的地消费水平
- 省钱建议需合法合规
- 汇率信息需标注更新时间
- 预留10-15%应急资金

---

## 核心功能

### 1. 费用预估
- 机票/火车票费用估算
- 酒店住宿费用估算
- 餐饮费用估算
- 交通费用估算
- 景点门票费用估算
- 购物/其他费用估算

### 2. 预算分配
- 根据目的地消费水平智能分配
- 按旅行风格调整预算比例
- 淡旺季价格差异提醒
- 预算上限预警

### 3. 省钱攻略
- 机票购买最佳时机
- 酒店预订优惠技巧
- 当地交通省钱方法
- 餐饮省钱建议
- 免费景点推荐

### 4. 多人分摊
- 费用按人头平均分摊
- 支持按消费比例分摊
- 生成费用明细表
- 结算建议

## 触发场景

1. **预算规划**
   - "去日本玩7天大概要花多少钱？"
   - "帮我算一下三亚5日游的预算"
   - "欧洲15天游，人均2万够吗？"

2. **费用明细**
   - "东京5日游的费用明细"
   - "帮我分析一下预算分配"

3. **省钱建议**
   - "怎么订机票最便宜？"
   - "东京住宿怎么省钱？"
   - "有什么免费景点推荐？"

4. **多人分摊**
   - "4个人去泰国，费用怎么分摊？"
   - "帮我算一下这次旅行的账单"

## 输入参数

| 参数 | 类型 | 必填 | 说明 |
|:---|:---|:---|:---|
| destination | string | 是 | 目的地 |
| duration_days | int | 是 | 旅行天数 |
| travelers | int | 否 | 旅行人数，默认1 |
| budget_level | string | 否 | 预算等级：ECONOMY/STANDARD/COMFORT/LUXURY |
| departure_city | string | 否 | 出发城市 |
| include_flight | bool | 否 | 是否包含大交通，默认true |

## 预算等级说明

| 等级 | 人均日预算 | 住宿 | 餐饮 | 特点 |
|:---|:---|:---|:---|:---|
| ECONOMY | ¥300-500 | 青旅/民宿 | 便利店/快餐 | 背包客首选 |
| STANDARD | ¥500-1000 | 商务酒店 | 定食屋/连锁 | 性价比最高 |
| COMFORT | ¥1000-2000 | 四星酒店 | 特色餐厅 | 舒适体验 |
| LUXURY | ¥2000+ | 五星酒店 | 米其林/怀石 | 奢华享受 |

| 接口名称 | 功能 | 参数 |
|:---|:---|:---|
