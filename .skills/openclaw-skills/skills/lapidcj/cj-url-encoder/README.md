# URL Encoder / Decoder Skill

一个轻量级、纯本地的 URL 编码与解码工具。无需联网，保护隐私，秒级响应。

## ✨ 功能
- **URL 编码 (Encode)**：将中文、空格、特殊符号转换为 `%XX` 格式，适用于构建 API 请求、分享含参链接。
- **URL 解码 (Decode)**：将 `%XX` 格式的字符串还原为可读文本，用于查看加密参数或修复乱码。
- **全自动识别**：智能判断用户意图，无需手动选择模式。

## 🚀 使用示例

### 1. 编码
**用户输入**:
> 把 "https://example.com/search?q=人工智能 教程" 编码一下

**助手回复**:
> ✅ **编码结果**:
> `https%3A%2F%2Fexample.com%2Fsearch%3Fq%3D%E4%BA%BA%E5%B7%A5%E6%99%BA%E8%83%BD%20%E6%95%99%E7%A8%8B`

### 2. 解码
**用户输入**:
> 解码：%E4%BD%A0%E5%A5%BD

**助手回复**:
> ✅ **解码结果**:
> `你好`

## 🛠 技术细节
- **语言**: Python 3.6+
- **依赖**: 仅使用 Python 标准库 `urllib.parse`，无第三方依赖。
- **标准**: 遵循 RFC 3986，使用 UTF-8 字符集。

## 🔒 隐私安全
所有处理均在本地完成，数据不会上传到任何服务器。
