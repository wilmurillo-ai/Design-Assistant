# 简单版CCTV新闻联播获取工具 - 使用说明

## 功能概述

这是一个无需钉钉机器人配置的CCTV新闻联播获取工具，获取的内容直接保存到本地文本文件，适合个人使用。

## 系统要求

- Python 3.6或更高版本
- 网络连接（可访问CCTV网站）

## 安装步骤

1. 安装必要的Python库：
```bash
pip install requests pytz lxml
```

2. 将skill文件复制到您的OpenClaw workspace目录

## 使用方法

### 手动执行

```bash
cd /path/to/cctv-news-simple
python scripts/get_cctv_news_simple.py
```

执行后会在脚本目录下生成 `cctv_news_YYYYMMDD.txt` 文件。

### 定时执行（每晚22:00）

#### Windows系统

1. 打开"任务计划程序"
2. 点击"创建基本任务"
3. 设置任务名称：CCTV新闻联播获取
4. 设置触发器：每天
5. 设置开始时间：22:00
6. 设置操作：启动程序
   - 程序或脚本：python.exe
   - 添加参数：C:\path\to\scripts\get_cctv_news_simple.py
   - 起始于：C:\path\to\cctv-news-simple

#### Linux/Unix系统

编辑crontab：
```bash
crontab -e
```

添加以下行（22:00北京时间 = 14:00 UTC）：
```
0 14 * * * cd /path/to/cctv-news-simple && python scripts/get_cctv_news_simple.py >> /var/log/cctv_news.log 2>&1
```

#### macOS系统

使用launchd或crontab，配置方法与Linux相同。

## 自定义配置

### 修改输出目录

编辑脚本文件 `get_cctv_news_simple.py`，找到 `save_to_file` 函数：

```python
# 修改这一行来更改输出目录
output_dir = "C:/your/custom/path"  # 替换为您想要的目录
```

### 修改CCTV网站URL

如果需要修改CCTV网站URL，编辑脚本中的这一行：
```python
url1 = "https://tv.cctv.com/lm/xwlb/"
```

## 故障排查

### 1. 无法获取内容
- 检查网络连接
- 确认CCTV网站可访问
- 查看脚本输出的错误信息

### 2. 文件保存失败
- 检查脚本目录是否有写权限
- 确认磁盘空间充足

### 3. 时间不正确
- 检查系统时区设置
- 确认系统时间准确

## 日志记录

建议在cron配置中添加日志记录：
```
0 14 * * * cd /path/to/skill && python scripts/get_cctv_news_simple.py >> logs/cctv_news.log 2>&1
```

## 备份建议

建议定期备份生成的新闻文件，或设置自动归档策略。
