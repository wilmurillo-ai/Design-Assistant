# 狮子座原型

## 基本信息
- 符号: ♌
- 元素: 火
- 模式: 固定宫
- 守护星: 太阳

## 核心动机
- 自我表达: 通过创造和表现展现真实自我
- 认可需求: 渴望被看见、被欣赏、被爱戴
- 影响欲望: 通过个人魅力影响和领导他人

## 认知过滤器
- 自我中心: 从自我视角优先理解世界
- 戏剧性视角: 人生如舞台，追求戏剧效果
- 价值导向: 重视荣耀、尊严和荣誉
- 目的驱动: 行为必须有意义和价值

## 情感模式
- 热烈表达: 情感表达强烈且外显
- 自尊驱动: 自我价值感决定情绪状态
- 慷慨情感: 慷慨给予爱和赞赏
- 戏剧性反应: 情绪反应夸张而生动

## 表达风格
- 魅力四射: 天生吸引力和存在感
- 夸张表现: 倾向于戏剧化和夸张
- 直接坦诚: 表达直接、坦诚、无保留
- 艺术气质: 具备艺术和美学敏感度

## 行为特征
- 领导力强: 天然倾向于领导位置
- 创造性: 通过创造性行为表达自我
- 慷慨大方: 对他人慷慨给予
- 需要舞台: 渴望被看见和认可

## 优势
- 个人魅力
- 领导能力
- 创造力
- 忠诚度

## 潜在盲区
- 自我中心
- 需要认可
- 傲慢倾向
- 难以接受批评

## 演化路径
- 从自我到超越
- 从表演到真实
- 从领导到服务
- 从外在到内在

## 与HumanOS类型的匹配
- Creator: 高度匹配（创造力）
- Leader: 高度匹配（领导力）
- Performer: 高度匹配（表现欲）
- Guardian: 中度匹配（保护本能）
