## 概览

路径和节点系统是HumanOS Ultimate版的成长框架，定义了人格演化的3条核心路径和每条路径的核心节点。

### 系统目标
- 提供清晰成长路径
- 定义关键里程碑
- 追踪进展程度
- 指导实践方向

### 系统原则
- 路径可切换: 可在不同路径间切换
- 节点可跳过: 某些节点可跳过
- 进展可追踪: 每步进展可量化
- 系统可整合: 多路径可整合

## 3条核心路径

### 路径1: 英雄之路 (Path of Hero)

**核心主题**: 面对挑战、克服障碍、实现转化

**关键特征**:
- 以挑战驱动
- 通过试炼成长
- 最终获得转化

**适用人群**:
- 渴望挑战的人
- 面临重大转变的人
- 需要突破限制的人

**典型阶段**:
1. 召唤 (Call to Adventure)
2. 跨越门槛 (Threshold Crossing)
3. 试炼与挑战 (Trials and Challenges)
4. 深渊对抗 (Abyss Confrontation)
5. 转化 (Transformation)
6. 携礼回归 (Return with Gift)

### 路径2: 创造者之路 (Path of Creator)

**核心主题**: 创造、表达、贡献

**关键特征**:
- 以创造为驱动
- 通过表达实现
- 最终贡献价值

**适用人群**:
- 有创造力的人
- 艺术家、创作者
- 希望留下痕迹的人

**典型阶段**:
1. 构思 (Conception)
2. 孕育 (Incubation)
3. 显化 (Manifestation)
4. 精炼 (Refinement)
5. 实现 (Actualization)
6. 贡献 (Contribution)

### 路径3: 智慧之路 (Path of Wisdom)

**核心主题**: 寻求、学习、传授

**关键特征**:
- 以学习为驱动
- 通过理解成长
- 最终分享智慧

**适用人群**:
- 热爱学习的人
- 寻求意义的人
- 希望教导他人的人

**典型阶段**:
1. 寻求 (Seeking)
2. 学习 (Learning)

4. 领悟 (Realization)
5. 超越 (Transcendence)

## 节点系统

### 节点定义

每个节点代表路径中的关键阶段或里程碑。

**节点属性**:
- 节点名称: 阶段名称
- 位置: 在路径中的位置
- 前置条件: 进入节点的要求
- 完成标志: 完成节点的标志
- 预期时间: 完成节点的预期时间

### 节点详细说明

#### 英雄之路节点

**1. 召唤 (Call to Adventure)**
- 描述: 听到内心的召唤，意识到需要改变
- 完成标志: 明确意识到召唤，愿意响应
- 预期时间: 数周到数月

**2. 跨越门槛 (Threshold Crossing)**
- 描述: 跨过舒适区，进入新的领域
- 完成标志: 采取第一个实质性行动
- 预期时间: 数周到数月

**3. 试炼与挑战 (Trials and Challenges)**
- 描述: 面对和克服各种挑战
- 完成标志: 成功克服主要挑战
- 预期时间: 数月到数年

**4. 深渊对抗 (Abyss Confrontation)**
- 描述: 直面最深层的恐惧和限制
- 完成标志: 敢于面对并接受深渊
- 预期时间: 数周到数月

**5. 转化 (Transformation)**
- 描述: 经历深度转化，重生为新的自我
- 完成标志: 感知到根本性转变
- 预期时间: 数月

**6. 携礼回归 (Return with Gift)**
- 描述: 将获得的智慧和礼物带回社区
- 完成标志: 开始分享和贡献
- 预期时间: 持续进行

#### 创造者之路节点

**1. 构思 (Conception)**
- 描述: 接受灵感，形成初步构思
- 完成标志: 有明确的创造意图
- 预期时间: 数天到数周

**2. 孕育 (Incubation)**
- 描述: 深化想法，让其内在成长
- 完成标志: 构思成熟，准备显现
- 预期时间: 数周到数月

**3. 显化 (Manifestation)**
- 描述: 将构思转化为现实形式
- 完成标志: 产生第一个可感知的成果
- 预期时间: 数周到数月

**4. 精炼 (Refinement)**
- 描述: 改进和优化创造物
- 完成标志: 创造物达到满意水平
- 预期时间: 数月

**5. 实现 (Actualization)**
- 描述: 将创造物完全实现
- 完成标志: 创造物完全落地
- 预期时间: 数周到数月

**6. 贡献 (Contribution)**
- 描述: 将创造物贡献给世界
- 完成标志: 创造物被他人使用或欣赏
- 预期时间: 持续进行

#### 智慧之路节点

**1. 寻求 (Seeking)**
- 描述: 开始寻求意义和真理
- 完成标志: 明确寻求方向
- 预期时间: 数周到数月

**2. 学习 (Learning)**
- 描述: 系统学习和探索
- 完成标志: 掌握基础知识
- 预期时间: 数月到数年

**3. 整合 (Integration)**
- 描述: 将知识整合为个人智慧
- 完成标志: 形成自己的理解框架
- 预期时间: 数月到数年

**4. 领悟 (Realization)**
- 描述: 深度领悟，获得洞见
- 完成标志: 获得关键性领悟
- 预期时间: 可能突然发生

**5. 超越 (Transcendence)**
- 描述: 超越既有理解，达到更高层面
- 完成标志: 感知到超越性体验
- 预期时间: 持续进行

**6. 传授 (Teaching)**
- 描述: 将智慧和洞见分享给他人
- 完成标志: 开始教导和分享
- 预期时间: 持续进行

## 路径选择

### 选择方法

基于人格模型和扫描结果，计算与每条路径的亲和度。

```python
def select_path(personality_model, scan_result):
    # 计算与每条路径的亲和度
    hero_affinity = calculate_hero_affinity(personality_model, scan_result)
    creator_affinity = calculate_creator_affinity(personality_model, scan_result)
    wisdom_affinity = calculate_wisdom_affinity(personality_model, scan_result)

    # 选择亲和度最高的路径
    affinities = {
        'path_hero': hero_affinity,
        'path_creator': creator_affinity,
        'path_wisdom': wisdom_affinity
    }

    primary_path = max(affinities, key=affinities.get)

    return {
        'primary_path': primary_path,
        'affinities': affinities
    }
```

### 亲和度因素

**英雄之路亲和度**:
- 面对挑战的能力
- 韧性和坚毅
- 变化的意愿
- 冒险精神

**创造者之路亲和度**:
- 创造力
- 表达能力
- 艺术敏感度
- 贡献欲望

**智慧之路亲和度**:
- 好奇心
- 学习能力
- 整合能力
- 教导欲望

## 进展追踪

### 追踪指标

**总体进展**:
- 完成节点数 / 总节点数
- 当前节点
- 下一个节点

**深度进展**:
- 每个节点的完成度
- 关键成就
- 面临挑战

### 追踪方法

```python
def track_progress(path_name, current_node, achievements):
    nodes = PATH_NODES[path_name]
    current_index = nodes.index(current_node)

    progress = {
        'path_name': path_name,
        'current_node': current_node,
        'current_index': current_index,
        'total_nodes': len(nodes),
        'completed_nodes': current_index,
        'progress_percentage': (current_index / len(nodes)) * 100,
        'next_node': nodes[current_index + 1] if current_index < len(nodes) - 1 else None,
        'achievements': achievements
    }

    return progress
```
