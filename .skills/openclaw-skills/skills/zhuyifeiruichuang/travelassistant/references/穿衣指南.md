# 穿衣指南参考

## 按温度分层

### 炎热天气（30°C以上）
- **男性**：轻薄短袖、短裤、透气运动鞋、遮阳帽
- **女性**：轻薄连衣裙、短裤、凉鞋、防晒衣
- **儿童**：纯棉短袖、防蚊裤、遮阳帽
- **老人**：轻薄长袖（防晒）、宽松长裤、舒适凉鞋

### 温暖天气（20-30°C）
- **男性**：短袖/薄长袖、长裤、运动鞋、薄外套
- **女性**：衬衫/薄针织衫、裙装/长裤、平底鞋
- **儿童**：长袖T恤、长裤、运动鞋、外套
- **老人**：舒适长袖、裤子、软底鞋、保暖外套

### 凉爽天气（10-20°C）
- **男性**：长袖、薄毛衣、夹克、长裤、皮鞋/运动鞋
- **女性**：针织衫、薄毛衣、风衣、裙装+打底裤
- **儿童**：保暖内衣、长袖、外套、裤子
- **老人**：保暖内衣、棉衣、羽绒服、围巾、帽子

### 寒冷天气（10°C以下）
- **男性**：保暖内衣、羽绒服、毛衣、保暖裤、棉鞋、手套、围巾、帽子
- **女性**：保暖内衣、羽绒服、高领毛衣、保暖裤、雪地靴、手套、围巾、帽子
- **儿童**：加绒保暖内衣、羽绒服、棉裤、保暖鞋、手套、帽子
- **老人**：加绒保暖内衣、厚羽绒服、棉裤、棉鞋、围巾、帽子、暖宝宝

## 特殊场景

### 雨季出行
- 防水外套/冲锋衣
- 雨伞或雨衣
- 防滑鞋
- 防水背包套

### 海边/沙滩
- 泳衣/泳裤
- 沙滩裙/沙滩裤
- 防晒霜（SPF50+）
- 墨镜
- 沙滩鞋/拖鞋
- 遮阳帽

### 高海拔地区
- 保暖衣物（即使夏季也需带）
- 防风外套
- 保暖帽子
- 手套
- 墨镜（防紫外线）
- 防晒霜

### 户外徒步
- 速干衣裤
- 登山鞋/徒步鞋
- 防晒衣
- 遮阳帽
- 保暖层（备用）
- 雨衣

### 商务出行
- 正装/衬衫
- 西装外套（视情况）
- 正装皮鞋
- 领带/丝巾（视情况）
- 商务休闲装

## 颜色建议

### 避免的颜色（特定场景）
- **户外/森林**：避免亮红色、黄色（可能吸引昆虫/蜂类）
- **野生动物保护区**：避免白色、亮色
- **夜晚户外**：避免深黑色（不易被车辆识别）

### 建议的颜色
- **夏季户外**：浅色系（白色、米色、浅蓝）
- **拍照打卡**：根据景点背景选择对比色或协调色
- **日常出行**：舒适为主，避免过于暴露
