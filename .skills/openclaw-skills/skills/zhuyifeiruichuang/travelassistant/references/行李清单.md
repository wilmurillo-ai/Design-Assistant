# 行李清单参考

## 证件类

| 物品 | 说明 |
|-----|-----|
| 身份证 | 国内游必备 |
| 护照 | 出境游必备 |
| 签证 | 确认有效期 |
| 驾驶证 | 自驾必备 |
| 机票/火车票 | 电子票或打印件 |
| 酒店预订单 | 电子版或打印件 |
| 保险单 | 旅行保险 |
| 银行卡/信用卡 | 确认额度 |
| 现金 | 适量（部分场景需现金） |

## 电子设备类

| 物品 | 说明 |
|-----|-----|
| 手机 | 确认充电器 |
| 充电宝 | 新国标（≤100Wh），随身携带 |
| 充电器 | 手机、电脑等 |
| 充电线 | 各设备充电线 |
| 转换插头 | 境外游 |
| 耳机 | 旅途娱乐 |
| 相机/自拍杆 | 如有需要 |
| 存储卡/读卡器 | 摄影爱好者 |

## 特殊人群额外物品

| 人群 | 额外物品 |
|-----|---------|
| 老年人 | 老花镜、助行器、常用医疗记录 |

## 违禁品提醒

### 禁止随身携带
- 超过100Wh的充电宝
- 液体超过100ml（除航空规定豁免）
- 打火机（部分场所）
- 管制刀具

### 禁止托运
- 锂电池
- 充电宝
- 电子烟

### 特殊物品申报
- 超过10万现金
- 动植物制品
- 药品（带处方证明）
