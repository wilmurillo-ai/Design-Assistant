# 版式微调规则

本文档定义了生成稳定、一致 PPT 的精确版式规则。所有渲染实现必须严格遵循本文档，不得随意偏离。

> **重要**：完整的视觉规范（包括精确坐标、颜色 hex 值、字体大小）请参见 `references/visual-rendering-spec.md`。本文档是规则索引和高层约束，两者必须配合使用。

---

## 1. 字号优先级（绝对规则）

当内容放不下时，优先顺序：
1. **拆页**
2. 精简次要信息
3. 调整布局
4. **最后**才是缩小字体

**禁止**：从一开始就把正文字号压到 13pt 以下。

---

## 2. 标题与正文必须拉开层级

- 标题字号（header 内）必须 ≥ 25pt
- 正文 bullet 必须 ≥ 13pt
- 层级差至少 6pt

---

## 3. 坐标与边距规则

- 所有 x 坐标以 0.1" 为最小单位对齐
- 内容左边界不得小于 0.48"（略大于 0.5" 安全区）
- 内容右边界不得大于 13.1"（留 0.23" 给页码）
- 正文区垂直起点约 1.05"–1.2"，底部不超过 7.1"

详见 `visual-rendering-spec.md` 第 1 节。

---

## 4. 颜色规则（禁止自创）

所有颜色必须来自以下固定色板，不得在渲染代码中自创 hex 值：

| 用途 | Hex |
|------|-----|
| navy（主背景）| `13293D` |
| navy2（侧栏）| `1B3A57` |
| blue | `2A628F` |
| teal（主强调）| `2CA6A4` |
| gold（金色强调）| `D5A44A` |
| light（浅背景）| `F5F8FC` |
| white | `FFFFFF` |
| dark（正文）| `18222D` |
| mid（次要文字）| `5C6773` |
| line（边框）| `D6E1EB` |

---

## 5. 字体规则

- 所有文字框必须显式指定 `fontFace`
- 不得依赖 PptxGenJS 默认字体行为
- 标题类使用 `Aptos Display`，正文类使用 `Aptos`
- 详见 `visual-rendering-spec.md` 第 3 节

---

## 6. 阴影规则

统一使用 `softShadow()` 函数返回的对象；**每次调用必须返回新对象**，不得复用同一个对象引用。详见 `visual-rendering-spec.md` 第 4 节。

---

## 7. 结果页图片规则

- 图片区域固定：`x=4.82", y=1.03", w=8.02", h=3.98"`
- 必须使用 `sizing: { type: 'contain' }`，不得裁切原图
- 图片在图片区域内水平垂直居中
- 当同一张 Figure 需拆成 2–3 页讲解时，允许在多页中重复放置同一张原图
- 这种拆页默认不切图，仅更换每页标题与解释重点
- 详见 `visual-rendering-spec.md` 第 5.4 节

---

## 8. 目录页与结束页

### 8.1 目录页
- 编号圆直径 0.5"，居中对齐
- 项目标题 21pt bold，bullet 正文 18pt
- 详见 `visual-rendering-spec.md` 第 5.2 节

### 8.2 结束页
- **必须使用深色背景**（navy + navy2），不混用浅色
- "谢谢！"字号 30pt bold，"感谢批评指正！"字号 21pt
- 不得使用白色或浅色背景作为结束页
- 详见 `visual-rendering-spec.md` 第 5.8 节

---

## 9. 关于动画

- 即使没有原生 PowerPoint 动画，也应通过拆页、章节页、路径式讲述来提供节奏感
- 优先"逐步呈现的页序设计"，而不是在单页上塞满所有信息
- 若当前工具链无法稳定写入动画，不要为了动画牺牲文件稳定性

---

## 10. 渲染脚本结构要求

渲染脚本必须遵循 `visual-rendering-spec.md` 第 8 节规定的结构，包括：
- 颜色常量集中声明
- 工具函数（softShadow、sanitize）统一实现
- 每个 slide_type 有独立布局函数
- 输出文件名使用论文标题

禁止：
- 在渲染函数内部声明颜色常量（导致不可复用）
- 硬编码魔法数字（如 `x: 4.83` 而非 `x: 4.82`）
- 在 slide 循环外做单例 shadow 对象复用

---

## 11. 质量自检清单（渲染层面）

渲染完成后，逐项确认：
- [ ] 所有 slide 的 header navy 条高度一致（h=0.78"）
- [ ] 所有 slide 的 footer 位置一致（y=7.02"）
- [ ] 所有文字显式指定 fontFace，无默认字体依赖
- [ ] 所有颜色值来自固定色板，无自创颜色
- [ ] 所有 shadow 调用使用 `softShadow()` 函数（每次新对象）
- [ ] 所有图片使用 `sizing: { type: 'contain' }`，不裁切
- [ ] 结果页图片区域：x=4.82", y=1.03", w=8.02", h=3.98"
- [ ] 机制页左侧路径卡：x=0.62", w=5.0"
- [ ] 结束页使用深色背景（navy + navy2），不混用浅色
- [ ] 所有 bullet 正文字号 ≥ 13pt
- [ ] 输出文件名使用论文标题，无泛化名称
