# 首选站点优先级表

## 1. 航班 / 机票
1. 携程机票 / Trip.com 实时页
2. 航司官网 / 航司官方小程序
3. 航旅类官方或机场动态页
4. FlightConnections / DirectFlights（用于确认航线，不用于最终价格）
5. 公开搜索结果页（仅初筛）

## 2. 酒店 / 房态
1. 携程酒店 / 飞猪 / 去哪儿实时页
2. 酒店官网
3. 地图平台酒店页
4. 公开搜索结果页

## 3. 商品价格 / 参数 / 可售状态
1. 官方商品页
2. 京东 / 天猫 / 淘宝 / 拼多多等实时页
3. 垂直评测站
4. 社区口碑（知乎 / 小红书 / B站）
5. 公开搜索结果页

## 4. GitHub / 技术文档 / API
1. 官方 docs
2. GitHub 仓库 / release / issues
3. Stack Overflow / 社区 Q&A
4. 普通搜索引擎结果页

## 5. 中文经验 / 口碑 / 对比
1. 知乎
2. 小红书
3. Bilibili
4. 微博
5. 百度 / 必应中文

## 6. 新闻 / 热点 / 最新动态
1. 官方来源 / 原始媒体
2. 主流新闻搜索入口
3. 微博 / 讨论社区（看舆情，不当最终事实）
4. 聚合页

## 7. PDF / 报告 / 白皮书
1. 官方机构站点
2. `site:` + `filetype:pdf` 搜索
3. 学术搜索 / 论文站点
4. 普通搜索结果页

## 8. 学术 / 研究
1. Google Scholar
2. arXiv / 论文主页 / 官方实验室
3. 普通 Google / DuckDuckGo
4. 社区解读文章

## 使用规则
- 先去**最接近原始来源**的站点
- 社区内容适合补充体验，不适合替代事实页
- 聚合页适合发现候选，不适合直接下结论
