# 站点直达模板

## 国内

### 知乎
- 内容搜索：`https://www.zhihu.com/search?type=content&q={keyword}`
- 问题搜索：`https://www.zhihu.com/search?type=content&q={keyword}`

### Bilibili
- 综合搜索：`https://search.bilibili.com/all?keyword={keyword}`
- 视频搜索：`https://search.bilibili.com/video?keyword={keyword}`

### 微博
- 综合：`https://s.weibo.com/weibo/{keyword}`

### 豆瓣
- 搜索：`https://www.douban.com/search?q={keyword}`

### 微信文章
- 搜狗微信：`https://wx.sogou.com/weixin?type=2&query={keyword}`

### 小红书 fallback
- Google 站内：`https://www.google.com/search?q=site:xiaohongshu.com+{keyword}`
- 百度 站内：`https://www.baidu.com/s?wd=site:xiaohongshu.com+{keyword}`

## 国际

### GitHub
- Google 站内：`https://www.google.com/search?q=site:github.com+{keyword}`
- DuckDuckGo Bang：`https://duckduckgo.com/html/?q=!gh+{keyword}`

### Stack Overflow
- DuckDuckGo Bang：`https://duckduckgo.com/html/?q=!so+{keyword}`
- Google 站内：`https://www.google.com/search?q=site:stackoverflow.com+{keyword}`

### Google Scholar
- `https://scholar.google.com/scholar?q={keyword}`

### arXiv
- Google 站内：`https://www.google.com/search?q=site:arxiv.org+{keyword}`

### 官方文档
- `https://www.google.com/search?q=site:docs.{domain}+{keyword}`
- `https://www.google.com/search?q=site:developer.{domain}+{keyword}`

## 出行类

### 航班
- 百度：`https://www.baidu.com/s?wd={from}+{to}+直飞+航班`
- 必应中文：`https://cn.bing.com/search?q={from}+{to}+航班+时刻表&ensearch=0`
- Google：`https://www.google.com/search?q={from}+{to}+direct+flight`

### 酒店
- 百度：`https://www.baidu.com/s?wd={city}+酒店+推荐`
- Google：`https://www.google.com/search?q={city}+hotel+best`
