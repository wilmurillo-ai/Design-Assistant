# 查询意图分类

先判断问题属于哪一类，再选搜索路径。

## 1. 导航型
目标：找官网、入口、页面、站点。

### 适合路径
- 官方站点优先
- `site:` 搜索优先

## 2. 事实型
目标：找一个相对确定的事实答案。

### 适合路径
- 官方来源
- 结构化来源
- 多源交叉

## 3. 对比型
目标：在多个选项中做选择。

### 适合路径
- 官方参数 + 社区口碑 + 价格页
- 不要只看单一搜索结果

## 4. 动态型
目标：查实时变化信息。

### 适合路径
- 实时页/官方页优先
- 公开搜索只做初筛

## 5. 经验型
目标：看别人怎么用、怎么评价、踩过什么坑。

### 适合路径
- 知乎 / 小红书 / B站 / Reddit / 论坛

## 6. 资料型
目标：找 PDF、白皮书、报告、论文、文档。

### 适合路径
- `site:` + `filetype:`
- 官方站 / 论文站 / 文档站

## 快速判断示例
- “官网在哪” → 导航型
- “今天直飞还有吗” → 动态型
- “A 和 B 哪个更值” → 对比型
- “大家怎么评价” → 经验型
- “给我找白皮书 PDF” → 资料型
