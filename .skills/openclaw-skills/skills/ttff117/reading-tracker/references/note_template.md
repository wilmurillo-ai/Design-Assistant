# 读书笔记模板 / Book Note Templates

## 强制输出三问 / The 3 Mandatory Questions

读完一本书后，必须回答以下三个问题。
> **EN**: After finishing a book, you must answer these 3 questions before marking it as complete.

---

### 1. 一句话总结 / One-Sentence Summary

用一句话概括这本书的核心观点。
> **EN**: Summarize the book's core idea in one sentence.

**提示 / Tips：**
- 不要罗列多个观点，只选最重要的一个
- 尽量用自己的话表达
- 如果朋友问你这本书讲什么，你会怎么回答？

> - Don't list multiple points — pick the most important one
> - Use your own words
> - How would you explain it to a friend in one sentence?

**示例 / Example：**
- ❌ "这本书讲了很多关于习惯养成的内容..."
- ✅ "习惯是自我提高的复利，微小改进带来巨大改变"
- ✅ EN: "Habits are the compound interest of self-improvement — tiny changes lead to remarkable results"

---

### 2. 认知改变 / Mindset Shift

这本书改变了你什么想法？
> **EN**: What did this book change about how you think?

**提示 / Tips：**
- 读之前你是怎么想的？读之后有什么不同？
- 哪个观点让你有"啊哈"时刻？

> - What did you believe before? What's different now?
> - Which idea gave you an "aha" moment?

**示例 / Example：**
- ❌ "这本书写得很好"
- ✅ "以前觉得养成习惯靠意志力，现在意识到应该设计环境"
- ✅ EN: "I used to think habits required willpower; now I know it's about designing your environment"

---

### 3. 行动计划 / Action Plan

你会采取什么具体行动？
> **EN**: What specific action will you take?

**提示 / Tips：**
- 必须具体、可执行，有明确的时间和场景
- 从小处着手，不要贪多

> - Must be specific and actionable with a clear time/context
> - Start small — one action is better than ten intentions

**示例 / Example：**
- ❌ "我要养成好习惯" / "I'll build better habits"
- ✅ "起床后立刻喝水（已有习惯）→ 做5个俯卧撑（新习惯）"
- ✅ EN: "After waking up (existing habit) → drink water → do 5 push-ups (new habit)"

---

## 书摘记录格式 / Quote Format

```markdown
### YYYY-MM-DD 第X章/Ch.X（进度%）
> "原文金句 / Original quote"

**我的想法/My thought**：你的思考、联想、质疑
```

## 复习记录格式 / Review Log Format

```markdown
- [YYYY-MM-DD] 复习内容/Review: 回顾了第X章关于XXX的观点
  新感悟/New insight: ...
```
