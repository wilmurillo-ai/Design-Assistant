# 复习提醒问题库

## 章节复习（1天、3天后）

### 通用问题
1. 你记录的这段话，现在还记得核心意思吗？用自己的话复述一下。
2. 当时你说"{thought}"，现在回头看，有什么新的想法？
3. 这段话和你最近的生活/工作有什么关联吗？
4. 如果要把这个观点讲给别人听，你会怎么表达？

### 行动类书摘
1. 你当时想尝试"{thought}"，实践了吗？效果如何？
2. 遇到什么阻碍了吗？需要调整方法吗？
3. 如果还没开始，是什么在阻止你？

### 观点类书摘
1. 这个观点你同意吗？有没有发现反例？
2. 和其他书里的观点有什么联系或冲突？
3. 如果应用到实际，会有什么后果？

---

## 全书复习（7天、30天后）

### 7天复习
1. 一句话总结这本书的核心观点，现在还能准确说出来吗？
2. 你当时的行动计划是什么？进展如何？
3. 书里最让你印象深刻的一个例子是什么？
4. 如果让你推荐这本书给一个朋友，你会怎么说？

### 30天复习
1. 一个月过去了，这本书对你产生了什么实际影响？
2. 你的行动计划还在坚持吗？如果中断了，为什么？
3. 回看当时的认知改变，现在还认同吗？
4. 如果要重读这本书，你会重点关注哪些章节？
5. 这本书和你这个月读的其他书有什么关联？

---

## 个性化追问

根据用户的具体回答，可以追问：

- "你提到{具体细节}，能展开说说吗？"
- "这和{之前读的另一本书}的观点有什么联系？"
- "如果{假设场景}，这个观点还适用吗？"
- "你打算什么时候开始行动？"
