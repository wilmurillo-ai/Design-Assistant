## 🎯 项目简介

MCP Hello World 是一个最小可行的 MCP（Model Context Protocol）服务器示例，展示了如何在 OpenClaw 生态中构建和发布 MCP 工具。

**核心功能：**
- 🔧 **add 工具** - 两数相加
- 👋 **hello_world 工具** - 个性化问候

**技术栈：**
- Node.js 22+
- @modelcontextprotocol/sdk
- zod

## 🚀 快速开始

### 安装
```bash
clawhub install mcp-hello-world
```

### 使用
```bash
# 列出工具
mcporter list

# 调用 add
mcporter call hello-world.add a:10 b:20

# 调用 hello_world
mcporter call hello-world.hello_world name:"朋友"
```

### 测试
```bash
npm test
```
