## 💡 OpenClaw 集成方式

### 方式 1: 通过 mcporter CLI（已验证 ✅）

在 OpenClaw 聊天中，可以通过执行 mcporter 命令调用：

```
mcporter call hello-world.add a:10 b:20
```

### 方式 2: 通过 Agent 工具调用（推荐）

OpenClaw Agent 可以自动识别 mcporter 配置的工具，通过自然语言触发：

**用户输入：** "帮我计算 10 加 20"
**Agent 响应：** 调用 `hello-world.add` 工具，返回结果

**用户输入：** "跟我打个招呼"
**Agent 响应：** 调用 `hello-world.hello_world` 工具，返回问候语

## ⚠️ 坑点记录

### 坑点 3: mcporter 配置格式

**问题：** 初始配置使用 `servers` 键，mcporter 期望 `mcpServers`

**错误信息：**
```
ZodError: Invalid input: expected record, received undefined
```

**解决：** 修改配置为：
```json
{
  "mcpServers": {
    "hello-world": { ... }
  }
}
```

**经验：** 参考 mcporter 官方文档的配置格式
