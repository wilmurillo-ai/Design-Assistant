## 📋 开发流程

### 1. 环境检查 ⚠️

**坑点 1：Python 版本过低**
- 系统 Python: 3.6.8
- MCP SDK 要求：Python 3.10+
- **解决方案：** 改用 Node.js 方案（系统已有 Node.js 22.22.0）

**决策记录：**
- 原计划：Python + FastMCP
- 实际采用：Node.js + @modelcontextprotocol/sdk
- 理由：云端环境 Node.js 版本新，无需额外安装

### 2. 依赖安装 ⚠️

**坑点 2：npm 安装超时**
- 问题：默认 npm registry 连接超时
- **解决方案：** 切换到淘宝镜像
```bash
npm config set registry https://registry.npmmirror.com
npm install @modelcontextprotocol/sdk zod
```
- 结果：8 秒完成安装

### 3. 服务器开发 ✅

**文件：** `src/server.js`

**关键代码：**
```javascript
import { McpServer } from "@modelcontextprotocol/sdk/server/mcp.js";
import { StdioServerTransport } from "@modelcontextprotocol/sdk/server/stdio.js";
import { z } from "zod";

const server = new McpServer({ name: "hello-world", version: "1.0.0" });

server.tool("add", "描述", { a: z.number(), b: z.number() }, async ({ a, b }) => {
  return { content: [{ type: "text", text: `${a} + ${b} = ${a+b}` }] };
});
```

**经验：**
- 使用 ES Modules (`"type": "module"`)
- Zod 用于参数验证和类型定义
- 服务器日志输出到 stderr，MCP 消息通过 stdin/stdout

## 🎯 技术要点总结

### MCP 协议核心

1. **初始化握手**
```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "initialize",
  "params": {
    "protocolVersion": "2024-11-05",
    "clientInfo": { "name": "xxx", "version": "1.0.0" }
  }
}
```

2. **工具列表**
```json
{
  "jsonrpc": "2.0",
  "id": 2,
  "method": "tools/list"
}
```

3. **工具调用**
```json
{
  "jsonrpc": "2.0",
  "id": 3,
  "method": "tools/call",
  "params": {
    "name": "add",
    "arguments": { "a": 3, "b": 5 }
  }
}
```

### Node.js MCP SDK 使用

**导入：**
```javascript
import { McpServer } from "@modelcontextprotocol/sdk/server/mcp.js";
import { StdioServerTransport } from "@modelcontextprotocol/sdk/server/stdio.js";
import { z } from "zod";
```

**创建服务器：**
```javascript
const server = new McpServer({
  name: "your-server",
  version: "1.0.0",
  description: "描述"
});
```

**添加工具：**
```javascript
server.tool(
  "工具名",
  "描述",
  {
    参数名：z.类型().describe("描述").optional().default(默认值)
  },
  async ({ 参数 }) => {
    return {
      content: [{ type: "text", text: "结果" }]
    };
  }
);
```

**启动：**
```javascript
const transport = new StdioServerTransport();
await server.connect(transport);
```
