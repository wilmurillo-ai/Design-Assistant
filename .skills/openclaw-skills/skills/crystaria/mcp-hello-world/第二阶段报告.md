## ⚠️ 坑点与解决方案

### 坑点 1: Python 版本过低
- **问题：** 系统 Python 3.6.8，MCP SDK 要求 3.10+
- **解决：** 改用 Node.js 方案（系统已有 Node.js 22.22.0）
- **经验：** 提前检查云端环境，选择合适技术栈

### 坑点 2: npm 安装超时
- **问题：** 默认 npm registry 连接超时
- **解决：** 切换到淘宝镜像
```bash
npm config set registry https://registry.npmmirror.com
```
- **效果：** 安装时间从超时缩短到 8 秒
