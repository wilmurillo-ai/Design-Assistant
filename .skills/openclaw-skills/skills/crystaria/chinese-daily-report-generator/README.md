# 职场人日报/周报生成器

**版本：** 1.1.0  
**作者：** Crystaria (with Paw and Kyle)

---

## 📖 简介

这是一个帮你自动生成结构化日报/周报的 AI 助手技能。

**核心价值：**
- ✅ 自动整理工作记录
- ✅ 结构化呈现完成项、待办、重要信息
- ✅ **问题复盘板块** — 记录问题发现、分析、解决过程，沉淀经验

---

## 🚀 快速开始

### 1. 安装

在 OpenClaw 中运行：
```bash
clawhub install chinese-daily-report-generator
```

### 2. 使用

直接对 AI 说：

```
帮我生成今天的日报
```

或

```
帮我生成这周的周报
```

### 3. AI 会问你

- 今天/本周完成了什么工作？
- 有什么待办事项？
- 有什么重要信息需要记录？
- 遇到了什么问题或挑战？

**提供这些信息后，AI 会自动生成完整报告并保存。**

---

## 📁 报告保存位置

| 报告类型 | 保存路径 |
|----------|----------|
| 日报 | `~/reports/daily/日报_YYYY-MM-DD.md` |
| 周报 | `~/reports/weekly/周报_YYYY-MM-DD_至_YYYY-MM-DD.md` |

**注：** `~` 指工作区根目录（即 OpenClaw workspace 的根目录）

---

## 📝 示例

### 用户输入

```
请帮我生成今天的日报

今天完成了：
- 完成了 XX 项目的需求文档
- 开了 3 个会
- 修复了 2 个 bug

待办：
- 明天要和设计对稿
- 周五前提交测试

重要：
- 老板下周一出差
- 项目 deadline 是下周五

问题：
- 会议太多，下午没时间写代码
```

### AI 输出

生成完整 Markdown 报告，包含：
- ✅ 完成的工作
- ⏳ 待办事项
- 📌 重要信息
- 🧩 问题与解决方案（含发现过程、原因分析、解决过程、经验总结）

---

## 🎯 报告标准

| 报告类型 | 长度 | 问题复盘要求 |
|----------|------|-------------|
| 日报 | 300-800 字 | 至少 1 个 |
| 周报 | 800-2000 字 | 3-5 个 |

---

## ❓ 常见问题

**Q: 我可以说"今天没什么问题"吗？**

A: 可以。"问题"可以是：
- 技术限制（某功能无法自动化）
- 沟通偏差（理解错需求）
- 流程优化（会议太多）
- 工具缺失（缺少 XX 工具）

**Q: 报告可以自定义模板吗？**

A: 当前版本使用固定模板，确保报告格式统一。未来版本可能支持自定义。

**Q: 如果我只说"生成日报"，没提供工作内容怎么办？**

A: AI 会主动询问你，引导你提供必要信息。
