## 📋 项目概述

基于 mcp-hello-world 模板，开发一个用于小红书语言景观分析的 MCP 工具，提供情感分析、关键词提取和周报生成功能。

## 🎯 核心需求分析

### 需求 1：接入小红书数据

**现状：**
- 现有数据源：`/home/admin/.openclaw/workspace/github_xiaohongshu/`
- 数据格式：CSV + Markdown 报告
- 数据位置：`/home/admin/.openclaw/workspace/任务成果/小红书运营周报/`

**技术方案：**
由于小红书官方 API 需要授权和审核，建议采用以下方案：

**方案 A：CSV 文件导入（推荐 ✅）**
- 优点：无需 API、零成本、快速实现
- 实现：读取 CSV 文件，解析笔记数据
- 数据字段：标题、内容、点赞、收藏、评论、发布时间

**方案 B：网页爬虫（备选）**
- 优点：数据实时
- 缺点：需要维护、可能违反平台规则
- 实现：使用 Puppeteer 或 Playwright

**方案 C：第三方 API（备选）**
- 优点：数据完整
- 缺点：需要付费、需要 API Key
- 推荐：新榜、蝉妈妈等

**决策：采用方案 A（CSV 文件导入）**
- 理由：快速 MVP、零成本、符合云端优先原则

---

### 需求 2：情感分析

**技术方案：**

**Node.js 方案（推荐 ✅）**
- 库：`sentiment` (npm)
- 优点：轻量、快速、支持中文
- 安装：`npm install sentiment`
- 使用：
```javascript
import Sentiment from 'sentiment';
const sentiment = new Sentiment();
const result = sentiment.analyze('这个产品很好用');
// result: { score: 3, comparative: 0.75, tokens: [...], words: [...], positive: [...] }
```

**Python 方案（备选）**
- 库：`SnowNLP` 或 `jieba` + `senta`
- 优点：中文支持更好
- 缺点：需要 Python 环境

**决策：采用 Node.js + sentiment 库**
- 理由：与 mcp-hello-world 技术栈一致

---

### 需求 3：关键词提取

**技术方案：**

**Node.js 方案（推荐 ✅）**
- 库：`keyword-extractor` (npm)
- 优点：简单、支持自定义语言
- 安装：`npm install keyword-extractor`
- 使用：
```javascript
import extractor from 'keyword-extractor';
const keywords = extractor.extract("这是一段文本", { language: "chinese" });
```

**中文分词方案（备选）**
- 库：`nodejieba` (npm)
- 优点：中文分词准确
- 缺点：需要词典支持

**决策：采用 keyword-extractor + nodejieba 组合**
- 理由：平衡准确性和易用性

---

### 需求 4：定时生成周报

**技术方案：**

**方案 A：MCP 工具触发（推荐 ✅）**
- 实现：提供 `generate_weekly_report` 工具
- 触发：用户通过 mcporter 调用
- 输出：Markdown 格式报告

**方案 B：Cron 定时任务（备选）**
- 实现：OpenClaw cron 定时执行
- 触发：每周五 17:00 自动执行
- 输出：保存到文件 + 发送给用户

**决策：采用方案 A + 方案 B 组合**
- MCP 工具：手动触发，灵活控制
- Cron 任务：自动执行，定期提醒

## 📦 功能设计

### 工具 1：analyze_sentiment

**功能：** 分析文本的情感倾向

**输入：**
```json
{
  "text": "这个产品很好用，推荐购买",
  "language": "zh" // 或 "en"
}
```

**输出：**
```json
{
  "score": 5,
  "comparative": 0.625,
  "sentiment": "positive",
  "confidence": 0.85,
  "analysis": "这段文本表达了积极的情感，包含 5 个正面词汇"
}
```

---

### 工具 2：extract_keywords

**功能：** 从文本中提取关键词

**输入：**
```json
{
  "text": "小红书运营数据分析报告...",
  "limit": 10,
  "language": "zh"
}
```

**输出：**
```json
{
  "keywords": ["小红书", "运营", "数据", "分析", "报告", ...],
  "count": 10,
  "categories": {
    "platform": ["小红书"],
    "action": ["运营", "分析"],
    "object": ["数据", "报告"]
  }
}
```

---

### 工具 3：list_notes

**功能：** 列出小红书笔记数据

**输入：**
```json
{
  "source": "csv", // 或 "markdown"
  "path": "/path/to/data.csv",
  "limit": 20,
  "sortBy": "likes", // 或 "time", "collects"
  "order": "desc"
}
```

**输出：**
```json
{
  "notes": [
    {
      "title": "笔记标题",
      "content": "笔记内容...",
      "likes": 1234,
      "collects": 567,
      "comments": 89,
      "time": "2026-03-15"
    }
  ],
  "total": 20,
  "source": "csv"
}
```

---

### 工具 4：generate_weekly_report

**功能：** 生成语言景观周报

**输入：**
```json
{
  "startDate": "2026-03-10",
  "endDate": "2026-03-17",
  "outputPath": "/path/to/output.md",
  "includeSentiment": true,
  "includeKeywords": true,
  "template": "default" // 或 "detailed", "simple"
}
```

**输出：**
```json
{
  "status": "success",
  "reportPath": "/path/to/output.md",
  "summary": {
    "totalNotes": 15,
    "avgSentiment": 0.65,
    "topKeywords": ["小红书", "运营", "数据"],
    "topNote": {
      "title": "爆款笔记标题",
      "likes": 5678
    }
  }
}
```
