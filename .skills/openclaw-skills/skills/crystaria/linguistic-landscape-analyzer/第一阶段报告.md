## ⚠️ 坑点记录

### 坑点 1：nodejieba 兼容性问题

**问题：** nodejieba 需要 GLIBC_2.38，系统版本不兼容

**错误信息：**
```
Error: version `GLIBC_2.38' not found
```

**解决：**
1. 卸载 nodejieba
2. 实现简单的中文分词函数 `simpleChineseTokenize()`
3. 使用字符串匹配替代专业分词

**经验：** 云端环境优先使用纯 JS 库，避免原生模块

### 坑点 2：路径计算问题

**问题：** __dirname 在 ES Modules 中需要特殊处理

**错误：** 数据文件路径计算错误

**解决：**
```javascript
const projectRoot = path.resolve(__dirname, "..");
const dataDir = path.join(projectRoot, "data");
```

**经验：** ES Modules 中使用 path.resolve() 更可靠
