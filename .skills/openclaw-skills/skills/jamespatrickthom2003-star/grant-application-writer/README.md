# Grant Application & Funding Proposal Writer

Funder-ready grant applications in minutes, not weeks. Describe your project, name the funder, get a complete application with need statement, budget, theory of change, and monitoring framework.

## What It Does

Give it a project description and funder name. Get back a professional grant application you can submit directly or adapt to a specific form:

- Executive summary with funder alignment
- Evidence-based statement of need with statistics and sources
- Project description with timeline and milestones
- Theory of change and logic model
- Detailed, line-item budget with match funding
- Monitoring and evaluation framework
- Sustainability plan, risk register, and organisational capacity statement
- Language matched to funder type (NLCF, Arts Council, UKRI, local authority, trusts, Innovate UK, and more)

## Quick Start

Full application:
```
/grant-application-writer Youth mentoring programme in Bradford, applying to National Lottery Community Fund, asking for £50k over 2 years
```

Just a budget:
```
/grant-application-writer Budget for a community garden project, 12 months, around £25k
```

Theory of change only:
```
/grant-application-writer Theory of change for a digital skills training programme for over-65s
```

Adapt existing content for a different funder:
```
/grant-application-writer Adapt this for Arts Council England: [paste existing application]
```

## Why It Exists

Professional grant writers charge GBP 500-2,000+ per application. A successful bid can unlock GBP 10k-1M+. Most small charities and social enterprises cannot afford a grant writer but lose out on funding because their applications lack structure, evidence, or funder-specific language.

This skill produces the same structured output a professional grant writer would deliver -- complete with budgets, logic models, risk registers, and funder-matched language.

## Who It's For

- Charities and community organisations applying for grants
- Social enterprises seeking start-up or growth funding
- Researchers preparing UKRI or Innovate UK bids
- Small businesses applying for innovation grants
- Fundraisers managing multiple applications
- Consultants supporting clients with bid writing
- Anyone who has a great project but struggles to get it onto paper

## What It Covers

| Section | What You Get |
|---------|-------------|
| Executive Summary | Organisation overview, amount, key outcomes, funder alignment |
| Statement of Need | Evidence-based problem, demographics, gap analysis, urgency |
| Project Description | Activities, methodology, partnerships, timeline |
| Theory of Change | Inputs to impact chain with narrative |
| Logic Model | Indicators, targets, data sources, frequency |
| Budget | Line-item costs, unit rates, match funding, value for money |
| M&E Framework | What, how, when, who, data protection |
| Sustainability | Post-funding plan, income generation, scaling |
| Risk Register | Risks with likelihood, impact, mitigation |
| Organisational Capacity | Track record, governance, policies, compliance |

## Funder Presets

Name the funder and the skill adapts language and emphasis automatically:

- **National Lottery Community Fund** -- "people-led", "community power", strengths-based
- **Arts Council England** -- "Let's Create", creative case for diversity
- **UKRI / Research Councils** -- impact pathways, TRL, knowledge exchange
- **Local Authority** -- place-based, prevention, cost-benefit
- **Innovate UK** -- market opportunity, scalability, IP strategy
- **Charitable Trusts** -- matched to trust objects, clear and direct
- **Corporate Foundations** -- social ROI, brand alignment, measurability
- **UKSPF / ESF** -- levelling up, skills gaps, labour market

## UK-Specific

All output uses UK English and references UK frameworks, pay scales (NJC), regulatory bodies (Charity Commission, CIC Regulator), and compliance requirements (DBS, safeguarding, GDPR, public liability insurance).

## Limitations

- Always check the funder's own guidance notes and adapt accordingly
- Does not guarantee funding success
- Does not submit applications or access funder portals
- Does not fabricate evidence -- all statistics should be verified against original sources
- Not a substitute for reading the funder's criteria
