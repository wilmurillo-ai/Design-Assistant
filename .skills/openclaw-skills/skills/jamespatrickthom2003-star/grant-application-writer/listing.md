# Agent37 Listing -- Grant Application & Funding Proposal Writer

## Skill Name
Grant Application & Funding Proposal Writer

## Tagline
Funder-ready grant applications in minutes. Describe your project, name the funder, get a complete bid with need statement, budget, theory of change, and monitoring framework -- ready to submit.

## Category
Fundraising & Non-Profit

## Description

**The problem:** Grant applications take days or weeks to write. Professional grant writers charge GBP 500-2,000+ per bid. Small charities, social enterprises, and community groups lose out on funding they deserve because they cannot afford specialist bid writers -- or because their applications lack the structure, evidence, and funder-specific language that panels expect.

**What this skill does:** Describe your project in plain English. Name the funder if you know it. Get back a complete, professional grant application:

- Executive summary with funder alignment and key outcomes
- Evidence-based statement of need with statistics, sources, and gap analysis
- Project description with activities, partnerships, and timeline
- Theory of change: inputs to impact, with narrative explanation
- Logic model with SMART indicators, targets, and data sources
- Detailed budget with unit costs, match funding, and value-for-money notes
- Monitoring and evaluation framework with data collection tools and GDPR compliance
- Sustainability plan: what happens when the money runs out
- Risk register with likelihood, impact, and mitigation for every risk
- Organisational capacity statement with governance, policies, and track record

**Funder-matched language:** Name the funder and the skill adjusts automatically. Built-in presets for National Lottery Community Fund, Arts Council England, UKRI, Innovate UK, local authorities, charitable trusts, corporate foundations, and UKSPF. Each preset mirrors the funder's own strategy language and assessment criteria.

**UK-specific:** UK English, NJC pay scales, Charity Commission compliance, DBS references, safeguarding, GDPR, public liability insurance, Equality Impact Assessments -- all handled.

**Why this saves money:** A successful GBP 50,000 grant application costs GBP 1,000-2,000 to have written professionally. This skill produces equivalent output for a fraction of the cost, and you can generate multiple versions for different funders from the same project brief.

## Example Prompts
- "Youth mentoring programme in Bradford, applying to National Lottery Community Fund, GBP 50k over 2 years"
- "Community garden project, 12 months, GBP 25k from a local charitable trust"
- "Digital skills training for over-65s, Arts Council or NLCF, around GBP 80k"
- "Just the budget for a 3-year research project on homelessness, UKRI funding"
- "Theory of change for an employment support programme for refugees"
- "Adapt this application for Innovate UK: [paste existing content]"
