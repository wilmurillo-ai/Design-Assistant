# Meeting Notes to Actions

Paste your messy meeting notes. Get a structured summary, action items with owners and deadlines, individual follow-up emails for every attendee, and a ready-made agenda for the next meeting.

## What It Does

Takes any form of meeting notes -- raw bullets, Zoom transcripts, voice memo dumps, Slack threads, scribbled shorthand -- and produces five outputs:

1. **Meeting summary** -- key decisions, discussion points, open questions
2. **Action items table** -- specific actions, owners, deadlines, priorities
3. **Follow-up emails** -- one personalised email per attendee, ready to send
4. **Next meeting agenda** -- auto-generated from open items and actions
5. **Project status update** -- on track / at risk / blocked, with context

## Quick Start

```
/meeting-notes-actions just had a call with Sarah and Tom about the website redesign.
Sarah to send updated wireframes by Friday. Tom handling the CMS migration, needs API
access from DevOps. We agreed to push launch from March to April. Next sync in 2 weeks.
```

```
/meeting-notes-actions standup -- paste your standup notes
```

```
/meeting-notes-actions actions only -- paste notes when you just need the action items
```

## Interaction Modes

- **Quick process** -- paste notes, get everything (default)
- **Meeting type preset** -- "client call", "standup", "board meeting", "1:1", "sprint retro", "sales call", "interview"
- **Follow-up only** -- just the emails
- **Actions only** -- just the action items table
- **Status update** -- just the project status

## Meeting Type Presets

Each preset adjusts the focus and adds type-specific output:

| Type | Extra Output |
|---|---|
| Client call | Client-facing summary email (no internal jargon) |
| Standup / scrum | Sprint board update format |
| Board meeting | Formal board minutes |
| 1:1 | Private notes format |
| Sprint retro | Retro summary with themes |
| Sales call | CRM update notes |
| Interview | Candidate scorecard |

## Why It Exists

Every consultant, project manager, and account manager has meetings. Otter.ai and Fireflies cost GBP 10-20/month and require integrations. This skill works from any pasted notes -- raw, messy, or transcribed -- and produces structured output instantly. No subscriptions, no integrations, no setup.

## Who It's For

- Freelancers and consultants who need to send professional follow-ups after client calls
- Project managers drowning in meeting notes and action tracking
- Account managers juggling multiple client relationships
- Team leads running standups, retros, and 1:1s
- Anyone who leaves meetings thinking "I should really send a follow-up" and never does

## What It Accepts

- Raw typed notes (bullet points, fragments, shorthand)
- Meeting transcripts (Zoom, Teams, Google Meet, Otter.ai, Fireflies)
- Voice memo transcriptions
- Slack or Teams thread summaries
- Descriptions of handwritten notes

## Note

This skill processes meeting notes you provide. It does not record, transcribe, or access meetings directly. Ensure you have permission to share meeting content.
