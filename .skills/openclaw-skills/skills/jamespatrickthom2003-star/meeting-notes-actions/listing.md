# Agent37 Listing -- Meeting Notes to Actions

## Skill Name
Meeting Notes to Actions

## Tagline
Paste messy meeting notes, get structured summaries, action items, follow-up emails, and next meeting agendas -- ready to send in seconds.

## Category
Productivity

## Description

**The problem:** Every professional has meetings. Every meeting produces notes. Those notes sit in a notebook or a Google Doc and slowly rot. Actions get forgotten. Follow-ups never get sent. The next meeting starts with "sorry, what did we agree last time?"

Transcription tools like Otter.ai and Fireflies cost GBP 10-20/month, require integrations, and still leave you with a wall of text. You still have to extract the actions yourself. You still have to write the follow-up emails yourself.

**What this skill does:** Paste any meeting notes -- raw bullets, full transcripts, voice memo dumps, Slack threads, even descriptions of your handwritten scribbles. Get back:

- **Structured meeting summary** with key decisions, discussion points, and open questions
- **Action items table** with specific actions, owners, deadlines, and priorities
- **Individual follow-up emails** for every attendee -- personalised, ready to copy and send
- **Auto-generated next meeting agenda** built from open items and outstanding actions
- **Project status update** (on track / at risk / blocked) when the meeting relates to a project

**Seven meeting type presets** adjust the focus and add type-specific output:
- **Client call** -- adds a polished client-facing summary (no internal jargon)
- **Standup / scrum** -- adds sprint board update format
- **Board meeting** -- adds formal board minutes with resolutions and votes
- **1:1** -- adds private notes format with feedback and development items
- **Sprint retro** -- adds retro summary with themes and improvement actions
- **Sales call** -- adds CRM update notes with needs, objections, and probability
- **Interview** -- adds candidate scorecard with ratings and recommendation

**Five modes:** Quick process (everything), meeting type preset, follow-up emails only, actions only, or status update only.

**Why it works:** Actions are specific and measurable ("Send wireframes to Sarah by Friday"), not vague ("Look into the wireframes"). Emails are written to send, not to edit. Agendas start with reviewing what was agreed last time.

## Example Prompts
- "Just had a call with Sarah and Tom about the website redesign. Sarah to send updated wireframes by Friday. Tom handling CMS migration, needs API access from DevOps. Agreed to push launch to April."
- "client call -- Met with Acme Corp, discussed Q3 deliverables, they want the dashboard by end of month, concerned about mobile responsiveness, budget approved for phase 2"
- "standup -- Alice finished the auth flow, Bob is blocked on the API integration waiting for credentials, Charlie is starting the payment page today"
- "board meeting -- Discussed annual accounts, approved budget for next FY, appointed new non-exec director Jane Smith, reviewed risk register"
- "1:1 with David -- discussed his promotion timeline, feedback on his presentation skills, he wants to lead the next client project"
- "actions only -- quick sync about the office move, need to book removals, order new furniture, set up IT in new office, update address with Companies House"
- "sprint retro -- deployment went well but testing was rushed, need better QA process, team morale is good, want to try pair programming next sprint"
- "sales call -- spoke with procurement at BigCo, they have budget but need sign-off from CTO, main objection is migration timeline, follow up with case study"
