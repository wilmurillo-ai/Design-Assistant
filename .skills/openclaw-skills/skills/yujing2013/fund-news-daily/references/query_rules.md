# 时间查询规则

## 查询类型

| 查询类型 | 触发词 | 时间范围 |
|---------|--------|---------|
| 今日新闻 | "今日"、"今天"、默认 | 当日 00:00 - 当前 |
| 过去七天 | "过去七天"、"最近一周" | 当日向前推7天 |
| 指定日期 | "YYYY年MM月DD日" | 指定日期 00:00-24:00 |
| 指定日期范围 | "YYYY年MM月DD日至..." | 初始日 00:00 到 截止日 23:59 |

## 执行流程

1. **首次使用检测依赖** → 自动安装缺失依赖
2. **解析查询类型** → 确定时间范围
3. **抓取五大平台** → 使用 Agent Browser CLI
4. **过滤与排序** → 应用筛选规则
5. **格式化输出** → 结构化文本 或 Word文档

## 输出形式

| 查询类型 | 输出形式 |
|---------|---------|
| 今日 | 文本回复 |
| 指定日期 | 文本回复 |
| 过去七天 | 文本回复 + Word文档 |
| 指定日期范围 | 文本回复 + Word文档 |
