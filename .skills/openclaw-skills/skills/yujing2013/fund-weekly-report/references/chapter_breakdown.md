# 章节话术拆分：固定部分 vs AI撰写部分

本文档明确区分每个章节中哪些是固定话术，哪些需要AI根据数据撰写。

---

## 第1章：主要市场指数周度表现回顾

### 1.1 宽基指数

**固定话术：**
```
最近一周（{start_date}-{end_date}），A股主要指数{涨跌描述}。
债券市场方面，中债-新综合财富指数本周{涨跌描述}；
商品资产方面，原油价格本周{涨跌描述}，黄金价格本周{涨跌描述}。

主要市场指数PE估值分化，{估值描述}。
```

**AI需要撰写：**
- {涨跌描述}：根据数据判断用"跌多涨少"还是"涨多跌少"
- {估值描述}：根据估值分位数据生成描述

**数据来源：**
- ❌ Excel中没有市场指数数据（需要单独提供）

行业指数方面，各申万一级行业{涨跌描述}，{领涨行业}行业{涨跌描述}，周涨幅为{return}%；

与之相对，{领跌行业}行业周跌幅超{threshold}%。

## 总结

### 固定话术（不变的部分）

1. **时间描述**："最近一周（{start_date}-{end_date}）"、"年初以来（0101-{end_date}）"
2. **连接词**："与之相对"、"另有"、"其中"
3. **结构词**："分类型来看"、"估值方面"、"ETF资金流动方面"
4. **风险提示**：完全固定

### AI需要撰写的部分

1. **数据提取**：从Excel中提取具体数值
2. **涨跌判断**：根据数据判断用"涨多跌少"还是"跌多涨少"
3. **排名识别**：识别领涨/领跌行业、基金
4. **阈值判断**：判断"超X%"、"在X%以内"
5. **数量统计**：统计涨跌数量、类型数量
6. **公司识别**：识别基金公司名称
7. **主题识别**：识别基金主题
8. **集中上报识别**：识别多家公司上报同一主题的情况

### 需要单独提供的数据

1. **市场指数数据**：第1章需要
2. **ETF资金流动数据**：第4章需要（可能）
3. **市场资讯**：第8章需要
