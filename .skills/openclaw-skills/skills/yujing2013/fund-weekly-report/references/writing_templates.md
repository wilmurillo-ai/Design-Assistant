# 基金周报话术模板

本文档定义了基金周报的专业话术风格，确保生成的文字符合分析师风格。

---

## 话术风格特点

### 1. 数据密集型
一段话包含多个数据点，信息密度高。

**示例：**
```
最近一周（0309-0313），A股主要指数跌多涨少，创业板指和沪深300指数周涨幅分别为2.51%和0.19%，中证1000、中证500和科创50全周分别下跌0.48%、1.44%和2.88%；港股和美股主要指数表现分化，恒生指数下跌1.13%，恒生科技指数上涨0.62%；标普500指数和纳斯达克100指数分别下跌1.60%、1.06%。
```

### 2. 对比结构
用"与之相对"、"另有"等连接正反两面。

**示例：**
```
行业指数方面，各申万一级行业跌多涨少，煤炭行业大涨5.03%，另有电力设备和建筑装饰行业周涨幅分别为4.55%和4.12%；与之相对，综合、石油石化、国防军工行业周跌幅超4%。
```

### 3. 具体名称
提到具体基金名称、基金公司、基金经理。

**示例：**
```
最近一周（0309-0313），资金积极流入中游制造/新能源主题相关标的，华夏中证电网设备主题ETF全周净申购额达31.40亿元, 易方达国证新能源电池ETF、华泰柏瑞中证全指电力公用事业ETF全周净申购均超过10亿元。
```

### 4. 专业术语
使用金融专业术语。

**术语列表：**
- PE-TTM：市盈率（滚动）
- 分位水平：历史分位数
- 净流入/流出：资金流向
- 超额收益：相对基准的收益
- 中位数：50%分位数
- 首尾差异：最高与最低的差距

## 话术模板

### 模板1：主动权益基金周度表现

**标题：** 收益分布：各类主动权益基金周收益中位数收负，尾部产品周跌幅超9%

**近一周话术：**
```
最近一周（{start_date}-{end_date}），普通股票/偏股混合/灵活配置/平衡混合基金周收益率中位数分别为{median_1}/{median_2}/{median_3}/{median_4}；首尾基金收益差异较大，头部绩优产品周收益超{max_return}，尾部产品周跌幅超过{min_return}。
```

**年初以来话术：**
```
年初以来（0101-{end_date}），普通股票/偏股混合/灵活配置/平衡混合型基金收益率中位数分别为{ytd_median_1}/{ytd_median_2}/{ytd_median_3}/{ytd_median_4}，正收益基金数量占比超五成；头部绩优产品累计收益超过{ytd_max_return}。
```

### 模板2：行业主题基金表现

**标题：** 行业主题基金：各类主动权益基金跌多涨少，新能源主题基金平均周涨幅1.27%

**近一周话术：**
```
分类型来看，最近一周（{start_date}-{end_date}）各类主动权益基金跌多涨少，{top_industry_1}、{top_industry_2}主题基金周收益均值分别为{return_1}和{return_2}，其余行业主题基金周收益均值收负；其中，{bottom_industry}主题基金平均周跌幅最大，为{return_bottom}，另有{other_industry_1}和{other_industry_2}主题基金平均周跌幅超{threshold}；全市场基金（主动管理）和主动量化基金周收益均值分别为{return_active}和{return_quant}。
```

**年初以来话术：**
```
年初以来（0101-{end_date}），除{negative_industry_1}、{negative_industry_2}、{negative_industry_3}、{negative_industry_4}主题基金平均收益率收负外，其余基金实现整体上涨，{top_industry}主题基金领涨，平均涨幅高达{return_top}。
```

### 模板3：固定收益基金表现

**标题：** 收益分布：纯债基金净值稳步上涨，各类含权债基周收益中位数全部收负

**近一周话术：**
```
最近一周（{start_date}-{end_date}），纯债基金净值稳步上涨，短期纯债和中长期纯债型基金周收益率中位数分别为{return_short}和{return_long}；各类含权债基周收益率中位数全部收负，一级债基、二级债基和偏债混合型基金周收益率中位数分别为{return_level_1}、{return_level_2}和{return_level_3}。
```

**年初以来话术：**
```
年初以来（0101-{end_date}），短期纯债/中长期纯债/一级债基的收益率中位数分别为{ytd_return_1}/{ytd_return_2}/{ytd_return_3}，中位收益集体跑赢货币基金；二级债基和偏债混合基金年初以来收益率中位数分别为{ytd_return_4}和{ytd_return_5}，头部绩优产品收益超{ytd_max}。
```

**标题：** 被动指基：新能源主题指基平均上涨5.14%，油价震荡市场资金大笔赎回油气主题标的

```

**ETF资金流动话术：**

```

## 连接词使用规则

### 对比连接词

| 场景 | 连接词 |
|-----|--------|
| 正反对比 | 与之相对 |
| 补充说明 | 另有 |
| 递进说明 | 此外 |
| 具体举例 | 例如、如 |
| 总结概括 | 整体来看 |

### 数据连接词

| 场景 | 连接词 |
|-----|--------|
| 并列数据 | 分别为、分别为...和... |
| 超过阈值 | 超、超过、高达 |
| 范围限定 | 在...以内、介于...之间 |
| 变化趋势 | 上涨、下跌、收负、收正 |

| 数据范围 | 格式示例 |
|---------|---------|
| 正数 | 5.14%、1.27% |
| 负数 | -5.91%、-0.59% |

| 数据范围 | 格式示例 |

| 场景 | 格式示例 |

## 话术生成原则

1. **数据优先**：每句话必须包含具体数据
2. **对比鲜明**：用"与之相对"突出正反两面
3. **信息密集**：一段话包含多个数据点
4. **专业术语**：使用金融专业术语
5. **具体名称**：提到具体基金名称和公司
6. **结构清晰**：先总后分，先涨后跌
