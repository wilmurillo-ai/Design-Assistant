# 数据处理逻辑

## 市场状态判断逻辑

### 行业主题基金/主动权益基金标题判断（2.2章节）

| 条件 | 标题用词 |
|-----|---------|
| 所有类型收益均为负 | "普遍下跌" |
| 所有类型收益均为正 | "普遍上涨" |
| 下跌数量 > 上涨数量 × 1.5 | "跌多涨少" |
| 上涨数量 > 下跌数量 × 1.5 | "涨多跌少" |
| 其他情况 | "涨跌互现" |

### FOF基金收益排序逻辑（5.1章节）

当FOF基金集体上涨或下跌时，描述中产品类型按**收益率绝对值从大到小**排序：

```
涨幅示例：目标风险-积极/普通FOF-偏股/目标日期-[2045年,2060年]基金平均涨幅分别为2.15%/1.89%/1.65%
跌幅示例：目标风险-积极/普通FOF-偏股/目标日期-[2045年,2060年]基金平均跌幅分别为1.71%/1.59%/1.34%
```

**规则说明：**
- 无论涨幅还是跌幅，都按绝对值降序排列，数值大的在前
- 高权益仓位FOF：普通FOF-偏股、目标日期-[2045年,2060年]、目标风险-积极
- 低权益仓位FOF：普通FOF-偏债、目标风险-稳健、目标日期-[2025年,2035年)

## 收益率描述规则

| 收益率范围 | 描述词 |
|-----------|--------|
| > 5% | 大涨、飙升 |
| 1% ~ 5% | 上涨、涨幅 |
| 0% ~ 1% | 微涨、小幅上涨 |
| -1% ~ 0% | 微跌、小幅下跌 |
| -5% ~ -1% | 下跌、跌幅 |
| < -5% | 大跌、重挫 |

## 排名描述

| 排名 | 描述 |
|-----|------|
| 第1名 | 领涨、涨幅最大 |
| 前3名 | 涨幅居前 |
| 倒数第1 | 领跌、跌幅最大 |
| 后3名 | 跌幅居前 |

## 占比描述

| 占比范围 | 描述 |
|---------|------|
| > 80% | 绝大多数 |
| 60% ~ 80% | 超六成 |
| 50% ~ 60% | 超五成 |
| 40% ~ 50% | 近半数 |
| 20% ~ 40% | 约1/3 |
| < 20% | 少数 |
