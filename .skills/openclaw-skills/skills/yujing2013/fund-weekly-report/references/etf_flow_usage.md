# ETF资金流动数据使用说明

本文档说明如何使用ETF资金流动跟踪Excel文件。

---

## 输入文件

**文件名：** ETF资金流动跟踪.xlsx

**必需Sheet：**
- 板块结果展示
- 个基结果展示
- 已上市ETF
- 核心宽基赛道
- 热门行业主题

---

## 数据提取

### 1. 板块资金流动

**Sheet：** 板块结果展示

**提取字段：**
- 二级分类 / Unnamed: 5 → 板块名称
- 资金流入总额（亿元）→ 板块资金流入
- 基金数量 → 板块基金数量
- 平均涨跌幅 → 板块平均涨跌幅

**示例输出：**
```json
{
  "板块": "大市值指数",
  "资金流入总额": -125.17,
  "基金数量": 131,
  "平均涨跌幅": 0.001795
}
```

---

### 2. 净申购TOP

**Sheet：** 个基结果展示

**提取字段：**
- 基金代码
- 基金名称
- 资金流入规模（亿元）
- 最新规模（亿元）
- 最新涨跌幅

**示例输出：**
```json
{
  "基金代码": "159326.OF",
  "基金名称": "华夏中证电网设备主题ETF",
  "资金流入规模": 31.40,
  "最新规模": 336.04,
  "最新涨跌幅": -0.027925
}
```

---

### 3. 净赎回TOP

**Sheet：** 已上市ETF

**提取字段：**
- 证券代码 → 基金代码
- 证券简称 → 基金名称
- 基金管理人简称 → 基金公司
- 周净申赎额 → 净申赎额（筛选负值）
- 基金规模 → 基金规模

**示例输出：**
```json
{
  "基金代码": "510300.OF",
  "基金名称": "华泰柏瑞沪深300ETF",
  "基金公司": "华泰柏瑞基金",
  "净申赎额": -29.24,
  "基金规模": 2058.03
}
```

---

### 4. 核心宽基ETF资金流动

**Sheet：** 核心宽基赛道

**提取字段：**
- 跟踪指数
- 指数类型
- 基金数量
- 合计规模（亿元）
- 净申购额（亿元）
- 代表产品
- 基金简称
- 净申购额（亿元）.1 → 代表产品净申购额

**示例输出：**
```json
{
  "跟踪指数": "沪深300指数",
  "指数类型": "大市值",
  "基金数量": 30,
  "合计规模": 5725.11,
  "净申购额": -54.17,
  "代表产品": "510300.OF",
  "基金简称": "华泰柏瑞300ETF",
  "代表产品净申购额": -29.24
}
```

---

## 话术模板

### 模板1：板块资金流动

```
ETF资金流动方面，跟踪沪深300/中证500/中证1000/创业板指/科创50等核心宽基指数的头部ETF标的{遭遇大额赎回/获得大额申购}，{板块1}/{板块2}/{板块3}板块全周净{流出/流入}额分别高达{金额1}亿元/{金额2}亿元/{金额3}亿元；与之相对，市场资金积极{申购/赎回}{主题1}、{主题2}、{主题3}等{热门/冷门}主题对应标的，{板块A}/{板块B}/{板块C}ETF板块全周分别净{流入/流出}{金额A}亿元/{金额B}亿元/{金额C}亿元。
```

---

### 模板2：净申购TOP

```
最近一周（{start_date}-{end_date}），{公司1}、{公司2}基金旗下{主题}主题ETF产品获得大额申购，周净申购额分别为{金额1}亿元、{金额2}亿元；{公司3}、{公司4}基金旗下{主题2}主题ETF净申购额分别为{金额3}亿元、{金额4}亿元；另有{主题3}、{主题4}、{主题5}等主题ETF净申购额超过{threshold}亿元。
```

---

### 模板3：净赎回TOP

```
净流出方面，{ETF名称1}全周净赎回额均超百亿元，合计净流出{金额1}亿元，另有{ETF名称2}、{ETF名称3}、{ETF名称4}净赎回额也均超过百亿元，十大核心宽基ETF标的周净赎回额高达{总金额}亿元。结合上述标的属于大型机构投资者重仓布局品种的背景，不排除此类行为是机构端为平抑市场过热情绪、助力市场降温，以呵护市场平稳上涨的逆周期调节手段。
```

## 使用示例

```python
from read_excel import extract_etf_flow_data

# 提取ETF资金流动数据
etf_data = extract_etf_flow_data("ETF资金流动跟踪.xlsx")

# 获取板块资金流动
sector_flow = etf_data['sector_flow']

# 获取净申购TOP
top_inflow = etf_data['top_inflow'][:10]

# 获取净赎回TOP
top_outflow = etf_data['top_outflow'][:10]

# 获取核心宽基ETF资金流动
core_index_flow = etf_data['core_index_flow']

# 获取热门行业主题资金流动
hot_theme_flow = etf_data['hot_theme_flow']
```

## 注意事项

1. **日期范围**：ETF资金流动Excel中的日期范围可能与基金周度收益Excel不一致，需要注意匹配
2. **数据单位**：资金流入/流出单位为亿元
3. **正负号**：正值表示净申购（流入），负值表示净赎回（流出）
4. **排序**：净申购TOP按资金流入规模降序排列，净赎回TOP按净申赎额升序排列（赎回最多的排前面）
