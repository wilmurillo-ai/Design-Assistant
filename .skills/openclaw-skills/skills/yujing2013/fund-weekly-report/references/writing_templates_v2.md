# 基金周报话术模板 - 改进版

本文档基于与用户周报对比分析，更新话术模板。

## 一、标题生成规则

### 规则：标题应包含关键数据点和亮点

**错误示例：**
```
2.1 收益分布：各类主动权益基金周收益表现
```

**正确示例：**
```
2.1 收益分布：普通股票/偏股混合基金中位收益超1%，首尾基金收益差异巨大
```

**标题生成公式：**
```
章节标题 = 章节主题 + 关键数据点 + 亮点描述
```

**示例：**
| 章节 | 主题 | 关键数据点 | 亮点描述 | 最终标题 |
|-----|------|----------|---------|---------|
| 2.1 | 收益分布 | 普通股票/偏股混合基金中位收益超1% | 首尾基金收益差异巨大 | 收益分布：普通股票/偏股混合基金中位收益超1%，首尾基金收益差异巨大 |
| 4.1 | 被动指基 | 核心宽基ETF全周净流出近2000亿元 | 市场资金流入活跃行业主题标的 | 被动指基：核心宽基ETF全周净流出近2000亿元，市场资金流入活跃行业主题标的 |
| 7.1 | 基金成立 | 新成立基金23只 | 合计募资192.94亿元 | 基金成立：新成立基金23只，合计募资192.94亿元 |
| 7.2 | 基金发行 | 富国基金新发行创新产品 | 中证智选船舶产业ETF | 基金发行：富国基金新发行创新产品中证智选船舶产业ETF |

## 二、话术风格改进

### 2.1 首尾差异描述

**错误：**
```
首尾基金收益差异较大，头部绩优产品周收益超15.87%，尾部产品周跌幅超-7.05%。
```

**正确：**
```
首尾基金收益差异巨大，头部绩优产品周收益超10%，尾部产品周跌幅近5%。
```

**规则：**
1. 用"巨大"而不是"较大"
2. 用"近X%"而不是"超X%"（更保守）
3. 数据要保守，不要过于激进
4. 正数用"超X%"，负数用"近X%"或"超X%"

---

### 2.2 正收益占比描述

**错误：**
```
正收益基金数量占比超五成
```

**正确：**
```
正收益基金数量占比约九成
```

**规则：**
1. 准确计算正收益基金占比
2. 用"约X成"而不是"超五成"
3. 数据要准确，不能模糊

---

### 2.3 年初以来收益描述

**错误：**
```
头部绩优产品累计收益超过35.04%
```

**正确：**
```
头部绩优产品累计收益在30%上下
```

**规则：**
1. 用"在X%上下"而不是"超过X%"
2. 数据要保守，用区间而不是精确值

---

### 2.4 行业主题基金描述

**错误：**
```
TMT主题基金领涨，平均涨幅高达11.92%。
```

**正确：**
```
TMT主题基金强势领涨，平均涨幅高达11.92%，医药生物和军工主题基金平均涨幅在10%左右；金融地产主题基金表现落后，平均涨幅仅为1.00%。
```

**规则：**
1. 用"强势领涨"而不是"领涨"
2. 补充涨幅居前的行业（如"医药生物和军工主题基金平均涨幅在10%左右"）
3. 提到表现落后的行业（如"金融地产主题基金表现落后"）
4. 用"表现落后"而不是"领跌"（更中性）

## 三、ETF资金流动（重大缺失）

### 3.1 数据要求

**需要的数据：**
1. 核心宽基ETF净流入/流出金额
2. 板块净流入/流出金额
3. 具体ETF名称和净申购额
4. 具体基金公司名称

### 3.2 话术模板

**段落1：总体流动**
```
ETF资金流动方面，跟踪沪深300/中证500/中证1000/创业板指/科创50等核心宽基指数的头部ETF标的{遭遇大额赎回/获得大额申购}，{板块1}/{板块2}/{板块3}板块全周净{流出/流入}额分别高达{金额1}亿元/{金额2}亿元/{金额3}亿元；与之相对，市场资金积极{申购/赎回}{主题1}、{主题2}、{主题3}等{热门/冷门}主题对应标的，{板块A}/{板块B}/{板块C}ETF板块全周分别净{流入/流出}{金额A}亿元/{金额B}亿元/{金额C}亿元。
```

**段落2：净申购TOP**
```
最近一周（{start_date}-{end_date}），{公司1}、{公司2}基金旗下{主题}主题ETF产品获得大额申购，周净申购额分别为{金额1}亿元、{金额2}亿元；{公司3}、{公司4}基金旗下{主题2}主题ETF净申购额分别为{金额3}亿元、{金额4}亿元；另有{主题3}、{主题4}、{主题5}等主题ETF净申购额超过{threshold}亿元。
```

**段落3：净流出TOP**
```
净流出方面，{ETF名称1}全周净赎回额均超百亿元，合计净流出{金额1}亿元，另有{ETF名称2}、{ETF名称3}、{ETF名称4}净赎回额也均超过百亿元，十大核心宽基ETF标的周净赎回额高达{总金额}亿元。结合上述标的属于大型机构投资者重仓布局品种的背景，不排除此类行为是机构端为平抑市场过热情绪、助力市场降温，以呵护市场平稳上涨的逆周期调节手段。
```

### 3.3 数据来源

**当前状态：** ❌ Excel中没有ETF资金流动数据

**需要：** 单独提供ETF资金流动数据

## 四、新基金细节

### 4.1 基金成立

**错误：**
```
本周新成立12只被动指数型基金，主动权益基金发行保持较高热度。
```

**正确：**
```
本周指数型基金成立数量最多，为9只，华宝和景顺长城旗下中证全指电力公用事业ETF募资规模分别为7.19亿元和16.67亿元，大成和南方基金各成立一只中证电池主题ETF。

主动权益/含权债基/量化策略/FOF/其他类型基金新成立数量分别为7只/2只/2只/2只/1只。易方达平衡精选（偏股混合）募资最高，为34.08亿元；路博迈中证A股增强和易方达中证800增强分别募资12.62亿元和21.50亿元。
```

**规则：**
1. 提到募资规模最大的基金名称和金额
2. 提到具体基金公司和基金名称
3. 按类型统计数量
4. 提到募资规模数据

### 4.2 基金发行

**错误：**
```
最近一周全市场新发行基金25只...
```

**正确：**
```
最近一周（0112-0116）全市场新发行基金25只，主动权益/含权债基/指数/FOF基金新发数量分别为7只/5只/12只/1只。

指数型基金方面，泰康基金和景顺长城各新发行一只中证有色金属矿业主题ETF，富国基金新发行一只中证智选船舶产业ETF，为市场首只船舶主题创新ETF基金。
```

**规则：**
1. 提到具体基金公司和基金名称
2. 提到创新产品（如"为市场首只船舶主题创新ETF基金"）
3. 按类型统计数量

### 4.3 基金申报

**错误：**
```
最近一周全市场新申报基金共43只，包括主动权益基金15只、固定收益基金1只，指数型基金24只及FOF基金3只。
```

**正确：**
```
最近一周（0112-0116）全市场新申报基金共43只，包括主动权益基金15只、固定收益基金1只，指数型基金24只及FOF基金3只。

主动权益基金方面，南方、广发、汇添富、海富通、东方红资管等公司申报行业主题基金，覆盖科技、港股医药、A股/港股周期等方向。

指数型基金方面，中银、工银瑞信、兴业基金各申报一只有色金属主题ETF基金或场外指数基金，华夏、华安、兴业基金各申报一只跟踪中证电池主题指数的ETF基金或场外指数基金，永赢基金新申报国证价值100ETF基金。
```

**规则：**
1. 识别多家公司集中上报同一主题的情况
2. 提到具体基金公司和主题
3. 按类型分类描述

## 五、QDII基金细节

**错误：**
```
最近一周主动QDII基金平均收益为5.73%，油气主题QDII基金表现亮眼。
```

**正确：**
```
最近一周（0109-0115）主动QDII基金平均收益为1.87%，华夏新时代周涨幅达9.90%，国富全球科技互联、万家全球成长、富国全球科技互联网、易方达全球配置近一周涨幅也均超过5%。

年初以来（0101-0115）主动QDII基金平均收益达4.52%，工银新经济、华夏新时代等头部产品年初以来收益均超过12%。
```

**规则：**
1. 提到涨幅最大的具体基金名称和涨幅
2. 提到头部产品名称（多个）
3. 用"达X%"而不是"为X%"

## 六、固定收益基金数据

**错误：**
```
各类含权债基周收益率中位数全部收负
```

**正确：**
```
各类含权债基周收益中位数集体收正
```

**规则：**
1. 准确判断正负
2. 用"集体收正"而不是"全部收正"

---

## 七、数据准确性检查

### 需要检查的数据：

1. **正收益基金占比** - 不能用"超五成"，要准确计算
2. **固定收益基金正负** - 检查数据提取逻辑
3. **日期范围** - 确保各章节日期一致
4. **平均收益** - 确保计算准确

## 八、话术改进总结

| 改进点 | 错误 | 正确 |
|-------|------|------|
| 差异程度 | 较大 | 巨大 |
| 数据表达 | 超15.87% | 超10% / 在30%上下 |
| 占比表达 | 超五成 | 约九成 |
| 领涨描述 | 领涨 | 强势领涨 |
| 表现落后 | 领跌 | 表现落后 |
| 收正/收负 | 全部收负 | 集体收正 |
| 平均收益 | 为5.73% | 达4.52% |

## 九、优先级改进

### 高优先级（必须改进）
1. ✅ 标题包含关键数据点
2. ✅ 话术准确性（"巨大"vs"较大"，数据保守）
3. ✅ 新基金细节（具体基金名称和募资规模）
4. ❌ ETF资金流动数据（需要数据源）

### 中优先级（应该改进）
1. ✅ 行业主题基金补充（涨幅居前和表现落后的行业）
2. ✅ QDII基金细节（具体基金名称）
3. ✅ 新基金申报集中上报（识别集中上报情况）

### 低优先级（可以改进）
1. ✅ 创新产品亮点（"市场首只XXX"）
2. ✅ 分析解读（ETF资金流动的解读）
