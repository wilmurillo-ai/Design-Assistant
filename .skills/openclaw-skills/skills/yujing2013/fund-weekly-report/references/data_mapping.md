## 数据类型转换规则

### 百分比格式

Excel中的小数值（如0.0514）需要转换为百分比显示（如5.14%）：

```python
def format_percent(value):
    return f"{value * 100:.2f}%"
```

### 日期格式

Excel中的日期需要统一格式：

- 近一周日期范围：`MMDD-MMDD`（如 `0309-0313`）
- 年初以来日期范围：`0101-MMDD`（如 `0101-0313`）
- 基金成立日：`YYYY-MM-DD`

### 数字格式

发行规模保留2位小数：

```python
def format_number(value, decimals=2):
    return f"{value:.{decimals}f}"
```

## 数据验证规则

### 必填字段检查

- 基金代码：不能为空
- 基金名称：不能为空
- 收益率：不能为空

### 异常值处理

- 收益率超过±100%：标记为异常
- 发行规模为负：标记为异常
- 日期格式错误：跳过该条记录

## 文字生成规则

### 收益率描述

根据收益率正负选择描述词：

- 正收益：上涨、领涨、涨幅
- 负收益：下跌、收负、跌幅
- 接近零：持平、微涨、微跌

### 排名描述

根据排名选择描述：

- 第1名：领涨、涨幅最大
- 前3名：涨幅居前
- 倒数第1：领跌、跌幅最大
- 后3名：跌幅居前

### 统计描述

根据统计数据生成描述：

- 中位数：收益率中位数
- 最高值：头部绩优产品
- 正收益占比：超五成、不足三成等
