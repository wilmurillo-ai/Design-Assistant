# 基金周报话术模板 - 学习版（基于真实周报）

本文档基于真实周报模板学习，总结关键话术模式。

## 一、标题生成规则

### 规则：标题必须包含关键数据点和亮点

**错误示例：**
```
2.1 收益分布：各类主动权益基金周收益表现
```

**正确示例：**
```
2.1 收益分布：普通股票/偏股混合基金中位收益超1%，首尾基金收益差异巨大
```

**标题公式：**
```
章节主题 + 关键数据点 + 亮点描述
```

**更多示例：**

| 章节 | 标题 |
|-----|------|
| 2.1 | 收益分布：普通股票/偏股混合基金中位收益超1%，首尾基金收益差异巨大 |
| 4.1 | 被动指基：核心宽基ETF全周净流出近2000亿元，市场资金流入活跃行业主题标的 |
| 7.1 | 基金成立：新成立基金23只，合计募资192.94亿元 |
| 7.2 | 基金发行：富国基金新发行创新产品中证智选船舶产业ETF |

## 二、主动权益基金话术

### 2.0 标题生成规则

#### 2.1 收益分布标题规则

**判断逻辑：**
1. **全部同号**（4个中位数全正或全负）→ `各类主动权益基金周收益中位数收正/收负`
2. **多数为正**（正数多于负数）→ `普通股票/偏股混合/灵活配置基金周收益中位数均为正值`（列出为正的基金类型）
3. **多数为负**（负数多于正数）→ `普通股票/偏股混合/灵活配置基金周收益中位数均为负值`（列出为负的基金类型）
4. **正负各半** → `各类主动权益基金周收益中位数涨跌互现`

**尾部跌幅阈值：** 超过5%时才添加到标题中

**示例：**
```
# 全部为正
2.1 收益分布：各类主动权益基金周收益中位数收正，尾部产品周跌幅超过5%

# 全部为负
2.1 收益分布：各类主动权益基金周收益中位数收负，尾部产品周跌幅超过10%

# 多数为正（比如3正1负）
2.1 收益分布：普通股票/偏股混合/灵活配置基金周收益中位数均为正值

# 多数为负（比如3负1正）
2.1 收益分布：普通股票/偏股混合/灵活配置基金周收益中位数均为负值，尾部产品周跌幅超过8%
```

#### 2.2 行业主题基金标题规则

**先判断涨跌情况，再加领涨行业：**
- 跌多涨少（下跌行业数 > 上涨行业数 * 1.5）：`各类主动权益基金跌多涨少，xxx主题基金平均上涨xx%`
- 涨多跌少（上涨行业数 > 下跌行业数 * 1.5）：`各类主动权益基金涨多跌少，xxx主题基金平均上涨xx%`
- 涨跌互现：`各类主动权益基金涨跌互现，xxx主题基金平均上涨xx%`
- 全部下跌：`各类主动权益基金普遍下跌，xxx主题基金跌幅相对较小`

**示例：**
```
2.2 行业主题基金：各类主动权益基金涨多跌少，周期主题基金平均上涨6.25%
2.2 行业主题基金：各类主动权益基金跌多涨少，金融地产主题基金平均下跌2.41%
```

---

### 2.1 近一周收益描述

**模板：**
```
最近一周（{start_date}-{end_date}），普通股票/偏股混合/灵活配置/平衡混合基金周收益率中位数分别为{median1}/{median2}/{median3}/{median4}；首尾基金收益差异巨大，头部绩优产品周收益超{max}%，尾部产品周跌幅近{min}%。
```

**关键点：**
- ✅ 用"差异巨大"而不是"差异较大"
- ✅ 用"近X%"而不是"超X%"（更保守）
- ✅ 用"头部绩优产品"而不是"头部产品"
- ✅ 用"尾部产品"而不是"尾部基金"

---

### 2.2 年初以来收益描述

**模板：**
```
年初以来（0101-{end_date}），普通股票/偏股混合/灵活配置/平衡混合基金收益率中位数分别为{ytd_median1}/{ytd_median2}/{ytd_median3}/{ytd_median4}，正收益基金数量占比约{ratio}成；头部绩优产品累计收益在{ytd_max}%上下。
```

**关键点：**
- ✅ 用"约X成"而不是"超五成"（更准确）
- ✅ 用"在X%上下"而不是"超过X%"（更保守）
- ✅ 用"累计收益"而不是"累计收益超过"

---

### 2.3 行业主题基金描述

**模板：**
```
分类型来看，最近一周（{start_date}-{end_date}）各类主动权益基金{涨跌情况}，{领涨行业}主题基金领涨，平均周涨幅达{return}%，另有{行业2}和{行业3}主题基金平均涨幅超{threshold}%，{表现落后行业}和{表现落后行业2}主题基金平均跌幅超{threshold2}%；全市场基金（主动管理）和主动量化基金周收益均值分别为{return_active}和{return_quant}。
```

**涨跌情况选择：**
- 跌多涨少：`跌多涨少`
- 涨多跌少：`涨多跌少`
- 涨跌互现：`涨跌互现`
- 普遍下跌：`普遍下跌`

**关键点：**
- ✅ 标题和正文的涨跌描述必须一致
- ✅ 用"领涨"而不是"涨幅最大"
- ✅ 补充涨幅居前的行业："另有科技和周期主题基金平均涨幅超2%"
- ✅ 提到表现落后的行业："金融地产和军工主题基金平均跌幅超1%"
- ✅ 全市场基金和主动量化基金收益从数据中提取，不写死

**年初以来模板：**
```
年初以来（0101-{end_date}），各类型基金均实现整体上涨，{领涨行业}主题基金强势领涨，平均涨幅高达{return}%，{涨幅居前行业1}和{涨幅居前行业2}主题基金平均涨幅在{threshold}%左右；{表现落后行业}主题基金表现落后，平均涨幅仅为{return_bottom}%。
```

**关键点：**
- ✅ 用"强势领涨"而不是"领涨"
- ✅ 补充涨幅居前的行业："医药生物和军工主题基金平均涨幅在10%左右"
- ✅ 提到表现落后的行业："金融地产主题基金表现落后，平均涨幅仅为1.00%"
- ✅ 用"表现落后"而不是"领跌"（更中性）

## 三、固定收益基金话术

### 3.0 标题生成规则

#### 3.1 收益分布标题规则

**标题格式：** `收益分布：纯债基金净值{状态1}，各类含权债基周收益率中位数{状态2}`

**纯债基金状态判断：**
- 两个中位数均 > 0：`稳步上涨`（全部为正）
- 两个中位数均 < 0：`略有回调`（全部为负）
- 一正一负：`表现分化`

**含权债基状态判断：**
- 全部为负：`集体收负`
- 全部为正：`集体收正`
- 负数多于正数：`负多正少`
- 正数多于负数：`正多负少`
- 正负各半：`表现分化`

**示例：**
```
3.1 收益分布：纯债基金净值稳步上涨，各类含权债基周收益率中位数集体收正
3.1 收益分布：纯债基金净值略有回调，各类含权债基周收益率中位数负多正少
```

---

### 3.1 近一周收益描述

**模板：**
```
最近一周（{start_date}-{end_date}），纯债基金净值{状态}，短期纯债和中长期纯债型基金周收益率中位数分别为{median_short}%和{median_long}%；各类含权债基周收益率中位数{状态}，一级债基、二级债基和偏债混合型基金周收益率中位数分别为{median_primary}%、{median_secondary}%和{median_hybrid}%。
```

**关键点：**
- ✅ 纯债基金状态与标题一致
- ✅ 含权债基状态与标题一致
- ✅ 列出三个含权债基的中位数

---

### 3.2 年初以来收益描述

**模板：**
```
年初以来（0101-{end_date}），短期纯债/中长期纯债/一级债基的收益率中位数分别为{ytd_short}%/{ytd_long}%/{ytd_primary}%；二级债基和偏债混合基金年初以来收益率中位数分别为{ytd_secondary}%和{ytd_hybrid}%，中位收益集体跑赢/集体跑输货币基金，头部绩优产品收益在{top_return}%上下。
```

**跑赢跑输判断：**
- 二级债基和偏债混合基金中位数均 > 货币基金收益（约0.12%）→ `集体跑赢货币基金`
- 否则 → `集体跑输货币基金`

**头部收益阈值：**
- 二级债基或偏债混合基金中位数 > 5% → `头部绩优产品收益在15%左右`
- 否则 → `头部绩优产品收益在5%上下`

## 五、话术对比总结

| 场景 | 错误 | 正确 |

| 表现落后 | 领跌 | 表现落后 |

| 上涨描述 | 集体上涨 | 集体反弹 / 均实现正收益 |
| 收益为正 | 收益为正 | 收益收正 |

## 七、涨跌描述的多样性

### 1. 根据市场情况选择不同表达

| 市场情况 | 表达方式 | 示例 |
|---------|---------|------|
| 涨多跌少 | 涨多跌少 | "各申万一级行业涨多跌少，有色金属行业涨幅达6.43%" |
| 涨跌各半 | 涨跌互现 | "各类主动权益基金涨跌互现，TMT主题基金领涨" |
| 集体上涨 | 集体反弹 | "A股、港股主要指数集体反弹，美股持续上涨" |
| 全部上涨 | 均实现正收益 | "除消费和生物医药主题基金收益小幅收负外，各类主动权益基金均实现正收益" |
| 收益为正 | 收益收正 | "各类含权债基收益收正" |

### 2. 涨幅描述

| 涨幅 | 表达方式 |
|-----|---------|
| >5% | 大涨 / 涨幅达X% |
| 3-5% | 涨幅超X% |
| 1-3% | 上涨X% |
| 0-1% | 小幅上涨 |

### 3. 跌幅描述

| 跌幅 | 表达方式 |
|-----|---------|
| >5% | 大跌 / 跌幅达X% |
| 3-5% | 跌幅超X% |
| 1-3% | 下跌X% |
| 0-1% | 小幅收负 / 小幅下跌 |

## 八、ETF资金流动详细描述

### 1. 净流入描述

| 全部上涨 | 集体上涨 / 均实现正收益 |
| 全部下跌 | 集体下跌 |
