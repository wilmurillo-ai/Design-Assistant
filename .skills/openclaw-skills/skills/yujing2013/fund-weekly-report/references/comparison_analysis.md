本文档对比用户做的周报和我生成的周报，找出差异和需要改进的地方。

---

## 一、标题差异

### 用户的标题风格

**特点：包含关键数据点和亮点**

| 章节 | 用户标题 | 我的标题 | 差异 |
|-----|---------|---------|------|

## 二、话术差异

### 2.1 首尾差异描述

**用户：**
```
首尾基金收益差异巨大，头部绩优产品周收益超10%，尾部产品周跌幅近5%。
```

**我：**
```
首尾基金收益差异较大，头部绩优产品周收益超15.87%，尾部产品周跌幅超-7.05%。
```

**差异：**
1. ❌ "巨大" vs "较大" - 用户用词更强烈
2. ❌ "近5%" vs "超-7.05%" - 用户用"近"，我用"超"，且我的数据可能不准确
3. ❌ 用户数据更保守，我的数据可能过于激进

**改进：**
- 用"巨大"而不是"较大"
- 用"近X%"而不是"超X%"
- 数据要准确，不能太激进
