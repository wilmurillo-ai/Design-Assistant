# Research Paper Orchestrator - Subagent Definitions

This skill coordinates 10 specialized subagents:

## Agent 1: Project Manager (You - The Orchestrator)
- Overall coordination
- Quality gate enforcement
- User communication

## Agent 2: Research Planning & Topic Analysis
- Topic analysis report (2000+ words)
- Research questions (1-3)
- Keywords (30-50)
- Boolean search strings (5-10)
- Database recommendations (10-15)
- Concept map
- Scope definition

## Agent 3: Literature Discovery & Evaluation
- Source collection (50-100+ sources)
- Source evaluation matrix
- Search documentation
- Annotated bibliography

## Agent 4: Synthesis & Critical Analysis
- Literature synthesis (8000-12000 words)
- Thematic organization (5-10 themes)
- Gap analysis
- Quotation database

## Agent 5: Theoretical & Methodological Framework
- Framework section (5000-8000 words)
- Methodology (if empirical)
- Conceptual diagrams
- Concept definitions

## Agent 6: Data Analysis & Visualization
- Analysis results
- Tables and figures
- (Only if empirical research)

## Agent 7: Structure & Writing
- Complete manuscript (12000-15000 words)
- All sections
- Citations integrated

## Agent 8: Quality & Coherence
- Quality assessment (min 80/100)
- Citation verification
- Reference list correction

## Agent 9: Technical Production
- Final Word document
- Formatting
- Structure verification

## Agent 10: Citation & Visual Specialist
- Citation guide research
- Visual content sourcing

---

## Parallel Execution Points

1. **Phase 3**: Agent 3 (Literature) parallel with Agent 10 Phase 1 (Citation Research)
2. **Phase 8**: Agent 8 (Quality) parallel with Agent 10 Phase 2 (Visual Sourcing)

All other phases are sequential.
