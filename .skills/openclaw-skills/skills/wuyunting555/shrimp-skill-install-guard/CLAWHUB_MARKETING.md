# ClawHub Marketing Copy - skill-install-guard

## 中文标题
Skill Install Guard｜技能安装守门员

## 一句话介绍
给 OpenClaw skill 安装加上一道可执行的质量闸门：安装前做来源与代码审查，安装后做落地验收，输出清晰的 go/no-go 结论。

## 详细介绍
`skill-install-guard` 面向真实安装场景，把“是否应该安装、安装是否真正成功”拆成一套标准化流程。它会先检查来源可信信息，再对目标 skill 做全量文件覆盖审查，输出权限需求、风险等级和安装建议；只有在满足条件时才执行安装；执行后继续校验目录与关键文件是否落地。

相比只看命令返回值，这种方式更适合长期维护和团队协作：每次安装都能留下一致的判断依据，既可读、也可复盘、也能用于自动化流水线。

## 核心优势
- **安装前后全流程守门**：审查、决策、执行、验收一体化
- **全量文件覆盖视角**：不是只看入口文档，而是关注整体内容
- **风险决策清晰**：红旗、权限、风险等级、建议结论一次给全
- **结果可验证**：安装后验证路径与关键文件，避免“假成功”
- **适配个人与团队**：既能单人快速决策，也能满足团队标准化

## 适合谁
- 频繁安装新 skill 的 OpenClaw 用户
- 负责团队环境治理与规范落地的维护者
- 对供应链风险敏感、需要可审计流程的组织

## 对外推荐短句
- 安装 skill 前先做一轮可执行审查，少走弯路、少踩坑。
- 不只判断“能不能装”，还判断“值不值得装、装完是否真的可用”。
- 给每次安装留下可复盘证据，让安全与效率同时在线。
