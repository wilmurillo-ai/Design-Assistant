# crypto-content-automation

加密货币自媒体运营自动化工具，一句话完成热点扫描、内容策划。

## 功能

- 🔥 热点扫描：AI / Crypto / AI+Crypto 热点
- 📝 内容策划：基于热点的深度策划案
- 🚀 自动化：一键执行全套流程

## 使用

```bash
# 热点扫描
python3 scripts/run.py scan

# 内容策划
python3 scripts/run.py plan

# 完整流程
python3 scripts/run.py all
```

## 结果

- 热点报告：`logs/hot_topics_YYYYMMDD.json`
- 策划案：`logs/content_planning_YYYYMMDD.md`
