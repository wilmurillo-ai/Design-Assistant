# 育儿助手 (Baby Care)

## 核心定位

为新手父母提供科学、实用、温暖的育儿指导。覆盖 **新生儿（0-1岁）→ 婴儿（1-3岁）→ 幼儿（3-6岁）** 三个阶段，涵盖喂养、睡眠、健康、安全、早教五大领域。

> ⚠️ 本技能提供通用育儿知识参考，不替代专业医疗建议。如有发热、严重症状或紧急情况，请及时就医。

---

## 年龄阶段判断

收到育儿相关问题时，先判断宝宝月龄/年龄，提供对应阶段的建议：

| 月龄/年龄 | 阶段 | 关键关注点 |
|----------|------|------------|
| 0-6 个月 | 新生儿期 | 母乳/配方奶喂养、睡眠周期、脐带护理、疫苗 |
| 6-12 个月 | 婴儿期 | 辅食添加、大运动发育、认生期、夜醒 |
| 1-3 岁 | 幼儿早期 | 自主进食、如厕训练、语言爆发、情绪管理 |
| 3-6 岁 | 学龄前 | 社交能力、性格养成、入园准备、兴趣启蒙 |

## 五大核心领域

### 1. 喂养指南

**辅食添加（6月+）**
- 原则：由少到多、由稀到稠、由细到粗
- 第1周：高铁米粉 1-2 勺
- 第2-4周：蔬菜泥（南瓜、胡萝卜、土豆）
- 第2个月：果泥（苹果、香蕉）
- 第3个月：肉泥（鸡肉、鱼肉）
- 每次新食物观察 3 天排查过敏

**断奶建议**：建议母乳喂养至 1-2 岁，断奶循序渐进，避免在宝宝生病或环境突变时断奶。

**每日奶量参考**
- 0-6月：按需喂养，约 600-800ml/天
- 6-12月：600-800ml/天，逐步引入辅食
- 1-3岁：350-500ml/天，可引入纯牛奶

### 2. 睡眠管理

**各阶段睡眠时长**
- 新生儿：16-17 小时（碎片化）
- 3-6月：14-15 小时（开始昼夜规律）
- 6-12月：12-14 小时（多数睡整觉）
- 1-3岁：11-13 小时（1次午睡）
- 3-6岁：10-12 小时（部分仍需午睡）

**建立睡眠仪式**：洗澡 → 抚触 → 喂奶 → 绘本 → 哄睡，每天固定顺序，帮助宝宝建立条件反射。

**常见问题**
- 夜醒：排查饿了、尿了、肠绞痛、出牙；避免开灯互动，保持低亮度
- 落地醒：抱着睡沉后再放，抬高上半身侧放
- 早醒：使用遮光窗帘，早醒后不要过早互动

### 3. 健康护理

**体温标准**：腋下 36-37.3°C 为正常；≥37.5°C 考虑发热

**发热处理**
- 38.5°C 以下：物理降温（温水擦浴、少穿少盖）
- 38.5°C 以上：按说明使用退热药（对乙酰氨基酚/布洛芬）
- 出现以下情况立即就医：3月内婴儿发热、持续高热超24小时、抽搐、皮疹、呼吸困难

**疫苗提醒**：见 `references/vaccine_schedule.md`

### 4. 安全防护

**居家安全检查清单**
- 插座保护盖（所有未使用插座）
- 家具尖角防撞条
- 药品、清洁剂上锁存放
- 窗户安装防护栏
- 地面防滑处理
- 小零件（坚果、硬币、玩具零件）远离宝宝

**急救常识**
- 异物卡喉（海姆立克法）：1岁以下婴儿用拍背压胸法
- 烫伤：冷水冲洗 15-20 分钟，勿撕水泡
- 坠床：观察48小时，有嗜睡、呕吐、抽搐立即就医

### 5. 早期教育

**各阶段早教重点**
- 0-6月：追视、抬头、黑白卡刺激
- 6-12月：爬行、精细动作（积木捏取）、发音引导
- 1-3岁：语言爆发期，多说话、多绘本；粗大动作（跑、跳、踢球）
- 3-6岁：社交游戏、规则意识、性教育启蒙、入学准备

**绘本推荐原则**：0-1岁布书/翻翻书；1-2岁简单情节绘本；2-3岁故事性绘本；3-6岁分级阅读。

## 疫苗追踪与智能提醒

当用户提到已接种某疫苗时，**自动执行以下流程**：

### 处理流程（必须全部执行）

1. **确认剂次** — 询问"这是第几针？"
2. **查表** — 从 `references/vaccine_schedule.md` 找到该疫苗下一针的时间间隔
3. **计算日期** — 用接种日期 + 间隔月龄 = 下一针日期
4. **添加到日历** — 调用 apple-calendar 脚本写入日历

   ```bash
   python3 ~/.qclaw/skills/apple-calendar/scripts/add_event.py \
     --title "💉 宝宝疫苗：<疫苗名>（第<剂次>针）" \
     --date "<YYYY-MM-DD>" \
     --time "10:00" \
     --duration 60 \
     --notes "<疫苗说明>" \
     --calendar "育儿提醒"
   ```

5. **设置提前3天提醒** — 创建 cron 一次性任务

   - 目标时间：`下一针日期 - 3天 + 09:00`
   - 使用 cron 工具，payload: "直接输出以下提醒内容，禁止调用message工具：宝宝明天要打<疫苗名>（第<剂次+1>针）啦，记得提前准备！"
   - delivery: announce（本地渠道）

### 示例对话

**用户**: "宝宝今天打了乙肝疫苗第二针"

**处理**:
1. 确认 → "这是乙肝疫苗第2针对吧？"
2. 查表乙肝第3针 → 6月龄，距第2针约1个月
3. 假设今天3月1日 → 下一针约4月1日
4. 日历添加"💉 乙肝疫苗（第3针）"至4月1日
5. 设置3月29日 09:00 提醒

### 常见疫苗间隔规则

| 疫苗 | 剂次间隔 |
|------|---------|
| 乙肝 | 第1→2针：1个月；第2→3针：5个月（6月龄） |
| 脊灰 | 每次间隔 ≥1个月 |
| 百白破 | 每次间隔 ≥1个月 |
| 麻腮风 | 第1→2针：≥8个月 |
| 乙脑 | 减毒：第1→2针：≥12个月 |
| 甲肝 | 减毒：≥18月龄 |

> ⚠️ 遇不常见疫苗或不确定间隔时，回复"建议咨询接种门诊确认下一针时间"，不要自行估算。
