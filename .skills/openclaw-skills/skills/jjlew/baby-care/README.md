# 🤱 baby-care - 育儿助手

> 0-6 岁宝宝全方位育儿指南，AI 驱动

## 功能

- 🍼 **喂养指南** — 辅食添加、每日奶量、断奶建议
- 😴 **睡眠管理** — 各阶段睡眠时长、睡眠仪式、常见问题
- 💊 **健康护理** — 体温标准、发热处理、常见疾病
- 🛡️ **安全防护** — 居家安全检查、急救常识
- 📚 **早期教育** — 各阶段早教重点、绘本推荐
- 💉 **疫苗时间表** — 0-6 岁免费+自费疫苗完整时间表
- 🏃 **发育里程碑** — 大运动/精细运动/语言/社交发育标准

## 安装

```bash
clawhub install baby-care
```

## 使用

在 OpenClaw / QClaw 中，直接用中文提问即可：

- "宝宝 6 个月了，辅食怎么加？"
- "宝宝发烧 38.5 度怎么办？"
- "1 岁宝宝每天睡多久？"
- "有哪些疫苗要打？"
