# 宝宝常见疾病护理

## 感冒（普通感冒）

**症状**：鼻塞、流涕、轻微咳嗽、低热
**护理**：
- 多喝温水，保持室内湿度50-60%
- 生理盐水滴鼻缓解鼻塞
- 咳嗽不严重时不需止咳，保持空气清新
- 观察3天无好转或加重就就医

## 幼儿急疹

**特点**：6月-2岁多见，高热3-5天后出疹，热退疹出
**护理**：
- 退热为主（体温≥38.5°C用退热药）
- 多喝水，无需特别皮肤护理
- 出疹后观察，无需特别处理
- 幼儿急疹本身无害，重点是排查其他发热原因

## 腹泻

**常见原因**：病毒性（轮状、诺如）、细菌性、乳糖不耐、辅食不当
**护理**：
- 补液盐（ORS）预防脱水是核心
- 继续母乳喂养，配方奶喂养可临时换无乳糖奶粉
- 蒙脱石散可缩短病程
- 出现以下情况立即就医：出现脱水表现（眼窝凹陷、无泪、尿少）、血便、持续高热

## 湿疹（特应性皮炎）

**特点**：0-2岁多发，脸颊、四肢伸侧为主，干燥、瘙痒
**护理**：
- 保湿是基础：厚涂润肤霜，每天2-3次
- 纯棉衣物，避免过度穿衣
- 排查过敏原（食物、尘螨等）
- 严重时用弱激素药膏（地奈德），安全可控
- 保持指甲短避免抓伤

## 咳嗽

**何时需要担心**：
- 犬吠样咳嗽（喉炎，急！）
- 喘息性咳嗽、呼吸困难
- 咳嗽超过2周
- 小婴儿咳嗽
- 发热伴咳嗽超过3天

**护理**：
- 1岁以下蜂蜜（避免，1岁以下禁用）
- 1岁以上睡前喝勺蜂蜜缓解
- 保持室内湿度，开窗通风
- 拍背帮助排痰

## 鼻塞/流鼻涕

- 生理盐水/海盐水滴鼻
- 加湿器保持湿度
- 吸鼻器轻柔吸出鼻涕
- 避免用力擤鼻（易引发中耳炎）

## 热性惊厥

**处理（保持冷静）**：
1. 让宝宝侧卧，防止窒息
2. 不要往嘴里塞东西
3. 不要按住宝宝
4. 记录惊厥开始时间
5. 惊厥超过5分钟或反复发作立即就医

## 就医信号（红灯）

出现以下任意情况请立即就医：
- 3个月内婴儿发热（任何体温）
- 呼吸急促（婴儿>60次/分，幼儿>40次/分）
- 嘴唇/指甲发紫
- 意识不清、嗜睡难以唤醒
- 抽搐
- 脱水表现
- 皮疹伴发热（需排查脑膜炎）
- 持续哭闹无法安抚
