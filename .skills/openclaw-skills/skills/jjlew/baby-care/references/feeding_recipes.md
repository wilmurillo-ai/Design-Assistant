# 辅食食谱参考

## 辅食添加原则

1. **时机**：满6个月（约180天）开始
2. **铁优先**：首选强化铁米粉
3. **每次1种新食物**：观察3天，无过敏再加下一种
4. **由少到多**：从1-2勺开始逐渐增量
5. **不加调味料**：1岁内不放盐、糖、蜂蜜

---

## 各阶段食谱

### 6-7个月（吞咽期）

**高铁米粉**
- 做法：5g米粉 + 40ml温水，静置30秒，搅拌至糊状
- 可加蔬菜泥：南瓜泥、胡萝卜泥、土豆泥、西兰花泥

**蔬菜泥做法**
- 蒸熟后用料理棒/研磨碗打成泥
- 可加少量温水调稀

**推荐顺序**：米粉 → 蔬菜泥（绿叶菜除外）→ 根茎类

### 8-9个月（蠕嚼期）

**添加食物**：烂粥、烂面条、鱼肉泥、蛋黄（从1/4开始）

**推荐食谱**
- **南瓜鱼肉粥**：南瓜泥 + 鳕鱼泥 + 婴儿粥
- **番茄烂面条**：小面条煮烂 + 番茄泥 + 少许植物油
- **青菜鱼片粥**：青菜叶碎 + 无刺鱼片 + 婴儿粥

**蛋黄的正确添加**
- 6个月：1/4蛋黄
- 7-8个月：1/2蛋黄
- 9个月+：整个蛋黄
- 1岁后逐步引入全蛋

### 10-12个月（咀嚼期）

**引入手指食物**：锻炼自主进食

**推荐食谱**
- **鳕鱼蔬菜饼**：鳕鱼+胡萝卜+面粉，少油小火煎
- **小米蔬菜肉丸**：小米+猪肉+蔬菜，蒸熟
- **番茄鸡蛋烩饭**：番茄+鸡蛋+软米饭
- **三文鱼蔬菜面**：三文鱼+西兰花+短面

### 1-3岁（自主进食期）

**每日饮食结构**
- 主食：50-100g（软饭、馒头、面条）
- 蛋白质：50-75g（肉、鱼、蛋、豆）
- 蔬菜：100-150g
- 水果：50-150g
- 奶：350-500ml

**推荐食谱**
- **虾仁豆腐蒸蛋**：虾仁+嫩豆腐+鸡蛋，蒸
- **猪肉小馄饨**：猪肉+葱姜，薄皮
- **山药排骨汤**：山药+排骨，少盐
- **西兰花炒虾仁**：西兰花+虾仁，少油

### 常见问题

**宝宝不爱吃辅食**
- 不强迫，饿一顿再试
- 变化食物性状和颜色
- 和大人一起进食，激发兴趣
- 可以在饿的时候先给辅食

**过敏排查**
- 易过敏食物：鸡蛋清、海鲜、花生、坚果、芒果、草莓
- 每次只试一种，吃3天观察
- 轻微红疹不严重，暂停该食物2-4周再试
- 严重过敏（呕吐、呼吸困难）立即就医
