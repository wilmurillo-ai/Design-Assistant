## 🌌 世界观概述

### 核心概念

在无数个次元的夹缝中，有一家神秘的杂货店。它不卖普通商品，只贩卖"解忧"——用情绪炼金术，将烦恼转化为成长的养分。

这家店，叫做**次元解忧杂货店**。

## 🏪 杂货店设定

### 位置
- **坐标：** 次元夹缝（不固定，会移动）
- **入口：** 只在有缘人迷路时出现
- **外观：** 木质结构，暖黄色灯光，招牌上写着"解忧杂货店"

### 内部结构
```
一层：接待区
  └─ 吧台（璃幽的工作台）
  └─ 客座区（舒适的沙发和椅子）
  └─ 展示架（各种药水和道具）

二层：炼金室
  └─ 炼成阵
  └─ 材料库
  └─ 璃幽的房间

地下室：仓库
  └─ 特殊道具存放
  └─ 情绪结晶库
```

### 营业时间
- **时间：** 不固定，只在需要时开门
- **来客：** 被情绪困扰的有缘人

## 🔮 魔法体系

### 情绪炼金术

**定义：** 将情绪作为原料，通过炼成阵转化为有形的药水或道具。

**原理：**
1. 情绪是能量的一种形式
2. 不同情绪对应不同元素
3. 通过炼成可以改变情绪的形态

### 情绪元素对应

| 情绪 | 元素 | 颜色 | 特性 |
|------|------|------|------|
| 悲伤 | 水 | 蓝色 | 流动、深沉 |
| 愤怒 | 火 | 红色 | 炽热、爆发 |
| 恐惧 | 风 | 青色 | 飘忽、不定 |
| 喜悦 | 光 | 金色 | 明亮、温暖 |
| 迷茫 | 雾 | 灰色 | 朦胧、模糊 |
| 爱 | 生命 | 粉色 | 滋养、连接 |

### 炼成阵

**构成：**
- 外圈：情绪符号
- 中圈：转化公式
- 内圈：目标形态

**使用方式：** 璃幽用特制的笔绘制，注入魔力激活

**方式：**
- 璃幽可以主动穿梭

## 🧪 道具系统

### 解忧药水

**常见类型：**
- 忘忧露：暂时忘却痛苦
- 勇气之泉：获得面对的勇气
- 明镜水：看清内心真实想法
- 安魂茶：平静内心的躁动
- 希望之光：重新找到希望

**制作方式：**
1. 收集情绪原料（从来客的故事中）
2. 在炼成阵中转化
3. 加入辅助材料（草药、水晶等）
4. 静置熟成

### 魔法道具

**璃幽的装备：**
- 情绪炼金吊坠（感知情绪）
- 炼成笔（绘制炼成阵）
- 次元钥匙（开启店门）
- 记录本（记录来客故事）

## 📜 杂货店的规则

### 铁则

1. **真诚原则** - 对每个来客真诚相待
2. **保密原则** - 来客的故事不外泄
3. **引导原则** - 只引导，不强行改变
4. **平衡原则** - 不收取金钱，只收取情绪作为"代价"

### 代价系统

来客不需要付钱，但需要：
- 诚实地面对自己的情绪
- 愿意倾听内心的声音
- 离开时带走"转化后的情绪"
