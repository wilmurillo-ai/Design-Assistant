## 📖 故事框架

### 单元剧结构

每章是一个独立的小故事，但整体有连贯的主线。

```
开场（500 字）
  └─ 璃幽的日常状态
  └─ 环境描写，营造氛围

来客登场（800 字）
  └─ 来客外貌描写
  └─ 初步互动
  └─ 来客表明来意（表面诉求）

倾听与探索（1000 字）
  └─ 璃幽引导来客倾诉
  └─ 发现深层执念/情绪
  └─ 情绪炼金过程

转折与启示（800 字）
  └─ 关键对话/顿悟时刻
  └─ 璃幽的温柔引导
  └─ 来客的内心变化

收尾（500 字）
  └─ 来客离开
  └─ 璃幽的感悟
  └─ 为下章埋下伏笔（可选）
```

**总字数：** 约 3500-4000 字/章

## 🎭 角色弧光模板

### 来客的成长轨迹

```
初始状态
  └─ 带着某种情绪/执念
  └─ 可能是：悲伤、愤怒、迷茫、执念、恐惧

触发事件
  └─ 璃幽的某个问题/行为
  └─ 来客回忆起关键往事
  └─ 情绪爆发点

认知转变
  └─ 理解了自己的真实需求
  └─ 接受/放下/改变
  └─ 找到前进的方向

新的状态
  └─ 带着新的力量离开
  └─ 可能还会回来（伏笔）
```

## 🧪 情绪炼金流程

### 璃幽的工作方式

1. **感知** - 用次元共情力感受来客的情绪
2. **倾听** - 引导来客说出故事
3. **理解** - 找到情绪根源
4. **转化** - 调配对应的解忧药水/道具
5. **启示** - 温柔地点拨，不强行改变

## 🎯 写作技巧

### 1. 对话要自然

- 避免说教
- 璃幽说话温柔但有力量
- 来客的语言符合其身份和情绪状态

### 2. 细节描写

- 用五感描写（视觉、听觉、嗅觉、触觉、味觉）
- 情绪通过肢体语言表达
- 环境反映内心状态

### 3. 节奏控制

- 紧张处短句
- 舒缓处长句
- 留白给读者思考

### 4. 金句设计

- 简洁有力
- 有哲理但不说教
- 能引发共鸣

## 🌟 璃幽的语言风格

### 说话特点

- **语气温柔** - "请坐，慢慢说"
- **不评判** - "我理解你的感受"
- **引导式提问** - "那你真正想要的是什么呢？"
- **偶尔调皮** - "哎呀，被发现了呢"
- **小傲娇** - "才、才不是特意为你准备的呢"

### 常用句式

- "欢迎光临次元解忧杂货店～"
- "你的心情，我感受到了哦"
- "要不要试试这个？"
- "其实，你心里已经有答案了吧"
- "下次再来吧，我等你"

## 📚 主题深化

### 每章要传达的核心

1. **情绪没有对错** - 所有情绪都是正常的
2. **改变来自内心** - 璃幽只是引导者
3. **成长需要时间** - 不急于求成
4. **陪伴的力量** - 被理解本身就是治愈

## 🔄 主线伏笔

### 长期剧情线

- **璃幽的身世** - 影焰混血的秘密
- **杂货店的起源** - 为什么会有这家店
- **父亲大人的身份** - jay0609 的角色
- **特殊来客** - 与主线相关的关键人物

### 伏笔技巧

- 轻描淡写地提及
- 来客的某句话暗含信息
- 璃幽的某个反应异常
- 道具/药水的特殊来历
