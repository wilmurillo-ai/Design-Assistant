💡个体工商户 · 合规节税建议
善用季度免税门槛
小规模纳税人季度销售额 ≤ 30万元免征增值税。若业务量可控，合理安排开票节奏，避免单季超过30万触发纳税义务，可有效降低增值税及附加税负担。
个税专项附加扣除全部申报
子女教育、大病医疗、住房贷款利息、赡养老人等专项附加扣除每月最高可减免数千元税基。通过"个人所得税"App 填报，不申报即视为放弃，每年汇算清缴前务必核查是否遗漏。
成本费用据实列支，保留凭证
办公场地租金、水电、员工社保、设备折旧、业务招待费（50%限额扣除）等均可在经营所得中税前扣除。每笔支出必须取得合法发票，否则税务稽查时无法税前扣除，面临补税风险。
合理选择纳税人身份
年销售额临近500万时，需综合评估：升为一般纳税人可抵扣进项税但税率更高；维持小规模适用低征收率但不能开专票影响B端客户。建议提前测算两种方案的综合税负，再做决策。
核定征收 vs 查账征收的选择
若实际成本费用较高、账务健全，查账征收可能更合算；若成本难以举证或账务不完整，税务机关核定的应税所得率可能比实际利润率低，核定征收反而节税。建议咨询当地税务顾问后决定。
⚠️合规红线提示
• 虚开发票（含购买他人发票）：补税+罚款+可能追究刑事责任
• 隐匿收入不申报：税务稽查后按偷税论处，最高加收5倍罚款
• 逾期申报：按日加收万分之五滞纳金，情节严重面临行政处罚
• 节税方案须以真实业务为基础，任何没有实质的"筹划"均涉嫌违法
以上建议仅供参考，具体税务决策建议咨询持证注册税务师（CTA）或税务师事务所。
