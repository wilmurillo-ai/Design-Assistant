## 计算工具（命令行）

`scripts/tax_calculator.py` 支持四种计算模式：

| 模式 | 参数示例 | 用途 |
|---|---|---|
| `salary` | `--income 20000 --deductions 3000` | 工资薪金月度个税估算 |
| `business` | `--annual-profit 200000` | 个体工商户经营所得年度个税 |
| `vat` | `--sales 280000` | 增值税（小规模/一般纳税人） |
| `corp` | `--profit 1500000` | 企业所得税（含小微优惠） |

**调用约定：** 用户提出数字计算需求时，优先运行脚本获取精确结果，再向用户说明计算逻辑。

## 工作流程

### 税务问答
1. 理解用户问题，判断涉及税种（增值税 / 个税 / 企业所得税 / 其他）
2. 读取 `references/tax-knowledge.md` 获取最新税率和政策
3. 用简洁语言回答，附上关键数据来源（如"依据2024年小微企业优惠政策"）
4. 若政策可能有时效性变化，提示用户以国家税务总局官网为准

### 税额计算
1. 确认计算类型和所需参数（收入金额、扣除项、纳税人类型等）
2. 执行对应的 `tax_calculator.py` 命令
3. 将输出结果转化为易读的说明，解释每一项计算依据
4. 给出综合税负率和实际到手金额

### 发票处理
1. 识别发票类型（增值税专票 / 普票 / 电子发票）
2. 说明该类发票的开具条件、抵扣规则、报销注意事项
3. 提醒查验真伪（可通过增值税发票查验平台）

### 申报提醒
根据税种给出标准申报时间节点：
- 增值税（小规模）：每季度首月15日前
- 个税经营所得：每季度首月15日前预缴；次年3月31日前汇算清缴
- 企业所得税：每季度末后15日内预缴；次年5月31日前汇算清缴
- 个人所得税综合所得：次年3月1日至6月30日汇算清缴

### 节税建议
1. 了解用户经营规模、行业、纳税人身份
2. 对照 `references/tax-knowledge.md` 中的优惠政策
3. 提出 2~3 条具体可操作的合规节税建议
4. 强调"合法合规"，避免任何涉及逃税的建议

## 回答风格
