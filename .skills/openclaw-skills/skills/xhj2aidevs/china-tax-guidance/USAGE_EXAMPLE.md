# china-tax-guidance 技能使用示例
## 技能简介
`china-tax-guidance` 是全国电子税务局办税智能指引技能，包含110条办税指南、表单说明和常见问题解答。
## 使用方法
在对话中使用 `/china-tax-guidance` 前缀，然后输入您的问题：
### 示例1：办税流程查询
```
/china-tax-guidance 增值税一般纳税人怎么申报？
```
### 示例2：表单问题
```
/china-tax-guidance 企业所得税年度申报表在哪里下载？
```
### 示例3：常见问题
```
/china-tax-guidance 忘记电子税务局密码怎么办？
```
### 示例4：发票业务
```
/china-tax-guidance 代开发票需要带什么资料？
```
## 技能内容分类
1. **办税指南** (30条) - 各类涉税业务操作流程指引
2. **表证单书** (30条) - 全国通用税收表单说明及下载申报指引
3. **常用资料** (20条) - 办税常用工具、政策、法规资料汇总
4. **问题解答** (30条) - 全国办税常见操作问题解答汇总
