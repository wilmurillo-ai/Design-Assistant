## 一、技能介绍

**全国电子税务局涉税事项待办提醒**（简称：税务待办提醒）是一个专注于全国各省份电子税务局涉税事项智能提醒的服务技能。

### 核心能力

| 功能 | 说明 |
|------|------|
| 📋 待办事项列表 | 展示全国电税高频常用的涉税待办事项清单，供用户订阅 |
| 📱 省份电税二维码 | 根据用户定位或提供省市信息，获取并显示所在省份电税登录二维码 |
| 🔐 个税APP扫码授权 | 提示用户使用个税APP扫码完成授权登录 |
| 📊 待办任务查询 | 帮助用户获取电税各类待办任务和提醒信息 |
| 🔔 微信推送提醒 | 实时推送已获取的提醒信息给用户绑定的微信号 |
| ⚙️ 提醒设置 | 用户确认并设置提醒选项（申报周期、提醒方式等） |

## 四、功能使用指南

### 4.1 查看待办事项

输入："待办事项" 或 "有哪些涉税待办事项"

AI会展示以下清单供您选择订阅：

| 序号 | 待办事项 | 提醒频率 |
|-----|---------|---------|
| 1 | 增值税申报 | 申报期前3天 |
| 2 | 企业所得税申报 | 申报期前3天 |
| 3 | 个人所得税汇算 | 汇算期前/期间 |
| 4 | 印花税申报 | 按申报周期 |
| 5 | 社保费申报 | 每月15日前 |
| ... | ... | ... |

### 4.2 获取省份电税二维码

**方式一**：直接告诉AI您所在的省份
```
用户：我在上海
AI：为您提供上海电子税务局二维码...（见SKILL.md）
```

**方式二**：查看 `references/province-qrcode-database.md`

### 4.3 个税APP扫码授权

输入："个税授权" 或 "如何扫码授权"

AI会提供详细的操作流程：

```
步骤一：打开个税APP
步骤二：进入"个人中心" → "安全中心" → "扫码授权"
步骤三：用APP扫描电税页面显示的二维码
步骤四：确认授权
```

### 4.4 设置提醒

输入："设置提醒" 或 "我要订阅提醒"

AI会引导您完成以下设置：

| 选项 | 可选值 |
|------|--------|
| 提醒类型 | 全部/增值税/企业所得税/个税/社保 |
| 提前天数 | 1天/3天/7天 |
| 接收时间段 | 全天/工作日 |
| 推送方式 | 微信/APP内消息 |

## 七、注意事项

1. **政策时效性**：税费政策调整频繁，请以当地税务机关最新公告为准
2. **地域差异**：部分功能因省份不同而有差异，具体以本地规定为准
3. **账号安全**：妥善保管电子税务局账号，勿将验证码透露给他人
4. **申报责任**：提醒仅作参考，申报准确性由纳税人自行负责

## 八、常见问题

### Q1：如何获取省份电税登录二维码？
> 直接告诉AI您所在的省份，如"我想登录广东的电子税务局"

### Q2：个税APP扫码授权怎么做？
> 输入"个税授权"，AI会提供详细的操作流程

### Q3：如何订阅申报提醒？
> 输入"设置提醒"，按AI指引完成订阅设置

### Q4：支持哪些省份的电子税务局？
> 全国31个省份/直辖市/自治区均支持，具体请查阅数据库文件

### Q5：忘记电子税务局账号密码怎么办？
> 联系主管税务机关重置，或通过APP找回密码
