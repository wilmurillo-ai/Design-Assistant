### TD005 个人所得税综合所得年度汇算

| 项目 | 内容 |
|-----|------|
| **事项代码** | IIT_Comprehensive |
| **事项名称** | 个人所得税综合所得年度汇算 |
| **法律依据** | 个人所得税法第十一条 |
| **申报期限** | 每年3月1日至6月30日 |
| **申报方式** | 个税APP / 自然人电子税务局 / 办税服务厅 |
| **适用范围** | 工资薪金、劳务报酬、稿酬、特许权使用费 |
| **应税收入** | 60000元基本减除+专项扣除+专项附加扣除+依法其他扣除 |
| **税率** | 3%-45%超额累进 |
| **退税条件** | 已预缴税额>应纳税额 → 申请退税 |
| **补税情形** | 已预缴税额<应纳税额 → 补税（≤400元可豁免） |

> ⚠️ **特别提醒**：2026年个税汇算正在进行中，3月1日-6月30日可办理

## 四、处理流程图

### 4.1 增值税申报流程

```
1. 登录电子税务局
   ↓
2. "我要办税" → "税费申报及缴纳"
   ↓
3. 选择"增值税申报"
   ↓
4. 填写/确���申���表数据
   ↓
5. 检查数据准确性
   ↓
6. 点击"申报" → "确认"
   ↓
7. 如需补税 → 点击"缴款"
   ↓
8. 申报完成
```

### 4.2 个税年度汇算流程

```
1. 打开个税APP或登录自然人电子税务局
   ↓
2. 首页 → "综合所得年度汇算"
   ↓
3. 选择"申报年度：2025"
   ↓
4. 确认基本信息 → "下一步"
   ↓
5. 查看收入及扣除数据
   ↓
6. 如有奖金选择计税方式
   ↓
7. 计算应补/应退税额
   ↓
8. 确认"提交申请"
   ↓
9. 完成（补税/退税）
```
