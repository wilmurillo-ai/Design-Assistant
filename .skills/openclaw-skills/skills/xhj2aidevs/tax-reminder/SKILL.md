# 🧾 全国电子税务局涉税事项待办提醒

> 版本：v1.1.0 | 更新日期：2026-04-15 | 数据来源：国家相关官方政策法规库
>
> 全国各省份电子税务局（Web端/APP端）涉税待办事项的智能提醒服务，涵盖申报提醒、个税扫码授权、待办任务查询、二维码获取等功能。

## 一、技能能力概览

### 1.1 核心功能模块

| 功能编号 | 功能名称 | 说明 |
|---------|---------|------|
| F01 | 📋 **待办事项列表** | 展示全国电税高频常用的涉税待办事项清单，供用户订阅 |
| F02 | 📱 **省份电税二维码** | 根据用户定位或提供省市信息，获取并显示所在省份电税登录二维码 |

| F04 | 📊 **待办任务查询** | 帮助用户获取电税各类待办任务和提醒信息 |
| F05 | 🔔 **微信推送提醒** | 实时推送已获取的提醒信息给用户绑定的微信号 |
| F06 | ⚙️ **提醒设置** | 用户确认并设置提醒选项（申报周期、提醒方式等） |

### 2.3 F03：个税APP扫码授权

指引用户使用**个人所得税APP**完成扫码授权，用于自动获取个税申报信息：

#### 🔐 个税 APP 扫码授权流程

**第一步：打开个税APP**
> 手机应用市场搜索"个人所得税"下载安装，或已有APP请更新至最新版本

**第二步：进入授权页面**
> APP首页 → [个人中心] → [安全中心] → [扫码授权] → [授权登录]

**第三步：扫码授权**
> 用APP扫描电脑端/网页端显示的授权二维码，确认授权

**第四步：授权成功**
> 授权成功后，可自动获取您在当前设备上的个税申报信息

> ⚠️ **注意**：
> - 授权有效期通常为**30分钟**，超时需重新扫码
> - 仅授权读取信息，**不会**自动申报或操作
> - 如不再使用，请在APP中[取消授权]

### 2.4 F04：待办任务查询

帮助用户获取其在电子税务局中的**待办任务列表**：

#### 📊 查询方式

方式一：**扫码登录后查询**（推荐）
> 使用省份电税二维码扫码登录 → 首页"我要办" → "待办事项"查看

方式二：**APP查询**
> 下载对应省份税务APP → 登录后查看"待办事项"

方式三：**用户告知，我帮您分析**
> 告诉我您在电子税务局看到的待办事项名称，我帮您分析处理方法

#### 🔍 常见待办事项处理指引

| 待办事项名称 | 处理路径 | 注意事项 |
|------------|---------|---------|
| 增值税申报 | 我要办税 → 常用申报 → 增值税申报 | 选对纳税人资格（一般/小规模） |
| 企业所得税申报 | 我要办税 → 申报纳税 → 企业所得税 | 季度末次年4月、年度5月31日前 |
| 个人所得税汇算 | 申报纳税 → 个人所得税 → 年度汇算 | 3月1日-6月30日 |
| 财务报表报送 | 我要办税 → 财务  表报送 | 季度/月度报送 |
| 社保费申报 | 我要办税 → 社保费申报 | 确认缴费基数 |
| 发票验旧 | 我要领票 → 发票验旧 | 准备空白发票 |

#### 🔔 推送内容

| 推送类型 | 内容 | 推送时机 |
|---------|------|---------|
| 📅 申报提醒 | 增值税/所得税/个税等申报截止前3天 | 申报期前 |
| ⚠️ 逾期预警 | 申报/缴费即将逾期 | 逾期前3天 |
| 📢 政策变更 | 税费优惠政策变化 | 政策发布时 |
| 💰 退库通知 | 退税到账 | 退库完成时 |

#### 📲 绑定方式

**方案一：通过技能绑定（推荐）**
> 告诉我"我要绑定微信"，按指引提供：
> 1. 微信号（用于通知推送）
> 2. 需要订阅的提醒类型
> 3. 提醒接收时间段

**方案二：通过公众号绑定**
> 关注指定税务公众号 → 输入"绑定" → 按指引操作

> ⚠️ **隐私说明**：仅用于接收提醒推送，不会发送任何营销信息，随时可解绑

## 五、注意事项

1. **时效性**：税费政策调整频繁，请以**当地税务机关最新解释为准**
2. **地域差异**：部分功能因省份不同而有差异，具体以本地规定为准
3. **账号安全**：妥善保管电子税务局账号，勿将验证码透露给他人
4. **申报责任**：提醒仅作参考，申报准确性由纳税人自行负责
