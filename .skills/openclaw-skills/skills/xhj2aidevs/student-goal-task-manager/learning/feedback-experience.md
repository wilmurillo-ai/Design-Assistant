# 助学星技能 · 用户反馈与经验教训

> 目的：避免同类问题重复发生，持续优化交互质量。

## 反馈记录

### [F-001] Phase 2-1 材料引导缺失 (2026-04-19)

**场景**：用户上传课程表图片后，技能未能自动引导上传成绩表，导致目标推荐缺乏成绩风险评估。

**用户原话**：提供课程表后没有被主动询问成绩情况。

**改进**：Phase 2-1 末尾强制附加「📎 材料上传引导」模板，明确列出课程表+成绩表+文字描述三种方式。

**影响范围**：Step 2-1、Step 2-2、Step 2-3

## 经验教训

### [E-001] 注入页面必须包含倒计时显示

**场景**：用户是期中考试倒计时3-5天的紧急场景，但初始注入页面没有倒计时显示。

**经验**：当用户提到具体考试时间（期中/期末/补考/考证），生成的注入页面必须包含动态倒计时。

**应用规则**：Step 2-3 生成注入页面时，检查任务中是否含考试截止日期，若有则自动计算并显示倒计时。

---

### [E-002] 家长角色需要更直接的风险提示

**场景**：家长用户更关心"孩子会不会挂科"，而不是"目标偏离率"。

**经验**：对家长角色，学科风险评估应使用直观的红黄绿标识（🔴高危/🟡中危/🟢稳定），避免使用百分比偏离率等抽象指标。

**应用规则**：Step 2-2 中，当角色=家长时，自动将评估结果转换为颜色编码的风险等级。
