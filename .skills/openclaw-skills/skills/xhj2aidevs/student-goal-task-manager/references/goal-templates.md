# 助学星目标模板索引（按角色×阶段）

> 本文件供 Phase 2 目标推荐时查阅，根据用户角色和阶段快速匹配最相关的目标方向和模板。

---

## 一、目标方向总览（助学星内置）

助学星内置 7 大目标方向 × 4 个学习阶段：

| 方向 | 适用阶段 | 核心价值 |
|------|----------|----------|
| 考公 | 大学、高中 | 公务员考试备考规划 |
| 考编 | 大学 | 事业单位/教师编制备考 |
| 考研 | 大学 | 考研备考全流程管理 |
| 留学 | 大学、高中 | 语言考试+申请材料+时间线 |
| 创业 | 大学 | 项目孵化+资源积累 |
| 就业 | 大学、高中 | 简历+实习+校招时间线 |
| 综合素养 | 全阶段 | 证书/竞赛/社实/社团/体测 |

---

## 二、角色×阶段 → 推荐目标矩阵

### 🎓 学生-大学

**常见痛点**：不知道毕业去向、课程+证书+竞赛乱、入团入党流程不清、保研/考研节点不明

**推荐目标组合**：

| 用户痛点 | 推荐目标方向 | 推荐理由 |
|----------|-------------|----------|
| 毕业去向不明确 | 考研 / 就业 / 考编 | 明确方向，倒推时间线 |
| 证书规划无头绪 | 综合素养 | 四六级/计算机/教资/普通话 |
| 竞赛经历空白 | 综合素养 | 数学建模/挑战杯/学科竞赛 |
| 入团入党不清楚 | 综合素养 | 流程分解+时间节点提醒 |
| 想保研 | 考研（保研方向） | GPA+科研+竞赛三合一规划 |
| 想出国 | 留学 | 雅思/托福+选校+文书时间线 |
| 想创业 | 创业 | 校内赛事+导师+融资路径 |

**可分解的子目标示例**（大学-综合素养）：
- 英语四级（目标分数、备考计划、模考安排）
- 英语六级
- 计算机二级（Python/Office）
- 普通话水平测试
- 教师资格证（笔试+面试）
- 入团流程（申请书→团课→考试→注册）
- 入党流程（积极分子→培养→预备→转正）
- 社会实践（暑期/寒期/志愿服务）
- 奖学金申请

---

### 🎓 学生-高中

**常见痛点**：高考倒计时管理差、学科偏科、志愿填报无方向、竞赛不知如何参加

**推荐目标组合**：

| 用户痛点 | 推荐目标方向 | 推荐理由 |
|----------|-------------|----------|
| 高考备考混乱 | 综合素养（高考专项） | 各科任务+模考安排+错题本 |
| 不知报什么专业 | 就业/考研 | 提前规划职业方向 |
| 想参加竞赛加分 | 综合素养 | 联赛/信息学/数学竞赛路径 |
| 想出国读大学 | 留学 | 语言+选校+文书时间线 |

**可分解的子目标示例**（高中-综合素养）：
- 高考各科周计划（语数英理/文+理化生/政史地）
- 模拟考安排（月考、联考、模拟卷）
- 竞赛报名时间线（全国联赛/省赛）
- 艺术特长/体育加分材料准备
- 志愿填报研究（院校+专业+分数线）

---

### 🎓 学生-初中

**常见痛点**：中考压力、学科偏科、作业拖延、升高中志愿填报

**推荐目标组合**：

| 用户痛点 | 推荐目标方向 | 推荐理由 |
|----------|-------------|----------|
| 中考备考 | 综合素养（中考专项） | 各科计划+刷题+模考 |
| 作业+复习混乱 | 综合素养 | 日/周计划模板 |
| 想进重点高中 | 综合素养 | 体测+综评+学科竞赛 |

**可分解的子目标示例**（初中-综合素养）：
- 中考各科月度计划
- 错题本整理（数学/英语/物理）
- 体育成绩提升（1000米/50米/跳绳）
- 竞赛/特长加分项准备

---

### 🎓 学生-小学

**常见痛点**：作业计划性差、阅读/英语打基础、习惯养成

**推荐目标组合**：

| 用户痛点 | 推荐目标方向 | 推荐理由 |
|----------|-------------|----------|
| 作业拖延 | 综合素养（习惯养成） | 每日任务打卡 |
| 英语启蒙 | 综合素养 | 单词/听力/口语每日计划 |
| 阅读习惯 | 综合素养 | 每周阅读量目标+书单 |

---

### 👨‍👩‍👧 家长-大学生子女

**常见痛点**：孩子迷茫不知方向、不清楚孩子学了什么、孩子拖延不执行

**推荐目标组合**：同"学生-大学"，但侧重可见度+督导任务。

**推荐增加**：
- 每周与孩子对账目标进度（备忘录形式）
- 重要节点提醒（四六级报名截止/校招开始等）

---

### 👨‍👩‍👧 家长-高中生子女

**常见痛点**：高考倒计时焦虑、孩子不自律、志愿填报资料不足

**推荐目标组合**：同"学生-高中"，家长视角增加：
- 高考政策跟进（新高考/平行志愿规则）
- 招生简章/分数线收集备忘录
- 复习物资/报名费时间节点

---

### 👩‍🏫 老师-辅助学生

**常见痛点**：班级任务追踪混乱、学生目标差异大、提醒下发不及时

**推荐目标组合**：
- 班级作业/考试时间线（综合素养）
- 重要节点提醒模板（期中/期末/竞赛报名/志愿填报）
- 个别学生跟踪备忘录

## 三、通用目标模板（快速引用）

### 📅 证书考试时间线（大学通用）
```
近期任务示例：
- 英语四级报名（截止日期前1个月提醒）
- 四级冲刺：每天1套模拟题
- 计算机二级报名
- 普通话测试预约
```

### 📚 每周学习计划模板
```
- 周一：整理上周错题 + 本周重难点预习
- 周三：完成作业+当周习题册
- 周五：模拟测试/刷题
- 周日：本周复盘 + 下周计划
```

### 🎯 目标分解三层结构（助学星子目标格式）
```
总目标 → 阶段里程碑 → 可执行周任务
例：
考研目标
  └ 数学：暑期完成一轮复习
      └ 每天高数2小时（张宇/武忠祥）
  └ 英语：每天50个单词 + 1篇阅读
  └ 专业课：完成教材精读1遍
```

## 四、推荐话术参考

### Phase 2-2 目标推荐（大学生，痛点=证书+保研）

```
根据你的情况，建议重点规划以下目标：

🎯 目标一：综合素养 — 证书考试规划
   从四六级→计算机二级→普通话→教资，系统性安排，避免冲突撞期

🎯 目标二：考研（保研方向）— GPA+竞赛+科研三合一
   保研需要GPA前20%、竞赛获奖、科研经历，现在大一开始布局最合适

🎯 目标三：综合素养 — 入团入党流程
   流程长但有固定节点，用助学星设置好提醒，不漏掉任何一步

这些方向满足你的预期吗？
可以直接确认，也可以补充你自己的想法。
```
