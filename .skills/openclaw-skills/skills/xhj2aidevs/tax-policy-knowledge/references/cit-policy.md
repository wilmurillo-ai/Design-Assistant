# 企业所得税政策详解（2024-2026年）

> 更新时间：2026-04-19 | 政策来源：国家税务总局、财政部

## 一、小微企业所得税优惠

### 1.1 延续至2027年12月31日的优惠

| 优惠项目 | 具体内容 | 适用条件 |
|---------|---------|---------|
| 实际税负5% | 年应纳税所得额300万元以下 | 符合小微企业认定标准 |
| 设备器具扣除 | 500万元以下设备可一次性扣除 | 2027年底前购置 |
| 研发费用加计扣除 | 科技型企业120%，制造业100% | 符合条件的企业 |

### 1.2 小微企业认定标准

同时符合以下四个条件：
1. **行业要求**：从事国家非限制和禁止行业
2. **所得额**：年度应纳税所得额不超过300万元
3. **从业人数**：不超过300人
4. **资产总额**：不超过5000万元

### 1.3 税负计算示例

**年应纳税所得额100万元：**
```
应纳所得税 = 100万 × 5% = 5万元
```

**年应纳税所得额300万元：**
```
应纳所得税 = 100万 × 5% + 200万 × 25% = 5万 + 50万 = 55万元
有效税负率 ≈ 18.3%
```

## 二、高新技术企业优惠

### 2.1 优惠内容

| 优惠项目 | 普通企业 | 高新技术企业 |
|---------|---------|-------------|
| 企业所得税税率 | 25% | 减按15% |
| 研发费用加计扣除 | 75% | 100%-120% |
| 技术转让所得减免 | 500万内免征 | 500万内免征 |

### 2.2 高新技术企业认定条件

1. **知识产权**：拥有主要产品核心知识产权的所有权
2. **技术领域**：属于《国家重点支持的高新技术领域》
3. **科技人员**：占企业当年职工总数10%以上
4. **研发费用**：占销售收入比例符合要求
5. **高新收入**：高新产品收入占企业总收入60%以上
6. **企业成长性**：四项指标综合评分70分以上

## 三、研发费用加计扣除

### 3.1 加计扣除比例

| 企业类型 | 加计扣除比例 | 适用期间 |
|---------|-------------|----------|
| 制造业企业 | 100% | 2023年起 |
| 科技型中小企业 | 100% | 2022年起 |
| 其他企业 | 100% | 2023年起 |
| 集成电路/工业母机企业 | 120% | 2023年起 |

### 3.2 研发费用归集范围

| 费用类别 | 主要内容 |
|---------|---------|
| 人员人工费用 | 直接从事研发人员工资薪金、五险一金 |
| 直接投入费用 | 材料、燃料、动力费用 |
| 折旧费用 | 仪器、设备折旧 |
| 无形资产摊销 | 软件、专利权等摊销 |
| 其他相关费用 | 专家咨询费、差旅费等（不超过总额10%） |

### 3.3 申报要点

1. **项目判定**：区分研究阶段和开发阶段
2. **辅助账**：需设置研发费用辅助账
3. **报表填写**：A107012研发费用加计扣除优惠明细表

## 四、固定资产加速折旧

### 4.1 一次性扣除政策

| 资产类型 | 金额限制 | 适用期间 |
|---------|---------|---------|
| 设备器具 | 单价500万元以下 | 无时间限制 |
| 特定行业设备 | 单价500万元以上 | 特定行业/地区 |

## 五、亏损弥补与结转

### 5.1 亏损弥补规则

| 亏损类型 | 结转年限 |
|---------|---------|
| 普通亏损 | 5年 |
| 高新技术企业/科技型中小企业 | 10年 |
| 电影行业企业 | 无限期 |
| 防疫企业 | 无限期 |

### 5.2 弥补顺序

1. 本年度税前利润弥补以前年度亏损
2. 当年所得不足弥补的，准许以后年度继续弥补
3. 连续弥补不得超过规定年限

## 六、税收优惠备案与申报

### 6.1 优惠备案要求

| 优惠类型 | 备案方式 | 备查资料 |
|---------|---------|---------|
| 小微优惠 | 自行判断、申报享受 | 留存备查 |
| 高新优惠 | 需先取得高新资质 | 证书+主要证明 |
| 研发加计扣除 | 申报享受 | 辅助账+说明 |

### 6.2 年度汇算清缴时间

| 申报类型 | 截止日期 |
|---------|---------|
| 季度预缴 | 每季度末15日前 |
| 年度汇算 | 次年5月31日前 |
| 关联申报 | 次年5月31日前 |
