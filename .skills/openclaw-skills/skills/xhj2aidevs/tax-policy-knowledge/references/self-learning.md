# 自学习与自动更新机制详解

> 更新时间：2026-04-19 | 版本：v1.2.0

## 一、机制概述

本技能具备**被动触发式自动更新**和**交互式自主学习**两大能力：

- **自动更新**：每次用户交互时，根据对话内容智能判断是否需要联网搜索最新政策，触发知识库增量更新
- **自学习**：根据用户提问内容、反馈纠正、追问方向，持续积累高频问题和最佳答案，补充到知识库

## 二、自动更新触发条件

满足以下任一条件时，自动联网搜索最新政策并更新知识库：

| 触发条件 | 说明 | 优先级 |
|---------|------|--------|
| 月份更替 | 每月1日自动搜索上月新政策 | 中 |
| 申报期关键节点 | 每年1/4/7/10月上旬搜索季度新政策 | 高 |
| 用户主动要求 | 用户说"更新"、"最新政策"、"联网查询"时 | 高 |
| 政策敏感词检测 | 用户提到具体文号、公告名称时，自动核实最新状态 | 高 |
| 指标数据过期 | 用户询问预警指标且数据库版本超过30天未更新 | 中 |

## 三、自动更新搜索策略

### 3.1 搜索优先级

1. **国家税务总局官网** (chinatax.gov.cn) —— 政策原文第一来源
2. **财政部官网** (mof.gov.cn) —— 财税公告来源
3. **中国政府网** (gov.cn) —— 重要政策来源
4. **专业财税平台**（税屋、水5网等）—— 实操解读

### 3.2 搜索关键词组合

```
[月份] 财税政策 最新公告 财政部 税务总局
[月份] 税务风险预警 金税四期 最新指标
[月份] 增值税 企业所得税 最新优惠 申报口径
[月份] 税务风险预警 申报提醒
```

## 四、自学习触发条件

| 触发场景 | 学习内容 | 示例 |
|---------|---------|------|
| 用户纠正 | AI回答被用户指出错误 | "这个不对，应该是..." |
| 技能未命中 | 用户问题未被识别触发 | 多次尝试才命中skill |
| 高频问题 | 同一问题出现3次以上 | 多个用户问同一问题 |
| 追问深化 | 用户追问细节超出知识库 | "能具体说说吗" |
| 用户建议 | 用户主动提供补充资料 | "这个政策还有补充..." |
| 反馈正面 | 用户确认回答有帮助 | "这个很有用" |
| 指标相关 | 用户补充具体案例/数据 | 分享实际稽查案例 |

## 七、脚本使用说明

### 7.1 自学习脚本

```bash
# 查看学习统计
python self_learning.py --stats

# 分析对话学习
python self_learning.py --learn "用户问题" "正确答案"

# 合并已验证学习
python self_learning.py --merge

# 导出学习报告
python self_learning.py --export
```

### 7.2 自动更新脚本

```bash
# 全量更新
python auto_update_knowledge.py

# 月度更新
python auto_update_knowledge.py --monthly

# 风险预警专项更新
python auto_update_knowledge.py --risk

# 按关键词搜索
python auto_update_knowledge.py --policy "增值税"

# 仅检查版本
python auto_update_knowledge.py --check
```

## 八、安全与隐私

### 8.1 安全边界

- **隐私保护**：学习内容不包含用户个人信息
- **可追溯**：所有学习记录有时间戳和来源标识
- **人工可控**：重大更新需人工确认方可生效
- **版本管理**：每次更新记录到 update_log.md

### 8.2 免责声明

自动更新内容为辅助参考，最终内容请以国家税务总局官网为准。
