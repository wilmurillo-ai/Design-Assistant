# 增值税政策详解（2024-2026年）

> 更新时间：2026-04-19 | 政策来源：国家税务总局、财政部

## 一、增值税法施行（2026年1月1日起）

### 1.1 核心配套政策

| 政策文件 | 文号 | 核心内容 |
|---------|------|---------|
| 增值税法 | 全国人大 | 正式确立增值税法律地位 |
| 增值税法实施条例 | 国务院令第826号 | 细化纳税人和征税范围，共6章54条 |
| 增值税优惠政策衔接公告 | 财政部 税务总局公告2026年第10号 | 明确延续和调整的优惠政策 |
| 进口环节增值税优惠政策衔接公告 | 财政部 海关总署 税务总局公告2026年第17号 | 进口环节优惠政策衔接 |

### 1.2 重要变化

- **留抵退税法定化**：存量留抵退税政策由阶段性措施上升为制度性安排
- **进项抵扣范围放宽**：旅客运输服务抵扣范围进一步扩大
- **视同销售范围精简**：部分视同销售行为调整为不征增值税
- **生物制品3%简易征收政策终结**：生物制品恢复13%税率
- **起征点调整为10万元/月**：2026年3月起正式执行（小规模纳税人）

## 二、小规模纳税人优惠政策

### 2.1 延续至2027年12月31日的优惠政策

| 优惠项目 | 具体内容 | 政策依据 |
|---------|---------|---------|
| 免征增值税 | 月销售额10万元以下（含本数） | 2026年第10号公告 |
| 征收率优惠 | 适用3%征收率的减按1%征收 | 2026年第10号公告 |
| 预缴优惠 | 适用3%预征率的减按1%预缴 | 2026年第10号公告 |
| 起征点 | 小规模纳税人起征点为10万元/月 | 增值税法 |

### 2.2 申报要点

1. **季度申报**：小规模纳税人按季度申报增值税
2. **开票规定**：享受免税的，开具普通发票；开具专票按3%征收
3. **未开票收入**：如实填报"未开票销售额"
4. **申报表填写**：
   - 第10行：小规模企业免税销售额
   - 第11行：免税项目销售额
   - 第18行：本期免税额

## 三、一般纳税人政策

### 3.1 简易计税选择权

自2026年1月1日至2027年12月31日，一般纳税人发生符合条件的应税交易，可选择适用简易计税方法。

**适用简易计税的项目（常见）：**
- 公共交通运输服务
- 电影放映服务
- 仓储服务
- 装卸搬运服务
- 收派服务
- 文化体育服务

### 3.2 进项抵扣要点

**可抵扣进项税额：**
- 增值税专用发票
- 海关进口增值税专用缴款书
- 农产品收购发票/销售发票
- 旅客运输服务票据（按9%计算抵扣）
- 收费公路通行费票据（按5%计算抵扣）

**不可抵扣进项税额：**
- 简易计税项目对应进项
- 免征增值税项目对应进项
- 集体福利、个人消费
- 贷款服务、餐饮服务、娱乐服务

## 四、发票管理要点

### 4.1 开票规定

| 开票类型 | 税率/征收率 | 适用范围 |
|---------|-------------|----------|
| 增值税专用发票 | 13%/9%/6%/0% | 一般纳税人之间 |
| 增值税普通发票 | 13%/9%/6%/0% | 所有纳税人 |
| 增值税电子专用发票 | 同专用发票 | 一般纳税人 |
| 增值税电子普通发票 | 同普通发票 | 所有纳税人 |
| 机动车销售统一发票 | 13%/9% | 机动车销售 |
| 二手车销售统一发票 | 3%减按0.5% | 二手车交易 |

### 4.2 勾选抵扣规定

- **勾选期限**：纳税申报前完成勾选确认
- **统计口径**：勾选即统计，申报时抵减
- **撤销勾选**：已统计未申报可撤销，申报后不可撤销

## 五、即征即退/超税负返还

### 5.1 即征即退项目

| 项目 | 退还比例 | 备注 |
|------|---------|------|
| 软件产品 | 税款超过3%的部分 | 动漫企业适用 |
| 管道运输服务 | 税款超过6%的部分 | - |
| 有形动产融资租赁 | 税款超过6%的部分 | - |
| 飞机维修劳务 | 税款超过6%的部分 | - |
| 资源综合利用产品 | 税款超过30%的部分 | 需取得认定资质 |

### 5.2 增值税留抵退税

**适用条件：**
- 连续6个月增量留抵税额均大于零
- 第6个月增量留抵税额不低于50万元
- 纳税信用等级为A级或B级
- 申请前36个月未发生骗取留抵退税、出口退税或虚开增值税专用发票行为

**计算公式：**
```
退还的增量留抵税额 = 增量留抵税额 × 进项构成比例 × 100%
```
