# 缴费计算公式与待遇测算参考

## 一、月缴费金额计算

### 职工社保月缴费公式

```
月缴费基数 = min(max(本人月工资, 基数下限), 基数上限)

个人月缴费 = 月缴费基数 × 个人缴费率
单位月缴费 = 月缴费基数 × 单位缴费率

养老个人 = 基数 × 8%
养老单位 = 基数 × 16%（部分省市14%）
医疗个人 = 基数 × 2% + 3元（大病统筹）
医疗单位 = 基数 × 6%~10%（各地不同）
失业个人 = 基数 × 0.5%
失业单位 = 基数 × 0.5%~1%
```

### 灵活就业人员缴费公式

```
选择缴费基数比例（60%/70%/80%/90%/100%/200%/300%）

缴费基数 = 当地社平工资 × 选择比例

养老保险费 = 缴费基数 × 20%
医疗保险费 = 缴费基数 × 医疗费率（各地约9%~12%）
```

## 二、养老金待遇测算

### 基础养老金公式

```
基础养老金 = (退休时统筹地区上年度在岗职工月均工资 + 本人指数化月均缴费工资) ÷ 2 × 缴费年限 × 1%

本人指数化月均缴费工资 = 退休时当地社平工资 × 本人平均缴费指数
本人平均缴费指数 = 历年缴费基数之和 ÷ (历年当地社平工资之和)
```

### 个人账户养老金

```
个人账户养老金 = 个人账户累计储存额 ÷ 计发月数

计发月数参照表（按退休年龄）：
  50岁：195个月
  55岁：170个月
  60岁：139个月
  65岁：101个月
  70岁：56个月
```

### 过渡性养老金（部分省市有）

```
过渡性养老金 = 退休时统筹地区上年度社平工资 × 本人平均缴费指数 × 1992年底前缴费年限 × 过渡系数

过渡系数：各省市规定不同，一般为1%~1.4%
```

### 月养老金估算示例

**假设条件：**
- 在北京缴费30年，平均缴费指数0.8
- 退休时社平工资：12000元/月

```
基础养老金 = (12000 + 12000×0.8) ÷ 2 × 30 × 1%
           = (12000 + 9600) ÷ 2 × 30 × 1%
           = 10800 × 30 × 1%
           = 3240 元/月

个人账户月均储存（假设缴30年，平均基数8000元）：
个人账户累计 = 8000 × 8% × 12 × 30 = 230400 元
个人账户养老金 = 230400 ÷ 139 ≈ 1658 元/月

月养老金合计 ≈ 3240 + 1658 = 4898 元/月
```

## 三、缴费规划建议框架

### 在职职工

| 情况 | 建议 |
|------|------|
| 收入低于基数下限 | 按下限缴纳，合规最低成本 |
| 收入在正常范围 | 按实际工资缴纳 |
| 收入高于基数上限 | 按上限缴纳（封顶） |
| 换工作有断缴风险 | 以灵活就业身份续保，避免断缴超3个月 |

### 灵活就业人员

| 目标 | 建议缴费档次 |
|------|------------|
| 最低保障 | 60%社平工资（最低档） |
| 平衡性价比 | 80%~100%社平工资 |
| 提高待遇 | 200%~300%社平工资（收益递增但成本高） |
| 临近退休补缴 | 尽量选高档，因计发年限缩短，高缴费见效快 |

### 居民养老保险

| 档次 | 缴费额 | 政府补贴（参考） | 建议人群 |
|------|-------|---------------|---------|
| 低档 | 200元/年 | 40元 | 经济困难人群 |
| 中档 | 500-1000元/年 | 60-80元 | 一般农村居民 |
| 高档 | 2000-5000元/年 | 100元+ | 有经济能力者 |

## 四、断缴影响分析

| 断缴时长 | 养老影响 | 医疗影响 |
|---------|---------|---------|
| 1~2个月 | 累计缴费年限减少 | 当月医疗待遇暂停 |
| 3个月以上 | 需重新满足待遇享受条件 | 各地不同，部分需3~6月等待期 |
| 补缴规则 | 补缴利息按银行同期存款利率 | 部分地区可补缴，部分不可 |

> **提示**：医疗保险断缴影响较大，建议在职期间尽量避免中断；养老保险可后期补缴。

| 1980年+ | 63岁 | 58岁 | 55岁 | 20年 |

> 数据基于2024年9月出台的延迟退休政策，以官方最终公布数据为准。
