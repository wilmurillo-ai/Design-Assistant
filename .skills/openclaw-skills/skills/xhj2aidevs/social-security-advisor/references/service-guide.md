# 社保业务办理与查询指南

## 一、全国统一查询渠道

### 1. 全国社会保障公共服务平台
- **网址**：https://www.12333.gov.cn
- **功能**：个人权益记录查询、参保凭证打印、跨省转移状态查询
- **使用**：实名注册，绑定身份证

### 2. 国家医疗保障局官网
- **网址**：https://www.nhsa.gov.cn
- **功能**：医保政策查询、异地就医备案

### 3. 人力资源和社会保障部
- **网址**：https://www.mohrss.gov.cn
- **功能**：政策发布、退休办理指引

### 4. 全国统一热线
- **12333**：人社部统一热线，可查社保政策、投诉、咨询

## 三、常见业务办理流程

### 3.1 首次参保登记

**在职职工（由用人单位办理）：**
1. 用人单位携带营业执照、组织机构代码证等至当地社保局
2. 提交员工名单、身份证复印件
3. 系统录入，员工激活社保卡

**灵活就业人员：**
1. 携带身份证、户口本（部分城市）至户籍地/居住地社保局
2. 填写《灵活就业人员参保申请表》
3. 选择缴费档次，签订缴费协议
4. 设置银行代扣账户

### 3.2 社保转移接续

**跨省转移流程（养老保险）：**
1. 在新就业地参保，由新参保地社保机构发起转移申请
2. 原参保地审核并转出基金（个人账户余额 + 12%统筹基金）
3. 全程线上，通过12333平台可查询进度
4. 办理时间：约45个工作日

**医疗保险转移：**
- 向新就业地社保机构提出申请
- 原则：就业时参加职工医保，失业时参加居民医保

### 3.3 退休申请流程

1. 达到法定退休年龄前1~3个月，向用人单位或居住地社保局提出申请
2. 提交：身份证、社保卡、个人信息表、档案
3. 社保局审核缴费年限、年龄
4. 审核通过后次月起开始领取养老金
5. 每年进行养老金资格认证（线上认证为主）

### 3.4 异地就医备案

1. 在医保APP（国家医保服务平台）进行长期居住地备案
2. 选择就医地区，勾选就诊医院（三甲优先）
3. 审核通过后，异地就医可直接刷卡报销

## 四、社保卡功能说明

| 功能 | 说明 |
|------|------|
| 身份核验 | 全国统一身份凭证 |
| 医疗结算 | 就医时刷卡结算，自动划扣个人账户 |
| 社保查询 | 可在ATM机/自助终端查询余额及记录 |
| 养老金领取 | 银行账户功能，养老金直接发放 |
| 公共服务 | 部分城市用于坐公交、图书馆等 |

### 社保卡挂失/补卡
- 通过发卡银行APP或社保局网厅办理
- 挂失后次工作日生效

## 五、业务订阅与提醒建议

### 需要关注的重要时间节点

| 时间节点 | 注意事项 |
|---------|---------|
| 每年3-4月 | 上一年度个人所得税汇算清缴，影响次年社保基数申报 |
| 每年6-7月 | 各地发布新社保缴费基数上下限（基于上年度社平工资） |
| 每年7月起 | 新缴费基数正式执行 |
| 每年1月 | 养老金调整方案公布 |
| 退休前6个月 | 开始准备退休申请材料，确认缴费记录 |
| 每年12月 | 居民医保缴费截止，未缴则次年无医保 |

### 建议设置的提醒
1. **6月底**：关注当地社保新基数公告
2. **11月底**：居民医保缴费倒计时
3. **生日前后**：确认延迟退休政策对自己的影响
4. **换工作时**：及时核实新单位社保参保情况

## 六、常见问题FAQ

**Q：社保断缴一个月会有什么影响？**
A：养老保险断缴一个月仅减少一个月累计年限，不影响其他月份；医疗保险断缴后当月医疗待遇暂停，各地处理不同。

**Q：异地工作如何缴纳社保？**
A：一般在工作地参保缴纳，退休时在缴费年限最长的地方或最后一段时间工作地申领。

**Q：个人账户的钱能取出来吗？**
A：正常情况不能取出，仅用于退休后逐月领取，或死亡后继承。特殊情况（出国定居等）可一次性提取。

**Q：缴费基数可以自己选吗？**
A：职工社保由工资决定，灵活就业人员可在60%~300%社平工资范围内选择档位。

**Q：补缴社保需要什么条件？**
A：各地规定不同，一般需用人单位或本人提出申请，按规定缴纳滞纳金，部分地区仅允许单位补缴，个人不能单独补缴养老金。
