# 各省市社保政策详细参考

> 数据以2024-2025年度为准，实际缴费基数上下限以当地人社局最新公告为准。

---

## 📍 北京市

### 缴费基数（2024年度）
- **上限**：35283 元/月
- **下限**：6821 元/月
- **2024年度社平工资**：11761 元/月

### 职工五险缴费比例
| 险种 | 单位 | 个人 |
|------|------|------|
| 养老 | 16% | 8% |
| 医疗 | 10% | 2% + 3元 |
| 失业 | 0.8% | 0.2% |
| 工伤 | 0.2%～1.9% | 0 |
| 生育 | 0.8% | 0 |

### 灵活就业参保
- 养老：20%（缴费基数：社平工资60%～300%）
- 医疗：9.8%（最低按社平60%基数）

### 居民医保（2024年）
- 成人：340元/年
- 学生：160元/年

### 退休待遇
- 2024年养老金调整：涨幅约3%
- 北京市基础养老金标准：全国领先

## 📍 广东省（广州市）

### 广州市缴费基数（2024年度）
- **上限**：29559 元/月
- **下限**：2300 元/月（最低工资标准）
- **社平工资**：9853 元/月

### 职工五险缴费比例
| 险种 | 单位 | 个人 |
|------|------|------|
| 养老 | 14% | 8% |

| 失业 | 0.7% | 0.3% |
| 工伤 | 0.2%～1.9% | 0 |
| 生育 | 0.5% | 0 |

> 广东省养老单位缴费比例由16%降至14%（阶段性政策延续）

### 深圳市特殊规定
- 深圳医保分一档、二档
- 一档（全覆盖）：单位 6.2% + 个人 0.5%
- 二档：单位 5.2% + 个人 0%
- 深圳最低工资：2360元/月（2024年）
- 深圳社保基数下限：2360元，上限：23088元
