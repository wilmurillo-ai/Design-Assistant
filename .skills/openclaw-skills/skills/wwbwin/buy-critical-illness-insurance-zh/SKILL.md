# 买重大疾病保险不踩雷

基于中国保险行业协会官方发布的《中国人身保险业重大疾病经验发生率表（2020）编制报告》，
通过交互式心理测试，分析个人风险画像，输出专业精算角度的重疾险购买建议，
最后用消费心理学视角给出直白的购买建议。

## 核心能力

1. **交互式风险评估** - 通过 15-20 道心理测试题，收集用户风险因素
2. **多维风险分析** - 整合性别、地域、职业、生活习惯、经济环境、遗传因素
3. **精算数据支撑** - 引用 2020 版重疾表官方统计数据
4. **双重建议输出** - 专业精算建议 + 消费心理学直白建议

## 工作流程

```
用户触发技能
  → 启动交互式问卷（15-20题）
  → 收集：基础信息 + 生活方式 + 职业风险 + 经济环境 + 遗传因素
  → 查询实时经济形势数据（可选）
  → 计算个人重疾风险评分
  → 匹配 2020 重疾表数据
  → 生成个性化购买建议报告（专业版 + 直白版）
```

## 综合风险评分计算

```
综合风险系数 = 基础系数 × 生活方式系数 × 职业系数 × 健康系数 × 经济系数

其中：
- 生活方式系数 = 睡眠 × 吸烟饮酒 × 运动 × 饮食 × 久坐
- 职业系数 = 工作性质 × 工作压力 × 环境因素
- 健康系数 = BMI × 慢性病 × 遗传病史 × 体检情况
- 经济系数 = 收入稳定性 × 负债压力

风险等级划分：
- 低风险：综合系数 < 1.5
- 中低风险：1.5 ≤ 综合系数 < 2.5
- 中风险：2.5 ≤ 综合系数 < 4.0
- 高风险：4.0 ≤ 综合系数 < 6.0
- 极高风险：综合系数 ≥ 6.0
```

## 使用说明

### 启动方式

用户输入以下任一关键词触发：
- "我要买重疾险"
- "帮我分析重疾险"
- "重大疾病保险建议"
- "保险避坑指南"
- "重疾险推荐"

### 交互流程

1. **问候与说明** - 介绍技能目的和数据来源
2. **逐题询问** - 每道题提供选项，用户回复字母或数字
3. **实时计算** - 后台累计风险系数
4. **数据查询**（可选）- 如用户同意，查询当前经济形势
5. **生成报告** - 输出完整报告（专业版 + 直白版）

### 注意事项

1. **免责声明**：每次使用必须输出"仅供参考，不构成购买建议"声明
2. **数据来源**：明确标注数据来自《中国人身保险业重大疾病经验发生率表（2020）》
3. **隐私保护**：不存储用户个人健康信息
4. **专业边界**：不涉及具体产品推荐，仅提供配置建议框架
5. **双重输出**：必须同时输出专业精算建议和消费心理学直白建议
