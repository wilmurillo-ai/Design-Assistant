# 重大疾病距离模拟器 | Disease Distance Simulator

> "看清自己的健康未来，比什么都重要。"

## 核心理念

**从"冰冷概率"转向"可视化的健康未来"**

传统健康评估：填表 → 算概率 → 给结论 → 用户一脸懵  
**本技能**：对话 → 画像建模 → 时间轴可视化 → 教育性建议 → 用户自主决策

用户喜欢看"可视化未来"，不喜欢"冰冷概率"。本技能用AI+RAG把精算数据变成"你的重大疾病还有X年预警期"的可感知信息。

---

## 工作流程

```
用户触发（自然语言，无需填表）
  ↓
【阶段1：渐进式画像解析】
  Agent通过自然对话逐步收集：
  - 基础画像：性别、年龄、地域
  - 生活方式：睡眠、饮食、运动、吸烟饮酒、久坐
  - 职业环境：工作类型、压力、出差/夜班
  - 健康因素：BMI、慢性病、家族史、体检情况
  ↓
【阶段2：RAG实时数据查询】
  基于画像，查询：
  - 2020重疾表：年龄段性别发生率
  - 2025生命表：预期寿命、死亡率
  - 国家癌症中心数据：地域癌症谱
  - GBD流行病学数据：风险因素权重
  ↓
【阶段3：AI风险预测建模】
  大模型综合分析：
  - 基础发生率 × 生活方式系数 × 职业系数 × 健康系数
  - 生成"疾病距离"时间轴
  - 识别高发疾病类型
  ↓
【阶段4：2段式教育报告输出】
  ① 距离可视化：你的重大疾病预计还有X年"预警期"
  ② 身体信号+缓解行动清单：个性化、可执行
```

## 交互设计：渐进式自然对话

### 触发方式

用户输入任一自然语言即可触发：
- "帮我看看我离重大疾病有多远"
- "我的健康生命线还有多长"
- "我想知道我的重疾风险"
- "帮我做个疾病距离模拟"

### 对话引导策略

**不要求一次性填完表单**，而是通过自然对话逐步获取信息：

```
【示例对话】

用户: 帮我看看我离重大疾病有多远

Agent: 好的，让我帮你做个"疾病距离"模拟。先问几个简单问题：
       你是男士还是女士？今年大概多少岁？

用户: 男，45岁，做销售的

Agent: 收到。45岁男性，销售工作——这个职业比较辛苦，应酬多、压力大。
       你目前在哪个城市？平时的生活习惯怎么样？比如睡眠、运动、吸烟喝酒这些？

用户: 北京，睡眠一般6小时，很少运动，偶尔喝酒

Agent: 北京一线城市，工作节奏快。6小时睡眠偏少，偶尔喝酒还好。
       你身高体重多少？家里父母或兄弟姐妹有没有什么慢性病史？比如高血压、糖尿病、癌症这些？

用户: 175cm，85kg，我爸有高血压，其他没什么

Agent: BMI约27.8，属于超重。父亲有高血压，这个有遗传倾向。
       最后问一下：你最近两年有做过体检吗？有没有查出什么异常指标？

用户: 去年体检说血脂偏高，其他还好

Agent: 好的，信息收集完毕。正在为你生成"疾病距离模拟报告"...
       [生成报告中...]
```

### 信息收集清单

| 维度 | 关键信息 | 对话中自然获取 |
|------|---------|---------------|
| 基础画像 | 性别、年龄、常驻城市 | ✅ |
| 生活方式 | 睡眠、运动、吸烟饮酒、饮食习惯、久坐时长 | ✅ |
| 职业环境 | 工作类型、压力水平、出差/夜班频率 | ✅ |
| 健康因素 | 身高体重(BMI)、慢性病史、家族史、体检异常 | ✅ |

> **交互原则**：像朋友聊天一样，不要像审问犯人。如果用户主动提供信息，直接记录；如果没有，自然地询问。

## 数据底座：RAG实时查询

### 核心数据源

| 数据源 | 提供信息 | 更新频率 |
|--------|---------|---------|
| **《中国人身保险业重大疾病经验发生率表（2020）》** | 6病种/25病种/28病种分年龄性别发生率 | 静态 |
| **《中国人身保险业经验生命表(2025)》** | 预期寿命、死亡率、生存概率 | 2025年新发布 |
| **国家癌症中心统计** | 分省份癌症发病率、死亡率、癌谱分布 | 年度更新 |
| **GBD全球疾病负担数据** | 风险因素权重、慢性病流行病学数据 | 年度更新 |

### RAG查询策略

当用户画像确定后，Agent通过联网搜索实时查询：

```
查询示例：
1. "45岁男性 重大疾病发生率 2020重疾表"
2. "北京 肺癌发病率 2025年统计"
3. "销售职业 健康风险 久坐 心血管"
4. "BMI 27 高血压家族史 心血管风险"
```

**为什么用RAG而不是硬编码？**
- 硬编码数据会过时，RAG实时查询确保最新
- 不同地域癌症谱差异大（如沿海肝癌高发、内陆食管癌高发）
- 新版生命表（2025版）刚发布，旧版数据已不再适用

## AI风险预测模型

### 风险系数计算

```
综合风险指数 = 基础发生率 × 生活方式系数 × 职业系数 × 健康系数 × 地域系数

其中：
- 基础发生率：来自2020重疾表，分年龄性别
- 生活方式系数：睡眠×运动×吸烟饮酒×饮食×久坐
- 职业系数：工作类型×压力×环境因素
- 健康系数：BMI×慢性病×家族史×体检异常
- 地域系数：来自国家癌症中心分省份数据
```

### "疾病距离"计算逻辑

```
疾病距离（年）= 预期发病年龄 - 当前年龄

预期发病年龄估算：
- 基于重疾发生率曲线，找到累计发病率达到50%的年龄点
- 根据个人风险系数调整（高风险提前，低风险延后）
```

### 预警期分级

| 预警期 | 含义 | 建议 |
|--------|------|------|
| >20年 | "远期预警" | 保持健康生活方式，定期体检 |
| 10-20年 | "中期预警" | 关注风险因素，考虑专项筛查 |
| 5-10年 | "近期预警" | 加强健康管理，咨询专业医生 |
| <5年 | "紧急预警" | 立即就医检查，加强健康管理 |

## 技术实现要点

### 1. 渐进式信息收集

- 使用对话状态追踪，记录已收集的信息
- 不强制一次性收集所有信息，允许用户分多次对话完成
- 如果用户主动提供额外信息，及时更新画像

### 2. RAG实时查询

- 基于用户画像生成搜索关键词
- 调用联网搜索API获取最新数据
- 对搜索结果进行信息提取和整合

### 3. 风险预测模型

- 基于重疾表数据建立基础发生率模型
- 叠加生活方式、职业、健康因素系数
- 生成"疾病距离"时间轴可视化

### 4. 报告输出

- 纯健康教育导向，不涉及任何产品推荐
- 以数据可视化帮助用户理解自身健康风险
