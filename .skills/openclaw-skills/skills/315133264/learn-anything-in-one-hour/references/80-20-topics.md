# Pre-curated 80/20 Key Point Lists for Common Topics

## Python (beginner)
Core logic: Python is a high-level readable programming language used for automation, data work, and web apps.
20% key points covering 80% usage:
1. Variables & basic data types (int, str, float, bool, list, dict)
2. Control flow (if/else, for loops, while loops)
3. Functions (def, parameters, return values)
4. Basic input/output (print(), input(), reading/writing text files)
5. Importing standard libraries (like math, random, os)

## Git
Core logic: Git is a version control system that tracks changes to files and lets you collaborate with others.
20% key points covering 80% usage:
1. git init / git clone: Create or download a repository
2. git add / git commit: Save changes locally
3. git push / git pull: Sync changes with remote repo
4. git status: Check current state of your files
5. git checkout: Switch branches or restore files

## HTML
Core logic: HTML is the markup language that defines the structure of web pages.
20% key points covering 80% usage:
1. Basic document structure (!DOCTYPE, html, head, body tags)
2. Common content tags (h1-h6, p, a, img, ul/ol/li, div, span)
3. Tag attributes (href for links, src for images, class/id for styling)
4. Form basics (form, input, button tags)
5. Semantic tags (header, footer, section, nav)
