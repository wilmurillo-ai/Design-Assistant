# Poster Templates Reference

## Event Poster Template

### Description
General-purpose event poster with clear information hierarchy.

### Layout
- **Header**: Event name (large, bold)
- **Subheader**: Tagline or event type
- **Date/Time**: Prominent placement
- **Location**: Below date or in footer
- **Visual**: Central illustration or photo
- **Footer**: Tickets/RSVP info, sponsors, social

### Default Prompt Template

```
A professional event poster design, [STYLE], featuring:
- Central [IMAGE_SUBJECT] illustration
- Clean typography for event details
- [COLOR_SCHEME] color palette
- Balanced composition with clear visual hierarchy
- Modern event poster aesthetic suitable for digital and print
```

### Variables

| Variable | Description | Example |
|----------|-------------|---------|
| `eventName` | Main event title | "Summer Music Festival" |
| `eventType` | Category | "Concert", "Conference", "Workshop" |
| `date` | Event date | "July 15, 2026" |
| `time` | Event time | "7:00 PM" |
| `venue` | Location | "Madison Square Garden" |
| `style` | Visual style | "modern minimalist", "bold colorful", "elegant" |
| `imageSubject` | Central visual | "musical instruments", "city skyline", "abstract patterns" |
| `colorScheme` | Colors | "deep blues and golds", "vibrant neons on black" |

## Concert/Music Poster Template

### Description
Music-focused poster with band/artist emphasis.

### Layout
- **Hero**: Band/artist name (largest element)
- **Visual**: Performance imagery or album art style
- **Details**: Date, venue, ticket info
- **Setlist/Features**: Supporting acts, special guests
- **Footer**: Socials, streaming links

### Default Prompt Template

```
A dynamic concert poster in [STYLE], featuring:
- [GENRE]-inspired visual elements
- [IMAGE_SUBJECT] as central artwork
- Bold typography suitable for band promotion
- [COLOR_SCHEME] with high contrast
- Energetic composition that conveys live music atmosphere
- Music poster aesthetic inspired by [REFERENCE_STYLE]
```

### Variables

| Variable | Description | Example |
|----------|-------------|---------|
| `bandName` | Main artist/band | "The Midnight Echo" |
| `genre` | Music genre | "indie rock", "jazz", "electronic" |
| `venue` | Concert venue | "The Fillmore" |
| `referenceStyle` | Inspiration | "vintage rock posters", "modern electronic", "jazz age" |
| `imageSubject` | Central visual | "electric guitar", "vinyl records", "sound waves" |

## Product Showcase Template

### Description
E-commerce style product presentation.

### Layout
- **Hero**: Product image (high quality, centered)
- **Headline**: Product name + tagline
- **Features**: 3-4 key benefits
- **Price**: Prominent CTA area
- **Trust**: Reviews, guarantees, shipping info

### Default Prompt Template

- Professional product photography style composition
- [BACKGROUND_TYPE] background
- Elegant lighting highlighting product features
- [COLOR_SCHEME] color palette

### Layout

- **Discount**: Percentage or amount off (largest text)
- **Visual**: Products or abstract sale imagery
- **Details**: Dates, exclusions, terms

## Typography Guidelines

### Font Pairings (for text overlay)

- **Modern**: Sans-serif headline + Sans-serif body
- **Classic**: Serif headline + Sans-serif body
- **Bold**: Heavy sans-serif headline + Light sans-serif body
- **Elegant**: Script accent + Serif body

### Sizing Hierarchy

1. **Hero text**: 48-72pt (event name, product name)
2. **Secondary**: 24-36pt (date, price, tagline)
3. **Tertiary**: 14-18pt (details, fine print)
4. **Accent**: 12-14pt (socials, contact)

## Composition Patterns

### Rule of Thirds
Place key elements at intersection points.

### Center Focus
Ideal for product showcases, symmetrical designs.

### Diagonal Energy
Creates movement, good for concerts and sales.

### Frame Within Frame
Draws eye to center, adds depth.

### Z-Pattern
Guides eye: headline → image → CTA (Western reading).

### F-Pattern
Top-heavy information hierarchy.
