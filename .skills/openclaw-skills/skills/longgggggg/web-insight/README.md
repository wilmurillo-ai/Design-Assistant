# 互联网内容洞察 (WebInsight)

面向全域网络信息监测与情报分析场景，提供多条件组合精准检索能力，可覆盖全网新闻、社交、资讯等多渠道公开内容。

支持关键词、情感、时间、平台等多条件组合查询，依托 NLP 智能解析，实现内容去重、关键信息抽取与数据结构化输出，一键供 AI Agent 调用。适用于品牌管理、市场分析、竞品追踪、风险感知等场景，高效提炼高价值信息，为业务决策提供数据支撑。

## 功能

- **舆情搜索** - 通过自然语言查询全网舆情数据
- **时间解析** - 智能解析"最近3天"、"上周"等时间表达
- **地区解析** - 支持省市区名称到国标区域代码的转换
- **多平台覆盖** - 抖音、微博、小红书、微信、B站等50+媒体平台

## 配置

### 1. 获取 API Key

前往 [https://www.feedax.cn](https://www.feedax.cn) 申请 API Key。

### 2. 配置环境变量

本技能**必须**配置 Feedax API Key；可参考仓库中的 `.env.example`，在项目根目录创建 `.env`（该文件已在 `.gitignore` 中，**请勿**将含真实 Key 的 `.env` 提交到 Git）：

```env
FEEDAX_SEARCH_API_KEY=your_api_key_here
```

每人应使用自己在 [feedax.cn](https://www.feedax.cn) 申请的 Key，勿共用以免配额耗尽或泄露。

## 使用示例

安装后，直接用自然语言与智能体对话：

- "搜索最近3天关于新能源汽车的舆情"
- "查找微博上关于iPhone的负面评论"
- "检索江苏省医疗相关的热门内容"
