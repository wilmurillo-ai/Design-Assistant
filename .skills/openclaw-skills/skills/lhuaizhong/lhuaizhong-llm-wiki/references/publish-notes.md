# Publish Notes

Use this wording when presenting the skill externally.

## One-line pitch

Turn scattered notes, docs, transcripts, and chat history into a compounding Markdown wiki that an LLM can continuously maintain.

## What makes it different

- Local-first and Markdown-native
- Obsidian-friendly
- Uses compiled wiki pages instead of one-shot document retrieval
- Explicit ingest, query, reindex, and lint workflows
- Keeps provenance and cross-links in human-readable files

## Typical use cases

- personal second-brain / knowledge vault
- project memory across many sessions
- research synthesis across heterogeneous sources
- team knowledge bases backed by readable Markdown
