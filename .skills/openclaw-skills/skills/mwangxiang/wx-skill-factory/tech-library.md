# Skill Factory 技术库

> **配置模板**：请根据你自己的 AI 服务填写具体的密钥和地址。
> 此文件包含 API 密钥等敏感信息，请勿提交到公开仓库。

---

## 使用说明

skill-factory 运行时会读取此文件获取 AI 服务的 API 配置。

**配置规则**：
- 所有服务统一使用 OpenAI 兼容格式
- 不同密钥对应不同模型族，按密钥分组配置
- 调用时根据任务类型选择对应的密钥 + 模型组合
- 选择逻辑详见 `call-guide.md` 的场景-AI 对照表

---

## 统一入口

| 配置项 | 值 |
|--------|-----|
| 主力站 | `https://<你的主力API地址>` |
| 主力 endpoint | `https://<你的主力API地址>/v1/chat/completions` |
| 备用站 | `https://<你的备用API地址>`（可选） |
| 备用 endpoint | `https://<你的备用API地址>/v1/chat/completions` |
| 协议 | OpenAI 兼容格式 |

## 默认降级链

```
任务进来 → 根据 call-guide.md 场景表选择密钥+模型
│
├─ 首选组合调用成功 → 返回结果
└─ 首选失败（连续2次）→ 降级到同组备选模型
    └─ 仍失败 → 降级到其他密钥组的对应模型
        └─ 全部失败 → AI 自身执行
```

**降级路径示例**：
- 写作任务：主力 GPT → 备用 GPT → Gemini → AI 自身
- 素材整理：主力 Gemini → 备用 Gemini → AI 自身
- 代码任务：Codex → 备用代码模型 → AI 自身
- 高质量输出：Claude 首选 → Claude 备选 → AI 自身
