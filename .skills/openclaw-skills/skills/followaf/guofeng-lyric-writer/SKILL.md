# Guofeng Lyric Writer

Create authentic traditional Chinese-style (Guofeng) lyrics with proper classical vocabulary, imagery, rhyme, and poetic structure.

## Quick Start

```text
User: "帮我写一首关于江南的国风歌词"
AI: [Reads references/imagery.md → Creates lyrics using proper imagery and vocabulary]
```

## Core Workflow

### 1. Understand Request
Identify: theme (离别/思念/山水/历史), emotion, story context

### 2. Select Imagery
- Load [references/imagery.md](references/imagery.md) for theme-appropriate imagery
- Choose 3-5 core images that work together

### 3. Apply Vocabulary
- Use [references/vocabulary.md](references/vocabulary.md) for word substitutions
- Transform modern language into classical expression

### 4. Structure Rhyme & Rhythm
- Consult [references/rhyme.md](references/rhyme.md) for rhyme scheme selection
- Apply appropriate sentence length patterns

### 5. Compose Lyrics
Follow [references/workflow.md](references/workflow.md) for complete creation process

### 6. Polish & Refine
Check emotional depth, rhythm flow, and classical authenticity

## Quality Standards

- **Authenticity**: Use genuine classical vocabulary, avoid forced ancient style
- **Coherence**: Images should connect and build unified atmosphere
- **Musicality**: Rhyme naturally, match melodic rhythm
- **Emotional Truth**: Express sincere feelings, not empty decoration

## Avoid Common Mistakes

- ❌ Over-stacking obscure references → incomprehensible lyrics
- ❌ Excessive classical language → loses modern accessibility
- ❌ Forced ancient style without real emotion → hollow beauty
- ❌ Ignoring rhythm → unsingable lyrics
