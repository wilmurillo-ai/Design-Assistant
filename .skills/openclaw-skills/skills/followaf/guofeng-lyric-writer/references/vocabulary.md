## Modern → Classical Translation Examples

### Love Expressions
```
Modern: "我真的很喜欢你，每天都想你"
Classical: "吾甚悦君，日日思君，夜夜念君"

Modern: "我们在一起很幸福"
Classical: "与君相伴，欢愉满怀，怡然自得"

Modern: "我会一直爱你，直到永远"
Classical: "此心属君，矢志不渝，直至白头"
```

### Longing Expressions
```
Modern: "我很想念你，想见你"
Classical: "思君如满月，夜夜减清辉，愿见君颜"

Modern: "每天都在想你，晚上做梦都梦见你"
Classical: "日日思君，夜夜梦君，梦里相见"

Modern: "我们分开很久了，我还在等你"
Classical: "离别已久，仍伫立候君，愿君早归"
```

### Sadness Expressions
```
Modern: "我很难过，因为你要走了"
Classical: "伤怀不已，缘君将离去，愁绪满怀"

Modern: "离开你我很孤独"
Classical: "离君而去，孤独寂寞，独倚栏杆"

Modern: "我很伤心，眼泪都流出来了"
Classical: "伤心欲绝，泪湿罗衫，泣不成声"
```

### Beauty Expressions
```
Modern: "你很美，我很喜欢"
Classical: "君容貌绝世，美如画卷，吾甚悦君"

Modern: "这里风景很美"
Classical: "此地风景如画，美不胜收"

Modern: "月亮很美，照亮了夜空"
Classical: "皎月如银，照亮夜空，美轮美奂"
```

### Time Expressions
```
Modern: "时间过得真快"
Classical: "光阴荏苒，岁月如梭，不觉已过数载"

Modern: "我们认识很久了"
Classical: "相识已久，往来数载，情谊深厚"

Modern: "以后我们会再见面"
Classical: "来日方长，必有相见之时"
```

## Usage Guidelines

### 1. Choose Appropriate Substitution
- Consider context and tone
- Match classical word's original meaning
- Maintain natural flow

### 2. Avoid Over-Translation
- Don't substitute every word
- Keep some modern elements for accessibility
- Balance classical and modern

### 3. Context Matters
- "君" fits romantic context
- "汝" fits conversation
- "吾" fits self-reference in narration

### 4. Emotional Matching
- Use "思" for gentle longing
- Use "念" for intense missing
- Use "忆" for past memories

### 5. Combine with Imagery
- Word substitution enhances imagery
- "思君如满月" combines substitution with imagery
- Create multi-layered expression
