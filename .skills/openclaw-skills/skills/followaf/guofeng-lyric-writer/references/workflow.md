## Phase 1: Theme Understanding

### Step 1.1: Identify Core Theme
Ask user or determine from context:

| Theme Category | Key Concepts |
|----------------|--------------|
| 离别 (departure) | 天涯、断肠、归雁、城门、离别、等待 |
| 思念 (longing) | 相思、红豆、月、梦、念君、天涯 |
| 山水 (landscape) | 江河、云雾、清风、明月、山水、亭台 |
| 历史 (history) | 朝代、古城、战场、将军、帝王、沧桑 |
| 爱情 (love) | 相思、缘分、誓言、红豆、月、琴瑟 |
| 孤独 (solitude) | 独、孤、寂寞、孤灯、空楼、夜深 |
| 春天 (spring) | 春风、春雨、桃花、杨柳、燕子、春光 |
| 秋天 (autumn) | 秋风、落叶、秋雨、霜、雁、愁绪 |

### Step 1.2: Determine Emotional Tone
- **Intensity**: Gentle (思念/山水) vs Strong (断肠/离别)
- **Direction**: Hopeful (等待/春天) vs Sad (离别/秋天)
- **Depth**: Surface (景色描写) vs Deep (情感表达)

### Step 1.3: Clarify Story Context (if applicable)
- Historical background?
- Personal story?
- Myth/legend adaptation?
- Pure emotion expression?

## Phase 2: Imagery Selection

### Step 2.1: Load Imagery Dictionary
Read [references/imagery.md](references/imagery.md)

### Step 2.2: Select Core Images (3-5)

**Selection Criteria**:
| Criterion | Check |
|-----------|-------|
| Relevance | Does it match theme? |
| Coherence | Do images work together? |
| Emotion | Does it express intended feeling? |
| Visual | Can listeners imagine it? |

**Example Selection for "离别主题"**:
1. 月 (思念、永恒)
2. 雨 (忧伤、离别)
3. 城门 (分离、等待)
4. 归雁 (书信、回归)
5. 天涯 (遥远、分离)

### Step 2.3: Organize Image Flow
Determine image appearance order:
```
Verse: Background setting images (月、雨、城门)
Chorus: Core emotional images (天涯、归雁)
Outro: Lingering image (月 - 循环回归)
```

## Phase 3: Vocabulary Transformation

### Step 3.1: Load Vocabulary Dictionary
Read [references/vocabulary.md](references/vocabulary.md)

### Step 3.2: Transform Key Words

**Before**: "我在城门口等你，看着月亮，心里很想你"
**After**: "伫立城门候君归，仰望明月，思君如流水"

| Modern | Classical | Applied |
|--------|-----------|---------|
| 我 | 伫立/吾 | 伫立 (standing + waiting) |
| 等 | 候 | 候君归 |
| 看 | 仰望/望 | 仰望明月 |
| 很想你 | 思君如流水 | 思君如流水 |

### Step 3.3: Balance Classical & Modern
- **70% classical vocabulary** in verses
- **50% classical vocabulary** in chorus (more accessible)
- **Keep emotional words natural** (爱、思念 can stay modern-ish)

### Step 3.4: Avoid Over-Translation
❌ "吾伫立于城门之侧，翘首以望明月之辉，思君之情如流水之长"
- Too dense, unnatural

✅ "伫立城门候君归，仰望明月，思君如流水"
- Balanced, readable

## Phase 4: Structure Design

### Step 4.1: Determine Song Structure

| Section | Purpose | Length |
|---------|---------|--------|
| Verse 1 (主歌1) | Set scene, introduce theme | 4-6 lines |
| Verse 2 (主歌2) | Develop story/emotion | 4-6 lines |
| Chorus (副歌) | Emotional peak, core message | 4-8 lines |
| Bridge (过渡) | Turn, new perspective | 2-4 lines |
| Outro (结尾) | Resolve, lingering feeling | 2-4 lines |

### Step 4.2: Assign Rhyme Scheme
Read [references/rhyme.md](references/rhyme.md) for rhyme selection

**Example Structure**:
```
Verse 1: 尤韵 (楼、愁、秋) - Set melancholy tone
Verse 2: 尤韵 - Continue tone
Chorus: 侵韵 (心、琴、深) - Emotional depth
Bridge: 寒韵 (山、难、关) - New perspective
Outro: 尤韵 - Return to opening tone
```

### Step 4.3: Design Sentence Length Pattern

```
Verse: 4字 + 6字 (叙事铺垫)
  "明月照高楼，相思无处寄"

Chorus: 5字 + 7字 (情感高潮)
  "相思入我心，离别意难申"

Bridge: 变化句式 (转折)
  "天涯两端，各守一方"
```

## Phase 5: Lyric Composition

### Step 5.1: Write Verse 1

**Purpose**: Establish setting and mood

**Template**:
```
[背景意象] + [场景描写]
[时间/季节] + [动作]
[情感铺垫]
```

**Example**:
```
明月照高楼（背景：月、楼）
流光正徘徊（场景描写）
上有愁思妇（人物：思妇）
悲叹有余哀（情感铺垫）
```

### Step 5.2: Write Verse 2

**Purpose**: Develop story/emotion

**Template**:
```
[延续意象]
[情感加深]
[时间流逝/变化]
[期待/等待]
```

**Example**:
```
秋风起寒山（延续：秋、山）
落叶满庭院（场景变化）
日日思君颜（情感加深）
夜夜梦相见（等待梦境）
```

### Step 5.3: Write Chorus

**Purpose**: Emotional peak, core message

**Template**:
```
[核心意象] + [核心情感]
[情感高潮句]
[反复句] (optional)
[情感收束]
```

**Example**:
```
相思红豆新（核心：相思、红豆）
离别愁断魂（情感高潮）
天涯两端连（核心：天涯、分离）
缘分一线牵（情感收束）
```

### Step 5.4: Write Bridge

**Purpose**: Turn perspective, new angle

**Template**:
```
[转折意象]
[新视角]
[情感转变]
```

**Example**:
```
孤雁度关山（转折：雁、关山）
书信无处传（新视角）
唯有明月知（情感转变）
```

### Step 5.5: Write Outro

**Purpose**: Resolve, lingering feeling

**Template**:
```
[回归意象]
[情感收束]
[留白/余韵]
```

**Example**:
```
明月照楼头（回归：月、楼）
相思夜夜愁（收束）
君归何时有（余韵/疑问）
```

## Phase 6: Polish & Refine

### Step 6.1: Check Rhyme Flow
- Are rhymes natural?
- Do end words match intended rhyme scheme?
- Is there variety or too repetitive?

### Step 6.2: Check Tonal Balance
- Avoid too many consecutive same tones
- Line endings favor 平声
- Match rhythm to likely melody

### Step 6.3: Check Imagery Coherence
- Do images build unified atmosphere?
- Are there conflicting images?
- Is visual flow clear?

### Step 6.4: Check Emotional Depth
- Is emotion sincere, not forced?
- Does it progress naturally?
- Does chorus reach emotional peak?

### Step 6.5: Check Readability
- Can modern listeners understand?
- Is classical vocabulary balanced?
- Is flow natural for singing?

### Step 6.6: Final Polish
- Adjust word choice for better rhyme
- Simplify over-dense classical language
- Strengthen weak emotional lines
- Add parallelism if appropriate

## Common Mistakes to Avoid

### Content Mistakes
❌ **Over-stacking obscure references**: 堆砌生僻典故
❌ **Excessive classical language**: 过度文言难懂
❌ **Forced ancient style without emotion**: 形式复古但情感空洞
❌ **Ignoring modern listener**: 完全不考虑现代理解度

### Technical Mistakes
❌ **Mixed rhyme in same section**: 同一段混用韵脚
❌ **Wrong tone at line end**: 句末用仄声难押韵
❌ **Unnatural rhythm**: 韵律不符合演唱节奏
❌ **Ignoring melody structure**: 不考虑歌曲旋律结构

### Expression Mistakes
❌ **Fake emotion**: 情感虚假不真挚
❌ **Vulgar language**: 语言粗俗破坏美感
❌ **Conflicting imagery**: 意象矛盾破坏意境
❌ **Stiff expression**: 表达生硬不流畅

## Quality Checklist

After composition, verify:

✅ **Authenticity**: 真正的古典词汇，不是生硬复古
✅ **Coherence**: 意象连贯，统一氛围
✅ **Musicality**: 韵律自然，适合演唱
✅ **Emotional Truth**: 情感真挚，有感染力
✅ **Accessibility**: 现代听众能理解
✅ **Structure Clear**: 主歌副歌过渡清晰
✅ **Imagery Visual**: 有画面感
✅ **Rhyme Natural**: 韵脚自然不牵强
