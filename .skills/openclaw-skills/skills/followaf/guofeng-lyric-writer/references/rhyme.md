## Rhyme Scheme Basics

### When to Change Rhyme
- **Within verse**: Keep same rhyme throughout one verse
- **Between verses**: Can switch rhyme for contrast
- **Chorus/bridge**: Often uses different rhyme from verses

### Rhyme Placement
- **End rhyme**: Last word of each line (most common)
- **Internal rhyme**: Mid-line words also rhyme (adds density)
- **Alternate rhyme**: Lines 1-3 rhyme, lines 2-4 rhyme differently

## Common Rhyme Categories

### Level Tone Rhymes (平声韵)

#### 东韵 - Grand, Open Scenes
**韵脚字**: 东、风、空、同、中、通、逢、匆、浓、重、梦、容、红、穷、封、松、葱、龙、融、丰、逢
**适用场景**: 山水辽阔、气势宏大、自然景观
**示例**:
```
明月照高楼（楼）
流光正徘徊（徊）✗
上有愁思妇（妇）✗
悲叹有余哀（哀）✗
---
正确示例：
清风拂杨柳（柳-尤韵）
明月照高楼（楼-尤韵）
相思寄红豆（豆-尤韵）
离别添新愁（愁-尤韵）
```

#### 江韵 - Water, Longing Scenes
**韵脚字**: 江、阳、长、香、堂、张、望、凉、霜、乡、芳、伤、央、墙、窗、扬、航、忙、茫、肠、方、狂、郎、浪
**适用场景**: 水景、思念、离别、家乡
**示例**:
```
春江花月夜（夜-陌韵）
明月照江阳（阳-江韵）
离人望故乡（乡-江韵）
相思夜夜长（长-江韵）
```

#### 支韵 - Thinking, Memory Scenes
**韵脚字**: 支、诗、时、思、知、迟、痴、姿、辞、私、疑、离、枝、儿、吹、飞、归、悲、眉、违、围、微、衣、期、棋、奇、骑、宜、仪、移、仪、遗、疑、怡、漪
**适用场景**: 思念、回忆、情感抒发
**示例**:
```
夜深人静时（时-支韵）
相思无处寄（寄-支韵）
独坐思往事（事-支韵）
泪湿青衫衣（衣-支韵）
```

#### 微韵 - Parting, Waiting Scenes
**韵脚字**: 微、飞、归、悲、眉、违、围、衣、期、非、扉、菲、霏、妃、辉、挥、徽、威、追、炊、吹、谁、随、垂、为、维
**适用场景**: 离别、等待、思念
**示例**:
```
春风吹柳絮（絮-御韵）
落花随流水（水-微韵）✗
---
正确示例：
明月照谁归（归-微韵）
相思几多悲（悲-微韵）
泪湿罗衣飞（飞-微韵）
```

#### 真韵 - Emotion, People Scenes
**韵脚字**: 真、人、身、门、痕、尘、晨、春、亲、邻、贫、巾、津、新、心、闻、文、分、纷、芬、焚、魂、温、存、恩、根、昏、坤、伦、轮、论
**适用场景**: 真情、人事、人物描写
**示例**:
```
相思红豆新（新-真韵）
离别愁断魂（魂-真韵）
梦里见君身（身-真韵）
醒来泪满痕（痕-真韵）
```

#### 元韵 - Time, Fate Scenes
**韵脚字**: 元、天、烟、连、缘、年、前、千、山、间、颜、闲、寒、还、欢、关、湾、盘、班、般、残、颜、全、权、宣、悬、弦、船、川、穿、传
**适用场景**: 时间流逝、缘分命运
**示例**:
```
相思已三年（年-元韵）
缘分天注定（定-径韵）✗
---
正确示例：
相思已三年（年-元韵）
梦里见君颜（颜-元韵）
天涯两端连（连-元韵）
缘分一线牵（牵-元韵）
```

#### 寒韵 - Cold, Longing Scenes
**韵脚字**: 寒、看、关、山、残、难、坛、滩、弹、叹、欢、还、安、间、班、般、颜、闲、湾、盘、关、冠、刊、团、完、丸、弯、湾、删、姗、珊、檀
**适用场景**: 寒冷季节、思念、离别
**示例**:
```
秋风扫落叶（叶-屑韵）✗
---
正确示例：
秋风起寒山（山-寒韵）
离别断肠看（看-寒韵）
相思夜夜难（难-寒韵）
孤雁度关关（关-寒韵）
```

### Oblique Tone Rhymes (仄声韵)

#### 庚韵 - Night, Thinking Scenes
**韵脚字**: 庚、影、醒、静、梦、鸣、名、情、行、清、晴、轻、惊、迎、盈、营、瓶、萍、屏、平、令、伶、铃、灵、龄、凌、陵、聆、零、领、岭
**适用场景**: 夜景、思考、情感细腻
**示例**:
```
月落乌啼夜（夜-屑韵）✗
---
正确示例：
月落影渐清（清-庚韵）
夜深人静醒（醒-庚韵）
相思梦中情（情-庚韵）
孤灯照窗明（明-庚韵）✗ - 明是平声，应改为静/影/醒
```

#### 尤韵 - Water, Sadness Scenes
**韵脚字**: 尤、流、愁、秋、楼、游、舟、头、收、留、求、谋、眸、柔、幽、羞、忧、油、由、游、悠、尤、邮、丘、邱、牛、抽、酬、绸、仇、俦、畴
**适用场景**: 水流、愁绪、离别
**示例**:
```
明月照高楼（楼-尤韵）
相思寄红豆（豆-尤韵）
离别添新愁（愁-尤韵）
孤雁向南游（游-尤韵）
```

#### 侵韵 - Depth, Inner Scenes
**韵脚字**: 侵、心、深、金、音、琴、阴、吟、襟、临、林、沉、寻、森、针、真、珍、斟、砧、簪、簪、参、衬、申、身、伸、神、臣、晨、辰、尘、沉、陈
**适用场景**: 深情、内心世界、音乐
**示例**:
```
相思入我心（心-侵韵）
离别意难申（申-侵韵）
孤坐抚瑶琴（琴-侵韵）
月落夜已深（深-侵韵）
```

## Sentence Length Patterns

| Length | Pattern Example | Best Usage |
|--------|-----------------|------------|
| 1字 | 归！错！错！错！ | 强调、感叹 |
| 2字 | 明月、清风、相思 | 简洁意象 |
| 3字 | 桃花落、春风起 | 简短叙述 |
| 4字 | 春暖花开、相思红豆 | 完整表达 |
| 5字 | 明月照高楼 | 古诗风格 |
| 6字 | 春风吹拂杨柳 | 描写叙事 |
| 7字 | 春风又绿江南岸 | 古诗叙事 |
| 8字+ | 长句叙事描写 | 详细叙事 |

### Common Verse Structures

**Verse (主歌)**: 多用4-6字句，叙事铺垫
**Chorus (副歌)**: 多用5-7字句，情感高潮
**Bridge (过渡)**: 句式变化，转换情绪

## Tonal Balance (Ping-Ze)

### Basic Concept
- **平声**: 1st/2nd tones (阴平、阳平) - smooth, extended
- **仄声**: 3rd/4th tones (上声、去声) - varied, short

### Balance Patterns

| Pattern | Example | Effect |
|---------|---------|--------|
| 平平仄仄 | 明月清风 | Smooth opening |
| 仄仄平平 | 断肠天涯 | Emotional impact |
| 平仄平仄 | 独倚栏杆 | Varied rhythm |
| 仄平仄平 | 夜深人静 | Alternating |

### Guidelines
- Line endings favor **平声** (extendable, rhyme-friendly)
- Avoid consecutive same tones (平平平平 or 仄仄仄仄)
- Match tone pattern to melody rhythm

## Parallelism Techniques

### Types

| Type | Definition | Example |
|------|------------|---------|
| 工对 | Exact match: same length, category, opposite tones | 春风吹杨柳 vs 秋雨淋梧桐 |
| 宽对 | Similar match: same length, similar category | 相思红豆寄情思 vs 离别青鸟传离愁 |
| 流水对 | Sequential meaning flow | 天涯两端各守候，何时相见诉衷肠 |

### Usage Guidelines
- Use in verse for structured beauty
- Avoid in chorus (focus on emotional flow)
- Don't force if breaks natural expression

## Rhyme Selection Guide

### By Theme

| Theme | Recommended Rhyme | Reason |
|-------|-------------------|--------|
| 离别 | 尤韵、侵韵、寒韵 | 柔和、愁绪 |
| 思念 | 支韵、真韵、微韵 | 细腻、深情 |
| 山水 | 东韵、江韵、庚韵 | 开阔、自然 |
| 历史 | 元韵、寒韵、删韵 | 时间、沧桑 |
| 爱情 | 江韵、真韵、侵韵 | 温柔、真挚 |
| 孤独 | 尤韵、侵韵、庚韵 | 寂静、深沉 |
| 春天 | 江韵、东韵、微韵 | 明朗、生机 |
| 秋天 | 尤韵、寒韵、侵韵 | 愁绪、离别 |

### By Emotion Intensity

| Intensity | Recommended Rhyme |
|-----------|-------------------|
| Strong emotion | 寒韵、尤韵、侵韵 |
| Gentle emotion | 支韵、微韵、真韵 |
| Neutral/Natural | 东韵、江韵、庚韵 |

### Rhyme Change Strategy

```
Verse 1: A韵 (铺垫情感)
Verse 2: A韵 (延续基调)
Chorus: B韵 (情感高潮，对比Verse)
Bridge: C韵 (转折情绪)
Verse 3: A韵 or B韵 (回归或升华)
Outro: A韵 (收束回归)
```

## Common Mistakes

❌ **Mixed rhyme in same verse**: Verse内混用不同韵脚
❌ **Wrong tone at line end**: 句末用仄声难押韵
❌ **Over-density**: 每字都押韵，过于刻意
❌ **Ignoring melody**: 韵律不符合旋律节奏
❌ **Forced parallelism**: 强行对仗破坏表达
