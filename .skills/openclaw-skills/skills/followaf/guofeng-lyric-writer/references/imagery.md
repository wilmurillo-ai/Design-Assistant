## Natural Imagery

### Sky & Weather
| Image | Meaning | Example Usage |
|-------|---------|---------------|
| 月 (moon) | longing, solitude, eternity | 明月、残月、皎月、月色 |
| 日 (sun) | hope, warmth, time passing | 朝阳、夕阳、暖阳 |
| 星 (stars) | brilliance, distance, eternal | 繁星、星辰、孤星 |
| 云 (clouds) | wandering, change, separation | 浮云、青云、暮云 |
| 雨 (rain) | sorrow, longing, separation | 烟雨、细雨、秋雨 |
| 风 (wind) | freedom, departure, gentleness | 清风、秋风、朔风 |
| 雪 (snow) | purity, coldness, solitude | 白雪、飞雪、残雪 |
| 霜 (frost) | coldness, sorrow | 清霜、寒霜 |
| 露 (dew) | fleeting, pure | 晨露、露珠 |
| 雾 (mist) | mystery, confusion | 烟雾、薄雾、晨雾 |

## Theme-Based Selection Guide

### 离别主题
**Core images**: 天涯、断肠、远行、归雁、杨柳、长桥、城门
**Supporting**: 月、雨、秋风、孤灯、古道

### 思念主题
**Core images**: 相思、梦、月、书信、归雁、天涯
**Supporting**: 夜、星、风、云、烛火

### 山水主题
**Core images**: 山、水、云、雾、清风、明月、流水、青竹
**Supporting**: 晨露、夕阳、湖、溪、亭、楼

### 历史主题
**Core images**: 朝代、帝王、将军、古城、城墙、烽火、古道
**Supporting**: 长河、沧海、月、风、雪

### 爱情主题
**Core images**: 相思、红豆、月、桃花、柳、缘、誓、心
**Supporting**: 梦、风、雨、琴、笛、书信

### 孤独主题
**Core images**: 独、孤、空、寂寞、孤灯、孤雁、空庭、空楼
**Supporting**: 夜、月、雨、秋风、寒霜

### 优雅/文雅主题
**Core images**: 琴、筝、诗、词、画、墨、宣纸、茶、香
**Supporting**: 月、清风、竹、兰、莲、亭、阁
