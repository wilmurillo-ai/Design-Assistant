# 章节审查模板（优化版）

每章生成后，按此模板进行审查，确保长篇小说剧情不偏离、不穿帮、不重复。

提供两种审查模式：
- **快速审查（3维度）**：日常创作使用，约3分钟
- **完整审查（9维度）**：关键章节或问题时使用，约10分钟

---

## 审查模式选择

| 场景 | 推荐模式 | 原因 |
|------|---------|------|
| 常规章节生成后 | 快速审查 | 效率高，发现主要问题 |
| 卷末高潮章节 | 完整审查 | 关键节点需全面检查 |
| 发现质量问题时 | 完整审查 | 深度诊断问题根源 |
| 用户要求详细审查 | 完整审查 | 满足用户需求 |
| 批量生成后 | 快速审查 + 抽样完整审查 | 平衡效率和质量 |
