## 章节分解表填写指南

### 核心任务

每章的核心任务必须明确、可衡量：
- ❌ "写一段打脸" — 太模糊
- ✅ "主角在拍卖会上碾压赵天罡，竞得神器" — 明确具体

### 爽点类型

| 类型 | 描述 | 适用频率 |
|------|------|---------|
| 打脸 | 被小看 → 展示实力 → 众人震惊 | 每1-2章 |
| 突破 | 等级/能力提升 | 每3-5章 |
| 获宝 | 获得重要物品/功法/传承 | 每5-8章 |
| 逆转 | 局势反转，绝地反击 | 卷终高潮 |
| 身份揭秘 | 真实身份曝光震惊众人 | 每10-15章 |
| 降维打击 | 用更高层次的力量碾压 | 大高潮 |

### 情绪曲线

| 曲线类型 | 适用场景 |
|---------|---------|
| 低开高走 | 常规章节，开局压抑结尾爽 |
| 层层递进 | 战斗章节，紧张感不断升级 |
| 反转爆发 | 转折章节，先绝望后逆转 |
| 过山车 | 高潮章节，情绪多次起伏 |

## 规划质量检查清单

规划完成后自检：

- [ ] 每章都有明确的"核心任务"
- [ ] 爽点分布均匀，没有连续3章无爽点
- [ ] 中高潮和卷终高潮位置合理
- [ ] 伏笔埋设和回收章节都已标注
- [ ] 新角色/新地点的出场章节已标注
- [ ] 本卷目标与大纲主线一致
- [ ] 预估总字数合理（符合每章2000-3000字）
- [ ] 与前卷衔接无断裂【新增】
- [ ] 风险点已识别并制定预防措施【新增】

## 规划更新规则

| 事件 | 更新内容 |
|------|---------|
| 章节写完 | 更新"状态"列为"已完成" |
| 章节核心任务调整 | 更新"核心任务"列，备注调整原因 |
| 新增角色/地点 | 在对应规划表中补充 |
| 伏笔回收 | 更新"伏笔规划"表状态 |
| 剧情偏离 | 在注意事项中记录偏离原因和修正方案 |

---

## 与大纲的关系

```
大纲（全局）
    ↓ 指导
卷级章节规划（本卷）
    ↓ 指导
单章生成（具体）
```

- 大纲决定全书走向
- 章节规划将大纲细化为每卷的章节任务
- 单章生成时必须对照章节规划执行
