# 直播电商日报 Skill

> 每天一句话，让你不错过直播电商的任何重要动态。

## 这个 Skill 能做什么？

每次输入「今日直播电商日报」，自动为你整合当日直播电商行业资讯，覆盖：

| 方向           | 内容                  |
| ------------ | ------------------- |
| 🚨 今日最值得关注   | 所有资讯中影响最大的一条，单独置顶   |
| 🔴 千川 / 巨量引擎 | 投放政策、产品更新、出价策略变化    |
| 🟠 抖音电商平台    | 平台规则、活动、流量分配变化      |
| 🟡 爆款素材方向    | 当前跑量素材风格、创意参考       |
| 🟢 竞品动态      | 同类商家动向、新打法          |
| 🔵 行业大盘      | 整体趋势、GMV 数据、平台变化    |
| 🗂 持续追踪中     | 多日跟踪的重要事件，标注天数和最新进展 |

每条资讯都附带「**品类化关注点**」——根据你的品类（家电/食品/美妆等），告诉你这件事对你的具体操作有什么影响。

---

## 快速开始

### 第一步：安装

将 `SKILL.md` 放入你的 Skill 目录：

```bash
# WorkBuddy 用户
mkdir -p ~/.workbuddy/skills/ecom-daily-report
cp SKILL.md ~/.workbuddy/skills/ecom-daily-report/
```

### 第二步：触发

重启工具后，直接输入：

```
今日直播电商日报
```

### 可选参数

```bash
今日直播电商日报                           # 自动搜索（默认）
今日直播电商日报 [粘贴文章标题/链接]        # 手动整理
今日直播电商日报 千川政策                  # 聚焦某个方向
今日直播电商日报 品类=家电清洁             # 品类定制关注点（推荐）
今日直播电商日报 保存到 ~/Documents/      # 自定义保存路径
今日直播电商日报 格式=html                # 输出 HTML 美化版
今日直播电商日报 品类=家电清洁 格式=html   # 组合使用
```

## 输出示例

```
# 直播电商日报 · 2026年3月30日
> 今日关键词：全域投放革命 · 食品安全新规 · 行业规范化

## 🔴 千川 / 巨量引擎
- **千川「乘方」新产品上线，全域投放成唯一主线**
  来源：36氪 | 2026-03-30
  摘要：全域+净成交出价成核心产品，标准投放进入退场倒计时
  ✅ 投手关注点：尽快迁移至全域投放，对比 ROI 变化
...
```

日报自动保存为 Markdown 文件到本地（路径可配置）。

## 搜索来源

优先抓取以下平台的内容：

- **36氪**、**今日头条**、**微博**（实时性强）
- 巨量引擎官方、抖音电商学习中心（政策第一手）

## 适合谁用？

- 千川投手：快速了解投放环境变化，不错过政策红利窗口
- 直播运营：掌握平台规则变化，提前规避风险
- 商家负责人：每天5分钟了解行业趋势，不被竞品甩开
