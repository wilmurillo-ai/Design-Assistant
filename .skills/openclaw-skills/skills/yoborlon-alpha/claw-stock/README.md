# claw-stock · 个股分析师

> OpenClaw Skill — 股票综合分析，动态六因素评分

## 简介

**claw-stock** 是一个专注于个股深度分析的 AI Skill。给出任意一只股票（A 股 / 港股 / 美股），它会：

1. 根据当前市场环境和个股特征，动态选出最关键的 **6 个分析因素**
2. 逐因素展开分析，每项单独打分
3. 汇总给出 **0–100 分的综合评分**
4. 给出一句话核心结论

分析维度覆盖基本面、技术面、估值面、资金面、行业面、政策面、成长面、心理面、催化剂、风险面，**不死板套用固定模板**，随市场重心动态调整关注点。

## 使用方式

直接在 ClawHub 聊天框中输入，例如：

```
分析一下宁德时代
茅台现在值得买吗
600519 打个分
帮我看看比亚迪
苹果股票怎么样
```
