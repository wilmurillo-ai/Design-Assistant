# Quick Start: ClawHub Publishing Mother Skill

## 🚀 In 5 Minutes:

1. **Install:**
   ```bash
   hermes skills install clawhub-publishing-mother-skill
   ```

2. **Test:**
   ```bash
   hermes run clawhub-publishing-mother-skill --help
   ```

3. **First Publish:**
   ```bash
   hermes run clawhub-publishing-mother-skill --skill your-skill --dry-run
   ```

## 🔧 Core Commands:

```bash
# Basic publishing
hermes run clawhub-publish --skill my-skill --personality dry-humor

# With promotion
hermes run clawhub-publish --skill my-skill --promote --platforms twitter,community

# Track impact
hermes run clawhub-track --skill my-skill --metrics

# Generate report
hermes run clawhub-report --format markdown
```

## 💡 Pro Tips:

1. **Start with --dry-run** to see what will be published
2. **Use --personality** to match your skill's voice
3. **Enable --tracking** from day one
4. **Engage early** with --community-feedback
