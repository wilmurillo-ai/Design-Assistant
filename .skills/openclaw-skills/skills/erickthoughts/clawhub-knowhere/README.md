# Knowhere ClawHub Skill

ClawHub-distributed skill for the Knowhere OpenClaw plugin.

This skill helps agents use the `knowhere_*` tools for document ingestion,
browse-first retrieval, job inspection, and cleanup.

## Prerequisite

Install the plugin before using this skill:

```bash
openclaw plugins install @ontos-ai/knowhere-claw
```

Then enable the `knowhere` plugin entry in OpenClaw and restart OpenClaw.

## Install the Skill

Install the skill from ClawHub:

```bash
clawhub install knowhere
```
