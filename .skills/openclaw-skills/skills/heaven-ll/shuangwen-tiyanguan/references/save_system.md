# 爽文体验馆 - 存档管理

## 存档独立原则

**核心规则**：凡人踏天行和武林趣闻的存档**完全独立**，数据不互通。

### 独立内容
- ✅ 角色名称、属性
- ✅ 物品、装备
- ✅ 进度、剧情
- ✅ 成就、称号
- ✅ 死亡/转世记录

### 不共享内容
- ❌ 角色数据跨游戏使用
- ❌ 物品转移
- ❌ 经验/等级继承
- ❌ 死亡惩罚传递

## 存档文件位置

### 凡人踏天行
```
workspace/
├── cultivation_save.md          # 主要存档
├── memory/
│   └── YYYY-MM-DD.md           # 每日记录
└── MEMORY.md                    # 长期记忆
```

### 武林趣闻
```
workspace/
├── wuxia_save.md               # 主要存档
├── memory/
│   └── YYYY-MM-DD.md          # 每日记录
└── MEMORY.md                   # 长期记忆
```

## 切换游戏说明

当用户从游戏 A 切换到游戏 B 时：

1. **自动存档** - 游戏 A 自动保存当前进度
2. **读取选择** - 回到体验馆选择界面
3. **加载游戏** - 选择游戏 B，读取对应存档
4. **独立继续** - 游戏 B 使用自己的存档继续

## 示例场景

### 场景 1：同时玩两个游戏
```
用户：开始爽文
AI：[显示选择菜单]
用户：1（选择凡人踏天行）
AI：[启动修仙游戏，创建角色"张三"]
...
用户：我想玩武侠
AI：[保存修仙进度，返回体验馆]
用户：2（选择武林趣闻）
AI：[启动武侠游戏，创建角色"李四"]
...
用户：继续修仙
AI：[保存武侠进度，返回体验馆，加载修仙存档]
```

### 场景 2：一个游戏死亡不影响另一个
```
用户：[在凡人踏天行中角色死亡]
AI：[显示转世选项，修仙存档重置或转世]
用户：开始爽文 → 选择武林趣闻
AI：[武侠角色完全正常，不受影响]
```
