# 爽文体验馆 - 核心逻辑

## 启动流程

### 1. 检测阶段
- 检查 `cultivation-chronicle-cn` 是否已安装
- 检查 `wuxia-quwen` 是否已安装
- 显示对应安装状态

### 2. 选择阶段
- 展示两个游戏选项
- 用户输入 1 或 2 做出选择
- 输入 help 查看详细说明

### 3. 安装阶段（如需要）
- 如选择的游戏未安装，提示安装
- 执行安装命令

### 4. 启动阶段
- 读取对应 skill 的启动文件
- 开始游戏

## 游戏入口

### 凡人踏天行（修仙）
启动命令：
```
read ~/.openclaw/workspace/skills/cultivation-chronicle-cn/references/ch00_start.md
```

### 武林趣闻（武侠）
启动命令：
```
read ~/.openclaw/workspace/skills/wuxia-quwen/references/ch00_start.md
```

## 安装命令

### 凡人踏天行
```
npx skills add cultivation-chronicle-cn -g -y
```
或
```
openclaw skills install cultivation-chronicle-cn
```

### 武林趣闻
```
npx skills add wuxia-quwen -g -y
```
或
```
openclaw skills install wuxia-quwen
```
