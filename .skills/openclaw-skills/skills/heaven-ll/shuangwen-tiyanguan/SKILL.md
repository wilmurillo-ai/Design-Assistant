# 爽文体验馆 🎮

> 一朝踏入爽文界，从此节操是路人

## 简介

**爽文体验馆**是四个风格迥异的文字冒险游戏的统一入口：

**🗡️ 热血爽文区：**
- ⚔️ **凡人踏天行** - 踏天之路，从凡人到飞升
- 🗡️ **武林趣闻** - 江湖风云，从菜鸟到大侠

**💕 情缘沉浸区：**
- 🎩 **民国风云** - 乱世枭雄，烽火绝恋
- 🌸 **青春校园** - 纯爱物语，青涩心动

**重要提示**：四个游戏世界**完全独立**，存档不互通。你可以在修仙世界渡劫飞升，在武侠世界成为一代宗师，在民国乱世既得天下又获真心，或在校园里谈一场青涩的恋爱——四条路互不影响，各生欢喜。

---

## 使用方法

### 启动体验馆
直接说：
- `"开始爽文"`
- `"进入体验馆"`
- `"启动爽文体验馆"`

### 选择游戏
根据提示输入：
- `1` - 进入凡人踏天行（修仙）
- `2` - 进入武林趣闻（武侠）
- `3` - 进入民国风云（民国）
- `4` - 进入青春校园（校园）
- `help` - 查看详细说明

### 游戏控制
每个游戏内都有独立的指令系统：
- **凡人踏天行**：`修炼`、`突破`、`查看状态`、`转世` 等
- **武林趣闻**：`打坐`、`闯荡`、`拜师`、`战斗` 等
- **民国风云**：根据剧情推进，涉及`属性`、`好感度`、`历史事件`等系统
- **青春校园**：根据剧情推进，涉及`属性`、`好感度`、`学业`、`社团`等系统

## 功能特性

1. **自动检测** - 检查本地是否已安装对应游戏 skill

## 存档说明

### 完全独立的存档系统
- 四个世界**数据不互通**
- 各自有独立的角色属性、物品、进度
- 一个世界的死亡/结局不会影响另一个世界
- 可同时拥有四个世界的角色/存档

### 存档位置
- **凡人踏天行**：`workspace/cultivation_save.md`
- **武林趣闻**：`workspace/wuxia_save.md`

## 游戏详细介绍

### 凡人踏天行 ⚔️
> 炼气→筑基→金丹→元婴→化神→渡劫→飞升

- **特色**：转世传承系统、灵根随机、道心抉择
- **结局**：飞升成仙/陨落转世/隐世高人
- **适合**：喜欢修仙网文、数值成长的玩家

### 武林趣闻 🗡️
> 无名小卒→江湖新秀→一方豪杰→名震四方→武林盟主→一代宗师→武林神话

**章节结构（6章）：**
| 章节 | 阶段 | 主要内容 |
|:---|:---|:---|
| **第一章** | 青松观拜师 | 三关考验、获得武功、同门相处 |
| **第二章** | 下山历练 | 初入江湖、采买物资、追查采花贼、邂逅苏红袖 |
| **第三章** | 师门大比 | 年度选拔、三关考验、决战大师兄、内门晋升 |
| **第四章** | 武林大会 | 华山之巅、群英荟萃、比武切磋、青年才俊榜 |
| **第五章** | 身世之谜 | 调查幽冥教、师门危机、天剑门遗孤、真相浮现 |
| **第六章** | 正邪大战 | 联盟成立、最终决战、复仇雪恨、重建天剑门 |

**可攻略女主（4位）：**
- **苏红袖** - 江湖游侠，红袖刀，豪爽直率
- **柳如烟** - 峨眉弟子，温柔娴静，医术精湛
- **冷凝霜** - 西域遗孤，冷若冰霜，身负血仇
- **二师姐** - 同门情深，温柔善良，青梅竹马

- **特色**：道德值系统、门派选择、奇遇事件、多女主攻略
- **结局**：武林至尊/隐退江湖/一代宗师/与佳人归隐
- **适合**：喜欢武侠、快意恩仇、感情线的玩家

### 民国风云 🎩
> 1900-1937，军阀二世祖的烽火绝恋

- **特色**：历史沉浸、五位可攻略女主、家国抉择
- **结局**：抗日英雄/乱世枭雄/海外寓公
- **适合**：喜欢历史、权谋、感情线的玩家

### 青春校园 🌸
> 高中三年，五个男生，一场青涩的恋爱

**章节结构（学期制 + 暑假篇）：**
| 章节 | 时间段 | 主要内容 |
|:---|:---|:---|
| **高一上** | 9月-1月 | 军训、社团招新、期中考试、元旦晚会、寒假 |
| **高一下** | 2月-7月 | 开学、春季踏青、篮球联赛、研学旅行、暑假 |
| **高二上** | 9月-1月 | 开学、**选科分班**、秋季运动会、期中考试、家长会、元旦晚会 |
| **高二下** | 2月-7月 | 开学、**学业水平考试（会考）**、期中考试、端午节、**高考旁观**、期末考 |
| **高三上** | 9月-1月 | 开学摸底、**百日冲刺**、**一模二模**、志愿选择、跨年约定 |
| **高三下** | 2月-6月 | 二轮复习、**三模**、**百日誓师**、**成人礼**、**毕业典礼**、**高考决战**、最终结局 |
| **暑假篇** | 假期 | 打工体验、夏令营、夏日旅行、感情升温 |

**学期特色活动：**
- **高一上**：军训（9月）、社团招新（10月）、期中（11月）、元旦晚会（12月）
- **高一下**：踏青（4月）、篮球联赛（5月）、研学旅行（6月）
- **高二上**：**选科分班**（9月）、秋季运动会（10月）、**家长会**（11月）
- **高二下**：**学业水平考试/会考**（5-6月）、**端午节**（6月）、**高考旁观**（6月）
- **高三上**：**百日冲刺**（9月起）、**一模**（10月）、**二模**（12月）
- **高三下**：**百日誓师**（2月）、**成人礼**（4月）、**毕业典礼**（5月）、**高考**（6月）

**五位可攻略男主：**
- **林逸辰** - 学霸班长，外冷内热，天文爱好者
- **顾阳** - 篮球队长，阳光暖男，运动系
- **沈墨** - 美术社成员，文艺画手，温柔细腻
- **陆景和** - 志愿者会长，家政全能，邻家哥哥
- **隐藏角色** - 转学生/青梅竹马（特殊路线）

**特色活动：**
- 军训、社团招新、期中/期末考试
- 春季踏青、秋季运动会、元旦晚会
- 篮球联赛、研学旅行、百日誓师、成人礼
- 打工体验、夏令营、暑假旅行

- **结局**：双向奔赴/各自天涯/后知后觉/独自精彩
- **适合**：喜欢纯爱、青春、虐心遗憾的玩家
