## 使用方法

### 1. 搜索知识库

使用 `feishu_search_doc_wiki` 搜索相关内容：
- query: 搜索关键词
- 可按文档类型筛选

### 2. 获取文档内容

使用 `feishu_fetch_doc` 获取具体文档：
- doc_id: 从搜索结果获取

### 3. 结合知识执行任务

当用户提出需求时：
1. 先搜索知识库相关主题
2. 获取具体操作指南
3. 按照知识库中的最佳实践执行
4. 如需创建新Skill，参考 Skill 创建指南
