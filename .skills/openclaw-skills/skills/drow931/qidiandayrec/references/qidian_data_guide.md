# 起点三江榜单数据获取指南

## 数据源概述

起点中文网的三江榜单数据通过起点图（qidiantu.com）第三方数据平台获取，无需登录、无反爬验证。

### 为什么用起点图而非起点官网？

| 对比项 | 起点官网 (qidian.com) | 起点图 (qidiantu.com) |
|--------|----------------------|----------------------|
| 反爬机制 | 严格（JS探测、Cookie校验） | 无 |
| 数据获取 | 需要无头浏览器 | 简单HTTP请求 |
| 额外数据 | 仅基本信息 | 收藏增长、首订数据 |
| 稳定性 | 频繁变更前端结构 | 相对稳定 |

## URL 结构

### 三江榜单列表

```
https://www.qidiantu.com/bang/1/6/{date}
```

- `{date}`: 榜单日期，格式 `YYYY-MM-DD`
- 榜单通常每周更新一期（周五）
- 示例: `https://www.qidiantu.com/bang/1/6/2026-04-10`

### 书籍详情页

```
https://www.qidiantu.com/info/{book_id}
```

- `{book_id}`: 起点书籍ID（纯数字）
- 示例: `https://www.qidiantu.com/info/1048197630`
- 部分书籍URL后缀带 `/d`，表示带数据图表

### 起点官网阅读页

```
https://www.qidian.com/book/{book_id}/
```

用于生成给用户的阅读链接。

| intro | 简介 | 完整简介文本 |
| first_order | 首订数 | 1429 |

## 三江榜单更新规律

- **更新周期**: 每周一期，通常在周五更新
- **每期书量**: 约15-20本
- **覆盖分类**: 玄幻、奇幻、都市、历史、科幻、轻小说、诸天无限等
- **推荐标准**: 编辑推荐+数据筛选，侧重新书潜力

## 注意事项

1. **请求频率**: 建议每次请求间隔 ≥1秒，避免对起点图服务器造成压力
2. **日期回退**: 如果指定日期无榜单数据，脚本会自动回退查找最近7天的数据
3. **book_id 格式**: 纯数字字符串，如 `1048197630`
4. **简介长度**: 部分书籍简介较长，建议显示时截断至300-500字
5. **数据时效**: 起点图数据通常比起点官网延迟1-2天更新
