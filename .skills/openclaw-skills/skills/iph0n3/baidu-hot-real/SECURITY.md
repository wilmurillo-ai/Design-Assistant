# 安全说明 (Security)

## 数据来源安全

- **唯一数据源**: `https://top.baidu.com/board`
- **访问类型**: 仅 HTTP GET 请求（公开网页）
- **无需认证**: 不要求任何 API Key 或凭证
- **无敏感数据**: 仅抓取公开热榜信息

## 脚本安全

### baidu_real.py
- ✅ 不读取任何本地文件
- ✅ 不写入任何文件（仅 stdout 输出）
- ✅ 不执行系统命令
- ✅ 不包含硬编码凭证
- ✅ URL 白名单验证（只允许访问 top.baidu.com）
- ✅ 合理的超时设置（15 秒）

### baidu_fetch.py
- ✅ 从 stdin 读取数据（不主动访问网络）
- ✅ 仅解析 HTML，不执行外部代码
- ✅ 无文件 I/O 操作
- ✅ 无系统命令执行

### fetch.sh
- ✅ 使用 `mktemp` 创建安全临时文件
- ✅ 使用 `trap` 确保退出时清理
- ✅ 限制临时文件权限为 600
- ✅ 参数验证（只接受数字或 "all"）
- ✅ 使用 `set -e` 错误立即退出
- ✅ URL 硬编码（防止注入）

## 临时文件安全

```bash
# 临时文件路径：/tmp/baidu_hot.XXXXXX.html
# 权限：600（仅当前用户可读写）
# 生命周期：脚本执行期间，退出后自动删除
```

## 网络请求限制

- **频率建议**: 间隔 ≥ 1 分钟（避免被百度限流）
- **超时**: 15 秒
- **最大响应**: 50,000 字符

## 无敏感操作

本技能**不执行**以下操作：
- ❌ 读取 ~/.ssh、.env、credentials 等敏感文件
- ❌ 访问系统配置文件（/etc/*）
- ❌ 执行 sudo 或提权操作
- ❌ 创建/修改用户账户
- ❌ 网络监听或反弹 shell
- ❌ 数据外发到非百度域名
- ❌ 修改系统时间、日志或其他基础设施

## 依赖安全
