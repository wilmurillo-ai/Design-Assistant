# ai-visibility-toolkit v0.2.0

This release turns the repository from a documentation-first AI visibility workflow into a more release-ready toolkit.

## Highlights

- Executable monitoring workflow with raw responses, score drafts, structured summaries, and weekly reports.
- Reproducible scoring through rubric, annotation protocol, and JSON schemas.
- Lightweight CLI and model leaderboard for faster onboarding and demoability.
- Repair-loop examples with T+7 and T+14 metric tracking.
- Collaboration improvements via issue templates and contribution guide.
