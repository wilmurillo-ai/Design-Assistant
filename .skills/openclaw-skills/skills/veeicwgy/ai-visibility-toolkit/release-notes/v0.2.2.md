# ai-visibility-toolkit v0.2.2

This release unifies the public name of the repository and skill under **AI Visibility Toolkit**.

## Highlights

- Renamed the GitHub repository and ClawHub skill to a less sensitive, more durable public brand.
- Added the new preferred CLI entry point: `python -m ai_visibility`.
- Renamed packaged sub-skills to the `visibility-*` family for more consistent positioning.
- Refreshed README, onboarding docs, playbook links, and sample report titles to match the new name.
