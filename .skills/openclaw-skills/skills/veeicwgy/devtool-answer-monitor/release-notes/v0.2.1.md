# ai-visibility-toolkit v0.2.1

This release expands the repository from an AI visibility baseline into a more complete growth workflow for developer tools, APIs, and scientific products.

## Highlights

- Stronger GitHub and ClawHub positioning with a clearer promise, starter prompts, and a 30-second result path.
- Activation-oriented AI visibility guidance that connects model visibility to installs, API calls, and agent invocation outcomes.
- Funnel-stage slices in summaries and weekly reports so teams can separate awareness, selection, integration, activation, and agent issues.
- Scientific product additions including Sciverse API query-pool examples and a complete sample run artifact set.
- Better onboarding through clearer getting-started docs, goal-based entry points, and richer default samples.
