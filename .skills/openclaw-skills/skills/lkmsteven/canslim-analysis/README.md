# CANSLIM Analysis Skill

A hybrid quantitative and qualitative CANSLIM analysis tool for US stocks that combines fundamental metrics with AI-driven qualitative insights to generate ranked investment candidates.

## Overview

This skill executes a two-step modular pipeline:
1. **Quantitative Analysis**: Screens stocks based on fundamental metrics (C, A, L, S criteria)
2. **AI Qualitative Analysis**: Uses AI capabilities to evaluate qualitative metrics (N, S, I criteria)
3. **Final Processing**: Generates a ranked CANSLIM report with composite scores

## Installation

### Prerequisites
- Python 3.8 or higher
- pip package manager

### Setup

1. Create a virtual environment:
   ```bash
   python -m venv canslim_analysis
   ```

2. Activate the virtual environment:
   - **Windows**: `canslim_analysis\Scripts\activate`
   - **macOS/Linux**: `source canslim_analysis/bin/activate`

3. Install dependencies:
   ```bash
   pip install --no-cache-dir -r Scripts/requirements.txt
   ```

## Usage

### Run the Full Analysis

```bash
# Step 1: Run quantitative analysis
python Scripts/quantitative_analyzer.py

# Step 2: AI qualitative analysis (performed automatically by the skill)
# The skill reads intermediate_canslim.json and enriches it with AI findings
# Create enriched_canslim.json with AI-discovered catalysts

# Step 3: Generate final report
python Scripts/final_process.py
```

### Configuration

You can adjust the following constants in `Scripts/quantitative_analyzer.py`:

| Constant | Default | Description |
|----------|---------|-------------|
| `MIN_EPS_Growth` | 0.25 | Minimum quarterly EPS growth (25%) |
| `MIN_ROE` | 0.17 | Minimum Return on Equity (17%) |
| `MIN_RS_RATING` | 80.0 | Minimum Relative Strength percentile |
| `MIN_VOLUME_RATIO` | 1.5 | Volume to 50-day average ratio threshold |
| `MAX_WORKERS` | 5 | Number of concurrent download workers |
| `DEFAULT_UNIVERSE_LIMIT` | None | Maximum stocks to analyze (None = no limit) |
| `MAX_RETRIES` | 3 | Number of retry attempts for failed requests |
| `RETRY_DELAY` | 2 | Delay between retry attempts (seconds) |

## CANSLIM Criteria

### The Analysis evaluates stocks against 7 criteria:

- **C - Current Earnings**: Quarterly EPS Growth ≥ 25%
- **A - Annual Earnings**: ROE ≥ 17% (proxy for annual growth)
- **N - New Products/Management**: AI-analyzed catalysts and news
- **S - Supply and Demand**: High volume + tight float analysis
- **L - Leader**: Relative Strength ≥ 80
- **I - Institutional Sponsorship**: AI-analyzed institutional quality
- **M - Market Direction**: Broad market trend assessment

## Output Files

### 1. `intermediate_canslim.json`
Generated by quantitative analysis. Contains:
- Metadata about the scan
- Stocks that passed quantitative filters
- AI qualitative fields (pending enrichment)

### 2. `enriched_canslim.json`
Generated by AI qualitative analysis. Contains:
- Same structure as intermediate
- AI-discovered catalysts filled in N, S, I fields

### 3. `final_canslim_report.json`
Generated by final processing. Contains:
- Ranked list of candidate stocks
- Final CANSLIM scores (out of 7)
- Met/missed criteria for each stock
- AI catalyst insights

### 4. `canslim_analysis.log`
Automatically generated log file containing:
- Detailed execution information
- Error messages and warnings
- Progress tracking

## Troubleshooting

### Common Issues

**Issue**: "Failed to fetch S&P 500 tickers"
- **Solution**: Check your internet connection. The script retries 3 times with delays.

**Issue**: "Insufficient historical data"
- **Solution**: Some stocks may not have a full year of trading data. These are automatically skipped.

**Issue**: Rate limiting errors
- **Solution**: Reduce `MAX_WORKERS` or increase `RETRY_DELAY` in the configuration.

**Issue**: Empty results
- **Solution**: This is normal during market downturns. Check the market environment in the output.

### Debug Mode

For detailed logging, check `canslim_analysis.log` for:
- Individual stock fetch attempts
- Failed requests and error details
- Criteria evaluation results
- Processing statistics
