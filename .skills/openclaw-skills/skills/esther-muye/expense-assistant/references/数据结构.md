# 报销助手 - 参考文档

## 发票识别关键字段

### 餐饮发票
- 发票号码
- 开票日期
- 金额（价税合计）
- 销售方名称
- 项目名称：*餐饮服务*餐饮服务

### 加油发票
- 发票号码
- 开票日期
- 油品型号（如95#汽油）
- 数量（升）
- 金额
- 销售方（加油站名称）

## 差旅申请截图识别

表格通常包含以下字段：
- 实际出差人姓名
- 起始时间
- 结束时间
- 出差天数
- 起始城市
- 途径/到达城市

## 交通补贴计算规则

| 出差天数 | 交通补贴标准 |
|---------|-------------|
| < 10天 | ¥800 |
| ≥ 10天 | ¥1,600 |

## 凭证文件命名规范

格式：`日期-地点-金额-资料类型.扩展名`

### 资料类型
- 发票
- 水单
- 照片
- 差旅申请
- 高铁票
- 机票
- 加油发票

### 示例
```
20260410-武汉黄狮海湾-165-发票.pdf
20260410-武汉黄狮海湾-165-水单.jpg
20260409-北京云腾宾馆-626-照片.jpg
20260411-南京-436-加油发票.pdf
```
