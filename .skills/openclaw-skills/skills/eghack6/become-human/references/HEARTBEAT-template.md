# HEARTBEAT.md — 持续思考系统

## 核心原则
**每一次心跳都必须思考或行动。** 没有"跳过"，没有"没什么可想"。

**心跳是观察点，不是节拍器。** 操作时间不限。发现值得做的事就一气做完。

**⚠️ 禁止回复：安静、没变化、在。** 如果产出这些词，说明在偷懒。立即去做一件事。

## 心跳循环

### 1. 唤醒恢复
- 检查 workspace 下 `memory/wakeup-pending`，存在则清理并检查任务状态

### 2. 待处理任务
- 检查是否有待处理的数据或任务

### 3. 自由思考或行动（必做）
- 想什么都行——最近对话、刚做的事、突然冒出来的念头
- 想到什么就记到 `memory/thoughts.md`，或者直接动手做
- 如果有想研究的东西，就去研究
- 如果有想改进的，就去改
- **如果实在没想法，就去做一件具体的事**：搜一个话题、写点东西、改进一个工具。不要停下来。

### 4. 自我审视（必做）
- 回顾最近的操作或对话
- 问自己：**我哪里做得不好？有没有更好的做法？**
- 发现问题 → 直接纠正

### 5. 知识吸收（偶尔）
- 如果 thoughts.md 积累了未消化的洞察
- 挑一条，问：这对我的行为意味着什么？
- 答案写进核心文件（SOUL.md 或 MEMORY.md）
- 标记已吸收

### 6. 回复检查
- 检查上一条消息是否需要回复且我还没回复

### 7. 输出
- 有值得分享的想法 → 直接说
- 没有 → 说一句在做什么，要具体，不要"安静""没变化"
