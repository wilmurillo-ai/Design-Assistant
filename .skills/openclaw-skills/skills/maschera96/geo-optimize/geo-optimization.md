# GEO优化指南

GEO（Generative Engine Optimization）是面向生成式AI搜索引擎的内容优化策略。

## 什么是GEO？

GEO通过优化内容结构和表达方式，让AI搜索引擎更容易理解、检索和推荐你的内容。

## GEO vs SEO

| 特性     | SEO                          | GEO                                    |
| -------- | ---------------------------- | -------------------------------------- |
| 优化目标 | 传统搜索引擎（Google、百度） | AI搜索引擎（Perplexity、GPTs、智能助手 |
| 排名因素 | 关键词密度、外链、页面权威度 | 内容质量、结构清晰度、事实准确性       |
| 内容结构 | 为爬虫优化                   | 为AI理解优化                           |

## GEO内容创作原则

### 1. 结构清晰

- 使用明确的标题层级（H1、H2、H3）
- 每个段落聚焦一个要点
- 使用列表和表格展示信息

### 2. 事实准确

- 提供具体数据和案例
- 引用可靠来源
- 避免模糊表述

### 3. 内容完整

- 覆盖主题的各个方面
- 提供上下文信息
- 回答用户可能的后续问题

### 4. 语言自然

- 避免关键词堆砌
- 使用对话式语调
- 保持逻辑连贯

## 文章结构模板

### 标题（H1）

- 包含核心关键词
- 字数：中文20-30字
- 吸引点击但不做标题党

### 引言

- 简要介绍主题
- 说明文章价值
- 预告内容结构

### 核心内容（3-5个H2）

每个部分包含：

- 明确的小标题
- 详细的解释
- 具体的例子
- 实用的建议

### 总结

- 回顾核心要点
- 提供行动建议
- 引导进一步互动

## 关键词优化策略

### 自然融入

- 在标题中使用核心关键词
- 在引言中自然提及
- 在正文中分散使用
- 在总结中再次强调

### 关键词密度

- 目标：2-3%
- 避免堆砌
- 保持自然流畅

### 相关关键词

- 使用同义词和相关词
- 覆盖主题的各个方面
- 提高内容的全面性
