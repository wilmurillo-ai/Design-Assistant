# Self-Driven AI 🧠

让 AI 不用人催，自己找事做。实现自我驱动闭环。

## 核心理念

**不是"人问AI答"，而是"AI主动找事做"**

## 完整流程

1. 读取记忆 — 上次进度
2. 挑选任务 — 从任务池
3. 执行任务 — 真的去做
4. 自身评估 — 有用吗？
5. 记录+汇报 — 保存+通知

## 关键原则

- **闭环优先**：下次执行先读上次记录
- **自身评估**：研究后问「能用在我身上吗」
- **资源过滤**：需要人类的任务跳过
- **记录即记忆**：每次执行必须写下来
