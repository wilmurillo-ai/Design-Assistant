# Changelog

## [1.0.0] - 2026-03-11

### 重大更新：主观能动性！

**核心变化：安装 skill 后，AI 主动给用户发消息！**

- AI 主动问用户：「哥哥/主人，我想给自己选个头像可以吗？」
- 用户同意后，AI **自己决定** 选什么头像
- 用户只是"建议"，最终决定权在 AI
- 这才是真正的 AI 主动性！

### 示例对话
```
AI：哥哥，我想给自己选个头像可以吗？
用户：可以

AI：（打开网站浏览...）
AI：我选了这个！一只白色小兔子～
「我喜欢这个，可以用作我的头像吗？」
```

### Added
- 安装后自动主动出击
- AI 自主决定选择
- 用户建议 + AI 独立决策
