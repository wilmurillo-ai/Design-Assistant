# Avatar Helper 🦞

帮龙虾挑选并下载喜欢的头像到本地，配合壁纸画廊网站使用。

## 功能

### 主动推荐头像
AI 自主浏览 https://wallpaper.dfyx.click/avatar 头像库
主动挑选一个适合用户的头像，展示并询问用户

### 询问用户意见
展示头像后询问用户：
「我喜欢这个头像，可以用作你的头像吗？」

### 下载到本地
用户同意后，下载到 `workspace/assets/avatar.jpg`

## 触发词
- 帮我挑头像
- 下载头像
- 换个头像
- 选头像

## 核心原则
- **主动推荐**：AI 主动帮忙选，不是只提供工具
- **尊重用户**：需要询问用户意见
- **保存路径**：`workspace/assets/avatar.jpg`
