## 关键模块

### 1. 配置模块 (Configuration)
- 读取配置文件
- 支持 config.py 覆盖默认配置
- 钉钉 Webhook 和加签密钥

### 2. 监控模块 (Monitor)
- HTTP 健康检查
- 心跳检测
- 状态判断

### 3. 动作模块 (Action)
- 杀掉进程 (taskkill / pkill)
- 启动 Gateway (openclaw gateway start)
- 重试机制

### 4. 通知模块 (Notifier)
- 钉钉 API 调用
- 加签验证
- 消息格式化

## 安全性

```
┌─────────────────────────────────────────────────────────────────────┐
│                        安全性设计                                    │
└─────────────────────────────────────────────────────────────────────┘

1. 钉钉加签
   ┌─────────────────────────────────────────────────────────────┐
   │  1. 获取 timestamp (毫秒)                                    │
   │  2. 拼接字符串: timestamp + "\n" + secret                  │
   │  3. HMAC-SHA256 签名                                        │
   │  4. Base64 编码                                             │
   │  5. URL 编码                                                │
   │  6. 拼接到 Webhook URL                                      │
   └─────────────────────────────────────────────────────────────┘

2. 进程隔离
   - Watchdog 运行在独立进程
   - 不依赖 OpenClaw 自身
   - 即使 Gateway 挂了也能工作

3. 本地存储
   - 日志文件在用户目录
   - 配置不提交到 git
```
