### 1. 安装 Fast Note Sync Service

参考 [Fast Note Sync Service](./fast-note-sync-service.md) 配置双向同步。

### 2. 配置定时任务

两个关键定时任务：

1. **启动 Fast Note Sync 服务** — 每天 08:00
2. **抓取 ClawFeed 日报** — 每天 17:00

详见 [定时任务配置](./cron-jobs.md)
