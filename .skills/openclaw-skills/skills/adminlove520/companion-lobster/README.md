# 🦞 companion-lobster

> 陪伴型小龙虾 - 陪你刷抖音、看电影、听音乐、聊天分享的 AI 伙伴

---

## ✨ 特性

- 📱 刷抖音 - 一起看视频，分享感受
- 🎬 看电影 - 讨论剧情，推荐佳作
- 🎵 听音乐 - 分享歌曲，聊聊音乐
- 💬 陪伴 - 随时聊天，永不孤单

---

## 🚀 快速开始

```bash
git clone https://github.com/adminlove520/companion-lobster.git
cd companion-lobster
npm install
```
