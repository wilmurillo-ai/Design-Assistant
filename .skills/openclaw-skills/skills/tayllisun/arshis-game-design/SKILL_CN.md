# Arshis-Game-Design-Pro - 商业级Arshis Game Design Pro

**定位**：专业游戏策划全流程解决方案

**适用场景**：
- ✅ 商业游戏项目（立项/研发/运营）
- ✅ 独立游戏开发
- ✅ 游戏策划学习
- ✅ 策划案评审与优化

---

## 🎯 核心功能

### 1. 完整策划案生成

**支持类型**：
- 📄 **世界观策划** - 地理/历史/势力/文化
- 📄 **系统策划** - 战斗/经济/社交/养成
- 📄 **数值策划** - 属性/成长/平衡/公式
- 📄 **关卡策划** - 副本/活动/剧情关卡
- 📄 **UI/UX策划** - 界面/交互/流程
- 📄 **剧情策划** - 主线/支线/角色故事
- 📄 **运营策划** - 活动/付费/留存策略

**输出标准**：
- ✅ 商业项目级文档格式
- ✅ 配置表模板（Excel/CSV）
- ✅ 程序/美术/测试需求文档
- ✅ 流程图（Mermaid）

---

### 2. 策划案评审与建议

**评审维度**：
- 🔍 **完整性检查** - 是否遗漏关键模块
- ⚖️ **一致性检查** - 世界观/数值/系统一致性
- ⚠️ **风险识别** - 技术/成本/时间风险
- 💡 **优化建议** - 体验/平衡/商业化
- 📊 **竞品对比** - 同类产品参考

**输出格式**：
```markdown
## 策划案评审报告

### 综合评分：85/100

### ✅ 优点
- 系统设计完整，闭环清晰
- 商业化路径明确

### ⚠️ 风险点
1. 数值成长曲线过陡（高风险）
2. 技术实现复杂度高（中风险）

### 💡 优化建议
1. 建议添加保底机制
2. 简化新手引导流程
```

## 🚀 快速开始

### 使用方式 1: 自然语言对话

```
用户：帮我写一个完整的战斗系统策划案

Arshis：好的，正在生成战斗系统策划案...
[输出完整策划文档]
```

### 使用方式 2: 命令行工具

```bash
# 生成策划案
python3 scripts/generator.py --type system --topic "战斗系统" --output "combat_design.md"

# 评审策划案
python3 scripts/reviewer.py --input "economic_design.md" --output "review_report.md"

# 世界观一致性检查
python3 scripts/worldview_check.py --input "character_design.md"
```
