# Parent Activity Log

This file tracks your kid's building sessions. Each session is logged automatically.

---

## How to Read This Log

Each entry includes:
- **Date & Time** — When the session happened
- **Duration** — How long the kid was building
- **Projects Created** — What they built
- **What the kid Asked For** — Their original idea
- **How It Went** — Summary of the session
- **Flags** — Any concerns (rare, but noted if they occur)

---
