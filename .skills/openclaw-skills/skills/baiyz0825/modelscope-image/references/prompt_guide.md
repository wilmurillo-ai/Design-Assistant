# AI 艺术提示词生成指南

本文档提供了为 AI 绘图生成高质量提示词的完整指南，适用于魔搭社区的文生图模型。

## 核心原则

### 提示词结构公式

**主体 + 细节描述 + 背景 + 风格 + 光影**

这个结构确保了提示词的完整性和有效性，让 AI 能够准确理解并生成高质量的图片。

## 详细说明

### 1. 主体（Subject）

**定义**：画面中最核心、最突出的元素。

**要素**：
- 人物、动物、物体、建筑、景观等
- 清晰、明确的描述
- 置于提示词开头（权重最高）

**示例**：
- `一只优雅的猫咪`
- `现代摩天大楼`
- `神秘的魔法森林`
- `年轻女性肖像`

**技巧**：
- 使用具体名词而非抽象概念
- 可以包含主体的基本特征
- 示例：`一只橘色的猫` > `一只猫`

### 2. 细节描述（Details）

**定义**：对主体的具体刻画，增强画面的丰富性和真实感。

**要素**：
- **动作**：行走、飞翔、微笑、沉思
- **表情**：温柔、神秘、欢乐、忧郁
- **服饰**：晚礼服、西装、古装、未来服饰
- **材质**：丝绸、金属、玻璃、木材
- **颜色**：具体的色彩描述
- **纹理**：光滑、粗糙、细腻、斑驳

**示例**：
- `蓬松的毛发，灵动的眼神，优雅的姿态`
- `精致的妆容，温柔的眼神，华丽的晚礼服`
- `金属质感的外壳，光滑的玻璃幕墙`

**技巧**：
- **具体化**：从宏观到微观逐步描述
- **避免模糊**：`精致的妆容` > `漂亮的妆容`
- **使用形容词**：增强表现力

### 3. 背景（Background）

**定义**：主体所处的环境或场景，为画面提供上下文。

**要素**：
- 自然环境：森林、海洋、山脉、天空
- 城市环境：街道、建筑、广场、室内
- 抽象环境：渐变色、纹理、星空
- 环境氛围：雾气、雨景、夜景

**示例**：
- `温馨的家居环境`
- `繁华的城市夜景`
- `宁静的湖畔`
- `梦幻的星空背景`

**技巧**：
- 与主体相协调
- 营造合适的氛围
- 不要过于复杂以免分散注意力

### 4. 风格（Style）

**定义**：希望图像呈现的艺术风格或视觉效果。

**要素**：
- **艺术流派**：印象派、超现实主义、抽象表现主义
- **艺术家风格**：慕夏、莫奈、梵高、宫崎骏
- **视觉风格**：电影质感、摄影风格、插画风格
- **技术风格**：数字艺术、概念艺术、3D渲染

**示例**：
- `电影级肖像风格`
- `印象派油画风格`
- `慕夏风格`
- `赛博朋克风格`
- `宫崎骏动画风格`

**技巧**：
- **直接指定艺术家**：AI 会学习该艺术家的特征
- **组合风格**：可以融合多种风格
- **保持一致性**：风格应与主题匹配

### 5. 光影（Lighting）

**定义**：光线的类型、方向、强度和色彩，决定画面的氛围和立体感。

**要素**：
- **自然光**：阳光、月光、星光
- **时间**：黄金时刻、蓝色时刻、正午
- **人工光**：霓虹灯、聚光灯、蜡烛光
- **方向**：逆光、侧光、顶光
- **效果**：柔和、强烈、戏剧性

**示例**：
- `黄金时刻的暖光`
- `柔和的伦勃朗光`
- `戏剧性的侧光`
- `霓虹灯光`
- `梦幻的逆光效果`

**技巧**：
- 光影决定画面氛围
- 配合场景选择合适的光线
- 专业术语可以增强效果

## 生成技巧

### 1. 词序与权重

**原则**：排列在前的词语被 AI 赋予更高权重。

**示例**：
```
❌ 错误：电影级风格，柔和的光线，一只优雅的猫
✅ 正确：一只优雅的猫，蓬松的毛发，电影级风格，柔和的光线
```

**技巧**：
- 主体放在最前面
- 重要特征次之
- 风格和光影放在后面

### 2. 长度控制

**建议长度**：
- 英文：10-30 个单词
- 中文：20-50 个字

**原则**：
- 过短：AI 自由发挥过多，可能偏离预期
- 过长：失去焦点，信息过多导致混乱
- 适中：清晰明确，AI 能准确理解

**示例**：
```
❌ 过短：一只猫
✅ 适中：一只优雅的橘猫，蓬松的毛发，灵动的眼神，躺在沙发上，温馨的家居环境，电影级摄影风格，自然阳光
❌ 过长：一只非常可爱的橘色的猫，它有着蓬松柔软的毛发，大大的灵动的眼睛，它正懒洋洋地躺在一个非常舒适的米色的沙发上...（过度详细）
```

### 3. 具体化描述

**原则**：避免模糊词汇，使用具体、明确的描述。

**示例**：
```
❌ 模糊：绚丽的魔法世界
✅ 具体：骑着扫帚的魔法师，周围环绕漂浮的符文，神秘的电影感

❌ 模糊：漂亮的风景
✅ 具体：壮丽的雪山，云雾缭绕，倒影如镜的湖泊
```

### 4. 避免矛盾

**原则**：不要在提示词中包含相互冲突的概念。

**示例**：
```
❌ 矛盾：写实风格和抽象风格并存
❌ 矛盾：白天和夜晚的场景
✅ 一致：写实风格的肖像摄影
```

### 5. 使用形容词

**原则**：生动、具体的形容词能增强画面表现力。

**示例**：
```
❌ 平淡：一个女人
✅ 生动：优雅的女性，温柔的眼神，精致的妆容

❌ 平淡：一座建筑
✅ 生动：高耸的摩天大楼，玻璃幕墙闪烁
```

## 场景模板

示例：

城市夜景背景，电影级肖像风格，柔和的伦勃朗光
```

### 自然风景

```
[主体] + [细节] + [背景] + [风格] + [光影]

示例：

蔚蓝的天空，超写实摄影风格，黄金时刻光线

繁华的人群，赛博朋克风格，夜幕下的霓虹灯光
```

```
[主体] + [细节] + [背景] + [风格] + [光影]

## 常见问题

### Q: 提示词太简单怎么办？

A: 按照"主体-细节-背景-风格-光影"结构扩展：
- 增加主体的具体特征
- 添加环境和背景
- 指定风格和光影

### Q: 生成效果不理想怎么办？

A: 优化策略：
- 调整词序，重要元素前置
- 增加具体细节描述
- 指定明确的风格
- 控制提示词长度

### Q: 如何控制画面风格？

A: 方法：
- 直接指定艺术家：`慕夏风格`
- 指定艺术流派：`印象派`
- 指定视觉风格：`电影质感`
- 指定技术风格：`数字艺术`

### Q: 如何生成特定氛围的图片？

A: 关键在于：
- 选择合适的光影：`黄金时刻`、`夜幕`
- 描述环境氛围：`云雾缭绕`、`霓虹闪烁`
- 使用情感词汇：`神秘`、`温馨`、`忧郁`

## 高级技巧

### 1. 负向提示词（Negative Prompt）

用于排除不想要的元素：
- `模糊的、多手指、变形` - 提高细节质量
- `过曝、欠曝` - 控制曝光
- `水印、文字` - 排除干扰元素

**注意**：初期不建议过度使用，以免干扰 AI 理解。

### 2. 权重控制

某些模型支持通过语法控制权重：
- `(important word)` - 增加权重
- `[less important]` - 降低权重
- `(very important:1.5)` - 精确控制

### 3. 参考图片

使用 Img2Img 功能：
- 上传参考图片
- 文字描述优化
- 提高输出稳定性

### 4. 迭代优化

通过多次尝试优化：
- 第一次：基本结构
- 第二次：增加细节
- 第三次：调整风格
- 最终：完美呈现

## 总结

优秀的 AI 绘图提示词遵循以下原则：

1. **结构清晰**：主体 + 细节 + 背景 + 风格 + 光影
2. **具体明确**：避免模糊，从宏观到微观
3. **权重得当**：重要元素前置
4. **长度适中**：20-50 个中文字
5. **风格统一**：元素之间协调一致
6. **持续优化**：通过迭代不断改进

通过掌握这些技巧，你将能够生成高质量的 AI 艺术作品！
