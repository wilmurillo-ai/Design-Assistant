## 运镜提示词语法

### 强度增强
```
Dramatic [运镜词] -> 夸张效果
Extreme [运镜词] -> 极强效果
```

### 速度词汇
```
Slowly, Gradually -> 增加优雅感/沉重感
Fast, Rapidly, Burst -> 增加冲击力/惊恐感
```

### 镜头连接词
```
The camera starts with... and then... (先...然后...)
Seamless transition from [A] to [B] (无缝切换)
```

### 光效增强
- 恐惧镜头: Cinematic lighting, high contrast, chiascuro
- 温暖镜头: Golden hour, soft bokeh, sun flares

### 速度控制
- **Slow motion (24fps→120fps)**: 增加神圣感或悲剧感
- **Time-lapse**: 表现时间流逝或孤独
