# 脚本模板库

> 包含：5条完整脚本模板 + 通用脚本公式 + 标题公式 + 评论区话术

---

## 通用脚本公式

```
[1] 钩子（3秒）→ 抓注意力，反差/疑问/数字
[2] 自我介绍（5秒）→ 我是谁，专业背书
[3] 干货输出（20-40秒）→ 给价值，知识点/制作过程/故事
[4] 互动引导（3秒）→ 引导评论/关注
```

## 脚本一：台州菜 vs 粤菜对比（干货口播，45-60秒）

| 时间 | 画面 | 文案 |
|------|------|------|
| 0-3秒 | 看着镜头，微笑 | "台州菜和粤菜，表面上都是中餐，其实差别大了。" |
| 3-8秒 | 简单自我介绍 | "我是老挝一家中餐厅的厨师长，做了20多年中餐，今天说点内行话。" |
| 8-20秒 | 干货1 | "第一个区别：口味。粤菜讲究清淡原味，能清蒸就清蒸。台州菜呢？鲜甜为主，酱油糖少不了。" |
| 20-32秒 | 干货2 | "第二个区别：食材。粤菜偏爱海鲜，活鲜最重要。台州菜呢？东海海鲜是灵魂，带鱼、黄鱼不在话下。" |
| 32-42秒 | 干货3 | "第三个区别：做法。粤菜讲火候精准，台州菜讲入味彻底。一句话：粤菜吃鲜，台州菜吃味。" |
| 42-48秒 | 总结 | "你更爱吃台州菜还是粤菜？评论区告诉我。" |
| 48-50秒 | 结束 | "我是老挝厨师长，关注我，每天分享中餐干货。" |

**拍摄建议**：背景选厨房或餐厅；表情自信亲和；语速稍慢

## 脚本二：正宗糖醋鱼做法（做菜展示，60-90秒）

| 时间 | 画面 | 文案 |
|------|------|------|
| 0-3秒 | 油锅特写 | "糖醋鱼很多人做，但80%的人第一步就错了。" |
| 3-8秒 | 露脸+自我介绍 | "我是老挝中餐厅厨师长，今天教你正宗台州糖醋鱼。" |
| 8-15秒 | 讲解步骤1 | "第一步：鱼要选对。台州人用东海大黄鱼，个子要大，肉质要紧实。" |
| 15-25秒 | 讲解步骤2 | "第二步：挂糊。很多人在面粉里加水，正确做法是只加蛋黄，一滴水都不要。" |
| 25-40秒 | 讲解步骤3 | "第三步：油温。必须冒烟再下鱼，一次定型。这是外酥里嫩的关键。" |
| 40-50秒 | 讲解步骤4 | "第四步：糖醋汁。台州人放糖量比你想象的多，醋要后放，快出锅的时候。" |
| 50-58秒 | 完成品展示 | "出锅。看，皮脆肉嫩，酸甜适口。" |
| 58-65秒 | 互动 | "你们家做糖醋鱼有什么秘诀？评论区交流一下。" |

**拍摄建议**：双机位（面部+手部动作）；重点拍挂糊手法、油温状态、成品特写

## 脚本三：为什么坚持做正宗中餐（人物故事，45-60秒）

| 时间 | 画面 | 文案 |
|------|------|------|
| 0-3秒 | 看着镜头 | "在老挝做中餐，其实有很多选择。" |
| 3-10秒 | 讲现状 | "可以迎合当地口味，做老挝化中餐，成本低，客人多。" |
| 10-18秒 | 转折 | "但我没有。" |
| 18-30秒 | 讲原因 | "因为这里的华人，有的已经10年没回国了。他们想吃一口正宗的家常味道，太难了。" |
| 30-40秒 | 升华 | "我做的不只是一道菜，是他们对家乡的那份念想。" |
| 40-48秒 | 互动引导 | "你离开家乡多久了？还记得家里那道菜吗？评论区聊聊。" |
| 48-50秒 | 结束 | "我是老挝厨师长，关注我，带你看海外中餐人的真实生活。" |

**拍摄建议**：色调偏暖色；语速慢有情感；可穿插顾客吃饭画面

## 脚本四：厨师成长故事（人物故事，60-90秒）

| 时间 | 画面 | 文案 |
|------|------|------|
| 0-3秒 | 看着镜头 | "15年前，我从浙江一个小村子出来，手里只有200块钱。" |
| 3-12秒 | 回忆 | "第一份工作，在杭州一家小餐馆帮厨，切菜洗菜，一天才30块。" |
| 12-22秒 | 转折 | "第三年，终于上了灶台。师傅说：中餐没有捷径，只有手上有茧，才能出师。" |
| 22-35秒 | 经历 | "后来去了广东，拜师学粤菜。再后来，辗转去了东南亚。" |
| 35-45秒 | 现在 | "现在，我在老挝做厨师长，管着一个中餐厅团队。" |
| 45-55秒 | 感悟 | "这一路走过来，最深的体会就是：做菜这件事，走不得捷径。" |
| 55-65秒 | 互动 | "你在异乡打拼几年了？评论区和我说说你的故事。" |

**拍摄建议**：适合穿插老照片；可拍餐厅工作镜头；语速中等有故事感

## 评论区引导话术

**发布后必做**：
1. "感谢大家观看，有什么问题评论区见"
2. 根据评论内容回复3-5条
3. 引导关注："关注我，每天分享一个中餐干货"

**高互动话术**：
- "你更爱吃台州菜还是粤菜？评论区告诉我"

## 常见问题

**Q：说话紧张怎么办？**
A：先把脚本写好，对着念3遍再拍。不用追求完美，真实最重要。

**Q：一条拍不好怎么办？**
A：正常。先拍10条再看效果，前几条都是练手。

**Q：设备不够好能拍吗？**
A：完全能。手机+剪映+光线好，就够了。
