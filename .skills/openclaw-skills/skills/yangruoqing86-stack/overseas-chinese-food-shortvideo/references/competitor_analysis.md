# 竞品分析方法

## 分析步骤

### 1. 确定竞品范围

| 类型 | 示例 |
|------|------|
| 直接竞品 | 同城/同区域的中餐厅账号 |
| 间接竞品 | 泛中餐美食账号、餐饮从业者IP |
| 内容参考 | 爆款中餐视频博主（国内） |

### 2. 数据收集维度

对每个竞品账号记录：
- 粉丝数、点赞数
- 最近30天发布频率
- 播放量最高的3条视频
- 内容类型分布（干货/做菜/故事）
- 爆款视频的共同特征

### 3. 竞品内容拆解表

```
账号名：
粉丝量：
最近爆款视频：
├─ 标题：
├─ 播放量：
├─ 爆款原因（推测）：
└─ 可借鉴点：

内容类型分布：
├─ 干货知识：X%
├─ 做菜展示：X%
└─ 人物故事：X%
```

### 4. 差异化定位矩阵

| 维度 | 竞品做法 | 你的差异化 |
|------|---------|-----------|
| 菜系 | 粤菜为主 | 你的菜系特色 |
| 视角 | 纯做菜 | 厨师长专业视角 |
| 地域 | 国内为主 | 海外稀缺性 |
| 人设 | 不明显 | 真实创业故事 |
| 变现 | 不明确 | 到店引流闭环 |

### 5. 空白点识别

问自己：
- 竞品没人做的内容类型是什么？
- 竞品覆盖不到的人群是谁？
- 竞品没有的真实经历你有哪些？

**海外视角的独特优势**：
- 当地食材做中餐的融合创新（实验感）
- 海外华人对正宗中餐的渴望（情感共鸣）
- 中餐在海外的文化差异故事（稀缺性）
- 餐饮创业的真实经历（故事性强）

### 6. 竞品监测频率

- 建联表：前两周每天记录3个竞品账号动态
- 第3周起：每周汇总一次竞品爆款规律
- 每月：更新竞品分析表，优化内容方向

---

## 快速找到竞品的方法

1. 微信视频号/抖音搜索：所在城市+中餐/厨师/餐厅
2. 搜索：所在国家+中餐（如"老挝中餐"）
3. 搜索：中餐+厨师长/创始人IP
4. 关注同类账号的评论区，了解用户需求
