# 各类型脚本模板库

## 一、剧情反转类

### 模板1：身份反转

```
【类型】身份反转
【时长】60秒
【适用】涨粉、人设打造

0-3s：（震惊开场）
「你们知道我以前是做什么的吗？」
（画面：素颜/反差形象）

3-15s：（身份铺垫）
「我在工地搬了8年砖...
所有人都觉得我一辈子就这样了...」

15-35s：（转折）
「直到有一天，我女儿放学回家...
哭着问我：爸爸，为什么你不能像别人的爸爸一样...」

35-50s：（高潮）
「那一刻我下定决心要改变...
现在我有自己的公司，公司估值XXX...」

50-60s：（结尾钩子）
「想知道我是怎么做到的吗？关注我，下期告诉你们」
```

## 六、爆款标题公式

### 数字型
```
「3个动作，在家练出马甲线」
「5分钟学会，月薪3000到30000」
「7天瘦10斤，我是怎么做到的」
```

### 冲突型
```
「她农村出身，凭什么年入百万」
「好吃到邻居来敲门」
「月薪5000和50000的人差在哪里」
```

### 悬念型
```
「原来XX是这样来的...」
「难怪他越来越有钱...」
「我不敢发朋友圈的秘密...」
```

### 蹭热点型
```
「#狂飙 #高启强同款」
「#世界杯 梅西不告诉你的秘密」
「#双十一 这样买省1000」
```
