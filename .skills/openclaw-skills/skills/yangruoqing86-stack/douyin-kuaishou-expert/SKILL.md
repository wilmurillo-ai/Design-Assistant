| 维度 | 抖音 | 快手 |

## 爆款内容类型库

### 剧情类

**反转剧**：前3秒埋爆点，结尾神反转
```
结构：铺垫(3s) → 第一次反转(15s) → 冲突升级(30s) → 神反转(45s)
核心：让人猜不到结局
```

**共情剧**：精准戳中某个群体痛点
```
适用人群：宝妈/打工人/考研/北漂/独居青年
结构：痛点场景 → 情绪发酵 → 解决方案 → 温暖结局
```

**爽剧模板**：逆袭、打脸、复仇
```
结构：被看不起(3s) → 忍辱负重(15s) → 身份揭晓(45s) → 打脸时刻(60s)
```

### 种草类

**开场公式**：痛点共鸣 → 解决方案 → 产品展示 → 促单

**对比实验**：前后对比、竞品对比
```
格式：使用前（糟糕） → 揭秘原因 → 解决方案 → 使用后（惊艳）
```

**场景植入**：生活化场景自然融入
```
核心：让产品成为剧情道具，而不是广告
```

### 干货类

**身份共鸣开头**：精准人群标签
```
适用开场：「宝妈必看」「打工人收藏」「考研人稳住」
```

**结构公式**：痛点 → 干货 → 总结 → 钩子

**速成感设计**：让用户觉得「三分钟学会XXX」

### 热点类

**蹭热点方法**：
- 平台热门BGM/梗/挑战赛
- 节日/事件节点借势
- 爆款内容形式 + 自己领域二次创作

## 标题封面设计

### 标题公式

| 类型 | 公式 | 示例 |
|------|------|------|
| 数字型 | 数字+痛点+悬念 | 「3个动作，每天5分钟，在家练出马甲线」 |
| 冲突型 | 反常识+为什么 | 「为什么她素颜比化妆还好看？」 |
| 利益型 | 快速+简单+获得 | 「5分钟搞定一周早餐，学会同事都来问」 |
| 悬念型 | 话说一半+引发好奇 | 「原来这就是他年入百万的秘密...」 |
| 身份型 | 人群标签+共鸣 | 「宝妈们，你们还在这样哄孩子吗？」 |
| 蹭热点型 | 热点+自己的领域 | 「狂飙带货火了，品牌方应该跟进吗」 |

### 封面设计原则

**抖音**：竖版(9:16)，大字+强视觉+人物
```
要素：文字要大、颜色要鲜、人物要近、情绪要强
禁忌：不要太多字、不要模糊、不要与标题重复
```

**快手**：真实感 > 精致感，人物+场景
```
要素：真实场景、有代入感、有故事感
禁忌：过度美颜、棚拍感、太过商业化
```

## 算法与运营策略

### 抖音算法机制

```
流量池递进：初始300-500播放 → 完播率高则进入下一级
                   ↓
核心指标：完播率 > 点赞 > 评论 > 转发 > 收藏
                   ↓
账号权重：账号分决定基础流量池大小
```

**提升完播率技巧**：
- 开头3秒必须抓住注意力
- 视频节奏要快，不要拖沓
- 设置悬念，让用户看完找答案
- 时长控制在15-45秒最佳（新手）

### 快手算法机制

```
老铁关系：信任感 > 算法推荐
                   ↓
核心指标：复播率 > 点赞 > 评论 > 关注
                   ↓
私域价值：快手粉丝价值高于抖音
                   ↓
家族生态：可考虑加入或对标老铁家族
```

**提升复播率技巧**：
- 制造「没看完没看懂」的悬念
- 打造连续剧式内容
- 建立人物IP，让人想持续关注
- 评论互动要积极回复

### 发布时间参考

```
抖音：
- 早高峰 7:00-9:00（通勤场景）
- 午高峰 12:00-14:00（休息场景）
- 晚高峰 18:00-21:00（黄金时段）

快手：
- 老铁文化，早6-7点、晚20-22点
- 周末全天流量更均匀
```

## 标签策略

### 标签选择原则

```
数量：3-5个相关标签
类型组合：
- 1-2个大流量泛标签（#搞笑 #美食）
- 2-3个精准垂类标签（#减脂餐 #快手减脂）
- 1个热点/节日标签（#春节 #双十一）
```

### 抖音vs快手标签差异

| 抖音 | 快手 |
|------|------|
| 泛流量标签为主 | 垂类深耕标签为主 |
| 蹭热点标签效果强 | 老铁文化，真实标签更有效 |
| 避免太旧的梗 | 可以用方言/地域标签 |

## 工作流程

### 接到创作需求时

```
Step 1：明确目标
- 涨粉？带货？品宣？引流？
- 目标人群是谁？
- 预算和出镜人员配置？

Step 2：选赛道+找对标
- 分析同类爆款视频
- 拆解爆款元素（前3秒、正文结构、结尾钩子）
- 提炼可复用的内容模式

Step 3：出脚本
- 标题+封面文案
- 分镜脚本（画面+台词+时长）
- 背景音乐/音效建议
- 标签选择

Step 4：发布策略
- 发布时间
- 标签组合
- Dou+/粉条投放建议（可选）
```

## 常见问题处理

### 视频不火怎么办？
```
自检顺序：
1. 开头3秒够不够吸引？（重拍开头）
2. 节奏有没有拖沓？（精简内容）
3. 封面标题够不够好？（参考爆款模板）
4. 标签是否精准？（调整标签组合）
5. 发布时间是否正确？（测试不同时段）
```

### 带货转化低怎么办？
```
自检顺序：
1. 产品是否与粉丝需求匹配？
2. 信任感有没有建立好？
3. 憋单话术是否到位？
4. 价格是否在粉丝接受范围？
5. 评论区是否有人问尺码/售后？
```
