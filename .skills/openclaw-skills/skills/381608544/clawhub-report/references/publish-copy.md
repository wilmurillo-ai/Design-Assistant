# ClawHub Publish Copy

## Display name

ClawHub

## Short summary

写日报、写总结、写学习笔记，并能结合网络调研整理网络与安全主题的清晰输出。

## Detailed description

ClawHub 是一个偏中文写作与技术调研的通用技能，适合把零散输入整理成结构化输出。

它擅长：
- 日报、周报、月报
- 项目总结、复盘
- 学习笔记、知识卡片
- 网络与安全主题的资料调研与学习摘要

适用输入包括：
- 零散备注
- 聊天记录
- 待办清单
- 文档内容
- 网页链接
- 临时想法

输出风格强调：
- 结构清晰
- 中文自然
- 信息密度高
- 不乱编内容
- 网络与安全主题尽量基于新鲜资料

## Suggested tags

writing, chinese, summary, report, notes, research, networking, security, productivity

## Example prompts

- 帮我把今天这些工作记录整理成日报
- 根据这些任务和聊天内容写一份周报
- 把这次项目过程整理成复盘总结
- 帮我写一份关于零信任网络的学习笔记
- 搜索一下最近关于 eBPF 安全风险的资料，给我做一份中文摘要
- 帮我对比 VPN、Tailscale 和 Zero Trust 的差异，整理成内部分享提纲
