# SEO Scoring Rubric

Each dimension is scored 0–20. Total = 0–100.

## On-Page Optimization (0–20)

| Criteria | Points |
|---|---|
| Title tag present, 50–60 chars, contains primary keyword | 0–4 |
| Meta description present, 150–160 chars, has CTA | 0–3 |
| Single H1 tag, keyword-rich | 0–3 |
| Logical H2/H3 hierarchy | 0–2 |
| Image alt text on all/most images | 0–2 |
| Clean URL structure | 0–2 |
| Internal linking (3+ relevant links) | 0–2 |
| Schema markup present | 0–2 |

## Technical SEO (0–20)

| Criteria | Points |
|---|---|
| HTTPS | 0–4 |
| Mobile viewport meta tag | 0–4 |
| Canonical tag correct | 0–3 |
| No render-blocking issues (CSS/JS) | 0–3 |
| Structured data / JSON-LD | 0–3 |
| Robots meta not blocking | 0–3 |

## Content Quality (0–20)

| Criteria | Points |
|---|---|
| Word count appropriate for topic (800+ for most pages) | 0–5 |
| Keyword density 1–2% (not stuffed, not absent) | 0–4 |
| Content addresses user search intent clearly | 0–5 |
| No obvious duplicate content signals | 0–3 |
| Readability (Flesch > 60 for general audiences) | 0–3 |

## User Experience Signals (0–20)

| Criteria | Points |
|---|---|
| Page loads fast (infer from minimal inline styles, no heavy embeds) | 0–5 |
| Clear CTA above the fold | 0–4 |
| No intrusive interstitials/popups blocking content | 0–4 |
| Logical content structure (easy to scan) | 0–4 |
| Navigation accessible | 0–3 |

## Keyword Alignment (0–20)

| Criteria | Points |
|---|---|
| Primary keyword in title + H1 | 0–5 |
| Secondary keywords in H2s | 0–4 |
| LSI/semantic keywords naturally present | 0–4 |
| Meta description contains keyword | 0–3 |
| Keyword in URL slug | 0–4 |
