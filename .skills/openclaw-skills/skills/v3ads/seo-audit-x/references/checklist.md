# 50-Point SEO Audit Checklist

## Title & Meta (8 points)
- [ ] Title tag present
- [ ] Title 50–60 characters
- [ ] Primary keyword in title (near the front)
- [ ] Title is unique (not duplicated across pages)
- [ ] Meta description present
- [ ] Meta description 150–160 characters
- [ ] Meta description contains keyword + CTA
- [ ] Meta description is unique

## Heading Structure (6 points)
- [ ] Exactly one H1 per page
- [ ] H1 contains primary keyword
- [ ] H2s used for main sections
- [ ] H3s used for sub-sections
- [ ] No heading hierarchy skips (H1 → H3 without H2)
- [ ] Headings are descriptive, not generic ("Section 1")

## Content (8 points)
- [ ] 800+ words for informational pages
- [ ] Keyword density 1–2%
- [ ] Primary keyword in first 100 words
- [ ] LSI/semantic keywords present
- [ ] Content matches search intent (informational/transactional)
- [ ] No keyword stuffing
- [ ] Content is original (no obvious duplicate blocks)
- [ ] Readability score Flesch > 60

## Images (4 points)
- [ ] All images have alt text
- [ ] Alt text describes image (not just keyword-stuffed)
- [ ] Images are compressed (infer from page load feel)
- [ ] Descriptive file names (not img001.jpg)

## Internal Linking (4 points)
- [ ] 3+ internal links on content pages
- [ ] Anchor text is descriptive (not "click here")
- [ ] No broken internal links (infer from 404 signals)
- [ ] Links to relevant pages (topical relevance)

## URL Structure (3 points)
- [ ] URL contains primary keyword
- [ ] URL is short and readable
- [ ] No excessive parameters or session IDs

## Technical (10 points)
- [ ] HTTPS enabled
- [ ] Mobile viewport meta tag present
- [ ] Canonical tag present and correct
- [ ] robots meta tag not set to noindex
- [ ] No render-blocking CSS/JS (infer)
- [ ] JSON-LD or schema markup present
- [ ] Open Graph tags present (social sharing)
- [ ] Favicon present
- [ ] 404 page exists
- [ ] XML sitemap referenced

## Performance Signals (4 points)
- [ ] No large uncompressed images visible in source
- [ ] External fonts loaded efficiently
- [ ] No excessive third-party scripts
- [ ] Core Web Vitals signals (no layout shift triggers)

## Competitive Context (3 points)
- [ ] Page competes on target keyword (SERP analysis)
- [ ] Content is at least as long as top 3 ranking pages
- [ ] Unique angle/differentiator vs top results
