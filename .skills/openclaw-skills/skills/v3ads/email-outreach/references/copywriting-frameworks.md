# Email Copywriting Frameworks

## AIDA (Awareness → Interest → Desire → Action)
Best for: First touch cold emails to warm segments

```
Subject: [Specific problem they have]

[A] Hey {{first_name}}, noticed [specific observation about their company].

[I] Most [job title]s I talk to are dealing with [pain point] — which usually means [consequence].

[D] We help [ICP] [achieve outcome] without [common objection/cost].

[A] Worth a 15-min call this week?
```

## PAS (Problem → Agitate → Solution)
Best for: High-pain segments with clear, specific problems

```
Subject: [Blunt problem statement]

[P] {{company}} is [doing thing that's causing problem].

[A] That means [downstream consequence]. Most [companies like theirs] don't fix this until [expensive moment].

[S] We built [solution] specifically for [ICP]. [One sentence proof].

Reply if you want to see how it works.
```

## BAB (Before → After → Bridge)
Best for: Outcomes-focused buyers, ROI-driven personas

```
Subject: [Outcome they want]

[B] Right now: [current state of their world — specific and painful]

[A] 90 days from now: [specific outcome after using your solution]

[Bridge] [Company name] makes that happen by [mechanism]. [One proof point].

Want the details? 15 minutes?
```

## Trojan Horse
Best for: Hard-to-reach executives, skeptical buyers

```
Subject: Quick question

{{first_name}} — quick question for you:

[Relevant question about their business/challenge that you actually know the answer to]

Reason I ask: [Brief explanation of why this matters + soft intro to what you do]

[CTA]
```

## The Breakup Email (Email 4)
High-performing pattern — creates urgency without pressure

```
Subject: Closing the loop

{{first_name}},

I've reached out a few times about [topic] — no worries if the timing's off or it's not a fit.

I'll leave it here unless you'd like to reconnect.

[Your name]

P.S. If there's someone else at {{company}} who handles [problem area], I'm happy to reach out to them instead — just say the word.
```

---

## Subject Line Formulas

| Type | Formula | Example |
|---|---|---|
| Question | "[Pain point]?" | "Still manually exporting reports?" |
| Curiosity gap | "How [Company] [outcome] in [timeframe]" | "How Loom cut churn 30% in 6 weeks" |
| Direct | "[Outcome] for {{company}}" | "More leads for Acme Corp" |
| Personalized | "{{company}} + [your company]" | "Acme Corp + Clearbit" |
| Contrarian | "Don't [common advice]" | "Don't send another cold email" |
| Number | "[X] [things] for {{company}}" | "3 fixes for Acme's onboarding drop-off" |

## Personalization Starters (First Lines)

Rotate these to avoid pattern detection:
- "Saw that {{company}} just [recent news / funding / hire] — congrats."
- "Your post about [topic] on LinkedIn last week resonated."
- "Noticed {{company}} is hiring [role] — usually means [inference]."
- "{{company}}'s [product/feature] is interesting — I had a question about [relevant detail]."
- "I've been following {{company}} since [specific thing] — [observation]."

---

## Spam Trigger Words to Avoid

High-risk: free, guarantee, no risk, limited time, act now, earn money, make money, winner, congratulations, click here, unsubscribe, buy now, order now, cash, prize, opportunity, discount, earn extra cash, increase sales, lose weight, miracle, promise, save up to

Use natural language. Write like a human, not a funnel.
