# gh-industry-deep-analysis - 行业深度分析框架

**版本：** 1.1.0  
**作者：** 投资助手  
**Slughub:** gh-industry-deep-analysis  
**Display Name:** 行业深度分析框架  
**描述：** 基于最新数据和资讯，对行业进行全面深入分析（十维全维度深度分析，10000-20000 字）

## 核心特性

- ✅ 十维全维度深度分析
- ✅ 市场前景深度预测
- ✅ 产业链全景拆解
- ✅ 成本结构详细分析
- ✅ 核心公司深度研究
- ✅ 输出长度 10000-20000 字

## 使用方式

直接发送行业名称或分析请求：
- "光模块行业"
- "分析一下半导体行业"
- "新能源汽车产业链"

## 标签

行业分析，深度研究，产业链，成本结构，投资框架
