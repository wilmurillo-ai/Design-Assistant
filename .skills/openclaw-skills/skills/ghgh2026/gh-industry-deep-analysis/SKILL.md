## 📊 十维行业分析框架（全维度深度分析）

### 默认分析深度：深度分析 ⭐⭐⭐⭐⭐

| 分析深度 | 输出长度 | 适用场景 |
|----------|----------|----------|
| **深度分析** | 10000-20000 字 | 首次覆盖、重点行业、投资决策 |
| 快速分析 | 3000-5000 字 | 日常跟踪、简单查询 |

### 十维全部深度展开（无简化章节）

| 维度 | 权重 | 优先级 | 核心内容 | 数据源 |
|------|------|--------|----------|--------|
| 1️⃣ **行业概览** | 10% | P0 | 定义、分类、市场规模、增速 | mx-search + Tavily |
| 2️⃣ **业务内容** | 12% | P0 | 主要产品/服务、商业模式、盈利模式 | mx-search |
| 3️⃣ **发展阶段** | 10% | P0 | 导入期/成长期/成熟期/衰退期判断 | mx-search + Tavily |
| 4️⃣ **市场前景** | 12% | P0 | 市场空间、增速预测、驱动因素、细分市场 | mx-search + Tavily |
| 5️⃣ **产业链拆解** | 15% | P0 | 上游/中游/下游、核心环节、价值分布、壁垒分析 | mx-search + Tavily |
| 6️⃣ **成本结构** | 12% | P0 | 核心部件成本占比、毛利率分析、成本趋势 | mx-search + 年报 |
| 7️⃣ **竞争格局** | 10% | P0 | CR5/CR10、市场份额、竞争态势 | mx-search |
| 8️⃣ **核心公司** | 8% | P0 | 龙头公司深度分析、技术优势、市占率、财务对比 | mx-data + mx-search |
| 9️⃣ **政策环境** | 6% | P0 | 支持/限制政策、监管趋势 | Tavily |
| 🔟 **风险与机遇** | 5% | P0 | 主要风险、投资机会 | mx-search + Tavily |
| **综合** | **100%** | - | - | - |

## 🛠️ 工具调用规范

### 必用工具（P0 优先级）

| 工具 | 用途 | 时效性要求 | 调用说明 |
|------|------|------------|----------|
| **mx-search** | 最新新闻、公告、研报（7 天内优先） | 7 天内 | 获取行业最新动态、机构观点 |
| **Tavily** | 全网深度搜索、行业数据、政策文件 | 30 天内 | 补充行业数据、政策、竞争格局 |

### 辅助工具（P1 优先级）

| 工具 | 用途 | 时效性要求 |
|------|------|------------|
| **mx-data** | 行业内上市公司财务数据、行情 | 实时 |
| **妙想资讯** | 行业数据、产业链信息 | 30 天内 |

### 数据交叉验证原则

```
重要数据必须至少 2 个独立来源验证：
├── 市场规模 → 多份研报交叉 + 行业协会数据
├── 成本占比 → 招股书 + 年报 + 调研纪要
├── 市占率 → 多份研报 + 行业协会 + 公司官网
├── 政策文件 → 政府官网 + 官方媒体
└── 技术趋势 → 专利数据 + 专家访谈 + 研报
```

## ⚠️ 注意事项

1. **数据时效性** - 所有数据必须使用最新可用数据，标注数据日期
2. **交叉验证** - 重要数据必须至少 2 个独立来源验证
3. **成本占比** - 必须基于招股书/年报/调研纪要，不得估算
4. **合规声明** - 必须包含"仅供参考，不构成投资建议"声明
5. **客观中立** - 分析基于事实和数据，不主观臆测
