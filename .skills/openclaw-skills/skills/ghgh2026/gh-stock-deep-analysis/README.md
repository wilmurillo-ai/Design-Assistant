# gh-stock-deep-analysis - 股票深度分析框架 4.1

## 📖 使用说明

### 触发方式

直接发送以下内容即可触发股票深度分析：

1. **股票名称** - 如 `贵州茅台`、`宁德时代`、`腾讯控股`
2. **股票代码** - 如 `600519`、`300750`、`0700.HK`、`AAPL`
3. **明确请求** - 如 `分析一下贵州茅台`、`宁德时代深度分析`

### 分析框架

本 skill 使用**完整的专业分析框架 4.1**，包含八维深度分析：

| 维度 | 权重 | 核心内容 |
|------|------|----------|
| 1️⃣ 行业分析 | 23% | 市场空间、竞争格局、周期位置、技术趋势 |
| 2️⃣ 业务拆解 | 18% | 产品结构、市占率、竞争力评分 |
| 3️⃣ 产业链分析 | 13% | 上游供应商、下游客户 |
| 4️⃣ 财务分析 | 13% | 成长性、盈利能力、现金流 |
| 5️⃣ 估值分析 | 10% | PE/PB/PEG、历史分位、同业对比 |
| 6️⃣ 资金面分析 | 10% | 北向资金、基金持仓 |
| 7️⃣ 风险分析 | 5% | 系统性风险、行业风险、个股风险 |
| 8️⃣ 催化剂时间表 | 8% | 未来 1-6 个月关键事件 |

### 输出内容

- ✅ 核心数据速览
- ✅ 八维详细分析
- ✅ 综合评分 + 催化剂评分
- ✅ 投资建议（目标价、止损位、仓位）
- ✅ 风险提示

### 数据来源

- **mx-data** - 实时行情、财务数据
- **mx-search** - 最新新闻、公告、研报
- **妙想资讯** - 行业数据、产业链信息
- **妙想金融数据** - 财务指标、估值数据
- **Tavily** - 全网深度搜索补充

## 🔧 技术细节

### 触发正则表达式

```regex
/^(贵州茅台 | 宁德时代 | 腾讯控股 | 阿里巴巴 | 苹果公司 | [0-9]{6}|[0-9]{4}\.HK|SH[0-9]{6}|SZ[0-9]{6}|AAPL|TSLA|MSFT).*$/
```

### 工具调用顺序

1. mx-data → 获取实时行情、财务数据
2. mx-search → 搜索最新新闻、公告、研报
3. 妙想资讯 → 获取行业数据、产业链信息
4. 妙想金融数据 → 获取财务指标、估值数据
5. Tavily → 全网深度搜索补充

### 数据交叉验证

```
财务数据 → mx-data + 财报原文 + 研报
订单数据 → 公告 + 调研纪要 + 妙想资讯
市占率 → 多份研报交叉 + 行业协会
价格数据 → mx-data + 行业网站 + 新闻
```
