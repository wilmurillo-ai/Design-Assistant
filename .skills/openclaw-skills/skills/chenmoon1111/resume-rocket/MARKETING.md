### 标题
```
[分享] 做了个开源 skill：用 AI 把简历改成 JD 的满分匹配版
```

### 正文
```markdown
最近帮朋友跳槽改简历，改到想吐。想了想做成工具。

## 做了什么

Resume Rocket，OpenClaw 生态的一个 skill。

核心链路：
1. 简历（PDF/DOCX/MD）→ 解析成结构化数据
2. JD（Boss URL / 拉勾 / 纯文本）→ 提取关键词 + 必须项 + 加分项
3. 规则 + LLM 两层评分（100 分制，技术/年限/学历/亮点四维度）
4. LLM 重写经历描述：
   - 用 JD 原词表达（过 ATS）
   - STAR 结构化
   - 量化成果
   - **不虚构，只挖掘**

## 设计原则

- **不造假**：只重写候选人真做过的事
- **本地优先**：简历数据不传云，LLM 调用走用户自己的 Key
- **多 LLM 兼容**：阿里百炼 / DeepSeek / 智谱 / OpenAI 都行

## 技术栈

Python · python-docx · pdfplumber · OpenAI SDK（compat 模式打各家）

## 试用

```bash
clawhub install resume-rocket
openclaw skill run resume-rocket match-score --resume ./resume.pdf --jd "JD 文本"
```

## 定价

免费 1 次/天。付费 29/99/299（具体见 SKILL.md）

## 为什么不免费开源

开发成本 + 持续维护 + LLM prompt 优化的试错成本。
core 代码是开源的（MIT），付费部分是「面试卡 + 批量投递 + 激活码」。
你愿意自己 fork 改全开源版也完全可以。

## 不足

- Boss JD 抓取目前走静态 HTML 解析，拿不到动态加载的详情（下版接 Selenium）
- 只测了中文简历，英文简历还没调
- 面试卡生成偶尔过于套路化，prompt 还在调

欢迎 issue 和 PR：[链接]
```
