## 架构趋势

- **MoE（混合专家）**：成为主流，稀疏激活降成本
- **Mamba/SSM**：状态空间模型挑战 Transformer 效率
- **长上下文**：100K-1M token 成为高端模型标配
- **小模型崛起**：端侧、离线场景推动 1B-7B 高效模型

## 能力趋势

- 多模态原生融合（不只是"拼接"）
- Agent 能力爆发：规划、工具调用、记忆
- 推理时计算（Test-Time Compute）扩展
- 低成本推理成为竞争焦点
