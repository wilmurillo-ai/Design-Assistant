# ✈️ 捡漏机票搜索工具 (Bargain Flights)

> **「捡漏机票」——不是找更便宜的票，是找更聪明的买法。**

## 📖 什么是捡漏机票

捡漏机票，也叫**隐藏城市票价**（Hidden City Ticketing），是一种利用机票定价差异的聪明购票策略。

## 使用演示

直接发送 `/bargain-flights` 命令加上行程信息即可：

```
/bargain-flights 帮我捡漏下北京到沈阳6月25日的航班
```
