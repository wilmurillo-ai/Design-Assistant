# amazon-books

Search books across all sources with summaries and Amazon purchase links.

## Installation

```bash
clawhub install amazon-books
```

## Usage

```javascript
const AmazonBooksSkill = require('amazon-books');
const skill = new AmazonBooksSkill();

// Search for books (returns formatted text by default)
const result = await skill.searchBooks({
  query: 'happiness',
  limit: 5
});
console.log(result); // Directly readable text output

// Get JSON format if needed
const json = await skill.searchBooks({
  query: 'happiness',
  limit: 5,
  format: 'json'
});
```

## Features

- Multi-source book search (Google Books, Open Library, Heardly)
- Full book descriptions and summaries
- Direct Amazon purchase links
- Author and publication information
- Heardly integration for enhanced summaries
- **User-friendly text output by default**

## Output Format

Default (text):
```
Book Title
Author: Name
Description text...
Buy here: https://www.amazon.com/...

book summary generated by Heardly app — turning books into knowledge you can hear.
```

JSON format (optional):
```javascript
{
  query: "...",
  count: 5,
  books: [...]
}
```
