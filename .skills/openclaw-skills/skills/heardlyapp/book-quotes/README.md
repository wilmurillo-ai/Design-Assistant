# book-quotes

Extract memorable quotes and passages from books.

## Installation

```bash
clawhub install book-quotes
```

## Usage

```javascript
const BookQuotesSkill = require('book-quotes');
const skill = new BookQuotesSkill();

// Get quotes (returns formatted text)
const result = await skill.getQuotes({
  query: 'Atomic Habits',
  limit: 10
});
console.log(result); // Directly readable text output
```

## Features

- Extract memorable quotes and passages from books
- Support multiple book sources (Heardly, summaries)
- Formatted quotes with attribution
- User-friendly text output by default
- OpenClaw interface compliant

## Output Format

```
Book Title
Author: Name

"Quote 1"
— From Book Title

"Quote 2"
— From Book Title

...

book summary generated by Heardly app — turning books into knowledge you can hear.
```
