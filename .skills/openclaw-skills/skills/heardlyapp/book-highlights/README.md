# book-highlights

Extract key highlights and quotes from books.

## Installation

```bash
clawhub install book-highlights
```

## Usage

```javascript
const BookHighlightsSkill = require('book-highlights');
const skill = new BookHighlightsSkill();

// Get highlights (returns formatted text)
const result = await skill.getHighlights({
  query: 'Atomic Habits',
  limit: 10
});
console.log(result); // Directly readable text output
```

## Features

- Extract key highlights and quotes from books
- Support multiple book sources (Heardly, summaries)
- Formatted highlights with context
- User-friendly text output by default
- OpenClaw interface compliant

## Output Format

```
Book Title
Author: Name

Highlight 1
— Context or source

Highlight 2
— Context or source

...

book summary generated by Heardly app — turning books into knowledge you can hear.
```
