# About NexLink

## What is NexLink?

NexLink is an OpenClaw skill that connects **Microsoft Exchange** and **Nextcloud** into one practical workflow layer. It gives teams a single assistant interface for email, calendar, tasks, file management, and document understanding.

## Built for Teams That Use Exchange + Nextcloud

If your organization runs **Microsoft Exchange** (on-premises or hybrid) and **Nextcloud** for file storage, NexLink eliminates the context switching between mail client, calendar, task manager, and file browser.

## What You Can Do

| Area | Actions |
|------|---------|
| **Email** | Read, send, draft, reply, forward, download attachments |
| **Calendar** | View today/week, create meetings, respond to invites |
| **Tasks** | List, create, complete, trash — including delegate workflows |
| **Analytics** | Inbox stats, response time, top senders, heatmaps |
| **Files** | List, search, upload, download, move, copy, share |
| **Documents** | Extract text, summarize, ask questions, extract action items |

## Key Workflows

- **Send a Nextcloud file by email** — download from Nextcloud, attach to Exchange email
- **Save email attachments to Nextcloud** — download from Exchange, upload to Nextcloud
- **Turn documents into tasks** — extract action items from contracts, create Exchange tasks
- **Delegate task management** — manage tasks for team members with proper Exchange permissions

## How It Works

NexLink uses:
- **EWS (Exchange Web Services)** for Exchange operations
- **WebDAV + OCS APIs** for Nextcloud file operations
- **OpenClaw skill framework** for integration with the assistant layer

## Requirements

- Python 3.10+
- Exchange Server 2016/2019 (on-premises) or hybrid
- Nextcloud instance with app password access
- Network access to both services
