# 飞书消息智能摘要

自动分析和总结飞书消息内容，帮助用户快速掌握关键信息，提高沟通效率。

## 核心功能

1. **消息内容摘要**
   - 提取群聊或私聊的核心讨论内容
   - 识别关键决策和行动项
   - 归纳主要话题和观点

2. **智能分类整理**
   - 按主题自动分组消息
   - 区分重要和次要信息
   - 识别待办事项和截止日期

3. **参与者分析**
   - 统计发言频率和活跃度
   - 识别关键参与者和决策者
   - 追踪@提及和任务分配

4. **时间维度统计**
   - 按时间段分析消息分布
   - 识别高峰讨论时段
   - 追踪话题演变趋势

## 适用场景

- **会后复盘**：快速回顾群聊中的会议讨论，提取决策要点
- **日常跟进**：定期总结项目群的进展和待办事项
- **信息归档**：整理重要讨论内容，生成可存档的摘要文档
- **跨时区协作**：帮助错过实时讨论的成员快速了解关键信息
- **新人上手**：为新加入的成员提供历史讨论概览

## 使用示例

**基础用法：**
```
总结今天项目群里的消息
```

**指定时间范围：**
```
总结上周一到周五 #产品讨论群 的主要内容
```

**提取特定信息：**
```
从最近的消息中提取所有待办事项和截止日期
```

**生成结构化报告：**
```
分析本月技术群的讨论，按技术栈分类整理
```

## 输出格式

生成的摘要通常包含以下结构：

- **概要**：整体讨论主题和关键结论
- **重要决策**：达成的共识和决定事项
- **待办事项**：需要跟进的任务（含责任人和时间）
- **关键讨论**：主要话题和观点分歧
- **统计信息**：消息数量、参与人数、活跃时段等

## 最佳实践

1. **明确范围**：指定具体的群聊名称和时间范围，获得更准确的摘要
2. **定期总结**：建立定期总结的习惯，避免信息遗漏
3. **结合导出**：可将摘要保存为飞书文档，便于团队共享
4. **迭代优化**：根据输出结果调整提问方式，获得更符合需求的摘要
5. **权限注意**：确保有相应群聊的访问权限
