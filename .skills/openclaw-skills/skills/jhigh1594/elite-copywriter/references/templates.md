# Copywriting Templates Library

Ready-to-use templates for common content types. Reference when actively writing specific deliverables.

**IMPORTANT**: These templates show *structure and flow*, not literal labels to include. Don't use the section headers (CONTEXT, RECOMMENDATION, etc.) in your final output — write natural, human communication instead.

---

### Executive Email Template

**Structure guide — don't include these labels in your output:**

```
SUBJECT: [Specific outcome or decision needed]

[Lead with your key point — what do you need and why it matters]

[2-3 sentences of essential background if context is needed]

[If presenting options, state them clearly with brief descriptions]

[Your recommendation with top 2-3 reasons]

[What happens next and by when]

[Your name]
```

**Example (natural output):**

```
Subject: Global Attributes timeline — decision needed by Friday

We need to decide whether to delay the Connected Work Graph work to prioritize Global Attributes. This affects the EBSCO-scale deals in pipeline and our PI1 commitments.

Global Attributes was a key factor in the EBSCO loss. We have 3 similar deals in late stages where this gap has been raised. Team-to-team planning is also a competitive gap every enterprise prospect cites.

CWG is a strategic bet executives are making for the $6M Anvi target, but the revenue linkage isn't yet proven with customers.

My recommendation: Prioritize Global Attributes and team-to-team planning. These are proven deal risks we're actively losing. CWG can slip to PI2 without breaking customer commitments.

If approved, I'll re-plan PI1 scope and communicate with engineering by EOD Friday.

Jamie
```

## <� STYLE & TONE GUIDELINES

### Voice Characteristics
```
CONFIDENT: Assertive without arrogance
CLEAR: Simple language, avoid jargon
COLLABORATIVE: "We" not "I" focused
EVIDENCE-BASED: Data supports claims
FORWARD-LOOKING: Solution-oriented
```

### Tone Adaptation by Audience
```
EXECUTIVES: Concise, strategic, outcome-focused
TECHNICAL TEAMS: Detailed, process-oriented, implementation-focused
CUSTOMERS: Benefit-focused, empathetic, value-driven
PEERS: Collaborative, data-rich, nuanced
SALES: Competitive, differentiated, compelling
```

### Power Words by Category
```
URGENCY: Immediate, critical, deadline, limited, exclusive
AUTHORITY: Proven, expert, leading, benchmark, industry-standard
INNOVATION: Revolutionary, breakthrough, cutting-edge, pioneering
RESULTS: Accelerate, optimize, transform, eliminate, multiply
EASE: Simple, streamlined, automated, effortless, intuitive
```

## >� TESTING & OPTIMIZATION

### A/B Testing Framework
```
ELEMENT TO TEST: [Subject line, CTA, value prop, etc.]
HYPOTHESIS: "We believe [change] will [outcome] because [reason]"
METRICS: [Open rate, click rate, conversion, etc.]
SAMPLE SIZE: [Statistical significance threshold]
DURATION: [Test length for valid results]
```

### Message Testing Questions
```
CLARITY: Can a 12-year-old understand this?
RELEVANCE: Does this matter to my specific audience?
URGENCY: Is there a clear reason to act now?
CREDIBILITY: What proof supports this claim?
SIMPLICITY: Could I explain this in one sentence?
```

## Template vs. Reality

**Remember**: These templates show structure. Don't copy the labels into your output.

**Bad (artificial):**

```
BOTTOM LINE: We need to decide on CWG prioritization.

CONTEXT: Three initiatives are competing for PI1 capacity.

RECOMMENDATION: Prioritize competitive features because they're proven deal risks.

NEXT STEPS: Decision by Friday.

RATIONALE: EBSCO loss demonstrates the revenue impact.
```

**Good (natural):**

```
We need to decide on CWG prioritization this week.

Three initiatives are competing for the same PI1 capacity: CWG, Global Attributes, and team-to-team planning. We can only deliver one well.

I recommend prioritizing the competitive features. Global Attributes was a key factor in the EBSCO loss, and team-to-team planning is a gap every enterprise prospect cites. CWG is a strategic bet for the Anvi target, but the revenue linkage isn't yet proven.

Need a decision by Friday so I can update sprint planning.

Jamie
```
