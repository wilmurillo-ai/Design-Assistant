## 导入说明

本清单可直接导入主流项目管理工具：

| 工具 | 导入方式 |
| :--- | :--- |
| **Jira** | 创建 Epic → Story → Sub-task，字段映射：Story Points = 估算天数 × 系数 |
| **Linear** | 创建 Project → Cycle → Issue，优先级直接映射 P0/P1/P2 |
| **Notion** | 直接粘贴 Markdown，或使用数据库视图按 Epic 分组 |
| **飞书多维表** | 按表格结构导入，Task ID 作为唯一标识 |
