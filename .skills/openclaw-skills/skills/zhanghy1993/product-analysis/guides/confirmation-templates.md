# 确认话术模板

本文件提供每个步骤完成后的标准确认话术模板。

**使用规范**：将占位符 `[此处展示...]` 替换为实际产出内容，将 `[ ]` 改为 `[x]`，暂停并等待用户回复。
详细执行规范和错误示例见 `guides/step-confirmation-checklist.md`。
