# 周报生成器 - AI 智能周报自动化系统

## 📋 产品概述

周报生成器是一款基于 AI 的智能周报自动化工具，能够从 Git 提交记录、工作描述、工作截图中自动提取信息，生成高质量、业务化的周报文档，并自动发送到飞书。彻底解放双手，让写周报从"痛苦的回忆作业"变成"一句话的事"。

---

## ✨ 核心亮点

### 🤖 全自动化流程

- **一键生成**：一句"按照skill帮我生成周报"，剩下的全部自动完成
- **零配置使用**：无需手动统计代码、整理提交记录
- **智能发送**：生成后自动推送到飞书，无需手动操作

### 🧠 AI 智能优化

- **业务化表达**：自动将技术提交转换为业务语言
  - 从 `"SXGH-1183 新增封面"`
  - 到 `"搭建霍兰德测评核心框架，完成报告封面页面设计与实现"`
- **智能合并**：自动识别并合并相同功能的多次提交
- **分类归纳**：自动将工作内容分类为功能开发、工具建设、会议协作等

### 📊 全面数据统计

- **多仓库支持**：自动扫描指定目录下所有 Git 仓库
- **精准统计**：提交次数、代码行数、工作天数全自动计算
- **可视化数据**：清晰展示工作量和成果

### 🎯 专业格式输出

- **标准模板**：符合企业周报规范的专业格式
- **突出亮点**：自动识别并强调本周重要成果
- **可读性强**：清晰的分类结构，便于阅读和汇报
