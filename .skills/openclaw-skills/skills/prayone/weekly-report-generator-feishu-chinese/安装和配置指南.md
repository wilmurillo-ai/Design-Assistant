## 🚀 快速开始

### 1. 复制脚本文件

将 `scripts` 目录下的所有文件复制到你的工作目录（推荐：`~/ai/cline-skills/`）：

```bash
# 创建工作目录
mkdir -p ~/ai/cline-skills

# 复制脚本文件
cp scripts/auto-weekly-report.sh ~/ai/cline-skills/
cp scripts/send-to-feishu.sh ~/ai/cline-skills/

# 添加执行权限
chmod +x ~/ai/cline-skills/auto-weekly-report.sh
chmod +x ~/ai/cline-skills/send-to-feishu.sh
```

## ⚙️ 配置脚本

### 1. 配置 auto-weekly-report.sh

编辑 `~/ai/cline-skills/auto-weekly-report.sh`，修改以下配置：

```bash
# 工作目录（周报文件保存位置）
WORK_DIR="/Users/你的用户名/ai/cline-skills"

# 项目根目录（Git 仓库所在目录）
PROJECT_ROOT="/Users/你的用户名/project/你的项目目录"
```

### 2. 配置 send-to-feishu.sh

编辑 `~/ai/cline-skills/send-to-feishu.sh`，填写飞书配置：

```bash
APP_ID="your_app_id"           # 飞书应用 App ID
APP_SECRET="your_app_secret"   # 飞书应用 App Secret
RECEIVE_ID="your_open_id"      # 你的飞书 open_id
```

## 🔐 配置飞书

### 步骤 1：创建飞书应用

1. 访问 [飞书开放平台](https://open.feishu.cn/)
2. 登录并进入"开发者后台"
3. 点击"创建企业自建应用"
4. 填写应用名称（如：周报助手）和描述

### 步骤 2：获取应用凭证

1. 进入应用详情页
2. 在"凭证与基础信息"中获取：
   - **App ID**
   - **App Secret**
3. 复制这两项内容到 `send-to-feishu.sh` 配置区

### 步骤 3：配置应用权限

在"权限管理"中添加以下权限：

- `im:message` - 发送消息
- `im:message:send_as_bot` - 以应用身份发消息

点击"发布版本"使权限生效。

### 步骤 4：获取你的 open_id

方法一：通过机器人获取

1. 在飞书中搜索"获取 open_id"机器人
2. 发送任意消息，机器人会返回你的 open_id

方法二：通过 API 获取

```bash
# 使用你的 tenant_access_token 调用
curl -X GET "https://open.feishu.cn/open-apis/contact/v3/users/me" \
  -H "Authorization: Bearer YOUR_TENANT_ACCESS_TOKEN"
```

## ⏰ 配置定时任务（macOS）

### 步骤 1：复制 plist 文件

```bash
# 复制定时任务配置
cp scripts/com.user.weekly-report.plist ~/Library/LaunchAgents/

# 修改配置文件中的路径为你的实际路径
```

### 步骤 2：编辑定时配置

编辑 `~/Library/LaunchAgents/com.user.weekly-report.plist`：

```xml
<!-- 修改脚本路径 -->
<string>/Users/你的用户名/ai/cline-skills/auto-weekly-report.sh</string>

<!-- 修改工作目录 -->
<string>/Users/你的用户名/ai/cline-skills</string>

<!-- 修改定时时间（示例：每周四上午 09:52）-->
<key>Weekday</key>
<integer>4</integer>  <!-- 4 = 周四 -->
<key>Hour</key>
<integer>9</integer>
<key>Minute</key>
<integer>52</integer>
```

### 步骤 3：加载定时任务

```bash
# 加载任务
launchctl load ~/Library/LaunchAgents/com.user.weekly-report.plist

# 查看任务状态
launchctl list | grep weekly-report
```

### 管理定时任务

```bash
# 停止任务
launchctl unload ~/Library/LaunchAgents/com.user.weekly-report.plist

# 重新加载（修改配置后）
launchctl unload ~/Library/LaunchAgents/com.user.weekly-report.plist
launchctl load ~/Library/LaunchAgents/com.user.weekly-report.plist

# 查看日志
tail -f ~/ai/cline-skills/auto-weekly-report.log
```

## 📖 使用方法

### 手动生成周报

```bash
# 1. 生成周报数据
~/ai/cline-skills/auto-weekly-report.sh

# 2. 使用 AI 优化周报内容（在 AI 助手中）
# "请优化这份周报：~/ai/cline-skills/weekly-report-YYYYMMDD.md"

# 3. 发送到飞书
~/ai/cline-skills/send-to-feishu.sh ~/ai/cline-skills/weekly-report-YYYYMMDD.md
```

### 使用 AI Skill

在支持 Skills 的 AI 助手中：

```
帮我生成本周的周报
```

AI 会自动：

1. 调用 `auto-weekly-report.sh` 采集数据
2. 智能分析和优化周报内容
3. 调用 `send-to-feishu.sh` 发送到飞书

## 🔧 故障排查

### 问题 1：定时任务不执行

**检查任务状态**：

```bash
launchctl list | grep weekly-report
```

**查看日志**：

```bash
cat ~/ai/cline-skills/auto-weekly-report-stderr.log
```

**常见原因**：

- 路径配置错误
- 脚本没有执行权限
- 时间配置错误

### 问题 2：飞书发送失败

**测试脚本**：

```bash
~/ai/cline-skills/send-to-feishu.sh ~/ai/cline-skills/weekly-report-test.md
```

**常见原因**：

- APP_ID 或 APP_SECRET 错误
- RECEIVE_ID 错误
- 应用权限未配置
- 应用版本未发布

### 问题 3：Git 仓库扫描失败

**检查项目路径**：

```bash
# 确认路径是否正确
ls -la /Users/你的用户名/project/你的项目目录
```

**检查 Git 配置**：

```bash
git config user.name
git config user.email
```

## 📝 注意事项

1. **安全性**：
   - 不要将包含 APP_SECRET 的脚本提交到公共仓库
   - 建议使用环境变量存储敏感信息

2. **路径配置**：
   - 所有路径必须使用绝对路径
   - 确保路径中没有特殊字符或空格

3. **权限问题**：
   - 确保脚本有执行权限（chmod +x）
   - 确保定时任务能访问 Git 仓库

4. **日志管理**：
   - 定期清理日志文件，防止占用过多空间
   - 查看日志了解脚本执行情况
