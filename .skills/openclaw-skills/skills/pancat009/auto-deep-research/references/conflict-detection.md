# 冲突检测 Reference

## 什么是冲突？

当两个**独立来源**对同一事实给出**不同**结论时，形成冲突。

## 检测方法

### 1. 事实比对
对每个结论，检查是否有多个来源：

```
结论 X
- 来源 A 说：XXX
- 来源 B 说：YYY
→ 如果 A 和 B 说的不是同一件事，有冲突
```

### 2. 优先级
- **官方文档 > 博客 > 社交媒体**
- **最新 > 旧版**
- **多来源一致 > 单来源**

### 3. 标记格式

在 memo.md 中用 `???` 标记冲突：

```markdown
### 冲突点（待验证）
- 来源 A 说：XXX
- 来源 B 说：YYY
???
```

## 退出条件修正

**原逻辑**（有问题）：
- 弹药够 + 没矛盾 + 没新概念 → 退出

**修正后**：
- 所有子问题都有 ≥2 个来源确认 → 退出
- **即使有冲突**，也标注为"不确定点"，不阻塞退出
