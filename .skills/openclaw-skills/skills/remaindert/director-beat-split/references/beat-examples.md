## 甜宠短剧案例：《桃花运》- 2分钟

### 节奏公式

```
相遇 → 误解 → 互怼 → 解开 → 靠近 → 危机 → 告白 → 撒糖
```

### 关键要点

- 甜来自互动，不是硬撒糖
- 每次靠近都有合理动机
- 危机要体现保护欲

### 情绪曲线

```
情绪: 3→5→4→6→7→5→8→9→7→8
     ↑   ↑   ↑   ↑   ↑   ↑
标签: 相遇 误会 解释 危机 告白 撒糖
```

## 逆袭短剧案例：《王者归来》- 3分钟

### 节奏公式

```
低谷 → 被踩 → 羞辱 → 转机 → 修炼 → 打脸 → 逆袭
```

### 关键要点

- 被踩要够狠（3次以上）
- 转机要合理（有铺垫）
- 打脸要爽（对比要强）

## 悬疑短剧案例：《看不见的凶手》- 2分钟

### 节奏公式

```
谜题 → 线索 → 误导 → 真相 → 反转 → 实情
```

### 关键要点

- 线索要明显（导演要看得出）
- 误导要合理（不是硬骗）
- 反转要有依据（前面有伏笔）

## 伏笔类型详解

### 1. 物品伏笔

```
setup: 镜头特写 → 男人口袋的钻戒
payoff: 结尾求婚
```

### 2. 对话伏笔

```
setup: "这件衬衫，我留了很多��"
payoff: 女主穿的正是这件
```

### 3. 情绪伏笔

```
setup: 每次提到母亲，女主都沉默
payoff: 真相揭露-母亲去世
```

### 4. 视觉伏笔

```
setup: 背景里模糊的日历
payoff: 日历上的日期是案发时间
```

### 5. 音效伏笔

```
setup: 神秘电话铃声
payoff: 接听的是反派
```

## 诊断问题库

### 结构问题

| 症状 | 诊断 | 方案 |
|------|------|------|
| 第一幕太长 | 建置超时 | 从冲突开场或删背景 |
| 高潮太晚 | 30%后才高潮 | 提前冲突点 |
| 结尾太短 | resolution < 5% | 加收尾节奏 |

### 节奏问题

| 症状 | 诊断 | 方案 |
|------|------|------|
| 每个beat<5s | 节奏太碎 | 合并相邻 |
| 连续同类 | 类型单调 | 插入过渡 |
| 全程对话 | 无行动 | 加事件加行动 |

### 情感问题

| 症状 | 诊断 | 方案 |
|------|------|------|
| 全程一个情绪 | 无波动 | 加反转点 |
| 情绪太满 | 持续高压 | 加释放 |
| 无共鸣点 | 无情感钩子 | 加情感场景 |
