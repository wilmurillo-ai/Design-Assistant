# 🧬 Personal Genomics — 你的 AI 基因分析师

**[English Version](README_EN.md)**

> 把消费级基因检测数据变成真正有用的健康洞察。

你做过微基因、23andMe 或 AncestryDNA 的基因检测吗？拿到了原始数据却不知道怎么深入解读？这个 Claude Skill 就是为你准备的——它让 Claude 变成一位专业的基因组分析师，从你的原始数据出发，一步步为你输出个性化的健康风险报告、用药指导和补充剂方案。

## ✨ 它能做什么

整个分析流程分为四个阶段，像和一位专业医生对话一样逐步推进：

### Phase 1 · 数据接入
把你的基因数据文件丢给 Claude，它会自动识别来源和格式，告诉你数据质量如何、包含多少变异位点。

### Phase 2 · 全面初筛
扫描 **~120 个临床相关 SNP**，覆盖六大领域：

| 领域 | 覆盖内容 | 典型基因 |
|------|---------|----------|
| 🏥 健康风险 | 癌症、心血管、代谢、神经退行性疾病 | BRCA1/2, APOE, TCF7L2, 9p21.3 |
| 💊 药物基因组学 | 药物代谢类型、用药敏感性 | CYP2D6, CYP2C19, VKORC1, SLCO1B1 |
| 🥗 营养代谢 | 乳糖耐受、叶酸代谢、维生素D、Omega-3 | MTHFR, VDR, FADS1, ALDH2 |
| 🏃 运动基因 | 肌纤维类型、耐力倾向、恢复能力 | ACTN3, ACE, PPARGC1A |
| 🌍 祖源分析 | mtDNA/Y 染色体单倍群（需全基因组数据） | — |
| 🔬 深度风险面板 | 血脂、冠心病、痛风、糖尿病通路各 10-20 个位点 | LDLR, PCSK9, SLC2A9, ABCG2 |

每一项发现都附带已发表研究的 PMID 引用。

### Phase 3 · 深度分析
Claude 会询问你的：已知疾病、家族病史、当前用药、生活方式。然后针对**你最关心的疾病通路**，展开 50+ 个扩展位点的精细分析，结合家族史交叉解读风险。

### Phase 4 · 个性化建议
输出可落地的行动方案：
- **补充剂计划** — 按优先级分层（核心/推荐/可选），标注剂量、服用时间、药物交互
- **体检筛查表** — 根据风险谱定制检查项目和频率
- **年龄节点提醒** — 在关键年龄自动建议加做的筛查

可选输出格式：HTML 交互报告、Excel 服用计划表、一页纸医生沟通摘要。

## 📂 支持的数据格式

| 平台 | 文件类型 | 说明 |
|------|---------|------|
| 微基因 (WeGene) | `.txt` (TSV) | 直接使用导出的核心数据文件 |
| 23andMe | `.txt` (TSV) | 下载的原始数据文件 |
| AncestryDNA | `.txt` (TSV) | 下载的原始 DNA 数据 |
| 全基因组测序 | `.vcf` / `.vcf.gz` | 标准 VCF v4.x 格式 |
| 比对文件 | `.cram` / `.bam` | 用于关键变异的测序深度验证 |

支持同时导入多个文件——例如芯片数据 + VCF，Skill 会自动使用**双源查询策略**最大化覆盖率。

## 🚀 安装与使用

### 方法一：安装 .skill 文件（推荐）

1. 从 [Releases](../../releases) 页面下载 `personal-genomics.skill`
2. 在 Claude Desktop 中安装该 Skill
3. 将你的基因数据文件放入工作目录
4. 对 Claude 说："帮我分析一下我的基因数据"

### 方法二：从源码安装

```bash
git clone https://github.com/yanzhanglee/personal-genomics--.git
# 将克隆的目录配置为 Claude Skill 路径
```

### 使用示例

```
👤 我做了微基因的全基因组测序，数据文件在文件夹里了，帮我分析一下。

🤖 [自动检测格式 → 解析数据 → 生成初步报告]

👤 我有高血脂和痛风，父亲 54 岁心梗去世。能针对这些做深入分析吗？

🤖 [启动深度风险面板 → 血脂/冠心病/痛风通路 50+ 位点精细分析]

👤 根据分析结果，帮我制定一份补充剂方案和服用时间表。

🤖 [生成分层补充剂方案 + Excel 每日服用计划 + 定期复查时间表]
```
