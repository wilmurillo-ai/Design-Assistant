AccessMind Inspector

v1.0.0 - WCAG 2.2 Erişilebilirlik Denetimi

🚀 Chrome Extension Kurulumu

1

Chrome Extensions Sayfasını Aç

Chrome'da yeni bir sekme aç ve adres çubuğuna yaz:

chrome://extensions/

Enter'a bas.

2

Developer Mode Aç

Sağ üst köşede "Developer mode" switch'ini AÇIK konuma getir.

Toggle mavi olmalı.

3

Load Unpacked Butonuna Tıkla

Sol üstte "Load unpacked" butonuna tıkla.

Dosya seçici açılacak.

4

Extension Klasörünü Seç

Aşağıdaki klasörü seç:

/Users/sarper/.openclaw/workspace/skills/accessmind/extension/

"Select" veya "Aç" butonuna tıkla.

5

Kurulum Tamamlandı!

Extension listede "AccessMind Inspector" olarak görünmeli.

İkon olarak mor "A" harfi göreceksin.

⌨️ Klavye Kısayolları

  • Ctrl + Shift + A - Hızlı Erişilebilirlik Taraması
  • Ctrl + Shift + H - Vurgulamaları Temizle
  • Ctrl + Shift + V - Görsel AI Analizi (Ollama gerekli)

✨ Özellikler

🔍 Hızlı Tarama

Sayfadaki erişilebilirlik sorunlarını anında tespit eder.

⌨️ Klavye Testi

Tab navigasyonu ve focus trap tespiti.

🎨 Görsel Vurgulama

Sorunlu elementleri renk kodlarıyla vurgular.

📊 WCAG Raporu

WCAG 2.2 kriterlerine göre detaylı rapor.

🤖 AI Analizi

Qwen3-VL ile görsel erişilebilirlik analizi.

📋 DevTools Panel

Chrome DevTools'ta entegre panel.
