# RoboHire — AI Recruitment Agent

## What It Does

RoboHire is an **AI-powered end-to-end recruitment platform** that automates 80% of the hiring process:

- **AI Recruiting Consultant** — clarifies role requirements via conversation
- **AI JD Generator** — creates polished job descriptions
- **AI Resume Screening** — ranks candidates with reasoning (not keyword matching)
- **Automated Interview Outreach** — sends interview invites via link + QR code
- **AI Video Interviewer** — 24/7, supports 7 languages, real-time analysis
- **Multi-Agent Evaluation** — structured scorecards with cheating detection
