#### 使用示例

**检测场景**:
```python

def get_user_input(request):
    user_id = request.args.get('id')  # 污点源
    return process_user(user_id)

# file2.py - 数据处理
def process_user(user_id):
    return query_database(user_id)

# file3.py - 数据库操作
def query_database(user_id):
    cursor.execute(f"SELECT * FROM users WHERE id = {user_id}")  # 汇点
```

**v1.1.0**: 追踪完整调用链 file1 → file2 → file3，并检查 file1 是否有验证
