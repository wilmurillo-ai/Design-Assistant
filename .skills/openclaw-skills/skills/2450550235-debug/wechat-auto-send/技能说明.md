# 微信自动发送消息技能

## 技能概述
这是一个 OpenClaw 技能，用于通过 Python 脚本自动控制微信发送消息给指定联系人。

## 触发关键词
- 发微信
- 微信发送
- 给[某人]发消息
- 微信消息
- 自动发送微信
- wechat send

## 使用方式

### 自然语言指令
```
给张三发微信说"晚上7点开会"
给文件传输助手发送测试消息
告诉李四资料已准备好
微信发送消息给王五：项目已完成
```

### 技能执行流程
1. 识别用户指令中的联系人和消息内容
2. 检查微信是否已打开
3. 执行 Python 脚本发送消息
4. 返回执行结果

## 依赖要求
```txt
pyautogui>=0.9.53
pygetwindow>=0.0.9
pyperclip>=1.8.2
```

安装命令：
```bash
pip install pyautogui pygetwindow pyperclip
```

## 配置要求
1. 微信客户端必须已安装并登录
2. 执行脚本时微信窗口必须可见（不要最小化）
3. 需要有 Python 环境

## 安全注意事项
1. 该技能会控制鼠标和键盘操作
2. 执行过程中不要操作电脑
3. 只能发送文本消息，不支持文件发送
4. 联系人名称必须准确匹配微信中的备注或昵称

## 错误处理
- 微信未打开：提示用户先打开微信
- 依赖未安装：提示安装命令
- 发送失败：显示错误信息并建议解决方案
