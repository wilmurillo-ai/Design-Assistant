## 技能功能
- 自动发送微信消息给指定联系人
- 支持自定义消息内容
- 支持调整发送速度
- 提供详细执行日志

## 使用示例
用户指令：
- "给张三发微信说晚上7点开会"
- "给文件传输助手发送测试消息"
- "告诉李四资料已准备好"

## 执行步骤
1. 解析用户指令，提取联系人和消息内容
2. 检查微信是否已打开
3. 执行 Python 脚本发送消息
4. 返回执行结果

## 注意事项
- 微信必须已打开并可见
- 需要安装 Python 依赖
- 执行过程中不要操作鼠标键盘
