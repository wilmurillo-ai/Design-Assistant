# 微信自动发送消息技能

## 功能概述
通过 Python 脚本自动控制微信发送消息给指定联系人。

## 前提条件
1. 微信客户端已安装并登录
2. 微信窗口已打开并可见（不要最小化）
3. 已安装 Python 依赖：
   ```bash
   pip install pyautogui pygetwindow pyperclip
   ```

## 使用方法

### 基本用法
```bash
python wechat-send.py "联系人名称" "消息内容"
```

### 示例
```bash
# 发送给文件传输助手
python wechat-send.py "文件传输助手" "这是一条测试消息"

# 发送给特定联系人
python wechat-send.py "张三" "你好，在吗？"

# 自定义等待时间和倒计时
python wechat-send.py "李四" "会议资料已发" --wait 2 --countdown 3

# 显示详细执行信息
python wechat-send.py "王五" "晚上一起吃饭" --verbose
```

## 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `contact` | 联系人名称（微信备注或昵称） | 必填 |
| `message` | 要发送的消息内容 | 必填 |
| `--wait` | 操作等待时间（秒） | 1.5 |
| `--countdown` | 开始前倒计时秒数 | 5 |
| `--verbose` | 显示详细执行信息 | 否 |

## 执行流程
1. 检查依赖和微信窗口
2. 显示倒计时（默认5秒）
3. 聚焦微信窗口
4. 复制联系人 → Ctrl+F → 粘贴联系人 → Enter
5. 复制消息 → 粘贴消息 → Enter 发送
6. 显示发送结果

## 注意事项
1. **执行前**：确保微信已打开并处于前台
2. **执行中**：不要操作鼠标键盘
3. **安全中断**：鼠标快速移动到屏幕左上角可中断脚本
4. **联系人名称**：必须与微信中的备注或昵称完全一致
5. **网络延迟**：如果网络慢，可以调大 `--wait` 参数

## 常见问题

### Q: 脚本没有反应？
A: 检查：
1. 微信是否已打开
2. 微信窗口是否可见（不要最小化）
3. 依赖是否安装：`pip install pyautogui pygetwindow pyperclip`

### Q: 发送到错误的联系人？
A: 确保联系人名称完全正确，包括空格和标点。

### Q: 消息发送失败？
A: 调大等待时间：`--wait 2.5`

### Q: 如何中断脚本？
A: 快速将鼠标移动到屏幕左上角，或按 Ctrl+C。
