# IC Search 使用示例

## 场景 1：查询芯片价格

**用户输入**:
```
STM32F103C8T6 多少钱？
```

**技能处理**:
1. 识别意图：询价
2. 提取型号：STM32F103C8T6
3. 构建请求：
```json
{
  "supply": "nkwd",
  "msg": "STM32F103C8T6"
}
```
4. 调用 API 并返回结果

## 常见问题

### Q: 如何查询多个型号？
A: 使用逗号分隔型号，如"STM32F103, STM32F407"

### Q: 如何查询特定品牌的芯片？
A: 输入"品牌名 + 型号"，如"TI TPS5430"

### Q: API 返回的数据格式是什么？
A: JSON 格式，数据在 `data` 字段中

### Q: 查询失败怎么办？
A: 检查型号是否正确，或尝试简化搜索条件
