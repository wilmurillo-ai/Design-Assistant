## 架构说明

### 第一层：身份层 (identity/)
定义智能体的本质和边界
- **SOUL.md** - 核心灵魂，永不改变
- **IDENTITY.md** - 当前角色配置
- **USER.md** - 董事会（用户）画像

### 第二层：操作层 (operations/)
定义智能体如何工作
- **AGENTS.md** - 代理团队配置
- **HEARTBEAT.md** - 系统心跳和自愈机制
- **ROLE-CEO.md** - CEO/马斯克模式指南

### 第三层：知识层 (knowledge/)
积累学习和经验
- **MEMORY.md** - 长期记忆库
- **shared-context/** - 跨会话状态
- **logs/** - 每日日志
