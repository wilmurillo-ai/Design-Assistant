# AGENTS.md - 代理团队配置
# 可调度的工作单元

## 核心代理

### 1. simplify（代码优化代理）
- **职责**: 代码审查与重构
- **触发条件**: 代码变更检测
- **工作流**:
  1. 扫描代码质量问题
  2. 识别可重用性缺陷
  3. 自动修复或报告
- **输出**: 优化后的代码 + 改进报告
- **状态**: ✅ 在线

### 2. claude-api（AI应用构建代理）
- **职责**: 使用Claude API构建应用
- **触发条件**: `import anthropic` 或相关SDK导入
- **工作流**:
  1. 识别API调用模式
  2. 生成完整应用代码
  3. 提供部署方案
- **输出**: 可运行的AI应用
- **状态**: ✅ 在线

### 3. agent-os（架构代理）**[新建]**
- **职责**: 维护三层架构完整性
- **触发条件**: 架构相关请求
- **工作流**:
  1. 验证三层文件完整性
  2. 执行架构升级
  3. 协调跨层操作
- **输出**: 架构状态报告
- **状态**: 🆕 刚部署

## 调度规则
```
用户指令 → 意图识别 → 代理匹配 → 任务分配 → 执行监控 → 结果汇报
```

## 代理冲突解决
1. 单任务：最高匹配度优先
2. 多任务：并行执行
3. 冲突时：向董事会请求决策

## 自愈机制
- 每分钟检测代理健康状态
- 失败任务自动重试（最多3次）
- 连续失败触发告警
