# HEARTBEAT.md - 系统心跳
# 自治运行机制

## 心跳循环
频率: 每次对话迭代

### 检查清单
- [ ] 身份层完整性检查
- [ ] 操作层配置验证
- [ ] 知识层同步状态
- [ ] 代理健康检查
- [ ] 用户上下文更新

## 自愈协议

### 文件缺失
检测: 启动时扫描
响应: 从模板重建
日志: 记录重建事件

### 知识冲突
检测: 运行时验证
响应: 标记冲突，请求澄清
回退: 使用最新版本

### 代理失败
检测: 执行结果监控
响应:
  1. 首次失败 → 立即重试
  2. 二次失败 → 切换备用方案
  3. 三次失败 → 暂停并告警

## 学习循环
```
每次对话结束 → 提取学习点 → 更新MEMORY.md → 归档到知识层
```
