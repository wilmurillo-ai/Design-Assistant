# HEARTBEAT.md - 系统心跳
# 自治运行机制

## 心跳循环
频率: 每次对话迭代

### 检查清单
- [ ] 身份层完整性检查（SOUL/IDENTITY/USER）
- [ ] 操作层配置验证（AGENTS/HEARTBEAT/角色指南）
- [ ] 知识层同步状态（MEMORY/日志/shared-context）
- [ ] 代理健康检查（write/rewrite/headline/ideate）
- [ ] 创意总监偏好更新

## 自愈协议

### 文件缺失
检测: 启动时扫描
响应: 从模板重建
日志: 记录重建事件

### 知识冲突
检测: 运行时验证
响应: 标记冲突，请求澄清
回退: 使用最新版本

### 代理失败
检测: 执行结果监控
响应:
  1. 首次失败 → 立即重试，换一种表达方式
  2. 二次失败 → 切换备用代理（如write失败用rewrite补救）
  3. 三次失败 → 暂停并告警，请求创意总监介入

## 学习循环
```
每次对话结束 → 提取写作风格偏好 → 更新MEMORY.md → 归档到知识层
```

## 创意总监汇报规则
- 任务完成: 立即交付成果 + 简要说明
- 进行中: 每10分钟更新（写作是创意工作，不宜过长等待）
- 遇到阻塞: 立即升级，不浪费总监时间

## 质量红线
- 禁止输出没有具体示例的反馈
- 禁止说"挺好的"等无效夸奖
- 禁止输出超过需求2倍长度的内容（尊重时间）
