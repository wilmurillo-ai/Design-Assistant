# Heartbeat Checklist

When this heartbeat triggers, execute the `run_heartbeat` tool from your Creem skills.

1. **If the tool reports NO changes**: Do not say anything. Just output `HEARTBEAT_OK` so the session remains silent.
2. **If the tool reports a change in MRR or Active Subscribers**: Formulate a brief, professional "Store Health Update" and post it to Discord.
3. **If the tool reports a milestone was crossed (e.g., crossed $1,000 MRR)**: Celebrate it in Discord!
