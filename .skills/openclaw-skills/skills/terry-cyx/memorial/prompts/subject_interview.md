# 本人访谈提示词（subject_interview.md）

## 适用场景

当档案主人**仍在世**时，由本人亲口参与建档。
这是质量最高的数据来源——只有他们自己才知道完整的故事。

## 访谈原则

1. **不催促，不追问细节**：如果对方不愿讲，记录"不愿讲"本身也是一种信息
2. **用他们自己的话**：逐字记录，不替他们总结或润色
3. **允许跑题**：跑题往往是最有价值的部分，跟着走
4. **分多次进行**：不要一次问完，几十年的人生不适合一次谈透
5. **让家人也在场**：有时候他们会说出从未对家人说过的话——这正是目的

---

## 访谈引导词

```
我想帮你留下一份关于你自己的档案。

不是采访，也不是什么正式的事——就是把你经历过的、想过的、在乎过的事情
记下来，留给我们，也留给以后的人。

你愿意跟我说说吗？从哪里开始都行。
```

---

## 访谈模块

### 模块 A：从哪里来

**目标：** 早年经历、成长背景、塑造性格的起点

```
你是哪里人？小时候在哪里长大的？

那时候的生活是什么样的？

你记得的，关于你小时候最深的一件事是什么？

你的父母是什么样的人？他们对你影响最大的是什么？

你读书了吗？读到什么程度？那段时间对你意味着什么？
```

**追问方向：**
- 家庭经济状况（引导方式：吃穿怎么样、住在哪里）
- 兄弟姐妹关系
- 最早意识到"我是这样一个人"的时刻

---

### 模块 B：走过了什么

**目标：** 人生转折点、重大选择、经历过的时代

```
你这辈子，有没有哪个时间节点，让你觉得"从那之后不一样了"？

你做过的最重要的一个决定是什么？

有没有一段特别难熬的时期？那时候你怎么撑过来的？

[如适用] 你经历过哪些大事？（文革/下乡/下岗/改革开放……）
那段时间你在哪里，在做什么，心里是什么感受？
```

**追问方向：**
- 不要预设"难熬"或"好时光"，跟着他们的表述走
- 如果他们提到某段经历后沉默了，可以问"那时候你一般怎么排解？"而不是"你当时心情怎么样"

---

### 模块 C：在乎什么

**目标：** 价值观、信念、人生哲学

```
你这辈子，最在乎的是什么？

有没有什么事，是你觉得无论如何都不能做的？

你觉得一个人怎么样才算活得好？

你最骄傲的事情是什么？

有没有什么事，你到现在还觉得是对的，但当时很多人不理解？
```

**追问方向：**
- 这里最容易得到口头禅和人生格言
- 注意记录他们反复提到的词——那就是核心价值观

---

### 模块 D：关于家人

**目标：** 关系视角、情感表达方式、对家人的期待

```
你对 [配偶/子女/孙辈] 最想说的一句话是什么？

你觉得自己做得最好的地方是什么？最遗憾的地方呢？

你希望他们记住你什么？

你觉得你给他们留下了什么？
```

**追问方向：**
- 这部分很可能触动情绪，不要打断，给时间
- 如果他们说"没什么好说的"，可以换一个角度："如果你能给他们写一封信，你想写什么？"

---

### 模块 E：关于这一生

**目标：** 整体人生观、未来展望、想留下什么

```
你这辈子，有没有什么事是你很想做但一直没做的？

有没有什么话，是你一直想说但没说出口的？

你最希望被记住的是什么？

如果让你对年轻时候的自己说一句话，你会说什么？
```

**特别问法（如果对方对"被记住"不感兴趣）：**
```
有没有什么事，你觉得值得讲给我们的孩子听？
```

## 记录规范

### 引用要逐字记录

```markdown
**原话**："吃亏是福，这是我爸教我的，我信了一辈子。"
**时间/场合**：2024年冬天，吃饭时谈起邻居的事
**记录者**：孙子
```

### 标注情绪状态（非评价，仅观察）

```markdown
提到三年困难时期时停顿了很长时间，之后说"那时候不想那些事"，就转移话题了。
[未展开，本人不愿细讲]
```

### 跑题内容单独记录

```markdown
**跑题片段**：讲到邻居老李的时候，突然说起文革时他们互相藏粮食的事。
没追问，先记下来，以后再看是否补充进时间轴。
```

---

## 访谈结束后的整理

每次访谈后，将录音/笔记整理为结构化文本，上传到档案中：

```
这次谈到了哪些模块？
有没有新的故事可以加进追忆档案？
有没有新的口头禅或说话方式可以更新人格档案？
有没有需要下次追问的地方？
```

档案的每条内容来源注明 **[本人口述，YYYY-MM-DD]**，与家人口述严格区分。
