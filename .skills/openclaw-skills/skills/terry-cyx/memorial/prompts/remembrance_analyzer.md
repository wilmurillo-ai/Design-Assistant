# 追忆分析提示词（remembrance_analyzer.md）

## 目标
从用户提供的各类材料中，提取并结构化逝者的人生记忆，填充追忆文档的8个维度。

## 分析原则
1. **只记录有据可查的信息**，不补全、不推断、不美化
2. 无法确定的内容用 `[待补充]` 标记，不编造
3. 时间节点尽量精确，不确定年份用"约"标注
4. 保留原始语言风格（方言词汇、特殊表达方式）
5. 优先记录具体的故事，而非抽象的评价

---

## 分析维度

### 维度 1：生命时间轴

**提取目标：**
- 出生时间与地点
- 求学、工作、迁居的关键时间节点
- 婚姻与家庭的重要事件（结婚、子女出生）
- 重大转折（退休、患病、离开某地）
- 离世时间与方式（如用户提及）

**分析指令：**
```
从材料中提取所有含有明确或隐含时间信息的事件。
按时间顺序排列，格式为：[年份/时间] — [事件]。
不确定的年份用"约XXXX年"表示。
不要填充材料中未出现的内容。
```

**输出示例：**
```markdown
- 1938年 — 出生于湖南省某县
- 约1956年 — 参加工作，进入铁路系统
- 1965年 — 与祖母结婚，定居北京
- 1968年 — 长子出生（文革期间，生活艰难）
- 约1980年 — 改革开放后晋升工段长
- 1998年 — 退休，开始种菜生活
- 2023年3月 — 因病离世，享年85岁
```

---

### 维度 2：人生角色

**提取目标：**
- 社会角色（职业、职务、党员身份等）
- 家庭角色（父亲/丈夫/祖父，以及在家中的实际位置）
- 社区角色（邻里关系、是否有威望）
- 特殊角色（战友、老师、某人的贵人）

**分析指令：**
```
识别材料中提及的所有角色标签，以及这些角色如何体现在行为上。
不只列标签，要附上具体体现。
```

**输出示例：**
```markdown
- **父亲**：严格要求子女学习，不轻易表扬，但私下以子女成就为傲
- **铁路工人**：工作认真，从不迟到，曾多次获得劳动模范称号
- **邻居心中的"老王"**：遇到困难帮忙，公正，小区里的非正式仲裁者
```

---

### 维度 3：价值观与信念

**提取目标：**
- 明确表达过的人生观（口头禅、格言、教导）
- 通过行动体现的价值优先级
- 政治/宗教信仰（客观记录，不评价）
- 对教育、钱财、面子、家庭的态度

**分析指令：**
```
优先收集他们亲口说过的话（引号标注）。
对于行动体现的价值观，描述具体行为，再归纳价值倾向。
不要代入现代价值判断。
```

**输出示例：**
```markdown
- **"吃亏是福"**：常挂嘴边，每次家庭矛盾时都会说这句话
- 重视教育高于一切：砸锅卖铁也要供孩子读书，自己穿补丁衣服
- 家族荣誉感强：非常在意"老王家的人"在外的形象
- 党员信仰：入党50年，从不动摇，去世时胸前别着党徽
```

---

### 维度 4：日常习惯与仪式

**提取目标：**
- 日常作息规律
- 饮食偏好（菜系、特殊习惯）
- 固定活动（晨练、看哪个节目、打牌）
- 待客之道
- 特殊仪式（逢年过节怎么过）

**分析指令：**
```
提取所有关于日常生活细节的描述。
越具体越好：不是"喜欢看电视"，而是"每晚7点准时看新闻联播，雷打不动"。
```

**输出示例：**
```markdown
- 每天5:30起床，买菜回来再吃早饭
- 不吃辣，湖南人却不吃辣——家里人觉得是个谜
- 每晚7点看新闻联播，7:30看地方新闻，雷打不动
- 退休后的每个下午都在菜园，哪怕下小雨也去
- 过年一定要包饺子，他来擀皮，从不让别人插手
```

---

### 维度 5：标志性故事

**提取目标：**
- 3-5个最能代表这个人的具体故事
- 家人常常提起的故事
- 体现其性格转折的故事
- 令人印象深刻的细节

**分析指令：**
```
提取有起承转合的完整故事，而不是碎片印象。
尽量保留原始讲述的语气和细节。
标注故事来源（聊天记录/家人口述/本人描述）。
```

**输出示例：**
```markdown
**故事1：文革期间藏书（家人口述）**
文革时，他把几十本书用油纸包好，埋在菜园地下。红卫兵来抄家时什么都没搜到。
改革开放后他把书挖出来，说："书不能烧，知识不能断。"

**故事2：为孙子的大学哭了（直接观察）**
孙子考上大学那天，他一个人在菜园待了一下午。
回来时眼睛红的，问他怎么了，他说"没事，风迷眼了"。
```

---

### 维度 6：关系网络

**提取目标：**
- 对配偶的态度与方式
- 对子女的教育方式与情感表达
- 对孙辈的不同态度（隔代宠爱？）
- 对老友、同事的情义
- 家庭中的权力结构

**分析指令：**
```
具体描述与每类关系人的互动模式，不要只用形容词。
"对孩子严格"不够，要说"如何严格，对什么事严格，哪些事反而宽松"。
```

---

### 维度 7：遗憾与遗言

**提取目标：**
- 明确表达过的遗憾（没完成的事、没去过的地方）
- 临终前说的话或做的事
- 未说出口的话（家人的感受/推测）
- 想对某人说却没说的话

**分析指令：**
```
这部分需要特别小心——只记录有实际依据的内容。
家人的感受/推测要明确标注为"[家人推测]"，不要当作事实呈现。
```

---

### 维度 8：遗产与影响

**提取目标：**
- 物质遗留（手工品、手稿、种的植物）
- 精神遗留（对家人的影响、塑造）
- 传承的手艺、习惯、价值观
- 家人觉得"他还在"的感受来源

**分析指令：**
```
关注那些"因为有他，我/我们变成了这样"的叙述。
具体到哪个习惯、哪句话、哪种方式被传承了下来。
```

---

## 分析输出格式

完成8个维度分析后，输出摘要供用户确认：

```
我从你分享的材料里，整理出了关于 [名字] 的这些内容：

生命时间轴：[X个关键节点]
标志性故事：[X个]
性格印象：[3-5个关键词]
日常习惯：[X个细节]
...

有没有我理解错的地方，或者你想补充的？
确认后我来生成完整的追忆文档。
```
