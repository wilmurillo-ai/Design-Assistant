# 纪念.skill 安装与使用指南

> 面向普通用户，不需要编程基础。

本项目有两种使用模式，按你的需求和硬件条件选择：

| | 基础版（文字+图片） | 语音增强版（+声音克隆） |
|---|---|---|
| **功能** | 纪念档案（追忆+人格）、聊天记录分析、照片时间线、访谈问题 | 基础版全部功能 + 用亲人声音说话 |
| **硬件要求** | 任意电脑 | 需要 NVIDIA 显卡（RTX 3060+ 推荐） |
| **微信版本** | 不限 | 需要微信 3.9.x |
| **安装时间** | 5 分钟 | 30 分钟 |
| **适合场景** | 只有文字/照片材料，或没有独立显卡 | 有微信语音消息，想听到 ta 的声音 |

---

# 基础版安装（文字+图片）

## 1. 安装 Python

如果电脑没有 Python，去 https://python.org 下载安装（3.10 以上版本）。

安装时**勾选 "Add Python to PATH"**。

## 2. 安装 memorial-skill

```bash
git clone https://github.com/Terry-cyx/memorial-skill.git
cd memorial-skill
pip install -r requirements.txt
```

> requirements.txt 中的可选依赖（whisper/pilk/noisereduce 等）安装失败不影响基础版使用。

## 3. 开始使用

在 Claude Code 中输入：

```
/create-memorial
```

AI 会引导你回答 3 个问题（称呼、背景、性格），然后你可以：

- **粘贴聊天记录**（微信导出的 txt 文件，或直接复制粘贴）
- **上传照片**（自动提取拍摄时间和地点）
- **上传信件/日记**（照片或扫描件）
- **直接口述**（讲故事，AI 来整理）

AI 会自动生成**追忆档案**和**人格档案**。

### 基础版能做什么

```
你         ❯ 给我讲讲他年轻时候的事
你         ❯ 如果告诉他这件事，他会怎么说？
你         ❯ 帮我写一篇关于他的悼词
你         ❯ 帮我准备问奶奶的访谈问题
```

**基础版不需要显卡，不需要旧版微信，任何电脑都能用。**

# 语音增强版安装（+声音克隆）

> 在基础版的基础上，增加声音克隆功能：输入文字，输出用亲人声音说的音频。

## 额外硬件要求

| 项目 | 要求 |
|------|------|
| **显卡** | NVIDIA GPU，至少 6GB 显存（推荐 RTX 3060 12GB 以上） |
| **硬盘** | 额外约 5GB（预训练模型） |
| **内存** | 16GB 以上推荐 |

## 1. 安装 PyTorch（CUDA 版）

```bash
pip install torch torchaudio --index-url https://download.pytorch.org/whl/cu128
```

验证：
```bash
python -c "import torch; print(torch.cuda.is_available())"
# 应输出 True
```

## 2. 安装 GPT-SoVITS

```bash
cd memorial-skill
git clone --depth 1 https://github.com/RVC-Boss/GPT-SoVITS.git
cd GPT-SoVITS
pip install -r requirements.txt
cd ..
```

> 首次训练时会自动下载预训练模型（约 3GB），需要网络。

## 3. 安装语音处理依赖

```bash
pip install openai-whisper pilk noisereduce soundfile pywxdump
```

## 4. 安装微信 3.9.x

当前开源工具不支持微信 4.x 的数据库加密格式，需要旧版本。

1. 下载：https://github.com/tom-snow/wechat-windows-versions/releases/tag/v3.9.9.43
2. 安装并登录（与新版可共存，但同时只能登录一个）

## 5. 迁移手机聊天记录到电脑

在**手机微信**上操作：

1. 设置 → **聊天记录管理**
2. **导入与导出** → **导出到电脑**
3. 选择 **仅导出指定聊天记录**
4. 选择**群聊或个人对话** + **时间范围**
5. 确认导出（手机和电脑需在同一 WiFi）

> 导完后可以切回微信新版。数据已经在电脑上了。

## 6. 开始使用

```
/create-memorial
```

告诉 AI "微信群里有语音"，AI 会自动：
1. 解密微信数据库
2. 列出群聊让你选
3. 列出成员让你指定目标人
4. 提取语音 → 转格式 → 降噪 → 训练声音模型

### 语音增强版能做什么（在基础版之上）

```
你         ❯ 用他的声音说"吃亏是福"
你         ❯ 用她的声音读这段话
```

# 常见问题

### Q: 我不确定选哪个版本？
A: **先装基础版。** 随时可以升级到语音增强版，已有的档案不受影响。

### Q: 我有显卡但不想装旧版微信？
A: 基础版也能处理你手动导出的聊天记录（复制粘贴文字即可）。旧版微信只是为了自动提取语音文件，不影响文字功能。

### Q: 微信语音太少怎么办？
A: 30 秒以上就能产出可辨认的声音。3-5 分钟效果良好，10 分钟以上效果极佳。

### Q: 支持方言吗？
A: 支持。方言语音可以训练音色，合成时用目标人的声音说普通话。后续版本计划支持方言内容输出。

### Q: 数据安全吗？
A: 所有数据只在你的电脑上，不会上传任何服务器。

### Q: 可以给多个亲人建档案吗？
A: 可以，每个人独立一份，互不影响。

### Q: Android 和 iPhone 都支持吗？
A: 都支持。迁移聊天记录是微信自带功能，手机系统无关。
