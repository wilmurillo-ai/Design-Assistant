# materials/ 目录说明

此目录存放用于构建纪念档案的原始材料。

```
materials/
├── audio/
│   ├── raw/           # 原始语音文件（silk/amr，从微信提取）
│   └── processed/     # 预处理后的 WAV 文件（降噪、重采样）
│
├── chats/             # 聊天记录导出文件（可选）
├── photos/            # 照片文件（可选，用于时间线提取）
└── documents/         # 其他文档（信件、日记等，可选）
```

所有材料仅存储在本地，不会上传到任何服务器。
