# 新股申购助手 - 发布指南

## 📤 发布到 ClawHub

### 第一步：清理 Git 文件

```bash
# 删除 .git 文件夹（如果有）
Remove-Item -Path ".git" -Recurse -Force
```

### 第二步：打包 ZIP

```bash
# 在 skills 目录执行
Compress-Archive -Path "ipo-assistant\*" -DestinationPath "ipo-assistant.zip" -Force
```

### 第三步：上传到 ClawHub

1. 访问 https://clawhub.com
2. 登录 GitHub 账号
3. 点击 "Create Skill" 或 "Upload"
4. 上传 `ipo-assistant` 文件夹

### 第四步：填写信息

**基本信息：**
```
Slug: ipo-assistant
名称：新股申购助手
版本：1.0.0
描述：A 股新股申购必备工具 - 申购日历、基本面分析、中签率预测、上市溢价预测
```

**定价：**
```
价格：0.001 USDT/次
类型：按次计费
```

**标签：**
```
新股，申购，A 股，IPO，中签率，溢价预测，理财，投资
```

### 第五步：发布

点击 "Publish" 完成！
