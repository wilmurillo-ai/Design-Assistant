# 更新日志

## v1.1.0 (2026-03-08) - 功能完善版

### ✨ 新增功能

#### 1. 多数据源支持
- ✅ **东方财富**：主要数据源，实时新股数据
- ✅ **akshare**：补充数据源（可选安装）
- ✅ **Tushare**：专业数据源（需 token）
- 📊 **数据聚合**：自动去重、合并多个数据源
- 💾 **智能缓存**：12 小时自动更新

#### 2. 历史统计
- 📈 **近 90 天统计**：新股数量、平均涨幅、中签率
- 🏆 **行业排行**：按行业统计新股表现
- 📊 ** extremes**：最高/最低涨幅新股
- 📉 **中签率分析**：行业对比、趋势分析

#### 3. 新股对比
- 🔄 **多股对比**：支持 2 只及以上新股对比
- 📊 **对比维度**：发行价、涨幅、中签率、行业、板块
- 💡 **投资建议**：基于对比结果

#### 4. 智能提醒
- ⏰ **每日提醒**：周一至周五 9:00 自动推送
- 📅 **周历提醒**：周一推送本周新股
- 🔔 **多渠道**：支持飞书、微信等
- ⚙️ **可配置**：提醒时间、渠道可自定义

### 🔧 功能优化

#### 1. 数据获取
- 增加数据源数量至 3 个
- 优化 API 请求失败处理
- 添加数据缓存机制

#### 2. 用户体验
- 优化输出格式
- 添加友好提示
- 支持更多命令
- 改进错误处理

### 📝 新增命令

| 命令 | 功能 | 示例 |
|------|------|------|
| `stats` | 查看历史统计 | `py main.py stats` |
| `compare <代码 1> <代码 2>` | 对比新股 | `py main.py compare 301234 688123` |
| `reminder` | 今日申购提醒 | `py main.py reminder` |
