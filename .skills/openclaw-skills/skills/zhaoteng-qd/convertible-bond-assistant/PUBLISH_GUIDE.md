# 技能发布指南

## 📤 发布到 ClawHub

### 第一步：登录 ClawHub

```bash
clawhub login
```

这会打开浏览器让你登录 clawhub.com 账号。

### 第二步：验证登录

```bash
clawhub whoami
```

应该显示你的用户名。

### 第三步：发布技能

在技能目录下执行：

```bash
cd C:\Users\tzhao\.openclaw\workspace\skills\convertible-bond-assistant
clawhub publish .
```

或者从 workspace 根目录：

```bash
clawhub publish skills/convertible-bond-assistant
```

### 第四步：设置免费

登录 clawhub.com 网站：
1. 进入你的技能页面
2. 点击 "Edit"
3. 设置价格为 **免费** (0 USDT)
4. 添加标签：`可转债` `打新` `免费` `A 股` `理财`
5. 保存
