## 话题标签使用规则

1. **数量：** 3-10 个为宜，不要超过10个
2. **位置：** 放在正文末尾或评论区第一条
3. **搭配：** 1-2个大热门 + 3-5个精准标签 + 1-2个长尾词
4. **格式：** 以 # 开头，不加空格
5. **时效：** 蹭热点时使用当前热门话题标签
