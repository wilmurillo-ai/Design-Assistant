## 报告示例

### 快速概览
```markdown
宁德时代 - 快速概览

## 核心逻辑
全球动力电池龙头，储能业务高增驱动业绩创新高。技术壁垒深厚，估值合理，具备长期配置价值。

## 主要矛盾
**多头**：储能占比提升至 20%，Q3 净利率 19.1% 创新高
**空头**：美国 IRA 法案限制、行业产能过剩、地缘政治风险

## 关键数据
| 指标 | 数值 | 评价 |
|------|------|------|
| 营收增速 | +12.9% | 稳健 |
| ROE | 18.5% | 优秀 |
| PE-TTM | 19.5x | 合理 |

## 投资建议
- 投资倾向: 偏多
- 适合人群: 长期投资者
- 核心风险: 地缘政治、价格战
```
