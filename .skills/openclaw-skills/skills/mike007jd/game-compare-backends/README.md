# game-compare-backends

Part of the **Game Superpowers** skill system.

## Summary

Run a narrow compare between two or three backend profiles when architecture uncertainty is real. Use carefully, because it usually costs more tokens and time.

## Recommended usage

- Use `game-compare-backends` for this game project and keep the output evidence-based.
- Apply `game-compare-backends` to improve or assess an existing browser game in a focused way.
- Use `game-compare-backends` as part of a larger Game Superpowers workflow when the request clearly matches game compare backends.
