- Use `game-backend-selector` for this game project and keep the output evidence-based.
- Apply `game-backend-selector` to improve or assess an existing browser game in a focused way.
- Use `game-backend-selector` as part of a larger Game Superpowers workflow when the request clearly matches game backend selector.
