# game-audio-feedback-audit

Part of the **Game Superpowers** skill system.

## Summary

Audit the project's audio layer as UX feedback: UI sounds, success/failure signals, danger cues, layering, and semantic sound priorities.

## Recommended usage

- Use `game-audio-feedback-audit` for this game project and keep the output evidence-based.
- Apply `game-audio-feedback-audit` to improve or assess an existing browser game in a focused way.
- Use `game-audio-feedback-audit` as part of a larger Game Superpowers workflow when the request clearly matches game audio feedback audit.
