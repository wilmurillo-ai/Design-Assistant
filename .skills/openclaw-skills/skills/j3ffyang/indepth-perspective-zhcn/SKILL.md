# 内容设计与表达策略（SKILL.md）

## 核心结构

- **核心观点**：每篇文章只需要一个锐利、明确且可被记住的核心立场。这是整篇内容的“主心骨”。
- **副观点**：从 2–3 个互补角度支撑核心观点，使其更立体。例如：
  - 逻辑性论证（数据或事实）
  - 情感性共鸣（故事或体验）
  - 审美或社会意义（价值或趋势）
- **说服策略**：可混合使用以下路径：
  - 数据驱动（Data-driven）
  - 故事驱动（Storytelling）
  - 权威背书（Expert / Authority Quote）
  - 类比隐喻（Analogy / Metaphor）
  - 社会认同（Social Proof）

## 情绪与表达

- **情绪出发点**：根据目标读者触发的主要感受选择：
  - 共鸣（认同与理解）
  - 紧迫（时间压力与FOMO）
  - 好奇（未知与反差）
  - 优越（认知或地位优势）
  - 恐惧（风险与损失厌恶）
  - 归属（社群与身份认同）
- **金句设计**：提炼一句能脱离上下文独立传播的记忆锚点。应具备：
  - 节奏感（短句或对称结构）
  - 意象性（易被视觉化）
  - 普适共鸣（可被多人转发）

## 节奏与层次

- **情感曲线**：内容节奏应像电影一样起伏：平静 → 矛盾 → 解决 → 余韵。
- **情感层次**：
  1. 表层信息：现象、事实、观察  
  2. 中层价值：态度与判断  
  3. 深层身份：读者自我认同被唤起

## 论证与表现

- **论证多样性**：在叙述中交替使用故事、数据、类比、反问，增强阅读节奏。
- **视角转化**：
  - 第一人称：增强代入与真实感  
  - 第二人称：制造共鸣与互动  
  - 第三人称：提升客观与权威感

## 语言与互动

- **语言风格**：
  - 控制句长（短句推动节奏，长句承载思想）
  - 调整口语化程度以匹配平台语境
  - 保持修辞密度与节奏的变化，制造韵律
- **互动钩子**：
  - 结尾提问：如“你是否也有类似感受？”
  - 空白填句：引导参与（例：“真正改变命运的，其实是____。”）
  - 争议立场：激发讨论
  - 投票行为：让读者有行动反馈

> **核心金句模板：**  
> 让思想有锋芒，让表达有温度，让节奏有生命。
