# 风格设计生成技能

## 技能用途

本技能是一套风格设计方案生成引擎。它基于两份参考文件中的100个风格方案及其内在模板系统，提供以下五类操作：

1. **换主体** — 将方案中的"字体"槽位替换为任意其他主体（物体、人体、建筑、Logo等），生成该主体的同风格方案
2. **要素混搭** — 从不同方案中提取各槽位值（主题、形态、色彩、背景、细节、参数），自由组合成全新方案
3. **模板衍生** — 参照"字体→信息图"的变形模式，将风格模板改造为新的产物类型（漫画分镜、游戏UI、小说封面、产品包装等）
4. **新风格延展** — 在20个分组内延续美学逻辑，或跨组融合，扩展出第101号及以后的更多方案
5. **领域应用** — 将风格方案的具体槽位值映射到漫画、小说、游戏、影视、产品设计等领域的创作要素中

## 参考文件

两份参考文件存放在 `references/` 目录下，**原封不动保留**。工作时按需读取：

| 文件 | 内容 | 角色 |
|------|------|------|
| `references/字体设计提示词方案.md` | 6槽位模板 × 100个实例（20组×5） | **基因源** |
| `references/信息图设计提示词方案.md` | 变形模板 × 100个实例 | **衍生示范** |

## 文件内在结构

### 字体方案的结构（基因源）

每个方案遵循统一的6槽位模板：

```
模板：主题（主题定位） 字体（字体形态） 色彩（色彩策略） 背景（背景设计） 细节（细节增强） 参数（技术参数）
```

示例：
```
示例：主题（自然生态） 字体（藤蔓缠绕形态） 色彩（生态绿渐变） 背景（牛皮纸背景） 细节（叶脉纹理） 参数（3:4比例）
```

100个方案分布在 **20个分组** 中，每组5个：

| # | 分组 | 编号范围 |
|---|------|----------|
| 1 | 自然与有机 | 01-05 |
| 2 | 文化与艺术 | 06-10 |
| 3 | 情绪与氛围 | 11-15 |
| 4 | 科技与未来 | 16-20 |
| 5 | 美食与生活 | 21-25 |
| 6 | 节日与庆典 | 26-30 |
| 7 | 奇幻与虚构 | 31-35 |
| 8 | 城市与工业 | 36-40 |
| 9 | 复古与经典 | 41-45 |
| 10 | 抽象与概念 | 46-50 |
| 11-20 | （继续...） | 51-100 |

### 信息图方案的结构（衍生示范）

信息图方案是字体方案的**产物变换示范**——同一编号的方案保持相同的风格DNA，但输出壳变了：

```
字体方案模板：  主题（XX） 字体（XX形态） 色彩（XX） 背景（XX） 细节（XX） 参数（X:X）
                    ↓ 变形
信息图方案模板：  信息图风格（{风格描述}，艺术化排版...） 标题（文本名称） 文本（文本内容） 参数（9:16）
```

对比示例：
```
字体方案：示例：主题（自然生态） 字体（藤蔓缠绕形态） 色彩（生态绿渐变） 背景（牛皮纸背景） 细节（叶脉纹理） 参数（3:4比例）

信息图方案：藤蔓缠绕生态绿渐变风格，生态绿配色搭配牛皮纸背景呈现叶脉纹理效果
```

**关键发现**：信息图的每一行都是把字体方案的对应行的6个槽位值"翻译"成了信息图专用的描述语言——风格描述融合了字体形态+色彩+背景+细节，同时加入了信息图专属的排版术语。

## 五类操作的执行方法

### 操作一：换主体

**原理**：字体方案的"字体（XX体）"槽位本质上描述的是**主体形态**，不限于文字。可以将其替换为任何具象或抽象的主体。

**步骤**：

1. 从 `字体设计提示词方案.md` 中选取目标方案
2. 将"字体（XX体）"槽位中的形态描述替换为目标主体的对应形态描述
3. 其余5个槽位（主题、色彩、背景、细节、参数）保持不变
4. 输出新方案

**示例**（藤蔓生态方案，字体→人体姿态）：
```
原方案：示例：主题（自然生态） 字体（藤蔓缠绕形态） 色彩（生态绿渐变） 背景（牛皮纸背景） 细节（叶脉纹理） 参数（3:4比例）
新方案：示例：主题（自然生态） 主体（藤蔓缠绕的人体姿态） 色彩（生态绿渐变） 背景（牛皮纸背景） 细节（叶脉纹理） 参数（3:4比例）
```

可替换的主体类型包括但不限于：物体（日常用品、交通工具、食物、自然物）、人体（人物姿态、服装造型、面部表情）、建筑（建筑外观、室内空间、城市景观）、Logo/品牌标识、抽象图形、动植物等。

### 操作二：要素混搭

**原理**：6个槽位是独立的维度，可以从不同方案中分别取值，组合成新方案。

**步骤**：

1. 确定要混搭的源方案编号（至少2个不同方案）
2. 为每个槽位选择来源方案和对应的槽位值
3. 组合成新方案，确保各槽位值的风格协调性
4. 如用户指定了特定槽位约束，优先满足约束

**示例**（混搭自然生态 + 赛博朋克）：
```
取自自然生态方案：主题（自然生态）
取自自然生态方案：字体（藤蔓缠绕形态）
取自赛博朋克方案：色彩（霓虹粉青撞色）
取自赛博朋克方案：背景（潮湿反光街道）
取自自然生态方案：细节（叶脉纹理）
取自赛博朋克方案：参数（2.35:1）

结果：示例：主题（自然生态） 字体（藤蔓缠绕形态） 色彩（霓虹粉青撞色） 背景（潮湿反光街道） 细节（叶脉纹理） 参数（2.35:1）
```

### 操作三：模板衍生

**原理**：参照信息图方案对字体方案的变形方式，可以将6槽位模板改造为任意新的产物类型。变形的核心是：

- 保留100个风格的**美学DNA**（各槽位的风格值不变）
- 改变**输出壳**（模板格式、槽位名称、产物专属术语）
- 加入目标产物的**领域专用语**

**已验证的变形路径**：字体 → 信息图（见参考文件中的对应关系）

**通用衍生步骤**：

1. 定义新产物的模板格式（需要哪些槽位，槽位叫什么名字）
2. 确定每个新槽位如何映射回原始6槽位（一对一、合并、或新增）
3. 加入新产物类型的领域专用术语和格式要求
4. 用100个风格的值逐条填充，生成完整的100个新方案

**示例**（基于自然生态方案衍生为漫画分镜风格方案）：
```
原方案：示例：主题（自然生态） 字体（藤蔓缠绕形态） 色彩（生态绿渐变） 背景（牛皮纸背景） 细节（叶脉纹理） 参数（3:4比例）

定义漫画模板：分镜格数（X格） 画风（XX风格） 配色（XX方案） 场景背景（XX） 特效/速度线（XX） 构图比例（X:X）
映射关系：
  分镜格数 ← 新增（漫画独有）
  画风 ← 字体槽位（藤蔓缠绕形态 → 藤蔓缠绕风角色线条）
  配色 ← 色彩槽位（生态绿渐变 → 生态绿色板）
  场景背景 ← 背景槽位（牛皮纸背景 → 牛皮纸质感的森林场景）
  特效/速度线 ← 细节槽位（叶脉纹理 → 叶脉状速度线）
  构图比例 ← 参数槽位（3:4比例直接沿用）
结果：分镜格数（4格） 画风（藤蔓缠绕风角色线条） 配色（生态绿色板） 场景背景（牛皮纸质感的森林场景） 特效/速度线（叶脉状速度线） 构图比例（3:4比例）
```

### 操作四：新风格延展

**原理**：20个分组各自有独特的美学逻辑。在每个组内可以延续这种逻辑创造更多方案；也可以跨组融合产生杂交风格。

**组内延展方法**：

1. 选取目标分组，阅读该组内已有5个方案
2. 提取该组的共同美学特征（如"自然与有机"组都强调材质纹理+自然元素）
3. 保持分组的美学基调，引入新的自然/文化/情绪/科技意象
4. 按标准6槽位模板编写新方案
5. 编号为该组范围内的下一个可用编号（如101-105延续自然与有机组）

**跨组融合方法**：

1. 选取2-3个不同组的代表性方案
2. 提取各组最标志性的槽位值特征
3. 融合为一个杂交风格的新方案
4. 创建新分组或归入最接近的现有分组

**输出要求**：严格遵循6槽位模板格式，与原文件中100个方案的写法完全一致。

**示例**（在自然与有机组内延展一个新方案）：
```
已有：示例：主题（自然生态） 字体（藤蔓缠绕形态） 色彩（生态绿渐变） 背景（牛皮纸背景） 细节（叶脉纹理） 参数（3:4比例）
延展：示例：主题（海洋生态） 字体（珊瑚礁生长形态） 色彩（海蓝深浅渐变） 背景（沙质海底） 细节（贝壳珍珠点缀） 参数（16:9比例）
```

### 操作五：领域应用

**原理**：将一个风格方案的各槽位值"翻译"为目标领域的具体创作要素。

**通用映射方法**：

读取目标方案的完整6槽位值，然后针对目标领域逐一回答：

| 原始槽位 | 漫画领域映射 | 小说领域映射 | 游戏领域映射 | 影视领域映射 |
|----------|-------------|-------------|-------------|-------------|
| 主题 | 故事世界观基调 | 小说的题材/流派 | 游戏的世界观设定 | 影片的类型/基调 |
| 字体(形态) | 主角/角色的视觉设计 | 主角形象/叙事声音 | 玩家角色/主角的设计 | 主演角色/视觉符号 |
| 色彩 | 整体配色方案 + 情绪色板 | 叙事氛围/情感基调 | UI配色 + 场景色调 | 调色方案/视觉基调 |
| 背景 | 主要场景/舞台 | 故事发生的时空背景 | 关卡/地图环境 | 主要拍摄场景/美术设计 |
| 细节 | 画面装饰/标志性元素 | 文学修辞/细节描写 | 道具/特效粒子/UI装饰 | 道具/服装/视觉细节 |
| 参数 | 分镜构图比例 | 叙事节奏/章节长度 | 屏幕比例/视角 | 画幅/镜头语言偏好 |

**执行步骤**：

1. 从参考文件中选定基础风格方案（通过编号或关键词匹配）
2. 确定应用的目标领域
3. 按6个槽位逐一映射，生成该领域的具体创作指导
4. 如果领域不在上表中，根据领域特点自行建立合理的映射关系

**示例**（将自然生态方案应用到漫画领域）：
```
基础方案：示例：主题（自然生态） 字体（藤蔓缠绕形态） 色彩（生态绿渐变） 背景（牛皮纸背景） 细节（叶脉纹理） 参数（3:4比例）

漫画领域映射：
  主题 → 故事世界观基调：以自然界生态循环为核心的世界观，万物互联的生命哲学
  字体(形态) → 主角视觉设计：身体线条如藤蔓般蜿蜒流动，服装纹理融入植物纹样
  色彩 → 配色方案 + 情绪色板：主色调生态绿渐变，从嫩绿到深绿表现成长与成熟
  背景 → 主要场景：牛皮纸质感的手绘森林/湿地场景，温暖质朴的基调
  细节 → 标志性元素：画面中反复出现叶脉纹理作为转场和装饰符号
  参数 → 分镜构图：3:4竖向分镜，适合展现纵向生长和层叠的自然场景
```

## 使用指南

### 用户触发此技能时的典型说法

- "用第X号风格生成一套XX方案"
- "把A风格和B风格混搭"
- "基于字体方案衍生物漫画/游戏/小说风格"
- "将第X号风格应用到XX领域"
- "在XX分组里多生成几个新风格"

### 执行流程

1. 判断用户需求属于哪一类操作（换主体/混搭/衍生/延展/领域应用）
2. 按需读取参考文件中的相关方案
3. 应用对应的操作方法生成结果
4. 输出时保持与原文件一致的格式规范
