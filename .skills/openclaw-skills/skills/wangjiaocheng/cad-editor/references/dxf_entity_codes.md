# DXF 组码 (Group Codes) 速查表

## 概述
DXF 文件由**组码+值**对组成。每个组码定义了值的类型和含义。

## 常用通用组码

### 实体公共组码

| 组码 | 类型 | 说明 | 示例 |
|------|------|------|------|
| **0** | 字符串 | **实体类型标识** | `"LINE"`, `"CIRCLE"`, `"INSERT"` |
| **5** | 字符串 | 句柄（唯一ID） | `"1A2"` |
| **6** | 字符串 | 线型名 | `"Continuous"`, `"CENTER"` |
| **8** | 字符串 | **图层名** | `"0"`, `"A-WALL"` |
| **48** | 实数 | 线型比例 | `1.0` |
| **60** | 整数 | 对象可见性（0=可见,1=隐藏）| 0 |
| **62** | 整数 | **颜色号(ACI)** | `1`(红), `7`(白/黑), `0`=BYLAYER |
| **67** | 整数 | 空间标志（0=模型空间,1=纸空间）| 0 |
| **100** | 子类标记 | `"AcDbEntity"`, `"AcDbCircle"` |
| **310** | 二进制 | 二进制数据（代理对象） |
| **330** | 字符串 | 所有权句柄（指向所属块/字典） |
| **340** | 字符串 | 软指针句柄 |
| **347** | 字符串 | 材料对象句柄 |
| **420** | 整数 | 颜色索引（真彩色RGB） | `16711680`(#FF0000) |
| **430** | 字符串 | 颜色名称 | `"Red"` |
| **440** | 实数 | 透明度值 (0=不透明, 1=全透明) | 0.0 |

## 图元专用组码

### LINE（直线）
```
10,20,30 — 起点 X,Y,Z    （WCS坐标）
11,21,31 — 终点 X,Y,Z
```

### CIRCLE（圆）
```
10,20,30 — 圆心 X,Y,Z
40       — 半径
```

### ARC（圆弧）
```
10,20,30 — 圆心 X,Y,Z
40       — 半径
50       — 起始角度（度）
51       — 终止角度（度）
```

### POINT（点）
```
10,20,30 — 点坐标 X,Y,Z
```

### LWPOLYLINE（优化多段线）
```
90       — 顶点数
70       — 标志位（1=闭合）
38       — 高程（标高）
10,20    — 各顶点的 X,Y（重复N次，N=90的值）
40/41/42 — 每个顶点的起始宽度/结束宽度/凸度（可选）
```

### TEXT / MTEXT（文字）

#### TEXT（单行文字）
```
10,20,30 — 插入点 X,Y,Z
40       — 高度（字高mm）如 3.5, 5.0, 7.0
1        — **文字内容**
50       — 旋转角度（度）
41       — 宽度因子(X方向缩放) 默认1.0
51       — 倾斜角度（斜体）
7        — 字体样式名
71       — 文字生成标志（2=反向,4=倒置）
72       — 水平对齐 (0=左,1=中,2=右)
73       — 垂直对齐 (0=基线,1=顶,2=中,3=底)
11,21,32 — 对齐点（当72/73非0时使用）
```

#### MTEXT（多行文字）
```
10,20,30 — 插入点 X,Y,Z
40       — 文字高度
1        — 文本内容（可含格式代码\\P换行\\C颜色等）
50       — 旋转角度
41       — 参考矩形宽度
7        — 样式名
71       — 附着点 (1=左上,2=中上,3=右上 ... 9=右下)
```

MTEXT 格式代码：
| 代码 | 含义 |
|------|------|
| `\\P` | 换段（相当于回车）|
| `\\Cn;` | 颜色（n=ACI色号或RGB）|
| `\\Hnx;` | 字高倍率 |
| `\\Wn;` | 宽度因子 |
| `\\f字体名\|b0\|i1\|c238;` | 字体设置（粗体/斜体/颜色）|

### INSERT（图块插入）
```
2        — **块名**
10,20,30 — 插入点 X,Y,Z
41,42,43 — X/Y/Z 缩放比例
50       — 旋转角度
66       | 属性跟随标志（1=有属性）
```

### DIMENSION（尺寸标注）
```
2        — 尺寸标注块名（由DXF自动生成如 *D0）
10,20,30 — 定义点（标注线的位置）
11,21,31 — 文字中点位置
13,23,33 — 第一个延伸线原点
14,24,34 — 第二个延伸线原点
42       — **实际测量值**（DXF自动计算）
50       | 旋转角度
52,53    — 第一/第二尺寸界线偏移
70       | 标注类型 (0=线性/旋转,1=对齐,2=角,3=直径,
          |         4=半径,5=角三点,6= ordinate)
1        | 用户覆盖的文字（空则显示测量值）
```

### HATCH（填充图案）
```
2        — **图案名**（如 "ANSI31", "SOLID", "AR-CONC"）
70       — 填充方式 (0=NORMAL,1=OUTER,2=IGNORE)
63       — 填充颜色 ACI
10,20    — 边界路径起点（在边界路径实体内循环）
91       — 边界路径数量
92       | 边界路径标志 (1=外环,2=Polyline,4=Derived,
          | 8=Textbox,16=Outermost)
93       | 边界路径中的边数
72       | 边缘类型 (1=Line,2=CircularArc,3=EllipticArc,
          | 4=Spline)
```

### SOLID / TRACE（实心填充/宽线）
```
10,20,30 — 第1角点
11,21,31 — 第2角点
12,22,32 — 第3角点
13,23,33 — 第4角点
```

### ELLIPSE（椭圆）
```
10,20,30 — 中心点
11,21,31 | 长轴端点（相对于中心的方向+长度的一半）
40       | 短半轴与长半轴比（<1为扁椭圆）
```

### SPLINE（样条曲线）
```
70       | 标志位（1=闭合,2=周期性,4=有理,8=平面）
71       | 控制点数
72       | 拟合点数
73       | 节点向量数
74       | 数据点数（有理样条时）
40       | 各节点值（重复73次）
10,20    | 各控制点坐标（重复71次）
10,21    | 各拟合点坐标（重复72次）——注意组码相同！
41       | 权重（仅在有理样条时出现）
```

## 表节组码

### LAYER 表（图层定义）
```
2        — 图层名
70       | 标志位
62       — 颜色号（负数=冻结，如 -7 = 冻结且白色）
6        — 线型名
370      | 线宽（单位：百分之一毫米，-1=BYLAYER，
          | -3=默认，50 = 0.5mm）
290      | 开关（0=关/冻结, 1=开/解冻）
```

### LTYPE 表（线型定义）
```
2        — 线型名
70       | 标志
3        — 描述文本（如 "__ __ __ __"）
72       | 对齐方式（65=A, 58=字符/形）
73       | 划线元素数量
40       | 总图案长度
49       | 各段长度（正=落笔划线,负=抬笔间隔,0=点）
          | 重复73次
```

### STYLE 表（文字样式）
```
2        — 样式名
70       | 标志
40       | 固定文字高度（0=不固定）
41       | 宽度因子
50       | 倾斜角度
71       | 文字生成标志（2=反向,4=倒置）
3        | 主字体文件名（如 "txt.shx"）
4        | 大字体文件名（如 "hztxt.shx"，用于中文）
```

### VPORT 表（视口配置）
```
2        | 视口名
10,20,30 | 视图中心点
11,21,31 | 目标点（Snap基点）
12,22,32 | 从目标点到相机位置的矢量
40,41,42 | 视图高度/宽高比/焦距长度
45       | 前裁剪距离
44       | 后裁剪距离
68       | 状态（0=非活动,1=活动）
69       | ID
71       | 视图模式（按位或组合）:
          |   1=PERSPECTIVE, 2=FRONT_CLIPPING,
          |   4=BACK_CLIPPING, 8=UCSFOLLOWON,
          |   16=FRONT_ZOOM_NOT_AT_CLIP
72       | 圆缩放百分比
73       | 快速缩放标志（0=关闭,1=开启）
74       | UCSICON状态
75       | SNAP开启
76       | GRID开启
77       | SNAP样式（0=标准,1=等轴测）
78       | SNAP等轴测面（1=Top,2=Left,3=Right）
```

| $INSBASE | 10,20,30 | 原点(0,0,0) | 0,0,0 |

## DXF 版本对照表

| 版本代号 | ACADVER值 | AutoCAD版本 | 特点 |
|---------|-----------|-------------|------|
| R12 | AC1009 | AutoCAD 12 | 最后的经典格式 |
| R2000 | AC1015 | AutoCAD 2000 | **最广泛兼容**，推荐使用 |
| R2004 | AC1018 | AutoCAD 2004 | 压缩对象数据 |
| R2007 | AC1021 | AutoCAD 2007 | 更好的压缩 |
| R2010 | AC1024 | AutoCAD 2010 | 参数化约束 |
| R2013 | AC1027 | AutoCAD 2013 | 智能标注 |
| R2018 | AC1032 | AutoCAD 2018 | 最新稳定版 |

> **建议**: 使用 `AC1015`(R2000) 或 `AC1024`(R2010) 作为输出版本，兼容性最佳。

## ezdxf 常用操作速查

```python
import ezdxf

# 创建新文档
doc = ezdxf.new('R2010')           # 或 'AC1024'

# 获取模型空间
msp = doc.modelspace()

# 创建实体
msp.add_line((0,0), (100,0))                    # 直线
msp.add_circle((50,50), radius=25)               # 圆
msp.add_arc((0,0), radius=50, start_angle=0, end_angle=90) # 圆弧
pts = [(0,0),(100,0),(100,50),(0,50)]
msp.add_lwpolyline(pts, close=True)              # 多边形
msp.add_text("Hello", height=5, dxfattribs={'insert': (10,10)}) # 文字

# 图层操作
doc.layers.add('MY_LAYER', color=2)              # 添加图层
layer = doc.layers.get('MY_LAYER')
layer.color = ezdxf.rgb(255, 0, 0)               # 设置真彩色
layer.dxf.linetype = 'DASHED'                    # 设置线型

# 块操作
block = doc.blocks.new('MY_BLOCK')                # 定义块
block.add_line((-5,-5),(5,-5))                   # 块内画内容
msp.add_blockref('MY_BLOCK', insert=(100,100))   # 插入块引用

# 标注（ezdxf内置）
msp.add_linear_dim(
    base=(100,-10), p1=(0,0), p2=(100,0),
    dxfattribs={'dimstyle': 'Standard'}
)

# 保存
doc.savefile('output.dxf')

# 打开已有文档
doc = ezdxf.readfile('existing.dxf')
```
