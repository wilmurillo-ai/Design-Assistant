# 综合知识体系 Skill

## 概述

本 Skill 是一个结构化的综合知识系统，整合了三大知识域：

1. **投资哲学与市场理论** (6篇) — 从价格运动的几何必然性到人性觉悟的向内之路
2. **高级提示词工程** (9篇) — 从增强套件到模式提示的完整提示词体系
3. **AI辅助编程方法论** (3篇) — 提示词在编程中的具体深入应用，从灵感到实现的标准化工作流

---

## 三大域交叉关系

```
                    ┌─────────────────┐
                    │   投资哲学       │  ← 第一层：世界观与思维根基
                    │  (6篇·觉悟阶梯)  │
                    ├─────────────────┤
                    │   提示词工程     │  ← 第二层：语言工具与表达矩阵
                    │  (9篇·技能谱系)  │
                    ├─────────────────┤
                    │   编程方法论     │  ← 第三层：提示词的具体深入应用
                    │  (3篇·落地执行)  │
                    └─────────────────┘
```

- **投资哲学**提供**世界观和思维框架** — 如何看待不确定性、如何理解系统性规律、如何在混沌中找到秩序（认知层）
- **提示词工程**提供**语言工具和表达能力** — 精确传达意图、激发AI创造力的技术（表达层）；其中提示词01的⚙️执行系统被编程01直接引用为元协议，提示词02的领域专用也源自提示词01的模板
- **编程方法论**是**提示词在编程领域的具体深入应用** — 编程01工作流内部直接调用提示词01执行系统+编程02/03样本，三层知识库在编程01中实现物理汇聚（行动层）

三者构成 **"认知→表达→行动"** 的递进闭环：先建立对世界的理解（投资），再掌握精确表达的武器（提示词），最后将这套表达能力聚焦到最具体的执行场景——AI辅助编程（编程）。**提示词01的执行系统既是提示词层的核心，也是编程层工作流的元协议，是两层之间最直接的内容纽带。**

## 使用指南

### 当用户涉及以下场景时，加载本 Skill：

#### 投资与决策类（认知层）
- 市场分析、价格走势、交易系统构建
- 不确定性管理、风险控制、决策框架
- 投资哲学探讨、长期主义思考
- 人生重大选择、自由与安全的权衡

#### 提示词工程类（表达层）
- 高质量提示词编写与优化
- 创意内容生成（文字/图像/设计）
- 专业领域提示词定制
- 模式识别、转换与创新思维
- 信息图、字体等设计类任务

#### 编程开发类（行动层：提示词的具体应用）
- AI辅助编程工作流搭建
- 用提示词驱动的复杂功能需求拆解与描述
- 基于提示词的方法调用结构与代码实现
- 从想法到产品的全流程管理

### 加载后的行为规范

1. **优先引用参考文件内容**：回答时主动关联相关参考文件的核心概念和框架
2. **保持逻辑一致性**：遵循用户定义的逻辑顺序：
   - 大类顺序：**投资→提示→编程**（认知→表达→行动）
   - 投资内部：纵深递进的觉悟阶梯
   - 提示词内部：基础→专项→高阶三引擎（改进/迁移/构建）→套件集成（直用+三引擎）的能力谱系
   - 编程内部：总纲→细节的流程贯通（提示词的具体深入）
3. **跨域联动**：当问题涉及多个领域时，利用三大域的递进关系进行综合分析
4. **不修改参考文件**：references 目录下的 18 个文件为只读参考资料，严禁任何修改操作
5. **概念溯源**：引用核心概念时注明来源文件，便于用户追溯原始上下文
