## 成本测算公式汇总

### 基本公式
```
目标成本 = 报价 / (1 + 利润率)
实际成本 = 人工费 + 材料费 + 机械费 + 管理费
成本偏差 = 目标成本 - 实际成本
```

### 快速估算
```
单方造价 = 总造价 / 建筑面积
单方人工 = 人工费 / 建筑面积
单方材料 = 材料费 / 建筑面积

经验值（框架结构）：
- 单方人工：120~180元/m²
- 单方材料：800~1200元/m²
- 单方机械：60~100元/m²
- 单方管理：80~120元/m²
```
