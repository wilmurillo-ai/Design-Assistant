## 标题模板库

### 模板1：数字+痛点+解决方案
```
格式："[数字]个[痛点]，帮你解决[目标]！"
示例："3个方法，帮你解决今日头条粉丝增长难题！"
```

### 模板2：热点+悬念
```
格式："[热点事件]，但[反转/悬念]！"
示例："iPhone 16发布，但这一点你可能没想到！"
```

### 模板3：情感共鸣+提问
```
格式："为什么你总是[痛点]？可能是[原因]！"
示例："为什么你总是赚不到钱？可能是这3个原因！"
```

### 模板4：对比+反差
```
格式："别人[成就]，你却[现状]？[解决方案]"
示例："别人月入过万，你却还在为涨粉发愁？试试这3招！"
```

### 模板5：身份标签型
```
格式："[身份]靠[动作]月入[收益]：[数字]个别错过！"
示例："90后宝妈靠副业月入10万：这3个冷门赛道别错过！"
```

### 模板6：揭秘式
```
格式："揭秘：[领域]内部不为人知的[真相]"
示例："揭秘：工程造价行业内部不为人知的暴利秘密"
```

### 模板7：震惊式
```
格式："[震惊词]！[具体事件]，引[数量][人群]围观！"
示例："震惊！工地这样做预算，一年省下20万！"
```

### 模板8：实用技巧型
```
格式："[数字]个技巧，让你的[目标]翻倍！"
示例："5个技巧，让你的投标中标率翻倍！"
```

### 模板9：结果导向型
```
格式："学会这[数字]点，[期待结果]不是梦！"
示例："学会这3点，月入过万不是梦！"
```

### 模板10：故事型
```
格式："[身份]从[困境]到[成功]，只因做了[动作]"
示例："造价员从月薪3000到20000，只因掌握了这个技能"
```

## 标题优化技巧

### 前10字法则
标题前10个字决定点击率，必须包含核心关键词

### 数字使用
- 奇数更易记：3、5、7、9
- 具体数字比模糊数字好：3比几个好

### 符号使用
- "！" 制造情绪停顿
- "？" 激发好奇心
- "…" 引导点击
