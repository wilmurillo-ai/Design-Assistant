# 工程造价套定额助手 v2.0

> 一句话：施工企业投标必备，10分钟完成定额套取

## description
专业工程造价套定额工具，帮助施工企业快速完成定额计价、成本测算、企业定额编制。适用于投标报价、成本核算、工程结算等场景。

## trigger
- 套定额、定额计价、定额套取
- 工程预算、工程造价、成本测算
- 投标报价、综合单价、企业定额
- 清单计价、定额换算、材料调差
- 广联达、清华斯维尔、神机妙算

## steps

### 🎯 第一步：项目诊断
```
输入：项目名称 + 类型 + 定额版本
↓
输出：确认项目特征和适用定额
```

### 📋 第二步：清单录入
| 序号 | 清单编码 | 项目名称 | 单位 | 工程量 |
|------|---------|---------|------|-------|
| 1 | 输入 | 清单项目 | m³/m²/t | 数量 |

### 🔗 第三步：智能套定额
```
清单特征 → 自动匹配推荐定额 → 选择确认 → 套取
```

推荐定额库（按使用频率排序）：
1. **广西2018定额** - 本地首选
2. **全国2018定额** - 全国通用
3. **各省市定额** - 按需选择

### 🔄 第四步：定额换算
支持以下换算类型：
- 混凝土等级换算（C20→C30）
- 砂浆等级换算（M5→M10）
- 材料品种换算（钢筋规格）
- 机械台班换算

### 💰 第五步：材料调差
```
基准价 + 价差 = 市场价
自动计算价差金额
```

### 📊 第六步：取费输出
| 费用项目 | 计算公式 | 金额 |
|---------|---------|------|
| 人工费 | 定额含量 × 单价 | ¥X |
| 材料费 | 定额含量 × 单价 | ¥X |
| 机械费 | 定额含量 × 单价 | ¥X |
| 管理费 | (人+材+机) × 费率 | ¥X |
| 利润 | (人+材+机) × 利润率 | ¥X |
| 税金 | (直接费+管理费+利润) × 税率 | ¥X |
| **综合单价** | 合计 | **¥X** |

### 定额选择规则
| 优先级 | 规则 | 说明 |
|-------|------|------|
| 1 | 项目特征匹配 | 清单描述 = 定额工作内容 |
| 2 | 工艺做法相同 | 施工方法一致 |
| 3 | 计量单位一致 | 避免单位换算 |
| 4 | 地区版本优先 | 优先当地定额 |
