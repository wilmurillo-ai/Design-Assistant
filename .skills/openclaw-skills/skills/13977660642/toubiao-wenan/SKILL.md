# 投标文案智能生成器 v2.0

> 一句话：施工企业投标神器，一键生成全套投标文件

## description
专业投标文书生成工具，自动生成投标函、技术标、商务标、报价单等全套投标文件。适用于施工企业、工程公司投标竞标。

## trigger
- 写投标书、投标函、投标文件
- 技术标、商务标制作
- 投标报价、竞标方案
- 工程投标、企业资质
- 标书代写、投标代理

## steps

### 📋 第一步：收集项目信息
```
招标单位 | 项目名称 | 招标编号
资质要求 | 工期要求 | 付款方式
```

### 📝 第二步：生成投标函
| 文件 | 内容 | 模板 |
|-----|------|------|
| 投标函 | 正式投标声明 | 通用模板 |
| 法定代表人证明 | 身份证明 | 通用模板 |
| 授权委托书 | 代理人授权 | 按需填写 |
| 投标保证金 | 缴纳凭证 | 按招标文件 |

### 🔧 第三步：生成技术标
**目录结构**：
```
技术标
├── 1. 工程概况
├── 2. 施工部署
├── 3. 施工方案
├── 4. 进度计划
├── 5. 资源计划
├── 6. 质量保证
├── 7. 安全文明
└── 8. 应急预案
```

**内容自动生成**：
- 根据项目类型匹配施工方案
- 根据建筑面积计算工期
- 根据结构类型选择技术路线

### 💰 第四步：生成商务标
| 内容 | 说明 |
|-----|------|
| 投标总报价 | 大写 + 小写 |
| 报价明细 | 按清单逐项报价 |
| 主要材料表 | 品牌、规格、价格 |
| 优惠条件 | 工期优惠、付款优惠 |

### 🏢 第五步：生成资质文件
```
企业营业执照 → 扫描件上传
资质证书 → 按需提供
类似业绩 → 近3年项目
项目班子 → 注册建造师等
```

## 核心模板库

### 投标函模板
```
_____________________________________________

投  标  函

[招标单位名称]：

1. 我方已仔细研究了[项目名称]（招标编号：[XXX]）招标文件的全部内容，愿意以人民币[大写]元（RMB ¥[小写]元）的投标总报价，按招标文件规定的要求完成本项目的施工。

2. 我方承诺：
   (1) 工程质量达到[国家/行业]标准，一次性验收合格
   (2) 工期[XX]日历天
   (3) 保修期[XX]年
   (4) 项目经理：[XXX]，建造师：[XXX]

3. 我方理解，贵方有权不接受最低报价，也有权选择任何投标文件，且无需向任何投标人解释选择或放弃的原因。

4. 我方承诺在整个投标期间不修改投标文件内容。

5. 随本投标函附上的投标保证金：[金额]元整，已按招标文件要求提交。

投标人：[公司名称]
法定代表人或授权代表：[签字]
日期：[XX年XX月XX日]

_____________________________________________
```
