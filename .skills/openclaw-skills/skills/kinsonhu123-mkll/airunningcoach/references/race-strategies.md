# 比赛策略

---

## 配速策略

### 1. 匀速策略（推荐）

**最适合：** 大多数跑者，尤其是破 3 目标

**执行方式：**
- 全程保持目标配速（±5 秒/km）
- 前 5km 略慢 5-10 秒/km（抑制起跑冲动）
- 5-35km 稳定在目标配速
- 35km 后根据状态调整

**优点：**
- 能量利用最高效
- 避免后程崩盘
- 心理压力小

---

### 2. 负分割策略（Negative Split）

**最适合：** 有经验跑者、追求 PB

**执行方式：**
- 前 10km：比目标配速慢 5-10 秒/km
- 10-30km：目标配速
- 30-40km：维持或略快
- 最后 2km：全力冲刺

**优点：**
- 后程超越心理优势大
- 降低撞车风险
- 整体体验更好

---

### 3. 正分割策略（不推荐）

**常见错误：** 前快后慢

**结果：**
- 前 10km 感觉很好
- 20-30km 开始挣扎
- 30km 后严重掉速
- 最终成绩远低于预期

**避免方法：**
- 起跑前告诉自己"慢就是快"
- 前 5km 看心率不看配速
- 找配速员或兔子跟随

## 补给策略

### 赛前

| 时间 | 内容 |
|------|------|
| 赛前 3 天 | 碳水负荷（碳水占比 70%） |
| 赛前 1 天 | 清淡饮食，避免高纤维 |
| 赛前 2-3h | 早餐（面包 + 香蕉 + 咖啡） |
| 赛前 30min | 能量胶 1 支 + 水 |

### 赛中

| 距离 | 补给 |
|------|------|
| 每 8-10km | 能量胶 1 支 |
| 每个水站 | 少量水或运动饮料 |
| 20km 后 | 考虑盐丸（天气热时） |
| 30km 后 | 每 5km 胶 + 咖啡因胶（可选） |

**要点：**
- 赛前训练时练习补给
- 不要等饿了/渴了再吃
- 进站前提前减速，避免急停

## 心理策略

### 分段法

将 42km 分成若干段：

```
0-10km:  "热身段" - 抑制冲动，找节奏
10-21km: "巡航段" - 稳定配速，放松心态
21-30km: "坚持段" - 专注当下，不想剩余距离
30-35km: "艰难段" - 接受痛苦，分解目标
35-40km: "冲刺段" - 看到终点，激发斗志
40-42km: "荣耀段" - 享受最后，全力冲刺
```

### 积极自我对话

**避免：**
- "还有好多公里..."
- "我坚持不住了..."
- "今天状态不好..."

**替换为：**
- "按计划执行"
- "这种感觉正常"
- "训练就是为了这一刻"

## 常见问题应对

### 撞墙（30-35km）

**症状：** 双腿沉重、配速下降、想放弃

**应对：**
1. 接受现实，不要对抗
2. 降低配速 10-15 秒/km
3. 补充能量胶 + 水
4. 缩短目标（"再跑 1km 就好"）
5. 关注跑姿（抬腿、摆臂）

### 侧腹痛

**应对：**
1. 减速但不走路
2. 深呼吸（腹式呼吸）
3. 按压疼痛部位
4. 调整呼吸节奏（两步一吸、两步一呼）

### 抽筋

**预防：**
- 赛前充分热身
- 赛中补充电解质
- 平时加强力量训练

**应对：**
1. 减速或步行
2. 拉伸抽筋部位
3. 补充盐丸
4. 如严重，考虑退赛
