## 🔑 核心规范摘要

### 节点命名规则
- ✅ 纯中文或中英文混合
- ✅ 不含标点符号
- ✅ 不含企业/品牌名称
- ✅ 不含具体型号参数

### 层级深度
- 一级分类：通过调研发现（不预设）
- 最大层级：5-7 层
- 每层细分：5-15 个子节点

### 分类原则
1. "是什么"优先（技术/产品本质）
2. "用在哪"其次（应用场景）
3. 遵循行业标准
