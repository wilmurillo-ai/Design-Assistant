# Fund Keeper v2.4 - 股票数据整合

## 🎉 新增功能 (v2.4)

### 整合 stock-watcher 数据接口 ✅

**新增股票查询功能**，使用新浪财经 API（类似 stock-watcher）：

- ✅ 实时股票行情
- ✅ 涨跌幅榜
- ✅ 多市场支持（沪市/深市）
- ✅ 完全免费，无需 API key

---

## 📊 新增命令

### 1. 查询个股行情

```bash
py fund_keeper.py stock --code 000001
```

**输出示例**：
```
============================================================
【平安银行 (000001)】实时行情 2026-03-07 21:01:44
============================================================

当前价：  10.82 元
涨跌幅：  +0.09%
涨跌额：  +0.01 元

今开：  10.78 元
最高：  10.84 元
最低：  10.77 元
昨收：  10.81 元

成交量：47,657,680 手
成交额：514,733,550 元
```

---

### 2. 涨幅榜

```bash
py fund_keeper.py stock --gainers --count 10
```

**输出示例**：
```
============================================================
【涨幅榜 TOP10】2026-03-07 21:04
============================================================

排名   代码       名称                价格         涨幅
------------------------------------------------------------
1    000001   平安银行               10.82      +5.20%
2    000002   万科 A                 8.50       +4.80%
...
```

---

### 3. 跌幅榜

```bash
py fund_keeper.py stock --losers --count 10
```

## 💡 使用场景

### 场景 1: 查看持仓基金重仓股表现
```bash
# 先查看基金重仓股（从 stock-analysis）
# 再查询个股行情
py fund_keeper.py stock --code 000001
```

### 场景 2: 发现热门股票 → 对应基金
```bash
# 查看涨幅榜
py fund_keeper.py stock --gainers

# 发现芯片股大涨 → 查询芯片基金
py fund_keeper.py advice --fund 017470
```

### 场景 3: 股票 + 基金联动分析
```bash
# 使用 stock-analysis 的联动功能
py stock_fund_linkage.py --sector 芯片
```

2. **周末/节假日**：显示最后收盘价
