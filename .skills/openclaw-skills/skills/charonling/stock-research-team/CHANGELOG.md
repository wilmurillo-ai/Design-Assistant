# Changelog

## 1.0.0 (2026-03-13)

### Features

- Multi-role AI stock research team with 6 specialized analysts
- MCP Server (`stock-analyzer`) providing 5 financial data tools
- A-share support via akshare (real-time quotes, K-line, financials, technicals)
- US stock support via yfinance
- Bull-bear debate, trading strategy, and risk review workflow
- Composite scoring system (technical 25% + fundamental 30% + macro 20% + sentiment 25%)
- Auto-detect stock market (A-share vs US) based on symbol format
- HTML report template for visual presentation
- Cross-platform support: OpenClaw, WorkBuddy, Claude Desktop
- Automated setup script with Python venv and MCP registration
