# task-manager - 发布信息

## ClawHub 发布描述

### 名称
task-manager

### 描述
OpenClaw 任务管理系统 Skill - 自动记录和管理所有分配给助手的任务。

功能特性：
• 自动维护任务记录表 (Markdown)
• 支持一次性任务和持续性/定时任务
• 记录任务创建时间、状态、调用功能、权限、依赖软件
• 自动统计任务数据（总任务数/执行中/已完成）
• 任何新任务自动记录，保持任务追踪

使用方式：
安装后会自动生效，任何新任务都会被记录到工作区的 TASKS.md 文件中。
支持查询当前任务列表，自动更新状态和统计信息。

适用于：需要长期追踪多个任务和定时任务的用户，帮助保持工作条理清晰。
