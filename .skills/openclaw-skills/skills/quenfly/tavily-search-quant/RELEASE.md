# tavily-search - 发布信息

## ClawHub 发布描述

### 名称
tavily-search

### 描述
集成 Tavily AI 搜索引擎，为 OpenClaw 提供专业的搜索能力，特别适合金融舆情分析、市场研究和信息检索。

功能特性：
• 完整封装 Tavily AI Search API
• 支持深度搜索、基础搜索、问答搜索多种模式
• 提供命令行工具方便调用
• 支持搜索结果格式化和摘要提取
• 适合舆情监控、金融分析、新闻聚合等场景

使用方式：
1. 安装后配置 TAVILY_API_KEY 环境变量
2. 在对话中直接使用："搜索 XXX 相关信息"
3. 支持其他skill调用，如舆情分析策略报告自动获取新闻舆情

依赖：需要 Tavily API 密钥 (可在 https://tavily.com 获取)

适用于：需要高质量AI搜索结果、金融舆情分析、市场情报收集的场景。
