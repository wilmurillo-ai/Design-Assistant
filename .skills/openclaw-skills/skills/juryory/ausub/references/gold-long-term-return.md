# 黄金长期收益参考说明

## 结论

市场上经常会提到“黄金长期平均涨幅约 8%”这一说法。这个数字有一定参考意义，但更适合放在长期资产配置背景里，而不适合直接用于日常动态定投规则。

## 为什么它有参考意义

- 它能说明黄金作为长期配置资产并不是单纯的避险工具
- 它有助于解释为什么长期持有黄金可能具备一定回报潜力
- 它可以帮助用户建立合理的长期收益预期区间

## 为什么不能直接写进策略公式

- 长期平均回报不是短期交易信号
- 历史统计值并不是未来稳定常数
- 动态定投的加减仓，更应该依赖价格位置、趋势强弱和回撤状态
- 如果把长期平均回报直接写进策略公式，很容易把“战略参考”误用成“战术信号”

## 在 OpenClaw 中的正确用法

建议把这类信息放在：

- `README.md` 里作为策略背景说明
- `references/` 里作为研究参考笔记

不建议把它放进这些地方：

- 核心倍率判断
- 布林线修正
- MACD 确认
- 单期投入金额计算

## OpenClaw 的处理原则

OpenClaw 应把“黄金长期约 8% 的历史回报”视为长期背景信息，而不是当前周期的加仓或减仓信号。
