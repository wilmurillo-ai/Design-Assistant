## 关键指标定义

### 电动车特有指标
- **渗透率** = 新能源车销量 / 汽车总销量
- **BEV/PHEV占比** = 纯电/插混销量结构
- **单车带电量** = 平均电池容量(kWh)
- **续航里程** = 主流车型续航区间
- **充电桩车桩比** = 新能源车保有量 / 充电桩数量

### 电池关键指标
- **装机量** = 电池装车总量(GWh)
- **能量密度** = Wh/kg
- **材料体系占比** = 磷酸铁锂/三元/其他占比
- **价格走势** = 元/Wh

## 典型分析框架

### 电动车五力分析要点
1. **竞争强度**：极高，价格战持续，洗牌加速
2. **进入壁垒**：提高中，资金/技术/品牌门槛提升
3. **替代品**：燃油车、氢燃料、公共交通
4. **买方议价**：强，选择多，价格敏感
5. **供应商议价**：分化，电池厂议价强，其他弱

### 关键成功因素
1. **成本控制** - 规模效应+垂直整合
2. **产品力** - 续航/智能/设计
3. **品牌心智** - 差异化定位
4. **渠道能力** - 销售/服务网络
5. **供应链** - 电池等核心资源获取
