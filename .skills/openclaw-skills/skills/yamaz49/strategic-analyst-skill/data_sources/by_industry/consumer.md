## 关键指标定义

### 餐饮特有指标
- **翻台率** = 总餐桌使用次数 / 总餐桌数 / 营业天数
- **坪效** = 营业额 / 营业面积
- **客单价** = 总营业额 / 顾客数
- **同店增长** = 开业一年以上门店的同比增长

### 新茶饮关键数据点
1. **市场规模**：全国/区域现制茶饮销售额
2. **门店数量**：各品牌门店数及增速
3. **价格带分布**：高端(20+)、中端(10-20)、低端(<10)占比
4. **城市层级分布**：一线/新一线/二线/下沉市场
5. **头部品牌市占率**：喜茶、奈雪、茶百道、古茗等

## 典型分析框架

### 新茶饮五力分析要点
1. **竞争强度**：极高，价格战频繁，同质化严重
2. **进入壁垒**：中等，资金门槛低但品牌/点位壁垒高
3. **替代品**：瓶装饮料、咖啡、其他休闲食品
4. **买方议价**：中等，品牌忠诚度低，价格敏感
5. **供应商议价**：低，原材料标准化，供应商分散

### 关键成功因素
1. **选址能力** - 商圈/点位资源
2. **产品创新** - 推新速度和爆款率
3. **供应链** - 原材料成本控制和稳定性
4. **品牌运营** - 社交媒体营销
5. **数字化** - 小程序/会员运营
