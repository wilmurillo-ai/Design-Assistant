## 关键指标定义

### SaaS核心指标
| 指标 | 英文 | 定义 | 健康标准 |
|-----|------|-----|---------|
| 年经常性收入 | ARR | 年度订阅收入 | - |
| 月经常性收入 | MRR | 月度订阅收入 | - |
| 客户终身价值 | LTV | 单客户全生命周期价值 | - |
| 获客成本 | CAC | 获取单个客户成本 | LTV/CAC > 3 |
| 月流失率 | Churn | 月度客户流失比例 | < 5% |
| 净收入留存率 | NRR | 存量客户收入变化 | > 100% |
| _magic number_ | - | 销售效率 = (Q Rev - Q-1 Rev)×4 / 销售投入 | > 0.7 |

### 上市公司参考指标
- 毛利率：通常70%+
- 销售费用率：通常40-60%
- 研发费用率：通常20-40%
- 净利率：通常-30%到10%（增长期多为负）

## 典型分析框架

### SaaS五力分析要点
1. **竞争强度**：高，同质化严重，客户切换成本逐渐降低
2. **进入壁垒**：中等，技术壁垒降低但客户积累需要时间
3. **替代品**：自研系统、传统软件、低代码平台
4. **买方议价**：增强趋势，客户选型更理性
5. **供应商议价**：低，云基础设施 commoditized

### 关键成功因素
1. **产品-市场契合** - 解决真实痛点
2. **获客效率** - LTV/CAC健康度
3. **客户成功** - 留存和扩展销售
4. **产品体验** - 易用性决定口碑
5. **生态集成** - 与其他工具打通
