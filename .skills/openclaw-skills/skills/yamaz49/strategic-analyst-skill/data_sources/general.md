## 搜索技巧

### 关键词组合
```
行业 + 市场规模/市场容量/市场空间
行业 + 竞争格局/市场份额/CR5
行业 + 发展趋势/前景预测/增长率
行业 + 政策法规/监管/标准
行业 + 融资/投资/并购
公司名 + 财报/年报/招股书
```

### 文件类型限定
```
site:pdf 行业名 报告
filetype:pdf 行业名 市场规模
```

### 时间限定
优先搜索近2年数据，使用搜索引擎的时间筛选功能。

## 数据质量评估

### 可信度分级
| 等级 | 描述 | 使用建议 |
|-----|------|---------|
| ★★★★★ | 政府统计、上市公司财报 | 可直接引用 |
| ★★★★☆ | 头部券商、知名咨询机构 | 可引用，注明来源 |
| ★★★☆☆ | 行业媒体、第三方平台 | 需交叉验证 |
| ★★☆☆☆ | 自媒体、论坛 | 仅作参考线索 |

### 常见问题
1. **口径不一致** - 不同来源对"市场规模"定义可能不同
2. **预测偏差** - 咨询公司的预测往往偏乐观
3. **样本偏差** - 调研数据可能不具代表性
4. **时效性** - 注意数据发布时间，快速变化行业需特别留意
