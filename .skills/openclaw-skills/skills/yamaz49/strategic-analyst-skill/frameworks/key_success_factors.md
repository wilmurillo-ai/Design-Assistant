# 关键成功因素（CSF）分析框架

## 定义

关键成功因素（Critical Success Factors, CSF）是指在特定行业中，企业必须掌握或做到位才能取得成功的核心要素。

## 识别方法

### 1. 价值链分析法

沿着行业价值链，识别每个环节的关键能力：

```
研发 → 采购 → 生产 → 营销 → 销售 → 服务
 │      │      │      │      │      │
[关键] [关键] [关键] [关键] [关键] [关键]
能力1  能力2  能力3  能力4  能力5  能力6
```

### 2. 赢家分析法

研究行业领先者，提炼共同的成功要素：
- 领先者都做对了什么？
- 失败者都错在哪里？
- 近年来崛起的新秀靠什么成功？

### 3. 客户决策因素法

从客户购买决策角度反推：
- 客户选择供应商时最看重什么？
- 不同客群的关注点有何差异？
- 决策因素的重要性排序？

### 4. 生命周期法

不同发展阶段，CSF有所不同：

| 阶段 | 核心CSF | 次要CSF |
|-----|--------|--------|
| 导入期 | 产品创新、教育市场 | 品牌建设 |
| 成长期 | 渠道扩张、规模效应 | 产品迭代 |
| 成熟期 | 成本控制、差异化 | 运营效率 |
| 衰退期 | 现金管理、业务转型 | 成本控制 |

## CSF分类框架

### 按性质分类

| 类型 | 描述 | 示例 |
|-----|------|------|
| 资源类 | 必须具备的资源/资质 | 牌照、专利、矿产 |
| 能力类 | 核心竞争能力 | 研发、品牌、渠道 |
| 战略类 | 正确的战略选择 | 定位、时机、模式 |
| 执行类 | 卓越的运营执行 | 效率、质量、服务 |

### 按层次分类

1. **行业层面CSF** - 进入该行业的门票
2. **竞争层面CSF** - 在竞争中胜出的关键
3. **企业层面CSF** - 特定企业的独特优势

## 能力差距分析

### 分析步骤
1. 列出行业CSF（5-8项）
2. 评估目标企业在每项的表现（1-5分）
3. 评估主要竞争者在每项的表现
4. 识别能力差距和优势

### 差距矩阵模板

| CSF | 权重 | 本企业 | 竞品A | 竞品B | 行业标杆 | 差距 | 优先级 |
|-----|-----|-------|------|------|---------|-----|-------|
| | | | | | | | |

**解读：**
- 高权重+低得分 = 急需改进
- 高权重+高得分 = 核心优势，需保持
- 低权重+低得分 = 可暂缓
- 低权重+高得分 = 资源可能错配

## 常见行业CSF参考

### 消费品行业
1. 品牌建设能力
2. 渠道覆盖能力
3. 产品创新能力
4. 供应链效率
5. 消费者洞察

### SaaS行业
1. 产品体验
2. 获客效率（LTV/CAC）
3. 客户成功（留存/扩展）
4. 技术架构
5. 品牌信任

### 制造业
1. 成本控制
2. 质量管理
3. 交付能力
4. 技术工艺
5. 大客户资源

### 平台型行业
1. 网络效应
2. 双边运营
3. 技术平台
4. 资金实力
5. 监管合规
