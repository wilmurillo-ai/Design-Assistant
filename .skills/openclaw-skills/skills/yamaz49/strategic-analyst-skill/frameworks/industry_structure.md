# 行业结构分析框架

## 概述

基于迈克尔·波特的五力模型（Porter's Five Forces），分析行业竞争强度和盈利能力。

## 五力模型

### 1. 现有竞争者竞争强度

**评估维度：**
- 行业增长率（高增长→竞争较弱）
- 行业集中度（CR4/CR8/赫芬达尔指数）
- 产品差异化程度
- 退出壁垒高低
- 固定成本占比

**判断标准：**
| 指标 | 高强度 | 低强度 |
|-----|-------|-------|
| 行业增速 | <5% | >15% |
| CR4 | <30% | >60% |
| 产品差异化 | 同质化 | 高度差异化 |
| 退出壁垒 | 高（专用资产） | 低 |

### 2. 新进入者威胁

**进入壁垒类型：**
- **规模经济** - 最小有效规模 vs 市场规模
- **资本需求** - 初始投资金额
- **转换成本** - 客户转换供应商的成本
- **渠道控制** - 分销渠道的可获得性
- **政策壁垒** - 许可证、监管要求
- **品牌/学习曲线** - 先发优势

**评估问题：**
- 成功进入需要多少初始投资？
- 新进入者多久能达到盈亏平衡？
- 历史上新进入者的存活率如何？

### 3. 替代品威胁

**分析要点：**
- 替代品的性价比比较
- 客户转换到替代品的成本
- 替代品的技术发展趋势
- 客户对替代品的接受度

**常见替代品类型：**
- 技术替代（如流媒体替代DVD）
- 商业模式替代（如订阅替代购买）
- 需求替代（如外卖替代自炊）

### 4. 买方议价能力

**强议价能力的信号：**
- 买方集中度高
- 采购量占卖方收入比重大
- 产品标准化、易于比较
- 买方转换成本低
- 买方可后向一体化
- 买方信息充分透明

### 5. 供应商议价能力

**强议价能力的信号：**
- 供应商集中度高
- 替代品稀少
- 转换成本高
- 供应商可前向一体化
- 该原料占产品成本比重大

## 综合分析

### 行业吸引力评估矩阵

| 力量 | 强度(1-5) | 趋势 | 关键驱动因素 |
|-----|----------|-----|------------|
| 现有竞争 | | | |
| 新进入威胁 | | | |
| 替代品威胁 | | | |
| 买方议价 | | | |
| 供应商议价 | | | |

**综合评分：**
- 25-20分：行业结构优秀，利润潜力高
- 19-15分：行业结构良好，适度竞争
- 14-10分：行业结构一般，竞争激烈
- <10分：行业结构差，利润被挤压

## 输出模板

### 结构摘要
```
【行业结构总评】[优秀/良好/一般/差]

【竞争强度】[高/中/低] - 关键原因：
【进入壁垒】[高/中/低] - 主要壁垒：
【替代品威胁】[高/中/低] - 主要替代品：
【买方议价】[强/中/弱] - 主要买方：
【供应商议价】[强/中/弱] - 主要供应商：

【结构趋势】未来3-5年的演变方向
【战略启示】对企业定位的建议
```

## 案例应用提示

分析具体行业时，需结合：
1. 行业历史数据验证框架假设
2. 近期重大事件对结构的影响（并购、新技术、政策变化）
3. 细分市场的结构差异
4. 不同区域市场的结构特点
