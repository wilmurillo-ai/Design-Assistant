# 竞争格局分析框架

## 分析维度

### 1. 市场集中度分析

**衡量指标：**
- **CR4/CR8** - 前4/8家企业市场份额之和
- **HHI指数** - 赫芬达尔-赫希曼指数

**集中度判断：**
| 类型 | CR4 | HHI | 竞争特点 |
|-----|-----|-----|---------|
| 极高寡占 | >75% | >2500 | 龙头主导，跟随者边缘化 |
| 高集中寡占 | 50-75% | 1800-2500 | 头部竞争激烈，壁垒高 |
| 低集中寡占 | 35-50% | 1000-1800 | 多强竞争，差异化重要 |
| 竞争型 | <35% | <1000 | 充分竞争，价格战风险 |

### 2. 战略群组划分

**划分维度：**
- 目标客户群体（高端/大众/低端）
- 产品/服务定位（全面/细分）
- 地理覆盖（全国/区域/本地）
- 商业模式（直营/加盟/平台）
- 垂直整合程度

**竞争强度判断：**
- 同群组内竞争 > 跨群组竞争
- 群组间移动壁垒高低

### 3. 竞争者画像

**每家企业分析要点：**

**基础信息：**
- 规模（收入、员工、门店数等）
- 发展历程与关键里程碑
- 股权结构与资本背景

**战略定位：**
- 目标市场与客群
- 核心产品/服务
- 价值主张

**能力评估：**
| 能力维度 | 评分(1-5) | 核心优势 | 主要短板 |
|---------|----------|---------|---------|
| 产品力 | | | |
| 品牌力 | | | |
| 渠道力 | | | |
| 供应链 | | | |
| 技术力 | | | |
| 资金实力 | | | |
| 组织力 | | | |

**近期动态：**
- 战略举措（并购、新品、渠道扩张）
- 财务表现（增速、盈利、现金流）
- 人事变动
- 风险信号

### 4. 竞争动态追踪

**关键监测指标：**
- 价格战频率与幅度
- 新品推出速度
- 渠道扩张节奏
- 营销投入强度
- 人才流动方向

**竞争态势演变：**
- 行业处于什么阶段？（蓝海/红海/洗牌/稳定）
- 主要玩家的战略重心变化
- 新进入者的威胁程度
- 潜在并购整合可能性

## 数据来源

- 上市公司财报
- 工商注册信息
- 行业排名榜单
- 招聘数据（反映扩张意图）
- 媒体报道与公告
- 实地调研（门店观察、渠道访谈）
