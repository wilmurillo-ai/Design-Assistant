# 数据收集检查清单

## 市场规模数据

### 必查项
- [ ] 行业总体市场规模（TAM）
- [ ] 近3-5年历史增长率
- [ ] 未来5年增长预测
- [ ] 细分市场结构（如适用）
- [ ] 区域市场分布

### 验证项
- [ ] 至少2个来源交叉验证市场规模
- [ ] 确认统计口径（GMV/营收/产量）
- [ ] 确认货币单位和时间点
- [ ] 检查数据时效性（优先近2年）

---

## 竞争格局数据

### 必查项
- [ ] 头部企业名单（前5-10名）
- [ ] 市场份额数据（CR4/CR8）
- [ ] 各企业核心业务和定位
- [ ] 近期重要战略动作（并购/新品/扩张）
- [ ] 财务数据（上市公司）

### 验证项
- [ ] 市场份额数据来源
- [ ] 企业信息的时效性
- [ ] 是否包含新兴挑战者
- [ ] 是否有即将上市的新玩家

---

## 政策法规数据

### 必查项
- [ ] 行业监管主体和政策
- [ ] 最新政策变化（近1-2年）
- [ ] 政策趋势（收紧/放松/规范）
- [ ] 行业标准/准入要求

### 验证项
- [ ] 政策原文出处
- [ ] 政策实施时间
- [ ] 地方政策差异（如适用）

---

## 投融资数据

### 必查项
- [ ] 近1年重大融资事件
- [ ] 投资热点领域
- [ ] 主要投资机构
- [ ] 估值水平趋势

### 验证项
- [ ] 融资金额是否官方披露
- [ ] 估值数据可靠性

---

## 趋势数据

### 必查项
- [ ] 技术发展趋势
- [ ] 消费行为变化
- [ ] 供应链变化
- [ ] 替代品动态

---

## 数据源质量检查

### 来源分级使用
| 来源类型 | 使用方式 | 是否需要交叉验证 |
|---------|---------|----------------|
| 政府统计 | 直接引用 | 否 |
| 上市公司财报 | 直接引用 | 否 |
| 头部券商研报 | 引用+标注 | 建议 |
| 咨询机构报告 | 引用+标注 | 建议 |
| 行业媒体 | 参考+验证 | 必须 |
| 自媒体/论坛 | 仅作线索 | 必须 |

### 数据记录规范
每条数据必须记录：
- [ ] 数据项名称
- [ ] 数值
- [ ] 来源（具体到报告/文章标题）
- [ ] 来源链接（原始报告或数据发布页面的 URL）
- [ ] 发布时间
- [ ] 可信度评分（1-5星）
- [ ] 备注（口径说明/局限性）
