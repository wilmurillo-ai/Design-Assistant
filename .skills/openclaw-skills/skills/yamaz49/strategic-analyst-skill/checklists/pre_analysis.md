# 分析前检查清单

## 需求访谈检查项

### 用户身份确认
- [ ] 明确用户角色（CEO/高管/学生/新进入者）
- [ ] 了解用户所在企业/机构背景
- [ ] 确定用户的行业经验和认知水平
- [ ] 明确用户的时间紧迫程度

### 决策问题澄清
- [ ] 具体行业名称和边界范围
- [ ] 核心决策问题（进入/投资/竞争/学习）
- [ ] 决策时间框架（短期/中期/长期）
- [ ] 资源约束条件
- [ ] 关键假设和已知信息

### 分析范围确认
- [ ] 地理范围（全国/区域/全球）
- [ ] 细分市场（全市场/细分领域）
- [ ] 时间范围（历史回顾/现状/未来预测）
- [ ] 重点关注的分析维度

### 输出要求确认
- [ ] 报告形式（摘要/完整/学习版）
- [ ] 交付时间
- [ ] 是否需要可视化
- [ ] 是否需要持续监控建议

---

## 分析框架选择指南

### 根据决策问题选择框架

| 决策问题 | 必选框架 | 可选框架 | 重点输出 |
|---------|---------|---------|---------|
| 是否进入该行业？ | 五力模型 + 市场规模 | CSF、价值链 | 进入策略建议 |
| 竞争策略制定 | 竞争格局 + CSF | 五力、价值链 | 差异化定位 |
| 投资决策 | 市场规模 + 趋势 | 五力、竞争格局 | 投资价值评估 |
| 行业全面学习 | 全部框架 | - | 完整认知地图 |
| 趋势判断 | 趋势分析 + 情景规划 | 五力 | 战略应对建议 |

### 根据用户类型调整深度

| 用户类型 | 框架深度 | 方法论解释 | 案例需求 |
|---------|---------|-----------|---------|
| CEO | 应用层面 | 简要 | 1-2个相关 |
| 高管 | 应用+原理 | 适度 | 2-3个相关 |
| 学生 | 完整+方法论 | 详细 | 3-5个详细 |
| 新进入者 | 应用+入门 | 适度 | 2-3个基础 |

## 快速启动问题模板

### 开场问题（必问）
1. "您关注的是哪个具体行业？（请给出准确的行业名称）"
2. "您当前的角色是什么？（CEO/高管/学生/其他）"
3. "您面临的核心问题是什么？（想解决什么决策问题）"
4. "您对这个行业目前的了解程度如何？"
5. "您希望什么时候得到分析结果？"

### 深入问题（根据情况）
- "您是否已经有一些初步想法或假设？"
- "是否有特定的竞争对手需要重点关注？"
- "您的资源约束是什么？（资金/时间/人力）"
- "地理范围是全国还是特定区域？"
- "是否需要和您现有的业务进行对比分析？"
