## Formatting Triggers

| Trigger | Fix |
|---------|-----|
| ALL CAPS words (>1 word) | Sentence case |
| Excessive punctuation (!!!, ???) | Single punctuation mark |
| Excessive bold/color | Minimal emphasis, 1-2 bold phrases max |
| Red or bright-colored text | Default text color |
| Large font sizes (>16px in body) | 14–16px standard |
| Multiple font families (>2) | Single font family |
| Background colors on text | Remove or use subtle tones |
