## 📋 快速开始

### 前置要求
- [ ] 安装 Zotero 桌面应用
- [ ] 配置 Zotero 本地存储路径
- [ ] (可选) 获取 Zotero API Key

### 使用步骤

#### 1. 启动技能
```
触发词：文献综述 / literature review / Zotero 综述
```

#### 2. 任务确认
技能会自动与您确认以下信息：
- 检索范围（本地 Zotero / 联网搜索 / 两者结合）
- 文献主题和关键词
- 文献数量要求
- 字数要求
- 输出格式
- 时间预期
- 特殊要求

#### 3. 执行任务
技能自动执行 6 阶段流程：
```
任务确认 → 文献检索 → 验证阅读 → 内容提取 → 深度撰写 → 质量检查 → 文件整理
```

#### 4. 交付成果
- 文献综述主文档（Word + Markdown）
- 任务总结报告
- 项目索引
- 质量检查结果

## 🎯 适用场景

### ✅ 推荐使用
- 已有 Zotero 文献库，需要撰写综述
- 需要基于真实 PDF 内容的准确综述
- 需要深度分析而非简单罗列
- 需要符合学术规范的参考文献

### ❌ 不推荐使用
- 没有 Zotero 文献库
- 只需要简单文献列表
- 追求速度而非质量
- 可以接受编造文献信息

## ❓ 常见问题

### Q1: 我没有 Zotero 文献库，可以使用吗？
**A**: 可以，但建议选择"仅联网搜索"模式，技能会使用 Tavily 或 Multi-Search 搜索网络文献。

### Q2: 我的 Zotero 文献库很小，怎么办？
**A**: 建议选择"本地 Zotero + 联网搜索"模式，技能会结合本地文献和网络搜索结果。

### Q3: 可以只分析特定集合 (Collection) 的文献吗？
**A**: 可以，在任务确认时说明需要分析的 Zotero 集合名称。

### Q4: 可以导出为 EndNote/NoteExpress 格式吗？
**A**: 可以，在任务确认时说明需要的参考文献格式。

### Q5: 时间紧急，可以加急吗？
**A**: 可以，但建议分阶段交付，保证质量。加急可能影响深度分析。
