## 常见问题解答

### Q1: 如果找不到足够的文献怎么办？
**A**: 
1. 扩大关键词范围
2. 使用同义词和相关词
3. 检查文献库是否完整
4. 如确实不足，如实告知用户

### Q2: 如果 PDF 无法读取怎么办？
**A**:
1. 检查 PDF 是否加密
2. 尝试使用不同的 PDF 读取库 (PyMuPDF、pdfplumber)
3. 如是扫描件，使用 OCR 工具
4. 如确实无法读取，标注"无法提取内容"

### Q3: 如果时间不够怎么办？
**A**:
1. 优先保证准确性（不能编造信息）
2. 可以减少文献数量（需用户同意）
3. 可以分阶段交付（先完成核心部分）
4. 不能牺牲质量换取速度

### Q4: 如何处理"待验证"信息？
**A**:
1. 明确标注"期刊：待验证"、"DOI: 待验证"
2. 在参考文献列表中说明
3. 建议用户后续通过学术数据库验证
4. 绝不编造看似完整但实际虚假的信息
