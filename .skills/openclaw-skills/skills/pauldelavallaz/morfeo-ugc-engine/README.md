# morfeo-ugc-engine

OpenClaw skill para controlar el **Morfeo UGC Engine** desde Telegram.

Genera videos UGC para marcas argentinas con lip-sync, personajes IA y plot twist de Morfeo Labs — directamente desde el chat.

## Instalación

```bash
clawhub install morfeo-ugc-engine
```

O manual:
```bash
mkdir -p ~/clawd/skills/morfeo-ugc-engine
# Copiar SKILL.md a ese directorio
```

## Uso

Desde OpenClaw/Telegram:
- "generá un video de Havanna"
- "qué runs están en curso"
- "avanzá el run de Aquarius"
- "mostrá el estado del engine"
- "publicá el video de [run]"

## Requisitos

- Morfeo Engine corriendo en localhost:3336
- Repos: https://github.com/PauldeLavallaz/morfeo-engine
