# 城市参考池（供 flyai 参考使用）

**用途说明：**
- **主用途**：为 flyai 提供参考，让推荐更精准、符合人格特征
- **知识补充**：提供季节加成等 flyai 可能不了解的本地信息

**使用逻辑：**
1. **调用 flyai 时**，在 prompt 中引用本文件作为参考池
2. flyai 结合参考池的人格特征 + 实时搜索热度数据，输出精准推荐

## 参考池使用说明

本文件为 flyai 提供人格特征描述和季节信息，不直接输出给用户。

flyai 调用格式：
```bash
flyai keyword-search --query "{{人格类型}} {{月份}} 适合去哪玩" --reference cities.md
```
