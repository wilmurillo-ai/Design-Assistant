# 旅行人格测试

不想跟风打卡？来测测你的旅行人格。

10道趣味测试，发现你是追求独处的山系旅人、烟火气满满的城市猎人，还是随性自在的夜行者——并根据当下季节和实时热度，为你精选3个最适合现在出发的目的地。

---

## ✨ 特点

- 🎯 **轻量测试**：10题以内，2分钟完成，不填问卷
- 🎭 **多选支持**：可选1-2个选项，把更戳你的那个放前面
- 🌍 **实时推荐**：可选接入 flyai 获取热门目的地和机票价格
- 🎨 **6种人格**：海系、山系、城市、夜行、觅食、云旅
- 💬 **自然交互**：氛围感话术，不生硬不贴标签

---

## 🎭 人格类型

| 人格 | 关键词 | 一句话描述 |
|------|--------|------------|
| 🌊 海系旅人 | 治愈、慢节奏、刚刚好 | 不是非要看海，只是需要一种什么都可以不想的感觉 |
| 🏔️ 山系旅人 | 清净、独处、深呼吸 | 人群让你消耗，自然让你充电 |
| 🌆 城市猎人 | 烟火气、探索、真实感 | 菜市场、老巷子、路边摊才是你的旅行地图 |
| 🌙 夜行者 | 反差、神秘、不走寻常路 | 别人睡了你才出发，别人打卡的地方你不去 |
| 🍜 觅食人 | 吃货、烟火、值得专程去 | 你用味觉丈量每座城市，一碗面可以让你爱上一个地方 |
| ☁️ 云旅人 | 诗意、想象力、总有一天 | 你去过很多地方，只是还没用脚 |

## 🔧 集成 flyai

本 skill **依赖 flyai** 获取实时推荐数据。安装后可获取：
- 实时热门目的地数据
- 机票/酒店价格查询
- 搜索量趋势分析

### 安装 flyai

参考：https://github.com/alibaba-flyai/flyai-skill

### 配置 API Key（如需）

```bash
export FLYAI_API_KEY="your_api_key"
```

### 兜底机制

如果 flyai 调用失败（如配额用完），会自动使用 `cities.md` 中的参考数据作为兜底推荐。
