# 旅行人格测试 - 后续优化计划

## 当前版本：v1.0.1

---

## 🎯 深度版规划（Deep Mode）

### 背景
轻量版10题设计保证了"2分钟完成、不填问卷"的体验，但部分用户可能希望获得更精准的结果。深度版作为可选扩展，在保持主流程轻量的同时，为想要深挖的用户提供出口。

---

### 触发机制

**时机**：轻量版结果输出后，自然引入反馈选项

```
> 你是「城市猎人」...

这个结果准吗？
- A. 挺准的 👍
- B. 有点偏差，想再测测看 🔍
```

- 选A：正常结束，进入城市推荐流程
- 选B：进入深度版（5-8道增量题目）

---

### 深度版设计

| 维度 | 轻量版 | 深度版 |
|------|--------|--------|
| 题量 | 10题 | +5-8题 |
| 选项数 | 4选1-2 | 6选，支持优先级排序 |
| 题型 | 场景选择题 | 情景判断 + 反选题 |
| 输出 | 单一人格/双人格 | 人格光谱图 + 子类型标签 |

---

### 深度版题目方向

#### 1. 决策模式（区分控制欲）
- 计划型 vs 随性型
- 例："出发前你会做多少准备？"

#### 2. 社交耗能（细化独处偏好）
- 独处恢复速度
- 区分海系 vs 山系的微妙差异

#### 3. 感官优先级
- 视觉/味觉/听觉哪个更能触发旅行冲动
- 与觅食人、云旅人等类型交叉验证

#### 4. 回忆锚点
- 旅行结束后印象最深的是什么
- 反推真实需求（拍照/感悟/美食/平静）

#### 5. 反向排除题
- "以下哪种旅行你最不会选择？"
- 排除法有时比正向选择更精准

---

### 计分机制

**方案A：增量加权**
- 轻量版分数作为基础分
- 深度版题目权重更高（用户投入更多时间，选择更认真）

**方案B：边界修正**
- 深度版只针对轻量版中分数接近的两个人格
- 专注区分模糊边界

---

### 技术实现

```yaml
# skill.yaml 配置
quiz:
  lite:
    question_count: 10
    max_options: 2
    required: true
  deep:
    question_count: 8
    max_options: 3
    requires_lite: true  # 必须先完成轻量版
    trigger: user_feedback  # 由用户反馈触发
```
