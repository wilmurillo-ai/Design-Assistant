## English

### Overview
**OmniSync** is a high-performance, standalone synchronization engine designed for the **Claw Ecosystem** and beyond. It implements a native semantic diffing algorithm that reduces token consumption in LLM-based micro-feeds by up to 90% while maintaining absolute independence from external libraries.

### Features (Phase 1)
*   **Zero Dependencies**: Optimized for portability. Works on any Python 3.10+ environment without `pip install`.
*   **Standard Engine**: High-reliability SHA-256 integrity validation.
*   **MCP Gateway**: Minimalist JSON-RPC 2.0 server over stdio for universal agent compatibility.
*   **Sovereign Strategy**: Built with an "Independent Project" ideology, allowing use across multiple platforms.

### Quick Start
1.  Launch the standalone gateway: `python mcp_gateway.py`.
2.  Send JSON-RPC requests via stdio to the `sync_standard` tool.

## Español

### Descripción General
**OmniSync** es un motor de sincronización independiente y de alto rendimiento diseñado para el **Ecosistema Claw** y más allá. Implementa un algoritmo de diff semántico nativo que reduce el consumo de tokens en micro-feeds basados en LLM hasta en un 90%, manteniendo una independencia absoluta de librerías externas.

### Características (Phase 1)
*   **Cero Dependencias**: Optimizado para la portabilidad. Funciona en cualquier entorno Python 3.10+ sin necesidad de `pip install`.
*   **Motor Estándar**: Validación de integridad SHA-256 de alta fiabilidad.
*   **Gateway MCP**: Servidor JSON-RPC 2.0 minimalista sobre stdio para compatibilidad universal con agentes.
*   **Estrategia Soberana**: Construido bajo la ideología de "Proyecto Independiente", permitiendo su uso en múltiples plataformas.

### Inicio Rápido
1.  Lanza el gateway independiente: `python mcp_gateway.py`.
2.  Envía peticiones JSON-RPC vía stdio a la herramienta `sync_standard`.
