# SUPAH Research Intelligence

Professional-grade web research with multi-source verification and credibility scoring.

## Features

✨ **Multi-Source Research** - Query 3-5 authoritative sources per topic  
✅ **Fact Verification** - Cross-check claims across multiple sources  
📊 **Credibility Scoring** - Evaluate source reliability and authority  
📝 **Citation Tracking** - Auto-generate bibliographies  
📄 **Research Reports** - Comprehensive analysis with confidence scoring

## Installation

### OpenClaw CLI
```bash
openclaw skills install https://tools.supah.ai/skills/supah-research-intelligence
```

### GitHub
```bash
git clone https://github.com/supah-based/supah-research-intelligence.git
cd supah-research-intelligence
npm link
```

## Usage

### Commands

```bash
# Multi-source research
research "AI safety developments"

# Fact verification
verify "Earth is flat"

# Source credibility scoring
credibility "https://example.com/article"

# Full research report
report "quantum computing"
```

### Natural Language (via OpenClaw agent)

Ask your agent:
- "Research the latest AI breakthroughs"
- "Verify this claim: [statement]"
- "Check the credibility of this source: [url]"
- "Generate a research report on climate change"
