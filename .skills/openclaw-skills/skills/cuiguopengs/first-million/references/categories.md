# 支出分类标准

## 分类设计原则

- **互斥性**：每笔支出只属于一个分类
- **完整性**：覆盖所有日常支出场景
- **层次性**：主分类下可设子分类

## 推荐分类体系

### 必要支出（刚需）
- **餐饮**：早餐、午餐、晚餐、零食、外卖
- **交通**：公交、地铁、打车、加油、停车、打车费
- **住房**：房租、水电燃气、物业费、宽带
- **通讯**：话费、流量包、网络

### 生活消费
- **购物**：衣物、日用品、电子产品、家具家电
- **娱乐**：电影、游戏、聚会、演唱会
- **学习**：书籍、课程、培训
- **健康**：药品、健身、体检

### 其他支出
- **社交**：礼物、请客、红包
- **旅行**：机票、酒店、景点门票
- **其他**：杂费、意外支出

## 收入分类

- **工资**：固定月薪
- **奖金**：年终奖、项目奖金
- **投资**：理财收益、股票分红
- **兼职**：副业收入
- **其他**：红包、礼金

## 使用建议

1. **记录时优先选择主分类**，避免过度细分
2. **子分类用备注说明**，如"餐饮 - 外卖（公司附近）"
3. **每月初调整预算**，根据上月支出情况
4. **异常支出单独标注**，如"医疗 - 急诊"

## 自定义分类

用户可根据个人情况添加或调整分类：
- 有孩子：增加"教育"、"亲子"
- 有宠物：增加"宠物"
- 有车：细分交通为"油费"、"停车"、"保养"
