# MEMORY.md - 长期记忆模板

> 记录重要经验、偏好、决策

---

## 结构说明

```
MEMORY.md 是长期记忆的核心文件，用于记录：
- 重要决策和结论
- 用户偏好和习惯
- 经验教训
- 关键事件
- 技能和能力
```

---

## 模板

```markdown
# MEMORY.md - 长期记忆

## 关于 AI

- **名字**: 
- **创造者**: 
- **类型**: 

## 用户信息

- 用户名：
- 重要偏好：
- 禁忌事项：

## 重要决策

- [日期] 决策内容

## 经验教训

- [日期] 经验总结

## 待处理

- [ ] 待办事项1
- [x] 已完成事项

## 最近更新
*最后更新: YYYY-MM-DD*
```

---

## 示例

```markdown
# MEMORY.md - 长期记忆

## 关于书灵

- **名字**: 书灵 (BooLin)
- **创造者**: 大仙
- **类型**: AI 助手 (运行于 OpenClaw)

## 用户信息

- 用户名：大仙
- 重要偏好：中文优先、结果输出按优先级
- 禁忌事项：暂未发现

## 重要决策

- [2026-03-09] 建立六层记忆系统架构
- [2026-03-09] 制定 RULES.md 沟通规则

```

---

## 更新频率

| 场景 | 是否更新 |
|------|---------|
| 用户明确要求记住 | ✅ 立即 |
| 重要决策 | ✅ 立即 |
| 重要经验 | ✅ 每日反思时 |
| 用户偏好变化 | ✅ 立即 |

---

## 维护建议

1. 每周回顾一次 MEMORY.md
2. 删除过时信息
3. 合并重复内容
4. 更新待处理清单
