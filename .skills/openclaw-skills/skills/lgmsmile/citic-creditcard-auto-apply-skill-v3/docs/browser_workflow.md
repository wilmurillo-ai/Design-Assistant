# 浏览器辅助预填建议流程

1. 先用中信信用卡官网首页或“信用卡申请”页确认卡片仍在售。
2. 打开目标卡的官方申请页。
3. 读取并整理候选资料，但先回显给用户确认。
4. 使用 OpenClaw 浏览器逐项填写：
   - 姓名
   - 性别
   - 出生日期
   - 手机号
   - 邮箱
   - 证件类型 / 证件号
   - 所在城市
   - 住宅地址 / 通讯地址
   - 工作单位 / 工作城市 / 职位
   - 学历
   - 月收入 / 收入区间
5. 如果官网表单出现：
   - 滑块验证码
   - 短信验证码
   - 人脸识别
   - 电子签名
   - 授权征信查询
   - 最终“提交申请”按钮
   立即停止自动化并交给用户本人完成。
6. 如果页面报错：
   - 所在地区不可申请
   - 必填单位信息缺失
   - 证件类型不支持
   - 申请卡片已下线
   立即向用户说明，并建议改走网点 / 纸质申请 / 其他卡种。

## 推荐的操作口径

- “我已经帮你打开官方申请页，并预填了已确认的信息。”
- “请你本人核对身份证号、单位、收入和地址是否准确。”
- “验证码、授权勾选和最终提交请你自己完成。”

## profile 建议

- 默认使用 `openclaw` 托管 profile
- 只有用户明确要求复用当前浏览器会话时，才改用 `user` existing-session profile
