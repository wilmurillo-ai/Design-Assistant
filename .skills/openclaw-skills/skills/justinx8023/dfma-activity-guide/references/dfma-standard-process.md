# DFMA活动标准流程

DFMA（Design for Manufacturing and Assembly，面向制造与装配的设计）是一种优化产品设计的方法论，核心目标是降低制造成本、提升装配效率、减少生产风险。

## 前期准备阶段

### 核心目标
确保活动开展前所有必要信息和资源准备到位，避免讨论中断。

### 关键任务
1. **明确活动背景**：说明为什么开展本次DFMA评估，是新产品设计阶段评估还是现有产品降本优化
2. **确认评估范围**：明确本次评估覆盖的产品范围，是整产品还是某个子模块
3. **确定参与角色**：需要设计、工艺、制造、采购等多角色参与，确保覆盖不同维度
4. **提前发放资料**：将产品设计图纸、BOM、现有生产问题等资料提前分享给参会人员，让大家提前准备

## 过程推进阶段

### 1. 产品结构分解
- 按层级分解产品结构，从总成到子系统再到单个零件
- 整理所有零部件列表，明确每个零件的名称、材料、加工方式

### 2. 功能必要性分析
对每个零件进行提问：
1. 这个零件的功能是什么？
2. 是否可以取消这个零件，和其他零件合并？
3. 是否可以用更简单/低成本的工艺实现相同功能？

### 3. 可制造性风险识别
识别每个零件在制造过程中可能存在的风险：
- 加工难度：是否存在复杂曲面、高精度要求导致加工成本高
- 材料成本：是否使用了昂贵材料，是否可以替代
- 公差要求：公差是否过于严格，是否可以放宽不影响功能

### 4. 可装配性风险识别
识别装配过程中的问题：
- 是否需要特殊工具或装配技巧
- 装配顺序是否合理，是否存在无法装配或者装配效率低的问题
- 是否存在零件错误装配的风险

### 5. 改进方案提出
针对识别出的每个问题，提出对应的改进方案：
- 零件整合：合并多个零件减少装配工序
- 工艺简化：将复杂加工工艺改为简单工艺
- 材料替代：更换低成本材料
- 公差放宽：在满足功能的前提下放宽公差要求
- 结构优化：调整设计提升装配便利性

### 6. 方案评估排序
对提出的改进方案进行评估，从多个维度打分：
1. 降本效果：估算可以降低多少成本
2. 实施难度：评估方案实施的技术难度和开发周期
3. 风险程度：评估方案对产品功能质量的影响
4. 按收益/成本比排序，确定优先实施的方案

## 结果输出阶段

### 核心目标
整理活动成果，形成可执行的行动方案。

### 关键输出
1. 问题清单：识别出的所有DFMA问题汇总
2. 改进方案清单：按优先级排序的改进方案列表
3. 行动项：明确每个改进方案的责任人、计划完成时间
4. 活动报告：完整记录本次DFMA活动过程和结果
