# 会议纪要输出格式规范

## PDF格式规范

### 字体设置
- **主标题**：楷体，14pt，居中
- **章节标题**：楷体，12pt（左对齐）
- **正文**：楷体，10.5pt（五号）

### 行距与缩进
- **行距**：1.5倍行距
- **首行缩进**：2字符（仅正文段落）
- **段间距**：6pt

### 页面设置
- **纸张**：A4
- **页边距**：上下左右各25mm

### 标题层次
```
主标题（居中，加粗）：
[来源/类型] 关于 [核心议题] 的讨论纪要

章节标题（一、二、三 - 顶头）：
一、会议基本信息
二、讨论内容

次级标题（议题编号 - 左缩进10pt）：
议题1：主题名称

正文列表项（首行缩进）：
- 要点内容
- 要点内容

编号列表（顶头）：
1. 第一点
2. 第二点
```

## Word格式规范

### 字体设置
- **默认字体**：楷体，10.5pt
- **标题0级**：楷体，14pt，加粗，居中
- **标题1级**：楷体，12pt，加粗
- **标题2-3级**：楷体，10.5pt

### 段落格式
- **行距**：1.5倍
- **首行缩进**：2字符（正文）
- **对齐**：两端对齐

## 纪要结构模板

```
[来源/类型] 关于 [核心议题] 的讨论纪要

一、会议基本信息
1. 会议时间：YYYY年MM月DD日
2. 会议地点：XXX（如无写"未明确"）
3. 参与人员：列出核心发言方、机构及职务
4. 主持人：XXX（如无写"未明确"）

二、讨论内容

议题1：[主题名称]
 [要点1]
 [要点2]
 [要点3]

议题2：[主题名称]
 [要点1]
 [要点2]
 [要点3]

（依此类推）

===COMPANY_INFO===
公司名称1：行业类型1
===END_COMPANY_INFO===
```

## 禁止事项

1. ❌ 不使用Markdown符号（**、*、#、---、`）
2. ❌ 不生成"会议背景"描述
3. ❌ 不添加原文中没有的信息
4. ❌ 不省略实质性内容

## 文件命名规则

```
{来源机构}+{核心议题}+{日期}.{格式}

示例：
投研部+半导体行业讨论+20240315.pdf
专家咨询+AI发展趋势+20240320.docx
```
