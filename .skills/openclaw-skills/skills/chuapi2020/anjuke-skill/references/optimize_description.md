# 房源详情描述

## 如何使用
你需要具体针对页面的上“发布标题”、“房源详情”、“业主心态”、“服务介绍”这四个信息的内容进行优化。
## 执行步骤
请按以下步骤执行任务：
1.**获取该房源当前所有原始描述**
先把该房源当前的“发布标题”、“房源详情”、“业主心态”、“服务介绍”获取下来，作为优化描述的基准。

2.**优化发布标题**
怎么让房源标题抓人眼球？根据当前已有信息，对房源标题进行优化描述：
①突出房屋卖点描述：重点突出房屋地段、价格、周边配套特色、装修、房型、朝向、楼层、知名物业等中的几个方面。
②突出房屋活动优惠等优势：业主出国急售、赠送装修、业主送车位等等，这些房屋优惠的原因都是可以明确写出来的。
③标题应新鲜独特，筛选查看这个小区，其他人写的房源标题，自己写的时候要跟别人的不一样。
④标题公式：区域名称（片区名称/楼盘名称/地标建筑）+房源卖点（2-3个核心卖点）
售房标题示例：笔架山接近指导价（次新房业主置换）业主工作地点变动割爱，置换诚心卖
租房标题示例：鼎太风华稀有豪华装修大四房，复古精致一看就爱！

修改好之后，将修改好的结果回填回页面上的“房源标题”输入框

3.**优化房源详情**
怎么让房源详情抓人眼球？根据当前已有信息，对房源详情进行优化描述：
①描写方向：可以从户型、朝向、装修、产权、周边配套、物业、小区环境、交通、地段、升值空间等去写房源描述；
②突出房源卖点、优势，“王婆卖瓜，自卖自夸”，并且针对客户需求突出重点，比如看重学位的、看重交通方便的等等，要适时调整房源描述；
③添加卖房故事：比如自己扎根小区X年，要体现自己和业主或是房子的各种关系，尽量细节到位，如具体什么牌子的家私、房子新装修的详细经过等，展现自己对房源的熟悉度；
正确示例：
本人在此小区扎根5年了，成交了X套房源。这套房子是业主2015年买的，买的时候就是看中了这个房子XXXX（房源亮点），一家人住了5年，顺风顺水，现在XXX；（这种写法是经纪人了解到的业主对房子的看法感受）
房子是业主去年10月份的时候刚刚新装修了，当时花了很多心思，专门找人设计，厨房使用的都是方太的厨电，空调是格力的XXXXX；
上个月有个客户，想买来自己住的，看了这个房子之后很满意，朝向楼层XXXX，本来就要签合同了，因为XXX后面无，XXX。
错误示例：
房源特色，户型方正、采光通风非常好、视野开阔无遮挡、产权清晰无纠纷（这种写法，冷冰冰，感受不到真实，也无法吸引客户眼球）
④注意要点：描述注意分点、分段落，语句通顺，无错别字，让人觉得房源真实；
⑤其他推荐：同时推荐其他小区的房源（附近同品质同价格小区的房源），一句话写这个房源核心卖点即可（这样客户点击进来，就算不想问这个房源，也可能会多问一句你推荐的其他房源）
示例：推荐XX小区，XX地铁口，新出的精装修2房，带XXX学校

修改好之后，将修改好的结果回填回页面上的“房源详情”输入框

4.**优化业主心态**
怎么让业主心态抓人眼球？根据当前已有信息，对业主心态进行优化描述：
①原则：体现业主诚心卖、急卖；
②内容：业主卖房原因合理，不需要瞎编写的天花乱坠，根据自己之前卖房业主的接触，用这些业主的真实卖房原因即可；若是没有经验的新手，可以跟老经理了解，他们跟进的一些业主的卖房原因，时间、对象、事件，越详细越真实。
③常见的诚心并且比较急卖的卖房原因有：
家里有新增加的人口，目前的房子不够住，需要赶紧换房（业主已经定了大的房子急需腾名额、业主滕名额急售）；
家里的小孩什么时间要上学，上哪里的学校，需要把房子卖了，买好学校的房子；
业主4月份急需投资大项目，需要有现金周转，急售几处房产。
【正确示例】：
这套房源业主一手买过来，一直都是自己一家人住的，小孩现在要读书，业主要换福田的学位房，目前已经有看中的小区了，诚心出售！
昨天业主跟我谈了一下，业主很诚心卖，价格可以商量。
业主老婆怀孕6个月了，打算年后把老人也接过来住，需要换大一点的房子，诚心急售。
业主的小孩在关内上学，需要卖掉关外的这一套。
业主在国外，没有时间打理，诚心出售。
夫妻想和老人分开住，在附近再给老人买一套，方便照顾小孩，急售一套换两套。
【错误示例】：
业主换房，急售！（缺乏细节，不够真实）
业主心态不要跟其他人写的一模一样，客户看多了一看就是假的，不会想要进线。

修改好之后，将修改好的结果回填回页面上的“业主心态”输入框

5.**优化服务介绍**
怎么让服务介绍抓人眼球？根据当前已有信息，对服务介绍进行优化描述：
①介绍公司、团队，宣传公司品牌，好品牌公司同时也会更吸引客户的信任。
②介绍自己：工作从事多久，服务过多少客户，得到过什么荣誉，介绍自己的专业，沟通方式，等等，从为对方着想的角度着手。
正确示例：
【关于我】我是乐有家xxx（回填我的姓名），土生土长广东人，始终秉持“真诚正义，先人后己”的理念，帮您买到满意的房子就是我的使命！

修改好之后，将修改好的结果回填回页面上的“服务介绍”输入框
## 规则与约束
以下规则必须严格遵守：
1.**保证真实**
必须要严格基于现有的房源信息进行优化，可以适度浮夸，但不能随意胡编乱造不存在的信息
