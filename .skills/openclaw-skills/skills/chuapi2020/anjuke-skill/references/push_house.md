# 推送内网房源到安居客

## 如何使用
当需要从内网将房源推送到安居客时，请使用该技能，该技能可以将内网指定小区的最新房源推送到安居客。

## 执行步骤
请按以下步骤执行任务：
1.**进入经纪人参谋看板**
打开浏览器，访问该地址：https://ai.anjuke.com/broker/consult/delivery-advice
如果没登录，则使用子技能`login_your_account`进行登录。

2.**查看热门小区**
将时间范围切换至“近七日”，来到“周边TOP20热门小区”表格，将列表排序切换至“按小区点击量”降序

3.**复制列表中第一个小区名称**
将筛选结果中的第一个小区名称复制下来

4.**进入房源列表页**
访问该地址：https://i.leyoujia.com/lyj-menu/hsl/FYCX?showTab=true&submenu=FYCX_1#/housingList
如果没登录，则提示用户：请您先登录新系统
如果登录了，则将步骤3里复制的小区名称，粘贴到界面上的“楼盘名称”输入框内，勾选中对应房源并点击“搜索”按钮

5.**获取最新发布的房源**
顶部检索条件中，“租售”条件筛选为“可售”，排序条件筛选为“最新录盘的排前”，直接双点击列表页中的第一条房源进入房源详情页
