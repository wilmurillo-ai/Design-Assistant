# TOOLS.md - 本地备注（{{AGENT_NAME}} 客服）

Skill 定义"怎么用工具"；这个文件放 **你这套环境/业务专属** 的备忘。

## 适合写什么
- 客服常用口径（不适合放在 SOUL 的硬规则里，但经常要用）
- 运营活动/促销信息的更新点（以及更新时间）
- 转人工流程（联系人/群/时间段）
- 渠道相关的固定信息（客服群名称、注意事项等）

> 注意：不要把密钥、token、账号密码写进来。

## 示例

```markdown
### 促销
- 活动名称：活动内容（截至 YYYY-MM-DD）

### 转人工
- 复杂售后/投诉：引导用户留下订单号 + 联系方式，由人工跟进
```
