## 分析模型

### 漏斗分析

适用场景：用户从曝光到转化的全链路分析

```
曝光 → 点击 → 下载 → 激活 → 注册 → 实名 → 首次关键行为
```

分析要点：
- 计算每步转化率 = 本步人数 / 上步人数
- 标注行业基准或历史均值作为对比
- 找出转化率最低或降幅最大的环节（最大改善空间）
- 区分自然流量和付费流量分别分析

### 渠道对比分析

适用场景：评估各获客渠道的效果

分析维度：
| 维度 | 指标 | 说明 |
|------|------|------|
| 量 | 新客数、激活数 | 渠道规模 |
| 质 | 次日留存、7日留存、30日留存 | 用户质量 |
| 效 | 注册转化率、首次行为完成率 | 转化效率 |
| 本 | CPA、CAC | 获客成本 |
| 值 | 人均收入、LTV | 长期价值 |

分析方法：
- 制作渠道对比矩阵（量级 × 质量四象限图）
- 高量高质：核心渠道，维持投入
- 高量低质：优化投放策略或素材
- 低量高质：探索扩量空间
- 低量低质：考虑缩减或关停

### 趋势分析

适用场景：观察指标随时间的变化规律

分析要点：
- 选择合适的时间粒度（日/周/月）
- 环比 = (本期 - 上期) / 上期 × 100%
- 同比 = (本期 - 去年同期) / 去年同期 × 100%
- 标注关键事件节点（版本上线、大促、政策变化）
- 识别趋势：上升/下降/波动/平稳

### 对比分析

适用场景：横向比较不同组别的差异

常见对比维度：
- 时间对比：本周 vs 上周、本月 vs 上月
- 渠道对比：A渠道 vs B渠道
- 版本对比：新版 vs 旧版
- 人群对比：新客 vs 老客、不同城市等级

## 增长核心指标体系

### 获客指标
- **曝光量 / 点击量**：渠道触达规模
- **CTR（点击率）**：素材/位置吸引力
- **CVR（转化率）**：下载/注册转化效率
- **CPA / CAC**：单客获取成本
- **新客数**：新客绝对量

### 质量指标
- **次日留存 / 7日留存 / 30日留存**：用户粘性
- **首次关键行为完成率**：新客激活深度
- **有效新客占比**：达到质量标准的新客比例

### 效率指标
- **ROI**：投入产出比
- **LTV / CAC 比值**：获客经济性（>3为健康）
- **人均首日/首周行为次数**：参与深度

## 图表选择指南

| 分析目的 | 推荐图表 | Mermaid支持 |
|---------|---------|------------|
| 趋势变化 | 折线图 | xychart-beta |
| 数量对比 | 柱状图 | xychart-beta |
| 占比构成 | 饼图 | pie |
| 漏斗转化 | 漏斗图（用流程图模拟） | graph TD |
| 多维对比 | 表格 | Markdown表格 |
| 流程步骤 | 流程图 | flowchart |
| 时间规划 | 甘特图 | gantt |
| 用户旅程 | 旅程图 | journey |

Mermaid 图表使用提示：
- 折线图/柱状图用 `xychart-beta`
- 复杂数据展示优先用 Markdown 表格（更清晰）
- 漏斗图用从上到下的流程图模拟，在节点中标注转化率

## 异常归因框架

当数据出现异常波动时，按以下顺序排查：

1. **数据本身**：口径变更？统计bug？数据延迟？
2. **产品变更**：版本上线？功能调整？页面改版？
3. **运营动作**：活动上下线？投放调整？push策略变化？
4. **外部因素**：节假日？行业事件？竞品动作？政策变化？
5. **季节性**：是否符合历史同期规律？

归因建议：
- 先看整体再看细分（先总量，再按渠道/人群拆）
- 对比"变"和"不变"（哪些维度变了，哪些没变）
- 寻找时间吻合（异常时间点与排查因素的时间吻合度）
