# 文化设施建设招标文件信息抽取点

## 行业特有抽取点

### 设施类型
- 设施类型：博物馆、美术馆、图书馆、文化中心、剧院等
- 功能定位：文化展示、艺术表演、文化教育、公众服务
- 建筑规模：建筑面积、功能分区、观众容量
- 建设内容：主体建筑、配套设施、设备采购

### 建筑设计
- 建筑风格：建筑风格、文化元素融入
- 功能布局：功能分区、空间设计、流线组织
- 展陈设计：展览设计、展陈方式、展陈技术
- 舞台设计：舞台设计、观众席、后台设施

### 设备配置
- 展陈设备：展柜、展台、照明、温湿度控制
- 演艺设备：音响设备、灯光设备、舞台机械
- 信息化设备：票务系统、监控系统、导览系统
- 空调设备：空调系统、新风系统、节能措施

### 运营服务
- 开放时间：开放时间、节假日安排
- 服务内容：展览、演出、讲座、培训
- 服务人员：服务人员数量、资质要求
- 志愿服务：志愿者服务、志愿活动
