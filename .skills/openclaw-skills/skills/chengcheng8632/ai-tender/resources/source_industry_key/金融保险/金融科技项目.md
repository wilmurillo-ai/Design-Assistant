# 金融科技项目招标文件信息抽取点

## 行业特有抽取点

### 项目类型
- 项目类型：核心业务系统升级、数字银行建设、保险科技平台、金融监管科技等
- 技术架构：系统架构、技术栈、云服务、微服务
- 业务范围：业务功能、业务流程、业务场景
- 项目规模：系统规模、用户规模、交易规模

### 系统架构
- 技术架构：前后端分离、微服务架构、分布式架构
- 数据架构：数据架构、数据模型、数据治理
- 安全架构：信息安全、网络安全、数据安全
- 集成架构：系统集成、API接口、数据交换

### 数据治理
- 数据质量：数据质量管理、数据标准化、数据清洗
- 数据安全：数据加密、数据脱敏、数据访问控制
- 数据分析：大数据分析、风险建模、客户画像
- 数据合规：数据合规、数据隐私、GDPR

### 合规科技
- 反洗钱：反洗钱系统、可疑交易监测、KYC
- 监管报告：监管报告生成、监管数据报送
- 风险控制：风险预警、风险监控、风险处置
- 审计追踪：操作日志、审计追踪、合规审计
