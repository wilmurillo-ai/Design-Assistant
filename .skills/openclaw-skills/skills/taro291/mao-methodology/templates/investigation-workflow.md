# 调查研究工作流程

## 适用场景
- 需要分析复杂问题时
- 需要做出重要决策前
- 需要了解真实情况时

## 流程步骤

### 第一步：明确调查目的
```
提问模板：
- 我们要解决什么问题？
- 目前对这个问题了解多少？
- 还有哪些方面不了解？
- 调查的预期结果是什么？
```

### 第二步：收集材料
```
材料清单：
1. 第一手材料（亲自观察、访谈）
   - 现场观察记录
   - 与相关人员的访谈
   - 实际操作体验

2. 第二手材料（文档、报告）
   - 相关文档和报告
   - 统计数据
   - 他人经验总结

3. 材料质量检查
   - 是否丰富（不零碎）
   - 是否真实（无错觉）
   - 是否相关（有针对性）
```

### 第三步：分析加工
```
分析方法：
1. 去粗取精 - 留下关键信息，去掉无关细节
2. 去伪存真 - 验证信息真伪，排除虚假
3. 由此及彼 - 联系各方面，找出关联
4. 由表及里 - 从现象深入本质
```

### 第四步：得出结论
```
结论要素：
- 主要问题是什么
- 问题产生的原因
- 解决问题的方案
- 方案实施的路径
```

## 调查技术清单

| 技术 | 适用情况 | 操作要点 |
|-----|---------|---------|
| 调查会 | 需要多角度信息 | 3-8人，讨论式，自己记录 |
| 现场观察 | 需要看实际情况 | 亲临一线，详细记录 |
| 个别访谈 | 需要深入了解 | 不耻下问，耐心倾听 |
| 文档分析 | 需要历史背景 | 交叉验证，注意时效 |
