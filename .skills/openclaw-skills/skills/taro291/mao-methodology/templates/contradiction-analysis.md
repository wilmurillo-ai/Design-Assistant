# 矛盾分析工作流程

## 适用场景
- 需要分析复杂问题时
- 需要确定优先级时
- 需要制定策略时

## 分析框架

### 第一步：列出所有矛盾
```
矛盾清单模板：

问题：______________________

存在的矛盾：
1. ______________ vs ______________（矛盾双方）
   - 具体表现：____________________
   - 存在条件：____________________

2. ______________ vs ______________
   - 具体表现：____________________
   - 存在条件：____________________

3. ______________ vs ______________
   - 具体表现：____________________
   - 存在条件：____________________
```

### 第二步：分析矛盾特殊性
```
对每个矛盾分析：
- 时间因素：这个矛盾在什么时期更突出？
- 地点因素：这个矛盾在什么环境下更明显？
- 条件因素：什么条件会加剧或缓解这个矛盾？
- 发展趋势：这个矛盾会怎么演变？
```

### 第三步：找主要矛盾
```
判断标准：
- 哪个矛盾规定或影响其他矛盾的存在和发展？
- 哪个矛盾解决后，其他矛盾会随之缓解？
- 哪个矛盾最紧迫、影响最大？

主要矛盾：______________________
理由：______________________
```

### 第四步：分析矛盾主要方面
```
判断标准：
- 矛盾双方中，哪一方处于主导地位？
- 事物的性质主要由哪一方决定？

矛盾主要方面：______________________
当前事物性质：______________________
```

### 第五步：预判矛盾转化
```
转化分析：
- 在什么条件下，主要方面会转化为次要方面？
- 事物性质会如何变化？
- 需要做什么来促成转化？
```

## 决策输出

```
基于矛盾分析的决策：

矛盾主要方面：______________________
解决策略：______________________
重点行动：______________________
预期转化：______________________
```
