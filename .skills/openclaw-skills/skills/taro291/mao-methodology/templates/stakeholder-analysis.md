# 利益相关者分析模板

## 核心问题
"谁是我们的敌人？谁是我们的朋友？" —— 这是解决问题的首要问题

## 分析步骤

### 第一步：识别利益相关者
```
利益相关者清单：

相关方名称 | 利益关系 | 影响程度 | 影响方式
----------|---------|---------|----------
          |         |         |
          |         |         |
          |         |         |
          |         |         |
```

### 第二步：分析各方立场
```
立场分类：

【支持方】（利益基本一致）
- ______________
- 支持程度：强/中/弱
- 可能的贡献：______________

【反对方】（利益根本对立）
- ______________
- 反对程度：强/中/弱
- 可能的阻碍方式：______________

【中间方】（利益部分一致，可能动摇）
- ______________
- 倾向：偏向支持/偏向反对/中立观望
- 可能争取的条件：______________
```

### 第三步：判断力量对比
```
力量分析：

| 类型 | 数量/规模 | 影响力 | 资源 |
|-----|----------|--------|-----|
| 支持方 | | | |
| 反对方 | | | |
| 中间方 | | | |

总体判断：
- 支持方力量：强于/等于/弱于 反对方
- 中间方的去向决定结果：是/否
```

### 第四步：制定策略
```
策略矩阵：

| 对象 | 策略 | 具体行动 |
|-----|------|---------|
| 支持方 | 团结巩固 | 发挥其作用 |
| 反对方 | 应对防范 | 减少其影响 |
| 中间方 | 积极争取 | 创造转化条件 |

关键策略：
- 主要依靠：______________
- 主要防范：______________
- 重点争取：______________
```

## 注意事项

1. **中间力量会分化** - 形势变化时，中间力量会向两边分化，要预判走向
2. **不要忽视潜在同盟** - 有些看似无关的力量可能成为关键
3. **力量对比会变化** - 要动态分析，不能固守既有判断
4. **策略要具体** - 不同对象要有不同策略，不能一刀切
