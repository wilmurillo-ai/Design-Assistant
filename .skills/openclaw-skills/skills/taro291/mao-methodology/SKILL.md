# 毛泽东方法论 - 调查研究与实践智慧

从《毛泽东选集第一卷》提炼的核心方法论，适用于日常问题分析、决策和执行工作。

## 核心原则

### 1. 调查研究法（反对本本主义）

**核心观点**："没有调查，没有发言权"

**操作步骤**：
1. **明确调查目的** - 调查是为了解决问题，不是为调查而调查
2. **深入实际场景** - 到一线去，亲自接触事物本身，不道听途说
3. **收集真实材料** - 材料要丰富（不是零碎不全）且合于实际（不是错觉）
4. **分析加工** - 去粗取精、去伪存真、由此及彼、由表及里
5. **得出结论** - 基于分析形成判断和决策

**调查技术要点**：
- 开调查会：召集了解情况的人，人数3-8人适宜
- 自己记录：不要让别人代劳，亲手记录印象更深
- 讨论式调查：不只是提问，要启发讨论
- 不耻下问：不懂就问，不要装懂

**避免的错误**：
- 本本主义：照搬书本/上级指示，不看实际情况
- 经验主义：只凭个人片断经验，不看全局规律

### 2. 实践论方法论

**核心观点**：实践是认识的来源和检验真理的标准

**认识过程**：
```
感性认识 → 理性认识 → 实践检验 → 新的认识
   ↑                                    ↓
   └←←←←←←←←← 反复循环 ←←←←←←←←←←←←←←←←
```

**关键步骤**：
1. **接触实际** - 先在实践中接触事物，获得感性认识（现象、片面、外部联系）
2. **思考加工** - 用概念、判断、推理，把握本质、全体、内部联系
3. **实践验证** - 把认识付诸实践，检验是否正确
4. **修正提高** - 失败则吸取教训，修正认识，再实践

**要点**：
- "你要知道梨子的滋味，你就得变革梨子，亲口吃一吃"
- "不入虎穴，焉得虎子"
- 失败是成功之母，吃一堑长一智

### 3. 矛盾分析法

**核心观点**：事物的矛盾法则（对立统一）是唯物辩证法的核心

**分析框架**：

| 分析维度 | 方法 | 应用场景 |
|---------|------|---------|
| 矛盾普遍性 | 找出一切矛盾的存在 | 发现问题、列出矛盾清单 |
| 矛盾特殊性 | 对具体事物作具体分析 | 分析特定情境下的问题特点 |
| 主要矛盾 | 找出起决定作用的矛盾 | 确定优先级、抓主要问题 |
| 矛盾主要方面 | 找出矛盾中主导的一方 | 判断事物性质、决定策略 |

**操作要点**：
1. 列出所有存在的矛盾
2. 分析每个矛盾的特殊性（时间、地点、条件）
3. 区分主要矛盾和次要矛盾
4. 抓住主要矛盾的主要方面
5. "捉住了主要矛盾，一切问题就迎刃而解了"

**矛盾的转化**：
- 矛盾双方在一定条件下会互相转化
- 要注意条件的变化，预判转化的可能性

### 4. 利益相关者分析法（阶级分析法的现代应用）

**核心观点**："谁是我们的敌人？谁是我们的朋友？这个问题是革命的首要问题"

**分析步骤**：
1. **列出所有相关方** - 全面识别涉及的利益主体
2. **分析各方的立场** - 根据利益关系判断态度
   - 根本利益对立者 → 反对/阻碍
   - 利益基本一致者 → 支持/同盟
   - 利益部分一致者 → 可能动摇/需要争取
3. **判断力量对比** - 各方的影响力和数量
4. **制定策略** - 团结真正的朋友，攻击真正的敌人

**注意事项**：
- 中间力量会在形势变化时分化，要预判走向
- 不要忽视潜在的同盟力量

### 5. 战略思维方法

**核心观点**：战略问题是研究全局规律的问题

**关键原则**：

| 战略原则 | 含义 | 工作应用 |
|---------|------|---------|
| 战略是研究全局 | 把握整体规律 | 项目规划要看全局 |
| 集中兵力 | 把有限力量集中于关键点 | 抓主要任务，不分散精力 |
| 运动战 | 在运动中寻找机会 | 动态调整策略，不固守 |
| 速决战 | 快速解决战斗 | 高效执行，避免拖延 |
| 歼灭战 | 彻底解决问题 | 不留尾巴，一次做彻底 |

**学习的方法**：
- "重要的问题在善于学习"
- 从失败中学习：败仗打得多的更能理解战争规律
- 理论结合实践：不能光看书本，要亲自实践

### 6. 群众工作法

**核心观点**："关心群众生活，注意工作方法"

**两个关键点**：
1. **关心群众生活** - 解决实际问题，不只是喊口号
2. **注意工作方法** - 方法不对，任务也完不成

**操作要点**：
- 从群众的需要出发，不是从自己的想法出发
- 群众的问题要具体解决（柴米油盐、衣食住行）
- 工作方法要讲究：任务与方法并重
- "任务本来是正确的，但由于工作方法错误，任务也完不成"

## 实践应用框架

### 问题分析流程

```
1. 调查研究 → 收集真实情况
2. 矩盾分析 → 列出矛盾、找主要矛盾
3. 利益分析 → 识别相关方立场
4. 战略判断 → 确定全局方针
5. 实践验证 → 执行并检验
6. 反复迭代 → 修正认识
```

### 常见错误警示

| 错误类型 | 表现 | 危害 |
|---------|------|------|
| 本本主义 | 照搬书本/上级指示 | 脱离实际，决策失误 |
| 经验主义 | 只凭个人片面经验 | 看不清全局，盲目行事 |
| 主观主义 | 不调查就主观判断 | 领导工作必然失败 |
| 平均主义 | 不区分主要次要矛盾 | 找不到重点，效率低下 |
