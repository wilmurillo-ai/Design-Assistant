# Severity Matrix (Code/Service)

## P0
- Full outage, data loss risk, security exploit in progress
- Action: immediate containment + hotfix lane, comms every 15 min

## P1
- Major feature down for many users, high revenue impact
- Action: hotfix same day, comms every 30-60 min

## P2
- Partial degradation, workaround exists
- Action: prioritize next deploy window

## P3
- Low-impact defect
- Action: normal backlog
