⚡ 怎么安装和启动

安装方式：
npx clawhub install openclaw-newbie-faq

启动方式：
安装这个 Skill 之后和 OpenClaw 对话说"启动新手帮助"，就会自动启动这个 Skill，你就可以通过 http://localhost:34567 来访问了。
