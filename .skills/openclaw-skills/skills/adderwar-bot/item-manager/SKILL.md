# 在哪儿（item-manager）
极简实用物品管理 Skill，只需正常说话即可记录、查询物品位置。

## 核心功能
- 自动识别 **物品 在 位置** 句式，无需前缀指令
- 支持食品、药品保质期记录与过期提醒
- 自动分类：药品、食品、电子产品、工具、随身物品、其他
- LLM 智能校验物品名称，**不自动更正**
- 名称疑似错误：询问用户是否更正
- 无法判断：保留原名并标注疑似名称
- 支持查询、分类查看、快过期物品提醒

## 使用示例
### 记录
芬必得在客厅药箱，保质期2026-09-10
牛奶放在冰箱
钥匙放玄关鞋柜
粉笔得在抽屉

### 查询
芬必得在哪？
牛奶在哪里？

### 管理
列出药品
列出食品
