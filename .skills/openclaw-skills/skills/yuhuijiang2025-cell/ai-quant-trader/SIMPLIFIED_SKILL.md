# AI量化交易助手 - 简化版

## 概述
这是一个基于文本的AI量化交易助手，通过OpenClaw的对话功能实现所有交易分析功能。

## 核心功能（通过对话实现）

### 1. 📈 选股分析
```
用户: "帮我筛选沪深A股，非ST，非创业，非科创，非北交所"
AI: "好的，正在筛选..."
AI: "筛选条件: ROE>-20%且<15%，流动比率≥0.4，市值<100亿..."
AI: "推荐股票: 1. 600519 贵州茅台 2. 000858 五粮液..."
```

### 2. 🤖 策略生成
```
用户: "生成一个MACD金叉短线策略"
AI: "策略生成完成: 当MACD金叉时买入，死叉时卖出..."
```

### 3. 📊 技术分析
```
用户: "分析一下贵州茅台"
AI: "技术分析报告: MA多头排列，RSI 58，MACD金叉..."
```

## 使用方法

### 直接对话
```
"小火马，帮我筛选今天值得关注的股票"
"分析一下600519的技术面"
"生成一个价值投资策略"
```

### 预设命令
```
/选股筛选 [条件]
/技术分析 [股票代码]
/策略生成 [描述]
/风险评估 [组合]
```

## 筛选条件示例

### 基本面筛选
- ROE: -20% ~ 15%
- 流动比率: ≥0.4
- 市值: <100亿元

### 技术面筛选
- 2日涨跌幅: <5%
- 当日涨幅: <3%
- 振幅: <6%
- 量比: >0.8
- 换手率: 1%~10%

### 排序方式
- 周成交量环比增长（从大到小）

## 投资建议流程

### 第一步：选股
```
"筛选沪深A股，非ST，市值<100亿，换手率1-10%"
```

### 第二步：分析
```
"对筛选结果前5名进行技术分析"
```

### 第三步：策略
```
"为贵州茅台生成一个短线交易策略"
```

### 第四步：风控
```
"设置止损5%，止盈10%"
```
