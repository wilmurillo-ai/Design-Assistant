# scoring.md – How to score each job (0–100)

Start from 50 points, then add or subtract.

## Add points
- +20 if title contains exact target role ("Full Stack Developer")
- +15 if title contains any skill from your profile (e.g., "React", "Node.js")
- +10 if location matches preference (Casablanca, Rabat, remote)
- +10 if contract is CDI or Freelance
- +5 for each additional skill match (max +25)
- +5 if company is well-known or rated 4+ stars
- +5 if posted within last 3 days

## Subtract points
- -20 if any exclusion keyword appears (e.g., "stage", "internship")
- -15 if location is outside preferred cities and not remote
- -10 if contract is "CDD" or "temporary"
- -5 if posted 5–7 days ago
- -30 if salary mentioned is below expectation (optional)

## Scoring levels
- 90–100: Apply immediately
- 70–89: Strong candidate – apply this week
- 50–69: Apply if nothing better
- <50: Skip
