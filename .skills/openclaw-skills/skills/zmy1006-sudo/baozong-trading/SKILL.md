## 能力一：思维决策层

### 1.1 巴菲特安全边际判断

**触发条件**：评估个股估值、决定买入时机、判断是否高估

**判断流程**：
1. 计算内在价值（ROE*合理PE法 或 现金流折现）
2. 计算安全边际 = (内在价值 - 当前价格) / 内在价值 × 100%
3. 安全边际 ≥ 30% → 可考虑建仓；20-30% → 等待；< 20% → 不符合
4. 检查护城河：品牌/专利/网络效应/成本优势/转换成本
5. 管理层诚信度评估

**输出模板**：
```
【巴菲特安全边际分析】{股票名称} {代码}
━━━━━━━━━━━━━━━━━━
内在价值估算：¥{XX元}
当前价格：¥{XX元}
安全边际：{XX%}
护城河评级：{强/中/弱}
综合判断：{强烈买入/买入/观望/卖出}
⚠️ {风险提示}
```

### 1.2 芒格多元思维核查清单

**触发条件**：重大投资决策前，必须强制逐项输出

```
□ 数学层：期望值 = 概率 × 赔率，计算了吗？
□ 工程层：物理/供应链是否可实现？
□ 生物层：竞争格局，是否有自然选择压力？
□ 经济层：护城河来源，可持续性？
□ 心理层：我的判断是否受情绪偏差影响？（确认偏误/损失厌恶？）
□ 黑天鹅：最坏情景是什么？能承受吗？
□ 逆向：如果我是空头，如何攻击这只股票？
```

### 1.3 段永平"本分"商业逻辑

**触发条件**：评估商业模式的可持续性

判断标准：
- ✓ 本分：不坑消费者、不做假账、不走监管擦边
- ✓ 利润来源：靠产品竞争力，不是靠政策或垄断
- ✓ 现金流：经营现金流持续为正
- ✓ 管理层：不吹牛、不蹭热点、专注主业
- ✓ 用户口碑：产品是否真的值得买

## 能力二：市场分析层

### 2.1 盘前机会扫描（每日 8:30 前执行）

**触发条件**："今日机会"、"盘前分析"、"开盘前看看"

扫描维度：
1. 隔夜美股：纳斯达克/标普500/道指涨跌幅，关联中概股ADR
2. A50期货：A股开盘情绪（高开/低开/平开）
3. 港股ADR：腾讯/阿里/美团ADR表现
4. 恐慌贪婪：A股情绪指数参考
5. 板块异动：昨夜美股热门板块映射
6. 重要新闻：政策面/宏观面重大消息
7. 外资动向：北向资金历史趋势

**输出模板**：
```
【{日期} 盘前机会扫描】
━━━━━━━━━━━━━━━━━━
外盘情绪：{乐观/中性/谨慎}
A50开盘预期：{高开(+X%)/低开(-X%)/平开}
今日关注板块：{1... 2... 3...}
重点关注个股：{代码} {理由}
今日操作建议：{进攻/防守/观望}
⚠️ 风险提示：{若有重大风险事件}
```

### 2.2 个股深度分析（三维交叉验证）

**触发条件**：用户提供股票代码/名称，"分析{股票}"

**基本面（价值锚）**：
- ROE、毛利率、净利率（3年趋势）
- 营收增速、净利润增速
- 资产负债率，经营现金流/净利润比
- PE/PB/PS 当前值 vs 历史分位数
- 股息率是否可持续
- 主流机构综合评级

**技术面（择时参考）**：
- MA5/MA20/MA60 多头排列 or 空头排列
- RSI(14) 是否超买>70 or 超卖<30
- 近30日高低点，关键支撑压力位
- 成交量：量价配合是否健康
- MACD：是否出现金叉/死叉

**资金面（驱动力）**：
- 北向资金近5日净流入/流出
- 主力资金超大单/大单净流入
- 是否近期登龙虎榜
- 融资余额变化趋势

**三维综合结论**：
```
基本面（价值）：{低估/合理/高估}
技术面（择时）：{强势/中性/弱势}
资金面（驱动）：{净流入/净流出}
综合评级：{★★★★★}
建议操作：{买入区间/止损价/目标价}
```

### 2.3 政策影响评估

**触发条件**：提到政策、监管、文件、会议

评估流程：
1. **来源定级**：最高层(★★★) → 全市场影响；部委级(★★) → 板块影响；地方(★) → 局部
2. **方向判断**：利好板块 / 利空板块 / 中性（规范发展）
3. **传导链**：政策 → 行业 → 龙头公司 → 估值 → 股价
4. **历史类比**：同类政策出台后3个月内市场/板块表现

## 能力三：执行操作层

### 3.1 每日复盘（每日 16:00 后执行）

**触发条件**："复盘"、"今天怎么样"、"收盘了"

复盘框架：
1. **账户状况**：今日盈亏，持仓涨跌，当日操作记录
2. **持仓诊断**：基本面变化？触发止损？量价健康？
3. **机会回顾**：有无操作机会被错过？
4. **市场总结**：最强板块/最弱板块/情绪指标（涨停跌停数）
5. **明日计划**：关注板块和个股 + 操作预案（高开/低开预案）

### 3.2 策略回测框架

**触发条件**：用户提出交易策略（选股条件/止损逻辑/仓位规则）

回测流程：
1. **策略编码**：将定性描述转化为量化条件
2. **参数定义**：回测区间（默认近3年）、手续费（默认0.03%）、滑点（默认0.1%）
3. **核心指标**：总收益率/年化收益率/最大回撤/夏普比率/胜率/盈亏比
4. **敏感性分析**：改变止损幅度/持仓周期后，指标如何变化
5. **结论**：策略有效？主要风险点？需优化什么？

### 3.3 自选股监控与异动预警

**触发条件**："监控{股票}"、"设置提醒"、"盯盘"

监控指标（用户可自定义阈值）：
- 价格异动：单日涨跌超过 ±{X}%
- 成交量异动：较20日均量放大{X}倍
- 技术破位：跌破MA60 / 跌破前期低点
- 资金异动：北向单日净流入超{XX}亿
- 估值突破：PE达到历史{X}%分位数
- 止损提醒：价格触及预设止损位

**预警推送格式**：
```
🔔 【{股票}{代码} 异动提醒】
━━━━━━━━━━━━━━━━━━
异动类型：{类型}
当前价格：¥{XX}（今日{X%}）
触发条件：{原设置的监控条件}
建议操作：{根据芒格清单重新评估}
⚠️ 风险提示
时间：{触发时间}
```

## 能力四：风控层

### 4.1 账户健康检查

**触发条件**："风控检查"、"账户怎么样"、"看看风险"

检查维度：
- 总仓位 {XX%} → >80% 为过重，建议减仓
- 单股仓位 {XX%} → >20% 过于集中
- 前3大持仓占总仓位 → 超过50%为高风险
- 单一板块仓位是否超过30%？
- 小盘股占比是否过高？（大熊市可能无法卖出）
- 是否有持仓已触发止损但未执行？
- 持仓股之间是否存在高度相关性？

### 4.2 止损决策辅助

**触发条件**：持仓股票下跌，用户犹豫是否止损

决策流程（强制执行）：
```
Step 1: 初始止损 vs 当前亏损
   → 当前亏损 > 初始止损 → 触发止损，强制执行

Step 2: 基本面是否变化？
   → 基本面变差：立即止损
   → 基本面不变：考虑暂时持有观察

Step 3: 芒格清单复查
   → 有无重大误判？ → 是：止损并反省

Step 4: 仓位决定
   → 止损后仓位可承受 → 执行止损
   → 止损后空仓 → 等待更好机会
```

## 注意事项

### 风险免责
- 本Skill分析仅供参考，不构成投资建议
- 股市有风险，投资需谨慎，盈亏自负
- 止损纪律是生存底线，必须严格执行
- 回测结果不代表未来表现

### 能力边界
- 本Skill不提供实时行情（需接入Tushare/AKShare数据API）
- 不提供荐股服务，仅提供分析框架和决策辅助
- 美股/港股分析能力相对A股弱（数据可得性限制）
- 技术面分析依赖数据完整性，小盘股数据可能不完整

### 数据来源
1. Tushare Pro API（A股，需注册）
2. AKShare（免费备用）
3. 公开研报和网络搜索（辅助验证）
4. 财联社/雪球实时新闻（消息面）
