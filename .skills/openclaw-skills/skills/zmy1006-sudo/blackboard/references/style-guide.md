# 黑板海报风格指南

## 风格定义

**核心关键词**：深色海报、竖版封面、大字居中、高对比配色、极简设计

## 背景材质

| 类型 | 提示词关键词 | 适用场景 |
|------|------------|---------|
| 纯黑 | solid black background | 极致对比 |
| 深炭灰 | dark charcoal grey #1a1a1a | 主流推荐 |
| 深蓝灰 | dark slate blue #2C3E50 | 科技感 |
| 深墨绿 | dark forest green #1B3A2D | 自然/健康 |
| 深棕 | dark brown #2D1B0E | 复古/手作 |

## 边框

| 类型 | 提示词 |
|------|------|
| 细白边 | thin white border around edges |
| 无边 | frameless, no border |
| 双层边 | double border, white outer + dark inner |

## 悬挂方式

| 方式 | 提示词 |
|------|------|
| 长尾夹 | held by small black binder clips at top |
| 透明胶带 | taped to wall with small pieces of tape at top corners |
| 绳子挂 | hung by thin string from two nails |
| 无挂件 | flat design, no hanging elements |

## 字体风格

```
bold sans-serif Chinese text        # 无衬线粗体
handwritten style Chinese text     # 手写风格
modern geometric font              # 现代几何字体
elegant serif Chinese text         # 衬线优雅字体
```

## 常见错误避免

- ❌ 不要写 "text with shadow" → 中文AI生成会产生乱码
- ❌ 不要写中文在提示词里 → 用英文描述图标代替
- ✅ 用颜色关键词代替具体文字描述
- ✅ 先用白板skill生成布局，再用黑板skill做最终版

## 效果对比

| 特征 | 白板风格 | 黑板风格 |
|------|---------|---------|
| 背景 | 白 | 深灰/黑 |
| 文字 | 手写马克笔风 | 现代印刷风 |
| 格式 | 横版16:9 | 竖版9:16 |
| 氛围 | 办公/敏捷 | 极简/设计感 |
| 适合 | 流程图、复盘 | 封面、slogan |
