# MCP Registry Submissions

Most registries just need the GitHub URL and pull metadata from the repo automatically. The copy below is for any that ask for a description.

## Reusable description

> Superhuman exploratory data analysis. Disco automatically finds the feature interactions and subgroup effects in tabular data that LLMs and manual exploration miss — with p-values, effect sizes, and literature citations. Data goes in, validated insights come out. Already used to make novel scientific discoveries in plant biology, immunology, and meteorology. Free for public data.
