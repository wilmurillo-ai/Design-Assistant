- 🤖 **AIプロダクト** — エージェント、LLMインフラ、評価
- 📈 **グロース＆PLG** — アクティベーション、リテンション、バイラルループ
- 💰 **プライシング** — バリューメトリクス、AI料金体系、フリーミアム vs トライアル
- 🚀 **GTM＆セールス** — エンタープライズ営業、Go-to-Market
- 🏗️ **プロダクト戦略** — フレームワーク、ロードマップ、PMF
- 👥 **リーダーシップ** — 採用、チームビルディング
