Вы провели месяцы (или годы) с ChatGPT. Он знает ваш стиль письма, проекты, предпочтения. При переходе на OpenClaw ничего из этого не должно начинаться с нуля.

**Memory Transfer** извлекает всё, что ваш старый ИИ знает о вас, очищает данные и импортирует в систему памяти OpenClaw.

## Использование

Скажите вашему агенту OpenClaw:

```
Я перешёл с ChatGPT
```

## Поддерживаемые источники

| Источник | Метод | Ваши действия |
|----------|-------|---------------|
| **ChatGPT** | ZIP-экспорт данных | Нажмите «Экспорт» в настройках, загрузите ZIP |
| **Claude.ai** | Управляемый промпт | Скопируйте промпт, вставьте результат |
| **Gemini** | Управляемый промпт | Скопируйте промпт, вставьте результат |
| **Copilot** | Управляемый промпт | Скопируйте промпт, вставьте результат |
| **Claude Code** | Автосканирование | Ничего — автоматически |
| **Cursor** | Автосканирование | Ничего — автоматически |
| **Windsurf** | Автосканирование | Ничего — автоматически |

## Что переносится

| Категория | Назначение | Примеры |
|-----------|-----------|---------|
| Личность | `USER.md` | Имя, профессия, язык, часовой пояс |
| Стиль общения | `USER.md` | Тон письма, предпочтения форматирования |
| Знания | `MEMORY.md` | Проекты, экспертиза, инсайты |
| Поведенческие паттерны | `MEMORY.md` | Рабочие процессы, привычки, коррекции |
| Инструменты | `TOOLS.md` | Технологический стек, платформы |

## Установка

```bash
clawhub install openclaw-memory-transfer
```
