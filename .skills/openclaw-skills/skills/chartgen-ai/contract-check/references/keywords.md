# 合同审核关键词速查表（通用模板）

> 审核合同时，用以下关键词在合同文本中搜索，快速定位潜在风险条款。
> 数值类关键词需结合配置文件中的具体数值判断。

## 🚫 否决级关键词（出现即需重点审查）

| 关键词 | 风险 | 对应条款 | 处理方式 |
|--------|------|----------|----------|
| 永久保密 / 无期限保密 | 保密期无上限 | R-018 | 修改为配置的保密期限上限 |
| 共同拥有 / 共同享有知识产权 | 侵犯己方IP | R-004 | 限定为配置允许的范围内 |
| 全部源代码 / 源码交付 | 源码泄露 | R-002 | 按配置的源码策略处理 |
| 连带赔偿 / 无限连带 | 赔偿无上限 | R-006 | 修改为直接损失+责任上限 |
| 利润损失 / 间接损失 | 超范围赔偿 | R-006 | 按配置排除，限定为直接损失 |
| 惩罚性损害 | 超范围赔偿 | R-006 | 删除 |
| 赔偿不封顶 / 无上限赔偿 | 责任无上限 | R-005 | 设置为配置的责任上限 |
| 免费升级 | 过度承诺 | R-015 | 按配置决定是否允许 |
| 付款.*超过.{max_payment_cycle_days}日 | 回款风险 | R-011 | 最长为配置的天数 |
| 数据安全.*无限连带 | 数据责任无上限 | R-019 | 设置为配置的上限 |

> 注：`{max_payment_cycle_days}` 等占位符需替换为配置文件中的实际值。

## ✅ 必备条款确认清单

| 条款 | 必须包含 | 配置字段 |
|------|----------|----------|
| 违约责任上限 | 赔偿总额 ≤ 配置比例 × 实收费用 | `liability.cap_pct_of_actual_receipt` |
| 赔偿范围限定 | 排除配置的排除项 | `liability.excluded_losses` |
| 首款 ≥ 配置比例% | 签署后配置期限内 | `payment.first_payment_min_pct` |
| 付款周期 ≤ 配置天数 | 超期须提前报备 | `payment.max_payment_cycle_days` |
| 验收环节+支付节点 | 验收标准附件化 | — |
| 验收逾期违约金 | 每天配置‰率，上限配置% | `service_rights.acceptance_penalty_*` |
| 暂停服务权 | 逾期配置天数可停服 | `service_rights.suspend_after_days` |
| 送达条款 | 含电子送达 | — |
| 订阅自动终止 | 按配置处理 | `product_service.subscription_auto_terminate` |
| 买断维保限制 | 配置年数免费，续保由后续合同约定 | `product_service.warranty_period_years` |
