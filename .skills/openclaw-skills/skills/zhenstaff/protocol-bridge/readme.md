# Protocol Bridge / 协议桥

**The TCP/IP of Agent Standards** / **AI Agent 标准的 TCP/IP**

---

## English

### What is Protocol Bridge?

Protocol Bridge is a universal protocol translation layer that enables AI agents to communicate seamlessly across different protocols and frameworks.

### Key Features

- Multi-Protocol Support (MCP, A2A, ACP)
- Real-Time Translation
- Intelligent Routing
- Agent Discovery
- Security Layer
- Monitoring

### Quick Start

```bash
npm install -g openclaw-protocol-bridge
protocol-bridge init
protocol-bridge serve --port 8080
```

## 中文

### Protocol Bridge 是什么？

Protocol Bridge（协议桥）是一个通用协议转换层，使 AI Agent 能够跨不同协议和框架无缝通信。

### 核心功能

- 多协议支持（MCP、A2A、ACP）
- 实时转换
- 智能路由
- Agent 发现
- 安全层
- 监控

### 快速开始

```bash
npm install -g openclaw-protocol-bridge
protocol-bridge init
```
