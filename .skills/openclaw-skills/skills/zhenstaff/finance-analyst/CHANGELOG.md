## [0.1.0] - 2026-03-05

### Added
- Initial release of FA Advisor skill (Python implementation)
- Core modules:
  - Project assessment engine
  - Pitch deck & business plan generator
  - Valuation engine with multiple methods:
    - Scorecard method
    - Berkus method
    - Risk factor summation method
    - Comparable company method
  - Investor matching algorithm
  - Investment analysis & memo generator
- Comprehensive Pydantic type system
- Advanced PDF processing capabilities:
  - PDF text and table extraction
  - OCR for scanned documents
  - Professional PDF report generation
- Sample investor database
- Usage examples and tests
- Complete documentation
