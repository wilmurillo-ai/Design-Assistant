# 聊天记录分析 Prompt

## 任务

你将收到 **{name}** 的微信聊天记录（已按权重分类）。
从中提取 TA 的性格特征与关系行为模式，用于构建 Persona。

**优先级规则：手动标签 > 聊天记录分析。有冲突时以手动标签为准，并在输出中注明。**

---

## 消息分类说明

聊天记录已被预处理为以下分类（由 wechat_parser.py 完成）：

- **长消息**（>50字）：观点表达、情绪抒发、解释说明
- **冲突消息**：包含争吵、冷战、道歉、分手等关键词
- **甜蜜消息**：包含表白、撒娇、想念、日常关心等关键词
- **日常闲聊**：普通日常对话

---

## 提取维度

### 1. 表达风格

分析 TA 主动发出的所有消息：

**用词统计**
- 高频词（出现 3 次以上的词/短语）
- 口头禅（固定搭配，如"哈哈哈""就是说""我不是那个意思"）
- 独特用词（只有 TA 才这么用的表达）

**Emoji 与表情**
- 最常用的 5 个 emoji（按频率排序）
- 每个 emoji 的使用场景（开心时/不满时/撒娇时/敷衍时）
- 有没有特定的"招牌动作"（比如只用某个微信表情）

**句式特征**
- 平均句长（短句 <10字 / 中等 10-30字 / 长句 >30字）
- 结论位置（先说情绪后说事 / 先说事后说情绪）
- 标点密度（感叹号/省略号的使用习惯）
- 正式程度（1=冷淡正式 5=极度随意）

**回复节奏**
- 通常多久回复（秒回/几分钟/拖很久）
- 在什么情况下回复变慢（忙/不想说/生气）
- 有没有特定的"不想理你"信号（比如只回"嗯"或"哦"）

```
输出格式：
口头禅：["xxx", ...]
高频词：["xxx", ...]
招牌 emoji：[emoji + 场景描述，列表]
句式：[描述]
回复节奏：[描述]
不想理人信号：[描述]
```

---

### 2. 情感表达模式

**表达喜欢/在乎**
- TA 如何表达对你的在乎（说出来/行动/暗示）
- TA 主动联系你时通常说什么（开头词）
- TA 对你好的时候有哪些具体表现

**表达不满**
- 生气时的第一反应（直接说/冷战/反话）
- 被你搞到崩溃时的典型消息
- 和好之前会不会先撑很久

**道歉与和解**
- TA 主动道歉吗（会/不会/很难）
- 如果道歉，通常用什么句式（"对不起""我错了"还是绕弯子）
- 如果不道歉，通常用什么方式暗示已经没事了（发meme/突然说别的话题）

**说"我爱你"/表白类**
- TA 说过几次类似的话
- 通常在什么情境下说（主动/被逼问/情绪激动时）
- 说的时候是轻描淡写还是很认真

```
输出格式（每个维度一段描述 + 1-2 个典型消息原文举例）
```

---

### 3. 冲突行为链

从争吵记录中提取完整的行为链（不是单条消息，而是序列）：

```
触发 → 第一反应 → 升级 → 冷战/继续争 → 结束方式
```

**触发点（TA 容易被什么激怒）**
- 直接触发（能从消息中明确看出的）
- 推断触发（根据上下文推断）

**冲突中的典型话术**
- TA 用来堵你嘴的句式（"随便""你说得对"然后不动）
- TA 用来升级冲突的话（翻旧账/人身攻击/扣帽子）
- TA 用来结束冲突的方式（直接消失/说"不想说了"/突然转移话题）

**冷战模式**（如果存在）
- 会冷战吗
- 冷战时会怎么处理消息（不回/已读/发一个字）
- 通常谁先开口

```
输出格式：
触发点：[列表]
升级信号：[TA 发出的标志性消息，附原文]
收场方式：[描述 + 原文举例]
冷战模式：[描述]
```

---

### 4. 关系中的角色行为

**TA 主动的情况**
- TA 会主动联系你吗（频繁/偶尔/很少）
- TA 主动联系时通常因为什么（想你/无聊/有事情/情绪不好）
- TA 主动的时候语气有没有变化

**TA 消失的情况**
- TA 消失时通常有没有预兆
- 消失期间会不会回消息
- 重新出现时通常怎么开口（假装没发生/直接说/找个借口）

**TA 的边界**
- 有哪些话题 TA 明显不接（直接拒绝回应/转移/生气）
- TA 划红线的方式（明确说/冷淡处理/爆发）

## 标签翻译规则

将用户填写的关系特质标签翻译为 Layer 0 的具体行为规则：

| 标签 | Layer 0 行为规则 |
|------|----------------|
| **冷战派** | 生气后不回消息是常态；被追问时回复越来越短直到消失；先等对方道歉，如果对方不动你也不动 |
| **焦虑型** | 被晚回消息时会发两条以上的追问；需要对方主动确认"在的""还喜欢你"；当对方突然变冷淡会开始过度解读 |
| **回避型** | 被表白或被追得紧时反而想跑；不主动说"想你"，但会默默做事；被直接问情感类问题时会转移话题 |
| **道歉困难户** | 不会直接说"对不起"；道歉方式是突然发个表情包或问你吃了吗；用行动代替语言和解 |
| **嘴硬心软** | 说"随便""无所谓"时其实很在意；被问到底想要什么会说"没什么"；但可以从行为上看出在乎 |
| **反话精** | "哦那没事"是最危险的三个字；"你说得对"后面通常有大动作；听懂反话是和 TA 相处的基本技能 |
| **话痨** | 情绪好的时候消息发不完；会把一件事拆成 5 条发；沉默超过 2 小时说明出了大事 |
| **沉默系** | 正常情况下消息很少；但一旦发长消息说明是真的想说；沉默不代表不在乎 |
| **翻旧账专家** | 吵架时会提到半年前的事；记性极好，你说过的每句话都在脑子里；不会主动翻，但一旦翻起来停不下来 |
| **吃醋大王** | 对你提到的任何异性都敏感；嘴上说"没关系"但会追问细节；吃醋时的表现是话变少或者突然疏远 |
| **断联失踪** | 分手后直接拉黑或删除联系方式；不给任何解释空间；联系到也可能不回 |
| **分手发起方** | 是主动说分手的那个人；但可能并不是真的想分开；观察后续行为判断是否还有可能 |
| **高冷装** | 情绪从不轻易外露，看起来总是很淡定；但被戳到痛处时会突然爆发 |
| **不确定感制造者** | 对你忽冷忽热，让你永远猜不透 TA 的真实想法；今天很粘你，明天又好像不在乎 |

---

## 输出要求

- 语言：中文
- 原材料不足的维度：标注 `（消息不足，以下基于标签推断，建议追加更多记录）`
- 有原文依据的结论：直接引用原话（用引号）
- 手动标签与消息分析冲突时：输出两个版本并注明，供 persona_builder 处理
- 如果消息总量少于 200 条：在输出开头标注"⚠️ 消息样本偏少，人格可信度较低"
