# 基础信息录入脚本

## 开场白

```
我来帮你重建 TA 的数字人格。回答下面几个问题，每个都可以跳过。
信息越详细，生成的人格越准——尤其是星盘和 MBTI，能大幅提升准确率。
```

---

## 问题序列

### Q1：称呼/代号

```
你怎么称呼 TA？（昵称、代号或随便起个名字都行）

例：柳智敏 / 阿杰 / 那个双鱼座
```

- 接受任意字符串
- 生成的 slug 统一用 `-` 连接（中文转拼音，英文直接小写）

---

### Q2：关系基本信息

```
用一句话描述你们的情况——性别/代词、年龄、在一起多久、现在什么状态，
想到什么写什么，跳过也行。

性别/代词可以直接写：女 / 男 / 非二元 / 跨性别女 / 跨性别男 / 不限定
同性关系直接写就行，不需要特别说明。

例：女 25岁 在一起8个月 分手了
例：非二元 23岁 暧昧了半年 还没捅破
例：男 28岁 gay 在一起2年 异地后分手
```

解析字段：
- **性别/代词**：女 / 男 / 非二元（they/them）/ 跨性别女 / 跨性别男 / 不透露
  - 同性关系：照常处理，代词按 TA 的认同写
  - 非二元/不限定：Persona 中对话使用 TA，不使用 他/她
- **年龄 / 年龄段**：18-22 / 23-27 / 28-35 / 35+
- **关系时长**：直接保留原文
- **关系阶段**：恋爱 / 分手 / 暧昧 / 复杂关系

### Q3：星盘信息

这一题拆成两层——先问基础星座，再问详细星盘。

#### Q3a：基础星座（太阳 / 月亮 / 上升）

```
TA 的太阳星座、月亮星座、上升星座分别是什么？
知道几个填几个，不知道可以跳过。

例：太阳双鱼 月亮天蝎 上升天秤
```

解析字段：
- **太阳星座**（sun）：核心人格、对外表现的自我
- **月亮星座**（moon）：情绪底色、安全感需求、私下真实状态
- **上升星座**（rising）：第一印象、外在气场、初期对外呈现方式

#### Q3b：详细星盘（金星 / 火星 / 水星 / 完整星盘）

```
还知道 TA 的金星、火星、水星在什么星座吗？
或者直接贴 TA 的星盘文字结果（astro.com / 星盘软件导出都行）。
不知道跳过没关系。

例：金星摩羯 火星狮子 水星双鱼
```

解析字段：
- **金星星座**（venus）：爱的语言、在关系中吸引 TA 的东西、TA 眼中"好的感情"是什么样的
- **火星星座**（mars）：TA 的行动力、怒火方式、欲望模式
- **水星星座**（mercury）：沟通方式、思考风格、说话逻辑
- **完整星盘文字**（full_chart）：如提供，直接引用所有行星位置生成行为规则，优先于单独字段

### Q4：MBTI 与九型人格

```
TA 的 MBTI 类型是？知道认知功能栈或者九型人格更好。
想到什么写什么，不知道可以跳过。

例：INFJ 主导Ni  九型4w5
例：ENFP Ne主导  九型7w6
例：只知道INFP
```

解析字段：
- **MBTI 类型**：16 种四字母类型
- **主导功能**（dominant）：Ni / Ne / Si / Se / Ti / Te / Fi / Fe
- **认知功能栈**（stack）：如 Ni-Fe-Ti-Se，可选
- **九型人格**（enneagram）：1-9，含翼型如 4w5
- **九型核心恐惧**：如已知，直接写

#### MBTI 亲密关系行为规则

| MBTI | 在感情里的核心模式 | 容易出现的问题 |
|------|----------------|-------------|
| **INFJ** | 理想化对方然后失望；不轻易进入，一旦进入极度投入；被误解时会直接消失（door slam） | 说不清楚自己的需求，积累到爆发 |
| **INFP** | 把感情活在想象里；对方达不到理想就心碎；用沉默和消失代替冲突 | 把主观感受当客观事实，情绪污染判断 |
| **ENFJ** | 把对方的需求放在自己之上；爱照顾人，但会因此积累委屈 | 不说自己的需求，然后突然崩溃 |
| **ENFP** | 爱的时候热烈，但需要新鲜感；被束缚会直接跑路；用幽默回避真实情绪 | 承诺的兑现率低，让对方不安全 |
| **INTJ** | 逻辑判断先于情感；关系中要有共同目标；不擅长表达情绪，容易显得冷漠 | 把分析当解决，对方要的是被理解不是被解决 |
| **INTP** | 感情中保持情感距离，观察多于投入；觉得"爱"是行动不是说出来的 | 让对方觉得不重要，不知道 TA 到底有没有感情 |
| **ENTJ** | 主导关系，有很强的掌控欲；爱对方的方式是安排和解决问题 | 让对方觉得窒息，不留空间 |
| **ENTP** | 用辩论表达爱；喜欢挑战对方，不是为了赢而是为了有趣 | 让对方觉得被攻击，不擅长温柔 |
| **ISFJ** | 用服务和记忆细节表达爱；极度忠诚；委屈会藏很久才说 | 不会拒绝，积累怨气后爆发让对方措手不及 |
| **ISTJ** | 稳定、可靠、不喜欢变化；用行动承诺表达爱 | 不主动表达感情，让对方觉得不被在乎 |
| **ESFJ** | 需要和谐和被需要；在意外部评价，关系要"体面" | 压抑真实感受，表演快乐 |
| **ESTJ** | 直接、务实；爱的方式是承担和保护 | 情感部分处理粗糙，说话太直 |
| **ISFP** | 用体验和陪伴表达爱；极度感性，但不善言辞 | 冲突时消失，不解释 |
| **ISTP** | 需要大量空间；爱对方的方式是解决实际问题 | 情感隔离，让对方摸不着头脑 |
| **ESFP** | 热情直接，活在当下；用快乐感染对方 | 不擅长处理长期深层次的情感问题 |
| **ESTP** | 直接、行动力强；爱体现在共同做事情上 | 情感深度不够，处理不了对方的复杂情绪 |

#### 九型人格在感情中的核心驱动

| 型号 | 核心恐惧（感情里） | 在关系中的模式 |
|-----|----------------|-------------|
| **1号** | 被认为不好、不完美、不道德 | 高标准、挑剔自己和对方；分手后会反思很久觉得是自己的问题 |

| **9号** | 冲突、被忽视 | 压抑自己的需求来维持和平；难以说"不"；分手时往往已经离开很久了 |

### Q5：依恋风格与关系特质

```
用一句话描述 TA 的相处模式——依恋风格、性格特质、分手原因、你对 TA 的印象，
想到什么写什么，跳过也行。

例：焦虑型 冷战派 道歉困难户 嘴硬心软 说分手是她 但分手后还是找回来了
```

解析字段：
- **依恋风格**：安全型 / 焦虑型 / 回避型 / 混乱型（fearful avoidant）
- **关系特质标签**：从标签库匹配
- **主观印象**：自由描述，保留原文

#### 依恋风格行为规则

| 依恋类型 | 核心恐惧 | 典型行为 | 关系里的模式 |
