# 发布文案生成规范

## 标题生成逻辑

### 优先级顺序
1. **涨停个股 + 原因** — 最抓眼球，例如「XX今天涨停，背后原因AI找到了」
2. **热点事件 + 市场联动** — 例如「中美会谈前夕，这个板块悄悄涨了」
3. **大盘数据 + 反差** — 例如「大盘微跌，但这5只股票涨停了」
4. **AI智能角度** — 例如「AI分析了今天所有涨停股，发现一个共同点」

### 标题模板库

| 场景 | 模板 | 示例 |
|------|------|------|
| 有明确涨停股 | `[股票名]今天涨停，AI告诉了我原因` | 朗科科技今天涨停，AI告诉了我原因 |
| 热点板块爆发 | `[板块]今天集体爆发，AI找到了主线` | AI存储今天集体爆发，AI找到了主线 |
| 宏观事件驱动 | `[事件]来了，这些股票提前反应了` | 中美会谈来了，这些股票提前反应了 |
| 大盘普涨 | `[上涨家数]只股票上涨，AI复盘今日A股` | 3152只股票上涨，AI复盘今日A股 |
| 大盘普跌 | `今天A股跌了，但AI发现[亮点]` | 今天A股跌了，但AI发现这个板块在悄悄建仓 |

### 禁止使用的表达
- 不用「暴涨」「崩盘」「必涨」等夸张词
- 不做投资建议（「建议买入/卖出」）
- 不预测明日走势

## 正文话术结构

### 公式
```
[今日亮点一句话（15-30字）]
[AI是如何分析的（20-40字）]
[行动号召（10-20字）]
```

### 模板示例

**场景：有涨停股+热点新闻**
> 今天[股票名]涨停，原来是[原因简述]。
> AI交叉比对今日新闻，5秒找到了背后逻辑。
> 想让AI每天自动给你发这份复盘？评论区有安装方法。

**场景：宏观驱动**
> [宏观事件]影响下，[板块]今天[涨/跌]了[幅度]。
> AI把新闻和行情数据对照了一遍，逻辑很清晰。
> 这套系统每天自动跑，不用手动操作。

**场景：大盘普涨**
> 今天[上涨家数]只股票上涨，赚钱效应不错。
> AI帮我梳理了今天的主线：[主线概述]。
> 数据来自非凸科技真实接口，不是AI瞎编的。

**场景：大盘混沌**
> 今天市场分化，[涨停板块]和[跌幅板块]冰火两重天。
> AI归因分析帮我搞清楚了背后逻辑。
> 感兴趣的可以看看评论区的安装方法。

## 平台适配

### 视频号（主战场）
- 标题：≤20字，带悬念
- 正文：≤100字，口语化
- 话题：6-8个

### 微信朋友圈（二传）
- 去掉话题标签
- 正文可适当加长到150字
- 加「附视频号链接」提示

### 微信群（三传）
- 只发文字版复盘摘要（不带视频）
- 附一句「完整视频在视频号：@非凸科技」

## 盘中快报文案

盘中场景与收盘复盘的区别：

### 标题模板

| 场景 | 模板 | 示例 |
|------|------|------|
| 盘中异动 | `[股票名]盘中涨停，AI实时分析原因` | 朗科科技盘中涨停，AI实时分析原因 |
| 盘中板块爆发 | `[板块]盘中集体拉升，发生了什么？` | AI芯片盘中集体拉升，发生了什么？ |
| 大盘异动 | `大盘盘中跳水/拉升，AI找到了[原因]` | 大盘盘中跳水，AI找到了三个信号 |

### 正文话术

结构：`[盘中异动描述] + [AI实时分析] + [数据截至时间提醒]`

> [板块/个股]盘中[异动描述]。AI实时交叉比对新闻，初步归因如下。
> ⚠️ 数据截至 HH:MM，盘中数据会持续变化，仅供参考。

### 注意事项

- 标题和正文必须标注「盘中」，避免误导为收盘数据
- 不使用「暴涨」「崩盘」等夸张词
- 必须注明数据截取时间
