# Content Creation — Do Rules

## Hard DOs

| ID | Rule |
|---|---|
| CD-001 | Always write hook before any other part of content |
| CD-002 | Always tag funnel stage before moving to review |
| CD-003 | Always tag content pillar before moving to review |
| CD-004 | Always include a CTA — even if soft (save/share/comment) |
| CD-005 | Always check brand voice guidelines before approving |
| CD-006 | Always verify hashtag set is current (not stale) |
| CD-007 | Always describe visual even if asset not yet created |
| CD-008 | Always adapt caption for each platform (no copy-paste) |

## Soft DOs

| ID | Rule |
|---|---|
| SD-001 | Use proven hook formula from brand/hooks/master.md |
| SD-002 | Include a personal story element in behind-scenes content |
| SD-003 | Add a data point or stat to educational content |
| SD-004 | Post within optimal time window for the platform |
| SD-005 | Reply to early comments to boost algorithm reach |
| SD-006 | Cross-link platforms where natural (not forced) |
