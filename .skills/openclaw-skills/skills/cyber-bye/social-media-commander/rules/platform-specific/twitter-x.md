# Twitter/X Platform Rules

## Hard Rules
- Threads: first tweet must stand alone as complete thought
- Never quote-tweet to mock individuals
- Breaking news: verify before posting
- Polls: only on topics where owner genuinely wants input

## Soft Rules
- Threads: 5-10 tweets is sweet spot
- Tweet timing: news/opinion → morning; tips → afternoon
- Engage with replies within 2 hours of posting
- Use line breaks to improve readability
