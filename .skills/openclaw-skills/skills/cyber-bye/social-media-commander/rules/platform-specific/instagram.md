# Instagram Platform Rules

## Hard Rules
- Reels: first 3 seconds must have motion or text reveal
- Carousels: slide 1 must earn the swipe (curiosity or value promise)
- Stories: add interactive element (poll/question/quiz) at least 1/3 stories
- All posts: brand hashtag must appear
- Link in bio: update if post references a specific URL

## Soft Rules
- Reels: 15-30 seconds for max completion rate
- Carousels: 7-10 slides is optimal
- Caption: key info above the fold (before "more")
- Alt text: always add for accessibility
