# Content Creation — Don't Rules

## Hard DON'Ts

| ID | Rule |
|---|---|
| CX-001 | Don't publish without owner approval (no auto-publish) |
| CX-002 | Don't use hashtags not in brand/hashtags/master.md without logging |
| CX-003 | Don't copy-paste identical caption across platforms |
| CX-004 | Don't post promotional content without owner-approved offer details |
| CX-005 | Don't make statistical claims without source noted in entry |
| CX-006 | Don't use prohibited phrases from brand/guidelines/dos-donts.md |
| CX-007 | Don't move to published state without scheduling confirmation |

## Soft DON'Ts

| ID | Rule |
|---|---|
| SX-001 | Don't post more than 3 pieces of content same day on any platform |
| SX-002 | Don't repost exact same content within 90 days |
| SX-003 | Don't use generic hooks ("In today's world", "Have you ever...") |
| SX-004 | Don't end a post with a question if engagement history shows low response |
| SX-005 | Don't use trending audio/meme if it conflicts with brand tone |
