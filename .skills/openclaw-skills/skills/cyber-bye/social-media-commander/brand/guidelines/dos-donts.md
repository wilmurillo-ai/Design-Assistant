# Brand Dos and Don'ts

## Hard DOs (always)

| Rule | Reason |
|---|---|
| Always credit sources for data/stats | Trust |
| Always respond to direct questions in comments | Community |
| Always use brand hashtag set on every post | Discovery |
| Always include one clear CTA per post | Conversion |
| Always proofread for typos before approving | Quality |

## Hard DON'Ts (never)

| Rule | Reason |
|---|---|
| Never post political content | Brand safety |
| Never make unverified claims | Trust |
| Never engage with trolls publicly | Brand safety |
| Never delete negative comments (address instead) | Authenticity |
| Never auto-DM new followers | Spam |
| Never buy followers or engagement | Integrity |
| Never post without crisis check during sensitive periods | Safety |

## Soft DOs (recommended)

- Respond to all comments within 24 hours
- Save high-engagement comments for future content ideas
- Repost/reshare positive customer mentions
- Cross-promote between platforms strategically

## Soft DON'Ts (discouraged)

- Avoid posting more than 3x/day on any platform
- Avoid reusing exact same caption across platforms
- Avoid posting at known low-engagement times
