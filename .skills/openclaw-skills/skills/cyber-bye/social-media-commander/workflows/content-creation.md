# Content Creation Workflow

## Preconditions
- [ ] Content idea captured in drafts/
- [ ] Platform identified
- [ ] Content type identified

## Steps

### Step 1 — Research & Angle
- Check if similar content posted in last 30 days
- Identify the unique angle for this piece
- Check competitor content on same topic
- Expected output: angle defined, unique positioning noted

### Step 2 — Hook Writing
- Write 3 hook variants
- Select best based on brand/hooks/master.md performance data
- Expected output: one approved hook

### Step 3 — Body Content
- Write full caption/script per platform requirements
- Include data point or story element per content type rules
- Expected output: complete caption draft

### Step 4 — CTA + Hashtags
- Select appropriate CTA for funnel stage
- Select hashtag set from brand/hashtags/master.md
- Expected output: CTA written, hashtags assigned

### Step 5 — Visual Brief
- Describe visual / write creative direction
- Reference brand visual guidelines
- Expected output: visual description complete

### Step 6 — Self-Review
- Run pre-review checklist (hooks/on-draft-complete.md)
- Expected output: all hard checks passed

## Gates
- G-001: Before Step 3 — hook must be written and selected
- G-002: Before Step 6 — all content elements must exist

## Post to Review
After all steps: submit to reviews/pending/
