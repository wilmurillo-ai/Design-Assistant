# ths-stock-themes - 同花顺个股题材查询

获取同花顺个股题材/概念板块和人气排名数据。

## 功能特性

- 🏷️ **题材查询** - 查询单只股票在同花顺的题材/概念分类
- 🔥 **人气排名** - 获取同花顺个股人气排名榜单
- 📊 **概念板块** - 包含所属概念板块信息
- 📈 **实时数据** - 从同花顺问财获取最新数据

## 使用方式

### 查询单只股票题材

```bash
python3 ~/.jvs/.openclaw/workspace/skills/stock-data-monorepo/ths-stock-themes/scripts/fetch_themes.py [股票代码]
```

### 示例

```bash
# 查询贵州茅台
python3 scripts/fetch_themes.py 600519

# 查询中国平安
python3 scripts/fetch_themes.py 601318
```

### 获取人气排名

```bash
python3 scripts/fetch_popularity.py
```

## 数据字段

### 题材查询
| 字段 | 说明 |
|------|------|
| 股票代码 | 6 位数字代码 |
| 股票简称 | 股票名称 |
| 题材列表 | 所属题材/概念板块 |
| 核心题材 | 主要炒作题材 |

### 人气排名
| 字段 | 说明 |
|------|------|
| 排名 | 人气排名 |
| 股票代码 | 6 位数字代码 |
| 股票简称 | 股票名称 |
| 人气值 | 人气指数 |
| 涨跌幅 | 当日涨跌幅 |
