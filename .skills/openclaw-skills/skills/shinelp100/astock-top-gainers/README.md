# A股涨幅排行榜 (astock-top-gainers)

获取A股市场近N日涨幅排名前N只股票，自动排除ST股票，输出格式化表格。

## 功能特点

- 🎯 **精准排名**：获取指定天数内的涨幅排行榜
- 🚫 **智能过滤**：自动排除ST、*ST等风险股票
- 📊 **格式化输出**：清晰的表格展示，包含多个关键指标
- ⏱️ **实时数据**：数据来源同花顺问财，准实时更新

## 使用方法

### 基本用法

```
近10日涨幅排行
近5日涨幅排名前20
A股涨幅最大的股票
最近涨得好的股票
```

### 参数说明

- **天数**：可指定3、5、10、20、30天等，默认10天
- **数量**：默认显示前20只，可自定义

- **同花顺问财** (iwencai.com) - 国内领先的智能选股平台

## 注意事项

1. ST股票（名称含ST、*ST等）已自动过滤
2. 数据为实时/准实时更新
3. 异常涨幅股票会有风险提示
