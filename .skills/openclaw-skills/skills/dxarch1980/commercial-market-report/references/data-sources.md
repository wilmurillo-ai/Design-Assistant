# 数据来源清单

## 数据时效性原则（强制）

**时效标准：以报告日期为基准，反推2年内均为有效数据。**
- 报告日期2026年 → 2024年及以后数据均可用
- 行业报告（如戴德梁行/世邦魏理仕等）通常滞后半年，2025H2报告在2026年发布属正常

| 数据类型 | 最低时效要求 | 超时处理 |
|----------|------------|---------|
| 地价/土地成交 | 反推2年内 | 标注时间+说明趋势 |
| 租金水平 | 反推2年内 | 标注时间+说明趋势 |
| Cap Rate | 反推2年内 | 标注并用同类城市参照 |
| 商业体量/空置率 | 反推2年内 | 标注时间+判断供需趋势 |
| 人口/经济数据 | 反推3年内 | 可用，注意数据年份 |
| 竞品数据 | 反推2年内 | 实地调研补充 |
| 政策/规划 | **最新** | 随时关注更新 |

> **搜索时必须附加时间词**：`2024` `2025` `最新` `近一年` 等

### 周边业态扫描（自动执行，每项目必做）

使用高德 POI 分业态对项目周边进行生态扫描，量化分析周边商业配套：

**标准搜索分类：**

| 业态大类 | 搜索关键词 | 用途 |
|---------|-----------|------|
| 零售商业 | `商场 超市 便利店` | 判断零售饱和度 |
| 餐饮 | `餐厅 美食 饭店` | 餐饮密度和品类分析 |
| 教育培训 | `培训 教育` | 判断亲子业态机会 |
| 儿童亲子 | `儿童 亲子 游乐` | 竞争/机会分析 |
| 电影院/KTV | `影院 KTV` | 体验业态竞争分析 |
| 美容健身 | `美容 健身 SPA` | 配套完善度分析 |

**限流处理：** amap-lbs-skill 已内置 800ms 间隔 + 3次重试，批量搜索时会自动等待重试。

**输出格式：**
```
餐饮: 450条 | 密度高（>100条/平方公里）
零售: 6条   | 集中在华联/嘉华商圈
教育培训: 7条 | 以驾校/托管为主，连锁品牌少
```

**应用：** 纳入第2章竞争分析 + 第4章业态规划依据。

## 多引擎搜索（补充，无API key）

`multi-search-engine` skill 已安装，支持百度/搜狗/必应等。

**使用场景：**
- 搜索政府统计公报PDF/网页（国内搜索引擎对政府网站收录更好）
- 搜索特定来源（site:gov.cn）
- Tavily 搜索结果不够时，用百度补充

**示例：**
```
web_fetch("https://www.baidu.com/s?wd=槐荫区+2024+统计公报+常住人口")
```

## AI 推断与经验值标注规范

当真实数据无法获取时，使用行业经验值，但**必须严格标注**：

### 可用经验值的场景
| 数据 | 经验值范围 | 标注格式 |
|------|----------|---------|
| 建安成本（二线城市商业） | 3,500-5,000 元/m² | 「根据行业经验」 |
| 装修成本（商业精装） | 1,500-3,000 元/m² | 「参考同类项目」 |
| 运营成本占比 | 租金收入 15-25% | 「行业经验值」 |
| 商业 Cap Rate（二线） | 6-8% | 「参照同类城市」 |

### 禁止使用经验值的场景（必须搜真实数据）
- **土地价格（地价）** — 土地成本是最大项，必须真实
- **区域租金水平** — 直接影响收入测算
- **Cap Rate** — 直接影响估值，必须有市场依据

### 标注格式
```
土地楼面价：8,500 元/m²（注：参照 {城市} {区域} 2024年商业地块成交价，均价8,000-9,000元/m²）
商业 Cap Rate：7.5%（注：参照戴德梁行 {城市} 2024Q3 商业市场报告）
```

## 数据缺失处理

| 情况 | 处理方式 |
|------|---------|
| 搜索结果均为 3 年+ 前数据 | 标注「数据较旧，需实地调研」，使用数据但注明时效 |
| 搜索完全无结果 | 标注「**[待补充]**」，使用合理经验值并注明 |
| 数据矛盾（多个来源不一致） | 采用最权威来源，注明「不同来源数据存在差异」 |
