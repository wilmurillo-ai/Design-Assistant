# Word 报告格式规范

## 文档格式

- 页面：A4 纵向
- 页边距：上下2.54cm，左右3.17cm
- 页眉：项目名称 + 日期
- 页脚：页码（第X页共Y页）
- 字体：正文宋体小四，标题黑体

## 章节标题格式

| 级别 | 格式 | 示例 |
|------|------|------|
| 一级标题 | 黑体三号，居中 | 第1章 项目概况与区域分析 |
| 二级标题 | 黑体四号 | 1.1 项目概况 |
| 三级标题 | 楷体小四 | （1）区位条件 |
| 正文 | 宋体小四，1.5倍行距 | 段落文字... |

## 图表规范

- 图表标题：图题在图下方，表题在表上方
- 字体：宋体五号
- 图：嵌入 Word，不要浮动
- 表：规范表格，有表头

## 封面格式

```
项目名称
市场调研报告

[制作单位]（可选）
[日期]
```

## 目录

自动生成，标注页码

## 装订

左侧装订，胶装
