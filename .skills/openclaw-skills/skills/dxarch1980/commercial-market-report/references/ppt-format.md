# PPT 汇报稿格式规范

## 页面结构

| 页码 | 内容 |
|------|------|
| 1 | 封面 |
| 2 | 目录 |
| 3-4 | 第1章摘要 |
| 5-6 | 第2章摘要 |
| 7-8 | 第3章摘要 |
| 9-10 | 第4章摘要 |
| 11-12 | 第5章摘要 |
| 13 | 第6章结论与建议 |
| 14 | 封底 |

## 封面内容

- 项目名称
- 副标题：「市场调研汇报」
- 汇报日期
- 汇报人（可选）

## 每章摘要内容（1-2页）

- 章节标题
- 3-5个核心要点（bullet points）
- 关键数据/图表
- 一句话结论

## 结论页重点

- 核心结论（3条以内）
- 设计建议要点
- 谢谢观看 / 欢迎提问

## 视觉规范

- 主色调：项目VI色或专业蓝
- 字体：微软雅黑
- 每页要点不超过5条
- 善用图标/数据图表
- 保持视觉一致性
