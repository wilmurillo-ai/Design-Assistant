# 商业市调报告章节编写指南

---

## 数据时效性规范（强制）

所有数据必须为**近1-2年内**的权威来源。

**标注规范：**
- 数据年份 < 2年 → 不需要特别注明（视为有效）
- 数据年份 2-3年 → 标注「数据截至 XXXX 年，需更新」
- 数据年份 > 3年 或 搜不到 → 标注「**[待补充]**」，使用行业经验值并注明
- 估值类数据（租金/cap rate/IRR）→ 必须注明数据来源和时效

## 图表选择规则

| 数据特征 | 图表类型 | 章节应用 |
|----------|----------|----------|
| 部分占整体比例 | **饼图** | 业态配比、客群结构、投资构成 |
| 多项目对比同一数值 | **横向柱状图** | 竞品体量、各楼层面积对比 |
| 同一指标随情景/时间变化 | **分组柱状图** | 敏感性IRR、历年收入对比 |
| 多年度趋势（≥5年） | **折线图** | 10年NOI/FCF趋势、收入增长曲线 |
| 数据精确值 | **表格** | 财务明细表、楼层指标表、竞品详情 |

## 第1章 项目概况与区域分析

### 1.1 项目概况
- 项目名称、位置（经纬度/门牌号）
- **用地面积（m²）**、容积率、建筑密度、绿化率、限高
- 可建总面积 = 用地面积 × 容积率
- 物业组合及面积配比
- 一句话定位

### 1.2 区域概况
- 行政区划、地理区位
- **区域人口**（常住/流动，数据须为近3年内）
- 经济发展水平（GDP/社会消费品零售总额，须为最新）
- **市场供需现状**（供需比、空置率、存量体量 → 搜最新数据）

### 1.3 交通可达性分析
- 地铁/公交覆盖（高德 API 自动查询）
- 周边路网结构
- 停车条件

### 1.4 政策规划
- 区域规划定位（政府文件）
- 商业地产相关政策（扶持/限制）
- 地铁/道路新建规划（影响项目潜力）

### 2.4 SWOT 分析

**S 优势：**
- 区位（地铁上盖/主干道/政策重点）
- 体量（规模适中/定位精准）
- 品牌（开发商实力/招商资源）

**W 劣势：**
- 区域认知度（新区培育期）
- 竞品分流
- 招商难度

**O 机会：**
- 市场缺口（精品/亲子/夜经济）
- 人口导入
- 消费升级

**T 威胁：**
- 竞品分流加剧
- 电商冲击
- 成本上涨

**战略建议（SO/WO/ST/WT）：**
- SO：利用区位优势抓住市场空白
- WO：借助品牌资源弥补短板
- ST：差异化互补，避免正面竞争
- WT：控制成本，分期开发

## 第5章 财务测算与估值

> **所有数据必须为近1-2年内市场数据。** 地价/租金/cap rate 等每半年更新一次，报告中须注明数据来源和时效。

### 5.1 土地成本
```
土地总价 = 用地面积（m²）× 商业楼面价（元/m²）
```
- **搜索近1-2年内同城类似地块成交价格**
- 如无成交案例，用周边地块类比，注明估算依据

### 5.2 建设投资
```
建安成本 = 可建面积 × 建安单价（元/m²）
装修成本 = 商业面积 × 装修单价（元/m²）
其他费用 = 前期费 + 预备费（约建安×10%）
```
- 建安单价参考当地市场（搜最新数据）
- 装修单价分业态（精装/毛坯）

**图表：投资构成饼图**（土地/建安/装修/其他占比）

### 5.3 收入测算
```
年收入 = Σ(各业态面积 × 租金单价 × 12 × 出租率) + 其他收入
```
- **租金单价：必须搜索近1年内市场数据**，分业态给出
- 出租率假设：开业期 60-70% → 成熟期 90-95%
- 其他收入：停车费、广告费、多种经营

### 5.4 运营成本
```
运营成本 = 人员费用 + 物管费 + 营销费 + 保险 + 税费
           （约为租金收入的 15-25%）
```

### 5.5 经营性现金流（NOI）测算

**逐年 NOI 计算表（10年）：**

| 年份 | 潜在总收入 | 空置损失 | 实际总收入 | 运营成本 | NOI |
|------|----------|----------|----------|----------|-----|
| Y1 | | | | | |
| Y2 | | | | | |
| ... | | | | | |

```
NOI = 实际总收入 - 运营成本
```
- 假设 Y1-Y3 为培育期，出租率逐步提升
- Y4 起进入成熟期，NOI 稳定增长（2-3%/年）

**图表：10年 NOI 折线图**（展示培育期→成熟期趋势）

### 5.6 项目估值（NOI资本化法）

```
估值 = NOI（稳定年） ÷ Cap Rate
```

- **Cap Rate：搜索同城近1-2年同类商业成交案例**
- 典型 Cap Rate 范围：一线城市 4-6%，二线城市 6-8%，三线城市 8-10%
- 如无本地数据，用省会/同类城市数据类比，注明

**Cap Rate 敏感性估值表：**

| Cap Rate | 估值（万元） |
|----------|------------|
| 6% | |

| 8% | |
| 9% | |

**图表：Cap Rate 敏感性柱状图**

### 5.7 动态 IRR 测算

**10年自由现金流（FCF）表：**

| 年份 | NOI | 资本性支出 | FCF | 累计FCF |
|------|-----|----------|-----|---------|
| Y0（投资） | - | - | -土地-建安 | |
| Y1 | | | | |
| Y2 | | | | |
| ... | | | | |
| Y10（退出） | | | +退出估值 | |

```
退出估值 = Y10 NOI ÷ Cap Rate（退出）
IRR = 使得 ΣFCF/(1+r)^t = 0 的 r
```

- 退出 Cap Rate 通常比购入时高 0.5-1%（反映市场波动）
- 退出方式假设：整体交易或散售

**图表：10年累计FCF折线图**（展示回本节点）+ **IRR敏感性柱状图**

### 5.8 静态回报指标
```
成本利润率 = 累计净利 ÷ 总投资 × 100%
投资回收期 = 累计正现金流出现年份（不含折现）
```

## 第6章 结论与设计建议

### 6.1 核心结论
- 市场可行性（供需/竞争/需求缺口）
- 差异化机会
- 主要风险提示

### 6.2 风险分析
| 风险类型 | 风险描述 | 应对策略 |
|---------|---------|---------|
| 市场风险 | 供需失衡/电商冲击 | 差异化定位 |
| 招商风险 | 品牌商家入驻不及预期 | 备选品牌库 |
| 资金风险 | 融资成本上涨/销售滞缓 | 分期开发 |
| 运营风险 | 出租率低/租金下跌 | 动态调整业态 |

### 6.3 设计建议
- 建筑形态（街区+盒子）
- 动线设计
- 空间亮点
- 停车/物流

### 6.4 开发节奏
- 分期策略（一期先开哪部分）
- 开业节奏建议

### 6.5 投资建议摘要
- 投资价值判断（简明结论）
- 核心数据一览（体量/投资/IRR/回收期/估值）
