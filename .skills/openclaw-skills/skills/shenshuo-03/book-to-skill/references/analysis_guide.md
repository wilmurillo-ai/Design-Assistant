# 书籍分析指南

## 分析框架

### 1. 主题识别
- 书的标题和副标题
- 目录结构
- 作者的核心主张

### 2. 概念提取
- 关键术语
- 定义和解释
- 概念之间的关系

### 3. 方法论提炼
- 作者提出的方法
- 具体步骤
- 适用条件

### 4. 思维方式归纳
- 思考模式
- 认知偏见克服
- 决策框架

### 5. 实践导向
- 可操作的建议
- 练习和技巧
- 案例分析

## 常见书籍类型

### 商业/管理类
- 战略思维
- 领导力
- 创新方法
- 组织管理

### 心理学/自我提升
- 认知行为
- 习惯养成
- 情绪管理
- 决策心理

### 思维/方法论
- 批判性思维
- 系统思维
- 设计思维
- 第一性原理

### 科技/技术
- 技术趋势
- 开发方法
- 数据分析
- AI/机器学习

## 技能提取原则

1. **聚焦核心**: 选择1-3个最核心、最可复制的技能
2. **可操作性**: 确保技能可以直接应用
3. **通用性**: 优先选择跨场景适用的技能
4. **独特性**: 提取书籍独特的方法论，而非通用常识
