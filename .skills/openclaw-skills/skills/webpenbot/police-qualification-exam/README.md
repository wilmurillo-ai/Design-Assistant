# 警察执法资格考试机器人助手 (PQEBot助手)

## 项目概述

警察执法资格考试机器人助手（简称PQEBot助手）是一个专门为公安机关人民警察执法资格等级考试设计的智能助手。该技能提供考试大纲解析、历年真题、考点分析、模拟练习、学习进度跟踪等功能，帮助考生系统备考。

## 快速开始

### 安装方式
1. 下载技能ZIP包
2. 解压到OpenClaw技能目录
3. 重启Agent或刷新技能列表

### 使用方式
- 触发关键词："执法资格考试"、"警察执法资格"、"PQEBot"、"公安考试"
- 示例命令："查询2026年执法资格考试大纲"

## 核心功能

### 🎯 考试大纲解析
- 完整解析2026年版《公安机关人民警察执法资格等级考试大纲》
- 分科目、分章节的考点梳理
- 考试范围、题型、分值分布说明

### 📝 历年真题库
- 收集整理历年执法资格考试真题
- 提供答案解析和解题思路
- 按年份、科目、难度分类

### 🔍 考点分析
- 高频考点统计与分析（15个核心知识点）
- 易错题整理与解析
- 重要知识点归纳

### 📊 模拟练习
- 智能组卷功能
- 错题本管理
- 练习统计与评估

### 📅 学习计划
- 个性化学习计划制定
- 学习进度跟踪（30天计划示例）
- 复习提醒功能

## 数据内容

### 考试大纲（2026年版）
- **公安基础理论** (20%)：公安工作概述、组织管理、队伍建设
- **法律基础知识** (60%)：宪法、行政法、刑法、刑事诉讼法、警察法
- **警务实战技能** (20%)：警务指挥、现场处置、安全防护

### 高频考点TOP 5
1. 刑事立案标准 (出现频率95%)
2. 行政处罚程序 (出现频率92%)
3. 正当防卫认定 (出现频率88%)
4. 搜查与扣押程序 (出现频率85%)
5. 治安调解原则 (出现频率82%)

### 考试信息
- 考试级别：初级、中级、高级
- 总分：100分，及格线：60分
- 考试时长：120分钟
- 题型：单选题、多选题、判断题、案例分析题

## 文件说明

### 核心文件
- `skill.md` - 技能说明文档（主要配置文件）
- `exam_outline_2026.json` - 2026年考试大纲数据
- `knowledge_points.json` - 知识点库（15个核心考点）
- `pqebot-core.py` - 核心功能模块
- `pqebot-web.py` - Web搜索模块
- `usage_examples.md` - 使用示例（11个典型对话场景）

### 数据文件结构
- `exam_outline_2026.json`: 包含3大科目、15个章节、考试参数等
- `knowledge_points.json`: 15个核心知识点，按类别和重要性分级
