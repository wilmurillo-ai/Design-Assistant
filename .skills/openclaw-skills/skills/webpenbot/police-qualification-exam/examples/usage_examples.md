### 4. 查看具体真题
```
用户：看一下2025年的真题示例
助手：2025年公安机关人民警察执法资格等级考试（初级）真题示例：

1. 【单选题】根据《中华人民共和国人民警察法》规定，人民警察的任务不包括下列哪一项？
   A. 维护国家安全
   B. 维护社会治安秩序  
   C. 保护公民的人身安全、人身自由和合法财产
   D. 管理经济事务
   ✅ 正确答案：D
   📝 解析：根据《人民警察法》第2条规定，人民警察的任务是维护国家安全...管理经济事务不属于人民警察的任务。

2. 【多选题】下列哪些属于行政处罚的种类？
   A. 警告 B. 罚款 C. 行政拘留 D. 拘役 E. 没收违法所得
   ✅ 正确答案：A、B、C、E  
   📝 解析：根据《行政处罚法》第9条规定，行政处罚的种类包括...拘役属于刑罚，不是行政处罚。

3. 【判断题】正当防卫明显超过必要限度造成重大损害的，应当负刑事责任，但是应当减轻或者免除处罚。
   ✅ 正确答案：正确
   📝 解析：《刑法》第20条第2款规定...
```

```
用户：分析一个执法案例
助手：案例：民警接警处理邻里纠纷，一方当事人轻微受伤。

分析要点：
1. 现场处置：首先制止冲突，保护现场，救助伤者
2. 证据收集：记录现场情况，拍照录像，收集证人证言
3. 法律适用：判断是否构成治安案件或刑事案件

5. 后续处理：根据伤情决定调解或立案调查

易错点：忽视证据固定、程序不规范
```

### 10. 考试技巧
```
用户：有什么考试技巧？
助手：考试技巧建议：

1. 时间分配：单选题每题不超过1分钟，案例分析留足时间
2. 审题技巧：注意题干中的"不正确"、"除外"等关键词
3. 多选题策略：不确定的选项宁可不选，避免倒扣分
4. 案例分析：先看问题，再读案例，抓住争议焦点
5. 检查重点：检查个人信息、答题卡填涂、有无漏题
```
