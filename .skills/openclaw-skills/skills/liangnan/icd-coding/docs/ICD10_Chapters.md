## 第2章 肿瘤（C00-D48）

### 分类原则
- C00-C97：恶性肿瘤
- D00-D09：原位肿瘤
- D10-D36：良性肿瘤
- D37-D48：动态未定或未知肿瘤

### 重点类目
| 类目 | 名称 | 编码范围 |
|------|------|----------|
| C00-C14 | 唇、口腔和咽部恶性肿瘤 | C00-C14 |
| C15-C26 | 消化器官恶性肿瘤 | C15-C26 |
| C30-C39 | 呼吸和胸腔内器官恶性肿瘤 | C30-C39 |
| C40-C41 | 骨和关节软骨恶性肿瘤 | C40-C41 |
| C43-C44 | 皮肤黑色素瘤和其他恶性肿瘤 | C43-C44 |
| C50 | 乳腺恶性肿瘤 | C50 |
| C51-C58 | 女性生殖器官恶性肿瘤 | C51-C58 |
| C60-C63 | 男性生殖器官恶性肿瘤 | C60-C63 |
| C64-C68 | 泌尿器官恶性肿瘤 | C64-C68 |
| C69-C72 | 脑和神经系统恶性肿瘤 | C69-C72 |

### 肿瘤编码要点
1. **原发肿瘤**：首先编码原发部位
2. **继发肿瘤**：转移灶编码为C77.x
3. **原位癌**：D00-D09
4. **交界性肿瘤**：D39.1（卵巢）、D39.7（其他）

## 常用编码注意事项

### 1. 糖尿病编码要点
- 必须区分1型和2型
- 如有并发症，应同时编码并发症

### 2. 肿瘤编码要点
- 首次诊断以原发灶为主诊断
- 转移灶编码为C77.x
- 原位癌为D00-D09

### 3. 高血压编码要点
- 原发性高血压：I10
- 继发性高血压：I15.x
- 合并心力衰竭/肾病等：分别编码

### 4. 损伤编码要点
- 主要编码损伤部位
- 开放性/闭合性需区分
- 多部位损伤以最严重部位为主
