# 2024-2025年ICD编码研究进展与实践

> 基于病案编码实践的研究总结
> 更新时间：2026年3月

---

## 一、DRG/DIP支付下的编码新要求

### 1.1 医保结算清单规范（2024版）

根据国家医保局《医疗保障基金结算清单》要求：

| 项目 | 新要求 |
|------|--------|
| **主要诊断** | 必须选择导致患者本次住院主要原因的疾病/伤害 |
| **主要手术** | 必须与主要诊断相对应 |
| **其他诊断** | 应填尽填，体现患者全部诊疗情况 |
| **离院方式** | 明确区分医嘱离院/转院/死亡等 |

### 1.2 DRG分组核心要点

```
DRG分组逻辑：
1. 主要诊断 → MDC（主要诊断大类）
2. 主要手术/操作 → 外科/内科ADRG
3. 合并症/并发症 → 细分DRG
4. 年龄/新生儿体重 → 最终分组
```

## 二、主要诊断选择最新原则

### 2.1 "三最原则"详解

| 原则 | 解释 | 示例 |
|------|------|------|
| **对患者健康危害最大** | 疾病严重程度最高的 | 糖尿病并发肾衰竭 vs 糖尿病 |
| **消耗医疗资源最多** | 费用/住院时间最长的 | 恶性肿瘤手术 vs 良性肿瘤切除 |
| **住院时间最长** | 康复时间最久的 | 脑梗死后遗症 vs 脑梗死恢复期 |

### 2.2 常见病主要诊断选择（2024版）

#### 【案例1：恶性肿瘤】
```
❌ 错误：肺恶性肿瘤史 Z85.101
✅ 正确：肺恶性肿瘤 C34.900
     （本次住院为手术治疗）
```

#### 【案例2：糖尿病】
```
❌ 错误：2型糖尿病 E11.900
✅ 正确：2型糖尿病性酮症酸中毒 E11.001
     （本次住院主要治疗酮症酸中毒）
```

#### 【案例3：心血管介入】
```
❌ 错误：冠心病  I25.103
✅ 正确：冠状动脉粥样硬化性心脏病 
        + 经皮冠状动脉支架置入术
```

### 2.3 主要诊断"五个不能"

| 序号 | 不能作为主要诊断 |
|------|------------------|
| 1 | 症状/体征（除非病因不明） |
| 2 | 入院前已存在的慢性病 |
| 3 | 异常检查结果（待查/复查） |
| 4 | 手术/操作并发症（作为其他诊断） |
| 5 | 医疗机构获得性疾病（作为其他诊断） |

## 三、手术/操作编码新规

### 3.1 编码优先顺序

```
手术编码选择原则：
1. 主要手术 → 与主要诊断对应
2. 手术优先于操作
3. 复杂手术优先于简单手术
4. 根治性手术优先于姑息性手术
5. 多个手术 → 选择价值最高的
```

### 3.2 介入治疗编码要点（2024版）

| 手术类型 | 编码示例 | 注意事项 |
|----------|----------|----------|
| 冠脉支架 | 00.66/36.06/36.07 | 区分裸支架/药物支架/药物球囊 |
| 起搏器 | 37.80-37.83 | 区分单腔/双腔/三腔 |
| 射频消融 | 37.33/37.34 | 区分房颤/室上速/室速 |
| 介入栓塞 | 39.79 | 区分血管/肿瘤/止血 |

## 四、肿瘤编码最新规范

### 4.1 肿瘤相关编码顺序

```
正确编码顺序：
1. 主要诊断：原发恶性肿瘤 C77.- / C80
2. 其他诊断：继发恶性肿瘤
3. 治疗方式：手术/放疗/化疗编码
4. 肿瘤分期：TNM分期（作为其他诊断）
5. 形态学编码：/后缀
```

### 4.2 肿瘤编码示例

| 病情 | 主要诊断 | 形态学编码 |
|------|----------|------------|
| 肺腺癌手术 | 肺腺癌 C34.900 | 8140/3 |
| 骨转移 | 继发性骨恶性肿瘤 C79.500 | 原发癌形态学编码 |
| 化疗期间 | 恶性肿瘤维持性化疗 Z51.103 | - |

## 五、编码常见错误与质控（2024版）

### 5.1 高频错误类型

| 错误类型 | 占比 | 纠正方法 |
|----------|------|----------|
| **漏编并发症** | 35% | 仔细阅读病程录/手术记录 |
| **主要诊断错误** | 28% | 应用"三最原则"重新判断 |
| **手术编码不当** | 20% | 对照ICD-9-CM-3正确选择 |
| **编码过于笼统** | 12% | 使用最具体编码 |
| **其他诊断遗漏** | 5% | 应填尽填原则 |

### 5.2 编码质控检查清单

```
□ 主要诊断是否体现本次住院主要原因？
□ 主要诊断与主要手术是否对应？
□ 是否有漏编的并发症/合并症？
□ 肿瘤患者是否同时编码形态学？
□ 损伤中毒外部原因是否填写？
□ 入院病情与诊断是否一致？
□ 离院方式是否正确选择？
```

## 七、编码查询技巧

### 7.1 主导词查找法

```
步骤：
1. 确定疾病/手术的主导词
2. 在索引中查找
3. 核对分类章/亚目
4. 确认编码正确性

示例：病毒性心肌炎
主导词：心肌炎
索引：心肌炎-病毒性 I40.001
核对：I40 急性心肌炎
```

### 7.2 医保版vs临床版对照

| 医保版特点 | 临床版特点 |
|-----------|-----------|
| 代码更细致 | 代码相对简化 |
| 扩展码多 | 基础码为主 |
| 适应DRG/DIP | 适应病案统计 |
| 查询：code.nhsa.gov.cn | 查询：医院信息系统 |
