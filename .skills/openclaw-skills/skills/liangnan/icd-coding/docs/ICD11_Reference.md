# ICD-11 参考指南

> 本文档基于 ICD-11（MMS/Mortality and Morbidity Statistics）参考指南编制，系统介绍 ICD-11 与 ICD-10 的主要区别、编码结构、新增内容及在中国的实施进展。
>
> 📚 ICD-11 于 2022 年被 WHO 正式发布（ICD-11 for Mortality and Morbidity Statistics, MMS），2023年起在全球逐步实施应用。

---

## 一、ICD-11 概述

### 1.1 发布时间线

| 时间 | 事件 |
|------|------|
| 2010年 | ICD-11 开发工作启动 |
| 2018年 | ICD-11 提交至 WHO 成员国预览 |
| 2019年 | ICD-11 在 WHO 网站发布（基础版本） |
| 2022年 | ICD-11 MMS（死亡与疾病统计版）正式通过 |
| 2023年起 | 全球逐步过渡实施 |
| 预计2027+ | 中国正式全面启用 ICD-11 |

### 1.2 ICD-11 vs ICD-10 核心区别

| 对比维度 | ICD-10 | ICD-11 |
|----------|--------|--------|
| **编码结构** | 4位数字/字母+数字 | 字母数字混排，可达10+位 |
| **层级深度** | 3-4层（类目/亚目/细目） | 深层嵌套结构（Cluster coding）|
| **颗粒度** | ~14,000 条目 | ~17,000+ 条目 |
| **疾病分类** | 按病因/解剖 | 多维分类（病因+解剖+病理）|
| **扩展码** | 有限 | 支持_extension_（扩展码）|
| **在线工具** | 无 | 高度数字化，在线协作平台 |
| **更新机制** | 5-10年大版本更新 | 持续更新 |
| **死亡编码** | 单独列表 | 整合于同一分类体系 |

## 二、ICD-11 编码结构

### 2.1 编码格式

ICD-11 采用**字母+数字**格式：

```
主体码格式：[字母][数字][.][数字]...
扩展码格式：主体码 + & + 扩展内容
```

**示例：**

| ICD-11 | 名称 |
|--------|------|
| 1A00 | 霍乱 |
| 1A01 | 霍乱due to Vibrio cholerae 01 |
| BB20 | 急性髓性白血病（AML） |
| BB20.0 | 急性髓性白血病伴成熟分化 |
| 1A00.1 | 霍乱（急性） |

### 2.2 ICD-11 章节结构（27章）

| 章节 | 编码范围 | 内容 |
|------|---------|------|
| 第1章 | 1A00-1Z | 某些感染或寄生虫病 |
| 第2章 | 2A00-2F | 肿瘤 |
| 第3章 | 3A00-3C | 血液或造血器官疾病 |
| 第4章 | 5A00-5C | 内分泌、营养或代谢疾病 |
| 第5章 | 6A00-6D | 精神和行为障碍 |
| 第6章 | 8A00-8E | 神经系统疾病 |
| 第7章 | 9A00-9F | 眼或附器疾病 |
| 第8章 | AA00-AF | 耳或乳突疾病 |
| 第9章 | BA00-BF | 循环系统疾病 |
| 第10章 | CA00-CF | 呼吸系统疾病 |
| 第11章 | DA00-DF | 消化系统疾病 |
| 第12章 | EA00-EB | 皮肤疾病 |
| 第13章 | FA00-FC | 肌肉骨骼或结缔组织疾病 |
| 第14章 | GA00-GF | 泌尿生殖系统疾病 |
| 第15章 | JA00-JC | 妊娠、生产或产褥期 |
| 第16章 | KA00-KB | 起源于围生期的情况 |
| 第17章 | LA00-LC | 先天性畸形、变形或染色体异常 |
| 第18章 | MA00-MC | 精神和行为障碍（症状、体征） |
| 第19章 | NA00-NB | 损伤、中毒或外因后果 |
| 第20章 | PA00-PE | 疾病或死因的外因 |
| 第21章 | QA00-QF | 影响健康状态或与卫生保健接触的因素 |
| 第22章 | RA00-RA | 特殊目的编码（COVID-19等） |
| 补充章 | SA00-ST | 传统医学补充章节 |
| 功能章 | VB00-VC | 功能状态评估（ICF衔接） |

## 三、ICD-11 核心新特性

### 3.1 Cluster Coding（簇编码）

ICD-11 最重要的创新之一，允许**多个编码组合使用**。

**结构：**
```
主要编码 / 附加编码（用 / 连接）
```

**示例：**
```
2A31.1 & XN5VJ & XH6BG1
  │         │         │
  │         │         └── 形态学扩展（SRPT突变）
  │         └── 生化扩展（分子异常）
  └── 主要编码（急性髓性白血病）
```

**临床应用：**
- 肿瘤：部位码 + 形态学码（/前为解剖部位，/后为组织学）
- 感染：病因码 + 抗菌药物敏感性码
- 中毒：物质码 + 外部原因码

### 3.2 Extension Codes（扩展码）

用于提供额外信息，不改变主体分类：

| 扩展类型 | 前缀 | 示例 |
|----------|------|------|
| 严重程度 | ! | 病情严重程度标注 |
| 抗菌药物耐药性 | & | MRSA感染标注 |
| 软组织/皮肤损伤程度 | @ | 烧伤程度标注 |
| 治疗敏感性 | ^ | 肿瘤靶向治疗敏感性 |
| 分子异常 | XN... | 基因突变标注 |

**示例：**
```
急性细菌性脑膜炎，无耐药性
= 1C41.0 & XN2KJ（无耐药性）

急性细菌性脑膜炎，MRSA
= 1C41.0 & XN3DU（MRSA）

COVID-19肺炎，重症
= RA01.0 & !A（重症）
```

### 3.3 基础编码与簇编码的关系

```
ICD-10 思维：  疾病 → 单一编码
ICD-11 思维：  疾病 → 主体编码 + 扩展信息（簇编码/扩展码）
```

## 四、主要章节变化

### 4.1 肿瘤分类（第二章）

| 变化 | ICD-10 | ICD-11 |
|------|--------|--------|
| 结构 | C00-D48（部位）+ 形态学单独列表 | 整合为同一树状结构 |
| 编码 | C34（小细胞肺癌）单独列表 | 肿瘤部位+形态学组合 |
| 淋巴瘤 | 分散于多个类目 | 整合为血液肿瘤大类 |

**ICD-11 肿瘤编码示例：**

| 肿瘤类型 | ICD-11 | ICD-10 |
|---------|--------|--------|
| 肺鳞状细胞癌 | 2C22.0 & XH3XC5 | C34.3 + M8070/3 |
| 急性髓性白血病 | BB20 | C92.0 |
| 乳腺癌 | 2C60-2C6F（含亚目） | C50 + 形态学 |

### 4.2 心脏病学

| 新分类 | 说明 | ICD-10对应 |
|--------|------|-----------|
| 心脏瓣膜病统一分类 | 按瓣膜+病因+病理分类 | 分散于I34-I37 |
| 心肌病 | 新增遗传性心肌病详细分类 | Q24.4/I42 |
| 心律失常 | 新增猝死/心脏骤停分类 | I46/I49 |

### 4.3 神经系统

| 新增分类 | 说明 |
|----------|------|
| 痴呆 | 新增血管性痴呆、路易体痴呆等详细亚目 |
| 头痛 | 大幅扩展（原发性/继发性头痛各数十种亚目）|
| 癫痫 | 新增发作类型详细分类，与临床对应更好 |
| 运动障碍 | 扩展帕金森病及相关综合征分类 |

### 4.4 精神与行为障碍（第六章）

ICD-11 精神障碍章节的主要变化：

| 障碍类型 | ICD-11变化 |
|----------|-----------|
| 抑郁症 | 新增持续性抑郁障碍（恶劣心境）分类调整 |
| 双相障碍 | 细化分型，与临床更好对应 |
| 精神分裂症 | 保留F20，细化症状维度描述 |
| PTSD | 症状标准更接近ICD-11临床标准 |
| 游戏障碍 | **新增**「 Gaming disorder 」（游戏障碍）6C51 |
| 赌博障碍 | **新增** 6C51.0 |
| 自闭症谱系 | 与DSM-5更趋一致 |

### 4.5 传统医学补充章节（补充章）

ICD-11 **新增第23章（补充章）** —— 传统医学模块：

| 编码范围 | 内容 |
|----------|------|
| SA00-SA8 | 传统医学病证（中医/韩医/印度医学等）|
| SB00-SC | 传统医学证候分类 |

**意义：** 传统医学（特别是中医）疾病分类**首次**纳入WHO官方疾病分类体系。

## 五、ICD-11 在中国的实施

### 5.1 中国ICD-11工作进展

| 时间 | 进展 |
|------|------|
| 2019年 | WHO ICD-11中文翻译版完成 |
| 2021年 | 国家卫健委启动ICD-11本地化研究 |
| 2022年 | ICD-11 MMS正式发布 |
| 2023年 | 部分医院开始ICD-11试点 |
| 2025-2027年 | 预计完成本地化，试点扩大 |
| 2027+ | 预计逐步替代ICD-10 |

### 5.2 中国ICD-11本地化工作

1. **中文翻译**：完成ICD-11官方中文翻译
2. **扩展码制定**：制定适合中国国情的扩展码体系
3. **与现有编码映射**：建立ICD-10与ICD-11对照表
4. **DRG/DIP衔接**：ICD-11实施对现行DRG/DIP分组的影响研究

## 六、ICD-11 编码规则

### 6.1 基本编码规则

1. **始终首先使用最具体的编码**
2. **优先使用主体码，再考虑扩展码**
3. **感染性疾病需标注病原体**（如可能）
4. **肿瘤必须使用簇编码**（部位+形态学）
5. **多重分类用 / 连接**（簇编码）

### 6.2 编码优先级

```
1. 已知病因 > 未知病因
2. 已知解剖部位 > 未知部位
3. 特指 > 非特指
4. 主体码 > 扩展码（扩展码不改变主体分类）
```

### 6.3 特殊编码情况

**死亡编码（MR32规则）：**
- 死因编码需追溯至根本死因
- ICD-11 MMS 专用于死亡编码
- 编码规则与临床版略有不同

**COVID-19编码：**
```
ICD-11（COVID-19）:
  RA01.0  2019冠状病毒病（COVID-19）
  RA01.0 & !A  重症COVID-19
  RA01.1  COVID-19后遗症（后COVID-19状态）
```

## 七、ICD-11 在线工具与平台

### 7.1 WHO ICD-11浏览器

- **网址**：https://icd.who.int/browse/11
- **功能**：在线查找编码、查看分类结构
- **语言**：支持多语言（含中文）

### 7.2 ICD-11 MMS API

- WHO提供ICD-11编程接口（API）
- 支持机构接入电子病历系统
- 支持自动化编码辅助

### 7.3 编码质量审核

| 审核要点 | ICD-11要求 |
|----------|-----------|
| 编码完整性 | 是否使用了适当的颗粒度 |
| 簇编码完整性 | 肿瘤是否包含部位+形态学 |
| 扩展码使用 | 是否需要扩展码补充信息 |
| 编码一致性 | 与临床诊断描述是否一致 |

## 八、ICD-11 与 ICD-10 主要编码差异示例

### 8.1 常见疾病编码对照

| 疾病名称 | ICD-10 | ICD-11 |
|---------|--------|--------|
| 霍乱 | A00.0 | 1A00 |
| 1型糖尿病 | E10 | 5A10 |
| 2型糖尿病 | E11 | 5A11 |
| 原发性高血压 | I10 | BA00-BA05 |
| 急性心肌梗死 | I21 | BA80 |
| 脑梗死 | I63 | 8B11 |
| 肺癌 | C33-C34 | 2C22 |
| 乳腺癌 | C50 | 2C60-2C6F |
| 抑郁症 | F32 | 6A41 |
| 阿尔茨海默病 | G30 | 6D80 |
| 慢阻肺 | J44 | CA22 |
| 阑尾炎 | K35-K37 | DD50 |

### 8.2 ICD-10 灰码在 ICD-11 中的变化

| ICD-10灰码 | ICD-11处理 |
|-----------|-----------|
| A49.9（细菌感染，未特指）| 保留，但鼓励使用具体病原体编码 |
| I10（高血压）| 细分为BA00（原发）、BA01-BA05（继发等）|
| J18.9（肺炎，未特指）| 鼓励使用更具体分类 |
| E14（糖尿病，未特指）| 取消，要求区分1型/2型/其他 |

> ⚠️ **重要提示**：ICD-11对颗粒度要求更高，许多ICD-10中的"未特指"编码在ICD-11中被**删除或限制使用**，推动更精确的编码。

## 九、ICD-11 对医院的影响

### 9.1 编码员面临的挑战

| 挑战 | 说明 |
|------|------|
| **编码逻辑转变** | 从"一病一码"到"簇编码"思维 |
| **培训成本** | 需全面培训ICD-11新分类体系 |
| **信息系统改造** | 病案系统、HRP、DRG系统均需升级 |
| **数据迁移** | 历史ICD-10数据与ICD-11对照映射 |
| **DRG/DIP重新分组** | 新编码体系下DRG/DIP分组规则需重新制定 |

### 9.2 应对策略

1. **提前准备**：了解ICD-11结构，开始内部培训
2. **系统升级**：联系HIS/病案系统供应商规划升级
3. **建立对照表**：建立ICD-10与ICD-11本地对照表
4. **试点运行**：在部分地区/医院先行试点
5. **持续学习**：关注国家卫健委ICD-11实施动态
