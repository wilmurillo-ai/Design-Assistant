## 一、ICD-O编码概述

### 1.1 ICD-O组成结构

ICD-O编码采用** topography（部位编码）+ morphology（形态学编码）** 双维度编码系统：

```
ICD-O编码 = 部位编码(T) + 形态学编码(M) + 动态编码(Behavior) + 等级编码(Grade)
     示例：C50.914  乳腺浸润性导管癌
          ↓         ↓
        部位       形态学
```

### 1.2 编码格式

| 组成部分 | 位置 | 说明 |
|----------|------|------|
| **Topography (T)** | 第1位 | 肿瘤发生器官/部位 |
| **Topography (T)** | 第2-4位 | 亚部位 |
| **Morphology (M)** | 第5-6位 | 肿瘤组织学类型 |
| **Morphology (M)** | 第7-8位 | 肿瘤具体类型 |
| **动态编码** | 第9位 | 肿瘤生长方式 |
| **等级编码** | 第10位 | 肿瘤分化程度 |

### 7.1 新增编码

- 免疫检查点抑制剂相关编码
- CAR-T细胞治疗相关编码
- 分子病理检测相关编码
- 肿瘤液体活检相关编码

- 神经内分泌肿瘤分级系统更新
- 淋巴造血系统肿瘤分类更新（WHO 2022标准）
- 软组织肿瘤分类更新

### 7.3 常用ICD-O-4编码查询路径

```
步骤1：确定肿瘤部位（T编码）
  ↓
步骤2：确定组织学类型（M编码）
  ↓
步骤3：确定动态编码（/0-/6）
  ↓
步骤4：确定分化等级（/1-/4）
```
