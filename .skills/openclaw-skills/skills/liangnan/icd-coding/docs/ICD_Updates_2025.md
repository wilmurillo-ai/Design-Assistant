## 一、ICD编码体系概述

### 1.1 主要编码体系

| 编码体系 | 全称 | 应用场景 |
|----------|------|----------|
| **ICD-10** | 国际疾病分类第10版 | 疾病诊断编码（全球通用） |
| **ICD-10-CM** | 美国临床修改版 | 美国医疗结算 |
| **ICD-11** | 国际疾病分类第11版 | WHO新版，2022年起逐步实施 |
| **ICD-9-CM-3** | 美国第9版临床修改第3卷 | 手术操作编码（中国使用） |
| **ICD-O** | 肿瘤学分类 | 肿瘤疾病编码 |

### 1.2 中国医保版ICD编码

中国使用的是**国家医保局版ICD编码**：

- **疾病编码**：基于ICD-10，医保局略有扩展
- **手术编码**：基于ICD-9-CM-3，医保局版本
- **查询网站**：https://code.nhsa.gov.cn/

## 五、常见编码问题解答

### 5.1 医保版 vs 临床版区别

| 维度 | 医保版 | 临床版 |
|------|--------|--------|
| **发布机构** | 国家医保局 | 国家卫健委 |
| **版本** | 基于ICD-10国标版 | 基于ICD-10临床版 |
| **主要差异** | 扩展码更多 | 更贴近临床诊断 |
| **使用场景** | 医保结算 | 病案首页 |

### 5.2 主要诊断选择原则（2024版）

1. **三最原则**：选择对患者健康危害最大、花费医疗资源最多、住院时间最长的疾病
2. **主要手术原则**：主要手术应与主要诊断相对应
3. **出院情况**：选择患者出院的疾病

### 5.3 编码常见错误

| 错误类型 | 正确做法 |
|----------|----------|
| 笼统编码 | 使用具体编码（如D25.9→D25.1） |
| 漏编并发症 | 体现并发症的因果关系 |
| 手术编码错误 | 参考ICD-9-CM-3正确选择 |
