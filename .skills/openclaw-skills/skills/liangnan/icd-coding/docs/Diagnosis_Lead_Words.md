# 诊断主导词查找路径大全

> 本文档基于上传的《诊断主导词.xlsx》（51,201条词条）构建，提供ICD-10编码的主导词快速查找功能。支持：
> - ✅ 主导词 → ICD-10编码（精确/模糊）
> - ✅ ICD-10编码 → 主导词（反向查找）
> - ✅ 完整多级查找路径展示
>
> ⚡ 实时查询请使用：`docs/icd_lookup_tool.py`

## 一、主导词查找方法

### 1.1 查找步骤

```
第1步：确定主导词
    → 疾病性质优先："炎"、"瘤"、"病"、"综合征"
    → 解剖部位其次："肺"、"胃"、"心"
    → 病因/性质最后："急性"、"慢性"、"细菌性"
第2步：在诊断主导词表中查找主导词
第3步：核对ICD-10编码与疾病描述是否匹配
第4步：如需更精确，查看子级路径（一级~七级）
```

### 1.2 主导词选择原则

| 优先级 | 词类 | 示例 |
|:------:|------|------|
| 1 | 疾病本质 | 炎、瘤、癌、病、综合征 |
| 2 | 解剖部位 | 肺、胃、心、脑、肝 |
| 3 | 病因/性质 | 急性、慢性、细菌性、病毒性 |

### 1.3 ICD-10编码格式

- **类目**（3位）：`I20`（心绞痛）
- **亚目**（4位）：`I20.0`（不稳定性心绞痛）
- **细目**（5位）：`I20.00`（无ST段抬高心绞痛）

## 三、主导词前缀/后缀说明

| 符号 | 含义 | 示例 |
|------|------|------|
| `—` | 一级子项 | 表示主导词的第一级细分 |
| `——` | 二级子项 | 表示第二级细分 |
| `见` | 参见 | 需跳转至另一主导词 |
| `另见` | 另请参见 | 参考另一主导词 |
| `NEC` | 不可归类他处 | 表示"未在他处分类" |

### "见"与"另见"的区别

| 术语 | 含义 | 处理方式 |
|------|------|----------|
| **见（SEE）** | 必须跳转 | 跳转到指定的主导词重新查找 |
| **另见（SEE ALSO）** | 有条件跳转 | 如当前词条无合适编码，再跳转 |

## 四、特殊编码规则

### 4.1 星剑号编码规则

ICD-10中使用星号编码（`*`）和剑号编码（`†`）：

- **剑号（†）**：表示病因分类
- **星号（*）**：表示临床表现分类

**示例：** 结核性脑膜炎
- 剑号：`A17.0†`（结核性脑膜炎的病因分类）
- 星号：`G01*`（细菌性脑膜炎中的结核性脑膜炎）

> 在病案首页中，剑号编码优先作为主要编码，星号编码作为附加编码

### 4.2 双重分类（复合编码）

某些疾病需要同时编码：
- 慢性病急性发作：慢性编码 + 急性编码
- 疾病 + 并发症：病因编码 + 临床表现编码

### 4.3 肿瘤编码规则

| 情况 | 主要编码 | 附加编码 |
|------|----------|----------|
| 原发肿瘤 | 肿瘤部位（Cxx） | 形态学编码（M-xxx/x） |
| 转移瘤 | 继发性肿瘤（C7x） | 原发部位 |
| 良性/原位瘤 | 良性/原位瘤（Dxx） | 部位 |

### 示例1：肺炎的完整查找路径

```

↓ 索引中查找"肺炎"
↓ 找到多条亚目：

  - 大叶性肺炎 J18.1
```
