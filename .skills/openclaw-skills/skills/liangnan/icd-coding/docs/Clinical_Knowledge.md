## 一、症状学基础

### 1.1 常见症状与鉴别要点

症状是患者主观感受到的异常感觉或现象，是疾病诊断的重要线索。

#### 发热（ Fever, T>37.3°C）

| 类型 | 特点 | 常见病因 |
|------|------|---------|
| 稽留热 | 体温39-40°C，持续数日 | 大叶性肺炎、伤寒 |
| 弛张热 | 体温39°C以上，波动大 | 败血症、风湿热 |
| 间歇热 | 高热+无热期交替 | 疟疾、败血症 |
| 波状热 | 体温渐升渐降 | 布氏杆菌病 |
| 回归热 | 高热期与无热期交替 | 回归热 |
| 不规则热 | 无规律 | 结核病、风湿热 |

**发热的编码提示**：
- 发热为主诉但病因不明→R50.9（发热）
- 感染性发热→A49.9（感染）
- 肿瘤性发热→R50.2（药物热/肿瘤热）

#### 咳嗽与咳痰

| 类型 | 特点 | 提示疾病 |
|------|------|---------|
| 干咳 | 无痰 | 咽炎、支原体肺炎、肺癌 |
| 湿咳 | 有痰 | 支气管炎、肺炎 |
| 脓痰 | 黄/绿脓痰 | 细菌感染、支气管扩张 |
| 铁锈色痰 | 大叶性肺炎特征 | 肺炎链球菌肺炎 |
| 大量脓痰 | 支气管扩张特征 | 支扩、脓胸 |
| 痰中带血 | 咯血 | 肺结核、肺癌、支气管扩张 |

**咳嗽的编码提示**：
- 急性支气管炎→J20.x
- 慢性支气管炎→J42
- 肺炎→J12-J18（按病原体）
- 肺癌咳嗽→C33-C34

#### 胸痛

| 病因 | 特点 | 编码 |
|------|------|------|
| 心绞痛 | 压榨样，劳累诱发 | I20.x |
| 心肌梗死 | 剧烈持续，伴冷汗 | I21-I22 |
| 肺栓塞 | 突感胸痛+呼吸困难 | I26.x |
| 胸膜炎 | 刺痛，咳嗽加重 | J90 |
| 肺癌 | 持续性，伴消瘦 | C33-C34 |
| 带状疱疹 | 沿肋间神经分布 | B02.2 |

#### 腹痛

| 部位 | 常见病因 |
|------|---------|
| 上腹痛 | 胃十二指肠溃疡、急性胃炎、胆囊炎、胰腺炎 |
| 右上腹痛 | 胆囊炎、胆结石、肝炎、肝脓肿 |
| 左上腹痛 | 胰腺炎、胃癌、脾梗死 |
| 脐周痛 | 阑尾炎早期、肠梗阻、肠道蛔虫 |
| 右下腹痛 | 阑尾炎、肠系膜淋巴结炎、宫外孕 |
| 左下腹痛 | 结肠炎、直肠炎、盆腔炎 |
| 全腹痛 | 腹膜炎、肠梗阻 |

**腹痛的编码提示**：
- 急性阑尾炎→K35.x
- 急性胆囊炎→K81.0
- 急性胰腺炎→K85.x
- 消化性溃疡→K25-K27
- 急性腹膜炎→K65.x

#### 水肿

| 类型 | 特点 | 提示疾病 | 编码 |
|------|------|---------|------|
| 心源性水肿 | 对称性下肢肿→全身 | 心力衰竭 | I50.x |
| 肾源性水肿 | 眼睑/面部先肿 | 肾炎、肾病 | N00-N05 |
| 肝源性水肿 | 腹水为主 | 肝硬化 | K74.x |
| 营养不良性 | 下肢水肿+消瘦 | 低蛋白血症 | E40-E46 |
| 粘液性水肿 | 非凹陷性 | 甲减 | E03.x |

## 二、体格检查

### 2.1 生命体征

| 指标 | 正常范围 | 异常提示 |
|------|---------|---------|
| 体温(T) | 36-37.3°C | >38°C发热 |
| 脉搏(P) | 60-100次/分 | 快/慢/不齐 |
| 呼吸(R) | 12-20次/分 | 呼吸困难 |
| 血压(BP) | <140/90mmHg | 高血压诊断 |

### 2.2 心脏检查

**心界扩大**：
- 左心室扩大→靴形心→高血压/主动脉瓣关闭不全
- 右心室扩大→梨形心→二尖瓣狭窄
- 全心扩大→球形心→扩张型心肌病

**心脏杂音**：
| 杂音 | 提示疾病 | 编码 |
|------|---------|------|
| 收缩期杂音(3/6+) | 器质性二尖瓣/主动脉瓣狭窄 | Q23/I35 |
| 舒张期杂音 | 二尖瓣/主动脉瓣关闭不全 | Q23/I35 |
| 连续性杂音 | 动脉导管未闭 | Q25.0 |

### 2.3 肺部检查

**干啰音**：
- 鼾音→支气管炎
- 哮鸣音→哮喘/慢支

**湿啰音**：
- 大水泡音→肺水肿
- 中水泡音→支气管肺炎
- 小水泡音→细支气管肺炎

## 三、辅助检查

### 3.1 实验室检查

#### 三大常规

**血常规**：
| 指标 | 正常 | 异常 | 提示 |
|------|------|------|------|
| WBC | 4-10×10⁹/L | 升高 | 感染/白血病 |
| WBC | | 降低 | 病毒感染/再障 |
| Hb | 男>120g/L 女>110g/L | 降低 | 贫血 |
| PLT | 100-300×10⁹/L | 降低 | ITP/再障 |
| PLT | | 升高 | 骨髓增值性疾病 |

**尿常规**：
| 指标 | 异常 | 提示 |
|------|------|------|
| PRO（蛋白） | +~++++ | 肾炎/肾病 |
| BLD（隐血） | +~++++ | 肾炎/结石/肿瘤 |
| GLU（糖） | + | 糖尿病/肾性糖尿 |
| KET（酮体） | + | 糖尿病酮症/饥饿 |
| WBC | + | 尿路感染 |

**大便常规**：
| 指标 | 异常 | 提示 |
|------|------|------|
| OB（隐血） | + | 消化道出血 |
| 寄生虫 | 阳性 | 寄生虫感染 |

#### 肝功能

| 指标 | 升高 | 提示 |
|------|------|------|
| ALT/AST | ALT>AST | 肝细胞损伤（肝炎） |
| AST>ALT | 心肌损伤/酒精肝 |
| ALP/GGT | 梗阻性黄疸/骨病 |
| TB/DB | 直接↑ | 梗阻性黄疸 |
| TB/DB | 间接↑ | 溶血性黄疸 |
| ALB | 降低 | 慢性肝病/营养不良 |

#### 肾功能

| 指标 | 异常 | 提示 |
|------|------|------|
| BUN | 升高 | 肾功能不全/高蛋白饮食/消化道出血 |
| Cr | 升高 | 肾小球滤过率下降 |
| UA | 升高 | 痛风/肾功能不全 |
| eGFR | 降低 | CKD分期依据 |

#### 血脂

| 指标 | 异常 | 提示 |
|------|------|------|
| TC↑ | 高胆固醇血症 |
| TG↑ | 高甘油三酯血症 |
| LDL-C↑ | 动脉粥样硬化 |
| HDL-C↓ | 代谢综合征 |

#### 血糖

| 指标 | 值 | 提示 |
|------|---|------|
| FPG | >7.0mmol/L | 糖尿病 |
| 2hPG | >11.1mmol/L | 糖尿病 |
| HbA1c | >6.5% | 糖尿病 |
| 空腹血糖受损 | 6.1-7.0mmol/L | 糖尿病前期 |

### 3.2 影像学检查

#### X线（胸片）

| 影像 | 提示 |
|------|------|
| 大叶性实变 | 大叶性肺炎 |
| 肺纹理增粗 | 支气管炎/心衰 |
| 空洞/结节 | 结核/肿瘤 |
| 胸腔积液 | 胸膜炎/心衰/恶性肿瘤 |
| 气胸 | 自发性/创伤性 |
| 心影扩大 | 心脏疾病 |

#### CT/MRI

| 检查 | 适应证 |
|------|--------|
| 头颅CT | 脑出血/梗死/肿瘤 |
| 胸部CT | 肺栓塞/肺癌/肺炎 |
| 腹部CT | 肝胆胰脾肾疾病 |
| 冠脉CTA | 冠心病筛查 |

### 3.3 心电图

| 心电图表现 | 临床意义 | 编码 |
|-----------|---------|------|
| ST段弓背抬高 | 急性心肌梗死 | I21-I22 |
| ST段压低 | 心绞痛/心肌缺血 | I20.x |
| T波倒置 | 心肌缺血/心室肥大 |

## 四、主要疾病诊断标准

### 4.1 心血管疾病

#### 高血压诊断标准

| 分类 | 收缩压(SBP) | 舒张压(DBP) |
|------|------------|-------------|
| 正常 | <120 mmHg | 且 <80 mmHg |
| 正常高值 | 120-139 | 或 80-89 |
| 高血压1级 | 140-159 | 或 90-99 |
| 高血压2级 | ≥160 | 或 ≥100 |

#### 急性冠脉综合征（ACS）

| 类型 | 诊断要点 | 编码 |
|------|---------|------|
| STEMI | ST段弓背抬高+心肌酶升高 | I21.0-I21.3 |
| NSTEMI | ST压低/T波倒置+心肌酶升高 | I21.4 |
| 不稳定性心绞痛 | ST压低/T波+症状+心肌酶正常 | I20.0 |

#### 心力衰竭分类

| 类型 | 诊断要点 | 编码 |
|------|---------|------|
| 急性左心衰 | 突发呼吸困难+心源性休克 | I50.1 |
| 慢性心力衰竭 | 劳力性呼吸困难+体循环淤血 | I50.0/I50.2 |
| 射血分数降低(HFrEF) | EF<40% | I50.0 |
| 射血分数保留(HFpEF) | EF≥50%+舒张功能不全 | I50.0 |

### 4.2 呼吸系统疾病

#### 肺炎分类

| 类型 | 诊断标准 | 编码 |
|------|---------|------|
| CAP | 社区获得性，院外发病 | J18.9 |
| HAP | 入院48h后发病 | J18.9 |
| 大叶性肺炎 | 肺叶实变 | J18.1 |
| 支气管肺炎 | 双肺纹理增粗 | J18.0 |
| 支原体肺炎 | 干咳+胸片+血清学 | J18.8 |

#### 慢阻肺（COPD）诊断

- 症状：慢性咳嗽、咳痰、呼吸困难
- 病史：长期吸烟/粉尘接触
- 肺功能：FEV1/FVC < 70%
- 分期：GOLD 1-4级

### 4.3 消化系统疾病

#### 消化性溃疡

| 类型 | 主要症状 | 编码 |
|------|---------|------|
| 胃溃疡 | 餐后痛 | K25.x |
| 十二指肠溃疡 | 饥饿痛/夜间痛 | K26.x |
| 复合溃疡 | 胃+十二指肠 | K27.x |
| 伴出血 | 呕血/黑便 | K25.4/K26.4 |
| 伴穿孔 | 急性腹膜炎 | K25.5/K26.5 |

#### 急性胰腺炎

| 严重程度 | 诊断标准 | 编码 |
|---------|---------|------|
| 轻症AP | 器官功能正常+无局部/全身并发症 | K85.9 |
| 重症SAP | 器官功能衰竭+局部/全身并发症 | K85.0 |

### 4.4 内分泌疾病

#### 糖尿病诊断标准（WHO 2019）

| 指标 | 切点 |
|------|------|
| 空腹血糖(FPG) | ≥7.0 mmol/L |
| OGTT 2h血糖 | ≥11.1 mmol/L |
| HbA1c | ≥6.5% |
| 随机血糖 | ≥11.1 mmol/L+典型高血糖症状 |

#### 甲状腺疾病

| 疾病 | 临床表现 | 编码 |
|------|---------|------|
| 甲亢 | 怕热多汗+心悸+消瘦+突眼 | E05.x |
| 甲减 | 怕冷+嗜睡+粘液性水肿 | E03.x |
| 亚甲炎 | 甲状腺疼痛+发热+甲亢表现 | E06.1 |
| 桥本甲状腺炎 | 甲减+甲状腺肿大 | E06.3 |

### 4.5 神经系统疾病

#### 脑卒中分类

| 类型 | 诊断依据 | 编码 |
|------|---------|------|
| 缺血性脑卒中 | CT排除出血+DWI高信号 | I63.x |
| 脑出血 | CT高密度影 | I61.x |
| 蛛网膜下腔出血 | CT高密度影（脑池/脑沟） | I60.x |
| TIA | 症状<24h+CT无梗死 | G45.9 |

#### 癫痫诊断要点

| 要点 | 说明 |
|------|------|
| 发作性 | 突然发作，突然停止 |
| 短暂性 | 持续数秒至数分钟 |
| 重复性 | 可反复发作 |
| 刻板性 | 每次发作类似 |
| 发作类型 | 部分性/全面性 |

### 4.6 肾脏疾病

#### CKD分期（GFR标准）

| 分期 | eGFR (ml/min/1.73m²) | 说明 |
|:----:|---------------------|------|
| G1 | ≥90 | 肾功能正常 |
| G2 | 60-89 | 轻度下降 |
| G3a | 45-59 | 轻-中度下降 |
| G3b | 30-44 | 中-重度下降 |
| G4 | 15-29 | 重度下降 |
| G5 | <15 或透析 | 肾衰竭 |

### 4.7 血液系统疾病

#### 贫血分类

| 类型 | MCV | 病因 | 编码 |
|------|-----|------|------|
| 小细胞低色素性 | <80fL | 缺铁/地中海 | D50/D56 |
| 正细胞性 | 80-100fL | 溶血/慢性病/再障 | D55-D59/D60-D64 |
| 大细胞性 | >100fL | 巨幼细胞性/B12/叶酸 | D51-D52 |

## 五、病案信息学核心知识

### 5.1 病案首页主要项目

| 项目 | 填写要求 |
|------|---------|
| 主要诊断 | 参见主要诊断选择规则 |
| 其他诊断 | 入院时存在/住院期间新发生的疾病 |
| 病理诊断 | 病理号+肿瘤形态学编码 |
| 药物过敏 | 填写具体药物名称 |
| 入院/出院时间 | 准确记录 |
| 手术/操作 | 按时间顺序，主要操作优先 |

### 5.2 病案质量评分标准

| 项目 | 分值 | 扣分标准 |
|------|------|---------|
| 主要诊断正确性 | 10分 | 主诊断错误全扣 |
| 其他诊断完整性 | 5分 | 漏填重要诊断扣分 |
| 手术操作完整性 | 5分 | 漏填主要手术扣分 |
| 编码准确性 | 10分 | 编码错误全扣 |
| 病程记录 | 10分 | 记录不及时/不完整 |
| 知情同意书 | 5分 | 缺项/代签 |
| 病案首页 | 5分 | 填写不规范 |

### 5.3 编码审核要点

| 审核项目 | 审核内容 |
|----------|---------|
| 编码是否在类目/亚目范围内 | 检查编码位数 |
| 是否有更具体的编码可用 | 避免过度汇总 |
| 肿瘤是否有形态学编码 | 原位/良性/恶性区分 |
| 多诊断是否有优先顺序 | 编码互斥问题 |
| 是否使用灰码 | 灰码需谨慎使用 |
| 主要诊断与首页一致性 | 编码与诊断名称匹配 |
