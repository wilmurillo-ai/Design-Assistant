## 一、CHS-DRG2.0 概述

### 1.1 版本信息

| 项目 | 内容 |
|------|------|
| 全称 | 国家医疗保障疾病诊断相关分组（China Healthcare Security DRG） |
| 版本 | 2.0版 |
| 发布机构 | 国家医疗保障局 |
| 发布时间 | 2024年7月（医保发〔2024〕XX号） |
| 前身 | CHS-DRG 1.0版（2021年） |
| 应用场景 | 医保按病组付费（DRG/DIP支付改革核心工具）|

### 1.2 分组结构

```
MDC（Major Diagnostic Category）
  ↓ 26个主要诊断类别
ADRG（Adjacent DRG / 核心分组）
  ↓ 细分为
DRG（Diagnosis Related Groups / 诊断相关分组）
```

| 层级 | 名称 | 数量（2.0版）|
|------|------|-------------|
| 第一层 | MDC（主要诊断大类）| **26个**（MDCA～MDCZ，跳过部分字母）|
| 第二层 | ADRG（核心分组）| **409个** |
| 第三层 | DRG（诊断相关分组）| **655组**（含权重细分）|

## 六、并发症/合并症（CC/MCC）分级

DRG分组中，"伴/不伴严重合并症或并发症"是决定DRG细分的核心因素：

### 严重合并症或并发症（MCC）
- 器官衰竭（心衰、呼吸衰竭、肾衰等）
- 严重感染（脓毒症、重症肺炎等）
- 恶性肿疖
- 多发性创伤

### 一般合并症或并发症（CC）
- 糖尿病（无严重并发症）
- 高血压（无严重靶器官损害）
- 轻度感染
- 营养不良

### 不伴合并症或并发症
- 无其他诊断或仅有轻微诊断
- 住院时间短、检查性住院

## 七、低风险组说明

低风险组（Low Risk DRG）是指住院时间短、费用低、死亡率低的DRG组。医保绩效考核中，低风险组死亡率是重要质控指标。

常见低风险DRG包括：
- 单纯性阑尾炎（无并发症）
- 轻症肺炎
- 皮肤良性肿瘤切除
- 无并发症的胆囊结石
- 乳腺良性肿瘤

## 八、ICD编码对DRG分组的影响

### 8.1 主诊断决定MDC

错误的ICD编码可导致进入错误的MDC，进而影响整个DRG分组。

### 8.2 常见编码错误与DRG后果

| 编码错误 | DRG后果 |
|----------|---------|
| 肿瘤原发/转移混淆 | 进入错误ADRG，权重偏差 |
| 糖尿病分型错误（E10/E11）| 进入错误ADRG |
| 心梗分型错误（STEMI/NSTEMI）| 分组权重差异 |
| 损伤类型错误 | 进入错误的MDCR/循环/神经系统 |
| 漏填合并症/并发症 | 进入"不伴并发症"DRG，权重降低 |
| 使用灰码 | 可能无法入组，结算争议 |

## 九、CHS-DRG2.0 vs 1.0 主要变化

| 变化项 | 1.0版 | 2.0版 |
|--------|--------|--------|
| 发布时间 | 2021年 | 2024年7月 |
| ADRG数量 | ~400个 | **409个** |

| ICD-10版本 | 医保版1.1 | **医保版2.0** |
| ICD-9-CM-3版本 | 医保版2020 | **医保版2022** |
