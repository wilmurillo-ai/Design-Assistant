## 四、学习资源说明

### 4.1 资源特点

| 特点 | 说明 |
|------|------|
| **权威性** | 广西医保局官方发布 |
| **系统性** | 22章ICD-10 + 18章ICD-9-CM-3完整体系 |
| **实用性** | 适合医保结算清单编码学习 |
| **地域性** | 广西医保结算要求，可能有地方特色 |

### 4.2 学习建议

1. **按章节系统学习**：从总论开始，逐章掌握
2. **结合实际案例**：将所学知识应用到实际病案编码
3. **关注医保要求**：重点学习与DRG/DIP相关的编码规则
4. **定期复习**：编码规则需要反复记忆

## 五、与其他版本的区别

### 5.1 医保版 vs 临床版

| 维度 | 广西医保版 | 临床版 |
|------|-----------|--------|
| **发布机构** | 广西医保局 | 国家卫健委 |
| **版本** | 基于国标版 | 基于临床版 |
| **适用范围** | 广西医保结算 | 全国病案统计 |

### 5.2 注意事项

1. 广西医保版可能有地方性扩展码
2. 具体编码以当地医保局最新要求为准
3. 建议结合国家医保局编码库使用
