# 主要诊断与主要手术操作选择原则

> 病案首页填报的核心规则 — 依据国家卫生健康委员会相关规定

---

## 第一章 主要诊断选择原则

### 1.1 主要诊断定义

**主要诊断（Principal Diagnosis）** 是指患者本次住院期间**对患者健康危害最大**、**消耗医疗资源最多**、**住院时间最长**的疾病诊断。

> 📌 三最原则：危害最大、资源消耗最多、住院时间最长

---

### 1.2 主要诊断选择基本原则

#### 原则一：危害最大优先

**总则：** 选择对患者健康危害最大、医疗资源消耗最多的疾病作为主要诊断。

| 情况 | 处理 | 示例 |
|------|------|------|
| **疾病 vs 损伤** | 疾病优先 | 糖尿病酮症酸中毒入院 → 主要诊断：糖尿病酮症酸中毒 |
| **并发症 vs 原发病** | 原发病优先 | 糖尿病患者并发肺炎 → 主要诊断：肺炎 |
| **病危 vs 病重** | 病危优先 | 多器官功能衰竭 + 高血压 → 主要诊断：多器官功能衰竭 |
| **恶性肿瘤 vs 其他** | 恶性肿瘤优先 | 肺癌患者合并肺部感染 → 主要诊断：肺癌 |

#### 原则二：消耗资源最多

**总则：** 消耗医疗资源最多的诊断作为主要诊断。

| 情况 | 处理 | 示例 |
|------|------|------|
| **手术 vs 保守治疗** | 手术对应诊断优先 | 急性心肌梗死行PCI → 主要诊断：急性心肌梗死 |
| **重症 vs 轻症** | 重症优先 | 重症肺炎 + 轻度贫血 → 主要诊断：重症肺炎 |
| **复杂 vs 简单** | 复杂疾病优先 | 复杂先心 + 上呼吸道感染 → 主要诊断：复杂先天性心脏病 |

#### 原则三：住院时间最长

**总则：** 住院时间最长的诊断作为主要诊断。

| 情况 | 处理 | 示例 |
|------|------|------|
| **住院期间发现的疾病** | 入院原因优先 | 入院后发现高血压 → 主要诊断：入院原因疾病 |
| **后续发现疾病** | 不作为主要诊断 | 阑尾炎手术后发现糖尿病 → 不作为主要诊断 |

#### 【六】心血管疾病

| 情况 | 主要诊断选择 |

#### 【七】神经系统疾病

| 住院目的 | 主要诊断选择 |
|----------|--------------|

### 1.4 主要诊断选择"负面清单"

> ⚠️ 严禁以下情况作为主要诊断：

| 序号 | 禁止情况 | 正确做法 |
|------|----------|----------|
| 1 | **症状、体征** | 病因诊断（如：发热→感染性疾病） |
| 2 | **检验异常结果** | 病因诊断（如：高血糖→糖尿病） |
| 3 | **手术并发症** | 如因并发症入院，则可选 |
| 4 | **入院前已存在疾病** | 本次新发/加重的疾病 |
| 5 | **医院获得性疾病** | 院内感染等不应作为主要诊断 |
| 6 | **陈旧性病变** | 现患疾病（如：陈旧性心梗→冠心病） |
| 7 | **非本次住院费用** | 与本次住院相关的诊断 |

### 1.5 主要诊断选择示例

#### 示例1：糖尿病酮症酸中毒

```
患者因"多饮、多尿3天，加重1天"入院，
既往有2型糖尿病史，
入院后诊断：2型糖尿病酮症酸中毒
```

✅ **主要诊断：2型糖尿病酮症酸中毒 [E11.1]**

> 解释：糖尿病酮症酸中毒是危及生命的急重症，是本次住院的主要原因

---

#### 示例2：肺癌合并肺部感染

```
患者因"咳嗽、咳痰1周"入院，
既往肺癌病史，
入院后诊断：左肺腺癌 cT2N0M0 IVA期
           肺部感染
```

✅ **主要诊断：左肺腺癌 [C34.1 M8140/3]**

> 解释：肺癌是危害最大的疾病，消耗更多医疗资源

---

#### 示例3：阑尾炎术后切口感染

```
患者因"阑尾切除术后切口红肿、渗液3天"入院，
行切口切开引流术
```

✅ **主要诊断：手术后切口感染 [T81.4]**

> 解释：因术后并发症入院，手术并发症作为主要诊断

---

#### 示例4：脑梗死溶栓后出血转化

```
急性脑梗死患者行静脉溶栓后出现脑出血，
转神经外科手术治疗
```

✅ **主要诊断：脑梗死伴出血转化 [I63.9]**

> 解释：原发疾病为基础，转化为出血仍按原疾病选择

---

#### 示例5：冠心病+高血压+糖尿病

```
患者因"胸闷、胸痛3天"入院，
诊断：冠心病 不稳定型心绞痛
      高血压3级
      2型糖尿病
```

✅ **主要诊断：冠心病 不稳定型心绞痛 [I20.0]**

> 解释：冠心病是危害最大、消耗资源最多的疾病

## 第二章 主要手术操作选择原则

### 2.1 主要手术操作定义

**主要手术操作（Principal Procedure）** 是指患者本次住院期间**对患者健康危害最大**、**手术风险最高**、**技术难度最大**、**消耗医疗资源最多**的手术或操作。

> 📌 四最原则：危害最大、风险最高、技术最难、资源消耗最多

---

### 2.2 主要手术操作选择基本原则

#### 原则一：手术优先级

**总则：** 有手术必有诊断，手术与诊断相匹配。

| 优先级 | 手术类型 | 说明 |
|--------|----------|------|
| 1级 | 根治性手术 | 肿瘤根治、器官切除 |
| 2级 | 修复性手术 | 血管修复、神经修复 |
| 3级 | 姑息性手术 | 造口、引流 |
| 4级 | 诊断性手术 | 活检、探查 |
| 5级 | 操作性手术 | 穿刺、置管 |

#### 原则二：手术与诊断匹配

| 情况 | 主要手术选择 |
|------|--------------|
| 肿瘤手术 | 肿瘤根治术（如：胃癌根治术） |
| 外伤手术 | 损伤修复术 |
| 急诊手术 | 急诊手术（如：阑尾切除术） |
| 择期手术 | 择期手术（如：疝修补术） |

#### 原则三：多次手术选择

| 情况 | 主要手术选择 |
|------|--------------|
| 多部位手术 | 危害最大/难度最高手术 |
| 分期手术 | 本次完成的手术 |
| 手术+操作 | 手术优先于操作 |
| 手术+保守 | 手术作为主要 |

### 2.4 主要手术操作"负面清单"

> ⚠️ 严禁以下情况作为主要手术：

| 序号 | 禁止情况 | 正确做法 |
|------|----------|----------|
| 1 | **诊断性操作** | 治疗性手术优先（如：活检→根治术） |
| 2 | **姑息性手术** | 根治性手术优先 |
| 3 | **简单手术** | 复杂手术优先 |
| 4 | **操作性手术** | 手术优先（如：穿刺→引流） |
| 5 | **分期手术** | 本期完成手术 |
| 6 | **非本次住院手术** | 本次住院实施的手术 |

### 2.5 主要手术操作选择示例

#### 示例1：胃癌根治术

```
患者诊断：胃窦腺癌 cT3N1M0 II期
行"腹腔镜胃癌根治术(D2)胃大部分切除"
```

✅ **主要手术：腹腔镜胃癌根治术 [43.42]**

> 解释：根治性手术是肿瘤治疗的首选手术方式

---

#### 示例2：冠脉支架+造影

```
患者诊断：冠心病 不稳定型心绞痛
行"冠脉造影+前降支支架植入术"
```

✅ **主要手术：冠状动脉药物洗脱支架植入术 [36.07]**

> 解释：支架植入是治疗性手术，优先于诊断性造影

---

#### 示例3：腹腔镜胆囊切除+阑尾切除

```
患者诊断：胆囊结石伴慢性胆囊炎
        急性阑尾炎
行"腹腔镜胆囊切除+腹腔镜阑尾切除"
```

✅ **主要手术：腹腔镜胆囊切除术 [51.22]**

> 解释：胆囊切除是本次住院的主要治疗目的

---

#### 示例4：骨折内固定+清创

```
患者诊断：左股骨骨折
        左小腿软组织挫伤
行"左股骨骨折切开复位内固定术"
        "左小腿软组织清创术"
```

✅ **主要手术：左股骨骨折切开复位内固定术 [79.35]**

> 解释：骨折内固定是治疗骨折的主要手术

---

#### 示例5：脑出血开颅血肿清除

```
患者诊断：右侧基底节区脑出血
        脑疝形成
行"右侧开颅血肿清除术+去骨瓣减压术"
```

✅ **主要手术：开颅血肿清除术 [01.31]**

> 解释：血肿清除是抢救生命的紧急手术

## 第三章 DRG/DIP下的主要诊断/手术选择影响

### 3.1 主要诊断对DRG分组的影响

| 主要诊断类型 | DRG分组影响 |
|--------------|-------------|
| 并发症/合并症 | 进入有并发症组（权重更高） |
| 严重并发症 | 进入严重并发症组（权重更高） |
| 重要合并症 | 影响MDC分组 |
| 肿瘤分期 | 影响C/CI组划分 |

### 3.2 主要手术对DRG分组的影响

| 手术类型 | DRG分组影响 |
|----------|-------------|
| 根治性手术 | 进入手术组（权重更高） |
| 保守治疗 | 进入内科组（权重较低） |
| 手术+操作 | 组合影响分值 |
| 微创手术 | 特殊分组/加成 |

### 3.3 DIP病种选择要点

| 关键因素 | 影响 |
|----------|------|
| 主要诊断 | 决定核心病种 |
| 手术操作 | 决定分型（根治/姑息/保守） |
| 编码准确性 | 影响分值点数 |
| 编码违规 | 扣分/拒付 |

## 第四章 常见错误与纠正

### 4.1 主要诊断常见错误

| 错误类型 | 错误示例 | 正确选择 |
|----------|----------|----------|
| 症状作为主诊 | 发热 | 感染性疾病 |
| 检验异常主诊 | 高血糖 | 糖尿病 |
| 并发症作为主诊 | 除非因并发症入院 | 原发病 |
| 陈旧病作为主诊 | 陈旧性心梗 | 冠心病 |
| 非本次疾病 | 既往高血压 | 本次新发疾病 |

### 4.2 主要手术常见错误

| 错误类型 | 错误示例 | 正确选择 |
|----------|----------|----------|
| 诊断性操作主手术 | 活检 | 根治性手术 |
| 简单手术主手术 | 阑尾切除 | 如合并其他手术 |
| 操作性手术主手术 | 穿刺置管 | 治疗性手术 |
| 非本次手术 | 既往手术史 | 本次手术 |
