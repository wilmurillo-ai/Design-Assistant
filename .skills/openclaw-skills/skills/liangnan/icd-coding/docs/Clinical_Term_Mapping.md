## 二、映射字段说明

每个条目包含7个字段：

| 字段 | 说明 |
|------|------|
| **正名（中文）** | 规范临床术语（首选） |
| **英文名** | 对应英文名称 |
| **又称** | 目前允许使用的非规范性名称 |
| **曾称** | 已淘汰的旧名 |
| **ICD-10编码** | 国家临床版2.0编码 |
| **ICD-10名称** | 编码对应的标准疾病名称 |
| **科室** | 该术语所属的临床科室 |

## 五、查询工具使用

### 命令行查询

```bash
# 临床术语 → ICD-10编码
python icd_lookup_tool.py word 原发性肺癌 --clinical

# ICD-10编码 → 临床术语
python icd_lookup_tool.py code C34 --clinical

# 按科室查询
python icd_lookup_tool.py dept 眼科 --clinical
python icd_lookup_tool.py dept 心内科 --clinical
```

### 在ICD-Coding技能中直接使用

```
临床术语 原发性肺癌
临床术语 急性心肌梗死
ICD临床查码 C34
```

## 六、使用说明

### 国家临床版 vs 医保版区别

| 对比 | 国家临床版ICD-10 | 医保版ICD-10 |
|------|----------------|-------------|
| 发布机构 | 国家卫健委 | 国家医保局 |
| 版本 | 2.0版 | 基于国标版 |
| 扩展码 | 更贴近临床 | 适合医保结算 |
| 病案首页 | 首选临床版 | 也可接受 |

> ⚠️ 编码时注意区分临床版与医保版，不同版本的编码可能存在细微差异

### 编码审核要点

1. **正名优先**：应使用"正名（中文）"作为主要诊断名称
2. **又称合规**：又称虽允许使用，但建议逐步向正名过渡
3. **避免曾称**：曾称为已淘汰名称，病案编码中不应使用
4. **中英文对照**：有英文名时可参考国际编码习惯
5. **星剑号保留**：部分编码含星剑号（如E10.302+H28.0*），应完整填写
