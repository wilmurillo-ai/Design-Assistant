### 1.2 重点章节编码要点

#### 【第1章：某些传染病和寄生虫病 A00-B99】

| 疾病类型 | 编码范围 | 重点 |
|----------|----------|------|
| 肠道传染病 | A00-A09 | 霍乱、痢疾、感染性腹泻 |
| 结核病 | A15-A19 | 肺结核/肺外结核 |
| 病毒性疾病 | A80-A99 | 乙肝、丙肝、艾滋病 |
| 寄生虫病 | B00-B99 | 疟疾、血吸虫病 |

**编码要点**：
- 传染病需注明病原体
- 艾滋病编码：C20 → B24
- 病毒性肝炎需区分急慢性

#### 【第2章：肿瘤 C00-D48】

| 肿瘤类别 | 编码特点 |
|----------|----------|
| 恶性肿瘤 | C00-C97，原发/继发区分 |
| 原位癌 | D00-D09，/2后缀 |
| 良性肿瘤 | D10-D36，/0后缀 |
| 动态未定 | D37-D48，/1后缀 |

**肿瘤编码规则**：
1. 部位编码（C开头的恶性肿瘤）
2. 形态学编码（/后缀）
3. 行为编码（/0良性、/1交界性、/2原位、/3恶性）
4. 分化等级（G1-G4）

#### 【第9章：循环系统疾病 I00-I99】

| 疾病类别 | 编码范围 | 常见病 |
|----------|----------|--------|
| 急性风湿热 | I00-I02 | 风湿性关节炎 |
| 慢性风湿性心脏病 | I05-I09 | 二尖瓣狭窄 |
| 高血压病 | I10-I15 | 高血压性心脏病 |
| 缺血性心脏病 | I20-I25 | 冠心病、心绞痛、心肌梗死 |
| 脑血管病 | I60-I69 | 脑梗死、脑出血 |
| 动脉疾病 | I70-I79 | 动脉粥样硬化 |

**心脑血管病编码要点**：
- 心肌梗死：I21.-（STEMI/NSTEMI区分）
- 脑梗死：I63.-（责任血管定位）
- 高血压：区分原发性/继发性

## 三、编码核心规则

### 3.1 主导词查找法

```
步骤：
1. 找准主导词（疾病/手术的核心词）
2. 在索引中查找
3. 核对分类
4. 确认编码

示例：病毒性心肌炎
主导词：心肌炎
索引：心肌炎-病毒性 I40.001
```

### 3.2 主要诊断选择原则

| 原则 | 说明 |
|------|------|
| **三最原则** | 危害最大、资源消耗最多、住院时间最长 |
| **主要疾病** | 导致本次住院的疾病 |
| **并发症** | 需要治疗的并发症 |
| **手术治疗** | 有手术先选手术 |

### 3.3 肿瘤编码顺序

```
1. 主要诊断：原发恶性肿瘤
2. 其他诊断：继发恶性肿瘤
3. 手术：肿瘤切除术
4. 其他：并发症/合并症
```

## 四、医保结算相关

### 4.1 医保版vs临床版区别

| 维度 | 医保版 | 临床版 |
|------|--------|--------|
| 发布 | 国家医保局 | 国家卫健委 |
| 特点 | 扩展码多 | 基础码为主 |
| 用途 | 医保结算 | 病案统计 |

### 4.2 DRG分组关键

```
DRG分组要素：
├── 主要诊断 → MDC大类
├── 主要手术 → 外科/内科ADRG
├── 合并症/并发症 → 细分DRG
└── 年龄/体重 → 最终分组
```
