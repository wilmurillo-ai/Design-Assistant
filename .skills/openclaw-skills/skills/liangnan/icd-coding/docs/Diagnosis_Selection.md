# 主要诊断编码选择规则

> 本文档依据《主要诊断编码选择规则（黄锋版）》及《病案信息学》编制，详细阐述主要诊断选择的核心原则、具体规则与案例分析。
>
> 核心原则：**三最原则**——患者住院的主要原因；消耗医疗资源最多；影响患者康复最大

---

## 一、核心概念

### 1.1 主要诊断的定义

**主要诊断**是指患者本次住院的主要原因，即导致患者本次住院的主要疾病或情况，包括：
- 症状、体征或异常所见
- 疾病（含并发症）
- 损伤、中毒
- 行为障碍
- 先天畸形
- 其他影响健康的因素

### 1.2 其他诊断的定义

其他诊断是指患者住院期间存在的、影响治疗或增加医疗资源的疾病或情况，包括：
- 入院时已存在但非主要诊断的疾病
- 住院期间新发生或加重的疾病
- 医院感染
- 慢性疾病的急性加重
- 既往史（影响本次治疗）

### 1.3 主要手术操作的定义

主要手术操作是指：
1. 消耗医疗资源最多的手术
2. 反映主要疾病治疗的手术
3. 难度和复杂性最高的手术

---

## 二、主要诊断选择的基本原则

### 原则一：患者住院的主要原因为主诊断

| 情况 | 主要诊断选择 | 说明 |
|------|-------------|------|
| 因A病住院 | A病 | 最常见情况 |
| 因A病并发症B住院 | B病（并发症） | 并发症是主因时 |
| 因院内感染住院 | 院内感染 | 感染是主因时 |
| 因疾病急性发作住院 | 原发疾病 | 急性发作是主因时 |

### 原则二：消耗医疗资源最多的疾病为主诊断

医疗资源包括：床位费、药费、检查费、手术费、护理费等。

| 判断要素 | 说明 |
|----------|------|
| 住院天数 | 主要疾病通常住院时间最长 |
| 手术/操作 | 消耗最多资源的手术对应疾病 |
| 药品费用 | 治疗主要疾病的药品费用最高 |
| 检查检验 | 针对主要疾病的检查检验最多 |

### 原则三：影响患者康复最大的疾病为主诊断

| 情况 | 主要诊断选择 |
|------|-------------|
| 疾病与并发症同时存在 | 选择危害最大、处理最复杂的 |
| 多部位损伤 | 选择最严重的损伤 |
| 治疗目的冲突时 | 选择对生命威胁最大的 |

---

## 三、具体选择规则

### 3.1 一般规则

#### 规则1：病因诊断优先

**诊断书写规范要求**：病因诊断能包括疾病的临床表现时，应以病因诊断作为主要诊断。

| 示例 | 正确主要诊断 | 说明 |
|------|-------------|------|
| 肺炎→肺叶实变 | 细菌性肺炎 | 病因诊断优先 |
| 高血压→肾衰竭 | 高血压性肾病 | 病因诊断优先 |
| 糖尿病→酮症酸中毒 | 1型糖尿病伴酮症酸中毒 | 病因优先，糖尿病为主 |

#### 规则2：急性疾病优先于慢性疾病

| 示例 | 主要诊断 |
|------|---------|
| 慢支急性发作+肺气肿 | 慢性支气管炎急性加重 |
| 慢阻肺+急性加重 | 慢性阻塞性肺疾病急性加重 |
| 慢性肾炎+急性发作 | 慢性肾小球肾炎（急性加重期） |

#### 规则3：手术治疗优先于药物治疗

| 示例 | 主要诊断 |
|------|---------|
| 胆囊结石伴急性胆囊炎→腹腔镜胆囊切除 | 胆囊结石伴急性胆囊炎 |
| 急性心梗→冠脉支架植入 | 急性ST段抬高型心肌梗死 |

#### 规则4：同一疾病急性与恢复期，以急性期为优先

| 示例 | 主要诊断 |
|------|---------|
| 脑梗死恢复期+陈旧性心梗 | 脑梗死恢复期 |
| 脑梗死急性期 | 脑梗死（急性期） |

| 情况 | 主要诊断选择原则 |
|------|-----------------|
| 霍乱 | 霍乱（具体亚型） |
| 细菌性痢疾 | 细菌性痢疾 |

### 3.3 肿瘤的诊断选择

> ⚠️ 这是临床上最复杂的诊断选择之一，规则较多

#### 肿瘤诊断选择矩阵

| 治疗目的 | 主要诊断选择 | 附加诊断 |
|----------|-------------|----------|
| 首次确诊治疗（手术） | **原发肿瘤** | 转移/并发症/继发 |
| 手术+化疗 | **原发肿瘤**（手术肿瘤） | 化疗/放疗作为其他诊断 |
| 化疗为主 | **化疗目的的肿瘤** | 原发肿瘤→其他诊断 |
| 放疗为主 | **放疗目的的肿瘤** | 原发肿瘤→其他诊断 |
| 免疫治疗 | **免疫治疗目的的肿瘤** | 原发→其他诊断 |
| 靶向治疗 | **靶向治疗目的的肿瘤** | 原发→其他诊断 |
| 复发治疗 | **复发性肿瘤** | 原发→其他诊断 |
| 转移治疗 | **转移性肿瘤**（继发） | 原发→其他诊断 |
| 临终关怀 | **临终关怀**（Z51.8） | 肿瘤→其他诊断 |
| 复查（无阳性） | **肿瘤史**（Z85.x） | — |
| 复查（有阳性） | **新发现疾病** | 原发→其他诊断 |
| 并发症治疗 | **并发症** | 肿瘤→其他诊断 |

#### 肿瘤主要诊断选择案例

```
案例1：胃癌手术
患者：张某，男，62岁
住院经过：确诊胃癌，行胃癌根治术
主要诊断：胃恶性肿瘤 C16.9
其他诊断：淋巴结继发恶性肿瘤 C77.2（病理证实时）

案例2：结肠癌术后化疗
患者：李某，女，55岁
住院经过：结肠癌术后3周，入院行FOLFOX方案化疗
主要诊断：结肠癌术后化学治疗 Z51.1
其他诊断：结肠恶性肿瘤 C18.9

案例3：肺癌骨转移放疗
患者：王某，男，68岁
住院经过：肺癌3年，发现骨转移，行局部放疗
主要诊断：骨继发性恶性肿瘤 C79.5
其他诊断：肺恶性肿瘤 C34.9

案例4：胃癌出血急症手术
患者：赵某，男，72岁
住院经过：胃癌并发出血，行急症胃大部分切除术
主要诊断：胃恶性肿瘤伴出血 C16.9
其他诊断：失血性休克（急性出血时）
```

- 急性期：发病4周内（约1个月内）

- 后遗症期：发病6个月以后

## 四、不作为主要诊断的情况

### 4.1 常规不作为主诊断的疾病

| 疾病类型 | 示例 | 说明 |
|----------|------|------|
| 医院感染 | 院内获得性肺炎 | 除非是本次住院主要目的 |
| 既往病史 | 陈旧性心梗 | 除非是本次治疗全部内容 |
| 慢性疾病急性发作 | 慢支急性发作 | 可作为其他诊断 |
| 症状/体征 | 发热、腹痛待查 | 病因明确后以病因为主诊断 |
| 病理结果未出 | 肺部阴影（病理待除外） | 病理确诊后更正 |

### 4.2 可选为主诊断或其他的疾病

| 情况 | 可选主诊断 | 可选其他诊断 |
|------|-----------|-------------|
| 慢性病急性加重 | 慢性病急性加重 | 慢性病 |
| 疾病伴随非主要并发症 | 并发症 | 疾病本身 |
| 既往手术史 | — | 可作为其他诊断 |

## 五、多诊断情况的优先顺序

当患者有多个诊断时，按以下顺序选择主要诊断：

```
第1顺位：病因诊断（如：胆囊结石 → 胆囊炎）
第2顺位：急性疾病（如：急性心梗 → 陈旧心梗）
第3顺位：手术相关（如：术后并发症 → 原发病）
第4顺位：危重疾病（如：休克 → 感染）
第5顺位：消耗资源最多（如：肿瘤 → 合并高血压）
```

## 六、主要手术操作选择原则

### 四最原则

1. **最直接反映主要疾病治疗**的手术
2. **消耗医疗资源最多**的手术
3. **难度和复杂性最高**的手术
4. **手术级别最高**的手术

### 具体规则

| 情况 | 主要手术选择 |
|------|-------------|
| 多个手术同时进行 | 以主要治疗目的的手术为主 |
| 急诊→择期手术 | 以急诊手术为主 |
| 手术→保守治疗 | 以手术为主 |
| 诊断性→治疗性 | 以治疗性为主 |
| 手术并发症 | 以原手术目的为主 |
| 恶性肿瘤手术 | 以根治性切除为主 |

### 手术与操作的区别

| 手术（有创） | 操作（诊断性/治疗性） |
|-------------|---------------------|
| 需要切开/切除/重建 | 经自然腔道或穿刺完成 |
| 消耗更多医疗资源 | 消耗资源相对较少 |
| 通常作为主要手术 | 可作为主要诊断的补充 |

## 八、常见问题解答

**Q1：患者同时有多个疾病，如何选择主要诊断？**
A：选择消耗医疗资源最多、影响康复最大的疾病。优先考虑病因诊断、急性疾病、手术相关疾病。

**Q2：手术后发现其他疾病，如何选择主要诊断？**
A：如果发现的疾病需要继续住院治疗，应作为主要诊断；否则作为其他诊断。

**Q3：肿瘤患者复查如何选择主要诊断？**
A：复查无阳性发现→肿瘤史（Z85.x）；阳性发现→新发现疾病作为主要诊断。

**Q4：慢性病急性发作如何选择？**
A：一般选择慢性病作为主要诊断，急性发作作为其他诊断；但如果急性发作是住院的主要原因，也可选急性发作为主诊断。

**Q5：糖尿病患者入院调糖如何选择？**
A：以糖尿病调糖为目的→糖尿病（按类型）；因糖尿病急性并发症（DKA/HHS）入院→急性并发症为主诊断。

**Q6：病理未回报时如何编码？**
A：先以临床诊断编码，病理回报后及时更正。建议出院前完成病理分型。

**Q7：医疗资源相近时如何选主诊断？**
A：优先选择本科室疾病；本科室治疗为主；本科室医生书写的主要诊断。
