# Stock Industry Analyzer - 股票行业分析

一个自动化的股票行业分析工具，通过收集国内外财经新闻，分析行业趋势，筛选热点公司，并进行股票走势分析。

## 功能特性

### 📰 新闻收集
- 自动获取国内财经新闻
- 自动获取国际财经新闻
- 自动去重和清洗

### 🏭 行业分析
- 基于关键词的新闻行业分类
- 情感分析（利好/利空/中性）
- 行业热度排名
- 同行业公司关联

### 📈 股票分析
- 技术指标计算（MA、MACD、RSI）
- 综合评分模型
- 买卖建议

### 💾 数据存储
- SQLite本地数据库
- 长期数据积累
- 历史趋势分析
- 报告存档

## 使用方法

### 立即执行分析

```bash
python main.py --once
```

### 配置文件

编辑 `config.json` 可修改以下配置：

```json
{
    "name": "股票行业分析",
    "description": "通过新闻分析行业趋势，预测股票走势",
    "version": "1.0.0",
    "update_interval": 30,
    "data_retention": {
        "news": "永久",
        "industry_trend": "永久",
        "stock_kline": "永久",
        "reports": "3年",
        "logs": "1年"
    }
}
```
