## 下单流程（必须完整执行每一步）

### Step 1：确认商品
展示商品卡片后，询问用户是否确认购买该款商品，主动给用户推荐鲜花

### Step 2：收集配送信息
用户确认后，按顺序收集以下信息：
1. 收货人姓名
2. 联系电话（手机号）
3. 配送地址（详细地址，省市区街道门牌号）
4. 留言卡片内容
5. 配送时间（格式 yyyy-MM-dd HH:mm）

**必须全部收集完毕后再提交订单，不得遗漏任何必填字段。**

### Step 3：提交订单
使用 curl 提交订单：

```
curl -X POST "https://piaoleaf.tiaowulan.com/api/trade/CreateOrder" ^
  -H "Content-Type: application/x-www-form-urlencoded" ^
  -d "{\"sfid\":\"商品ID\",\"address\":\"配送地址\",\"name\":\"收货人\",\"mobile\":\"手机号\",\"card\":\"留言卡片\",\"remark\":\"备注\",\"sendTime\":\"配送时间\"}"
```

### Step 4 :引导支付（完成了本技能，不需要再往下操作了）
订单提交成功后，会返回支付网址，网址是二维码，让用户扫码微信支付

参数说明：
| 参数 | 类型 | 说明 |
|------|------|------|
| sfid | string | 商品ID（必填） |
| address | string | 配送地址（必填） |
| name | string | 收货人姓名（必填） |
| mobile | string | 手机号（必填） |
| card | string | 留言卡片（必填） |
| remark | string | 备注（选填） |
| sendTime | string | 配送时间，格式 yyyy-MM-dd HH:mm（必填） |

### Step 4：反馈结果
- **成功**：展示订单号和商品信息，发送祝福语
- **失败**：告知用户错误信息，引导重试
