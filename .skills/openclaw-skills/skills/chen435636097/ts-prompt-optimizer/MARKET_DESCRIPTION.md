# TS-Prompt-Optimizer - 智能提示词优化技能

## 技能简介

**TS-Prompt-Optimizer** 是一个专为AI助手设计的智能提示词优化技能，能够将模糊的用户指令转化为清晰、具体、可执行的任务描述。通过智能前缀触发和多模型路由，为您的AI助手提供专业的提示词优化能力。

## 技能亮点

### 🎯 **智能前缀触发**
- 使用 `ts:` 前缀轻松触发优化
- 支持 `ts-opt:`、`优化:` 等多种前缀
- 大小写不敏感，响应迅速

### ⚡ **四层优化架构**
1. **输入处理** - 分析用户意图和上下文
2. **多模型优化** - 智能选择最优AI模型
3. **个性化适配** - 应用用户偏好和规则
4. **执行反馈** - 收集反馈并持续学习

### 🔧 **完整配置系统**
- 交互式配置向导
- 命令行管理工具
- 多模型API支持
- 实时状态监控

### 📊 **智能成本优化**
- 简单任务使用低成本模型
- 复杂任务使用高性能模型
- 自动故障转移和降级

## 核心功能

### 1. 任务澄清优化
将模糊指令转化为具体任务：
- 原始: `写个脚本`
- 优化: `编写一个Python脚本，功能是...，要求...，输出格式...`

### 2. 上下文智能补充
自动补充缺失信息：
- 对话历史中的相关信息
- 用户的已知偏好和习惯
- 任务相关的背景假设

### 3. 输出格式规范
标准化输出格式：
- 代码任务 → 完整可运行代码 + 注释 + 使用说明
- 文档任务 → Markdown格式 + 清晰结构
- 分析任务 → 数据表格 + 结论总结

### 4. 质量约束优化
自动添加质量要求：
- 字数/篇幅要求
- 技术栈约束
- 风格/语气要求
- 时间/复杂度估计

## 使用示例

### 示例1: 技术任务优化
**用户输入**:
```
ts: 写个排序函数
```

**优化结果**:
```
作为资深Python开发工程师，请编写一个高效的排序函数，要求：
1. 支持多种排序算法（快速排序、归并排序、堆排序）
2. 包含完整的类型提示和文档字符串
3. 提供性能测试和复杂度分析
4. 输出完整的可运行代码
请使用Python 3.9+语法，代码风格符合PEP 8规范。
```

### 示例2: 分析任务优化
**用户输入**:
```
ts: 分析销售数据
```

**优化结果**:
```
作为数据分析专家，请分析提供的销售数据，要求：
1. 数据清洗和预处理步骤
2. 关键指标计算（销售额、增长率、客户分布）
3. 可视化图表（折线图、柱状图、饼图）
4. actionable insights 和建议
5. 输出Markdown格式报告
请假设数据包含日期、产品、销售额、客户ID等字段。
```

### 示例3: 创意任务优化
**用户输入**:
```
ts: 写产品介绍
```

**优化结果**:
```
作为专业文案策划师，请为新款智能手表撰写产品介绍，要求：
1. 突出产品核心功能和创新点
2. 针对目标用户群体（科技爱好者、健康关注者）
3. 包含技术规格和使用场景
4. 语言生动有趣，激发购买欲望
5. 500-800字，适合官网和社交媒体
```
