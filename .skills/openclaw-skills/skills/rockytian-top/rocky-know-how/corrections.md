# 纠正日志 — 模板

> 首次使用此技能时在 `~/.openclaw/.learnings/corrections.md` 创建。
> 保留最近 50 条纠正。旧条目评估后晋升或归档。

## 示例条目

```markdown
## 2026-02-19

### 14:32 — Code style
- **纠正:** "Use 2-space indentation, not 4"
- **上下文:** Editing TypeScript file
- **计数:** 1 (first occurrence)

### 16:15 — Communication
- **纠正:** "Don't start responses with 'Great question!'"
- **上下文:** Chat response
- **计数:** 3 → **PROMOTED to memory.md**

## 2026-02-18

### 09:00 — Project: website
- **纠正:** "For this project, always use Tailwind"
- **上下文:** CSS discussion
- **操作:** Added to projects/website.md
```

## 日志格式

每个条目包括：
- **时间戳** — 纠正何时发生
- **纠正** — 用户说了什么
- **上下文** — 什么触发了它
- **计数** — 发生了多少次（用于晋升跟踪）
- **操作** — 存储在哪里（如果晋升了）
