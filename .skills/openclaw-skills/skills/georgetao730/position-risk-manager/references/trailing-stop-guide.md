# 移动跟踪止盈 (Trailing Stop) 详细指南

## 核心理念

**让利润奔跑，但绝不让盈利变亏损。**

---

## 为什么需要移动跟踪止盈？

### 常见问题

| 问题 | 表现 | 后果 |
|------|------|------|
| **过早止盈** | 涨 10% 就跑，后面涨 50% | 错失大行情 |
| **过晚止盈** | 涨 50% 不卖，跌回成本 | 坐过山车 |
| **预测顶部** | "应该到顶了" | 主观判断，容易出错 |

### 移动跟踪止盈的优势

- ✅ **不预测顶部**：让市场告诉你何时离场
- ✅ **保护盈利**：确保盈利交易不变成亏损
- ✅ **抓住大趋势**：在趋势延续时保持持仓
- ✅ **克服人性**：机械化执行，避免贪婪和恐惧

---

## 移动跟踪止盈的实施方法

### 方法 1：均线跟踪法

**适用**：趋势明显的股票

| 周期 | 均线选择 | 特点 |
|------|----------|------|
| **短线** | 10 日均线 | 紧密跟踪，容易触发 |
| **中线** | 20 日均线 | 平衡，适合大多数情况 |
| **长线** | 60 日均线 | 宽松，抓住大趋势 |

**操作**：
```
股价上涨 → 均线上移 → 止盈位上移
股价下跌 → 均线走平 → 触及止盈位离场
```

**示例**：
```
中国海油 成本 30 元

第 1 周：股价 35 元，20 日均线 33 元 → 止盈位 33 元
第 2 周：股价 38 元，20 日均线 35 元 → 止盈位上移至 35 元
第 3 周：股价 42 元，20 日均线 38 元 → 止盈位上移至 38 元
第 4 周：股价 40 元，跌破 20 日均线 39 元 → 触发止盈，获利了结
```

---

### 方法 2：百分比回撤法

**适用**：波动较大的股票

| 浮盈比例 | 回撤阈值 | 说明 |
|----------|----------|------|
| **20-30%** | -10% | 浮盈较小时，允许较大回撤 |
| **30-50%** | -8% | 浮盈中等，收紧回撤 |
| **50%+** | -5% | 浮盈较大，严格保护 |

**操作**：
```
当前价 × (1 - 回撤阈值) = 止盈位

示例：
股价 42 元，浮盈 40% → 回撤阈值 8%
止盈位 = 42 × (1 - 0.08) = 38.64 元
```

---

### 方法 3：阶段性低点法

**适用**：震荡上行的股票

**识别阶段性低点**：
- 最近一个明显的回调低点
- 成交量萎缩的支撑位
- 前期突破的平台

**操作**：
```
股价突破新高 → 前高变为支撑 → 止盈位上移至前高

示例：
股价从 30 涨到 35 → 回调到 33 → 再涨到 40
止盈位：33 元（阶段性低点）
```

---

### 方法 4：ATR 波动法

**适用**：波动率变化大的股票

**ATR (Average True Range)**：平均真实波动范围

**操作**：
```
止盈位 = 当前价 - N × ATR

N 的选择：
- N=2：紧密跟踪
- N=3：适中
- N=4：宽松
```

**示例**：
```
股价 42 元，ATR=2 元，N=3
止盈位 = 42 - 3×2 = 36 元
```

---

## 移动跟踪止盈的纪律

### 必须遵守的规则

1. **只上移，不下移**
   - 止盈位只能随着股价上涨而上移
   - 绝不能因为股价回调而下移止盈位

2. **触及即执行**
   - 收盘价跌破止盈位 → 立即止盈
   - 不要抱幻想，不要等反弹

3. **盘中和收盘的区别**
   - 盘中跌破：观察，不急于行动
   - 收盘跌破：执行止盈

4. **特殊情况处理**
   - 跳空低开跌破：开盘即止盈
   - 涨停板：继续持有，止盈位上移

## 实战案例

### 案例 1：中国海油（成功）

```
买入：30 元
第 1 月：涨到 35 元，止盈位 33 元
第 2 月：涨到 38 元，止盈位 35 元
第 3 月：涨到 42 元，止盈位 38 元
第 4 月：跌破 38 元，止盈离场

最终盈利：(38-30)/30 = 26.7%
如果不用止盈：可能坐过山车回到 30 元
```

### 案例 2：某妖股（失败但正确）

```
买入：10 元
第 1 周：涨到 15 元，止盈位 13.5 元
第 2 周：涨到 20 元，止盈位 18 元
第 3 周：涨到 25 元，止盈位 22.5 元
第 4 周：跌破 22.5 元，止盈离场

最终盈利：125%
后续股价涨到 40 元？没关系，已经赚了 125%
```

**教训**：移动跟踪止盈不能保证卖在最高点，但能保证你抓住大部分趋势。

## 常见错误

### ❌ 错误 1：止盈位设置过紧

```
股价 42 元，止盈位 41 元（-2%）
结果：正常波动就被震出局
```

**正确**：根据股票波动率设置合理的回撤阈值

---

### ❌ 错误 2：止盈位下移

```
股价 42 元，止盈位 38 元
股价跌到 39 元，把止盈位下移到 37 元
结果：失去保护意义
```

**正确**：止盈位只上移，不下移

---

### ❌ 错误 3：触及不止盈

```
股价跌破止盈位 38 元
"等反弹再卖"
结果：继续跌到 30 元
```

**正确**：触及即执行，纪律第一

## 心理建设

### 克服"卖飞"的恐惧

**真相**：
- 卖飞是赚，不是亏
- 你不可能每次都卖在最高点
- 把鱼尾留给别人，你吃鱼身就够了

**心态**：
```
"我已经赚了 30%，这很好。
如果后面再涨，那是市场的礼物。
如果下跌，我保住了利润。"
```

### 克服"再等等"的贪婪

**真相**：
- 趋势会反转，不会永远上涨
- 盈利变亏损是最痛苦的事
- 移动止盈帮你克服人性弱点

**心态**：
```
"我不预测顶部，让市场告诉我何时离场。
跌破止盈位，就是市场在说：该走了。"
```

---

## 总结

移动跟踪止盈的核心：

1. **不预测顶部** → 让利润奔跑
2. **设置防守位** → 保护盈利
3. **只上移不下移** → 锁定利润
4. **触及即执行** → 纪律第一

**记住**：
> 交易的目的是赚钱，不是证明自己正确。
> 移动跟踪止盈帮你做到：赚的时候多赚，亏的时候少亏。
