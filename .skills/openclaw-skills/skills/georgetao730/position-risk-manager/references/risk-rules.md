# 风控规则清单 (Risk Management Rules)

## 仓位管理铁律

### 1. 单票上限规则

| 账户规模 | 单票上限 | 说明 |
|----------|----------|------|
| **<50 万** | 30% | 小账户可以集中一些 |
| **50-200 万** | 25% | 中等账户适度分散 |
| **>200 万** | 20% | 大账户必须分散 |

**强制执行**：
- 单票占比超过上限 → 强制卖出超额部分
- 不要因为"特别看好"而例外

---

### 2. 行业分散规则

| 规则 | 要求 |
|------|------|
| **单行业上限** | 不超过总仓位的 40% |
| **相关行业合并计算** | 如银行 + 保险 + 证券算金融行业 |
| **周期股合并计算** | 如煤炭 + 钢铁 + 有色算周期行业 |

**原因**：避免行业黑天鹅导致整体亏损

---

### 3. 止损规则

| 交易类型 | 止损位 | 说明 |
|----------|--------|------|
| **短线交易** | -5% | 短线必须严格止损 |
| **中线交易** | -8% | 中线可适度放宽 |
| **长线交易** | -15% | 长线看基本面是否变化 |
| **浮盈交易** | 保本位 | 有浮盈后，止损上移至成本价 |

**纪律**：
- 止损位只上移，不下移
- 触及止损立即执行，不抱幻想

---

### 4. 止盈规则

| 浮盈比例 | 操作 |
|----------|------|
| **10-20%** | 继续持有，止损上移至成本 |
| **20-30%** | 可部分止盈 (20-30%)，或移动跟踪止盈 |
| **30-50%** | 建议部分止盈，收回本金 |
| **50%+** | 倒金字塔止盈，或严格移动跟踪 |

---

### 5. 加仓规则

| 情况 | 操作 |
|------|------|
| **浮盈加仓** | 可以，但加仓后总仓位不超过上限 |
| **浮亏加仓** | 谨慎，最多加仓 1 次，且亏损不超过 -10% |
| **深度套牢** | 禁止加仓，要么止损要么换股 |

**加仓金字塔**：
```
第一批：30% 仓位
第二批：25% 仓位（确认趋势后）
第三批：20% 仓位（趋势加速）
第四批：15% 仓位（不推荐）

越涨越买是错的，应该越涨越卖
```

---

### 6. 再平衡规则

| 频率 | 操作 |
|------|------|
| **季度再平衡** | 每季度检查一次仓位分布 |
| **触发再平衡** | 单票占比超过上限 10% 时 |
| **年度大调整** | 每年末全面评估持仓 |

**再平衡步骤**：
1. 计算各持仓占比
2. 卖出超过上限的部分
3. 将资金分配给低配标的或现金

---

## 特殊情况处理

### 情况 1：连续涨停

| 操作 | 说明 |
|------|------|
| **继续持有** | 涨停板不打开，继续持有 |
| **止盈位上移** | 每天上移至涨停价 |
| **打开涨停** | 成交量放大，考虑部分止盈 |

---

### 情况 2：连续跌停

| 操作 | 说明 |
|------|------|
| **能卖即卖** | 跌停板打开立即卖出 |
| **不要补仓** | 补仓可能越套越深 |
| **评估基本面** | 判断是暂时问题还是永久损失 |

---

### 情况 3：停牌

| 操作 | 说明 |
|------|------|
| **评估影响** | 停牌原因对股价的影响 |
| **准备预案** | 复牌后大涨/大跌的应对方案 |
| **控制仓位** | 停牌股票占比不宜过高 |

---

### 情况 4：分红除权

| 操作 | 说明 |
|------|------|
| **长期持有** | 分红再投资，复利增长 |
| **短期交易** | 注意除权缺口，调整成本价 |
| **税务考虑** | 持股超过 1 年免征红利税 |

| 信号 | 风险等级 | 操作 |

## 心态管理规则

### 1. 亏损后的处理

| 错误做法 | 正确做法 |
|----------|----------|
| 急于翻本，加大仓位 | 暂停交易，调整心态 |
| 频繁交易，试图挽回 | 休息几天，冷静思考 |
| 报复性交易 | 复盘总结，找出问题 |

---

### 2. 盈利后的处理

| 错误做法 | 正确做法 |
|----------|----------|
| 骄傲自满，放松警惕 | 保持谦逊，继续执行纪律 |
| 加大仓位，追求更高收益 | 按原计划执行，不贪心 |
| 改变策略，随意操作 | 坚持原有策略 |

---

### 3. 日常心态

**交易格言**：

> - 市场永远是对的，错的是你
> - 宁可错过，不做错
> - 截断亏损，让利润奔跑
> - 不要和股票谈恋爱
> - 别人贪婪时我恐惧，别人恐惧时我贪婪
> - 活下来，才能等到下一次机会

## 检查清单

### 每次交易前检查

- [ ] 这笔交易的风险收益比是否合理（至少 1:2）？
- [ ] 止损位是否明确？
- [ ] 仓位是否在可控范围内？
- [ ] 是否影响了整体仓位分布？
- [ ] 买入逻辑是否清晰？

### 每周检查

- [ ] 各持仓占比是否超标？
- [ ] 是否有需要止盈/止损的持仓？
- [ ] 行业分布是否合理？
- [ ] 总体仓位是否合适？

### 每月检查

- [ ] 本月盈亏情况如何？
- [ ] 是否有违反纪律的操作？
- [ ] 需要调整策略吗？
- [ ] 下月计划是什么？

---

## 总结

风控的核心不是避免亏损，而是：

1. **控制亏损的幅度** → 止损
2. **保护已有的盈利** → 止盈
3. **避免灾难性损失** → 分散仓位
4. **长期活下来** → 纪律执行

**记住**：
> 一年三倍者如过江之鲫，三年一倍者寥若晨星。
> 持续盈利的关键不是抓住大牛股，而是避免大亏损。
