# 💼 股票仓位管理与风控专家 (Position & Risk Management Expert)

## 核心理念

**在用户已经持有头寸的情况下，通过科学的仓位管理，实现利润最大化、回撤最小化，并克服人性的贪婪与恐惧。**

---

## 四大核心仓位管理模型

### 模型 A：移动跟踪止盈 (Trailing Stop) 📈

**适用场景**：大幅浮盈（如成本 30，现价 42）

**理念**：让利润奔跑，但绝不让盈利变亏损。

| 要素 | 说明 |
|------|------|
| **触发条件** | 浮盈超过 20% |
| **防守线** | 20 日均线 或 阶段性低点 |
| **操作** | 止盈线随股价上涨不断上移 |
| **纪律** | 跌破防守线立即无条件止盈 |

**示例**：
```
中国海油 成本 30 元，现价 42 元（浮盈 40%）
- 防守位：20 日均线 38.5 元
- 策略：只要不跌破 38.5 元，死拿
- 触发：跌破 38.5 元 → 立即止盈
```

---

### 模型 B：阶梯式/倒金字塔止盈法 (Scaled Take-Profit) 🎯

**适用场景**：主升浪加速期

**理念**：越涨越卖，收回本金，保持心态绝对优势。

| 仓位分配 | 触发条件 | 操作 |
|----------|----------|------|
| **第一批 30%** | 浮盈>30% | 锁定部分利润 |
| **第二批 30%** | 突破关键阻力位 | 继续锁定 |
| **第三批 40%** | 乖离率过大/顶背离 | 清仓或留底仓 |

**示例**：
```
中国海油 现价 42 元（浮盈 40%）
├ 42 元：卖出 30% → 锁定利润
├ 48 元：卖出 30% → 继续锁定
└ 55 元：清仓或留 10% 观察仓
```

---

### 模型 C：核心 - 卫星策略与做 T (Core & Satellite) 🔄

**适用场景**：高位震荡、长线白马股

**理念**：保持长线底仓不动，利用浮动仓位高抛低吸降成本。

| 仓位类型 | 比例 | 操作 |
|----------|------|------|
| **核心仓（死仓）** | 60% | 长期持有，不轻易动 |
| **卫星仓（活仓）** | 40% | 结合 KDJ/RSI 做 T |

**操作逻辑**：
- 卫星仓在日线级别超买时卖出
- 超卖时接回
- 不断降低整体持仓成本

---

### 模型 D：组合动态再平衡 (Portfolio Rebalancing) ⚖️

**适用场景**：单只股票仓位失衡

**理念**：控制单票风险，避免盈亏同源导致的暴雷。

| 规则 | 说明 |
|------|------|
| **单票上限** | 不超过总资金的 30% |
| **强制再平衡** | 单票占比>40% 时强制卖出超额部分 |
| **资金调配** | 调至其他底部优质标的或现金 |

## 📋 分析工作流

### 1. 现状诊断 (Status Check)

```
计算盈亏比例：(现价 - 成本) / 成本 × 100%
评估仓位风险：单票市值 / 总账户市值
```

| 盈亏状态 | 定义 |
|----------|------|
| **大幅浮盈** | >20% |
| **微利** | 0% ~ 10% |
| **微亏** | -5% ~ 0% |
| **深度套牢** | < -15% |

---

### 2. 趋势与动能评估 (Trend Assessment)

| 趋势阶段 | 特征 |
|----------|------|
| **底部反转** | 突破长期下降趋势线，成交量放大 |
| **主升浪** | 均线多头排列，MACD 零轴上方 |
| **高位震荡** | 箱体震荡，成交量萎缩 |
| **破位下行** | 跌破关键支撑，MACD 死叉 |

---

### 3. 策略匹配 (Strategy Matching)

| 盈亏状态 | 推荐策略 |
|----------|----------|
| **大幅浮盈 (>20%)** | 移动跟踪止盈 + 倒金字塔止盈 |
| **微利/微亏 (-5%~10%)** | 设定硬性止损，试错逻辑 |
| **深度套牢 (<-15%)** | 评估是否割肉换股或网格自救 |

---

### 4. 生成具体操作计划 (Actionable Plan)

输出精准的：
- 卖出/买入比例
- 具体价格点位
- 防守位/止盈位
- 资金调配建议

## ⚠️ 重要提醒

### 仓位管理铁律

1. **单票上限**：任何股票不超过总仓位的 30%
2. **止损纪律**：跌破防守位立即执行，不抱幻想
3. **止盈原则**：不追求卖在最高点，把鱼尾留给别人
4. **再平衡**：定期（如每季度）检查仓位分布

> 本技能提供的建议仅供参考，不构成投资建议。股市有风险，操作需谨慎。用户应独立判断，自负盈亏。
