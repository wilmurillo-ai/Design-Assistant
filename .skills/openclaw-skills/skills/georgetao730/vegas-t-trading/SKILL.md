## 🔧 核心功能详解

### 1. 斐波那契计算

```python
def calculate_fibonacci(klines, lookback=100):
    # 获取 100 根 K 线内的最高点和最低点
    high = max(k.high for k in klines[-100:])
    low = min(k.low for k in klines[-100:])
    diff = high - low
    
    # 计算关键位
    return {
        "61.8%": high - 0.618 * diff,  # 黄金分割
        "50%": high - 0.5 * diff,
        "38.2%": high - 0.382 * diff,
        "78.6%": high - 0.786 * diff,
    }
```

### 2. 共振检测

```python
def check_fib_confluence(price, fib, tunnel_support):
    # 检查价格是否同时重合斐波那契位和隧道支撑
    tolerance = fib.diff * 0.01  # 1% 容差
    
    if abs(price - fib.level_618) < tolerance and abs(price - tunnel_support) < tolerance * 2:
        return "黄金共振：61.8% + 隧道支撑"
```

### 3. 评分系统

```python
def calculate_score(tunnel_60m, tunnel_15m, tunnel_5m, fib, price):
    score = 0
    # 60 分钟趋势 (30%)
    if price > tunnel_60m['ema144']: score += 15
    if tunnel_60m['ema144'] > tunnel_60m['ema169']: score += 15
    
    # 15 分钟回踩 (20%)
    if abs(price - tunnel_15m['ema144']) / price < 0.02: score += 20
    
    # 5 分钟金叉 (30%)
    if tunnel_5m['ema12'][-1] > tunnel_5m['ema13'][-1]: score += 15
    if price > tunnel_5m['ema169']: score += 15
    
    # 斐波那契 (20%)
    if abs(price - fib['61.8%']) < tolerance: score += 20
    
    return score
```

## ⚠️ 重要提醒

### 风险控制

1. **仓位管理**：做 T 仓位不超过总持仓的 50%
2. **止损设置**：单笔亏损不超过 2%，或跌破斐波那契 78.6%
3. **T+1 限制**：A 股当天买入不能当天卖出，需用底仓
4. **交易时间**：仅限交易日 9:30-15:00

### 技术分析局限

- 维加斯通道和斐波那契都是**滞后指标**
- 评分系统基于历史数据，不能预测突发消息
- 需结合**大盘情绪**、**板块轮动**、**消息面**综合判断
