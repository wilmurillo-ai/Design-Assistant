# 行业周期判断指南

## 为什么行业周期很重要？

**模型 B（历史低位反弹型）的核心逻辑**：好公司遇上烂时候

- 公司本身优秀（护城河、竞争力未变）
- 但行业处于周期低谷（产品价格下跌、需求萎缩）
- 一旦行业见底回升，股价弹性极大

---

## 常见周期性行业

### 强周期行业

| 行业 | 周期驱动因素 | 周期长度 |
|------|-------------|---------|
| **有色金属** | 全球经济、供需关系 | 3-5 年 |
| **石油化工** | 油价、产能周期 | 3-5 年 |
| **煤炭** | 能源政策、供需 | 2-4 年 |
| **钢铁** | 基建投资、房地产 | 2-3 年 |
| **水泥** | 基建、房地产开工 | 2-3 年 |
| **航运** | 全球贸易、运力 | 3-5 年 |
| **养猪** | 猪周期（能繁母猪存栏） | 3-4 年 |

### 弱周期行业

| 行业 | 特点 |
|------|------|
| **白酒** | 消费升级，品牌护城河 |
| **医药** | 刚需 + 创新，集采影响 |
| **家电** | 地产后周期，更新换代 |
| **保险** | 利率环境、新业务价值 |

## 行业见底信号

### 1. 价格信号

```
✅ 见底信号：
- 产品价格连续 3 个月止跌
- 期货价格开始回升
- 龙头企业率先提价

❌ 未见底：
- 产品价格仍在下跌
- 库存持续积压
- 企业竞相降价
```

### 2. 库存信号

```
✅ 见底信号：
- 行业库存降至历史低位
- 库存周转天数开始下降
- 企业开始补库存

❌ 未见底：
- 库存创历史新高
- 库存周转天数持续上升
```

### 3. 产能信号

```
✅ 见底信号：
- 落后产能退出（中小企业破产）
- 行业集中度提升
- 新增产能大幅减少

❌ 未见底：
- 产能仍在扩张
- 新玩家不断进入
```

### 4. 政策信号

```
✅ 见底信号：
- 行业扶持政策出台
- 供给侧改革
- 环保限产

❌ 未见底：
- 政策仍在收紧
- 监管趋严
```

### 5. 财务信号

```
✅ 见底信号：
- 龙头企业仍盈利（小亏可接受）
- 现金流健康
- 负债率可控

❌ 未见底：
- 大面积亏损
- 现金流断裂
- 负债率飙升
```

## 具体行业判断方法

### 养猪行业

**核心指标**：能繁母猪存栏量

```
见底信号：
- 能繁母猪存栏连续 3 个月下降
- 猪粮比<5:1（深度亏损）
- 大规模养殖户退出

数据来源：农业农村部月度报告
```

### 光伏行业

**核心指标**：硅料/硅片价格

```
见底信号：
- 硅料价格跌破成本线
- 二三线厂商停产
- 行业整合加速

数据来源：PV Infolink 周度报价
```

### 锂电行业

**核心指标**：碳酸锂价格

```
见底信号：
- 碳酸锂价格跌破 10 万元/吨
- 锂矿企业减产
- 下游排产回升

数据来源：上海有色网
```

### 航运行业

**核心指标**：BDI/SCFI 指数

```
见底信号：
- BDI 指数跌破 1000 点后企稳
- 运力增速放缓
- 全球贸易量回升

数据来源：波罗的海交易所
```

## 实战案例

### 案例：2023 年养猪周期见底

```
信号出现顺序：
1. 2023.06：猪粮比跌破 5:1（深度亏损）
2. 2023.08：能繁母猪存栏开始下降
3. 2023.10：大规模养殖户退出新闻增多
4. 2023.12：猪价开始反弹

布局时机：2023.09-10 月（信号 2+3 出现时）
```

### 案例：2024 年光伏周期

```
信号出现顺序：
1. 2024.03：硅料价格跌破 6 万元/吨（成本线）
2. 2024.05：二三线厂商停产
3. 2024.07：行业整合加速

布局时机：等待信号 3 确认后
```

## 注意事项

1. **周期底部可能持续很久**：不要急于满仓，分批布局
2. **警惕价值陷阱**：有些行业可能永远无法回到高点
3. **关注政策风险**：政策可能改变周期走向
4. **分散投资**：不要押注单一行业
