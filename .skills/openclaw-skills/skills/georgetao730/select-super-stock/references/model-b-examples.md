# 模型 B：历史低位反弹型典型案例

## 核心特征回顾

| 维度 | 要求 |
|------|------|
| **空间** | 距离历史高点回撤>50% |
| **企稳信号** | 双底/头肩底/MACD 底背离 |
| **当前状态** | 底部缩量筑底后温和放量 |
| **基本面** | 核心竞争力未丧失 |
| **行业信号** | 产品价格止跌/库存去化 |
| **消息面** | 大股东增持/回购/利空出尽 |

---

## 典型案例：万华化学 (600309)

### 技术面特征

```
空间回撤：
- 历史高点：150 元（假设）
- 最低点：60 元
- 回撤幅度：60%

底部形态：
- 周线级别出现双底结构
- MACD 底背离（价格创新低，MACD 不创新低）
- 成交量底部萎缩，反弹温和放量

当前状态：
- 股价从最低点反弹 20%-30%
- 站上 60 日均线
- 250 日均线开始走平
```

### 基本面特征

```
- 核心竞争力：MDI 全球龙头，技术壁垒高
- 股价下跌原因：行业周期低谷，产品价格下跌
- 财务健康：资产负债率合理，现金流稳定
- 估值低位：PE 处于历史 10% 分位以下
```

### 行业信号

```
- MDI 价格止跌回升
- 行业库存去化完成
- 中小产能退出，行业集中度提升
- 下游需求开始恢复
```

## 其他典型案例

### 海螺水泥 (600585)
- **特征**：水泥龙头，行业周期底部
- **回撤**：从高点回撤约 60%
- **信号**：基建投资回暖，水泥价格企稳

### 恒瑞医药 (600276)
- **特征**：创新药龙头，集采影响消化
- **回撤**：从高点回撤约 70%
- **信号**：创新药获批加速，海外授权突破

### 中国平安 (601318)
- **特征**：保险龙头，寿险改革阵痛
- **回撤**：从高点回撤约 60%
- **信号**：新业务价值增速转正，投资端改善

## 筛选要点

1. **确认是好公司**：行业龙头，核心竞争力未丧失
2. **下跌有原因**：行业周期/系统性风险，非公司自身问题
3. **技术面企稳**：底部形态 + 底背离 + 温和放量
4. **行业见底信号**：产品价格/库存/政策至少一项好转

---

## 建仓策略

```
左侧建仓（激进）：
- 底部区域分批建仓 40%
- 每下跌 10% 加仓 20%
- 总仓位不超过 60%

右侧建仓（稳健）：
- 等待突破 60 日均线确认
- 首次建仓 30%
- 回踩确认后再加仓 30%
- 突破年线后加仓 40%

止损：跌破底部最低点，止损离场
止盈：第一目标位 50% 回撤位，第二目标位年线
```

---

## 风险提示

⚠️ **价值陷阱风险**：有些股票看起来便宜，但基本面持续恶化，可能永远无法回到高点

**识别方法**：
- 营收连续 3 年下滑
- 毛利率持续下降
- 负债率持续上升
- 大股东/高管持续减持

遇到以上情况，即使符合技术面条件，也要谨慎！
