# 黑名单：垃圾股识别指南

## 典型代表：中远海发 (601866)

---

## 技术面特征

### 1. 暴涨暴跌

```
❌ 典型表现：
- 一周内涨停 3 次，跌停 2 次
- 单日振幅经常超过 15%
- K 线图呈现"心电图"或"织布机"走势

识别指标：
- 30 日内涨跌停天数 >= 3 天
- 平均日振幅 > 8%
- 股价走势极不连贯
```

### 2. 量价异常

```
❌ 典型表现：
- 底部无量突然连续涨停（3-5 个涨停板）
- 高位放出巨量，单日换手率>20%
- 涨停板频繁打开

识别指标：
- 单日换手率 > 20% → 警惕
- 单日换手率 > 30% → 高危
- 连续 3 日放量下跌 → 出货信号
```

### 3. K 线形态异常

```
❌ 典型表现：
- 频繁出现长上影线（冲高回落）
- 频繁出现长下影线（探底回升）
- "天地板"或"地天板"

识别方法：
- 统计 20 日内长影线天数
- 长影线天数 >= 8 天 → 庄家控盘迹象
```

### 4. 横盘震荡（织布机）

```
❌ 典型表现：
- 股价在窄幅区间（<10%）横盘超过 3 个月
- 每日涨跌幅在±2% 以内
- 成交量极度萎缩

识别指标：
- 60 日价格波动幅度 < 10%
- 日均换手率 < 1%
```

## 基本面特征

### 1. 业绩支撑不足

```
❌ 危险信号：
- 市盈率>100 倍或负值（亏损）
- 连续 3 年净利润为负
- 营收连续下滑
- 经营性现金流为负

识别方法：
- PE > 100 → 警惕
- PE < 0（亏损）→ 高危
- 连续亏损 → 规避
```

### 2. 蹭热点炒概念

```
❌ 典型表现：
- 公司主营业务与热点概念无关
- 频繁发布"战略合作"公告
- 蹭 AI、元宇宙、区块链等热点

识别方法：
- 查看公告内容是否实质性
- 对比主营业务与概念关联度
- 关注监管问询函
```

### 3. 股东结构异常

```
❌ 危险信号：
- 前十大流通股东中散户比例极高
- 存在明显牛散/私募扎堆
- 大股东持续减持

识别方法：
- 查看季报股东人数变化
- 股东人数激增 → 散户接盘
- 大股东减持 → 不看好公司
```

## 消息面特征

### 1. 小作文满天飞

```
❌ 典型表现：
- 股吧/论坛传闻不断
- 真假难辨的"内幕消息"
- 媒体集中吹捧

识别方法：
- 只相信官方公告
- 忽略非官方渠道消息
- 警惕"重大利好"后的高位放量
```

### 2. 监管关注

```
❌ 危险信号：
- 收到交易所问询函
- 被证监会立案调查
- 被实施 ST/*ST

识别方法：
- 查看公司公告
- 关注交易所官网
```

## 快速识别清单

**满足以下任意 3 项，坚决规避：**

- [ ] 30 日内涨跌停天数 >= 3 天
- [ ] 单日换手率 > 20%
- [ ] 20 日内长影线天数 >= 8 天
- [ ] 市盈率>100 倍或负值
- [ ] 连续 2 年亏损
- [ ] 大股东近期减持
- [ ] 收到监管问询函
- [ ] 股吧传闻满天飞

## 真实案例解析

### 案例 1：某妖股 A

```
表现：
- 1 个月内从 5 元涨到 25 元（5 倍）
- 连续 10 个涨停板
- 第 11 天高开低走，放出历史天量
- 随后连续跌停，回到 5 元

特征：
- 蹭热点（元宇宙概念）
- 无业绩支撑（PE>500）
- 大股东在高位减持
- 游资接力炒作

教训：不追高，不接最后一棒
```

### 案例 2：某庄股 B

```
表现：
- 长期横盘（6 个月波动<15%）
- 每日走势高度相似（织布机）
- 突然连续涨停
- 随后 A 字杀跌

特征：
- 庄家高度控盘
- 流动性差
- 散户无法及时出逃

教训：不碰长期横盘的庄股
```

---

## 保护原则

1. **宁可错过，不做错**：垃圾股就算涨了也不要后悔
2. **不追高**：连续涨停的股票不追
3. **不抄底**：连续跌停的股票不抄
4. **不信传闻**：只看官方公告
5. **设置止损**：买入后跌破支撑位立即离场
