# 模型 A：长线稳步上涨型典型案例

## 核心特征回顾

| 维度 | 要求 |
|------|------|
| **趋势** | 月线和年线完美多头排列 |
| **指标** | 月线/周线 MACD 零轴上方 |
| **量价** | 上涨放量、下跌缩量 |
| **ROE** | 常年>15% |
| **股息率** | >4% |
| **护城河** | 行业龙头/垄断 |

---

## 典型案例：中国海油 (600938)

### 技术面特征

```
月线级别：
- EMA12/13 长期在金叉状态
- EMA144/169（维加斯隧道）稳步向上
- MACD 零轴上方运行，偶有金叉

周线级别：
- 5/10/20/60 周线多头排列
- 股价依托 20 周线稳步上行

日线级别：
- 250 日均线（年线）持续向上
- 股价回调至年线附近即获得支撑
```

### 基本面特征

```
- 行业地位：中国最大海上油气生产商，垄断性资源
- ROE：常年保持在 15%-20%
- 股息率：约 6%-8%（行业较高水平）
- 市盈率：8-12 倍（合理区间）
- 现金流：经营性现金流持续为正且增长
```

### 消息面驱动

```
- 国际油价长期上涨趋势
- 国家能源安全战略支持
- 勘探开发投入持续增加
- 成本控制优秀，盈亏平衡点低
```

## 其他典型案例

### 贵州茅台 (600519)
- **特征**：消费龙头，品牌护城河极深
- **ROE**：常年 30%+
- **风险**：估值较高，政策风险

### 招商银行 (600036)
- **特征**：零售银行龙头，资产质量优秀
- **ROE**：15%-18%
- **股息率**：约 4%-5%

### 长江电力 (600900)
- **特征**：水电垄断，现金流稳定
- **ROE**：15% 左右
- **股息率**：约 4%-5%

---

## 筛选要点

1. **先看月线**：月线级别必须是上涨趋势，排除月线下跌的股票
2. **再看年线**：250 日均线必须向上，股价在年线上方
3. **验证基本面**：ROE>15%，股息率>4%，行业龙头
4. **排除暴涨**：60 日涨幅不超过 50%，避免短期过热

---

## 建仓策略

```
第一批：现价建仓 30%
第二批：回调至 60 日均线加仓 30%
第三批：回调至年线加仓 40%

止损：跌破年线且 3 日内无法收回，减仓 50%
止盈：不设固定止盈，趋势破坏时离场
```
