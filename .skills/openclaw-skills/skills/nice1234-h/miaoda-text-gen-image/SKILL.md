# 妙搭图片生成（真实感版）

> 让 AI 生成的图片更真实、不做作，符合你的创作思路

## 核心价值

大多数 AI 绘图工具生成的图片存在"假感"——过度美化、塑料质感、姿势僵硬。本 Skill 提供**真实感 Prompt 技巧**，让你生成的图片更像真实世界拍摄的照片/影片画面。

---

## 真实感 Prompt 公式

```
[主体] + [环境] + [光线] + [情绪/氛围] + [风格参考]
```

### 关键词分类

| 类别 | 关键词 | 作用 |
|------|--------|------|
| **光线** | 自然光、黄昏光、侧光、逆光、柔光、硬光、眼神光 | 真实光影 |
| **质感** | 皮肤质感、肌肉线条、织物纹理、电影颗粒、8mm 胶片 | 真实材质 |
| **相机** | Sony A7M4、Canon R5、35mm、50mm、f/1.8、ISO 100 | 真实设备感 |
| **风格** | 纪录片风格、电影感、日系写真、欧美时尚、肖像摄影 | 真实风格 |
| **情绪** | 若有所思、轻松自然、眼神忧郁、灿烂微笑、沉思 | 真实情感 |
| **构图** | 三分法、黄金分割、浅景深、纵深感、框架构图 | 真实构图 |

| ❌ 容易导致"假感"的词 | ✅ 建议替换 |

## 基础命令

### 快速参考

| 参数 | 说明 | 必需 | 默认值 |
|------|------|------|--------|
| `--prompt` | 图片描述（越详细效果越好） | 是 | - |
| `--ratio` | 宽高比 | 否 | 1:1 |
| `--watermark` | 添加 AI 水印 | 否 | false |
| `--output, -o` | 输出格式: text/json | 否 | text |

### 宽高比选择

| 用途 | 推荐 ratio |
|------|-----------|
| 头像、图标、Logo | `1:1` |
| 文章配图、横幅 | `4:3` 或 `3:2` |
| 演示文稿、视频封面 | `16:9` |
| 手机壁纸、竖版海报 | `9:16` |
| 社交媒体竖版 | `3:4` |

### 使用示例

```bash
# 真实感人像
miaoda-studio-cli text-to-image --prompt "傍晚时分，一个年轻女孩站在金色余晖的海边，海风吹起她的长发和白色连衣裙的裙摆，她侧身对着镜头，自然地撩头发，眼神看向远方，皮肤质感真实，Sony A7M4 拍摄，35mm 镜头，电影纪录片风格" --ratio 3:2

# 指定宽高比
miaoda-studio-cli text-to-image --prompt "阳光下的复古咖啡店，一个女孩坐在窗边看书，自然光从窗外洒入，佳能 R5 拍摄，日系清新人像" --ratio 4:3

# 获取图片 URL（用于程序调用）
miaoda-studio-cli text-to-image --prompt "城市天台夜景，霓虹灯光，赛博朋克风格" --ratio 16:9 --output json
```

## 常见问题

**Q: 为什么同样描述生成的效果差很多？**
A: prompt 越具体越好。加上相机型号、镜头焦距、光线方向等细节，AI 更容易理解你想要的感觉。

**Q: 人物总是像网红脸怎么办？**
A: 避免使用"完美"、"精致"、"盛世美颜"等词，改用"普通但舒服"、"气质自然"、"干净的妆容"。

**Q: 风格怎么选？**
A: 电影感适合短视频配图，日系适合清新风格，欧美时尚适合商业感。根据你的使用场景选择。
