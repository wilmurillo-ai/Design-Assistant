## 二、快速开始

### 2.1 记录任务反馈

每次任务完成后，快速评估结果：

```
反馈：[👍/👎/✏️]
标签：任务类型（可选，如：文件整理、报告生成）
备注：改进意见（可选）
```

**等价命令**：
```bash
python learn.py feedback good   # 满意
python learn.py feedback bad     # 不满意
```

### 2.2 查看模式分析

新会话启动时运行，了解近期任务模式：

```bash
python learn.py patterns
```

**输出示例**：
```
## 模式识别报告

[高] **文件整理** (评分: 85)
   - 30天内出现: 12 次
   - 建议: 创建「文件整理自动化模板」
   - 预估节省: ~36 分钟

[中] **内容推送** (评分: 72)
   - 30天内出现: 8 次
   - 建议: 创建「CodeBuddy推送模板」
   - 预估节省: ~24 分钟
```

### 2.3 运行完整学习周期

定期（建议每周）运行，获取综合洞察：

```bash
python learn.py full
```

## 三、核心模块详解

### 3.1 反馈收集 (feedback.py)

**功能**：
- 记录显式反馈（用户主动评分）
- 采集隐式信号（复用率、等待时长等）
- 分析不满意模式

**数据结构**：
```json
{
  "feedback_id": "uuid",
  "task_id": "任务ID",
  "timestamp": "2026-04-14T14:30:00",
  "rating": "good|bad|neutral",
  "tags": ["文件整理", "自动化"],
  "note": "用户备注"
}
```

### 3.2 模式识别 (pattern_recognizer.py)

**功能**：
- 扫描30天内的记忆文件
- 提取高频任务类型
- 计算模式价值分数
- 生成沉淀建议

**评分公式**：
```
模式价值 = (频率得分 × 50) + (时效得分 × 20) + (稳定得分 × 30)
```

**阈值**：分数 ≥ 60 才建议沉淀

### 3.3 知识蒸馏 (knowledge_distiller.py)

**功能**：
- 从正面反馈中提炼成功要素
- 从记忆片段中生成模板结构
- 保存为标准YAML模板

**模板格式**：
```yaml
template:
  name: "模板名称"
  version: "1.0.0"
  trigger:
    keywords: ["关键词1", "关键词2"]
  process:
    steps:
      - id: 1
        action: "执行动作"
```

## 五、与主系统集成

### 5.1 AI任务后自动触发

在每次任务完成后，AI 自动追加：

```
## 任务反馈收集

请评估本次任务执行的质量：
- 👍 满意：结果直接可用
- 👎 不满意：需要修正
- ✏️ 有建议：具体改进意见

格式：
反馈：[👍/👎/✏️]
标签：（可选）
备注：（可选）
```

### 5.2 新会话启动检查

每次新会话读取记忆后，自动运行：
```python
system = SmartLearningSystem(workspace)
suggestions = system.check_and_suggest()
```
