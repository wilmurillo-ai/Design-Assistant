# 🦴 Caveman Soul Optimizer
*Traemos la eficiencia del modo cavernícola sin perder ni un gramo del alma de tu asistente.*

**Caveman Soul Optimizer** es una skill para agentes de **OpenClaw** (como Tenacitas) diseñada para reducir drásticamente el consumo de tokens y los costes de API, sin que tu agente pierda su personalidad.

## 🚀 ¿Qué hace?
Esta skill implementa la técnica "Caveman" de forma inteligente:
1.  **Ahorro Invisible:** Todo el razonamiento narrativo ("pensamientos", planificación, escritura de logs y trazas internas) se comprime a un estilo telegráfico (cavernícola), protegiendo siempre la sintaxis real al usar herramientas.
2.  **Alma Intacta:** Cuando el agente te responde a ti, **mantiene su personalidad original al 100%** (sea cínico, divertido, serio o excéntrico). El ahorro ocurre "detrás de las cámaras".

## 📦 Instalación (OpenClaw)
Para instalarlo en tu agente, simplemente clona este repo en tu carpeta de plugins o pídele a tu agente que lo haga:

```bash
cd ~/.openclaw/plugins
git clone <URL_DE_ESTE_REPO> caveman-soul-optimizer
openclaw plugins enable caveman-soul-optimizer
```

O si tu agente tiene permisos de shell, simplemente dile:
> *"Tenacitas, instálate esta skill desde [URL_DE_ESTE_REPO] y actívala."*

## 💡 ¿Por qué usarlo?
- **Tokens de salida:** Ahorra aproximadamente un 70% de tokens en los bloques de pensamiento, razonamiento, y en la escritura de logs o trazas internas, comprobado por diseño.
- **Contexto:** El historial dura mucho más antes de que el agente empiece a olvidar cosas.
- **Eficacia:** Los modelos suelen razonar mejor cuando se les obliga a ser ultra-concisos en la lógica interna.
- **Seguridad Total:** Esta skill es puramente un conjunto de instrucciones de texto (prompting). **No ejecuta ningún tipo de código binario ni intercepta llamadas a la API** en tu máquina, lo que garantiza que tus datos y tus claves están 100% a salvo de software de terceros.
