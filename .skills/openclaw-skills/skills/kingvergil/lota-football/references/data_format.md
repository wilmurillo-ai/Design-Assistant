# 紧凑版特征文本数据格式说明

## 概述
紧凑版特征文本（compact-fet）是为AI分析优化的比赛数据表示格式，整合了多维度比赛信息，使用高度优化的记法压缩数据。

## 数据内容结构

### 1. 比赛基本信息
- **球队名称**：主队和客队名称
- **联赛信息**：联赛名称
- **比赛时间**：比赛开始时间
- **主客场信息**：主场/客场标识

### 2. 基本面分析数据
- **交手历史**：过往对战记录
- **球队排名**：联赛排名信息
- **近期表现**：主客队最近比赛结果

### 3. 赔率数据（使用优化紧凑记法）
- **离散赔率**：`t=Δt±m odds=h/d/a` - 时间差±分钟和主/和/客赔率
- **主流欧盘**：`t=Δt±m odds=h/d/a/r(rrr%)` - 时间差±分钟，主/和/客赔率及返还率
- **主流亚盘**：`t=Δt±m odds=h/handicap/a/r(rrr%)` - 时间差±分钟，主队赔率/让球盘口/客队赔率/返还率
- **大小球数据**：`t=Δt±m odds=o/handicap/u/r(rrr%)` - 时间差±分钟，大球赔率/盘口/小球赔率/返还率

### 4. 必发与公平盘数据
- **必发欧盘**：成交量(万)/价位/盈亏指数/市场指数/凯利值（主/和/客格式）
- **公平盘**：主客实力差(egd)/预期进球总和(etg)/预期主客队进球(xgh/xga)/ELO评级
- **让球欧盘信息**：平均欧盘和竞彩胜平负赔率

### 5. 可选数据（根据参数启用）
- **阵容信息**：首发名单（比赛开始前最后半小时出现）
- **进球预测**：预期进球模型预测结果
- **比分预测**：比分概率模型预测结果

## 赔率数据记法解析

### 时间表示
- `OPt-{delta_min}m=`：开盘时间，距离比赛开始的时间差（负值表示赛前）
  - 示例：`OPt-240m=` 表示赛前240分钟的开盘数据
- `Δt+{delta_min}m`：相对于前一个数据点的时间差（正值表示更接近比赛）
  - 示例：`Δt+30m` 表示距离前一个数据点30分钟

### 变化方向符号
- `↑`：赔率上升
- `↓`：赔率下降
- `→`：赔率不变
- 示例：`Δt+30m↑→↓` 表示主胜赔率↑、平赔→、客胜赔率↓

### 数据字段
- `h/d/a`：主胜/平局/客胜赔率（欧盘）
- `h/handicap/a`：主队赔率/让球盘口/客队赔率（亚盘）
- `o/handicap/u`：大球赔率/大小球盘口/小球赔率
- `r(rrr%)`：返还率（百分比），如 `r(95.5%)`

### 压缩算法
紧凑版特征文本采用多层压缩算法：

1. **差异编码**：只记录与前一个数据点的变化，减少冗余数据
2. **时间窗口过滤**：默认保留赛前24小时数据，过滤过时数据
3. **关键节点采样**：最多保留40个关键时间节点，优化数据密度
4. **微小波动过滤**：过滤无意义的微小赔率变动（<0.01变化）

## 必发与公平盘数据记法

### 必发欧盘数据格式
```
成交量(万)(主/和/客):1.23/2.34/3.45
```

包含五个维度数据（均为主/平/客三路格式）：
1. **成交量**：交易量（单位：万元）
2. **价位**：当前交易价格
3. **盈亏指数**：盈亏平衡点分析
4. **市场指数**：市场热度指标
5. **凯利值**：凯利公式计算值

### 公平盘数据字段
- `egd`：主客实力差 (Goal Difference)
- `etg`：主客进球和 (Expected Total Goals)
- `xgh`：预期主队进球 (Expected Goals Home)
- `xga`：预期客队进球 (Expected Goals Away)
- `ELO`：球队ELO评级分

## 示例解析

### 赔率数据示例
1. **开盘数据**：
   ```
   OPt-240m=1.85/3.40/4.20(r95.5%)
   ```
   - 赛前240分钟开盘
   - 赔率：主胜1.85，平局3.40，客胜4.20
   - 返还率：95.5%

2. **变化数据**：
   ```
   Δt+30m↑→↓1.90/3.40/4.10/95.0
   ```
   - 30分钟后变化
   - 主胜↑至1.90，平局→保持3.40，客胜↓至4.10
   - 返还率变为95.0%

### 完整数据段示例
```
[比赛信息] 英超:曼联 vs 切尔西 | 2025-03-15 20:00 | 未开赛
[基本面] 近5次交手:2胜2平1负 | 排名:3rd vs 5th | 近期:WWLDD vs LWDWW
[欧赔] OPt-240m=1.85/3.40/4.20(r95.5%) Δt+30m↑→↓1.90/3.40/4.10/95.0
[亚盘] OPt-240m=0.85/-0.5/1.05(r95.5%) Δt+30m→→→0.85/-0.5/1.05/95.0
[大小] OPt-240m=0.95/2.5/0.95(r95.5%) Δt+30m↓→↑0.90/2.5/1.00/95.0
[必发] 15.2万(65%/20%/15%):1.85/3.40/4.20
[公平盘] egd:+0.3 etg:2.8 xgh:1.5 xga:1.3 ELO:1850/1820
```

## 数据应用场景
紧凑版特征文本适用于多种分析场景：

1. **AI模型训练**：为机器学习模型提供结构化的特征数据
2. **投注决策支持**：提供全面的赔率变化和市场数据分析
3. **比赛预测**：整合多维度数据支持比赛结果预测
4. **市场分析**：分析赔率变化趋势和市场情绪
5. **学术研究**：足球比赛数据分析研究

## 技术特点
- **高压缩比**：通过差异编码和关键节点采样实现高压缩比
- **时间序列优化**：针对赔率时间序列数据专门优化
- **人类可读**：在保证机器可读性的同时，保持一定的人类可读性
- **多维度整合**：整合基本面、赔率、交易量等多维度数据

## 注意事项
1. 数据可能存在几分钟延迟
2. 部分可选数据仅在特定条件下提供（如阵容信息在赛前半小时出现）
3. 赔率数据经过过滤处理，微小波动可能被忽略
4. 不同比赛的数据完整度可能有所不同
