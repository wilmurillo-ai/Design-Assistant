# 更新日志

## 1.3.0 (2026-03-05)

### 修复：配置在容器重建后丢失（含跨平台兼容）

- **多路径配置查找**：Python 脚本按优先级自动查找配置文件（`FEISHU_ENV_FILE` > 技能根目录 > `~`），容器重建后不再丢失
- **新增 `save-config` 子命令**：用 Python 原生文件操作替代 `sed -i ''`，macOS/Linux 通用，幂等写入
- **SKILL.md 简化**：去掉 `sed`/`grep`/`source` 等 shell 配置脚本，改为调用 `save-config`
- **新增 `.gitignore`**：防止凭证文件被意外提交
- **删除不存在的权限 `im:message:urgent`**：飞书无此权限，应用内加急只需 `im:message` 即可；`im:message.urgent:phone` 和 `contact:user.id:readonly` 改为必需
- **安装说明规范化**：README 区分 OpenClaw / 本地两种安装方式，明确 clone 到 `~/.openclaw/workspace/skills/` 目录

## 1.2.0 (2026-03-02)

### 重大更新

- **新增电话加急**：`--phone-call` 参数，真正拨打手机电话提醒用户
- **新增 notify 命令**：仅发加急消息，不创建会议
- **架构重构**：从 1v1 视频呼叫改为会议 + 消息卡片 + 加急的三步组合
- **消息卡片**：发送带一键加入按钮的富文本消息卡片
- **降级策略**：卡片消息失败自动降级为文本；电话加急失败自动降级为应用内加急
- **完善错误提示**：每个常见错误码都有中文排查指引
- **文档完善**：新增权限开通步骤、机器人能力配置、常见错误排查表
