## 🎯 核心原则（强制遵守）

### 1. 真实 > 爆款
- ❌ 绝不编造数据、案例、收入
- ❌ 绝不夸大效果（"3天学会"、"月入8千"）
- ✅ 只讲真实发生的事情
- ✅ 所有数据必须有来源（自己测试/公开报告/真实学员）

### 2. 真诚 > 完美
- ❌ 不装专业、不说教、不卖弄
- ✅ 说人话、有温度、像朋友聊天
- ✅ 真诚的不足 > 虚构的完美

### 3. 长期 > 短期
- ❌ 不为短期流量牺牲长期信任
- ✅ 追求长期价值，建立信任
- ✅ 一次谎言毁掉所有信任

## 📋 完整流程（5步）

### Step 1: 选题（基于真实数据）

#### 选题来源（4种）

**1. 自己的AI工具测试**
```
示例：
- 我测试了5个AI写作工具，发现3个真相
- ChatGPT vs 文心一言 vs 通义千问，真实对比
- OpenClaw实战：我用它写了脚本、分析了数据

要求：
✅ 真实测试过（有录屏/截图）
✅ 优缺点都讲（不偏不倚）
✅ 数据真实（不夸大）
```

**2. 公开数据和新闻**
```
示例：
- 根据36氪报道，AI培训市场规模500亿
- OpenClaw登顶GitHub，超越Linux
- 2026年AI行业5大趋势

要求：
✅ 有明确来源（"根据XX报道"）
✅ 数据可验证（公开可查）
✅ 解读合理（不曲解）
```

**3. 自己的真实经历**
```
示例：
- 我学AI 6个月，花了2万，踩了10个坑
- 从0到1，我考证了：抖音&阿里云认证讲师
- 我做AI培训1年，踩过的坑和经验

要求：
✅ 真实经历（不编造）
✅ 有证据（截图/证书/账单）
✅ 不夸大（好就是好，不好就是不好）
```

**4. 真实学员案例（需授权）**
```
示例：
- 学员@小李（化名），30天学会AI，真实反馈
- 3个学员，3种学习路径，告诉你哪个最有效

要求：
✅ 真实学员（有授权）
✅ 真实案例（有证据）
✅ 不夸大（收入/效果真实）
```

#### 选题决策
```
用3个维度评估：
1. 流量潜力（用户想看 + 平台想推）
2. 制作难度（简单 > 中等 > 困难）
3. 转化价值（高 > 中 > 低）

选分值最高的
```

#### 禁用选题
```
❌ 虚构案例（"学员小王月入8000"）
❌ 夸大效果（"3天学会AI"）
❌ 制造焦虑（"不学AI就被淘汰"）
❌ 纯理论（"AI的底层逻辑是什么"）
```

---

### Step 2: 脚本（痛点+洞察+路径+诱饵）

#### 脚本结构（60秒）

**开头5秒（黄金5秒）**
```
公式1: 痛点提问
"你部署了OpenClaw但不会用？"
"很多人学AI，但90%都错了"

公式2: 数据冲击
"我测试了5个AI写作工具，发现3个真相"
"OpenClaw登顶GitHub，超越Linux"

公式3: 故事引入
"昨天有个学员问我：OpenClaw怎么部署？"
```

**中间40-50秒（核心内容）**
```
结构：痛点 + 洞察 + 路径

示例：
"很多人部署了OpenClaw，但不会用（痛点）
为什么？因为你不懂大模型的底层逻辑（洞察）
正确的学习路径是5步（路径）：
1. 起底AI底层逻辑
2. 学Coze智能体
3. 学AI编程
4. 学AI写作
5. 学AI漫剧"
```

**结尾10秒（转化）**
```
结构：立人设 + 给诱饵 + 行动指令

示例：
"我是大象，15年自媒体+AI方案架构师（立人设）
我开源了10节课，免费送你（给诱饵）
关注我，私信领10节课（行动指令）"
```

---

### Step 3: 拍摄（录屏+镜头+表演）

#### 拍摄方式

**1. 录屏（展示OpenClaw/AI工具）**
```
展示内容：
- 飞书对话界面（你问我答的过程）
- OpenClaw界面（模糊处理）
- AI工具对比（左右分屏）

备注：
- 可以快进展示
- 关键部分慢放
- 字幕提示
```

**2. 镜头（正面对镜头）**
```
场景：书架/纯色背景，光线充足
服装：休闲商务（衬衫/Polo衫）
表演：自然真诚，语速适中，眼神看镜头
```

**3. 口述（最简单）**
```
展示方式：
- 镜头对着你
- 身后有OpenClaw界面（模糊处理）
- 关键台词："我正在用OpenClaw优化脚本"
```

---

### Step 4: 剪辑（节奏+字幕+BGM）

#### 节奏
```
开头5秒：快节奏（快速切入）
中间部分：中速（清晰表达）
结尾10秒：上扬（引导关注）
```

#### 字幕
```
字体：黑体/微软雅黑
大小：40-50号
颜色：白字黑边
关键信息：放大+变色
```

#### BGM
```
开头：轻快节奏
中间：轻柔背景
结尾：上扬节奏
```

### Step 5: 复盘（数据追踪+优化）

#### 核心数据
```
播放量：______

点赞率：______%（>5%及格，>10%优秀）
评论率：______%（>1%及格，>3%优秀）
私信数：______个
```

```

1. 完播率>60%

3. 评论有人问"怎么学"
4. 有人私信咨询

应对策略：
1. 立即做系列内容（同一选题，3-5条）
2. 引导到私域（公众号/社群）
3. 促转化（限时优惠/名额限制）
```
