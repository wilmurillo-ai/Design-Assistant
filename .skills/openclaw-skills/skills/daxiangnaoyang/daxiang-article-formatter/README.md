# Article Formatter - 自媒体文章排版专家

## 简介

这是一个专为自媒体文章（公众号、小红书、知识星球等）设计的高质量排版技能。它可以将长文转化为易读、美观、移动端友好的格式。

## 核心特性

✅ **智能结构优化** - 自动识别文章类型，应用最佳排版策略  
✅ **视觉元素增强** - 使用emoji、图标、色块提升可读性  
✅ **信息呈现优化** - 对比内容表格化，案例引用块化  
✅ **移动端友好** - 所有排版都考虑了移动端阅读体验  
✅ **多平台适配** - 支持公众号、小红书、知识星球等平台

## 快速开始

### 基本使用

1. 准备你的原始文章文本
2. 告诉我目标平台（如：公众号、小红书）
3. 我会自动应用排版规则
4. 输出可直接使用的排版后文本

### 示例

**输入**：
```
Claude的3个核心优势
1. 超长文本处理能力
这是Claude最核心的护城河。ChatGPT的上下文窗口是128k tokens，约等于10万字。而Claude 3可以达到200k tokens，约等于15-20万字。
```

**输出**：
```
## Claude的3个核心优势

### 1. 超长文本处理能力 📊

这是Claude最核心的护城河。

- ChatGPT：128k tokens ≈ 10万字
- Claude 3：200k tokens ≈ 15-20万字
```

## 支持的文章类型

- **干货型** - 深度分析、方法论
- **评测型** - 产品对比、工具评测
- **教程型** - 操作步骤、指南
- **观点型** - 观点表达、见解分享

## 核心规则

### 移动端优化
- 段落≤ 5行
- 标题≤ 20字
- 句子≤ 30字
- 列表项≤ 2行

### 视觉系统
- ✅ 优势/正确
- ❌ 劣势/错误
- 💡 洞察/建议
- 📊 数据/案例
- ⚡ 快速/高效
- 🎯 目标/结论

### 颜色系统（深色模式）
- 背景：#000000
- 正文：#ffffff
- 高亮：#ff6b6b
- 边框：#ffd93d
- 引用块：#4a90e2

## 质量检查清单

- [ ] 所有段落≤ 5行
- [ ] 每个章节有清晰标题
- [ ] emoji使用有目的，不滥用
- [ ] 对比内容用表格呈现
- [ ] 案例用引用块
- [ ] CTA清晰明确
- [ ] 移动端友好
- [ ] 颜色使用不超过3种
- [ ] 无大段密集文字
- [ ] 逻辑流畅，过渡自然
