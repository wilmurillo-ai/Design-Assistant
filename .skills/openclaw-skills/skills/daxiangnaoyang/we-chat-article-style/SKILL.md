# 公众号文章创作风格指南（大象风格）

**适用场景**：公众号文章、知乎文章、知识分享类文章

**核心特征**：痛点驱动 + 实用教程 + 数据支撑 + 完整签名

---

## 📋 文章结构模板（7段式）

### 1. 前言：痛点引入（200-300字）

**目的**：让读者产生共鸣

**写法**：
- 描述一个具体场景
- 点出读者的困扰
- 提出问题："为什么？"
- 引出解决方案

**示例开头**：
```
你是不是也遇到过这种情况？

你的飞书群里有个AI助手，看起来很智能...

但当你让它：
- 📝 "帮我创建一个飞书文档"
- 📊 "整理成多维表格"

结果它告诉你："对不起，我没有权限..."

**为什么？** 因为你的飞书AI助手只是个"聊天机器人"，
而不是真正的"办公助理"。
```

**要点**：
- ✅ 用"你是不是也遇到过"开头
- ✅ 列举3-4个具体痛点（用emoji）
- ✅ 给出"为什么"的答案
- ✅ 引出下文

---

### 2. 更新介绍：抛出解决方案（300-400字）

**目的**：告诉读者有新方法了

**写法**：
- 宣布好消息
- 对比更新前后的差异
- 强调核心价值

**模板**：
```
## 🚀 飞书官方更新：OpenClaw插件全面升级

就在最近，[产品名] 迎来了重大更新！

这次更新解决了什么问题？

### ✅ 从"旧状态"到"新状态"

**更新前**：
- ❌ 问题1
- ❌ 问题2
- ❌ 问题3

**更新后**：
- ✅ 解决1
- ✅ 解决2
- ✅ 解决3
```

**要点**：
- ✅ 用emoji标题（🚀/✨/🎯）
- ✅ 对比式呈现（更新前vs更新后）
- ✅ 用✅和❌符号

---

### 3. 功能清单：详细说明（600-800字）

**目的**：展示完整能力

**写法**：
- 分类列出功能
- 每个功能给出使用场景

**模板**：
```
## 📋 完整功能清单

### 1️⃣ 功能名称
```
✅ 能力1
✅ 能力2
✅ 能力3
```

**使用场景**：
- 场景1
- 场景2
- 场景3

### 2️⃣ 功能名称
...
```

**要点**：
- ✅ 用数字emoji（1️⃣2️⃣3️⃣）
- ✅ 每个功能用✅符号
- ✅ 代码块展示功能列表
- ✅ 粗体强调关键点

---

### 4. 配置教程：Step by Step（800-1000字）

**目的**：让读者能跟着做

**写法**：
- 分步骤说明
- 关键点用粗体
- 给出代码示例

**模板**：
```
## 🛠️ 如何配置完整权限？

很多用户问："为什么我的AI助手还是不能用这些功能？"

答案：**需要完整的三步配置**

### 第一步：...
```bash
# 代码示例
命令
```

### 第二步：...
（详细说明）

### 第三步：开放平台配置（**关键！**）

1. 打开...
2. 找到...
3. 进入**权限管理**
4. 开通以下权限：

#### 必开权限列表：
- `permission1` - 说明
- `permission2` - 说明

5. **发布应用**（**重要！**权限变更后必须重新发布）
```

**要点**：
- ✅ 分步骤（第一步/第二步）
- ✅ 关键词加粗（**关键！**/**重要！**）
- ✅ 代码块用bash格式
- ✅ 权限列表用代码块
- ✅ 强调"必须"的操作

---

## 📐 格式规范

### Emoji使用规则
- 章节标题：🚀/📋/🛠️/🎯/💡/🔧/📊/🎉
- 功能列表：✅/❌
- 序号：1️⃣2️⃣3️⃣4️⃣5️⃣6️⃣7️⃣
- 重点：⭐/🔥/💡/⚠️

- 关键词：**必须**/**重要**/**关键**
- 强调：**三步配置**/**完整功能**
- 数字：**93%** ⬇️

```

```

- 章节之间：`---`
- 重要内容前后：`---`

## 🎯 写作要点总结

### 内容要求
1. ✅ **痛点驱动**：开头必须点出读者困扰
2. ✅ **实用性强**：提供可执行的步骤
3. ✅ **案例丰富**：至少3个真实场景
4. ✅ **数据支撑**：用量化数据说话
5. ✅ **FAQ完善**：解答常见疑问

### 风格要求
1. ✅ **口语化**：像朋友聊天，不说教
2. ✅ **有温度**：用"你"、"我"，不冷冰冰
3. ✅ **适度幽默**：轻松自然，不强行讲笑话
4. ✅ **专业但不装**：说人话，不堆术语

### 结构要求
1. ✅ **7段式结构**：前言→更新→功能→教程→案例→技巧→FAQ
2. ✅ **逻辑递进**：为什么→是什么→怎么做→效果如何
3. ✅ **首尾呼应**：开头痛点→结尾解决方案

### 格式要求
1. ✅ **Emoji丰富**：每个章节都有emoji
2. ✅ **粗体强调**：关键词加粗
3. ✅ **代码块**：命令/列表/对话用代码块
4. ✅ **表格**：数据对比用表格
5. ✅ **签名完整**：三个认证+三个账号+二维码
