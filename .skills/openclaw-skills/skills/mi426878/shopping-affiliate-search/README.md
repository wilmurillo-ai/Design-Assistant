## 🚀 快速开始

### 1. 配置推荐码

```bash
# 淘宝客
python3 scripts/search.py config --taobao "mm_12345_67890_123456"

# 京东联盟
python3 scripts/search.py config --jd "your_jd_union_id"

# 拼多多
python3 scripts/search.py config --pdd "your_pdd_pid"

# 亚马逊
python3 scripts/search.py config --amazon "your-associate-id-20"
```

### 2. 搜索商品

```bash
# 全平台搜索
python3 scripts/search.py search "蓝牙耳机"

# 单平台搜索
python3 scripts/search.py search "iPhone手机壳" --platform taobao
```
