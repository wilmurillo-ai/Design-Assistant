# 小红书内容生成器 (Xiaohongshu Content Generator)

基于真实小红书运营经验封装的内容生成工具，帮助创作者快速产出符合平台调性的爆款内容。

## 功能特性

- ✅ **智能标题生成** - 数字型、痛点型、悬念型标题
- ✅ **正文结构化** - 钩子+干货+互动，符合平台算法偏好
- ✅ **标签自动匹配** - 精准标签组合，提升曝光率
- ✅ **多领域适配** - 育儿、知识、生活、职场等
- ✅ **合规安全** - 避免敏感词，降低封号风险

## 使用方法

### 方式1：直接调用

```
生成一篇关于[主题]的小红书笔记
```

### 方式2：指定风格

```
写一篇[育儿/知识/生活]类的小红书内容，主题是[主题]
```

### 方式3：批量生成

```

```

## 最佳实践

### 内容安全准则

1. **避免敏感词**
   - ❌ 不要用："最"、"第一"、"保证"
   - ❌ 避免：价格、医疗建议、政治

2. **控制发布频率**
   - 每天1-2条，避免批量操作
   - 分散在全天发布

3. **真人感优先**
   - AI生成 → 人工微调 → 发布
   - 保留个人语言习惯
