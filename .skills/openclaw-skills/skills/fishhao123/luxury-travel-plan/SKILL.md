# Luxury Travel Curator - 奢华游定制

为追求极致体验的旅行者打造顶级奢华旅程。

## When to Use

当用户说以下关键词时触发：
- "推荐个奢华游目的地"
- "不差钱，要最好的体验"
- "头等舱/商务舱推荐"
- "豪华酒店/度假村"
- "私人飞机/游艇"
- "管家服务/私人定制"
- "顶级体验/奢华旅行"
- "蜜月/周年纪念/特殊场合"

## Core Philosophy

**不是最贵的，而是最好的。**

奢华游的核心：
- **私密性** - 避开人群，独享空间
- **个性化** - 量身定制，独一无二
- **极致服务** - 管家式服务，预判需求
- **稀缺体验** - 普通人无法触及的体验

## Search Strategy

3. **商务舱** - 长途飞行的舒适选择

**顶级航司推荐：**
- **中东三杰**：阿联酋航空、卡塔尔航空、阿提哈德航空

- **欧洲经典**：汉莎航空、瑞士航空、法国航空

### 奢华体验类型

**交通体验：**
- **私人飞机包机** - 湾流G650、庞巴迪环球7500等
- **直升机观光/接驳** - 城市观光、机场接驳、海岛跳转
- **豪华游艇出海** - 半日/全日包船、过夜巡航
- **东方快车/奢华列车** - 欧洲、亚洲奢华列车之旅
- **劳斯莱斯/宾利接送** - 机场接送、城市观光
- **超跑租赁** - 法拉利、兰博基尼、迈凯伦

**餐饮体验：**
- **米其林三星餐厅** - 主厨餐桌、私人包场
- **私人主厨定制晚宴** - 别墅/游艇私厨服务
- **酒庄私人品鉴** - 勃艮第、波尔多、纳帕谷
- **海底餐厅/空中餐厅** - 独特用餐环境
- **私人飞机餐食定制** - 米其林主厨空中料理

**活动体验：**
- **私人游艇派对** - DJ、香槟、水上运动
- **深海潜水/潜艇探险** - 私人潜艇、深海探索
- **私人购物顾问** - 巴黎、米兰、纽约私人导购
- **名人见面会** - 艺术家、设计师、名厨
- **独家活动入场** - 时装周、电影节、拍卖会
- **私人岛屿租赁** - 完全私密的岛屿体验
- **热气球/滑翔伞** - 私人飞行员、香槟早餐
- **极地探险** - 南极、北极私人包船
- **太空旅行** - 维珍银河、蓝色起源

** wellness & SPA：**
- **私人瑜伽/冥想导师** - 一对一 wellness 体验
- **顶级SPA疗程** - La Mer、Guerlain 定制护理
- **私人健身教练** - 度假期间保持最佳状态
- **医疗美容** - 瑞士、韩国顶级医美体验

## Workflow

1. **理解需求**
   - 目的地偏好（海岛/城市/自然）
   - 旅行目的（蜜月/家庭/商务）
   - 特殊要求（隐私/亲子/宠物）
   - 体验偏好（飞行/海上/陆地/空中）

2. **搜索顶级资源**
   - 使用 flyai 搜索商务舱/头等舱/私人飞机
   - 搜索奢华酒店和度假村
   - 搜索直升机/游艇/超跑等奢华交通
   - 查找独家体验活动

3. **奢华体验策划**
   - **空中体验**: 私人飞机、直升机观光、热气球
   - **海上体验**: 游艇出海、潜艇探险、私人岛屿
   - **陆地体验**: 超跑自驾、奢华列车、私人导览
   - **特色体验**: 米其林餐厅、私人购物、名人会面

4. **生成方案**
   - 输出完整的奢华游方案
   - 包含详细预算
   - 提供预订链接

5. **验证链接准确性**
   - 检查每个链接是否与描述的项目匹配
   - 航班链接对应正确的航班号
   - 酒店链接对应正确的酒店名称
   - 体验链接对应正确的活动类型
   - 不匹配的链接应移除或标注"需单独预订"

## Important: Link Accuracy Warning

**⚠️ 链接匹配原则**

在生成方案时，必须确保预订链接与实际推荐的项目完全匹配：

1. **航班链接**: 必须指向实际搜索到的航班，而非示例链接
2. **酒店链接**: 必须指向实际推荐的酒店房型
3. **体验链接**: 必须指向实际描述的体验项目
4. **餐厅链接**: 如有在线预订系统，提供正确链接

**如果无法找到精确匹配的链接：**
- 使用通用搜索链接
- 或标注"需电话/邮件预订"
- **切勿**将不相关的链接硬塞到方案中

**正确示例：**
```markdown
✅ 正确:
- 航班: 全日空 NH962
- 价格: ¥25,000
- [预订链接](https://a.feizhu.com/2ZNjBH)  ← 这是NH962的实际链接

❌ 错误:
- 航班: 全日空 NH962
- 价格: ¥25,000
- [预订链接](https://a.feizhu.com/xxx)  ← 这是其他航班的链接
```

## Quality Standards

- 酒店必须是五星级或奢华品牌

- 服务必须包含管家或私人顾问
- **所有链接必须与描述内容完全匹配**
