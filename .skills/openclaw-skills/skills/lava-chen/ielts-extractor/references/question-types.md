# 题型标准

## 8种核心题型

| type | 中文名 | 选项位置 |
|------|--------|---------|
| `matching-headings` | 段落标题配对 | 题组级 |
| `matching-sentence` | 句子首尾配对 | 题组级 |
| `matching-label` | 标签配对 | 题组级 |
| `multiple-choice-single` | 多项选择(单) | 每题独立 |
| `multiple-answer` | 多项选择(多) | 题组级 |
| `yes-no-not-given` | 判断题 | 无 |
| `sentence-completion` | 完成句子 | 无/题组级 |
| `fill-blank-summary` | 摘要填空 | 无/原文片段 |
| `table-completion` | 表格完成 | 图片+题目 |

## 题型检查清单

- [ ] 文章字数: 1500-2500词/篇
- [ ] 总题数: 40道
- [ ] 选项完整: 单选每题4个选项(A-D)
- [ ] 答案对应: 答案与题目匹配
