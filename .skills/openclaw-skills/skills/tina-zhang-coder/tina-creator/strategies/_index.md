# 策略库

在 Grow 阶段沉淀的可复用创作策略。每次沉淀新策略时，按以下格式追加到本文件末尾。

---

## 格式模板

每条策略按以下结构记录：

```markdown
### [策略名称]

- **适用场景：** [什么类型的主题/领域适合用这个策略]
- **标题公式：** [标题的抽象模式，用占位符表示可替换部分]
- **内容框架：** [正文的组织结构，如"痛点引入 → 3步解法 → 金句收尾"]
- **钩子类型：** [开头的抓眼方式，如"数字冲击"、"反差对比"、"提问共鸣"]
- **封面风格：** [配色 + 版式 + 文案风格]
- **示例摘要：** [首次使用该策略时的笔记标题，作为参考]
- **沉淀日期：** [YYYY-MM-DD]
```
