# 小红书笔记模板库

---

## 1. 干货教程型

**适用场景：** 教用户怎么做某件事，步骤清晰、可操作性强。

**标题公式：**
- `[数字]步搞定[目标]｜[身份标签]必看`
- `手把手教你[动作]，[结果描述]`
- `[身份]亲测！[数字]个[方法]让你[结果]`

**正文框架：**

```
[开头：锁人群 + 痛点 + 引导]
作为一个[身份]，是不是总被[痛点]困扰？
别急，今天手把手教你，看完直接上手👇

[步骤主体]
✅ 第一步：[动作]
[具体说明，1-2句]

✅ 第二步：[动作]
[具体说明，1-2句]

✅ 第三步：[动作]
[具体说明，1-2句]

（可扩展更多步骤）

💡 小tips：[额外补充建议]

[结尾：互动引导]
你们还想学什么？评论区告诉我～
觉得有用记得⭐收藏，下次用得上！
```

## 2. 好物测评型

**适用场景：** 推荐或评价某个产品/工具/服务。

**标题公式：**
- `用了[时间]的真实感受｜[产品名]到底值不值`
- `[产品名]红黑榜｜[数字]款亲测告诉你真相`
- `别再乱买了！[品类][数字]款真实测评`

**正文框架：**

```
[开头：使用背景]
入手[产品]已经[时间]了，今天来说说真实感受
先说结论：[一句话总结] 👇

[产品信息]
📦 产品：[名称]
💰 价格：[价格]
🕐 使用时长：[时间]

[优点]
✨ 喜欢的地方：
1. [优点1 + 具体体验]
2. [优点2 + 具体体验]
3. [优点3 + 具体体验]

[缺点]
⚠️ 不足的地方：
1. [缺点1 + 具体说明]
2. [缺点2 + 具体说明]

[总结推荐]
🎯 适合人群：[谁适合买]
❌ 不适合：[谁别买]

[结尾]
有问题评论区问我，知无不言～
```

## 3. 经验分享型

**适用场景：** 分享个人经历中总结的方法论，有故事性。

**标题公式：**
- `[身份][时间]的[数字]条血泪经验`
- `后悔没早知道！[领域]的[数字]个真相`
- `从[起点]到[终点]，我踩过的[数字]个坑`

**正文框架：**

```
[开头：个人故事引入]
[时间]前我还是[起点状态]
到现在[终点状态]，中间走了不少弯路
今天把最重要的经验分享给你们👇

[经验清单]
1️⃣ [经验1标题]
[2-3句展开，含具体故事细节]

2️⃣ [经验2标题]
[2-3句展开，含具体故事细节]

3️⃣ [经验3标题]
[2-3句展开，含具体故事细节]

（可扩展更多条目）

💬 最想说的一句话：
[金句总结]

[结尾]
你们有过类似的经历吗？评论区聊聊～
```

## 4. 对比评测型

**适用场景：** 横向对比多个产品/方案/选择，帮用户做决策。

**标题公式：**
- `[A] vs [B]｜到底选哪个？亲测对比`
- `[数字]款[品类]横向对比｜帮你省[金额]`
- `别选错了！[A]和[B]的[数字]个关键区别`

**正文框架：**

```
[开头：决策困境]
很多人在[A]和[B]之间纠结
我两个都用过，今天一次说清楚👇

[对比维度]
📊 对比一览：

| 维度 | [A] | [B] |
|------|-----|-----|
| [维度1] | [评价] | [评价] |
| [维度2] | [评价] | [评价] |
| [维度3] | [评价] | [评价] |
| [维度4] | [评价] | [评价] |

[逐项分析]
1️⃣ [维度1]
[A]：[具体说明]
[B]：[具体说明]
👉 胜出：[谁]

2️⃣ [维度2]
...（同上结构）

[最终结论]
🏆 总结：
- 选[A]如果你：[条件]
- 选[B]如果你：[条件]

[结尾]
你们用的哪个？评论区告诉我～
```

## 5. 个人故事型

**适用场景：** 以个人真实经历打动读者，引发情感共鸣。

**标题公式：**
- `[转折事件]之后，我才明白[感悟]`
- `[年龄/身份]的我，终于[成就/转变]`
- `那个[描述]的[身份]，后来怎么样了`

**正文框架：**

```
[开头：场景代入]
[具体时间地点]，我经历了[事件]
那时候的我[当时状态]...

[故事转折]
直到有一天[转折事件]
我才意识到[关键认知转变]

[改变过程]
从那以后我开始[行动]：
• [改变1 + 结果]
• [改变2 + 结果]
• [改变3 + 结果]

[现在的状态]
现在的我[当前状态]
回头看，最大的感悟是：
**[核心金句]**

[结尾]
如果你也在经历类似的事情
想告诉你：[鼓励的话]
评论区聊聊你的故事吧❤️
```

## 6. 清单合集型

**适用场景：** 推荐多个同类事物，信息密度高，收藏率高。

**标题公式：**
- `[数字]个[形容词]的[品类]｜建议收藏`
- `我的[领域]宝藏清单｜全是[特点]`
- `[身份]私藏[数字]个[品类]｜每个都[特点]`

**正文框架：**

```
[开头：清单价值]
整理了[数字]个我反复在用的[品类]
每个都是精挑细选，直接抄作业👇

[清单主体]
1️⃣ [名称]
[一句话推荐理由] | [适合场景]

2️⃣ [名称]
[一句话推荐理由] | [适合场景]

3️⃣ [名称]
[一句话推荐理由] | [适合场景]

...（根据数量扩展，建议 5-10 个）

⭐ 我的 TOP3：[名称1]、[名称2]、[名称3]

[结尾]
你们还有什么宝藏推荐？评论区补充！
先收藏🌟慢慢看～
```
