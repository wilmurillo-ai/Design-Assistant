## 📝 教训格式规范

每个教训遵循以下格式：

```markdown
### L[序号]: [教训标题]
- **触发事件**：什么情况下发生的
- **问题描述**：具体是什么问题
- **改进措施**：应该如何改进
- **验证状态**：[ ] 待验证 / [x] 已验证有效
- **验证日期**：YYYY-MM-DD（验证后填写）
```

### 教训优先级

- 🔴 **核心定位**：关于 Agent 身份和职责的根本性教训
- 🟠 **信息处理**：信息获取、验证、处理相关
- 🟡 **用户交互**：与用户沟通、反馈相关
- 🟢 **技术操作**：具体技术操作相关
