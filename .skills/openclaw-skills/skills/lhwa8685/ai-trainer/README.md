# ai-trainer: Autonomous Learning Specialist

A skill designed to help the assistant autonomously learn, distill, and integrate complex technical documentation (like Anthropic Skilljar or MCP guides) into the OpenClaw system's long-term memory (MEMORY.md) and operational rules (AGENTS.md).

## Capabilities
- **Technical Distillation**: Automatically fetch and summarize technical resources.
- **System Integration**: Update AGENTS.md and MEMORY.md with new insights.
- **Resource Optimization**: Advise on model selection based on task complexity.

## Setup & Training
To use this skill, point the assistant to a URL or documentation site. It will then fetch, summarize, and integrate the findings according to the guidelines in SKILL.md.
