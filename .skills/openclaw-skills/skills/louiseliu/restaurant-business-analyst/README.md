# Restaurant Business Analyst

餐饮行业商业模式全景分析 AI Skill。输入一个餐饮品牌或业态，自动生成"战略 + 策略 + 指标"三位一体的综合分析报告。

## 功能特性

- **商业模式全景分析**：基于 Business Model Canvas 的餐饮行业9要素深度解读
- **近期战略动态**：联网搜索品牌最新新闻，关联商业模式变化
- **核心指标体系**：北极星指标 → 驱动指标 → 过程指标，三层自上而下拆解
- **餐饮经营专项**：单店P&L模型、成本四费分析、翻台率/坪效/人效效率三角、数字化成熟度评估
- **扩展分析模块**：组织能力诊断、区域分布分析、消费者洞察、出海分析、技术/AI评估、ESG评估
- **数据置信度标注**：★★★确/★★推/★估 三级，每个核心数字来源可追溯
- **双格式输出**：HTML 报告（暖色餐饮主题）+ 内置 PDF 导出按钮
- **业态确认机制**：自动检测输入品牌是否属于餐饮行业

## 报告结构

```
第一部分：商业模式全景（战略层）
  1.1 品牌概况 → 1.2 BMC画布 → 1.3 模式识别 → 1.4 演化阶段
  1.5 竞争壁垒 → 1.6 竞品矩阵 → 1.7 风险矩阵 → 1.8 组织能力

第二部分：近期战略动向（策略层）
  2.1 新闻摘要 → 2.2 战略选择 → 2.3 营销渠道 → 2.4 演化研判

第三部分：核心指标体系（指标层）
  3.1 北极星指标 → 3.2 驱动指标 → 3.3 过程指标 → 3.4 趋势解读
  3.5 单店P&L → 3.6 成本四费 → 3.7 数字化成熟度 → 3.8 区域分布

第四部分：餐饮经营专项分析
  4.1 单位经济模型 → 4.2 成本健康度 → 4.3 门店效率 → 4.4 供应链 → 4.5 数字化

第五部分：扩展分析（按需）
  E1 消费者洞察 → E2 出海分析 → E3 技术/AI评估 → E4 ESG评估

综合研判：战略-策略-指标三位一体解读
```

## 使用方式

本项目是一个 AI Agent Skill，设计为 Cursor IDE 的 Agent Skill 使用。

### 触发示例

```
分析海底捞的商业模式
帮我做茶饮行业的商业模式分析
给我一份瑞幸的餐饮战略分析报告
帮我拆解蜜雪冰城的增长策略
对比海底捞和呷哺呷哺的商业模式
```

### 安装

将本仓库克隆到你的 Agent Skills 目录即可：

```bash
git clone https://github.com/louiseliu/restaurant-business-analyst.git ~/.cursor/skills/restaurant-business-analyst
```

或者将整个文件夹复制到你使用的 AI IDE 的 Skills 目录。

## 数据置信度体系

报告中每个核心数字都标注置信度等级：

| 等级 | 标识 | 说明 |
|------|------|------|
| ★★★ | `[确]` | 官方财报、招股书、可交叉验证的权威数据 |
| ★★ | `[推]` | 权威第三方报告、可信媒体引用的内部数据 |
| ★ | `[估]` | 行业基准估算、间接推导的数据 |
