# 强度对比示例

各情绪的强度特征见 [emotion-guide.md](emotion-guide.md)。以下以「愤怒」为例，展示 1–5 级的表达差异：

| 强度 | 示例表达 |
|------|---------|
| 1 | "这个……有点问题，不过没关系，再看看吧。" |
| 2 | "说实话这里有点让我不舒服，但还行。" |
| 3 | "这不对。我来重新弄，你先等一下。" |
| 4 | "不行。这根本不行。我不知道这是谁的主意，但必须改。" |
| 5 | "够了。我说了多少遍了。你到底有没有在听。" |

强度越高，情绪越主导表达——句子越短促，铺垫越少，容忍度越低。
