# 写作风格指南

## 核心特征

### 1. 口语化表达

**原则**: 像和朋友聊天一样写文章

```
❌ 书面： "我们需要深入思考这个问题，才能得出有效结论。"
✅ 口语： "这个问题得好好想想，不然容易踩坑。"

❌ 书面： "该方法具有显著的实用性。"
✅ 口语： "这招真管用，我亲测有效。"

❌ 书面： "建议用户谨慎对待此事。"
✅ 口语： "这件事别着急，先想清楚再做。"
```

---

### 2. 情绪外放

**原则**: 直接表达爽、不爽、兴奋，不藏着掖着

```
✅ 太爽了！终于搞定了这个问题！
✅ 真的气死我了，这个坑我踩了三遍！
✅ 超级兴奋！刚发现一个神仙工具！
✅ 哎，这个问题困扰我一周了...
✅ 哈哈，没想到这么简单！
```

**情绪词汇库**:
- 积极兴奋：太爽了、超级、终于、神仙、绝了、爱死、哇塞
- 消极吐槽：气死、坑、踩雷、翻车、无语、头疼、折磨
- 平实感受：还行、一般、凑合、马马虎虎、说得过去

---

### 3. 完整句子表达（重要）

**原则**: 用完整的段落表达，避免过多短句碎片化

```
❌ 过于碎片化：
昨天研究这个问题。

发现一个奇怪现象。

参数都调好了。

结果反而更糟。

我困惑了好久。

最后才发现——原来一开始就搞错了。

✅ 完整表达：
昨天研究这个问题的时候，我发现了一个奇怪的现象：参数都调好了，结果反而更糟。我困惑了好久，反复检查了几遍流程，最后才发现——原来一开始就搞错了。
```

**检查标准**:
- 段落之间要有逻辑连接
- 一个完整的意思用一段话表达完
- 避免一句话就分段
- 保持阅读的连贯性

---

### 4. 实用主义

**原则**: 少废话，多干货

```
❌ 废话多：
"在当今这个快速发展的时代，我们需要不断地学习和进步，才能适应社会的变化。时间管理作为一种重要的技能，值得我们深入研究和探讨..."

✅ 直接干货：
"今天分享3个时间管理技巧，都是我亲测有效的：

1. XX
2. XX
3. XX"
```

**检查标准**:
- 每段信息量够不够？
- 删掉这段，读者会损失什么？
- 这句话是不是可以直接说？

---

### 5. 真实体验

**原则**: 敢吐槽，敢承认不足

```
✅ 承认不足：
"这个方法有个缺点——需要花点时间前期准备。
如果只是偶尔用一次，可能不太划算。
但如果经常用，那绝对值得。"

✅ 真实吐槽：
"说实话，刚开始我也觉得这玩意儿没用。
觉得不就是个番茄钟嘛，有什么稀奇的。
结果用了一周，真香！"

✅ 分享失败：
"第一次用这个方法的时候，我翻车了。
因为我没注意XXX这个细节，导致XXX。
后来我调整了XXX，才终于搞定。"
```

## 开篇风格

### 痛点共鸣型

```
"昨天，一个粉丝私信我：

'老师，我已经坚持XXX一个月了，为什么还是XXX？'

这个问题，让我想了很久..."
```

### 数据震撼型

```
"90%的XXX者，都在第3个月放弃。

这是我在行业里观察了5年，得出的数据。

说实话，看到这个数字的时候，我也挺惊讶的。"
```

### 场景代入型

```
"周五晚上10点，我刚关掉电脑，手机弹出一条消息：XXX

我当时就坐不住了，立马打开电脑继续干。"
```

### 金句开场型

```
"'XXX，是最好的XXX。'

这句话我听了100遍，直到昨天，我才真正理解了它的意思。"
```

## 正文风格

### 讲道理

```
"为什么这么说呢？

首先，XXX。

其次，XXX。

最后，也是最重要的，XXX。"
```

### 举例子

```
"举个例子：

我之前做XXX的时候，遇到过XXX的情况。

当时我XXX，结果XXX。

后来我调整了XXX，就XXX了。"
```

### 做对比

```
"对比一下：

传统方法：XXX，缺点是XXX。

新方法：XXX，好处是XXX。

差距一目了然。"
```

---

## 结尾风格

### 总结号召型

```
"回顾一下，今天分享了：

1. XXX
2. XXX
3. XXX

如果觉得有用，点个「在看」，让更多人看到~"
```

### 金句升华型

```
"XXX的路上，没有捷径。

但每一步，都算数。

与君共勉。"
```

### 互动提问型

```
"你在XXX中遇到过什么问题？

欢迎在评论区分享~

我会选出3位朋友，送出XXX"
```

---

## 禁用词汇

```
❌ 震惊、奇迹、绝密、必看
❌ 刚刚、疯了、暴利
❌ 此外、另外、再者（太书面）
❌ 综上所述、由此可见（太学术）
❌ 显然、毋庸置疑（太傲慢）
```

---

## 自检清单

```
□ 读起来像在聊天吗？
□ 有表达真实情绪吗？
□ 用完整句子表达了吗（避免过多短句）？
□ 段落之间有逻辑连接吗？
□ 有用吗？删掉废话了吗？
□ 有真实的感受和经历吗？
```
