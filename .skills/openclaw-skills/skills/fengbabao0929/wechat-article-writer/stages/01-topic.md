# 阶段01：选题定位

## 目标
明确文章的三个核心问题：写什么、给谁看、解决什么问题

---

## 执行步骤

### 步骤1：了解初步想法

询问用户：
```
你初步想写什么主题？
```

记录用户的初步想法。

---

### 步骤2：三问定位法

问用户三个关键问题：

**问题1：这篇文章写给谁看？**
- 行业/身份：如"职场新人"、"自媒体运营"、"产品经理"
- 痛点需求：如"想涨粉"、"想提高效率"、"想转型"

**问题2：看完这篇文章，读者能得到什么？**
- 具体收获：如"学会一个工具"、"了解一个方法"、"避免一个坑"
- 情绪价值：如"被理解"、"被激励"、"产生共鸣"

**问题3：你希望读者看完后做什么？**
- 行动：如"尝试方法"、"购买产品"、"关注账号"
- 传播：如"转发分享"、"收藏备用"

---

### 步骤3：选题确认

用以下格式向用户确认选题：

```
【选题确认】
主题：XXX
目标读者：XXX
核心价值：XXX
期待行动：XXX

是否确认这个方向？
```

---

## 输出模板

确认后，记录到 `topic.yaml`：

```yaml
topic: "文章主题"
target_audience: "目标读者描述"
core_value: "读者能获得的核心价值"
call_to_action: "期待读者采取的行动"
article_type: "干货教程/观点文章/故事案例/列表盘点"
estimated_length: 1500
```
