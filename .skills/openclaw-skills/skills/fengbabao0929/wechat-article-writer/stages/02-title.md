# 阶段02：标题优化

## 目标
生成5-10个高点击率标题供用户选择

---

## 执行步骤

### 步骤1：提取核心关键词

从选题中提取：
- 核心主题词
- 痛点/收益词
- 目标读者标签

---

### 步骤2：套用标题公式

使用以下公式生成标题变体：

**公式1：数字+收益**
```
"7个技巧，帮你XXX"
"3个月，我从XXX到XXX"
"90%的人都不知道的XXX方法"
```

**公式2：疑问+痛点**
```
"为什么你XXX了这么久，还是XXX？"
"明明很努力，为什么XXX？"
"XXX的真相，可能和你想的不一样"
```

**公式3：对比+反差**
```
"别人XXX，你却在XXX"
"同样是XXX，为什么结果差这么多？"
"放弃XXX后，我反而XXX了"
```

**公式4：时间+紧迫感**
```
"2026年，XXX的趋势变了"
"现在开始XXX，还来得及吗？"
"XXX前，你必须知道的3件事"
```

**公式5：身份+场景**
```
"致所有XXX的人：写给XXX的一封信"
"当XXX遇上XXX：我是如何XXX的"
"XXX年，我的XXX复盘"
```

---

### 步骤3：呈现选项

用以下格式展示：

```
【标题选项】

1. XXX
2. XXX
3. XXX
...

推荐：第X个，理由：XXX

请选择或修改：
```

---

## 注意事项

- 标题控制在15-25字
- 避免过度夸张（"震惊"、"奇迹"）
- 确保标题与内容匹配
- 考虑目标读者的阅读场景
