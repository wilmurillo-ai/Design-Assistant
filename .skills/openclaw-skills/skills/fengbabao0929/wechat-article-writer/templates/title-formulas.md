# 公众号标题公式库

## 数字型公式

| 公式 | 示例 |
|---|---|
| 数字+收益 | "7个技巧，帮你快速涨粉" |
| 数字+时间 | "30天，我从0到1万粉" |
| 数字+痛点 | "3个坑，90%的新手都会踩" |
| 年份+趋势 | "2026年，自媒体的5个新趋势" |

## 疑问型公式

| 公式 | 示例 |
|---|---|
| 为什么+痛点 | "为什么努力了，还是没结果？" |
| 难道只有我+感受 | "难道只有我觉得XXX吗？" |
| 怎么+行动 | "怎么用3小时，做完别人1天的事？" |
| 是不是+猜测 | "是不是你的XXX出了问题？" |

## 对比型公式

| 公式 | 示例 |
|---|---|
| 同样+结果差异 | "同样做XXX，为什么结果差这么多？" |
| 别人VS你 | "别人在XXX，你却在XXX" |
| 放弃+反而+ | "放弃XXX后，我反而XXX了" |
| 以为+实际 | "我以为XXX，实际XXX" |

## 身份型公式

| 公式 | 示例 |
|---|---|
| 致+人群 | "致所有30岁的你" |
| 作为一个+身份 | "作为一个XXX，我想说..." |
| 我是+身份+成就 | "我是XXX，这是我的故事" |
| 写给+人群 | "写给所有正在XXX的人" |

## 时间型公式

| 公式 | 示例 |
|---|---|
| 时间+来不及 | "2026年了，现在开始XXX还来得及吗？" |
| 时间前+必须 | "30岁前，你必须知道的3件事" |
| 时间+变化 | "入行5年，我对XXX的看法变了" |
| 时间+总结 | "回顾2025：我的XXX复盘" |

## 好标题的检验标准

```
✅ 15-25字
✅ 包含目标读者能识别的词
✅ 承诺具体价值
✅ 激发好奇心
✅ 与内容匹配
```
