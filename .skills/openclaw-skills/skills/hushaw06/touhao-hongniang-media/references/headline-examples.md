# 爆款标题示例库

## 数据冲击型

| 标题 | 数据点 | 效果分析 |
|------|--------|----------|
| 结婚率腰斩，相亲却卷疯了 | 腰斩（50%） | 反差制造，数据+现象对比 |
| 2024年结婚人数创40年新低 | 40年新低 | 时间跨度大，冲击力强 |
| 每100对结婚，42对在排队离婚 | 42%离结比 | 数据具象化，画面感强 |
| 90后结婚率不足20% | 20% | 人群标签+低数字冲击 |

## 反差制造型

| 标题 | 反差点 | 效果分析 |
|------|--------|----------|
| 嘴上说恐婚，身体却在相亲 | 口是心非 | 击中年轻人矛盾心态 |
| 公园相亲角挤爆，年轻人去哪了？ | 老人vs年轻人 | 两代人行为差异 |
| 不婚的年轻人，正在疯狂相亲 | 不婚vs相亲 | 标签反转，制造悬念 |
| 结婚率暴跌，婚庆公司却涨价 | 暴跌vs涨价 | 供需矛盾反差 |

## 标签共鸣型

| 标题 | 人群标签 | 痛点 |
|------|----------|------|
| 90后：相亲鄙视链最底端 | 90后 | 被歧视的焦虑 |
| 大龄剩女的相亲困境 | 大龄女性 | 年龄焦虑 |
| 程序员的相亲简历有多卷 | 程序员 | 职业内卷 |
| 小镇青年的相亲困局 | 小镇青年 | 城乡差异 |

## 悬念提问型

| 标题 | 悬念点 |
|------|--------|
| 结婚率连跌9年，年轻人为什么不结婚？ | 深度追问 |
| 父母相亲角的简历，为什么没人看？ | 现象追问 |
| 相亲APP的算法，真的能帮你找到对的人吗？ | 质疑式追问 |
| 彩礼涨了10倍，爱情呢？ | 反问式追问 |

## 情绪点燃型

| 标题 | 情绪点 |
|------|--------|
| 这届年轻人，终于对婚姻下手了 | 解构传统 |
| 别催了，这届年轻人真的不想结婚 | 反抗情绪 |
| 相亲100次后，我终于放弃了 | 失望/释然 |
| 我在相亲角被羞辱的那一天 | 愤怒/共情 |

## 标题自检清单

写完标题后，用以下问题自检：

- [ ] 是否包含数字/人群/冲突词之一？
- [ ] 长度是否在15-25字之间？
- [ ] 是否有传播欲望（想转发给别人）？
- [ ] 是否避免了"震惊""必看"等low词？
- [ ] 是否与文章内容匹配（不做标题党）？
