# 交互模式使用指南

## 🎯 什么是交互模式？

交互模式通过**多轮问答**帮您细化和明确需求，生成更精准的原型。

---

## 🚀 快速开始

### 方式 1：OpenClaw 聊天

```
用户：创建一个产品列表页 -i
```

### 方式 2：命令行

```bash
cd ~/.openclaw/skills/htmlPrototype
python3 main.py "创建一个产品列表页" --interactive
```

## ❓ 问题说明

### 风格选择

| 选项 | 说明 | 主色调 | 适用场景 |
|------|------|--------|----------|
| **1️⃣ 现代简约** | 默认，紫色渐变 | #667eea | 通用，SaaS 产品 |
| **2️⃣ 商务专业** | 深蓝色系 | #2c5282 | 企业后台，金融系统 |
| **3️⃣ 活泼清新** | 绿色系 | #38a169 | C 端产品，电商 |

### 表格列定制

**示例输入**：
- `产品名称，价格，库存，状态`
- `订单号，客户，金额，状态，时间`
- `用户名，邮箱，手机号，注册时间`

**提示**：用逗号分隔，支持中文或英文逗号

### 筛选条件

| 选项 | 包含组件 |
|------|----------|
| **1️⃣ 搜索框** | 关键词搜索 |
| **2️⃣ 下拉选择** | 状态下拉框 |
| **3️⃣ 日期选择** | 时间范围选择 |
| **4️⃣ 都需要** | 以上全部 |

### 操作按钮

| 选项 | 按钮组合 |
|------|----------|
| **1️⃣ 新建** | 只有新建按钮 |
| **2️⃣ 导出** | 新建 + 导出 |
| **3️⃣ 批量删除** | 新建 + 批量删除 |
| **4️⃣ 都需要** | 新建 + 导出 + 批量删除 |

## 💡 技巧

### 1. 使用默认值

直接回车使用默认选项，快速生成：

```
您的选择 [回车使用默认]: [直接回车]
```

### 2. 跳过某些问题

目前不支持跳过，但问题都经过精心设计，建议全部回答。

### 3. 修改答案

生成后查看效果，不满意可以重新运行并修改答案。

### 4. 保存配置

将常用配置保存为脚本：

```bash
# 保存为 product_list.sh
python3 main.py "创建一个产品列表页" -i << EOF
2
产品名称，价格，库存，状态，操作
4
4
1
EOF

# 下次直接运行
bash product_list.sh
```

## 🎓 常见问题

### Q: 可以跳过某个问题吗？

A: 目前不支持跳过，但所有问题都有默认值，直接回车即可。

### Q: 能自定义问题吗？

A: 可以编辑 `main.py` 中的 `generate_clarifying_questions()` 函数。

### Q: 回答错了怎么办？

A: 重新运行命令即可，每次都是独立的。

### Q: 能保存我的配置吗？

A: 可以将答案保存为 JSON 文件，下次直接加载（待实现）。
