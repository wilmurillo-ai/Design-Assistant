## 📊 评审报告样例

### 样例输入

```markdown
# XX 稳健增长混合型证券投资基金 PRD

## 产品概述
本产品为混合型基金，风险等级 R3...

## 功能需求
1. 基金认购
2. 基金申购
3. 基金赎回
...
```

### 样例输出

```markdown
# PRD 评审报告

**文档名称**: XX 稳健增长混合型证券投资基金 PRD  
**评审时间**: 2026-02-26 22:45  
**总体评分**: 85/100 ⚠️ 良好

## ✅ 通过项
- [x] 产品概述完整
- [x] 功能需求清晰
- [x] 合规检查点覆盖 10/10

## ⚠️ 待改进项

### 1. 完整性问题
- [ ] 缺失数据需求章节（建议补充）

### 2. 一致性问题
- [ ] 风险等级：产品概述 R3，风险揭示 R2（建议统一为 R3）

### 3. 术语问题
- [ ] "认购费率"应为"申购费率"（开放式基金）

## 📈 改进建议
1. **优先修复**: 统一风险等级为 R3
2. **建议补充**: 数据需求章节
3. **可选优化**: 术语统一

## 🎯 评审结论
⚠️ 需改进，改进后复审
```

## 🔧 使用方式

### 命令示例

```bash
# 评审 PRD 文档
node scripts/review.mjs prd.md

# 评审指定产品类型
node scripts/review.mjs prd.md --type fund

# 生成详细报告
node scripts/review.mjs prd.md --detailed

# 只检查合规项
node scripts/review.mjs prd.md --compliance-only

# 输出 JSON 格式
node scripts/review.mjs prd.md --json
```

### API 调用

```javascript
const reviewer = require('./requirement-reviewer');

const report = await reviewer.review({
  document: prdContent,
  productType: 'fund',
  riskLevel: 'R3'
});

console.log(`总体评分：${report.overall_score}`);
console.log(`评审结论：${report.conclusion.text}`);
```
