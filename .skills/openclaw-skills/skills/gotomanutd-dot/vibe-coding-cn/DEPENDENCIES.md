## 📦 依赖清单

### 核心功能

**无需额外依赖** ✅

核心功能（index.js + executors）使用 Node.js 原生模块：
- `fs` - 文件系统（原生）
- `path` - 路径处理（原生）
- `child_process` - 打开文件夹（原生）

**用户无需安装任何依赖！**

---

### 可选功能

#### 可视化监控服务器

**依赖**: `ws` (WebSocket 库)

**用途**: 启动 HTTP+WebSocket 服务器，用于可视化监控

**安装**（可选）:
```bash
npm install ws
```

**使用**:
```bash
npm start
# 访问 http://localhost:3000/ui/vibe-dashboard-v2.html
```

**注意**: 这是可选功能，不影响核心功能使用。

## 💡 为什么无需依赖？

### 设计原则

1. **最小化依赖** - 只使用 Node.js 原生模块
2. **零配置** - 用户无需手动安装
3. **快速启动** - 安装即可使用
4. **安全可靠** - 无第三方依赖风险

### 技术实现

```javascript
// index.js - 只使用原生模块
const fs = require('fs').promises;  // 原生
const path = require('path');       // 原生

// 不需要 ws、axios 等第三方库
```

## 🆘 常见问题

### Q: 需要安装 Node.js 吗？

A: 是的，需要 Node.js >= 18.0.0。OpenClaw 会自动检测。

### Q: 需要安装 npm 包吗？

A: **不需要！** 核心功能无需额外依赖。

### Q: 可视化监控必须安装吗？

A: 不，这是可选功能。核心功能完全可用。

### Q: 依赖冲突怎么办？

A: 核心功能无依赖，不会冲突。可选的 ws 是独立安装。
