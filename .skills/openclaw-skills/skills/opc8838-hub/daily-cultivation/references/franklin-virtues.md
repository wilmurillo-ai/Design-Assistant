# 富兰克林十三美德

> Benjamin Franklin's 13 Virtues - 经典美德修炼体系

---

## 美德列表

| 周次 | 美德 | 英文名 | 定义 |
|:---:|:---|:---|:---|
| 1 | 节制 | Temperance | 食不过饱，饮不过量 |
| 2 | 缄默 | Silence | 言必于人于己有益，避免无益的聊天 |
| 3 | 秩序 | Order | 每样东西放在该放的地方，每件事在规定的时间做 |
| 4 | 决心 | Resolution | 该做的事坚决做，做就做好 |
| 5 | 节俭 | Frugality | 花钱必须于人于己有益，杜绝浪费 |
| 6 | 勤勉 | Industry | 不浪费时间，做有益之事 |
| 7 | 真诚 | Sincerity | 实事求是，不虚伪 |
| 8 | 正义 | Justice | 不损人利己，履行应尽义务 |
| 9 | 中庸 | Moderation | 避免极端，容忍他人 |
| 10 | 整洁 | Cleanliness | 身体、衣服、住所保持清洁 |
| 11 | 镇静 | Tranquility | 不因小事、常事或不可避免之事慌乱 |
| 12 | 贞洁 | Chastity | 克制欲望，不损人名誉 |
| 13 | 谦虚 | Humility | 效仿耶稣和苏格拉底 |

## 修炼方法

### 周循环制

- 每周专注修炼**一项美德**
- 13周为一个完整周期
- 一年可完成约 4 个完整周期

### 每日记录

在晚间复盘中：
1. 回顾今日是否践行本周美德
2. 记录成功或失败的案例
3. 给自己打分（1-10分）

### 美德追踪

创建 `virtue-status.md` 文件记录当前状态：

```markdown
# 美德状态

当前周：第X周
当前美德：[美德名]
美德定义：[定义]
时间范围：YYYY-MM-DD 至 YYYY-MM-DD
```

## 自定义美德

用户可以自定义美德体系，替换富兰克林13美德：

```yaml
virtue_system:
  type: custom
  virtues:
    - name: 专注
      definition: 全力投入当下之事
    - name: 勇气
      definition: 面对恐惧依然行动
    # ... 最多13项
```
