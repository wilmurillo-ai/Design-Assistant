## 模板结构说明

### 第一部分：信息流宪法检查

```
【信息流宪法检查】
今日收藏 ≤ 3 条：
当日消化完毕：
未清不取，明日方续：
```

**目的**：
- 控制信息摄入量
- 确保当天消化
- 避免信息囤积

**检查要点**：
- 今天收藏/保存了多少条信息？
- 是否超过3条？
- 是否当天消化理解？
- 未消化的留到明天

---

### 第二部分：生理地基（能量系统）

```
一、生理地基（能量系统）
睡眠：
运动：
饮食：
```

**三大能量支柱**：

| 维度 | 检查要点 | 理想状态 |
|:---|:---|:---|
| 睡眠 | 时长、质量、规律 | 7-8小时，深睡眠充足 |
| 运动 | 类型、时长、强度 | 每日30分钟以上 |
| 饮食 | 三餐、营养、节制 | 规律、均衡、不过量 |

**意义**：
身体是修行的基础，能量充足才能支撑高层修炼。

---

### 第三部分：神经控制（前额叶训练）

```
二、神经控制（前额叶训练）
今天是否控制了一次本能冲动？
最难控制的那个瞬间是什么？
```

**核心训练**：
- 识别本能冲动（食欲、懒惰、情绪）
- 有意识地暂停
- 理性选择替代行为

**常见冲动类型**：
- 饮食冲动（想吃垃圾食品）
- 娱乐冲动（想刷短视频）
- 情绪冲动（想发火、想抱怨）
- 懒惰冲动（想拖延）

**意义**：
前额叶是大脑的"CEO"，控制冲动就是锻炼意志力肌肉。

---

### 第四部分：微善系统（人格气质）

```
三、微善系统（人格气质）
今天是否减少了一次世界的摩擦？
有没有做一件让别人感到温暖的小事？
```

**两大修炼**：

1. **减少摩擦** — 不给他人添麻烦
   - 遵守规则
   - 礼貌待人
   - 不抱怨、不指责

2. **主动行善** — 给他人带来温暖
   - 一个微笑
   - 一句鼓励
   - 一次帮助

**意义**：
人格气质在日常细节中体现，微善积累成大德。

---

### 第五部分：美德修炼

```
四、美德修炼
本周美德：{美德名}（{美德定义}）

今日践行：
不说谎：
表达真实想法：
承认错误：

美德打分：___/10分
```

**美德践行检查**：

| 项目 | 说明 |
|:---|:---|
| 不说谎 | 今天是否诚实？ |
| 表达真实想法 | 是否勇于表达？ |
| 承认错误 | 是否承认过失？ |

**打分标准**：

| 分数 | 标准 |
|:---:|:---|
| 9-10 | 完美践行，有突破性感悟 |
| 7-8 | 良好践行，基本达标 |
| 5-6 | 一般，有改进空间 |
| 3-4 | 较差，需要反思 |
| 1-2 | 严重偏离，需要调整 |

---

### 第六部分：今日产出

```
五、今日产出
今天最重要的一件事是什么？
完成度如何？
```

**聚焦原则**：
- 每天只设定一件最重要的事
- 全力推进这件事
- 不求多，但求聚焦

**复盘要点**：
- 是否完成了？
- 如果没完成，卡在哪里？
- 明天如何推进？

---

六、今日感悟/金句

**来源**：

**意义**：
记录每日智慧，积累个人成长档案。

## 复盘价值

### 为什么复盘？

1. **觉察** — 看见自己的行为模式
2. **纠正** — 发现问题，及时调整
3. **积累** — 日拱一卒，持续精进
4. **记录** — 留下成长轨迹

### 复盘的力量

```
不复盘：今天就这样过去了 → 明天重复 → 一年365天，只有1天

有复盘：今天觉察 → 明天改进 → 日积月累 → 一年365天，有365次迭代
```

## 保存建议

建议将每日复盘保存到 Obsidian vault：

```
your-vault/
└── 每晚复盘/
    ├── 2026-04-14-晚复盘.md
    ├── 2026-04-15-晚复盘.md
    └── ...
```

长期积累后，你将拥有：
- 一年的成长记录
- 美德践行的轨迹
- 个人进化的证据

## 自动存档

用户完成复盘后，系统会自动保存到本地：

**推荐路径**：`{vault}/每晚复盘/{YYYY-MM-DD}-晚复盘.md`

**结合 Obsidian 使用**：

1. 将 vault 设置为 Obsidian vault
2. 每日复盘自动保存
3. 可在 Obsidian 中：
   - 搜索历史复盘（Ctrl/Cmd + P）
   - 建立 `#复盘` 标签
   - 使用 Dataview 插件分析美德践行趋势
   - 追踪个人成长曲线

**示例：Dataview 查询**

```dataview
TABLE
  file.ctime as "日期",
  choice(contains(file.content, "美德打分：8"), "优秀", "良好") as "表现"
FROM "每晚复盘"
SORT file.ctime DESC
LIMIT 7
```

**形成修炼库的价值**：

| 功能 | 说明 |
|:---|:---|
| 回顾 | 翻看过去，发现成长 |
| 搜索 | 快速找到特定内容 |
| 分析 | 用插件统计美德得分趋势 |
| 分享 | 可导出分享给导师/朋友 |
| 传承 | 长期积累，成为人生财富 |

## 自定义配置

### 修改复盘模板

在 `daily-cultivation-config.yaml` 中指定自定义模板：

```yaml
evening:
  reflection:
    template: templates/my-evening.md
```

### 自定义复盘模块

可以在模板中添加/删除模块：

- 添加"今日感恩"模块
- 添加"明日计划"模块
- 删除不需要的模块
