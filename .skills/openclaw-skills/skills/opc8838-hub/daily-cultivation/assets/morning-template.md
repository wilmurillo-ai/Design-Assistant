# 晨语模板

> 每日晨间智慧提醒 - 开启美好的一天

---

## 完整模板示例

```
🎓 **Master 晨语**

早安，朋友！🌅
今天是2026年04月14日，星期二

【信息流宪法】信息如饮食，过则为毒。日取三则，当日化之，未清不取。

今日美德：**整洁**（身体、衣服、住所保持清洁）

每日提醒：身体节律稳定是根基，情绪稳定是支柱，脑子清醒是内核

今日大师智慧

**段永平**：做对的事情，把事情做对。三十年以来，我们犯的错误就是比其他人少。人们关心我们做过什么，其实我们之所以成为我们，很重要的原因是我们不做的那些事情。

**富兰克林**：保持清洁不仅是身体的需要，更是心灵的修炼。环境有序，心智清明。

**万维钢**：秩序不是约束，而是释放。整理好你的环境，才能释放你的认知能力。

**冯仑**：把自己的事做好，把自己的屁股擦干净，这就是最大的尊严。

**乔布斯**：简单是终极的复杂。整理，就是剥离多余，留下本质。

**吴军**：一个人书房或桌面，反映了他思维的秩序程度。物理空间的混乱，往往源于思维的混乱。

【寄语】

整洁不只是一周的主题，更是一种生活态度的体现。从整理身边的秩序开始，让内心的秩序也随之清明。愿你在这周里，从小事做起，让整洁成为习惯，让习惯成为力量！

---

新的一天，整洁周继续。
```

## 模板结构说明

### 第一部分：问候与日期

```
早安，{user_greeting}！🌅
今天是{YYYY}年{MM}月{DD}日，星期{X}
```

**变量**：
- `{user_greeting}` - 用户称呼（可配置，如"朋友"、"dear friend"）
- `{日期}` - 自动获取当天日期

---

### 第二部分：信息流宪法

```
【信息流宪法】信息如饮食，过则为毒。日取三则，当日化之，未清不取。
```

**含义**：
- 信息如饮食，过量则成毒
- 每天只取3条，当天消化
- 未清空不取新信息

---

### 第三部分：今日美德

```
今日美德：**{美德名}**（{美德定义}）
```

**来源**：`virtue-status.md`

**富兰克林13美德**：

| 周次 | 美德 | 定义 |
|:---:|:---|:---|
| 1 | 节制 | 食不过饱，饮不过量 |
| 2 | 缄默 | 言必于人于己有益，避免无益的聊天 |
| 3 | 秩序 | 每样东西放在该放的地方，每件事在规定的时间做 |
| 4 | 决心 | 该做的事坚决做，做就做好 |
| 5 | 节俭 | 花钱必须于人于己有益，杜绝浪费 |
| 6 | 勤勉 | 不浪费时间，做有益之事 |
| 7 | 真诚 | 实事求是，不虚伪 |
| 8 | 正义 | 不损人利己，履行应尽义务 |
| 9 | 中庸 | 避免极端，容忍他人 |
| 10 | 整洁 | 身体、衣服、住所保持清洁 |
| 11 | 镇静 | 不因小事、常事或不可避免之事慌乱 |
| 12 | 贞洁 | 克制欲望，不损人名誉 |
| 13 | 谦虚 | 效仿耶稣和苏格拉底 |

---

### 第四部分：每日提醒

```
每日提醒：身体节律稳定是根基，情绪稳定是支柱，脑子清醒是内核
```

**四层监督体系**：
1. 生理地基 — 睡眠、运动、饮食
2. 神经控制 — 前额叶训练、情绪管理
3. 微善系统 — 不作恶、微主动、结构性善
4. 结构输出 — 个人稳定 → 小范围连接 → 长期影响

---

### 第五部分：大师智慧

```
今日大师智慧

**{大师1}**：{语录1}
**{大师2}**：{语录2}
**{大师3}**：{语录3}
**{大师4}**：{语录4}
**{大师5}**：{语录5}
**{大师6}**：{语录6}
```

**来源**：`quotes/*.md`

**抽取规则**：
- 每天从每位大师的语录库中随机抽取1条
- 避免重复（追踪已发送记录）
- 最多支持6位大师

---

### 第六部分：寄语

```
【寄语】

{美德相关感悟}

---

新的一天，{美德名}周继续。
```

**寄语生成**：
- 结合本周美德主题
- 提供激励性感悟
- 鼓励践行

## 自定义配置

### 修改问候语

在 `daily-cultivation-config.yaml` 中：

```yaml
user_greeting: "朋友"  # 可改为任意称呼
```

### 自定义大师

```yaml
morning:
  wisdom:
    masters:
      - name: 你的大师1
        quotes_file: quotes/大师1.md
      - name: 你的大师2
        quotes_file: quotes/大师2.md
      # ... 最多6位
    daily_count: 6
```

### 修改信息流宪法

可直接在模板中修改此部分。

---

## 发送效果

晨语将发送到用户配置的渠道：

- **飞书**：私聊或群聊
- **Discord**：指定频道
- **Telegram**：私聊或群组
