# 🌅 人格修行系统 (Daily Cultivation)

> 每天10分钟，晨间智慧，晚上记录精进，持续修炼更好的自己

## ✨ 这是什么？

人格修行系统是一套每日自我修炼的完整流程，帮助你：

- 📥 **晨间输入** — 接收大师智慧 + 美德提醒
- 📤 **晚间输出** — 自我复盘 + 美德践行
- 🔄 **日积月累** — 持续成为更强大、更稳定、更有影响力的自己

## 📋 模板详解

### 晨语模板结构与逻辑

#### 第一部分：问候与日期

```
早安，朋友！🌅
今天是2026年04月14日，星期二
```

**目的**：开启新的一天，建立仪式感。

---

#### 第二部分：信息流宪法

```
【信息流宪法】信息如饮食，过则为毒。日取三则，当日化之，未清不取。
```

**核心逻辑**：

| 原则 | 含义 | 实践方法 |
|:---|:---|:---|
| 信息如饮食 | 信息过量如饮食过量，会中毒 | 控制输入量 |
| 日取三则 | 每天最多关注3条信息 | 设定上限 |
| 当日化之 | 当天的信息当天消化 | 不拖延 |
| 未清不取 | 未消化完不取新信息 | 保持清爽 |

**为什么这样做？**

现代人的问题不是信息太少，而是信息太多。每天被无数信息轰炸，导致：
- 注意力被分散
- 深度思考能力下降
- 焦虑感增加

信息流宪法帮助你：
- 重新掌控注意力
- 只关注真正重要的信息
- 保持大脑清爽

---

#### 第三部分：今日美德

```
今日美德：**整洁**（身体、衣服、住所保持清洁）
```

**来源**：基于本周美德轮转

**为什么是每周一项美德？**

- 富兰克林亲身实践：每周专注一项，深入修炼
- 避免贪多：一次只修一项，不分散精力
- 形成习惯：一周足够让美德内化

---

#### 第四部分：每日提醒

```
每日提醒：身体节律稳定是根基，情绪稳定是支柱，脑子清醒是内核
```

**四层监督体系**：

```
┌─────────────────────────────────────┐
│  第四层：结构输出                     │
│  个人稳定 → 小范围连接 → 长期影响      │
├─────────────────────────────────────┤
│  第三层：微善系统                     │
│  不作恶、微主动、结构性善              │
├─────────────────────────────────────┤
│  第二层：神经控制 ← 你在这里           │
│  前额叶训练、情绪管理                 │
├─────────────────────────────────────┤
│  第一层：生理地基                     │
│  睡眠、运动、饮食                     │
└─────────────────────────────────────┘
```

**为什么强调前额叶？**

前额叶是大脑的"CEO"，负责：
- 决策和规划
- 冲动控制
- 情绪调节
- 长期目标追求

现代生活的诱惑（短视频、垃圾食品、冲动消费）都在攻击你的前额叶。

**每天提醒的目的**：
- 唤醒前额叶意识
- 提醒你掌控自己的大脑
- 不要被本能和情绪绑架

### 晚复盘模板结构与逻辑

#### 第一部分：信息流宪法检查

```
【信息流宪法检查】
今日收藏 ≤ 3 条：✓ 2条
当日消化完毕：✓ 已读完并整理
未清不取，明日方续：无未消化内容
```

**目的**：将晨间的提醒转化为晚间检查

**检查要点**：
- 今天收藏了多少信息？
- 是否超过3条？
- 是否消化完毕？

**如果不达标怎么办？**
- 反思为什么超量
- 明天调整策略
- 不追求完美，但要觉察

---

#### 第二部分：生理地基（能量系统）

```
一、生理地基（能量系统）
睡眠：昨晚 7.5 小时，质量良好
运动：步行 30 分钟
饮食：三餐规律，午餐略油腻
```

**三大能量支柱**：

| 维度 | 检查要点 | 理想状态 |
|:---|:---|:---|
| 睡眠 | 时长、质量、规律 | 7-8小时，深睡眠充足 |
| 运动 | 类型、时长、强度 | 每日30分钟以上 |
| 饮食 | 三餐、营养、节制 | 规律、均衡、不过量 |

**为什么从生理开始？**

身体是修行的基础：
- 睡眠不足 → 前额叶功能下降 → 自控力下降
- 缺乏运动 → 能量不足 → 情绪波动
- 饮食不当 → 血糖波动 → 专注力下降

**先搞定身体，再谈其他。**

#### 第三部分：神经控制（前额叶训练）

```
二、神经控制（前额叶训练）
今天是否控制了一次本能冲动？
是的，下午想吃甜食时忍住了。

最难控制的那个瞬间是什么？
晚饭后想刷短视频，但强迫自己先完成复盘。
```

**核心训练**：
- 识别本能冲动（食欲、懒惰、情绪）
- 有意识地暂停
- 理性选择替代行为

**常见冲动类型**：

| 冲动类型 | 表现 | 前额叶应对 |
|:---|:---|:---|
| 饮食冲动 | 想吃垃圾食品 | 喝水、吃健康零食 |
| 娱乐冲动 | 想刷短视频 | 设定时间、做其他事 |
| 情绪冲动 | 想发火、想抱怨 | 暂停、深呼吸 |
| 懒惰冲动 | 想拖延 | 先做5分钟 |

**为什么训练前额叶？**

前额叶像肌肉，可以训练：
- 每次控制冲动 = 做一次"前额叶俯卧撑"
- 长期训练 → 意志力增强
- 意志力增强 → 人生掌控力增强

**记录"最难控制"的价值**：
- 发现自己的弱点
- 针对性改进
- 追踪进步

---

#### 第四部分：微善系统（人格气质）

```
三、微善系统（人格气质）
今天是否减少了一次世界的摩擦？
帮同事解答了一个问题，节省了他半小时。

有没有做一件让别人感到温暖的小事？
给家人发了条关心消息。
```

**两大修炼**：

**1. 减少摩擦** — 不给他人添麻烦

- 遵守规则
- 礼貌待人
- 不抱怨、不指责
- 守时、守信

**2. 主动行善** — 给他人带来温暖

- 一个微笑
- 一句鼓励
- 一次帮助
- 一条关心消息

**为什么是"微"善？**

- 不是要做大善人
- 不是要牺牲自己
- 而是：每天一个小小的善意
- 积累起来 = 人格气质的改变

**结构性善**：
- 不是偶尔做一次
- 而是成为习惯
- 成为你的"出厂设置"

## 🚀 快速开始

### 安装

1. 下载 `daily-cultivation.skill` 文件
2. 在 OpenClaw 中安装 Skill
3. 首次使用时，Agent 会引导你完成配置

### 配置

创建配置文件 `daily-cultivation-config.yaml`：

```yaml
# 基础配置
system_name: "人格修行系统"
user_greeting: "dear friend"
timezone: "Asia/Shanghai"

# Obsidian vault 路径（用于自动存档）
obsidian_vault: "D:/obisian/master"

# 晨间设置
morning:
  enabled: true
  time: "08:00"
  wisdom:
    masters:
      - name: 段永平
        quotes_file: quotes/段永平.md
      - name: 富兰克林
        quotes_file: quotes/富兰克林.md
      # ... 最多6位
  virtue:
    enabled: true
    system: franklin  # 或 custom

# 晚间设置
evening:
  enabled: true
  time: "22:00"
  auto_save:
    enabled: true
    save_path: "每晚复盘"

# 发送渠道
channels:
  - type: feishu
    target: "user:ou_xxx"
```

### 语录库

创建语录文件 `quotes/段永平.md`：

```markdown
# 段永平语录

1. **做对的事情，把事情做对。**（浙大演讲）
2. **买股票就是买公司。**（投资原则）
3. **不懂不做，不熟不做。**（投资原则）
...
```

## 🎨 自定义

### 自定义大师

在配置文件中添加你喜欢的大师：

```yaml
morning:
  wisdom:
    masters:
      - name: 你的大师1
        quotes_file: quotes/大师1.md
      - name: 你的大师2
        quotes_file: quotes/大师2.md
      # ... 最多6位
```

### 自定义美德

完全自由定义你的美德体系：

```yaml
morning:
  virtue:
    system: custom
    virtues:
      - name: 专注
        definition: 全力投入当下之事
      - name: 勇气
        definition: 面对恐惧依然行动
      - name: 诚实
        definition: 对自己诚实，也对他人诚实
      - name: 感恩
        definition: 每天记录三件值得感恩的事
      # ... 数量自由定义，建议 4-13 个
```

**美德数量建议**：

| 数量 | 循环周期 | 适用场景 |
|:---:|:---|:---|
| 4 个 | 1个月 | 快速迭代，适合新手 |
| 6 个 | 1.5个月 | 平衡节奏 |
| 12 个 | 3个月 | 深度修炼 |
| 13 个 | 3个月 | 富兰克林经典 |

### 晚复盘自动存档

用户完成复盘后，自动保存到本地：

```yaml
evening:
  auto_save:
    enabled: true
    save_path: "每晚复盘"
    filename_format: "{YYYY}-{MM}-{DD}-晚复盘.md"
```

**推荐**：结合 Obsidian 使用

```
your-vault/
└── 每晚复盘/
    ├── 2026-04-01-晚复盘.md
    ├── 2026-04-02-晚复盘.md
    └── ...
```

**价值**：
- 形成个人修炼库
- 可在 Obsidian 中搜索、分析
- 长期追踪成长轨迹

## 🔄 工作原理

```
┌─────────────────────────────────────────────────┐
│                   每日循环                        │
├─────────────────────────────────────────────────┤
│                                                 │
│  🌅 晨间 (08:00)                                │
│  ┌─────────────┐                                │
│  │  大师智慧    │  ← 从语录库抽取                 │
│  │  +          │                                │
│  │  美德提醒    │  ← 从美德状态读取               │
│  └─────────────┘                                │
│       ↓ 输入                                    │
│                                                 │
│  💼 白天                                        │
│  ┌─────────────┐                                │
│  │  工作/生活   │                                │
│  │  美德践行    │                                │
│  └─────────────┘                                │
│       ↓ 践行                                    │
│                                                 │
│  🌙 晚间 (22:00)                                │
│  ┌─────────────┐                                │
│  │  自我复盘    │  → 记录成长轨迹                 │
│  │  +          │                                │
│  │  美德打分    │  → 量化修行进度                 │
│  └─────────────┘                                │
│       ↓ 输出                                    │
│                                                 │
└─────────────────────────────────────────────────┘
         ↓
    日积月累，持续精进
```

## 💡 设计理念

### 输入-输出闭环

**晨间输入**：大师智慧滋养心灵，美德提醒校准方向

**晚间输出**：自我复盘觉察问题，美德打分量化进度

### 日拱一卒

每天10分钟，不贪多、不求快，持续积累。

### 系统化修炼

- **生理地基** — 睡眠、运动、饮食（身体是基础）
- **神经控制** — 前额叶训练、情绪管理（掌控大脑）
- **微善系统** — 不作恶、微主动、结构性善（人格气质）
- **结构输出** — 个人稳定 → 小范围连接 → 长期影响（向外辐射）
