## Mapping Rules

1. Extract all recognizable imagery from the dream description
2. Each image maps to one visual_element
3. visual_elements array contains at most 5 elements (prevent visual clutter)
4. Prioritize imagery related to the main plot
5. Must include at least 1 atmosphere modifier element
6. mood takes the strongest emotion from the dream; color_scheme matches mood
7. When multiple traditions are equally relevant, prefer the atmosphere modifier that best matches the dream's emotional tone
