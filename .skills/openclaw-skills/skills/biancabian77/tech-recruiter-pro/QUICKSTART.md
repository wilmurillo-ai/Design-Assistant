## 🆘 遇到问题

### 搜索结果太少

**原因**: 条件太严格

**解决**:
```
放宽搜索条件
减少关键词数量
尝试不同平台
```

### 邮件回复率低

**原因**: 个性化不够

**解决**:
```
增加论文/项目细节
强调独特技术优势
调整邮件语气
```

### 候选人不回复

**原因**: 时机不对/不感兴趣

**解决**:
```
3-5 天后跟进一次
更换联系方式
调整目标人群
```

## 🎓 最佳实践

### ✅ 应该做的

- 个性化每封邮件（提及具体论文/项目）
- 跟进 2-3 次（间隔 3-5 天）
- 诚实介绍职位和团队
- 尊重候选人意愿

### ❌ 不应该做的

- 群发通用模板
- 每天跟进
- 夸大职位吸引力
- 频繁联系
