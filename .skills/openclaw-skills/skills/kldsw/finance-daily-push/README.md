# 📖 金融日报推送技能 - 使用说明

## 🎯 技能功能

本技能提供 A 股科技方向的金融日报推送服务，包含：

- **早报**（工作日 8:00）：隔夜外盘 + A 股预告 + 个股推荐
- **晚报**（工作日 20:00）：今日行情 + 消息解读 + 个股推荐
- **周报**（周日 20:00）：本周回顾 + 下周展望 + 个股推荐

## 📁 技能文件

```
finance-daily-push/
├── SKILL.md              # 技能说明
├── prompt-早报.md         # 早报模板
├── prompt-晚报.md         # 晚报模板
├── prompt-周报.md         # 周报模板
├── scripts/
│   └── fetch-quote.py    # 行情获取脚本
└── README.md             # 本文件
```

## 🚀 快速开始

### 方式一：使用 cron 定时推送

```bash
# 进入技能目录
cd ~/.openclaw/skills/finance-daily-push

# 添加早报任务（工作日 8:00）
cron add --name "科技金融早报" \
  --schedule "0 8 * * 1-5" \
  --prompt-file prompt-早报.md

# 添加晚报任务（工作日 20:00）
cron add --name "科技金融晚报" \
  --schedule "0 20 * * 1-5" \
  --prompt-file prompt-晚报.md

# 添加周报任务（周日 20:00）
cron add --name "科技周报·下周展望" \
  --schedule "0 20 * * 0" \
  --prompt-file prompt-周报.md
```

### 方式二：手动调用

直接使用对应模板的内容作为 prompt：

```
请整理一份科技方向的金融日报（晚报版），重点聚焦 A 股...
```

### 方式三：使用脚本获取行情

```bash
# 获取默认指数行情
python3 scripts/fetch-quote.py

# 获取指定股票行情
python3 scripts/fetch-quote.py sh000001,sz399006,sz000001
```

## 📊 数据来源配置

技能默认使用以下数据源：

| 数据类型 | 来源 | 说明 |
|----------|------|------|
| 行情数据 | 腾讯财经 API | 实时获取 |
| 重要消息 | 同花顺财经 | 24 小时内 |
| 重要消息 | 东方财富网 | 24 小时内 |
| 公司公告 | 交易所公告 | 实时 |

## ✅ 内容标准

- **行情数据**：必须标注来源和时间
- **重要消息**：15-20 条精选
- **个股推荐**：3-5 只，带逻辑分析
- **独立分析**：每条消息标注利好/利空/中性

## ❌ 不收录内容

- 北向资金（非实时数据）
- 超过 24 小时的旧闻
- 无法验证的自媒体内容
- 对 A 股无直接影响的消息

## 🔧 自定义配置

### 修改推送时间

编辑 cron 任务的 schedule 字段：

```json
{
  "schedule": {
    "kind": "cron",
    "expr": "0 8 * * 1-5"  // 修改这里
  }
}
```

### 修改推送内容

编辑对应的 prompt 模板文件：

- `prompt-早报.md`
- `prompt-晚报.md`
- `prompt-周报.md`

### 添加新的数据源

在 prompt 模板的"重要：数据与消息来源"部分添加新的数据源。

## ⚠️ 注意事项

1. **数据准确性**：所有行情数据必须来自腾讯财经 API
2. **消息时效性**：只收录 24 小时内的新消息
3. **独立分析**：每条消息必须有独立判断
4. **免责声明**：每条推送必须包含免责声明
