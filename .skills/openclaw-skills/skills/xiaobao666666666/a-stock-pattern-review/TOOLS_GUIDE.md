# A股复盘工具使用指南

## 一、情绪监控工具

### 880005（昨日涨停股表现）
- 数值>0：追板资金赚钱，市场情绪好
- 数值<-5%：冰点区，进攻窗口可能开启

### 连板率
| 连板率 | 市场状态 | 操作策略 |
|-------|---------|---------|
| >30% | 高潮期 | 锁定最强 |
| 15-30% | 回暖期 | 积极试错 |
| <15% | 冰点期 | 等待 |
| <5% | 冰点深化 | 空仓 |

### 炸板率
- >30%：情绪不稳，追板风险大
- <20%：情绪稳定
- >50%：退潮信号，禁止追板

### 红盘率
- >60%：追板有溢价
- <40%：控制仓位
- <30%：减少操作

## 四、主线识别

### 板块生命周期
```
启动期（1-2天）→ 龙头首板，识别难度大
发酵期（3-5天）→ 跟风批量涨停，确认主线
高潮期（5-10天）→ 龙头连板加速，高位分歧
退潮期 → 龙头跌停，主线结束
```

## 五、容量唯一性识别

### 容量标的特征
- 流通市值>50亿，日成交额>10亿
- 同题材中盘子最大
- 板块调整时抗跌

### 活口特征
- 退潮期唯一没跌停的高位股
- 分时图逆势拉升
- 成交回报显示大资金买入

### 流动性切换信号
- 老主线龙头高位跌停
- 新方向首板出现
- 资金快速切换到新板块
