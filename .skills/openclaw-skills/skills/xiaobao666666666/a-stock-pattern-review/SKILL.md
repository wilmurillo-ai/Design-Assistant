# A股短线交易复盘框架

## 触发条件
当用户发送以下关键词时触发：
- "复盘"、"盘面分析"、"市场情绪"
- "资金流向"、"主线"、"龙头"
- "涨停"、"冰点"、"加仓"、"撤退"
- "操作预案"、"明日剧本"

## 核心理念
> "做情绪拐点上的容量唯一性活口，或者流动性切换的新方向"

- **容量唯一性**：同一题材中，只有它能承载大资金
- **活口**：主线分歧时，唯一活下来的标的
- **流动性切换**：资金从老方向切换到新方向时的承接点

## 二、市场情绪周期

### 情绪钟摆位置

| 钟摆位置 | 特征 | 信号 |
|---------|------|------|
| **冰点区** | 涨停稀少、情绪低迷、连板率低 | 进攻窗口开启 |
| **回暖区** | 连板出现、昨日涨停有溢价 | 积极试错 |
| **高潮区** | 批量涨停、连板率高、龙头一字 | 锁定最强 |
| **退潮区** | 炸板率高、龙头跌停、亏钱效应扩散 | 收缩防守 |

### 判断工具
- **通达信880005**：昨日涨停股今日整体表现
- **连板率**：连板股占比
- **炸板率**：炸板股占比
- **红盘率**：昨日涨停今日红盘比例

### 输出格式
```
【情绪周期定位】
当前阶段：[增量/存量/退潮]
钟摆位置：[冰点/回暖/高潮/退潮]
关键指标：
 - 880005昨日涨停表现：[XX%]
 - 连板率：[XX%]
 - 炸板率：[XX%]
 - 红盘率：[XX%]
结论：[进攻窗口/积极试错/锁定最强/收缩防守]
```

## 三、资金流向与主线博弈

### 分析维度
1. **资金流向**：主力净流入板块排名，板块内个股结构
2. **持续性预期**：催化逻辑、板块容量、龙头表现
3. **核心标的识别**：

| 标的类型 | 定义 | 特征 |
|---------|------|------|
| **身位高标** | 当前连板最高或启动最早的标的 | 代表板块空间 |
| **弹性核心** | 小盘子、情绪驱动的涨停股 | 代表赚钱效应 |
| **容量中军** | 大盘子、机构/大游资参与的核心资产 | 代表资金容量 |

### 主线判断标准
```
有效主线特征：
✓ 有明确题材催化
✓ 涨停家数≥3家
✓ 有身位龙头（3板+）
✓ 板块内结构完整（龙头→跟风→补涨）
✓ 持续性≥2天

无效轮动特征：
✗ 一日游行情
✗ 无身位龙头
✗ 板块内个股独立炒作
✗ 纯消息刺激无实质逻辑
```

## 四、明日剧本与操作预案

### 三种情形

**情形A：进攻预案（冰点回暖时）**
```
触发：情绪从冰点回暖，出现首个3板+
目标：容量唯一性活口
买入：开盘恐慌低吸 / 板上打板确认
仓位：轻仓试错，确认后加仓
```

**情形B：锁仓预案（高潮延续时）**
```
触发：主线明确，龙头强势
操作：锁仓持有，不轻易下车
风控：浮盈回撤20%出
```

**情形C：收缩预案（退潮期）**
```
触发：连板率下降，炸板率高企
操作：空仓或极轻仓，不做新买入
风控：宁错过不做错
```

## 数据来源
- **thsdk**：实时行情、龙虎榜、资金流向
- **stockapi**：游资席位、板块资金
- **问财**：涨停原因、板块排名
- **Kronos**：5日预测辅助验证

## 风控原则
- 单票仓位 ≤ 10%（短线激进≤20%）
- 止损线：买入价 × 0.92（8%固定止损）
- 禁止高位追缩量涨停
- 退潮期不做新开仓
