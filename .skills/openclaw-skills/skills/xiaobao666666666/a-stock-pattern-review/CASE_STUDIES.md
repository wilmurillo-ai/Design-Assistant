# A股历史行情复盘案例

## 案例一：情绪冰点反转

### 背景
连续3日情绪低迷，涨停家数从80家降至20家，880005连续收阴。

### 盘面特征
```
涨停家数：18家（冰点）
连板率：8%（极低）
炸板率：45%（高危）
880005：-3.2%
主线：混沌期，无明显主线
```

### 关键结论
```
✓ 冰点次日情绪反转是最佳进攻时机
✓ 关注券商/大金融的先行异动
✓ 容量唯一性标的是首选
✓ 情绪拐点需要板块共振确认
```

## 案例二：主线退潮期操作

### 盘面特征
```
龙头：XX股票高位跌停
跟风股：批量跌停或大跌
连板率：5%
炸板率：52%（危险）
昨日涨停红盘率：28%
```

### 关键结论
```
✗ 退潮期禁止抄底
✗ 退潮期禁止博弈反抽
✗ 退潮期禁止追老主线跟风股
✓ 空仓是最好的风控
✓ 等待是散户最大的优势
```

## 案例三：容量唯一性实战

### 背景
新题材"固态电池"，需要识别容量标的。

### 容量分析
```
8亿标的：最多承载2亿资金
15亿标的：最多承载4亿资金
60亿标的：可以承载20亿+资金 ✓（唯一）
```

### 操作预案
```
标的：YY新能源（60亿，唯一容量标的）
买入时机：
 1. 开盘情绪恐慌时低吸
 2. 板上确认后打板
仓位：30%
止损：-5%
结果：+35%（3日）
```

### 关键结论
```
✓ 题材够大时，容量唯一性标的必须重视
✓ 大资金必须找到出口
✓ 锁仓需要成交回报确认大资金在场
```

## 案例四：新主线确认流程

### 判断三步法

**第一步：题材新鲜度**
- 新题材：过去1个月内没有炒作
- 政策催化：多部委联合出台

**第二步：板块结构**
- 身位龙头：3板+ ✓
- 弹性核心：小盘 ✓
- 容量中军：大盘 ✓

**第三步：持续性**
- 催化持续 ✓
- 资金认可 ✓
- 龙头继续表态 ✓
