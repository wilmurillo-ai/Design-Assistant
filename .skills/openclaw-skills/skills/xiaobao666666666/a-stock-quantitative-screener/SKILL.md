# A股量化交易股票筛选系统

## 触发条件
当用户发送以下指令时触发：
- "帮我筛选A股量化股票"
- "分析A股量化交易机会"
- "量化选股"
- "筛选可能拉升的股票"
- "识别主力资金介入"
- "分析量化信号"

## 工作流程

本技能采用**三角色辩论式分析**流程：

```
角色1（筛选） → 角色2（反驳） → 角色3（综判） → 最终推荐
```

## 阶段一：A股量化特点分析

1. **成交活跃度高** - 日均成交额约1-2万亿
2. **散户占比高** - 超额机会多
3. **政策影响显著** - 监管政策影响大
4. **波动性大** - 趋势跟踪策略有效
5. **小盘股溢价** - 流动性溢价明显
6. **板块轮动快** - 行业轮动策略适用
7. **技术指标有效** - 量价数据被广泛使用
8. **北向资金影响** - 外资定价权逐渐增强

## 阶段二：五维筛选标准

### 1. 资金流向维度（权重30%）
- 主力资金连续净流入 ≥ 3日
- 超大单净流入占比 > 5%
- 北向资金持股比例上升
- 融资融券余额变化

### 2. 量价技术维度（权重25%）
- 成交量较20日均量放大 ≥ 1.5倍
- 5日均线上穿20日均线（金叉）
- 股价站上BOLL中轨
- MACD零轴上方金叉

### 3. 趋势强度维度（权重20%）
- 区间涨幅 5%-25%
- 相对大盘超额收益 > 3%
- 股价创20日新高
- 多周期均线多头排列

### 4. 基本面维度（权重15%）
- 市值：主板50-500亿
- PE：0-50倍
- 近两年净利润正增长
- 近一年无重大违规

### 5. 筹码集中度（权重10%）
- 股东户数连续减少
- 户均持股数增加
- 筹码趋向集中

## 阶段三：三角色分析

### 角色1：量化研究员（筛选）
1. 分析A股量化特点
2. 结合当日最新市场行情
3. 应用筛选标准识别量化资金介入信号
4. 输出符合条件的前10只股票
5. 附具体代码、名称、入选理由、风险提示

### 角色2：量化研究员（反驳）
验证角色1结果的合理性：
1. **技术指标有效性** - 质疑滞后性、参数敏感性
2. **资金流向可靠性** - 质疑数据划分标准
3. **基本面矛盾** - 质疑流动性陷阱
4. **样本偏差** - 质疑幸存者偏差
5. **执行问题** - 质疑滑点、延迟、风控缺失

### 角色3：量化研究员（综判）
综合评估双方观点：
1. 评估辩论双方的论据强度
2. 调整优化筛选标准
3. 更新最终股票池
4. 提供优先级排序
5. 给出具体投资建议

## 阶段四：风险控制

- 单只股票仓位 ≤ 10%
- 设置8%止损线
- 持仓不超过20只
- 避免高位追涨

## 数据来源
- thsdk：实时行情、K线、资金流
- stockapi：龙虎榜、游资席位
- 问财：板块、资金排名

## 注意事项
- 量化策略存在失效风险
- 技术指标具有滞后性
- 资金流向数据可能存在误差
- 市场环境变化影响策略有效性
