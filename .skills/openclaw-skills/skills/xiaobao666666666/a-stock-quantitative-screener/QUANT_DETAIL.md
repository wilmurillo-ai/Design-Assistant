# A股量化筛选标准详细说明

## 一、资金流向维度（权重30%）

### 主力资金判定
- 单笔成交>100万为主力资金
- 主力净流入 = 超大单净流入 + 大单净流入

### 北向资金
- 持股数量/比例/市值变化

### 融资融券
- 融资买入=做多情绪，融券卖出=做空情绪

## 二、量价技术维度（权重25%）

### 均线金叉/死叉
- 5日均线>20日均线 → 买入信号
- 5日均线<20日均线 → 卖出信号

### MACD
- 参数：12/26/9日
- MACD柱由负转正+DIFF上穿DEA → 买入

### BOLL布林带
- 中轨：20日均线；上轨：中轨+2倍标准差
- 股价突破中轨+成交量放大 → 买入

### KDJ
- K、D、J值<20 → 超卖 → 买入机会
- K、D、J值>80 → 超买 → 卖出风险

## 三、趋势强度（权重20%）

### RSI
- RSI<30 → 超卖 → 买入机会
- RSI>70 → 超买 → 卖出风险

### 超额收益
- 超额收益 = 个股收益率 - 沪深300收益率

## 四、基本面（权重15%）

### 市值筛选
- 主板50-500亿，兼顾流动性和弹性

### 估值
- PE历史分位数<30%为低估，>70%为高估

## 五、筹码集中度（权重10%）

- 股东户数减少 → 筹码集中
- 户均持股数增加 → 主力吸筹

## 六、综合评分

```
总分 = 资金流向×0.30 + 量价技术×0.25 + 趋势强度×0.20 + 基本面×0.15 + 筹码×0.10
```

- ≥70分 → 重点关注
- 60-70分 → 观察标的
- <60分 → 暂不推荐

## 七、风控

- 止损：买入价×0.92（8%止损）
- 单票仓位≤10%
- 持仓上限20只
- 行业集中度≤30%
