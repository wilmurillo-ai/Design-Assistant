# 新能源职位搜索工具（碳索人才网）

## 功能概述
本技能用于搜索碳索人才网（新能源垂直行业招聘平台）上的职位信息。支持通过关键词、工作地点、所属行业进行精确筛选，并支持分页返回结果。助手会将用户的自然语言输入智能解析为结构化参数，然后调用碳索人才网的开放 API 获取数据，最后以易读的 Markdown 表格形式呈现给用户。

## 适用场景
- 用户想要查找某个地区的新能源行业职位，例如“北京的光伏工程师”。
- 用户希望了解特定行业（如风电、储能、新能源汽车）的招聘信息。
- 用户需要根据职位名称、公司或技能关键词进行搜索，例如“锂电池研发”。
- 用户希望浏览多个页面，通过“下一页”指令继续查看。

## 参数说明
以下为调用 API 时需要提供的参数。助手会从用户输入中提取相应的值，部分参数可省略。

| 参数名 | 类型 | 是否必需 | 默认值 | 说明 |
|--------|------|----------|--------|------|
| `keyWord` | string | 是 | - | 职位关键词，支持多个关键词以空格分隔。例如“工艺 总监”。 |
| `location` | string | 否 | 空 | 工作地点，支持城市名称（如“北京”、“上海”），也可输入省份（如“河北”）。 |
| `industry` | string | 否 | 空 | 所属行业，例如“光伏”、“风电”、“储能”、“新能源汽车”。 |
| `pageIndex` | int | 否 | 1 | 当前页码，从 1 开始。 |
| `pageSize` | int | 否 | 10 | 每页返回的职位数量，最大不超过 50。 |

## 如何从自然语言中提取参数
助手会通过语义理解，从用户的输入中提取出关键词、地点和行业。以下是常见的提取规则：

- **关键词**：通常为职位名称、技能或职责描述，例如“工艺总监”、“电池工程师”、“销售经理”。如果用户输入中同时包含多个部分，助手会将剩余部分作为关键词。
- **地点**：常见的地名（中国省份、城市）会被识别为 `location`。
- **行业**：新能源相关行业词汇（光伏、风电、储能、氢能、新能源汽车、锂电池等）会被识别为 `industry`。
