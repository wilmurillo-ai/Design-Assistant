# Podcast Script Templates

## Solo (单人主播)
1. 开场白 (15s) → 话题引入 (1min) → 主体分段 (5-15min) → 总结 (1min) → 结尾 (15s)

## Dialogue (双人对谈)
1. 双方问候 → A抛观点/B补充 → 交替深入 → 各自总结 → 结尾
