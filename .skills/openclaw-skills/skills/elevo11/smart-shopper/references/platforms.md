# Platform Reference

## Supported Platforms

### Amazon 🟠
- **URL**: amazon.com
- **Search**: `https://www.amazon.com/s?k={query}`
- **Price filter**: `&rh=p_36:{min*100}-{max*100}`
- **Shipping**: 1-2 days (Prime), 5-7 days (standard)
- **Return**: 30 days
- **Best for**: Electronics, books, household items, quality products

### Temu 🟡
- **URL**: temu.com
- **Search**: `https://www.temu.com/search_result.html?search_key={query}`
- **Shipping**: 7-15 days
- **Return**: 90 days free return
- **Best for**: Budget items, accessories, home goods, bulk buying

### SHEIN 🩷
- **URL**: shein.com
- **Search**: `https://www.shein.com/pdsearch/{query}/`
- **Shipping**: 7-12 days
- **Return**: 45 days
- **Best for**: Fashion, clothing, accessories, trendy items

## Budget Tiers

| Tier | Range | Best Platform |
|:-----|:------|:-------------|
| Low | $0-25 | Temu, SHEIN |
| Mid | $25-75 | All platforms |
| High | $75+ | Amazon |
