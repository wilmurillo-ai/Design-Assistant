# 课件制作指南

## 生成方式

**必须使用 `dragon-ppt-maker` 脚本生成 PPT**：

```bash
python ~/.workbuddy/skills/dragon-ppt-maker/ppt_maker.py --output 输出路径.pptx
```

该脚本支持：
- 渐变背景
- 图文混排
- 多种主题（tech/modern/corporate）
- 精美排版

**禁止**：使用 pptxgenjs 直接生成纯文字简陋 PPT。

## 课件结构建议

### 1. 封面页
- 课题名称
- 授课教师
- 所属学校/班级
- 配图：与课题相关的情境图

### 2. 目录页
- 出示本节课的学习内容
- 可以用流程图或时间轴展示

### 3. 情境导入页
- 呈现生活情境或数学问题
- 激发学生学习兴趣
- 引出本节课主题

### 4. 知识讲解页

#### 概念讲解
- 用不同颜色标注关键词
- 配合图示说明
- 适当留白，让学生思考

#### 公式推导
- 逐步展示推导过程
- 每一步加上解释
- 最后用框图总结

#### 例题解析
- 展示完整解题过程
- 标注关键步骤和易错点
- 提供多种解法对比

### 5. 练习巩固页
- 题目要循序渐进
- 基础题 → 提高题 → 拓展题
- 正确答案可以设置点击显示

### 6. 课堂总结页
- 用思维导图呈现知识结构（系统自动生成）
- 强调重点和难点
- 布置课后任务

**思维导图位置**：
- 复习课：第 2-3 页，作为知识回顾
- 新授课：倒数第 2-3 页，作为总结梳理
- 单元课：单独一页，展示完整知识体系

### 7. 板书设计页（可打印）
- 呈现完整板书内容
- 供学生整理笔记

## 配图要求

### 必须配图的位置
每页课件都应有适当的视觉元素，主要包括：

| 位置 | 配图类型 | 说明 |
|------|----------|------|
| 封面页 | 情境图 | 与课题相关的生活场景图 |
| 导入页 | 情境图 | 问题情境的插图 |
| 概念页 | 示意图 | 辅助理解数学概念的图示 |
| 例题页 | 例题图 | 包含题目条件和图示 |
| 练习页 | 题目图 | 题目可视化呈现 |
| 总结页 | 思维导图 | 知识结构图 |

### 配图原则
1. **相关性**：图片必须与教学内容直接相关
2. **清晰度**：图片要清晰、不模糊
3. **简洁性**：避免过于复杂的图片干扰重点
4. **美观性**：风格统一，色彩协调
5. **适量**：每页1-2张关键图即可，不要堆砌

### 配图来源
- 使用图像生成工具（如 DALL-E、Midjourney）生成教学情境图
- 使用在线数学图形工具（GeoGebra、几何画板）生成数学示意图
- **使用 diagram-generator 生成思维导图**（集成在本技能中）
- 使用流程图工具制作知识结构图
- 避免直接使用网络下载的未授权图片

### 思维导图生成

**本技能已集成 diagram-generator，可自动生成高质量思维导图**

支持格式：
- **Mermaid**：代码化格式，适合快速生成，可嵌入课件
- **DrawIO**：可编辑格式，功能强大，支持复杂布局
- **Excalidraw**：手绘风格，创意性强，适合吸引学生注意力

生成流程：
1. 系统自动识别适合使用思维导图的内容
2. 生成导图 JSON 规范
3. 调用 diagram-generator 生成导图文件
4. 将导图插入到课件相应位置
5. 保存原始文件到 `d:/WorkBuddy/MyTeacher/diagrams/` 目录

适用场景：
- 知识复习课：梳理单元知识点
- 概念关系复杂：展示层级或关联关系
- 逻辑推理步骤：定理推导、解题思路
- 对比总结：多个概念的异同点分析

详细使用说明请参考 `references/mindmap_guide.md`

### 配图标注
- 必要的图要有文字说明
- 几何图形要标注关键要素（角度、长度、名称）
- 图表要有标题

## 语文学科（古诗词）配图要求

### 古诗词课件必须配图的位置

| 位置 | 配图类型 | 具体内容 | 示例 |
|------|----------|----------|------|
| 封面页 | 情境图 | 与诗词主题相关的意境图 | 山川、日月、楼台 |
| 诗人介绍页 | 诗人画像 | 诗人的画像或照片 | 陆游画像 |
| 诗句讲解页 | 诗意配图 | 诗句描写的场景图 | 西湖山水图 |
| 情感分析页 | 背景图 | 时代背景相关图片 | 南宋临安城图 |
| 主题总结页 | 思维导图 | 情感线索图（系统自动生成） | 爱国情感脉络图 |

### 古诗词配图要点

1. **意境图**：选择能体现诗句意境的图片，如：
   - 山水田园诗 → 山水风景图
   - 边塞诗 → 大漠、草原图
   - 爱国诗 → 长城、江山图

2. **诗人画像**：每位诗人至少配一张画像

3. **时代背景图**：帮助学生理解诗歌创作背景

4. **图文对应**：每句重点诗句配一张相关图片

### 配图生成建议

- 使用 AI 图像生成工具创建诗意配图
- 关键词示例："中国古代山水画风格"、"南宋临安城"、"诗人画像古装"
- 优先使用中国风的图片素材

## 数学学科特殊要求

### 公式编辑
- 使用公式编辑器输入规范数学符号
- 公式要适当加大字号
- 重要公式用彩色标注

### 几何图形
- 图形要清晰、准确
- 动态演示几何变换过程
- 标注必要的角度、长度

### 动画设计
- 避免过于花哨的动画
- 重点内容可用动画强调
- 解题过程建议使用"擦除"或"出现"动画

### 色彩搭配
- 背景以浅色为主
- 文字与背景对比度要高
- 避免使用过多颜色（建议不超过3种）
