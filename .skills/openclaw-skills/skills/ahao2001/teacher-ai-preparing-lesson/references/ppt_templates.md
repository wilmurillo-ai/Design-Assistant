## 二、字体规范

### 2.1 中文字体
- **标题**：思源黑体 Bold / 微软雅黑 Bold
- **正文**：思源黑体 Regular / 微软雅黑
- **强调**：思源黑体 Medium / 加粗处理

### 2.2 英文字体
- **标题**：Segoe UI Bold / Arial Bold
- **正文**：Segoe UI / Calibri
- **数字**：Segoe UI / Arial (等宽更佳)

### 2.3 字号规范
| 元素 | 字号 | 用途 |
|------|------|------|
| 主标题 | 44-60pt | 封面、章节标题 |
| 副标题 | 28-36pt | 次级标题 |
| 小标题 | 24-28pt | 页面标题 |
| 正文 | 18-24pt | 内容文字 |
| 注释 | 14-16pt | 说明、页码 |

## 四、动画效果规范

### 4.1 推荐动画
| 动画类型 | 适用场景 | 持续时间 |
|----------|----------|----------|
| 淡入 (Fade) | 文字、图片出现 | 0.5s |
| 擦除 (Wipe) | 流程图、步骤展示 | 0.5s |
| 飞入 (Fly In) | 标题、重点内容 | 0.5s |
| 缩放 (Zoom) | 强调关键信息 | 0.3s |
| 路径动画 | 运动轨迹演示 | 1-2s |

### 4.2 动画原则
- 简洁为主，避免过度花哨
- 同一页面动画不超过3种
- 动画顺序符合阅读逻辑（左→右，上→下）
- 数学公式推导使用分步动画

## 五、页面尺寸与边距

### 5.1 标准尺寸
- **16:9 宽屏**：33.867cm × 19.05cm
- **4:3 标准**：25.4cm × 19.05cm（兼容旧设备）

### 5.2 安全边距
- 上下边距：2cm
- 左右边距：2.5cm
- 页眉页脚区域：1.5cm

## 六、Python-pptx 实现要点

### 6.1 母版设置
```python
# 设置幻灯片母版
slide_layouts = prs.slide_layouts

# 自定义布局
blank_layout = prs.slide_layouts[6]  # 空白布局作为基础

# 添加背景形状
background = slide.shapes.add_shape(
    MSO_SHAPE.RECTANGLE, 0, 0, prs.slide_width, prs.slide_height
)
background.fill.solid()
background.fill.fore_color.rgb = RGBColor(255, 255, 255)
```

## 七、模板使用说明

### 7.1 选择模板
1. 根据学科选择对应配色方案
2. 根据内容类型选择页面布局
3. 保持整课件风格统一

### 7.2 内容填充
1. 替换占位符文本
2. 插入学科相关图标
3. 调整图片和图表位置
4. 添加适当的动画效果

### 7.3 质量检查
- [ ] 配色是否符合学科风格
- [ ] 字体大小是否合适
- [ ] 页面布局是否整齐
- [ ] 动画是否流畅自然
- [ ] 内容是否完整准确
