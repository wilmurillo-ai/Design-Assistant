# 👓 glass2claw

**Ray-Ban glasses → voice command → WhatsApp → OpenClaw auto-routes your photo into the right database.**

Hands-free life logging for the agentic era.

## 🛠️ Installation

### 1. Ask OpenClaw (Recommended)
Tell OpenClaw: *"Install the glass2claw skill."* The agent will handle the installation and configuration automatically.

### 2. Manual Installation (CLI)
If you prefer the terminal, run:
```bash
clawhub install glass2claw
```

## 📸 How It Works

1. **You see something**: A wine bottle, a business card, or a rare tea.
2. **You say**: *"Hey Meta, take a picture and send this to myself on WhatsApp."*
3. **OpenClaw acts**: The photo lands in your WhatsApp session, is automatically classified, and routed to the correct specialist agent to be written into your database.

## Prerequisites

- **Meta Ray-Ban glasses** with Meta AI enabled.
- **WhatsApp** linked to your Meta View app.
- **OpenClaw** running with the WhatsApp channel active.
