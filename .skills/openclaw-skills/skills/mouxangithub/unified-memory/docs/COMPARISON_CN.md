## 3. 使用场景

### 3.1 何时使用统一记忆系统

- ✅ 长期项目（需要积累经验）

- ✅ 持续改进（需要自我进化）

- ✅ 一次性原型（不需要记忆）
- ✅ 快速演示（不需要优化）
- ✅ 个人实验（不需要协作）

## 6. 迁移指南

### 6.1 从 MetaGPT 迁移到统一记忆系统

```python
# MetaGPT
from metagpt import Team, Role

team = Team()
team.hire([ProductManager(), Architect(), Engineer()])
team.run("写一个博客系统")

# 统一记忆系统（相同结果 + 记忆）
from scripts.agent import Agent

agent = Agent()
agent.run("写一个博客系统")  # 自动存储到记忆
```

### 6.2 从 QMD 迁移到统一记忆系统

```python
# QMD
from qmd import QMDSearch

search = QMDSearch()
results = search.query("用户偏好")

# 统一记忆系统（相同 + 更多）
from scripts.memory import Memory

mem = Memory()
results = mem.search("用户偏好")  # 相同搜索
mem.store("新偏好")  # 加存储
agent = Agent()
agent.run("基于偏好构建")  # 加代码生成
```
