# 技术面评分标准

综合评分为 `-10 ~ +10` 区间，正数为多头，负数为空头。**0 为中性。**

每个指标打分后相加：

## 1. 均线排列（±3 分）

| 条件 | 分数 |
|---|---|
| MA5 > MA10 > MA20 > MA60（完美多头） | +3 |
| MA5 > MA10 > MA20（短中期多头，MA60 未参与） | +2 |
| MA5 > MA10，但 MA20 未参与 | +1 |
| 均线纠缠，无明显方向 | 0 |
| MA5 < MA10 < MA20 | -2 |
| MA5 < MA10 < MA20 < MA60（完美空头） | -3 |

## 2. RSI(14)（±2 分）

| 条件 | 分数 |
|---|---|
| RSI > 80（严重超买） | -2（风险警示） |
| 70 < RSI ≤ 80（超买） | -1 |
| 50 < RSI ≤ 70（健康上行） | +2 |
| 30 ≤ RSI ≤ 50（中性偏弱） | 0 |
| 20 ≤ RSI < 30（超卖反弹机会） | +1 |
| RSI < 20（严重超卖） | +2（反弹预期） |

注意：超买/超卖在短期是**反向信号**。

## 3. MACD（±2 分）

| 条件 | 分数 |
|---|---|
| 金叉 + MACD柱扩张 + 0 轴上方 | +2 |
| 金叉 + MACD柱扩张 + 0 轴下方 | +1 |
| 金叉但 MACD柱已经在收缩（顶背离警示） | 0 |
| 死叉 + MACD柱扩张 + 0 轴下方 | -2 |
| 死叉 + MACD柱扩张 + 0 轴上方 | -1 |

## 4. 布林带位置（±2 分）

| 条件 | 分数 |
|---|---|
| 价格突破上轨 > 3%（严重偏离） | -1（超买风险） |
| 价格在上轨附近（0~3%） | +1 |
| 价格在中轨上方 | +1 |
| 价格在中轨附近 | 0 |
| 价格在下轨附近 | -1 |
| 价格跌破下轨 > 3%（严重偏离） | +1（超卖反弹预期） |

## 5. KDJ（±1 分）

| 条件 | 分数 |
|---|---|
| J > 100（严重超买） | -1 |
| 80 ≤ J ≤ 100 | 0 |
| 20 ≤ J < 80 | 0 |
| 0 ≤ J < 20 | +1 |
| J < 0（严重超卖） | +1 |

## 6. 量能（±1 分）

| 条件 | 分数 |
|---|---|
| 近 5 日放量明显上涨 | +1 |
| 缩量上涨（上涨动能不足） | 0 |
| 放量下跌 | -1 |
| 缩量下跌（抛压减轻） | 0 |

---

## 评分映射

| 总分 | 结论 |
|---|---|
| +8 ~ +10 | 强烈看多，但注意超买 |
| +5 ~ +7 | 偏多 |
| +2 ~ +4 | 中性偏多 |
| -1 ~ +1 | 中性 |
| -4 ~ -2 | 中性偏空 |
| -7 ~ -5 | 偏空 |
| -10 ~ -8 | 强烈看空 |

**重要：** 技术面评分仅反映短期趋势，不反映基本面。即使评分高，基本面差仍要警示风险。
