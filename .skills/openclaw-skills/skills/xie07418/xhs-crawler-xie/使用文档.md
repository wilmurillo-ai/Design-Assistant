# 小红书爬虫飞书 Skill 使用文档

## 一、功能概述

本程序实现**飞书输入关键词 → 自动爬取小红书笔记 → 推送结果到飞书群**的完整流程。

### 核心功能
- ✅ 飞书群输入关键词，自动搜索小红书笔记
- ✅ Cookie 自动管理（检测有效期，过期自动二维码登录）
- ✅ 使用浏览器模拟搜索，绕过 API 限制
- ✅ 搜索结果以卡片形式推送到飞书群

---

## 二、文件结构

```
openclaw封装skill/
├── __init__.py                    # 包初始化
├── config.py                      # 配置文件（凭证、路径、参数）
├── feishu_app_bot.py             # 飞书应用机器人（App ID + App Secret）
├── cookie_manager.py             # Cookie 检测与提取
├── auto_login_with_qrcode.py     # 自动二维码登录
├── xhs_search_with_browser.py    # 浏览器搜索小红书
└── example_openclaw_skill.py     # Skill 入口（主程序）
```

### 各文件作用

| 文件 | 作用 | 是否必须 |
|------|------|---------|
| `config.py` | 配置飞书凭证、Cookie 路径、请求间隔等 | ✅ 必须 |
| `feishu_app_bot.py` | 飞书消息发送、图片上传 | ✅ 必须 |
| `cookie_manager.py` | 检测 Cookie 是否有效，从浏览器提取 Cookie | ✅ 必须 |
| `auto_login_with_qrcode.py` | Cookie 过期时自动截图二维码发送到飞书 | ✅ 必须 |
| `xhs_search_with_browser.py` | 使用浏览器模拟搜索小红书 | ✅ 必须 |
| `example_openclaw_skill.py` | 整合所有功能，提供统一接口 | ✅ 必须 |

## 四、配置说明

### config.py 关键配置

```python
# ==================== 飞书应用配置 ====================
FEISHU_APP_ID = "cli_a924d921ce7a9cbd"           # 飞书 App ID
FEISHU_APP_SECRET = "5QG92Lp8kvhAkgpPJTd57fIxshnCebEt"  # 飞书 App Secret

# ==================== 爬虫配置 ====================
DEFAULT_MAX_NOTES = 5           # 默认爬取笔记数量
REQUEST_DELAY_MIN = 2.0         # 请求最小间隔（秒）
REQUEST_DELAY_MAX = 5.0         # 请求最大间隔（秒）

# ==================== 防风控配置 ====================
ANTI_DETECT = {
    "max_checks_per_hour": 10,      # 每小时最多检查 Cookie 次数
    "min_check_interval": 300,       # 两次检查间隔至少 5 分钟
    "cooldown_after_461": 3600,      # 遇到 461 错误后冷却 1 小时
}
```


## 五、使用方式

### 方式一：命令行测试

```bash
# 进入项目目录
cd "D:\pycharm\projects\小红书关键词爬取\openclaw封装skill"

# 运行测试（搜索关键词"新燕宝2025"）
python example_openclaw_skill.py "新燕宝2025"
```

### 方式二：OpenClaw 集成

在 OpenClaw 中配置 Skill，调用 `example_openclaw_skill.py`：

```python
from example_openclaw_skill import xhs_search_skill
import asyncio

# 调用 Skill
result = asyncio.run(xhs_search_skill("新燕宝2025"))

# 处理结果
if result["success"]:
    print(f"获取到 {len(result['data'])} 条笔记")
else:
    print(f"失败: {result['message']}")
```

### 方式三：作为模块导入

```python
from openclaw封装skill import FeishuAppBot
from xhs_search_with_browser import search_with_browser
import asyncio

# 搜索笔记
results = asyncio.run(search_with_browser("新燕宝2025", max_notes=5))

# 发送到飞书
bot = FeishuAppBot()
bot.send_notes_summary("oc_29fbba0871ab7371a4c1a1ebe0350dac", "新燕宝2025", results)
```

## 六、工作流程

```
用户输入关键词
    ↓
检查 Cookie 有效性
    ↓
├── Cookie 有效 ──→ 继续搜索
└── Cookie 失效 ──→ 自动二维码登录
        ↓
    截图登录页面
        ↓
    发送到飞书群
        ↓
    等待用户扫码（最长 5 分钟）
        ↓
    保存新 Cookie
        ↓
使用浏览器搜索小红书
    ↓
提取笔记信息（标题、链接、作者、点赞数）
    ↓
发送结果到飞书群
```

## 七、常见问题

### Q1: IP 被风控（HTTP 461 错误）

**现象：**
```
IP存在风险，请切换可靠网络环境后重试
```

**解决：**
1. 停止运行，等待 2-4 小时自动解封
2. 或切换到手机热点/其他网络

### Q2: 搜索不到笔记

**原因：**
- 小红书搜索是精确匹配，某些关键词可能没有结果
- 尝试使用更通用的关键词，如"新燕宝2025"代替"新燕宝"

### Q3: Cookie 频繁失效

**解决：**
- 这是正常现象，小红书 Cookie 通常有效期为数天到数周
- 程序会自动触发二维码登录，无需手动干预

### Q4: 飞书消息发送失败

**检查：**
1. App ID 和 App Secret 是否正确
2. 机器人是否已添加到目标群聊
3. 群聊 ID 是否正确

## 八、注意事项

1. **请求频率**：已设置防风控机制，请求间隔 2-5 秒，每小时最多检查 10 次 Cookie
2. **网络环境**：避免频繁切换 IP，可能触发风控
3. **关键词选择**：某些小众关键词可能无搜索结果，建议使用相关热门词
4. **日志查看**：运行日志保存在 `logs/xhs_crawler.log`
