# Auto Tech Research Skill v2.1

全自动化技术深度调研 Skill，支持智能平台相关性评估和动态数量分配。

## ✨ 核心特性

🎯 **智能相关性评估** - 根据技术主题自动评估各平台相关性  
📊 **动态数量分配** - 高相关平台多获取，低相关平台少获取或跳过  
📝 **技术概览生成** - 报告开头自动生成 200-1000 字技术介绍  
🏷️ **完整来源标注** - 每条内容都标注来源平台  
📺 **视频直接跳转** - 视频类资源提供可点击的播放链接  
📈 **详细平台统计** - 每个平台的获取数量、相关性、质量评分完整统计

## 🎯 智能相关性评估

### 评估逻辑

Skill 会根据技术主题自动评估每个平台的相关性：

**高相关** (🔴) → 获取 1.3x 基础数量
- 主题与平台专业领域高度匹配
- 该平台有大量相关内容

**中相关** (🟡) → 获取 1.0x 基础数量
- 主题与平台有一定关联
- 有适量相关内容

**低相关** (🔵) → 获取 0.4x 基础数量（最少2条）
- 主题与平台关联度低
- 仅获取代表性内容

## 🔧 动态数量分配算法

```python
def calculate_fetch_count(platform, relevance_score):
    base_count = BASE_COUNTS[platform]  # 基础数量
    
    if relevance_score >= 0.7:      # 高相关
        multiplier = 1.3
    elif relevance_score >= 0.4:    # 中相关
        multiplier = 1.0
    else:                           # 低相关
        multiplier = 0.4
    
    return int(base_count * multiplier)
```

## 💡 设计亮点

1. **智能相关性评估** - 不是固定数量，而是根据主题动态调整
2. **技术概览生成** - 报告开头先给核心介绍，建立整体认知
3. **完整来源标注** - 每条内容都清楚标注来自哪个平台
4. **视频直接播放** - 视频链接可直接点击跳转
5. **详细统计分析** - 每个平台的相关性、数量、质量都有统计
6. **灵活可扩展** - 易于添加新平台和调整相关性规则
