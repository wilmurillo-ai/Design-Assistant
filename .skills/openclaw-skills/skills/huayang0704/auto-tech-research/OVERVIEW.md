## 核心模块详解

### 1. 关键词扩展引擎 (KeywordExpander)

**功能**：
- 将用户输入的技术主题扩展为搜索关键词矩阵
- 支持中英双语扩展
- 生成同义词、相关概念、应用场景等变体

**扩展策略**：
```python
{
    "zh": {
        "core": ["原始主题"],
        "tutorial": ["主题+教程", "主题+入门"],
        "practice": ["主题+实战", "主题+案例"],
        "advanced": ["主题+原理", "主题+源码"]
    },
    "en": {
        "core": ["topic"],
        "tutorial": ["topic tutorial", "topic guide"],
        "practice": ["topic example", "topic project"],
        "advanced": ["topic deep dive", "topic architecture"]
    }
}
```

### 2. 多平台搜索引擎 (MultiPlatformSearcher)

**平台分类**：

| 类别 | 平台 | 内容类型 | 可信度 |
|------|------|----------|--------|
| 学术 | arXiv, Google Scholar | 论文 | ★★★★★ |
| 国内 | 知乎, CSDN, 公众号, B站 | 实践 | ★★★★☆ |
| 国际 | GitHub, HN, Reddit, YouTube | 工程 | ★★★★☆ |
| 播客 | 小宇宙, Podcast | 讨论 | ★★★☆☆ |

**搜索策略**：
- 并行执行提高速度
- 针对不同平台优化查询
- 智能处理反爬机制

### 3. 内容分级系统 (ContentClassifier)

**分级算法**：

```python
def classify_level(content):
    title = content.title.lower()
    
    # Level 1: 入门关键词
    if any(kw in title for kw in ["入门", "简介", "什么是", "explain"]):
        return 1
    
    # Level 2: 实践关键词
    if any(kw in title for kw in ["教程", "实战", "案例", "tutorial"]):
        return 2
    
    # Level 4: 前沿关键词
    if any(kw in title for kw in ["最新", "前沿", "趋势", "2026"]):
        return 4
    
    # 默认 Level 3
    return 3
```

**质量评分**：
- 平台可信度权重
- 作者权威性
- 社区反馈（赞/评/转）
- 时效性

## 关键技术点

### 1. 内容去重

**策略**：
- URL精确匹配
- 标题相似度检测（编辑距离）
- 摘要语义相似度

```python
from difflib import SequenceMatcher

def similarity(a, b):
    return SequenceMatcher(None, a, b).ratio()

def deduplicate(items, threshold=0.8):
    unique = []
    for item in items:
        if not any(similarity(item.title, u.title) > threshold for u in unique):
            unique.append(item)
    return unique
```

### 2. 智能摘要

**方法**：
- 提取关键段落
- LLM生成摘要（可选）
- 保留核心观点

### 3. 链接管理
