# 文书类型判断规则

## 传票 / 出庭通知 → 建日历

**关键词：**
- 开庭审理
- 开庭时间
- 传唤
- 出庭通知

**判断逻辑：**
```python
is_summons = bool(re.search(r'开\s*庭|传\s*唤|出\s*庭\s*通\s*知', text))
```

**日历字段：**
- summary: ⚖️ 开庭：{案由}
- start date: {开庭时间}
- end date: {开庭时间+2小时}
- location: {地点}

---

## 撤诉裁定 → 仅汇报，不建日历

**关键词：**
- 撤诉
- 裁定准予

**判断逻辑：**
```python
is_dismissal = bool(re.search(r'撤诉|裁定准予', text))
```

---

## 应诉通知书 → 建日历

**关键词：**
- 应诉
- 答辩

---

## 其他文书 → 仅汇报

不触发日历创建，仅提取信息并汇报给用户。
