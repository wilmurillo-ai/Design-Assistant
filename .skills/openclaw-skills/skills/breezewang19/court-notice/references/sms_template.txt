# 法院短信标准模板

## 推荐格式

```
【{法院名称}】{法院名称}向您发送了{案号}案件及相关文书，请及时签收。点击链接查阅：{链接}
```

## 示例

```
【上海市第一中级人民法院】上海市第一中级人民法院向您发送了（2026）沪01民初1234号案件及相关文书，请及时签收。点击链接查阅：http://10.10.159.145:8765/test_court_notice.pdf
```

## 字段说明

- `{法院名称}` - 如：上海市第一中级人民法院
- `{案号}` - 如：（2026）沪01民初1234号
- `{链接}` - PDF下载链接

## 变体（简洁版）

```
【{法院名称}】
案号：{案号}
本院已向您送达案件文书，请及时签收。点击查阅：{链接}
```
