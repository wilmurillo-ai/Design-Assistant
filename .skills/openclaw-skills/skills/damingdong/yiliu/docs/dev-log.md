### 架构

```
yiliu-skill/
├── src/
│   ├── index.ts       # 入口（支持异步）
│   ├── commands/
│   │   └── note.ts    # 笔记命令（语义搜索）
│   ├── storage/
│   │   ├── db.ts      # SQLite + AI 集成
│   │   └── vector.ts  # 向量存储
│   ├── ai/
│   │   └── index.ts   # AI 能力
│   └── types/
│       └── index.ts   # 类型定义
└── data/
    ├── yiliu.db       # SQLite
    └── vectors.json   # 向量索引
```

### 技术决策

1. **向量存储**：采用内存向量索引 + JSON 持久化，避免 ChromaDB 的 Python 依赖问题
2. **AI API**：直接调用 OpenAI API，支持自定义 BASE_URL（兼容国内代理）
3. **降级策略**：未配置 API Key 时回退到关键词搜索
