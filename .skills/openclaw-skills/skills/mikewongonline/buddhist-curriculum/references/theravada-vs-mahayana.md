# 🌳 部派佛教 vs 大乘佛教 (Theravada vs Mahayana)

这是佛教史上两大主流的派别区分，主要差异在于**救度众生的方法论**和**终极追求的层面**。

## 🔹 部派佛教 (Theravada Buddhism)
*   **核心关注点：** 严格的戒律（律藏）、个人修行、个人解脱。
*   **核心概念：** 重视阿罗汉果的觉悟——即为自己彻底解脱，摆脱轮回（出家）。
*   **修行重点：** **戒律**（守戒是第一位的）、**禅定**（通过禅定达到心性的寂静）。
*   **代表意义：** 强调回归原始的、最纯粹的戒律和修行方法。

## 🔷 大乘佛教 (Mahayana Buddhism)
*   **核心关注点：** 广大、普世的慈悲心和彻底的解脱。
*   **核心概念：** 强调“菩萨道”——即不能只为自己解脱，必须发愿为一切众生作度。
*   **修行重点：** **智慧 (Prajna)** 和 **慈悲 (Karuna)** 的结合。菩萨需要具备修行的“圆满性”。
*   **代表意义：** 展现了极大的包容性和广大的众生利益范围。

**⚖️ 总结：**
*   **部派**：更强调“修完自己”，个人解脱是重点。
*   **大乘**：更强调“度尽所有”，集体的觉悟和利他精神是核心。
*   **关系：** 大乘思想吸收了部派的戒律和禅定，但将目标从“我”扩展到了“众生万有”。
