# ✨ 核心概念辨析 (The Core Concepts)

在佛学中，以下概念是相互关联，但含义不同的哲学基石：

## 1. 缘起 (Dependent Origination)
*   **定义：** 世界上的一切现象（万物）都是由各种“因”和“缘”共同作用而生起的，它没有独立、永恒的自性。
*   **关键词：** 因果、相互依存、没有自性。
*   **理解方式：** 这是一个描述“过程”的法则，告诉我们事物不是凭空产生的，而是有生有灭、有因有果。

## 2. 空性 (Emptiness)
*   **定义：** “空”并非指“什么都没有”，而是指事物**缺乏独立、永恒、不变的自性**。它是一种本体论的观点。
*   **关键词：** 无自性、缺乏实体、空是缘起的基础。
*   **理解方式：** 它回答的是“本质是什么”的问题。万法皆空，但“空”本身不是实体。

## 3. 无我 (No-Self)
*   **定义：** 否定“我”这个概念的实体性。指我们所认知的“自我”——由五蕴（色、受、想、行、识）这个过程构成的，但这个“我”不是一个独立存在的灵魂。
*   **关键词：** 五蕴、轮回、主体性幻象。
*   **理解方式：** 解决“我是谁”的问题。它指的不是没有自我，而是没有一个固定不变的“我是我”的实体。

**👉【关系链】**：**缘起**是万物生灭的方式；**空性**指出万物缺乏自性；**无我**则应用空性的思想，否定了那个“自性”存在的“我”。
