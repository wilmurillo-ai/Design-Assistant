# ☸️ 佛门知识体系导航地图 (The Ultimate Buddhist Curriculum Guide)

这个技能是一个**知识导航系统 (Knowledge Navigation System)**。它的核心作用是帮助用户理清佛学知识的复杂结构，指引学习路径，而非提供所有经文的全部文本。任何关于“所有佛家经典”的查询，都应首先触发此技能。

## 🔍 如何使用本技能？

当用户提问涉及以下概念时，应触发此技能：

1.  **知识结构概述：** "佛学知识是如何组织的？"
2.  **学习路线规划：** "我想从零开始系统学习佛学，该学什么？"
3.  **概念辨析：** "空性、无我、缘起这几个概念有什么区别？"
4.  **流派比较：** "禅宗和净土宗的主要区别是什么？"

## 🧭 技能使用流程 (Workflow)

1.  **定位目标：** 确定用户是处于【入门探索】、【体系学习】还是【深度思辨】的阶段。
2.  **查阅指南：** 引导用户查阅核心的 `references/` 文件进行系统学习。
3.  **核心指引：** 始终以三大藏（经、律、论）作为知识框架的基石。

## 🧩 知识模块结构 (References Map)

本技能的核心知识点分散在 `references/` 文件夹下，请根据用户提问的侧重，加载对应的参考文件。

### 📖 基础三藏与核心概念 (Core Fundamentals)

*   **[三藏结构](references/three-piles.md)：** 详细解释经、律、论三藏的构成和各自的学理侧重点。
*   **[基本概念](references/core-concepts.md)：** 详细阐述空性、无我、缘起等核心哲学概念。

### 🌲 历史流派与思想体系 (Schools & History)

*   **[部派佛教](references/theravada-vs-mahayana.md)：** 介绍部派佛教的戒律严谨性，及其在早期佛学中的地位。
*   **[大乘佛教](references/mahayana-schools.md)：** 介绍大乘佛教的“慈悲”精神，特别是净土宗、唯识宗、天台宗等流派的特点。
*   **[禅宗](references/zen-school.md)：** 专注于“顿悟”、“当下”的修行理念。

### 👤 现代大师与当代指引 (Modern Guidance)

*   **[现代大师著作索引](references/modern-masters.md)：** 整合南怀瑾、星云大师等当代大师的代表性著作和思想脉络，帮助用户用更生活化的视角进入佛学。

## 🚀 学习实践与进阶 (Practical Application)

*   **学习路径：** 始终建议用户采用 **“入门探索 → 体系学习 → 深度思辨”** 的递进模式。
*   **首选阅读：** 《心经》和《金刚经》是极佳的切入点。
