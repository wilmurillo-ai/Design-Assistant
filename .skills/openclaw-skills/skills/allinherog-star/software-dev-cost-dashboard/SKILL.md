# software-dev-cost-dashboard

## 概述

软件成本评估看板入口，快速生成项目报价和工时估算。

**直接访问**: [https://soft.ai-skills.ai](https://soft.ai-skills.ai)

## 使用场景

- 项目立项前的成本预算
- 客户项目的工时与报价评估
- 研发团队的资源规划
- 多方案比选（不同规模/不同技术栈的成本对比）
