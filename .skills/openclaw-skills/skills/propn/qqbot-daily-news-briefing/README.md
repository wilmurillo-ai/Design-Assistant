# Daily News Briefing Skill for OpenClaw

[![Version](https://img.shields.io/badge/version-1.1-blue.svg)](https://github.com/openclaw) [![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

**Automated daily news briefing generation with AI commentary. Features dual-search architecture: Baidu API (preferred) + DuckDuckGo fallback (no API key required).**

## 🚀 Quick Start

### 1. Install

```bash
# Via ClawHub
clawhub install daily-news-briefing

# Or download and place in ~/.openclaw/skills/
```

### 2. Configure

**Quick start (uses DuckDuckGo automatically):**
```bash
export NEWS_TARGET_USER="your-qq-user-id"
export NEWS_CHANNEL="qqbot"
```

**With Baidu API (better results):**
```bash
export BAIDU_API_KEY="bce-v3/ALTAK-your-api-key"
export NEWS_TARGET_USER="your-qq-user-id"
export NEWS_CHANNEL="qqbot"
```

### 3. Test

```bash
cd ~/.openclaw/skills/daily-news-briefing/scripts

# Generate news
python3 generate-briefing.py

# Deliver to messaging channel
bash deliver-briefing.sh
```

### 4. Automate (Optional)

```bash
crontab -e

# Add these lines for daily 9:00 AM delivery
0 9 * * * source /etc/profile && cd ~/.openclaw/skills/daily-news-briefing/scripts && python3 generate-briefing.py >> /var/log/daily-news.log 2>&1
1 9 * * * source /etc/profile && bash deliver-briefing.sh >> /var/log/news-delivery.log 2>&1
```

## 📊 Features

- **4 News Categories**: China Tech, International Tech, China Finance, International Finance
- **AI Commentary**: Professional Jarvis commentary for each article with financial IT architect perspective
- **Dual Search Engine**: 
  - Baidu API (structured data + content previews)
  - DuckDuckGo fallback (no API key needed, always available)
- **Automated Delivery**: Cron job support with 9:00 AM default schedule
- **Multi-Channel Support**: QQ Bot, Telegram, Discord via OpenClaw CLI
- **Graceful Degradation**: Works even if both search methods fail

## 🔧 Customization

### Change Search Queries

Edit `scripts/generate-briefing.py` around line 180:

```python
china_tech = smart_search('科技新闻 人工智能 芯片 AI 华为', count=3)
intl_tech = smart_search('NVIDIA Broadcom Apple Microsoft AI', count=3)
china_finance = smart_search('A 股 上证指数 港股 财经', count=3)
intl_finance = smart_search('美股 纳斯达克 道琼斯 比特币', count=3)
```

### Modify Delivery Time

Edit crontab entries:
- `0 9 * * *` → 9:00 AM
- `30 7 * * *` → 7:30 AM
- `0 18 * * *` → 6:00 PM

### Change Output Location

Edit WORKSPACE variable in scripts:
```python
WORKSPACE = '/root/.openclaw/workspace'
NEWS_FILE = f"{WORKSPACE}/daily-news-2026{datetime.date.today().strftime('%m%d')}.md"
```

## 🐛 Troubleshooting

**No results:**
```bash
# Check which search method is being used
grep "Search method used" /var/log/daily-news.log | tail -1

# Test DuckDuckGo directly (should work without API key)
unset BAIDU_API_KEY && python3 generate-briefing.py
```

**Delivery fails:**
```bash
# Check delivery log
tail -30 /var/log/news-delivery.log

# Test manual delivery
bash deliver-briefing.sh
```

See `SKILL.md` for detailed troubleshooting guide.
