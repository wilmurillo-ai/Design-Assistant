# 论文摘要模板

## 完整结构

```markdown
## <论文标题>

### 基本信息
- **作者**: 作者列表
- **机构**: 机构列表
- **arXiv ID**: XXXX.XXXXX
- **链接**: [arXiv](https://arxiv.org/abs/XXXX.XXXXX) | [PDF](./core-papers/xxx.pdf)
- **日期**: YYYY年MM月
- **来源**: arXiv / ICLR 2024 / NeurIPS 2024 / ICML 2024

### 摘要
> 原文摘要

### 核心贡献
1. 贡献点1
2. 贡献点2
3. 贡献点3

### 方法
- 方法1
- 方法2
- 方法3

### 实验
- 数据集
- 基线模型
- 主要结果

### 局限性
- 局限点1
- 局限点2

### 相关工作
- 相关论文1
- 相关论文2

### 引用
- 被引次数
- 引用论文列表

### 个人笔记
> 阅读时的思考和问题
```

## 快速模板

```markdown
### <标题>

- **作者**: ...
- **arXiv**: [ID](url)
- **核心贡献**: 
  1. ...
- **方法**: ...
- **链接**: [PDF](./xxx.pdf)
```

## 关键词提取模板

```markdown
### 关键词
- 关键词1 (出现次数: N)
- 关键词2 (出现次数: N)
- 关键词3 (出现次数: N)
```
