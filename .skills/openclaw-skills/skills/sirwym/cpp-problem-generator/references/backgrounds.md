# 随机题面背景库

1. **赛博朋克黑客入侵**：背景设定在 2077 年的霓虹都市，主角是顶尖黑客，需要突破巨型企业（如荒坂或微光）的冰墙防火墙、绕过追踪节点或窃取加密数据。
2. **中世纪魔法学院**：背景设定在霍格沃茨般的魔法世界，主角是见习魔法师，需要调配魔药（容量/体积限制）、布置魔法阵（几何/图论）、或者对抗魔物。
3. **星际联邦物流**：背景设定在银河系，主角是星际舰长或物流调度员，需要规划跨星系跃迁航道、分配有限的跃迁引擎能量、或在陨石带中寻找最优路径。
4. **末日废土生存**：背景设定在核战后的废土荒野，主角是拾荒者，需要在辐射区收集物资、躲避变异生物、或修复破损的地下避难所能源网络。
5. **修仙修真世界**：背景设定在东方玄幻世界，主角是修仙门派弟子，需要突破境界（状态转移）、炼制法宝（材料合成）、或探索秘境迷宫。
6. **现代金融商战**：背景设定在华尔街，主角是量化交易员或财团操盘手，需要进行高频交易预测、规避市场风险、或进行资金流转调度。
7. **克苏鲁神话调查**：背景设定在 1920 年代的阿卡姆镇，主角是调查员，需要解读古老文献寻找理智值（SAN）的最优消耗路径，或封印邪神祭坛。
