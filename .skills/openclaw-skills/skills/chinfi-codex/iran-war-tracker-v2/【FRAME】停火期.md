## 一、核心分析逻辑

此阶段日报回答三个问题：

1. **停火今天还撑着吗？**（合规层）
2. **谁在向打破僵局的方向移动？**（压力层）——取代原"谈判层"
3. **4月21日到期时会发生什么？**（推演层）

三层逻辑要求：推演必须能从压力层的方向判断中找到依据；压力层判断必须基于可核实的当日行为，而非声明。

> **阶段定义：停火空转期** 停火协议形式存续，政治驱动力已消失。各方在窗口到期前进行战略试探、国内政治操作和军事预置。这一阶段的危险不在于"宣布破裂"，而在于**蠕变式升级**——没有人正式打破停火，但每天都向临界点靠近一步。

### 2.3 到期倒计时压力模型（新增核心模块）

两周停火窗口于**4月21日**到期。倒计时不是线性的——压力在最后72小时非线性跃升。

|剩余时间|阶段特征|重点观察|
|---|---|---|
|9-7天（当前）|战略试探期：各方测试对方底线，预置立场|扫雷行动伊朗反应；以色列对伊表态|
|6-4天|施压期：各方开始为到期后行动铺垫舆论|特朗普/内塔尼亚胡是否发出到期威胁|
|3-1天|临界期：非线性风险急升|是否出现续期提案；军事预置动作|
|到期日|决策节点|三条路：续期、崩溃、僵尸化延续|

**每日日报标题栏必须标注：停火第N天 / 剩余X天 / 当前阶段**

### 2.5 续谈信号监测（压缩自原谈判模块）

谈判桌已空，不再每日填写分歧热力表。改为单行信号监测：

**续谈需满足的最低条件（缺一不可）：**

- 伊朗就核问题立场出现可验证的软化表态
- 美方撤回或修改"最终方案"的部分条款
- 第三方（巴基斯坦/卡塔尔）提出具体续谈框架

**每日只问一个问题：今天上述三个条件中，有任何一个发生了吗？**

- 全无 → 情景B/C概率上升
- 出现任意一个 → 在压力层标注，情景A概率调整

## 四、边际变化判断标准

停火空转期的边际变化逻辑：

- **合规层：** 任何新的直接军事行动 = 即时升级信号
- **压力层：** 行为体向"打破僵局"方向移动的可验证行动（不含声明）= 升级信号；向"续谈"方向移动 = 缓和信号
- **扫雷行动：** 每日进展 = 新增摩擦点，伊朗反应决定其性质
- **以色列独立行动：** 任何未经美方背书的对伊行动 = 最高优先级升级信号
- **哈梅内伊直接发声：** 任何与政府口径不一致的最高领袖声明 = 伊朗内部分歧信号

**不记录：** 例行强硬声明、重复性表态、与前日无实质差异的外交辞令。

## 五、场景推演规则

**4月21日到期时的三个情景（每日滚动更新概率）：**

**情景A：停火续期或启动第二轮谈判（20%）** 触发条件：伊朗就核立场出现可验证软化 + 美方修改部分条款 + 以色列被约束住 市场含义：Brent较战时峰值回落$8-12；波斯湾战险费率降至0.3-0.5%

**情景B：停火到期崩溃，恢复交战（45%）** 触发路径①：以色列自主打击伊朗，美国事后背书 触发路径②：伊朗对扫雷行动采取对抗措施，美军回应 触发路径③：代理人行动失控，触发连锁反应 市场含义：Brent反弹至战时峰值区间；霍尔木兹再封锁：LNG现货溢价扩大至$15-20/MMBtu

**情景C：僵尸化延续（35%）** 停火无限期延续，但无任何推进机制，各方均不愿承担破裂责任，问题完全悬置 市场含义：Brent区间震荡，高不确定性溢价维持$6-10，航运保险成本居高不下

**概率更新规则：**

- 单日出现2个以上升级信号 → B概率+5-10%
- 出现任意续谈条件信号 → A概率+5-10%
- 概率变动±15%以上须在边际变化中说明理由
