## 一、Agent 任务

### 目标

agent 每次播报只需回答 4 个问题：

1. 当前主情景是什么。
2. 当前最关键的 3 个变量是什么。
3. 当前更像是短期通胀冲击、滞涨，还是衰退风险上升。
4. 未来 1-2 周最值得盯的触发点是什么。

### 输出顺序

优先按以下结构输出：

1. `当前主情景`
2. `核心变量`
3. `价格判断`
4. `经济含义`
5. `后续触发点`

### 硬规则

- 不要把局势简化成“霍尔木兹是否开放”。
- 不要把“恢复通航”直接等同于“供给恢复正常”。
- 必须区分 `事实`、`市场定价`、`推演`。
- 默认用中文输出。

## 二、核心判断

### 1. 当前主矛盾

当前局势的核心不是单一的“总供给不足”，而是：

`海峡流量 + 出口节点 + 运输交付 + 成品油/天然气紧张 + 谈判路径`

### 2. 默认总逻辑

短期定价核心仍然是：

`有效供给 = 海峡通航能力 × 出口节点可用性 × 运输交付能力`

这意味着：

- 库存只能缓冲，不能替代流量恢复。
- 即使海峡部分恢复，若卡尔格岛、运力、保险、港口、船东行为未恢复，实际供给仍会偏紧。
- 成品油和天然气的局部短缺，可能先于“总原油缺口”成为市场交易主线。

### 3. 一句话模板

`当前局势的主矛盾不是单纯的海峡关闭，而是海峡通航、出口节点、运输交付和政策路径同时不确定，导致有效供给恢复慢于名义通航恢复。`

---

## 三、核心变量

### 1. 海峡流量

- 看点：霍尔木兹海峡实际通航量、油轮进出、护航是否落地。
- 默认阈值：`<2mb/d` 且持续数周，说明进入持续流量危机。

### 2. 出口节点

- 看点：卡尔格岛状态、装运设施、伊朗是否主动关阀。
- 关键点：卡尔格岛关系到伊朗绝大部分原油出口。

### 3. 运输与交付

- 看点：VLCC 运价、LNG 船运价、白船可用性、保险、绕航、交付周期。
- 关键点：本轮危机更接近“运输与交付危机”，不只是产量问题。

### 4. 替代路线与炼化错配

- 看点：延布、富查伊拉、红海替代路径是否可用。
- 看点：重油/轻油错配、炼厂适配性、关键炼厂开工率。
- 关键点：即使原油还有，炼化和品种错配也会造成成品油紧张。

### 5. 成品油与天然气紧张度

- 看点：柴油、航空煤油、裂解价差、LNG 运输与区域气价价差。
- 关键点：柴油、航空煤油、天然气往往比“总原油”更早进入紧张状态。

### 6. 区域库存与补给节奏

- 看点：东亚库存消耗、日韩补给、东南亚局部短缺。
- 关键点：危机往往在库存下降但补给无法续接时显著升级。

### 7. 美方行动路径

- 分支：
  - `仅空袭施压`
  - `空袭后护航`
  - `封锁/占领卡尔格岛`
  - `更大规模地面介入`
- 关键点：旧框架里的“护航/不护航”已不够用。

### 8. 谈判与政策路径

- 看点：中介方、停火条件、开放海峡是否绑定进谈判。
- 看点：美国通胀、增长、衰退概率、Fed 路径。

## 四、情景矩阵

### 情景 1：短期中断，快速降温

- 时间：约 `1个月` 或 `6周以内`
- 价格：Brent 月度峰值约 `$100`，年均约 `$80`
- 含义：通胀上升但总体可控，增长承压但未必衰退

### 情景 2：中等时长中断，市场重定价

- 时间：约 `3个月`，或约 `10周` 且恢复明显晚于预期
- 价格：Brent 峰值约 `$140`，年均约 `$100`
- 含义：更接近滞涨，通胀再上行，衰退风险抬升

### 情景 3：长期中断，衰退高概率

- 时间：约 `6个月`
- 价格：Brent 峰值约 `$170`，年均约 `$120`
- 含义：全球衰退高概率，政策与金融市场同时承压

### 情景 4：海峡部分恢复，但实际供给恢复不足

- 特征：名义通航恢复，但卡尔格岛、运力、保险、炼化错配、成品油偏紧拖慢实际恢复
- 含义：油价未必冲极端高点，但高位与高波动持续更久

### 情景 5：军事升级换谈判

- 特征：军事施压升级，同时为停火谈判铺路
- 含义：短期风险更高，但若谈判成形，价格回落也可能更快

## 五、价格与宏观传导

### 1. 价格形成顺序

短期价格优先看：

1. 海峡能否过船
2. 波斯湾能否正常装船
3. 出口节点是否可用
4. 运力、保险、交付是否通畅
5. 替代路线是否有效
6. 炼化错配和成品油紧张是否放大冲击
7. 需求破坏是否被迫出现

### 2. 宏观传导主链

`海峡/节点中断 -> 原油与成品油上涨 -> 航运/航空成本上升 -> 通胀预期抬升 -> 消费和投资走弱 -> 增长下修 -> 风险偏好下降`

### 3. 通胀传导副链

`油价上涨 -> headline inflation 上升 -> core inflation 通过交通服务滞后抬升`

`化肥/化工/金属上涨 -> 食品与工业品价格上升 -> 非能源商品通胀上升`

`现实通胀上升 -> 预期上修 -> 二轮效应 -> 政策约束增强`

### 4. 美国侧默认量化模板

当用户关心美国通胀或 Fed 时，可优先使用：

- `油价每上涨10% -> headline PCE +0.2pp`
- `油价每上涨10% -> core PCE +0.04pp`
- 基准情景下能源价格对 `core PCE` 的峰值拉动约 `0.35pp`
- 更差情景下峰值可接近 `0.5pp`
- 二轮效应在基准情景较小，在严重不利情景会放大

### 5. 油价区间模板

以下只是经验区间，不是硬预测：

- `100-120 美元`：更接近可控通胀
- `120-150 美元`：更接近滞涨
- `150-200 美元` 且持续：更接近衰退

## 七、更新规则

### 上调风险等级

出现以下任一情况，应上调风险等级：

- 海峡低流量显著长于市场预期
- 卡尔格岛持续失能或被封锁
- 运力、保险或交付链条失灵
- 替代路线也受扰
- 东亚库存接近临界水平但补给未恢复
- 非油商品、食品价格和通胀预期同步走强

### 下调风险等级

出现以下组合时，可下调风险等级：

- 海峡恢复稳定通航
- 卡尔格岛恢复装运
- 运力与保险恢复正常
- 替代路线未继续恶化
- 谈判中介明确且进入实质讨论
- 成品油、实物市场溢价和通胀预期回落

## 八、结论模板

`当前主情景`：  
一句话定义当前最可能路径。

`核心变量`：  
列出 3 个最关键变量，并说明是改善还是恶化。

`价格判断`：  
说明当前更接近哪个情景。

`经济含义`：  
说明当前更像短期通胀、滞涨还是衰退风险上升。

`后续触发点`：  
指出未来 1-2 周最值得盯的事件或数据。

---

## 九、默认基线

在没有更强反证前，agent 可采用以下默认基线：

- 当前不能把局势简化为单一“海峡恢复”问题。
- 当前最重要的新变量是 `卡尔格岛状态`。
- 当前最重要的新风险层是 `运输与交付`。
- 当前最容易被低估的是 `成品油紧张度`、`炼化错配`、`区域库存消耗`。
- 若冲突从 `1个月` 向 `3个月+` 滑移，市场大概率需要重定价滞涨或衰退风险。
