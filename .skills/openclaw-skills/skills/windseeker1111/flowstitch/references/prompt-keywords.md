## Shape → Natural Language Translations

| Technical | Natural Language |
|:---|:---|
| `rounded-none` | sharp, squared-off edges |
| `rounded-sm` | slightly softened corners |
| `rounded-md` | gently rounded corners |
| `rounded-lg` | generously rounded corners (12px) |
| `rounded-xl` | very rounded, pillow-like (16px) |
| `rounded-2xl` | softly rounded, approachable (20px) |
| `rounded-full` | pill-shaped, circular |

## Shadow Vocabulary

| Style | Description |
|:---|:---|
| Flat | No shadows, color blocking and borders only |
| Whisper-soft | Diffused, light shadow for subtle lift |
| Floating | High-offset, soft shadow for elements above surface |
| Inset | Inner shadow for pressable or nested elements |
| Dramatic | High-contrast, sharp shadow for bold statements |
