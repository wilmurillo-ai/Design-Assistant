# food-tour

Plan culinary travel experiences — local food tours, Michelin restaurants, street food crawls, cooking classes, food markets, and regional specialty tasting routes.

## Overview

Plan culinary travel experiences — local food tours, Michelin restaurants, street food crawls, cooking classes, food markets, and regional specialty tasting routes. Wraps the flyai-cli to provide real-time travel data with booking links.

## Quick Start

### Install Skill

```bash
clawhub install food-tour
# or
npx skills add alibaba-flyai/flyai-skill/tree/main/skills/food-tour
```

### Install CLI

```bash
npm i -g @fly-ai/flyai-cli
```

### Verify

```bash
flyai --version
```

## What It Does

Plan culinary travel experiences — local food tours, Michelin restaurants, street food crawls, cooking classes, food markets, and regional specialty tasting routes.
