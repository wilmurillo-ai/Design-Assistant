# family-hotel

Find hotels perfect for families — multi-bed rooms, kid-friendly amenities, nearby playgrounds, and family-oriented services.

## Overview

Find hotels perfect for families — multi-bed rooms, kid-friendly amenities, nearby playgrounds, and family-oriented services. Wraps the flyai-cli to provide real-time travel data with booking links.

## Quick Start

### Install Skill

```bash
clawhub install family-hotel
# or
npx skills add alibaba-flyai/flyai-skill/tree/main/skills/family-hotel
```

### Install CLI

```bash
npm i -g @fly-ai/flyai-cli
```

### Verify

```bash
flyai --version
```
