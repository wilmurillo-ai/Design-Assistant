# historical-sites

Explore ancient ruins, monuments, UNESCO World Heritage sites, and historical landmarks with detailed cultural context and visiting guides.

## Overview

Explore ancient ruins, monuments, UNESCO World Heritage sites, and historical landmarks with detailed cultural context and visiting guides. Wraps the flyai-cli to provide real-time travel data with booking links.

## Quick Start

### Install Skill

```bash
clawhub install historical-sites
# or
npx skills add alibaba-flyai/flyai-skill/tree/main/skills/historical-sites
```

### Install CLI

```bash
npm i -g @fly-ai/flyai-cli
```

### Verify

```bash
flyai --version
```

## What It Does

Explore ancient ruins, monuments, UNESCO World Heritage sites, and historical landmarks with detailed cultural context and visiting guides.
