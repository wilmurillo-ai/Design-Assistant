# airport-pickup-service

Book airport transfer services — private cars, shared shuttles, and limo services for comfortable, stress-free arrivals and departures.

## Overview

Book airport transfer services — private cars, shared shuttles, and limo services for comfortable, stress-free arrivals and departures. Wraps the flyai-cli to provide real-time travel data with booking links.

## Quick Start

### Install Skill

```bash
clawhub install airport-pickup-service
# or
npx skills add alibaba-flyai/flyai-skill/tree/main/skills/airport-pickup-service
```

### Install CLI

```bash
npm i -g @fly-ai/flyai-cli
```

### Verify

```bash
flyai --version
```

## What It Does

Book airport transfer services — private cars, shared shuttles, and limo services for comfortable, stress-free arrivals and departures.

**Command type:** Single-command
