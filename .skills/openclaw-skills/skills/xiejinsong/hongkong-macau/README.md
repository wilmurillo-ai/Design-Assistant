# hongkong-macau

## Overview

Plan your Hong Kong and Macau experience — Victoria Peak views, dim sum crawls, Macau casino strips, heritage walks, and duty-free shopping sprees. Wraps the flyai-cli to provide real-time travel data with booking links.

## Quick Start

### Install Skill

```bash
clawhub install explore-hongkong-macau
# or
npx skills add alibaba-flyai/flyai-skill/tree/main/skills/explore-hongkong-macau
```

### Install CLI

```bash
npm i -g @fly-ai/flyai-cli
```

### Verify

```bash
flyai --version
```
