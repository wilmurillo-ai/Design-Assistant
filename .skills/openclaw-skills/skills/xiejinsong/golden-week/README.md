# golden-week

Plan Golden Week (National Day) or Spring Festival trips — beat the crowds with smart timing, find available hotels during peak season, and save on inflated holiday prices.

## Overview

Plan Golden Week (National Day) or Spring Festival trips — beat the crowds with smart timing, find available hotels during peak season, and save on inflated holiday prices. Wraps the flyai-cli to provide real-time travel data with booking links.

## Quick Start

### Install Skill

```bash
clawhub install golden-week-trip
# or
npx skills add alibaba-flyai/flyai-skill/tree/main/skills/golden-week-trip
```

### Install CLI

```bash
npm i -g @fly-ai/flyai-cli
```

### Verify

```bash
flyai --version
```
