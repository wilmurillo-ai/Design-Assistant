# ancient-town-explorer

Discover preserved ancient towns, water villages, and traditional settlements — stone bridges, wooden houses, and time-frozen streetscapes.

## Overview

Discover preserved ancient towns, water villages, and traditional settlements — stone bridges, wooden houses, and time-frozen streetscapes. Wraps the flyai-cli to provide real-time travel data with booking links.

## Quick Start

### Install Skill

```bash
clawhub install ancient-town-explorer
# or
npx skills add alibaba-flyai/flyai-skill/tree/main/skills/ancient-town-explorer
```

### Install CLI

```bash
npm i -g @fly-ai/flyai-cli
```

### Verify

```bash
flyai --version
```

## What It Does

Discover preserved ancient towns, water villages, and traditional settlements — stone bridges, wooden houses, and time-frozen streetscapes.

**Command type:** Single-command
