# group-tour

Find organized group tours and travel packages with professional guides, planned itineraries, meals included, and hassle-free travel for those who prefer structure.

## Overview

Find organized group tours and travel packages with professional guides, planned itineraries, meals included, and hassle-free travel for those who prefer structure. Wraps the flyai-cli to provide real-time travel data with booking links.

## Quick Start

### Install Skill

```bash
clawhub install group-tour
# or
npx skills add alibaba-flyai/flyai-skill/tree/main/skills/group-tour
```

### Install CLI

```bash
npm i -g @fly-ai/flyai-cli
```

### Verify

```bash
flyai --version
```
