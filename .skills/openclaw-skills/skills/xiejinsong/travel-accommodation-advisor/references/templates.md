# Templates

## 标准问询清单
1. 你要围绕的地标是哪个？（景点/商圈/车站/机场/会展中心）
2. 城市是哪里？（用于 `--city-name` / `--dest-name`）
3. 入住和离店日期？（`--check-in-date` / `--check-out-date`）
4. 预算上限是多少？（`--max-price`，可为空）
5. 希望住多近？（半径默认 3km，可改 1/2/5km）
6. 更看重距离还是热度评分？（决定 `distance_asc` 或 `rate_desc`）

## 参数填充模板
```text
城市=<城市>
地标=<地标名称>
入住=<yyyy-mm-dd>
离店=<yyyy-mm-dd>
预算=<数字，可空>
半径=<1|2|3|5 km，默认3>
排序=<distance_asc|rate_desc，默认distance_asc>
星级=<3,4,5，可空>
```
