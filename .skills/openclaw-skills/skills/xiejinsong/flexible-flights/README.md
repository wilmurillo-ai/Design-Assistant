# flexible-flights

Find the cheapest day to fly within a date range. Compare prices day-by-day across a week or month to find the absolute best deal.

## Overview

Find the cheapest day to fly within a date range. Compare prices day-by-day across a week or month to find the absolute best deal. Wraps the flyai-cli to provide real-time travel data with booking links.

## Quick Start

### Install Skill

```bash
clawhub install flexible-flights
# or
npx skills add alibaba-flyai/flyai-skill/tree/main/skills/flexible-flights
```

### Install CLI

```bash
npm i -g @fly-ai/flyai-cli
```

### Verify

```bash
flyai --version
```

## What It Does

Find the cheapest day to fly within a date range. Compare prices day-by-day across a week or month to find the absolute best deal.

**Command type:** Single-command
