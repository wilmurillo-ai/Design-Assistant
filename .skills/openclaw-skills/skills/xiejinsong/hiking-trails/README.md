# hiking-trails

Find hiking trails from easy walks to challenging mountain treks. Includes difficulty levels, elevation, duration, and what to pack.

## Overview

Find hiking trails from easy walks to challenging mountain treks. Includes difficulty levels, elevation, duration, and what to pack. Wraps the flyai-cli to provide real-time travel data with booking links.

## Quick Start

### Install Skill

```bash
clawhub install hiking-trails
# or
npx skills add alibaba-flyai/flyai-skill/tree/main/skills/hiking-trails
```

### Install CLI

```bash
npm i -g @fly-ai/flyai-cli
```

### Verify

```bash
flyai --version
```

Find hiking trails from easy walks to challenging mountain treks. Includes difficulty levels, elevation, duration, and what to pack.
