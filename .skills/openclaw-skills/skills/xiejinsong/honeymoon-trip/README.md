## Overview

Plan a dreamy honeymoon — luxury hotels, romantic activities, couples' spa, sunset dinners, and unforgettable experiences for newlyweds. Wraps the flyai-cli to provide real-time travel data with booking links.

## Quick Start

### Install Skill

```bash
clawhub install honeymoon-trip
# or
npx skills add alibaba-flyai/flyai-skill/tree/main/skills/honeymoon-trip
```

### Install CLI

```bash
npm i -g @fly-ai/flyai-cli
```

### Verify

```bash
flyai --version
```
