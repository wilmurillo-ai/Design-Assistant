# garden-parks

Explore classical Chinese gardens, city parks, botanical gardens, and royal gardens — perfect for relaxing walks and cultural appreciation.

## Overview

Explore classical Chinese gardens, city parks, botanical gardens, and royal gardens — perfect for relaxing walks and cultural appreciation. Wraps the flyai-cli to provide real-time travel data with booking links.

## Quick Start

### Install Skill

```bash
clawhub install garden-parks
# or
npx skills add alibaba-flyai/flyai-skill/tree/main/skills/garden-parks
```

### Install CLI

```bash
npm i -g @fly-ai/flyai-cli
```

### Verify

```bash
flyai --version
```
