# Templates

## 标准问询清单
1. 你的护照国籍是哪里？
2. 计划去哪个国家/地区？是否单一目的地？
3. 预计停留多久、计划哪天出发？
4. 是否接受电子签/落地签，还是必须贴签？
5. 你准备在哪个城市办理签证？
6. 是否需要把签证办理与机票/酒店一起规划？

## 参数填充模板
```text
护照国籍=<国家/地区>
目的地=<国家/地区>
出发日期=<yyyy-mm-dd>
停留时长=<x天>
办理城市=<城市，可空>
是否多国行程=<是/否>
转机地=<城市/国家，可空>
预算=<数字，可空>
```

## Markdown 输出模板
```markdown
# 旅行签证建议（<护照国籍> -> <目的地>）

## 结论
- 当前行程建议：<免签/电子签/贴签/需进一步核验>。
- 建议最晚在 <日期> 前启动办理，预留 <x> 个工作日缓冲。

## 办理路径
1. 核验签证政策与材料清单。
2. 确认办理地点（领馆/签证中心）与预约要求。
3. 按时间线准备递签与取签。

## 关键核验项
| 项目 | 结果 | 备注 |
|---|---|---|
| 是否需要签证 | <是/否> | <说明> |
| 是否可电子签 | <是/否> | <说明> |
| 预计办理时效 | <x 工作日> | <旺季可能延长> |

## 办理地点（如有）
### <地点名称>
![](<picUrl>)

```
