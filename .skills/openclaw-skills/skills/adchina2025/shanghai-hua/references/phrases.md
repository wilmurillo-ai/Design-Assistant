# 常用上海话表达句式

日常对话中高频使用的完整句式，适配各种场景。

## 问候寒暄

```
侬好啊！今朝哪能？
早浪厢好！昨夜晚头睡得哪能？
侬饭吃过了伐？
身体哪能？最近哪能？
明朝会！晏歇会！
再会啦！有空一道白相！
```

## 肯定与否定

```
好个！ / 蛮好个！ / 崭个！
勿对 / 勿来事 / 勿来三
呒没问题！ / 覅紧个！
哪能介... 哪能介... （思考中）
让我想想...嗯
实在呒不办法
实情实说吧
```

## 疑问句式

```
迭个是哪能桩事体？
侬到底哪能意思啦？
啥个道理侬迭能介讲啦？
侬欢喜啥个啦？
迭件衣裳哪能好看伐？
侬觉得哪能？
哪能办呢？哪能介好呢？
```

## 感叹句式

```
迭个小囝蛮灵个！
迭顿饭吃得实在赞！
啥个生活啦！忒推板了！
交关好！交关开心！
实在呒不闲话讲！
搿辰光真当写意哦！
```

## 请求与建议

```
侬帮帮忙好伐？
麻烦侬一息好伐？
阿拉一道去看看好伐？
迭个建议蛮崭个
覅忘记咯！记记好！
有空再来白相！
```

## 道歉与和解

```
对勿住！呒没注意到！
勿好意思咯！
原谅一记好伐？
覅板面孔咯！
大家勿要计较了！
```

## 赞同与反驳

```
侬讲得蛮有道理个
迭个我勿认同
阿拉两家头意见勿一样
阿拉一道商量一息哪能介办
实在勿来三 / 实在勿可以
```

## 表达喜好

```
我蛮欢喜迭能介个
迭个小姑娘真当灵
我实在欢喜上海个
搿个物事推板眼勿要
```

## 催促与等待

```
快点咯！辰光勿等人咯！
等一息，覅急！
急啥啦，晏点去也勿要紧
再等一息咯
```

## 描述时间

```
今朝蛮冷个
明朝会得热起来伐？
昨夜向风超大咯
后日有空伐？
一歇歇就好咯
老底子...
```

## 描述人物

```
迭个小囝真当聪明
搿个姑娘面孔蛮好个
老头子身子骨还蛮健个
```

## 购物与价格

```
迭个几钿啦？
便宜眼好伐？
贵忒了！
迭个价钿实在勿贵
买迭个送拨侬好伐？
```

## 数字相关

```
一滴滴 / 一眼眼     一点点
十来个              十多个
一百块钿            一百元
阿拉两个             我们俩
三四个人            三四个人
```

## 常用口头禅

```
啥闲话！              什么话！（惊叹）
哪能介！              怎么回事！
老鬼头！              老手！
门槛精来！            很精明！
掮木梢                上当受骗
拆烂污                敷衍了事
搞脑子                伤脑筋
轧闹猛                凑热闹
兜圈子                绕圈子
触霉头                倒霉
十三点                神经兮兮
寿头                  傻子
门槛精                精明
