# 核心辩论参考

用于补充主 skill 的辩论执行细节，不重复主流程。

## 1. 角色追问清单

### 价值派
- 护城河是否真实存在
- ROE / ROIC 是否可持续
- 现金流是否匹配利润
- 当前估值是否已具备安全边际

### 成长派
- 增速来自哪里
- 是否存在第二曲线
- 渗透率提升是否还在早中期
- 市场是否低估中期成长空间

### 资金派
- 机构是否在加减仓
- 资金流向是否支持当前逻辑
- 市场是在交易复苏还是交易下行
- 当前位置是否已经 price in 某种预期

### 宏观 / 政策派
- 利率、汇率、流动性是否有利
- 行业监管方向是否边际改善或恶化
- 是否存在补贴、集采、关税、出口限制等变量
- 海外敞口是否受地缘冲击影响

### 空头代表
- 市场最可能没有定价的风险是什么
- 关键叙事里最脆弱的假设是什么
- 哪个变量一旦反向，将直接击穿多头逻辑
- 哪些说法只是故事，不是证据

---

## 2. 争议点设计

好的争议点应满足：
- 是当前股价背后的主要矛盾
- 可以通过新增证据推进
- 会改变持有 / 加仓 / 回避决策

常见争议点：
- 这是价值修复，还是价值陷阱？
- 增长是短期脉冲，还是中期趋势？
- 当前估值在交易复苏，还是交易幻觉？
- 风险已被市场充分定价，还是尚未体现？
- 政策是顺风，还是未来最大的回撤来源？

---

## 3. 主持人追问

当出现空话时，可直接追问：
- 你的依据是什么？
- 这是事实、推断，还是市场传闻？
- 这个判断对应的是哪个季度、哪个政策周期？
- 有没有更高优先级来源能验证？
- 如果这个假设错了，哪条逻辑先坏？

- 当前最强空头证据：...
