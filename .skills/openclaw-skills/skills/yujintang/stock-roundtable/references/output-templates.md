# 输出模板参考

只提供可复用输出骨架，不放完整示例节奏。

## 1. 快速交锋模板

```markdown
# {股票名称} 快速交锋

## 核心争议
{一句话}

## 多方观点
- 多头：...
- 空头：...

## 实时查证
- [来源 | 时间] ...
- [来源 | 时间] ...

## 迷你结论
- 当前判断：...
- 置信度：高 / 中 / 低
- 最大风险：...
- 下一验证点：...
```

## 2. 标准辩论模板

```markdown
# {股票名称} 标准辩论

## 开场
- 当前价 / 市值 / 估值：...
- 核心矛盾：...
- 当前最重要变量层：公司 / 行业 / 政策 / 宏观 / 地缘

## Round 1
- 争议：...
- 查证后裁决：...

## Round 2
- 争议：...
- 查证后裁决：...

## 当前结论
- 一句话判断：...
- 最强多头逻辑：...
- 最强空头逻辑：...
- 空头否决项是否解除：...
- 置信度：高 / 中 / 低

## 跟踪清单
1. ...
2. ...
3. ...
```

## 3. 深度审判模板

```markdown
# {股票名称} 深度审判

## 开场画像
- 价格 / 市值 / 估值 / 近期表现
- 用户持仓背景
- 主要矛盾层
- 次要风险层

## 多层审判
- Round 1：公司层
- Round 2：行业层
- Round 3：政策 / 宏观 / 地缘层
- Round 4：估值与仓位决策

## 终局裁决
- 一句话结论
- 多空力量对比
- 空头否决项
- 置信度

## 持仓行动指南
- 继续持有条件
- 转入观察条件
- 减仓 / 止损 / 止盈条件

## 后续跟踪
- 关键财报指标
- 关键政策事件
- 关键宏观 / 地缘变量
```
