# 财报模式参考

适用于财报发布后 24-48 小时内的快速反应场景。

## 1. 模式目标

不是复述财报，而是快速回答：
- 财报到底超预期还是低于预期？
- 超 / 低预期来自哪里？
- 市场反应是否合理？
- 这会改变中期逻辑，还是只是短期噪音？

## 2. 财报模式额外必查

- 最新财报核心数据：营收、净利润、毛利率、经营现金流
- 管理层指引
- 市场预期 / 一致预期（若可获取）
- 财报发布后股价表现
- 资金 / 分析师 / 市场关注点

## 3. 建议辩论顺序

- Round 1：财报本身好不好
- Round 2：超 / 低预期来自主营变化还是一次性因素
- Round 3：指引与下季度验证点是否可信
- Round 4：市场反应是在交易事实还是情绪

## 4. 财报季专属追问

- 增长是不是靠低基数？
- 毛利率变化是结构改善还是暂时因素？
- 现金流是否跟上利润？
- 管理层指引是保守还是已经转弱？
- 市场今天的反应，是否提前 price in 了？

## 5. 财报季结尾模板

```markdown
## 财报季裁决
- 这份财报：超预期 / 符合预期 / 低于预期
- 最重要的正面变化：...
- 最重要的负面变化：...
- 市场反应：合理 / 偏激 / 仍未充分定价
- 中期逻辑是否变化：...
- 下一次最重要验证点：...
```
