# 行业打法参考

只补充行业特有的观察点，通用辩论流程仍以主 skill 为准。

## 1. 消费 / 白马
重点看：
- 品牌力与渠道控制力
- 提价能力
- 现金流与库存
- 历史估值分位

空头常见质疑：
- 品牌是否弱化
- 价格带是否被竞争对手击穿
- 增长是否进入平台期

## 2. 科技 / 成长
重点看：
- 增速来源
- 产品周期
- 渗透率与 TAM
- 估值是否透支远期预期

空头常见质疑：
- 增速一旦掉到某阈值，估值会压缩到哪里
- 第二曲线是否还只是故事

## 3. 医药 / 医疗器械
重点看：
- 产品管线 / 核心产品结构
- 审批节奏
- 集采与价格压力
- 海外拓展

空头常见质疑：
- 集采或降价后利润还能剩多少
- 单一大单品依赖是否过高

## 4. 周期 / 制造
重点看：
- 价格周期
- 产能投放
- 订单与库存
- 资本开支

空头常见质疑：
- 当前是不是周期高点附近
- 扩产是否会反过来压垮利润

## 5. 银行 / 保险
重点看：
- 资产质量
- 资本充足率
- 负债成本
- 投资收益与利差

空头常见质疑：
- 隐性不良是否被低估
- 利率变化会不会持续侵蚀利润

## 6. 平台互联网
重点看：
- 用户时长 / GMV / 变现效率
- 回购与股东回报
- 监管边际变化
- 海外业务敞口

空头常见质疑：
- 如果没有增长，市场还愿意给多少倍估值
- 监管与反垄断是否会重来

## 7. ETF / 指数
重点看：
- 底层成分结构
- 风格暴露
- 估值分布
- 再平衡与拥挤度

空头常见质疑：
- 这个 ETF 是否装着很多你单独不会买的公司
- 当前买入本质是在赌哪种风格
