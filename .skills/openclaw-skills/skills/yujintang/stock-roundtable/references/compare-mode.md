# 对比模式参考

适用于用户提问：
- A 和 B 哪个更值得买？
- 两只股票谁更适合我当前仓位？
- 同赛道哪个公司更强？

## 一、比较原则

对比分析必须避免“各说各好”，应强制落到统一维度：
- 商业模式
- 财务质量
- 增长质量
- 估值水平
- 政策 / 宏观 / 地缘暴露
- 适合的投资者类型

## 二、推荐流程

### Step 1：统一画像
先给两只标的做同口径快照：
- 市值
- 估值
- 最新财报增速
- 核心业务
- 当前主要争议

### Step 2：逐维度辩论
每轮只比一个维度，例如：
- Round 1：谁的商业模式更稳
- Round 2：谁的增长更可信
- Round 3：谁的估值和风险收益比更优
- Round 4：谁更受政策 / 地缘冲击

### Step 3：强制裁决
每轮必须判定：
- A 胜
- B 胜
- 平局

如果平局，必须说明为什么平，以及哪个变量最可能打破平局。

## 四、空头专属任务

空头必须回答：
- 如果只能淘汰一个，先淘汰谁？为什么？
- 哪个看起来更便宜，但其实更危险？
- 哪个当前市场误判最严重？
