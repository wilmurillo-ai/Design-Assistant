# 示例参考

注意：本文件只展示结构与节奏，**不要把示例占位符当成真实数据**。执行时必须依赖实时搜索结果。

## 1. 单轮争议示例

```markdown
## Round 1：增长是真复苏，还是短期脉冲？

### 当前争议
公司近一季收入反弹，究竟意味着景气反转，还是一次性补库存？

### 先发言
- 价值派：如果现金流与利润同步改善，我愿意承认不是假复苏。
- 成长派：只要核心业务增速连续两个季度抬升，市场会重新定价。
- 资金派：关键不只是增速，而是机构有没有回流。
- 空头：先别讲复苏，证明这不是渠道压货。

### 证据缺口
- 收入改善是否伴随现金流改善
- 增长是终端需求拉动还是库存错配
- 市场是否已经提前交易复苏

### 实时查证
- 搜索：最新财报 经营现金流 / 存货 / 应收
- 搜索：最近两周行业销量 / 出货 / 渠道信息
- 搜索：机构持仓或资金流变化

### 证据回场
- [财报 | 最新季度] 经营现金流改善 / 未改善
- [行业数据 | 最近1个月] 出货量变化
- [资金数据 | 最近区间] 北向 / 南向 / 机构流向变化

### 立场更新
- 若现金流和出货同步改善：成长派被加强
- 若收入改善但库存与应收恶化：空头被加强
- 若资金仍未回流：资金派保持谨慎
```

## 2. 持仓场景结尾示例

```markdown
## 当前裁决
- 一句话结论：当前更像“带验证条件的持有”，不是确定性加仓点。
- 最强多头逻辑：核心业务若连续两个季度改善，估值有修复空间。
- 最强空头逻辑：政策与外需变量尚未出清，利润质量仍需验证。
- 置信度：中

## 接下来最该看
1. 下一次财报里的营收增速与经营现金流
2. 行业政策是否继续收紧 / 放松
3. 海外需求、汇率或地缘风险是否恶化
```
