# 支持的股票代码格式

TradingAgents-CN 支持三种股票市场的分析：A股、港股、美股。

## A股市场

### 代码格式
- **6位数字**，如：`600519`
- 无需添加前缀或后缀

### 交易所识别

| 代码开头 | 交易所 | 示例 |
|----------|--------|------|
| 60xxxx | 上海主板 | 600036 招商银行 |
| 00xxxx | 深圳主板 | 000001 平安银行 |
| 30xxxx | 创业板 | 300750 宁德时代 |
| 68xxxx | 科创板 | 688981 中芯国际 |

## 数据源说明

| 市场 | 主要数据源 | 备用数据源 |
|------|-----------|-----------|
| A股 | Tushare, AkShare | BaoStock |
| 港股 | Yahoo Finance | - |
| 美股 | FinnHub, Yahoo Finance | - |

### 数据获取限制

- **Tushare**：需要注册获取 Token，免费版每日 500 次请求
- **AkShare**：免费，无需注册，但有频率限制
- **FinnHub**：免费版每分钟 60 次请求
- **Yahoo Finance**：免费，但可能不稳定

## 示例用法


## 常见问题

### Q: A股代码输入 600519 提示找不到？

检查：
1. 是否配置了 Tushare Token 或 AkShare 正常运行
2. 尝试先运行数据初始化

### Q: 港股代码格式错误？

港股代码格式为 `数字.HK`，如 `0700.HK`，注意：
- 前面的 0 不能省略
- 必须包含 `.HK` 后缀

### Q: 美股代码区分大小写？

美股代码不区分大小写，`AAPL` 和 `aapl` 都可以。
