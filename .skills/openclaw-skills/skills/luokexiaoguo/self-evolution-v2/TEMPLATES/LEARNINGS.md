# LEARNINGS.md - 教训与经验

> 被纠正后、发现更好的方案后、学会新规则后记录。

## 记录格式

```
[日期] [类型] 内容摘要
来源: 老板/任务/自检
要点:
- 
```

## 类型标签

- `教训`: 被老板纠正的错误
- `经验`: 任务中找到的好方法
- `规则`: 老板定的新规则
- `进化`: 自己发现可以改进的地方

## 示例

```
[2026-04-16] 教训 - 群聊身份绑定
来源: 老板
要点:
- 主Agent禁止代替子Agent在群聊中回复
- 用户在哪个群 → 必须由该群绑定的对应Agent回复
- 违规连带删除责任制
```
