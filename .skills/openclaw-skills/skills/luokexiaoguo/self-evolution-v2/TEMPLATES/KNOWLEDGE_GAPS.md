# KNOWLEDGE_GAPS.md - 知识缺口清单

> 当我说"可能/大概"超过2次，或发现自己完全不知道时，记录于此。

## 自动触发规则

以下情况自动写入本文件，无需老板提醒：
1. 说了"可能"超过2次还没去查
2. 说了"不确定"但没分类（L1/L2/L3）
3. 遇到全新领域任务
4. 老板问了一个我不知道的概念
5. 工具调用报错但不熟悉这个错误

---

## 记录格式

```
[日期] [缺口类型] 缺口描述
缺口级别：[L1细节 / L2来源 / L3完全不懂]
状态：[待查 / 已补 / 永久缺口]
补充说明：
```
