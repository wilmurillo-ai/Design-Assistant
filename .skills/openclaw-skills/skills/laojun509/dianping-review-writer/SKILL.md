# 大众点评评论生成器

帮助用户快速生成高质量、真实感强的大众点评评论，提升账号等级。

## 核心功能

- **多类型商家支持**：餐厅、美容美发、休闲娱乐、运动健身、酒店等
- **多种风格可选**：客观详实型、故事感型、简短精悍型
- **账号升级优化**：评论长度、关键词、配图建议均按大众点评算法优化
- **批量生成**：一次可生成多条评论供选择

## 使用方法

用户只需提供：

1. **商家类型**（必填）：餐厅/美容/KTV/健身房/酒店等
2. **具体品类**（可选）：如餐厅细分到火锅、日料、烧烤等
3. **消费体验**（可选）：简单描述好坏（如"味道不错但上菜慢"、"环境很好"）
4. **评分意向**（可选）：4星/5星/中立
5. **生成数量**（可选）：默认3条

## 评论生成策略

### 高权重评论特征（利于账号升级）

| 要素 | 建议 |
|------|------|
| **字数** | 100-200字最佳，Lv4+建议200+字 |
| **配图** | 3-9张，带图评论权重翻倍 |
| **关键词** | 包含具体菜名、服务细节、环境描述 |
| **结构化** | 环境+菜品+服务+总结 |
| **时效性** | 到店当天或次日发布 |

### 风格模板

**客观详实型**（适合Lv4-Lv8升级）
```
环境：描述装修、氛围、座位舒适度
菜品：具体点名+口味评价+推荐/踩雷
服务：上菜速度、态度、响应及时性
总结：性价比、是否值得再来
```

**故事感型**（适合吸引点赞）
```
以具体场景切入："朋友聚会"、"约会"、"加班后的深夜"
加入细节："晚上8点到的"、"老板娘推荐的"
情绪共鸣："意外惊喜"、"有点小遗憾"
```

**简短精悍型**（快速刷量）
```
一句话评价核心亮点
一句话补充
适合Lv1-Lv3快速升级
```

## 生成流程

1. 询问用户商家类型和体验
2. 根据类型调用对应prompt模板
3. 生成3条不同风格的评论供选择
4. 提供配图建议和标签推荐

**用户**：帮我写个火锅店的评论

**助手**：
- 哪家火锅店？（品牌/类型）
- 体验怎么样？
- 打几星？

**标签推荐**：#重庆火锅 #毛肚 #朋友聚会 #性价比高

## 脚本使用

如需程序化生成，使用 `scripts/generate.py`：

```bash
python scripts/generate.py \
  --type restaurant \
  --cuisine "日料" \
  --experience "食材新鲜但价格偏高" \
  --rating 4 \
  --count 3
```

参数说明：
- `--type`: 商家类型（restaurant/beauty/entertainment/fitness/hotel）
- `--cuisine`: 细分品类（可选）
- `--experience`: 体验简述（可选）
- `--rating`: 目标评分 1-5（可选）
- `--count`: 生成数量，默认3
- `--style`: 风格（detailed/story/short），不传则生成混合风格

## 注意事项

1. **内容真实**：虽然AI生成，但建议基于真实消费体验，避免虚假宣传
2. **不要完全相同**：同一商家不要发布完全一样的评论，会被判定刷评
3. **发布间隔**：批量评论建议间隔几天发布，不要同一天大量发
4. **配图原创**：尽量使用自己拍的照片，网图可能被识别

## 升级策略建议

**Lv1 → Lv3（快速）**：
- 每天1-2条，每条50-100字
- 带图评论优先

**Lv3 → Lv5（稳定）**：
- 每周3-5条，每条150-200字
- 覆盖不同类型商家
- 多写优质点评（被点赞、回复）

**Lv5 → Lv8（质量）**：
- 专注写长文优质点评（300+字）
- 成为"VIP会员"（每月写4条优质点评）
- 争取"优质点评"标签
