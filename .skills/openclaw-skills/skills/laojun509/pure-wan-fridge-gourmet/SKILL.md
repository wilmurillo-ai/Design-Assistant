## [一句话总览]
拍一张冰箱照片，AI帮你生成一道精致料理的概念图 + **真实可信的完整菜谱**，支持50+种菜系、4种万相模型，还能输出烹饪视频。

## [标准流程图版]

**STEP1 | 拍冰箱**
打开手机，对着冰箱内部拍一张照片，上传给AI。

**STEP2 | 选风格**
从50+种菜系中选择：想要川菜还是寿司？法式大餐还是墨西哥taco？
选择难度：简单快手、适中、挑战级
选择时间：15分钟、30-45分钟、1小时+

**STEP3 | 等出图 + 菜谱**
AI分析你冰箱里的食材，生成：
- 一张精致料理的概念图（或视频！）
- 完整的菜谱（食材清单 + 烹饪步骤 + 技巧）

## 使用示例

```bash
# 基础用法（生成图片+菜谱）
python3 scripts/generate_gourmet.py 冰箱照片.jpg

# 指定风格：日式简单快手菜
python3 scripts/generate_gourmet.py 冰箱照片.jpg --cuisine japanese --difficulty easy --cooking-time quick

# 使用万相2.7图生图（默认，质量最好）
python3 scripts/generate_gourmet.py 冰箱照片.jpg --model 2.7 -c sichuan -d medium -t moderate

# 使用万相2.1文生视频（会基于菜谱生成烹饪视频）
python3 scripts/generate_gourmet.py 冰箱照片.jpg --model 2.1-t2v -c chinese -d medium -t moderate

# 图生视频（基于冰箱照片生成）
python3 scripts/generate_gourmet.py 冰箱照片.jpg --model 2.1-i2v -c italian

# 加额外要求
python3 scripts/generate_gourmet.py 冰箱照片.jpg --custom "不要辣，减脂期"

# 只生成图片，不生成菜谱
python3 scripts/generate_gourmet.py 冰箱照片.jpg --no-recipe

# 查看所有可用的菜系
python3 scripts/generate_gourmet.py --list-cuisines

# 查询任务状态
python3 scripts/generate_gourmet.py --query <task_id>
```

## 🆕 v3.0 真实菜谱系统

### 真实经典菜谱库
- 收录经过验证的经典家常菜做法
- 优先从数据库匹配真实菜谱，而非AI随意生成
- 包含：番茄炒蛋、青椒炒蛋、黄瓜炒蛋、胡萝卜炒蛋、洋葱炒蛋、时蔬杂炒等

### 详细烹饪技法
每个步骤包含：
- **【火候】** 大火/中火/小火，具体温度范围
- **【时长】** 精确到秒/分钟
- **【判断标准】** 如何判断这一步完成（如"蛋液凝固成块"）
- **💡 关键技巧** 真实有用的烹饪窍门

```
   2. 番茄：2个中等大小
   3. 葱花：1根
   4. 盐：1/2茶匙
   5. 白糖：1茶匙

👨‍🍳 烹饪步骤

番茄洗净，顶部划十字，用开水烫10秒后去皮，切成滚刀块。
鸡蛋打入碗中，加1/4茶匙盐，用筷子充分搅打至起泡（约30秒）。
   【火候】无
   【时长】3分钟
   💡 蛋液充分搅打炒出来更蓬松

步骤 2. 【炒鸡蛋】
热锅凉油，大火烧至油温六成热（约160°C），倒入蛋液。
待蛋液底部凝固，用铲子轻轻推动，炒至七分熟盛出备用。
   【火候】大火
   【时长】30秒
   💡 不要炒太老，七分熟就要盛出

步骤 3. 【炒番茄】
锅中再加少许油，中火下番茄块翻炒。加1/4茶匙盐，炒至番茄出汁变软。
   【火候】中火
   【时长】2分钟
   💡 加盐能加速番茄出汁

步骤 4. 【合炒调味】
倒入炒好的鸡蛋，加白糖提鲜，快速翻炒均匀。撒葱花，关火盛出。
   【火候】中火
   【时长】30秒
   💡 糖能中和番茄酸味，提鲜关键

============================================================
💡 烹饪技巧
============================================================
   1. 蛋液要充分搅打，炒出来才蓬松
   2. 番茄去皮口感更好，但不去皮营养更丰富
   3. 炒鸡蛋时油温要高，这样蛋才嫩滑
```

## 烹饪技法速查

| 火候 | 温度 | 适用场景 | 判断方法 |
|------|------|----------|----------|
| **大火** | 180-200°C | 爆炒、快炒、油炸 | 油面大量青烟，剧烈滋滋声 |
| **中火** | 140-160°C | 煎、炒、煮、焖 | 油面微微波动，少量青烟 |
| **小火** | 80-100°C | 慢炖、煨、保温 | 油面平静，偶有细小气泡 |
