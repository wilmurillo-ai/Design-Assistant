# Matic-MQuant-Assistant

> **MQuant 策略程序生成 OPENCLAW SKILL**

## 简介

Matic-MQuant-Assistant 是一个为 **Matic MQuant 模块** 编写的策略生成助手SKILL。目标是能够根据用户需求快速生成**可直接运行**的 Mquant Python 策略代码，提升工作效率。

**核心理念：代码可运行 > 完美逻辑**

- 优先成能运行的代码，策略逻辑由用户持续优化；
- 严格遵守 Matic MQuant API 文档中对接口和结构体的定义，避免常见错误；
- 详尽的日志记录，便于问题排查；

## 免责声明

**本 Skill 仅生成代码框架，具体策略逻辑由用户自行实现。**

- 本工具仅提供策略代码的基础模板
- 为方便调试，所有策略参数（买入条件、卖出条件、仓位管理等）有默认值，但最终应由用户自行配置
- 生成的代码应在 Matic 的测试环境中充分验证后再用于实盘
- 使用本工具生成的策略进行交易，风险由用户自行承担
