# custom_docs/ - 用户自定义文档目录

此目录存放用户自定义的文档和模板，Skill 永远不会更新此目录下的任何文件。

## 文档说明

| 文件 | 用途 |
|------|------|
| `strategy_templates.user.md` | 个人策略模板库 - 存放常用的代码框架、公共函数 |
| `api_cookbook.user.md` | API 用法速查 - 常用 API 的示例和注意事项 |
| `deployment_checklist.user.md` | 上线检查清单 - 实盘前的检查项 |
| `debugging_guide.user.md` | 问题排查手册 - 记录遇到的问题和解决方法 |
| `exchange_notes.user.md` | 交易所特性笔记 - 不同交易所的规则差异 |

## 使用建议

- 这些文件完全由你管理，可随时增删改
- Skill 在生成策略时会优先参考这些文档
- 建议定期整理，形成自己的知识体系
