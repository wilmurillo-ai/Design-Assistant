# Matic-MQuant-Assistant

> **MQuant 策略程序生成 OPENCLAW SKILL**

## 简介

Matic-MQuant-Assistant 是一个为 **Matic MQuant 模块** 编写的策略生成助手SKILL。目标是能够根据用户需求快速生成**可直接运行**的 Mquant Python 策略代码，提升效率。

**核心理念：可运行 > 完美**

- 先生成能运行的代码，策略逻辑可以后续优化
- 严格遵守 Matic MQuant API 文档，避免常见错误
- 详尽的日志记录，便于问题排查

## 免责声明

**本 Skill 仅生成代码框架，具体策略逻辑由用户自行实现。**

- 本工具仅提供策略代码的基础框架和模板
- 所有策略参数（买入条件、卖出条件、仓位管理等）由用户自行配置
- 生成的代码需在 Matic 测试环境中充分验证后再用于实盘
- 使用本工具生成的策略进行交易，风险由用户自行承担

## 主要特性

### 快速生成
- 支持 **Python** 策略生成
- 6 步工作流程：目录检测 → 语言选择 → 版本控制 → 参数向导 → 代码生成 → 代码解释
- 基于模板生成，确保代码可直接运行

### 质量保证
- **11 字符文件名限制** - 符合 Matic 平台要求
- 严格遵守 A 股交易规则
- **API 严格核对** - 每个接口和字段都核对文档
- **详尽日志** - 4阶段完整日志（初始化/行情/判断/执行）

### 个人化机制
- Skill 文档与个人笔记分离
- `.user.md` 文件永远不会被更新覆盖
- 记录个人习惯、踩过的坑、调试技巧



## 快速开始

### 1. 首次使用

首次使用时会自动创建个人笔记文件：
- `COMMON_ERRORS.user.md` - 个人错误笔记
- `TRADING_RULES.user.md` - 个人交易规则
- `TRADING_PHILOSOPHY.user.md` - 个人交易理念

### 2. 生成策略

直接告诉助手你想要的策略：

举例：
```
用户：我要一个双均线策略，标的用平安银行
助手：好的，正在生成...
```

### 3. 运行策略

生成的代码保存在 `M-quant/用户名/` 目录下，在 Matic 策略执行监控中加载运行。

| 文件类型 | 更新方式 | 用途 |
