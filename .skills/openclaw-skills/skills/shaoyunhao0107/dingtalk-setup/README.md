# DingTalk Channel Setup

OpenClaw skill for quick DingTalk enterprise bot channel integration.

## Features

- 🚀 Quick setup with interactive configuration
- 🤖 Stream mode (no public IP required)
- 💬 DM and group chat support
- 🎨 AI interactive cards (optional)
- 🛠️ Automated setup script included
- 🇨🇳 China network optimized

## Quick Start

```bash
# Install plugin
openclaw plugins install @soimy/dingtalk

# Configure channel
openclaw configure --section channels

# Restart gateway
openclaw gateway restart
```

## Prerequisites

- OpenClaw installed
- DingTalk Open Platform enterprise internal app created
- Client ID (AppKey) and Client Secret (AppSecret)
