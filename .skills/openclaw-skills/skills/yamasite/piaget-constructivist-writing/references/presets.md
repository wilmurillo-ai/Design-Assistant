# Presets: 场景预设

用途：当用户没有给完整输入时，用预设补齐 Audience/Goal/Constraints，并给出默认结构与成功标准方向。

## 预设 1：科普文章（大众读者）

- Audience：大众；允许少量中学水平背景
- Goal：理解 + 能复述 + 不产生关键误解
- Constraints：少术语；先现象后机制；类比必须写清边界
- 默认结构建议：
  - 现象/问题（锚点）
  - 旧知解释为何不够（冲突）
  - 新概念与机制（支架）
  - 例子/反例（稳固）
  - 误解纠偏 + 自测

## 预设 2：课程讲义/备课（老师）

- Audience：学生（可指定年级与基础）
- Goal：能应用 + 能迁移（会做题/会用方法）
- Constraints：定义清楚；步骤与原因对应；必须有练习与易错点
- 默认结构建议：
  - 旧知回顾（锚点）
  - 反例/难题引入（冲突）
  - 新概念（定义 + 直觉）
  - Worked Example（例题带思路）
  - Faded Guidance（变式题逐步抽提示）
  - 易错点纠偏 + 迁移练习

## 预设 3：学习笔记（自学）

- Audience：自己/同学；希望后续快速复盘
- Goal：能复述 + 能检验理解（自测）
- Constraints：短；层级清楚；保留关键例子与反例
- 默认结构建议：
  - 一句话总结
  - 概念地图（最小集合）
  - 冲突点：我原来以为 X，但解释不了 Y
  - 新模型：如何解释 Y
  - 自测题与错因
