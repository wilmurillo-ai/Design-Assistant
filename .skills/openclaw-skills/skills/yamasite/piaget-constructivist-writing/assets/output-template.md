# 输出模板索引（建构写作）

按目标与时长选择模板：

- `compact-template.md`
  - 适用：短讲解、短科普、知识点解释（读者需要快速理解）
  - 结构：锚点 -> 冲突 -> 支架 -> 迁移练习 -> 纠偏 -> 评估

- `lesson-template.md`
  - 适用：课程讲义/备课（读者需要会做题/会迁移）
  - 结构：旧知回顾 -> 冲突 -> 新概念 -> 例题 -> 变式 -> 易错点 -> 迁移练习 -> 评估

默认映射建议：

- `科普文章` -> `compact-template.md`
- `学习笔记` -> `compact-template.md`
- `课程讲义/备课` -> `lesson-template.md`
