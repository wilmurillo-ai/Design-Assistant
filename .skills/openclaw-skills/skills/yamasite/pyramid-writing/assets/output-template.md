# 输出模板索引（Pyramid Writing）

按场景选择模板，不要默认所有任务都用同一份完整版：

- `compact-template.md`
  - 适用：关键邮件、IM 总结、短周报、状态同步
  - 特点：最短路径传达结论，保留必要 CTA

- `standard-template.md`
  - 适用：常规汇报、说明文、复盘总结、对外技术传播
  - 特点：结构完整，但允许裁剪风险/行动项

- `decision-template.md`
  - 适用：一页纸方案评审、立项/取舍、关键决策说明
  - 特点：强化决策结论、决策表、替代方案与 trade-off

默认映射建议：

- `评审/立项/决策` -> `decision-template.md`
- `汇报/复盘/对外文章` -> `standard-template.md`
- `邮件/IM/短周报` -> `compact-template.md`
