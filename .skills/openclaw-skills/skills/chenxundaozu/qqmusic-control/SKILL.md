# QQ Music Control

## 用法

### 播放/暂停
```
AutoHotkey64.exe scripts/qqmusic_media_keys.ahk 1
```

### 下一首
```
AutoHotkey64.exe scripts/qqmusic_media_keys.ahk 2
```

### 上一首
```
AutoHotkey64.exe scripts/qqmusic_media_keys.ahk 3
```

## 说明
- 依赖 AutoHotkey v2（路径可按需调整）
- QQ音乐需已打开（后台也可）
- 使用系统媒体键，不依赖窗口焦点
