# SEO Strategy Reference

Strategic SEO methodology for affiliate content. Used by S3 Blog & SEO skills.

## Topical Authority Model

Google rewards sites that demonstrate comprehensive expertise on a topic. Random keyword targeting is dead — topical clusters win.

### How Topical Authority Works
1. **Pick a topic** (not a keyword) — e.g., "AI video editing tools"
2. **Map every subtopic** — features, comparisons, tutorials, use cases, pricing, alternatives
3. **Create content for each subtopic** — each piece targets a cluster of related keywords
4. **Interlink everything** — hub page links to all subtopics, subtopics link back and to each other
5. **Result** — Google sees you as THE authority on this topic, ranks you higher for all related queries

### Content Architecture: Hub & Spoke
```
                    HUB PAGE
              "Best AI Video Tools"
              /    |    |    \
        SPOKE   SPOKE  SPOKE  SPOKE
       "HeyGen  "AI    "How   "HeyGen
        Review"  Video   to     vs
                 Pricing" Use   Synthesia"
                         HeyGen"
```

- **Hub page**: Broad, high-volume keyword. Links to all spokes.
- **Spoke pages**: Long-tail, specific keywords. Link back to hub + related spokes.
- **Ratio**: 1 hub per 5-15 spokes. Don't create a hub until you have 5+ spokes.

## Keyword Clustering Methodology

### Step 1: Seed Keywords
Start with 3-5 seed keywords from your niche:
- Product name: "HeyGen"
- Category: "AI video tools"
- Problem: "create videos without camera"

### Step 2: Expand
For each seed, find related keywords by intent:
- **Informational**: "what is [product]", "how does [product] work", "[product] tutorial"
- **Commercial**: "best [category]", "[product] review", "[product] alternatives"
- **Transactional**: "[product] pricing", "[product] free trial", "[product] discount code"
- **Navigational**: "[product] login", "[product] affiliate program"

### Step 3: Cluster
Group keywords that share the same search intent (same SERP = same cluster):
```
Cluster: "HeyGen Review"
  - heygen review
  - heygen review 2024
  - is heygen worth it
  - heygen pros and cons
  → ONE page targets all of these
```

### Step 4: Prioritize
Score each cluster:
- **Search volume**: How many people search this monthly
- **Competition**: How strong are the current top 10 results
- **Commercial intent**: Will this keyword lead to affiliate clicks
- **Your authority**: Do you already rank for related terms

Priority = (Volume × Intent) ÷ Competition

## Content Moat Calculation

A content moat is the minimum content needed to establish topical authority in a niche.

### Formula
```
Content Moat = Average competitor pages × 1.5

Where "competitor" = sites ranking in top 10 for your hub keyword
```

### Assessment Steps
1. Search your hub keyword
2. Analyze top 5 ranking sites
3. Count their pages on this topic (use site:domain.com + topic)
4. Average the count
5. Multiply by 1.5 (you need MORE than them)
6. That's your content moat target

### Go/No-Go Decision
- **< 20 pages needed**: Green light — achievable solo
- **20-50 pages**: Yellow — need consistent output (3-5 months)
- **50-100 pages**: Orange — consider narrowing the niche
- **100+ pages**: Red — too broad, find a sub-niche

## Content Decay Detection

Published content loses rankings over time. Detecting and refreshing decaying content is the highest-ROI SEO activity.

### Decay Signals
1. **Ranking drop**: Position dropped 5+ spots in 30 days
2. **Traffic decline**: 20%+ traffic drop month-over-month
3. **Outdated information**: Product features, pricing, or comparisons are stale
4. **Competitor refresh**: A competitor published a newer, better version
5. **SERP changes**: New SERP features (featured snippets, AI overviews) displaced your result

### Refresh Priority Matrix
| Signal | Priority | Action |
|---|---|---|
| Ranking drop + high traffic | P0 — Critical | Immediate refresh |
| Outdated pricing/features | P1 — High | Update within 1 week |
| Competitor published better version | P1 — High | Rewrite to surpass |
| Traffic declining slowly | P2 — Medium | Schedule refresh |
| Minor ranking fluctuation | P3 — Low | Monitor for 2 weeks |

### Refresh Checklist
- [ ] Update all facts, figures, and dates
- [ ] Add any new product features or pricing changes
- [ ] Improve content depth (add sections competitors have)
- [ ] Update internal links (new related content to link to)
- [ ] Refresh meta title and description
- [ ] Add schema markup if missing
- [ ] Check and fix broken links

## Internal Linking Strategy

### Hub-and-Spoke Linking Rules
1. **Hub → all spokes**: The hub page links to every spoke page
2. **Spoke → hub**: Every spoke links back to the hub (usually in intro or conclusion)
3. **Spoke → related spokes**: Each spoke links to 2-3 related spokes
4. **Contextual anchors**: Use descriptive anchor text, not "click here"
5. **Link depth**: Every page should be reachable within 3 clicks from homepage

### Link Equity Flow
- Pages with the most backlinks should link generously to pages you want to rank
- New pages need internal links from established pages within 48 hours
- Review link structure quarterly — orphan pages (no internal links) get zero link equity

### Anchor Text Rules
- Use the target keyword as anchor text (naturally, not forced)
- Vary anchor text — don't use the same exact anchor for every link to a page
- Avoid over-optimization — keep it natural and helpful to readers
