## 图片视频验证工具

1. **Google Images 反向搜索**：images.google.com
2. **TinEye**：tineye.com
3. **InVID/WeVerify**：验证视频片段
4. **Fotoforensics**：分析图片元数据

## 官方信息查询

- **政府官网**：gov.cn 结尾的官方网站
- **企业官网**：正规企业官方网站
- **学术期刊**：CNKI, PubMed, JSTOR
- **法院判决书**：中国裁判文书网

## 数据核实

- **统计数据**：国家统计局 worldbank.org
- **疫情数据**：WHO, 各国卫生部门
- **气象数据**：各国气象局

## 常见谣言类型

1. **旧闻新炒**：将旧新闻重新发布
2. **移花接木**：图片或视频与文字不匹配
3. **断章取义**：引用部分内容改变原意
4. **伪造来源**：冒充权威媒体或机构
5. **阴谋论**：无法验证的宏大叙事
6. **情绪操控**：利用恐惧、愤怒等情绪

## 核查清单

- [ ] 新闻来源是权威媒体吗？
- [ ] 有其他媒体同步报道吗？
- [ ] 消息源是否明确？
- [ ] 报道时间是最近的吗？
- [ ] 事实有官方确认吗？
- [ ] 图片视频是否真实？
- [ ] 是否有明显的偏见或煽动？
- [ ] 转发者是否核实过？
