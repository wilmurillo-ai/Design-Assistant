# 趋势启动扫描器 (Trend Launch Scanner)

## 简介

基于历史数据验证的趋势跟踪选股系统。通过对比"上涨组"与"对照组"的技术指标差异，找出真正有区分度的信号，用于筛选当前处于上升趋势初期的潜力股。

## 核心理念

**不预测，只跟踪。**

这套系统不试图预测股价涨跌，而是通过历史数据验证，找出"已经启动上涨"的股票在启动前具备哪些共性特征，然后用这些特征实时扫描市场，发现当前处于类似状态的股票。

## 验证结果

基于 117 个上涨样本（之后20日涨幅>15%）与 186 个对照样本（之后20日涨幅<5%）的对比分析：

### 有区分度的信号（启动组 vs 对照组）

| 信号 | 启动组 | 对照组 | 差值 | 权重 |
|------|--------|--------|------|------|
| MACD柱线为正 | 62% | 36% | +26% | 25分 |
| 价格>MA20 | 62% | 40% | +21% | 20分 |
| RSI在上升 | 59% | 40% | +19% | 15分 |
| RSI>50 | 60% | 42% | +18% | 15分 |
| 窗口内价格上升 | 55% | 37% | +18% | 10分 |
| 均线多头排列 | 42% | 26% | +16% | 10分 |
| MACD柱线增大 | 51% | 36% | +15% | 5分 |

### 关键发现

✅ **启动组的特征：**
- MACD柱线为正（最强信号）
- 价格在MA20上方（趋势向上）
- RSI偏强且在上升（动能增强）
- 均线多头排列（趋势确认）

❌ **常见误区（启动组反而更少）：**
- 不是低位蓄势（启动组在布林上半区）
- 不是缩量（启动组量比略高）
- 不是大幅回调（启动组回撤更小）

**核心结论：真正的启动点是"强势延续"，而不是"低位反弹"。**

## 评分标准

满分 100 分，筛选标准 >= 60 分：

| 信号 | 分值 |
|------|------|
| MACD柱线为正 | 25分 |
| 价格>MA20 | 20分 |
| RSI在上升 | 15分 |
| RSI>50 | 15分 |
| 窗口内价格上升 | 10分 |
| 均线多头排列 | 10分 |
| MACD柱线增大 | 5分 |

## 脚本说明

### 核心脚本

| 脚本 | 用途 |
|------|------|
| `trend_scanner.py` | 实时扫描，筛选潜力股 |
| `verify_plan_c_tencent.py` | 验证分析，对比启动组与对照组 |

### 数据文件

| 文件 | 说明 |
|------|------|
| `launch_vs_control_v3.csv` | 验证数据（启动组+对照组） |
| `trend_scan_YYYY-MM-DD.csv` | 每日扫描结果 |

## 使用方法

### 每日扫描

```bash
python scripts/trend_scanner.py
```

输出：
- 扫描统计
- TOP 20 潜力股
- 信号分布
- CSV 文件保存路径

### 重新验证规律

```bash
python scripts/verify_plan_c_tencent.py
```

用于验证评分标准的有效性，生成新的对照组对比数据。

## 注意事项

1. **数据源**：使用腾讯财经 API，前复权日K线
2. **样本范围**：沪深主板 + 创业板（不含科创板、ST股）
3. **更新频率**：建议每日收盘后运行
4. **风险提示**：
   - 高分股票也可能下跌
   - 5日涨幅过大的股票可能短期过热
   - 布林位置接近上轨（>1.0）需警惕回调
