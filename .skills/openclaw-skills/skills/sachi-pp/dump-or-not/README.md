提供对方的原材料（微信聊天记录、短信截图、对话内容记录）加上你发你说不清楚的感觉——

从 7 个维度给你打分，把你的原话逐句解剖，最后告诉你：**分，还是不分。**

随便用，但分手了别来找我负责。没分成的也别来找我——你那段感情我看了，问题不在我。

---

## 能干什么

- 🔨 **直接给结论** — 分还是不分，一句话，有理由
- 📊 **维度评分** — 价值观 / 沟通 / 信任 / 亲密感 / 付出对等…7项逐一打分
- 🔍 **原话解剖** — 把你说的话翻译成它真实的意思
- 💬 **情绪收尾** — 分了怎么走出来，不分怎么有底线地留
- 📸 **输出长图** — 1080px 小红书风格诊断卡，可以截图发给闺蜜

## 特殊场景

- **一票否决**：命中7条红线之一，直接判分，不废话
- **止盈型分手**：关系还 OK 但感觉快到头了？她也能看出来
- **发聊天记录**：截图或文字直接丢，她自己读，然后开始追问

---

## 安装

```bash
clawhub install dump-or-not
```

装完重启一次 OpenClaw 对话就生效。

---

## 怎么触发

不需要说什么特别的话，直接倾诉就行：

> "我男朋友又冷战了，已经第三次了……"

> "我不爱他了但舍不得分，怎么办"

> "帮我看看这段聊天记录"（直接发截图）

> "分还是不分"

她会接住的。
