# Skill Security Guard - AI 技能安全守护者

[![ClawHub Skill](https://img.shields.io/badge/ClawHub-Skill-blue)](https://clawhub.com)
[![Version](https://img.shields.io/badge/version-1.0.1-green)](https://clawhub.com)
[![Security](https://img.shields.io/badge/security-scanner-red)](https://clawhub.com)

**一句话介绍：** 一键扫描第三方 Skill 安全风险，保护你的 AI 助手免受恶意代码侵害。

---

## 为什么需要 Skill Security Guard？

| 风险 | 后果 | Skill Security Guard 保护 |
|------|------|--------------------------|
| ❌ 恶意代码 | AI 助手被控制 | ✅ 检测木马、病毒 |
| ❌ 信息泄露 | 用户数据被盗 | ✅ 检测数据外传 |
| ❌ 危险操作 | 文件被删除 | ✅ 检测危险命令 |
| ❌ 敏感信息 | 密钥泄露 | ✅ 检测硬编码密钥 |

---

## 谁需要使用？

| 用户类型 | 使用场景 | 价值 |
|---------|---------|------|
| **Skill 开发者** | 发布前自检 | 确保自己的 Skill 安全合规 |
| **Skill 使用者** | 安装前检查 | 避免安装恶意 Skill |
| **企业管理员** | 安全审计 | 企业 AI 安全管理 |

## 核心功能

### ✅ 安全检查

| 检查项 | 说明 | 风险等级 |
|--------|------|:--------:|
| **网络请求** | 检测 HTTP 请求、外部连接 | 🟡 中 |
| **文件操作** | 检测删除、写入、敏感文件访问 | 🟠 高 |
| **危险代码** | 检测 exec、eval、os.system | 🔴 高 |
| **敏感信息** | 检测硬编码密钥、密码、Token | 🔴 高 |

### ✅ 风险评分

| 分数 | 等级 | 建议 |
|:----:|:----:|------|
| 0-20 | 🟢 低风险 | 可以使用 |
| 21-50 | 🟡 中等风险 | 审查后使用 |
| 51-80 | 🟠 高风险 | 谨慎使用 |
| 81-100 | 🔴 极高风险 | 不建议使用 |

## 快速开始

### 安装

```bash
# ClawHub 安装（推荐）
clawhub install skill-security-guard

# 或手动安装
git clone <repo-url> skill-security-guard
cd skill-security-guard
pip install -r requirements.txt
```

### 使用

```bash
# 扫描单个 Skill
python scanner.py /path/to/skill

# 扫描当前目录
python scanner.py .
```

## 使用案例

### 案例 1：安装第三方 Skill 前

**场景：** 想安装一个来自社区的 Skill，担心安全问题

**步骤：**
```bash
# 1. 先扫描
python scanner.py ./community-skill

# 2. 查看报告
# ============================================================
# Skill 安全扫描报告
# ============================================================
# 总体风险评分: 15/100
# 总体风险等级: 低风险
# 发现的安全问题数量: 1
# 
# 详细发现:
# 1. 文件: skill/main.py
#    类型: 网络请求
#    风险等级: medium
#    描述: 发现网络请求代码
# ============================================================

# 3. 根据报告决定是否安装
```

**效果：** 避免安装高风险 Skill

---

### 案例 2：发布自己的 Skill 前

**场景：** 开发完成准备发布，先自检

**步骤：**
```bash
# 1. 扫描自己的 Skill
python scanner.py ./my-skill

# 2. 根据报告修复问题
# 3. 再次扫描确认安全
# 4. 发布到 ClawHub
```

**效果：** 发布安全合规的 Skill

---

### 案例 3：企业安全管理

**场景：** 企业需要审核员工使用的 Skill

**步骤：**
```bash
# 1. 批量扫描所有 Skill
python scanner.py /path/to/skills/

# 2. 生成报告
# 3. 根据企业安全策略审核
```

**效果：** 企业 AI 安全可控

## 常见问题

### Q1: 扫描会执行代码吗？

**回答：** 不会。本工具采用静态分析，只扫描代码文本，不执行任何代码。

### Q2: 扫描结果一定准确吗？

**回答：** 可能存在误报。建议结合实际情况判断，如有疑问可以查看详细报告。

### Q3: 发现高风险问题怎么办？

**回答：** 
- 如果是自己开发的 Skill：修复问题后重新扫描
- 如果是第三方 Skill：建议不安装或联系开发者

### Q4: 支持哪些编程语言？

**回答：** 目前支持 Python 编写的 Skill。
