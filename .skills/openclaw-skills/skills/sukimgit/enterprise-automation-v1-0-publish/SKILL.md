## 💡 核心功能

### 1. 智能文件搜索
- ✅ 支持中文路径（完美解决编码问题）
- ✅ 精准匹配 + 模糊匹配
- ✅ 按文件类型过滤（.pdf/.docx/.xlsx 等）
- ✅ 显示文件大小和修改时间
- ✅ 支持批量搜索

**示例：**
```bash
# 搜索某某科技公司的合同
python tools/file_searcher.py "D:\工作" "某某科技公司" .pdf

# 搜索所有合同文件
python tools/file_searcher.py "D:\工作" "合同"
```

### 2. 邮件自动发送
- ✅ 支持附件（单个/批量）
- ✅ 邮件模板（合同/报价/跟进等）
- ✅ 自动确认流程（发送前飞书确认）
- ✅ 发送状态追踪
- ✅ 支持 QQ 邮箱/163 邮箱/Gmail

**示例：**
```bash
# 发送文件到邮箱
python tools/file_sender.py "D:\工作" "合同" .pdf "client@example.com"
```

### 3. 飞书实时通知
- ✅ 任务完成通知
- ✅ 文件发送确认
- ✅ 重要事项提醒
- ✅ 自定义 Webhook
- ✅ 支持@指定人员

**通知示例：**
```

大小：111 KB
收件人：client@example.com
状态：已发送
```

### 4. 一体化工作流
- ✅ 文件搜索 → 飞书确认 → 邮件发送 → 完成通知
- ✅ 全自动流程，无需人工干预
- ✅ 支持自定义工作流
- ✅ 日志记录，便于追溯
