## 📝 功能需求

### 1. 邮箱连接（P0）

**功能描述：**
- 支持 QQ 邮箱/163 邮箱/Gmail
- IMAP 协议接收邮件
- SMTP 协议发送邮件
- SSL 加密连接

**技术要求：**
```python
# IMAP 配置
imap_host = "imap.qq.com"
imap_port = 993
imap_ssl = True

# SMTP 配置
smtp_host = "smtp.qq.com"
smtp_port = 465
smtp_ssl = True
```

**验收标准：**
- [ ] 能成功连接邮箱
- [ ] 能读取收件箱邮件
- [ ] 能发送邮件

---

### 2. 商机识别（P0）

**功能描述：**
- 根据关键词识别商机邮件
- 分类：优先/咨询/验证/垃圾
- 不同类别不同处理方式

**关键词配置：**

| 类别 | 关键词 | 处理方式 |
|------|--------|---------|
| **优先（商机）** | 订单/付款/定制/开发/报价/价格/购买/order/payment/custom/develop/quote/price/buy | 回复 + 飞书通知 |
| **咨询** | 咨询/询问/了解/详情/功能/技能/服务/consult/inquiry/question | 回复服务信息 |
| **验证** | 验证/确认/激活/注册/登录/验证码/verify/confirm/activate/code | 忽略 |
| **垃圾** | 发票/广告/促销/优惠/抽奖/赌博/贷款/spam/promotion | 标记为垃圾 |

**验收标准：**
- [ ] 能准确识别商机邮件
- [ ] 能正确分类邮件
- [ ] 不同类别处理方式正确

---

### 3. 自动回复（P0）

**功能描述：**
- 中英双语回复模板
- 包含服务内容和价格
- 包含支付方式
- 发送频次控制

**回复模板：**
```
【中文】
您好！感谢联系 OpenClaw 技能开发服务。

【服务内容】
- OpenClaw 技能定制开发：5000-20000 元/个
- AI 自动化系统集成：10000-50000 元/项目
- 企业培训/技术咨询：20000 元/场

【支付方式】
- 支付宝：your_email@example.com（某用户）
- PayPal：私信获取账号
- Wise：Account 242009405

【English】
Hello! Thank you for contacting OpenClaw skill development services.

【Services】
- Custom Skill Development: 5000-20000 CNY
- AI Automation Integration: 10000-50000 CNY

【Payment】
- Alipay: your_email@example.com
- PayPal: DM for account

Best regards,
客户 | OpenClaw Team
```

**频次控制：**
- 同一发件人 24 小时内只回复 1 次
- 每小时最多回复 5 封
- 每天最多回复 20 封
- 免打扰时段：23:00-07:00

**验收标准：**
- [ ] 回复模板正确
- [ ] 频次控制有效
- [ ] 免打扰时段不回复

### 5. 垃圾过滤（P1）

**功能描述：**
- 根据关键词过滤垃圾邮件
- 根据发件人黑名单过滤
- 自动标记为垃圾邮件

**黑名单配置：**
```json
{
  "ignore_keywords": ["发票", "广告", "促销", "spam"],
  "ignore_senders": ["10000@qq.com", "no-reply@"]
}
```

**验收标准：**
- [ ] 垃圾邮件正确识别
- [ ] 自动标记为垃圾
- [ ] 不触发回复
