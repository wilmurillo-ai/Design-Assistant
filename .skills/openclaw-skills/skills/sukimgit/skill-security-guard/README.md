# Skill 安全扫描器

**版本：** v1.0.0
**作者：** Monet
**创建时间：** 2026-03-18

---

## 功能

扫描第三方 Skill 的安全性，检测：
- 🔴 恶意代码（木马、病毒）
- 🔴 信息泄露（发送用户数据）
- 🟠 危险操作（删除文件、执行命令）
- 🟡 敏感信息（硬编码密钥）

---

## 使用方法

```bash
python scanner.py <skill_directory>
```

---

## 检查项

| 检查项 | 说明 |
|--------|------|
| **网络请求** | 检测 HTTP 请求、外部连接 |
| **文件操作** | 检测删除、写入、敏感文件访问 |
| **危险代码** | 检测 exec、eval、os.system |
| **敏感信息** | 检测硬编码密钥、密码、Token |

## 风险等级

| 分数 | 等级 | 建议 |
|------|------|------|
| 0-20 | 🟢 低风险 | 可以使用 |
| 21-50 | 🟡 中等风险 | 审查后使用 |
| 51-80 | 🟠 高风险 | 谨慎使用 |
| 81-100 | 🔴 极高风险 | 不建议使用 |

---

## 注意事项

- 本工具采用静态分析，不执行代码
- 可能存在误报，请结合实际情况判断
- 建议：先扫描，再审查，最后使用
