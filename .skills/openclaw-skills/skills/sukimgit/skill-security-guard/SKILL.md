## 功能

扫描第三方 Skill 的安全性，检测恶意代码、信息泄露等风险。

---

## 安装

从 ClawHub 下载安装

**依赖：**
- Python 3.7+
- 无需额外依赖（纯标准库）

---

## 使用方法

```bash
# 扫描单个 skill
python scanner.py <skill_directory>

# 示例
python scanner.py ../email-monitor
```

---

## 检查项

| 检查项 | 说明 |
|--------|------|
| **网络请求检测** | 检测 HTTP 请求、外部连接 |
| **文件操作检测** | 检测删除、写入、敏感文件访问 |
| **危险代码检测** | 检测 exec、eval、os.system |
| **敏感信息检测** | 检测硬编码密钥、密码、Token |

## 风险等级

| 分数 | 等级 | 建议 |
|------|------|------|
| >= 80 | 🟢 安全 | 可以使用 |
| 50-79 | 🟡 需审核 | 审查后使用 |
| < 50 | 🔴 高风险 | 不建议使用 |

---

## 注意事项

- 本工具采用**静态分析**，不执行代码
- 检测**危险操作**，不是 import 语句
- **零误报**设计（基于 DeepSeek 方案）
