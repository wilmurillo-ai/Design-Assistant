# 湖南凤凰古城 (Fenghuang Ancient Town)

> **📅 数据最后更新**: 2026-03

## 基本信息

| 项目 | 详情 |
|------|------|
| 位置 | 湖南省湘西土家族苗族自治州凤凰县 |
| 开放时间 | 全天开放 |
| 建议游览时长 | 1-2 天 |
| 门票 | 古城免费，部分景点收费 |
| 最佳季节 | 春秋 (3-5 月，9-11 月) |
| 交通 | 吉首乘大巴/高铁 |

## 主要景点

- 沱江：穿城而过
- 吊脚楼：湘西特色建筑
- 虹桥：古城标志
- 沈从文故居：文学圣地
- 东门城楼：古城门

## 游览路线

```
东门城楼 → 沱江 → 虹桥 → 
吊脚楼 → 沈从文故居 → 夜景
```

## 历史文化

**建造历史**：
- 唐代：开始建城
- 明清：鼎盛时期
- 2001 年：国家历史文化名城

**沈从文**：
著名作家，《边城》作者，凤凰人。

**冷知识**：
- 历史：1300 多年
- 民族：苗族/土家族为主
- 特色：吊脚楼建筑
- 夜景：沱江夜景最美
