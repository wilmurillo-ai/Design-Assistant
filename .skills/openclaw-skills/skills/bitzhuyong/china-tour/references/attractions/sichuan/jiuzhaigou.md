## 推荐路线

### 经典 1 日游 ⭐ 推荐

```
入口 → 观光车 → 诺日朗中心 → 
日则沟 (箭竹海→熊猫海→五花海→珍珠滩) → 
诺日朗中心 → 则查洼沟 (长海→五彩池) → 
树正沟 (火花海→树正群海) → 出口

特点：
• 游览时间：8-10 小时
• 步行距离：约 15km
• 难度：中等
• 适合：第一次游览
```

### 深度 2 日游 ⭐⭐⭐

**Day 1**:
```
入口 → 诺日朗中心 → 日则沟全天
(箭竹海→熊猫海→五花海→珍珠滩→镜海)

特点：
• 游览时间：6-8 小时
• 步行距离：约 10km
• 难度：低
• 适合：深度游览日则沟
```

**Day 2**:
```
入口 → 诺日朗中心 → 则查洼沟 → 
树正沟全天 (长海→五彩池→火花海→树正群海)

特点：
• 游览时间：6-8 小时
• 步行距离：约 10km
• 难度：低
• 适合：深度游览剩余景点
```

## 冷门知识点

1. **世界遗产**：1992 年列入世界自然遗产
2. **名称由来**：因沟内有 9 个藏族寨子得名
3. **海子数量**：108 个海子
4. **水质**：国家一类水质，可直接饮用
5. **彩林**：秋季树叶变色，形成五彩斑斓景观
6. **钙化瀑布**：诺日朗瀑布是中国最宽钙化瀑布
7. **海拔**：2000-3100 米
8. **保护**：2017 年地震后修复，2019 年重新开放

## 游览贴士

### 必带物品
- ✅ 身份证 (入园验证)
- ✅ 舒适运动鞋 (步行多)
- ✅ 保暖外套 (高原温差大)
- ✅ 防晒用品 (高原紫外线强)
- ✅ 水和干粮

### 选带物品
- ⭐ 氧气瓶 (预防高反)
- ⭐ 相机 (长焦拍远景)
- ⭐ 雨衣 (天气多变)

### 注意事项
- ⚠️ 高原地区，不要剧烈运动
- ⚠️ 禁止吸烟 (保护森林)
- ⚠️ 不要投喂野生动物
- ⚠️ 不要踩踏栈道外区域
- ⚠️ 看好小孩 (水边危险)
