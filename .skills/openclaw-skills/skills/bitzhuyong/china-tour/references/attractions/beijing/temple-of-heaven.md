# 天坛 (Temple of Heaven)

> **📅 数据最后更新**: 2026-03

## 基本信息

| 项目 | 详情 |
|------|------|
| 位置 | 北京市东城区天坛内街 1 号 |
| 开放时间 | 08:00-17:30 (旺季 4-10 月)<br>08:00-17:00 (淡季 11-3 月) |
| 建议游览时长 | 2-3 小时 |
| 门票 | 旺季 15 元，淡季 10 元 (联票 35 元) |
| 最佳季节 | 春秋 (3-5 月，9-11 月) |
| 官网 | http://www.tiantanpark.com |
| 交通 | 地铁 5 号线天坛东门站 |

## 推荐路线

### 经典游 (2-3 小时)
```
南门 → 圜丘坛 → 皇穹宇 → 回音壁 → 
丹陛桥 → 祈年殿 → 北门

特点：
• 不走回头路
• 覆盖所有核心景点
• 2-3 小时完成
```

---

## 服务设施

- **休息区**：各景点附近均有
- **卫生间**：南门/北门/祈年殿附近
- **餐饮**：南门/北门有小卖部
- **交通**：建议南门进北门出
