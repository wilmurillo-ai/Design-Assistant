# 北京圆明园 (Old Summer Palace)

> **📅 数据最后更新**: 2026-03

## 基本信息

| 项目 | 详情 |
|------|------|
| 位置 | 北京市海淀区清华西路 28 号 |
| 开放时间 | 07:00-19:00 (旺季 4-10 月)<br>08:00-17:30 (淡季 11-3 月) |
| 建议游览时长 | 2-3 小时 |
| 门票 | 10 元 (通票 25 元) |
| 最佳季节 | 春秋 (3-5 月，9-11 月) |
| 官网 | http://www.yuanmingyuan.gov.cn |
| 交通 | 地铁 4 号线圆明园站 |

## 主要景点

| 景点 | 亮点 |
|------|------|
| 大水法 | 西洋楼遗址标志 |
| 海晏堂 | 十二生肖兽首 |
| 远瀛观 | 西洋建筑精华 |
| 福海 | 最大湖面 |
| 长春园 | 园中园 |

## 游览路线

```
南门进 → 西洋楼遗址 (大水法/海晏堂) → 
福海 → 长春园 → 绮春园 → 东门出
```
