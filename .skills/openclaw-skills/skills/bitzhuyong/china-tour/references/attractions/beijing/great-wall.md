## 推荐路线

### 路线 1：精华游 (2-3 小时) ⭐ 推荐

```
关城 → 北一楼 → 北二楼 → 北三楼 → 北四楼 (好汉坡) → 原路返回

特点：
• 游览时间：2-3 小时
• 步行距离：约 3km
• 难度：中等
• 适合：大部分游客

亮点：
• 打卡"好汉坡"
• 北四楼视野最佳
• 不用爬完全程，节省体力
```

### 路线 2：全程游 (4-5 小时) ⭐⭐⭐

```
关城 → 北一楼 → ... → 北八楼 (最高点) → 原路返回 或 南线下

特点：
• 游览时间：4-5 小时
• 步行距离：约 8km
• 难度：较高
• 适合：体力好的游客

亮点：
• 登顶北八楼 (888m)
• 俯瞰长城全貌
• 完整体验长城
```

### 路线 3：轻松游 (1-2 小时) ⭐⭐

```
关城 → 北一楼 → 北二楼 → 原路返回

特点：
• 游览时间：1-2 小时
• 步行距离：约 1.5km
• 难度：低
• 适合：老人/小孩/体力一般

亮点：
• 轻松游览
• 体验长城氛围
• 不用爬太高
```

### 路线 4：南线游 (2-3 小时)

```
关城 → 南一楼 → 南二楼 → 南三楼 → 南四楼 → 原路返回

特点：
• 游览时间：2-3 小时
• 步行距离：约 3km
• 难度：中等
• 适合：避开人流

亮点：
• 游客较少
• 安静拍照
• 南线风光
```

## 冷门知识点

1. **八达岭名称由来**：居庸关外有四个山口，此为第八个，故名"八达岭"
2. **建造时间**：明弘治十八年 (1505 年) 开始修建，距今 500 多年
3. **墙体高度**：平均高 7.8 米，顶部宽 5.8 米，可容五马并骑
4. **敌楼作用**：驻兵、储存武器、瞭望、射击
5. **烽火台**：古代通讯系统，白天燃烟，夜晚举火
6. **修复历史**：1957 年首次修复，1961 年列为全国重点文物保护单位
7. **接待记录**：已接待 5 位美国总统、40 多位国家元首
8. **最长一天**：2019 年国庆，单日接待 7.4 万人次

## 游览贴士

### 必带物品
- ✅ 身份证 (预约入园)
- ✅ 舒适运动鞋 (石板路滑)
- ✅ 水和干粮 (景区内贵)
- ✅ 防晒用品 (帽子/防晒霜)
- ✅ 充电宝 (拍照耗电)

### 选带物品
- ⭐ 登山杖 (省力)
- ⭐ 手套 (扶城墙用)
- ⭐ 相机 (广角镜头)
- ⭐ 薄外套 (山顶风大)

### 注意事项
- ⚠️ 不要翻越城墙 (危险且违法)
- ⚠️ 不要刻字涂鸦 (文物保护)
- ⚠️ 注意防滑 (石板路滑)
- ⚠️ 量力而行 (不要硬撑)
- ⚠️ 看好小孩 (城墙边缘危险)
