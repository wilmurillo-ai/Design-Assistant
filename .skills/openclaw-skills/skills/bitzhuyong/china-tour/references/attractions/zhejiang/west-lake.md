## 推荐路线

### 路线 1：经典环湖游 (5-6 小时) ⭐ 推荐

```
断桥残雪 → 白堤 → 平湖秋月 → 孤山 → 
苏堤春晓 → 花港观鱼 → 乘船 → 三潭印月 → 
雷峰塔 → 净慈寺

特点：
• 游览时间：5-6 小时
• 步行距离：约 8km
• 难度：中等
• 适合：第一次游览

亮点：
• 覆盖西湖十景
• 乘船游湖体验
• 拍照机位齐全
```

### 路线 2：轻松半日游 (3 小时) ⭐⭐

```
断桥残雪 → 白堤 → 平湖秋月 → 孤山 → 
乘船 → 三潭印月 → 返回

特点：
• 游览时间：3 小时
• 步行距离：约 4km
• 难度：低
• 适合：老人/小孩/时间紧

亮点：
• 核心景点
• 轻松不累
• 乘船体验
```

### 路线 3：深度一日游 (8 小时) ⭐⭐⭐

```
断桥残雪 → 白堤 → 平湖秋月 → 孤山 → 
苏堤春晓 → 花港观鱼 → 乘船 → 三潭印月 → 
雷峰塔 → 净慈寺 → 灵隐寺

特点：
• 游览时间：8 小时
• 步行距离：约 12km
• 难度：较高
• 适合：深度游览

亮点：
• 包含灵隐寺
• 完整体验西湖文化
• 佛教文化体验
```

## 服务设施

### 休息区
- 白堤沿线有多处座椅
- 孤山公园有休息区
- 苏堤沿线有亭子可休息

- 断桥附近
- 平湖秋月附近
- 苏堤南端
- 雷峰塔附近
- 湖滨路沿线

### 餐饮
- **湖滨路**：
  - 外婆家 (人均 80 元)
  - 绿茶餐厅 (人均 70 元)
  - 知味观 (人均 50 元)
- **景区内**：
  - 小卖部 (水 5 元，饮料 8-15 元)
  - 茶室 (龙井茶 50-100 元/杯)
- 建议：湖滨路用餐，选择多

- **类型**：画舫船/手划船/自划船
- **价格**：
  - 画舫船：55 元 (含三潭印月)
  - 手划船：150 元/小时
  - 自划船：100 元/小时
- **码头**：
  - 中山公园码头
  - 花港观鱼码头
  - 湖滨码头

## 冷门知识点

1. **西湖面积**：6.39 平方公里，周长 15 公里
2. **白堤非白居易修建**：原名白沙堤，后人误传
3. **苏堤是苏轼修建**：苏轼疏浚西湖，用淤泥建堤
4. **雷峰塔历史**：原塔 1924 年倒塌，2002 年重建
5. **三潭印月**：一元人民币背景图案
6. **西湖十景**：南宋时期命名，距今 800 多年
7. **世界遗产**：2011 年列入世界文化遗产
8. **免费开放**：2002 年起免费，是国内首个免费 5A 景区

## 游览贴士

### 必带物品
- ✅ 身份证 (部分景点需要)
- ✅ 舒适运动鞋 (步行多)
- ✅ 水和干粮
- ✅ 防晒用品
- ✅ 充电宝

### 选带物品
- ⭐ 汉服 (拍照好看)
- ⭐ 雨伞 (杭州多雨)
- ⭐ 相机 (长焦拍远景)
- ⭐ 自行车 (环湖骑行)

### 注意事项
- ⚠️ 不要踩踏草坪
- ⚠️ 不要投喂鱼类
- ⚠️ 不要攀爬树木
- ⚠️ 小心黑车/黑导游
- ⚠️ 看好小孩 (水边危险)

### 避坑指南
- ❌ 不要在景区买贵重茶叶
- ❌ 不要相信"低价游"
- ❌ 不要在湖边大排档用餐
- ✅ 龙井茶去正规茶庄购买
- ✅ 用餐去湖滨路
- ✅ 乘船去正规码头
