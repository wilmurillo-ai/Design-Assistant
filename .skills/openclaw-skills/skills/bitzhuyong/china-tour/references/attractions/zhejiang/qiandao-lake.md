## 游览路线

**1 日游**：
```
游客中心 → 乘船 → 梅峰岛 → 月光岛 → 
龙山岛 → 返回
```

**2 日游**：
```
Day1: 中心湖区（梅峰岛/月光岛/龙山岛）
Day2: 森林氧吧/骑行环湖
```

## 历史文化

**形成历史**：
- 1959 年：新安江水电站建成
- 蓄水形成：1078 个岛屿
- 面积：573 平方公里
- 水质：国家一类水质

**水下古城**：
- 狮城：唐代建造，距今 1300 年
- 贺城：汉代建造
- 现淹没水下 30-40 米

**冷知识**：
- 岛屿数量：1078 个
- 湖面积：573 平方公里
- 水质：可直接饮用
- 千岛湖啤酒：当地特产
