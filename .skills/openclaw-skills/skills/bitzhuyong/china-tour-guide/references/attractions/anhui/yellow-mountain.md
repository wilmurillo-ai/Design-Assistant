## 推荐路线

### 经典 2 日游 ⭐ 推荐

**Day 1**:
```
云谷寺 → 云谷索道 → 始信峰 → 
北海宾馆 (入住) → 狮子峰 → 
丹霞峰 (看日落)

特点：
• 游览时间：6-8 小时
• 步行距离：约 10km
• 难度：中等
• 适合：第一次游览
```

**Day 2**:
```
北海 → 光明顶 → 鳌鱼峰 → 
莲花峰 → 迎客松 → 玉屏索道下山

特点：
• 游览时间：4-6 小时
• 步行距离：约 8km
• 难度：中等
• 适合：轻松下山
```

### 快速 1 日游 ⭐⭐

```
云谷索道 → 始信峰 → 北海 → 
光明顶 → 迎客松 → 玉屏索道下山

特点：
• 游览时间：6-8 小时
• 步行距离：约 12km
• 难度：较高
• 适合：时间紧
```

## 冷门知识点

1. **世界遗产**：1990 年列入世界文化与自然双重遗产
2. **黄山四绝**：奇松、怪石、云海、温泉
3. **最高峰**：莲花峰，海拔 1864 米
4. **迎客松**：树龄 800 多年，黄山标志
5. **徐霞客**：「五岳归来不看山，黄山归来不看岳」
6. **云海概率**：冬季最高 (约 50%)
7. **峰林数量**：72 峰，36 大峰，36 小峰
8. **摩崖石刻**：200 多处，历代名人题刻

## 游览贴士

### 必带物品
- ✅ 身份证 (入园验证)
- ✅ 舒适运动鞋 (徒步多)
- ✅ 雨具 (山区天气多变)
- ✅ 保暖外套 (山顶温度低)
- ✅ 水和干粮 (山顶贵)
- ✅ 登山杖 (省力)

### 选带物品
- ⭐ 手套 (爬天都峰用)
- ⭐ 相机 (长焦拍远景)
- ⭐ 晕车药 (山路弯多)
- ⭐ 手电筒 (看日出用)

### 注意事项
- ⚠️ 山区天气多变，带雨具
- ⚠️ 山顶温度低，带外套
- ⚠️ 不要攀爬危险区域
- ⚠️ 看好小孩 (悬崖边危险)
- ⚠️ 猴子多，不要喂食
- ⚠️ 天都峰/莲花峰轮休开放
