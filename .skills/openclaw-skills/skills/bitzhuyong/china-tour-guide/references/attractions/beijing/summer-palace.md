## 推荐路线

### 路线 1：经典游 (4 小时) ⭐ 推荐

```
东宫门 → 仁寿殿 → 乐寿堂 → 长廊 → 排云殿 → 
佛香阁 → 石舫 → 十七孔桥 → 新建宫门出

特点：
• 游览时间：4 小时
• 步行距离：约 5km
• 难度：中等
• 适合：第一次游览

亮点：
• 覆盖所有核心景点
• 体验皇家园林精华
• 拍照机位齐全
```

### 路线 2：全景游 (6 小时) ⭐⭐⭐

```
北宫门 → 苏州街 → 四大部洲 → 佛香阁 → 
排云殿 → 长廊 → 石舫 → 西堤 → 
十七孔桥 → 南湖岛 → 新建宫门出

特点：
• 游览时间：6 小时
• 步行距离：约 8km
• 难度：较高
• 适合：深度游览

亮点：
• 前后山全覆盖
• 昆明湖环游
• 完整体验
```

### 路线 3：轻松游 (2 小时) ⭐⭐

```
东宫门 → 仁寿殿 → 长廊 → 石舫 → 原路返回

特点：
• 游览时间：2 小时
• 步行距离：约 2km
• 难度：低
• 适合：老人/小孩/时间紧

亮点：
• 轻松游览
• 核心景点
• 不用爬山
```

## 服务设施

### 休息区
- 仁寿殿前广场
- 长廊沿线有多处座椅
- 佛香阁平台
- 十七孔桥东岸

### 卫生间
- 东宫门入口处
- 仁寿殿附近
- 长廊中段
- 十七孔桥东岸
- 北宫门入口

### 餐饮
- **听鹂馆**：宫廷菜 (人均 200-500 元)
- **颐和园餐厅**：中式快餐 (人均 50-80 元)
- **小卖部**：水 5 元，饮料 8-15 元
- 建议：自备水和干粮

### 游船
- **类型**：画舫船、脚踏船、电瓶船
- **价格**：
  - 画舫船：单程 30 元
  - 脚踏船：100 元/小时
  - 电瓶船：200 元/小时
- **码头**：
  - 排云殿码头
  - 十七孔桥码头
  - 石舫码头

## 拍照注意事项

✅ **允许拍照**：
- 室外建筑、长廊、湖景
- 大部分区域可拍照

❌ **禁止拍照**：
- 部分室内展馆 (有标识)
- 文物特展

💡 **拍照建议**：
- 最佳光线：上午 8-10 点，下午 3-5 点
- 避开人流：开门第一时间或下午 4 点后
- 穿着建议：鲜艳衣服与园林对比强烈
