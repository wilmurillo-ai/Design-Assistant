# 北京恭王府 (Prince Gong's Mansion)

> **📅 数据最后更新**: 2026-03

## 基本信息

| 项目 | 详情 |
|------|------|
| 位置 | 北京市西城区前海西街 17 号 |
| 开放时间 | 08:30-17:00 (周一闭馆) |
| 门票 | 40 元 |
| 游览时长 | 2-3 小时 |
| 亮点 | 清代王府/和珅府邸/福字碑 |

## 主要景点

- 银安殿：王府正殿
- 嘉乐堂：和珅居室
- 福字碑：康熙御笔福字
- 大戏楼：清代戏楼
- 花园：王府花园

## 游览路线

```
南门进 → 银安殿 → 嘉乐堂 → 福字碑 → 
大戏楼 → 花园 → 北门出
```
