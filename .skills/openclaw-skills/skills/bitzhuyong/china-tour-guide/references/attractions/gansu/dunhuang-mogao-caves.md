## 推荐路线

### 经典 A 类票游 (3-4 小时) ⭐ 推荐

```
数字展示中心 (30 分钟电影) → 
摆渡车 (15 分钟) → 
洞窟参观 (8 个洞窟，2 小时) → 
藏经洞陈列馆 (1 小时) → 返程

特点：
• 游览时间：3-4 小时
• 步行距离：约 3km
• 难度：低
• 适合：第一次游览
```

### 深度 B 类票游 (2-3 小时) ⭐⭐

```
摆渡车 (15 分钟) → 
洞窟参观 (4 个洞窟，1.5 小时) → 
藏经洞陈列馆 (30 分钟) → 返程

特点：
• 游览时间：2-3 小时
• 步行距离：约 2km
• 难度：低
• 适合：时间紧
```

### 特窟深度游 (5-6 小时) ⭐⭐⭐

```
数字展示中心 → 摆渡车 → 
普通洞窟 (8 个) → 
特窟 (2-3 个，另付费) → 
陈列馆 → 返程

特点：
• 游览时间：5-6 小时
• 步行距离：约 4km
• 难度：中等
• 适合：深度游览
```

- 九层楼外观

- 陈列馆 (部分)

## 冷门知识点

1. **世界遗产**：1987 年列入世界文化遗产
2. **历史**：始建于前秦建元二年 (366 年)，距今 1600 多年
3. **洞窟数量**：735 个洞窟，4.5 万平方米壁画，2000 多身彩塑
4. **藏经洞**：1900 年发现，出土文物 5 万多件
5. **九层楼**：高 45 米，莫高窟标志性建筑
6. **保护等级**：中国首批全国重点文物保护单位
7. **数字敦煌**：已完成 200+ 个洞窟数字化
8. **特窟**：部分精品洞窟需另付费参观

## 游览贴士

### 必带物品
- ✅ 身份证 (入园验证)
- ✅ 预约码 (电子/纸质)
- ✅ 舒适运动鞋 (步行多)
- ✅ 防晒用品 (沙漠紫外线强)
- ✅ 水和干粮

### 选带物品
- ⭐ 手电筒 (洞窟内光线暗，导游会用)
- ⭐ 相机 (只能拍外观)
- ⭐ 外套 (洞窟内温度低)

### 注意事项
- ⚠️ 洞窟内禁止拍照 (保护文物)
- ⚠️ 不要触摸壁画/彩塑
- ⚠️ 不要大声喧哗
- ⚠️ 听从导游安排
- ⚠️ 特窟需另付费
