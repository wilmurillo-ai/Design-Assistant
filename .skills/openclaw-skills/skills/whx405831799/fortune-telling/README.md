# Fortune Telling 算命 Skill 🔮

基于中国周易的运势预测 skill，根据出生年月日测算今天、本周、本月的运势。

## 特性

- 🌏 **双语支持**：自动识别中英文，返回对应语言的结果
- ☯️ **周易八卦**：基于传统周易六十四卦进行推算
- 🔢 **五行分析**：根据出生日期分析五行属性
- 📊 **多维度运势**：整体、事业、财运、感情、健康五大维度
- 📅 **多时间范围**：支持今天、本周、本月运势

## 安装

```bash
clawhub install fortune-telling
```

## 使用方法

### 中文

```
算命，我是1990年5月15日出生的
```

```
帮我看看本周运势，生日是1988年农历三月初八
```

### English

```
Read my fortune. I was born on May 15, 1990.
```

```
What's my weekly fortune? Birthday: March 8, 1988.
```

## 算命原理

### 周易八卦

- 乾（天）、兑（泽）、离（火）、震（雷）
- 巽（风）、坎（水）、艮（山）、坤（地）

### 五行

- 金、木、水、火、土
- 相生：金生水、水生木、木生火、火生土、土生金
- 相克：金克木、木克土、土克水、水克火、火克金
