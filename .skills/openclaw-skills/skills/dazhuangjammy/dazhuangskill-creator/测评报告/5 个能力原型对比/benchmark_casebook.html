Method Reminder

为什么这里的对比是能成立的

因为每个版本拿到的是同一个 brief、同一个 task、同一个代表性 query、同一个评分逻辑。这里不是比较文采，而是比较在同样任务压力下，哪个 creator 生成的 skill 更能把模型锁进正确轨道。

  • 同一 brief JSON：任务描述、artifact hint、downstream tasks、trigger query 全部相同。
  • 同一 Codex / 同一模型 / 同一机器 / 同一运行时。
  • 每个 task 看 3 次重复平均，不把一次偶然波动当结论。
  • 页面里展示的是 repeat 1 输出，但分数显示的是 repeat 平均值。
