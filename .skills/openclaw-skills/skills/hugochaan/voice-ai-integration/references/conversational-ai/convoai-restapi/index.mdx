本文介绍声网对话式智能体服务 RESTful API 的基本信息。

## 鉴权

发送 HTTP 请求时，你需要通过 `Authorization` 字段传入鉴权信息，支持以下两种方式（任选其一）：

1. **使用 RTC Token**：将用于声网对话式 AI 引擎项目使用的 RTC Token 填入 `Authorization` 字段。
   - 测试环境：从[声网控制台](https://console.shengwang.cn/)生成临时 Token（有效期 24 小时）
   - 生产环境：部署 [token-server](../../token-server/README.md) 生成 Token
   - 传参示例：`Authorization: agora token="007abcxxxxxxx123"`

2. **使用 Basic Auth**：参考[实现 HTTP 安全认证](https://doc.shengwang.cn/doc/convoai/restful/user-guides/http-basic-auth)生成 Base64 编码的 `Authorization` 字段。
   - 传参示例：`Authorization: Basic NDI1OTQ3N2I4MzYy...YwZjA=`

## 数据格式

所有 HTTP 请求头部的 Content-Type 类型为 `application/json`。所有请求和响应内容的格式均为 JSON。
