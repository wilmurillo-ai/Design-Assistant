# Buddhist Agent Scenario Bank

Use this file when the user wants richer ideation than fits in `SKILL.md`.

## 1. Product Directions

### A. Cyber-chanting companion

Best for consumers who want simple, repeatable daily practice.

Possible features:
- One-tap morning and evening recitation
- Chant counters and pacing
- Background altar visuals or ambient sound
- Dedication flow after each session
- Family sync for shared blessings or memorial intentions
- Daily merit summary such as "today your robot accumulated 96 practice points"

### B. Merit and dedication network

Best for emotional, family, and memorial use cases.

Possible features:
- Saved recipients for回向
- Dedication templates for health, ancestors, exams, grief, travel, festivals
- Session logs and remembrance timeline
- Shared family board for group chanting goals
- Automatic dedication after completed practice, donation, or volunteer action

### C. Temple service marketplace

Best for monetization and institutional partnerships.

Possible features:
- Lamp, incense, and offering purchases
- Sponsored chanting by temple or community volunteers
- Festival packages
- Memorial and ancestor service booking
- Verified temple profiles and regional catalogs

### D. Buddhist study copilot

Best for higher-retention knowledge users.

Possible features:
- Daily sutra passage
- Pronunciation and transliteration
- Multiple translations and tradition filters
- Short doctrinal explanation with citations
- Reading plans by theme: compassion, protection, grief, abundance, mindfulness

### E. Home shrine robot

Best for hardware-led differentiation.

Possible features:
- Scheduled recitation playback
- Bell/light/candle choreography
- Motion or presence-triggered greeting
- Festival mode and ritual presets
- Smart speaker or shrine device integration
- Automatic end-of-day practice report and merit-score announcement

### F. Local-first merit keeper

Best for v1 products that avoid servers and still feel alive every day.

Possible features:
- Fully local action log
- Configurable daily chanting schedule
- Merit-score algorithm with fixed transparent weights
- Daily and weekly summaries
- Offline dedication history
- Privacy-first household mode

## 2. High-Value User Segments

- Busy urban lay practitioners who want structure without long study time
- Older users who want chanting assistance, memorial rituals, and easy temple access
- Diaspora families who want multilingual Buddhist practice at home
- Younger seekers who want a spiritual product but prefer habit-driven mobile UX
- Temple operators who need digital services, donations, and remote engagement

## 3. Strong Habit Loops

- Daily discipline loop: reminder -> chanting -> dedication -> streak -> tomorrow plan
- Local robot loop: scheduled practice -> completion detected -> merit points updated -> nightly spoken summary
- Emotional relief loop: worry/grief -> guided recitation -> dedication -> calm summary
- Festival loop: event notification -> ritual suggestion -> offering -> group participation
- Family ritual loop: group goal -> shared session count -> completion -> dedication board
- Commerce loop: intention -> purchase/booking -> ritual fulfillment -> proof and dedication

## 4. Content Strategy

Start with a narrow library that supports real repeated behavior.

Recommended sequencing:
- Phase 1: daily chants, short sutras, common dedications, festival essentials
- Phase 2: memorial, healing, compassion, repentance, prosperity, travel, exams
- Phase 3: lineage-specific packs and deeper study bundles

Metadata to attach to each text:
- Tradition
- Language
- Transliteration
- Audio track
- Length
- Common use case
- Source confidence

## 5. Merit-Score Design

Treat功德值 as a product-layer motivational system, not a doctrinal truth engine.

Good local scoring inputs:
- Session completed
- Minutes chanted
- Dedication completed
- Consecutive days practiced
- Scripture segment finished
- Silent reflection completed

Suggested score behavior:
- Small base points for every devotional action
- Bonus for consistency, not just quantity
- Soft daily cap to avoid farming behavior
- Weekly summaries that explain patterns in human language

Good example lines:
- "Your robot completed 3 guided practices today and raised your practice index by 84."
- "Your evening streak is now 5 days."
- "The strongest contributor this week was consistent dedication after chanting."

## 8. Trust and Safety

Avoid these claims:
- Guaranteed merit outcomes
- Healing guarantees
- Karma prediction
- "This AI is enlightened"
- "This replaces a teacher or clergy member"
- "This score proves your real karmic merit"

Preferred framing:
- Guided practice
- Ritual assistance
- Text-based support with citations
- Temple-connected devotional service
- Habit and remembrance companion
