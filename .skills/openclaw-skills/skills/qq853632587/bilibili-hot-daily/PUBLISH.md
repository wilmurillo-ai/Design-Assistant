## 📋 发布前检查清单

- [x] SKILL.md 格式正确
- [x] package.json 配置完整
- [x] 代码测试通过
- [x] README.md 文档完善

## 📦 发布步骤

### 1️⃣ 安装 ClawHub CLI

```bash
npm install -g clawhub@latest
```

### 2️⃣ 登录 ClawHub

```bash
clawhub login
```

### 3️⃣ 发布技能

```bash
cd skills/bilibili-hot-daily
clawhub publish
```

### 4️⃣ 验证发布

```bash
clawhub search bilibili-hot-daily
```
