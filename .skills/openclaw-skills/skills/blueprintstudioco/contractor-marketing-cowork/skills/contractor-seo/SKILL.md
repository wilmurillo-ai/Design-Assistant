## Local SEO for contractors

### Page structure
Every service page: title tag under 60 chars "[Service] in [City] | [Business Name]". Meta description under 155 chars with CTA. One H1 with primary keyword. 3+ internal links. Schema markup: LocalBusiness (homepage), Service (service pages), BreadcrumbList (all).

### Service area pages
One page per city/county in service area with genuinely unique content about that area. Never duplicate content with city name swapped. Reference local landmarks, neighborhoods, common property types.

### Citation building (NAP consistency)
Exact same Name, Address, Phone on every directory. Priority order: data aggregators (Data Axle, Localeze, Foursquare) > major (GBP, Apple Maps, Bing, Yelp, BBB) > industry (Angi, HomeAdvisor, Thumbtack) > general (YP, MapQuest). Monthly audit on 1st.

### Blog strategy
Target striking-distance keywords (positions 4-20). 1,200-1,800 words. Open with real scenarios, not "Are you looking for..." Include local references. FAQ section for People Also Ask. Monthly calendar: educational, project showcase, seasonal, local guide.

### Technical SEO
Mobile-first responsive. Image alt tags with keywords. XML sitemap submitted to Search Console. robots.txt. Canonical URLs. Open Graph tags. Page speed under 3 seconds.

### What NOT to do
- Never stuff keywords
- Never duplicate content across service area pages
- Never buy backlinks
- Never use "serving [city] and surrounding areas" on every page
- Never ignore mobile experience
