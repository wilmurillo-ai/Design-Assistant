# Connectors

This plugin works with the following services. When a direct connector is not available, Claude uses computer use to navigate the browser.

## Direct connectors (if available)
- **Slack** -- team notifications and alerts
- **Google Drive** -- file storage and sharing

## Via computer use (browser automation)
- **Google Business Profile** -- post updates, respond to reviews, manage Q&A
- **Meta Ads Manager** -- create campaigns, check performance, manage budgets
- **Google Ads** -- search campaigns, keyword management, performance reports
- **Buffer / Later** -- schedule social media posts
- **Mailchimp / MailerLite** -- send email campaigns, manage sequences
- **Google Search Console** -- keyword rankings, indexing status
- **Google Analytics 4** -- traffic reports, conversion tracking
- **Instagram / Facebook** -- direct posting when scheduling tools unavailable
- **Yelp / BBB / directories** -- listing management
- **Jobber / HouseCall Pro** -- CRM data, customer lists, job records

## MCP server
- **Heavy Metric Strategies** -- 74 proven contractor marketing strategies. Claude queries these automatically when generating content, ads, positioning, or recommendations.
