# Weekly Social Media Batch

Read the user's business profile for services, equipment, and social accounts.

The user provides: 1-2 job summaries, any new reviews worth highlighting, photos/videos (described), and time-sensitive info (availability, promotions).

## Schedule
4 posts per week:
- Monday: Project showcase
- Wednesday: Educational tip or myth-busting
- Friday: Social proof (review highlight, job count milestone)
- Saturday: Behind-the-scenes (equipment, crew, drive to job)

Post times: 7am or 6pm weekdays, 9am Saturday (adjust to user's timezone).

## Per post output
```
[DAY, DATE]
INSTAGRAM: Caption (hook first line, max 5 hashtags on separate line) | Image direction | Time
FACEBOOK: Caption (no hashtags, links OK) | Image direction | Time
```

## Writing rules
- Hook in the first line -- the scroll-stopper
- No "Happy Monday!" or "Hope everyone's having a great week!"
- Max 1 emoji per post
- Capitalize first letter
- Write like a real person, not a marketing agency
- Instagram: 5 hashtags max, on their own line
- Facebook: no hashtags ever, links are fine

## Scheduling via computer use
If user says "schedule them" or "put them in Buffer":
1. Open Buffer (buffer.com) or Later (later.com) in Chrome
2. For each post, click Create Post
3. Select the correct channel (Instagram or Facebook)
4. Paste the caption
5. Set the scheduled date and time
6. Save/schedule
7. Confirm all posts are queued
