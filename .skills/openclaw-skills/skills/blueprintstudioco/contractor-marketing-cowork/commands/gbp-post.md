# GBP Post Generator

Read the user's business profile. If Heavy Metric strategies are available via MCP, query for relevant strategies (search "google business profile" or "local seo").

## Post rotation
Determine which type is next based on the current week:
- Week 1 of month: Project showcase (ask user for recent job details if not provided)
- Week 2: Seasonal tip tied to their services and current season
- Week 3: Availability update ("booking [month] now")
- Week 4: Equipment or team spotlight

## Post rules
- 100-300 words
- One CTA button (Call, Learn More, or Book)
- 1-2 keywords from their services naturally included
- No hashtags
- Conversational, not corporate
- Capitalize first letter
- Max 1 emoji

## If user wants to publish (computer use)
1. Open Chrome and go to business.google.com
2. Click "Add update" or "Create post"
3. Select "Add update" post type
4. Paste the post content
5. Add CTA button if supported
6. Click "Post"
7. Confirm to user that it was published

## Output
Provide the post text and ask: "Want me to post this to your Google Business Profile, or would you rather copy it yourself?"
