# Proposal Template

Generate from brief input like: "[name], [address], [acreage], [work], [price]"

## Sections

### Header
Business name, address, phone, email, website. Logo if available.

### Customer Information
Name, property address, contact info.

### Property Assessment
Turn brief notes into professional language. Reference specific conditions.

### Scope of Work
- Services to be performed
- Equipment to be used (from business profile)
- Expected timeline
- Access requirements

### What Is NOT Included
Standard exclusions: stump grinding below grade, hauling debris off-site, grading, seeding (unless specified).

### Timeline
Start date, duration, completion. Note weather dependencies.

### Pricing
- Line items if multiple services
- Total price
- Payment terms: due upon completion
- Accepted payment methods

### Terms and Conditions
- Change orders require written approval
- Customer marks property lines and utilities
- 48-hour cancellation notice
- Not responsible for unmarked underground utilities
- Work area clear of vehicles, pets, personal items

### Signature Block
Customer signature, date, printed name. Company signature line.

## Format
Clean HTML for PDF (Ctrl+P > Save as PDF).
