# 通用公文格式标准

适用于：报告、总结、方案、意见等非标准文种

---

## 一、格式规范

### 1. 标题

- **字体**：方正小标宋简体
- **字号**：二号（22pt）
- **对齐**：居中
- **内容**：由发文机关名称、事由、文种组成（可省略发文机关名称）

示例：
```
关于2025年工作总结的报告
广东省政务服务数据管理局2025年工作要点
```

### 2. 正文

- **字体**：仿宋_GB2312
- **字号**：三号（16pt）
- **对齐**：两端对齐
- **缩进**：首行缩进2字符
- **行距**：固定值28磅

### 3. 标题层级

**一级标题：**
- **字体**：黑体
- **字号**：三号（16pt）
- **格式**：一、二、三...
- **缩进**：首行缩进2字符

**二级标题：**
- **字体**：楷体_GB2312
- **字号**：三号（16pt）
- **格式**：（一）（二）（三）...
- **缩进**：首行缩进2字符

**三级标题：**
- **字体**：仿宋_GB2312加粗
- **字号**：三号（16pt）
- **格式**：1. 2. 3. ...
- **缩进**：首行缩进2字符

### 4. 落款

- **发文机关**：右对齐
- **日期**：右对齐，在发文机关下方
- **字体**：仿宋_GB2312
- **字号**：三号（16pt）
- **日期格式**：汉字数字（二〇二五年三月五日）或阿拉伯数字（2025年3月5日）

示例：
```
                                        广东省政务服务和数据管理局
                                        二〇二五年三月五日
```

### 5. 页边距

- 上：25mm
- 下：25mm
- 左：25mm
- 右：25mm

---

## 二、基本结构

```
标题

正文（包含多个层级标题）

落款（发文机关+日期）
```

**基本结构**：标题 → 正文（包含多个层级标题） → 落款（发文机关+日期）
