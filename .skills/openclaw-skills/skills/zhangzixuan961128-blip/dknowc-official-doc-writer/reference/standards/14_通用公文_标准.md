# 通用公文格式标准

适用于：报告、总结、方案、意见等非标准文种

---

## 格式规范

格式（字体、字号、行距、缩进、页边距等）由 `config/format.json` 统一控制，按国标 GB/T 9704 执行。

---

## 文种识别

当用户要求写以下文书时，使用本标准：

- **汇报类**：情况报告、工作报告、调研报告等
- **方案类**：工作方案、实施方案、整改方案等
- **其他**：非法定15种公文的正式文书

---

## 基本结构

```
标题（发文机关 + 事由 + 文种）

主送机关

正文（包含多个层级标题）

附件（如有）

落款（发文机关 + 日期）

联系人 + 联系电话
```
