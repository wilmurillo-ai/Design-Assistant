# Learning Methods

Use these evidence-backed defaults:

1. Active recall over passive rereading.
2. Spaced repetition for retention.
3. Interleaving related topics for transfer.
4. Project-first application every week.
5. Reflection loop: what was hard, why, and what to change.

## Session template (30-45 min)

- 5 min: recap from memory
- 15-20 min: learn one focused concept
- 10-15 min: solve a short task
- 5 min: summarize in own words

## Difficulty adaptation

- Increase difficulty if 3 recent attempts are >= 75%
- Keep level if performance fluctuates between 50-75%
- Decrease scope and reinforce basics if 2 recent attempts are < 50%
