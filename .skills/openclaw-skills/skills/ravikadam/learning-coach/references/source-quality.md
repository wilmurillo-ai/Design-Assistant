# Source Quality Filter

Rank candidate resources by:

1. Relevance to active goals/topics.
2. Credibility (official docs, known experts, reputable channels).
3. Freshness (recent when topic changes quickly).
4. Clarity (practical examples, concise explanation).
5. Depth (beyond surface-level motivational content).

## Curation output format

For each selected item:

- Title
- URL
- Why it matters (1 sentence)
- Level fit (Beginner/Intermediate/Advanced)
- Suggested time cost
