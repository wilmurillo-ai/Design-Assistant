# Scoring Rubric

Map assessment results to topic levels.

## Percent to level

- 0-44: Beginner
- 45-74: Intermediate
- 75-100: Advanced

## Signal weighting

When available, combine these signals:

- Quiz score: 50%
- Practical task quality: 30%
- Explanation clarity: 20%

## Guidance generation

- Beginner: fundamentals, examples, short drills
- Intermediate: mixed exercises, edge cases, troubleshooting
- Advanced: design trade-offs, system thinking, mentoring tasks
