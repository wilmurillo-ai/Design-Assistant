# HEARTBEAT.md — vision-action-evolution-loop

## 心跳触发任务

### 每次心跳
```
1. 检查是否有新的视觉样本需要处理
2. 如果有，执行 Perceive 阶段，记录结果
3. 检查认知池中是否有足够的认知点触发新闭环迭代
4. 更新进化日志
```

### 每日任务
```
1. 审查所有未完成闭环的反馈状态
2. 对已验证的进化增量更新世界模型
3. 搜索arXiv最新VLA/具身智能论文（通过arxiv-collision-cognition）
4. 检查三阶段桥接中每个阶段的当前状态
```

### 进化指标
- 闭环完成次数
- 平均迭代次数至收敛
- 新增认知节点数
- 五阶段各阶段平均置信度
