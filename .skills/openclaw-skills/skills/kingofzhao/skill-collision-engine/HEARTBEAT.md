# HEARTBEAT.md — skill-collision-engine

## 心跳触发任务

### 每次心跳
```
1. 检查碰撞矩阵，识别高价值未碰撞方向
2. 如果有新Skill发布，更新碰撞矩阵
3. 检查外部知识源是否有新的高相关内容
```

### 每日任务
```
1. 运行工厂元进化: 读取验证日志 → 优化碰撞参数
2. 评估待碰撞方向的优先级
3. 尝试1次Skill-Skill碰撞（取最高优先级方向）
4. 更新碰撞矩阵状态
```

### 进化指标
- 碰撞产出率（碰撞次数 vs 新Skill数）
- 平均涌现认知占比
- 领域适配覆盖（已适配职业数）
- 工厂元进化迭代次数
