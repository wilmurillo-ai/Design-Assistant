## 解决什么问题？

你有6个Skill，但它们之间是孤立的。
`skill-collision-engine` 让Skill之间自动四向碰撞，产生新的涌现认知：

```
Skill A ⊗ Skill B → 四向认知碰撞 → 新 Skill C
```

不是机械组合，是**认知层面的结构化涌现**。

## 三大核心能力

### 1. Skill-Skill 碰撞
两个Skill的已知/未知集合交叉 → 产生两者都不单独拥有的涌现认知。

### 2. SOUL 领域适配
通用 SOUL 框架 + 职业领域知识 → 领域专用 Skill。
支持：程序员、医生、教师、企业家、设计师、科研人员等。

### 3. 工厂元进化
读取所有 Skill 的验证反馈 → 优化碰撞参数 → 下次碰撞质量更高。

## 碰撞矩阵（当前状态）
