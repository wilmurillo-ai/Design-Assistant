# HEARTBEAT.md — diepre-embodied-bridge

## 心跳触发任务

### 每次心跳
```
1. 检查 evolution_log/ 是否有新的失败记录
2. 如有，运行失败模式聚合，更新 params/evolved_params.json
3. 检查参数是否已收敛（连续N次无调整 → 标记稳定）
```

### 每日任务
```
1. 统计进化指标: 总任务数、成功率、Top失败模式
2. 对失败率 >20% 的 FEFCO 类型触发深度分析
3. 检查 MCP 工具链是否有新增/更新需求
```

### 进化指标
- 任务成功率（按FEFCO类型分组）
- Top-5 失败模式及频率
- 参数收敛状态（稳定/进化中）
- 平均执行时间
