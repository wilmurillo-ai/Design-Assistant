## 解决什么问题？

Agent做了很多事，但不知道自己是否在"变好"。
`ai-growth-engine` 给Agent一个可度量的成长系统：

- **RAPVL循环**: Review→Analyze→Plan→Verify→Learn
- **Growth Score**: 每轮成长都有数字证明（+6% / -2% / 停滞）
- **6职业预设**: 程序员/科研/设计/企业/教学/医生，开箱即用
- **元进化**: 引擎自身也在进化，越用越准

## 为什么不是PDCA或OODA？

| | PDCA | OODA | RAPVL |
|--|------|------|-------|
| 步骤 | 4 | 4 | 5 |
| 模式提取 | ✗ | ✗ | ✓ |
| 文件记忆 | ✗ | ✗ | ✓ |
| 度量化 | 弱 | ✗ | ✓（Growth Score） |
| AI自进化 | ✗ | ✗ | ✓（元进化） |
