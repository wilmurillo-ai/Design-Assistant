# 彩票分析系统 V2.0

专业的彩票分析工具，支持双色球和大乐透。

## Windows 兼容特性

- 无 emoji 字符，完美兼容 Windows 控制台
- 自动处理文件编码（UTF-8/GBK）
- 完整功能保留

## 功能特性

- [基础统计] 热号、冷号、和值分布、频率分析
- [遗漏分析] 号码遗漏期数、回补概率
- [智能推荐] 6种策略（均衡/热号/冷号/遗漏回补/连号/区间）
- [杀号推荐] 自动排除过热、遗漏过大号码
- [胆拖推荐] 自动生成胆码+拖码组合
- [历史回测] 验证预测策略准确率
- [趋势数据] 可视化图表数据
- [报告导出] JSON格式完整分析报告
