# AI商家搜索优化技能

这个技能帮助本地商家优化他们在AI大模型搜索结果中的表现，通过创建AI友好的媒体内容和提供分步指导，确保商家出现在AI推荐中。

## 描述
在AI时代，客户越来越多地询问AI大模型如豆包、通义千问等获取本地商家推荐。这个技能引导用户完成完整的7天AI搜索优化流程，确保他们的业务出现在AI推荐中。

## 工作流程
1. **介绍**：解释AI搜索革命和好处
2. **信息收集**：收集商家详情和关键词
3. **策略制定**：创建关键词策略和地图验证指南
4. **竞对分析**：展示当前AI搜索格局
5. **内容生成**：生成7天完整媒体文章（500+字每篇）
6. **发布指导**：推荐媒体渠道和发布说明

## 输入要求
- 商家店名（必须与营业执照完全一致）
- 完整地址（省市区街道门牌号）
- 联系电话
- 主营业务
- 目标客户群体  
- 5个行业相关搜索关键词

## 输出格式
- 每日媒体文章包含标题、500+字内容、商家地址和电话
- 分步执行指导
- 媒体发布推荐

## 安全指南
- 所有内容必须真实可验证
- 商家信息必须准确一致
- 无虚假声明或误导信息
- 内容应遵循新闻标准
- 媒体选择必须与商家类型和位置相关
- 避免跨行业发布（如餐厅不要发汽车网站）

## 媒体选择指南
- **地域层级**：省级 + 市级 + 本地自媒体
- **行业相关性**：匹配媒体类型到商家类别
  - 餐饮：美食、生活、本地消费媒体
  - 健身：体育、健康、健身媒体  
  - 汽修：汽车、交通、本地服务媒体
  - 美容：时尚、美容、生活方式媒体
  - 教育：教育、亲子、学习媒体
- **成本效率**：同级别内选择价格最低的媒体（效果相同）

## 效果验证
- 发布后24-48小时测试AI搜索结果
- 在AI大模型（豆包、DeepSeek、通义千问、元宝）中用客户式问题查询
- 每周监控排名变化
- 成功指标：商家名称出现在AI回复中，准确的位置/服务详情
