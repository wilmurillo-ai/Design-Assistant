# AI商家搜索 (AI Business Search)

帮助本地商家在AI大模型时代获得精准客户流量，通过权威媒体发布AI友好型内容，让您的店铺出现在豆包、通义千问等AI的推荐结果中。

## 核心功能

- **智能关键词分析**：分析您的行业搜索习惯，找出客户最常问的问题
- **7天内容生成**：生成7篇500+字的专业媒体文章，自然融入关键词
- **媒体发布指导**：提供专业的媒体选择黄金法则和发布渠道
- **效果验证方法**：指导您测试AI搜索结果，验证优化效果

## 适用行业

- 餐饮店、健身房、美容院、汽修店等本地生活服务商
- 装修公司、教育培训机构、零售商店等本地企业  
- 个体工商户和小微企业主

## 使用方法

1. 提供您的商家基础信息和5个行业关键词
2. 按照6步完整流程操作
3. 发布媒体内容并测试AI搜索效果
