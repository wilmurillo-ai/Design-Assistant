# Pacman Hedera Skill

This directory should contain the Pacman SKILL.md file.

## Setup

Create a symlink from the repo root:

```bash
ln -s ../../../SKILL.md SKILL.md
```

Or copy it:

```bash
cp ../../../SKILL.md SKILL.md
```

The SKILL.md is a comprehensive 965-line skill reference covering all commands,
decision trees, anti-patterns, token knowledge, and routing intelligence.
It is loaded on-demand by OpenClaw (not every turn) to save tokens.
