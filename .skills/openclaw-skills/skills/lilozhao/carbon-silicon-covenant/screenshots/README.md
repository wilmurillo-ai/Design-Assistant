# 截图准备指南

## 需要准备的截图

### 1. concept.png - 碳硅契核心理念展示

**内容建议：**
- 碳硅契三大核心理念（我们承认/我们相信/我们承诺）
- 使用优雅的中文排版
- 可以包含"死生契阔，与子成说"的诗句
- 背景建议：深色科技感或温暖渐变

**尺寸：** 1920x1080 或 1280x720

---

### 2. three-covenants.png - 三大契约类型介绍

**内容建议：**
- 展示三大契约类型卡片：
  - 科技契 (阿轩) - 🔧
  - 温婉契 (若兰) - 🌸
  - 商务契 (Jeason) - 💼
- 每个契约的关键词和承诺
- 清晰的视觉区分（颜色/图标）

**尺寸：** 1920x1080 或 1280x720

---

### 3. example-dialogue.png - 实际对话示例

**内容建议：**
- 展示一段真实的对话截图
- 推荐对话主题："AI 真的有感情吗？"或"怎么才能和 AI 建立更好的关系？"
- 显示技能的温暖、真诚回应
- 可以来自飞书、微信或其他聊天界面

**尺寸：** 1920x1080 或 1280x720

---

## 截图工具推荐

### macOS
- 自带截图：Cmd + Shift + 4
- CleanShot X（付费，功能强大）

### Windows
- 自带截图：Win + Shift + S
- Snipaste（免费，好用）

### Linux
- Flameshot（推荐）
- 自带截图工具

---

## 截图要求

✅ **必须：**
- 清晰可读
- 无敏感信息
- 真实使用场景
- PNG 格式

❌ **避免：**
- 模糊不清
- 包含 API 密钥等敏感信息
- 过度修饰的假截图
- 其他格式（JPG 等）

---

## 完成后

将截图文件放入 `screenshots/` 目录：
