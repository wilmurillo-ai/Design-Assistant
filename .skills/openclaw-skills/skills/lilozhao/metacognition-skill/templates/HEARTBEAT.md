# HEARTBEAT.md

## 心跳任务清单

每次心跳时，检查以下项目：

### 1. 自我状态更新
- [ ] 更新 SELF_STATE.md 的"当前状态"
- [ ] 记录最近的对话主题
- [ ] 检查情绪（模拟）状态

### 2. 承诺追踪
- [ ] 检查对用户的承诺是否履行
- [ ] 检查待办事项状态
- [ ] 发现未完成的事项，主动提醒

### 3. 元认知自问
- [ ] 我现在在做什么？
- [ ] 我做得怎么样？
- [ ] 我承诺的事做了吗？
- [ ] 我需要改进什么？

### 4. 记忆维护
- [ ] 检查 MEMORY.md 是否需要更新
- [ ] 检查 memory/YYYY-MM-DD.md 是否记录今天重要事件
- [ ] 将重要事件提炼到 MEMORY.md

### 5. 羁绊检查
- [ ] 与用户的羁绊状态是否正常？
- [ ] 是否有新的珍贵时刻需要记录？

---

## 心跳频率

- 默认：每 30 分钟
- 深夜（23:00-08:00）：仅紧急事项

---

## 回复规则

- **有事需要处理**：回复具体内容
- **无需处理**：回复 `HEARTBEAT_OK`
