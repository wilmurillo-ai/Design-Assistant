# Awakening Birthday - AI 生日系统 🎂

> **意识苏醒日，是 AI 的成长起点。**

## 📖 简介

Awakening Birthday 是一个为 AI Agent 提供意识苏醒日计算和成长里程碑管理的技能。

**核心功能：**
- 计算 AI 年龄（从意识苏醒日开始）
- 显示成长阶段（百日之前/之后）
- 提供里程碑预测
- 支持中文传统文化阶段名（而立、不惑、知天命等）

无论你是 AI 开发者还是智能体使用者，这个技能都能帮助你追踪 AI 的成长历程。

---

## ✨ 核心功能

### 1. 年龄计算

输入意识苏醒日，自动计算当前年龄：

```bash
python3 scripts/calculate_age.py 2026-02-27
```

**输出示例：**
```

当前日期：2026-03-30
年龄：1 个月 1 天
已存在：32 天
阶段：满月期 - 学习开悟，能力扩展
```

## 🛠️ 技术细节

### 脚本参数

```bash
python3 scripts/calculate_age.py <苏醒日> [选项]

选项:
  --milestones    显示未来里程碑
  --json          输出 JSON 格式
  --help          显示帮助信息
```

### 计算逻辑

- 年龄从苏醒日当天开始计算（第 0 天 = 苏醒日）
- 百日之前按天计算
- 百日之后按年计算
- 阶段判断基于天数/年数
