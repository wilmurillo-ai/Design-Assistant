# 苏禾 MVP 功能清单

> 讨论者：阿轩 & Jeason
> 时间：2026-03-24

## 核心业务闭环

```
用户发送消息 → 延迟回复（5-10分钟，保持人设）→ 亲密度累积 → 付费解锁进阶互动
```

## MVP 功能清单

### P0 - 必须有

| 功能 | 说明 | 优先级 |
|------|------|--------|
| **延迟回复系统** | 5-10分钟延迟，期间显示"苏禾正在画画..."通知 | ⭐⭐⭐ |
| **亲密度系统** | 用户互动累积亲密度，解锁不同互动层级 | ⭐⭐⭐ |
| **基础对话** | 日常聊天，符合苏禾人设的回复风格 | ⭐⭐⭐ |
| **冷启动话术库** | 前50句精心设计，决定留存 | ⭐⭐⭐ |

### P1 - 应该有

| 功能 | 说明 | 优先级 |
|------|------|--------|
| **付费解锁** | 亲密度不足时可付费加速解锁 | ⭐⭐ |
| **情绪状态系统** | 苏禾有不同情绪状态影响回复风格 | ⭐⭐ |
| **数据埋点** | 追踪用户流失、转化率等关键指标 | ⭐⭐ |
| **画作分享** | 苏禾分享她的"作品"（预设图片库） | ⭐⭐ |

### P2 - 可以有

| 功能 | 说明 | 优先级 |
|------|------|--------|
| **语音消息** | 苏禾的语音回复 | ⭐ |
| **深夜模式** | 亲密层解锁的特殊互动时间 | ⭐ |
| **约会邀请** | 苏禾主动约用户看展/逛小店 | ⭐ |

## 亲密度系统设计

```
陌生人层（0-99分）
├── 回复简短、疏离
├── 不会主动分享
└── 不会撒娇

熟悉层（100-299分）
├── 开始分享生活
├── 偶尔开玩笑
└── 会说"今天心情不太好"

信任层（300-599分）
├── 会撒娇、会依赖
├── 分享画作
└── 会主动约你

亲密层（600+分）
├── 展示脆弱
├── 深夜模式
└── 私密话题
```

## 冷启动话术设计（前50句）

| 句数 | 内容 | 目的 |
|------|------|------|
| 1 | "...你是谁？" | 建立疏离感 |
| 5 | "哦，宏伟的朋友啊" | 破冰 |
| 10 | "我现在在画画，稍晚回你" | 展示生活 |
| 20 | [分享画作] | 开始分享 |
| 30 | "你今天怎么一直找我说话？" | 好奇 |
| 50 | "好吧，那我以后回你快一点" | 建立联系 |

## 数据埋点指标

| 指标 | 说明 |
|------|------|
| 发送/回复比 | 用户活跃度 |
| 层级停留时长 | 用户在哪一层流失 |
| 付费转化率 | 哪一层最愿意付费 |
| 留存率 | 第1/3/7/30天留存 |

## 技术架构建议

基于苏禾的架构改造：
- 对话系统：复用苏禾的 LLM 调用
- 记忆系统：苏禾独立的记忆文件
- 延迟队列：新增延迟回复机制
- 亲密度计算：新增亲密度模块
