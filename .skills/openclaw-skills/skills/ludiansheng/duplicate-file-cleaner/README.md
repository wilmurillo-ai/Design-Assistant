# 重复文件清理工具 - duplicate-file-cleaner

一个智能的重复文件检测与整理工具，帮助你快速识别和清理计算机中的重复文件，释放宝贵的磁盘空间。

## 功能特性

- **精准检测**：基于文件内容哈希比对，准确识别完全相同的重复文件
- **智能过滤**：支持按文件类型、大小进行筛选，聚焦高价值重复文件
- **安全删除**：提供智能删除建议，标注高风险文件，避免误删
- **详细报告**：生成结构化扫描报告，清晰展示重复文件分布和可释放空间
- **灵活执行**：支持自动删除、手动确认、备份后删除多种模式

## 适用场景

- 📁 **文件整理**：清理重复下载的文件，释放磁盘空间
- 🖼️ **照片库管理**：删除重复的图片和视频，特别是多设备同步后的照片
- 📄 **项目清理**：清理项目文件中的冗余备份
- 💾 **空间优化**：快速释放磁盘空间，提升系统运行效率
- 🔄 **数据同步后处理**：处理云同步、多设备备份产生的重复文件

## 安装要求

- Python 3.8 或更高版本
- 操作系统：Windows、macOS、Linux 均支持

## 快速开始

### 1. 扫描照片库

```bash
python duplicate-file-cleaner/scripts/duplicate_scanner.py \
  --directory ~/Pictures \
  --output report.json \
  --min-size 10240 \
  --extensions jpg,png,heic
```

### 2. 扫描下载目录

```bash
python duplicate-file-cleaner/scripts/duplicate_scanner.py \
  --directory ~/Downloads \
  --output report.json \
  --min-size 102400
```

### 3. 全盘扫描（谨慎使用）

```bash
python duplicate-file-cleaner/scripts/duplicate_scanner.py \
  --directory ~ \
  --output report.json \
  --min-size 1024
```

## 参数说明

| 参数 | 必填 | 说明 | 默认值 |
|------|------|------|--------|
| `--directory` | 是 | 要扫描的目录路径 | - |
| `--output` | 否 | 输出报告文件路径 | 终端输出 |
| `--min-size` | 否 | 最小文件大小（字节） | 1024（1KB） |
| `--extensions` | 否 | 文件扩展名过滤，逗号分隔 | 全部文件 |

## 使用建议

### 安全优先
- 操作前先备份重要数据
- 建议优先从用户目录开始扫描（Documents、Pictures、Downloads）
- 避免直接扫描系统目录（如 Windows/Program Files）

### 效率优化
- 先设置较大的 `--min-size` 过滤小文件
- 使用 `--extensions` 限制扫描的文件类型
- 大目录扫描可能需要较长时间，请耐心等待

### 智能决策
- 优先清理文件名、修改时间明显相同的重复文件
- 注意文件路径中的"backup"、"copy"等关键词
- 对不确定的文件选择"备份后删除"模式

## 输出报告说明

扫描完成后，工具会生成结构化的 JSON 报告，包含：
- 重复文件组列表
- 每组的文件路径、大小、修改时间
- 可释放的总空间统计
- 智能删除建议和风险标注

## 常见问题

**Q: 会删除重要的系统文件吗？**
A: 工具只检测内容完全相同的文件，并提供删除建议，最终删除操作需要你确认。建议避免扫描系统目录。

**Q: 扫描速度慢怎么办？**
A: 可以通过设置 `--min-size` 过滤小文件，或使用 `--extensions` 限制扫描文件类型来提高速度。

**Q: 能检测视觉相似的图片吗？**
A: 不能。工具只检测内容完全相同的文件，不检测视觉相似度。

**Q: 误删了文件怎么办？**
A: 建议使用"备份后删除"模式，或在进行删除操作前完整备份重要数据。
