## 模板使用指南

### 模板选择原则

根据内容目标选择合适模板：

| 内容目标 | 推荐模板 | 适用场景 |
|---------|---------|---------|
| 展示产品优势 | 产品评测模板 | 新产品发布、产品介绍 |
| 帮助用户决策 | 选购指南模板 | 用户选购阶段、教育市场 |
| 对比竞品 | 对比分析模板 | 竞品对比、差异化展示 |
| 解决用户问题 | 教程指南模板 | 使用指导、问题解决 |
| 回答常见问题 | FAQ 模板 | 用户疑问解答 |
| 建立专业形象 | 深度分析模板 | 行业分析、趋势解读 |

### 模板使用步骤

1. **确定内容目标**：明确内容要达成的目标
2. **选择合适模板**：根据目标选择模板
3. **填充核心内容**：按照模板结构填充内容
4. **添加数据支撑**：补充统计数据和权威引用
5. **优化结构细节**：检查标题层级、段落长度等
