# 电子书智能阅读器 (Ebook Reader)

[![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)](https://github.com)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)
[![Python](https://img.shields.io/badge/python-3.8+-orange.svg)](https://www.python.org)

智能电子书阅读助手，支持 EPUB/PDF 等常见格式解析，提供章节总结、关键内容摘录、深度分析与落地应用建议。

---

## 核心功能

### 1. 电子书格式解析
- 支持 **EPUB** 和 **PDF** 两种主流格式
- 自动提取书籍元数据（书名、作者、章节数）
- 结构化输出 JSON 格式，便于后续处理

### 2. 章节智能总结
- 核心观点提炼（1-3句话概括）
- 关键论据梳理（3-5个支撑点）
- 逻辑结构分析（因果/并列/递进/对比）

### 3. 关键内容摘录
- 启发性观点识别
- 可执行方法论提取
- 经典案例与数据摘录
- 金句与隐喻收藏

### 4. 深度内容分析
- **内容层面**：主题深度、观点创新性、论据充分性
- **逻辑层面**：论证方式、思维模型、逻辑严密性
- **价值层面**：理论贡献、实践意义、适用边界

### 5. 落地应用建议
- **个人层面**：认知升级、习惯养成、技能提升
- **工作层面**：方法应用、决策优化、流程改进
- **社会层面**：价值传播、体系构建、影响力扩大

## 快速开始

### 安装依赖

```bash
pip install ebooklib==0.18 pypdf==3.17.0 beautifulsoup4==4.12.0
```

### 基本用法

#### 1. 解析电子书

```bash
# 解析整本书
python scripts/ebook_parser.py --input book.epub --output result.json

# 解析指定章节
python scripts/ebook_parser.py --input book.epub --chapter 1 2 3

# 解析 PDF 文件
python scripts/ebook_parser.py --input document.pdf --output output.json
```

#### 2. 使用智能体分析

上传电子书后，向智能体发送指令：

```
请帮我阅读这本书并生成每章的总结
```

```
请从第3章中摘录所有可执行的方法论
```

```
请分析第5章，并给出如何在工作中的应用建议
```

## 输出格式

### 电子书解析输出

```json
{
  "book_title": "书名",
  "author": "作者",
  "total_chapters": 10,
  "chapters": [
    {
      "number": 1,
      "title": "章节标题",
      "content": "章节内容文本"
    }
  ]
}
```

### 章节总结输出

```
【核心观点】习惯的形成不是靠意志力，而是靠设计合理的系统。
【关键论据】
  1. 英国自行车队的边际增益案例
  2. 两分钟法则的实践效果
  3. 环境设计对行为的影响研究
【逻辑结构】递进结构：现象 → 原理 → 方法 → 案例
```

### 摘录输出格式

```
【摘录】你不会达到目标的高度，只会落到系统的水平。
【位置】第1章：系统优于目标
【价值】用对比手法深刻揭示了系统思维的重要性
```

## 使用场景

### 场景 1：系统化阅读
**用户需求**：阅读《原子习惯》并生成完整读书报告

**执行流程**：
1. 解析电子书文件
2. 逐章生成总结（核心观点、关键论据、逻辑结构）
3. 汇总输出完整阅读报告

**预期输出**：包含所有章节总结的结构化报告

---

### 场景 2：方法论提取
**用户需求**：从《深度工作》中提取所有可执行的方法论

**执行流程**：
1. 解析全书内容
2. 按摘录标准筛选方法论内容
3. 输出带位置和价值标注的方法论清单

**预期输出**：可直接应用的方法论列表

---

### 场景 3：落地应用规划
**用户需求**：分析《原则》第5章并给出工作应用建议

**执行流程**：
1. 解析指定章节内容
2. 从内容、逻辑、价值三个维度分析
3. 结合工作场景生成具体应用建议

**预期输出**：具体的行动方案和预期效果

## 技术特点

### 智能体能力应用
- **内容理解**：智能体理解章节内容并提炼核心观点
- **文本分析**：智能体识别高价值内容并摘录
- **推理分析**：智能体进行多维度深度分析
- **知识转化**：智能体生成可执行的落地建议

### 脚本能力应用
- **格式解析**：处理 EPUB/PDF 二进制文件格式
- **结构提取**：提取章节结构和元数据
- **数据转换**：输出标准化的 JSON 格式

---

## 注意事项

- **格式支持**：EPUB 解析效果最佳，PDF 可能存在格式损失
- **扫描版 PDF**：建议先使用 OCR 工具转换为文本版
- **章节识别**：基于 HTML 结构自动识别章节标题
- **编码处理**：自动处理 UTF-8 编码问题
- **内容客观性**：分析时保持客观，结合书籍整体主题
