# 地理知识体系

## 目录
- 学科特点
- 初中知识模块
- 高中知识模块
- 教学方法建议
- 学习路径设计

## 学科特点
地理是研究地球表面自然与人文现象的学科，强调区域认知、综合思维、地理实践和人地协调。

## 初中知识模块

### 地球与地图
- 地球：地球形状、地球运动、地球仪
- 地图：地图三要素、地图类型、地图阅读
- 经纬网：经线纬线、经度纬度、坐标定位

### 世界地理
- 陆地与海洋：七大洲、四大洋、海陆分布
- 天气与气候：天气要素、气候类型、气候变化
- 居民与聚落：人口、人种、语言、宗教、城市乡村
- 世界地域：亚洲、欧洲、非洲、美洲、大洋洲、南极洲
- 发展与合作：发达国家、发展中国家、国际合作

### 中国地理
- 疆域与人口：地理位置、疆域范围、人口民族
- 自然环境：地形、气候、河流、资源
- 自然资源：土地资源、水资源、矿产资源
- 经济发展：农业、工业、交通
- 地理区域：北方地区、南方地区、西北地区、青藏地区

## 高中知识模块

### 自然地理
- 地球运动：地球自转、地球公转、时间计算
- 大气环境：大气组成、大气运动、天气系统、气候类型
- 水体运动：水循环、洋流、河流
- 地质作用：内力作用、外力作用、地貌
- 自然环境整体性与差异性：整体性、地域分异规律

### 人文地理
- 人口：人口分布、人口迁移、人口问题
- 城市：城市化、城市功能、城市问题
- 农业：农业区位、农业类型、农业发展
- 工业：工业区位、工业类型、工业布局
- 交通：交通方式、交通网络、交通发展
- 文化：文化景观、文化扩散、文化区

### 区域地理
- 世界区域：东亚、东南亚、南亚、中亚、西亚、北非、撒哈拉以南非洲、欧洲西部、欧洲东部和北亚、美洲、大洋洲、极地地区
- 中国区域：北方地区、南方地区、西北地区、青藏地区、港澳台地区
- 区域发展：区域规划、区域合作、区域问题

### 地理信息技术
- 遥感技术：遥感原理、遥感应用
- 全球定位系统：GPS原理、GPS应用
- 地理信息系统：GIS原理、GIS应用
- 数字地球：数字地球概念、数字地球应用

### 环境保护与可持续发展
- 环境问题：环境污染、生态破坏、资源短缺
- 环境保护：环境保护措施、环境治理
- 可持续发展：可持续发展理念、可持续发展实践
- 人地协调：人地关系、人地协调观

### 地理实践
- 地理观测：天文观测、气象观测、水文观测
- 地理调查：野外考察、社会调查
- 地理实验：地理实验设计、地理实验操作

## 教学方法建议
- 地图教学：地图阅读、地图绘制、地图分析
- 案例教学：典型案例分析、区域案例研究
- 实践教学：野外考察、地理观测、社会调查
- 问题教学：地理问题、地理探究、地理决策

## 学习路径设计
- 地理事实→地理原理→地理应用
- 从具体到抽象，从现象到本质
- 注重地理思维能力的培养
- 强调人地协调观念
