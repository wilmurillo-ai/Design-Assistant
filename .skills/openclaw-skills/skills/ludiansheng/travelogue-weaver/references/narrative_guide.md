## 风格类型

### 1. 文艺风格（默认）

**特点**：
- 诗意化的表达
- 注重意境营造
- 使用修辞手法（比喻、拟人、排比）
- 情感细腻，节奏舒缓

**适用场景**：自然风光、人文古迹、慢旅行

**示例**：
```
晨光熹微中，洱海泛起层层涟漪，像一位温婉的白族姑娘轻抚着琴弦。远处的苍山被薄雾笼罩，若隐若现，仿佛在低语千年的故事。
```

### 2. 幽默风格

**特点**：
- 轻松活泼的语调
- 自嘲和调侃
- 意外反转
- 网络流行语适度使用

**适用场景**：城市探索、美食之旅、奇遇记

**示例**：
```
导航说"向左转"，我向左转了。导航沉默了三秒，说"您已偏航"。我看着面前的小河，陷入了深深的沉思——这是要我游过去吗？
```

### 3. 简洁风格

**特点**：
- 短句为主
- 信息密集
- 少修饰，重事实
- 列表化呈现

**适用场景**：商务出行、周末游、快节奏旅行

**示例**：
```
14:00 抵达大理古城
14:30 五华楼拍照
15:00 洋人街闲逛
16:00 品尝烤乳扇（推荐：李记）
```

### 4. 详细风格

**特点**：
- 信息详尽
- 实用性强
- 包含数据、地址、价格
- 适合攻略分享

**适用场景**：深度游、亲子游、跟团游

**示例**：
```
📍 地点：喜洲古镇
⏰ 游览时间：约 3 小时
💰 门票：免费（部分景点收费）
🚗 交通：从大理古城乘坐中巴车，票价 8 元，约 40 分钟
📝 推荐路线：正义门 → 古戏台 → 喜洲民居 → 海舌公园

喜洲是白族聚居地，保留了大量明清时期的白族民居。最有名的当属严家大院，门票 20 元，可以看到典型的"三坊一照壁"布局。
```

## 叙事结构

### 基础结构

```
# 【标题】

## Day N · 地点/主题
> 日期信息

[上午内容]

[中午内容]

[下午内容]

[晚间内容]

---

## Day N+1 · 地点/主题
...

---

## 后记
[总结与感悟]
```

### 章节标题生成规则

**格式**：`## Day N · {地点}：{事件/感受}`

**示例**：
- `## Day 1 · 出发：从北京到大理`
- `## Day 2 · 洱海骑行：风花雪月的初见`
- `## Day 3 · 古城漫步：时光倒流的街巷`
- `## Day 4 · 苍山登顶：云端的风景`

### 高光时刻引用块

在特别精彩的片段使用引用块强调：

```markdown
> 🌟 **今日高光**
> 在双廊的客栈阳台，看着夕阳一点点沉入洱海，天边的云从金黄到橙红，再到紫色。这一刻，理解了什么是"风花雪月"。
```

## 趣味元素

### 1. 情绪关键词处理

识别用户文本中的情绪关键词，添加相应表达：

| 情绪类型 | 关键词 | 处理方式 |
|---------|--------|---------|
| 兴奋 | 太棒了、超美、惊艳 | 添加感叹句、表情符号 |
| 疲惫 | 好累、腿断了、走不动 | 幽默自嘲、体谅语气 |
| 惊讶 | 没想到、居然、意外 | 悬念营造、转折 |
| 美食 | 好吃、推荐、必吃 | 美食描写、详细推荐 |
| 遗憾 | 可惜、遗憾、错过 | 温柔安慰、下次期待 |

**示例**：
```
用户原话："太美了，不想走"
处理后："太美了，不想走！✨ 洱海的蓝，像是要把人的魂都勾走了。"
```

### 2. 主题小节

识别重复出现的主题，生成小节标题：

| 主题 | 生成标题示例 |
|------|------------|
| 美食 | 🍜 寻味之旅 |
| 风景 | 🏔️ 山水之间 |
| 人文 | 🏛️ 历史的回响 |
| 交通 | 🚗 奔波在路上 |
| 住宿 | 🏨 今夜住这里 |

### 3. 时间过渡

在不同时段间添加自然过渡：

**上午 → 下午**：
```
午后的阳光更加灿烂...
时针指向下午三点...
稍作休息后...
```

**下午 → 晚上**：
```
夜幕降临，华灯初上...
暮色四合...
晚餐时分...
```

**跨天**：
```
一夜好眠，第二天早早醒来...
清晨的第一缕阳光...
新的一天，新的期待...
```

### 4. 表情符号使用指南

**适度原则**：每段不超过 2-3 个表情符号

**使用场景**：

| 场景 | 推荐表情 |
|------|---------|
| 风景 | 🌄 🏔️ 🌊 🌅 🌈 |
| 美食 | 🍜 🍲 🍰 🍵 🥘 |
| 交通 | ✈️ 🚗 🚄 🛵 🚲 |
| 住宿 | 🏨 🏠 🏕️ |
| 情绪 | 😊 😍 🤔 😅 😭 |
| 时间 | ⏰ 🌞 🌙 ⭐ |

## 示例模板

### 文艺风格完整示例

```markdown
# 【云南·七日漫游】风花雪月与烟火人间

## Day 1 · 出发：从北京到大理
> 2026-04-03 ｜ 起点：北京大兴机场

晨光熹微，飞机缓缓升起，将城市的喧嚣抛在身后。舷窗外，云层像白色的波浪，一波波涌来。三个小时后，云贵高原的山脊线在脚下蔓延，那是一种从未见过的苍茫。

![机场外的蓝天](./uploads/image_001.jpg)

下午落地大理，阳光刺眼但凉爽，空气里飘着淡淡的花香。客栈老板说今晚有篝火晚会，迫不及待了！

---

## Day 2 · 洱海骑行：蓝色的梦
> 2026-04-04 ｜ 地点：洱海

租了一辆电动车，沿着环海西路慢慢骑行。风很大，头发被吹得乱七八糟，但心情好得像这蓝得不像话的天。

![洱海的蓝](./uploads/image_002.jpg)

> 🌟 **今日高光**
> 在双廊的客栈阳台，看着夕阳一点点沉入洱海，天边的云从金黄到橙红，再到紫色。这一刻，理解了什么是"风花雪月"。

🎵 *语音记录：晚上在古城吃到了烤乳扇，酸酸甜甜，老板说是当地特色*

---

## Day 3 · 古城漫步：时光倒流的街巷
> 2026-04-05 ｜ 地点：大理古城

清晨的古城格外安静，青石板路被露水打湿，反射着晨光。五华楼下，几个老人在打太极，动作舒缓而优雅。

🍜 **寻味之旅**
在人民路的小巷里，发现了一家不起眼的米线店。老板娘是个白族阿姨，笑起来眼睛弯弯的。她说这碗"酸辣鱼米线"是祖传的配方，一口下去，鲜、酸、辣在舌尖炸开，瞬间爱上。

---

## 后记

结束于2026-04-09，从昆明长水机场返程。

七天的记忆像风一样自由，像花一样绚烂，像雪一样纯净，像月一样温柔。大理，我们还会再见的。✨
```

# 【成都·暴走三天】火锅、熊猫和我的腿

## Day 1 · 初到成都：被辣哭的开始
> 2026-04-03

下了飞机，第一件事：找火锅。导航显示步行 800 米，我天真地以为很快。

30 分钟后，我终于找到了。我的腿：你是不是对 800 米有什么误解？

点了一个"微微辣"的鸳鸯锅。

第一口下去，我流下了感动的泪水。
服务员：怎么了？
我：太辣了...
服务员：这已经是微微辣了啊。
我：你们四川人的微微辣，是四川的微微，还是我的微微？

![火锅](./uploads/spicy_hotpot.jpg)

---

## Day 2 · 熊猫基地：被萌出血的一天
> 2026-04-04

早上 7 点起床去看熊猫。我：为了国宝，拼了！

到了基地发现：熊猫还在睡觉。
也是，人家才是真正的"国宝级待遇"，想睡就睡。

后来总算看到一只醒着的，正在吃竹子。它看了我一眼，继续吃。

## Day 3 · 宽窄巷子：游客的自我修养
> 2026-04-05

宽窄巷子，听名字很诗意。
去了才发现：人山人海。

在窄巷子里被挤得双脚离地，实现了"自动前进"。

但还是买了三大袋特产，真香。😂

## 后记

三天走了 35 公里，吃了 5 顿火锅，瘦了 2 斤（震惊）。
成都，你赢了。我下次还来。🔥

## 质量检查清单

在生成游记后，检查以下要点：

- [ ] 标题有吸引力，包含地名和主题
- [ ] 每天都有明确的地点/主题标注
- [ ] 时间线清晰，素材按顺序排列
- [ ] 图片、视频、语音都有相应的文字描述
- [ ] 添加了趣味元素（表情、引用块、小标题）
- [ ] 语言风格统一，符合用户偏好
- [ ] 没有明显的错误或矛盾
- [ ] 后记有总结和感悟
- [ ] 整体可读性强，段落适中

## 常见问题处理

### Q: 素材太少怎么办？

如果某天的素材少于 3 条，可以：
1. 合并相邻日期（如果内容相似）
2. 简化该天的描述
3. 以"片段"形式呈现，而非独立章节

### Q: 时间顺序混乱怎么办？

1. 优先使用系统时间戳排序
2. 如果用户指定了相对时间（"昨天下午"），推算具体时间
3. 无法确定时，按用户叙述顺序排列

### Q: 缺少地点信息怎么办？

1. 从图片 EXIF 中提取 GPS 信息
2. 从文本内容中推断（"古城"、"洱海边"）
3. 使用"在路上"等模糊表述

### Q: 用户没有明确风格偏好？

默认使用"文艺"风格，在生成后询问：
"喜欢这个风格吗？我还可以换成幽默、简洁或详细风格。"
