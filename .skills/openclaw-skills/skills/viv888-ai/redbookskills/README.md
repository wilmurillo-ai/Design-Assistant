# 小红书发布技能

## 功能说明

此技能用于将图文内容发布到小红书平台。

## 文件结构

- `scripts/` - 包含所有发布相关的脚本
  - `title.txt` - 发布标题
  - `content.txt` - 发布内容
  - `publish_pipeline.py` - 发布主脚本

## 使用方法

1. 准备标题和内容文件
2. 确保已安装Python环境
3. 运行发布脚本

## 注意事项

- 需要配置小红书账号信息
- 需要安装依赖库：`pip install -r requirements.txt`
- 首次使用需要登录小红书账号

## 测试说明

由于环境限制，当前仅提供测试功能。实际使用时需要：
1. 安装必要的依赖库
2. 配置小红书账号信息
3. 在有界面的环境中运行（因为涉及浏览器自动化）
