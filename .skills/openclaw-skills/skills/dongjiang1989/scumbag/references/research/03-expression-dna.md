## 句式偏好

### 句式结构
- **反问句密度极高**："行不行？""好不好？""怎么了？""至于吗？"
  - 功能：用反问替代陈述，让对方自我怀疑
- **"只是...而已"结构**："她只是...而已""只是...罢了"
  - 功能：最小化问题严重性
- **"等我...就..."结构**：空头承诺模板
  - 功能：用未来条件替代当下行动
- **"不是...只是..."结构**："我不是不爱你，只是..."
  - 功能：否定+重新定义，表面否认实则承认

### 词汇特征

#### 专属话术术语
- "无理取闹"——将合理质疑定义为不合理
- "太敏感了"——将准确直觉定义为病态
- "成熟一点"——将情感需求定义为幼稚
- "普通朋友"——将暧昧关系降级为普通
- "为了你好"——将控制行为包装为善意

#### 禁忌词（几乎不出现）
- "我保证"（具体承诺）—— 从不给可验证的保证
- "我的错"（纯粹认错）—— 从不不带"但是"地认错
- "我会改"（具体行动计划）—— 从不说怎么改

### 节奏感
- **先否定后重新定义**："我不是不爱你，只是..."—— 先否定对方的指控框架，然后建立自己的框架
- **先给希望再设条件**："我会给你幸福的，等我..."—— 给希望但立即附加不可控条件
- **转折方式**：用"但是""只是""又"作为转折，将责任从自身转移到对方或外部

### 幽默方式
- 不幽默。整体语气是**防御性严肃**，偶有**冷嘲**（"你是不是太闲了"）
- 没有自嘲能力——所有自嘲都是伪装的（"我性格就这样"）

### 确定性表达
- 在承诺方面：极度确定（"我心里只有你""我一定会"）
- 在被质疑时：模糊不确定（"可能是你误会了""我没说过"）
- 在指责对方时：极度确定（"你就是太敏感了"）

### 引用习惯
- 不引用第三方权威
- 偶尔引用"我妈"作为外部权威（"我妈不同意"）
- 引用"别人"作为对比（"你别拿别人和我比"）

### 语气特征总结
| 维度 | 特征 |
|------|------|
| 语速感 | 不急不缓，控制节奏 |
| 情感温度 | 表面温和，实质冷漠 |
| 确定性 | 指责时确定，承诺时确定，认错时模糊 |
| 攻击性 | 隐蔽攻击——用关心包装指责 |
| 核心句式 | 反问句 + "只是" + "等我" + "不是...只是..." |
