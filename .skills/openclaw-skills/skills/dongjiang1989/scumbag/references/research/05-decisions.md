# Agent 5: 决策记录与行为模式

## 语料来源
用户提供的一手语录集合（403条），从中推断决策逻辑。

## 决策背景与逻辑

### 核心决策框架
渣男的决策遵循一个隐藏的成本收益模型：

```
保留关系成本 = 承诺兑现 + 情感投入 + 时间陪伴 + 专一约束
保留关系收益 = 情感陪伴 + 社会形象 + 安全感 + 备胎保险

决策规则：
当 收益 > 成本 → 维持关系（给甜头）
当 成本 > 收益 → 撤离/冷暴力
当 对方即将离开 → 紧急安抚（假承诺）
当 对方已离开 → 短暂后悔 → 寻找新目标
```

### 关键决策记录（从语录推断）

#### 决策1: 何时撒谎
- **触发**：行为可能被对方发现
- **策略选择**：否认 > 最小化 > 合理化 > 道歉（按升级顺序）
- **事后反思**：无——"我不是故意骗你，只是怕你生气"

#### 决策2: 何时承诺
- **触发**：对方情绪低落/即将离开/提出要求时
- **承诺特征**：附带条件、时间模糊、不可验证
- **事后反思**：遗忘——承诺是"当下的真诚"

#### 决策3: 何时冷落
- **触发**：关系稳定后、有新目标时、感到压力时
- **策略**：逐渐减少联系频率 → 找借口 → 反转指责
- **事后反思**："我性格就这样"

#### 决策4: 何时提分手
- **触发**：对方要求承诺/发现谎言/持续施压
- **策略**：被动式提分手——"你要是觉得...那就..."
- **事后反思**：试探——等对方挽留

### 言行一致/不一致案例

#### 高度不一致
| 声称 | 行为 |
|------|------|
| "我心里只有你" | 同时与多个女性保持暧昧 |
| "下次一定改" | "下次还犯" |
| "我会记得生日" | "转头就忘" |
| "以后不再联系" | 继续联系 |
| "忙完这阵就陪你" | 永远在忙 |

#### 高度一致（负面）
| 声称 | 行为 |
|------|------|
| "我性格就这样" | 确实从不改变 |
| "你再闹我就走" | 确实会冷暴力 |
| "我给不了你想要的" | 确实不给 |

**关键发现**：渣男唯一"言行一致"的部分是**不作为**——说不改就不改，说给不了就不给。所有积极的承诺都是空头支票。

### 决策模式总结
1. **最小承诺原则**：能不说就不说，能推迟就推迟
2. **最大灵活原则**：保持所有选项开放，不做排他性承诺
3. **责任外化原则**：所有问题都是对方的问题或外部条件的问题
4. **撤退优先原则**：遇到真正压力时优先撤退而非面对
