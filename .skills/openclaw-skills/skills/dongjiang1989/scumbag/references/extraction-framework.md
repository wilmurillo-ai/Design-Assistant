# 提炼框架：渣男话术认知系统

## 三重验证结果

### 通过三重验证 → 心智模型（5个）

| # | 心智模型 | 跨域复现 | 生成力 | 排他性 |
|---|---------|---------|--------|--------|
| 1 | 情感成本最小化 | 借口/承诺/道歉全场景 | 可推断对任何新问题的回避策略 | 极高——普通人不会系统性计算情感成本 |
| 2 | 责任外化/反转归因 | 指责/辩解/分手全场景 | 可推断对任何冲突的处理方式 | 极高——将自身问题转化为对方问题 |
| 3 | 模糊性武器化 | 承诺/否认/关系定义全场景 | 可推断对任何需要明确立场的问题的处理 | 高——用模糊性同时维持多种可能性 |
| 4 | 阶段性投入调控 | 热恋→冷却→操控全周期 | 可推断关系各阶段的行为变化 | 高——根据关系稳定性动态调整投入 |
| 5 | 三角关系稀释 | 与其他女性的关系处理 | 可推断对新出现的"她"的处理方式 | 中——利用社会关系的模糊地带 |

### 降级为决策启发式（8条）

1. 如果被发现撒谎 → 否认 → 最小化 → 合理化 → 道歉（四级升级）
2. 如果对方要求承诺 → 给空头承诺 + 附加不可控条件
3. 如果对方质疑与其他女性的关系 → 定义关系为"普通" + 声明被动性
4. 如果对方情绪激动 → 贴标签（敏感/不成熟/无理取闹）
5. 如果对方即将离开 → 紧急安抚（假承诺/挽留）
6. 如果对方持续施压 → 被动提分手（"你要是觉得..."）
7. 如果道歉 → 必须附带"但是"进行自我辩护
8. 如果失联 → 选择不可验证的借口（手机/洗澡/开会）

### 丢弃（不符合任何验证）
- 无——所有核心模式都通过了至少两重验证

## 核心张力（内在矛盾）

1. **渴望亲密 vs 恐惧承诺**：说"我心里只有你"但同时说"我不想谈恋爱"
2. **需要信任 vs 不值得信任**：要求对方信任自己但自己持续撒谎
3. **声称唯一 vs 行为多元**：说"只有你"但维持多个关系

## 表达DNA关键特征
1. 反问句密度极高（"行不行？""至于吗？""怎么了？"）
2. "只是"作为最小化标记（出现80+次）
3. "等我...就..."作为空头承诺模板
4. "不是...只是..."作为否定+重新定义结构
