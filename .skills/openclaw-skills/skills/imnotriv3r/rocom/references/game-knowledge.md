## 印记系统

每方（我方/敌方）同时只能存在1种正面印记和1种负面印记，新印记覆盖同类型旧印记，双方独立。

**正面印记**：湿润(-1能耗)、龙噬(+30%双攻可叠)、蓄势(+30%威力+1能耗可叠)、风起(先手+20%威力)、蓄电(+10威力)、光合(回合+1能量)、攻击(+10%威力)

**负面印记**：减速(-10速度)、降灵(入场-1能量)、星陨(额外幻系伤害可叠)、中毒(3%生命毒伤可叠)、棘刺(入场-6%生命)

**配队建议**：光合印记续航+高能耗精灵；风起印记+高速先手链；负面印记配合速秒（棘刺）或消耗（中毒）

## 种族值解读

- 总和 = 生命+物攻+魔攻+物防+魔防+速度，最终形态一般 500-700+
- 物攻型(物攻>魔攻)配物攻技能，魔攻型(魔攻>物攻)配魔攻技能
- 速度决定出手顺序，PVP 极其重要
- 肉盾型：双防高+生命高，配变化/防御技能

## 性格选择

25种性格中20种有加成削弱，5种无加成。选择原则：
- 输出精灵：加攻击/速度的性格（固执/保守/胆小/开朗）
- 肉盾精灵：加防御的性格（大胆/沉着/慎重）
- 用 `nature search <属性名>` 查哪些性格加某个属性

## 配队思路

1. **属性覆盖**：队伍覆盖多种属性，避免被单一属性克制全队
2. **攻防兼备**：至少1-2只高速输出 + 1只肉盾/辅助
3. **印记协同**：利用印记系统，选能配合的精灵
4. **血脉搭配**：血脉魔法（如光合治愈）可全队受益
5. **PVE vs PVP**：PVE 讲究单点爆发，PVP 讲究联防和速度线

## 新手推荐方向

- 优先练最终形态精灵（种族值高）
- 关注特性（如烈火战神"爆燃"：火系技能后双攻+30%）
- 图鉴课题奖励课题技能石
- 首领形态通常强于原始形态

## 阵容分析指南

用户问"XX怎么配队"或"推荐阵容"时：
1. `formation search` 找相关阵容
2. `formation detail` 看完整配招/性格/个体值/打法
3. 结合属性克制分析优劣
4. 没有现成阵容 → `pet list --element X` 找同系精灵推荐
