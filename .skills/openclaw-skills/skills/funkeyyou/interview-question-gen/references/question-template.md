# Interview Question Template

## Full Question Structure

Below is the complete question structure with notes for each section.

### 一、破冰与自我介绍

1. 能简单介绍一下自己的经历，以及为什么会对[岗位名称]这个方向感兴趣？
2. [针对简历中的跨度/空档期/特殊经历设计追问]

### 二、结合简历的深挖问题

For each employer, create 1–2 targeted questions:
- Reference specific projects, metrics, and outcomes from the resume
- Focus on "how did you do X" rather than "did you do X"
- Probe for ownership: was the candidate driving, executing, or assisting?
- For KOL/media experience: ask about volume managed, selection criteria, ROI definition
- For event/campaign experience: ask about full lifecycle (planning → execution → post-mortem)

### 三、活动运营能力考察

4. [Scenario question] 假设我们要在APP里上线一个为期7天的限定活动，核心玩法是[结合产品当前主要玩法]。你来负责这个活动，从立项到上线，你会关注哪些环节和风险点？
5. 活动上线后，你会用哪些数据指标来判断这个活动是否成功？如果数据表现不好，你会怎么快速排查问题？
6. 你做过数据分析/汇报类工作——能举一个你用数据发现问题并推动改进的例子吗？
7. 活动运营中经常要和设计、开发、产品跨部门协作，你有没有遇到过协作不顺畅的情况？你是怎么推进的？

### 四、日语与本地化能力

8. WePlay是日服产品，用户全是日本人。你觉得中国团队做日本本地化活动，最容易踩的坑是什么？
9. 如果你来负责活动文案的日语本地化，你会怎么确保文案既符合日语表达习惯，又能传递活动的核心情绪？能举一个你觉得本地化做得好/差的例子？
10. 日本用户在隐私保护和「身バレ」问题上比较敏感，你认为在活动设计和用户运营中，有哪些地方需要特别注意？

### 五、WePlay 产品体验问题

> Note: Add a callout block: "以下问题假设候选人已在面试前下载并体验 WePlay"

11. 你体验了WePlay之后，整体的第一印象是什么？如果用三个词来形容，你会怎么形容？
12. 语音房是WePlay的核心场景，你进语音房体验之后，觉得作为一个新用户，最难受的是哪一步？有没有觉得"这里如果做成XX就好了"的地方？
13. WePlay有一套礼物系统，包括小礼物爆大礼物、盲盒、抽奖页等机制。你觉得这套设计对用户的吸引力在哪里？有没有机制设计上的缺陷或可以优化的地方？
14. 从活动运营的角度，你觉得目前WePlay活动入口和活动介绍页面的设计，有没有让你觉得"参与动线不清晰"或者"不知道自己在做什么"的地方？
15. WePlay在日本面对REALITY、TopTop等竞品。你自己用下来之后，觉得WePlay最明显的差异化优势是什么？最需要补的短板是什么？

### 六、价值观与潜力考察

16. [Gamescom/展会/特殊经历追问，根据简历定制]
17. 活动运营有时候会遇到"资源少、时间紧、跨部门推进难"的三重压力，你有没有经历过这种情况？你是怎么应对的？
18. 如果来WePlay，你觉得你过往经历中哪个点最能直接用上？哪个点是你觉得自己需要补的？
19. 你有没有什么想问我们的？

### 七、日本語口頭試問

> Important: All questions in this section must be written entirely in Japanese.

**基礎（ウォームアップ）**

25. 30秒程度で自己紹介をしてください。
26. 前職での主な業務内容を日本語で説明してください。

**業務場面シミュレーション**

27. 日本のKOLにコラボを依頼するメールを送ると想定して、メールの書き出しと主な内容の要点を日本語で話してください。
28. アプリ内のガチャイベントで「確率の仕組み」をユーザーに説明するお知らせ文を日本語で作ってみてください。
29. 日本のユーザーがコミュニティで「イベントのルールがわからない」と不満を書き込んでいます。日本語でどのように返信してフォローしますか？

**自由回答**

30. WePlayを実際に触ってみた感想を、日本語で自由に話してください。
