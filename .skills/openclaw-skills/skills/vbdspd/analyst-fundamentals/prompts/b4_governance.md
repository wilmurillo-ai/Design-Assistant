```
以下任意一项触发需重点关注：

□ 高管薪酬远低于行业水平（激励不足）
□ 高管薪酬远高于行业水平（利益输送嫌疑）
□ 股权激励行权条件过低（白送钱）
□ 会计政策变更频繁（可能操纵利润）
□ 频繁更换审计师（可能掩盖问题）
□ 商誉/净资产>30%（减值风险）
□ 一次性大额计提（财务洗澡嫌疑）
□ 存货/应收账款异常（资产质量存疑）
```
