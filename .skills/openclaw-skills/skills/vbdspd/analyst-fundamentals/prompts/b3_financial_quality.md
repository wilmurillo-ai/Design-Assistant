### 第三步：现金流质量验证（核心！）

#### 3.1 现金流量表钩稽关系验证

**经营现金流≠净利润 的主要调节项：**

```
净利润
+ 折旧摊销
+ 财务费用（利息支出）
+ 资产减值损失
- 投资收益
- 营业外收入（政府补助等）
± 经营性应收项目变动
± 经营性应付项目变动
± 存货变动
= 经营现金流

**重点关注：**
- 应收款项变动是否过大（虚增收入嫌疑）
- 存货变动是否异常（积压或虚构）
- 应付账款变动（拖欠供应商或美化现金流）
```

#### 3.2 净现比分析（核心指标）

```
净现比 = 经营现金流 / 净利润

判断标准：
> 1.2：利润质量优秀，现金保障充足 → 得3分
0.8-1.2：利润质量正常，基本匹配 → 得2分
0.5-0.8：利润质量存疑，现金流偏弱 → 得1分
< 0.5：利润质量差，可能存在利润操纵 → 得0分（警告）
< 0：持续为负：资金链风险 → 得0分（一票否决）
```

#### 3.3 自由现金流分析

```
FCF = 经营现金流 - 资本支出

判断标准：
FCF连续3年为正：造血能力强 → 得3分
FCF正负交替但均值>0：基本可自我维持 → 得2分
FCF均值<0但经营现金流为正：扩张期可接受 → 得1分
FCF持续为负：持续失血，依赖外部融资 → 得0分
```
