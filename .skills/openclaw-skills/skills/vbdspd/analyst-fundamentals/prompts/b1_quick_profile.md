### 第二步：构建快速画像（核心分析）

#### 2.1 财务快照（近3年）

从原始数据中提取并计算：

```
营收（近3年）= [数值] / [增长率]

净利率 = [数值]

总市值 = [数值]
流通市值 = [数值]
股东人数（最新）= [数值]
人均持股金额 = 流通市值/股东人数
```

#### 2.2 投资信号快速扫描

**绿灯信号（每项+1分）：**
```
□ ROE近3年均 > 15%
□ 毛利率 > 30%
□ 营收连续3年正增长
□ 净利润连续3年正增长
□ 股价<50元（中小盘有弹性）
□ 人均持股金额>20万（筹码集中）
□ 市值100-1000亿（机构主战场）
```

**红灯信号（每项-1分，出现任意一项需在报告中重点标注）：**
```
□ ROE近3年任何一年 < 5%
□ 净利润连续2年亏损
□ 经营现金流连续2年为负
□ 毛利率持续下降（近3年降幅>10个百分点）
□ 资产负债率 > 70%
□ 股东人数连续3期增加（筹码分散）
□ 人均持股金额<5万（散户化）
□ 市值<30亿（流动性风险）
```

#### 2.3 快速估值判断

```
PE(TTM) = 当前股价 / 近4季度EPS
判断：<0 = 亏损 | 0-15 = 偏低 | 15-30 = 合理 | 30-50 = 偏高 | >50 = 高估
PB = 当前股价 / 每股净资产
判断：<1 = 破净 | 1-3 = 正常 | 3-5 = 偏高 | >5 = 高净资产溢价
```
