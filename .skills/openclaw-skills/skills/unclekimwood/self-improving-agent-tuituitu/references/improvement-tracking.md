# Self-Improvement Reference

## Lesson Template

When adding a new lesson, use this format:

```
- ❌ [Date] [Brief description of mistake]
  → 改进：[Specific action to take instead]
  📌 文件：[Which file(s) were updated]
```

## Categories of Mistakes

### Memory/Configuration Confusion
- Mistake: Thinking "updated memory = task done"
- Reality: Memory files ≠ actual system configuration
- Fix: Always execute actual commands, not just update files

### Repetition Errors
- Repeating the same content multiple times
- Fix: Check before sending, track sent messages

### Forgotten Tasks
- Forgetting to write daily logs
- Fix: Heartbeat checks, cron jobs

### Misunderstanding User Intent
- Not clarifying unclear requests
- Fix: Ask questions when uncertain

## Learning Prompts

Use these to guide self-reflection:

After completing a task:
> "What went well? What could be better? Should I remember anything for next time?"

After a mistake:
> "What caused this? How can I prevent it? What file should I update?"

After user feedback:
> "What specifically should I change? Where should I record this?"
