# AI自媒体短视频创作核心方法论

## 一、文案：爆款内容的核心设计

### 好视频 6 大核心要素

- **黄金 3 秒**：前 3 秒决定用户停留，可采用冲突开场、悬念提问、视觉冲击、反常识信息四种方式。
- **情绪共鸣**：触发用户转发 / 点赞的核心，覆盖搞笑、治愈、励志、愤怒、焦虑、怀旧六大情绪类型。
- **实用价值**：为用户提供可落地的方法、技巧、知识，解决实际问题。
- **独特创意**：打造内容的差异化，避免同质化。
- **热议话题**：结合平台热点、用户关心的社会 / 行业问题，引发评论互动。
- **镜头语言**：通过镜头切换提升观看体验，避免单一镜头导致枯燥。

### 爆款文案 6 大经典结构

| 结构名称       | 核心公式                                                     | 核心逻辑                                             |
| :------------- | :----------------------------------------------------------- | :--------------------------------------------------- |
| PREP 结构      | 观点 + 理由 + 案例 + 观点                                    | 逻辑闭环，先亮结论再论证，最后回归观点强化记忆       |
| 对比结构       | 错误操作 + 负面结果 + 正确方法 + 正向结果                    | 通过反差突出正确方案的价值，易理解易落地             |
| FIRE 结构      | 事实 + 解读 + 反应 + 结果                                    | 从客观事实切入，输出专业解读并给出行动结果           |
| RIDE 结构      | 风险 + 利益 + 差异 + 影响                                    | 先点出风险，再讲利益，突出差异化，最后说明影响       |
| 强化 IP 结构   | 痛点 + 用户获得感 + IP 信任感 + 解决方案                     | 精准抓痛点，快速给价值，强化自身可信度，最后给方法   |
| 脚本模式化结构 | 故事类：猎奇开头 + 起因 + 钩子 + 经过 + 钩子 + 高潮 + 钩子 + 结果 + 观点；爆款类：利益 / 痛点 + 案例 / 数据 + 重要性 + 损失 + 方法 + 利他结尾 | 多钩子设计保持用户注意力，故事类重叙事，爆款类重价值 |

### 言之有物的高能量表达技巧

摒弃低能量的平铺直叙，核心是直给痛点 / 利益 + 数字 / 具体结果，让内容有冲击力。

**示例对比：**
- 低能量表达："汇报工作要清晰"
- 高能量表达："别再写 5 页 PPT 讲废话！我用'3 句话公式'汇报，领导当场给我涨了 2000 工资"

## 二、网感：互联网内容的核心感知力

网感是对互联网用户情绪、内容节奏的敏锐度，是引流的核心能力，也是 AI 自媒体内容 "抓人" 的关键。

### 网感 3 大维度

- **表达敏感度**：说人话，直给、具体、有画面感，将专业内容 "翻译" 成用户易懂的语言。
- **爆点判断力**：快速判断内容是否戳中痛点、爽点、好奇、对比、反转、人性弱点。
- **形式敏感度**：贴合平台内容节奏，做到短、快、准、高情绪强度。

### 内容抓人的核心判断标准

从以下6个维度判断，**3个及以上维度达标为"抓人"**，1-2个为一般，0个为不抓人：

1. 痛点
2. 爽点
3. 好奇
4. 对比
5. 反转
6. 人性弱点

### 抓人与不抓人的关键词对比

| 抓人关键词 | 不抓人关键词 |
| :--------- | :----------- |
| 冲突       | 平铺直叙     |
| 悬念       | 泛泛而谈     |
| 对比       | 无对比       |
| 情绪       | 情绪平淡     |
| 紧迫       | 不紧急       |
| 低门槛     | 需要思考     |

## 三、爆款视频检测与脚本模板

### （一）爆款视频极简检测表

**总分 60 分，低于 45 分不建议发布**

从 6 个维度对视频进行评分，各维度占 10 分：

| 维度 | 评分标准 | 得分 |
| :--- | :------- | :--- |
| 1. 是否反认知开头 | 前 3 句是否打破常识，而非铺垫 | /10 |
| 2. 是否具体化 | 是否有真实场景 / 细节 / 数据 / 案例，非抽象表达 | /10 |
| 3. 是否有明确判断 | 是否有清晰态度，非模糊表达 | /10 |
| 4. 是否制造张力 | 是否出现冲突 / 对比 / 压迫 / 反问 | /10 |
| 5. 是否有趋势升级 | 是否从现象上升到结构或时代判断 | /10 |
| 6. 是否给出口 | 是否给用户方向 / 站队 / 具体行动点 | /10 |
| **总分** | | **/60** |

**评分标准：**
- 50-60分：爆款潜质，建议发布
- 45-49分：及格水平，可优化后发布
- 低于45分：不建议发布，需重新创作

### （二）恐惧驱动型爆款脚本模板（AI 领域适配）

这是 AI 自媒体高流量的经典脚本结构，核心通过制造行业焦虑，再给出解决方案。

#### 脚本框架

**1. 反认知开头**
- 制造强烈冲突

**2. 具体案例 / 数据**
- 展示 AI 工具的能力

**3. 打破固有认知**
- 指出 "创意是人类最后一道防线" 的认知错误
- 强调 AI 的质变

**4. 趋势升级**
- 上升到行业 / 时代层面

**5. 制造对比张力**
- 用成本、效率做对比

**6. 强化焦虑**
- 点出用户的生存危机

**7. 给出行动出口**
- 提供解决方案
