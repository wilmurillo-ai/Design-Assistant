# 抖音AI视频脚本生成器

## 概述

本技能帮助用户生成适合抖音平台的AI主题短视频脚本。核心特点：

1. **大模型直接生成** - 由AI助手基于《AI自媒体短视频创作核心方法论》自由创作，无代码框架限制
2. **自动热点搜索** - 未提供主题时，自动获取当前热门AI话题
3. **质量评分检测** - 使用爆款视频检测表（60分制）评估脚本质量
4. **高质量输出** - 目标评分≥45分，确保脚本具备爆款潜质

## 工作流程

```
用户请求生成脚本
    ↓

    ↓ 否                    ↓ 是

    ↓

```

## 使用方法

### 场景1：用户指定主题

用户说："帮我生成一个关于AI编程助手的抖音脚本"

执行步骤：
1. 直接使用用户提供的主题
2. **大模型基于方法论自由生成**：
   - 参考《AI自媒体短视频创作核心方法论》
   - 使用恐惧驱动型爆款脚本模板（7步骤）
   - 确保6大维度达标
3. **自评质量**：使用爆款视频检测表评分
4. **保存脚本**：保存到项目根目录

### 场景2：用户未指定主题（自动获取实时热点）

用户说："帮我生成一个抖音AI视频脚本"

执行步骤：
1. **实时搜索AI热点**：使用 `search_web` 工具搜索 "2026年2月AI人工智能最新热点"
2. **解析热点话题**：从搜索结果中提取最热门的AI话题
3. **大模型基于方法论生成**：围绕热点自由创作脚本
4. **保存脚本**：评分后保存到项目根目录

### 场景3：查看当前AI热点

用户说："最近有什么AI热点话题？"

执行步骤：
1. 使用 `search_web` 搜索 "AI人工智能最新热点新闻"
2. 解析并展示热点列表
3. 供用户选择或自动生成脚本

## 脚本生成方法

### 核心方法论参考

**恐惧驱动型爆款脚本模板（7步骤）**：

1. **反认知开头** - 制造强烈冲突（黄金3秒）
2. **具体案例/数据** - 展示AI工具能力
3. **打破固有认知** - 指出传统观念错误
4. **趋势升级** - 上升到行业/时代层面
5. **制造对比张力** - 成本效率对比
6. **强化焦虑** - 点出生存危机
7. **给出行动出口** - 提供解决方案

### 爆款视频检测表（60分制）

| 维度 | 分值 | 检测标准 |
|------|------|----------|
| 反认知开头 | 10分 | 前3句是否打破常识 |
| 具体化程度 | 10分 | 是否有场景/数据/案例 |
| 明确判断 | 10分 | 是否有清晰态度 |
| 张力制造 | 10分 | 是否有冲突/对比/反问 |
| 趋势升级 | 10分 | 是否上升到时代判断 |
| 行动出口 | 10分 | 是否有具体行动引导 |

**评分标准：**
- 50-60分：爆款潜质
- 45-49分：及格水平
- <45分：需重新创作
