## 二、核心功能

### 2.1 FlyAI真实票价查询

**技术亮点**：
- 集成飞猪实时数据API
- 真实航班票价（无模拟数据）
- 真实预订链接（可点击验证）

**示例数据**：
- 北京→香港：CX393 ¥1605
- 预订链接：https://a.feizhu.com/0bUklm

---

### 2.2 高德vs腾讯双源验证

**技术亮点**：
- POI数量对比（验证一致性）
- 路径距离对比（误差<10%）
- 双源数据交叉验证

**创新点**：
- 防止单一数据源偏差
- 提高推荐准确性

---

### 2.3 美团美食+麦当劳兜底

**技术亮点**：
- 美团优先推荐优惠套餐
- 麦当劳兜底保证可用性
- 预订链接一键生成

---

### 2.4 地图截图生成

**技术亮点**：
- 高德静态地图API
- 实时生成目的地截图
- 可嵌入HTML攻略

---

## 三、技术架构

```
前端: Flask + HTML
后端: Python 3.8
MCP集成: FlyAI + 高德 + 腾讯 + 美团 + 麦当劳
数据库: 无（实时API调用）
```
