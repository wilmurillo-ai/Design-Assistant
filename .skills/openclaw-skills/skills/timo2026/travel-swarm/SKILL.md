# openclaw-travel-swarm-v8

多MCP集成智能旅行规划系统 - FlyAI+高德+腾讯+美团+麦当劳

## 功能

- FlyAI真实票价查询（飞猪实时数据）
- 高德vs腾讯POI验证
- 美团美食推荐
- 麦当劳兜底方案
- 地图截图生成

## 使用

触发口令：
- 旅行规划
- 旅游攻略
- 订票
- 美食推荐

## 安装

```bash
clawhub install openclaw-travel-swarm-v8
```

## 配置

需要配置以下API密钥：
- FLYAI_API_KEY
- GAODE_API_KEY
- TENCENT_MAP_KEY（可选）
- MEITUAN_API_KEY（可选）
