## 正文

### 背景

作为一名制造业从业者，我见证了太多因报价失误导致的损失：
- 材料计算错误，亏损 30%
- 工艺冲突未发现，订单取消
- 交期评估不准，违约赔偿

传统报价依赖老师傅经验，但人的精力有限，错误在所难免。

### 解决方案

我花了 6 个月时间，从零构建了一套 **CNC 智能报价系统**：

```
输入: 材料 + 尺寸 + 表面处理 + 数量
输出: 报价 + 风险提示 + 优化建议
时间: < 2 秒
准确率: 94% (±10%)
```

### 核心能力

#### 1️⃣ 规则引擎 + AI 混合架构

```
规则引擎 (确定性计算)
├── 材料费 = 体积 × 密度 × 单价
├── 加工费 = 时间 × 时薪 × 复杂度
└── 表面费 = 面积 × 单价 + 附加费

AI 模块 (智能判断)
├── 工艺冲突检测
├── 异常订单识别
└── 优化建议生成
```

#### 2️⃣ RAG 知识增强

- **训练数据**: 1213 条真实报价记录
- **材料库**: 111 种材料参数
- **工艺库**: 11 种表面处理规则

检索准确率: **84%**，置信度: **0.84**

#### 3️⃣ 多通道集成

| 通道 | 用途 | 响应时间 |
|------|------|----------|
| QQ Bot | 即时报价 | < 3 秒 |
| 邮件 | 批量报价 | < 1 分钟 |
| API | 系统集成 | < 2 秒 |

### 实战成果

#### 📊 数据统计

| 累计报价 | 1213 单 |

#### 🎯 典型案例

**案例 1: 工艺冲突检测**

```
客户要求: 铝合金 + 阳极氧化 + 镀铬
系统检测: ⚠️ 工艺冲突！阳极氧化和镀铬不能同时进行
结果: 及时沟通，避免订单取消
```

**案例 2: 成本优化**

```
订单: 500 件复杂零件
系统建议: 分 10 批生产，批量折扣 12%
结果: 客户省 ¥19,200，我方利润率提升 7%
```

**案例 3: 材料推荐**

```
客户指定: 碳钢 (户外使用)
系统建议: 推荐 304 不锈钢，防腐蚀
结果: 6 个月后客户感谢，成为回头客
```

### 技术栈

- **后端**: Python 3.8 + Flask
- **AI**: DashScope API (Qwen3.5-Plus)
- **存储**: SQLite + JSON
- **集成**: OpenClaw 框架
- **部署**: 阿里云 ECS
