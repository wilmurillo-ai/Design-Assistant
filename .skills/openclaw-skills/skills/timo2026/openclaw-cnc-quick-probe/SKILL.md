# CNC快速探明 Skill

## 触发条件

| 条件 | 说明 |
|------|------|
| 意图 = CNC报价 | 识别到报价请求 |
| 收敛度 < 80% | 参数不完整 |
| 无文件/文件解析失败 | 缺少STEP/PDF |

## 执行流程

```
检测触发条件
    ↓
启动参数收集器
    ↓
生成5参数提问
    ↓
等待用户回答
    ↓
更新收敛度
    ↓
┌─ ≥80% → 调用报价Skill
└─ <80% → 继续追问
```

## 5个关键参数

| # | 参数 | 必需 | 默认值 |
|---|------|------|--------|
| 1 | 材料 | ✅ | 铝合金6061 |
| 2 | 数量 | ✅ | 1件 |
| 3 | 精度 | ✅ | ±0.1mm |
| 4 | 表面处理 | ✅ | 本色 |
| 5 | Ra | ❌ | Ra 3.2 |
