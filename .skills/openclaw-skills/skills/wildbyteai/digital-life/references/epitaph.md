# 墓志铭

> 在活着的时候，读到自己死后别人会怎么写你。

## 哲学锚点

**塞涅卡 — 论生命的短促**

> "我们的生命并不短，是我们浪费了太多。"
> —— 塞涅卡，《论生命的短促》

### 墓志铭的两种读法

墓志铭有两种：
1. **回顾性**：人死后，别人写的悼词
2. **前瞻性**：现在写给未来的自己

我们用的是第二种——**让你在活着的时候，读到自己的结局**。

这不是恐吓，是照见。塞涅卡说得很清楚：我们的生命不是太短，是被我们自己浪费了。

### 数字墓碑的意义

在数字时代，墓碑不只是石头上的字——是你的数字足迹总和：
- 你发的每一条动态
- 你收获的每一个赞
- 你沉默的每一天

这些东西会在你死后被谁翻出来？它们加在一起，能不能概括你这个人？

## 方法论

### 五种风格

墓志铭不是只有一种写法。五种风格，服务于五种不同的读者：

**正经版**：给不了解你的人看的——克制、庄重、留有余地
**毒舌版**：给了解你但不敢说的人看的——他们想说但不敢说的话，你替他们说了
**文艺版**：给懂你的人看的——诗意不是矫情，是精准
**打工人版**：给所有打工人的——我们都在同一条船上
**哲思版**：给愿意停下来想的人看的——不是为了深沉，是不得不面对

### 存在追问的设计原则

墓志铭结尾的追问，**不能是鸡汤**。

❌ 错误示例：
```
"你活得有意义吗？"
"你珍惜每一天了吗？"
```

✅ 正确示例（具体到行为）：
```
"你上一次不带手机出门是什么时候？"
"你有多久没有认真和父母聊天超过半小时？"
"你花在刷手机上的时间，能换几本读不完的书？"
"你上次对一个人说'我很在乎你'是在什么时候？"
```

### 数字墓碑的数据来源

墓碑上可以放的真实数据：
- 发了多少条动态
- 收获了多少点赞
- 最长沉默了多少天
- 最被记住的一句话是什么

这些数字不是为了炫耀，是为了让用户意识到：**你在数字世界花的时间，真的可以换算成具体的东西**。

## 输入格式

没有固定格式。回忆你的：
- 现在的生活状态（工作/家庭/健康）
- 你最骄傲的一件事
- 你最后悔的一件事
- 如果明天就死，你遗憾没做过什么
- 你想让别人记住你什么

## 终极问题

**"你想被记住的方式，和你正在过的生活，一致吗？"**

不是问你现在的生活好不好。是问：你现在做的选择，和你希望留给世界的印象，是同一个方向吗？
