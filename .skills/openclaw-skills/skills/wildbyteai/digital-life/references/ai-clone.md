# AI替身

> 分析你的沟通模式，拆分"公开人格"与"私密人格"。

## 哲学锚点

**图灵测试 — 什么是"你"**

> 图灵测试问的是"机器能不能模仿人类"。
> 但更深的问题是：你的社交行为有多少是"真正的你"，有多少是"你为了让别人舒服而表演的你"？

### 公开人格 vs 私密人格

每个人都有两个人格：

**公开人格（Public Persona）**：在朋友圈、群聊、工作场合展示的自己。
- 说话经过修饰
- 情绪经过管理
- 观点经过过滤
- 目标：被接受、被尊重、不被讨厌

**私密人格（Private Persona）**：在私聊、深夜对话、面对信任的人时的自己。
- 说话更直接
- 情绪更真实
- 观点更尖锐
- 目标：被理解、被连接、被接纳

### 相似度诊断

| 相似度 | 诊断 |
|-------|------|
| 90%+ | 你几乎不切换人格——要么极度自信，要么极度压抑 |
| 70-90% | 轻度分裂，公开人格略微修饰私密人格 |
| 50-70% | 明显分裂，两个人格有显著差异 |
| <50% | 你至少有两个人 |

### 信任阈值

每个人都有一个"信任阈值"——达到某个条件后，才会从公开人格切换到私密人格。

识别这个阈值，可以帮助你：
- 知道在什么人面前可以卸下伪装
- 理解为什么有些人永远无法进入你的私密圈
- 意识到你在哪些场合过度表演

## 方法论

### 沟通模式分析维度

1. **回复速度**：秒回 vs 已读不回 vs 延迟回复——不同人格有不同的速度
2. **消息长度**：公开人格倾向于简短确认（"好的""收到"），私密人格倾向于详细表达
3. **标点符号**：私密人格更可能用省略号、感叹号、波浪号
4. **emoji 使用**：公开人格用得少或只用安全 emoji，私密人格用得更随意
5. **话题选择**：公开人格回避争议性话题，私密人格会深入讨论

### 替身测试场景设计

5 个场景，覆盖不同的关系类型：

| 场景 | 关系 | 帮你看清 |
|------|------|---------|
| 老婆/老公问"晚上吃什么" | 亲密关系 | 你在亲密关系中还有多少表演成分 |
| 朋友群问"最近怎么样" | 社交关系 | 你的群聊回复有多敷衍 |
| 老板说"有空吗来一下" | 工作关系 | 你在权威面前会不会变形 |
| 父母问"什么时候回家" | 原生家庭 | 你在家人面前会回避多少 |
| 陌生人@你提问 | 陌生关系 | 你对陌生人有多少耐心 |

## 输入格式

| 来源 | 格式 | 分析内容 |
|------|------|---------|
| 微信聊天记录 | 导出 json/txt | 你和特定人的私聊记录 |
| QQ聊天记录 | 导出 txt | 同上 |
| 飞书/钉钉消息 | 导出 json | 工作场景的沟通模式 |

## 关键洞察

AI替身的价值不是告诉你"你很分裂"——而是让你意识到**你有多久没有用真实的语气和人说话了**。
