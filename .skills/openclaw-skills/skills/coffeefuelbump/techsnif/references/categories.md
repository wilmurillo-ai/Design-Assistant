# TechSnif Categories

Use the `key` value for `--category` filters. Case-sensitive. Only these values are valid.

| Key | Description |
|-----|-------------|
| `AI` | Artificial intelligence and machine learning — generative AI, LLMs, text-to-image/video, speech recognition, predictive analytics. Companies building AI and the ethical issues it raises. |
| `Startups` | Tech startup news — launches, growth, trajectory across every stage and industry. |
| `Venture` | Venture capital — funding rounds, VC firms, investment trends, valuations, IPOs, and the money behind tech startups. |
| `Robotics` | Robotics, automation, autonomous systems — industrial robots, warehouse automation, humanoid robots, drone technology. |
