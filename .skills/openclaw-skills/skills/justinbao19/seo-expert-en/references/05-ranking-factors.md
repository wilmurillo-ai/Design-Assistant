| Backlinks | 13% |

| Brand Trust | 11% |
| Niche Expertise | 10% |

- Source: "2025 Google Algorithm Ranking Factor Changes"

### 5.2 User Behavior Signals (Critical)

- **Dwell Time**: How long users stay on the page
- **Bounce Rate**: Whether users quickly return to search results
- **Pages Per Visit**: Whether users continue browsing
- Improvement method: Offer free trials so users actually use the tool
  - Source: "An Effective Method to Improve Page Rankings"

### 5.3 Sitelinks Acquisition Conditions

- Usually appear for brand term searches
- Requires good navigation structure
- Important pages need sufficient internal and external links
  - Source: "What Are Sitelinks and Mini Sitelinks?"
