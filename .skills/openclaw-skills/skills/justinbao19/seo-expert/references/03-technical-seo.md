## III. Technical SEO

### 3.1 Sitemap Configuration

- **Multi-language sites**: Need separate Disallow rules for each language
```
Disallow: /people/
Disallow: /ja/people/
Disallow: /fr/people/
Disallow: /ko/people/
```
- Source: "Multi-language Website robots.txt Configuration Guide"

### 3.2 Page Depth

- Best practice: Any page reachable within 3 levels
- Homepage → Category page → Detail page
  - Source: "How to Create SEO-Friendly Web Pages"

### 3.3 Accelerating Site Indexing

- List new content on the homepage to attract crawlers
- Use the "Bean Roasting Method": Detect unindexed pages, continuously expose them on homepage
- Check indexing status with: site:domain/url
  - Source: "Finding Pages with 5.9 Million Monthly Growth"
