## I. Keyword Research

### 1.1 New Keyword Strategy (Core Theory)

- **"New Keywords for New Sites" Strategy**: New keywords have low competition, allowing new sites to rank quickly
  - Source: "Finding New Keywords: An Always-Effective Website Building Strategy"
  
- **Consensus-Driven Search Demand (CDSD)**
  - When users independently search for the same term, consensus forms
  - Stronger consensus = higher search volume
  - Source: "Search Volume Comes from Consensus"

- **How to Identify New Keywords**:
  - Use Google Trends to find when trends started
  - Check intitle search result counts
  - New keyword characteristics: sudden search volume surge, few competing pages
  - Source: "Tutorial: One Trick to Determine When a Keyword Appeared"

### 1.2 KGR Formula (Keyword Golden Ratio)

```
KGR = intitle results / monthly search volume

KGR < 0.25: Easy to rank (blue ocean keywords)
KGR 0.25-1: Medium difficulty
KGR > 1: Difficult (red ocean keywords)
```
- Source: "Using the KGR Formula to Quantify New Keywords and Find Blue Ocean Terms"

### 1.3 Keyword Selection Principles

- **One Keyword Per Page**: Each page should optimize for only one core keyword
- **CPC Reference Value**: High CPC indicates advertisers willing to pay for traffic, but not essential
- **Understand Search Intent**: What does the user want when searching? Content/tools/videos/products?
  - Source: "When Analyzing Keywords, Does CPC Have Reference Value?"
