# 基本面分析框架

## 一、业绩分析

### 核心财务指标

#### 1. 营收分析
| 指标 | 计算方法 | 健康标准 |
|------|----------|----------|
| 营收增速 | (本期营收 - 上期营收)/上期营收 | >15% 优秀，>30% 高增长 |
| 营收质量 | 经营现金流/营收 | >80% 为健康 |
| 营收结构 | 各业务占比 | 核心业务>50% 为佳 |

#### 2. 利润分析
| 指标 | 计算方法 | 健康标准 |
|------|----------|----------|
| 净利润增速 | (本期净利 - 上期净利)/上期净利 | >20% 优秀 |
| 扣非净利润 | 扣除非常性损益后的净利润 | 应接近净利润 |
| 净利率 | 净利润/营收 | 行业对比，稳定或提升为佳 |

#### 3. 盈利能力
| 指标 | 计算方法 | 健康标准 |
|------|----------|----------|
| 毛利率 | (营收 - 成本)/营收 | >30% 较好，>50% 优秀 |
| 净利率 | 净利润/营收 | >15% 较好 |
| ROE | 净利润/净资产 | >15% 优秀，持续>20% 极佳 |
| ROIC | 息前税后利润/投入资本 | >10% 较好 |

#### 4. 现金流分析
| 指标 | 意义 | 健康标准 |
|------|------|----------|
| 经营现金流 | 主业造血能力 | 持续为正且增长 |
| 自由现金流 | 经营现金流 - 资本开支 | 持续为正最佳 |
| 现金流/净利润 | 利润质量 | >1 为健康 |

### 业绩预警信号
- [ ] 营收增速大幅下滑（>50% 降幅）
- [ ] 净利润增速<营收增速（利润率下降）
- [ ] 经营现金流持续为负
- [ ] 应收账款大幅增加
- [ ] 存货周转率下降
- [ ] 毛利率持续下滑

## 二、业务分析

### 业务结构划分

#### 1. 核心支柱业务
- **定义**：贡献主要营收和利润的业务
- **特征**：成熟稳定、市场份额高、现金流好
- **分析要点**：
  - 市场地位（市占率）
  - 竞争壁垒（技术、品牌、渠道）
  - 增长空间（行业天花板）
  - 盈利能力（毛利率、净利率）

#### 2. 增长型业务
- **定义**：高速增长、未来潜力大的业务
- **特征**：增速快、可能亏损、需要投入
- **分析要点**：
  - 增速（营收、用户、订单）
  - 市场空间（TAM）
  - 竞争格局
  - 盈利路径（何时盈亏平衡）

#### 3. 长期布局业务
- **定义**：战略性投入、长期才能见效的业务
- **特征**：投入大、周期长、不确定性高
- **分析要点**：
  - 战略意义
  - 技术/资源积累
  - 投入产出比
  - 风险控制

### 业务分析方法

#### 1. 营收拆解
```
总营收 = 业务 A + 业务 B + 业务 C + ...
分析：
- 各业务占比及变化
- 各业务增速对比
- 增长驱动力识别
```

#### 2. 利润拆解
```
总利润 = 业务 A 利润 + 业务 B 利润 + ...
分析：
- 各业务利润率
- 利润贡献度
- 费用分摊合理性
```

#### 3. 增长驱动分析
- **量增**：销量、用户数、订单数增长
- **价增**：单价、ARPU 值提升
- **新品**：新产品/服务贡献
- **并购**：外延式增长

### 竞争格局分析

#### 1. 市场地位
| 地位 | 市占率 | 特征 |
|------|--------|------|
| 龙头 | >30% | 定价权、品牌优势 |
| 领先 | 10%-30% | 追赶者、差异化竞争 |
| 跟随 | <10% | 价格竞争、生存压力 |

#### 2. 竞争壁垒
- **技术壁垒**：专利、know-how、研发能力
- **品牌壁垒**：知名度、美誉度、忠诚度
- **渠道壁垒**：销售网络、客户关系
- **规模壁垒**：成本优势、资金实力
- **政策壁垒**：牌照、准入限制

#### 3. 竞争态势
- **行业集中度**：CR3、CR5、HHI 指数
- **竞争烈度**：价格战、营销战、人才战
- **进入者**：新进入者威胁
- **替代品**：替代技术/产品威胁

## 三、估值分析

### 估值方法选择

| 公司类型 | 适用方法 | 关键指标 |
|----------|----------|----------|
| 成熟盈利 | PE、PEG | 净利润、增速 |
| 高成长 | PEG、PS | 营收增速、PS |
| 重资产 | PB | 净资产、ROE |
| 周期股 | PB、EV/EBITDA | 资产、现金流 |
| 亏损股 | PS、用户价值 | 营收、用户数 |
| 金融股 | PB、PE | 净资产、坏账率 |

### PE 估值法

#### 适用条件
- 公司持续盈利
- 盈利相对稳定
- 非周期或弱周期行业

#### 计算方法
```
合理 PE = 净利润增速（%）
例如：增速 30%，合理 PE 约 30 倍

PEG = PE / 净利润增速
PEG < 1：低估
PEG = 1-1.5：合理
PEG > 1.5：高估
```

#### PE 分位判断
- 当前 PE 在历史 PE 区间的位置
- 当前 PE 与行业平均 PE 对比
- 结合增速判断合理性

### PB 估值法

#### 适用条件
- 重资产行业（银行、地产、钢铁）
- 周期股底部
- 盈利波动大但资产稳定

#### 计算方法
```
PB = 市值 / 净资产
合理 PB = ROE × 合理 PE
```

#### PB 分位判断
- 当前 PB 在历史 PB 区间的位置
- 当前 PB 与行业平均 PB 对比
- 结合 ROE 判断合理性

### PS 估值法

#### 适用条件
- 高增长但未盈利公司
- SaaS、互联网平台
- 早期成长阶段公司

#### 计算方法
```
PS = 市值 / 营收
合理 PS 参考：
- 增速<20%：PS 1-3 倍
- 增速 20%-50%：PS 3-8 倍
- 增速>50%：PS 8-15 倍
```

### 估值对比分析

#### 1. 历史对比
- 当前估值 vs 历史中位数
- 当前估值 vs 历史极值
- 估值分位计算

#### 2. 行业对比
- 公司 PE/PB vs 行业平均
- 公司增速 vs 行业平均
- 溢价/折价合理性分析

#### 3. 可比公司对比
- 选择 3-5 家可比公司
- 对比 PE、PB、PS、PEG
- 分析差异原因（增速、质量、风险）

## 四、管理层分析

### 管理层评估要点

#### 1. 能力评估
- **战略眼光**：过往战略决策正确率
- **执行能力**：目标达成情况
- **资本配置**：投资、并购、分红决策

#### 2. 诚信评估
- **信息披露**：是否透明、及时
- **承诺兑现**：是否言行一致
- **利益输送**：是否存在损害中小股东行为

#### 3. 激励机制
- **持股情况**：管理层持股比例
- **股权激励**：是否与股东利益一致
- **薪酬结构**：是否与业绩挂钩

### 管理层预警信号
- [ ] 频繁更换高管
- [ ] 高管大幅减持
- [ ] 信息披露不及时
- [ ] 承诺未兑现
- [ ] 关联交易频繁
- [ ] 财务造假历史

## 五、行业分析

### 行业生命周期

| 阶段 | 特征 | 投资策略 |
|------|------|----------|
| 导入期 | 增速快、亏损、不确定性高 | 高风险高回报，小仓位 |
| 成长期 | 高增长、盈利改善、竞争加剧 | 重点配置，享受成长 |
| 成熟期 | 增速放缓、盈利稳定、格局清晰 | 价值投资，分红收益 |
| 衰退期 | 负增长、盈利下滑、退出加速 | 回避或做空 |

### 行业景气度判断

#### 1. 需求端
- 行业增速（营收、销量）
- 下游需求变化
- 渗透率提升空间

#### 2. 供给端
- 产能利用率
- 新增产能情况
- 行业集中度变化

#### 3. 价格端
- 产品价格走势
- 原材料成本变化
- 毛利率趋势

### 政策环境影响
- **支持政策**：补贴、税收优惠、准入放宽
- **限制政策**：产能限制、环保要求、价格管制
- **监管政策**：反垄断、数据安全、行业规范

## 六、基本面分析检查清单

### 买入前检查
- [ ] 营收增速>15% 或有明确改善趋势
- [ ] 净利润增速>营收增速（或合理原因）
- [ ] 毛利率稳定或提升
- [ ] 经营现金流为正
- [ ] ROE>15% 或有提升趋势
- [ ] 负债率合理（<60% 或行业合理水平）
- [ ] 核心业务竞争力强
- [ ] 估值处于合理或低估区间
- [ ] 无明显利空未释放

### 持仓中跟踪（每季度）
- [ ] 业绩是否符合预期
- [ ] 核心指标是否恶化
- [ ] 竞争格局是否变化
- [ ] 估值是否合理
- [ ] 是否需要调整仓位

### 卖出信号
- [ ] 业绩连续不及预期
- [ ] 核心逻辑被证伪
- [ ] 竞争格局显著恶化
- [ ] 估值严重高估
- [ ] 发现更好的投资标的
- [ ] 管理层诚信问题
