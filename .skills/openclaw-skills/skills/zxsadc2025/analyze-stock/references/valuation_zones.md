# 估值区间判断方法

## 一、估值分位计算

### PE/PB 分位计算公式
```
分位 = (当前 PE - 历史最小 PE) / (历史最大 PE - 历史最小 PE) × 100%
```

### 数据来源
- 历史 PE/PB 数据：过去 5-10 年
- 行业对比：同行业可比公司
- 自身历史：公司历史估值区间

## 二、五档估值区间

| 区间 | 估值分位 | 特征 | 操作策略 |
|------|----------|------|----------|
| 超低估区 | <20% | 极度悲观，利空出尽 | 分批低吸，20%-40% 仓位 |
| 低估区 | 20%-40% | 低于合理价值 | 逢低布局，40%-60% 仓位 |
| 合理区 | 40%-60% | 公允价值附近 | 持有为主，60%-80% 仓位 |
| 高估区 | 60%-80% | 高于合理价值 | 减仓止盈，30%-50% 仓位 |
| 泡沫区 | >80% | 严重高估，情绪狂热 | 清仓避险，0%-10% 仓位 |

## 三、技术面确认信号

### 超低估区买入信号
- [ ] 缩量企稳（成交量较前期高点萎缩 50%+）
- [ ] 股价在支撑位附近震荡
- [ ] MACD 底背离或金叉
- [ ] RSI<30 超卖区

### 低估区加仓信号
- [ ] 回踩重要支撑位（20/60 日均线）
- [ ] 量能温和放大（较前日放大 20%-50%）
- [ ] 突破短期下降趋势线
- [ ] 均线开始走平或拐头向上

### 合理区持有信号
- [ ] 5/10/20 日均线多头排列
- [ ] 趋势线完整未破
- [ ] 量能稳定，无异常放大
- [ ] MACD 在零轴上方运行

### 高估区减仓信号
- [ ] 放量滞涨（成交量放大但股价不涨）
- [ ] 高位十字星或长上影线
- [ ] MACD 顶背离
- [ ] RSI>70 超买区

### 泡沫区清仓信号
- [ ] 高位巨量（成交量创阶段新高）
- [ ] 加速赶顶走势
- [ ] 利好消息密集发布但股价反应钝化
- [ ] 跌破上升趋势线或关键均线

## 四、行业估值对比

### 相对估值法
1. **PE 对比**：公司 PE / 行业平均 PE
   - <0.8：相对低估
   - 0.8-1.2：合理
   - >1.2：相对高估

2. **PB 对比**：公司 PB / 行业平均 PB
   - 适用于重资产行业
   - 银行、地产、钢铁等

3. **PEG 对比**：PE / 净利润增速
   - <1：低估（增速快于估值）
   - 1-1.5：合理
   - >1.5：高估

## 五、特殊情况处理

### 周期股
- 使用 PB 而非 PE（周期底部 PE 可能失真）
- 关注行业周期位置
- 结合商品价格判断

### 成长股
- 可容忍较高 PE（关注 PEG）
- 重点看营收/利润增速
- 关注市场空间和竞争格局

### 亏损股
- 使用 PS（市销率）估值
- 关注用户数、GMV 等运营指标
- 评估盈利时间点

### 困境反转
- 区分暂时性困难 vs 永久性损伤
- 关注资产负债表健康度
- 等待明确反转信号

## 六、估值陷阱警示

### 低估值陷阱
- 业绩持续下滑的低 PE
- 行业衰退期的低 PB
- 财务造假的"便宜股"

### 高估值陷阱
- 透支未来多年增长
- 竞争格局恶化仍高估值
- 概念炒作无业绩支撑

## 七、快速判断清单

**买入前检查：**
- [ ] 估值分位<40%
- [ ] 技术面出现企稳信号
- [ ] 无重大利空未释放
- [ ] 基本面未恶化
- [ ] 仓位在可控范围

**卖出前检查：**
- [ ] 估值分位>60%
- [ ] 技术面出现见顶信号
- [ ] 基本面出现恶化迹象
- [ ] 有更好的投资标的
- [ ] 需要控制风险
