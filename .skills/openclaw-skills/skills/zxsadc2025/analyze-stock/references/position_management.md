# 仓位管理策略

## 一、总原则

### 仓位控制核心
- **永不梭哈**：单只股票最高仓位不超过 80%
- **分批操作**：建仓/加仓分 2-3 批，避免一次性满仓
- **留有余地**：始终保留 20% 以上现金应对极端情况
- **动态调整**：根据市场环境和个股情况灵活调整

## 二、初始建仓策略

### 稳健型建仓（推荐）
```
总计划仓位：40%
├─ 第一批：15%（试探仓，确认方向）
├─ 第二批：15%（确认企稳后加仓）
└─ 第三批：10%（突破关键位后补仓）
```

### 建仓条件
- [ ] 股价进入超低估区或低估区
- [ ] 技术面出现企稳信号
- [ ] 无明显利空未释放
- [ ] 单只股票不超过总资金 20% 首次建仓

### 建仓位置选择
| 位置类型 | 特征 | 建仓比例 |
|----------|------|----------|
| 强支撑位 | 前期低点、重要均线、缺口 | 可建首批 |
| 突破回踩 | 突破后回踩确认 | 可建第二批 |
| 趋势反转 | 底部形态完成、均线拐头 | 可建第三批 |

## 三、加仓规则

### 加仓条件（需同时满足）
1. **技术面**：
   - 股价企稳回升
   - 突破关键压力位
   - 量能同步放大

2. **基本面**：
   - 无重大利空
   - 业绩符合或超预期
   - 行业景气度向上

3. **仓位状态**：
   - 当前仓位<60%
   - 浮盈>5%（安全垫足够）
   - 总仓位可控

### 加仓金字塔
```
当前股价上涨 → 加仓比例递减
├─ 首次加仓：+15%（确认趋势）
├─ 二次加仓：+10%（趋势延续）
└─ 三次加仓：+5%（趋势加速）
```

### 禁止加仓情况
- [ ] 已处于高估区或泡沫区
- [ ] 股价跌破成本价（不补仓摊平）
- [ ] 基本面恶化
- [ ] 总仓位已>80%
- [ ] 市场整体风险高

## 四、减仓规则

### 减仓条件（满足任一）
1. **估值过高**：进入高估区或泡沫区
2. **技术走弱**：
   - 放量滞涨
   - 跌破关键均线/支撑位
   - 趋势线破位
3. **基本面变化**：
   - 业绩不及预期
   - 行业政策利空
   - 竞争格局恶化
4. **仓位控制**：
   - 单只股票仓位>50% 且浮盈>20%
   - 需要平衡组合风险

### 减仓阶梯
```
估值进入高估区 → 逐步减仓
├─ 高估区初期：减至 50% 仓位
├─ 高估区中期：减至 30% 仓位
└─ 泡沫区：减至 10% 或清仓
```

### 减仓方法
- **分批减仓**：分 2-3 次卖出，避免卖在最低点
- **逢高减仓**：反弹时卖出，不追跌
- **保留底仓**：优质股可保留 10%-20% 底仓

## 五、止损策略

### 止损类型

#### 1. 固定比例止损
| 投资风格 | 止损幅度 | 适用场景 |
|----------|----------|----------|
| 短线 | 5%-8% | 题材股、波段操作 |
| 中线 | 10%-15% | 价值股、趋势投资 |
| 长线 | 15%-20% | 核心资产、长期持有 |

#### 2. 技术位止损
- **支撑位止损**：跌破重要支撑位（前低、均线、趋势线）
- **形态止损**：头部形态完成（双顶、头肩顶）
- **时间止损**：买入后 X 天不涨反跌

#### 3. 基本面止损
- 业绩大幅不及预期
- 核心逻辑被证伪
- 行业政策重大利空
- 财务造假或诚信问题

### 止损执行原则
- **严格执行**：触及止损位坚决执行，不抱侥幸心理
- **不补仓摊平**：亏损时不加仓，避免越陷越深
- **快速止损**：发现错误快速离场，不拖延
- **接受亏损**：亏损是交易的一部分，不要情绪化

## 六、止盈策略

### 止盈类型

#### 1. 目标位止盈
| 投资风格 | 止盈目标 | 执行方式 |
|----------|----------|----------|
| 短线 | 10%-20% | 一次或分批止盈 |
| 中线 | 20%-40% | 分 2-3 批止盈 |
| 长线 | 40%+ | 估值过高时减仓 |

#### 2. 估值止盈
- **高估区**：开始减仓，保留 30%-50%
- **泡沫区**：大幅减仓或清仓
- **方法**：参考估值区间判断

#### 3. 技术止盈
- **趋势破位**：跌破上升趋势线或 20 日均线
- **顶部形态**：双顶、头肩顶完成
- **量价背离**：股价新高但量能萎缩

### 止盈执行方法

#### 分批止盈（推荐）
```
浮盈>20% 开始止盈
├─ 第一批：20% 仓位（锁定部分利润）
├─ 第二批：30% 仓位（估值进入高估区）
└─ 第三批：剩余仓位（趋势破位或泡沫区）
```

#### 移动止盈
- 股价上涨，止盈位上移
- 例如：从最高点回撤 10% 止盈
- 保护利润的同时让利润奔跑

## 七、不同风格投资者仓位建议

### 保守型（风险厌恶）
- **仓位上限**：50%
- **操作区间**：只做超低估区、低估区
- **止损**：严格执行 5%-8% 止损
- **止盈**：10%-15% 即可止盈
- **特点**：宁可错过，不做错

### 平衡型（风险中性）
- **仓位上限**：80%
- **操作区间**：低估区、合理区
- **止损**：10% 左右
- **止盈**：20%-30% 分批止盈
- **特点**：平衡风险与收益

### 进取型（风险偏好）
- **仓位上限**：80%-100%
- **操作区间**：合理区为主，可参与高估区
- **止损**：15% 左右
- **止盈**：30%-50% 或趋势止盈
- **特点**：追求高收益，承受高波动

## 八、仓位管理检查清单

### 建仓前检查
- [ ] 估值处于合理或低估区间
- [ ] 技术面有明确买入信号
- [ ] 单只股票不超过总资金 20% 首仓
- [ ] 总仓位在可控范围
- [ ] 止损位已设定

### 持仓中检查（每周）
- [ ] 估值区间是否变化
- [ ] 技术趋势是否延续
- [ ] 基本面是否有变化
- [ ] 仓位是否需要调整
- [ ] 止损止盈位是否需要更新

### 减仓前检查
- [ ] 是否达到止盈目标
- [ ] 是否触发止损条件
- [ ] 估值是否过高
- [ ] 是否有更好的投资标的
- [ ] 市场环境是否恶化

## 九、极端行情处理

### 市场暴跌
- 检查持仓股基本面是否变化
- 如逻辑未变，可逢低加仓（分批）
- 如逻辑破坏，果断止损
- 保持 20% 以上现金

### 市场暴涨
- 逐步减仓，不追高
- 锁定利润，落袋为安
- 等待回调机会
- 不轻易做空

### 黑天鹅事件
- 第一时间评估影响
- 如影响重大，先减仓避险
- 待情况明朗后再决策
- 保持流动性
