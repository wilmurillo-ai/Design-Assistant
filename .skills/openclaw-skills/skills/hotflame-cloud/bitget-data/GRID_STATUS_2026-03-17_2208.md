## 📋 网格原理

### 完整网格策略

```
价格上方 → 卖单 5 (卖出)
          ↓
价格上方 → 卖单 4 (卖出)
          ↓
价格上方 → 卖单 3 (卖出)
          ↓
当前价格 → 卖单 2 (卖出) ← 需要持仓
          ↓
当前价格 → 买单 1 (买入) ← 不需要持仓
          ↓
价格下方 → 买单 2 (买入)
          ↓
价格下方 → 买单 3 (买入)
```

### 工作流程

1. **价格下跌** → 买单成交 → 获得币种
2. **价格上涨** → 卖单成交 → 卖出币种获利
3. **循环往复** → 持续赚取差价
