## 🎯 盈利模式

### 工作原理
1. **低位买入** - 价格下跌时逐个成交买单
2. **高位卖出** - 成交后自动在上方挂卖单
3. **循环获利** - 价格波动中持续赚取差价
4. **自动维护** - 每 5 分钟检查并补单

### 利润来源
- **网格间距利润** - 每格 0.2-0.5% 利润
- **波动收益** - 日波动 2-5% 带来多次成交
- **复利效应** - 利润再投资扩大仓位
