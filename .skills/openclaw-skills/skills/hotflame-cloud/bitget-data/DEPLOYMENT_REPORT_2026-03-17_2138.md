```json
{
  "sol": {
    "amount": 15,  // 12 → 15 USDT
    "minOrderValue": 2  // 1 → 2 USDT
  },
  "eth": {
    "amount": 8,  // 5 → 8 USDT
    "minOrderValue": 2
```


### 方案 C：分批部署

先部署靠近当前价的订单，成交后再补充：

```bash

node deploy-highfreq-grids.js --range=0.05
```
