**遇到的错误:**
1. `Parameter size cannot be empty` - 需要 size 参数
2. `Parameter force error` - force 参数值不对
3. `orderType cannot be empty` - 需要 orderType

**尝试的解决方案:**
1. ✅ 使用自定义 HTTP 请求 - 失败
2. ✅ 使用官方 SDK - 无法安装 (npm 权限问题)
3. ✅ 查阅官方文档 - 参数不匹配

**根本原因:**
Bitget API v2 的下单接口可能需要特定的参数组合或格式，与公开文档不完全一致。
