---

## 🆘 问题排查

### 成交过少
1. 检查价格是否在区间内
2. 加密网格（增加 gridNum）
3. 缩小价格区间

### 成交过多
1. 降低网格密度
2. 扩大价格区间
3. 检查是否有异常波动

### 系统异常
1. 查看 `auto_monitor.log` 错误信息
2. 检查 Bitget API 连接
3. 重启监控脚本
