## 核心功能

- **每日摸鱼报告**：时长统计 + 幽默心得吐槽 + 老板版伪装日志
- **周摸鱼排行榜**：Top3 罪魁祸首 App/活动，一周摸鱼总时长
- **一键触发**：说“今天摸鱼了”“摸鱼日记”“本周摸鱼排行”即可
- **全本地存储**：数据只保存在 `~/.openclaw/moyu-journal/`，隐私拉满

## 安装方式

1. **ClawHub 一键安装**（推荐）  

   clawhub install moyu-journal-auto


或在 ClawHub 搜索 “摸鱼日记自动整理器” 直接点安装。

2. **手动安装**（开发测试用）  
把这个文件夹复制到 `~/.openclaw/workspace/skills/moyu-journal-auto`  
然后在 OpenClaw 聊天里说：`reload skills` 或重启实例。

**首次使用前请开启权限**（终端运行一次就好）：

openclaw config set tools.allow_filesystem true
openclaw restart
