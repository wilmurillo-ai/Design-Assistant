# Funnel Alignment & Repurposing Strategy

This document defines the strict classification rules for the Mad SEO Content Pipeline to ensure balanced production and maximum distribution ROI.

## 1. Funnel Stage Classification (TOFU/MOFU/BOFU)

Every piece of content generated by `mad_seo:plan_content` MUST be classified into one of these three stages:

### TOFU (Top of Funnel) - Awareness
- **Goal:** Reach new audiences, generate massive organic traffic (SEO).
- **Format:** Ultimate Guides, "What is X", Trends, Broad Industry Problems.
- **Example:** "The Ultimate Guide to OpenClaw Agents in 2026."

### MOFU (Middle of Funnel) - Consideration
- **Goal:** Build deep trust, capture emails, help the user evaluate solutions.
- **Format:** "How-to" tutorials, "X vs Y" comparisons, Framework breakdowns.
- **Example:** "Mad SEO vs Standard AI Writers: A 70-Point Comparison."

### BOFU (Bottom of Funnel) - Decision
- **Goal:** Drive direct action, overcome final objections, sales enablement.
- **Format:** Case studies, ROI calculators, Deep-dive workflow demos.
- **Example:** "How We Scaled WordPress Traffic by 300% using Mad SEO (Case Study)."

*Pipeline Rule:* A healthy monthly calendar should be roughly **50% TOFU, 30% MOFU, and 20% BOFU**.

---

## 2. The Repurposing Engine (One-to-Many)

For every Pillar Post (Long-form article) completed, the `mad_seo:repurpose_content` tool MUST generate the following distribution bundle:

1. **The Core Asset:** 1x In-depth SEO Blog Post (1,500+ words).
2. **LinkedIn Authority Post:** Rewrite the core hook and top 3 insights into a narrative, professional format suitable for B2B engagement.
3. **Twitter/X Thread:** Break the article down into 7-10 punchy, high-value tweets using statistical burstiness.
4. **Newsletter Teaser:** A 150-word personal intro that creates curiosity and links to the full post.

*Golden Rule:* Never create once and forget. 80% of the effort should go into distributing the 20% of high-quality pillar content.
