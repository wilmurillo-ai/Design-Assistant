# Author EEAT Benchmarks (HCU Compliance)

To rank in 2026, content must be tied to a verifiable entity. This audit ensures the author has a "Digital Footprint" that search engines can verify.

## 1. Professional Transparency (O07)
- [ ] **Author Bio**: Is there a consistent bio (150-300 words) across the site and socials?
- [ ] **Profile Image**: Is there a professional image? (AI avatars are acceptable if branded properly).
- [ ] **Social Links**: Are LinkedIn, Twitter/X, or GitHub links present and active?

## 2. Niche Authority (O08)
- [ ] **Citations**: Does the author have mentions or articles on third-party sites in the same niche?
- [ ] **Consistency**: Does the author only write about this niche, or is the profile scattered?
- [ ] **Contact Info**: Is there a clear way to contact the expert directly?

## 3. Experience Markers (HCU Mandatory)
- [ ] **Case Studies**: Does the author reference their own previous work?
- [ ] **Proprietary Data**: If data is used, is it clearly attributed to the author's research?
