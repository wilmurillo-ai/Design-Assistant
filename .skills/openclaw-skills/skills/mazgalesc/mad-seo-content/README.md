# Mad SEO Manager 🚀 (V6.0.1)

**Mad SEO Writer** is the elite blogging and topical authority engine for OpenClaw. It merges advanced market intelligence with surgical-grade editorial standards, specifically optimized for **Google HCU** (Helpful Content) and **GEO** (Generative Engine Optimization).

## 🌟 The "Citation-First" Advantage
Unlike standard SEO skills, Mad SEO Writer optimizes for **AI Citation Likelihood**. Every article is built to become the primary primary source that AI search engines (Gemini, Perplexity, GPT) cite in their responses.

## 🛠️ Tools & Agent Triggers

- **`mad_seo:research_strategy`** - *Trigger: "Build a cluster for [Topic]"*
  - **Strategic Research**: Automates the "Skyscraper 2.0" and "Hub-and-Spoke" planning phases.
- **`mad_seo:plan_content`** - *Trigger: "Design a 3-article-per-week content plan"*
  - **Editorial Management**: Autonomous 4-week calendar generation (3 posts/week) using Top-10 resilient research.
- **`mad_seo:site_wide_audit`** - *Trigger: "Audit my entire website"*
  - **New V3 Autonomous Manager**: Resilient-crawling of authorized sitemaps via Scrapling.
- **`mad_seo:draft_article`** - *Trigger: "Write a pillar post about [Topic]"*
  - Generates 1,500-2,500+ word articles with 7 GEO Factors and Experience markers.
- **`mad_seo:audit_eeat`** - *Trigger: "Audit this draft for EEAT"*
  - Performs the 70-point Scientific SEO audit (0-70 scoring).
- **`mad_seo:headline_pro`** - *Trigger: "Generate catchier headlines"*
  - Access 50+ high-CTR title formulas.
- **`mad_seo:humanize_content`** - *Trigger: "Humanize my content"*
  - **Human-GEO Framework**: Breaks AI patterns, removes robotic vocabulary, and injects statistical burstiness for 2026-level stealth content.

## 📂 Architecture: The Knowledge Bridge
The skill functions as a **Strategic Bridge**. Instead of burying prompts in code, it uses a rich `/references` library that the agent "consults" for every task. This keeps the agent's context clean and its output high-precision.

## 📖 Example Workflow

**User**: "Write a pillar post about 'Solo Game Dev Marketing' targeting the keyword 'indie game marketing guide 2026'."

**Agent Output**:
1. **Strategy**: Identifies content gaps in existing GDC and Reddit threads.
2. **Drafting**: Produces a 3,500-word piece featuring:
   - **Semantic Summary Table** (for AI Overview snippets).
   - **Experience Markers** ("When I marketed X, we found Y...").
   - **Sentence Chunking** for maximum extractability.
3. **Audit**: Scores the draft (e.g., 68/70) and highlights the final Top 10

## 📖 Deployment Guide

### Installation
1. Move the `mad_seo` folder to your OpenClaw workspace skills directory.
2. The skill is standalone and requires no external system dependencies.

## 🛡️ Compliance & Responsible Use
- **Authorized Audits Only**: `site_wide_audit` is for use on websites you own or have explicit permission to scan.
- **Public Data Research**: Planning tools are for analyzing publicly available competitive search information.
- **Terms of Service**: Users must ensure all automated discovery and research tasks comply with the target site's `robots.txt` and Terms of Service.
