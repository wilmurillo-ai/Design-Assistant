# 查询视频人脸融合任务 API 文档

> SDK: `tencentcloud.vclm.v20240523`

## 接口信息

- **接口域名**：vclm.tencentcloudapi.com
- **Action**：DescribeVideoFaceFusionJob
- **Version**：2024-05-23
- **类型**：同步查询接口

## 输入参数

| 参数名称  | 必选 | 类型   | 描述                                          |
| --------- | ---- | ------ | --------------------------------------------- |
| JobId     | 是   | String | 提交视频人脸融合任务时返回的 JobId               |

## 输出参数

| 参数名称        | 类型   | 描述                                                              |
| --------------- | ------ | ----------------------------------------------------------------- |
| Status          | String | 任务状态：WAIT（等待中）、RUN（执行中）、FAIL（失败）、DONE（成功） |
| ErrorCode       | String | 任务执行错误码。状态不为 FAIL 时为空字符串                           |
| ErrorMessage    | String | 任务执行错误信息。状态不为 FAIL 时为空字符串                         |
| ResultVideoUrl  | String | 结果视频 URL。**有效期 24 小时**                                   |
| RequestId       | String | 唯一请求 ID                                                        |

## 任务状态说明

| 状态值 | 含义     | 说明                          |
| ------ | -------- | ----------------------------- |
| WAIT   | 等待中   | 任务已提交，正在排队等待处理    |
| RUN    | 执行中   | 任务正在处理中                  |
| DONE   | 处理完成 | 任务成功完成，可获取结果视频     |
| FAIL   | 处理失败 | 任务执行失败，查看错误信息       |
