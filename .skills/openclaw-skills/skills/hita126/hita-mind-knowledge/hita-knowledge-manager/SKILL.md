# hita Knowledge Manager - 知识库智能管理器

hita 开发的三层知识库管理系统，是 **Hita-Mind & Knowledge** 知识体系的一部分。

## 定位

| 模块 | 作用 |
|------|------|
| **Mind** | AI 的记忆 — 偏好、决策、教训 |
| **Knowledge**（本模块） | AI 的知识 — 技巧、方法、规则 |

## 核心概念

```
L1 热数据 ─── 最近使用/高频/常用知识
│            完整内容，直接塞进 context
│
│ ↓ 30天未访问 或 超过50条
│
L2 温数据 ─── 中期知识/次要内容
│            摘要+标签，节省 context 空间
│
│ ↓ 90天未访问
│
L3 冷数据 ─── 历史归档
              只保留元数据索引
```

## CLI 用法

### 添加知识
```bash
node index.js add "标题" "完整内容" "标签1" "标签2"
```
示例：
```bash
node index.js add "冰箱除异味" "放一杯小苏打在冰箱里，第二天味道就没了" "生活技巧"
```

### 搜索知识
```bash
node index.js search "关键词"
```

### 列出知识
```bash
node index.js list [L1|L2|L3]
```

### 构建 AI 上下文
```bash
node index.js context [最大条数]
```

### 执行压缩
```bash
node index.js compress
```

## 分层策略

| 层级 | 容量 | 保留内容 | 降级条件 |
|------|------|---------|----------|
| L1 | 50条 | 完整内容 | 30天未访问 或 满容量 |
| L2 | 200条 | 摘要+标签 | 90天未访问 |
| L3 | 无限制 | 仅元数据索引 | - |

## 优先级

- **high** - 重要知识，不自动降级
- **medium** - 普通内容
- **low** - 低频内容，优先降级
