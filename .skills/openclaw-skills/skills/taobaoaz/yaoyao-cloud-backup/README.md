# yaoyao-cloud-backup

云端备份与同步套件

## 脚本列表

| 脚本 | 功能 |
|------|------|
| sync_ima.py | IMA 知识库同步 |
| sync_samba.py | Samba/NAS 同步 |
| backup_manager.py | 本地备份管理 |
| backup_restore.py | 备份恢复 |
| memory_exporter.py | 数据导出 |
| migrate.py | 配置迁移 |
| paths.py | 路径管理 |
