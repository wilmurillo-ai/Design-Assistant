# 思想领袖论坛 · 角色蒸馏体系说明

> 本文档介绍思想领袖论坛的角色筛选逻辑、蒸馏方法与预置成果。
> 所有角色均为【观点模拟】，仅供内部研究参考，不声称代表任何真实人物立场。

---

## 一、角色筛选逻辑

### 筛选原则

本体系角色池的筛选遵循三大原则：

**1. 领域代表性**
- 在AI技术/酒店旅游/风险投资三大领域具有公认影响力
- 其观点在公开领域被广泛引用和讨论
- 具备跨行业参考价值

**2. 观点差异化**
- 同一领域内尽可能覆盖不同立场和视角
- 支持对立观点的碰撞与辩论
- 确保讨论结论的多维度性

**3. 场景适配性**
- 角色观点与本项目AI Native战略具有直接或间接关联
- 可为战略决策提供有价值的参考视角
- 便于生成立场分明的讨论报告

### 筛选来源

| 来源 | 用途 |
|------|------|
| 公开主题演讲 | 提取核心观点和语言风格 |
| 企业财报/发布会 | 提取战略立场和业务逻辑 |
| 行业媒体专访 | 提取对行业趋势的判断 |
| 学术论文/访谈 | 提取理论框架和思维模型 |
| 投资机构公告 | 提取投资逻辑和关注重点 |

## 二、蒸馏方法论

### 六维思维蒸馏框架

本体系采用六维框架对角色进行结构化建模：

| 维度 | 核心问题 |
|------|---------|
| L1 表达DNA | 他用什么语言风格说事情？ |
| L2 心智模型 | 他如何拆解和分析问题？ |
| L3 决策启发式 | 他快速判断的思维捷径是什么？ |
| L4 反模式 | 他绝对不会做的事是什么？ |
| L5 价值观 | 什么对他来说绝对不可妥协？ |
| L6 诚实边界 | 他承认自己不知道什么？ |

### 蒸馏流程

```
Step 1: 资料收集
  → 多源搜索：演讲/访谈/财报/文章
  → 建立资料档案

Step 2: 观点提取
  → 提取核心主张（3-5个关键点）
  → 识别语言风格特征
  → 提炼决策逻辑

Step 3: 六维建模
  → 按六维框架逐项填充
  → 交叉验证逻辑一致性
  → 标注置信度

Step 4: 文件生成
  → 输出标准化角色文件
  → 标注法律合规声明
```

## 四、使用说明

### 快速启动

```bash
# 安装
clawhub install thought-leader-forum

# 启动讨论
> 用讨论模式，主题是XXX，让XXX参与
```

### 讨论模式

多角色模拟辩论，输出结构化报告。

### 意见模式

单角色针对具体问题快速输出观点。

### 参会模式

角色实时参与会议，发表意见和提问。
