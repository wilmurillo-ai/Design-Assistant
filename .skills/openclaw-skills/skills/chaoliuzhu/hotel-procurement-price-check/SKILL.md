# hotel-procurement-price-check

> 酒店采购实时比价助手 v1.0
> 一键查询农产品/电商全网价格，支持多源头比价

## 触发条件

**激活词**（任一）：
- "查价格"
- "采购比价"
- "多少钱"
- "价格查询"
- "农产品价格"
- "今日报价"
- "市场价"
- "比价"

**适用场景**：
- 酒店餐饮采购询价
- 农产品（蔬菜/水产/肉类）批发价查询
- 电商全网（淘宝/京东/1688/拼多多）价格对比
- 成本参考底线分析

## 能力范围

### ✅ 支持查询

| 类别 | 数据源 | 说明 |
|------|--------|------|
| 农产品批发价 | **农业农村部「农产品批发价格200指数」** | 每日更新，gov.cn官方 |
| 农产品 | **一亩田 YMT** | B端批发平台，品类最全 |
| 蔬菜 | **中国蔬菜网 vegnet.com.cn** | 批发均价 |
| 水产 | **惠农网 cnhnb.com** | 活鲜/冰鲜 |
| 肉类 | **农业农村部畜禽频道** | 猪肉/牛肉/羊肉批发价 |
| 电商全网 | **淘宝/天猫/京东/1688/拼多多** | 多平台同时查 |
| 小红书/抖音 | **社交媒体价格趋势** | 参考性价格 |

### 📍 核心免费数据源URL

**农业农村部（权威官方）**：
- 每日指数： http://www.scs.moa.gov.cn/jcyj/ （农产品批发价格200指数）
- 批发价格： http://www.agri.cn/sj/jcyj/ （国内外农产品市场动态）

**一亩田（国内最大农产品B端平台）**：
- 搜索页： https://www.ymt.com/ （支持按品类/产地搜索）
- 行情数据： https://m.ymt.com/supplylist/ （批发价搜索）

**中国蔬菜网（专业蔬菜）**：
- 批发均价： http://www.vegnet.com.cn/

**惠农网（水产/果蔬/综合）**：
- 行情页： https://www.cnhnb.com/pi/list

**北京新发地（北方最大农批）**：
- 批发价： https://www.xinfadi.com.cn/ （北京新发地批发市场）

### ⚠️ 局限性

- **农产品**：异地查询仅作参考，本地实际采购价以当地为准
- **电商**：实时价格以查询时刻为准，可能存在波动
- **小红书/抖音**：数据获取不稳定，作为补充参考

## 工作流程

### Step 1：解析商品

```
输入：商品名称/描述
  ↓
识别商品类型（农产品/电商商品/混合）
  ↓
确定优先查询的数据源
```

### Step 2：多源头并发查询

**农产品 → 优先查**：
1. 农业农村部批发价格指数
2. 一亩田 APU接口
3. 惠农网实时行情
4. 当地大型批发市场数据（如有地域信息）

**电商商品 → 全网查**：
1. 1688批发价（关键词搜索）
2. 淘宝/天猫零售价
3. 京东自营价
4. 拼多多百亿补贴
5. 参考小红书/抖音价格带

### Step 3：价格分析与呈现

输出标准格式：
```
📊 [商品名称] 实时价格报告

【批发市场参考价】（来源/更新时间）
  最低价：¥XX / 规格
  最高价：¥XX / 规格
  均价：¥XX / 规格

【电商全网价】（来源）
  1688批发价：¥XX 起（最小起订量）
  淘宝/天猫：¥XX
  京东自营：¥XX
  拼多多：¥XX

【价格分析】
  价格区间：¥XX - ¥XX
  建议采购价：¥XX - ¥XX
  ⚠️ 注意：[特殊说明，如物流/季节/规格差异]

【最低成本参考】
  基于全网最低价分析，该商品合理成本底线约：¥XX
```

## 成本逆向拆解（进阶能力）

当用户明确要求「剥离成本」「分析原材料价格」「真实成本」时触发：

```
终端零售价
  ↓ 减去
品牌溢价（行业平均15-30%）
  ↓ 减去
渠道利润（行业平均10-20%）
  ↓ 减去
营销/物流（行业平均5-15%）
  ↓
= 原材料/生产成本底线
```

**注意**：此功能为AI估算，基于行业平均系数，无法做到100%精确，仅供参考。

## 输出格式规范

1. **价格优先**：数字要醒目，单位明确
2. **来源标注**：每个价格必须标注来源和查询时间
3. **建议明确**：给出明确的采购价格建议区间
4. **风险提示**：标注异地采购、物流影响等风险

## 错误处理

- 数据源查询失败 → 自动切换备用数据源
- 所有数据源失败 → 返回「暂无数据」，提示用户手动查询
- 价格差异过大 → 标注异常，提示可能存在假货/二手/临期品

## 技术说明

- 使用 web_search + web_fetch 并发查询多个源头
- 优先使用免费/开放数据源
- 1688 API需要独立配置key（可选）
- 结果缓存10分钟，避免重复查询浪费资源
