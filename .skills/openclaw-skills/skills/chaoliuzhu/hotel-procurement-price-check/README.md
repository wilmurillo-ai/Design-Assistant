# 🛒 hotel-procurement-price-check
**酒店采购实时比价助手**

> 酒店采购人员的比价神器！输入商品名称，一键查询农产品批发价 + 电商全网价，自动给出采购建议。

## ✨ 功能特点

### 🔍 多源头并发查询
- **农产品**：农业农村部批发价、一亩田、惠农网、中国蔬菜网
- **电商全网**：1688、淘宝、天猫、京东、拼多多
- **社交参考**：小红书、抖音价格带

### 📊 智能分析输出
- 最低价 / 最高价 / 均价
- 建议采购价区间
- 成本底线估算（进阶功能）

### 🌐 适用场景
- 酒店餐饮原材料采购询价
- 蔬菜、水产、肉类批发价查询
- 全网电商价格对比
- 成本分析与砍价参考

## ⚙️ 配置

### 可选：1688 API Key
如需更精准的1688数据，可配置1688开放平台Key：
1. 注册 1688 开放平台：https://open.1688.com
2. 创建自用型应用，获取 Key 和 Secret
3. 配置到工具Key管理表

### 环境要求
- bash
- curl
- web_search / web_fetch 工具

## 📋 成本逆向拆解说明

进阶功能可剥离商品各层成本：

```
终端零售价
  - 品牌溢价（15-30%）
  - 渠道利润（10-20%）
  - 营销/物流（5-15%）
  = 原材料成本底线
```

> ⚠️ 此为AI估算，基于行业平均系数，仅供参考。
