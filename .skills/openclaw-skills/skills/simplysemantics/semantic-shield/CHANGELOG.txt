--------------------
Initial ClawhHub release.

Features:
  - Real-time AI skill safety lookups via REST API
  - 100% human-reviewed by US-based security experts (30+ years experience)
  - Safety scoring 0–100 with risk levels: critical, high, medium, low, safe
  - Threat detection: prompt injection, data exfiltration, over-privileged
    permissions, malicious code, privacy violations
  - Free skill search endpoint (no quota cost)
  - Full trust detail lookups (counts against monthly quota)
  - Submit unvetted skills for expert evaluation with request tracking
  - Continuous 0-day vulnerability monitoring
  - Webhook alerts for Pro+ tiers on threat detection
  - API-key authentication for agent/bot integration
  - MCP-compatible endpoints for AI agent frameworks
  - Interactive dashboard with usage metrics and settings
  - Free tier: 25 lookups + 5 inquiries per month
  - Pro tier: 100 lookups + 20 inquiries per month, priority evaluations
  - Enterprise tier: unlimited, private queue, SLA, dedicated support
