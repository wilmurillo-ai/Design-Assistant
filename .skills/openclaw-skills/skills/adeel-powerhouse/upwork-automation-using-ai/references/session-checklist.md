# Session Checklist

1. Confirm active tab/session.
2. Confirm login state (avatar/messages/dashboard indicators).
3. Reconfirm criteria and disqualifiers.
4. Search and shortlist.
5. Open selected job.
6. Open proposal form.
7. Fill fields.
8. Final confirmation: not submitted.

## Credentials Handling
- Accept credentials only when user explicitly provides them.
- Do not echo passwords back in chat.
- If already logged in, do not ask again.
