# De-AI Checklist

12 markers that make Chinese text sound AI-generated. The agent must audit every draft
against these markers and rewrite any flagged sentences.

## Detection Markers

### Structural Markers

| # | Marker | Example | Fix |
|---|--------|---------|-----|
| 1 | **Parallel sentence structures** — 3+ sentences with identical grammar pattern | 我喜欢X，我热爱Y，我追求Z | Break the pattern; vary grammar |
| 2 | **Three-part formula (三段式)** — intro-body-conclusion with clear transitions | 首先...其次...最后 | Drop the scaffolding; just say it |
| 3 | **Uniform sentence length** — all sentences roughly the same character count | Every sentence 15-20 chars | Mix short (5-8) with medium (15-25) |
| 4 | **Bullet-point prose** — text reads like a formatted list without the bullets | Point 1. Point 2. Point 3. | Merge ideas into flowing text |

### Vocabulary Markers

| # | Marker | Example | Fix |
|---|--------|---------|-----|
| 5 | **AI cliché words** — 赋能、助力、深耕、沉淀、拥抱、赛道、底层逻辑、认知 | 深耕技术领域多年 | Use plain words: 做了好几年 |
| 6 | **Excessive connectives** — 此外、与此同时、不仅如此、值得一提的是 | 此外，我还关注了... | Drop the connector; start directly |
| 7 | **Grandiose metaphors** — 灯塔、宝藏、殿堂、星辰大海 | Linux.do 是技术人的灯塔 | Just say what it is plainly |
| 8 | **Negation-based emphasis** — 不是X而是Y pattern repeated | 不是简单的学习，而是深度的交流 | State it directly without the contrast |

### Tone Markers

| # | Marker | Example | Fix |
|---|--------|---------|-----|
| 9 | **Over-polished tone** — no colloquialisms, no casual markers | Reads like a press release | Add 吧、嘛、其实、反正、就是 |
| 10 | **Propaganda tone** — 共同进步、携手前行、共建生态 | 希望和大家共同进步 | Say what you'll actually do |
| 11 | **Perfect grammar throughout** — no fragments, no run-ons | Every sentence is grammatically flawless | Allow natural imperfections |
| 12 | **Flattery without substance** — 久仰大名、慕名而来、心向往之 | 久仰 Linux.do 大名 | Replace with how you actually found it |

## Audit Process

1. Read the draft sentence by sentence
2. Flag any sentence matching 1+ markers above
3. For each flagged sentence, rewrite using the "Fix" column guidance
4. Re-read the full text aloud — does it sound like someone talking, not writing an essay?
5. Final check: would a human reviewer think "this person actually wrote this"?

## Quick Smell Test

Ask these 3 questions about the final draft:
- **Could you swap the applicant's details and reuse this for someone else?** If yes → too generic, rewrite.
- **Does every sentence add new information?** If any sentence is pure filler → cut it.
- **Would you text this to a friend?** If it feels too formal → loosen the tone.
