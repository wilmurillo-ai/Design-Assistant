# Platform Copy Specs

> Used by Phase 8 (optional multi-platform distribution) to generate platform-specific content from the canonical article.

---

## Xiaohongshu (小红书)

### Title
- 15-20 chars, one sentence of value
- Format: `core highlight｜scene/identity tag`
- Numbers at the start increase impact
- Max 0-1 emoji in title

### Body
- 150-250 chars, max 300
- Tone: friend sharing, not a tutorial
- Short paragraphs (1-3 sentences each)
- Use number emoji lists (1️⃣2️⃣3️⃣)
- End with `.` on its own line, then tags below
- Say "链接在评论区" — no long URLs in body

### Tags
8-12 tags, mixing:
- high-traffic tags (#AI工具 #AI编程)
- precise tags (#ClaudeCode #Skill)
- scene tags (#一个人的团队 #独立开发)
- content tags (topic-specific)

### Prohibitions
- no "震惊" "竟然" style clickbait
- no emoji spam
- no overly academic titles

## Jike (即刻)

### Platform character
Tech/creator community. Users default to knowing the background. Write like you're chatting with peers, not teaching.

### Style
- 200-400 chars (longer than XHS, shorter than WeChat)
- First sentence throws out the most interesting point
- Include specific details: tool names, numbers, operations
- Personal perspective and judgment, not objective news
- Relaxed paragraph rhythm — short, with whitespace
- End naturally, no hard CTA

### Structure
```
[one-sentence hook]
[background, 2-3 sentences]
[process, expanded, multiple paragraphs]
[core takeaway, 1-2 sentences]
[resource link + community prompt]
```

### Circle tags
Select at post time (not in body text). Common: #Claude Code, #AI工具, #独立开发, #AI编程

### Prohibitions
- no XHS style ("姐妹们！" "绝绝子")
- no WeChat style ("本文将介绍")
- no excessive emoji
- no @ influencers for traffic

## Xiaoyuzhou (小宇宙 / Podcast)

### Script rules
- 1500-2000 chars, ~5-8 minutes when read
- Conversational oral style, like telling a story
- Short sentences, natural paragraph transitions
- Avoid "本期我们将介绍" — use "你看" "说白了" "关键来了"
- Numbers must be spelled out
- No stage directions like "(停顿)"

### Script structure
```
[Opening] identity + one-sentence hook (most attractive result)
[Result] show outcome first, create curiosity
[Process] specific how-to: tools, steps, details
[Methodology] distill a reusable takeaway
[CTA] WeChat ID + keyword + closing
```

### Show notes
Always include full transcript text in show notes (listeners can read while listening).

## WeChat Moments (朋友圈)

### Style
- Like a status update to friends, not a marketing post
- First sentence grabs attention (suspense/result/reversal)
- 3-5 lines of body text (max 6 — longer gets folded)
- 1-2 emoji at most
- Optional trailing question or teaser

### Structure
```
[one-sentence hook]
[2-3 sentences of core info]
[optional: link/comment prompt]
```
