# 公众号发布操作指南

> 后续每次写公众号文章，照着这个流程走就行。

## 推荐工具

### 首选：doocs/md（开源、免费、功能强）

- **地址**：https://md.doocs.org
- **特点**：开源免费、自定义主题色、支持代码高亮、自动处理外链转脚注、支持段落缩进
- **GitHub**：https://github.com/doocs/md （11.8k+ stars）

### 备选：mdnice（主题丰富）

- **地址**：https://editor.mdnice.com
- **特点**：海量内置主题、一键复制、支持知乎/掘金等多平台

### 备选：墨滴社区版

- **地址**：https://mdnice.xieyufei.com
- **特点**：开源版 mdnice，自部署

## 发布流程（3 分钟）

### 第 1 步：打开编辑器

用 Chrome 浏览器打开 https://md.doocs.org

### 第 2 步：粘贴 Markdown

把 `blog-post-wechat.md` 的内容全选复制，粘贴到编辑器左侧的编辑区。

### 第 3 步：选主题和配色

右侧侧边栏配置：

- **主题**：选「经典」或「优雅」
- **主题色**：推荐选「薰衣紫」（与 Product Sense Coach 品牌色一致），或「经典蓝」
- **字体**：无衬线
- **字号**：正常
- **段落首行缩进**：关闭（技术/产品类文章不需要）
- **段落两端对齐**：开启
- **微信外链转底部引用**：开启（公众号不支持外链跳转）

### 第 4 步：复制

点击顶部的 **「复制」** 按钮。

### 第 5 步：粘贴到公众号

打开微信公众号后台编辑器 → `Cmd+V`（Mac）或 `Ctrl+V`（Windows）粘贴 → 完成。

### 第 6 步：微调

- 检查图片是否正常显示
- 调整封面图
- 填写摘要
- 预览 → 发布

## Markdown 写作注意事项

公众号有一些特殊限制，写 Markdown 时注意：

1. **不要用代码块展示安装命令**：公众号读者不会在手机上装东西，改为文字描述
2. **代码块尽量短**：超宽代码块在手机上体验差
3. **外链写明文**：公众号不支持超链接跳转，写成 `github.com/xxx` 这种明文
4. **场景描述用引用块（>）而不是代码块（```）**：引用块在公众号里渲染更好看
5. **表格列数控制在 3 列以内**：手机屏幕窄，4 列以上会挤
6. **图片宽度**：建议 900px 宽度，公众号会自动缩放
