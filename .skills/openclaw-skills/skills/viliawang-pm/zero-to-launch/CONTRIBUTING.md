# Contributing to Product Sense Coach

First off, thank you for considering contributing! This skill is built for PMs by PMs, and your real-world experience makes it better.

## Ways to Contribute

### 1. Share Your Clarity Map

Used the skill to think through a product idea? Share it!

- Open an issue titled "Clarity Map: [Your Product Idea]"
- Include the full conversation (or highlights)
- Tell us what worked and what didn't

### 2. Improve the Conversation Prompts

Found a question that fell flat? Have a better way to phrase something?

- Open a PR modifying `SKILL.md`
- Explain why your version works better
- Include an example conversation if possible

### 3. Add Examples

More examples help people understand how to use this skill.

- Add to `examples/CONVERSATIONS.md`
- Follow the existing format
- Real conversations are better than hypothetical ones

### 4. Translate

Want to help PMs in other languages?

- Create `README.zh-CN.md` or similar
- Translate the core content
- Keep the examples culturally relevant

### 5. Report Issues

Something didn't work as expected?

- Open an issue with the tag "bug" or "feedback"
- Describe what you expected vs. what happened
- Include the conversation context if possible

## Code of Conduct

- Be constructive, not critical
- Remember that every idea starts as intuition
- This is a thinking partner, not a gatekeeper

## Questions?

Open an issue with the tag "question" — we're happy to help.

Thanks for making product thinking more accessible. ⭐
