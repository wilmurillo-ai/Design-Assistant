## v0.5.0 (2026-03-02)

### 新功能
- **多用户授权**：支持按用户分别授权，各自 token 独立存储
- **加密存储**：token 使用 AES-256-CBC 加密存储，安全可靠
- 新增 `use` 命令：使用指定用户的 token 执行命令
- 新增 `users` 命令：列出所有已授权用户

### 安全特性
- token 加密存储到 `config/users.enc`
- 支持环境变量设置加密密钥

### 修复
- 修复变量重复声明 bug
