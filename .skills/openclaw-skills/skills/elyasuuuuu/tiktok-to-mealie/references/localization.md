# Localization

## General rule
Match the user's language when reconstructing the recipe.

## Translation rule
Translate:
- title
- description
- ingredients
- steps
- categories/tags if useful

Do not keep a mixed-language recipe unless the user explicitly wants that.

## Ingredient style
Prefer natural cooking language over literal translation.
Examples:
- finely chopped -> finement haché
- to taste -> au goût
- drizzle -> arroser / un filet de
- shredded -> émincé / râpé depending on context

## Quantity uncertainty
If TikTok text is incomplete:
- keep approximate wording explicit
- avoid inventing exact grams without basis
- prefer a usable recipe with light uncertainty over false precision
