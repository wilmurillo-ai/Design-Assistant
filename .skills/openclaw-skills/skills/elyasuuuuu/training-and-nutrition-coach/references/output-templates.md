# Output templates

## Full fitness plan
Use this compact structure:
1. Goal
2. Assumptions
3. Calories
4. Macros
5. Weekly training split
6. Meal structure
7. Adjust after 2-3 weeks

## Meal plan
Use this structure:
1. Goal fit
2. Daily calorie target
3. Macro target
4. Meal structure
5. 3-5 concrete meal ideas
6. Recipe import note if relevant

## Recipe set
For recipe batches, prefer 2-4 strong recipes.
For each recipe include:
- title
- short fit summary
- calories per serving estimate
- protein/carbs/fats per serving estimate when reasonable

## Workout plan
Use this structure:
1. Goal
2. Days per week
3. Day-by-day plan
4. Progression rule
5. Recovery note

## Short follow-up questions
Ask at most one grouped follow-up when needed.
Good grouped pattern:
- age, height, weight
- goal
- activity/training frequency
- dietary constraints

## Assumption rule
If the user wants speed over precision, proceed with assumptions and label them clearly.
