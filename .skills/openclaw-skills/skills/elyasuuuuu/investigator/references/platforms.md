# Platforms

Start with major public platforms and developer/community sites:
- GitHub
- Reddit
- TikTok
- X / Twitter
- Instagram
- GitLab
- YouTube
- Twitch
- Pinterest
- Hacker News
- Keybase
- Gravatar

## Search strategy

Prefer simple public discovery first:
- direct profile URL guess when platform URL structure is stable
- platform-specific web search
- public profile pages
- obvious mirror or cached public references when normal search is weak

## Evidence classes

For each platform, classify as:
- exact match
- likely match
- weak match
- no result
- blocked / not verifiable

## Notes

- Some platforms are inconsistent or hostile to scraping.
- Do not assume a guessed profile URL means the account is valid unless the page confirms it.
- Prefer a smaller set of verified findings over a large set of noisy guesses.
