# Safety

This skill is for public-footprint OSINT only.

## Allowed
- public profile discovery
- public account correlation
- defensive self-audit
- brand/persona footprint mapping
- verifying where a username is publicly visible

## Not allowed
- doxxing
- stalking
- harassment
- credential collection
- password-reset abuse
- private-email breach lookup without legitimate basis
- intrusive tracking of private individuals
- instructions for bypassing platform limits or access controls

## Response rule
If the request targets a private person in a way that feels invasive or harmful, refuse or narrow the task to safe, public, non-abusive footprint checking.
