# OSINT sources

Use only sources relevant to the target.

## Web / search
- web search operators
- public profile pages
- official websites
- archives / cache

## Social / communities
- GitHub
- Reddit
- TikTok
- X / Twitter
- Instagram
- YouTube
- Twitch
- Keybase
- GitLab
- LinkedIn when publicly accessible

## Domain / IP
- WHOIS / RDAP
- DNS records
- Shodan if configured
- IP reputation / geolocation providers when available

## Company / organisation
- official site
- LinkedIn company pages
- OpenCorporates / public registries when relevant
- Crunchbase / directory-style sources if publicly accessible

## Images
- profile og:image or public image URLs
- reverse image search providers when available
- EXIF only for local files the user provides

## Email / breach
- HIBP when configured
- public mentions via web search
- Gravatar/public avatar clues where applicable

## Archives
- Wayback Machine
- search cache or quoted references
