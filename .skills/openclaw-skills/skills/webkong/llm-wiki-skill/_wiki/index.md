This is the LLM-maintained knowledge wiki for this vault.

## What this is

A synthesized knowledge layer — not raw notes, but distilled understanding. Built incrementally via `/wiki-ingest`, queried via `/wiki-query`, and audited via `/wiki-health`.

## Skills

- `/wiki-ingest <source>` — extract insights from a file, URL, or topic into the wiki
- `/wiki-query <question>` — search and synthesize an answer; files new insights back automatically
- `/wiki-health` — audit for orphans, stubs, broken links, and stale pages

## Principles

- The wiki compounds over time. Every query is an opportunity to improve it.
- Synthesize, don't copy. Pages contain understanding, not raw quotes.
- Wikilinks are first-class. Isolated pages are a smell.
- One concept per page. Split broad topics.
