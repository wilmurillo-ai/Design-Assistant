## AI与数字化新趋势分析

### 大模型/AI应用现状评估

```
关键问题：
- 是否在用AI相关产品？（ChatGPT企业版/文心一言/通义千问）
- 是否有AI落地项目在推进？（智能客服/AI写作/代码助手）
- IT团队是否有AI相关招聘？（提示词工程师/AI产品经理）

判断信号（从招聘JD中发现）：
✓ 招聘"大模型工程师" → 自建AI意愿强
✓ 招聘"RPA工程师" → 流程自动化需求旺
✓ 招聘"数据分析师" → BI/数仓升级需求
✓ JD中提到"AI原生" → 引领级数字化
```

### 国产化替代评估（政策驱动机会）

| 场景 | 政策背景 | 用友机会 |
|------|---------|---------|
| 央国企去Oracle/SAP | 信创政策 | 用友BIP替换 |
| 政府单位去国外ERP | 等保/信创要求 | 用友GRP |
| 军工单位系统自主可控 | 保密要求 | 用友军工版 |
| 医疗机构医保合规 | 医保局要求 | 用友医疗云 |

### 云化迁移评估

```
现状类型：
- 纯本地部署 → 有上云意向的，推云端/混合部署方案
- 已用SaaS → 数据安全/定制化痛点，推私有云方案
- 混合架构 → 数据整合痛点，推统一平台方案
```
