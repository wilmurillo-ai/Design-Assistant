# 客户分级ABCD模型

本文档提供客户分级管理模型，帮助销售团队合理分配资源，提高赢单效率。

---

## 一、分级模型概述

### 1.1 分级目的

- **资源优化**：把有限资源投入到高价值客户
- **过程管控**：不同级别客户不同跟进策略
- **预测管理**：通过分级预测业绩达成
- **团队协作**：明确主攻和协攻分工

### 1.2 分级维度

| 维度 | 权重 | 说明 |
|------|------|------|
| 需求匹配度 | 30% | 产品与客户需求的匹配程度 |
| 预算落实度 | 25% | 预算是否确定、充足 |
| 决策链清晰度 | 20% | 能否找到决策人 |
| 竞争态势 | 15% | 竞争格局是否有利 |
| 时间窗口 | 10% | 成交时间紧迫性 |

---

## 二、ABCD分级标准

### 2.1 A类客户（高价值）

**特征**：
- [ ] 需求与产品高度匹配
- [ ] 预算已落实（XX万+）
- [ ] 决策链清晰，关键人已搞定
- [ ] 竞争态势有利或有信心赢
- [ ] 3个月内可签约

**资源投入**：
- 销售总监/资深销售主攻
- 高层拜访（至少1次）
- 个性化解决方案
- 商务谈判重点跟进
- 投入时间：50%

**跟进策略**：
```
1. 高层攻关：总监及以上层级拜访
2. 需求深挖：全面了解痛点
3. 方案定制：针对性制作PPT/方案
4. 商务谈判：价格、合同条款攻坚
5. 逼单签约：创造签约条件
```

### 2.2 B类客户（潜力）

**特征**：
- [ ] 需求基本匹配
- [ ] 预算有但不确定
- [ ] 决策链部分清晰
- [ ] 竞争态势不明或略处下风
- [ ] 3-6个月可签约

**资源投入**：
- 主力销售跟进
- 定期价值传递
- 保持竞争意识
- 投入时间：30%

**跟进策略**：
```
1. 需求验证：确认核心需求
2. 关系建立：找到关键人
3. 竞争壁垒：构建差异化优势
4. 预算推动：协助客户申请预算
5. 时机等待：等待决策窗口
```

### 2.3 C类客户（培育）

**特征**：
- [ ] 需求匹配度一般
- [ ] 预算未知或不足
- [ ] 决策链不清晰
- [ ] 竞争激烈或信息不足
- [ ] 6个月以上或不确定

**资源投入**：
- 标准化跟进
- 内容营销影响
- 市场活动转化
- 投入时间：15%

**跟进策略**：
```
1. 需求探索：了解真实需求
2. 价值传递：发送案例、资料
3. 关系维护：保持存在感
4. 机会转化：等待需求启动
5. 适时升级：B类信号及时升级
```

### 2.4 D类客户（无效）

**特征**：
- [ ] 需求完全不匹配
- [ ] 无预算或预算严重不足
- [ ] 决策链复杂且无法突破
- [ ] 竞争格局明显不利
- [ ] 超过12个月无可能

**处理策略**：
- [ ] 减少投入，礼貌性维护
- [ ] 放入线索池等待机会
- [ ] 适时放弃，转向其他客户

## 三、分级评估表

### 3.1 评分标准

| 评分项 | 5分 | 3分 | 1分 |
|-------|-----|-----|-----|
| 需求匹配 | 高度匹配 | 基本匹配 | 不匹配 |
| 预算 | 已落实 | 在申请 | 无预算 |
| 决策链 | 搞定关键人 | 找到接口人 | 不清楚 |
| 竞争 | 明显优势 | 势均力敌 | 明显劣势 |
| 时间 | <3个月 | 3-6个月 | >6个月 |

### 3.2 分级计算

```
总分 = 需求匹配×0.3 + 预算×0.25 + 决策链×0.2 + 竞争×0.15 + 时间×0.1

- A类：总分 ≥ 4.0
- B类：总分 3.0-3.9
- C类：总分 2.0-2.9
- D类：总分 < 2.0
```

### 3.3 示例评估

**客户：北京能源集团**

| 评分项 | 得分 | 说明 |
|-------|------|------|
| 需求匹配 | 5 | AI+集团管控+信创，完美匹配 |
| 预算 | 4 | 预算充足，已在申请 |
| 决策链 | 4 | 信息中心李主任支持 |
| 竞争 | 3 | 中国软件已在，需抢份额 |
| 时间 | 4 | 招标在即，Q2要签约 |
| **总分** | **4.0** | **A类客户** |

## 四、分级跟进策略

### 4.1 A类客户攻略

**攻关节奏**：
| 周次 | 动作 | 目标 |
|------|------|------|
| 第1周 | 高层拜访 | 搞定董事长/总经理 |
| 第2周 | 方案汇报 | 获得初步认可 |
| 第3周 | 商务谈判 | 价格条款协商 |
| 第4周 | 逼单签约 | 拿下订单 |

**必做动作**：
- [ ] 至少1次高层拜访
- [ ] 个性化解决方案
- [ ] 客户案例参观
- [ ] 高层联合决策

### 4.2 B类客户攻略

**跟进节奏**：
| 周期 | 动作 | 目标 |
|------|------|------|
| 每2周 | 价值传递 | 保持热度 |
| 每月 | 需求沟通 | 深化关系 |
| 季度 | 方案更新 | 响应变化 |
| 随时 | 升级评估 | 捕捉时机 |

**必做动作**：
- [ ] 找到关键决策人
- [ ] 差异化优势传递
- [ ] 预算申请协助
- [ ] 竞争壁垒构建

### 4.3 C类客户攻略

**跟进节奏**：
| 周期 | 动作 | 目标 |
|------|------|------|
| 每月 | 内容触达 | 价值传递 |
| 季度 | 活动邀请 | 深度接触 |
| 半年 | 评估升级 | 识别机会 |

**必做动作**：
- [ ] 定期发送成功案例
- [ ] 邀请参加市场活动
- [ ] 保持基础关系
- [ ] 监控需求启动信号

## 七、分级话术

### 7.1 询问预算

> "王总，这个项目贵司计划投入多少预算？我们好根据预算匹配最合适的方案。"

### 7.2 了解决策链

> "除了您之外，这个项目还有哪些负责人参与决策？方便的话我分别拜访一下，确保方案能覆盖大家的需求。"

### 7.3 评估时间

> "贵司计划什么时候启动这个项目？我们好提前安排资源，确保项目顺利落地。"

### 7.4 升级请求

> "X总，这个项目对我们非常重要，我想邀请我们总监来拜访您一次，当面汇报我们的解决方案，您看可以吗？"

## 九、案例：北京能源集团分级分析

### 9.1 评估结果

| 维度 | 得分 | 说明 |
|------|------|------|
| 需求匹配 | 5 | AI业务+集团管控+信创，三大需求全匹配 |
| 预算落实 | 4 | 预算充足，具体金额在确认 |
| 决策链 | 4 | 李主任支持，需突破董事长 |
| 竞争态势 | 3 | 中国软件已在，需抢份额 |
| 时间窗口 | 4 | 招标已发，Q2要签约 |
| **总分** | **4.0** | **A类** |

### 9.2 攻关策略

1. **搞定高层**：董事长/总经理拜访
2. **差异化**：用友BIP + AI能力 + 信创适配
3. **抢份额**：从小模块切入，展示能力
4. **快速响应**：提供个性化方案

### 9.3 预期收益

- 签约金额：300-500万
- 战略意义：能源行业标杆
- 复制价值：可复制到其他能源集团
