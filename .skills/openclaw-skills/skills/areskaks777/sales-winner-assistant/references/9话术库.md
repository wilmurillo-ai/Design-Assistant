## 一、微信/飞书跟进话术

### 初次接触后跟进（当天发）

```
[情景：参加了论坛/展会后]
"[姓名]总，今天在[活动名]跟您交流很受启发！
您提到的[具体业务痛点]正好是我们最近帮[同行企业]解决的核心问题。
我整理了一份案例，方便的话发给您参考？"
```

```
[情景：电话拜访后]
"[姓名]总，刚才电话打扰了，感谢您耐心聆听。
针对您提到的[核心痛点]，我梳理了一份针对[行业]的解决方案，
已经发到您邮箱了，有时间看看，如有问题随时联系我 🙏"
```

### 方案发送后跟进（3天内）

```
"[姓名]总，上次发的方案您有机会看看了吗？
里面有一个[行业]的案例，[企业名]跟贵公司规模很接近，
他们上线后[具体收益：如月结从15天缩短到3天]，
您看这个方向跟贵公司的诉求match吗？"
```

### 长期未回复跟进（超过1周）

```
"[姓名]总，冒昧打扰～
最近我们有个[行业]的新客户刚上线，有几个数据挺有意思的，
跟您分享：[1个具体数据]。
不知道贵公司今年在这块有没有新的规划？"
```

```
[价值型，不直接推销]
"[姓名]总，看到一篇关于[行业趋势]的文章，
想到上次我们聊到的[话题]，转发给您，
周末愉快 😊"
```

## 二、拒绝处理话术（30+ 场景）

### 💰 价格类拒绝（6个）

**1. "太贵了"**
```
"您说的完全理解，我们的定价确实不是最低的。
不过您算过用现在方式的隐性成本吗？
比如[具体场景：对账每月要3个财务加班5天]，
一年下来这部分人力成本大概是多少？
我们系统的费用通常12-18个月就能回本，
您觉得这个ROI可以接受吗？"
```

**2. "预算不够"**
```
"我理解今年预算已经定了，这很正常。
有两个思路可以考虑：
一是我们可以做分期方案，首年只需要[X万]；
二是我们先跑一个[最小MVP模块]，
下半年追加预算的时候再扩展，
您看哪个思路更适合贵公司的节奏？"
```

**3. "比金蝶便宜多少？"**
```
"[姓名]总，我直接说：光比价格可能比不出来，
因为两边产品覆盖范围不一样。
我建议我们把需求列出来，分别对应看各家报价，
这样比出来的才有意义——
您最核心的需求是哪几个模块，我先列个清单？"
```

**4. "别人家打5折了"**
```
"价格战我们也能打，但最后吃亏的是客户。
便宜的那家可能明年就不在了，
服务没人管，系统没人升级，
您还得重新选型，二次成本更高。
我们不拼价格，拼的是客户成功率和长期价值。"
```

**5. "能送我们一些模块吗"**
```
"可以谈。您希望送哪几个模块？
不过我建议先确认核心需求——
有些模块看起来有用，但实际业务可能用不上。
我们先盘点一下，哪些是必须的，哪些是锦上添花的，
这样送的才有价值。"
```

**6. "价格能到XX万吗"**
```
"这个价格确实做到了我们的成本线。
这样吧，我申请一下，但需要您支持两个条件：
一是签约时间能否提前到本月；
二是首付比例能否提高到50%。
如果可以，我全力帮您申请。"
```

---

### ⏰ 时间/节奏类拒绝（5个）

**7. "现在不是时候，等等再说"**
```
"[姓名]总，您说的'等等'大概是什么时间节点？
我问这个不是要催您，而是想了解
是预算周期、还是内部项目节奏、还是其他原因？
不同情况我们的配合方式不一样。"
```

**8. "太忙了，没时间搞这个"**
```
"您忙我很理解，其实我们实施周期比您想的短，
[同行案例]当时也是边经营边上线，分批推进，
核心业务不停，我们来适应您的节奏——
最小化干扰，这是我们专门给忙的企业设计的方案。"
```

**9. "等明年再说"**
```
"明年的规划已经有了吗？
我问是因为如果您打算明年上，
今年Q4开始启动选型和内部立项
是比较合适的节奏——
这样明年Q1就能开始实施，避免拖到下半年。
要不我给您梳理一个时间规划表？"
```

**10. "我们刚换了ERP，短期内不会换"**
```
"理解，刚换系统确实需要稳定期。
不过我们不是要替换您现有的系统，
而是补充——比如您的财务系统很好，
但供应链或人事模块可能有升级空间。
等这批系统稳定了，我们可以先做个评估，
看看有没有互补的地方，不着急。"
```

**11. "等项目上线后再谈"**
```
"完全理解，项目上线期间不宜分心。
这样，我保持低频跟进，每个月给您发一条行业动态，
等您项目稳定了咱们再细聊。
如果您这边有任何问题，随时找我。"
```

### 🤔 信任/关系类拒绝（5个）

**12. "再看看其他家"**
```
"完全应该多看看，这是大项目，
选型严谨才对。
我建议您用这4个维度来评估：
产品成熟度/行业案例/实施团队本地化/售后响应速度——
我们可以一起列个评测表，每家填一遍，
一目了然。您觉得这样可以吗？"
```

**13. "我们已经在跟金蝶谈了"**
```
"那挺好的，也说明贵公司确实在认真推这个事。
我不是要您放弃他们，而是建议在最终决定前
给我们一个展示的机会——
就2小时，我带着[行业]案例来，
如果没有价值您当场叫停，怎么样？"
```

**14. "上次有家供应商承诺了很多没兑现"**
```
"我理解这种担心，确实行业里有这种情况。
您愿意跟我说说当时是哪方面没兑现吗？
我想知道我们能不能做到——
如果能，我们把它写进合同里，白纸黑字。"
```

**15. "我们对用友有偏见"**
```
"我理解，每个品牌都有被误解的时候。
您方便说说是什么让您有这个印象吗？
我想知道真实原因，这样也能让您更客观地评估。
如果您说的确实存在，我也不会回避；
如果只是误会，咱们就事论事。"
```

**16. "你们服务不行"**
```
"您说的服务不行，具体是哪方面？
是响应速度、实施质量、还是后续支持？
我想了解细节，这样才能判断是个案还是系统性问题。
如果是我们的责任，我负责到底；
如果不是，也欢迎监督。"
```

### 🔧 技术/实施类拒绝（5个）

**17. "怕实施周期太长，影响正常运营"**
```
"这个担心很正常，实施风险是大项目最大的坑。
我们有一套'最小打扰'实施方法论：
第一阶段只上核心财务，2-4周上线；
业务系统老方式并行跑，过渡期3个月；
我给您看[类似规模客户]的实施时间表——
比您预想的快很多。"
```

**18. "我们IT团队不够，实施资源跟不上"**
```
"我们的实施不需要您这边大量IT资源，
主要靠我们的顾问团队主导，
您这边只需要一个业务对接人就够了。
[同行案例]当时IT只有3个人，也顺利上线了。"
```

**19. "担心数据安全"**
```
"您担心数据安全完全正确，这应该放在第一位。
我们支持私有化部署——数据在您自己服务器上，
我们访问不到任何您的业务数据。
等保三级认证、ISO27001我们都有，
需要我给您发一份安全白皮书吗？"
```

**20. "系统太复杂，担心员工不会用"**
```
"这个担心我理解，新系统上线确实有学习成本。
不过我们有完整的培训体系：
上线前集中培训+上线后一对一带教+操作手册+视频课程。
[同行案例]当时50多岁的财务阿姨一周就上手了，
后来她说比旧系统还好用。"
```

**21. "怕跟现有系统不兼容"**
```
"我们支持API对接，主流系统都能打通。
上线前我们会做接口测试，确保数据互通。
您现有系统是什么？我们来评估兼容性。"
```

如果您认可这个方向，
我们一起想想怎么给老板汇报才能通过——

**23. "董事会要开会讨论"**

我们之前帮[类似客户]做过董事会汇报材料，
您这边汇报人是谁？
我们来帮您准备，让决策过程更顺畅。"

"招标我们很熟悉，各地的流程都走过。
我来帮您梳理招标文件的技术要求，

您这边需要走流程的时候随时可以用。
另外，采购最怕的是技术不达标出问题，

### 🎯 需求/意向类拒绝（4个）

**27. "我们不需要"**
```
"理解，您目前业务可能运转得很好。
不过我想了解一下：您现在是用什么方式管理[业务]？
方便的话简单说说，我看看有没有提升空间——
不一定现在要换，了解一下行业趋势。"
```

**28. "我们有内部开发"**
```
"内部开发确实灵活，但长期维护成本很高。
招人难、留人难、升级难——
我们见过太多做到一半放弃的案例。
您内部开发投入了多少人力？一年维护成本多少？
我们可以做个对比分析。"
```

**29. "上这个系统不是我的KPI"**
```
"我理解，那您的KPI主要是哪些指标？
也许这个系统能帮您达成KPI？
比如提升效率、降低成本这些。
如果我们能证明帮您完成KPI，是不是就有动力推了？"
```

**30. "再考虑考虑"**
```
"考虑是应该的，这是大事。
不过我想知道您主要考虑哪些方面？
是价格、方案、还是其他？
了解清楚我也好帮您分析，
不然我心里也没底。"
```

**31. "最近没时间"**
```
"完全理解您忙。
这样，我只需要15分钟，
您方便时我打给您，或者您给我个回电时间？
我把最核心的东西快速过一下，不耽误您太久。"
```

## 三、复购/续费/增购话术

### 合同到期前3个月

```
"[姓名]总，您的合同大概[X月]到期了，
这一年用下来感觉怎么样？
有没有什么功能用得不顺手的地方？
续约的同时我们可以一起看看有没有要优化的。"
```

### 增购模块话术

```
"[姓名]总，最近我看到您们在招[职位]，
是有在考虑[相关业务]这块扩展吗？
我们的[新模块]刚好是针对这个场景的，
已有客户在用，效果挺好，要不要了解一下？"
```

```
"[姓名]总，感谢贵公司一直的支持和信任，
我最近有个任务：希望多了解一些[行业]的优质企业。
您在圈子里认识的朋友里，有没有也在考虑数字化升级的？
给我引荐一下，我保证不给您的朋友添麻烦 😊"
```
