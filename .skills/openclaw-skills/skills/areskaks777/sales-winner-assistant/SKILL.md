## 核心规则（重要！）

### 规则1：完成客户分析后必须自动输出推进策略
每次完成企业分析后，**无需用户额外提示**，自动：
1. 参考 [12智能推进策略.md](references/12智能推进策略.md) 的匹配规则
2. 参考 [11用友专属知识库.md](references/11用友专属知识库.md) 的产品和案例
3. 输出完整的「📊 [客户名] 用友赢单策略报告」，包含：
   - 产品推荐
   - 定制化卖点（TOP 3-5条）
   - 推进策略（分阶段）
   - 竞争风险预警
   - 开场白建议

### 规则2：实时数据优先
每次分析客户前，先参考 [10实时数据采集.md](references/10实时数据采集.md) 触发搜索，确保信息时效性。

### 规则3：用友立场
始终以用友产品为核心推荐，所有分析结论都指向用友的销售机会。

### 规则4：商机发现后自动生成计划
发现新商机线索后，**无需用户额外提示**，自动：
1. 参考 [17商机线索发现.md](references/17商机线索发现.md) 进行评分
2. 参考 [16客户分级ABCD模型.md](references/16客户分级ABCD模型.md) 确定优先级
3. 参考 [18工作计划制定.md](references/18工作计划制定.md) 生成今日/本周任务
4. 输出完整的商机线索报告，包含：评分、等级、行动建议

### 规则5：拜访记录分析后自动更新状态
用户提交拜访记录后，**无需用户额外提示**，自动：
1. 参考 [19拜访记录分析.md](references/19拜访记录分析.md) 提取关键信息
2. 更新客户画像中的动态信息
3. 根据商机阶段自动推进漏斗
4. 参考 [20下一步行动建议.md](references/20下一步行动建议.md) 生成下一步行动
5. 询问是否需要加入今日工作计划

### 规则6：主动推荐下一步行动
用户询问"这个客户怎么跟进"或"下一步做什么"时，自动：
1. 参考 [20下一步行动建议.md](references/20下一步行动建议.md)
2. 结合当前商机阶段和客户态度
3. 输出具体可执行的行动建议（含话术、资料、准备事项）

### 信息收集优先级

| 优先级 | 来源 | 内容 |
|--------|------|------|
| P0 | 企业官网 | 简介、产品、新闻、投资者关系 |
| P1 | 财报/研报 | 财务数据、战略方向 |
| P1 | 招投标网 | 中标记录、采购需求 |
| P2 | 企查查/启信宝 | 股权架构、工商信息 |
| P2 | 行业报告 | 市场地位、竞争格局 |
| P3 | 脉脉/媒体 | 关键人信息、组织动态 |

### 信息时效性要求

- 财务数据：年度财报为准
- 股权结构：近3个月
- IT系统：近6个月
- 项目信息：近12个月
- 人员变动：近3个月
