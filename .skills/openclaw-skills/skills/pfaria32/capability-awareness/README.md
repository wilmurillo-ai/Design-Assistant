# Capability Awareness System

Makes OpenClaw agents aware of custom skills and capabilities. Skills-first approach with on-demand documentation loading. Zero overhead when not in use. Foundation for skill marketplace discovery.

## Features

- Skill descriptions in agent prompt
- On-demand SKILL.md loading when relevant
- Zero overhead when skills not in use
- Simple skills-first approach (no complex routing)
- Foundation for skill ecosystem

## Use Cases

- Make agents aware of custom capabilities
- Enable skill discovery without complexity
- Build toward skill marketplace
- Improve agent self-awareness

## Installation

```bash
clawhub install capability-awareness
```
