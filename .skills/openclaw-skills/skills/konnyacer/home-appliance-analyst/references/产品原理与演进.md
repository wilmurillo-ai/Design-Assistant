# 产品原理与技术演进 — 分析方法论

本文档是分析框架，不是百科全书。具体品类的原理与演进路线见 `品类知识库/` 目录。

---

## 一、原理分析通用框架

分析任何家电产品原理时，按以下维度展开：

### 核心三问

1. **这个产品是什么能量转换？**
   - 电能→热能（加热类）
   - 电能→动能（电机类）
   - 能量搬运（热泵类）
   - 电化/化学反应（消毒类）

2. **核心元器件是什么？谁是成本大户？**
   - 列出该品类3~5个关键元器件
   - 注明各元器件的主流方案和代表供应商

3. **主要技术路线差异在哪？**
   - 找出各技术路线的核心差异维度（效率/寿命/成本/安全性）
   - 列出各路线的代表品牌/机型

## 四、技术路线共性规律（跨品类适用）

以下规律在多个家电品类中反复出现：

| 规律 | 说明 | 出现品类 |
|------|------|---------|
| **定频→变频** | 电机/压缩机从固定转速到可调转速 → 能效提升、控温精准、噪音降低 | 空调、冰箱、洗衣机、油烟机、净水器 |
| **有刷→无刷（BLDC）** | 电机换向方式变化 → 效率提升、寿命延长、噪音降低 | 吸尘器、吹风机、破壁机、扫地机器人 |
| **机械→电子** | 机械开关/恒温器 → 单片机/传感器精准控制 → 功能丰富、精度提升 | 几乎所有品类 |
| **单一功能→多功能集成** | 单机 → 多合一（洗地机+吸尘器、集成灶等） | 厨房电器、清洁电器 |
| **非智能→联网智控** | 本地控制 → WIFI/蓝牙APP → AI自适应 | 空调、扫地机器人、净水器、空气净化器 |
| **非可视→数显/APP可视化** | 状态不可知 → 实时数据/历史记录 → 用户决策更理性 | 空气净化器（甲醛数显）、净水器（TDS）、扫地机器人（地图） |
| **耗材被动换→主动提醒** | 用户不知何时换滤芯 → APP提醒+智能监测 → 体验提升 | 空气净化器、净水器、扫地机器人 |
| **过滤介质迭代** | 活性炭→HEPA→静电→光触媒 | 空气净化器 |
| **加热方式升级** | 电阻→PTC→IH/电磁→红外/远红外 | 电饭煲、取暖器、烤箱 |

## 五、跨品类类比推理指南

当遇到知识库里没有的品类时，用以下类比路径推理：

| 待分析品类 | 最近似品类 | 可类比的技术演进规律 |
|-----------|-----------|------------------|
| 除湿机 | 空气净化器/空调 | 压缩机→变频→智能湿度控制；过滤系统可类比空气净化器 |
| 加湿器 | 除湿机（反过程） | 超声波/蒸发式类似空气净化器的滤网层级进化 |
| 擦窗机器人 | 扫地机器人 | LDS导航→视觉→结构光，演进规律完全相同；真空吸附原理类比 |
| 挂烫机 | 电熨斗/吹风机 | 发热原理同PTC/电阻丝，蒸汽原理类比加湿器 |
| 净水器 | 空气净化器 | 过滤层级逻辑相同（粗滤→精滤→末端），RO类比HEPA为最细过滤 |
| 浴霸 | 取暖器/空调 | 灯暖（红外辐射）→风暖（PTC热风），类比小太阳→暖风机 |
| 干衣机 | 洗衣机/空调 | 热泵干衣≈空调制热原理（热泵循环），冷凝≈空调除湿 |
| 吸尘器 | 洗地机/吹风机 | 有刷→BLDC电机演进相同，风机技术类比吹风机高速电机 |
