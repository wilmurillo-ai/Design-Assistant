# 信息搜索方法论

## 数据优先级原则

**原则：权威来源 > 第三方平台 > 社交媒体 > 品牌自述**

| 信息类型 | 首选来源 | 次选来源 | 禁用作数据源 |
|---------|---------|---------|------------|
| 市场数据 | 奥维云网、GfK、中怡康、Statista | 艾瑞、头豹、洞见研报 | 自媒体文章 |
| 价格数据 | 京东/天猫/品牌官网（实时） | 什么值得买历史价格 | 社交媒体报价 |
| 产品参数 | 品牌官网/产品页 | 京东SKU页 | 第三方测评 |
| 用户口碑 | 京东/天猫真实评价 | 小红书（带图） | 品牌评论区 |
| 供应链信息 | 上市公司财报（招股书/年报） | 行业协会报告 | 论坛帖 |
| 合规标准 | 国家标准全文公开系统（samr.gov.cn） | 工信部/认监委官网 | 第三方网站摘录 |

## 各类型信息搜索路径

### 行业市场数据

```
搜索词示例：
"[品类] 市场规模 2024"
"[品类] 奥维云网"
"[品类] GfK 中国"
"[品类] 市场报告"

重点来源：
- 奥维云网 avcidata.com（家电零售监测，最权威）
- GfK 中怡康（中怡康零售监测）
- Statista.com（全球数据）
- 前瞻产业研究院、中国报告大厅
- 艾瑞咨询、头豹研究院
- 巨量算数（抖音/字节系品类数据）
```

### 产品比价

```
搜索词示例：
"[型号] 价格"
"[型号] 京东"
"[型号] 参数"
"[型号] 对比"
"[品类] 性价比 推荐"

重点来源：
- 京东（销量/评价最真实）
- 天猫（品牌控价参考）
- 拼多多（低价参考）
- 什么值得买（历史价格走势）
- 价格雷达（各平台比价）
```

### 用户痛点分析

```
搜索词示例：
"[型号] 差评"
"[型号] 后悔"
"[品类] 常见问题"
"[品类] 智商税"
"[型号] 真实评价"

重点来源：
- 京东评价（按差评排序）
- 天猫追评（使用后反馈）
- 小红书（真实使用体验）
- 知乎（深度讨论）
```

### 供应链与成本

```
搜索词示例：
"[部件名称] 供应商"
"[品类] 物料成本"
"[原材料] 价格走势"
"[压缩机/电机] 品牌"

重点来源：
- 上市公司年报/招股书（供应商信息、成本结构）
- 同花顺/东方财富（行业研报）
- 上海有色网（金属原材料）
- 卓创资讯/隆众资讯（化工原材料）
- 产业在线（chinaiol.com，家电产业链数据）
```

### 合规标准

```
搜索词示例：
"[品类] 国家标准"
"[品类] 3C认证"
"[品类] 能效标准"
"[品类] GB 4706"

重点来源：
- 国家标准全文公开系统：https://open.samr.gov.cn/
- 中国认证认可信息网：www.cnca.gov.cn
- 工信部标准定额司
- 认监委官网
```

### 竞品分析

```
搜索词示例：
"[品类] 品牌格局"
"[品类] CR3"
"[品牌] 市占率"
"[品类] 竞品分析"

重点来源：
- 奥维云网（零售监测）
- 公司年报（竞争对手披露）
- 艾瑞/头豹行业报告
```

## 品牌财报重点财务数据读取指南

上市公司年报中与家电分析最相关的数据：

- **营收构成**：各品类营收占比（找到想分析的具体品类）
- **毛利率**：各品类毛利率（反映品牌溢价能力）
- **关联交易**：向关联方采购原材料/零部件的价格（供应链成本参考）
- **前五大供应商/客户**：了解供应链结构
- **存货周转**：判断产品销售速度
- **研发投入**：新技术方向

重点公司：
- 美的集团（000333）：全品类
- 格力电器（000651）：空调为核心
- 海尔智家（600690）：冰箱/洗衣机/高端品牌矩阵
- 苏泊尔（002032）：厨房小电
- 九阳股份（002242）：厨房小电
- 小熊电器（002959）：创意小电
- 石头科技（688169）：扫地机器人
- 火星人（300894）：集成灶
