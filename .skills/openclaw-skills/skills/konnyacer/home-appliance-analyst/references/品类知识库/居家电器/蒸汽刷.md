# 蒸汽刷 / 干洗刷 / 挂烫刷

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 功率 | 800~2000W | 值越大出蒸汽越快 |
| 蒸汽量 | 20~40g/min | 值越大熨烫越快 |
| 水箱 | 50~300mL | 手持款50~100mL；立式款200mL+ |
| 预热时间 | 15~60秒 | 手持快，立式慢 |
| 类型 | 手持蒸汽刷/立式蒸汽挂烫刷 | 按使用频率选择 |

---

## 与挂烫机的区别

| 对比 | 挂烫机 | 蒸汽刷（手持） |
|------|--------|--------------|
| 体积 | 大（立式+衣架） | 小（手持，迷你） |
| 适用场景 | 整件衣物/西装/大衣 | 局部褶皱/出差旅行 |
| 功率 | 1000~3000W | 800~1500W |
| 蒸汽量 | 大（40~90g/min） | 中（20~40g/min） |
| 储水量 | 大（1~3L） | 小（50~200mL） |
| 衣架要求 | 需要专用衣架 | 不需要 |

---

## 技术演进路线

```
[第一代] 简易手持蒸汽刷（约2010~2015年）
- 核心原理：小型加热器+水箱→产生蒸汽→手持刷头喷出蒸汽→衣物表面
- 核心元器件：PTC加热片+小水泵+ABS外壳+尼龙刷毛
- 主要局限：蒸汽量小（20~25g/min），只能处理轻微褶皱

[第二代] 大蒸汽手持刷（约2015~2020年）
- 核心升级：
  - 功率提升（1200~1800W）
  - 蒸汽量提升（30~40g/min）
  - 即热技术（30秒即热）
  - 配套毛刷/扁嘴等多种配件
  - 可平烫（加装小型熨烫底板）

[第三代] 智能蒸汽刷+杀菌（约2020~至今）
- 核心升级：
  - APP/温度显示（实时了解当前温度）
  - 持续大蒸汽（不间断蒸汽输出）
  - 杀菌除螨功能（蒸汽温度>100℃，杀菌率99.9%）
  - 纳米釉面底板（光滑不粘，防护衣物）
  - 除静电功能（减少衣物静电）
```
