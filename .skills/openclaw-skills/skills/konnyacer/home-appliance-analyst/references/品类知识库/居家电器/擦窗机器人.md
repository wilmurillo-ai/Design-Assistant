# 擦窗机器人

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 清洁速度 | 2~5㎡/min | 窗户面积/分钟 |
| 吸力 | 2500~5000Pa | 值越大越安全 |
| 噪音 | 60~75dB | 清洁时噪音较大 |
| 电池 | 内置锂电池（约500mAh） | 断电保护备用 |
| 适用玻璃厚度 | 单层/双层/三层中空 | 需确认是否适配 |
| 安全绳 | 承重50~150kg | 必须连接，防止坠落 |

---

## 主要类型与技术路线

| 类型 | 原理 | 优点 | 缺点 | 适用场景 |
|------|------|------|------|---------|
| **真空吸附式** | 电机抽真空，吸盘紧贴玻璃 | 对光滑表面吸附强 | 对粗糙表面（如磨砂玻璃）吸附差 | 光滑玻璃/镜子/瓷砖 |
| **风机吸附式** | 大功率风机产生负压吸附 | 吸附力强，可用于粗糙面 | 噪音大（>70dB） | 有缝隙墙面/瓷砖 |
| **磁吸式（无绳）** | 内外两块磁铁夹住玻璃 | 无需安全绳 | 需匹配玻璃厚度；已淘汰 | 双层中空玻璃（早期） |

## 技术演进路线

```
[第一代] 磁吸式擦窗机（约2008~2012年）
- 核心原理：外侧磁块+内侧把手，通过磁力夹住玻璃，用户移动把手带动擦布清洁
- 核心元器件：永磁铁（钕铁硼）、纤维擦布
- 主要特点：
  - 无需电源，完全物理吸附
  - 需匹配玻璃厚度（单层/双层/三层各有对应磁力规格）
  - 笨重（两块磁铁操作繁琐）
  - 已基本被市场淘汰
- 代表：iRobot（早期尝试）、国内仿制品牌

[第二代] 真空风机吸附机器人（约2012~2017年）
- 核心突破：风机产生负压，机器人自动在玻璃上移动
- 核心原理：大功率风机抽走机器人底部与玻璃之间的空气→大气压将机器人压在玻璃上→履带/轮子驱动机器人移动
- 核心元器件：
  - 真空风机（BLDC电机，吸力2500~3500Pa）
  - 履带轮（驱动机器人移动）
  - MCU主控板
  - 边缘检测传感器（防坠落）
  - 安全绳（高强度纤维绳，防坠落）
  - 锂电池（断电备用保护，续航~30分钟）
  - 清洁布（超细纤维布，湿擦）
- 主要升级：
  - 自动清洁（机器人自主移动，无需人工操作移动路径）
  - 边缘检测（防止从玻璃边缘跌落）
  - 断电保护（内置电池+安全绳，防止坠落）
  - 遥控器控制（部分机型）
- 代表品牌：科沃斯（窗宝系列）、玻妞（Hobot）

[第三代] 智能规划+喷水擦窗（约2017~2020年）
- 核心升级：
  - **弓字形路径规划**（类似扫地机器人的SLAM算法，路径不重复，覆盖率高）
  - **自动喷水**（内置水箱+微型水泵，喷水湿润玻璃→清洁布擦拭）
  - **双边雾化喷水**（水雾均匀覆盖，清洁效果更好）
  - **智能边缘感应**（精确检测窗户边框，防止跌落）
  - **遥控+APP双控制**
- 核心元器件：
  - 水箱+微型隔膜泵（自动喷水）
  - 履带轮防滑设计（应对湿滑玻璃）
  - 激光测距/边缘传感器
- 代表品牌：
  - 科沃斯（窗宝WINBOT 880/920）
  - 玻妞（Hobot 388/268）
  - 小米（擦窗机器人1代）
- 主要局限：
  - 清洁速度仍然偏慢（2~3㎡/min）
  - 对无边框玻璃（淋浴房）吸附不稳定

[第四代] 大吸力+多功能+精细清洁（约2020~至今）
- 核心升级：
  - **超大风压**（吸力提升至4000~5000Pa，可在粗糙表面工作）
  - **恒压悬浮系统**（擦布始终保持合适压力，清洁均匀）
  - **双向交叉雾化喷水**（两组喷嘴双向喷水，覆盖面积更大）
  - **自动感应脏污程度**（部分高端机型，根据脏污自动调节喷水量）
  - **擦刮同步**（先刮后擦，清洁效果接近人工）
  - **超薄设计**（9cm以下，可清洁防盗窗内玻璃）
- 核心元器件：
  - 无刷直流风机（更大吸力）
  - 双向喷水系统（2组微型泵+雾化喷嘴）
  - 交叉清洁布（先湿擦后干擦）
- 代表品牌：
  - 科沃斯（窗宝W1 PRO，2022年旗舰）
  - 玻妞（Hobot 2S/3S系列）
  - 赫特（HOBOT）—— 专注擦窗机器人
  - 追觅（尚未入局）

[第五代] AI智能化+全场景（约2023~至今）
- 核心升级：
  - **AI路径优化**（深度学习优化路径，覆盖率更高）
  - **超静音设计**（70dB→60dB以下）
  - **多种表面适用**（玻璃/瓷砖/大理石/木质墙面）
  - **全屋清洁**（不止擦窗，拓展至浴室镜/厨房墙面/地板）
  - **自动加液**（内置清洁液仓，一次加满多次使用）
```

## 选购核心维度

1. **吸附方式**：真空风机（主流） > 磁吸（已淘汰）
2. **吸力**：>3000Pa为安全线（越高越好）
3. **清洁功能**：自动喷水（湿擦）> 无水（干擦）
4. **路径规划**：弓字形规划 > 随机路线
5. **安全保护**：安全绳（必须）+ 断电保护 + 边缘传感
6. **噪音**：<65dB为优秀（70dB以上较吵）
7. **适用场景**：有无边框（部分机型对无边框支持差）
8. **超薄设计**：如有防盗窗，选<9cm厚度
