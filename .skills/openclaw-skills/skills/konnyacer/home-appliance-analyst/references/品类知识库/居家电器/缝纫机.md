# 缝纫机

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 缝纫速度 | 800~1300针/分钟 | 值越高效率越高 |
| 线迹类型 | 10~30种 | 线迹越多功能越全 |
| 最大针距 | 5mm | 决定最大针脚宽度 |
| 梭芯容量 | 小型/中型 | 决定连续缝纫时间 |
| 功率 | 50~100W | 家用机型 |
| 功能 | 锁边/钉扣/锁扣眼/刺绣/自动穿线 | 功能越多价格越高 |

---

## 主要类型

| 类型 | 特点 | 适合场景 |
|------|------|---------|
| **机械式缝纫机** | 手动调节，无电子控制 | 基础缝补，预算有限 |
| **电脑缝纫机** | LCD显示，微电脑控制 | 家用手工爱好者 |
| **电子缝纫机** | 步进电机驱动，精确控制 | 精细缝纫 |
| **工业缝纫机** | 高速（5000~10000针/分钟），专业用 | 裁缝店/服装厂 |
| **包缝机（锁边机）** | 专业锁边，防止布料脱散 | 面料锁边专用 |
| **刺绣机** | 大型/小型，专业刺绣 | 定制刺绣 |

## 技术演进路线

```
[第一代] 手动机械缝纫机（约1790s~1870s）
- 核心发明：1790年英国Thomas Saint发明第一台木质缝纫机（用于皮革）；1830年代法国Barthélemy Thimonnier工业化生产
- 核心原理：手持针穿透布料→线从针眼穿过→线环勾住另一根线→收紧成结（链式线迹）
- 主要局限：
  - 结构粗糙，可靠性差
  - 只能缝直线，速度慢
  - 专利争议（Walter Hunt vs Elias Howe vs Isaac Merritt Singer）
- 重要事件：1846年Elias Howe获得美国缝纫机专利（可缝锁式线迹）

[第二代] 工业缝纫机（约1850s~1950s）
- 核心突破：Singer（胜家）1855年推出商业化缝纫机，1860年代大量普及
- 核心技术：锁式线迹（双线连锁，牢固）；踏板驱动（解放双手）
- 核心元器件：主轴、曲柄连杆机构、挑线杆、送布牙、梭芯机构
- 发展里程碑：
  1860s-1880s：脚踏驱动普及
  1890s-1920s：电动机替代脚踏，缝纫速度从200→1000针/分钟
  1930s-1950s：工业缝纫机成熟，自动化程度提升
- 代表：Singer（美国）、胜家（JUKI前身）、兄弟（Brother，1908年）

[第三代] 家用电动缝纫机（约1950s~1990s）
- 核心升级：交流电动机替代脚踏板
- 核心技术变化：
  - 电机驱动（踩踏板→按电键，解放双脚）
  - 变速控制（转速无级可调）
  - 张力调节（上线/下线张力独立调节）
  - 多种线迹（直线、锯齿、锁扣眼等）
- 核心元器件：
  - 交流电机（功率50~70W）
  - 针杆机构（上下运动，穿线）
  - 送布牙（前进/后退送布）
  - 梭芯机构（旋转供线）
  - 挑线杆（收紧线迹）
- 代表品牌：
  - 胜家（Singer）— 1960~80年代全球垄断
  - 兄弟（Brother）— 1970年代日本电子化先驱
  - 真善美（Janome）— 日本机械品质代表

[第四代] 电脑缝纫机（约1990s~2015年）
- 核心突破：从机械控制→微电脑芯片控制
- 核心技术升级：
  - **微电脑控制**：LCD显示屏，设置参数一键完成
  - **步进电机驱动**：取代传统皮带轮传动，精确控制每一针
  - **自动穿线**：一键自动穿线，降低使用门槛
  - **线迹存储**：预设线迹从10种→30~200种
  - **自动张力调节**：根据面料厚度自动调整上线张力
  - **一键倒缝**：自动倒缝加固线迹
  - **自由绣花**：配合专用绣花框，可做小型刺绣
- 核心元器件：
  - 微电脑控制板（MCU芯片）
  - 步进电机×2~3个（主轴+送布+挑线）
  - LCD液晶显示屏（单色或彩色）
  - 编码器（实时检测针位置）
- 代表品牌：
  - 兄弟（Brother）— 开创电脑缝纫机时代，Innov-is系列
  - 真善美（Janome）— 日本高端电脑机
  - 重机（JUKI）— 从工业缝纫机延伸至家用高端（HZL系列）
  - 胜家（Singer）— 相对落后于日本品牌

[第五代] 智能缝纫机（约2015~至今）
- 核心突破：AI辅助、大屏触控、WIFI连接、云端线迹库
- 核心技术升级：
  - **大屏彩色触控**（3~5寸触摸屏，类似智能手机操作）
  - **AI面料感应**：自动识别面料类型（棉/丝/牛仔）→自动设置针距/张力/压力
  - **WIFI线迹下载**：联网下载设计师线迹图案
  - **自动穿线升级**：激光/摄像头引导穿线
  - **USB接口**：导入导出绣花图案
  - **语音助手**：语音提示操作步骤（适合新手）
  - **LED投影定位**：将刺绣图案投影到绣花区域，精准定位
- 代表品牌/产品：
  - 兄弟（Brother）— Innov-is V3（大型彩屏+300+线迹）
  - 真善美（Janome）— MC9400（7寸彩屏+950种线迹）
  - 胜家（Singer）— ONE+（大屏+WIFI，国内定位偏入门）
  - 哈根（Husqvarna）— Emerald系列（瑞典，高端）

## 核心元器件详解

### 锁式线迹原理（最常见的缝纫原理）
```
上下两根线（A=上线/B=下线）：

1. 针带上线A下刺布料（A从针眼穿过）
2. 针到达最低点，梭钩抓住上线A的线环
3. 梭旋转，将线A绕住下线B
4. 针回升，收紧线环，形成牢固的锁式线迹
5. 送布牙前移，布料前进一个针距
```

### 送布牙（送布系统）
- 位于针板下方，带锯齿的金属片
- 每次针刺入→送布牙前进→布料移动一个针距
- 精细面料：送布牙高度调低
- 厚重面料：送布牙高度调高

## 工业 vs 家用：核心差异

| 维度 | 工业缝纫机 | 家用电脑缝纫机 |
|------|-----------|--------------|
| 速度 | 3000~10000针/分钟 | 800~1300针/分钟 |
| 连续工作时间 | 24小时连续运转 | 建议<2小时/次 |
| 线迹 | 单一（直线/锁边） | 10~500种 |
| 适用面料 | 牛仔布/皮革/薄纱/工业布料 | 全品类家用面料 |
| 操作难度 | 高（需专业技能） | 低（有引导） |
| 价格 | 2000~30000元 | 300~8000元 |

## 选购核心维度

1. **类型**：日常基础缝补（机械式入门即可）；手工爱好者（电脑款）；精细刺绣（带绣花功能）
2. **线迹数量**：越多越贵，>30种为佳
3. **自动功能**：自动穿线 > 自动倒缝 > 自动张力
4. **适用面料**：厚布料（牛仔/帆布）需大功率；丝绸需精细张力调节
5. **刺绣功能**：需要小型刺绣选带绣花功能款
6. **品牌选择**：兄弟（综合强）、真善美（机械品质）、JUKI（工业延伸）、胜家（历史品牌，功能偏弱）
