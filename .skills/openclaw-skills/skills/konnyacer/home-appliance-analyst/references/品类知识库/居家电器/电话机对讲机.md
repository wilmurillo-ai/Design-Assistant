# 电话机 / 对讲机

## 电话机

### 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 类型 | 有线座机/无线子母机/录音电话 | 按场景选择 |
| 来电显示 | FSK/DTMF双制式 | 现代必备 |
| 屏幕 | LCD显示/无屏 | 录音电话必备 |
| 功能 | 免提/重拨/静音/三方通话 | 高端功能更多 |

### 技术演进路线

```
[第一代] 有线旋转拨号电话（约1950s~2000年）
- 机械旋转拨号盘（脉冲拨号），通过拨盘闭合电路次数传输号码
- 1980年代被按键式（DTMF）取代

[第二代] 有线按键电话（约1980~2010年）
- 按键式DTMF双音多频，拨号快速准确
- FSK/DTMF来电显示
- 免提通话、号码存储等功能

[第三代] 无线子母机（约1995~至今）
- 模拟/DECT无绳电话，室内可移动通话
- 子机充电底座+母机+多子机扩展
- 录音电话：内置答录机（可存储留言）

[第四代] 网络电话+智能座机（约2015~至今）
- VOIP网络电话（WiFi/以太网连接）
- 触屏智能座机（类平板，可视频通话）
- AI录音转写（通话内容实时转文字）
```

## 对讲机

### 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 类型 | 民用对讲机/物业对讲/楼宇对讲/婴儿监护器 | 场景差异大 |
| 频段 | 409MHz（民用免执照）/ UHF/VHF | 距离决定频段 |
| 通话距离 | 3~10km（室外开阔地） | 城市内大幅缩短 |
| 功率 | 0.5~5W | 值越大距离越远 |
| 供电 | 锂电池/AA电池 | 可换电池更方便 |

### 技术演进路线

```
[第一代] 模拟对讲机（约1980s~2010年）
- FM模拟信号传输
- 功率0.5~5W，通话距离有限
- 代表：Motorola Talkabout系列

[第二代] 数字对讲机（约2010~至今）
- DMR数字标准（欧洲ETSI标准）
- 更清晰的语音质量
- 更低的功耗
- 支持短信传输

[第三代] 公网对讲机（约2015~至今）
- 基于4G/5G网络，无需自建基站
- 全国范围内对讲（有手机信号即可）
- 无通话距离限制
- 代表：海能达（Hytera）、对讲机APP（如对讲机微信小程序）
```

## 楼宇对讲/可视对讲（家用）

```
[第一代] 语音对讲（约1990s~2010年）
- 门口机（按键呼叫）+ 室内分机（听筒对讲）
- 局限性：无法看到访客

[第二代] 可视对讲（约2005~2015年）
- 门口摄像头+室内LCD屏幕
- 开门按钮（电动门锁）
- 访客留影/留言功能

[第三代] 智能云对讲（约2015~至今）
- APP远程接听（不在家也能开门口锁）
- 人脸识别开门
- 云端存储访客记录
- 与智能门锁联动（临时密码/远程解锁）
```
