# 擦鞋器 / 刷鞋机

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 类型 | 手持式/立式/鞋油一体 | 家用→手持；商用→立式 |
| 刷子材质 | 马毛/尼龙/超细纤维 | 马毛最传统，尼龙耐磨 |
| 功能 | 上油/抛光/除尘 | 高端一体机全功能 |
| 安全 | 过热保护 | 电动型必须有 |

---

## 技术演进路线

```
[第一代] 手动擦鞋器（约1950s~2000年）
- 手持鞋油盒+刷子/绒布，手动擦鞋

[第二代] 电动抛光擦鞋器（约2000~2015年）
- 小型电机驱动绒布轮，高速旋转抛光
- 家用手持款（2~5W），商业立式（20~50W）

[第三代] 智能擦鞋一体机（约2015~至今）
- 上油+抛光+除尘三合一
- 红外感应（放鞋自动启动）
- UV杀菌（上油同时消毒）
- 代表：商用立式擦鞋机（机场/酒店/写字楼）
```
