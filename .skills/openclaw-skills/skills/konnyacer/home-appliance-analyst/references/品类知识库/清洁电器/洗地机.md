# 洗地机

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 吸力/滚刷转速 | 5000~10000次/分钟（滚刷） | 值越高擦地越干净 |
| 清水箱 | 500~900mL | 决定单次清洁面积 |
| 污水箱 | 500~720mL | 需小于清水箱 |
| 续航 | 25~60分钟 | 强档约10~20分钟 |
| 自清洁 | 离心甩干/热风烘干 | 热风烘干防异味 |
| 噪音 | 70~85dB | 建议选带降噪款 |

---

## 主要类型

| 类型 | 特点 | 适合场景 |
|------|------|---------|
| **单滚刷洗地机** | 结构简单，操控灵活 | 日常清洁 |
| **双滚刷洗地机** | 清洁效率更高，一次拖两遍 | 大面积 |
| **添可式（单滚刷+智能）** | 智能变频+APP | 智能体验 |
| **追觅式（双滚刷+双向助力）** | 双向驱动，推拉都省力 | 力气小的用户 |
| **干湿分离洗地机** | 污水和垃圾分开 | 减少倾倒难度 |

## 技术演进路线

```
[第一代] 基础洗地机（约2016~2019年）
- 核心原理：清水喷出→滚刷擦地→吸力回收污水→污水箱存储
- 核心元器件：直流电机（吸水）、清水泵、污水泵、滚刷（PVC/绒布）
- 主要局限：
  - 清水污水共箱（容易产生异味）
  - 无自清洁（滚刷需手动清洗）
  - 续航短（约20~30分钟）
  - 操控重（重心设计差）
- 代表：早期BISSELL（必胜）产品，国内市场较少

[第二代] 智能洗地机（约2020~2021年）
- 核心突破：添可（TINECO）芙万系列引爆中国市场
- 核心技术：单片机智能感应地面脏污程度→自动调节出水量和吸力
- 核心元器件：
  - 智能脏污传感器（红外光电传感器检测水体浊度）
  - 变频控制（MCU调节清水泵+吸水电机功率）
  - 活水清洁系统（清水持续喷淋滚刷，滚刷实时自清洁）
  - 自清洁底座（放回充电座→自动清洗滚刷+管道）
- 主要升级：
  - 活水系统（清水管和污水管分离，滚刷始终干净）
  - 智能变频（根据地面脏污自动档位，无需手动调节）
  - 语音提示（状态提醒、故障提醒）
  - 自清洁（放回底座，一键清洗滚刷）
- 代表：添可芙万1.0（2020爆款）、美的洗地机

[第三代] 双滚刷+双向驱动（约2021~2022年）
- 核心突破：追觅H系列首创双向驱动（推拉都省力）
- 核心技术：
  - 双滚刷（两个滚刷同向或对向旋转，一次清洁两遍）
  - 双电机驱动（滚刷电机+轮毂电机，提供前后双向助力）
  - 固液分离（干垃圾和液体分开收集）
- 主要升级：
  - 清洁效率：双滚刷一次清洁面积更大
  - 操控体验：双向驱动，老人/女性使用更轻松
  - 边角清洁：双滚刷覆盖更好
- 代表：追觅H12/H13、石头（Roborock）洗地机

[第四代] 热风烘干+超长续航（约2022~至今）
- 核心升级：
  - **热风烘干**（自清洁后热风吹干滚刷，解决异味问题）
  - **电解水杀菌**（清水箱内置电解模块，将水分解为含氯消毒剂，杀菌率99.9%）
  - **长续航**（电池优化，标准档60分钟）
  - **180°躺平**（机身可完全放倒，深入床底/沙发底清洁）
- 核心元器件：
  - PTC加热模块（热风烘干，30~60分钟烘干）
  - 电解水模块（电解槽，阳极氧化钛涂层）
  - 180°灵活转轴（万向轮设计）
- 代表：添可芙万3.0（离心烘干+智能照明）、追觅H13 Pro（双向助力+热风烘干）、滴水（DIISEA）洗地机（水尘环流，噪音极低）

[第五代] 旗舰全能基站（约2023~至今）
- 核心升级：基站集成自动上下水+自动添加清洗液+滚刷UV杀菌
- 方向1（添可路线）：智能变频+AI脏污识别+空间站（自动集尘+加水+充电）
- 方向2（追觅路线）：双滚刷+大吸力+180°躺平+热风烘干
- 方向3（小米/石头路线）：高性价比+米家/石头APP生态
```

## 添可 vs 追觅：核心差异

| 维度 | 添可（芙万） | 追觅（H系列） |
|------|------------|-------------|
| 滚刷 | 单滚刷 | 双滚刷 |
| 助力 | 单向（向前） | 双向驱动（推拉都行） |
| 智能化 | 脏污感应（芙万杀手锏） | 吸力大，但智能脏污感应稍弱 |
| 清洁效率 | 中等 | 双滚刷更快 |
| 躺平 | 部分支持 | 全系180°躺平 |
| 生态 | 独立品牌 | 小米生态链 |

## 选购核心维度

1. **滚刷结构**：双滚刷（清洁效率高） vs 单滚刷（灵活）
2. **自清洁方式**：离心甩干（快但滚刷潮湿） vs 热风烘干（彻底但耗时）
3. **助力系统**：有无双向驱动（老人/女性推荐双向）
4. **续航**：标准档>40分钟，强档>15分钟
5. **电解水/UV杀菌**：有则更卫生
6. **能否躺平**：180°躺平对家具底部清洁至关重要
