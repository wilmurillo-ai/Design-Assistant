# 电动拖把 / 电动扫把

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 类型 | 电动拖把（湿拖）/电动扫把（干扫） | 两者功能不同 |
| 功率 | 20~150W（无线） | 值越大动力越强 |
| 转速 | 1000~3000转/分钟 | 电动拖把转速决定擦地力 |
| 续航 | 15~60分钟 | 无线使用 |
| 重量 | 1~3kg | 轻便性 |

---

## 技术演进路线

```
[电动拖把]
[第一代] 简单旋转拖把（约2008~2015年）
- 核心原理：两个旋转拖布盘，电机驱动旋转，代替手动拖地
- 局限：需外接电源，拖布面积小

[第二代] 无线旋转电动拖把（约2015~至今）
- 核心升级：
  - 充电锂电池（无线使用）
  - 双向旋转拖布
  - 喷水功能（按键喷水，湿拖+擦干同步）
  - 自清洁桶（部分机型，拖布放桶内自动清洗甩干）
- 代表品牌：洒哇地咔（SWDK）—— 国内电动拖把领先品牌、博世（BOSCH）

[第三代] 蒸汽电动拖把（约2020~至今）
- 核心突破：电动拖把+蒸汽加热，杀菌同时强力去污
- 特点：即热式蒸汽（10秒出蒸汽）+ 旋转擦布 = 电动拖把+蒸汽拖把二合一

---

[电动扫把]
[第一代] 电动清洁刷（约2010~2015年）
- 核心原理：电动刷头（旋转刷毛）代替手动扫帚
- 局限：只能清洁光滑地面

[第二代] 无线电动扫把（约2015~至今）
- 核心升级：
  - 无刷电机（低噪音）
  - LED灯（照亮角落灰尘）
  - 扁薄设计（深入家具底部）
  - 可换刷头（地板刷/地毯刷）
- 核心场景：代替传统扫帚，轻便省力，尤其适合清洁毛发和灰尘
- 代表：戴森（Dyson）V12 Detect Slim（绿光显尘电动扫把）
```
