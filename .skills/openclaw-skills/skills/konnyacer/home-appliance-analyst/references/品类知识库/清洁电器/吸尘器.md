# 吸尘器

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 吸力 | 10~30kPa（有线）/ 5~20kPa（无线手持） | kPa值越高越好 |
| 功率 | 100~2000W（有线）/ 300~800W（无线） | 无线功率受电池限制 |
| 续航 | 15~90分钟（无线） | 实际续航受档位影响大 |
| 尘杯容量 | 0.3~2L | 容量越大清理频率越低 |
| 噪音 | 60~85dB | 建议<75dB |

---

## 主要类型

| 类型 | 形态 | 核心原理 | 适合场景 |
|------|------|---------|---------|
| **桶式/卧式** | 有线大型 | 电机带动叶轮产生负压→吸入灰尘 | 大面积清洁、综合最强 |
| **立式** | 有线，机身一体 | 同上，但一体化立式设计 | 家庭主力清洁 |
| **手持式** | 无绳，轻便 | 充电电池供电，小型电机 | 车内/沙发/键盘 |
| **推杆式（无线）** | 无绳主流形态 | 充电电池+BLDC电机 | 家庭日常使用 |
| **洗地机** | 无绳，功能复合 | 清水擦地+污水回收 | 地面湿拖 |
| **扫地机器人** | 全自动 | LDS导航+边刷+风机 | 全屋自动清扫 |

## 技术演进路线

```
[第一代] 有线桶式/卧式吸尘器（约1950s~2000s）
- 核心原理：串激电机（有刷）高速旋转→叶轮产生吸力→尘气分离（尘杯/集尘袋）
- 核心元器件：串激电机（iron core，噪音大）、HEPA前级过滤、集尘袋/尘杯
- 代表品牌：伊莱克斯、三星（早期）、美的（90年代国产跟进）
- 主要局限：
  - 串激电机（有刷）：碳刷磨损（寿命~500小时），噪音>80dB
  - 有线（拖线麻烦）
  - 功能单一（仅干吸）

[第二代] 无绳手持/推杆吸尘器（约2006~2015年）
- 核心突破：锂电池（18650电芯）+ BLDC无刷电机 → 无绳化成为可能
- 核心元器件：18650锂电池组（3.6V×n颗串联）、BLDC电机（转速每分钟3万转）、多级气旋分离（解决无尘袋后的吸力衰减问题）
- 代表：戴森V系列（V1约2006年开创无绳推杆品类）
- 关键升级：
  - 多级气旋分离（尘气彻底分离，滤网不堵塞→吸力不衰减）
  - BLDC电机（无刷，寿命长，效率高，转速可达10~15万转/分钟）
  - HEPA后级过滤（避免二次扬尘）
  - 轻量化（2~3kg vs 桶式的5~8kg）

[第三代] 智能无绳+多功能（约2016~2020年）
- 核心升级：电机功率/续航双提升 + 除螨刷/地毯刷等配件
- 核心元器件：21700电芯（续航提升30%）、BLDC电机（转速提升至12~15万转）、LED探照灯
- 代表国产品牌：追觅（2017年创立）、小狗、莱克（LEXY）
- 主要升级：
  - 吸力从100AW→200AW（戴森V11达185AW，追觅达200AW）
  - 续航从30分钟→90分钟（强力档仍偏短，约10~15分钟）
  - 附件丰富：除螨刷、扁嘴吸、二合一刷
  - 智能屏显（实时显示剩余电量/吸力档位）

[第四代] 吸拖一体+自动集尘（约2020~至今）
- 核心升级：前吸后拖（磁吸式水箱+拖布） + 自动集尘基站
- 核心元器件：自动集尘基站（电机+尘袋）、电控水箱（精准控水）、抗菌拖布
- 代表产品：
  - 戴森V12 Detect（激光探测微尘→可视化）
  - 追觅V16（16万转电机，210AW吸力）
  - 莱克吉米（莱克高端品牌）
- 近年趋势：
  - 绿光显尘（LED侧向照射，使地面微尘可见）
  - 智能吸力调节（根据地面类型自动调整）
```

## 吸尘器 vs 洗地机 vs 扫地机器人：怎么选？

| 维度 | 吸尘器 | 洗地机 | 扫地机器人 |
|------|--------|-------|-----------|
| 干垃圾 | ✅ 强 | ✅ 强 | ✅ 中等 |
| 湿垃圾/液体 | ❌（水尘款除外） | ✅✅ 强 | ❌ |
| 地毯深层清洁 | ✅✅ 强 | ❌ 弱 | ❌ |
| 全自动程度 | ❌（需人工） | ❌（需人工） | ✅✅ 全自动 |
| 地面维护清洁 | 中等 | ✅✅ 强 | ✅ |
| **推荐场景** | 有地毯/宠物/多场景 | 重度厨房/卫生间 | 日常维护/懒人 |
