# 沙发/布艺/地毯清洗机

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 功率 | 400~1200W | 值越大清洁力越强 |
| 吸力 | 10~20kPa | 吸走污水能力 |
| 清水箱 | 1~4L | 决定单次清洁面积 |
| 污水箱 | 0.8~3L | 收集污水 |
| 刷头类型 | 地毯刷/布艺刷/缝隙刷 | 多配件适用多场景 |

---

## 技术演进路线

```
[第一代] 简易布艺刷+吸尘器（约2000s~2010年）
- 核心原理：在手持吸尘器上加装布艺刷配件→刷头刷洗布面→吸力吸走污水
- 主要局限：刷洗能力弱，污水无法有效回收

[第二代] 专用沙发清洗机（约2010~2018年）
- 核心突破：净水+刷洗+吸水三合一
- 核心原理：清水喷出→刷头刷洗→大吸力吸走污水→污水箱收集
- 核心元器件：
  - 喷水泵（将清水喷向织物表面）
  - 刷洗电机（驱动刷头旋转）
  - 吸水电机（干湿两用电机）
  - 清水/污水双水箱
- 代表品牌：必胜（BISSELL）—— 沙发/地毯清洗机全球领导者

[第三代] 大功率+多功能（约2018~至今）
- 核心升级：
  - **功率提升**（450W→1200W，清洁力大幅提升）
  - **多种刷头**（地毯刷/布艺刷/硬地面刷/缝隙刷）
  - **热水清洁**（热水溶解油污效果更好）
  - **一键自清洁**（清洗管道和刷头）
  - **手柄延长杆**（可清洁大面积地毯/楼梯）
- 代表品牌：必胜（BISSELL）Pro系列、海尔（布艺清洗机）、卡赫（Kärcher）
```

---

## 选购核心维度

1. **吸力**：>15kPa为强
2. **刷头配件**：配件越丰富适用场景越多
3. **热水功能**：有则清洁油污更强
4. **自清洁功能**：有则维护更方便
