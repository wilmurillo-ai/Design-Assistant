# 扫地机器人

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 吸力 | 2000~11000Pa | Pa值越高吸力越强 |
| 导航方式 | 随机碰撞→LDS激光→视觉→结构光 | 导航决定智能化程度 |
| 电池/续航 | 5200mAh（最大）/ 3000mAh | 标准档约90~180分钟 |
| 尘杯容量 | 350~600mL | 容量与基站规格相关 |
| 水箱/拖布 | 80~300mL（基站版更大） | 拖地功能差异大 |
| 避障 | 红外→机械碰撞→视觉AI→结构光 | 决定是否卡困 |

---

## 主要类型与技术路线

| 导航代际 | 核心原理 | 优点 | 缺点 | 代表产品 |
|---------|---------|------|------|---------|
| **随机碰撞** | 随机方向移动，碰障反弹 | 便宜 | 覆盖率低、效率差 | 早期Roomba |
| **陀螺仪导航** | 惯性导航+随机路径 | 比随机好 | 无法建图、定位差 | 入门产品 |
| **LDS激光雷达** | 顶部旋转激光360°扫描建图 | 建图精准、可规划路径 | 无法识别低矮障碍物 | 小米1代、石头S系列 |
| **视觉vSLAM** | 顶部摄像头+算法建图 | 体积小、成本低 | 弱光/黑暗环境差 | iRobot高端 |
| **AI视觉避障** | LDS+AI摄像头识别障碍物类型 | 识别电线/宠物粪便 | 算法要求高 | 石头T7 Pro |
| **3D结构光** | LDS+结构光投射3D轮廓 | 精准3D避障 | 成本高 | 石头G10S、追觅X10 |

## 技术演进路线（按代际）

```
[第一代] 随机碰撞式（约1996~2008年）
- 核心原理：随机方向运动，碰障碍物后反弹，随机算法决定新方向
- 核心元器件：机械碰撞开关、红外测障（辅助）、有刷电机
- 代表：iRobot Roomba 400系列（1996）、伊莱克斯三叶虫（1999）
- 主要局限：覆盖率低（大量重复清扫同一区域，大量漏扫）

[第二代] 陀螺仪+摄像头辅助导航（约2008~2014年）
- 核心原理：陀螺仪记录移动方向+摄像头辅助视觉，记录已清扫区域
- 核心元器件：MEMS陀螺仪、摄像头、轮速计（记录移动距离）
- 代表：Neato（激光导航先驱，2010年推出Botvac）
- 局限：无法精确建图，地图精度差

[第三代] LDS激光雷达导航（约2010~至今，成熟期2016年后）
- 核心原理：顶部旋转激光发射→接收反射光→TOF测距→360°实时建图
- 核心元器件：
  - LDS激光模块（核心，约2万点/秒扫描）
  - LDS电机（带动激光头旋转）
  - 主控芯片（ARM Cortex-A系列）
  - 沿墙传感器（红外）
  - 悬崖传感器（防跌落）
- 代表：小米米家扫地机器人1代（2016，石头代工）、石头S5系列
- 核心优势：
  - 实时SLAM建图（手机可见清扫地图）
  - 路径规划（弓字形清扫，覆盖率高）
  - 虚拟墙/禁区（APP划区）
  - 指哪扫哪/断点续扫
- 局限：无法识别低矮/透明障碍物（电线、袜子、宠物粪便）

[第四代] LDS+AI视觉避障（约2018~至今）
- 核心原理：在LDS建图基础上，增加AI视觉摄像头，识别障碍物类型并智能绕行
- 核心元器件：
  - RGB摄像头（分辨率~200万像素）
  - AI视觉芯片（专用NPU，如地平线机器人AI芯片）
  - AI障碍识别算法（深度学习模型）
- 升级点：
  - 主动避障（识别后绕行而非碰撞）：电线、拖鞋、宠物粪便、玩具
  - 语义识别（识别家具位置，如"餐桌周围"）
- 代表：石头T7 Pro（AI双目避障）、科沃斯AIVI

[第五代] LDS+3D结构光/ToF（约2020~至今）
- 核心原理：3D结构光（或ToF传感器）投射光斑，精确测量障碍物3D轮廓
- 核心元器件：
  - 3D结构光发射器+摄像头
  - 或双摄ToF（两摄像头测距差计算深度）
  - LDS模块（同第三代）
- 升级点：
  - 可识别<3cm的细小障碍物
  - 暗光环境仍可精准避障
  - 从"碰撞避障"升级为"预判避障"
- 代表：石头G10S（AI 3D结构光）、追觅X10、科沃斯X1

[第六代] 全能基站+AI大模型（约2022~至今）
- 核心突破：基站集成自动集尘+自动洗拖布+自动上下水+热风烘干
- 核心元器件：
  - 基站：清水泵+污水泵+充电管理+集尘风机+自动上下水模块
  - 拖布：旋转式拖布（代替平板振动拖布）
  - AI芯片：端侧AI（地平线/高通QCS系列）
- 主要升级：
  - **自动集尘**（基站内置尘袋，1~2个月换一次）
  - **自动洗拖布**（回基站→清水冲洗→甩干→热风烘干）
  - **自动上下水**（接水管，不用手动加水/倒水）
  - **旋转加压拖地**（双盘旋转拖布，模拟人工下压力，拖地效果大幅提升）
  - **AI语义地图**（"清扫客厅沙发下面"等自然语言指令）
- 代表：石头G10S Pro、追觅X10、科沃斯X1 OMNI、云鲸J3

[第七代] AI大模型+具身智能萌芽（约2023~至今）
- 核心技术：LLM大模型+端侧AI，理解复杂语义指令
- 发展方向：
  - 语义地图+自然语言指令（"餐厅餐桌周围扫2遍"）
  - 具身智能：识别物体并主动规划（跨品类能力扩展）
  - 家庭环境感知：与智能家居联动（开灯→开始清扫）
```

## LDS激光雷达详解（核心元器件）

LDS是扫地机器人最重要的传感器，原理类似蝙蝠的超声波定位：

1. 发射激光脉冲（905nm近红外）
2. 测量光往返时间（TOF原理）
3. 360°旋转扫描，每秒~10圈
4. 实时生成2D平面地图（SLAM算法）

成本：LDS模块约100~200元/个，是扫地机器人最贵的零部件之一。

## 石头 vs 科沃斯 vs 云鲸 vs 追觅：核心差异

| 品牌 | 核心优势 | 代表系列 | 导航方案 | 拖地方案 |
|------|---------|---------|---------|---------|
| 石头（Roborock） | 算法最强、稳定性高 | G系列（旗舰）/S系列 | LDS+结构光 | 声波振动平板拖布 |
| 科沃斯（Ecovacs） | 产品线最全、国内最早 | X系列/T系列 | LDS+AI视觉 | 旋转加压拖布 |
| 云鲸（NARWAL） | 拖地效果最好、自动洗布 | J系列 | LDS激光 | 旋转加压拖布（拖地强项） |
| 追觅（Dreame） | 吸力最大、高性价比 | X系列/W系列 | LDS+AI避障 | 旋转拖布+热风烘干 |
| iRobot | 导航算法先驱 | Roomba i/s系列 | vSLAM（视觉） | 无拖地（欧美需求不同） |

## 选购核心维度

1. **导航方案**：LDS是基础（有LDS才有基本保障），AI避障推荐有宠/有孩家庭
2. **吸力**：>6000Pa为旗舰水平，>4000Pa为中端，>2500Pa为基础
3. **基站功能**：自动集尘（懒人必备） > 自动洗拖布 > 自动上下水
4. **拖地效果**：旋转加压 > 声波振动 > 平板拖布
5. **续航**：5200mAh→可清扫200㎡以上
6. **APP体验**：地图精度、禁区设置、指哪扫哪、语音控制
