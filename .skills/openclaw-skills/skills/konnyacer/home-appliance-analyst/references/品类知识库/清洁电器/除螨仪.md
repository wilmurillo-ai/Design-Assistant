# 除螨仪

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 拍打频率 | 5000~15000次/分钟 | 值越高拍出深层尘螨越有效 |
| 吸力 | 5~20kPa | 值越高吸螨虫效果越好 |
| 杀菌方式 | UV-C紫外/超声波/高温蒸汽 | UV-C最常见 |
| 功率 | 200~600W | 功率大通常吸力也大 |
| 滤网 | HEPA H12/H13 | 防止二次扬尘 |

---

## 技术演进路线

```
[第一代] 简单拍打+吸尘（约2005~2012年）
- 核心原理：小型电机驱动拍打棒（偏心轮）→拍打床铺→扬起尘螨→吸力吸入尘杯
- 核心元器件：拍打电机、吸尘电机（小型）、尘杯、无HEPA
- 主要局限：吸力弱，无杀菌，拍打不彻底

[第二代] UV-C紫外线杀菌（约2012~2018年）
- 核心升级：
  - **UV-C紫外灯**（253.7nm，杀菌/杀螨虫）
  - 拍打频率提升（8000~10000次/分钟）
  - HEPA滤网（避免二次扬尘）
  - **热风功能**（55~60℃热风，减少床铺湿度，抑制螨虫滋生）
- 核心元器件：
  - UV-C灯管（功率3~11W）
  - HEPA H12滤网
  - PTC加热器（热风）

[第三代] 超声波+强吸力（约2018~至今）
- 核心升级：
  - **超声波除螨**（特定频率超声波，干扰螨虫神经系统，使其脱离织物）
  - **强吸力**（15~20kPa，深层吸螨）
  - **紫外线+超声波双重**
  - **智能感应**（抬起自动关闭UV灯，防止伤人）
- 代表品牌：戴森（V7 Mattress手持除螨仪）、小狗（Dyson系追谁）、美的（Midea）、苏泊尔（SUPOR）

[第四代] 吸尘器除螨配件化（约2020~至今）
- 趋势：高端手持吸尘器标配除螨刷头，无需单独购买除螨仪
- 除螨仪成为独立品类的意义下降（已被手持吸尘器覆盖）
```

---

## 选购核心维度

1. **拍打频率**：>10000次/分钟为优秀
2. **吸力**：>10kPa为强
3. **杀菌方式**：UV-C（主流）+ 超声波（辅助）> 单一功能
4. **HEPA滤网**：H12/H13级（必需，防止二次污染）
5. **热风功能**：有则抑制螨虫滋生（55℃以上热风）
