# 超声波清洗机

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 清洗频率 | 40~60kHz | 值越低穿透力越强 |
| 功率 | 30~200W | 值越大清洗力越强 |
| 内槽容量 | 300mL~30L | 小型（眼镜/首饰）/大型（零件） |
| 定时 | 1~30分钟 | 自动清洗定时 |
| 加热温度 | 常温~70℃ | 加热提升清洗效果 |

## 技术演进路线

```
[第一代] 工业超声波清洗机（约1960s~1990s）
- 核心原理：超声波换能器（压电陶瓷）→产生40kHz超声波→液体中产生空化效应→冲击波剥离污垢
- 核心公式：空化泡在声压负压区膨胀→正压区突然收缩爆裂→产生局部高温高压→冲击波剥离污垢
- 主要局限：体积大、价格高，仅工业使用

[第二代] 小型超声波清洗机（约2000~2015年）
- 核心突破：体积小型化，成本大幅降低，进入家庭消费市场
- 核心元器件：
  - 压电陶瓷换能器（40~60kHz）
  - 超声波发生器（电路板）
  - 不锈钢内槽（sus304）
  - ABS外壳
- 主要应用：
  - **眼镜清洗**（最主流，缝隙中的污垢手工无法清理）
  - **首饰清洗**（项链/戒指/耳环）
  - **手表表带**（金属表带缝隙）
  - **牙套/假牙**（医疗辅助）
- 代表品牌：洁盟（深圳，专业超声波清洗机）、日本BK（Branson系）、小米（超声波清洗机，众筹版）

[第三代] 加热+UVC杀菌（约2018~至今）
- 核心升级：
  - **加热功能**（40~70℃热水，清洗更彻底）
  - **UVC杀菌**（清洗+杀菌一步完成）
  - **多频技术**（40kHz（大面积）+60kHz（精细）双频可选）
  - **低噪音设计**（改进隔音结构）
  - **小型化**（便携式，眼镜盒大小）
```

## 空化效应原理详解

```
超声波清洗的核心是"空化效应"：

1. 超声波在液体中传播（频率40kHz=每秒40000次压缩/拉伸）
2. 液体在负压区产生微小气泡（空化泡）
3. 空化泡在正压区迅速坍缩（爆裂）
4. 爆裂产生局部冲击波：
   - 温度：瞬间可达数千摄氏度
   - 压力：数百个大气压
   - 速度：微射流以400km/h速度冲击表面
5. 冲击波将污垢从物体表面剥离

关键参数：
- 频率：40kHz（深穿透，适合复杂零件） vs 60kHz（精细，适合珠宝/眼镜）
- 功率密度：>0.3W/cm² 才能产生有效空化
```

## 选购核心维度

1. **频率**：眼镜/首饰选60kHz；复杂零件选40kHz；双频可选更灵活
2. **功率**：>50W清洗效果明显；>100W工业级
3. **内槽容量**：眼镜选300~500mL；餐具选1L以上
4. **加热功能**：有则清洗油污更强（眼镜清洗不需加热）
5. **定时功能**：有则更方便（自动停止）
