# 冰箱除味器

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 工作方式 | 臭氧/活性炭/负离子/光触媒 | 决定除味原理 |
| 功率 | 1~5W | 极低功耗 |
| 供电 | 充电锂电池/USB充电 | 充电一次可用数周 |
| 适用容积 | 50~500L | 按需匹配 |
| 功能 | 除味/杀菌/保鲜/降解农残 | 高端款功能更多 |

---

## 主要类型

| 类型 | 原理 | 优点 | 缺点 |
|------|------|------|------|
| **活性炭包（被动）** | 活性炭吸附异味分子 | 便宜、无需电 | 需定期更换（1~3个月） |
| **臭氧除味器（主动）** | 臭氧氧化分解异味分子 | 杀菌彻底、可降解农残 | 臭氧浓度需控制（对人体有害） |
| **负离子除味器** | 负离子与异味分子结合沉降 | 无耗材 | 效果中等 |
| **光触媒+UV（主动）** | 光触媒在UV催化下分解有机物 | 无耗材、持续有效 | 需要UV照射 |

---

## 技术演进路线

```
[第一代] 活性炭被动除味（约1990s~2010年）
- 简单活性炭包放入冰箱，吸附异味
- 局限：被动吸附，需定期更换，效果有限

[第二代] 电子除味器（约2010~2018年）
- 臭氧/负离子发生器，小型电池供电
- 主动释放臭氧或负离子，分解异味分子

[第三代] 智能臭氧+光触媒（约2018~至今）
- 臭氧+光触媒双技术
- APP定时开关
- 充电锂电池（续航>30天）
- 评价两极分化：臭氧效果显著，但浓度控制存在安全隐患（超标臭氧刺激呼吸道）
```
