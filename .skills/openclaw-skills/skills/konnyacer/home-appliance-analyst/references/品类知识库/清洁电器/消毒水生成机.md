# 消毒水生成机

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 生成方式 | 电解（盐+水） | 次氯酸钠电解原理 |
| 浓度 | 100~1000mg/L | 值越高消毒力越强 |
| 容量 | 300~1000mL | 即时制作即使用 |
| 功率 | 5~20W | 极低功耗 |
| 生成时间 | 3~10分钟 | 一键制作 |

---

## 技术原理

```
次氯酸钠电解消毒水生成原理：

1. 原料：食盐（NaCl）+ 自来水
2. 电解：阳极（+）氧化Cl⁻ → Cl₂ → 与水反应 → HClO（次氯酸）
   阴极（-）还原H₂O → H₂↑
3. 生成物：次氯酸（HClO）+ 氢氧化钠（NaOH）
4. 次氯酸是84消毒液的有效成分，杀菌率>99.9%

关键优势：
- 无需购买化学消毒剂
- 现做现用（浓度稳定）
- 无残留（分解后为盐和水）
- 可用于：果蔬清洗/餐具消毒/物体表面/衣物消毒
```

---

## 技术演进路线

```
[第一代] 大型次氯酸钠发生器（约1980s~2000年）
- 仅工业/商业使用（自来水厂/医院）
- 电解槽大型化，成本高

[第二代] 家用电解消毒水机（约2015~2020年）
- 家用小型化，原料仅需食盐+水
- 一键制作，操作简单
- 代表：星美（XINGMEI）、国产各品牌

[第三代] 智能浓度控制（约2020~至今）
- 浓度可调（不同场景不同浓度）
- LCD显示浓度
- 容量从300mL→1000mL
- 与洗碗机/洗衣机联动（加入消毒水）
```

---

## 选购核心维度

1. **浓度可调**：可调浓度 > 固定浓度
2. **容量**：1000mL（大容量减少制作次数）
3. **安全性**：电解过程无危险气体释放（需有CCC认证）
4. **耗材**：仅需食盐（极便宜），无其他耗材
