# 擦地机器人

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 清洁方式 | 振动擦地/旋转擦地/喷水擦地 | 旋转擦地清洁力最强 |
| 水箱容量 | 80~500mL | 决定单次清洁面积 |
| 导航方式 | 随机碰撞→LDS视觉 | 决定覆盖率 |
| 噪音 | 40~65dB | 工作噪音 |
| 功能 | 干擦/湿擦/扫拖一体 | 功能越多价格越高 |

---

## 与扫地机器人的关系

| 对比 | 扫地机器人 | 擦地机器人 |
|------|-----------|----------|
| 主要功能 | 扫地（吸尘） | 擦地（湿拖/干擦） |
| 吸力 | ✅ 有（核心功能） | ❌ 无/弱 |
| 拖地 | 附带（平板振动/旋转） | 专业（擦布面积更大） |
| 水箱 | 小（80~200mL） | 中大（200~500mL） |
| 清洁专注度 | 清扫地面灰尘 | 去除地面污渍 |
| **互补性** | 互补关系，先扫后擦效果最佳 |

## 技术演进路线

```
[第一代] 干擦机器人（约2002~2010年）
- 核心原理：超声波加湿器蒸发水分→湿气在空气扩散→加湿室内空气
- 核心元器件：超声波振子（1.7MHz）、小型水箱（30~100mL）、USB充电
- 主要局限：加湿效果微弱，仅适合小范围

[第二代] 拖地机器人（约2010~2016年）
- 核心突破：在扫地机器人上加装拖布，实现简单湿拖
- 核心原理：扫地机器人+尾部拖布（湿布）→拖布在前行中擦拭地面
- 主要问题：
  - 拖布水量不可控（漏水）
  - 无擦布自清洁（拖布脏了继续擦越擦越脏）
  - 扫地功能弱化

[第三代] 专用擦地机器人（约2015~2020年）
- 核心突破：专业擦地（无扫地功能，专注于拖地）
- 核心技术路线：
  - **干擦模式**（静电擦布，吸附灰尘毛发）
  - **湿擦模式**（微控渗水+纤维擦布）
  - **路径规划**（弓字形路径，覆盖均匀）
  - **精准控水**（电控水箱，按需渗水，不泡地板）
- 核心元器件：
  - 电控微泵（精准控制出水量）
  - 超细纤维擦布（吸水性强，不留水渍）
  - LDS激光导航（高端机型）
- 代表品牌：
  - iRobot（Braava系列，干湿两擦）
  - 科沃斯（Ecovacs）—— 地宝擦地系列
  - 云鲸（Narwal）J1（旋转擦布+自动洗布）

[第四代] 扫拖一体+自清洁（约2020~至今）
- 核心突破：扫拖一体机器人成为主流，擦地功能大幅升级
- 核心技术升级：
  - **旋转加压擦地**（双盘拖布高速旋转+下压力，清洁力大幅提升）
  - **自动回洗拖布**（拖布脏了自动回基站清洗）
  - **热风烘干**（清洗后热风烘干，防止异味）
  - **自动上下水**（接水管，不用手动加水/倒水）
  - **先扫后擦模式**（扫地吸尘→拖布擦地，分工清洁）
  - **基站自清洁**（基站自动清洗，不用手动清理）
- 旋转擦地原理详解：
  - 两个圆形拖布盘（直径约12cm）
  - 以每分钟180~200转高速旋转
  - 同时施加向下的压力（模拟人手用力擦地）
  - 清洁力 >> 平板振动拖地
- 代表品牌/产品：
  - 云鲸（Narwal）J2/J3—— 旋转擦布+自动洗布+自动上下水
  - 科沃斯（Ecovacs）X1 OMNI—— 旋转擦地+全能基站
  - 追觅（Dreame）W10/W10 Pro—— 旋转擦地+热风烘干
  - 石头（Roborock）G10/G10S—— 声波振动擦地+基站自清洁
```

## 旋转擦地 vs 声波振动：核心差异

| 维度 | 旋转擦地（云鲸/科沃斯） | 声波振动擦地（石头） |
|------|---------------------|------------------|
| 原理 | 双圆盘高速旋转+下压力 | 高速往复振动（每分钟3000次） |
| 清洁力 | 强（物理摩擦力大） | 中（适合日常维护） |
| 适合场景 | 重污渍/厨房/餐后清洁 | 日常轻度清洁 |
| 水渍 | 可能留轻微水痕 | 更均匀，水渍少 |
| 代表机型 | 云鲸J3、科沃斯X1 | 石头G10/G10S |

---

## 选购核心维度

1. **清洁方式**：旋转加压（强） vs 声波振动（日常）
2. **自清洁功能**：有基站自清洁+热风烘干 >> 无（需手动洗拖布）
3. **扫拖分离 vs 扫拖一体**：先扫后擦效果更好（扫地+擦地机器人分开最优）
4. **水箱容量**：>400mL减少加水频率
5. **自动上下水**：有则真正解放双手
