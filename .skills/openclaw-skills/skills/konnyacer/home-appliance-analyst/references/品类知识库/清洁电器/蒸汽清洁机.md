# 蒸汽清洁机（综合）

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 功率 | 1000~2200W | 值越大蒸汽越足 |
| 蒸汽温度 | 100~150℃ | 值越高杀菌清洁力越强 |
| 蒸汽压力 | 0~5bar | 值越大穿透力越强 |
| 水箱 | 0.3~2L | 决定连续使用时间 |
| 预热时间 | 3~10分钟 | 值越短越好 |

---

## 与蒸汽拖把的区别

| 对比 | 蒸汽拖把 | 蒸汽清洁机 |
|------|---------|----------|
| 主体 | 手持式拖把（地面为主） | 多配件清洁枪（全屋） |
| 配件数量 | 少量（拖布+毛刷） | 丰富（多种喷嘴） |
| 主体功能 | 地面拖地+杀菌 | 油烟机/厨房/浴室/玻璃/缝隙 |
| 共同点 | 都用高温蒸汽杀菌 | 均可地面清洁 |

---

## 技术演进路线

```
[第一代] 多功能蒸汽清洁机（约2005~2015年）
- 核心原理：高压蒸汽（1~3bar）通过多种配件喷出→高温溶解油污/水垢→清洁表面
- 核心元器件：
  - 电热管/铸铝发热体（1000~2000W）
  - 高压蒸汽泵（产生压力）
  - 安全阀（超压泄压保护）
  - 多喷嘴配件组（直喷头/圆刷头/刮板/织物刷等）
- 主要局限：预热时间较长（5~10分钟）

[第二代] 即热式蒸汽清洁机（约2015~至今）
- 核心升级：
  - **即热技术**（30秒~3分钟预热）
  - **多档蒸汽量**（可调档位，适应不同表面）
  - **可躺卧设计**（180°可躺，深入家具底部）
  - **附件丰富**（刷头从3件套→15+配件）
  - **配件磁吸**（快速拆装）
- 代表品牌：
  - 卡赫（Kärcher）—— SC系列（德国，蒸汽清洁机领导者）
  - 德尔（DeLonghi）—— 意大利，蒸汽清洁机
  - Shark（鲨客）—— 美国，多功能蒸汽清洁机
  - 国内：德尔/小熊（Bear）—— 性价比

[第三代] 蒸汽+吸水二合一（约2020~至今）
- 核心突破：蒸汽喷射清洁→同时吸走污水→表面即擦即干
- 原理：蒸汽清洁+洗地机吸水功能二合一
- 适合场景：厨房重油污（油烟机/灶台）+ 浴室水垢
```
