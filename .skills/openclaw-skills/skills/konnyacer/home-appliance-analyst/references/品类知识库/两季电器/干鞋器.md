# 干鞋器

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 功率 | 15~250W | PTC型功率低，暖风型功率高 |
| 类型 | PTC发热体/暖风式/UV紫光 | 决定干鞋原理 |
| 适用鞋型 | 拖鞋/运动鞋/靴子 | 注意鞋型适配 |
| 安全 | 自动断电/防过热 | 必须有 |

---

## 主要类型

| 类型 | 原理 | 优点 | 缺点 |
|------|------|------|------|
| **PTC干鞋器（最常见）** | PTC片发热，热传导至鞋内 | 小巧、低功耗、便宜 | 无风力，干燥慢 |
| **暖风式干鞋器** | PTC加热+小风机送风 | 干燥快、带杀菌 | 噪音、价格高 |
| **UV紫光+加热** | UV照射杀菌+热烘干 | 杀菌+除臭 | 杀菌效果有限 |

---

## 技术演进路线

```
[第一代] 简易PTC干鞋器（约2000s~2010年）
- PTC发热片插入鞋内，热传导，无风力
- 小巧便携（鞋拔子形状）

[第二代] 暖风式干鞋器（约2010~至今）
- PTC+小风机，热风通过风管吹入鞋内
- 干燥速度提升3~5倍

[第三代] 智能杀菌干鞋器（约2020~至今）
- 暖风+UV紫光/臭氧杀菌
- 定时功能（1~12小时）
- 适用多种鞋型（含靴子款）
- 代表：苏美（干鞋器专业品牌）
```
