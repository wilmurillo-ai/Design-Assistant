# 电热毯

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 功率 | 50~150W（双人）/ 25~80W（单人） | 碳纤维/石墨烯可低至25W |
| 控温档位 | 1~10档 | 高端支持分区分控 |
| 面料 | 棉/绒/法兰绒/仿兔毛 | 面料决定触感 |
| 安全功能 | 过热保护/自动断电/防水（部分） | 必须有过热保护 |

---

## 主要类型与技术路线

| 类型 | 发热原理 | 优点 | 缺点 | 适合场景 |
|------|---------|------|------|---------|
| **电阻丝电热毯** | 通电后镍铬合金电阻丝发热 | 价格低（50~200元）、技术成熟 | 存在电磁辐射（吐槽）、不能水洗 | 基础使用 |
| **水暖毯（水循环）** | 主机加热温水→毯内水管循环 | 无电磁辐射、温控精准、不干燥 | 需加水/主机有噪音、价格偏高 | 母婴/敏感人群 |
| **碳纤维电热毯** | 碳纤维长丝发热 | 远红外理疗、无电磁辐射、可水洗 | 价格偏高 | 高端/健康需求 |
| **石墨烯电热毯** | 石墨烯涂层发热 | 升温快、远红外释放、可水洗 | 新材料，价格高 | 高端 |

---

## 技术演进路线

```
[第一代] 电阻丝电热毯（约1980s~2010年）
- 核心元器件：镍铬合金电阻丝（NiCr 80/20）、机械恒温器（双金属片）
- 结构：电阻丝蛇形缝入毯体→机械式温度开关控制通断
- 特点：价格低（50~200元），技术成熟
- 局限：
  - 电磁辐射（吐槽：低频50Hz电磁场）
  - 局部高温（电阻丝集中处温度高）→ 低温烫伤风险
  - 不能水洗（电阻丝浸水后安全隐患）

[第二代] 智能恒温电热毯（约2010~2018年）
- 核心升级：单片机控制+温度传感器（热敏电阻NTC）替代机械恒温器
- 核心元器件：NTC测温探头、单片机（MCU）控制板、继电器开关
- 主要升级：
  - 精确控温（±1℃）
  - 多档位（3~10档）
  - 定时功能（1~12小时自动关闭）
  - 过热保护（双重保护：NTC+机械保险丝）
  - 分区控温（双人毯左右独立控温）
- 代表：南极人/彩虹/环高（国产老品牌）

[第三代] 水暖毯（约2015~至今）
- 核心技术：主机（水泵+加热器）→ 毯内软管循环温水
- 核心元器件：小型水泵、不锈钢加热管、分集水器、温度传感器
- 主要优点：
  - 零电磁辐射（隔绝了电阻丝直接发热）
  - 温度均匀（热水在管内流动散热）
  - 不干燥（毯体温度<40℃，空气湿度损失小）
  - 安全性高（毯体本身不导电）
- 主要局限：
  - 需加注纯净水（2~3个月补充）
  - 主机有轻微水泵声（夜间敏感人群可能不适）
  - 价格较高（300~800元）

[第四代] 碳纤维/石墨烯电热毯（约2018~至今）
- 碳纤维路线：
  - 核心：碳纤维长丝束替代电阻丝，发热均匀
  - 优势：远红外8~15μm波段（与人体红外辐射相近，号称理疗效果）
  - 可整体水洗（碳纤维耐水）

- 石墨烯涂层路线：
  - 核心：石墨烯涂层的金属纤维复合丝发热
  - 优势：升温更快（30秒升温）、远红外+石墨烯双重
  - 可轻柔水洗（电压低于12V，低压安全）

[第五代] 智能物联+分区精准控温（约2022~至今）
- 核心技术：WIFI/蓝牙+APP控制 + AI学习用户睡眠习惯
- 分区升级：从左右两区→头/身/脚三区分控
- 安全升级：低压直流供电（12V/24V），毯体完全不带电
- 代表：华为智选电热毯、A.O.史密斯水暖毯（高端）
```

## 主要元器件及成本

| 元器件 | 方案 | 成本占比 |
|--------|------|---------|
| 发热体 | 电阻丝/碳纤维丝/石墨烯线 | ~30% |
| 控制板 | MCU控制板+NTC测温+继电器 | ~25% |
| 面料层 | 棉/绒/法兰绒+绗缝层 | ~25% |
| 安全元件 | 温度保险丝+NTC/热保护器 | ~10% |

## 安全要点（必须掌握的）

1. **禁止事项**：禁止折叠使用、禁止使用超过10年（电阻丝老化）、禁止刺穿毯体
2. **必须功能**：过热保护（双金属片+保险丝双保险）
3. **低温烫伤**：皮肤持续接触>45℃超过2小时会低温烫伤；水暖毯<40℃相对安全
4. **孕妇/婴幼儿**：推荐水暖毯或碳纤维毯（无电磁辐射、低电压）
5. **选购标准**：认准3C认证 + 过热保护 + 定时关闭
