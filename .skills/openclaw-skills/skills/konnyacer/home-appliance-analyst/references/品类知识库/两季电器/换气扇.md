# 换气扇 / 排气扇

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 风量 | 100~500m³/h | 值越大换气越快 |
| 功率 | 20~100W | 功耗极低 |
| 噪音 | 25~55dB | 静音款可选 |
| 安装方式 | 吊顶/窗式/壁挂 | 按场景选择 |
| 类型 | 单向流/双向流/热交换 | 热交换更节能 |

---

## 技术演进路线

```
[第一代] 普通换气扇（约1970s~2000年）
- 核心原理：交流电机驱动叶片旋转→排出室内浊气
- 核心元器件：交流电机+叶片+百叶风口
- 主要局限：单向排气，冬天热量流失大，噪音大

[第二代] 双向流换气扇（约2005~2015年）
- 核心升级：
  - 送风+排风双向（冬天可预热进风）
  - 热交换芯（回收排出空气的热量）
  - 低噪音BLDC电机

[第三代] 智能换气系统（约2015~至今）
- 核心升级：
  - CO₂/湿度传感器联动（CO₂超标自动开启）
  - 变频EC电机（按需调速，更省电）
  - 别墅级热交换系统（全热交换，换气同时回收热能）
```
