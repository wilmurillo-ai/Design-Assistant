# 电子灭蚊器

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 类型 | 紫外灯诱捕/电流电击/吸入式 | 主流为吸入式 |
| 功率 | 3~20W | 极低功耗 |
| 覆盖面积 | 20~100m² | 按房间面积选择 |
| 诱蚊光源 | UV紫光/仿体温/CO₂模拟 | 高端组合多种诱蚊 |
| 供电 | USB/电池/插电 | 户外需电池/USB |

---

## 主要类型

| 类型 | 原理 | 优点 | 缺点 |
|------|------|------|------|
| **电击式** | UV吸引蚊子→蚊虫接触电网→电击死亡 | 效果直接 | 啪啪声、有焦味 |
| **吸入式（主流）** | UV+仿体温吸引→风扇吸入→风干而死 | 无噪音无异味、干净 | 需定期清理 |
| **光触媒式** | UV+光触媒催化CO₂（人体呼吸模拟） | 诱蚊效果最强 | 价格高 |

---

## 技术演进路线

```
[第一代] 电击式灭蚊灯（约1980s~2000年）
- UV灯管（365nm紫光）吸引蚊虫→接触高压电网→电击烧焦
- 局限：啪啪声、烧焦味、蚊虫尸体飞溅

[第二代] 吸入式灭蚊灯（约2005~2015年）
- UV吸引→风扇吸入蚊虫→风干致死→收集盒
- 优点：无噪音、无异味、干净卫生
- 代表：雅格（YAGE）、久量（DP久量）

[第三代] 光触媒+复合诱蚊（约2015~至今）
- 核心升级：
  - **光触媒涂层**（UV光催化释放CO₂+水蒸气=仿人体呼吸）
  - **仿体温发热体**（PTC加热，模拟人体37℃体温）
  - **UV紫光365nm+395nm双波长**
  - **静音直流风机**（BLDC，<25dB，适合卧室）
  - **智能光控**（天黑自动开启，天亮关闭）
```
