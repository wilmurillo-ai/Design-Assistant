# 暖足器 / 暖脚宝 / 暖脚器

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 功率 | 50~250W | 暖风型功率高 |
| 类型 | 包裹式（脚套）/ 脚踏式（平板） | 脚套更暖 |
| 温度 | 35~65℃ | 可调节 |
| 安全 | 过热保护/倾倒断电 | 必须有 |

---

## 技术演进路线

```
[第一代] 简易发热垫（约2000s~2010年）
- 电热丝缝入绒布垫，通电发热
- 局限：热量分布不均，安全性差

[第二代] PTC暖足器（约2010~至今）
- PTC加热片+包裹式脚套，保温效果好
- PTC自动恒温（安全）+ 多档温度

[第三代] 暖风型暖足器（约2018~至今）
- PTC+小风机，暖风包裹脚部
- 加热更快、更均匀
- 带除湿功能（冬天脚汗问题）
```
