# 吊扇 / 吊扇灯

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 扇叶直径 | 36~72英寸（90~180cm） | 36"=1~2人/ 48"=3~4人/ 60"=5~6人 |
| 功率 | 15~70W | 直流变频可低至5W |
| 转速 | 80~300转/分钟 | 多档可调 |
| 安装高度 | 离地2.3~2.8m | 安全距离 |
| 噪音 | 25~55dB | 直流变频静音 |

---

## 技术演进路线

```
[第一代] 普通交流吊扇（约1940s~2000年）
- 核心元器件：交流电机（iron core，有刷），5叶/3叶扇叶，吊杆
- 主要局限：噪音大、档位少（3档）、能耗高（50~70W）

[第二代] 直流变频吊扇（约2010~至今）
- 核心突破：BLDC无刷直流电机取代交流电机
- 核心升级：
  - 功率降至5~25W（省电60~80%）
  - 档位细化（6~8档）
  - 正转（制冷）= 夏季向下吹风，促进蒸发降温
  - 反转（冬季）= 夏季的反向，促进室内暖气均匀分布
  - 遥控/APP控制
  - 静音（低档<30dB）

[第三代] 吊扇灯（约2015~至今）
- 核心突破：风扇+LED灯具二合一（取代客厅主灯）
- 核心元器件：
  - BLDC电机（风扇）
  - LED光源（主照明，20~40W）
  - 遥控/墙壁开关
- 优势：节省安装空间；一台设备满足照明+风扇需求
- 代表品牌：永怡御风（国内吊扇灯领导者）、雷士（照明品牌吊扇灯）
```
