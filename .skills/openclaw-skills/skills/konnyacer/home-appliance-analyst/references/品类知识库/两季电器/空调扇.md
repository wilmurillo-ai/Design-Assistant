# 空调扇

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 功率 | 50~150W | 比空调省电90%+ |
| 制冷效果 | 低于室温3~8℃ | 非真正制冷，蒸发降温 |
| 水箱容量 | 4~30L（大型塔扇型） | 容量决定续航 |
| 冰晶盒 | 1~4块（附带） | 冰晶盒越多制冷效果越好 |
| 风量 | 300~800m³/h | 值越大送风越远 |
| 噪音 | 35~65dB | 低于空调室内机 |

---

## 主要类型

| 类型 | 特点 | 适用场景 |
|------|------|---------|
| **小型冰晶箱式** | 4~8L水箱，冰晶辅助降温，台式/立式 | 小空间/个人使用 |
| **塔式冷风扇** | 冰晶+大水箱+大出风口，送风范围广 | 客厅/中等面积 |
| **工业/商业冷风扇** | 大型，移动式，大水箱（20~30L） | 车间/商铺/户外 |
| **空调扇+暖风两用** | 夏季制冷+冬季暖风，一机两用 | 全年使用 |

## 技术演进路线

```
[第一代] 基础冷风扇（约1990s~2010年）
- 核心原理：风扇抽风→风经过湿润的湿帘/冰晶→水分蒸发吸热→冷风送出
  （本质是"蒸发制冷"，不是真正的制冷，与空调的压缩机制冷完全不同）
- 核心元器件：
  - 交流电机（风扇）
  - 湿帘（纸蜂窝结构，吸水增湿）
  - 冰晶盒（冰晶蓄冷，增强降温效果）
  - 水泵（将水抽至湿帘）
  - 水箱（4~8L）
- 主要特点：
  - 降温效果有限（室温-3~8℃）
  - 比风扇凉爽，但远不如空调
  - 功耗极低（50~100W，省电）
  - 需定期加冰晶/冰块
- 主要局限：
  - **不能降低室内温度**（只能局部降温，风吹到的地方才凉爽）
  - 湿度大幅提升（室内湿度增加，对南方潮湿天气不友好）
  - 冰晶效果持续时间短（1~3小时）
  - 水箱若不及时清理，易滋生细菌

[第二代] 大型塔式+冰晶优化（约2010~2018年）
- 核心升级：
  - **塔式结构**：高塔型，出风口面积更大，送风范围更广
  - **大冰晶盒**：配套2~4块冰晶，蓄冷量增加
  - **大水箱**（10~30L，减少加水频率）
  - **负离子发生器**（部分机型，净化空气+产生负离子）
  - **上下左右摇头**（送风范围更大）
  - **可控定时**（1~12小时定时）
  - **加湿功能**（部分机型，冰晶化水后继续加湿）
- 核心元器件：
  - 大型塔式蒸发湿帘（纸质蜂窝，吸水性强）
  - 冰晶盒（高分子凝胶，蓄冷性能好）
  - 水位显示（可视化水箱）
- 代表品牌：
  - 艾美特（Airmate）—— 空调扇领域领先品牌
  - 美的（Midea）—— 高性价比
  - 格力（ Gree）—— 大型塔扇
  - 先锋（SINGFEN）—— 价格亲民

[第三代] 变频直流+两用+智能化（约2018~至今）
- 核心升级：
  - **直流变频电机**（BLDC）：噪音降低（35~50dB），功耗更低，支持多档位
  - **冷暖两用**：夏季制冷+冬季暖风（PTC加热元件），一机四季
  - **智能湿度控制**（湿度传感器，防止室内过度加湿）
  - **WIFI/APP控制**
  - **大屏数显**（温度/湿度/档位实时显示）
  - **自动摆风**（上下+左右，360°送风）
  - **大滤网**（过滤空气灰尘，部分机型叠加HEPA）
- 核心元器件：
  - BLDC无刷直流电机
  - PTC加热模块（暖风模式，1000~2000W）
  - 负离子/UV杀菌（杀菌净化）
- 代表品牌/产品：
  - 艾美特（Airmate）—— 冷暖两用塔扇旗舰
  - 美的（Midea）—— 高性价比直流变频款
  - 米家（小米）—— 米家直流变频空调扇（APP联动）
  - 戴森（Dyson）—— Pure Cool Me（无叶，气流净化+凉爽）

[空调扇的技术局限（必须告知用户）]
- ❌ 不能降低房间温度（只能局部降温）
- ❌ 会增加室内湿度（不适合潮湿地区）
- ❌ 冰晶效果衰减快（每2~3小时需重新冷冻）
- ✅ 正确用途：空调的补充（空调冷气过强时缓解"空调病"）/ 无空调的过渡季节
```

## 关键元器件及供应商

| 元器件 | 主流方案 | 备注 |
|--------|---------|------|
| 冰晶盒 | 高分子凝胶（羧甲基纤维素） | 反复冷冻使用，蓄冷量决定降温效果 |
| 湿帘 | 纸蜂窝结构 | 吸水性和蒸发面积决定制冷效率 |
| 电机 | 普通交流/ BLDC直流 | BLDC更静音 |
| 水泵 | 小型隔膜泵 | 将水抽至湿帘 |
| PTC加热（暖风版） | PTC陶瓷加热片 | 提供冬季暖风 |

## 选购核心维度

1. **类型**：小型（单人或桌面）→ 塔式（客厅/中等面积）→ 工业型（商铺/车间）
2. **冰晶效果**：冰晶盒数量和体积（越多越大，制冷效果越好）
3. **功率**：>100W可接受，太低风量不足
4. **水箱容量**：>10L减少加水频率
5. **暖风需求**：冬天需要暖风→选冷暖两用款
6. **噪音**：卧室选<45dB（BLDC变频款）
7. **正确定位**：空调的补充品，而非空调的替代品
