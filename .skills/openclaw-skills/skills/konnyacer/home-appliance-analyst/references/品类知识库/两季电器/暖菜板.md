# 暖菜板 / 暖菜垫 / 暖手宝

## 一、暖菜板（暖菜盘）

### 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 加热温度 | 40~120℃（可调） | 暖菜40~80℃，加热需100℃以上 |
| 功率 | 100~500W | 整板型功率大 |
| 形态 | 整板型/单盘型/卷折型 | 按需选择 |
| 定时 | 1~8小时 | 自动关闭更安全 |
| 表面材质 | 钢化玻璃/金属板 | 决定耐用性和清洁便利性 |

---

### 主要类型

| 类型 | 特点 | 适用场景 |
|------|------|---------|
| **整板式暖菜板** | 大面积均匀发热，整桌暖菜 | 家庭聚餐/多人使用 |
| **单盘式暖菜板** | 每个位置独立加热，可拆分 | 小家庭/个人 |
| **卷折式暖菜垫** | 柔性发热垫，可卷起收纳 | 租房/小户型 |
| **火锅保温盘** | 大面积恒温，用于火锅/烧烤保温 | 火锅场景 |

---

### 技术演进路线

```
[第一代] 基础整板式暖菜板（约2010~2016年）
- 核心原理：电热膜/电热管加热→导热板表面升温→持续保温（40~80℃）
- 核心元器件：
  - 电热膜（金属氧化物浆料印刷于玻璃/金属板上）
  - 温控器（双金属片式恒温控制）
  - 玻璃面板（钢化玻璃，表面美观易清洁）
- 主要特点：
  - 解决冬天饭菜凉得快的问题（北方冬季暖气房以外）
  - 表面温度恒定在55~70℃（保温而非加热）
  - 价格：100~300元
- 主要局限：
  - 温度不均（电热膜分布不均匀导致热点）
  - 无精确控温（机械恒温器，滞后明显）
  - 整板温度固定，无法分区域控制

[第二代] 分区控温+精准温控（约2016~2020年）
- 核心升级：
  - **分区控温**（整板分2~4区，各区独立温控）
  - **MCU电子温控**（NTC传感器实时测温，±5℃精确控温）
  - **多档温度可选**（40℃/60℃/80℃/100℃/120℃）
  - **触控面板**（LED/数码显示）
  - **预约功能**（提前开启，到点即暖）
- 核心元器件：
  - NTC温度传感器
  - MCU控制板
  - 分区电热膜/铝合金加热板
  - 钢化玻璃面板（耐高温易清洁）
- 代表品牌：
  - 美的（Midea）—— 入局暖菜板，699元搅局
  - 苏泊尔（SUPOR）—— 分区控温旗舰
  - 小熊（Bear）—— 性价比，小型单盘
  - 炊大皇（COOKAM）—— 火锅保温盘

[第三代] 智能暖菜板（约2020~至今）
- 核心升级：
  - **远红外加热**（远红外线直接加热食物内部，减少面板温度梯度）
  - **变频恒温**（根据菜量自动调节功率，节能）
  - **保温+快速加热双模式**（快速加热模式>100℃，适合冻品解冻）
  - **卷折收纳**（柔性发热垫，卷起收纳节省空间）
  - **WIFI/APP远程控制**（提前开启）
  - **抗菌表面**（纳米银涂层，抑制细菌滋生）
- 代表品牌/产品：
  - 尚朋堂（高端整板式，远红外加热）
  - 小熊（Bear）—— 暖菜垫/卷折型
  - 苏泊尔（黑金刚系列，变频恒温）
```

## 二、暖手宝（暖手器）

### 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 加热方式 | 电热丝热水袋/锂电池暖手宝/陶瓷暖手宝 | 主流为充电式 |
| 温度 | 45~65℃ | 恒温控制 |
| 续航 | 3~12小时 | 电容量决定 |
| 电容量 | 5000~20000mAh | 20000mAh可当充电宝 |
| 安全 | 电热丝防爆（电极式已淘汰） | 必须有防爆保护 |

---

### 主要类型与技术路线

| 类型 | 原理 | 优点 | 缺点 | 安全性 |
|------|------|------|------|--------|
| **电极式热水袋（已淘汰）** | 电极直接加热液体 | 便宜 | 爆炸风险极高 | ❌ 危险，已禁售 |
| **电热丝热水袋** | 电热丝隔离加热液体 | 价格适中 | 有液体，携带不便 | ✅ 有防爆保护 |
| **锂电池暖手宝** | 锂电池+电热片（无液体） | 轻便、安全 | 续航有限 | ✅ 安全 |
| **陶瓷暖手宝** | PTC陶瓷片加热 | 恒温、安全 | 需插电/充电 | ✅ 安全 |

---

### 技术演进路线

```
[第一代] 电极式热水袋（约2000年前后，危险）
- 核心原理：两个金属电极直接插入水中→通电后水被加热
- 危险原因：电极直接接触水，无隔离，一旦温控失灵→爆炸
- 结果：国家于2010年前后明令禁止生产电极式热水袋

[第二代] 电热丝热水袋（约2005~2015年）
- 核心改进：电热丝包裹在绝缘层内→不与水直接接触
- 核心元器件：
  - 电热丝（镍铬合金）
  - 温控器（双金属片自动断电）
  - 压力保护（泄压阀）
  - 充电底座（直接连接220V）
- 主要升级：
  - 防爆保护（温控+压力保护双保险）
  - 自动断电（加热完成后自动断电）
- 主要局限：
  - 有液体，携带不便，有漏液风险
  - 充电时不能使用

[第三代] 锂电池暖手宝（约2015~至今）
- 核心突破：充电锂电池+电热片，无液体，安全性大幅提升
- 核心元器件：
  - 锂电池（18650/聚合物，5000~10000mAh）
  - PTC电热片（自动恒温）
  - 温控保护（NTC+保护板）
  - USB充电（5V/2A）
- 主要升级：
  - **无液体**（安全，可带上飞机）
  - **可充电循环使用**（USB充电，经济环保）
  - **可当充电宝**（20000mAh款可给手机充电）
  - **多档温度**（45℃/55℃/65℃）
  - **小巧轻便**（口袋书大小）
- 代表品牌：素士（Soocas）、倍思（Baseus）、飞科（Flyco）

[第四代] 智能暖手宝（约2020~至今）
- 核心升级：
  - **远红外发热**（远红外辐射，加热更均匀，舒适度高）
  - **数显温度**（LED显示实时温度）
  - **APP控制**（部分高端款）
  - **超大电容量**（20000mAh，续航12小时+）
  - **时尚设计**（猫耳暖手宝/围巾暖手等造型）
```

## 选购核心维度

### 暖菜板
1. **类型**：家庭聚餐→整板式；小家庭→单盘式；租房→卷折垫式
2. **温控精度**：电子温控（分档）> 机械温控
3. **分区功能**：多分区更实用（不同菜品不同温度）
4. **表面材质**：钢化玻璃（易清洁）> 金属板（耐用）
5. **安全功能**：自动断电 + 防干烧

### 暖手宝
1. **类型**：有液体（保温好但不便携） vs 无液体（轻便安全）
2. **安全性**：电极式（禁）→ 电热丝式（可）→ 锂电池式（推荐）
3. **续航**：>8小时为优秀
4. **附加功能**：充电宝功能（实用）、多档温控
5. **安全认证**：必须3C认证，有防爆保护
