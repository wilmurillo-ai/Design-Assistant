# 暖风机（专项）

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 功率 | 1000~3000W | 值越大制热越快 |
| 加热方式 | PTC陶瓷/电热丝 | PTC更安全、恒温 |
| 防水等级 | IPX0~IPX4 | 浴室用需≥IPX2 |
| 噪音 | 35~60dB | 直流款可低至35dB |
| 类型 | 台式/壁挂/塔式/迷你 | 按场景选择 |

---

## 与其他取暖器的区别

| 对比 | 暖风机 | 踢脚线 | 小太阳 | 油汀 |
|------|--------|--------|--------|------|
| 原理 | PTC加热+强制吹风 | 自然对流（底部进冷空气） | 红外辐射（热射线） | 热油散热 |
| 优点 | 即热、浴室可用、定向吹 | 温度均匀、无声 | 即热、价格低 | 无光、保温好 |
| 缺点 | 有风/噪音、干燥 | 预热慢、干燥 | 局部取暖、干燥 | 预热慢、重 |
| 适合场景 | 浴室/临时/定向取暖 | 客厅/全屋整夜 | 小范围 | 卧室/书房 |

## 技术演进路线

```
[第一代] 电热丝暖风机（约1980s~2000s）
- 核心原理：电热丝通电发热→小型风扇将暖风吹出
- 核心元器件：镍铬电热丝、有刷电机风扇、机械开关
- 主要局限：
  - 电热丝温度高（>200℃），有明火风险
  - 无恒温，过热风险
  - 噪音大

[第二代] PTC陶瓷暖风机（约2005~至今）
- 核心突破：PTC（正温度系数热敏电阻）取代电热丝
- 核心优势：
  - PTC温度上升→电阻增大→自动限流→永不超温（自限温特性）
  - 热量不集中，安全可靠
  - 寿命极长（>10000小时）
- 核心元器件：PTC陶瓷发热片（自带恒温特性）、BLDC无刷直流风扇、温控NTC

[第三代] 浴室防水暖风机（约2010~至今）
- 核心升级：
  - IPX1~IPX4防水认证
  - PTC+摇头功能（广角送暖）
  - 多档位（冷风/低热/高热）
  - 小巧台壁两用（可放地面/挂墙）
  - 遥控器+定时

[第四代] 智能变频暖风机（约2020~至今）
- 核心升级：
  - DC直流变频（BLDC电机，噪音<35dB）
  - AI恒温（精确控温±1℃，节能）
  - 居浴两用（IPX4防水）
  - APP/WIFI控制
  - 等离子杀菌（部分机型，暖风同时净化空气）
- 代表品牌：美的（Midea）暖风机系列、格力（ Gree）浴用暖风机
```

## 选购核心维度

1. **浴室使用**：必须有IPX2+防水认证
2. **功率**：浴室选1500W+，小房间1000W即可
3. **噪音**：卧室选<40dB（BLDC直流）
4. **安全**：PTC发热体 > 电热丝
5. **功能**：摇头（送暖范围大） > 固定方向
