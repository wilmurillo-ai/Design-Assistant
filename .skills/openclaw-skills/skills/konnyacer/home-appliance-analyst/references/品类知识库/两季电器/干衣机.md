# 干衣机

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 容量 | 3~10kg | 值越大烘得越多 |
| 干衣方式 | 冷凝式/热泵式/排气式 | 热泵最节能 |
| 功率 | 500~2500W | 热泵功率低但效率高 |
| 噪音 | 45~65dB | 滚筒转动声 |
| 能效 | 定频/变频（热泵） | 热泵省电50~60% |
| 附加功能 | 除菌/除螨/除皱/防皱 | 高端功能 |

---

## 主要类型

| 类型 | 原理 | 优点 | 缺点 | 能效 |
|------|------|------|------|------|
| **排气式** | 加热空气→穿过衣物→带走水分→排出湿气 | 价格低 | 需排气管、安装麻烦 | ⭐（最耗电） |
| **冷凝式** | 加热空气→冷凝器凝结水蒸气→收集水箱 | 无需排气管 | 能耗较高 | ⭐⭐ |
| **热泵式** | 制冷剂循环，类似空调原理回收热量 | 最节能（省电50%） | 价格高（2000+） | ⭐⭐⭐⭐⭐ |

## 技术演进路线

```
[第一代] 排气式干衣机（约1970s~2000年）
- 核心原理：加热器（电热管）加热空气→热风穿过滚筒→带走衣物水分→湿气通过排气管排出室外
- 核心元器件：电热管（1500~2500W）、滚筒电机、排气管接口、温度保险丝
- 主要特点：
  - 必须连接排气管到室外（或窗户）
  - 能耗高（全部热量随湿气排走）
  - 烘干温度高（80~90℃），对衣物损伤大
- 主要局限：
  - 安装麻烦
  - 耗电（热效率约30%）
  - 高温损伤精细衣物（羊毛/丝绸）

[第二代] 冷凝式干衣机（约2000~2015年）
- 核心突破：去掉排气管，室内使用成为可能
- 核心原理：加热器加热空气→穿过滚筒→湿气进入冷凝器→冷却凝结成水→水收集到水箱
- 核心元器件：
  - 电热管（1500~2000W）
  - 冷凝器（铜管+铝翅片）
  - 冷凝水箱（1.5~3L）
  - 滚筒BLDC电机
- 主要升级：
  - **无需排气管**（室内使用，随处摆放）
  - 烘干温度降至60~80℃
  - 水箱满了自动停机提示
- 主要局限：
  - 能耗仍偏高（约50%热效率）
  - 冷凝器需定期清洁

[第三代] 热泵式干衣机（约2010~至今，成熟2015年后）
- 核心突破：热泵技术（类似空调逆循环）回收废热，能耗大幅降低
- 核心原理：制冷剂在蒸发器吸收筒内湿气热量→压缩机压缩→冷凝器释放热量加热空气→热空气再进入滚筒（循环）
  - 整个过程热量被回收再利用，而非排出室外
- 核心元器件：
  - 压缩机（变频/定频）
  - 蒸发器（铜管+铝翅片）
  - 冷凝器（热交换核心）
  - 节流装置（电子膨胀阀，变频机型）
  - 热泵专用滚筒（低转速+正反转交替，防止衣物缠绕）
- 主要升级：
  - **能耗降低50~60%**（对比冷凝式）
  - **烘干温度低（50~60℃）**（对衣物更温和，保护精细面料）
  - **正反转防缠绕**（滚筒正转1分钟→反转30秒，减少褶皱）
  - **绒毛收集盒**（过滤绒毛，保护衣物）
  - **除静电**（减少衣物静电）
- 代表品牌：
  - 小天鹅（Little Swan）—— 国内热泵干衣机领导者（被美的收购）
  - 海尔/卡萨帝—— 热泵干衣旗舰
  - 博世/西门子（欧洲品牌）—— 热泵技术领先
  - LG/三星（韩国）—— 双变频（压缩机+电机双变频）

[第四代] 热泵变频+智能（约2020~至今）
- 核心升级：
  - **双变频**（变频压缩机+BLDC滚筒电机，精准控温，更护衣）
  - **紫外线/巴氏杀菌**（烘干同时杀菌除螨，除菌率>99.9%）
  - **AI智能感控**（传感器检测衣物干燥程度，自动调整时间）
  - **羊毛蓝标认证**（精确低温烘干，适合羊毛/丝绸）
  - **热泵静音技术**（噪音从65dB→45dB）
- 代表产品：
  - 小天鹅H36WT（热泵旗舰，10kg）
  - 海尔GBN100-189U1（热泵变频+巴氏杀菌）
  - 卡萨帝（Casarte）—— 高端，热泵+AI
```

## 选购核心维度

1. **类型**：热泵（推荐，节能护衣） > 冷凝 > 排气式
2. **容量**：3kg（单人）/ 8kg（家庭）/ 10kg（大件被子）
3. **能效**：热泵变频 > 热泵定频 > 冷凝式
4. **护衣功能**：正反转防缠绕、羊毛认证、低温柔烘
5. **杀菌功能**：UV/巴氏杀菌（母婴/有宠物家庭推荐）
6. **噪音**：<50dB为安静
7. **安装方式**：热泵/冷凝（独立式，无需安装） > 排气（需排气管）
