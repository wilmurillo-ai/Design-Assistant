# 取暖桌 / 电暖茶几

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 功率 | 600~2400W（分区控制） | 整桌分区加热 |
| 桌面材质 | 钢化玻璃/大理石/岩板/木面 | 大理石/岩板导热好 |
| 发热方式 | 电热膜/碳纤维/ PTC | 碳纤维发热最均匀 |
| 升降功能 | 固定高度/电动升降 | 电动升降可作茶几/餐桌 |
| 安全 | 过热保护/倾倒断电/童锁 | 必须有 |

---

## 技术演进路线

```
[第一代] 简易电暖桌（约2005~2015年）
- 核心原理：电热管/电热膜铺在桌面下方→加热桌面→人体手部接触取暖
- 核心局限：温度不均，只有手部接触位置暖和

[第二代] 碳纤维远红外电暖桌（约2015~2020年）
- 核心突破：碳纤维发热膜替代电热管
- 核心优势：
  - 远红外辐射加热（类似晒太阳，人体感受更舒适）
  - 热量分布均匀
  - 寿命长（碳纤维抗氧化，不衰减）
- 配套功能：电磁炉内嵌（冬季一边烤火一边火锅）

[第三代] 智能多功能电暖桌（约2020~至今）
- 核心升级：
  - **电动升降**（茶几模式/餐桌模式/书桌模式一键切换）
  - **分区独立控温**（桌面分区加热，节能）
  - **暖脚功能**（桌子下方空间加热，形成局部暖区）
  - **岩板/大理石桌面**（导热快，质感好）
  - **APP远程控制**
  - **电磁炉+茶杯架**（内嵌设计）
  - **脚轮可移动**（可移动位置）
```
