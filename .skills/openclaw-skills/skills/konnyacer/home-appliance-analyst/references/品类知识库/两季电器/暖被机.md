# 暖被机 / 烘被机

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 功率 | 300~800W | 值越大暖被越快 |
| 暖被时间 | 15~60分钟 | 典型30分钟 |
| 功能 | 暖被/烘鞋/除螨/杀菌 | 功能越多价格越高 |
| 风管 | 软管/伸缩风管 | 连接被子 |
| 安全 | 自动断电/防过热 | 必须有 |

---

## 技术演进路线

```
[第一代] 热风暖被机（约2000s~2010年）
- 核心原理：PTC加热器产生热风→软管插入被窝→热风在被中循环→带走湿气+加热
- 核心元器件：PTC加热片+小型风机+软管
- 主要局限：功能单一

[第二代] 多功能暖被机（约2010~至今）
- 核心升级：
  - **多风道设计**（暖被+烘鞋+衣柜+除螨多配件）
  - **除螨功能**（60℃以上热风持续30分钟，除螨率>99%）
  - **恒温控制**（NTC控温，不会过热）
  - **可折叠设计**（节省收纳空间）
- 代表品牌：爱丽思（Iris Ohyama，日本）、日本amadana、国产（彩虹/小熊）
```
