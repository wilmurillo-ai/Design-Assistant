# 毛球修剪器 / 除毛器

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 刀网类型 | 不锈钢刀片/微孔刀网 | 决定剪切效果 |
| 功率 | 2~5W | 极低功耗 |
| 充电 | USB充电/干电池 | USB更方便 |
| 集尘盒 | 透明可拆洗 | 收集剪下的毛球 |

---

## 技术原理

```
毛球修剪器工作原理：

1. 刀网（金属网格）紧贴衣物表面
2. 刀片在刀网下方旋转
3. 衣物上的毛球凸起部分穿过刀网孔洞
4. 刀片剪切掉凸起的毛球
5. 吸风装置将剪下的毛球吸入集尘盒

关键：刀网孔洞大小决定剪切精细度；刀片锋利度决定剪切干净程度
```

---

## 技术演进路线

```
[第一代] 干电池式毛球修剪器（约2000s~2015年）
- 核心元器件：小型直流电机+刀片+刀网+干电池
- 主要特点：便宜（20~50元），但刀片钝得快，效果一般

[第二代] USB充电+大面积刀网（约2015~至今）
- 核心升级：
  - USB充电锂电池（反复使用）
  - 大面积刀网（修剪效率提升）
  - 透明集尘盒（可水洗）
  - 多级刀片（剪切更干净）
- 代表品牌：飞科（Flyco）、超人（Chaoren）、日本 Miyako
```
