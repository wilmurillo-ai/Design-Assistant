# 电热水袋

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 加热方式 | 电极式（禁）/电热丝式/USB充电式/锂电池式 | 安全性差异大 |
| 容量 | 500~2000mL | 值越大保温越久 |
| 温度 | 40~70℃ | 可调温更安全 |
| 充电方式 | 220V直充/USB充电 | USB更安全 |
| 安全认证 | 3C认证+防爆夹 | 必须有 |

---

## 主要类型与安全性

| 类型 | 原理 | 安全性 | 备注 |
|------|------|--------|------|
| **电极式（已淘汰）** | 电极直接接触水加热 | ❌ 危险，已禁售 | 爆炸风险高 |
| **电热丝式** | 电热丝包裹绝缘层后置于水中 | ✅ 有防爆保护 | 传统主流 |
| **USB充电暖手宝** | PTC/PTC陶瓷加热，USB充电 | ✅ 安全 | 无液体，便携 |
| **锂电池式** | 聚合物锂电池+电热片 | ✅✅ 最安全 | 可循环充电 |

---

## 技术演进路线

```
[第一代] 电极式电热水袋（约2000年前后，危险）
- 核心原理：两个金属电极直接插入水袋水中→通电加热
- 危险：一旦温控失灵→水袋膨胀→爆炸
- 结果：2010年前后国家明令禁止电极式

[第二代] 电热丝式电热水袋（约2005~至今）
- 核心改进：电热丝外包绝缘套→不直接接触水
- 核心元器件：电热丝+温控器+压力保护+防爆充电夹
- 防爆夹原理：充电时温度过高→压力过大→防爆夹自动断开电源
- 仍存风险：温控器失灵仍有隐患；建议认准3C认证

[第三代] USB充电暖手宝（约2015~至今）
- 核心突破：充电宝式设计，无液体，USB充电
- 核心元器件：聚合物锂电池（5000~10000mAh）+ PTC电热片
- 优势：无液体、无漏液风险、可边充电边用、可当充电宝
- 代表：素士/倍思/飞科

[第四代] 智能恒温+按摩暖手宝（约2020~至今）
- 核心升级：
  - 多档温度可调（40/50/60℃）
  - 按摩功能（震动按摩）
  - 智能数显（LED显示实时温度）
  - 超大电容量（20000mAh，续航12小时+）
```
