# 除湿机

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 日除湿量 | 10~50L/D（30℃/80%RH） | 主流12~30L/D |
| 名义除湿量 | 0.3~1.5kg/h（27℃/60%RH） | 国家标准工况，更准确 |
| 功率 | 200~800W | 除湿量越大功率越高 |
| 水箱容量 | 2~8L | 决定倒水频率 |
| 适用面积 | 20~120m² | 与除湿量正相关 |
| 噪音 | 35~55dB | 压缩机款噪音较大 |
| 除湿技术 | 压缩机式/转轮式/半导体式 | 决定适用场景 |

---

## 主要类型与技术路线

| 类型 | 原理 | 优点 | 缺点 | 适用场景 |
|------|------|------|------|---------|
| **压缩机式** | 制冷剂循环，与空调除湿相同 | 除湿量大、能效高 | 噪音大（压缩机）、低温效果差 | 家庭/仓库（>15℃） |
| **转轮式（干燥剂）** | 吸湿转轮转动，干燥区加热再生 | 低温适用（<10℃可用）、静音 | 除湿量小、需加热能耗 | 地下室/低温环境 |
| **半导体式（冷凝）** | 珀尔帖效应，半导体制冷+冷凝水 | 小型静音、轻便 | 除湿量极小 | 小空间/衣柜/车载 |
| **热石式** | 加热吸附→冷却脱附循环 | 静音、低温可用 | 能耗高、除湿量中等 | 卧室/小型空间 |

---

## 技术演进路线

```
[第一代] 压缩机式基础款（约1980s~2010年）
- 核心原理：与空调制冷循环相同（压缩机→冷凝器→节流装置→蒸发器→吸气）
  - 湿空气通过蒸发器翅片，空气中水蒸气冷凝成水珠→落入水箱
  - 干燥空气经过冷凝器加热后吹出（温度略升高）
- 核心元器件：旋转式压缩机（主流）、冷凝器（铜管+铝翅片）、蒸发器、毛细管/电子膨胀阀
- 主要特点：
  - 除湿量大（12~30L/D）
  - 能效比（EER）约1.5~2.5 kg/kWh
  - 价格：300~800元
- 主要局限：
  - 压缩机噪音大（40~55dB）
  - 低温环境（<15℃）除湿效率急剧下降（蒸发器结霜）
  - 水箱较小（2~4L），需频繁倒水

[第二代] 大容量+低噪音款（约2010~2018年）
- 核心升级：
  - 大水箱（4~8L，减少倒水频率）
  - 直流变频压缩机（降噪+节能）
  - 万向轮（方便移动）
  - 湿度设定（RH 30%~80%可调，到达设定湿度自动停机）
  - 干衣模式（强风+低热，加速衣服干燥）
  - 除霜功能（低温自动除霜）
- 核心元器件：变频压缩机（直流变频）替代定频
- 代表品牌：德业（Deye）、欧井（Eurgeen）、多乐信（Duolexin）、松下（高端静音）
- 代表产品：德业DYD-E12A3（12L/D）、欧井OJ-128E

[第三代] 智能化+多功能（约2018~2021年）
- 核心升级：
  - 湿度实时数显（LED/LCD显示）
  - APP远程控制（开关机+湿度设定）
  - 24小时定时
  - 自动排水（外接水管，不用倒水）
  - 负离子/UV杀菌（空气净化功能叠加）
  - 干衣功能升级（冷风/热风可选）
- 核心元器件：
  - 湿度传感器（高精度电容式）
  - WIFI模块（IoT连接）
  - UV灯（杀菌功能）
- 代表品牌：小米（米家除湿机）、德业（高端变频）、追觅（尚未入局）

[第四代] 大除湿量+中央/全屋型（约2020~至今）
- 核心突破：除湿量大幅提升（30~50L/D）+ 接入中央除湿/全屋系统
- 技术升级：
  - 压缩机技术成熟：大除湿量机型（30L/D以上）稳定可靠
  - 静音技术：松下纳米水离子压缩机（28~35dB）
  - 工业级/商用级：德业DYD-T22A3（22L/D，适合60㎡）
  - 全屋除湿：接入新风机/中央新风系统
- 主要升级点：
  - 除湿量从12L→50L/D（大功率机型）
  - 噪音从55dB→28dB（松下旗舰）
  - 全屋覆盖：适合地下室/复式/别墅（需大功率机型）
  - 智能联动：与米家/Aqara等智能家居联动（湿度>60%自动开启）

[第五代] 低温型+净化除湿一体（约2022~至今）
- 低温型（转轮式）进化：
  - 原理：吸湿转轮（分子筛/硅胶）吸收湿气→低温区吸湿→高温区脱附再生
  - 优点：在5℃以下仍可除湿（压缩机式在<10℃基本失效）
  - 代表：德业（低温型）、格力（低温除湿机）
- 净化除湿一体：
  - 增加HEPA滤网+活性炭（除湿+PM2.5+甲醛）
  - 代表：松下F-46C7YXW（除湿+空气净化）
```

## 压缩机式 vs 转轮式：核心对比

| 维度 | 压缩机式 | 转轮式（干燥剂） |
|------|---------|----------------|
| 工作温度 | 15~40℃（最佳） | 1~40℃（均可） |
| 低温性能 | 差（<15℃结霜衰减） | 优秀（地下室/冬天可用） |
| 除湿量 | 大（12~50L/D） | 小（5~20L/D） |
| 能耗 | 低（能效比1.5~2.5） | 高（含加热再生耗电） |
| 噪音 | 40~55dB（压缩机声） | 30~45dB（静音优势） |
| 价格 | 300~2000元 | 1000~5000元 |
| 适合场景 | 家庭客厅/卧室/南方回南天 | 地下室/酒窖/北方冬天/制药厂 |
| 代表品牌 | 德业、欧井、美的、松下 | 德业（低温型）、格力（工业） |

## 关键元器件及供应商

| 元器件 | 主流方案 | 代表供应商 | 备注 |
|--------|---------|----------|------|
| 压缩机 | 旋转式压缩机（定频/变频） | 美芝（GMCC）、春飞（上海）、松下 | 压缩机是核心，占BOM约30% |

| 膨胀装置 | 毛细管（定频）/电子膨胀阀（变频） | 盾安 | 变频款控温更精准 |

## 选购核心维度

1. **除湿量**：看名义除湿量（kg/h，30℃/60%RH，国家标准工况）
2. **适用面积**：除湿量（L/D）× 2≈适用面积（m²）
3. **水箱容量**：>5L减少倒水频率，或选外接排水款
4. **噪音**：卧室选<40dB，客厅可接受45dB
5. **低温适用**：地下室/冬天/<10℃环境选转轮式
6. **功能叠加**：干衣/空气净化/UV杀菌（按需选择）
7. **耗材成本**：滤网/除湿剂定期更换成本
