# 空气净化器

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| CADR值 | 300~700m³/h | 越大越好，适用面积≈CADR×0.1 |
| CCM值 | P4/F4（最高级） | 滤网累计净化量，值越高寿命越长 |
| 噪音 | 25~65dB | 睡眠档<35dB为佳 |
| HEPA等级 | H11~H14 | H13以上为医疗级 |
| 甲醛CADR | 100~400m³/h | 专项除甲醛机型才有 |
| 功率 | 30~80W（常规） | 滤网等级越高通常功率越大 |

## 主要类型与技术路线

| 路线 | 原理 | 适用场景 | 代表产品 |
|------|------|---------|---------|
| **HEPA过滤型** | HEPA滤网拦截颗粒物 | 雾霾/PM2.5/花粉 | 小米Pro系列、352 X50 |
| **活性炭吸附型** | 活性炭多孔吸附甲醛/VOC | 甲醛/异味 | IQAir（医疗级活性炭） |
| **光触媒/光催化型** | 紫外光+催化剂分解甲醛 | 甲醛（但效果争议大） | 部分中端机型 |
| **负离子型** | 负离子沉降颗粒物+杀菌 | 辅助净化 | 亚都、夏普部分机型 |
| **静电集尘型** | 高压静电使颗粒物带电后吸附 | 大风量低噪音 | IQAir（医用款） |
| **新风机** | 引入室外空气过滤后注入室内 | 新装修/CO₂浓度高 | 米家新风机、果麦 |

## 技术演进路线

```
[第一代] 被动式过滤（约2000年前）
- 核心原理：风机抽风→初效滤网（大颗粒）→HEPA滤网（细颗粒）→活性炭→出风
- 核心元器件：初效滤网、HEPA滤网（玻璃纤维）、活性炭层
- 代表品牌：IQAir（瑞士，医疗级标准）、Blueair（瑞典）
- 主要特点：
  - 以HEPA为核心，CADR值300~500m³/h
  - 活性炭主要用于除异味（非除甲醛主要手段）
- 主要局限：
  - 滤网消耗品（3~6个月换一次，一套滤网500~2000元）
  - 除甲醛能力弱（活性炭吸附饱和后二次释放）

[第二代] HEPA+强效活性炭+甲醛数显（约2010~2018年）
- 核心升级：活性炭负载量增加（针对甲醛）+ 甲醛传感器+甲醛数显
- 核心元器件：
  - 改性活性炭（椰壳活性炭，碘值更高，吸附甲醛更强）
  - 甲醛传感器（电化学传感器，实时数显）
  - TVOC传感器
- 主要升级：
  - 甲醛CADR值成为重要指标（部分机型可达100~200m³/h）
  - 智能模式（根据空气质量自动调节风量）
  - APP远程控制+历史数据查看
- 代表品牌：352（全效空气净化器X80C）、小米（Pro系列）、IAM（英国）

[第三代] 多技术融合除甲醛（约2018~2021年）
- 核心升级：多种除甲醛技术叠加，试图解决活性炭饱和问题
- 技术组合：
  - 改良活性炭（椰壳+催化剂）
  - 负离子（沉降颗粒物+一定除甲醛效果）
  - 紫外灯（杀菌+辅助光触媒）
  - 冷触媒/光触媒（催化剂分解）
- 主要问题：市场混乱，消费者难以区分真实效果
  - 权威测评显示：大部分光触媒在室内光照条件下（<1000lux）甲醛分解效率极低
  - 负离子除甲醛效果微弱且产生臭氧风险
  - 活性炭仍是最有效的被动吸附手段

[第四代] 甲醛分解不依赖活性炭（约2021~至今）
- 核心突破：找到不依赖活性炭饱和的甲醛分解路线
- 两条主要技术路线：
  路线A：**氨基酸降解（化学法）**
    - 原理：氨基酸与甲醛反应，生成无害物质（羟甲基氨基酸）
    - 代表：352 Y106C（氨基酸除醛晶体）
    - 优点：不饱和、可重复反应
    - 缺点：氨基酸会消耗，需定期更换
    
  路线B：**锰系催化剂（氧化法）**
    - 原理：锰氧化物催化氧化甲醛为CO₂+H₂O
    - 代表：小米全效空气净化器Ultra（固态甲醛传感器+催化分解）
    - 优点：催化剂不消耗，长期有效
    - 缺点：需要较高温度才有最佳活性

- 其他升级：
  - 固态甲醛传感器（比电化学传感器更稳定，不受酒精干扰）
  - PM2.5+甲醛+温湿度+CO₂五维数显
  - 静音设计（睡眠档≤25dB）
  - 消毒级净化（部分机型增加UVC紫外+等离子杀菌）

[第五代] 全链路消毒+智能感知（约2023~至今）
- 核心升级：从PM2.5净化 → 全链路空气质量管理
- 传感器扩展：PM2.5+甲醛+CO₂+温湿度+TVOC（5~6种传感器）
- 消毒功能：等离子+UVC双重杀菌（细菌/病毒消杀率>99.9%）
- 智能感知：室内CO₂传感器（解决密闭空间CO₂浓度过高问题）
- 新风集成：部分机型集成新风功能（引入室外空气+过滤）
- 代表：小米全效空气净化器Ultra（分解甲醛）、A.O.史密斯（嵌入式新风）
```

## CADR值计算公式

```
适用面积（m²）≈ CADR值（m³/h）× 0.07 ~ 0.12
（0.07为保守估算，0.12为宽松估算，0.1为中间值）

示例：CADR=500m³/h → 适用面积≈35~60m²
```

## 选购核心维度

1. **除甲醛**：看甲醛CADR值（越大越好）和甲醛CCM值（F4最高）
2. **颗粒物净化**：看颗粒物CADR + HEPA等级（H13以上）
3. **传感器配置**：甲醛数显 > 无数显（避免盲区）
4. **滤网更换成本**：年耗材费用（元/天）= 滤网价格÷使用天数
5. **噪音**：睡眠档<35dB为优秀
6. **消毒功能**：UVC+等离子双杀菌 > 单一UVC
