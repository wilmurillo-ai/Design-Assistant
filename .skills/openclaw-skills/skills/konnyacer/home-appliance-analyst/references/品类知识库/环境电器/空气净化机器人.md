# 空气净化机器人

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| CADR值 | 200~500m³/h | 值越大净化越快 |
| 导航方式 | 随机碰撞→视觉/LDS激光 | 决定覆盖率 |
| 移动能力 | 全屋自主移动 | vs 固定位置净化器 |
| 噪音 | 30~65dB | 移动时噪音 |
| 滤网 | H13 HEPA+活性炭 | 同固定式 |

---

## 与固定式空气净化器的核心区别

| 对比 | 固定式空气净化器 | 空气净化机器人 |
|------|---------------|--------------|
| 工作原理 | 固定位置净化附近空气 | 自主移动→覆盖全屋 |
| 覆盖率 | 仅单房间有效 | 全屋覆盖（需多次全屋清扫） |
| CADR效率 | 即时净化（但范围有限） | 需要时间（移动遍历全屋） |
| 适用场景 | 单房间持续净化 | 全屋无死角净化 |

## 技术演进路线

```
[第一代] 固定式HEPA净化器（约2000~2015年）
- 核心原理：风扇抽风→HEPA滤网过滤→洁净空气排出（已在空气净化器品类详述）
- 局限：仅能净化净化器所在房间

[第二代] 可移动净化器概念（约2015~2018年）
- 核心探索：在固定式净化器底部加装轮子，实现简单移动
- 局限：无法自主导航，需人工推动

[第三代] LDS激光导航空气净化机器人（约2018~2021年）
- 核心突破：扫地机器人类导航技术移植到空气净化器
- 核心技术：
  - LDS激光雷达（同扫地机器人，360°扫描建图）
  - 自主移动（全屋路径规划）
  - HEPA净化模块（与固定式相同）
  - 甲醛/PM2.5传感器（实时显示）
  - 避障（防跌落，防碰撞）
- 核心元器件：
  - LDS激光模块（同扫地机器人）
  - 驱动电机（两轮差速驱动）
  - HEPA H13级滤网
  - BLDC风机
- 主要特点：
  - **自主移动**：充电座充电→自动出发→全屋巡航净化
  - **全屋覆盖**：扫地机器人清扫地面，净化机器人净化空气，互补
  - **指定区域净化**：APP指定某个房间重点净化
- 主要局限：
  - 移动速度慢（全屋一次需2~4小时）
  - 噪音（移动+净化双重）
  - 适用场景有限（不如固定式高效）

- 代表产品：
  - 科沃斯（Ecovacs）—— AIR FRESH系列（首款真正意义的空气净化机器人）
  - 石头（Roborock）—— 石头空气净化器（h系列产品，非移动）
  - 小米—— 米家空气净化器（移动款概念）
```

## 选购核心维度

1. **导航方式**：LDS激光 > 视觉 > 陀螺仪（与扫地机器人同理）
2. **CADR值**：>300m³/h为优秀
3. **HEPA等级**：H13以上
4. **移动避障**：多传感器避障（防缠绕/防跌落）
5. **APP控制**：指定区域净化
