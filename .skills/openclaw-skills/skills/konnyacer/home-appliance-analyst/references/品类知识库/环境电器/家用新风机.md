# 家用新风机

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 风量 | 50~300m³/h | 值越大换气越快 |
| 适用面积 | 50~200m² | 按风量×2估算 |
| 热回收效率 | 60~85% | 值越高越节能 |
| 噪音 | 25~50dB | 低档静音 |
| 过滤等级 | G4~H13 | 户外新风过滤精度 |
| 类型 | 单向流/双向流/全热交换 | 热回收决定节能性 |

---

## 主要类型

| 类型 | 原理 | 优点 | 缺点 |
|------|------|------|------|
| **单向流新风机** | 强制送风或强制排风 | 结构简单、价格低 | 无热回收，冬天耗能大 |
| **双向流新风机** | 送风+排风同时 | 可热回收（部分机型） | 安装复杂 |
| **全热交换新风机** | 双向流+全热交换芯 | 热+湿双向回收，最节能 | 价格高，安装要求高 |

---

## 技术演进路线

```
[第一代] 简易换气扇（约1980s~2000年）
- 核心原理：单向排风（排出室内浊气）或单向送风（注入室外空气）
- 局限：冬天室外冷空气直接进入室内，热量损失大

[第二代] 双向流热回收（约2005~2015年）
- 核心突破：送风+排风双向同时运行，热交换器回收排出空气的热量
- 核心元器件：
  - 送风风机（低噪音EC直流）
  - 排风风机
  - 热交换芯（纸质/铝制/陶瓷）
  - 初效+G4滤网（过滤室外花粉/大颗粒）
- 热交换原理：夏季排出冷气→预冷进入的室外热空气；冬季排出暖空气→预热进入的室外冷空气
- 主要升级：
  - **热回收效率60~75%**（排出空气热量的60~75%被回收利用）
  - 全热交换（热+湿双回收，室内湿度更舒适）
  - 减少空调/暖气负荷

[第三代] 智能变频+多重过滤（约2015~2020年）
- 核心升级：
  - **EC直流变频风机**（根据CO₂浓度自动调节风量）
  - **高效过滤**（F7~H11级HEPA，过滤PM2.5）
  - **CO₂传感器联动**（室内CO₂浓度>1000ppm自动加大风量）
  - **低噪音设计**（<30dB@低档）
  - **冷热源辅助**（配合空调/暖气工作）

[第四代] 壁挂式新风机（约2018~至今，爆发期）
- 核心突破：壁挂式新风机出现，安装无需全屋管道，工程量大幅降低
- 核心技术：
  - **壁挂单台安装**（只需墙上打一个孔）
  - **全热交换**（热+湿回收）
  - **PM2.5过滤**（H13级HEPA，进风洁净）
  - **APP智能控制**（CO₂/PM2.5双显示，联动空调/除湿机）
  - **静音技术**（<25dB，适合卧室）
  - **电辅热**（冬天室外极冷时，辅助加热进风）
- 核心元器件：
  - EC直流变频风机（两组，一送一排）
  - 全热交换芯（陶瓷/纸质，不可水洗）
  - H13 HEPA滤网
  - CO₂传感器（NDIR红外式）
  - PM2.5传感器（激光粉尘传感器）
  - 电辅热模块（PTC，约500~1000W）
- 代表品牌：
  - 果麦（GMK）—— 壁挂新风机爆款，高性价比
  - 小米（米家新风机）—— 接入米家生态
  - 352（Skin）—— 全热交换壁挂式
  - 皓庭（Haoting）—— 专业新风机品牌
  - 松下（Panasonic）—— 管道式+壁挂式双线
```

## 关键元器件及供应商

| 元器件 | 主流方案 | 备注 |
|--------|---------|------|
| 热交换芯 | 陶瓷（可水洗）/纸质（便宜） | 陶瓷可重复使用，纸质需定期更换 |
| HEPA滤网 | H13级 | 决定PM2.5过滤效果 |
| EC直流风机 | 直流无刷EC电机 | 变频+静音的关键 |
| CO₂传感器 | NDIR红外式 | 高端机型配备 |
| 电辅热模块 | PTC加热片 | 极寒天气辅助加热 |

## 选购核心维度

1. **类型**：壁挂式（适合已装修） vs 管道式（适合新装修）
2. **热回收**：全热交换（推荐） > 显热交换 > 无热回收
3. **过滤精度**：H13 HEPA（过滤PM2.5） > H11 > G4初效
4. **风量**：适用面积≈风量×1.5~2
5. **噪音**：卧室选<30dB
6. **CO₂联动**：有则更智能（按需通风）
7. **滤网成本**：年耗材成本 = HEPA+初效滤芯费用
