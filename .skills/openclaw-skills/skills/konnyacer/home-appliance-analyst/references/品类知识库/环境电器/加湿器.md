# 加湿器

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 加湿量 | 200~600mL/h | 值越大适用面积越大 |
| 适用面积 | 20~60m² | 加湿量×10≈适用面积 |
| 水箱容量 | 3~8L | 决定续航时间 |
| 噪音 | 25~55dB | 超声波款噪音低 |
| 控湿精度 | ±5%RH（高端） | 智能款可恒定湿度 |
| 加湿技术 | 超声波/蒸发式/电热式 | 决定安全性 |

---

## 主要类型与技术路线

| 类型 | 原理 | 优点 | 缺点 | 安全性 |
|------|------|------|------|--------|
| **超声波式** | 超声波振子产生水雾，扩散加湿 | 功耗低（15~30W）、噪音小、加湿速度快 | 水雾携带细菌/水垢风险（白粉）；对水质要求高 | ⚠️ 需用纯净水 |
| **蒸发式（无雾）** | 风机吹过湿膜，水分蒸发到空气中 | 无白粉、无雾气、安全性高 | 加湿速度慢、噪音相对大 | ✅ 可用自来水 |
| **电热式（蒸汽）** | 电热管加热水→产生蒸汽→扩散加湿 | 杀菌效果好、加湿均匀、无白粉 | 能耗高（300~800W）、有烫伤风险 | ⚠️ 需防烫 |
| **复合型** | 超声波+蒸发双重 | 综合两者优点 | 价格偏高 | 中 |

---

## 技术演进路线

```
[第一代] 电热蒸汽加湿器（约1980s~2000s）
- 核心原理：电热管加热水至沸腾→产生蒸汽→蒸汽自然扩散→加湿空气
- 核心元器件：电热管（不锈钢/石英）、水箱（不锈钢）、温度保险丝、水位开关
- 主要特点：
  - 加湿均匀，无白粉（蒸汽纯净）
  - 高温杀菌（细菌99.9%被杀灭）
  - 能耗高（500~800W）
  - 有烫伤风险（水箱和出汽口高温）
- 代表：亚都超声波前身的早期加湿器，国产品牌仿制日本产品

[第二代] 超声波加湿器（约2000~2015年）
- 核心突破：超声波雾化技术成熟，成本大幅下降
- 核心原理：超声波振子（压电陶瓷，1.7MHz/2.4MHz）→水分子雾化成1~5μm微粒→风扇吹出
- 核心元器件：
  - 超声波振子（压电陶瓷片，雾化核心）
  - 风扇（低噪音直流无刷）
  - 小型雾化片（雾化量可叠加）
  - 水位传感器（低水位保护）
- 主要升级：
  - 能耗极低（15~30W，比蒸汽款省电20倍）
  - 体积小巧（桌面级加湿器普及）
  - 价格低廉（30~200元）
  - 有白粉问题（自来水中钙镁离子随水雾扩散，产生白色粉末）
- 主要局限：
  - **白粉问题**：自来水中的钙镁离子随水雾扩散，沉降后形成白色粉末
  - **细菌风险**：超声波无法杀菌，水箱内细菌可能随水雾扩散
  - 儿童/哮喘/老人不建议用超声波+自来水

[第三代] 蒸发式无雾加湿器（约2015~2019年）
- 核心突破：蒸发式（冷蒸发）技术家用化，解决超声波的安全隐患
- 核心原理：风机→带动湿润滤网→水分子自然蒸发→清洁空气加湿（无雾气）
- 核心元器件：
  - 蒸发滤网（无纺布+纸蜂窝结构，吸水面积大）
  - 无刷直流风机
  - 水位感应（感应水位，自动停机）
  - 水箱UV灯（部分机型，抑菌）
- 主要优点：
  - **无白粉**（水分子蒸发，矿物质留在滤网）
  - **可用自来水**（无任何安全隐患）
  - 均匀加湿，无局部过湿
- 主要局限：
  - 加湿速度慢（同等功耗下比超声波慢3~5倍）
  - 滤网需定期清洗/更换（3~6个月）
  - 噪音相对大（风机工作声）
- 代表品牌：舒乐氏（SOLEUSAIR）、IAM（ SZ系列）、亚都（YC-D204）、格力（加湿器品类）

[第四代] 智能变频+杀菌加湿（约2019~2022年）
- 核心升级：
  - **银离子/UV杀菌**（水箱内置银离子盒或UVC紫外灯，杀菌抑菌）
  - **智能恒湿**（湿度传感器+自动调节雾量，恒定设定湿度）
  - **APP远程控制**（开关+湿度设定）
  - **上加水设计**（直接打开盖子加水，无需拆卸水箱）
  - **复合滤芯**（活性炭+银离子+无机抗菌剂）
- 核心元器件：
  - 湿度传感器（电容式，实时检测室内湿度）
  - UV-C紫外灯（254nm，杀菌）
  - 银离子发生器（电解释放Ag+，抑菌）
  - 变频风机（根据湿度调节风量，节能降噪）

[第五代] 高端无雾+多功能（约2022~至今）
- 核心升级：
  - **大风量无雾加湿**：加湿量达600mL/h（大蒸发式，接近超声波速度）
  - **滤芯超长寿命**：抗菌滤网寿命12个月以上
  - **空气净化+加湿一体**（部分机型加入HEPA+活性炭模块）
  - **香薰功能**（预留香薰盒）
  - **零水雾设计**（蒸发式彻底取代超声波成主流）
  - **婴儿级安全**：无雾+银离子杀菌+智能恒湿三合一
- 代表品牌：IAM（ SZ600）、舒乐氏（SOLEUSAIR 600D1）、大宇（DAEWOO）、巴慕达（BALMUDA，日本高端）
- 近年趋势：
  - **蒸发式取代超声波成为主流**（消费者安全意识提升）
  - **加湿+净化二合一**（一机两用，节省空间）
  - **智能家居联动**（与温湿度计/空调/除湿机联动，自动调节室内湿度）
```

## 选购核心维度

1. **加湿技术**：蒸发式（无雾，推荐） > 超声波（需纯净水） > 电热式（能耗高）
2. **加湿量**：适用面积×10=所需加湿量（如30㎡选300mL/h）
3. **水箱容量**：>4L可减少加水频率
4. **噪音**：卧室选<35dB
5. **杀菌功能**：UV杀菌/银离子 > 无杀菌（蒸发式+杀菌=最安全组合）
6. **恒湿功能**：有湿度传感器+自动调节 > 手动档位
7. **耗材成本**：滤网更换周期和单价（年耗材费用）
8. **水质要求**：自来水可用选蒸发式；纯净水可选超声波
