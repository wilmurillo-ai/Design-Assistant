# 电磁炉 / 电陶炉

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 功率 | 1000~3000W | 值越大加热越快 |
| 火力档位 | 8~99档 | 档位越多控火越精细 |
| 面板材质 | 微晶玻璃/黑晶面板 | 决定耐热性和价格 |
| 线盘直径 | 150~280mm | 值越大加热越均匀 |
| 热效率 | 86~95% | 电磁炉远高于电陶炉 |

---

## 主要类型对比

| 类型 | 原理 | 热效率 | 适用锅具 | 优点 | 缺点 |
|------|------|--------|---------|------|------|
| **电磁炉** | 电磁感应，锅体直接发热 | 90~95% | 仅铁系锅具 | 热效率高、省电、加热快 | 仅适用铁锅 |
| **电陶炉** | 电阻丝/卤素管加热面板 | 60~70% | 所有锅具 | 不挑锅、可小件加热 | 热效率低、耗电、表面烫 |
| **双炉（电磁+电陶）** | 两者结合 | — | 铁/陶/玻璃均可 | 一机两用 | 价格高、体积大 |

## 技术演进路线

### 电磁炉

```
[第一代] 家用电磁炉（约1980s~2000s）
- 核心原理：交流电→整流滤波→IGBT高频开关→高频电流→线圈盘产生交变磁场→锅底产生涡流发热
- 核心元器件：
  - 线圈盘（铜线绕制，150~180mm直径）
  - IGBT管（绝缘栅双极晶体管，高频开关核心）
  - 整流桥（AC→DC）
  - LC振荡电路（产生高频电流）
  - 普通微晶玻璃面板
- 主要特点：
  - 1980年代德国Schott发明首个家用电磁炉
  - 1990年代日本普及，国产2000年后跟进
  - 功率：1000~2000W
  - 面板：普通黑晶，耐用性一般
- 主要局限：
  - 线圈小（加热不均，大锅易中间糊）
  - 只能用铁系锅具（不锈刚/铁锅）
  - 风扇噪音大

[第二代] 大线盘+多样化（约2005~2015年）
- 核心升级：
  - **线盘直径扩大**（180→200→220mm，加热更均匀）
  - **功率提升**（2000→2200→3000W）
  - **多功能**（炒/煮/煲/汤/火锅/定时）
  - **防水设计**（IGBT风道优化，降低进水风险）
  - **持续小火加热**（解决普通电磁炉小火间断加热的问题）
  - **一键猛火/文火**（快速切换）
- 核心元器件：
  - 大直径线圈盘（220mm，铜线更粗）
  - 高品质微晶面板（德国肖特Schott/日本NEG）
  - 高速静音风扇（滚珠轴承BLDC风扇）
- 代表品牌：
  - 美的（Midea）—— 国内市场绝对霸主
  - 九阳（Joyoung）—— 豆浆机品牌延伸
  - 苏泊尔（SUPOR）—— 锅具+灶具全布局
  - 尚明堂（尚朋堂）—— 专业电磁炉

[第三代] 智能IH+持续加热（约2016~至今）
- 核心突破：持续低功率加热（取代传统间断加热），实现类明火烹饪
- 核心技术升级：
  - **恒匀火技术**（Continuous Heating，持续小火不间断）
  - **多段IH功率控制**（精确到1W调节）
  - **大线盘（280mm）**（覆盖更大锅底，加热更均匀）
  - **低噪音设计**（无刷风机，噪音<40dB）
  - **黑科技面板**（汉森/微晶玻璃，防刮花、防渗透）
  - **防辐射设计**（IGBT频率优化，EMC电磁兼容）
  - **WIFI/APP联动**（部分高端机型，远程控温）
- 代表品牌/产品：
  - 美的（Midea）—— 恒匀火系列（旗舰）
  - 苏泊尔（SUPOR）—— 秘镜系列（大线盘）
  - 九阳（Joyoung）—— F2/F5系列（智能控温）
  - 尚明堂（尚朋堂）—— 日本原装进口电磁炉（顶级线盘）
```

### 电陶炉

```
[补充] 电陶炉技术路线
- 原理：卤素管/镍铬合金发热丝加热面板→面板导热→锅具受热
- 第一代（2000~2010）：卤素灯管（热得快但发热丝寿命短）
- 第二代（2010~至今）：镍铬合金发热盘（寿命更长，控温更稳定）
- 核心优势：不挑锅（陶/瓷/玻璃/铁均可）；缺点：热效率低（仅60~70%），面板烫伤风险
- 代表品牌：米家（小米）电陶炉（红外加热+APP联动）
```

## 选购核心维度

1. **功率**：家用2200W足够，爆炒可选3000W
2. **线盘直径**：>200mm为优秀（加热均匀）
3. **小火持续加热**：选支持"恒匀火"/"持续小火"的机型（炒菜必备）
4. **面板材质**：德国肖特/NEG > 国产微晶
5. **适用锅具**：只有铁锅→电磁炉；多类型锅具→电陶炉或双炉
6. **火力档位**：>8档为基本，>15档为精细
7. **噪音**：<45dB为安静
