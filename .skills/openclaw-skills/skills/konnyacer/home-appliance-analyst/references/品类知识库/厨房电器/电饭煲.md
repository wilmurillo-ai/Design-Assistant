# 电饭煲

## 核心参数速查

| 参数 | 典型值 | 说明 |
|------|-------|------|
| 容量 | 1.5~5L | 1.5L（1~2人）/ 4L（3~5人） |
| 功率 | 400~1500W | IH款功率更大 |
| 内胆材质 | 铝合金/黑晶/球釜/铁釜/土锅 | 材质决定导热和米饭口感 |
| 加热方式 | 底盘加热/IH电磁加热/压力IH | 加热方式决定米饭品质 |
| 功能 | 煮饭/煲汤/蒸煮/蛋糕/粥/杂粮 | 功能越多样价格越高 |

---

## 主要类型与技术路线

| 类型 | 原理 | 优点 | 缺点 | 米饭品质 |
|------|------|------|------|---------|
| **底盘加热式** | 电热盘加热内胆底部，热传导到米饭 | 价格低、结构简单 | 加热不均（底部糊/上层夹生） | ⭐⭐ |
| **IH电磁加热** | 电磁线圈在锅体四周产生涡流，直接加热内胆 | 立体加热均匀 | 价格偏高 | ⭐⭐⭐⭐ |
| **压力IH** | IH加热+微高压（1.2~1.5大气压） | 沸点>100℃，米饭糊化更好 | 贵，炖汤需小心泄压 | ⭐⭐⭐⭐⭐ |

---

## 技术演进路线

```
[第一代] 底盘加热式电饭煲（约1950s~2000年）
- 核心原理：电热盘（铝合金铸造）加热→热量传导至内胆底部→加热米饭
- 核心元器件：
  - 电热盘（铝合金铸造，400~800W）
  - 普通内胆（铝合金+不粘涂层）
  - 磁钢限温器（通过磁性感应饭熟，自动断电）
  - 普通定时器（机械定时）
- 主要特点：
  - 1955年日本东芝发明首个电饭煲（自动保温）
  - 功能单一（仅煮饭）
  - 米饭质量一般（底部温度过高，米饭干硬）
- 代表：美的（Midea）、苏泊尔（SUPOR）早期产品

[第二代] IH电磁加热（约2005~2015年）
- 核心突破：日本虎牌/松下于1980s发明IH电饭煲，2010年代国产化普及
- 核心技术：电磁线圈（IH线圈）在内胆周围产生电磁场→金属内胆产生涡流发热→内胆整体均匀发热
- 核心元器件：
  - IH线圈组（缠绕在内胆侧面和底部）
  - IGBT控制芯片（精确控制加热功率）
  - 铁釜/陶釜内胆（铁/陶内胆导磁+蓄热好）
  - 微电脑控制板（MCU+NTC测温）
  - 复合内胆（铝合金+不粘涂层+导磁层）
- 主要升级：
  - **立体加热**（底部+侧面同时加热，米饭均匀熟透）
  - **精确控温**（NTC实时监测，糊化温度精准控制）
  - **多种米饭模式**（精华饭/快煮/粥/糙米/蒸炖）
  - **24小时预约**（提前放入原料，预约时间自动煮）
  - **保温功能**（煮完后自动进入保温状态）
- 代表品牌：
  - 虎牌（Tiger，日本高端，JKT/JBW系列）
  - 松下（Panasonic，SR-某系列）
  - 美的（MB-系列，IH普及推动者）
  - 苏泊尔（SF-系列）

[第三代] 压力IH+高端内胆（约2015~2020年）
- 核心突破：加入微压（1.2~1.5个大气压），沸腾温度>100℃，米饭完全糊化，口感大幅提升
- 核心技术：
  - **微压IH**（高压锅原理用于电饭煲，1.2~1.5大气压）
  - **土锅涂层**（模仿土锅远红外效果）
  - **可变压力技术**（煮饭不同阶段施加不同压力）
  - **多段IH**（底部+侧面+顶部多组线圈，分段加热）
- 核心元器件：
  - 加压电磁阀（精确控制压力）
  - 土锅内胆（陶晶涂层+远红外效果）
  - 多段IH线圈（3~7段）
  - 压力传感器（检测内压）
- 代表产品：
  - 虎牌JKT系列（压力IH+土锅内胆，顶级口感）
  - 松下SR-AFX系列（变频压力IH）
  - 美的MB-系列（压力IH国产旗舰）
  - 米家（小米）IH电饭煲（性价比，APP联动）

[第四代] 智能+微压高端化（约2020~至今）
- 核心升级：
  - **微压2.0**（压力IH成为高端标配，300元以上的IH机型普遍带微压）
  - **APP智能联动**（云端食谱，自动识别米种，匹配最优加热曲线）
  - **米种识别**（东北米/丝苗米/泰国香米，自动匹配程序）
  - **微气泡技术**（底部产生微气泡，使米饭更蓬松）
  - **不粘内胆革命**（大金PFA粉体涂层，不粘性+耐用性大幅提升）
  - **小型化**（1~1.5L小容量，满足单身/小家庭需求）
  - **低糖电饭煲**（米汤分离技术，降低淀粉含量）
- 代表品牌：
  - 虎牌/松下/象牌（日本三强，高端市场）
  - 米家（小米，IH智能系列）
  - 美的（压力IH旗舰）
  - 松下（迷你电饭煲系列）
```

## 内胆材质进化详解

| 材质 | 特点 | 成本 | 品牌偏好 |
|------|------|------|---------|
| 铝合金+不粘涂层 | 导热快、轻便 | 低 | 入门款 |
| 黑晶内胆 | 硬度高，不粘好 | 中 | 中端款 |
| 球釜内胆 | 球形底部，热对流好 | 中 | 中高端 |
| 铁釜内胆 | 蓄热好，IH兼容 | 中高 | IH机型 |
| 土锅内胆 | 远红外效果，口感最佳 | 高 | 高端旗舰 |

## 选购核心维度

1. **加热方式**：压力IH > IH > 底盘加热（按预算选）
2. **内胆材质**：铁釜/土锅 > 球釜 > 普通不粘锅
3. **容量**：1.5L（1~2人）/ 3L（2~4人）/ 4L（4~6人）
4. **米种识别**：有则更智能（对米种自动适配）
5. **功能需求**：低糖饭/煲汤/蒸炖（按需选择）
6. **APP控制**：云端食谱更新（年轻人偏好）
