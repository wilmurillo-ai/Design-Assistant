# YumFu Deep Narrative Engine (DNE)
# 深度叙事引擎设计规范 v1.0

## 核心哲学

**每一个选择都是起点，不是终点。**

无论玩家选什么，故事都要继续。
没有"死路"，只有不同的路。
每条路都有30+个决策节点。

---

## 🎭 通用叙事结构

所有YumFu世界都遵循以下三弧结构：

```
Arc 1: 建立 (全程20%)  - 角色定位，世界观建立，第一次危机
Arc 2: 发展 (全程60%)  - 核心冲突，人物关系，道德困境
Arc 3: 高潮 (全程20%)  - 最终抉择，多种结局
```

**最少节点数**: 每个世界30个，每个角色路线20个
**结局数量**: 每个世界最少6种结局

---

## 📊 通用隐藏指标系统

每个世界都追踪以下指标（数值0-100）：

```json
{
  "core_stats": {
    "reputation": 50,        // 声望 - 影响NPC态度
    "moral_alignment": 50,   // 道德倾向 (0=黑暗, 100=光明)
    "risk_exposure": 0,      // 暴露度/危险度
    "relationship_karma": 50 // 人际因果
  },
  "world_specific": {
    // 每个世界额外2-4个专属指标
    // 笑傲江湖: 武功值/江湖声望/门派地位
    // Harry Potter: 魔力值/学院积分/黑魔法倾向
    // F15 Down: 愤怒值/信任值/父亲关系
    // Warrior Cats: 战士等级/氏族荣誉
  },
  "npc_trust": {
    // 每个重要NPC独立信任值
    // 影响他们愿意分享的信息和提供的帮助
  }
}
```

## 🔄 决策后果系统

### 立即后果 (Immediate)
- 场景变化
- NPC反应
- 资源获得/失去

### 延迟后果 (Delayed)
- 3-5个节点后触发
- "你之前的决定产生了影响..."
- 玩家会感受到蝴蝶效应

### 累积后果 (Cumulative)
- 多次相似选择积累效果
- 影响最终结局判定
- 某些结局需要特定指标阈值

### 隐藏后果 (Hidden)
- 玩家不知道某个选择的后果
- 直到很久以后才显现
- 增加重玩性

**核心主题**: 自由 vs 名利 vs 爱情

- Day 16-25: 正邪大战，立场危机

**6种结局**:
1. 笑傲江湖 - 隐居山林，与爱人逍遥
2. 大侠 - 成为武林盟主，守护正义
3. 邪教掌主 - 统一魔教，称霸武林
4. 殉道 - 为保护朋友牺牲
5. 出走 - 离开中原，浪迹海外
6. 归隐 - 假死脱身，普通人生活

## 🎲 通用随机事件系统

每3-5个节点，触发一次随机事件：

**类型**:
- 遭遇 (Encounter): 遇到新NPC
- 危机 (Crisis): 紧急状况需要应对
- 发现 (Discovery): 发现隐藏信息
- 机遇 (Opportunity): 特殊资源/技能获取
- 回响 (Echo): 之前决定的延迟后果

**世界专属随机事件**:
- 笑傲江湖: 偶遇独孤九剑传人/找到武功秘籍
- Harry Potter: 发现禁忌书籍/被抓到用魔法
- Warrior Cats: 发现领地入侵者/受伤的小猫
- F15 Down: 被特工跟踪/发现秘密文件

## 📝 NPC记忆系统

每个重要NPC记住：
```json
{
  "npc_id": "yusuf",
  "trust": 65,
  "remembered_events": [
    "player_refused_assassination",
    "player_helped_with_food",
    "player_lied_about_location"
  ],
  "current_attitude": "cautious_ally",
  "secrets_shared": ["faction_funding_source"],
  "secrets_hidden": ["yusuf_is_double_agent"]
}
```

高信任值 → NPC分享更多秘密，提供更多帮助
低信任值 → NPC可能出卖你，提供错误信息

## 🎯 节点设计模板

每个叙事节点必须包含：

```markdown
## [世界] Day X, [时间] - [节点名称]

### 场景描述 (2-3段)
沉浸式描述，不超过200字

### 对话/事件
NPC对话或关键事件

### 隐藏信息 (DM Only)
玩家不知道的背景信息
对未来节点的影响

### 玩家选择 (3-4个)
A. [选择]
   - 立即效果
   - 延迟效果（隐藏）
   - 影响的指标

B. [选择]
   ...

### 触发条件
如果某指标达到阈值，解锁额外选项D/E

### 下一节点
A → 节点X+1a
B → 节点X+1b
C → 节点X+1c
```
