# YumFu v1.7.1 — Support Link Update 💖

## Highlights

This is a lightweight documentation release focused on making it easier for players and supporters to back YumFu development.

## What's New

- Added a **Stripe support badge** to `README.md`
- Added a **Support YumFu** section near the top of the GitHub README
- Added a **Donate via Stripe** support line to the user-visible top section of `SKILL.md`

## Why this release

YumFu is growing across GitHub and ClawHub, and this update adds a simple, low-friction way for players and supporters to donate voluntarily.

This is a support link only — not a paywall, not a gameplay lock, and not a subscription requirement.

## No gameplay changes

- No story logic changes
- No world changes
- No save format changes
- No command changes

## Release Focus

**Main theme of v1.7.1:**
> Add a clean, optional donation path for people who want to support YumFu.
