# YumFu 门派与武功数据

## 九大门派

- **初始武功**: 罗汉拳

- **著名人物**: 少林方丈、扫地僧

## 传世秘籍

### 九阴真经
- **类型**: 内功 + 外功
- **效果**: 内力+30, 攻击+15, 防御+15, 速度+10
- **招式**: 九阴白骨爪、摧心掌
- **获取**: 峨嵋派任务、华山思过崖
- **限制**: 需悟性≥13

### 九阳神功
- **类型**: 内功
- **效果**: 内力+50, 体力恢复速度+200%, 免疫寒毒
- **招式**: 无（纯内功）
- **获取**: 昆仑山冰窟、少林藏经阁
- **限制**: 需悟性≥14

### 独孤九剑
- **类型**: 剑法
- **效果**: 攻击+20, 速度+15, 破招能力
- **招式**: 破剑式、破刀式、破枪式等九式
- **获取**: 华山思过崖、风清扬传授
- **限制**: 需悟性≥15, 无内功要求

### 降龙十八掌
- **类型**: 掌法
- **效果**: 攻击+25, 威力极大
- **招式**: 亢龙有悔、飞龙在天、神龙摆尾等
- **获取**: 丐帮帮主传授、洪七公传授
- **限制**: 需内力≥150, 善恶值≥50

### 葵花宝典
- **类型**: 内功 + 剑法
- **效果**: 速度+30, 攻击+20, 内力+40
- **招式**: 天下武功，唯快不破
- **获取**: 日月神教密藏、福威镖局
- **限制**: 需付出"代价"（自宫）
- **风险**: 练成后性情大变

### 六脉神剑
- **类型**: 指法（远程）
- **效果**: 攻击+18, 速度+12, 远程攻击
- **招式**: 少商剑、商阳剑等六剑
- **获取**: 大理段氏传授
- **限制**: 需内力≥200, 悟性≥16
- **特点**: 内力消耗极大

### 易筋经
- **类型**: 内功
- **效果**: 体力+40, 内力+40, 防御+20
- **招式**: 无（纯内功）
- **获取**: 少林派高级任务
- **限制**: 需少林弟子, 等级≥30

### 吸星大法
- **类型**: 内功（邪派）
- **效果**: 可吸取敌人内力
- **招式**: 吸人内力、转化为己用
- **获取**: 日月神教、梅庄地牢
- **限制**: 善恶值≤-20
- **风险**: 多种内力混杂，易走火入魔

## 著名NPC

- **身份**: 丐帮帮主、襄阳城守将
- **性格**: 憨厚正直、侠之大者
- **可学武功**: 降龙十八掌、九阴真经
- **好感度**: 需善恶值≥50, 完成襄阳守城任务
- **位置**: 襄阳城

- **可学武功**: 弹指神通、碧海潮生曲
- **好感度**: 需完成奇怪的任务（如陪他下棋）

- **好感度**: 需华山派弟子，悟性≥15
