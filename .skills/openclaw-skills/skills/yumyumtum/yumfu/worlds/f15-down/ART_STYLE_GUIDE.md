# Shadow War - Art Style Guide

## 🎨 Visual Style: Command & Conquer RTS Aesthetic

**Reference**: Command & Conquer Generals, Red Alert series, World in Conflict

---

## 🖼️ Character Portraits

### Style Requirements
- **NOT photorealistic** - stylized 3D renders
- Command & Conquer Generals commander portrait style
- Dramatic, heroic poses
- Cinematic lighting with faction color accents
- Age 35-50, battle-hardened but not gritty/realistic

### Prompt Template
```
Portrait of {name}, a {role} from {faction}.
Command & Conquer Generals style military commander portrait.
3D render, stylized but detailed.
Wearing tactical uniform with rank insignia and {faction_color} accents.
Command center background with holographic tactical displays.
Heroic determined pose, looking at camera.
Cinematic dramatic lighting with red/blue accent lights.
RTS game character portrait aesthetic - similar to C&C Generals faction leaders.
{personality_trait}
NOT photorealistic, NOT documentary style.
```

### Faction Color Coding
- **Phoenix State**: Blue/white accents
- **Mountain Republic**: Red/black accents
- **Western Alliance**: Navy blue/gold accents
- **Gulf Emirates**: Teal/white accents
- **Eastern Coalition**: Red/yellow accents
- **Desert Kingdom**: Green/gold accents

### Personality Visual Traits
- **Aggressive**: Fierce expression, forward-leaning pose, clenched jaw
- **Diplomatic**: Calm demeanor, hand gestures, thoughtful gaze
- **Pragmatic**: Sharp calculating eyes, arms crossed, confident stance
- **Moral**: Compassionate but firm, open posture, steady gaze

## 🗺️ Tactical Maps

### Style Requirements
- Isometric or top-down battlefield view
- C&C minimap aesthetic with unit icons
- Territory markers and control zones
- HUD overlays with threat indicators
- Red/blue faction color coding
- NOT satellite imagery, NOT photorealistic

### Prompt Template
```
Command & Conquer style tactical battlefield map.
Isometric view of {location}.
Military RTS game minimap aesthetic.
Red enemy positions (Mountain Republic) vs Blue friendly (Phoenix State).
Unit icons: tanks, aircraft, missile sites, buildings.
Territory control zones highlighted.
HUD overlays with threat indicators and movement vectors.
Dark blue background with grid lines.
Glowing tactical markers and waypoints.
C&C Generals war room holographic display style.
NOT satellite photo, NOT realistic.
```

---

## 💥 Combat Scenes

### Style Requirements
- Dynamic action like C&C cutscenes
- Stylized explosions and effects
- Dramatic camera angles
- RTS game cinematic quality
- NOT gritty/realistic war photography

### Prompt Template
```
Command & Conquer style {combat_scene}.
RTS game cinematic cutscene aesthetic.
Dramatic action shot: {description}.
Stylized explosions with bloom effects.
Military units in dynamic poses.
Cinematic camera angle from low/side.
Faction colors: {faction_colors}.
Sunset/dramatic sky background.
Similar to C&C Generals mission briefing cinematics.
NOT realistic war photography, NOT documentary style.
```

### Example Scenes
- **Airstrike**: F-35s dropping bombs on facility, stylized explosions
- **Missile Launch**: Ballistic missiles launching from underground silos, dramatic sky
- **Drone Swarm**: 50 drones flying in formation, red targeting lasers
- **Naval Battle**: Destroyer firing missiles at coastline, ocean spray
- **Special Forces Raid**: Commandos breaching door with flashbangs, tactical lights

## 🏙️ Location Backgrounds

### Style Requirements
- Stylized military base/city renders
- RTS game environment quality
- Cinematic lighting and atmosphere
- Clear silhouettes, not too detailed
- Faction-appropriate architecture

### Prompt Template
```
{location_name} in Command & Conquer game style.
RTS game environment render.
{location_type} with {architectural_style}.
Faction colors and flags: {faction}.
Cinematic atmosphere: {time_of_day}.
Military structures: hangars, radar towers, defensive walls.
Clear readable silhouettes.
Similar to C&C Generals base construction view.
NOT photorealistic, NOT generic stock photos.
```

## 🎭 UI Elements

### HUD Overlays
- Tactical minimap (top-right corner)
- Resource counters (top-left)
- Threat indicators (red/yellow/green)
- Unit health bars
- Faction emblems
- Day/time counter
- Mission objectives panel

### Color Scheme
- **Background**: Dark blue-gray (#1a2332)
- **Friendly**: Blue (#4a9eff)
- **Enemy**: Red (#ff4a4a)
- **Neutral**: Yellow (#ffd54a)
- **UI Accents**: Teal (#00ffcc)

## ❌ What to Avoid

### DO NOT Use:
- ❌ Photorealistic military stock photos
- ❌ Gritty war documentary style
- ❌ Real news footage aesthetic
- ❌ Overly detailed realistic gore
- ❌ Generic modern military shooter look
- ❌ Desaturated color palette

### Instead Use:
- ✅ Stylized 3D renders
- ✅ Saturated faction colors
- ✅ Heroic/cinematic framing
- ✅ RTS game aesthetic
- ✅ Clean readable visuals
- ✅ Dramatic lighting
