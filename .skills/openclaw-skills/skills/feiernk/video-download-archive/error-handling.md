# Error Handling

## Immediate Failures
以下情况直接终止：
- 输入不是合法 URL
- 链接不可访问
- yt-dlp 无法识别为可下载视频

## Retry Policy
- 单阶段最多重试 3 次
- 默认单任务最长运行时间 10 分钟
- 不允许无限重试

## Failure Classes
### 1. 输入错误
- URL 非法
- 空输入

### 2. 访问错误
- 页面打不开
- 超时
- 403 / 404 / 地区限制

### 3. 识别错误
- 不是视频页
- yt-dlp 无法提取
- 需要登录但 cookies 也无法解决

### 4. 下载错误
- 下载失败
- 合并失败
- 格式选择失败
- 磁盘空间不足

### 5. 归档错误
- 元数据补全失败
- 飞书写表失败
- 附件上传失败

## Required Behavior
- 所有失败都要返回明确原因
- 下载成功但飞书记表失败时：保留本地文件，不回滚下载结果
- 如果只是缩略图拿不到：留空，不要把整个任务判定为失败
- 如果用户手动移动了文件：优先反查，不要因为路径变化就直接重下
