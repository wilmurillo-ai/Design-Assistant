# Slogan Maker Tips

## 好广告语的特征

1. **简短有力** — 最好不超过10个字
2. **朗朗上口** — 押韵或有节奏感
3. **突出差异** — 表达独特卖点(USP)
4. **情感共鸣** — 触动目标受众的情感
5. **易于记忆** — 听一遍就能记住

## 创作技巧

| 技巧       | 示例                        |
|-----------|---------------------------|
| 对比法     | 小身材，大味道               |
| 押韵法     | 人头马一开，好事自然来        |
| 数字法     | 27层净化                    |
| 疑问法     | 你的益达？不，是你的益达      |
| 双关法     | 衣衣不舍（洗衣液）           |
| 重复法     | 今年过节不收礼，收礼只收脑白金 |

## 经典广告语分析

- **Nike: Just Do It** — 简洁、有力、普适
- **Apple: Think Different** — 突出品牌调性
- **可口可乐: 分享快乐** — 情感连接
- **去哪儿: 总有你要的低价** — 功能承诺

## 测试方法

- A/B测试不同版本
- 朗读测试（听着顺不顺）
- 10秒回忆测试（看一遍能否复述）
- 负面联想检查（有无歧义）
