# Funnel Analyzer Tips

## 漏斗分析核心

1. **定义清晰步骤** — 每步必须是可量化的用户行为
2. **关注最大流失点** — 优化投入产出比最高的环节
3. **分群分析** — 不同用户群的漏斗表现可能截然不同
4. **时间窗口** — 考虑合理的转化时间窗口

## 常见漏斗类型

| 类型        | 步骤示例                               |
|------------|---------------------------------------|
| 电商购买    | 浏览 → 加购 → 下单 → 支付 → 完成        |
| SaaS注册   | 访问 → 注册 → 激活 → 留存 → 付费        |
| 内容转化    | 曝光 → 点击 → 阅读 → 互动 → 转化        |
| App下载    | 广告展示 → 点击 → 商店 → 下载 → 打开     |
| 招聘       | 投递 → 筛选 → 面试 → offer → 入职      |

## 行业基准参考

- 电商加购率：3-8%
- SaaS试用→付费：2-5%
- 注册页转化率：20-40%
- 支付页完成率：60-85%

## 优化优先级

优先优化靠近底部的步骤 — 越靠近转化终点，每提升1%的商业价值越大。

## 注意事项

- 漏斗不一定是线性的，用户可能跳步或回退
- 设置合理的归因窗口（7天/30天）
- 对比不同时段的漏斗，发现趋势变化
