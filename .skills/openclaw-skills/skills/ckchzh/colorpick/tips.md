# Color Palette Tips 🎨

## 配色基础

1. **60-30-10法则** — 主色60%，辅色30%，点缀色10%
2. **对比度很重要** — 文字至少4.5:1(AA级)，大文字3:1
3. **不超过5种颜色** — 一个方案里3-5种色足够
4. **先选主色** — 再用harmony命令生成搭配色

## 常用场景

- `generate` — 不知道用什么色？给个关键词自动推荐
- `harmony` — 已有品牌色，需要搭配色
- `contrast` — 检查文字在背景上是否可读
- `convert` — 设计稿给了HEX，代码需要RGB

## WCAG对比度标准

- **AAA级(7:1)** — 最佳可读性
- **AA级(4.5:1)** — 正常文字最低要求
- **AA大文字(3:1)** — 18px+或14px粗体
- **不合格(<3:1)** — 避免用于文字
