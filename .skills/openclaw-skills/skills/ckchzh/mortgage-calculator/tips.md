# 🏠 房贷实用指南

## 买房前必知

### 📌 我到底能买多大的房？
1. 算月供承受能力: 月供不超过家庭月收入的50%（安全线30%）
2. 算首付: 首套房首付20%~30%（各城市不同）
3. 公式: **可买总价 = 首付款 ÷ 首付比例**
4. 例: 手里有60万首付，首付比例30% → 最高可买200万的房

### 📌 房贷两种还款方式

| 对比项 | 等额本息 | 等额本金 |
|--------|---------|---------|
| 月供 | 固定不变 | 逐月递减 |
| 前期压力 | 较小 | 较大 |
| 总利息 | 较多 | 较少 |
| 适合人群 | 收入稳定 | 收入较高 |
| 前期还的 | 利息为主 | 本金为主 |

**💡 100万贷30年(3.5%):**
- 等额本息: 月供4490，总利息61.7万
- 等额本金: 首月5694→末月2786，总利息52.7万
- 差额: 等额本金少付约9万利息

### 📌 公积金贷款 vs 商业贷款

| 对比 | 公积金 | 商贷 |
|------|--------|------|
| 利率 | 3.1%（2024） | 3.5%~4.2% |
| 额度上限 | 各地不同(60~120万) | 看征信和收入 |
| 审批速度 | 慢(1~3个月) | 快(2~4周) |
| 申请条件 | 连续缴存6~12个月 | 征信良好 |

**💡 建议: 优先用公积金贷款，不够的部分用商贷（组合贷）**

## 提前还款攻略

### 📌 什么时候提前还划算？
- **还款前1/3时间** — 最划算（利息还得多的阶段）
- **等额本息比等额本金** — 提前还更划算（因为利息占比更大）
- **利率高的先还** — 商贷利率>公积金，先还商贷

### 📌 提前还款的两种方式
1. **缩短年限（推荐）** — 月供不变，提前还清，省利息最多
2. **减少月供** — 年限不变，月供减少，减轻月压力

### 📌 什么情况不建议提前还？
1. 房贷利率 < 理财收益率（如房贷3%，理财5%）
2. 有其他高息负债（先还信用卡、花呗）
3. 手里没留3~6个月应急资金
4. 利率已经很低了（3%以下）
5. 还款已过2/3周期（剩的主要是本金了）

### 📌 提前还款节省利息参考
100万贷30年(3.5%)等额本息:
| 已还 | 提前还20万 | 省利息 | 缩短 |
|------|-----------|--------|------|
| 1年 | ✅ | ~15万 | ~7年 |
| 5年 | ✅ | ~11万 | ~6年 |
| 10年 | ⚠️ | ~7万 | ~4年 |
| 20年 | ❌ | ~2万 | ~2年 |

## 房贷利率知识

### 📌 LPR是什么？
- LPR = 贷款市场报价利率
- 每月20日更新（关注5年期LPR）
- 你的房贷利率 = LPR + 加点（签合同时确定）
- 每年1月1日自动调整（按最新LPR）

### 📌 2024年利率参考
- 5年期LPR: 3.95%（2024.2）→ 持续下降趋势
- 首套房: LPR-20bp = 3.75%（部分城市更低）
- 二套房: LPR+20bp = 4.15%
- 公积金: 3.1%（首套）/ 3.575%（二套）

### 📌 利率每降0.1%能省多少？
100万30年等额本息:
- 3.5% → 月供4490 → 总利息61.7万
- 3.4% → 月供4432 → 总利息59.6万
- 3.3% → 月供4375 → 总利息57.5万
- **每降0.1%，30年省约2万利息**

## 贷款能力评估

### 📌 银行审批看什么？
1. **月收入** — 月供不超过月收入的50%
2. **征信报告** — 近2年无逾期
3. **负债率** — 所有负债月供/月收入 < 50%
4. **工作稳定性** — 在职时间、公司规模
5. **流水** — 近6个月银行流水

### 📌 提高贷款额度的方法
1. 增加首付比例
2. 延长贷款年限（最长30年）
3. 选择利率更低的银行
4. 夫妻共同贷款（收入合并计算）
5. 减少其他负债（信用卡、花呗）

## 买房避坑

1. ❌ 只看月供，不看总利息
2. ❌ 贷款期限太短，月供压力大
3. ❌ 掏空六个钱包付首付，月供还不起
4. ❌ 忽略税费（契税、中介费、装修费）
5. ❌ 没留应急资金就买房
6. ❌ 不比较多家银行的利率和服务
