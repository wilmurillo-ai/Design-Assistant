# Startup Tools — Prompt Tips

## When User Asks for Startup Help

1. Identify the stage: idea / validation / MVP / launch / growth / scale
2. Run: `bash scripts/startup.sh <command> [stage] [industry]`
3. Present actionable guidance
4. Connect related tools (checklist → register → mvp → fundraise)

## Startup Stages

1. **Idea** — problem validation, market sizing, competitor analysis
2. **Validation** — customer interviews, landing page test, pre-sales
3. **MVP** — core features only, build-measure-learn cycle
4. **Launch** — go-to-market, PR, initial users
5. **Growth** — product-market fit, scaling channels
6. **Scale** — team building, process optimization, fundraising

## China-specific Considerations

- Company types: 有限责任公司, 股份有限公司, 个体工商户, 合伙企业
- Registration: 工商局, 税务登记, 银行开户, 社保公积金
- Compliance: 网络安全法, 个人信息保护法, ICP备案
- Funding: 天使轮, Pre-A, A轮, B轮...

## Key Metrics by Stage

- Pre-PMF: engagement, retention, NPS
- Post-PMF: CAC, LTV, MRR/ARR, churn rate
- Growth: burn rate, runway, unit economics
