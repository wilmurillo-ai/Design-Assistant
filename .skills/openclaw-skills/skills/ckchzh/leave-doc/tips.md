# 请假条使用技巧与最佳实践

## 📋 请假的核心原则

### 1. 提前规划
- 非紧急请假，至少提前3天申请
- 长假（5天+）建议提前2周申请
- 跟团队沟通好工作交接

### 2. 理由得体
不需要编造复杂理由，简单真实即可。
- ✅ "个人事务" — 万能理由，不用解释太多
- ✅ "身体不适" — 附上就医证明更好
- ❌ 不要写过于详细的私人原因

### 3. 交接到位
请假不可怕，工作没人管才可怕。
- 列出紧急事务和负责人
- 留好联系方式

## 🎯 使用场景速查

### 场景A：普通请假（1-2天）
```bash
# 生成标准请假条
doc.sh leave "事假" "1" "个人事务"
doc.sh leave "病假" "2" "身体不适需就医"
```

### 场景B：请长假（3天+）
```bash
# 自动计算工作日，排除周末
doc.sh multi-day "年假" "5" "出国旅行"

# 系统会自动：
# - 从明天开始计算
# - 跳过周末
# - 列出每天的日期和星期
# - 计算实际日历天数
```

### 场景C：突发情况
```bash
# 生成紧急请假模板 + 微信消息模板
doc.sh emergency

# 包含：
# - 第一时间通知的简短消息
# - 正式请假条（事后补）
# - 三种微信消息模板（直接复制使用）
```

### 场景D：规划年假
```bash
# 输入你的年假天数
doc.sh annual-plan "10"

# 输出：
# - 所有法定节假日概览
# - 拼假方案（按性价比排序）
# - 针对你的天数的分配建议
# - 注意事项和小技巧
```

## 📊 多天请假功能详解

`multi-day` 命令的特殊之处：
- **自动排除周末**：请5天假 = 5个工作日，周末不算
- **逐日列出**：清楚看到哪天请假、哪天是周末
- **自动计算返回日期**：知道什么时候回来上班
- **包含交接模板**：紧急联系人、进行中项目

示例输出：
```
工作日1: 03/11 周一  ← 请假
工作日2: 03/12 周二  ← 请假
工作日3: 03/13 周三  ← 请假
------: 03/14 周六  ← 周末休息
------: 03/15 周日  ← 周末休息
工作日4: 03/16 周一  ← 请假
工作日5: 03/17 周二  ← 请假
```

> ⚠️ 注意：法定节假日需要手动调整（调休日也算工作日）

## 🏖️ 年假规划功能详解

`annual-plan` 根据你的年假天数，推荐最优拼假方案：

### 5天年假
- 方案1：全部用在春节或国庆 → 拼10天+
- 方案2：春节2天 + 五一3天
- 方案3：三次小长假，各1-2天

### 10天年假
- 方案1：春节3天 + 五一3天 + 清明2天 + 机动2天
- 方案2：攒一个超长假期（7天年假+国庆 = 14天+）
- 方案3：每季度一次3天小假

### 15天年假
- 方案1：每季度一次5天假
- 方案2：春节5天 + 暑假5天 + 零散5天
- 方案3：一次超长假出国旅行

### 拼假原理
```
周五请1天 + 周末 = 3天假（性价比最高）
周一请1天 + 周末 = 3天假（同上）
周四五请2天 + 周末 = 4天假
周一二请2天 + 周末 = 4天假
```

## 🆘 紧急请假攻略

### 第一时间做什么
1. **通知领导**（30秒内能搞定的消息）
2. **交代最紧急的工作**（找人代班）
3. **保持手机畅通**

### 微信消息怎么发
`emergency` 命令提供三种模板：
- 病假模板：身体不适
- 家中急事模板：家人需要照顾
- 突发意外模板：紧急情况

直接复制修改即可，不用自己想措辞。

### 事后补什么
- 正式请假条
- 病假：医院证明/病历
- 事假：简要说明
- 如果请假超过预期：及时通知延长

## ⚖️ 请假法律知识

### 年假天数（法定）
| 工龄 | 年假天数 |
|------|---------|
| 1-10年 | 5天 |
| 10-20年 | 10天 |
| 20年以上 | 15天 |

### 其他假期（法定）
- 婚假：3天（各地可能有额外天数）
- 产假：98天 + 各地延长假
- 陪产假：各地不同，7-30天
- 丧假：直系亲属1-3天

### 常见问题
- Q: 试用期能请年假吗？
  A: 法律上可以，但很多公司规定转正后才能休

- Q: 年假没休完怎么办？
  A: 公司应支付300%日工资补偿

- Q: 病假扣工资吗？
  A: 病假工资不低于最低工资的80%

## 💡 小技巧

1. **周三请假法**：周三请1天，把一周变成两个"短周末"
2. **提前规划**：年初就把全年假期排好
3. **错峰休假**：避开同事的假期，更容易被批准
4. **留后手**：不要一次用完年假，留1-2天应急
