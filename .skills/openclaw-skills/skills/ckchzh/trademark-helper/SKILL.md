# Trademark Helper

商标注册助手。商标查询、分类选择、注册流程、驳回应对。Trademark registration helper with search, classification, process guide. 商标注册、品牌保护、知识产权。Use when registering trademarks.

## 常见问题

**Q: 这个工具适合谁用？**
A: 任何需要trademark helper的人，无论是个人还是企业用户。

**Q: 输出格式是什么？**
A: 主要输出Markdown格式，方便复制和编辑。

## 专业建议

- 商标分类: 45大类, 选对类别很重要**
- 查询: 中国商标网 sbj.cnipa.gov.cn 免费查**
- 申请流程: 查询→提交→形式审查(1月)→实质审查(6月)→公告(3月)**
- 费用: 官方300元/类(网上申请270元)**
- 驳回: 近似驳回最常见, 可以做驳回复审**
