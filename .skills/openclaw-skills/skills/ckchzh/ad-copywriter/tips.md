# 广告投放实战技巧 💡

## 一、ROI优化核心

### 1. ROI计算基础
```
GMV ROI = 广告带来的收入 ÷ 广告花费 × 100%
净利ROI = (收入×利润率 - 广告花费) ÷ 广告花费 × 100%
```
- GMV ROI > 200% 是基本线
- 净利ROI > 30% 才算真正赚钱
- 不要只看GMV，要看扣除成本后的真实利润

### 2. ROI提升公式
```
ROI = (客单价 × 转化率) ÷ CPC
```
提升ROI只有三条路：
- 降CPC（优化质量分/出价策略）
- 提转化率（优化落地页/文案/定向）
- 提客单价（组合装/满减/关联销售）

### 3. 预算分配法则
```
测试期:  20% 预算，多素材小金额测试
放量期:  60% 预算，集中投放已验证的素材
衰退期:  20% 预算，维持曝光+测试新素材
```

## 二、A/B测试方法论

### 4. A/B测试黄金法则
- **一次只测一个变量**（标题/图片/CTA/定向）
- 流量50/50随机分配
- 至少跑3-7天（覆盖工作日+周末）
- 每组≥1000次展示才有统计意义
- 置信度≥95%才判定胜出

### 5. 测试优先级
```
影响最大 → 先测
1. 标题/Hook（决定是否点击）
2. 主图/视频（决定第一印象）
3. 定向人群（决定流量质量）
4. CTA按钮（决定转化动作）
5. 落地页（决定最终转化）
```

### 6. 常见A/B测试维度
| 维度 | 版本A | 版本B | 测试目标 |
|------|-------|-------|---------|
| 标题句式 | 疑问句 | 陈述句 | CTR |
| 文案角度 | 痛点切入 | 利益切入 | CTR+CVR |
| 价格呈现 | 原价对比 | 日均价格 | CVR |
| 信任方式 | 数据证明 | 用户故事 | CVR |
| 紧迫感 | 限时 | 限量 | CVR |

### 7. 素材迭代策略
- 每周产出3-5组新素材
- 淘汰CTR低于平均值的素材
- 胜出素材提取核心元素复用
- 关注竞品素材变化，保持差异化
- 建立品牌素材库，记录所有测试结果

## 三、平台适配策略

### 8. 抖音/巨量引擎
- **用户特点**: 刷短视频，注意力极短
- **核心指标**: 完播率>35%，互动率>5%
- **文案风格**: 口语化、故事感、有情绪
- **前3秒**: 必须有Hook抓住注意力
- **投放建议**: 千川投流+DOU+组合，先测后放

### 9. 微信/朋友圈
- **用户特点**: 社交场景，信任度高
- **核心指标**: 点击率>1.5%，加粉率>3%
- **文案风格**: 像朋友分享，不像硬广
- **关键**: 第一句话决定是否展开阅读
- **投放建议**: 精准定向+小程序落地页

### 10. 百度搜索
- **用户特点**: 有明确搜索意图，转化最强
- **核心指标**: 点击率>5%，转化率>3%
- **文案风格**: 包含关键词，专业可信
- **关键**: 标题必须含搜索词，提高质量度
- **投放建议**: 核心词高价精准+长尾词低价广覆

### 11. Google Ads
- **用户特点**: 全球覆盖，适合跨境出海
- **核心指标**: CTR>3%, CVR>2%, QS≥7
- **文案风格**: 英文为主，简洁直接
- **关键**: Responsive Search Ads 10+ headlines
- **投放建议**: 先Exact Match测试，再Phrase扩量

### 12. 平台选择矩阵
| 需求 | 推荐平台 | 预算建议 |
|------|---------|---------|
| 品牌曝光 | 抖音+朋友圈 | ¥5000+/天 |
| 精准获客 | 百度+Google | ¥2000+/天 |
| 电商转化 | 抖音千川 | ¥1000+/天 |
| 私域引流 | 微信+朋友圈 | ¥3000+/天 |
| 本地获客 | 抖音本地+百度 | ¥500+/天 |

## 四、文案创作技巧

### 13. 广告文案AIDA法则
```
Attention (引起注意) → 标题/Hook
Interest  (产生兴趣) → 痛点/共鸣
Desire    (激发欲望) → 利益/场景
Action    (引导行动) → CTA/限时优惠
```

### 14. 高CTR标题公式
- 数字法: 「3个方法让你的XX提升200%」
- 对比法: 「别人花5000，我只花了500」
- 悬念法: 「用了这个之后，我再也没有...」
- 痛点法: 「为什么你的XX总是不好用？」
- 从众法: 「100万人都在用的XX」

### 15. CTA（行动号召）优化
```
❌ 弱CTA: 「了解更多」「点击查看」
✅ 强CTA: 「立即抢购，限时3折」「免费领取，仅限今日」
```
- CTA要有紧迫感（限时/限量）
- CTA要有利益点（折扣/赠品）
- CTA要有行动指引（立即/马上）
- 不同平台CTA风格不同

## 五、投放数据分析

### 16. 核心指标看板
```
曝光量 → 点击率(CTR) → 转化率(CVR) → 客单价(AOV)
         优化素材       优化落地页      优化SKU
                                        ↓
                    ROI = AOV × CVR ÷ CPC
```

### 17. 数据诊断清单
- CTR低：素材/标题不吸引人 → 换素材
- CTR高但CVR低：流量质量差或落地页差 → 优化定向/落地页
- CPC太高：竞争激烈或质量分低 → 优化质量分/换时段
- ROI波动大：季节性/竞品变化 → 监控竞品+调整策略
- 整体下滑：素材疲劳 → 更新素材+拓展新渠道
