# 📈 基金投资实用指南

## 基金入门

### 📌 什么是基金？
简单说: 你出钱，基金经理帮你买一篮子股票/债券。
- **门槛低** — 10元起投
- **分散风险** — 一只基金持有几十到几百只股票
- **专业管理** — 基金经理全职研究

### 📌 基金类型一览
| 类型 | 风险 | 预期年化 | 适合 |
|------|------|---------|------|
| 货币基金 | ⭐ | 1.5%~2.5% | 现金管理 |
| 纯债基金 | ⭐⭐ | 3%~5% | 稳健保本 |
| 混合债基 | ⭐⭐☆ | 4%~7% | 稳中求进 |
| 指数基金 | ⭐⭐⭐ | 6%~12% | 定投首选 |
| 混合基金 | ⭐⭐⭐☆ | 5%~15% | 有经验者 |
| 股票基金 | ⭐⭐⭐⭐ | 8%~20%+ | 高风险承受 |
| QDII | ⭐⭐⭐⭐ | 不定 | 全球配置 |

## 定投——最适合普通人的方式

### 📌 为什么定投？
1. **不用择时** — 不需要判断高低点
2. **摊低成本** — 高位少买，低位多买
3. **强制储蓄** — 每月自动扣款
4. **长期有效** — 时间越长，复利越猛

### 📌 定投的数学魔力
每月投3000元，年化10%:
- 5年: 投入18万 → 市值23.4万 (+30%)
- 10年: 投入36万 → 市值62.0万 (+72%)
- 20年: 投入72万 → 市值229.7万 (+219%)
- 30年: 投入108万 → 市值678.1万 (+528%)

**💡 复利是世界第八大奇迹 — 爱因斯坦**

### 📌 定投实操要点
1. **选什么** — 宽基指数基金（沪深300、中证500）
2. **投多少** — 月收入的10%~20%
3. **什么时候投** — 发薪日自动定投
4. **频率** — 月投和周投差别不大，月投更方便
5. **坚持** — 至少3年，最好5年以上

### 📌 定投止盈策略
- **目标收益率法**: 总收益达25%~30%就止盈一半
- **估值止盈法**: PE>历史80%分位时分批止盈
- **永不止损**: 定投越跌越该买，不要割肉

## 资产配置——鸡蛋别放一个篮子

### 📌 三种经典配置方案

**🛡️ 保守型（年化3%~5%）**
- 货币30% + 纯债40% + 指数20% + 黄金10%
- 适合: 3年内要用的钱、退休人群

**⚖️ 稳健型（年化6%~10%）**
- 货币10% + 债券25% + 指数35% + 行业15% + 海外15%
- 适合: 大多数人、3-5年的中长期投资

**🚀 激进型（年化10%~15%+）**
- 货币5% + 指数30% + 行业25% + 主动股基20% + 海外15% + 另类5%
- 适合: 年轻人、5年以上的长期投资

### 📌 配置原则
1. **100减年龄法则** — 100-你的年龄 = 股票类资产比例
2. **核心+卫星** — 70%放指数基金(核心)，30%放主题基金(卫星)
3. **不同市场分散** — A股+美股+港股+债券+黄金
4. **定期再平衡** — 每半年检查，偏离>5%就调整

## 再平衡——被忽视的超级武器

### 📌 什么是再平衡？
当某类资产因涨跌偏离了目标比例，通过买卖恢复到目标配置。
- 本质: **卖涨买跌**（逆人性操作）
- 效果: 长期可提升年化收益1%~2%

### 📌 再平衡实操
1. 设定目标配置（如股票40%、债券40%、货币20%）
2. 每半年检查一次实际比例
3. 偏离>5%就触发调整
4. 方法一: 卖出超配的，买入低配的
5. 方法二: 新增资金全部投入低配资产

### 📌 再平衡示例
目标: 股票50% + 债券50%，总计10万
- 年初: 股票5万 + 债券5万
- 年末: 股票7万(涨40%) + 债券5.2万(涨4%)
- 实际: 股票57% + 债券43%
- 操作: 卖出股票0.85万 → 买入债券0.85万
- 结果: 各6.1万，恢复50:50

## 基金投资避坑

### 📌 新手常犯错误
1. ❌ 追涨杀跌 — 看涨了才买，跌了就卖
2. ❌ 频繁交易 — 手续费吃掉收益
3. ❌ 只看短期收益 — 过去涨得好≠未来涨得好
4. ❌ 不分散 — all in单一行业基金
5. ❌ 借钱投资 — 杠杆 + 波动 = 灾难
6. ❌ 忽略费率 — 选C类短持、A类长持
7. ❌ 定投中断 — 坚持不下去=前功尽弃

### 📌 选基金看什么？
1. **成立时间** — 至少3年以上
2. **规模** — 2亿~100亿（太小清盘风险，太大船大难掉头）
3. **费率** — 管理费<1.5%，指数基金<0.5%
4. **基金经理** — 任职>3年，经历过熊市
5. **最大回撤** — 你能承受的最大亏损幅度
