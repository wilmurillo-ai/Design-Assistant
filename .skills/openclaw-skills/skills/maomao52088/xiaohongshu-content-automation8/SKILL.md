## 快速开始

### 选题挖掘流程

1. 确定领域关键词（如"AI工具"、"个人成长"）
2. 分析该领域爆款笔记共性
3. 提取标题公式、封面特征、内容结构
4. 生成10个选题 + 优先级排序
5. 验证可行性

### 爆款标题公式

- 数字开头：`"坚持30天，我发生了什么变化"`
- 对比反差：`"月薪3千vs月薪3万的人，区别在哪"`
- 痛点直击：`"如果你也...一定要看这篇"`
- 身份认同：`"写给所有...的人"`

### 文案模板示例

**干货型（高收藏）**：
```
【标题】数字+结果承诺

开头：痛点直击 OR 结果预告

正文：
1. 第一点（核心方法）
2. 第二点
3. 第三点

结尾：总结 + 互动提问
```
