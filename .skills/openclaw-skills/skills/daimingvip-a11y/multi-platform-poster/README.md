## 🎯 解决什么问题？

你写了一篇优质内容，想要发布到：
- 微信公众号
- 微博
- 知乎
- 小红书
- 抖音

**传统方式：**
1. 手动复制粘贴到每个平台
2. 调整格式适配各平台要求
3. 压缩/裁剪图片
4. 编写不同风格的标题和描述
5. 分别点击发布
6. 逐个平台查看数据

**耗时：2-3 小时**

**使用 Multi-Platform Poster：**
1. 输入一篇 Markdown 文章
2. 选择目标平台
3. 一键发布

**耗时：2 分钟** ⚡

## ✨ 核心功能

### 1. 智能内容适配

自动根据各平台规则调整内容：

| 平台 | 自动处理 |
|------|---------|
| 微信公众号 | Markdown → HTML 精美排版 |
| 微博 | 自动压缩到 140 字以内 |
| 知乎 | 保留完整长文格式 |
| 小红书 | 添加 emoji 和热门标签 |
| 抖音 | 生成短视频描述文案 |

### 2. 一键多平台发布

```
输入：一篇 3000 字文章
↓
自动处理:
• 生成 5 个平台适配版本
• 优化图片尺寸和格式
• 添加平台专属标签
• 设置最佳发布时间
↓
输出：5 个平台同时/定时发布
```

### 3. 智能发布时间

根据各平台用户活跃时间自动排期：

- **微信公众号**: 20:00-22:00 (晚间阅读高峰)
- **微博**: 12:00-13:00 (午休刷博)
- **知乎**: 09:00-10:00 (上班路上)
- **小红书**: 18:00-20:00 (下班放松)
- **抖音**: 20:00-22:00 (睡前刷视频)

### 4. 数据追踪面板

## 📋 配置步骤

### 第一步：授权平台账号

在 ClawHub 技能设置中绑定：

- [ ] 微信公众号 (需要 AppID + AppSecret)
- [ ] 微博开发者账号

输入你的 Markdown 文章，然后说：
```
发布到所有平台
```

## ❓ 常见问题

### Q: 需要每个平台都授权吗？
A: 不需要。可以只授权你使用的平台，未授权的平台会跳过。

### Q: 内容会被平台限流吗？
A: 不会。每个平台的内容都是独立适配的，符合各平台规则。

### Q: 支持视频发布吗？
A: 当前版本支持图文，视频发布功能在开发中 (v0.2.0)。

### Q: 可以撤回已发布内容吗？
A: 支持撤回最近 1 小时内发布的内容，超过 1 小时需要手动在各平台删除。

### Q: 数据追踪是实时的吗？
A: 数据每 15 分钟更新一次，可在设置中调整频率。
