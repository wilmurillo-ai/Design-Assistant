# WeChat Social Automation 💕

> 微信公众号 + 朋友圈自动化管理技能

## 🚀 快速开始

### 1. 安装技能

```bash
clawhub install wechat-social-automation
```

### 2. 配置微信公众号

在 `TOOLS.md` 文件中添加你的公众号配置:

```markdown
### 微信公众号

- AppID: wx1234567890abcdef
- AppSecret: your-app-secret-here
- 公众号名称：你的公众号
```

### 3. 开始使用

```
发布一篇公众号文章，标题"如何月入过万"
```

## ✨ 功能特性

### 📝 文章发布
- 一键发布文章到公众号
- 支持草稿箱管理
- 多图文消息支持
- 富文本编辑

### ⏰ 定时推送
- 设置定时推送时间
- 自动发布到公众号
- 支持重复推送计划

### 📊 数据分析
- 文章阅读量统计
- 粉丝增长趋势
- 互动率分析
- 最佳发布时间建议

### 👥 粉丝管理
- 粉丝标签管理
- 自动回复配置
- 关键词触发
- 用户画像分析
