# 客户对接准备工作清单

## 📋 对接前准备

### 资料准备
- [ ] 营业执照复印件
- [ ] 组织机构代码证
- [ ] 法人身份证复印件
- [ ] 经办人身份证复印件及授权委托书

### 技术准备
- [ ] 确定对接负责人和技术联系人
- [ ] 准备开发环境（服务器、网络等）
- [ ] 确认开发人员具备相关技术能力
- [ ] 准备测试环境和生产环境

---

## 📝 第 1 步：确定接口

**需要填写的文档**：
- [ ] 外部系统接入确认单
- [ ] 接口需求确认表

**需要确认的信息**：
- [ ] 需要使用的接口列表
- [ ] 联机交易类型
- [ ] 业务场景说明

---

## 🌐 第 2-3 步：网络信息

**需要提供的信息**：
- [ ] 企业网络类型（专线/互联网）
- [ ] 出口 IP 地址
- [ ] 服务器 IP 地址
- [ ] 服务器端口
- [ ] 访问协议（HTTP/HTTPS/TCP）
- [ ] 加密方式要求

---

## 🔧 第 6 步：联调测试

**测试前准备**：
- [ ] 开发环境准备就绪
- [ ] 接口程序开发完成
- [ ] 自测通过

**测试中工作**：
- [ ] 按测试用例逐项测试
- [ ] 记录测试结果
- [ ] 反馈测试问题
- [ ] 配合问题排查

**测试完成标志**：
- [ ] 所有测试用例通过
- [ ] 测试报告签署确认

---

## 🚀 第 7-8 步：上线

**上线前**：
- [ ] 提交上线申请
- [ ] 确认上线窗口时间
- [ ] 生产环境配置完成
- [ ] 回滚方案准备

**上线后**：
- [ ] 小额交易验证
- [ ] 功能完整性验证
- [ ] 签署上线确认单
- [ ] 进入运维阶段

---

## 📞 联系方式

**技术支持**：
- 对接过程中遇到任何问题，请及时联系银行技术支持团队

**紧急联系**：
- 上线期间如遇紧急情况，请拨打紧急联系电话

---

> 💡 **提示**：建议在对接开始前指定专人负责对接工作，保持与银行的沟通畅通。
