# CPS赚钱

目标与原则：
- 目标：在90天内用一个主渠道跑通稳定的CPS收入闭环
- 原则：真实评价、不夸大承诺、遵守平台与广告法、可量化与可复制

平台与账号：
- 联盟平台：阿里妈妈（淘宝客）、京东联盟、抖音精选联盟、拼多多多多进宝
- 分发平台：抖音/快手/小红书/视频号/B站/微信公众号/知乎/独立站/社群
- 账号矩阵：1个主号 + 2–3个辅助号，定位清晰，风格一致

选品方法：
- 维度：高转化（痛点明确）+ 高佣金 + 高搜索/讨论热度 + 低售后风险
- 数据源：联盟后台榜单、站内热搜、社群反馈、竞品内容表现
- 测试：每次并行测试3–5款，观察点击率、转化率、EPC与退款率

内容矩阵：
- 种草内容：场景化对比/避坑指南/清单推荐/真实测评
- 转化内容：开箱演示/使用教程/结果展示/用户口碑集合
- 长尾内容：关键词问答/细分场景/购买前后注意事项

获取链接与埋点：
- 申请商品与佣金计划，生成专属CPS链接或口令
- 使用短链与UTM参数区分渠道与内容（source/campaign/content）
- 保留跳转页与独立推荐页两套方案（对比转化与风控）

转化页面与文案：
- 着陆结构：痛点引入 → 方案与亮点 → 真实证据 → 风险与边界 → 明确行动
- 视觉与信任：清晰图文/视频、资质与评价、交付与售后承诺
- 行动引导：按钮与引导语统一，重复出现但不过度打扰

流量获取与投放：
- 免费流量：SEO/平台搜索/社群口碑/互推合作
- 付费流量：信息流小预算测试，跑通后再加码；素材两两对照
- 达人协作：分佣合作或内容共创，确保追踪与结算规则明确

数据追踪与指标：
- 关键指标：点击率CTR、转化率CVR、EPC、GMV、佣金率、退款率、LTV/CAC
- 数据看板：按渠道与内容维度对比，周复盘并更新黑白名单
- 结算核对：联盟后台与自有追踪交叉验证，异常单独复核

标准SOP（单商品）：
- 01 选品与申请：筛选标准与佣金确认
- 02 素材与脚本：输出3个方向的文案/视频版本
- 03 链接与埋点：生成专属链接与UTM参数，短链化
- 04 分发与投放：主渠道上线，辅渠道矩阵转发；小预算付费测试
- 05 复盘与迭代：记录CTR/CVR/EPC，淘汰差素材，放大优质组合
- 06 售后与口碑：收集评价与问题，完善常见问答并优化文案

30日执行计划：
- 第1周：搭建账号与看板；选3–5款商品，完成脚本与链接
- 第2周：主渠道发布并跑数；整理黑白名单与优化素材
- 第3周：扩展到第二渠道；尝试小预算付费；建立FAQ与转化页
- 第4周：总结复盘，固定每周选品与内容节奏；制定90日目标

工具清单：
- 内容：剪映/CapCut、飞书/Notion、Canva/Figma
- 追踪：UTM参数、短链、联盟后台报表、表格看板
- 协作：群管理工具、素材库、模板库

合规与风险：
- 不夸大效果，不虚构体验；明确适用边界与售后说明
- 遵守平台政策；避免违规词与夸大承诺；保护用户隐私
- 版权与授权：素材来源合法，合作达成书面约定

常见坑与优化：
- 只看佣金不看退货 → 关注退款率与真实口碑
- 只做单平台 → 建立矩阵与长尾内容，提升稳定性
- 无数据归因 → 全量使用UTM/短链并复盘，形成模板化

扩展与升级：
- 横向：同品类拓展与清单化，打造系列内容与站点
- 纵向：自有转化页/独立站，沉淀邮件与社群，提升复购与会员化
