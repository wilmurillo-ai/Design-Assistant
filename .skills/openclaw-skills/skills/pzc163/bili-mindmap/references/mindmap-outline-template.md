- 视频概览
  - 链接 / BV 号
  - UP 主 / 频道
  - 发布时间
  - 主题一句话总结
- 核心内容
  - 核心观点 1
    - 论据 / 解释
    - 例子 / 对比
  - 核心观点 2
    - 论据 / 解释
  - 核心观点 3
    - 关键结论
- 关键细节
  - 方法 / 步骤
  - 数据 / 案例
  - 术语 / 概念
- 评论反馈
  - 高价值补充
  - 主要争议
  - 常见疑问
- 总结 / 行动项
  - 适合记住的结论
  - 可执行建议
  - 后续可延伸的问题

## 整理规则

- 先依据字幕或 ASR 全文组织主干，再用 AI 总结和热门评论补充。
- 用短语而不是长句，避免把整段口语原样搬进导图。
- 把相近观点合并到同一分支，避免碎片化节点。
- 对来源不确定的内容单独标记为“评论补充”或“AI 总结补充”。
