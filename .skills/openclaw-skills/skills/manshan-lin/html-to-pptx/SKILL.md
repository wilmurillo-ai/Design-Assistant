# HTML to PPTX Converter

## Description
Convert beautiful HTML presentation slides generated by LLMs into PPTX format for easy sharing and wide distribution.

## Capabilities
- Convert HTML slides into PPTX
- Preserve layout and styling
- Support multi-slide HTML structures

## Input
- HTML content (string or file)

## Output
- PPTX file

## Usage
Provide HTML content, and the tool will return a downloadable PPTX file.

## Tags
presentation, pptx, html, conversion, slides
