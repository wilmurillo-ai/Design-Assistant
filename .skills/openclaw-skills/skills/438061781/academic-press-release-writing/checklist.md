# 学术新闻稿质量检查清单
---

## 撰写前检查
- [ ] 核心需求确认：发布渠道、目标读者、宣传重点
- [ ] 发布时间确认：是否有时间要求，是否允许公开
- [ ] 素材收集完整：论文/成果介绍、团队资料、数据素材
- [ ] 关键信息核实：成果名称、期刊、发表时间、核心数据

## 内容检查
- [ ] 五段式结构完整：导语、成果详情、应用价值、团队介绍、展望
- [ ] 导语包含5W要素：Who/What/When/Where/Why
- [ ] 所有数据准确：与原始素材核对一致
- [ ] 人名/单位名准确：无错别字，写法符合官方规范
- [ ] 无夸大宣传：不使用"世界第一"、"全球首创"等绝对化表述
- [ ] 无涉密内容：符合单位宣传规范
- [ ] 语言通俗易懂：避免过多专业术语，必要时做解释
- [ ] 逻辑流畅：段落衔接自然，无生硬跳转
- [ ] 字数符合要求：800-1200字，不超过1500字

## 格式检查
- [ ] 标题符合规范：20-30字，醒目有吸引力
- [ ] 排版规范：宋体小四号、1.5倍行距、段首缩进2字符
- [ ] 配图清晰：每张图配有说明文字，位置合适
- [ ] 页眉页脚正确：包含单位logo、发布日期等必要元素
- [ ] 联系方式完整（如果需要）：联系人、邮箱、电话

## PDF导出检查
- [ ] 排版与Word一致：无错位、无乱码
- [ ] 图片清晰：分辨率足够，无模糊
- [ ] 页码正确：连续无缺失
- [ ] 链接可点击：如果有链接，测试是否能正常打开
- [ ] 文件大小合适：不超过10MB，便于发送
- [ ] 文件名规范：`[单位]_[成果名称]_[发布日期].pdf`

## 最终确认
- [ ] 通读全文2次以上，无错别字、不通顺语句
- [ ] 他人复核：至少1名同事/团队成员审核通过
- [ ] 需求方确认：符合所有要求，同意发布
