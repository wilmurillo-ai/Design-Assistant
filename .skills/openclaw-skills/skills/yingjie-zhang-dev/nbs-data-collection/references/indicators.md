# 指标说明

## 核心指标

### 1. 国内生产总值（GDP）

#### 定义
一定时期内一国生产的全部最终产品和服务的市场价值。

#### 分类
- **名义GDP**：按当年价格计算
- **实际GDP**：按不变价格计算（用于去除价格变动影响）

#### 数据频率
- 季度（1992年Q1至今）
- 年度（1952年至今）

#### 数据来源优先级
1. 国家统计局季度初步核算结果
2. 统计年鉴（最终核实数）
3. data.stats.gov.cn数据库

#### 注意事项
- 季度GDP数据通常在季度结束后15天左右发布
- 年度最终核实数在次年1月发布
- 初步核算数与最终核实数可能有差异

---

### 2. GDP增速

#### 定义
GDP的同比增长率，反映经济增长速度。

#### 计算公式
```
GDP增速(%) = (本期GDP / 上年同期GDP - 1) × 100%
```

#### 类型
- **同比增速**：与上年同期相比
- **环比增速**：与上季度相比
- **累计增速**：与上年全年累计相比

#### 数据频率
季度同比为主

---

### 3. 产出缺口

#### 定义
实际GDP与潜在GDP之差，反映经济运行状态。

#### 计算公式
```
产出缺口(亿元) = 实际GDP - 潜在GDP
产出缺口率(%) = (实际GDP - 潜在GDP) / 潜在GDP × 100%
```

#### 潜在GDP估算方法
**HP滤波法**（Hodrick-Prescott Filter）

```python
def hp_filter(y, lamb=1600):
    """
    分离趋势成分（潜在GDP）和周期成分（产出缺口）
    lamb: 平滑参数
        - 季度数据: λ=1600
        - 年度数据: λ=100
        - 月度数据: λ=14400
    """
    from scipy.signal import diff_press
    import numpy as np
    
    n = len(y)
    I = np.eye(n)
    D = np.diff(I, n=2)
    tau = np.linalg.solve(I + lamb * D.T @ D, y)
    cycle = y - tau
    return tau, cycle
```

#### 经济含义
| 产出缺口率 | 经济状态 |
|-----------|---------|
| > 0% | 经济偏热，存在通胀压力 |
| ≈ 0% | 经济正常运行 |
| < 0% | 经济偏冷，存在通缩压力 |

#### 典型历史验证
| 时期 | 预期产出缺口率 | 实际合理范围 |
|------|---------------|-------------|
| 2009Q1 | 金融危机冲击 | -15% ~ -10% |
| 2020Q1 | COVID-19冲击 | -20% ~ -15% |

### 4. 人均GDP

#### 定义
GDP除以人口，反映人均产出水平。

#### 计算公式
```
人均GDP(元/季度) = GDP(亿元) × 1亿 / 季度平均人口(万)
                 = GDP(亿元) × 10000 / 季度平均人口(万)
```

#### 数据频率
季度（需要季度人口数据，可用年除以4估算）

#### 注意
- 人口数据频率通常为年度，季度人口需估算
- 估算方法：季度人口 ≈ 年末人口（近似）

---

### 5. 居民消费价格指数（CPI）

#### 定义
反映居民家庭购买的消费品和服务价格水平变动的指标。

#### 分类
- **CPI**：居民消费价格指数
- **核心CPI**：不包括食品和能源的CPI

#### 数据频率
月度（1951年至今）

#### 转换为季度均值
```python
quarterly_cpi = monthly_cpi.resample('QE').mean()
```

#### 同比计算
```
CPI同比(%) = (本期CPI / 上年同期CPI - 1) × 100
```
注意：CPI基期已调整为上年同月=100

---

### 6. 生产者出厂价格指数（PPI）

#### 定义
反映工业企业产品出厂价格变动的指标。

#### 分类
- 生产资料PPI
- 生活资料PPI

#### 数据频率
月度（1996年至今）

#### 转换为季度均值
同CPI

---

### 7. GDP平减指数

#### 定义
名义GDP与实际GDP之比，反映整体价格水平变动。

#### 计算公式
```
GDP平减指数 = 名义GDP / 实际GDP × 100
```

#### 估算方法（由于无法直接获取）
⚠️ **注意**：当无法获取GDP平减指数时，可用CPI季度链式指数估算：
```
GDP平减指数 ≈ CPI季度均值链式指数
```
需标注为"估算值"

## 派生指标关系图

```
实际GDP
    │
    ├── 人均GDP = GDP ÷ 人口
    │
    └── 产出缺口 = HP滤波(实际GDP)
                       │
                       └── 潜在GDP = 趋势成分

CPI/PPI月度 → 季度均值 → 同比增速

GDP平减指数 ≈ CPI链式指数（估算）
```
