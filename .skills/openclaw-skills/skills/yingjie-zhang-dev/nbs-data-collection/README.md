# 国家统计局数据采集 (nbs-data-collection)

## 简介

面向中国宏观经济研究的数据采集工具，专注于从国家统计局获取GDP、CPI、PPI等核心经济指标。

## 功能

- 📊 **数据采集**：自动从国家统计局获取季度GDP、月度CPI/PPI等指标
- 🔗 **来源追溯**：自动为每个数据点添加可验证的来源链接（统计年鉴/统计公报）
- 📈 **衍生计算**：自动计算产出缺口（HP滤波）、人均GDP、GDP平减指数等
- 📉 **可视化**：自动为时间序列数据添加折线图
- ✅ **质量核验**：内置数据抽查和链接验证机制

## 适用场景

- 宏观经济研究
- GDP/CPI/PPI指标追踪
- 产出缺口分析
- 经济周期研究

## 数据指标

| 指标 | 频率 | 说明 |
|------|------|------|
| 实际GDP | 季度 | 1992年至今 |
| GDP增速 | 季度 | 同比/累计同比 |
| 产出缺口 | 季度 | HP滤波(λ=1600) |
| 人均GDP | 季度 | 派生计算 |
| CPI | 月度/季度 | 1951年至今 |
| PPI | 月度/季度 | 1996年至今 |
| GDP平减指数 | 年度 | ⚠️估算 |

## 数据来源

- 国家统计局统计年鉴（历史数据）
- 国家统计局统计公报（最新数据）
- data.stats.gov.cn数据平台

## 输出格式

Excel文件（.xlsx），包含：
- 多个Sheet（每个指标一个）
- 数据来源列（可点击链接）
- 折线图（趋势可视化）
- 数据说明Sheet（计算方法、修订记录）

## 使用要求

- Python 3.10+
- openpyxl（用于Excel操作）
