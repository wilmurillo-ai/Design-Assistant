# MM Score 评分函数详细说明

## 指标 1: PnL/Vol 比率 (权重 30%)

```
ratio = abs(PnL) / Volume

< 0.1%  → 100 分 (几乎确定是做市商)
< 0.5%  → 80 分
< 1%    → 60 分
< 3%    → 40 分
< 5%    → 20 分
≥ 5%    → 0 分  (不像做市商)
```

## 指标 2: 决策频率 (权重 25%)

使用"头寸数/天"而非"交易笔数/天"。

```
decisions_per_day = total_positions / active_days

> 50  → 100 分
> 20  → 80 分
> 10  → 60 分
> 5   → 40 分
> 2   → 20 分
≤ 2   → 0 分
```

## 指标 3: 买卖平衡度 (权重 20%)

```
balance = min(buy_count, sell_count) / max(buy_count, sell_count)

> 0.9  → 100 分 (几乎完美平衡 = 做市商)
> 0.7  → 70 分
> 0.5  → 40 分
> 0.3  → 20 分
≤ 0.3  → 0 分  (明显单边 = 方向性交易者)
```

## 指标 4: 平均持仓时间 (权重 15%)

```
avg_hours = median(holding_hours)

< 1h   → 100 分 (分钟级持仓 = 做市商)
< 4h   → 80 分
< 12h  → 60 分
< 24h  → 40 分
< 72h  → 20 分
≥ 72h  → 0 分  (持仓超过 3 天 = 不是做市商)
```

## 指标 5: 金额均匀度 (权重 10%)

```
cv = std(trade_amounts) / mean(trade_amounts)

< 0.3  → 100 分 (金额非常均匀 = 做市商)
< 0.5  → 70 分
< 0.8  → 40 分
< 1.2  → 20 分
≥ 1.2  → 0 分  (金额波动大 = 不是做市商)
```

## 总分计算

```
MM Score = pnl_vol × 0.30 + frequency × 0.25 + balance × 0.20 + holding × 0.15 + uniformity × 0.10
```

## HFT 检测 (独立于 MM Score)

满足以下 3 条件中的 2 条 → HFT:

1. 决策频率 > 10 头寸/天
2. 中位数持仓时间 < 4 小时
3. 中位数单笔金额 < $50

## 做市商 vs 方向性交易者对比

| 维度 | 做市商 | 方向性交易者 |
|------|--------|------------|
| 目标 | 赚价差 | 赚方向 |
| 持仓方向 | 中性（买卖平衡） | 单边（看多或看空） |
| 持仓时间 | 极短（秒~分钟） | 较长（小时~天~周） |
| 决策频率 | 极高（每天 50+ 头寸） | 较低（每天 < 5 头寸） |
| PnL/Vol | 极低（< 1%） | 较高（> 5%） |
| 跟单价值 | ❌ 无法跟单 | ✅ 可以跟单 |
