# 中国历史年份查询技能

## 功能概述

将用户输入的年份转换为多种纪年方式，并提供该年中国与世界的重要历史事件。

## 支持的输入格式

1. **公元纪年**：如 1234、公元前221、2024
2. **朝代年号纪年**：如 绍定六年、康熙元年、开元天宝
3. **干支纪年**：如 甲子年、乙丑年、壬寅年

## 输出格式

提供以下信息：
- 公元纪年
- 中国朝代年号纪年（含朝代名称）
- 干支纪年
- 中国重要历史事件
- 世界重要历史事件

## 使用流程

### 1. 解析输入年份

- 公元年份：直接使用
- 年号年份：查询 references/dynasty_data.md 转换为公元年份
- 干支年份：使用天干地支公式计算公元年份

### 2. 干支纪年计算公式

天干：甲(0)、乙(1)、丙(2)、丁(3)、戊(4)、己(5)、庚(6)、辛(7)、壬(8)、癸(9)
地支：子(0)、丑(1)、寅(2)、卯(3)、辰(4)、巳(5)、午(6)、未(7)、申(8)、酉(9)、戌(10)、亥(11)

公元年份计算：
- 天干：(公元年份 - 4) % 10
- 地支：(公元年份 - 4) % 12

反向计算（干支转公元）：
- 甲子年对应公元4年，此后每60年一循环
- 可用公式：公元年份 = 60 * n + 4 + 天干值 - 地支值（需调试）

### 3. 查询历史事件

使用 web_fetch 工具从百度获取历史事件：
- 中文查询：`https://www.baidu.com/s?wd=XXXX年+历史事件`
- 或直接搜索百度百科词条：`https://zh.wikipedia.org/wiki/XXXX年`

示例：
```
https://www.baidu.com/s?wd=1234年+历史事件
```

### 4. 输出格式化

按以下格式整理输出：
```
## 📅 年份信息

- 公元纪年：XXXX年
- 朝代年号：X朝 X帝 X年
- 干支纪年：X年

## 🇨🇳 中国历史事件

1. 事件1
2. 事件2
...

## 🌍 世界历史事件

1. 事件1
2. 事件2
...
```

## 新功能：皇帝年号查询

### 功能说明
当用户发送中国古代皇帝名字时，查询并展示：
1. 该皇帝在位期间使用的全部年号
2. 每个年号对应的公元纪年
3. 每个年号对应的干支纪年
4. 皇帝的功绩与历史地位概括

### 使用方式
1. 解析用户输入中的皇帝名字
2. 在 references/emperor_data.md 中查找匹配的皇帝
3. 列出其所有年号及对应年份
4. 搜索并概括皇帝的历史功绩

### 输出格式
```
## 👑 皇帝信息

- 皇帝姓名：XXX
- 朝代：XXX
- 在位时间：XXX-XXX年

## 📅 年号列表

| 年号 | 公元年份 | 干支纪年 |
|------|----------|----------|
| 元年 | XXXX年 | 甲子年 |
| X年 | XXXX年 | 乙丑年 |
...

## 📖 功绩与历史地位

简要概括...
```

### 皇帝数据来源
- references/emperor_data.md 内置主要皇帝年表数据
- 可通过网络搜索补充查询

## 新功能：历史事件查询

### 功能说明
当用户发送中国历史重要事件名称时，查询并展示：
1. 该事件发生的**公元纪年**
2. 该事件发生的**年号纪年**
3. 该事件发生的**干支纪年**
4. 事件的**简要介绍**

### 使用方式
1. 解析用户输入中的事件名称
2. 在 references/events_data.md 中查找匹配的事件
3. 列出事件的年份信息
4. 通过网络搜索补充事件详情

### 输出格式
```
## 📜 历史事件

- 事件名称：XXX
- 公元纪年：XXXX年
- 年号纪年：XXX朝 XXX年
- 干支纪年：XXX年

### 事件简介

事件详细介绍...
```

### 事件数据来源
- references/events_data.md 内置重要历史事件数据
- 可通过网络搜索补充查询

### 支持的事件类型
- 政治变革：商鞅变法、戊戌变法、靖难之役等
- 军事战争：赤壁之战、淝水之战、鸦片战争等
- 外交出使：张骞出使西域、郑和下西洋等
- 都城变迁：盘庚迁殷、迁都北京等
- 宫廷政变：玄武门之变、神龙政变等
- 条约签订：辛丑条约、马关条约等

## 注意事项

- 公元前纪年：公元前221年 = -220（计算时用负数）
- 朝代年号需先确定朝代和皇帝，再查具体年份
- 搜索历史事件时，优先使用中文关键词
- 对于模糊输入，尽量给出最可能的结果
