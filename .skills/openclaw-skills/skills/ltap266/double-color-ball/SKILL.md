# 双色球查询技能

当用户输入"双色球"关键词时，自动返回近一个月双色球开奖号码、走势分析及下期预测推荐。

## 触发条件

- 关键词：双色球、双色球开奖、双色球结果

## 执行方式

```bash
python C:\Users\duyun\.openclaw\workspace\skills\double-color-ball\scripts\lottery.py
```

## 功能

1. **开奖号码查询**：获取近一个月双色球开奖结果
2. **走势分析**：红球分区、连号、重号、热号等统计
3. **预测推荐**：娱乐性质地推荐2注下期号码

## 数据来源

- 内置预存数据（每月更新）

## 使用示例

```
用户：双色球
罗汉：<运行脚本返回开奖号码+走势+预测>
```

## 注意事项

- 预测推荐纯属娱乐，无科学依据
- 购彩需理性，请勿沉迷
