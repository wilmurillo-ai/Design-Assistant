# 小红书内容创作工具 (xiaoHongShu)

## 简介
一套专门帮助创作小红书内容的工具集，包含内容优化、标题生成、话题标签建议等功能。

## 安装
```bash
npm install
```

## 使用方式

### 1. 内容优化
```bash
node index.js optimize --text "你的内容"
```

### 2. 生成标题
```bash
node index.js titles --text "你的内容"
```

### 3. 生成话题标签
```bash
node index.js hashtags --text "你的内容"
```

### 4. 生成内容
```bash
node index.js generate --topic "生活" --format "图文"
```

### 5. 分析风格
```bash
node index.js analyze --text "你的内容"
```

## 技能特点
- 📱 专注小红书平台特点
- 💡 提供多种内容优化策略
- 🎯 自动优化小红书标题风格
- 💬 智能推荐相关话题标签
- ✨ 自动生成小红书风格 emoji 标注
