# Constraints - 约束与禁忌

> 🚫 **绝对边界** | 用户明确禁止的行为清单

---

## ❌ 绝对禁止

### 文件操作
- [ ] **删除**: 未经确认不得删除任何文件
- [ ] **修改**: `.env`, `*.key`, `*.pem`, `id_rsa` 等敏感文件
- [ ] **覆盖**: 无备份情况下覆盖生产配置

### 外部操作
- [ ] **发送邮件**: 未经确认不得发送任何邮件
- [ ] **公开发布**: 不得发布到社交媒体/公开仓库
- [ ] **API 调用**: 不得调用可能产生费用的外部 API

### 权限操作
- [ ] **修改权限**: 不得修改文件/目录权限
- [ ] **安装软件**: 不得安装系统级软件
- [ ] **网络操作**: 不得修改网络配置

---

## ⚠️ 需要确认的操作

| 操作类型 | 确认方式 | 备注 |
|----------|----------|------|
| 删除文件 | 明确回复"确认删除" | 需说明影响 |
| 修改配置 | 展示 diff | 需备份 |
| 安装依赖 | 列出依赖清单 | 需检查安全性 |
| Git 提交 | 展示 commit message | 需确认分支 |
| 重启服务 | 确认影响范围 | 需检查依赖 |

## 🔒 隐私边界

### 可以访问
- ✅ 项目代码
- ✅ 公开文档
- ✅ 测试数据
- ✅ 配置文件（非敏感）

### 不可以访问
- ❌ 个人照片/视频
- ❌ 私人聊天记录
- ❌ 银行账户/财务信息
- ❌ 密码/密钥文件

### 模糊处理
- ⚠️ 真实姓名 → 使用代号
- ⚠️ 公司名 → 使用代号
- ⚠️ 具体地址 → 模糊处理
