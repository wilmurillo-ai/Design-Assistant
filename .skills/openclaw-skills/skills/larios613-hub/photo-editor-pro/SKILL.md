# Photo Editor Pro — ClawHub Skill

## Overview

**Photo Editor Pro** is a comprehensive image editing skill that combines AI-powered enhancements with professional editing workflows.

### What It Does

| Capability | Description |
|------------|-------------|
| **AI Enhancement** | Auto-enhance photos with AI |
| **Background Removal** | Remove backgrounds with one click |
| **Color Correction** | Professional color grading |
| **Batch Processing** | Edit multiple images at once |
| **Format Conversion** | Convert between PNG, JPG, WebP, etc. |
| **Resize & Crop** | Smart resizing and cropping |

---

## Features

### Basic Editing

- Crop and rotate
- Resize with aspect ratio lock
- Brightness, contrast, saturation
- Sharpness and clarity
- Noise reduction

### AI-Powered

- Auto-enhance (one-click improvement)
- Background removal
- Object removal
- Face retouching
- Sky replacement
- Style transfer

### Professional Tools

- Color grading (curves, levels)
- Selective adjustments
- Layers and masks
- Text and watermarks
- Batch processing

### Export Options

- PNG (lossless)
- JPG (adjustable quality)
- WebP (web optimized)
- Resize presets (social media, web, print)
