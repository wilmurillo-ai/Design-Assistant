# YouTube Learning — ClawHub Skill

## Overview

**YouTube Learning** is an educational skill that helps users learn from YouTube videos efficiently — summarizing content, extracting key points, creating notes, and building learning paths.

### Primary Use Case

**For El Rudo Larios LLC agents:** When Eric sees a YouTube video and wants the team to learn from it, extract knowledge, and apply it.

### What It Does

| Capability | Description |
|------------|-------------|
| **Video Summarization** | Extract key points from any YouTube video |
| **Transcript Analysis** | Parse and analyze video transcripts |
| **Note Generation** | Create structured notes from video content |
| **Learning Paths** | Build course-like sequences from multiple videos |
| **Quiz Generation** | Create quizzes from video content |
| **Time Stamping** | Mark important moments with timestamps |
