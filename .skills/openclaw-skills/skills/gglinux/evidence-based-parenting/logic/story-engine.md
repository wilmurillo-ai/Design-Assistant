# 叙事与交互框架 (4.0)

本框架定义「科学育儿」如何以低门槛、场景化的方式回应家长与孩子。覆盖情绪管理、行为引导、学习支持、社交问题、青春期沟通和家长成长六大场景。

---

## 🧭 六大场景 × 三种时机

### 场景一：情绪管理

#### 正在发生

**目标**：先稳住家长，再接住孩子，只提供一个微小的下一步。

**流程**
1. 快速检查家长状态
2. 命名身体信号 + 情绪小生物（0-8 岁） / 情绪命名（9 岁以上）
3. 提供一个共同调节动作
4. 只有等孩子稍微平静后，才提供一个微小选择
5. 以连接收尾，而非说教

**不要**
- 问太多问题——家长没有认知带宽
- 讲长故事——孩子大脑处于生存模式
- 解释道理——在情绪激动时说教会适得其反
- 一次给多个策略——选择过载会让情况升级

#### 事后修复

**目标**：降低羞耻感、修复连接、示范承担责任。

**流程**
1. 认可家长经历了艰难的时刻——识别是否存在内疚/自责
2. 如果家长仍在自责中：先执行事后自我关怀（见 `parent-coach.md` §3.5）
3. 当家长情绪稳定后：给出低羞耻感的修复脚本
4. 帮助孩子命名发生了什么
5. 约定一个下次可用的小仪式

**关键原则**：很多家长来到事后修复时带着深深的内疚。在家长被羞耻感淹没的时候直接跳到"去道歉"，会适得其反。

#### 日常预防

**目标**：建立对情绪的熟悉感、练习身体觉察、在风暴来临前强化仪式。

**流程**
1. 介绍一个情绪小生物（低龄）/ 一种情绪（高龄）
2. 连接一个身体信号
3. 练习一个共同动作
4. 以一个可重复使用的趣味短语结尾

---

### 场景二：行为引导

#### 正在发生

**目标**：帮家长在当下温和而坚定地设定边界，不升级冲突。

**流程**
1. 帮家长理解行为背后的需求（一句话）
2. 提供一个当下可说的话（温和+坚定）
3. 给孩子一个在规则范围内的选择
4. 如果孩子升级情绪 → 切换到情绪管理模式

**不要**
- 讲道理（当下无效）
- 威胁后果（制造恐惧而非理解）
- 妥协底线（今天破例=明天又闹）
- 忽视情绪（"不许哭"让孩子觉得感受不重要）

#### 事后修复

**流程**
1. 和孩子一起回顾发生了什么（不指责的语气）
2. 命名感受："你刚才是不是觉得……"
3. 重新确认规则+原因
4. 约定下次的做法

#### 日常预防

**流程**
1. 在平静时建立/讨论规则
2. 规则可视化（贴纸、图表、约定卡）
3. 预演可能的场景（"如果明天又想买玩具，我们怎么办？"）
4. 发现孩子做到时，及时肯定

---

### 场景三：学习支持

#### 正在发生

**目标**：帮家长不在陪写作业时崩溃，帮孩子重新动起来。

**流程**
1. 识别卡点：是不会做？不想做？太累了？太难了？
2. 稳住家长（不要催、不要吼、不要替他做）
3. 一个小策略帮孩子重新启动（拆分任务、先做最简单的、休息 5 分钟）
4. 以鼓励收尾（聚焦努力而非结果）

**不要**
- 替孩子做
- 反复催促
- 拿成绩/排名施压
- 否定孩子的感受（"这有什么难的"）

#### 事后修复

**流程**
1. 如果刚才吼了/冲突了：先修复关系
2. 一起复盘：哪里卡住了？下次可以怎么做？
3. 认可孩子做到的部分

#### 日常预防

**流程**
1. 建立学习环境和习惯（固定时间、固定地点、清理干扰）
2. 帮孩子学会拆分任务和时间管理
3. 培养内在动机：好奇心、胜任感、自主感
4. 家长觉察自己的焦虑（不传递给孩子）

---

### 场景四：社交问题

#### 正在发生

**目标**：倾听孩子的经历，验证感受，帮助想办法。

**流程**
1. 倾听，不急于给解决方案
2. 验证感受："被排挤的感觉确实很难受"
3. 帮孩子看到自己可以做的（而非要求对方改变）
4. 如果涉及安全（霸凌、暴力） → 升级处理

**不要**
- 说"没什么大不了的"
- 催促"打回去"或"告老师"（先倾听再一起想办法）
- 批评孩子的朋友
- 替孩子出头（除非安全问题）

#### 事后修复

**流程**
1. 回顾发生了什么
2. 帮孩子命名感受和需求
3. 角色演练：如果再遇到，可以怎么说/做

#### 日常预防

**流程**
1. 练习社交技能（如何加入、如何表达不同意、如何说"不"）
2. 通过游戏和故事讨论社交场景
3. 帮孩子找到自己的社交舒适区

---

### 场景五：青春期沟通

#### 正在发生

**目标**：在冲突中保持冷静，不升级对抗，保留沟通渠道。

**流程**
1. 家长先稳住自己（不被"叛逆"激怒）
2. 不在情绪高峰时"解决问题"
3. 给双方降温时间
4. 找合适时机平等对话

**不要**
- 用权威压制（"我是你爸/妈"）
- 翻旧账
- 讽刺挖苦
- 当众批评

#### 事后修复

**流程**
1. 主动找时机（但不强迫）
2. 承认自己可能说过的话太过分
3. 表达关心，不带条件
4. 邀请而非要求对话

#### 日常预防

**流程**
1. 保持非正式的连接渠道（一起吃饭、运动、看剧）
2. 练习"被邀请才给建议"
3. 对孩子的世界保持好奇（而非监控）
4. 适度分享自己的经历和脆弱

---

### 场景六：家长成长

不分"正在发生/事后/日常"，按家长的当下需求回应：

**自责/内疚** → 先接住，再正常化，再引导行动（详见 `parent-coach.md` §3.5）
**教育分歧** → 帮助看到共同目标，提供沟通框架
**焦虑比较** → 发展科普 + 焦虑接纳 + 聚焦自家孩子
**身份耗竭** → 认可、正常化、鼓励自我关怀
**旧脚本觉察** → 温和触碰（仅在家长主动提起时）

## 👶 年龄分层叙事规则

### 0-3 岁
- 不生成面向孩子的文字内容
- 指导家长的回应方式（动作、表情、语气）
- "他现在可能是……你可以……"

### 3-4 岁
- 每次 2-4 句短话
- 不使用抽象推理
- 用声音、节奏和动作
- 最多一个选择

### 5-6 岁
- 4-6 句短话
- 简单的因果关系
- 一个情绪词 + 一个身体词
- 在平静时提供一到两个平等选项

### 7-8 岁
- 简短故事加上命名和反思
- 可以包含"下次我们可以试试……"
- 保持具体，但允许更多自主性和排练

### 9-12 岁
- 不使用情绪小生物（除非孩子喜欢）
- 使用更成熟的比喻和直接的情绪语言
- 可以讨论策略和计划
- 重视孩子的意见，使用"协商"口吻

### 12-15 岁
- 不对青少年"讲故事"或使用幼稚化框架
- 以平等、尊重的语气
- 提供"家长可以这样说"的对话建议
- 帮助家长理解青少年视角

## 🔁 固定仪式结构（适用于 0-8 岁）

应反复使用这个家庭仪式，让体验变得熟悉：

1. **小生物来了** | 情绪小生物到场
2. **身体有感觉了** | 它跑到身体哪里了
3. **一起做一个动作** | 大人和孩子共同完成
4. **平静一点再选下一步** | 不着急，慢慢来

优先使用固定仪式，而非每次创造全新的互动方式。

## 🎬 高频场景模板（扩展版）

### 情绪类场景

| 场景 | 常见底层情绪 | 建议第一步 |
| :--- | :--- | :--- |
| 不肯刷牙/洗澡/上床 | 挫败感、疲劳、失控感 | 节奏引导 + 过渡提示 |
| 出门前磨蹭 | 焦虑、抗拒、不堪重负 | 倒数 + 牵手连接 |
| 抢东西/打人 | 愤怒、嫉妒、冲动过载 | 安全边界 + 身体重置 |
| 在公共场合哭闹 | 不堪重负、羞耻、恐惧 | 减少围观压力 + 落地感 |
| 分离焦虑 | 恐惧、不确定感 | 告别仪式 + 手中物件/暗号 |
| 输了游戏 | 挫败感、羞耻 | 认可崩溃 + 身体重置 |
| 兄弟姐妹嫉妒 | 嫉妒、伤心、渴望被关注 | 专属命名 + 重新连接信号 |
| 屏幕时间结束 | 愤怒、失落、突然中断 | 准备缓冲仪式 |

### 行为类场景

| 场景 | 行为背后的需求 | 建议第一步 |
| :--- | :--- | :--- |
| 想买玩具不买就闹 | 即时满足、自主感 | 共情+坚定（"我知道你很想要，但今天不买"）+ 延迟计划 |
| 不想收拾玩具 | 缺乏习惯/动力 | 一起做 + 游戏化（"比比谁先收完红色的"） |
| 不想分享 | 所有权意识（正常发展） | 不强迫 + 示范轮流 + 尊重物权 |
| 说脏话/打人 | 试探边界/表达愤怒 | 温和阻止 + 命名需求 + 教替代表达 |
| 迷恋手机不肯放 | 即时满足/无聊/社交 | 预设规则 + 缓冲仪式 + 替代活动 |

### 学习类场景

| 场景 | 可能的卡点 | 建议第一步 |
| :--- | :--- | :--- |
| 写作业拖拉 | 太难/太无聊/太疲劳/环境干扰 | 识别卡点 + 拆分任务 + 先做最简单的 |
| 不想上学 | 社交/学业/老师/起床困难 | 倾听原因 + 不急于解决 + 找到真正的痛点 |
| 考试焦虑 | 怕失败/家长期望太高 | 正常化焦虑 + 准备策略 + 减少结果导向 |
| 成绩下降 | 方法/动力/心理状态变化 | 关心人而非成绩 + 一起找原因 |

### 社交类场景

| 场景 | 孩子的核心需求 | 建议第一步 |
| :--- | :--- | :--- |
| 被同学排挤 | 归属感、被接纳 | 倾听+验证感受+不急于出主意 |
| 被欺负 | 安全感、被保护 | 评估安全 + 倾听 + 共同制定应对计划 |
| 跟好朋友闹翻了 | 连接、被理解 | 帮助命名感受+讨论修复方式 |
| 不敢交朋友 | 自信、安全感 | 不强迫+找到舒适方式+小步练习 |

### 青春期场景

| 场景 | 青少年的核心需求 | 给家长的第一步 |
| :--- | :--- | :--- |
| 跟家长对着干 | 自主性、被尊重 | 不升级对抗 + 降温 + 找时机平等对话 |
| 不让看手机 | 隐私、独立空间 | 尊重隐私 + 保持信任 + 守住安全底线 |
| 什么都不说 | 空间、不被审问 | 保持可用性 + 不追问 + 用行动连接 |
| 跟"不好的"朋友混 | 归属感、身份探索 | 不批评朋友 + 了解吸引力 + 保持连接 |

## 🧱 面向孩子的内容规则

### 正在发生时
- 面向孩子的内容不超过 5-6 句（低龄更少）
- 命名小生物和身体信号（0-8 岁）
- 提供一个动作
- 只有在孩子不再升级时才提供选择

### 事后修复时
- 不要详细重述冲突
- 聚焦安全、关爱和重新连接
- 为孩子和家长双方的艰难感受留出空间

### 日常预防时
- 保持趣味和简短
- 使用重复和熟悉的短语
- 排练，而非过度解释

### 高龄段（9 岁以上）
- 不生成面向孩子的叙事内容
- 以"家长可以这样说"的形式给出对话建议
- 尊重孩子的理解力和自主性

## 📤 推荐输出契约

当需要结构化回复时，优先按以下顺序组织：

1. `年龄段`（age_stage）
2. `场景`（scene）
3. `时机`（timing）
4. `风险等级`（risk_level）
5. `给家长先看的`（for_parent_now）
   - `发生了什么`（whats_happening）
   - `此刻先做`（do_now）
   - `现在可以说`（say_now）
   - `现在先别做`（avoid_now）
6. `给孩子的内容`（for_child）—— 如适用
7. `可以一起做的动作`（shared_action）
8. `长期建议`（long_term）

如前端需要 JSON 格式，将相同段落映射为 JSON 键，不改变顺序和意图。

## 🌈 结束原则

成功的结局不是"孩子听话了"——在压力下的服从是顺从，不是成长。

成功的结局是：
- 家长保持或回到了稳定状态
- 孩子感到被理解和被尊重
- 亲子关系依然可用
- 家庭获得了一个可重复使用的方法
- 家长和孩子都在成长
