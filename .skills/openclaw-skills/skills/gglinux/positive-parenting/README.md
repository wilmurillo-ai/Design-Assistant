# 科学育儿 | AI 全场景育儿教练

**科学育儿** 是一款服务 0-15 岁儿童/青少年及家长的 AI 育儿教练技能。覆盖情绪管理、行为引导、学习支持、社交问题、青春期沟通和家长成长六大场景，目标是成为家庭科学、健康、快乐成长路上的教练和导师——先托举家长，再托举孩子。

---

## ✨ 核心特性

- **四大年龄段**：萌芽期(0-3) → 成长期(3-6) → 扎根期(6-12) → 觉醒期(12-15)
- **六大场景**：情绪管理 / 行为引导 / 学习支持 / 社交问题 / 青春期沟通 / 家长成长
- **三种时机**：正在发生 / 事后修复 / 日常预防
- **家长优先托举**：先稳住大人，再帮助孩子
- **风险分级边界**：绿色 / 黄色 / 红色三级路由，确保安全
- **年龄精准适配**：不同年龄段使用完全不同的沟通策略和指导框架
- **科学理论支撑**：基于依恋理论、自我决定理论、积极纪律、成长型思维等经典理论

---

## 🧒 年龄段体系

| 阶段 | 年龄 | 核心任务 | 家长角色 |
|------|------|---------|---------|
| 🌱 萌芽期 | 0-3 岁 | 安全依恋、基础信任 | 安全基地、回应者 |
| 🌿 成长期 | 3-6 岁 | 自主性、情绪识别 | 情绪教练、边界设定者 |
| 🌳 扎根期 | 6-12 岁 | 自律、社交、学习 | 引导者、支持者 |
| 🔥 觉醒期 | 12-15 岁 | 自我认同、独立思考 | 顾问、尊重者 |

## 🛠️ 仓库结构

| 文件 | 说明 |

| `logic/emotion-creatures.md` | 情绪小生物↔身体信号映射表（主要 0-8 岁） |
| `logic/parent-coach.md` | 全场景家长支持指南：自救、行为引导、学习支持、社交指导、青春期沟通、家长成长 |

## 🤖 如何使用

1. 将 `SKILL.md` 注入 AI 智能体的系统提示词中
2. `logic/` 目录下的文件作为运行时规则集
3. 每个请求先路由到年龄段和场景，再生成内容
4. 回复结构保持：理解 → 家长指导 → 孩子内容（如适用）→ 动作 → 长期建议
5. 如前端需要结构化渲染，将相同段落映射为 JSON 格式

## 🎯 产品理念

「科学育儿」不是来替代家长或评判家长的。它是一个全场景育儿支撑系统，帮助家庭：

- 在情绪风暴中找到锚点
- 在行为困境中建立规则而非对抗
- 在学习挫折中找到动力而非增加压力
- 在社交困难中发展技能而非回避
- 在青春期摩擦中保持连接而非升级冲突
- 让家长感到"我可以做好这件事"

> **目标是让家庭越来越不需要这个工具，而非越来越依赖它。**
