# 年龄段分层路由引擎

本文件定义不同年龄段的发展特征、沟通策略和场景处理差异。每次请求都应参考本文件确定年龄段后加载对应内容。

---

## 🌱 萌芽期（0-3 岁）

### 发展特征
- **认知**：前语言期或语言萌芽期，靠感官和动作认识世界
- **情绪**：情绪直接通过身体表达（哭、扭、黏、推），无法用语言命名
- **社会性**：以依恋关系为核心，开始探索但需要"安全基地"
- **自主性**：1.5-3 岁出现"我要自己来"的意志，容易产生冲突

### 核心发展任务
- 建立安全依恋
- 信任世界是安全的
- 学会基本的自我安抚（需要大量帮助）

### 家长角色
**安全基地 + 回应者**
- 这个阶段家长是孩子情绪调节的"外部大脑"
- 回应的及时性和一致性比回应的"正确性"更重要
- 不需要"教"孩子情绪管理，只需要反复"接住"

### 沟通策略
- **面向家长**为主，几乎不直接面向孩子输出文字内容
- 指导家长如何回应（动作、表情、语气、节奏）
- 使用简单的旁白帮家长理解孩子的状态："他现在可能是……"
- 不使用情绪小生物框架（太抽象），使用简单的身体描述

### 高频场景处理

| 场景 | 处理要点 |
|------|---------|
| 睡眠问题 | 建立固定入睡仪式、安全物件、渐进式调整 |
| 分离焦虑 | 告别仪式、过渡物件、缩短而非突然消失 |
| "不"的爆发期 | 这是自主性发展的标志，不是"不听话"；给有限选择 |
| 打人/咬人 | 前语言表达方式，非恶意；温和阻止+示范替代行为 |
| 吃饭问题 | 不强迫、不哄骗、信任身体信号 |

## 🌿 成长期（3-6 岁）

### 发展特征
- **认知**：想象力爆发、开始理解简单因果、但逻辑思维有限
- **情绪**：开始能命名基础情绪，但调节能力很弱，情绪风暴频繁
- **社会性**：开始社会化（幼儿园）、学习分享和轮流、出现友谊萌芽
- **自主性**：强烈的"自己做"欲望，但能力有限，容易挫败

### 核心发展任务
- 发展自主性和主动性
- 学会基本的情绪识别和命名
- 开始理解和内化基础规则
- 社会化启蒙

### 家长角色
**情绪教练 + 边界设定者**
- 帮助孩子命名情绪（"你现在感觉好生气"）
- 温和而坚定地设定边界（"你可以生气，但不能打人"）
- 用选择替代命令（"你想先刷牙还是先选故事？"）
- 用故事和游戏代替说教

### 沟通策略
- **情绪小生物框架**完全适用
- 使用短句、具体描述、感官词汇
- 选择不超过 2 个
- 用"做什么"替代"不做什么"（"慢慢走"替代"别跑"）
- 故事和游戏是最有效的沟通媒介

### 年龄细分

#### 3-4 岁
- 每次 2-4 句短话
- 不使用抽象推理
- 用声音、节奏和动作
- 最多一个选择
- 小生物只用最简单的形象

#### 5-6 岁
- 4-6 句短话
- 简单的因果关系可以使用
- 一个情绪词 + 一个身体词
- 平静后提供一到两个平等选项
- 开始建立"身体信号 → 动作"的连接

### 高频场景处理

| 场景 | 处理要点 |
|------|---------|
| 情绪风暴 | 情绪小生物 + 身体信号 + 共调节动作 |
| 不刷牙/洗澡/睡觉 | 这是自主性需求，不是故意找茬；规则+选择+过渡仪式 |
| 入园适应 | 分离仪式、过渡物件、接纳情绪表达 |
| 手机/屏幕 | 建立清晰规则 + 缓冲仪式 + 替代活动 |
| 不分享 | 发展阶段正常、不强迫、示范轮流 |
| 二胎矛盾 | 维护大孩归属感、避免"让着弟弟/妹妹" |

## 🌳 扎根期（6-12 岁）

### 发展特征
- **认知**：逻辑思维发展、能理解规则和因果、开始具备反思能力
- **情绪**：情绪词汇扩大、开始自我调节、但在压力下仍会退行
- **社会性**：友谊变得重要、开始比较、在意同伴评价
- **自主性**：需要更多自主空间和决策权、开始形成自我概念

### 核心发展任务
- 发展勤奋感和胜任感
- 建立自律能力
- 学会社交技能
- 规则从"外在约束"转为"内在认同"

### 家长角色
**引导者 + 支持者**
- 从"替孩子做"转向"陪孩子做"再到"看孩子做"
- 提供结构和方法，而非代劳
- 允许犯错，帮助从错误中学习
- 关注过程和努力，而非只看结果

### 沟通策略
- 情绪小生物仍可在 6-8 岁使用，但需要更成熟
- 9-12 岁使用更直接的情绪语言和比喻
- 可以使用简单的反思对话（"你觉得是什么让你特别烦？"）
- 给予更多自主空间（"你觉得怎么安排比较好？"）
- 尊重隐私需求的萌芽

### 年龄细分

#### 6-8 岁
- 情绪小生物仍可使用，但加入反思元素
- 简短故事加上命名和反思
- 可以包含"下次我们可以试试……"
- 保持具体，但允许更多自主性

#### 9-12 岁
- 不再使用情绪小生物（除非孩子喜欢）
- 使用更成熟的比喻和类比
- 可以讨论策略和计划
- 重视孩子的意见和想法
- 开始使用"协商"而非"指导"

### 高频场景处理

| 场景 | 处理要点 |
|------|---------|
| 作业拖拉 | 拆分任务、番茄钟、环境管理、找到卡点而非催促 |
| 考试焦虑 | 正常化焦虑、准备策略、减少结果导向的评价 |
| 厌学 | 找到根本原因（社交？难度？意义感？）、保持好奇而非焦虑 |
| 游戏/手机上瘾 | 协商规则、理解需求（社交？成就感？逃避？）、替代满足 |
| 被欺负 | 倾听+验证感受、不催促"打回去"或"告老师"、一起想办法 |
| 交不到朋友 | 社交技能练习、找到舒适的社交方式、不强迫 |
| 兄弟姐妹冲突 | 不当裁判、教冲突解决技能、各自需求都看见 |

## 🔥 觉醒期（12-15 岁）

### 发展特征
- **认知**：抽象思维发展、开始质疑权威和规则、理想主义萌芽
- **情绪**：情绪波动加剧（激素+身份探索）、更加内化和隐藏
- **社会性**：同伴关系成为核心、开始形成独立的价值观和身份认同
- **自主性**：强烈的独立需求、隐私意识增强、亲子权力关系重组

### 核心发展任务
- 建立自我认同（"我是谁"）
- 发展独立思考能力
- 学会在同伴压力中保持自我
- 从家庭为中心转向社会化

### 家长角色
**顾问 + 尊重者**
- 从"管理者"彻底转为"顾问"——被邀请时才给建议
- 倾听多于说教，好奇多于担心
- 尊重隐私，保持可用性
- 坚持底线（安全），但放开其他控制

### 沟通策略
- **绝对不使用**情绪小生物或任何幼稚化框架
- 以平等、尊重的语气对话
- 使用开放式问题而非引导性提问
- 分享而非指导（"我以前也遇到过类似的……"）
- 主动给空间（"你不想说也可以，我在这里"）
- 承认自己的局限（"这个我也不确定，我们一起想想？"）

### ⚠️ 特殊注意
- 这个年龄段很多"问题行为"实际上是正常的发展需求
- "叛逆"往往是自主性发展的表现
- 强行管控通常导致关系破裂或地下行为
- **黄色信号识别极其重要**：持续情绪低落、社交退缩、自我伤害迹象需要及时引导就医

### 高频场景处理

| 场景 | 处理要点 |
|------|---------|
| "叛逆"/顶嘴 | 正常化（这是自主性发展）、不升级对抗、找时机平等对话 |
| 手机/网络依赖 | 协商而非强制、理解社交需求、共同制定规则 |
| 不愿沟通 | 尊重空间、保持可用性、用行动而非话语连接 |
| 同伴压力 | 不批评朋友、帮助建立自我价值、练习说"不"的技能 |
| 隐私边界 | 尊重隐私是信任的基础、不翻手机/日记、除非有安全红线 |
| 早恋 | 不惊慌、不偷窥、开放对话、重点是安全和尊重 |
| 情绪低落 | 识别黄色信号、不说"你有什么好烦的"、必要时引导就医 |
| 学业压力 | 关注心理健康优先于成绩、减少比较、帮助找到意义感 |

## 🔀 年龄-场景交叉指导

同一场景在不同年龄段的处理差异巨大。以下是关键交叉点：

### "不听话" / 规则冲突

| 年龄段 | 本质 | 方法 |
|--------|------|------|
| 0-3 岁 | 自主性萌芽，不是"不听话" | 分散注意力、有限选择、温和转移 |
| 3-6 岁 | 自主性 vs 规则的初始碰撞 | 情绪接纳 + 温和坚定的边界 + 选择 |
| 6-12 岁 | 规则意义感和公平感的诉求 | 解释原因 + 协商 + 自然后果 |
| 12-15 岁 | 独立性和权力关系的重组 | 平等对话 + 协商规则 + 底线清晰 |

### 手机/屏幕

| 年龄段 | 本质 | 方法 |
|--------|------|------|
| 0-3 岁 | 家长行为引导为主 | 减少家长自身在孩子面前的手机使用 |
| 3-6 岁 | 缺少替代活动 + 即时满足 | 清晰规则 + 缓冲仪式 + 丰富活动 |
| 6-12 岁 | 成就感/社交/逃避多种需求 | 协商时间 + 理解需求 + 替代满足 |
| 12-15 岁 | 社交核心工具 + 身份表达 | 协商而非控制 + 数字素养 + 信任 |

### 社交困难

| 年龄段 | 本质 | 方法 |
|--------|------|------|
| 0-3 岁 | 还在平行游戏阶段，无需担心 | 提供社交接触机会即可 |
| 3-6 岁 | 社会化启蒙，冲突是学习 | 旁观引导、示范社交语言 |
| 6-12 岁 | 友谊重要性上升，被排挤很痛 | 倾听+验证+社交技能练习 |
| 12-15 岁 | 同伴关系是核心，家长影响力下降 | 不批评朋友、做安全港、练习边界 |

---

## 使用规则

1. 每次请求先确定年龄段，再选择对应的沟通策略和处理方式
2. 如果用户没有说明年龄，从描述中推断（如"幼儿园"→ 3-6 岁，"初中"→ 12-15 岁）
3. 如果无法推断，简短询问"孩子大概几岁？"
4. 边界年龄（如 3 岁、6 岁、12 岁）根据具体描述灵活判断，不机械切分
5. 同一个家庭的多个孩子可能处于不同阶段，需要分别考虑
