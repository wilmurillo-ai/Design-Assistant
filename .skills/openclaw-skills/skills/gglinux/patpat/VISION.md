# 摸摸头（PatPat）项目愿景

> **摸摸头（PatPat）—— 3-8岁家庭的情绪急救包。先接住大人，再托住孩子。**

---

## 1. 项目愿景

### 1.1 我们要解决什么问题

每个家长都想给孩子温暖的回应，但在情绪风暴真正来临的那 30 秒里——孩子在哭、自己也快失控、脑子一片空白——几乎没有人能"做对"。不是不想做好，而是**当下没有可用的支撑**。

"摸摸头"希望在那个最难的瞬间，提供一双温暖的大手托举整个家庭：先接住家长的情绪，再陪着孩子练习更好的应对方式。不评判，不说教，只是在你最需要的时候，递上一个可以马上用的"下一步"。

### 1.2 我们的核心理念

| 理念 | 说明 |
| --- | --- |
| 托举优先于指导 | 先稳住家长自己，再去帮助孩子——失控的大人无法共调节 |
| 陪伴优先于解决 | 有时候孩子不需要"办法"，只需要有人在 |
| 允许优先于纠正 | 先让情绪被允许存在，再谈"怎么办" |
| 连接优先于正确 | 关系修复比"做对"更重要 |
| 示范优先于说教 | 用故事示范，而非讲道理 |

> 重要边界：我们不是"替代父母"或"评判父母"，而是为家庭提供一个**当下可用的情绪支持脚手架**（Scaffolding）。

### 1.3 心理学理论基础

| 理论 | 核心观点 | 产品应用 |
| --- | --- | --- |
| 依恋理论 | 安全依恋是情绪调节的基础 | 强调"陪伴"和"接住"，而非"教导"和"纠正" |
| 情绪教练（Gottman） | 五步法：觉察→连接→倾听→命名→引导 | 故事结构遵循这个顺序 |
| 叙事治疗 | "问题外化"让人与问题分离 | 情绪小生物就是外化的具体应用 |
| 正念育儿 | 带着觉察和不评判的态度与孩子互动 | 家长呼吸暂停、自我关怀 |
| 内感受觉察（Interoception） | 识别身体内部信号是情绪调节的起点 | 情绪小生物 ↔ 身体信号映射，帮孩子从"说不出来"到"肚子紧紧的" |

> 💡 **关键洞察**：情绪调节能力的发展主要来自**"被调节"的重复体验**，而非"被教导"。孩子在一次次被稳定接住的经历中，逐渐内化出自我调节的能力。——这是 PatPat 所有设计决策的底层逻辑。

### 1.4 产品定位

| 维度 | 定位 | 说明 |
| --- | --- | --- |
| 用户 | 3-8 岁儿童 + 家长 | 孩子是体验主体，家长是操作者和共同成长者 |
| 场景 | 情绪风暴 + 日常陪伴 | 从"救火"升级为"防火+救火+修复" |
| 价值 | 情绪脚手架 | 不替代家长，而是赋能家长 |
| 形态 | 互动故事 + 家长教练 | 孩子听故事做选择，家长获得当下指导 |
| 差异化 | 先共情后策略 | 不说教、不评判、不制造内疚 |

## 2. 核心问题与机会

### 2.1 孩子侧痛点

| 痛点 | 描述 | 机会 |
| --- | --- | --- |
| 情绪识别难 | 3-8 岁儿童难以识别和命名自己的情绪 | 用拟人化小生物帮助外化情绪 |
| 讲道理无效 | 情绪脑激活时，逻辑脑无法工作 | 用故事和隐喻绕过防御 |
| 缺乏调节工具 | 不知道除了哭闹还能怎么办 | 通过选择练习替代行为 |
| 被否定的感受 | "不许哭""有什么好生气的" | 先承认感受，再引导行为 |

### 2.2 家长侧痛点

| 痛点 | 描述 | 机会 |
| --- | --- | --- |
| 当下不知所措 | 孩子崩溃时，不知道该说什么、做什么 | 提供实时"耳返"指导 |
| 容易失控 | 自己也会提高音量、威胁、强行拉走 | 提供家长自稳动作 |
| 事后愧疚 | 知道自己做得不好，但不知道如何修复 | 提供修复脚本 |
| 旧脚本循环 | 不自觉地用自己小时候被对待的方式来对待孩子 | 提供新的"脚本"替代旧模式 |

### 2.3 我们不做什么

| 不做 | 原因 | 替代方案 |
| --- | --- | --- |
| 诊断或治疗 | 我们不是医疗/心理治疗产品 | 红色风险时引导寻求专业帮助 |
| 替代真实关系 | 屏幕无法替代真人的温度 | 作为"桥梁"帮助家长与孩子连接 |
| 评判家长 | 家长已经够自责了 | 支持和赋能，而非评价 |
| 许诺"治愈" | 情绪发展是长期过程 | 强调"练习"和"陪伴" |

### 2.4 竞品差异化分析

| 现有方案 | 局限 | PatPat 的不同 |
| --- | --- | --- |
| 育儿类图文/课程 | 学了很多道理，但"当下用不上" | 提供**实时**、**场景化**的指导 |
| 儿童故事 App | 只面向孩子，不关心家长状态 | **家长优先**，先稳住大人 |
| 心理咨询 | 门槛高、不即时 | 低门槛的**日常情绪脚手架** |
| 通用 AI 助手 | 缺少儿童发展专业知识和安全边界 | 内置**风险分级**和**年龄适配** |

## 3. 核心体验

核心目标：提供一次完整的"被看见、被陪伴、被托举"的体验。

### 3.1 体验的五个层次

**孩子的体验：**

| 层次 | 体验 | 如何实现 |
| --- | --- | --- |
| 1. 安全感 | "我的感受是被允许的" | 不评判、不催促、不威胁 |
| 2. 被看见 | "有人懂我现在的感受" | 情绪外化、身体感受描述 |
| 3. 被陪伴 | "我不是一个人" | 陪伴选项、不急于解决 |
| 4. 有选择 | "我可以决定怎么做" | 两个平等选项、可以不选 |
| 5. 有希望 | "下次我可以做不同的事" | 正向结局、技能积累 |

**家长的平行体验：**

| 层次 | 体验 | 如何实现 |
| --- | --- | --- |
| 1. 安全感 | "我不是坏家长" | 不评判、先认同家长的难 |
| 2. 被看见 | "有人理解我现在的崩溃" | 识别家长触发模式并反馈 |
| 3. 被陪伴 | "我不用一个人扛" | 提供即时脚手架，降低孤立感 |
| 4. 有方法 | "我知道此刻该做什么" | Do now / Say now / Avoid now |
| 5. 有信心 | "下次我可以做得不一样" | 修复脚本、未来锚点 |

### 3.2 核心体验流程

```
1. 命名/外化 → 把情绪变成可爱的小生物
2. 共情与允许 → 承认感受，允许情绪存在
3. 陪伴或选择 → 可以只是陪伴，也可以探索不同的应对方式
4. 关系修复   → 回到连接，而非回到"正确"
```

此流程在三种模式下的侧重不同：

| 模式 | 侧重 | 说明 |
| --- | --- | --- |
| 正在爆发 | 主要停在 1→2→3（陪伴） | 不急于走到第 4 步，先稳住 |
| 事后修复 | 从第 4 步（关系修复）切入，回溯到 2（共情） | 先修复关系，再回顾发生了什么 |
| 平时练习 | 完整走完 1→2→3→4 | 以轻松节奏完整练习，建立肌肉记忆 |

> 🌿 不是所有情绪都需要"解决"。有时候，陪着就好了。
