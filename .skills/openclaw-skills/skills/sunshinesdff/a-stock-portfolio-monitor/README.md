# A股持仓监控助手

> 每日自动持仓报告，止损止盈提醒，让炒股更轻松

## 📦 功能介绍

- ✅ **持仓实时监控** - 自动获取最新股价，计算盈亏
- ✅ **每日收盘报告** - 生成日终持仓总结  
- ✅ **止损止盈提醒** - 自动检测触发条件
- ✅ **智能选股** - 基于涨幅/量能/估值多因子评分

## 🚀 快速开始

### 1. 安装

```bash
# 克隆到本地
git clone <技能仓库地址>
cd a-stock-portfolio-monitor

# 安装依赖（Python 3.8+）
pip install -r requirements.txt
```

### 2. 配置持仓

编辑 `data/portfolio.json`，添加你的持仓：

```json
{
  "600803": {"name": "新奥股份", "cost": 21.76, "qty": 1800},
  "600010": {"name": "包钢股份", "cost": 3.30, "qty": 15000},
  "600845": {"name": "宝信软件", "cost": 23.92, "qty": 3000}
}
```

### 3. 运行

```bash
python scripts/run_monitor.py
```

## ⚙️ 高级配置

### 定时任务（Windows）

创建 `monitor.bat`：

```batch
@echo off
cd /d C:\path\to\a-stock-portfolio-monitor
python scripts/portfolio.py analyze
```

使用任务计划程序，每天15:05运行。

### 选股参数调整

编辑 `scripts/selector.py`，修改评分阈值：

```python
# 选股阈值（默认>=50分）
MIN_SCORE = 50

# 股票池（可自定义）
STOCK_POOL = ['002594', '300750', ...]
```

## 📈 评分逻辑

| 维度 | 权重 | 说明 |
|------|------|------|
| 涨幅 | 0-30分 | 涨停30分，大涨25分 |
| 成交额 | 0-25分 | 10亿25分，5亿20分 |
| 强势程度 | 0-20分 | 接近最高价20分 |
| 估值 | 0-15分 | PE<30给15分 |
| 流动性 | 0-10分 | 成交量大给分 |
