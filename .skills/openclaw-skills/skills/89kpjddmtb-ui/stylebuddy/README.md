# StyleBuddy - AI穿搭闺蜜 🌸

**OpenClaw 生态首个生活场景 Skill**

你的专属 AI 穿搭顾问，像闺蜜一样懂你的风格。

## 🎯 一句话介绍

> 拍照管理衣橱 + 智能推荐搭配 + 逛街种草咨询 = 你的 AI 穿搭闺蜜

## 🌟 核心功能

| 功能 | 说明 | 示例 |
|-----|------|------|
| 📸 **衣橱录入** | 拍照自动识别衣服信息 | "录入这件衬衫" |
| 👗 **穿搭推荐** | 根据天气/场合智能推荐 | "今天穿什么？" |
| 🛍️ **种草咨询** | 逛街实时参谋 | "这件适合我吗？" |
| 📊 **衣橱分析** | 洞察衣橱配置，科学购物 | "帮我分析衣橱" |

## 💡 为什么选 StyleBuddy？

- **自然交互** — 像聊天一样，无需记指令
- **多模态支持** — 能看懂图片，也能纯文字使用
- **智能降级** — 无论你的 OpenClaw 配置如何，都能正常使用
- **越用越懂你** — 用户画像系统记录你的偏好

## 🚀 快速开始

安装后，直接说：

```
今天穿什么？
```

StyleBuddy 就会根据今天的天气和你的衣橱，推荐一套搭配。
