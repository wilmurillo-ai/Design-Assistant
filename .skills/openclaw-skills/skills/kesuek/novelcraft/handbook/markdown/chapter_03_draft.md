# Chapter 3: The Workflow Step-by-Step

## The 6 Phases at a Glance

| Phase | Module | Duration | Order |
|-------|--------|----------|-------|
| 1 | Concept | 1-2h | First |
| 2 | Writer Extras (Prolog) | 30-60min | Before chapters |
| 3 | Images | 40-55min | **Parallel** to phase 4 |
| 4 | Chapters | 4-8h | After concept |
| 5 | Writer Extras (Epilog) | 30-60min | After chapters |
| 6 | Publication | 15-30min | Last |

## Phase 1: Concept

This is where the idea is born:
- Define genre & theme
- Develop 3-5 main characters
- Plot structure (3-act model)
- Worldbuilding (setting, rules, history)
- Plan flashbacks

**Output:** Concept document in `00-concept/`

## Phase 2 & 5: Writer Extras (Optional)

- **Prolog:** Before chapters — background, world, setup
- **Epilog:** After chapters — outlook, resolution

Both are optional and configurable via `module-writer-extras.md`.

## Phase 3: Images (Optional)

Start image generation — but **DON'T WAIT**.

The server reports ~40-55 minutes execution time. Use this time to start with chapters immediately.

## Phase 4: Chapters (Core Phase)

**Important: Strict sequence!**

1. Chapter 1: Draft → Review → Revision → Approval
2. Then Chapter 2: Draft → Review → Revision → Approval
3. And so on...

**Never in parallel!** Chapters build on each other.

### Review Process

Each chapter is automatically scored:

| Score | Decision | Action |
|-------|----------|--------|
| 8.0-10.0 | ✅ APPROVED | Copy to `02-chapters/`, next chapter |
| 6.0-7.9 | ⚠️ MINOR_REVISION | Specific fixes, max 3 attempts |
| 4.0-5.9 | 🔧 MAJOR_REVISION | Major rewrite, max 3 attempts |
| 0.0-3.9 | ❌ REJECTED | Complete rewrite, max 3 attempts |

## Phase 6: Publication

**Create immediately:**
1. PDF/EPUB **without visuals**

**Then check:**
- Are images from Phase 3 ready?
  - **Yes:** Create final version with visuals
  - **No:** Wait, check again later
