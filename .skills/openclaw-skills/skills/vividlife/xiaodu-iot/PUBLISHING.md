# 小度设备控制技能 - 发布说明

## 技能概述

这是一个用于控制小度智能设备的OpenClaw技能，支持：
- 小度音箱设备控制（语音指令、播报）
- IoT设备控制（灯光、窗帘、开关等）
- 设备列表查看和状态监控
- 语音播报和批量控制

## 发布准备检查清单

### ✅ 已完成的安全检查
1. **敏感信息清理**：所有真实Access Token、CUID、Client ID已替换为占位符
2. **个人信息移除**：设备名称通用化，无个人标识
3. **配置占位符**：使用`YOUR_*`格式的配置变量
4. **示例代码安全**：所有示例使用通用设备信息

### 🔧 用户配置步骤
用户需要配置以下信息：

1. **获取Access Token**
   - 通过百度DuerOS开放平台申请
   - 有效期30天，需要定期刷新

2. **获取设备信息**
   ```bash
   # 首次运行获取设备列表
   mcporter call xiaodu.list_user_devices
   # 从中提取CUID和Client ID
   ```

3. **配置技能**
   - 替换所有`YOUR_ACCESS_TOKEN_HERE`
   - 替换所有`YOUR_DEVICE_CUID`
   - 替换所有`YOUR_CLIENT_ID`

### 📦 技能打包
```bash
# 使用OpenClaw技能打包工具
openclaw skills package xiaodu-control

# 或使用提供的脚本
python scripts/init_skill.py --package
```
