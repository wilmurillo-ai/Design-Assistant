# Discord Bot Feature Map

## Core
- Guilds, channels, roles, members
- Messages, embeds, attachments
- Webhooks

## Interactions
- Slash commands
- User and message context commands
- Buttons, select menus
- Modals
- Autocomplete

## Realtime
- Gateway events
- Presence and intents

## Community
- Threads
- Forum channels
- Scheduled events

## Safety and moderation
- Auto moderation rules
- Audit log awareness

## Optional surfaces
- Activities, voice, and stage features (if needed)
