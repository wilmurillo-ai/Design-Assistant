# GoldSkill OpenClaw 安装说明

## 文件结构

```
openclaw_skill/
├── SKILL.md          # Skill 描述文件
├── agent.py          # 核心执行逻辑
└── requirements.txt  # Python 依赖
```

## 安装步骤

### 1. 复制 Skill 目录

将 `openclaw_skill` 文件夹复制到 openclaw 的 skills 目录：

```powershell
cp -r "c:\Users\14278\Desktop\goldskill_v3.2\gs_v32\openclaw_skill" "$HOME\.openclaw\skills\goldskill"
```

### 2. 安装依赖

```powershell
cd "$HOME\.openclaw\skills\goldskill"
pip install -r requirements.txt
```

### 3. 重启 openclaw

```powershell
openclaw restart
```

## 使用方法

在 openclaw 对话中直接说：

- `"分析黄金"` - 黄金行情分析
- `"白银行情"` - 白银行情分析
- `"原油期货分析"` - WTI原油分析
- `"天然气"` - 天然气行情
- `"铜价"` - 铜期货分析
- `"查看全部大宗商品"` - 全品种分析

## 支持品种

| 品种 | 代码 | 名称 |
|------|------|------|
| 黄金 | XAUUSD | 黄金现货/期货 |
| 白银 | XAGUSD | 白银现货/期货 |
| 原油 | CRUDEOIL | WTI原油 |
| 天然气 | NATGAS | 天然气 |
| 铜 | COPPER | 铜期货 |

## 功能特性

- 实时行情获取（Yahoo Finance / Stooq / metals.live）
- 技术指标分析（RSI、MACD、布林带、CCI、随机指标等）
- 新闻情绪分析
- 量化策略信号（做多/做空/观望）
- 止损止盈建议
