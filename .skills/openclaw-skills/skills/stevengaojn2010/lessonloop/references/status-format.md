# LessonLoop status/report format

## Suggested trigger phrases
- `LessonLoop Status`
- `LessonLoop Report`

## Suggested output

```text
LessonLoop Report
period: 7d
captured: <n>
daily: <n>
candidateLongTerm: <n>
longTerm: <n>
latest: <short latest lesson>
```

Optional expanded fields:
- lesson types breakdown
- source breakdown
- false positive notes
