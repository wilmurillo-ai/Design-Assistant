# 产品评论分类指南

本文档提供产品评论分类的详细标准和示例。

## 分类定义

### Bug（产品错误）

产品功能异常、故障、崩溃或行为不符合预期。

**关键特征**：
- 描述错误行为、故障现象
- 提及崩溃、卡死、报错
- 功能无法正常使用
- 数据丢失或损坏

**示例**：

| 评论 | 分类 | 模块 |
|------|------|------|
| 导出数据时软件闪退了 | Bug | 数据导出 |
| 点击保存按钮没反应，页面卡死 | Bug | 数据保存 |
| 报错了，显示Error 500 | Bug | 系统稳定性 |
| 导出的PDF文件内容错乱，表格重叠 | Bug | 文件导出 |
| 登录后显示的不是我的账号信息 | Bug | 用户登录 |

---

### 建议（功能改进）

对现有功能的优化意见，不涉及新增功能。

**关键特征**：
- 使用"希望"、"建议"、"可以"等词
- 针对已有功能的改进
- 提升易用性、效率、体验

**示例**：

| 评论 | 分类 | 模块 |
|------|------|------|
| 希望导出支持Excel格式 | 建议 | 数据导出 |
| 建议把导出按钮放在更显眼的位置 | 建议 | 界面布局 |
| 可以增加批量删除功能吗 | 建议 | 数据管理 |
| 希望支持深色模式 | 建议 | 界面主题 |
| 建议增加搜索历史记录 | 建议 | 搜索功能 |

---

### 需求（功能新增）

希望添加产品当前不存在的全新功能。

**关键特征**：
- "需要"、"想要"、"能不能加"
- 当前产品不具备的能力
- 新的业务场景支持

**示例**：

| 评论 | 分类 | 模块 |
|------|------|------|
| 需要批量导入用户的功能 | 需求 | 用户管理 |
| 想要数据可视化图表功能 | 需求 | 数据分析 |
| 能不能加个自动化定时备份 | 需求 | 数据备份 |
| 需要支持移动端APP | 需求 | 多端支持 |
| 想要API接口对接第三方系统 | 需求 | 系统集成 |

---

### 其他（非功能相关）

咨询、评价、客服反馈、与其他功能无关的内容。

**关键特征**：
- 咨询类问题
- 服务评价
- 感谢/抱怨
- 与产品功能无关的讨论

**示例**：

| 评论 | 分类 |
|------|------|
| 客服响应很快，态度很好 | 其他 |
| 怎么联系技术支持？ | 其他 |
| 谢谢你们的帮助 | 其他 |
| 价格有点贵 | 其他 |
| 什么时候发新版本？ | 其他 |
| 这个软件是免费的吗 | 其他 |

---

## 边界情况处理

### Bug vs 建议

| 评论 | 分类 | 理由 |
|------|------|------|
| 导出速度太慢了 | 建议 | 功能正常，只是性能待优化 |
| 导出时一直转圈，最后超时 | Bug | 功能无法完成 |

### 建议 vs 需求

| 评论 | 分类 | 理由 |
|------|------|------|
| 希望搜索功能支持模糊搜索 | 建议 | 搜索功能已存在 |
| 需要添加搜索功能 | 需求 | 搜索功能不存在 |

### 无效内容

以下内容应标记为 `is_valid: false`：

- 纯表情符号：😊😊😊
- 无意义字符：asdfgh
- 空白或只有标点
- 明显的测试内容：测试123、哈哈、好的
