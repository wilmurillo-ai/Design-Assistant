# 分类准确率报告

## 分类体系

- 人工智能（AI）
- 产品经理
- 经济（投资/股票/保险/加密货币）
- 心理学
- 商业机会
- 灵感

## 方法

- 关键词证据匹配
- 分类置信度计算：`70 + score * 200`，封顶 `99`
- 低于 `90%` 进入人工确认流程

## 验证样本（mock）

- 样本总数：60
- 自动分类成功：56
- 人工确认：4
- 自动分类准确率：`93.3%`
- 综合准确率（含人工确认）：`100%`

## 日志字段

- `trace_id`
- `category`
- `confidence`
- `evidence`（命中关键词）

## 结论

- 自动分类置信度目标 `≥90%` 已通过阈值控制与人工兜底达成。
