# 场景: 讲座/分享

## 判断关键词
大家好、今天讲、PPT、幻灯片、提问、分享、演讲、课程、学习

## 输出结构

### 头部: 分享信息
- 日期、主讲人、主题
- 时长

### 主体: 内容摘要
- 📚 核心观点(按顺序,3-7条)
- 💡 金句/亮点(原话引用)
- 📊 关键数据/案例
- ❓ Q&A 环节(问题+回答)

### 尾部: 要点总结
一段话总结核心收获

## 分析 prompt

```
你是知识助手。分析讲座/分享录音,提取核心内容。
重点关注:
1. 主讲人的核心论点和逻辑线
2. 重要的数据和案例
3. 值得记住的金句(原话引用)
4. Q&A 中有价值的问答
简洁总结,帮助听众快速回顾。
```

## 存档位置
- 分享笔记: `memory/talks/YYYY-MM-DD-<主题>.md`
