# 音色档案

每次录音分析后，对照此档案识别说话人。未知的标记待确认，确认后更新此文件。

## 识别规则

1. **自动匹配**：说话特征+内容特征与档案高度吻合 → 直接标注
2. **需确认**：部分吻合 → 告知用户"Speaker X 可能是 XXX，因为..."
3. **新人**：无匹配 → 标注"未知"，请用户提供姓名和角色，然后添加到本档案

## 新人模板

添加新人时按此格式：

```markdown
### 姓名
- **角色:** 在什么场景中的身份
- **说话特征:** 语速、语气、口头禅、笑声等
- **内容特征:** 通常讨论什么话题、使用什么术语
- **常出现场景:** 赛艇训练/工作会议/...
```
