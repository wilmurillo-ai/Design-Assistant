## Install

### Via ClawHub (recommended)

```bash
clawhub install MyClaw-AI/openclaw-marketing-skills
```

Or install individual skills:

```bash
clawhub install MyClaw-AI/openclaw-marketing-skills --skill copywriting page-cro
```

### Manual

Clone and copy the skills folder:

```bash
git clone https://github.com/LeoYeAI/openclaw-marketing-skills.git
cp -r openclaw-marketing-skills/skills/* ~/.agents/skills/
```

## Start here

Before using any other skill, run `product-marketing-context` to create `.agents/product-marketing-context.md`. All 33 skills automatically read this file — you only describe your product once.

```
Set up my product marketing context
```

Then use any skill naturally:

```
Optimize this landing page for conversions
→ uses page-cro

Write homepage copy for my SaaS
→ uses copywriting

Audit our SEO
→ uses seo-audit

Create a 5-email welcome sequence
→ uses email-sequence

Help me with Google Ads
→ uses paid-ads
```
