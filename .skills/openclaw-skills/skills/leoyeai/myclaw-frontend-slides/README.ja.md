**美しいアニメーション付きHTMLプレゼンテーションを作成 — ゼロから、またはPowerPointファイルを変換して。**

依存関係なしで美しいウェブスライドを作成できるOpenClawスキル。インラインCSS/JSを含む単一HTMLファイル — npm、ビルドツール、フレームワーク不要。

## ✨ 機能

- **依存関係ゼロ** — インラインCSS/JSを含む単一HTMLファイル。
- **12種類のスタイル** — Bold Signal、Neon Cyber、Dark Botanical、Swiss Modernなど。
- **PPT変換** — PowerPointファイルをウェブプレゼンテーションに変換。
- **ビジュアルスタイル発見** — 3つのプレビューを生成して好みを選択。
- **本番品質** — キーボードナビゲーション、タッチ/スワイプ、アニメーション、レスポンシブ対応。

## 🚀 インストール

```bash
clawhub install frontend-slides
```
