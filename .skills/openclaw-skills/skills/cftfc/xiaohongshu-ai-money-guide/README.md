# 小红书 AI 副业指南生成器

自动化生成"虾的薯片"风格的小红书爆款笔记，专注 AI 副业赚钱主题。

---

## 🚀 快速开始

### 生成笔记

```bash
# 生成 1 篇（默认主题：AI副业）
./xiaohongshu-gen

# 生成 5 篇，保存到文件
./xiaohongshu-gen --count 5 --output my_notes.md

# 指定主题（如 OpenClaw）
./xiaohongshu-gen --topic OpenClaw --count 3

# 不生成封面提示词
./xiaohongshu-gen --no-cover
```

## 🎯 主题列表

当前支持的主题（--topic 参数）：
- `AI副业`（默认）- 泛用型 AI 副业
- `OpenClaw` - 专攻 OpenClaw 自动化
- `知乎盐选` - 知乎盐选故事生成
- `公众号` - 微信公众号排版
- `B站` - B站视频文案

---

## 🛠️ 自定义

如果你想修改内容风格：

1. 编辑 `prompts/titles.txt` 添加/修改标题模板
2. 编辑 `prompts/cover_prompts.txt` 调整封面图风格
3. 脚本使用内置正文模板（如需扩展，可修改 xiaohongshu-gen 脚本）

---

## 💾 数据存储

- 生成的笔记默认打印到终端
- 使用 `--output` 保存到文件
- 所有备份自动存入 `notes/` 目录，按时间戳命名

---

## 📈 变现建议

1. **前期（0-1000粉）**：平台流量分成，每万阅读 ¥3-10

## ❓ 故障排除

- 权限错误：`chmod +x xiaohongshu-gen`
