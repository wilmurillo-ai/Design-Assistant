# 文献调研技能

## When to Use
- 查找并分析特定领域的高分文献
- 做文献汇报/文献综述时使用

## Procedure
1. 调用 Zotero API 搜索相关文献
2. 获取文献元数据（标题、期刊、年份、DOI、摘要）
3. 评估期刊影响力和引用情况
4. 进行网络补充搜索
5. 提取关键发现和创新点
6. 整理成结构化文献汇报

## Notes
- Zotero API 需要 userID 和 API key
- 优先选择 PNAS, Nature, Science, Cell 等顶刊
