### 5.3 测试脚本示例

```java
@Test
@DisplayName("TC-001: Amazon平台基础图片搜索")
void testAmazonBasicImageSearch() {
    // 构造请求
    ImageSearchApiRequest request = new ImageSearchApiRequest();
    request.setImageUrl("https://img.alicdn.com/imgextra/xxx.jpg");
    request.setPlatform("amazon");
    request.setRegion("US");
    request.setRecallNum(10);

    // 调用接口
    ApiResultModel<ImageSearchApiResponse> result =
        oppSelectionHsfService.imageSearchApi(request, "test_user_001");

    // 断言验证
    assertNotNull(result);
    assertEquals("SUCCESS", result.getCode());
    assertNotNull(result.getModel());
    assertEquals("amazon", result.getModel().getPlatform());
    assertEquals("US", result.getModel().getRegion());
    assertTrue(result.getModel().getTotalCount() > 0);
    assertNotNull(result.getModel().getProductList());
    assertFalse(result.getModel().getProductList().isEmpty());

    // 验证商品字段
    ProductApiVO product = result.getModel().getProductList().get(0);
    assertNotNull(product.getProductId());
    assertNotNull(product.getProductName());
    assertNotNull(product.getImageUrl());
    assertNotNull(product.getPrice());
}
```

### 9.3 注意事项

1. **测试环境要求**:
   - 确保EMRepository（图片向量化服务）正常可用
   - 确保ProductRepository（OpenSearch）正常可用
   - 确保测试图片URL可访问

2. **测试数据注意**:
   - 使用真实的商品图片URL
   - 避免使用敏感或版权图片
   - 准备多种类型的商品图片

3. **性能测试注意**:
   - 性能测试应在独立环境进行
   - 避免影响生产环境
   - 记录详细的性能指标

4. **测试执行顺序**:
   - 先执行正常场景，确保基本功能可用
