## 2. 需求背景

### 2.1 业务需求
从 global-1688-ai-sel 项目中提取"一张图召回全平台下游Top同款商品"的能力，封装为独立的对外API接口，供外部系统调用。

### 2.2 现有实现分析
项目中已存在图片搜索功能：
- **HSF接口**: `OppImageSearchService.recallImgs()`
- **核心实现**: `OppImageSearchServiceImpl`
- **主要流程**:
  1. 接收图片URL和裁剪区域
  2. 调用embedding模型获取图片向量
  3. 基于向量相似度在OpenSearch中召回商品
  4. 返回同款商品图片URL列表

### 2.3 参考实现
已有的关键词搜索HTTP API（`keywordSearchApi`）可作为参考：
- 入参校验机制
- 错误处理规范
- 响应码枚举
- VO转换模式

## 3. 接口设计

### 3.1 接口概述

| 项目 | 内容 |
|------|------|
| 接口名称 | 图片商品召回API |
| 接口功能 | 根据用户上传的图片URL，通过AI图像识别和向量检索，返回全平台同款/相似商品列表 |
| 请求方式 | POST |
| Content-Type | application/json |
| 响应耗时 | 10秒内 |
| Endpoint | https://api.alphashop.cn/opp.selection.image.search/1.0 |

### 3.2 入参设计

#### 3.2.1 入参结构
```java
public class ImageSearchApiRequest {
    /**
     * 图片URL（必填）
     */
    private String imageUrl;

    /**
     * 图片裁剪区域（可选）
     * 格式: "x,y,width,height"
     */
    private String cropRegion;

    /**
     * 目标平台（必填）
     * 枚举值: amazon/tiktok
     */
    private String platform;

    /**
     * 目标国家/地区（必填）
     * 取值范围受platform参数限制
     */
    private String region;

    /**
     * 召回数量（可选）
     * 默认值: 10
     * 取值范围: 1-50
     */
    private Integer recallNum;

    /**
     * 相似度阈值（可选）
     * 默认值: 0.5
     * 取值范围: 0.0-1.0
     */
    private String similarityValue;

    /**
     * 用户ID（必填，HTTP Header传递）
     */
    private String userId;
}
```

#### 3.2.2 入参约束
1. **platform 与 region 的合法性校验**：
   - Amazon 平台支持: US, UK, ES, FR, DE, IT, CA, JP
   - TikTok 平台支持: ID, VN, MY, TH, PH, US, SG, BR, MX, GB, ES, FR, DE, IT, JP

2. **图片URL校验**：
   - 必须是合法的HTTP/HTTPS URL
   - 支持的图片格式: JPG, JPEG, PNG, WEBP
   - 图片大小限制: 不超过10MB

3. **召回数量限制**：
   - 默认: 10
   - 最小: 1
   - 最大: 50

4. **相似度阈值**：
   - 默认: 0.5
   - 取值范围: 0.0-1.0
   - 值越大，返回结果越精准但数量可能越少

## 4. 核心流程设计

### 4.1 整体流程图

```
Client Request
    ↓
入参校验
    ↓
图片URL有效性检查
    ↓
调用Embedding服务获取图片向量
    ↓
构建向量检索查询条件
    ├─ platform过滤
    ├─ region过滤
    ├─ embedding向量
    └─ similarityValue阈值
    ↓
调用OpenSearch向量检索
    ↓
商品列表过滤和排序
    ↓
OppProduct转换为ProductApiVO
    ↓
返回结果
```

### 4.2 关键实现步骤

#### 4.2.1 入参校验
```java
private ApiResultModel<ImageSearchApiResponse> validateImageSearchRequest(
    ImageSearchApiRequest request) {

    // 1. 基础参数校验
    if (request == null) {
        return ApiResultModel.fail(ApiResCode.REQUEST_PARAM_EMPTY);
    }

    // 2. 图片URL校验
    if (StringUtils.isBlank(request.getImageUrl())) {
        return ApiResultModel.fail(ApiResCode.IMAGE_URL_EMPTY);
    }
    if (!isValidImageUrl(request.getImageUrl())) {
        return ApiResultModel.fail(ApiResCode.IMAGE_URL_ILLEGAL);
    }

    // 3. 平台和地区校验
    if (StringUtils.isBlank(request.getPlatform())) {
        return ApiResultModel.fail(ApiResCode.TARGET_PLATFORM_EMPTY);
    }
    if (StringUtils.isBlank(request.getRegion())) {
        return ApiResultModel.fail(ApiResCode.TARGET_REGION_EMPTY);
    }

    // 4. 平台和地区匹配校验
    if (!isValidPlatformRegionPair(request.getPlatform(), request.getRegion())) {
        return ApiResultModel.fail(ApiResCode.TARGET_REGION_ILLEGAL);
    }

    // 5. 召回数量校验
    if (request.getRecallNum() != null) {
        if (request.getRecallNum() < 1 || request.getRecallNum() > 50) {
            return ApiResultModel.fail(ApiResCode.RECALL_NUM_ILLEGAL);
        }
    }

    // 6. 相似度阈值校验
    if (StringUtils.isNotBlank(request.getSimilarityValue())) {
        try {
            double similarity = Double.parseDouble(request.getSimilarityValue());
            if (similarity < 0.0 || similarity > 1.0) {
                return ApiResultModel.fail(ApiResCode.SIMILARITY_VALUE_ILLEGAL);
            }
        } catch (NumberFormatException e) {
            return ApiResultModel.fail(ApiResCode.SIMILARITY_VALUE_ILLEGAL);
        }
    }

    return null; // 校验通过
}
```

#### 4.2.2 图片向量化
```java
private ImgEmbeddingResponse getImageEmbedding(String imageUrl, String cropRegion) {
    try {
        EMImgEmbeddingRequest embeddingRequest =
            new EMImgEmbeddingRequest(imageUrl, cropRegion);
        ImgEmbeddingResponse response = emRepository.askImageEmbeddings(embeddingRequest);

        if (response == null || response.getEmbeddings() == null) {
            log.error("Failed to get embeddings for image: {}", imageUrl);
            throw new OppSelectException(ApiResCode.IMAGE_EMBEDDING_ERROR);
        }

        return response;
    } catch (Exception e) {
        log.error("Error getting image embeddings", e);
        throw new OppSelectException(ApiResCode.IMAGE_EMBEDDING_ERROR);
    }
}
```

#### 4.2.3 构建检索条件
```java
private List<Property> buildSearchProperties(
    ImageSearchApiRequest request,
    ImgEmbeddingResponse embeddingResponse) {

    List<Property> properties = new ArrayList<>();

    // 1. 添加平台过滤
    if (StringUtils.isNotBlank(request.getPlatform())) {
        properties.add(new UniqueProperty(
            OppItemFields.PLATFORM.getField(),
            request.getPlatform()
        ));
    }

    // 2. 添加地区过滤
    if (StringUtils.isNotBlank(request.getRegion())) {
        properties.add(new UniqueProperty(
            OppItemFields.COUNTRY_CODE.getField(),
            request.getRegion()
        ));
    }

    // 3. 添加向量embedding
    String embeddingStr = buildEmbeddingString(
        embeddingResponse.getEmbeddings(),
        request
    );
    properties.add(new UniqueProperty(
        OppItemFields.EMBEDDING.getField(),
        embeddingStr
    ));

    // 4. 添加相似度阈值
    String similarityValue = Optional.ofNullable(request.getSimilarityValue())
        .orElse(oppDiamondConfig.getSearchProductHa3ParamConfig()
            .getSimilarityValue());
    properties.add(new UniqueProperty(
        OppItemFields.IMAGE_SF.getField(),
        similarityValue
    ));

    return properties;
}
```

#### 4.2.4 商品召回
```java
private List<OppProduct> recallProducts(
    List<Property> properties,
    Integer recallNum) {

    // 设置分页参数
    QueryPagination pagination = new QueryPagination();
    pagination.setLimit(Optional.ofNullable(recallNum).orElse(10));

    // 执行OpenSearch查询
    List<OppProduct> products = productRepository.listByPropertiesWithPagination(
        properties,
        pagination
    );

    log.info("Image search recalled {} products", products.size());
    return products;
}
```

#### 4.2.5 结果转换
```java
private List<ProductApiVO> convertToProductApiVO(List<OppProduct> oppProducts) {
    if (CollectionUtils.isEmpty(oppProducts)) {
        return Collections.emptyList();
    }

    return oppProducts.stream()
        .map(this::convertSingleProduct)
        .filter(Objects::nonNull)
        .collect(Collectors.toList());
}

private ProductApiVO convertSingleProduct(OppProduct product) {
    ProductApiVO vo = new ProductApiVO();
    vo.setProductId(product.getOfferId());
    vo.setProductName(product.getTitle());
    vo.setImageUrl(product.getImageUrl());
    vo.setMainImageUrl(product.getMainImageUrl());

    // 价格信息
    if (product.getPrice() != null) {
        PriceVO priceVO = new PriceVO();
        priceVO.setAmount(product.getPrice().toString());
        priceVO.setCurrencySymbol(getCurrencySymbol(product.getCountryCode()));
        priceVO.setAmountWithSymbol(
            priceVO.getCurrencySymbol() + priceVO.getAmount()
        );
        vo.setPrice(priceVO);
    }

    // 相似度评分（从扩展信息中获取）
    String similarityScore = product.getExtInfo(ProductExtInfoKeys.SIMILARITY_SCORE);
    vo.setSimilarityScore(similarityScore);

    // 销售信息
    if (product.getSalesVolume30d() != null || product.getSalesAmount30d() != null) {
        SalesInfoVO salesInfo = new SalesInfoVO();
        salesInfo.setSoldCnt30d(formatNumber(product.getSalesVolume30d()));

        if (product.getSalesAmount30d() != null) {
            AmountVO amountVO = new AmountVO();
            amountVO.setAmount(formatNumber(product.getSalesAmount30d()));
            amountVO.setCurrencySymbol(getCurrencySymbol(product.getCountryCode()));
            amountVO.setAmountWithSymbol(
                amountVO.getCurrencySymbol() + amountVO.getAmount()
            );
            salesInfo.setSoldAmt30d(amountVO);
        }
        vo.setSalesInfo(salesInfo);
    }

    // 评分和评论数
    vo.setRating(formatRating(product.getReviewScore()));
    vo.setReviewCount(formatNumber(product.getReviewCount()));

    // 供应商信息
    if (StringUtils.isNotBlank(product.getSupplierId())) {
        SupplierInfoVO supplierInfo = new SupplierInfoVO();
        supplierInfo.setSupplierId(product.getSupplierId());
        supplierInfo.setSupplierName(product.getSupplierName());
        vo.setSupplierInfo(supplierInfo);
    }

    return vo;
}
```

## 6. 性能考虑

### 6.1 响应时间要求
- 总体响应时间: ≤10s
- Embedding服务: ≤5s
- OpenSearch检索: ≤3s
- 数据转换: ≤1s

### 6.2 并发处理
- 支持并发请求: 100 QPS
- 使用连接池管理外部服务连接
- 实现请求限流和熔断机制

### 6.3 缓存策略
- 相同图片URL的embedding结果可缓存15分钟
- 缓存Key: MD5(imageUrl + cropRegion)

## 7. 错误处理

### 7.1 异常分类
1. **客户端错误** (4xx)：参数错误、格式错误等
2. **服务端错误** (5xx)：系统异常、依赖服务失败等

### 7.2 异常处理原则
1. 所有异常必须被捕获并转换为统一的响应码
2. 记录详细的错误日志（包含traceId）
3. 敏感信息不暴露给客户端
4. 提供有意义的错误提示信息
