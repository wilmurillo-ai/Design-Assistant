# 分销新人完整参考手册

本文件包含各环节的详细操作步骤和完整参数说明，供 agent 在用户需要深入了解时读取。

## 一、开店准备完整流程

### 操作步骤

| 步骤        | 操作内容                                             | 关键要点               |
| ----------- | ---------------------------------------------------- | ---------------------- |
| 1. 账号准备 | 注册 1688 账号、准备身份证/银行卡/手机号             | 确保信息真实有效       |
| 2. 确定方向 | 选择下游平台、店铺类型、经营类目                     | 根据自身资源和能力选择 |
| 3. 平台入驻 | 按平台要求提交资料、缴纳保证金（0-2000 元不等）      | 关注各平台最新政策     |
| 4. 基础设置 | 设置店铺头像/名称/介绍、运费模板、退换货规则         | 完善店铺基础信息       |
| 5. 工具准备 | 下载对应平台商家后台 APP（千牛/抖店/拼多多商家版等） | 熟悉后台操作界面       |

### 平台详细对比

| 平台   | 运营重点                                   | 优势                                           | 劣势                                     | 适合人群                               |
| ------ | ------------------------------------------ | ---------------------------------------------- | ---------------------------------------- | -------------------------------------- |
| 淘宝   | 商品素材优化、直通车、客服响应             | 规则完善、流量稳定、用户购买意图强、生态成熟   | 竞争激烈、无货源监管严、新手起步慢       | 有耐心、愿意学习规则、追求长期稳定经营 |
| 抖音   | 短视频内容、直播带货、千川投放、体验分维护 | 流量爆发快、内容驱动、适合趋势款/视觉款        | 体验分要求高、内容制作门槛高、售后压力大 | 擅长内容创作、有表现力、能快速响应热点 |
| 拼多多 | 低价策略、活动报名、多多进宝、客服回复率   | 起量快、用户对价格敏感、白牌友好               | 仅退款率高、利润薄、客服压力大、罚款多   | 能接受低利润走量、抗压能力强、执行力高 |
| 小红书 | 笔记种草、KOC 合作、搜索优化、私域沉淀     | 用户粘性高、种草转化好、溢价空间大、竞争相对小 | 转化周期长、流量不稳定、电商基建较弱     | 擅长内容种草、有审美、愿意长期经营私域 |

### 开店避坑

- 不要盲目选择高保证金类目：新店启动期避开 3C 数码、母婴等高保证金类目。
- 不要跳过基础设置：运费模板、退换货规则等必须提前配置好。

## 三、选品策略详解

### 操作步骤

| 步骤                  | 操作内容                                                   | 关键要点                             |
| --------------------- | ---------------------------------------------------------- | ------------------------------------ |
| 1. 进入分销 AI 工作台 | 1688 首页 → 分销 → 一件代发频道 → AI 工作台                | —                                    |
| 2. 搜索商品           | 词搜：输入关键词（如"夏季女装"）；图搜：上传图片找同款     | 词搜用精准长尾词，图搜注意原图清晰度 |
| 3. 筛选条件           | 勾选"晚揽必赔"、"渠道面单"，按"代发订单数"排序             | 优先选择履约能力强的供应商           |
| 4. 供应商评估         | 查看标识（实力商家/诚信通/超级工厂）、综合服务分、买家评价 | 选择有资质、口碑好的供应商           |
| 5. 商品选择           | 新店期选出单福利品，成长期选潜力新品，稳定期选正价品       | 根据店铺阶段调整选品策略             |
| 6. 合规检查           | 使用 AI 检测、检查品牌词、擦边词、图片风格一致性           | 避免违规风险                         |

### 供应商判断标准

查看认证标识（牛头标志/诚字/超级工厂）、成立时间、综合服务分、交易勋章和买家评价。

### 选品避坑

- **高风险类目慎选**：资质门槛高、物流成本高、售后复杂的类目尽量避开。
- **品牌商品不能碰**：标题/图片带品牌名称、擦边大牌表述的商品坚决不选。
- **图片风格要统一**：拍摄风格差异大的商品可能非原创，存在侵权风险。
- **关注代发订单数**：比总订单数更能反映商品的分销热度。

### 搜索技巧

搜索不到想要的商品时，尝试更换关键词组合，或使用图搜功能。关键词不要太宽泛，可加入属性词（如"纯棉短袖女夏"）。

## 四、铺货操作详解

### 操作步骤

| 步骤        | 操作内容                                              | 关键要点               |
| ----------- | ----------------------------------------------------- | ---------------------- |
| 1. 订购工具 | 通过 1688 分销频道或下游服务市场订购铺货工具          | 选择官方推荐工具       |
| 2. 授权绑店 | 完成工具订购 → 跳转下游平台 → 授权店铺 → 确认"已授权" | 确保授权状态正常       |
| 3. 铺货配置 | 设置价格公式、运费模板、库存同步、标题过滤规则        | 首次铺货必须完成配置   |
| 4. 执行铺货 | 单品铺货/批量铺货/自动铺货（建议每日 ≤50 件）         | 自动铺货后必须人工检查 |
| 5. 结果检查 | 查看铺货日志、处理失败商品、验证商品信息准确性        | 确保价格/库存/图片正确 |

### 铺货失败常见原因

- 运费模板同步异常 → 重新配置同步规则
- SKU 信息匹配错误 → 检查 SKU 映射
- 图片空间不足 → 清理图片空间

### 价格设置详解

基准公式：货源代发价 x 130% + 5 元。包邮商品少量加价，不包邮商品需扣除运费成本。初期可略低于同行 5-10% 提高出单率。

### 铺货避坑

- **必须删除的词汇**：专供、源头、包邮、一件代发、厂家、直销、批发、跨境、工厂、现货。
- **自动铺货不等于不管**：每日上新不超过 50 件（淘宝/抖店），铺货后必须全面检查。
- **库存同步要设置**：SKU 库存 <20 自动归零，避免超卖。
- **物流模板要完善**：设置偏远地区禁售、合理配置运费。

### 广告法违禁词完整列表

禁用词：最、第一、国家级、全网首发、独家、百分百、绝对、永久。
替代词：优选、热门、深受喜爱、值得信赖、值得尝试。

## 五、采购管理详解

### 操作步骤

| 步骤            | 操作内容                                                 | 关键要点                 |
| --------------- | -------------------------------------------------------- | ------------------------ |
| 1. 自动下单设置 | 开启自动下单功能，设置采购时效；确认"密文下单"已开启     | 开启密文下单             |
| 2. 利润保护配置 | 设置订单利润下限、利润率下限、宝贝数量上限、采购运费上限 | 防止亏损订单             |
| 3. 特殊订单处理 | 开启买家留言/备注自动跳过、相同地址跳过、多子订单跳过    | 避免异常订单             |
| 4. 下单频率控制 | 控制下单频率，避免触发平台风控                           | 频率过快会导致账号受限   |
| 5. 异常处理     | 发现亏损提示 → 点击"更换货源" → 核实密文面单 → 评估利润  | 采购后无法挽回则改价止损 |

### 密文面单详解

密文面单 = 加密的快递单，买家看不到你的真实姓名、电话、地址，供应商也看不到买家的完整信息，保护双方隐私。淘宝、抖音、拼多多等主流平台强制要求使用密文面单，不用密文面单发货可能判定违规，限制流量或处罚。

确认供应商是否支持密文面单的方法：

- 在 1688 商品详情页代发板块查看商品支持密文代发哪些平台
- 在 AI 工作台选品时勾选"对应出单平台的渠道面单"
- 与供应商客服确认："请问支持密文面单发货吗？"

### 自动下单时间建议

推荐"30 分钟后自动下单"，给买家留出考虑时间，也可添加赠品。立即下单无法处理临时要求，手动下单可能漏单。

### 采购避坑

- **利润保护必须设**：设置合理的利润下限和利润率下限，防止亏损订单。
- **特殊订单要跳过**：买家留言、相同地址、多子订单、赠品订单需特殊处理。

## 六、售后处理详解

### 操作步骤

| 步骤        | 操作内容                                             | 关键要点                             |
| ----------- | ---------------------------------------------------- | ------------------------------------ |
| 1. 退款处理 | 未发货开启"自动处理退款"，发货后依赖平台物流拦截     | 分阶段处理，提高效率                 |
| 2. 运费控制 | 谨慎开通运费险（每单约 0.5 元），选择包邮供应商      | 控制长期成本                         |
| 3. 退货地址 | 向 1688 商家索取指定退货地址；建议开通官方仓退货服务 | 默认退货地址建议设置为官方仓退货地址 |
| 4. 小额补偿 | 轻微瑕疵/主观不满用小额补偿代替退货                  | 避免拉低品退率影响流量               |
| 5. 信息同步 | 手动处理数据错乱情况，补充准确售后记录               | 确保上下游信息一致                   |

### 运费承担规则

- 未发货 → 自动处理
- 发货后 → 平台自动拦截
- 收货后无品质问题 → 买家承担
- 收货后品质问题 → 供应商承担（需确认配置运费险）

### 售后避坑

- **退款顺序不能错**：未发货自动处理，发货后依赖拦截，收货后区分责任。
- **运费成本要控制**：上游不包邮 + 下游包邮 = 中间商承担差额，务必选择包邮供应商。
- **退货地址要准确**：系统默认可能是个人地址，必须更改为供应商指定仓库。
- **善用小额补偿**：频繁退货拉低品退率影响流量，小额补偿更划算。
