## 案例一：老王队长（脑科学护眼）

### 背景
- 13年军旅经历，20年行业经验
- 从旧房改造转型脑科学护眼咨询
- 目标：帮助8-15岁近视孩子的家长找到度数上涨的根本原因

### 第一节：活儿

| 要素 | 内容 |
|------|------|
| 手艺领域 | 儿童脑科学护眼 + 个人IP定位咨询 |
| 拿手绝活 | 九维测评解读、验光单分析、IP三节棍定位 |
| 独门套路 | 三节棍定位法 + 九维测评体系 + 查修验增效闭环 |
| 差异价值 | 13年军旅不忽悠 + 脑科学找根因不只治标 |
| 一句话定位 | 13年军旅20年行业，用脑科学和实战经验帮你避坑 |

### 第二节：样

| 要素 | 内容 |
|------|------|
| 标志形象 | 穿立领的中年实干家 |
| 穿衣调性 | 简洁硬朗，不花哨 |
| 视觉符号 | 手拿验光单，用笔圈出关键数据 |
| 怎么称呼 | 老王队长 |
| 嘴边话 | "这个问题我见太多了" |
| 标志动作 | 每条视频开头展示一个真实数据或案例 |

### 第三节：魂

| 要素 | 内容 |
|------|------|
| 底线信念 | 不忽悠、不套路、不低价抢客 |
| 为什么干 | 看了太多人被坑，想帮普通人避坑 |
| 要到哪去 | 帮1000个家庭找到孩子学习的根因 + 帮100个创业者立住IP |
| 碰到谁的心 | 普通人对"被忽悠"的愤怒和无力感 |

### 一句话定位
> 我要用九维脑力测评在儿童护眼领域帮8-15岁近视孩子的家长找到度数上涨的根本原因

### 选题方向
1. **手艺活60%**：验光单解读、近视防控干货、测评案例分析
2. **跨界20%**：军旅故事、育儿经验、学习方法
3. **行业猛料20%**：配镜避坑、护眼产品真相、行业黑幕

## 案例二：琴子（私域电商）

### 背景
- 同时经营红酒、翡翠、鲜花三个品类
- 痛点：一个人干三个人的活，内容难写，客户跟进乱

### 第一节：活儿

| 要素 | 内容 |
|------|------|
| 手艺领域 | 生活方式美学 + 私域选品 |
| 拿手绝活 | 把普通产品讲出故事感、精准匹配客户需求 |
| 独门套路 | "场景+情感+稀缺"三段式文案法 |
| 差异价值 | 不卖货，卖生活方式；三个品类统一在"美"之下 |
| 一句话定位 | 帮你选对生活美学好物的闺蜜 |

### 第二节：样

| 要素 | 内容 |
|------|------|
| 标志形象 | 有品位的精致生活家 |
| 穿衣调性 | 优雅知性，有艺术感 |
| 视觉符号 |  always手持一杯红酒或一束花 |
| 怎么称呼 | 琴子姐 |
| 嘴边话 | "这个你一定要试试" |
| 标志动作 | 每条内容开头展示一个生活场景 |

### 第三节：魂

| 要素 | 内容 |
|------|------|
| 底线信念 | 只卖自己用过的好东西，不为了卖货而卖货 |
| 为什么干 | 想让更多人过上"不贵但精致"的生活 |
| 要到哪去 | 成为10000个家庭的"生活美学顾问" |
| 碰到谁的心 | 想要品质生活但不知道从何下手的焦虑感 |

### 一句话定位
> 我要用"场景+情感+稀缺"三段式文案法在私域电商领域帮25-40岁女性选对生活美学好物

## 案例三：曾嘉亿（大健康多身份）

### 背景
- 四重身份：非遗传承、徒手变美、中医康复、救援队
- 痛点：四个身份内容难写，专业形象与量产矛盾

### 第一节：活儿

| 要素 | 内容 |
|------|------|
| 手艺领域 | 传统中医养生 + 徒手面部调理 |
| 拿手绝活 | 非遗手法面部提升、中医体质调理 |
| 独门套路 | "内调外养"双轨调理法 |
| 差异价值 | 非遗传承人背书 + 不打针不吃药的天然变美 |
| 一句话定位 | 用老祖宗的方法，让你自然变美 |

### 第二节：样

| 要素 | 内容 |
|------|------|
| 标志形象 | 传统与现代结合的养生专家 |
| 穿衣调性 | 中式元素+现代剪裁，有文化底蕴 |
| 视觉符号 | 手持非遗工具或展示手法动作 |
| 怎么称呼 | 嘉亿老师 |
| 嘴边话 | "这个手法我外婆传给我的" |
| 标志动作 | 每条视频展示一个手法细节 |

### 第三节：魂

| 要素 | 内容 |
|------|------|
| 底线信念 | 不用化学成分、不夸大效果、不制造焦虑 |
| 为什么干 | 想让更多人知道中医不是慢，是用对方法才快 |
| 要到哪去 | 让非遗手法走进10000个家庭 |
| 碰到谁的心 | 害怕医美副作用、想要自然变美的安全感 |

### 一句话定位
> 我要用非遗传承的"内调外养"双轨调理法在中医养生领域帮30-50岁女性实现不打针不吃药的自然变美

## 案例四：高中数学老师

### 背景
- 15年教龄，带出过多个高考数学满分
- 痛点：家长沟通耗时、备课效率低、个人品牌弱

### 第一节：活儿

| 要素 | 内容 |
|------|------|
| 手艺领域 | 高考数学提分 + 学习方法指导 |
| 拿手绝活 | 把难题拆解成步骤、让学生"听得懂、做得出" |
| 独门套路 | "三步拆题法"：拆考点→拆步骤→拆变式 |
| 差异价值 | 15年只教数学，每个题型都研究透了 |
| 一句话定位 | 让数学从拖分项变成提分项 |

### 第二节：样

| 要素 | 内容 |
|------|------|
| 标志形象 | 严谨但不严肃的专业老师 |
| 穿衣调性 | 衬衫+西装外套，知识分子气质 |
| 视觉符号 | 白板上写满解题步骤 |
| 怎么称呼 | XX老师（用真名） |
| 嘴边话 | "这道题我讲过，换个数字就不会了？" |
| 标志动作 | 每条视频开头写一道题，结尾给出答案 |

### 第三节：魂

| 要素 | 内容 |
|------|------|
| 底线信念 | 不制造焦虑、不贬低学生、不只讲套路 |
| 为什么干 | 自己当年被数学折磨，想让学生少走弯路 |
| 要到哪去 | 帮1000个学生数学突破130分 |
| 碰到谁的心 | 数学差学生的自卑和无力感 |

### 一句话定位
> 我要用"三步拆题法"在高考数学领域帮高三学生把数学从拖分项变成提分项

## 案例五：贷款中介

### 背景
- 帮助中小企业解决融资难题
- 痛点：获客效率低、话术重复、客户信任度低

### 第一节：活儿

| 要素 | 内容 |
|------|------|
| 手艺领域 | 中小企业融资咨询 |
| 拿手绝活 | 快速匹配最优贷款方案、提高通过率 |
| 独门套路 | "三查三比"融资诊断法 |
| 差异价值 | 不收前期费用、不过件不收费 |
| 一句话定位 | 让好企业不被资金卡脖子 |

### 第二节：样

| 要素 | 内容 |
|------|------|
| 标志形象 | 靠谱的金融顾问 |
| 穿衣调性 | 商务休闲，专业但不刻板 |
| 视觉符号 | 手持计算器或贷款方案对比表 |
| 怎么称呼 | XX经理 |
| 嘴边话 | "这个方案我帮你算过了，能省XX万" |
| 标志动作 | 每条视频展示一个真实案例数据 |

### 第三节：魂

| 要素 | 内容 |
|------|------|
| 底线信念 | 不骗贷、不套路、不隐瞒费用 |
| 为什么干 | 看过太多好企业被高利贷拖垮 |
| 要到哪去 | 帮1000家中小企业解决融资难题 |
| 碰到谁的心 | 企业主对资金断裂的恐惧和对靠谱渠道的渴望 |

### 一句话定位
> 我要用"三查三比"融资诊断法在中小企业融资领域帮企业主找到最优贷款方案，不被资金卡脖子
