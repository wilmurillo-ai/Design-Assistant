# 三节棍定位法

> 老王队长独创的IP定位方法论
> 13年军旅+20年行业经验的实战总结
> 帮助创业者、老板、想做IP的人快速找到自己的定位

---

## 描述

三节棍定位法是老王队长原创的个人IP定位方法论，将IP定位拆解为三个核心维度：

- **活儿**：你能干什么（手艺、绝活、独门套路）
- **样**：别人怎么看你（形象、气质、记忆点）
- **魂**：别人信不信你（态度、底线、初心）

三节必须连在一起，才能打出力量。

---

## 使用场景

1. **自我定位**：梳理自己的IP定位，找到一句话说清你是谁
2. **帮别人定位**：作为咨询师/顾问，帮客户快速完成IP定位分析
3. **团队定位**：帮助团队成员找到自己的专业标签
4. **品牌重塑**：已有IP需要升级或调整定位方向

直接告诉我你想定位谁（自己或别人），我会通过提问引导你完成三节棍分析。

**示例**：
- "帮我做一下三节棍定位"
- "帮我朋友分析一下他的IP定位"

## 输出内容

完成定位后，你将获得：

1. **三节棍定位报告**（一页纸）
   - 活儿分析（5个要素）
   - 样分析（6个要素）
   - 魂分析（4个要素）

2. **一句话定位**
   - 公式：我要用【独门套路】在【手艺领域】帮【什么人】解决【什么问题】

3. **视觉锤方案**
   - 标志形象建议

   - 标志动作设计

   - 3个核心选题方向

5. **人设避坑清单**
   - 四大坑检查
   - 改进建议

## 核心方法论

### 第一节：活儿（你能干什么）

| 要素 | 核心问题 |
|------|----------|
| 手艺领域 | 你在哪个赛道有真本事？ |
| 拿手绝活 | 别人花钱找你，图你哪一手？ |
| 独门套路 | 你有没有自己的方法体系？ |
| 差异价值 | 你和别人有什么不同？ |
| 一句话定位 | 别人怎么介绍你？ |

**检验标准**：
- 有人愿意为你的"活儿"付费吗？
- 你的"独门套路"能教给别人吗？
- 离开平台，你还能靠这手活儿吃饭吗？

### 第二节：样（别人怎么看你）

| 要素 | 核心问题 |
|------|----------|
| 标志形象 | 别人看到什么就能想到你？ |
| 穿衣调性 | 你平时的穿着风格？ |
| 视觉符号 | 让人看一眼就记住的标志 |
| 怎么称呼 | 别人叫你什么？ |
| 嘴边话 | 你经常说的话？ |
| 标志动作 | 你每次出场的固定动作？ |

**检验标准**：
- 别人能在3秒内描述你的样子吗？
- 视频静音播放，别人能认出是你吗？
- 粉丝能模仿你的口头禅吗？

### 第三节：魂（别人信不信你）

| 要素 | 核心问题 |
|------|----------|
| 底线信念 | 你绝对不做什么？ |
| 为什么干 | 你为什么做这件事？ |
| 要到哪去 | 你最终想做到什么？ |
| 碰到谁的心 | 你能唤起什么情感？ |

**检验标准**：
- 你敢在公开场合说一句得罪同行的话吗？
- 粉丝会在评论区替你说话吗？
- 有人因为认同你的态度，愿意无条件支持你吗？

### 三节棍铁律

> 一节再硬，其他两节是散的，三节棍就是三根棍子，打不出力。

**互连检验**：
- **活儿 ↔ 样**：视频静音看一遍，别人能猜出你是干什么的吗？
- **样 ↔ 魂**：粉丝能用一个词形容你的性格吗？
- **魂 ↔ 活儿**：如果明天行业规则全变了，你还会干这行吗？

## 速查卡（30分钟定位流程）

### 第一步：聊活儿（10分钟）
- "你在哪个领域有真本事？"
- "别人花钱找你，图你哪一手？"
- "你有没有自己的方法？能教给别人吗？"
- "用一句话说清你是干什么的？"

### 第二步：看样（10分钟）
- "你平时怎么穿？给别人什么感觉？"
- "你有没有口头禅？"
- "你的视频静音播放，别人能认出你吗？"
- "给你起个外号，你希望别人怎么叫你？"

### 第三步：挖魂（10分钟）
- "你绝对不做什么？哪怕给钱也不干？"
- "你为什么做这件事？不是为了赚钱的那个原因"
- "你觉得这个行业最大的坑是什么？"
- "你最想改变什么？"

### 第四步：出结果
- 三节棍定位报告（一页纸）
- 一句话定位（可放简介）
- 三个选题方向
- 一个标志动作建议

## 参考案例

### 老王队长（脑科学护眼方向）

**活儿**：
- 手艺领域：儿童脑科学护眼 + 个人IP定位咨询
- 拿手绝活：九维测评解读、验光单分析、IP三节棍定位
- 独门套路：三节棍定位法 + 九维测评体系 + 查修验增效闭环
- 差异价值：13年军旅不忽悠 + 脑科学找根因不只治标

**样**：
- 标志形象：穿立领的中年实干家
- 穿衣调性：简洁硬朗，不花哨
- 视觉符号：手拿验光单，用笔圈出关键数据
- 怎么称呼：老王队长
- 嘴边话："这个问题我见太多了"
- 标志动作：每条视频开头展示一个真实数据或案例

**魂**：
- 底线信念：不忽悠、不套路、不低价抢客
- 为什么干：看了太多人被坑，想帮普通人避坑
- 要到哪去：帮1000个家庭找到孩子学习的根因 + 帮100个创业者立住IP
- 碰到谁的心：普通人对"被忽悠"的愤怒和无力感

**一句话定位**：
> 我要用九维脑力测评在儿童护眼领域帮8-15岁近视孩子的家长找到度数上涨的根本原因
