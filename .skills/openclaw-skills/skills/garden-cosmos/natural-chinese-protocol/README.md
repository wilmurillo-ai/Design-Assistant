# natural-chinese-protocol

A core protocol to eliminate AI traces in Chinese text generation. | 中文 AI 写作去痕与自然化表达核心协议。

> 自然中文协议 (Natural Chinese Protocol) 大模型时代的中文表达重塑白皮书

## 理念源起
大模型生成的中文常充斥着AI感，例如死板对称、显性连词泛滥、翻译腔严重。本协议是一套基于原生中文语感的认知过滤系统。核心使命是剥离机械痕迹，让文本逻辑、语体和流线性回归真实中文母语者的表达习惯。

## 四大核心原则

**一、视觉净化：去符号化与反模板**
严控表情符号（如 1️⃣、🔥）作为列表装饰，打破“大标题-列表-解释”的刻板线性推演，让文本视觉回归纯粹与严肃。

**二、气韵贯通：节奏重塑与流线性**
拒绝将长句强行切碎的干瘪“电报体”，恢复中文应有的延展度。通过长短句交错，消除为凑排比而生造的对称结构。

**三、语境回归：破除翻译腔与零主语复位**
强制剔除无生命系统滥用代词“它”的恶习（恢复高语境下的“零主语”习惯），瓦解“带来了……的更新”等生搬硬套的英文句法，消除语体杂糅。

**四、意合之美：隐性逻辑与认知留白**
摒弃“值得注意的是”等画蛇添足的显性逻辑标记。剥离“无疑”、“绝对”等教科书式的确信感，保留真实认知的主观边界与留白。

## 使用指南

* **适用范围**：不受限于特定文体，广泛适用于所有通用中文写作与交流场景。
* **调用指令**：在 Prompt 或工作流中直接挂载声明：“请应用《自然中文协议》重写/生成以下内容……”
* **执行链路**：
  1. **界定场景**：锚定真实语境，防止跨文体生硬篡改。
  2. **病灶扫描**：侦测并抓取文本中的“翻译腔”、“过度显性逻辑”、“宏大叙事”等 AI 痕迹。
  3. **同频重塑**：在绝对保留原意的前提下，运用四大原则进行语感重铸。
