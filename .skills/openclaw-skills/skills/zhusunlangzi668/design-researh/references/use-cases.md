# Common Use Cases

## 1. 活动宣传页或专题页

优先看：

- 传播目标
- 受众注意力争夺点
- 首屏视觉张力
- 信息层级和转化动作

适合收集的参考：

- 同行业活动页
- 时尚和媒体专题页
- 强叙事首屏案例

## 2. 课程内容包装或讲义视觉

优先看：

- 内容理解门槛
- 知识结构可视化
- 学习氛围和亲和度
- 长内容阅读疲劳

适合收集的参考：

- 教育内容编排
- 信息图表风格
- 卡片化知识展示

## 3. 品牌传播物料

优先看：

- 品牌一致性
- 语气和气质
- 传播主题放大方式
- 系列化延展性

适合收集的参考：

- 品牌 campaign
- 社媒视觉系列
- KV 体系案例

## 4. 交互或产品引导场景

优先看：

- 用户任务场景
- 认知负担
- 关键路径
- 反馈与状态表达

适合收集的参考：

- onboarding
- 空状态和引导
- 信息层级强的产品案例

## 5. 动效或氛围探索

优先看：

- 节奏感
- 信息出现顺序
- 情绪调动方式
- 实现复杂度

适合收集的参考：

- 品牌片头
- UI motion
- 社媒短视频包装
