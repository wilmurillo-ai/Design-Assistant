# Example Scenario: 活动专题页前期研究

## 示例输入

我们要做一个面向家长用户的春季主题课程活动专题页，目标是提升课程咨询和试听报名。现在只有一版比较粗的 brief：希望整体感觉有春天、生长、陪伴感，不要太幼稚，也不要太像传统教育广告。页面可能会包含主视觉、课程亮点、老师介绍、学员反馈和报名引导。领导希望先看前期方向，不要直接出 final 视觉。

## 理想输出重点

### 1. 需求翻译

这不是单纯做一个“好看”的活动页，而是要在“教育可信度”和“春季情绪感”之间找到平衡，让家长既感到专业可信，又愿意继续了解课程并产生报名动作。当前最需要先回答的问题是：首页应该更偏品牌感、内容说服感，还是活动氛围感。

### 2. 已知 / 未知

已知：

- 面向家长用户
- 目标是咨询和试听报名
- 希望有春季、生长、陪伴感
- 不想太幼稚，也不想像传统教育广告

未知：

- 核心年龄段是否明确
- 是高客单课程还是普适型课程
- 品牌主色和禁用风格是否有限制
- 报名转化最关键的信息点是什么

### 3. 关键调研问题

- 家长最在意的是课程效果、老师资质，还是孩子体验
- 同类教育活动页最常见的说服结构是什么
- 春季主题在教育场景里常见的视觉表达是否已经同质化
- 首页更适合先打情绪，还是先打课程价值
- 用户更容易被真实课堂感打动，还是被品牌感包装打动

### 4. 灵感方向

方向 A：自然生长型

- 关键词：春芽、留白、柔和层次、陪伴感
- 适合参考：生活方式品牌、亲子内容、轻教育品牌
- 风险：容易偏软，课程说服力可能不足

方向 B：内容可信型

- 关键词：清晰结构、学习成果、专业信息层级
- 适合参考：知识产品页、课程介绍页、功能型专题页
- 风险：可能偏理性，春季氛围不够鲜明

方向 C：品牌故事型

- 关键词：春日叙事、场景化情绪、人物陪伴
- 适合参考：品牌 campaign、长专题页、轻叙事活动页
- 风险：如果内容组织弱，容易只有氛围没有转化力

### 5. 建议下一步

建议先做 moodboard，而不是直接做页面框架。当前最大风险不是信息缺失本身，而是方向感还没有收敛。先确认视觉和叙事基调，后面的结构草案会更稳。

## 这个示例适合说明什么

- 这个技能不会直接开始画设计稿，而是先把问题拆清楚
- 这个技能会同时给研究问题、灵感方向和下一步动作
- 这个技能适合在“先给我看前期思路”的阶段快速产出结构化结果
