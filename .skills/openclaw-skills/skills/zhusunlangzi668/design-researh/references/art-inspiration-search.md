# 美术灵感图搜索指引

当用户说“找美术灵感图”“找画风参考”“找国际插画感”“找绘本感参考”时，用这份规则。

## 搜索目标

优先找这些内容：

- 画风
- 色彩气质
- 笔触和材质
- 构图节奏
- 情绪氛围
- 叙事感

不要把重点放在这些内容上：

- 纯 UI 截图
- 普通官网截图
- 品牌介绍页
- 官方产品页

## 默认站点

只在下面四个站点里找：

- Behance
- 站酷
- 花瓣
- 小红书

每次搜索都要显式覆盖这四个站点，不能漏掉其中任何一个。

## 推荐关键词包

按需求方向组合使用主题词、风格词、场景词和站点词，不要只搜一个总词。

### 绘本感

- 绘本风 插画 花瓣
- picture book illustration Behance
- children editorial illustration Behance
- 儿童 插画 站酷
- 绘本风格 插画 小红书

### 高级艺术感

- editorial illustration Behance
- art direction poster illustration 站酷
- 色彩 氛围 插画 花瓣
- narrative illustration Behance
- 艺术感 插画 配色 小红书

### 教育场景可借鉴

- education illustration Behance
- 教育 插画 站酷
- 儿童 学习 海报 花瓣
- learning visual illustration Behance
- 教育 视觉设计 儿童插画 小红书

## 筛选标准

优先保留：

- 一眼能看出画风完成度高的项目
- 色彩语言明确的项目
- 构图有故事感的项目
- 有系列感和统一视觉语言的项目
- 经 `references/quality-gate.md` 判断后仍明显强于其他候选的项目

直接淘汰：

- 只有一张封面的项目
- 只有排版，没有明确美术表达的页面
- 质量一般，只是关键词碰巧匹配的结果
- 和已选结果高度重复的结果

## 返回方式

最好按 3 类整理：

- 偏绘本叙事
- 偏高级艺术感
- 偏教育场景可借鉴

每类给 3-5 个结果即可，不要堆太多。

每个结果都补一句质量判断，说明为什么值得保留。

返回时必须写出四站覆盖情况：

- Behance：找到强结果 / 只有弱结果 / 没有可用结果
- 站酷：找到强结果 / 只有弱结果 / 没有可用结果
- 花瓣：找到强结果 / 只有弱结果 / 没有可用结果
- 小红书：找到强结果 / 只有弱结果 / 没有可用结果
