# @ref_realestate_commercial - 商业地产

## 画面基调
高端商务，投资价值，城市地标，突出商业潜力和升值空间

## 核心元素比例
建筑外观展示(40%)、商业布局(30%)、投资价值(20%)、政策信息(10%)

## 运镜风格
大气运镜+城市视角，航拍镜头，空间展示，突出规模感

## 色调
商务灰色调，金色点缀，营造高端商务印象

## 音效
商务BGM+城市音效，脚步声，商务洽谈声，营造专业氛围

## 合规要点
- 预售许可证号必须显著标注
- 建筑面积/套内面积必须区分标注
- 不得承诺投资回报率
- 位置信息必须准确
- 周边配套设施描述必须真实
