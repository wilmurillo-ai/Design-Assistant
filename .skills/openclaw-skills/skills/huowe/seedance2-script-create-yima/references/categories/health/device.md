# @ref_health_device - 医疗器械

## 画面基调
专业可靠，科技医疗，安全可信，突出专业性和安全性

## 核心元素比例
产品专业展示(40%)、使用演示(30%)、安全信息(20%)、认证标识(10%)

## 运镜风格
专业运镜+科学演示，稳定镜头，细节特写，突出专业感

## 色调
医疗蓝白色调，专业冷色，营造可信赖印象

## 音效
专业BGM+医疗音效，仪器运行声，安全提示声，营造专业氛围

## 合规要点
- 注册证号必须显著标注
- 适用范围和禁忌症必须明确
- "请在医生指导下使用"必须标注
- 不得宣传治疗效果
- 产品名称必须与注册证一致
