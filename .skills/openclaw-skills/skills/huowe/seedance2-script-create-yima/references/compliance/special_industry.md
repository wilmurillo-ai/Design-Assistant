# 特殊行业合规要求

## 概述

根据中国广告法，以下特殊行业有额外的合规要求和强制标注义务。

## 最新更新 (2024年)

### 新增行业监管
- **互联网服务**：APP、小程序、网站等数字服务
- **教育科技**：在线教育、AI教育产品
- **跨境电商**：进口商品、海外品牌
- **直播电商**：主播带货、直播营销
- **算法推荐**：个性化推荐广告

### 加强监管行业
- **医疗健康**：进一步规范医疗广告
- **金融科技**：加强FinTech广告监管
- **保健食品**：完善功效宣称管理

## 行业合规表

| 行业 | 强制标注 | 禁止内容 | 其他要求 |
|-----|---------|---------|---------|

| **金融服务** | "投资有风险，入市需谨慎" | 保本保收益、预期收益率、零风险承诺 | 需明确标注风险等级、投资期限 |
| **教育培训** | "效果因人而异" | 保过、保分、包就业、100%通过率 | 需标注师资资质、课程大纲 |
| **房地产** | 预售许可证号、建筑面积/套内面积 | 投资回报承诺、学区房承诺、升值保证 | 必须提供预售证号、面积说明 |
| **医疗器械** | 注册证号、禁忌内容、注意事项 | 治愈率、有效率、替代医生建议 | 必须标注适用范围、禁忌症 |
| **化妆品** | 备案号、成分表、生产许可证号 | 医疗作用、绝对功效、治疗疾病描述 | 必须标注全成分、使用期限 |

| **互联网服务** | "广告"字样、数据使用声明 | 虚假用户评价、刷单数据 | 必须标注数据收集使用目的 |

## 详细要求

### 1. 保健食品

#### 强制标注
- **主标注**："保健食品不是药物，不能代替药物治疗疾病"
- **标识要求**：必须显示蓝帽子标识（保健食品标志）
- **备案信息**：必须标注备案号或批准文号

#### 禁止内容
- 任何治疗疾病或预防疾病的描述
- 使用医疗术语或医生形象
- 暗示替代药物或治疗效果

#### 示例
```
✅ 合规：本品为保健食品，具有辅助调节功能
❌ 违规：本品能治疗高血压，效果显著
```

### 2. 金融服务

#### 强制标注
- **风险提示**："投资有风险，入市需谨慎"
- **产品信息**：必须明确标注产品类型、风险等级
- **收益说明**：历史收益不代表未来表现

#### 禁止内容
- 保本保收益承诺
- 预期收益率或回报率承诺
- "零风险"、"稳赚不赔"等表述

#### 示例
```
✅ 合规：本产品为中风险，过往业绩不预示未来表现
❌ 违规：年化收益8%，保本保息，零风险
```

### 3. 教育培训

#### 强制标注
- **效果说明**："效果因人而异"
- **资质公示**：必须标注机构资质、教师资质
- **课程说明**：明确标注课程内容、课时安排

#### 禁止内容
- "保过"、"保分"、"包就业"承诺
- 100%通过率或成功率
- 虚假成功案例或学员评价

#### 示例
```
✅ 合规：本课程帮助学员提升技能，效果因人而异
❌ 违规：参加本课程保证通过考试，不过退款
```

### 4. 房地产

#### 强制标注
- **证件信息**：预售许可证号（必须）
- **面积说明**：建筑面积/套内面积（必须区分）
- **位置信息**：准确的地理位置描述

#### 禁止内容
- 投资回报率或升值承诺
- 学区房承诺（除非有官方文件）
- 虚假交通、配套设施描述

#### 示例
```
✅ 合规：预售证号：X预字(2024)001号，建筑面积90㎡
❌ 违规：投资本项目，年回报率15%，学区房保证
```

### 5. 医疗器械

#### 强制标注
- **注册信息**：医疗器械注册证号
- **安全信息**：禁忌内容、注意事项、警示说明
- **适用范围**：明确的适用人群和病症范围

#### 禁止内容
- 治愈率、有效率数据
- 替代医生诊断或治疗建议
- 夸大适用范围或效果

#### 示例
```
✅ 合规：注册证号：国械注准2024XXXX，请在医生指导下使用
❌ 违规：使用本仪器，颈椎病治愈率95%
```

### 6. 化妆品

#### 强制标注
- **备案信息**：化妆品备案号
- **成分信息**：全成分表
- **生产信息**：生产许可证号、使用期限

#### 禁止内容
- 医疗作用或治疗疾病描述
- "绝对功效"、"神奇效果"等表述
- 使用医疗术语或专家背书

#### 示例
```
✅ 合规：备案号：国妆备进字J2024XXXX，限期使用日期见包装
❌ 违规：本产品能祛斑治疗皮肤病，效果显著
```

### 7. 母婴产品

#### 强制标注
- **年龄范围**：必须明确标注适用年龄范围（如"6个月以上"）
- **安全标准**：必须标注符合国家安全标准
- **材质成分**：必须标注主要材质成分

#### 禁止内容
- "绝对安全"、"宝宝专用"等绝对化表述
- 暗示产品能促进智力发育或身体发育
- 使用"天然无害"等未经证实的表述

#### 示例
```
✅ 合规：适用年龄：6-36个月，符合GB6675安全标准，主要材质：棉
❌ 违规：绝对安全，宝宝专用，促进智力发育，天然无害
```

### 8. 互联网服务 (2024新增)

#### 强制标注
- **广告标识**："广告"字样（算法推荐广告必须）
- **数据声明**："本广告基于您的兴趣推荐"
- **服务信息**：必须标注服务提供商、联系方式

#### 禁止内容
- 虚假用户评价或下载量
- 刷单或虚假数据
- 夸大功能或效果

#### 示例
```
✅ 合规：广告 - 本应用提供在线服务，数据仅用于个性化推荐
❌ 违规：下载量突破1000万，评分5星，用户口碑爆棚
```

### 9. 直播电商 (2024新增)

#### 强制标注
- **广告声明**：主播必须口头声明"这是广告"
- **产品信息**：必须展示真实价格和优惠
- **售后保障**：明确标注退换货政策

#### 禁止内容
- 虚假销量数据
- 夸大优惠幅度
- 虚假用户互动

#### 示例
```
✅ 合规：主播说"这是广告"，展示真实优惠券和售后政策
❌ 违规：直播间人气爆棚，销量突破万单，限时优惠99折
```

### 10. 跨境电商 (2024新增)

#### 强制标注
- **进口标识**：清晰标注"进口商品"
- **原产国信息**：准确标注原产国和进口商
- **质量标准**：标注符合中国质量标准

#### 禁止内容
- 虚假进口资质
- 伪造国际认证
- 虚假原产地宣传

#### 示例
```
✅ 合规：原产国：美国，进口商：XXX公司，符合中国质量标准
❌ 违规：正品进口，美国直邮，原装正品，国际品质
```

## 执行要求

### 标注位置
1. **视频广告**：必须在画面显著位置显示，持续时间不少于3秒
2. **图文广告**：必须在主要位置标注，字号不小于正文
3. **音频广告**：必须在广告结尾清晰播报
4. **直播广告**：主播必须口头声明，字幕同步显示
5. **算法推荐**：在推荐内容旁显著标注"广告"

### 验证流程
1. **自动检测**：系统根据产品分类自动添加相应标注
2. **人工审核**：特殊行业广告必须经过人工合规审核
3. **最终验证**：输出前验证所有强制标注是否齐全

### 违规处理
- **一级违规**：缺少强制标注，禁止发布
- **二级违规**：标注不完整，要求修改
- **三级违规**：标注位置不显著，要求调整
