# @ref_compliance_check - 合规检查器

## 概述

广告法合规检查系统，用于确保所有广告内容符合中国广告法规要求。

## 最新更新 (2024年)

### 法规更新
- **《广告法》修订**：2024年最新修订案正式实施
- **《互联网广告管理暂行办法》**：加强互联网广告监管
- **《医疗广告管理办法》**：进一步规范医疗广告
- **《保健食品广告审查办法》**：完善保健食品广告审查

### 新增要求
- **算法推荐广告**：必须标注"广告"字样
- **直播带货**：主播需明确声明广告性质
- **短视频广告**：所有时长必须标注"广告"
- **数据合规**：涉及用户数据必须获得明确同意

## 检查流程

1. **违禁词检查** - 检查一级违禁词和二级限制词
2. **特殊行业验证** - 验证特殊行业的强制标注要求
3. **合规替换** - 自动替换违禁词为合规词汇
4. **新媒体合规** - 检查互联网广告特殊要求

## 使用方法

在广告脚本生成过程中，自动调用合规检查：

```markdown
【合规引用】@ref_compliance_check
```

## 检查规则

### 1. 违禁词检查
- 自动扫描广告文案中的一级违禁词
- 提示二级限制词需要人工验证
- 自动应用合规替换词库

### 2. 特殊行业标注
- 根据产品分类自动添加强制标注
- 验证标注格式和位置是否符合要求
- 确保禁止内容不被使用

### 3. 互联网广告要求 (2024新增)
- **短视频平台**：所有时长必须标注"广告"
- **直播电商**：主播必须声明"这是广告"
- **算法推荐**：必须明确标识广告内容
- **数据使用**：涉及用户画像必须获得同意

### 4. 输出验证
- 最终输出必须包含合规声明
- 特殊行业必须显示强制标注
- 违禁词检查状态必须明确
