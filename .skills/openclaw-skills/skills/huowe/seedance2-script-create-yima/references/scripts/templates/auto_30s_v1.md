## 注意事项

1. **时间分配**：30秒允许深入展示，确保信息层次清晰
2. **细节丰富**：充分利用时间展示产品细节和功能
3. **场景多样**：展示多种驾驶场景，增加说服力
4. **合规完整**：合规信息必须完整且清晰显示
5. **品牌故事**：适当融入品牌故事，增强品牌记忆
6. **工况标注**：涉及油耗/续航数据时，必须标注NEDC或CLTC工况
