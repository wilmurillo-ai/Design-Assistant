**运镜技巧**：
- 产品360度旋转

- 功能特写镜头

### 8-12秒：场景应用 (Scenario Application)

- 场景环境音（办公室、家庭、户外）

**运镜技巧**：

## 特殊合规要求

### 电子产品合规要点
1. **参数准确性**：所有技术参数必须准确，不得夸大
2. **功能真实性**：演示功能必须真实可用，不得虚构
3. **对比公正性**：与其他产品对比必须公正客观
4. **认证标识**：必要的安全认证标识必须显示

### 禁止内容
- "最快"、"最强"、"最佳"等绝对化描述
- 未经证实的性能数据
- 误导性的功能演示
- 虚假的用户评价或测试结果
