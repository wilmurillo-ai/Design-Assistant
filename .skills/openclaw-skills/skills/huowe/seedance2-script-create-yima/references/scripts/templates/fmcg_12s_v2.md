## 场景详细说明

### 0-3秒：开场钩子 (Hook)
**目的**：建立用户需求共鸣，引发兴趣
**视觉元素**：
- 问题场景呈现
- 情感共鸣画面
- 快速引入产品概念
**音频元素**：
- 场景环境音
- 疑问或需求的音效
- BGM渐起
**运镜技巧**：
- 场景建立镜头
- 人物引入
- 问题特写

### 3-7秒：产品展示 (Product)
**目的**：介绍产品核心功能
**视觉元素**：
- 产品核心展示
- 使用过程演示
- 功能特点视觉化
**音频元素**：
- 产品使用音效
- 功能解说
- BGM持续
**运镜技巧**：
- 产品特写
- 使用过程跟拍
- 功能展示

### 7-10秒：利益展示 (Benefit)
**目的**：展示产品带来的价值
**视觉元素**：
- 使用前后对比
- 用户满意反应
- 效果可视化
**音频元素**：
- 效果音效
- 用户赞叹声
- BGM高潮
**运镜技巧**：
- 对比展示
- 用户反应捕捉
- 效果特写

### 10-12秒：行动号召 (CTA)
**目的**：促使用户行动，强化品牌记忆
**视觉元素**：
- 品牌标识展示
- 购买指引
- 最终形象
**音频元素**：
- BGM高潮
- 号召性口号
**合规标注**：
- 显示合规信息
