## 场景详细说明

### 0-3秒：动态开场 (Dynamic Hook)
**目的**：建立驾驶氛围，吸引汽车爱好者
**视觉元素**：
- 动态驾驶场景
- 汽车流畅登场
- 性能特点视觉化
**音频元素**：
- 引擎声浪
- 驾驶音乐
**运镜技巧**：
- 跟车拍摄
- 动态引入

### 3-8秒：产品展示 (Vehicle Display)
**目的**：展示汽车外观和内饰特点
**视觉元素**：
- 外观设计展示
- 内饰细节特写
- 核心功能演示
**音频元素**：
- 驾驶音效
- BGM增强
**运镜技巧**：
- 360度旋转
- 内饰跟拍
- 功能特写

### 8-12秒：驾驶体验 (Driving Benefit)
**目的**：展示驾驶体验和性能优势
**视觉元素**：
- 实际驾驶场景
- 性能优势表现
- 安全舒适体现
**音频元素**：
- 驾驶环境音
- 性能音效
- BGM高潮
**运镜技巧**：
- 驾驶跟拍
- 场景切换
- 体验特写

### 12-15秒：行动号召 (CTA)
**目的**：促使用户行动，强化品牌记忆
**视觉元素**：
- 品牌标识展示
- 购买指引清晰
- 汽车最终形象
**音频元素**：
- BGM完美收尾
- 品牌音效
**合规标注**：
- 完整显示合规信息

## 注意事项

1. **时间分配**：合理分配15秒时间，确保每个环节充分展示
2. **信息层次**：信息呈现要有逻辑层次，从吸引到行动
3. **情感递进**：情感从驾驶渴望到满足递进
4. **合规完整**：15秒足够展示完整合规信息，必须包含
5. **品牌强化**：品牌信息要在多个环节重复强化
6. **行动明确**：购买指引要清晰明确，易于操作
7. **工况标注**：涉及油耗/续航数据时，必须标注NEDC或CLTC工况
