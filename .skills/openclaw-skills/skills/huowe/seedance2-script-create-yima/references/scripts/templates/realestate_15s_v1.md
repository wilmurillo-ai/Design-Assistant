## 注意事项

1. **时间分配**：合理分配15秒时间，确保每个环节充分展示
2. **信息层次**：信息呈现要有逻辑层次，从吸引到行动
3. **情感递进**：情感从理想向往到满足递进
4. **合规完整**：15秒足够展示完整合规信息，必须包含
5. **品牌强化**：品牌信息要在多个环节重复强化
6. **行动明确**：咨询指引要清晰明确，易于操作
