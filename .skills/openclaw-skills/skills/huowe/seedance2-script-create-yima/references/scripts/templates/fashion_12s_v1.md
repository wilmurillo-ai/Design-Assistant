## 场景详细说明

### 0-2秒：时尚开场 (Fashion Hook)
**目的**：建立时尚氛围，吸引目标消费者
**视觉元素**：
- 模特优雅姿态
- 时尚元素流动
- 品牌调性展示
**音频元素**：
- 时尚BGM进入
- 轻盈脚步声
- 环境氛围音
**运镜技巧**：
- 时尚运镜
- 模特走秀
- 元素特写

### 2-6秒：产品展示 (Product Display)
**目的**：展示产品细节和工艺特点
**视觉元素**：
- 产品细节特写
- 材质工艺展示
- 设计亮点呈现
**音频元素**：
- 时尚音效
- 材质触感音
- BGM节奏增强
**运镜技巧**：
- 产品特写
- 细节展示
- 工艺跟拍

### 6-9秒：魅力体现 (Charm Benefit)
**目的**：展示穿搭效果和时尚魅力
**视觉元素**：
- 穿搭整体效果
- 时尚魅力表现
- 场合适用展示
**音频元素**：
- 满意反馈音
- 时尚氛围音
- BGM高潮
**运镜技巧**：
- 整体造型
- 场景展示
- 魅力捕捉

### 9-12秒：品牌号召 (Brand CTA)
**目的**：强化品牌时尚形象，引导购买
**视觉元素**：
- 品牌时尚标识
- 产品系列展示
- 购买渠道指引
**音频元素**：
- BGM高潮收尾
- 品牌时尚音效
- 行动号召语音
**合规标注**：
- 材质成分标注
- 尺码信息提示

## 注意事项

1. **时间分配**：合理分配12秒时间，确保时尚感充分展现
2. **材质标注**：必须标注面料成分和保养方式
3. **尺码提示**：提供尺码参考和试穿建议
4. **效果免责**：标注"实际效果因人而异"
5. **合规标注**：12秒内必须展示完整材质信息
6. **时尚氛围**：整体风格保持优雅时尚调性
