## 场景详细说明

### 0-2秒：快速科技开场 (Quick Tech Hook)
**目的**：快速吸引科技用户注意力
**视觉元素**：
- 产品快速浮现
- 科技元素闪现
- 创新点瞬间展示
**音频元素**：
- 快速电子音效
- BGM急促进入
**运镜技巧**：
- 快速光线追踪
- 产品闪现动画

### 2-5秒：核心功能演示 (Core Demo)
**目的**：快速展示产品核心价值
**视觉元素**：
- 关键功能操作
- 产品细节特写
**音频元素**：
- 操作音效
- 功能提示音
**运镜技巧**：
- 快速旋转
- 功能跟拍

### 5-7秒：效果展示 (Effect Show)
**目的**：展示使用效果
**视觉元素**：
- 效果快速呈现
- 效率提升表现
**音频元素**：
- 效果音效
- BGM高潮
**运镜技巧**：
- 快速切换
- 效果特写

### 7-9秒：行动号召 (CTA)
**目的**：促使用户行动
**视觉元素**：
- 品牌展示
- 购买指引
**音频元素**：
- BGM收尾
- 品牌音效
**合规标注**：
- 快速显示合规信息

## 注意事项

1. **节奏快速**：9秒时间紧凑，信息传递要高效
2. **重点突出**：突出产品最核心的卖点
3. **视觉冲击**：利用科技感创造视觉冲击
4. **合规完整**：合规信息必须完整显示
5. **平台适配**：针对抖音、快手等短视频平台优化
