## 场景详细说明

### 0-3秒：温馨开场 (Cozy Hook)
**目的**：建立温馨家居氛围，引发舒适向往
**视觉元素**：
- 温馨家居环境
- 产品自然融入场景
- 舒适生活片段
**音频元素**：
- 温馨背景音乐
- 轻柔环境音
**运镜技巧**：
- 场景平移
- 产品柔和引入

### 3-8秒：产品展示 (Product Display)
**目的**：详细展示产品特点和品质
**视觉元素**：
- 产品材质特写
- 功能特点演示
- 细节工艺展示
**音频元素**：
- 产品操作音效
- BGM持续
**运镜技巧**：
- 多角度展示
- 材质质感特写
- 功能演示跟拍

### 8-12秒：舒适体验 (Comfort Benefit)
**目的**：展示产品带来的舒适体验
**视觉元素**：
- 实际使用场景
- 舒适体验表现
- 生活品质提升
**音频元素**：
- 场景环境音
- 用户满意声
- BGM增强
**运镜技巧**：
- 场景切换
- 用户互动
- 体验特写

### 12-15秒：行动号召 (CTA)
**目的**：促使用户行动，强化品牌记忆
**视觉元素**：
- 品牌标识展示
- 购买指引清晰
- 温馨家居形象
**音频元素**：
- BGM完美收尾
- 品牌音效
**合规标注**：
- 完整显示合规信息

## 注意事项

1. **时间分配**：合理分配15秒时间，确保每个环节充分展示
2. **信息层次**：信息呈现要有逻辑层次，从吸引到行动
3. **情感递进**：情感从舒适向往到满足递进
4. **合规完整**：15秒足够展示完整合规信息，必须包含
5. **品牌强化**：品牌信息要在多个环节重复强化
6. **行动明确**：购买指引要清晰明确，易于操作
