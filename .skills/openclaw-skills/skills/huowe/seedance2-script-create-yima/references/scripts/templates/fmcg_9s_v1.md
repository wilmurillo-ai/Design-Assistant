## 场景详细说明

### 0-3秒：开场钩子 (Hook)
**目的**：在3秒内吸引用户注意力
**视觉元素**：
- 产品惊艳亮相（从黑暗中浮现、旋转出现等）
- 核心卖点视觉化展示
- 快速剪辑，节奏感强
**音频元素**：
- 简短有力的开场音效（0.5-1秒）
- 快速切入主BGM
**运镜技巧**：
- 快速推近（zoom in）
- 旋转展示（product rotation）
- 动态转场

### 3-6秒：产品展示 (Product)
**目的**：详细展示产品特点和功能
**视觉元素**：
- 产品特写（材质、细节、颜色）
- 功能演示（使用过程、效果对比）
- 多角度展示
**音频元素**：
- 产品使用音效（开启声、涂抹声、咬合声等）
- BGM持续，节奏适中
**运镜技巧**：
- 微距特写（macro shot）
- 平移跟踪（dolly shot）
- 焦点转换（rack focus）

### 6-9秒：行动号召 (CTA)
**目的**：促使用户采取行动，强化品牌印象
**视觉元素**：
- 产品在使用场景中的效果
- 品牌标识或slogan展示
- 愉悦的使用者表情或场景
**音频元素**：
- BGM达到高潮后收尾
- 品牌专属音效（如果有）
- 清晰的结束音
**合规标注**：
- 画面角落添加必要标注
- 标注显示时间不少于2秒

## 注意事项

1. **时长控制**：严格控制在9秒内，每个场景时间分配精确
2. **节奏把握**：前3秒要足够吸引人，中间展示要清晰，结尾要有力
3. **合规优先**：必须添加必要的合规标注
4. **品牌一致性**：保持品牌调性和视觉风格一致
5. **平台适配**：针对短视频平台（抖音、快手、小红书）优化
