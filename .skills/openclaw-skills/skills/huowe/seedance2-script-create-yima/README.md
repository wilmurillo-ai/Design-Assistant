# Advertising Skill for Seedance 2.0

一个 Claude Code Skill，用于生成即梦（Seedance 2.0）广告视频的脚本和提示词，内置广告法合规检查。

## 能做什么

输入产品信息和需求，自动输出：

- 分镜脚本（4段结构：钩子 → 产品展示 → 效果 → CTA）
- 可直接粘贴到即梦的视频生成提示词
- 配套的图片生成提示词

支持 9s / 12s / 15s / 30s 四种时长，覆盖 25 个行业分类。

## 合规检查

生成前自动扫描违禁词（如"第一"、"最佳"、"神器"等），并替换为合法表达。保健品、金融、教育、房产等特殊行业自动添加必要免责声明。

## 使用方式

将本项目放入 `.claude/skills/` 目录，在 Claudecode 中描述你的产品即可触发，例如：

```
帮我做一个15秒的护肤品广告，主打保湿，目标用户25-35岁女性
```
