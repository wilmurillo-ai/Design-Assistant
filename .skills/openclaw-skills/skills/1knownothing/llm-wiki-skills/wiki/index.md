## Theme Overview

This wiki focuses on **Mechanistic Interpretability** - the field of understanding how AI models compute their outputs. Key themes include:

1. **Feature Extraction**: Using sparse autoencoders to find interpretable features in model activations
2. **Circuit Analysis**: Understanding how features interact to produce model behavior
3. **Safety Applications**: Using interpretability techniques for AI safety
