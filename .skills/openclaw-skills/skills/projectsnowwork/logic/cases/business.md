# Case — Business: Why a Seemingly Good Product Is Not Growing

## Situation
A product has strong internal enthusiasm, positive user interviews, and a polished feature set, but growth is weak.

## Structural Reduction

### Objective
Find the real constraint on growth.

### Constraints
- limited budget
- limited time
- product already built
- user acquisition not infinite

### Possible Surface Explanations
- weak marketing
- poor branding
- bad content
- wrong channels

### Governing Questions
- is demand real and recurring?
- does the product solve a painful enough problem?
- where does the value loop break?
- is acquisition failing, activation failing, or retention failing?

## Mechanism
Growth usually fails because one of these breaks:
1. not enough pull
2. weak activation
3. low repeat value
4. bad distribution fit

## Load-Bearing Variables
- problem severity
- time-to-value
- repeat usage trigger
- distribution efficiency

## Recommendation Structure
Do not start with more marketing.
First isolate whether the real break is:
- demand
- activation
- retention
- channel fit

## Fragile Assumption
The most dangerous assumption is: “users like it” means “users need it enough to return or tell others.”

## Lesson
Growth analysis should begin with the broken mechanism in the value loop, not with visible promotion tactics.
