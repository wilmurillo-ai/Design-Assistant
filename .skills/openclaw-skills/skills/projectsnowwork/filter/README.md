# Filter

Filter is a reusable rule-based filtering skill for OpenClaw workflows.

## What it can do
- create reusable filter rules
- store local rules for repeated use
- support priority-based filtering
- support allowlists and blocklists
- prepare rule sets for future filtering workflows

## What it can filter
- email
- news
- search results
- structured task streams
- JSON-like data

## Typical scenarios
- reduce inbox noise
- clean up noisy news feeds
- narrow down search results
- keep only high-priority items
- create reusable filtering rules for recurring workflows
