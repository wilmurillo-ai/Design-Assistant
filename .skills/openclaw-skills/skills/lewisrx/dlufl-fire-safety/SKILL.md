# 大连外国语大学消防安全制度技能

## 技能简介

本技能整合了大连外国语大学消防安全相关的四份核心制度文件，用于回答关于学校消防安全工作的各类问题。

## 源文件

- 《大连外国语大学消防安全综合管理办法（试行）》（2026年1月） - 纲领性文件
- 《大连外国语大学消防安全宣传教育与培训实施细则》 - 教育培训专项
- 《大连外国语大学消防监督检查工作手册（试行）》 - 监督检查专项
- 《大连外国语大学火灾事故应急预案汇编（2026版）》 - 应急预案

## 核心架构

### 一、责任体系（四级责任体系）

| 层级 | 责任主体 | 主要职责 |
|------|----------|----------|
| 学校 | 党委书记、校长（组长） | 全面领导，决策重大事项 |
| 职能部门 | 党政办、教务处、后勤处等 | 管业务必须管安全 |
| 二级单位 | 学院/部门党政负责人 | 主体管理责任 |
| 个人 | 师生员工 | 个人义务 |

### 二、关键数字速查

| 项目 | 要求 |
|------|------|
| 新生消防安全教育 | ≥4学时 |
| 教职工培训 | 每年至少1次，覆盖率≥90% |
| 校级演练 | 每学年至少1次 |
| 二级单位演练 | 每学期至少1次 |
| 重点部位演练 | 每半年至少1次 |
| 学生演练参与率 | ≥90% |
| 学校层面检查 | 每季度1次 |
| 保卫处检查 | 每月1次 |
| 二级单位自查 | 每月至少1次 |
| 重点部位巡查 | 每日防火巡查 |
| 宿舍夜间巡查 | 每晚不少于2次 |
| 食堂燃气自查 | 每月至少1次 |
| 油烟管道清洗 | 每季度至少1次 |
| 动火后观察期 | ≥30分钟 |

### 三、重大隐患判定

**直接判定（11种情形）**
1. 未建立消防安全责任体系
2. 学生宿舍/幼儿园设在地下室或半地下室
3. 人员密集场所违规使用易燃易爆危险品
4. 采用低于A级的彩钢夹芯板
5. 未按标准设置火灾自动报警系统等消防设施
6. 电动自行车（含电池）入楼
7. 私拉乱接电线、"飞线"充电
8. 未经消防审批擅自投入使用
9. 安全出口数量不足或被封堵
10. 门窗设置影响逃生的障碍物
11. 实验室违规存放危化品

**综合判定（11项要素，符合3条及以上）**
- 消防车道被占用
- 防火间距不足
- 消防设施损坏超过50%
- 消防控制室值班人员无证
- 安全出口被封堵
- 疏散指示标志损坏超过50%
- 防火门损坏超过20%
- 等等

### 四、动火作业"三色证"

| 等级 | 颜色 | 适用范围 | 审批层级 |
|------|------|----------|----------|
| 一级 | 红色 | 易燃易爆区域、重大危险源附近、人员密集场所核心区 | 施工单位→使用单位→后勤→保卫处→分管校领导 |
| 二级 | 黄色 | 非易燃爆区域的一般维修改造 | 施工单位→使用单位→后勤→保卫处 |
| 三级 | 蓝色 | 小型、低风险、短时作业 | 施工单位→使用单位→后勤 |

### 五、应急响应分级

| 级别 | 情形 | 启动主体 |
|------|------|----------|
| Ⅲ级（一般） | 单一房间≤50㎡，无伤亡，初起可控 | 事故发生单位 |
| Ⅱ级（较大） | 疏散≥50人/面积≥100㎡/重点部位 | 保卫处报告指挥部后启动 |
| Ⅰ级（重大） | 死亡/重伤/受困≥3人/多栋建筑 | 学校主要负责人 |

### 六、信息报告时限

- 15分钟内：事发单位口头向保卫处初报
- 30分钟内：保卫处向学校主要领导电话初报（较大及以上）
- 1小时内：形成初步书面报告
- 3小时内：书面续报详细情况
- 重大火情：一小时一报，处置结束后终报

### 七、"一票否决"情形

1. 发生亡人火灾事故
2. 发生两起及以上一般火灾且负有管理责任
3. 重大隐患挂牌督办后未按期整改
4. 瞒报、谎报、迟报火灾事故
5. 严重履职缺失
6. 其他严重违反法律法规情形

### 八、核心禁止行为

- 电动自行车（含电池）进入建筑物
- "飞线"充电
- 宿舍使用热得快、电饭锅等违章电器
- 堵塞消防通道、安全出口
- 违规动火作业
- 人员密集场所门窗设置防盗网（未设逃生口）

### 九、关键岗位

| 岗位 | 资质要求 |
|------|----------|
| 消防控制室值班人员 | 持证上岗（消防设施操作员） |
| 电工、焊工 | 特种作业操作证 |
| 危险化学品管理人员 | 专项安全培训 |

### 十、档案保存期限

- 每日巡查记录：≥1年
- 自查记录：≥3年
- 学校大检查记录：≥5年
- 培训演练资料：≥5年
- 建筑消防设施原始资料：永久

## 适用场景

- 回答关于学校消防安全制度的问题
- 查询责任体系、检查频次、审批流程
- 了解应急响应程序和报告时限
- 确认重大隐患判定标准
- 查找禁止性行为和处罚依据

## 注意事项

- 本技能基于2026年版本的制度文件
- 具体问题应结合最新制度和实际情况回答
- 涉及责任认定和处罚的，应以正式制度文本和调查结果为准
