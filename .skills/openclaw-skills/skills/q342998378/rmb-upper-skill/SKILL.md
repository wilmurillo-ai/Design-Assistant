# RMB Upper Skill

将数字金额转换为人民币大写字符串。

## 功能

- 支持整数和小数金额
- 支持负数（显示"负"前缀）
- 符合财务规范的人民币大写格式

## 示例

| 输入 | 输出 |
|------|------|
| 1023.5 | 壹仟零贰拾叁元伍角 |
| 1002003.01 | 壹佰万贰仟零叁元零壹分 |
| -12.3 | 负拾贰元叁角 |

## 使用

通过工具 `rmb_upper_tool` 调用，参数 `amount` 为数字金额字符串。
