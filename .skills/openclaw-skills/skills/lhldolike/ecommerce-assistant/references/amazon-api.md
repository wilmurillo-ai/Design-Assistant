## 推荐方案：公开数据聚合

使用以下免费数据源组合：

1. **CamelCamelCamel** - 历史价格数据 (API 需申请)
2. **Keepa** - 价格追踪 (免费 tier)
3. **Google Shopping API** - 产品搜索 (免费额度)
4. **RapidAPI 免费套餐** - 多种电商 API

## 快速启动

使用 RapidAPI 的免费套餐：
- Amazon Price Tracker
- Real-Time Amazon Data
- Product Search API

免费额度：通常 100-500 请求/月
