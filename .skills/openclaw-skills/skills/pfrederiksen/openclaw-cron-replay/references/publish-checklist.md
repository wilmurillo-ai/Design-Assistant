# Publish Checklist

Repo link:
- https://github.com/pfrederiksen/openclaw-cron-replay

Before publishing to ClawHub or GitHub:
- verify SKILL.md frontmatter is present
- verify the repo link is included
- verify there are no remote install commands in SKILL.md
- verify no secrets or private tokens appear
- verify the skill is read-only / replay-oriented and does not send messages
- verify prerequisites are explicit
