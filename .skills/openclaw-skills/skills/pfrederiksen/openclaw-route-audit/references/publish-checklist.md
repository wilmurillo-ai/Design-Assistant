# Publish Checklist

Repo link:
- https://github.com/pfrederiksen/openclaw-route-check

Before publishing to ClawHub or GitHub:
- verify SKILL.md frontmatter is present
- verify the repo link is included
- verify scripts are readable shell/python, no obfuscation
- verify no secrets or private tokens appear
- verify commands are read-only or clearly labeled when they modify config
- verify local paths are accurate for your OpenClaw setup
- verify the skill does not auto-send messages or auto-install packages
