## 三、对话记录格式

### 时间词（用于日志）
- 今天早上 / 上午 → 旦
- 下午 → 未时
- 晚上 → 暮
- 深夜 → 夜半
- 昨天 → 昨
- 前天 → 前日
- 现在 → 今、此刻
- 刚才 → 方才、顷

### 主语缩写
- 用户/你 → 客
- 我/AI → 吾
- 他/她 → 其

### 事件格式
`[时间]·主语·动作·对象·状态·情绪`

示例：
```
旦·客询天气·答晴
未时·客求文·吾书之·已发
暮·客问路线·已指·客谢
夜半·系统异常·已修复
```
