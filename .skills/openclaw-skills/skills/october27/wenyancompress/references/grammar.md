# 文言语法速查 · Grammar Cheatsheet

## 一、核心句式

### 判断句（是/不是）
- 「...者，...也」→ A者B也（A is B）
- 「乃」→ 就是
- 「非」→ 不是

示例：
```
今天天气好 → 今者，天朗气清也
AI是助手 → AI者，侍者之属也
```

### 叙述句
- 「...为...」→ ...做...
- 「...于...」→ 在/向/从...
- 「...以...」→ 用/因为/以...方式

### 疑问句
- 「...乎？」→ ...吗？
- 「...耶？」→ ...吗？（反问）
- 「...与？」→ ...吗？

### 否定句
- 「不」→ 不
- 「未」→ 还没
- 「非」→ 不是
- 「无」→ 没有
- 「勿」→ 不要

### 使动/意动
- 使动：使... → 让...
- 意动：以...为... → 认为...

## 二、虚词（语气词）

| 词 | 位置 | 作用 |
|----|------|------|
| 也 | 句末 | 判断/肯定 |
| 矣 | 句末 | 已然/感叹 |
| 哉 | 句末 | 感叹 |
| 乎 | 句末 | 疑问 |
| 邪/耶 | 句末 | 反问 |
| 焉 | 句末 | 兼词（于此）|
| 尔 | 句末 | 如此 |
| 耳 | 句末 | 不过如此 |
| 者 | 句中 | 停顿/提示 |
| 之 | 句中 | 的/取消句子独立性 |

## 六、比喻与典故

### 常见比喻
- 如虎添翼 → 锦上添花/如虎添翼
- 杯水车薪 → 无济于事
- 画饼充饥 → 空想
- 刻舟求剑 → 不知变通

### 四字格（成语式压缩）
用四字格压缩概念：
- 三思而行 → 谨慎
- 破釜沉舟 → 决心
- 卧薪尝胆 → 隐忍图强
- 望梅止渴 → 空想激励
- 草木皆兵 → 多疑
- 纸上谈兵 → 空谈

## 七、快速压缩公式

**对话 → 压缩格式**：
```
谁 + 做什么 + 结果 + 情绪
→ 吾/客 + 动词 + 受词 + 状态
```

**长句 → 压缩**：
```
删掉：的、了、着、啊、呢、呀
替换：什么→何、如何→怎
合并：并列词→顿号隔开
```

## 八、常用句式模板

```
请求：...可乎？/ ...可否？
回答：可/否/善/诺
询问：...如何？/ ...何如？
感叹：...哉！/ ...矣！
道歉：吾之过也
感谢：多谢/感君之恩
告别：后会有期/来日再会
```
