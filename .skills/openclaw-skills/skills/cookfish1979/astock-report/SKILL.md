## 量化情绪打分规则（6维，满分100）

| 维度 | 满分 | 区间得分 |
|------|------|---------|
| ①涨停数量 | 15 | ≥150→15, ≥100→12, ≥60→9, ≥30→5, ≥10→2, <10→0 |
| ②涨跌停比 | 15 | ≥8→15, ≥5→12, ≥3→9, ≥1.5→5, ≥0.5→2, <0.5→0；跌停=0时直接15分 |
| ③北向资金 | 20 | ≥100亿→20, ≥50→15, ≥20→10, ≥0→6; 流出≥-50→3, <-50→0 |
| ④主力资金（全市场） | 20 | ≥500亿→20, ≥200→16, ≥50→12, ≥0→8; -200→5, -500→2, <-500→0 |
| ⑤炸板率（炸板数/涨停数） | 15 | <15%→15（强封板）, <25%→12, <35%→8, <45%→5, ≥45%→3；无数据→⚠️暂缺 |
| ⑥沪深300 | 10 | ≥+2%→10, ≥+1%→8, ≥0%→6; -1%→4, <-1%→0 |

**综合评级**：≥85→🟢极强多头 | ≥68→🟡多头偏强 | ≥52→⚪中性震荡 | ≥36→🟠偏空谨慎 | <36→🔴极弱空头

**⚠️ 评分保护**：任意一项数据暂缺 → 整体等级显示"⚠️数据暂缺，无法评分"（不是0分，不计入）。

**市场风险偏好**（晚报专用，独立于情绪，纯分类）：
`两融比例 = 融资买入额 / 全市场成交额 × 100%`
<7% → 保守 | 7~11% → 中性 | >11% → 过热

**两融余额占比预警**（晚报专用）：
`两融余额占比 = 两融余额 / A股流通市值 × 100%`
>3.2% → ⚠️ 预警区（战略撤退） | >3.5% → 高危区（必须撤离）

⚠️ **数据日期一致性原则（关键）**：
- 两融余额、两融交易额：均为 T-1 日盘后更新，属昨日数据
- 全市场成交额、流通市值：T 日实时数据
- 比例行下方须注明分子和分母各自的日期，格式：
  ```
  分子日期：YYYY年MM月DD日（T-1日）
  分母日期：YYYY年MM月DD日
  ```
- `两融余额 / 流通市值`：分子分母均为 T-1 日数据
- `两融交易额 / 全市场成交额`：两融交易额为 T-1 日，成交额应取 T-1 日（若可获取），两者日期须保持一致
- 若 T 日流通市值或成交额无法获取，回退至 T-1 日数据，注明对应日期

## 数据源（Python脚本实现，SKILL.md只描述逻辑）

> ⚠️ **核心原则**：今日数据暂缺时，脚本返回 `None`，由 AI 第三步搜索补全；**禁止取昨日数据凑数**（数据不可比，违反可比性原则）。

| 数据项 | 正确来源 | 单位换算 |
|--------|----------|---------|
| 六大指数 | 腾讯 qt.gtimg.cn field[32]涨跌幅 | URL格式 `qt.gtimg.cn/q=sh000001,sz399001...` |
| 涨停家数 | akshare `stock_zt_pool_em(date=今天)` | — |
| 北向资金 | akshare `stock_hsgt_fund_flow_summary_em` | 列名 `成交净买额`（万元→÷10000=亿）；**今日数据18:00后才更新**，暂缺时返回 `None`，AI 搜索补全 |
| 两融余额 | akshare `stock_margin_account_info` | **融资余额 + 融券余额**（两列均已是亿元，无需换算）；今日未更新时返回 `None`，AI 搜索补全 |
| 融资买入额 | akshare `stock_margin_account_info` | 列名 `融资买入额 + 融券卖出额`（已是亿元）；今日为0时**不要**取昨日数据，由 AI 搜索补全 |
| 全市场成交额 | 沪市 `stock_sse_deal_daily` + 深市 `stock_szse_summary` + 北交所 `stock_bse_summary` | 沪市直接是**亿元**；深市**元÷1e8**；北交所**元÷1e8**；三者相加 |
| 流通市值 | 沪市 `stock_sse_deal_daily` + 深市 `stock_szse_summary` | 沪市直接是**亿元**；深市**元÷1e8**；两者相加 |
| 沪深300PE | 腾讯 sh000300 field[39] | 直接是数字（倍） |
| 中证全指PE | 腾讯 sh000985 field[39] | 直接是数字（倍） |
| 10年国债收益率 | akshare `bond_china_yield`（国债收益率曲线，10年列） | 返回%（如1.81） |
| 风险溢价 | 计算值 | `1/中证全指PE - 10年国债收益率`，单位% |
| 沪深300近5年历史分位 | akshare `stock_index_pe_lg(symbol='000300')` | 滚动市盈率近5年分位点估算 |
| 申万行业涨跌 | akshare `stock_sector_spot()`（无参数） | — |
| 全市场主力净流入 | ①妙想 MX API → ②akshare个股加总后备 | 妙想每日50次限额；妙想可用时显示"妙想数据"，不可用时返回 `None`，AI 搜索补全 |
| 行业主力资金流向 | 妙想 MX API（分两批，每批10行业） | 妙想不可用时返回空 |
