# 오픈소스 프로젝트 런칭 마케팅 플레이북

> 버전: 5.0 (최종판) | 저자: Iris (生姜iris)

## 핵심 철학

> 오픈소스의 본질: 개발자에게 영향력을 기여받고, B2B 고객에게 돈을 기여받는다 💰. SOP를 통해 오픈소스 런칭으로 수백~수천 개의 스타를 얻고 많은 SMB 리드를 확보할 수 있다. **오픈소스화 전에 비즈니스 모델을 준비하여 트래픽을 잡아라.**

## 전체 타임라인

| 단계 | 시기 | 핵심 태스크 |
|:---|:---|:---|
| 전략 기획 | T-4주 | 포지셔닝, 밸류 프로포지션, 비즈니스 모델 |
| 자산 준비 | T-3주 | GitHub Readme, 웹사이트, PR 기사, 문서 |
| 채널 준비 | T-2주 | KOL 아웃리치, Reddit 계정, 그룹 리스트 |
| 팀 분담 | T-1주 | 예산 승인, 태스크 할당, 커뮤니케이션 |
| 런칭 스프린트 | T-3일 | 카피 확인, 자동화 설정, 최종 체크 |
| 🚀 **런칭일** | T-0 | 멀티채널 집중 노출, 트래픽 플라이휠 |
| 지속 운영 | T+1~T+7 | 매일 100+ 스타, KOL Quote 폭격 |
