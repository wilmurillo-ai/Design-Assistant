# オープンソースプロジェクト・ローンチ・マーケティング・プレイブック

> バージョン: 5.0 (最終版) | 著者: Iris (生姜iris)

## コア理念

> オープンソースの本質：開発者に影響力を貢献してもらい、B2Bクライアントにお金を貢献してもらう💰。SOPを通じて、オープンソースローンチで数百〜数千のスターを獲得し、多くのSMBリードを得ることができる。**オープンソース化する前にビジネスモデルを準備し、トラフィックを捉えよう。**

## 全体タイムライン

| フェーズ | タイミング | コアタスク |
|:---|:---|:---|
| 戦略企画 | T-4週 | ポジショニング、バリュープロポジション、ビジネスモデル |
| アセット準備 | T-3週 | GitHub Readme、ウェブサイト、PR記事、ドキュメント |
| チャネル準備 | T-2週 | KOLアウトリーチ、Redditアカウント、グループリスト |
| チーム分担 | T-1週 | 予算承認、タスク割り当て、コミュニケーション |
| ローンチスプリント | T-3日 | コピー確認、自動化設定、最終チェック |
| 🚀 **ローンチ日** | T-0 | マルチチャネル集中露出、トラフィックフライホイール |
| 継続運営 | T+1〜T+7 | 毎日100+スター、KOL Quote爆撃 |
