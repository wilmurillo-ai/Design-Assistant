# KOL Collaboration List & Strategy

## Strategy

- **High-value KOL**: `Thread + Quote` for deep collaboration
- **Quote KOL**: "Volume tactics" — massive Quotes for screen-flooding effect
- **Target**: 20-30 Quotes/day × 5 days = 100-150 total

## Collaboration Flow

```
Confirm budget → Match KOLs → Get quotes → Confirm intent 
    → Send Content Package → Approve draft → Pay → Publish → Track
```
