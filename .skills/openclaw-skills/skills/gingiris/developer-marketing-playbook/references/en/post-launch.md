# Post-Launch Operations (T+1 to T+7)

## Core Target

> **Daily new Stars ≥ 100**. If not met, immediately review and add channel traffic.

---

## Daily Operations Rhythm

### 1. GitHub Star Monitoring
- **Frequency**: Morning and evening
- **Tools**: Star-history.com, OSSInsight.io

### 2. KOL Maintenance
- **Target**: 20-30 Quotes per day, 5 consecutive days
- **Total**: 100-150 Quotes

### 3. Reddit Operations
- **Target**: 3-5 new Subreddits per day
- **Tool**: idea-hunt.com for finding discussions

### 4. Group Distribution
- **Target**: 50-100 groups per day (Telegram/Discord/LinkedIn)

### 5. Content Re-distribution
- Share Medium articles to Dev.to
- Comment on related articles with project links

### 6. YouTube Ads
- **Budget**: $100
- **Target**: Website visits → GitHub
- **Region**: India, Indonesia (cost-effective)

### 7. User Feedback
- Monitor GitHub Issues
- Respond to community and social media feedback

| Metric | Tool | Frequency | Target |

| **GitHub Stars** | Star-history.com | 2x daily | >100/day |
| **Repo Traffic** | GitHub Insights | Daily | Analyze sources |
| **Website Traffic** | Google Analytics | Daily | User behavior |
| **Channel Conversion** | UTM/Bitly | Real-time | High ROI channels |
| **Community Activity** | Discord/Telegram | Daily | New members |

## Contributor Management

| Step | Action | Goal |
|:---|:---|:---|
| **1. Lower barriers** | Tag simple issues with `Help Wanted` | Attract first-time contributors |
| **2. Screen talent** | Review PRs for code quality | Identify core contributors |
| **3. Deepen collaboration** | DM high-potential contributors, interview | Convert to core team |
| **4. Incentive system** | Build long-term ecosystem incentives | Maintain community activity |

## Daily Checklist

- [ ] **Morning**: Record GitHub Star count
- [ ] **AM**: Complete 20-30 KOL Quotes
- [ ] **PM**: Post in 3-5 new Subreddits
- [ ] **PM**: Distribute to 50-100 groups
- [ ] **Evening**: Comment on 5-10 related articles
- [ ] **Night**: Reply to all new comments/DMs
- [ ] **EOD**: Record Stars, review channels, adjust next day strategy
