# 发布日执行清单 (T-0)

## 核心原则

- **高强度、高密度、多渠道的集中曝光**
- **目标**: 短时间内引爆流量，冲上各大平台热榜，触发"流量飞轮"效应
- **时区**: 目标美国市场，核心活动从北京时间 21:00（美东 9:00）开始

## 发布日 Checklist

### GitHub & 官网
- [ ] 将仓库设为 Public
- [ ] 确认 Readme、文档链接无误
- [ ] 上线最新版官网
- [ ] 确认所有 CTA 链接正确

### Twitter
- [ ] 官方账号发布核心 Thread
- [ ] 确认第一批核心 KOL 已发布内容
- [ ] 开启 AI 自动评论任务
- [ ] 确认第二批 KOL 已发布内容

### 内容平台
- [ ] Medium 发布核心 PR 文章
- [ ] Hacker News 提交
- [ ] Product Hunt 提交
- [ ] V2EX 发帖

### Reddit
- [ ] 在 3-5 个目标 Subreddit 完成发帖
- [ ] 监控帖子反馈，及时回复评论

### 开源平台
- [ ] DevHunt 提交
- [ ] dev.to 发布
- [ ] 其他开源平台提交

### 数据监控
- [ ] 记录 24 小时 Star 数量
- [ ] 记录各渠道流量来源
- [ ] 记录 Discord/Telegram 新增成员

## 紧急情况处理

| 情况 | 处理方式 |
|:---|:---|
| **KOL 未按时发布** | 立即联系确认，准备备用 KOL |
| **负面评论出现** | 不要删除，积极真诚地回应，将批评视为改进机会 |
| **Star 增长缓慢** | 立即补充新流量来源，增加 Reddit 发帖、KOL Quote |
| **网站/服务宕机** | 技术团队立即响应，同步在社媒说明情况 |
