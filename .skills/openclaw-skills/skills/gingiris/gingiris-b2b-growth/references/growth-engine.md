# 第二部分：构建增长引擎

## 1. PLG + SLG 混合增长模型

### 两种模式的定位

| 模式 | 定义 | 优势 |
|:---|:---|:---|
| **PLG** | 以产品本身为核心营销工具，用户自行体验、评估、购买 | 极低获客成本，极高扩展性 |
| **SLG** | 以销售团队为驱动力，主动触达、方案定制、商务谈判 | 攻克复杂决策流程，高LTV |

### PQL（产品认证线索）定义

PQL = 通过产品内行为表现出强烈购买意向的用户，转化率远高于MQL

| 维度 | 示例 | 说明 |
|:---|:---|:---|
| **激活程度** | 成功导出3个以上视频 | 已深度体验核心价值 |
| **使用频率** | 连续7天登录 | 已融入日常工作流 |
| **协作广度** | 邀请3名以上团队成员 | 已产生网络效应 |
| **企业画像** | 财富500强邮箱后缀 | 高付费潜力 |

## 2. 联盟营销 — 三种佣金模型

### 案例对比

| 公司 | 佣金模型 | Cookie有效期 | 适用产品 |
|:---|:---|:---|:---|
| **InVideo** | **50%首单高额返佣** | 120天 | 中低客单价、高转化率工具 |
| **HeyGen** | **20%持续12个月** | 90天 | 订阅制、高留存SaaS |
| **Deel** | **$500-$1500固定赏金** | 90天 | 高客单价企业级服务 |

### 联盟营销六步搭建

1. **选择管理平台：** PartnerStack, Impact, ShareASale
2. **设计佣金结构：** 根据客单价、利润率、LTV设计
3. **创建推广素材：** Affiliate Kit（Logo、截图、视频、Banner、邮件模板）
4. **招募首批联盟客：** 从忠实用户、社区成员、已评测博主中招募
5. **赋能与沟通：** 建立专属沟通渠道，同步产品更新和推广技巧
6. **合规与反作弊：** 禁止品牌词竞价、虚假宣传、机器人流量

## 3. 渠道伙伴生态

### 四种核心角色

| 伙伴类型 | 核心价值 | 合作模式 | 案例 |
|:---|:---|:---|:---|
| **技术集成伙伴** | 扩展产品能力 | API对接，应用市场互上 | Vercel上的Headless CMS |
| **解决方案伙伴** | 行业解决方案与实施服务 | 咨询、实施、转售 | Lark亚太区伙伴 |
| **云市场伙伴** | 借助云巨头渠道 | 上架AWS/Azure/GCP Marketplace | Snowflake, Datadog |
| **战略联盟** | 品牌背书，共同开拓大客户 | 联合市场活动，联合销售 | OpenAI × Microsoft |

### Deel 案例：分级认证体系

四个等级：**注册级 → 银牌 → 金牌 → 铂金**

不同等级享有不同权益：
- 佣金比例
- 市场基金
- 技术支持
- 品牌曝光

**成果：** 伙伴计划贡献超过 **$90M ARR**

### Vercel × Zapier × v0 案例

- 过去：新集成需要数周时间进行技术评估和原型开发
- 现在：产品经理用自然语言描述需求，v0 几分钟内生成功能完善的React原型
- **效率提升：100倍**

### AWS 全球创业计划

为入选创业公司提供：
- 云资源抵扣券
- **联合销售支持 (Co-selling Support)** — AWS销售团队一起拓展客户

## 4. 直销作战单元

### SDR + AE + SE + CSM 模型

| 角色 | 职责 |
|:---|:---|
| **SDR** (销售发展代表) | 线索筛选和培育，判断BANT，转交SQL给AE |
| **AE** (客户主管) | 跟进SQL，产品演示、需求挖掘、方案提供、商务谈判、签约 |
| **SE** (解决方案工程师) | 技术搭档，复杂技术问题、PoC、系统集成 |
| **CSM** (客户成功经理) | Onboarding、持续使用、续约，确保客户获得价值 |

### 大客户多线程作战

对于大型企业客户，采购决策链条复杂，涉及多个部门和角色。

**策略：** 同时与多个关键角色建立联系和信任

关键人物识别：
- **Champion（支持者）**
- **Economic Buyer（经济买家）**
- **Technical Buyer（技术买家）**

## 5. 伙伴体系搭建要点

1. **设计伙伴计划：** 明确类型、合作模式、权益与义务
2. **建立伙伴门户：** 资源下载、线索报备、佣金查询、培训学习
3. **招募与筛选：** 优先从现有客户、社区领袖发展
4. **赋能与支持：** 设立Partner Manager角色
5. **建立伙伴目录：** 展示认证伙伴的专长和成功案例
