# 第四部分：合同模板体系

## 合同体系概览

| 文件 | 说明 |
|:---|:---|
| **SOR** (服务需求书) | 由客户撰写，描述业务目标、功能需求 |
| **SOW** (实施工作说明书) | 由服务方撰写，定义范围、交付物、时间表 |
| **MSA** (主服务协议) | 法律主体合同，定义通用条款 |
| **PCR** (项目变更申请单) | 管理需求变更 |
| **报价单** | 费用构成、支付方式 |

## 3. 主服务协议 (MSA) 核心条款

| 条款 | 要点 |
|:---|:---|
| **服务范围** | 定义服务内容，引用具体SOW |
| **付款条款** | 支付方式、周期、货币、逾期违约责任 |
| **知识产权** | 定制开发部分归客户，通用框架归服务方 |
| **保密协议** | 双方对商业秘密和技术信息予以保密 |
| **数据处理与安全** | 数据所有权、使用权、安全措施（GDPR/CCPA） |
| **SLA** | 可用性承诺（如99.9%）、故障响应时间、补偿措施 |
| **陈述与保证** | 双方保证有权签署并履行本协议 |
| **责任限制** | 最大赔偿责任上限（通常不超过合同总金额） |
| **期限与终止** | 有效期、提前终止条件 |
| **法律适用** | 适用法律、争议解决方式（仲裁/法院） |
