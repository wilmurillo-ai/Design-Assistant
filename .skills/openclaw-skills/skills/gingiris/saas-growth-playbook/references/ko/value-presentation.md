# 3부: 가치 제시의 기술

## 1. 밸류 프로포지션 케이스

### Supabase: "Build in a weekend, scale to millions."
- **"Build in a weekend"** — 빠른 프로토타이핑에 대한 개발자 페인 포인트 직격
- **"Scale to millions"** — 프로덕션 환경 사용에 대한 우려 해소
- **한 문장으로 개인 개발자와 엔터프라이즈 의사결정자 모두 커버**

### Vercel: "The Frontend Cloud"
- 기능 설명 대신 새로운 카테고리 정의
- "클라우드"와 연관 (확장성, 고가용성)
- "Next.js 호스팅"에서 "차세대 웹 인프라 리더"로 승격

> **좋은 B2B 밸류 프로포지션의 3요소:**
> 1. 원라인 포지셔닝
> 2. 정량화된 성과
> 3. 소셜 프루프

## 2. B2B 세일즈 덱 9페이지 구성

| 페이지 | 내용 | 목적 |
|:---|:---|:---|
| 1 | **표지** | 기업 로고 + 고객 로고 + 밸류 프로포지션 |
| 2 | **문제** | 고객의 페인 포인트 묘사 |
| 3 | **솔루션** | 어떻게 문제를 해결하는가 |
| 4 | **데모** | 핵심 기능 스크린샷/비디오 |
| 5 | **가치** | 정량화된 베네핏 ("30% 효율 향상") |
| 6 | **케이스 스터디** | 동일 업계 성공 사례 |
| 7 | **엔터프라이즈** | 보안, 컴플라이언스, 지원 |
| 8 | **가격** | 티어와 기능 비교 |
| 9 | **요약과 다음 단계** | 요약 + 명확한 액션 플랜 |

## 3. 고객 성공 스토리 — 히어로즈 저니

### Vercel × Frame.io 케이스

**1막: 챌린지**
- Frame.io 웹사이트는 이터레이션이 느리고 퍼포먼스가 나쁨
- 구식 기술 스택이 비즈니스 병목

**2막: 깨달음**
- 프론트엔드 경험을 극한까지 높일 필요성 인식
- 마케팅 유연성 + 엔지니어링 컨트롤 필요

**3막: 솔루션**
- Vercel + Next.js 선택
- ISR, Edge Functions, Preview Deployments 활용

**4막: 결과**
- 페이지 로드 퍼포먼스 **3배** 향상
- 새 페이지 퍼블리싱 **며칠 → 몇 분**
- 글로벌 가용성 **99.99%**
- 최종적으로 Adobe에 인수

> **핵심 인사이트:** 고객을 "히어로"로 묘사하고, 제품을 승리를 돕는 "마법의 무기"로 포지셔닝
