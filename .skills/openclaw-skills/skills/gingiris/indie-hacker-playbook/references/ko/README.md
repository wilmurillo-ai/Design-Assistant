# AI 제품 글로벌 런칭 액션 가이드

> 저자: Iris (生姜iris) | 버전: Final v7.0 (2026년 2월)

## 핵심 원칙

- **사용자 우선, 가치에서 시작** — 모든 콘텐츠와 채널 전략은 타겟 사용자에게 실제 가치를 창출하는 것에서 시작
- **콘텐츠가 왕, 채널은 증폭기** — 고품질 콘텐츠가 확산의 기반
- **글로벌하게 생각, 로컬하게 실행** — 글로벌 브랜드 일관성을 유지하면서 각 시장에 맞게 현지화
- **질이 양보다 중요** — 예산을 핵심 영향력 노드에 집중, 뿌리지 말것

## 전체 타임라인

| 단계 | 시기 | 주요 마일스톤 |
|:---|:---|:---|
| 전략 준비기 | L-6주 | 목표, ICP, 가치 제안, 키워드, 예산 정의 |
| 자산 제작기 | L-5~L-4주 | 웹사이트 최적화, 브랜드 비디오, 기능 비디오 |
| 파트너십 확정기 | L-4~L-3주 | KOL 선별, UGC 모집, 미디어 아웃리치 |
| 콘텐츠 준비기 | L-3~L-2주 | KOL 콘텐츠 패키지, UGC 스크립트, Reddit 전략 |
| 최종 확인기 | L-2~L-1주 | 초안 승인, 런칭 타이밍 확정 |
| 🚀 **제품 런칭** | 런칭일 | Launch Week Day 1-5 |
| 모멘텀 구축기 | L+1~+4주 | 지속적 운영, 모멘텀 축적 |
