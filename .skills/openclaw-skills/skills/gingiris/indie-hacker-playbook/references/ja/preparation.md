# 準備段階SOP（L-6〜L-1週）

## 1. 戦略定義（L-6週）

| コア質問 | 成果物 |
|:---|:---|
| 誰がユーザーか？ | ICPドキュメント |
| 何を解決するか？ | バリュープロポジション（タグライン + 3つのポイント） |
| 目標は何か？ | ローンチ目標ドキュメント（定量化可能なKPI） |
| 予算は？ | 予算表 |
| キーワードは？ | キーワードリスト（競合10 + シナリオ10） |

---

## 2. ウェブサイト最適化（L-5〜L-2週）

### 必須チェックリスト

| セクション | 必要コンテンツ |
|:---|:---|
| **ヒーロー** | メイン見出し + サブタイトル + CTA + 製品スクリーンショット |
| **機能** | 3つのコア販売ポイント（シナリオGIF付き） |
| **テスティモニアル** | 実際のテスティモニアル（アバター + 役職 + 会社） |

**コンバージョン率ベンチマーク：** 最低10%、優秀30-40%

---

## 3. ビデオ制作（L-5〜L-1週）

### ローンチビデオ構成
- 長さ：60-90秒
- 構成：スローガン → 3つの機能ハイライト → スローガン
