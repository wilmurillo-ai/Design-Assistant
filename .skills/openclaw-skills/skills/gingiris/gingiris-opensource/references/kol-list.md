## 合作策略

- **高价值 KOL**: 采用 `Thread + Quote` 形式深度合作，制造核心声量

```
确认预算 → 精准匹配 KOL → 询价 → 确认合作意向 
    → 发送 Content Package → 确认 Draft → 支付 → 发布 → 效果追踪
```

## AI 自动化引流

### 核心工具

- **Phantombuster**: Twitter/LinkedIn 自动化 DM 和评论
- **Opencord.ai**: Twitter AI 自动评论
- **Sheet0.com**: 抓取 Twitter 用户列表
- **转化率**: 约 1%

### 目标用户列表

- **竞品 Follower**: 抓取竞品（如 Suno, Vercel, Supabase）的 Follower 列表
- **KOL Follower**: 抓取行业关键 KOL 的 Follower 列表
- **精准开发者**: 使用筛选过的开发者账号列表

### 执行频率

每天发送 30 条自动评论，连续执行 7 天
