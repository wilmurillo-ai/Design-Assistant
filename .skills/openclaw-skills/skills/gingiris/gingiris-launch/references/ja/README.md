# AI製品グローバルローンチ・アクションガイド

> 著者：Iris（生姜iris）| バージョン：Final v7.0（2026年2月）

## 基本原則

- **ユーザーファースト、価値から始める** — すべてのコンテンツとチャネル戦略は、ターゲットユーザーに真の価値を創造することから始まる
- **コンテンツが王、チャネルは増幅器** — 高品質なコンテンツが拡散の基盤
- **グローバルに考え、ローカルに実行** — グローバルなブランド一貫性を維持しながら、各市場にローカライズ
- **質が量より重要** — 予算を重要な影響力のあるノードに集中し、散布しない
