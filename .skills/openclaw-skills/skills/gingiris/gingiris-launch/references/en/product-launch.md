# Product Launch (Launch Week)

## Launch Week Principles

- **Content Rhythm:** One theme per day, progressive depth
- **Channel Synergy:** Simultaneous multi-channel publishing
- **Goal-Oriented:** Core goal is user growth and community activation
- **Momentum Building:** Prepare for PH launch

---

## Day 1 (Monday): Main Launch Day

**Theme:** Official product release, showcase core value and demo video
**Goal:** Maximize initial exposure, GitHub Stars sprint (100-120 on day 1)

| Time (PST) | Platform | Content |
|:---|:---|:---|
| 00:01 | Twitter/X | Main launch tweet (with demo video) |
| 00:01 | Discord/Telegram | Community announcement |
| 00:30 | GitHub | README update |
| 01:00-04:00 | Twitter/X | KOL first wave threads |
| 02:00 | Reddit | Posts (10 posts) |
| 04:00-08:00 | Overseas groups | Group distribution |
| All day | UGC channels | 20 UGC posts across platforms |

### Day 1 Checklist
- [ ] Main tweet with demo video published
- [ ] Community announcements sent
- [ ] GitHub README updated
- [ ] Core KOL threads published
- [ ] Reddit posts published (10)
- [ ] Overseas group messages sent
- [ ] 20 UGC posts published

---

## Day 2 (Tuesday): Tutorial Day

**Theme:** Step-by-step getting started tutorial
**Goal:** Improve new user activation, cumulative 200-240 Stars

---

## Day 3-4 (Wed-Thu): Feature Deep Dives

**Theme:** Deep dive into core features 1 and 2
**Goal:** Attract users interested in specific features, cumulative 400-480 Stars

---

## Day 5 (Friday): Community & Summary

**Theme:** Community activities, Launch Week summary, PH preview
**Goal:** Strengthen community cohesion, prepare for PH

- Host AMA (Ask Me Anything)
- Publish summary tweet, preview PH
- Send thank you newsletter

---

## Post-Launch Week: Momentum Building (L+1 to PH-1 Week)

- **Continuous Content:** 1-2 valuable posts per week
- **Collect User Cases:** Case studies and testimonials for PH page
- **Monitor GitHub Stars:** Key reference for PH timing
- **PH Preview:** Start teasing 1-2 weeks before: "We're coming to Product Hunt!"
