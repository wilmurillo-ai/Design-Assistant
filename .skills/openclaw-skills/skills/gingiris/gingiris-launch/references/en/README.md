# AI Product Global Launch Action Guide

> Author: Iris (生姜iris) | Version: Final v7.0 (Feb 2026)

## Core Principles

- **User First, Start with Value** — All content and channel strategies begin with creating real value for target users
- **Content is King, Channels are Amplifiers** — High-quality content is the foundation of distribution
- **Think Global, Execute Local** — Maintain global brand consistency while localizing for different markets
- **Quality Over Quantity** — Focus budget on key influential nodes, not spray-and-pray

## Overall Timeline

| Phase | Timing | Key Milestone |
|:---|:---|:---|
| Strategic Planning | L-6 weeks | Define goals, ICP, value proposition, keywords, budget |
| Asset Production | L-5 to L-4 weeks | Website optimization, brand video, feature videos |
| Partnership Lock-in | L-4 to L-3 weeks | KOL screening, UGC recruitment, media outreach |
| Content Preparation | L-3 to L-2 weeks | KOL content packages, UGC scripts, Reddit strategy |
| Final Confirmation | L-2 to L-1 weeks | Draft approval, launch timing lock |
| 🚀 **Product Launch** | Launch Day | Launch Week Day 1-5 |
| Momentum Building | L+1 to +4 weeks | Continuous operations, momentum accumulation |
| 🎯 **PH Launch** | PH Day | 24-hour battle plan |
