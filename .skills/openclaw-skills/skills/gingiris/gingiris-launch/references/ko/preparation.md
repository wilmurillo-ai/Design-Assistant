# 준비 단계 SOP (L-6 ~ L-1주)

## 1. 전략 정의 (L-6주)

| 핵심 질문 | 산출물 |
|:---|:---|
| 사용자는 누구인가? | ICP 문서 |
| 어떤 문제를 해결하는가? | 가치 제안 (태그라인 + 3가지 포인트) |
| 목표는 무엇인가? | 런칭 목표 문서 (정량화 가능한 KPI) |
| 예산은? | 예산표 |
| 키워드는? | 키워드 리스트 (경쟁사 10 + 시나리오 10) |

---

## 2. 웹사이트 최적화 (L-5 ~ L-2주)

### 필수 체크리스트

| 섹션 | 필요 콘텐츠 |
|:---|:---|
| **히어로** | 메인 헤드라인 + 서브타이틀 + CTA + 제품 스크린샷 |
| **기능** | 3가지 핵심 판매 포인트 (시나리오 GIF 포함) |
| **추천사** | 실제 추천사 (아바타 + 직함 + 회사) |

**전환율 벤치마크:** 최소 10%, 우수 30-40%

---

## 3. 비디오 제작 (L-5 ~ L-1주)

### 런칭 비디오 구조
- 길이: 60-90초
- 구조: 슬로건 → 3가지 기능 하이라이트 → 슬로건

---

## 4. 인플루언서 전략 (L-4주)

### 인플루언서 가치 순위

| 유형 | 가치 |
|:---|:---|
| **뉴스레터 창립자** | **최고** |
| **개발자/엔지니어** | **매우 높음** |
| **임원** | **매우 높음** |
| **창업자/인디** | **높음** |

---

## 5. KOL 협업

### 선정 기준
- 팔로워: 1만~10만이 이상적
- 참여율: 뷰/팔로워 > 10%

### 플랫폼 가격 참고
| 플랫폼 | 형식 | 가격대 |
|:---|:---|:---|
| Twitter/X | 1 스레드 | $50-$200 |
| YouTube | 롱폼 비디오 | $600-$1,500 |

---

## 6. UGC 성장 엔진

**핵심 철학:** UGC 크리에이터가 아닌, **브랜드 앰배서더**를 찾아라

**열정 > 경험, 아마추어 > 인플루언서, 질 > 양**
