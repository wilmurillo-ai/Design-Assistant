# 预算分配建议

## 预算分配原则

1. **不要在无效的Hype账号上浪费一分钱** — 将预算集中投入到能一锤定音的关键节点
2. **人脉优先，付费补充** — 高价值KOL优先通过人脉关系免费触达
3. **内容是最大的杠杆** — Launch Video的制作质量直接决定所有后续传播的天花板

---

## $10,000 精简版（早期创业团队）

| 类别 | 预算 | 说明 |
|:---|:---|:---|
| **Launch Video 制作** | $4,000 | 聘请专业自由职业者制作，最重要的单笔支出 |
| **核心KOL 合作（1-2人）** | $3,000 | 选择1-2位精准的中腰部KOL（1-20万粉），单帖$500-$2,500 |
| **媒体新闻稿发布** | $1,500 | PR Newswire或类似平台，保证基础媒体覆盖 |
| **备用/广告助推** | $1,500 | 对核心推文进行小范围广告助推 |
