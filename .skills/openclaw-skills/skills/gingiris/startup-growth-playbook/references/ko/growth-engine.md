## 2. 어필리에이트 마케팅 — 3가지 커미션 모델

| 기업 | 모델 | 쿠키 기간 | 최적 용도 |
|:---|:---|:---|:---|
| **InVideo** | **50% 첫 주문** | 120일 | 중저가, 높은 CV율 도구 |
| **HeyGen** | **20% × 12개월** | 90일 | 구독, 높은 리텐션 SaaS |
| **Deel** | **$500-$1500 고정** | 90일 | 고단가 엔터프라이즈 |

### 6단계 셋업
1. 플랫폼 선택 (PartnerStack, Impact, ShareASale)
2. 커미션 구조 설계
3. 프로모션 자료 제작 (Affiliate Kit)
4. 초기 어필리에이트 모집
5. 활성화와 커뮤니케이션
6. 컴플라이언스와 부정 방지

## 3. 채널 파트너 에코시스템

### 4가지 핵심 역할

| 파트너 유형 | 핵심 가치 | 협업 모델 |
|:---|:---|:---|
| **기술 통합** | 제품 기능 확장 | API 통합, 마켓플레이스 등록 |
| **솔루션** | 업계 솔루션과 구현 | 컨설팅, 구현, 재판매 |
| **클라우드 마켓플레이스** | 클라우드 대기업 채널 활용 | AWS/Azure/GCP 등록 |
| **전략적 제휴** | 브랜드 신용 | 공동 마케팅, 공동 판매 |

### Deel 케이스: 단계별 인증
4단계: **등록 → 실버 → 골드 → 플래티넘**

**결과:** 파트너 프로그램이 **$90M ARR** 기여
