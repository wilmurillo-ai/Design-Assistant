# 第4部：契約テンプレート体系

## 契約システム概要

| 文書 | 説明 |
|:---|:---|
| **SOR**（要件定義書） | クライアント作成、ビジネス目標と要件を記述 |
| **SOW**（作業範囲記述書） | ベンダー作成、範囲、成果物、タイムラインを定義 |
| **MSA**（基本サービス契約） | 法的マスター契約、一般条件を定義 |
| **PCR**（プロジェクト変更申請） | 範囲変更を管理 |
| **見積書** | 費用構成、支払条件 |

## 3. MSAコア条項

| 条項 | 要点 |
|:---|:---|
| **サービス** | 範囲を定義、SOWを参照 |
| **支払い** | 方法、サイクル、通貨、遅延ペナルティ |
| **知的財産** | カスタム → クライアント; フレームワーク → ベンダー |
| **機密保持** | 相互NDA |
| **データとセキュリティ** | 所有権、使用、GDPR/CCPA準拠 |
| **SLA** | アップタイム（99.9%）、応答時間、救済措置 |
| **責任制限** | 上限（通常≤契約金額） |
| **期間と終了** | 期間、早期終了条件 |
| **準拠法** | 適用法、紛争解決 |
