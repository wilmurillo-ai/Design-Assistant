# 第5部：データ駆動成長ループ

## 1. 北極星指標

**定義：** 製品が顧客に創出するコアバリューを最もよく測定する単一指標

**要件：**
- 全社（製品、エンジニアリング、マーケ、営業）がこれを中心に結集
- 顧客価値を直接反映

**例：**
| 製品タイプ | 北極星指標 |
|:---|:---|
| AI動画ツール | 週間エクスポート動画数 |
| AI会議アシスタント | 週間生成会議サマリー数 |
| コラボプラットフォーム | 週間アクティブコラボチーム数 |

---

## 2. AARRRファネル指標

| ステージ | コア質問 | 主要指標 |
|:---|:---|:---|
| **獲得** | ユーザーはどこから？ | チャネルUV、サインアップ、CAC |
| **アクティベーション** | コアバリューを体験した？ | 「Ahaモーメント」CV率、オンボーディング完了率 |
| **リテンション** | 戻ってくる？ | D1/D7/D30リテンション |
| **収益** | 支払う？ | CV率、ARPU、LTV |
| **紹介** | 推薦する？ | NPS、K-factor |

---

## 3. 主要指標定義

### 獲得指標
| 指標 | 定義 |
|:---|:---|
| **CAC** | 顧客獲得コスト = 総セールス&マーケ費用 / 新規顧客数 |
| **LTV** | 顧客生涯価値 = ARPU × 平均顧客寿命 |
| **LTV/CAC** | 獲得効率、健全値 > 3:1 |

### リテンション指標
| 指標 | 計算式 |
|:---|:---|
| **D1リテンション** | 2日目復帰者 / 1日目新規ユーザー |
| **D7リテンション** | 8日目復帰者 / 1日目新規ユーザー |

### 収益指標
| 指標 | 定義 |
|:---|:---|
| **MRR** | 月間経常収益 |
| **ARR** | 年間経常収益 = MRR × 12 |
| **NRR** | 純収益リテンション（拡大、解約含む） |

## 6. 成長実験フレームワーク

### ICEスコアリング
| 次元 | 説明 |
|:---|:---|
| **Impact** | 北極星への潜在的影響 |
| **Confidence** | 仮説への確信度 |
| **Ease** | 実装の難易度 |

**優先度 = Impact × Confidence × Ease**
