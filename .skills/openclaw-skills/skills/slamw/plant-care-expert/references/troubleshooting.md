# 植物问题排查手册

本文档提供常见植物问题的诊断方法和解决方案，帮助用户快速定位并解决植物养护中遇到的问题。

---

## 一、水分相关问题

### 浇水过多

**识别症状**

- 叶片发黄、暗淡无光泽
- 嫩叶和老叶同时发黄
- 土壤长期潮湿不干
- 叶片萎蔫但土壤湿润
- 茎基部变软变色
- 盆底托盘长期有积水

**可能原因**

- 浇水频率过高
- 花盆排水不良
- 土壤透气性差
- 通风不良，水分蒸发慢
- 冬季植物休眠期仍按常规浇水

**解决方案**

1. 立即停止浇水
2. 松动表层土壤增加透气
3. 将花盆放在通风良好的地方
4. 用筷子在盆土上扎几个小孔透气
5. 如已烂根，需翻盆修根换土
6. 调整浇水频率，遵循见干见湿原则

**预防措施**

- 选择排水良好的花盆
- 配土时加入珍珠岩或粗沙
- 浇水前用手指测试土壤湿度
- 冬季减少浇水频率

### 浇水过少

**识别症状**

- 叶片萎蔫、下垂
- 叶片边缘卷曲、焦枯
- 土壤干燥开裂
- 叶片颜色变浅
- 盆土与盆壁分离
- 落叶（严重时）

**可能原因**

- 忘记浇水
- 浇水只浇表面，未浇透
- 花盆过大，根系吸收不到水分
- 环境过于干燥，水分蒸发快

**解决方案**

1. 立即浇透水，确保水分渗透到底部
2. 对于极度干燥的土壤，采用浸盆法
3. 剪除枯萎严重的叶片
4. 移至阴凉处缓苗
5. 增加周围空气湿度

**浸盆操作步骤**

1. 准备一盆温水（与室温相近）
2. 将花盆放入水中，水位至盆土高度的三分之一
3. 浸泡 20-30 分钟
4. 取出放在通风处控干多余水分
5. 之后正常养护

### 水质问题

**识别症状**

- 叶片出现斑点或焦斑
- 土壤表面有白色水渍
- 生长缓慢
- 叶脉间发黄

**可能原因**

- 使用了未经处理的自来水
- 水中矿物质或盐分过高
- 使用了洗碗水或洗衣水

**解决方案**

1. 自来水需放置 24 小时后再使用
2. 收集雨水或使用纯净水
3. 定期用清水冲洗土壤
4. 换用透气性好的土壤

## 二、光照相关问题

### 光照过强

**识别症状**

- 叶片出现灼伤斑块
- 叶片边缘焦枯
- 叶片颜色变浅、发白
- 叶片脱水萎蔫
- 生长停滞

**可能原因**

- 夏季阳光直射
- 突然从室内移至室外
- 玻璃聚焦阳光产生灼伤
- 使用补光灯距离过近

**解决方案**

1. 立即移至阴凉处
2. 剪除严重灼伤的叶片
3. 保持土壤湿润
4. 逐步增加光照，避免再次灼伤
5. 夏季注意遮荫

### 光照不足

**识别症状**

- 叶片颜色变淡、发黄
- 茎节拉长（徒长）
- 叶片变小
- 生长缓慢或停滞
- 新叶颜色浅淡
- 开花植物不开花
- 叶片向光源方向倾斜

**可能原因**

- 摆放位置光线过暗
- 长期不换位置
- 窗户被遮挡
- 冬季日照时间短

**解决方案**

1. 逐步移至光线更好的位置
2. 定期旋转花盆均匀受光
3. 使用补光灯补充光照
4. 修剪徒长枝条
5. 减少浇水，徒长植物需控水

**补光灯使用建议**

- 选择全光谱植物灯
- 每天补充 4-6 小时
- 距离植物 20-40 厘米
- 避免距离过近造成灼伤

## 三、温度相关问题

### 高温伤害

**识别症状**

- 叶片萎蔫、下垂
- 叶缘焦枯
- 叶片出现褐色斑点
- 生长停滞
- 果实变形或脱落

**可能原因**

- 夏季高温未遮荫
- 靠近暖气或空调出风口
- 花盆放在发热物体旁
- 密闭车内温度过高

**解决方案**

1. 移至阴凉通风处
2. 向叶面喷水降温
3. 增加浇水频率（仅限耐热植物）
4. 使用遮阳网或窗帘遮挡
5. 远离热源

### 低温冻害

**识别症状**

- 叶片变软出水
- 叶片颜色变暗、发黑
- 茎部软腐
- 整株萎蔫
- 新叶脱落

**可能原因**

- 冬季忘记移入室内
- 夜晚忘记关闭窗户
- 放置在门口或窗边冷风处
- 暖气突然停止

**解决方案**

1. 立即移至温暖处（10°C 以上）
2. 不要立即浇水
3. 剪除冻伤严重的叶片和茎
4. 保持土壤偏干状态
5. 不要暴晒，逐步适应温暖环境
6. 放置 24 小时后再少量浇水

### 温差过大

**识别症状**

- 叶片出现畸形
- 生长点受损
- 叶片脱落
- 开花期花瓣受损

**可能原因**

- 昼夜温差超过 15°C
- 白天暴晒夜晚低温
- 冷暖气流直吹

**解决方案**

1. 调整摆放位置
2. 避免冷热风直吹
3. 冬季夜间移至室内
4. 逐步适应温度变化

## 四、施肥相关问题

### 施肥过量（肥害）

**识别症状**

- 叶片边缘或尖端焦枯
- 叶片表面有水渍状斑块
- 土壤表面有白色盐渍
- 叶片变厚、畸形
- 整株萎蔫
- 根系变褐色

**可能原因**

- 施肥浓度过高
- 施用了未腐熟的有机肥
- 施肥过于频繁
- 干燥状态下施肥

**解决方案**

1. 立即用大量清水冲淋土壤
2. 重复冲洗 2-3 次
3. 放在通风处晾干
4. 如严重，需翻盆换土
5. 修剪枯萎部分
6. 之后减少施肥量

### 施肥不足

**识别症状**

- 叶片颜色变淡
- 叶片变小
- 生长缓慢
- 下部叶片发黄脱落
- 开花植物花朵变小
- 茎干变细

**可能原因**

- 长期未施肥
- 土壤贫瘠
- 植物生长旺盛，消耗大
- 换盆时未施底肥

**解决方案**

1. 补充氮肥促进叶片生长
2. 薄肥勤施，避免浓肥
3. 生长季增加施肥频率
4. 定期换盆换土补充养分

## 五、病虫害问题

### 虫害识别与处理

#### 蚜虫

**识别特征**

- 绿色、黑色或棕色的小虫
- 聚集在嫩芽、花蕾、叶片背面
- 叶片卷曲变形
- 有黏稠的蜜露（蚂蚁喜食）

**解决方案**

1. 用清水冲洗叶片
2. 用棉签蘸酒精擦拭
3. 喷洒稀释的肥皂水
4. 严重时使用吡虫啉
5. 引入瓢虫等天敌

#### 红蜘蛛

**识别特征**

- 微小红色螨虫
- 叶片出现细小白点
- 叶片背面有蛛网
- 叶片干燥脱落

**解决方案**

1. 增加空气湿度
2. 用清水冲洗叶片背面
3. 喷洒苦楝油或矿物油
4. 使用阿维菌素等杀螨剂
5. 隔离病株防止蔓延

#### 介壳虫

**识别特征**

- 棕褐色或白色圆形小虫
- 固定在叶片或茎上
- 叶片出现黄斑
- 有黏稠分泌物

**解决方案**

1. 用牙刷或棉签刷除
2. 用酒精擦拭
3. 喷洒矿物油或马拉硫磷
4. 严重时需整株浸泡

#### 粉虱

**识别特征**

- 白色小飞虫
- 受到惊扰会飞散
- 叶片背面产卵
- 叶片变黄、生长不良

**解决方案**

1. 使用黄色粘虫板诱捕
2. 喷洒稀释的肥皂水
3. 使用吡虫啉或啶虫脒
4. 保持良好通风

### 病害识别与处理

#### 叶斑病

**识别特征**

- 叶片出现褐色、黑色或灰色斑点
- 斑点逐渐扩大
- 严重时叶片枯死
- 可能出现同心轮纹

**解决方案**

1. 剪除病叶销毁
2. 降低环境湿度
3. 加强通风
4. 喷洒多菌灵或百菌清
5. 避免叶面浇水

#### 根腐病

**识别特征**

- 根部变黑软腐
- 茎基部变色
- 叶片萎蔫
- 土壤有异味

**解决方案**

1. 立即翻盆检查
2. 剪除腐烂根系
3. 用多菌灵浸泡消毒
4. 更换新土
5. 控制浇水，保持通风

#### 白粉病

**识别特征**

- 叶片表面有白色粉末状物质
- 叶片卷曲变形
- 影响光合作用
- 湿度高时易发

**解决方案**

1. 剪除严重病叶
2. 喷洒小苏打溶液（5g/L）
3. 使用硫磺粉或百菌清
4. 增加通风，降低湿度

#### 病毒病

**识别特征**

- 叶片出现斑驳、条纹
- 叶片畸形
- 生长停滞
- 开花异常

**解决方案**

1. 无有效治疗方法
2. 销毁病株防止传播
3. 消毒工具
4. 使用健康种苗

## 六、生理障碍

### 缺素症

#### 缺氮

- 症状：老叶先黄，叶色浅淡
- 解决：施用尿素或氮肥

#### 缺磷

- 症状：叶色深绿但无光泽，生长缓慢
- 解决：施用磷肥或骨粉

#### 缺钾

- 症状：叶缘焦枯，老叶先受影响
- 解决：施用钾肥或草木灰

#### 缺铁

- 症状：新叶叶脉绿色，叶肉发黄
- 解决：施用硫酸亚铁

#### 缺镁

- 症状：叶脉间发黄，叶脉绿色
- 解决：施用硫酸镁

#### 缺钙

- 症状：新叶畸形，生长点枯死
- 解决：施用硝酸钙

### 元素中毒

- 症状：叶片出现焦斑、畸形
- 原因：施肥过量或使用不当
- 解决：大量浇水冲淡，严重时换土

## 七、环境适应问题

### 换盆应激

**识别症状**

- 叶片暂时萎蔫
- 少量落叶
- 生长暂时停滞

**处理方法**

1. 换盆后放在阴凉处缓苗 1-2 周
2. 保持土壤微潮
3. 不要立即施肥
4. 避免阳光直射
5. 等待植物恢复后再正常养护

### 运输应激

**识别症状**

- 叶片掉落
- 茎叶萎蔫
- 花苞脱落

**处理方法**

1. 放在温暖明亮处
2. 保持土壤湿润
3. 不要换盆或施肥
4. 耐心等待恢复
5. 1-2 周后可逐步正常养护

### 季节转换应激

**识别症状**

- 春季：新芽生长缓慢
- 夏季：高温休眠
- 秋季：叶片变色或脱落
- 冬季：生长停滞

**处理方法**

1. 根据季节调整养护
2. 避免突然改变环境
3. 逐步适应温度变化
4. 相应调整浇水和施肥

## 八、特殊问题

### 开花问题

#### 不开花

- 光照不足：增加光照
- 氮肥过多：减少氮肥，增施磷钾肥
- 未到成熟年龄：耐心等待
- 修剪不当：正确修剪促花芽

#### 落花落蕾

- 环境骤变：保持稳定
- 浇水不当：调整浇水
- 施肥不足：补充养分
- 通风不良：加强通风

### 果实问题

#### 落果

- 水分剧烈变化：保持稳定
- 营养不足：补充养分
- 环境不适：改善条件

### 徒长

**识别特征**

- 茎节拉长变细
- 叶片间距变大
- 整体株型松散
- 颜色变浅

**原因**

- 光照不足
- 氮肥过多
- 浇水过勤

**解决方法**

1. 增加光照
2. 减少氮肥
3. 控制浇水
4. 修剪徒长部分
5. 如严重，考虑扦插重新繁殖

## 九、急救流程图

### 叶片萎蔫急救流程

```
叶片萎蔫
    │
    ├── 土壤干燥？── 是 ──→ 立即浇透水 / 浸盆
    │                      │
    │                      └── 观察 2-4 小时
    │                          │
    │                          └── 恢复？── 是 ──→ 正常养护
    │                                              │
    │                                              └── 否 ──→ 检查根系
    │
    └── 土壤湿润？── 是 ──→ 检查根系是否腐烂
                             │
                             ├── 烂根 ──→ 翻盆修根换土
                             │
                             └── 无烂根 ──→ 增加通风，减少浇水
```

### 叶片发黄急救流程

```
叶片发黄
    │
    ├── 老叶发黄，新叶正常？
    │       │
    │       ├── 是 ──→ 自然老化，无需处理
    │       │
    │       └── 否 ──→ 检查新叶
    │                   │
    │                   ├── 新叶也黄 ──→ 缺铁或浇水过多
    │                   │                │
    │                   │                ├── 缺铁 ──→ 施硫酸亚铁
    │                   │                │
    │                   │                └── 浇水过多 ──→ 控水通风
    │                   │
    │                   └── 嫩叶萎蔫 ──→ 可能是肥害或缺水
    │                                   │
    │                                   └── 大量浇水冲淡
    │
    └── 叶缘焦枯？── 是 ──→ 光照过强或施肥过量
                            │
                            ├── 光照问题 ──→ 移至阴凉处
                            │
                            └── 施肥问题 ──→ 大量浇水冲淡
```

### 烂根急救流程

```
发现烂根迹象（萎蔫、茎软、土壤异味）
    │
    ├── 立即脱盆
    │
    ├── 清理旧土
    │
    ├── 检查根系
    │   │
    │   ├── 烂根比例 < 50% ──→ 保留健康根系
    │   │                        │
    │   │                        ├── 剪除烂根
    │   │                        ├── 多菌灵浸泡 20 分钟
    │   │                        ├── 晾干
    │   │                        └── 换新土上盆
    │   │
    │   └── 烂根比例 > 50% ──→ 保留健康茎叶
    │                            │
    │                            ├── 剪除烂根和大部分叶片
    │                            ├── 多菌灵浸泡
    │                            ├── 晾干
    │                            └── 重新扦插
    │
    └── 上盆后养护
            │
            ├── 放在阴凉通风处
            ├── 控制浇水
            ├── 不施肥
            └── 等待新根长出（约 2-4 周）
```
