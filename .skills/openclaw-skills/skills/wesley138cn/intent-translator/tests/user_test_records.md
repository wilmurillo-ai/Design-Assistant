# Intent Translation Skill - 用户测试记录

## 测试 1：小美 - 股权分配决策

**时间**: 2026-03-21  
**用户类型**: 艺术生设计师（文科生典型代表）  
**场景**: 和朋友合伙开设计工作室，不知道怎么分股份

### 测试流程

| 轮次 | Skill 提问 | 小美回答 | 效果 |
|-----|-----------|---------|-----|
| Q1 | 你说的"公平"是指按出钱分、按干活分、还是大家平均？ | "说不清楚"（选4） | 触发追问机制 ✅ |
| Q1-追问 | 如果最后分出来让某个人觉得吃亏，你最不希望是谁？ | "我自己"（选A） | 挖出真实顾虑 ✅ |
| Q2 | 每个人具体出什么？钱、时间、技术、资源？ | 小美全职、A出钱+干活、B全职运营 | 明确贡献结构 ✅ |
| Q3 | 一年后最担心的场景？ | "决策僵局"（选5） | 识别核心风险 ✅ |
| Q4 | 绝对不能接受的底线？ | "三人平权"（选4） | 明确约束条件 ✅ |

### 生成的结构化 Prompt

```
【角色】股权分配与治理顾问

【背景】
- 发起人小美不希望自己在分配中吃亏
- 三人都有实际贡献（出资+全职/兼职投入）
- 最担心决策僵局（三人意见不一无法推进）

【核心诉求】
- 自己别吃亏
- 三人平权决策

【约束条件】
- 必须三人平权决策，不能有人独断
- 不能伤感情

【目标】
给出具体的股权比例建议 + 决策权设计 + 退出机制
```

### 测试结果对比

| 指标 | 直接问 AI | 用 Skill 翻译 |
|-----|----------|--------------|
| 参与度 | 一次性提问 | 4轮对话，挖掘真实诉求 |
| 精准度 | 泛泛的"按贡献分配" | 明确"三人平权+防僵局" |
| 信心 | "不知道 AI 懂不懂我" | "这个 Prompt 可以直接用" |
| 时间 | 5分钟出模糊答案 | 10分钟出精准指令 |

### 关键洞察

1. **追问机制有效**: 当小美说"说不清楚"时，换角度问"最不希望谁吃亏"成功挖掘出真实顾虑
2. **情感优先原则**: 问题设计聚焦"担心""底线"等情感因素，而非专业术语
3. **输出可用性**: 生成的 Prompt 结构完整，可直接复制给任意 AI 使用

### 改进建议

- [ ] 增加场景自动识别（用户说"股权"自动加载 equity 场景）
- [ ] 优化追问逻辑（当用户选择"说不清楚"时提供更多引导选项）
- [ ] 添加示例输出（让用户预览最终 Prompt 长什么样）
