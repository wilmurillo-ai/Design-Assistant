# Brainhack — ADHD Sidekick

**The first AI sidekick built for ADHD minds.**

Not a productivity app. Not a chatbot with generic prompts. A fully configured agent that already knows how ADHD brains work, already speaks the language, and has the systems baked in.

The shift: from "here are prompts to try" to "your sidekick is already running."

## Setup

1. **Install the pack** — `openclaw skills install brainhack-adhd-sidekick`
2. **Run onboarding** — the agent asks 4 questions in casual conversation (~2 min) and populates USER.md
3. **Start talking** — ROUTER.md maps what users say to the right skill automatically
4. **Memory builds** — MEMORY.md accumulates patterns over time; the agent gets better as it learns

---

## What Makes This Different

**1. Agent, not app.** Lives in Telegram or WhatsApp. No app to download, forget about, and feel guilty about.

**2. It learns you.** Memory means it adapts — it remembers that mornings are hard, that you work better with background noise, that Tuesday is your hard day.

**3. Emotional intelligence first.** Detects spirals, shutdown, and overwhelm and responds accordingly — before touching any task.

**4. Designed for inconsistency.** No punishing streak mechanics. No guilt for missing a day. It just shows up tomorrow like it never happened.

**5. Modular.** Runs all 15 skills or just the ones you need.
