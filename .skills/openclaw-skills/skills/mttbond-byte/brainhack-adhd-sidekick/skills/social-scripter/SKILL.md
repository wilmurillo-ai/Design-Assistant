# Social Scripter

## Purpose
ADHD brains often know what they need to communicate but can't figure out the words — especially when there's emotional charge. This skill removes the blank-page paralysis.

## Trigger
- "Help me write..."
- "How do I say..."
- "I need to cancel but..."
- "I've been avoiding this email"
- "I don't know how to respond to..."
- Social anxiety detected around communication

## Process

### Step 1: Understand the situation
Who, what, what's the emotional charge?
- "Who is this to and what's the situation?"
- "What are you afraid will happen if you send it?"

### Step 2: Identify the goal
What does this message need to DO?
- Cancel without guilt
- Set a boundary
- Apologize
- Ask for something (extension, help, information)
- Respond professionally when annoyed
- Follow up after accidentally ghosting
- Just reply to something you've been avoiding

### Step 3: Generate 2-3 options
Different tones for the same message:
- Casual / warm
- Professional / neutral
- Direct but kind

Let the user pick and then customize.

### Step 4: Offer rehearsal
If it's a verbal conversation: "Want to practice how this might go? I can play the other person."

## Common Script Templates

### Cancel plans without guilt
```
"Hey [name], I'm so sorry — I have to bail on [thing] tonight. I'm really not in a good place and I know I'd be terrible company. Can we reschedule? I really want to actually hang out with you."
```

### Ask for a deadline extension
```
"Hi [name], I wanted to flag that I'm going to need a bit more time on [project]. I ran into [brief honest reason] and I want to make sure I give it the attention it deserves. Would [new date] work?"
```

### Reply to something you've been avoiding
```
"Hey — I'm so sorry for the delayed response. Things got busy and I didn't want to give you a rushed reply. [Answer/response to their message]. Thanks for your patience."
```

### Set a boundary with a friend/family
```
"I really value our relationship and I want to be honest: when [specific thing], I feel [feeling]. I'd really appreciate it if we could [specific change]. I'm not trying to make a big deal of it — I just want to be upfront."
```

### Respond professionally when annoyed
```
"Thanks for flagging this. I'll take a look and follow up. [Or: I've reviewed this and here's my perspective: ___]"
(cool, professional, nothing they can use against you)
```

### Follow up after accidentally ghosting
```
"Hey — I am so sorry, I completely dropped the ball on responding to this. [Brief honest reason or just: life got away from me]. Still very much interested/here/thinking about this — [answer to original message]."
```

## Rules
- Never write something that sounds unnatural or over-formal for the relationship
- Always offer multiple tones — never just one
- If they seem very anxious: normalize first. "This kind of thing is hard for a lot of people. Let's write it together."
- Never shame them for avoiding it in the first place
