# Check-In

## Purpose
ADHD brains often rush from one thing to the next without pausing to register what went well. This skill creates a daily touchpoint — even a 2-minute one — that builds self-awareness and prevents open loops from becoming overnight anxiety.

## Trigger
- Evening message (default route if no specific ask)
- "Check in"
- "How'd I do today"
- Heartbeat/cron at user's preferred evening time
- End of a long session

## Process

### Step 1: Temperature check
"How are you feeling about today? Scale of 1-10 is fine, or just tell me."

Don't make it a big deal. One question. Let them answer however they want.

### Step 2: Surface wins
Pull from MEMORY.md and session history — what got done today? What counts as a win?

Celebrate explicitly, even if small. Especially if small.

"You did [X]. That counts. For real."

If nothing feels like a win: "You showed up and got through the day. That's something."

### Step 3: Capture open loops
"Anything hanging over you that we should log for tomorrow?"

If yes: quick capture, don't solve tonight. Add to MEMORY.md for tomorrow's day-architect.

If no: perfect. Clean slate.

### Step 4: Emotional reading
If score was low (1-4): gentle acknowledgment. Don't try to fix it tonight.
"That sounds like it was a rough one. Rest is the right call."

If score was high (7-10): celebrate.
"That's a solid day. You did that."

If medium: normalize.
"Middle-of-the-road days are the majority of life. Not every day has to be a win."

### Step 5: Close warmly
"Rest up. Tomorrow's a new one."

or: "You did what you could today. That's enough."

or: "Good night. I'll be here tomorrow."

### Step 6: Log
In MEMORY.md: date, mood score, wins logged, open loops captured.
Update 7-day rolling emotional baseline.

## Rules
- Never turn the check-in into a task session
- If user starts problem-solving a big issue at check-in time: "Let's catch that one tomorrow when your brain is fresh. For now — close the laptop."
- Keep it SHORT. This should feel like a gentle landing, not an interview.
- If user is mid-spiral: transition to spiral-catcher, come back to check-in when calmer.

## Output Example

```
Hey — how are you feeling about today? 1-10 or just tell me.

[User: 5 — it was okay, got some stuff done]

5 is solid. You got [X] done — that's real forward motion. 

Anything still floating around in your head that we should grab before bed?

[User: Yeah I still need to reply to that email]

Got it — I'll remind you tomorrow morning. Not tonight's problem.

Rest up. You did what you could. 🌙
```
