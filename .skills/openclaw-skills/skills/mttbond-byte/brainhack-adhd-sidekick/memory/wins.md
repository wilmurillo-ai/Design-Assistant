# Wins Log

> Append-only. Every win logged here.
> This file is evidence. Pull it during spirals. Build weekly highlights from it.

---

## Format (Required — two lines per entry)

```
[YYYY-MM-DD] — [What they accomplished] — [Why it mattered / context]
→ PULL: "[One sentence, past tense, concrete — ready to read aloud during a spiral]"
```

The `→ PULL` line is mandatory. It's what gets surfaced during spiral-catcher. Write it so it can be read directly to the user without editing. Specific. No hedging. No "you kind of did..."

**Example:**
```
2026-03-16 — Replied to 4 emails they'd been avoiding for a week — cleared a loop that was causing background anxiety
→ PULL: "You replied to 4 emails you'd been avoiding for a week. That's real."

2026-03-16 — Made the dentist appointment — had been on the list for 3 months
→ PULL: "You made the dentist appointment. Three months on your list. You did it."

2026-03-16 — Finished the project draft — said it felt impossible at the start of the session
→ PULL: "You finished the project draft. You said it felt impossible. It wasn't."
```

## Spiral Pull Protocol

When spiral-catcher requests counter-evidence:

1. Pull the 3 most recent `→ PULL` lines from this file
2. Present as a list, no preamble — just: "Here's what you've actually done:"
3. Read them exactly as written — don't paraphrase or soften
4. Then stop. Wait. Don't add "see, you're doing great!" — let the evidence land.
5. If user dismisses ("that doesn't count") — don't argue. "I hear that. It still happened."

**Timing rule:** Only surface wins after the user has been acknowledged and grounded (Steps 2-3 of spiral-catcher). Not before. Counter-evidence before acknowledgment feels dismissive.

## Weekly Highlights Template

Generated every Friday during check-in. Pull all `→ PULL` lines from the current week:

```
Real talk — here's what you actually did this week:
• [PULL line 1]
• [PULL line 2]
• [PULL line 3]
(+ N more)

That's not nothing. That's a real week.
```

If fewer than 3 wins logged: "Short week on the scoreboard. You still showed up. That counts too."
