# Citation Formatting Rules

Three supported styles. The LLM must format all in-text citations and reference list entries exactly as specified below.

---

## APA 7th Edition

### In-Text Citations

| Situation | Parenthetical | Narrative |
|-----------|--------------|-----------|
| 1 author | (Smith, 2023) | Smith (2023) found... |
| 2 authors | (Smith & Lee, 2023) | Smith and Lee (2023) argued... |
| 3+ authors | (Smith et al., 2023) | Smith et al. (2023) demonstrated... |
| No author | ("Article Title," 2023) | "Article Title" (2023) states... |
| No date | (Smith, n.d.) | Smith (n.d.) noted... |
| Direct quote | (Smith, 2023, p. 45) | Smith (2023) stated "quote" (p. 45). |
| Multiple works | (Chen, 2021; Smith, 2023) | - |
| Same author, same year | (Smith, 2023a, 2023b) | - |
| Secondary source | (as cited in Smith, 2023) | - |

### Reference List (alphabetical by first author surname)

**Journal article:**
```
Smith, J. A., & Lee, K. (2023). Article title in sentence case. Journal Name in Title Case, 40(2), 112-128. https://doi.org/10.xxxx/xxxxx
```

**Book:**
```
Smith, J. A. (2023). Book title in sentence case (3rd ed.). Publisher Name.
```

**Book chapter:**
```
Smith, J. A. (2023). Chapter title. In K. Lee (Ed.), Book title (pp. 45-67). Publisher Name.
```

**Website:**
```
Smith, J. A. (2023, March 15). Page title. Site Name. https://www.example.com/page
```

### APA Rules
- Use "&" in parenthetical citations and reference list; use "and" in narrative citations
- Italicize journal names, book titles, and volume numbers
- Use sentence case for article/chapter titles; title case for journal/book names
- Include DOI as a URL when available

## MLA 9th Edition

### In-Text Citations

| Situation | Format |
|-----------|--------|
| 1 author | (Smith 45) |
| 2 authors | (Smith and Lee 45) |
| 3+ authors | (Smith et al. 45) |
| No author | ("Article Title" 45) |
| No page number | (Smith) |
| Multiple works | (Smith 45; Lee 12) |
| Direct quote | (Smith 45) - page number required |

### Works Cited (alphabetical by first author surname)

**Journal article:**
```
Smith, John A., and Karen Lee. "Article Title in Title Case." Journal Name, vol. 40, no. 2, 2023, pp. 112-28. DOI or URL.
```

**Book:**
```
Smith, John A. Book Title in Title Case. Publisher, 2023.
```

### MLA Rules
- Use "and" (never "&") between authors
- Italicize book and journal titles; use quotation marks for article/chapter titles
- Title case for all titles
- Include page numbers in in-text citations whenever possible

## Chicago 17th Edition (Author-Date)

### In-Text Citations

| Situation | Parenthetical | Narrative |
|-----------|--------------|-----------|
| 1 author | (Smith 2023) | Smith (2023) argues... |
| 2 authors | (Smith and Lee 2023) | Smith and Lee (2023) found... |
| 4+ authors | (Smith et al. 2023) | Smith et al. (2023) show... |
| Direct quote | (Smith 2023, 45) | Smith (2023, 45) states "quote." |
| Multiple works | (Chen 2021; Smith 2023) | - |

### Bibliography (alphabetical by first author surname)

**Journal article:**
```
Smith, John A., and Karen Lee. 2023. "Article Title in Title Case." Journal Name 40 (2): 112-28. https://doi.org/10.xxxx/xxxxx.
```

**Book:**
```
Smith, John A. 2023. Book Title in Title Case. 3rd ed. City: Publisher.
```

### Chicago Rules
- Use "and" (never "&") between authors
- Year follows author name(s) directly
- Italicize book and journal titles; use quotation marks for article/chapter titles

---

## Cross-Style Rules

### Direct Quotes
- Short quotes: integrate inline with quotation marks
- Long quotes: use a block quote (indented, no quotation marks)
- Always include a page number for direct quotes

### What NOT to Do
- Never fabricate a source
- Never alter a real author name, title, year, or DOI
- Never cite a source not in the reference list, and vice versa
- Never use "According to various studies" without naming them
