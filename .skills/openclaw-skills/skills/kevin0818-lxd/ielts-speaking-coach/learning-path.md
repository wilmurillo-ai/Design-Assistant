## 2. 词汇进阶阶梯

### Band 5→6：从 generic 过渡到 topic-specific

**目标特征**: d=2-3, register=spoken/neutral，能在熟悉话题中使用场景化替换词

**典型进阶路线**（base → ZPD targets）:
- good → solid, decent, worthwhile
- bad → rough, disappointing, tricky
- important → meaningful, crucial, essential
- interesting → engaging, fascinating, eye-opening
- like → be into, be fond of, be passionate about
- think → believe, suppose, reckon
- big → massive, major, enormous
- help → support, give a hand, pitch in
- say → mention, point out, bring up
- really → genuinely, incredibly, honestly

**搭配模式**: 开始注意 topic-specific 替换（如 food 语境用 flavourful 而非 good；person 语境用 supportive/considerate 而非 nice）

### Band 6→7：从 topic-specific 过渡到 less common & idiomatic

**目标特征**: d=3-4, register=neutral/formal，能自然使用 idiomatic 表达和精准搭配

**典型进阶路线**:
- good → outstanding, top-notch（performance 语境）
- happy → thrilled, over the moon, fulfilled
- important → paramount, pivotal, far-reaching
- think → reflect on, contemplate, mull over
- interesting → captivating, intriguing, compelling
- problem → dilemma, hurdle, stumbling block
- big → wide-ranging, extensive, significant
- beautiful → breathtaking, picturesque, striking
- change → transform, reshape, shift
- start → embark on, venture into, dive into

**搭配模式**: 掌握 collocation 精度（pretty + demanding, highly + beneficial），注意 register 切换（Part 3 可用 formal）

### Band 7→8：精准化 + 学术语域 + 低频精确词

**目标特征**: d=4-5, register=flexible，能灵活使用 uncommon items 和 idiomatic language

**典型进阶路线**:
- important → indispensable（d=5, academic）
- need → necessitate（d=5, academic）
- bad → deteriorating（health 语境, d=4）
- boring → mundane, tedious, monotonous
- use → leverage, employ（technology/formal 语境）
- tell → narrate, recount（story 语境）
- show → illustrate, demonstrate（data 语境）
- beautiful → exquisite, radiant（person 语境）
- feel → perceive（deep 语境, d=4）
- believe → to the best of my knowledge（d=4, formal）

**搭配模式**: 掌握 paraphrasing 技巧（同一概念多种表达），discourse-level 词汇切换

## 3. 语法进阶阶梯

### Band 5→6 语法目标

**当前水平**: 基本简单句控制良好，复杂结构有尝试但常出错
**ZPD 目标结构**:
1. **关系从句**（who/which/that）: "The teacher who taught me was very patient."
2. **时间/原因状语从句**（when/because/although）: "Although I was nervous, I managed to finish."
3. **基础条件句**（if + would）: "If I had more time, I would travel more."
4. **比较结构**（more...than / as...as）: "It's way more convenient than before."
5. **被动语态**（基础）: "I was told that..."

**密度目标**: 每 18 词至少 0.5 个复杂结构 → 目标提升到 1 个

### Band 6→7 语法目标

**当前水平**: 能使用复合句但灵活度有限，复杂结构常有错误
**ZPD 目标结构**:
1. **完成时态**（present/past perfect）: "I've been studying English for about 10 years."
2. **混合条件句**: "If I had started earlier, I would be fluent by now."
3. **间接引语**: "She mentioned that she had always wanted to..."
4. **强调句**（cleft sentences）: "What I really enjoy is the atmosphere."
5. **高级关系从句**（whose/in which）: "It's a city in which tradition and modernity coexist."

**密度目标**: 每 18 词至少 1 个复杂结构 → 目标提升到 1.5 个

### Band 7→8 语法目标

**当前水平**: 复杂结构使用频繁但偶有错误
**ZPD 目标结构**:
1. **倒装句**: "Not only does it improve health, but it also boosts confidence."
2. **虚拟语气**: "Had I known about it earlier, I would have applied."
3. **复杂名词短语**: "The rapidly growing demand for sustainable solutions..."
4. **高级让步/对比**: "Much as I appreciate technology, I think face-to-face interaction is irreplaceable."
5. **嵌套从句**: "What fascinates me is how people who come from completely different backgrounds can still find common ground."

**密度目标**: 每 18 词至少 1.5 个复杂结构 → 目标提升到 2 个

## 5. 先修链

学习高级词汇前，确保已掌握其先修词汇。以下为核心先修关系：

### 词汇先修链
```
good → decent (d=2) → solid (d=2) → worthwhile (d=3)
bad → rough (d=2) → disappointing (d=2) → problematic (d=3)
think → figure out (d=2) → reflect on (d=3) → contemplate (d=4)
say → point out (d=2) → argue (d=3) → articulate (d=4)
important → crucial (d=3) → paramount (d=4) → indispensable (d=5)
really → absolutely (d=2) → incredibly (d=3) → remarkably (d=4)
like → be into (d=2) → be fond of (d=3) → cherish (d=4)
help → support (d=2) → mentor (d=3) → facilitate (d=4)
interesting → engaging (d=3) → captivating (d=4) → riveting (d=5)
big → massive (d=2) → significant (d=3) → substantial (d=4)
```

### 搭配先修链
```
pretty + good → pretty + demanding (d=3)
highly + important → highly + beneficial (d=3)
really + good → really + solid (d=2) → genuinely + rewarding (d=3)
```

### 语法先修链
```
simple sentence → compound (and/but) → relative clause (who/which)
relative clause → adverbial clause (although/because) → conditional (if...would)
conditional → perfect tense → mixed conditional
mixed conditional → cleft sentence → inversion
```

## 6. 每日练习建议模板

根据 ZPD 区间，建议用户每日完成以下任务：

### Band 5→6 每日练习
- 从本周 ZPD 目标词中选 3 个，各造 2 个口语句子
- 练习 1 个新语法结构，在 Part 1 回答中使用 2 次
- 录音 1 段 1 分钟的 Part 2 回答，注意使用本周目标词

### Band 6→7 每日练习
- 从本周 ZPD 目标词中选 3 个，用不同话题各造 1 个句子（共 3 句）
- 练习 1 个新搭配模式，替换 3 个旧表达
- 录音 1 段 Part 3 讨论回答（2-3 分钟），确保使用 2 个以上复杂结构

### Band 7→8 每日练习
- 选 2 个 d=4-5 的表达，在不同语域中各使用 1 次（spoken vs formal）
- 练习 1 个高级语法结构（如倒装或虚拟），写出 3 个变体
- 录音 1 段完整 Part 2+3 模拟（5 分钟），自我评估 LR 和 GRA 密度
