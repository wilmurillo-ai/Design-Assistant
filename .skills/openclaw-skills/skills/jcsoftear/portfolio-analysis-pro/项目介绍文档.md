# 持仓分析系统 (Portfolio Analysis System)

## 项目概述

持仓分析系统是一个功能完整的股票投资管理工具，提供交互式命令行界面和网页界面，支持持仓管理、实时价格更新、盈亏分析和报告生成等核心功能。

## 项目结构

```
portfolio-analysis/
├── templates/
│   └── index.html        # 网页版前端模板
├── portfolio.db          # SQLite数据库文件
├── portfolio_ui.py       # 交互式命令行界面
├── portfolio_web.py      # 基于Flask的网页版
├── update_all_prices.py  # 批量更新价格工具
├── update_portfolio.py   # 持仓更新和分析脚本
├── simple_app.py         # 简化版应用
├── simple_web_app.py     # 简化版网页应用
└── web_dashboard.py      # 网页仪表盘
```

## 核心功能

### 1. 持仓管理
- **添加持仓**：录入股票代码、名称、持仓数量和成本价
- **编辑持仓**：修改持仓数量、成本价或当前价格
- **删除持仓**：移除不需要的持仓记录
- **持仓列表**：查看所有持仓的详细信息

### 2. 实时价格更新
- **手动更新**：按需更新单个或所有持仓价格
- **自动更新**：网页版支持定时自动更新（默认5分钟）
- **多源获取**：支持从新浪财经和东方财富获取实时价格

### 3. 数据分析
- **持仓总览**：显示总市值、总成本、总盈亏和盈亏比例
- **个股分析**：计算每只股票的盈亏额、盈亏比例和仓位占比
- **盈亏统计**：统计盈利和亏损股票数量
- **操作建议**：基于盈亏情况提供简单的操作建议

### 4. 操作记录
- **自动记录**：所有操作（添加、编辑、删除、更新）都会被记录
- **历史查询**：查看最近的操作记录
- **操作详情**：记录详细的操作内容和时间

### 5. 报告导出
- **生成报告**：导出持仓报告到Markdown文件
- **报告内容**：包含持仓总览、个股详情和盈亏分析

### 6. 网页界面
- **响应式设计**：适配PC和移动设备
- **实时数据**：通过API获取和显示数据
- **交互式操作**：支持添加、编辑、删除持仓
- **数据可视化**：提供图表展示持仓分布

## 技术实现

### 1. 数据存储
- **SQLite数据库**：本地轻量级数据库，存储持仓和操作记录
- **数据表结构**：
  - `holdings`：存储持仓信息（代码、名称、数量、成本价、当前价等）
  - `operation_log`：存储操作记录（操作类型、股票代码、详情、时间）

### 2. 价格获取
- **API调用**：使用requests库调用第三方金融API
- **多源备份**：支持多个数据源，提高数据获取成功率
- **错误处理**：网络异常时的容错处理

### 3. 界面实现
- **命令行界面**：基于Python标准库的交互式界面
- **网页界面**：基于Flask框架和Bootstrap 5
- **API接口**：提供RESTful API供前端调用

### 4. 核心类
- **PortfolioDashboardUI**：命令行界面主类
- **PortfolioWebDashboard**：网页界面主类
- **StockPriceAPI**：股票价格获取类

## 系统特点

1. **轻量级**：基于Python和SQLite，无需复杂的环境配置
2. **多界面**：同时提供命令行和网页界面，满足不同使用场景
3. **实时数据**：自动获取并更新股票实时价格
4. **数据分析**：提供详细的盈亏分析和操作建议
5. **操作记录**：完整记录所有操作历史
6. **局域网访问**：网页版支持局域网内多设备访问
7. **响应式设计**：网页界面适配不同屏幕尺寸

## 使用方法

### 命令行界面
```bash
python portfolio_ui.py
```

### 网页界面
```bash
python portfolio_web.py
```
访问地址：http://localhost:5000 或 http://<本地IP>:5000

### 价格更新
```bash
python update_all_prices.py
```

### 每日汇总分析
```bash
python update_portfolio.py
```

## 系统要求

- Python 3.6+
- 依赖库：
  - requests
  - flask
  - akshare (用于update_portfolio.py)

## 数据安全

- **本地存储**：所有数据存储在本地SQLite数据库
- **备份功能**：支持数据库备份
- **操作记录**：完整记录所有操作，便于追溯
