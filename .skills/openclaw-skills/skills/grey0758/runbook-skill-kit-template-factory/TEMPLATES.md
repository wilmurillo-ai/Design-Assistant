## Skill Bundle Template

```text
<skill-name>/
  SKILL.md
  README.md
  WORKFLOW.md
  FAQ.md
  CHANGELOG.md
```

## SKILL.md Template Hints

- include a precise trigger description
- reference bundled docs with `{baseDir}`
- keep the rules and constraints explicit
- keep the bundle self-contained
