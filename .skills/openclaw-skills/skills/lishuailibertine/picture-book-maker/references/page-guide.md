## 页面布局原则

### 图文比例

**全图页（Full Image）**
- 适用：情节高潮、情感强烈、需要沉浸感的场景
- 文字：极少或无文字（10-30字）
- 布局：文字通常在空白角落或使用半透明背景
- 例子：
  ```
  画面：全景展现森林大火，小熊勇敢地冲向被困的小兔
  文字："别怕，我来救你！"
  ```

**主图+文字区（Image + Text Area）**
- 适用：大多数场景，平衡叙事和视觉
- 文字：中等（50-100字）
- 布局：画面占70-80%，底部或侧边留出文字区
- 例子：
  ```
  画面：中景，小熊和小兔在河边交谈
  文字区（底部）：小熊低下头，轻声说："我一直在找你。小兔抬起头，眼泪还挂在脸上。
  ```

**分割页（Split Page）**
- 适用：对比两个场景、同时发生的两个事件
- 文字：可分两段（50-80字×2）
- 布局：页面分为上下或左右两部分
- 例子：
  ```
  左侧：森林里，小熊焦急地寻找
  右侧：山洞中，小兔孤单地等待
  文字左："小熊，小熊，你在哪里？"
  文字右："谁来救救我……"
  ```

**多格页（Panel Page）**
- 适用：快速展现多个连续动作
- 文字：每格少量文字（20-40字×3-4格）
- 布局：2×2或3×2的网格布局
- 例子：
  ```
  格1：小熊听到哭声
  格2：小熊循声跑去
  格3：小熊发现山洞
  格4：小熊冲进山洞
  文字1："什么声音？"
  文字2："等等我！"
  文字3："找到了！"
  文字4："我来了！"
  ```

### 视觉层次

**焦点处理**
- 主要角色：位置稍偏中心，体型略大
- 重要物品：通过光线、色彩、位置突出
- 背景元素：简化处理，避免喧宾夺主

**引导视线**
- 从左到右、从上到下（符合阅读习惯）
- 使用人物的视线引导观众
- 利用线条、道路等元素引导

### 构图技巧

**中心构图**
- 主体在画面中心
- 适用于：角色特写、重要对话
- 效果：稳定、突出主体

**三分法构图**
- 主体在三分线的交叉点
- 适用于：风景场景、动态画面
- 效果：平衡、有活力

**对角线构图**
- 主体沿对角线排列
- 适用于：动作场面、对比场景
- 效果：动感、张力

**框架构图**
- 利用前景元素形成框架
- 适用于：强调探索、发现
- 效果：引导视线、增加深度

## 画面描述技巧

### 完整的画面描述模板

```markdown
## 页面 X：[场景名称]

### 基本信息
- **情节节点**：[对应故事节拍]
- **情绪基调**：[2-3个情绪词]
- **构图方式**：[如"中心构图，中景"]

### 画面元素
**场景**
- 环境：[具体地点+时间+天气]
- 背景：[远处景色，简化处理]
- 前景：[近处物体，可增加层次感]

**角色**
- 主角：[位置+动作+表情+服饰细节]
- 配角：[如有，同上]

**光影与色彩**
- 光源：[方向+色温，如"从左上方射来的温暖阳光"]
- 主色调：[2-3种主色]
- 色彩倾向：[如"温暖明亮"、"冷色调"、"高对比"]

### 画面描述（直接用于生成图像）
[用一段自然流畅的语言描述整个画面，确保包含所有关键元素]

### 文字安排
- **文字内容**：[该页的完整文字]
- **文字位置**：[如"底部居中"、"右上角空白区"]
- **文字样式建议**：[如"白色文字+半透明黑色背景"]
```

### 描述质量检查清单

**角色一致性**
- [ ] 角色外貌特征准确
- [ ] 表情符合情节和性格
- [ ] 服装/配饰保持一致
- [ ] 角色大小比例合理

**场景一致性**
- [ ] 场景符合故事设定
- [ ] 光线方向连贯
- [ ] 天气/时间符合情节
- [ ] 色调与情绪匹配

**构图合理性**
- [ ] 焦点突出，层次分明
- [ ] 视线引导清晰
- [ ] 留白适当，不拥挤
- [ ] 与上一页、下一页有视觉连贯性

**文字适配**
- [ ] 文字量适合目标年龄
- [ ] 文字位置不遮挡关键画面
- [ ] 文字内容推动情节
- [ ] 文字语言风格统一

### 情绪渲染技巧

**通过颜色表达情绪**
```
开心/温暖 → 暖色（黄色、橙色、粉色）
平静/宁静 → 柔和色（浅蓝、浅绿、米白）
悲伤/失落 → 冷色（蓝色、灰色、紫色）
紧张/恐惧 → 对比色（红黑、深蓝+亮黄）
神秘/奇幻 → 深邃色（深紫、深蓝、金色点缀）
```

**通过视角表达情绪**
```
亲切/平等 → 平视（与角色眼睛同高）
渺小/脆弱 → 仰视（从下往上拍）
强大/威严 → 俯视（从上往下拍）
神秘/未知 → 远景（小人物在大环境中）
亲密/温暖 → 特写（近距离聚焦表情）
```

**通过构图表达情绪**
```
稳定/安全 → 居中构图，对称平衡
冲突/矛盾 → 分割构图，对比强烈
动态/活力 → 对角线构图，倾斜感
开放/自由 → 留白构图，广阔空间
```

## 不同类型场景

### 开场场景
**目标**：建立世界，介绍主角

**画面特点**
- 宏观视角，展现环境
- 主角自然地融入场景
- 色调温暖，建立友好氛围

**描述示例**
```
画面：清晨的森林，阳光透过树叶洒下金色的光斑。小熊坐在河边，手中拿着钓鱼竿，神情悠闲。远处的山脉轮廓在晨雾中若隐若现。整体色调温暖明亮，使用柔和的绿色、棕色和黄色。
```

### 对话场景
**目标**：展现角色互动，推动情节

**画面特点**
- 中景或近景，聚焦角色表情
- 角色位置体现关系（并排、对峙、拥抱等）
- 背景简化，避免干扰

**描述示例**
```
画面：中景，小熊和小兔并排坐在草地上。小熊转头看向小兔，表情关切。小兔低着头，肩膀微微抽动，手中攥着一块手帕。背景是模糊的绿色草地和远处的树林，光线柔和。色调以暖色为主，体现关怀。
```

### 动作场景
**目标**：展现动态，增加张力

**画面特点**
- 动态构图（对角线、倾斜）
- 捕捉动作关键瞬间
- 使用速度线或动态模糊（风格化处理）

**描述示例**
```
画面：小熊在森林中奔跑，身体前倾，双臂张开保持平衡。树叶在他身边飞舞，表现出速度感。镜头从右下角向上仰拍，突出小熊的勇敢。背景是模糊的绿色线条，增强动态感。色调明亮，充满活力。
```

### 情感场景
**目标**：表达内心世界，引发共情

**画面特点**
- 特写或大特写，聚焦表情
- 光线强化情绪（侧光、逆光、柔光）
- 背景虚化或简化

**描述示例**
```
画面：大特写，聚焦小熊的脸庞。他微微皱眉，眼神中透露出犹豫和担忧。光线从侧上方照射，在他脸上形成柔和的阴影，增强立体感。背景完全虚化，只有温暖的橙色光晕。色调以暖色为主，体现内心的温柔。
```

### 高潮场景
**目标**：情节顶点，情感爆发

**画面特点**
- 戏剧性构图
- 强烈的色彩对比
- 突出关键动作或情感

**描述示例**
```
画面：全景，小熊站在高高的岩石上，张开双臂拥抱初升的太阳。阳光金灿灿的，照亮整个森林，小熊的轮廓被光线勾勒得闪闪发光。下方的动物们抬头仰望，表情惊叹。色调高对比，金色与深色森林形成强烈对比。
```

### 结局场景
**目标**：情感释放，留下余韵

**画面特点**
- 和谐的构图
- 温暖的色调
- 象征性元素

**描述示例**
```
画面：夕阳西下，森林笼罩在柔和的金色和紫色光晕中。小熊和小兔并肩坐在山顶，看着远处的落日。他们的影子被拉得很长，交织在一起。天空中有几只鸟儿飞过，象征着自由和希望。色调温暖而宁静，营造满足和温馨的氛围。
```

## 风格与色调

### 常见绘本风格

**卡通风格**
- 特点：线条简洁、色彩鲜艳、造型夸张
- 适用：低龄儿童、轻松主题
- 描述要点：强调轮廓，颜色饱和度高

**写实风格**
- 特点：细节丰富、光影真实、色彩自然
- 适用：科普类、自然主题
- 描述要点：强调质感，光影准确

**水彩风格**
- 特点：色彩晕染、笔触柔和、透明感强
- 适用：温柔主题、情感表达
- 描述要点：强调色彩渐变和笔触痕迹

**水墨风格**
- 特点：黑白为主、留白意境、笔法自由
- 适用：传统文化、哲学主题
- 描述要点：强调墨色层次和留白

**剪纸风格**
- 特点：色块清晰、边缘硬朗、层次分明
- 适用：民间故事、手工感
- 描述要点：强调色块对比和轮廓清晰

### 色彩统一方案

**单色系方案**
- 选择一个主色，使用不同明度和纯度的变化
- 优点：统一感强，适合情感集中
- 示例：绿色系（森林主题）、蓝色系（海洋主题）

**对比色方案**
- 选择一对对比色，营造视觉张力
- 优点：突出重点，适合冲突情节
- 示例：橙+蓝（活力）、红+绿（节日）

**邻近色方案**
- 选择色环上相邻的3-4种颜色
- 优点：和谐自然，适合日常场景
- 示例：黄+橙+红（温暖）、蓝+青+绿（宁静）

**三色方案**
- 选择三种颜色作为主色调
- 比例：60%主色+30%副色+10%点缀色
- 优点：丰富但不杂乱，适合大多数场景

### 风格一致性维护

**角色设计统一**
- 所有角色使用相同的线条风格（粗细、硬度）
- 所有角色使用相同的阴影方式（平涂、渐变等）
- 所有角色使用相同的色彩饱和度

**场景风格统一**
- 所有场景使用相同的透视规律
- 所有场景使用相同的背景处理方式（简化程度、细节程度）
- 所有场景使用相同的笔触风格

**全局色调统一**
- 确定整本书的主色调
- 确定情绪色调的变化规律（如：开场明亮→发展复杂→高潮强烈→结局温暖）
- 避免一页与周围页面色调差异过大
