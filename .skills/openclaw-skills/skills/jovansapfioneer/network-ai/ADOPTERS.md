## How to Add Yourself

1. Fork the repository
2. Add a row to the appropriate table above
3. Columns:
   - **Organization / Project**: your company or project name
   - **Use Case**: one-line description (e.g. "Multi-agent code review pipeline", "Federated budget enforcement across 5 LLM agents")
   - **Since**: version or month you started (e.g. `v4.0.0` or `2026-02`)
   - **Public Reference**: optional — link to a blog post, repo, or leave blank
4. Open a pull request titled `adopters: add <your org/project name>`

No email required. Anonymous entries (use case only, no org name) are accepted.
