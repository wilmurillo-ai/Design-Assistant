# Shot List — Structure Reference
*Compiled from StudioBinder, Boords, and Milanote industry standards*

## What Is a Shot List?
A shot list is a strategic document that lists all the shots to be captured in a specific shoot day (or entire production). It's how the director communicates the visual plan to the DP and 1st AD.

## Core Elements (Required)
Every shot list must contain:
1. **Scene Number** — syncs with script and schedule
2. **Shot Number** — sequential within each scene (reset to 1 for each new setup)
3. **Shot Description** — what's happening in the shot
4. **Shot Size** — ECU / CU / MCU / MS / MWS / WS / XWS / OTS
5. **Camera Movement** — Static / Pan / Tilt / Dolly / Handheld / Steadicam / Crane / Drone

## Extended Elements (Professional)
6. **Angle** — Eye Level / High / Low / Dutch / POV / Bird's Eye
7. **Lens** — e.g., 35mm, 50mm, 85mm, 24mm, anamorphic
8. **Frame Rate** — 24fps / 48fps / 120fps (slow-mo)
9. **Equipment Required** — Steadicam, jib, slider, drone, gimbal
10. **Audio Notes** — Dialogue / Ambience / Wild lines / MOS
11. **Notes** — Lighting setup, mood, reference image, special FX
12. **VFX Tag** — Whether shot requires VFX in post
13. **Stunt Flag** — Shots involving stunts or special risk
14. **Est. Duration** — Estimated minutes to achieve the shot

## Shot Sizes Reference
| Code | Name | Description |
|------|------|-------------|
| ECU | Extreme Close-Up | Eyes, hands, detail |
| CU | Close-Up | Head and shoulders |
| MCU | Medium Close-Up | Chest up |
| MS | Medium Shot | Waist up |
| MWS | Medium Wide Shot | Full body with context |
| WS | Wide Shot | Full environment |
| XWS | Extreme Wide Shot | Establishing, geography |
| OTS | Over the Shoulder | Dialogue coverage |
| POV | Point of View | Subjective shot |
| 2S | Two Shot | Two subjects |
| INS | Insert | Object detail |

## Camera Movement Reference
- **Static** — Locked-off, no movement
- **Pan** — Horizontal rotation on tripod
- **Tilt** — Vertical rotation on tripod
- **Dolly In/Out** — Track-based push or pull
- **Track Left/Right** — Lateral movement on tracks
- **Handheld** — Camera operator carrying camera
- **Steadicam** — Stabilized walking movement
- **Crane/Jib** — Arm-mounted vertical movement
- **Drone** — Aerial
- **Zoom In/Out** — Optical zoom (different from dolly)

## Shot List Best Practices
- Build by scene, number shots within each scene
- Add reference images or storyboard thumbnails when possible
- Color-code shot types for quick visual scanning
- Share with DP 48hrs before shoot for equipment prep
- On-set: 1st AD tracks completed shots against the list
- Mark PRIORITY shots — shots you MUST get vs. nice-to-haves
