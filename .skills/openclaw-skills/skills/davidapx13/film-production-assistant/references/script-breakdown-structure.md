# Script Breakdown — Structure Reference
*Compiled from StudioBinder, Filmustage, and The Film Fund*

## What Is a Script Breakdown?
The first step in pre-production: analyzing the screenplay scene-by-scene to extract every element needed to shoot it. Drives budgeting, scheduling, and departmental planning.

## The Two Core Concepts

### 1. Page Eighths
Each script page is divided into 8 equal sections. Scenes are measured in eighths:
- A scene spanning half a page = 4/8
- A 1.5 page scene = 1 4/8
- Faster to schedule and estimate time than whole pages

**Rule of thumb:**
- 1 page = ~1 minute of screen time
- 8/8 page = 1 full day for complex scenes (rarely)
- 1/8 page = fast pickup shot or single line exchange

### 2. Breakdown Sheet (One Per Scene)
One sheet per scene in the film. Contains all elements required for that specific scene.

## Script Breakdown Sheet Template

```
PRODUCTION: [Title]          BREAKDOWN PAGE: ___
SCENE #: ___                 SCENE NAME: _______________
INT/EXT: ___                 DAY/NIGHT: ___
LOCATION: _______________    PAGE COUNT: ___ / 8
STORY DAY: ___              SEQUENCE: ___

SYNOPSIS:
[2-3 sentence description of what happens]

ELEMENT CATEGORIES:
```

## Element Categories (Color-Coded System)

| Color | Category | What Goes Here |
|-------|----------|---------------|
| 🔴 Red | Cast (Speaking Roles) | All speaking characters |
| 🟡 Yellow | Extras / Background | Non-speaking atmosphere |
| 🟢 Green | Extras Atmosphere | Specific background talent |
| 🔵 Blue | Special Effects (SFX) | Practical on-set FX (rain, fire, explosions) |
| 🔹 Light Blue | VFX | Visual effects to be added in post |
| 🟣 Purple | Props | Objects actors interact with |
| ⚪ Gray | Set Dressing | Furniture, decoration (non-interactive) |
| ⭕ Circle | Wardrobe / Costumes | Character outfits (per story day) |
| ✨ Asterisk | Hair & Makeup | Special makeup, prosthetics, wounds |
| 💗 Pink | Vehicles / Animals | Picture cars, live animals |
| 🟠 Orange | Stunts | Any risky sequences |
| 🟤 Brown | Sound Notes | Key audio, wild lines, MOS shots |
| 🟡 Gold | Lighting Notes | Practical lights, natural light timing |
| ⬛ Box | Camera / Special Equipment | Drones, cranes, Steadicam |

## Detailed Category Breakdown

### Cast (Red)
- Speaking roles only (actors with lines)
- Note character names + cast numbers assigned
- Include any featured background who react on camera
- Child actors require guardian + tutors on set
- Animal trainers included when animals present

### Props (Purple)
- Items that actors PHYSICALLY USE or interact with
- Not set decoration (things just sitting there)
- Examples: phone, gun, briefcase, cup of coffee
- Flag: consumables (food, breakable items — need multiples)
- Flag: hero props (need picture-perfect condition)

### Set Dressing (Gray)
- Items that dress the set but aren't touched
- Examples: paintings on wall, furniture, lamps
- Set decorator's domain (not props department)

### Wardrobe (Circle)
- Label by: Character Name + Story Day + Outfit Number
- Flag items that get damaged/destroyed in scene (need multiples)
- Continuity critical: must match across coverage

### Makeup (Asterisk)
- ONLY unusual makeup goes here
- Prosthetics, wounds, aging, blood, etc.
- Standard makeup not noted (assumed for all performers)

### SFX (Blue)
- On-set practical effects
- Rain rigs, wind machines, pyrotechnics
- Mechanical effects, squibs, breakaway glass
- Fog machines

### VFX (Light Blue)
- Green screen elements
- Digital additions to be composited in post
- Wire removal shots
- Matte paintings / background replacement

### Vehicles (Pink)
- Picture vehicles (cars that appear ON camera)
- Note if vehicle moves or is parked static
- Classic/specialty vehicles need early sourcing
- Driver required? Stunt driver?

### Stunts (Orange)
- Any scene with physical risk
- Fight choreography
- Falls, crashes, fire
- Requires stunt coordinator + safety protocols

### Sound Notes (Brown)
- MOS shots (camera only, no sound) — saves time
- Wild lines (audio recorded separately)
- Atmospheric sound requirements
- Playback (music playing on set during scene)

## Complete Scene Breakdown Example

```
PRODUCTION: "Night Falls"     BREAKDOWN PAGE: 12
SCENE #: 24                  SCENE NAME: The Confrontation
INT/EXT: INT                 DAY/NIGHT: NIGHT
LOCATION: HAYES APARTMENT - KITCHEN
PAGE COUNT: 2 4/8            STORY DAY: Day 3

SYNOPSIS: Hayes confronts Maria about the missing evidence. 
Tension escalates. Maria smashes a coffee mug. Hayes pulls his badge.

CAST (Red):
- DETECTIVE HAYES (#1)
- MARIA (#2)

EXTRAS (Yellow):
- None

PROPS (Purple):
- Coffee mug x3 (smashed — need multiples)
- Detective's badge
- Evidence folder with photos
- Half-empty whiskey bottle
- Phone (Hayes's)

SET DRESSING (Gray):
- Kitchen appliances (fridge, stove)
- Wall calendar showing date
- Dirty dishes in sink (continuity from Sc.22)

WARDROBE (Circle):
- Hayes - Story Day 3 - Outfit 2 (wet shirt from rain)
- Maria - Story Day 3 - Outfit 1 (robe)

MAKEUP (Asterisk):
- Hayes: cut on cheek from Sc.20 (continuity wound)

SFX (Blue):
- None

VFX (Light Blue):
- None

SOUND (Brown):
- Rain ambience playback (storm heard outside)
- Mug smash — practical or foley

CAMERA / EQUIPMENT:
- Handheld second camera for coverage
- Tight quarters — 35mm preferred

SPECIAL INSTRUCTIONS:
- 3 coffee mugs required (breakaway for safety, real for close-up insert)
- Confirm continuity photos from Sc.22 for Hayes wound
```
