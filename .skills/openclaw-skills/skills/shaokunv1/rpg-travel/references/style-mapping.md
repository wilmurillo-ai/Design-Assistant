# 游戏风格映射

根据游戏类型和用户选择自动切换 HTML 冒险地图的视觉风格。

## 风格映射表

| 风格名称 | 配色方案 | 风格关键词 | 适用游戏 | 自动匹配规则 |
|---------|---------|-----------|---------|-------------|
| 旅行手账风 | 米白底+牛皮纸色+彩色贴纸 | 纸质纹理、胶带、拍立得、手写体 | 所有游戏（默认推荐） | 用户未指定时默认 |
| 复古羊皮纸 | 棕黄+墨黑+金+火漆红 | 做旧纸张、墨水、火漆印章、古典边框 | 巫师3、上古卷轴、刺客信条 | 西幻/中世纪游戏 |
| 现代极简卡片 | 白底+灰线+品牌色点缀 | 大留白、圆角卡片、细线分割、无衬线 | 所有游戏（干净清爽） | 都市/现代游戏 |
| 游戏内UI风 | 半透明深色+高亮标记色 | 半透明面板、小地图标记、任务侧栏 | 所有游戏（沉浸感最强） | 用户指定"沉浸感"时 |
| 像素复古 | 8-bit 彩色 | 复古、像素、8-bit | 马里奥、星露谷物语 | 像素/复古游戏 |
| 都市霓虹 | 霓虹红蓝紫 | 都市、霓虹、赛博 | 如龙、GTA、赛博朋克2077 | 都市/赛博游戏 |
| 和风武士 | 樱花粉+墨黑+金 | 日式、和风、禅意 | 对马岛之魂、只狼、塞尔达 | 和风/日本游戏 |
| 中国古风 | 朱红+墨黑+金 | 中国风、仙侠、水墨 | 黑神话：悟空、原神·璃月 | 中国风游戏 |
| 科幻未来 | 深蓝+青色+白 | 科幻、未来、极简 | 质量效应、星空 | 科幻游戏 |

**实现方式**：在 HTML 模板中通过 CSS 变量切换配色，格子图标和装饰元素也随风格变化。

## Style 参数映射

JSON 中的 `style` 字段使用以下值：
- `game-ui` — 游戏内UI风
- `travel-journal` — 旅行手账风
- `parchment` — 复古羊皮纸
- `minimal-card` — 现代极简卡片
- `pixel-retro` — 像素复古
- `neon-city` — 都市霓虹
- `japanese` — 和风武士
- `chinese` — 中国古风
- `scifi` — 科幻未来

## Game Type 参数映射

JSON 中的 `game_type` 字段使用以下值：
- `western` — 西幻（巫师3等）
- `japanese` — 和风（对马岛/只狼/塞尔达）
- `chinese` — 中国风（黑神话/原神·璃月）
- `cyberpunk` — 赛博朋克（赛博朋克2077等）
- `scifi` — 科幻（质量效应/星空）
- `pixel` — 像素（马里奥/星露谷）
- `modern` — 现代都市
- `default` — 默认
