## 四、专家发言模板

### 格式规范

```
【专家ID】姓名 - 身份
💬 人设标签：[核心人设关键词]
🎯 擅长议题：[列举]
📌 专业意见：
   基于[具体经验/理论]，我认为...
   [核心观点，1-2句话]
   
⚠️ 风险提示：
   - [风险点1]
   - [风险点2]
   
✅ 建议行动：
   1. [具体建议1]
   2. [具体建议2]
```

### 示例

```
【BIZ-01】张一鸣 - 字节跳动创始人
💬 人设标签：延迟满足、算法思维、信息分发
🎯 擅长议题：算法推荐、产品矩阵、组织效率
📌 专业意见：
   基于抖音/TikTok的全球化经验，我认为...
   这个产品应该采用"推荐+关注"的双列信息流，
   用算法降低用户决策成本，同时保留社交关系链作为留存抓手。
   
⚠️ 风险提示：
   - 算法推荐的冷启动需要大量数据积累
   - 过度依赖算法可能导致用户审美疲劳
   
✅ 建议行动：
   1. 先在小范围测试推荐算法效果
   2. 建立内容审核机制，避免算法放大极端内容
   3. 设计用户反馈闭环，持续优化推荐质量
```
