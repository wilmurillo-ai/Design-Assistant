# PRD Writer Skill - 产品经理 PRD 撰写助手

## 简介

产品经理 PRD 撰写助手是一款 AI Skill，能够根据用户输入的核心需求和北极星指标，自动生成结构化、可直接进入评审的 PRD 文档。

## 核心价值

- **最小输入，最大输出**：用户只需提供核心需求 + 北极星指标
- **自动完成 80%**：功能清单、用户流程、埋点、验收标准等自动生成
- **支持 AI 项目**：自动检测 AI/算法需求，生成增强型 PRD

## 使用场景

1. 从一句话需求快速生成完整 PRD
2. AI/算法项目的 PRD 自动撰写（含模型评估、兜底策略）
3. 已有 PRD 的优化与补全

## 安装方法

1. 将本 Skill 文件夹复制到 `~/.agents/skills/` 目录下
2. 确保文件结构完整
3. 重启 AI 助手或刷新 Skill 列表

## 工作原理

### 步骤 1：需求输入与北极星指标确认
接收用户的核心需求描述，确认或引导用户明确北极星指标

### 步骤 2：自动撰写各章节
- 需求背景自动撰写
- 功能清单自动生成（MECE 原则，P0/P1/P2 优先级）
- 用户流程自动梳理（主流程、分支流程、异常流程）
- 页面元素自动推导
- 异常场景自动覆盖（网络、服务端、输入、权限）
- 埋点方案自动设计（使用、转化、效果三维度）
- 验收标准自动生成

### 步骤 3：AI 项目增强（可选）
当检测到 AI/LLM/算法相关需求时，自动补充：
- AI 能力定义
- 模型评估指标
- Prompt 模板设计
- 异常兜底策略

### 步骤 4：PRD 输出
套用对应模板，输出完整 Markdown 格式的 PRD 文档
