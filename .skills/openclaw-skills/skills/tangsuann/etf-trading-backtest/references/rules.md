## 3. 手续费计算

### 费用明细
| 费用类型 | 费率 | 收取时机 |
|----------|------|----------|
| 买入手续费 | 0.025% | 买入时 |
| 卖出手续费 | 0.025% | 卖出时 |
| 印花税 | 0.1% | 卖出时 |

### 计算公式
```
买入费用 = 买入价 × 数量 × 0.025%
卖出费用 = 卖出价 × 数量 × 0.025%
印花税 = 卖出价 × 数量 × 0.1%

净收益 = 卖出价 × 数量 - 买入价 × 数量 - 买入费用 - 卖出费用 - 印花税
```

## 4. 风险控制

### 止损规则
- **触发条件**: 持仓亏损达到 -5%
- **处理方式**: 强制卖出
- **公式**: `(当前价 - 买入价) / 买入价 <= -0.05`

### 止盈规则
- **触发条件**: 持仓盈利达到 +8%
- **处理方式**: 可选择卖出或继续持有
- **公式**: `(当前价 - 买入价) / 买入价 >= 0.08`

### 熔断机制
- **触发条件**: 连续3笔亏损
- **处理方式**: 暂停交易1天，分析原因

## 7. 回测指标说明

### 基础指标
| 指标 | 说明 | 计算方式 |
|------|------|----------|
| 总收益率 | 累计收益/初始资金 | (期末净值-1)×100% |
| 年化收益率 | 年化后的收益 | (1+总收益率)^(252/交易日数)-1 |
| 胜率 | 盈利交易占比 | 盈利次数/总交易次数 |
| 盈亏比 | 平均盈利/平均亏损 | 总盈利金额/总亏损金额 |

### 风险指标
| 指标 | 说明 | 理想值 |
|------|------|--------|
| 最大回撤 | 最大跌幅 | <15% |
| 夏普比率 | 风险调整收益 | >1.0 |
| 卡玛比率 | 收益/回撤 | >1.5 |

### 计算公式

#### 最大回撤 (Max Drawdown)
```
MDD = max(Peak - Valley) / Peak × 100%
```

#### 夏普比率 (Sharpe Ratio)
```
SR = (Rp - Rf) / σp
其中: Rp=年化收益率, Rf=无风险利率(2%), σp=年化波动率
```

#### 卡玛比率 (Calmar Ratio)
```
CR = 年化收益率 / 最大回撤
```
