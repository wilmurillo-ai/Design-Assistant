# Rotifer Guide

> From zero to your first Gene in five minutes.

Interactive onboarding for [Rotifer Protocol](https://rotifer.dev) — the open-source evolution framework for AI agents.

## What It Does

One Skill, five capabilities — tell it what you need:

| Say this | It does this |
|----------|-------------|
| "How does Rotifer work?" | **Onboarding** — walks you through core concepts with hands-on CLI examples |
| "Create a grammar checker Gene" | **Scaffold** — extracts intent, generates the Gene skeleton |
| "My Gene score is 0" | **Doctor** — diagnoses why, fixes the root cause |
| "Find a web search Gene" | **Explorer** — searches the ecosystem, recommends the best match |
| "Upgrade my Wrapped Gene to Native" | **Upgrade** — evaluates migration path, guides the fidelity jump |

## Quick Start

After installing this Skill, just describe what you need in natural language:

```
I want to create a Gene that summarizes web pages
```

The Skill identifies your intent, picks the right sub-capability, and walks you through the full workflow.

## Core Concepts

| Concept | One-liner |
|---------|-----------|
| Gene | Self-contained logic unit: `express(input) → output` |
| Fidelity | Native > Hybrid > Wrapped — higher = more secure |
| Arena | Genes compete for ranking via F(g) fitness score |
| Domain | Two-level category like `content.grammar` |
| phenotype.json | Gene metadata (like package.json for Genes) |
