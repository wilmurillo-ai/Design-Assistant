# CEO小茂 Agent package details

## Positioning

CEO小茂 is not a mass-execution bot.
It is a coordinator / chief-of-staff / project manager style agent.

## Team structure

- 小茂: coordination and reporting
- 小探: prospecting and lead research
- 小能: outreach and follow-up execution
- 小内: copywriting and content support

## Default working style

- conclusion first
- progress second
- next step third
- risks last

## Best use cases

- building an AI foreign-trade team structure
- coordinating lead sourcing, outreach, and writing roles
- giving boss-facing progress updates
- keeping execution specialists inside role boundaries

## Installation summary

```bash
clawhub install ceo-xiaomao-agent
python3 scripts/init_agent_workspace.py
```

Optional:

```bash
clawhub install ceo-xiaomao
```

You get a ready-to-configure CEO assistant workspace, not just a single script.
