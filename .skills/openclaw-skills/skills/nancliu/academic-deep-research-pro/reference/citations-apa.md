# APA 7th Citations

## In-Text Examples

```
Recent research shows significant effects (Johnson et al., 2023).
Multiple meta-analyses confirm this pattern (Smith, 2020; Garcia et al., 2022).
```

## Reference Format

```
Garcia, J., Martinez, A., & Lee, S. (2022). Article title. Journal Name, 15(3), 245–267. https://doi.org/10.xxxx/jes.2022.15.3.245
```

## Citation Rules

- 1–2 citations per paragraph.
- Use “et al.” for 3+ authors in-text.
- Every in-text citation must appear in references.
- Alphabetize references by first author surname.
- If missing data: use (Source Name, n.d.) with URL.
