# 风格与风险护栏

适用场景：

- 续写时担心人物跑偏、视角漂移或 AI 味变重
- 对单章做扩写、润色、重写、净化
- 女频、言情、大女主或强感情线题材
- 网文平台连载、商业化题材或长期更新项目
- 需要处理借鉴边界、争议关系或高敏感价值观

## 1. 语言护栏

- 少空泛感叹、总结性感悟、抽象华丽词和 AI 高发词。
- 少写“他感到很”“她觉得”一类抽象心理，优先让情绪落在动作、反应、对话和后果里。
- 长短句交替，避免整段都像说明文或总结陈词。
- 高风险套语不做机械黑名单，但一旦成串复用，优先改写成更具体的动作、物件、环境、代价或因果。
- 对话避免剧本化堆叠；连续三句以上直接对话时，复查是否该补动作、环境或心理承接。

## 2. POV 护栏

- 当前 POV 只能直接写他此刻可见、可闻、可感、可想、可推断到的内容。
- 非当前 POV 的内心、真实动机和即时判断，默认不要直接写死。
- 视角稳定优先于“把信息说全”；不要在同一段里同时钻进两个人的脑内。
- 切 POV 必须有明确收益，且最好通过场景切换或明显段落转换完成。
- 第三人称称呼也要服从当前 POV 和人物关系，避免忽远忽近。

## 3. 感情线与女频敏感检查

- 关键配对的情感边界要尽早交代清楚，不要长期制造暧昧脏账。
- 宽恕必须建立在代价、补偿、认知变化和主动选择上，不能写成默认吃亏是美德。
- 大女主、御姐、独立型角色可以成长、柔软、恋爱，但不能为了推进感情突然失去主体性和核心判断。
- 生育工具化、极端年龄差或权力差、背叛洗白、雌竞和高争议价值观，都要处理失衡、后果和叙事立场。
- 如果标榜大女主，核心破局、成长弧和价值兑现必须主要落在女主自己的判断、行动和承担上。

### 3.1 大女主偏移预警

- 不要写成“设定上大女主，执行上爱男世界观”。如果打的是不媚男、大女主或女性成长标签，执行层就要兑现女性主体性。
- 男主可以支持、理解、共情，但不要长期占据女性议题的解释权、命名权和导师位。涉及身体羞耻、身份压迫、主体觉醒等议题，优先让女性角色自己发声、判断和行动。
- 不要把男主写成脱离时代、身份和职业逻辑的“完美进步样板”，尤其不要用超时代觉悟去偷女主成长线。若必须设置超前认知，必须补清来源、代价、局限和他为何偏偏懂这一题。
- 女主的高光、破局、价值判断和关键表达，不要频繁改写成“男主来教”“男主来认证”“男主来兜底”。
- 不要用女性角色集体降智、花痴、互踩、羞辱或雌竞，去抬男主魅力、开明、稀缺或正确。
- 女性配角要有自己的欲望、立场和关系线，不能只是用来证明男主值得爱、最懂女人、最会尊重女性。
- 如果出现“所有女性角色最后都围着男主转”的结构趋势，优先回查：是否偷走了女主议题表达，是否让女性配角失去自身任务，是否把女性冲突写成了男主魅力展示台。

### 3.2 平台连载风控预警

- 不要把影射现实、夹带私货、和平台审核斗心眼当成“高级表达”或“作者聪明”。平台连载首先是长期商业创作，不是拿账号和项目寿命去赌侥幸。
- 不要预设审核、编辑和读者“看不出来”。这种写法一旦出问题，受损的不是某一章，而是整本书的大纲、连载节奏和作者信用。
- 高风险表达如果与主线无关，直接删；如果只是为了情绪发泄、立场宣泄或阴阳怪气，也不要硬塞进正文。
- 真正稳的网文作者，把聪明用在题材兑现、冲突推进、角色记忆点和更新稳定性上，不用在擦边试探上证明自己。
- 如果某段内容的主要价值是“我想借角色说点别的”，优先回查：它是否真的服务人物、冲突和情节，还是在拖垮作品的聚焦度与平台安全性。

## 4. 原创性边界

- 可以借用开放式或公共框架设定，但不要直接照搬角色履历、关系模板、标志桥段和剧情骨架。
- 真正的原创性要落在规则边界、代价系统、冲突来源、人物欲望、推进链和结局变化上。
- 如果用户提供外部材料，先提炼稳定原则，再决定写进 skill 还是写进单项目文档。
- 只适用于单个项目的经验，不写进通用规则。

## 5. 写后净化检查

- 检查是否出现成串的感官套语、微表情套语、解释腔和抽象形容。
- 检查章节结尾是否在替作者总结主题；若有，改停在动作、发现、选择或危险上。
- 检查主角受挫后是否至少获得补偿基础、信息收益、关系回应或价值确认。
- 检查正文是否还能看见明显的 Beat 编号感；若有，补动作余波、情绪惯性和因果桥接。
