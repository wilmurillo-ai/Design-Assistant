# 世界观逻辑专项规则

适用场景：

- 从零搭建世界观
- 设定很多，但像清单，没有内在关联
- 冲突和人物动机长不出来自然逻辑
- 世界规则经常为剧情临时服务

## 1. 核心认知

世界观不是设定百科，而是一个动态运转系统。

不要只回答：

- 这个世界有什么

更要回答：

- 这个世界怎么运转
- 设定之间如何互相影响
- 什么能改，什么不能改

补充原则：

- 世界观首先要承载故事，而不是炫示宏大
- 不是越大越好，而是要和叙事需求匹配

## 1.1 现实映射原则

虚构世界往往不是凭空产生的，而是对现实规则网络的提炼、重组和放大。

现实中的世界同样由多重规则交织而成：

- 法律
- 道德
- 人情
- 经济规律
- 物理条件

这些规则共同决定：

- 谁更容易获得资源
- 谁更容易被推到边缘
- 人能以什么姿态活着

写虚构世界时，最好也能回答类似问题，而不是只给出地名、种族和设定名词。

## 1.2 不同写法中世界的功能

### 严肃文学取向

世界更像一张铁网。

重点往往在于：

- 规则有多严苛
- 人如何在规则前挣扎
- 命运如何压过个人愿望

### 网络文学取向

世界通常依旧冷硬，但会给主角留下打破规则、逆袭翻盘的缝隙。

重点往往在于：

- 规则有多压人
- 主角如何拿到破局筹码
- 如何给读者补偿感

两者都能成立，关键是要清楚自己的故事更偏哪种功能。

## 1.3 问题清单式设定法

世界观发虚时，不要靠灵感硬填名词，优先改用“连续提问”的方式。

不一定真的凑满多少问，关键是连续追问，直到设定能自己运转。

至少追到这些方向：

- 这个世界最值钱、最危险、最禁忌的是什么
- 谁因此受益，谁因此被压制
- 普通人每天怎么活，主角为什么活得更难或更特别
- 哪些规则绝对不能碰，碰了会怎样
- 资源怎么分，权力怎么分，代价怎么落
- 这些规则如何直接卡住主角的目标

如果问到一半就答不上来，通常说明这个世界还只是装饰板，不是叙事系统。

## 2. 两层结构模型

世界观建议拆成两层：

### A. 自然层

客观存在，即使没人类也成立。

包含：

- 地理环境
- 资源分布
- 生命形态
- 物理规律
- 因果与代价

### B. 文明层

由意识与社会组织创造。

包含：

- 势力格局
- 社会结构
- 制度
- 价值观
- 历史秩序
- 资源分配方式

## 2.1 三向抓手

如果用户写的是架空世界，或世界观虽然多但抓不住重点，可以先从三个方向同时下手：

### A. 自然环境

它不只是地图和气候，还决定正文的“滤镜”。

优先思考：

- 这个世界整体更偏压抑、冷硬、潮湿、荒凉，还是明亮、松弛、流动、丰饶
- 环境如何改变人物的日常动作、身体感受和风险判断
- 环境本身会不会参与制造冲突

自然环境写得稳，正文的气味、光线、温度、尺度感才会统一。

### B. 人文社会

它不只是背景资料，而是故事里的边界线和通行规则。

优先思考：

- 谁能越线，谁不能越线
- 什么行为会被奖赏、默许、羞辱、惩罚
- 这些规则如何直接卡住主角的目标、关系或生存

人文社会设定如果和主线无关，往往只是好看的摆设。

### C. 核心设定

核心设定不是“这个世界还有很多东西之一”，而是最能点火的那一个故事引擎。

先问：

- 这个世界里最值钱、最危险、最稀缺、最禁忌的是什么
- 谁会因此受益
- 谁会因此被猎捕、排斥、压榨或改造
- 它如何从设定直接长成冲突链

更稳的链条通常是：

`核心设定的价值/禁忌 -> 利益争夺或生存风险 -> 人物被卷入 -> 冲突升级`

## 2.2 运转逻辑的五层视角

除了自然层 / 文明层的二分，还可以用“五层运转逻辑”来判断故事真正由什么驱动。

这不是替代两层模型，而是更适合拿来做叙事前台选择。

### 第一层：人性

包括：

- 贪婪
- 恐惧
- 爱
- 求生
- 嫉妒
- 执念

特点：

- 是所有故事的默认底座
- 不需要发明，但需要被环境激活

### 第二层：硬性法则

包括：

- 物理规律
- 资源有限
- 生老病死
- 宇宙法则
- 世界承载上限

特点：

- 不可协商
- 只能理解、利用、绕开或付代价对抗

### 第三层：软性逻辑中的制度性

包括：

- 权力结构
- 经济分配
- 继承制度
- 宗门秩序
- 官僚体系

特点：

- 人造规则
- 理论上可改变，实际上很难撼动

### 第四层：软性逻辑中的文化性

包括：

- 礼教
- 忠义
- 风俗
- 信仰
- 道德期待

特点：

- 决定“应该怎么活”
- 变化更慢，更容易内化到人物身上

### 第五层：外部关系

包括：

- 文明与文明
- 阵营与阵营
- 人与自然
- 人类与异族

特点：

- 把内部矛盾放进更大参照系
- 常用于在中后期把格局往外推

## 2.3 不同题材的构建基石

### 现实题材

核心在认知。

重点不是发明规则，而是看清真正起作用的规则：

- 职场逻辑
- 圈层门槛
- 资源分配
- 潜规则

### 历史题材

核心在研究。

重点不是把现代情绪硬套进古代，而是理解那个时代的人：

- 为什么会那样活
- 被什么规训
- 又在什么地方偷偷反抗

### 虚构题材

核心在逻辑。

重点不是“炫”，而是：

- 规则能不能自洽
- 资源怎么循环
- 代价如何发生
- 角色为什么会被这套规则改变

## 2.2 前台逻辑的选择

多层逻辑可以同时存在，但最好有一层被推到前台。

前台逻辑会直接决定故事质感：

- 制度性前台：偏权力博弈、资源分配、阶层压迫
- 文化性前台：偏价值冲突、伦理撕扯、传统束缚
- 硬性法则前台：偏生存极限、文明困境、规则反抗
- 混合前台：张力更大，但控制难度也更高

判断问题：

- 这本书最核心的矛盾，主要由哪一层逻辑点燃
- 其他层级是背景支撑，还是也要被推到台前
- 如果抽掉这一层，故事是否还能成立

## 3. 四个维度

### 1. 自然形态

可见的世界硬件：

- 山川河流
- 大陆板块
- 星球结构
- 气候与天灾
- 资源点位
- 物种生态

作用：

- 给文明划边界
- 决定谁能在哪里生存
- 决定哪些资源会成为争夺中心

### 2. 自然规则

不可见的底层逻辑。

建议再分两层：

#### 底层规则

铁律，设定后不能硬破。

例如：

- 资源总量有限
- 能量守恒
- 因果律
- 代价法则

#### 表层规则

可以突破，但必须支付清晰代价。

例如：

- 寿命限制可被延缓
- 技术瓶颈可被突破
- 修为壁垒可被打破

判断原则：

- 不是不能变
- 是不能无代价地变

### 3. 文明形态

文明的可见产物：

- 国家
- 宗门
- 家族
- 城乡结构
- 法律制度
- 武器工具
- 货币系统
- 建筑风格

作用：

- 让抽象规则变成读者可见的生活现实

### 4. 文明规则

文明内部真正运转的逻辑：

- 权力如何合法化
- 资源如何分配
- 谁可以上升
- 谁天然被排除
- 价值观如何排序
- 社会禁忌如何形成

关键特点：

- 它们不是自然真理
- 它们可以被挑战、推翻、重建

## 4. 层级关系

默认关系：

1. 自然规则决定自然形态的边界
2. 自然形态影响文明形态的生长方向
3. 文明形态沉淀出文明规则

反向作用同样存在：

1. 文明规则塑造文明形态
2. 文明形态改造自然形态
3. 自然形态变化反过来挤压文明

因此世界观不是静止背景，而是循环系统：

自然塑造文明 -> 文明改造自然 -> 改变后的自然再塑造文明

## 5. 历史的作用

历史不是背景资料，而是当前秩序的来源。

优先回答：

- 这个制度是谁建立的
- 这个仇恨从哪一代开始
- 这个资源为什么归某个势力控制
- 这个禁忌背后曾发生过什么

如果这些问题答不出来，很多“现在如此”的设定会显得空。

## 5.1 三大核心设定的咬合

故事最核心的三块不是分开的：

- 世界观：舞台与规则
- 人物：欲望与行动载体
- 冲突：推动叙事前进的对抗与代价

它们应当互相咬合，而不是各写各的。

### 世界观 -> 人物

世界观决定人物处境。

例如：

- 资源稀缺，会逼出争夺压力
- 等级森严，会制造上升阻碍
- 阵营对立，会制造身份撕裂

人物的欲望、恐惧、能力边界，本质上都是世界规则的具体体现。

### 人物 -> 冲突

冲突不是凭空来的，而是人物带着欲望进入世界规则时被点燃的。

例如：

- 追求长生，撞上资源稀缺
- 追求自由，撞上等级压制
- 追求忠诚，撞上阵营冲突

### 冲突 -> 世界观

冲突是世界观的显影剂。

读者经常不是靠设定说明理解世界，而是靠冲突后果理解世界。

例如：

- 被强者欺压，让读者感到弱肉强食
- 因出身被拒，让读者感到等级森严

### 判断原则

无论从哪一端切入，最后都要回到三者咬合：

- 先有世界观：追问这个世界会逼出什么人、什么冲突
- 先有人物：追问这类人只能长在什么样的世界里
- 先有冲突：追问这种冲突需要什么规则和土壤支撑

反面信号：

- 世界观像装饰，删掉也不影响剧情
- 人物像悬浮标签，和世界没关系
- 冲突像临时拧出来的戏剧场面，没有结构根基

## 6. 空间尺度与故事匹配

常见误区：

- 以为世界越大越高级
- 用宇宙级设定去承载一个本来只需要城镇尺度的故事

更稳的原则：

- 故事需要多大空间，世界观就先开多大
- 当前空间还能持续长出新矛盾时，不急着升级世界

判断问题：

- 当前故事的核心矛盾发生在村镇、地区、国家还是文明尺度
- 当前空间是否已经足够承载人物关系、资源争夺和规则压力
- 扩大空间后，是否真的会生出新故事，而不是只多了名词

世界观的价值取决于承载力，不取决于物理尺寸。

## 7. 扩展与升级

### 横向扩展

同一层级内的广度拓展。

常见形式：

- 从一个地区走向多个地区
- 从一种地貌扩展到多种地貌
- 从一个文明中心扩展到多方文明板块

效果：

- 世界更丰富
- 读者感到“这个世界真大”

### 纵向升级

进入更高层级的规则和格局。

常见形式：

- 村镇 -> 城池 -> 王朝
- 凡界 -> 灵界 -> 仙界
- 地方势力博弈 -> 国家层级博弈 -> 文明层级博弈

效果：

- 读者感到“原来之前看到的只是冰山一角”

### 何时扩展，何时升级

优先扩展：

- 当前层级还没被写透
- 新地区能提供明显不同的生态、文明、利益结构

优先升级：

- 当前层级的冲突承载已接近上限
- 原有规则框架已无法支撑角色目标和故事野心

避免：

- 旧问题没写透就强行飞升
- 换了新地图，但故事模式完全没变

## 8. 开放与封闭

同一空间尺度内，还要决定它更偏开放还是封闭。

### 封闭结构

特点：

- 资源有限
- 外来变量少
- 决策后果更集中

适合：

- 生存压力
- 密室感
- 被困感
- 高压关系戏

### 开放结构

特点：

- 空间可延展
- 外来变量持续进入
- 未来可能性强

适合：

- 冒险
- 旅行
- 多文明接触
- 长篇延展

### 更成熟的做法

局部封闭，远处开放。

这样既能让当前阶段有压迫感，又能给未来升级和扩展留出口。

## 9. 铁律与故事空间

### 不可随意改动的部分

- 底层自然规则

这些是可信度支点。

### 可以推动剧情改变的部分

- 自然形态
- 文明形态
- 文明规则
- 表层自然规则

这些才是故事真正能施展变化的地方。

## 10. 从世界逻辑推导故事要素

世界观不是背景板，而是故事发生器。

优先思路：

- 先看资源和规则
- 再看谁受益、谁受损
- 再看谁会打破现状
- 最后顺着系统反应推出情节

### A. 冲突如何长出来

最常见的底层来源是：

- 资源稀缺
- 规则不公

推导顺序：

1. 哪些资源最稀缺
2. 谁控制这些资源
3. 谁被排除在分配体系外
4. 主角站在体系的哪个位置

这样推出来的冲突，通常不是私人吵架，而是结构性对立。

### B. 人物站位如何长出来

人物不要先写标签，再硬塞进世界。

更稳的做法是先问：

- 他是规则受益者、受害者、夹层者，还是规则设计者
- 他的动机与恐惧是否来自他所在的位置
- 他为什么会维护、依附、逃离或挑战现有秩序

常见站位：

- 规则受益者：有动力维护现状
- 规则受害者：有动力挑战现状
- 规则夹层者：既依赖秩序，又被秩序压迫
- 规则设计者：最害怕秩序崩塌

### C. 情节如何自然推进

情节卡住，常见原因不是没想到桥段，而是没有写出“打破规则后的系统反应”。

推荐模型：

1. 主角打破某个表层规则
2. 世界系统开始反应
3. 资源关系、权力关系、阵营关系发生变化
4. 新的结构性矛盾出现
5. 主角被迫做下一轮选择

这样情节更像“世界在反弹”，而不是作者在硬推。

### D. 升级体系如何稳定

体系容易崩，通常因为两件事：

- 升级没有不可逆代价
- 力量没有清晰边界

稳定锚点：

- 代价机制：寿命、记忆、人性、关系、资源消耗、道德代价
- 天花板设定：绝对上限、位阶边界、世界承载极限

判断标准：

- 每次升级是否都只是时间换数值
- 后期是否还能靠规则维持张力

### E. 题材侧重点

| 题材 | 优先设计的维度 | 重点 |
|------|----------------|------|
| 修仙/玄幻 | 自然规则 + 文明规则 | 资源结构、修炼逻辑、主角站位 |
| 科幻 | 自然规则 + 自然形态 | 技术边界、代价、陌生环境 |
| 权谋 | 文明规则 + 文明形态 | 权力结构、利益分配、站队后果 |
| 都市/现实 | 文明规则 | 行业规则、圈层壁垒、现实代价 |

### F. 抓主干，放细节

世界观不必一开始就写到面面俱到。

更有效的做法是：

- 先抓和主线冲突直接相关的规则主干
- 其他细节在创作过程中按需生长

判断问题：

- 抽掉这个设定，主线会不会断
- 如果不会断，它大概率还不是当前最需要写细的部分

### G. 不做全知全能作者

作者可以知道世界更大，但正文不必把一切都摊开。

更好的状态是：

- 人物只看到自己脚下的路
- 读者通过人物逐步感到世界的厚度
- 未写出的部分仍然有回声

### H. 验证顺序

无论最初灵感来自人物、脑洞、冲突还是一张画面，动笔前都尽量按这个顺序回查：

1. 底层逻辑：这个世界究竟怎么运转
2. 天然矛盾：这套运转逻辑自然会逼出什么冲突
3. 核心人物：谁最会在这套矛盾中被挤压、受益或撕裂
4. 冲突升级路径：当人物行动后，世界会如何反弹并升级

如果第一步没想清，后面常会出现：

- 情节越写越人工
- 人物越写越悬浮
- 冲突越来越像作者硬拧出来的

## 11. 设计模板

```markdown
# 世界观

## 前台逻辑层级
```

## 12. 写前检查

每次动笔前至少确认：

1. 本章动用的是哪一层世界规则
2. 角色行为是否符合其文明环境
3. 如果角色打破规则，代价是否已经铺垫
4. 本章事件是否会反过来改变局部世界状态
5. 当前故事是否真的需要扩展或升级世界边界
6. 本章推进是否来自世界系统反应，而不只是作者硬推
7. 当前前台逻辑层级是否依然清晰
8. 现在最该写细的，是规则主干还是枝叶细节

## 13. 写后检查

- [ ] 世界观不是清单，而是能解释事件因果
- [ ] 自然层与文明层逻辑一致
- [ ] 底层规则没有被剧情硬破坏
- [ ] 表层规则被打破时写清了代价
- [ ] 历史不是摆设，而是影响当前秩序
- [ ] 冲突和人物动机能追溯到世界结构
- [ ] 世界尺度和故事需求匹配
- [ ] 如果扩展或升级了世界，新的空间真的带来了新矛盾或新规则
- [ ] 人物站位与其所处规则结构匹配
- [ ] 主角扰动规则后，世界出现了连锁反应
- [ ] 升级体系存在代价和边界，不是单纯数值膨胀
- [ ] 当前前台逻辑层级明确，其他层级没有喧宾夺主
- [ ] 故事能按“逻辑 -> 矛盾 -> 人物 -> 升级路径”闭环解释
- [ ] 当前题材所需的那类规则已经足够真实或足够自洽
- [ ] 细节没有抢走主干的位置

## 14. 高风险失败模式

- 地图很大，但和剧情无关
- 设定很多，但互相不影响
- 角色突破规则没有成本
- 制度存在，却不影响人物命运
- 历史只在旁白里存在，不进入当下剧情
- 世界一味做大，但剧情仍在重复旧套路
- 新世界只是换皮，没有新的利益结构和规则压力
- 主角做出大动作，世界却像没被碰过一样
- 人物只是标签，站位和动机不来自结构
- 现实题材没有现实规则，只有空泛情绪
- 历史题材没有时代逻辑，只有现代人换装
