# 日常叙事专项规则

适用场景：

- 休整章
- 关系升温或关系转冷
- 世界烟火气展示
- 需要自然埋伏笔或传递信息
- 过渡章节容易写成流水账

## 1. 核心认知

日常不是注水，也不是“没事可写时的填充物”。

它真正的价值在于：

- 让人物活起来
- 让关系有根基
- 让世界有温度
- 让节奏有呼吸
- 让伏笔有地方藏
- 让信息自然传递

可以把它理解成：

- 冲突是骨头
- 日常是血肉

## 2. 日常的六个功能

### A. 人物功能

冲突更容易写出“人物能做什么”。
日常更容易写出“人物到底是什么样的人”。

优先通过小动作、小偏好、小抉择暴露性格。

### B. 世界功能

通过吃、住、聊、买、卖、修炼、赶路、做工等日常场景，让世界长出烟火气。

关键不是堆细节，而是让读者知道：

- 这个世界的人怎么活
- 不同阶层、阵营、地域的人活法有什么差异

### C. 节奏功能

日常是“松”，冲突是“紧”。

没有松，故事会一直绷到疲劳。
没有紧，故事会散成闲逛。

### D. 关系功能

关系往往不是在大事里立住，而是在小事里扎根。

例如：

- 一起吃饭
- 一起赶路
- 一起发呆
- 小事上的试探、照顾、误会、推让

### E. 伏笔功能

日常是最自然的伏笔掩体。

高压剧情里埋伏笔容易显刻意，日常里更容易藏住。

### F. 信息功能

酒馆闲聊、交易行情、旅途见闻、修炼琐事，都可以成为传信息的通道。

## 3. 三条核心原则

### 原则 1：日常必须有任务

写前先问：

- 这段日常到底要服务什么

常见任务：

- 展示人物
- 推进关系
- 传递信息
- 调节节奏
- 埋设伏笔
- 展现世界

### 原则 2：日常也要有钩子

不是所有日常都要大冲突，但至少要让读者觉得“这段有点东西”。

钩子可以是：

- 性格细节
- 关系微妙变化
- 新鲜世界细节
- 隐性伏笔
- 小摩擦、小误会、小试探

### 原则 3：功能完成就收

日常最怕写顺手了停不下来。

任务完成，及时收住，不要恋战。

## 4. 常见类型

### 1. 休整型日常

冲突后让人物和读者一起喘口气。

### 2. 关系型日常

用小事给关系打地基。

### 3. 世界型日常

把设定转成生活感，而不是导览词条。

### 4. 铺垫型日常

看起来像闲笔，回头看才发现是伏笔。

### 5. 反差型日常

大冲突前后的平静，会让前后两端的情绪都更重。

## 5. 高风险失败模式

- 写成流水账，事无巨细但没有重点
- 日常太长，冲淡主线
- 和主线完全脱节，各写各的
- 人物在日常里失焦，只剩空壳
- 只有平静，没有任何变化、钩子或收获

## 6. 写前检查

1. 这段日常的任务是什么
2. 这段日常有没有至少一个钩子
3. 这段日常会让人物、关系、世界、伏笔或信息中的哪一项更具体
4. 功能完成后准备在哪里收住

## 7. 写后检查

- [ ] 这段日常不是流水账
- [ ] 人物在日常中仍然有鲜明性格
- [ ] 关系在小事中推进，而不是口头宣布
- [ ] 世界通过生活细节变得可感
- [ ] 日常段落里有轻微变化、钩子或收获
- [ ] 日常和主线存在连接
