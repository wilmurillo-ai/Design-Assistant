# 质量检查清单

交付章节前使用此清单确保质量。

## 整体检查

### 基础要素

- [ ] **章节有明确标题**
  - 标题与内容相关
  - 吸引人但不过度透露
  - 纯正文章节可以把标题放在文件名或大纲里，不要求正文首行重复标题

- [ ] **字数符合预期**
  - 本 skill 默认章节：3000-5000 字
  - 如用户明确要求短章，以用户要求为准

- [ ] **章节完整性**
  - 有开头、发展、高潮
  - 不是片段，是完整叙事单元

- [ ] **时间地点清晰**
  - 读者知道何时何地
  - 转换时有明确标记

---

## 开头检查

- [ ] **前 3 段内抓住读者**
  - 有行动/冲突/悬念
  - 不是天气或日常流程

- [ ] **开篇人物密度可控**
  - 没有一上来就堆太多新人物和新名字
  - 关键人物数量控制在读者能快速记住的范围内

- [ ] **认知钩子与情感钩子都已启动**
  - 读者会问“这是什么情况”
  - 读者也开始关心“这个人会怎样”

- [ ] **若采用文学性开篇，双钩子成立**
  - 有认知钩子，不只是漂亮句子
  - 有情感钩子，不只是结构炫技

- [ ] **与上一章有连接**
  - 回应上一章结尾
  - 或明确时间/地点跳跃

- [ ] **背景信息不过量**
  - 没有大段信息倾倒
  - 信息自然融入动作

- [ ] **五百字内完成基本契约**
  - 欲望与阻力已可感知
  - 故事类型和调性开始明确

- [ ] **若采用非线性结构，工具服务故事**
  - 不是为了显得高级而打乱顺序
  - 当前信息缺口对读者有实际吸引力

---

## 内容检查

### 情节推进

- [ ] **本章有核心事件**
  - 发生了不可删除的事
  - 不是"什么都没发生"的过渡章

- [ ] **推动主线剧情**
  - 揭示新信息
  - 或改变人物关系
  - 或升级冲突

- [ ] **故事发生了改变**
  - 当前内容没有回到原点
  - 不是一连串事件过去后什么都没变

- [ ] **逻辑自洽**
  - 事件因果关系合理
  - 没有巧合驱动剧情
  - 人物行为符合动机

- [ ] **三大设定互相咬合**
  - 世界观能解释人物处境和冲突来源
  - 人物欲望能自然点燃冲突
  - 能说清“世界逻辑 -> 矛盾 -> 人物 -> 升级路径”的闭环

- [ ] **借熟设定时，原创性落在剧情和人物上**
  - 即便使用公共框架，人物欲望、关系张力、推进链和代价也不是现成影子
  - 没有直接照搬他人角色履历、关系模板或标志桥段

### 冲突与张力

- [ ] **有明确冲突**
  - 人与人、人与环境、人与自己
  - 冲突推动本章事件

- [ ] **冲突有根源**
  - 能追溯到资源稀缺、规则压力或历史积怨
  - 不是为了推进剧情临时硬造

- [ ] **张力有变化**
  - 不是平铺直叙
  - 有紧张和缓解的交替

- [ ] **有转折或新信息**
  - 不是线性可预测
  - 有意外或新发现

- [ ] **冲突有代价**
  - 冲突之后有人失去、受伤、暴露或被迫选择
  - 不是吵完就像没发生过

- [ ] **三层冲突互相咬合**
  - 微观事件能映照中观格局或宏观压力
  - 不是只有私人恩怨，没有更大结构支撑

- [ ] **冲突链已经形成**
  - 当前困境来自之前的选择或代价
  - 不是一段一段热闹但互不相干

- [ ] **冲突生成链清楚**
  - 能说清现象、欲望、矛盾、冲突如何一步步长出来
  - 不是只有结果，没有生成过程

- [ ] **存在不可逆起爆点**
  - 前期至少有一次关键行动或决定让故事无法轻易回到原点
  - 主冲突不是靠零碎事件慢慢堆出来的

- [ ] **冲突会升级而不是原地打转**
  - 至少能看出从个人/局部走向更大格局的趋势
  - 不是换一批事件重复同一种摩擦

- [ ] **冲突不是消失，而是换形**
  - 某阶段的解法本身会制造下一阶段的问题
  - 旧冲突会以新形式返回，而不是被简单清零

- [ ] **存在假性缓和期或压制期**
  - 如果中段出现和平、修复或繁荣，底层仍有倒计时
  - 缓和不是泄气，而是为了放大后续反弹

- [ ] **冲突层级匹配体量**
  - 主线级、阶段级、场景级冲突分工清楚
  - 不是只有场景热闹，没有中长线支撑

- [ ] **冲突网存在**
  - 配角和阵营之间也有被引爆的命运线
  - 不是只有主角一个人在推动故事

- [ ] **赌注带有真实死亡感**
  - 这里的“死亡”可以是生理、社会、关系或心理层面的毁灭
  - 读者清楚知道主角到底输不起什么

- [ ] **主角与麻烦被结构绑住**
  - 冲突里至少存在生存威胁、制度绑定或时间锁中的一种
  - 不是主角理论上随时都能转身走人

### 情绪补偿

- [ ] **本章有明确情绪回报**
  - 至少有一种前台回报：反制、兑现、回应、揭示、收获、确认
  - 不是只有压力和消耗

- [ ] **主角受挫不是纯受气**
  - 吃亏后带来了反制基础、信息收益、关系变化或资源落袋
  - 不是长期悬空的委屈

- [ ] **深度没有裸奔**
  - 价值判断通过选择、代价、后果和人物关系体现
  - 不是角色直接替作者讲道理

- [ ] **爽点不靠机械堆砌**
  - 爽点来自具体兑现，而不是关键词重复
  - 不只依赖单一种类的刺激

- [ ] **期待感没有被提前清空**
  - 阶段回报给到了，但更大的问题、危险或代价仍在前台
  - 不是为了圆满把追读动力一起收掉

- [ ] **逆袭回路成立**
  - 如果写的是爽文/升级文/逆袭文，读者能看清低位处境、压制来源、翻盘筹码和地位改写
  - 不是只说主角很惨、很强、很狂，但中间缺少结构性兑现

- [ ] **复仇或打脸有实际结果**
  - 轻视、羞辱或伤害主角的人，后续付出了具体代价
  - 不是只有主角放狠话，没有局势变化

- [ ] **爽感不只停在嘴上**
  - 回报除了面子，还落在身份、资格、资源、站队或规则变化上
  - 不是吵赢一句就当赢了全场

- [ ] **配角不是排队送脸**
  - 旁观者、对手、盟友各有立场，不是清一色只会嘲讽或震惊
  - 爽感来自反差和兑现，不是全世界一起降智

---

## 人物检查

- [ ] **人物行为一致**
  - 符合已建立的性格
  - 如不一致，有解释

- [ ] **人物动机清楚**
  - 关键行为背后有内在理由
  - 不是作者为了推进剧情硬推

- [ ] **人物有反应**
  - 对事件有情绪/行动
  - 不是被动道具

- [ ] **人物有声音**
  - 对话能区分角色
  - 每人说话方式不同

- [ ] **人物展示而非讲述**
  - 通过行动/对话表现性格
  - 不是直接陈述"他很勇敢"

- [ ] **人物有局限**
  - 存在能力、认知、性格或环境层面的短板
  - 不是无成本全能角色

- [ ] **成长弧光可信**
  - 改变来自困境和过程积累
  - 关键转变有转折点支撑，而不是突然顿悟

- [ ] **高智角色不是空标签**
  - 如果角色被写成聪明、会算、善谋或高智，正文里至少有可见的观察、筛选、判断或布局兑现
  - 不是只靠头衔、旁白和他人吹捧维持说服力

- [ ] **高智结论可回溯**
  - 角色的重要推断能追溯到前文已给出的线索、异常或信息差
  - 不是作者事后补一句“他早就知道”

- [ ] **高智不靠全员降智衬托**
  - 其他角色可以看漏、迟疑或走常规解法，但不能集体失智只为抬一人
  - 衬托服务于证明能力，不是把世界写傻

- [ ] **高智也有边界**
  - 角色的聪明有领域、条件、代价或视野限制
  - 不是任何场景都能无成本开上帝视角

### 日常检查

- [ ] 日常段落不是流水账
- [ ] 日常段落有明确任务（人物/关系/世界/伏笔/信息/节奏）
- [ ] 日常里有轻微变化、钩子或收获
- [ ] 日常与主线存在连接，不是各写各的

---

## 视角检查

- [ ] **本章主 POV 明确**
  - 读者知道当前主要跟着谁看、谁想、谁误判
  - 不是写着写着突然换了代入对象

- [ ] **没有乱切脑内**
  - 同一段或同一场景里没有直接跳进多个角色内心
  - 不是刚写完 A 的心思，下一句又直接宣布 B 的真实想法

- [ ] **信息暴露符合 POV 边界**
  - 当前 POV 不该知道的信息，没有被作者提前说漏
  - 其他角色的真实动机、判断和结论没有越权暴露

- [ ] **未知被保留下来**
  - 对他人的看法、敌意、轻蔑、算计等优先通过动作和细节暗示
  - 没有因为解释过早而削弱悬念和代入

---

## 对话检查

- [ ] **每句对话有目的**
  - 推动情节/揭示人物/制造冲突
  - 没有"你好""吃了吗"等无意义对话

- [ ] **对话简洁自然**
  - 删除冗余词语
  - 符合真实说话方式

- [ ] **有潜台词**
  - 不是所有话都直说
  - 有言外之意

- [ ] **标签使用正确**
  - 能辨识时省略标签
  - 不过度使用副词

- [ ] **直接对话密度受控**
  - 高压场景可以密，但长段对白前后有动作、环境、心理或转述支撑
  - 连续三句以上直接对话时，检查是否每句都必要

- [ ] **没有剧本化倾向**
  - 不是只有台词在来回抛信息
  - 对话前后有铺垫、承接或收束

---

## 悬念检查

- [ ] **结尾有钩子**
  - 使用至少一种悬念技巧
  - 让读者想看下一章

- [ ] **主问题明确**
  - 读者能说出当前“还没解决的是什么”
  - 不只是发生了一件事
  - 这个问题已经被读者清楚感知到

- [ ] **悬念强度适当**
  - 与故事位置匹配
  - 高潮章节悬念更强

- [ ] **不是虚假悬念**
  - 不是机械误会
  - 不是无意义的"突然"

- [ ] **读者在乎悬念对象**
  - 悬念绑定人物命运、关系或明确代价
  - 不是无关紧要的信息缺口

- [ ] **赌注与不确定性成立**
  - 读者知道输赢后果是什么
  - 结果不是一眼可判

- [ ] **为下一章铺垫**
  - 设置下一章的冲突
  - 埋下伏笔

- [ ] **局部解答会引出新问题**
  - 回答旧问题时没有掐断悬念链
  - 解答后仍有追读动力

- [ ] **悬念类型搭配合理**
  - 长线没有只依赖单一悬念类型
  - 同一事件的追问方向是清晰的

- [ ] **追读循环没有断裂**
  - 没有过早把答案一次性说透
  - 旧问题没有长时间失联到被读者忘掉

- [ ] **章节结尾仍有未决问题**
  - 读者还有继续看下一章的理由
  - 没有把主线、支线、章节问题同时清零
  - 没有出现“问题真空”

- [ ] **中间章节也有钩子意识**
  - 钩子不只靠最后一句，中段场景也在持续堆期待感
  - 回答旧问题时没有把追读动力一起回答没了

- [ ] **存在持续性悬念而不只是一瞬意外**
  - 读者有时比角色更早知道风险或炸点
  - 不是只靠突然爆一下，然后立刻归零

---

## 展示而非讲述检查

### 常见"讲述"标记

检查并修正以下模式：

| 讲述（避免） | 展示（使用） |
|-------------|-------------|
| 他很愤怒 | 他握紧拳头，指节发白 |
| 她很美丽 | 他凝视着她，忘记说话 |
| 他很紧张 | 他反复调整领带 |
| 房间很乱 | 衣服扔在沙发上，外卖盒堆在桌上 |
| 他很富有 | 他从口袋里掏出一叠现金 |

### 自查问题

- [ ] 是否直接陈述情绪？（改为身体反应）
- [ ] 是否用形容词总结？（改为具体描写）
- [ ] 是否跳过了关键场景？（补充展示）
- [ ] 场景里的人物是否真的和环境发生了交互，而不是悬浮对白
- [ ] 关键细节是否至少有部分做到“一笔多用”，而不是纯装饰画面

---

## 节奏检查

- [ ] **句子长度有变化**
  - 没有连续 3 句长度相同
  - 长短交错

- [ ] **段落长度适当**
  - 避免大段文字墙
  - 动作场景用短段落

- [ ] **信息密度有变化**
  - 高密度（动作/对话）
  - 低密度（描写/内心）

- [ ] **文笔层次有分工**
  - 旁白、画面、思想、情绪没有全挤成一种说明腔
  - 情绪主要落在动作、环境、细节和节奏里，而不是一直直说

- [ ] **内部推进没有提纲切块感**
  - 读者不会明显感觉到“又切到下一个 Beat 了”
  - 不是一段只做一件事然后立刻硬跳下一段

- [ ] **转场和转折有桥接**
  - 相邻推进点之间有动作余波、情绪惯性、环境反馈、视线转移或因果触发
  - 重大转折前后至少有半拍承接，而不是生硬折断

---

## 语言检查

- [ ] **没有 AI 写作痕迹**
  - 避免"此外""然而""强调"等 AI 词汇
  - 避免四字成语堆砌
  - 句式多样化

- [ ] **语气词不过量**
  - “呢”“啊”“啦”等主要服务角色口吻，不是整段默认口气
  - 高频语气词没有把叙述写虚、写飘、写腻

- [ ] **高风险 AI 套语没有成串复用**
  - 如“瞳孔骤缩”“倒吸一口凉气”“不可名状”这类表达，只可偶发，重复就改
  - 优先改写成具体动作、环境、物件、代价或后果

- [ ] **解释腔和说明文化连接词受控**
  - 少靠“也就是说”“换言之”等词硬推逻辑
  - 因果尽量落在场面和结果里，不靠作者补讲

- [ ] **结尾没有总结升华**
  - 没有情感总结、主题点题或替作者发言
  - 收尾优先停在动作、发现、选择或危险上

- [ ] **"的"字不密集**
  - 没有连续多个"的"
  - 简化修饰结构

- [ ] **用词精确**
  - 避免模糊词（"一些""某种"）
  - 使用具体词汇

---

## 连贯性检查

- [ ] **与前文连贯**
  - 上一章的悬念有回应
  - 已知信息一致

- [ ] **伏笔有呼应**
  - 早期埋下的线索有进展
  - 或即将揭示

- [ ] **时间线一致**
  - 时间流逝合理
  - 事件顺序正确

---

## 类型特定检查

### 悬疑类

- [ ] 有线索揭示
- [ ] 有新谜题提出
- [ ] 逻辑无漏洞

### 言情类

- [ ] 关系有进展
- [ ] 有情感张力
- [ ] 读者在意配对
- [ ] 关键配对的情感边界清晰，没有长期暧昧脏账消耗代入
- [ ] 主角没有被迫靠吃亏、忍让或无条件原谅来维系关系
- [ ] 角色没有为了爱情突然降智、失去主体性或偏离核心人设
- [ ] 高争议价值观有明确后果和叙事立场，没有偷渡成“正确答案”
- [ ] 生育、生子或系统任务没有把女主工具化、数量化或去人格化
- [ ] 极端年龄差或强权力差关系有清楚的边界与后果，不是简单甜化失衡关系
- [ ] 如果打的是“大女主”标签，核心破局与成长主要由女主完成，不是“女强男更强”长期代打
- [ ] 女性身体、身份或羞耻议题主要由女性角色自己发声和推进，没有长期改写成男主教育、男主定义或男主认证
- [ ] 男主没有被写成脱离时代与身份逻辑的全能进步导师；如果他明显超前，文本给了来源、代价、局限或代偿解释
- [ ] 女主的重要高光、判断与破局没有频繁被改写成“男主来教、男主来救、男主来收尾”
- [ ] 没有用女性角色集体花痴、降智、互踩或雌竞来抬男主魅力与正确性
- [ ] 女性配角拥有独立欲望、任务和关系线，不只是为男主服务或为配对背书
- [ ] 如果作品标榜不媚男或大女主，世界观执行没有滑向“全员为男主服务”

### 群像类

- [ ] 主题已拆成多个切面
- [ ] 主题已超出单个角色的承载范围，确实需要群像化处理
- [ ] 本章主 POV 明确
- [ ] 切线有明确收益，不是机械切换
- [ ] POV 切换后信息边界仍然稳定，没有一边切一边乱钻所有人脑内
