# 世界观展现专项规则

适用场景：

- 世界观设定很多，但正文读起来像资料
- 读者说“设定懂了，但没感觉”
- 开篇背景交代太重，沉浸感差
- 想让世界观和故事推进自然融合

## 1. 核心认知

世界观设计和世界观展现不是一回事。

- 设计：作者知道这个世界如何运转
- 展现：读者如何在阅读中自然感知这个世界

目标不是让读者背设定，而是让读者进入世界。

## 2. 开篇设定的使用原则

以下题材更需要在开篇给读者基本认知框架：

- 历史题材
- 复杂奇幻
- 神话、玄幻、科幻
- 势力和规则门槛较高的故事

但原则不是“先讲清一切”，而是：

- 只交代理解当下情节所必须的最低信息
- 开篇提到的设定，后续要真正用到
- 如果后续暂时用不到，不要为了完整感硬塞

## 3. 正文中的徐徐展开

### A. 角色视角展开

让角色和读者一起认识世界。

适合：

- 主角初入新环境
- 升级流、冒险流、成长流

优点：

- 代入感强
- 设定和剧情天然绑定

限制：

- 只能自然展现角色接触范围内的信息

### B. 上帝视角展开

由叙述者直接给出全局信息。

适合：

- 宏观格局
- 多方势力并行
- 历史总览

风险：

- 信息容易压过情绪
- 读者变成旁观者

使用原则：

- 只在角色视角无法承载时使用
- 尽量短、准、服务当前情节

## 4. 四种自然展现方式

### 1. 对话展现

用信息差自然传递设定。

适合：

- 老手对新人
- 本地人对外来者
- 上位者对下位者

避免：

- 两个都知道的人硬互相讲常识
- 为了交代设定故意演讲

### 2. 冲突展现

最有效。

让读者通过后果感到规则存在。

例如：

- 主角因出身被拒，读者自然理解阶层规则
- 主角被强者压制，读者自然感到弱肉强食

关键：

- 设定要落到人物命运上
- 让规则有重量，而不是停留在说明层

### 3. 载体展现

通过“物”承载世界信息。

常见载体：

- 典籍
- 书信
- 地图
- 壁画
- 遗迹
- 传说
- 回忆

适合：

- 历史
- 规则来历
- 消亡文明
- 失落知识

### 4. 感受展现

通过日常与身体感受让世界“活”起来。

环境首先决定正文的前台滤镜。

可以先问：

- 这个地方压人还是养人
- 人走进来先感到冷、湿、刺、闷、亮，还是松、缓、软、空
- 这种环境会让人物本能地更警惕、更克制，还是更松弛、更外放

例如：

- 吃饭方式
- 行礼规矩
- 穿着材质
- 方言与称呼
- 城市气味
- 宗门禁忌

原则：

- 只点关键处
- 不求面面俱到

如果环境气质总是抓不稳，可以先找 3-5 张参考图，只借：

- 光线
- 色温
- 材质
- 空间压迫感

不要照抄具体景点或画面构图，只把它们当成作者侧的氛围校准工具。

## 5. 设定展现的三条铁律

### 铁律 1：设定要兑现

开篇提到的关键元素，后续要真正进入情节。

否则会让读者觉得：

- 设定很多
- 但没有用

### 铁律 2：设定清单是作者工具

设定清单主要给作者自己理顺逻辑。

正文里应优先转译成：

- 冲突
- 关系
- 风俗
- 场景体验
- 选择代价

### 铁律 3：保留未知

读者不需要一次知道世界全部。

适当保留：

- 未展开地区
- 未解释禁忌
- 未揭开的旧历史

会让世界更像真实世界，而不是说明书。

## 6. 写前检查

每章涉及新设定时先问：

1. 这段设定有没有必要现在出现
2. 这段设定最自然的载体是什么
3. 能不能通过角色经历而不是旁白说明
4. 这个设定后续会不会真正兑现

## 7. 写后检查

- [ ] 新设定不是整段灌输
- [ ] 设定通过视角、对话、冲突或载体自然出现
- [ ] 规则对角色造成了实际影响
- [ ] 开篇提到的设定元素后续有兑现可能
- [ ] 读者能感到世界存在，而不是只读到资料
- [ ] 保留了适量未知，没有一次讲完
