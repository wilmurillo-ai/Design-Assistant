# Presentation

使用 Slidev 创建的演示文稿

## 预览

```bash
npx slidew dev
```

## 导出

```bash
# 导出为 PDF
npx slidew export

# 导出为 PPTX
npx slidew export --format pptx

# 导出为 HTML
npx slidew export --format html
```

## 主题

- 名称：科技紫

修改 `slides.md` 文件来更新内容
