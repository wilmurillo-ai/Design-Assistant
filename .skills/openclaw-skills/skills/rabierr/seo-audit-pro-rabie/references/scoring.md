# Scoring Rubric

## Overall Score (out of 100)

| Score | Grade | Meaning |
|-------|-------|---------|
| 90–100 | A | Excellent — minor tweaks only |
| 75–89 | B | Good — a few issues to address |
| 60–74 | C | Fair — several important fixes needed |
| 45–59 | D | Poor — significant SEO problems |
| 0–44 | F | Critical — major overhaul required |

## Category Breakdown

### Technical SEO (40 points)
| Check | Points |
|-------|--------|
| HTTPS enabled | 15 |
| TTFB < 800ms | 10 (5 for 800ms–2s) |
| robots.txt exists | 5 |
| sitemap.xml exists | 5 |
| Viewport meta tag | 5 |

### On-Page SEO (40 points)
| Check | Points |
|-------|--------|
| Title tag present & correct length (30-70 chars) | 10 |
| Meta description present & correct length (70-160) | 8 |
| Single H1 tag | 8 |
| Canonical tag | 4 |
| Robots meta (not blocking) | 5 |
| OG tags | 5 |

### Content Quality (20 points)
| Check | Points |
|-------|--------|
| Word count > 300 | 5 |
| All images have alt text | 5 |
| Schema.org structured data | 3 |
| Internal/external link balance | 4 |
| Content freshness signals | 3 |

## Keyword Score (separate from main score)
When a keyword is provided, rate keyword optimization:
- **Strong (3/3):** In title + H1 + meta description
- **Good (2/3):** In two of the three above
- **Weak (1/3):** Only in one location
- **Missing (0/3):** Not found in key locations
