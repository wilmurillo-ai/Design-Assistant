### Domain Verification

**How to verify official domain**:
1. Check Twitter bio for official link
2. Check project docs (docs.projectname.xyz)
3. Cross-reference with Etherscan
4. Check Discord announcement channel

**Common Phishing Patterns**:
```
Real: scroll.io
Fake: scroll-airdrop.io
Fake: scrolls.io
Fake: scroll-network.io
```

## Quick Reference: Search Workflow

### Daily Search Routine (10 minutes)

```
1. Twitter (5 min)
   - Search: "airdrop" after:today
   - Check KOL accounts
   - Check protocol accounts

2. Galxe/Zealy (2 min)
   - Check trending campaigns
   - Filter by "active"

3. DefiLlama Airdrops (2 min)
   - Check new listings
   - Note deadlines

4. RootData (1 min)
   - Check recent funding rounds
   - New projects with funding
```

### Weekly Deep Dive (30 minutes)

```
1. VC Portfolio Updates
   - Check new investments
   - Note projects in stealth

2. Ecosystem Maps
   - New projects in top L2s
   - Upcoming launches

3. Token Unlocks
   - Check for TGE dates
   - Note airdrop windows
```

## Red Flags in Sources

### Unreliable Sources
- Telegram DMs from strangers
- Twitter replies with links
- Unverified Discord announcements
- Emails claiming "you won airdrop"

### Verification Checklist
```
□ Source is official account (check verification badge)
□ Link matches Twitter bio
□ Contract address matches Etherscan
□ Announcement exists on official Discord
□ Multiple sources confirm (not just one tweet)
```
