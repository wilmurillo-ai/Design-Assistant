---

## Usage Notes

1. Replace all `[PLACEHOLDERS]` with actual content
2. Remove empty sections (e.g., no scam alerts = omit that section)
3. Maximum 5 projects per report to avoid overwhelm
4. Always include at least one quick tip
5. End with standard disclaimer
