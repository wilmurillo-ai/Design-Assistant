## 二、平台详细分析

### 2.1 COZE/扣子 🔍

**平台地址：** https://www.coze.cn/store/market/bot

**自动化能力：**
- ✅ 有 REST API 支持（`api.coze.cn`）
- ✅ 可通过 API 创建/更新 Bot
- ⚠️ 发布到商店需要人工审核
- ⚠️ API 文档为中文，需申请开发者权限

**API 功能：**
- 创建 Bot
- 上传 Bot 配置文件
- 查询 Bot 状态
- ❌ 无法绕过人工审核提交

**自动化方案：**
1. 使用 COZE API 自动创建 Bot（开发者测试用）
2. 生成标准提交文本，包含 Bot 配置信息
3. 人工审核后正式发布

**所需 Token：**
- `COZE_TOKEN` - 从 https://www.coze.cn/settings/api 获取
- 需要个人 Access Token（不是 API Token）

**难度：** ⭐⭐⭐ 中等

---

### 2.2 腾讯元器 🔍

**平台地址：** https://yuanqi.tencent.com/market

**自动化能力：**
- ⚠️ 有基础 API 支持
- ⚠️ 智能体发布需要腾讯账号 + 人工审核
- ⚠️ 主要面向微信小程序生态

**API 功能：**
- 创建技能/智能体
- 配置技能参数
- ❌ 发布到市场需要人工操作

**自动化方案：**
1. 通过 API 创建技能（测试用）
2. 生成标准提交文本，包含技能描述、图标、介绍
3. 提供腾讯元宝市场的提交页面链接
4. 人工提交后等待审核

**所需 Token：**
- `YUANQI_TOKEN` - 腾讯云 API Key

**难度：** ⭐⭐⭐⭐ 较高

---

### 2.3 阿里百炼 🔍

**平台地址：** https://bailian.console.aliyun.com/model-market

**自动化能力：**
- ⚠️ 有阿里云 ModelScope API 支持
- ⚠️ 智能体/Dataset 发布需要阿里云账号
- ⚠️ 主要面向 AI 模型和服务市场

**API 功能：**
- 调用模型 API
- 管理数据集
- ❌ 智能体发布需要人工操作控制台

**自动化方案：**
1. 生成标准提交文本
2. 提供阿里云百炼市场的提交页面链接
3. 人工提交后等待审核

**所需 Token：**
- `BAILIAN_TOKEN` - 阿里云 API Key

**难度：** ⭐⭐⭐⭐ 较高

---

### 2.4 SkillzWave 🔍

**平台地址：** https://skillzwave.ai/submit/

**自动化能力：**
- ⚠️ 需要 GitHub OAuth 登录
- ⚠️ 提交表单需要人工填写
- ✅ 审核相对快速

**提交信息：**
- GitHub 仓库 URL
- Skill 描述
- 截图/预览
- 分类标签
- 开发者联系信息

**自动化方案：**
1. 生成完整的提交文本（包含所有字段）
2. 生成 GitHub OAuth 授权链接
3. 提供预填充的提交表单 URL
4. 人工确认提交

**所需 Token：**
- GitHub OAuth（通过浏览器交互获取）

**难度：** ⭐⭐ 较低

## 三、自动化架构设计

### 3.1 Token 安全存储

```
本地文件系统 (绝对不上传)
    │
    ├── skill-publisher/
    │   ├── .env                    ← Token 存储位置
    │   │   ├── GITHUB_TOKEN=xxx
    │   │   ├── CLAWHUB_TOKEN=xxx
    │   │   └── ...
    │   ├── .gitignore             ← 确保 .env 忽略
    │   └── data/
    │       └── platforms.json      ← 平台元数据
    │
    └── 本地 .env 由用户维护
```

**安全措施：**
1. `.env` 已在 `.gitignore` 中
2. Token 绝不写入 Git 提交
3. 用户通过交互式向导输入 Token
4. Token 仅在运行时加载到内存

### 3.2 分层发布策略

```
┌─────────────────────────────────────┐
│  交互式发布向导 (setup_tokens.py)   │
│  - Token 配置                       │
│  - 平台选择                         │
│  - 状态查看                         │
└────────────────┬────────────────────┘
                 │
                 ▼
┌─────────────────────────────────────┐
│  自动发布层（可全自动）              │
│  ✅ ClawHub - npx clawhub publish   │
│  ✅ GitHub   - git push             │
│  ✅ SkillsMP - GitHub 触发同步      │
└────────────────┬────────────────────┘
                 │
                 ▼
┌─────────────────────────────────────┐
│  半自动层（需用户确认）              │
│  ⚠️ COZE    - API 创建 + 手动提交   │
│  ⚠️ 元器    - 手动提交              │
│  ⚠️ 百炼    - 手动提交              │
└────────────────┬────────────────────┘
                 │
                 ▼
┌─────────────────────────────────────┐
│  生成提交文本层                      │
│  📝 为每个平台生成标准化提交说明     │
│  📝 提供一键复制                     │
│  📝 提供直接跳转链接                 │
```

## 四、平台 API 自动化可能性总结

| 平台 | API 自动化 | 绕过审核 | 推荐程度 | 说明 |
|------|-----------|---------|---------|------|
| ClawHub | ✅ 完全 | N/A | ⭐⭐⭐⭐⭐ | 官方 CLI 支持 |
| GitHub | ✅ 完全 | N/A | ⭐⭐⭐⭐⭐ | REST/Git API |
| SkillsMP | ✅ 完全 | N/A | ⭐⭐⭐⭐ | GitHub 自动同步 |
| SkillzWave | ⚠️ 登录 | ❌ | ⭐⭐⭐ | GitHub OAuth + 手动 |
| COZE | ⚠️ 创建 | ❌ | ⭐⭐⭐ | API 有限，审核必备 |
| 元器 | ⚠️ 创建 | ❌ | ⭐⭐ | 主要靠手动 |
| 百炼 | ⚠️ 创建 | ❌ | ⭐⭐ | 主要靠手动 |

**结论：**
- **可全自动（3个）：** ClawHub、GitHub、SkillsMP → 直接集成
- **需生成提交文本（4个）：** SkillzWave、COZE、元器、百炼 → 生成标准文本 + 链接

## 五、Token 获取指南

### GitHub
1. https://github.com/settings/tokens
2. Generate new token (classic)
3. 勾选 `repo` 权限
4. 复制 Token

### ClawHub
1. https://clawhub.ai
2. 登录 → 个人设置
3. 复制 API Token

### COZE
1. https://www.coze.cn/settings/api
2. 需申请开发者权限
3. 创建个人 Access Token

### 腾讯元器
1. https://yuanqi.tencent.com
2. 登录腾讯账号
3. 个人中心 → API 密钥

### 阿里百炼
1. https://bailian.console.aliyun.com
2. 登录阿里云账号
3. API Key 管理
