# 保险代理人客户档案管理工具

> 一站式解决客户资料整理、保单信息提取、客户跟进记录管理

## 💡 解决什么问题

你是不是经常遇到这些烦恼？

| 烦恼                          | 用了这个工具后           |
| --------------------------- | ----------------- |
| 客户资料散落在 Excel、PDF、微信里，找起来麻烦 | ✅ 一个文件夹，自动整理成统一档案 |
| 客户问"我有哪些保单"，翻半天找不到          | ✅ 3秒钟语音/文字查询，立刻回答 |
| 保单快到期了，客户都忘了                | ✅ 自动提醒，永不遗漏       |

## 🎯 核心功能

1. **📂 一键整理客户资料**
   - 从 Excel 表格批量提取客户信息
   - 从 PDF 电子保单提取条款内容
   - 从 PPT 方案书提取产品对比
   - 从 Word 情况说明提取客户背景
   - 从图片（身份证、证件照）提取身份信息
2. **🔍 秒级查询客户档案**
   - "张总有哪些保单？" → 立刻列出所有保单
   - "王女士的家庭成员有哪些？" → 家庭关系图谱
   - "最近3个月需要续费的客户有哪些？" → 续费提醒清单
3. **📋 统一管理客户档案**
   - 多份保单自动合并，去重处理
   - 冲突信息智能判断，保留最准确的
   - 完整的数据来源追溯
4. **💬 智能客户跟进助手**
   - 记录每次沟通内容
   - 自动提取关键需求和承诺
   - 到期提醒：续费、生日、周年
5. **📊 保障缺口分析**
   - 自动计算客户现有保障额度
   - 根据年龄、收入、家庭责任推荐保障缺口
   - 生成产品推荐方案

## 👥 适用人群

- **保险代理人**（个险渠道）- 整理客户档案，提升展业效率
- **保险代理人**（银保渠道）- 管理高净值客户，一键查询
- **保险经纪人** - 跨公司产品对比，客户统一管理
- **保险团队主管** - 团队客户管理，业绩分析
- **保险内勤/组训** - 报表整理，数据分析

## 🔧 安装与配置

### 前置要求

- Python 3.8+
- API Key（MiniMax 或 OpenAI）

### 安装依赖

```bash
pip install pandas openai python-docx python-pptx pdfplumber
```

### 快速开始

```python
from src.config import SkillConfig
from src.tools import extract_single_document, query_customer_profile

# 初始化配置（只需配置一次）
config = SkillConfig(
    api_key="your-api-key",  # 首次使用时输入
    base_url="https://api.minimax.chat/v1",  # MiniMax API
    model="MiniMax-M2.7"
)

# 整理客户保单文件
result = extract_single_document("客户保单.xlsx", config)

# 自然语言查询客户信息
result = query_customer_profile(
    profiles_dir="./客户档案",
    query="这个客户的联系方式和所有保单信息？",
    config=config
)
```

## 🔒 安全说明

- ⚠️ **数据流向说明**：本工具会将提取的文本内容发送到你配置的 LLM API 端点（如 MiniMax/OpenAI）进行处理。**请确保你使用的 LLM 服务符合你的数据安全要求**。
- ⚠️ **敏感信息**：在发送给 LLM 前，建议对客户敏感信息（如完整身份证号、银行卡号等）进行脱敏处理。
- ✅ API Key 仅用于调用你配置的 LLM 服务，不会发送给其他第三方
- ✅ 本地文件读写操作仅在本地执行
- ⚠️ **合规责任**：使用前请确认符合《个人信息保护法》等法规要求，并取得客户同意
- ✅ 无需担心信息泄露
