# Source Quality Guide

## Tier 1 — Preferred sources (use first)

| Source | Type | Why |
|---|---|---|
| Established press (Guardian, FT, Telegraph, Times, BBC) | News/analysis | Factual, edited, named journalists |
| Academic (Google Scholar, researchgate, university presses) | Research/evidence | Peer-reviewed, cited |
| Government (.gov.uk), GDS, NHS Digital | Policy/research | Authoritative, primary source |
| Think tanks (Demos, IPPR, Nesta, RSA) | Analysis | Well-sourced, named authors |
| Industry reports (McKinsey, Gartner, Nielsen) | Data | Cited, methodology clear |

## Tier 2 — Acceptable with caveats

| Source | Caveat |
|---|---|
| Blog posts / Medium | Always cross-check claims |
| Company blogs | Check for conflicts of interest |
| Reddit / forums | Use only for sentiment, never facts |
| YouTube | Only for demonstrations, not claims |
| Newsletter archives | Verify recency |

## Tier 3 — Do not cite as primary source

- Aggregate job boards (Indeed, Glassdoor, LinkedIn) — use only as discovery, cite the primary job listing
- Aggregator event listings — always link to the primary event page, not the aggregator
- Press releases — use as leads only, always verify independently
- Unsigned web content without methodology

## Source checklist before citing

- [ ] Author named (not anonymous)
- [ ] Publication date visible
- [ ] Claims backed by evidence or named source
- [ ] Not an advertorial
- [ ] Primary source, not derivative
- [ ] Domain matches a known credible publication or institution

## For listings/events

Always find the primary event page. Do not cite:
- Time Out aggregator (link to the actual venue/event page)
- Skiddle as the primary link for an event that has its own venue page
- Facebook event pages (unreliable, paywalled or deleted without notice)

Valid listing link: direct to venue website or official booking page.
