# 题目模板与出题规范

> 本文件规定各题型的标准格式、出题原则和样例，供技能生成题目时参照。

---

## 一、题型规范

### 1. 选择题

**格式模板：**
```
【第X题】（选择题）
[题目正文，确保有且仅有一个正确选项]

A. [选项A]
B. [选项B]
C. [选项C]
D. [选项D]
```

**出题原则：**
- 题干简洁，无歧义
- 4个选项，只有1个正确答案
- 干扰项要有"迷惑性"，能覆盖典型错误
- 各选项长度尽量接近

**样例（初中·一元一次方程）：**
```
若方程 2x - 3 = 7 的解为 x = a，则 a 的值是（  ）

A. 2    B. 5    C. -2    D. -5
```
答案：B（2x = 10，x = 5）

---

### 2. 填空题

**格式模板：**
```
【第X题】（填空题）
[题目正文，用 ____ 标记填空位置]

答：____
```

**出题原则：**
- 每个填空位置答案唯一确定
- 避免有多种合理解读的开放性填空
- 填空位置通常在句末，除非特殊需要

**样例（初中·一次函数）：**
```
一次函数 y = 2x - 1 的图像过点 (1, ____)，
当 x = 3 时，y = ____。
```
答案：1；5

---

### 3. 解答题

**格式模板：**
```
【第X题】（解答题，共X分）
[题目正文]

（要求：写出完整解题步骤）
```

**出题原则：**
- 明确说明解答要求（写步骤/只写结果等）
- 分值标注，体现步骤得分
- 题目条件充分，无冗余也无缺漏

**样例（初中·二元一次方程组）：**
```
解方程组：
⎧ 2x + y = 7
⎨
⎩ x - y = 2

（共6分：设方程2分，解方程过程3分，验证1分）
```

---

### 4. 应用题

**格式模板：**
```
【第X题】（应用题，共X分）
[情境描述，不超过3-4句话]
[明确的问题，以问号结尾]

（要求：设未知数，列方程（组），求解，并写明答语）
```

**出题原则：**
- 情境贴近学生生活（购物、出行、面积计算等）
- 数据合理，有实际意义
- 答案通常为正整数或简单小数/分数
- 必须要求写答语

**样例（初中·一元一次方程应用）：**
```
小明家距学校4.5千米，他先步行一段路后，改乘公交车去学校。
已知步行速度为4千米/时，公交车速度为36千米/时，
共用了20分钟到达学校。
问：小明步行了多少千米？

（共8分）
```

## 二、难度分级标准

### 基础题（★）
- 直接套用公式/定义
- 单一知识点，步骤≤3步
- 数字计算简单（整数为主）
- 目标：检验概念理解是否到位

### 中等题（★★）
- 需要1-2步变形或转化
- 涉及1-2个知识点的综合
- 可能包含分数、根号等计算
- 目标：检验知识的灵活运用

### 拔高题（★★★）
- 需要多步推理或多种方法结合
- 跨知识点综合，需要构造辅助工具
- 计算量较大，需要合理化简
- 目标：培养解题策略和创新思维

## 三、出题检查清单

生成题目后，自动检查以下项目：

- [ ] 知识点是否在该年级教材范围内（不超纲）
- [ ] 题目条件是否完整（无漏条件、无矛盾条件）
- [ ] 答案是否唯一确定
- [ ] 选择题的干扰项是否具有迷惑性
- [ ] 应用题的数据是否符合实际
- [ ] 题目表述是否清晰无歧义
- [ ] LaTeX 公式是否渲染正确

## 四、整套试卷结构参考

### 小学（1-6年级）期末练习卷结构
```
选择题：5题 × 2分 = 10分
判断题：5题 × 2分 = 10分
填空题：10题 × 3分 = 30分
计算题：5题 × 4分 = 20分
解决问题：5题 × 6分 = 30分
总分：100分
```

### 初中（7-9年级）单元测验结构
```
选择题：8题 × 3分 = 24分
填空题：6题 × 4分 = 24分
解答题：6题 × 8-10分 = 52分
总分：100分
```

### 高中（10-12年级）章节测验结构
```
选择题：10题 × 5分 = 50分（单选）
填空题：4题 × 5分 = 20分
解答题：6题 × 8-14分 = 150分
总分：150分
```
