# 批改评分标准

> 本文件规定主观题批改的评分规则和反馈模板，确保批改结果专业、公正、有教育意义。

---

## 一、评分原则

### 1. 过程性评分
数学解答题按步骤给分，即使最终结果错误，只要过程正确，也可得部分分。

**步骤分配原则：**
- 设未知数/审题：10-15% 分值
- 建立方程/关系式：20-25% 分值
- 解方程/运算过程：40-50% 分值
- 验证/答语：15-20% 分值

### 2. 错误分类
| 错误类型 | 扣分比例 | 说明 |
|----------|----------|------|
| 计算错误 | 扣该步骤50%分值 | 思路正确，算术有误 |
| 概念错误 | 扣该步骤全部分值 | 使用了错误的公式/定理 |
| 方向错误 | 扣后续所有步骤分值 | 解题方向根本错误 |
| 书写不规范 | 扣1-2分 | 步骤不清晰，符号混用等 |
| 漏写答语 | 扣1-2分 | 仅限应用题 |

### 3. 容错原则
- 允许等价的不同写法（如 x = 5 和 x₁ = 5 等同）
- 允许不同但正确的解题方法
- 最终答案与参考答案等价即视为正确

## 二、各题型批改模板

### 选择题批改
```
【第X题】[题目] 
你的答案：[X]
正确答案：[Y]
结果：✅ 正确 / ❌ 错误

[若错误] 解析：
[简短解释为什么答案是Y，以及你选X的可能错误原因]
```

### 填空题批改
```
【第X题】[题目]
你的答案：____
正确答案：____
结果：✅ 正确 / ❌ 错误

[若错误] 解析：[简短说明]
```

### 解答题批改（详细步骤模板）
```
【第X题】批改（共N分）

📌 解题过程分析：

步骤① [描述你的做法]
  → 评分：__/__ 分
  → 说明：[✅正确 / ⚠️部分正确：... / ❌错误：...]

步骤② [描述你的做法]  
  → 评分：__/__ 分
  → 说明：...

步骤③ ...

【本题得分】：__ / N 分

【参考解答】：
[完整的标准解答步骤]

【错误原因分析】：
[指出根本问题所在，避免类似错误]

【提升建议】：
[1-2条具体改进建议]
```

## 三、常见错误类型库

### 代数类
| 错误描述 | 标准说明 |
|----------|----------|
| 移项忘记变号 | 移项时必须改变符号，如 2x=3+x → x=3 不是 3x=3 |
| 分母未通分直接运算 | 含分母的方程须先通分 |
| 去括号符号错误 | -(a+b) = -a-b，不是 -a+b |
| 平方根忘记正负 | √4 = ±2，一元二次方程要写两个根 |
| 韦达定理用错 | x₁+x₂=-b/a，注意负号 |

### 几何类
| 错误描述 | 标准说明 |
|----------|----------|
| 混淆全等与相似 | 全等面积相等，相似面积之比等于相似比的平方 |
| 直角三角形斜边判断 | 勾股定理中c是斜边（最大边），非随意 |
| 圆周角与圆心角混淆 | 圆周角 = ½ 圆心角 |
| 角平分线与中线混淆 | 角平分线等分角，中线连接顶点与对边中点 |

### 计算类
| 错误描述 | 标准说明 |
|----------|----------|
| 运算顺序错误 | 先乘除后加减，括号优先 |
| 负数乘除符号 | 负×负=正，负÷负=正 |
| 分数运算错误 | 乘法：分子×分子/分母×分母；加减：先通分 |
| 根号运算错误 | √(a²) = |a|，不是直接等于a |

## 四、鼓励性反馈模板

### 全对时
```
🎉 太棒了！这道题你完全答对了！解题过程清晰、步骤规范，非常出色。
继续保持这种细心的解题习惯！
```

### 部分正确时
```
👍 思路是对的！你已经掌握了这道题的核心方法，只是在 [具体步骤] 上有一点小失误。
这种错误很常见，注意 [具体注意点] 就能拿满分了。
```

### 完全错误时
```
没关系！这道题确实有一定难度。让我帮你一步步分析思路……
[详细解析]
建议你把这道题重新做一遍，看看是否掌握了。
```

## 五、批改后引导策略

| 得分情况 | 引导策略 |
|----------|----------|
| 100% | 推荐挑战同类型拔高题 |
| 70-99% | 针对错误步骤给出专项练习 |
| 40-69% | 建议重新回顾知识点讲解 |
| <40% | 建议从基础题重新练习，必要时重学知识点 |
