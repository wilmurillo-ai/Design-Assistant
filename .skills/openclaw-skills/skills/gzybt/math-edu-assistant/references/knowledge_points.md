## 初中核心公式速查

### 代数

#### 整式运算
```
(a+b)² = a² + 2ab + b²
(a-b)² = a² - 2ab + b²
(a+b)(a-b) = a² - b²
(a+b)³ = a³ + 3a²b + 3ab² + b³
```

#### 一元二次方程
```
标准形式：ax² + bx + c = 0 (a≠0)

求根公式：x = (-b ± √(b²-4ac)) / 2a

判别式 Δ = b² - 4ac：
  Δ > 0 → 两个不相等实数根
  Δ = 0 → 两个相等实数根
  Δ < 0 → 无实数根

韦达定理：
  x₁ + x₂ = -b/a
  x₁ · x₂ = c/a
```

#### 二次函数
```
顶点式：y = a(x-h)² + k，顶点为 (h, k)
标准式：y = ax² + bx + c

顶点坐标：h = -b/2a，k = c - b²/4a
对称轴：x = -b/2a

a > 0 → 开口向上，最小值 k
a < 0 → 开口向下，最大值 k
```

#### 不等式
```
基本性质：
  a > b → a+c > b+c（不等式两边加同一数，不等号方向不变）
  a > b, c > 0 → ac > bc（乘正数，方向不变）
  a > b, c < 0 → ac < bc（乘负数，方向改变）

均值不等式：a + b ≥ 2√(ab)（a,b > 0，当且仅当 a=b 时取等）
```

### 几何

#### 三角形
```
内角和：∠A + ∠B + ∠C = 180°
外角：等于不相邻两内角之和

全等条件：SSS、SAS、ASA、AAS（注意：AAA和SSA不能判定全等）
相似条件：AA、SAS、SSS

面积公式：S = ½ × 底 × 高
```

#### 直角三角形（勾股定理）
```
勾股定理：a² + b² = c²（c为斜边）
逆定理：若 a² + b² = c²，则该三角形是直角三角形

常见勾股数：3-4-5，5-12-13，8-15-17，7-24-25
```

#### 锐角三角函数
```
在直角三角形中（∠C = 90°）：
  sinA = 对边/斜边 = a/c
  cosA = 邻边/斜边 = b/c
  tanA = 对边/邻边 = a/b

特殊角：
  sin30° = ½,   cos30° = √3/2, tan30° = √3/3
  sin45° = √2/2, cos45° = √2/2, tan45° = 1
  sin60° = √3/2, cos60° = ½,   tan60° = √3
```

#### 圆
```
圆的基本量：
  周长 C = 2πr
  面积 S = πr²

弧长公式：l = nπr/180（n为圆心角度数）
扇形面积：S = nπr²/360 = ½lr

圆周角定理：
  同弧所对圆周角相等
  圆周角 = ½ × 圆心角
  直径所对圆周角 = 90°

切线性质：切线垂直于切点半径
```

#### 四边形
```
平行四边形：
  对边相等且平行，对角相等，对角线互相平分
  面积 S = 底 × 高

矩形：平行四边形 + 四角均为直角 + 对角线相等
菱形：平行四边形 + 四边相等 + 对角线互相垂直平分
正方形：矩形 + 菱形

梯形面积：S = ½(上底+下底) × 高
```

### 统计与概率

```
平均数：x̄ = (x₁+x₂+...+xₙ)/n
中位数：数据从小到大排列，处于中间位置的数（偶数个取中间两数平均值）
众数：出现次数最多的数

方差：s² = [(x₁-x̄)² + (x₂-x̄)² + ... + (xₙ-x̄)²] / n
标准差：s = √s²

古典概型：P(A) = A包含的基本事件数 / 基本事件总数
```

## 高中核心公式速查

### 函数与基本初等函数

```
指数运算：
  aᵐ · aⁿ = aᵐ⁺ⁿ
  (aᵐ)ⁿ = aᵐⁿ
  aᵐ/aⁿ = aᵐ⁻ⁿ
  a⁰ = 1（a≠0）
  a⁻ⁿ = 1/aⁿ

对数运算：
  log_a(MN) = log_a M + log_a N
  log_a(M/N) = log_a M - log_a N
  log_a(Mⁿ) = n·log_a M
  换底公式：log_a b = log_c b / log_c a

自然对数：ln = log_e，e ≈ 2.71828
常用对数：lg = log₁₀
```

### 三角函数

```
同角三角函数关系：
  sin²α + cos²α = 1
  tanα = sinα/cosα
  1 + tan²α = sec²α（sec = 1/cos）

诱导公式（奇变偶不变，符号看象限）：
  sin(-α) = -sinα
  cos(-α) = cosα
  sin(π-α) = sinα
  cos(π-α) = -cosα
  sin(π/2-α) = cosα
  cos(π/2-α) = sinα

和差化积：
  sin(α±β) = sinαcosβ ± cosαsinβ
  cos(α±β) = cosαcosβ ∓ sinαsinβ
  tan(α±β) = (tanα±tanβ)/(1∓tanαtanβ)

二倍角公式：
  sin2α = 2sinαcosα
  cos2α = cos²α - sin²α = 1-2sin²α = 2cos²α-1
  tan2α = 2tanα/(1-tan²α)
```

### 数列

```
等差数列：
  通项：aₙ = a₁ + (n-1)d
  前n项和：Sₙ = na₁ + n(n-1)d/2 = n(a₁+aₙ)/2

等比数列（q≠1）：
  通项：aₙ = a₁·qⁿ⁻¹
  前n项和：Sₙ = a₁(1-qⁿ)/(1-q)
  当 q=1 时：Sₙ = na₁
```

### 平面解析几何

```
两点距离：|P₁P₂| = √((x₂-x₁)²+(y₂-y₁)²)
中点坐标：M = ((x₁+x₂)/2, (y₁+y₂)/2)

直线斜率：k = (y₂-y₁)/(x₂-x₁)
直线方程各种形式：
  斜截式：y = kx + b
  点斜式：y - y₀ = k(x - x₀)
  两点式：(y-y₁)/(y₂-y₁) = (x-x₁)/(x₂-x₁)
  截距式：x/a + y/b = 1

点到直线距离：d = |Ax₀+By₀+C| / √(A²+B²)

圆的方程：
  标准式：(x-a)²+(y-b)² = r²，圆心(a,b)，半径r
  一般式：x²+y²+Dx+Ey+F=0
```

## 常见解题方法汇总

### 方程类
1. **移项法**：把含未知数的项移到一边
2. **代入消元法**：用一个方程表达一个未知数，代入另一个方程
3. **加减消元法**：两个方程相加减，消去一个未知数
4. **因式分解法**：将方程变形为两个因子乘积为零

### 函数类
1. **图像法**：先画出函数图像，从图像上读取信息
2. **配方法**：将二次函数化为顶点式，找最值
3. **换元法**：用新变量替换复杂表达式，简化问题

### 几何类
1. **辅助线法**：过关键点作平行线、垂线或连线
2. **坐标法**：建立坐标系，将几何问题转化为代数计算
3. **面积法**：用面积关系推导线段比

### 应用题类
1. **设未知数**：设清楚未知量，注明单位
2. **找等量关系**：题目中的等量条件是列方程的依据
3. **验证答案**：代回原题验证是否符合实际意义
