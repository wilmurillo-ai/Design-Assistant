## 技能详细描述（Full Description）
> 显示在技能详情页，支持 Markdown

```markdown
## 数学教材同步教育助手

专为 K12 学生设计的智能数学学习伙伴，覆盖小学（1–6年级）、初中（7–9年级）、
高中（10–12年级）全阶段，严格对标人教版 / 北师大版教材内容，绝不超纲。

### ✨ 核心功能

#### 📚 教材同步讲解
按年级、章节精准讲解知识点，配合例题和易错提醒，帮助学生真正理解，而不是死记硬背。

#### 📝 智能出题
- 支持选择题、填空题、解答题、应用题四种题型
- 按年级 / 章节 / 难度（基础 / 中等 / 拔高）灵活生成
- 可一键生成单元测验或期末模拟卷

#### ✅ 自动批改
- 客观题秒批，给出正确答案和解析
- 主观题按解题步骤逐步评分，思路对了就有分
- 给出错误原因分析和针对性改进建议

#### 💡 智能答疑
- 先给启发性提示，引导学生自己思考
- 分步拆解复杂题目，每步单独讲清楚
- 解答后提炼「这类题的通用方法」，举一反三

### 🗣️ 使用方式
无需记忆任何指令，用自然语言说话即可：

> "帮我讲一下二元一次方程组"
> "出 5 道八年级二次函数的填空题，难度中等"
> "这道题我不会，帮我讲思路"
> "帮我批改这道题，我的答案是 x=5"
> "总结一下三角函数的公式"

### 📖 覆盖内容
| 学段 | 年级 | 主要内容 |
|------|------|---------|
| 小学 | 1–6 年级 | 四则运算、分数、小数、几何图形、面积体积 |
| 初中 | 7–9 年级 | 代数、方程组、函数、几何证明、统计概率 |
| 高中 | 10–12 年级 | 集合、三角函数、数列、解析几何、概率统计 |

### ⚠️ 使用须知
- 本技能仅处理数学学科内容
- 严格按教材范围输出，不引入超纲内容
- 所有题目均为原创生成，非版权试卷内容
```
