# Lean Six Sigma Tools Reference

## DMAIC Methodology

### Define Phase
- Project Charter
- SIPOC Diagram
- CTQ Tree
- VOC/VOB Collection

### Measure Phase
- Data Collection Plan
- MSA (GR&R, Bias, Linearity, Stability)
- Baseline Capability (Ppk/Cpk)
- Process Flow Map

### Analyze Phase
- Control Charts (Xbar-R, Xbar-S, I-MR)
- 8 Out-of-Control Rules
- Hypothesis Testing
- Root Cause Analysis (5 Why, Fishbone)

### Improve Phase
- DOE (Full Factorial, Fractional, RSM, Taguchi)
- Poka-Yoke
- Solution Selection Matrix
- Pilot Testing

### Control Phase
- Control Plans
- OCAP (Out-of-Control Action Plan)
- SPC Implementation
- Lessons Learned

## Lean Tools
- 5S (Sort, Set in order, Shine, Standardize, Sustain)
- VSM (Value Stream Mapping)
- Kanban
- SMED (Single-Minute Exchange of Dies)
- TPM (Total Productive Maintenance)
- FMEA (Failure Mode and Effects Analysis)
