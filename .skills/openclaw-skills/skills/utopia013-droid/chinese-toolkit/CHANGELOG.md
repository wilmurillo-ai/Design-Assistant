## 发布说明

### 版本命名规则
本项目使用语义化版本控制：
- **主版本号**: 不兼容的API变更
- **次版本号**: 向后兼容的功能新增
- **修订号**: 向后兼容的问题修复

### 升级指南
- 从0.x升级到1.0: 可能需要调整API调用
- 1.x版本之间: 通常可以直接升级
- 建议在升级前阅读变更日志

### 兼容性
- Python: >= 3.8
- 操作系统: Windows, macOS, Linux
- OpenClaw: >= 2026.2.0
