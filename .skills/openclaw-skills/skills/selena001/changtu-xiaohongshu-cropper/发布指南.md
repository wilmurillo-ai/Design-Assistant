## 📋 技能信息

### 基本信息
- **名称：** 长图转小红书图片切割工具
- **Slug：** changtu-xiaohongshu-cropper
- **版本：** 1.0.0
- **标签：** latest, image, xiaohongshu, cropper, social-media

### 功能描述
将任意长图按小红书 3:4 比例（1080×1440px）自动切割，支持：
- ✅ 自定义起始位置
- ✅ 自动切除底部多余内容（可配置高度）
- ✅ 移动端协作流程
- ✅ GUI 可视化操作
- ✅ 批量处理

### 使用场景
1. 小红书博主需要将长图转换成图文笔记
2. 移动端快速处理（手机发图 → AI 切割 → 返回结果）
3. 批量处理多张长图

---

## 🎯 核心特性

### 1. 小红书 3:4 比例
- 宽度：1080px
- 高度：1440px
- 完美适配小红书图文笔记

### 2. 自动切除底部
- 默认切除底部 160px（可配置）
- 支持自定义调整

### 3. 移动端协作
- 手机发图给 AI
- 告知起始位置（如"从顶部开始"）
- AI 切割后返回结果
- 随时随地完成

### 4. GUI 可视化
- 完整长图预览（可滚动）
- 鼠标点击选择起始位置
- 实时显示切割线
- 切割完成自动打开目录

## 📖 使用示例

### 命令行方式
```bash
# 基础用法
python cropper.py 长图.png

# 指定起始位置
python cropper.py 长图.png --start-y 400

# 完整参数
python cropper.py 长图.png \
  --start-y 0 \
  --logo-height 160 \
  --width 1080 \
  --height 1440 \
  --output ./output/
```

### GUI 方式
```bash
python cropper_gui.py
```

### 移动端协作
```
1. 手机发送长图给 AI
2. 告知起始位置（如"从顶部开始"）
3. AI 切割后逐张返回
4. 直接发布到小红书
```
