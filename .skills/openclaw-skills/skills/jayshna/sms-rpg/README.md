# SMS RPG Skill Bundle

This folder is the self-contained ClawHub/OpenClaw package for SMS RPG.

Included:

- `SKILL.md`
- `package.json`
- `code/`
- `dist/`
- `prompts/`
- `tsconfig.json`

Run:

```bash
npm run dialog -- --user demo-user --input "开始新游戏"
```

Or:

```bash
node dist/dialog_engine.js --user demo-user --input "开始新游戏"
```
