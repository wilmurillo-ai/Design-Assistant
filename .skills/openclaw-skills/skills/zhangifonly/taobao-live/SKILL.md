## 直播选品策略

### 三款式排品法

| 类型 | 占比 | 定价策略 | 作用 |
|------|------|----------|------|
| **引流款/福利款** | 15-20% | 成本价或微亏，9.9/19.9 | 拉升在线人数，激活直播间权重 |
| **利润款/主推款** | 50-60% | 正常利润 30-50% | 主要盈利来源，重点讲解 |
| **形象款/品牌款** | 20-30% | 高客单价 | 提升直播间调性和 UV 价值 |

### 排品节奏

标准 2 小时直播排品示例：

| 时间 | 品类 | 目的 |
|------|------|------|
| 0-10min | 福利款（秒杀） | 拉人气，留住观众 |
| 10-30min | 利润款 A | 趁人气高峰转化 |
| 30-40min | 福利款（抽奖） | 防止流失，再次拉升 |
| 40-70min | 利润款 B+C | 核心转化时段 |
| 70-80min | 形象款 | 提升客单价 |
| 80-100min | 利润款返场 | 二次转化 |
| 100-120min | 福利款收尾 | 感谢粉丝，预告下场 |

### 选品原则

- 应季应需：跟随季节和消费热点选品
- 价格优势：对比天猫旗舰店有明显让利空间
- 视觉冲击：适合镜头前展示，有对比效果
- 高复购率：日用消耗品优先，培养回购习惯
- 低售后率：避免尺码敏感、易碎、主观性强的品类

## 流量运营

### 淘宝直播流量来源

| 流量入口 | 占比参考 | 获取方式 |
|----------|----------|----------|
| **推荐流量（猜你喜欢）** | 30-50% | 直播间数据表现好，系统自动推荐 |
| **搜索流量** | 10-20% | 优化直播标题关键词 |
| **关注流量** | 15-25% | 粉丝基础，开播推送触达 |
| **超级推荐** | 10-30% | 付费投流，精准人群定向 |
| **短视频引流** | 5-15% | 开播前发预热短视频 |
| **微淘/逛逛** | 5-10% | 日常内容运营积累 |

### 直播预告策略

开播前 24 小时预热清单：

1. **发布预告短视频**：展示今日主推款，制造期待感
2. **微淘/逛逛发图文**：公布福利清单和开播时间
3. **粉丝群通知**：群内发红包 + 开播提醒
4. **设置直播预告**：淘宝直播后台设置预约功能
5. **店铺首页挂直播入口**：引导浏览店铺的用户进入直播间

### 超级推荐投流策略

| 投放阶段 | 预算分配 | 出价策略 | 目标 |
|----------|----------|----------|------|
| **冷启动期** | 日预算 200-500 | 高出价抢量 | 积累初始数据 |
| **爬坡期** | 日预算 500-2000 | 逐步降低出价 | 扩大流量规模 |
| **稳定期** | 日预算 2000+ | ROI 导向出价 | 稳定产出 |

人群定向建议：
- 店铺人群：已购买/收藏/加购用户（转化率最高）
- 相似人群：与店铺人群相似的潜在用户
- 行业人群：品类兴趣人群
- 达摩盘自定义人群：精细化标签组合

### 浮现权获取

淘宝直播浮现权是获得公域流量推荐的关键：

获取条件：
- 店铺信誉：一钻以上
- 直播时长：每月不低于一定时长（通常 20 小时以上）
- 内容质量：无违规记录，内容评分达标
- 互动数据：点赞、评论、分享等互动指标健康

维护要点：
- 保持稳定的开播频率和时长
- 避免任何违规行为（虚假宣传、诱导好评等）
- 持续提升直播间互动率和转化率

## 数据分析

### 关键指标

| 指标 | 计算方式 | 健康值参考 | 优化方向 |
|------|----------|-----------|----------|
| **观看人数（PV）** | 累计进入直播间人次 | 因账号体量而异 | 优化标题、封面、投流 |
| **观看人数（UV）** | 去重独立访客数 | PV/UV 比值反映回访率 | 提升内容吸引力 |
| **平均停留时长** | 总停留时长 / UV | >1.5 分钟为合格 | 优化留人话术和内容节奏 |
| **互动率** | 互动人数 / UV | >5% 为优秀 | 设计互动玩法和福利 |
| **转化率** | 成交人数 / UV | >3% 为优秀 | 优化产品讲解和促单话术 |
| **ARPU** | 成交额 / 成交人数 | 因品类而异 | 提升客单价和连带率 |
| **粉丝转化率** | 新增粉丝 / UV | >5% 为优秀 | 优化关注引导话术 |

### 淘宝直播特色指标

**亲密度体系**：
- 粉丝通过观看、互动、购买积累亲密度
- 亲密度等级：新粉 → 铁粉 → 钻粉 → 挚爱粉
- 高亲密度粉丝享受专属权益（优先购、专属价）

**粉丝等级权益设计**：

| 等级 | 亲密度要求 | 建议权益 |
|------|-----------|----------|
| 新粉 | 0-99 | 新人专属优惠券 |
| 铁粉 | 100-499 | 铁粉专属价、优先发货 |
| 钻粉 | 500-1999 | 钻粉专属款、免费试用 |
| 挚爱粉 | 2000+ | 最低价保障、1v1 客服 |

### 复盘方法论

每场直播结束后执行「五步复盘法」：

1. **数据导出**：从淘宝直播中控台导出完整数据
2. **流量分析**：各入口流量占比，哪个渠道效果最好
3. **转化漏斗**：观看 → 互动 → 点击 → 加购 → 成交，找到流失环节
4. **单品分析**：每款产品的讲解时长、点击率、转化率对比
5. **优化计划**：针对问题制定下场直播的改进方案

## 平台规则

### 淘宝直播违规红线

- **虚假宣传**：夸大产品功效、虚构使用效果
- **绝对化用语**：最好、第一、全网最低
- **诱导好评**：好评返现、好评送礼
- **价格欺诈**：虚标原价、虚假折扣
- **引导站外交易**：引导加微信、私下转账

### 处罚机制

| 违规等级 | 处罚措施 |
|----------|----------|
| 轻微违规 | 警告、扣分 |
| 一般违规 | 限流 3-7 天、暂停直播 |
| 严重违规 | 封禁直播权限 30 天以上 |
| 极端违规 | 永久封号、清退 |

### 最佳实践

- 保持每周 3-5 场的稳定开播频率
- 每场直播不少于 2 小时，培养粉丝观看习惯
- 固定开播时间，让粉丝形成期待
- 直播前 1 小时发布预热短视频
- 直播中穿插福利环节，每 20-30 分钟一次
- 重视售后服务，好口碑带来自然回购
