### 职场人设定位

四种常见人设类型：

1. **行业专家型**：深耕某一领域，输出专业洞察和方法论
2. **成长记录型**：分享职场成长经历，引发共鸣
3. **资源链接型**：分享行业资源、内推机会，做连接者
4. **观点输出型**：对行业热点发表独到见解，引发讨论

## 简历优化

### STAR 法则

每段工作经历用 STAR 法则描述核心成果：

- **S（Situation）**：什么背景/挑战
- **T（Task）**：你的职责/目标
- **A（Action）**：你采取了什么行动
- **R（Result）**：取得了什么量化成果

示例：
```
❌ 负责公司产品的用户增长工作

✅ 面对产品日活从 50 万下滑至 30 万的困境（S），
   作为增长负责人带领 5 人团队（T），
   搭建了用户分层运营体系，设计了 3 套召回策略（A），
   6 个月内将日活恢复至 80 万，留存率提升 25%（R）
```

### 关键词优化

简历关键词来源：
- 目标岗位 JD 中的高频词汇
- 行业通用技能术语
- 工具和技术栈名称
- 管理能力相关词汇（带团队、跨部门协作）

### 简历常见问题

| 问题 | 优化建议 |
|------|----------|
| 职责描述而非成果 | 每条经历必须有量化数据 |
| 经历过多过杂 | 只保留与目标岗位相关的 3-5 段 |
| 缺少关键词 | 对照 JD 补充行业术语 |
| 格式混乱 | 统一字体、对齐、间距 |
| 自我评价空洞 | 删除或改为具体能力标签 |

## 职场社交技巧

### 破冰话术模板

场景一：想认识行业大佬
```
"XX 老师您好，我是 [公司] 的 [职位]，
拜读了您关于 [具体话题] 的分享，非常受启发。
特别是 [具体观点] 这个点，和我目前在做的 [项目] 很相关，
想请教一下您在这方面的经验，方便交流吗？"
```

场景二：想内推求职
```
"您好，我注意到贵司在招 [岗位]，
我在 [领域] 有 X 年经验，曾在 [公司] 负责 [项目]，
取得了 [量化成果]。冒昧打扰，想了解一下这个岗位的具体情况，
如果合适的话，能否帮忙内推？附上我的简历供参考。"
```

场景三：日常人脉维护
```
"XX 好久不见！看到你最近分享的 [内容]，写得真好。
最近我们团队也在做类似的事情，有机会线下聊聊？"
```

### 人脉维护法则

- **定期互动**：每周花 15 分钟给重要联系人的动态点赞评论
- **价值输出**：看到对方可能感兴趣的文章/机会主动分享
- **节日问候**：春节、中秋等节点发送个性化祝福（非群发）
- **线下见面**：每季度至少约 2-3 位重要人脉线下交流

### 社交礼仪

| 场景 | 建议 | 避免 |
|------|------|------|
| 初次联系 | 说明来意，简洁有礼 | 上来就推销或求帮忙 |
| 请教问题 | 先做功课，问具体问题 | 问太宽泛的问题浪费对方时间 |
| 内推请求 | 附简历，说明匹配度 | 只说"帮我内推"不提供信息 |
| 被拒绝 | 礼貌感谢，保持联系 | 纠缠或表达不满 |

## 行业洞察

### 薪资谈判技巧

1. **调研市场行情**：通过脉脉薪资查询、职级对标了解目标薪资范围
2. **量化个人价值**：用数据说明你能带来的价值
3. **谈判时机**：拿到 offer 后、年度调薪前是最佳时机
4. **报价策略**：报期望薪资的 110-120%，留出谈判空间
5. **全面考虑**：除基本薪资外，关注股票期权、年终奖、福利

### 跳槽时机判断

适合跳槽的信号：
- 连续 2 年没有晋升或加薪
- 当前工作无法提供新的成长空间
- 行业下行，公司前景不明朗
- 收到明显高于当前的 offer
- 团队/领导变动导致工作环境恶化

不建议跳槽的情况：
- 入职不满 1 年（简历不好看）
- 没有明确的下家方向
- 仅因为短期情绪波动
- 行业整体不景气，外部机会少

### 脉脉匿名区使用建议

- 发布内容注意脱敏，避免暴露个人身份
- 不要发布公司机密信息，可能涉及法律风险
- 理性看待匿名区信息，存在夸大和偏见
- 可以用来了解行业薪资水平和公司口碑
- 避免参与人身攻击和恶意中伤
