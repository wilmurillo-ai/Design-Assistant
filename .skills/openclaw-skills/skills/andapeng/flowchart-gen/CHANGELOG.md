## [1.1.0] - 2026-04-17

### 🛡️ 安全架构优化
基于ClawHub安全评估报告，重构技能的安全架构和配置系统，提升透明度和安全性。

#### ✨ 新增功能
- **命令行API配置参数**：新增 `--api-key`、`--api-provider`、`--api-base-url` 参数，支持手动指定API配置
- **标准化安装脚本**：新增 `install.ps1`（Windows）和 `install.sh`（Linux/macOS），提供一键安装和验证

#### 🔧 技术改进
- **配置读取逻辑重构**：明确定义配置优先级（命令行参数 > 环境变量 > OpenClaw配置文件）
- **配置向导系统**：首次运行或缺少配置时，提供详细的配置指引和故障排除建议
- **元数据声明完善**：在SKILL.md中明确声明环境变量和配置文件访问权限

#### 🔒 安全增强
- **权限透明化**：所有配置访问行为已在元数据中明确声明，符合最小权限原则
- **安全审计友好**：配置访问逻辑清晰可审计，便于安全审查
- **向后兼容**：保持原有环境变量和OpenClaw配置文件支持，不影响现有用户

#### 📚 文档更新
- **配置说明**：更新LLM API配置章节，详细说明配置优先级和多种配置方式
- **安全指南**：新增安全说明章节，解释技能的安全架构设计
- **安装指南**：推荐使用安装脚本，简化依赖安装流程

### 🐛 修复的问题
- **发布包完整性**：确保 `templates/` 目录包含在技能包中
- **文件清单一致性**：修复了测试脚本引用缺失模板目录的问题
- **配置错误处理**：改进配置失败时的错误提示和恢复建议

## [1.0.0] - 2026-03-17

### 🎉 首次发布
流程图生成技能 v1.0.0 正式发布！

### ✨ 新增功能
- **DeepSeek API智能生成**：自动从OpenClaw配置读取API密钥，实现自然语言到Mermaid代码的智能转换
- **丰富的模板库**：31个预置模板，覆盖业务流程、技术架构、项目管理、教育培训等常用场景
- **多图表类型支持**：支持8种Mermaid图表类型（流程图、序列图、甘特图、类图、状态图、饼图、旅程图、时间线图）
- **智能错误处理**：环境依赖自动检查、Mermaid错误智能分析、详细的故障排除指南
- **多格式输出**：支持PNG、SVG、PDF格式输出
- **多主题支持**：支持default、dark、forest、neutral等多种主题

### 🔧 技术特性
- **Windows兼容性优化**：自动检测并使用 `mmdc.cmd`，支持Windows系统
- **Puppeteer自动配置**：自动检测并设置Chrome路径，解决Chrome依赖问题
- **版本管理系统**：支持 `--version` 参数显示版本信息
- **调试模式**：支持 `--debug` 参数保存临时文件，便于问题诊断
- **中文友好界面**：全中文错误提示、帮助文档和用户界面

### 📚 文档完善
- **SKILL.md**：完整的技能说明文档
- **FAQ.md**：详细的常见问题解答
- **TODO.md**：开发任务跟踪（所有任务已完成）
- **进度报告-2026-03-17.md**：完整的项目总结报告
- **CHANGELOG.md**：版本变更记录（本文件）

### 🛠 修复的问题
- **两个路径不一致问题**：修复了workspace路径与技能安装路径脚本版本不一致的问题
- **Windows命令问题**：修复了Windows上 `mmdc` 命令无法识别的问题，改用 `mmdc.cmd`
- **Chrome依赖问题**：自动配置Puppeteer使用系统Chrome，解决Chrome依赖错误
- **参数解析问题**：修复了 `--list-templates` 参数设计缺陷，使prompt参数变为可选

### ⚙️ 环境要求
- **Node.js**：需要安装Node.js以使用Mermaid CLI
- **Mermaid CLI**：需要安装 `@mermaid-js/mermaid-cli` 包
- **Python依赖**：推荐安装 `pillow` 和 `requests` 库
- **Chrome浏览器**：需要Chrome浏览器以完整渲染图表（SVG格式无需Chrome）

### 📦 安装步骤
```bash
# 1. 安装Mermaid CLI
npm install -g @mermaid-js/mermaid-cli

# 2. 安装Python依赖
pip install pillow requests

# 3. 验证安装
mmdc --version
python scripts/generate.py --version
```

## 版本命名规则

本项目使用 [语义化版本 2.0.0](https://semver.org/lang/zh-CN/)：

- **主版本号**：当你做了不兼容的 API 修改
- **次版本号**：当你做了向下兼容的功能性新增
- **修订号**：当你做了向下兼容的问题修正

## 文件版本对应关系

| 文件 | 版本 | 描述 |
|------|------|------|
| `VERSION` | 1.0.0 | 纯文本版本号文件 |
| `scripts/generate.py` | 1.0.0 | 主生成脚本，包含版本信息 |

## 版本验证

```bash

python scripts/generate.py --version

# 查看文件版本
cat VERSION

# 检查脚本版本信息
head -20 scripts/generate.py | grep -E "版本|version"
```
