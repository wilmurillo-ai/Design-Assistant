# Snowflake Data Engineering Skill

Version: 1.0.0

## Overview
Governed Snowflake read-only execution layer with enforced SELECT-only queries,
row limiting, and execution metadata return.

## Capabilities
- SELECT-only query execution
- Forbidden keyword blocking
- Row limiting
- Query timeout enforcement
- Structured JSON response

## Security
- Read-only Snowflake role
- No DDL/DML permissions
- Statement timeout enforced
