# 生肖描述参考

## 十二生肖特点

### 鼠 (Rat)
- **性格特点**: 机智灵活、善于交际、适应力强
- **优点**: 聪明、勤奋、节俭
- **缺点**: 有时过于谨慎、缺乏安全感
- **幸运元素**: 水、蓝色、数字1,4,6

### 牛 (Ox)
- **性格特点**: 勤奋踏实、有耐心、责任感强
- **优点**: 可靠、稳定、坚持
- **缺点**: 固执、缺乏变通
- **幸运元素**: 土、黄色、数字3,7,9

### 虎 (Tiger)
- **性格特点**: 勇敢果断、有领导力、充满活力
- **优点**: 热情、自信、冒险精神
- **缺点**: 冲动、控制欲强
- **幸运元素**: 木、绿色、数字1,3,4

### 兔 (Rabbit)
- **性格特点**: 温和善良、谨慎小心、人际关系好
- **优点**: 体贴、优雅、有艺术感
- **缺点**: 过于敏感、逃避冲突
- **幸运元素**: 木、绿色、数字3,4,9

### 龙 (Dragon)
- **性格特点**: 充满活力、有创造力、自信
- **优点**: 热情、慷慨、有远见
- **缺点**: 自负、要求高
- **幸运元素**: 土、金色、数字1,6,7

### 蛇 (Snake)
- **性格特点**: 智慧敏锐、直觉强、神秘
- **优点**: 聪明、冷静、有魅力
- **缺点**: 多疑、嫉妒心强
- **幸运元素**: 火、红色、数字2,8,9

### 马 (Horse)
- **性格特点**: 自由奔放、精力充沛、乐观
- **优点**: 热情、独立、行动力强
- **缺点**: 急躁、缺乏耐心
- **幸运元素**: 火、红色、数字2,3,7

### 羊 (Goat)
- **性格特点**: 温和体贴、艺术感强、善良
- **优点**: 有创造力、同情心、优雅
- **缺点**: 优柔寡断、依赖性强
- **幸运元素**: 土、黄色、数字3,4,9

### 猴 (Monkey)
- **性格特点**: 聪明机智、适应力强、好奇
- **优点**: 灵活、幽默、学习能力强
- **缺点**: 狡猾、缺乏专注
- **幸运元素**: 金、白色、数字4,7,8

### 鸡 (Rooster)
- **性格特点**: 勤奋守时、注重细节、诚实
- **优点**: 有条理、可靠、观察力强
- **缺点**: 挑剔、爱批评
- **幸运元素**: 金、白色、数字5,7,8

### 狗 (Dog)
- **性格特点**: 忠诚可靠、责任感强、正直
- **优点**: 诚实、勇敢、有同情心
- **缺点**: 悲观、固执
- **幸运元素**: 土、黄色、数字3,4,9

### 猪 (Pig)
- **性格特点**: 乐观豁达、真诚善良、享受生活
- **优点**: 慷慨、诚实、有福气
- **缺点**: 天真、容易受骗
- **幸运元素**: 水、黑色、数字2,5,8

## 生肖相生相克

### 三合生肖（和谐）
- 鼠、龙、猴
- 牛、蛇、鸡
- 虎、马、狗
- 兔、羊、猪

### 六合生肖（最佳配对）
- 鼠与牛
- 虎与猪
- 兔与狗
- 龙与鸡
- 蛇与猴
- 马与羊

### 相冲生肖（冲突）
- 鼠与马
- 牛与羊
- 虎与猴
- 兔与鸡
- 龙与狗
- 蛇与猪
