# 颜色含义参考

## 颜色在运势中的象征意义

### 红色 (Red)
- **正面意义**: 热情、活力、勇气、好运、喜庆
- **运势关联**: 增强行动力，吸引机会，提升自信
- **适合场合**: 重要会议、社交活动、创业开始
- **搭配建议**: 与金色、黑色搭配增强效果

### 金色 (Gold)
- **正面意义**: 财富、成功、尊贵、智慧、光明
- **运势关联**: 吸引财富，提升地位，获得认可
- **适合场合**: 商务谈判、投资决策、重要展示
- **搭配建议**: 与黑色、白色、红色搭配

### 蓝色 (Blue)
- **正面意义**: 平静、智慧、信任、忠诚、专业
- **运势关联**: 增强沟通，促进合作，提升思考
- **适合场合**: 学习考试、重要会议、谈判沟通
- **搭配建议**: 与白色、银色、灰色搭配

### 绿色 (Green)
- **正面意义**: 成长、健康、和谐、平衡、新生
- **运势关联**: 促进成长，改善健康，带来和谐
- **适合场合**: 健康养生、学习成长、家庭和谐
- **搭配建议**: 与棕色、白色、黄色搭配

### 紫色 (Purple)
- **正面意义**: 灵性、创意、神秘、高贵、智慧
- **运势关联**: 激发创意，提升直觉， spiritual growth
- **适合场合**: 创意工作、 spiritual practice、艺术创作
- **搭配建议**: 与银色、白色、粉色搭配

### 白色 (White)
- **正面意义**: 纯洁、清晰、新的开始、和平、简洁
- **运势关联**: 清理过去，新的开始， clarity of mind
- **适合场合**: 重大决策、新的开始、清理整顿
- **搭配建议**: 与任何颜色都搭配

### 黑色 (Black)
- **正面意义**: 保护、力量、优雅、神秘、专业
- **运势关联**: 提供保护，增强力量，保持低调
- **适合场合**: 需要保护、重要决策、 formal occasions
- **搭配建议**: 与金色、银色、红色搭配

### 银色 (Silver)
- **正面意义**: 直觉、未来、科技、优雅、平衡
- **运势关联**: 增强直觉，连接未来，平衡能量
- **适合场合**: 科技工作、未来规划、直觉决策
- **搭配建议**: 与蓝色、紫色、黑色搭配

### 黄色 (Yellow)
- **正面意义**: 快乐、智慧、沟通、创造力、乐观
- **运势关联**: 提升 mood，增强沟通，激发创意
- **适合场合**: 社交活动、创意工作、学习交流
- **搭配建议**: 与蓝色、绿色、棕色搭配

### 橙色 (Orange)
- **正面意义**: 创造力、热情、社交、活力、温暖
- **运势关联**: 激发热情，促进社交，增强活力
- **适合场合**: 团队活动、创意项目、社交聚会
- **搭配建议**: 与蓝色、绿色、棕色搭配

## 颜色使用建议

### 日常穿搭
- **工作日**: 蓝色、灰色、白色（专业、专注）
- **社交日**: 红色、黄色、橙色（热情、友好）
- **休息日**: 绿色、棕色、米色（放松、自然）
- **特殊场合**: 金色、黑色、紫色（正式、重要）

### 环境布置
- **办公室**: 蓝色、绿色（提升 focus 和 creativity）
- **卧室**: 粉色、米色、浅蓝色（放松、安宁）
- **客厅**: 黄色、橙色、绿色（温暖、和谐）
- **学习区**: 绿色、蓝色（促进学习、思考）

### 数字时代
- **手机壁纸**: 根据当天运势选择颜色
- **电脑桌面**: 蓝色、绿色减少 eye strain
- **社交媒体**: 使用幸运色作为主题色
- **数字文档**: 使用幸运色 highlight 重要内容
