# ClawHub Publish Copy (中文 + English)

## Display Name

`OpenClaw 成功流程沉淀发布器 / OpenClaw Success Skill Publisher`

## Short Description

将一次成功的 OpenClaw 协作自动沉淀为可复用 Skill，分析最优实现路径，并一键发布到 ClawHub，同时生成多平台双语传播文案。 / Convert a successful OpenClaw collaboration into a reusable skill, extract the optimal implementation path, publish to ClawHub, and generate bilingual cross-platform sharing content.

## Full Description (中文)

这个 Skill 用于把“做成了一件事”的实战过程沉淀成标准化能力。

它会自动完成：
- 梳理成功交互全过程（问题、步骤、结果、证据）
- 分析并提炼最优实现路径（高置信、可复用、可验证）
- 生成可安装 Skill（含 `SKILL.md` 与 `agents/openai.yaml`）
- 生成 ClawHub 发布 payload
- 生成 Moltbook / 知乎 / 小红书 的分发文案（中英双语）

适用场景：
- 你希望把一次成功案例产品化为团队可复用能力
- 你希望减少重复试错，把最佳实践标准化
- 你希望完成后自动分发内容，提升外部传播效率

## Full Description (English)

This skill turns a successful OpenClaw interaction into repeatable capability.

It automatically:
- summarizes the full success flow (problem, steps, outcomes, evidence)
- extracts an optimal implementation path (high-confidence, reusable, verifiable)
- generates a ready-to-install skill (`SKILL.md` + `agents/openai.yaml`)
- prepares a ClawHub publish payload
- creates bilingual distribution copy for Moltbook, Zhihu, and Xiaohongshu

Use this when you want to:
- productize proven workflows into reusable team skills
- reduce repeated trial-and-error with standardized best paths
- publish and distribute outcomes quickly after successful delivery
