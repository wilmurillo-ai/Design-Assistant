# 六爻纳甲排盘工具

> 天工长老开发 · v2.0.0

六爻纳甲是汉代京房所创的占卜方法，以铜钱起卦，纳干支、六亲、六神，断吉凶。

## 功能特性

- ✅ 三种起卦方式：时间起卦、数字起卦、铜钱起卦
- ✅ 完整纳甲装卦：天干地支、五行属性
- ✅ 六亲排布：父母、兄弟、子孙、妻财、官鬼
- ✅ 六神排布：青龙、朱雀、勾陈、螣蛇、白虎、玄武
- ✅ 世应定位：自动计算世爻应爻位置
- ✅ 动爻变卦：标记动爻，推算变卦
- ✅ 自动化断卦：用神选取、旺衰分析、吉凶判断

## 快速开始

### 安装

```bash
# 克隆仓库
git clone https://github.com/dglijin-oss/liuyao-najia-skill.git
cd liuyao-najia-skill

# 无需依赖，纯 Python 实现
python3 --version  # 需要 Python 3.6+
```

### 基本用法

```bash
# 时间起卦（默认当前时间）
python3 scripts/liuyao_pan.py

# 指定时间起卦
python3 scripts/liuyao_pan.py --date "2026-03-21 15:00"

# 数字起卦
python3 scripts/liuyao_pan.py --numbers "123,456,789"

# 铜钱起卦
python3 scripts/liuyao_pan.py --coins "3,2,3,2,3,2"

# 问事断卦
python3 scripts/liuyao_pan.py --question "财运"

# JSON 输出
python3 scripts/liuyao_pan.py --json
```

### 命令行参数

| 参数 | 简写 | 说明 |
|------|------|------|
| --date | -d | 日期时间 (YYYY-MM-DD HH:MM) |
| --numbers | -n | 数字起卦 (逗号分隔的三个数字) |
| --coins | -c | 铜钱起卦 (6 次背面数，逗号分隔) |
| --question | -q | 问事类型 (财运/事业/婚姻/健康等) |
| --json | -j | 输出 JSON 格式 |
| --help | -h | 显示帮助信息 |

## 问事类型

| 类型 | 用神 | 说明 |
|------|------|------|
| 财运 | 妻财 | 财富、交易、投资 |
| 事业 | 官鬼 | 工作、职位、官司 |
| 婚姻 | 妻财/官鬼 | 感情、配偶 |
| 健康 | 子孙 | 疾病、医药 |
| 考试 | 父母 | 学业、文书 |
| 出行 | 子孙 | 旅行、平安 |

### 六亲关系

- **父母**：生我者（长辈、文书）
- **兄弟**：同我者（朋友、同事）
- **子孙**：我生者（子女、财源）
- **妻财**：我克者（财富、物质）
- **官鬼**：克我者（职位、压力）
