# 七政四余排盘工具

> 天工长老开发 · v2.0.0

七政四余是中国古代占星术，源自果老星宗。七政为日月五星，四余为罗睺、计都、月孛、紫气。

## 功能特性

- ✅ 精确星曜位置计算（简化天文算法）
- ✅ 黄经度数显示（0-360°）
- ✅ 二十八宿排布
- ✅ 十二宫位完整排布
- ✅ 星曜庙旺落陷判断
- ✅ 多种格局判断（日月拱照、日月合璧、金水相生等）
- ✅ 自动化断语生成
- ✅ 四柱八字显示

## 快速开始

### 安装

```bash
# 克隆仓库
git clone https://github.com/dglijin-oss/qizheng-siyu-skill.git
cd qizheng-siyu-skill

# 无需依赖，纯 Python 实现
python3 --version  # 需要 Python 3.6+
```

### 基本用法

```bash
# 当前时间排盘
python3 scripts/qizheng_pan.py

# 指定时间排盘
python3 scripts/qizheng_pan.py --date "2026-03-21 15:00"

# 指定地点（经纬度）
python3 scripts/qizheng_pan.py --date "2026-03-21 15:00" --lat 39.9 --lon 116.4

# JSON 输出
python3 scripts/qizheng_pan.py --json
```

### 命令行参数

| 参数 | 简写 | 说明 |
|------|------|------|
| --date | -d | 日期时间 (YYYY-MM-DD HH:MM) |
| --lat | | 纬度（默认 39.9） |
| --lon | | 经度（默认 116.4） |
| --json | -j | 输出 JSON 格式 |
| --help | -h | 显示帮助信息 |

## 七政四余基础

### 七政（日月五星）

| 星曜 | 五行 | 吉凶 | 含义 |
|------|------|------|------|
| 太阳 | 火 | 大吉 | 事业、名声、父亲 |
| 太阴 | 水 | 吉 | 情感、母亲、女性 |
| 木星 | 木 | 吉 | 财运、扩张、智慧 |
| 火星 | 火 | 凶 | 行动、竞争、冲突 |
| 土星 | 土 | 凶 | 压力、责任、限制 |
| 金星 | 金 | 吉 | 爱情、艺术、财富 |
| 水星 | 水 | 中 | 智慧、沟通、学习 |
