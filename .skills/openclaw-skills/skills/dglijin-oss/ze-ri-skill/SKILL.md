# 择日学技能 v2.0.0

> "择吉避凶，顺天应时。"

## 功能

- ✅ 建除十二神查询
- ✅ 黄道黑道日查询
- ✅ 神煞吉凶查询
- ✅ 宜忌事项查询
- ✅ 冲煞查询（冲生肖/煞方位）
- ✅ 分类择日（婚嫁/开业/动土/出行等）
- ✅ 吉日推荐（月度/年度）

## 使用

```bash
python3 scripts/ze_ri.py --date "2026-04-15"
python3 scripts/ze_ri.py --month "2026-04" --event "婚嫁"
```
