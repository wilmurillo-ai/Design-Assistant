## 5. 太乙基础知识

### 太乙积年

**计算公式：**
```
太乙积年 = 从上元甲子至今的年数
基准年：公元前 2917 年（上元甲子）
积年 = 当前年份 + 2917
```

**示例：**
- 2026 年积年 = 2026 + 2917 = 4943 年

## 6. 主客算推算

### 计算公式

```
主算 = (太乙积年 + 局数) mod 72
客算 = (太乙积年 - 局数) mod 72
```

### 判断规则

| 比较 | 含义 |
|------|------|
| 主算 > 客算 | 主方胜、内力强 |
| 客算 > 主算 | 客方胜、外力强 |
| 主算 = 客算 | 平局、相持 |

### 数值含义

| 数值范围 | 含义 |
|----------|------|
| 1-18 | 弱 |
| 19-36 | 中 |
| 37-54 | 强 |
| 55-72 | 极强 |

## 8. 断局规则

### 8.1 吉凶判断

**吉象：**
- 太乙落旺宫（离、乾、巽）
- 主算强旺（37-72）
- 吉神临用（贵人、青龙、文昌）
- 无凶神冲克

**凶象：**
- 太乙落衰宫（坎、坤）
- 客算强于主算
- 凶神临用（白虎、螣蛇、始击）
- 有冲克破害

### 8.2 应期推算

| 局数 | 应期 |
|------|------|
| 1-3 局 | 近期（1-3 月） |
| 4-6 局 | 中期（3-6 月） |
| 7-9 局 | 远期（6-12 月） |

### 8.3 分类断语

**国运：**
- 太乙落离宫 → 国运昌隆，文化兴盛
- 太乙落乾宫 → 政治稳定，领导有力
- 太乙落坎宫 → 需谨慎，防内忧

**天灾：**
- 天目落坎宫 → 防水灾
- 天目落离宫 → 防火灾、旱灾
- 地目落坤宫 → 防地震

**经济：**
- 文昌旺 → 文化产业兴盛
- 始击旺 → 竞争激烈
- 客算强 → 外资影响大
