# 风水堪舆技能 v2.0.0

> "风水者，藏风得水也。"

## 功能

- ✅ 八宅分类（东四宅/西四宅）
- ✅ 命卦计算（出生年份推算）
- ✅ 游年九星排布（伏位/生气/天医等）
- ✅ 方位吉凶分析
- ✅ 布局建议（门窗/卧室/厨房/卫生间）
- ✅ 煞气识别与化解

## 使用

```bash
python3 scripts/fengshui_pan.py --year 1990
python3 scripts/fengshui_pan.py --year 1990 --direction "坐北朝南"
```
