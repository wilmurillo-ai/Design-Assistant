## 3. How to use

### 调用方式

```bash

# 指定时间排课

# 指定问事类型
```

### 三传起法

**贼克法：**
- 下克上为"贼"，上克下为"克"
- 有一克取一克，无克取比用
- 比用：与日干阴阳相同者优先

**比用法：**
- 两课俱克或俱无克，取与日干比和者

**涉害法：**
- 比用后仍有多者，取涉害深者

**遥克法：**
- 无克无比，取遥相克者

**昴星法：**
- 无遥无克，阳日取酉上神，阴日取酉下神

**别责法：**
- 四课不全，取干合上神或支三合

**八专法：**
- 干支同位，阳日顺数，阴日逆数

**伏吟法：**
- 天地盘相同，取刑冲

**返吟法：**
- 天地盘对冲，取冲位

- 贵人、青龙、太常、六合、太阴、天后临用神
- 三传生干支
- 财官旺相
- 无刑冲破害

**凶象：**
- 螣蛇、朱雀、勾陈、白虎、玄武、天空临用神
- 三传克干支
- 鬼爻旺相
