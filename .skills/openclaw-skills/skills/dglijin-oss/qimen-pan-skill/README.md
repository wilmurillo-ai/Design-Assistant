# 奇门遁甲排盘技能

> "易与天地准，故能弥纶天地之道" ——《周易·系辞》

[![Version](https://img.shields.io/badge/version-2.0.1-blue.svg)](CHANGELOG.md)

奇门遁甲是中国古代最高层次的预测学，与太乙、六壬合称"三式"。本技能提供高精度的奇门遁甲排盘和断卦解盘功能。

---

## 🌟 特性

### 核心功能
- ✨ **精确农历转换**（1900-2100 年，精度±1 分钟）
- ✨ **精确节气计算**（天文算法）
- ✨ **真太阳时校正**（支持城市/经度）
- ✨ **四柱排盘**（年柱、月柱、日柱、时柱）
- ✨ **九宫落位**（八门、九星、八神）
- ✨ **断卦解盘自动化**（v2.0.1 新增）
- ✨ **用神选取规则**（10 种问事类型）
- ✨ **吉凶判断逻辑**（门/星/神三级判断）

### 技术特点
- 🔧 纯 Python 实现，无外部依赖
- 🔧 内置农历和节气算法
- 🔧 支持 JSON 格式输出
- 🔧 命令行界面友好

## 🚀 快速开始

### 安装

无需安装，直接运行：

```bash
# 确保 Python 3.8+ 已安装
python3 --version

# 赋予执行权限
chmod +qimen_pan_v2.py
```

### 基本用法

```bash
# 当前时间排盘
python3 qimen_pan_v2.py

# 指定时间排盘
python3 qimen_pan_v2.py --date "2026-03-20 10:00"

# 指定问事类型（带断卦分析）
python3 qimen_pan_v2.py --date "2026-03-20 10:00" --question "财运"

# 真太阳时校正（按城市）
python3 qimen_pan_v2.py --date "2026-03-20 10:00" --city "北京"

# 输出 JSON 格式
python3 qimen_pan_v2.py --date "2026-03-20 10:00" --json
```

### 问事类型

| 类型 | 参数 | 用神 |
|------|------|------|
| 通用 | `通用` | 综合判断 |
| 财运 | `财运` | 生门、戊土、六合 |
| 事业 | `事业` | 开门、庚金、值符 |
| 婚姻 | `婚姻` | 六合、乙庚、太阴 |
| 健康 | `健康` | 天芮星、死门、白虎 |
| 官司 | `官司` | 惊门、辛金、玄武 |
| 出行 | `出行` | 开门、丁奇、九天 |
| 求子 | `求子` | 生门、丁奇、九地 |
| 失物 | `失物` | 杜门、辛金、玄武 |
| 考试 | `考试` | 景门、丁奇、天辅 |
| 交易 | `交易` | 生门、戊土、六合 |

## 📚 断卦规则

### 用神选取

用神是断卦的核心，根据问事类型选取：

- **财运**：生门（利润）、戊土（资本）、六合（交易）
- **事业**：开门（工作）、庚金（阻力）、值符（领导）
- **婚姻**：六合（媒合）、乙庚（夫妻）、太阴（隐私）
- **健康**：天芮（病星）、死门（凶兆）、白虎（血光）

### 吉凶判断

#### 门吉凶
- **吉门**：开门、休门、生门
- **中门**：杜门、景门
- **凶门**：伤门、死门、惊门

#### 星吉凶
- **吉星**：天冲、天辅、天禽、天心、天任
- **中星**：天英
- **凶星**：天蓬、天芮、天柱

#### 神吉凶
- **大吉**：值符
- **吉神**：太阴、六合、九地、九天
- **凶神**：螣蛇、白虎、玄武

### 五行生克

| 宫位 | 五行 |
|------|------|
| 坎宫 | 水 |
| 坤宫 | 土 |
| 震宫 | 木 |
| 巽宫 | 木 |
| 中宫 | 土 |
| 乾宫 | 金 |
| 兑宫 | 金 |
| 艮宫 | 土 |
| 离宫 | 火 |

**生克关系：**
- 相生：木→火→土→金→水→木
- 相克：木→土→水→火→金→木

## 🔧 API 参考

### 命令行参数

```
usage: qimen_pan_v2.py [-h] [--date DATE] [--timestamp TIMESTAMP]
                       [--longitude LONGITUDE] [--city CITY]
                       [--question QUESTION] [--json]

奇门遁甲排盘工具（高精度版 v2.0.1）

optional arguments:
  -h, --help            显示帮助信息
  --date, -d DATE       日期时间 (YYYY-MM-DD HH:MM)
  --timestamp, -t TIMESTAMP
                        Unix 时间戳
  --longitude, -lon LONGITUDE
                        当地经度（真太阳时校正）
  --city, -c CITY       城市名（自动获取经度）
  --question, -q QUESTION
                        问事类型（财运/事业/婚姻/健康/官司/出行/求子/失物/考试/交易）
  --json, -j            输出 JSON 格式
```

### Python 调用

```python
from qimen_pan_v2 import qimen_pan, format_output

# 排盘
result = qimen_pan(
    date_str="2026-03-20 10:00",
    question_type="事业"
)

# 格式化输出
print(format_output(result, "事业"))

# 或直接获取字典数据
print(result['断卦分析'])
```
