```

    ↓         ↓         ↓
```

## [2.0.0] - 2026-03-18

### 🎉 新增功能

#### 精确农历转换
- ✨ 基于天文算法的农历计算
- ✨ 支持范围：1900 年 - 2100 年
- ✨ 精度：±1 分钟
- ✨ 内置农历数据表，无需外部依赖

#### 精确节气计算
- ✨ 基于天文算法的节气计算
- ✨ 24 节气精确判断
- ✨ 阴阳遁自动判断

#### 真太阳时校正
- ✨ 支持按城市自动获取经度
- ✨ 支持手动指定经度
- ✨ 均时差（Equation of Time）计算
- ✨ 时辰精确判断

### 📝 改进优化

- 📝 版本号更新至 2.0.0
- 📝 代码注释完善
- 📝 错误处理优化
- 📝 输出格式美化

### 🔧 技术细节

#### 新增类
- `LunarCalendar` - 农历计算类
- `JieQiCalculator` - 节气计算类
- `TrueSolarTime` - 真太阳时计算类

#### 核心算法
- 农历转换：基于儒略日计算
- 节气计算：基于太阳黄经
- 真太阳时：经度差 + 均时差

## 📚 版本规范

本项目遵循语义化版本规范（Semantic Versioning）：

- **主版本号（Major）**：不兼容的 API 变更
- **次版本号（Minor）**：向后兼容的功能新增
- **修订号（Patch）**：向后兼容的问题修复

### 版本号格式

```
主版本号。次版本号。修订号
例：2.0.1
```
