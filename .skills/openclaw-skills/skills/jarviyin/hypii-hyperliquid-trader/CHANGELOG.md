# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-06

### Added
- Initial release of Hypii Hyperliquid Trading Agent
- Portfolio management (balance, positions, P&L)
- Real-time price queries
- DCA (Dollar Cost Averaging) strategy
- Grid trading strategy
- AI trading signals with trend analysis
- Automated trade execution (market/limit orders)
- SkillPay integration for micropayments
- Free tier: 5 calls per day
- Pay-per-use pricing: 0.01-0.1 USDT per call
- Pro subscription: 19.9 USDT/month unlimited
- Risk management features (slippage protection, position sizing)
- Support for Hyperliquid testnet

### Security
- Private key never exposed in logs
- Slippage protection on market orders
- Position size warnings
