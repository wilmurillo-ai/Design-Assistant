# 国网研究报告文档风格规范

## 一、整体风格定位

国网公司研究报告属于**政策性研究文书**，兼具学术严谨性与实践指导性。其文风特征为：
- 正式、客观、准确，具有国有企业的"公文气质"
- 理论联系实际，有政策依据，有数据支撑
- 层次分明，结构清晰，便于领导层快速把握重点
- 语言凝练，避免冗余，重点突出

---

## 二、标题层级规范

### 完整层级体系

| 层级 | 格式 | 示例 | 字体要求 |
|------|------|------|----------|
| 一级标题 | 一、二、三、 | 一、原集体企业考核条线现状分析 | 黑体，16pt，居中 |
| 二级标题 | （一）（二）（三） | （一）省级条线考核机制现状 | 黑体，14pt，左对齐 |
| 三级标题 | 1. 2. 3. | 1. 考核主体与授权关系 | 仿宋加粗，12pt |
| 四级标题 | （1）（2）（3） | （1）省公司层面的职责边界 | 仿宋加粗，12pt |
| 列举项 | ①②③ 或 一是二是三是 | ①指标设计缺乏差异化 | 正文字体 |

### 注意事项
- **不得跳级**：二级标题下不能直接出现四级标题
- **层级内容量**：三级以下内容如果超过3个条目，考虑重新规划为单独章节
- 列举项在正文段落中使用，不作为独立标题行

## 三、章节结构模式

### 分析性章节（现状/问题分析）
```
（一）[主题]现状概述
    —— 1-2段综述性描述，点明主要特征

（二）主要问题与挑战
    1. 问题一：[具体问题标题]
       [描述现象] + [成因分析]
    2. 问题二：……
    3. 问题三：……

（三）原因深层分析
    —— 从制度设计、历史沿革、管理机制等角度深入剖析
```

### 建议性章节（对策/方案设计）
```
（一）总体思路
    —— 原则、目标、路径的高度概括（配合政策依据）

（二）具体措施
    1. 措施一：[标题]
       [操作内容] + [预期效果]
    2. 措施二：……

（三）实施保障
    —— 组织保障、制度保障、工作机制
```

### 综合性章节（机制设计）
```
（一）[机制名称]总体框架
    —— 结构图描述或要素说明

（二）核心要素设计
    1. 要素一
    2. 要素二

（三）运行机制
    —— 流程、接口、协同关系
```

## 四、核心语言规范

### 问题描述句式
- "……存在……问题，主要表现在以下几个方面"
- "当前，……尚未建立完善的……机制"
- "实践中，……与……之间存在明显的……矛盾"
- "从调研情况来看，……普遍反映……"

### 原因分析句式
- "究其深层原因，主要有以下几点：一是……；二是……；三是……"
- "这一问题的形成有其历史背景，……"
- "从制度设计层面分析，……导致了……"

### 政策引用句式
- "根据《[文件全称]》[章条款]的规定，……"
- "按照国网公司……工作部署要求，……"
- "依据……原则，……应当……"
- "《[文件名]》明确指出，……"

### 建议表述句式
- "为此，建议……，具体可从以下几个维度着力"
- "在……方面，重点做好……工作"
- "结合[单位]实际，……可采取……方式推进"
- "近期可……；中期着力……；远期形成……"

### 评价性句式
- "……是……的重要支撑"
- "……对于……具有重要现实意义"
- "……有助于……，为……奠定基础"
- "……在一定程度上制约了……"

## 五、专业术语规范

### 国企改革类
- 统一使用"国有企业"（非"国企"，正文中可简称）
- "三项制度改革"：劳动、人事、分配制度改革（注意不要混淆为其他"三项"）
- "原集体企业"：国网体系内历史上由国有单位兴办的集体性质企业
- "分类改革"：按功能定位对国有企业实施差异化改革管理

### 国网体系类
- 省公司：国网[省份]电力有限公司
- 地市公司/地市供电公司：下属地市级单位（正式称谓应为"国网[地市]供电公司"）
- 平台公司：省级或地市级的原集体企业管理平台（如晋电科技）
- 主办单位：对原集体企业承担主要管理责任的上级单位
- 原主办单位：历史上的主办单位（改革后职责可能已调整）

### 绩效考核类
- 考核链条：考核主体 → 考核对象形成的纵向传导链
- 指标分解：上级指标向下级单位的拆解与落地
- 考核周期：季度/半年/年度考核的节奏安排
- 考核结果运用：将考核结论与薪酬、晋升、奖惩挂钩的机制

### 薪酬分配类
- 工资总额：企业年度工资发放总量（受主管单位核定约束）
- 岗效工资制：以岗位等级和绩效结果为依据的工资制度
- 绩效工资：与考核结果挂钩的可变动部分
- 分配系数：岗位之间薪酬拉开差距的比率设计

## 六、避免的表述习惯

| 不当用法 | 规范替换 |
|---------|---------|
| 我认为/笔者认为 | 研究认为 / 本报告认为 / 可以认为 |
| 搞好/做好（程度弱） | 推进/落实/深化/完善 |
| 非常重要/很关键 | 具有重要意义 / 是……的关键所在 |
| 问题很多/有很多问题 | 存在若干问题 / 主要问题集中在…… |
| 等等（随意列举） | 等 / 等方面（列举后收尾用单字"等"） |
| 首先……然后……最后 | 一是……；二是……；三是…… |
| 通过……来……（双"来"结构） | 通过……，实现…… / 以……推动…… |

## 七、表格使用规范

### 适合用表格呈现的内容
- 对比分析（两种模式、两类主体、改革前后的对比）
- 指标体系（维度-要素-权重-标准的多列展示）
- 职责边界划分（谁做什么、责任清单）
- 数据汇总（调研数据、样本统计）

### 表格标题格式
- 正文引出：在表格前一段末尾写"如表X所示"或"详见表X"
- 表格标题：紧接在表格上方，居中，格式为"表X　[标题内容]"
- 例："表1　省级原集体企业考核链条对比分析"

### 表格内容原则
- 表头文字简洁（2-6字）
- 单元格内容不超过50字；如需长文字，考虑改用正文
- 合并单元格时注意逻辑一致性

---

## 八、段落节奏控制

一份高质量的国网研究报告，段落节奏应体现"重要内容先说、充分论证、有始有终"：

### 开篇段（每章节）
点明本章议题的背景意义，1-3句话，不宜过长。

### 主体段
每段聚焦一个核心观点：**提出观点 → 说明理由/引用依据 → 举例/数据 → 小结或过渡**

### 过渡与收尾
- 章末可加1句小结："综上所述，……是制约……的核心因素，亟需从……入手加以解决。"
- 章节之间的逻辑过渡："在厘清……现状的基础上，进一步分析……"
