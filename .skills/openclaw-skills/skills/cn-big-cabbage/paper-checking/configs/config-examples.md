## 查重阈值参考

不同场景的推荐阈值设置：

| 场景 | 推荐阈值 | 说明 |
|------|---------|------|
| 本科论文查重 | 13-15 | 学校通常要求总重复率<30% |
| 硕士论文查重 | 12-14 | 学校通常要求总重复率<20% |
| 博士论文查重 | 10-12 | 学校通常要求总重复率<10% |
| 标书查重 | 10-13 | 串标检测更严格 |
| 项目申报书 | 11-14 | 避免重复申报 |

## 常用配置参数

### 内存配置
- 最低要求：1.5GB 可用内存
- 推荐配置：4GB+ 内存（处理大批量文件）

### 进程配置
- 默认：CPU逻辑核心数 - 2
- 可手动调整并发进程数

### 文件格式支持
- 支持：.doc, .docx, .pdf, .txt
- 推荐：.docx 格式（兼容性最佳）

## 批量查重配置示例

### 查重批次配置
```
批次名称：2024年春季毕业论文
待查目录：D:\2024春季\待查文件\
比对库：D:\比对库\全部
输出目录：D:\2024春季\查重结果\
阈值设置：14
关键词过滤：启用（keywords.txt）
```

### 防串标配置
```
批次名称：某项目投标文件查重
查重模式：横向查重
文件列表：batch_files.txt
输出目录：D:\串标检测结果\
阈值设置：12
```

## 中断恢复配置

系统自动保存进度到：
```
%TEMP%\paper_checking_progress.dat
```

重启后自动检测并提示恢复。
