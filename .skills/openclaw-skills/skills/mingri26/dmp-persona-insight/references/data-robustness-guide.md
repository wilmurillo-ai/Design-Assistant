### 问题背景
在分析DMP画像数据时，容易遇到以下问题导致**维度数据不完整**：

| 问题 | 原因 | 影响 |
|------|------|------|
| **地域维度缺失** | Sheet有热力图表头，真实表头在第3行 | 无法读取地域分布TGI数据 |
| **应用维度无TGI** | 应用偏好只有覆盖度，无TGI值 | 无法进行标准化分析 |
| **列名识别失败** | 通用的列名匹配逻辑不完善 | 特征提取结果为空 |
| **占比列多种格式** | 不同数据源的占比列名不统一 | 无法计算相对排名 |
| **特殊格式Sheet** | 某些Sheet有非标准的表头结构 | 数据读取失败 |

## 关键改进细节

### 1. 自动热力图表头处理

**问题场景：**
```
地域分布Sheet的实际结构：
Row 0: 居住地热力图 | https://datapan.getui.com/... | NaN | NaN | NaN
Row 1: 工作地热力图  | https://datapan.getui.com/... | NaN | NaN | NaN
Row 2: 一级标签      | 二级标签                      | 洞察人群 | 全库人群 | TGI  ← 真实表头
Row 3: 居住地区      | 一线城市                      | 0.0999 | 0.0984 | 1.016
```

**脚本处理：**
```python
# 识别策略：查找包含"一级标签"、"二级标签"、"TGI"等关键词的行
for idx, row in df_raw.iterrows():
    row_str = " ".join(str(x) for x in row.values).lower()
    if '一级标签' in row_str or 'tgi' in row_str:
        header_row_idx = idx  # 找到表头行
        break

# 从该行开始重新构建DataFrame
df = df_raw.iloc[header_row_idx:]
df.columns = df.iloc[0]  # 设置为表头
df = df.iloc[1:]  # 取数据行
```

### 2. 多维度列名识别

**支持的列名格式：**

**TGI列：**
- `tgi`, `TGI`, `index`, `指数`, `tgi值`

**标签列：**
- `二级标签` (优先)
- `标签`, `特征`, `名称`
- `一级标签` (降级)
- `三级标签`, `四级标签` (最后降级)

**占比列：**
- `占比` (优先)
- `coverage`, `比例`, `percentage`, `百分比`
- `ratio`, `覆盖`, `覆盖率`
- `洞察人群`, `洞察人群覆盖`

**全库数据列（用于TGI计算）：**
- `全库人群`, `total`, `全库`, `总库`

### 3. 智能的TGI计算

**场景：应用偏好维度无TGI，但有洞察人群和全库人群数据**

```python
# 自动检测并计算TGI
if not tgi_col and insight_col and total_col:
    df['_computed_tgi'] = df[insight_col] / df[total_col]
    tgi_col = '_computed_tgi'
    result['tgi_computed'] = True  # 标记为计算型
```

## 常见场景与解决

### 场景1：地域分布无法读取

**症状：**
```
分析维度: 地域分布... （无TGI数据）
```

**原因：** Sheet有热力图表头，真实表头在第3行

**v4.1处理：** ✅ 自动检测并跳过热力图表头

### 场景2：应用偏好无TGI值

**症状：** 
```
分析维度: 应用偏好... （无TGI数据）
```

**原因：** 应用偏好维度只有覆盖度数据

**处理方案：**
1. v4.1会标注为"无TGI数据"
2. 可选：使用覆盖度作为参考指标
3. 或：手动计算TGI（洞察人群/全库人群）

### 场景3：占比列名不统一

**v3.0问题：** 只能识别"占比"列名，其他格式无法识别

**v4.1解决：** 支持8种占比列名格式自动识别

```python
coverage_col = self._find_column_robust(df, [
    '占比', 'coverage', '比例', 'percentage', 
    '百分比', 'ratio', '覆盖', '覆盖率',
    '洞察人群', '洞察人群覆盖'
])
```
