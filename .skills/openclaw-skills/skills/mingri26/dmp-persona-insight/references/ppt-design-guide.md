## 🎨 设计规范

### 配色方案

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色调 | #0070C0（蓝） | 标题、关键指标、重点突出 |
| 辅助色 | #595959（深灰） | 副标题、普通文字 |
| 强调色 | #FF6B6B（红） | 数据对比、关键差异 |
| 文字色 | #404040（深灰） | 正文文本 |
| 背景色 | #F5F5F5（浅灰） | 内容块背景 |

### 排版规范

#### 字体大小

| 元素 | 大小 | 说明 |
|------|------|------|
| 页标题 | 32-36pt | 深蓝色，加粗 |
| 一级标题 | 24pt | 深灰色，加粗 |
| 二级标题 | 18pt | 深灰色 |
| 关键数据 | 20-24pt | 蓝色，加粗（突出显示） |
| 说明文字 | 12-14pt | 深灰色，常规 |
| 备注文字 | 10pt | 浅灰色，常规 |

#### 间距与布局

- **页边距**：上 0.5"，下 0.5"，左 0.5"，右 0.5"
- **内容块间距**：0.3" 纵向间距
- **行距**：1.5 倍行距（提高可读性）
- **段落间距**：段前 6pt，段后 12pt

### 信息设计

#### 内容块结构

每个内容块遵循以下模式：

```
[分类标题]
[关键数据（大）] + [TGI值]
[说明文字]
[补充数据列表]
```

**示例：**
```
运动健身兴趣
38.7%
TGI 1.76

用户群体中 38.7% 具有运动健身兴趣，
TGI 值 1.76 显示该特征显著高于平均水平。

细分特征：
- 减肥需求 13.5% (TGI 2.26)
- 养生保健 14.1% (TGI 1.65)
```

#### 数据展示优先级

1. **大数字优先**：百分比（占比）> TGI值 > 其他指标
2. **颜色突出**：关键数据用蓝色大字，普通数据用灰色小字
3. **关联展示**：占比和TGI成对出现，便于对标理解

## 🔄 修改指南

### 如何快速定制报告

1. **替换标题和数据**：修改各页文本框
2. **更新图表**：替换现有图表数据
3. **调整配色**：全局替换 RGB 值
4. **添加 Logo**：在标题页和末页插入公司 Logo

### 常见修改场景

| 场景 | 操作 |
|------|------|
| 更改主题色 | 查找替换 `#0070C0` → 新色值 |
| 添加脚注 | 在末页增加"公司名称""部门"等信息 |
| 删除某维度 | 移除对应页面，调整目录 |
| 突出特殊指标 | 改用强调色（红色）标记 |
