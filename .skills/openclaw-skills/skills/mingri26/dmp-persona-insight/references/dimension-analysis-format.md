# 维度分析三步法严格格式指南 v7.0

本文档定义了各维度分析必须遵循的严格格式规范，确保分析过程的规范性和可追踪性。

---

## 📋 总体要求

每个维度的分析都必须按照以下三个步骤进行，并严格按照规定的格式呈现：

### ✅ 第一步：筛选 TGI ≥ 1.0 的特征
### ✅ 第二步：按 TGI 降序排名（维度内排序）
### ✅ 第三步：取前 40% 的特征（动态阈值截取）

每一步都需要详细说明计算过程和逻辑，确保分析的透明度和可验证性。

---

## 第一步：筛选 TGI ≥ 1.0 的特征

### 目的
识别高于平均水平的有效特征，排除无效特征（TGI < 1.0）

### 输出内容

```markdown
### 第一步：筛选 TGI ≥ 1.0 的特征

原始特征列表（xxx条）：
  TGI排序：[x.xx, x.xx, x.xx, ...]

筛选规则：保留 TGI ≥ 1.0，排除 TGI < 1.0

筛选结果（前n条示例）：
  ✓ 特征名1    TGI: x.xx
  ✓ 特征名2    TGI: x.xx
  ✓ 特征名3    TGI: x.xx
  ... (共m条 TGI ≥ 1.0)
  
排除数据：TGI < 1.0 的特征（共n条）

统计：有效特征 m 条 / 总特征 xxx 条 = x.x%
```

### 关键数据
- 原始特征总数
- 有效特征数（TGI ≥ 1.0）
- 排除特征数（TGI < 1.0）
- 占比百分比

---

## 第二步：按 TGI 降序排名

### 目的
在维度内进行相对排序，建立特征优先级体系

### 排序规则
- **主排序**：按 TGI 从高到低
- **次排序**：相同 TGI 时，按占比从高到低

### 输出格式（表格形式）

```markdown
### 第二步：按 TGI 降序排名

维度内排名（按TGI从高到低）：

排名 | 特征名称        | TGI   | 占比    | 说明
-----|-----------------|-------|---------|--------
1    | 特征A          | 1.50  | 12%     | 最高TGI
2    | 特征B          | 1.42  | 8%      |
3    | 特征C          | 1.32  | 9%      |
4    | 特征D          | 1.24  | 7%      | (并列)
5    | 特征E          | 1.24  | 6%      | (并列)
...  | ...            | ...   | ...     | ...
```

### 说明列内容

标注以下信息：
- **最高/最低TGI** - 在该维度中的极值
- **并列特征** - 相同TGI值的特征
- **相对评价** - 最强/次强/普通等
- **突出特点** - 如果占比特别高或低

---

## 第三步：取前 40% 的特征（动态阈值）

### 目的
根据维度规模自动调整截取比例，平衡覆盖度与精准度

### 动态阈值公式

```
维度大小分类：

1. 小维度（<20条有效特征）
   公式：max(int(count * 0.4), 5)
   含义：至少保留5条，否则按40%计算
   
   例：8条有效特征
      计算：max(int(8 * 0.4), 5) = max(3, 5) = 5
      → 保留5条（因为3 < 5，所以取最小值5）

2. 中维度（20-50条有效特征）
   公式：int(count * 0.4)
   含义：直接取40%
   
   例：40条有效特征
      计算：int(40 * 0.4) = 16
      → 保留16条

3. 大维度（>50条有效特征）
   公式：min(int(count * 0.4), 20)
   含义：最多保留20条，否则按40%计算
   
   例：850条有效特征
      计算：min(int(850 * 0.4), 20) = min(340, 20) = 20
      → 保留20条（因为340 > 20，所以取上限20）
```

### 输出格式

```markdown
### 第三步：取前 40% 的特征（动态阈值）

维度大小：xxx条有效特征（属于小/中/大维度）

动态阈值计算：
  
  小/中/大维度公式：xxx(int(count * 0.4), y)
  计算：xxx * 0.4 = yyy → 取值 = zzz
  
核心特征数：n条

核心特征最终列表：
  1.  特征名1       TGI: 1.50  ✓ 入选
  2.  特征名2       TGI: 1.42  ✓ 入选
  3.  特征名3       TGI: 1.32  ✓ 入选
  ...
  n.  特征名n       TGI: x.xx  ✓ 入选
  
潜力特征（未入选但高相关）：
  n+1. 特征名x      TGI: 1.10  △ 补充
  ...
```

### 关键内容

1. **维度大小分类** - 明确该维度属于小/中/大
2. **阈值公式** - 使用的具体公式
3. **计算过程** - 完整的数学计算
4. **最终数量** - 核心特征的确定数量
5. **特征清单** - 分别列出核心和潜力特征
