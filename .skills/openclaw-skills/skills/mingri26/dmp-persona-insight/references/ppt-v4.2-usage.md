## 🚀 一句话总结

新版 PPT 脚本 v4.2 **自动生成** 文字大、排版紧凑、图形丰富的专业演示文稿，**无需手动调整**。

## 📊 核心改进

### ✅ 文字铺满整页
- **标题**: 40-44pt (清晰明确)
- **正文**: 12-14pt ↑ (从11pt 提升，更易阅读)
- **数据**: 13-16pt (突出强调)

### ✅ 适当加入图形元素
- 侧边装饰条 (3mm，增强视觉等级)
- 背景色块 (浅色，区分内容区域)
- 彩色标签 (重点数据突出)
- Emoji 图标 (快速导航)
- 彩色编号圆 (目录强化)

### ✅ 左右分栏排版
- **维度分析页**: 左列(特征) + 右列(数据)，充分利用空间
- **地域页**: 上下分区，优先布局 vs 潜力城市
- **策略页**: 三分区设计，清晰的逻辑流

### ✅ 重点数据突出
- 🔴 **Top 3 特征**: 加粗 + 红色
- 🔵 **最高 TGI**: 大号 + 加粗
- 🟠 **关键定位**: 高亮背景色块
- 🟢 **策略建议**: 左侧彩色条指示

## 📖 使用步骤

### Step 1: 准备 Markdown 报告
```bash
# 先用 analyze_persona.py 生成报告
python scripts/analyze_persona.py data.xlsx 人群名称 --output reports

# 输出: reports/人群名称_洞察分析报告_v3.md
```

### Step 2: 生成 PPT (v4.2)
```bash
# 一行命令，自动生成新版 PPT

# 输出: reports/人群名称_洞察分析报告_v3_标准格式_v4.2.pptx
```

### Step 3: 打开查看
```bash
# PowerPoint 直接打开
open reports/人群名称_洞察分析报告_v3_标准格式_v4.2.pptx
```

**就这样！✅ 无需手动调整。**

## 🔧 自定义选项

### 调整文字大小
编辑脚本中的字号：
```python
# 标题
self._add_title(slide, text, size=44)  # 默认44pt

# 正文
self._add_bullet_list(slide, items, size=12)  # 默认12pt

# 调整示例
self._add_title(slide, text, size=40)  # 改为40pt
self._add_bullet_list(slide, items, size=14)  # 改为14pt
```

### 调整颜色
编辑 COLORS 字典：
```python
COLORS = {
    'primary': RGBColor(0, 112, 192),  # 改为其他RGB值
    'accent': RGBColor(255, 107, 107),  # 改为其他RGB值
    ...
}
```

### 调整分栏比例
编辑维度分析页面方法：
```python
# 当前: 左4.8" + 右4.2"
# 改为: 左5" + 右4"
slide.shapes.add_textbox(Inches(0.5), ..., Inches(5), ...)
slide.shapes.add_textbox(Inches(5.6), ..., Inches(4), ...)
```

## ⚠️ 常见问题

### Q: PPT 生成后文字还是太小?
A: 可能是你的 PowerPoint 软件缩放设置。试试：
- 打开 PPT → View → Zoom → 100%

### Q: 左右分栏内容不对齐?
A: 这是正常的 (因为内容长度不同)。可选择：
- 编辑每页内容长度
- 调整行距 (frame.paragraphs[i].space_after)

### Q: 能否添加真实图表?
A: v4.2 暂不支持，v4.3 计划添加柱状图、雷达图。

### Q: 生成速度慢?
A: 正常。维度越多越慢。通常 10 维以内 < 3秒。

### Q: 如何批量生成多个人群的 PPT?
A: 可写简单脚本：
```bash
for file in reports/*_洞察分析报告_v3.md; do
    python scripts/generate_ppt_report.py "$file"
done
```
