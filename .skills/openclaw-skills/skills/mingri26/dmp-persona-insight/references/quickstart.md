# 快速开始指南

## 5分钟上手

### 第一步：准备数据

**文件格式：** Excel (.xlsx) 或 CSV

**数据结构：** 多维度标签数据
```
工作表示例：
├── 基础标签（年龄、性别、婚育等）
├── 地域分布（省份、城市）
├── 兴趣偏好
├── 应用偏好
└── 其他维度...
```

**必需列：**
- 标签名称
- TGI值
- 占比（或百分比）

### 第二步：上传文件

向 AI 助手发送：
```
我要分析这份用户画像数据
[上传文件]
```

### 第三步：确认输出格式

选择你需要的报告格式：
- ✅ **Markdown 报告**（推荐，快速查看）
- ✅ **PPT 演示文稿**（管理层汇报）
- ✅ **Excel 表格**（数据汇总）

### 第四步：获取分析结果

系统会输出：

**1. 用户画像总结**
```
一句话核心特征 + 核心定位标签
```

**2. 地域深耕方向**
```
优先省份 → 潜力城市 → 减少投入区域
```

**3. 各维度详细分析**
```
每个维度的核心特征 + TGI 对标 + 占比分布
```

**4. 综合策略建议**
```
产品定位 + 渠道布局 + 内容营销 + 用户运营
```

**5. 深度分析方向**
```
推荐单维度深挖方向 + 交叉分析组合 + 优先级排序
```

## 常见问题

### Q: 数据维度最少需要多少个？

A: **建议 7-10 个维度**，最少 3 个。维度越多，画像越立体。

### Q: TGI 值没有，怎么办？

A: 如果你有占比数据，系统可以自动计算 TGI。只需提供标签和占比。

### Q: 报告会包含敏感信息吗？

A: 系统只分析数据中的特征和趋势，不涉及个人隐私信息。

### Q: 可以修改报告吗？

A: 可以。系统输出 Markdown 或 Excel 格式，你可以自由编辑。
