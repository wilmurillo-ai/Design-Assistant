# 律师法律检索与案例查找方法论

## 📚 律师找案例的专业习惯

### 一、案件分析流程

#### 1. 事实梳理（Fact Pattern Analysis）
律师在查找案例前，会先完整梳理案件事实：

- **当事人信息**：自然人/法人、年龄/注册资本、住所地
- **法律关系**：合同关系、侵权关系、婚姻关系等
- **时间线**：关键事件发生的时间顺序
- **争议焦点**：双方争执的核心问题
- **诉讼请求**：希望法院支持的具体主张
- **证据情况**：现有证据及证明力

#### 2. 法律关系定性
确定案件属于哪个法律领域：

```
民事案件 → 合同/侵权/婚姻/继承/物权/劳动争议
刑事案件 → 罪名确定（盗窃/诈骗/伤害等）
行政案件 → 行政行为类型（处罚/许可/强制等）
```

#### 3. 请求权基础分析
寻找支持诉讼请求的法律依据：

- **主要请求权基础**：直接支持诉求的法条
- **辅助请求权基础**：计算标准、程序规定
- **抗辩权基础**：预判对方可能的抗辩理由

### 二、案例检索策略

#### 1. 检索平台选择

**官方平台**：
- 中国裁判文书网（https://wenshu.court.gov.cn）- 最权威
- 中国执行信息公开网（http://zxgk.court.gov.cn）
- 各省市高级人民法院官网

**商业数据库**：
- 北大法宝（https://www.pkulaw.com）- 学术性强
- 威科先行（https://law.wkinfo.com.cn）- 外资所常用
- 法信（https://www.faxin.cn）- 最高法官方
- 律商联讯（https://www.lexiscn.com）

#### 2. 检索关键词设计

**层级化检索**：

```
第一层：案由
  - "借款合同纠纷"
  - "劳动争议"
  - "离婚纠纷"

第二层：争议焦点
  - "诉讼时效"
  - "举证责任"
  - "违约金调整"
  - "违法解除"

第三层：关键事实
  - "借条" + "转账凭证"
  - "业绩不达标" + "培训"
  - "分居满两年"

第四层：法律适用
  - "民法典第 675 条"
  - "劳动合同法第 40 条"
```

**关键词组合技巧**：
- 使用"AND"、"OR"、"NOT"逻辑运算符
- 使用通配符（*）扩大检索范围
- 使用引号精确匹配短语

#### 3. 筛选条件设置

**优先级排序**：

| 优先级 | 筛选条件 | 理由 |
|--------|----------|------|
| ⭐⭐⭐⭐⭐ | 法院层级 | 最高法 > 高院 > 中院 > 基层 |
| ⭐⭐⭐⭐ | 裁判时间 | 近 3 年优先，法律未修改 |
| ⭐⭐⭐⭐ | 地域 | 同一省份/同一法院优先 |
| ⭐⭐⭐ | 案由 | 完全相同 > 相似 |
| ⭐⭐ | 争议焦点 | 核心争议点一致 |
| ⭐ | 标的金额 | 相近金额参考性更强 |

#### 4. 案例阅读要点

阅读裁判文书时，重点关注：

**① 基本信息**：
- 案号：(2023) 京 01 民终 1234 号
- 法院：北京市第一中级人民法院
- 案由：借款合同纠纷
- 裁判日期：2023 年 5 月 10 日

**② 当事人信息**：
- 原告/上诉人：身份、诉讼地位
- 被告/被上诉人：身份、抗辩理由

**③ 案情简介**：
- 原告诉称
- 被告辩称
- 法院查明的事实

**④ 裁判要旨**（最重要）：
- 法院认为部分
- 法律适用理由
- 裁判规则提炼

**⑤ 裁判结果**：
- 支持/驳回
- 金额计算方式
- 诉讼费承担

### 三、类案比对分析

#### 1. 相似性分析维度

```
事实相似性：
  ✓ 法律关系相同
  ✓ 关键事实相似
  ✓ 争议焦点一致
  ✓ 证据类型相近

法律适用相似性：
  ✓ 适用同一法条
  ✓ 解释方法一致
  ✓ 裁判思路相同

程序相似性：
  ✓ 审级相同
  ✓ 地域相同
  ✓ 时间相近
```

#### 2. 区分不利案例

如发现对己方不利的案例，分析：

- **事实区别**：找出本案与不利案例的事实差异
- **法律区别**：法律适用是否不同
- **程序区别**：是否存在程序问题
- **时间效力**：法律法规是否已修改

#### 3. 提炼裁判规则

从案例中提炼可复用的裁判规则：

```
格式：
"在 [某类案件] 中，法院认为 [核心观点]，
因此 [裁判结果]，适用 [法律条文]。"

示例：
"在借款合同纠纷中，法院认为仅有借条而无
转账凭证的，不能证明借款事实成立，因此
驳回原告诉讼请求，适用《民法典》第 667 条。"
```

### 五、常见纠纷类型的检索要点

#### 1. 借款合同纠纷

**检索重点**：
- 借贷关系成立的证据标准
- 现金交付的认定规则
- 利息计算标准（LPR）
- 诉讼时效中断事由

**关键词**：
```
"民间借贷" + "借条" + "转账凭证"
"现金交付" + "举证责任"
"诉讼时效" + "催收"
```

#### 2. 劳动合同纠纷

**检索重点**：
- 违法解除的认定标准
- 经济补偿金/赔偿金计算
- 加班费举证责任
- 竞业限制效力

**关键词**：
```
"违法解除" + "劳动合同法第 40 条"
"赔偿金" + "2N"
"加班费" + "举证责任"
```

#### 3. 离婚纠纷

**检索重点**：
- 感情破裂的认定标准
- 子女抚养权归属原则
- 夫妻共同财产分割
- 家暴证据认定

**关键词**：
```
"感情破裂" + "分居满两年"
"抚养权" + "子女最佳利益"
"共同财产" + "隐匿转移"
```

#### 4. 买卖合同纠纷

**检索重点**：
- 货物交付的认定
- 质量异议期限
- 违约金调整标准
- 表见代理认定

**关键词**：
```
"货款" + "送货单" + "对账单"
"质量异议" + "合理期限"
"违约金" + "过高调整"
```

### 六、检索技巧与注意事项

#### 1. 提高检索效率的技巧

- **由宽到窄**：先宽泛检索，再逐步缩小范围
- **由新到旧**：优先查看最新案例
- **由上到下**：优先查看上级法院案例
- **正向 + 反向**：既检索支持己方的，也检索支持对方的

#### 2. 避免常见错误

❌ **错误做法**：
- 只看裁判结果，不看裁判理由
- 忽视案例的地域性和时效性
- 盲目相信单一案例
- 不核实法律法规是否有效

✅ **正确做法**：
- 全面阅读裁判文书
- 注意案例的参考效力
- 多案例交叉验证
- 核实法律依据的现行有效性

#### 3. 案例的参考效力

```
指导性案例（最高法发布）> 公报案例 > 典型案例
        ↓
    各级法院生效裁判
        ↓
    同一法院同类案件
```

**注意**：中国不是判例法国家，案例仅具有参考价值，不具有约束力（指导性案例除外）。

### 七、法律更新追踪

#### 1. 法律法规更新

- 关注全国人大、国务院、最高法、最高检官网
- 订阅法律数据库的更新提醒
- 关注专业法律公众号

#### 2. 司法解释更新

- 最高法定期发布司法解释
- 注意新旧法的衔接适用
- 关注司法解释的理解与适用文章

#### 3. 裁判观点变化

- 定期检索同类案件
- 关注裁判尺度变化
- 参加专业培训和研讨

## 📋 总结

律师查找案例的核心方法论：

1. **事实梳理** → 明确案件争点
2. **法律定性** → 确定检索方向
3. **关键词设计** → 精准检索
4. **筛选排序** → 优先权威案例
5. **阅读提炼** → 提取裁判要旨
6. **类案比对** → 分析相似与区别
7. **形成意见** → 支持诉讼主张

**关键原则**：
- 全面性：不遗漏重要案例
- 准确性：正确理解裁判规则
- 时效性：优先最新案例
- 权威性：优先上级法院案例
