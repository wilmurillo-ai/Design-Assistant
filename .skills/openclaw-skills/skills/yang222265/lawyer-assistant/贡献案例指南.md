## 📝 如何提交案例

### 方法一：对话提交

在对话中输入：

```
贡献案例
纠纷类型：劳动合同纠纷
案号：(2024) 京 01 民终 1234 号
法院：北京市第一中级人民法院
日期：2024-01-15
案情描述：公司违法解除劳动合同，员工工作 3 年...
争议焦点：违法解除赔偿金计算
裁判结果：公司支付赔偿金 18 万元
裁判要旨：公司未提供培训或调岗证据，构成违法解除
证据清单：劳动合同、解除通知、工资流水
诉讼金额：18 万元
备注：典型案例，值得参考
```

### 方法二：简化提交

```
贡献案例
纠纷类型：消费纠纷
案情描述：网购买到假货，花 5000 元
裁判结果：退一赔三，共 2 万元
```

### 方法三：命令行提交

```bash
python3 lawyer_assistant.py "贡献案例 <案例信息>"
```

## ⚖️ 审核规则

### 审核标准

**通过条件**：
- ✅ 案例真实
- ✅ 信息完整
- ✅ 有参考价值
- ✅ 不涉密

**拒绝条件**：
- ❌ 虚假信息
- ❌ 信息不全
- ❌ 涉及隐私
- ❌ 重复提交

### 审核时间

- 普通用户：1-3 个工作日
- 贡献者：24 小时内
- 专家/达人：优先审核

## 🎯 优秀案例标准

### 什么样的案例受欢迎？

1. **典型性**: 常见纠纷类型
2. **参考价值**: 裁判要旨清晰
3. **完整性**: 信息详细
4. **时效性**: 近 3 年案例
5. **指导意义**: 对类似案件有参考

### 示例：优秀案例

```
纠纷类型：劳动合同纠纷
案号：(2024) 京 02 民终 2345 号
法院：北京市第二中级人民法院
案情描述：公司以业绩不达标为由辞退员工，
          但未提供培训或调岗证据。
          员工工作 5 年，月工资 1.8 万元。
争议焦点：违法解除劳动合同赔偿金计算
裁判结果：公司支付赔偿金 18 万元（5×1.8 万×2）
裁判要旨：公司以不能胜任工作为由解除合同，
          需经过培训或调岗程序。未经过该程序
          直接解除，构成违法解除。
证据清单：劳动合同、解除通知、工资流水、
          业绩考核表
备注：典型案例，明确了培训调岗程序的必要性
```

## ❓ 常见问题

### Q1: 我可以提交别人的案例吗？

A: 可以，但需确保信息来源合法，不侵犯隐私。

### Q2: 提交后多久能审核？

A: 一般 1-3 个工作日，贡献者 24 小时内。

### Q3: 审核不通过怎么办？

A: 根据审核意见修改后重新提交。

### Q4: 积分会过期吗？

A: 积分长期有效，不会过期。

### Q5: 可以删除已提交的案例吗？

A: 已发布的案例不能删除，但可以申请隐藏。
