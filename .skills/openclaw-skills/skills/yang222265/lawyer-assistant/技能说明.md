# 律师助手技能完整说明

## 📌 技能概述

**律师助手**是一个基于法律专业知识构建的 AI 辅助工具，模拟专业律师的案件分析和法律检索流程，为用户提供结构化的案件分析报告。

---

## 🎯 律师找案例的专业习惯总结

根据法律实务经验，律师在查找案例时遵循以下专业流程：

### 1️⃣ 案件信息结构化

律师会先梳理案件的完整信息：

| 信息要素 | 内容说明 | 示例 |
|---------|---------|------|
| **当事人** | 姓名/名称、身份 | 张三（自然人）、A 公司（法人） |
| **对手** | 对方当事人的完整信息 | 李四、XX 科技公司 |
| **纠纷类型** | 法律关系类型 | 借款合同、劳动合同、离婚 |
| **起因** | 案件发生的事实经过 | 借款 10 万未还、违法解除劳动合同 |
| **诉求** | 希望达到的法律结果 | 归还本金、赔偿金、离婚 |
| **证据** | 现有证据清单 | 借条、合同、转账记录 |

### 2️⃣ 法律关系定性

根据纠纷类型确定适用的法律领域：

```
借款纠纷 → 民法典（合同编）
劳动纠纷 → 劳动合同法 + 劳动争议调解仲裁法
离婚纠纷 → 民法典（婚姻家庭编）
买卖纠纷 → 民法典（合同编）
侵权纠纷 → 民法典（侵权责任编）
```

### 3️⃣ 案例检索策略

#### 检索平台优先级：
1. **中国裁判文书网**（官方、最权威）
2. **北大法宝/威科先行**（商业数据库、功能强大）
3. **法信**（最高法官方平台）

#### 检索关键词设计：
```
第一层：案由（借款合同纠纷）
   ↓
第二层：争议焦点（诉讼时效、举证责任）
   ↓
第三层：关键事实（借条、转账凭证）
   ↓
第四层：法律适用（民法典第 675 条）
```

#### 筛选条件优先级：
1. ⭐⭐⭐⭐⭐ **法院层级**：最高法 > 高院 > 中院
2. ⭐⭐⭐⭐ **裁判时间**：近 3 年优先
3. ⭐⭐⭐⭐ **地域**：同一省份优先
4. ⭐⭐⭐ **案由**：完全相同 > 相似

### 4️⃣ 案例阅读要点

律师阅读裁判文书时重点关注：

- **裁判要旨**：法院的核心观点
- **争议焦点**：双方争执的核心问题
- **证据认定**：法院采信了哪些证据
- **法律适用**：引用了哪些法条
- **裁判结果**：支持/驳回及理由

### 5️⃣ 类案比对分析

将检索到的案例与本案进行比对：

**相似性分析**：
- ✓ 法律关系是否相同
- ✓ 关键事实是否相似
- ✓ 争议焦点是否一致
- ✓ 证据类型是否相近

**区分不利案例**：
- 找出本案与不利案例的事实差异
- 分析法律适用是否不同
- 注意法律法规是否已修改

## 🤖 律师助手技能功能

### 核心功能

#### 1. 案件信息自动提取
从用户输入中自动识别和提取：
- 当事人信息
- 对手信息
- 纠纷类型
- 案件起因
- 诉讼诉求
- 证据情况

#### 2. 法律分析
- 确定法律关系类型
- 匹配适用律师
- 列出核心法条
- 提示诉讼时效

#### 3. 案例检索建议
- 推荐检索平台
- 提供检索关键词
- 建议筛选条件

#### 4. 证据收集指导
根据不同纠纷类型提供：
- 通用证据清单
- 特有证据清单
- 证据收集建议

#### 5. 风险提示
- 举证不能风险
- 诉讼时效风险
- 执行风险
- 诉讼成本分析

#### 6. 策略建议
- 诉前准备建议
- 程序选择（诉讼/仲裁/调解）
- 诉讼请求设计

## 📖 使用示例

### 示例 1：借款纠纷

**用户输入**：
```
请帮我分析这个案件：
- 当事人：张三
- 对手：李四
- 纠纷类型：借款合同纠纷
- 起因：2023 年 1 月李四向张三借款 10 万元，约定 2023 年 12 月归还，至今未还
- 诉求：要求归还本金及利息
- 证据：借条、转账记录
```

**技能输出**：
```
📋 案件分析报告

【案件概要】
当事人：张三
对手方：李四
纠纷类型：借款合同
案件起因：2023 年 1 月李四向张三借款 10 万元...
诉讼诉求：要求归还本金及利息
现有证据：借条、转账记录

【法律分析】
法律关系：民间借贷法律关系
适用律师：民法典
核心法条：第 667 条，第 675 条，第 676 条
诉讼时效：3 年（民法典第 188 条）

【类似案例检索建议】
推荐检索平台：
  1. 中国裁判文书网
  2. 北大法宝
  3. 威科先行

检索关键词：借款合同纠纷、借条、转账凭证...

【证据收集建议】
借款纠纷特有证据：
  ✓ 借条/借款合同
  ✓ 转账凭证
  ✓ 催收记录

【风险提示】
⚠️ 诉讼时效风险：超过 3 年可能丧失胜诉权

【策略建议】
1️⃣ 诉前准备：完善证据链、发送律师函
2️⃣ 程序选择：优先协商，协商不成再诉讼
3️⃣ 诉讼请求：本金 + 利息 + 诉讼费
```

### 示例 2：劳动纠纷

**用户输入**：
```
当事人：王某
对手：XX 科技公司
纠纷：劳动合同纠纷
起因：公司以业绩不达标为由辞退，但未提供培训或调岗
```

**技能输出**：
```
【法律分析】
法律关系：劳动合同法律关系
适用律师：劳动合同法
核心法条：第 39 条，第 40 条，第 47 条，第 87 条

【证据收集建议】
劳动纠纷特有证据：
  ✓ 劳动合同
  ✓ 工资银行流水
  ✓ 社保缴纳记录
  ✓ 解除/终止劳动合同通知书

【策略建议】
- 劳动纠纷需先仲裁后诉讼
- 赔偿金计算：2N（工作年限×月工资×2）
```

## ⚠️ 重要声明

### 技能定位
- 本技能为**辅助工具**，不能替代执业律师
- 适用于**初步案件分析**和**法律检索指导**
- 复杂案件或重大利益事项**务必咨询专业律师**

### 使用限制
1. **不构成法律意见**：分析报告仅供参考
2. **法律法规更新**：具体适用以最新规定为准
3. **个案差异**：具体案件需具体分析
4. **地域差异**：不同地区司法实践可能不同

### 适用场景
✅ **适合使用**：
- 初步了解案件性质
- 学习法律知识
- 准备咨询律师前的自我梳理
- 简单案件的参考分析

❌ **不适合使用**：
- 重大财产纠纷
- 刑事案件
- 复杂商事纠纷
- 涉及人身关系的案件
