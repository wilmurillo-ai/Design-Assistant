## 🎉 重大更新

---

## ✨ 核心功能

### 1️⃣ 评价回复功能

**回复类型**：
- ✅ 系统回复 - 自动回复
- ✅ 律师回复 - 被评价律师回复
- ✅ 管理员回复 - 运营人员回复

**回复场景**：
```
用户评价：3 星 赔偿计算不太准确

系统回复：感谢您的反馈！我们已记录您的问题，
         会尽快优化赔偿计算功能。

律师回复：您好，我是张律师。关于您提到的问题，
         建议提供更详细的案情，我可以帮您重新计算。
```

**功能特点**：
- 支持多级回复
- 显示回复者身份
- 记录回复时间
- 通知用户（可选）

---

### 2️⃣ 评价标签系统

**标签分类**：

**正面标签**（8 个）：
- 专业准确、分析详细、实用建议、响应快速
- 法条清晰、案例相关、律师匹配、界面友好

**负面标签**（8 个）：
- 分析不准、建议无用、响应慢、法条错误
- 案例不相关、律师不匹配、计算错误、内容简单

**功能标签**（7 个）：
- 赔偿计算、胜诉率、律师推荐、法条详解
- 案例推荐、用户贡献、文书模板

**标签生成**：
- **自动生成**：根据评分和文字评价自动打标签
- **手动添加**：用户/管理员可手动添加标签
- **智能推荐**：根据评价内容推荐标签

**示例**：
```
用户评价：5 星 法条详解很专业，案例很相关

自动标签：['专业准确', '法条清晰', '案例相关', '分析详细']
```

---

### 3️⃣ 满意度趋势图

**多维度支持**：
- 📊 **平台满意度** - 整体服务满意度趋势
- ⚖️ **律师满意度** - 律师推荐满意度趋势
- 🔄 **支持切换** - 可查看不同维度

**图表特点**：
- ASCII 艺术图表（终端友好）
- 按月统计趋势
- 显示最高/最低分
- 显示平均满意度
- 显示评价数量

## 📊 使用方式

### 方式 1：评价并添加标签

```bash
# 提交评价（自动标签）
评价 5 星 法条详解很专业，案例很相关

# 输出：
# ✅ 感谢您的评价！
# 自动标签：['专业准确', '法条清晰', '案例相关']
```

### 方式 2：查看满意度趋势

```bash
# 查看平台满意度趋势
查看满意度趋势

# 查看律师满意度趋势
查看满意度趋势 律师

# 输出 ASCII 趋势图
```

### 方式 3：查看评价标签

```bash
查看评价标签

# 输出：
# 总评价数：3
# 已标签评价：1
# TOP 标签：
#   专业准确：1 次
#   案例相关：1 次
#   法条清晰：1 次
```

### 方式 4：回复评价

```python
# 代码方式（管理员）
rs.add_reply(rating_id, {
    'reply_type': 'system',
    'content': '感谢您的反馈！',
    'reply_by': '系统管理员'
})
```

## 🔧 技术实现

### 评价回复

**数据结构**：
```json
{
  "rating_id": "R_xxxxx",
  "replies": [
    {
      "reply_id": "RP_xxxxx",
      "reply_type": "system",
      "content": "感谢...",
      "reply_by": "系统管理员",
      "reply_time": "2026-03-21T00:49:26"
    }
  ]
}
```

### 标签系统

**自动标签算法**：
1. 根据平均评分判断正面/负面
2. 从文字评价提取关键词
3. 匹配预定义标签
4. 去重后保存

### 趋势图

**数据聚合**：
- 按月聚合评价数据
- 分维度计算平均分
- 支持多维度切换

**图表渲染**：
- ASCII 字符绘制
- 自适应高度
- 显示关键统计
