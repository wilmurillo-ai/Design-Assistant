### 待优化问题

**问题 1：混合式输入解析不够准确**

测试输入：
```
公司违法辞退我（工作 5 年，月工资 2 万），
有劳动合同和解除通知，能赔多少钱？胜算大吗？
```

当前输出：
```
【案件概要】
当事人：
对手方：
纠纷类型：
...
```

**问题**：信息提取不完整

**原因**：解析算法需要优化，支持括号内信息提取

**改进方案**：
```python
def parse_mixed_input(text):
    """优化混合式输入解析"""
    # 提取括号内的补充信息
    bracket_info = re.findall(r'（(.*?)）', text)
    
    # 提取关键信息
    case_info = {
        '纠纷类型': detect_dispute_type(text),
        '起因': extract_cause(text),
        '诉求': extract_claims(text),
        '补充信息': bracket_info
    }
    
    # 从补充信息提取工作年限、工资等
    for info in bracket_info:
        if '工作' in info and '年' in info:
            case_info['工作年限'] = extract_years(info)
        if '工资' in info:
            case_info['月工资'] = extract_amount(info)
    
    return case_info
```
