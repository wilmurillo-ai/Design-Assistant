## 🚀 快速开始

### 方法一：在对话中直接使用（推荐）

在支持该技能的平台上，直接输入案件信息即可：

```
请帮我分析这个案件：
- 当事人：张三
- 对手：李四
- 纠纷类型：借款合同纠纷
- 起因：2023 年 1 月李四向张三借款 10 万元，约定 2023 年 12 月归还，至今未还
- 诉求：要求归还本金及利息
- 证据：借条、转账记录
```

### 方法二：命令行使用

```bash
python3 ~/.openclaw/skills/lawyer-assistant/lawyer_assistant.py "当事人：张三 对手：李四 纠纷：借款 起因：借款 10 万未还 诉求：归还"
```

### 方法三：运行测试案例

```bash
cd ~/.openclaw/skills/lawyer-assistant/test_cases
bash run_tests.sh
```

## 📋 输出内容说明

技能会生成结构化的案件分析报告，包含以下部分：

### 1️⃣ 案件概要
- 当事人、对手方、纠纷类型
- 案件起因、诉讼诉求、现有证据

### 2️⃣ 法律分析
- 法律关系定性
- 适用律师
- 核心法条
- 诉讼时效

### 3️⃣ 类似案例检索建议
- 推荐检索平台（裁判文书网、北大法宝等）
- 检索关键词建议
- 筛选条件建议

### 4️⃣ 证据收集建议
- 通用证据清单
- 特定纠纷类型的特有证据

### 5️⃣ 风险提示
- 举证不能风险
- 诉讼时效风险
- 执行风险
- 诉讼成本

### 6️⃣ 策略建议
- 诉前准备
- 程序选择（协商/调解/诉讼）
- 诉讼请求设计

## 💡 使用技巧

### 提供完整信息
尽可能提供完整的案件信息，包括：
- ✅ 当事人和对手的详细信息
- ✅ 纠纷的具体类型
- ✅ 案件起因的完整经过
- ✅ 明确的诉讼请求
- ✅ 现有证据清单

### 信息格式灵活
支持多种输入格式：

**格式一（推荐）：**
```
- 当事人：张三
- 对手：李四
- 纠纷类型：借款合同
- 起因：...
- 诉求：...
- 证据：...
```

**格式二（单行）：**
```
当事人：张三 对手：李四 纠纷：借款 起因：... 诉求：... 证据：...
```

**格式三（自然语言）：**
```
我是张三，李四欠我 10 万元借款不还，有借条和转账记录，想要起诉要回本金和利息
```

## ⚠️ 重要声明

### 技能定位
- 本技能为**辅助工具**，不能替代执业律师
- 适用于**初步案件分析**和**法律检索指导**
- 复杂案件或重大利益事项**务必咨询专业律师**

### 使用限制
1. **不构成法律意见**：分析报告仅供参考
2. **法律法规更新**：具体适用以最新规定为准
3. **个案差异**：具体案件需具体分析
4. **地域差异**：不同地区司法实践可能不同

### 适用场景
✅ **适合使用**：
- 初步了解案件性质
- 学习法律知识
- 准备咨询律师前的自我梳理
- 简单案件的参考分析

❌ **不适合使用**：
- 重大财产纠纷
- 刑事案件
- 复杂商事纠纷
- 涉及人身关系的案件

## 🔧 故障排除

### 问题 1：技能无法识别
**解决**：确保技能已正确安装
```bash
ls -la ~/.openclaw/skills/lawyer-assistant/
```

**解决**：检查 Python 环境
```bash

```
