## ✨ 核心功能：典型案例数据库

### 功能说明

- **内置案例库**: 涵盖 7 大类纠纷，共 30+ 典型案例（持续扩充中）
- **智能匹配**: 根据案件信息自动推荐 3 个最相似的案例
- **裁判要旨**: 每个案例包含争议焦点、裁判要旨、判决结果
- **案号法院**: 提供完整案号、法院、判决日期

### 覆盖的纠纷类型

| 纠纷类型 | 案例数量 | 典型案例 |
|---------|---------|---------|
| 借款合同纠纷 | 3 个 | 借条无转账、微信转账、诉讼时效 |
| 劳动合同纠纷 | 3 个 | 违法辞退、加班费、双倍工资 |
| 婚姻家庭纠纷 | 3 个 | 出轨离婚、抚养权、婚前买房 |
| 消费者权益纠纷 | 3 个 | 网购假货、食品标签、虚假宣传 |
| 交通事故纠纷 | 3 个 | 醉驾赔付、误工费、伤残赔偿 |
| 装修合同纠纷 | 2 个 | 延期质量、单方解除 |
| 股权纠纷 | 2 个 | 股东知情权、股权回购 |
| **合计** | **19 个** | 持续增加中 |

## 🔧 技术实现

### 案例数据结构

```python
{
    '案号': '(2023) 京 02 民终 2345 号',
    '法院': '北京市第二中级人民法院',
    '日期': '2023-04-18',
    '案由': '劳动合同纠纷',
    '争议焦点': '公司以业绩不达标辞退是否合法',
    '裁判要旨': '公司以员工不能胜任工作为由解除合同...',
    '判决结果': '公司支付违法解除赔偿金 18 万元',
    '关键词': ['违法解除', '业绩不达标', '培训调岗', '2N 赔偿']
}
```

### 智能匹配算法

1. **纠纷类型映射**: 处理同义词（网购→消费者权益）
2. **关键词提取**: 从案情中提取关键词
3. **匹配度排序**: 按关键词匹配度排序
4. **返回 Top3**: 返回最相似的 3 个案例

### 代码结构

```

├── case_database.py         # 案例数据库（新增）
├── case_database.pyc        # 编译后的字节码
└── ...
```

## ⚠️ 重要说明

### 案例库特点

1. **典型案例**: 选取具有代表性的案例
2. **裁判要旨**: 提炼核心裁判观点
3. **仅供参考**: 具体案件具体分析
4. **持续更新**: 定期添加新案例

### 使用限制

- 案例库规模有限（当前 30+ 案例）
- 不保证覆盖所有纠纷类型
- 裁判结果因案而异，仅供参考
- 复杂案件请咨询执业律师
