**现在**：说"网购"、"假货"、"欺诈"都能自动识别！

### 支持的同义词

| 日常用语 | 识别为 |
|---------|--------|
| 网购、电商、网络购物 | 消费者权益 |
| 假货、假冒伪劣、欺诈 | 消费者权益 |
| 离婚、出轨、婚外情 | 婚姻家庭 |
| 借款、借贷、欠款 | 借款合同 |
| 辞退、开除、裁员 | 劳动合同 |
| 车祸、撞人、交通事故 | 交通事故 |
| 分红、股东、股权 | 股权纠纷 |

## ✨ 新功能 3：赔偿计算（最实用！）

**以前**：只告诉你能起诉，没说能赔多少  
**现在**：直接给你计算公式和说明！

### 📊 支持的纠纷类型

#### 1️⃣ 劳动纠纷赔偿计算

**适用场景**：被违法辞退、拖欠工资、加班费等

```
💰 赔偿计算：
  • 违法解除赔偿金
    公式：工作年限 × 月工资 × 2
    说明：工作满 1 年支付 2 个月工资

  • 经济补偿金
    公式：工作年限 × 月工资
    说明：N 或 N+1

  • 未休年假工资
    公式：月工资 ÷ 21.75 × 未休天数 × 300%

  • 加班费
    公式：月工资 ÷ 21.75 ÷ 8 × 加班小时 × 倍数
    说明：工作日 1.5 倍，周末 2 倍，法定节假日 3 倍
```

**举例**：
- 工作 3 年，月工资 15000 元，违法辞退
- 违法解除赔偿金 = 3 × 15000 × 2 = **90000 元**

---

#### 2️⃣ 消费纠纷赔偿计算

**适用场景**：买到假货、商家欺诈、网购纠纷等

```
💰 赔偿计算：
  • 三倍赔偿
    公式：商品价款 × 3
    说明：不足 500 元按 500 元计算

  • 退货退款
    公式：商品价款
    说明：可要求退货并返还全部价款

  • 其他损失
    公式：实际损失金额
    说明：交通费、误工费、鉴定费等
```

**举例**：
- 花 5000 元买到假货，商家欺诈
- 三倍赔偿 = 5000 × 3 = **15000 元**
- 总计可主张：15000（赔偿）+ 5000（退款）= **20000 元**

---

#### 3️⃣ 离婚纠纷赔偿计算

**适用场景**：离婚、子女抚养、财产分割等

```
💰 赔偿计算：
  • 共同财产分割
    公式：共同财产总额 ÷ 2
    说明：原则上均分，照顾子女和女方权益

  • 损害赔偿
    公式：根据实际情况确定
    说明：因出轨、家暴等过错可主张

  • 抚养费
    公式：月收入的 20%-30%
    说明：抚养一个子女的比例
```

**举例**：
- 夫妻共同财产 200 万元
- 共同财产分割 = 200 万 ÷ 2 = **100 万元**
- 如男方出轨，女方还可主张损害赔偿

---

#### 4️⃣ 交通事故赔偿计算

**适用场景**：车祸受伤、人身损害等

```
💰 赔偿计算：
  • 医疗费 = 实际医疗支出
  • 误工费 = 误工天数 × 日均收入
  • 护理费 = 护理天数 × 护理人员日收入
  • 残疾赔偿金 = 城镇居民人均可支配收入 × 20 年 × 伤残系数
```

---

#### 5️⃣ 借款纠纷赔偿计算

**适用场景**：借钱不还、欠款拖欠等

```
💰 赔偿计算：
  • 本金 = 实际借款金额
  • 利息 = 本金 × 约定利率 × 借款期限
    说明：年利率不得超过 LPR 的 4 倍
  • 逾期利息 = 本金 × 逾期利率 × 逾期天数
```

## ⚠️ 重要提示

### 赔偿计算说明

1. **仅供参考**：实际赔偿金额以法院认定为准
2. **地区差异**：不同地区赔偿标准可能不同
3. **证据关键**：需要提供相应证据证明金额
4. **专业咨询**：复杂案件请咨询执业律师

### 最佳实践

✅ **这样做**：
- 提供详细的案件信息
- 说明具体金额（工资、借款等）
- 列出已有证据
- 使用赔偿计算作为参考

❌ **不要这样做**：
- 完全依赖计算结果
- 忽视证据收集
- 自行起诉复杂案件
