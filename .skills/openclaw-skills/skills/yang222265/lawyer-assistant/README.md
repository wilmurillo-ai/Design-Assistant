# 律师助手技能 (Lawyer Assistant)

## 📖 技能介绍

律师助手是一个专业的法律案件分析工具，根据案件信息（当事人、对手、纠纷类型、起因等）智能分析案件，提供法律依据、案例检索建议和诉讼策略。

## ✨ 核心功能

1. **案件信息结构化** - 自动提取和整理案件关键信息
2. **法律分析** - 确定法律关系、适用律师和核心法条
3. **案例检索建议** - 提供检索平台和关键词建议
4. **证据指导** - 根据不同纠纷类型提供证据收集建议
5. **风险评估** - 分析诉讼风险和注意事项
6. **策略建议** - 提供诉讼策略和程序选择建议

## 📝 使用方法

### 在对话中使用

直接在对话中提供案件信息，技能会自动识别并分析：

```
请帮我分析这个案件：
- 当事人：张三
- 对手：李四
- 纠纷类型：借款合同纠纷
- 起因：2023 年 1 月李四向张三借款 10 万元，约定 2023 年 12 月归还，至今未还
- 诉求：要求归还本金及利息
- 证据：借条、转账记录
```

### 命令行使用

```bash
python3 lawyer_assistant.py "当事人：张三 对手：李四 纠纷类型：借款合同纠纷 起因：借款 10 万未还 诉求：要求归还"
```

## 📋 支持的信息格式

技能支持多种输入格式，以下格式都可以识别：

**格式一（推荐）：**
```
当事人：张三
对手：李四
纠纷类型：借款合同纠纷
起因：...
诉求：...
证据：...
```

**格式二（单行）：**
```
当事人：张三 对手：李四 纠纷类型：借款合同纠纷 起因：... 诉求：... 证据：...
```

**格式三（自然语言）：**
```
我是张三，李四欠我 10 万元借款不还，有借条和转账记录，想要起诉要回本金和利息
```

## 🎯 支持的纠纷类型

目前支持的纠纷类型包括：
- 借款合同纠纷
- 劳动合同纠纷
- 买卖合同纠纷
- 租赁合同纠纷
- 婚姻家庭纠纷
- 交通事故纠纷
- 侵权责任纠纷
- 知识产权纠纷
- 建设工程纠纷
- 股权纠纷

## 📊 输出内容

技能会生成结构化的案件分析报告，包括：

1. **案件概要** - 当事人、对手、纠纷类型、起因、诉求、证据
2. **法律分析** - 法律关系、适用律师、核心法条、诉讼时效
3. **案例检索建议** - 推荐平台、检索关键词
4. **证据收集建议** - 通用证据 + 特有证据
5. **风险提示** - 举证、时效、执行、成本等风险
6. **策略建议** - 诉前准备、程序选择、诉讼请求设计

## ⚠️ 重要声明

**本技能为辅助工具，具有以下限制：**

1. 不构成正式法律意见或律师 - 当事人关系
2. 法律法规可能存在更新，具体适用以最新规定为准
3. 案例分析基于一般情况，具体案件需具体分析
4. 复杂案件或重大利益事项，请务必咨询执业律师
5. AI 分析仅供参考，不能替代专业法律服务
