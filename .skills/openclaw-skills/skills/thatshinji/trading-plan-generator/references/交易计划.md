# 交易计划（Trading Plan）

交易计划回答"**我该怎么买、怎么卖、怎么止损、怎么持仓？**"

---

## 一、交易计划四要素

每笔交易必须包含以下四个明确部分：

1. **进场信号** — 何时进入
2. **止损点** — 止损离场的价格水平
3. **止盈点** — 止盈离场的价格水平
4. **退出机制** — 何时、如何获取利润

---

## 二、进出场信号

### 建仓信号（进场）

等待价格回落到**实际支撑区域**形成确认后买入；等待价格回升到**实际阻力区域**形成确认后卖空。

> "成功的回调趋势交易，核心是找到支撑区域以买入和阻力区域以卖出。"

### 止损点（最关键）

**永远要有止损。** 每次交易必须预先确定：
- 止损离场的价格位置
- 止损金额（每笔交易风险资金）

### 止损的两种情况

1. **初始止损** — 入市后马上设置，保护资本
2. **移动止损** — 入市有利后，通过移动止损锁定利润（趋势交易者常见做法）

> "你之所以买入，是因为你相信市场已经触及潜在的支撑线，并且这个支撑线将抬高。你在某一水平上止损离场，是因为你认为自己的设想是错误的。"

---

## 三、资金管理（风险控制）

### 资金管理的目的

- **盈利时增加交易规模**
- **亏损时减少交易规模**

### 固定百分比法（推荐给专业交易者）

每笔交易承担账户余额的固定百分比作为风险：

```
每笔交易风险金额 = 账户余额 × 风险百分比
合约数量 = 账户余额 × 风险百分比 / 每份合约风险金额
```

**推荐风险百分比：** 每笔交易风险不超过账户的 **1%~2%**

### 固定比率法（适合小额账户）

每增加一份合约需要先实现固定的利润增量：

```
每份合约需要盈利 = 账户增量的固定百分比
```

当亏损导致账户余额下降时，**减少合约数量**；当盈利时，**增加合约数量**。

### 风险管理目的

> "盈利时增大交易规模，亏损时减少交易规模。"

---

## 四、预期值（Expectation）

### 预期公式

```
预期值 = (准确率 × 平均收益/平均损失) - (1 - 准确率)
```

### 目标指标

作为趋势交易者，建议达到以下最低目标：

| 指标 | 目标值 |
|------|--------|
| 准确率 | ≥ 33%（10次交易中约3-4次盈利） |
| 平均收益/平均损失 | ≥ 3:1 |
| 预期值 | ≥ 0（即正值） |

> "作为风险管理者，你应该制定方法以达到一定的预期值，而不是某个准确度。"

### 小洞察

**趋势交易不追求准确率，而是追求预期值。** 一个33%准确率、3:1盈亏比的策略，可以产生正预期值，长期来看是盈利的。

---

## 五、心理因素

### 必须管理的情绪

- **恐惧** — 人们见下跌就害怕，频繁调整止损点
- **贪婪** — 人们总想赚更多，过早获利了结
- **希望** — 人们期盼反转，不愿意承认失败

### 正确心态

> "你应该为了抓住赚取期望利润的机会而交易，你不该为了赚得短期利润的喜悦而交易。"

**记住：** 长期趋势交易准确率低，亏损交易数量远超盈利交易。做好心理准备——**趋势交易者的生活是痛苦的**。

---

## 六、系统终止点（权益动量止损）

除了单笔交易的止损，还需要对整个**交易方法**设置终止点。

### 三个作用

1. 提前预警，告知你的交易策略是否开始失效
2. 当权益曲线跌破系统终止点时，**停止交易**
3. 当权益曲线恢复到终止点以上时，**重新开始交易**

### 推荐做法

- 建立**单一合约权益曲线**（不含资金管理影响）
- 设定财务边界（如：风险资金损失到$10,000时停止）
- 监控权益曲线的移动平均线或波动最低点

---

## 七、执行清单

每笔交易前逐项确认：

1. [ ] 入场信号明确
2. [ ] 止损价格已设定
3. [ ] 止损金额不超过账户的1-2%
4. [ ] 交易计划符合当前趋势方向
5. [ ] 心理状态稳定（不因恐惧/贪婪/希望而随意调整计划）
6. [ ] 目的性正确（为预期值交易，不为短期利润）

---

## 八、崩溃风险

> "除非爆仓风险趋近0，否则不应该交易。"

**爆仓风险 = 一个统计量，告诉你账户在某段时间内被抹平的数学概率。**

降低爆仓风险的方式：
- 降低单笔交易风险资金
- 提高交易准确率
- 提高平均收益/平均损失比率

---

## 九、关键原则

1. **永远带止损** — 没有止损的交易等同于赌博
2. **为预期交易** — 不是为了猜对方向
3. **长期持有正预期策略** — 一个好的趋势策略80%时间在亏损，20%时间赚大钱
4. **简单执行** — 不要过度复杂化交易系统
5. **管理心理** — 趋势交易是痛苦的，保持纪律性
