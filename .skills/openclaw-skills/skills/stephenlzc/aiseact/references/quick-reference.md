## 可靠性等级说明 / Reliability Key

| 符号 / Symbol | 含义 / Meaning |
|------|------|
| ✅ 可靠 / Reliable | 有编辑团队、事实核查可靠 / Editorial oversight, fact-checked |
| ⚠️ 半可靠 / Semi-reliable | 视场景判断 / Context-dependent |
| 🔄 应替代 / Should Replace | 建议用更可靠的来源替代 / Use better source if available |
| ❌ 不可靠 / Unreliable | 缺乏编辑审核 / No editorial review |
| ⛔ 弃用 / Deprecated | 社区共识禁止使用 / Community consensus: do not use |

## 使用建议 / Usage Recommendations

1. **政治新闻 / Political News** → AP、Reuters、BBC
2. **中国相关 / China** → 财新 / Caixin、联合早报 / Zaobao、南华早报 / SCMP
3. **事实核查 / Fact-check** → Snopes、PolitiFact、FactCheck.org
4. **技术文档 / Tech Docs** → 官方文档 / Official docs、GitHub、StackOverflow
5. **学术研究 / Academic** → 学术期刊 / Journals、官方数据库 / Official databases
