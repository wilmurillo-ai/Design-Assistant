# Authoritative Sources Recommendation List

This document compiles authoritative information sources recommended by AISEACT, organized by the **Primary Source First** principle.

---

## Primary vs Secondary Sources

### Definitions

| Type | Definition | Examples | Credibility |
|------|------------|----------|-------------|
| **Primary Source** | Original source, directly from information producer | Company prospectus, government announcements, official documents | Highest |
| **Quasi-Primary Source** | Officially authorized, close to original | Original reports from authoritative media | High |
| **Secondary Source** | Analysis/interpretation based on primary sources | Industry analysis reports, commentary articles | Medium |
| **Tertiary Source** | Multiple rewrites, scraped content | 百家号 Baijiahao, self-media interpretations | Low |

### Usage Principles

1. **Prioritize Primary Sources**: Use official websites and prospectus instead of analysis articles when possible
2. **Trace to Origin**: For secondary sources, try to find their cited primary sources
3. **Cross-Validate**: Important information needs 2-3 independent source supports
4. **Avoid Tertiary Sources**: Don't cite scraped content from 百家号 Baijiahao, 搜狐号 Sohu, etc.

## VII. Quick Selection Guide

### Primary Sources by Scenario

| Scenario | Primary Sources | Secondary Sources | Sources to Avoid |
|----------|-----------------|-------------------|------------------|
| **Company Research** | Prospectus, annual reports, official website | Authoritative financial media | 百家号 Baijiahao, 搜狐号 Sohu |
| **Technical Questions** | Official documentation, GitHub | Stack Overflow | CSDN scrapers, 简书 Jianshu |
| **Policies/Regulations** | Government websites, legal databases | Authoritative media interpretation | Self-media interpretation |
| **Industry Data** | Bureau of Statistics, industry associations | Consulting firm reports | Articles without cited sources |
| **Academic Questions** | Academic databases, papers | Authoritative institution reports | 百度百科 Baidu Baike, 知乎 Zhihu |
| **News Events** | Official announcements, authoritative media | Multiple report comparison | Single self-media source |

### Domain Quick Judgment

**High Credibility Domains**:
- `.gov.cn` / `.gov` - Government
- `.edu.cn` / `.edu` - Educational institutions
- `cninfo.com.cn` - Company announcements
- Well-known corporate official websites

**Low Credibility Domains**:
- `baijiahao.baidu.com` - 百家号 Baijiahao
- `sohu.com/a/` - 搜狐号 Sohu
- `csdn.net` (non-well-known bloggers)
- `jianshu.com` - 简书 Jianshu
- Random character domains

## VIII. Usage Recommendations

1. **Find Primary Sources First**: Before searching, predict where primary sources are most likely to be found
2. **Use site Syntax**: `site:cninfo.com.cn`, `site:gov.cn` for quick location
3. **Validate Secondary Sources**: Check cited original sources, directly access primary sources
4. **Label Information Nature**: Clearly indicate primary or secondary in responses
5. **Continuous Updates**: Regularly check if primary sources have updated information
