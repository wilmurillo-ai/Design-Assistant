# Translation Modes

Choose the scope based on what the user means by "all text":

- `slides only`
  Best for ordinary visible deck translation with minimal inherited-text churn.

- `slides + notes`
  Use when the user explicitly wants speaker notes translated.

- `slides + notes + layouts + masters`
  Use when the user requests that no editable source-language text remain anywhere in the deck package.

Guidance:

- Layouts and masters often contain inherited placeholder text that users never see during presentation, but they still count as editable text.
- Translating layouts and masters is useful for strict verification or when the template itself will be reused later.
- If design fidelity is fragile, translate layouts and masters only when requested or when zero-source-text verification is a hard requirement.
