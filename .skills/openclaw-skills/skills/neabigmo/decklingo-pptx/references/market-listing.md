## Long Description

### English

DeckLingo for PPTX is built for people who need to translate real presentation decks, not rebuild them from scratch. It works directly on editable PPTX text, supports target-language selection, separate interaction language, glossary control, notes/layout/master scope selection, and post-translation verification.

### Chinese

DeckLingo for PPTX 面向真正需要翻译演示文稿的人，而不是重新手工做一份 PPT。它直接处理 PPTX 中的可编辑文本，支持目标翻译语言选择、交互语言分离、术语表控制、备注/版式/母版范围选择，以及翻译后的校验。

### Japanese

DeckLingo for PPTX は、プレゼン資料を一から作り直すのではなく、既存の資料をそのまま翻訳したい人のためのツールです。編集可能な PPTX テキストを直接処理し、翻訳先言語の選択、操作言語の分離、用語集の適用、ノート/レイアウト/マスターの範囲指定、翻訳後の検証に対応しています。
