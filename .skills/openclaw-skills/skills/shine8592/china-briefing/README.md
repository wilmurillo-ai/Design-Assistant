# 📰 多尺度中文资讯简报

**从一条街道到全球，任意地理范围均可生成。**

---

## ✨ 能做什么

| 你说的 | 生成什么 |
|--------|---------|
| "江川路街道简报" | 街道民生、政务、社区动态 |
| "上海简报" | 城市政策、经济、社会热点 |
| "国际简报" | 全球头条、地缘、财经 |
| "AI行业简报" | 行业动态、研报、社区声音 |

---

## 🔍 多源获取

从 **21+ 个权威来源** 自动抓取，按尺度匹配：

- **全球**：Reuters、BBC、CNN、Al Jazeera、NYT中文、FT中文
- **全国**：新华社、央视、澎湃、观察者网
- **省市**：本地官媒、政府官网、省级日报
- **区县/街道**：政务公众号、政府公告、本地社区
- **社交**：微博、小红书、Twitter/X、Reddit

每条信息标注出处，不编造、不标题党。

---

## 📋 固定输出结构

```
🌍 核心结论（1-2句总结）
📌 头条3条（含嘟嘟独家点评）
📊 分类热点（2-4个主题）
📈 数据看板（可选）
🎯 行动建议（高/中/低优先级）
```

---

## ⚙️ 技术

- **依赖**：Python 3（仅标准库）
- **数据来源**：通过 OpenClaw 的 `web_search` 和 `web_fetch` 获取
- **无需 API Key**：不依赖任何第三方付费服务
- **辅助脚本**：`scripts/briefing_helper.py` 自动识别用户需求的地理/行业尺度
