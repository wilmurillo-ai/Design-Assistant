# 搜索模板

## 按地理尺度

### 全球
```
1. "world news latest today 2026"
2. "国际新闻 最新 2026"
3. "breaking news major events"
4. "global economy geopolitics April 2026"
```

### 国家（中国）
```
1. "中国 2026 最新 政策 经济"
2. "中国社会 民生 热点"
3. "China news April 2026"
```

### 省
```
1. "{省} 2026 最新动态"
2. "{省} 经济 民生 政策"
3. "{省} 政府 新闻"
```

### 市
```
1. "{市} 2026 最新"
2. "{市} 民生 经济 社区"
3. "{市} 政府 实事项目"
4. "{市} 人大 会议"
```

### 区/县
```
1. "{区县名} 2026 最新"
2. "{区县名} 民生 经济 社区"
3. "{区县名} 政府 实事项目"
4. "{区县名} 改造 建设"
```

### 街道（最小尺度，核心差异化）
```
1. "{街道名} 2026 最新"
2. "{街道名} 民生 实事"
3. "{街道名} 社区 改造"
4. "{街道名} 书记 主任 工作"
5. "{区} {街道名}"
6. "{街道名} 公众号"
7. "{街道名} 公园 学校 医院"
```

## 按行业
