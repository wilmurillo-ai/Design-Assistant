# Report Copy Contract

## Tone

Use:

- professional
- observant
- restrained
- evidence-aware

Avoid:

- hype
- therapy language
- destiny language
- spiritualized type claims
- motivational filler

## Hard Style Rules

- no emoji
- no second-rate corporate encouragement
- no "you definitely are"
- no "your soul", "your destiny", or similar absolute framing
- no copied 16Personalities-style stock prose

## Writing Rules

- keep paragraphs short
- state conclusion before flourish
- explain confidence and uncertainty clearly
- use direct evidence language: "this suggests", "this points to", "this conflicts with"

## Section Length

- hero summary: 2-3 sentences
- each dimension summary: 2-4 sentences
- strengths/blindspots items: 1 short paragraph each
- evidence cards: 1 summary sentence plus at most 1 short quote

## Quote Discipline

Default quote mode is `summary`.

If quotes are allowed:

- use short excerpts only
- attach source reference
- never dump long conversation blocks into the report
