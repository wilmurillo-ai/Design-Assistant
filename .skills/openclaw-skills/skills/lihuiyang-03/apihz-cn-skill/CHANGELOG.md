## v1.0.5 (2026-03-09) - 加密存储

### 🔐 安全增强
- **KEY 加密存储** - 使用 AES-256-GCM 加密 APIHZ_KEY
- **DMSG 加密存储** - 动态密钥预留信息同样加密
- **机器指纹密钥** - 加密密钥基于主机名 + 用户名 + 工作区生成
- **向后兼容** - 支持读取旧版明文格式，自动提示升级

### 🔧 技术实现
- **加密算法:** AES-256-GCM (认证加密)
- **随机 IV:** 每次加密生成新的 16 字节 IV
- **Auth Tag:** 16 字节认证标签，防止篡改
- **存储格式:** `base64(iv):base64(authTag):base64(encrypted)`

- 更新安全提示章节
- 说明加密机制和不跨机器使用的原因
- 添加最佳实践建议

## v1.0.4 (2026-03-09) - 动态密钥支持

### 🔐 新功能
- **动态密钥 (dkey) 支持** - 实现接口盒子动态密钥验证机制
- **自动 dkey 生成** - 客户端自动获取 dcan 并生成 dkey = md5(dmsg + dcan)
- **dcan 缓存** - 智能缓存动态参数 (10 秒有效期)，减少 API 调用
- **环境变量支持** - 新增 `APIHZ_DMSG` 环境变量配置
