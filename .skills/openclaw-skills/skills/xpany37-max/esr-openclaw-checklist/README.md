## 🔍 安全检查项目

脚本执行8项安全检查：

1. **默认端口使用情况** - 检查是否使用默认端口18789
2. **服务监听地址检查** - 检查是否绑定到0.0.0.0
3. **内网穿透工具检测** - 检测未授权穿透工具
4. **Node.js版本检查** - 检查Node.js版本是否>=18
5. **密码登录模式检查** - 检查是否启用密码登录
6. **Skill数量及官方来源检查** - 检查非官方Skill
7. **配置文件权限检查** - 检查openclaw.json权限
8. **OpenClawd深度安全审计** - 检查Sandbox等安全配置

## 🧠 运行思路

### 1. 手动执行流程
```
用户: "运行ESR安全检查"
    ↓
OpenClaw: 调用 python3 scripts/openclaw_checklist.py
    ↓
Python脚本: 执行8项安全检查
    ↓
输出: 标准化格式的结果 + 信息安全提示
    ↓
保存: 生成时间戳命名的JSON文件
```

### 2. 定时任务流程
```
系统时间: 每天17:00
    ↓
OpenClaw Cron: 触发定时任务
    ↓
AI代理: 执行Python脚本
    ↓
Python脚本: 
    1. 执行8项安全检查
    2. 格式化钉钉消息
    3. 发送到钉钉群
    ↓
AI代理: 回复完成消息
    ↓

```
