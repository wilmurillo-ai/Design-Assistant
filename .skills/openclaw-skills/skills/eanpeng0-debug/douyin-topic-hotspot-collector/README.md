# 抖音主题热点采集器

## 功能简介
根据用户输入的主题与时间范围，搜索抖音相关热点视频，并按互动数据整理输出前10条热点结果。

## 当前版本
1.0.0

## 支持能力
- 单主题抖音热点采集
- 时间范围限制（最近24小时 / 最近7天 / 最近30天）
- 前10条热点表格输出
- 结果导出

## 适合谁用
- 抖音创作者
- 内容运营
- 选题分析人员
- 学习者

## 不支持
- 跨平台采集
- 自动生成脚本
- 自动发布
- 多主题批量分析

## 依赖
- 可访问抖音的网络环境
- 浏览器能力
- 页面可见字段读取能力
- 导出时需具备写文件权限

## 安全边界
- 仅访问公开可见页面
- 不自动登录账号
- 不自动发布内容
- 不修改非指定路径文件
- 不伪造数据
