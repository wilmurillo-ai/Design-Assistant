## 使用场景

### 适用场景
1. **多代理协作**：主代理与子代理间的记忆共享
2. **项目记忆管理**：写作、分析等项目的记忆组织
3. **系统状态监控**：定期检查记忆系统健康状态
4. **文件生命周期管理**：活跃文件→归档文件的管理

### 推荐工作流
1. **每日/每周**：使用`memory_check.ps1`检查状态
2. **按需**：使用`archive_tool.ps1`归档文件
3. **定期**：运行`cleanup_example.ps1`进行系统维护
4. **监控**：使用`agent_check.ps1`检查代理状态
