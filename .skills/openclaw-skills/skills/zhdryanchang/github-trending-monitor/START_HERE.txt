📡 API ENDPOINTS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  POST /notify            - Trigger notification
  POST /subscribe         - Subscribe to daily reports
  POST /unsubscribe       - Cancel subscription
  GET  /subscription/:id  - Check status
  GET  /health            - Health check
