# Obsidian Headless 使用手册

## 简介

Obsidian Headless 是一个在无显示器/无 GUI 环境下通过自然语言指令管理 Obsidian 笔记仓库的工具。

## 快速参考

### 指令速查表

| 指令 | 功能 | 示例 |
|------|------|------|
| `obs创建笔记 [文件名] [内容]` | 创建新笔记 | `obs创建笔记 待办清单 买东西` |
| `obs删除笔记 [文件名]` | 删除笔记（带确认） | `obs删除笔记 旧笔记` |
| `obs查看笔记 [文件名]` | 显示笔记内容 | `obs查看笔记 欢迎` |
| `obs搜索标题 [关键词]` | 搜索文件名 | `obs搜索标题 openclaw` |
| `obs搜索内容 [关键词]` | 搜索文件内容 | `obs搜索内容 tavily` |
| `obs模糊搜索 [关键词]` | 标题+内容搜索 | `obs模糊搜索 home` |
| `obs今天日记 [内容]` | 创建/追加日记 | `obs今天日记 今天...` |
| `obs列出所有` | 列出所有笔记 | `obs列出所有` |
| `obs列出文件夹` | 列出所有文件夹 | `obs列出文件夹` |
| `obs最近笔记` | 最近10篇笔记 | `obs最近笔记` |
| `obs修改库路径` | 修改仓库路径 | `obs修改库路径` |

### 格式说明

**obs 前缀格式（大小写不敏感）：**
- `obs指令` - 推荐
- `obs 指令` - 带空格
- `obs-指令` - 带连字符
- `OBS指令` - 大写

### 路径支持

```bash
# 根目录
obs创建笔记 笔记名

# 子目录
obs创建笔记 文件夹/笔记名
obs创建笔记 文件夹/子文件夹/笔记名
```

## 安装

```bash
cd ~/.openclaw/skills/obsidian-headless
./install.sh
```

或直接运行：
```bash
~/.openclaw/skills/obsidian-headless/obs "指令"
```

## 配置

### 首次使用

运行任意命令时会提示输入仓库路径：

```
首次使用 Obsidian Headless
==========================

请输入 Obsidian 仓库路径: /path/to/your/vault
✓ 有效的 Obsidian 仓库
✓ 已保存配置到: ~/.config/obsidian-headless/vault-path
```

### 修改配置

```bash
# 方法1: 使用环境变量（临时）
export OBSIDIAN_VAULT=/new/path

# 方法2: 删除配置重新输入
rm ~/.config/obsidian-headless/vault-path
./obs "obs创建笔记 测试"

# 方法3: 使用修改库路径命令
./obs "obs修改库路径"
```

## 指令格式

所有指令都支持 `obs` 前缀（**大小写不敏感**），`obs` 和指令之间可以有**空格、-、_、: 等连接符**：

| 格式 | 示例 |
|------|------|
| `obs指令` | `obs创建笔记 新笔记` |
| `obs 指令` | `obs 创建笔记 新笔记` |
| `obs-指令` | `obs-创建笔记 新笔记` |
| `obs_指令` | `obs_创建笔记 新笔记` |
| `OBS指令` | `OBS创建笔记 新笔记` |
| `Obs::指令` | `Obs::创建笔记 新笔记` |

## 完整指令列表

### 1. 创建笔记

```bash
obs创建笔记 [文件名] [内容]
```

**说明：**
- `文件名` 支持路径，如 `文件夹/笔记名` 或 `笔记名`
- 如果**只输入文件名**（无内容），创建**空笔记文件**
- 如果**输入了内容**，内容**直接写入文件**，不会自动添加标题
- 笔记的标题需要用户在 `内容` 中自行定义
- **文件名和内容之间可以用空格或换行符分隔**

**示例（空格分隔）：**
```bash
obs创建笔记 待办清单
obs创建笔记 项目想法 这个项目的核心目标是...
obs创建笔记 日记/2024-03-14 今天完成了...
```

**示例（换行符分隔）：**
```bash
obs创建笔记 AI/总结
# AI 总结

今天学习了...

obs创建笔记 想法
# 新想法 #idea
## 详细描述
具体的内容在这里...
```

### 2. 删除笔记

```bash
obs删除笔记 [文件名]
```

**说明：**
- 支持模糊匹配
- 找到唯一匹配时显示详情并要求确认
- 找到多个匹配时列出带序号的列表，用户选择

**示例：**
```bash
obs删除笔记 旧笔记
obs删除笔记 AI/测试
```

**删除确认流程：**

**单个匹配时：**
```
即将删除笔记:
  文件名: xxx
  路径: /path/to/xxx.md

  内容预览 (共 15 行):
    ...

请回复确认:
  [Y] 确认删除
  [N] 取消
```

**多个匹配时：**
```
找到多个匹配的笔记:

  [1] 笔记A
      位置: folder1/
      大小: 20 行

  [2] 笔记B
      位置: folder2/
      大小: 35 行

请回复要删除的笔记编号 (或 0 取消)
```

### 3. 查看笔记

```bash
obs查看笔记 [文件名]
```

**示例：**
```bash
obs查看笔记 欢迎
obs查看笔记 AI/总结
```

### 4. 搜索笔记标题

```bash
obs搜索标题 [关键词]
```

**说明：** 只搜索文件名（路径+文件名）

**示例：**
```bash
obs搜索标题 openclaw
obs搜索标题 2024
```

### 5. 搜索笔记内容

```bash
obs搜索内容 [关键词]
```

**说明：** 搜索所有 `.md` 文件的内容

**示例：**
```bash
obs搜索内容 home assistant
obs搜索内容 tavily
```

### 6. 模糊搜索

```bash
obs模糊搜索 [关键词]
```

**说明：** 同时搜索标题和内容

**示例：**
```bash
obs模糊搜索 obsidian
obs模糊搜索 项目
```

### 7. 今天日记

```bash
obs今天日记 [可选内容]
```

**说明：**
- 自动在 `日记/` 文件夹下创建 `YYYY-MM-DD.md`
- 如果今天日记已存在，追加内容

**示例：**
```bash
obs今天日记
obs今天日记 今天完成了技能开发...
```

### 8. 列出所有笔记

```bash
obs列出所有
```

**说明：** 列出仓库中所有 `.md` 文件

### 9. 列出所有文件夹

```bash
obs列出文件夹
```

**说明：** 列出仓库中所有文件夹（排除 `.obsidian`）

### 10. 最近笔记

```bash
obs最近笔记
```

**说明：** 显示最近修改的 10 篇笔记

### 11. 修改库路径

```bash
obs修改库路径
obs修改库目录
obs更改路径
obs切换仓库
```

**说明：** 交互式修改仓库路径

## 使用示例

### 日常工作流

```bash
# 1. 创建新笔记
obs创建笔记 会议纪要 今天讨论了Q3计划...

# 2. 查看笔记
obs查看笔记 会议纪要

# 3. 搜索相关内容
obs搜索内容 Q3计划

# 4. 记录今天的工作
obs今天日记 完成了会议纪要整理

# 5. 列出所有笔记检查
obs列出所有
```

### 项目管理

```bash
# 创建项目结构
obs创建笔记 项目/项目A/需求文档 需求分析...
obs创建笔记 项目/项目A/进度跟踪 当前进度...

# 搜索项目相关笔记
obs模糊搜索 项目A

# 查看具体笔记
obs查看笔记 项目/项目A/需求文档
```

### 知识整理

```bash
# 创建分类笔记
obs创建笔记 AI/ChatGPT技巧 使用技巧总结...
obs创建笔记 AI/Claude技巧 使用技巧总结...
obs创建笔记 前端/React技巧 使用技巧总结...

# 模糊搜索所有技巧
obs模糊搜索 技巧
```

## 故障排除

### 提示"未找到 Obsidian 仓库路径"

```bash
# 设置环境变量
export OBSIDIAN_VAULT=/path/to/your/vault

# 或使用修改库路径命令
obs修改库路径
```

### 搜索速度慢

安装 ripgrep：
```bash
# Ubuntu/Debian
sudo apt install ripgrep

# macOS
brew install ripgrep
```

### 删除时找不到笔记

使用更精确的标题，或使用模糊搜索先找到准确标题：
```bash
obs模糊搜索 关键词
obs删除笔记 准确文件名
```

### 配置文件位置

```
~/.config/obsidian-headless/vault-path
```

删除此文件可重置配置。

## 依赖要求

### 必需依赖

以下命令通常在 Linux/macOS 系统中已预装：

- `bash` - 脚本解释器
- `find` - 查找文件
- `grep` - 文本搜索
- `mkdir`, `touch`, `rm`, `cat` - 文件操作
- `date`, `chmod` - 系统命令
- `head`, `tail`, `cut`, `awk` - 文本处理

### macOS 用户注意

旧版 macOS 可能缺少 `realpath` 命令：

```bash
# 安装 coreutils（包含 realpath）
brew install coreutils
```

### 可选依赖（推荐）

- `ripgrep (rg)` - 更快的搜索
