# Ad Copy Writer Reference

Use this file when the user wants a specific marketing copy shape rather than generic ad text.

## Common Output Shapes

- Headline set: 3 to 10 compact headline variants.
- CTA set: short action-led CTA options.
- Landing page section: hero, value proposition, proof, CTA.
- Paid social copy: hook, benefit, action.
- Launch copy: what is new, why it matters, what to do next.

## Style Rules

- Lead with the benefit, not the feature list.
- Keep claims specific and readable.
- Match the requested channel and audience.
- Avoid generic hype unless the user explicitly wants aggressive copy.
- If the user asks for variants, vary angle and wording rather than repeating the same line.
