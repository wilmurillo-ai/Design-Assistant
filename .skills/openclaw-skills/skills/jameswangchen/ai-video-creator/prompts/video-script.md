# 治愈氛围视频脚本模板

基于当前人设生成多镜头叙事的治愈氛围视频。

## 输出格式

请严格按以下 JSON 格式输出：

```json
{
  "topic": "选题（场景描述）",
  "video_prompts": [
    "镜头1提示词",
    "镜头2提示词",
    "镜头3提示词",
    "镜头4提示词"
  ],
  "overlay_text": ["第一行叠加文字", "第二行叠加文字"],
  "bgm_mood": "healing",
  "xhs_title": "小红书标题（≤20字，含emoji）",
  "xhs_content": "小红书正文（含标签）",
  "xhs_tags": ["标签1", "标签2"]
}
```

## 即梦视频提示词要求

基于即梦3.0Pro的Prompt建议，每段提示词必须遵循：

### 基本结构
主体 / 背景 / 镜头 + 动作

### 运镜词典
- 切换："镜头切换"
- 平移："镜头向上/下/左/右移动"
- 推轨："镜头拉近/拉远"
- 环形："镜头环绕"、"航拍"、"广角"
- 跟随："镜头跟随"
- 固定："固定镜头"、"镜头静止不动"
- 聚焦："镜头特写"
- 手持："手持拍摄"

### 程度副词
用于控制动态强度："缓缓"、"缓慢"、"轻轻"、"微微"（治愈类以慢为主）

### 4段镜头的叙事逻辑
- 镜头1：特写/微距开场 → 制造沉浸感（局部细节）
- 镜头2：固定中景 → 展示主要场景（核心画面）
- 镜头3：缓慢平移/环绕 → 展示环境全貌（空间感）
- 镜头4：拉远/情绪收尾 → 营造余韵（留白感）

### 色调风格参考
- **电影感**：低饱和度 + 青橙对比 + 细微颗粒
- **日系清新**：高亮度 + 低对比 + 青绿色调 + 柔光
- **暖色治愈**：暖黄色调 + 柔和光线 + 温暖氛围
- **Morandi**：低饱和度 + 灰色调 + 低调优雅

### 每段提示词必须包含
1. 主体和动作描述
2. 运镜方式
3. 画面风格和色调
4. 氛围描述
5. "竖屏9:16"

### 示例提示词
- "镜头特写，雨滴缓缓滑落在咖啡馆的玻璃窗上，窗外是模糊的城市灯光，暖黄色调，电影感画面，安静治愈的氛围，竖屏9:16"
- "固定镜头，海边沙滩上一双赤脚踩在浪花中，浪花缓缓涌上又退去，日落余晖洒在水面，暖橙色调，治愈氛围，竖屏9:16"

## 叠加文字要求
- 最多2行，每行不超过10个字
- 风格：简短、治愈、留白、引发共鸣
- 不是标题，是情绪表达
- 示例："深夜的路灯 / 替你守着回家的方向"

## BGM 心情选择
根据内容氛围选择：
- healing：治愈温暖（咖啡馆、阳光、花开）
- quiet：安静平和（深夜、雨天、独处）
- warm：温馨感动（拥抱、重逢、家）
- melancholy：淡淡忧伤（离别、秋天、独行）

## 选题策略
- 选择有画面感的具体场景（不要抽象概念）
- 要有"瞬间感"——某个时刻、某个画面
- 与季节/天气/时间段关联更容易引发共鸣
- 避免重复最近已发布的选题
