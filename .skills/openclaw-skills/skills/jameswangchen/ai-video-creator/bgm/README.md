# BGM 背景音乐

将无版权音乐文件（`.mp3`）放入对应心情子目录中。

## 目录结构

```
bgm/
├── healing/     # 治愈温暖（咖啡馆、阳光、花开）
├── quiet/       # 安静平和（深夜、雨天、独处）
├── warm/        # 温馨感动（重逢、家、童年回忆）
├── melancholy/  # 淡淡忧伤（离别、秋天、独行）
├── dreamy/      # 梦幻飘渺（云海、极光、星河）
├── citynight/   # 都市夜景（霓虹灯、便利店、深夜街头）
└── nature/      # 自然之声（森林、溪流、海浪）
```
