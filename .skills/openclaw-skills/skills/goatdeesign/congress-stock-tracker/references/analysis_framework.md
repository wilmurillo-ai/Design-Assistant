# 国会交易分析框架

## 分析维度

### 1. 交易信号强度评估

按以下维度评估每笔交易的信号强度（1-5分）：

| 维度 | 1分（弱） | 3分（中） | 5分（强） |
|------|---------|---------|---------|
| 金额规模 | < $15K | $50K-$250K | > $500K |
| 议员数量 | 单人交易 | 2-3人同标的 | 4人以上同标的 |
| 委员会相关性 | 无直接关联 | 间接关联 | 直接管辖行业 |
| 时间集中度 | 分散 | 2周内 | 1周内密集 |
| 跨党派程度 | 单一党派 | - | 两党共同买入 |

**综合信号强度** = 各维度平均分
- ⚡ 强信号（≥4）：高度关注，可能有重大政策/事件催化
- 🔔 中信号（2.5-4）：值得跟踪，需结合其他信息判断
- 💤 弱信号（<2.5）：常规交易，参考价值有限

### 2. 事件关联分析流程

```
交易数据 → 识别标的 → 查找议员委员会 → 匹配行业关联
    ↓
近期立法/监管动态 → 时间线比对 → 判断是否存在信息优势
    ↓
生成事件假设 → 评估概率 → 输出分析报告
```

**关键问题清单：**
1. 该议员属于哪个委员会？该委员会近期有何动态？
2. 交易标的所属行业是否在该委员会管辖范围内？
3. 近期是否有相关法案/听证会/监管决定？
4. 是否有其他议员做出类似交易？
5. 交易时机是否与特定事件存在可疑的时间关联？
