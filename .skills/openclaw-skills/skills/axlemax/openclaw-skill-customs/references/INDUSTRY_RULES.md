# 行业特殊申报要素参考

> 📖 **使用说明**：根据结果中商品 HS 编码的**前 2 位**，在下方查找对应章节。
> 仅当结果中存在该章节商品时才需阅读对应部分，未命中则跳过本文件。

## 第21章 — 食品类（HS 21xx）

### 申报要素注意事项

| 要素 | 规范要求 | 常见遗漏 |
|------|----------|----------|
| 成分含量 | 须注明各成分及百分比 | 仅写"食品"不够，需写具体成分 |
| 制作方法 | 加工方式（烘焙/冷冻/腌制等） | 常遗漏 |
| 包装规格 | 每包净含量 | 需注明单位（克/毫升） |
| 食品添加剂 | 须注明具体名称及用途 | 不可笼统写"含添加剂" |
| 保质期形式 | 标注保质期表达方式 | — |

### 常见提示

- 预包装食品须标注净含量（如"每袋 500g"）
- 含动植物成分的需关注检验检疫要求

## 第22章 — 酒类饮料（HS 22xx）

### 申报要素注意事项

| 商品 | 必须申报的特殊要素 |
|------|-------------------|
| 🍷 葡萄酒 | 产区中英文名称、酒庄中英文名称、葡萄品种、酒精度(%) |
| 🍺 啤酒 | 每升麦芽汁浓度(°P)、容器材质（罐/瓶）、酒精度(%) |
| 🥃 蒸馏酒 | 酒精度(%)、原料（谷物/水果等） |
| 🧃 非酒精饮料 | 成分、是否含糖、包装容量 |

### 常见提示

- 零售包装酒类：重量须**包含瓶/罐重量**（不只是液体重量）
- 散装葡萄酒（桶装）与瓶装葡萄酒的 HS 编码不同
- 酒精度超过 0.5% 即归入酒类税号

## 第39章 — 塑料及制品（HS 39xx）

### 申报要素注意事项

| 产品形态 | 必须申报的特殊要素 |
|----------|-------------------|
| 初级形状 | 外观（颗粒/粉末/液态）、比重、成分、用途 |
| 板/片/膜 | 是否泡沫制、是否增强/层压、厚度 |
| 管/棒/型材 | 截面形状、是否硬质 |
| 再生塑料 | ⚠️ **底料来源**（进口废塑料/国产废塑料/回收利用）|
| 塑料制品 | 用途、材质（塑料品种）、品牌、型号 |

### 常见提示

- ⚠️ 再生塑料须注明底料来源，这是海关重点审核项
- 聚乙烯(PE)、聚丙烯(PP)、聚氯乙烯(PVC) 等须注明具体塑料品种
- "其他塑料制品"(3926) 须明确用途，不可仅写"塑料制品"

## 第41章 — 皮革（HS 41xx）

### 申报要素注意事项

| 要素 | 规范要求 | 说明 |
|------|----------|------|
| 鞣制方法 | 铬鞣/植鞣/半铬鞣/醛鞣 | 必填，不可留空 |
| 动物种类 | 牛/羊/猪/马/爬行动物/其他 | 须具体到物种 |
| 加工程度 | 湿蓝皮/坯皮/成品革/再鞣革 | 直接影响税号归类 |
| 加工方法 | 整张/半张/裁切 | — |
| 厚度 | 须注明（若适用） | — |

### 常见提示

- 濒危动物皮革（鳄鱼、蟒蛇等）需 CITES 许可证
- 湿蓝皮(wet-blue)和成品革的 HS 编码不同

## 第64章 — 鞋靴（HS 64xx）

### 申报要素注意事项

| 要素 | 规范要求 | 代码/分类 |
|------|----------|----------|
| 鞋面材料 | 须具体注明材质 | 皮革/纺织/塑料/橡胶 |
| 外底材料 | 须具体注明材质 | 橡胶/塑料/皮革/其他 |
| 类型 | 须注明用途 | 运动鞋/皮鞋/拖鞋/靴子等 |
| 尺码 | 内底长度 | 厘米或国际码 |
| 性别/年龄 | 男式/女式/儿童 | 影响税率 |

### 常见提示

- 鞋面材料面积占比 ≥ 50% 的材质为主要材质，决定 HS 归类
- 防水鞋靴有专门税号
- "运动鞋"需区分：有钉（田径/足球）vs 无钉（跑鞋/篮球鞋）

## 第85章 — 电子电气产品（HS 85xx）

### 申报要素注意事项

| 商品 | 必须申报的特殊要素 |
|------|-------------------|
| 芯片/集成电路 | ⚠️ **流片地**（境内/境外及具体国家）、工艺制程（纳米） |
| 手机/通讯设备 | 适用网络种类（蜂窝/无线/有线）、是否加密 |
| 电池 | 类型（锂离子/镍氢等）、容量(mAh/Wh)、电压 |
| 电机/马达 | 功率(W/kW)、电压、交流/直流 |
| 变压器/适配器 | 功率、输入输出电压、用途 |
| 电线电缆 | 额定电压、是否有接头、结构类型 |

### 常见提示

- ⚠️ 芯片流片地是近年海关重点关注项，须准确申报
- 通讯设备的"是否加密"字段须明确回答"未加密"或具体加密方式，不可留空
- 锂电池运输有特殊安全要求（UN3480/UN3481）
