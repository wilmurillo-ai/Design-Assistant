# qr-campaign-studio format reference

## Batch CSV columns

- name: output base name (required)
- type: url|text|wifi|vcard (default: url)
- url/content
- utm_source/utm_medium/utm_campaign/utm_term/utm_content (optional)
- logo (optional)

## vCard fields

- vcard_name (required for vcard)
- vcard_phone
- vcard_email
- vcard_org
- vcard_title
- vcard_url
- vcard_note
