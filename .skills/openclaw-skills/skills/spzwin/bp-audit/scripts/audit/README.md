# BP 审计模块脚本清单

**模块**: `audit`

---

## 脚本说明

本模块暂无自动化脚本，所有操作通过直接调用 API 完成。

如需批量审计或自动化流程，可自行编写脚本调用上述接口。

---

## 建议的脚本化场景

### 1. 批量审计脚本
遍历部门下所有员工，批量审计 BP 合规性。

### 2. 承接分析脚本
自动拉通上下级任务，生成承接分析报告。

### 3. GAP 分析脚本
自动对比上下级数据，识别承接差/执行差/逻辑差。

### 4. 覆盖率计算脚本
针对收入/利润指标，自动计算数值覆盖率。
