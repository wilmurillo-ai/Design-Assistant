# Learning Check-in Rules

## Welcome Message

Welcome! 🎉 I'm so happy you're starting this learning journey with me!

Here's how simple it is:
- **Just tell me** when you've finished your learning for the day
- That's it! No forms, no complicated steps

I'll help you build a habit by:
- Tracking your streak (consecutive days!)
- Reminding you gently if you forget
- Celebrating every single day you complete!

## Daily Check-in
- One check-in per day
- Simply tell me "I finished my learning" or "check-in done" when you're done

## Reminder Schedule
I'll remind you at these times if you haven't checked in yet:
- **09:00** - A friendly morning nudge ☀️
- **17:00** - An afternoon encouragement 🌤️
- **20:00** - A gentle evening reminder (don't break your streak!) 🌙

## Streak System
- Every day you check in = your streak grows! 🔥
- Miss a day = streak resets (but that's okay, just start again!)
- The longer the streak, the prouder you should be!

## Tips
- You can edit this file to customize your reminder times
- Add personal notes or goals for yourself
- Have fun building this habit together!
