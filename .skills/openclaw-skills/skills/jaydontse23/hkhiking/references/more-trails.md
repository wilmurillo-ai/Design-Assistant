# 香港更多遠足路徑資料

## 數據來源

- **HKO 遠足路徑天氣服務**：https://www.hko.gov.hk/tc/sports/index.html
- **GeoJSON 數據 API**：`https://www.hko.gov.hk/hiking/geojson/walks.geojson`
  - 包含 61 條路徑的完整地理數據（路線座標、長度、時間、難度）
  - 無需 API Key，可直接存取
- **郊野樂行路徑詳情**：`https://www.hiking.gov.hk/trail/info/id/{encoded_id}`
  - 每條路徑均有對應的詳情頁面

## 路徑分類說明

香港遠足路徑除四大長途遠足徑外，還分為三大類：

| 類型 | 英文 | 代碼前綴 | 條數 | 特點 |
|------|------|---------|------|------|
| 郊遊徑 | Country Trail | `cty_` | 31 條 | 中等難度，適合有一定體能的遠足人士 |
| 家樂徑 | Family Walk | `fmy_` | 13 條 | 難度最低，適合家庭及長幼 |
| 自然教育徑 | Nature Trail | `nat_` | 17 條 | 以自然認識為主，多設有解說牌 |

## HKO 遠足路徑天氣 API 使用方法

### 獲取路徑地理數據

```
GET https://www.hko.gov.hk/hiking/geojson/walks.geojson
```
返回所有 61 條郊遊徑／家樂徑／自然教育徑的完整 GeoJSON 數據，包含：
- `TRAIL_ID`：路徑代碼（如 `cty_01`、`fmy_1`、`nat_1`）
- `C_NAME`：中文名稱
- `NAME`：英文名稱
- `TIME_`：預計所需時間（分鐘）
- `MEA_LEN`：長度（米）
- `DIF_LEVEL`：難度等級（1-5）

### 獲取路徑逐小時天氣預報點位

```
GET https://www.hko.gov.hk/hiking/geojson/hourlySegPts_{TRAIL_ID}.geojson
```
例如：`hourlySegPts_cty_01.geojson`

### 獲取四大長途遠足徑數據

```
GET https://www.hko.gov.hk/hiking/geojson/data_fixed_longDistanceTrails.geojson
```
包含麥理浩徑（mc_1~10）、港島徑（hk_1~8）、鳳凰徑（lt_1~12）、衛奕信徑（ws_1~10）的地理數據。

### 郊野樂行路徑詳情頁

```
https://www.hiking.gov.hk/trail/info/id/{encoded_id}
```
每條路徑對應的 `encoded_id` 已內嵌於 HKO JS 檔案中，可直接引用。

## 郊野樂行網站（hiking.gov.hk）

「郊野樂行」是漁農自然護理署（AFCD）官方遠足資訊平台，提供：
- 各郊野公園內的特色景點及路徑資訊
- 按興趣、體能及經驗篩選路線
- 路徑地圖、難度評分、景點介紹

**路徑詳情頁格式**：`https://www.hiking.gov.hk/trail/info/id/{encoded_id}`

> 注意：`hiking.gov.hk` 網站目前伺服器端可能出現技術問題，部分頁面未必可直接存取。建議優先使用 HKO GeoJSON API 取得路徑數據。
