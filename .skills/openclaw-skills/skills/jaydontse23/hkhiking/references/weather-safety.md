## 香港天文台天氣API

### 九天天氣預報（免費，無需 API Key）

```
GET https://data.weather.gov.hk/weatherAPI/opendata/weather.php?dataType=fnd&lang=tc
```

參數：
- `dataType=fnd`：九天天氣預報
- `lang=tc`：繁體中文（可選 `en` 英文、`sc` 簡體中文）

回傳欄位說明：
```json
{
  "generalSituation": "天氣概況描述",
  "weatherForecast": [
    {
      "forecastDate": "20260415",
      "week": "星期三",
      "forecastWind": "東至東南風 4至5級",
      "forecastWeather": "大致多雲，有幾陣驟雨",
      "forecastMaxtemp": { "value": 26, "unit": "C" },
      "forecastMintemp": { "value": 22, "unit": "C" },
      "forecastMaxrh": { "value": 95, "unit": "percent" },
      "forecastMinrh": { "value": 75, "unit": "percent" },
      "ForecastIcon": 54,
      "PSR": "低"
    }
  ],
  "updateTime": "2026-04-15T16:00:00+08:00"
}
```

### 當前天氣觀測

```
GET https://data.weather.gov.hk/weatherAPI/opendata/weather.php?dataType=rhrread&lang=tc
```

返回即時氣溫、濕度、天氣圖標等。

### 熱帶氣旋/惡劣天氣警告

```
GET https://data.weather.gov.hk/weatherAPI/opendata/weather.php?dataType=warningInfo&lang=tc
```

## 天氣對行山的影響

### 溫度與濕度
| 溫度 | 濕度 | 行山建議 |
|------|------|----------|
| 15-25°C | 50-75% | 最佳行山天氣 |
| 25-30°C | 75-85% | 需多補水，避免長時間暴曬 |
| >30°C | >85% | 高溫高濕，建議清晨出發，量力而為 |
| <10°C | - | 需注意保暖，山頂可能有霧 |

### 能見度
- 能見度 > 10km：視野開闊，拍照最佳
- 能見度 5-10km：尚可，部分遠景模糊
- 能見度 < 5km：有霧或煙霞，導航需謹慎

### 降雨
- 小雨（< 10mm/小時）：可繼續行山，注意防滑
- 中至大雨（10-30mm/小時）：建議暫停，尋找遮蔽
- 暴雨（> 30mm/小時）：立即停止，山洪暴發風險

## 警告信號與行山建議

### 颱風信號
| 信號 | 意義 | 行山建議 |
|------|------|----------|
| 1號 | 熱帶氣旋在800公里內 | 可行山，密切注意天氣 |
| 3號 | 烈風或暴風 | 不建議行山 |
| 8號或以上 | 烈風 | **禁止行山，留在室內** |

### 暴雨警告
| 信號 | 意義 | 行山建議 |
|------|------|----------|
| 黃色 | 預料1小時內降雨量達30mm | 謹慎行山，注意積水 |
| 紅色 | 預料1小時內降雨量達50mm | **不建議行山** |
| 黑色 | 預料1小時內降雨量達70mm | **立即下山，尋找遮蔽** |

### 山火危險警告
- 黃色/紅色山火危險警告期間，避免在山野生火

## 各季節行山指南

### 秋季（10月-12月）⭐最佳季節
- 溫度：15-25°C
- 濕度：較低
- 特色：天高氣爽，能見度高，是行山最佳季節
- 推薦路線：麥理浩徑全程、衛奕信徑、任何路線

### 冬季（1月-3月）⭐推薦
- 溫度：10-18°C
- 特色：涼爽舒適，偶有霧氣
- 注意：山頂可能有霧，帶備外套
- 推薦路線：港島徑、鳳凰徑（大東山芒草）

### 春季（4月-5月）⚠️注意
- 溫度：18-26°C
- 特色：回南天，濕度高，能見度低
- 注意：路面濕滑，山頂多雲霧
- 建議：選擇較易路線，注意防滑

### 夏季（6月-9月）⚠️謹慎
- 溫度：28-33°C
- 特色：炎熱潮濕，颱風季節
- 注意：中暑風險高，颱風頻繁
- 建議：早出早回（6-9點出發），選短線，多補水
- 不建議：長距離高難度路線

## 行山安全守則

### 出發前準備
1. 查看天氣預報，確認無惡劣天氣警告
2. 告知家人或朋友行山路線和預計返回時間
3. 充電手機，下載離線地圖（推薦：Wikiloc、AllTrails）
4. 準備足夠食水（每人每小時至少500ml）
5. 帶備急救包、雨具、充電器

### 裝備建議
- 鞋：防滑行山鞋（必備）
- 衣物：排汗速乾衣物，備防風外套
- 食水：每人至少2公升（夏季3公升）
- 食物：高能量零食
- 地圖：下載離線地圖或帶紙質地圖

### 行山中注意事項
- 保持隊伍整齊，不落單
- 注意路標，避免迷路
- 遇到山洪，立即向高處撤離
- 遇到山火，向逆風方向逃離
