| 數據集 | 說明 | 官方頁面 |
|--------|------|----------|
| 露營地點 | 41個指定露營地詳情 | https://data.gov.hk/tc-data/dataset/hk-afcd-afcdlist-campsite |

> **注意**：以上數據集的直接 API 端點需在 `data.gov.hk` 頁面查看最新資源 URL，格式可能為 JSON 或 CSV 下載。

直接到訪漁護署官網獲取最新設施資訊：

- 各個露營地詳情：`https://www.afcd.gov.hk/tc_chi/country/cou_vis/cou_vis_cam/cou_vis_cam_cam/cou_vis_cam_{編號}_{英文名}.html`

**鉛礦坳營地詳情**（麥理浩徑第8段+衛奕信徑第7段交匯）：
- 位置：城門郊野公園鉛礦坳
- 設施：燒烤爐、檯、櫈、污水穴、涼亭
- 特色：步行30分鐘可達草山山頂，遠眺北區風景
- 前往：由城門菠蘿壩或大埔元墩下前往

**浪茄灣營地詳情**（麥理浩徑第1-2段）：
- 位置：西貢東郊野公園浪茄灣
- 設施：基本（無烤爐）
- 水源：溪澗（季節性，旱季需自備）
- 特色：香港最美海灘之一，水清沙幼
- 前往：沿麥理浩徑第1段步行約3.5小時

**鹹田灣營地詳情**（麥理浩徑第2段）：
- 位置：西貢東郊野公園鹹田灣
- 設施：洗手間（距營地約200米）
- 水源：水喉
- 特色：大浪四灣之一，可遠眺蚺蛇尖（468米）
- 前往：沿麥理浩徑第2段

## 郊野公園概覽

香港現有 **24 個郊野公園**，佔全港面積約 40%，由漁農自然護理署（漁護署）管理。

**衛奕信徑**：穿越8個郊野公園（由南至北）
- 大潭郊野公園 → 香港仔郊野公園 → 獅子山郊野公園 → 金山郊野公園 → 城門郊野公園 → 大帽山郊野公園（部分）→ 八仙嶺郊野公園

**港島徑**：穿越港島多個郊野公園
- 薄扶林郊野公園 → 香港仔郊野公園 → 大潭郊野公園

**鳳凰徑**：大嶼山郊野公園
- 南大嶼郊野公園 → 大嶼山郊野公園（北大嶼）→ 大嶼山郊野公園（南大嶼）

### 郊野公園設施類型

| 設施 | 說明 |
|------|------|
| 標準洗手間 | 沖水式，有洗手盤 |
| 旱廁 | 非沖水式，較基本 |
| 水喉 | 自來水，全年供應 |
| 溪澗（季節性）| 雨季有水，旱季可能斷流，需自備 |
| 燒烤爐 | 部分營地提供，需自備炭 |
| 加水站 | 郊野公園內設有多個飲用水加水站 |

## 露營注意事項

### 預約要求
- **荃錦營地**（第 24 號）：需提前預約，詳情見漁護署官網
- 其他營地：一般無需預約，先到先得

### 水源注意事項
- 水喉營地：全年有自來水供應
- 溪澗（季節性）營地：**旱季（11月-4月）可能無水**，必須自備足夠飲用水
- 建議每人每天至少準備3公升飲用水

### 露營守則
- 只可在指定露營地紮營，違者可被檢控
- 禁止在郊野公園內生火（除指定燒烤區）
- 離開時帶走所有垃圾
- 保持安靜，不影響其他露營者
