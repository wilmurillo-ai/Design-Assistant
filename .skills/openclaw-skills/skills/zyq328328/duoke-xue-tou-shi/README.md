# 多科学透视 Multi-disciplinary Perspective

> 用多个学科的视角透视任意想法 🌈

## 简介

多科学透视是一个 OpenClaw Skill，通过多学科视角对任意想法、对话或笔记进行深度透视分析，帮助发现"未知的未知"，拓展认知边界。

## 功能

- 💎 提取核心洞察和金句
- 🌈 从认知心理学、历史哲学、教育学、系统思维等多维度分析
- 🔷 发现意想不到的历史故事和经典连接
- ✍️ 提供差异化写作方向
- 💡 引发深度探索问题

## 安装

```bash
clawhub install duoke-xue-tou-shi
```

## 使用方法

在对话中直接粘贴你想要分析的内容，例如：

> "我最近在思考一个问题：为什么孩子学东西比大人快？"

然后我会用多科学透视从多个维度为你分析。

## 分析维度

1. **认知心理学** — 元认知、抽象思维、隐喻理解
2. **历史哲学** — 时代精神、思想实验、历史先例
3. **教育学** — 苏格拉丁式提问、深度学习、学习动机
4. **系统思维** — 多维思考、反馈循环、涌现效应
5. **中国传统文化** — 儒道佛思想、古典典故
6. **意外连接** — 跨学科的有趣类比

## 示例

输入：
```
我和孩子聊历史，他说"历史很有风韵，就像乐谱有波折和平坦"。
```

输出：
```
💎 核心洞察
"历史很有风韵，就像乐谱有波折和平坦"

🌈 认知光谱分析
- 认知心理学：8岁孩子已具备抽象比喻能力
- 历史哲学：波折/平坦暗合"文明兴衰周期论"
- 教育学：自由追问激发了深度类比
- 系统思维：历史不是直线进步，而是波浪式演进
- 中国传统文化：阴阳之道，循环往复

...（完整分析）
```
