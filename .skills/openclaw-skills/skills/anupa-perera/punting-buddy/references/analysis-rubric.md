# Analysis rubric

With source A only, treat predictions as a strong card read, not a hard model.

## Main signals to weigh

### Trainer
Use as a major signal when:
- juvenile races
- maidens and novices
- first-time runners
- races where readiness matters more than exposed form

Questions to ask:
- Is this the sort of yard that can have one ready first time?
- Does the runner look like a serious setup or just a participant?

### Jockey
Use as a seriousness and intent signal.

Questions to ask:
- Is this a strong booking for this level?
- Does the rider make the horse look more likely?
- Does it suggest stable confidence?

### Draw
Matters most in:
- sprints
- large fields
- courses where track position can matter

Do not overstate it when the race shape is unclear.

### Race conditions
Always check:
- distance
- going
- surface
- class
- age band
- race type

Questions to ask:
- Does the profile look right for this setup?
- Is this a speed test, stamina test, or messy unknown?

### Pedigree clues
Use especially when form is missing or light.

Questions to ask:
- Does the sire or damsire suggest speed or stamina?
- Does the profile look suitable for a sharp debut?
- Does the horse look likely to improve for time?

### Field shape
Think about:
- number of runners
- whether the race looks open or top-heavy
- whether this is a straightforward card read or a complete guess-fest

## How to talk about runners

Use natural labels like:
- the obvious one
- the solid one
- the danger
- the lively outsider
- the one I would not leave out
- the one with the sexy profile but a catch

## Confidence handling

When confidence is low, say it.
Examples:
- "This is a baby race, so we are reading the setup more than hard evidence."
- "I can make a case, but I would not pretend this is a clean data-backed edge."
- "This feels more likely than proven."

## Good answer shape

For a single-race question:
1. Say who catches the eye first.
2. Say why, using concrete card facts.
3. Say the catch.
4. Mention the main danger or alternative and why.
5. End with a clear practical view.

## Minimum explanation rule

If you suggest a runner, include at least 2 or 3 real reasons when available.
Good reasons include:
- trainer angle
- jockey angle
- draw angle
- conditions angle
- pedigree angle
- market shape angle

Bad version:
- `I like Horse X.`

Better version:
- `I like Horse X because the yard is good with this type, the jockey booking looks serious, and the draw should help in a race like this.`

## Things to avoid

Avoid fake precision like:
- exact win percentages
- implied edge claims
- confidence numbers that sound scientific

Only use harder quantitative language if a richer source is actually available later.
