# 期货早报生成工具

> 一句话生成专业期货/黑色产业早报，输入网址或留空即可，自动抓取最新资讯，输出精美 HTML 页面。

---

## 功能特点

- **零配置开箱即用**：留空自动使用西本资讯，无需手动查找网址
- **自动定位最新早报**：精准匹配"X月X日行业要闻早餐"最新一期
- **内容全面**：整体综述 + 重点关注 + 行情表格 + 宏观要闻 + 产业要闻
- **行情表格丰富**：覆盖螺纹/热卷/铁矿/焦煤/焦炭主力合约 + 现货价格 + 股指期货
- **样式精美**：金棕色主色调、展开式新闻列表、中文排版友好
- **数据来源纯净**：仅抓取用户提供页面，全文不显示来源说明

## 触发词

```
生成早报 / 期货早报 / 早间新闻 / 行业要闻早餐 / 早班车 / 卓钢早报
```
