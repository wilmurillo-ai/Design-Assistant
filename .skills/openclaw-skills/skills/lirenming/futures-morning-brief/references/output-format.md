# 期货早报 HTML 输出格式规范

## 整体结构

早报 HTML 由以下七个部分组成，按顺序排列：

1. **顶部横幅区** - 品牌 Banner 图
2. **整体综述** - 综合分析段落（200字左右），放在顶部横幅之后第一个内容版块
3. **重点关注** - 3条精选资讯，带下划线加粗标题 + 详情段落
4. **信息速览** - 行情概述文字 + 品种行情数据表格（品种/收盘价/涨跌/幅度四列）
5. **宏观要闻** - 10条，**标题列表 + 每条展开标题 + 内容详情**
6. **产业要闻** - **6条**，**标题列表 + 每条展开标题 + 内容详情**
7. **页脚** - 卓钢链黑色研究院联系信息

**重要：全文不显示任何数据来源、来源等说明文字。**

## 关键样式规范

### 配色
- 主色调：#d2bd8b（金棕色）
- 标题文字色：rgb(117, 79, 68)（棕色）
- 正文色：rgb(51, 51, 51)
- 宏观要闻标题色：rgb(117, 79, 68)（棕色）
- 产业要闻标题色：rgb(79, 129, 189)（蓝色）
- 背景色：rgb(238, 238, 238)（灰白）

## 注意事项

1. 只使用用户提供的网站内容，不搜索其他来源
2. 行情表格涨跌：正数 → 红色 rgb(204,0,0)；负数 → 绿色 rgb(0,128,0)
3. 宏观/产业要闻：**每条新闻直接展开为标题+内容**，不使用折叠、不使用滚动区
4. 整体综述放在顶部横幅之后第一位
5. 产业要闻只显示6条
6. 全文不显示任何数据来源、来源等说明文字
