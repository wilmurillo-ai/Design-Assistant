# Virtual Try-On

**Virtual Try-On** transforms clothing images into professional e-commerce product photos with AI models wearing the garments. Upload up to 4 clothing images and receive a high-quality product photo ready for e-commerce platforms.

---

## ✨ Features

- **Multi-Garment Processing**: Upload up to 4 clothing images simultaneously
- **AI Model Integration**: Automatically generates product photos with realistic model presentations
- **E-Commerce Ready**: Produces professional-quality images optimized for online retail
- **Flexible Input Order**: Image upload order doesn't matter
- **Batch Generation**: Get a professional product photo from your inputs
- **Async Execution**: Robust asynchronous API handling for complex AI generation tasks

---

## 🚀 How It Works

1. **Prepare Clothing Images**:
   - Upload 1-4 clothing/garment images
   - Images should be clear product photos or design mockups
   - Order of upload doesn't matter

2. **Submit to Workflow**:
   - Send images to the Virtual Try-On API
   - Workflow processes all inputs simultaneously

3. **Get Results**: Receive a professional e-commerce product photo with AI models wearing your garments

## 🧪 Example

### Input
```json
{

  "clothing_2": "https://example.com/pants-design.png",
  "clothing_3": "https://example.com/jacket-design.png",
  "clothing_4": "https://example.com/accessories.png"

```

**What happens:**
- The workflow analyzes each clothing image
- AI models are dressed with your garments
- A professional product photo is generated
- You receive an e-commerce ready image
