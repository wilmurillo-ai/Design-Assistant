# 答案格式规范

## 图片要求

### 推荐格式
- PNG 或 JPEG
- 分辨率不低于 1024x768
- 文字清晰可读

### 拍摄建议
- 光线均匀，避免阴影
- 保持水平，不要倾斜
- 包含题号和答案

## 答案识别格式

### 支持的格式

**格式 1：编号 + 答案**
```
1. 答案内容
2. 答案内容
3. 答案内容
```

**格式 2：括号格式**
```
1) 答案内容
2) 答案内容
3) 答案内容
```

**格式 3：顿号分隔**
```
1、答案内容
2、答案内容
3、答案内容
```

### 答案内容示例

数学答案示例：
```
1. 125
2. x = 3
3. 36平方厘米
4. 2/3
5. 180度
```

## 注意事项

1. **每题一行**：一个答案占一行
2. **题号连续**：题号应该是连续的（1, 2, 3...）
3. **避免复杂格式**：OCR 对复杂格式识别可能不准确
4. **检查识别结果**：AI 识别后请老师确认答案是否正确
