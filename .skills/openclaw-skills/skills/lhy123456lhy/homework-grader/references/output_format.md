# 输出格式说明

## Excel 统计表格式

### Sheet 1: 学生统计

| 列名 | 说明 | 示例 |
|------|------|------|
| 姓名/序号 | 学生姓名或序号 | 张三 / 学生1 |
| 正确题数 | 做对的题目数量 | 8 |
| 错误题数 | 做错的题目数量 | 2 |
| 正确率 | 正确率百分比 | 80% |
| 错题列表 | 错误的题号列表 | [1, 5] |

### Sheet 2: 错题统计

| 列名 | 说明 | 示例 |
|------|------|------|
| 题号 | 题目编号 | 第1题 |
| 错题人数 | 错误该题的人数 | 15 |
| 错误率 | 错误率百分比 | 75% |
| 是否重点 | 是否为重点错题 | ⭐ 重点 |

### Sheet 3: 班级概况

| 项目 | 说明 | 示例 |
|------|------|------|
| 统计时间 | 生成报表的时间 | 2024-01-15 14:30 |
| 班级人数 | 总学生数 | 20 |
| 题目总数 | 总题目数 | 10 |
| 班级平均正确率 | 班级整体正确率 | 75% |
| 重点错题数量 | 错误率≥阈值的题目数 | 3 |
| 重点错题题号 | 重点错题列表 | 第1题, 第5题, 第8题 |

## PDF 报告格式

### 页面结构

1. **标题区**：报告标题 + 生成时间
2. **班级概况**：人数、题目数、平均正确率、重点错题数
3. **错题统计表**：所有题目的错误率统计
4. **重点错题讲解建议**：错误率≥阈值的题目 + 讲解建议
5. **学生得分明细**：每个学生的得分情况

### 重点标记

- 错误率 ≥ 配置阈值（默认40%）的题目
- 在表格中用红色/高亮标记
- 题目后显示 "⭐ 重点" 标签

## JSON 数据格式

### 学生数据 (students_data.json)

```json
[
  {
    "name": "张三",
    "index": "学生1",
    "answers": [
      {"question": 1, "answer": "125", "correct": true},
      {"question": 2, "answer": "x=3", "correct": false}
    ]
  }
]
```

### 正确答案 (correct_answers.json)

```json
[
  {"question": 1, "answer": "125"},
  {"question": 2, "answer": "x=3"}
]
```

### 统计数据 (stats.json)

```json
{
  "students": [...],
  "questions": {
    "1": {"incorrect_count": 5, "total": 20, "error_rate": 25.0}
  },
  "total_questions": 10,
  "total_students": 20
}
```
