# 全方位智能股票/ETF分析助手

自动执行六维分析：基本面 → 技术面 → 资金面 → 消息面 → 综合评分 → 操作建议

## 使用方法

直接在对话中输入股票代码或名称，例如：

- "分析一下 515050"
- "帮我看看 600519"
- "腾讯还能买吗"
- "贵州茅台的操作建议"
- "帮我诊断一下宁德时代"

## 分析维度

1. **基本信息** — 价格、涨跌幅、52周高低、年内涨跌、跟踪指数
2. **技术面** — 均线多头排列、MACD、RSI、KDJ、布林带、成交量、动量、威廉指标
3. **资金面** — OBV能量潮、资金流向、量价关系
4. **消息面** — 最新政策、行业动态、成分股表现、宏观背景
5. **综合评分** — 六维度量化打分（1-5星）
6. **操作建议** — 明确给出：持仓者/空仓者/短线者的具体操作建议

## 数据来源

- **yfinance**: 美股、港股、A股ETF
- **akshare**: A股个股行情
- **tushare**: A股基本面数据
- **Web搜索**: 最新政策和行业消息

## 支持的品种

- A股（上交所 6开头、深交所 0/3开头）
- 港股（5位数股票代码）
- 美股（字母代码如 AAPL、TSLA）
- 全市场ETF
