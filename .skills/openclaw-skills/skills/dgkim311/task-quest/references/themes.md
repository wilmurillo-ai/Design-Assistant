# Task Quest — 테마 시스템

`quest-state.md`의 `theme` 필드로 칭호 분위기를 선택.
현재 레벨에 맞는 칭호를 `title` 필드에 저장.

## 테마 개요

| 테마 ID | 분위기 | 특징 |
|---------|--------|------|
| `rpg` | 판타지 모험 | 견습생 → 기사 → 레전드. 클래식 RPG 감성 |
| `space` | SF 탐험 | 훈련생 → 파일럿 → 제독. 우주 개척 감성 |
| `academic` | 학술 성장 | 학부생 → 박사 → 석학. 연구자에게 특히 잘 맞음 |
| `minimal` | 숫자 중심 | Lv.1 → Lv.10 → Lv.20. 군더더기 없이 깔끔 |

## 테마 변경 절차

1. 사용자 요청 파싱: "테마 바꿔줘 → 우주" / "academic으로 변경" 등
2. theme ID 매핑:
   - "rpg" / "판타지" / "모험" → `rpg`
   - "space" / "우주" / "SF" → `space`
   - "academic" / "학술" / "학교" / "연구" → `academic`
   - "minimal" / "미니멀" / "숫자" / "심플" → `minimal`
3. quest-state.md 업데이트:
   - `theme` 필드 변경
   - `title` 필드를 현재 level에 맞는 새 테마 칭호로 변경
4. 확인 메시지:
   > "테마를 'academic'으로 변경했습니다. 현재 칭호: 박사 (Level 9)"
