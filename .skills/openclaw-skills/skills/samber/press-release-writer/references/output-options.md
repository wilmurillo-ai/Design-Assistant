# Output Options

Read this file when deciding on the delivery format for the press release.

## Available Formats

### Markdown (default)

Clean, readable, copy-pasteable. Use when the user will paste into a CMS, email client, or word processor.

### Word Document

Use the `docx` skill if available for formal distribution. Professional formatting with company letterhead template if provided.

### Email-Ready Format

Press release pasted directly in the email body (never as attachment) with a 2-3 line pitch above it. See [journalist email pitch](journalist-email-pitch.md) for the pitch structure and hook types.

### Bilingual

Side-by-side or sequential FR/EN (or other language pairs) for international distribution. Each version is culturally adapted, not word-for-word translated. See [regional conventions](regional-conventions.md) for localization rules.

### Press Kit

Complete package for major announcements:

1. The press release (plain text + formatted PDF)
2. High-res images with captions
3. Executive bios and headshots
4. Company factsheet (founding date, HQ, headcount, key metrics)
5. Product screenshots or demo video
6. FAQ document
7. Previous relevant coverage (2-3 top clips)
8. Brand guidelines / logo files
