# 六层架构工作空间规则

## 架构概述

本工作空间采用六层架构设计：

| 层级 | 名称 | 职责 |
|------|------|------|
| L1 | 核心认知层 | 身份、规则、引导 |
| L2 | 记忆上下文层 | 存储、检索、日记 |
| L3 | 任务编排层 | 路由、工作流、调度 |
| L4 | 能力执行层 | 技能执行、结果验证 |
| L5 | 稳定治理层 | 安全、权限、审计 |
| L6 | 基础设施层 | 路径、插件、性能 |

## 使用说明

1. 每个层级有明确的职责边界
2. 层级之间通过标准接口通信
3. 所有操作遵循安全规范
