# Taxonomy

## Fixed domains

Use exactly these domains:
- strategy
- business
- organization
- finance
- legal
- project
- operations
- tech
- routines
- personal
- meta
- misc

## Module policy

- Modules are reusable topic buckets under a domain.
- Do not precreate a full module tree.
- Create modules only when repeated use justifies them.
- Reuse existing names whenever possible.
- Avoid synonym drift.

## Entity vs module vs free tag

Use a module when the concept is a reusable topic bucket.
Use an entity when the concept is a stable object: person, project, task, system, platform, customer, policy.
Use a free tag when the concept is specific, temporary, or too granular for a reusable module.
