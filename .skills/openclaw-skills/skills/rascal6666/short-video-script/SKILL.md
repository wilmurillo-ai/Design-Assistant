# 短视频脚本创作技能

## 概述

为短视频平台创作高效脚本，提升完播率、互动率和转化率。

## 适用场景

- 抖音/快手/视频号/B站 脚本创作
- 知识类干货分享
- 热点蹭流内容
- 产品推广文案
- 剧情/情感类内容

## 脚本结构

### 黄金3秒开头（抓住注意力）

```
公式1：提问式 - "你知道...吗？"
公式2：夸张式 - "99%的人都不知道..."
公式3：冲突式 - "年薪百万的工作居然是..."
公式4：结果前置 - "3分钟学会..." + 悬念
```

### 核心内容结构

```
[开场3秒] → [痛点/悬念] → [干货/情节] → [高潮/反转] → [结尾引导]
```

### 结尾引导

```
- "关注我，带你了解更多..."
- "下期讲..."
- "评论区告诉我你的想法"
- "点赞收藏防丢失"
```

## 脚本模板

### 知识类模板

```
【标题】：[吸引眼球的标题]

【时长】：1-3分钟

【开头3秒】：
[提出问题/痛点/悬念]

【正文结构】：
1. [核心知识点1] - 30秒
2. [核心知识点2] - 30秒  
3. [核心知识点3] - 30秒

【结尾】：
[总结+引导互动]
```

### 热点类模板

```
【标题】：[热点相关+悬念]

【开头3秒】：
[热点引入+自己的观点]

【正文】：
1. [热点简述] - 10秒
2. [深度分析] - 40秒
3. [独特观点] - 30秒

【结尾】：
[引导评论讨论]
```

## 平台特性

### 抖音
- 15秒-3分钟为主
- 节奏快，金句多
- BGM很重要
- 完播率优先

### 快手
- 较抖音更接地气
- 老铁文化
- 真实性重要

### B站
- 可以更长（5-15分钟）
- 内容深度要求高
- 弹幕互动

### 视频号
- 社交推荐为主
- 私域引流
- 适合知识类

## 标题公式

```
数字型：3个技巧、5个方法
疑问型：为什么...？怎么做...？
悬念型：原来...真相是...
对比型：月薪3千vs3万的差距
冲突型：千万不要...否则...
```

## 封面设计建议

```
- 文字简洁有力
- 突出关键词
- 色彩对比强
- 人物+文字组合
```

## 创作流程

1. **确定选题**：结合热点/痛点/定位
2. **设计开头**：3秒抓住注意力
3. **搭建结构**：逻辑清晰
4. **填充内容**：金句/干货/情感
5. **优化结尾**：引导互动
6. **检查节奏**：控制时长

## 注意事项

- 内容要有价值，避免纯标题党
- 遵守平台规则，避免违规词
- 保持更新频率
- 数据驱动优化
- 真人出镜效果更好

## 常用工具

- 热点查询：抖音热榜、微博热搜
- 数据分析：蝉妈妈、飞瓜数据
- 剪辑工具：剪映、Premiere
