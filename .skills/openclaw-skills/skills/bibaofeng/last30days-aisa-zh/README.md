# last30days-zh

中文 ClawHub 发布包。

包含：

- 运行时脚本
- 中文 `SKILL.md`
- `.codex-plugin/plugin.json`
- 许可证和包元数据

刻意不包含：

- tests
- 历史文档
- hooks
- fixtures
- Gemini 扩展元数据

运行时摘要：

- `AISA_API_KEY` 提供规划、重排、综合、X/Twitter、YouTube、Polymarket 和 grounded web search。
- Reddit 与 Hacker News 走公开路径。
- GitHub 仍走官方 GitHub API 路径，需要时再配置 `GH_TOKEN` 或 `GITHUB_TOKEN`。
