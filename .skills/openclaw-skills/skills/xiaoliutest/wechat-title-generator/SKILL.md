# 公众号标题生成器

> 生成高点击率的微信公众号标题，支持多种爆款技巧和吸引力评分

## 功能

- 📊 **标题吸引力评分** (0-100分)
- 🎯 **10种爆款标题技巧**自动应用
- 📝 **批量生成候选标题**供A/B测试
- 🔍 **根据文章内容智能生成**
- ⚡ **即时生成，无需脚本**

## 使用方法

向 AI 描述你的需求：

```
用 wechat-title-generator 帮我生成公众号标题
主题：[你的文章主题]
类型：[干货/情感/新闻/故事/盘点]
目标受众：[你的目标读者]
```

## 爆款标题技巧

### 1. 数字法
用具体数字增加可信度和吸引力
- "3个方法让你月入过万" (85分)
- "这5本书改变了我的命运" (80分)

### 2. 疑问法
引发好奇心，激发点击欲望
- "为什么90%的人都在犯这个错误？"
- "月薪3千和月薪3万的人，差在哪里？"

### 3. 对比法
制造反差，突出价值
- "从负债50万到年入百万，我只做对了这件事"
- "同样是工作5年，为什么他能买房你还在租房？"

### 4. 悬念法
留下悬念，勾起探索欲
- "那个每天加班的同事，离职后..."
- "我把这份工作辞了，原因是..."

### 5. 借势法
借助热点或名人效应
- "马斯克都在用的时间管理法"
- "看了《狂飙》，我悟出了职场3条铁律"

### 6. 痛点法
直击痛点，引发共鸣
- "总是被领导忽视？你可能犯了这3个错误"
- "存不下钱？这个方法让我3年攒了30万"

### 7. 否定法
用否定语气制造反差
- "别再这样吃早餐了，真的很伤身"
- "别做那个"老好人"，职场真的不吃这一套"

### 8. 故事法
用故事感吸引读者
- "那年我25岁，裸辞后..."
- "一个电话，改变了我的一生"

### 9. 揭秘法
满足窥探欲和求知欲
- "揭秘：那些年薪百万的人，都在偷偷做什么"
- "行业内幕：电商运营不会告诉你的3个真相"

### 10. 干货法
直接展示价值
- "Excel最常用的20个公式，收藏这一篇就够了"
- "新手入门理财，看这篇就够了"

## 标题吸引力评分标准

| 维度 | 权重 | 说明 |
|:---|:---|:---|
| 点击欲望 | 40% | 是否让人想点开看 |
| 相关性 | 20% | 是否匹配目标受众 |
| 独特性 | 20% | 是否新颖、不套路 |
| 可信度 | 10% | 是否让人信服 |
| 传播性 | 10% | 是否让人想分享 |

### 评分等级
- **90-100分**：爆款标题，预期打开率15%+
- **80-89分**：优秀标题，预期打开率10-15%
- **70-79分**：良好标题，预期打开率5-10%
- **60-69分**：合格标题，预期打开率3-5%
- **60分以下**：需要优化

## A/B测试建议

1. 同时准备3-5个候选标题
2. 选择不同技巧组合
3. 根据打开率数据优化
4. 建立"爆款标题库"

## 注意事项

- 标题要匹配内容，避免"标题党"
- 了解目标受众的痛点和兴趣
- 保持人设一致性
- 热点要快速响应
