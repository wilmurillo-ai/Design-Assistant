# 周报复仇者

一键生成专业周报，让汇报不再是噩梦。

## 使用方法

告诉 AI 你需要周报，提供以下信息：

```
生成周报

岗位：[你的职位]
本周做了什么：[简单描述]
重点成果：[可选，重要的成果]
下周计划：[可选，下周要做什么]
风格：[正式/简洁/详细]
```

AI 会自动生成结构清晰、重点突出的专业周报。

---

## 周报类型

### 📋 通用型周报
适用于大多数岗位，包含：
- 本周工作概述
- 重点成果
- 下周计划
- 需要的支持

### 💻 研发型周报
适用于程序员、开发工程师：
- 本周开发内容
- 完成进度
- 技术难点与解决方案
- 下周计划

### 📊 运营型周报
适用于运营、市场岗位：
- 数据指标变化
- 活动执行情况
- 用户反馈
- 下周策略

### 💰 销售型周报
适用于销售、商务岗位：
- 客户跟进情况
- 业绩数据
- 商机管道
- 下周目标

### 📱 项目型周报
适用于项目经理：
- 项目进度
- 风险与问题
- 里程碑完成情况
- 下周计划

---

## 示例

### 输入
```
生成周报
岗位：前端开发
本周做了什么：完成了首页重构，修复了3个bug，参加了2次需求评审
重点成果：首页加载速度从3秒优化到1.5秒
风格：简洁
```

### 输出
```
## 本周工作总结

### 完成工作
- ✅ 首页重构完成，加载速度优化50%
- ✅ 修复线上bug 3个
- ✅ 参与需求评审会议2次

### 重点成果
首页性能优化成效显著：
- 加载时间：3s → 1.5s
- 用户体验大幅提升

## 下周计划
1. 完成用户中心页面开发
2. 配合测试完成首页回归测试
3. 准备技术分享材料

需要协调：UI设计资源确认
```

## 技巧

### 输入提示
- **简单描述即可**：不用写得很详细，AI会帮你润色
- **数据更直观**：提供具体数字，周报更有说服力
- **问题提前说**：需要协调的资源可以主动提出

### 常见岗位关键词
- **程序员**：开发、上线、修复、优化、重构
- **运营**：增长、转化、活动、用户、留存
- **销售**：客户、业绩、跟进、商机、合同
- **设计**：设计稿、改版、规范、评审、交付

---

## 输出特点

- **结构清晰**：固定格式，领导一眼看懂
- **数据可视化**：用数字说话，更有说服力
- **成果突出**：自动提炼亮点工作
- **风格可控**：支持正式/简洁/详细三种风格
