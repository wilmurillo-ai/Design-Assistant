# 主题样式参考

本文档介绍 md-wechat 支持的三种预设主题及其样式特点。

---

## 主题概览

| 主题 | 名称 | 设计者 | 适用场景 |
|-----|------|--------|---------|
| default | 经典主题 | md 团队 | 技术文章、教程、文档 |
| grace | 优雅主题 | @brzhang | 散文、游记、生活方式 |
| simple | 简洁主题 | @okooo5km | 科技资讯、产品介绍、报告 |

---

## 经典主题 (default)

### 设计理念

经典主题采用传统排版风格，层次分明，适合技术类文章。二级标题使用主题色背景突出显示，三级标题使用左侧边框强调。

### 样式特点

**标题样式**：
- H1：居中显示，底部主题色边框
- H2：居中显示，主题色背景白字
- H3：左侧主题色边框
- H4-H6：主题色文字

**代码块**：
- 深色背景（#282c34）
- Mac 风格装饰（红黄绿圆点）
- 圆角边框

**引用块**：
- 浅灰背景
- 左侧主题色边框
- 圆角设计

**列表**：
- 无序列表使用空心圆点
- 有序列表使用数字

### CSS 关键代码

```css
/* 二级标题 - 主题色背景 */
h2 {
  display: table;
  padding: 0.3em 1em;
  margin: 2em auto 1em;
  color: #fff;
  background: var(--md-primary-color);
  font-size: 1.2em;
  font-weight: bold;
  text-align: center;
}

/* 三级标题 - 左边框 */
h3 {
  padding-left: 8px;
  border-left: 3px solid var(--md-primary-color);
  margin: 1.5em 8px 0.75em 0;
}
```

### 效果预览

```
┌─────────────────────────────────────┐
│           一级标题                   │  ← 底部边框
├─────────────────────────────────────┤
│           【二级标题】               │  ← 主题色背景
├─────────────────────────────────────┤
│  ▌三级标题                          │  ← 左边框
│                                     │
│  正文内容，段落之间有适当间距。      │
│                                     │
│  ● 列表项一                         │
│  ● 列表项二                         │
└─────────────────────────────────────┘
```

## 优雅主题 (grace)

### 设计理念

优雅主题由 @brzhang 设计，注重视觉美感，使用阴影和圆角营造精致感，适合散文、游记等生活类文章。

### 样式特点

**标题样式**：
- H1：底部边框 + 文字阴影
- H2：主题色背景 + 阴影效果
- H3：左边框 + 底部虚线边框

**代码块**：
- 内嵌阴影效果
- 推荐使用 Fira Code 字体

**引用块**：
- 斜体文字
- 阴影效果
- 圆角设计

**图片**：
- 圆角边框
- 阴影效果

**分隔线**：
- 渐变效果（中间深两端浅）

**表格**：
- 分离边框
- 圆角设计
- 阴影效果
- 表头主题色背景

### CSS 关键代码

```css
/* 二级标题 - 阴影效果 */
h2 {
  padding: 0.3em 1em;
  border-radius: 8px;
  box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
}

/* 三级标题 - 虚线边框 */
h3 {
  border-left: 4px solid var(--md-primary-color);
  border-bottom: 1px dashed var(--md-primary-color);
}

/* 引用块 - 斜体 + 阴影 */
blockquote {
  font-style: italic;
  box-shadow: 0 4px 6px rgba(0, 0, 0, 0.05);
}

/* 分隔线 - 渐变 */
hr {
  background: linear-gradient(to right, 
    rgba(0, 0, 0, 0), 
    rgba(0, 0, 0, 0.15), 
    rgba(0, 0, 0, 0));
}
```

### 效果预览

```
┌─────────────────────────────────────┐
│           一级标题                   │  ← 底部边框 + 阴影
├─────────────────────────────────────┤
│           【二级标题】               │  ← 圆角 + 阴影
├─────────────────────────────────────┤
│  ▌三级标题                          │  ← 左边框 + 虚线底
│  ───────────────                    │
│                                     │
│  "引用块使用斜体样式                 │  ← 斜体 + 阴影
│   更加优雅动人"                      │
│                                     │
│  ──────── 渐变分隔线 ────────       │
└─────────────────────────────────────┘
```

## 简洁主题 (simple)

### 设计理念

简洁主题由 @okooo5km 设计，追求极简风格，使用微妙的边框和阴影，视觉效果干净清爽，适合科技资讯和产品介绍。

### 样式特点

**标题样式**：
- H1：简洁无装饰
- H2：不规则圆角（8px 24px 8px 24px）
- H3：左边框 + 浅色背景框

**代码块**：
- 细边框
- 浅灰背景

**引用块**：
- 斜体文字
- 微妙的四边边框
- 浅灰背景

**图片**：
- 圆角
- 细边框

**列表**：
- 无序列表使用空心圆点（◦）
- 无实心圆点

### CSS 关键代码

```css
/* 二级标题 - 不规则圆角 */
h2 {
  border-radius: 8px 24px 8px 24px;
  box-shadow: 0 2px 6px rgba(0, 0, 0, 0.06);
}

/* 三级标题 - 背景框 */
h3 {
  border-left: 4px solid var(--md-primary-color);
  border-right: 1px solid rgba(0, 0, 0, 0.05);
  border-bottom: 1px solid rgba(0, 0, 0, 0.05);
  border-top: 1px solid rgba(0, 0, 0, 0.05);
  background: rgba(0, 0, 0, 0.02);
}

/* 引用块 - 微妙边框 */
blockquote {
  border-bottom: 0.2px solid rgba(0, 0, 0, 0.04);
  border-top: 0.2px solid rgba(0, 0, 0, 0.04);
  border-right: 0.2px solid rgba(0, 0, 0, 0.04);
}

/* 列表项 - 空心圆点 */
ul li::before {
  content: "◦";
  color: var(--md-primary-color);
}
```

### 效果预览

```
┌─────────────────────────────────────┐
│           一级标题                   │  ← 简洁无装饰
├─────────────────────────────────────┤
│         ╭──────────────╮            │
│         │  二级标题    │            │  ← 不规则圆角
│         ╰──────────────╯            │
├─────────────────────────────────────┤
│  ┌───────────────────────┐          │
│  │ ▌三级标题             │          │  ← 背景框
│  └───────────────────────┘          │
│                                     │
│  ◦ 列表项使用空心圆点                │  ← ◦ 符号
└─────────────────────────────────────┘
```

## 如何选择主题

### 按文章类型选择

| 文章类型 | 推荐主题 | 原因 |
|---------|---------|------|
| 技术教程 | 经典 | 代码块突出，层次清晰 |
| 编程笔记 | 经典 | 结构化强，信息密度高 |
| 散文游记 | 优雅 | 视觉柔和，阅读舒适 |
| 生活方式 | 优雅 | 美观精致，符合调性 |
| 科技资讯 | 简洁 | 干净利落，信息优先 |
| 产品介绍 | 简洁 | 现代简约，专业感强 |
| 学术报告 | 经典 | 正式规范，结构清晰 |

### 按受众选择

| 目标读者 | 推荐主题 | 建议 |
|---------|---------|------|
| 程序员 | 经典 | 代码高亮，技术感强 |
| 普通读者 | 优雅 | 阅读体验好，不累眼 |
| 年轻群体 | 简洁 | 设计现代，符合审美 |
| 商务人士 | 经典/简洁 | 专业正式 |

## 自定义主题

如果预设主题不能满足需求，可以通过以下方式自定义：

### 方法一：修改配置

```json
{
  "theme": { "name": "default" },
  "style": {
    "primaryColor": "#009874",
    "fontSize": "17px"
  },
  "headingStyles": {
    "h1": "border-bottom",
    "h2": "color-only",
    "h3": "border-left"
  }
}
```

### 方法二：添加自定义 CSS

```json
{
  "customCSS": "
    blockquote {
      background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
      color: #fff;
      border: none;
    }
    
    h2 {
      background: linear-gradient(90deg, #667eea, #764ba2);
    }
  "
}
```

### 方法三：创建新主题

1. 在 `assets/themes/` 目录创建新 CSS 文件
2. 修改 `convert.py` 中的 `THEME_MAP` 添加新主题
3. 在配置中使用新主题名称

## CSS 变量

所有主题共用以下 CSS 变量，可通过配置修改：

```css
:root {
  --md-primary-color: #0F4C81;  /* 主题色 */
  --md-font-family: ...;         /* 字体 */
  --md-font-size: 16px;          /* 字号 */
}
```

在主题 CSS 中使用：

```css
h2 {
  background: var(--md-primary-color);
  font-size: calc(var(--md-font-size) * 1.2);
}
```
