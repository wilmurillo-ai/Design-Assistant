# 龙虾日记

中文文档 | [English](README.md)

一个 WorkBuddy 技能，用于快速记录日记和备忘内容，自动按年月日组织目录结构。

## 功能特点

- 📝 **快速记录** - 只需说"帮我记一下"即可开始记录
- 📁 **自动归档** - 按年/月/日自动组织目录结构，方便检索
- 🎯 **智能命名** - 自动从内容中提取标题，或使用用户指定的标题
- 🏷️ **标签支持** - 支持为日记添加标签分类
- 💾 **Markdown 格式** - 所有日记以标准 Markdown 格式存储
- 📊 **每日摘要** - 自动生成每日日记的摘要文件

## 目录结构

```
龙虾日记/
└── YYYY/           # 年份目录（如 2026）
    └── MM/         # 月份目录（如 03）
        └── DD/     # 日期目录（如 11）
            ├── README.md     # 每日摘要
            └── *.md          # 具体的日记文件
```

示例：
```
龙虾日记/
└── 2026/
    └── 03/
        └── 11/
            ├── README.md
            ├── 22-00-技能开发.md
            └── 22-30-会议纪要.md
```

## 使用方法

### 基本记录

```
用户：帮我记一下
智能体：好的，请继续说，我会记录下来。说"结束记录"完成记录。
用户：今天完成了技能开发任务，创建了一个龙虾日记技能
用户：结束记录
智能体：已记录完成！日记保存在：龙虾日记/2026/03/11/22-00-技能开发.md
```

### 带标题记录

```
用户：帮我记一下会议纪要
智能体：好的，请继续说，我会记录下来。说"结束记录"完成记录。
用户：今天讨论了新产品的设计方案，确定了三个核心功能
用户：结束记录
```

### 指定工作空间

```
用户：在 Claw 空间帮我记一下
智能体：好的，请继续说，我会记录下来。说"结束记录"完成记录。
用户：我的学习笔记：今天学习了 Python 异步编程
用户：结束记录
```

## 安装方式

### 通过技能中心安装
在 WorkBuddy 技能中心搜索"龙虾日记"并安装。

### 手动安装
将技能包解压到 `.codebuddy/skills/crayfish-diary/` 目录即可。

## 配置

技能可以通过 `clawhub.json` 进行配置：

```json
{
  "name": "crayfish-diary",
  "version": "1.0.0",
  "author": "italks",
  "triggers": ["帮我记一下", "结束记录"]
}
```

## 技术细节

- **技能名称**：crayfish-diary
- **触发词**：
  - 开始："帮我记一下", "help me record", "start recording"
  - 结束："结束记录", "end recording", "finish recording"
- **文件格式**：Markdown (.md)
- **脚本语言**：Python 3

## 每日摘要格式

每天目录下都会自动生成 `README.md` 摘要文件：

```markdown
# 2026-03-11 日记摘要

## 记录列表

1. **22:00 技能开发**: 今天完成了技能开发任务，创建了一个龙虾日记技能...
2. **22:30 会议纪要**: 今天讨论了新产品的设计方案，确定了三个核心功能...

---
*共 2 条记录*
```
