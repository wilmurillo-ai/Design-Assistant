# Sum2Slides Lite 安全使用指南
# 版本: 1.0.1 (安全修复版)

## 🎯 概述

本文档提供 Sum2Slides Lite 的详细安全使用指南，帮助用户安全地安装、配置和使用本软件。

## 🔒 核心安全原则

### **1. 最小权限原则**
```
只授予完成工作所需的最小权限
```

### **2. 透明性原则**
```
所有操作对用户可见，无隐藏行为
```

### **3. 用户控制原则**
```
用户始终控制所有操作和权限
```

### **4. 安全默认原则**
```
默认配置是安全的，需要用户显式启用增强功能
```

## ⚙️ 安全配置指南

### **环境变量安全配置**
```bash
# .env 文件示例 (安全版本)

# 🔐 飞书凭证 (可选)
# 如果不需要上传功能，可以留空
FEISHU_APP_ID=optional_app_id_here
FEISHU_APP_SECRET=optional_app_secret_here

# 📁 输出目录 (必需)
# 建议使用用户有权限的目录
OUTPUT_DIR=~/Desktop/Sum2Slides

# ⚙️ 安全设置 (推荐启用)
SAFE_MODE=true
REQUIRE_CONFIRMATION=true
```

### **安全配置建议**
1. **敏感信息**: 使用环境变量，不要硬编码
2. **输出目录**: 使用用户可控制的目录
3. **网络功能**: 默认禁用，需要时显式启用
4. **自动化功能**: 默认禁用，需要时显式启用

### **3. 实际使用**
```python
from sum2slides import sum2slides

# 安全使用示例
result = sum2slides(
    conversation_text="会议讨论内容...",
    title="安全测试",
    software="powerpoint",
    safe_mode=True,  # 启用安全模式
    require_confirmation=True  # 需要用户确认
)

# 检查结果
if result['success']:
    print(f"✅ PPT生成成功: {result['ppt_info']['file_path']}")
else:
    print(f"⚠️ 生成失败: {result['error']}")
    print(f"   建议: {result.get('suggestion', '请检查配置')}")
```

## ⚠️ 风险识别与防范

### **1. 文件系统风险**
**风险**: 文件被保存到错误位置
**防范**: 
- 明确指定输出目录
- 使用安全模式
- 定期检查输出文件

### **2. 网络风险**
**风险**: 敏感信息通过网络传输
**防范**:
- 使用HTTPS加密传输
- 默认禁用网络功能
- 安全存储API凭证

### **3. 自动化风险**
**风险**: 自动化操作可能出错
**防范**:
- 需要用户确认
- 提供操作预览
- 实现优雅降级

### **4. 依赖风险**
**风险**: 第三方库可能存在安全问题
**防范**:
- 使用知名、维护良好的库
- 固定版本号
- 定期更新依赖
