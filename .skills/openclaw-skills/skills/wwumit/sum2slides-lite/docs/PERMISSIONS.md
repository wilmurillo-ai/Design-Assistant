# Sum2Slides Lite 权限说明与安全指南
# 版本: 1.0.1 (安全修复版)

## 🎯 概述

Sum2Slides Lite 是一个免费的对话总结成PPT工具。本文档详细说明软件所需的权限、安全设计和使用建议。

## 📋 所需权限清单

### **1. 文件系统权限 (必需)**
**用途**: 保存生成的PPT文件
**范围**: 用户指定的输出目录
**安全说明**: 
- 仅写入权限，不读取其他文件
- 用户可以随时更改输出目录
- 不访问系统敏感目录

### **2. 网络权限 (可选)**
**用途**: 飞书云盘文件上传
**范围**: 仅访问飞书官方API (open.feishu.cn)
**安全说明**:
- 需要用户显式启用
- 使用HTTPS加密传输
- 不访问其他网站

### **3. AppleScript自动化权限 (可选, 仅macOS)**
**用途**: 自动化控制PowerPoint/WPS软件
**范围**: 仅控制指定的PPT软件
**安全说明**:
- 需要用户手动在系统设置中授权
- 仅用于PPT生成和保存
- 可以随时撤销权限

## 🛡️ 安全设计

### **1. 最小权限原则**
```
权限要求 = 功能必需的最小权限
```
- 必需权限: 仅文件保存
- 可选权限: 用户控制启用
- 平台特有权限: 明确标注

### **2. 优雅降级机制**
```
权限状态 → 功能可用性
```
| 权限状态 | PPT生成能力 | 结果 |
|---------|------------|------|
| **完全授权** | ✅ 全自动化 | 功能完整，体验最佳 |
| **无AppleScript** | ✅ 标准生成 | 生成标准 .pptx，两种软件都能打开 |
| **无网络权限** | ✅ 本地保存 | 文件保存在桌面，可手动分享 |
| **无文件权限** | ⚠️ 提示更改目录 | 用户选择有权限的目录 |

### **3. 透明操作**

## 🔒 凭证安全处理

### **1. 不硬编码原则**
```python
# ❌ 错误: 硬编码凭证
API_KEY = "sk_live_123456"

# ✅ 正确: 通过环境变量获取
API_KEY = os.getenv("FEISHU_API_KEY", "")
```

### **2. 安全存储建议**
```
✅ 推荐方式:
1. 环境变量 (系统级)
2. .env 文件 (项目级，不上传Git)
3. 密钥管理器 (企业级)

❌ 不推荐方式:
1. 硬编码在代码中
2. 写在配置文件中并上传Git
3. 存储在不安全的位置
```

### **3. 传输安全**
```
传输层: HTTPS (强制)
加密: TLS 1.2+
验证: 服务器证书验证
```

## ⚠️ 风险与限制

### **1. 已知风险**
- **平台限制**: AppleScript仅支持macOS
- **软件依赖**: 需要PowerPoint或WPS安装
- **网络依赖**: 飞书上传需要网络连接

### **2. 安全限制**
- 不执行系统命令
- 不修改系统设置
- 不收集用户数据
- 不访问敏感文件

### **3. 使用限制**
- 最大对话长度: 10,000字符
- 支持的模板: 商务/技术/教育
- 输出格式: .pptx 文件
