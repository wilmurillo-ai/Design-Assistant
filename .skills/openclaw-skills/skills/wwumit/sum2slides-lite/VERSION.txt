Sum2Slides Lite v1.1.6
===================================================

## 🎯 版本说明

### **版本更新: v1.1.5 → v1.1.6**

**主要修复:**
✅ **修复元数据不一致问题**
✅ **明确环境变量的可选性**
✅ **提高安全审查信心等级**

### **问题修复:**
```
1. ❌ 之前: 注册表元数据错误标记FEISHU_*为必需环境变量
2. ✅ 现在: 正确标记为可选环境变量
3. ✅ 澄清: 纯本地模式完全不需要环境变量
4. ✅ 明确: 飞书上传模式需要设置可选环境变量
```

### **核心功能不变:**
```
✅ 纯本地PPT生成功能
✅ 可选飞书上传功能
✅ 智能对话分析
✅ 多种模板支持
```

## 🔒 安全特性

### **环境变量澄清:**
```
✅ FEISHU_APP_ID: 可选 (仅当使用飞书上传时)
✅ FEISHU_APP_SECRET: 可选 (仅当使用飞书上传时)
✅ 纯本地模式: 不需要任何环境变量
```

### **使用模式:**
```
1. 🔒 纯本地模式 (最安全)
   - 不设置任何环境变量
   - 完全无网络活动
   - 数据100%在本地

2. 🌐 飞书上传模式 (可选)
   - 设置FEISHU_APP_ID/SECRET
   - 启用飞书API功能
   - 数据上传到飞书云
```

## 🚀 安装要求

### **基础安装 (纯本地模式):**
```
✅ 不需要任何环境变量
✅ 不需要网络连接
✅ 不需要API凭证
```

### **完整安装 (飞书上传模式):**
```
✅ 设置可选环境变量
✅ 需要网络连接
✅ 需要飞书API凭证
```
