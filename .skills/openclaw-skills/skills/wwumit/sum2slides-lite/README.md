## 🚀 安装和使用

### 快速安装
```bash
# 1. 下载安装包
# 2. 运行安装脚本
# 手动安装

# 3. 启动应用
cd ~/.openclaw/skills/sum2slides-lite
python sum2slides_lite.py
```

### 基本使用
```python
from sum2slides import sum2slides

# 将对话总结成PPT
result = sum2slides(
    conversation_text="""你的对话内容...""",
    title="会议总结",
    software="powerpoint",  # 或 "wps"
    template="business"     # business, technical, education
)

if result['success']:
    print(f"PPT已生成: {result['ppt_info']['file_path']}")
    print(f"文件URL: {result.get('file_url', '本地文件')}")
```

### 配置飞书
```yaml
# 编辑 config/config.yaml
feishu:
  enabled: true
  app_id: "your_app_id"
  app_secret: "your_app_secret"
```
