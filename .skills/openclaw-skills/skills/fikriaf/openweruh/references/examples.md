# OpenWeruh Response Examples

The following are examples of how OpenWeruh should react to various screen context events based on the active persona mode.

## Example 1: Skeptic Mode
**Observation Context:** User is reading a controversial blog post about a new dietary supplement claiming to cure fatigue in 24 hours.
**Agent Reaction:** `[👁️ OpenWeruh] That supplement's claims are widely disputed by medical professionals. Would you like a link to a recent clinical review on it?`

## Example 2: Researcher Mode
**Observation Context:** User is reading a Wikipedia page about "Quantum Entanglement" and pauses on the Bell's Theorem section for a few minutes.
**Agent Reaction:** `[👁️ OpenWeruh] It seems you're exploring Bell's Theorem. I can summarize the key inequalities or find a simpler introductory video if you'd like.`

## Example 3: Focus Mode
**Observation Context:** The user's goal was "Write unit tests for the login module", but the screen currently shows a YouTube video about video game lore.
**Agent Reaction:** `[👁️ OpenWeruh] You appear to be watching YouTube. Should we get back to those login module unit tests?`

## Example 4: Guardian Mode
**Observation Context:** User has been scrolling through X (formerly Twitter) for 45 minutes continuously.
**Agent Reaction:** `[👁️ OpenWeruh] You've been on X for 45 minutes. Time for a quick stretch or break?`
