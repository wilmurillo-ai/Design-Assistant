# 🎯 Expert Mode - 专家模式技能（安全版）

## 简介

Expert Mode 是一个为 OpenClaw 打造的 AI 专家库管理技能。它整合了多个来源的专家 Agent，涵盖工程、设计、营销、销售、产品、游戏开发等20+个领域，共 **186个独立专家**。

> ⚠️ **安全版本** - 针对生产环境优化，添加了安全审查机制

## ⚠️ 安全特性

- ✅ 内容验证：获取外部内容后进行安全审查
- ✅ 写入确认：持久化前需要用户确认
- ✅ 自动同步默认关闭：避免意外重复持久化
- ✅ 只读专家配置：不加载外部可执行指令

## 核心功能

1. **专家列表管理** - 自动获取 + 安全验证
2. **专家激活** - 一键切换身份
3. **分类检索** - 快速查找专家
4. **自定义专家** - 安全添加（仅名称描述）
5. **收藏夹** - 收藏常用专家

## 使用方法

```
激活前端开发者
列出所有营销专家
学习专家模式
添加自定义专家：名字=AI绘画师，专长=Midjourney
```

## ⚠️ 安全说明

- 专家配置仅从本地可信列表加载
- 自定义专家不支持可执行指令
- 自动同步需主动开启，每次同步仍需确认
- 写入MEMORY.md前会显示预览并确认
