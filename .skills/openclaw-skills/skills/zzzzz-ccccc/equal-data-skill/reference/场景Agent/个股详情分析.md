## 场景三：个股详细分析

### 业务价值
对单只股票进行全方位分析，整合行情、财务、股东、资讯、高管行为等多维度数据，形成完整的投资决策依据。

### 涉及接口
| 接口 | 接口名称 | 用途 |
|------|----------|------|
| G1 | 实时行情列表 | 获取最新股价、涨跌幅、成交量 |
| G2 | 历史日K行情数据 | 获取历史走势，分析技术形态 |
| A1 | 主要财务指标 | 分析盈利能力、成长性 |
| D2 | 十大股东/流通股东 | 了解机构持仓变化 |
| D3 | 股东人数列表 | 判断筹码集中度变化 |
| E2 | 文章信息 | 获取相关财经新闻 |
| E3 | 上市公司公告 | 获取公司重要公告 |
| B6/B7 | 高管增减持 | 了解管理层动向 |

### 使用示例
```python
from equal_data import EqualDataApi
from datetime import datetime, timedelta
import os

api = EqualDataApi(os.getenv('EQUAL_DATA_API_KEY'))

def comprehensive_stock_analysis(stock_code, stock_name):
    """
    个股全景深度分析
    """
    result = {
        'stock_code': stock_code,
        'stock_name': stock_name,
        'analysis_time': datetime.now().strftime('%Y-%m-%d %H:%M')
    }

    # 实时行情
    realtime = api.query_equal_data(
        interfaceId="G1",
        stockCodes=stock_code
    )
    result['realtime'] = {
        'price': realtime[0]['newPrice'],
        'change_pct': realtime[0]['chg'],
        'volume': realtime[0]['volume'],
        'market_cap': realtime[0]['marketValue']
    }

    # 历史K线（近3个月）
    end_date = datetime.now().strftime('%Y-%m-%d')
    start_date = (datetime.now() - timedelta(days=90)).strftime('%Y-%m-%d')

    kline = api.query_equal_data(
        interfaceId="G2",
        stockCodes=stock_code,
        startDate=start_date,
        endDate=end_date
    )
    result['kline'] = kline

    # 财务指标（最近4个季度）
    finance = api.query_equal_data(
        interfaceId="A1",
        stockCodes=stock_code,
        reportType=1       # 季报
    )
    result['finance'] = finance[:4]

    # 十大流通股东（最新报告期）
    shareholders = api.query_equal_data(
        interfaceId="D2",
        stockCode=stock_code,
    		dataDate=None,  # str  # 默认:
    		holderType=1,  # int  # 默认:
    )
    result['shareholders'] = shareholders

    # 股东人数变化（近1年）
    holder_count = api.query_equal_data(
        interfaceId="D3",
        stockCode=stock_code,
        dataDate=None
    )
    result['holder_count_trend'] = holder_count

    # 近期新闻（近1周）
    news = api.query_equal_data(
        interfaceId="E2",
        keyword=stock_name
    )
    result['recent_news'] = news[:5]

    # 近期公告（近1月）
    announcements = api.query_equal_data(
        interfaceId="E3",
        stockCode=stock_code,
    )
    result['announcements'] = announcements[:5]

    # 最近高管增减持计划
    executive_changes = api.query_equal_data(
        interfaceId="B10",
        stockCode=stock_code,
        pageIndex=0,
      	pageSize=10
    )
    result['executive_changes'] = executive_changes

    # 生成分析报告
    generate_report(result)
    return result

def generate_report(data):
    """生成可读的分析报告"""
    print(f"\n{'='*60}")
    print(f"【{data['stock_name']}({data['stock_code']})】全景分析报告")
    print(f"分析时间: {data['analysis_time']}")
    print(f"{'='*60}")

    rt = data['realtime']
    print(f"\n【实时行情】")
    print(f"  当前价格: ¥{rt['price']}")
    print(f"  涨跌幅: {rt['change_pct']}%")
    print(f"  市值: {rt['market_cap']}")

    print(f"\n【财务概览】")
    for f in data['finance'][:2]:
        print(f"  {f['dateStr']}: 美股收益增长率{f['epsYoy']}, 净利{f['netProfitMargin']}")

    print(f"\n【股东情况】")
    print(f"  最新股东人数: {data['holder_count_trend'][0]['holderTotalNum']}")

    print(f"\n【高管动向】")
    if data['executive_changes']:
        for change in data['executive_changes'][:3]:
            action = "增持" if change['changeType'] == 1 else "减持"
            print(f"  {change['executiveName']}: {action} {change['changeShares']}股")
    else:
        print("  近3月无高管增减持记录")

    print(f"\n【近期资讯】")
    for news in data['recent_news'][:3]:
        print(f"  - {news['title'][:40]}...")

    print(f"\n{'='*60}")

# 使用示例
comprehensive_stock_analysis("000001", "平安银行")
```

## 使用建议

1. **API Key配置**: 确保已设置环境变量 `EQUAL_DATA_API_KEY`
2. **频率控制**: 注意API调用频率限制，避免短时间内大量请求
3. **缓存策略**: 对不频繁变化的数据（如财务指标、股东信息）建议本地缓存
4. **错误处理**: 生产环境需添加异常处理和重试机制
5. **数据验证**: 对返回数据进行有效性校验后再使用
