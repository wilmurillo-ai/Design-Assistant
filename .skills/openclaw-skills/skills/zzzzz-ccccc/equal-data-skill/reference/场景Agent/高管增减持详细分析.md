## 场景一：高管增减持详细分析

### 业务价值
全面了解上市公司高管增减持行为，包括历史变动、未来计划和持股分析，辅助判断管理层对公司前景的信心。

### 涉及接口
| 接口 | 接口名称 | 用途 |
|------|----------|------|
| B6 | 高管增减持金额排行榜 | 获取近期增减持金额排名，快速定位热点 |
| B7 | 查询所有高管增减持变动数据 | 获取详细的历史变动记录 |
| B8 | 高管增减持计划数据 | 获取未来增减持计划预告 |
| B11 | 高管增减持数据上涨概率分析 | 分析高管持股结构和上涨概率趋势 |

### 使用示例
```python
from equal_data import EqualDataApi
import os

api = EqualDataApi(os.getenv('EQUAL_DATA_API_KEY'))

# 获取近1月高管增持金额排行榜
top_increase = api.query_equal_data(
    interfaceId="B6",
    period=1,           # 近1月
    changeType=1,       # 增持
    pageIndex=0,
    pageSize=20
)

# 遍历排名靠前的股票，查询详细变动数据
for stock in top_increase['data'][:5]:
    stock_name = stock['stockName']

    # 查询该股票所有高管变动记录
    changes = api.query_equal_data(
        interfaceId="B7",
        stockName=stock_name,
        pageIndex=0,
        pageSize=50
    )

    # 查询该股票的增减持计划
    plans = api.query_equal_data(
        interfaceId="B8",
        stockName=stock_name,
        pageIndex=0,
        pageSize=10
    )

    # 高管持股数据分析
    analysis = api.query_equal_data(
        interfaceId="B11",
        stockCode=stock['stockCode'],
        period=4           # 近1年
    )

    # 综合分析输出
    print(f"\n=== {stock_name} 高管增减持综合分析 ===")
    print(f"近期增持金额: {stock['amount']}元")
    print(f"历史变动次数: {len(changes['data'])}次")
    print(f"未来计划数: {len(plans['data'])}个")
    print(f"持股分析: {analysis['data']}")
```

## 使用建议

1. **API Key配置**: 确保已设置环境变量 `EQUAL_DATA_API_KEY`
2. **频率控制**: 注意API调用频率限制，避免短时间内大量请求
3. **缓存策略**: 对不频繁变化的数据（如财务指标、股东信息）建议本地缓存
4. **错误处理**: 生产环境需添加异常处理和重试机制
5. **数据验证**: 对返回数据进行有效性校验后再使用
