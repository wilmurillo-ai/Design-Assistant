## 场景二：龙虎榜游资追踪详细分析

### 业务价值
追踪活跃游资的操作风格和胜率，识别市场热点龙头股，为短线交易提供参考。

### 涉及接口
| 接口 | 接口名称 | 用途 |
|------|----------|------|
| B1 | 龙虎榜-近1月胜率排行 | 筛选近期胜率高的优质游资 |
| B5 | 龙虎榜-市场高度 | 识别市场热点连板股 |
| C3 | 龙虎榜-游资特色指标分析 | 分析游资历史操作的涨跌概率 |
| C4 | 游资基本信息 | 获取游资简介和关联营业部 |

### 使用示例
```python
from equal_data import EqualDataApi
import os

api = EqualDataApi(os.getenv('EQUAL_DATA_API_KEY'))

# 获取近1月胜率最高的游资列表
top_yz = api.query_equal_data(
    interfaceId="B1",
    period=1,           # 近1月
    pageIndex=0,
    pageSize=10
)

# 遍历优质游资，获取详细信息
for yz in top_yz['data'][:3]:
    yz_name = yz['youziName']
    win_rate = yz['winRate']

    # 获取游资基本信息和关联营业部
    yz_info = api.query_equal_data(
        interfaceId="C4",
        youziName=yz_name
    )

    # 分析该游资的操作特色指标
    yz_analysis = api.query_equal_data(
        interfaceId="C3",
        youziName=yz_name,
        period=12          # 近1年数据
    )

    print(f"\n=== 游资: {yz_name} ===")
    print(f"近1月胜率: {win_rate}%")
    print(f"关联营业部: {yz_info['data']['departments']}")
    print(f"次日上涨概率: {yz_analysis['data']['nextDayUpProb']}%")
    print(f"3日上涨概率: {yz_analysis['data']['threeDayUpProb']}%")

# 查看当前市场高度（情绪判断）
market_height = api.query_equal_data(
    interfaceId="B5",
    tradeDate=None      # 最新交易日
)

print(f"\n=== 当前市场高度 ===")
print(f"最高连板数: {market_height['data'][0]['limitUpDays']}板")
print(f"龙头股: {market_height['data'][0]['stockName']}")
```
