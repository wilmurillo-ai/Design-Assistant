# 音频提取能力

## 功能命名

- `extract_audio`

## 作用

- 从视频中提取音频，支持输出 `mp3` 或 `m4a`， 默认输出格式为**m4a** 。

## 参数

| 参数名    | 类型   | 必填 | 说明                                       |
| --------- | ------ | ---- | ------------------------------------------ |
| video_url | string | ✅   | 输入视频 URL，支持 `http://` 或 `https://` |
| format    | string | ❌   | 输出格式：`mp3` \| `m4a`，默认 `m4a`       |

## 返回数据

- task_id(str): 任务查询 id
- requst_id(str): 日志 id
