# Bilibili 弹幕 XML 格式说明

## 文件结构

```xml
<?xml version="1.0" encoding="UTF-8"?>
<i>
  <chatserver>chat.bilibili.com</chatserver>
  <chatid>12345678</chatid>
  <d p="时间,类型,字号,颜色,发送时间戳,弹幕池,用户Hash,弹幕ID">弹幕内容</d>
  ...
</i>
```

## `<d p="...">` 属性说明

| 位置 | 字段 | 说明 |
|------|------|------|
| 0 | **播放时间** | 弹幕出现的视频时间点（秒，浮点），**本 Skill 使用此字段** |
| 1 | 弹幕类型 | 1=滚动弹幕，4=底部固定，5=顶部固定，6=逆向，7=高级，8=代码 |
| 2 | 字号 | 25=标准 |
| 3 | 颜色 | 十进制 RGB，16777215=白色 |
| 4 | 发送时间戳 | Unix 时间戳（非视频时间） |
| 5 | 弹幕池 | 0=普通，1=字幕 |
| 6 | 用户 Hash | 匿名化用户标识 |
| 7 | 弹幕 ID | 数据库 ID |

## 常见高能弹幕词汇参考

| 类型 | 关键词 |
|------|--------|
| 笑点 | `哈哈`、`哈哈哈`、`www`、`笑死`、`笑不活了`、`hhh` |
| 认可/精彩 | `666`、`牛`、`绝了`、`厉害`、`神`、`强`、`真的假的` |
| 感叹 | `awsl`、`可爱`、`这也行`、`我了个去`、`nb` |
| 名场面 | `名场面`、`经典`、`高光`、`封神` |
| 共鸣 | `我也是`、`真的`、`对对对`、`就是` |
| 惊讶 | `？？？`、`离谱`、`不是吧`、`wtf` |

## 弹幕密度与情绪分析

- **密度 > 100条/分钟**：极高能时刻（大量观众同时发弹幕）
- **密度 50-100条/分钟**：高能时刻
- **密度 10-50条/分钟**：普通内容
- **密度 < 10条/分钟**：低互动区域（可能是过渡/前情提要）

## 示例

```xml
<d p="125.338,1,25,16777215,1620000000,0,abc123,9999999">666这操作！</d>
```
→ 视频第 125.3 秒处，滚动弹幕，白色，内容"666这操作！"
