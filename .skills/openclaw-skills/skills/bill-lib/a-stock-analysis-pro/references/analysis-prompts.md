# 六大章节分析提示词

> 将数据整理表填完后，用以下提示词逐章节分析。提示词可直接复制给 LLM 使用。

---

## 章节一：核心观点及研判

```
你是专业A股策略分析师。根据以下数据，为研报「核心观点」章节输出三段判断，
每段限80~120字，语言专业直接，避免模糊词汇。

股票：{简称}（{代码}），今日价 {价格}（{涨跌幅}%）
52周定位：{X%}（底部/低位/中位/高位/顶部区域）
今日量比：{X}（缩量/平量/放量），换手率 {X%}
最新财报（{报告期}）：营收同比 {X%}，归母净利润同比 {X%}
核心事件（最近最重要的1~2条）：{事件描述}
行业景气：{产品价格趋势，一句话}
大股东动向：{如有减持/增持}
ST状态：{是/否}

请输出：

【价格走势】（80~120字）
- 描述今日价格表现和量能特征
- 说明52周定位和近期趋势
- 结尾给出技术形态一句话判断（如"典型缩量阴跌/放量突破/高位震荡"）

【基本面逻辑】（80~120字）
- 描述最新财报核心数字
- 说明利润变化的主因（原材料/需求/价格/减值）
- 给出催化剂强度：★1~5，并说明能否短期兑现

【策略概要】（80~120字）
- 明确操作建议：追涨/等待/防守/观望（选一个）
- 给出具体入场/观察条件（"当……时，才考虑……"）
- 说明近期最重要的1个时间节点（具体日期或可量化条件）
```

## 章节二：关键价格结构

```
根据以下价格数据，识别关键价格结构，输出7行表格：

今日收盘：{价格}，52周区间：{低}~{高}，发行价：{价格（如有）}
近期明显高点：{价格+日期}，近期明显低点：{价格+日期}
每股净资产：{X}（PB=1.0x 对应价格 = 每股净资产）
是否ST（±5%限制）：{是/否}

输出格式（7行）：
项目名称 | 价格区间（CNY） | 逻辑说明（1~2句，含具体理由）

行1：第一阻力位（近期明显压力区）
行2：第二阻力位（发行价/中期均值/前期平台，任选最相关一个）
行3：今日收盘价（附量能特征）
行4：第一支撑位（52周低点或近期低点）
行5：强支撑位（净资产锚或历史密集成交区）
行6：极值支撑（上市以来最低价/理论极值）
行7：当前状态（"放量突破/缩量阴跌/高位震荡/低位筑底/盘整待变"之一）
```

## 章节三：深度数据透视

```
根据以下完整数据，输出数据透视章节的6个段落（每段50~150字）：

{粘贴完整的 Step 2 数据整理表内容}

当前10年期国债收益率参考值：约2.0%~2.3%

段落输出要求：

【段落1 市值与估值】
- PE是否有参考意义（亏损/高PE时说明为何及替代指标）
- PB当前值是否高于/低于行业均值（化工类均值约1.5~2.0x，消费类约3~5x）
- 股息率与国债收益率比较，判断配置价值
- 净资产底部价格（= 每股净资产，PB=1.0x）

【段落2 今日资金异动】
- 量比是多少，是放量还是缩量多少%
- 主力资金净流向方向和金额
- 换手率类型（对照换手率标准）
- 综合判断资金行为性质（主动买入/被动承接/游资博弈/无人问津）

【段落3 近期行情回顾】
- 列举近期3~4个关键价格节点（格式：X月X日 价格 + 事件）
- 涨停/跌停记录（如有）
- 公司异动公告情况（如有，核心内容一句话）
- 龙虎榜机构/游资行为

【段落4 核心业绩数据】
- 营收和净利润同比数据
- 扣非净利润和归母净利润的差值解释（非经常性损益来源）
- 单季度趋势（是否加速/减速/转折）
- 利润变化核心原因（2~3个）
- 资产负债率和财务费用健康度

【段落5 核心催化剂】
- 列举最重要的1~3个催化剂
- 每个催化剂强度评级★1~5，说明评级理由
- 最近兑现时间节点
- 如无催化剂，明确说明"当前缺乏明确催化剂，修复依赖{X}"

【段落6 主营业务结构表】（公司有3个以上业务时才输出）
- 格式：业务名 | 特点 | 应用领域及景气描述
- 3~6行，每行突出与当前股价最相关的信息
```

## 章节四：情景模拟

```
根据以下信息，推演三个情景（概率之和必须=100%）：

股票：{简称}，当前价格 {价格}，52周定位 {X%}
核心催化剂：{描述}，强度：★{N}
业绩趋势：{上升/下降/转折中/持续低迷}
大股东：{无动作/减持X%/增持X%}
融资余额/流通市值：{X%}（>5%=杠杆风险偏高）
ST状态：{是/否}

概率分配参考（根据股票类型选择，三情景概率和=100%）：
- 底部低估型（52周<25%，无重大风险）：A=25% / B=50% / C=25%
- 强势上涨型（近期涨停连板，高位）：A=20% / B=50% / C=30%
- ST/合规风险型：A=15% / B=55% / C=30%
- 概念炒作高位型（52周>80%）：A=15% / B=50% / C=35%
- 中位震荡型：A=20% / B=55% / C=25%

输出格式：

情景A（乐观，对应绿色背景）：
  触发条件：[具体可验证事件，如"4月23日年报净利润同比转正>20%"]
  入场信号：[技术面具体条件，如"成交量>1500万股并放量突破X元"]
  目标价区间：[含计算依据，如"PE从18x修复至22x对应EPS，目标Y~Z元"]
  时间窗口：[预计何时]
  概率：{A}%

情景B（基准，对应黄色背景）：
  核心逻辑：[维持现状的2个主要原因]
  震荡区间：[X~Y元]
  适合策略：[持有分红/轻仓观望/避险]
  概率：{B}%

情景C（悲观，对应红色背景）：
  触发条件：[具体负面事件]
  参考止损价：[具体价格及选取依据，如"跌破52周低点X元"]
  预估最大回撤：[百分比]
  最需警惕的风险：[1~2个最重要的]
  概率：{C}%
```

## 章节五：风险提示

```
根据以下股票信息，从五类风险中选4~6项最相关的输出风险表：

股票：{简称}，行业：{行业}
主营业务风险：{原材料依赖/产品价格敏感/需求周期性}
融资余额/市值：{X%}（>5%杠杆风险高）
大股东动向：{减持/增持/无变化}
合规状态：{正常/ST/立案中}
ST股：{是/否，如是则必须加入合规退市风险}

五类风险来源（每次必选最相关的，不要每次都选同样的）：
A. 行业/宏观：产品价格持续下行、政策收紧、行业产能过剩、原材料涨价
B. 基本面：业绩持续恶化、财务数据不实、主业不可逆衰退、商誉减值
C. 市场流动性：大盘系统风险、成交量枯竭、机构强制清仓
D. 资金面：大股东减持、融资盘强平（余额/流通市值>5%时高风险）、限售解禁
E. 合规法律：处罚升级/退市（ST股专属）、诉讼赔偿、信披违规重演

风险等级：
★★★★★ 触发概率>50%，股价影响>-20%
★★★★☆ 触发概率30~50%，影响-10%~-20%
★★★☆☆ 触发概率<30%，影响-5%~-10%
★★☆☆☆ 触发概率<15%，影响<-5%

输出表格（每行）：
风险类别（≤15字）| 具体描述（2~3句，含触发条件+影响路径）| 等级（★☆组合）
```

## 章节六：总结评价

```
根据前五章节分析结果，撰写总结（分两部分）：

输入：{将前五章节的核心结论摘要填入此处}

Part 1：核心评价（高亮段落，3~5句）
要求（每句均要有数据支撑，拒绝模糊措辞）：
- 句1：定性公司（"是一家______类型的企业，当前处于______阶段"）
- 句2：当前价格驱动力（"股价下跌/上涨的核心驱动是______而非______"）
- 句3：最大风险（"当前最需警惕的是______，触发条件是______"）
- 句4：适合策略（"当前价格更适合______策略而非______策略"）
- 句5：操作框架（"核心不是______，而是等待______信号"）

Part 2：三大关键节点
节点1（最近确定性时间节点）：【时间】事件名称——对股价的意义
节点2（中期验证节点）：【时间/条件】事件名称——意义
节点3（持续跟踪指标）：【持续观察】指标名+阈值——意义
```
