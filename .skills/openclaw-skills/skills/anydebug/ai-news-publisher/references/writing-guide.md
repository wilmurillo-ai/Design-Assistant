# 爆款写作与选题指南

## 1. 选题策划（从 10 选 1）
从每日新闻中选出**1个最爆款**，评估标准如下：
1. **热点性** - 关联当前热点/趋势
2. **争议性** - 有观点冲突
3. **实用性** - 解决读者痛点
4. **新鲜感** - 独家/首发/新视角
5. **情绪价值** - 激发好奇/焦虑/希望

## 2. 爆款标题公式
| 类型 | 示例 |
|-----|------|
| 数字+痛点 | 《AI淘汰了50%程序员，这3个技能让你翻盘》 |
| 对比反差 | 《月薪3千和3万，差的不是技术》 |
| 悬念好奇 | 《OpenAI深夜发布，硅谷炸了》 |
| 利益承诺 | 《学会这个AI工具，效率提升10倍》 |
| 情绪触发 | 《被AI取代的第1天，我哭了》 |

**标题自检清单**：
- [ ] 有具体数字？
- [ ] 触发情绪？
- [ ] 制造好奇？
- [ ] 承诺价值？
- [ ] 30字以内？

## 3. 开头钩子（3秒定生死）
- **扎心问题**：你有没有想过，为什么你的AI文章没人看？
- **意外故事**：昨天，一个程序员朋友被裁员了。不是因为他技术不行，而是因为他不会用AI。
- **惊人数据**：ChatGPT上线2个月，用户破亿。TikTok用了9个月，Instagram用了2年半。
- **认知反差**：很多人以为AI会让程序员失业。错。AI会让不会用AI的程序员失业。

## 4. 内容节奏
**黄金结构**：钩子 → 痛点 → 分析 → 方案 → 行动

**节奏控制**：
- 每 300 字一个小高潮（新观点/新数据/小故事/新问题）
- 每段结尾留悬念
- 适时插入金句

**金句示例**：
- 💡 记住：工具不会淘汰人，会工具的人淘汰不会工具的人。
- ⚡ AI不是威胁，是杠杆。
- 📌 不是AI替代你，是用AI的人替代你。

## 5. 结尾行动
- **互动引导**：你怎么看？评论区聊聊。有收获？点个"在看"让更多人看到。
- **行动号召**：现在就去试试，3分钟搞定。收藏这篇文章，明天就能用上。

## 6. 常用结构模板
**热点事件类**：
1. 开头：事件概述（3句话）
2. 痛点：这个事件对你意味着什么
3. 分析：为什么发生，背后逻辑
4. 影响：对你的影响（利/弊）
5. 行动：你应该怎么做
6. 结尾：金句总结 + 互动引导

**新功能/产品类**：
1. 开头：产品/功能介绍
2. 痛点：解决什么问题
3. 功能：具体怎么用
4. 价值：能帮你做什么
5. 案例：真实使用场景
6. 行动：如何获取
7. 结尾：鼓励尝试 + 互动

**趋势分析类**：
1. 开头：趋势现象描述
2. 数据：支撑数据/证据
3. 分析：为什么会这样
4. 判断：未来会怎样
5. 建议：你应该怎么做
6. 结尾：引发思考 + 互动
