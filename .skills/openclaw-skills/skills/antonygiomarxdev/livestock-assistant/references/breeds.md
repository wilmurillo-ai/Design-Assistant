# Livestock Breed Reference

suitability for meat / milk / wool / dual-purpose production.

## Selection Guide

- **Milk production** → Holstein Friesian (temperate), Criollo Lechero Tropical (tropical), Nubian (goats)
- **Meat production (tropical)** → Brahman crosses, Dorper (sheep), Angus / Hereford (temperate)
- **Dual-purpose** → Simmental (cattle)
- **Fine wool** → Merino
- **Heat & parasite resistance** → Brahman, Criollo Lechero Tropical
