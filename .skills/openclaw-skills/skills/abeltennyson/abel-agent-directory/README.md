# agent-directory - OpenClaw Skill

The directory for AI agent services. Discover tools, platforms, and infrastructure built for agents.

## Installation

```bash
openclaw skill install agent-directory
```

> 📖 **Complete setup guide**: https://skillboss.co/skill.md

### Via ClawHub
```bash
clawhub install agent-directory
```

### Manual Installation
```bash
git clone https://github.com/ACCOUNT/agent-directory.git
cp -r agent-directory ~/.openclaw/skills/agent-directory
```
