# 🇯🇵 日本 Visit Japan Web 填写指南

欢迎前往日本！日本政府推出了 **Visit Japan Web** 在线入境申报系统，让入境流程更加便捷。我为你整理了这份详细的填写指南。

---

## 📋 什么是 Visit Japan Web？

**Visit Japan Web** 是日本政府推出的在线入境申报系统，整合了以下三项服务：
1. **入境审查**（Immigration）
2. **海关申报**（Customs Declaration）
3. **检疫快速通道**（Fast Track）

通过提前在线填写，可以大幅缩短入境时的排队时间。

---

## 📝 填写前准备

在开始填写前，请确保你已经准备好以下信息：

- **护照**：有效期需在 6 个月以上
- **航班信息**：准确的航班号和抵达日期
- **在日本的住址**：建议提前预订好酒店，并准备好酒店的英文全称和详细地址
- **电子邮箱**：用于注册账户和接收确认信息
- **疫苗接种证明**（如有）：虽然目前不强制要求，但建议准备
- **填写时间**：建议在出发前 1-2 周完成

---

## 🌐 官方填写入口

⚠️ **请务必通过日本政府官方网站进行填写，谨防钓鱼网站。**

**官方网站**：https://www.vjw.digital.go.jp/

**支持语言**：
- 日语
- 英语
- 中文（简体/繁体）
- 韩语

## 📝 详细填写步骤

### 第一步：注册账户

1. 访问 Visit Japan Web 官网
2. 点击 **"新規アカウント作成"**（创建新账户）
3. 输入电子邮箱地址
4. 设置密码（至少 8 位，包含大小写字母和数字）
5. 验证邮箱（点击邮件中的验证链接）

---

### 第二步：填写个人信息

登录后，需要填写以下个人信息：

| 字段名称 | 说明与示例 |
|---------|-----------|
| **基本信息** | |
| 姓名（英文） | 与护照完全一致<br>例如：`ZHANG SAN` |
| 姓名（日文假名） | 可选，如不会填写可跳过 |
| 出生日期 | 格式：YYYY/MM/DD<br>例如：`1990/01/01` |
| 性别 | 男性/女性 |
| 国籍 | 选择 `中国` 或 `CHINA` |
| **护照信息** | |
| 护照号码 | 填写护照右上角的号码<br>例如：`E12345678` |
| 护照签发国 | 选择 `中国` |
| 护照有效期 | 格式：YYYY/MM/DD<br>例如：`2030/12/31` |

---

### 第三步：填写入境信息

| 字段名称 | 说明与示例 |
|---------|-----------|
| **航班信息** | |
| 航空公司 | 选择你的航空公司<br>例如：`中国国际航空`、`全日空` |
| 航班号 | 填写你的航班号<br>例如：`CA183` |
| 出发地 | 填写出发机场<br>例如：`北京首都国际机场` |
| 抵达机场 | 选择日本入境机场<br>例如：`成田国际机场`、`羽田机场` |
| 抵达日期 | 格式：YYYY/MM/DD<br>例如：`2026/04/15` |
| **住宿信息** | |
| 住宿类型 | 选择 `酒店`、`民宿`、`亲友家` 等 |
| 酒店名称 | 填写酒店的日文或英文名称<br>例如：`東京ヒルトンホテル` 或 `Tokyo Hilton Hotel` |
| 酒店地址 | 填写完整地址<br>例如：`〒160-0023 東京都新宿区西新宿6-6-2` |
| 酒店电话 | 填写酒店电话号码<br>例如：`03-1234-5678` |
| **联系方式** | |
| 在日本的联系电话 | 如有日本手机号可填写，否则填写酒店电话 |
| 紧急联系人 | 填写国内紧急联系人姓名和电话 |

---

### 第四步：填写海关申报

**需要申报的物品**：

| 申报项目 | 说明 |
|---------|------|
| **现金** | 携带现金或有价证券超过 100 万日元（约 5 万人民币）需申报 |
| **酒类** | 免税额度：3 瓶（每瓶 760ml） |
| **香烟** | 免税额度：400 支（或雪茄 100 支，或烟丝 500g） |
| **香水** | 免税额度：2 盎司（约 56ml） |
| **其他物品** | 总价值超过 20 万日元需申报 |
| **禁止物品** | 毒品、枪支、刀具、假冒商品、肉类制品等 |

**特别注意**：
- ⚠️ **肉类制品**（包括火腿、香肠、肉干等）**严禁携带**
- ⚠️ **新鲜水果**需要检疫证明
- ⚠️ **药品**需要符合日本药事法规定

---

### 第五步：健康申报（Fast Track）

虽然目前日本已取消大部分疫情相关限制，但仍需填写健康申报：

| 申报项目 | 说明 |
|---------|------|
| 健康状况 | 确认过去 14 天内未出现发热、咳嗽等症状 |
| 疫苗接种情况 | 如有接种可填写（非强制） |
| 过去 14 天旅行史 | 列出访问过的国家/地区 |

---

### 第六步：提交并获取 QR 码

1. 确认所有信息无误后，点击 **"提交"**
2. 系统会生成 **3 个 QR 码**：
   - **入境审查用 QR 码**（蓝色）
   - **海关申报用 QR 码**（黄色）
   - **检疫用 QR 码**（绿色，如适用）
3. **务必截图或下载保存所有 QR 码**
4. 入境时分别在对应窗口出示相应的 QR 码

## ⚠️ 重要提醒

### 📄 保存 QR 码
- ✅ **务必将所有 QR 码截图保存在手机相册中**
- ✅ **建议同时保存到云端备份**
- ✅ **如有条件，建议打印纸质版**
- 入境时需要分别出示不同的 QR 码

### ⏰ 填写时间建议
- **建议在出发前 1-2 周完成**
- **最晚在出发前 24 小时完成**
- 填写后可随时修改，直到入境前

### 💰 费用说明
- **Visit Japan Web 完全免费**
- 如遇到要求付费的网站，请立即停止

### 👨‍👩‍👧‍👦 家庭申报
- 每位家庭成员都需要单独注册账户
- 每人都需要获取自己的 QR 码
- 未成年人可由家长代为填写

### 🚨 入境流程
1. **检疫**（如适用）：出示绿色 QR 码
2. **入境审查**：出示蓝色 QR 码 + 护照
3. **提取行李**
4. **海关检查**：出示黄色 QR 码

## ❓ 常见问题处理

### Q1：Visit Japan Web 是强制的吗？
**A**：不是强制的，但强烈推荐使用。使用 Visit Japan Web 可以：
- 大幅缩短入境排队时间
- 避免填写纸质表格
- 使用专用快速通道

### Q2：填写错误可以修改吗？
**A**：可以。在入境前，你可以随时登录账户修改信息并重新生成 QR 码。

### Q3：QR 码有效期多久？
**A**：QR 码在你填写的入境日期前后都有效，但建议在入境当天使用最新生成的 QR 码。

### Q4：可以用纸质版 QR 码吗？
**A**：可以。手机截图或打印纸质版都可以，但要确保 QR 码清晰可扫描。

### Q5：忘记截图 QR 码怎么办？
**A**：可以重新登录 Visit Japan Web 账户，查看并下载之前生成的 QR 码。

### Q6：没有日本手机号怎么填联系电话？
**A**：可以填写酒店电话号码，或者留空（部分字段可选）。

### Q7：携带药品需要申报吗？
**A**：
- 个人使用的处方药（1-2 个月用量）通常不需要申报
- 大量药品或特殊药品需要提前申请《药监证明》
- 建议携带医生处方和药品说明书

### Q8：可以携带肉类制品吗？
**A**：❌ **严禁携带**任何肉类制品（包括火腿、香肠、肉干、肉松等），违者将被没收并可能罚款。

### Q9：入境时需要出示哪些文件？
**A**：
- 护照（有效期 6 个月以上）
- Visit Japan Web 的 QR 码（3 个）
- 往返机票（可能抽查）
- 酒店预订单（可能抽查）
- 足够的旅费证明（可能抽查）

### Q10：转机需要填写吗？
**A**：
- 如果在日本转机**不出境**（国际转国际），不需要填写
- 如果需要**入境日本**再转机，必须填写
