# 🇮🇩 印尼 All Indonesia Arrival Card 填写指南

欢迎前往美丽的印尼！从 2025 年 8 月起，印尼启用了全新的 **All Indonesia 入境申报系统**，将原本分开填写的入境卡、海关申报和健康声明整合在一起。别担心，流程很简单，我为你整理了这份详细的填写指南。

---

## 📋 填写前准备

在开始填写前，请确保你已经准备好以下信息：

- **护照**：有效期需在 6 个月以上
- **航班信息**：准确的航班号和抵达日期
- **在印尼的住址**：建议提前预订好酒店，并准备好酒店的英文全称和详细地址
- **签证信息**（如已办理）：落地签号或电子签号
- **电子邮箱**：用于接收确认信息
- **填写时间**：最早可在出发前 3 天（含出发当天）填写

---

## 🌐 官方填写入口

⚠️ **请务必通过印尼移民局官方网站进行填写，谨防钓鱼网站。**

**官方网站**：https://allindonesia.imigrasi.go.id/

**其他方式**：
- 扫描官方二维码
- 下载 APP "All Indonesia"

---

## 📝 详细填写步骤

整个流程分为 **"选择语言"**、**"填写信息"** 和 **"保存 QR Code"** 三个部分。

### 第一步：选择语言并进入填写页面

1. 进入官网后，点击右上角 **"Languages"** 菜单
2. 选择 **"中国"** 切换为简体中文界面（推荐）
3. 选择 **"外国访客"** 入口开始填写

---

### 第二步：填写个人信息

**详细填写表格**：

| 填写项目 | 说明与示例 |
|---------|-----------| 
| **个人信息 (Personal Information)** | |
| 姓名 (Name) | 与护照信息页一致，用拼音填写，姓和名之间用空格隔开<br>例如：`ZHANG SAN` |
| 护照号 (Passport No.) | 填写护照右上角的号码<br>例如：`E12345678` |
| 国籍 (Nationality) | 选择 `CHINA` |
| 出生日期 (Date of Birth) | 按照护照信息页格式填写<br>例如：`01/01/1990` |
| 性别 (Gender) | 男性选 `Male`，女性选 `Female` |
| 护照有效期 (Date of Passport Expiry) | 按照护照信息页填写<br>例如：`31/12/2030` |
| 签证号码 (Visa No.) | 如已办理落地签或电子签，填写签证号<br>例如：`1234567890` |
| 电子邮箱 (Email Address) | 填写一个常用且能正常接收邮件的邮箱<br>例如：`zhangsan@example.com` |
| 手机号码 (Mobile No.) | 填写你的中国手机号，前面加上国家代码 `+86`<br>例如：`+86 138 0000 0000` |

💡 **小贴士**：系统支持护照扫描功能，可以选择"护照扫描"让系统自动识别信息，但建议核对后再提交。

---

### 第三步：填写旅行详情

| 填写项目 | 说明与示例 |
|---------|-----------| 
| **旅行信息 (Travel Details)** | |
| 抵达日期 (Date of Arrival) | 你抵达印尼的日期<br>例如：`15/04/2026` |
| 离开日期 (Date of Departure) | 你离开印尼的日期<br>例如：`22/04/2026` |
| **交通方式 (Transportation)** | |

| 航班号 (Flight No.) | 填写你的航班号<br>例如：`CZ303` |
| 抵达机场 (Airport of Arrival) | 填写机场代码或名称<br>常见机场见下方参考 |
| **住宿信息 (Accommodation)** | |
| 住宿类型 (Accommodation Type) | 住酒店选择 `Hotel`，住亲友家选择 `Home` |
| 酒店名称/地址 (Name/Address) | 填写酒店的英文全称和街道地址<br>例如：`Grand Hyatt Bali, Jalan Kayu Aya No. 1` |

### 第四步：填写海关申报

⚠️ **请仔细阅读申报要求，如实填写，避免违反印尼海关规定。**

**海关申报表格**：

| 申报项目 | 说明 |
|---------|------|
| **行李数量** | 填写托运行李和手提行李的总数量 |
| **需申报物品** | |
| 现金金额 | 如携带现金超过 1 亿印尼盾（约 5000 美元）**必须申报** |
| 酒精饮料 | 年满 21 岁可免税携带 1 升洋酒 |
| 香烟 | 年满 21 岁可免税携带 200 支香烟 |
| 电子产品 | 如携带多部电子产品用于商业用途需申报 |
| 食品 | 肉类、水果、蔬菜等动植物产品需申报 |
| **禁止携带物品** | 毒品、武器、色情物品、濒危动植物制品等 |

💡 **重要提示**：
- 印尼对毒品法律极其严格，携带毒品可判死刑
- 未申报现金超过限额将被罚款 10% 的携带金额
- 酒精超限将被没收，不允许补缴关税

---

### 第五步：健康声明

| 声明项目 | 说明 |
|---------|------|
| 健康状况 | 确认过去 14 天内未出现发热、咳嗽、呼吸困难等症状 |
| 接触史 | 确认过去 14 天内未接触过传染病患者 |
| 携带物品 | 确认未携带动植物、动植物产品、病原体等 |

---

### 第六步：提交并保存 QR Code

1. 确认所有信息无误后，点击 **"提交" (Submit)**
2. 系统会生成一个 **QR Code（二维码）**
3. **务必截图、下载或打印保存 QR Code**
4. 入境时向移民官员和海关人员出示 QR Code 即可

---

## ⚠️ 重要提醒

### 📄 保存 QR Code
- ✅ **务必将 QR Code 截图保存在手机相册中**
- ✅ **建议同时保存到云端备份（如 iCloud、Google Photos）**
- ✅ **如有条件，建议打印一份纸质版随身携带**
- 入境时移民官员和海关人员会扫描 QR Code

### ⏰ 填写时间限制
- **最早可在出发前 3 天（含出发当天）填写**
- **建议在出发前 1-2 天完成，避免临时忘记**
- 如未提前填写，也可在抵达印尼后入境前填写，但会增加排队等候时长

### 💰 费用说明
- **填写 All Indonesia Arrival Card 完全免费**
- 如在填写过程中系统要求付费，说明访问的不是官方网站，请立即停止
- 落地签（VoA）费用需单独支付，约 50 万印尼盾（约 35 美元）

### 👨‍👩‍👧‍👦 团体/家庭申报
- 团体或家庭可以共用一个二维码
- 点击"添加旅客"填写所有成员信息
- 入境时由主旅客出示二维码，但所有成员需一同入境

## ❓ 常见问题处理

### Q1：填写错误了怎么办？
**A**：如果发现填写错误，可以重新填写一份新的申报，系统会生成新的 QR Code。入境时使用最新的 QR Code 即可。

### Q2：QR Code 可以打印出来吗？
**A**：可以打印，但不是必须的。手机截图保存即可，入境时向官员出示手机屏幕上的 QR Code。建议将屏幕亮度调到最高。

### Q3：需要打印纸质版吗？
**A**：不是必须的，但建议打印一份备用。如果手机没电或网络问题，纸质版可以作为备用方案。

### Q4：填写后多久有效？
**A**：QR Code 在你填写的抵达日期当天有效。如果行程变更，需要重新填写。

### Q5：可以提前 4 天填写吗？
**A**：不可以。系统最早只允许在出发前 3 天填写，日期选择器会自动限制可选日期范围。

### Q6：填写了入境卡还需要签证吗？
**A**：需要。入境卡和签证是两个完全不同的手续。持有中国护照的游客需要办理落地签（VoA）或提前申请电子签（e-VOA）。

### Q7：海关申报物品忘记填了怎么办？
**A**：如实申报即可。如果不确定是否需要申报，建议选择申报。印尼海关对未申报物品的处罚很严格，罚款金额可能高达物品价值的数倍。

### Q8：入境时需要出示哪些文件？
**A**：
- 护照（有效期 6 个月以上）
- 签证（落地签或电子签）
- All Indonesia Arrival Card QR Code
- 酒店预订确认单（建议准备）
- 返程机票（建议准备）

### Q9：系统打不开或加载很慢怎么办？
**A**：
- 检查网络连接
- 尝试更换浏览器（推荐 Chrome、Safari）
- 尝试使用官方 APP "All Indonesia"
- 如仍无法解决，可在抵达印尼后在机场填写（但会增加排队时间）

### Q10：忘记截图 QR Code 了怎么办？
**A**：可以返回官网，点击"查询申报"（Check Registration），输入护照号和国籍，重新下载或截图 QR Code。
