## 🇲🇾 马来西亚 MDAC 填写流程

填写马来西亚电子入境卡（MDAC）是入境前的必要步骤。别担心，流程并不复杂，我为你整理了一份详细的填写指南，你跟着一步步操作就行。

---

### 📋 填写前准备

在开始填写前，请确保你已经准备好以下信息，这样填写起来会更顺畅：

- **护照**：有效期需在 6 个月以上
- **航班信息**：准确的航班号和出发地
- **在马来西亚的住址**：建议提前预订好酒店，并准备好酒店的英文全称和详细地址
- **电子邮箱**：用于接收确认邮件和 PIN 码
- **填写时间**：最早可在出发前 3 天（含出发当天）填写

---

### 🌐 官方填写入口

⚠️ **请务必通过马来西亚移民局官方网站进行填写，谨防钓鱼网站。**

**官方网站**：https://imigresen-online.imi.gov.my/mdac/main

---

### 📝 详细填写步骤

整个流程分为 **"注册填写"** 和 **"验证下载"** 两个部分。

#### 第一步：注册并填写信息 (REGISTER)

1. 进入官网后，点击红色的 **REGISTER** 按钮开始填写
2. 根据页面提示，如实填写个人信息和旅行信息
3. ⚠️ **所有信息建议使用英文大写字母填写**

**详细填写表格**：

| 填写项目 | 说明与示例 |
|---------|-----------|
| **个人信息 (Personal Information)** | |
| 姓名 (Name) | 与护照信息页一致，姓和名之间用空格隔开<br>例如：`ZHANG SAN` |
| 护照号 (Passport No.) | 填写护照右上角的号码<br>例如：`E12345678` |
| 国籍 (Nationality) | 选择 `CHINA` |
| 出生日期 (Date of Birth) | 按照 **日/月/年** 的格式填写<br>例如：`01/01/1990` |
| 性别 (Sex) | 男性选 `MALE`，女性选 `FEMALE` |
| 护照有效期 (Date of Passport Expiry) | 按照护照信息页填写<br>例如：`31/12/2030` |
| 电子邮箱 (Email Address) | 填写一个常用且能正常接收邮件的邮箱<br>例如：`zhangsan@example.com` |
| 手机号码 (Mobile No.) | 填写你的中国手机号，前面加上国家代码 `+86`<br>例如：`+86 138 0000 0000` |
| **旅行信息 (Traveling Information)** | |
| 抵达日期 (Date of Arrival) | 你抵达马来西亚的日期<br>例如：`15/04/2026` |
| 旅行方式 (Mode of Travel) | 飞机入境选择 `AIR` |
| 离开日期 (Date of Departure) | 你离开马来西亚的日期<br>例如：`22/04/2026` |
| 出发地 (Last Port of Embarkation) | 你登机前的最后一个国家<br>例如：`CHINA` |
| 航班号 (Flight No.) | 填写你的航班号<br>例如：`MH370` |
| 住宿类型 (Accommodation of Stay) | 住酒店选择 `HOTEL/MOTEL/RESTHOUSE` |
| 酒店地址 (Address in Malaysia) | 填写酒店的英文全称和街道地址<br>例如：`Grand Hyatt Kuala Lumpur, 12 Jalan Pinang` |
| 州/城市/邮编 (State/City/Postcode) | 根据酒店所在地选择或填写，常用城市邮编见下方参考 |

**常用城市及邮编参考**：

| 城市 | 州 (State) | 城市 (City) | 邮编 (Postcode) |
|------|-----------|------------|----------------|
| 吉隆坡 (Kuala Lumpur) | WP KUALA LUMPUR | KUALA LUMPUR | 50000 |
| 槟城 (Penang) | PULAU PINANG | GEORGE TOWN | 11600 |
| 仙本那 (Semporna) | SABAH | SEMPORNA | 91308 |
| 马六甲 (Malacca) | MELAKA | MELAKA | 75000 |
| 新山 (Johor Bahru) | JOHOR | JOHOR BAHRU | 80000 |
| 兰卡威 (Langkawi) | KEDAH | LANGKAWI | 07000 |

4. 确认所有信息无误后，点击 **SUBMIT** 提交

---

#### 第二步：验证并下载确认信 (CHECK REGISTRATION)

1. 提交后，系统会提示你一封确认邮件已发送至你的邮箱。**请立刻登录邮箱查收**
2. 在邮件中找到并复制 **PIN 码**
3. 返回 MDAC 官网首页，点击 **CHECK REGISTRATION** 按钮
4. 依次输入：
   - 你的护照号
   - 国籍 (`CHINA`)
   - 刚刚收到的 PIN 码
5. 完成人机验证后点击 **SUBMIT**
6. 页面会显示你填写的完整信息，在页面底部找到下载图标，点击即可下载 **PDF 格式的正式入境卡确认信**

---

### ⚠️ 重要提醒

#### 📄 保存凭证
- ✅ **务必将下载好的 PDF 确认信打印一份纸质版随身携带**
- ✅ **同时也在手机里保存好电子版**
- 入境时移民官员可能会要求出示

#### 🚪 自助通关
- 持有 6 个月以上有效期护照的中国公民
- 在吉隆坡国际机场等指定口岸可以使用自助通关闸门（Autogate）入境
- 非常便捷，节省排队时间

#### ✈️ 转机情况
- 如果你的航班在马来西亚转机，且需要更换航站楼或重新托运行李
- 也算作入境，同样需要填写 MDAC

#### 🆓 豁免人群
以下人群可豁免填写：
- 持有马来西亚长期通行证
- 马来西亚永久居民
- 新加坡、文莱等特定国家护照的人员

### 🔧 常见问题处理

#### Q1: 没有收到 PIN 码邮件怎么办？
**A:** 
1. 检查垃圾邮件箱
2. 确认填写的邮箱地址正确
3. 等待 5-10 分钟后重新检查
4. 如仍未收到，可以重新提交一次

#### Q2: 填写错误可以修改吗？
**A:** 
- 提交前：可以随时修改
- 提交后：需要重新填写一份新的，使用新的 PIN 码下载

#### Q3: 可以提前多久填写？
**A:** 
- 没有明确的提前时间限制
- 建议在确定行程后再填写
- 最晚在出发前完成即可

#### Q4: 入境时需要出示什么？
**A:** 
- PDF 确认信（纸质版或电子版）
- 护照
- 往返机票
- 酒店预订单（可能抽查）
