# 🇰🇷 韩国 K-ETA + e-Arrival Card 填写指南

欢迎前往韩国！韩国有两个重要的入境申报系统：**K-ETA（电子旅行许可）** 和 **e-Arrival Card（电子入境卡）**。我为你整理了这份详细的填写指南。

---

## 📋 系统说明

韩国入境需要完成两个系统的申报：

### 1. K-ETA（Korea Electronic Travel Authorization）
- **性质**：电子旅行许可（类似签证）
- **必填性**：✅ **强制要求**（免签国家公民）
- **费用**：10,300 韩元（约 55 人民币）
- **有效期**：2 年（或护照有效期，以先到期者为准）
- **申请时间**：建议出发前至少 72 小时申请

### 2. e-Arrival Card（电子入境卡）
- **性质**：入境申报表
- **必填性**：✅ **强制要求**（所有外国人）
- **费用**：免费
- **有效期**：单次入境有效
- **填写时间**：可在出发前或飞机上填写

## 📝 Part 1: K-ETA 申请指南

### 第一步：准备材料

| 材料 | 说明 |
|------|------|
| **护照** | 有效期需在 6 个月以上 |
| **护照照片页扫描件** | 清晰的彩色扫描件或照片 |
| **个人照片** | 6 个月内拍摄的证件照（白底） |
| **信用卡/借记卡** | 用于支付申请费用 |
| **电子邮箱** | 用于接收审批结果 |
| **韩国住宿地址** | 酒店名称和地址 |

---

### 第二步：在线申请

访问 K-ETA 官网后，按照以下步骤填写：

#### 1. 基本信息

| 字段名称 | 说明与示例 |
|---------|-----------|
| **护照信息** | |
| 护照号码 | 填写护照右上角的号码<br>例如：`E12345678` |
| 护照类型 | 选择 `普通护照` |
| 护照签发国 | 选择 `中国` |
| 护照签发日期 | 格式：YYYY-MM-DD<br>例如：`2020-01-01` |
| 护照有效期 | 格式：YYYY-MM-DD<br>例如：`2030-12-31` |
| **个人信息** | |
| 姓名（英文） | 与护照完全一致<br>例如：`ZHANG SAN` |
| 姓名（中文） | 填写中文姓名<br>例如：`张三` |
| 性别 | 男性/女性 |
| 出生日期 | 格式：YYYY-MM-DD<br>例如：`1990-01-01` |
| 出生地 | 填写出生城市<br>例如：`北京` |
| 国籍 | 选择 `中国` |

#### 2. 联系信息

| 字段名称 | 说明与示例 |
|---------|-----------|
| 电子邮箱 | 填写常用邮箱<br>例如：`zhangsan@example.com` |
| 手机号码 | 填写中国手机号，前面加 `+86`<br>例如：`+86 138 0000 0000` |
| 居住地址 | 填写中国的居住地址（英文）<br>例如：`123 Beijing Road, Shanghai, China` |

#### 3. 旅行信息

| 字段名称 | 说明与示例 |
|---------|-----------|
| 访问目的 | 选择 `旅游`、`商务`、`探亲` 等 |
| 预计停留时间 | 填写预计在韩国停留的天数<br>例如：`7 天` |
| 韩国住宿地址 | 填写酒店的完整地址（英文或韩文）<br>例如：`123 Gangnam-daero, Gangnam-gu, Seoul` |
| 韩国联系电话 | 填写酒店电话或韩国联系人电话 |

#### 4. 安全问题

需要回答一系列安全相关的问题（通常选择"否"）：

- 是否有犯罪记录
- 是否有传染病
- 是否曾被拒绝入境韩国
- 是否有非法滞留记录
- 等等

⚠️ **重要**：必须如实回答，虚假申报可能导致拒签或入境被拒。

---

### 第三步：上传文件

1. **上传护照照片页**：
   - 格式：JPG、PNG
   - 大小：不超过 5MB
   - 要求：清晰、完整、无反光

2. **上传个人照片**：
   - 格式：JPG、PNG
   - 大小：不超过 5MB
   - 要求：6 个月内拍摄的证件照，白底，正面免冠

---

### 第四步：支付费用

- **费用**：10,300 韩元（约 55 人民币）
- **支付方式**：信用卡/借记卡（Visa、MasterCard、银联）
- **注意**：支付成功后无法退款

---

### 第五步：等待审批

- **审批时间**：通常 24-72 小时
- **结果通知**：通过邮件发送
- **审批结果**：
  - ✅ **批准**：收到 K-ETA 批准通知
  - ❌ **拒绝**：收到拒绝通知及原因
  - ⏳ **补充材料**：可能要求提供额外信息

---

### 第六步：保存 K-ETA

- 批准后，登录 K-ETA 官网下载批准通知
- **务必截图或打印保存**
- 入境时出示 K-ETA 批准通知（电子版或纸质版）

---

### 第三步：填写海关申报

**需要申报的物品**：

| 申报项目 | 说明 |
|---------|------|
| **现金** | 携带现金超过 10,000 美元（或等值外币）需申报 |
| **酒类** | 免税额度：1 瓶（1 升以下） |
| **香烟** | 免税额度：200 支（或雪茄 50 支，或烟丝 250g） |
| **香水** | 免税额度：60ml |
| **其他物品** | 总价值超过 600 美元需申报 |
| **禁止物品** | 毒品、枪支、刀具、假冒商品、肉类制品等 |

**特别注意**：
- ⚠️ **肉类制品**（包括火腿、香肠、肉干等）**严禁携带**
- ⚠️ **新鲜水果**需要检疫证明
- ⚠️ **药品**需要符合韩国药事法规定

---

### 第四步：健康申报

| 申报项目 | 说明 |
|---------|------|
| 健康状况 | 确认过去 14 天内未出现发热、咳嗽等症状 |
| 过去 14 天旅行史 | 列出访问过的国家/地区 |
| 是否接触过传染病患者 | 如实填写 |

---

### 第五步：提交并获取 QR 码

1. 确认所有信息无误后，点击 **"提交"**
2. 系统会生成一个 **QR 码**
3. **务必截图或下载保存 QR 码**
4. 入境时向移民官员出示 QR 码

## ⚠️ 重要提醒

### 📄 K-ETA vs e-Arrival Card

| 对比项 | K-ETA | e-Arrival Card |
|-------|-------|----------------|
| **性质** | 电子旅行许可 | 入境申报表 |
| **费用** | 10,300 韩元 | 免费 |
| **有效期** | 2 年 | 单次入境 |
| **申请时间** | 出发前至少 72 小时 | 出发前或飞机上 |
| **必填性** | 强制（免签国家） | 强制（所有外国人） |

### ⏰ 时间安排建议

1. **K-ETA**：出发前 **1-2 周** 申请（预留审批时间）
2. **e-Arrival Card**：出发前 **1-2 天** 填写（或飞机上填写）

### 💰 费用说明

- **K-ETA**：10,300 韩元（约 55 人民币）
- **e-Arrival Card**：完全免费
- ⚠️ 谨防第三方代办网站收取高额费用

### 👨‍👩‍👧‍👦 家庭申报

- **K-ETA**：每位家庭成员都需要单独申请
- **e-Arrival Card**：每位家庭成员都需要单独填写
- 未成年人可由家长代为填写

### 🚨 入境流程

1. **检疫**（如适用）
2. **入境审查**：出示护照 + K-ETA + e-Arrival Card QR 码
3. **提取行李**
4. **海关检查**：出示海关申报

## ❓ 常见问题处理

### Q1：K-ETA 是强制的吗？
**A**：是的。对于免签国家（包括中国）的公民，K-ETA 是强制要求的。没有 K-ETA 将无法登机。

### Q2：K-ETA 审批需要多久？
**A**：通常 24-72 小时，但建议提前 1-2 周申请，以防审批延迟。

### Q3：K-ETA 被拒绝怎么办？
**A**：
- 查看拒绝原因
- 根据原因补充材料或修改信息
- 重新申请（需重新支付费用）
- 如多次被拒，建议申请正式签证

### Q4：e-Arrival Card 可以在飞机上填写吗？
**A**：可以。但建议提前在线填写，节省入境时间。

### Q5：忘记截图 QR 码怎么办？
**A**：可以重新登录 e-Arrival Card 官网，查看并下载之前生成的 QR 码。

### Q6：K-ETA 可以多次入境吗？
**A**：可以。K-ETA 有效期为 2 年，期间可以多次入境韩国，每次停留不超过 90 天。

### Q7：携带药品需要申报吗？
**A**：
- 个人使用的处方药（1-2 个月用量）通常不需要申报
- 大量药品或特殊药品需要提前申请许可
- 建议携带医生处方和药品说明书

### Q8：可以携带肉类制品吗？
**A**：❌ **严禁携带**任何肉类制品（包括火腿、香肠、肉干、肉松等），违者将被没收并可能罚款。

### Q9：入境时需要出示哪些文件？
**A**：
- 护照（有效期 6 个月以上）
- K-ETA 批准通知
- e-Arrival Card QR 码
- 往返机票（可能抽查）
- 酒店预订单（可能抽查）
- 足够的旅费证明（可能抽查）

### Q10：转机需要 K-ETA 吗？
**A**：
- 如果在韩国转机**不出境**（国际转国际），不需要 K-ETA
- 如果需要**入境韩国**再转机，必须申请 K-ETA
