# HTML to PPTX Converter

Converts HTML presentations generated by frontendslides into native PowerPoint (PPTX) files with virtual environment management and optional frontendslides integration.

## Architecture

```
html2pptx/
├── SKILL.md              # Skill manifest and documentation
├── requirements.txt      # Python dependencies
├── src/                  # Source code
│   ├── main.py           # Main entry point
│   ├── converter.py      # HTML to PPTX conversion logic
│   ├── venv_manager.py   # Virtual environment management
│   └── skill_integration.py # Frontendslides integration
├── tests/                # Test files
│   └── test_converter.py
└── examples/             # Example usage files
```

## Usage

```bash
# Direct HTML to PPTX conversion
python src/main.py input.html -o output.pptx

# End-to-end generation (conceptual)
python src/main.py "Create a presentation about..." 
```

## Features

- Converts HTML slides to PowerPoint format with preserved structure
- Automatic Python virtual environment management
- Smart input detection for both HTML files and user requests
- Maintains visual elements like charts and images
- Handles different slide types appropriately
