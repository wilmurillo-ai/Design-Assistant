## 失败处理示例

### 情况1：小红书搜不到相关内容

**用户输入：**
> "找个隔音特别好的酒店，要完全安静的"

**处理：**

```bash
# Step 1: 尝试多个关键词
opencli xiaohongshu search "隔音好的酒店"
opencli xiaohongshu search "安静酒店 推荐"

# 如果还是没有相关内容
```

**输出：**

```
抱歉，在小红书上没有找到"隔音特别好的酒店"相关内容。

建议：
1. 换个说法试试："安静酒店"、"睡眠友好酒店"
2. 在预订时备注"需要安静的房间"，酒店通常会尽量安排
3. 查看飞猪评分中的"环境"分数，选择环境评分高的酒店
```

### 情况2：酒店在飞猪搜不到

**处理：**

```
在小红书上找到了【某某民宿】的推荐，但飞猪上暂时没有预订信息。

可能原因：
- 该酒店未入驻飞猪
- 酒店名称不准确

建议：
1. 直接联系酒店预订（搜索酒店官方电话）
2. 尝试其他预订平台（携程、美团等）
3. 查看酒店官方公众号或小程序
```

### 情况3：用户需求太模糊

**用户输入：**
> "推荐个酒店"

**处理：**

```
好的！为了给你更精准的推荐，需要了解一些信息：

1. 目的地是哪里？
2. 入住时间？
3. 预算范围？
4. 有什么特殊需求吗？（如：带宠物、适合老人、亲子等）

告诉我这些，我来帮你找最适合的酒店！
```

## 最佳实践

1. **多关键词搜索**：一个需求可能需要多个搜索词
   - "宠物友好" → "带狗住"、"宠物入住"、"狗狗友好"
   - "安静酒店" → "隔音好"、"睡眠友好"、"远离闹市"

2. **信息交叉验证**：
   - 小红书评价 + 飞猪评分
   - 用户实拍图 + 酒店官方图
   - 多个用户的评价综合判断

3. **标注信息来源**：
   - "信息来自小红书用户分享"
   - "建议入住前与酒店确认具体设施"

4. **价格和预订信息**：
   - 飞猪价格仅供参考，实际价格以预订时为准
   - 提供预订链接，方便用户直接下单
