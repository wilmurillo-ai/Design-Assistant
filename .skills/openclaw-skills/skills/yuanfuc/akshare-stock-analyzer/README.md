# Akshare-Stock-Analyzer

> 默认显示中文说明；如需英文请查看 [English README](README.en.md)。

Akshare-Stock-Analyzer 是一个基于 Akshare 的轻量级 A 股分析工具，用来：
- 看“今天”一只股票的盘中行情（价格、涨跌幅、成交量、日内高低）；
- 看最近一段时间（一周到几个月）的日线趋势、信号和评分。

你可以把它理解成一个“短线技术面体检助手”。

---

## 怎么跟它对话

不需要记命令行，只要用自然语言提问，例如：

- 「今天平安银行的行情怎么样？」
- 「帮我看一下中国太保最近一周和今天的走势。」
- 「过去三个月，方大炭素是偏多头还是空头？」
- 「这只股现在适合持有还是该减仓？」

系统会在后台调用本项目的 today / fetch / analyze 脚本，给出：
- 今日盘中快照：价格、涨跌幅、成交量、日内高低；
- 日线趋势、周涨跌幅、今日涨跌幅；
- 交易信号（buy / sell / hold）和 0~10 的评分；
- 以及均线结构、MACD、RSI 等辅助信息。

---

## 趋势分析方法（简述）

内部的分析逻辑主要基于：

- 日线 K 线：最近一段时间的收盘价、最高价、最低价、成交量；
- 均线系统：MA7 / MA14 / MA20 / MA60 的排列和价格所处位置；
- MACD：金叉/死叉、在 0 轴上还是下、动能是增强还是减弱；
- RSI：当前强弱区间、是否接近超买/超卖、是走强还是走弱；
- 趋势强度与风险：
  - `trend_strength` 量化多空强弱；
  - `reversal_hint` 用来给顶部/底部的转折预警；
  - `risk_level` 标记是否存在追高/杀跌或趋势过度发散的风险。

综合这些指标，模型会给出：
- `trend`：当前是多头、空头还是震荡；
- `weekly_change_pct` / `today_change_pct`：一周和今天涨跌幅；
- `signal`：偏向 buy / sell / hold 的操作建议；
- `score`：0~10 的短期技术面打分。

你可以把输出理解为：
- 趋势 + 动能 + 情绪 的简化合成结果，只做技术参考。

---

## 风险提示

- 所有结论仅基于技术指标和价格行为，不构成任何投资建议；
- 真实交易还需要结合基本面、消息面、仓位和风险承受能力综合判断；
- 实时行情依赖外部接口和网络环境，偶尔可能出现短暂不可用或延迟的情况。
