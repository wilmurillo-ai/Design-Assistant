# Easy2Pe Knowledge Base

Easy2Pe is an Indian multi-service recharge and payments platform offering mobile recharge, bill payments, travel booking, PAN services, and more through a single platform.

## Quick Reference

- **Website:** https://easy2pe.com
- **Market:** India
- **Users:** 5,000+ consumers
- **Key value prop:** Multiple services on a single platform

## Services at a Glance

| Service | Details |
|---------|---------|
| Prepaid & DTH | Mobile recharge, DTH recharge, live response |
| Bill Payments | Postpaid, landline, broadband, electricity, water, insurance (via BBPS) |
| M-POS | Mini ATM, 24/7, max commission for agents |
| AEPS | Aadhaar-linked banking |
| Travel | Domestic buses (8,020+ routes), domestic/intl flights & hotels |
| PAN Services | New registration, edits/corrections |
| Gift Cards | 8 categories (entertainment, fashion, electronics, etc.) |
| E-Commerce | Marketplace + development services |
