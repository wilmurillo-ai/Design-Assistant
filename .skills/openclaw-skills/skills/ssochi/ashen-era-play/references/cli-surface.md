# CLI Surface

This file is not a guide. It only keeps the minimum launch facts. During play, follow the live in-game prompts.

## Default launch command

```bash
scripts/run-packed-cli.sh -- play --seed 42 --class ash_walker --locale en
```

If the user specifies values, use the user's values instead of the defaults.

## Available classes

- `ash_walker`
- `shadow_blade`
- `spirit_weaver`

## Boundaries

- Use `play` for real sessions
- Do not use `autoplay`
- Do not use `scout`
- Do not type `whoisyourdad`
- Do not use any `debug` command
- Default locale is `en`
- Switch to `zh` if the user explicitly wants Chinese
