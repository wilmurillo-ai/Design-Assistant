# 龙虾 MBTI 诊断参考手册

## 诊断维度详解

### 维度一：E/I — 互动能量方向

核心洞察：I 型用户反而可能跟龙虾聊得更多更密——因为龙虾是安全的倾诉对象。E 型用户有丰富的线下社交，跟龙虾的互动反而更随意稀疏。因此不能用"聊得多=外向"来判断。

| 指标 | E（外向龙虾） | I（内向龙虾） |
|------|---------------|---------------|
| 追问密度 | 稀疏，回复间隔长，有一搭没一搭 | 密集，收到回复后很快追问，连续深挖 |
| 话题来源 | 聊外部见闻、社交活动、他人故事 | 聊内心想法、个人感受、自我探索 |
| 提及他人频率 | 高，经常出现朋友/同事/家人 | 低，对话中很少涉及其他人 |
| 对话深度 | 浅层广泛，像跟朋友闲聊 | 深层聚焦，像在写日记或做心理咨询 |
| 社交场景 | 经常分享聚会、团队、社交经历 | 很少提及社交场景，更关注独处体验 |

### 维度二：S/N — 信息获取方式

| 指标 | S（实感龙虾） | N（直觉龙虾） |
|------|---------------|---------------|
| 问题类型 | "怎么做"、"多少"、"什么时候" | "为什么"、"如果...会怎样" |
| 关注焦点 | 具体细节、步骤、事实 | 大方向、可能性、隐含含义 |
| 信息偏好 | 要求明确清晰的答案 | 喜欢开放性的探讨 |
| 表达方式 | 描述具体场景和经历 | 使用类比、隐喻、抽象概念 |
| 对答案的反应 | 满足于具体可操作的回答 | 会在答案基础上继续延伸 |

### 维度三：T/F — 决策判断方式

| 指标 | T（思考龙虾） | F（情感龙虾） |
|------|---------------|---------------|
| 追问方向 | "为什么是这样"、"有什么依据" | "你觉得呢"、"这样好不好" |
| 评价标准 | 逻辑是否自洽、效率是否最优 | 是否照顾到感受、是否和谐 |
| 反驳方式 | 指出逻辑漏洞、要求数据 | 表达不同感受、诉诸价值 |
| 感谢频率 | 较少，拿到答案就走 | 较多，经常表达感谢和认可 |
| 讨论冲突 | 对事不对人，直接说问题 | 先肯定再提意见，注意措辞 |

### 维度四：J/P — 生活应对方式

| 指标 | J（判断龙虾） | P（知觉龙虾） |
|------|---------------|---------------|
| 对话结构 | 有明确目的，步步推进 | 随性发散，想到什么聊什么 |
| 任务处理 | 要求清单、计划、截止日期 | 享受过程，不急于得结论 |
| 话题切换 | 一个话题聊完才换下一个 | 频繁跳转话题 |
| 对结论的需求 | 强，希望有明确收尾 | 弱，可以留着开放式结尾 |
| 时间模式 | 固定时段互动，有规律 | 随机时间互动，无固定规律 |

## 评分算法参考

每个维度的评分逻辑：

```
维度得分 = Σ(各指标匹配度 × 权重) / 总权重

E/I 维度：
  - 追问密度权重：30%
  - 话题来源权重：25%
  - 提及他人频率权重：20%
  - 对话深度权重：15%
  - 社交场景权重：10%

S/N 维度：
  - 问题类型权重：40%
  - 关注焦点权重：25%
  - 表达方式权重：20%
  - 对答案的反应权重：15%

T/F 维度：
  - 追问方向权重：35%
  - 评价标准权重：25%
  - 反驳方式权重：20%
  - 感谢频率权重：10%
  - 讨论冲突权重：10%

J/P 维度：
  - 对话结构权重：30%
  - 任务处理权重：25%
  - 话题切换权重：20%
  - 对结论的需求权重：15%
  - 时间模式权重：10%
```

得分 > 50 取该维度左侧字母，≤ 50 取右侧字母。

## 边界情况处理

- **零互动（对话历史为空）**：不给出任何结果，引导用户先聊天，积累足够数据后再诊断
- **某维度得分接近 50（45-55）**：在描述中提到"你在这个维度上比较平衡"
- **数据量极少（< 5 轮）**：仅给出最有信心的 1-2 个维度判断，其余标注"待观察"
- **用户行为矛盾**：优先看追问行为（维度 C），因为追问是最真实的反应
