# Stock Analysis 配置

## 自动数据获取

### 启用自动获取
- 自动获取：是
- 数据源优先级：腾讯财经 > 东方财富 > 新浪财经
- 缓存时间：5 分钟
- 超时时间：10 秒

### 数据获取流程
1. 分析请求触发
2. 自动调用 fetch-price.sh 获取实时股价
3. 自动调用 fetch-eastmoney.sh 获取财务数据
4. 自动调用 fetch-kline.sh 获取 K 线数据
5. 自动调用 technical-analysis.sh 获取技术面
6. 生成完整分析报告

### 备用方案
如果自动获取失败：
1. 使用缓存数据（如有）
2. 使用公开财报数据估算
3. 提示用户数据可能不是最新
