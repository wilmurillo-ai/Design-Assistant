# 🏠 家里有什么

解决"东西在哪"、"家里有什么"的日常困惑。

记录家中所有物品存放位置，需要时快速找到。

**适用场景：** 家庭物品管理、搬家整理、清点物资、记录杂物

## 功能概述

| 功能 | 描述 | 示例 |
|------|------|------|
| 初始化套房 | 创建新房并定义房间 | `设置一套房叫工作室，有客厅、卧室` |
| 新增房间 | 在已有套房下添加房间 | `给工作室加一个房间叫厨房` |
| 新增家具 | 在房间中添加家具 | `客厅有个电视柜` |
| 添加物品 | 记录物品存放位置 | `把遥控器放在客厅电视柜里，数量2` |
| 查询物品 | 快速定位物品位置 | `我的遥控器在哪？` |
| 查看套房 | 列出所有物品清单 | `查看工作室` |
| 物品搬动 | 变更物品存放位置 | `把遥控器搬到卧室衣柜` |
| 删除物品 | 移除物品记录 | `删除遥控器` |
| 搜索物品 | 模糊搜索物品 | `搜索遥控` |
| 备份 | 备份数据到文件 | `备份数据` |
| 列出备份 | 查看所有备份 | `列出备份` |
| 恢复 | 从备份恢复数据 | `恢复 20260325_143000` |
| 导出 | 导出数据（文本/JSON） | `导出数据` |

## 数据结构

```
🏠 套房 (Suite)
  └── 📦 房间 (Room)
      └── 🪑 家具 (Furniture)
          └── 📦 物品 (Item)
```

## 物品属性

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | UUID | ✅ | 唯一标识 |
| name | string | ✅ | 物品名称 |
| quantity | number | ✅ | 数量，默认1 |
| addedAt | datetime | ✅ | 放入时间 |
| updatedAt | datetime | ✅ | 更新时间 |
| note | string | ❌ | 备注 |

## 时间戳字段（所有层级）

| 层级 | 字段 | 说明 |
|------|------|------|
| 套房 | createdAt, updatedAt | 创建/更新时间 |
| 房间 | createdAt, updatedAt | 创建/更新时间 |
| 家具 | createdAt, updatedAt | 创建/更新时间 |
| 物品 | addedAt, updatedAt | 放入/更新时间 |

## 命令格式参考

### 初始化套房
| 格式 | 示例 |
|------|------|
| `设置一套房叫{名称}，有{房间1}、{房间2}...` | `设置一套房叫老家，有客厅、主卧、次卧` |
| `添加套房：{名称}，包含{房间1}和{房间2}` | `添加套房：工作室，包含客厅和厨房` |

### 添加房间
| 格式 | 示例 |
|------|------|
| `给{套房}加一个房间叫{房间}` | `给工作室加一个房间叫储物间` |
| `{套房}新增{房间}` | `工作室新增阳台` |

### 添加家具
| 格式 | 示例 |
|------|------|
| `{房间}有个{家具}` | `客厅有个电视柜` |
| `{房间}有个{数量}{家具}` | `主卧有个大衣柜` |

### 添加物品
| 格式 | 示例 |
|------|------|
| `把{物品}放在{房间}{家具}里` | `把遥控器放在客厅电视柜里` |
| `把{物品}放在{房间}{家具}里，数量{数量}` | `把被子放卧室衣柜里，数量2` |

### 查询
| 格式 | 示例 |
|------|------|
| `我的{物品}在哪？` | `我的钥匙在哪？` |
| `{物品}在哪里` | `遥控器在哪里` |
| `查看{套房}` | `查看工作室` |

### 搜索
| 格式 | 示例 |
|------|------|
| `搜索{关键词}` | `搜索遥控` |
| `找一下{关键词}` | `找一下手机` |

### 搬动
| 格式 | 示例 |
|------|------|
| `把{物品}搬到{房间}{家具}` | `把遥控器搬到卧室衣柜` |

### 删除
| 格式 | 示例 |
|------|------|
| `删除{物品}` | `删除遥控器` |

### 备份与恢复

### 导出

## 技术信息

- **数据存储**：`{workspace}/data/home_storage.json`（默认为 `~/.openclaw/workspace/data/home_storage.json`）
- **备份目录**：`{workspace}/data/backups/`
- **数据格式**：JSON
- **依赖**：Python 3
