# 伤寒论 · 倪海厦核心笔记

> 倪海厦注解 | 2010-05-09定稿 | 整理自《倪注伤寒论》原文摘录

## 一、六经辨证框架

倪师以**阴阳平衡**为伤寒论核心指导思想，强调"张仲景都是在阳上面治疗，不管病在哪里，开的大多是用阳药"。

### 1. 太阳病（表证）

| 分类 | 主证 | 脉象 |
|------|------|------|
| **中风** | 发热、汗出、恶风 | 脉浮缓 |
| **伤寒** | 恶寒、体痛、呕逆、无汗（或已发热/或未发热） | 脉阴阳俱紧 |

**核心病机**：
- 中风：风邪袭表，营卫不和，汗出病解（水排得出去）
- 伤寒：寒邪束表，水停体表，水排不出去（故骨节酸痛）

> 倪师特别强调："滤过性病毒千百种，只分这两种就好了，按症处方，就是辨症论治。"

**节气定义**：霜降后至春分前，被寒气所伤称为伤寒（非疫病，不传染）。

### 2. 阳明病（里实热证）

- **病位**：消化系统（阳明属胃、大肠经）
- **主证**：发热、不恶寒、便秘、胃家实
- **代表方**：调胃承气汤、小承气汤、大承气汤
- **牙病**：上牙手阳明大肠经，下牙足阳明胃经 → 牙龈红肿烂掉=肠胃太热

### 3. 少阳病（半表半里）

- **病位**：胸胁部位
- **主证**：往来寒热、胸胁苦满、心烦喜呕、嘿嘿不欲饮食
- **代表方**：小柴胡汤

### 4. 太阴病（里虚寒证）

- **病位**：脾、胃
- **主证**：腹满、吐利、食不下
- **代表方**：理中汤、四逆汤

### 5. 少阴病（里虚寒重证）

- **病位**：心、肾
- **主证**：脉微细、但欲寐、四肢厥逆、下利清谷
- **代表方**：四逆汤（生附子壮肾阳 + 干姜炙甘草壮脾阳）

### 6. 厥阴病（寒热错杂证）

- **病位**：肝、心包
- **主证**：寒热错杂、上热下寒、消渴、气上撞心
- **代表方**：麻黄升麻汤（寒热并用）、干姜黄连黄芩人参汤

## 二、各经证治要点

### 太阳病

**桂枝汤证（中风）**：
- 症状：发热、汗出、恶风、头痛、脉浮缓
- 舌象：舌苔薄白
- 变证：
  - 兼喘 → 桂枝汤加厚朴杏仁
  - 兼湿热（酒客）→ 葛根芩连汤
  - 发汗过度 → 桂枝加附子汤（汗漏不止、恶风、小便难）

**麻黄汤证（伤寒）**：
- 症状：无汗、恶寒、体痛、呕逆、脉阴阳俱紧
- 特点：水排不出去 → 全身骨节酸痛

**葛根汤证**：
- 病邪进入太阳阳明之间，项背强几几

**麻杏甘石汤证**（发汗后表邪入肺/肺炎）：
- 症状：汗出而喘、无大热
- 特点：表已虚不可再发汗，用石膏清肺热

### 阳明病

**调胃承气汤**：
- 主证：便秘、上牙痛、牙龈红肿烂掉（肠胃太热）
- 大黄：消炎；芒硝：破瘀软坚

**小承气汤 / 大承气汤**：
- 主证：便秘腹痛、潮热、谵语

### 太阴/少阴病

**甘草干姜汤**（二神汤）：
- 吐血神方：胃寒吐血用热药
- 干姜壮脾阳，炙甘草壮胸阳

**四逆汤**：
- 主证：四肢厥逆、小便清利、下利清谷
- 生附子壮肾阳 + 干姜炙甘草壮脾阳

**芍药甘草汤**（去杖汤）：
- 治脚弱无力、腹痛
- 白芍破血活血

### 厥阴病

**麻黄升麻汤**：
- 主证：表证无汗、咽喉溃烂、肠中虚寒下利
- 寒热错杂：上焦热（石膏、黄芩） + 下焦寒（干姜、白术）

**干姜黄连黄芩人参汤**：
- 主证：食入口即吐（上热下寒）
- 虚热非实热

## 三、倪海厦独特见解

### 1. 阴阳观念（核心思想）

> **"人的阴阳是平衡的，阴不足时不用滋阴要壮阳，阳一壮就起来了；阳不足的时候，也要壮阳。"**

- 张仲景不管病在哪里，开的大多是用**阳药**
- "把阴拉起来" = 壮阳
- 治病重点在**阳**，不在阴

### 2. 处方思想

- **桂枝汤可做补药**：平常可以保养用
- **甘草作用**：
  - 停在胃让药能直接到肺
  - 分解未消化的食物
  - 解百毒
- **石膏**：白虎，非常寒凉，去肺/胃壮热
- **杏仁**：去肺热、化痰、润肤（肺主皮毛）
- **厚朴**：去脾湿，收敛

### 3. 脉诊要诀

- **寸为阳，尺为阴**
- 阴阳脉：寸脉与尺脉皆浮紧 = 伤寒
- **弦脉 = 水病**
- 脉微弱 = 本有久寒

### 4. 舌诊辨证

| 舌象 | 意义 |
|------|------|
| 舌苔薄白 | 正常/表证 |
| 舌苔厚腻黄 | 湿热（酒客） |
| 舌红无苔 | 阴虚 |

### 5. 特殊辨证

- **渴不渴**：不渴 = 有水饮
- **结胸 vs 协热利**：误下后根据症状判断
- **脾胃功能恢复**：血回到经络 → 出血自止

## 五、临床应用要点

### 1. 处方原则

- **主证为主，兼证兼治**：有主证就开主方，兼证同时加药
- **新病优先**：张仲景治病注重新病，旧恙兼及而已
- **问胃口**：气喘胃口好→麻杏甘石汤；胃口不好→大青龙汤（甘姜枣开胃）

### 2. 药量比例

**麻黄杏仁甘草石膏汤**：
- 麻黄三钱，石膏四~六钱，杏仁二~五钱，甘草二钱
- 先煮麻黄去上沫（去气留味，不让麻黄冲到头部）

**大青龙汤**：
- 麻黄六钱（比麻杏甘石汤重），石膏四钱，杏仁二钱
- 杏仁+石膏 = 与麻黄量相等 → 不伤津液

**调胃承气汤**：
- 大黄:炙甘草 = 2:1
- 炙甘草不可太重，否则大黄芒硝被固住

### 3. 临床辨证关键

- **水病诊断**：不渴 + 舌苔湿 + 脉弦
- **寒湿辨别**：拉出黏稠湿湿大便 = 寒湿排出
- **热喘**：虚汗多、舌苔黄、气喘久（虚证）
- **寒喘**：痰清稀、舌苔白（实证）

### 4. 特殊情况

- **吐家**（胃溃疡/胃癌）：不可给桂枝汤 → 芍药甘草汤
- **酒客**：不给桂枝汤 → 葛根芩连汤
- **胰脏癌**：麻黄升麻汤 / 旋覆代赭汤加干姜吴茱萸
- **术后肠黏连**：独参汤（人参单味）

### 5. 针灸配穴（气喘）

- 公孙、内关（胃心胸）
- 足三里、太白（土生金）
- 临泣、太白（合化土）
- 云门、肺俞

## 六、核心理念总结（倪师语录）

1. **"滤过性病毒千百种，只分中风、伤寒两种，按症处方即可。"**
2. **"张仲景都是在阳上面治疗，开的大多是用阳药。"**
3. **"阴不足时不用滋阴要壮阳，阳一壮就起来了。"**
4. **"甘草可以解百毒。"**
5. **"治病注重新病，旧恙兼及而已。"**
6. **"石膏的目的是去热，杏仁是去痰、润肺。"**
