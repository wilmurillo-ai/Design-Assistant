# 黄帝内经 · 倪海厦核心笔记

> 整理自倪海厦《黄帝内经》原文摘录（461页原文关键页面），以倪海厦视角提炼，非泛泛养生。

---

## 一、核心理论体系

### 1. 天地之道——养生的最高准则
倪海厦强调，《黄帝内经》所有篇章的核心就一句话：**"法于阴阳，和于术数"**。养生、治病、诊断皆以此为准绳。天地有阴阳消长，人必须顺应，不能违背。

> 歧伯回答，上古之人，知"天地之道"，遵循阴阳之消长，行事符合天文术数历法不背天而行。

### 2. 真气（免疫系统）是健康之本
倪海厦将"真气"直接类比为现代医学的**免疫系统**。真气在体内畅行无阻，人则健康；真气受损，人则生病。

> 人体内的真气"免疫系统"必能强盛起来，因无欲而更强，因无耗而更壮。

### 3. 形与神俱——形神兼备方能尽终天年
健康的标准不仅是形体不衰，更是**精神内守**。形神俱备，才能"尽终其天年，度百岁乃去"。

> 食饮有节，起居有常，不妄作劳。故能形与神俱，而尽终其天年，度百岁乃去。

### 4. 肾为先天之本，脾胃为后天之本
男女生命周期以肾气盛衰为轴线：女子以七为节，男子以八为节。肾气决定生殖、生长、衰老。

倪海厦特别指出：**肾主水，受五藏六府之精而藏之**，五脏六腑的精华都会聚于肾，肾是全身精气的总汇。

> 女人七岁时，肾气始盛……七七任脉虚，太冲脉衰少，天癸竭，地道不通，故形坏而无子也。

### 5. 治未病——圣人之道
倪海厦多次引用**"圣人不治已病，治未病；不治已乱，治未乱"**，强调预防医学才是中医的最高境界。

> 是故圣人不治已病，治未病。不治已乱，治未乱，此之谓也。

## 二、倪海厦独特见解

### 1. 把"真气"直接等同于"免疫系统"
倪海厦在讲解时多次直接标注"真气=免疫系统"，用现代语言解释古代概念，让学生秒懂。这是他教学的核心特色——**让中医说现代话**。

> 人体内的真气"免疫系统"必能强盛起来，因无欲而更强，因无耗而更壮。

### 2. 四季脉象的临床应用
倪海厦特别强调脉象与四季的对应关系：
- 春应中规（圆滑）
- 夏应中矩（洪大）
- 秋应中衡（浮毛）
- 冬应中权（沉石）

> 微妙在脉，不可不察。察之有纪，从阴阳。

### 3. 情志伤人——怒伤阴，喜伤阳
倪海厦详解情志致病的机理：
- **暴怒伤阴**：伤津液
- **暴喜伤阳**：伤动能
- 气一旦离经叛道向上行，血脉过满则体形必伤

> 暴怒伤阴，暴喜伤阳。厥气上行，满脉去形。

### 4. 脉证合参——由外知内
倪海厦非常重视脉象在诊断中的核心地位，指出诊断的五个关键：
1. 脉依四季如何变化
2. 如何测知病之所在
3. 如何察知病由内而起
4. 如何知道病由外而入

### 5. 肾气余刃——老人有子之谜
倪海厦解释为何有些老人仍能生育：**其肾气尚有余刃（残余之力）**，并非筋骨解堕，而是肾气未完全衰竭。

> 今五脏皆衰，筋骨解堕，天癸尽矣。故发鬓白，身体重，走路不直，故而无子。

## 三、临床应用要点

### 1. 阴阳不相应，病名曰关格
脉象与四季不合：
- 应太过而不足 → 为精（健康）
- 应不足而有余 → 为消（已损）
- 阴阳不容，互相抗争相隔 → **死兆**

### 2. 气血结滞的诊断（倪海厦详述）
| 症状 | 病机 |
|------|------|
| 阳气结，四肢肿 | 气伤形 |
| 阴气结，便血（一升≈40CC） | 形伤气 |
| 阴阳结，阴盛阳弱 | 石水（死水），小腹肿，小便不出 |
| 二阳结 | 消（渴饮消瘦） |
| 三阳结 | 隔（大小便不出） |
| 三阴结 | 水（水肿） |
| 一阴一阳结 | 喉痹 |

### 3. 特殊脉象临床意义
- 阴搏阳别 → 有子（怀孕）
- 阴虚阳搏 → 崩（血崩）
- 阳加于阴 → 汗
- 三阴俱搏 → 二十日夜半死

### 4. 脾胃为五脏六腑之海
太阴（脾）为三阴，阳明（胃）为五脏六腑之海。脾胃以膜相连，脾为胃行其津液到三阴三阳，**四肢不得禀胃气**则病。

> 藏府各因其经而受气于阳明，故为胃行其津液。

## 四、养生核心原则

### 1. 饮食有节，起居有常，不妄作劳
倪海厦特别强调：**今时之人不然——以酒为浆，以妄为常，醉以入房，以欲竭其精，以耗散其真**。这是半百而衰的根本原因。

### 2. 虚邪贼风，避之有时
上古圣人之教：身体空虚为邪之始生，风气糜烂耗败人心为贼风。**随时避之，保持身心平淡无求**。

### 3. 恬淡虚无，真气从之
- 志闲而少欲
- 心安而不惧
- 形劳而不倦
- 气从以顺，各从其欲，皆得所愿

### 4. 春夏养阳，秋冬养阴
违背四时之气的后果：
- 逆春气 → 少阳不生，肝气内变
- 逆夏气 → 太阳不长，心气内洞
- 逆秋气 → 太阴不收，肺气焦满
- 逆冬气 → 少阴不藏，肾气独沉

### 5. 冬季养生——闭藏之道
- 早睡晚起，必待日光
- 精神藏志隐匿不出
- 避寒近温，汗不出皮肤
- 不过度运动流汗伤身
- **违反则伤肾，春季生痿病、骨病、四肢麻木**

## 五、名言/金句（倪海厦原话摘录）

> **"以其德全不危也。"**
上古之人度百岁而不衰，因其道德完美，不图一丝侥幸之心而招危。

> **"道者圣人行之，愚者佩之。从阴阳则生，逆之则死。"**
得道者圣人奉行，愚者只是佩戴在身上。从阴阳则生，逆之则死。

> **"重阴必阳，重阳必阴。"**
阴阳是动态平衡，阴盛到极点必转阳，阳盛到极点必转阴。这是病机传变的根本规律。

> **"夫四时阴阳者，万物之终始也，死生之本也。逆之则灾害生，从之则苛疾不起。"**
四时阴阳是万物从生到死的根本规律，逆之则灾害生，从之则苛疾不起。

> **"故曰：冬伤于寒，春必温病。春伤于风，夏生飧泄。夏伤于暑，秋必痎疟。秋伤于湿，冬生咳嗽。"**
四时伏邪发病规律——这是倪海厦临床辨证的核心依据。

> **"肾者主水，受五藏六府之精而藏之，故五藏盛乃能写。"**
肾是全身精气的储藏室和分配中心，五脏强盛才能把精气泻出使用。

> **"脾者，土也。治中央。常以四时长四藏，各十八日寄治，不得独主于时也。"**
脾土治中央，通过四季将精气输布到肝、心、肺、肾，四季各占十八日。
