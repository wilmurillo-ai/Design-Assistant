## 二、倪海厦医学理论体系

### 1. 学术渊源

倪海厦自称"汉唐中医"，其学术思想来源于：

- **《内经》** - 基础理论，构建天人合一、阴阳五行框架
- **《难经》** - 脉学、脏腑辨证
- **《伤寒论》** - 六经辨证体系
- **《金匮要略》** - 杂病辨证
- **《千金方》《外台秘要》** - 方剂补充

**核心特点**：经方派，以汉张仲景、唐孙思邈的方剂为主。

### 2. 辨证方法

倪海厦采用**六经辨证与脏腑辨证双轨并行**：

```
六经辨证 → 《伤寒论》为主 → 辨病邪传变
脏腑辨证 → 《内经》《难经》《千金》→ 辨脏腑虚实
```

### 3. 病因学观点（重点！）

倪海厦将现代疾病分为三类病因：

| 病因类型 | 占比 | 说明 |
|----------|------|------|
| **心理因素** | 70%+ | 七情六欲致病，精神压力大 |
| **文明病** | ~20% | 饮食添加剂、空调、抗生素等 |
| **正常生病** | ~10% | 天气外因 |

> **核心观点**：现代疾病七成以上由心理压力造成，西医称的"亚健康"实际是不健康，是中医大显身手的时候。

### 4. 疾病观

- **癌与疮同源**：体内毒气向外发则为疮痈，向内闭则为癌症
- **感冒发烧是自我保护**：退烧是压制正气，使毒气内陷
- **乳汁逆流成癌**：癌症根源在于乳汁不能正常下行而逆流入脏腑
- **激素的危害**：强行调动肾精，等于提前耗尽人的寿命

## 三、方剂核心功效与应用

### 1. 补肾系列（HT-46/48/49/99）

**六味地黄丸（HT-46）** - 肾阴虚
- 组成：熟地、山茱萸、山药、泽泻、茯苓、丹皮
- 应用：肾阴不足，腰膝酸软，头晕耳鸣

**金匮肾气丸（HT-48）** - 肾阳虚
- 组成：六味加桂枝、附子
- 应用：肾阳不足，水肿、畏寒
- **重要提醒**：糖尿病等肾虚证不能用六味，必须用桂附八味！

**起阳丸（HT-49）** - 元阳不足
- 功效：振奋阳气、生精
- 比较：可与威而刚对比，无西药副作用

**斑龙丸（HT-99）** - 补肾填精
- 功效：填精补髓，强壮筋骨

### 2. 活血化瘀系列

**抵挡丸（HT-11）** - 太阳蓄血证
- 主治：子宫干血、膀胱受伤、血积膀胱
- 原文对应：《伤寒论》太阳蓄血证
- 症状：人自发狂

**紫金丹（HT-35）** - 胸腔内伤
- 主治：上身瘀血、胸胁伤痛

**天青丸（HT-36）** - 妇女痛经
- 主治：痛经兼治下身瘀血

**强力天青丸（HT-60）** - 重症瘀血
- 功效比HT-36更强

### 3. 糖尿病专方（HT-24）

**降血糖中医之上中下三消渴症**

**核心病机**：肾不藏精 → 肝不藏精 → 视力模糊/白内障

**倪海厦治糖尿病思路**：
```
强心必先强肝（肝主双眼）→ 强肝 → 强心 → 避免胸痹证
强心 → 强肾 → 避免肾衰竭
```

**治法**：强肾健脾，恢复人体正常代谢功能

**重要禁忌**：晚餐后、睡觉前**绝对禁食**（这是主要病因）

### 4. 心脏病系列（HT-87/乌梅丸）

**倪海厦治心脏病思路**：
- 心脏病人心包膜上充满黏稠痰湿
- 舌苔黄厚腻 → 心包肺中有湿痰
- 治法：去湿痰 + 强心

**乌梅丸（HT-40）**新用：
- 原治：蛔虫病
- 新用：去除心包膜黏痰湿（2-3天舌苔尽退）
- 预防心脏病复发、治疗晕眩

## 四、倪海厦独特见解

### 1. 对西医的批判

| 西医问题 | 倪海厦观点 |
|----------|------------|
| 退烧消炎 | 压制正气，使毒气内陷，致癌根源 |
| 抗生素滥用 | 使体内毒气发不出来 |
| 激素治疗 | 提前耗尽肾精，缩短寿命 |
| 降糖药 | 控制血糖但最后还要肾衰竭，无效控制 |
| 疫苗 | 封压胎毒，今后变癌症 |

### 2. 对温病派的批判

- 温病派不懂内景变化
- 清热解毒压制法 → 造成麻疹不满浆 → 形成麻子
- 未得师传，不懂乳汁变化月经的真正机理
- 红斑狼疮用激素只是把肾精向外耗，无法根治

### 3. 师承与根基

倪海厦属于**正统纯中医经方派**：
- 师传于台湾（台湾中国古文化保存更完整）
- 有真正的师承内景功夫
- 其方剂来自《伤寒论》《千金》《外台》

### 4. 对乳汁与月经的独特解释（颠覆西医认知！）

**关键论点**：
- 月经不是子宫内膜代谢产物
- 月经是乳汁下行变化而来
- 乳汁下行入子宫为月经
- 乳汁逆流入心、血脉 → 红斑狼疮、血癌等

### 5. 治病次序观

```
急治标 → 缓治本 → 标本兼治
先安未受邪之地 → 已病防传
```

## 六、临床应用要点

### 1. 诊断要点

- **舌诊**：黄厚腻苔 → 心包肺有痰湿
- **脉诊**：肾气虚 → 糖尿病主要脉象
- **望诊**：面色、眼神

### 2. 用药原则

- **经方派不等于只用姜附**：要看医理，不是看药性寒热
- **有毒中药**：在有师承指导下可以使用，效果显著
- **桂枝黄连半夏白术**是常用组合（理肝气、清痰热）

### 3. 禁忌与注意事项

| 病证 | 禁忌 |
|------|------|
| 糖尿病 | 晚餐后、睡前禁食 |
| 心脏病人 | 禁生冷、黏腻、痰生之物 |
| 皮肤病 | 禁发物（鱼虾等） |
| 癌症 | 慎用寒凉压制法 |

### 4. 服药方法

- **HT-7胃痈丸**：三餐饭后15-20粒
- **HT-11抵挡丸**：早7点30粒，5小时无排再服30粒
- **乌梅丸**：早晚饭后20粒，重症30粒

### 5. 疾病预防观

> "真气内守，病安从来"——《内经》

- 正气充足则不易受邪
- 疫苗、激素是压制非透发
- 正常透发才能根治

## 七、经典原文摘录（重要）

### 1. 对西医治法的批判

> "现在西医已经走到穷途了，经常在我们看病当中就碰到很多，特别是小孩子受到的伤害最大。"

### 2. 对现代疾病病因的总结

> "七成甚至七成以上都是心理影响出来的毛病，除此之外两成是现在的称为'文明病'。只有一成，属于人的正常生病。"

### 3. 对中医优势的论述

> "当西医走向穷途的时候，便是我们中医发展的时候，所以大家应当对纯中医有信心。"

### 4. 对经方派定义

> "经方派不等于全部用姜附，要从医理上看。倪海厦的方子基本上是从汉之张仲景与唐之孙思邈出来的。"

### 5. 对糖尿病的论断

> "西医的控制是控制血糖，但是病人最终还要发生肾中毒尿毒症，这是属于无效的控制。"

## 八、学习方法建议

倪海厦中医思想的特点：
1. **懂的人一看就明白**——不保密，但需要根基
2. **去除浮躁心理**——不要只想找秘方
3. **从内经与难经顿悟**——重基础理论
4. **结合临床内景**——有师传懂脏腑变化

**核心提醒**：学中医要从根本上理解医理医道，不能只记方药名称，要理解疾病发生、变化、转归、治愈的全过程。
