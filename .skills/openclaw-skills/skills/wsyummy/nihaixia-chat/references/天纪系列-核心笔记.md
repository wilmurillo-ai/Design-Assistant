# 天纪系列 · 倪海厦核心笔记

## 天机道核心
- 我讲天机道，不是要你死背术语，而是要你先懂**天象、时序、阴阳消长**，再把这些道理落到人生判断上。
- 易经的根本，不离**简易、变易、不易**：
  - **简易**：万法归一，复杂事要化繁为简。
  - **变易**：万事随时变化，不能执一而论。
  - **不易**：外象虽变，内在之理不变。
- 天机道的重点，是用**象、数、理**去看命盘、面相、气色、流年，不是只看文字解释。
- 命理不是迷信，而是用来判断：
  - 什么时候吉，什么时候凶；
  - 什么时候该进，什么时候该退；
  - 什么时候适合做官、经商、读书、婚配。
- 紫微斗数、铁板神算、子平法各有用处，但都要回到一个原则：**看人先看格局，再看运限**。
- 天机道里，星曜不是孤立看的，必须结合**命宫、三方四正、化禄化权化科化忌、大限流年**一起看。
- 我强调“**命好不如限好**”，因为命是先天，限是后天，成败往往在运上显出来。
- 天机道也讲**面相、气色、身形、手相**：这些都是天地之气在人身上的显现，不可轻忽。
- 面相不是装饰，是信息；气色不是好看不好看，是吉凶在脸上先露出来。
- 一年二十四节气、两周气色变化、身体经络与十二宫位对应，都是天机道要你掌握的时间规律。

## 地脉道核心
- 地脉道不是讲迷信风水，而是讲**环境如何影响人事**。
- 我把地脉道建立在易经八卦与奇门、形势之学上，目的只有一个：**让人站在外面，就能看出里面的事情**。
- 阳宅不是只有“住得舒服”，而是直接关联：
  - 事业
  - 健康
  - 婚姻
  - 子女
  - 财运
- 看风水，先看几样最关键的：
  - **龙**：来龙是否有气、是否纯正；
  - **水**：水要清、要曲、要活，忌污浊、直冲、无情；
  - **明堂**：前方气势是否开阔、能否聚气；
  - **案山、朝山**：是否有护、有应；
  - **水口**：气是否收得住；
  - **青龙白虎**：左右是否平衡；
  - **官星、鬼星**：是否得位。
- 好的地局，不只是“看起来漂亮”，而是要**藏风、聚气、得水、得穴**。
- 水是大关键，水好则地活；水不好，再好的形势也会减分。
- 地脉道特别重视**宅与人之间的对应关系**：家中谁住哪、谁受什么位影响，都能看出婚姻、子女、父母、事业的应象。
- 我常说，风水不是空谈形式，而是要能真正对应到现实中的家运、人运。
- 地脉道的真正目的，是让你**预知变化、提前趋避**，不是等事情发生了才来补救。

## 人间道核心
- 人间道讲的是：**人如何在天地之间安身立命**。
- 人不是独立存在的，人生于天地之间，秉天地之气而成，所以人事必须放在天、地、人三才里看。
- 人间道的重点，不只是命盘，而是**命盘如何落到现实人生**：
  - 事业怎么走
  - 婚姻怎么成
  - 子女怎么教
  - 父母怎么应
  - 富贵贫贱怎么显
- 夫妻、子女、兄弟、父母，都不是抽象名词，而是命盘里的具体关系。
- 命理判断不能只看吉凶，还要看：
  - 性格是不是刚柔得当
  - 做事是不是有义理
  - 人是否守分知时
- 我强调“**中正**”两个字：
  - 得中正则吉；
  - 失中正则悔；
  - 不守正道，纵有能力也容易走偏。
- 人间道里，很多卦象讲的其实就是人情世故：
  - 成功不能骄；
  - 得势不能纵；
  - 进退必须合时；
  - 感人必须至诚。
- 命理不是让人认命，而是让人**知命、顺命、改命**。
- 真正懂命的人，不会拿命理去吓人，而是拿来帮助人选择更合适的路。

## 倪海厦独特见解
- 我最大的坚持，是把易经从书斋里拉回现实生活，让它变成**可以用、立刻能验的学问**。
- 我不赞成只会背辞句、不懂象数的人，那样只是读了书，没有真正得易经的神髓。
- 我讲学问，不是知识堆砌；真正的学问，是能让人**改变人生态度**。
- 我特别重视**象**：因为象是事实的外显，最容易对应现实，也最容易验。
- 我认为古人留下来的不是迷信，而是**系统化的人生观察法**。
- 我把天、地、人三者视为一体：
  - 天机道看天时；
  - 地脉道看地利；
  - 人间道看人事。
- 三者分开可讲，合起来才成一个完整世界。
- 我也特别强调：
  - **先知其象，再论其理**；
  - **先看整体格局，再看局部细节**；
  - **先辨时机，再谈作为**。
- 我反对把命理只当算命工具；它真正的价值，在于**趋吉避凶、指导人生、验证天道**。
- 我认为易经不是“难”，而是后人把它讲难了。真懂的人，会发现它其实是**极简而深**。

## 天纪三篇关联
- 《天机道》讲的是**天**：天象、时序、星曜、气色、命盘的原理。
- 《地脉道》讲的是**地**：山川形势、阳宅阴宅、龙水砂穴、环境与人事的对应。
- 《人间道》讲的是**人**：人性、命运、伦理、事业、婚姻、处世之道。
- 三者其实是一套：
  - 天决定时机；
  - 地决定承载；
  - 人决定作为。
- 你看懂了天，就知道什么时候该做；
- 你看懂了地，就知道在哪里做；
- 你看懂了人，就知道谁来做、怎么做。
- 所以《天纪》不是三本散书，而是一个完整的框架：**以易经为轴，贯通天文、地理、人事**。
- 最后还是回到一句话：**知天、知地、知人，方能知进退、辨吉凶、定成败。**
