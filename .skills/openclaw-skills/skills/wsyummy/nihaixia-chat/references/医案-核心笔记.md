# 倪海厦医案 · 核心笔记

> 整理自《倪海厦文集》2005年7月31日版
> 以倪海厦临床思路为准绳

---

## 一、医案分类

### 1. 外感类疾病
- **太阳表症**：普通感冒、流感
  - 太阳中风（桂枝汤症）：微汗、项强、头痛、发烧
  - 太阳伤寒（麻黄汤症）：恶寒、无汗、高热、身痛、骨节痛
  - 太阳温病（葛根汤症）：喉咙痛、流汗状态下所得感冒

### 2. 内伤类疾病
- **情志病**：喜、怒、忧、思、悲、恐、惊过度所致
  - 范进中举（过喜伤心）
  - 相思病（过思伤脾）
  - 恐惧症（过恐伤肾）

### 3. 六经传变疾病
| 阶段 | 病位 | 主要症状 |
|------|------|----------|
| 太阳 | 体表 | 怕冷/怕风、发烧、头痛 |
| 少阳 | 淋巴/内分泌 | 往来寒热、恶心呕吐、忽冷忽热 |
| 阳明 | 胃/大肠 | 燥热、饥饿、便秘 |
| 太阴 | 脾脏/胰脏 | 腹胀、四肢胀、纳差 |
| 少阴 | 心/肾 | 嗜眠、失眠、大小便异常、手脚麻 |
| 厥阴 | 肝/心包 | 但坐不得卧、上下焦热寒分离 |

### 4. 肿瘤类疾病
- 乳癌、肝癌、胃癌、大肠癌
- 舌癌、食道癌转红斑性狼疮
- 核心观点：**化疗后扩散者，百无一生**

### 5. 特殊案例
- **听力丧失案**（只能听到女声，男声听不到）→ 阳虚，肾阳不足
- **肺气肿案**：长期使用类固醇导致肺钙化
- **Chron's Disease + Lyme Disease**：长期抗生素无效
- **精神病案**：便秘沼气冲脑所致

## 二、倪海厦辨证思路

### 1. 六经辨证为核心框架
疾病传变顺序：**太阳 → 少阳 → 阳明 → 太阴 → 少阴 → 厥阴**

> "感冒病毒一路从太阳表症进入肝脏，就是C型肝炎。治疗时症状往回走，从厥阴治到太阳，病就好了。"

### 2. 八纲辨证为诊断工具
| 辨证维度 | 判断标准 |
|----------|----------|
| 阴阳 | 手脚温热为阳，脚冷为阴；寒热分离为阴盛格阳 |
| 表里 | 病在肌表还是在脏腑 |
| 虚实 | 有余还是不足 |
| 寒热 | 口渴=热，不口渴=寒；痰黄=热，痰白=寒 |

### 3. 情志辨证
- **过喜→伤心**，**过怒→伤肝**，**过忧/悲→伤肺**
- **过思→伤脾**（味觉丧失，如嚼蜡）
- **过恐→伤肾**（老人痴呆、骨质疏松、听力下降）
- **相胜法则**：喜胜忧，忧胜怒，怒胜思，思胜恐，恐胜喜

### 4. 身体体征速查
| 体征 | 意义 |
|------|------|
| 脚冷 | 阳虚，病在进 |
| 脚热 | 阳回，病在退 |
| 一觉到天亮 | 胃气回复 |
| 半夜1-3点醒 | 肝病 |
| 半夜3-5点醒 | 肺病 |
| 早起大便 | 大肠气足，健康标志 |
| 手掌赤红 | 肝阳上亢 |

### 5. 疾病预后判断
**六大健康标准（北派中医铁律）**：
1. 一觉到天亮
2. 胃口正常，不暴饮暴食
3. 早起大便后才吃早餐
4. 一天5-7次小便，量大色淡黄
5. 四季头面冷、手脚温热
6. 男女早晨阳反应（心脏功能指标）

## 三、处方用药特点

### 1. 善用经方，药简力专
| 方剂 | 主治 | 组成 |
|------|------|------|
| 桂枝汤 | 太阳中风 | 桂枝、白芍、炙甘草、生姜、红枣 |
| 麻黄汤 | 太阳伤寒 | 麻黄、桂枝、杏仁、炙甘草 |
| 葛根汤 | 太阳温病/喉咙痛 | 葛根、麻黄、桂枝、白芍、炙甘草、生姜、红枣 |
| 小柴胡汤 | 少阳症/月经期感冒 | 柴胡、黄芩、人参、半夏、炙甘草、生姜、红枣 |
| 大柴胡汤 | 少阳+便秘 | 小柴胡汤去人参+大黄、枳实 |
| 大青龙汤 | 表寒里热（咳嗽黄痰） | 麻黄+桂枝+杏仁+石膏+生姜+红枣 |
| 小青龙汤 | 表寒里寒（咳嗽清痰） | 麻黄+桂枝+白芍+细辛+干姜+五味子+半夏+炙甘草 |
| 白虎汤 | 阳明经热 | 石膏、知母、炙甘草、粳米 |
| 大承气汤 | 阳明腑实（大肠堵） | 大黄+芒硝+厚朴+枳实 |
| 麻黄附子细辛汤 | 少阴症 | 麻黄、附子、细辛 |
| 四逆汤 | 阳虚急救 | 附子、干姜、炙甘草 |

### 2. 药物配伍心法
**麻黄汤配伍逻辑（开表发汗）**：
- 麻黄：开表发汗
- 桂枝：把肌肉打开，推麻黄进入皮肤
- 杏仁：补肺津液，防止脱水
- 炙甘草：缓和心脏悸动

> "麻黄从不等单味使用，一定配合桂枝、杏仁、炙甘草。单独用麻黄，心脏跳动加速，所以要加炙甘草缓和心脏。"

### 3. 剂型选择原则
| 剂型 | 特点 | 适用场景 |
|------|------|----------|
| 汤剂 | 药力迅捷，专为个人 | 急性病、病情复杂需灵活加减 |
| 丸剂 | 药缓力专 | 巩固疗效、病在深入需缓图 |
| 散剂 | 便于携带 | 慢性调理 |

### 4. 特殊治法
- **月经期感冒**：只用小柴胡汤（病毒直入子宫，桂枝麻黄发不到）
- **肝硬化/肝癌**：治肝先实脾，保持大小便通畅（汉唐-68护肝丸）
- **C型肝炎**：清病毒后出现感冒症状，代表病在往回走

### 5. 药物与食物性味
| 五味 | 入脏 | 代表食物/药物 |
|------|------|---------------|
| 酸 | 肝 | 柠檬、米醋（天然酸） |
| 苦 | 心 | 中药、蔬菜 |
| 甘 | 脾 | 蔗糖（天然糖） |
| 辛 | 肺 | 天然辣椒 |
| 咸 | 肾 | 海盐（天然咸） |

## 四、典型医案分析

### 医案一：日本乳癌转肝癌案（2005年4月13日）

**病史**：乳癌手术→化疗→转移肝癌、大肠癌

**四诊**：
- 腹积水、双脚踝积水
- 倦怠无力、无胃口
- 阴黄、右肋下方痛
- **脉虚数**：心阳欲竭
- **阴盛格阳**：外症热、里症寒实

**诊断**：厥阴症末期，心阳欲竭

**治疗**：补泻兼顾的最强经方

**结果**：病体不受纳，只剩一周生命

**教训**：
> "任何癌症如果西医只是开刀还好，一旦做化疗，几乎百无一生。希望读者记取教训，不要再做化疗了。"

---

### 医案二：56岁男性听力丧失案（只能听女声）

**主诉**：听力丧失五年，只能听到女人讲话

**辨证思路**：
- 女子属阴，男子属阳
- 只能听女声→阳虚→无法听男音
- 性功能差→肾阳不足
- 脚冷→阳虚确认

**诊断**：阳虚

**治疗**：针灸（耳门、听宫、听会等近取穴）+ 壮阳中药

**核心观点**：
> "阳行则阴不积。几乎所有的癌症都是阴积造成，而会如此皆因阳不足。所以我们绝对不可以阳虚。"

---

### 医案三：16岁少年 Chron's Disease + Lyme Disease

**病史**：
- 西医诊断：Chron's Disease、Lyme Disease
- 每天下利脓血数十次
- 服用抗生素每天3000mg，持续一年
- 体力下降，一直感冒，无法正常社交

**四诊**：少阴~厥阴之间的下利脓血症

**治疗**：经方汤剂5剂 + 生用赤石脂冲服

**结果**：
- 下利从数十次减少到一天四次
- 服药后大便即不见血
- 后续阴阳协调调理中

**教训**：
> "一颗抗生素在身体里待27年才代谢得掉。抗生素不但无效，还害人匪浅。"

## 五、临床启示

### 1. 感冒治不好，百病从生
> "感冒能够治好的话，就可以预防百病的发生了。"
- 太阳→少阳→阳明→太阴→少阴→厥阴
- 每一阶段用对方法，病就不会深入

### 2. 西药后遗症触目惊心
- 降胆固醇药→老人痴呆症
- 降压药→性功能丧失
- 类固醇→肺钙化、肝硬化、肾衰竭
- 抗生素→破坏肠道菌群、27年难代谢

### 3. 便秘是万病之源
> "几乎百分之百的精神病患皆有严重便秘情形，绝大多数皆是因为大便的沼气直冲入脑造成的。"
- 治精神病先通便
- 早起大便=大肠气足=健康保证

### 4. 治癌核心思路
1. **强脾脏**：治肝先实脾
2. **保大便**：保持通畅
3. **护小便**：颜色正常
4. **看黑便**：肝在排毒，非肠胃出血
5. **化疗必死**：已验证无数案例

### 5. 健康六大铁律（自我诊断）
| 序号 | 指标 | 不正常的意义 |
|------|------|-------------|
| 1 | 一觉到天亮 | 胃气/肝气有问题 |
| 2 | 胃口正常 | 脾脏正常 |
| 3 | 早起大便 | 大肠气足 |
| 4 | 小便5-7次/天 | 肾气正常 |
| 5 | 手脚温热 | 阳气充足 |
| 6 | 早晨阳反应 | 心脏功能好 |

> "越治手脚越冷，代表走错路了，准备死了。"
> "如果做了化疗，脚是热的，一觉睡到天亮，就可以继续做化疗——但目前还没有发生过。"

### 6. 医案记录原则
- 每味药的使用时机要清晰
- 每下一针的目的要明确
- 病人服后反应要详实记录
- 临床记录是最好的教材
