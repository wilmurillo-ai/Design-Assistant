# 针灸教程 · 倪海厦核心笔记

## 针灸理论基础

### 一、中医是物理医学
倪师强调：**中医是物理医学**，如同爱因斯坦站在外面看到里面。核心观念：
- **痛来自压力，而非发炎** —— 压力纾解、环境改变，病就好了
- **针灸不是杀细菌，而是改变环境**
- **站在外面就知道里面** —— 通过外部观察（时辰、脉象、舌苔等）诊断内部病变

### 二、经络系统架构
- **正经十二条**：六阳经（胆、胃、大肠、膀胱、三焦、小肠）+ 六阴经（肝、心、脾、肺、心包、肾）
- **奇经八条**：任脉、督脉、冲脉、带脉、阳蹻、阴蹻、阳维、阴维
- **核心原则**：手不过肘，足不过膝 —— 四肢肘膝以下穴位最常用

### 三、井荣俞原经合（五腧穴）
阴经属性：木 → 火 → 土 → 金 → 水（井木荣火俞土经金合水）
阳经属性：金 → 水 → 木 → 火 → 土（井金荣水俞木马经火合土）

| 穴位 | 原理 | 临床应用 |
|------|------|----------|
| **井穴** | 病在脏取井穴 | 井主心下满，指头/趾头末稍，动脉血管堵塞 |
| **荣穴** | 病变于色取荣穴 | 颜色变化相关病症 |
| **俞穴** | 病在时间者取俞穴 | 定时发作的疾病（根据地支歌） |
| **经穴** | 病变于音者取经穴 | 声音相关病变 |
| **合穴** | 病起于饮食取合穴 | 饮食相关疾病 |
| **原穴** | 五脏不平衡时用原穴 | 阳经独有，脏腑问题 |

### 四、五行与阴阳
- **五行**：木火土金水，相生相克
- **阴阳**：分清阴经阳经，诊治思路完全不同
- **干支歌**：诊断时间性疾病的利器

## 辨证论治体系

### 一、时间诊断法（地支歌）
倪师最重视的诊断工具：
> **肺寅大卯胃辰宫，脾巳心午小未中，申胱酉肾心包戌，亥焦子胆丑肝通**

**临床应用**：
- 肝病 → 1-3点（丑时）不能睡 → 治后能睡 = 肝病好转
- 肺癌 → 3-5点（寅时）不能睡
- 胆结石 → 23-1点（子时）发作
- **不需要X光**，问睡眠就知道病是否治好

### 二、阴阳辨证
- **脏病（阴）**：病在深部，取井穴
- **腑病（阳）**：病在浅部，取合穴
- **虚实分辨**：补泻手法的基础

### 三、八纲辨证
阴阳、表里、虚实、寒热 → 决定针灸手法与穴位选择

### 四、特殊诊断法
- **眼诊法**：癫、痫、狂的眼白辨证
- **触诊法**：压痛点找穴位（经络痛压就下针）
- **时辰法**：按时辰定点下针（子午流注精神）

## 倪海厦独特见解

### 一、"痛来自压力"的革命性观点
> **痛根本是来自压力，而不是来自发炎。**

这与西医完全不同，西医止痛用消炎药；倪师用针灸疏通压力。

### 二、脚热则无癌
> **当你全身充满气的时候，是没有癌症的，像个热水瓶一样。脚是热的就没有癌症。**

- 正常人站立，气往上走，人感觉不到体重
- 脚是全身最底层，却应该是温热的（气化状态）
- 要想办法把脚热起来

### 三、糖尿病西医制造论
> **西医分两种，胰岛素依赖型（第一型）跟第二型糖尿病，实际上依赖型根本不存在，是西医制造出来的。**

打疫苗会导致胰脏失能。

### 四、小儿鼻血不发高烧
> **小孩十四岁以下，流鼻血叫做正常，不要去治它。会流鼻血的小孩不发高烧！**

塞住鼻血反而会让高烧出来。

### 五、脊椎骨不能动
> **不要让人家动你的脊椎骨，因为一伤到就很难治，脊椎骨牵扯太广。**

### 六、近取穴与远取穴
- **近取**：病在肩膀，就在肩膀下针
- **远取**：病在肩膀，在远处循经取穴
- **对侧取穴**：治疗以对侧取穴最好

## 临床应用要点

### 一、针灸时机与原则
- **隔姜灸**：鲜姜片上艾灸
- **隔盐灸**：神阙（肚脐）用盐
- **隔蒜灸**：蒜片上灸
- **米粒灸**：小艾粒直接灸
- **急救法**：针灸急救三招（人中、十宣、放血）

### 二、补泻手法
1. **捻转针法**：顺时针补，逆时针泻
2. **呼吸配合**：呼气进针，吸气出针（补）；吸气进针，呼气出针（泻）
3. **提插补泻**：重插轻提（补），轻插重提（泻）

### 三、透针技术
- **合谷透后溪**：治疗中风手挛
- **肩髃透极泉**：治疗狐臭
- **条口透承山**：糖尿病

### 四、诊断为先
> **知道穴道的位置跟经络的走向，不见得一定要下针，但诊断用得到。**

### 五、火罐应用
- 放血用火罐（实证放血）
- 肩背痛、肌肉痛常用
- 脓疮、炎症也可使用

### 六、治症思路
1. **俞募治疗**：病在脏取俞穴+募穴
2. **会郄治疗**：腑会中脘+胃经郄穴梁丘 → 胃病
3. **对称治疗**：左边有病治右边
4. **导引针法**：循经取穴，引导气血

### 七、针灸疗程
- 急性病：每日针
- 慢性病：隔日针或每周2-3次
- 疗程中感冒：继续治，感冒标症兼治

### 八、禁针穴位
- 五里穴（有大动脉）
- 巨骨穴（下面是肺）
- 重要器官附近穴位：浅针为宜

### 九、度骨找穴法
从已知椎位（第七椎、第十四椎）往回找，不用从第一椎开始数，避免出错。
