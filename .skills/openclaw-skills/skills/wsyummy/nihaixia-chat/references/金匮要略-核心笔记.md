# 金匮要略 · 倪海厦核心笔记

> 倪海厦注《金匮要略》—— 勤求古训，博采众方

---

## 一、杂病分类框架

倪海厦将《金匮要略》杂病按**脏腑经络体系**分为十六篇，核心逻辑是**按病位（五脏六腑）、病性（风寒湿热）、病形（虚实积聚）**三维分类：

| 篇序 | 病类 | 病位/病性 | 代表病证 |
|------|------|-----------|---------|
| 第一 | 藏府经络先后病 | 整体/先后 | 病因判断、表里先后治则 |
| 第二 | 痉湿暍病 | 太阳经/津液 | 刚痉、柔痉、湿病、暑病 |
| 第三 | 百合狐惑阴阳毒 | 肺/心肝 | 情志病、热毒证 |
| 第四 | 疟病 | 少阳/膜原 | 疟母、温疟、寒疟 |
| 第五 | 中风历节 | 肝/风寒湿 | 中风、痛风历节 |
| 第六 | 血痹虚劳 | 脾/气血 | 气血两虚、劳伤 |
| 第七 | 肺痿肺痈咳嗽上气 | 肺 | 咳嗽、哮喘、肺痈 |
| 第八 | 奔豚气 | 肝/冲气 | 气上冲胸 |
| 第九 | 胸痹心痛短气 | 心/寒痰 | 冠心病、胸闷 |
| 第十 | 腹满寒疝宿食 | 胃肠/寒食 | 腹满、寒疝、食积 |
| 第十一 | 五藏风寒积聚 | 五脏 | 积聚、脏腑病 |
| 第十二 | 痰饮咳嗽 | 三焦/水湿 | 痰饮、悬饮、溢饮、支饮 |
| 第十三 | 消渴小便不利淋 | 肾/津液 | 消渴、淋证 |
| 第十四 | 水气 | 脾肾/水停 | 风水、皮水、正水、石水 |
| 第十五 | 黄疸 | 肝胆/湿热 | 阴黄、阳黄 |
| 第十六 | 惊悸吐衄下血瘀血 | 心/血分 | 出血、瘀血 |

**倪海厦特别强调**：五脏风寒积聚一篇"断简残篇"最甚，但最重要，临床价值极高。

## 二、辨证论治要点

### 1. 表里先后治则
- **表里并病时，先治里，再解表**。如霍乱兼太阳证，先止吐泻，再解表。
- **脉诊定位**：寸脉浮→病在表（上焦）；尺脉浮→病在里（下焦）。寸脉浮→感冒（麻黄汤证/桂枝汤证/葛根汤证）。

### 2. 六经辨证融入杂病
- 太阳病传变：津液丧失过多→葛根汤证；表热传少阳→小柴胡证；入阳明→里实热；入膀胱→小便不利。
- **湿病治则**：湿在关节→从汗解（麻黄加术汤）；"离毛孔近者从汗解"。

### 3. 津液观贯穿全书
- 倪海厦反复强调：**津液是抵抗病邪的根本**。运动/出汗后得感冒→葛根汤证（津液丧失在前）。
- 津液枯竭→温病→白虎加人参汤。

### 4. 阴阳辨证
- **厥阳**："有阳无阴"，阳独行→阴虚发热。
- **阴阳毒**：阳毒→面赤斑斓如锦纹；阴毒→面目青黑。
- **判断预后**：脉微涩→四逆汤证，心阳不振，寒水不化。

### 5. 汗法微妙
- 麻黄加术汤→微汗（黏汗）→治阴黄（湿家黑黄脸）。
- 小续命汤→得汗则解，坐姿助汗。
- **不可大汗伤津**：大汗→津液丧失→变证。

## 三、倪海厦独特见解

### 1. 阳气与经络优先
- **"上焦如雾"**：心肺是阳气根本，水气蒸发如雾，升头顶再宣泄。
- 肺中风→热上加热→横膈膜上去下不来→喘。
- 中风核心：风寒湿三气夹杂，麻黄桂枝并用才能祛风散寒。

### 2. 续命汤系列（小续命汤）的临床价值
- 小续命汤：**治中风第一方**，手足不仁兼口不能言、神志不清、四肢拘急。
- 特点：麻黄桂枝同用（南派视为蛇蝎，倪海厦视为救逆圣药）。
- **得汗则解**：采坐姿，汗出即愈。
- 千金三黄汤→手足拘急、无力；术附汤→风虚头眩、口淡无味。

### 3. 湿病专著见解（最详尽）
- 湿家但头汗出→湿阻三焦→丹田有热、胸上有寒。
- 麻黄加术汤→**阴黄**（黎黑黄），不是阳黄（鲜黄）。
- 湿家不可下→下之早则哕、胸满、小便不利。

### 4. 痰饮病水液代谢观
- 痰饮核心：脾肾阳虚→水液不化→停留三焦。
- **离膀胱近→从小便解；离毛孔近→从汗解**。

### 5. 易筋经养生（倪海厦亲授）
- **文式易筋经**：寸劲发力的中国功夫，不靠外力（举重），内壮筋骨。
- **效果**：60岁头发仍黑、精神好、仅睡5小时。
- **动作要点**：背挺直、眼瞪怒、舌顶上颚、下盘绷紧、寸劲下按。

### 6. 预防观念
- 张仲景：**"服食节其冷热，苦酸辛甘，不遣形体有衰，病则无由入其腠理"**。
- 腠理→津液循环的门户→养生核心在保津液。

## 五、临床应用要点

### 1. 中风急救三原则
> **小续命汤为主，千金三黄汤为辅，术附汤收尾**
- 小续命汤：口不能言+神志不清+四肢拘急→**主方**。
- 千金三黄汤：手足不仁+心烦+小便秘涩。
- 术附汤：风虚头眩+口淡无味+里虚。
- **服药方式**：坐姿，得汗则解；无汗继续服。

### 2. 湿病三禁
- **禁下**（下之早→哕、胸满、小便不利）。
- **禁火**（火攻→发黄）。
- **禁汗过多**（汗太过→痉）。
- **正确治法**：微汗（麻黄加术汤）。

### 3. 痰饮治法层次
- 离表近→汗解（麻黄加术汤）。
- 离里近→吐解（瓜蒂散）。
- 离下焦近→下解（十枣汤）。
- **核心**：给病邪找出路，顺势利导。

### 4. 霍乱辨证
- 霍乱自吐下→先治里。
- 兼太阳表证→先止吐利，再解表。
- **脉微涩**→四逆汤证（心阳不振→寒水不化）。

### 5. 黄疸分阴阳
- **阳黄**（鲜黄）→茵陈蒿汤（湿热在胃）。
- **阴黄**（黎黑）→麻黄加术汤（湿在肌肉）。
- 不可混淆！阴黄误用茵陈蒿汤则伤阳。

### 6. 养生防病原则
- **"腠者，津液渗泄之所"**→腠理是津液流通的门户。
- 节气太过不及皆致病。
- 运动以微微汗出为度（不是大汗淋漓）。
- **津液比血更重要**：津液枯竭则病入膏肓。

### 7. 小续命汤使用细节（倪海厦临床经验）
- 麻黄+桂枝同用：南派畏惧，倪海厦力倡。
- 石膏五钱（平常）→壮热时加重。
- 当归+川芎等量（各五钱至一两）→补血活血并行。
- 得汗停药，坐姿助发汗。
