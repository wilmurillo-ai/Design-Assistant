# Batch Resume Screener

批量简历初筛技能，支持多候选人、多岗位的自动化简历评估与筛选。

## 功能特点

- **批量处理**: 支持批量处理多份简历，评估多个岗位
- **多格式支持**: 支持 PDF、ZIP 压缩包等多种简历格式
- **七维度评分**: 学术背景、职业稳定性、技术栈能力、项目匹配、问题解决能力、学习能力、加分项
- **动态权重**: 根据岗位级别（初级/中级/高级）自动调整评分权重
- **硬性要求检查**: 学历、工作年限等硬性要求一票否决机制
- **置信度评分**: 每份评估结果包含置信度，标记需要人工复核的简历
- **多格式输出**: Markdown 报告、Excel 数据、对比表格、亮点摘要

## 使用方法

### 触发方式

当用户请求以下任务时，自动触发此技能：

- 批量筛选简历
- 评估多位候选人
- 处理简历 ZIP 包
- 针对多个岗位进行简历筛选

### 输入要求

1. **岗位要求文档**: 每个岗位一份要求文档
2. **简历文件**: 多份简历（PDF/DOC/DOCX）或包含简历的 ZIP 压缩包

### 示例用法

```
请帮我筛选这批简历，岗位要求如下：
1. Java高级工程师：5年以上Java开发经验，本科及以上学历
2. 后端开发工程师：3年以上开发经验

简历文件：resumes.zip
```

## 评分体系

### 动态权重分配

| 评分维度 | 初级 | 中级 | 高级 |
|---------|------|------|------|
| 学术背景 | 15% | 10% | 5% |
| 职业稳定性 | 5% | 10% | 10% |
| 技术栈能力 | 15% | 15% | 15% |
| 项目匹配 | 20% | 25% | 25% |
| 问题解决能力 | 10% | 15% | 20% |
| 学习能力 | 20% | 15% | 10% |
| 加分项 | 15% | 10% | 15% |

### 硬性要求（一票否决）

- 学历门槛：是否满足岗位明确要求的最低学历
- 工作年限：是否满足岗位明确要求的最低年限
- 其他硬性要求：岗位明确标注"必须/不可或缺"的要求

### 总分评级参考

| 分数区间 | 评级 | 说明 |
|---------|------|------|
| 90-100 | 优秀候选人 | 远超岗位要求，核心维度表现突出 |
| 75-89 | 推荐面试 | 完全满足岗位要求，核心维度匹配度高 |
| 60-74 | 可备选 | 刚满足基本要求，有明显短板 |
| <60 | 不推荐 | 满足硬性门槛，但核心能力差距较大 |

## 输出格式

### 1. Markdown 综合报告

包含统计概览、候选人排名、通过/未通过汇总、低置信度候选人标记等。

### 2. Excel 就绪数据

CSV 格式，可直接导入 Excel 进行筛选和排序。

### 3. 候选人对比表

横向对比多位候选人的各维度得分。

### 4. 亮点摘要

为面试官提供的候选人核心优势和面试建议。

## 依赖

- Python 3.x
- pdfplumber（PDF 文本提取）

安装依赖：
```bash
pip install pdfplumber
```

## 注意事项

- 所有评估仅基于简历文本内容，不做无根据假设
- 低置信度评估结果建议人工复核
- 评分严格遵循给定权重和评分规则
- 每位候选人推荐得分最高的岗位
