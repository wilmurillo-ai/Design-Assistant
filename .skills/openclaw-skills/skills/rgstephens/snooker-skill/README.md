# Snooker Skill

This skill wraps the excellent [snooker.org API](https://api.snooker.org) site by [Hermund Årdalen](http://hermund.ardalen.com/).

To use the skill and underlying API you'll need to request an API key.

After installing the skill, set your API key:

```bash
{baseDir}/scripts/snooker.py setup --api-key YOUR_KEY
```
