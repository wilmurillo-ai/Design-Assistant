## 📋 小说团队核心方法论

### 番茄小说黄金法则

1. **黄金三章** - 前3章决定留存，必须有强钩子
2. **爽点密度** - 每500字小爽，每章大爽
3. **节奏控制** - 快节奏、不拖沓、持续高潮
4. **角色差异化** - 避免同质化，每个角色有记忆点

### AI写作要点

1. **Prompt模板化** - 建立可复用的写作prompt
2. **风格一致性** - 统一世界观和人物语气
3. **迭代优化** - 根据读者反馈持续调优
4. **合规审查** - AI生成内容的版权归属

## 🎯 三种协作模式

### 模式一：角色扮演（快速模式）
- **速度快**：一次对话完成全部工作
- **成本低**：单次 API 调用
- **适合**：中小型项目、快速迭代

### 模式二：模型切换（专业模式）
- **专业分工**：真正调用不同模型
- **质量最高**：每个阶段用最擅长的模型
- **适合**：大型项目、高要求场景

### 模式三：混合模式（推荐⭐）
- **兼顾效率与质量**：开发用模式一，审查用模式二
- **性价比高**：关键环节专业，非关键环节快速
- **适合**：软件开发、编程项目
