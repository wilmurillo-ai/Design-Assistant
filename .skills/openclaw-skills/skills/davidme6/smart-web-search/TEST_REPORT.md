## ⚠️ 发现问题

### 1. 网络连接问题
- **现象**: DuckDuckGo 连接超时
- **原因**: 国际网络不稳定
- **解决**: 
  - 使用备用引擎（Qwant、Startpage）
  - 用中文搜索替代（搜索"亚马逊爆品"）

### 2. 结果提取问题
- **现象**: 搜索结果包含大量 JavaScript
- **原因**: 动态加载内容
- **解决**: 
  - 增加 `extractMode="text"` 参数
  - 使用 `web_fetch` 二次抓取详情页

### 3. 广告过滤
- **现象**: 搜索结果包含推广内容
- **解决**: 
  - 识别"Sponsored"标记
  - 优先选择有机结果
