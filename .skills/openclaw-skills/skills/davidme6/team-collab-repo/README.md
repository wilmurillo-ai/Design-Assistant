## 简介

Team Collab 是一个多模型团队协作系统，把不同的 AI 模型组织成一个专业团队，像公司一样协作完成项目。

## 团队角色

| 角色 | 推荐模型 | 核心能力 | 职责 |
|------|----------|----------|------|
| **产品经理** | Qwen-Plus | 需求分析、逻辑规划 | 需求文档、功能设计 |
| **程序员** | GLM-5 | 代码生成、Bug修复 | 核心开发、功能实现 |
| **美术设计师** | Qwen3.5-Plus | 视觉设计、创意表达 | UI设计、配色方案 |
| **测试工程师** | DeepSeek-Coder | Bug发现、测试用例 | 功能测试、问题定位 |
| **代码审查员** | Qwen-Max | 代码审查、性能优化 | 代码质量、安全检查 |

## 工作流程

```
1. 产品经理 → 需求分析 → 需求文档
2. 程序员   → 开发实现 → 代码
3. 设计师   → UI设计   → 界面
4. 测试     → 测试验收 → 测试报告
5. 审查员   → 代码审查 → 审查报告
6. 交付
```
