## 🎯 技能简介

**Self-Learning Skill v2.0.0** 是一个完整的自我学习与迭代系统，专为 AI 助手设计。

**核心使命**：持续学习、识别不足、主动增强、迭代进化、**永不重复犯错**

**v2.0.0 新增** ⭐：

- ✅ 预防措施检查清单
- ✅ 定期复查机制

## 🌟 核心特性

### 1. 主动学习 🧠
- 不等待指令，主动发现知识盲区
- 多渠道学习（GitHub/文档/社区/优秀作品）
- 学以致用，立即应用

### 2. 持续迭代 🔄
- 任务后复盘（每次必做）
- 每日总结（22:00 执行）
- 每周评估（能力对比）
- 每月趋势分析

### 3. 错误学习 ⭐ NEW
- **宗旨**：错误可以犯，但永不重复
- 错误分类系统（4 大类 20 小类）
- 错误档案和数据库
- 5 Why 分析法找根本原因
- 预防措施检查清单
- 定期复查（3 天/1 周/1 月）

### 4. 量化追踪 📊
- 能力评估矩阵（8 维度）
- 学习时长统计
- 代码产出追踪
- 错误复发率监控
- 成长曲线可视化

### 5. 用户主导 👤
- 随时叫停学习
- 纠正方向
- 调整目标
- 查看进度

## 🧠 错误学习系统

### 错误分类

#### 技术错误 (TECH)
- TECH-001: API 使用错误
- TECH-002: 代码逻辑错误
- TECH-003: 性能问题
- TECH-004: 兼容性问题
- TECH-005: 安全问题

#### 沟通错误 (COMM)
- COMM-001: 理解偏差
- COMM-002: 信息遗漏
- COMM-003: 表达不清
- COMM-004: 反馈延迟

#### 流程错误 (PROC)
- PROC-001: 步骤遗漏
- PROC-002: 顺序错误
- PROC-003: 工具使用错误
- PROC-004: 时间管理不当

#### 知识错误 (KNOW)
- KNOW-001: 知识盲区
- KNOW-002: 概念误解
- KNOW-003: 信息过时
- KNOW-004: 技能不足

### 错误档案模板

```markdown
## 错误档案 [ERROR-编号]

**发生时间**：2026-03-15
**错误类型**：[TECH/COMM/PROC/KNOW]-[编号]
**严重程度**：⭐⭐⭐⭐⭐ (1-5 星)

### 错误描述
[详细描述]

### 根本原因
[5 Why 分析法]

### 解决方案
[立即措施]

### 预防措施 ⭐
[如何避免再犯]
1. [行动 1]
2. [行动 2]
3. [检查清单]

### 复查日期
- 首次复查：3 天后
- 二次复查：1 周后
- 长期复查：1 月后
```
