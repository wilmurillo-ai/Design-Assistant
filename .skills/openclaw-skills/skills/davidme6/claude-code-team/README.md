## 📋 任务分配模板

### 标准任务分配

```javascript
function assignTask(teamName, taskDescription) {
    const teamConfig = getTeamConfig(teamName);
    
    // 自动启动所有成员
    const agents = teamConfig.members.map(member => {
        return sessions_spawn({
            label: `${teamName}-${member.role}`,
            model: member.model,
            task: `${taskDescription}\n\n你的职责：${member.responsibility}`
        });
    });
    
    // 等待所有结果
    return Promise.all(agents);
}
```

## ⚡ 性能优化

### Gateway 超时解决

| 问题 | 解决方案 |
|------|----------|
| 连接泄漏 | 任务完成后自动关闭连接 |
| 超时设置 | 延长到 120 秒 |
| 并发限制 | 动态调整（默认 10） |
| 内存压力 | 定期清理旧连接 |

### 优化后的 sessions_spawn

```javascript
sessions_spawn({
    mode: 'run',          // 一次性任务
    runTimeoutSeconds: 120, // 超时 120 秒
    cleanup: 'delete',    // 完成后自动清理
    label: '角色名'        // 便于追踪
});
```
