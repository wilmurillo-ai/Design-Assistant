# 快递承运商识别规则

## 单号格式速查

### 顺丰速运
正则：^(SF|sf|13[0-9]{9})[0-9]{9}$
长度：12-15位
前缀：SF开头或13开头手机号
示例：SF1234567890

### 中通快递
正则：^(ZT|zt|78|38|01|02)[0-9]{11}$
长度：12-13位
前缀：78/38/01/02开头
示例：781234567890

### 圆通速递
正则：^(YT|yt|YT|88|99|66)[0-9]{10,12}$
长度：12-13位
前缀：88/99/66/YT开头
示例：881234567890

### 韵达速递
正则：^(YD|yd|12|10|39|30)[0-9]{11}$
长度：13位
前缀：12/10/39/30开头
示例：1281234567890

### 申通快递
正则：^(ST|st|88|99|66|22)[0-9]{11}$
长度：12-13位
前缀：88/99/66/ST开头
示例：881234567890

### 极兔速递
正则：^(JT|jt|JT|TG|tg)[0-9]{12}$
长度：12-13位
前缀：JT/TG开头
示例：JT12345678901

### 京东物流
正则：^(JD|jd|JD|16|18|36)[0-9]{12,15}$
长度：13-15位
前缀：JD/16/18/36开头
示例：JD12345678901

### 邮政快递
正则：^(YZ|yz|PG|pg|99)[0-9]{13,15}$
长度：13-15位
前缀：YZ/PG/99开头
示例：9812345678901

### 德邦物流
正则：^(DP|dp|TC|tc)[0-9]{9,12}$
长度：9-12位
前缀：DP/TC开头
示例：DP123456789

### 安能物流
正则：^(ANE|ane|56|52|68)[0-9]{9,12}$
长度：10-12位
前缀：ANE/56/52/68开头
示例：561234567890

### 中铁快运
正则：^(KT|kt)[0-9]{7,10}$
长度：7-10位
前缀：KT开头
示例：KT1234567

### 跨越速运
正则：^(KY|ky)[0-9]{12,13}$
长度：12-13位
前缀：KY开头
示例：KY12345678901

### 联邦快递
正则：^[0-9]{12,22}$
长度：12-22位
前缀：纯数字为主
示例：123456789012

## 智能识别算法

### 步骤一：前缀匹配

如果单号以SF或sf开头，判断为顺丰速运

如果单号以JT、jt、TG、tg开头，判断为极兔速递

如果单号以JD或jd开头，判断为京东物流

如果单号以YT或yt开头，判断为圆通速递

如果单号以ZT或zt开头，判断为中通快递

如果单号以ST或st开头，判断为申通快递

如果单号以YD或yd开头，判断为韵达速递

如果单号以DP或dp开头，判断为德邦物流

如果单号以ANE或ane开头，判断为安能物流

如果单号以KT或kt开头，判断为中铁快运

如果单号以KY或ky开头，判断为跨越速运

如果单号以YZ、yz、PG、pg开头，判断为邮政快递

### 步骤二：长度和特征匹配

当长度为12位时：

如果以78或38或01或02开头，判断为中通快递

如果以88或99或66开头，判断为圆通速递或申通快递

当长度为13位时：

如果以12或10或39或30开头，判断为韵达速递

如果以96开头，判断为邮政快递

当长度为15位时：

如果以96或98开头，判断为邮政快递

### 步骤三：兜底判断

如果前面都不匹配，返回"未知快递公司"

## 快速识别伪代码

def识别快递公司(单号):
    号码 = 单号.strip().upper()
    
    # 第一步：前缀精确匹配
    if 号码.startswith('SF'):
        return '顺丰速运'
    if 号码.startswith('JT') or 号码.startswith('TG'):
        return '极兔速递'
    if 号码.startswith('JD'):
        return '京东物流'
    if 号码.startswith('YT'):
        return '圆通速递'
    if 号码.startswith('ZT'):
        return '中通快递'
    if 号码.startswith('ST'):
        return '申通快递'
    if 号码.startswith('YD'):
        return '韵达速递'
    if 号码.startswith('DP'):
        return '德邦物流'
    if 号码.startswith('ANE'):
        return '安能物流'
    if 号码.startswith('KY'):
        return '跨越速运'
    if 号码.startswith('YZ') or 号码.startswith('PG'):
        return '邮政快递'
    
    # 第二步：长度和特征匹配
    长度 = len(号码)
    
    if 长度 == 12:
        if 号码.startswith(('78', '38', '01', '02')):
            return '中通快递'
        if 号码.startswith(('88', '99', '66'):
            return '圆通速递'
    
    if 长度 == 13:
        if 号码.startswith(('12', '10', '39', '30')):
            return '韵达速递'
        if 号码.startswith('96'):
            return '邮政快递'
    
    # 第三步：兜底
    return '未知快递公司'

## 注意事项

单号格式会变：各公司单号格式更新频繁，需要定期更新正则规则

混合模式：有些单号在快递100可查多家，不需要精确识别

特殊单号：跨境快递、DHL、UPS等不在本表范围

手机号混淆：138开头的可能是顺丰手机号会员卡
