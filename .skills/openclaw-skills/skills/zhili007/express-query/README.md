# 快递物流查询 Skill

## 简介

一个能够自动识别快递公司、生成查询方案、解析物流轨迹的Skill。

## 功能

- 单号识别：自动识别12+家主流快递公司，输出置信度
- 查询方案：根据不同快递公司生成最优查询路径
- 轨迹解析：支持解析物流时间线
- 异常判断：自动判断物流异常（停滞、派送超时、签收未收到等）
- 分级查询：一级→二级→三级→四级查询策略

## 支持的快递公司

顺丰速运、极兔速递、京东物流、圆通速递、中通快递、申通快递、韵达速递、德邦物流、安能物流、跨越速运、邮政快递、中铁快运

## 使用方法

### 1. 识别快递公司

```bash
cd scripts
python3 identify.py <单号>
```

示例：
```bash
python3 identify.py SF1234567890
# 输出：顺丰速运 (95%)
```

### 2. 查询快递

```bash
python3 query.py <单号>
```

示例：
```bash
python3 query.py SF1234567890
```
