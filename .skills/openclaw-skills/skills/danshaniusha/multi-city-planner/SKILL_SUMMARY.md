# ✅ Multi-City Planner Skill - 格式已固化

## 📋 输出格式规范（严格执行）

### HTML 报告必须包含以下 9 个模块：

1. **头部信息** - 标题、副标题、国旗、日期
2. **方案价格对比表** - 7 种方案，含排名/路线/价格/差价
3. **交通方式对比卡片** - 飞机/火车/大巴 3 种
4. **推荐方案详情** - 至少 2 个方案，含航班表 + 优缺点
5. **详细行程建议** - 按天列出，含景点推荐
6. **预算估算表** - 7 项费用 + 总计
7. **完整注意事项** - 10 大类（必须全部包含）
8. **推荐总结** - 推荐方案 + 价格 + 亮点
9. **页脚** - 数据来源 + 生成时间

### 10 大类注意事项（必须全部包含）

| 类别 | 内容 |
|------|------|
| 📋 出行前准备 | 签证、护照、保险、机票、酒店、货币 |
| 🎒 行李准备 | 证件、电器、衣物、药品、其他 |
| ✈️ 交通注意事项 | 廉航行李、机场交通、火车订票、市内交通 |
| 🏨 住宿注意事项 | 入住时间、城市税、安全、民宿 |
| 🍽️ 餐饮注意事项 | 用餐时间、小费、饮用水、美食 |
| 🎫 景点注意事项 | 提前预订、免费日、城市通票、着装 |
| 🔒 安全注意事项 | 防盗、地铁、景点骗局、紧急电话、大使馆 |
| 📱 通讯与网络 | 漫游、WiFi、APP 推荐 |
| 💡 其他实用提示 | 时差、电压、退税、周日营业、语言 |
| ⚠️ 特别提示 | 针对目的地的特殊注意事项 |
