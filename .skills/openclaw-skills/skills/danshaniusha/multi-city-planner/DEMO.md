# 🗺️ Multi-City Planner 使用演示

## 场景：北京出发，游玩大阪和东京

### 标准输出格式

```bash
multi-city-planner --origin "北京" --cities "大阪，东京" --dep-date 2026-04-15 --return-date 2026-04-25
```

**输出**：
```

日期：2026-04-15 ~ 2026-04-25 (10 天)

📊 行程类型价格对比（从低到高）

| 排名 | 行程类型 | 飞行路线 | 总价格 | 差价 |
|------|----------|----------|--------|------|
| 1 | 多程联订 | 北京→大阪 → 大阪→东京 → 东京→北京 | ¥3,041 | - |
| 2 | 缺口程 + 单程 | 北京→大阪 + 东京→北京 + 大阪→东京 | ¥3,041 | +¥0 |
| 3 | 三个单程 | 北京→大阪 → 大阪→东京 → 东京→北京 | ¥3,041 | +¥0 |

### 1. 多程联订 - ¥3,041

**航班详情:**

| 航段 | 日期 | 时间 | 航班 | 价格 |
```

---

## 输出格式说明

### 1. 行程类型价格对比表

| 列名 | 说明 |
|------|------|
| 排名 | 按价格从低到高排序 |
| 行程类型 | 多程联订、缺口程、单程组合等 |
| **飞行路线** | **清晰展示每段飞行路径** |
| 总价格 | 该方案总费用 |
| 差价 | 相比最便宜方案的差价 |

### 2. 各类型详细航班对比

对每种行程类型，展示：
- **路线**: 简洁的行程概述
- **航班详情表**: 每段的日期、时间、航班号、价格
- **优缺点**: 该类型的优势和劣势

### 3. 推荐总结

- 🏆 推荐的行程类型
- 💰 推荐方案价格
- 💵 相比最贵方案的节省金额
