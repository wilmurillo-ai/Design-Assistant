# OSNK, OSNP, OSN & IOI Syllabus - Complete Reference

## OSN-K (Kabupaten/Kota) 2026

**Format:**
- 30-50 soal, 2.5 jam
- 3 Bagian: A, B, C

### Bagian A: Abstraksi Berpikir Komputasional
- Soal cerita bergambar (mirip Bebras)
- Computational thinking

### Bagian B: Pemecahan Masalah Komputasional
- Studi kasus pemrograman kompetitif
- Setiap studi kasus = 3 soal pemahaman

### Bagian C: Pemahaman Algoritma dalam Bahasa C++
- Analisis kode C++

## Materi OSN 2026 (Mengacu pada IOI Syllabus)

### 1. Dasar-dasar Pemrograman
- Sintaks dan semantik dasar
- Variabel, tipe data, ekspresi, assignment
- Masukan dan keluaran dasar
- Percabangan dan perulangan
- Fungsi dan parameter

### 2. Operasi Logika dan Bitwise
- Operator logika (konjungsi, disjungsi, implikasi, biimplikasi, disjungsi eksklusif)
- Tabel kebenaran
- Modus Ponens dan modus Tollens

### 3. Aritmetika
- Bilangan bulat, operasi, perbandingan
- Sifat bilangan bulat (tanda, paritas, keterbagian)
- Operasi modular
- Perpangkatan modular
- Bilangan prima
- Bilangan pecahan, persentase
- Teori bilangan dan himpunan

### 4. Aturan Berhitung
- Aturan penjumlahan dan perkalian
- Barisan aritmetika dan geometri
- Bilangan Fibonacci
- Permutasi dan kombinasi
- Probabilitas
- Pigeonhole principle
- Prinsip inklusi dan eksklusi
- Segitiga Pascal, teorema binomial

### 5. Rekursi
- Konsep rekursi
- Fungsi matematis rekursi
- Prosedur rekursi sederhana
- Divide-and-conquer
- Backtracking

### 6. Pencarian dan Pengurutan
- Linear search, Binary search
- Bubble sort, insertion sort
- Quick sort, merge sort, heap sort

### 7. Strategi Pemecahan Masalah
- Brute-force / Complete search
- Greedy
- Divide-and-conquer
- Dynamic programming

### 8. Struktur Data
- Tipe data primitif
- Array (multidimensi)
- String dan operasinya
- Stack dan queue
- Binary heap
- Disjoint set
- Point Update, Range Query (Fenwick tree, Segment tree)

### 9. Graf dan Tree
- Tree dasar (rooted tree)
- Graf berarah dan tak berarah
- Graf berbobot dan tak berbobot
- Representasi graf
- BFS, DFS, keterhubungan
- Shortest path (Dijkstra, Bellman-Ford, Floyd-Warshall)
- Minimum spanning tree (Prim, Kruskal)
- Lowest Common Ancestor (LCA)

### 10. Geometri Dasar
- Garis, segmen garis, sudut
- Segitiga, persegi, persegi panjang, lingkaran
- Koordinat Kartesius 2D
- Jarak Euclidean
- Teorema Pythagoras
- Definisi Convex hull

## IOI Syllabus Categories

### Included, unlimited:
- Integer
- Basic arithmetic operations
- Basic data types
- Basic programming language constructs

### Included, to be defined:
- Directed graph
- Queue
- Sorting algorithms
- Dynamic programming

### Topics:
- **AL1:** Basic Logic
- **AL2:** Number representation
- **AL3:** Propositional logic, Boolean algebra
- **AL4:** Truth tables
- **AL5:** Implication
- **AL6:** Logic engineering

- **NU1:** Basic number theory
- **NU2:** Divisibility, remainder
- **NU3:** Prime numbers
- **NU4:** Floor/ceiling functions

- **DS1:** Variables, types, expressions
- **DS2:** Functions, parameter passing
- **DS3:** Basic data structures
- **DS4:** Counting (sum rule, product rule, permutations, combinations)
- **DS5:** Graphs and trees
- **DS6:** Discrete probability

- **AR1:** Complexity analysis
- **AR2:** Searching (linear, binary)
- **AR3:** Sorting (comparison-based)
- **AR4:** Algorithm design techniques
