## 📦 发布流程

### 1. 准备阶段

**检查清单**：
- [ ] SKILL.md 已创建
- [ ] scripts/ 已创建并测试
- [ ] references/ 已创建
- [ ] assets/ 已创建（可选）
- [ ] 所有脚本已测试

### 2. 打包 SKILL

```bash
cd /home/admin/openclaw/workspace/skills/weight-management
python /opt/jvs-claw/base/lib/node_modules/openclaw/skills/skill-creator/scripts/package_skill.py .
```

**输出**：`weight-management.skill`

### 3. 发布到 ClawHub

**步骤**：
1. 访问 https://clawhub.ai/
2. 登录账号
3. 点击"发布 SKILL"
4. 上传 `weight-management.skill` 文件
5. 填写 SKILL 信息：
   - 名称：Weight Management
   - 描述：减肥项目管理与数据分析
   - 分类：Health & Fitness
   - 标签：weight-loss, health, fitness, diet
6. 提交审核

### 4. 审核与发布

**审核时间**：1-3 个工作日

**审核标准**：
- SKILL.md 格式正确
- 描述清晰
- 无安全问题
- 功能实用
