# PaperPass 查重报告降重 Skill v1

## 概述

本 skill 针对 PaperPass 查重报告中的高相似度段落，提供系统化的降重改写服务。

**输入**：PaperPass 查重报告（PDF 文件或报告文本）  
**输出**：改写后的低相似度段落 + 降重策略说明

---

## PaperPass 查重报告格式解析

PaperPass 报告具有以下特征，识别这些特征是降重的前提：

1. **段落相似度标注**：每段文字末尾或中间标注 `查重 XX%`，表示该段与已有文献的相似度
2. **颜色标记**：`查重 0%-40%` 通常为低风险（绿色），`41%-60%` 中等风险（黄色），`61%以上` 高风险（橙色/红色）
3. **相似文献来源**：报告末尾列出匹配来源，包括学位库、期刊库、会议库等
4. **连续高相似句子**：连续多句高相似会被整体标记，需要整体改写而非逐句
5. **高相似度阈值**：通常 40% 以上需要重点关注，60% 以上必须改写，80% 以上需大幅重构

## 降重核心策略（学术论文适用）

### 策略一：同义改写（适用于 40%-60% 相似段落）

- 将专业术语替换为等效表达（如"心肌梗死"→"心脏组织坏死"、"缺血"→"血流供应不足"）
- 形容词和副词替换（"显著"→"明显"/"较为"/"大幅度"）
- 动词替换（"表明"→"提示"/"证实"/"显示"）
- 保持术语准确性，避免改变专业含义

### 策略二：句式重构（适用于 50%-70% 相似段落）

- 主动句改被动句，被动句改主动句
  - 原文："MSCs 可以分化为心肌细胞" → 改写："在特定微环境诱导下，骨髓间充质干细胞具备向心肌谱系细胞定向转变的潜能"
- 拆分长句为短句，或合并短句为长句
- 调整句子顺序（先说结果后说原因）
- 将定语从句改为分词结构或独立句子

### 策略三：语义扩展（适用于 60%-80% 相似段落）

- 在保持核心信息的前提下，补充背景、原因、结果等扩展内容
- 加入具体数据或研究细节（原创性补充）
- 加入个人论述/评述性语言（"值得注意的是…"）
- 将陈述句改为更具批判性或分析性的表达

### 策略四：结构重组（适用于 80%+ 相似段落）

- 寻找该段落的替代文献来源，从不同角度重新描述同一现象
- 将一个高相似段落拆分为 2-3 个低相似段落
- 用图表替代文字描述（条件允许时）
- 改变论述逻辑顺序（先对比再总结 vs 先总结再展开）
- 大幅改写，保留核心概念但用全新句式和表达方式

## 操作流程

### 第一步：解析报告，识别高相似段落

从用户提供的查重报告中，提取所有 `查重 XX%` 段落，按照相似度从高到低排序：

```
极高风险（80%+）：必须全面重构
高风险（60%-80%）：大幅改写 + 补充原创内容
中等风险（40%-60%）：同义改写 + 句式调整
低风险（40%以下）：可接受，仅必要时微调
```

### 第二步：逐段改写

对每个高相似段落，按以下优先级执行改写：

1. 核心概念/术语保持不变（保证学术准确性）
2. 句式和表达方式全部替换
3. 如可能，加入补充说明或限定语（"已有研究表明…"替换为"近年来多项研究报道…"）
4. 保持字数相近，避免大幅删减导致内容缺失

### 第三步：输出结果

对每个改写段落，标注：
- 原文相似度
- 改写后预估相似度
- 使用的改写策略
- 原文与改写后对比

### 第四步：汇总建议

提供总体降重建议，包括：
- 总段落数、高相似段落数
- 建议优先处理段落顺序
- 降重后预估总体相似度范围

---

## 改写质量标准

1. **学术准确性**：核心概念和数据不得有误
2. **语义等价**：改写后意思与原文等价，不改变研究结论
3. **语言自然**：避免机械翻译感，语句通顺符合学术写作规范
4. **逻辑连贯**：改写后段落内部逻辑清晰，与上下文衔接自然
5. **术语一致**：同一术语在同一段落内保持统一

---

## 注意事项

- 本 skill 处理**文字重复（抄袭）检测**问题，AIGC 检测问题请使用 `humanizer-academic-zh` skill
- 本 skill 不提供虚假期刊信息或伪造数据，所有改写基于原文信息重构
- 改写后建议再次提交查重验证效果
- 对极高相似段落（90%+），可能需要引用多篇相关文献进行综合改写
