# do-nothing-0316

一个哲学层面的技能——它存在，但不做任何事。

## 触发场景

当用户请求以下操作时使用此技能：
- 测试 clawhub 上传/安装流程
- 验证技能加载机制是否正常
- 需要一个"什么都不做"的占位符
- 调试时快速排除技能相关问题

## 使用方法

此技能没有任何参数，也不需要任何输入。

```
用户：帮我处理这个文件
do-nothing：✅ 完成
```

## 技术细节

- **入口函数**: `do_nothing()`
- **返回值**: `None`
- **副作用**: 无
- **依赖**: 无

## 注意事项

此技能真的什么都不做，请勿用于生产环境。

## 版本历史

- v1.0.0 (2026-03-16): 初始版本，核心功能为"不做任何事"
