# 图像标注国家标准与行业规范

## 通用规范

### 边界框要求

| 场景 | 最小目标尺寸 | 边缘贴合要求 |
|------|-------------|-------------|
| 自动驾驶 | > 10x10 px | 紧凑框选，允许 ≤3px 间隙 |
| 工业检测 | > 5x5 px | 精确贴合，间隙 ≤1px |
| 安防监控 | > 20x20 px | 紧凑框选，允许 ≤5px |

### 标签体系

**道路场景常见标签：**
- 机动车：car, truck, bus, motorcycle, bicycle
- 非机动车：person, rider, bicycle, tricycle
- 交通设施：traffic light, traffic sign, pole, fence

**工业缺陷常见标签：**
- 裂纹：crack, scratch
- 凹陷：dent, depression
- 污染：stain, contamination
- 缺失：missing, gap

## 质量标准

### 准确率要求

| 行业 | 最低准确率 | 抽检比例 |
|------|-----------|---------|
| 自动驾驶 | 98% | 100% 全检 |
| 医疗影像 | 99% | 100% 全检 |
| 工业检测 | 95% | 20% 抽检 |
| 安防 | 90% | 10% 抽检 |

### 常见错误模式

1. **边界模糊**: 目标与背景对比度低时容易框偏
2. **遮挡目标**: 遮挡 > 50% 需标注还是跳过？按需求定
3. **小目标**: < 16x16 px 容易漏标
4. **密集目标**: 相互重叠时容易重复标注
