# 🔒 安全审查检查清单

## 注入防护
- [ ] SQL 使用参数化查询（非字符串拼接）
- [ ] XSS 输出正确转义
- [ ] 命令执行无用户输入拼接
- [ ] SSRF 防护（白名单域名/IP）
- [ ] LDAP/NoSQL 注入防护

## 认证与授权
- [ ] 接口有认证校验
- [ ] 资源有权限校验（非越权）
- [ ] 会话管理安全（超时、刷新）
- [ ] 密码使用安全哈希（bcrypt/argon2）
- [ ] MFA 关键操作有二次验证

## 数据保护
- [ ] 敏感信息不写入日志
- [ ] 配置文件无硬编码密钥
- [ ] 传输使用 HTTPS/TLS
- [ ] 敏感数据加密存储
- [ ] PII 数据脱敏处理

## 输入验证
- [ ] 用户输入有类型校验
- [ ] 文件上传有类型和大小限制
- [ ] API 参数有边界校验
- [ ] 反序列化使用安全方法
- [ ] XML 解析禁用外部实体

## 依赖安全
- [ ] 无可知漏洞依赖（CVE）
- [ ] 依赖版本锁定
- [ ] 不使用已废弃/不安全的库
- [ ] 第三方服务调用有超时和重试限制
