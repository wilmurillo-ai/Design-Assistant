# Changelog

## v1.1.0 (2026-04-18)

### 新增
- **记忆管理系统**：联系人记忆全局共享（`memories/contacts/`），按时间增量更新，支持身份、偏好、聊天风格、话题记忆等维度
- **消息来源识别**：sender_id + 前缀双重判断，支持识别"用户身份发送的 Bot 消息"
- **认识新朋友机制**：遇到未知群/联系人时先征求用户同意，支持群成员发现和新联系人录入
- **首次使用权限检查**：自动检测飞书 OAuth 权限，缺失时降级处理并提示授权
- **发送身份区分**：明确 Bot 身份 vs 用户身份的使用场景

### 优化
- **配置结构升级**：contacts 全局共享 + 群引用，同一联系人跨群共用一份配置
- **消息格式铁律强化**：5 条硬性规则（at 在前、post 类型、用户身份工具、prefix 自动取、2 元素结构）
- **快捷脚本**：`send_group_message.sh` 支持图片上传（`--image`）和手动 image_key（`--image-key`）
- **自适应回复逻辑**：根据 config.json 自动判断是否 @、用什么前缀

### 修复
- 前缀从分开的 text 元素改为合并在 at 后的单一 text 元素，避免显示异常
