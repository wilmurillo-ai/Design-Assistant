# Output Shapes

Pick the lightest artifact that matches the user's ask.

## 1. Evolution Chain

Use when the user wants the shortest useful answer.

Format:

- `seed proposition`
- `key shaping turns`
- `current mature proposition`
- `open tensions`

Best for: quick explanation, chat replies, compact thinking support.

## 2. Proposition Dossier

Use when the user wants a stable document for later reuse.

Include:

- proposition title
- early formulation
- current formulation
- why it qualifies as core
- shaping contributions with type + effect + confidence
- unresolved questions

Best for: notes, memory systems, graph seeding, documentation.

## 3. Turning-Point Map

Use when the important thing is how the proposition changed, not just what supports it.

Organize by:

- before
- turning point
- what changed
- after

Best for: understanding correction, reframing, or reconstruction.

## 4. Active-Line Snapshot

Use when the user cares about present direction, not just retrospective history.

Include:

- active proposition line
- current evidence that the line is still moving
- what kinds of events now count as likely future contributions
- what signals would mean the line is weakening or splitting

Best for: direction finding and ongoing recognition.

## 5. External Explanation

Use when the user wants to explain the proposition to someone else.

Include:

- one-sentence mature proposition
- why it did not appear suddenly
- 3 to 5 key shaping contributions
- what the final view now includes that the early view did not

Best for: essays, public writing, or concise external communication.
