# Cold Pitch Email Template

## Subject
`[Business Name] — Your New Website is Ready to View`

## Body

```
Hi [Owner Name / "there"],

I noticed [Business Name] doesn't have a website (or the current one needs some work), and I went ahead and built you one as a free demo.

You can see it here: [DEMO URL]

It's clean, fast, mobile-friendly, and built specifically for [trade] businesses in [City].

I'm [Your Name] from MJW Design Studio — we build websites for local businesses starting at a flat rate with no monthly fees.

If you'd like to keep it, customize it, or just have a quick 10-minute call, just reply to this email.

No pressure at all — just wanted to show you what's possible.

[Your Name]
MJW Design Studio
mjwdesignstudio.com
```

## Personalization Checklist

- [ ] Replace `[Business Name]` with actual business name
- [ ] Replace `[DEMO URL]` with the live VPS URL
- [ ] Replace `[trade]` with their specific trade (plumber, electrician, etc.)
- [ ] Replace `[City]` with their city
- [ ] Replace `[Owner Name / "there"]` with owner name if known from web search

## Follow-up (Day 3 if no reply)

Subject: `Re: [Business Name] — Quick follow-up`

```
Hi again,

Just wanted to make sure my last email didn't get buried. Your demo site is still live at [DEMO URL] if you'd like to take a look.

Happy to hop on a quick call or answer any questions.

[Your Name]
```

## Tips

- Send between 9am–11am local time (highest open rates)
- Never send more than 20/day from one Gmail account
- Track all sends in leads.md
