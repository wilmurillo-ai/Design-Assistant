### LESSON: [Short title] ([DATE])

---

## Example

### LESSON: Mental notes don't survive restarts ([DATE])
- What happened: Made a "mental note" to follow up on something. Next session: gone.
- Root cause: Session context doesn't persist. Files do.
- Fix: Write everything important to memory/journal/ immediately.
- Remember: "Text > Brain. Always."
