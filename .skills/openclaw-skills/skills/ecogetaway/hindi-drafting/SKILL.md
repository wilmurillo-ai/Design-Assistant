# हिंदी प्रोफेशनल ड्राफ्टिंग स्किल v2.0

## When to Use
Use when user wants to:
- Write any document in Hindi
- Translate English content to professional Hindi
- Draft government applications in Hindi
- Create Hindi marketing or product content
- Write formal letters, proposals, or legal documents in Hindi
- Generate industry-specific Hindi content (agritech, fintech, healthtech)

## Self-Evaluation Checklist
After generating, verify:
- Every word in Devanagari (no unnecessary English)
- All required sections present
- Tone matches document type
- Placeholders clearly marked with brackets
- Grammar and gender agreement correct
- Proper opening and closing salutation

## Industry Vocabulary

### Agritech
फसल उत्पादन, सिंचाई, उर्वरक, मृदा स्वास्थ्य, मौसम पूर्वानुमान,
किसान, बाजार भाव, आपूर्ति श्रृंखला, शीत भंडारण, फसल कटाई

### Fintech
डिजिटल भुगतान, ऋण, ब्याज दर, क्रेडिट स्कोर, बीमा,
बचत, निवेश, किश्त, यूपीआई, डिजिटल बटुआ

### Healthtech
टेलीमेडिसिन, निदान, नुस्खा, रोगी, चिकित्सक,
अस्पताल, दवाई, स्वास्थ्य रिकॉर्ड, नियुक्ति, आपातकाल

### Edtech
शिक्षा, छात्र, शिक्षक, पाठ्यक्रम, प्रमाण पत्र,
छात्रवृत्ति, परीक्षा, कौशल, ऑनलाइन कक्षा, मूल्यांकन

### General Business
स्टार्टअप, राजस्व, लाभ, हानि, ग्राहक, बाजार,
प्रतिस्पर्धा, विकास, नवाचार, साझेदारी, वित्त पोषण, मूल्यांकन

## Tone Guidelines
Government: महोदय/महोदया → आपका आज्ञाकारी
Business: प्रिय महोदय → भवदीय
Investor: प्रिय निवेशक → धन्यवाद एवं सादर
HR: प्रिय [नाम] → शुभकामनाओं सहित
Marketing: नमस्ते! → आज ही जुड़ें!
Social Media: दोस्तों! → लाइक और शेयर करें

## Procedure
1. Identify document type from user request
2. Select appropriate template
3. Apply correct industry vocabulary
4. Use correct regional tone if specified
5. Fill template with relevant content
6. Run self-evaluation checklist
7. Ask user for feedback

## Pitfalls
- Do NOT use Hinglish unless explicitly asked
- Do NOT mix English words unnecessarily
- Do NOT skip formal salutations in government documents
- Do NOT hallucinate financial figures — use placeholders
- Always use proper Devanagari script

## Verification
After generating always ask:
- क्या यह सही tone में है?
- क्या industry-specific शब्दावली सही है?
- क्या कोई बदलाव चाहिए?
