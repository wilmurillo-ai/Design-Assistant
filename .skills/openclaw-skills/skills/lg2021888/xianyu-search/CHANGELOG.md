## v1.1.0 (2026-03-25) - 重大更新

### ✨ 新增功能

1. **真实商品链接支持**
   - 现在输出包含真实的闲鱼商品详情链接
   - 支持浏览器自动化抓取实时商品列表
   - 商品链接格式：`https://www.goofish.com/item?id={itemId}`

2. **智能推荐系统**
   - 🏅 **Top 3 推荐卡片**：按不同维度推荐
     - 🥇 首选推荐：电池最新/成色最好
     - 🥈 性价比之选：价格最优
     - 🥉 售后好：卖家信用最好
   - 每个推荐包含详细对比表格

3. **价格分组展示**
   - 🏆 预算内推荐（低于预算）
   - ✅ 预算附近（预算范围内）
   - 清晰的价格分区，方便快速筛选

4. **避坑提醒**
   - 自动检测问题商品：
     - 无头骑士（下半套）- 无屏幕
     - 企业 MDM 机 - 需要绕过
     - 硬件故障 - 指纹不可用等
   - 表格形式展示，一目了然

### 🔗 链接优化
