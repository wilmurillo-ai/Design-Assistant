# 委员会裁决整合规则 v2

## 加权评分（各 20%）

综合得分 = (芒格 + 马克斯 + 段永平 + 德鲁肯米勒 + 西蒙斯) / 5

## 评级体系

| 综合得分 | 评级 | 建议仓位 |
|---|---|---|
| 8.0–10.0 | ⭐⭐⭐⭐⭐ 强烈买入 | 15–25% |
| 6.5–7.9 | ⭐⭐⭐⭐ 买入（保守） | 8–15% |
| 5.0–6.4 | ⭐⭐⭐ 等待/观望 | 0–5%（等回调） |
| 3.5–4.9 | ⭐⭐ 谨慎 | 0%（观察名单） |
| <3.5 | ⭐ 回避 | 0% |

> 仓位为相对仓位（占该资产类别配置比例），非绝对金额。

## 否决规则

**硬否决**：≥3 位委员评分 ≤ 4/10 → 评级「回避」，仓位 0%。
**软否决（各自独立触发，叠加生效）**：
- 马克斯评分 ≤ 5 且情绪=「贪婪/极度贪婪」→ 仓位减半
- 段永平动机审查不通过 → 仓位 ≤ 30%
- 西蒙斯动量「看空」→ 减仓 20–30% 或分批建仓

## 满仓条件

四位或以上委员评分 ≥ 7 → 可考虑上限仓位（不超过该评级建议上限）。

## 冲突处理

| 冲突场景 | 处理方式 |
|---|---|
| 芒格看好 + 马克斯说贵 | 买入但减半，等回调 |
| 段永平否决 + 西蒙斯看多动量 | 动机优先，放弃或极小仓位 |
| 德鲁肯米勒说时机不对 | 列入观察，等催化剂明确 |
| 西蒙斯看空 + 其余四人看好 | 减仓 20–30%，分批建仓 |

## 特殊资产注记

**加密 / 黄金**：芒格和段永平因无现金流资产天然给低分。报告中注明「宏观/量化视角（马克斯、德鲁肯米勒、西蒙斯）在本标的权重更具参考价值」。

## 报告必含要素

1. 五委员各自评分 + 核心判断（1–2 句）
2. 加权综合得分（保留 1 位小数）
3. 最终评级
4. 建议仓位 %
5. 核心逻辑（1–2 句共识）
6. 主要风险（1 句，最悲观委员的核心担忧）
7. 止损条件（1 句，来自德鲁肯米勒或西蒙斯）
8. 若触发硬否决 / 软否决，明确标注
9. 若 7 日内有财报，增加「财报时机建议」字段
