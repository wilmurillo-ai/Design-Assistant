# Security Policy

If you discover a security issue, please report it responsibly.

## Scope

This tool processes SAP journal entry exports (CSV/Excel files). It does not:
- Access live SAP systems
- Store or transmit data externally
- Process personal or sensitive data beyond what is in the export files

## Reporting

Send details to:
- https://radarroster.com/#contact

Please include:
- Steps to reproduce
- Impact assessment
- Suggested fix (if any)

## Response

We aim to acknowledge reports within 48 hours and provide a fix timeline within 7 days for confirmed issues.
