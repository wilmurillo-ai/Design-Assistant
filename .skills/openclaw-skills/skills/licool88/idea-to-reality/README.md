# 💡 idea-to-reality - 从想法到落地：八步落地法

> 你是否有很多想法躺在脑子里，从来没有开始？或者做了一半就放弃？这套方法论帮你真正把想法变成结果。

## 🎯 解决什么痛点

❌ 常见问题：
- 脑子里全是想法，从来没有落地
- 想做一件事，但不知道从哪里开始
- 经常半途而废，坚持不下去
- 努力了半天，方向错了越走越远
- 做成一次也无法复制，下次还是靠运气

✅ 这个技能给你：
- **一套可直接照着走的完整流程**，从筛选想法到复制经验，每一步都有方法和模板
- **弥补了经典框架缺失的关键环节**：前置筛选、预设止损、动力管理、经验复制
- **更符合人性**，不鼓吹意志力，接受倦怠和不确定性，设计容错路径

## ✨ 核心框架：八步落地法

```
┌─────────────────────────────────────────────────────┐
│              从想法到现实：八步落地法                │
├─────────────────────────────────────────────────────┤
│  🎯  【前置筛选】这件事到底值不值得做？             │
│  1️⃣  定目标：清晰定义，结果导向                    │
│  2️⃣  拆任务：拆解到第一步就能立即行动              │
│  3️⃣  找路径：资源盘点+风险预案+止损线              │
│  4️⃣  抓关键：找到关键路径，排出优先级              │
│  5️⃣  去执行：PDCA循环+动力管理                    │
│  6️⃣  做闭环：复盘总结+明确决策输出                 │
│  7️⃣  可复制：沉淀方法论，固化经验                  │
└─────────────────────────────────────────────────────┘
```

## 🚀 核心亮点

| 亮点 | 说明 |
|------|------|
| **前置筛选** | 先想清楚值不值得做，方向错了努力白费 |
| **预设止损** | 到点就止损，不被沉没成本绑架 |
| **人性友好** | 接受倦怠，不鼓吹意志力，环境设计比自律靠谱 |
| **关键路径** | 找到20%产生80%结果的关键动作，不平均用力 |
| **闭环决策** | 复盘必须输出明确决策（继续/止损/扩大），不是空话 |
| **经验复制** | 把一次成功变成可复制的能力，越做越顺 |

## 📋 使用方式

1. 打开 `templates/planning.md` 模板
2. 按照八步一步步填写
3. 填完你的思路就清晰了，直接开始执行
4. 每周对照复盘，调整方向

**一句话：填完模板，你就知道该怎么做了。**
