## Safety Checklist

Before placing an order, verify:
- [ ] Domain matches a trusted site (not a spoofed domain)
- [ ] Connection is HTTPS
- [ ] Not accessed via a paid search ad
- [ ] Price is reasonable (not suspiciously low)
- [ ] Payment page uses a known provider (Klarna / Stripe / Adyen / Swish)
