# 趋势洞察指南（trend）

> 命令为只读；依赖 AK 签名，无需业务 token。

## CLI 调用

```bash
python3 {baseDir}/cli.py trend --query "大码女装"
```

- `--query`：必填；关键词需“宽泛/不太细”，如类目、品类、品牌。建议 ≤5 个汉字或 ≤2 个英文单词；过长/过细可能无结果。

## 触发与自适应策略

- 必须有明确的单一 `query`（niche / 类目 / 品类 / 产品线）。一次只接受一个 query；若用户给出多个词，需拆成多次调用。
- 无 query 时先向用户索取；若上下文已明确意图，可先概括出1-2个候选类目词供用户确认后再调接口。
- 适用场景：周期趋势、行业/类目走势、商机规模、价格分布、淘宝爆款概况等。
- 若返回空/失败且 query 过长或过细：提示用户换更宽泛关键词（≤5 汉字或 ≤2 英文词）后再试；不在脚本层自动截断，需与用户协商。

## 输出理解

- 正常：`bizData` 为 Markdown 字符串，含“商机查询结果/趋势查询结果/淘宝爆款概况”等段落。
- 如需个性化呈现，可结合用户关注点（如价格分布、热品概况）挑重点重述，避免机械逐段照搬。
- 无结果：Markdown 会提示“未查询到相关趋势信息”；可提示用户换更宽泛关键词再试。

## 异常处理

- 通用 HTTP 异常（400/401/429/500）处理见 `references/common/error-handling.md`。
