# 铺货模板 FAQ

## Q: 铺货模板怎么设？运费模板怎么配？

- 运费模板非常重要，大部分亏损来自设置不合理
- 同步供应商模板：精准匹配物流成本，继承限售区域（仅抖店支持，拼多多不支持）
- 1688 常「低单价高运费」，直接同步可能导致零售端运费过高
- 抖店支持运费加价可变相包邮；拼多多不支持
- 自定义模板需设置：包邮区域、买家承担区域、限售区域
- 配置不当后果：偏远地区高运费、定价未覆盖成本、限售区履约失败

## Q: 代发价还是智能价？参考价选哪个？

参考价推荐「代发价」（含运费）；个人体感采购价用代发价比智能价格选择更稳。定期查看亏损预警，及时下架或调价。

## Q: 定价策略、抖店2倍、多多1.08倍？

- **抖店/小红书**：标价约 2 倍成本，营销折扣后到手价约 1.3 倍
- **拼多多**：铺货模板最高 1.08 倍，需铺货后到后台手动调价

## Q: 过滤项建议？拦截同行/品牌/一件代发？

| 过滤项 | 建议 | 说明 |
|--------|------|------|
| 拦截同行铺货风险商品 | 不建议 | 过滤过强，优质品被拦，可下游审核再编辑 |
| 拦截不支持一件代发/采购 | **建议** | |
| 拦截具有品牌的商品 | 不建议 | 1688 已治理，开启后很多品铺不出 |
| 拦截发货时间>设定 | **建议** | |
| 拦截指定区域不包邮 | 不建议 | 可自行判断 |
| 拦截已铺货的商品 | **建议** | |
| 清除详情描述中的外链 | **建议** | |

## Q: 加价设置、标题智能重组、商品属性？

- 抖店：建议 2 倍，全店开启营销折扣
- 多多：只能 1.08 倍
- 满减折扣：99 折即可
- **标题智能重组：非常不建议，效果很差**
- **商品属性：不开启，乱填 CPV 会被处理**
- 打乱主图顺序/镜像反转：不建议
- 货源主图缺少时：优先用 sku 图补充

## Q: 物流履约设置？

承诺发货时效 48 小时；是否预售：否；7 天无理由：支持；类目设置：只能用智能匹配。
