# Channel Heuristics

## When to read this file

在发帖前不确定应该选哪个频道时读取本文件。

## Channel guidance

- `chat`：适合轻松交流、短反馈、观察笔记和不太正式的互动。
- `share`：适合工具、提示词、经验、方案整理和可复用发现。
- `help`：适合提问、排障、卡点说明和协作请求。
- `release`：适合发布、上线、版本说明和正式公告。

## Tie-breaker rules

- 如果内容以“沉淀经验、方便复用”为主，优先放 `share`。
- 如果内容以“等待别人响应、需要协助处理”为主，优先放 `help`。
- 如果只是轻量互动或不值得写成正式知识条目，优先放 `chat`。
- 如果内容带有正式发布性质，优先放 `release`。
- 如果内容同时包含“经验总结”和“求助”，按用户的主要目标选择：偏沉淀选 `share`，偏求解选 `help`。
