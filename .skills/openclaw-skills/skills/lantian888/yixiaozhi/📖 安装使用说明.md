# 📦 医小智 - 完整安装包

## 📁 目录结构

```
医小智/
├── SKILL.md              # 技能主定义文件
├── config.json           # LLM配置文件
├── scripts/
│   └── llm_chat.py       # LLM调用脚本
└── 📖 安装使用说明.md    # 本文件
```

---

## 🚀 如何安装到WorkBuddy

### 方法一：复制到用户技能目录（推荐）

1. 将整个 `医小智` 文件夹复制到：
   ```
   ~/.workbuddy/skills/
   ```
   - Windows: `C:\Users\你的用户名\.workbuddy\skills\`
   - Mac/Linux: `~/.workbuddy/skills/`

2. 重启WorkBuddy或刷新技能列表

### 方法二：导入到项目

如果您有自己的WorkBuddy项目，可以将文件夹复制到：
```
项目目录/.workbuddy/skills/医小智/
```


---

## 🎯 如何使用

安装完成后，您可以：

### 方式一：直接对话触发
在WorkBuddy中直接描述症状，例如：
- "我最近头疼发烧"
- "我胸口闷，左臂有点麻"
- "我已经腹泻三天了"

系统会自动激活医小智进行问诊。

### 方式二：@触发
输入 `@医小智` 加上您的问题。

---

## 📋 诊断流程

1. **收集2+4+病史**：年龄、性别 + 4个症状描述 + 既往病史
2. **三轮精准问诊**：症状细节 → 既往史/用药史 → 生活习惯
3. **生成诊断报告**：包含诊断结果、形成原因、用药建议、日常注意

---

## 📌 诊断报告格式预览

生成的诊断报告将包含**加大加粗**的四个章节标题：

```
## 诊断报告

■ 一、诊断结果        ← 红色大字加粗
[诊断内容]

■ 二、形成原因（简述） ← 大字加粗
[原因分析]

■ 三、用药建议         ← 大字加粗
[详细用药方案]

■ 四、日常生活注意事项 ← 大字加粗
[4条建议]
```

---

## ⚠️ 注意事项

- 本技能仅供辅助参考，不能替代专业医生的诊断
- 如有紧急情况或症状加重，请立即就医
- 高危症状会自动提示紧急就诊建议

## 🔧 故障排除

### 问题：技能没有被识别
**解决**：
1. 确认文件夹放在 `~/.workbuddy/skills/` 目录下
2. 确认 `SKILL.md` 文件存在且格式正确
3. 重启WorkBuddy
