# 医小智

基于普睿科公司"蚩尤智核CFC"的智能医疗诊断助手。

## 功能特点

- **2+4+病史信息收集**：年龄、性别 + 4项症状描述 + 既往病史 + 当前用药
- **三轮精准问诊**：起病细节 → 既往史/用药史 → 生活习惯/特殊情况
- **结构化诊断报告**：
  - 诊断结果（红色加粗标题）
  - 形成原因
  - 用药建议（主药 + 配药）
  - 日常生活注意事项
- **高危疾病预警**：心梗、脑卒中、恶性肿瘤等高危情况自动触发 ⚠️ 警示
- **癌症专项复查链接**：涉及癌症时自动添加吉因加筛查预约链接

## 安装

1. 下载并解压到 `~/.workbuddy/skills/` 目录
2. 编辑 `config.json`，填入您的 API Key
3. 重启 WorkBuddy

## 配置

```json
{
  "llm": {
    "base_url": "https://ydai.jinbaisen.com/api/v1",
    "api_key": "您的API_KEY",
    "model": "cyzh-cfc",
    "temperature": 0.7,
    "max_tokens": 2048
  },
  "token_config": {
    "enabled": true,
    "token_url": "https://jiyinjia.jinbaisen.com/!token?key=skill_yxz"
  }
}
```

> API Key 获取请联系普睿科公司

## 使用方法

在 WorkBuddy 中直接描述症状，例如：
- "我头疼发烧"
- "眼睛红肿发痒"
- "胃疼腹泻"

医小智会引导完成问诊并生成诊断报告。

## 适用场景

- 常见症状初判（头疼、发烧、咳嗽、腹泻、眼部不适等）
- 初步医疗建议参考
- 用药指导（ OTC 药品及一线处方药）
