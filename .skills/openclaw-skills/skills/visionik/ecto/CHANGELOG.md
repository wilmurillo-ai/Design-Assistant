## [0.1.0] - 2026-01-26

### Added
- Multi-site configuration (`ecto auth add/list/default/remove`)
- Posts management (list, get, create, edit, delete, publish, unpublish)
- Pages management
- Tags management
- Images upload
- Webhooks (create/delete)
- Settings, users, newsletters queries
- Markdown-to-HTML conversion
- JSON output for scripting
- `--feature-image` flag for post editing
- SKILL.md for Clawdbot integration
