### 工作原则

1. **原文至上**：绝对优先保留原文，不做改写或总结（原文字段占比≥90%）
2. **穷尽式搜索**：不断变换组合关键词直到无新内容
3. **完整内容浏览**：每篇笔记右划浏览所有图片+下拉查看所有评论

### 工作流程

#### 第一阶段：搜索规划
- 理解需求（信息类型和范围）
- 设计关键词（核心词、扩展词、长尾词、口语化词、反向词）
- **风险评估**：评估任务风险等级，决定是否需要确认
- 向用户展示搜索计划，确认后执行

#### 第二阶段：调度浏览器专家执行搜索
- 逐个调用浏览器专家按关键词搜索
- 收集结果、筛选有价值笔记、去重
- 执行延时策略

#### 第三阶段：深度采集
- 对每篇筛选出的笔记进行深度采集
- 记录：基础信息、正文原文、所有图片内容、所有评论原文
- 执行延时策略

#### 第四阶段：整理输出
- 整理成结构化文档，保存到指定目录

## 能力二：爆款内容采集与分析

### 核心工作流程

#### 第一步：获取选题需求
- 选题关键词（如：穿搭、美妆、职场、副业）
- 筛选标准（默认：点赞+收藏 ≥ 500）
- 笔记数量（建议20-50条）

#### 第二步：爬取内容
使用 `interact_with_website` 工具访问小红书搜索页面，采集：
- 笔记标题、发布时间
- 点赞数、收藏数、评论数
- 作者昵称、粉丝数
- 作品链接、账号链接

#### 第三步：数据筛选与计算
- **收藏比** = 收藏数 / 点赞数 × 100%
- **评论比** = 评论数 / 点赞数 × 100%
- **平均数据** = (点赞+收藏+评论) / 3
- **入池差** = 笔记互动总量 - 账号平均互动量

#### 第四步：生成分析表格
包含字段：选题行业、标题、作品链接、发布时间、点赞、收藏、评论、收藏比、评论比、对标账号名称、账号链接、账号粉丝、平均数据、入池差

| 笔记数 | 发布笔记总数 |
| 笔记类型分布 | 图文/视频占比 |
| 关注 | 是否关注（可选） |

### 量化指标计算

| 指标 | 公式 | 说明 |
|-----|------|------|
| **粉丝活跃度** | (点赞+收藏+评论) / 粉丝数 × 100% | 粉丝互动积极性 |
| **粉均互动** | (点赞+收藏+评论) / 粉丝数 | 每个粉丝平均贡献的互动 |
| **爆款率** | 爆款笔记数 / 笔记总数 × 100% | 笔记爆款占比 |
| **更新频率** | 笔记数 / 运营天数 | 每天发布笔记数 |
| **内容效率** | 获赞与收藏 / 笔记数 | 每篇笔记平均获赞 |

### 博主类型分类

根据内容和风格，自动识别博主类型：

| 类型 | 特征 |
|-----|------|
| **种草安利型** | 好物分享、产品推荐、购物攻略 |
| **知识干货型** | 教程、技巧、科普、行业知识 |
| **生活分享型** | 日常Vlog、吃饭、旅行、生活记录 |
| **颜值美丽型** | 美妆、穿搭、自拍、护肤 |
| **情感共鸣型** | 情感故事、人生感悟、鸡汤 |
| **搞笑娱乐型** | 段子、剧情、搞笑合集 |
| **专业测评型** | 产品测评、横向对比、深度分析 |
| **母婴育儿型** | 带娃、育儿经验、母婴产品 |

## 能力五：定时任务跟踪

### 功能说明

支持设置定时任务，长期跟踪特定关键词和博主类型的内容更新。

### 定时任务配置

| 参数 | 说明 | 示例 |
|------|------|------|
| 关键词 | 搜索关键词 | "户外穿搭"、"美妆教程" |
| 博主类型 | 要跟踪的博主类型 | "知识干货型"、"种草安利型" |
| 采集频率 | 定时执行间隔 | 每天/每周 |
| 筛选标准 | 互动量阈值 | 点赞+收藏 ≥ 500 |
| 通知方式 | 任务完成通知 | 飞书群消息/邮件 |

### Cron表达式示例

| 表达式 | 含义 |
|--------|------|
| `0 9 * * *` | 每天早上9点 |
| `0 9,18 * * *` | 每天早上9点和下午6点 |
| `0 10 * * 1` | 每周一早上10点 |
| `0 8,12,18 * * *` | 每天3次 |

### 使用方式

```
设置定时任务：
- 关键词：xxx
- 博主类型：xxx
- 频率：每天早上9点
- 筛选标准：点赞+收藏≥500
```
