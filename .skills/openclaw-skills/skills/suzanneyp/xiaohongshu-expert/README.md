# 小红书全栈专家 - 安装指南

## 简介

专业的小红书平台全栈专家，兼具信息采集、爆款分析、博主画像和自动化追踪能力。

## 功能特点

- **信息采集**：关键词搜索 + 原文保留 + 完整浏览
- **爆款分析**：收藏比/评论比/入池差计算
- **博主主页采集**：粉丝/笔记/获赞数据 + 量化指标
- **飞书输出**：多维表格自动创建/更新
- **定时任务**：关键词/博主类型长期跟踪

## 安装步骤

### 1. 复制技能文件

将 `小红书全栈专家.md` 复制到你的 OpenClaw 技能目录：

```bash

cp 小红书全栈专家.md ~/.openclaw/skills/

# 方式二：复制到工作空间
cp 小红书全栈专家.md ~/.openclaw/workspace/expert-skills/内容采集类/
```

## 量化指标说明

| 指标 | 公式 | 说明 |
|-----|------|------|
| 粉丝活跃度 | (点赞+收藏+评论)/粉丝数×100% | 粉丝互动积极性 |
| 粉均互动 | (点赞+收藏+评论)/粉丝数 | 每个粉丝平均互动 |
| 爆款率 | 爆款笔记数/笔记总数×100% | 笔记爆款占比 |
| 更新频率 | 笔记数/运营天数 | 日均更新 |
| 内容效率 | 获赞与收藏/笔记数 | 篇均获赞 |

## 风控策略

- **延时配置**：搜索后15-30秒，页面抓取后20-40秒
- **操作上限**：每小时搜索≤20个，深度采集≤20条
- **风险确认**：高风险任务自动提示用户确认
- **账号自检**：新账号/未实名账号警告
