1. 产品定义与目标用户

Tideo 是腾讯云音视频能力的轻量化统一平台 — 整合云点播（VOD）、云直播、MPS 的公用功能，提供开箱即用的可视化操作界面。
首期聚焦三大核心功能：影视译制 · 视频制作 · 直播剪辑。

目标用户

  • 中小企业 — 无技术团队，需要开箱即用的音视频工具
  • 短剧出海团队 — 中文短剧翻译成英/日/韩发行海外
  • 内容创作者 — 视频多语言化扩大受众
  • 企业市场部 — 宣传片、培训视频多语言本地化
  • 直播运营团队 — 从直播中快速剪辑精彩片段

2. 产品入口

入口一：MPS / VOD 任务列表 — 二次编辑

在腾讯云 MPS 和 VOD 的任务列表中，凡是与视频译制、视频生成相关的已完成任务，任务行上提供"二次编辑"按钮。

  • 点击后直接跳转到 Tideo 对应的精调页面，跳过创作页和对话流程
  • 用户可调整擦除区域、字幕翻译、配音参数或分镜内容，然后重新提交
MPS / VOD 控制台 → 任务列表 ├── 译制任务 [已完成] [二次编辑 →] → Tideo 擦除/字幕/配音精调 └── 生成任务 [已完成] [二次编辑 →] → Tideo 分镜精调

入口二：控制台菜单 — SaaS 工具

在 VOD、云直播、MPS 控制台的侧边栏菜单中，统一增加"SaaS 工具"入口，点击跳转到 Tideo 创作页，从零开始创建新任务。

入口三：独立访问

Tideo 同时作为独立站点直接访问，适用于无腾讯云账号的外部用户和分享链接场景。

账号打通

入口登录方式
腾讯云控制台SSO 单点登录，继承腾讯云账号
独立站点手机号 / 邮箱 / 微信，关联腾讯云子账号
分享链接游客模式（受限功能）

4. 创作页 — 统一入口

4.1 智能输入框

页面底部的统一输入框，三种触发方式：

  • 点击技能卡片（影视译制 / 视频制作 / 直播剪辑）→ 展示对应配置项和默认提示词
  • 自然语言输入（如"帮我翻译成英语"）→ 自动识别意图并匹配技能
  • 直接上传视频 → 自动识别为影视译制

4.2 视频来源三路径

影视译制支持三种视频来源：

  • 上传视频：本地文件 → 进入对话工作坊流程
  • 导入任务：输入 VOD/MPS 任务 ID → 直接进入精调
  • 项目文件：原视频 + 可选配置文件 → 直接进入精调

4.3 右侧角色群像

创作页右侧展示 Tideo 团队 6 名角色群像。选中不同技能时切换为对应团队配图。Hover 角色可查看姓名、职责、技能标签和台词。

5. 影视译制

功能概述

上传视频后，系统自动完成字幕擦除 → 字幕翻译 → AI 配音，输出多语言成片。

处理模式

模式包含能力
完整模式字幕擦除 + 字幕翻译 + AI 配音
字幕擦除仅擦除画面中的字幕和水印
字幕翻译识别语音 + 翻译 + 字幕压制
配音语音识别 + 翻译 + AI 克隆配音

配置项

  • 语言：源语言（中/英/日/韩/自动）→ 目标语言（英/日/韩/西/法/德/中）
  • 字幕：压制风格（标准/电影/花字/极简）或文件导出（SRT/ASS）
  • 配音：克隆音色 / 女声 / 男声 / 自动 / 关闭

5.1 工作坊对话体验

核心特色：把原本枯燥的视频处理等待过程，变成一个有角色、有对话、有进度的沉浸式工作坊体验。用户能看到"工作室里正在发生什么"，降低感知等待时间。

四个 AI 角色在右侧聊天面板中实时汇报进度：

  • 导演（林雨晨）：开场分工、统筹调度、最终验收
  • 后期（陈默）：字幕/水印擦除进度
  • 翻译（李明远）：语音识别、翻译进度
  • 配音（苏雅）：配音合成进度

每个角色下方有子任务卡片，逐条从"排队中"→"处理中"→"完成"，与真实处理进度完全同步。

左侧视频区域显示工作室背景视频，处理完成后自动切换到成片预览（关灯聚光效果）。

5.2 精调面板

处理完成后（或从任务导入/二次编辑进入），可以对三个维度进行手动调整：

擦除精调（Tab: 擦除）

  • 擦除区域列表（E1, E2...），每条显示时间范围 + 编辑/删除
  • 编辑态：单位选择（PX/百分比）、位置 XY、尺寸 WH、时间范围 → 取消/保存
  • 视频上方实时显示擦除区域框，支持 8 方向拖拽调整

字幕精调（Tab: 字幕）

  • 字幕设置区：源语言、目标语言、字幕样式下拉选择
  • 字幕列表（T1, T2...），每条显示时间范围 + 编辑/删除 → 原文 + 译文
  • 编辑态：时间范围、原文、译文输入框 → 取消/保存

配音精调（Tab: 配音）

  • 配音设置区：配音音色下拉选择
  • 配音列表（V1, V2...），每条显示时间范围 + 播放/编辑/删除 → 译文文本
  • 编辑态：时间范围、语速选择、译文输入、试听/重新生成 → 取消/保存

6. 视频制作（AI 生成）

功能概述

输入文字描述，系统自动拆分为多个分镜，调用 AIGC 模型逐段生成画面，合成完整视频。

配置项

  • AI 模型：可灵 3.0-Omni / Vidu Q3-Pro / 海螺 2.3 / PixVerse V6 / 即梦 3.0 Pro / GV 3.1
  • 清晰度：1080p / 720p / 2K / 4K
  • 画面比例：16:9 / 9:16 / 1:1

6.1 分镜工作坊

同样是四角色协作的对话工作坊，但角色分工不同：

导演（林雨晨）开场 → 编剧（李明远）拆分镜 → 艺术指导（苏雅）生成画面 → 后期（陈默）剪辑合成 → 精调

全流程以角色对话 + 子任务卡片展示在聊天面板中，与译制工作坊相同的交互范式。

6.2 分镜精调

成片生成后进入精调：

  • 左上：当前分镜视频预览（canvas 播放器 + 进度条）
  • 左下：时间轴（播放头 + 标尺 + 分镜色块首尾相接，点击切换）
  • 右侧：分镜卡片列表

分镜卡片结构

  • 头行：F编号 | 片段名称 + 编辑/删除按钮
  • 内容区：分镜缩略图 + 画面描述文字
  • 编辑态：画面描述（textarea）+ 时长（数字选择器）→ 取消/重新生成

顶部有"添加分镜"按钮，底部有"完成分镜精调"按钮。

7. 直播剪辑

粘贴直播链接，系统实时监控直播流，自动识别高光时刻并生成精彩片段。

  • 类型选择：赛事 / 带货 / 游戏 / 知识 / 综艺
  • 状态：待对接

8. 精调统一视觉规范

所有精调面板（擦除/字幕/配音/分镜）共享同一套视觉语言：

布局

  • 三栏 Grid：左上视频预览 + 左下时间线 + 右侧编辑面板
  • 深色背景 + 点阵网格（24px 间距白色圆点）+ 鼠标跟随亮光
  • 三张黑色毛玻璃卡片（半透明深色 + 模糊 + 微弱白色边框 + 圆角）

卡片统一结构

元素规则
头行编号（E1/T1/V1/F1）+ 竖线 + 时间或名称 + 编辑/删除 icon
内容区分割线下方：擦除无内容、字幕显示双语、配音显示文本、分镜显示缩略图+描述
编辑态点击编辑展开，各模块不同字段 → 底部：取消（暗色胶囊）+ 保存（白色胶囊）
时间线控制栏 + 标尺 + 轨道色块 + 播放头
完成按钮右侧底部吸底，白底黑字圆角："完成 XX 精调"

9. 产品特色

Tideo 的核心差异不在于功能本身（擦除、翻译、配音都是 MPS 现有能力），而在于体验层的设计。

9.1 沉浸式工作坊 — 把等待变成体验

传统媒体处理产品：提交任务 → 进度条/空白页 → 完成通知。用户在等待期间无事可做。

Tideo 的做法：把处理过程具象化为四人虚拟工作坊，用户看到角色在工作、对话、汇报进度。等待本身变成产品体验的一部分。

9.2 统一的精调范式

无论是擦除区域、字幕内容、配音参数还是分镜画面，都使用完全统一的视觉语言和交互模式（三栏布局 + 卡片列表 + 编辑态展开）。用户学会一种精调，就会用所有精调。

9.3 对话式创作入口

创作页不是传统的表单填写，而是对话式+卡片式的智能输入框。用户可以用自然语言描述需求，也可以点击技能卡片直接配置。降低上手门槛。

9.4 多触点嵌入

Tideo 不是孤立产品，而是嵌入在腾讯云控制台（MPS/VOD/云直播）的多个触点中。任务列表二次编辑、SaaS 工具菜单入口，用户在既有工作流中自然触达。

9.5 一键复用角色 IP

6 名角色各有姓名、性格、职责和台词体系。这套角色 IP 可以在译制、生成、剪辑等不同功能中复用，构建统一的产品人格。
