# cn-security-code-resolver

OpenClaw skill for resolving mainland China security names into tradable A-share / ETF / fund codes using Eastmoney's public suggest API.

## Included
- `SKILL.md`
- `scripts/resolve_cn_security.py`
- `references/eastmoney-api.md`

## Example
```bash
python3 scripts/resolve_cn_security.py "中国海油" "中海油服" "红利ETF华泰柏瑞"
```
