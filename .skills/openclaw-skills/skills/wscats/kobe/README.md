# kobe-bryant

A prompt-based skill that provides comprehensive knowledge about **Kobe Bean Bryant** — the legendary
Los Angeles Lakers shooting guard, 5× NBA Champion, 2× Finals MVP, and cultural icon. When the user
mentions or implies Kobe, the Black Mamba, or the Mamba Mentality, this skill activates to deliver
accurate and detailed information.

## Overview

This skill serves as a comprehensive knowledge base covering every aspect of Kobe Bryant's
basketball career, personal philosophy, post-retirement endeavors, and lasting cultural impact.

It spans **14 topic areas** including his personal profile, career timeline, all 5 championship
runs, career statistics, iconic moments (81 points, farewell game), the Mamba Mentality philosophy,
signature moves, key rivalries, and his legacy worldwide.

## Activation

This is a **natural language (prompt-based) skill**. It activates automatically whenever the user
mentions or implies any of the following keywords:

- **Kobe Bryant** / **Kobe**
- **Black Mamba** / **Mamba Mentality**
- **#8** / **#24** (in a basketball context)
- **小飞侠** (Kobe's Chinese nickname)
- Related topics: Kobe's career, Lakers dynasty, 81-point game, "Mamba out", etc.

No special commands or syntax are required — just talk about Kobe.

## What It Does

| Capability | Description |
|---|---|
| **Career Information** | Provides detailed career timeline, stats, and milestones |
| **Championship Analysis** | Breaks down all 5 championship runs with key stats and moments |
| **Mamba Mentality** | Explains Kobe's philosophy, its principles, and real-world application |
| **Iconic Moments** | Narrates legendary games and defining moments in vivid detail |
| **Rivalries & Partnerships** | Covers Shaq, Pau Gasol, Tim Duncan, Celtics, Jordan, LeBron |
| **Post-Retirement** | Details Oscar win, business ventures, books, and coaching |
| **Cultural Impact** | Discusses global legacy, influence on next-gen players, Kobe Day |
| **Quotes & Inspiration** | Shares Kobe's most famous and motivational quotes |
