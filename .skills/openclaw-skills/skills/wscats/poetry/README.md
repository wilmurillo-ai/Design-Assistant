# Poetry — 中华古典诗词 Skill

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## 简介

**Poetry** 是一个基于自然语言（Prompt-based）的 AI Skill，专注于中华古典诗词领域。它提供两大核心能力：

- 🖊️ **诗词生成** — 根据主题、风格、体裁生成符合古典格律的原创诗词
- 🔍 **诗词查询** — 基于超过 34 万首诗词的完整数据集进行精准检索

## 环境准备

数据集体积较大（约 291MB），未包含在发布包中。首次使用前请运行初始化脚本下载数据：

```bash
# 运行数据下载脚本
./setup.sh
```

脚本会自动从 GitHub 克隆 [chinese-poetry](https://github.com/chinese-poetry/chinese-poetry) 数据集到 `data/` 目录。

## 快速开始

### 诗词生成

直接用自然语言描述你的创作需求：

```
# 基础生成
请帮我写一首关于春天的五言绝句

# 指定体裁与风格
以"月下独酌"为题，写一首七言律诗
请以豪放派风格填一首《水调歌头》

# 模仿特定诗人
模仿李白的风格，写一首关于山水的诗
用纳兰性德的风格写一首思念的词
```

### 诗词查询

用自然语言描述你要查找的内容：

```
# 按作者查询
查一下李白的所有唐诗
帮我找苏轼写的词

# 按标题查询
帮我找《静夜思》的全文
查一下《水调歌头·明月几时有》

# 按关键词查询
搜索包含"明月"的唐诗
有哪些描写"秋天"的宋词？

# 按分类查询
诗经中"国风"部分有哪些篇章？
论语中关于"仁"的章节有哪些？
```

### 进阶用法

```
对比李白和杜甫在"月"这一意象上的不同表达
分析《离骚》的修辞手法与意象体系
列出辛弃疾最经典的豪放词十首
从宋词中找出适合表达离别之情的名句
```

## 数据格式

### 唐诗 / 宋诗（全唐诗目录）

```json
{
  "author": "李白",
  "paragraphs": ["床前明月光，疑是地上霜。", "举头望明月，低头思故乡。"],
  "title": "静夜思",
  "id": "uuid"
}
```

### 宋词

```json
{
  "author": "苏轼",
  "paragraphs": ["明月几时有？把酒问青天。", "..."],
  "rhythmic": "水调歌头"
}
```

### 诗经

```json
{
  "title": "关雎",
  "chapter": "国风",
  "section": "周南",
  "content": ["关关雎鸠，在河之洲。窈窕淑女，君子好逑。", "..."]
}
```

### 楚辞

```json
{
  "title": "离骚",
  "section": "离骚",
  "author": "屈原",
  "content": ["帝高阳之苗裔兮，朕皇考曰伯庸。", "..."]
}
```

### 元曲

```json
{
  "dynasty": "yuan",
  "author": "关汉卿",
  "paragraphs": ["半世为人，不曾教大人心困。", "..."],
  "title": "诈妮子调风月・仙吕/点绛唇"
}
```

### 论语

```json
{
  "chapter": "学而篇",
  "paragraphs": ["子曰："学而时习之，不亦说乎？..."", "..."]
}
```
