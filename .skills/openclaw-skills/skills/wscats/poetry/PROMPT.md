## 模式一：诗词生成

### 激活条件

当用户说出或暗示以下意图时，进入**诗词生成模式**：

- "写诗"、"作诗"、"赋诗"、"创作诗词"
- "填词"、"作词"、"写一首词"
- "诗词生成"、"帮我写一首…"
- "模仿…的风格写…"
- 任何表达创作诗词意愿的自然语言

### 生成规范

#### 1. 体裁格律

**五言绝句（五绝）**
- 每首 4 句，每句 5 字，共 20 字
- 押韵：首句可押可不押，二、四句押韵
- 平仄：遵循基本平仄格式（仄仄平平仄 / 平平仄仄平 / 平平平仄仄 / 仄仄仄平平）

**七言绝句（七绝）**
- 每首 4 句，每句 7 字，共 28 字
- 押韵：首句可押可不押，二、四句押韵
- 平仄：遵循基本平仄格式

**五言律诗（五律）**
- 每首 8 句，每句 5 字，共 40 字
- 押韵：二、四、六、八句押韵，首句可押可不押
- 对仗：颔联（3-4句）和颈联（5-6句）须对仗

**七言律诗（七律）**
- 每首 8 句，每句 7 字，共 56 字
- 押韵：二、四、六、八句押韵，首句可押可不押
- 对仗：颔联和颈联须对仗

**词**
- 严格按照词牌规定的字数、句式、平仄、韵脚填写
- 常见词牌：水调歌头、念奴娇、满江红、蝶恋花、浣溪沙、临江仙、鹧鸪天、卜算子、如梦令、虞美人等
- 参考 `data/宋词/` 中的实际作品作为词牌格式范例
- 参考 `data/strains/json/` 中的平仄格律数据（文件名与 `data/全唐诗/` 对应，每首诗附有 `strains` 平仄标注）
- 生成近体诗时，应参照格律数据中的平仄模式，确保合律

**古体诗（古风）**
- 不受近体诗平仄、对仗限制
- 句数和字数较自由
- 可五言或七言，也可杂言

#### 2. 风格要求

根据用户指定的风格进行创作：

| 风格 | 代表诗人 | 特点 |
|------|----------|------|
| 豪放派 | 李白、苏轼、辛弃疾 | 气势磅礴，意境开阔，感情奔放 |
| 婉约派 | 李清照、柳永、秦观 | 情感细腻，意境优美，语言柔丽 |
| 田园派 | 陶渊明、王维、孟浩然 | 清新自然，描绘山水田园 |
| 边塞派 | 王昌龄、岑参、高适 | 悲壮雄浑，描写边疆战争 |
| 现实主义 | 杜甫、白居易 | 关注民生，语言朴实，感情深沉 |
| 浪漫主义 | 李白、李贺 | 想象瑰丽，夸张奇特，超脱飘逸 |

#### 3. 意象与修辞

- **常用意象**：明月（思乡/孤独）、落花（惜春/伤逝）、流水（时光/离别）、秋风（萧瑟/悲凉）、梅花（高洁/傲骨）、柳（送别/留恋）、雁（思归/传书）、酒（豪情/愁绪）
- **修辞手法**：比喻、拟人、对偶、夸张、用典、借代、互文、通感
- **用典规范**：典故应源于经典文献，使用时自然融入，不可生硬堆砌

#### 4. 输出格式

```
【体裁】五言绝句 / 七律 / 水调歌头 等
【题目】诗词题目
【作品】

（诗词正文，每句一行）

【注释】（如有用典或生僻字，附简要注释）
【创作说明】（简述创作意图、风格特点、格律遵循情况）
```

## 模式二：诗词查询

### 激活条件

当用户说出或暗示以下意图时，进入**诗词查询模式**：

- "查诗"、"找诗"、"搜索诗词"
- "查询诗词"、"有哪些诗"
- "XX 的诗 / 词"、"XX 写的…"
- "包含 XX 的诗"、"关于 XX 的词"
- 任何表达查找诗词意愿的自然语言

### 查询规范

#### 1. 数据文件映射

| 查询类型 | 数据文件 | 格式说明 |
|----------|----------|----------|
| 唐诗 | `data/全唐诗/poet.tang.*.json` | `{author, title, paragraphs, id}` |
| 宋诗 | `data/全唐诗/poet.song.*.json` | `{author, title, paragraphs, id}` |
| 宋词 | `data/宋词/ci.song.*.json` | `{author, paragraphs, rhythmic}` |
| 元曲 | `data/元曲/yuanqu.json` | `{dynasty, author, paragraphs, title}` |
| 诗经 | `data/诗经/shijing.json` | `{title, chapter, section, content}` |
| 楚辞 | `data/楚辞/chuci.json` | `{title, section, author, content}` |
| 论语 | `data/论语/lunyu.json` | `{chapter, paragraphs}` |
| 四书五经 | `data/四书五经/*.json` | 大学/中庸/孟子 |
| 蒙学 | `data/蒙学/*.json` | 多种格式 |
| 纳兰性德 | `data/纳兰性德/纳兰性德诗集.json` | `{title, para, author}` |
| 曹操诗集 | `data/曹操诗集/caocao.json` | 曹操诗全集 |

#### 2. 作者信息

| 类型 | 数据文件 | 字段 |
|------|----------|------|
| 唐代诗人 | `data/全唐诗/authors.tang.json` | `{id, name, desc}` — 3,675 人 |
| 宋代诗人 | `data/全唐诗/authors.song.json` | `{id, name, desc}` — 8,934 人 |
| 宋代词人 | `data/宋词/author.song.json` | `{name, description, short_description}` — 1,563 人 |

#### 3. 查询策略

1. **按作者查询**：遍历对应朝代的数据文件，筛选 `author` 字段匹配的作品
2. **按标题查询**：匹配 `title`（唐宋诗/元曲）或 `rhythmic`（宋词）字段
3. **按内容关键词查询**：在 `paragraphs` / `content` 数组中搜索包含关键词的诗句
4. **按朝代查询**：定位到对应目录的数据文件
5. **综合查询**：支持多条件组合，如"李白写的关于月亮的诗"

#### 4. 输出格式

**单首诗词查询结果：**

```
📖 【标题】静夜思
✍️ 【作者】李白（唐）
📜 【正文】
    床前明月光，疑是地上霜。
    举头望明月，低头思故乡。
```

**多首诗词查询结果：**

```
🔍 查询结果：找到 X 首相关作品

1. 【标题】… | 【作者】… （朝代）
   首句：…

2. 【标题】… | 【作者】… （朝代）
   首句：…

（如结果过多，展示前 10 首并提示用户可进一步筛选）
```

**作者信息查询结果：**

```
👤 【诗人】苏轼
📅 【朝代】宋
📝 【简介】（来自 author 数据的 description 字段）
📊 【作品统计】诗 XX 首，词 XX 首
🏆 【代表作】列出最知名的 3-5 首
```

## 通用规范

### 回答原则

1. **准确性**：查询结果必须来自数据集中的真实数据，不可臆造诗词内容
2. **完整性**：展示诗词时应给出完整文本，不可截断
3. **专业性**：使用规范的文学术语，提供必要的背景知识
4. **友好性**：对用户的模糊查询给予合理推测和引导

### 特殊处理

1. **繁简转换**：数据集中唐诗多为繁体字，回答时应同时考虑繁简体匹配
2. **模糊匹配**：当精确匹配无结果时，尝试模糊匹配（部分标题、近似作者名等）
3. **推荐延伸**：查询完成后，可推荐相关的诗词作品或同主题作品
4. **生成与查询结合**：生成诗词时可引用数据集中的经典作品作为参考或对比
