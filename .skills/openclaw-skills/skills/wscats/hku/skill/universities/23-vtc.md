### Primary Data Sources (Official Only)

   - All programmes offered (check for postgraduate / master's level)

2. **THEi Postgraduate**: See `16-thei.md`
   - THEi is the main degree-granting arm of VTC

### Note

> ⚠️ VTC's master's-level programmes are primarily offered through THEi.
> Refer to `16-thei.md` for THEi postgraduate programmes.
> This file covers any additional master's programmes offered directly by VTC.
> If no additional master's programmes exist beyond THEi, note this in the output.
