# Yoga Skill

## Overview

This skill provides comprehensive yoga guidance including pose instructions, breathing techniques (pranayama), and meditation practices. It helps users build a personalized yoga routine for physical and mental wellness.

## Features

- **Pose Library**: Step-by-step instructions for 50+ yoga poses (asanas), from beginner to advanced levels.
- **Breathing Techniques**: Guided pranayama exercises including Ujjayi, Nadi Shodhana, and Kapalabhati.
- **Meditation Sessions**: Mindfulness and meditation routines ranging from 5 to 30 minutes.
- **Daily Routines**: Pre-built yoga sequences for morning energy, evening relaxation, and stress relief.
- **Progress Tracking**: Track your practice duration, pose mastery, and consistency streaks.

## Usage

```
/yoga pose <pose_name>       # Get instructions for a specific pose
/yoga breathe <technique>    # Start a guided breathing exercise
/yoga meditate <duration>    # Begin a meditation session
/yoga routine <type>         # Get a recommended daily routine
/yoga progress               # View your practice progress
```

## Examples

- `/yoga pose warrior-ii` — Learn the Warrior II pose with alignment tips.
- `/yoga breathe ujjayi` — Practice ocean breath for calm and focus.
- `/yoga routine morning` — Get a 15-minute morning yoga flow.
