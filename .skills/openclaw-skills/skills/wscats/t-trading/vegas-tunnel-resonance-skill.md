## 📐 第二维度：斐波那契回撤与扩展

### 2.1 斐波那契回撤位计算方法

```
确定最近一波明确的趋势运动（波段高点 H 与波段低点 L）：

上升趋势回撤位（自高点回落时的支撑位）:
  0.236 回撤位 = H - (H - L) × 0.236  → 浅度回撤，强势回调
  0.382 回撤位 = H - (H - L) × 0.382  → 正常回撤，第一支撑
  0.500 回撤位 = H - (H - L) × 0.500  → 中度回撤，关键位
  0.618 回撤位 = H - (H - L) × 0.618  → 黄金回撤，强支撑
  0.786 回撤位 = H - (H - L) × 0.786  → 深度回撤，最后防线

下降趋势回撤位（自低点反弹时的压力位）:
  0.236 回撤位 = L + (H - L) × 0.236  → 弱反弹压力
  0.382 回撤位 = L + (H - L) × 0.382  → 正常反弹压力
  0.500 回撤位 = L + (H - L) × 0.500  → 中度反弹压力
  0.618 回撤位 = L + (H - L) × 0.618  → 黄金反弹压力
  0.786 回撤位 = L + (H - L) × 0.786  → 强反弹压力
```

### 2.2 斐波那契与 EMA 通道的共振判定

```
【黄金共振 (Golden Resonance)】 → +20 分
  条件: 斐波那契 0.618 回撤位 与 EMA144/169 通道重合（误差 < 1%）
  含义: 极强支撑/压力，高概率反转点
  操作: 可重仓介入

【白银共振 (Silver Resonance)】 → +15 分

  含义: 较强支撑/压力

  含义: 一般支撑/压力

【无共振 (No Resonance)】 → +0 分
  条件: 斐波那契位与 EMA 通道无重合
  含义: 支撑/压力可靠性降低
  操作: 需要更多确认信号
```

```

维度三 → 对各周期逐一评分并加权汇总
维度四 → 合成信号，生成交易决策
```

### Step 3: 输出报告

```
使用上述"完整分析报告"模板输出

给出具体的价格位和操作建议
始终包含风险提示
```

## 📚 附录：EMA 计算参考

```
EMA 计算公式:
  EMA(today) = Price(today) × K + EMA(yesterday) × (1 - K)
  其中 K = 2 / (N + 1), N 为周期数

各 EMA 的平滑系数 K:
  EMA12:  K = 2/13  ≈ 0.1538
  EMA13:  K = 2/14  ≈ 0.1429
  EMA144: K = 2/145 ≈ 0.0138
  EMA169: K = 2/170 ≈ 0.0118
  EMA576: K = 2/577 ≈ 0.0035
  EMA676: K = 2/677 ≈ 0.0030

维加斯通道 EMA 周期来源:
  12, 13     → 短期动量（斐波那契相邻数）
  144, 169   → 中期趋势（12², 13²）
  576, 676   → 长期趋势（144×4, 169×4）
```
