# Blast Article Anatomy

Use this file when a draft feels too flat or too much like ordinary web fiction.

## Core principle
A LOFTER blast-style article is packaged around click-through and emotional payoff density, not slow-burn prestige prose.

## Reader reward ladder
1. Reader identifies the CP/IP immediately.
2. Reader sees a visible abnormal event or high-emotion promise.
3. Reader receives a hook within one screen.
4. Reader gets a new payoff every 1-3 paragraphs.
5. Reader leaves with one quote-point, screenshot line, or next-click impulse.

## Preface function
Keep it short. Use it to:
- mark `ooc`, `if`, `私设`, `慎入`
- mark key pairings or direction
- signal the article's emotional center
- optionally invite red-heart/blue-hand interaction

## Opening function
The opening should do one of the following immediately:
- reveal the abnormal condition
- place the pair in a familiar scene with one disruptive twist
- create a public misunderstanding
- start the argument before the explanation

## Mid-section function
The middle should not become travel notes or pure exposition. Add one of:
- stronger misunderstanding
- public reaction
- jealousy
- overprotection
- old feelings leaking out
- one exposed preference that feels like hidden love
- a direct emotional question that cannot be dodged

## Ending function
End with one of:
- relationship confirmation
- public/private line that seals the pairing
- stronger unresolved hook for the next numbered chapter
- one line that can be screenshotted and quoted in comments

## Common failure modes
- pretty but vague title
- preface too long
- three paragraphs of setup before the first hook
- middle section becomes repetitive banter without escalation
- ending lands softly without sealing the article's promise

## Recovery moves
- Make the title one level more specific.
- Move the first embarrassment or conflict into paragraph one.
- Cut repeated banter and replace it with a reveal or stronger reaction.
- End chapters on a question, kiss, confrontation, or social-chaos beat.
