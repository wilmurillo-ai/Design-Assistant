# End-to-End Examples

## Example 1: Product + Engineering Team

### User intent
"I want to create a product-engineering team to ship a new SaaS feature in 6 weeks."

### Proposed roles
Core:
- product-manager
- tech-architect
- backend-engineer
- frontend-engineer
- qa-engineer

Optional:
- ux-designer
- devops-engineer
- data-analyst

Assumptions:
- Scope includes strategy + execution.
- Team operates in one primary channel and one sync channel.

### Output sample
- PM owns PRD and milestone plan.
- Architect owns service boundaries + API contracts.
- Backend/Frontend own implementation deliverables.
- QA owns test plan and release readiness report.
- PM consolidates final status to user.

### Smoke test prompt
"Design a minimal feature plan, split tasks by role, identify blockers, and return a release-readiness summary."

## Example 2: Growth + Marketing Team

### User intent
"Build a growth team for acquisition and conversion optimization."

### Proposed roles
Core:
- growth-lead
- content-strategist
- campaign-operator
- performance-analyst

Optional:
- seo-specialist
- brand-designer
- martech-engineer

Assumptions:
- Weekly experiment cadence.
- KPI focus: CAC, CVR, retention proxy metrics.

### Output sample
- Growth lead defines experiment roadmap.
- Content strategist produces campaign narratives.
- Campaign operator executes distribution and channel ops.
- Performance analyst reports metrics and recommends next iteration.

### Smoke test prompt
"Propose a 2-week experiment plan with hypotheses, channel actions, expected KPI movement, and weekly reporting format."
