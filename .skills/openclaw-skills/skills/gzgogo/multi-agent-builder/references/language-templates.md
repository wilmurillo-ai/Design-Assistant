# Language Templates

Mirror user's language. If mixed language, follow the dominant language.

## English kickoff template
"Great — I can help design and bootstrap this team. To make the setup accurate, I need 4 quick inputs: (1) target outcome, (2) scope depth, (3) hard constraints, (4) operating channel(s). If you want, I can also start with a default role proposal now."

## Chinese kickoff template
"好，我可以帮你把这个团队设计并落地。为了保证配置准确，我先确认 4 个信息：1）目标结果，2）工作范围深度，3）硬约束（时间/预算/合规/技术栈），4）运行渠道。你也可以让我先给一版默认角色方案。"

## English proposal template
"Based on your goal, here's a proposed team:\nCore: ...\nOptional: ...\nAssumptions: ...\nPlease confirm additions/removals before I proceed to creation setup."

## Chinese proposal template
"基于你的目标，我建议团队如下：\n核心角色：...\n可选角色：...\n默认假设：...\n请先确认增删，我再进入创建流程配置。"

## English confirmation template
"Please confirm: (1) final role list, (2) lead role, (3) channel mapping per role, (4) proceed with creation now (yes/no)."

## Chinese confirmation template
"请确认：1）最终角色清单，2）牵头角色，3）每个角色的渠道映射，4）是否现在执行创建（是/否）。"
