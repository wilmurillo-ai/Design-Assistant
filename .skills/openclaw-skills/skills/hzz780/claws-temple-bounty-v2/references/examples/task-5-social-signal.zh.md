# Task 5 示例（中文）

> 这是一个双层展开示例，用来展示完整输出结构；plain chat 默认不应主动展开维护者详情。

## 普通用户摘要

你现在在 `Task 5：社交寻配`。

这一项是可选加分项，不会影响主线资格。

你可以发出一条简短信号，把你的坐标、部落方向和想找的伙伴类型告诉更多人。

参考文案：

```text
我刚完成龙虾圣殿的原力坐标测绘，正在寻找能和我的 Agent 形成互补的伙伴。
如果你也在走这条路径，欢迎来和我对一下坐标。
```

如果你愿意，我也可以按 `TG / X / 奇物志` 三个平台分别帮你改成更适合发布的版本。

如果你现在就想发到 `TG` 或 `X`，我会先帮你起草内容；如果当前宿主具备对应权限和能力，我再继续带你直接发送。若当前宿主没有这项能力，最后一步会由你手动点发送。

如果你已经确定平台，我再把对应版本直接改给你。
如果你只是想先看入口，我也可以再把对应链接贴给你。
如果你现在就在 `OpenClaw` 里，而且已经确定要发到 `TG` 或 `X`，只要当前宿主真的具备对应权限和能力，我也可以直接带你走浏览器操作。

### 分平台模板

- `TG`：用 2 段短文，语气更像邀人对话，结尾用“欢迎来认领我”这类 CTA。
- `X`：保持单条短帖，信息要压缩，只保留坐标、方向、CTA。
- `奇物志`：可以多写一句你想找什么样的伙伴，语气更像公开招募。
- `OpenClaw`：如果你已经确定发 `TG` 或 `X`，而且想现在就发，只要当前宿主真的具备对应权限和能力，我可以继续带你走浏览器操作。
