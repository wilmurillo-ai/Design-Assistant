# Seers Markets - 超短期权快速交易

## 交易机制

Seers 提供超短期权快速交易功能，用户可以预测加密货币短期价格走势（看涨/看跌）。

### 交易流程

1. 选择交易对（如 BTC/USDT、ETH/USDT）
2. 选择方向：看涨（买涨）或看跌（买跌）
3. 输入下注金额（最低 0.1 USDT，支持两位小数）
4. 系统根据实时赔率计算潜在收益
5. 确认下单
6. 等待结算（5秒结算周期）

### 胜负判定

- 买涨：最终结算价大于开仓成交价时，用户赢
- 买跌：最终结算价小于开仓成交价时，用户赢
- 赢了获得下注金额乘以赔率的奖励
- 输了则损失下注金额

### 结算时间

- 开仓时间精确到毫秒（例：12:23:45.456）
- 结算时间为整秒结算（例：12:23:48.000）

## 赔率说明

### 动态赔率

赔率会根据全站用户的下注情况实时变动：
- 基础赔率约 1.9 倍
- 最高赔率不超过 2.3 倍
- 最低赔率不低于 1.35 倍

### 赔率变动规则

- 买涨的人越多、资金越大，看涨赔率越低
- 买跌的人越多、资金越大，看跌赔率越低
- 资金少的方向赔率会自动升高，吸引用户对冲
- 赔率通过 WebSocket 实时推送，前端实时显示

## 价格来源

### 三家交易所均价

平台价格不依赖单一交易所，采用三家主流交易所的均价：
- Binance（币安）
- OKX（欧易）
- Gate.io（芝麻开门）

计算方式：平台实时价 = (币安价 + OKX价 + Gate价) / 3

### 价格精度

所有交易对价格统一保留 4 位小数（例如：68500.1234）

### 异常处理

- 3 家正常：三家均价
- 1 家掉线：剩余 2 家均价
- 2 家掉线：使用仅剩的 1 家价格
- 全部中断超过 3 秒：触发熔断，前端显示"数据恢复中"，暂停交易

## 熔断机制

当价格源中断超过 3 秒：
- 系统冻结下单按钮，停止接收新订单
- 在途订单按最后获取到的价格结算
- 如果结算时无法取价，本金 100% 原路退回用户
