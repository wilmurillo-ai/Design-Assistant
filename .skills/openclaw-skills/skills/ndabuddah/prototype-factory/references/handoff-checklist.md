# Prototype Handoff Checklist

1. **README.md** – Explain stack, commands, credentials, demo data
2. **Assets** – Compress hero images, include licenses/attribution links
3. **Env vars** – Provide `.env.example` with dummy keys
4. **Scripts** – Add `setup.sh` or `npm run dev` instructions
5. **Video/GIF** – Record 30–60s Loom showcasing core flow
6. **Issue log** – Track known limitations / next steps
7. **Cost note** – Document external API usage or paid fonts
