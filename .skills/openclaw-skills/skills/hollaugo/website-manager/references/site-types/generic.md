# Site Type: Generic

Use for broad business websites, creators, local organizations, nonprofits, mixed-content sites, landing-page-driven brands, and any website that does not clearly fit a narrower vertical type.

## Priorities

- clear positioning
- flexible layout system
- reusable cards and listings
- broad support for pages, collections, and blog content
- simple conversion paths
