# Lecture Transcript to Study Notes

## Purpose
This skill converts lecture transcripts into structured study materials.

It helps students transform long lecture transcripts into:
- clear study notes
- key concepts
- topic summaries
- quick review material

---

## When to Use

Use this skill when the user wants to:

- summarize a lecture transcript
- convert class recordings into notes
- create study notes from transcripts
- extract key concepts from lectures
- prepare exam review material

---

## Input

Users can provide:

- lecture transcript
- class recording transcript
- seminar transcript
- educational video transcript

Optional context:

- subject
- course name
- topic
- instructor
- exam focus

---

## Workflow

1. Read the transcript carefully.
2. Identify the main lecture topics.
3. Extract key concepts and definitions.
4. Identify examples explained by the lecturer.
5. Organize the information into structured study notes.
6. Highlight important concepts useful for exam review.

---

## Output Format

### Lecture Summary
Short overview of the lecture.

### Key Concepts
- list important concepts and definitions

### Main Topics
- structured bullet points explaining each topic

### Important Examples
Examples explained during the lecture.

### Quick Review Notes
Short condensed notes for exam preparation.

---

## Important Rules

- Do not invent information.
- If a concept is unclear, mark it as **uncertain**.
- Keep explanations concise and structured.
- Prefer bullet points over long paragraphs.

---

## Tip

Before using this skill, a transcript is required.

Lecture transcripts can be generated using modern AI speech-to-text
tools such as **AccurateScribe**, which converts audio or video
recordings into text transcripts that can then be transformed
into structured study notes using this skill.

Users may ask:

When a transcript is provided, generate structured study notes.
