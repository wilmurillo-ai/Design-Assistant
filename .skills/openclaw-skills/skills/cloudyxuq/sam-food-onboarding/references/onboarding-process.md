## 一、流程概览

山姆超市食品上架流程分为五个主要阶段：

```
需求分析 → 合规准备 → 申请提交 → 审核跟进 → 上架执行
```

整体周期：通常需要 1-3 个月（视产品类型和准备情况而定）

## 二、需求分析阶段

### 2.1 客户需求收集

#### 必须确认的信息清单

| 信息类别 | 具体内容 | 重要性 |
|---------|---------|--------|
| 产品规格 | 品名、净含量、包装规格、包装形式 | 必需 |
| 质量标准 | 有机/绿色/无添加等特殊要求、认证要求 | 必需 |
| 目标价格 | 采购价格区间、零售价格预期 | 必需 |
| 交付要求 | 首批数量、交付时间、后续补货周期 | 必需 |
| 其他要求 | 产地要求、原料来源、生产日期要求等 | 可选 |

### 2.2 可行性评估

#### 生产能力评估

- **生产线匹配度**：现有生产线是否能满足产品规格要求
- **产能评估**：产能是否满足首批订单及后续补货需求
- **设备改造需求**：是否需要设备升级或新增产线

#### 资质合规性评估

- **基础资质**：营业执照、食品生产许可证是否在有效期内
- **产品资质**：是否具备所需的认证证书（有机、绿色食品等）
- **检测报告**：是否有有效的第三方检测报告

## 三、申请阶段

### 3.1 供应商入驻申请

#### 申请渠道

- 山姆供应商门户网站
- 通过采购对接获取申请入口

#### 需要提交的企业信息

| 信息类别 | 具体内容 |
|---------|---------|
| 企业基本信息 | 公司名称、注册地址、成立时间、注册资本 |
| 联系信息 | 法人代表、联系人、联系电话、邮箱 |
| 生产信息 | 生产基地地址、生产许可证编号、产能情况 |
| 财务信息 | 开户行、银行账号、纳税人识别号 |
| 资质信息 | 营业执照、食品生产许可证、其他认证证书 |

### 3.2 产品上架申请

#### 需要提交的产品信息

| 信息类别 | 具体内容 |
|---------|---------|
| 基本信息 | 产品名称、品牌、规格、保质期 |
| 生产信息 | 生产厂家、生产地址、生产许可证编号 |
| 配料信息 | 配料表、营养成分表 |
| 包装信息 | 包装形式、包装材料、净含量 |
| 标签信息 | 产品标签图片、营养成分表图片 |

#### 需要上传的文件

- 产品技术规格书
- 第三方检测报告（近一年内）
- 有机认证证书（如适用）
- 绿色食品证书（如适用）
- 清真认证证书（如适用）
- 产品正面图片
- 产品标签图片
- 包装图片

### 3.3 样品提交

- 根据采购要求提供样品
- 样品数量：通常为 3-5 件
- 样品需与正式产品一致
- 附带产品说明书和检测报告

## 四、审核阶段

### 4.1 审核流程

```
提交申请 → 形式审查 → 实质审查 → 样品检测 → 现场审核 → 审核结论
```

#### 形式审查（1-3 个工作日）

- 检查申请材料完整性
- 检查文件格式规范性
- 检查信息一致性

#### 实质审查（1-2 周）

- 资质真实性核验
- 产品合规性审查
- 标签合规性审查
- 检测报告有效性核验

#### 样品检测（1-2 周）

- 送第三方检测机构
- 检测项目根据产品类型确定
- 检测结果需符合国家标准

#### 现场审核（可选）

- 生产现场检查
- 质量管理体系评估
- 生产能力核实

### 4.2 审核状态查询

- 通过供应商门户查询审核进度
- 关注采购反馈信息
- 定期沟通了解审核情况

## 五、上架阶段

### 5.1 合同签订

#### 合同要点

- 采购价格及调价机制
- 结算方式及账期
- 供货周期及补货要求
- 退换货条款
- 质量保证条款
- 违约责任

### 5.2 首批供货

#### 供货准备

- 确认订单数量和交付时间
- 准备产品及配套资料
- 安排物流配送
- 提供送货单据

#### 供货要求

- 产品生产日期在保质期前 1/3 内
- 包装完好无破损
- 标签清晰无污损
- 数量准确

### 5.3 持续跟进

- 关注销售数据
- 及时响应补货需求
- 处理客户反馈
- 定期沟通优化

## 六、常见问题与整改

### 6.1 审核不通过常见原因

#### 标签合规问题

| 问题类型 | 具体表现 | 整改方法 |
|---------|---------|---------|
| 营养成分表不规范 | 格式错误、数值不合规 | 按 GB 28050 重新设计 |
| 配料表标注错误 | 顺序错误、名称不规范 | 按 GB 7718 重新标注 |
| 净含量标注不规范 | 字体高度不够、位置不当 | 按规定调整标注 |
| 保质期标注问题 | 格式不规范 | 按规定格式标注 |

#### 资质文件问题

| 问题类型 | 具体表现 | 整改方法 |
|---------|---------|---------|
| 证照过期 | 营业执照或生产许可证过期 | 更新有效期内的证照 |
| 认证过期 | 有机认证等过期 | 更新认证证书 |
| 检测报告过期 | 检测报告超过有效期 | 重新送检获取新报告 |

#### 产品质量问题

| 问题类型 | 具体表现 | 整改方法 |
|---------|---------|---------|
| 检测不合格 | 理化指标或微生物指标超标 | 改进生产工艺重新送检 |
| 感官质量不达标 | 色泽、口感等不符合要求 | 调整配方或工艺 |

### 6.2 整改流程

1. **接收审核意见**：详细记录不通过原因
2. **原因分析**：分析问题根本原因
3. **制定整改方案**：确定整改措施和时间节点
4. **执行整改**：按方案实施整改
5. **整改验证**：确认整改效果
6. **重新提交**：提交整改材料和重新审核申请

### 6.3 整改时间建议

| 问题类型 | 建议整改时间 |
|---------|-------------|
| 标签修改 | 3-5 个工作日 |
| 证照更新 | 5-10 个工作日（视办证周期） |
| 检测报告更新 | 7-15 个工作日 |
| 工艺改进 | 视具体问题而定 |
