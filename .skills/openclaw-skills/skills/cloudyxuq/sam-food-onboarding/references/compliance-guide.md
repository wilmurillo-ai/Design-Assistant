## 一、基础资质要求

### 1.1 企业资质

#### 必需证照

| 证照名称 | 发证部门 | 有效期 | 注意事项 |
|---------|---------|--------|---------|
| 营业执照 | 市场监管部门 | 长期 | 经营范围需包含食品生产或销售 |
| 食品生产许可证 | 市场监管部门 | 5年 | 需覆盖申请产品类别 |
| 组织机构代码证 | 市场监管部门 | - | 已与营业执照三证合一 |

#### 证照要求

- 所有证照需在有效期内
- 证照信息需与申请信息一致
- 需提供原件扫描件（清晰、完整）
- 部分情况需要提供复印件加盖公章

### 1.2 生产许可细则

#### 食品生产许可证分类

| 产品类别 | 许可证类别 | 备注 |
|---------|-----------|------|
| 糕点、饼干 | 糕点 | 需包含相应品种 |
| 坚果、炒货 | 炒货食品及坚果制品 | - |
| 肉制品 | 肉制品 | 需注明具体品种 |
| 乳制品 | 乳制品 | 需注明具体品种 |
| 饮料 | 饮料 | 需注明具体品种 |
| 调味品 | 调味品 | 需注明具体品种 |

#### 生产许可证审查要点

- 许可范围是否包含申请产品
- 生产地址是否与实际一致
- 有效期是否充足（建议剩余有效期超过6个月）

## 二、产品资质要求

### 2.1 检测报告

#### 检测报告要求

| 要求项 | 具体标准 |
|-------|---------|
| 检测机构 | 具备CMA或CNAS资质的第三方检测机构 |
| 检测时间 | 近一年内（部分产品要求更短） |
| 检测项目 | 根据产品类别确定，需覆盖国家标准要求 |
| 样品信息 | 需与申请产品信息一致 |

#### 常见产品检测项目

**糕点类**
- 理化指标：水分、脂肪、蛋白质
- 微生物：菌落总数、大肠菌群、霉菌、沙门氏菌等
- 添加剂：防腐剂、甜味剂、色素等

**坚果炒货类**
- 理化指标：酸价、过氧化值
- 微生物：菌落总数、大肠菌群、霉菌
- 污染物：黄曲霉毒素

**肉制品**
- 理化指标：蛋白质、脂肪、水分、盐分
- 微生物：菌落总数、大肠菌群、致病菌
- 添加剂：亚硝酸盐、防腐剂等

**乳制品**
- 理化指标：蛋白质、脂肪、非脂乳固体
- 微生物：菌落总数、大肠菌群、致病菌
- 污染物：重金属、农残等

### 2.2 型式检验报告

- 需提供全项目检测报告
- 检测依据为国家标准或行业标准
- 检测结果需全部合格

## 三、标签合规要求

### 3.1 强制标示内容

根据 GB 7718《预包装食品标签通则》，预包装食品标签必须标示：

| 标示项目 | 具体要求 |
|---------|---------|
| 食品名称 | 反映食品真实属性 |
| 配料表 | 按递减顺序排列，使用规范名称 |
| 净含量和规格 | 使用法定计量单位 |
| 生产者信息 | 名称、地址、联系方式 |
| 生产日期 | 年月日格式 |
| 保质期 | 明确保质期限 |
| 贮存条件 | 符合产品特性的贮存要求 |
| 食品生产许可证编号 | SC 开头的编号 |
| 产品标准代号 | 执行的标准编号 |

### 3.2 营养成分表要求

根据 GB 28050《预包装食品营养标签通则》：

#### 必须标示的营养成分

| 营养成分 | 计量单位 | 标示要求 |
|---------|---------|---------|
| 能量 | 千焦(kJ) | 必须 |
| 蛋白质 | 克(g) | 必须 |
| 脂肪 | 克(g) | 必须 |
| 碳水化合物 | 克(g) | 必须 |
| 钠 | 毫克(mg) | 必须 |

#### 营养成分表格式

```
营养成分表
─────────────────────────────
  项目     每100克   NRV%
─────────────────────────────
 能量      XX kJ     XX%
 蛋白质    XX g      XX%
 脂肪      XX g      XX%
碳水化合物  XX g      XX%
 钠        XX mg     XX%
─────────────────────────────
```

#### 营养成分表注意事项

- 数值需为实际检测值或计算值
- 需标注每 100g（或每份）的含量
- 需标注 NRV%（营养素参考值百分比）
- 字体大小需符合规定（最小高度 1.0mm）

### 3.3 配料表标注要求

#### 标注原则

- 按递减顺序排列（按质量或体积）
- 使用规范名称
- 复合配料需展开标注
- 添加剂使用 INS 号或标准名称

#### 特殊配料标注

- **过敏原**：需明确标注（如含有麸质、乳、蛋、坚果等）
- **转基因**：如使用转基因原料需标注
- **辐照**：如使用辐照原料需标注

## 四、特殊认证要求

### 4.1 有机产品认证

#### 认证要求

| 要求项 | 具体内容 |
|-------|---------|
| 认证机构 | 经国家认监委批准的有机认证机构 |
| 认证范围 | 原料、生产过程、加工均需符合有机标准 |
| 证书有效期 | 1年 |
| 标识使用 | 需在产品上使用有机认证标识 |

#### 有机认证申请材料

- 有机产品认证申请书
- 营业执照、食品生产许可证
- 有机生产/加工计划
- 产地环境检测报告
- 有机管理体系文件

#### 有机产品标签要求

- 使用中国有机产品认证标志
- 标注认证机构名称
- 标注有机码（可追溯）

### 4.2 绿色食品认证

#### 认证要求

| 要求项 | 具体内容 |
|-------|---------|
| 认证机构 | 中国绿色食品发展中心 |
| 认证等级 | AA 级、A 级 |
| 证书有效期 | 3年 |
| 标识使用 | 需使用绿色食品标志 |

#### 绿色食品申请材料

- 绿色食品标志使用申请书
- 营业执照、食品生产许可证
- 产品执行标准
- 原料来源证明
- 质量保证体系文件

### 4.3 清真认证

#### 认证要求

- 原料需符合清真要求
- 生产设备专用或经过清真清洗
- 生产过程符合清真规范
- 需获得清真认证机构认证

#### 清真认证机构

- 中国伊斯兰教协会
- 各省市伊斯兰教协会
- 国际认可的清真认证机构

### 4.4 其他认证

| 认证类型 | 认证机构 | 适用范围 |
|---------|---------|---------|
| HACCP认证 | 第三方认证机构 | 食品安全管理体系 |
| ISO22000认证 | 第三方认证机构 | 食品安全管理体系 |
| 无公害农产品认证 | 农业部门 | 农产品 |

## 五、进口食品要求

### 5.1 进口资质

#### 进口商资质

- 进出口经营权
- 食品经营许可证
- 进口食品收货人备案

#### 进口产品要求

| 要求项 | 具体内容 |
|-------|---------|
| 原产地证明 | 出口国官方出具 |
| 卫生证书 | 出口国官方出具 |
| 检测报告 | 符合中国国家标准 |
| 中文标签 | 符合 GB 7718 要求 |

### 5.2 进口食品标签

#### 中文标签要求

- 必须粘贴中文标签
- 标签内容符合 GB 7718 要求
- 标注原产国及国内进口商信息
- 标注国内进口商联系方式

#### 中文标签内容

```
食品名称：XXX
原产国：XXX
进口商：XXX公司
进口商地址：XXX
进口商联系方式：XXX
生产日期：XXX
保质期：XXX
贮存条件：XXX
净含量：XXX
配料表：XXX
营养成分表：XXX
```

### 5.3 进口食品检测

- 入境检验检疫
- 中文标签审核
- 安全卫生项目检测
- 检测合格后方可销售

## 六、常见合规问题

### 6.1 标签问题

| 问题类型 | 具体表现 | 合规要求 |
|---------|---------|---------|
| 配料表顺序错误 | 未按递减顺序 | 按 GB 7718 重新标注 |
| 营养成分表错误 | 数值不准确、格式不规范 | 按 GB 28050 重新设计 |
| 净含量标注不规范 | 字体高度不够 | 最小字体高度符合规定 |
| 日期标注不规范 | 格式不统一 | 按"年月日"格式标注 |

### 6.2 认证问题

| 问题类型 | 具体表现 | 解决方法 |
|---------|---------|---------|
| 认证过期 | 证书超过有效期 | 重新申请认证 |
| 认证范围不符 | 产品超出认证范围 | 扩展认证范围或更换产品 |
| 标识使用不规范 | 未使用或错误使用标识 | 按规定使用认证标识 |

### 6.3 检测问题

| 问题类型 | 具体表现 | 解决方法 |
|---------|---------|---------|
| 检测报告过期 | 超过有效期 | 重新送检 |
| 检测项目不全 | 未覆盖国家标准要求 | 补充检测项目 |
| 检测机构无资质 | 不具备 CMA/CNAS 资质 | 选择有资质的检测机构 |
