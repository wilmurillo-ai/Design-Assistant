## 概述

本指南为食品包装材料选择提供系统性参考，涵盖常用材料的特性、适用范围、优缺点及选型建议。

## 材料分类体系

### 包装材料分类

```
食品包装材料
├── 塑料类
│   ├── 热塑性塑料 (PE/PP/PET/PS/PA)
│   └── 热固性塑料
├── 金属类
│   ├── 铝合金 (铝箔/铝罐)
│   └── 钢材 (马口铁/无锡钢)
├── 玻璃类
│   ├── 钠钙玻璃
│   ├── 硼硅玻璃
│   └── 铅晶质玻璃
├── 纸类
│   ├── 纸板
│   ├── 特种纸
│   └── 纸塑复合材料
└── 复合材料
    ├── 纸塑复合
    ├── 铝塑复合
    └── 多层共挤
```

## 塑料材料选择

### 1. 聚乙烯(PE)

#### 特性

| 特性 | HDPE | LDPE | LLDPE |
|------|------|------|-------|
| 密度 | 0.94-0.96 | 0.91-0.94 | 0.92-0.94 |
| 透明度 | 半透明 | 半透明 | 半透明 |
| 刚性 | 高 | 低 | 中 |
| 柔韧性 | 一般 | 好 | 好 |
| 耐温 | -50~110℃ | -50~80℃ | -50~90℃ |
| 阻隔性 | 中 | 低 | 中 |

#### 典型应用

| 类型 | 应用场景 | 包装形式 |
|------|----------|----------|
| HDPE | 奶制品/饮料瓶 | 瓶/桶 |
| LDPE | 保鲜膜/包装袋 | 薄膜/袋 |
| LLDPE | 复合袋内层 | 复合膜 |

#### 选型建议

- 优点：成本低、无毒、耐化学性好
- 缺点：阻隔性一般、耐温有限
- 适用：一般食品包装、冷冻食品
- 不适用：高温杀菌食品、油性食品长期存储

### 2. 聚丙烯(PP)

#### 特性

| 特性 | 说明 |
|------|------|
| 密度 | 0.90-0.91 |
| 透明度 | 较好(结晶改性的) |
| 刚性 | 高 |
| 耐温 | -20~130℃ |
| 阻隔性 | 中等 |
| 刚性 | 适合蒸煮 |

#### 典型应用

| 应用场景 | 包装形式 | 特点 |
|----------|----------|------|
| 微波食品 | 托盘/盒子 | 可蒸煮、微波 |
| 饮料瓶 | 瓶身/瓶盖 | 轻量、透明 |
| 酸奶杯 | 杯体/盖膜 | 耐温、印刷性好 |

#### 选型建议

- 优点：耐高温、可蒸煮、透明
- 缺点：低温脆性、阻氧性一般
- 适用：蒸煮包装、微波食品
- 不适用：冷冻食品(-20℃以下)

### 3. 聚酯(PET)

#### 特性

| 特性 | 说明 |
|------|------|
| 密度 | 1.34 |
| 透明度 | 优异 |
| 刚性 | 高 |
| 耐温 | -40~70℃(非耐温) / 100℃(耐温型) |
| 阻隔性 | 优异(对O₂/CO₂) |
| 强度 | 高 |

#### 典型应用

| 应用场景 | 包装形式 | 特点 |
|----------|----------|------|
| 碳酸饮料 | PET瓶 | 阻CO₂、透明 |
| 食用油 | PET瓶 | 阻氧、货架期长 |
| 药品包装 | 吸塑/泡罩 | 阻隔性、刚性 |

#### 选型建议

- 优点：阻隔性好、透明、强度高
- 缺点：不耐高温(普通型)
- 适用：碳酸饮料、油品、干燥食品
- 不适用：高温蒸煮、热灌装

### 4. 聚苯乙烯(PS)

#### 特性

| 特性 | 说明 |
|------|------|
| 密度 | 1.05 |
| 透明度 | 优异 |
| 刚性 | 高 |
| 耐温 | -20~70℃ |
| 加工性 | 优良(发泡) |

#### 典型应用

| 应用场景 | 包装形式 | 说明 |
|----------|----------|------|
| 快餐包装 | 发泡托盘 | 保温、成本低 |
| 酸奶盖 | PS片材 | 易揭、阻湿 |
| 蛋盒 | 发泡PS | 缓冲、保护 |

#### 选型建议

- 优点：成本低、透明度好、易加工
- 缺点：脆性、阻隔性差、不耐油
- 适用：一次性包装、干燥食品
- 不适用：油脂食品、需高阻隔产品

### 5. 尼龙(PA)

#### 特性

| 特性 | 说明 |
|------|------|
| 阻氧性 | 优异 |
| 耐温 | -60~200℃ |
| 强度 | 高 |
| 阻香性 | 好 |

#### 典型应用

| 应用场景 | 包装形式 | 说明 |
|----------|----------|------|
| 肉类真空包装 | 复合膜 | 高阻隔 |
| 高温蒸煮袋 | 多层共挤 | 耐蒸煮 |
| 咖啡包装 | 复合膜 | 阻氧、香气保留 |

#### 选型建议

- 优点：阻隔性优异、耐高温
- 缺点：成本高、吸潮
- 适用：高阻隔需求、高温杀菌
- 不适用：纯PA单层包装(因吸潮)

## 金属材料选择

### 1. 铝合金

#### 特性

| 特性 | 说明 |
|------|------|
| 阻隔性 | 极佳(完全阻隔) |
| 重量 | 轻 |
| 导热性 | 好 |
| 加工性 | 好(可冲压/拉伸) |
| 回收性 | 100%可回收 |

#### 典型应用

| 应用场景 | 罐型 | 说明 |
|----------|------|------|
| 饮料罐 | 两片罐/三片罐 | 轻量、密封 |
| 气雾罐 | 气雾罐 | 耐压、密封 |
| 软管 | 铝管 | 保护性高 |

#### 选型建议

- 优点：阻隔性极佳、轻量、可回收
- 缺点：成本较高、不能微波
- 适用：饮料、气雾产品、高保护需求

### 2. 马口铁

#### 特性

| 特性 | 说明 |
|------|------|
| 基材 | 低碳钢 |
| 镀层 | 锡层 |
| 阻隔性 | 优异 |
| 耐压性 | 好 |
| 刚性 | 高 |

#### 典型应用

| 应用场景 | 罐型 | 说明 |
|----------|------|------|
| 食品罐头 | 三片罐 | 密封、耐压 |
| 饮料罐 | 三片罐 | 成本低 |
| 茶叶罐 | 异型罐 | 密封、遮光 |

#### 选型建议

- 优点：阻隔性好、耐压、成本适中
- 缺点：重量较重、有镀锡层限制
- 适用：罐头食品、需要高温杀菌

## 材料对比分析

### 综合性能对比

| 材料 | 阻氧 | 阻湿 | 强度 | 耐温 | 成本 | 环保 |
|------|------|------|------|------|------|------|
| PE | 中 | 中 | 中 | 低 | 低 | 中 |
| PP | 中 | 中 | 高 | 高 | 中 | 中 |
| PET | 高 | 高 | 高 | 中 | 中 | 中 |
| PA | 高 | 中 | 高 | 高 | 高 | 低 |
| 铝 | 极佳 | 极佳 | 高 | 高 | 高 | 好 |
| 玻璃 | 高 | 高 | 高 | 高 | 中 | 低 |

### 成本层级

```
成本从高到低：
铝 > 玻璃 > PA > PET > PP > PE/纸
```

### 选型决策树

```
开始
  │
  ├─ 需高温蒸煮?
  │    └─ 是 → 选择PP/PA/RCPP结构
  │
  ├─ 需要高阻隔?
  │    ├─ 是 → 铝箔复合 or PA/EVOH
  │    └─ 否 → PET/PE即可
  │
  ├─ 成本优先?
  │    ├─ 是 → PE/LDPE单层 or 纸塑
  │    └─ 否 → 按性能需求选材
  │
  └─ 环保要求?
       ├─ 是 → 纸/单一材质/可回收
       └─ 否 → 按性能需求选材
```

## 常见问题

| 问题 | 解答 |
|------|------|
| 油脂食品选什么? | PET/PA复合,避免PS |
| 冷冻食品选什么? | PE共挤膜,抗低温脆裂 |
| 微波食品选什么? | PP托盘,无金属复合 |
| 碳酸饮料选什么? | PET,耐CO₂渗透 |
| 高温杀菌选什么? | 马口铁罐/铝罐/耐高温蒸煮袋 |
