## 概述

本清单用于规范食品包装法规合规性检查，确保包装设计符合国家相关法律法规要求。

## 法规标准体系

### 核心法规

| 法规名称 | 编号 | 适用范围 |
|----------|------|----------|
| 食品安全法 | 主席令第21号 | 食品包装基本要求 |
| 食品生产许可管理办法 | 市场监管总局令 | 生产许可相关 |
| 食品标识管理规定 | 总局令 | 标签标识 |

### 包装材料标准

| 标准类型 | 标准编号 | 适用范围 |
|----------|----------|----------|
| 塑料材料 | GB 4806.7 | 食品接触用塑料树脂 |
| 金属材料 | GB 4806.9 | 食品接触用金属材料 |
| 纸制品 | GB 4806.8 | 食品接触用纸和纸板 |
| 玻璃制品 | GB 4806.5 | 食品接触用玻璃 |
| 涂层材料 | GB 4806.10 | 食品接触用涂料 |
| 橡胶材料 | GB 4806.11 | 食品接触用橡胶 |

## 材料合规检查

### 1. 塑料包装检查

#### 树脂品种确认

| 检查项 | 要求 | 检查依据 |
|--------|------|----------|
| 树脂品种 | 在GB 4806.6附录或公告名单内 | 官方公告 |
| 添加剂量 | 符合GB 9685规定 | 添加剂使用标准 |
| 迁移限量 | 总迁移量≤10mg/dm² | GB 31604.8 |

#### 常见问题

| 问题类型 | 风险描述 | 处理建议 |
|----------|----------|----------|
| 未备案树脂 | 使用未经审批的新材料 | 更换为已备案材料 |
| 超范围使用 | 使用了不允许的树脂品种 | 确认适用范围 |
| 添加剂超标 | 添加剂量超过限量 | 调整配方 |

### 2. 金属包装检查

| 检查项 | 要求 | 检测方法 |
|--------|------|----------|
| 基材 | 铝合金/马口铁 | 材质证明 |
| 镀层 | 符合GB/T 2520 | 厚度检测 |
| 内涂 | 食品级涂料 | 迁移量检测 |
| 铬溶出量 | ≤0.05mg/kg | GB 31604.7 |

### 3. 纸包装检查

| 检查项 | 要求 | 检测方法 |
|--------|------|----------|
| 荧光物质 | 阴性(254nm/365nm) | GB 31604.47 |
| 重金属 | 符合GB 4806.8限值 | GB 31604.9 |
| 微生物 | 符合相应要求 | GB 4789.18 |
| 甲醛 | ≤5mg/kg | SN/T 2333 |

### 4. 玻璃包装检查

| 检查项 | 要求 | 检测方法 |
|--------|------|----------|
| 铅溶出 | ≤0.5mg/L(普通玻璃) | GB/T 5009.62 |
| 镉溶出 | ≤0.25mg/L | GB/T 5009.62 |
| 砷溶出 | ≤0.05mg/L | GB/T 5009.62 |

## 标签标识检查

### 1. 强制标识内容

| 标识项目 | 位置 | 字高要求 | 检查要点 |
|----------|------|----------|----------|
| 产品名称 | 显著位置 | ≥最小字高 | 反映真实属性 |
| 配料表 | 配料标题下 | ≥1.8mm | 完整列出 |
| 净含量 | 显著位置 | 按规格要求 | 单位正确 |
| 生产日期 | 明确位置 | ≥3mm | 清晰持久 |
| 保质期 | 明确位置 | ≥3mm | 明确标注 |
| 贮存条件 | 明确位置 | - | 完整准确 |
| 生产许可证 | 包装上 | - | QS标志 |
| 生产者信息 | 明确位置 | - | 名称地址 |

### 2. 标签版面检查

| 检查项 | 要求 | 违规类型 |
|--------|------|----------|
| 文字规范 | 使用规范汉字 | 繁体字/错别字 |
| 外文使用 | 与汉字对应 | 仅外文违规 |
| 声称规范 | 有科学依据 | 虚假宣传 |
| 图案规范 | 不涉及禁忌 | 违规图案 |

### 3. 营养标签检查

| 检查项 | 要求 | 标准依据 |
|--------|------|----------|
| 格式要求 | 符合GB 28050格式 | 格式规范 |
| 营养素顺序 | 按标准顺序 | 强制标示 |
| 数值准确 | 与检测值一致 | 误差范围 |
| 声称规范 | 符合含量要求 | 功能声称 |

## 安全限量检查

### 1. 通用安全指标

| 指标 | 限值 | 检测标准 | 风险等级 |
|------|------|----------|----------|
| 总迁移量 | ≤10mg/dm² | GB 31604.8 | 高 |
| 高锰酸钾消耗量 | ≤10mg/kg | GB 31604.2 | 中 |
| 重金属(以Pb计) | ≤1mg/kg | GB 31604.9 | 高 |
| 脱色试验 | 阴性 | GB 31604.4 | 中 |

### 2. 特定迁移限量(SML)

| 物质 | SML | 物质 | SML |
|------|-----|------|-----|
| 铅(Pb) | 0.01mg/kg | 镉(Cd) | 0.002mg/kg |
| 甲醛 | 15mg/kg | 苯乙烯 | 0.1mg/kg |
| 丙烯腈 | 0.02mg/kg | 己内酰胺 | 15mg/kg |

### 3. 特定物质禁止使用

| 物质类型 | 说明 | 检测要求 |
|----------|------|----------|
| 荧光增白剂 | 纸包装禁止 | 荧光检测 |
| 偶氮染料 | 禁用于接触食品 | 偶氮检测 |
| 塑化剂 | 邻苯类禁用于脂肪类 | DEHP等检测 |
| 双酚A | 婴儿奶瓶禁用 | BPA检测 |

## 检查流程

```
┌─────────────────────────────────────────────────────────┐
│                    法规合规检查流程                       │
├─────────────────────────────────────────────────────────┤
│  1. 材料确认 → 2. 标准对照 → 3. 检测验证 → 4. 标识审核  │
│         ↓              ↓              ↓              ↓   │
│   树脂品种核查    限值标准确认    迁移量检测       标签检查  │
│         ↓              ↓              ↓              ↓   │
│   添加剂确认     合规判定        不合格整改      修改确认   │
└─────────────────────────────────────────────────────────┘
```

## 常见违规案例

| 违规类型 | 典型案例 | 处罚依据 |
|----------|----------|----------|
| 材料未备案 | 使用新型树脂未备案 | 食品安全法125条 |
| 标签不规范 | 净含量标注错误 | 定量包装规定 |
| 声称违规 | 普通食品声称保健功能 | 广告法/食安法 |
| 迁移超标 | 总迁移量超标 | 食安法111条 |
| 标识缺失 | 缺生产日期 | 食安法67条 |
