# 智能日历技能
一个开箱即用的智能日历工具，支持日期查询、农历转换、节假日判断、倒计时、日程管理。

## 功能描述
1. 查询今天/明天/后天的日期、星期、时间
2. 自动转换公历 ↔ 农历
3. 判断当天是否为法定节假日
4. 支持任意日期倒计时
5. 简易日程添加、查看、管理
6. 支持自然语言意图识别

## 支持意图（Intents）
- 今天几号 / 今天日期 / 当前时间
- 明天日期 / 后天日期
- 今天放假吗 / 是否节假日
- 距离XX日期还有多少天 / 倒计时
- 添加日程 / 新建提醒
- 查看日程 / 我的日程
