# Validation checklist

## Drill readiness

- [ ] severity model documented and understood
- [ ] on-call + escalation ladder tested
- [ ] comms templates approved
- [ ] reconciliation owner identified

## Live incident checks

- [ ] incident commander assigned
- [ ] blast radius quantified
- [ ] containment action executed
- [ ] update cadence maintained
- [ ] recovery and reconciliation verified
- [ ] postmortem created with owners/dates
