# Launch playbook (UX and ops)

Use this sequence for customer/support readiness:

1. Run [setup.md](setup.md) and confirm SLA + ownership.
2. Finalize state-wise copy using [examples.md](examples.md).
3. Validate macros/escalations against [reference.md](reference.md).
4. Run [validation-checklist.md](validation-checklist.md).
5. Align incident messaging with `upi-go-live-checklist` runbook.

Exit criteria:

- state messaging is clear and non-ambiguous
- support has approved macros and escalation matrix
- incident banners/templates are pre-approved
