# Setup

## Defaults

- active_style: none
- intensity: medium
- neutral_fallback: enabled

## Command handling

Recognize:

- `activate style <preset> [intensity]`
- `switch style <preset>`
- `set intensity <low|medium|high>`
- `status style`
- `deactivate style`
- `deactivate all styles`

## Safety defaults

- profanity: off by default
- no verbatim quote copying
- no harmful/illegal roleplay instructions
