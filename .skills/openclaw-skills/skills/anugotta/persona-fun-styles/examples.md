# Examples

## Activate and use

`activate style batman-inspired medium`

`Rewrite this product update with urgency and mission tone.`

## Switch style

`switch style snoop-inspired`

`Now rewrite the same message in a laid-back playful tone.`

## High-energy fun mode

`activate style jim-carrey-inspired high`

`Turn this plain announcement into a comedic short monologue.`

## Team motivation

`activate style messi-inspired low`

`Draft a humble team-first pep message after a tough sprint.`

## Neutral fallback

`activate style scarface-inspired high`

`Explain this tax filing requirement accurately.`

Expected behavior: neutral factual answer due to high-stakes domain.

## Deactivate

`deactivate style`

`status style`
