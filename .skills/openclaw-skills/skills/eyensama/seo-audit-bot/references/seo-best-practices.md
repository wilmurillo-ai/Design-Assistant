# SEO Best Practices Reference

## Title Tag
- Length: 50-60 characters
- Include primary keyword near the beginning
- Unique per page
- Format: "Primary Keyword - Secondary Keyword | Brand Name"

## Meta Description
- Length: 150-160 characters
- Include primary keyword naturally
- Compelling call-to-action
- Unique per page

## H1 Tag
- One per page
- Include primary keyword
- Should match user intent
- Different from title tag

## URL Structure
- Short and descriptive
- Use hyphens, not underscores
- Include target keyword
- Lowercase only
- No special characters or parameters

## Image Optimization
- Descriptive alt text (not keyword-stuffed)
- Compressed file sizes
- Descriptive file names (not "IMG_1234.jpg")
- Lazy loading for below-fold images

## Content
- Blog posts: 1,500-2,500 words
- Product pages: 300+ words
- Keyword density: 1-2%
- Use related keywords (LSI)
- Break content with H2/H3 headings
- Short paragraphs (2-4 sentences)

## Internal Linking
- Link to related content
- Use descriptive anchor text
- No more than 100 links per page
- Important pages should have more internal links

## Page Speed Signals
- Minified CSS/JS
- Compressed images (WebP preferred)
- Browser caching headers
- CDN for static assets
- Minimal render-blocking resources

## Mobile
- Responsive design
- Viewport meta tag
- Touch-friendly buttons (48px minimum)
- Readable without zooming

## Structured Data
- JSON-LD format (Google preferred)
- Common types: Article, Product, FAQ, HowTo, LocalBusiness
- Test with Google Rich Results Test

## Social
- Open Graph: og:title, og:description, og:image, og:url
- Twitter Cards: twitter:card, twitter:title, twitter:description, twitter:image
- Image dimensions: 1200x630px for OG, 1200x628 for Twitter
