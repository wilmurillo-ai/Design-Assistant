# BNBOT Mascot Skill

Generate BNBOT's lobster-bot mascot in various poses and scenarios with consistent character design.

## Features

- Style-consistent mascot generation using reference images
- Multiple reference poses (front, side, action, cute, logo)
- Transparent PNG output via green-screen chromakey
- CLI and Python API

## Install

```bash
npx clawhub install bnbot-mascot
```

## Usage

Ask your AI agent:

- "Generate a mascot image of the lobster waving hello"
- "Make a BNBOT sticker with the lobster holding a trophy"
- "Create a tweet image with the lobster celebrating"

## Character

BNBOT's mascot is a **Lobster Bot** — a lobster-robot hybrid with a golden retro TV head, LED pixel face (heart eye + wink), golden mechanical claws, and a red-orange lobster body.
