# Memory Orchestrator

> **全栈智能记忆系统** - 让 AI 拥有长期记忆、情感感知和自我进化能力

## 🚀 快速开始

```bash
# 一键安装
bash install.sh

# 启动服务
bash run.sh

# 开始使用
memory_search "上次解决 Git 冲突的方法"
```

## 📚 详细文档

查看 [SKILL.md](SKILL.md) 获取完整的功能介绍、安装指南、使用示例和架构图。

## 🎯 核心功能

- 🧠 **语义搜索** - FAISS + qwen2.5
- 🔄 **自动化同步** - Syncthing + Git-crypt
- 📸 **多模态理解** - CLIP + Whisper
- 🕸️ **知识图谱** - NetworkX 可视化
- ❤️ **情感标记** - 自动标注情绪
- 🦠 **自进化引擎** - 持续自我优化

## 📦 安装

```bash
# 方式一：iflow (推荐)
iflow skill install memory-orchestrator

# 方式二：手动安装
git clone https://github.com/openclaw/clawhub.git
cd clawhub/skills/memory-orchestrator
bash install.sh
```

## 📖 使用示例

```bash
# 语义搜索
memory_search "关键词"

# 多模态处理
memory_multimodal process image.jpg

# 生成知识图谱
memory_knowledge_graph generate

# 情感分析
memory_evolve tag-emotions

# 自进化优化
memory_evolve run
```

## 🛠️ 技术栈

- **AI 模型**: qwen2.5:7b, all-MiniLM-L6-v2, CLIP, Whisper
- **存储**: FAISS, Git, Syncthing
- **图数据库**: NetworkX, PyVis
- **自动化**: iflow, Cron, Systemd
