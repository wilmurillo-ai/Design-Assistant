## 📖 Overview（概述）

**quality-os** 是 3Q 质量保障体系的操作系统，整合所有质量相关技能。

通过**六大模块**（Task/Quality/Decision/Output/Knowledge/AI），提供统一的质量管理框架。

**适用场景**：
- 建立项目质量体系
- 定义质量标准
- 质量流程设计
- 质量指标追踪

**质量评分**: 15/15（S 级技能）

**依赖关系**: quality-prevention-milestone, self-challenge-3q, subagent-brief-template

**被依赖技能**:
- `quality-os-trigger` - 质量触发器
