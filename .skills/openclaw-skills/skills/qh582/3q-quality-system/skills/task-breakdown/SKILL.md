# task-breakdown-v3 - 任务拆解 +3Q 适配

## 🌐 English Summary

**Task Breakdown with 3Q Quality Adaptation**

Breaks down complex tasks into executable steps with automatic 3Q version matching:

**4 Quadrant Classification**:
- **Mechanical** (×0.5) → 3Q-Lite (5-10 min, ≥12/15)
- **Creative** (×1.5) → 3Q-Pro (20-30 min, ≥13/15)
- **Decision** (×1.0) → 3Q-Decision (30-45 min, ≥13/15)
- **Integration** (×2.0) → 3Q-Max (45-60 min, ≥14/15)

**Effect**: Task classification accuracy 100%, 3Q match rate 100%

**Triggers**: `任务拆解`, `分解任务`, `task breakdown`, `task classification`

## 🏗️ 任务分类矩阵

### 4 象限分类法

```
                    影响力
                      ↑
         ┌────────────┼────────────┐
         │   整合类   │   创作类   │
         │  (×2.0)    │  (×1.5)    │
         │  3Q-Max    │  3Q-Pro    │
         │            │            │
    复杂度 ────────────┼─────────────
         │            │            │
         │   机械类   │   决策类   │
         │  (×0.5)    │  (×1.0)    │
         │  3Q-Lite   │3Q-Decision │
         │            │            │
         └────────────┼────────────┘
                      →
                   风险
```

### 任务类型详解

| 类型 | 特征 | 权重 | 3Q 版本 | 质量要求 | 示例 |
|------|------|------|--------|---------|------|
| **机械类** | 重复性/标准化/低风险 | ×0.5 | 3Q-Lite | B 级 (10-11 分) | 数据录入/文件整理 |
| **创作类** | 创意/差异化/中风险 | ×1.5 | 3Q-Pro | A 级 (12-13 分) | 文档写作/内容创作 |
| **决策类** | 选择/风险评估/中风险 | ×1.0 | 3Q-Decision | A 级 (12-13 分) | 技术选型/方案选择 |
| **整合类** | 多模块/系统化/高风险 | ×2.0 | 3Q-Max | S 级 (14-15 分) | 系统设计/QualityOS |

## 🔧 3Q 版本详情

### 3Q-Lite（机械类任务）

**适用场景**：数据录入/文件整理/格式转换

**特点**：快速检查（3-5 分钟），确保准确性

```markdown
## 3Q-Lite 检查清单

### 逻辑问（准确性）
- [ ] 步骤是否完整？
- [ ] 数据是否准确？
- [ ] 有无遗漏？

**评分**: ___ / 5

### 用户问（可用性）
- [ ] 结果是否可用？
- [ ] 格式是否正确？
- [ ] 有无明显错误？

**评分**: ___ / 5

### 竞争问（效率）
- [ ] 效率是否提升？
- [ ] 有无更好的方法？
- [ ] 是否可自动化？

**评分**: ___ / 5

---

**总分**: ___ / 15
**质量等级**: B(10-11)/C(8-9)/D(<8)
```

**质量要求**：B 级（10-11 分）即可

---

## 📝 使用流程

### 标准流程

```
1. 任务输入 → 2. 任务分类 → 3. 确定 3Q 版本 → 
4. 任务拆解 → 5. 执行 → 6. 3Q 检查 → 7. 质量评分
```

### 快速流程（简单任务）

```
1. 任务输入 → 2. 识别为机械类 → 3. 3Q-Lite 检查 → 4. 输出
```

### 完整流程（复杂任务）

```
1. 任务输入 → 2. 识别为整合类 → 3. 3Q-Max 检查 → 
4. 任务拆解（WBS） → 5. 执行 → 6. 13 检查点 → 
7. 3Q 评分 → 8. S 级质量标签 → 9. 知识库沉淀
```
