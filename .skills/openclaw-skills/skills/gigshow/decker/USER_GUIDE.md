# Decker OpenClaw/Slack 사용자 체험 가이드

**대상**: Slack에서 @deckerclaw로 Decker를 처음 사용하는 사용자  
**목적**: 5분 안에 시그널·포지션·자동주문을 체험할 수 있도록 안내

---

## 1. 사전 준비 (1회)

| 단계 | 내용 |
|------|------|
| 1 | [decker-ai.com](https://decker-ai.com) 가입 (무료) |
| 2 | 로그인 후 [decker-ai.com/decker-link](https://decker-ai.com/decker-link)에서 **Slack 사용자 ID** 입력 |
| 3 | Slack ID 확인: Slack 프로필 클릭 → 더보기 → 멤버 ID 복사 (U로 시작) |
| 4 | Slack에서 @deckerclaw 봇이 있는 채널 또는 DM으로 이동 |

---

## 2. 체험 시나리오 (순서대로 해보기)

### 2.1 인사·기능 확인

| 입력 | 기대 응답 |
|------|-----------|
| `@deckerclaw 하이` | 서비스 소개 + 기능 목록 |
| `@deckerclaw 뭐 할 수 있어?` | 시그널·포지션·주문·자동주문 등 기능 요약 |

### 2.2 시그널 조회

| 입력 | 기대 응답 |
|------|-----------|
| `@deckerclaw 비트코인 시그널 알려줘` | BTCUSDT 시그널 (롱/숏, 신뢰도, 진입가·목표가·손절가) + "치킨먹자?" |
| `@deckerclaw 이더 시그널` | ETHUSDT 시그널 |

### 2.3 자동주문 체험 (핵심)

| 입력 | 기대 응답 |
|------|-----------|
| `@deckerclaw 이더리움 자동주문 해줘` | "✅ ETHUSDT 자동주문 설정 완료! (0.05개, 손절 2%, 익절 4%, 모의거래)" |
| `@deckerclaw 자동주문 설정 보여줘` | 설정된 규칙 목록 (ETHUSDT 등) |
| `@deckerclaw 자동주문 어떻게 써?` | 3단계 체험 가이드 (설정→확인→해제) |
| `@deckerclaw 이더리움 자동주문 끄줘` | "✅ ETHUSDT 자동주문 중지됐어요" |

**동작**: Decker가 4시간마다 시그널을 체크해, 설정한 종목에 좋은 시그널(신뢰도 50% 이상)이 나오면 **모의거래**로 자동 주문. 처음엔 모의거래로 안전하게 체험.

### 2.4 포지션·수익 (연동 필요)

| 입력 | 기대 응답 |
|------|-----------|
| `@deckerclaw 포지션 보여줘` | 포트폴리오 요약 또는 연동 안내 |
| `@deckerclaw 수익현황` | 총 평가액·손익 |

**연동 안 됐으면**: "Decker에 가입하고 Slack 연동해 주세요" + decker-link URL

### 2.5 주문 (승인 플로우)

| 입력 | 기대 응답 |
|------|-----------|
| `@deckerclaw ETH 0.01 매수해줘` | "승인 요청이 Slack으로 발송되었습니다. 승인/취소 버튼을 확인해 주세요." |

## 3. 자주 쓰는 말 모음

| 기능 | 예시 |
|------|------|
| 시그널 | 비트코인 시그널 알려줘, 이더 시그널 |
| 자동주문 | 이더리움 자동주문 해줘, 자동주문 설정 보여줘, 이더리움 자동주문 끄줘 |
| 포지션 | 포지션 보여줘, 포트폴리오, 수익현황 |
| 주문 | BTC 0.01 매수해줘, ETH 매도해줘 |
| 뉴스 | 뉴스 보여줘, 소식 보여줘 |
| 가이드 | 어떻게 써?, 뭐 할 수 있어? |

## 4. 텔레그램 사용자

텔레그램 봇(@deckerclawbot)에서도 동일한 명령 사용 가능:

- 시그널 알려줘, 비트코인 시그널
- 이더리움 자동주문 해줘, 자동주문 설정 보여줘
- 뉴스 보여줘

## 5. 에이전트 vs 대시보드 (혼동 방지)

| 채팅에서 가능 | 대시보드에서만 |
|---------------|----------------|
| 시그널·시세·포지션·주문·청산·자동주문·뉴스 보여줘 | 가입·연동·설정·API 키·알림 켜기·포트폴리오 리셋·전략 빌더 |

**자세한 구분**: `docs/에이전트_대시보드_사용자_가이드_20250312.md`

## 6. 문제 해결

| 증상 | 조치 |
|------|------|
| "연동해 주세요" | decker-link에서 Slack ID 재입력 |
| 시그널 없음 | judgment_signals에 최근 데이터 필요 (24시간 내) |
| 자동주문 미실행 | 4시간 주기 대기, auto_order_rules status=ACTIVE 확인 |
| 주문 승인 버튼 없음 | Slack DM 확인, order-request API 연동 상태 점검 |
| 포트폴리오 리셋해줘 | "decker-ai.com → 포트폴리오 → 자산 리셋 버튼" (채팅 불가) |
