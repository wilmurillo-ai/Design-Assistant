# 文档生成规范

通过 @documentation-gen 手动触发。

## README 模板
- 简介：一句话描述项目用途
- 快速开始：最少步骤跑起来
- 架构概览：核心模块及关系
- 开发指南：环境搭建、常用命令
- API 文档（如适用）
- 贡献指南（如适用）

## 变更日志
- 使用 Keep a Changelog 格式
- 按 Added / Changed / Fixed / Removed 分类
- 每条关联 issue 或 PR 编号

## API 文档
- 包含请求方法、路径、参数、响应示例、错误码
- 提供 curl 示例
