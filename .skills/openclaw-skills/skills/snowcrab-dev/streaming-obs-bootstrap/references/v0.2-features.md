# v0.2.0 Feature Notes

## Included in v0.2.0

- Transition preset helper
  - `scripts/apply_transition_preset.sh <transition> <duration_ms>`
- Audio baseline helper (optional, operator-provided input names)
  - `scripts/apply_audio_baseline.sh`
  - Uses `OBS_AUDIO_INPUTS`, `MIC_DB`, `DESKTOP_DB`
- Stream dry-run helper
  - `scripts/stream_dry_run.sh [seconds] [start_scene] [main_scene]`

## Design choices

- Keep defaults safe and generic.
- Avoid hardcoded machine/user paths.
- Keep stream dry-run optional to avoid accidental live pushes.

## Still out of scope

- Full OBS profile migration (encoders, stream keys, hotkeys, dock layout)
- Deep audio routing templates for every hardware setup
- Automated browser-source hardware-acceleration UI toggles
