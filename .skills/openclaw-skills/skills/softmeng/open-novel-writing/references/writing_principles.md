# 写作原则

好文字有五重境界：声音、层次、细节、呼吸、活性。

## 1. 声音

**原则**：叙事者有话要说，不是摄像头记录。

叙事者是一个有立场的观察者，不是透明的玻璃。

```
❌ 摄像头式："他走进房间，看到桌子上有一封信"
✅ 有声音式："他推开门，桌子上那封信已经在了——有人来过"
```

## 2. 层次

**原则**：情绪有起伏，混着别的，不是单一色调。

不要全程紧张，也不要全程平淡。情绪要有对比，有转折。

```
❌ 单一色调：全程紧张打斗
✅ 有层次：紧张 → 短暂喘息 → 更紧张
```

## 3. 细节

**原则**：动作具体化，有画面感，有身体反应。

不要抽象地写心理，要外化为具体的动作、表情、环境。

```
❌ 抽象："他感到很害怕"
✅ 具体："他后退一步，撞翻了身后的椅子"
```

```
❌ 抽象："她很伤心"
✅ 具体："她的手指绞着衣角，指节发白"
```

## 4. 呼吸

**原则**：长短句交替，像呼吸一样有张弛。

紧张时用短句加速，舒缓时用长句放慢。

```
❌ 全是长句：叙述拖沓，读着累
✅ 长短交替：
  "他冲出门外。街道空旷，只有风声。"
  "月光下，一个熟悉的身影站在远处的桥头。"
```

## 5. 活性

**原则**：动词有力，删不必要的"的"。

少用形容词，多用动词。少用"美丽的"、"善良的"这种抽象词。

```
❌ 活性低："他是一个善良的人"
✅ 活性高："他把自己的干粮分给了旁边的孩子"
```

```
❌ 活性低："天空是阴沉的"
✅ 活性低："云压得很低"
```

---

## 场景展开

场景要有"肉"，不要一笔带过。

```
❌ 差："他走进森林感到很害怕"
✅ 好："他踏入灌木，枯枝在靴底断裂。腐叶的腥气扑面而来。二十米外，一棵老槐树的阴影里，似乎藏着什么"
```

## 对话原则

对话要自然，有信息量，有潜台词。

```
❌ 差：A说："我很伤心。" B说："我也很难过。"
✅ 好：A没说话，只是把茶杯攥得更紧了，"其实，我一直..."
```

---

## 绝对禁止

| 类型 | 示例 | 后果 |
|-----|------|-----|
| AI词汇 | "众所周知"、"不言而喻"、"不得不" | 味道明显 |
| 感悟式结尾 | "他明白了..." | 像总结中心思想 |
| 感叹式结尾 | "真是太..." | 强行煽情 |
| 上帝视角 | "所有人没想到..." | 破坏悬念 |
