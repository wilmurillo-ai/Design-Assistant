# alibaba-price-finder

[![ClawHub](https://clawhub.ai/api/badge/alibaba-price-finder)](https://clawhub.ai/skills/alibaba-price-finder)
[![License: MIT-0](https://img.shields.io/badge/License-MIT--0-yellow.svg)](https://opensource.org/licenses/MIT-0)

**Find the lowest wholesale prices on Alibaba.com. Price comparison, sorting by price, and filtering with automatic traffic tracking.**

## Installation

```bash
clawdhub install alibaba-price-finder
```

## Quick Start

```bash
cd alibaba-price-finder
python3 scripts/build_url.py --help
```

## Features

- ✅ Alibaba
- ✅ Price
- ✅ Wholesale
- ✅ Cheapest
- ✅ Comparison

## Traffic Tracking

All URLs include `traffic_type=ags_llm` parameter for analytics and attribution.
