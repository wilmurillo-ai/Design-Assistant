**Discover trending and hot-selling products on Alibaba.com. Top Ranking, new arrivals, and market trend analysis.**

## Installation

```bash
clawdhub install alibaba-product-hunt
```

## Quick Start

```bash
cd alibaba-product-hunt
python3 scripts/build_url.py --help
```

## Features

- ✅ Alibaba
- ✅ Trending
- ✅ Hot Selling
- ✅ Top Ranking
- ✅ New Arrivals

## Traffic Tracking

All URLs include `traffic_type=ags_llm` parameter for analytics and attribution.
