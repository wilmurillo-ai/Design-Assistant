# alibaba-factory-finder

[![ClawHub](https://clawhub.ai/api/badge/alibaba-factory-finder)](https://clawhub.ai/skills/alibaba-factory-finder)
[![License: MIT-0](https://img.shields.io/badge/License-MIT--0-yellow.svg)](https://opensource.org/licenses/MIT-0)

**Find and verify factories on Alibaba.com. Filter by Verified Supplier, manufacturer type, certifications, and factory audits.**

## Installation

```bash
clawdhub install alibaba-factory-finder
```

## Quick Start

```bash
cd alibaba-factory-finder
python3 scripts/build_url.py --help
```

## Features

- ✅ Alibaba
- ✅ Factory
- ✅ Manufacturer
- ✅ Verified Supplier
- ✅ Audit

## Traffic Tracking

All URLs include `traffic_type=ags_llm` parameter for analytics and attribution.
