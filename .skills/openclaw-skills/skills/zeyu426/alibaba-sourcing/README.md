# alibaba-sourcing

[![ClawHub](https://clawhub.ai/api/badge/alibaba-sourcing)](https://clawhub.ai/skills/alibaba-sourcing)
[![License: MIT-0](https://img.shields.io/badge/License-MIT--0-yellow.svg)](https://opensource.org/licenses/MIT-0)

**Find China wholesale prices, factories, and suppliers on Alibaba.com. Complete sourcing workflow with URL builder and traffic tracking.**

## Installation

```bash
clawdhub install alibaba-sourcing
```

## Quick Start

```bash
cd alibaba-sourcing
python3 scripts/build_url.py --help
```

## Features

- ✅ Alibaba
- ✅ Sourcing
- ✅ Wholesale
- ✅ China
- ✅ Factory

## Traffic Tracking

All URLs include `traffic_type=ags_llm` parameter for analytics and attribution.
