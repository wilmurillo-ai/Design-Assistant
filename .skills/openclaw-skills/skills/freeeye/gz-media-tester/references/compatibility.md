| 屏幕类型 | 关键问题 | 测试方法 |
|---------|---------|---------|
| 刘海屏/水滴屏 | 内容被遮挡 | 打开各页面，检查顶部状态栏 |
| 折叠屏（展开） | 布局错位 | 展开后横屏/竖屏分别检查 |

| 曲面屏 | 边缘点击失效 | 边缘按钮可点击 |

## 四、网络环境测试矩阵

| 环境 | 场景 | 关注点 |
|------|------|--------|
| WiFi（5G）| 正常环境 | 功能完整，无异常 |
| 4G | 移动网络 | 视频播放流畅度 |
| 3G | 弱网 | 内容加载时间，显示友好提示 |
| 2G | 极弱网 | 超时处理，离线提示 |
| WiFi→4G切换 | 场景切换 | 无中断，不崩溃 |
| 弱网（模拟）| 使用Charles/Fiddler限速 | 加载提示，超时处理 |
