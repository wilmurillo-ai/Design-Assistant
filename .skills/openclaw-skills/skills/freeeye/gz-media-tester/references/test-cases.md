### 三端差异注意

| 功能点 | Android | iOS | HarmonyOS |
|--------|---------|-----|-----------|
| 推送通道 | 厂商通道（华为/小米/OPPO/Vivo） | APNs | 华为Push+鸿蒙Push |
| 安装包格式 | APK/AAB | IPA | HAP |
| 权限弹窗 | 运行时申请 | 首次触发时系统弹窗 | 同Android |
| 热更新 | 整包更新或Tinker | App Store审核 | 应用市场审核 |
| 屏幕适配 | 全面屏/水滴屏/折叠屏 | 刘海屏/灵动岛 | 折叠屏/平板 |

## 七、性能基准

| 指标 | 目标值 | 测试方法 |
|------|--------|---------|
| APP冷启动时间 | ≤ 3s | 记录启动到首页渲染完成时间 |
| APP热启动时间 | ≤ 1s | 退出后快速返回 |
| 接口响应时间（P99） | ≤ 1s | 压测工具记录P99延迟 |
| H5首屏时间 | ≤ 2s | Chrome DevTools或Lighthouse |
| 页面切换帧率 | ≥ 55 FPS | Android Profiler / Instruments |
| 内存占用（空闲） | ≤ 200MB | Android Memory Profiler |
| 电量消耗 | 10分钟耗电 ≤ 3% | 电池监控 |
| Crash率 | ≤ 0.1% | 统计7日崩溃率 |
