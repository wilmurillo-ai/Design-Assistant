# Bug单模板

## 标准格式（便于导入禅道/Jira）

```
【Bug标题】简洁描述问题现象，一行说明

【所属项目】融媒云APP（Android）/ 新花城APP（iOS）/ 融媒云后台 / H5 / 小程序 / 接口
【所属模块】登录 / 新闻 / 任务 / 推送 / 搜索 / 分享 / 支付 / ...
【严重级别】P0（致命）/ P1（严重）/ P2（一般）/ P3（建议）
【优先级】高 / 中 / 低

【Bug类型】功能缺陷 / UI问题 / 性能问题 / 安全问题 / 兼容性问题 / 接口问题

【复现概率】必现 / 经常 / 偶发 / 极偶发

【复现步骤】
1.
2.
3.

【实际结果】
（描述实际发生的错误行为）

【预期结果】
（描述期望的正确行为）

【环境信息】
- APP版本：v2.x.x
- 操作系统：Android 14 / iOS 17.x / HarmonyOS 4.x
- 设备型号：[设备名称]
- 网络环境：WiFi / 4G / 5G
- 渠道：[应用宝 / App Store / 华为应用市场]

【截图/日志】
[在此插入截图或Logcat/Console日志]

【复现设备清单】
- [设备1]：[版本] — 复现/不复现
- [设备2]：[版本] — 复现/不复现
- [设备3]：[版本] — 复现/不复现

【开发备注】
[留给开发人员填写的备注]
```

## 严重级别说明

| 级别 | 定义 | 示例 |
|------|------|------|
| P0 致命 | 功能不可用、崩溃、数据丢失、安全漏洞 | APP启动崩溃、无法登录、支付金额错误 |
| P1 严重 | 核心功能受损，影响大量用户 | 新闻无法加载、搜索无结果、推送失效 |
| P2 一般 | 非核心功能问题，可绕过或有临时方案 | 分享图标偏移、页面轻微错位 |
| P3 建议 | 体验优化，不影响功能 | 文案措辞建议、动画流畅度优化 |

## Bug单示例

```
【Bug标题】新花城APP Android版新闻详情页图片无法横屏全屏查看

【所属项目】新花城APP（Android）
【所属模块】新闻详情 - 图片查看
【严重级别】P2
【优先级】中

【Bug类型】功能缺陷

【复现概率】必现

【复现步骤】
1. 打开新花城APP，进入新闻详情页
2. 找到含有多张图片的新闻
3. 点击任意图片进入图片预览模式
4. 将手机横屏

【实际结果】
图片保持竖屏居中，顶部/底部有黑边，无法填满横屏

【预期结果】
图片自动旋转并填满横屏全屏显示

【环境信息】
- APP版本：v4.2.1
- 操作系统：Android 14（OnePlus 12）
- 网络环境：WiFi
- 渠道：应用宝

【截图/日志】
[截图已附件：bug_screenshot_001.png]

【复现设备清单】
- OnePlus 12（Android 14）— 复现
- 小米14（Android 14）— 复现
- 三星S24（Android 14）— 复现
- 华为Mate60（HarmonyOS 4）— 不复现（正常）
```
