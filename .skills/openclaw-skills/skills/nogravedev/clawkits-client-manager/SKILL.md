# Client Manager — Automated Client Communication

Monitor client channels, auto-triage by urgency, draft responses using conversation history, and schedule follow-ups. Built for freelancers and agencies managing multiple clients.

- **Auto-triage system** — categorizes incoming messages as 🔴 Urgent, 🟡 Normal, 🟢 Low priority
- **Smart response drafting** — pulls context from previous conversations, matches client communication style, includes project details
- **Follow-up scheduling** — tracks promises, sets reminders for pending responses, escalation rules for overdue items
- **Per-client context tracking** — project status, communication preferences, past issues, key contacts, contract/payment status
- **Channel integration** — works across email, Slack, Discord, WhatsApp
- **Weekly interaction summaries**
