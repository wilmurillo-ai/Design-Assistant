# 微信 PC 端朋友圈发布工作流（稳定版）

## 适用范围
- 在 Windows 桌面端微信发布朋友圈（含图片 + 文案）。
- 依赖已登录微信；若被踢下线需人工扫码登录。

## 固定前提
- 微信 EXE 示例：`C:\Program Files\Tencent\Weixin\Weixin.exe`
- 目标图片路径示例：`C:\path\to\image.png`
- 朋友圈发布入口在**独立“朋友圈”窗口顶部三按钮**里，**中间相机按钮**为发布入口。

## 核心流程（概要）
1. **确认微信主界面已打开**（聊天列表页）
2. **打开独立朋友圈窗口**
   - 从微信左侧栏点击“发现/朋友圈”入口
3. **进入发布页**
   - 在朋友圈窗口顶部三按钮中，横向点击直到弹出“选择文件”窗口（中间相机按钮）
4. **选择图片**
   - 在文件选择框输入完整路径并回车
5. **填写文案 + 发表**
   - 在“这一刻的想法”输入区粘贴文案
   - 点击“发表”按钮
6. **验证**
   - 发布后编辑页会消失，通常跳回“收藏/内容预览”

## 常见失败与处理
- **还在聊天页**：说明未打开独立朋友圈窗口；继续切换左侧栏入口。
- **发布页未出现**：顶部按钮点位不准，需横向扫描三按钮区域。
- **文案未粘贴成功**：OCR 可能识别不到；发布前人工确认或多粘贴一次。
- **出现二维码**：必须由用户扫码登录后才能继续自动化。
