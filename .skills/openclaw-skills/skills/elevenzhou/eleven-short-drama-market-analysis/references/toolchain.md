## 四、角色一致性解决方案

### 方案一：参考图法（推荐入门）
- 每个角色保存3-5张标准参考图
- 文生图时使用 `style reference` + `character reference`
- 提取角色特征词：发色/瞳色/服装/配饰

### 方案二：Lora模型训练（进阶）
- 训练专属角色Lora
- 适合固定IP的长期运营
- 工具：海艺AI、即梦Lora

## 五、全流程工具链推荐

### AI漫剧（动漫风格）
```
剧本 → 即梦AI/Vidu（图生角色）→ 可灵AI（图生视频）→ 配音 → 剪映合成
```

### AI仿真人剧（真人风格）
```
剧本 → 即梦AI（文生图）→ 可灵AI（图生视频）→ Suno（BGM）→ 剪映合成
```

### 自动化工作流
- **Toonflow**：小说转视频流水线
- **Moki**：动画短片/网文短剧/故事绘本
- **ComfyUI + FramePack**：本地离线生成
