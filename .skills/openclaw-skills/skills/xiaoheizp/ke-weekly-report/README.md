# 周报生成器 - 使用说明

## 🚀 快速开始

### 方法 1: 命令行使用

```bash
# 基础用法
python3 generator.py "本周完成了用户模块开发，修复了 3 个 bug，参加了产品评审会"

# 输出到文件
python3 generator.py "工作内容..." > weekly-report.md
```

### 方法 2: 作为 OpenClaw 技能

```bash
# 安装技能
clawhub install weekly-report-generator

# 使用技能
clawhub run weekly-report-generator --input "工作内容..."
```

### 方法 3: 在线演示（计划中）

访问：https://weekly-report.demo.com

---

## 📝 输入格式

### 简单模式
直接输入工作内容，用逗号或换行分隔：
```
完成了用户模块开发，修复了 3 个 bug，参加了产品评审会，协助新人熟悉项目
```

### 详细模式
按类别输入：
```
【重点工作】
1. 完成了用户模块开发
2. 修复了支付流程的 3 个 bug

【常规工作】
1. 参加了产品需求评审会
2. 协助新人熟悉项目

【问题】
测试环境不稳定，影响开发进度

【下周计划】
1. 开始订单模块开发
2. 输出技术文档
```

## 💎 高级功能（付费版）

### 1. Word/PPT 输出
```bash
python3 generator.py "工作内容..." --format word --output report.docx
```

### 2. 多模板选择
```bash
# 互联网风格
python3 generator.py "工作内容..." --style tech

# 国企风格
python3 generator.py "工作内容..." --style formal

# 创业公司风格
python3 generator.py "工作内容..." --style casual
```

### 3. 数据可视化
```bash
python3 generator.py "工作内容..." --add-charts
```

### 4. 批量生成
```bash
python3 generator.py --batch input.xlsx --output reports/
```
