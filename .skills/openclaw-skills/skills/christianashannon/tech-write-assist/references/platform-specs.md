# 各平台内容规范速查表

## X (Twitter)

| 项目 | 规范 |
|------|------|
| 单条推文字符限制 | 280 字符 (标准账户), 25000 字符 (Premium) |
| Thread 推荐长度 | 5-15 条 |
| 图片格式 | PNG / JPEG, 最大 5MB |
| 图片尺寸 | 推荐 1200x675 (16:9) 或 1200x1200 (1:1) |
| 每条推文图片数 | 最多 4 张 |
| 视频长度 | 最长 2:20 (标准), 最长 60 分钟 (Premium) |
| 标签 | 2-3 个 hashtag, 不要过多 |
| 最佳发布时间 (全球) | 周二至周四, 9:00-12:00 EST |
| AI 社区活跃时间 | 论文发布日 (周一/周四), 美西时间上午 |

## 小红书

| 项目 | 规范 |
|------|------|
| 标题字数 | 最多 20 字 (含 emoji) |
| 正文字数 | 最多 1000 字 (建议 300-800 字) |
| 话题标签 | 最多 10 个 (建议 5-8 个) |
| 图片数量 | 1-18 张轮播图 |
| 图片格式 | JPG / PNG |
| 封面图推荐尺寸 | 1080x1080 (1:1) 或 1080x1440 (3:4) |
| 内容图推荐尺寸 | 1080x1440 (3:4) |
| 视频长度 | 最长 15 分钟 |
| 最佳发布时间 | 周五至周日, 18:00-22:00; 工作日 12:00-14:00 |
| AI/科技内容活跃时间 | 工作日午间 12:00-13:00, 晚间 20:00-22:00 |

### 小红书内容审核注意事项
- 避免直接提及竞品商标 (可用缩写或暗示)
- 避免绝对化用语: "最好" "第一" "唯一"
- 不使用未经授权的他人图片
- AI 生成内容建议标注

## 微信公众号

| 项目 | 规范 |
|------|------|
| 标题字数 | 最多 64 字 (建议 20-40 字) |
| 摘要字数 | 最多 120 字 |
| 正文字数 | 无硬性限制 (建议 1500-5000 字) |
| 封面图尺寸 | 900x383 (2.35:1) 或 1068x455 |
| 正文图片 | 宽度建议 1080px, 高度不限 |
| 正文图片格式 | JPG / PNG / GIF, 单张最大 10MB |
| 每篇图片总量 | 无限制 (建议 6-15 张) |
| 排版 | 支持 Markdown 导入 (通过第三方工具如 mdnice) |
| 最佳发布时间 | 工作日 7:30-9:00 (通勤), 12:00-13:00 (午间), 20:00-22:00 (晚间) |
| 技术文最佳时间 | 工作日 8:00-9:00, 12:00-13:00 |

### 微信公众号排版建议
- 正文字号: 15-16px
- 行间距: 1.75-2 倍
- 字间距: 0.5-1px
- 两端缩进: 建议 16px
- 重点文字使用加粗, 不建议使用下划线
- 引用块用于专家评论或数据引用
- 分割线用于章节分隔
