# 小红书 — 技术揭秘/大厂黑科技曝光风格 写作模板

## 平台定位

小红书科技圈层用户，对 AI/互联网技术有一定关注但不一定是从业者。他们关注:
- 大厂又搞了什么"黑科技"
- 技术背后的内幕和信息差
- 看起来很厉害但能用人话解释的技术
- 行业趋势和发展方向

## 风格要求

| 要素 | 规范 |
|------|------|
| 语言 | **简体中文**，专业术语 + 通俗解读结合 |
| 标题 | 15-25 字，悬念感强，含 emoji |
| 正文 | 400-800 字 |
| 段落 | 短段落（2-4 句），emoji 分段符号 |
| 话题标签 | 5-8 个 |
| 人称 | 第一人称或第三人称客观视角均可 |
| 语气 | 科技博主揭秘内幕，"懂行"人设，权威但不学术 |

## 关键词库（自然融入）

- 曝光 / 揭秘 / 深扒 / 拆解
- 黑科技 / 硬核 / 炸裂
- 圈内 / 业内 / 大厂
- 震动 / 炸了 / 刷屏
- 信息差 / 提前知道
- 藏不住了 / 瞒不住了
- 关键数据 / 核心技术

## 标题公式

从以下模板中选择最合适的:

1. `🔍 [公司名]最新论文曝光：[技术名]到底有多强？`
2. `⚡ [技术名]技术揭秘！[公司名]这次真藏不住了`
3. `🧠 刚刚，[公司名]发布了[技术名]，圈内炸了`
4. `📢 业内震动！[核心发现]，[一句话影响]`
5. `🔬 深扒[公司名][技术名]：[最亮眼的数据点]`
6. `💥 [技术名]硬核拆解！[核心优势]碾压[竞品/前代]`
7. `🚨 [公司名]放大招了！[技术名]，[行业反应]`
8. `👀 全网刷屏的[技术名]，我来给你讲讲到底怎么回事`

## 术语处理策略

与好物推荐风格不同，这种风格可以保留专业术语，但必须在首次出现时给出通俗解释:

**格式**: `[专业术语]（[通俗解释]）`

示例:
- "采用了 Mixture of Experts（混合专家）架构，简单说就是让多个专长不同的小 AI 分工合作"
- "通过知识蒸馏（把大模型的能力'浓缩'到小模型里）"
- "在推理阶段（也就是 AI 实际回答问题的时候）"
