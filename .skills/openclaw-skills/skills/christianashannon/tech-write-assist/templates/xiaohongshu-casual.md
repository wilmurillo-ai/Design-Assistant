# 小红书 — 好物推荐/日常种草风格 写作模板

## 平台定位

小红书泛科技用户群体，以 20-35 岁年轻职场人为主。他们关注:
- AI 工具能不能帮我提效/省事
- 有什么好玩的新技术可以体验
- 大厂又搞了什么有意思的东西
- 能不能用通俗语言让我秒懂

## 风格要求

| 要素 | 规范 |
|------|------|
| 语言 | **简体中文**，大量网络用语和口语化表达 |
| 标题 | 15-25 字，**必须含 emoji**，疑问/惊叹句式 |
| 正文 | 300-600 字 |
| 段落 | 极短段落（1-3 句），大量换行留白 |
| 话题标签 | 5-8 个，放在正文末尾 |
| 人称 | 第一人称，"我发现了…" "我试了下…" |
| 语气 | 闺蜜分享发现的惊喜感，真诚推荐 |

## 常用网络用语（自然融入，不要堆砌）

- 绝了 / 绝绝子 / yyds
- 救命 / 真的会谢
- 吹爆 / 安利 / 种草
- 打工人 / 效率翻倍
- 天花板 / 这也太强了
- 我不允许还有人不知道
- 谁懂啊 / 懂的都懂

## 标题公式

从以下模板中选择最合适的，根据论文内容填充:

1. `[emoji] [技术/产品名]用了之后效率翻倍！AI打工人狂喜`
2. `[emoji] 天花板级[功能/能力]！这个AI我吹爆`
3. `[emoji] 救命！终于有[技术效果]了`
4. `[emoji] [公司名]偷偷搞的新技术，用过的都说绝`
5. `[emoji] 我不允许还有人不知道这个[技术类型]`
6. `[emoji] 试了下[技术/产品名]，这也太[效果]了吧`
7. `[emoji] [数字]个让你[好处]的AI神器，第[N]个绝了`
8. `[emoji] 被[技术名]种草了！[一句话效果]`

常用标题 emoji: ✨ 🔥 💡 🤯 💪 🎯 ⚡ 🚀 ❗ 📢

## 术语转化策略

将所有专业术语翻译为日常语言。必须让完全不懂技术的人也能理解:

| 专业术语 | 小红书表达 |
|---------|-----------|
| 大模型 / LLM | AI 大脑 / 超级 AI |
| 推理 (Inference) | AI 思考/回答 |
| 训练 (Training) | 教 AI 学习 |
| 微调 (Fine-tuning) | 给 AI 做专项训练 |
| Benchmark | 跑分测试 / 能力考试 |
| 多模态 | 能看图能听话能写字 |
| Token | AI 处理的文字单元 |
| 延迟 (Latency) | 等待时间 / 响应速度 |
| 参数量 | AI 的"脑容量" |
| 上下文窗口 | AI 的"记忆力" |
| Transformer | AI 的核心引擎 |
| 量化 (Quantization) | AI 瘦身术 / 压缩技术 |
| RAG | AI + 知识库搜索 |
| Agent | AI 助手 / AI 自动帮手 |
