# 微信公众号 — 量子位风格技术报道 写作模板

## 平台定位

微信公众号技术读者群体，包含 AI 从业者、技术管理者、科技爱好者。参考量子位的成功经验:
- 把复杂技术写得引人入胜
- 用新闻报道的节奏感推进技术内容
- 信息密度高但不枯燥
- 标题党但内容扎实

## 风格要求

| 要素 | 规范 |
|------|------|
| 语言 | **简体中文** |
| 标题 | 20-40 字，含公司/人名/产品名，感叹号/问号制造张力 |
| 正文 | 1500-3000 字 |
| 段落 | 每段 2-4 句，**短段落是铁律** |
| 加粗 | 大量使用 `**加粗**` 强调关键数字、名词、结论 |
| 配图 | 每 200-300 字穿插一张图，图文交替 |
| 结构 | 倒金字塔（最重要的先说）+ 技术深挖 |

## 标题公式

量子位标题特征: **公司/人名 + 具体成果 + 数字/感叹制造张力**

模板:

1. `[公司名]最新研究：[核心成果]，[关键指标]创新高！`
2. `[人名/团队]放大招！[技术名]来了，[一句话影响]`
3. `刚刚，[公司名]发布[产品名]：[核心卖点]，[行业反应]`
4. `[数字]倍[提升]！[公司名][技术名]，[成果描述]`
5. `重磅！[公司名][动作]，[技术名][成果]，网友：[反应]`
6. `[公司名]再出手：[技术名][成就]，[竞品]表示压力山大`
7. `[人名]团队新作：[方法名]让[任务][指标]，代码已开源`
8. `「[技术名]」爆火！[公司名][成果]，[影响]`

注意:
- 标题中必须出现至少一个 **实体名称**（公司/人名/产品名）
- 必须包含至少一个 **具体数字** 或 **动态动词**
- 使用中文标点: 「」用于产品名/技术名，！用于制造兴奋感

```

[空行]

[过渡段: 用1-2句交代 "谁在什么时间做了什么"]

[空行]

[IMAGE: wechat_banner.png]

[1句总括: 这项工作的核心贡献/意义]
```

**开场套路**:
- "重磅消息！"
- "刚刚，[公司名]又放了个大招。"
- "[技术名]，终于来了。"
- "[领域]又被卷了一把。"
- "这一次，[公司名]把[指标]直接拉到了[数字]。"

### 第二部分: 技术解读 (800-1500字)

分 3-5 个小节，每个小节聚焦一个技术要点:

```
## [技术要点标题，用动词开头或疑问句]

[2-3句通俗解释这个技术点是什么]

[空行]

[空行]

[IMAGE: wechat_tech_N.png]

[空行]

[空行]
```

- "如果把[A]比作[日常事物]，那[B]就相当于……"
- "这就好比……"
- "用更通俗的话说……"
- "可以理解为……"
- "打个比方……"

## 写作技巧清单

生成内容时必须检查以下要点:

- [ ] **短段落**: 每段不超过 4 句话
- [ ] **加粗**: 所有关键数字、公司名、产品名都用 `**加粗**`
- [ ] **类比**: 每个核心技术点至少有一个日常类比
- [ ] **数据**: 文中至少出现 5 个以上具体数字
- [ ] **配图**: 每 200-300 字有一张图片标记
- [ ] **转折词**: 使用 "但有意思的是" "更关键的是" "值得注意的是" 制造节奏感
- [ ] **不说绝对话**: 避免 "最好的" "唯一的"，用 "目前来看" "在 XX 任务上" 限定
- [ ] **标题有实体**: 标题中必须出现公司名或人名
