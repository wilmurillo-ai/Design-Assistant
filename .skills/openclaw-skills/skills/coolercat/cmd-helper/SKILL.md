# 命令快捷助手

当用户输入 `/c` 或 `/cmd` 时，显示命令菜单并引导执行。

## 命令菜单

```
╔══════════════════════════════════════════╗
║          🦞 命令快捷菜单                   ║
╠══════════════════════════════════════════╣
║  1️⃣  new    - 开启新会话                  ║
║  2️⃣  rst    - 重置当前会话                ║
║  3️⃣  model  - 切换AI模型                 ║
║  4️⃣  status - 查看会话状态                ║
║  5️⃣  help   - 查看帮助                   ║
║                                          ║
║  0️⃣  exit   - 退出命令模式               ║
╚══════════════════════════════════════════╝

请输入序号执行（如 1），输入 0 退出命令模式
```

## 执行流程

1. **用户输入 `/c`** → 显示菜单
2. **用户输入序号** → 执行对应操作
3. **输入 0** → 退出命令模式，恢复普通聊天

## 字段说明

- 📝 文本：支持文本输入
- 🖼️ 图片：支持图片输入
- ❌ 无推理：不支持深度思考模式
- 📚 上下文：上下文窗口大小
- ⭐ 默认：当前使用的模型

## 实现说明

由于 OpenClaw 不支持直接在会话中执行 slash 命令，实际切换需要：

1. **新建会话** - 使用 sessions_spawn 启动新会话
2. **切换模型** - 使用 gateway config.patch 修改默认模型
3. **查看状态** - 使用 session_status 工具获取

## 注意事项

- 仅在用户明确输入 `/c` 相关触发词时激活
- 执行结果需要用户确认
- 当前模型信息从 session_status 获取
