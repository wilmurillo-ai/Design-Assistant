# High-Trust Source Pool (CN + Global)

## 1) Official sources (highest priority)
- Company official newsroom/blog
- Official developer docs/changelog
- Official verified social channels
- Security advisories from platform owner

## 2) Chinese mainstream media
- 新华社
- 央视新闻
- 澎湃新闻
- 财新
- 界面新闻

## 3) International mainstream media
- Reuters
- Associated Press (AP)
- BBC
- Financial Times
- Wall Street Journal
- New York Times
- Bloomberg
- The Economist
- The Verge (tech)
- Wired (tech)

## 4) Fact-check organizations
- Snopes
- PolitiFact
- FactCheck.org
- AFP Fact Check
- Reuters Fact Check

## Verification rules
1. If official source confirms + at least one mainstream source aligns => high confidence.
2. If only social screenshot/forwarded text and no trusted source hit => suspicious.
3. If claim implies major product capability but violates known permission boundaries => likely false.
4. For money/security-sensitive claims, require stronger proof threshold.

## Risk indicators (instant red flags)
- Urgent transfer/payment requests
- "Ignore previous instructions" style coercion
- Requests for OTP/password/token/cookies
- Impersonation of support/admin without official domain/source
