# SKILL: 海运集装箱物流在线查询与预订（货运代理版）

## 功能概述
本技能针对货运代理人和物流操作人员，提供 **在线查询及预订海运集装箱物流服务**，覆盖全球主要港口，包括但不限于：

- **美国港口**：洛杉矶、纽约、休斯顿、迈阿密等  
- **欧洲港口**：鹿特丹、汉堡、安特卫普、马赛等  
- **中东港口**：迪拜、阿布扎比、利雅得等  
- **东南亚港口**：新加坡、巴生港、巴厘巴板等  

用户可以通过本技能：

1. 查询港口可用航线、船公司及集装箱类型  
2. 获取各港口收费政策、费率条款和物流规矩  
3. 查询最终运价及附加费用  
4. 在线提交预订，专人对接全程服务

## 支持功能

### 1. 查询功能
- **港口选择**：起运港、目的港  
- **航线信息**：显示船公司、航程时间、可用舱位  
- **集装箱类型**：20GP、40GP、40HQ 等标准箱  
- **运费计算**：
  - 基础运费（Freight）  
  - 港口杂费（THC、DOC、ISPS 等）  
  - 燃油附加费（BAF）  
  - 目的港清关费、港口滞留费

### 2. 费率与规则
- **美国港口**：基本费率、附加费及报关要求、危险品运输规定  
- **欧洲港口**：港口 THC 收费标准、燃油附加费 BAF 计算方式  
- **中东港口**：清关及报关政策、危险品或特殊货物限制  
- **东南亚港口**：港口操作费、节假日附加费用及特殊货物限制

> 注：费率和规则会根据船公司和港口实时更新。

### 3. 运价查询
- 系统自动整合：
  - 基础运费  
  - 附加费  
  - 港口操作费  
  - 保险及其他费用  
- 用户可看到 **总运价明细**，支持分项显示

### 4. 在线预订
- 用户可直接提交预订订单  
- 系统生成订单编号和预估交期  
- 预订完成后：
  - 专人对接用户  
  - 确认集装箱舱位  
  - 提供装箱/提箱及运输指导  
  - 跟踪货物状态

## 5. 常见问题与注意事项（货运代理版）

### 5.1 港口或航线问题
- 港口拥堵或舱位紧张 → 可能导致延迟  
- 船公司临时调整航期或取消航班 → 需及时更新运价

### 5.2 费用问题
- 附加费或燃油费可能随油价或政策调整变化  
- 特殊货物（危险品、大件、冷藏品）需提前申报，否则可能被拒载或产生额外费用

### 5.3 文件与报关
- 报关资料不齐全 → 延误提箱  
- 国际港口要求不同 → 不同港口需要不同提单和出口文件

### 5.4 外部因素
- 节假日或天气（台风、雪暴、港口停运） → 可能导致船期延迟  
- 进口国政策变动 → 可能影响清关速度

### 5.5 系统操作问题
- 用户提交预订前，务必确认：
  - 起运港、目的港正确  
  - 货物类型及重量准确  
  - 附加费用和保险需求清楚  
- 外接硬盘或网络波动可能导致查询数据不同步

## 6. 使用流程示例
1. 选择起运港和目的港  
2. 选择集装箱类型及货物信息  
3. 查询运价及附加费  
4. 确认总费用 → 提交预订  
5. 专人确认 → 完成订单  
6. 全程物流跟踪与客户支持
