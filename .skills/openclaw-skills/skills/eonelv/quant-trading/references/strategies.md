# 量化交易经典策略

## 1. 双均线策略

**原理**：短期均线 > 长期均线 → 买入（金叉）
短期均线 < 长期均线 → 卖出（死叉）

**适用场景**：趋势明显的行情

```python
# 伪代码
if short_ma > long_ma and not holding:
    buy()
elif short_ma < long_ma and holding:
    sell()
```

## 2. 网格交易策略

**原理**：在价格区间划分网格，每下跌一格买入，每上涨一格卖出

**适用场景**：震荡行情

**关键参数**：
- 网格数量
- 每格金额
- 止盈位

## 3. 突破策略

**类型**：
- 高低点突破：高于N日高点买入，低于低点卖出
- 布林带突破：突破上轨买入，突破下轨卖出

**适用场景**：趋势启动阶段

## 4. 动量策略

**原理**：时间序列动量，追涨杀跌

```python
# 基于过去N日收益率
return = (current_price - price_N_days_ago) / price_N_days_ago
if return > threshold:
    buy()
```

## 5. 配对交易（套利）

**原理**：找两只相关性极高的股票，当价差偏离历史均值时反向操作

- 做多"便宜"的，做空"贵的"
- 价差回归时平仓获利
