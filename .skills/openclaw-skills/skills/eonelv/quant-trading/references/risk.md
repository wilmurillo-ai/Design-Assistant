# 风险指标详解

## 1. 夏普比率 (Sharpe Ratio)

**公式**：(策略收益 - 无风险收益) / 策略收益标准差

**评价**：
- > 1.5：优秀
- 1.0 - 1.5：良好
- 0.5 - 1.0：一般
- < 0.5：较差

## 2. 最大回撤 (Max Drawdown)

**定义**：从历史最高点到最低点的最大跌幅

**评价**：
- < 10%：极好
- 10%-20%：良好
- 20%-30%：一般
- > 30%：风险较高

## 3. 胜率 (Win Rate)

**公式**：盈利交易次数 / 总交易次数

**评价**：
- > 50%：优秀
- 40%-50%：良好
- < 40%：需要改进

## 4. 盈亏比 (Profit/Loss Ratio)

**公式**：平均盈利金额 / 平均亏损金额

**评价**：
- > 2.0：优秀
- 1.5 - 2.0：良好
- 1.0 - 1.5：一般
- < 1.0：需要改进

## 5. 卡玛比率 (Calmar Ratio)

**公式**：年化收益 / 最大回撤

**评价**：
- > 2.0：优秀
- 1.0 - 2.0：良好
- < 1.0：一般
