# 🌸 花粉监测技能 (Pollen Monitor) v1.0.0

查询中国大陆城市花粉浓度，获取过敏防护建议。

> ⚠️ **支持范围：仅支持中国大陆 53 个城市**

## 快速使用

```bash
# 查询北京（默认）
python3 ~/.openclaw/workspace/skills/pollen-monitor/pollen.py

# 查询其他城市
python3 ~/.openclaw/workspace/skills/pollen-monitor/pollen.py shanghai 上海
python3 ~/.openclaw/workspace/skills/pollen-monitor/pollen.py guangzhou 广州
```

**直辖市**：北京、上海、天津、重庆

**省会城市**：广州、深圳、成都、杭州、南京、武汉、西安、郑州、济南、沈阳、长春、哈尔滨、石家庄、太原、兰州、西宁、银川、乌鲁木齐、呼和浩特、福州、合肥、南昌、长沙、南宁、海口、昆明、贵阳、拉萨

**其他城市**：包头、鄂尔多斯、赤峰、大连、烟台、淄博、聊城、承德、沧州、保定、泊头、南充、扬州、无锡、乌海、乌兰浩特、延安、咸阳、榆林、六盘水、南阳等

> 📋 完整列表：https://m.weather.com.cn/huafen/index.html?id=101010100

## 花粉等级说明

| 等级 | Emoji | 数值 | 说明 | 防护建议 |
|------|-------|------|------|----------|
| 未检测到 | ⚪ | - | 无数据 | - |
| 很低 | 🟢 | 1-70 | 不易引发过敏 | 正常活动 |
| 低 | 🟡 | 71-150 | 轻度过敏 | 适当防护，对症用药 |
| 中 | 🟠 | 151-290 | 易过敏 | 加强防护，对症用药 |
| 高 | 🔴 | 291-520 | 易过敏 | 加强防护，规范用药 |
| 很高 | 🔴🔴 | ≥521 | 极易过敏 | 减少外出，持续规范用药 |

## 数据来源

- 中国天气网
- 全国多家医院联合监测
- 每日 08:00 更新

## 花粉季时间

- **春季**：3 月中旬 - 5 月（树木花粉：柏树、杨树、柳树、榆树等）
- **秋季**：8 月 - 9 月（草本花粉：蒿草、葎草等）

## 注意事项

- ❌ 不支持港澳台及海外城市
- ⚠️ 部分城市可能无监测数据
- 🕐 城市拼音需严格匹配（如 `xian` 不是 `xi'an`）
