# Query Patterns

根据问题类型灵活组合中英文查询，不要求固定全查，但必须覆盖高质量来源。

## 通用中文

- `"<主题> 最佳实践"`
- `"<主题> 生产环境"`
- `"<主题> 已知问题"`
- `"<主题> GitHub issue"`
- `"<主题> changelog"`
- `"<主题> 替代方案"`
- `"<主题> 值不值得用"`

## 通用英文

- `"<topic> best practices"`
- `"<topic> production ready"`
- `"<topic> known issues"`
- `"<topic> GitHub issues"`
- `"<topic> changelog"`
- `"<topic> release notes"`
- `"<topic> alternatives"`
- `"<topic> community feedback"`

## 选型 / 对比

中文：
- `"<主题> 对比 <替代方案>"`
- `"<主题> 缺点"`
- `"<主题> 适用场景"`

英文：
- `"<topic> vs <alternative>"`
- `"<topic> limitations"`
- `"<topic> tradeoffs"`

## 升级 / 兼容性

中文：
- `"<主题> breaking changes"`
- `"<主题> 升级 指南"`
- `"<主题> 回滚"`

英文：
- `"<topic> breaking changes"`
- `"<topic> migration guide"`
- `"<topic> rollback"`
- `"<topic> regression"`

## 社区口碑 / 成熟度

中文：
- `"<主题> 社区评价"`
- `"<主题> 踩坑"`
- `"<主题> 不推荐"`

英文：
- `"<topic> community feedback"`
- `"<topic> problems"`
- `"<topic> not recommended"`
