# 卡片类型详解

## 概述

知识卡片共有 10 种布局类型，分为 3 类：

- **系列框架卡**：cover（封面）、summary（总结）— 用于系列的首尾
- **内容卡**：tips、comparison、definition、steps、qa、matrix、quote、mindmap — 承载核心知识
- **自动匹配**：根据知识点的结构特征自动选择最适合的卡片类型

---

## tips（要点清单）

编号列表 + 图标，适合 3-7 个并列要点。

**布局结构**：

```
┌──────────────────────┐
│     [主标题]          │  ← 标题区 15%
│     [副标题]          │
├──────────────────────┤
│  ① [图标] 要点一      │
│     简要说明文字       │
│                      │  ← 内容区 65%
│  ② [图标] 要点二      │
│     简要说明文字       │
│                      │
│  ③ [图标] 要点三      │
│     简要说明文字       │
├──────────────────────┤
│   [角色区/装饰区]     │  ← 底部 20%
└──────────────────────┘
```

**提示词模板**：

```
[标题] - 要点清单知识卡片

布局：要点清单 — 竖向编号列表，每条配图标

标题区：
- 主标题：[大号醒目文字]
- 副标题：[补充说明，可选]

内容区：
- ① [图标描述] [要点标题]：[一句话说明]
- ② [图标描述] [要点标题]：[一句话说明]
- ③ [图标描述] [要点标题]：[一句话说明]
（重复 3-7 条）

装饰元素：[编号圆圈、分隔线、小图标]
```

## comparison（对比卡）

左右分栏或上下对比，适合两个选项/方案的并列比较。

**布局结构**：

```
┌──────────────────────┐
│     [主标题]          │  ← 标题区 12%
├──────────┬───────────┤
│          │           │
│  [A 方案] │  [B 方案]  │
│          │           │
│  · 优点1  │  · 优点1   │  ← 内容区 70%
│  · 优点2  │  · 优点2   │
│  · 缺点1  │  · 缺点1   │
│          │           │
├──────────┴───────────┤
│   [角色区/结论区]     │  ← 底部 18%
└──────────────────────┘
```

**提示词模板**：

```
[标题] - 对比知识卡片

布局：对比卡 — 左右分栏，中间有视觉分隔

标题区：
- 主标题：[A] vs [B]

左侧 — [A 方案名称]：
- [优点/特点 1]
- [优点/特点 2]
- [缺点/限制 1]

右侧 — [B 方案名称]：
- [优点/特点 1]
- [优点/特点 2]
- [缺点/限制 1]

分隔：[竖线/VS标识/渐变过渡]
底部：[结论或推荐，可选]
```

## definition（定义卡）

大标题 + 核心解释 + 示例，适合单个概念的深度解释。

**布局结构**：

```
┌──────────────────────┐
│                      │
│   [术语/概念]         │  ← 标题区 25%
│   [拼音/英文，可选]    │
│                      │
├──────────────────────┤
│                      │
│   [核心定义]          │  ← 定义区 35%
│   一段简洁的解释       │
│                      │
├──────────────────────┤
│   例子：              │
│   [具体示例]          │  ← 示例区 25%
│                      │
├──────────────────────┤
│   [角色区/装饰]       │  ← 底部 15%
└──────────────────────┘
```

**提示词模板**：

```
[标题] - 定义知识卡片

布局：定义卡 — 大号术语 + 解释 + 示例，层级分明

标题区：
- 术语：[大号醒目，突出显示]
- 注音/英文：[可选辅助信息]

定义区：
- [一段简洁清晰的定义，2-3 句话]

示例区：
- 例子：[具体场景或代码示例]

装饰元素：[引号装饰、下划线强调、标签式术语框]
```

## steps（步骤卡）

编号步骤 + 连接线 + 简要说明，适合 3-6 个有序步骤。

**布局结构**：

```
┌──────────────────────┐
│     [主标题]          │  ← 标题区 12%
├──────────────────────┤
│  Step 1 ──→          │
│  [步骤名] [说明]      │
│       │              │
│  Step 2 ──→          │  ← 内容区 70%
│  [步骤名] [说明]      │
│       │              │
│  Step 3 ──→          │
│  [步骤名] [说明]      │
├──────────────────────┤
│   [角色区/装饰]       │  ← 底部 18%
└──────────────────────┘
```

**提示词模板**：

```
[标题] - 步骤知识卡片

布局：步骤卡 — 竖向编号步骤，有连接线/箭头表示顺序

标题区：
- 主标题：[流程/教程名称]

步骤区：
- Step 1 [步骤名称]：[一句话说明]
- Step 2 [步骤名称]：[一句话说明]
- Step 3 [步骤名称]：[一句话说明]
（重复 3-6 步）

连接元素：[步骤间的箭头或连接线]
装饰元素：[编号圆圈、进度指示、小图标]
```

## qa（问答卡）

问题 + 答案的配对结构，适合 FAQ 或常见误区。

**布局结构**：

```
┌──────────────────────┐
│     [主标题]          │  ← 标题区 12%
├──────────────────────┤
│  Q: [问题文字]        │
│  ─────────────        │
│  A: [答案文字]        │  ← 内容区 70%
│                      │
│  Q: [问题文字]        │
│  ─────────────        │
│  A: [答案文字]        │
├──────────────────────┤
│   [角色区/装饰]       │  ← 底部 18%
└──────────────────────┘
```

**提示词模板**：

```
[标题] - 问答知识卡片

布局：问答卡 — Q&A 配对，问题突出，答案清晰

标题区：
- 主标题：[FAQ 主题]

问答区：
- Q1：[问题，较大字号，醒目颜色]
  A1：[答案，标准字号，解释清晰]
- Q2：[问题]
  A2：[答案]
（重复 2-4 对）

装饰元素：[Q/A 标签、分隔线、对话气泡]
```

## matrix（矩阵卡）

2×2 或多格矩阵布局，适合分类和象限分析。

**布局结构**：

```
┌──────────────────────┐
│     [主标题]          │  ← 标题区 12%
├──────────┬───────────┤
│          │           │
│  [象限1]  │  [象限2]   │
│  · 内容   │  · 内容    │  ← 矩阵区 70%
│          │           │
├──────────┼───────────┤
│          │           │
│  [象限3]  │  [象限4]   │
│  · 内容   │  · 内容    │
│          │           │
├──────────┴───────────┤
│   [角色区/轴标签]     │  ← 底部 18%
└──────────────────────┘
```

**提示词模板**：

```
[标题] - 矩阵知识卡片

布局：矩阵卡 — 2×2 象限，有轴标签

标题区：
- 主标题：[分类/分析主题]

轴标签：
- 横轴：[左] ←→ [右]
- 纵轴：[下] ←→ [上]

象限内容：
- 左上 [象限名]：[2-3 个要点]
- 右上 [象限名]：[2-3 个要点]
- 左下 [象限名]：[2-3 个要点]
- 右下 [象限名]：[2-3 个要点]

装饰元素：[象限分隔线、颜色区分、小图标]
```

## quote（金句卡）

大字金句 + 出处/解读，视觉冲击力强。

**布局结构**：

```
┌──────────────────────┐
│                      │
│                      │
│   「                  │
│     [金句内容]        │  ← 金句区 50%
│     [大号文字]        │
│                  」   │
│                      │
├──────────────────────┤
│   — [出处/作者]       │  ← 出处区 15%
├──────────────────────┤
│   [解读/点评]         │  ← 解读区 20%
├──────────────────────┤
│   [角色区/装饰]       │  ← 底部 15%
└──────────────────────┘
```

**提示词模板**：

```
[标题] - 金句知识卡片

布局：金句卡 — 大字金句居中，出处和解读在下方

金句区：
- 金句：[大号文字，居中，占据画面主要空间]
- 引号装饰：[大号装饰性引号]

出处区：
- 作者/来源：[署名]

解读区（可选）：
- [一句话解读或应用场景]

装饰元素：[大号引号、线条装饰、留白充分]
```

## mindmap（脑图卡）

中心主题向外辐射分支，适合展示知识脉络和概念关系。

**布局结构**：

```
┌──────────────────────┐
│                      │
│        [分支1]       │
│          ↑           │
│  [分支4]←[中心]→[分支2]│  ← 脑图区 80%
│          ↓           │
│        [分支3]       │
│                      │
├──────────────────────┤
│   [角色区/装饰]       │  ← 底部 20%
└──────────────────────┘
```

**提示词模板**：

```
[标题] - 脑图知识卡片

布局：脑图卡 — 中心辐射式，主题在中央，分支向外展开

中心主题：[核心概念，大号醒目，居中]

分支：
- 分支 1 [名称]：[2-3 个子要点]
- 分支 2 [名称]：[2-3 个子要点]
- 分支 3 [名称]：[2-3 个子要点]
- 分支 4 [名称]：[2-3 个子要点]
（4-8 个分支）

连接元素：[从中心向外的连线，颜色区分各分支]
装饰元素：[分支颜色编码、小图标、圆角节点]
```

## cover（封面卡）

系列首图，用于吸引注意力和建立系列主题。

**布局结构**：

```
┌──────────────────────┐
│                      │
│                      │
│   [系列标题]          │
│   [大号醒目]          │  ← 标题区 40%
│                      │
├──────────────────────┤
│                      │
│   [主视觉元素]        │  ← 视觉区 35%
│   [与主题相关的图形]   │
│                      │
├──────────────────────┤
│   [副标题/要点预告]   │  ← 副标题区 10%
├──────────────────────┤
│   [角色区/装饰]       │  ← 底部 15%
└──────────────────────┘
```

**提示词模板**：

```
[标题] - 封面知识卡片

布局：封面卡 — 大标题 + 主视觉，吸引眼球

标题区：
- 系列标题：[大号醒目，画面核心]
- 副标题：[一句话概括内容，字号较小]

视觉区：
- 主视觉元素：[与主题相关的图形/插画描述]

内容预告（可选）：
- [本系列包含的要点数量或亮点预告]

装饰元素：[品牌色块、几何装饰、序号标识"共N张"]
```

## summary（总结卡）

系列末图，回顾核心要点，引导关注/收藏。

**布局结构**：

```
┌──────────────────────┐
│     [总结标题]        │  ← 标题区 12%
├──────────────────────┤
│                      │
│  ✓ 要点 1 回顾        │
│  ✓ 要点 2 回顾        │  ← 回顾区 50%
│  ✓ 要点 3 回顾        │
│  ✓ 要点 4 回顾        │
│                      │
├──────────────────────┤
│   [行动号召 CTA]      │  ← CTA 区 20%
│   收藏 · 关注 · 转发   │
├──────────────────────┤
│   [角色区/装饰]       │  ← 底部 18%
└──────────────────────┘
```

**提示词模板**：

```
[标题] - 总结知识卡片

布局：总结卡 — 核心回顾 + 行动号召

标题区：
- 总结标题：[如"今天学到了什么？"或"核心要点回顾"]

回顾区：
- ✓ [要点 1 的一句话总结]
- ✓ [要点 2 的一句话总结]
- ✓ [要点 3 的一句话总结]
（与内容卡的核心要点对应）

行动号召：
- [引导文字，如"觉得有用？点赞收藏不迷路！"]
- [社交平台图标或互动提示]

装饰元素：[打勾图标、星星、收藏/点赞图标]
```
