# Show Archetypes and Candidate Seeds

Use this file to form a first-pass candidate list before doing web verification.

These are **starting points**, not a replacement for verification. The goal is to avoid treating every request like a blank-slate crawl.

## Archetypes

### 1. Global flagship shows

Best when the team wants:
- maximum market visibility
- executive meetings
- broad category coverage
- benchmark competition and category messaging

Common tradeoff:
- expensive
- broad audience
- easier to waste booth spend if ICP is too narrow

### 2. Regional market-entry shows

Best when the team wants:
- buyer access in one target geography
- distributors, channel partners, or local credibility
- proof of demand in a new region

Common tradeoff:
- smaller scale
- less global brand impact

### 3. Specialist / niche shows

Best when the team wants:
- tighter buyer fit
- deeper technical conversations
- better signal quality than a giant flagship event

Common tradeoff:
- lower total volume
- weaker brand halo

### 4. Launch / press / innovation shows

Best when the team wants:
- product launch visibility
- media coverage
- ecosystem attention

Common tradeoff:
- attention can skew toward hype rather than buying intent

### 5. Supply-chain / manufacturing ecosystem shows

Best when the team wants:
- OEM relationships
- component partners
- plant operations or production buyers

Common tradeoff:
- not always the best fit for software vendors unless the workflow is operationally critical

## Heuristics by Objective

### Pipeline generation

Favor:
- strong buyer concentration
- repeatable exhibitor scale
- direct ICP relevance

Discount:
- events that are famous but too broad

### Distributor / partner search

Favor:
- regional market-entry shows
- events with strong channel presence
- events known for local market operators

### Brand visibility / launch

Favor:
- flagship shows
- innovation-heavy events
- shows with strong media and analyst presence

### Competitive learning before committing booth budget

Favor:
- attend-first recommendation at a major show
- events where the user can validate booth density, competitor presence, and buyer quality before exhibiting next year

## Output Guidance

When using these archetypes:
- name the archetype if it helps explain the recommendation
- do not overfit to fame; a niche show can win if buyer fit is materially better
- always explain the tradeoff behind the recommendation
