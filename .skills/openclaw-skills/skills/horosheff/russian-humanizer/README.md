# 🇷🇺 Russian Humanizer (OpenClaw Plugin)

Продвинутый нативный плагин для OpenClaw и Cursor AI, который автоматически находит и безжалостно уничтожает "воду", канцелярит и нейросетевые штампы (AI slop) в русском тексте.

## 🔥 Ключевые возможности
- 🧠 **Продвинутая морфология и Regex-словари:** Ищет штампы во всех падежах и подсказывает, на какой человечный аналог их заменить.
- 📉 **Детектор "Правила трех" и метрики ритма:** Ловит излюбленные паттерны ChatGPT (например, "быстрый, надежный и безопасный") и вычисляет дисперсию длины предложений.
- ✨ **Авто-очистка (`auto_fix_slop`):** Вырезает вводные конструкции ("справедливости ради", "очевидно") и лишнее "мы" без изменения смысла.
- 🔌 **Интеграция с API Главреда (`analyze_glavred`):** Встроенная связь с Главредом (glvrd.ru) по 10-балльной шкале.
- 🛡 **Системный Промпт:** Используйте прилагаемый `anti_slop_prompt.md` для жестких инструкций вашему агенту "на старте".

## 📦 Установка

Менеджер плагинов OpenClaw поддерживает прямую загрузку из GitHub. В терминале, где работает шлюз, выполните:

```bash
openclaw plugins install https://github.com/Horosheff/russian-humanizer
```

Локальная установка (если вы склонировали этот репозиторий):
```bash
openclaw plugins install ./russian-humanizer
```

*(Не забудьте перезапустить шлюз OpenClaw после установки)*

## ⚙️ Настройки
Ваш `config.json` в OpenClaw может включать параметры для плагина:
```json
{
  "plugins": {
    "entries": {
      "russian-humanizer": {
        "config": {
          "mode": "business"
        }
      }
    }
  }
}
```

## 🛠 Доступные Инструменты (Agent Tools)
Сразу после установки агенту станут доступны:
1. `analyze_russian_slop` — Глубокий аудит текста (Считает 'Slop Score' и 'Ритм').
2. `auto_fix_slop` — Идеальный инструмент-редактор, который автоматически подчищает текст перед отправкой пользователю.
3. `analyze_glavred` (опционально) — Выполняет оценку через API Главреда для двойного скоринга.

## 💡 Использование в Cursor AI
1. Скачайте файл `anti_slop_prompt.md` из репозитория.
2. Скопируйте его содержимое в ваши настройки Cursor AI (**Rules for AI** или **System Prompt**).
3. Теперь ИИ будет с первой попытки писать плотным, живым текстом.
