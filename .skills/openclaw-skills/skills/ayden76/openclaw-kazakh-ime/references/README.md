# 哈萨克语输入法 (OpenClaw Kazakh IME)

一个功能完整、用户友好的哈萨克语输入法，支持阿拉伯文和西里尔文两种书写系统，可在任何网页中直接使用。

## 功能特性

### 核心功能

- **三种输入模式**：英文模式、哈萨克语阿拉伯文模式、哈萨克语西里尔文模式
- **循环切换**：按 `Ctrl+Shift+K` 循环切换三种输入模式
- **虚拟键盘**：内置可拖动的虚拟键盘，支持触屏设备
- **字符映射**：完整的英文键盘到哈萨克文字符的映射
- **组合键支持**：支持 `Ctrl+C`、`Ctrl+V` 等系统快捷键
- **智能识别**：自动适配输入元素，支持动态添加的输入框

### 用户界面

- **状态按钮**：显示当前输入模式，点击可切换模式
- **键盘切换**：一键显示/隐藏虚拟键盘
- **拖动手柄**：可自由拖动虚拟键盘到任何位置
- **视觉反馈**：模式切换时的颜色变化和动画效果
- **响应式设计**：适配不同屏幕尺寸

## 安装方法

## 使用说明

### 切换输入模式

**方法 1：快捷键**

- 按下 `Ctrl+Shift+K` 循环切换输入模式
- 顺序：英文模式 → 阿拉伯文模式 → 西里尔文模式 → 英文模式

**方法 2：状态按钮**

- 点击右下角的状态按钮切换输入模式
- 按钮颜色和文字会随模式变化：
  - 英文模式：灰色背景，显示 "قا"
  - 阿拉伯文模式：粉色渐变背景，显示 "قا"
  - 西里尔文模式：蓝色渐变背景，显示 "қ"

### 输入哈萨克语

1. **切换到目标模式**：
   - 阿拉伯文模式：输入哈萨克语阿拉伯文字符
   - 西里尔文模式：输入哈萨克语西里尔文字符
2. **使用英文键盘**：
   - 直接使用英文键盘输入，字符会自动转换为对应模式的哈萨克文字符
   - 支持大写字母映射，按 Shift 键输入大写字符

### 使用虚拟键盘

1. **显示虚拟键盘**：点击右下角的 ⌨ 按钮
2. **拖动键盘**：点击并拖动键盘顶部的 ☰ 区域
3. **输入字符**：点击虚拟键盘上的按键
4. **切换大小写**：点击虚拟键盘上的 ⇧ 按钮
5. **关闭键盘**：点击虚拟键盘顶部的 ✕ 按钮

## 技术特点

- **模块化设计**：清晰的代码结构，易于维护和扩展
- **事件委托**：使用事件委托减少事件监听器数量，提高性能
- **MutationObserver**：支持动态添加的输入元素，自动应用输入法
- **拖放功能**：虚拟键盘支持鼠标拖动，自由调整位置
- **响应式 UI**：适配不同屏幕尺寸，提供良好的用户体验
- **IIFE 封装**：避免污染全局命名空间，保持代码干净

## 字体设置

### 内置字体配置

输入法默认使用以下字体配置：

- **阿拉伯文模式**：`UKK TZK1, Microsoft Uighur, sans-serif`
- **西里尔文模式**：`UKK TZK1, sans-serif`
- **英文模式**：默认系统字体

### 自定义字体

你可以在 `openclaw-kazakh-ime.js` 文件开头的 `FONT_SETTINGS` 对象中修改字体设置：

```javascript
const FONT_SETTINGS = {
    // 阿拉伯文字体：支持哈萨克语阿拉伯文书写系统
    arabic: 'UKK TZK1, Microsoft Uighur, sans-serif',
    // 西里尔文字体：支持哈萨克语西里尔文书写系统
    cyrillic: 'UKK TZK1, sans-serif',
    // 英文字体：默认系统字体
    english: ''
};
```

### 字体推荐

- **UKK TZK1**：推荐的哈萨克语字体，支持阿拉伯文和西里尔文
- **Microsoft Uighur**：支持哈萨克语阿拉伯文书写系统的字体
- **其他选项**：如果系统中没有推荐字体，可以使用其他支持哈萨克语的字体
