## Recommendation Heuristics

- Prefer `ubuntu-linux` as the default when the user wants "a Linux playground" without more detail.
- Prefer `ubuntu-desktop` when the user needs a GUI rather than just a shell.
- Prefer `kali-linux` for security, pentest, or offensive-security distro requests.
- Prefer `rhel` for enterprise Linux requests unless the user explicitly names another distro.
- Prefer `alpine` for minimal image, small footprint, or stripped-down environment requests.
- Prefer `archlinux` for rolling-release comparisons.
- When the user asks "which distro should I use", recommend at most three Linux distro playgrounds and briefly explain the tradeoff.
