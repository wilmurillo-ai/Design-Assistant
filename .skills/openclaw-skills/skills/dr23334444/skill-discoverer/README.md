# Skill Discoverer

**Skill Discoverer** 是一款帮助你 **自动巡查 Skill 广场**、**智能推荐** 与 **风险评估** 的 OpenClaw 插件。它能够：

- 定期（每日/每周/每月）扫描 Skill 广场，发现最新上架的 skill；
- 根据你的工作背景（读取 `USER.md` 与最近 7 天的 `memory/*.md`）自动生成搜索关键词；
- 将搜索结果按照 **工具型** 与 **增强型** 分类，给出分数、风险标记（✅ / △ / ⚠️ / 🚨）以及安装选项（A/B/C/D）；
- 支持创建 **cron** 任务，把同一搜索关键词的巡查自动化并把结果推送到当前聊天渠道。

## 主要触发词

```
帮我看看有没有新 skill
有没有新 skill
最近有新 skill 吗
skill 广场有什么新东西
智能推荐 skill
定时发现新 skill
每天帮我看新 skill
skill discoverer
any new skills
```

## 使用流程概览

1. **首次运行** → 自动保存渠道信息并询问巡查频率。
2. **手动/自动触发** → 根据关键词或智能推荐模式搜索 Skill 广场。
3. **生成报告** → 按照严格的排版铁律展示每个 skill 的分数、风险与安装选项。
4. **用户确认** → 只在用户明确回复后执行安装、学习或创建 cron。
5. **后续巡查** → 若用户选择定时任务，系统会在约定时间自动重复步骤 2‑4。

## 安全与权限

- 所有写文件操作仅限于 `~/.openclaw/workspace/skill-discoverer/`（配置、日志、已知 skill 列表）。
- **不** 会执行任何需要 `sudo` 或系统级权限的命令。
- **所有** 安装与增强型 skill 的写入都必须二次确认，防止意外修改 `MEMORY.md`、`SOUL.md` 等核心文件。
