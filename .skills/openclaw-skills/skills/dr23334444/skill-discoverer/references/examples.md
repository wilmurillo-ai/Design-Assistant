## 用户指定安装 — 有影响（增强型）

```

功能：让 Agent 从被动响应变为主动预判需求，包含 WAL Protocol 和自主 Cron。

影响项：
  ⚠️ 会改变交互方式：Agent 会主动发起对话（当前是纯被动响应）
  ⚠️ 会新增 cron 任务（每日蒸馏 + 每周提炼），需确认与现有 cron 无冲突
  🚨 不可逆改动：覆盖 SOUL.md 第 3 节"工作风格"

是否确认安装？[是 / 否]
```
