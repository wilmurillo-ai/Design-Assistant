## For ClawHub Users

Install via:
```bash
openclaw skills install https://github.com/loudmouthedmedia/context-bridge
```

Or manual:
1. Download archive from GitHub releases
2. Extract to `~/.openclaw/skills/context-bridge/`
3. Run `./setup.sh`
