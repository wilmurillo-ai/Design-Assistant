# Citation Style Guide

## APA 7th Edition (most common in social sciences & management)

### In-text
- Single author: (Kowalski, 2023) or Kowalski (2023)
- Two authors: (Kowalski & Nowak, 2023)
- Three+ authors: (Kowalski et al., 2023)
- Direct quote: (Kowalski, 2023, s. 45) — use "s." for Polish, "p." for English
- Multiple sources: (Kowalski, 2020; Nowak, 2023) — alphabetical, semicolon

### Bibliography entry patterns
- Book: Nazwisko, I. (Rok). *Tytuł książki*. Wydawnictwo.
- Chapter: Nazwisko, I. (Rok). Tytuł rozdziału. W: I. Redaktor (red.), *Tytuł książki* (s. XX–XX). Wydawnictwo.
- Journal: Nazwisko, I. (Rok). Tytuł artykułu. *Nazwa Czasopisma*, *nr tomu*(nr zeszytu), s. XX–XX. https://doi.org/xxxxx
- Web: Nazwisko, I. (Rok, dzień miesiąc). Tytuł strony. Nazwa serwisu. https://url (dostęp: DD.MM.RRRR)
- Standard/Norm: ISO/IEC 22123-1:2023. *Cloud computing — Concepts and terminology — Part 1*. ISO.

### Common APA mistakes to flag
- Missing DOI when available
- Inconsistent use of "i" vs "&" (Polish: "i", English: "&")
- Title case vs sentence case confusion
- Missing issue number
- "Ibidem" or "op. cit." — NOT used in APA

## Harvard Style

### In-text
- Same as APA: (Author, Year) pattern
- Key difference: page refs use "p." always

### Bibliography
- Similar to APA but: titles not italicized for articles, different punctuation
- Book: Nazwisko, I., Rok. *Tytuł*. Miejsce: Wydawnictwo.

## Chicago (Notes & Bibliography)

### In-text
- Uses footnotes, not parenthetical citations
- First citation: full reference in footnote
- Subsequent: shortened form (Nazwisko, *Krótki tytuł*, s. XX)

### Bibliography
- Book: Nazwisko, Imię. *Tytuł*. Miejsce: Wydawnictwo, Rok.
- Inverted name order only for first author

## Polish Academic Conventions

### General rules
- Foreign terms in *italics*, not "quotes"
- Abbreviations defined on first use, then abbreviation only
- Tables: "Tabela 1." with title above, "Źródło:" below
- Figures: "Rysunek 1." with caption below, "Źródło:" below
- "Źródło: opracowanie własne" for original tables/figures
- "Źródło: opracowanie własne na podstawie: [citation]" for derived works
- Page ranges: use en-dash (s. 45–67), not hyphen

### Common issues in Polish academic texts
- Mixing Polish and English citation styles
- Using "tamże" / "ibidem" — check if style allows it
- Inconsistent transliteration of foreign names
- Missing "dostęp:" date for web sources
- Not specifying edition for standards (e.g., ITIL 4 vs ITIL v3)
