# 文件处理技能模板

## 核心脚本

### file-ops.py
通用文件操作：读写、重命名、移动、删除、统计

### convert.py
文件格式转换：PDF ↔ DOCX, CSV ↔ JSON, 图片格式转换

### batch-process.py
批量处理：遍历目录、批量操作、进度显示

## 支持的格式

| 格式 | 输入 | 输出 | 依赖 |
|------|------|------|------|
| CSV | ✅ | ✅ | pandas |
| JSON | ✅ | ✅ | 无 |
| PDF | ✅ | ✅ | pdfplumber |
| 图片 | ✅ | ✅ | pillow |

## 错误处理

| 错误类型 | 处理策略 |
|---------|---------|
| 文件不存在 | 提示用户重新指定 |
| 权限错误 | 提示权限问题 |
| 格式不支持 | 列出支持的格式 |
