# 海地革命独立费用历史研究 Skill

## 描述
这个skill用于研究和分析海地革命（1791-1804年）后海地被迫向法国支付独立费用的历史背景、原因和影响。

## 历史背景

### 海地革命概述
- 时间：1791-1804年
- 地点：圣多明各（现海地）
- 结果：海地成为世界上第一个黑人共和国

### 独立费用（1825年协议）
- **金额**：1.5亿法郎（后减至9000万法郎）
- **支付期限**：1825-1947年（历时122年）
- **原因**：法国以承认海地独立为条件要求赔偿

## 技能功能

### 1. 历史查询
- 查询海地革命的关键事件时间线
- 了解独立费用协议的具体条款
- 分析支付对海地经济发展的影响

### 2. 数据分析
- 计算独立费用的实际价值（考虑通货膨胀）
- 比较海地与其他拉美国家的独立过程
- 分析长期债务对国家发展的制约

### 3. 国际关系分析
- 法国要求赔偿的法律和政治依据
- 国际社会对此事件的反应
- 现代赔偿运动的进展

## 使用方法

```bash
# 查询海地革命基本信息
haiti-revolution timeline

# 计算独立费用详细信息  
haiti-revolution payments --year 1825-1947

# 分析经济影响
haiti-revolution impact --economic --social
```
