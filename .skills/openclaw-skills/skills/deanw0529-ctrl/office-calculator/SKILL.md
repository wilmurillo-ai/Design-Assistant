# Office Calculator · 办公室面积测算器

> 「先定面积，再做分区，最后设计详图——省去反复测算的高额咨询费。」
> 作者：王至德（至德老师）｜基于20年全国办公选址经验

## 背景

本工具基于20年全国办公楼选址经验总结，将复杂的面积需求转化为可量化的参数模型。帮助企业、行政人员快速算出"租多大"，避免租赁后再发现不够用、或租大了浪费。

**本工具输出结果含完整面积分解，用户可直接拿着结果去跟业主谈判，或交给设计师出分区图。**

## 触发词（AI 识别）

当用户提到以下任意关键词时，使用本技能：
- 办公室面积 / 租赁多大 / 需要多大办公室
- 工位数 / 多少人 / 多少平米
- 办公室规划 / 选址 / 办公区设计
- 行政测算 / 办公选址

## 参数说明

| 参数 | 必填 | 默认值 | 说明 |
|------|------|--------|------|
| `--seats` | ✅ | - | 总人数（含所有人：业务、行政、财务等） |
| `--finance` | ❌ | 0 | 财务部人数（财务室独立占用面积） |
| `--seat-size` | ✅ | - | 单工位面积（㎡），支持 1.44（1.2×1.2m）或 1.96（1.4×1.4m） |
| `--meeting` | ❌ | 0 | 全部会议室总面积（㎡） |
| `--reception` | ❌ | 0 | 客户接待区域面积（㎡） |
| `--pantry` | ❌ | 0 | 茶水间、卫生间等辅助区总面积（㎡） |
| `--finance-room` | ❌ | 0 | 财务办公室独立占用面积（㎡） |
| `--has-manager-office` | ❌ | 否 | 是否设置经理独立办公室 |
| `--manager-area` | ❌ | 12 | 经理办公室面积（㎡），建议 12~15㎡ |

## 自然语言调用示例

直接用自然语言描述参数，AI 会解析并调用脚本：

```
"50人，3个财务，工位1.2×1.2，会议室40㎡，接待室20㎡，茶水间15㎡，财务室25㎡，租多大？"
"45人办公室规划，有经理独立办公室12㎡，帮我算算需要多少面积"
"我们30人，要多大办公室？财务3人，会议室大概30㎡"
"120人，大工位1.4×1.4，会议室80㎡，接待40㎡，茶水30㎡，财务50㎡"
```

## 典型场景示例

**场景1：科技公司，50人，标准配置**
```
总人数50 / 财务3人 / 1.2×1.2m工位 / 会议室40㎡ / 接待室20㎡ / 茶水间15㎡ / 财务室25㎡
→ 建议租赁：约 297㎡
```

**场景2：成长型企业，45人，有经理独立办公室**
```
总人数45 / 财务3人 / 经理独立办公室12㎡ / 1.2×1.2m工位 / 会议室40㎡ / 接待室20㎡ / 茶水间15㎡ / 财务室25㎡
→ 建议租赁：约 312㎡
```

**场景3：大型办公室，120人，大工位配置**
```
总人数120 / 财务5人 / 1.4×1.4m工位 / 会议室80㎡ / 接待室40㎡ / 茶水间30㎡ / 财务室50㎡
→ 建议租赁：约 700㎡
```

## 规划流程建议

```
Step 1：用本工具测算面积 → 知道"要租多大"
Step 2：按比例分区（工位区 / 会议室 / 接待 / 茶水 / 财务）
Step 3：找设计师出详细布局图
          ↑ 此时面积已锁定，不用反复改图
          ↑ 省去第一步反复测算的高额咨询费
```

## 输出说明

- **RECOMMENDED TOTAL**：最低建议租赁面积（㎡）
- **+10% buffer**：考虑实际楼盘得房率（通常65-75%），建议在此基础上增加10-15%选房
- **完整面积分解**：工位区、会议室、接待室、茶水间、财务室、管理层办公室分项列出
- 本测算不含：电梯、楼道、洗手间等楼宇公共区域
