# Master Resume Template

Use this template when creating a master resume for the user.

## Structure

```markdown
# [Full Name]

## Contact Information
- **Email:** [email@example.com]
- **Phone:** [+XX XXX XXX XXXX]
- **LinkedIn:** [linkedin.com/in/username]
- **Portfolio/Website:** [optional]

## Professional Summary

[2-3 sentences that capture:
- Current role/expertise
- Key differentiator (what makes them unique)
- Career focus/objective
]

Example:
> Technical Product Lead & AI Specialist with a background in East Asian Studies. A hybrid profile combining Japanese linguistic fluency with data engineering and AI-Assisted Rapid Prototyping. Focus: Leveraging Context Engineering and AI tools to build end-to-end technical solutions.
```

## Guidelines for Bullet Points

**Good bullets:**
- Led cross-functional team of 5 to deliver project 2 weeks early
- Reduced processing time by 40% through automation
- Built React web app using AI-assisted code generation

**Bad bullets (avoid):**
- Responsible for team management
- Hard worker with proven track record
- Results-oriented professional

## Tips

1. **Quantify when possible** - Numbers make achievements concrete
2. **Focus on outcomes** - Not just what you did, but the impact
3. **Use action verbs** - Led, Built, Created, Reduced, Improved
4. **Keep it scannable** - Recruiters spend 6-10 seconds on first pass
5. **Tailor for ATS** - Include relevant keywords from job descriptions
