## 技能输入参数

支持JSON格式传入参数，适配ClawHub平台调用规则，参数明细如下：

```json
{
    "lotteryType": "",
    "issueNum": "",
    "lotteryDate": "",
    "winningNumbers": [],
    "specialNumber": null,
    "historyDataCount": 50
}
```

- lotteryType：彩种类型（必填，如双色球、大乐透）

- issueNum：开奖期号（必填，格式统一）

- lotteryDate：开奖日期（必填，格式：YYYY-MM-DD）

- winningNumbers：开奖基础号码（必填，数组格式）

- specialNumber：特殊号码（选填，蓝球/后区号码）

- historyDataCount：复盘历史期数（选填，默认50期）

## 核心功能模块

### 1. 基础开奖数据统计

自动解析当期开奖号码，计算核心基础指标，包含期号、开奖日期、完整开奖号码、号码和值、号码跨度、奇偶分布占比、大小号分布、区间分布、012路占比，全方位梳理当期开奖数据特征，形成基础数据台账。

### 2. 冷热号&遗漏值分析

基于设定的历史复盘期数，统计各号码出现频次，划分热号、温号、冷号三大梯队；同步计算单号码当前遗漏期数、平均遗漏期数、最大遗漏期数，梳理号码遗漏规律，辅助历史数据复盘。

### 3. 中奖走势特征分析

复盘多期开奖走势，分析连号、同尾号、重号、斜连号等走势特征，统计各类走势出现频次，输出多期走势变化规律，仅做历史走势回溯，无预测导向。

### 4. 专项号码分析

针对双色球蓝球、大乐透后区等特殊号码，单独做大小、奇偶、遗漏、走势专项分析，细化数据维度，满足精细化数据复盘需求。

### 5. 标准化报告输出

自动整合所有统计、分析结果，生成简洁易懂的文字报告，格式统一、重点清晰，支持直接查看、留存复盘，无需手动整理数据。

## 技能执行流程

1. 接收ClawHub平台传入的当期开奖数据参数

2. 校验参数完整性、格式合规性，异常数据自动提示

3. 调取对应历史开奖数据，完成各项指标计算

5. 生成合规、无预测导向的标准化数据报告

6. 向平台返回分析结果，完成单次技能调用
