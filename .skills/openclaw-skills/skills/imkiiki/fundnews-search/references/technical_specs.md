# 技术实现细节

## 抓取策略

### 核心原则：宽搜索 + 严过滤

**搜索阶段**：使用广泛关键词，确保不遗漏
**过滤阶段**：严格执行过滤规则，确保质量

---

### 搜索关键词策略

#### 之前（❌ 太严格）
```bash
# 只搜索"公募基金"，漏掉大量新闻
公募基金 site:chnfund.com after:2026-03-09
```

**问题**：
- 漏掉"证监会 基金"类政策新闻
- 漏掉"证券投资基金"类正式表述
- 漏掉"基金业"类行业新闻

#### 当前策略（✅ 宽搜索）
```bash
# 搜索"基金"，然后过滤
基金 site:chnfund.com after:2026-03-09
基金 site:cs.com.cn after:2026-03-09
基金 site:zqrb.cn after:2026-03-09
基金 site:stcn.com after:2026-03-09
基金 site:cnstock.com after:2026-03-09
```

**优势**：
- ✅ 捕获所有基金相关新闻
- ✅ 包括政策、行业、产品等各层面
- ✅ 通过后置过滤保证质量

---

### 过滤策略（后置严过滤）

#### 第一层：过滤非公募基金

| 过滤类型 | 关键词 | 示例 |
|---------|--------|------|
| 私募基金 | `私募`、`私募基金`、`私募股权` | 私募证券基金、私募股权基金 |
| 产业基金 | `产业基金`、`引导基金`、`政府基金` | 京津冀产业基金、政府引导基金 |
| 股权基金 | `股权投资基金`、`创投基金`、`风投基金` | 创业投资基金、风险投资基金 |
| 房产基金 | `房产基金`、`地产基金`、`REITs`（非公募） | 越秀房产基金 |
| 海外基金 | `海外基金`、`离岸基金`、`外币基金` | 外币计价基金 |

#### 第二层：过滤单一主体

| 过滤类型 | 关键词 | 示例 |
|---------|--------|------|
| 单一产品发行 | `发行中 `、` 正在发行`、`新基金发行` | 华夏中证 A500ETF 联接发行中 |
| 单一产品公告 | `停牌 `、` 复牌`、`分红公告`、`开放申购` | 嘉实标普油气 ETF 停牌 |
| 单一公司系统 | `系统上线 `、` 系统优化`、`投资交易系统` | 恒生 O45 在长盛基金上线 |
| 单一公司活动 | `足球赛 `、` 邀请赛`、`鲲鹏杯`、`活动` | 鹏扬基金鲲鹏杯足球赛 |

#### 第三层：过滤非实质内容

| 过滤类型 | 关键词 | 示例 |
|---------|--------|------|
| 短期涨跌 | `涨逾 `、` 跌幅`、`收益率`、`净值涨` | 多只 ETF 涨逾 4% |
| 资金流向 | `资金净流入 `、` 资金净流出`、`获融资净买入` | 279 只 ETF 获融资净买入 |

## 执行流程

```
1. 解析用户查询
   └─ 确定时间范围

2. Exa 宽搜索（五大平台）
   ├─ "基金 site:stcn.com after:日期"
   ├─ "基金 site:cs.com.cn after:日期"
   ├─ "基金 site:zqrb.cn after:日期"
   ├─ "基金 site:cnstock.com after:日期"
   └─ "基金 site:chnfund.com after:日期"

3. 三层过滤
   ├─ 第一层：过滤非公募基金（私募、产业、股权等）
   ├─ 第二层：过滤单一主体（产品发行、公告、活动等）
   └─ 第三层：过滤非实质内容（涨跌、流向、人事等）

4. 质量检查
   ├─ 是否为行业整体/监管政策？
   ├─ 是否影响公募基金行业？
   └─ 是否具有长期意义？

5. 格式化输出
   ├─ 按日期分组
   ├─ 按平台排序
   ├─ 按时间倒序
   └─ 生成 Word 文档（过去七天/日期范围）
```

## 注意事项

1. **宽搜索**：搜索词用"基金"，不用"公募基金"
2. **严过滤**：严格执行三层过滤规则
3. **质量优先**：宁可少抓，不可错抓
4. **监管政策优先**：证监会、基金业协会政策新闻必须抓取
5. **行业整体现状**：规模、发行、趋势等整体数据必须抓取
6. **单一产品过滤**：单一产品发行、公告、活动全部过滤
