# Lighting Patterns Reference

## Natural Light

### Golden Hour
```
Time: 1 hour after sunrise, 1 hour before sunset
Color: Warm orange/amber tones
Quality: Soft, directional
Characteristics: Long shadows, warm skin tones, magic hour glow
Prompt: "golden hour light, warm amber tones, soft directional sunlight, long gentle shadows"
```

### Blue Hour
```
Time: 20-40 min before sunrise, after sunset
Color: Cool blue tones, warmer accents in city lights
Quality: Soft, even
Characteristics: Twilight atmosphere, city lights pop, cool palette
Prompt: "blue hour, cool blue tones, twilight sky, city lights starting to glow"
```

### Overcast / Cloudy
```
Quality: Soft, diffused, even
Characteristics: No harsh shadows, natural fill, flattering for portraits
Color: Neutral, slightly cool
Prompt: "soft overcast light, diffused daylight, no harsh shadows, natural even illumination"
```

### Direct Sun
```
Quality: Hard, directional, high contrast
Characteristics: Strong shadows, punchy, warm
Pitfalls: Harsh shadows on face, squinting
Prompt: "direct sunlight creating defined shadows, high contrast, punchy warm tones"
```

### Window Light
```
Quality: Soft, directional, diffused through glass
Characteristics: Natural, flattering, variable
Variations: Direct window (harder), sheer curtains (softer)
Prompt: "soft window light from the left, natural side lighting, subtle shadows"
```

## Studio Lighting Setups

### One Light (Classic)
```
Key light only
Simple: Front light (flat, even)
Butterfly: Above and centered (dramatic, glamour)
Side: 90° from subject (graphic, high contrast)
Prompt: "single softbox key light at 45 degrees, soft shadows, clean look"
```

### Two Light (Standard)
```
Key light + Fill light
Key: Main illumination
Fill: Shadow reducer (usually weaker, opposite side)
Ratio: 2:1 for subtle, 4:1 for dramatic
Prompt: "soft key light from left at 45 degrees, subtle fill from right, professional portrait lighting"
```

### Three Light (Full)
```
Key + Fill + Rim/Hair light
Key: Main subject lighting
Fill: Shadow fill
Rim: Edge/separating light from background
Hair: Top accent
Prompt: "main key light, soft fill, strong rim light creating separation from background"
```

### Butterfly / Paramount
```
Light: Directly above and in front
Shadow: Small under nose (butterfly shape)
Characteristics: Glamour, beauty, flattering
Used: Fashion, beauty, portrait
Prompt: "butterfly lighting from above, glamour portrait, soft shadows under chin"
```

### Rembrandt Lighting
```
Light: 45° to side and above
Shadow: Triangle of light on cheek (nose shadow touches cheek shadow)
Characteristics: Dramatic, artistic, dimensional
Named after: Dutch painter
Prompt: "Rembrandt lighting, dramatic triangle of light on cheek, 45-degree angle, cinematic"
```

### Loop Lighting
```
Light: 30-45° to side, slightly above
Shadow: Small oval under nose (loop)
Characteristics: Universal flattering, slight dimension
More versatile than Rembrandt
Prompt: "loop lighting, small loop shadow under nose, slightly above, flattering for most faces"
```

### Split Lighting
```
Light: 90° to side (directly side-on)
Shadow: Half face lit, half in darkness
Characteristics: Dramatic, graphic, split personality
Prompt: "split lighting, half face illuminated, half in shadow, dramatic side light"
```

### Broad Lighting
```
Light: On side of face toward camera
Shadow: Opposite side
Characteristics: Makes face appear wider, fuller
Better for: Narrower faces
Prompt: "broad lighting, lit side facing camera, wider appearance, full illumination"
```

### Short Lighting
```
Light: Opposite side from camera
Shadow: Majority of face in shadow
Characteristics: Slimming, dramatic, mysterious
Better for: Broader faces
Prompt: "short lighting, shadow side facing camera, slimming effect, mysterious mood"
```

### Clamshell
```
Light: One above/front, one below/front (反射)
Creates: Minimal shadows, very even
Beauty: Favors skin, minimal imperfection
Prompt: "clamshell lighting, ring light or reflector below, even illumination, beauty setup"
```

## Advanced Lighting Effects

### Rim / Hair Light
```
Placement: Behind subject
Direction: Toward camera (through subject edges)
Effect: Separation from background, glowing edges
Color: Can differ from key (blue sky, orange sun)
Prompt: "strong rim light creating halo effect, blue fill from sky, subject separated from background"
```

### Catchlights
```
Effect: Light reflections in eyes
Shape: Window reflection (portrait), round (ring light)
Position: 10-11 o'clock or 1-2 o'clock in iris
Prompt: "catchlights in eyes, window reflection, lively eyes"
```

### Practical / Available Light
```
Source: Visible lamps, screens, candles
Characteristics: Environmentally contextual, narrative
Color: Varies (warm tungsten, cool blue screens)
Prompt: "practical lamp in frame, warm tungsten glow, intimate interior lighting"
```

### Treetment / Gobo
```
Technique: Light through pattern/shape
Effect: Dappled light, pattern shadows
Nature: "sunlight through trees"
Prompt: "dappled light through tree leaves, soft shadow patterns on subject"
```

### Volumetric Light
```
Technique: Atmosphere (fog, haze, dust)
Effect: Light rays visible in air
Natural: Morning mist, dust motes, fog
Prompt: "volumetric light rays through morning mist, god rays, atmospheric depth"
```

### Light Painting
```
Technique: Long exposure, moving light source
Effect: Trails, written words, sculpted light
Subject: Usually stationary
Prompt: "light painting effect, neon trails in dark studio, creative long exposure photography"
```

## Color Gels & Gobos

| Effect | Description |
|--------|-------------|
| Orange gel | Warm up tungsten, match sunset |
| Blue gel | Cool down daylight, night effect |
| Green gel | Cancel green cast (fluorescents) |
| CTO | Color temperature orange (full/half) |
| CTB | Color temperature blue |
| Grid/egg crate | Control light spread |
| Colorful gels | Creative color accents |
