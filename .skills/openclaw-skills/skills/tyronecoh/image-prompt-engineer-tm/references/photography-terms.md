# Photography Terminology Reference

## Camera & Lens

| Term | Correct Usage | Vague Equivalent |
|------|--------------|-----------------|
| **Focal length** | "85mm lens", "24mm wide angle" | "big lens", "zoom lens" |
| **Depth of field** | "shallow DOF at f/1.4", "deep focus f/11" | "blurry background" |
| **Bokeh** | "creamy bokeh from f/1.8" | "nice blur" |
| **Wide angle** | "24mm for environmental portrait" | "goes really wide" |
| **Telephoto** | "85mm for flattering portraits" | "long lens" |
| **Macro** | "90mm macro for product details" | "close-up lens" |

## Aperture

| Effect | How to Describe |
|--------|----------------|
| Shallow DOF | "f/1.4 for creamy bokeh and subject isolation" |
| Deep DOF | "f/11 for sharp focus throughout scene" |
| Starburst | "f/16 with point light creating starburst" |
| Sweet spot | "f/5.6 for sharpest lens performance" |

## Shutter Speed

| Effect | How to Describe |
|--------|----------------|
| Freeze motion | "1/1000s to freeze fast action" |
| Motion blur | "1/15s for intentional blur" |
| Light trails | "30s exposure for light painting" |
| Silky water | "2s for smooth water effect" |

## Composition

| Term | Meaning |
|------|---------|
| **Rule of thirds** | Subject at intersection points |
| **Golden ratio** | Spiral composition guide |
| **Leading lines** | Lines drawing eye to subject |
| **Negative space** | Empty space around subject |
| **Symmetry** | Mirror image balance |
| **Frame within frame** | Natural frames (windows, doors) |
| **Depth layering** | Foreground, midground, background |
| **Headroom** | Space above subject's head |
| **Look room** | Space in direction subject faces |
| **Center framing** | Subject dead center |
| **Diagonal composition** | Dynamic angled lines |

## Lighting Direction

| Pattern | Description |
|---------|-------------|
| **Front light** | Light directly facing subject |
| **Side light** | Light from 45° or 90° to subject |
| **Back light** | Light behind subject (rim lighting) |
| **Rembrandt** | Triangle of light on cheek (3/4 view) |
| **Butterfly** | Light from above (classic Hollywood) |
| **Split** | Half face lit, half in shadow |
| **Loop** | Small shadow under nose (between Rembrandt and butterfly) |
| **Broad** | More of face lit (opposite of short) |
| **Short** | Less of face lit (3/4 view away from light) |

## Lighting Quality

| Quality | Characteristics |
|---------|-----------------|
| **Hard light** | Sharp shadows, high contrast, direct source |
| **Soft light** | Soft shadows, gradual transition, diffused |
| **Specular** | Mirror-like reflections, highlights |
| **Diffuse** | Even, wrap-around, minimal shadows |
| **Volumetric** | Light visible in air (fog, dust, haze) |
| **Directional** | Clear shadow direction |
| **Ambient** | Available light, environmental |

## Color Temperature

| Description | Technical |
|-------------|----------|
| Warm golden | "5600K daylight with orange gel" |
| Cool blue | "6500K with blue gel" |
| Neutral | "Pure white balance, no color cast" |
| Mixed | "Warm key light, cool fill from window" |

## Post-Processing Terms

| Effect | Description |
|--------|-------------|
| **High key** | Bright, low contrast, light tones |
| **Low key** | Dark, high contrast, deep shadows |
| **HDR** | High dynamic range, detail in shadows/highlights |
| **Silhouette** | Subject as dark outline against light |
| **Split toning** | Different colors in shadows vs highlights |
| **Dodging/burning** | Selective lightening/darkening |
| **Clarity** | Midtone contrast, definition |
| **Vibrance** | Intelligent saturation (less on skin) |
| **Saturation** | Overall color intensity |
| **Exposure compensation** | +1EV or -1.5EV |

## Subject Poses (Portrait)

| Pose | Description |
|------|-------------|
| **Straight on** | Facing camera directly |
| **Three-quarter** | Body at 45° to camera |
| **Profile** | Side view |
| **Hands on face** | Intimate, editorial |
| **Over shoulder** | Looking back at camera |
| **Seated** | Casual, relaxed |
| **Standing** | Full length or 3/4 |
| **Looking away** | Contemplative, cinematic |
| **Laughing** | Genuine, warm |
| **Serious** | Intense, dramatic |

|-------|------------|
