# Tavily Search 配置指南

## 📋 简介

Tavily 是一个专门用于 AI 应用的智能搜索引擎，提供精准、最新的新闻和信息搜索结果。

**投资研究报告技能已集成 Tavily**，可以自动搜索：
- 公司股价动态和投资资讯
- 产品和技术相关信息  
- 行业政策和市场热点
- 并对搜索结果进行情感分析和股价影响评估

## 🔑 获取 API Key

### 1. 注册账号
访问 [https://app.tavily.com/](https://app.tavily.com/) 注册账户

### 2. 创建 API Key
- 登录后进入 Dashboard
- 点击 "API Keys" 
- 创建新的 API Key

### 3. 价格计划
- **免费**: 每天 1000 次搜索额度（足够个人使用）
- **Pro**: $49/月，无限制搜索

## ⚙️ 配置步骤

### 方式一：编辑 .env 文件（推荐）
```bash
# 编辑 OpenClaw 的 .env 文件
vim ~/.openclaw/.env
```

添加一行：
```bash
TAVILY_API_KEY=tvly-xxxxxxxxxxxxxxxxxxxxxx
```

保存后重启 OpenClaw 即可生效。

### 方式二：设置环境变量
```bash
# Linux/Mac
export TAVILY_API_KEY="tvly-xxxxxxxxxxxxxxxxxxxxxx"

# Windows PowerShell
$env:TAVILY_API_KEY="tvly-xxxxxxxxxxxxxxxxxxxxxx"
```

## 📊 搜索功能说明

### 自动搜索内容

每次运行股票分析时，会自动搜索：

1. **公司动态** (最近 7 天)
   - 股票代码 + 公司名称 + "股价 投资 最新"
   - 搜索深度：advanced（深度搜索）
   - 返回数量：前 3 条

2. **产品与业务**
   - 公司名称 + "产品 技术 主营业务"
   - 搜索深度：basic（快速搜索）
   - 返回数量：前 3 条

3. **行业动态**
   - 行业关键词 + "行业 政策 市场"
   - 搜索深度：basic
   - 返回数量：前 3 条

## ❗ 常见问题

### Q: 没有设置 API Key 会怎样？
A: 分析程序会跳过新闻搜索部分，其他技术分析功能正常工作。

### Q: 搜索失败怎么办？
A: 
1. 检查网络连接
2. 确认 API Key 是否正确
3. 查看 Tavily 服务状态

### Q: 如何提高搜索准确性？
A:
1. 使用 advanced 搜索深度
2. 添加更具体的关键词
3. 调整搜索范围天数
