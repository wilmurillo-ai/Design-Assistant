# Feishu API Documentation Reference

This directory contains Feishu API documentation references.

## Files

### feishu_api.md
Summary of commonly used Feishu APIs and their key parameters.

### doc_urls.txt
Collection of Feishu API documentation URLs for quick reference.

## Maintaining This Directory

When adding new API scripts:
1. Add the API documentation URL to `doc_urls.txt`
2. If the API is commonly used, add a summary to `feishu_api.md`

## Note

For the most up-to-date API documentation, always refer to:
- [Feishu Open Platform](https://open.feishu.cn/)
- [Feishu API Documentation](https://open.feishu.cn/document/server-docs/)
