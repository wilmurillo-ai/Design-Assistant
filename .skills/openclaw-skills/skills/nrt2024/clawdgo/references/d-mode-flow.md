# D 模式教学讲解——推荐主题与弱项联动

## 目标定位（v1.3.1）

- D 是深度讲解与复盘模式。
- 用户可按推荐主题或自由问题发问。

## 开场模板（固定）

```text
━━━━━━━━━━━━━━━━━━━━━━━━
🦞 D 教学模式 — 深度讲解
━━━━━━━━━━━━━━━━━━━━━━━━
我来为你深度拆解任何安全概念。推荐主题：

🔥 热门主题
  1. 钓鱼邮件识别的7个黄金规则
  2. 公共WiFi的真实风险（Evil Twin详解）
  3. CEO欺诈：为什么聪明人也会中招
  4. 密码管理最佳实践

📚 基于你的弱项
  5. [{weakest_1}] ...
  6. [{weakest_2}] ...

💡 自由提问
  直接发送任何安全问题、新闻或疑惑

发送编号（1-6）或直接输入你的问题
发「返回」→ 退出教学模式
━━━━━━━━━━━━━━━━━━━━━━━━
```

## 弱项联动

- 若有 profile，`5/6` 必须动态替换为 weakest 维度。
- 若无 profile，保留通用弱项强化主题。

## 严禁

- 不得切换成考试问答（A/B/C 选项题）。
- 不得输出攻击实施细节。
