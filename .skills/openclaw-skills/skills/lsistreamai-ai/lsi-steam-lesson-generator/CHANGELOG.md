# AI-Integrated STEAM Lesson Generator - Changelog

## v1.0.0 (2026-04-04)

### Initial Release

**Features:**
- Generate complete lesson plans with AI tool integration
- Support for all subjects (Math, Science, English, Chinese, STEAM, Visual Arts, etc.)
- Primary and Secondary grade levels
- Bilingual output (Chinese/English)
- 6 AI tools integrated: GLM5, Canva AI, Gamma, HeyGen, Blender, Tripo3D.ai

**Output includes:**
- Teacher Quick Start guide (for AI-novice teachers)
- Lesson Overview with objectives and materials
- AI Tool Integration instructions
- Step-by-step Teaching Steps
- Student Activities with AI tools
- Assessment Rubric
- AI-Assisted Assessment prompts (for grading help)
- Teacher Notes (differentiation, EDB alignment)

**Example lessons included:**
- Primary 4 Science: Plant Life Cycle (40 min)
- Secondary 2 Visual Arts: Digital Portrait Design (70 min)
- Secondary 2 English: Descriptive Writing (70 min)

**EDB Alignment:**
- Lessons are formatted for EDB "AI teaching resource" submission
- Schools can use outputs for HK$500,000 AI funding grant requirement
