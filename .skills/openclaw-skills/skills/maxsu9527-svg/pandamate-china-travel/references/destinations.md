# 中国目的地深度攻略（20+城市）

## 一线/超大城市

### 北京 �🇳
**必去景点**：
- 故宫（紫禁城）：提前7天预约，¥60
- 天安门广场：免费，登城楼¥15
- 长城：八达岭/慕田峪（建议后者，人少）
- 颐和园：¥30，皇家园林
- 天坛：¥34，建筑之美
- 798艺术区：文艺青年轻奢地

**美食推荐**：
- 北京烤鸭：全聚德（大栅栏老店体验差）、便宜坊、大董
- 炸酱面：海碗居
- 铜锅涮肉：聚宝源、东来顺
- 小吃：簋街夜市

**交通**：
- 地铁：覆盖完善，¥3-10/次
- 机场：首都机场（大兴较远）
- 火车站：北京站/北京西/北京南

**住宿建议**：
- 王府井/天安门：景点集中
- 三里屯：使馆区，国际化
- 五道口：大学区，年轻氛围

**注意事项**：
- ⚠️ 故宫每周一闭馆
- ⚠️ 长城距市区远，报团需谨慎
- ⚠️ 空气污染时口罩必备

---

### 上海
**必去景点**：
- 外滩：夜景最佳，步行/观光隧道
- 陆家嘴：东方明珠¥199、环球金融中心¥180
- 豫园+城隍庙：传统园林+小吃
- 田子坊：石库门艺术区
- 南京路步行街：购物
- 上海博物馆：免费，周一闭馆

**美食推荐**：
- 本帮菜：绿波廊、南翔馒头店
- 生煎：小杨生煎
- 小笼：鼎泰丰（贵）、佳记
- 国际美食：日料、意大利菜（日本人聚居区古北）

**交通**：
- 地铁：最方便，¥3-10/次
- 磁悬浮：浦东机场→龙阳路¥50（体验速度）
- 机场：浦东（国际）、虹桥（国内）

**住宿建议**：
- 外滩/人民路：景观好
- 静安寺/法租界：洋气、文艺
- 虹桥：交通枢纽，商务

---

### 广州
**必去景点**：
- 珠江夜游：游船¥100-300
- 沙面岛：租界风情
- 上下九步行街：岭南建筑
- 广州塔（小蛮腰）：¥150-450
- 陈家祠：岭南建筑艺术¥10
- 白云山：城市绿肺¥5

**美食推荐**（粤菜发源地）：
- 早茶：点都德、陶陶居、广州酒家
- 烧腊：烧鹅、叉烧（随身入）
- 肠粉：银记
- 糖水：文明路糖水铺
- 海鲜：黄沙水产交易市场

**交通**：
- 地铁：覆盖广，¥2-10/次
- 有轨电车：广州塔→万胜围¥2
- 机场：白云机场（距市区远，地铁1h）

**住宿建议**：
- 天河区：天河城/太古汇
- 越秀区：北京路/公园前
- 荔湾区：沙面/上下九

---

### 深圳
**必去景点**：
- 世界之窗：各国微缩景观¥220
- 欢乐谷：大型游乐园¥230
- 东部华侨城：大侠谷/茶溪谷
- 海上世界：滨海休闲
- 深圳湾公园：骑行¥0

**特点**：
- 年轻城市，无历史古迹
- 购物天堂（万象城、海岸城）
- 香港后花园（多个口岸通港）

**交通**：
- 地铁：方便，覆盖广
- 机场：宝安机场（距市区适中）
- 火车站：深圳站（罗湖）、深圳北（高铁）

---

### 成都
**必去景点**：
- 宽窄巷子：老成都风情
- 锦里：夜景美
- 武侯祠：三国文化¥50
- 杜甫草堂：诗圣故居¥50
- 大熊猫繁育基地：必去¥55
- 青城山：道教名山¥60（前山）/¥20（后山）
- 都江堰：古代水利¥90

**美食推荐**（美食之都）：
- 火锅：蜀大侠、小龙坎、大龙燚
- 串串：钢管厂五区
- 冒菜：芙蓉树下
- 兔头：双流老妈兔头
- 小吃：宽窄巷子、祥和里

**交通**：
- 地铁：覆盖完善
- 机场：双流（近）/天府（远但新）
- 高铁：成都东/成都西/成都南

**住宿建议**：
- 春熙路/太古里：市中心
- 锦里/武侯祠：游客观光
- 高新区：商务、现代

## 推荐路线

### 7天经典（北京+上海）
```
D1-3：北京（故宫、长城、颐和园）
D4：北京→上海高铁
D5-7：上海（外滩、豫园、田子坊）
```

### 14天深度（华北+华东+西南）
```
D1-4：北京（+天津一日）
D5-7：上海（+苏州/杭州当日）
D8-14：成都（+九寨沟/峨眉山）
```
