# 中国签证与入境完整指南（2026最新版）

## 免签政策一览

### 1. 240小时过境免签（最常用）

**适用国家**（55个）：
- 欧洲：英国、法国、德国、意大利、西班牙、俄罗斯（团体）、瑞士、瑞典、挪威、丹麦、荷兰、比利时、奥地利、希腊、芬兰、爱尔兰、葡萄牙、捷克、匈牙利、斯洛文尼亚
- 亚太：澳大利亚、新西兰、日本、韩国、新加坡、马来西亚、泰国、印尼
- 美洲：美国、加拿大、巴西、阿根廷、墨西哥
- 中东：阿联酋、卡塔尔、沙特阿拉伯
- 及其他符合条件国家

**可入境城市**（20+）：
| 城市 | 口岸类型 | 可活动范围 |
|------|---------|-----------|
| 上海 | 浦东/虹桥 | 上海市+苏州/嘉兴 |
| 北京 | 首都/大兴 | 北京市+天津 |
| 广州 | 白云 | 广州市+佛山/肇庆 |
| 成都 | 双流/天府 | 成都市+绵阳 |
| 杭州 | 萧山 | 杭州市+绍兴 |
| 深圳 | 宝安+陆路口岸 | 深圳市+惠州 |
| 西安 | 咸阳 | 西安市+咸阳 |
| 昆明 | 长水 | 昆明市+玉溪 |
| 厦门 | 高崎/翔安 | 厦门市+漳州 |
| 重庆 | 江北 | 重庆市 |
| 南京 | 禄口 | 南京市 |
| 大连 | 周水子 | 大连市 |
| 青岛 | 胶东 | 青岛市 |
| 武汉 | 天河 | 武汉市 |
| 长沙 | 黄花 | 长沙市 |

**申请条件**：
1. 持有有效期≥3个月的护照
2. 持有已确认的第三国机票/船票（陆路需要车票）
3. 来自55个免签国之一
4. 没去过中国限制入境地区

**注意事项**：
- ⚠️ 离境必须是入境城市或城市群内
- ⚠️ 不能提前太久入境（航班日期±1天内）
- ⚠️ 团体旅游也可申请，需旅行团担保

### 2. 15天免签（互免签证）

**适用国家**（9个）：
- 新加坡（2024年更新）
- 文莱
- 日本（2024年恢复）
- 毛里求斯
- 塞舌尔
- 巴巴多斯
- 巴哈马
- 斐济
- 格林纳达

**条件**：
- 护照有效期≥6个月
- 来华目的为旅游/商务/探亲
- 不超过15天

### 3. 72小时过境免签（已大幅缩减）
仅剩少数城市保留，建议直接用240h政策。

## 落地签（VOA）

**适用情况**：
- 紧急旅行（来不及办签证）
- 所在国无中国使领馆
- 团体旅游（旅行团担保）

**落地签口岸**（仅限）：
- 北京首都机场
- 上海浦东/虹桥机场
- 广州白云机场
- 深圳机场
- 珠海机场（仅拱北口岸陆路）

**费用**：$30-60 USD（因国籍而异）

**所需材料**：
1. 护照（有效期≥6个月）
2. 往返机票
3. 酒店预订确认
4. 签证申请表（机场领取）
5. 照片（部分机场需要）
6. 资金证明（部分情况）

## 签证类型选择

| 签证类型 | 适用场景 | 有效期 | 停留期 | 办理时间 |
|---------|---------|-------|-------|---------|
| L（旅游） | 自由行、团体游 | 3个月-10年 | 15-180天 | 4-15天 |
| M（商务） | 商务访问、会议 | 3个月-10年 | 30-90天 | 4-10天 |
| F（交流） | 学术交流、访问 | 3个月-5年 | 30-90天 | 4-10天 |
| Q1（探亲） | 探亲（≥180天） | 1-5年 | 180天 | 2-4周 |
| Q2（探亲） | 探亲（<180天） | 1-10年 | 90天 | 1-2周 |
| S1（工作） | 外国人在华工作 | 90天-5年 | 90天-5年 | 2-4周 |
| X1（学习） | 长期留学 | 90天-5年 | 180天-5年 | 2-4周 |

## 申请流程（旅游签证L）

### 中国签证申请服务中心

**所需材料**：
1. 护照原件（有效期≥6个月，至少2页空白）
2. 签证申请表（在线填写+打印）
3. 照片（白底，33×48mm）
4. 往返机票预订单
5. 酒店预订确认
6. 在职/在读证明（可选）

**申请步骤**：
1. 访问 **https://www.visaforchina.cn**（中国签证申请服务中心）
2. 选择你的国家/地区
3. 在线填写申请表
4. 预约递交时间
5. 前往签证中心递交材料
6. 缴费（可刷卡/现金）
7. 领取回执
8. 取护照（可邮寄）

**费用**：$130-170 USD（因国籍、办理时间而异）

## 新政策动态（2025-2026）

### 已生效变化
1. **马来西亚互免签证**（2024年12月）：15天免签
2. **欧洲国家更多适用240h**：范围扩大
3. **在线填表优化**：减少填写项
4. **落地通关便利化**：多城市机场支持自助通关

### 即将生效变化
1. **入境卡电子化**：部分口岸已试点
2. **签证在线预约**：覆盖更多使领馆

## 常见问题

**Q: 护照过期但有有效签证怎么办？**
A: 旧护照+新护照+签证转移，或者重新申请签证。

**Q: 可以用落地签进入后转其他签证吗？**
A: 不可以，签证类型在入境时确定。

**Q: 240h过境免签可以多次入境吗？**
A: 每次入境需要新的机票/车票，不能重复使用。

**Q: 随身带多少现金合适？**
A: 建议$200-500 USD等值人民币，部分城市机场兑换方便。
