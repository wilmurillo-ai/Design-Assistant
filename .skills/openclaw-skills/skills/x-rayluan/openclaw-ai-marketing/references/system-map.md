# System Map

## Goal
Run an AI marketing team as a repeatable operating system rather than disconnected agent tasks.

## Main flow
1. Hunter research intel
2. Pain Map
3. Selection layer
4. marketing-assets sync
5. JK packaging (optional but useful)
6. Elon social publishing
7. Tony blog publishing
8. Jenny lifecycle / activation
9. Peter QA closeout
10. Karen truth gate
11. Mission Control truth update

## Main assets
- company positioning / brand source of truth
- marketing-assets hook / angle / CTA / proof files
- same-day receipts
- Mission Control state files

## What good looks like
- upstream research shapes downstream messaging
- no lane claims completion without evidence
- social, blog, and lifecycle all point at real conversion goals
- truth files stay aligned with actual delivery
