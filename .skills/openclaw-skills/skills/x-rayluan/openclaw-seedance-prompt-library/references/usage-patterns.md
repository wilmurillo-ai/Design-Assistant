# Usage patterns

## Use this library well

Do not treat the repository as a copy-paste vault.

Use it to:
- find patterns that already work
- understand motion / camera / pacing language
- build stronger prompts for a new request

## Search workflow

1. Start with the user's target:
   - subject
   - genre
   - camera feel
   - energy level
   - realism vs stylization
2. Search the English and Chinese README files
3. Pull 3-8 examples max
4. Compare what repeats:
   - camera movement
   - scene transitions
   - action verbs
   - lighting language
   - duration / aspect ratio notes
5. Synthesize a new prompt

## Rewrite workflow

When the user gives a rough idea, improve it by tightening:
- subject clarity
- scene clarity
- motion sequence
- camera path
- style consistency
- ending beat

## Good response pattern

### Inspiration request
- Shortlist
- Why each example matters
- One merged prompt suggestion

### Rewrite request
- Original
- Improved version
- 3 short upgrade notes

### Batch generation request
Offer 3 versions:
- safe
- stylized
- experimental

## Bilingual pattern

If the user may generate in Chinese or English interfaces, provide both.
Translate for generation quality, not word-for-word literal fidelity.

## Avoid

- dumping dozens of prompts with no curation
- overfitting to copyrighted characters unless the user explicitly accepts the risk
- bloated adjective stacks with no motion logic
- pretending an adapted community prompt is fully original
