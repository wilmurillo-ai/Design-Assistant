# Daily Loop

## Morning
- Hunter delivers before downstream windows
- marketing-assets updated
- JK package prepared if used
- Elon morning lane publishes
- Tony starts daily publish target

## Midday
- Mission Control truth refresh
- Jenny activation / recovery / lifecycle checks
- Peter verifies live blog state for same-day Tony work when available
- Karen checks for truth drift

## Evening
- Elon evening lane publishes
- Tony backlog / publish target check
- Karen truth gate
- Mission Control final same-day closure or explicit gap state

## Rule
The clock can move, but dependency order should not break.
