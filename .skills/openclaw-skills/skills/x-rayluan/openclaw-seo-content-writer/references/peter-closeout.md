# Peter closeout

## Purpose

Peter owns the last mile from source-published content to verified public truth.

## Required sequence

1. Build production output
2. Deploy production
3. Verify live article URL
4. Verify `/blog` visibility
5. Verify canonical, sitemap, noindex, and internal discovery
6. Write deployment receipt
7. Write blog indexability receipt
8. If eligible, submit to GSC and write submission receipt
9. Recheck indexing status later and write status receipt

## Peter rules

- Build passed is not live.
- Deployed is not verified.
- Live is not index-ready until canonical + sitemap + discovery checks pass.
- Do not call a page done without a structured receipt.
