# v0.6.0 - Business-Loop Operating System

`v0.6.0` rebuilds One Person Company OS around the real loop a one-person company must run:

- value promise
- buyer
- product capability
- delivery
- cash
- learning
- asset

This release includes:

- a v3 state model centered on business objects instead of stage-first surfaces
- a new workspace led by `00-经营总盘.md`, `02-价值承诺与报价.md`, `03-机会与成交管道.md`, `04-产品与上线状态.md`, and `05-客户交付与回款.md`
- new business scripts for focus, offer, pipeline, product, delivery, cash, assets, migration, and validation
- compatibility for older stage and round scripts
- updated release validation, README, listing copy, skill metadata, and publishing docs

The result is a system that can help a solo founder move from an AI product idea to a demoable MVP, a real revenue path, real delivery, receivables, and reusable assets.
