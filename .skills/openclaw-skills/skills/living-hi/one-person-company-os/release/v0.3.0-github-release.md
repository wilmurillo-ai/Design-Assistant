# v0.3.0 - Round-Based Solo Company OS

This release is a full V2 rebuild of `one-person-company-os`.

The old shape was closer to a startup operating manual with heavier documents and a weekly review loop.  
The new shape is a faster AI-native control system for solo companies:

- 创建公司
- 启动回合
- 推进回合
- 触发时才校准
- 瓶颈变化时切阶段

## What Changed

### 1. The skill now has a much clearer product shape

The new `SKILL.md` is centered on five concrete user intents instead of a broad “company OS” description:

- create the company
- start a round
- advance the round
- calibrate the round
- transition stages

### 2. Chinese-first is now the default for Chinese users

Chinese users now get:

- Chinese workspace names
- Chinese file names
- Chinese role names
- Chinese operating language

Only technical identifiers stay in their original form when needed.

### 3. The operating loop is now round-based

The main loop is no longer “weekly review”.

The new loop is:

- define one current round
- move that round forward
- enter calibration only when triggered
- transition stages only when the bottleneck changes

### 4. The role system is smaller and more practical

The V2 rewrite introduces `总控台` as the central coordinating role and reduces the default starter role set.

The default skeleton is now:

- 创始人
- 总控台
- 产品负责人
- 工程负责人

Other roles are activated only when needed.

### 5. The local workflow has been rebuilt

New and updated scripts now support:

- Chinese workspace initialization
- role brief generation
- round start
- round update
- round calibration
- stage transition
- release validation

## Included In This Release

- rewritten `SKILL.md`

## Upgrade Summary

If you used the previous version, the main mental model has changed:

- before: setup + weekly review
- now: setup + one current round + trigger-based calibration

This is the intended product direction for V2.
