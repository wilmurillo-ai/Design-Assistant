# v0.5.7 - AI-Era Fast Loop And Lightweight Stages

`v0.5.7` makes the product feel closer to how an AI-native solo company should actually run.

The key change is a shift in the visible workflow:

- validate demand quickly
- ship the smallest useful MVP
- launch narrowly
- collect feedback and production reality
- improve before scaling growth

This release adds:

- `12-AI时代快循环.md` inside generated workspaces
- lighter stage definitions that behave like bottleneck labels instead of bureaucracy
- updated founder-start guidance and template copy across the workspace
- aligned GitHub, ClawHub, and release materials around the fast-loop positioning
- release validation for the new fast-loop workspace guide

The internal control system still keeps rounds, calibration, and stage transitions.
The difference is that founders now see a simpler, faster operating model first.
