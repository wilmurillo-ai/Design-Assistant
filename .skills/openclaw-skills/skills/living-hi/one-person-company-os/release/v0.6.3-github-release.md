# One Person Company OS v0.6.3

`v0.6.3` is a cleanup patch on top of `v0.6.2`.

## What Changed

- removed leftover `当前状态.json.lock` artifacts from founder workspaces after successful state saves
- kept the merged-save and file-lock protection introduced in `v0.6.2`

## Why This Matters

The concurrency protection is important, but the lock file itself should not leak into the user-visible workspace.

This patch keeps the safer persistence model while restoring a clean founder-facing file surface.
