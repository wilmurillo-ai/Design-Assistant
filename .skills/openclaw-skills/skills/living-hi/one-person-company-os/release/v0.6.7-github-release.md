# One Person Company OS v0.6.7

`v0.6.7` adds a download-friendly reading layer on top of the existing workspace model.

## What Changed

- every generated workspace now includes a localized HTML reading directory
- Chinese founders get `阅读版/00-先看这里.html`
- English founders get `reading/00-start-here.html`
- the core dashboard, offer, pipeline, product, delivery, cash, asset, and deliverable-overview pages now export as HTML reading pages
- markdown remains the editable working source and numbered DOCX files remain the formal deliverable format

## Why This Matters

The package already produced a good working workspace, but the downloaded result still looked too source-heavy for direct founder review.

This patch closes that gap by turning the workspace into a three-layer output:

- markdown for editing
- HTML for reading
- DOCX for formal delivery
