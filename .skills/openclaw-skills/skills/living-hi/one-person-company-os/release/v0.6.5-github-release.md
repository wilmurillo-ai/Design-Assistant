# One Person Company OS v0.6.5

`v0.6.5` is a workspace-surface and trust patch on top of `v0.6.4`.

## What Changed

- localized the full founder-visible workspace surface for Chinese and English founders
- moved machine state to the hidden stable path `.opcos/state/current-state.json`
- localized role brief filenames, support work surfaces, artifact paths, and flow files
- added layout harmonization so older workspaces can be cleaned up into the new structure
- strengthened release validation to check Chinese and English workspace separation explicitly

## Why This Matters

The previous release fixed the direction-first contract, but the filesystem still told a different story:

- English founders could still get Chinese-visible file names
- Chinese founders could still see English business directories
- internal state could still leak into the founder-visible surface

That left the product feeling partially translated instead of intentionally localized.

This patch fixes that mismatch.
The founder now gets a workspace that matches the language they actually use, while automation keeps one hidden stable state path behind the scenes.
