# B站UP主成长助手

**B站UP主全能运营工具** - 热门监控、数据分析、竞品研究、内容策划

## 功能

- 📊 **热门监控** - 实时追踪B站热门视频，生成热门报告
- 🔍 **UP主分析** - 分析对标UP主数据、粉丝趋势、内容风格
- 📈 **视频分析** - 单视频播放分析、互动数据
- 💡 **内容策划** - 基于热门数据给出内容建议
- 📝 **报告生成** - 自动生成B站运营日报/周报

## 使用方法

```
监控B站热门视频
分析B站UP主[UP主名称]
看看这个视频[BV号]
生成B站运营日报
```

## 技术

- 需要配合 browser 或 agent-reach 工具获取数据
- 数据存储在 /tmp/bilibili-data/
- 支持 OpenClaw + macOS/Linux
