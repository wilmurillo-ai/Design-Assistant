📊 OpenClaw PPT生成器示例

技能简介

这是一个乔布斯风格的PPT生成器，可以将用户输入的讲稿自动转换成精美的HTML演示文稿。

🎨 极简设计

采用乔布斯风格的极简主义设计，专注于内容本身

📱 响应式布局

完美适配手机、平板和电脑屏幕

⌨️ 多种控制

支持键盘、鼠标、触摸等多种操作方式

使用方法

在OpenClaw中，你可以这样使用：

帮我创建一个关于人工智能发展趋势的PPT
内容如下：
人工智能正在改变世界...
...

技能特点

  • 智能内容分割：自动将长文本分割成合适的幻灯片
  • 多种幻灯片类型：支持标题页、列表页、内容页等
  • 可定制样式：可调整颜色、字体等样式参数
  • 全屏演示：支持全屏模式，适合正式演示
  • 离线使用：生成的HTML文件可离线使用
立即试用

技术实现

这个技能基于以下技术：

  • Node.js：后端处理逻辑
  • HTML5/CSS3：前端展示界面
  • JavaScript：交互逻辑
  • OpenClaw Skills API：技能集成接口
