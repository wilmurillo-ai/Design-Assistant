# 封面模板说明

本目录用于存放小红书笔记封面模板资源。

## 封面设计规格

- **尺寸：** 1080 × 1440 px（3:4 竖版）
- **格式：** PNG / JPG / WebP
- **文件大小：** 建议 < 5MB

## 封面类型模板

### 1. 文字型封面（干货/教程类）
```
┌──────────────────────┐
│                      │
│   [主标题大字]        │
│   [副标题/补充说明]   │
│                      │
│   [装饰元素/图标]     │
│                      │
└──────────────────────┘
```
- 背景色：纯色或渐变
- 主标题：白色/深色大字，居中或偏上
- 适合：教程、攻略、清单类笔记

### 2. 拼图型封面（对比/合集类）
```
┌──────────┬──────────┐
│          │          │
│  图片 A   │  图片 B   │
│          │          │
├──────────┴──────────┤
│     [标题文字]        │
└──────────────────────┘
```
- 上方：产品/效果对比图
- 下方：标题文字条
- 适合：对比测评、好物合集

### 3. 人像型封面（穿搭/美妆类）
```
┌──────────────────────┐
│  [人物照片]           │
│                      │
│  ┌────────────────┐  │
│  │ [标题文字]      │  │
│  └────────────────┘  │
└──────────────────────┘
```
- 人物占 60-70%
- 底部或侧边叠加文字卡片
- 适合：穿搭、美妆、生活方式

### 4. 前后对比型
```
┌──────────┬──────────┐
│  BEFORE  │  AFTER   │
│  [效果前] │  [效果后] │
├──────────┴──────────┤
│   [标题 + 结果数据]   │
└──────────────────────┘
```
- 左右对比，视觉冲击强
- 底部标注关键数据
- 适合：改造、减肥、装修前后

### 5. 信息图表型
```
┌──────────────────────┐
│  [图表/数据可视化]    │
│                      │
│  [标题文字]           │
│  [来源/说明]          │
└──────────────────────┘
```
- 信息密度较高
- 配色统一协调
- 适合：科普、数据分析、行业报告

## 使用 AI 生成封面

推荐使用 Image Generation 技能，根据笔记类型选择合适的封面风格：

1. 干货/教程 → 文字型封面
2. 种草/推荐 → 人像型或拼图型
3. 测评/对比 → 前后对比型
4. 数据/科普 → 信息图表型

## 注意事项

- 封面文字不要太小（建议 ≥ 48pt）
- 标题不超过 15 个字
- 配色与账号整体调性一致
- 避免使用模糊或低质量图片
