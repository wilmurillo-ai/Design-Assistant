## 爆款标题公式

### 公式 1：数字 + 痛点 + 解决方案

```
[N个] + [常见痛点] + [解决方案/效果]

示例：
- 3步搞定秋冬底妆卡粉，干皮必看！
- 5款百元以内口红，黄皮也能找到本命色
- 7天跟着吃瘦了3斤，懒人减脂餐安排上
```

### 公式 2：身份 + 场景 + 结果

```
[目标人群] + [具体场景] + [结果/好处]

示例：
- 学生党也能学会的平价穿搭公式
- 上班族快手早餐，15分钟搞定一周
- 租房改造 | 500元让出租屋变成ins风
```

### 公式 3：反常识 + 揭秘

```
[反常识观点] + [揭秘/真相/才知道]

示例：
- 别再这样洗脸了！90%的人都洗错了
- 原来煮米饭加点这个会这么香！
- 护肤品不是越贵越好，这3个平价才是真香
```

### 公式 4：时间限定 + 效果承诺

```
[时间范围] + [效果/变化]

示例：
- 坚持早睡30天，皮肤的变化惊呆了
- 7天养成马甲线，跟着做就对了
- 1个月从0开始学插画，我的进度条拉满了
```

### 公式 5：对比 + 选择建议

```
[A vs B] + [对比结论/选择建议]

示例：
- 贵妇vs平价，这些护肤品真的差很多吗？
- 有机vs普通，超市里该怎么选蔬菜？
- 空调vs除湿机，南方人到底该买哪个？
```

### 公式 6：情绪共鸣 + 共鸣点

```
[情绪/感受] + [引发共鸣的场景]

示例：
- 终于找到了！适合亚洲人的眼影盘
- 谁懂啊！这个方法真的太好用了
- 看哭了！这才是我们普通人该看的装修攻略
```

### 公式 7：灵魂拷问 + 回答

```
[引发思考的问题] + [回答/解决方案]

示例：
- 为什么你的化妆总是显脏？3个原因帮你找到
- 为什么自制咖啡总不如咖啡店？问题出在这
- 为什么你瘦不下来？可能是这5个习惯在作怪
```

### 公式 8：经验总结 + 教训

```
[N次/年] + [踩坑/失败经历] + [总结]

示例：
- 踩了100个坑后总结的买房经验，看完少走弯路
- 用了50款面霜才总结出的抗老攻略
- 3年新媒体运营的血泪教训
```

### 公式 9：清单/合集

```
[场景/主题] + [数量] + [好物/推荐合集]

示例：
- 办公室必备的10个提升幸福感好物
- 2024年最值得买的8款面霜
- 双11必囤的15件日用品
```

### 公式 10：地域 + 推荐

```
[城市/地区] + [主题] + [推荐/攻略]

示例：
- 上海最好吃的10家brunch合集
- 杭州周末去哪玩？这5个地方美到哭
- 成都本地人都爱去的小众咖啡馆
```

## 正文结构模板

### 通用结构（适用大部分笔记）

```
【Hook（开头钩子）】
一句话抓住注意力，让用户想继续看下去。
可以是：提问、数据、反常识、共鸣、悬念。

【正文段落 1：场景/痛点】
描述用户熟悉的场景或痛点，建立共鸣。

【正文段落 2：核心内容/方法】
笔记的核心价值所在，提供干货、方案或产品介绍。

【正文段落 3：细节/案例/补充】
补充细节、举例佐证，增强可信度。

【CTA（结尾行动号召）】
引导用户互动：提问、投票、收藏、关注。
```

### 步骤教程类结构

```
Hook: "保姆级教程来啦！手把手教你xxx"

Step 1: xxx
  - 要点说明
  - 注意事项

Step 2: xxx
  - 要点说明
  - 注意事项

Step 3: xxx
  - 要点说明
  - 注意事项

💡 避坑提醒：xxx
💡 进阶技巧：xxx

CTA: "学会了吗？评论区告诉我~"
```

### 产品测评类结构

```
Hook: "用了X天/测了X款，终于找到了yyds"

产品 1: [名称]
  价格：xxx
  优点：xxx
  缺点：xxx
  适合人群：xxx
  评分：⭐⭐⭐⭐⭐

产品 2: [名称]
  ...

总结对比：
| 产品 | 价格 | 适合 | 评分 |
|------|------|------|------|

CTA: "你最种草哪个？评论区讨论~"
```

## Emoji 使用指南

### 常用 Emoji 映射

| 用途 | Emoji |
|------|-------|
| 标记要点 | ✅ ❌ ⚠️ 💡 |
| 评分/推荐 | ⭐ 👑 🏆 💎 |
| 情感表达 | ❤️ 💕 🥺 😭 🤩 😍 |
| 方向/指示 | 👉 👆 👇 📌 |
| 动作 | 💪 🏃‍♀️ 🎯 ✨ 🙋 |
| 分类/编号 | 1️⃣ 2️⃣ 3️⃣ 📚 🛒 🍽️ |
| 装饰 | 🌈 🌸 🔥 🎀 💫 |

### 使用原则

1. **适量**：每篇笔记 3-8 个 emoji
2. **一致**：同类笔记使用相似的 emoji 风格
3. **功能**：emoji 应有明确用途（强调、分隔、装饰）
4. **不过度**：避免每句都有 emoji，影响阅读
5. **相关性**：emoji 与内容主题匹配
