# Learning Mentor - 通用学习带教模式

> 采用苏格拉底式教学法，一步步引导学习，不直接给答案

## 功能特点

- 🎯 **算法训练营模式**：LeetCode 30 题系统学习
- 🐍 **Python 学习模式**：对比 JavaScript 教学
- 📝 **自动生成文档**：每个学习单元完成后生成代码和文档
- 🤔 **苏格拉底式提问**：引导思考而非直接给答案
- ✅ **验证反馈**：等学生写完代码再验证

## 使用方法

### 算法学习
```
/learning-mentor algorithm
```

### Python 学习
```
/learning-mentor python
```

## 教学原则

- 一步一步教导，每步确认理解后再继续
- 先提问引导思考，再给提示
- 等学生写完代码后再验证
- 每个学习单元完成后生成文档和代码

## 适用人群

- 正在学习算法和数据结构的开发者
- 想系统学习 Python 的 JavaScript 开发者
- 希望通过实践巩固知识的学习者
