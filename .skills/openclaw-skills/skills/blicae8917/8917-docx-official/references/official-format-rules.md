# 公文格式规则速查

## 适用范围
用于将 Markdown 或已有文本内容转换为符合中国党政机关公文/正式文件格式要求的 `.docx` 或 `.pdf`。

## 标题层级映射

| Markdown | 公文层级 | 自动编号示例 | 字体 |
|----------|---------|------------|------|
| `#` | 文档标题 | 居中显示，不编号 | 方正小标宋_GBK 小二号 |
| `##` | 一级标题 | 一、主要目标 | 方正黑体_GBK 三号 |
| `###` | 二级标题 | （一）数据要素运营 | 方正楷体_GBK 三号 |
| `####` | 三级标题 | 1. 公共数据授权运营 | 方正仿宋_GBK 三号加粗 |
| `#####` | 四级标题 | （1）数据确权登记 | 方正仿宋_GBK 三号 |
| 普通段落 | 正文 | — | 方正仿宋_GBK 三号 |

## 关键规则

1. 标题不要手写编号，脚本会自动生成。
2. `-t` 参数设置标题时，Markdown 中的 `#` 标题仅作为备用来源。
3. 正文首行缩进 2 字符，固定行距 28 磅。
4. 页边距：上 3.3cm / 下 3.7cm / 左右 2.8cm。
5. 页码使用 `-1-` 格式，页面底部居中。

## 常见参数

| 参数 | 说明 |
|------|------|
| `-o` | 输出文件路径（.docx 或 .pdf） |
| `-t` | 文档主标题 |
| `-s` | 副标题 / 发文机关（支持 `\n` 换行） |
| `-a` | 署名落款 |
| `-d` | 日期 |
| `--pdf` | 同时生成 PDF |
| `--pdf-only` | 仅保留 PDF |
| `--pdf-path` | 自定义 PDF 输出路径 |

## 依赖
- Python 3
- `python-docx`
- LibreOffice（仅 PDF 输出需要）
