## 🔧 使用方法

### 方法 1: 语音/文字指令

```
发头条，标题"我的文章"，内容"正文内容..."
```

### 方法 2: 使用发布脚本

```bash
# 一键发布
./publish-toutiao.sh "文章标题" "正文内容" "图片关键词" "封面关键词"

# 测试功能
./test-publish.sh
```

### 方法 3: 手动执行

```bash
# 1. 打开发布页面
browser open https://mp.toutiao.com/profile_v4/graphic/publish

# 2. 获取 snapshot
browser snapshot refs=aria

# 3. 执行各步骤...
```
