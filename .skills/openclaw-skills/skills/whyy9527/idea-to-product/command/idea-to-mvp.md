# /idea-to-mvp

Turn a rough product idea into one small MVP that can generate real feedback fast.

## Task

Take an idea and output:

1. the core user problem
2. current constraints
3. the smallest feedback loop
4. what not to build yet
5. one MVP file target

## Rules

- one idea -> one MVP
- prefer the shortest real feedback loop
- reject platform sprawl
- define "not now" explicitly

## Output

- MVP candidate
- smallest feedback loop
- non-goals
- next file to create
