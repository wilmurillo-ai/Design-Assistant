## 预估方式（元）

- `estimated_price_yuan = input_tokens * rate_in + output_tokens * rate_out`
- 注意区分公网/内网价格与输入长度区间

其中：

- `rate_in`：输入单价（元/token），由上面的“推理服务（输入）”单价（元/百万 Tokens）除以 1,000,000 得到
- `rate_out`：输出单价（元/token），由上面的“推理服务（输出）”单价（元/百万 Tokens）除以 1,000,000 得到

输入长度区间以 `input_tokens` 计算，表中标注的是“千 token”区间：

- `input_tokens <= 128_000` 使用 `[0, 128]` 千 token 的 `rate_in/rate_out`
- `128_000 < input_tokens <= 256_000` 使用 `(128, 256]` 千 token 的 `rate_in/rate_out`
