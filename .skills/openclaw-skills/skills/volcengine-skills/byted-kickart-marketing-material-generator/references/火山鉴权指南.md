# 火山鉴权指南

## 如何验证火山鉴权是否成功
1. 执行`python authentication.py read`，获取环境变量中的火山引擎的App Key和Secret Key。
2. 如果成功获取到App Key和Secret Key，说明火山鉴权已成功配置。
3. 如果未获取到App Key和Secret Key，说明火山鉴权配置有误。请检查是否按照指南正确配置了App Key和Secret Key。

## 如何配置火山鉴权
1. 要求用户已经注册火山引擎账号，并且具有相应的权限。
2. 用户需要在火山引擎控制台中创建一个新的应用，获取到App Key和Secret Key。
3. 用户需要告知获取到的App Key和Secret Key，然后执行`python authentication.py write --app-key <app_key> --secret-key <secret_key>`。
4. 执行`python authentication.py read`，验证是否成功写入环境变量。
