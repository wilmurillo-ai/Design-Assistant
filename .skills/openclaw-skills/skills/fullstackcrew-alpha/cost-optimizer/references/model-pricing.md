## 成本场景模拟

### 场景 A：日常开发（无优化）
- 模型：全部使用 claude-opus-4-6
- 日均 token：500K（输入 400K + 输出 100K）
- 日均成本：400K/1M × $15 + 100K/1M × $75 = $6.00 + $7.50 = **$13.50/天**
- 月均成本：**~$405/月**

### 场景 B：日常开发（智能路由 balanced）
- Heartbeat (20%): deepseek/v3 → $0.084/1M
- 简单操作 (30%): haiku → $1.44/1M
- 代码生成 (40%): sonnet → $5.40/1M
- 复杂任务 (10%): opus → $27.00/1M
- 日均 token：500K
- 日均成本：500K × (0.2×$0.084 + 0.3×$1.44 + 0.4×$5.40 + 0.1×$27.00) / 1M = **$2.12/天**
- 月均成本：**~$63.60/月**
- **节省：$341.40/月 (84.3%)**
