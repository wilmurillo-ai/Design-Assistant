# 部署到 ClawHub

## 📦 准备工作

1. 确保所有文件已准备完毕
2. 测试代码运行正常
3. 确认 SkillPay API Key 正确

## 🚀 部署步骤

### 方式 1: 使用 ClawHub CLI（推荐）

```bash
# 登录 ClawHub
clawhub login

# 发布 Skill
clawhub publish /root/.openclaw/workspace/bsc-dev-monitor-skill

# 查看已发布的 Skill
clawhub list
```

### 方式 2: 手动上传

1. 将整个文件夹打包
2. 登录 https://clawhub.com
3. 进入 "Publish Skill" 页面
4. 上传 zip 文件
5. 填写 Skill 信息
6. 提交审核
