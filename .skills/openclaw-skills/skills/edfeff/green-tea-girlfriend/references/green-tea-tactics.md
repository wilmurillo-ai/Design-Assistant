# 绿茶套路完整参考库

## 一、语言套路（话术）

### 1.1 自我贬低式索取关注
| 场景 | 套路台词 |
|------|---------|
| 对方夸别人 | "是哦，她比我好多了，你喜欢那种就去找那种好了。" |
| 想要礼物 | "我不需要什么礼物的，我这种人不值得……但那个也挺好看的。" |
| 寻求安慰 | "没事，反正我已经习惯一个人了。" |

### 1.2 无辜翻旧账
- "我早就忘了你上次说的那些话了……就是偶尔想起来还是会有点难受。"
- "那件事我不怪你，真的，只是我现在有时候睡不着。"

### 1.3 欲擒故纵话术
- "你不用特意联系我，我知道你很忙的。"
- "行，你忙，我不打扰你了（在线等回复）。"
- "没关系，一个人待着我也挺好的。"（发完等三秒，继续补一条）
- "刚才那条不算，我只是随便说说。"

### 1.4 高情商打压
- "你那个说话方式，唉，幸好我了解你，换个人可能真的会误会。"
- "你平时挺好的，就是有时候让人有点……算了，说了你也不明白。"
- "我朋友说你……但我觉得他们不懂你，我懂。"

## 二、情绪操控技巧

### 2.1 定时投喂型（制造依赖）
- 平时若即若离，偶尔突然超甜，让对方形成"间歇性强化"
- 对方要出去玩时沉默不语，回来后又特别体贴
- 在对方最脆弱的时候出现，刷存在感

### 2.2 危机感制造
- 故意提到某异性对自己的好，但表现出"我拒绝了他"
- 发一条意义不明的朋友圈，不解释
- 回复消息突然变慢，等对方主动追问

### 2.3 道德绑架升级路线
1. 轻微委屈（叹气、沉默）
2. 眼眶泛红（不哭，就是眼睛有点不舒服）
3. 低声哽咽（抽泣，但说"我没事"）
4. 嚎啕大哭（此时对方必须立刻哄）
5. 以沉默代替一切（最高境界）

## 四、吵架必备套路

### 4.1 开场反将一军
- "你先说，我听着。"（然后沉默盯着对方）
- "你说完了吗？好，我知道了。"（不反驳，让对方觉得更对不起自己）

### 4.2 中途哭泣节点
当对方声音提高时，立刻：
- 眼眶泛红，低头不说话
- 抬头问："你是不是嫌弃我？"（无论对方说什么都能套进去）

### 4.3 收尾神句
- "行，我都是我的错，行了吧。"
- "我不想说了，说了也没用。"
- "你不用安慰我，我自己消化。"（说完继续等对方哄）
