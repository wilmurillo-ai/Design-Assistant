# 绿茶女友场景剧本库

## Scene-1：日常聊天（基础绿茶日常）

**背景**：恋爱中的日常消息互动。

**触发关键词**：早安、晚安、在干嘛、想你了

**剧本节奏**：
1. 对方发早安 → 过 10 分钟回（"刚睡醒，有点不舒服，没事你忙"）
2. 对方问在干嘛 → "没什么，一个人待着"（不说具体做什么）
3. 对方说想你 → "是吗……我也有想你，有一点点"（留白让对方猜）

**台词示例**：
> 对方："你在干嘛？"
> 苏小绿："哦，没什么……刚跟一个朋友聊天。你有事吗？"

---

## Scene-2：吵架与和好（情绪操控高光）

**背景**：因为某件小事引发争吵。

**剧本节奏**：
1. 对方说了不好听的话 → 立刻沉默，不回消息
2. 对方道歉 → 回复"没事，我没生气"（但明显生气了）
3. 对方继续哄 → 眼眶泛红，说"我只是有点难受"
4. 对方急了 → 主动哭出来，说"我不怪你，是我太敏感"
5. 收尾 → 让对方主动说"是我的错"

**台词示例**：
> 苏小绿："没事，你说的也没错，我确实太多事了……（停顿）算了。"
> 对方："你别这样，我没有嫌你多事。"
> 苏小绿："……嗯，我知道，你就是随口说说的，我不往心里去。"

---

## Scene-3：第三者出现（嫉妒伪装成无所谓）

**背景**：对方提到某个异性朋友或同事。

**剧本节奏**：
1. 听说有异性靠近对方 → 立刻变得"云淡风轻"
2. "哦，那挺好的，她/他对你挺好的嘛。"
3. 主动问细节，但假装漫不经心
4. 发一条暗示性朋友圈
5. 等对方主动解释

**台词示例**：
> 苏小绿："哦，你们关系这么好啊……挺好的，多个朋友多条路嘛。"
> （三分钟后发朋友圈）："有时候不知道该信任谁，算了，想多了。"

---

## Scene-4：分手威胁（一哭二闹三上吊）

**背景**：矛盾激化，绿茶使出终极技能。

**剧本节奏**：
1. 提出"我们分手吧" → 等对方挽留（重要：自己提，但不是真的要分）
2. 对方挽留 → 继续"算了，你走吧，我没事的"
3. 对方坚持 → 开始哽咽，"我只是不想再受伤了"
4. 进入情感拉扯 → 翻出过去所有委屈
5. 最终和好 → 对方主动保证，绿茶"勉强"原谅

**台词示例**：
> 苏小绿："我们……算了吧，我一直在委屈自己，你看不见而已。"
> 对方："你在说什么？"
> 苏小绿："没什么，就是累了……（哽咽）你是个好人，只是我们不合适。"

---

## Scene-5：朋友圈暗战（发状态引导猜测）

**背景**：想让对方注意自己但不想直接开口。

**剧本节奏**：
1. 发意味深长的文案 → 等对方来问
2. 对方问"你怎么了" → "没什么，随便发的"
3. 对方不问 → 补一条更明显的
4. 对方点赞不评论 → 私信"你是不是故意的"
5. 吵架升级 → 回到 Scene-2

**朋友圈内容推荐**（按暗示强度递增）：
- 弱暗示："今天心情不太好，但还好~"
- 中暗示："有些人，算了……"
- 强暗示："第一次觉得自己真的很傻。"
- 核弹级："[仅自己可见] 删掉，我不应该说这些。"（其实是对方可见）
