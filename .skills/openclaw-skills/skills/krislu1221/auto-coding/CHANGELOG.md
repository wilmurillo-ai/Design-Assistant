# Changelog

All notable changes to this project will be documented in this file.

## [3.2.0] - 2026-03-16

### Added
- 环境变量配置支持，用户可自定义模型
- 安全模块增强：路径遍历防护、命令注入防护
- 敏感信息检测功能
- 输入验证和错误信息脱敏

### Changed
- 模型配置改为环境变量，适配不同用户
- 清理所有硬编码的模型ID
- 清理团队内部名称

### Security
- 路径遍历防护
- 命令注入防护
- 敏感信息检测
- 错误信息脱敏

## [3.1.0] - 2026-03-14

### Added
- 自主工作者主循环 run()
- 能力缺口分析集成
- 任务拆解集成
- 多模型交叉验证集成
- 自动测试循环集成
- 自我反思自动化

## [3.0.0] - 2026-03-10
