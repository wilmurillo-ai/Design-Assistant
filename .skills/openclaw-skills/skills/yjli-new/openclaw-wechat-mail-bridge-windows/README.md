# WeChat Mail Bridge for OpenClaw

ClawHub release bundle for a Windows WeChat desktop automation, File Transfer Assistant testing, and BHMailer/OpenClaw mail bridge workflow.

Top-level user entry points:

- `SKILL.md`: skill instructions used by OpenClaw and indexed by ClawHub.
- `references/`: concise install, config, operations, and release notes docs.
- `config/`: clean config templates to copy from.
- `scripts/`: Windows batch helpers.
- `bundle/`: shipped plugin and sidecar release sources.

Recommended path:

1. Read `references/install-windows.md`.
2. Copy from `config/` into your local editable files.
3. Use `scripts/` to install deps, start the plugin, and start the sidecar.
4. Use `bundle/` when you need the shipped OpenClaw plugin, Windows sidecar, or BHMailer bridge source tree.
