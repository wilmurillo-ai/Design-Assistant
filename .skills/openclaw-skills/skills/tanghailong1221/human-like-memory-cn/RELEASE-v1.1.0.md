## ⚠️ 注意事项

### 首次使用

1. **模型下载**: 首次运行会自动下载 bge-m3 模型（约 200MB）
2. **网络要求**: 需要稳定网络连接（仅首次）
3. **存储空间**: 确保有 300MB 可用空间
4. **内存要求**: 建议 512MB 以上可用内存

### 网络问题解决方案

如果模型下载失败：

```bash
# 方案 1: 使用国内镜像
export HF_ENDPOINT=https://hf-mirror.com
npm install

# 方案 2: 手动下载模型
git lfs install
git clone https://hf-mirror.com/Xenova/bge-m3 ~/.cache/models

# 方案 3: 临时关闭向量搜索
# 在 config.json 中添加:
{ "useVectorSearch": false }
```
