# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-31

### ✨ Added
- 初始版本发布
- 人类记忆模拟机制
  - 自动记忆编码（识别用户明确指示、任务相关性、情感强度等）
  - 5 维度重要性评分算法（用户指示 40% + 任务相关 25% + 情感 15% + 可复用 12% + 唯一性 8%）
  - 三层记忆架构（HOT 短期/WARM 中期/COLD 长期）
  - 记忆复习机制（7 天间隔，复习 +10 分，未复习 -20 分）
  - 遗忘机制（<30 分自动清理）
- 自动上下文压缩
  - 触发阈值：上下文使用率>70%
  - 分级压缩策略（移除低分→摘要化→归档）
  - 保护核心记忆（≥80 分不压缩）
- 记忆关联网络
  - 支持记忆间双向链接
  - 标签系统（3-5 个标签/记忆）
  - 分类系统（8 种记忆类型）
- 完整的 CLI 命令接口
  - `/remember` - 手动编码记忆
  - `/memory search` - 搜索记忆
  - `/memory review` - 复习记忆
  - `/memory stats` - 查看统计
  - `/memory compact` - 手动压缩
  - `/memory export` - 导出记忆

### 🔧 Technical
- 配置文件：`config.json`（可自定义评分权重、阈值、间隔）
- 数据存储：JSON 格式（HOT/WARM/COLD 分离）
- 模板系统：`templates/memory-template.md`
- 统计追踪：`stats.json`（编码/复习/遗忘统计）
- 复习队列：`review-queue.json`（到期记忆管理）

### 📚 Documentation
- 完整使用文档（README.md）
- 技能说明（SKILL.md）
- 记忆模板示例
- 故障排除指南

### 🎯 Use Cases
- 长对话上下文管理（节省 30-50% 空间）
- 用户偏好持久化（跨会话记忆）
- 项目信息归档（自动升级 HOT→WARM→COLD）
- 重要决策记录（永不遗忘≥90 分记忆）
