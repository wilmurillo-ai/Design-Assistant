## 核心功能

1. **向量化记忆检索** — 语义搜索，相似度>0.7 视为相关
2. **智能摘要压缩** — 按重要性自动压缩（原文/摘要/关键词）
3. **按需上下文注入** — 每轮对话动态加载最相关记忆
4. **三层记忆架构** — HOT（7 天）/ WARM（7-30 天）/ COLD（永久）
5. **记忆遗忘机制** — 自动清理低价值信息
