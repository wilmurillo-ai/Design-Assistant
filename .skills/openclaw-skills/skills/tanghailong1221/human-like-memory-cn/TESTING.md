## 故障排除

### 问题 1: 模型下载失败

**错误信息：**
```
Error: Failed to download model
```

**解决方案：**
```bash
# 手动下载模型
export TRANSFORMERS_CACHE=/home/admin/openclaw/workspace/skills/human-like-memory/.cache
node -e "require('@xenova/transformers').pipeline('feature-extraction', 'Xenova/bge-m3')"
```

### 问题 2: 内存不足

**错误信息：**
```
JavaScript heap out of memory
```

**解决方案：**
```bash
# 增加 Node.js 内存限制
export NODE_OPTIONS="--max-old-space-size=1024"
```

### 问题 3: 向量化速度慢

**优化方案：**
```bash
# 使用量化版本（已在配置中启用）
# 或降级到关键词检索（临时方案）
```

在 config.json 中添加：
```json
{
  "useVectorSearch": false  // 临时关闭向量搜索
}
```
