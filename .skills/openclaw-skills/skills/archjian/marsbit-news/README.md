# Marsbit News Skill

Crypto and Web3 news API for AI agents.

Marsbit News Skill provides structured crypto, blockchain and Web3 news data that can be used by AI assistants and autonomous agents.

Marsbit is a crypto and Web3 media platform providing market news, analysis, and real-time updates.

## Features

The Marsbit News Skill enables AI systems to:

- Retrieve the latest crypto and Web3 articles
- Retrieve full article content
- Search articles by keyword
- Retrieve latest flash news
- Search flash updates by keyword

These APIs allow AI assistants to access reliable news data from the Marsbit platform.

## Skill Definition

The AI skill specification is defined in:

skill.md

AI agents can read this file to understand how to call the Marsbit APIs.

https://api.marsbit.co/info/ai

### Articles

/articles/latest  
/articles/detail  
/articles/search  

### Flash News

/flashes/latest  
/flashes/search  

## Example Use Cases

AI assistants can use this skill to:

- Retrieve latest crypto news
- Track breaking Web3 developments
- Search news related to Bitcoin or Ethereum
- Summarize crypto market events
- Monitor real-time flash updates
