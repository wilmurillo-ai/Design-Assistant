## 一、万物类象的推导原理（重点）

> 梅花易数之"活"，核心在于类象不是死记硬背，而是从八卦本质推导。

### 1. 八卦本质：从爻象到性质

| 卦 | 爻象结构 | 核心性质 |
|---|----------|----------|
| **乾** ☰ | 三连阳爻 | 刚健、至阳、圆满、主动 |
| **兑** ☱ | 上缺阴爻 | 喜悦、毁折、口开、外破 |
| **离** ☲ | 中虚阴爻 | 光明、丽、文明、中空 |
| **震** ☳ | 初阳动爻 | 震动、奋起、决断、起步 |
| **巽** ☴ | 初阴入爻 | 进入、传播、柔顺、渗透 |
| **坎** ☵ | 中实阳爻 | 危险、流动、陷落、智慧 |
| **艮** ☶ | 上实阳爻 | 停止、阻碍、稳固、突出 |
| **坤** ☷ | 三连阴爻 | 柔顺、承载、静止、包容 |

### 2. 从性质到类象的推导

**推导公式：爻象结构 → 核心性质 → 万物类象**

举例：

| 卦 | 性质 | 推导 | 古代类象 | 现代类象 |
|---|------|------|----------|----------|
| 乾 | 刚健、圆、领导 | 最坚固的→天，最能跑的→马，最有权的→父/贵族 | 天、马、头、玉、金 | 电脑、汽车、飞机、互联网、权威机构 |
| 兑 | 喜悦、毁折、口开 | 泽水浸润→低洼，少女→柔，嘴→开口说话 | 泽、少女、口、破损器 | 手机、社交、音响、合同、口腔 |
| 离 | 光明、丽、文明 | 火照→明亮，眼睛能见，书载文明 | 火、日、书、眼睛、文明 | 电力、屏幕、媒体、电子产品 |
| 震 | 震动、动、起 | 雷声震动，龙善动，足行走 | 雷、龙、惊惧、足 | 交通工具（震动）、手机响动、突发事件 |
| 巽 | 入、传播、柔顺 | 风无孔不入，长女柔顺，直物穿透 | 风、长女、草木、绳直 | 网络传播、航班、高铁、广告、指令 |
| 坎 | 危险、流动、陷 | 水流动，雨从天落，低洼危险 | 水、雨、豕、耳、险 | 雨、网络流、黑客、液体、危险 |
| 艮 | 止、阻、门、鼻 | 山静止不动，门止出人，手指止物 | 山、门、狗、指、止 | 障碍、门禁、刹车、鼻子（凸出） |
| 坤 | 柔顺、承载、方 | 大地承载万物，布方能包，母柔顺 | 地、母、布、囊、方 | 大地、平台、仓库、数据、容器 |

### 3. 活解要点

**不要背死类象，要掌握推导逻辑：**

1. **抓住核心性质** - 阳主动、阴主静；中空外实、中实外虚
2. **联想相关事物** - 从性质出发，联想符合这一性质的事物
3. **结合实际情境** - 同一卦在不同场景下类象不同
4. **与体用结合** - 看卦象与所问之事的五行生克关系

## 三、体用关系

> 体为求测者/主体，用为所问之事/客体

### 体用生克与吉凶

| 体用关系 | 吉凶 | 含义 | 建议 |
|----------|------|------|------|
| **体克用** | 吉 | 掌握主动权，事情向利己发展 | 积极推进 |
| **用克体** | 凶 | 被动受压，事情受阻 | 审时度势，必要时暂停 |
| **体生用** | 平中带耗 | 付出多于收获，劳心费力 | 评估性价比 |
| **用生体** | 吉 | 外部资源倾斜，有意外收获 | 把握机会 |
| **体用比和** | 大吉 | 双方和谐，做事顺遂 | 放手去做 |

### 体用生克的力量判断

1. **月令影响**：体卦旺则抗克能力强，体卦衰则抗克能力弱
2. **旺相之时**：体卦旺相，能克衰败之用卦
3. **休囚之时**：体卦休囚，难以克制旺盛之用卦

## 四、主互变三卦综合判断

### 主卦（本卦）
代表**当前状态**和**事情主体**，动爻所在位置是**变化的触发点**。

### 互卦
代表**事情发展过程**和**中间状态**，体现事情演变的中间环节。

### 变卦
代表事情**最终结果**和**变化后的状态**。

### 综合判断要点

1. **先看主卦体用关系** — 确定基本吉凶基调
2. **再看动爻变化** — 动爻是变化的转折点
3. **参考互卦** — 了解事情发展过程
4. **以变卦定结局** — 变卦是最终呈现的结果
5. **结合月令** — 五行旺衰影响吉凶程度

## 五、动爻分析

### 动爻位置含义

| 爻位 | 含义 |
|------|------|
| 初爻 | 事情刚开始，基础层面 |
| 二爻 | 内部阶段，方案层面 |
| 三爻 | 中期阶段，执行层面 |
| 四爻 | 接近目标，外部层面 |
| 五爻 | 关键位置，领导层面 |
| 上爻 | 最终结局，顶层层面 |

### 动变规则
动爻之阴阳发生改变，阳爻变阴爻，或阴爻变阳爻，从而产生变卦。

## 六、应期判断

> 应期判断是梅花易数最难的部分，需要结合体用、象数、实际情况综合判断

### 应期数字计算法

以动爻位置数 + 用卦数（或变卦卦数）之和，用月令或日辰起算。

**口诀**：
- 近事以日计
- 短期事以月计
- 长期事以年计

### 应期判断要点

1. **体用关系转化时** — 体用关系由凶转吉或由吉转凶的转折点
2. **用卦旺相之时** — 用卦当令或临长生时，事情容易发生
3. **冲合之时** — 逢冲或逢合的日期
4. **动爻逢值** — 动爻地支当值之日
5. **变卦应期** — 以变卦的卦象配合判断

### 实用应期判断

| 询问范围 | 应期参考 |
|----------|----------|
| 几日 | 1-10日 |
| 近期 | 一月之内 |
| 短期 | 三月之内 |
| 中期 | 半年之内 |
| 长期 | 一年之内或更久 |

## 七、断卦步骤

1. **确认问题** — 明确用户所问之事
2. **获取排盘** — 执行程序获取卦象
3. **分析体用** — 判断体用生克关系及旺衰
4. **解读动爻** — 分析动爻位置和变化
5. **综合三卦** — 主卦看现状，互卦看过程，变卦看结果
6. **判断应期** — 事情发生的大致时间
7. **综合结论** — 给出合理的推断和建议
