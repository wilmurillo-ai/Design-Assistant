## 一、月令旺相（五行旺衰）

> 五行在十二个月中的旺衰状态，是断卦的重要依据

### 旺相休囚死释义

- **旺**：最旺盛，当令之气（同我者旺）
- **相**：次旺，受旺气所生（我生者相）
- **休**：休息，生我者休
- **囚**：被困，克我者囚
- **死**：最衰，我克者死

### 五行旺衰规则

> 规则：同我者旺，我生者相，生我者休，克我者囚，我克者死

### 月令断卦要点

```
例：春季（寅卯月）占卦
  - 木旺：震卦、巽卦得令
  - 火相：离卦次旺
  - 金囚：乾卦、兑卦衰
  - 水休：坎卦不旺
  - 土死：坤卦、艮卦衰
```

## 二、体用生克

### 基础概念

- **体卦**：主卦中**不动**的卦，代表求测者、本人、主体
- **用卦**：主卦中**动爻所在**的卦，代表所问之事、对方、环境

> 例如：天地否，初爻动，初爻在下卦坤，则下卦坤为用卦，上卦乾为体卦

### 生克关系

| 关系 | 名称 | 吉凶 | 解释 |
|------|------|------|------|
| 体克用 | 财 | 吉 | 以少胜多，主导权在握 |
| 用克体 | 官鬼 | 凶 | 祸害临头，压力巨大 |
| 体生用 | 泄气 | 耗 | 劳心费力，付出多回报少 |
| 用生体 | 生气 | 吉 | 外来助力，事半功倍 |
| 体用比和 | 比和 | 吉 | 和谐顺利 |

### 动爻的影响

梅花易数**一卦一爻动**，动爻发生变化产生变卦，需要综合分析本卦、互卦、变卦三者。

### 标准流程

  ↓

## 四、断卦要诀

### 1. 先看体用
> 体用生克是根本

- 体克用 → 吉
- 用克体 → 凶
- 体生用 → 耗
- 用生体 → 吉
- 体用比和 → 吉

### 2. 次看月令
> 旺相之地如同得助

- 用卦在月令旺相 → 力量强
- 用卦在月令休囚 → 力量弱

### 3. 再看动爻
> 动则变，变则通

- 动爻生体 → 大吉
- 动爻克体 → 大凶
- 静爻只做辅助参考

### 4. 参看三要十应
> 内外感应，天人合一

- 外应与卦象相符 → 更准
- 外应与卦象相悖 → 需细辨

### 5. 结合万物类象
> 象数合一，取象准确

- 根据所问之事选取合适象意
- 卦象+象意=具体事件

## 五、应期推断

### 基本方法

> **成卦之数 + 时数 = 应期**

计算公式：
- 成卦之数 = 上卦先天数 + 下卦先天数（1-8 + 1-8 = 2-16）
- 时数 = 时辰地支序数（子=1, 丑=2, ... 亥=12）
- **应期数字范围：3-28**

### 应期断法核心

> **必须根据所问之事的性质，结合现实逻辑来推断时间单位！**

不是数字大就是年，数字小就是日。要看问的是什么：

| 问的事情 | 时间单位推断示例 |
|----------|------------------|
| 外卖/快递 | 分钟（13分钟） |
| 今天的 事 | 时辰/小时 |
| 本月的事 | 日/旬 |
| 今年的事 | 月 |
| 长期的事 | 年 |

### 断应期要点

1. **先问什么**：问今天的事，按日计；问长远的事，按年计
2. **灵活判断**：数字是参考，关键符合现实逻辑
3. **参考月令**：旺相之时应期近，休囚之时应期远
4. **不固守**：有的卦无需应期，切勿强求

### 示例

```
例1：问外卖何时到
    天地否卦，上卦乾1 + 下卦坤8 = 9
    丑时（2），应期=9+2=11
    断：11分钟内送到

例2：问今日财运
    数字11，巳时（6），应期=17
    断：约17天内有财运

例3：问今年工作
    数字15，亥时（12），应期=27
    断：约27个月后（约2年后）有转机
```

---

*数字只是参考，关键看问什么事。问外卖送到，13就是13分钟；问升职加薪，13可能就是13个月。*

## 六、梅花易数核心原则

1. **一卦一爻动**：梅花易数只有一爻动，多爻动为特殊情况
2. **体用为本**：体用生克是断卦核心
3. **心诚则灵**：起卦时需心无杂念
4. **灵活变通**：不可死板套用，需结合实际

## 七、案例示范

**案例**：问今日财运

```
起卦：时间起卦得《乾》卦上、《巽》卦下
本卦：《天风姤》
互卦：《乾》
变卦：《风天小畜》

分析：
1. 体卦：乾卦（金）
2. 用卦：巽卦（木）
3. 体用：乾金克巽木 → 体克用 → 财
4. 月令：假设秋季申酉月，金旺之时
5. 断卦：体克用为财，金旺之时得财
6. 三要：耳闻笑声，目见金色
7. 取象：金为财，木为财源
8. 结论：今日财运旺盛，必有收获！
```
