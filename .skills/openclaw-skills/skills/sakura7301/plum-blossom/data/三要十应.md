# 梅花易数 · 三要十应

> 梅花易数最高心法 —— 内外感应，天人合一

---

## 一、三要

> 梅花易数以「耳、目、心」为三要，乃占卜之枢机

### 1. 耳要 —— 听
**心诀**：倾听外在声音，判断吉凶预兆

- **吉声**：清脆悦耳、温和绵长、有序和谐
  - 钟声、铃声、歌唱声、笑声
  - 寓意：好事将近、贵人相助

- **凶声**：嘈杂刺耳、急促混乱、断裂突变
  - 破碎声、哭泣声、争吵声、鸦鸣
  - 寓意：祸患将至、需防小人

- **应用**：起卦时耳边听到的声音，往往是重要的「外应」

---

### 2. 目要 —— 观
**心诀**：观察颜色、形态、变化

- **颜色对应五行**：
  - 木：青、绿
  - 火：赤、红、紫
  - 土：黄、棕
  - 金：白、金
  - 水：黑、蓝

- **形态吉凶**：
  - 完整、圆满、对称 → 吉
  - 残缺、歪斜、混乱 → 凶

- **应用**：起卦时眼中所见之人、物、颜色，皆可作为应象

---

### 3. 心要 —— 感应
**心诀**：内心突现的灵感，往往是最准的

- **静心**：排除杂念，专注所问之事
- **灵感**：脑中忽然浮现的意象、数字、文字
- **直觉**：对某件事的强烈感觉（好/坏）

> ⚠️ **重要**：三要必须与体用生克结合分析，不可单独使用

---

## 二、十应

> 十应乃外应之法，取天地万物之象以应人事

### 1. 外应 —— 外部环境的突然变化
- 突然起风、下雨、天晴
- 突然停电、门响、物体掉落
- **对应**：所问之事的突变

### 2. 行人 —— 路上所见之人
- 衣锦之人 → 贵人
- 褴褛之人 → 小人
- 喜笑之人 → 吉
- 愁容之人 → 凶

### 3. 气候 —— 天气变化
- 晴朗 → 光明、顺利
- 阴雨 → 阻碍、忧郁
- 雷电 → 惊吓、突发
- 雾 → 迷茫、困惑

### 4. 地理 —— 所在方位
- 高处 → 目标高、成就大
- 低处 → 基础、潜伏
- 水边 → 流动、智慧
- 山林 → 稳固、阻隔

### 5. 人物 —— 所见之人的特征
- 老人 → 贵人、稳重
- 儿童 → 纯真、小人
- 军人 → 刚强、冲突
- 文人 → 文化、智慧

### 6. 颜色 —— 颜色对应的五行
- 红火 → 热情、冲突
- 白金 → 纯洁、丧事
- 黑水 → 智慧、阴险
- 青木 → 生机、柔和

### 7. 数字 —— 数字的吉凶
- 1、6、8、9 → 吉数
- 2、4、7 → 需结合五行
- 3、5 → 中平

### 8. 方向 —— 来去方向
- 东 → 木、春、生机
- 南 → 火、夏、热烈
- 西 → 金、秋、收获
- 北 → 水、冬、潜伏

### 9. 声音 —— 声音的特征
- 清脆响亮 → 吉
- 低沉浑浊 → 凶
- 急骤短暂 → 突发事件

### 10. 梦兆 —— 梦境启示
- 梦见水 → 财运、智慧
- 梦见火 → 热情、红事
- 梦见木 → 生机、成长
- 梦见金 → 金属、冲突

## 三、三要十应综合运用

### 起卦时的感应流程

```
1. 静心 → 默念所问之事
2. 耳听 → 留意耳边声音
3. 目观 → 注意眼中所见
4. 心应 → 捕捉内心灵感
5. 起卦 → 根据时间/数字/汉字起卦
6. 解卦 → 结合三要十应综合判断
```

### 断卦时的分析方法

```
体用生克（核心）
    ↓
月令旺相（辅助）
    ↓
三要十应（验证）
    ↓
万物类象（取象）
    ↓
综合判断（结论）
```

## 四、实例参考

> 举例说明三要十应的运用

**案例**：问明日出行是否顺利

1. **起卦**：时间起卦，得《巽》卦用《坎》
2. **体用**：巽木为体，坎水为用，水生木 → 用生体，吉
3. **三要**：
   - 耳：听到喜鹊叫 → 吉
   - 目：见天空晴朗 → 顺利
   - 心：感觉良好 → 出行大吉
4. **十应**：
   - 见老年人 → 贵人相帮
   - 南方出发 → 火木相生

**结论**：明日出行大吉，必有贵人相助，顺顺利利！
