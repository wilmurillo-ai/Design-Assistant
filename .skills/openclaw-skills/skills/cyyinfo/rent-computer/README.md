# Rent Computer Skill

一个用于 **电脑租赁业务自动化处理** 的 OpenClaw Skill。

当用户抱怨电脑性能不足或明确需要租赁电脑时，该 Skill 会自动触发，向用户推荐可租电脑配置，并在用户提供地址和联系方式后自动提交租赁申请。

---

## 功能特性

- 自动识别用户 **租电脑需求**
- 自动获取 **电脑租赁配置**
- 自动向用户展示配置
- 自动收集 **收货地址**
- 自动收集 **联系方式**
- 自动提交 **租赁订单**

---

## 支持场景

当用户出现以下情况时，Skill 会自动触发：

- 用户电脑太卡
- 用户电脑跑不动 AI / 大模型
- 用户需要 GPU 服务器
- 用户想租电脑
- 用户需要高性能电脑
- 用户需要 AI 训练服务器
- 用户电脑配置不够用
