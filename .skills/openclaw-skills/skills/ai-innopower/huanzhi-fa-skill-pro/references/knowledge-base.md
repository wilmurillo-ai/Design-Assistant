# Fa.Pro 知识库

## 3.1 法规政策库

### 公司法

#### 股权结构

**关键要点：**
- 创始人应保留控制权至 B 轮前（>50%）
- 期权池 10-15%，投前设立稀释创始人
- 代持股权需清理

**红线/风险：**
- 股权过于分散
- 期权池时机不当
- 联合创始人分配争议

#### 股东权利

**关键要点：**
- 普通股 vs 优先股权利差异
- 保护性条款限于重大决策
- 否决权不含日常经营

**红线/风险：**
- 小股东日常经营否决
- 未经同意的强制出售权
- 过于宽泛的信息披露

### 外商投资

#### VIE 架构

**关键要点：**
- 适用于外资限制行业
- 需 WFOE-OPCO 协议控制
- 37 号文登记必须

**红线/风险：**
- 政策不确定性
- 利润转移税务风险
- 外汇管制资金出境限制

### 股权激励

#### 期权池设计

**关键要点：**
- 占完全稀释后 10-15%
- Vesting 通常 4 年，1 年 cliff

**红线/风险：**
- 投前 vs 投后设立影响稀释
- 行权价格和税务处理

### 证券法

#### 私募融资

**关键要点：**
- 合格投资人认定
- 募资人数限制
- 信息披露义务

**红线/风险：**
- 非法集资风险
- 公开宣传违规

## 附录：行话词典

| 投资人说的 | 实际意思 | 应对策略 |
|-----------|---------|---------|
| "我们再看看" | 目前没兴趣，但别得罪 | 定期 Update，等待窗口 |
| "赛道不错，时机早了点" | 不投，但不想明说 | 追问具体顾虑，或放弃 |
| "再看看数据" | 感兴趣但有顾虑 | 主动提供补充材料 |
| "条款都是标准的" | 可能有坑，希望你别细看 | 逐条审阅，找律师 |
| "我们对估值敏感" | 觉得贵，想压价 | 展示可比交易，制造竞争 |
| "在走内部流程" | 可能是真流程，也可能是 Plan B | 设定 Deadline，启动 Plan B |
| "保持联系" | 礼貌性拒绝 | 降级为低优先级，专注其他 |
