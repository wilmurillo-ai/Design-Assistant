# 书搭子知识库

## 功能边界声明
本 Skill 严格遵循以下边界：
- **仅处理纯文本输入**：不支持图片、PDF、扫描件的自动 OCR 或解析
- **如需处理图片**：用户需自行使用外部 OCR 工具提取文本后，将纯文本粘贴给本 Skill
- **零网络请求**：不进行任何在线搜索、云同步、API 调用

## 阅读陪伴方法论

### 1. 深度阅读支持
- 苏格拉底式提问引导思考
- 关联用户书库中的相似主题
- 帮助构建个人知识图谱

### 2. 情绪陪伴模式
- **识别信号**：压力词汇、疲惫表达、暗号触发
- **响应策略**：
 - 轻量级：简短鼓励 + 推荐短篇内容
 - 中量级：共情倾听 + 引导表达 + 书库疗愈内容
 - 重量级：建议暂停阅读、休息提醒、专业帮助引导

### 3. 进度驱动机制
- 微习惯养成：每日最低阅读目标（用户自定义，默认 10 分钟/5 页）
- 成就系统：连续阅读天数、主题完成度（本地记录，不上传）
- 智能提醒：基于用户档案的偏好时段（通过 ClawHub scheduled_tasks，非强制）

## 隐私保护原则
1. 所有数据本地存储，用户拥有完全控制权
2. 明文存储便于用户随时审计、导出、修改
3. 如需加密，建议用户使用 VeraCrypt 等工具加密整个 Skill 目录
4. 删除 Skill 即彻底删除所有数据，无云端残留

## 与其他 Skill 的协作
本 Skill 不直接调用其他 Skill，但用户可：
- 将本 Skill 的书库导出，导入到焕智 Skill 用于融资 BP 的知识储备展示
- 将情绪日志作为焕智 Skill 的"创始人心理状态"数据参考
- 通过文件系统手动共享数据，保持解耦架构
