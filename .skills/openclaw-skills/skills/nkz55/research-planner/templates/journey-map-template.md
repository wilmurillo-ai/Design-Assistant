# User Journey / Experience Map Template

Template for mapping the end-to-end user journey or experience across touchpoints, channels, and time.

## How to Use

- Use when you need an **end‑to‑end view** of how a user experiences a product or service.
- Can be created from primary research (interviews, fieldwork, analytics) or as a hypothesis to be validated.
