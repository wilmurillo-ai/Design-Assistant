# Storyboard Template

Template for creating storyboards that visualize key moments in a user’s experience over time.

## How to Use

- Use to illustrate sequences of key moments in a user’s story in a **visual, narrative form**.
- Storyboards are particularly helpful for socializing insights and designing future experiences.

## Storyboard Template

```markdown
# Storyboard – [Scenario / User]

Version: v[...]
Date: [...]
Author: [...]

## 1. Scenario

- Brief description of the situation and goal:
  - [...]
- Persona / user type:
  - [...]

## 2. Key Moments (Frames)

For each frame, capture:

- Frame number:
- Title:
- Setting:
- What the user is doing:
- What the user is thinking/feeling:
- System or service response:
```
