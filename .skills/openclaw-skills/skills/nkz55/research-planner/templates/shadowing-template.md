# Shadowing Template

Template for planning user shadowing sessions, where researchers observe users performing tasks over time.

## How to Use

- Use shadowing when you want to see work unfold **in situ and over time**, not just in a single interview.
- Combine with shadowing notes template for consistent documentation.
