# Empathy Map Template

Template for building empathy maps that capture what a user says, thinks, does, and feels.

## How to Use

- Use as a **synthesis tool** after qualitative research (interviews, observation).
- Keep it grounded in data; label inferences clearly.

## Empathy Map Template

```markdown
# Empathy Map – [Persona / User Segment]

Version: v[...]
Date: [...]
Author: [...]

## 1. Context

- Scenario or situation:
  - [e.g., "When managing monthly invoicing", "When first trying our product"]
- Based on which research sources:
  - [e.g., "10 user interviews in Q1", "field observations from 3 sites"]

## 2. Quadrants

### Says

- Direct quotes:
  - "[...]"
  - "[...]"

### Thinks

- Thoughts, worries, aspirations (informed by research, not pure speculation):
  - [...]

### Does

- Key behaviors and actions:
  - [...]

### Feels

- Emotions and attitudes:
  - [...]
```

## Key References for Empathy Maps

- Dave Gray, *Gamestorming* – includes empathy map exercises.
