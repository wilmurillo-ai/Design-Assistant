# Rapid Iterative Testing and Evaluation (RITE) Template

Template for planning and documenting RITE-style rapid usability testing cycles with frequent design changes between sessions.

## How to Use

- Use when you want **fast learning cycles** with design changes between sessions.
- Requires close collaboration between research and design/engineering to apply changes quickly.
