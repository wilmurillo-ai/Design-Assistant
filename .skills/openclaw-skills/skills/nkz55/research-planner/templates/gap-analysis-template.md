# Gap Analysis Template

Template for systematically identifying gaps between current and desired states in user experience, product capabilities, or processes.

## How to Use

- Use when stakeholders ask “Where are we today vs. where do we want to be?” in terms of UX or capabilities.
- Combine with journey maps and service blueprints for richer context.
