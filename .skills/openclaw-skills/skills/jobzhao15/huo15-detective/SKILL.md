# huo15-detective - 企业/人员背景调查技能

企业/人员背景调查专用技能，包含调查流程、平台选择、风险识别和报告生成。

## 触发条件

当用户要求调查以下内容时自动触发：
- 调查公司/企业背景
- 调查个人/人员背景
- 查公司信息
- 企业信用调查
- 背景调查
- 关联企业分析

---

## 调查平台推荐

| 平台 | 优点 | 缺点 |
|------|------|------|
| **企查查（qcc.com）** | 数据全面、更新快、页面清晰 | 需要会员查看详细数据 |
| **天眼查** | 数据丰富 | 页面结构复杂 |
| **国家企业信用信息公示系统** | 官方数据、免费 | 数据展示不够直观 |

**推荐**：优先使用企查查

## 调查流程

### 第一步：确定调查对象

#### 公司调查
- 使用完整公司名称搜索
- 注意：可能有曾用名、简称
- 示例：`中物汇通（青岛）供应链有限公司`

#### 人员调查
- ⚠️ **重要**：同名不同人极多，必须通过关联公司确认
- 搜索结果中的"张三"可能有上百个同名人物
- 必须结合公司信息确认身份

### 第二步：获取基本信息

从企查查获取以下核心信息：

| 信息项 | 重要性 | 说明 |
|--------|--------|------|
| 统一社会信用代码 | ⭐⭐⭐ | 企业唯一标识 |
| 注册资本 | ⭐⭐⭐ | 认缴资本（可认而不缴） |
| 实缴资本 | ⭐⭐⭐⭐ | 可能虚假，需验资报告确认 |
| 成立日期 | ⭐⭐⭐ | 判断企业经营时长 |
| 法定代表人 | ⭐⭐⭐ | 实际控制人之一 |
| 股东信息 | ⭐⭐⭐⭐ | 股权结构核心 |
| 员工人数 | ⭐⭐ | 经营规模参考 |
| 注册地址 | ⭐⭐ | 判断实际经营地 |
| 联系电话/邮箱 | ⭐⭐⭐ | 判断关联企业 |

### 第三步：分析股权关系

1. 查看股东及出资信息
2. 追溯最终实际控制人
3. 识别企业族群（如"中物汇达科技"）
4. **关键**：区分直接股权和间接股权

### 第四步：查看企业年报

**必须查看**，获取真实数据：
- 股东及出资信息（实缴资本）
- 对外投资信息
- 社保参保人数
- 企业资产状况（是否公示）
- 修改记录（资本变更等）

**年报查看地址**：
```
https://www.qcc.com/webReportYearDetail/{公司ID}_2024.html
```

### 第五步：风险评估

| 风险类型 | 检查项 |
|----------|----------|
| 法律风险 | 裁判文书、失信被执行人、限制高消费 |
| 经营风险 | 经营异常、行政处罚、欠税 |
| 资本风险 | 实缴资本与认缴资本差异、资本变更频繁 |

## 实缴资本虚假识别

### 常见作伪手法

1. **资金过账**：验资后立即转出（抽逃出资）
2. **虚假验资**：找中介垫资，验资后归还
3. **延迟实缴**：年报显示已实缴，但实际资金未到账

### 核实方法

如需确认真实出资情况，要求提供：
- ✅ 公司银行流水（验资账户）
- ✅ 验资报告（会计师事务所出具）
- ✅ 财务报表审计报告

### 关键判断

- 实缴资本 ≤ 认缴资本：正常
- 实缴资本 = 0：可能未实际出资
- 实缴时间晚于认缴期限：可能虚假
- 2024年才公示实缴：信息滞后

## 关联企业判断

### 判断标准

1. **同地址办公**：同一栋楼、同一房间号
2. **共用联系方式**：电话、邮箱
3. **股权关联**：股东、实际控制人相同
4. **业务关联**：同一行业、上下游

### 重要结论

- 同地址办公 ≠ 股权关联
- 共用电话/邮箱 ≠ 股权关联
- **必须通过股权结构图确认关系**

## 人员背景调查注意事项

### 同名人员识别

1. **必须结合公司**：通过关联公司确认身份
2. **查看身份证号**：部分平台显示部分身份证号
3. **查看历史任职**：公司历史变更记录
4. **风险关联**：查看该人关联公司是否有风险记录

### 风险人员特征

- 担任多家公司法定代表人
- 关联公司有失信/限高记录
- 公司频繁变更股权/法人
- 实际控制但非显名股东

## 调查报告输出规范

### 必含内容

1. **调查概述**：核心结论放前面
2. **基本信息**：注册资本、实缴资本、成立日期、股东
3. **股权关系**：实际控制人、关联企业
4. **风险提示**：法律风险、经营风险、资本风险
5. **核实建议**：需要进一步确认的材料

### 格式要求

- 使用 huo15-doc-template 技能生成 Word 文档
- 字体：仿宋正文、黑体标题
- 数据标注来源和时间
- 风险内容加粗提示
- 文件命名：`调查报告-{公司名}-{YYYYMMDD}.docx`
