# Chinese Simplified Semiconductor Terminology (简体中文 — Mainland China)

## The 11 Critical Mainland vs Taiwan Divergences

| Concept | Mainland (简体) | Taiwan (繁體) |
|---|---|---|
| Silicon | **硅** (gui) | 矽 (xi) |
| Chip | **芯片** | 晶片 |
| Integrated circuit | **集成电路** | 積體電路 |
| Photoresist | **光刻胶** | 光阻 |
| Lithography | **光刻** | 微影 |
| Etch | **刻蚀** | 蝕刻 (reversed!) |
| Epitaxial | **外延** | 磊晶 |
| Plasma | **等离子体** | 電漿 |
| Nanometer | **纳米** | 奈米 |
| Process (node) | **工艺** | 製程 |
| IP core | **IP核** | 矽智財 |

**ALWAYS search both variants for cross-strait research.**

| Section (Chinese) | English |
|---|---|
| **前五名供应商采购额** | Top-5 supplier procurement amounts (MANDATORY disclosure) |
| **前五名客户销售额** | Top-5 customer sales amounts |
| **供应商集中度** | Supplier concentration |
| **原材料采购** | Raw material procurement |
| **关联交易** | Related party transactions |
