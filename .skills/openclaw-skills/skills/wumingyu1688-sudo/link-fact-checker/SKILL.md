# 链接事实核验工具

## 功能
1. 检查给出的URL是否可以正常访问
2. 抓取资讯内容，核验资讯真实性
3. 交叉对比多个来源确认事实
4. 输出标准化核验结果

## 使用方式
- 用户给出需要核验的链接：直接访问并返回状态
- 用户给出资讯内容：搜索多个来源交叉验证

## 输出格式
```
【核验结果】
- 链接：URL
- 状态：可正常访问 / 无法访问 / 404 / 连接超时
- 资讯真实性：✅ 信息匹配 / ⚠️ 部分存疑 / ❌ 虚假信息
- 备注：简要说明
```

## 依赖
- 使用 `web_fetch` 工具访问链接
- 使用 `web_search` 搜索交叉验证
