# 道氏理论规则速查

## 牛熊市定义
- **牛市**：主要上升趋势，连续高点抬高+连续低点抬高
- **熊市**：主要下降趋势，连续高点压低+连续低点压低

## 相互验证表
| 指数组合 | 验证信号 |
|---------|---------|
| 工业+运输指数同创新高 | 牛市确认 |
| 工业+运输指数同创新低 | 熊市确认 |
| 仅一个指数创新高 | 趋势存疑，谨慎 |
| 两个指数新低不同步 | 底部可能更强 |

## 次级折返判断
- 折返幅度：主要趋势的33%~66%
- 折返时间：3周~数月
- 超过66% = 趋势可能逆转

## 道氏理论缺陷
1. 滞后性：信号来得太晚
2. 人为操纵：主要趋势初期难辨
3. 两大指数并非总是同步
4. 短期趋势无法预测
