# 筹码分布分析体系

筹码分布（CYQ）是东方财富特有指标，将持仓成本可视化，揭示主力吸筹/派发区间。

## 核心概念

### 筹码区间的五个位置
```
高位密集：股价上方筹码占比>70% → 主力出货/上方套牢重
低位单峰：股价下方筹码集中 → 主力吸筹充分
双峰峡谷：上下各一筹码峰，中间筹码少 → 主力洗盘
发散状态：筹码散乱 → 趋势不明
成本均匀：各价格筹码差不多 → 等待方向
```

### 成本集中度（CYQKKD）
```
>70%：高度集中，主力控盘强
50%~70%：中度集中
<50%：筹码分散
```

### 获利盘比例（CYS）
```
CYS > 30%：多数筹码盈利 → 短线回调风险大
CYS < -30%：多数筹码套牢 → 反弹空间大
CYS 在0附近：成本接近现价 → 突破概率大
```

## 典型形态与操作

| 形态 | 信号 | 操作 |
|------|------|------|
| 低位单峰+缩量 | 主力吸筹 | 持仓或买入 |
| 双峰填谷（向上突破）| 上升中继 | 回调颈线买 |
| 高位密集+放量大阴 | 主力出货 | 清仓 |
| 筹码快速下移 | 主力派发 | 离场 |
| 成本收敛后发散 | 趋势启动 | 跟进 |

## 支撑阻力计算
```
支撑1 = 筹码集中峰下沿
阻力1 = 筹码集中峰上沿
密集区突破后，支撑变阻力
```
