# 宏观指标速查

## 关键阈值
- CPI > 3%：通胀风险，政策收紧预期
- PPI > 5%：工业景气，供需两旺
- PMI > 50：制造业扩张
- M2增速 > 12%：流动性充裕
- 10年国债 > 3.5%：风险资产估值承压

## 政策信号汇总
| 政策工具 | 信号含义 |
|---------|---------|
| 降准 | 放水，利好 |
| 降息（MLF/LPR）| 降低融资成本，利好 |
| 麻辣粉(MLF)超量续作 | 释放流动性 |
| 逆回购到期 | 回收流动性 |
| 专项债加速发行 | 财政发力，利好基建 |

## A股日历效应
- 年初（1~2月）：春季躁动，布局全年
- 两会（3月）：政策预期，科技题材
- 年中（6~8月）：中报行情，业绩为王
- 年末（11~12月）：估值切换，调仓
