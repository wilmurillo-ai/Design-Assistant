# 行业板块轮动体系

核心逻辑：不同经济周期下，不同行业表现差异巨大，顺应轮动规律可以大幅跑赢大盘。

## 美林时钟四大周期

| 周期 | 特征 | 最佳资产 | 代表行业 |
|------|------|---------|---------|
| 复苏期 | GDP↑通胀↓ | 股票>债券>现金 | 科技/消费/医药 |
| 过热期 | GDP↑通胀↑ | 商品>股票>债券 | 有色/煤炭/钢铁 |
| 滞胀期 | GDP↓通胀↑ | 现金>商品>债券 | 能源/医药/农业 |
| 衰退期 | GDP↓通胀↓ | 债券>现金>股票 | 金融/基建/消费 |

## A股常用板块分类

### 周期类（顺周期）
- 大金融：银行/券商/保险/地产
- 强周期：煤炭/钢铁/有色/化工/建材

### 防御类（弱周期）
- 消费：食品饮料/白酒/家电
- 医药：创新药/医疗器械/中药

### 成长类（独立周期）
- 科技：AI/半导体/消费电子/通信
- 新能源：光伏/锂电/风电/储能
- 军工：航空航天/舰船/军工电子

### 政策驱动类
- 题材：商业航天/AI应用/低空经济/固态电池

## 资金轮动规律
```
经济预期改善 → 券商→周期股→消费→科技
经济预期恶化 → 科技→消费→医药→黄金
政策催化 → 题材股爆发（短线）
美股映射 → 科技股跟涨/跟跌
```

## 板块择时信号
1. 涨幅排名：连续3天出现在前10 → 主线确立
2. 资金净流入：板块指数资金连续净流入
3. 板块内部涨停数量：≥5只 → 板块效应强
4. 龙头股先行：板块中军先启动

## 操作节奏
```
确认主线：板块连续强势+资金确认
分歧回调：板块内部分化，龙头独立
高位退潮：龙头滞涨，跟风股补跌
下一主线切换：资金撤出后流向新板块
```
