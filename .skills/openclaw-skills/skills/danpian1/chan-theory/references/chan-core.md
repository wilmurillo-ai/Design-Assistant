# 缠论核心规则速查

## 分型定义
- **顶分型**：三根K线，第二根最高，第三根更低
- **底分型**：三根K线，第二根最低，第三根更高
- 包含关系处理：处理顶底分型的包含关系，取向上/向下处理

## 笔的严格定义
1. 顶底分型之间至少5根不共用K线
2. 必须处理包含关系
3. 笔破坏 = 新笔反向创新高/低

## 线段定义
- 由连续三笔构成
- 三笔必须有重叠区间
- 线段破坏：反向走势突破前一线段端点

## 中枢级别
| 级别 | 对应周期 | 操作周期 |
|------|---------|---------|
| 本级 | 日线 | 日线一笔 |
| 次级别 | 60分钟 | 日线一段 |
| 再次级别 | 5分钟 | 日线走势 |

## 背驰段判断标准
```
1. 趋势力度比较：连接同向走势的MACD面积
2. 空间比较：上涨段高低点差 vs 之前同级别上涨段
3. 时间比较：回调/反弹幅度 vs 上一段
同时满足2项以上 → 背驰确认
```

## 各周期配合
- 日线：定方向（30%
- 60分钟：找买卖点（次级别）
- 5分钟：精确止损（次次级别）
