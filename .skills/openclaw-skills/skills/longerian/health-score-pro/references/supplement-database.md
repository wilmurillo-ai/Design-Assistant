## 📚 补剂分类说明

### 分级标准

**基础补剂**（强烈推荐）:
- 安全性高，副作用少
- 研究证据充分
- 大多数人受益
- 基础健康支持

**进阶补剂**（可选）:
- 特定目标或人群
- 需要更多研究支持
- 可能需要医嘱
- 个性化需求

**特殊补剂**（需医嘱）:
- 针对特定健康状况
- 需要医疗监督
- 剂量需要精确控制
- 潜在副作用较高

## ⚠️ 补剂安全性

### 高风险补剂（需医嘱）

1. **激素类**: DHEA、褪黑素（高剂量）
2. **代谢类**: 二甲双胍、雷帕霉素
3. **草药类**: 可能与药物相互作用（银杏、人参等）

### 安全补剂（一般人群）

1. **维生素类**: 维生素C、D3、K2、复合B
2. **矿物质类**: 镁、锌、硒
3. **脂肪酸**: Omega-3
4. **运动补剂**: 肌酸、β-丙氨酸

### 补剂相互作用

**避免同时服用**:
- 钙 + 铁（竞争吸收）
- 锌 + 铜（长期锌需补铜）
- 镁 + 某些抗生素（间隔2小时）

**增强吸收**:
- 维生素D3 + K2 + 钙（协同）
- 维生素C + 铁（促进吸收）
- 姜黄素 + 黑胡椒（提高20倍）
- 脂溶性维生素 + 脂肪（随餐）
