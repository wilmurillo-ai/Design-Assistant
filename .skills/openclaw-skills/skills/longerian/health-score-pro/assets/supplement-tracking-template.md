## ⚠️ 补剂安全检查清单

### 每月必查

- [ ] **检查过期日期**: 所有补剂是否在有效期内
- [ ] **检查保存条件**: 是否按要求保存（冷藏、避光等）
- [ ] **检查副作用**: 是否有任何不适反应
- [ ] **检查药物相互作用**: 如有服用药物，咨询医生
- [ ] **检查剂量**: 是否仍在推荐范围内

### 每3个月必查

- [ ] **血液检查**: 相关指标（维生素D、肝肾功能等）
- [ ] **效果评估**: 补剂是否达到预期效果
- [ ] **调整方案**: 根据检查结果调整补剂

### 每6个月必查

- [ ] **全面体检**: 包括相关营养素水平
- [ ] **重新评估需求**: 是否仍需要所有补剂
- [ ] **咨询专业人士**: 营养师或医生
