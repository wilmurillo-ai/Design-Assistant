# AI SEO Article Skill Workflow

Goal: produce SEO-friendly article assets (cover + inline visuals + metadata image variants).

## Inputs

- primary keyword
- secondary keywords
- article angle
- target audience

## Prompt Template

```text
Create a complete visual asset plan for an SEO article.
Primary keyword: {primary_keyword}
Secondary keywords: {secondary_keywords}
Angle: {article_angle}
Audience: {audience}
Output:
1) cover image prompt,
2) 3 inline image prompts,
3) social preview image prompt,
4) image naming suggestions using SEO-friendly slugs.
```

## Media Generation Tips

- Generate 1:1 for social previews and 16:9 for article banners.
- Keep visual entities aligned with the article keyword intent.
