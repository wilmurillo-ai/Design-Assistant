# AI Comic Drama Workflow

Goal: turn a short story idea into a scene-by-scene image/video production plan.

## Inputs

- genre
- main character
- story arc
- tone

## Prompt Template

```text
Transform this story idea into a comic-drama production brief.
Genre: {genre}
Main character: {main_character}
Story arc: {story_arc}
Tone: {tone}
Output:
1) 8 scene prompts (visual storyboard),
2) 2 transition video prompts,
3) character consistency checklist,
4) style lock instructions.
```

## Media Generation Tips

- Reuse character descriptors exactly to improve consistency.
- Use image keyframes first, then animate selected scenes.
