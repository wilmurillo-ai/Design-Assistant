# idea-spark

Generate actionable project ideas by scanning real pain points from Hacker News, Reddit, and GitHub. Complements `idea-check` for validation.

## Requirements

- OpenClaw with `web_search` tool (built-in)
- `mcporter` + `idea-reality` MCP server (optional, for automatic validation)

## Setup

No setup required. The skill uses the built-in `web_search` tool.

For automatic idea validation, install `idea-check`:
```bash
clawhub install idea-check
```

## Usage

> "Give me project ideas for developer tools"

> "What should I build in the AI agent space?"

> "Startup ideas around health tracking"

## Install

```bash
clawhub install idea-spark
```
