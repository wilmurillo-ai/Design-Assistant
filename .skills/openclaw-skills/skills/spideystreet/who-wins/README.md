# who-wins

Query the PinchBench AI agent leaderboard — benchmarks for LLMs on standardized OpenClaw coding tasks.

## Requirements

- `curl` — fetches the leaderboard page
- `python3` — parses and formats the data

## Setup

No setup required. The skill scrapes public data from [pinchbench.com](https://pinchbench.com/).

## Usage

> "Show me the PinchBench leaderboard"

> "Which model is cheapest for OpenClaw?"

> "How does Claude compare to Gemini on PinchBench?"

## Install

```bash
clawhub install who-wins
```
