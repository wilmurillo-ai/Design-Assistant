# OpenClaw Update

Check for OpenClaw updates and self-update the installation.

## Requirements

- `npm` — OpenClaw is distributed as an npm package

## Setup

No setup required. The skill uses the globally installed `openclaw` package.

## Usage

> "Check for updates"

> "Update openclaw"

> "Switch to beta channel"

## Install

```bash
clawhub install oc-self-update
```
