# Skill Builder

Create or optimize OpenClaw skills following official best practices. Guides the agent through a structured workflow from intent gathering to publish-ready skill.

## Requirements

- `jq` — for reading `openclaw.json` metadata

## Setup

No setup required. The skill is self-contained.

## Usage

**Create a new skill:**
> "I want to create a skill that tracks my reading list"

> "Build me a skill for checking server health"

**Optimize an existing skill:**
> "Review my reminder skill"

> "Prepare sm-saver for ClawHub publishing"

**Audit a SKILL.md:**
> "Check if my workout-track skill follows best practices"

## Install

```bash
clawhub install skill-build-helper
```
