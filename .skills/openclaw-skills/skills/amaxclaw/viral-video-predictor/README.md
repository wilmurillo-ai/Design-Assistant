# 短视频爆款预测脚本

> 基于历史爆款数据，预测短视频脚本的爆款概率

## 功能介绍

专为短视频创作者、MCN 机构设计的爆款预测工具。分析脚本的钩子、节奏、情绪曲线，给出优化建议。

### 📊 输出内容

| 模块 | 内容 |
|------|------|
| 爆款评分 | 0-100 分，预测爆款概率 |
| 钩子分析 | 前 3 秒吸引力检测 + 优化建议 |
| 节奏分析 | 信息密度、转折点、段落结构 |
| 情绪曲线 | 按时间轴的情绪变化建议 |
| 优化建议 | 具体的脚本改进方向 |
| 爆款案例 | 相似品类爆款参考 |
