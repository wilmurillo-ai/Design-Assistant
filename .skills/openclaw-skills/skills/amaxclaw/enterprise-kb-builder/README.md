# 企业级智能客服知识库构建器

> 上传产品文档/FAQ/历史对话，自动生成结构化知识库

## 功能介绍

专为企业客服团队、产品经理设计的知识库构建工具。自动解析原始内容，生成意图分类、话术优化、多轮对话设计。

### 🔍 核心功能

- **意图识别**：7 大类意图 + 30+ 子意图自动分类
- **话术优化**：按行业（电商/SaaS/教育/金融/医疗）和语气（专业/亲切/正式/活泼）优化
- **多轮对话**：自动设计对话流程，包含分支和 fallback
- **敏感词过滤**：5 类敏感词检测（绝对化用语/承诺风险/竞品攻击/违规内容/隐私风险）
- **覆盖度分析**：知识库覆盖度评估 + 缺失意图提示

### 📊 输出内容

- 结构化知识条目（意图/问题/答案/关键词/场景）
- 意图分类图谱（主意图/子意图/关联意图）
- 多轮对话流程设计
- 敏感词检查报告
- 知识库覆盖度分析 + 优化建议
