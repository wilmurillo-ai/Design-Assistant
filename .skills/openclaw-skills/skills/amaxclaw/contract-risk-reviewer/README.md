# 法律合同风险审查系统

> 专业级合同风险审查工具，自动识别 20+ 类常见风险条款

## 功能介绍

专为创业者、法务、自由职业者设计的合同风险审查工具。输入合同文本，自动识别风险并给出修改建议和法律依据。

### 🔍 20+ 类风险识别

| 类别 | 风险等级 | 说明 |
|------|---------|------|
| 违约责任不明确 | 🔴 高 | 未约定具体赔偿金额或计算方式 |
| 管辖权陷阱 | 🔴 高 | 约定在对方所在地，增加维权成本 |
| 权利义务不对等 | 🔴 高 | 一方权利明显多于另一方 |
| 知识产权归属不明 | 🔴 高 | 涉及成果交付但未约定归属 |
| 付款条款模糊 | 🔴 高 | 缺少金额/时间/条件 |
| 解除权不对等 | 🔴 高 | 仅单方享有解除权 |
| 免责条款过宽 | 🔴 高 | 可能免除故意/重大过失责任 |
| 连带责任风险 | 🔴 高 | 可能承担超出预期的责任 |
| 竞业限制不合理 | 🔴 高 | 缺少期限或补偿约定 |
| 个人信息保护缺失 | 🔴 高 | 未约定数据合规义务 |
| 保密条款缺失 | 🟡 中 | 涉及商业信息但未约定保密 |
| 合同期限缺失 | 🟡 中 | 未明确有效期限 |
| 自动续约陷阱 | 🟡 中 | 未约定提前通知期 |
| 通知送达条款 | 🟡 中 | 未约定视为送达规则 |
| 定金/押金风险 | 🟡 中 | 未明确退还条件 |
| 违约金过高 | 🟡 中 | 超过合同总额 30% |
| 合同转让限制 | 🟡 中 | 未限制单方转让 |
| 争议解决成本 | 🟡 中 | 未约定维权费用承担 |
| 不可抗力条款 | 🟢 低 | 未约定不可抗力处理 |
| 合同变更规则 | 🟢 低 | 未约定书面变更要求 |

### 📊 输出内容

- **风险评分**：0-100 分综合评分
- **风险清单**：按严重程度排序的详细列表
- **修改建议**：每条风险的安全条款模板
- **法律依据**：引用《民法典》《劳动合同法》等法律条文
- **整体建议**：审查摘要和优先级建议

## 使用示例

**输入**：
```json
{
  "contract_text": "甲方委托乙方提供技术服务...",
  "contract_type": "服务合同",
  "my_role": "乙方",
  "depth": "全面"
}
```

**输出**：
```
风险评分：45/100
识别风险：8 项（3 高 / 4 中 / 1 低）

🔴 高风险 1：违约责任条款存在但未明确赔偿金额
   建议：约定具体违约金金额和计算方式...
   法律依据：《民法典》第 584 条
   修改后条款：任何一方违约，应向守约方支付...
```
