# 竞品情报监控系统

> 输入竞品名称/链接，自动生成完整竞品分析报告

## 功能介绍

专为产品经理、市场运营、创业者设计的竞品情报利器。一键生成包含定价策略、营销动作、用户评价、SWOT 分析和应对策略的完整报告。

### 📊 报告模块

| 模块 | 内容 |
|------|------|
| 执行摘要 | 300 字核心发现 |
| 定价策略 | 检测 6 种定价策略 + 应对方案 |
| 营销分析 | 6 大营销渠道评估 |
| 用户评价 | 好评/差评点分析 + 情感分布 |
| SWOT | 竞品优势/劣势/机会/威胁 |
| 应对策略 | 短期/中期/长期行动建议 |

### 🏭 支持行业

电商 / SaaS / 教育 / 金融 / 内容 / 硬件 / 通用

### 💡 核心壁垒

- **行业分析框架**：每个行业有独立的分析维度
- **定价策略库**：6 种常见定价策略检测 + 应对方案
- **用户评价模式**：正/负面评价识别规则
- **应对策略引擎**：基于 SWOT 自动生成短期/中期/长期策略
