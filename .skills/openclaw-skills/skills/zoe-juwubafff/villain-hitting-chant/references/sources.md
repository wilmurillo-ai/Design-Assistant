# 打小人仪式参考资料
## 文化来源
香港打小人是岭南地区传统的民间民俗仪式，最知名的聚集地为香港铜锣湾鹅颈桥底，通常由资深的神婆（打小人婆）主持，惊蛰前后为高峰期，核心作用是帮信众宣泄负面情绪、驱走小人、化解霉运、祈求贵人扶持。

## 传统仪式流程
1. 奉神：请神助力
2. 确认小人：指定要打的小人对象
3. 打小人：用鞋底拍打纸小人，配合口诀
4. 化解：烧纸虎、化解小人煞气
5. 祈福：为信众祈求顺风顺水、贵人相助

## 传统口诀规范
- 必须为粤语口语，有节奏、有气势，对应身体部位的诅咒
- 核心句式为「打你个XX，等你XX」，符合民间传统韵律
- 内容以宣泄情绪、化解霉运为主，不涉及极端恶意的违法内容

## 语言规范
- 粤语口语优先，可适配港普，禁用书面化、官方化话术
- 必须保留地道粤语俚语，如「冇定抖」「仆街」「啃亲」等传统表述
