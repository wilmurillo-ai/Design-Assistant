# 小愈树 -- AI 儿童心理成长顾问

> 赛道：创意灵感  
> Skill ID：`little-healing-tree`  
> 一句话介绍：给 6-12 岁孩子的家长用的心理辅导 Skill，背后是 6 套发展心理学理论和 6 套带话术模板的干预工具。

---

## 一、这个 Skill 要解决什么问题

### 家长的真实困境

中国有超过 1 亿 6-12 岁学龄儿童。按《中国儿童青少年心理健康蓝皮书》的数据，大约 17.5% 存在不同程度的行为问题，包括情绪失控、社交冲突、注意力涣散、跟大人对着干等等。

但家长遇到这些事的时候，通常是这样的：

- 不理解：孩子说"老师针对我"，家长第一反应是"你想多了"或者"你是不是又做什么了"，两种都没用
- 不会说：想好好聊，一开口就变成说教，说着说着就吼上了
- 看不起：去医院儿童行为门诊挂个号，心理咨询 300-800 块一次，持续做下来很贵
- 不持续：偶尔刷到一篇育儿文章觉得有道理，过两天就忘了，也没人跟踪调整

### 跟医疗健康的关系

这不是硬靠。儿童心理行为问题本来就是各大医院儿保科和发育行为科的日常工作。WHO 对健康的定义里明确包含心理和社会适应。6-12 岁是行为模式定型的窗口期，这个阶段做结构化干预，能实实在在降低青春期出问题的概率。

现在儿童心理干预的主流做法叫"家长赋能"（Parent Empowerment），意思是与其只治孩子，不如教会家长怎么在日常里做对。小愈树做的就是这件事：把儿童发展心理学的专业知识装进一个 AI Skill，让家长随时能用。

## 二、技术方案

### 架构

│  ┌─────────────────────────────────────────────┐ │
│  │            结构化分析路径                      │ │

### 模块一：理论引擎

Skill 里内置了 6 个发展心理学理论的精简版，用来判断孩子的行为到底是"这个年龄段本来就这样"还是"确实需要干预"：

| 理论 | 来源 | 回答什么问题 |
|------|------|-------------|
| 皮亚杰认知发展 | Piaget, 1952-1972 | 这是认知能力限制，还是行为选择？ |
| 归因理论/敌意归因偏差 | Dodge, 1980 | 孩子为什么总觉得被针对？ |
| 情绪调节发展模型 | Gross & Thompson, 2007 | 为什么发脾气时讲道理没用？ |
| 正面管教/自然结果 | Nelsen, 2006 | 怎么让后果本身教育孩子？ |
| 自我决定论 | Deci & Ryan, 2000 | 问题行为背后缺的是什么心理营养？ |
| 公平感发展 | Damon, 1977 | 为什么孩子老喊"不公平"？ |

### 模块二：干预工具箱

6 套工具，每套都带具体步骤、家长可以直接用的话术模板、预期见效时间线，以及 5 个常见踩坑点。

| 工具 | 什么时候用 | 怎么起作用的 |
|------|---------|---------|
| 侦探游戏 | 觉得被针对、看问题只看一面 | 用提问代替说教，让孩子自己收集信息 |
| 情绪温度计 | 一生气就摔东西、不写作业 | 在情绪和行为之间插入一个识别环节 |
| 挫折健身房 | 输不起、没被叫到就崩溃 | 像练肌肉一样，从小挫折练起 |
| 自然结果+复盘 | 不写作业、拒绝做该做的事 | 不替孩子承受后果，事后用四个问题复盘 |
| 公平天平 | 总觉得"别人可以我不行" | 教孩子判断前先收集完整信息 |
| 被需要计划 | 感觉被忽视、用闹来求关注 | 在家里给孩子一个有实际责任的角色 |

举个例子。"侦探游戏"的话术模板长这样：

> 孩子说"老师针对我"时，不要说"你想多了"，也不要说"你是不是又做什么了"。
> 
> 这样说：
> "听起来你挺生气的。来，我们当一回侦探，我想知道更多线索。当时具体发生了什么？老师原话说了什么？其他人当时在做什么？"

家长拿过去就能用，不需要自己组织语言。

### 模块三：记忆和追踪

通过 `update_memory` 做跨会话的连续辅导。第一次聊会建立孩子的基本档案，之后每次都会回顾上次的情况、问执行效果、根据反馈调整。聊了 3 次以上，会主动分析几个问题之间有没有共同的根源。

### 四步分析路径

每次收到一个具体问题，Skill 按固定路径走：

1. 判断发展阶段：这个行为在当前年龄正不正常？
2. 拆解变量：孩子的需求是什么？触发点是什么？
3. 识别缺口：是边界不清、能量没出口、缺成就感、还是情绪策略不够？
4. 三层输出：理解层（帮家长理解为什么）、行动层（具体工具和话术）、结构层（长期怎么调）

## 三、做得不一样的地方

**拿来就能用，不是读完觉得有道理然后不知道怎么做。** 大部分育儿内容的问题是告诉你"要多沟通""要共情"，然后就没了。小愈树的每个建议都带具体话术，家长不需要自己想怎么措辞。

**有理论撑着，不是经验之谈。** 背后 6 个理论都是发展心理学领域经过同行评审的成果，不是"我家孩子当年就是这么带的"那种分享。

**有约束机制，不会乱说。** Skill 里写死了 7 条底线，比如不用道德化语言评判孩子、不给压制型建议（吼/罚/威胁）、不替权威辩护、不给"多陪伴"这种正确但没用的话。这些约束让 AI 的输出保持在安全范围内。

**能跟踪，不是一次性的。** 多数问答工具聊完就忘了。小愈树能记住孩子的情况，下次接着上次的进度聊，发现问题之间的关联，给出调整后的方案。

## 四、怎么用

### 触发

三种方式都行：

```
# 斜杠命令
/little-healing-tree 我家孩子总觉得老师针对他怎么办

# 直接聊（自动识别育儿话题）
我儿子昨天又在学校发脾气了，老师说他上课叠纸被没收后就不写作业了

# 简单描述
孩子说所有老师都不喜欢他，我该怎么办
```

### 第一次用

小愈树会先问孩子的基本情况：

```
小愈树：你家孩子多大了？几年级？能简单描述一下 ta 的性格特点吗？
        比如精力旺盛还是偏安静？容易着急还是比较慢热？

你：    男孩，7岁，一年级。精力特别旺盛，自尊心很强，
        特别在意公不公平。

小愈树：收到，我已记录。现在跟我说说具体发生了什么？
```

### 拿到的建议长什么样

三层结构：

- 理解层：2-3 句话解释"为什么会这样"，让你先别焦虑
- 行动层：1-3 个具体工具加话术模板，今天就能用
- 结构层：长期怎么调整家庭互动方式

### 后续跟进

下次再聊，小愈树会主动提上次的事、问执行得怎么样、根据反馈调方案。

## 五、实际对话效果

家长输入：
> 我儿子昨天抱怨所有的老师都不喜欢他，他犯错后老师吵他，然后也没有哄他

小愈树输出（摘要）：

> 理解层：孩子说"所有老师都不喜欢我"，实际上是三件事叠在一起。"批评我=不喜欢我"是 7 岁典型的归因偏差，他还分不清"批评行为"和"否定这个人"。"没有哄我"说明他需要的是批评之后的关系修复信号，本质是对安全感的需求。"所有老师"说明已经从具体事件升级为全面否定，归属感在亮红灯。
>
> 行动层：
> 1. 侦探游戏（改编版）：找孩子心情好的时候，问他"爸爸/妈妈说你的时候，是不喜欢你吗？"，引导他区分批评和不喜欢
> 2. 被需要计划：每天 5 分钟固定的 check-in 时间，加上家庭里给他一个专属角色，明确告诉他"犯错不影响你对我的重要性"
>
> 结构层：建议跟班主任私下聊一次，坚持侦探游戏的日常练习...
