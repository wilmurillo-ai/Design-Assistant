# cp2k-crosscode-input-studio

A publish-ready OpenClaw skill for computational chemistry and materials workflows.

It does two core things well:
1. Generate conservative CP2K input drafts from natural-language requests and structure files.
2. Convert those CP2K drafts into draft inputs for other electronic-structure codes.

## What it supports

### CP2K draft generation
- Single-point energy
- Geometry optimization
- Cell optimization
- Short MD draft setup
- Vibrational-analysis draft setup

### Cross-code draft conversion
- Gaussian (`.gjf`)
- VASP (`INCAR`, `POSCAR`, `KPOINTS`)
- ORCA (`.inp`)
- Quantum ESPRESSO (`pw.in`)

## Design philosophy

- Prefer conservative defaults over fake certainty
- Prefer runnable drafts over overcomplicated automation
- Always surface warnings when physical meaning may change
- Do not pretend different codes are exactly equivalent

## Best use case

This skill works best when:
- the user describes a calculation in plain language
- a structure file is available (`xyz`, `cif`, etc.)
- the goal is to obtain a practical first-pass input draft
- the user may later port that draft to another code

## Important limits

- This is a **draft generator and translator**, not a convergence-validation engine
- Cross-code conversion is best for **CP2K drafts generated by this skill itself**
- Arbitrary hand-written CP2K inputs are only partially supported
- Pseudopotential/basis/library equivalence across codes is not guaranteed
- `POTCAR` for VASP and exact QE pseudopotential filenames are **not** fabricated
- The bundled scripts do **not** require a local CP2K executable or installed CP2K basis/potential libraries; when exact library-resolved names are unavailable, the output stays conservative and uses placeholders or warnings

## Key files

- `SKILL.md` — OpenClaw skill definition
- `scripts/normalize-request.py` — normalize request JSON
- `scripts/render-cp2k-input.py` — render CP2K draft input
- `scripts/generate_cp2k_bundle.py` — one-shot CP2K bundle generation
- `scripts/convert-cp2k-input.py` — convert CP2K drafts to other codes
- `examples/example-structures.md` — plain-text structure snippets for local testing

## Example workflow

Generate a CP2K bundle:

```bash
python3 scripts/generate_cp2k_bundle.py examples/water_request.json examples/water.xyz -d out
```

Convert the generated CP2K input to Gaussian:

```bash
python3 scripts/convert-cp2k-input.py out/job.inp gaussian -o out_gaussian
```

Convert the generated CP2K input to VASP:

```bash
python3 scripts/convert-cp2k-input.py out/job.inp vasp -o out_vasp
```
