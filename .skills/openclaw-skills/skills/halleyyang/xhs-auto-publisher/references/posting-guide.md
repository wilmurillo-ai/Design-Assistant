# 帖子创作规范

通用规范，适用于任意垂直领域的小红书账号运营。

---

## 标题规范

- **字数**：严格 ≤20字（超出会被小红书截断！）
- **风格**：有个人感受（惊讶/好奇/有用），像朋友在说话
- **禁用烂大街句式**：「我测了一下」「刷到震惊的」「太好用了吧」「绝了」「OMG」

**好标题示例**：

| 领域 | 示例 |
|------|------|
| 知识科普 | `清明节，比你想的更厚重` |
| 生活方式 | `这个习惯我坚持了三年` |
| 美食 | `这碗面我吃了十年` |
| 旅行 | `去了才知道有多值` |
| 职场 | `入职三个月，我学到什么` |
| 情感 | `一个人也可以过得很好` |

---

## 正文规范

- **字数**：200-400字
- **结构**：开门见山说事 → 具体细节（读者能理解的角度）→ 个人感受/观点 → 结尾互动问句
- **语气**：朋友聊天，可以有立场，不要面面俱到
- **格式禁忌**：不用 `━━━` 横线分割，不过度堆 emoji，一段话不超过4行

**结尾互动示例**：
- `你有没有类似的经历？`
- `你觉得这种做法值得试试吗？`
- `有同感的来聊聊`

---

## 话题标签策略

每次选 **3-5个**，保持多样性，不要每次选一样的。

**通用高流量话题**（任何内容都可考虑）：

| 话题 | 适用场景 |
|------|----------|
| 知识分享 | 科普/干货类 |
| 生活方式 | 日常/习惯类 |
| 传统文化 | 节日/历史/文化类 |
| 情感共鸣 | 情感/心理类 |
| 好物推荐 | 产品/工具类 |
| 旅行攻略 | 旅游/出行类 |
| 美食探店 | 餐饮/食谱类 |
| 职场干货 | 工作/成长类 |

**加 1-2 个与内容高度相关的具体标签**（如「清明节」「春日旅行」「职场新人」等）。

> 💡 根据你的账号垂直领域，固定 2-3 个主标签，其余灵活替换。

## 封面图模板

脚本内置 6 套封面模板，根据标题 hash **自动选择**，同一标题始终对应同一套模板。

| 编号 | 名称 | 风格 | 适合内容 |
|------|------|------|----------|
| 0 | 珊瑚红·撞色 | 上下分割，大字冲击 | 热点、节日、话题性内容 |
| 1 | 奶油白·极简 | 白底黑字，高级留白 | 知识、观点、深度内容 |
| 2 | 墨绿·自然风 | 文艺清新 | 人文、生活、旅行、历史 |
| 3 | 深蓝·知性风 | 沉稳理性 | 科普、职场、财经 |
| 4 | 柠黄·活力风 | 明亮年轻 | 美食、穿搭、娱乐 |
| 5 | 丁香紫·治愈风 | 圆润柔和 | 情感、心理、治愈系 |

**subtitle 写法**：
- 2-3句话，用 `\n` 换行，总字数 30-50字
- 是对标题的补充说明，引发读者好奇

**好的 subtitle 示例**：
```
「它不只是扫墓\n藏着2500年的家国情怀\n今天聊聊它真正的故事」
「不需要大预算\n三个动作养成好习惯\n我坚持了整整一年」
「很多人都做错了\n这个细节改变之后\n整个状态都不一样了」
```
