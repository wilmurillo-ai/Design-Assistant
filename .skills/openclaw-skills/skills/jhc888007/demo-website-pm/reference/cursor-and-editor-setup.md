# Cursor 与编辑器准备

## 适用场景
- 你第一次用 Cursor。
- 你要打开网站项目并开始改内容。
- 你是零技术用户，需要按步骤操作。

## 前置准备
- 一台可联网电脑。
- 可正常打开浏览器。
- 一个可用邮箱（用来注册账号）。
- 你已拿到项目文件夹路径。

## 页面英文看不懂时怎么做
- 在浏览器里右键页面，点“翻译成中文”。
- 看不懂按钮文字时，先看位置：右上角常见“登录/注册”，中间常见“继续/下一步”。
- 用手机拍照翻译也可以，重点看按钮和报错红字。
- 看到不认识的英文，不要猜，先截图再问。

## 步骤清单
1. 打开浏览器，进入 Cursor 官网下载页。  
   你会看到什么：页面上有下载按钮，通常在中间或右上。
2. 点击下载按钮。  
   你会看到什么：浏览器开始下载安装包。
3. 双击安装包并完成安装。  
   你会看到什么：桌面或应用列表里出现 Cursor 图标。
4. 打开 Cursor，选择“登录”或“注册”。  
   你会看到什么：出现邮箱登录页面，要求输入邮箱或验证码。
5. 完成登录后，点击“Open Folder（打开文件夹）”。  
   你会看到什么：系统弹出文件夹选择窗口。
6. 选中你的项目文件夹并确认。  
   你会看到什么：左侧出现项目文件列表。
7. 在左侧点开一个文件，确认可以编辑。  
   你会看到什么：中间区域显示文件内容，光标可输入文字。
8. 按 `Command + S`（mac）或 `Ctrl + S`（Windows）保存一次。  
   你会看到什么：文件不再显示“未保存”提示。

## 成功判定
- 你能正常打开 Cursor。
- 你已登录账号。
- 你能打开项目文件夹。
- 你能修改并保存任意一个文件。

## 常见问题与处理
- 问题：下载很慢或失败。  
  处理：先换网络，再刷新页面重试；仍失败就换浏览器。
- 问题：登录收不到验证码。  
  处理：先查垃圾邮件；等 1 分钟后重发；确认邮箱地址无拼写错误。
- 问题：看不到项目文件。  
  处理：重新点“Open Folder”，确认选的是项目根目录，不是上一级目录。
- 问题：无法保存文件。  
  处理：检查文件是否只读；把项目移到“文稿/桌面”等可写目录后重试。

## 何时中止并回报
- 连续 3 次安装失败。
- 连续 3 次登录失败（含验证码问题）。
- 已确认选对文件夹，但文件仍不显示。
- 出现系统权限弹窗且你不确定是否允许。
- 中止后请回报：你做到哪一步、看到了什么提示、发一张截图。
