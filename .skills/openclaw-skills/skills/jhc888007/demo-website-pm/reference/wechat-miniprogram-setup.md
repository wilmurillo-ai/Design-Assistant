# 微信开发者工具与小程序注册指南

**目标受众**：零技术基础用户
**使用场景**：当用户选择制作“微信小程序”且状态处于 S0（准备环境阶段）时，读取此文档一步步指导。不要向用户询问 ID，让用户自己查找并填写到 环境变量 或 网站配置 中。

---

## 步骤 1：注册小程序账号（获取“基础户口”）
*话术引导：要让别人能在微信里看到你的程序，我们需要先去腾讯那里免费注册一个“户口”。*

1. **打开官网**：在电脑浏览器打开 [微信公众平台](https://mp.weixin.qq.com/)。
2. **开始注册**：点击右上角的「立即注册」，账号类型选择**「小程序」**。
3. **填写信息**：用一个没注册过微信公众号的邮箱进行注册，跟着网页提示激活邮箱、登记主体信息（个人做 demo 选择“个人”即可）。
4. **获取 AppID**：注册成功并登录后，在左侧菜单找到 **「开发」 -> 「开发管理」 -> 点击「开发设置」**标签页。
5. **复制保存**：找到 **AppID (小程序ID)**，这是一串字母和数字组合，把它复制下来并记下来（这是你小程序的唯一身份证，后面写代码必须用到）。

## 步骤 2：完成认证与备案（给小程序上“牌照”）
*话术引导：根据现在的规定，所有小程序都需要实名“上牌照”才能正式发给别人看。这个手续审核大概需要几天，但别担心，我们现在先把它提交上去，这期间完全不耽误我们在电脑上继续做 demo 看效果！*

1. **微信认证（确认身份）**：
   - 在左侧菜单找到 **「设置」 -> 「基本设置」**。
   - 往下看找到 **「微信认证」**，点击「详情」或「开通」。
   - 跟着向导填写资料。注意：微信官方对个人开发者会收取 30 元/年的身份验证审核费（这是直接交给腾讯的）。
2. **小程序备案（合法登记）**：
   - 同样在 **「设置」 -> 「基本设置」** 页面里，找到 **「小程序备案」** 入口。
   - 点击去备案，它会像网购填地址一样让你补充一些基本信息，需要你上传身份证正反面照片，并用你的手机微信扫屏幕上的二维码，做一次刷脸验证。
   - **提交后就不用管了**。审核进度它会通过微信通知你，我们直接进入下一步！

## 步骤 3：下载“施工工具”
*话术引导：户口和牌照都在办理中了，接下来我们需要安装一个类似于“工坊”的软件，用来把大模型写的代码变成真正的小程序界面。*

1. **下载链接**：打开 [微信开发者工具下载页](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)。
2. **选择版本**：点击 **「稳定版 Stable Build」**，根据你的电脑是 Windows 还是 Mac 下载对应版本。
3. **安装并登录**：像装普通软件一样安装好。打开后，用你**刚才注册小程序的那个微信**扫码登录。

## 步骤 4：开通“云开发”（存放数据的云端仓库）
*话术引导：我们的 demo 需要记录数据（比如用户填的表单），微信提供了一个非常方便的“云仓库”，我们去把它打开。*

1. **新建空项目**：在开发者工具里点击「+」号新建项目。填入刚才的 AppID，后端服务选择 **「微信云开发」**，点击确定。
2. **点击云开发按钮**：进入主界面后，点击最上方工具栏里的 **「云开发」** 按钮（一朵小云的图标）。
3. **开通服务**：弹出的窗口里点击「开通」，同意协议。（*注：云开发有免费的基础版或几块钱的测试套餐，根据页面提示开通最基础的即可*）。
4. **获取环境 ID**：开通成功后，在云开发控制台的「概览」或「设置」里，找到你的 **「环境 ID」**（通常是类似 `test-1234abcd` 的格式）。把它复制发给我。

**本阶段结束标志**：用户成功安装了微信开发者工具，了解了备案的必要性，并确认完成了 ID 配置。
