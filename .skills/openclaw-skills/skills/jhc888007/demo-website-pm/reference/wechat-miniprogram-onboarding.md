# 微信小程序真机预览与上线体验指南

**目标受众**：零技术基础用户
**使用场景**：当代码已经由 AI 生成，用户需要把代码跑起来，或者想把做好的 demo 发给朋友看（状态 S3 阶段）时使用。不要向用户询问 ID，让用户自己查找并填写到 环境变量 或 网站配置 中。

---

## 阶段 A：在电脑上跑通（本地预览）
*话术引导：代码已经写好了！现在我们把它放进你的“工坊”里，看看长什么样。*

1. **导入代码文件夹**：打开「微信开发者工具」，点击主界面的「导入项目」（或者点「+」号）。
2. **选择目录**：选中 AI 帮你生成的那整个代码文件夹。`AppID` 填你之前注册的那个，点击「确定」。
3. **填入环境 ID**：
   - 找到代码目录里的 `app.js` 文件，点击打开。
   - 里面有一行写着 `env: '你的云开发环境ID'`，把它替换成你真实的云开发环境 ID（保留单引号）。
   - 保存文件。
4. **编译看效果**：点击顶部工具栏最左侧的 **「编译」** 按钮。如果一切正常，工具左侧的“手机模拟器”里就会显示出你的小程序界面了！你能看到画面吗？

*(如果报错，请用户提供模拟器下面的“控制台”最后几行红字截图，转入排查流程。)*

## 阶段 B：发给朋友看（上传与体验版）
*话术引导：电脑上看没问题了，现在我们把它变成一个可以发给朋友扫一扫的二维码！*

1. **点击上传**：在「微信开发者工具」顶部工具栏，点击 **「上传」** 按钮（一个向上箭头的图标）。
2. **填版本号**：弹出的窗口里，版本号填 `1.0.0`，项目备注随便写（比如“第一个demo测试”），点击确定上传。
3. **设为体验版**：
   - 回到浏览器，登录 [微信公众平台](https://mp.weixin.qq.com/)。
   - 在左侧菜单找到 **「管理」 -> 「版本管理」**。
   - 往下拉找到「开发版本」，你会看到刚才上传的那条记录。点击右侧下拉箭头，选择 **「选为体验版」**。
4. **添加朋友的微信**（必需步骤，否则别人打不开）：
   - 在左侧菜单找 **「管理」 -> 「成员管理」**。
   - 点击「编辑」->「添加成员」，输入你朋友的微信号，给他们勾选上“体验者”权限。
5. **分享体验**：回到「版本管理」，点击你的“体验版”二维码，把这幅图发给刚才添加的朋友，他们用微信扫一扫就能体验你的产品啦！

**本阶段结束标志**：用户成功在手机上扫码看到了自己的小程序 demo，或遇到了具体报错需要排查。
