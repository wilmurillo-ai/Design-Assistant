# 黄金走势分析 Skill

全面分析黄金价格走势，整合多维度数据，生成专业的分析报告。

## 功能说明

### 核心能力
1. **数据源整合**
   - 金十数据：实时金价、财经日历、宏观数据
   - 金价网站：Kitco、Bloomberg、新浪财经
   - 财经新闻：律动 BlockBeats、华尔街见闻

2. **分析维度**
   - 技术面：K线、均线、MACD、RSI、布林带
   - 基本面：美元指数、实际利率、通胀数据、央行政策
   - 情绪面：CFTC持仓、恐慌指数、市场情绪指标
   - 地缘政治：战争、大选、地缘风险事件

3. **输出内容**
   - 当前金价及近期走势
   - 技术指标分析（支撑位/阻力位）
   - 宏观因素影响分析
   - 短期/中期/长期展望
   - 可操作的投资建议

## 使用方式

### 快速开始

```bash
# 生成完整分析报告
gold-analysis --full

# 只看技术面
gold-analysis --technical

# 只看基本面
gold-analysis --fundamental

# 指定时间周期
gold-analysis --period daily
gold-analysis --period weekly
gold-analysis --period monthly
```

## 分析框架

### 一、技术面分析

#### 1. 价格走势
- 当前金价（美元/盎司、人民币/克）
- 24小时涨跌幅
- 近1周/1月/3月/1年走势
- 对比历史同期表现

#### 2. 技术指标
- **均线系统**：MA5、MA10、MA20、MA60、MA120、MA250
  - 价格与均线关系（站上/跌破）
  - 均线排列（多头/空头/震荡）

- **趋势指标**：MACD
  - DIF、DEA、柱状图
  - 金叉/死叉信号
  - 顶背离/底背离

- **震荡指标**：RSI (14)
  - 当前RSI数值
  - 超买（>70）/超卖（<30）区间
  - 背离信号

- ** volatility指标**：布林带
  - 上轨、中轨、下轨位置
  - 价格在布林带中的位置
  - 带宽变化（收敛/发散）

- **支撑与阻力**
  - 关键支撑位（近期低点、前期平台）
  - 关键阻力位（近期高点、前期平台）
  - 整数关口（2000、2100、2200、2300美元）

### 二、基本面分析

#### 1. 美元指数 (DXY)
- 当前美元指数
- 近期走势
- 与黄金的负相关性分析
- 美元走势展望（美联储政策、美国经济）

#### 2. 实际利率
- 美国10年期国债收益率
- 通胀预期（CPI、PCE）
- 实际利率 = 名义利率 - 通胀
- 实际利率与黄金的负相关性

#### 3. 通胀数据
- 美国CPI（同比、环比）
- 核心CPI
- PCE物价指数
- 通胀趋势分析

#### 4. 央行政策
- 美联储（FOMC）最新决议
- 利率路径预期（点阵图）
- 资产负债表变化（QT）
- 其他主要央行（欧央行、日央行、英国央行）

#### 5. 经济数据
- 非农就业数据
- GDP增速
- 制造业PMI
- 消费者信心指数

### 三、情绪面分析

#### 1. CFTC持仓报告
- 商业持仓（套保）
- 非商业持仓（投机）
- 净多头/净空头变化
- 持仓极值信号

#### 2. 市场情绪指标
- 贪婪恐慌指数
- 金价波动性（GVZ）
- 看跌/看涨期权比率
- ETF持仓变化（GLD、IAU）

#### 3. 媒体与舆情
- 主流媒体报道倾向
- 分析师观点汇总
- 社交媒体情绪
- 一致性预期（反向指标）

### 四、地缘政治与事件风险

#### 1. 地缘政治
- 战争与冲突（中东、乌克兰、台海）
- 大选与政治事件
- 国际关系变化
- 恐怖袭击与突发事件

#### 2. 经济事件
- 央行决议（FOMC、ECB等）
- 重要数据发布（CPI、非农、GDP）
- 地缘风险事件升级
- 黑色历史日期（9月、10月）

### 五、展望与建议

#### 1. 短期展望（1周-1月）
- 技术面信号
- 重要事件日历
- 操作建议（买入/卖出/观望）
- 止损止盈位置

#### 2. 中期展望（1月-3月）
- 趋势判断
- 关键价位
- 仓位建议
- 风险提示

#### 3. 长期展望（3月-1年）
- 宏观大趋势
- 战略配置建议
- 风险因素
- 预期收益空间

## 最佳实践

1. **数据验证**：多个数据源交叉验证，避免单一数据源错误
2. **周期搭配**：短中长期结合，避免单一周期误判
3. **动态跟踪**：重大事件发生后及时更新分析
4. **风险意识**：任何分析都要有止损，不要盲目乐观
5. **持续学习**：定期回顾历史分析，总结经验教训
