# 黄金技术指标参考

## RSI（相对强弱指数）

- **计算周期**：通常 14 日
- **超买区间**：RSI > 70，市场可能过热
- **超卖区间**：RSI < 30，市场可能超卖
- **背离信号**：价格创新高但 RSI 走低 → 潜在顶背离

## MACD（指数平滑异同移动平均线）

- **组成**：DIF 线、DEA 线、MACD 柱状图
- **金叉信号**：DIF 上穿 DEA，看涨
- **死叉信号**：DIF 下穿 DEA，看跌
- **零轴位置**：零轴上方偏多，下方偏空

## 移动平均线（MA）

- **短期均线**：5 日、10 日
- **长期均线**：30 日、60 日、120 日
- **金叉**：短期上穿长期，多头信号
- **死叉**：短期下穿长期，空头信号

## 支撑位与阻力位

- **支撑位**：价格下跌时的底部支撑
- **阻力位**：价格上涨时的顶部压力
- **整数关口**：如 5000、5100、5200 美元/盎司
