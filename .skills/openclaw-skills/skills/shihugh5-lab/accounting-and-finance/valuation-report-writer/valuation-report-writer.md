# 估值报告撰写

## 📚 一、报告结构

### 1.1 标准章节

```
1. 执行摘要
2. 公司概况
3. 行业分析
4. 财务分析
5. 估值分析
6. 敏感性分析
7. 投资建议
8. 风险提示
9. 附录
```

### 1.2 报告要素

| 要素 | 说明 |
|:---|:---|
| **投资评级** | 买入/持有/卖出 |
| **目标价** | 具体价格区间 |
| **关键假设** | 核心参数假设 |
| **风险因素** | 主要风险点 |

---

## 📝 二、撰写要点

### 执行摘要

- 一页纸总结核心结论
- 投资评级和目标价
- 3-5个关键投资逻辑

### 估值部分

- 多种估值方法结果
- 估值区间而非单点
- 与当前股价对比

### 投资建议

- 明确的评级
- 目标价和时限
- 触发条件（上调/下调）
