# 自由现金流计算：从利润到现金

## 📚 一、理论背景与核心概念

### 1.1 为什么利润不等于现金

**会计利润的问题**：
- 包含非现金项目（折旧、摊销）
- 收入确认与现金收付时点不一致
- 资本支出与费用化处理差异

**案例**：
```
某公司今年：
- 销售收入1亿元（其中30%未回款）
- 购买设备5000万（资本化，10年折旧）
- 会计利润：3000万
- 实际现金流：-2000万
```

### 1.2 自由现金流的定义

**核心概念**：企业在维持正常经营和必要增长后，可以自由支配的现金。

**两个层次**：

| 层次 | 定义 | 用途 |
|:---|:---|:---|
| **保全性FCF** | 维持现有规模产生的现金流 | 评估生存底线 |
| **扩张性FCF** | 支持增长后的剩余现金流 | 分红、偿债、并购 |

### 1.3 自由现金流的理论基础


> "自由现金流是企业经营活动创造的、扣除维持和增长所需资本支出后的剩余现金，是企业真实价值创造能力的体现。"

**价值创造逻辑**：
```
企业价值 = ∑ 未来自由现金流 / (1+WACC)^t
```

### 1.4 与会计指标的本质区别

| 指标 | 性质 | 问题 | FCF优势 |
|:---|:---|:---|:---|
| 净利润 | 会计概念 | 易受操纵 | 现金真实 |
| 经营现金流 | 现金概念 | 含营运资本波动 | 剔除波动性 |
| EBITDA | 近似概念 | 忽视资本支出 | 考虑资本需求 |
| **自由现金流** | **价值概念** | **完整准确** | **真实价值** |

## 📝 二、详细操作步骤

### 步骤1：获取基础数据

**所需报表**：
- 重构后的资产资本表（期初、期末）
- 重构后的股权价值增加表
- 现金流量表
- 财务报表附注（资本支出明细）

**核心数据字段**：
```
利润表数据：
- 税后经营净利润(NOPAT)
- 折旧与摊销
- 营运资本变动

资产负债表数据：
- 经营资产变动
- 经营负债变动
- 固定资产原值变动

现金流量表数据：
- 经营活动现金流
- 购建固定资产支付的现金
```

### 步骤2：计算营业现金流

**方法一：从NOPAT出发**
```
营业现金流 = NOPAT + 折旧与摊销 - 营运资本增加
```

**方法二：从经营现金流出发**
```
营业现金流 = 经营活动现金流
           + 利息支出 × (1-税率)
           - 金融性投资收益
```

**营运资本计算**：
```
营运资本 = 经营流动资产 - 经营流动负债
         = (货币资金(经营) + 应收票据 + 应收账款 + 预付款项 + 存货)
         - (应付票据 + 应付账款 + 预收款项 + 合同负债 + 应付职工薪酬 + 应交税费)

营运资本增加 = 期末营运资本 - 期初营运资本
```

### 步骤3：计算资本支出

**资本支出(CapEx)构成**：
```
资本支出 = 维持性资本支出 + 扩张性资本支出

维持性资本支出 = 折旧与摊销（近似）
扩张性资本支出 = 总资本支出 - 维持性资本支出
```

**数据获取**：
```
总资本支出 = 购建固定资产、无形资产支付的现金
           + 取得子公司支付的现金
           - 处置固定资产收回的现金
```

### 步骤4：计算两个层次的自由现金流

**保全性自由现金流**：
```
FCF_保全 = NOPAT + 折旧与摊销 - 营运资本增加 - 维持性资本支出
         = NOPAT - 营运资本增加
```

**扩张性自由现金流（企业自由现金流）**：
```
FCF_企业 = NOPAT + 折旧与摊销 - 营运资本增加 - 总资本支出
         = NOPAT - 营运资本增加 - 扩张性资本支出
```

### 步骤5：计算股权自由现金流

**从企业FCF到股权FCF**：
```
FCF_股权 = FCF_企业 - 税后利息支出 + 净债务增加
         = FCF_企业 - 利息支出 × (1-税率) + (新增借款 - 偿还借款)
```

**简化公式**：
```
FCF_股权 = 净利润 + 折旧与摊销 - 营运资本增加 - 资本支出 + 净债务增加
```

## 🔢 三、公式与计算逻辑

### 3.1 自由现金流公式体系

**企业自由现金流(FCFF)**：
```
FCFF = EBIT × (1 - T) + 折旧与摊销 - 资本支出 - 营运资本增加
     = NOPAT + 折旧与摊销 - 资本支出 - 营运资本增加
```

**股权自由现金流(FCFE)**：
```
FCFE = 净利润 + 折旧与摊销 - 资本支出 - 营运资本增加 + 净债务增加
     = FCFF - 利息支出 × (1 - T) + 净债务增加
```

### 3.2 现金流层次结构

```
                    营业收入
                       │
                       ▼
              ┌─────────────────┐
              │   减：营业成本   │
              │   减：经营税费   │
              │   减：经营费用   │
              └────────┬────────┘
                       ▼
                     EBIT
                       │
                       ▼
              EBIT × (1 - T)
                       │
                       ▼
                    NOPAT
                       │
                       ▼
              ┌─────────────────┐
              │  + 折旧与摊销    │
              │  - 营运资本增加  │
              └────────┬────────┘
                       ▼
                 营业现金流
                       │
                       ▼
              ┌─────────────────┐
              │  - 维持性资本支出│ ← 保全性自由现金流
              │  - 扩张性资本支出│
              └────────┬────────┘
                       ▼
               企业自由现金流
                       │
                       ▼
              ┌─────────────────┐
              │  - 税后利息支出  │
              │  + 净债务增加    │
              └────────┬────────┘
                       ▼
               股权自由现金流
```

### 3.3 资本支出的估算方法

| 方法 | 公式 | 适用场景 |
|:---|:---|:---|
| **直接法** | 现金流量表"购建固定资产" | 数据完整时 |
| **间接法** | 固定资产增加 + 折旧 | 快速估算 |
| **维持法** | ≈ 折旧与摊销 | 成熟企业 |
| **收入比例法** | 收入 × 行业CapEx率 | 预测时 |

## 💡 四、实例演示：万华化学

### 4.1 基础数据

**万华化学2023年数据**（单位：亿元）：

| 指标 | 2023年 | 2022年 | 说明 |
|:---|---:|---:|:---|
| 营业收入 | 1753.0 | 1655.0 | |
| NOPAT | 296.2 | 265.0 | 已重构 |
| 折旧与摊销 | 85.0 | 75.0 | 附注 |
| 经营流动资产 | 338.0 | 310.0 | 已重构 |
| 经营流动负债 | 245.0 | 225.0 | 已重构 |
| 固定资产原值 | 1050.0 | 920.0 | 附注 |
| 资本支出 | 180.0 | 150.0 | 现金流量表 |

**计算中间指标**：
```
营运资本(2023) = 338.0 - 245.0 = 93.0亿
营运资本(2022) = 310.0 - 225.0 = 85.0亿
营运资本增加 = 93.0 - 85.0 = 8.0亿

固定资产增加 = 1050.0 - 920.0 = 130.0亿
```

### 4.2 自由现金流计算

**营业现金流**：
```
= NOPAT + 折旧与摊销 - 营运资本增加
= 296.2 + 85.0 - 8.0
= 373.2 亿元
```

**保全性自由现金流**：
```
= NOPAT - 营运资本增加
= 296.2 - 8.0
= 288.2 亿元

或：
= 营业现金流 - 折旧与摊销
= 373.2 - 85.0
= 288.2 亿元 ✓
```

**企业自由现金流**：
```
= NOPAT + 折旧与摊销 - 营运资本增加 - 资本支出
= 296.2 + 85.0 - 8.0 - 180.0
= 193.2 亿元
```

**扩张性资本支出**：
```
= 总资本支出 - 折旧与摊销
= 180.0 - 85.0
= 95.0 亿元
```

### 4.3 自由现金流分析

| 指标 | 金额 | 占收入比 | 评价 |
|:---|---:|---:|:---|
| NOPAT | 296.2 | 16.9% | 盈利能力强 |
| 营业现金流 | 373.2 | 21.3% | 现金创造能力强 |
| 保全性FCF | 288.2 | 16.4% | 生存底线充足 |
| 企业FCF | 193.2 | 11.0% | 价值创造良好 |
| 资本支出 | 180.0 | 10.3% | 扩张投入大 |

**FCF质量分析**：
```
FCF转化率 = 企业FCF / NOPAT
         = 193.2 / 296.2
         = 65.2%

评价：FCF转化率较高，说明盈利质量良好，
      资本支出虽大但仍在可控范围。
```

## ⚠️ 五、常见问题与注意事项

### 5.1 营运资本变动的波动性

**问题**：营运资本变动大，导致FCF波动

**处理建议**：
1. 使用多年平均营运资本变动
2. 分析变动原因（应收/存货/应付）
3. 区分正常增长vs管理问题

**示例**：
```
情形A：营收增长20%，应收增长50% → 管理问题 ⚠️
情形B：营收增长20%，应收增长20% → 正常增长 ✓
```

### 5.2 资本支出的周期性

**问题**：资本支出大小年差异大

**处理建议**：
1. 使用3-5年平均资本支出
2. 区分维持性vs扩张性
3. 了解公司资本开支计划

### 5.3 负自由现金流的解读

**情形1：成长期企业**
- 大规模资本支出
- FCF为负正常
- 关注投资回报

**情形2：衰退期企业**
- 经营利润下滑
- FCF为负危险
- 关注生存风险

**情形3：营运资本恶化**
- 应收/存货激增
- 需要调查原因
- 可能是管理问题

### 5.4 金融性现金流的影响

**注意**：
- 理财收益应剔除（不属于经营）
- 利息支出已反映在FCF中
- 股权融资不影响FCF

### 5.5 不同行业的FCF特征

| 行业 | FCF特征 | 关注点 |
|:---|:---|:---|
| **轻资产** | FCF高 | 可持续竞争优势 |
| **重资产** | FCF低 | 资本效率、产能利用率 |
| **高成长** | FCF负 | 增长质量、投资回报 |
| **成熟稳定** | FCF稳定 | 分红能力 |

### 二、自由现金流质量分析

#### 2.1 FCF转化率分析

```python
# FCF转化率 = 企业FCF / NOPAT
# 衡量：每1元经营利润转化为多少自由现金流

FCF转化率 = {
    '2021': 148 / 248 = 59.7%,
    '2022': 140 / 265 = 52.8%,
    '2023': 193 / 296 = 65.2%
}

3年平均FCF转化率 = 481 / 809 = 59.5%

评价标准：
- > 70%：优秀，盈利质量高，资本支出可控
- 50-70%：良好，正常水平
- 30-50%：一般，资本支出较大
- < 30%：较差，可能存在过度投资或盈利质量差

万华评价：59.5%属于良好水平，盈利质量正常
```

#### 2.2 资本支出强度分析

```python
# 资本支出强度 = 资本支出 / 折旧摊销
# 衡量：投资是维持性还是扩张性

资本支出强度 = {
    '2021': 150 / 65 = 2.31,
    '2022': 185 / 75 = 2.47,
    '2023': 180 / 85 = 2.12
}

3年平均 = 515 / 225 = 2.29

解读：
- 强度 = 1.0：仅维持性投资（替换折旧）
- 强度 = 1.5：轻度扩张
- 强度 > 2.0：积极扩张

万华强度2.29，属于积极扩张期
3年累计扩张性投资 = 515 - 225 = 290亿元
```

#### 2.3 现金创造效率分析

| 指标 | 2021 | 2022 | 2023 | 3年平均 | 行业平均 | 评价 |
|:---|---:|---:|---:|---:|---:|:---|
| 营业现金流/收入 | 20.5% | 19.6% | 21.3% | 20.5% | 15% | ✅ 优秀 |
| 企业FCF/收入 | 10.2% | 8.5% | 11.0% | 9.9% | 6% | ✅ 优秀 |
| 企业FCF/NOPAT | 59.7% | 52.8% | 65.2% | 59.5% | 50% | ✅ 良好 |
| CapEx/折旧 | 2.31 | 2.47 | 2.12 | 2.29 | 1.5 | ⚠️ 扩张期 |

**结论**：
- 现金创造效率高于行业平均 ✅
- 处于积极扩张期，资本支出强度高 ⚠️
- FCF质量良好，但需关注投资回报

### 五、FCF对估值的影响

#### 5.1 DCF估值计算

```python
# 两阶段DCF模型

# 预测期（2024-2028）
```
