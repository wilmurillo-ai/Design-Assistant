# WACC计算

## 一、理论背景与核心概念

### 1.1 什么是WACC？

**定义**：
WACC（Weighted Average Cost of Capital，加权平均资本成本）是企业各种资本来源成本的加权平均值，反映企业整体融资成本。

**核心作用**：
1. **DCF估值的折现率**：将未来现金流折现到当前
2. **投资决策基准**：项目回报率需高于WACC
3. **资本结构优化**：寻找最低WACC的资本结构
4. **企业价值评估**：WACC影响企业估值水平

### 1.2 WACC的经济含义

**投资者的视角**：
- 股权投资者要求回报率（Ke）：承担经营风险
- 债权投资者要求回报率（Kd）：承担信用风险
- WACC是两者的加权平均

**企业的视角**：
- WACC是企业的"资金成本"
- 投资项目必须产生高于WACC的回报
- 否则将毁灭股东价值

### 1.3 WACC与投资决策

**决策规则**：
```
如果 项目ROIC > WACC：
    → 接受项目，创造价值
    
如果 项目ROIC < WACC：
    → 拒绝项目，毁灭价值
    
如果 项目ROIC = WACC：
    → 盈亏平衡，无增量价值
```

---

## 二、WACC计算公式

### 2.1 基本公式

```
WACC = (D/V) × Kd × (1 - T) + (E/V) × Ke

其中：
D = 债务资本的市场价值
E = 股权资本的市场价值
V = D + E = 企业总价值
D/V = 债务权重
E/V = 股权权重
Kd = 债务成本率
Ke = 股权成本率
T = 企业所得税率
```

## 三、详细操作步骤

### 步骤1：收集基础数据

**所需数据清单**：
```python
所需数据 = {
    '利润表': ['利息支出', '净利润', '所得税费用', '利润总额'],
    '资产负债表': ['短期借款', '长期借款', '应付债券', '股东权益', '货币资金'],
    '市场数据': ['股价', '总股本', '无风险利率', '市场收益率'],
    '其他': ['Beta系数', '目标资本结构']
}
```

**数据来源**：
- 财务报表：年报、季报
- 市场数据：Wind、Bloomberg、雅虎财经
- 宏观数据：10年期国债收益率

### 步骤2：计算债务成本（Kd）

**方法一：基于历史数据**
```python
Kd = 利息支出 / 平均债务余额

平均债务 = (期初债务 + 期末债务) / 2
```

**方法二：基于市场利率**
```python
Kd = 无风险利率 + 信用利差

信用利差根据信用评级确定：
- AAA级：0.5-1.0%
- AA级：1.0-1.5%
- A级：1.5-2.5%
- BBB级：2.5-4.0%
```

**计算税后债务成本**：
```
税后Kd = Kd × (1 - T)
```

### 步骤3：计算股权成本（Ke）

**使用CAPM模型**：
```python
Ke = Rf + β × (Rm - Rf)

其中：
Rf = 无风险利率（10年期国债收益率）
β = 股票贝塔系数
Rm = 市场预期收益率
(Rm - Rf) = 市场风险溢价
```

**参数获取**：
1. **无风险利率(Rf)**：中国10年期国债收益率（约2.5-3.5%）
2. **市场风险溢价**：历史数据（中国约5-7%）
3. **Beta系数**：
   - 金融终端查询
   - 自行计算（回归法）
   - 使用行业平均

### 步骤4：计算资本结构权重

**计算市场价值**：
```python
# 债务市场价值
债务市场价值 = 短期借款 + 长期借款 + 应付债券
             + 一年内到期的非流动负债

# 股权市场价值
股权市场价值 = 股价 × 总股本

# 企业总价值
V = 债务市场价值 + 股权市场价值

# 权重计算
债务权重 = 债务市场价值 / V
股权权重 = 股权市场价值 / V
```

**注意**：使用市场价值而非账面价值

### 步骤5：计算WACC

**代入公式计算**：
```python
WACC = 债务权重 × 税后Kd + 股权权重 × Ke
```

**验证计算**：
- 检查各组成部分合理性
- 与行业平均对比
- 确认计算无误

### 步骤6：敏感性分析与调整

**关键变量敏感性测试**：
| 变量 | 基准值 | 变动范围 | 对WACC影响 |
|:---|:---|:---|:---|
| Beta | 1.2 | ±0.3 | ±1.8% |
| 风险溢价 | 6% | ±1% | ±0.6% |
| 债务权重 | 40% | ±10% | ±0.3% |

**最终确定**：
- 使用基准情景WACC
- 在估值中考虑WACC区间

### 4.1 基础数据收集

**万华化学2023年基础数据**：

| 数据类别 | 项目 | 数值 | 来源 |
|:---|:---|---:|:---|

| | **债务合计** | **350亿** | 计算 |

| **股权数据** | 股价 | 110元 | 市场数据 |
| | 总股本 | 31.4亿股 | 股本数据 |
| | **股权市值** | **3454亿** | 计算 |

| | **有效税率** | **15%** | 计算 |
| **市场参数** | 无风险利率(Rf) | 3.0% | 10年国债 |
| | Beta系数 | 1.2 | 行业平均 |
| | 市场风险溢价 | 6.0% | 历史数据 |

### 4.2 详细计算过程

**步骤1：计算债务成本(Kd)**
```
税前Kd = 利息支出 / 平均债务
      = 20 / 350
      = 5.71%

税后Kd = 5.71% × (1 - 15%)
      = 5.71% × 0.85
      = 4.85%
```

**步骤2：计算股权成本(Ke)**
```
Ke = Rf + β × (Rm - Rf)
   = 3.0% + 1.2 × 6.0%
   = 3.0% + 7.2%
   = 10.2%
```

**步骤3：计算资本结构权重**
```
债务市场价值(D) = 350亿
股权市场价值(E) = 3454亿
企业总价值(V) = 350 + 3454 = 3804亿

债务权重(D/V) = 350 / 3804 = 9.2%
股权权重(E/V) = 3454 / 3804 = 90.8%
```

**步骤4：计算WACC**
```
WACC = (D/V) × 税后Kd + (E/V) × Ke
     = 9.2% × 4.85% + 90.8% × 10.2%
     = 0.45% + 9.26%
     = 9.71%
```

**修正：使用目标资本结构**

实际计算中，为避免市场波动影响，使用目标资本结构：
```
目标债务权重 = 30%（行业平均）
目标股权权重 = 70%

WACC = 30% × 4.85% + 70% × 10.2%
     = 1.46% + 7.14%
     = 8.6%
```

### 4.4 合理性验证

### 2.2 公式拆解

**债务成本部分**：
```
债务权重 × 税后债务成本
= (D/V) × Kd × (1 - T)

解释：
- 债务成本可税前扣除（税盾效应）
- 实际债务成本 = Kd × (1 - T)
```

**股权成本部分**：
```
股权权重 × 股权成本
= (E/V) × Ke

解释：
- 股权成本不可税前扣除
- 股权投资者承担更高风险，要求更高回报
```

## 五、债务成本详解计算（Kd）

### 3.1 计算方法

**方法一：基于历史利息支出**
```
Kd = 利息支出 / 平均债务余额

其中：
利息支出来自利润表
平均债务余额 = (期初债务 + 期末债务) / 2
```

**方法二：基于市场利率**
```
Kd = 无风险利率 + 信用利差

其中：
无风险利率：国债收益率
信用利差：根据企业信用评级确定
```

### 3.2 税盾效应

**税后债务成本**：
```
税后Kd = Kd × (1 - T)

示例：
Kd = 6%
T = 25%
税后Kd = 6% × (1 - 25%) = 4.5%

税盾价值 = 利息支出 × T
```

### 3.3 案例计算

**万华化学**：
```
利息支出：20亿元
平均债务：400亿元
税率：25%

Kd = 20 / 400 = 5.0%
税后Kd = 5.0% × (1 - 25%) = 3.75%
```

## 六、股权成本详解计算（Ke）

### 4.1 CAPM模型

**资本资产定价模型（CAPM）**：
```
Ke = Rf + β × (Rm - Rf)

其中：
Rf = 无风险利率
β = 股票贝塔系数
Rm = 市场预期回报率
(Rm - Rf) = 市场风险溢价
```

### 4.2 参数确定

**无风险利率（Rf）**：
```
常用选择：
- 10年期国债收益率（中国：约2.5-3.5%）
- 5年期国债收益率（更短期视角）
- 国债收益率曲线对应期限

当前建议：Rf = 3.0%
```

**市场风险溢价（Rm - Rf）**：
```
历史数据法：
- 中国股市长期风险溢价：5-7%
- 美国市场长期风险溢价：4-6%

当前建议：市场风险溢价 = 6.0%
```

**Beta系数（β）**：
```
β的含义：
- β = 1：股票波动与市场相同
- β > 1：股票波动大于市场（进攻型）
- β < 1：股票波动小于市场（防御型）
- β < 0：与市场反向变动（罕见）

估算方法：
1. 回归法：股票收益率 vs 市场收益率
2. 可比公司法：同行业平均Beta
3. 去杠杆再杠杆：调整资本结构影响
```

### 4.3 Beta计算详解

**回归法**：
```python
import numpy as np
from sklearn.linear_model import LinearRegression

def calculate_beta(stock_returns, market_returns):
    """
    通过回归计算Beta
    
    stock_returns: 股票月度收益率序列
    market_returns: 市场月度收益率序列（如沪深300）
    """
    # 线性回归：股票收益 = α + β × 市场收益
    model = LinearRegression()
    model.fit(market_returns.reshape(-1, 1), stock_returns)
    
    beta = model.coef_[0]
    alpha = model.intercept_
    
    return beta, alpha

# 示例
月度股票收益 = np.array([0.02, -0.01, 0.03, ...])  # 24个月数据
月度市场收益 = np.array([0.01, -0.005, 0.02, ...])

beta, alpha = calculate_beta(月度股票收益, 月度市场收益)
print(f"Beta: {beta:.3f}")
print(f"Alpha: {alpha:.4f}")
```

**去杠杆与再杠杆**：
```
当可比公司资本结构不同时，需要调整Beta：

去杠杆公式：
βu = βl / [1 + (1 - T) × (D/E)]

其中：
βu = 无杠杆Beta（资产Beta）
βl = 有杠杆Beta（权益Beta）

再杠杆公式（调整到目标资本结构）：
βl_target = βu × [1 + (1 - T) × (D/E)_target]
```

### 4.4 股权成本案例

**万华化学**：
```
Rf = 3.0%
Beta = 1.2（化工行业平均）
市场风险溢价 = 6.0%

Ke = 3.0% + 1.2 × 6.0%
   = 3.0% + 7.2%
   = 10.2%
```

## 七、资本结构权重

### 5.1 市场价值vs账面价值

**原则**：使用市场价值而非账面价值

**原因**：
- 市场价值反映当前资本成本
- 账面价值是历史成本，已过时
- 投资者按市场价值要求回报

**计算方法**：
```
股权市场价值（E）：
E = 股价 × 流通股数

债务市场价值（D）：
- 上市公司债券：按债券市场价格
- 银行贷款：近似账面价值（或按现金流折现）

权重：
E/V = E / (E + D)
D/V = D / (E + D)
```

### 5.2 案例计算

**万华化学**：
```
股价：110元
总股本：31.4亿股
股权市场价值 = 110 × 31.4 = 3,454亿元

债务账面价值：400亿元
（假设债务市场价值≈账面价值）

总价值 V = 3,454 + 400 = 3,854亿元

股权权重 = 3,454 / 3,854 = 89.6%
债务权重 = 400 / 3,854 = 10.4%
```

**注意**：实际计算中，股权价值应该使用**目标资本结构**或**行业平均资本结构**，而非当前市场价值，以避免市场波动影响。

### 5.3 目标资本结构

**确定方法**：
```
1. 历史平均法：公司5年平均资本结构
2. 行业对标法：同行业可比公司平均
3. 管理层目标：公司 announced 的目标
4. 最优资本结构：理论计算的最小WACC点
```

**万华化学目标结构**：
```
债务/权益 = 40/60
债务权重 = 40%
股权权重 = 60%
```

| 60% | 40% | 6.8% | ⚠️ 风险高 |

**WACC敏感性矩阵**（Beta vs 资本结构）：

| Beta \ D/V | 30% | 40% | 50% |
|:---:|:---:|:---:|:---:|
| 1.0 | 7.2% | 6.9% | 6.6% |
| 1.2 | 7.9% | 7.6% | 7.3% |
| 1.4 | 8.6% | 8.3% | 8.0% |

### 7.3 对估值的影响

**示例**：FCF = 100亿元，g = 3%

| WACC | 企业价值 | 差异 |
|:---:|:---:|:---:|

| 9.0% | 1,667亿 | -28% |

**结论**：WACC对估值影响巨大，需谨慎确定。

## 十、行业WACC对比

### 8.1 不同行业WACC

| 行业 | 典型WACC | 原因 |
|:---|:---:|:---|
| 公用事业 | 5-7% | 低风险、稳定现金流 |
| 消费品 | 6-8% | 品牌护城河、稳定需求 |
| 制造业 | 7-9% | 中等风险、周期性 |
| 科技 | 8-12% | 高风险、高增长 |
| 生物医药 | 9-13% | 研发风险、长周期 |
| 金融 | 不适用 | 特殊估值方法 |

### 8.2 中国市场特点

**利率环境**：
- 中国无风险利率低于美国
- 但市场风险溢价可能更高
- 整体WACC与美国相近

**Beta特点**：
- A股市场波动大，Beta普遍偏高
- 大盘股Beta更接近1
- 小盘股Beta可达1.5-2.0

---

## 十一、常见问题与注意事项

### 9.1 常见问题

**Q1：用账面价值还是市场价值？**
- **A**：理论上用市场价值
- 但为避免波动，可用目标资本结构或行业平均

**Q2：债务成本用历史还是当前？**
- **A**：用当前市场利率更好
- 但历史利息支出可作为参考

**Q3：Beta如何获取？**
- **A**：
  1. 金融终端（Wind、Bloomberg）
  2. 自行计算（需收益率数据）
  3. 使用行业平均Beta

**Q4：WACC多久更新一次？**
- **A**：至少每年更新
- 利率环境变化时需及时更新

### 9.2 注意事项

1. **税率的准确性**：使用有效税率而非名义税率
2. **债务的完整性**：包括所有带息债务（租赁负债等）
3. **Beta的稳定性**：检查Beta是否稳定，异常时需调查
4. **市场风险溢价**：不同市场、不同时期差异大

## 附录：实战案例集

### A.1 高Beta科技股案例

**背景**：某AI芯片公司，高成长、高风险

**参数确定**：
```
债务成本：
- 银行贷款：6%
- 无债券发行
- Kd = 6%

股权成本：
- Rf = 3.0%
- Beta = 2.0（科技股高波动）
- 风险溢价 = 6%
- Ke = 3% + 2.0×6% = 15%

资本结构：
- 债务：100亿（20%）
- 股权：400亿（80%）

WACC计算：
WACC = 20%×6%×(1-25%) + 80%×15%
     = 0.9% + 12%
     = 12.9%
```

**分析**：
- 高Beta导致股权成本极高
- WACC 12.9%处于科技行业正常范围
- 投资项目需产生>13%回报

### A.2 低Beta公用事业案例

**背景**：某电力公司，稳定现金流

**参数确定**：
```
债务成本：
- Kd = 4%（AAA级信用）

股权成本：
- Beta = 0.6（防御型）
- Ke = 3% + 0.6×6% = 6.6%

资本结构：
- 债务：60%
- 股权：40%

WACC = 60%×4%×75% + 40%×6.6%
     = 1.8% + 2.64%
     = 4.44%
```

**分析**：
- 低Beta + 高杠杆 = 极低WACC
- 适合投资回报率要求低的项目
- 但债务风险需关注

### A.3 跨国企业案例

**背景**：某跨国制造企业在华子公司

**特殊考虑**：

1. 债务成本：
   - 境内借款：5%
   - 境外母司借款：4%

2. 股权成本：
   - 需考虑国别风险溢价

## 进阶话题

### B.1 最优资本结构

**理论最优**：
使WACC最小的债务/权益比例

**计算方法**：
```python
def find_optimal_capital_structure(roic, tax_rate, risk_free_rate, beta_unlevered):
    """
    计算理论最优资本结构
    """
    min_wacc = float('inf')
    optimal_debt_ratio = 0
    
    for debt_ratio in [i/100 for i in range(0, 101, 5)]:
        # 随杠杆增加，债务成本上升
        kd = risk_free_rate + 0.02 * (debt_ratio / 0.3)
        
        # 再杠杆Beta
        beta_levered = beta_unlevered * (1 + (1 - tax_rate) * debt_ratio / (1 - debt_ratio))
        ke = risk_free_rate + beta_levered * 0.06
        
        wacc = debt_ratio * kd * (1 - tax_rate) + (1 - debt_ratio) * ke
        
        if wacc < min_wacc:
            min_wacc = wacc
            optimal_debt_ratio = debt_ratio
    
    return optimal_debt_ratio, min_wacc
```

### B.2 不同时期WACC变化

**历史演变**：
```
2015年：WACC = 9%（高利率环境）
2020年：WACC = 7%（疫情后降息）
2024年：WACC = 7.6%（当前）

变化原因：
- 无风险利率下降
- 但市场风险溢价上升
- 综合影响WACC略有下降
```

### B.3 WACC与ROIC的关系

**价值创造公式**：
```
企业价值 = 投入资本 + PV(经济利润)

经济利润 = 投入资本 × (ROIC - WACC)

当ROIC > WACC：创造价值
当ROIC < WACC：毁灭价值
```

## 常见错误与陷阱

### C.1 错误1：使用账面价值权重

**错误做法**：
```
股权账面价值：300亿
债务账面价值：400亿
权重：E=43%, D=57%
```

**问题**：
- 股权账面价值严重低估（尤其是成长股）
- 导致债务权重虚高
- WACC偏低，可能过度投资

**正确做法**：使用市场价值或目标结构

### C.2 错误2：忽视税盾

**错误做法**：
```
WACC = 40%×6% + 60%×10% = 8.4%（未扣税）
```

**正确做法**：
```
WACC = 40%×6%×75% + 60%×10% = 7.8%
```

### C.3 错误3：Beta选择不当

**错误**：使用短期Beta（波动大）
**正确**：使用3-5年Beta，或行业平均

### C.4 错误4：风险溢价过时

**错误**：使用20年前风险溢价4%
**正确**：使用近10年平均，或当前市场隐含

## 总结

WACC是估值和投资决策的核心参数，计算时需注意：

1. **债务成本**：使用当前市场利率，考虑税盾
2. **股权成本**：CAPM参数需谨慎确定，特别是Beta
3. **资本结构**：用目标结构，避免市场波动影响
4. **敏感性**：WACC对估值影响大，需充分测试
5. **合理性**：与行业对比，检查是否在合理范围

掌握WACC计算，是进行专业财务分析和估值的基础能力。
