# 现金周期分析

## 📚 一、现金周期构成

```
现金周期 = 存货周转天数 + 应收周转天数 - 应付周转天数
```

| 周期 | 公式 | 意义 |
|:---|:---|:---|
| 存货周期 | 365/(成本/存货) | 库存管理效率 |
| 应收周期 | 365/(收入/应收) | 回款效率 |
| 应付周期 | 365/(成本/应付) | 占用供应商资金 |

## 📝 二、分析要点

### 同业对比
- 行业平均现金周期
- 领先企业水平

### 趋势分析
- 现金周期变化趋势
- 各分项贡献

### 优化空间
- 缩短存货周期
- 加快应收回款
- 延长应付（不影响关系）
