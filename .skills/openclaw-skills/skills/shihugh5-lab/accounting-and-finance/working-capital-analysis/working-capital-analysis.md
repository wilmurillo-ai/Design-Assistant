# 营运资本分析

## 一、理论背景与核心概念

### 1.1 什么是营运资本？

**传统定义**：
```
营运资本 = 流动资产 - 流动负债
（净营运资本 = 流动资产 - 流动负债）
```

```
周转性经营投入 = 经营性流动资产 - 经营性流动负债

经营性流动资产：
- 应收票据/账款（经营相关）
- 预付款项（经营相关）
- 存货
- 合同资产
- 其他应收款（经营相关）

经营性流动负债：
- 应付票据/账款（经营相关）
- 预收款项/合同负债
- 应付职工薪酬
- 应交税费
- 其他应付款（经营相关）
```

**核心区别**：
- 传统营运资本包含金融资产和金融负债
- 周转性经营投入只包含经营性项目
- 周转性经营投入反映真实的经营资金需求

### 1.2 营运资本的经济含义

**营运资本是"投入"**：
- 企业为维持日常经营必须投入的资金
- 包括：应收款占用的资金、存货占用的资金
- 减：应付款节省的资金

**现金周期**：
```
现金周期 = 存货周转天数 + 应收账款周转天数 - 应付账款周转天数

含义：从支付现金购买原材料到收回销售现金的平均天数
```

**目标**：
- 缩短现金周期，减少营运资本占用
- 提高营运资本周转效率
- 释放资金用于价值创造

## 二、现金周期分析

### 2.1 现金周期公式

```
现金周期（CCC）= DIO + DSO - DPO

其中：
DIO = Days Inventory Outstanding = 存货周转天数
DSO = Days Sales Outstanding = 应收账款周转天数
DPO = Days Payable Outstanding = 应付账款周转天数
```

### 2.2 分项计算

**存货周转天数（DIO）**：
```
存货周转率 = 营业成本 / 平均存货
存货周转天数 = 365 / 存货周转率
```

**应收账款周转天数（DSO）**：
```
应收账款周转率 = 营业收入 / 平均应收账款
应收账款周转天数 = 365 / 应收账款周转率

注意：应收账款应包括应收票据、合同资产
```

**应付账款周转天数（DPO）**：
```
应付账款周转率 = 营业成本 / 平均应付账款
应付账款周转天数 = 365 / 应付账款周转率

注意：应付账款应包括应付票据、合同负债
```

### 2.3 行业对比标准

| 行业 | 现金周期 | DIO | DSO | DPO | 特征 |
|:---|:---:|:---:|:---:|:---:|:---|
| **零售** | -30~0天 | 30-60 | 0-10 | 60-90 | 现金交易，应付账期长 |
| **白酒** | 100-300天 | 300-800 | 10-30 | 30-60 | 存货周转慢（基酒陈化） |
| **家电** | 30-60天 | 30-50 | 20-40 | 40-70 | 快速周转 |
| **医药** | 60-120天 | 60-90 | 60-120 | 60-90 | 应收账款周期长 |
| **化工** | 30-90天 | 30-60 | 20-40 | 30-60 | 标准化产品，周转较快 |
| **房地产** | 300-1000天 | 300-800 | 10-30 | 90-180 | 项目周期长 |

### 2.4 现金周期优化策略

**缩短现金周期的措施**：

**1. 降低DIO（加快存货周转）**：
- JIT（准时制）生产
- 精益库存管理
- 需求预测优化
- 加快物流配送

**2. 降低DSO（加快收款）**：
- 信用政策优化
- 应收账款保理
- 现金折扣激励
- 加强催收管理

**3. 延长DPO（延长付款）**：
- 供应商关系管理
- 付款期限谈判
- 供应链金融
- 不损害信用的前提下延期

**注意**：
- 过度压缩DPO可能损害供应商关系
- 过度放松DSO可能增加坏账风险
- 过度降低DIO可能导致缺货

## 三、应收账款分析

### 3.1 应收账款结构分析

```
应收账款构成
├── 应收票据（银行承兑/商业承兑）
├── 应收账款（账龄结构）
│   ├── 1年以内
│   ├── 1-2年
│   ├── 2-3年
│   └── 3年以上
└── 合同资产（已完工未结算）
```

### 3.2 账龄分析

```python
# 账龄结构分析
def 应收账款账龄分析(账龄结构, 坏账准备):
    """
    账龄结构：{"1年以内": 金额, "1-2年": 金额, ...}
    """
    总应收账款 = sum(账龄结构.values())
    
    分析结果 = {}
    for 账期, 金额 in 账龄结构.items():
        占比 = 金额 / 总应收账款 * 100
        分析结果[账期] = {
            "金额": 金额,
            "占比": 占比
        }
    
    # 风险判断
    逾期占比 = 账龄结构.get("1-2年", 0) + 账龄结构.get("2-3年", 0) + 账龄结构.get("3年以上", 0)
    逾期比例 = 逾期占比 / 总应收账款
    
    if 逾期比例 < 0.05:
        风险等级 = "🟢 低风险"
    elif 逾期比例 < 0.15:
        风险等级 = "🟡 中低风险"
    elif 逾期比例 < 0.30:
        风险等级 = "🟠 中风险"
    else:
        风险等级 = "🔴 高风险"
    
    分析结果["逾期比例"] = 逾期比例
    分析结果["风险等级"] = 风险等级
    
    return 分析结果
```

### 3.3 坏账准备充足性

```
坏账准备充足性 = 坏账准备余额 / 逾期应收账款 × 100%

评价标准：
- > 100%：准备充足 ✅
- 50-100%：准备基本充足 ⚠️
- < 50%：准备不足，存在风险 ❌
```

### 3.4 应收账款质量指标

| 指标 | 计算公式 | 评价标准 |
|:---|:---|:---|
| **DSO** | 平均应收账款/日均收入 | 越低越好，对比行业 |
| **逾期率** | 逾期应收/总应收 | <5% ✅，>15% ❌ |
| **坏账率** | 坏账损失/营业收入 | <1% ✅，>3% ❌ |
| **周转率** | 营业收入/平均应收 | 越高越好 |

## 四、存货分析

### 4.1 存货结构分析

```
存货构成
├── 原材料
├── 在产品（半成品）
├── 产成品
└── 周转材料

分析要点：
1. 各类型占比是否合理
2. 是否存在呆滞库存
3. 存货跌价准备是否充足
```

### 4.2 存货周转分析

```python
# 存货周转指标
def 存货周转分析(营业成本, 平均存货, 存货结构):
    """
    分析存货周转效率
    """
    存货周转率 = 营业成本 / 平均存货
    存货周转天数 = 365 / 存货周转率
    
    # 分类周转（如有数据）
    分类周转 = {}
    if 存货结构:
        for 类别, 金额 in 存货结构.items():
            # 假设各类别周转与整体相同（简化）
            分类周转[类别] = {
                "金额": 金额,
                "占比": 金额 / sum(存货结构.values())
            }
    
    return {
        "存货周转率": 存货周转率,
        "存货周转天数": 存货周转天数,
        "分类结构": 分类周转
    }
```

### 4.3 存货风险识别

**风险信号**：
1. **存货大幅增长**：销售不畅或过度生产
2. **周转天数增加**：管理效率下降
3. **产成品占比过高**：可能存在滞销
4. **跌价准备不足**：存货减值风险
5. **库龄结构恶化**：长库龄存货增加

### 4.4 存货管理优化

**策略**：
```
1. 需求驱动生产（Demand-Driven）
   - 基于订单生产
   - 缩短生产周期

2. 供应链协同
   - VMI（供应商管理库存）
   - JIT配送

3. 库存分类管理
   - ABC分类：重点管理A类（高价值）
   - XYZ分类：重点管理X类（需求稳定）
```


### 5.2 付款政策分析

```python
def 付款政策分析(应付账款, 采购成本, dpo):
    """
    分析付款政策合理性
    """
    # 理论DPO（基于合同付款期）
    合同付款期 = 30  # 假设平均30天
    
    if dpo > 合同付款期 + 15:
        评价 = "⚠️ 付款期显著延长，需关注供应商关系"
    elif dpo > 合同付款期:
        评价 = "✅ 适度利用供应商融资"
    elif dpo < 合同付款期 - 10:
        评价 = "⚠️ 付款过于积极，未充分利用信用期"
    else:
        评价 = "✅ 付款政策合理"
    
    return {
        "实际DPO": dpo,
        "合同付款期": 合同付款期,
        "评价": 评价
    }
```

## 常见问题与注意事项

### Q1：数据口径不一致怎么办？

**A**：优先使用年报数据，确保口径一致。如有差异，在分析中说明。

### Q2：如何处理异常值？

**A**：识别异常原因，使用多年平均或中位数平滑。

### Q3：定性分析与定量分析如何结合？

**A**：定量为主，定性为辅。定量发现问题，定性解释原因。

### 注意事项

1. **数据质量优先**：确保基础数据准确
2. **多年度对比**：避免单年度偶然因素
3. **业务实质**：透过数字看业务本质
4. **动态分析**：关注趋势而非时点

### 二、现金周期计算

#### 2.1 分项周转天数

```python
# 基础数据
营业收入 = 1753亿元
营业成本 = 1315亿元

# 存货周转天数（DIO）
平均存货 = 185亿元
存货周转率 = 1315 / 185 = 7.11次
DIO = 365 / 7.11 = 51.3天

# 应收账款周转天数（DSO）
平均应收账款 = (28.5 + 62.0) / 2 = 45.25亿元  # 含应收票据
DSO = 365 / (1753 / 45.25) = 365 / 38.74 = 9.4天

# 应付账款周转天数（DPO）
平均应付账款 = (35.0 + 125.0) / 2 = 80亿元  # 含应付票据
DPO = 365 / (1315 / 80) = 365 / 16.44 = 22.2天
```

**注意**：万华化学采用预收款模式，大量客户预付款导致应收账款极低

#### 2.2 现金周期

```
现金周期 = DIO + DSO - DPO
        = 51.3 + 9.4 - 22.2
        = 38.5天
```

| DSO（应收） | 9.4天 | 35天 | ✅ 极其优秀（预收款模式）|

- 1年以内应收占比93.5%，账龄结构健康 ✅

#### 3.2 低DSO的原因分析

```
万华化学DSO仅9.4天的原因：

1. **产品稀缺性**
   - MDI全球仅7家生产商
   - 供不应求，议价能力强

2. **预收款模式**
   - 合同负债45亿（预收款）
   - 部分客户需提前付款

3. **客户结构**
```
