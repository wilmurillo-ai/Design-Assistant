# DCF固定增长模型（Gordon Growth Model）

## 一、理论背景与核心概念

### 1.1 模型的起源与发展

**Gordon Growth Model历史**：
> DCF固定增长模型由经济学家Myron J. Gordon和Eli Shapiro于1956年提出，又称Gordon-Shapiro模型。这是最简单的现金流折现模型，为后续多阶段DCF模型奠定了理论基础。

**模型的经济学基础**：
- **永续经营假设**：企业将持续经营 indefinitely
- **稳定增长假设**：现金流以固定速率永续增长
- **市场均衡假设**：增长率 eventually 收敛到经济整体增速
- **风险补偿原理**：折现率反映资金时间价值和风险溢价

### 1.2 为什么需要固定增长模型？

**企业生命周期视角**：
```
初创期 → 成长期 → 成熟期 → 衰退期
  ↓        ↓        ↓        ↓
不适用   两阶段   固定增长   不适用
        三阶段   模型适用
```

**成熟企业的特征**：
| 特征 | 成长期企业 | 成熟企业 |
|:---|:---|:---|
| 收入增长率 | >GDP增速 | ≈GDP增速 |
| 市场份额 | 快速扩张 | 稳定 |
| 竞争格局 | 激烈变化 | 格局已定 |
| 资本支出 | 扩张性为主 | 维持性为主 |
| 盈利能力 | 波动大 | 稳定 |
| 适用模型 | 两阶段/三阶段 | 固定增长模型 |

**模型的核心假设**：
1. **永续经营**：企业将无限期持续经营
2. **固定增长**：FCF以固定增长率g永续增长
3. **增长约束**：g < WACC（增长必须小于资本成本）
4. **稳定状态**：企业处于稳定状态，无重大变化


> "对于成熟期的企业，可以使用单阶段DCF模型，即固定增长模型。永续增长率一般取2-3%，不应超过GDP增速。"

1. 成熟企业的FCF预测相对容易（稳定性高）
2. 永续增长率应保守估计（2-3%）
3. 资本支出与折旧摊销应大致相等
4. 营运资本变动应较小

### 1.4 模型的适用边界

**适用条件检查清单**：
- [ ] 企业是否处于成熟期？（收入增长率≈GDP增速）
- [ ] 行业竞争格局是否稳定？（市场份额变化小）
- [ ] 资本支出是否稳定？（维持性为主）
- [ ] 盈利能力是否稳定？（ROE波动小）
- [ ] 是否有长期竞争优势？（可持续获取平均回报）

**不适用情况**：
| 情况 | 原因 | 推荐模型 |
|:---|:---|:---|
| 收入增长率>10% | 增长过快，不适用于固定增长 | 两阶段DCF |
| 周期性行业 | 盈利波动大，无常数g | 正常化盈利+两阶段 |
| 转型期企业 | 业务模式变化大 | 两阶段/三阶段 |
| 衰退期企业 | 现金流持续下降 | 清算价值法 |
| 初创企业 | 盈利不稳定，可能为负 | VC法/收入倍数 |

### 步骤1：判断模型适用性

**1.1 定量判断标准**：
```python
def 判断固定增长模型适用性(财务指标):
    """
    判断企业是否适合使用固定增长模型
    """
    检查项 = {
        '收入增长率': {
            '条件': 财务指标['收入增长率'] < 0.10,  # <10%
            '权重': 0.25
        },
        '增长率稳定性': {
            '条件': 财务指标['增长率标准差'] < 0.05,  # 波动小
            '权重': 0.20
        },
        '市场份额稳定性': {
            '条件': 财务指标['市场份额变化'] < 0.02,  # <2%
            '权重': 0.20
        },
        '资本支出稳定性': {
            '条件': 财务指标['资本支出波动'] < 0.15,
            '权重': 0.15
        },
        'ROE稳定性': {
            '条件': 财务指标['ROE标准差'] < 0.03,
            '权重': 0.20
        }
    }
    
    得分 = 0
    for 指标, 配置 in 检查项.items():
        if 配置['条件']:
            得分 += 配置['权重']
    
    if 得分 >= 0.7:
        return '高度适用', 得分
    elif 得分 >= 0.5:
        return '基本适用', 得分
    else:
        return '不适用', 得分
```

### 步骤2：确定当前自由现金流（FCF₀）

**2.1 FCF计算方法**：

**方法一：从经营现金流出发**（推荐）
```
FCF = 经营活动现金流净额 - 保全性资本支出

其中：
保全性资本支出 = 维持现有产能所需的资本支出
              ≈ 折旧摊销（简化处理）
```

**方法二：从NOPAT出发**
```
FCF = NOPAT + 折旧摊销 - 资本支出 - 营运资本增加

其中：
NOPAT = EBIT × (1 - 税率)
```

**方法三：简化公式**（成熟期企业）
```
FCF ≈ 净利润 + 折旧摊销 - 资本支出
    ≈ 净利润（当资本支出≈折旧时）
```

**2.2 万华化学FCF计算**：

| 项目 | 金额（亿元） | 说明 |
|:---|---:|:---|
| 净利润 | 296 | 2023年 |
| 加：折旧摊销 | 150 | 非现金支出 |
| 减：资本支出 | 180 | 维持+扩张 |
| 减：营运资本增加 | 30 | 业务增长 |
| **FCF** | **236** | 自由现金流 |

**2.3 FCF调整注意事项**：

**非经常性项目调整**：
```python
def 调整非经常性项目(报表FCF, 非经常性项目):
    """
    剔除一次性因素，得到可持续FCF
    """
    调整项 = {
        '加回': ['资产处置损失', '一次性费用', '异常减值'],
        '扣除': ['资产处置收益', '一次性收入', '保险赔偿']
    }
    
    可持续FCF = 报表FCF
    for 项目 in 调整项['加回']:
        可持续FCF += 非经常性项目.get(项目, 0)
    for 项目 in 调整项['扣除']:
        可持续FCF -= 非经常性项目.get(项目, 0)
    
    return 可持续FCF
```

**周期调整**（周期性行业）：
```python
def 周期调整FCF(历史FCF, 当前FCF, 周期位置):
    """
    对周期性企业的FCF进行正常化处理
    """
    if 周期位置 == '高点':
        # 使用历史平均而非当前高点
        正常化FCF = np.mean(历史FCF)
    elif 周期位置 == '低点':
        # 使用历史平均而非当前低点
        正常化FCF = np.mean(历史FCF)
    else:
        正常化FCF = 当前FCF
    
    return 正常化FCF
```

### 步骤3：确定永续增长率（g）

**3.1 g的确定原则**：

**硬性约束**：
```
1. g < WACC（必须严格小于，否则模型失效）
2. g ≤ 长期GDP增速（企业不能永远快于经济）
3. g ≤ 行业长期增速（行业约束）
4. g > 0（模型假设增长，虽可为负但少见）
```

**软性参考**：
```
1. 通胀水平：g通常略高于长期通胀
2. 人口增长：与目标市场人口增长相关
3. 行业生命周期：成熟期行业增速低
4. 竞争格局：垄断行业可略高于竞争行业
```

**3.2 g的确定方法**：

**方法1：GDP锚定法**（最常用）
```python
def GDP锚定法(国家GDP增速目标, 企业溢价=0.005):
    """
    以GDP增速为锚，给予小幅溢价
    """
    g = 国家GDP增速目标 + 企业溢价
    
    # 约束检查
    if g > 0.04:
        print("警告：永续增长率超过4%，可能过高")
    
    return g

# 中国示例
g_中国 = GDP锚定法(0.025, 0.005)  # 2.5% + 0.5% = 3.0%
# 美国示例
g_美国 = GDP锚定法(0.020, 0.005)  # 2.0% + 0.5% = 2.5%
```

**方法2：行业长期增速法**
```python
def 行业增速法(行业历史收入, 预测年数=20):
    """
    基于行业历史增速外推
    """
    行业CAGR = (行业历史收入[-1] / 行业历史收入[0]) ** (1/len(行业历史收入)) - 1
    
    # 成熟行业增速通常低于历史平均
    g = 行业CAGR * 0.8  # 保守估计
    
    return max(g, 0.01)  # 最低1%
```

**方法3：名义GDP法**
```python
def 名义GDP法(实际GDP增速, 通胀目标):
    """
    永续增长率 ≈ 名义GDP增速
    """
    名义GDP增速 = (1 + 实际GDP增速) * (1 + 通胀目标) - 1
    
    # 企业增速略低于名义GDP（保守）
    g = 名义GDP增速 - 0.005
    
    return g

# 中国示例
g = 名义GDP法(0.025, 0.020)  # ≈ 4.5% - 0.5% = 4.0%（可能偏高，建议3%）
```

### 步骤4：确定WACC

**4.1 WACC计算方法**：

**标准公式**：
```
WACC = E/V × Re + D/V × Rd × (1-T)

其中：
E/V = 股权价值/企业总价值（目标资本结构）
D/V = 债务价值/企业总价值（目标资本结构）
Re = 股权成本（CAPM计算）
Rd = 债务成本（平均利率）
T = 有效税率
```

**4.2 各参数确定**：

**股权成本（Re）- CAPM**：
```
Re = Rf + β × (Rm - Rf) + 国家风险溢价

其中：
Rf = 无风险利率（10年期国债收益率）
β = Beta系数（股票系统性风险）
Rm - Rf = 市场风险溢价（通常5-6%）
```

| 参数 | 万华化学 | 来源 |
|:---|:---:|:---|
| Rf | 2.5% | 中国10年期国债 |
| β | 1.1 | 回归计算 |
| 市场风险溢价 | 6.0% | 历史平均 |
| **Re** | **9.1%** | CAPM计算 |

**债务成本（Rd）**：
```
Rd = 平均借款利率
    = Σ(各笔债务 × 利率) / 总债务
```

| 债务类型 | 金额 | 利率 | 权重 | 加权利率 |
|:---|---:|---:|---:|---:|
| 短期借款 | 100亿 | 4.0% | 20% | 0.8% |
| 长期借款 | 300亿 | 4.5% | 60% | 2.7% |
| 债券 | 100亿 | 3.8% | 20% | 0.8% |
| **合计** | 500亿 | - | 100% | **4.3%** |

**资本结构**：
```
目标资本结构：
- 股权比例（E/V）：60%
- 债务比例（D/V）：40%
```

**4.3 WACC计算**：
```
WACC = 60% × 9.1% + 40% × 4.3% × (1-15%)
     = 5.46% + 1.46%
     = 6.92% ≈ 7.0%
```

**4.4 WACC合理性检查**：

| 检查项 | 万华化学 | 合理性 |
|:---|:---:|:---:|
| WACC > Rf | 7.0% > 2.5% | ✅ 风险补偿充足 |
| WACC < 15% | 7.0% < 15% | ✅ 不过高 |
| Re > Rd | 9.1% > 4.3% | ✅ 股权风险溢价合理 |
| 与同业对比 | 7.0% vs 6.5-7.5% | ✅ 合理区间 |

### 步骤5：计算企业价值

**5.1 核心公式应用**：

**公式回顾**：
```
企业价值 = FCF₁ / (WACC - g)
        = FCF₀ × (1 + g) / (WACC - g)
```

**万华化学计算**：
```python
# 参数
FCF_0 = 236  # 亿元（2023年FCF）
g = 0.03     # 3% 永续增长率
WACC = 0.07  # 7% 加权平均资本成本

# 计算
FCF_1 = FCF_0 * (1 + g)  # = 236 × 1.03 = 243亿元

企业价值 = FCF_1 / (WACC - g)
         = 243 / (0.07 - 0.03)
         = 243 / 0.04
         = 6,075亿元
```

**5.2 结果验证**：

**合理性检查**：
```python
def 估值合理性检查(企业价值, 财务指标):
    """
    检查估值结果是否合理
    """
    检查项 = {
        'EV/Revenue': {
            '计算': 企业价值 / 财务指标['收入'],
            '合理区间': (1.0, 5.0),
            '说明': '企业价值/收入倍数'
        },
        'EV/EBITDA': {
            '计算': 企业价值 / 财务指标['EBITDA'],
            '合理区间': (5, 15),
            '说明': '企业价值/EBITDA倍数'
        },
        'Implied PE': {
            '计算': (企业价值 - 财务指标['净债务']) / 财务指标['净利润'],
            '合理区间': (8, 25),
            '说明': '隐含市盈率'
        }
    }
    
    检查结果 = {}
    for 指标, 配置 in 检查项.items():
        计算值 = 配置['计算']
        下限, 上限 = 配置['合理区间']
        是否合理 = 下限 <= 计算值 <= 上限
        检查结果[指标] = {
            '计算值': 计算值,
            '是否合理': 是否合理
        }
    
    return 检查结果

# 万华化学检查

    '收入': 1753,
    'EBITDA': 446,
    '净利润': 296,
    '净债务': 240
}

检查结果 = 估值合理性检查(6075, 万华财务)
# 结果：EV/Revenue=3.5✅, EV/EBITDA=13.6✅, Implied PE=19.7✅
```

**6.1 股权价值计算**：
```
股权价值 = 企业价值 - 净债务

其中：
净债务 = 有息负债 - 现金及等价物
```

```
每股价值 = 股权价值 / 总股本
```

```python
股权价值 = 5835  # 亿元
```

## 三、公式与计算逻辑详解

### 3.1 Gordon增长模型公式推导

**基本公式**：
```
企业价值 = FCF₁ / (WACC - g)
```

**推导过程**：

**第一步：永续现金流现值公式**
企业价值是未来所有FCF的现值之和：
```
V = FCF₁/(1+r) + FCF₂/(1+r)² + FCF₃/(1+r)³ + ...
```

**第二步：应用固定增长假设**
由于FCF以固定增长率g增长：
```
FCF₂ = FCF₁ × (1+g)
FCF₃ = FCF₂ × (1+g) = FCF₁ × (1+g)²
FCF_n = FCF₁ × (1+g)^(n-1)
```

**第三步：代入现值公式**
```
V = FCF₁/(1+r) + FCF₁(1+g)/(1+r)² + FCF₁(1+g)²/(1+r)³ + ...

令 k = (1+g)/(1+r)，则：
V = FCF₁/(1+r) × [1 + k + k² + k³ + ...]
```

**第四步：无穷等比数列求和**
当 g < r（即 k < 1）时，无穷等比数列收敛：
```
1 + k + k² + k³ + ... = 1/(1-k) = (1+r)/(r-g)
```

**第五步：最终公式**
```
V = FCF₁/(1+r) × (1+r)/(r-g)
  = FCF₁ / (r-g)

即：企业价值 = FCF₁ / (WACC - g)
```

**约束条件**：
- g < WACC（必须严格小于，否则级数不收敛）
- g > -100%（增长率不能低于-100%，即不能为负无穷）

### 3.2 零增长特殊情况（g=0）

**当g=0时**：
```
企业价值 = FCF / WACC

这被称为"零增长模型"或"永续年金模型"
```

**适用场景**：
- 企业已进入完全稳定状态
- 无增长、无收缩
- 常见于公用事业评估

**万华化学零增长估值**：
```
如果假设g=0：
企业价值 = 236 / 7% = 3,371亿元
每股价值 = (3371-240)/31.4 = 100元

比g=3%时的186元低46%
说明永续增长假设对估值影响巨大
```

### 3.3 负增长的特殊情况（g<0）

**当g<0时**（衰退期企业）：
```
企业价值 = FCF₁ / (WACC - g)

由于g为负，(WACC - g) > WACC
→ 分母变大，估值降低
```

**示例**：
```
假设g = -2%（每年衰退2%）
企业价值 = 236 × 0.98 / (7% - (-2%))
         = 231 / 9%
         = 2,567亿元（比g=3%时低58%）
```

**适用性**：
- 衰退期企业理论上可用
- 但通常推荐使用清算价值法
- 或有限期DCF（非永续）

### 3.4 隐含增长率计算

**从市场价格反推隐含增长率**：
```
已知：
- 当前股价
- 当前FCF
- WACC

求解：市场隐含的g是多少？
```

**公式变形**：
```
股价 = (FCF₁ / (WACC - g) - 净债务) / 总股本

解得：
g = WACC - FCF₁ / (股价 × 总股本 + 净债务)
```

**万华化学示例**：
```
当前股价 = 110元
总股本 = 31.4亿股
净债务 = 240亿元
FCF₁ = 243亿元
WACC = 7%

股权价值 = 110 × 31.4 = 3,454亿元
企业价值 = 3454 + 240 = 3,694亿元

隐含g = 7% - 243/3694
      = 7% - 6.6%
      = 0.4%

结论：市场隐含永续增长率仅0.4%，
远低于我们假设的3%，存在预期差
```

### 3.5 调整现值法（APV）对比

**APV方法**：
```
APV = 无杠杆企业价值 + 税盾现值

其中：
无杠杆企业价值 = FCF / (Ru - g)
税盾现值 = D × T（简化）
```

**WACC法 vs APV法**：

| 对比项 | WACC法 | APV法 |
|:---|:---|:---|
| **适用性** | 稳定资本结构 | 资本结构变化大 |
| **复杂性** | 简单 | 复杂 |
| **精确度** | 足够 | 更精确 |
| **常用性** | 更常用 | 较少用 |

**万华化学两种方法对比**：
```
WACC法：
- WACC = 7.0%
- 企业价值 = 6,075亿元

APV法：
- Ru（无杠杆股权成本）= 8.0%
- 无杠杆价值 = 243 / (8%-3%) = 4,860亿元
- 税盾价值 = 500 × 15% = 75亿元
- APV = 4,860 + 75 = 4,935亿元

差异：WACC法比APV法高23%
原因：WACC法隐含的税盾价值更高
```

## 五、常见问题与注意事项

### 5.1 固定增长模型 vs 两阶段模型

**Q：什么时候用固定增长模型，什么时候用两阶段模型？**

**A**：

| 判断标准 | 固定增长模型 | 两阶段模型 |
|:---|:---:|:---:|
| 收入增长率 | <8% | >10% |
| 增长阶段 | 成熟期 | 成长期 |
| 竞争优势 | 稳定 | 快速建立 |
| 资本支出 | 维持性为主 | 扩张性为主 |
| 行业特征 | 公用事业、成熟消费 | 科技、新能源 |

**万华化学判断**：
- MDI业务：收入增速6%，用固定增长模型 ✅
- 新材料业务：收入增速80%，用两阶段模型 ✅
- 整体公司：使用SOTP分部估值

### 5.2 永续增长率的设定误区

**Q：永续增长率设定过高会有什么后果？**

**A**：

**误区1：g >= WACC**
```
后果：
- 分母为0或负数
- 估值趋于无穷大或为负
- 模型失效

正确做法：
g必须严格小于WACC（至少差1-2%）
```

**误区2：g > GDP增速**
```
假设：g=5%，GDP增速=3%
后果：
- 企业终将超过整个经济体
- 逻辑上不可能

正确做法：
g ≤ GDP长期增速
```

**误区3：g = 当前增长率**
```
假设：当前增长10%，设g=10%
后果：
- 假设企业永远高速增长
- 与生命周期理论矛盾

正确做法：
g应反映长期稳态，非当前增速
```

### 5.3 FCF的确定难点

**Q：如何确定"可持续"的FCF？**

**A**：

**调整项清单**：
```
FCF调整 = 报表FCF
        + 非经常性损失
        - 非经常性收益
        + 异常减值
        - 异常处置收益
        ± 周期调整（如需要）
```

**万华化学示例**：
```
报表FCF = 236亿元
调整项：
+ 资产处置损失 15亿
- 政府补助 8亿
+ 存货减值 5亿
= 调整后FCF 248亿元

为保守起见，使用236亿元
```

### 5.4 周期性行业的特殊处理

**Q：化工、钢铁等周期性行业能用固定增长模型吗？**

**A**：

**原则上可以，但需调整**：

**方法1：正常化FCF**
```
正常化FCF = 历史平均FCF（覆盖完整周期）

万华化学：
5年平均FCF = (85+200+320+280+236)/5 = 224亿元
vs 当前FCF = 236亿元
使用224亿元（更保守）
```

**方法2：情景加权**
```
构建多情景：
- 周期高点：FCF=300亿，g=4%
- 周期中点：FCF=220亿，g=3%
- 周期低点：FCF=150亿，g=2%

概率加权估值：
= 高点估值×20% + 中点估值×60% + 低点估值×20%
```

**方法3：使用两阶段模型**
```
第一阶段：预测完整周期（5-7年）
第二阶段：永续增长
```

### 5.5 负增长率的情况

**Q：衰退期企业（负增长）能用固定增长模型吗？**

**A**：

**技术上可以，但不推荐**：
```
假设g = -2%
企业价值 = FCF₁ / (WACC - (-2%))
         = FCF₁ / (WACC + 2%)

估值会降低，但仍有价值
```

**更好的方法**：
- **清算价值法**：资产变现价值
- **有限期DCF**：假设企业有有限寿命
- **相对估值法**：PB、PS等

### 5.6 多业务公司的估值

**Q：多元化公司如何用固定增长模型？**

**A**：

**分部估值法（SOTP）**：
```
企业价值 = Σ 各业务价值

各业务：
- 成熟业务：固定增长模型
- 成长业务：两阶段模型
- 初创业务：收入倍数法

万华化学示例：
- MDI业务（成熟）：固定增长模型 → 3000亿
- 新材料业务（成长）：两阶段模型 → 1500亿
- 精细化工（成熟）：固定增长模型 → 1000亿
- 合计：5500亿元
```

### 5.7 注意事项

**建模注意事项**：
1. **保守性原则**：关键假设应保守估计
2. **一致性原则**：假设要与业务战略一致
3. **可比性原则**：参数要与行业对比
4. **敏感性原则**：必须做敏感性分析

**常见错误**：
- ❌ g >= WACC（模型失效）
- ❌ g > GDP增速（逻辑错误）
- ❌ 使用未调整的报表FCF（不准确）
- ❌ 忽略周期性（周期高点估值偏高）
- ❌ 不做敏感性分析（风险不明）
