# CAPM股权资本成本计算

## 📚 一、理论背景与核心概念

### 1.1 什么是股权资本成本

**定义**：
> 股权资本成本是股东投资于企业所要求的最低回报率，也是企业使用股东资金的机会成本。

**经济含义**：
- 股东放弃其他投资机会的代价
- 企业必须为股东创造的最低回报
- 投资决策的最低门槛收益率

**公式表达**：
```
股权资本成本(Ke) = 无风险利率 + β × 市场风险溢价
                 = Rf + β × (Rm - Rf)
```

### 1.2 CAPM的理论基础

**资本资产定价模型**（Capital Asset Pricing Model）：

**核心假设**：
1. 投资者风险厌恶，追求效用最大化
2. 市场完全竞争，无摩擦
3. 所有投资者对预期一致
4. 可以无风险利率自由借贷

**风险分解**：
```
总风险 = 系统性风险 + 非系统性风险

CAPM认为：
- 非系统性风险可通过分散化消除
- 只有系统性风险需要补偿
- β衡量系统性风险
```

### 1.3 三个参数的含义

| 参数 | 符号 | 含义 | 影响 |
|:---|:---:|:---|:---|
| **无风险利率** | Rf | 无任何风险的投资收益率 | 时间价值基准 |
| **β系数** | β | 股票相对于市场的波动程度 | 风险衡量 |
| **市场风险溢价** | Rm-Rf | 市场超额回报 | 风险补偿 |

### 1.4 CAPM的适用与局限

**适用场景**：
- 上市公司估值
- 资本预算决策
- 业绩评价基准

**局限性**：
- 假设过于理想化
- β不稳定，随时间变化
- 难以解释小规模效应、价值效应等异常

**替代模型**：
- Fama-French三因子模型
- 套利定价理论(APT)
- 股息增长模型(DDM)

## 📝 二、详细操作步骤

### 步骤1：确定无风险利率(Rf)

**选择标准**：
- 长期国债收益率（10年期）
- 与现金流预测期限匹配
- 使用当前市场数据

**数据来源**：
- 中国：中债10年期国债收益率
- 美国：美国10年期国债收益率
- Yahoo Finance、Wind等金融终端

**示例**：
```
2024年1月：
- 中国10年期国债收益率：2.5%
- 美国10年期国债收益率：4.0%
```

### 步骤2：确定β系数

**β的定义**：
```
β = Cov(Ri, Rm) / Var(Rm)

其中：
Ri = 股票收益率
Rm = 市场收益率
```

**β的含义**：
| β值 | 含义 | 风险特征 |
|:---:|:---|:---|
| β = 1 | 与市场同步波动 | 平均风险 |
| β > 1 | 比市场波动更大 | 高风险高弹性 |
| β < 1 | 比市场波动小 | 低风险防御型 |
| β < 0 | 与市场反向波动 | 罕见 |

**获取方法**：

**方法1：使用已计算β**
- Yahoo Finance
- Bloomberg
- Wind、同花顺等

**方法2：自行计算**
```
1. 收集股票日收益率（1-3年）
2. 收集市场指数日收益率（同期）
3. 计算协方差和方差
4. β = Cov(股票,市场) / Var(市场)
```

**方法3：可比公司β**
- 选择3-5家可比上市公司
- 取平均β
- 适用于非上市公司

### 步骤3：确定市场风险溢价

**定义**：
```
市场风险溢价 = 市场期望收益率 - 无风险利率
             = Rm - Rf
```

**历史数据法**：
```
使用历史长期数据（20-30年）：
- 股票市场平均收益率
- 减去同期无风险利率
- 得到历史风险溢价
```

**中国市场参考值**：5%-7%
**美国市场参考值**：4%-6%

**当前市场法**：
- 使用隐含风险溢价
- 基于当前市场估值反推

### 步骤4：计算股权资本成本

**公式**：
```
Ke = Rf + β × (Rm - Rf)
```

**示例**：
```
Rf = 2.5%
β = 1.2
Rm - Rf = 6%

Ke = 2.5% + 1.2 × 6%
   = 2.5% + 7.2%
   = 9.7%
```

### 步骤5：结果验证与调整

**合理性检验**：
| Ke区间 | 评价 | 适用企业 |
|:---:|:---|:---|
| 7%-9% | 低风险 | 公用事业、消费龙头 |
| 9%-12% | 中等风险 | 一般制造业 |
| 12%-15% | 高风险 | 科技、成长股 |
| >15% | 极高风险 | 初创、小盘股 |

**调整因素**：
- 国家风险溢价（新兴市场）
- 规模溢价（小公司）
- 特定风险（单一业务、高负债）

## 🔢 三、公式与计算逻辑

### 3.1 CAPM公式体系

**基础公式**：
```
Ke = Rf + β × ERP

其中：
Ke = 股权资本成本
Rf = 无风险利率
β = 系统性风险系数
ERP = 市场风险溢价 = Rm - Rf
```

**β的计算**：
```
β = ρ(i,m) × σi / σm

其中：
ρ(i,m) = 股票与市场的相关系数
σi = 股票收益率标准差
σm = 市场收益率标准差
```

### 3.2 参数估计方法

**无风险利率**：
```
推荐使用：10年期国债到期收益率

中国：中债国债收益率曲线
美国：Treasury Yield Curve

注意：使用名义利率（含通胀预期）
```

**β的调整**：
```
原始β（含杠杆）→ 去杠杆 → 行业平均β → 加杠杆

去杠杆公式：
βu = βl / [1 + (1-T) × D/E]

加杠杆公式：
βl = βu × [1 + (1-T) × D/E]
```

**市场风险溢价**：
```
几何平均法（推荐）：
ERP = [(1+Rm)/(1+Rf)]^(1/n) - 1

算术平均法：
ERP = Average(Rm - Rf)
```

### 3.3 调整的CAPM

**国家风险溢价**：
```
Ke = Rf + β × ERP + CRP

CRP = 国家风险溢价（新兴市场）
```

**规模溢价**：
```
Ke = Rf + β × ERP + Size Premium

小公司通常有额外2%-4%溢价
```

## 💡 四、实例演示：万华化学

### 4.1 参数确定

**无风险利率(Rf)**：
```
2024年中国10年期国债收益率：2.5%
```

**β系数**：
```
方法1：Yahoo Finance查询
万华化学β ≈ 1.15

方法2：自行计算（使用2年数据）
计算得β = 1.18

取平均值：β = 1.17
```

**市场风险溢价(ERP)**：
```
中国市场历史风险溢价：约6%
当前隐含风险溢价：约5.5%

取保守值：ERP = 6%
```

### 4.2 CAPM计算

```
Ke = Rf + β × ERP
   = 2.5% + 1.17 × 6%
   = 2.5% + 7.02%
   = 9.52%

取整：Ke ≈ 9.5% 或 10%
```

### 4.3 同业对比

| 公司 | β | Rf | ERP | Ke | 评价 |
|:---|:---:|:---:|:---:|:---:|:---|
| 万华化学 | 1.17 | 2.5% | 6% | 9.5% | 中等风险 |
|  competitor A | 1.05 | 2.5% | 6% | 8.8% | 低风险 |
|  competitor B | 1.35 | 2.5% | 6% | 10.6% | 高风险 |

### 4.4 敏感性分析

**β敏感性**：
| β | Ke | 变化 |
|:---:|:---:|:---:|

| 1.00 | 8.5% | -1.0% |
| **1.17** | **9.5%** | 基准 |
| 1.30 | 10.3% | +0.8% |
| 1.50 | 11.5% | +2.0% |

**ERP敏感性**：
| ERP | Ke | 变化 |
|:---:|:---:|:---:|
| 5% | 8.4% | -1.1% |
| 5.5% | 8.9% | -0.6% |
| **6%** | **9.5%** | 基准 |
| 7% | 10.7% | +1.2% |

## ⚠️ 五、常见问题与注意事项

### 5.1 β的稳定性问题

**β随时间变化**：
- 业务结构变化导致β变化
- 财务杠杆变化影响β
- 建议使用2-3年平均β

**β的预测性**：
- 历史β不一定代表未来
- 重大事件后β可能突变
- 定期更新β估计

### 5.2 无风险利率的选择

**期限匹配**：
- DCF估值用10年期
- 短期项目用短期利率
- 长期基础设施用30年期

**名义vs实际利率**：
- 通常使用名义利率
- 如果现金流是实际值，使用实际利率
- 保持一致性

### 5.3 市场风险溢价的争议

**历史数据vs前瞻估计**：
- 历史数据：客观但有 survivorship bias
- 前瞻估计：更合理但主观性强

**地理差异**：
- 成熟市场：4%-6%
- 新兴市场：6%-8%

### 5.4 非上市公司的处理

**可比公司法**：
1. 选择3-5家可比上市公司
2. 计算平均β
3. 去杠杆调整
4. 按目标公司资本结构加杠杆

**行业平均β**：
- 使用行业平均β
- 调整规模差异
