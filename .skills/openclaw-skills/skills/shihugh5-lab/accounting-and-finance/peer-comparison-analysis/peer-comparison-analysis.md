# 同业对比分析

## 一、理论背景与核心概念

### 1.1 为什么要做同业对比？

**核心问题**：
1. 公司在行业中处于什么地位？
2. 与竞争对手相比，优势和劣势在哪里？
3. 财务指标是否优于/劣于行业平均？
4. 估值水平是否合理？

**分析原则**：
- 没有绝对的好坏，只有相对的优劣
- 不同行业不能简单比较
- 要找对可比公司
- 要理解差异背后的原因

### 1.2 可比公司选择

**选择标准**：

| 维度 | 标准 | 示例 |
|:---|:---|:---|
| **行业相同** | 同一细分行业 | 万华化学 vs 巴斯夫、科思创 |
| **业务相似** | 主要产品/服务相同 | MDI生产企业 |
| **规模相近** | 收入规模差异 < 3倍 | 收入500亿 vs 200-1500亿 |
| **市场相似** | 主要市场/客户相同 | 都是面向全球市场 |
| **生命周期** | 发展阶段相似 | 都是成熟期企业 |

**可比公司来源**：
1. **直接竞争对手**：市场份额竞争关系
2. **上下游企业**：产业链相关
3. **行业龙头**：学习标杆
4. **次新股**：同时间段上市

## 三、公式与计算逻辑

### 3.1 对比分析核心公式

**相对优势指数**：
```
相对优势指数 = (公司指标值 - 行业平均值) / 行业平均值 × 100%

解释：
- > 20%：显著优势，竞争力强
- 10%-20%：适度优势，略优于行业
- -10%-10%：平均水平，与行业相当
- -20%--10%：适度劣势，略逊于行业
- < -20%：显著劣势，需关注
```

**百分位排名**：
```
百分位排名 = (公司在行业中的排名 / 行业公司总数) × 100%

解释：
- 前10%：行业顶尖
- 10%-25%：行业上游
- 25%-50%：行业中游
- 50%-75%：行业中下游
- 后25%：行业落后
```

**Z-score标准化**：
```
Z = (X - μ) / σ

其中：
X = 公司指标值
μ = 行业平均值
σ = 行业标准差

解释：
- Z > 2：显著高于行业
- 0 < Z < 2：高于行业平均
- -2 < Z < 0：低于行业平均
- Z < -2：显著低于行业
```

### 3.2 综合评分模型

**评分体系**：
```python
综合评分 = Σ(单项指标得分 × 权重)

权重设置（示例）：
盈利能力指标：30%
  - ROE：10%
  - 毛利率：10%
  - 净利率：10%

营运效率指标：25%
  - 资产周转率：10%
  - 现金周期：10%
  - 存货周转：5%

财务风险指标：20%
  - 资产负债率：10%
  - 利息保障倍数：10%

成长性指标：15%
  - 收入增长率：8%
  - 利润增长率：7%

估值指标：10%
  - PE：5%
  - PB：5%
```

**单项指标得分计算**：
```
如果 公司值 ≥ 行业平均 × 1.2：
    得分 = 100
elif 公司值 ≥ 行业平均：
    得分 = 80 + (公司值/行业平均 - 1) × 100
elif 公司值 ≥ 行业平均 × 0.8：
    得分 = 60 + (公司值/行业平均 - 0.8) × 100
else：
    得分 = 公司值/行业平均 × 75
```

### 3.3 统计显著性检验

**t检验（公司与行业差异是否显著）**：
```
t = (公司均值 - 行业均值) / (行业标准差 / √n)

判断：
- |t| > 2：差异显著
- |t| < 2：差异不显著
```

**Wilcoxon秩和检验（非参数检验）**：
```
适用于：数据分布不正态时
判断公司是否显著优于/劣于行业
```

## 二、数据标准化与对比方法

### 2.1 数据标准化方法

**问题**：不同规模公司的绝对值指标无法直接对比

**解决方案**：

#### 方法一：比率标准化
```
所有指标转换为比率形式：
- ROE = 净利润 / 股东权益
- 毛利率 = 毛利 / 营业收入
- 周转率 = 营业收入 / 平均总资产
```

#### 方法二：Z-score标准化
```python
def z_score_standardize(value, industry_values):
    """
    Z-score标准化：消除量纲影响
    
    公式：Z = (X - μ) / σ
    
    其中：
    X = 公司指标值
    μ = 行业平均值
    σ = 行业标准差
    """
    import numpy as np
    
    mean = np.mean(industry_values)
    std = np.std(industry_values)
    
    if std == 0:
        return 0
    
    z_score = (value - mean) / std
    
    return z_score

# 示例：万华化学ROE标准化
roe_万华 = 0.264
roe_行业 = [0.264, 0.185, 0.152, 0.220, 0.120, 0.150, 0.080, 0.030]
z_score = z_score_standardize(roe_万华, roe_行业)
# 结果：Z = 1.85（高于行业平均1.85个标准差）
```

#### 方法三：百分位排名
```python
def percentile_rank(value, industry_values):
    """
    计算行业分位排名
    """
    sorted_values = sorted(industry_values)
    n = len(sorted_values)
    
    # 找到value的位置
    position = sorted_values.index(value)
    
    # 计算百分位（越高越好）
    percentile = (position + 1) / n * 100
    
    return percentile

# 示例
percentile = percentile_rank(0.264, roe_行业)
# 结果：前12.5%（行业前10%）
```

### 2.2 对比差异显著性检验

**问题**：公司指标与行业平均的差异是偶然还是显著？

**t检验方法**：
```python
def significance_test(company_value, industry_values, alpha=0.05):
    """
    单样本t检验：判断公司与行业平均的差异是否显著
    
    H0: 公司指标 = 行业平均（无显著差异）
    H1: 公司指标 ≠ 行业平均（有显著差异）
    """
    import numpy as np
    from scipy import stats
    
    industry_mean = np.mean(industry_values)
    industry_std = np.std(industry_values, ddof=1)
    n = len(industry_values)
    
    # 计算t统计量
    t_stat = (company_value - industry_mean) / (industry_std / np.sqrt(n))
    
    # 计算p值（双侧检验）
    p_value = 2 * (1 - stats.t.cdf(abs(t_stat), df=n-1))
    
    # 判断显著性
    is_significant = p_value < alpha
    
    return {
        "公司指标": company_value,
        "行业平均": industry_mean,
        "差异": company_value - industry_mean,
        "t统计量": t_stat,
        "p值": p_value,
        "显著性": "显著" if is_significant else "不显著",
        "结论": f"{'显著' if is_significant else '不显著'}高于行业平均" if company_value > industry_mean else f"{'显著' if is_significant else '不显著'}低于行业平均"
    }

# 示例：万华化学ROE显著性检验
result = significance_test(0.264, roe_行业)
# 结果：t=3.52, p=0.009 < 0.05，显著高于行业平均
```

### 2.3 对比指标计算方法

#### 相对优势指数
```
相对优势指数 = (公司指标 - 行业平均) / 行业平均 × 100%

解释：
- > 20%：显著优势
- 10%~20%：适度优势
- -10%~10%：平均水平
- -20%~-10%：适度劣势
- < -20%：显著劣势
```

#### 综合得分计算
```python
def comprehensive_score(company_metrics, industry_metrics, weights):
    """
    计算综合竞争力得分
    
    company_metrics: 公司各项指标值 dict
    industry_metrics: 行业平均各项指标值 dict
    weights: 各项指标权重 dict
    """
    total_score = 0
    detail_scores = {}
    
    for metric, weight in weights.items():
        company_value = company_metrics[metric]
        industry_value = industry_metrics[metric]
        
        # 计算相对得分（假设指标越高越好）
        if industry_value != 0:
            relative_score = (company_value / industry_value) * 100
        else:
            relative_score = 100 if company_value == 0 else 200
        
        # 限制得分范围（50-150）
        relative_score = max(50, min(150, relative_score))
        
        weighted_score = relative_score * weight
        total_score += weighted_score
        detail_scores[metric] = {
            "公司值": company_value,
            "行业均值": industry_value,
            "相对得分": relative_score,
            "权重": weight,
            "加权得分": weighted_score
        }
    
    return total_score, detail_scores

# 示例：万华化学综合得分
metrics_万华 = {
    "ROE": 0.264,
    "毛利率": 0.30,
    "资产周转率": 1.22,
    "收入增长率": 0.15
}

metrics_行业 = {
    "ROE": 0.12,
    "毛利率": 0.25,
    "资产周转率": 0.85,
    "收入增长率": 0.08
}

weights = {
    "ROE": 0.30,
    "毛利率": 0.25,
    "资产周转率": 0.25,
    "收入增长率": 0.20
}

score, details = comprehensive_score(metrics_万华, metrics_行业, weights)
# 结果：综合得分128.5分（高于行业平均28.5%）
```

```python
def roe_comparison(company_roe, industry_roes):
    """
    ROE对比分析
    """
    industry_avg = sum(industry_roes) / len(industry_roes)
    industry_max = max(industry_roes)
    industry_min = min(industry_roes)
    
    # 计算分位数
    sorted_roes = sorted(industry_roes + [company_roe])
    percentile = sorted_roes.index(company_roe) / len(sorted_roes) * 100
    
    result = {
        "公司ROE": company_roe,
        "行业平均": industry_avg,
        "行业最高": industry_max,
        "行业最低": industry_min,
        "公司排名": f"前{100-percentile:.0f}%"
    }
    
    # 评价
    if company_roe > industry_avg * 1.2:
        result["评价"] = "✅ ROE显著高于行业平均"
    elif company_roe > industry_avg * 0.9:
        result["评价"] = "✅ ROE处于行业平均水平"
    else:
        result["评价"] = "⚠️ ROE低于行业平均，需分析原因"
    
    return result
```

## 十一、常见问题与注意事项

### 11.1 可比公司选择的常见问题

**Q1：找不到完全可比的公司怎么办？**

**A**：
- 放宽选择标准（如规模差异从3倍放宽到5倍）
- 选择业务相似度高的跨行业公司
- 使用行业平均值作为对比基准
- 分别与不同维度的标杆对比

**Q2：可比公司数量多少合适？**

**A**：
- 最少3家（太少缺乏代表性）
- 建议5-8家（平衡全面性与操作成本）
- 最多10家（太多增加分析复杂度）

### 11.2 数据对比的常见问题

**Q3：不同会计政策的公司如何对比？**

**A**：
- 尽量调整至同一口径
- 使用受会计政策影响小的指标
- 在分析中说明会计政策差异

**Q4：如何处理极端值？**

**A**：
- 识别极端值原因
- 使用中位数代替平均值
- 分别报告包含/不包含极端值的结果

### 11.3 分析判断的注意事项

**注意事项1：避免简单排名**
- 分析指标的驱动因素
- 对比指标的质量
- 结合行业特性判断

**注意事项2：关注指标的关联性**
- 综合分析，识别矛盾信号
- 高ROE+高杠杆=风险高

**注意事项3：考虑行业周期**
- 使用多年平均数据
- 区分周期性和结构性因素
