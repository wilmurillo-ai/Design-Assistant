# 银行股估值方法

## 一、理论背景与核心概念

### 1.1 银行的特殊性

**为什么银行需要特殊估值方法？**

**高杠杆经营**：
```
银行杠杆倍数：权益乘数通常10-20倍
vs 一般企业：权益乘数通常1.5-3倍

影响：
- 微小资产质量变化对权益影响巨大
- 传统PE估值失效（盈利波动大）
- PB估值更适用（资产为基础）
```

**监管约束**：
| 监管指标 | 要求 | 影响 |
|:---|:---:|:---|
| 资本充足率 | ≥10.5% | 限制资产扩张 |
| 核心一级资本充足率 | ≥7.5% | 限制分红能力 |
| 拨备覆盖率 | ≥150% | 影响利润释放 |
| 不良贷款率 | 监管关注 | 影响资产质量 |

**资产负债表驱动**：
```
银行盈利模式：
资产（贷款、债券）× 利差 - 成本 = 利润

vs 一般企业：
收入 - 成本 = 利润

银行利润更多由资产规模驱动，而非收入增长
```

### 1.2 银行估值的核心方法

**方法一：PB-ROE框架**（最常用）
```
合理PB = (ROE - g) / (Ke - g)

其中：
ROE = 净资产收益率
Ke = 股权成本
g = 永续增长率

目标价值 = 每股净资产 × 合理PB
```

**方法二：股息贴现模型（DDM）**
```
价值 = D1 / (Ke - g)

其中：
D1 = 下期预期股息
Ke = 股权成本
g = 股息增长率
```

**方法三：剩余收益模型**
```
价值 = 账面价值 + 剩余收益现值

剩余收益 = 净利润 - 股权成本 × 期初账面价值
```

### 1.3 银行关键指标

**盈利能力指标**：
| 指标 | 公式 | 优秀标准 |
|:---|:---|:---:|
| ROE | 净利润/净资产 | >15% |
| ROA | 净利润/总资产 | >1% |
| NIM | 净利息收入/生息资产 | >2% |
| 成本收入比 | 管理费用/营业收入 | <35% |

**资产质量指标**：
| 指标 | 公式 | 安全标准 |
|:---|:---|:---:|
| 不良贷款率 | 不良贷款/总贷款 | <1.5% |
| 拨备覆盖率 | 拨备/不良贷款 | >200% |
| 关注类贷款率 | 关注类贷款/总贷款 | <3% |

**资本充足指标**：
| 指标 | 监管要求 | 优秀标准 |
|:---|:---:|:---:|
| 资本充足率 | ≥10.5% | >12% |
| 一级资本充足率 | ≥8.5% | >10% |
| 核心一级资本充足率 | ≥7.5% | >9% |

## 二、详细操作步骤

### 步骤1：评估资产质量

**1.1 不良资产分析**：
```python
def 资产质量评估(不良贷款余额, 关注类贷款余额, 总贷款余额, 拨备余额):
    """
    评估银行资产质量
    """
    不良率 = 不良贷款余额 / 总贷款余额
    关注率 = 关注类贷款余额 / 总贷款余额
    拨备覆盖率 = 拨备余额 / 不良贷款余额
    拨贷比 = 拨备余额 / 总贷款余额
    
    # 资产质量评级
    if 不良率 < 0.01 and 拨备覆盖率 > 300:
        质量评级 = '优秀'
    elif 不良率 < 0.015 and 拨备覆盖率 > 200:
        质量评级 = '良好'
    elif 不良率 < 0.025 and 拨备覆盖率 > 150:
        质量评级 = '一般'
    else:
        质量评级 = '较差'
    
    return {
        '不良率': 不良率,
        '关注率': 关注率,
        '拨备覆盖率': 拨备覆盖率,
        '拨贷比': 拨贷比,
        '质量评级': 质量评级
    }
```

### 步骤2：评估盈利能力

**2.1 ROE分解**：
```python
def ROE分解分析(净利润, 净资产, 总资产, 营业收入):
    """
    用杜邦分析法分解银行ROE
    """
    ROE = 净利润 / 净资产
    ROA = 净利润 / 总资产
    权益乘数 = 总资产 / 净资产
    净利润率 = 净利润 / 营业收入
    资产周转率 = 营业收入 / 总资产
    
    return {
        'ROE': ROE,
        'ROA': ROA,
        '权益乘数': 权益乘数,
        '净利润率': 净利润率,
        '资产周转率': 资产周转率,
        '杜邦分解': ROE == ROA * 权益乘数
    }
```

### 步骤3：PB-ROE估值

**3.1 计算合理PB**：
```python
def PB_ROE估值(ROE, 股权成本, 永续增长率, 每股净资产):
    """
    使用PB-ROE框架估值
    """
    if 股权成本 <= 永续增长率:
        return None, "股权成本必须大于永续增长率"
    
    合理PB = (ROE - 永续增长率) / (股权成本 - 永续增长率)
    
    目标价值 = 每股净资产 * 合理PB
    
    return {
        '合理PB': 合理PB,
        '每股净资产': 每股净资产,
        '目标价值': 目标价值,
        '隐含ROE': ROE
    }

# 工商银行演示（国有大行代表）
估值结果 = PB_ROE估值(0.115, 0.09, 0.02, 8.9)
# 结果：合理PB=1.36倍，目标价值12.1元
```

### 步骤4：股息贴现估值

**4.1 DDM估值**：
```python
def 股息贴现估值(每股股息, 股权成本, 股息增长率):
    """
    使用戈登增长模型估值
    """
    if 股权成本 <= 股息增长率:
        return None, "股权成本必须大于股息增长率"
    
    价值 = 每股股息 * (1 + 股息增长率) / (股权成本 - 股息增长率)
    
    return {
        '每股股息': 每股股息,
        '股息增长率': 股息增长率,
        '股权成本': 股权成本,
        '价值': 价值
    }
```

### 步骤5：资本充足率分析

**5.1 资本充足评估**：
```python
def 资本充足评估(核心一级资本, 一级资本, 总资本, 风险加权资产):
    """
    评估银行资本充足情况
    """
    核心一级资本充足率 = 核心一级资本 / 风险加权资产
    一级资本充足率 = 一级资本 / 风险加权资产
    资本充足率 = 总资本 / 风险加权资产
    
    评估 = {
        '核心一级资本充足率': (核心一级资本充足率, 0.075, '核心资本'),
        '一级资本充足率': (一级资本充足率, 0.085, '一级资本'),
        '资本充足率': (资本充足率, 0.105, '总资本')
    }
    
    结果 = {}
    for 指标, (实际值, 监管要求, 类型) in 评估.items():
        缓冲 = 实际值 - 监管要求
        if 缓冲 > 0.02:
            评价 = '充足'
        elif 缓冲 > 0:
            评价 = '达标'
        else:
            评价 = '不足'
        
        结果[指标] = {
            '实际值': 实际值,
            '监管要求': 监管要求,
            '缓冲': 缓冲,
            '评价': 评价
        }
    
    return 结果
```

## 三、公式与计算逻辑

### 3.1 PB-ROE模型公式

**合理PB推导**：
```
根据戈登增长模型：
P = D1 / (Ke - g)

银行股分红率通常稳定，假设分红率为b：
D1 = E1 × b = B0 × ROE × b

PB = P / B0 = (ROE × b) / (Ke - g)

由于 g = ROE × (1-b)，代入得：
PB = (ROE - g) / (Ke - g)

简化版（当银行稳定分红时）：
合理PB = ROE / Ke
```

### 3.2 股息贴现模型

**戈登增长模型**：
```
价值 = D1 / (Ke - g)

其中：
D1 = D0 × (1 + g)
D0 = 当前每股股息
Ke = 股权成本（CAPM计算）
g = 永续增长率（通常2-3%）
```

### 3.3 净息差（NIM）分析

**NIM计算**：
```
NIM = 净利息收入 / 平均生息资产

净利息收入 = 利息收入 - 利息支出

影响因素：
- 资产端收益率（贷款、债券利率）
- 负债端成本率（存款、同业利率）
- 资产负债结构
```

## 四、实例演示：PB-ROE框架应用

### 4.1 工商银行PB-ROE估值案例

虽然工行是典型银行，但可用PB-ROE框架演示估值逻辑：

**参数设定**：
| 参数 | 数值 | 说明 |
|:---|---:|:---|
| ROE | 11.5% | 工行实际ROE |
| 股权成本(Ke) | 9% | CAPM计算 |
| 永续增长率(g) | 3% | GDP增速 |
| 每股净资产 | 70元 | 工行实际BPS |

**合理PB计算**：
```
合理PB = (ROE - g) / (Ke - g)
       = (26% - 3%) / (9% - 3%)
       = 23% / 6%
       = 3.83倍

目标价值 = 70元 × 3.83 = 268元
```

**与当前股价对比**：
- 当前股价：5.0元
- 目标价值：12.1元
- 潜在涨幅：144%

**注意**：这是基于PB-ROE框架的理论估值，工行作为国有大行，实际估值需结合行业特性。

**假设某银行数据**：

## 五、常见问题与注意事项

### 5.1 PB-ROE框架适用所有银行吗？

**Q：PB-ROE框架适用于所有银行吗？**

**A**：基本适用，但需调整：
- **大型银行**：ROE稳定，框架适用性好
- **成长型银行**：g估计难度大，需调整
- **问题银行**：资产质量差，需下调ROE
- **特殊银行**：投行、资管为主，需调整模型

### 5.2 如何处理银行资产质量问题？

**Q：银行不良资产如何影响估值？**

**A**：
```
方法一：调整净资产
调整后净资产 = 账面净资产 - 潜在不良损失

方法二：调整ROE
调整后ROE = 报告ROE × 资产质量系数

方法三：直接估值折扣
目标PB = 理论PB × 资产质量折扣（0.5-1.0）
```

### 5.3 银行股估值的特殊风险

**银行特有风险**：
1. **资产质量风险**：不良贷款爆发
2. **利率风险**：息差收窄
3. **监管风险**：政策变化
4. **系统性风险**：金融危机
5. **流动性风险**：挤兑风险

## 附录：银行估值深度分析补充材料

### A.1 PB-ROE关系详解

**PB-ROE线性关系**：
```
理论上：PB = 1 + (ROE - r) / (r - g)

当 ROE = r 时，PB = 1
当 ROE > r 时，PB > 1
当 ROE < r 时，PB < 1

其中r为要求回报率，g为增长率
```

**工商银行PB-ROE关系**：
```
ROE = 11.5%
r = 9%
g = 3%

PB = 1 + (26% - 9%) / (9% - 3%)
   = 1 + 17% / 6%
   = 1 + 2.83
   = 3.83倍

与公式一致：(26%-3%)/(9%-3%) = 3.83
```

### A.5 常见问题速查

**Q1：银行股为什么通常PB<1？**
A：市场担忧资产质量，给予折价。若ROE持续低于股权成本，PB<1合理。

**Q2：高ROE银行一定好吗？**
A：不一定，可能伴随高风险（高杠杆、高风险资产）。需结合资产质量。

**Q3：银行股适合长期投资吗？**
A：适合。银行股分红稳定，股息率高，适合作为防御性配置。

**Q4：利率下行对银行的影响？**
A：短期息差收窄（负面），长期贷款需求增加（正面）。总体中性偏负面。

### A.7 更多估值案例

**案例1：高ROE成长型银行**
```
假设：
- ROE = 18%（高于平均）
- 增长率 = 8%（成长型）
- 股权成本 = 10%
- 每股净资产 = 12元

合理PB = (18% - 8%) / (10% - 8%) = 10% / 2% = 5倍
目标价 = 12 × 5 = 60元

特点：高成长支撑高估值，类似印度银行股
```

**案例2：低ROE价值型银行**
```
假设：
- ROE = 8%（低于平均）
- 增长率 = 2%
- 股权成本 = 10%
- 每股净资产 = 15元

合理PB = (8% - 2%) / (10% - 2%) = 6% / 8% = 0.75倍
目标价 = 15 × 0.75 = 11.25元

特点：ROE低于要求回报，应折价交易，类似欧洲银行股
```

**案例3：工商银行估值详解（国有大行代表）**
```
使用银行估值框架评估工行：

工行参数：
- ROE = 11.5%（稳健）
- 增长率 = 2%
- 股权成本 = 9%
- 每股净资产 = 8.9元

合理PB = (11.5% - 2%) / (9% - 2%) = 9.5% / 7% = 1.36倍
目标价 = 8.9 × 1.36 = 12.1元

当前股价 = 5.0元
低估幅度 = (12.1 - 5.0) / 5.0 = 142%

注：这是基于2023年数据估算，实际需结合最新财报
```

### A.8 银行估值关键假设

**关键假设及敏感性**：

| 假设 | 基准值 | 变动 | 对估值影响 |
|:---|:---:|:---:|:---:|
| ROE | 12% | ±1% | ±8% |
| 股权成本 | 10% | ±1% | ±10% |
| 永续增长率 | 3% | ±1% | ±5% |
| 分红率 | 30% | ±5% | ±3% |

**最敏感因素**：股权成本和ROE
