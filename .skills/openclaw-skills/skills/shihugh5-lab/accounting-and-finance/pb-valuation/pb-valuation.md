# 市净率(PB)估值方法

## 一、理论背景与核心概念

### 1.1 PB的本质与意义

**什么是市净率？**
> 市净率(Price-to-Book Ratio, PB)是股价与每股净资产的比率，反映市场对公司账面价值的溢价程度。

**PB的经济含义**：
```
PB = 股价 / 每股净资产(BPS)
   = 市值 / 股东权益

解释：
- PB = 1.0：股价等于账面价值（平价）
- PB > 1.0：股价高于账面价值（溢价）
- PB < 1.0：股价低于账面价值（折价）
```

**PB与清算价值**：
```
清算价值 = 资产变现价值 - 负债

PB < 1 意味着：
市值 < 账面净资产
可能低于清算价值（如果资产不贬值）
→ 理论上存在投资价值
```

### 1.2 PB vs PE的区别

| 维度 | PE估值 | PB估值 |
|:---|:---|:---|
| **估值基础** | 盈利能力 | 资产价值 |
| **核心指标** | 净利润 | 净资产 |
| **适用行业** | 盈利稳定企业 | 重资产/金融机构 |
| **盈利要求** | 必须盈利 | 可以为负 |
| **资产要求** | 不重要 | 核心指标 |
| **周期影响** | 大（盈利波动） | 小（资产稳定） |

**选择原则**：
- 盈利稳定 → PE
- 重资产/资产驱动 → PB
- 盈利波动大 → PB
- 金融机构 → PB

### 1.3 PB的财务理论基础

**PB与ROE的理论关系**：

从Gordon增长模型推导：
```
P = D₁ / (r - g)

其中：
D₁ = E₁ × b = (B₀ × ROE) × b
（E₁ = B₀ × ROE，b为分红率）

PB = P / B₀ = (ROE × b) / (r - g)

由于 g = ROE × (1-b)，代入得：
PB = (ROE - g) / (r - g)
```

**关键洞察**：
```
当 ROE = r 时，PB = 1（合理平价）
当 ROE > r 时，PB > 1（溢价合理）
当 ROE < r 时，PB < 1（折价合理）
```

### 1.4 PB的生命周期特征

**不同阶段的典型PB水平**：

| 企业阶段 | 典型PB | 原因 |
|:---|:---:|:---|
| **初创期** | 3-5倍 | 轻资产，成长预期高 |
| **高速成长期** | 2-4倍 | 资产快速增值 |
| **稳定成长期** | 1.5-2.5倍 | ROE高于要求回报 |
| **成熟期** | 1-1.5倍 | ROE接近要求回报 |
| **衰退期** | 0.5-1倍 | 资产减值风险 |

**万华化学PB判断**：
- 重资产化工企业
- ROE 26% > 要求回报9%
- 合理PB应在1.5-2.5倍区间


> "市净率估值适用于重资产行业和金融机构。分析时要注意账面价值是否真实反映资产价值，对于重资产行业，资产质量比账面数字更重要。"

1. **资产质量**：账面价值≠变现价值，需评估资产质量
2. **ROE匹配**：PB应与ROE匹配，高ROE支持高PB
3. **行业特性**：不同行业PB水平差异大，需同业对比
4. **周期性调整**：周期底部PB低可能是机会

## 二、详细操作步骤

### 步骤1：确定PB适用性

**1.1 PB估值的适用条件**：

**高度适用**（优先使用PB）：
- [x] 重资产行业（制造业、能源、公用事业）
- [x] 金融机构（银行、保险）
- [x] 房地产行业
- [x] 周期性行业（盈利波动大）
- [x] 亏损企业（无法用PE）

**轻度适用**（PB作为参考）：
- 一般制造业
- 有形资产占比较高的企业

**不适用**（避免使用PB）：
- 轻资产行业（科技、互联网、服务业）
- 无形资产占比高的企业
- 负净资产企业

**万华化学适用性**：✅ 高度适用
- 重资产化工企业
- 固定资产占比高
- 有周期性特征

### 步骤2：计算账面价值

**2.1 每股净资产计算**：
```python
def 计算每股净资产(股东权益合计, 优先股权益, 总股本):
    """
    计算每股净资产（账面价值）
    """
    普通股权益 = 股东权益合计 - 优先股权益
    每股净资产 = 普通股权益 / 总股本
    
    return {
        '股东权益合计': 股东权益合计,
        '优先股权益': 优先股权益,
        '普通股权益': 普通股权益,
        '总股本': 总股本,
        '每股净资产': 每股净资产
    }

# 万华化学示例
股东权益 = 840  # 亿元
优先股 = 0      # 无优先股
总股本 = 31.4   # 亿股

# 结果：每股净资产 = 26.75元
```

### 步骤3：计算当前PB

**3.1 PB计算**：
```python
def 计算PB(股价, 每股净资产):
    """
    计算市净率PB
    """
    if 每股净资产 <= 0:
        return None, "净资产为负，PB无意义"
    
    PB = 股价 / 每股净资产
    
    return {
        '股价': 股价,
        '每股净资产': 每股净资产,
        'PB': PB,
        '溢价率': (PB - 1) * 100  # 相对账面价值的溢价/折价
    }

# 重新计算：
PB结果 = 计算PB(110, 70)
# 结果：{'PB': 1.57, '溢价率': 57%}
```

### 步骤4：分析ROE-PB关系

**4.1 ROE-PB理论关系应用**：
```python
def ROE_PB分析(ROE, 要求回报率, 增长率, 当前PB):
    """
    基于ROE分析PB的合理性
    """
    # 理论合理PB
    理论PB = (ROE - 增长率) / (要求回报率 - 增长率)
    
    # PB判断
    if 当前PB < 理论PB * 0.8:
        评价 = '低估'
        建议 = '买入'
    elif 当前PB <= 理论PB * 1.2:
        评价 = '合理'
        建议 = '持有'
    else:
        评价 = '高估'
        建议 = '卖出'
    
    return {
        '理论PB': 理论PB,
        '当前PB': 当前PB,
        '差异': (当前PB - 理论PB) / 理论PB,
        '评价': 评价,
        '建议': 建议
    }

# 万华化学
分析结果 = ROE_PB分析(
    ROE=0.26,           # 26%
    要求回报率=0.09,     # 9%
    增长率=0.03,         # 3%
    当前PB=1.57
)
# 结果：理论PB=2.88，当前PB低估46%，评价=低估
```

| ROE \ PB | <1.0 | 1.0-1.5 | 1.5-2.0 | >2.0 |

| >20% | 极度低估 | 低估 | 合理 | 略高 |
| 15-20% | 低估 | 合理 | 略高 | 高估 |

### 步骤6：计算可比PB区间

**6.1 可比PB统计**：
```python
def 可比PB分析(可比公司PB列表, 可比公司ROE列表, 目标ROE):
    """
    分析可比PB并调整
    """
    import numpy as np
    
    基础统计 = {
        '平均PB': np.mean(可比公司PB列表),
        '中位数PB': np.median(可比公司PB列表),
        'PB区间': (min(可比公司PB列表), max(可比公司PB列表))
    }
    
    # ROE调整
    平均可比ROE = np.mean(可比公司ROE列表)
    ROE差异 = 目标ROE - 平均可比ROE
    
    # ROE每高5%，PB可高0.5倍
    ROE调整 = ROE差异 / 0.05 * 0.5
    
    调整后PB = 基础统计['中位数PB'] + ROE调整
    
    return {
        '基础统计': 基础统计,
        'ROE调整': ROE调整,
        '调整后PB': 调整后PB,
        '合理区间': (调整后PB * 0.9, 调整后PB * 1.1)
    }

# 万华化学
可比PB = [1.8, 2.2, 1.5]
可比ROE = [0.20, 0.18, 0.12]
目标ROE = 0.26

结果 = 可比PB分析(可比PB, 可比ROE, 目标ROE)
# 结果：调整后PB=2.3，合理区间(2.1, 2.5)
```

### 步骤7：计算目标价值

**7.1 PB估值计算**：
```python
def PB估值计算(每股净资产, PB区间, 当前股价):
    """
    使用PB区间计算估值
    """
    保守估值 = 每股净资产 * PB区间[0]
    基准估值 = 每股净资产 * PB区间[1]
    乐观估值 = 每股净资产 * PB区间[2] if len(PB区间) > 2 else None
    
    if 乐观估值:
        平均估值 = (保守估值 + 基准估值 + 乐观估值) / 3
    else:
        平均估值 = (保守估值 + 基准估值) / 2
    
    上涨空间 = (平均估值 - 当前股价) / 当前股价
    
    return {
        '保守估值': 保守估值,
        '基准估值': 基准估值,
        '乐观估值': 乐观估值,
        '平均估值': 平均估值,
        '上涨空间': 上涨空间
    }

# 万华化学
估值结果 = PB估值计算(70, (2.1, 2.3, 2.5), 110)
# 结果：保守147元，基准161元，乐观175元，平均161元，上涨空间46%
```

## 三、公式与计算逻辑

### 3.1 PB-ROE关系公式

**理论公式**：
```
PB = (ROE - g) / (r - g)

变形：
PB = 1 + (ROE - r) / (r - g)

隐含ROE：
ROE = PB × (r - g) + g
```

**简化判断**：
```
当 ROE > r 时，合理PB > 1
当 ROE = r 时，合理PB = 1  
当 ROE < r 时，合理PB < 1
```

### 3.2 Tobin's Q（托宾Q值）

**概念**：
```
Tobin's Q = 市场价值 / 资产重置成本
          = (股权市值 + 债务市值) / 资产重置成本

近似计算：
Q ≈ PB × (账面资产 / 重置成本)
```

**判断**：
- Q > 1：市场价值高于重置成本，投资有利
- Q < 1：市场价值低于重置成本，并购有利

### 3.3 清算价值计算

**公式**：
```
清算价值 = 资产变现价值 - 负债
         = Σ(各项资产 × 变现折扣率) - 负债

常见折扣率：
- 现金：100%
- 应收账款：70-80%
- 存货：50-70%
- 固定资产：40-60%
- 无形资产：10-30%
```

**清算价值PB**：
```
如果 股价 < 清算价值 → 绝对低估
如果 股价 < 账面价值 → 相对低估
```

## 五、常见问题与注意事项

### 5.1 PB<1一定低估吗？

**Q：PB小于1一定说明公司被低估吗？**

**A**：不一定，可能原因：
1. **资产质量差**：账面价值虚高，实际价值低
2. **盈利能力差**：ROE低于要求回报
3. **行业衰退**：资产面临减值
4. **清算风险**：可能资不抵债

**判断标准**：
```
PB < 1 且 ROE > r → 低估
PB < 1 且 ROE < r → 合理（可能继续跌）
```

### 5.2 轻资产公司能用PB吗？

**Q：科技、互联网等轻资产公司能用PB估值吗？**

**A**：不推荐，原因：
1. **无形资产未入账**：品牌、技术、用户价值不在账上
2. **账面价值失真**：不能反映真实价值
3. **PB可能极高**：几十倍PB无参考意义

**替代方法**：
- PE估值（盈利驱动）
- PS估值（收入驱动）
- 用户价值法（互联网公司）

### 5.3 账面价值何时需要调整？

**Q：什么情况下需要对账面价值进行调整？**

**A**：以下情况需要调整：

| 情况 | 调整方向 | 示例 |
|:---|:---:|:---|
| 资产减值未充分计提 | 下调 | 应收账款坏账 |
| 存货跌价 | 下调 | 过时库存 |
| 资产重估增值 | 上调 | 房地产升值 |
| 无形资产价值 | 上调 | 品牌价值 |
| 表外负债 | 下调 | 担保、诉讼 |

### 5.4 PB估值的局限性

**主要局限**：
1. **账面价值失真**：历史成本计量，可能偏离市场价值
2. **无形资产缺失**：品牌、专利未体现在账上
3. **资产质量差异**：同样账面价值，质量可能天差地别
4. **会计政策差异**：不同折旧方法影响账面价值
5. **负净资产**：PB为负，无意义
