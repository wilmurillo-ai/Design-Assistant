# DCF三阶段模型估值

## 📚 一、理论背景与核心概念

### 1.1 三阶段模型的核心思想

**企业生命周期理论**：
```
成长轨迹：高速增长 → 增速放缓 → 稳定永续

  增长率
    │    ╭────╮
高  │   ╱      ╲____
    │  ╱              ╲___
    │ ╱                     ╲______
    │╱                              ╲____
    └──────────────────────────────────────
      高速增长期   过渡期    永续稳定期
      (5-10年)   (3-5年)    (永续)
```

**阶段划分**：
| 阶段 | 名称 | 时长 | 增长率特征 |
|:---:|:---|:---:|:---|
| 第一阶段 | 高速增长期 | 5-10年 | 高增长率（10%-30%） |
| 第二阶段 | 过渡期 | 3-5年 | 增长率线性递减 |
| 第三阶段 | 永续期 | 永续 | 稳定增长（2%-4%） |

### 1.2 与两阶段模型的区别

| 对比维度 | 两阶段模型 | 三阶段模型 |
|:---|:---|:---|
| 阶段数 | 2 | 3 |
| 增长轨迹 | 突变式 | 渐进式 |
| 适用企业 | 成熟期/稳定期企业 | 成长期企业 |
| 预测精度 | 较粗 | 较细 |
| 计算复杂度 | 中等 | 较高 |

### 1.3 过渡期的设计逻辑

**为什么要过渡期？**
- 企业不会从高增长突然跳到稳定
- 竞争加剧、市场饱和导致增速逐步放缓
- 过渡期模拟真实的增长衰减过程

**增长率递减方式**：
```
线性递减：每年增长率减少相同幅度

例如：
第1年末增长率 = 20%
过渡期5年，最终增长率 = 3%

每年递减 = (20% - 3%) / 5 = 3.4%

第1年：20% → 第2年：16.6% → ... → 第5年：3%
```

### 1.4 模型的适用与局限

**适用企业**：
- 高速成长的科技公司
- 处于扩张期的行业龙头
- 有明显成长周期的企业

**局限性**：
- 参数较多，敏感性高
- 预测难度大，不确定性高
- 需要较强的假设依据

## 📝 二、详细操作步骤

### 步骤1：设定三阶段参数

**第一阶段（高速增长）**：
```
- 时长：n1年（通常5-10年）
- 增长率：g1（根据业务预测）
- FCFF基数：当前FCFF
```

**第二阶段（过渡）**：
```
- 时长：n2年（通常3-5年）
- 起始增长率：g1（第一阶段末）
- 最终增长率：g2（永续增长率）
- 递减方式：线性递减
```

**第三阶段（永续）**：
```
- 时长：永续
- 增长率：g2（2%-4%，通常3%）
```

### 步骤2：预测各阶段FCFF

**第一阶段FCFF**：
```
FCFF_t = FCFF_0 × (1 + g1)^t

其中t = 1, 2, ..., n1
```

**第二阶段FCFF**：
```
第n1+1年增长率：g1 - (g1-g2)/(n2+1)
第n1+2年增长率：g1 - 2×(g1-g2)/(n2+1)
...
第n1+n2年增长率：g2

FCFF逐年计算：FCFF_t = FCFF_{t-1} × (1 + g_t)
```

**第三阶段价值**：
```
终值 = FCFF_{n1+n2} × (1 + g2) / (WACC - g2)
```

### 步骤3：确定WACC

**注意**：三阶段可使用不同WACC
- 第一阶段：当前WACC（高风险）
- 第二阶段：逐步降低
- 第三阶段：稳定期WACC

**简化处理**：全程使用同一WACC

### 步骤4：折现计算

**第一阶段现值**：
```
PV1 = Σ[FCFF_t / (1+WACC)^t]  (t=1到n1)
```

**第二阶段现值**：
```
PV2 = Σ[FCFF_t / (1+WACC)^t]  (t=n1+1到n1+n2)
```

**第三阶段现值**：
```
PV3 = 终值 / (1+WACC)^(n1+n2)
```

**企业价值**：
```
企业价值 = PV1 + PV2 + PV3
```

### 步骤5：计算股权价值与每股价值

```
股权价值 = 企业价值 - 净金融负债
每股价值 = 股权价值 / 总股本
```

## 🔢 三、公式与计算逻辑

### 3.1 完整公式体系

**第一阶段**：
```
FCFF_t = FCFF_0 × (1 + g1)^t  (t = 1, 2, ..., n1)

PV1 = FCFF_0 × [(1+g1)/(1+WACC) + (1+g1)²/(1+WACC)² + ... + (1+g1)^{n1}/(1+WACC)^{n1}]

简化（当g1≠WACC）：
PV1 = FCFF_0 × (1+g1)/(WACC-g1) × [1 - ((1+g1)/(1+WACC))^{n1}]
```

**第二阶段**（线性递减）：
```
增长率序列：
g_{n1+1} = g1 - Δg
g_{n1+2} = g1 - 2×Δg
...
g_{n1+n2} = g2

其中：Δg = (g1 - g2) / (n2 + 1)

逐年计算FCFF并折现求和
```

**第三阶段**：
```
终值 = FCFF_{n1+n2} × (1 + g2) / (WACC - g2)

PV3 = 终值 / (1 + WACC)^(n1+n2)
```

### 3.2 计算流程图

```
输入参数
    │
    ├─ FCFF_0（当前自由现金流）
    ├─ g1（高速增长率）
    ├─ n1（高速年数）
    ├─ g2（永续增长率）
    ├─ n2（过渡年数）
    └─ WACC
    │
    ▼
┌─────────────────────────────────────────┐
│ 第一阶段：高速增长                       │
│ FCFF_t = FCFF_0 × (1+g1)^t              │
│ PV1 = Σ FCFF_t / (1+WACC)^t             │
└─────────────────┬───────────────────────┘
                  │
                  ▼
┌─────────────────────────────────────────┐
│ 第二阶段：过渡期                         │
│ 增长率线性递减                           │
│ 逐年计算FCFF并折现                       │
│ PV2 = Σ FCFF_t / (1+WACC)^t             │
└─────────────────┬───────────────────────┘
                  │
                  ▼
┌─────────────────────────────────────────┐
│ 第三阶段：永续期                         │
│ 终值 = FCFF × (1+g2) / (WACC-g2)        │
│ PV3 = 终值 / (1+WACC)^{n1+n2}           │
└─────────────────┬───────────────────────┘
                  │
                  ▼
          企业价值 = PV1 + PV2 + PV3
                  │
          减：净金融负债
                  │
                  ▼
          股权价值 → 每股价值
```

## ⚠️ 五、常见问题与注意事项

### 5.1 阶段长度的确定

**高速增长期**：
- 科技公司：7-10年
- 消费品牌：5-7年
- 制造业：3-5年

**考虑因素**：
- 行业生命周期
- 竞争格局变化
- 技术迭代速度
- 市场渗透率

### 5.2 增长率的合理性

**第一阶段增长率上限**：
- 不能超过行业天花板
- 要考虑市场份额提升空间
- 需验证历史增长可持续性

**参考标准**：
```
收入增长率 ≤ min(行业增长率×2, 30%)
FCFF增长率 ≤ 收入增长率 + 利润率改善
```

### 5.3 过渡期设计

**过渡期过短的问题**：
- 从高增长突然跳变不合理
- 估值波动大

**过渡期过长的影响**：
- 计算复杂
- 参数敏感性增加

**建议**：3-5年为合理区间

### 5.4 与两阶段模型的对比使用

| 场景 | 推荐模型 |
|:---|:---|
| 简单分析 | 两阶段模型 |
| 详细估值 | 三阶段模型 |
| 保守估值 | 两阶段（短高增长期）|
| 乐观估值 | 三阶段（长高增长期）|

## 补充理论框架

### 扩展理论背景

在深入理解本Skill的基础上，还需要考虑以下扩展理论：

**理论1：信息不对称理论**
管理层拥有比外部投资者更多的内部信息，会计政策选择可能成为信息不对称的表现形式。保守的会计政策可以减少信息不对称，但可能低估企业价值。

**理论2：契约理论**
会计政策选择受到各种契约关系的影响：
- 债务契约：可能促使选择增加资产的会计政策
- 薪酬契约：可能促使选择增加利润的会计政策
- 政治成本：大企业可能选择减少利润的会计政策

**理论3：信号传递理论**
会计政策变更可能向市场传递信号：
- 变更为更保守的政策：可能预示风险增加
- 变更为更激进的政策：可能预示管理层信心增强

### 实务操作补充

**数据处理技巧**：
1. 批量获取多家公司会计政策数据
2. 建立行业标准数据库
3. 自动化对比分析流程
4. 可视化呈现对比结果

**质量控制要点**：
1. 确保数据来源可靠（年报、审计报告）
2. 理解会计政策背后的业务逻辑
3. 避免机械对比，注重实质分析
4. 定期更新分析框架

2. 详细分析：分维度深度分析

4. 风险提示：潜在风险点提示
5. 改进建议：针对性改进建议

### 与其他分析方法的结合

**结合财务比率分析**：
会计政策选择会影响财务比率计算，分析时需要考虑政策差异的影响。

**结合现金流分析**：
会计政策影响利润但不影响现金流，通过对比可以识别政策影响。

**结合估值分析**：
不同会计政策下，需要调整估值模型参数以确保可比性。

### 高级分析技术

**蒙特卡洛模拟在会计估计中的应用**：
对于具有高度不确定性的会计估计（如减值测试），可以使用蒙特卡洛模拟来评估不同情景下的影响。

基本步骤：
1. 确定关键假设的概率分布
2. 设定模拟次数（通常10,000次以上）
3. 随机抽样生成情景
4. 计算各情景下的估计值
5. 分析结果分布

**敏感性分析的进阶应用**：
不仅分析单一假设变化的影响，还应分析：
- 多变量同时变化的组合影响
- 非线性关系的影响

**机器学习在会计估计预测中的应用**：

- 最佳估计值

import numpy as np
import pandas as pd

def sensitivity_analysis(base_value, variables, ranges):
    """
    多变量敏感性分析
    """
    results = []
    for var, range_vals in zip(variables, ranges):
        for val in range_vals:
            # 计算新估计值
            new_value = calculate_estimate(base_value, var, val)
            results.append({
                'variable': var,
                'value': val,
                'estimate': new_value,
                'change_pct': (new_value - base_value) / base_value
            })
    return pd.DataFrame(results)
