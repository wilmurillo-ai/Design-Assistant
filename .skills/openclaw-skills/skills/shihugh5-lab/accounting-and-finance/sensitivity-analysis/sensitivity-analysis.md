# 敏感性分析：评估估值的不确定性与风险

## 📚 一、理论背景与核心概念

### 1.1 敏感性分析的意义

**为什么需要敏感性分析**：
> 估值是基于一系列假设的预测，而这些假设都存在不确定性。敏感性分析帮助我们了解：哪些假设对估值影响最大？估值结果的稳健性如何？在什么情况下投资逻辑会失效？

**敏感性分析的价值**：
| 价值 | 说明 |
|:---|:---|
| **识别关键假设** | 找出对估值影响最大的变量 |
| **评估稳健性** | 判断估值结果对假设变化的敏感程度 |
| **确定区间** | 给出估值范围而非单点估计 |
| **风险管理** | 识别可能导致投资亏损的情景 |
| **沟通工具** | 向决策者展示不确定性 |

### 1.2 敏感性分析的类型

**单因素敏感性分析**：
```
一次只改变一个变量，其他保持不变
→ 识别各变量独立影响
→ 龙卷风图展示
```

**多因素敏感性分析**：
```
同时改变多个变量
→ 评估组合影响
→ 情景分析
```

**情景分析**：
```
设定不同情景（乐观/基准/悲观）
→ 评估极端情况
→ 压力测试
```

### 步骤2：建立估值模型

**基础估值模型**：

```python
def DCF估值(参数):
    """
    DCF估值计算
    """
    # 预测期现金流现值
    预测期现值 = 0
    for 年 in range(1, 6):
        FCF = 计算FCF(参数, 年)
        现值 = FCF / (1 + 参数['WACC'])**年
        预测期现值 += 现值
    
    # 终值
    终值FCF = 计算FCF(参数, 5) * (1 + 参数['永续增长率'])
    终值 = 终值FCF / (参数['WACC'] - 参数['永续增长率'])
    终值现值 = 终值 / (1 + 参数['WACC'])**5
    
    # 企业价值
    企业价值 = 预测期现值 + 终值现值
    
    # 股权价值
    股权价值 = 企业价值 - 参数['净负债']
    
    return {
        '企业价值': 企业价值,
        '股权价值': 股权价值,
        '每股价值': 股权价值 / 参数['股本']
    }
```

### 步骤3：单因素敏感性分析

**逐一变动变量**：

```python
def 单因素敏感性分析(基础参数, 变量列表, 变动幅度=0.1):
    """
    单因素敏感性分析
    """
    基础估值 = DCF估值(基础参数)
    基础每股 = 基础估值['每股价值']
    
    结果 = []
    
    for 变量名 in 变量列表:
        for 方向 in [-1, 1]:  # -10% 和 +10%
            # 复制参数
            调整参数 = 基础参数.copy()
            
            # 调整变量
            调整值 = 基础参数[变量名] * (1 + 方向 * 变动幅度)
            调整参数[变量名] = 调整值
            
            # 重新估值
            调整估值 = DCF估值(调整参数)
            调整每股 = 调整估值['每股价值']
            
            # 计算敏感度
            价值变动率 = (调整每股 - 基础每股) / 基础每股
            敏感度 = 价值变动率 / (方向 * 变动幅度)
            
            结果.append({
                '变量': 变量名,
                '方向': f'{方向*10:+d}%',
                '每股价值': 调整每股,
                '变动率': f'{价值变动率:.1%}',
                '敏感度': abs(敏感度)
            }
)
    
    return pd.DataFrame(结果)
```

### 步骤4：生成龙卷风图

**可视化敏感度**：

```python
def 生成龙卷风图(敏感性结果, 基础价值):
    """
    生成龙卷风图数据
    """
    # 按敏感度排序
    排序结果 = 敏感性结果.groupby('变量')['敏感度'].max().sort_values()
    
    图表数据 = []
    for 变量 in 排序结果.index:
        变量数据 = 敏感性结果[敏感性结果['变量'] == 变量]
        
        负面 = 变量数据[变量数据['方向'] == '-10%']['每股价值'].values[0]
        正面 = 变量数据[变量数据['方向'] == '+10%']['每股价值'].values[0]
        
        图表数据.append({
            '变量': 变量,
            '下限': min(负面, 正面),
            '上限': max(负面, 正面),
            '基准': 基础价值
        })
    
    return pd.DataFrame(图表数据)
```

def 情景分析估值(基础参数, 情景假设):

    多情景估值分析
    """
    情景结果 = {}
    
    for 情景名, 假设调整 in 情景假设.items():

## 🔢 三、公式与计算

### 3.1 敏感度系数

```
敏感度 = (价值变动率) / (变量变动率)

解释：
- 敏感度 = 1：价值与变量同比例变动
- 敏感度 > 1：价值变动幅度大于变量（敏感）
- 敏感度 < 1：价值变动幅度小于变量（不敏感）
- 敏感度 = 0：价值不受变量影响

示例：
收入增长率变动+10%，估值变动+20%
敏感度 = 20% / 10% = 2
```

### 3.2 估值区间计算

```
估值区间 = [悲观估值, 乐观估值]

或加权平均：
预期估值 = Σ(情景估值 × 情景概率)

估值标准差 = √[Σ(概率 × (估值 - 预期)^2)]
```
