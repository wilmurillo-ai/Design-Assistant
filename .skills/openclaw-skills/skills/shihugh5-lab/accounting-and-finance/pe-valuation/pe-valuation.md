# 市盈率(PE)估值方法

## 一、理论背景与核心概念

### 1.1 PE的本质与意义

**什么是市盈率？**
> 市盈率(Price-to-Earnings Ratio, PE)是最常用的相对估值指标，反映投资者为每1元净利润支付的价格倍数。

**PE的经济含义**：
```
PE = 股价 / 每股收益(EPS)
   = 市值 / 净利润

解释：
- PE = 10倍：投资者支付10元获得1元盈利
- PE = 20倍：投资者支付20元获得1元盈利
- 倒数(1/PE) = 盈利收益率（假设全分红）
```

**PE的两种视角**：

| 视角 | 理解 | 应用 |
|:---|:---|:---|
| **投资回收期** | PE=10意味着10年回收投资 | 长期价值投资 |
| **市场预期** | 高PE反映高增长预期 | 成长股筛选 |

**PE vs 盈利收益率**：
```
PE = 10倍 → 盈利收益率 = 10%
PE = 20倍 → 盈利收益率 = 5%
PE = 5倍 → 盈利收益率 = 20%

结论：PE越低，盈利收益率越高（假设利润全部分红）
```

### 1.2 PE的财务理论基础

**股利折现模型推导**：
```
根据Gordon增长模型：
P = D₁ / (r - g)

其中：
P = 股价
D₁ = 下期股利 = E₁ × b（b为分红率）
r = 要求回报率
g = 永续增长率

PE = P / E₁ = b / (r - g)
```

**关键洞察**：
PE由三个因素决定：
1. **分红率(b)**：分红率越高，PE越高
2. **要求回报率(r)**：风险越高，PE越低
3. **增长率(g)**：增长越快，PE越高

**公式变形**：
```
PE = (1 - 留存率) / (r - g)
   = 分红率 / (r - g)

或：
PE = 1 / (r - g) × 分红率
```

### 1.3 PE的生命周期特征

**不同阶段的典型PE水平**：

| 企业阶段 | 典型PE | 原因 |
|:---|:---:|:---|
| **初创期** | 不适用 | 盈利为负或极低 |
| **高速成长期** | 30-50倍 | 高增长预期 |
| **稳定成长期** | 20-30倍 | 增长放缓但仍快 |
| **成熟期** | 10-20倍 | 增长稳定 |
| **衰退期** | 5-10倍 | 增长停滞或下滑 |

**万华化学PE判断**：
- MDI业务：成熟期，合理PE 12-15倍
- 新材料业务：成长期，合理PE 20-25倍
- 综合PE：15-18倍（考虑业务结构）


> "市盈率估值相对简单，但要注意盈利的质量和可持续性。对于周期性行业，应使用正常化盈利计算PE，避免在周期高点买入。"

1. **盈利质量**：PE分母是净利润，需确保盈利真实可持续
2. **周期性调整**：周期股应使用平均盈利而非当期盈利
3. **同业对比**：PE估值核心是可比公司选择
4. **成长匹配**：高PE需匹配高增长，否则就是高估

## 二、详细操作步骤

### 步骤1：确定适用性

**1.1 PE估值的适用条件**：

**适用情况**（检查清单）：
- [x] 企业盈利稳定（非大幅波动）
- [x] 净利润为正值
- [x] 有可比上市公司
- [x] 会计政策可比

**不适用情况**：
| 情况 | 原因 | 替代方法 |
|:---|:---|:---|
| 亏损企业 | PE为负，无意义 | PS估值、PB估值 |
| 盈利波动极大 | PE不稳定 | 周期调整PE、PB估值 |
| 重资产企业 | 盈利受折旧影响大 | PB估值、EV/EBITDA |
| 金融行业 | 杠杆高，盈利特殊 | PB估值 |

**万华化学适用性**：✅ 适用
- 盈利稳定且为正
- 有可比化工公司
- 周期性但可调整

### 步骤2：选择PE类型

**2.1 三种PE类型对比**：

| 类型 | 计算公式 | 优点 | 缺点 | 适用场景 |
|:---|:---|:---|:---|:---|
| **静态PE** | 股价/上年EPS | 数据确定 | 滞后 | 年报发布后 |
| **动态PE** | 股价/本年预测EPS | 前瞻性 | 预测不确定 | 投资决策 |
| **滚动PE(TTM)** | 股价/过去4季度EPS | 最新数据 | 计算复杂 | 日常估值 |

**计算公式**：
```python
def 计算PE类型(股价, 历史EPS列表, 预测EPS):
    """
    计算三种PE
    """
    # 静态PE（最近年报）
    静态PE = 股价 / 历史EPS列表[-1]
    
    # 滚动PE(TTM)（最近4季度）
    TTM_EPS = sum(历史EPS列表[-4:]) if len(历史EPS列表) >= 4 else None
    滚动PE = 股价 / TTM_EPS if TTM_EPS else None
    
    # 动态PE（本年预测）
    动态PE = 股价 / 预测EPS if 预测EPS else None
    
    return {
        '静态PE': 静态PE,
        '滚动PE(TTM)': 滚动PE,
        '动态PE': 动态PE
    }

# 万华化学示例
股价 = 110
历史EPS = [2.5, 3.2, 4.5, 6.0, 7.8, 8.5, 9.0, 9.4]  # 最近8个季度
预测EPS = 10.5  # 本年预测

PE结果 = 计算PE类型(股价, 历史EPS, 预测EPS)
# 结果：静态PE=11.7倍(使用EPS=9.4)，滚动PE=11.7倍，动态PE=10.5倍
```

**2.2 选择建议**：
```python
def PE类型选择建议(当前时间, 年报发布时间):
    """
    根据时间选择最合适的PE类型
    """
    if 当前时间 < 年报发布时间 + 30天:
        # 刚发布年报，静态PE最新
        推荐 = '静态PE'
    elif 当前时间 > 年报发布后 + 9个月:
        # 接近下年年报，数据较老，用滚动PE或动态PE
        推荐 = '滚动PE(TTM)'
    else:
        # 中间时段，三者都可用
        推荐 = '动态PE（优先）或滚动PE'
    
    return 推荐
```

def 计算PE(股价, 总股本, 净利润):

    计算PE倍数
    """

    PE = 市值 / 净利润

        'PE': PE,

### 步骤4：选择可比公司

**4.1 可比公司选择标准**：

**定性标准**：
| 维度 | 选择标准 | 权重 |
|:---|:---|:---:|
| **行业** | 同一细分行业 | 30% |
| **商业模式** | 相似的收入来源 | 20% |
| **规模** | 收入规模相近 | 20% |
| **成长性** | 增长率相近 | 15% |
| **生命周期** | 处于相同阶段 | 15% |

**定量筛选**：
```python
def 筛选可比公司(目标公司, 候选公司列表, 阈值=0.3):
    """
    基于财务指标筛选可比公司
    """
    可比公司 = []
    
    for 公司 in 候选公司列表:
        相似度评分 = 0
        
        # 规模相似度（收入规模差异）
        规模差异 = abs(公司['收入'] - 目标公司['收入']) / 目标公司['收入']
        if 规模差异 < 0.5:
            相似度评分 += 0.3 * (1 - 规模差异)
        
        # 成长性相似度（增长率差异）
        增长差异 = abs(公司['增长率'] - 目标公司['增长率'])
        if 增长差异 < 0.15:
            相似度评分 += 0.3 * (1 - 增长差异/0.15)
        
        # 盈利能力相似度（ROE差异）
        ROE差异 = abs(公司['ROE'] - 目标公司['ROE'])
        if ROE差异 < 0.1:
            相似度评分 += 0.2 * (1 - ROE差异/0.1)
        
        # 财务杠杆相似度
        杠杆差异 = abs(公司['负债率'] - 目标公司['负债率'])
        if 杠杆差异 < 0.2:
            相似度评分 += 0.2 * (1 - 杠杆差异/0.2)
        
        if 相似度评分 >= 阈值:
            可比公司.append({
                '公司': 公司['名称'],
                '相似度': 相似度评分,
                'PE': 公司['PE']
            })
    
    # 按相似度排序
    可比公司.sort(key=lambda x: x['相似度'], reverse=True)
    
    return 可比公司[:5]  # 取前5家
```

**5.1 可比PE统计**：

def 可比PE统计(可比公司PE列表):

**5.2 确定合理PE区间**：
```python
def 确定合理PE区间(目标公司, 可比统计, 调整因子):
    """
    根据目标公司特征调整可比PE
    """
    基准PE = 可比统计['中位数']
    
    # 成长性调整
    if 目标公司['增长率'] > 可比统计['平均增长']:
        增长溢价 = min((目标公司['增长率'] - 可比统计['平均增长']) * 2, 3)
    else:
        增长溢价 = max((目标公司['增长率'] - 可比统计['平均增长']) * 2, -3)
    
    # 盈利能力调整
    if 目标公司['ROE'] > 可比统计['平均ROE']:
        盈利溢价 = min((目标公司['ROE'] - 可比统计['平均ROE']) * 10, 2)
    else:
        盈利溢价 = max((目标公司['ROE'] - 可比统计['平均ROE']) * 10, -2)
    
    # 调整后PE
    调整后PE = 基准PE + 增长溢价 + 盈利溢价
    
    # 合理区间
    下限 = max(调整后PE - 2, 可比统计['25分位'])
    上限 = min(调整后PE + 2, 可比统计['75分位'])
    
    return {
        '基准PE': 基准PE,
        '调整后PE': 调整后PE,
        '合理区间': (下限, 上限),
        '调整说明': f'成长性溢价{增长溢价:+.1f}，盈利溢价{盈利溢价:+.1f}'
    }

# 万华化学
万华 = {'增长率': 0.06, 'ROE': 0.26}
可比基准 = {'平均增长': 0.07, '平均ROE': 0.16}

结果 = 确定合理PE区间(万华, {**统计, **可比基准}, 1)
# 结果：基准PE=14.5，调整后PE=16.0，合理区间(14.5, 16.0)
```

### 步骤6：计算目标价值

**6.1 估值计算**：
```python
def PE估值计算(EPS, PE区间, 当前股价):
    """
    使用PE区间计算估值
    """
    下限估值 = EPS * PE区间[0]
    基准估值 = EPS * PE区间[1]
    上限估值 = EPS * PE区间[2] if len(PE区间) > 2 else None
    
    if 上限估值:
        平均估值 = (下限估值 + 基准估值 + 上限估值) / 3
    else:
        平均估值 = (下限估值 + 基准估值) / 2
    
    上涨空间 = (平均估值 - 当前股价) / 当前股价
    
    return {
        '保守估值': 下限估值,
        '基准估值': 基准估值,
        '乐观估值': 上限估值,
        '平均估值': 平均估值,
        '上涨空间': 上涨空间
    }

# 万华化学
估值结果 = PE估值计算(9.43, (14.5, 16.0, 18.0), 110)
# 结果：保守137元，基准151元，乐观170元，平均153元，上涨空间39%
```

### 步骤7：PEG交叉验证

**7.1 PEG计算**：
```python
def 计算PEG(PE, 盈利增长率):
    """
    计算PEG比率
    PEG = PE / 盈利增长率
    """
    if 盈利增长率 <= 0:
        return None, '增长率为负，PEG不适用'
    
    PEG = PE / (盈利增长率 * 100)  # 增长率用百分比
    
    if PEG < 0.8:
        评价 = '显著低估'
    elif PEG < 1.0:
        评价 = '低估'
    elif PEG < 1.2:
        评价 = '合理'
    elif PEG < 1.5:
        评价 = '略高估'
    else:
        评价 = '高估'
    
    return PEG, 评价

# 万华化学
PEG, 评价 = 计算PEG(11.7, 0.06)  # PE=11.7，增长6%
# 结果：PEG=1.95，评价=高估（但考虑ROE优势，可接受）
```

**注意**：PEG对低增长企业惩罚较重，需结合ROE综合判断。

## 三、公式与计算逻辑

### 3.1 PE驱动因素公式

**从Gordon模型推导**：
```
PE = 分红率 / (r - g)

变形：
PE = (1 - b) / (r - g)
其中b为留存率

或：
PE = 1 / (r - g) × 分红率
```

**PE的四大驱动因素**：
| 因素 | 与PE关系 | 万华化学情况 |
|:---|:---:|:---|
| 增长率(g) | 正相关 | 6%，适中 |
| 要求回报率(r) | 负相关 | 9%，适中 |
| 分红率 | 正相关 | 40%，一般 |
| 风险 | 负相关 | 中等 |

### 3.2 周期调整PE

**周期性行业PE陷阱**：
```
周期高点：盈利高 → PE低 → 看似便宜（实际贵）
周期低点：盈利低 → PE高 → 看似贵（实际便宜）

解决方案：使用周期调整PE
```

**周期调整PE公式**：
```
周期调整PE = 股价 / 正常化EPS

正常化EPS = 过去5-10年平均EPS
          或：周期中点EPS
```

**万华化学周期调整**：
```
历史EPS：4.5, 6.0, 7.8, 9.0, 9.4
正常化EPS = 7.3元
周期调整PE = 110 / 7.3 = 15.1倍

vs 当前PE = 11.7倍

结论：当前PE低估约25%（受周期高点影响）
```

### 3.3 PE Band（PE区间带）

**历史PE区间分析**：
```python
def 计算PEBand(历史股价, 历史EPS, 未来预测EPS):
    """
    计算PE区间带
    """
    历史PE = [股价 / EPS for 股价, EPS in zip(历史股价, 历史EPS)]
    
    PE统计 = {
        '最低PE': min(历史PE),
        '最高PE': max(历史PE),
        '平均PE': np.mean(历史PE),
        '25分位': np.percentile(历史PE, 25),
        '75分位': np.percentile(历史PE, 75)
    }
    
    # 未来股价区间
    目标区间 = {
        '悲观': 未来预测EPS * PE统计['25分位'],
        '基准': 未来预测EPS * PE统计['平均PE'],
        '乐观': 未来预测EPS * PE统计['75分位']
    }
    
    return PE统计, 目标区间
```

| 泰和新材 | 18.2 | 14% | 12% | 前20% |
| **平均值** | **14.2** | **16%** | **7%** | - |
| **中位数** | **14.5** | - | - | - |

### 4.4 合理PE确定

**调整因素**：
| 因素 | 万华vs可比 | 调整 |
|:---|:---|---:|
| ROE | 26% vs 16% | +1.0倍 |
| 增长率 | 6% vs 7% | -0.2倍 |
| 负债率 | 43% vs 50% | +0.3倍 |
| 龙头溢价 | 行业第一 | +0.5倍 |
| **合计调整** | - | **+1.6倍** |

**合理PE**：14.5 + 1.6 = **16.1倍**

**合理区间**：14-18倍

### 4.5 估值结果

**目标价计算**：
| 情景 | PE | 目标价 | 上涨空间 |
|:---|:---:|:---:|:---:|
| 悲观 | 14倍 | 132元 | 20% |
| 基准 | 16倍 | 151元 | 37% |
| 乐观 | 18倍 | 170元 | 55% |
| **平均** | - | **151元** | **37%** |

### 4.6 PEG验证

**PEG计算**：
```
PE = 11.7倍
增长率 = 6%
PEG = 11.7 / 6 = 1.95

评价：PEG > 1.5，按传统标准偏高
```

**但考虑ROE调整**：
```
ROE = 26%，远高于行业平均16%
调整PEG = PE / (增长率 + ROE/10)
         = 11.7 / (6 + 2.6)
         = 11.7 / 8.6
         = 1.36

调整后评价：合理
```

|:---|:---|

| 目标价 | 151元（37%上涨空间） |
| 评级 | 买入 |
| 风险 | 周期下行、盈利下滑 |

## 五、常见问题与注意事项

### 5.1 亏损企业能用PE吗？

**Q：亏损企业的PE如何计算？**

**A**：亏损企业PE为负数，无意义，应使用：
- **PS估值**：市销率
- **PB估值**：市净率
- **EV/EBITDA**：企业价值倍数

### 5.2 如何处理周期性行业的PE？

**Q：化工、钢铁等周期股PE波动大怎么办？**

**A**：
1. **使用正常化盈利**：5-10年平均盈利
2. **周期调整PE**：PE Band方法
3. **结合PB估值**：资产为基础
4. **多情景分析**：高点/中点/低点

### 5.3 高PE一定高估吗？

**Q：PE 50倍一定是高估吗？**

**A**：不一定，要看：
- 增长率是否匹配（PEG）
- 行业特征（科技股普遍高PE）
- 成长阶段（初创期高PE正常）

### 5.4 PE估值的局限性

**主要局限**：
1. 盈利为负时失效
2. 盈利波动大时不稳定
3. 会计政策差异影响可比性
4. 无法反映资产价值
5. 忽略了资本结构差异

### A.6 PE估值快速检查清单

**估值前自查**：
- [ ] 盈利是否稳定？（非亏损/大幅波动）
- [ ] 是否找到合适可比公司？（至少3-5家）
- [ ] 是否考虑成长性差异？（PEG验证）
- [ ] 是否考虑周期性？（周期调整）
- [ ] 当前PE处于历史什么位置？（历史分位）

**估值后验证**：
- [ ] 目标价与当前价差异是否合理？（±50%内）
- [ ] 与DCF估值是否一致？（差异<30%）
- [ ] 同业对比是否支持？（分位合理）
- [ ] PEG是否合理？（1-2倍区间）

### A.7 常见估值误区提醒

**误区1：只看静态PE**
- ❌ "PE 10倍，便宜"
- ✅ 要看动态PE、PEG、成长性

**误区2：跨行业比较PE**
- ❌ "银行PE 5倍，科技PE 50倍，银行更便宜"
- ✅ 同行业比较才有意义

**误区3：忽视盈利质量**
- ❌ "PE低就是低估"
- ✅ 要分析盈利是否可持续

**误区4：周期股在高点买入**
- ❌ "周期股PE 5倍，最便宜"
- ✅ 周期高点PE低是陷阱，要用周期调整PE
