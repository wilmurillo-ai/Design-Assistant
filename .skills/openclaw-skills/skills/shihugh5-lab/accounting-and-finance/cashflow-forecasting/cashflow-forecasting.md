# 现金流预测：预见未来的资金流动

## 📚 一、理论背景与核心概念

### 1.1 为什么需要现金流预测

**现金流预测的战略价值**：
> 现金流是企业的血液。预测现金流就是预测企业的生命线，是财务规划、投资决策、融资安排的基础。没有现金流预测，就无法评估企业的未来价值。

**现金流预测的应用场景**：

| 应用场景 | 预测用途 | 预测期限 | 精度要求 |
|:---|:---|:---:|:---:|
| **DCF估值** | 计算企业内在价值 | 5-10年 | 中长期趋势准确 |
| **融资规划** | 确定融资规模和时间 | 1-3年 | 年度数据准确 |
| **偿债分析** | 评估还款能力 | 与债务期限匹配 | 关键时点准确 |
| **营运资本管理** | 优化资金使用 | 月度/季度 | 短期精确 |
| **投资评估** | 评估项目可行性 | 项目周期 | 项目期准确 |

**现金流预测 vs 利润预测**：

| 维度 | 现金流预测 | 利润预测 |
|:---|:---|:---|
| **关注点** | 资金收付 | 收入成本配比 |
| **时间基础** | 收付实现制 | 权责发生制 |
| **波动性** | 更大（投资、融资集中） | 相对平滑 |
| **可验证性** | 高（银行对账单） | 低（会计估计） |
| **对估值影响** | 直接（DCF） | 间接（通过现金流） |

### 1.2 现金流预测的方法论

**预测方法分类**：

```
现金流预测方法
├── 直接法
│   ├── 销售百分比法
│   ├── 明细预测法
│   └── 项目现金流法
├── 间接法
│   ├── 利润调整法
│   └── 资产负债表推导法
└── 混合法
    └── 综合预测法（推荐）
```

**方法选择指南**：

| 方法 | 适用场景 | 优点 | 缺点 |
|:---|:---|:---|:---|
| **销售百分比法** | 稳定经营企业 | 简单快速 | 假设销售驱动一切 |
| **明细预测法** | 业务复杂企业 | 精确详细 | 数据要求高 |
| **利润调整法** | 有完整利润预测 | 利用现有数据 | 忽略营运资本变化 |
| **混合法** | 一般企业 | 平衡精确与效率 | 需要专业判断 |

### 1.3 现金流预测的核心驱动因素

**经营现金流驱动因素**：

```
经营现金流 = f(销售收入, 成本结构, 营运资本效率, 资本支出)

核心驱动：
1. 收入增长
   - 销量增长
   - 价格变化
   - 产品结构

2. 盈利能力
   - 毛利率
   - 费用率
   - 经营杠杆

3. 营运资本效率
   - 应收账款周转
   - 存货周转
   - 应付账款周转

4. 资本支出
   - 维持性投资
   - 扩张性投资
   - 折旧摊销
```

**预测假设的重要性**：
> 现金流预测的质量取决于假设的合理性。Garbage in, garbage out。关键假设需要经过 sensitivity analysis。

### 步骤2：建立预测模型

**销售百分比法模型**：

```python
def 销售百分比法预测(历史数据, 预测假设, 预测年限=5):
    """
    使用销售百分比法预测现金流
    
    参数：
        历史数据：历史财务数据
        预测假设：包含收入增长率、成本率等假设
        预测年限：预测年数
    
    返回：
        预测期现金流数据
    """
    预测结果 = []
    
    # 计算历史比率
    历史收入 = 历史数据['利润表历史']['收入']
    历史成本率 = [c/r for c, r in zip(历史数据['利润表历史']['成本'], 历史收入)]
    历史应收收入比 = [a/r for a, r in zip(历史数据['资产负债表历史']['应收账款'], 历史收入)]
    历史存货收入比 = [i/r for i, r in zip(历史数据['资产负债表历史']['存货'], 历史收入)]
    历史应付成本比 = [p/c for p, c in zip(历史数据['资产负债表历史']['应付账款'], 历史数据['利润表历史']['成本'])]
    
    # 使用平均比率作为预测基础
    平均成本率 = sum(历史成本率) / len(历史成本率)
    平均应收收入比 = sum(历史应收收入比) / len(历史应收收入比)
    平均存货收入比 = sum(历史存货收入比) / len(历史存货收入比)
    平均应付成本比 = sum(历史应付成本比) / len(历史应付成本比)
    
    # 起始值
    上期收入 = 历史收入[-1]
    上期应收 = 历史数据['资产负债表历史']['应收账款'][-1]
    上期存货 = 历史数据['资产负债表历史']['存货'][-1]
    上期应付 = 历史数据['资产负债表历史']['应付账款'][-1]
    
    for year in range(1, 预测年限 + 1):
        # 收入预测
        收入增长率 = 预测假设.get(f'收入增长率_{year}', 0.05)
        收入 = 上期收入 * (1 + 收入增长率)
        
        # 成本预测
        成本 = 收入 * 平均成本率
        
        # 营运资本预测
        应收 = 收入 * 平均应收收入比
        存货 = 收入 * 平均存货收入比
        应付 = 成本 * 平均应付成本比
        
        # 营运资本变动
        应收增加 = 应收 - 上期应收
        存货增加 = 存货 - 上期存货
        应付增加 = 应付 - 上期应付
        营运资本增加 = 应收增加 + 存货增加 - 应付增加
        
        # 经营现金流预测
        净利润 = 收入 - 成本  # 简化，实际需扣除费用和税
        折旧 = 预测假设.get(f'折旧_{year}', 0)
        经营现金流 = 净利润 + 折旧 - 营运资本增加
        
        # 资本支出预测
        资本支出 = 预测假设.get(f'资本支出_{year}', 0)
        
        # 自由现金流
        自由现金流 = 经营现金流 - 资本支出
        
        预测结果.append({
            '年份': year,
            '收入': 收入,
            '成本': 成本,
            '净利润': 净利润,
            '折旧': 折旧,
            '营运资本增加': 营运资本增加,
            '经营现金流': 经营现金流,
            '资本支出': 资本支出,
            '自由现金流': 自由现金流
        })
        
        # 更新上期值
        上期收入 = 收入
        上期应收 = 应收
        上期存货 = 存货
        上期应付 = 应付
    
    return pd.DataFrame(预测结果)
```

### 步骤4：敏感性分析

**关键假设敏感性测试**：

```python
def 现金流敏感性分析(基础预测, 关键假设, 变动范围):
    """
    对关键假设进行敏感性分析
    
    参数：
        基础预测：基准情景预测结果
        关键假设：需要测试的关键假设列表
        变动范围：假设变动范围（如±10%）
    
    返回：
        各假设变动对FCF的影响
    """
    敏感性结果 = {}
    
    for 假设 in 关键假设:
        敏感性结果[假设] = {}
        
        for 变动百分比 in 变动范围:
            # 调整假设
            调整后的假设 = 基础预测['假设'].copy()
            调整后的假设[假设] = 调整后的假设[假设] * (1 + 变动百分比)
            
            # 重新预测
            调整后预测 = 重新预测(调整后的假设)
            
            # 计算FCF变化
            基础FCF = sum(基础预测['自由现金流'])
            调整后FCF = sum(调整后预测['自由现金流'])
            FCF变动率 = (调整后FCF - 基础FCF) / 基础FCF
            
            敏感性结果[假设][变动百分比] = {
                'FCF变动率': FCF变动率,
                'FCF变动额': 调整后FCF - 基础FCF
            }
    
    return 敏感性结果
```

## 🔢 三、核心公式与计算

### 3.1 经营现金流预测公式

```
经营现金流 = 净利润
          + 折旧摊销
          - 营运资本增加
          + 其他非现金调整

营运资本增加 = Δ应收账款 + Δ存货 + Δ预付账款
             - Δ应付账款 - Δ预收账款 - Δ应付职工薪酬

简化公式（销售百分比法）：
经营现金流 = 收入 × (1 - 成本率 - 费用率) × (1 - 税率)
          + 折旧
          - 收入 × 营运资本收入比
```

### 3.2 自由现金流预测公式

```
自由现金流(FCF) = 经营现金流 - 资本支出
                = EBIT × (1 - 税率) + 折旧摊销
                - 营运资本增加 - 资本支出

股东自由现金流(FCFE) = FCF - 利息 × (1 - 税率) + 净借债
```

### 3.3 预测准确性评估

```
预测误差 = |实际值 - 预测值| / 实际值

平均绝对百分比误差(MAPE) = 平均(预测误差)

判断标准：
- MAPE < 5%：预测优秀
- MAPE 5-10%：预测良好
- MAPE 10-20%：预测一般
- MAPE > 20%：预测较差
```

## ⚠️ 五、常见问题

### 5.1 如何处理不确定性？

**情景分析法**：
```
构建多情景预测：
- 乐观情景：市场超预期，增长20%
- 基准情景：市场如预期，增长10%
- 悲观情景：市场不及预期，增长5%

概率加权：
预期FCF = 0.2×乐观 + 0.6×基准 + 0.2×悲观
```

### 5.2 预测偏差怎么办？

**滚动预测机制**：
```
每季度更新预测：
- 实际 vs 预测对比
- 分析偏差原因
- 调整未来假设
- 更新全年预测
```

#### 5.2 DCF估值输入

```python
# DCF估值参数
预测期FCF = [122, 181, 245, 310, 340]  # 2024-2028
永续增长率 = 0.03  # 3%
WACC = 0.076  # 7.6%

# 预测期现值计算
预测期现值 = sum([fcf / (1+WACC)**(i+1) for i, fcf in enumerate(预测期FCF)])
# = 122/1.076 + 181/1.076² + 245/1.076³ + 310/1.076⁴ + 340/1.076⁵
# = 113 + 156 + 197 + 231 + 236
# = 933亿

# 终值计算
终值FCF = 340 * 1.03 = 350亿
终值 = 350 / (0.076 - 0.03) = 7,609亿
终值现值 = 7609 / (1.076)**5 = 5,280亿

# 企业价值
企业价值 = 933 + 5280 = 6,213亿
```
