# 保险股估值方法

## 一、理论背景与核心概念

### 1.1 保险业务的特殊性

**为什么保险需要特殊估值方法？**

**长期性特征**：
```
寿险保单期限：10-30年甚至终身
收入确认：保费收入需在保险期内逐步确认
负债确认：需计提长期责任准备金

vs 一般企业：
收入当期确认，无长期准备金负债
```

**负债驱动模式**：
```
保险公司经营逻辑：
1. 收取保费（形成负债-准备金）
2. 投资运作（获取投资收益）
3. 赔付给付（履行保险责任）
4. 利差损益（投资与负债成本差异）

盈利来源 = 承保利润 + 投资收益 - 准备金提转差
```

**精算假设复杂**：
| 假设类型 | 内容 | 影响 |
|:---|:---|:---|
| 死亡率/生存率 | 生命表假设 | 准备金计提 |
| 疾病发生率 | 重疾、医疗概率 | 健康险准备金 |
| 退保率 | 保单继续率 | 现金流预测 |
| 投资收益率 | 未来投资回报 | 贴现率 |
| 费用率 | 运营费用假设 | 成本预测 |

### 1.2 保险估值的核心方法

**方法一：内含价值法（Embedded Value）**
```
内含价值(EV) = 调整后净资产 + 有效业务价值(VIF)

调整后净资产：
= 净资产 + 资产减值准备 - 负债调整项

有效业务价值(VIF)：
= 未来保单现金流现值 - 资本成本
  = Σ(预期利润/(1+贴现率)^t)
```

**方法二：新业务价值法（VNB）**
```
新业务价值(VNB) = 当年新保单的未来利润现值

NBV margin = VNB / 首年保费收入

反映新业务的盈利能力和质量
```

**方法三：P/EV倍数法**
```
合理P/EV = 目标价值 / 内含价值

类比：P/B估值，但EV比B更反映保险业务价值
```

### 1.3 保险关键指标

**价值类指标**：
| 指标 | 公式 | 优秀标准 |
|:---|:---|:---:|
| 内含价值(EV) | 调整后净资产+VIF | 持续增长 |
| 新业务价值(VNB) | 新保单利润现值 | 正增长 |
| NBV margin | VNB/首年保费 | >30% |
| EV增长率 | ΔEV/期初EV | >10% |

**业务类指标**：
| 指标 | 公式 | 健康标准 |
|:---|:---|:---:|
| 偿付能力充足率 | 实际资本/最低资本 | >120% |
| 综合成本率 | (赔付+费用)/保费 | <100% |
| 投资收益率 | 投资收益/投资资产 | >4% |
| 保单继续率 | 续期保费/上年保费 | >85% |

## 二、详细操作步骤

### 步骤1：计算调整后净资产

**1.1 资产端调整**：
```python
def 计算调整后净资产(净资产, 资产减值, 递延税项, 其他调整):
    """
    计算保险内含价值中的调整后净资产
    """
    调整后净资产 = (净资产 
                    + 资产减值 
                    - 递延税项 
                    + 其他调整)
    
    return {
        '净资产': 净资产,
        '资产减值调整': 资产减值,
        '递延税调整': -递延税项,
        '其他调整': 其他调整,
        '调整后净资产': 调整后净资产
    }
```

### 步骤2：计算有效业务价值(VIF)

**2.1 现金流预测**：
```python
def 预测保单现金流(保单组合, 精算假设):
    """
    预测现有保单的未来现金流
    """
    现金流预测 = []
    
    for 年度 in range(1, 预测年限+1):
        保费收入 = 预测保费(保单组合, 年度, 精算假设)
        赔付支出 = 预测赔付(保单组合, 年度, 精算假设)
        费用支出 = 预测费用(保单组合, 年度, 精算假设)
        投资收益 = 预测投资收益(准备金, 年度, 精算假设)
        
        年度利润 = 保费收入 + 投资收益 - 赔付支出 - 费用支出
        现金流预测.append(年度利润)
    
    return 现金流预测
```

**2.2 贴现计算VIF**：
```python
def 计算有效业务价值(未来利润现金流, 贴现率, 资本成本率):
    """
    计算有效业务价值
    """
    # 利润现值
    利润现值 = sum([利润 / (1+贴现率)**(t+1) 
                   for t, 利润 in enumerate(未来利润现金流)])
    
    # 资本成本
    资本成本 = 计算资本成本(未来利润现金流, 资本成本率)
    
    VIF = 利润现值 - 资本成本
    
    return {
        '利润现值': 利润现值,
        '资本成本': 资本成本,
        '有效业务价值VIF': VIF
    }
```

### 步骤3：计算内含价值(EV)

**3.1 EV计算**：
```python
def 计算内含价值(调整后净资产, 有效业务价值):
    """
    计算内含价值EV
    """
    EV = 调整后净资产 + 有效业务价值
    
    return {
        '调整后净资产': 调整后净资产,
        '有效业务价值': 有效业务价值,
        '内含价值EV': EV,
        '每股EV': EV / 总股本
    }

# 示例（假设数据）
EV结果 = 计算内含价值(800, 600)  # ANW=800亿, VIF=600亿
# 结果：EV=1400亿
```

### 步骤4：计算新业务价值(VNB)

**4.1 VNB计算**：
```python
def 计算新业务价值(当年新保单, 精算假设, 贴现率):
    """
    计算当年新业务价值
    """
    未来利润 = 预测新业务利润(当年新保单, 精算假设)
    
    VNB = sum([利润 / (1+贴现率)**(t+1) 
               for t, 利润 in enumerate(未来利润)])
    
    首年保费 = sum([保单['首年保费'] for 保单 in 当年新保单])
    
    NBV_margin = VNB / 首年保费 if 首年保费 > 0 else 0
    
    return {
        '新业务价值VNB': VNB,
        '首年保费': 首年保费,
        'NBV_margin': NBV_margin
    }
```

### 步骤5：P/EV估值

**5.1 P/EV倍数法**：
```python
def PEV估值(内含价值, 目标PEV倍数, 每股净资产):
    """
    使用P/EV倍数估值
    """
    目标价值 = 内含价值 * 目标PEV倍数
    每股价值 = 目标价值 / 总股本
    
    当前PEV = 当前市值 / 内含价值
    
    return {
        '内含价值': 内含价值,
        '目标PEV': 目标PEV倍数,
        '当前PEV': 当前PEV,
        '目标价值': 目标价值,
        '每股价值': 每股价值
    }
```

## 三、公式与计算逻辑

### 3.1 内含价值公式

**基本公式**：
```
内含价值(EV) = 调整后净资产(ANW) + 有效业务价值(VIF)

调整后净资产 = 净资产 
             + 资产减值准备 
             - 递延所得税负债 
             + 其他调整项

有效业务价值 = Σ(未来年度利润/(1+贴现率)^t) - 资本成本
```

### 3.2 新业务价值公式

**VNB计算**：
```
新业务价值(VNB) = Σ(新保单未来利润/(1+贴现率)^t)

NBV margin = VNB / 首年保费收入

通常NBV margin > 30%为优秀
```

### 3.3 P/EV估值公式

**合理P/EV推导**：
```
合理P/EV = (ROEV - g) / (r - g)

其中：
ROEV = EV回报率（类似ROE）
g = EV增长率
r = 要求回报率

或简化：
合理P/EV = 1 + (ROEV - r) / (r - g)
```

```
假设：
- ROEV = 12%
- 永续增长率 = 3%
- 股权成本 = 10%

合理P/EV = (12% - 3%) / (10% - 3%)
          = 9% / 7%
          = 1.29倍

目标价 = 14元 × 1.29 = 18元
上涨空间 = (18 - 12) / 12 = 50%
```

**投资建议**：买入，目标价18元

## 五、常见问题与注意事项

### 5.1 精算假设对估值影响大吗？

**Q：精算假设变化对EV影响大吗？**

**A**：影响巨大，特别是：
- **贴现率**：每变动1%，EV变动10-15%
- **投资收益率**：假设下降0.5%，EV下降5-10%
- **死亡率**：生命表更新可能影响准备金

**投资建议**：关注精算假设的保守性

### 5.2 P/EV<1一定是低估吗？

**Q：P/EV小于1一定说明低估吗？**

**A**：不一定，可能原因：
- ✅ **真实低估**：市场悲观，EV被低估
- ⚠️ **精算乐观**：EV计算假设过于乐观
- ⚠️ **业务恶化**：新业务价值下降
- ❌ **资产质量问题**：投资资产减值风险

### 5.3 财险与寿险估值差异

**Q：财险和寿险估值方法相同吗？**

**A**：方法类似，但重点不同：

| 维度 | 寿险 | 财险 |
|:---|:---|:---|
| 核心指标 | EV、VNB | 综合成本率、ROE |
| 估值重点 | 长期价值 | 当期盈利 |
| 关键假设 | 死亡率、退保率 | 赔付率、费用率 |
| 周期特性 | 长期稳定 | 短期波动 |

**财险更简单**：可用PB+ROE框架，类似银行

### A.1 内含价值详细构成

**内含价值构成分解**：

| 成熟公司 | 50% | 50% | 业务均衡 |
| 老公司 | 40% | 60% | 存量业务大，VIF高 |

### A.3 P/EV估值详解

**P/EV与ROEV关系**：
```
P/EV = (ROEV - g) / (r - g)

假设：
- ROEV = 12%
- g = 3%
- r = 10%

P/EV = (12% - 3%) / (10% - 3%) = 1.29

若ROEV提升至15%：
P/EV = (15% - 3%) / (10% - 3%) = 1.71
```

**ROEV每提升1%，P/EV提升约0.14倍**

### A.7 常见问题速查

**Q1：EV和净资产有什么区别？**
A：EV=调整后净资产+有效业务价值，比净资产多包含存量保单未来价值。

**Q2：为什么中国保险P/EV<1？**
A：市场担忧代理人改革、投资收益率下行、房地产敞口等风险。

**Q3：财险和寿险哪个更好估值？**
A：财险更简单（类似一般企业），寿险复杂（需计算EV）。

**Q4：NBV margin多少算好？**
A：>30%优秀，20-30%良好，<20%一般。

### A.10 保险估值关键要点总结

**估值核心逻辑**：
1. 内含价值(EV)是保险股的账面价值
2. P/EV类似PB，反映市场对保险业务的估值
3. ROEV驱动P/EV，高ROEV享受高P/EV
4. NBV growth是估值的重要支撑

**投资建议**：
- P/EV < 0.8：可能低估（需验证精算假设）
- P/EV 0.8-1.2：合理区间
- P/EV > 1.5：可能高估（除非是高增长公司）

**风险提示**：
- 精算假设过于乐观
- 投资收益率下行
- 大规模退保
- 监管政策变化
