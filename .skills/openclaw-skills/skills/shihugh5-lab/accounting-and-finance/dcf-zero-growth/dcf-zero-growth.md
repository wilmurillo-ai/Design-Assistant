# DCF零增长模型估值

## 📚 一、理论背景与核心概念

### 1.1 零增长模型的适用场景

**假设前提**：
```
企业自由现金流(FCFF)保持不变，永续增长率为0
```

**适用企业类型**：
| 类型 | 特征 | 案例 |
|:---|:---|:---|
| **成熟公用事业** | 现金流稳定，受监管限制增长 | 电力、水务、燃气公司 |
| **REITs** | 租金收入稳定，大部分收益分红 | 商业地产信托 |
| **特许经营企业** | 特许权到期后无法增长 | 收费公路、机场 |
| **资产清算价值** | 估值区间下限参考 | 任何企业 |

### 1.2 零增长模型的理论基础

**戈登模型特例**：
```
当增长率 g = 0 时，戈登增长模型简化为：

企业价值 = FCFF / WACC
```

**经济含义**：
> 企业价值等于将永续不变现金流按资本成本折现的现值。

**隐含假设**：
1. 企业永续经营
2. 现金流完全稳定
3. 无增长、无衰退
4. 资本结构不变

### 1.3 零增长vs稳定增长

| 对比维度 | 零增长 | 稳定增长 |
|:---|:---|:---|
| 增长率 | 0% | 2%-4% |
| 再投资 | 仅维持性资本支出 | 维持+增长性投资 |
| 适用阶段 | 成熟期/衰退期 | 稳定期 |
| 估值结果 | 保守/下限 | 合理/中等 |

### 1.4 模型的局限性

**不适用场景**：
- 成长期企业（增长率>0）
- 周期性行业（现金流波动大）
- 转型期企业（业务结构变化）

**使用方式**：
- 作为估值区间的下限
- 与稳定增长模型对比
- 敏感性分析的基准

## 📝 二、详细操作步骤

### 步骤1：计算当前FCFF

**公式**：
```
FCFF = NOPAT + 折旧与摊销 - 资本支出 - 营运资本增加
```

**数据来源**：
- 来自`free-cashflow-calculation` Skill
- 或使用最近年度实际FCFF

### 步骤2：确定WACC

**公式**：
```
WACC = (E/V) × Ke + (D/V) × Kd × (1 - T)
```

**数据来源**：
- 来自`wacc-calculation` Skill
- 或使用行业平均WACC

### 步骤3：计算企业价值

**公式**：
```
企业价值 = FCFF / WACC
```

**注意**：这是永续不变现金流的现值

### 步骤4：计算股权价值

**公式**：
```
股权价值 = 企业价值 - 净金融负债
         = 企业价值 - (金融负债 - 金融资产)
```

### 步骤5：计算每股价值

**公式**：
```
每股价值 = 股权价值 / 总股本
```

## 🔢 三、公式与计算逻辑

### 3.1 完整公式推导

**永续年金公式**：
```
PV = C / r

其中：
PV = 现值（企业价值）
C = 每期现金流（FCFF）
r = 折现率（WACC）
```

**应用到DCF零增长**：
```
企业价值 = FCFF / WACC
```

### 3.2 从企业价值到股权价值

```
┌─────────────────────────────────────┐
│            企业价值                  │
│         = FCFF / WACC               │
└───────────────┬─────────────────────┘
                │
        减：净金融负债
                │
                ▼
┌─────────────────────────────────────┐
│           股权价值                   │
│    = 企业价值 - 净金融负债           │
└───────────────┬─────────────────────┘
                │
        除以：总股本
                │
                ▼
┌─────────────────────────────────────┐
│           每股价值                   │
└─────────────────────────────────────┘
```

### 3.3 敏感性分析公式

**WACC敏感性**：
```
当WACC变化Δr时：

价值变化率 ≈ -Δr / WACC

例如：WACC从10%降至9%（Δr = -1%）
价值变化 ≈ -(-1%) / 10% = +10%
```

**FCFF敏感性**：
```
价值变化率 = ΔFCFF / FCFF

线性关系：FCFF增加10%，价值增加10%
```

## 💡 四、实例演示：某电力公司

### 4.1 基础假设

**公司特征**：
- 区域性电力公司
- 受电价监管，增长受限
- 现金流稳定可预测

**基础数据**：
| 指标 | 数值 |
|:---|---:|
| 营业收入 | 100.0亿 |
| NOPAT | 15.0亿 |
| 折旧与摊销 | 8.0亿 |
| 资本支出 | 8.0亿 |
| 营运资本变动 | 0 |
| FCFF | 15.0亿 |
| WACC | 8.0% |
| 金融负债 | 50.0亿 |
| 金融资产 | 5.0亿 |
| 总股本 | 10.0亿股 |

### 4.2 估值计算

**步骤1：企业价值**
```
企业价值 = FCFF / WACC
        = 15.0 / 8.0%
        = 187.5亿
```

**步骤2：股权价值**
```
净金融负债 = 50.0 - 5.0 = 45.0亿

股权价值 = 187.5 - 45.0
        = 142.5亿
```

**步骤3：每股价值**
```
每股价值 = 142.5 / 10.0
        = 14.25元/股
```

### 4.3 估值分析

| 指标 | 数值 | 说明 |
|:---|---:|:---|
| 企业价值 | 187.5亿 | 12.5倍FCFF |
| 股权价值 | 142.5亿 | |
| 每股价值 | 14.25元 | |
| 隐含EV/EBITDA | 约8倍 | 公用事业合理水平 |

### 4.4 敏感性分析

| WACC | 企业价值 | 股权价值 | 每股价值 |
|:---:|:---:|:---:|:---:|
| 7.0% | 214.3 | 169.3 | 16.93 |
| 7.5% | 200.0 | 155.0 | 15.50 |
| **8.0%** | **187.5** | **142.5** | **14.25** |
| 8.5% | 176.5 | 131.5 | 13.15 |
| 9.0% | 166.7 | 121.7 | 12.17 |

**结论**：WACC每变化0.5%，估值变化约6-7%。

## ⚠️ 五、常见问题与注意事项

### 5.1 FCFF的确定

**方法1：使用最近年度FCFF**
- 简单直接
- 适用于极其稳定的企业

**方法2：使用正常化FCFF**
- 剔除一次性项目
- 使用多年平均

**方法3：使用预测FCFF**
- 基于未来预期
- 更前瞻性

### 5.2 WACC的选择

**当前WACC vs 目标WACC**：
- 当前资本结构可能不是最优
- 可使用目标资本结构计算WACC

**行业平均WACC**：
- 可比公司平均
- 更客观

### 5.3 模型的保守性

**零增长假设的保守性**：
- 即使企业有轻微增长，零增长估值也是下限
- 可用于设置安全边际

**适用作为**：
- 估值区间下限
- 清算价值参考
- 保守投资者买入价

### 5.4 与市净率的对比

**重置成本估值**：
```
如果企业资产可重置：
零增长价值 ≈ 资产重置成本
         ≈ 账面价值 × 适当溢价
```

**破净分析**：
- 如果股价 < 零增长价值：可能被低估
- 但需确认FCFF可持续
