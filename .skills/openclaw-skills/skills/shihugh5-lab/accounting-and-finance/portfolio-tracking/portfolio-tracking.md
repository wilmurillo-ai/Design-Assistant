# 投资组合跟踪与绩效分析

## 一、理论背景与核心概念

### 1.1 投资组合跟踪的重要性

**为什么需要跟踪投资组合？**
> 投资组合跟踪不仅是计算收益，更是评估投资决策的有效性、识别风险来源、优化资产配置的基础。通过系统的跟踪分析，投资者可以了解：收益来自哪里？风险暴露在哪？是否需要调整？

**跟踪分析的核心价值**：
| 价值 | 说明 |
|:---|:---|
| **收益归因** | 识别收益来源（选股、择时、行业配置）|
| **风险监控** | 监控组合风险敞口和集中度 |
| **绩效评估** | 对比基准，评估投资能力 |
| **优化决策** | 数据驱动的调仓依据 |
| **纪律执行** | 避免情绪化交易 |

### 1.2 投资组合收益计算

**时间加权收益率(TWR)**：
```
TWR = (1+R1) × (1+R2) × ... × (1+Rn) - 1

其中R为各期收益率，适用于有资金进出的组合
消除资金进出对收益的影响
```

**资金加权收益率(MWR/IRR)**：
```
MWR = 使现金流现值为0的折现率

考虑资金进出的时间和金额
反映投资者实际体验
```

**简单收益率**：
```
收益率 = (期末价值 - 期初价值 + 分红) / 期初价值

适用于无资金进出的静态组合
```

### 1.3 风险调整收益指标

**夏普比率(Sharpe Ratio)**：
```
夏普比率 = (组合收益率 - 无风险利率) / 组合波动率

衡量单位风险带来的超额收益
夏普比率 > 1 通常认为优秀
```

**索提诺比率(Sortino Ratio)**：
```
索提诺比率 = (组合收益率 - 无风险利率) / 下行波动率

只考虑下行风险，更符合投资者偏好
```

**最大回撤(Max Drawdown)**：
```
最大回撤 = (峰值 - 谷值) / 峰值

反映组合历史上最大的亏损幅度
衡量极端风险
```

**卡玛比率(Calmar Ratio)**：
```
卡玛比率 = 年化收益率 / 最大回撤

衡量收益与极端风险的平衡
```

### 步骤2：计算组合收益

**2.1 日收益率计算**：
```python
def 计算日收益率(当日市值, 昨日市值, 当日现金流):
    """
    计算日收益率（时间加权）
    """
    if 昨日市值 == 0:
        return 0
    
    # 调整现金流影响
    调整后市值 = 当日市值 - 当日现金流
    日收益率 = (调整后市值 - 昨日市值) / 昨日市值
    
    return 日收益率
```

**2.2 累计收益计算**：
```python
def 计算累计收益(日收益率列表):
    """
    计算累计收益率
    """
    累计收益 = 1
    for 收益率 in 日收益率列表:
        累计收益 *= (1 + 收益率)
    
    return 累计收益 - 1
```

### 步骤3：风险指标计算

**3.1 波动率计算**：
```python
import numpy as np

def 计算波动率(收益率列表, 年化=True):
    """
    计算收益率标准差（波动率）
    """
    日波动率 = np.std(收益率列表)
    
    if 年化:
        年化波动率 = 日波动率 * np.sqrt(252)  # 252个交易日
        return 年化波动率
    else:
        return 日波动率
```

**3.2 最大回撤计算**：
```python
def 计算最大回撤(净值序列):
    """
    计算最大回撤
    """
    峰值 = np.maximum.accumulate(净值序列)
    回撤 = (峰值 - 净值序列) / 峰值
    最大回撤 = np.max(回撤)
    
    return 最大回撤
```

**3.3 夏普比率计算**：
```python
def 计算夏普比率(年化收益率, 年化波动率, 无风险利率=0.03):
    """
    计算夏普比率
    """
    夏普比率 = (年化收益率 - 无风险利率) / 年化波动率
    
    return 夏普比率
```

### 步骤4：绩效归因分析

**4.1 Brinson归因模型**：
```python
def Brinson归因(组合收益, 基准收益, 组合权重, 基准权重, 个股收益):
    """
    Brinson绩效归因模型
    分解为：配置效应 + 选股效应 + 交互效应
    """
    # 配置效应（行业配置贡献）
    配置效应 = sum([(组合权重[i] - 基准权重[i]) × 基准收益[i]] 
                   for i in range(len(组合权重))])
    
    # 选股效应（个股选择贡献）
    选股效应 = sum([基准权重[i] × (个股收益[i] - 基准收益[i]] 
                   for i in range(len(个股收益))])
    
    # 交互效应
    交互效应 = sum([(组合权重[i] - 基准权重[i]) × (个股收益[i] - 基准收益[i]] 
                   for i in range(len(组合权重))])
    
    return {
        '配置效应': 配置效应,
        '选股效应': 选股效应,
        '交互效应': 交互效应,
        '超额收益': 组合收益 - sum(基准权重[i] * 基准收益[i] for i in range(len(基准收益)))
    }
```

### 步骤5：风险监控

**5.1 集中度分析**：
```python
def 集中度分析(持仓权重):
    """
    分析组合集中度
    """
    排序权重 = sorted(持仓权重, reverse=True)
    
    return {
        'CR1': 排序权重[0],  # 最大持仓占比
        'CR5': sum(排序权重[:5]),  # 前5大持仓占比
        'CR10': sum(排序权重[:10]),  # 前10大持仓占比
        '赫芬达尔指数': sum([w**2 for w in 持仓权重])  # 集中度指标
    }
```

**5.2 行业暴露分析**：
```python
def 行业暴露分析(持仓数据, 行业分类):
    """
    分析行业配置暴露
    """
    行业权重 = {}
    for 持仓 in 持仓数据:
        行业 = 行业分类[持仓['代码']]
        行业权重[行业] = 行业权重.get(行业, 0) + 持仓['权重']
    
    return 行业权重
```

## 三、公式与计算逻辑

### 3.1 收益率公式

**时间加权收益率**：
```
TWR = ∏(1 + Ri) - 1

其中Ri为各期收益率
消除资金进出影响
```

**年化收益率**：
```
年化收益 = (1 + 总收益)^(365/天数) - 1
或简化：日收益 × 252
```

### 3.2 风险指标公式

**标准差（波动率）**：
```
σ = √[Σ(Ri - R̄)² / (n-1)]

年化波动率 = 日波动率 × √252
```

**贝塔系数**：
```
β = Cov(Rp, Rm) / Var(Rm)

衡量组合相对市场的波动性
β > 1：比市场波动大
β < 1：比市场波动小
```

### 3.3 归因分析公式

**Brinson模型**：
```
超额收益 = 配置效应 + 选股效应 + 交互效应

配置效应 = Σ(Wp_i - Wb_i) × Rb_i
选股效应 = ΣWb_i × (Rp_i - Rb_i)
交互效应 = Σ(Wp_i - Wb_i) × (Rp_i - Rb_i)
```

## 四、实例演示：组合跟踪案例

### 4.1 假设投资组合

**组合构成**（2024年初）：
| 股票 | 权重 | 行业 | 成本价 | 当前价 | 收益率 |
|:---|:---:|:---|---:|---:|---:|
| 万华化学 | 40% | 化工 | 100 | 110 | +10% |
| 招商银行 | 30% | 银行 | 35 | 32 | -9% |
| 中国平安 | 20% | 保险 | 50 | 45 | -10% |
| 现金 | 10% | - | - | - | 0% |
| **合计** | **100%** | - | - | - | **+1.3%** |

**基准**：沪深300指数，同期收益 +5%

### 4.2 绩效分析

**收益分析**：
```
组合收益：+1.3%
基准收益：+5.0%
超额收益：-3.7%（跑输基准）

分析：
- 万华贡献：40% × 10% = +4.0%
- 招行贡献：30% × (-9%) = -2.7%
- 平安贡献：20% × (-10%) = -2.0%
- 现金贡献：10% × 0% = 0%
```

**风险分析**：
```
假设数据：
- 组合波动率：18%
- 基准波动率：15%
- 最大回撤：-12%
- 夏普比率：0.3（较低）

结论：组合风险高于基准，但收益更低
```

### 4.3 归因分析

**行业配置效应**：
```
组合：化工40%，金融50%，现金10%
基准：化工15%，金融25%，其他60%

配置效应 = (40%-15%) × 化工基准收益 + (50%-25%) × 金融基准收益
         = 25% × 8% + 25% × 0%
         = +2.0%

解释：超配化工（表现好）带来正贡献
```

**个股选择效应**：
```
选股效应 = 基准权重 × (个股收益 - 行业收益)
         = 15% × (10%-8%) + 25% × (-4.5%-0%) + ...
         = -1.5%

解释：个股选择能力一般，招行和平安跑输行业
```

**总结**：
- 配置效应：+2.0%（成功）
- 选股效应：-5.7%（失败）
- 总超额：-3.7%

### 4.4 优化建议

**诊断结果**：
1. **选股问题**：金融股选择不佳，招行、平安跑输行业
2. **集中度风险**：单一个股40%权重过高
3. **现金拖累**：10%现金在牛市中拖累收益

**优化建议**：
1. 降低个股集中度（单一个股<25%）
2. 金融股换为行业ETF或选择更好的个股
3. 减少现金比例（除非判断市场顶部）
4. 增加行业分散度

## 五、常见问题与注意事项

### 5.1 多长时间评估一次组合？

**Q：组合跟踪应该多久进行一次？**

**A**：不同频率不同目的：
| 频率 | 目的 | 关注点 |
|:---|:---|:---|
| 每日 | 风险监控 | 仓位、止损 |
| 每周 | 交易复盘 | 买卖点 |
| 每月 | 绩效评估 | 收益归因 |
| 每季 | 战略调整 | 资产配置 |
| 每年 | 年度总结 | 整体表现 |

### 5.2 如何选择比较基准？

**Q：应该选择什么基准进行比较？**

**A**：根据组合特点：
- **全市场组合**：沪深300或中证500
- **行业主题**：对应行业指数
- **风格偏好**：价值/成长风格指数
- **全球配置**：MSCI全球指数

### 5.3 夏普比率多高算好？

**Q：夏普比率多少算优秀？**

**A**：参考标准：
| 夏普比率 | 评价 |
|:---:|:---|
| < 0 | 不如无风险资产 |
| 0 - 0.5 | 较差 |
| 0.5 - 1.0 | 一般 |
| 1.0 - 2.0 | 良好 |
| > 2.0 | 优秀（罕见）|

### 5.4 组合跟踪的局限性

**主要局限**：
1. **历史不代表未来**：过去表现不预示未来
2. **数据质量依赖**：需要准确的数据
3. **归因模型局限**：Brinson模型有假设
4. **短期噪音**：短期波动可能误导
5. **过度交易**：频繁跟踪可能导致过度交易

### A.3 组合优化模型

**均值-方差优化**：
```python
import numpy as np
from scipy.optimize import minimize

def 组合优化(预期收益, 协方差矩阵, 目标收益):
    """
    马科维茨均值-方差优化
    """
    n = len(预期收益)
    
    # 目标函数：最小化方差
    def 目标函数(权重):
        return 权重.T @ 协方差矩阵 @ 权重
    
    # 约束条件
    约束 = [
        {'type': 'eq', 'fun': lambda w: np.sum(w) - 1},  # 权重和为1
        {'type': 'eq', 'fun': lambda w: w.T @ 预期收益 - 目标收益}  # 达到目标收益
    ]
    
    # 边界条件（允许做空）
    边界 = tuple((0, 1) for _ in range(n))
    
    # 优化
    初始权重 = np.array([1/n] * n)
    结果 = minimize(目标函数, 初始权重, method='SLSQP', bounds=边界, constraints=约束)
    
    return 结果.x
```

### A.4 风险预算模型

**风险平价策略**：
```
目标：各资产对组合风险贡献相等

风险贡献 = 权重 × 边际风险
边际风险 = Cov(资产, 组合) / 组合波动率

优化问题：
min Σ(风险贡献_i - 平均风险贡献)²
```

### A.5 绩效归因进阶

**多因子归因**：
```
收益 = α + β1 × 市场因子 + β2 × 价值因子 + β3 × 规模因子 + β4 × 动量因子 + ε

其中：
- α：超额收益（选股能力）
- β：因子暴露
- ε：残差
```

### A.6 常见错误与避免

**错误1：过度优化**
- 问题：根据历史数据过度拟合
- 解决：样本外测试，留出一部分数据验证

**错误2：忽视交易成本**
- 问题：频繁调仓导致成本侵蚀收益
- 解决：设置调仓阈值，批量调整

**错误3：追热点**
- 问题：根据短期表现调整配置
- 解决：坚持长期配置，定期再平衡

**错误4：集中度过高**
- 问题：单一个股/行业占比过高
- 解决：设置集中度上限，分散投资
