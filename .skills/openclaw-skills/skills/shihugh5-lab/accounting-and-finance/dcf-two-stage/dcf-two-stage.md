# DCF两阶段估值模型

## 一、理论背景与核心概念

### 1.1 为什么需要两阶段模型？

**单阶段模型的局限**：
单阶段DCF模型假设企业从当前开始就以固定的永续增长率增长，这存在明显缺陷：
- 不适合高速增长期企业（增长率>WACC时模型失效）
- 无法反映企业从高速增长向稳定增长的过渡过程
- 对近期现金流预测准确性要求高
- 无法区分不同发展阶段的风险特征

**两阶段模型的理论依据**：
根据企业生命周期理论，企业的发展通常经历：
```
初创期 → 成长期 → 成熟期 → 衰退期
  ↓        ↓        ↓        ↓
不适用    适用     适用     不适用
         两阶段    单阶段
```

**两阶段模型的优势**：
| 优势 | 说明 |
|:---|:---|
| **符合生命周期** | 第一阶段反映高增长，第二阶段反映稳定期 |
| **更准确** | 对近期现金流详细预测，远期简化处理 |
| **更灵活** | 可调整预测期长度适应不同企业 |
| **风险分层** | 不同阶段可采用不同折现率 |

### 1.2 理论基础

**现金流折现原理**：
企业价值等于未来所有自由现金流的现值之和：
```
企业价值 = Σ FCF_t / (1+r)^t
```

**两阶段模型的经济学基础**：
- 第一阶段：企业享有竞争优势，获得超额收益，高增长
- 第二阶段：竞争均衡，只获得资本成本回报，稳定增长

**适用条件**：
- 当前处于高速增长期（g > GDP增速）
- 预计10-15年后进入稳定增长期
- 行业具有生命周期特征（如科技、新能源）


> "对于成长型企业，应采用两阶段甚至三阶段DCF模型。第一阶段详细预测，第二阶段永续增长。预测期的选择要根据企业所处行业和竞争格局确定。"

1. 预测期要充分反映竞争优势期
2. 永续增长率不应超过GDP增速
3. 资本支出要与折旧摊销匹配
4. 营运资本变动要符合行业特征

## 二、详细操作步骤

### 步骤1：确定模型适用性

**检查清单**：
- [ ] 企业是否处于成长期？（收入增长率>10%）
- [ ] 增长是否有可持续性？（竞争优势可维持5-10年）
- [ ] 是否有明确的催化剂？（产能扩张、产品升级）
- [ ] 行业是否有周期性？（周期位置判断）

**不适用情况**：
- 初创企业（现金流为负且不稳定）
- 衰退期企业（收入持续下滑）
- 周期性行业顶部（高盈利不可持续）
- 并购重组中（业务结构将大变）

**2.2 增长率假设设定**：

**第一阶段增长率（g1）**：
```python
# 方法一：历史趋势外推
def 历史增长率外推(历史收入, 年数):
    CAGR = (历史收入[-1] / 历史收入[0]) ** (1/年数) - 1
    return CAGR

# 方法二：行业空间法
def 行业空间法(市场空间, 当前份额, 目标份额):
    增长空间 = 目标份额 / 当前份额
    年数 = 10  # 预测期
    所需增长率 = 增长空间 ** (1/年数) - 1
    return 所需增长率

# 方法三：产能规划法
def 产能规划法(现有产能, 新增产能, 产能爬坡期):
    达产后产能 = 现有产能 + 新增产能
    产能增长率 = (达产后产能 / 现有产能) ** (1/产能爬坡期) - 1
    return 产能增长率
```

**第二阶段增长率（g2）**：
```
确定原则：
1. 不应超过长期GDP增速（中国2-3%，全球1-2%）
2. 不应超过行业长期增速
3. 必须小于WACC（否则模型失效）

推荐值：

- 全球成熟企业：1.5-2.5%
```

```python
def 收入预测(当前收入, 增长率序列):
```

**3.2 利润率预测**：
```python
# NOPAT率预测
def NOPAT率预测(当前NOPAT率, 目标NOPAT率, 收敛年数):
    NOPAT率序列 = []
    for t in range(1, 收敛年数+1):
        # 线性收敛到目标水平
        NOPAT率 = 当前NOPAT率 + (目标NOPAT率 - 当前NOPAT率) * t / 收敛年数
        NOPAT率序列.append(NOPAT率)
    # 后续保持目标水平
    for t in range(收敛年数+1, 11):
        NOPAT率序列.append(目标NOPAT率)
    return NOPAT率序列

# 万华化学：当前16.9%，目标15%，5年收敛
NOPAT率序列 = NOPAT率预测(0.169, 0.15, 5)
```

**3.3 资本支出预测**：
```python
# 资本支出与收入增长匹配
def 资本支出预测(收入增长, 资本密集度):
    """
    资本密集度 = 资本支出 / 收入增长
    制造业通常：1.0-2.0（每1元收入增长需1-2元资本支出）
    """
    资本支出 = 收入增长 * 资本密集度
    return 资本支出

# 万华化学（资本密集型）
资本密集度 = 1.5
```

**3.4 营运资本变动预测**：
```python
# 营运资本变动 = 收入变动 × 营运资本占收入比
def 营运资本变动预测(收入序列, 营运资本占收入比):
    营运资本变动序列 = []
    for i in range(1, len(收入序列)):
        收入变动 = 收入序列[i] - 收入序列[i-1]
        营运资本变动 = 收入变动 * 营运资本占收入比
        营运资本变动序列.append(营运资本变动)
    return 营运资本变动序列

# 万华化学：营运资本占收入比约15%
营运资本占比 = 0.15
```

**3.5 FCF计算汇总**：
```python
def 计算FCF序列(收入序列, NOPAT率序列, 折旧摊销率, 资本密集度, 营运资本占比):
    FCF序列 = []
    for t in range(len(收入序列)):
        收入 = 收入序列[t]
        NOPAT = 收入 * NOPAT率序列[t]
        折旧摊销 = 收入 * 折旧摊销率
        
        if t == 0:
            资本支出 = 0  # 当前期不计算
            营运资本变动 = 0
        else:
            收入增长 = 收入 - 收入序列[t-1]
            资本支出 = 收入增长 * 资本密集度
            营运资本变动 = 收入增长 * 营运资本占比
        
        FCF = NOPAT + 折旧摊销 - 资本支出 - 营运资本变动
        FCF序列.append(FCF)
    
    return FCF序列
```

```python
def 第一阶段现值(FCF序列, WACC):
    现值序列 = []
    for t, FCF in enumerate(FCF序列, 1):
        折现因子 = 1 / (1 + WACC) ** t
        现值 = FCF * 折现因子
        现值序列.append(现值)
    
    第一阶段现值合计 = sum(现值序列)
    return 第一阶段现值合计, 现值序列
```

**5.2 终值计算**：
```python
def 计算终值(最后一年FCF, 永续增长率, WACC):
    """
    Gordon增长模型
    终值 = FCF_{n+1} / (WACC - g) = FCF_n × (1+g) / (WACC - g)
    """
    if 永续增长率 >= WACC:
        raise ValueError("永续增长率必须小于WACC")
    
    终值 = 最后一年FCF * (1 + 永续增长率) / (WACC - 永续增长率)
    return 终值

def 终值现值(终值, WACC, 预测期年数):
    终值现值 = 终值 / (1 + WACC) ** 预测期年数
    return 终值现值
```

## 三、公式与计算逻辑详解

### 3.1 核心公式推导

**两阶段DCF基本公式**：
```
企业价值 = 第一阶段价值 + 第二阶段价值

         n
        Σ   FCF_t / (1+WACC)^t  +  TV / (1+WACC)^n
        t=1

其中：
TV = FCF_n × (1+g) / (WACC - g)   [Gordon增长模型]
```

**公式推导过程**：

**第一步：价值可加性原理**
企业价值可以分解为预测期价值和终值两部分：
```
V = V_预测期 + V_终值
```

**第二步：预测期价值计算**
预测期内各年FCF分别折现：
```
         n
V_预测 = Σ   FCF_t / (1+WACC)^t
        t=1
```

**第三步：终值推导**
从第n+1年开始，FCF以固定增长率g永续增长：
```
TV = FCF_{n+1}/(1+WACC) + FCF_{n+2}/(1+WACC)^2 + ...
   = FCF_{n+1}/(1+WACC) × [1 + (1+g)/(1+WACC) + (1+g)^2/(1+WACC)^2 + ...]

令 r = (1+g)/(1+WACC)，当 g < WACC 时，r < 1

TV = FCF_{n+1}/(1+WACC) × 1/(1-r)
   = FCF_{n+1}/(1+WACC) × (1+WACC)/(WACC-g)
   = FCF_{n+1} / (WACC-g)
   = FCF_n × (1+g) / (WACC-g)
```

**第四步：终值现值计算**
将终值折现到当前：
```
V_终值 = TV / (1+WACC)^n
```

**最终公式**：
```
         n                                          FCF_n × (1+g)
V =     Σ   FCF_t / (1+WACC)^t  +  ─────────────────────────────────
        t=1                           (WACC-g) × (1+WACC)^n
```

### 3.2 关键参数确定方法

**3.2.1 预测期长度确定**：
```
预测期 = 竞争优势期 + 过渡期

竞争优势期：企业能维持超额收益的时间
- 技术壁垒：专利保护期
- 品牌优势：消费者认知建立期
- 规模效应：产能爬坡期
- 网络效应：用户增长期

过渡期：从高速增长到稳定增长的过渡
- 通常2-3年
- 增长率从g1逐步降至g2
```

**3.2.2 增长率预测方法**：

**历史增长率法**：
```
收入增长率 = (收入_t / 收入_{t-n})^(1/n) - 1

考虑因素：
- 历史增长是否可持续
- 行业周期位置
- 竞争格局变化
- 宏观经济环境
```

**行业空间法**：
```
目标市场份额 = min(当前份额×(1+增长率)^n, 合理市场份额上限)
所需增长率 = (目标份额/当前份额)^(1/n) - 1
```

**产能规划法**：
```
产能利用率 = 产量 / 产能
收入增长 = 产能增长 × 产能利用率变化 × 价格变化
```

**3.2.3 永续增长率确定**：
```
g_永续 ≤ min(GDP长期增速, 行业长期增速, WACC-1%)

推荐值：
- 中国：2.0-3.0%
- 美国：1.5-2.5%
- 新兴市场：3.0-4.0%
```

**3.2.4 WACC确定**：
```
WACC = E/V × Re + D/V × Rd × (1-T)

其中：
Re = Rf + β × (Rm - Rf)  [CAPM]
Rd = 债务成本（平均利率）
E/V, D/V = 目标资本结构
T = 有效税率
```

### 3.3 特殊情况处理

**3.3.1 FCF为负的情况**：
```
初创期/扩张期企业FCF可能为负

处理方法：
1. 延长预测期，直到FCF转正
2. 使用收入倍数法估值早期价值
3. 假设第n年后才开始折现
4. 改用其他估值方法（如风险投资法）
```

**3.3.2 周期性行业的处理**：
```
周期高点：盈利高，但不可持续
周期低点：盈利低，但可能反弹

处理方法：
1. 使用周期平均盈利
2. 正常化FCF = 平均利润率 × 当前收入
3. 预测期覆盖完整周期
4. 敏感性分析包含周期情景
```

**3.3.3 多业务公司的处理**：
```
企业价值 = Σ 各业务单元价值 + 协同效应价值

各业务单元：
- 分别预测FCF
- 分别确定WACC（风险不同）
- 分别计算价值
- 加总得到企业价值
```

## 五、常见问题与注意事项

### 5.1 两阶段模型与三阶段模型的选择

**Q：什么时候需要用三阶段模型？**

**A**：
| 情况 | 推荐模型 | 说明 |
|:---|:---:|:---|
| 高速成长→稳定增长 | 两阶段 | 大多数情况 |
| 高速成长→减速→稳定 | 三阶段 | 增长放缓明显 |
| 超高速增长（>30%） | 三阶段 | 增长不可持续 |
| 成熟企业 | 单阶段/两阶段 | 增长已稳定 |

**三阶段模型公式**：
```
企业价值 = 高增长期现值 + 过渡期现值 + 稳定期终值现值

第一阶段（1-5年）：g1 = 20%
第二阶段（6-10年）：g2 = 10%
第三阶段（永续）：g3 = 3%
```

### 5.2 永续增长率的设定陷阱

**Q：永续增长率设定过高会有什么后果？**

**A**：

**陷阱1：永续增长率 > WACC**
```
如果 g >= WACC，终值公式分母为负或零，模型失效

后果：
- 终值无限大或负数
- 估值失去意义

正确做法：
永续增长率必须严格小于WACC（至少差1-2%）
```

**陷阱2：永续增长率 > GDP增速**
```
假设永续增长率5%，GDP增速2%
→ 企业规模终将超过GDP
→ 逻辑上不可能

正确做法：
永续增长率 ≤ GDP长期增速
```

**陷阱3：永续增长率与行业脱节**
```
衰退行业设定3%永续增长
→ 与行业趋势矛盾

正确做法：
永续增长率参考行业长期增速
```

### 5.3 预测期长度的确定

**Q：预测期应该选5年还是10年？**

**A**：

| 因素 | 短预测期（5年） | 长预测期（10年） |
|:---|:---:|:---:|
| **适用企业** | 成熟期、低增长 | 成长期、高增长 |
| **优势** | 预测准确度高 | 充分反映增长期 |
| **劣势** | 终值占比高 | 预测难度大 |
| **终值占比** | 70-80% | 60-70% |
| **推荐** | 传统行业 | 科技、医药 |

**经验法则**：
- 终值占比不应超过80%（否则模型不稳定）
- 预测期应覆盖竞争优势期
- 预测期FCF应转正（初创企业除外）

### 5.4 如何处理FCF为负的情况

**Q：扩张期企业FCF为负，还能用DCF吗？**

**A**：

**情况1：短期FCF为负**
```
处理方法：
1. 延长预测期直到FCF转正
2. 前几年的负FCF正常折现
3. 终值只计算正FCF阶段
```

**情况2：长期FCF为负**
```
不适用DCF的情况：
- 商业模式未验证
- 盈利时间不确定
- 需要持续大额融资

替代方法：
- 风险投资法（VC Method）
- 收入倍数法
- 实物期权法
```

### 5.5 周期性行业的特殊处理

**Q：化工、钢铁等周期性行业如何用DCF？**

**A**：

**方法1：正常化盈利**
```
步骤：
1. 计算历史平均利润率（覆盖完整周期）
2. 用正常化利润率替代当前利润率
3. 基于正常化盈利预测FCF

万华化学示例：
当前NOPAT率 = 16.9%（周期高点）
历史平均NOPAT率 = 14.5%（正常化）
使用14.5%作为预测基础
```

**方法2：情景加权**
```
构建多情景DCF：
- 周期高点情景：估值200元
- 周期中点情景：估值150元
- 周期低点情景：估值100元

概率加权：
期望值 = 200×0.2 + 150×0.6 + 100×0.2 = 150元
```

### 5.6 多业务公司的估值

**Q：多元化公司如何估值？**

**A**：

**分部估值法（SOTP）**：
```
企业价值 = Σ 各业务价值 + 协同价值

步骤：
1. 将公司按业务线分拆
2. 各业务线独立预测FCF
3. 各业务线使用不同WACC（风险不同）
4. 分别计算各业务价值
5. 加总得到企业价值

万华化学示例：
- MDI业务：估值3000亿，WACC 7%
- 新材料业务：估值1500亿，WACC 9%
- 精细化工：估值1000亿，WACC 8%
- 企业价值 = 3000 + 1500 + 1000 = 5500亿
```

### 5.7 注意事项

**建模注意事项**：
1. **一致性原则**：预测假设要与业务战略一致
2. **可比性原则**：参数设定要与行业对比
3. **敏感性原则**：关键假设要做敏感性分析
4. **合理性原则**：估值结果要与常识对比

**常见错误**：
- ❌ 永续增长率等于预测期增长率
- ❌ WACC计算使用当前资本结构而非目标结构
- ❌ 不考虑营运资本变动
- ❌ 资本支出与折旧不匹配
- ❌ 忽略非经营性资产/负债
