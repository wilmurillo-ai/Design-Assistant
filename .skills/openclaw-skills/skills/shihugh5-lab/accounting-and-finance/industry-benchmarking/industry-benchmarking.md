# 行业基准对标：在行业中找准位置

## 📚 一、理论背景与核心概念

### 1.1 行业基准的意义

**为什么需要行业基准**：
> 脱离行业谈企业表现没有意义。行业基准提供了客观的参照系，帮助企业判断自身是领先、平均还是落后。同一指标，在不同行业可能有完全不同的评价标准。

**行业基准的核心作用**：

| 作用 | 说明 | 应用场景 |
|:---|:---|:---|
| **定位** | 企业在行业中的相对位置 | 知己知彼，明确竞争地位 |
| **对标** | 与行业最佳实践的差距 | 学习标杆，寻找改进空间 |
| **预警** | 偏离行业平均的风险信号 | 及时发现问题，防范风险 |
| **目标** | 设定合理的业绩目标 | 基于行业水平制定可行目标 |
| **估值** | 判断估值是否合理 | 与行业平均估值对比 |

**脱离行业基准的误判案例**：
```
案例1：毛利率判断
- 公司A毛利率30%
- 如果不看行业，可能觉得一般
- 但行业平均只有15%，公司A是优秀水平

案例2：周转率判断
- 公司B周转率0.8次
- 看起来不高
- 但重化工行业平均0.6次，公司B是良好水平

案例3：负债率判断
- 公司C负债率60%
- 看起来偏高
- 但房地产行业平均70%，公司C是保守水平
```

### 1.2 行业基准的类型

**按统计口径分类**：

| 类型 | 计算方法 | 适用场景 | 优缺点 |
|:---|:---|:---|:---|
| **行业平均** | 所有公司简单平均 | 整体情况了解 | 受极端值影响 |
| **行业中位数** | 排序后取中间值 | 排除极端值影响 | 更能代表典型 |
| **行业前25%** | 领先企业水平 | 设定高目标 | 标杆作用 |
| **行业前10%** | 顶尖企业水平 | 追求卓越 | 挑战性目标 |
| **行业后25%** | 落后企业水平 | 风险预警 | 底线参考 |

**按公司范围分类**：

```
行业基准范围选择

│   ├── 全部A股公司
```

**不同基准的适用性**：

| 分析目的 | 推荐基准 | 理由 |
|:---|:---|:---|
| 初步了解 | 行业平均 | 简单直观 |
| 精准定位 | 行业中位数 | 排除极端值 |
| 设定目标 | 行业前25% | 挑战先进 |
| 识别风险 | 行业后25% | 底线思维 |
| 投资决策 | 多基准综合 | 全面判断 |

**基准计算方法**：

```python
import statistics

def 计算行业基准(行业公司列表, 指标, 剔除异常值=True):
    """
    计算行业基准值
    
    参数：
        行业公司列表：包含各公司指标数据的列表
        指标：要计算的指标名称
        剔除异常值：是否剔除极端值
    
    返回：
        各类基准值字典
    """
    # 提取数据
    数据 = [公司[指标] for 公司 in 行业公司列表 if 指标 in 公司 and 公司[指标] is not None]
    
    if not 数据:
        return None
    
    # 剔除异常值（如有要求）
    if 剔除异常值 and len(数据) > 10:
        Q1 = statistics.quantiles(数据, n=4)[0]
        Q3 = statistics.quantiles(数据, n=4)[2]
        IQR = Q3 - Q1
        下限 = Q1 - 1.5 * IQR
        上限 = Q3 + 1.5 * IQR
        数据 = [x for x in 数据 if 下限 <= x <= 上限]
    
    # 计算各类基准
    基准 = {
        '平均值': statistics.mean(数据),
        '中位数': statistics.median(数据),
        '标准差': statistics.stdev(数据) if len(数据) > 1 else 0,
        '最小值': min(数据),
        '最大值': max(数据),
        '25分位': sorted(数据)[int(len(数据)*0.25)],
        '75分位': sorted(数据)[int(len(数据)*0.75)],
        '10分位': sorted(数据)[int(len(数据)*0.10)],
        '90分位': sorted(数据)[int(len(数据)*0.90)],
        '样本数': len(数据)
    }
    
    return 基准
```

### 1.4 行业基准的动态性

**行业基准随时间变化**：

| 周期阶段 | 基准特征 | 分析注意 |
|:---|:---|:---|
| **行业景气高点** | 利润率、ROE普遍偏高 | 避免以此为常态预期 |
| **行业景气低点** | 指标普遍偏低 | 可能蕴含投资机会 |
| **技术变革期** | 新旧模式指标分化 | 关注转型公司 |
| **政策调整期** | 合规成本变化 | 关注政策影响 |

**基准更新频率建议**：
- **季度更新**：高波动行业（如科技、周期股）
- **半年更新**：中等波动行业（如消费品）
- **年度更新**：稳定行业（如公用事业）

```python
def 获取行业数据(行业代码, 数据类型='财务指标', 时间范围='最近年度'):
    """
    获取行业数据
    
    参数：
        行业代码：申万或GICS行业代码

        时间范围：数据时间范围
    
    返回：
        行业公司数据列表
    """
    # 1. 从金融终端获取数据
    数据 = 从金融终端获取(行业代码, 数据类型, 时间范围)
    
    # 2. 数据清洗
    数据 = 清洗行业数据(数据)
    
    # 3. 异常值处理
    数据 = 处理异常值(数据)
    
    # 4. 补充计算
    数据 = 补充衍生指标(数据)
    
    return 数据

def 清洗行业数据(原始数据):
    """
    清洗行业数据
    """
    清洗后数据 = []
    
    for 公司 in 原始数据:
        # 剔除ST公司（如需）
        if 公司.get('是否ST', False):
            continue
        
        # 剔除数据不完整公司
        if 数据完整性检查(公司) < 0.8:
            continue
        
        # 剔除异常值（如净利润为负但ROE为正）
        if 逻辑一致性检查(公司) == False:
            continue
        
        清洗后数据.append(公司)
    
    return 清洗后数据
```

### 步骤3：企业定位分析

**四分位定位法**：

```python
def 企业行业定位(企业值, 行业基准):
    """
    确定企业在行业中的位置（四分位法）
    
    参数：
        企业值：企业的指标数值
        行业基准：行业基准数据字典
    
    返回：
        定位结果
    """
    p10 = 行业基准.get('10分位')
    p25 = 行业基准.get('25分位')
    p50 = 行业基准.get('中位数')
    p75 = 行业基准.get('75分位')
    p90 = 行业基准.get('90分位')
    
    # 判断是高优指标还是低优指标
    # 默认假设高优（如ROE、利润率）
    # 对于低优指标（如负债率、周转天数）需反向判断
    
    if 企业值 >= p90:
        定位 = '行业顶尖（前10%）'
        等级 = 'A+'
        分位 = 'Top 10%'
    elif 企业值 >= p75:
        定位 = '行业领先（前25%）'
        等级 = 'A'
        分位 = 'Top 25%'
    elif 企业值 >= p50:
        定位 = '行业中上（25%-50%）'
        等级 = 'B'
        分位 = '50%-75%'
    elif 企业值 >= p25:
        定位 = '行业中下（50%-75%）'
        等级 = 'C'
        分位 = '25%-50%'
    elif 企业值 >= p10:
        定位 = '行业落后（后25%）'
        等级 = 'D'
        分位 = '10%-25%'
    else:
        定位 = '行业垫底（后10%）'
        等级 = 'D-'
        分位 = 'Bottom 10%'
    
    # 计算百分位排名
    # 简化为线性插值
    if 企业值 >= p75:
        百分位 = 75 + (企业值 - p75) / (p90 - p75) * 15 if p90 != p75 else 75
    elif 企业值 >= p50:
        百分位 = 50 + (企业值 - p50) / (p75 - p50) * 25 if p75 != p50 else 50
    elif 企业值 >= p25:
        百分位 = 25 + (企业值 - p25) / (p50 - p25) * 25 if p50 != p25 else 25
    else:
        百分位 = (企业值 - p10) / (p25 - p10) * 25 if p25 != p10 else 0
    
    百分位 = max(0, min(100, 百分位))
    
    return {
        '定位': 定位,
        '等级': 等级,
        '分位': 分位,
        '百分位排名': f'前{100-百分位:.1f}%'
    }
```

### 步骤4：差距分析

**与行业标杆的差距分析**：

```python
def 差距分析(企业指标, 行业基准):
    """
    分析与行业标杆的差距
    """
    差距 = {}
    
    for 指标名, 企业值 in 企业指标.items():
        if 指标名 not in 行业基准:
            continue
        
        行业值 = 行业基准[指标名]['中位数']
        行业前25 = 行业基准[指标名]['75分位']
        行业前10 = 行业基准[指标名]['90分位']
        
        差距[指标名] = {
            '企业值': 企业值,
            '行业中位': 行业值,
            '行业前25': 行业前25,
            '行业前10': 行业前10,
            '与行业差距': 企业值 - 行业值,
            '差距比例': (企业值 - 行业值) / 行业值 if 行业值 != 0 else 0,
            '与领先差距': 企业值 - 行业前25,
            '追赶空间': (行业前25 - 企业值) / 行业前25 if 行业前25 != 0 else 0
        }
    
    return 差距
```

### 3.2 行业排名计算

```
行业百分位 = (低于该公司的家数 / 总家数) × 100%

排名等级：
- 前10%：行业顶尖 ⭐⭐⭐⭐⭐
- 10-25%：行业领先 ⭐⭐⭐⭐
- 25-50%：行业中上 ⭐⭐⭐
- 50-75%：行业中下 ⭐⭐
- 后25%：行业落后 ⭐

百分位越高，排名越靠前
```

### 3.3 差距分析公式

```
绝对差距 = 企业值 - 行业基准值
相对差距 = (企业值 - 行业基准值) / |行业基准值|
追赶空间 = (行业标杆值 - 企业值) / 行业标杆值

差距解释：
- 相对差距 > 20%：显著优于行业
- 相对差距 0-20%：略优于行业
- 相对差距 -20%-0：略低于行业
- 相对差距 < -20%：显著低于行业
```

## ⚠️ 五、常见问题与注意事项

### 5.1 行业边界模糊怎么办？

**处理方法**：
- 选择主要业务所属行业
- 参考公司自身行业分类
- 对比多个相关行业基准

### 5.2 行业样本量太小怎么办？

**解决方案**：
- 扩大至相关行业
- 纳入上下游公司
- 使用历史平均值

### 5.3 如何处理行业内的极端值？

**处理方法**：
- 使用中位数而非平均值
- 剔除ST公司
- 剔除明显异常数据

### 5.4 行业基准分析中的常见误区

**误区识别与纠正**：

| 误区 | 错误做法 | 正确做法 | 后果 |
|:---|:---|:---|:---|
| **静态看待** | 只用一期数据 | 观察趋势变化 | 误判周期位置 |
| **忽视规模** | 大小公司混在一起 | 分规模区间对比 | 规模效应干扰 |
| **指标单一** | 只看一个指标 | 多维度综合分析 | 以偏概全 |
| **忽视差异** | 直接比较不同模式 | 区分商业模式 | 错误结论 |
| **样本偏差** | 只选表现好的公司 | 客观全面选取 | 高估行业水平 |

**周期行业的特殊处理**：
```
周期行业（如化工、钢铁、航运）分析要点：

1. 使用多年平均数据
   - 覆盖完整周期（通常3-5年）
   - 避免单年极端值影响

2. 分周期阶段对比
   - 区分景气高点、低点、常态
   - 分析企业在各阶段表现

3. 关注成本曲线位置
   - 在行业中处于成本什么分位
   - 成本优势是否持续

4. 产能利用率对比
   - 与行业平均利用率比较
   - 判断供需关系
```

### 5.5 跨国行业对比的注意事项

**跨国对比调整因素**：

| 因素 | 影响 | 调整方法 |
|:---|:---|:---|
| **会计准则** | 中美欧准则差异 | 统一调整至同一口径 |
| **税制差异** | 税率影响净利润 | 使用税前指标对比 |
| **发展阶段** | 新兴市场vs成熟市场 | 考虑增长阶段差异 |
| **资本成本** | 不同市场WACC不同 | 使用相对指标 |
| **汇率波动** | 影响报表折算 | 使用本币或统一货币 |

**跨国对比示例**：
```
中国化工企业 vs 国际化工巨头：

对比前调整：
1. 统一至IFRS会计准则
2. 调整为相同税率（25%）
3. 统一货币（美元）
4. 调整至相同折旧政策

对比维度：
- 成本竞争力（吨成本对比）
- 技术水平（研发投入占比）
- 全球化程度（海外收入占比）
- ESG表现（环保投入占比）
```

### 5.6 行业基准在估值中的应用

**相对估值法应用**：
```
PE估值的行业调整：

企业合理PE = 行业平均PE × 企业质量系数

企业质量系数计算：
质量系数 = (企业ROE/行业ROE) × 0.4 + 
          (企业增速/行业增速) × 0.3 +
          (企业稳定性评分) × 0.3

示例：
- 行业平均PE：15倍
- 企业ROE是行业2倍
- 企业增速是行业1.5倍
- 稳定性评分1.2

质量系数 = 2×0.4 + 1.5×0.3 + 1.2×0.3 = 0.8+0.45+0.36 = 1.61
合理PE = 15 × 1.61 = 24.15倍
```

**PB-ROE行业定位**：
```
PB-ROE关系：
合理PB = ROE / (r - g)

行业对比应用：
- 如果企业ROE高于行业，但PB低于行业
→ 可能被低估

- 如果企业ROE低于行业，但PB高于行业
→ 可能被高估

示例：
行业平均ROE 10%，平均PB 1.5倍
万华ROE 34%，当前PB 3.5倍

按行业比例：34%/10% × 1.5 = 5.1倍
实际PB 3.5 < 5.1
→ 相对行业可能低估
```
