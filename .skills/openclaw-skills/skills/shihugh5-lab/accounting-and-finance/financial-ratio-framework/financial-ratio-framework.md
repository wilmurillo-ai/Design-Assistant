# 财务指标综合分析框架：四维评估体系

## 📚 一、理论背景与核心概念

### 1.1 为什么需要综合框架

**单一指标的局限性**：

| 单一指标 | 局限性 | 案例 |
|:---|:---|:---|
| ROE高 | 可能是高杠杆风险 | 某地产公司ROE 25%，负债率85% |
| 收入高增长 | 可能是并购堆砌 | 某公司收入增长50%，但主要为并购 |
| 利润率高 | 可能是费用资本化 | 某科技公司高利润，但研发资本化激进 |
| 现金流好 | 可能是压缩营运资本 | 某公司通过延长应付改善现金流 |

**综合框架的价值**：
> 单一指标只能反映企业财务状况的一个侧面，容易以偏概全。综合框架从盈利能力、运营效率、偿债能力、成长能力四个维度全面评估，形成立体化的财务画像，才能得出可靠结论。

**综合评估的核心思想**：
```
企业财务健康 = f(盈利能力, 运营效率, 偿债能力, 成长能力)

四维平衡：
- 盈利是目的
- 效率是手段
- 偿债是保障
- 成长是延续
```

### 1.2 四维分析框架的理论基础

**框架来源**：
该框架源于20世纪初的财务分析实践，经过近百年的发展完善，已成为财务分析的标准范式。最早由杜邦公司在1919年提出的杜邦分析体系发展而来，后经过多位学者和实践者的扩展。

**四维度逻辑关系**：
```
┌─────────────────────────────────────────────────────────┐
│                  财务分析四维框架                        │
├─────────────────────────────────────────────────────────┤
│                                                          │
│    盈利能力 ←──────────────→ 运营效率                   │
│   （赚不赚钱）              （转得快不快）               │
│        │                        │                       │
│        │         企业价值        │                       │
│        │            ↑            │                       │
│        └────────────┼────────────┘                       │
│                     │                                   │
│    偿债能力 ←───────┴───────→ 成长能力                   │
│   （扛不扛得住）            （有没有未来）               │
│                                                          │
│  逻辑关系：                                              │
│  - 盈利能力是核心目标                                    │
│  - 运营效率是实现盈利的手段                              │
│  - 偿债能力是持续经营的保障                              │
│  - 成长能力是价值增长的源泉                              │
│  - 四者相互制约、相互支撑                                │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

**四维度的平衡艺术**：

| 组合类型 | 特征 | 评价 |
|:---|:---|:---:|
| 高盈利+高效率+低杠杆+高成长 | 完美型 | 🟢 卓越 |
| 高盈利+高效率+低杠杆+低成长 | 现金牛型 | 🟢 稳健 |
| 高盈利+低效率+高杠杆+高成长 | 激进型 | 🟡 风险 |
| 低盈利+高效率+低杠杆+高成长 | 潜力型 | 🟡 观察 |
| 低盈利+低效率+高杠杆+低成长 | 危机型 | 🔴 危险 |

### 1.3 四维指标体系的设计原则

**指标选取原则**：

1. **代表性**：能代表该维度的核心特征
2. **可获取性**：数据容易从财报中获取
3. **可比性**：便于跨公司、跨期对比
4. **互补性**：各指标相互补充，不重复
5. **敏感性**：能反映财务状况的变化

**各维度核心指标**：

| 维度 | 核心指标 | 指标数量 | 重要性 |
|:---|:---|:---:|:---:|
| **盈利能力** | ROE、ROIC、毛利率、净利率 | 4 | ⭐⭐⭐⭐⭐ |
| **运营效率** | 周转率、现金周期、资产周转 | 3 | ⭐⭐⭐⭐ |
| **偿债能力** | 资产负债率、流动比率、利息保障 | 3 | ⭐⭐⭐⭐ |
| **成长能力** | 收入增长率、利润增长率、FCF增长 | 3 | ⭐⭐⭐ |

### 步骤2：计算各维度指标

**盈利能力指标计算**：

```python
def 计算盈利能力指标(利润表, 资产负债表, 重构数据=None):
    """
    计算盈利能力核心指标
    
    参数：
        利润表：标准化利润表数据
        资产负债表：标准化资产负债表数据
        重构数据：重构后的资产资本表（如有）
    
    返回：盈利能力指标字典
    """
    指标 = {}
    
    # 基础数据
    净利润 = 利润表['净利润']
    营业收入 = 利润表['营业收入']
    营业成本 = 利润表['营业成本']
    EBIT = 利润表.get('EBIT', 营业收入 - 营业成本 - 利润表.get('销售费用',0) - 利润表.get('管理费用',0) - 利润表.get('研发费用',0))
    
    # 平均股东权益
    期初权益 = 资产负债表.get('股东权益_期初', 资产负债表['股东权益'])
    期末权益 = 资产负债表.get('股东权益_期末', 资产负债表['股东权益'])
    平均权益 = (期初权益 + 期末权益) / 2
    
    # 1. ROE
    指标['ROE'] = 净利润 / 平均权益 if 平均权益 != 0 else 0
    
    # 2. ROIC（如有重构数据）
    if 重构数据:
        NOPAT = EBIT * (1 - 利润表.get('有效税率', 0.25))
        投入资本 = 重构数据['投入资本']
        指标['ROIC'] = NOPAT / 投入资本 if 投入资本 != 0 else 0
    else:
        # 简化计算
        指标['ROIC'] = EBIT * 0.75 / 平均权益 if 平均权益 != 0 else 0
    
    # 3. 毛利率
    毛利 = 营业收入 - 营业成本
    指标['毛利率'] = 毛利 / 营业收入 if 营业收入 != 0 else 0
    
    # 4. 净利率
    指标['净利率'] = 净利润 / 营业收入 if 营业收入 != 0 else 0
    
    # 5. EBIT利润率
    指标['EBIT利润率'] = EBIT / 营业收入 if 营业收入 != 0 else 0
    
    return 指标
```

**运营效率指标计算**：

```python
def 计算运营效率指标(利润表, 资产负债表):
    """
    计算运营效率核心指标
    """
    指标 = {}
    
    # 基础数据
    营业收入 = 利润表['营业收入']
    营业成本 = 利润表['营业成本']
    
    # 平均数计算
    平均总资产 = (资产负债表.get('总资产_期初',0) + 资产负债表.get('总资产_期末',0)) / 2
    平均存货 = (资产负债表.get('存货_期初',0) + 资产负债表.get('存货_期末',0)) / 2
    平均应收 = (资产负债表.get('应收账款_期初',0) + 资产负债表.get('应收账款_期末',0)) / 2
    平均应付 = (资产负债表.get('应付账款_期初',0) + 资产负债表.get('应付账款_期末',0)) / 2
    
    # 1. 总资产周转率
    指标['总资产周转率'] = 营业收入 / 平均总资产 if 平均总资产 != 0 else 0
    
    # 2. 存货周转天数
    指标['存货周转天数'] = 365 / (营业成本 / 平均存货) if 平均存货 != 0 and 营业成本 != 0 else 0
    
    # 3. 应收周转天数
    指标['应收周转天数'] = 365 / (营业收入 / 平均应收) if 平均应收 != 0 and 营业收入 != 0 else 0
    
    # 4. 应付周转天数
    指标['应付周转天数'] = 365 / (营业成本 / 平均应付) if 平均应付 != 0 and 营业成本 != 0 else 0
    
    # 5. 现金周期
    指标['现金周期'] = 指标['存货周转天数'] + 指标['应收周转天数'] - 指标['应付周转天数']
    
    return 指标
```

**偿债能力指标计算**：

```python
def 计算偿债能力指标(资产负债表, 利润表=None):
    """
    计算偿债能力核心指标
    """
    指标 = {}
    
    # 基础数据
    总资产 = 资产负债表['总资产']
    总负债 = 资产负债表['总负债']
    股东权益 = 资产负债表['股东权益']
    流动资产 = 资产负债表.get('流动资产', 0)
    流动负债 = 资产负债表.get('流动负债', 0)
    存货 = 资产负债表.get('存货', 0)
    
    # 1. 资产负债率
    指标['资产负债率'] = 总负债 / 总资产 if 总资产 != 0 else 0
    
    # 2. 权益乘数
    指标['权益乘数'] = 总资产 / 股东权益 if 股东权益 != 0 else 0
    
    # 3. 流动比率
    指标['流动比率'] = 流动资产 / 流动负债 if 流动负债 != 0 else 0
    
    # 4. 速动比率
    指标['速动比率'] = (流动资产 - 存货) / 流动负债 if 流动负债 != 0 else 0
    
    # 5. 利息保障倍数（如有利润表）
    if 利润表 and 'EBIT' in 利润表 and '利息费用' in 利润表:
        EBIT = 利润表['EBIT']
        利息费用 = 利润表['利息费用']
        指标['利息保障倍数'] = EBIT / 利息费用 if 利息费用 != 0 else float('inf')
    else:
        指标['利息保障倍数'] = None
    
    return 指标
```

**成长能力指标计算**：

```python
def 计算成长能力指标(本期数据, 上期数据):
    """
    计算成长能力核心指标
    
    参数：
        本期数据/上期数据：包含收入、利润、资产、FCF的字典
    """
    指标 = {}
    
    # 1. 收入增长率
    上期收入 = 上期数据.get('营业收入', 0)
    本期收入 = 本期数据.get('营业收入', 0)
    指标['收入增长率'] = (本期收入 - 上期收入) / 上期收入 if 上期收入 != 0 else 0
    
    # 2. 利润增长率
    上期利润 = 上期数据.get('净利润', 0)
    本期利润 = 本期数据.get('净利润', 0)
    指标['利润增长率'] = (本期利润 - 上期利润) / abs(上期利润) if 上期利润 != 0 else 0
    
    # 3. 资产增长率
    上期资产 = 上期数据.get('总资产', 0)
    本期资产 = 本期数据.get('总资产', 0)
    指标['资产增长率'] = (本期资产 - 上期资产) / 上期资产 if 上期资产 != 0 else 0
    
    # 4. FCF增长率（如有）
    上期FCF = 上期数据.get('FCF', 0)
    本期FCF = 本期数据.get('FCF', 0)
    if 上期FCF != 0:
        指标['FCF增长率'] = (本期FCF - 上期FCF) / abs(上期FCF)
    else:
        指标['FCF增长率'] = None
    
    return 指标
```

### 步骤3：四维度评分

**综合评分模型**：

```python
def 四维财务健康评分(指标, 行业基准):
    """
    四维财务健康度综合评分
    
    满分100分，各维度25分
    
    参数：
        指标：四维指标字典
        行业基准：行业平均水平字典
    
    返回：各维度得分和总分
    """
    评分 = {}
    
    # ===== 维度1：盈利能力（25分）=====
    盈利分 = 0
    
    # ROE评分（10分）
    ROE = 指标['盈利能力'].get('ROE', 0)
    if ROE >= 0.20:
        盈利分 += 10
    elif ROE >= 0.15:
        盈利分 += 7
    elif ROE >= 0.10:
        盈利分 += 5
    else:
        盈利分 += 2
    
    # ROIC评分（8分）
    ROIC = 指标['盈利能力'].get('ROIC', 0)
    if ROIC >= 0.15:
        盈利分 += 8
    elif ROIC >= 0.10:
        盈利分 += 5
    elif ROIC >= 0.08:
        盈利分 += 3
    else:
        盈利分 += 1
    
    # 毛利率评分（7分）
    毛利率 = 指标['盈利能力'].get('毛利率', 0)
    行业毛利率 = 行业基准.get('毛利率', 0.20)
    if 毛利率 > 行业毛利率 * 1.2:
        盈利分 += 7
    elif 毛利率 > 行业毛利率:
        盈利分 += 5
    elif 毛利率 > 行业毛利率 * 0.8:
        盈利分 += 3
    else:
        盈利分 += 1
    
    评分['盈利能力'] = min(盈利分, 25)
    
    # ===== 维度2：运营效率（25分）=====
    效率分 = 0
    
    # 总资产周转率（10分）
    周转率 = 指标['运营效率'].get('总资产周转率', 0)
    if 周转率 >= 1.2:
        效率分 += 10
    elif 周转率 >= 0.8:
        效率分 += 7
    elif 周转率 >= 0.5:
        效率分 += 4
    else:
        效率分 += 1
    
    # 现金周期（10分）
    现金周期 = 指标['运营效率'].get('现金周期', 0)
    if 现金周期 < 30:
        效率分 += 10
    elif 现金周期 < 60:
        效率分 += 7
    elif 现金周期 < 90:
        效率分 += 4
    else:
        效率分 += 1
    
    # 周转天数综合（5分）
    存货天数 = 指标['运营效率'].get('存货周转天数', 0)
    应收天数 = 指标['运营效率'].get('应收周转天数', 0)
    应付天数 = 指标['运营效率'].get('应付周转天数', 0)
    
    if 存货天数 < 60 and 应收天数 < 60:
        效率分 += 5
    elif 存货天数 < 90 and 应收天数 < 90:
        效率分 += 3
    else:
        效率分 += 1
    
    评分['运营效率'] = min(效率分, 25)
    
    # ===== 维度3：偿债能力（25分）=====
    偿债分 = 0
    
    # 资产负债率（10分）
    负债率 = 指标['偿债能力'].get('资产负债率', 0)
    if 0.40 <= 负债率 <= 0.50:
        偿债分 += 10  # 最优区间
    elif 0.30 <= 负债率 < 0.40 or 0.50 < 负债率 <= 0.60:
        偿债分 += 8  # 良好
    elif 0.60 < 负债率 <= 0.70:
        偿债分 += 5  # 一般
    else:
        偿债分 += 2  # 风险
    
    # 流动比率（8分）
    流动比率 = 指标['偿债能力'].get('流动比率', 0)
    if 流动比率 >= 2.0:
        偿债分 += 8
    elif 流动比率 >= 1.5:
        偿债分 += 6
    elif 流动比率 >= 1.0:
        偿债分 += 4
    else:
        偿债分 += 1
    
    # 利息保障（7分）
    利息保障 = 指标['偿债能力'].get('利息保障倍数', 0)
    if 利息保障 and 利息保障 >= 8:
        偿债分 += 7
    elif 利息保障 and 利息保障 >= 5:
        偿债分 += 5
    elif 利息保障 and 利息保障 >= 3:
        偿债分 += 3
    else:
        偿债分 += 1
    
    评分['偿债能力'] = min(偿债分, 25)
    
    # ===== 维度4：成长能力（25分）=====
    成长分 = 0
    
    # 收入增长率（10分）
    收入增速 = 指标['成长能力'].get('收入增长率', 0)
    if 收入增速 >= 0.20:
        成长分 += 10
    elif 收入增速 >= 0.15:
        成长分 += 8
    elif 收入增速 >= 0.10:
        成长分 += 6
    elif 收入增速 >= 0.05:
        成长分 += 4
    else:
        成长分 += 2
    
    # 利润增长率（8分）
    利润增速 = 指标['成长能力'].get('利润增长率', 0)
    if 利润增速 > 收入增速:  # 利润增长快于收入，质量高
        成长分 += 8
    elif 利润增速 > 0:
        成长分 += 5
    else:
        成长分 += 2
    
    # FCF增长率（7分）
    FCF增速 = 指标['成长能力'].get('FCF增长率', None)
    if FCF增速 and FCF增速 > 0:
        成长分 += 7
    elif FCF增速 and FCF增速 > -0.1:  # 小幅下降
        成长分 += 4
    else:
        成长分 += 2
    
    评分['成长能力'] = min(成长分, 25)
    
    # 总分
    评分['总分'] = sum([评分['盈利能力'], 评分['运营效率'], 
                      评分['偿债能力'], 评分['成长能力']])
    
    return 评分
```

## 🔢 三、核心指标体系与评价标准

### 3.1 盈利能力指标体系

| 指标 | 计算公式 | 优秀 | 良好 | 一般 | 较差 | 行业适用性 |
|:---|:---|:---:|:---:|:---:|:---:|:---|
| **ROE** | 净利润/平均权益 | >20% | 15-20% | 10-15% | <10% | 通用 |
| **ROIC** | NOPAT/投入资本 | >15% | 10-15% | 8-10% | <8% | 通用 |
| **毛利率** | 毛利/收入 | >行业+5% | 行业±5% | 行业-5%~-10% | <行业-10% | 行业差异大 |
| **净利率** | 净利润/收入 | >15% | 10-15% | 5-10% | <5% | 通用 |
| **EBIT利润率** | EBIT/收入 | >18% | 12-18% | 8-12% | <8% | 通用 |

### 3.2 运营效率指标体系

| 指标 | 计算公式 | 优秀 | 良好 | 一般 | 较差 | 行业差异 |
|:---|:---|:---:|:---:|:---:|:---:|:---|
| **总资产周转率** | 收入/平均总资产 | >1.2 | 0.8-1.2 | 0.5-0.8 | <0.5 | 重资产行业低 |
| **存货周转天数** | 365/(成本/平均存货) | <45天 | 45-60天 | 60-90天 | >90天 | 零售低、制造高 |
| **应收周转天数** | 365/(收入/平均应收) | <30天 | 30-60天 | 60-90天 | >90天 | B2B高、B2C低 |
| **现金周期** | 存货天数+应收天数-应付天数 | <30天 | 30-60天 | 60-90天 | >90天 | 零售可负、制造正 |

### 3.3 偿债能力指标体系

| 指标 | 计算公式 | 优秀 | 良好 | 一般 | 较差 | 适用场景 |
|:---|:---|:---:|:---:|:---:|:---:|:---|
| **资产负债率** | 负债/资产 | 40-50% | 30-40%或50-60% | 60-70% | >70%或<30% | 通用 |
| **权益乘数** | 资产/权益 | 1.5-2.0 | 2.0-2.5 | 2.5-3.0 | >3.0 | 通用 |
| **流动比率** | 流动资产/流动负债 | >2.0 | 1.5-2.0 | 1.0-1.5 | <1.0 | 通用 |
| **速动比率** | (流动资产-存货)/流动负债 | >1.5 | 1.0-1.5 | 0.8-1.0 | <0.8 | 通用 |
| **利息保障倍数** | EBIT/利息 | >8 | 5-8 | 3-5 | <3 | 高负债行业 |

### 3.4 成长能力指标体系

| 指标 | 计算公式 | 优秀 | 良好 | 一般 | 较差 | 阶段适用 |
|:---|:---|:---:|:---:|:---:|:---:|:---|
| **收入增长率** | (本期-上期)/上期 | >20% | 15-20% | 10-15% | <10% | 成长期 |
| **利润增长率** | (本期-上期)/上期 | >收入增速 | =收入增速 | 略低于收入 | 远低于收入 | 通用 |
| **资产增长率** | (本期-上期)/上期 | 10-20% | 5-10% | 0-5% | <0% | 扩张期 |
| **FCF增长率** | (本期-上期)/上期 | >利润增速 | =利润增速 | 略低于利润 | 负增长 | 成熟期 |

## 💡 四、实例演示：万华化学四维分析

### 4.1 基础数据（2023年）

| 项目 | 金额/数值 |
|:---|---:|
| 营业收入 | 1,753.0亿元 |
| 净利润 | 277.5亿元 |
| EBIT | 348.5亿元 |
| 总资产 | 1,475.0亿元 |
| 股东权益 | 840.0亿元 |
| 总负债 | 635.0亿元 |
| 流动资产 | 585.0亿元 |
| 流动负债 | 270.0亿元 |
| 存货 | 185.0亿元 |
| 应收账款 | 62.0亿元 |
| 营业成本 | 1,305.0亿元 |

### 4.2 四维指标计算

**盈利能力指标**：
```
ROE = 277.5 / [(840+780)/2] = 277.5 / 810 = 34.3% ✅ 优秀
ROIC = 296.2 / 933 = 31.7% ✅ 优秀（使用重构后数据）
毛利率 = (1753-1305) / 1753 = 448 / 1753 = 25.6% ✅ 良好
净利率 = 277.5 / 1753 = 15.8% ✅ 优秀
EBIT利润率 = 348.5 / 1753 = 19.9% ✅ 优秀
```

**运营效率指标**：
```
总资产周转率 = 1753 / [(1475+1350)/2] = 1753 / 1412.5 = 1.24次 ✅ 良好
存货周转天数 = 365 / (1305 / 185) = 365 / 7.05 = 52天 ✅ 优秀
应收周转天数 = 365 / (1753 / 62) = 365 / 28.3 = 13天 ✅ 优秀
应付周转天数 = 365 / (1305 / 245) = 365 / 5.33 = 68天 ✅ 良好
现金周期 = 52 + 13 - 68 = -3天 ✅ 优秀（负周期）
```

**偿债能力指标**：
```
资产负债率 = 635 / 1475 = 43.1% ✅ 良好
权益乘数 = 1475 / 840 = 1.76倍 ✅ 良好
流动比率 = 585 / 270 = 2.17倍 ✅ 优秀
速动比率 = (585-185) / 270 = 400 / 270 = 1.48倍 ✅ 良好
利息保障倍数 = 348.5 / 22 = 15.8倍 ✅ 优秀
```

**成长能力指标（2022→2023）**：
```
收入增长率 = (1753-1655) / 1655 = 5.9% 🟡 一般
利润增长率 = (277.5-252) / 252 = 10.1% 🟢 良好
资产增长率 = (1475-1350) / 1350 = 9.3% 🟡 一般
FCF增长率 = (193-180) / 180 = 7.2% 🟡 一般
```

### 4.3 综合评分

| 维度 | 得分计算 | 得分 | 评级 |
|:---|:---|:---:|:---:|
| 盈利能力 | ROE 34.3%→10分, ROIC 31.7%→8分, 毛利率25.6%→7分 | 25/25 | 🟢 优秀 |
| 运营效率 | 周转率1.24→7分, 现金周期-3天→10分, 周转天数→5分 | 22/25 | 🟢 优秀 |
| 偿债能力 | 负债率43.1%→8分, 流动比率2.17→8分, 利息保障15.8→7分 | 23/25 | 🟢 优秀 |
| 成长能力 | 收入增长5.9%→4分, 利润增长10.1%→5分, FCF增长7.2%→4分 | 13/25 | 🟡 一般 |
| **总分** | | **83/100** | 🟢 **良好** |

### 4.4 分析结论

**优势维度**：
1. **盈利能力卓越**：ROE 34.3%远超行业平均，ROIC 31.7%显示核心经营能力极强
2. **运营效率优秀**：现金周期为负（-3天），显示极强的营运资本管理能力
3. **偿债能力优秀**：负债率适中，流动性和利息覆盖能力充足

**短板维度**：
1. **成长能力一般**：收入和利润增速放缓，从高速成长进入稳定期

**综合评价**：
万华化学财务健康度评级为**良好（83分）**，盈利、效率、偿债三维表现优秀，成长维度放缓但仍为正增长。整体财务质量高，风险可控，是典型的成熟期优质企业。

**投资建议**：
- 适合价值投资者长期持有
- 关注成长放缓后的估值调整
- 重点监控行业周期对盈利能力的影响

## ⚠️ 五、常见问题与注意事项

### 5.1 行业差异如何处理？

**不同行业的指标差异巨大**：

| 行业类型 | 典型特征 | 指标调整 |
|:---|:---|:---|
| **重资产行业**（化工、钢铁） | 周转率低、杠杆高 | 周转率标准降低，关注ROIC |
| **轻资产行业**（软件、服务） | 周转率高、杠杆低 | 周转率标准提高，关注毛利率 |
| **高杠杆行业**（银行、地产） | 杠杆高、周转慢 | 杠杆容忍度提高，关注风控 |
| **零售业** | 周转极快、利润薄 | 周转率标准极高，关注现金流 |

**处理方式**：
1. 使用行业相对值而非绝对值
2. 与行业均值对比而非通用标准
3. 关注行业特有的关键指标

### 5.2 四维指标冲突怎么办？

**常见冲突情景**：

| 冲突情景 | 例子 | 解读 |
|:---|:---|:---|
| 高盈利+低成长 | ROE 30%，增长5% | 成熟期现金牛，稳健但无惊喜 |
| 高成长+低盈利 | 增长50%，亏损 | 烧钱扩张期，高风险高回报 |
| 高效率+高杠杆 | 周转快，负债高 | 激进经营，需警惕风险 |
| 高偿债+低效率 | 现金多，周转慢 | 过于保守，资金利用不足 |

**处理原则**：
1. 识别主要矛盾和次要矛盾
2. 结合企业生命周期判断
3. 关注冲突背后的商业模式

### 5.3 评分模型如何调整？

**根据分析目的调整权重**：

| 分析目的 | 权重调整 | 适用场景 |
|:---|:---|:---|
| 价值投资 | 盈利40%+偿债30% | 长期持有 |
| 成长投资 | 成长40%+盈利30% | 成长股筛选 |
| 信用评估 | 偿债50%+盈利25% | 债券投资 |
| 并购评估 | 效率30%+偿债30%+盈利25% | 整合难度评估 |

**自定义评分**：
```python
自定义评分 = (盈利分×W1 + 效率分×W2 + 偿债分×W3 + 成长分×W4) / (W1+W2+W3+W4)

其中W1-W4可根据投资偏好调整
```

### 5.4 数据质量问题

**常见数据质量问题**：

| 问题 | 影响 | 处理 |
|:---|:---|:---|
| 新上市公司历史短 | 成长指标不可靠 | 延长观察期，降低成长权重 |
| 会计政策变更 | 同比数据不可比 | 追溯调整或分段分析 |
| 重大资产重组 | 指标剧烈波动 | 剔除非经常性影响 |
| 季节性强的公司 | 单期数据不具代表性 | 使用TTM或季度平均 |
