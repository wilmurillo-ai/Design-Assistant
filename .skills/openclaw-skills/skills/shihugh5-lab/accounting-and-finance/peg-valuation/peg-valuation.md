# PEG估值方法

## 一、理论背景与核心概念

### 1.1 PEG的起源与本质

**什么是PEG？**
> PEG(Price/Earnings to Growth Ratio)是市盈率与盈利增长率的比率，由投资大师彼得·林奇(Peter Lynch)推广使用，用于评估成长股估值是否合理。

**PEG的经济含义**：
```
PEG = PE / 盈利增长率

解释：
- PEG = 1.0：每1%增长对应1倍PE，估值与增长匹配
- PEG < 1.0：PE低于增长率，可能被低估
- PEG > 1.0：PE高于增长率，可能被高估

直观理解：
PEG衡量"每单位增长所需的估值代价"
```

**彼得·林奇的投资法则**：
> "寻找PEG小于1的股票，这意味着你以低于增长率的价格购买了增长。"

### 1.2 为什么需要PEG？

**传统PE估值的局限**：

| 情况 | PE局限 | PEG解决方案 |
|:---|:---|:---|
| **高成长股** | PE 30倍看似高 | 如果增长30%，PEG=1，合理 |
| **低成长股** | PE 10倍看似低 | 如果增长5%，PEG=2，高估 |
| **跨公司比较** | 不同增长率公司PE不可比 | PEG标准化后可直接比较 |
| **周期股** | 周期高点PE低，实际贵 | PEG结合增长预期更准确 |

**示例说明**：
```
公司A：PE=30倍，增长率=30% → PEG=1.0（合理）
公司B：PE=15倍，增长率=5%  → PEG=3.0（高估）

只看PE：B更便宜（15 vs 30）
看PEG：A更合理（1.0 vs 3.0）

结论：PEG更能反映估值与成长的匹配度
```

### 1.3 PEG的财务理论基础

**从Gordon模型推导PEG**：
```
由 Gordon 模型：
PE = b / (r - g)

其中：
b = 分红率
r = 要求回报率
g = 增长率

PEG = PE / g = b / [g × (r - g)]

当 g = r/2 时，PEG达到最小值
```

**PEG与增长率的关系**：
```
PEG对g敏感，但并非线性关系
- 低增长时：PEG通常>1（增长稀缺溢价）
- 中增长时：PEG可能≈1（估值与增长匹配）
- 高增长时：PEG通常<1（高增长折扣）

但注意：极高增长（>30%）难以持续，PEG可能失真
```

### 1.4 PEG的生命周期特征

**不同阶段的典型PEG水平**：

| 企业阶段 | 增长率 | 典型PE | 典型PEG | 评价 |
|:---|:---:|:---:|:---:|:---|
| **初创期** | >50% | 40-60倍 | 0.8-1.2 | 高风险高回报 |
| **高速成长期** | 30-50% | 25-40倍 | 0.8-1.0 | 成长股最佳区间 |
| **稳定成长期** | 15-30% | 15-25倍 | 1.0-1.2 | 平衡型 |
| **成熟期** | 5-15% | 8-15倍 | 1.0-2.0 | 价值型 |
| **衰退期** | <5% | 5-10倍 | >2.0 | 价值陷阱 |

**万华化学PEG判断**：
- 增长率：6%（成熟期）
- PE：11.7倍
- PEG = 11.7 / 6 = 1.95
- 评价：PEG偏高，但考虑高ROE可接受


> "PEG是成长股估值的重要工具，但要注意增长率的可持续性。不能只看当期高增长，要分析增长的驱动因素和质量。"

1. **增长率质量**：内生增长优于外延并购
2. **增长持续性**：3-5年增长预期比1年增长更可靠
3. **ROE匹配**：高ROE可支撑较高PEG
4. **行业差异**：不同行业PEG基准不同

## 二、详细操作步骤

### 步骤1：确定PEG适用性

**1.1 PEG估值的适用条件**：

**高度适用**：
- [x] 盈利为正（PE有定义）
- [x] 增长率为正（PEG有意义）
- [x] 成长股（增长率>10%）
- [x] 增长可持续（非一次性增长）

**轻度适用**：
- 一般成长型企业
- 周期股（需调整）

**不适用**：
- 亏损企业（PE无定义）
- 零增长或负增长
- 极不稳定增长

**1.2 万华化学适用性**：⚠️ 轻度适用
- 增长率为正（6%）但较低
- 盈利稳定
- PEG可能偏高，需结合ROE调整

### 步骤2：计算PE

**2.1 PE计算**：
```python
def 计算PE(股价, EPS):
    """
    计算市盈率PE
    """
    if EPS <= 0:
        return None, "EPS为负，无法计算PEG"
    
    PE = 股价 / EPS
    return PE

# 万华化学
股价 = 110
EPS = 9.43
PE = 计算PE(股价, EPS)
# 结果：11.7倍
```

### 步骤3：确定增长率

**3.1 增长率选择标准**：

| 增长率类型 | 适用场景 | 优点 | 缺点 |
|:---|:---|:---|:---|
| **历史增长率** | 趋势稳定企业 | 数据确定 | 不代表未来 |
| **预测增长率** | 投资决策 | 前瞻性 | 预测误差 |
| **复合增长** | 3-5年平均 | 平滑波动 | 可能失真 |

**推荐**：使用未来3-5年预期增长率

**3.2 增长率确定方法**：
```python
def 确定增长率(历史盈利, 分析师预测, 行业增长):
    """
    综合确定用于PEG的增长率
    """
    # 历史CAGR（3年）
    历史CAGR = (历史盈利[-1] / 历史盈利[-3]) ** (1/3) - 1
    
    # 权重分配
    权重 = {
        '历史': 0.3,
        '预测': 0.5,
        '行业': 0.2
    }
    
    综合增长率 = (历史CAGR * 权重['历史'] + 
                  分析师预测 * 权重['预测'] + 
                  行业增长 * 权重['行业'])
    
    return 综合增长率

# 万华化学
历史CAGR = 0.08  # 8%
分析师预测 = 0.06  # 6%
行业增长 = 0.05  # 5%

增长率 = 确定增长率([100, 110, 120], 0.06, 0.05)
# 结果：约6.4%
```

**3.3 增长率约束**：
```
增长率上限：30%（超过30%难以持续，PEG失真）
增长率下限：3%（低于3%PEG意义不大）

如果增长率 > 30%，使用30%计算PEG（保守）
如果增长率 < 3%，PEG不适用
```

### 步骤4：计算PEG

**4.1 PEG计算**：
```python
def 计算PEG(PE, 增长率):
    """
    计算PEG比率
    增长率使用百分比数值（如15%用15）
    """
    if 增长率 <= 0:
        return None, "增长率为负，PEG不适用"
    
    # 增长率上限约束
    调整后增长率 = min(增长率 * 100, 30)
    
    PEG = PE / 调整后增长率
    
    # PEG评价
    if PEG < 0.5:
        评价 = '严重低估'
    elif PEG < 1.0:
        评价 = '低估'
    elif PEG < 1.5:
        评价 = '合理'
    elif PEG < 2.0:
        评价 = '偏高'
    else:
        评价 = '高估'
    
    return {
        'PE': PE,
        '增长率': 增长率,
        '调整后增长率': 调整后增长率,
        'PEG': PEG,
        '评价': 评价
    }

# 万华化学
PEG结果 = 计算PEG(11.7, 0.06)
# 结果：PEG=1.95，评价=高估
```

### 步骤5：ROE-PEG综合评估

**5.1 传统PEG的局限**：
```
传统PEG仅考虑增长，未考虑ROE（资本效率）

示例：
公司A：PE=20，增长率=20%，ROE=30% → PEG=1.0
公司B：PE=20，增长率=20%，ROE=10% → PEG=1.0

只看PEG：两者相同
但公司A资本效率更高，应给予更高估值
```

**5.2 ROE-PEG调整方法**：
```python
def ROE调整PEG(PE, 增长率, ROE, 行业平均ROE):
    """
    基于ROE调整PEG评价
    """
    基础PEG = PE / (min(增长率 * 100, 30))
    
    # ROE调整因子
    if ROE > 行业平均ROE:
        # ROE高于行业，PEG应上调（可以更高）
        ROE溢价 = (ROE - 行业平均ROE) / 0.05 * 0.3
        调整后PEG上限 = 1.5 + ROE溢价
    else:
        # ROE低于行业，PEG应下调
        ROE折价 = (行业平均ROE - ROE) / 0.05 * 0.3
        调整后PEG上限 = 1.5 - ROE折价
    
    # 评价调整
    if 基础PEG <= 调整后PEG上限:
        评价 = '合理或低估'
    else:
        评价 = '偏高'
    
    return {
        '基础PEG': 基础PEG,
        '调整后PEG上限': 调整后PEG上限,
        '评价': 评价,
        'ROE溢价': ROE溢价 if ROE > 行业平均ROE else -ROE折价
    }

# 万华化学
调整结果 = ROE调整PEG(11.7, 0.06, 0.26, 0.16)
# 结果：基础PEG=1.95，调整后上限=2.1，评价=合理
```

### 步骤6：可比公司PEG比较

**6.1 同业PEG对比**：
```python
def 同业PEG对比(本公司PEG, 可比公司PEG列表, 可比公司ROE_list, 本公司ROE):
    """
    与同业比较PEG
    """
    import numpy as np
    
    平均PEG = np.mean(可比公司PEG列表)
    中位数PEG = np.median(可比公司PEG列表)
    
    # ROE调整后可比PEG
    调整后可比PEG = []
    for peg, roe in zip(可比公司PEG列表, 可比公司ROE_list):
        if roe > 0:
            调整后PEG = peg * (本公司ROE / roe) ** 0.5
            调整后可比PEG.append(调整后PEG)
    
    平均调整后PEG = np.mean(调整后可比PEG)
    
    return {
        '本公司PEG': 本公司PEG,
        '同业平均PEG': 平均PEG,
        '同行业中位数PEG': 中位数PEG,
        'ROE调整后平均PEG': 平均调整后PEG,
        '相对位置': '前25%' if 本公司PEG < np.percentile(可比公司PEG_list, 25) else '后25%'
    }

# 万华化学
可比PEG = [1.5, 2.2, 1.8, 2.5]
可比ROE = [0.20, 0.18, 0.15, 0.12]

对比结果 = 同业PEG对比(1.95, 可比PEG, 可比ROE, 0.26)
# 结果：万华PEG低于ROE调整后平均水平
```

## 三、公式与计算逻辑

### 3.1 PEG基本公式

**标准公式**：
```
PEG = PE / 盈利增长率

其中：
- PE = 当前市盈率
- 盈利增长率 = 未来3-5年预期增长率（百分比数值）

示例：
PE = 20倍
增长率 = 20%
PEG = 20 / 20 = 1.0
```

### 3.2 合理PE反推

**从PEG反推合理PE**：
```
合理PE = 目标PEG × 增长率

假设：
目标PEG = 1.2（合理偏高）
增长率 = 15%
合理PE = 1.2 × 15 = 18倍

如果当前PE = 15倍
→ 低估，有上涨空间
```

### 3.3 增长-PE关系表

**PEG=1时的合理PE**：

| 增长率 | 合理PE(PEG=1) | 合理PE(PEG=1.2) | 合理PE(PEG=0.8) |
|:---:|:---:|:---:|:---:|
| 5% | 5倍 | 6倍 | 4倍 |
| 10% | 10倍 | 12倍 | 8倍 |
| 15% | 15倍 | 18倍 | 12倍 |
| 20% | 20倍 | 24倍 | 16倍 |
| 25% | 25倍 | 30倍 | 20倍 |
| 30% | 30倍 | 36倍 | 24倍 |

## 五、常见问题与注意事项

### 5.1 PEG对低增长企业的惩罚

**Q：为什么万华PEG=1.95显示"高估"，但PE只有11.7倍？**

**A**：传统PEG对低增长企业不公平：
```
问题：增长率分母小，PEG被放大

万华：PE=11.7，增长=6% → PEG=1.95
成长股：PE=30，增长=30% → PEG=1.0

万华PE更低，但PEG更高
→ 传统PEG"惩罚"了低增长企业

解决方案：使用ROE调整PEG
或：对低增长企业（<10%）PEG参考值放宽到2.0
```

### 5.2 增长率预测的风险

**Q：如何降低增长率预测错误的风险？**

**A**：
1. **多情景分析**：乐观/基准/悲观三种增长率
2. **历史验证**：用历史数据验证预测合理性
3. **行业对比**：与行业增长对比
4. **敏感性分析**：测试增长率变动对PEG影响

### 5.3 PEG的适用边界

**PEG不适用情况**：
- 增长率 > 50%（难以持续，PEG失真）
- 增长率 < 3%（分母太小，PEG无意义）
- 盈利为负（PE无定义）
- 极不稳定增长（PEG波动大）

### 5.4 PEG与价值陷阱

**Q：低PEG一定是好投资吗？**

**A**：不一定，可能是价值陷阱：
```
公司A：PE=5，增长=10% → PEG=0.5（看似低估）
但增长不可持续，明年可能负增长

公司B：PE=25，增长=25% → PEG=1.0（看似合理）
但增长可持续，ROE高

结论：PEG只是工具，需结合质量和可持续性分析
```
