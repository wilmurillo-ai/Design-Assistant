# 私域流量运营助手（微信群/朋友圈/企微）

## 功能概述

**输入**：业务类型 + 客户画像 + 运营目标
**输出**：完整的私域运营方案，包括朋友圈剧本、群发话术、社群 SOP、活动方案
**适用场景**：微信群运营、企业微信、朋友圈营销、客户分层管理

## 使用场景

- 刚加了一批新客户，不知道怎么运营激活
- 社群变成死群，用户活跃度越来越低
- 朋友圈发了很多但没人点赞咨询
- 想做促销活动，但没有完整的运营方案
- 客户分层管理混乱，高价值客户流失

## 核心功能

### 1. 朋友圈剧本生成器

输出每日朋友圈发布计划（7天/30天滚动）：

| 时间 | 内容类型 | 目的 |
|------|---------|------|
| 07:30-08:30 | 早安问候 + 今日正能量 | 建立信任 |
| 09:30-10:30 | 干货分享（1-3条） | 价值输出 |
| 12:00-13:00 | 产品软广（生活化场景） | 种草 |
| 17:30-18:30 | 限时活动 / 客户案例 | 促活转化 |
| 21:00-22:00 | 互动话题 / 晚安问候 | 增强粘性 |

**朋友圈内容类型矩阵：**

```
┌─────────────┬─────────────┬─────────────┐
│  干货型30%  │  产品型30%  │  生活型20%  │
│  行业知识   │  软性种草   │  个人IP打造 │
│  教程技巧   │  客户案例   │  日常分享   │
├─────────────┼─────────────┼─────────────┤
│  互动型10% │  活动型10%  │             │
│  问答投票   │  限时福利   │             │
│  话题讨论   │  抽奖秒杀   │             │
└─────────────┴─────────────┴─────────────┘
```

### 2. 社群 SOP 生成器

输出完整的社群运营 SOP，包含：

**社群生命周期管理：**

```

□ 发布入群欢迎语
□ 介绍社群价值

□ 发起第一次破冰活动

【第二阶段：激活期】（Day 2-7）
□ 每日早安问候
□ 定时干货分享
□ 话题讨论引导
□ 轻度促销活动

【第三阶段：维护期】（Day 8-30）
□ 定期抽奖活动
□ 会员专属福利
□ 客户案例分享
□ 定期调研收集反馈

【第四阶段：裂变期】（定期）
□ 邀请有礼活动
□ 拼团/分销机制
□ 口碑传播激励
□ KOC培养计划
```

**每日群运营时间表：**

### 4. 客户分层管理方案

输出客户分层运营策略：

```
【客户分层模型】
┌────────────────────────────────────────┐
│  A类：KOC（高价值传播者）              │
│  特征：消费高、活跃强、愿意分享       │
│  策略：专属福利、优先体验、邀请裂变   │
│  维护成本：高（1对1服务）             │
├────────────────────────────────────────┤
│  B类：忠实客户                         │
│  特征：复购稳定、偶尔互动             │
│  策略：积分体系、专属优惠、节日关怀   │
│  维护成本：中（社群维护即可）         │
├────────────────────────────────────────┤
│  C类：潜力客户                         │
│  特征：有需求但未成交或低频消费       │
│  策略：定向优惠、案例种草、限时活动   │
│  维护成本：中（自动化推送）           │
├────────────────────────────────────────┤
│  D类：沉睡客户                         │
│  特征：30天无互动或已取关             │
│  策略：唤醒活动、问卷调研、流失激励   │
│  维护成本：低（自动化触达）           │
└────────────────────────────────────────┘
```

### 5. 活动策划方案

输出完整活动策划案：

**裂变活动 SOP：**

```
活动名称：[xxx]邀请有礼
活动目的：新增XX个精准客户
活动时间：2026年X月X日- X月X日
参与门槛：已消费客户

【活动机制】
1. 老客户每邀请1位新客户添加微信
   → 双方各获得[礼品/积分/优惠券]
2. 邀请排行榜前10名
   → 额外获得[大奖]
3. 邀请达3人
   → 升级为[VIP身份]

【推广节奏】
Day 1-2：预热（朋友圈+社群预告）
Day 3：正式启动（群发+私发）
Day 5：中期助推（成果展示+榜单公示）
Day 7：收尾（最后机会+感谢）

【物料清单】
□ 活动海报（多版本）
□ 邀请文案（群发版+私信版）
□ 客服话术（FAQ）
□ 统计数据表
□ 礼品兑换流程
```
