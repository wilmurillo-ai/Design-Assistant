# 短视频带货脚本生成器（抖音/快手/视频号）

## 功能概述

**输入**：商品链接/图片 + 卖点描述 + 目标平台
**输出**：完整短视频带货脚本，包含开头钩子、话术、节奏建议、评论区引导
**支持平台**：抖音、快手、视频号、小红书（自动适配各平台风格）

## 使用场景

- 刚拿到一款产品，不知道怎么拍带货视频
- 视频播放量还行，但转化率低（话术不行）
- 想批量生产带货视频，但缺脚本创意
- 达人合作需要提供标准话术参考

## 核心功能

### 1. 智能话术生成

输入商品信息，输出完整可拍摄的话术脚本：

| 模块 | 内容 | 时长建议 |
|------|------|---------|
| 开场钩子 | 3秒抓人开场白 | 0-3秒 |
| 痛点切入 | 引发共鸣的场景描述 | 3-8秒 |
| 产品介绍 | 核心卖点逐条拆解 | 8-40秒 |
| 价格钩子 | 限时优惠/对比价格 | 40-50秒 |
| 行动号召 | 引导点击下方链接 | 50-60秒 |

### 2. 多风格模板

| 风格 | 适用场景 | 语气 |
|------|---------|------|
| 咆哮体 | 工厂/源头好货 | 高亢激情 |
| 种草分享 | 美妆/母婴/日用 | 亲切真实 |
| 知识干货 | 数码/家电/健康 | 专业可信 |
| 剧情植入 | 食品/服装/家居 | 软性植入 |
| 对比测评 | 电器/工具/功能产品 | 客观中立 |

### 3. 爆款开头生成器

自动生成 10+ 种爆款开头方案：

```
A. 反常识型：「没想到这东西才卖29块9！」
B. 问题引导型：「你是不是也跟我一样，被这个问题困扰了10年？」
C. 身份认同型：「做电商3年，这款产品让我月销破了100万」
D. 场景代入型：「每天早上赶时间，早餐总是将就……」
E. 数据冲击型：「100个人里面，只有1个人知道这个秘密」
```

### 4. 评论区话术

自动生成 10 条引导购买和互动的话术：

```
[购买引导]
- 点击下方链接，领完券只要XX元
- 库存不多了，需要的抓紧
- 看完还不会买的，评论区问我

[互动引导]
- 你们遇到过这种情况吗？评论区说说
- 觉得有用的点个赞，支持一下
- 关注我，每天分享更多好货

[顾虑消除]
- 质量问题包退，放心拍
- 7天无理由，买了不亏
- 收到不满意直接退，不收运费
```

### 5. 分镜提词器

输出带时间戳的分镜脚本，方便拍摄：

```
[00:00-00:03] 🔥 开场
画面：手机怼脸拍摄，表情夸张
话术：「姐妹们！这个XX真的绝了！」
动作：手指向镜头，语气惊讶

[00:03-00:08] 🤕 痛点
画面：切换场景（厨房/客厅）
话术：「以前我用传统的XX，特别麻烦，每次都要搞半天……」
动作：皱眉演示旧方法的不便

[00:08-00:25] 🎁 产品展示
画面：产品全景 + 细节特写
话术：「现在用这个，3步搞定，而且只要XX元！」
动作：边说边演示操作流程

[00:25-00:35] 💰 价格钩子
话术：「今天直播间专属价，比平时便宜XX元！」
动作：展示原价和直播价的对比

[00:35-00:45] ⚡ 行动号召
话术：「想要的赶紧下单，30分钟后恢复原价！」
动作：手点击购物车图标
```
