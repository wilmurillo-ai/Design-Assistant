# 淘宝/天猫/闲鱼多平台商品Listing生成器

## 功能概述

**输入**：商品基本信息（名称、功能特点、规格参数、目标人群、 价格带）
**输出**：完整可上架的商品Listing，包含标题、主图文案、详情描述、标签关键词
**支持平台**：淘宝、天猫、闲鱼、拼多多（自动适配各平台规则）

## 使用场景

- 闲鱼卖家快速生成商品 listing
- 淘宝新手开店批量上传商品
- 拼多多商家优化商品信息
- 跨境卖家做国内平台 listing

## 工作流程

### 第一步：收集商品信息

用户需提供以下信息（可语音或文字随意描述，我会整理）：

```
商品名称：
商品类目：
核心功能/卖点（3-5个）：
目标人群：
价格区间：
规格选项（如有）：
特殊说明（如有）：
```

### 第二步：生成标题

**规则（淘宝/天猫）：**
- 核心关键词放前 8 个字
- 包含品牌词 + 核心属性 + 热门长尾词
- 避免违规词（最、第一、绝对等）
- 字数：28-30字

**规则（闲鱼）：**
- 口语化、接地气
- 直接说明商品状况（全新/闲置/二手）
- 标注价格优势

**规则（拼多多）：**
- 低价格敏感词 + 高性价比词
- 强调量词（10包、5斤等）

### 第三步：生成主图文案

输出 5 张主图的图文搭配建议：

| 图片位置 | 文案内容 | 风格 |
|---------|---------|------|
| 第1张 | 核心卖点一句话 | 大字醒目 |
| 第2张 | 商品全貌+规格 | 清晰展示 |
| 第3张 | 细节图+功能说明 | 场景化 |
| 第4张 | 对比图/效果图 | 可信度高 |
| 第5张 | 售后保障+购买引导 | 促行动 |

### 第四步：生成详情页描述

结构化输出：

```
【商品介绍】
（2-3段产品描述，场景化文案）

【规格参数】
（表格形式列出规格）

【使用场景】
（真人使用场景描述，引发共鸣）

【买家秀/评价引导】
（软性引导好评的话术）

【售后保障】
（退换货、运费险、七天无理由等）
```

### 第五步：生成标签关键词

输出 20 个可用标签（#形式），覆盖：
- 类目词
- 属性词
- 长尾词
- 场景词
- 人群词
