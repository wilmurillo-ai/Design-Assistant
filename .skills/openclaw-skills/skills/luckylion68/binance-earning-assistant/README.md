# 币安撸毛助手

自动获取币安最新撸毛活动信息，包括理财产品、活动奖励、空投预告等。

## 功能特性

- ✅ **自动获取** - 从币安公告 API 自动获取活动信息
- ✅ **实时查询** - 按需查询，实时更新
- ✅ **分类展示** - 理财产品、活动奖励、空投预告
- ✅ **详细信息** - 奖池、截止时间、参与方式、链接
- ✅ **空投监控** - 从 alpha123.uk 获取空投预告

## 使用方法

### 查看今天的撸毛信息

直接对 OpenClaw 说：
```
今日撸毛信息
```

### 查看特定分类

```
理财产品
活动奖励
空投预告
```

## 输出格式

每个活动包含以下信息：

- 🔗 **链接** - 币安公告链接
- 💰 **奖池** - 奖励金额或 APR
- 📅 **截止时间** - 活动截止日期
- 🎯 **门槛** - 参与门槛
- 📝 **参与方式** - 参与步骤
- ⚠️ **风险** - 风险提示

## 活动分类

| 分类 | 说明 |
|------|------|
| 💰 理财产品 | 币安理财产品，包括活期、定期 |
| 🎁 活动奖励 | 交易竞赛、奖励活动 |
| 📢 其他活动 | 知识问答等其他活动 |
| 🚀 ALPHA 空投预告 | Alpha 板块空投预告 |

## 依赖

- requests（已内置）

## 注意事项

1. 所有数据来自币安官方公告和 alpha123.uk
2. 活动信息可能随时变更，请以官方页面为准
3. 空投有风险，参与需谨慎
4. 理财非存款，产品有风险
