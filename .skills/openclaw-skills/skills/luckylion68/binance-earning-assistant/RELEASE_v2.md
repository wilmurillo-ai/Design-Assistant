# 币安撸毛助手 v2.0.0

## 更新说明

### v2.0.0 新功能
- ✅ 实时从币安 API 获取活动列表（catalogId: 93）
- ✅ 每个活动都调用详情 API 获取准确截止日期
- ✅ 智能区域限制检测（过滤非华语区活动）
- ✅ 活动类型自动识别（理财/奖励/竞赛等）
- ✅ 智能去重（活动类型 + 代币作为 key）
- ✅ 广场任务识别（CreatorPad/Square 关键词）
- ✅ Alpha 空投实时获取（alpha123.uk）
- ✅ UTC 时间转 UTC+8（华语区时间）
- ✅ 中文名称映射表
- ✅ 代币智能识别

### v1.0.x 功能
- 基础活动列表展示
- 触发词自动响应
- 分类展示（理财/活动奖励/广场任务）

## 核心文件
- `SKILL.md` - 技能配置
- `summary_table.py` - 主程序（实时获取 + 智能过滤）
- `activity_names.py` - 中文映射 + 代币识别 + 区域检测

## 使用方法
```bash
python3 summary_table.py
```
