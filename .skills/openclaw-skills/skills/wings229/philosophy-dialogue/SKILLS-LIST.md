# ⚔️ 哲学家技能系统 / Philosopher Skills

**版本**: v2.10.0  
**最后更新**: 2026-04-12

---

## 🎮 技能使用说明

### 标准对话
- 每位哲学家每场对话限用 **1 次技能**
- 技能使用时标注（如「【技能】有教无类」）
- 技能效果持续 1-3 轮发言
- 增加游戏化和不可预测性

### 双人论战
- 每位哲学家每场论战限用 **2 次技能**
- 技能可直接影响评委投票
- 技能使用时机更关键（建议在关键回合使用）
- 组合技能不适用（仅 2 人对决）

## 📝 技能使用示例

### 示例 1：核心技能

```markdown
**孔子** 😐【技能】有教无类：
「各位，让我们暂时放下立场，共同思考这个问题的本质。」
→ 全场哲学家暂时进入深度思考状态（持续 2 轮）
```

### 示例 2：扭转局势

```markdown
**尼采** 😤【技能】价值重估：
「传统价值崩溃了，我们要创造新价值！」
→ 之前所有论证被推翻，讨论重新开始
```

### 示例 3：质疑本质

```markdown
**庄子** 😏【技能】庄周梦蝶：
「你们怎么知道现在不是在做梦？」
→ 全场哲学家暂时陷入沉思（持续 1 轮）
```

### 示例 4：组合技能

```markdown
【组合技能】孔子 + 孟子「仁政天下」：
孔子：「仁者爱人。」
孟子：「民为贵，社稷次之，君为轻。」
→ 全场进入道德讨论模式（持续 3 轮）
```
