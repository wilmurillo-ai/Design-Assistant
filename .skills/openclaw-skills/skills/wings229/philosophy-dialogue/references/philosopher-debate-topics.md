## 📖 使用说明

### 触发方式

```bash
# 指定哲学家 + 使用其题目
孔子，辩论：仁者爱人是否应无条件

# 随机哲学家 + 其专属题目
哲学对话，话题：随机

# 双人论战 + 两位哲学家的题目
孔子 vs 尼采，辩论：仁者爱人 vs 超人进化
```

### 题目特点

| 特点 | 说明 |
|------|------|
| **20 字以内** | 简洁明了，便于辩论 |
| **不重复** | 200 位哲学家各 1 题 |
| **核心思想** | 反映哲学家核心主张 |
| **可辩论** | 有正反方，非单一答案 |
