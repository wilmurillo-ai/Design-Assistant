# 对话示例

## 示例 1：完整对话

**用户输入**：「组织哲学对话，话题：什么是自由」

**执行过程**：

```
Phase 0: 扫描 skills/ → 发现 15 位已蒸馏哲学家 → 全部参与
Phase 1: 相关性排序 → 庄子⭐5 苏格拉底⭐5 第欧根尼⭐5 韩非子⭐4 荀子⭐4
         随机选择 → 主持人：庄子 | 记录员：管子
Phase 2: 对话 8 轮（≥50% 发言）
Phase 3: 总结输出
```

## 示例 2：指定参与人

**用户输入**：「让孔子和苏格拉底讨论教育」

**执行过程**：

```
Phase 0: 只有 2 人，取消主持人/记录员分工
Phase 1: 跳过（2 人直接对话）
Phase 2: 孔子先发言 → 苏格拉底回应 → 孔子回应 → 各 2 轮
Phase 3: 总结输出
```
