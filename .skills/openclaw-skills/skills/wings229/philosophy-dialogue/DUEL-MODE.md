# ⚔️ 双人论战模式 / Duel Mode

**版本**: v3.1.0  
**最后更新**: 2026-04-13

---

## 🎯 功能概述

双人论战模式是哲学对话 Skill 的核心功能之一，支持两位哲学家进行 1v1 直接对抗。

### 核心特性

| 功能 | 描述 |
|------|------|
| ⚔️ **1v1 对决** | 两位哲学家直接对抗 |
| 🎯 **回合控制** | 默认 20 回合，可设置 5-50 回合 |
| 🏳️ **主动放弃** | 一方可随时认输，对方直接获胜 |
| 📊 **评委打分** | 15-20 位哲学家作为评委 |
| 🏆 **胜负判定** | 按得票数分胜负 |
| 🔄 **避免重复** | 每回合内容不重复，观点递进 |
| 🗳️ **集中投票** | 仅在中场休息和结束时投票 |

---

## 🚀 快速上手

### 触发方式

```bash
# 随机选两人论战
双人论战，话题：什么是自由

# 指定两人论战
尼采 vs 庄子，话题：人生的意义

# 设置回合数
双人论战，话题：人性善恶，回合：15
```

### 完整参数

```bash
双人论战
话题：什么是真正的自由
选手：尼采 vs 庄子
回合：25
评委：18 人
```

## 🎮 回合控制

### 默认设置
- **默认回合数**：20 回合
- **可设置范围**：5-50 回合
- **发言顺序**：哲学家 A → 哲学家 B
- **投票时机**：第 10 回合（中场）、第 20 回合（结束）

### 避免重复规则 ⚠️

| 规则 | 说明 |
|------|------|
| **观点递进** | 每回合必须推进论点，不能重复之前说过的话 |
| **新论据** | 每回合至少引入 1 个新论据或新视角 |
| **回应对方** | 必须直接回应对方的上一个论点，不能自说自话 |
| **深度递进** | 从表层→深层，从具体→抽象，逐步深入 |
| **主持人监督** | 如发现重复，主持人可提醒哲学家换角度 |

### 提前结束条件

| 条件 | 说明 |
|------|------|
| **主动放弃** | 一方可随时说「我认输」，对方直接获胜 |
| **压倒性胜利** | 一方得分领先超过 70%，可提前结束 |
| **达到回合数** | 达到设定回合数，按总票数分胜负 |

### 平局处理
- 双方票数相同
- 加赛 3 回合
- 或宣布平局

## 📊 打分机制

### 评委选择

**评委人数**：**15-20 位哲学家**（扩大评委阵容，增加代表性）

**选择标准**：
- 中立立场（不偏向任何一方）
- 有判断力（善于分析论证）
- 与话题相关但不直接参与
- 涵盖中西哲学家（多元化视角）

### 投票规则 ⚠️ **重要更新 v3.1.0**

| 规则 | 说明 |
|------|------|
| **每票分值** | 1 票 = 1 分 |
| **投票时机** | **仅在第 10 回合（中场）和第 20 回合（结束）投票** |
| **评委人数** | **15-20 位哲学家**（扩大评委阵容） |
| **投票限制** | **每位评委每次投票只能投 1 票**（不能投双方） |
| **弃权** | 可以弃权（认为双方平手） |
| **回避** | 不能投给自己（如评委也参与论战） |

### 投票流程

```
第 1-9 回合：论战进行，无投票
第 10 回合：中场休息 → 15-20 位评委集中投票（每人 1 票）
第 11-19 回合：论战进行，无投票
第 20 回合：最终投票 → 15-20 位评委集中投票（每人 1 票）
```

---

### 胜负判定

| 情况 | 判定 |
|------|------|
| **正常结束** | 达到设定回合数，按总票数分胜负 |
| **认输** | 哲学家主动说「我认输」，对方直接获胜 |
| **压倒性胜利** | 一方得分领先超过 70%，可提前结束 |
| **平局** | 双方票数相同，加赛 3 回合或宣布平局 |

## ⚔️ 技能规则

### 标准对话 vs 双人论战

| 对话类型 | 技能使用 |
|----------|----------|
| 标准对话 | 1 次/哲学家 |
| 双人论战 | 2 次/哲学家 |

### 技能使用时机
- 建议在关键回合使用（如第 10、15、20 回合）
- 技能可直接影响评委投票
- 组合技能不适用（仅 2 人对决）

### 技能反制
| 技能 | 反制技能 |
|------|----------|
| 「庄周梦蝶」 | 「我思故我在」 |
| 「价值重估」 | 「绝对命令」 |
| 「宁我负人」 | 「非暴力」 |
