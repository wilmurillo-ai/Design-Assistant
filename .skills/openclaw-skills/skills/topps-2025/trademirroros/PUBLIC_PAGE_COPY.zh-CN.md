# TradeMirrorOS Public Copy（中文版）

TradeMirrorOS 是一个面向交易记忆工作流的最小技能包。

它不替你交易。
它不会把你的数据自动推送到任何地方。
它提供的是一套结构化技能说明，用来把计划、交易、复盘与行为模式沉淀为可复用的长期记忆指令。

## 定位

TradeMirrorOS 适合这类需求：

- 需要交易记录与复盘任务的路由层
- 希望把个人经验整理成可复用 skill prompts
- 希望用长期记忆框架替代短聊天上下文
- 希望清晰区分计划、执行复盘与纪律分析

## 安全姿态

- instruction-only 技能包
- 不捆绑运行时代码
- 不包含自动同步
- 不预设任何凭证需求
