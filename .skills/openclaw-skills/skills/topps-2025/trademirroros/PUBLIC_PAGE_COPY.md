# TradeMirrorOS Public Copy

TradeMirrorOS is a minimal skill bundle for trading-memory workflows.

It does not trade for you.
It does not push your data anywhere.
It gives a structured way to turn plans, trades, reviews, and behavior patterns into reusable long-term memory instructions.

## Positioning

TradeMirrorOS is for traders who want:

- a routing layer for journaling and review tasks
- reusable skill prompts grounded in personal evidence
- a long-horizon memory frame instead of short chat context
- clearer separation between planning, execution review, and discipline analysis

## Safety Posture

- instruction-only package
- no bundled runtime code
- no sync automation
- no credential expectations
