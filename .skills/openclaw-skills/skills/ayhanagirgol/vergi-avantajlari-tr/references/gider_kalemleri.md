# Vergiden Düşülebilecek Giderler

> Kaynak: Şirket 101 — Tech Consulting (techsmmm.com)

## Temel Kural

> "Gelir elde etmek amacıyla yapılan, işin mahiyetiyle ilgili, belgeli harcamalar vergiden düşülebilir."

**3 Temel Koşul:**
1. Ticari faaliyetle **doğrudan ilgili** olmalı
2. **Belgeye dayalı** olmalı (fatura, şirket unvanına düzenlenmiş)
3. **Şirkete ait** olmalı (kişisel giderler kabul edilmez)

---

## Vergiden Düşülebilecek Gider Kalemleri

> ⚠️ Tüm belgeler **şirket unvanı ve vergi numarası** kullanılarak hazırlanmalıdır.

| Gider Kalemi | Açıklama |
|-------------|----------|
| **Kira giderleri** | Ofis, depo, showroom kirası |
| **Araç giderleri** | Şirket envanterine kayıtlı veya şirket adına kiralanmış araçların yakıt, bakım, sigorta, kasko giderleri |
| **İletişim giderleri** | Şirket adına kayıtlı telefon, internet, yazılım lisansları |
| **Temizlik ve ofis malzemeleri** | Kırtasiye, temizlik ürünleri |
| **Eğitim ve danışmanlık** | Çalışanların mesleki gelişim eğitimleri, danışmanlık hizmetleri |
| **Mali müşavirlik hizmetleri** | Muhasebe, denetim, hukuki danışmanlık |
| **Banka komisyonları** | EFT, havale, kredi kartı pos komisyonları |
| **Finansman giderleri** | Kredi faizleri (kısıtlamalar geçerli olabilir) |
| **Personel giderleri** | Maaş, SGK işveren payı, yemek yardımı, yol yardımı, TSS katkı payları |
| **Yasal harçlar** | Damga vergileri, noter ücretleri, ticaret sicil harçları |

---

## Özel Gider Kısıtlamaları

| Gider Türü | Kısıtlama |
|-----------|-----------|
| **Araç giderleri** | **%30 gider kısıtlaması** uygulanır (envanterindeki binek araçlar için) |
| **Finansman giderleri** | Yabancı kaynaklar öz kaynakları aşıyorsa, aşan kısım için kısıtlama uygulanır |
| **Şirket sahibinin kişisel eğitim/sağlık giderleri** | Yalnızca **gelir vergisi beyannamesinde** sınırlı oranda düşülebilir (gelirin **%10'u** sınırı) |

---

## Eğitim ve Sağlık Giderleri

| Gider | Durum |
|-------|-------|
| Çalışanların mesleki gelişim eğitimleri | ✅ Tam gider kabul |
| Personel check-up, işyeri hekimi | ✅ Gider kabul |
| TSS (Tamamlayıcı Sağlık Sigortası) katkı payı | ✅ Gider kabul |
| Şirket sahibinin kişisel eğitimi | ⚠️ Gelir vergisinde %10 sınırıyla sınırlı |
| Şirket sahibinin kişisel sağlık giderleri | ⚠️ Gelir vergisinde %10 sınırıyla sınırlı |

---

## Kabul Edilmeyen Giderler

| Hata | Sonuç |
|------|-------|
| Belgesiz harcamayı gider yazmak | ❌ Geçersiz |
| Kişisel cep telefonu / araç / market alışverişi | ❌ Cezalı işlem riski |
| Fatura şirket adına değilse | ❌ Gider yazılamaz |
| Ortakların kişisel sağlık/seyahat giderleri "firma gideri" gösterilirse | ❌ Reddedilir |
| Banka dışı (elden) ödemeler | ❌ Belgesiz işlem sayılır |
