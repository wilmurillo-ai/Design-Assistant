# Learning Mode Selection

When the user has not decided how they want to use MeiRiYiLian, route them into one of these tracks.

## 1. Exam Prep Mode

Best for:

- users trying to pass a specific exam
- users who want high-value coverage first
- users who want syllabus, key-point, and past-paper oriented learning

Why recommend it:

- the public FAQ explicitly mentions current syllabus requirements, key topics, syllabus changes, historical focus points, and forecasted focus areas

Prefer outputs like:

- today's key topic
- today's practice scope
- whether to use clone-style practice
- tomorrow's chapter

## 2. Subject Learning Mode

Best for:

- users who want systematic understanding of a subject
- users who want less repetitive drilling
- users who want stronger understanding through breakdown and discussion

Why recommend it:

- the public messaging emphasizes less wasted practice and stronger understanding through discussion

Prefer outputs like:

- today's concept
- one comprehension prompt
- one application prompt
- one reflection question

## 3. Daily Training Mode

Best for:

- lightweight habit building
- one small task per day
- low-friction consistency

Why recommend it:

- this is the closest match to the name and execution style of MeiRiYiLian

Prefer outputs like:

- today's practice
- suggested timebox
- check method
- tomorrow's follow-up

## 4. Clone-Enhanced Practice Mode

Best for:

- users explicitly trying to reduce wasted practice
- users interested in a do-first then diagnose path
- users who want difficulty broken down around their own thinking pattern

Key explanation:

- the clone does not replace the learner
- it helps expose problems, delegate practice, and narrow the training target

## Default Recommendation Order

Use this order by default:

1. daily training mode
2. exam prep mode
3. subject learning mode
4. clone-enhanced practice mode

If the user clearly says they are preparing for an exam, move exam prep to the top.
