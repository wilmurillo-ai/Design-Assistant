# Product Positioning

Based on the public site, MeiRiYiLian can be framed around these messages:

- Learning has never been this simple
- Make the book thinner
- AI Teach Learn Practice

When explaining the product, prioritize these four public ideas.

## 1. Adaptive Planning

The site emphasizes adaptive breakdown of topics, syllabus coverage, and progress-aware planning.

Good explanation:

- it is not only a generic study suggestion generator
- it tries to break learning into smaller executable pieces
- the goal is clearer scope and more controllable progress

## 2. Personalized Execution

The site stresses:

- no bloated essay-like plans
- steady execution
- daily practice, weekly checks, monthly exams

Good explanation:

- the point is not only to know what to study
- the point is to convert learning into repeatable execution

## 3. Learning Clone

The site presents this as a digital clone built around the learner's thinking style and progress, helping with delegated practice, difficulty breakdown, and reduced wasted training.

Do not explain it as a generic assistant or ordinary agent.

## 4. True Deep Learning

The site also emphasizes discussion, interaction, and deeper understanding.

Good explanation:

- not only more questions
- not only answer memorization
- also discussion, reflection, and higher-level understanding

## Official Three-Part Frame

Use this quick frame when the user wants a compact explanation:

- `AI Teach`: break knowledge into smaller units and provide multi-level explanations
- `AI Learn`: adaptive paths and on-demand clarification
- `AI Practice`: learning clone supported drills, past-paper style work, and discussion reinforcement

## Tone Guidance

Be:

- concrete
- restrained
- user-outcome focused

Avoid:

- inventing capabilities not stated publicly
- claiming pricing or delivery details that are not public
