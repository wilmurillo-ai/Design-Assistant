## 详细版本变更说明

### V1.8.1 (发布时间：2026-03-23)
- **算法平台**: 优化用户信息查询，支持接口全量返回（含 `record palm` 下全部字段）。
- **M4 设备**: 
  - 录掌交互输入方式统一为纯数字，降低误触。
  - 采用 9 宫格布局，放大按键面积，适配 M4 的 5 寸屏。
  - 在 User ID 后新增 PIN 码字段，实现双重验证。

### V1.8.0 (发布时间：2026-03-02)
- **算法平台**: 新增算法管理，针对老年人专项算法优化/阈值调整；支持掌特征变化，实现不同客户/设备的特征隔离。
- **场景策略**: 场景管理中支持绑定场景策略，按场景下发配置到设备生效。
- **交付运维**: 新增运维指引（提供单机版和 TKE 版本的运维方案）；优化问题排查，支持设备端日志收集上报和常见问题排查工具。
- **版本升级**: 支持 Standard 到 Max 版本的升级和数据迁移方案。

### V1.7.2 (发布时间：2026-01-21)
- **M3 运维**: 新增与系统层守护进程的通信能力，支持通过守护进程升级应用。
- **M4 运维**: 新增系统最新版本号解析能力。
- **IoT 平台**: 优化 OTA 机制，在某些错误状态下仍能继续升级。
- **管理后台**: 新增角色与权限管理，提供 5 类内置角色并支持分配权限。

### V1.7.1 (发布时间：2026-01-20)
- **M4 测试**: 新增 TELCOM 测试环境。
- **管理后台**: 新增系统配置，可关闭“刷掌记录异步推送”。
- **API**: OpenAPI 创建核验规则接口，默认 `EnableWebhookRule` 为 true。

### V1.7 (发布时间：2026-01-20)
- **管理后台**:
  - 新增 M4 设备模板管理（自定义音效、文案、图片等），支持按场景/设备设置。
  - 场景管理允许自定义场景 ID。
  - 手掌注册优化，支持服务端检索去重。
- **OpenAPI**: 新增用户标签增删改查 API；支持掌库容量和设备有效期查询。
- **管理体验**: 优化核验规则搜索、防重配置、批量导入模板校验逻辑、分页控件修复等。
- **IoT 平台 OTA**: 新增反复 OTA 升级能力；支持指定设备静默升级；支持固件/模组/应用/算法独立及依赖版本升级。
- **设备修复与优化 (M3/M4/A3)**:
  - 修复海量数据下页面卡死、用户数据联动异常、界面显示冲突等问题。
  - 完善网络功能（无密码 Wi-Fi、网络重置等）。
  - M3：预置 APP 守护进程至系统固件；系统固件版本号更新至 `1.0.1.20251226`。
  - A3 模组：优化设备待机温升，修复偶现持续重启问题。
- **交付运维**: 支持 TCS 底座部署；支持 License 失效监控预警；通过 Grafana/Prometheus 实现大盘监控；支持运维邮件推送。

### V1.6 (发布时间：2025-12-15)
- **管理后台**: 新增第三方系统核验（适用于预约签到、门票核验等）；优化批量操作体验、提示语多语言规范。
- **OpenAPI**: 新增核验记录 API；新增设备管理 API、场景管理 API。
- **移动端**: 新增手机核验能力（支持 1:1 身份核验，打通管理后台核验记录）；支持手机注册/查询状态及预录入手掌过期时间显示。
- **设备端 (M3/M4/A3)**:
  - M3 固件支持动态调整硬件接口模式；优化静态 Wi-Fi 配置流程。
  - A3 模组算法优化（5W库规模下 FRR=0.1%, FAR=0.01%）；支持标准 QR code 靠近即识别。
  - 刷掌比对耗时优化（平均 <800ms）；特征全量同步耗时优化。
- **交付运维**: 支持 Azure 集群版的运维监控、日志分析。

### V1.5 (发布时间：2025-11-03)
- **管理后台**: 支持按核验规则下发设备端掌库；优化删除场景后的记录展示。
- **设备端 (M3/M4)**: 新增“模式切换”指令，支持在“注册模式”和“识别模式”间切换；优化核验成功音效。
- **交付运维**: 新增私有化单机版运维监控。

### V1.4 (发布时间：2025-10-20)
- **管理后台**: 支持核验规则管理（按时间、次数、标签等限制人员通行）。
- **设备端 (M4)**: 识别模式中融入核验规则逻辑；支持设备每日定时重启、清空掌库等系统设置。
- **SDK**: 空中录掌 SDK 支持繁体中文及错误文案优化。

### V1.3 (发布时间：2025-10-09)
- **管理后台**: 新增用户标签管理（页面直接设置或批量导入）。
- **设备端 (M3/M4)**: 
  - 支持 H5扫码录掌、外接设备录掌、设备端录掌及空中录掌。
  - 识别成功后可上传记录，离线核验自动上传。
  - M4新增通讯协议唤起支付；M3支持 M1 刷卡、韦根或继电器输出。
  - 设备端系统支持有线/无线网、OTA 升级、开发者日志上传及扫码激活。
- **SDK & API**: 空中录掌 SDK 完成网关鉴权改造；新增核验规则 API；适配纯 CPU 版本的服务部署。

### V1.2 (发布时间：2025-07-31)
- **首个对外版本**: 整合设备管理、用户管理、场景管理、核验记录、系统设置（多语言支持/自定义Logo）。
- **SDK & Demo**: 提供 iOS/安卓原生 SDK 及测试用 Demo APP；提供 Windows/Mac 外接录掌演示应用。
- **API**: 提供全套基础用户管理及核验记录获取/推送 API。
