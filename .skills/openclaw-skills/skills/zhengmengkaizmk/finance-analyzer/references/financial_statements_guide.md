# 财务报表结构与指标参考

## 一、三大财务报表结构

### 1. 损益表（利润表 / Income Statement）

损益表反映企业在一定会计期间的经营成果。核心结构如下：

| 项目 | 英文名 | 说明 |
|------|--------|------|
| 营业收入 | Revenue / Net Sales | 企业主营业务产生的收入 |
| 营业成本 | Cost of Revenue / COGS | 生产商品或提供服务的直接成本 |
| **毛利润** | **Gross Profit** | **= 营业收入 - 营业成本** |
| 销售费用 | Selling Expenses | 销售商品过程中发生的费用 |
| 管理费用 | Administrative Expenses | 管理活动产生的费用 |
| 研发费用 | R&D Expenses | 研究与开发活动的费用 |
| 财务费用 | Financial Expenses | 融资活动产生的利息等费用 |
| 营业利润 | Operating Income / EBIT | = 毛利润 - 各项费用 + 其他收益 |
| 利润总额 | Profit Before Tax | = 营业利润 + 营业外收支净额 |
| 所得税费用 | Income Tax Expense | 应缴纳的企业所得税 |
| **净利润** | **Net Income** | **= 利润总额 - 所得税费用** |

### 2. 资产负债表（Balance Sheet）

资产负债表反映企业在特定日期的财务状况。核心等式：**资产 = 负债 + 所有者权益**

| 项目 | 英文名 | 说明 |
|------|--------|------|
| 流动资产合计 | Total Current Assets | 预期在一年内变现的资产 |
| 非流动资产合计 | Total Non-Current Assets | 长期使用的资产 |
| **资产总计** | **Total Assets** | **= 流动资产 + 非流动资产** |
| 流动负债合计 | Total Current Liabilities | 一年内需偿还的负债 |
| 非流动负债合计 | Total Non-Current Liabilities | 长期负债 |
| **负债合计** | **Total Liabilities** | **= 流动负债 + 非流动负债** |
| **所有者权益合计** | **Total Equity** | **= 资产总计 - 负债合计** |

### 3. 现金流量表（Cash Flow Statement）

现金流量表反映企业现金的流入和流出情况。分为三大类：

| 项目 | 英文名 | 说明 |
|------|--------|------|
| 经营活动现金流量净额 | Operating Cash Flow | 日常经营活动产生的净现金 |
| 投资活动现金流量净额 | Investing Cash Flow | 购买/出售资产等投资活动产生的净现金 |
| 筹资活动现金流量净额 | Financing Cash Flow | 借款、还款、分红等融资活动产生的净现金 |

## 二、核心财务指标

### 盈利能力指标

| 指标 | 公式 | 含义 | 一般标准 |
|------|------|------|----------|
| **ROE（净资产收益率）** | 净利润 / 股东权益 × 100% | 衡量股东投入资本的回报率 | >15% 优秀，10-15% 良好 |
| **ROA（总资产收益率）** | 净利润 / 总资产 × 100% | 衡量企业利用全部资产的盈利能力 | >5% 较好，行业差异大 |
| **毛利率** | 毛利润 / 营业收入 × 100% | 衡量产品/服务的基础盈利能力 | 行业差异大，越高越好 |
| **净利率** | 净利润 / 营业收入 × 100% | 衡量企业最终的盈利能力 | >10% 较好，行业差异大 |

### 偿债能力指标

| 指标 | 公式 | 含义 | 一般标准 |
|------|------|------|----------|
| **流动比率** | 流动资产 / 流动负债 | 衡量短期偿债能力 | 1.5-2.0 较为安全 |
| **资产负债率** | 总负债 / 总资产 × 100% | 衡量企业负债程度 | 40-60% 适中 |

### 杜邦分析体系

ROE 可以分解为三个驱动因素：

```
ROE = 净利率 × 总资产周转率 × 权益乘数
    = (净利润/收入) × (收入/总资产) × (总资产/股东权益)
```

- **净利率**：反映盈利能力
- **总资产周转率**：反映运营效率
- **权益乘数**：反映财务杠杆

## 三、常见 Excel 报表格式

### 中文报表常见格式
- 第一列通常为科目名称
- 后续列为不同期间（如"本期金额"、"上期金额"，或具体年月）
- 可能存在合并行、小计行
- Sheet 名称通常包含"利润"、"资产负债"、"现金流"等关键字

### 英文报表常见格式
- 第一列为 Line Item 名称
- 后续列为期间（如 FY2023, FY2024, Q1 2024 等）
- Sheet 名称通常包含 "Income", "Balance", "Cash Flow" 等关键字

### 数据解读注意事项
- 金额单位：注意是否为"万元"、"百万元"或"元"
- 括号表示负数：(100) 代表 -100
- 百分比与绝对值的区分
- 合并报表与母公司报表的区分
