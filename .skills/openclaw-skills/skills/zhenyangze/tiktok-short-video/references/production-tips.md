# TikTok 短视频制作技巧

## 视频拍摄基础

### 设备要求
```
【最低配置】
├─ 手机: 最新款智能手机
├─ 稳定器: 手机支架
└─ 补光灯: LED补光灯

【推荐配置】
├─ 手机: iPhone 14+/安卓旗舰
├─ 稳定器: 手机云台/三脚架
├─ 补光: 主灯+补光灯+背景灯
├─ 收音: 领夹麦克风/无线麦
└─ 背景: 纯色/渐变背景纸
```

### 拍摄角度
| 角度 | 适用场景 | 效果 |
|------|---------|------|
| 平视 | 常规对话 | 自然亲切 |
| 俯视 | 展示产品 | 专业展示 |
| 仰视 | 权威感 | 增强气场 |
| 45度角 | 美颜效果 | 显瘦显高 |

## 剪辑技巧

### 常用转场
- 硬切: 直接切换，节奏感强
- 淡入淡出: 柔和过渡
- 缩放: 强调重点
- 旋转: 增加动感

### 节奏控制
```
【快节奏】
├─ 切换间隔: 1-2秒
├─ 适用: 娱乐、搞笑
└── 氛围: 活泼、刺激

【慢节奏】
├─ 切换间隔: 3-5秒
├─ 适用: 情感、故事
└── 氛围: 温馨、治愈
```

### 字幕使用
- 字体: 清晰易读的黑体/圆体
- 大小: 适配手机屏幕
- 位置: 屏幕下方1/3
- 颜色: 白字+黑边 (万能对比)

## BGM 选择技巧

### BGM匹配原则
```
【情绪匹配】
├─ 欢快内容 → 动感BGM
├─ 情感内容 → 温馨BGM
├─ 悬疑内容 → 紧张BGM
└── 励志内容 → 激昂BGM

【节奏匹配】
├─ 快剪 → 快节奏BGM
├─ 慢剪 → 慢节奏BGM
└── 卡点剪辑 → 强节奏BGM
```

### 热门BGM来源
- TikTok 热歌榜
- 抖音神曲
- 网红歌曲片段
- 影视金曲
