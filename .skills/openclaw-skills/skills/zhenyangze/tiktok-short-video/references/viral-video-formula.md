# TikTok 爆款视频公式

## 爆款视频底层逻辑

### 流量推荐机制

```
视频发布 → 初始流量池 (200-500) → 数据反馈 → 下一级流量池
    ↓
完播率 > 互动率 > 分享率 > 关注率
```

### 核心指标权重

| 指标 | 权重 | 优化方向 |
|------|------|----------|
| 完播率 | 40% | 内容质量、节奏把控 |
| 互动率 | 30% | 引导评论、点赞 |
| 分享率 | 20% | 价值输出、情感共鸣 |
| 关注率 | 10% | 人设吸引、内容预期 |

## 爆款公式

### 公式 1: 痛点 + 方案

```\n
爆款 = 共同痛点 + 意外方案 + 效果验证
```

**示例**:
- 痛点："每天都在为 X 烦恼？"
- 方案："试试这个方法..."
- 验证："看！问题解决了！"

**适用**: 教程类、产品种草

---

### 公式 2: 反差 + 反转

```\n
爆款 = 预期铺垫 + 意外反转 + 情绪释放
```

**示例**:
- 铺垫："以为又是智商税..."
- 反转："结果真香了！"
- 释放："现在离不开它了"

**适用**: 测评类、剧情种草

---

### 公式 3: 数字 + 结果

```爆款 = 具体数字 + 时间 + 惊人结果```

**示例**:
- "30 天瘦 20 斤，我只做了这 3 件事"
- "7 天粉丝破 10 万，方法分享给你"
- "省钱攻略：100 元过一周"

**适用**: 经验分享、干货类

---

### 公式 4: 身份 + 秘密

```爆款 = 权威身份 + 行业内幕/秘密
```

**示例**:
- "作为皮肤科医生，我想说..."
- "10 年配方师告诉你真相"
- "柜姐不会告诉你的事"

**适用**: 专业科普、行业内幕

---

### 公式 5: 挑战 + 围观

```爆款 = 挑战目标 + 过程记录 + 最终结果
```

**示例**:
- "挑战 100 元改造房间"
- "连续用 X 产品 30 天，效果是..."
- "不花钱过一周，可能吗？"

**适用**: 挑战类、记录类

## 热门 BGM 选择

### BGM 类型

| 类型 | 适用场景 | 特点 |
|------|---------|------|
| 卡点音乐 | 变装、展示 | 节奏感强 |
| 轻音乐 | 教程、科普 | 不抢注意力 |
| 热门歌曲 | 日常、Vlog | 自带流量 |
| 音效 | 转折、强调 | 突出关键点 |

### BGM 使用技巧

- 音量控制在 -18dB ~ -12dB
- 高潮部分配合视频高潮
- 卡点视频精确到帧
- 关注热门音乐榜单

## 标签策略

### 标签组合

```
3 个大词标签 (1 亿 + 播放)
  + 3 个中词标签 (1000 万 -1 亿播放)
  + 2 个长尾标签 (100 万 -1000 万播放)
  + 1 个品牌/个人标签
```

### 热门标签

**通用**:
- #TikTokMadeMeBuyIt
- #MustHave
- #LifeHacks
- #Viral
- #ForYou

**类目**:
- 美妆：#Beauty、#Makeup、#Skincare
- 服饰：#OOTD、#Fashion、#Style
- 家居：#HomeDecor、#HomeHacks
- 美食：#FoodTok、#Recipe、#Cooking

## 数据复盘

### 关键指标

| 指标 | 健康值 |
|------|--------|
| 完播率 | > 30% |
| 互动率 | > 5% |
| 分享率 | > 1% |
| 关注转化率 | > 0.5% |

### 优化方向

- 完播率低 → 优化开头、精简内容
- 互动率低 → 增加互动引导
- 分享率低 → 增强价值输出
- 关注低 → 强化人设、内容预期
