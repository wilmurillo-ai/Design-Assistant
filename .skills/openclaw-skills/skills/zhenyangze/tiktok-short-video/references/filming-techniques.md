## 布光技巧

### 三点布光法

```
        主光
         ↓
    主体 ← 辅光
         ↓
       轮廓光
```

| 灯光 | 位置 | 作用 |
|------|------|------|
| 主光 | 前侧 45° | 主要照明 |
| 辅光 | 前侧另一方 | 补阴影 |
| 轮廓光 | 后方 | 勾勒轮廓 |

### 常见布光场景

**产品展示**:
- 45°侧光突出质感
- 底光消除阴影
- 背光勾勒轮廓

**人物拍摄**:
- 正面柔光美化皮肤
- 侧光塑造立体感
- 发灯增加层次

## 运镜技巧

### 基础运镜

| 运镜 | 操作 | 效果 |
|------|------|------|
| 推 | 镜头靠近主体 | 聚焦、强调 |
| 拉 | 镜头远离主体 | 释放、交代 |
| 摇 | 左右/上下转动 | 展示环境 |
| 移 | 横向/纵向移动 | 跟随主体 |
| 环绕 | 绕主体旋转 | 立体展示 |

### 运镜组合

**产品展示组合**:
```
1. 环绕运镜 → 展示全貌
2. 推进特写 → 突出细节
3. 拉远全景 → 整体效果
```

**变装视频组合**:
```
1. 推进遮挡 → 换装
2. 拉远展示 → 新造型
3. 快速切换 → 多套服装
```

## 常见问题解决

### 画面抖动

**原因**: 手持不稳
**解决**:
- 使用三脚架
- 开启防抖功能
- 双手持机

### 画面模糊

**原因**: 对焦不准/手抖
**解决**:
- 锁定对焦
- 使用稳定器
- 提高快门速度

### 曝光不准

**原因**: 光线复杂
**解决**:
- 锁定曝光
- 使用测光工具
- 后期调整

### 颜色失真

**原因**: 白平衡不准
**解决**:
- 手动设置白平衡
- 使用灰卡校准
- 后期校色
