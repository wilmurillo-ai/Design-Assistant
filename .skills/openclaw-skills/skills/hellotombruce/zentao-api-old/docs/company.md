# 公司 (Company) API 文档

## 概述
公司管理相关的API接口。

## API 列表

### 公司首页

| 属性 | 值 |
|------|-----|
| 方法 | GET |
| 路径 | `/company.json` |
| 描述 | 公司首页 |

**参数列表**

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| 无参数 | - | - | - |

---

### 浏览公司

| 属性 | 值 |
|------|-----|
| 方法 | GET |
| 路径 | `/company-browse-[deptID].json` |
| 描述 | 浏览公司部门结构 |

**参数列表**

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| deptID | int | 否 | 部门ID |

---
