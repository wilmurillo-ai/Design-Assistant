### 切换定时任务执行状态

| 属性 | 值 |
|------|-----|
| 方法 | GET |
| 路径 | `/cron-toggle-[cronID]-[status].json` |
| 描述 | 启动或停止定时任务 |

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| cronID | int | 否 | 定时任务ID |
| status | int | 否 | 状态（1:启动，0:停止） |

---
