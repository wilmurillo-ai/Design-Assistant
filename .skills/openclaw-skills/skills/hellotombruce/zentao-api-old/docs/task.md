# 任务 (Task) API 文档

## 概述
任务管理相关的API接口。

## API 列表

### 创建任务

| 属性 | 值 |
|------|-----|
| 方法 | GET/POST |
| 路径 | `/task-create-[projectID]-[storyID]-[moduleID]-[taskID]-[todoID].json` |
| 描述 | 创建新任务 |

**参数列表**

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| projectID | int | 否 | 项目ID |
| storyID | int | 否 | 需求ID |
| moduleID | int | 否 | 模块ID |
| taskID | int | 否 | 父任务ID |
| todoID | int | 否 | 待办ID |

---
