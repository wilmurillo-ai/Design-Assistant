# 自定义 (Custom) API 文档

## 概述
自定义设置相关的API接口。

## API 列表

### 自定义首页

| 属性 | 值 |
|------|-----|
| 方法 | GET |
| 路径 | `/custom.json` |
| 描述 | 自定义设置首页 |

**参数列表**

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| 无参数 | - | - | - |

---

### 模块设置

| 属性 | 值 |
|------|-----|
| 方法 | GET/POST |
| 路径 | `/custom-module-[module].json` |
| 描述 | 自定义模块设置 |

**参数列表**

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| module | string | 否 | 模块名称 |
