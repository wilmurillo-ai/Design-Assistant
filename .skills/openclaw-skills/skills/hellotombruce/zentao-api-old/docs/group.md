### 编辑用户组

| 属性 | 值 |
|------|-----|
| 方法 | GET/POST |
| 路径 | `/group-edit-[groupID].json` |

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|

### 删除用户组

| 属性 | 值 |
|------|-----|
| 方法 | GET |
| 路径 | `/group-delete-[groupID]-[confirm].json` |
| 描述 | 删除用户组 |

**参数列表**

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| groupID | int | 否 | 用户组ID |
| confirm | string | 否 | 确认删除 |
