# 需求 (Story) API 文档

## 概述
需求管理相关的API接口。

## API 列表

### 创建需求

| 属性 | 值 |
|------|-----|
| 方法 | GET/POST |
| 路径 | `/story-create-[productID]-[branch]-[moduleID]-[storyID]-[projectID]-[bugID]-[planID]-[todoID]-[extra]-[type].json` |
| 描述 | 创建新的需求 |

**参数列表**

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| productID | int | 否 | 产品ID |
| branch | int | 否 | 分支ID |
| moduleID | int | 否 | 模块ID |
| storyID | int | 否 | 父需求ID |
| projectID | int | 否 | 项目ID |
| bugID | int | 否 | BugID |
| planID | int | 否 | 计划ID |
| todoID | int | 否 | 待办ID |
| extra | - | 否 | 额外参数 |
| type | string | 否 | 需求类型 |

---
