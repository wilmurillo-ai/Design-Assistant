# 部门 (Dept) API 文档

## 概述
部门管理相关的API接口。

## API 列表

### 浏览部门

| 属性 | 值 |
|------|-----|
| 方法 | GET |
| 路径 | `/dept-browse-[deptID].json` |
| 描述 | 浏览部门详情及其子部门 |

**参数列表**

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| deptID | int | 否 | 部门ID |

---
