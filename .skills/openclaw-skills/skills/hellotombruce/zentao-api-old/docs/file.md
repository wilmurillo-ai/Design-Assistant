# 附件 (File) API 文档

## 概述
附件管理相关的API接口。

## API 列表

### 构建上传表单

| 属性 | 值 |
|------|-----|
| 方法 | GET |
| 路径 | `/file-buildForm-[fileCount]-[percent]-[filesName]-[labelsName].json` |
| 描述 | 构建文件上传表单 |

**参数列表**

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| fileCount | - | 否 | 文件数量 |
| percent | - | 否 | 上传进度百分比 |
| filesName | - | 否 | 文件名称 |
| labelsName | - | 否 | 标签名称 |

---
