### 邮件日志

| 属性 | 值 |
|------|-----|
| 方法 | GET |
| 路径 | `/mail-log-[recTotal]-[recPerPage]-[pageID].json` |
| 描述 | 查看邮件发送日志 |

**参数列表**

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| recTotal | int | 否 | 总记录数 |
| recPerPage | int | 否 | 每页记录数 |
| pageID | int | 否 | 页码 |

---
