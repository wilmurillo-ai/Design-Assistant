# 待办 (Todo) API 文档

## 概述
待办事项管理相关的API接口。

## API 列表

### 创建待办

| 属性 | 值 |
|------|-----|
| 方法 | GET/POST |
| 路径 | `/todo-create-[date]-[account].json` |
| 描述 | 创建待办事项 |

**参数列表**

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| date | string/date | 否 | 日期 |
| account | string | 否 | 用户账号 |

---
