# Caselib API 文档

## 概述
用例库管理相关的API接口。

## API 列表

### 用例库首页

| 属性 | 值 |
|------|-----|
| 方法 | GET |
| 路径 | `/caselib.json` |
| 描述 | 用例库首页 |

**参数列表**

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| 无参数 | - | - | - |

---

### 浏览用例库

| 属性 | 值 |
|------|-----|
| 方法 | GET |
| 路径 | `/caselib-browse-[libID]-[orderBy].json` |
| 描述 | 浏览用例库 |

**参数列表**

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| libID | int | 否 | 用例库ID |
| orderBy | string | 否 | 排序方式 |
