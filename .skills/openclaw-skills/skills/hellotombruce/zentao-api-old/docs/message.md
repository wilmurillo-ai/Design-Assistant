# 消息 (Message) API 文档

## 概述
消息管理相关的API接口。

## API 列表

### 消息首页

| 属性 | 值 |
|------|-----|
| 方法 | GET |
| 路径 | `/message.json` |
| 描述 | 消息首页 |

**参数列表**

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| 无参数 | - | - | - |

---

### 浏览消息

| 属性 | 值 |
|------|-----|
| 方法 | GET |
| 路径 | `/message-browse-[type]-[recTotal]-[recPerPage]-[pageID].json` |
| 描述 | 浏览消息列表 |

**参数列表**

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| type | string | 否 | 消息类型 |
| recTotal | int | 否 | 总记录数 |
| recPerPage | int | 否 | 每页记录数 |
| pageID | int | 否 | 页码 |

---
