# WMS 模块字段规范

## 仓管人员

### 筛选条件
| 字段 | 类型 | 说明 |
|------|------|------|
| 姓名/工号 | 搜索框 | 模糊搜索 |
| 所属仓库 | 下拉选择 | 全部/北京仓/上海仓/广州仓/成都仓 |
| 班组 | 下拉选择 | 全部/一班/二班/三班 |
| 用工形式 | 下拉选择 | 全部/正式/外包/实习 |
| 岗位类型 | 下拉选择 | 全部/仓管员/复核员/分拣员/配送员 |
| 状态 | 下拉选择 | 全部/在职/离职 |

### 列表字段
| 字段 | 说明 |
|------|------|
| 工号 | monospace字体，如 `EMP-001` |
| 姓名 | 加粗显示 |
| 性别 | 显示图标或文字 |
| 岗位 | 标签样式 |
| 用工形式 | 正式/外包/实习 |
| 所属仓库 | 仓库名称 |
| 所属班组 | 班组名称 |
| 联系方式 | 手机号脱敏 `138-****-0001` |
| 入职日期 | YYYY-MM-DD |
| 转正日期 | YYYY-MM-DD |
| 状态 | 在职/离职 |
| 操作 | 详情/编辑 |
