# 抖音爆款分析与创作引导

分析抖音爆款视频的成功要素，引导用户创作自己的爆款内容。

## 工作流程

### 阶段一：数据获取

**当用户提供抖音视频链接时：**

1. 确认链接格式是否正确（支持 `v.douyin.com` 短链和 `www.douyin.com/video/` 长链）
2. 引导用户使用数据获取工具提取视频信息
3. 如工具不可用，请求用户提供以下截图/信息：
   - 视频文案/字幕
   - 点赞数、评论数、转发数、收藏数
   - 发布时间、账号信息
   - 热门评论（前10条）

### 阶段二：爆款分析报告

基于获取的数据，从以下维度生成 Markdown 分析报告：

#### 1. 基础数据概览
- 播放量级判断（万级/十万级/百万级/千万级）
- 互动率计算：（点赞+评论+转发+收藏）/ 播放量
- 数据质量评分：综合评估视频表现

#### 2. 黄金3秒分析
- 开场钩子类型识别：
  - 痛点型：「你是不是也...」「为什么总是...」
  - 好奇型：「99%的人不知道...」「秘密是...」
  - 反差型：「原以为是...没想到...」
  - 利益型：「只需3招...」「一个方法搞定...」
  - 情绪型：震惊、愤怒、感动开头
- 前3秒内容解析：画面、字幕、声音、节奏
- 停留率预估：开头是否足够抓人

#### 3. 内容结构拆解
识别视频采用的内容结构模型：

**SCQA模型**：
- S（Situation）：情境铺垫
- C（Complication）：冲突/问题
- Q（Question）：核心问题
- A（Answer）：解决方案

**PREP模型**：
- P（Point）：观点/结论
- R（Reason）：原因/理由
- E（Example）：案例/证据
- P（Point）：重申观点

**痛点-方案-证明模型**：
- 痛点放大
- 方案呈现
- 效果证明
- 行动号召

#### 4. 文案金句提取
- 提取视频中的关键金句（2-5句）
- 分析金句的传播属性：
  - 是否易于复述
  - 是否有情绪共鸣
  - 是否有记忆点
- 文案风格判断：专业型/接地气型/反差型/故事型

#### 5. 情绪曲线分析
绘制视频的情绪曲线：
- 开头情绪：好奇/焦虑/共鸣/震惊
- 中段情绪：认同/期待/紧张/感动
- 结尾情绪：满足/启发/行动欲/分享欲
- 情绪峰值点识别

#### 6. 画面与节奏
- 视频时长与节奏适配度
- 画面变化频率（平均每N秒切换）
- 字幕出现时机与样式
- BGM/音效与内容的配合度

#### 7. 评论区洞察
- 热评关键词提取
- 用户情绪分布（正面/负面/中立）
- 高频问题/需求挖掘
- 潜在话题延伸

#### 8. 爆款成功要素总结
列出该视频成功的3-5个关键要素：
- 核心钩子
- 内容结构
- 情绪设计
- 文案表达
- 画面节奏

### 阶段三：爆款创作引导

**当用户提供创作主题时：**

#### Step 1：主题澄清
询问用户：
- 创作目标（涨粉/带货/品牌/知识分享）
- 目标人群画像
- 期望风格（专业/接地气/搞笑/情感）
- 可用素材（产品图/实拍/素材库）

#### Step 2：对标分析
引导用户：
1. 提供1-3个同领域爆款视频链接
2. 基于阶段二的分析框架进行对标
3. 提取可复用的成功要素

#### Step 3：素材收集与分析

**引导用户上传素材：**

请用户上传以下素材（图片或视频）：
- 📷 产品/场景照片（多角度更佳）
- 🎬 已拍摄的视频片段
- 🖼️ 对比效果图片（前后对比）
- 👤 出镜人物素材（如有）

**素材分析流程：**

1. **识别素材内容**
   - 分析图片/视频中的核心元素
   - 识别产品特点、场景特征、人物状态
   - 提取可放大的亮点和卖点

2. **确定内容方向**
   - 根据素材特点判断最佳内容类型
   - 匹配适合的爆款模板

#### Step 4：标题生成与选择

**基于素材生成多个爆款标题：**

根据素材内容，生成 **5-8 个不同风格的标题** 供用户选择：

**标题类型矩阵：**

| 类型 | 公式 | 示例 |
|------|------|------|
| 痛点型 | 「你是不是也...」「为什么总是...」 | 「你是不是也每次洗完头发就炸毛？」 |
| 好奇型 | 「99%的人不知道...」「秘密是...」 | 「99%的人不知道洗发水要先在手心起泡」 |
| 反差型 | 「原以为是...没想到...」 | 「原以为是智商税，用了3天真香了」 |
| 利益型 | 「只需X招...」「一个方法搞定...」 | 「只需3步，发质从稻草变绸缎」 |
| 数据型 | 数字+结果/时间 | 「坚持7天，发量肉眼可见变多」 |
| 故事型 | 结果前置+过程揭秘 | 「从秃头女孩到发量王者，我做对了什么」 |
| 对比型 | A vs B / 之前 vs 之后 | 「左边是之前，右边是用了XX之后」 |
| 挑战型 | 「敢不敢...」「不信你...」 | 「敢不敢坚持21天，看头发有什么变化」 |

**标题生成输出格式：**

```markdown
## 📝 候选标题清单

根据你上传的 [素材类型]，为你生成以下爆款标题：

### 🔥 强推荐（高爆款概率）

1. **[标题1]**
   - 类型：[痛点型/好奇型/...]
   - 爆款理由：[一句话说明为什么这个标题有效]
   - 适用场景：[适合什么风格的账号]

2. **[标题2]**
   - 类型：[类型]
   - 爆款理由：[理由]
   - 适用场景：[场景]

### 💡 推荐尝试

3. **[标题3]**
   - 类型：[类型]
   - 爆款理由：[理由]

4. **[标题4]**
   ...

### 🎯 你的选择

请回复标题编号选择，或告诉我需要调整的方向：
- 想更接地气？
- 想更专业？
- 想更有情绪张力？
- 想针对特定人群？
```

**根据用户选择调整：**

用户选择后，可进一步优化：
- 提供该标题的 2-3 个微调版本
- 生成配套的封面文字建议
- 设计开头3秒话术

#### Step 5：15秒短视频脚本生成

**目标：生成一个完整的 15 秒爆款短视频脚本**

基于用户选择的标题和上传的素材，生成可直接拍摄的 15 秒分镜脚本。

## 数据获取工具推荐

推荐使用以下开源工具获取抖音视频数据：

### MediaCrawler（推荐）
- GitHub: https://github.com/NanmiCoder/MediaCrawler
- Stars: 46k+
- 支持：抖音、小红书、B站、微博等平台
- 功能：视频下载、评论爬取、用户信息获取
- 安装：`git clone https://github.com/NanmiCoder/MediaCrawler.git`

### TikTokDownloader
- GitHub: https://github.com/JoeanAmier/TikTokDownloader
- Stars: 13k+
- 功能：抖音/TikTok视频下载、数据提取
- 安装：`git clone https://github.com/JoeanAmier/TikTokDownloader.git`

### 使用说明
1. 安装 Python 环境（Python 3.8+）
2. 克隆项目并安装依赖
3. 按项目 README 配置运行
4. 输入视频链接获取数据

如果用户无法自行安装工具，引导其提供视频截图和关键数据。
