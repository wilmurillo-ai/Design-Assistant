# 录音文件识别标准版使用指南

## 方案优势
- 价格更低：约2.5元/小时，相比极速版4.5元/小时节省44%成本
- 支持更长音频：最长可支持10小时音频识别（极速版为2小时）
- 适合批量处理：适合大量音频文件批量转写场景

## 开通步骤
1. **开通TOS对象存储服务**
   - 访问火山引擎控制台，开通「对象存储TOS」服务
   - 创建一个私有存储桶，选择就近地域，开启服务端加密
   - 配置存储桶生命周期规则：上传后30天自动删除文件，避免不必要的存储费用

2. **开通录音文件识别标准版服务**
   - 访问火山引擎控制台，开通「智能语音服务」->「录音文件识别2.0」->「标准版」
   - 获取API访问密钥（AccessKey / SecretKey），确保密钥同时拥有TOS和语音服务的权限

3. **配置权限**
   - 给API密钥授予TOS存储桶的上传权限和文件读取权限
   - 确保语音服务有读取TOS文件的权限（可以通过设置存储桶策略允许语音服务账号访问）

## 调用流程
1. 上传音频文件到TOS存储桶
2. 生成TOS文件的预签名URL，有效期设置为1小时
3. 调用录音文件识别标准版API，传入预签名URL作为音频地址
4. 轮询识别结果，完成后生成会议纪要
5. TOS中的文件会根据生命周期规则自动在30天后删除，无需手动清理

## 价格说明
- 录音文件识别标准版：2.5元/小时（不满1小时按实际时长折算）
- TOS存储费用：0.12元/GB/月，流量费用0.5元/GB（内网访问免费）
- 综合成本比极速版降低约40%，适合大用量场景
