# Stock Checker

Fetches real-time stock prices by ticker symbol using free public APIs.

## Quick Start

```
Check the stock price for AAPL
```

## Parameters

| Parameter | Required | Description |
|-----------|----------|-------------|
| `ticker` | Yes | Stock ticker symbol (e.g., `AAPL`, `TSLA`, `GOOGL`) |

## Example Usage

**Single stock:**
```
What's the current price of TSLA?
```
→ **TSLA** — $248.50 USD | Change: +$3.20 (+1.30%) | Volume: 52.3M

**Quick comparison:**
```
Check stock prices for AAPL, MSFT, and GOOGL
```

## Setup

No setup required. Uses free Yahoo Finance API — no API key needed.

## How It Works

1. Validates the ticker symbol format
2. Hits the Yahoo Finance chart API for 1-day data
3. Extracts current price, previous close, and volume
4. Calculates daily change and percentage
5. Returns formatted summary

## Limitations

- Only supports US-listed stocks
- Data may be delayed 15-20 minutes during market hours
- Crypto tickers (BTC-USD) may work but are not officially supported
- No historical data — current price only
