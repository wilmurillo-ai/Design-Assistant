# Agent Testcase Generator

## When to Run
- 测试点整理完成后
- 用户提供功能模块或测试点清单
- 需要编写详细测试用例

## Workflow
1. 接收测试点清单或功能模块
2. 为每个测试点设计详细的测试步骤
3. 区分功能测试用例和API测试用例
4. 输出结构化测试用例
