# 🎰 彩票预测技能 V2.15

基于高等数学模型的双色球预测工具

## 功能特点

- ✅ 7 种数学模型综合预测
- ✅ 基于 3430+ 期历史数据
- ✅ 纯离线分析，无 API 调用
- ✅ 自动防错（阻止跨期预测）
- ✅ 输出可解释性报告

## 安装方法

```bash
# 方法 1：使用 openclaw 命令
openclaw skill install lottery-predictor-v2.15

# 方法 2：手动安装
cp -r ~/.openclaw/workspace/skills/lottery-predictor-v2.15 ~/.openclaw/skills/
```

## 使用示例

### 预测下一期

```
帮我预测双色球 2026035 期
```

### 使用技能命令

```
/lottery-predictor-v2.15 predict --issue 2026035
```

### 验证准确率

```
/lottery-predictor-v2.15 verify --issue 2026034 --actual 3,6,13,21,28,29,6
```

## 输出说明

### 红球推荐
- TOP10 号码池（按概率排序）
- 第一梯队（6 码）：均值回归 + 正态分布
- 第二梯队（4 码）：大数定律 + 卡方检验

### 蓝球推荐
- TOP4 号码（按概率排序）

### 推荐组合
- 第 1 组：均值回归组合
- 第 2 组：正态分布组合
- 第 3 组：大数定律组合

## 核心算法

| 模型 | 权重 | 说明 |
|------|------|------|
| 均值回归 | 25% | 极端值向平均值回归 |
| 正态分布 | 22% | 和值分布预测 |
| 大数定律 | 18% | 历史频率估计 |
| 卡方检验 | 15% | 冷热号判断 |
| 马尔可夫链 | 8% | 状态转移概率 |
| 时间序列 | 7% | 近期趋势分析 |
| 泊松分布 | 5% | 连号/同尾号概率 |
