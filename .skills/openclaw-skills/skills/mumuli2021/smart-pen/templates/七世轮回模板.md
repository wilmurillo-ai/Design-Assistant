# 七世轮回模板

## 📖 模板说明

适用于多世轮回题材的言情小说，可调整为三世、五世、九世等。

---

## 🏗️ 整体框架

### 标准结构（44-45 回）
```
楔子（2 回）→ 第一世（6 回）→ 第二世（6 回）→ 第三世（6 回）→ 
第四世（6 回）→ 第五世（6 回）→ 第六世（6 回）→ 第七世（6 回）+ 尾声
```

### 时间线设计
| 部分 | 时代 | 年份 | 说明 |
|------|------|------|------|
| 楔子 | 古代 | 前 1000 年 | 诅咒起源 |
| 第一世 | 古代 | 前 1000 年 | 初见殉情 |
| 第二世 | 古代 | 650 年 | 身份悬殊 |
| 第三世 | 古代 | 1450 年 | 正邪对立 |
| 第四世 | 近代 | 1930 年 | 家国情怀 |
| 第五世 | 现代 | 1990 年 | 现实阻碍 |
| 第六世 | 当代 | 2026 年 | 记忆觉醒 |
| 第七世 | 未来 | 2050 年 | 最终圆满 |

## 📝 每世模板（6 回）

### 第 1 回：相遇
**功能：** 建立初见场景，埋下情感种子

**必备元素：**
- [ ] 场景描写（五感并用）
- [ ] 一见钟情（心动瞬间）
- [ ] 内心吐槽（幽默调节）
- [ ] 悬念埋设（伏笔）

**情感基调：** 甜蜜、心动

**字数：** 2500-3000 字

**示例：**
```
桃花林初见（第一世）
醉仙楼听琴（第二世）
魔教总坛相认（第三世）
雨中相遇（第四世）
图书馆偶遇（第六世）
```

---

### 第 2 回：发展
**功能：** 推进感情，建立内部矛盾

**必备元素：**
- [ ] 表白/提亲场景
- [ ] 内部矛盾（绯闻/误会/三角恋）
- [ ] 配角登场（小翠/阿左等）
- [ ] 甜蜜互动

**情感基调：** 甜蜜 + 小波折

**字数：** 2500-3000 字

**示例：**
```
将军提亲（第一世）
右相提亲（第二世）
选择相信（第三世）
杀父之仇（第四世）
车祸失忆（第五世）
```

---

### 第 3 回：转折
**功能：** 外部阻力出现，情节转折

**必备元素：**
- [ ] 外部阻力（战争/身份/立场）
- [ ] 随机事件（改变剧情走向）
- [ ] 情感挣扎（选择困境）
- [ ] 小反转

**情感基调：** 紧张、挣扎

**字数：** 3000-3500 字

**示例：**
```
西戎犯境（第一世）
右相陷害（第二世）
正道围攻（第三世）
抗日爆发（第四世）
家族反对（第五世）
```

---

### 第 4 回：高潮
**功能：** 情感爆发，配角牺牲

**必备元素：**
- [ ] 配角支线高潮
- [ ] 牺牲/成全场景
- [ ] 情感爆发（哭戏）
- [ ] 大反转

**情感基调：** 悲壮、挣扎

**字数：** 3000-3500 字

**示例：**
```
阿左战死（第一世）
右相之子成全（第二世）
冷锋牺牲（第三世）
军阀千金成全（第四世）
未婚妻阴谋（第五世）
```

---

### 第 5 回：悲剧
**功能：** 悲剧发生，情感崩溃

**必备元素：**
- [ ] 死亡/错过场景
- [ ] 真相揭露
- [ ] 情感崩溃（哭戏）
- [ ] 最虐场景

**情感基调：** 崩溃、绝望

**字数：** 3000-3500 字

**示例：**
```
凤卿中咒（第一世）
凤卿自尽（第二世）
两人战死（第三世）
凤卿挡炸弹（第四世）
凤卿车祸（第五世）⭐ 最虐
```

---

### 第 6 回：过渡
**功能：** 收尾，过渡到下世

**必备元素：**
- [ ] 殉情/悬念
- [ ] 三生石场景
- [ ] 下世预告
- [ ] 夜玄观察

**情感基调：** 释然、期待

**字数：** 2500-3000 字

**示例：**
```
陈叙殉情（第一世）
陈叙未殉情（第二世）
三生石相聚（第三世）
陈叙继续抗日（第四世）
冷冻 24 年（第六世）
```

## 📋 冲突类型库

### 外部冲突（每世至少 1 种）
| 类型 | 示例 |
|------|------|
| 战争 | 西戎犯境、抗日 |
| 身份 | 状元×琴师、豪门×贫困生 |
| 立场 | 锦衣卫×魔教圣女 |
| 仇恨 | 杀父之仇 |
| 家族 | 家族反对、逼婚 |

### 内部冲突（每世至少 2 种）
| 类型 | 示例 |
|------|------|
| 绯闻 | 公主婚约 |
| 误会 | 以为背叛、以为贪财 |

## 🔧 使用说明

### 1. 直接套用
- 确定核心设定
- 选择时代背景
- 套用 6 回模板
- 填充具体内容

### 2. 灵活调整
- 可调整每世回数（4-8 回）
- 可调整时间线（古代→现代→未来）
- 可调整结局（BE/HE）

### 3. 创新变化
- 保持核心公式
- 创新冲突类型
- 创新人物设定
- 创新结局设计
