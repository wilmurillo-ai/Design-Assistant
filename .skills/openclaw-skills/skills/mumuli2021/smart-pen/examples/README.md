## 🔧 使用指南

### 查找示例
1. 确定需要的类型（情感/场景/对话/人物/冲突）
2. 查看对应分类
3. 找到具体示例
4. 参考应用到新作

### 学习示例
1. 阅读原文件
2. 分析写作技巧
3. 总结规律
4. 应用到实践
