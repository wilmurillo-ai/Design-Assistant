# 成哥写作模型·自检清单

## 📋 写作前检查

### 核心设定
- [ ] 核心设定是否清晰？（一句话能说清）
- [ ] 时间线是否合理？（古代→现代→未来）
- [ ] 世界观是否完整？（规则明确）

### 人物设计
- [ ] 主角核心特质是否明确？
- [ ] 主角成长弧光是否完整？
- [ ] 配角是否有独立动机？
- [ ] 反派是否可理解？

### 情节规划
- [ ] 每世冲突类型是否多样？（至少 3 种）
- [ ] 是否有随机事件？（5-7 个/世）
- [ ] 伏笔是否埋设？（至少 4 个）

### 情感设计
- [ ] 情感曲线是否合理？（甜→虐→甜）
- [ ] 情感层次是否丰富？（至少 3 种）
- [ ] 是否有情绪释放点？

---

## ✍️ 写作中检查

### 语言风格
- [ ] 对话是否口语化？
- [ ] 是否适度使用网络梗？（3-5 个/回）
- [ ] 是否有内心吐槽？

### 场景描写
- [ ] 是否五感并用？（视觉/听觉/嗅觉/触觉/味觉）
- [ ] 是否有核心意象？
- [ ] 是否有情感投射？

### 情感表达
- [ ] 情感是否真实？（不为虐而虐）
- [ ] 是否有层次？（表面/深层/元情感）
- [ ] 是否有递进？（甜→虐→甜）

### 节奏控制
- [ ] 是否有反转？（每 3 回小反转）
- [ ] 是否有悬念？（短期/中期/长期）
- [ ] 是否有张力？（冲突不断）

---

## ✅ 写作后检查

### 内容完整
- [ ] 【】标记是否清除？
- [ ] 章回标题是否保留？
- [ ] 世数标题是否保留？
- [ ] 分隔线是否清除？

### 逻辑自洽
- [ ] 时间线是否连贯？
- [ ] 因果链是否完整？
- [ ] 伏笔是否有呼应？
- [ ] 人物行为是否合理？

### 质量保证
- [ ] 情感是否真实？
- [ ] 人物是否立体？
- [ ] 节奏是否有张力？
- [ ] 读者是否能共鸣？

---

## 📊 质量评估表

### 评分标准（1-5 分）

| 维度 | 1 分 | 3 分 | 5 分 |
|------|------|------|------|
| **情感真实** | 为虐而虐 | 部分真实 | 完全真实 |
| **人物立体** | 工具人 | 部分立体 | 完全立体 |
| **逻辑自洽** | 多处矛盾 | 小问题 | 完全自洽 |
| **节奏张力** | 平铺直叙 | 部分有张力 | 全程有张力 |
| **读者共鸣** | 无共鸣 | 部分共鸣 | 强烈共鸣 |

### 总分评估
- 20-25 分：优秀作品 ⭐
- 15-19 分：良好作品
- 10-14 分：需要改进
- 5-9 分：重写

---

## 🎯 核心原则检查

### 不可妥协的
- [ ] 情感必须真实
- [ ] 人物必须立体
- [ ] 逻辑必须自洽
- [ ] 节奏必须有张力

### 可以调整的
- [ ] 网络梗频率是否合适？
- [ ] 场景描写是否适度？
- [ ] 章回数量是否合理？
- [ ] 配角戏份是否合适？
