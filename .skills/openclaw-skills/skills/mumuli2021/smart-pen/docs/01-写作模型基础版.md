# 一、写作风格模型

## 1.1 语言风格

### 核心特点
| 特点 | 说明 | 示例 |
|------|------|------|
| **00 后幽默** | 轻松诙谐，网络梗适度使用 | "直接好家伙"、"破防了"、"磕到了" |
| **内心吐槽** | 主角/配角内心独白增加趣味 | "我说了啥？？？" |
| **对话简短** | 口语化，节奏快 | "你信我吗？""我信。" |
| **场景细腻** | 五感并用，画面感强 | "粉白的花瓣层层叠叠，像是天边最软的云霞" |

### 对话风格
```
✅ 短句为主，少用长句
✅ 情绪直接，不拐弯抹角
✅ 适度留白，给读者想象空间
✅ 符合人物身份（将军/状元/少帅等不同语气）
```

### 叙述风格
```
✅ 第一人称/第三人称灵活切换
✅ 视角标签清晰（【陈叙视角】等，正文中移除）
✅ 时间线清晰，每世独立又连贯
✅ 情感递进，从甜到虐到圆满
```

---

## 1.2 情感表达

### 情感层次（6 种）
| 情感类型 | 比例 | 作用 | 代表场景 |
|----------|------|------|----------|
| **缠绵** | 20% | 甜蜜、撒糖 | 新婚夜、提亲 |
| **暧昧** | 20% | 拉扯、试探 | 桃花林初见、醉仙楼听琴 |
| **情爱** | 15% | 深情、承诺 | "我找了你七世" |
| **争吵** | 15% | 冲突、张力 | 杀父之仇、误会贪财 |
| **误解** | 20% | 虐心、揪心 | 休书、失忆 |
| **释怀** | 10% | 和解、成长 | 放下仇恨、原谅 |

### 情感递进公式
```
第一世：懵懂初恋（甜→虐）
第二世：热烈奔放（甜→虐→遗憾）
第三世：挣扎矛盾（纠结→悲壮）
第四世：克制压抑（虐→释然）
第五世：崩溃绝望（最虐）⭐
第六世：主动成长（甜→悬念）
第七世：圆满永恒（HE）⭐
```

---

## 1.3 场景描写

### 五感并用原则
| 感官 | 描写要点 | 示例 |
|------|----------|------|
| **视觉** | 颜色、光影、形状、动态 | "粉白的花瓣如雪般飘落" |
| **听觉** | 声音大小、远近、音色 | "琴声如泣如诉" |
| **嗅觉** | 气味浓淡、层次 | "桃花的甜香，混着泥土的清新" |
| **触觉** | 温度、质感、疼痛 | "春风拂面，带着些许暖意" |
| **味觉** | 苦涩、甘甜、血腥 | "嘴角溢血，腥甜" |

### 场景描写公式
```
环境铺垫（视觉 + 听觉）+ 人物出场（视觉细节）+ 情感烘托（嗅觉 + 触觉）= 完整场景
```

# 二、写作思路模型

## 2.1 故事架构

### 核心框架：七世轮回
```
楔子（2 回）→ 第一世（6 回）→ 第二世（6 回）→ 第三世（6 回）→ 
第四世（6 回）→ 第五世（6 回）→ 第六世（6 回）→ 第七世（6 回）+ 尾声
```

### 时间线设计
| 世数 | 时代 | 年份 | 身份设定 |
|------|------|------|----------|
| 楔子 | 西周初年 | 前 1000 年 | 夜玄×阿瑶（诅咒起源） |
| 第一世 | 西周 | 前 1000 年 | 将军×大夫 |
| 第二世 | 唐朝 | 650 年 | 状元×琴师 |
| 第三世 | 明朝 | 1450 年 | 锦衣卫×圣女 |
| 第四世 | 民国 | 1930 年 | 少帅×学生 |
| 第五世 | 现代 | 1990 年 | 豪门×贫困生 |
| 第六世 | 当代 | 2026 年 | 学霸×写手 |
| 第七世 | 未来 | 2050 年 | AI 工程师×仿生人 |

### 每世结构（6 回模板）
```
第 1 回：相遇（心动场景 + 暧昧）
第 2 回：发展（提亲/表白 + 内部矛盾）
第 3 回：冲突（外部阻力 + 随机事件）
第 4 回：高潮（牺牲/误会 + 配角支线）
第 5 回：悲剧（死亡/错过 + 情感爆发）
第 6 回：结局（殉情/悬念 + 三生石过渡）
```

---

## 2.2 人物设计

### 主角设定
| 人物 | 核心特质 | 成长弧光 |
|------|----------|----------|
| **陈叙** | 深情、执着、每世身份不同 | 冲动→坚持→珍惜→担当→成长→圆满 |
| **凤卿** | 温婉、善良、主动牺牲 | 被动等待→主动牺牲→并肩作战→追求理想→觉醒 |

### 配角设定（独立弧光）
| 配角 | 身份 | 核心动机 | 结局 |
|------|------|----------|------|
| **阿左** | 守护者 | 生生世世守护陈叙 | 第七世解脱转世 |
| **夜玄** | 邪神→救赎 | 从嫉妒到相信爱情 | 解除诅咒，与阿瑶团聚 |
| **右相之子** | 深情男二 | 爱而不得，选择成全 | 牺牲 |
| **冷锋** | 义兄 | 守护凤卿 | 牺牲成全 |
| **宋晚晴** | 军阀千金 | 高傲→放手 | 幸福 |

### 人物关系网络
```
核心：陈叙×凤卿（七世情缘）
守护：阿左（七世陪伴）
反派→救赎：夜玄（诅咒→解除）
成全配角：右相之子/冷锋/宋晚晴/林婉
```

---

## 2.3 情节设计

### 冲突类型（每世至少 3 种）
| 冲突类型 | 第一世 | 第二世 | 第三世 | 第四世 | 第五世 |
|----------|--------|--------|--------|--------|--------|
| **外部** | 战争 | 身份悬殊 | 正邪对立 | 杀父之仇 | 家族反对 |
| **内部** | 绯闻 | 误会 | 立场 | 仇恨 | 失忆 |
| **配角** | 青梅竹马 | 右相之子 | 义兄 | 军阀千金 | 未婚妻 |

### 随机事件设计
- 每世插入 5-7 个随机事件
- 改变剧情走向，避免单调
- 增加悬念和反转

### 伏笔与呼应
| 伏笔 | 位置 | 呼应 | 位置 |
|------|------|------|------|
| 凤凰印记 | 楔子 | 第七世觉醒 | 第 32 回 |
| 阿左誓言 | 楔子 | 每世守护 | 每世 |
| 三生石预言 | 楔子 | 第七世圆满 | 第 42 回 |
| 夜玄 backstory | 楔子 1-2 | 第七世救赎 | 第 39 回 |

# 三、写作规则模型

## 3.1 内容规则

### 必须保留
- ✅ 章回标题（从文件名提取）
- ✅ 世数标题（第一世、第二世等）
- ✅ 对话内容
- ✅ 心理活动（无标记）
- ✅ 场景描写

### 必须移除
- ❌ 【】标记（【陈叙视角】等）
- ❌ 分隔线（════════）
- ❌ "第 X 回 完"标记
- ❌ 作者备注

### 格式规范
```
段落：每段独立，适当留白
对话：引号清晰，一人一段
心理：无标记，自然融入
场景：五感并用，画面感强
```

---

## 3.2 节奏规则

### 每回字数
| 部分 | 字数 | 说明 |
|------|------|------|
| 楔子 | 3000 字/回 | 铺垫 backstory |
| 正文章回 | 2500-3000 字/回 | 情节推进 |
| 高潮回 | 3000-3500 字/回 | 情感爆发 |

### 情感节奏
```
每世节奏：甜（1-2 回）→ 虐（3-5 回）→ 悲（6 回）
整体节奏：第一世（甜虐）→ 第五世（最虐）→ 第七世（圆满）
```

### 悬念设计
- 每世至少 1 个核心悬念
- 每 3 回一个小反转
- 每世一个大反转

---

## 3.3 质量规则

### 自检清单
- [ ] 章回标题是否显示
- [ ] 【】标记是否清除
- [ ] 情感层次是否丰富（至少 3 种）
- [ ] 场景描写是否五感并用
- [ ] 对话是否口语化、简短
- [ ] 情节是否有反转
- [ ] 伏笔是否有呼应
- [ ] 配角是否有独立弧光

### 读者体验
- ✅ 阅读流畅（无阅读障碍）
- ✅ 情感共鸣（能打动读者）
- ✅ 逻辑自洽（时间线/因果链）
- ✅ 期待感强（悬念/伏笔）

# 四、创作流程模型

## 4.1 创作步骤

```
1. 确定核心设定（七世轮回 + 诅咒）
   ↓
2. 设计人物关系网（主角 + 配角）
   ↓
3. 规划每世框架（身份 + 冲突 + 结局）
   ↓
4. 编写详细大纲（每回情节）
   ↓
5. 正式写作（00 后风格 + 场景描写）
   ↓
6. 自我审查（清除【】+ 检查标题）
   ↓
7. 生成网页版（移动端适配 + 导航）
   ↓
8. 发布与反馈（内网分享 + 收集意见）
```

## 4.2 工具使用

| 工具 | 用途 |
|------|------|
| Markdown | 正文写作 |
| Python | 数据处理、HTML 生成 |
| HTTP Server | 内网分享 |
| Word | 导出文档 |
