# 成哥写作模型·核心公式集

## 📖 语言公式

### 好文字公式
```
好文字 = 30% 画面描写 + 40% 情绪表达 + 20% 对话推进 + 10% 留白
```

### 对话设计
```
好对话 = 简短口语化 + 情绪直接 + 适度留白
```

### 场景描写
```
好场景 = 核心意象（桃花）+ 情感投射（飘落=离别）+ 五感细节
```

### 五感分配
```
视觉 40% + 听觉 20% + 嗅觉 15% + 触觉 15% + 味觉 10%
```

---

## 💕 情感公式

### 虐心公式
```
虐心 = 希望 × 落差 × 无力感

示例：
第一世：希望（提亲成功）× 落差（战争分离）× 无力感（中咒）= 虐
第五世：希望（记忆恢复）× 落差（婚礼当天）× 无力感（车祸）= 最虐 ⭐
```

### 情感层次
```
表层情感（角色感受到的）
    ↓
深层情感（读者感受到的）
    ↓
元情感（作者想传达的）
```

### 情感递进
```
第一世：懵懂初恋（甜→虐）
第二世：热烈奔放（甜→虐→遗憾）
第三世：挣扎矛盾（纠结→悲壮）
第四世：克制压抑（虐→释然）
第五世：崩溃绝望（最虐）⭐
第六世：主动成长（甜→悬念）
第七世：圆满永恒（HE）⭐
```

### 六层情感
| 情感 | 比例 | 作用 |
|------|------|------|
| 缠绵 | 20% | 甜蜜、撒糖 |
| 暧昧 | 20% | 拉扯、试探 |
| 情爱 | 15% | 深情、承诺 |
| 争吵 | 15% | 冲突、张力 |
| 误解 | 20% | 虐心、揪心 |
| 释怀 | 10% | 和解、成长 |

## 📚 结构公式

### 好故事公式
```
好故事 = 熟悉感（模板）× 新鲜感（变化）× 期待感（悬念）
```

### 七世结构
```
楔子（2 回）→ 七世（各 6 回）→ 尾声
总计：44-45 回
```

### 每世六回模板
| 回次 | 功能 | 情感 | 必备元素 |
|------|------|------|----------|
| 第 1 回 | 相遇 | 心动 | 场景描写 + 一见钟情 |
| 第 2 回 | 发展 | 甜蜜 | 表白/提亲 + 内部矛盾 |
| 第 3 回 | 转折 | 紧张 | 外部阻力 + 随机事件 |
| 第 4 回 | 高潮 | 挣扎 | 配角牺牲 + 情感爆发 |
| 第 5 回 | 悲剧 | 崩溃 | 死亡/错过 + 真相揭露 |
| 第 6 回 | 过渡 | 释然 | 三生石 + 下世预告 |

### 时间线设计
```
好时间线 = 历史纵深（厚重）+ 现实连接（代入）+ 未来想象（升华）

示例：
西周（前 1000）→ 唐（650）→ 明（1450）→ 民国（1930）→ 
1990 → 2026 → 2050
```

## 👥 人物公式

### 好主角
```
好主角 = 核心特质（深情）× 成长变化（七世学会不同东西）
```

### 好配角
```
好配角 = 独立动机 × 完整弧光 × 与主线交织
```

### 好反派
```
好反派 = 可理解动机 × 完整弧光 × 救赎可能
```

### 人物关系网络
```
核心：主角×主角（七世情缘）
守护：守护者（七世陪伴）
反派→救赎：反派（诅咒→解除）
成全配角：多个配角（各自成全）
```

## 🎭 悬念公式

### 好悬念
```
好悬念 = 短期（30%）+ 中期（40%）+ 长期（30%）

短期：本章内解决
中期：本世内解决
长期：全书结束解决
```

### 伏笔设计
```
好伏笔 = 早期埋设 + 中期呼应 + 后期回收
```

## 😊 共鸣公式

### 三层共鸣
```
表层共鸣：情节共鸣（"我也经历过"）
    ↓
深层共鸣：情感共鸣（"我也这样感受过"）
    ↓
元共鸣：存在共鸣（"这就是人生"）
```

### 读者情绪曲线
```
好情绪 = 铺垫（30%）× 积累（40%）× 释放（30%）

示例：
期待（楔子）→ 甜蜜（1-2 回）→ 紧张（3-4 回）→ 
崩溃（5 回）→ 释然（6 回）→ 圆满（7 回）
```

## ✍️ 创作流程公式

### 创作步骤
```
设定 → 人物 → 框架 → 大纲 → 写作 → 审查 → 发布
```

### 质量保障
```
好质量 = 写作前规划（30%）+ 写作中控制（40%）+ 写作后检查（30%）
```

## 📊 自检公式

### 内容自检
```
好内容 = 章回标题 ✅ + 【】清除 ✅ + 情感层次 ✅ + 
        场景描写 ✅ + 逻辑自洽 ✅
```

### 节奏自检
```
好节奏 = 每世 3 种冲突 + 每 3 回小反转 + 每世大反转
```

## 🎯 核心原则

### 不可妥协的
1. ✅ 情感必须真实（不能为虐而虐）
2. ✅ 人物必须立体（不能工具人）
3. ✅ 逻辑必须自洽（不能前后矛盾）
4. ✅ 节奏必须有张力（不能平铺直叙）

### 可以调整的
1. ⚙️ 网络梗的使用频率
2. ⚙️ 场景描写的详细程度
3. ⚙️ 每世的章回数量
4. ⚙️ 配角的戏份比例
