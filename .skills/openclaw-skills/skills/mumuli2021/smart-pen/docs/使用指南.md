## 🎯 使用场景

### 场景 1：新项目启动

**步骤：**
1. 阅读 `docs/核心文档/00-聪明的笔总纲.md`
2. 使用 `templates/七世轮回模板.md` 搭建框架
3. 参考 `examples/人物示例.md` 设计人物
4. 使用 `tools/自检清单.md` 检查设定

**预计用时：** 30-60 分钟

---

### 场景 2：日常创作

**步骤：**
1. 写作前：阅读 `docs/核心文档/03-创作原则.md`
2. 写作中：参考 `docs/核心文档/02-核心公式集.md`
3. 写作后：使用 `tools/自检清单.md` 检查

**预计用时：** 每次 5-10 分钟

---

### 场景 3：质量评估

**步骤：**
1. 使用 `tools/自检清单.md` 逐项检查
2. 使用 `tools/质量评估表.md` 打分
3. 根据得分决定是否需要修改

**预计用时：** 15-30 分钟

## 📋 核心公式速查

### 写作公式
```
好文字 = 30% 画面 + 40% 情绪 + 20% 对话 + 10% 留白
虐心 = 希望 × 落差 × 无力感
好场景 = 核心意象 + 情感投射 + 五感细节
好故事 = 熟悉感 × 新鲜感 × 期待感
```

### 人物公式
```
好主角 = 核心特质 × 成长变化
好配角 = 独立动机 × 完整弧光 × 与主线交织
好反派 = 可理解动机 × 完整弧光 × 救赎可能
```

### 悬念公式
```
好悬念 = 短期 30% + 中期 40% + 长期 30%
好情绪 = 铺垫 30% × 积累 40% × 释放 30%
```

## 🎯 常见问题

### Q1：如何设计一个新项目？
**A：** 使用 `templates/七世轮回模板.md`，按步骤填写：
1. 确定核心设定
2. 设计时间线
3. 设计人物
4. 规划每世冲突

### Q2：如何保证情感真实？
**A：** 参考 `docs/核心文档/03-创作原则.md` 中的"情感必须真实"原则：
- 情感递进（有过程）
- 情绪铺垫（有希望）
- 真实反应（符合性格）

### Q3：如何检查质量？
**A：** 使用 `tools/自检清单.md`：
- 写作前检查（设定）
- 写作中检查（语言/场景/情感）
- 写作后检查（内容/逻辑/质量）

### Q4：如何学习写作技巧？
**A：** 参考 `examples/` 中的示例：
- 情感示例（缠绵/暧昧/虐心）
- 场景示例（初见/战斗）
- 对话示例（表白/虐心）
- 人物示例（主角/配角）
