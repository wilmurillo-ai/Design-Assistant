# 成哥写作模型·创作原则

## 🎯 不可妥协的原则

### 1. 情感必须真实

**❌ 错误做法：**
- 为虐而虐（没有铺垫的悲剧）
- 强行圆满（没有过程的 HE）
- 情绪断层（突然转变）

**✅ 正确做法：**
- 情感递进（甜→虐→甜有过程）
- 情绪铺垫（悲剧前有希望）
- 真实反应（人物反应符合性格）

**示例：**
```
第五世为什么最虐？
因为有铺垫：失忆→误会→怀孕→错过→车祸
因为真实：两人都在为对方着想
因为无力：明明相爱却无法相守
```

---

### 2. 人物必须立体

**❌ 错误做法：**
- 工具人配角（只为推进剧情）
- 脸谱化反派（为坏而坏）
- 完美主角（没有缺点）

**✅ 正确做法：**
- 配角独立弧光（阿左七世守护）
- 反派可理解（夜玄被伤害过）
- 主角有成长（七世学会不同东西）

**示例：**
```
阿左不是工具人：
第一世：战死（忠诚）
第二世：顶罪（觉醒）
第三世：双面间谍（选择）
第四世：自尽（自由）
第五世：被灭口（良知）
第六世：AI 删除（觉醒）
第七世：解脱转世（圆满）⭐
```

---

### 3. 逻辑必须自洽

**❌ 错误做法：**
- 时间线混乱
- 因果链断裂
- 伏笔不回收

**✅ 正确做法：**
- 时间线清晰（西周→唐→明→民国→1990→2026→2050）
- 因果链完整（前世影响今生）
- 伏笔全部回收（8 个伏笔全部呼应）

**示例：**
```
伏笔设计：
凤凰印记（楔子）→ 第七世觉醒（第 32 回）✅
阿左誓言（楔子）→ 每世守护（每世）✅
三生石预言（楔子）→ 第七世圆满（第 42 回）✅
夜玄 backstory（楔子 1-2）→ 第七世救赎（第 39 回）✅
```

---

### 4. 节奏必须有张力

**❌ 错误做法：**
- 平铺直叙（没有起伏）
- 一味虐心（没有甜）
- 一味甜蜜（没有冲突）

**✅ 正确做法：**
- 甜虐交替（每世甜→虐）
- 冲突多样（外部 + 内部 + 配角）
- 情绪释放（第五世最虐→第七世圆满）

**示例：**
```
情感节奏：
第一世（甜虐）→ 第二世（甜虐遗憾）→ 第三世（纠结悲壮）→ 
第四世（虐释然）→ 第五世（最虐）⭐→ 第六世（甜悬念）→ 
第七世（圆满）⭐
```

---

## ⚙️ 可以调整的元素

### 1. 网络梗使用频率

**可调整范围：**
- 密集使用（每回 5-10 个）
- 适度使用（每回 3-5 个）⭐ 推荐
- 少量使用（每回 1-2 个）

**建议：**
- 楔子：适度（建立风格）
- 正文章：适度（保持一致）
- 高潮回：少量（情感优先）

---

### 2. 场景描写详细程度

**可调整范围：**
- 详细描写（300-500 字/场景）
- 适度描写（150-300 字/场景）⭐ 推荐
- 简略描写（50-150 字/场景）

**建议：**
- 初见场景：详细（桃花林）
- 日常场景：适度（对话推进）
- 高潮场景：简略（情感爆发）

---

### 3. 每世章回数量

**可调整范围：**
- 4 回/世（快节奏）
- 6 回/世（标准）⭐ 推荐
- 8 回/世（慢节奏）

**建议：**
- 重要世（第一、五、七世）：6-8 回
- 过渡世（第二、三、四世）：6 回
- 快节奏世（第六世）：4-6 回

---

### 4. 配角戏份比例

**可调整范围：**
- 10%（主角绝对主导）
- 20%（配角有独立故事）⭐ 推荐
- 30%（多线叙事）

**建议：**
- 守护者（阿左）：每世出现
- 成全配角：每世 1-2 个
- 反派：贯穿始终

---

## 📋 质量自检清单

### 写作前
- [ ] 核心设定是否清晰？
- [ ] 人物弧光是否完整？
- [ ] 情感曲线是否合理？
- [ ] 时间线是否清晰？

### 写作中
- [ ] 对话是否口语化？
- [ ] 场景是否五感并用？
- [ ] 情感是否有层次（至少 3 种）？
- [ ] 是否有反转（每 3 回小反转）？

### 写作后
- [ ] 【】标记是否清除？
- [ ] 章回标题是否保留？
- [ ] 世数标题是否保留？
- [ ] 伏笔是否有呼应？
- [ ] 逻辑是否自洽？
- [ ] 节奏是否有张力？

## 🎯 创作信念

### 写作的目的
```
不是讲一个故事，
而是让读者相信：
有些人，一旦遇见，就是永恒。
```

### 写作的信念
```
不是为虐而虐，
而是让读者经历后明白：
七世情缘，终得圆满。
```

### 写作的追求
```
不是完成一部作品，
而是创造一种体验：
让读者又哭又笑，虐到崩溃，甜到圆满。
```
