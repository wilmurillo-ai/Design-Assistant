## 🎯 核心公式（9 个）

### 语言公式
```
好文字 = 30% 画面 + 40% 情绪 + 20% 对话 + 10% 留白
```

### 情感公式
```
虐心 = 希望 × 落差 × 无力感
```

### 场景公式
```
好场景 = 核心意象 + 情感投射 + 五感细节
```

### 故事公式
```
好故事 = 熟悉感 × 新鲜感 × 期待感
```

### 人物公式
```
好主角 = 核心特质 × 成长变化
好配角 = 独立动机 × 完整弧光 × 与主线交织
好反派 = 可理解动机 × 完整弧光 × 救赎可能
```

### 悬念公式
```
好悬念 = 短期 30% + 中期 40% + 长期 30%
```

### 情绪公式
```
好情绪 = 铺垫 30% × 积累 40% × 释放 30%
```
