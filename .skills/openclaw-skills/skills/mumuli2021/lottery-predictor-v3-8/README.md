# 🎰 双色球预测技能 V3.8 - 机器学习增强版

基于机器学习的双色球预测工具，使用随机森林和 Gradient Boosting 模型，结合规则引擎进行混合集成预测。

## 核心特性

- ✅ **机器学习模型**：随机森林 + Gradient Boosting
- ✅ **15 维特征工程**：遗漏、频率、形态、区间、号码属性
- ✅ **混合集成**：ML 50% + 规则引擎 50%
- ✅ **大样本训练**：2000 期历史数据，62,700 样本
- ✅ **高准确率**：红球 80.8%，蓝球 93.8%

## 安装依赖

```bash
pip3 install scikit-learn numpy pandas
```

## 使用方法

### 预测下一期

```bash
python3 scripts/v3.8_ml_model.py
```

### 运行回测

```bash
python3 scripts/backtest_v3.8.py --periods 1000
```
