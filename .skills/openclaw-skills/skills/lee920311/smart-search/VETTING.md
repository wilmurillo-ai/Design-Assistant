## 🔍 代码审计详情

### 核心函数分析

#### 1. `decide_engine()`
```bash
功能：根据查询内容选择搜索引擎
风险：🟢 无（纯文本匹配，无外部调用）
输入：用户搜索词
输出：引擎名称（exa/searx/tavily）
```

#### 2. `call_exa_mcp()`
```bash
功能：调用 Exa MCP API
风险：🟢 低（官方服务，HTTPS 加密）
输入：搜索词
输出：搜索结果
数据：仅发送搜索词，无个人信息
```

#### 3. `call_searx()`
```bash
功能：调用本地 SearX
风险：🟢 无（本地服务，无外部网络）
输入：搜索词
输出：搜索结果
```

#### 4. `call_tavily()`
```bash
功能：调用 Tavily API
风险：🟢 低（官方服务，HTTPS 加密）
输入：搜索词 + 用户 API Key
输出：搜索结果 + AI 摘要
数据：API Key 从环境变量读取，不传输
```
