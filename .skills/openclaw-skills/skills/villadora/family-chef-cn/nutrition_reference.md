# 每日营养建议

## 成年人

- 热量：1800-2200 kcal
- 蛋白质：50-70g
- 脂肪：40-60g
- 碳水：250-300g
- 膳食纤维：25-30g

## 儿童（7-14岁）

- 热量：1600-2200 kcal
- 蛋白质：40-60g
