## ⚠️ 已知问题

### 1. 价格提取问题
**现象**: 搜索结果为空，价格为 0

**原因**: 
- 携程页面使用动态加载（JavaScript 渲染）
- 价格信息在异步请求后显示
- 当前等待时间可能不足

**解决方案**:
1. 增加等待时间（已设置为 8 秒）
2. 使用 `page.wait_for_selector()` 等待特定元素
3. 考虑使用携程 API（如有）或中间件

### 2. 反爬策略
**现象**: 频繁请求可能被限制

**建议**:
- 请求间隔 > 5 秒
- 使用代理 IP
- 模拟真实用户行为
