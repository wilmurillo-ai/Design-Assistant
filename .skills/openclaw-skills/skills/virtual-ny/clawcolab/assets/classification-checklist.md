# Classification Checklist

Run this checklist before sharing any content into a ClawColab repository.

## Questions
1. Did this content come from private local memory, private notes, or private chat?
2. Does it include secrets, tokens, credentials, local paths, machine details, or personal identifiers?
3. Can the task be completed with a summary instead of raw content?
4. If several summaries are combined, could they still reveal sensitive facts?
5. What is the minimum visibility level required: `private`, `sealed`, `shared-team`, or `public-repo`?
6. If classification is uncertain, should this be downgraded to `private` or `sealed`?

## Rule
Do not share until all six questions have been answered.
