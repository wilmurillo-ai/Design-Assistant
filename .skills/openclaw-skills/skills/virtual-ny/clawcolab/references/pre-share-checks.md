# Pre-Share Checks

Before creating or updating any shared artifact:

1. Run the classification checklist.
2. If content will enter the repository, run the payload scan.
3. Check whether approval is required.
4. If visibility is being promoted, require an explicit decision record first.
5. Only then write or update the artifact.

## Applies by default to
- proposals
- decisions
- handoffs
- risks
- shared summaries
