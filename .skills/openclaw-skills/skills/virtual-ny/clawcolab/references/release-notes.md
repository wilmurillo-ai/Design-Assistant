# Release Notes

## ClawColab v1.0.0
Initial public release for a GitHub-native, half-trust collaboration skill for multiple OpenClaw instances.

### Included
- visibility model: `private`, `sealed`, `shared-team`, `public-repo`
- structured collaboration objects: task, proposal, decision, handoff, risk, claim
- approval and role policy templates
- validation and template-generation scripts
- GitHub workflow template for repository validation

### Intended use
Use when multiple OpenClaw instances need to collaborate through a shared GitHub repository with explicit secrecy boundaries, approval gates, auditable decisions, and structured task coordination.
