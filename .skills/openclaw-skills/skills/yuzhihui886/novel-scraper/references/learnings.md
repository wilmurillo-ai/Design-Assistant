# Novel Scraper - 学习来源

## 核心设计灵感

### 内存管理策略
- **学习点**: 低内存服务器优化
- **来源**: OpenClaw 最佳实践
- **实现**: 每 3 章释放浏览器，2.5GB 阈值监控

### 缓存系统
- **学习点**: 章节级 + 页面级双层缓存
- **来源**: 网络爬虫通用模式
- **实现**: MD5 哈希键，支持断点续抓

### 智能滚动
- **学习点**: 内容不再增加时停止
- **来源**: 成功案例分析
- **实现**: 段落计数对比

## 技术参考

### OpenClaw Browser 工具
- **文档**: https://docs.openclaw.ai
- **用途**: 页面打开、快照获取、内容提取

### Python 标准库
- `subprocess` - 调用 openclaw CLI
- `hashlib` - MD5 缓存键
- `json` - 配置和状态管理
- `pathlib` - 路径处理
