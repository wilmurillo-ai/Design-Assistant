## Common Character Errors

### High-Frequency Homophone Confusions

These are the most common typos in Chinese, caused by pinyin-based input methods presenting wrong candidate characters.

#### The 的/地/得 Rule

This is the single most frequent error in Chinese writing.

| Particle | Function | Pattern | Example |
|----------|----------|---------|---------|
| **的** | Attributive (modifies noun) | adj/noun + 的 + noun | 漂亮**的**裙子, 我**的**书 |
| **地** | Adverbial (modifies verb) | adv/adj + 地 + verb | 自由**地**飞翔, 认真**地**学习 |
| **得** | Complement (describes result) | verb/adj + 得 + result | 高兴**得**跳起来, 跑**得**很快 |

Quick test: 的 answers "what kind?", 地 answers "how?", 得 answers "to what degree/result?"

#### Other High-Frequency Pairs

| Error pair | Rule |
|-----------|------|
| **在/再** | 在 = at/in (location, state: 在家, 在工作); 再 = again/then (再见, 再说, 再三) |
| **以/已** | 以 = with/by (以后, 可以, 以为); 已 = already (已经, 已有, 已知) |
| **做/作** | 做 = concrete actions (做饭, 做事, 做梦); 作 = abstract/literary (作为, 作品, 作用). When both work, 做 is preferred in modern usage |
| **即/既** | 即 (jí) = immediately/if (即使, 即将, 立即); 既 (jì) = already/since (既然, 既定, 既往) |
| **权力/权利** | 权力 = power/authority (国家权力); 权利 = rights/entitlements (公民权利) |
| **必须/必需** | 必须 = must (verb, 你必须来); 必需 = essential/needed (adj, 必需品) |
| **反映/反应** | 反映 = reflect/report (反映情况); 反应 = react/response (化学反应, 反应速度) |
| **截止/截至** | 截止 = deadline has passed (报名已截止); 截至 = up to (截至目前) |
| **其他/其它** | 其他 = general use (recommended); 其它 = for things only (less common) |
| **象/像** | 象 = elephant (大象); 像 = resemble/image (好像, 画像). Note: 好象 is always wrong → 好像 |

### Similar-Looking Character Errors

| Error pair | Distinction |
|-----------|-------------|
| **已/己/巳** | 已 (yǐ, already) — opening faces right; 己 (jǐ, self) — fully open; 巳 (sì, earthly branch) — fully closed |
| **未/末** | 未 (wèi, not yet) — top stroke shorter; 末 (mò, end) — top stroke longer |
| **拔/拨** | 拔 (bá, pull out); 拨 (bō, dial/allocate) — note right component difference |
| **辩/辨/辫** | 辩 (argue) — middle is 言; 辨 (distinguish) — middle is a point+line; 辫 (braid) — middle is 纟 |
| **壁/璧** | 壁 (wall, 墙壁); 璧 (jade, 完璧归赵) |
| **陷/馅** | 陷 (fall into, 陷阱); 馅 (filling, 饺子馅) |

## Punctuation Rules

### Core Rules

Based on GB/T 15834-2011 (标点符号用法):

| Rule | Correct | Wrong |
|------|---------|-------|
| Chinese sentences use full-width punctuation | 你好，世界！ | 你好,世界! |
| Period is 。not . | 结束了。 | 结束了. |
| Ellipsis is six dots | 他说…… | 他说... or 他说...... |
| Em-dash occupies two character widths | 这件事——很重要 | 这件事—很重要 |
| Enumeration comma (顿号) only between parallel words | 苹果、香蕉、橘子 | Used between clauses |
| No stacked exclamation/question marks | 真的吗？ | 真的吗？？？ |
| Nested quotes: outer " " inner ' ' | 他说"她说'好的'" | Reverse nesting |
| Book title marks for published works only | 《红楼梦》 | 《春节联欢晚会》(event, wrong) |
| Cannot combine ellipsis with 等 | 苹果、香蕉…… OR 苹果、香蕉等 | 苹果、香蕉……等 |

### Positioning Rules

| Mark | Cannot appear at |
|------|-----------------|
| 。，、；：！？ | Line start |
| Opening brackets " ' 《 〈 （ [ { | Line end |
| Closing brackets " ' 》 〉 ） ] } | Line start |

### Number Usage

Based on GB/T 15835-2011 (出版物上数字用法):

| Context | Format | Example |
|---------|--------|---------|
| Statistics, measurements | Arabic | 增长 5.3%，全长 1200 公里 |
| Formal dates | Chinese+Arabic | 2026年3月19日 |
| Technical dates | ISO format | 2026-03-19 |
| Approximate numbers | Chinese characters | 三四天, 十几个 |
| Idioms and set phrases | Chinese characters | 一心一意, 三令五申 |
| Number ranges | En-dash or wave | 10–20 or 10~20, not 10-20 |
| 4-digit numbers | No separator | 5000 |
| 5+ digit numbers | Space grouping | 100 000 |

## CJK-Latin Mixing Rules

Based on CY/T 154-2017 and the Sparanoid Chinese Copywriting Guidelines.

### Spacing Rules

| Rule | Correct | Wrong |
|------|---------|-------|
| Space between Chinese and English | 在 LeanCloud 上 | 在LeanCloud上 |
| Space between Chinese and numbers | 花了 5000 元 | 花了5000元 |
| Space between numbers and units | 10 GB | 10GB |
| NO space for % and degrees | 90%, 233° | 90 %, 233 ° |
| NO space around full-width punctuation | 好开心！ | 好开心 ！ |
| Space around hyperlinks in Chinese text | 请 [提交 issue] 并分配 | 请[提交issue]并分配 |

### Punctuation in Mixed Text

| Context | Use | Example |
|---------|-----|---------|
| Chinese sentence with embedded English words | Full-width punctuation | 我买了 iPhone，好开心！ |
| Complete English sentence within Chinese text | Half-width punctuation for the English | Stay hungry, stay foolish. |
| English parenthetical within Chinese | Half-width parentheses with spaces | (optional) 这是可选的 |

### Proper Noun Capitalization

Always use the brand's official capitalization:

| Correct | Wrong |
|---------|-------|
| GitHub | github, GITHUB, Github |
| iOS | ios, IOS |
| iPhone | iphone, IPHONE, Iphone |
| macOS | MacOS, MACOS |
| JavaScript | javascript, Javascript |
| TypeScript | typescript, Typescript |
| LinkedIn | Linkedin, LINKEDIN |
| YouTube | Youtube, YOUTUBE |
| WeChat | wechat, Wechat |
| WiFi | wifi, WIFI, Wifi |
| App | APP (全大写在中文语境中常见但不规范) |

Avoid unnecessary abbreviations in formal text: use TypeScript not TS, HTML5 not H5 (H5 is acceptable in casual/industry contexts where meaning is clear).
