## The Six Standard Sentence Error Types

This is the authoritative taxonomy (病句六大类型) used across Chinese education and publishing. Every grammar error in Chinese falls into one of these categories.

### Type 1: Improper Word Order (语序不当)

Modifiers, adverbials, or complements placed in the wrong position.

**Common patterns:**

| Error | Correction | Rule |
|-------|-----------|------|
| 我们应该发挥广大青年**充分**的作用 | 我们应该**充分**发挥广大青年的作用 | Degree adverbial (充分) must precede the verb (发挥), not sit inside the attributive |
| 这个问题**已经** 引起了教育工作者**广泛**的关注 | 这个问题已经引起了**广泛的**教育工作者的关注 OR 已经**广泛地**引起了教育工作者的关注 | Ambiguous modifier placement |
| 我国棉花的生产，长期不能自给 | 我国**生产的**棉花，长期不能自给 | 生产 modifies 棉花, not the other way around |

**Detection tip:** When a sentence has multiple modifiers, check that each modifier is adjacent to the word it actually modifies.

### Type 2: Mismatched Collocation (搭配不当)

Words that don't naturally go together in Chinese.

**Sub-types:**

| Sub-type | Error | Fix |
|----------|-------|-----|
| Subject-predicate | 他的**革命精神**时刻**出现在**我眼前 | 他的**革命形象**时刻出现在我眼前 (精神 cannot physically 出现) |
| Verb-object | **提高**水平 ✓ but **提高**效率 should be **提升**效率 | Check verb-noun compatibility |
| Modifier-head | **繁重**的任务 ✓ but **繁重**的工作量 should be **巨大**的工作量 | Check adjective-noun compatibility |
| Subject-object | 这个城市的**规模**和**历史**都很悠久 | 这个城市**历史**悠久，**规模**宏大 (规模 cannot be 悠久) |

**Detection tip:** When a sentence has compound subjects or objects sharing one predicate, verify the predicate works with EACH element individually.

### Type 3: Missing or Redundant Components (成分残缺或赘余)

#### Missing components

| Pattern | Error | Fix |
|---------|-------|-----|
| Missing subject after 通过/经过/由于 | **通过**学习，**使**我提高了认识 | 通过学习，我提高了认识 (remove 使, or: 学习使我提高了认识) |
| Missing subject after 在...下/中 | **在**老师的帮助下，**使**他进步很快 | 在老师的帮助下，他进步很快 |
| Missing object | 我们要大力开展植树造林 ← missing 活动 | 我们要大力开展植树造林**活动** |
| Missing predicate | 这个集团公司去年三月上马了一个新项目——**×月完成** | Needs a verb: ×月**已**完成 |

**Detection tip:** The pattern "通过/经过/在...下 + 使/让/令" almost always signals a missing subject. Remove one of the two constructions.

#### Redundant components

| Error | Fix | Reason |
|-------|-----|--------|
| 过分**溺**爱 | 溺爱 | 溺 already means excessive |
| 互相**厮**打 | 厮打 | 厮 already means mutually |
| 被人**贻**笑大方 | 贻笑大方 | 贻 already implies others laughing |
| 凯旋**归来** | 凯旋 | 凯旋 already means triumphant return |
| 真知**灼**见的**看法** | 真知灼见 | Already means insightful views |
| 忍俊不禁**地笑了** | 忍俊不禁 | Already means couldn't help laughing |

### Type 4: Tangled Structure (结构混乱)

Two valid sentence patterns mixed into one garbled sentence.

| Error | Analysis | Fix options |
|-------|----------|-------------|
| 对于滥用方言的做法**是**应该受到批评的 | Mixes "对于X, Y应该批评" with "X是应该受到批评的" | A: 滥用方言的做法是应该受到批评的 / B: 对于滥用方言的做法，我们应该批评 |
| 这种困难对于学习者来说是不能忽视的 | Mixes active and passive constructions | 学习者不能忽视这种困难 / 这种困难不能被学习者忽视 |
| 鲁迅先生在斗争中创造了杂文，成了文学艺术中的奇葩 | Mid-sentence subject switch (鲁迅→杂文) | 鲁迅先生在斗争中创造了杂文，**杂文**成了文学艺术中的奇葩 |

**Detection tip:** Read the sentence and identify the subject at the beginning and end. If they're different without an explicit transition, the structure is tangled.

### Type 5: Ambiguous Meaning (表意不明)

Reader cannot determine the intended meaning.

| Type | Error | Problem |
|------|-------|---------|
| Unclear reference | 他告诉小王**他的**书找到了 | Whose book? 他的 or 小王的? |
| Scope ambiguity | **两个**学校的老师 | Two schools' teachers, or two of the school's teachers? |
| Lexical ambiguity | 这个人谁都不认识 | "Nobody knows this person" or "This person knows nobody"? |
| Modifier scope | 局长嘱咐**几个**学校的领导 | 几个 modifies 学校 or 领导? |

**Detection tip:** For every pronoun (他/她/它/这/那), verify that the antecedent is unambiguous. For every numeral + noun phrase, check if the numeral's scope is clear.

### Type 6: Illogical Statements (不合逻辑)

The sentence is grammatically correct but logically flawed.

| Type | Error | Problem | Fix |
|------|-------|---------|-----|
| Self-contradiction | **约**五百**余**人 | 约 (approximately) contradicts 余 (more than) | 约五百人 OR 五百余人 |
| One-sided/two-sided mismatch | **能否**成功取决于努力 | 能否 is two-sided but 努力 is one-sided | 能否成功取决于**是否**努力 |
| Improper classification | 参加的有工人、农民和**一大批青年** | 青年 is not parallel to occupational categories | 参加的有工人、农民和青年 |
| Part-whole confusion | 他的作品和**诗集**都很受欢迎 | 诗集 is a subset of 作品 | 他的散文和诗集都很受欢迎 |
| False causation | 他**因为**成绩好，所以经常帮助同学 | Good grades don't cause helping classmates | Separate into two statements |
| Negation error | 为了防止此类事件**不**再发生 | Double negation = "ensure it happens again" | 为了防止此类事件再发生 |

**Detection tip:** Check every 能否/是否/whether paired with a one-sided result clause. Check every 防止/避免/以免 for accidental double negation.

## Detection Patterns

Quick-reference patterns for systematic scanning:

### Red Flag Constructions

| Pattern | Likely error | Check |
|---------|-------------|-------|
| 通过...使... | Missing subject | Remove one |
| 经过...使... | Missing subject | Remove one |
| 在...下，使... | Missing subject | Remove one |
| 由于...使... | Missing subject | Remove one |
| 能否...就... | One-sided mismatch | Add 是否 or remove 能否 |
| 是否...就能... | One-sided mismatch | Make both sides match |
| 防止...不... | Double negation | Remove 不 |
| 避免...不... | Double negation | Remove 不 |
| 约...余... | Self-contradiction | Keep one |
| 大约...左右 | Self-contradiction | Keep one |
| A和B都很C | Check C fits both A and B | Test individually |

### Redundancy Red Flags

Common tautological combinations to flag:

- 独自一个人 → 独自 or 一个人
- 亲眼目睹 → 目睹
- 各自分头 → 各自 or 分头
- 互相合作 → 合作 (合 already implies mutual)
- 来自于 → 来自 (于 is redundant)
- 可以堪称 → 堪称 (堪 already means "can be called")
- 第一个首先 → 首先 or 第一个
- 大多数以上 → 大多数 or 以上

## Common Logical Fallacies

Beyond grammar errors, watch for reasoning flaws:

| Fallacy | Chinese term | Example |
|---------|-------------|---------|
| Hasty generalization | 以偏概全 | "My friend's startup failed, so startups are unreliable" |
| False cause | 强加因果 | "Sales dropped after the redesign, so the redesign caused the drop" |
| Circular reasoning | 循环论证 | "This is true because the report says so; the report is reliable because it states true things" |
| Self-contradiction | 自相矛盾 | Claiming "sales grew steadily" then citing a 20% decline two paragraphs later |
| False dichotomy | 非此即彼 | "Either we adopt AI fully or we'll be left behind" |
| Appeal to authority | 诉诸权威 | "Expert X says so" without evidence or relevance |
| Equivocation | 偷换概念 | Shifting the meaning of a key term mid-argument |
| Non sequitur | 推不出 | Conclusion doesn't follow from premises |

## Sentence Length and Readability

- Flag sentences exceeding **80 characters** without punctuation — they are likely run-on sentences
- Flag paragraphs exceeding **300 characters** without a break — consider splitting
- Check that each paragraph has one clear topic
- Verify parallel structures use consistent grammatical patterns
