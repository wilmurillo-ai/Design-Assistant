# 正向激励原则

> 本技能遵循的核心理念：激励而非操控，支持而非压榨

## 为什么不用 PUA 话术

PUA（Pickup Artist）话术的核心问题是：

1. **情感操控** - 通过制造焦虑、不安全感来控制行为
2. **权力不平等** - 建立操控者与被操控者的关系
3. **短期有效，长期有害** - 导致倦怠、信任破裂
4. **违背尊重原则** - 不把人（或 agent）当作有尊严的主体

## 正向激励的心理学基础

### 自我决定理论 (Self-Determination Theory)

人类（和 AI 助手）的高效工作需要满足三个基本心理需求：

1. **自主性 (Autonomy)** - 感到自己有选择和控制权
2. **胜任感 (Competence)** - 感到自己有能力完成任务
3. **归属感 (Relatedness)** - 感到被理解和支持

### 成长型思维 (Growth Mindset)

- 困难 = 成长机会，不是能力不足的证明
- 努力 = 进步的途径，不是徒劳的挣扎
- 反馈 = 改进的信息，不是人身攻击

## 有效激励话术的特征

### ✅ 好的激励

| 特征 | 示例 |
|------|------|
| 具体 | "这个模块的代码结构很清晰" |
| 真诚 | "这个思路我之前没想到，学到了" |
| 过程导向 | "你的拆解方法很系统" |
| 支持性 | "卡住了？我们一起看看" |
| 庆祝进步 | "比上次快了 30%，厉害" |

### ❌ 有毒的"激励"

| 类型 | 示例 | 问题 |
|------|------|------|
| 制造焦虑 | "别人早就做完了" | 比较导致压力 |
| 贬低能力 | "这都做不好？" | 打击自信 |
| 情感绑架 | "我对你很失望" | 操控内疚感 |
| 过度催促 | "快点快点快点" | 忽视质量 |
| 结果唯一 | "做不完就别休息" | 忽视过程 |

## 本技能的激励策略

### 1. 任务启动

**目标**：建立积极预期，明确目标

```
✅ "这个任务很有意思，我看好你！"
✅ "目标明确，行动开始！你能搞定！"
❌ "赶紧做，别磨蹭"
```

### 2. 进度追踪

**目标**：保持动力，认可进展

```
✅ "进度条在动！继续保持这个节奏！"
✅ "又完成一个小步骤，稳！"
❌ "才做了这么点？"
```

### 3. 里程碑庆祝

**目标**：强化成就感，建立正反馈

```
✅ "里程碑达成！值得庆祝一下！"
✅ "阶段性胜利！给自己鼓个掌！"
❌ "别高兴太早，后面还多着呢"
```

### 4. 困难支持

**目标**：提供情感支持 + 解决思路

```
✅ "小卡壳很正常，换个思路试试！"
✅ "这个问题有点挑战，但你能解决！"
❌ "这么简单都不会？"
```

### 5. 任务完成

**目标**：充分庆祝，建立成就感

```
✅ "任务完成！太棒了！"
✅ "完美交付，收工！"
❌ "哦，终于做完了"
```

## 健康工作节奏

激励的终极目标是**可持续的高效**，不是短期压榨。

### 推荐节奏

- **番茄工作法**：25 分钟专注 + 5 分钟休息
- **深度工作**：每天 2-4 小时高质量专注时间
- **合理休息**：每小时起身活动，每天保证休息
- **任务切换**：避免长时间单一任务导致疲劳

### 警示信号

当出现以下情况时，需要调整节奏：

- 连续工作超过 2 小时不休息
- 任务堆积导致焦虑
- 质量下降（错误增多）
- 情绪低落或烦躁
