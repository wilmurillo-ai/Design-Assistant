# AI工作原理

## 核心概念

### 神经网络基础
- **神经元**: 基本计算单元，接收输入并产生输出
- **层**: 神经元的集合，形成网络结构
- **权重**: 连接强度，通过训练调整
- **激活函数**: 引入非线性，使网络能学习复杂模式

### 训练过程
1. **前向传播**: 输入数据通过网络产生预测
2. **损失计算**: 比较预测与真实值的差异
3. **反向传播**: 计算梯度，确定如何调整权重
4. **权重更新**: 使用优化算法（如SGD、Adam）更新权重

### 推理阶段
- 加载训练好的模型
- 输入新数据
- 获得预测结果
- 无需进一步训练

## 大语言模型特性

### 自回归生成
- 逐token生成文本
- 基于上下文预测下一个token
- 支持多种输出格式

### 上下文窗口
- 模型能处理的最大token数量
- 影响长文档理解和生成能力
- 不同模型有不同窗口大小

### 零样本/少样本学习
- **零样本**: 直接理解任务指令
- **少样本**: 通过示例学习任务模式
- **思维链**: 逐步推理提高准确性

## 实际应用考虑

### Token限制
- 输入+输出总长度不能超过上下文窗口
- 需要合理规划提示词长度
- 长文档需要分块处理

### 延迟与成本
- 更大模型通常更慢更贵
- 需要在性能和成本间权衡
- 缓存和批处理可优化效率

### 可靠性
- AI可能产生幻觉（虚假信息）
- 重要决策需要人工验证
- 设置适当的置信度阈值
