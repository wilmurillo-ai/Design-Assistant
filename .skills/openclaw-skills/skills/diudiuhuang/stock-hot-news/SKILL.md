# 财经热点新闻爬取与话题归纳系统

## 功能特性

* **主力财经网站**：3大权威财经网站网稳定爬取，后续逐步适配其他财经网站
* **混合评分系统**：本地规则(55分) + 大模型评估(45分)智能评分
* **专业HTML报告**：可视化设计，自动打开浏览器，响应式布局
* **实时快讯采集**：获取专业财经网站实时快讯
* **自动化工作流**：一键执行，自动清理旧数据

## 快速开始

```bash
cd skills/stock-hot-news
python main.py --module3-mode playwright
```
