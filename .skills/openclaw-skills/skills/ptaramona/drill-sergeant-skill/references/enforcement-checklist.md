# Enforcement Checklist

Use this checklist before publishing any discipline report.

## Core checks
- Is each finding tied to concrete evidence?
- Is each action specific and executable now?
- Is ownership explicit?
- Are duplicates collapsed into one item?
- Is severity justified?

## Escalation rules
Escalate when:
- Same failure repeats 2+ times in one window
- Wrong-audience disclosure risk appears
- Work is blocked by unclear ownership

Do not escalate when:
- One-off formatting misses
- Benign duplicates already corrected
- No user impact and no repetition
