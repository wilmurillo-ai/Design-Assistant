# Startup Checklist (Cerebro-First)

Use this at the beginning of operational tasks.

- [ ] Domain identified
- [ ] Read `COMPANY.md` (or domain root doc)
- [ ] Read most specific skill/runbook
- [ ] Read today + yesterday daily memory
- [ ] Known State prepared (Last done / Open items / Next step)
- [ ] Action mapped to explicit Cerebro rule/section

If any item is unchecked, pause execution and complete it first.
