## 📝 发布文案

### 标题（中文）
财报智析 - PDF财务报告智能分析工具

### 标题（英文）
Financial Report Analyzer - Intelligent PDF Financial Report Analysis Tool

### 简短描述（100字以内）
智能解析PDF财务报告，自动提取关键财务指标，生成可视化分析图表和投资参考报告。支持A股上市公司年报、季报分析，本地处理确保数据安全。

### 详细描述

#### 功能特点
📊 **智能PDF解析**
- 自动提取三大报表数据（资产负债表、利润表、现金流量表）
- 识别关键财务指标：营业收入、净利润、ROE、毛利率等16项指标
- 支持多页复杂格式财报

📈 **可视化分析**
- 财务健康度雷达图
- 盈利能力柱状图
- 关键指标摘要表

📝 **智能报告生成**
- 自动生成财务分析报告
- 包含执行摘要、盈利能力分析、偿债能力分析
- 风险提示和投资建议

🔒 **数据安全保障**
- 本地处理，数据不上传云端
- 只读访问，不修改原文件
- 代码开源透明，可审计

#### 适用场景
- 📈 投资者快速了解公司财务状况
- 📊 分析师批量处理财报数据
- 🎓 学生学习财务分析
- 💼 企业财务部门文档处理

#### 支持格式
- A股上市公司年报、季报
- PDF格式财务报告
- 中文财报优化

## 🔧 技术说明

### 依赖库
- `pdfplumber` - PDF文本和表格提取
- `pandas` - 数据处理
- `matplotlib` - 图表生成
- `numpy` - 数值计算

### 支持平台
- Windows
- macOS
- Linux

### 文件结构
```
financial-report-analyzer/
├── SKILL.md                    # Skill说明文件
├── scripts/
│   ├── extract_pdf.py         # PDF提取
│   ├── parse_financial_data.py # 数据解析
│   ├── generate_charts.py     # 图表生成
│   └── generate_report.py     # 报告生成
├── references/
│   ├── financial_metrics.md   # 财务指标说明
│   └── report_template.md     # 报告模板
└── assets/                    # 资源文件
```
