# 财务指标详解

## 📊 盈利能力指标

### 营业收入 (Operating Revenue)
- **定义**: 企业在日常经营活动中产生的收入总额
- **计算公式**: 主营业务收入 + 其他业务收入
- **解读**: 
  - 反映企业市场规模和经营能力
  - 持续增长表明企业处于扩张期
  - 需结合行业特点判断合理性

### 净利润 (Net Profit)
- **定义**: 扣除所有成本、费用、税金后的最终盈利
- **计算公式**: 利润总额 - 所得税费用
- **解读**:
  - 反映企业最终盈利水平
  - 关注扣非净利润（扣除非经常性损益）
  - 与营业收入结合看盈利能力

### 毛利率 (Gross Profit Margin)
- **定义**: 销售收入扣除成本后的利润占比
- **计算公式**: (营业收入 - 营业成本) / 营业收入 × 100%
- **参考标准**:
  - > 40%: 优秀（如白酒、医药）
  - 20-40%: 良好
  - < 20%: 较低（如零售、制造业）
- **解读**: 反映产品竞争力和定价能力

### 净利率 (Net Profit Margin)
- **定义**: 净利润占营业收入的比例
- **计算公式**: 净利润 / 营业收入 × 100%
- **参考标准**:
  - > 20%: 优秀
  - 10-20%: 良好
  - < 10%: 较低
- **解读**: 反映整体盈利能力，包含管理效率

### ROE - 净资产收益率 (Return on Equity)
- **定义**: 股东权益的投资回报率
- **计算公式**: 净利润 / 平均净资产 × 100%
- **参考标准**:
  - > 20%: 优秀（巴菲特标准）
  - 15-20%: 良好
  - < 10%: 较低
- **解读**: 衡量股东投资回报的核心指标

### ROA - 总资产收益率 (Return on Assets)
- **定义**: 总资产的盈利能力
- **计算公式**: 净利润 / 平均总资产 × 100%
- **参考标准**:
  - > 10%: 优秀
  - 5-10%: 良好
  - < 5%: 较低
- **解读**: 反映资产使用效率

## 📈 成长能力指标

### 营收增长率 (Revenue Growth Rate)
- **定义**: 营业收入的同比增长速度
- **计算公式**: (本期营收 - 上期营收) / 上期营收 × 100%
- **参考标准**:
  - > 30%: 高速增长
  - 15-30%: 稳健增长
  - < 15%: 增长放缓
- **解读**: 反映企业扩张速度

### 净利润增长率 (Profit Growth Rate)
- **定义**: 净利润的同比增长速度
- **计算公式**: (本期净利润 - 上期净利润) / 上期净利润 × 100%
- **解读**:
  - 应高于营收增长率（规模效应）
  - 持续正增长表明盈利能力提升
  - 负增长需警惕

## 💰 偿债能力指标

### 资产负债率 (Debt-to-Asset Ratio)
- **定义**: 负债占总资产的比例
- **计算公式**: 总负债 / 总资产 × 100%
- **参考标准**:
  - < 40%: 保守
  - 40-60%: 适中
  - > 60%: 偏高（需结合行业）
  - > 70%: 高风险
- **解读**: 
  - 反映财务杠杆水平
  - 不同行业标准差异大（银行、地产通常较高）

### 流动比率 (Current Ratio)
- **定义**: 流动资产对流动负债的覆盖能力
- **计算公式**: 流动资产 / 流动负债
- **参考标准**:
  - > 2: 优秀
  - 1.5-2: 良好
  - < 1: 短期偿债压力大
- **解读**: 反映短期偿债能力

### 速动比率 (Quick Ratio)
- **定义**: 剔除存货后的短期偿债能力
- **计算公式**: (流动资产 - 存货) / 流动负债
- **参考标准**:
  - > 1: 优秀
  - 0.8-1: 良好
  - < 0.5: 风险较高
- **解读**: 更严格的短期偿债指标

## 🔄 运营能力指标

### 总资产周转率 (Total Asset Turnover)
- **定义**: 总资产的运营效率
- **计算公式**: 营业收入 / 平均总资产
- **参考标准**:
  - > 1: 优秀（零售业可能更高）
  - 0.5-1: 良好
  - < 0.5: 资产使用效率较低
- **解读**: 反映资产运营效率

### 应收账款周转天数 (Days Sales Outstanding)
- **定义**: 平均收回应收账款所需天数
- **计算公式**: 365 / (营业收入 / 平均应收账款)
- **参考标准**:
  - < 30天: 优秀
  - 30-60天: 正常
  - > 90天: 回款风险较高
- **解读**: 反映应收账款管理效率

### 存货周转天数 (Days Inventory Outstanding)
- **定义**: 平均销售存货所需天数
- **计算公式**: 365 / (营业成本 / 平均存货)
- **参考标准**:
  - 因行业差异大
  - 快消品: < 30天
  - 制造业: 30-60天
  - 房地产: > 365天
- **解读**: 反映存货管理效率

## 🔍 综合分析框架

### 杜邦分析法
```
ROE = 净利率 × 总资产周转率 × 权益乘数

其中:
- 净利率: 盈利能力
- 总资产周转率: 运营效率
- 权益乘数: 财务杠杆
```

### 健康企业特征
1. **盈利稳定**: 毛利率、净利率稳定或提升
2. **成长持续**: 营收、利润持续增长
3. **负债合理**: 资产负债率适中，偿债能力强
4. **运营高效**: 资产周转快，回款及时

### 风险信号
1. **盈利下滑**: 连续两季度毛利率下降
2. **负债过高**: 资产负债率>70%且持续上升
3. **现金流差**: 经营现金流持续为负
4. **周转变慢**: 应收账款、存货周转天数显著增加

---

*注：以上标准仅供参考，实际分析需结合行业特点和公司具体情况。*
