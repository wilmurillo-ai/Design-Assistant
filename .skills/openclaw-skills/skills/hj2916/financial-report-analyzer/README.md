# 财报智析 (Financial Report Analyzer)

智能解析PDF财务报告，自动提取关键财务指标，生成可视化分析图表和投资参考报告。

## ✨ 功能特点

- 📊 **智能PDF解析** - 自动提取三大报表数据
- 📈 **可视化分析** - 雷达图、柱状图、摘要表
- 📝 **智能报告** - 自动生成财务分析报告
- 🔒 **数据安全** - 本地处理，不上传云端

## 🚀 快速开始

### 安装
```bash
clawhub install financial-report-analyzer
```

### 使用
上传PDF财报文件，AI将自动：
1. 提取财务数据
2. 生成可视化图表
3. 输出分析报告

## 📋 支持的财务指标

### 盈利能力
- 营业收入、净利润
- 毛利率、净利率
- ROE、ROA

### 偿债能力
- 资产负债率
- 流动比率、速动比率

### 成长能力
- 营收增长率
- 净利润增长率

### 运营能力
- 总资产周转率
- 应收账款周转天数
- 存货周转天数
