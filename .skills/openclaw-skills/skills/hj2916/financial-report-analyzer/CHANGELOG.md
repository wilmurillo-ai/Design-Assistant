# 更新日志

## [1.0.0] - 2026-03-28

### 新增
- 初始版本发布
- PDF财报文本提取功能
- 16项财务指标自动识别
- 财务健康度雷达图
- 盈利能力柱状图
- 关键指标摘要表
- 智能分析报告生成
- 支持A股上市公司财报格式

### 测试验证
- ✅ 东方财富2024年报（金融）
- ✅ 美的集团2024三季报（家电）
- ✅ 比亚迪2025三季报（汽车）

### 技术栈
- Python 3.12
- pdfplumber - PDF解析
- pandas - 数据处理
- matplotlib - 图表生成
