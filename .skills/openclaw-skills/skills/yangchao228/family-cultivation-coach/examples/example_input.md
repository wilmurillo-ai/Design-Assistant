## 第1组：孩子画像

**姓名 / 昵称**：果果

**年龄**：5岁4个月

**当前阶段**：幼儿园大班（秋季升小学）

**性格特点**：
活跃外向，好奇心强，但注意力容易分散；喜欢和人互动，对新事物接受快；情绪比较直接，遇到不想做的事会直接说"不要"；规则意识还在建立中，有时需要多次提醒才能执行。

**当前兴趣**：
- 特别喜欢恐龙，能背出十几种恐龙名称
- 喜欢积木和乐高类拼插玩具
- 喜欢听故事（尤其是爸爸讲的，不喜欢自己看书）
- 最近对英文动画有兴趣（Bluey、Peppa Pig）

**当前优势**：
- 语言表达能力强，会用复杂句子讲故事
- 记忆力好，听过的故事能大段复述
- 社交能力好，在幼儿园有几个好朋友

**当前短板 / 挑战**：
- 专注时间短，主动学习超过15分钟就开始走神
- 精细动作偏弱（握笔姿势不好，不喜欢写写画画）
- 规则执行力弱，需要反复提醒（如收玩具、洗手）
- 对主动阅读兴趣不高，更喜欢被动听故事

**家长最担心的问题**：
1. 快升小学了，但对"坐下来学"非常抵触，担心入学适应困难
2. 精细动作（握笔）明显比同龄人弱，担心影响入学后写字
3. 英语基础几乎为零，而周围很多孩子已经开始上外教课

## 第3组：培养目标

**当前最重要的 3 个培养方向**：
1. 入学准备：帮助果果建立"坐下来专注做一件事"的能力，时间逐步延长
2. 英语启蒙：建立英语兴趣和初步听力基础（不要求读写，只要愿意开口）
3. 精细动作：通过游戏化方式练习手部精细动作，不强迫写字

**近期最急需解决的问题**：
入学适应焦虑（家长焦虑>孩子焦虑）——希望在接下来3-4个月内通过实际执行，让全家人对果果的准备状态有信心，而不是一直感到"差很多"。

**专项重点**：
- 英语（听力输入+开口兴趣）
- 自我管理（规则执行，例程建立）
- 精细动作（握笔准备，但不直接练写字）
