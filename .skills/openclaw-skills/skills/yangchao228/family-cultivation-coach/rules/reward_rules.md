# 游戏化与奖励规则（Reward & Gamification Rules）

> 本文件定义轻量游戏化设计和奖励机制的原则与示例，供课表设计时参考。

---

## 一、轻量游戏化原则

游戏化的目的是**降低启动门槛、增加参与感**，不是靠外部奖励替代内在动机。

### 核心原则

1. **轻量**：机制简单，家长不需要花大量精力维护
2. **即时**：反馈要快，不能"等一个月再看结果"
3. **过程导向**：奖励行为本身（做了/尝试了），而非只奖励结果（做对了/做好了）
4. **可退出**：如果某个机制孩子不喜欢，可以换，不要强迫
5. **避免通货膨胀**：奖励不要太多太滥，否则失去意义

## 二、奖励设计原则

### 2.1 奖励什么
✅ 推荐奖励的行为：
- 主动开始（不需要催促就开始了某个任务）
- 完成了（不论质量，完成本身值得认可）
- 尝试了新事物（即使没做好）
- 保持了某个习惯 N 天
- 帮助了家人

❌ 不建议只奖励：
- 成绩/得分（容易导致焦虑和结果导向）
- 比较（"比上次好"比"比别人好"更健康）
- 听话/不哭（这是情绪压抑，不是奖励目标）

### 2.2 奖励的形式
按孩子年龄和偏好选择，以下为推荐方向：

| 类型 | 示例 | 适合年龄 | 注意 |
|------|------|---------|------|
| 贴纸 / 印章 | 完成一个任务贴一颗星 | 3-7岁 | 贴纸板要孩子能看到 |
| 积分 / 金币 | 每次完成+1币，攒够兑换 | 5岁+ | 兑换物要现实可兑换 |
| 称号 / 徽章 | "今日专注小勇士" | 4-8岁 | 称号要有新鲜感 |
| 选择权 | 完成后孩子选今晚玩什么 | 4岁+ | 选择范围家长提前划定 |
| 特殊时光 | 攒够X颗星，爸爸带你去__ | 5岁+ | 兑换时间不能太远 |
| 口头认可 | 具体、真诚的语言认可 | 全年龄 | 避免空洞表扬"你真棒" |

## 三、贴纸 / 积分 / 称号系统示例

### 示例 A：简单贴纸板
```
【本周任务板】
□ 英语跟读 × 3次  → 完成3次：贴1颗金星
□ 自主阅读 × 5次  → 完成5次：贴1颗金星
□ 运动 × 4次      → 完成4次：贴1颗金星

本周攒够 3 颗金星 → 周末选一个家庭活动
```

### 示例 B：积分兑换制
```
每次完成任务 → +1 点
主动开始（无需提醒）→ +2 点（加倍）
连续完成 3 天 → +3 点（连击奖励）

积分兑换：
10 点 → 可选今晚故事由孩子来讲
20 点 → 可选一个周末小活动
30 点 → 家庭"你说了算日"（孩子做主一件大事）
```

### 示例 C：称号系统（适合喜欢角色扮演的孩子）
```
称号名称        获得条件
──────────────────────────────
🔰 探索小学徒  → 开始第一周
⭐ 坚持小勇士  → 连续执行 7 天
🌟 专注小达人  → 单次专注完成 ≥ 20 分钟
🏆 自律小冒险家 → 主动开始 5 次（无需提醒）
```

## 四、关于避免过度奖励的提醒

### 4.1 过度奖励的副作用
- 孩子开始"只为奖励而做"，撤销奖励后行为消失
- 孩子开始讨价还价："做完这个给我什么？"
- 家长维护成本越来越高

### 4.2 如何避免
1. **逐步减少外部奖励**：习惯稳定后，减少贴纸/积分，改为偶尔惊喜型认可
2. **强调内在感受**：引导孩子说出"做完之后感觉怎么样"，而不只是给奖励
3. **奖励要有意义**：奖励物/活动要孩子真正在乎，不要流于形式
4. **不要每件小事都奖励**：保留一些"该做的事就是该做的"的心理空间

### 4.3 何时暂停奖励机制
- 孩子已经养成了自发习惯时，可以降低奖励频率
- 当孩子开始抵触或觉得无聊时，换一种机制
- 当家长觉得维护成本太高时，简化或暂停

## 五、游戏化设计快查表

| 目标行为 | 推荐游戏化方式 | 示例 |
|---------|-------------|------|
| 主动开始任务 | 倒计时游戏 + 积分加倍 | "3分钟内自己开始，加倍得分" |
| 坚持每日习惯 | 连击奖励 | 连续3天不断链，获特别贴纸 |
| 接受挑战性任务 | 勇气徽章 | "敢于尝试"比"做对了"更值得奖励 |
| 自主整理/规则 | 检查表 + 盖章 | 完成晚间例程→自己盖章 |
| 英语开口 | 录音/角色扮演 | 录下来给家人"表演"，降低压力 |
| 运动 | 打卡地图 | 每次运动在地图上走一步，走到目标地点 |
