# Professional E-commerce Color Palette

Recommended color presets for the AI Recoloring Agent to suggest to users for high-end fashion results.

| Look & Style | Recommended Hex | Best For | Visual Rationale |
| :--- | :--- | :--- | :--- |
| **Morandi Gray** | `#B8B8B8` | Linen, Cotton, Basics | Calm, stable, minimalist aesthetic. |
| **Latte Beige** | `#C2B280` | Cardigans, Overcoats | Warm, cozy, premium texture enhancement. |
| **Peacock Blue** | `#192135` | Business Suits, Coats | Sophisticated, semi-glossy, elegant. |
| **Terracotta Red** | `#8D2122` | Sweaters, Scarves | Trending vintage red, highly compatible. |

## Strategy
1. **Contrast Management**: If the original garment is white, prioritize mid-saturation Morandi tones. If the original is black, prefer deep-tone shifts.
2. **Global Trend Mapping**: Always recommend these colors if the user asks for a "fashion-forward" or "premium" look without providing a specific color sample.
