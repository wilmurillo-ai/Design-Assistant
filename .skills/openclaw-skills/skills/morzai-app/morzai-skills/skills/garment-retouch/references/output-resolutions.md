# Output Resolutions

This guide defines practical output-resolution recommendations for ecommerce retouching workflows.

---

## Recommended defaults

| Use case | Recommended resolution | Notes |
| :--- | :--- | :--- |
| Standard ecommerce PDP | 2K | Balanced for web detail and processing efficiency |
| Shopify / DTC storefront | 2K | Good default for zoom-friendly storefront assets |
| Premium detail review | 4K | Use when fabric, hardware, or texture detail is critical |
| Batch cleanup workflows | 2K | Better throughput and consistency |

---

## Platform considerations

### Amazon
- Use high enough resolution to support zoom and sharp detail.
- Keep the product clean and centered rather than over-optimizing for file size.

### Shopify / DTC
- 2K is usually a strong default.
- Move to 4K when close-up texture fidelity is a selling point.

### Marketplace bulk workflows
- Prefer 2K for operational speed and consistency unless the user explicitly requests higher fidelity.

---

## Agent implication

- Default to 2K unless the user explicitly requests a premium or close-up-heavy output.
- If the job is highly texture-sensitive, recommend 4K.
- If the task is batch retouching, favor stable 2K production unless there is a strong reason to upscale.
