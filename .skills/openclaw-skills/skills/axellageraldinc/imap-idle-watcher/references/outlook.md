# Outlook / Hotmail / Live Setup

## Create App Password

1. Go to: https://account.live.com/proofs/AppPassword
2. Sign in → follow prompts to generate an app password
3. Copy the password

## If App Passwords is unavailable
- Enable 2FA first: https://account.live.com/proofs/manage/additional

## Connection details
| Setting | Value |
|---------|-------|
| Host | `outlook.office365.com` |
| Port | `993` |
| SSL | Yes (required) |

## Note
Microsoft 365 / Exchange accounts may require admin approval for IMAP access.
