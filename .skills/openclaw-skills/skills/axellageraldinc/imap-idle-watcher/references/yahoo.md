# Yahoo Mail Setup

## Create App Password

1. Go to: https://login.yahoo.com/account/security/app-passwords
2. Sign in → select "Other App" → type a label
3. Click **Generate** → copy the password

## Enable "Allow less secure apps" (if needed)
Yahoo sometimes requires this toggle in account security settings.

## Connection details
| Setting | Value |
|---------|-------|
| Host | `imap.mail.yahoo.com` |
| Port | `993` |
| SSL | Yes (required) |
