# 分析框架参考

## 一级市场尽调框架

- 市场空间与需求量化拆解（TAM/SAM/SOM + 单位经济学）
- 技术/产品壁垒与代际演进
- 客户结构深度（大客户绑定、导入周期、多线扶持风险）
- 产能与产线经济学（利用率、爬坡节奏、扩产CapEx）
- 成本结构与降本路径
- 竞争格局逐家拆解（国内+国际对标）
- 供应链安全与去风险（制裁环境下必做）
- 财务健康度（烧钱速度、盈利时间线、累计亏损）
- 关联交易、集团关系与独立性
- 团队与治理
- 资本运作与退出路径
- 合规与政策风险

## 二级市场研究框架

- 估值水平（PE/PB/PS 与同行对比）
- 成长性（收入/利润增速 + 驱动因子拆解）
- 产能利用率与爬坡进度（制造业核心）
- 催化剂与风险事件
- 筹码结构（股东、质押、减持）

## 行业扫描框架

- 市场规模与增速（量价拆解）
- 产业链上中下游（各环节利润分配）
- 竞争格局（CR3/CR5 + 逐家状态）
- 技术路线与演进（代际对比）
- 供应链安全与国产替代进展
- 行业整合趋势（退出者、阵营化）
- 政策与监管

## 制造业/半导体专项维度

以下维度在制造业尽调中**必须覆盖**：
- 产线规格（制程/尺寸/月产能/产品方向）
- 产能利用率及满产时间表
- 设备折旧周期与剩余压力
- 良率水平与提升路径
- 国产化率（设备/材料/衬底）
- 去风险预案（极端制裁场景）
- 降本路径量化（材料/面积/规模各降多少%）
