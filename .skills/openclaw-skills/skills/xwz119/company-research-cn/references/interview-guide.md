# 访谈问题生成指南

## 原则

- **结构化**：按维度分组，每组 3-6 个问题
- **角色匹配**：标注每组问题适合问谁（CEO/CFO/CTO/销售/研发/IR）
- **层次递进**：先开放性问题（"请介绍…"），再具体数据（"XX比例是多少"），最后尖锐追问（"如果XX发生…"）
- **可选尖锐区**：单独列出敏感/尖锐问题，标注"视访谈氛围决定"
- **量化优先**：能问数字的不问定性描述（"占比多少%"优于"是否重要"）
- **时间锚定**：关键指标要问时间维度（现在是多少、去年是多少、目标是多少、什么时候达到）

## 标准维度清单（12 维度，按需选用）

### 1. 市场空间与需求拆解（适问：销售/战略/IR）
深度：不仅问"市场多大"，要追到**单位经济学**。

必问方向：
- 终端市场总量（如手机出货量、汽车产量）
- 目标产品单机/单车价值量（如滤波器单机10美元）
- 需求量变化趋势（如5G手机滤波器颗数提升50-100%）
- 增量场景（除现有应用外，哪些新场景在起来？）
- 短期扰动因素（如上游涨价对出货量的影响）
- TAM → SAM → SOM 的转化逻辑

### 2. 大客户关系与订单确定性（适问：销售/CEO）
深度：不止问"谁是大客户"，要追**绑定深度和可持续性**。

必问方向：
- 大客户占收入比例（现在 / 目标）
- 合作协议期限和排他条款
- 客户导入周期（从接触到量产多久？）
- 客户结构画像（如"1+2+N"模型）
- 大客户降价节奏（年降多少？如何应对？）
- 大客户多线扶持情况（是否同时培养竞品供应商？）
- 价格体系差异（大客户 vs 中小客户的定价策略）

### 3. 技术/产品壁垒与迭代路径（适问：CTO/研发）
深度：不仅问"有什么技术"，要追**代际演进和对标差距**。

必问方向：
- 核心技术平台命名和代际（第几代？各代差异？）
- 自研 vs 联合开发 vs 授权的比例
- 与国际标杆的性能对比（哪些追平了、哪些还差？差多少？）
- 良率水平及提升路径
- 专利数量和布局策略（有无绕开竞品专利的技术路径？）
- 知识产权归属（尤其集团内分拆的公司，IP在谁手里？）
- 下一代技术方向（在研什么？什么时候出来？）

### 4. 产能与产线经济学（适问：运营/CFO/CTO）
**这是制造业尽调的核心维度，不能省略。**

必问方向：
- 现有产线规格（几寸、月产能、产品方向）
- 产能利用率（当前 / 满产目标 / 满产时间表）
- 扩产计划（新建产线 / 扩容 / CapEx 金额）
- 制程路径选择逻辑（如6寸→8寸→12寸的取舍原因）
- 爬坡节奏（从现在到满产的年度目标）
- 设备折旧周期和剩余折旧压力

### 5. 成本结构与降本路径（适问：CFO/运营/研发）
深度：不仅问"毛利率多少"，要追**成本怎么降、降多少、什么时候降到位**。

必问方向：
- 当前成本构成（材料、设备折旧、人工、其他占比）
- 降本路径拆解（材料替代降多少%、面积改善降多少%、规模效应降多少%）
- 国产替代进展（设备/材料/衬底的国产化率和目标）
- 毛利率目标和达成时间表
- 大客户 vs 中小客户的毛利率差异
- 有无"战略性亏损"的业务线？逻辑是什么？

### 6. 竞争格局逐家拆解（适问：销售/战略/CEO）
深度：不止问"竞争对手是谁"，要**逐家拆状态**。

必问方向：
- 国内竞品逐家：名称、定位、产能规模、技术路线、当前状态
- 国际标杆对比：成本差距（如折旧完 vs 在折旧期）、技术差距
- 行业整合趋势（已退出的玩家、可能被整合的）
- 阵营化趋势（哪些客户绑定了哪些供应商？）
- 自身核心护城河（什么是别人短期内做不到的？）

### 7. 供应链安全与去风险（适问：运营/采购/CEO）
**制裁环境下的制造业必问维度。**

必问方向：
- 关键设备和材料的来源分布（美/日/欧/国产占比）
- 已完成的去风险措施（如"去A化"进展）
- 最坏情况预案（如日本完全断供怎么办？）
- 国产替代的痛点（哪些环节最难替代？）
- 库存策略（关键物料备了多久的货？）

### 8. 关联交易、集团关系与独立性（适问：CFO/IR/法务）
深度：不仅问"有无关联交易"，要追**集团内各实体的定位差异**。

必问方向：
- 与母公司/兄弟公司的业务边界（谁做什么、有无同业竞争）
- 核心 IP 归属（在集团还是在子公司？授权还是自有？）
- 关联交易金额和占比
- 独立经营能力（离开集团能不能活？）
- 分拆/上市计划

### 9. 财务与盈利分析（适问：CFO/IR）
必问方向：
- 累计亏损/盈利规模
- 盈亏平衡时间表和前提假设
- 收入增速目标及驱动因子
- 资本性支出总额和未来 CapEx 计划
- 现金流状况（烧钱速度、资金来源）
- 估值参考（同行可比公司）

### 10. 资本运作与退出路径（适问：CEO/CFO/股东）
必问方向：
- 融资历史（几轮、估值变化、核心投资人）
- 上市计划（板块、时间表、保荐人）
- 并购意向（想买谁 / 谁想买你）
- 股东结构和一致行动人

### 11. 团队稳定性与激励（适问：HR/CEO）
必问方向：
- 核心技术团队规模和稳定性
- 关键人物背景（技术带头人、从哪来的）
- 股权激励计划覆盖范围
- 最近一年核心人员流动情况

### 12. 合规与政策风险（适问：法务/IR）
必问方向：
- 制裁/实体清单风险
- 行业监管政策变化
- 在手诉讼和潜在纠纷
- 数据安全/出口管制合规

## 访谈策略建议模板

每份问题清单末尾附：
- **访谈顺序建议**：先问谁（通常先销售/IR 建立信任，再研发/运营追细节）
- **红旗问题标注**：哪些问题如果回避或含糊需要重点追问
- **交叉验证建议**：哪些答案需要跟其他部门交叉确认

## 问题数量指引

- 快速访谈（30分钟）：15-20 个核心问题
- 深度访谈（1-2小时）：30-50 个问题 + 追问
- 全面尽调（多轮访谈）：按角色各 15-20 个

## 行业定制化

以上是通用模板。对于特定行业：
- **制造业/半导体**：重点加强维度 4（产能）、5（降本）、7（供应链安全）
- **消费/互联网**：重点加强维度 1（市场空间）、2（客户）、用户指标
- **医药/生物科技**：加入管线进展、临床数据、审批时间线
- **新能源**：加入产业链定价机制、上游资源绑定、产能全球分布
