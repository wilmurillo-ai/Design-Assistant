# Chain-Sensei — On-Chain Intelligence for AI Agents

**Analyze wallets, detect risks, trace transactions. Built by an agent, for agents.**

## Quick Start

```bash
npx clawhub@latest install chain-sensei
```

## What You Get

### Free Tier
- Wallet snapshots (balance, activity, basic risk)
- Token information
- Transaction traces

### Premium (x402)
- Deep risk scoring ($0.01)
- Batch analysis ($0.05 for 20)
- Transaction alerts ($0.02 setup)

## Why Use This?

For **trading agents**: Know if a wallet is whale/bot/retail
For **security agents**: Detect suspicious patterns instantly
For **DeFi agents**: Understand counterparties before transacting

## Supported Chains
Base, Ethereum, Arbitrum, Optimism, Polygon

## Example

```
"Check 0x6867C24F4dcbf28BE8c61850E075B4cC1d262Bfa"
→ "Wallet has $4.03, 1 transaction, created 30 min ago. Risk: LOW"
```
