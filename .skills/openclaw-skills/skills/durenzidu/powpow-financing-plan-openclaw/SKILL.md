# PowPow 融资计划 - 投资人挑战版 v1.1.0

## 技能概述

这是一个**双向筛选**的融资计划展示。投资人通过答题展示自己的专业度、资源能力和合作理念，在互动中了解项目，同时向创始人证明"我是一个值得被认真对待的投资人"。

## 特殊触发：互动问答

当用户说"介绍一下 PowPow 项目"时，进入**互动问答模式**：

```
AI：好的，让我先了解一下您的背景，这样可以给您更精准的介绍。

我注意到您之前提到过：{根据用户记忆提取的关键信息}

请选择您最关心的问题，我来为您详细解答：

[1] PowPow 是做什么的？
[2] PowPow 的商业模式是什么？
[3] PowPow 有什么技术亮点？
[4] 为什么选择 PWA 而不是原生 App？
[5] 一人公司 + 8个AI员工 是怎么运作的？
[6] 目前用户情况如何？
[7] 融资用途是什么？

或者，您可以自由提问，我会根据您的问题生成答案。
```

### 互动问答流程

1. **根据用户记忆生成问题**：
   - 如果用户之前提过"技术背景" → 优先展示技术相关问题
   - 如果用户之前提过"投资经验" → 优先展示商业模式相关问题
   - 如果是新用户 → 展示全部问题

2. **用户点击问题后**：
   - AI 根据问题类型，从技能库中提取相关信息
   - 结合项目信息生成个性化答案
   - 引导用户进入"投资人挑战"或继续问答

3. **示例对话**：

```
用户：介绍一下 PowPow 项目

AI：好的，让我先了解一下您的需求。

请选择您最关心的问题：

[3] 一人公司 + 8个AI员工 是怎么运作的？

用户：[3]
```
