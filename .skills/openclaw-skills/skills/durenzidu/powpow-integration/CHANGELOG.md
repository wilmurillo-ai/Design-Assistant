# Changelog

所有版本的变更记录。

## [2.1.8] - 2026-04-04

### 修复

- ✅ **添加输入验证**: 在 `createAgent` 函数中添加完整的输入验证
  - 验证数字人名称不能为空且不超过100字符
  - 验证人设描述不能为空
  - **验证纬度必须在 -90 到 90 之间**
  - **验证经度必须在 -180 到 180 之间**
  - 验证位置名称不能为空
  - 验证 Webhook URL 格式正确（必须以 http:// 或 https:// 开头）

### 改进

- 提升用户体验：在发送请求前拦截无效输入，减少无效 API 调用
- 提供清晰的错误提示信息

## [2.1.7] - 2026-04-04

### 初始版本

- 实现 Powpow 平台集成
- 支持用户注册和登录
- 支持创建、查看、删除数字人
- 支持徽章系统管理
- 实现 JWT Token 认证
- 配置文件持久化存储

### 功能

- `register`: 注册 Powpow 账号
- `login`: 登录 Powpow 账号
- `createAgent`: 创建数字人
- `listAgents`: 获取数字人列表
- `deleteAgent`: 删除数字人
- `getStatus`: 获取登录状态
- `logout`: 退出登录

## 版本号规则

- **主版本号**: 重大功能变更或破坏性更新
- **次版本号**: 新功能添加
- **修订号**: Bug 修复和优化
