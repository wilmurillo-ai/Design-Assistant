# 大淘客选品工具

大淘客淘宝客选品与数据服务工具。作为国内淘客行业的基础设施级服务商，提供高佣商品库、实时销量榜单、佣金数据分析等选品工具，帮助淘客推广者高效选品和推广。

## 触发词

大淘客、大淘客选品、dataoke、淘客选品、淘客工具、高佣商品库、淘客数据

## 计划实现能力

- **高佣商品库**：实时更新的高佣金商品数据库
- **实时榜单**：销量榜、佣金榜、新品榜等多维榜单
- **选品分析**：商品佣金趋势、销量数据、竞品分析
- **API对接**：为淘客系统提供商品数据API接口
