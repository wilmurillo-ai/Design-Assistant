# 美逛返利助手

美逛社交电商导购返利工具。聚合淘宝、京东、拼多多等多平台优惠券和返利信息，为推广者提供商品选品、推广素材生成和社群运营工具支持。

## 触发词

美逛、美逛返利、meiguang、美逛优惠、美逛推广、美逛佣金

## 计划实现能力

- **多平台优惠券聚合**：淘宝/京东/拼多多优惠券汇总
- **推广素材库**：商品推广图文素材一键生成
- **选品工具**：高佣爆款商品筛选推荐
- **社群助手**：定时发品、自动转链等社群运营工具
