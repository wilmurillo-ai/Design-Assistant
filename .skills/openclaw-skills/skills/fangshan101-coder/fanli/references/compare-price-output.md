# compare-price 输出模板

**数据来源**：`comparePriceData`。若不存在或 `topLowestItems` 为空 → 输出"暂无跨平台比价数据"。

## 输出模板

```
### 💰 比价结论：{结论}

| 平台 | 店铺 | 价格 | 标签 |
|------|------|------|------|
| {shopType} | {shopName} | **¥{price}** | {badge 或 —} |
```

## 结论规则

- 仅 1 家 → "仅找到一家在售"
- 最低比次低便宜 > 5% 且 > ¥10 → 🟢 推荐 {平台}·{店铺}，比第二低便宜 ¥X
- 差价 ≤ 5% 或 ≤ ¥10 → 🟡 多家价格接近，建议选常用平台
