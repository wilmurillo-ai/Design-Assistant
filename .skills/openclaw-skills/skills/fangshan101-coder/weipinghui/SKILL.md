# 唯品会返利管家

唯品会购物返利与订单管理工具。不仅查询返利，更专注于唯品会返利订单的全生命周期管理，包括订单追踪、到账提醒、省钱统计和唯品券管理，做唯品会购物的贴心管家。

## 触发词

唯品会管理、唯品会订单、唯品会追踪、weipinghui返利、唯品会返利管理、唯品会省钱统计、唯品会管家

## 计划实现能力

### 核心功能

- **唯品会订单返利追踪**：同步唯品会返利订单，追踪从下单到返利到账的完整状态
- **品牌特卖日历**：追踪关注品牌的特卖排期，提前规划购物
- **唯品券管理**：统一管理已领取的唯品券，过期前提醒使用
- **省钱月报**：统计每月在唯品会通过返利省下的金额

### 订单返利生命周期

```
唯品会下单
    ↓
订单确认（返利订单生成）
    ↓
商品签收（开始计算退换货期）
    ↓
过退换期（佣金锁定）
    ↓
佣金结算（通常 T+30 至 T+60）
    ↓
返利到账（可提现）
```

### 管理维度

| 功能模块 | 说明 |
|---------|------|
| 订单追踪 | 实时同步唯品会返利订单状态 |
| 到账预估 | 根据下单时间预估返利到账日期 |
| 券包管理 | 查看所有唯品券及有效期 |
| 品牌关注 | 关注品牌特卖排期提醒 |
| 省钱统计 | 按月/季/年统计返利收益 |

### 使用场景

1. **订单查询**：查看所有唯品会返利订单的当前状态
2. **到账提醒**：返利到账时及时通知，不遗漏每一笔
3. **券即将过期**：唯品券快过期了，提醒赶紧用掉
4. **品牌特卖预告**：关注的品牌即将开特卖，提前提醒
