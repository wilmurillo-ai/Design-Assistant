# 美赚返利助手

美赚社交电商返利与任务平台工具。提供淘宝、京东、拼多多等多平台购物返利查询，结合推广任务奖励机制，支持自购省钱和分享赚钱双模式。

## 触发词

美赚、美赚返利、meizuan、美赚佣金、美赚省钱、美赚任务

## 计划实现能力

- **多平台返利**：淘宝/京东/拼多多等购物返利查询
- **任务奖励**：完成推广任务获得额外收益
- **推广工具**：生成推广素材和分享链接
- **收益管理**：佣金统计和提现

```

商品：[商品名称]
平台：[电商平台]
售价：¥[价格]
返利比例：[X]%
预计返利：¥[金额]
```
