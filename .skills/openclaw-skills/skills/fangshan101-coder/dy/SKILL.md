# 抖音返利管家

抖音电商返利订单管理工具。专注于抖音平台的返利订单全生命周期管理，追踪精选联盟佣金从下单到结算的完整流程，区分直播间、短视频、商城等不同来源的订单数据。

## 触发词

抖音返利管理、抖音订单返利、抖音佣金追踪、dy返利、dy管家、抖音返利统计、抖音省钱报告、精选联盟管理

## 计划实现能力

### 核心功能

- **抖音订单返利追踪**：同步精选联盟返利订单，追踪佣金结算状态
- **多来源分类管理**：区分直播间、短视频带货、抖音商城等不同来源订单
- **达人带货分析**：统计通过哪些达人/主播购买的商品返利最多
- **省钱数据报告**：统计每月抖音购物返利收益和购物行为分析

### 抖音订单返利流程

```
抖音下单（直播间/短视频/商城）
    ↓
订单确认（精选联盟记录佣金）
    ↓
商品签收
    ↓
过售后期（7-15天）
    ↓
佣金结算（T+7 至 T+30）
    ↓
返利到账
```

### 管理维度

| 模块 | 说明 |
|------|------|
| 订单同步 | 自动同步精选联盟返利订单 |
| 来源分类 | 直播间/短视频/商城分别统计 |
| 达人分析 | 哪个主播带货买的最多、返利最高 |
| 佣金追踪 | 实时更新佣金结算状态 |
| 优惠券管理 | 抖音优惠券有效期和使用状态 |
| 月度报告 | 抖音购物返利月度数据分析 |

### 数据分析维度

| 维度 | 说明 |
|------|------|
| 按来源 | 直播间 vs 短视频 vs 商城的返利占比 |
| 按达人 | 各达人/主播带货返利排行 |
| 按品类 | 服饰、美妆、食品等各品类返利统计 |
| 按时间 | 日/周/月返利趋势，识别购物高峰 |
| 退货率 | 各来源的退货率和返利失效率 |

### 使用场景

1. **日常管理**：查看所有抖音返利订单的最新状态
2. **结算确认**：佣金到账通知，确认金额无误
3. **购物复盘**：每月分析抖音购物习惯，看是否理性消费
4. **达人评估**：分析哪些主播推荐的商品性价比最高
