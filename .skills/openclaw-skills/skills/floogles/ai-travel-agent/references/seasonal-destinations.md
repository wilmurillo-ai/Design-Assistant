# Seasonal Destination Guide

Use this when: (a) user doesn't know where to go and needs suggestions, or (b) user wants to go somewhere but doesn't know when, and you need to recommend the best time of year.

## Best Time to Visit — By Destination

### Europe
| Destination | Best Months | Notes |
|---|---|---|
| Barcelona | Apr–Jun, Sep–Oct | Avoid Jul–Aug (extreme heat, crowds) |
| Amalfi Coast | May–Jun, Sep | Jul–Aug = packed, expensive |
| Greek Islands (Santorini, Mykonos) | May–Jun, Sep | Jul–Aug hot and very crowded |
| Croatia (Dubrovnik, Split) | May–Jun, Sep | Jul–Aug overcrowded |
| Portugal (Lisbon, Algarve) | Apr–Jun, Sep–Oct | Mild, less crowded than summer |
| Amsterdam | Apr–May, Sep | Tulip season Apr–May |
| Prague/Vienna | Apr–Jun, Sep–Oct | Winter also charming |
| Iceland | Jun–Aug (midnight sun), Dec–Feb (Northern Lights) | Two very different experiences |
| Scandinavia | Jun–Aug | Only comfortable summer window |
| Alps (ski) | Dec–Mar | Peak ski season |
| Tuscany | Apr–Jun, Sep–Oct | Harvest season Sep–Oct is special |

## Destination Suggestions by Vibe & Season

### Beach / Relaxation
- **Spring (Mar–May):** Canary Islands, Morocco, UAE
- **Summer (Jun–Aug):** Greece, Croatia, Algarve, Italian coast
- **Autumn (Sep–Oct):** Algarve, Cyprus, Crete, Maldives
- **Winter (Nov–Feb):** Maldives, Caribbean, Bali, Thailand

### City Break / Culture
- **Any season:** Lisbon, Porto, Amsterdam, Krakow, Budapest, Prague
- **Winter:** New York, Tokyo, Vienna, Marrakech
- **Spring:** Kyoto (cherry blossom), Paris, Rome

### Adventure / Outdoors
- **Summer:** Iceland, Norway, Alps hiking, Scottish Highlands
- **Winter:** Swiss Alps (ski), Canadian Rockies (ski), Lapland
- **Year-round:** New Zealand, Costa Rica, Peru

### Safari
- **Jun–Oct:** Kenya, Tanzania (Great Migration)
- **Jan–Feb:** South Africa (Kruger, less crowded)

## When Recommending Dates — Calendar Logic
1. Load user's Google Calendar via google-calendar skill
2. Find windows of 4+ days (city break) or 7+ days (holiday) with no conflicts
3. Cross-reference with best months table above
4. Suggest 2–3 date window options ranked by: best season fit + calendar availability
