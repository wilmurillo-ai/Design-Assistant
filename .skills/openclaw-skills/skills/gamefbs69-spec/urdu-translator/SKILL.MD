# Urdu Master Pro Translator

An advanced translation skill for OpenClaw that handles both **Urdu Script** and **Roman Urdu**.

## Features
- **Bi-Directional Context:** Translates any language into fluent Urdu.
- **Script Support:** Choose between native Urdu script or Roman Urdu (English letters).
- **Nuanced Translation:** Understands cultural idioms and natural flow.

## How to use
You can trigger this skill by asking the agent:
- "Translate this into Urdu: [Text]"
- "Write this in Roman Urdu: [Text]"

## Parameters
- `content`: (String) The text you want to translate.
- `format`: (Enum) Either `script` (default) or `roman`.
