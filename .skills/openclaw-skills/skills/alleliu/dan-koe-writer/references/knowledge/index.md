# 积木库知识管理

> 基于 Dan Koe 写作方法论，所有爆款内容拆解后的创意积木存储于此。
> 使用 Obsidian 风格管理，支持双向链接。

---

## 📁 文件结构

```
knowledge/
├── hooks.md              # 钩子库（Attention）
├── paradoxes.md         # 悖论库（Paradoxes）
├── arcs.md              # 转化弧（Transformation Arc）
├── core_problems.md     # 核心问题（Core Problems）
├── golden_phrases.md    # 金句库（Golden Phrases）
├── structures.md        # 结构模式（Structures）
└── perspectives.md      # 观点库（Perspectives）
```

---

## 📝 贡献积木的格式

每条积木按以下格式添加到对应文件：

```markdown
## [积木标题] (来源: 原文链接, 日期: YYYY-MM-DD)

**原文片段:**
> 摘录的原文

**解析:**
为什么这个积木有效？

**适用场景:**
什么样的内容适合使用这个积木？

---
```

---

## 🔄 积木添加流程

1. 用 `goldmine.py` 拆解爆款内容
2. 提取：金句、悖论、转化弧、核心问题、钩子、结构
3. 按格式添加到对应文件
4. 在 `index.md` 更新日期
