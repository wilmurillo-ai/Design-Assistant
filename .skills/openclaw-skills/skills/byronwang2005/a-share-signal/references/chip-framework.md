# 筹码分布框架

当用户希望严格按这套自定义筹码体系判断，而不是按泛化的技术分析口径判断时，读取这份参考。

## 四步结构

1. 筛选：先判断这只股票是否适合进入该体系。
2. 结构：判断筹码结构是否健康。
3. 触发：识别是否出现了真实可执行的入场触发。
4. 风控与退出：定义失效条件和止盈方式。

## 筛选

优先：
- 近期存在催化的股票
- 符合市场主线方向的股票
- 有题材驱动的股票
- 在主动帮用户筛选股票时，优先选择市值100亿元以下且换手充分的股票

回避：
- 长期单边阴跌、没有明显反转迹象的股票
- 基本面明显恶化，例如业绩大幅下滑或重大利空

## 结构

优选结构：

- 低位单峰密集
- 成本带逐步上移
- 筹码在相对狭窄区间内高度集中

偏弱或较差的结构：

- 上方多峰套牢盘
- 大幅拉升后形成高位单峰
- 成本带持续下移

解读规则：

- 如果价格跌到主成本区下方，不要把这个结构说成健康。
- 如果平均成本和主密集区持续下移，应视为派发。
- 如果突破尝试立刻失败，价格重新跌回筹码密集区内部或下方，应判定为触发失败。

## 触发

核心触发是向上突破主筹码区上沿。

默认要求同时满足：

1. 价格站上主筹码区上沿
2. 成交量放大到均量的 `1.5x-2x`
3. 换手率明显提升，小市值品种 `5%+` 尤其重要
4. 上方没有明显套牢区，或已经接近历史新高区域

强但非必须的辅助确认：

- 板块共振
- 龙头属性
- 持续资金流入

## 执行

默认建仓节奏：

- 首次突破：`30%`
- 回踩确认成功：再加 `30%`
- 二次放量上攻：再加 `40%`

避免：

- 追一字板
- 在缩量上涨中重仓

## 风控与退出

先看结构，再看目标价。

目标价的前提：

- 只有在威科夫视角已接近或进入 `markup`、且缠论视角已完成中枢上破并更接近主升段时，才把目标价当作相对明确的上行测算
- 如果价格还运行在箱体内部、仍处于吸筹末期确认前、或缠论上仍在中枢震荡，那么目标价只能写成“突破成立后的条件性量度目标”

目标价测算优先顺序：

1. 箱体高度等幅上移
2. 最近有效中枢高度等幅上移
3. 吸筹区完成后 `markup` 第一段常规量度空间

表达要求：

- 已进入主升时：直接给出第一目标位，并注明量度基础
- 未进入主升时：必须明确写出“目标价成立的前提是假定向上突破箱体 / 中枢上沿”
- 如果出现派发迹象、跌回箱体、或突破不被接受，则该目标价失效

止损逻辑：

- 如果价格跌破筹码密集区下沿，视为退出或清仓信号
- 如果价格跌破密集区中轴，可先减仓

失效信号：

- 放量下跌
- 成本带开始下移
- 关键支撑跌破后看不到承接

止盈逻辑：

- 出现新的高位筹码峰
- 形成多峰结构
- 反复缩量上涨，说明上攻乏力
- 高换手但价格不前进，说明可能在派发

分批止盈参考：

- `+10%`：减 `30%`
- `+20%`：减 `30%`
- 剩余仓位跟随趋势

## 核心原则

- 重结构，不重拍脑袋预测价格
- 筹码分析是辅助工具，不是孤立系统
- 必须结合成交量、换手率和情绪环境
- 风险控制优先于收益最大化
