# 网红城市

## 筛选项
- 难度/里程/海拔/路况（铺装/土路）
- 季节/气候/最佳时段/是否需要预约或许可
- 安全（补给点/紧急撤退/信号覆盖）/亲子或萌宠友好

## 返回字段
- 路线图/用时/交通方式/补给点与洗手间
- 风景亮点/拍摄位/人流情况/风险点
- 天气预警/装备清单/环保与无痕原则

## 示例请求
- “杭州 网红城市 雨季友好路线（坡度≤8%）”
- “深圳 网红城市 城市步道串联（地铁直达）”
