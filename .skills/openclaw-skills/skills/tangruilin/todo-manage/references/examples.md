# 使用示例

## 创建待办

**用户**：新增待办 完成项目文档

**助手**：
1. 创建待办「完成项目文档」
2. 是否关联 GitHub issue？请提供链接，或回复"不需要"。

**用户**：https://github.com/owner/repo/issues/123

**助手**：
已关联 issue。是否关联路径？可提供多个路径，回复"完成"结束。

**用户**：/Users/reilly/project-a

**助手**：
检测到 git 项目。当前分支：main。使用此分支还是指定其他？

**用户**：就用 main

**助手**：
已添加路径 `/Users/reilly/project-a` (分支: main)。继续添加路径，或回复"完成"结束。

**用户**：完成

**助手**：
```

   创建时间: 2026-04-04 21:00
   状态: 未开始
```
