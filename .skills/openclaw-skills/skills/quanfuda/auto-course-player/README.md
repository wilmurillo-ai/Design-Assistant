# 自动网课播放器 - 使用指南 🎓

## ⚠️ 重要提醒

**使用前请确认：**
- ✅ 符合学校规定和学术诚信政策
- ✅ 了解可能的风险（账号检测、学习记录异常等）
- ✅ 建议合理使用，不要完全跳过内容

---

## 📋 准备工作

### 1. 安装 Chrome 扩展

确保已安装 **OpenClaw Browser Relay** 扩展：

1. 打开 Chrome 网上应用店
2. 搜索 "OpenClaw Browser Relay"
3. 安装并启用

### 2. 登录网课平台

1. 打开 Chrome 浏览器
2. 访问：https://studentjxjyzx.qdu.edu.cn/
3. 使用手机扫码登录
4. 进入课程列表页面
5. **保持浏览器打开状态**

### 3. 启用 Browser Relay

1. 点击 Chrome 工具栏的 OpenClaw 扩展图标
2. 确保扩展已启用（图标高亮）
3. 保持当前标签页激活

## 🔧 故障排除

### 问题 1: 浏览器无法接管

**解决：**
1. 确保 Chrome 扩展已启用
2. 刷新扩展权限
3. 重启 Chrome 浏览器

### 问题 2: 视频不自动播放

**解决：**
1. 检查浏览器是否允许自动播放
2. 手动点击一次播放按钮
3. 检查音量是否静音

### 问题 3: 无法切换到下一课

**解决：**
1. 检查页面结构是否变化
2. 手动点击下一课
3. 告诉我页面布局，我调整脚本

### 问题 4: 登录过期

**解决：**
1. 重新扫码登录
2. 告诉我"继续播放"
3. 从上次进度恢复

## ⚡ 最佳实践

1. **分批完成** - 每次 2-3 小时，避免异常检测
2. **定期检查** - 每 30 分钟查看一次进度
3. **保持网络** - 确保网络连接稳定
4. **人工介入** - 遇到验证码立即处理
5. **保存进度** - 停止前确保进度已保存
