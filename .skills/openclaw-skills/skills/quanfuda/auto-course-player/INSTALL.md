## 📝 第 2 步：安装自动播放脚本

### 方法 A：直接复制安装（推荐）

**1. 打开 Tampermonkey 仪表板**
- 点击 Chrome 右上角的 Tampermonkey 图标（🧩 拼图形状）
- 选择"添加新脚本"或"管理面板"

**2. 复制以下代码**

打开文件：`/Users/fuda/.openclaw/workspace/skills/auto-course-player/scripts/course-autoplay.user.js`

全选复制所有内容（Cmd+A → Cmd+C）

**3. 粘贴到 Tampermonkey**
- 在 Tampermonkey 编辑器中，删除所有默认内容
- 粘贴复制的代码
- 按 `Cmd+S` 保存
- 看到"已安装"提示即成功

## 🎮 使用自动播放器

**1. 访问网课平台**
```
https://studentjxjyzx.qdu.edu.cn/
```

**2. 扫码登录并进入课程**

**3. 页面右上角会出现控制面板**
```
┌─────────────────────────┐
│ 🎓 网课自动播放      × │
├─────────────────────────┤
│ 状态：⏸️ 已暂停          │
│ 已完成：0 个视频         │
├─────────────────────────┤
│   ▶️ 启动自动播放        │
└─────────────────────────┘
```

**4. 点击"启动自动播放"**

系统会自动：
- ✅ 静音播放当前视频
- ✅ 检测视频播放完成
- ✅ 自动切换到下一课
- ✅ 关闭弹窗
- ✅ 继续静音播放

## ⚙️ 配置选项

在脚本顶部可以修改：

```javascript
const CONFIG = {
    checkInterval: 10000,        // 检查间隔（10 秒）
    videoEndThreshold: 0.95,     // 95% 视为完成
    autoPlayDelay: 2000,         // 切换后 2 秒播放
    debugMode: false             // 调试模式
};
```

## 🛑 控制自动播放

- **暂停/继续** - 点击控制面板的按钮
- **关闭面板** - 点击面板右上角的 ×
- **重新打开** - 刷新页面即可

## 📊 查看进度

## ❓ 故障排除

**问题：控制面板不显示**
- 刷新页面
- 检查 Tampermonkey 是否启用
- 检查脚本是否已启用（Tampermonkey 仪表板）

**问题：视频不自动播放**
- 检查浏览器是否允许自动播放
- 手动点击一次播放按钮
- 检查脚本是否启用

**问题：不切换下一课**
- 检查课程列表结构是否匹配
- 查看浏览器控制台（F12）是否有错误
