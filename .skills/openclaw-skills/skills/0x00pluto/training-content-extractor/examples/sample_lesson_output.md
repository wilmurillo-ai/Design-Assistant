# 示例：Lesson 细致文档（节选）

## L1 平替逻辑与对标判断

### 本课要解决的问题
- 如何判断一个项目是否具有可执行的确定性？
- 如何避免只看故事、不看成本结构？

### 要点细化（结论→解释→做法→边界）
1. 先做对标，再谈创意  
   - 结论：没有可比较对标，确定性判断容易失真。  
   - 解释：对标能提供成本与收益基准线。  
   - 做法：建立“成本/收益/风险”对照表。  
   - 边界条件：对标对象必须同品类或同消费场景。  

### 案例拆解（来自原文）
- 案例背景：烧饼门店与低成本摊位模型对比。
- 拆解步骤：先比成本，再比收益，再看风险。
- 可复用方法：任何线下轻创业项目都可套用该框架。

### 作业（提交物）
- 提交物名称：《项目确定性评估表》
- 提交格式：1 页表格
- 完成标准：至少含 3 条关键假设及验证方式
