# Polymarket Command Center

Read-only Polymarket interface for OpenClaw — trending markets, detailed odds, search, and watchlists. Zero API key required.

## Install

```bash
clawhub install polymarket-command-center
```

## Quick Start

No API key needed. Install and start using immediately.

## Commands

| Command | Description |
|---------|-------------|
| `trending` | Top 10 markets by trading volume |
| `odds [market]` | Detailed probability bars for a market |
| `search [query]` | Find active markets by keyword |
| `watchlist` | Track your configured markets |
