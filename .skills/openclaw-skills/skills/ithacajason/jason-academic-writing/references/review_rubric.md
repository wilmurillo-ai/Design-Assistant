# Review Rubric - Detailed Scoring Criteria

## Editor-in-Chief (30% weight)

### Novelty (25 points)
| Score | Criteria |
|-------|----------|
| 25 | Novel approach to significant problem, clear innovation |
| 20 | Incremental improvement with clear contribution |
| 15 | Known method applied to new domain |
| 10 | Replication or minor extension |
| 5 | No clear novelty |

### Significance (25 points)
| Score | Criteria |
|-------|----------|
| 25 | Paradigm-shifting impact on field |
| 20 | Practical impact on practitioners |
| 15 | Interesting to subset of field |
| 10 | Limited applicability |
| 5 | Marginal contribution |

### Journal Fit (25 points)
| Score | Criteria |
|-------|----------|
| 25 | Perfect fit, highly relevant to readership |
| 20 | Good fit, appropriate scope |
| 15 | Somewhat relevant |
| 10 | Tangentially related |
| 5 | Not appropriate for journal |

### Writing Quality (25 points)
| Score | Criteria |
|-------|----------|
| 25 | Excellent structure, clear arguments, polished |
| 20 | Good structure, minor issues |
| 15 | Adequate, some confusion |
| 10 | Poor organization, hard to follow |
| 5 | Unreadable |

---

## Methodology Reviewer (25% weight)

### Research Design (30 points)
| Score | Criteria |
|-------|----------|
| 30 | Rigorous design, controls, clear hypotheses |
| 25 | Solid design with minor gaps |
| 20 | Acceptable design |
| 15 | Weak design, confounding factors |
| 10 | Flawed design |

### Data Quality (25 points)
| Score | Criteria |
|-------|----------|
| 25 | Large sample, reliable source, clean data |
| 20 | Adequate sample, good source |
| 15 | Limited sample, questionable source |
| 10 | Small sample, unreliable |
| 5 | Insufficient data |

### Statistical Analysis (25 points)
| Score | Criteria |
|-------|----------|
| 25 | Appropriate methods, correct interpretation |
| 20 | Correct methods, minor interpretation issues |
| 15 | Acceptable methods |
| 10 | Questionable methods or interpretation |
| 5 | Flawed analysis |

### Reproducibility (20 points)
| Score | Criteria |
|-------|----------|
| 20 | Full code, data, clear documentation |
| 15 | Code available, partial data |
| 10 | Methods described, no code |
| 5 | Insufficient detail |
| 0 | Not reproducible |

---

### Related Work (30 points)
| Score | Criteria |
|-------|----------|
| 30 | Comprehensive, accurate, critical analysis |
| 25 | Good coverage, minor gaps |
| 20 | Adequate coverage |
| 15 | Significant gaps or uncritical |
| 10 | Major omissions |

### Theory Framework (30 points)
| Score | Criteria |
|-------|----------|
| 30 | Solid theoretical foundation, clear definitions |
| 25 | Good foundation, minor gaps |
| 20 | Adequate framework |
| 15 | Weak theoretical basis |
| 10 | Missing framework |

### Literature Review (25 points)
| Score | Criteria |
|-------|----------|
| 25 | Comprehensive synthesis, critical evaluation |
| 20 | Good synthesis |
| 15 | Adequate listing |
| 10 | Minimal review |
| 5 | Missing |

### Field Contribution (15 points)
| Score | Criteria |
|-------|----------|
| 15 | Advances field understanding |
| 12 | Useful contribution |
| 8 | Minor contribution |
| 5 | No clear contribution |
| 0 | None |

## Devil's Advocate (15% weight)

### Weakness Detection (40 points)
| Score | Criteria |
|-------|----------|
| 40 | Identifies fatal flaw(s) |
| 30 | Identifies significant weaknesses |
| 20 | Identifies moderate weaknesses |
| 10 | Minor criticisms only |
| 5 | No substantial criticism |

### Counterexamples (30 points)
| Score | Criteria |
|-------|----------|
| 30 | Strong counterexamples with evidence |
| 25 | Reasonable counterexamples |
| 20 | Possible counterexamples |
| 10 | Speculative only |
| 5 | None |

### Overclaim Detection (20 points)
| Score | Criteria |
|-------|----------|
| 20 | Identifies all overclaims |
| 15 | Identifies major overclaims |
| 10 | Identifies some |
| 5 | Misses significant ones |
| 0 | None detected |

### Alternative Explanations (10 points)
| Score | Criteria |
|-------|----------|
| 10 | Plausible alternative with support |
| 8 | Reasonable alternative |
| 5 | Speculative alternative |
| 0 | None |

---

## Decision Thresholds

| Score Range | Decision | Next Step |
|-------------|----------|-----------|
| ≥80 | Accept | Proceed to Finalize |
| 65-79 | Minor Revision | Revise + Re-check integrity |
| 50-64 | Major Revision | Revise + Re-review |
| <50 | Reject | Return to Research/Write |

---

## Revision Priority Mapping

| Priority | Description | Must Fix? |
|----------|-------------|-----------|
| High | Fatal flaw or critical issue | Yes |
| Medium | Significant weakness | Recommended |
| Low | Minor improvement | Optional |
