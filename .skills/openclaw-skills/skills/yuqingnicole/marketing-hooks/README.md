# Marketing Hooks

**Puzzle-Driven Content Model for Marketing**

An AI skill for creating compelling marketing hooks and content structures based on proven psychological frameworks.

## 🎯 What This Skill Does

Helps you create attention-grabbing marketing content using the **Puzzle-Driven Model** - a framework that leverages how the human brain responds to questions and extreme cases.

## 🧠 Core Framework

### Four Question Types

1. **How-Type** (Method) - "如何快速成为有钱人？"
2. **Why-Type** (Reason) - "为什么可以成为董事长？"
3. **What-Type** (Definition) - "什么是供给侧改革？"
4. **WoW-Type** (Extreme Case) ⭐ - "00后用 AI 月入10万"

### Content Tree Structure

```
问题 (Question) = 树根 (Root)
    ↓
观点 (Viewpoint) = 树干 (Trunk)
    ↓
理由 (Reasoning) = 树枝 (Branches)
    ↓
证据 (Evidence) = 树叶 (Leaves)
```

## 🎓 Key Concepts

### WoW-Type Content (Highest Engagement)

**Formula:** Create probability-defying extreme cases

- **Discover extremes** → Get traffic
- **Become extreme** → Build brand

**Why it works:** The brain automatically fills in missing logic, creating engagement stickiness.

### Gan Method Content Model

1. **Core viewpoint** (树干)
2. **Sub-arguments** (树枝)
3. **Evidence** (树叶) - Must be:
   - Unexpected (出乎意料)
   - Memorable (印象深刻)
   - Vivid (生动鲜活)
