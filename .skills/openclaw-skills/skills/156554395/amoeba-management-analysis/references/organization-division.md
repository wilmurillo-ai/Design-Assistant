# 阿米巴组织划分指南

## 划分原则

阿米巴划分的三个条件：
1. **独立核算** — 收入和支出可以明确计算
2. **独立业务** — 具有完整的业务流程
3. **贯彻整体** — 与公司整体目标一致

## 划分维度

| 维度 | 适用场景 | 示例 |
|------|---------|------|
| 按产品 | 不同产品线差异大 | A产品巴、B产品巴 |
| 按工序 | 流水线生产 | 切削巴、研磨巴、组装巴 |
| 按区域 | 区域差异明显 | 华北巴、华南巴 |
| 按客户 | 大客户独立运营 | X客户巴、Y客户巴 |
| 按职能 | 支持型部门 | 设计巴、IT巴、物流巴 |

## 划分步骤

1. 梳理企业价值链（从采购到销售）
2. 识别可独立核算的环节
3. 评估各环节的收入来源和成本结构
4. 确定阿米巴的合理规模（通常10-30人）
5. 明确各阿米巴的核算边界
6. 建立内部交易机制

## 常见问题

- **太大** → 核算不精细，员工参与感弱
- **太小** → 管理成本过高，协调困难
- **边界不清** → 内部扯皮，核算失真
- **支持部门不划分** → 缺乏成本意识，效率低下

## 分析要点

做组织划分分析时重点关注：
1. 现有组织是否满足独立核算三条件
2. 各阿米巴规模是否合理（人数、产值）
3. 内部交易链是否完整闭环
4. 支持部门（职能部门）是否纳入核算体系
