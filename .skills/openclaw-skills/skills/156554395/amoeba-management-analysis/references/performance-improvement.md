# 业绩改善诊断

## 诊断框架

从三个维度诊断阿米巴业绩：

### 维度一：单位时间附加值
```
附加值 = 收入 - 外购成本
提升路径：增收（提价/增量/结构优化）或 降本（材料/外协/杂费）
```

### 维度二：总劳动时间
```
效率提升路径：
- 减少无效时间（等待、返工、会议）
- 提高产出速度（改进工艺、自动化）
- 优化排班（消除闲置工时）
```

### 维度三：人均产值
```
人均产值 = 附加值 ÷ 人数
提升路径：提高技能、优化流程、合理调配
```

## 常见改善工具

1. **PDCA循环** — 计划→执行→检查→行动
2. **瓶颈分析** — 找到限制产出的最弱环节
3. **价值流图** — 识别增值与非增值活动
4. **阿米巴经营会议** — 每月召开，巴长汇报、全员讨论
5. **改善提案制度** — 鼓励一线员工提出改善建议

## 诊断检查清单

- [ ] 单位时间附加值是否达成月度目标？
- [ ] 与上月/同期相比，变动的主要原因是什么？
- [ ] 材料费占收比是否合理？有无异常波动？
- [ ] 加班工时是否过多？原因是什么？
- [ ] 内部交易的定价是否需要调整？
- [ ] 有无因前工序问题导致本阿米巴效率下降？
- [ ] 下月的改善目标和具体措施是什么？
