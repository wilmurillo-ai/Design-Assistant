# 经营会计与财务会计对比

## 核心区别

| 维度 | 财务会计 | 经营会计（阿米巴） |
|------|---------|-----------------|
| 目的 | 向外部报告经营成果 | 向内部提供决策依据 |
| 对象 | 企业整体 | 各阿米巴单元 |
| 频率 | 月/季/年 | 日/周/月 |
| 核心 | 利润 | 单位时间附加值 |
| 规则 | 会计准则 | 企业自定（简洁易懂） |
| 受众 | 股东/监管 | 全体员工 |
| 导向 | 过去结果 | 未来改善 |

## 经营会计报表体系

### 1. 阿米巴经营核算表（月度）
各阿米巴每月填报，反映收入、成本、附加值、单位时间核算。

### 2. 阿米巴经营汇总表（月度）
汇总所有阿米巴数据，呈现公司整体经营状况。

### 3. 阿米巴间交易明细表
记录内部交易：谁卖给谁、数量、单价、金额。

### 4. 费用分摊表
间接费用按合理基准分摊到各阿米巴。

## 费用分摊方法

| 费用类型 | 分摊基准 | 示例 |
|---------|---------|------|
| 厂房租金 | 面积 | 按各巴使用面积 |
| 水电气 | 用量 | 按各巴实际用量 |
| 管理人员工资 | 人数 | 按各巴人数比例 |
| IT费用 | 使用量 | 按各巴系统使用量 |
| 折旧 | 资产归属 | 按各巴使用资产 |

**原则：** 能直接归集的不分摊；必须分摊的用最合理的基准；保持简单透明。
