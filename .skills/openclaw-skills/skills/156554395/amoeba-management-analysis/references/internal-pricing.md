# 内部交易定价

## 核心原则

阿米巴之间的交易定价遵循：
1. **市场原则** — 参考市场价格，不搞内部保护
2. **双方认可** — 买卖双方阿米巴协商同意
3. **透明公开** — 定价依据全员可见
4. **及时调整** — 市场变化时及时修正

## 定价方法

| 方法 | 适用场景 | 定价依据 |
|------|---------|---------|
| 市场价法 | 有外部参考市场 | 外部市场价 × 折扣 |
| 成本加成法 | 无外部市场 | 成本 × (1+利润率) |
| 协商定价 | 双方有议价空间 | 买卖双方协商 |
| 目标利润倒推 | 有明确利润目标 | 售价 - 目标利润 = 内部价 |

## 内部交易分析要点

1. 内部交易占各阿米巴收入的比例
2. 内部价格与外部市场价格的偏差
3. 上下游阿米巴之间的利润分配是否合理
4. 是否存在"内部保护"导致竞争力下降
5. 内部交易的结算是否及时准确
