# 单位时间核算详解

## 核心概念

单位时间核算是阿米巴经营的核心工具，用"每小时创造的附加值"衡量各阿米巴的经营绩效。

与传统财务会计的区别：
- 财务会计：面向过去、面向股东、按月/季/年
- 经营会计：面向未来、面向全员、按日/周/月实时

## 核算公式

```
=== 收入侧 ===
生产总值（总产值）= 外部销售额 + 内部销售额 + 库存增减

=== 成本侧 ===
外部采购成本 = 原材料费 + 外协加工费 + 杂费（差旅、通信等）

=== 附加值 ===
附加值 = 生产总值 - 外部采购成本

=== 单位时间指标 ===
总劳动时间 = 正常工时 + 加班工时
单位时间附加值 = 附加值 ÷ 总劳动时间
单位时间人事费 = 人事费总额 ÷ 总劳动时间
单位时间利润 = 单位时间附加值 - 单位时间人事费
```

## 关键分析维度

1. **纵向对比** — 本月 vs 上月、本月 vs 去年同期
2. **横向对比** — 同类阿米巴之间的对比
3. **目标对比** — 实际 vs 年度计划/月度目标
4. **趋势分析** — 连续12个月单位时间附加值走势
