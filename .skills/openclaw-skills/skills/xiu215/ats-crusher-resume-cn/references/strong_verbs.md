- 压缩 / 精简 / 减少
- 加速 / 优化 / 改善

### 数据类

## 需避免的弱动词

❌ 做了 / 参与了 / 帮助了
❌ 负责了 / 完成了 / 处理过
❌ 使用过 / 学习过 / 了解过
❌ 协助 / 辅助 / 支持

## 动词强度对比示例

### 弱表达 → 强表达
- "负责后端开发" → "主导后端架构设计与核心功能开发"
- "参与了性能优化" → "重构核心模块，提升系统性能 30%"
- "使用过 Python" → "基于 Python 构建数据处理流水线，支撑千万级数据分析"
- "帮助团队完成任务" → "带领 5 人团队，3 周内交付核心业务系统"
