# ATS 关键词指南 - 简历筛选系统优化

## ATS 工作原理

ATS (Applicant Tracking System) 是招聘企业使用的简历筛选系统，会自动扫描简历中的关键词匹配度。

## 关键词使用技巧

### ✓ 正确做法
- 在技能部分列出核心技术栈
- 在工作经历中自然地融入技术关键词
- 使用标准的技术术语
- 既写全称也写缩写（如: "Amazon Web Services (AWS)"）

### ✗ 错误做法
- 关键词堆砌（无上下文）
- 使用非标准缩写
- 拼写错误
- 使用图片展示技能（ATS 无法识别）
