# SEO Checklist

Use this checklist before publishing any content.

## 🔍 On-Page SEO

### Title Tags
- [ ] Include target keyword near the beginning
- [ ] Keep under 60 characters
- [ ] Make it compelling and clickable
- [ ] Unique for each page

### Meta Descriptions
- [ ] Include target keyword
- [ ] Keep under 160 characters
- [ ] Add a clear call-to-action
- [ ] Unique for each page

### Content
- [ ] Include target keyword in first 100 words
- [ ] Use H1 for main heading
- [ ] Use H2-H3 for subheadings
- [ ] Add relevant images with alt text
- [ ] Include internal links (2-5)
- [ ] Include external links to authoritative sources
- [ ] Content is 500+ words (1000+ recommended)

### URL Structure
- [ ] Use hyphens (-) to separate words
- [ ] Include target keyword
- [ ] Keep URLs short and descriptive
- [ ] Use lowercase only

### Images
- [ ] Add descriptive alt text
- [ ] Compress images for fast loading
- [ ] Use descriptive file names

## ⚡ Technical SEO

### Performance
- [ ] Page loads under 3 seconds
- [ ] Mobile-friendly design
- [ ] Minified CSS/JS
- [ ] Enable browser caching

### Structure
- [ ] Valid HTML
- [ ] Proper heading hierarchy (H1 → H2 → H3)
- [ ] XML sitemap submitted to Google Search Console
- [ ] robots.txt configured

### Schema Markup
- [ ] Add structured data for articles
- [ ] Include schema for organization
- [ ] Test with Rich Results Test

## 🔗 Link Building

### Internal Links
- [ ] Link to related content
- [ ] Use descriptive anchor text
- [ ] Create logical site hierarchy

### External Links
- [ ] Link to authoritative sources
- [ ] No broken links
- [ ] Open external links in new tabs (optional)

## 📱 Mobile SEO

- [ ] Responsive design
- [ ] Readable without zooming
- [ ] Tap targets are large enough
- [ ] No horizontal scrolling

## 🧪 Testing

- [ ] Test with Google PageSpeed Insights
- [ ] Test with Google Mobile-Friendly Test
- [ ] Check Google Search Console for errors
- [ ] Verify structured data with Rich Results Test
