# SEO Best Practices

## 🎯 Keyword Strategy

### Keyword Research
- **Focus on intent**: Understand what users are searching for
- **Long-tail keywords**: Less competition, higher conversion
- **Keyword clustering**: Group related keywords together
- **Monitor competitors**: See what keywords they rank for

### Keyword Placement
| Location | Priority | Tips |
|----------|----------|------|
| Title | 🔴 High | First 50 characters most important |
| H1 | 🔴 High | Include main keyword |
| First 100 words | 🟠 Medium | Natural inclusion |
| Subheadings | 🟠 Medium | LSI keywords |
| Alt text | 🟡 Lower | Describe image context |
| URL | 🟠 Medium | Include primary keyword |

## 📝 Content Best Practices

### Content Quality
1. **E-E-A-T**: Experience, Expertise, Authoritativeness, Trustworthiness
2. **Original content**: Never duplicate from other sources
3. **Comprehensive**: Cover topics thoroughly
4. **Updated regularly**: Keep content fresh
5. **Answer questions**: Address user intent

### Content Structure
```
H1: Main Topic
├── H2: Major Section
│   ├── H3: Sub-section
│   └── H3: Sub-section
├── H2: Major Section  
│   ├── H3: Sub-section
│   └── H3: Sub-section
└── H2: Conclusion
```

### Word Count Guidelines
| Type | Recommended |
|------|-------------|
| Blog posts | 1000-2000 |
| Product pages | 300-500 |
| Guides | 2000-5000 |
| Landing pages | 500-1000 |

## 🔧 Technical Best Practices

### Page Speed
- **Target**: Under 3 seconds
- **Tools**: PageSpeed Insights, GTmetrix
- **Optimizations**:
  - Compress images (WebP format)
  - Minify CSS/JS
  - Enable lazy loading
  - Use CDN
  - Enable caching

### Mobile-First
- **Design mobile-first**: Mobile is primary index
- **Test on real devices**: Don't rely only on emulators
- **Touch targets**: Minimum 48x48px
- **Viewport**: Proper meta viewport tag

### Schema Markup
Recommended types:
- `Article` - Blog posts, news
- `Product` - E-commerce products
- `FAQPage` - FAQ content
- `HowTo` - Tutorial content
- `Organization` - Business info
- `LocalBusiness` - Local businesses

## 🔗 Link Building

### Internal Links
- **Anchor text**: Use descriptive text, not "click here"
- **Contextual**: Links within content, not navigation only
- **Hierarchy**: Main pages get more internal links
- **Deep links**: Link to subpages, not just homepage

### External Links
- **Authority**: Link to trustworthy sites (.edu, .gov, established sites)
- **Relevance**: Link to content that supports your points
- **No-follow**: Consider for paid links or untrusted content
- **Don't overdo**: Too many external links can look spammy

## 📊 Analytics & Monitoring

### Key Metrics
- **Organic traffic**: Visitors from search
- **Keyword rankings**: Position in SERPs
- **Click-through rate (CTR)**: Impressions vs clicks
- **Bounce rate**: Users leaving quickly
- **Time on page**: Engagement indicator

### Tools to Use
| Tool | Purpose |
|------|---------|
| Google Search Console | Performance, indexing |
| Google Analytics | Traffic analysis |
| Ahrefs/SEMrush | Keyword research |
| PageSpeed Insights | Performance |
| Screaming Frog | Technical audit |

## 🚫 Common Mistakes

### Avoid
1. **Keyword stuffing**: Don't overuse keywords
2. **Duplicate content**: Each page should be unique
3. **Ignoring mobile**: Mobile-first is mandatory
4. **Slow pages**: Speed affects rankings
5. **No clear CTA**: Guide users to action
6. **Thin content**: Provide real value

### Never Do
- Buy links (violates guidelines)
- Hide content from users
- Use doorway pages
- Copy content from other sites
- Spam comment sections

## 📅 Ongoing Tasks

### Weekly
- Monitor keyword rankings
- Check for broken links
- Review analytics

### Monthly
- Update old content
- Add new internal links
- Check competitor rankings

### Quarterly
- Technical audit
- Content gap analysis
- Link profile review

## Quick Reference

| Do | Don't |
|----|-------|
| Write for users first | Write for search engines |
| Use descriptive URLs | Use generic URLs like /page1 |
| Add alt text to images | Leave alt text empty |
| Update content regularly | Publish and forget |
| Build quality links | Buy or spam links |
| Be patient | Expect instant results |

**Remember**: SEO is a marathon, not a sprint. Focus on providing value to users and the rankings will follow.
