## Twitter/X

### Platform Overview
- **Monthly Active Users:** 400M+
- **Best For:** News, updates, conversations, threads, real-time engagement
- **Algorithm:** Recency + Engagement + Relevance

### Technical Specs
| Feature | Free Account | Premium |
|---------|--------------|---------|
| Character Limit | 280 | 25,000 |
| Images per tweet | 4 | 4 |
| Videos | 2:20 max | 2:20 max |
| Hashtag limit | No strict limit | No strict limit |
| Optimal hashtags | 2-3 | 2-3 |

### Best Posting Times (Local Time)
- **Weekdays:** 9:00 AM, 12:00 PM, 5:00 PM - 6:00 PM
- **Best Days:** Tuesday, Wednesday, Thursday
- **Frequency:** 3-5 tweets/day for active accounts

### Content Strategy
1. **Threads** - Use for longer content (up to 25 posts)
2. **Polls** - High engagement driver
3. **Quote Tweets** - Add value, don't just RT
4. **Replies** - Engage within first hour

### Do's and Don'ts
✅ **DO:**
- Keep hooks in first 280 characters
- Ask questions to drive replies
- Use 1-2 relevant hashtags
- Post consistently
- Engage with replies

❌ **DON'T:**
- Tweet only links without context
- Use more than 3 hashtags
- Over-promote (keep it 80/20 ratio)
- Ignore mentions
- Be overly promotional

### Engagement Tips
- Reply to trending topics when relevant
- Use GIFs strategically
- Pin important tweets
- Embed media directly (not links)
- Cross-promote with other accounts

## Instagram

### Platform Overview
- **Monthly Active Users:** 2B+
- **Best For:** Visual storytelling, brand awareness, community
- **Algorithm:** Engagement + Relationships + Timeliness + Profile Searches

### Technical Specs
| Feature | Limit |
|---------|-------|
| Caption length | 2,200 characters |
| Hashtags | 30 max (optimal: 5-15) |
| Image size | 1080 x 1080 (square), 1080 x 1350 (portrait) |
| Video length | 60 seconds (feed), 15 seconds (stories) |
| Reels | 90 seconds max |

### Best Posting Times (Local Time)
- **Weekdays:** 11:00 AM - 1:00 PM, 7:00 PM - 9:00 PM
- **Best Days:** Monday, Wednesday, Thursday
- **Frequency:** 3-5 feed posts/week, daily stories

### Content Types (Best Engagement)
1. **Reels** - Highest reach potential
2. **Carousels** - Highest save rate
3. **Single Images** - Good for consistency
4. **Stories** - Daily engagement, ephemeral

### Do's and Don'ts
✅ **DO:**
- Put key info in first 125 characters
- Use line breaks for readability
- End with CTA ("Save this", "Tag a friend")
- Use relevant hashtags (mix of sizes)
- Post Stories daily

❌ **DON'T:**
- Write novels in captions
- Use broken/irrelevant hashtags
- Post too frequently (flooding)
- Ignore DMs and comments
- Use watermarked content (Reels)

### Hashtag Strategy
```
Industry (2-3): #TechNews #Innovation
Niche (3-5): #StartupTips #FounderLife  
Branded (1-2): #YourBrand
Popular (2-3): #Trending #Explore
```

### Instagram Story Best Practices
- Post 3-5 stories daily
- Use stickers (polls, questions, sliders)
- Mix content: BTS, memes, product, engagement
- Add highlights to categorize

## LinkedIn

### Platform Overview
- **Monthly Active Users:** 900M+
- **Best For:** B2B, professional networking, thought leadership
- **Algorithm:** Engagement + Connections + Timeliness

### Technical Specs
| Feature | Limit |
|---------|-------|
| Post length | 3,000 characters (optimal: 1,500-2,000) |
| Hashtags | 10 max (optimal: 3-5) |
| Comments | Unlimited |
| Attachments | Documents, images, videos |

### Best Posting Times (Local Time)
- **Weekdays:** 8:00 AM, 12:00 PM, 5:00 PM
- **Best Days:** Tuesday, Wednesday, Thursday
- **Frequency:** 2-5 posts/week

### Content Strategy
1. **Personal Stories** - Authenticity wins
2. **Professional Insights** - Share expertise
3. **Industry News** - Commentary adds value
4. **How-To Posts** - Educational content
5. **Milestones** - Celebrate wins

### Post Structure (High Engagement)
```
[Personal hook or question]
-
[Main content with 2-3 key points]
Use bullet points or numbered lists
-
[Call to action / Question]
"What's your experience?" 👇
[3-5 relevant hashtags]
```

### Do's and Don'ts
✅ **DO:**
- Start with a hook in first line
- Use professional yet personal tone
- Add relevant media (images, documents)
- Engage with comments within first hour
- Post consistently (2-5x/week)

❌ **DON'T:**
- Be overly salesy
- Copy-paste everywhere
- Write walls of text
- Use excessive hashtags
- Auto-post without customization

### LinkedIn Newsletter
- Weekly or bi-weekly consistent schedule
- 1,500+ words per edition
- Build subscribers organically
- Repurpose blog content

## Facebook

### Platform Overview
- **Monthly Active Users:** 3B+
- **Best For:** Community building, local business, brand awareness
- **Algorithm:** Friends/Family > Groups > Pages

### Technical Specs
| Feature | Limit |
|---------|-------|
| Post length | 63,206 characters |
| Hashtag limit | Facebook discourages hashtags |
| Video length | 240 minutes (up to 4GB) |
| Image size | 1,200 x 630 (recommended) |

### Best Posting Times (Local Time)
- **Weekdays:** 1:00 PM - 4:00 PM
- **Evenings:** 8:00 PM - 9:00 PM
- **Weekends:** 10:00 AM - 1:00 PM
- **Frequency:** 1-2 posts/day

### Content Strategy
1. **Native Videos** - Highest engagement (not links)
2. **Questions** - Drive comments
3. **Behind-the-scenes** - Humanizes brand
4. **User-generated content** - Social proof
5. **Events** - If applicable

### Do's and Don'ts
✅ **DO:**
- Use native video (not YouTube links)
- Ask open-ended questions
- Share authentic content
- Engage in Groups
- Keep posts conversational

❌ **DON'T:**
- Use too many hashtags (1-2 max)
- Only post links without value
- Over-promote
- Ignore comments/messages
- Buy fake engagement

### Facebook Groups
- More reach than Pages
- Build community
- Consistent engagement required
- Valuable for local/niche markets

## Cross-Platform Tips

### Content Repurposing Matrix
| Original | Twitter | Instagram | LinkedIn | Facebook |
|----------|---------|-----------|----------|----------|
| Blog Post | Thread | Carousel | Article | Summary + Video |
| Video | Clips | Reels | Short | Native Video |
| Podcast | Clips | Reels | Highlights | Clips |
| Customer Quote | Tweet | Post | Article | Post |

### Universal Best Practices
1. **Know your audience** - Different platforms, different personas
2. **Be consistent** - Regular posting builds algorithm favor
3. **Engage authentically** - Real interactions > automated
4. **Use analytics** - Track what works, double down
5. **Stay current** - Adapt to platform changes
