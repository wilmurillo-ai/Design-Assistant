# 获取B站热门视频排行榜
## 核心功能
1. 无需key值即可直接获取bilibili热门推荐视频的名称和UP主名字
2. 实时抓取 Bilibili 首页“热门”分区的视频榜单（默认前20）
3. 结构化输出 每条视频包含： 视频标题 、UP 主（作者）昵称 
4. 自动更新 每次调用均返回最新榜单，确保内容时效性

## 使用方法
### 直接使用以下任一指令即可触发：
1. 获取B站热门视频
2. 获取bilibili热门视频排行榜
3. 帮我看看现在B站最火的视频有哪些

### 使用场景
1. 想快速知道今天 B 站大家都在看什么？
2. 寻找近期爆火的优质内容或潜力 UP 主？
3. 做内容分析、热点追踪或竞品调研？

## 脚本位置
- 脚本路径：`scripts/bilibili-hot-recommend.py`
- 数据来源：Bilibili 官方热门推荐接口（无需登录）
- 输出格式：按热度从高到低排序的列表，每项包含【标题 + 作者】
💡 提示：该榜单反映的是全站综合热度（包括播放、点赞、投币、收藏、评论等维度），是了解中文互联网年轻群体兴趣风向的重要窗口
