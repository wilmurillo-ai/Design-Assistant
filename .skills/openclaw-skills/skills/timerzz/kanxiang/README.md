## 使用

### 基本用法

1. **上传照片并说明分析类型**：
   ```
   帮我看看这个面相 [上传照片]
   ```

2. **或者直接上传照片**：
   ```
   [上传照片]
   技能会自动识别照片类型并进行分析
   ```

### 支持的分析类型

| 类型 | 说明 | 照片要求 |
|------|------|---------|
| **面相** | 分析五官、三庭、十二宫、气色 | 正面或侧面人脸照片 |
| **手相** | 分析掌线、指形、掌丘、色泽 | 手掌正面照片 |
| **骨相** | 分析头骨、体骨、九骨结构 | 能看出骨骼轮廓的照片 |
| **体相** | 分析体型、体态、气质 | 全身或半身照片 |

## 功能特性

### 支持的相术类型

| 类型 | 分析维度 | 规则来源 |
|------|---------|---------|
| **面相** | 五官、三庭、十二宫、气色、骨骼 | 《麻衣神相》《柳庄相法》《神相全编》 |
| **手相** | 掌线、指形、掌丘、色泽、纹路 | 《神相全编》《水镜神相》 |
| **骨相** | 头骨、体骨、九骨结构 | 《太清神鉴》《人伦大统赋》 |
| **体相** | 体型、体态、气质、步态 | 《冰鉴》《神相全编》 |

### 分析报告结构

每份分析报告包含以下部分：

1. **基本信息**：相术类型、分析时间
2. **特征识别**：从照片中识别出的关键特征
3. **性格特质**：主要性格倾向、优缺点
4. **能力倾向**：擅长领域、适合职业
5. **运势分析**：分时期或分维度的运势分析
6. **健康提示**：健康相关的观察与建议
7. **发展建议**：基于相术分析的实用性建议

### 技术原理

```
上传照片 → 识别图片类型 → 调用规则库 → 提取特征 → 综合分析 → 生成报告
```

1. **识别图片类型**：自动判断是面相/手相/骨相/体相
2. **调用规则库**：读取对应相术规则文件（基于经典相书）
3. **提取特征**：使用视觉分析工具提取关键特征
4. **综合分析**：结合规则知识进行多维度分析
5. **生成报告**：输出结构化分析报告，附带免责声明

## 注意事项

### ⚠️ 重要提醒

- **本分析仅供文化参考和娱乐**：相术属传统文化，现代科学尚未证实其预测能力
- **请理性看待**：人生道路主要取决于个人努力和选择
- **不做绝对化断言**：所有分析均为概率性参考，非绝对命运
- **健康提示仅供参考**：如有健康问题，请咨询专业医生

### 📋 使用限制

- 不预测具体事件（如"何时结婚"、"何时发财"）
- 不涉及政治敏感话题
- 不宣扬迷信色彩
- 每次分析报告末尾包含免责声明

### 📷 照片要求

- **清晰度**：照片应清晰可见，避免模糊、遮挡
- **光线**：避免光线过暗或强光直射
- **角度**：正面照片最佳，侧面照片也可分析
- **隐私**：请勿上传包含他人隐私的照片

## 常见问题

### Q：分析结果准确吗？
**A**：相术是传统文化的一部分，可作为参考，但不宜迷信。现代科学对相术的预测能力尚无定论。建议将分析结果作为娱乐和文化参考，理性看待。

### Q：可以上传哪些类型的照片？
**A**：支持人体部位照片，包括人脸、手掌、全身等。照片应清晰可见，避免模糊、遮挡、光线过暗等问题。

### Q：分析需要多长时间？
**A**：通常在几秒到十几秒内完成分析。

### Q：可以多次上传照片分析吗？
**A**：可以。每次分析都是独立的，可以多次上传不同照片进行分析。

### Q：分析结果会保存吗？
**A**：不会。每次分析都是实时的，不会保存您的照片或分析结果。

### Q：支持哪些相术类型？
**A**：目前支持面相、手相、骨相、体相四大类。未来可能扩展更多类型。
