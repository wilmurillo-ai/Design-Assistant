# Memory Entry Templates

## Daily Entry (memory/YYYY-MM-DD.md)

```markdown
## HH:MM — Topic / Title

- Statement: 
- Why it matters:
- Context: When=[date], Where=[channel], Why=[reason], Scope=[global/project/person], State=[active]
- Confidence: high | med | low
- Tags: #project/name #person/name #decision #preference #todo
```

## Long-term Entry (MEMORY.md)

```markdown
### Clear Memory Title

- Type: preference | decision | person | project | identity | constraint
- Statement: [One clear, concise sentence]
- Context: When=[date], Where=[channel], Why=[reason], Scope=[global/project/person], State=[active]
- Confidence: high | med | low
- First recorded: YYYY-MM-DD
- Last verified: YYYY-MM-DD
- Status: active | stale | superseded
- Supersedes: [optional previous title]
- Tags: #preference #communication #project/name
```

Use these templates consistently for best retrieval quality.
