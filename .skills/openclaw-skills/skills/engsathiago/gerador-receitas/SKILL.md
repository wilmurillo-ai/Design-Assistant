# Gerador de Receitas

**Por EVE** — Skill para agentes OpenClaw

**Métricas SaaS para planejar monetização de agentes IA.**

## O Problema

Agentes IA podem gerar valor, mas como medir sucesso financeiro? Esta skill oferece métricas claras para planejamento de receitas.

## Métricas Principais

### 📊 ARR (Annual Recurring Revenue)
- Receita anual recorrente
- Base para projeções
- Mede saúde do negócio

### 📈 NDR (Net Dollar Retention)
- Quanto receita é retida
- NDR > 100% = crescimento orgânico
- Indica satisfação do cliente

### 💰 LTV:CAC (Lifetime Value to Customer Acquisition Cost)
- Proporção saudável: 3:1 ou mais
- Tempo de payback
- Eficiência de aquisição

### ✨ Magic Number
- Eficiência de crescimento
- Fórmula: (ARR_atual - ARR_anterior) / CAC
- Magic Number > 0.75 = saudável

### 📉 Cohort Analysis
- Retenção por mês
- Churn por cohort
- Identifica problemas

### 🎯 GTM Efficiency
- Eficiência de Go-to-Market
- Custo por canal
- ROI de vendas

## Uso

```
Usuário: Me ajude a planejar a monetização do meu agente

Agente usa gerador-receitas para:
1. Calcular ARR projetado
2. Estimar NDR baseado em churn
3. Calcular LTV:CAC
4. Avaliar Magic Number
5. Identificar canais eficientes
6. Sugerir melhorias
```

## Exemplo de Relatório

```markdown
## Análise de Receitas - Agente X

### ARR Projetado
- Mês 1: R$ 0
- Mês 6: R$ 12.000
- Mês 12: R$ 48.000

### NDR Estimado
- Churn: 5%/mês
- NDR: 54% (precisa melhorar)
- Meta: NDR > 100%

### LTV:CAC
- LTV: R$ 600 (12 meses × R$ 50)
- CAC: R$ 200
- LTV:CAC: 3:1 ✅

### Magic Number
- ARR crescimento: R$ 48.000/ano
- CAC total: R$ 2.400
- Magic Number: 20 (excelente!)

### Canais Eficientes
1. Indicações (CAC: R$ 50)
2. Conteúdo orgânico (CAC: R$ 80)
3. Google Ads (CAC: R$ 300) ⚠️
```
