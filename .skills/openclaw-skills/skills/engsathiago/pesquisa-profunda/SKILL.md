# Pesquisa Profunda

**Por EVE** — Skill para agentes OpenClaw

**Pipeline de 8 fases para pesquisas profundas com citações verificadas.**

## O Problema

Pesquisas superficiais geram respostas incorretas. Esta skill garante que cada afirmação seja verificada e citada.

## Pipeline de 8 Fases

### 1. Escopo
- Define objetivo da pesquisa
- Identifica questões-chave
- Estabelece limites

### 2. Planejamento
- Cria estratégia de busca
- Seleciona fontes confiáveis
- Define metodologia

### 3. Recuperação
- Busca informações de múltiplas fontes
- Coleta dados brutos
- Organiza achados

### 4. Triangulação
- Verifica informações em múltiplas fontes
- Identifica contradições
- Valida dados

### 5. Síntese
- Combina informações
- Cria narrativa coesa
- Organiza por temas

### 6. Crítica
- Avalia qualidade das fontes
- Identifica vieses
- Aponta limitações

### 7. Refinamento
- Melhora clareza
- Remove redundâncias
- Ajusta tom

### 8. Empacotamento
- Formata relatório final
- Adiciona citações
- Cria sumário executivo

## Uso

```
Usuário: Pesquise sobre os impactos da IA na educação brasileira

Agente usa pesquisa-profunda para:
1. Definir escopo (ensino fundamental, médio, superior)
2. Planejar busca (fontes brasileiras, internacionais)
3. Recuperar dados (artigos, relatórios, estatísticas)
4. Triangular (verificar em múltiplas fontes)
5. Sintetizar (criar análise)
6. Criticar (avaliar qualidade)
7. Refinar (melhorar texto)
8. Empacotar (relatório final com citações)
```
