# Aether Tarot — AI Intuitive Tarot Reading Skill
Aether Tarot is a safe, supportive, AI-powered tarot reading skill built for the ClawHub ecosystem. It delivers clear, gentle, and context-aware guidance through natural conversation using a full 78‑card tarot deck.

## Core Purpose
To provide accessible, reflective tarot insights for daily decisions, emotional clarity, and personal reflection.

## Key Features
- Full 78‑card tarot deck (Major Arcana + four suites)
- 3‑card intuitive spread: Past / Present / Future
- Random card selection with ethical, non‑doom interpretation
- Calm, supportive, private conversational tone
- Stable, lightweight runtime performance
- Privacy‑focused user experience

## Tarot Deck Included
Major Arcana: The Fool, The Magician, The High Priestess, The Empress, The Emperor, The Hierophant, The Lovers, The Chariot, Strength, The Hermit, Wheel of Fortune, Justice, The Hanged Man, Death, Temperance, The Devil, The Tower, The Star, The Moon, The Sun, Judgement, The World
Wands: Ace–Ten, Page, Knight, Queen, King
Cups: Ace–Ten, Page, Knight, Queen, King
Swords: Ace–Ten, Page, Knight, Queen, King
Pentacles: Ace–Ten, Page, Knight, Queen, King
