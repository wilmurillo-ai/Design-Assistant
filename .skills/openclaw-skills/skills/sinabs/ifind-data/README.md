# iFinD 金融数据查询 Skill

同花顺 iFinD 量化数据接口的 [OpenClaw](https://openclaw.com) Skill 封装。通过自然语言即可查询 A 股、基金、债券、期货、指数的实时行情、历史数据、财务指标、宏观经济数据等，覆盖 18 个 API 接口。

## 功能一览

| 类别 | 功能 | 说明 |
|------|------|------|
| **行情数据** | 实时行情 | 最新价、涨跌幅、成交量、买卖盘口 |
| | 历史行情 | 日K/周K/月K，支持前复权/后复权 |
| | 高频序列 | 分钟级数据 + MACD/KDJ/RSI 等技术指标 |
| | 日内快照 | 逐笔/分钟级盘口快照 |
| **基本面** | 基础数据 | ROE、EPS、PE、PB 等财务指标 |
| | 日期序列 | 按时间序列获取指标数据 |
| **宏观经济** | 经济数据库 (EDB) | GDP、CPI、PMI 等宏观指标 |
| **基金** | 基金估值(分钟) | 实时分钟级基金估值 |
| | 基金估值(日) | 日终基金估值 |
| **专题** | 专题报表 | REITs、龙虎榜等专题数据 |
| | 智能选股 | 按条件筛选股票 |
| **工具** | 组合管理 | 模拟组合新建/交易/监控/绩效分析 |
| | 日期查询 | 交易日历查询 |
| | 日期偏移 | 基准日前推后推 |
| | 代码转换 | 证券代码转同花顺代码 |
| | 公告查询 | 上市公司公告检索下载 |
| | 数据量查询 | 查询账号已用数据量 |
| | 错误信息查询 | 查询错误码含义 |

## 获取 Refresh Token

使用本 Skill 需要 iFinD 量化数据终端的 `refresh_token`。

### 还没有 iFinD 账号

1. 访问 [iFinD 量化数据终端](https://ft.10jqka.com.cn) 申请开通
2. 开通后按下方步骤获取 token

### 已有 iFinD 账号

**方式一：客户端获取**
1. 打开 iFinD 超级命令客户端
2. 菜单 → 工具 → refresh_token 查询
3. 复制显示的 token

**方式二：网页获取**
1. 登录 [iFinD 量化平台](https://quantapi.51ifind.com)
2. 在「账号信息」页面查看 refresh_token

### 配置 Token

在聊天中直接把 token 发给 AI，它会自动帮你配置好，无需手动操作。

## 使用示例

在聊天中用自然语言提问即可，以下是一些常见场景：

1. 帮我看一下同花顺(300033)和中信证券(600030)的最新行情
2. 贵州茅台现在什么价？涨跌幅和成交量也给我看看
3. 查一下 300033 最近 3 个月的日K线数据，要前复权
4. 给我看看沪深300指数(000300.SH)过去一年的周K线，包含成交量
5. 比较一下同花顺和东方财富(300059)的 ROE 和 EPS
6. 查一下宁德时代(300750)最近三年每年的净利润和营收
7. 帮我看一下 300033 今天的分钟级 MACD 和 KDJ 指标
8. 查一下中国最近 5 年的 GDP 和 CPI 数据
9. 查一下华夏大盘精选(000001.OF)的最新实时估值
10. 帮我筛选一下个股热度排名
11. 2025 年上交所有哪些交易日？
12. 查一下同花顺和浦发银行最近的年报公告
13. 帮我查一下 300033 的同花顺标准代码
