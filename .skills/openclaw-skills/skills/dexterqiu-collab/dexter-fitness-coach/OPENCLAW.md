# 🏋️ AI健身教练 - OpenClaw安装指南

## 在飞书OpenClaw中安装

### 方式1：直接搜索（推荐）

在飞书的OpenClaw对话框中输入：
```
/install dexter-fitness-coach
```

### 方式2：搜索关键词

```
/search fitness coach
```
或
```
/search 健身
```

找到 **AI健身教练 (dexter-fitness-coach)** 后点击安装

---

## 安装后使用

安装完成后，优先直接用自然语言对话：

```
你好，我是新来的，想开始健身
```

AI 教练会引导你完成：
1. 身体数据收集（身高、体重）
2. 目标设定（减脂/增肌）
3. 训练频率选择
4. 个性化计划生成

---

## 功能清单

| 功能 | 命令示例 |
|-----|---------|
| 查看今日计划 | 今天的计划是什么 |
| 记录训练 | 今天练了胸，卧推4组12次 |
