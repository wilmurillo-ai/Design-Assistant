# AI健身教练 | OpenClaw技能

> 24/7陪伴的个性化健身指导智能体

## 快速开始

在飞书OpenClaw中安装：
```
/install dexter-fitness-coach
```

或搜索：`健身` / `fitness`

## 功能

- 🧠 **智能记忆** - 记住你的训练历史和偏好
- 📋 **计划生成** - 根据目标生成个性化训练计划
- 📊 **进度追踪** - 自动记录训练，统计连续打卡
- 📱 **飞书集成** - 自动同步到飞书多维表格
- ❤️ **情绪陪伴** - 不只是计划，更是懂你的教练

## 使用示例

```
# 开始使用
你好，我是新来的，想开始健身

# 查看计划
今天的训练计划是什么？

# 记录训练
今天练了胸，卧推4组12次，引体向上4组10次

# 查看进度
我的训练进度怎么样？
```

## 配置

当前最稳的集成方式是把它作为 markdown skill 使用。Python 文件保留为参考实现；是否会被宿主直接执行，取决于你的 OpenClaw 运行环境。

## 技术栈

- Python 3.8+
- 智谱GLM API
- OpenClaw Agent Framework
- 飞书多维表格 API
