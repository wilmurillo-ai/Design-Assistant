# Example prompts

## Chinese prompts

- `帮我查一下今天 OpenAI API 有什么新发布`
- `搜索一下最近一周关于 OpenClaw 的更新`
- `帮我看看 2026 年 3 月以来 AI 搜索产品的新闻`
- `查一下美国最新对华关税政策`

## English prompts

- `Look up the latest OpenClaw release notes`
- `Search recent news about browser agents`
- `Find current coverage of AI coding benchmark results`

## Expected response pattern

1. State that the answer is based on live web search when timeliness matters.
2. Summarize the key findings in one short paragraph.
3. List the most relevant sources or domains.
4. Flag uncertainty if sources disagree.
5. Never invent results when the API is unavailable.
