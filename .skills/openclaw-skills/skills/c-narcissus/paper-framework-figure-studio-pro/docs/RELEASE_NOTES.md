# Release Notes

This package is prepared for OpenClaw / ClawHub publishing.

## Package intent
A publish-ready skill focused on paper framework diagrams and method overview figures.

## Notable publishing notes
- `SKILL.md` includes YAML frontmatter with `name`, `description`, and `version`.
- License should be set to **MIT-0** on the ClawHub publish page.
- The workflow explicitly ends by asking whether the user also wants figure caption / annotation text.
