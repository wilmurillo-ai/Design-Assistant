# Paper Framework Figure Studio Pro

Multi-round co-designer for reviewer-friendly **paper framework diagrams** and method overview figures.

This skill turns a paper deep-reading report, method description, model overview, or figure idea into a publication-ready framework diagram through a structured human-in-the-loop workflow.

## What it is for
Use this skill when you need:
- a main paper framework figure,
- a method overview diagram,
- a why-ours comparison figure,
- a mechanism explanation figure,
- or an integrated one-page paper figure.

## How it works
1. You provide a deep-reading report, method summary, or model description.
2. The skill compresses it into a figure-oriented visual summary.
3. It proposes a small set of figure families.
4. It generates multiple candidate diagrams each round.
5. You choose a direction.
6. The skill refines the figure family, substyle, layout, density, and local elements.
7. It finalizes 1–3 strong framework diagrams.

After the final image, the skill explicitly asks whether you also want:
- short in-figure labels,
- a polished figure caption,
- panel-by-panel explanation text,
- or slide-friendly annotation text.

## Best use cases
- top-tier CS conference / journal submissions
- framework figures for method papers
- reviewer-friendly explanation diagrams
- iterative figure design with approval at each step

## Typical first prompt
"Design a main framework figure for this method. Here is my deep-reading report / method summary: ..."

## Notes
This skill is optimized for **framework diagrams**, not simple numeric charts.
