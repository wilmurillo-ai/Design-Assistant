# Summary Template

Use this structure after fetching report tasks with `scripts/fetch_report_tasks.js`.

## Daily digest

1. Overall themes
2. Completed work
3. Blockers and risks
4. Planned next steps
5. People or teams that need follow-up

## Weekly recap

1. Cross-team trends
2. Major wins
3. Open issues
4. Work that rolled over into next week
5. Suggested actions for managers
