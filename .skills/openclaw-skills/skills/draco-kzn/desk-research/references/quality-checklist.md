# Desk Research Quality Checklist (Submission Gate)

Before final delivery, verify all boxes:

## 1) Problem-fit
- [ ] Research answers the 2 core questions clearly
- [ ] Scope is explicit (time, geography, sample boundary)
- [ ] No major section is off-topic

## 2) Evidence quality
- [ ] Each major claim has source + date
- [ ] Each key framework / stage model / analytical lens has source, or is clearly labeled as author synthesis
- [ ] No critical conclusion depends on one weak source
- [ ] Conflicting evidence is acknowledged and reconciled

## 3) Case depth
- [ ] Each promising case uses the full 9-part framework
- [ ] Each case has >=3 website evidence snippets
- [ ] Each case has metric anchors (MRR/revenue/etc.)
- [ ] Each case includes risk + falsification condition
- [ ] Each case has final summary paragraph (做什么/用户是谁/商业模式/为什么付费)

## 4) Decision usefulness
- [ ] Every major section ends with “so-what” implication
- [ ] Cross-category comparison leads to explicit prioritization
- [ ] Final takeaways can be pasted into an article conclusion

## 5) Writing quality
- [ ] Conclusion-first, evidence-following
- [ ] Facts vs interpretations clearly separated
- [ ] Language is concise and non-generic
- [ ] Uncertainty and next verification steps are explicit
