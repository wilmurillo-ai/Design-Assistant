# 品质报告模板选择（请选择模板）

当用户请求**周报**或**月报**时，技能必须先展示以下模板选择，待用户回复 1、2、3 或 4 后再继续。

---

## 请选择模板

**1. 简单模板**  
一页摘要：核心 KPI（良率、PPM、客诉）+ 1～2 条主要问题 + 下阶段重点。适合产线/班组内、邮件汇报、快速阅读。

**2. 完整过程模板**  
从封面到结尾的完整周报/月报，含：封面（报告名称、编号、日期、部门）→ 质量状况综述（市场反馈、成品管理、外检、过程、计量、体系）→ 总体不良率与客户工程/客户市场 → 客户投诉统计（可三地/多工厂）→ 质量信息跟进表 → 成品抽样/批量不良 → 外协抽检及分布图 → 装配/焊涂各线下线统计与明细 → 趋势图 → 外检进货情况 → 外协主要问题及饼图 → 计量与体系管理 → 下周/下月工作计划 → 谢谢浏览。适合质量部正式汇报、管理层、PPT 全文。

**3. 三地/多工厂汇总模板**  
多工厂（如工厂A、工厂B、工厂C）数据汇总：三地投诉总次数与年度闭环率、各工厂分项投诉、客户工程/客户市场退回责任划分（我司责任/用户责任）、各地产线或成品数据。适合集团或三地工厂品质周报。

**4. 首检送检制程模板**  
按**目录**五模块组织：① 首检合格率及不良分析（首检每日表、**趋势图**、周对比、调机员合格率表与**柱状图**、主要质量问题）② 送检合格率及不良分析（送检每日表、**趋势图**、周对比、操作员送检合格率表与**柱状图**）③ 首检及送检不良项目统计（首检/送检不良类别**饼图**、主要不良款号与描述）④ 制程质量异常状况（异常明细表、主要因素汇总、制程异常数量**周对比图**）⑤ 来料质量状况（来源合格率表、合格率**柱状图**、**升降趋势图**）⑥ 出货合格率。**生成时须含上述图表**（Mermaid 或图表数据规格），不得仅输出表格。适合制程/产线品质周报（机加工、CNC、调机员与操作员维度）。

---

- 用户回复 **1** 或「简单」→ 使用简单版模板（SKILL 中的 Weekly/Monthly Simple）。
- 用户回复 **2** 或「完整」「完整过程」→ 使用完整过程模板（见 `examples/sample-full-process-weekly.md`）。
- 用户回复 **3** 或「三地」「多工厂」→ 使用三地/多工厂汇总模板（见 `examples/sample-san-di-weekly.md`）。
- 用户回复 **4** 或「首检送检」「制程来料」→ 使用首检送检制程模板（见 `examples/sample-shoujian-songjian-weekly.md`）。
