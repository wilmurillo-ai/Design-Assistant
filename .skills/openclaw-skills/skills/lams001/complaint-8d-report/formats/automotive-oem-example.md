# Format: Automotive OEM (example)

Example of a customer-specific format. Section titles and order match a typical OEM CAR template. Use these headings when the user selects "automotive-oem" or "Automotive OEM".

## Section order (use these headings in the report)

1. **Cover sheet / Basic information**
2. **D1 – Team formation** (not "Form the team")
3. **D2 – Problem description (5W2H)**
4. **D3 – Interim containment actions**
5. **D4 – Root cause analysis**
6. **D5 – Permanent corrective actions – selection and verification**
7. **D6 – Implementation and validation**
8. **D7 – Recurrence prevention**
9. **D8 – Team recognition and closure**

## Optional: extra fields this customer requires

- In Basic information: add **RMA number**, **Supplier response due date**, **Cost of quality (if applicable)**.
- After D8: **Customer approval signature / date** (placeholder).

## Table columns (if customer template differs from standard)

- **Team**: Role | Name | Department | Responsibility | Contact
- **Containment**: # | Action | Owner | Due date | Verification
- **Permanent actions**: ID | Action | Owner | Due date | Verification result
