## D4 Root cause analysis — example

- **5-Why summary**:  
  1. Why scratched? → Parts contacted metal container edges after blanking.  
  2. Why contact? → Container had no liner; parts rubbed the frame.  
  3. Why no liner? → This part number had no liner requirement; old process did not cover it.  
  4. Why not covered? → Design/process review did not include appearance and handling method.  
- **Direct cause**: Parts rubbing metal container edges.  
- **Root cause**: No scratch protection (liner/soft contact) for post-blanking handling containers, and process did not define it.  
- **Verification**: Same parts handled in lined containers—appearance OK; reproduction test confirmed no scratches.

**Root cause statement**: Due to lack of scratch protection on handling containers and no process requirement, parts contacted container edges and sustained surface scratches.
