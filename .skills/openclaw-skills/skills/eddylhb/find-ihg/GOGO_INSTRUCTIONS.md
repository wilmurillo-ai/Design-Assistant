# 🏨 Gogo调用IHG江浙沪酒店查询技能指南 (增强版 v2.1.0)

## 📋 技能基本信息
- **技能ID**: `ihg-monitor`
- **技能名称**: IHG江浙沪积分房查询
- **技能版本**: v2.1.0 (OpenClaw 4.5兼容版)
- **功能描述**: 查询江浙沪地区IHG酒店积分房情况和优惠活动，支持品牌筛选、行政酒廊过滤、智能推荐
- **覆盖范围**: 江浙沪3省市，23家酒店，8个品牌类型
- **最后更新**: 2026-04-08

---

## 🎯 核心功能亮点

### 1. 🗺️ **地理范围扩展**
- **江浙沪全覆盖**: 上海、江苏、浙江三省市
- **三级距离体系**: 
  - **近程** (1-2小时): 上海本地及昆山、太仓等
  - **中程** (2-3小时): 苏州、无锡、宁波、嘉兴、绍兴、湖州
  - **远程** (3-4小时): 杭州、南京、常州、镇江、扬州、南通
- **23家精选酒店**: 涵盖IHG全品牌体系

### 2. 🏷️ **品牌完整支持**
- **顶级奢华**: 洲际(InterContinental)、丽晶(Regent)
- **中式奢华**: 华邑(HUALUXE)
- **精品设计**: 金普顿(Kimpton)、英迪格(Indigo)
- **高端商务**: 皇冠假日(Crowne Plaza)
- **生活方式**: voco、逸衡(even)
- **中端商务**: 假日(Holiday Inn)
- **❌ 排除**: 智选假日(Holiday Inn Express)

### 3. 🍽️ **行政酒廊筛选**
- **明确标识**: 13家有行政酒廊，10家无行政酒廊
- **智能筛选**: 支持"必须有/必须无/不限"三种模式
- **商务友好**: 快速筛选适合商务出行的酒店

### 4. 🧠 **智能推荐系统**
- **综合评分算法**: 品牌等级+行政酒廊+距离+价值比
- **精选推荐**: 每次返回3-4家最优酒店
- **品牌多样性**: 自动保证不同品牌选择
- **价值分析**: 积分vs现金价值比计算与建议

| "皇冠"、"皇冠假日" | 高端商务品牌 | `brand_type=皇冠` |
| "voco" | 生活方式品牌 | `brand_type=voco` |
| "even"、"逸衡" | 健康生活品牌 | `brand_type=even` |

| "金普顿" | 精品奢华品牌 | `brand_type=金普顿` |

#### 特征关键词映射  
| 用户说 | Gogo理解 | 映射参数 |
|--------|----------|----------|
