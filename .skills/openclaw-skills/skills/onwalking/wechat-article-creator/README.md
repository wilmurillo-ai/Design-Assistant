# wechat-article-creator

根据主题自动生成微信公众号文章草稿

## 功能

1. 根据用户提供的主题搜索相关内容
2. AI 总结分析搜索到的信息
3. 排版美化成适合公众号阅读的格式
4. 自动保存到微信公众号草稿箱

## 前置要求

- 微信公众号开发者账号
- 已认证的公众号（或个人订阅号）
- 配置了 IP 白名单的服务器

## 配置

在运行目录创建 `.env` 文件：

```
WECHAT_APPID=your_appid
WECHAT_SECRET=your_secret
```

或使用环境变量导出。

## 使用方法

```bash
# 创建文章
wechat-article-creator "人工智能发展趋势"

# 指定字数
wechat-article-creator "新能源汽车市场分析" --words 2000

# 指定风格
wechat-article-creator "健康饮食指南" --style "轻松科普"
```

## 流程

1. **搜索阶段**：用主题关键词搜索最新资讯
2. **分析阶段**：提取核心观点、数据、案例
3. **写作阶段**：生成标题、导语、正文、结语
4. **排版阶段**：添加公众号样式（重点标红、引用框、分隔线等）
5. **保存阶段**：调用微信 API 创建草稿

## 输出

- 文章预览（Markdown 格式）
- 草稿保存成功提示
- 公众号后台草稿链接
