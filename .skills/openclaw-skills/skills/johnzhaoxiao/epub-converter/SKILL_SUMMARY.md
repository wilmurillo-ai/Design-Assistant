## 🚀 核心功能

### 1. 自动依赖管理
- 首次运行自动创建虚拟环境
- 自动安装 `ebooklib` 和 `opencc-python-reimplemented`
- 智能路径管理，无需手动配置

### 2. 双向转换
- **繁→简** (t2s): Traditional Chinese → Simplified Chinese
- **简→繁** (s2t): Simplified Chinese → Traditional Chinese

### 3. 完整转换
- ✅ 所有章节内容（HTML/XHTML）
- ✅ 书籍元数据（标题、作者等）
- ✅ 目录结构（TOC）
- ✅ 自动修复损坏的目录条目

### 4. 智能输出
- 自动生成输出文件名（添加 `_简体` 或 `_繁體` 后缀）
- 支持自定义输出路径
- 保持原始文件格式和结构

## 🎓 使用方式

### 方式1：在OpenClaw对话中
```
用户：[上传EPUB文件] 帮我把这本繁体书转成简体
OpenClaw：[自动调用技能] → 返回简体版文件
```

### 方式2：命令行直接使用
```bash
# 基础转换（繁→简）
python3 scripts/convert_epub.py input.epub

# 指定输出
python3 scripts/convert_epub.py input.epub -o output.epub

# 反向转换（简→繁）
python3 scripts/convert_epub.py input.epub --direction s2t
```

## 🔧 技术架构

### 依赖库
- **ebooklib** (0.20) - EPUB文件解析和写入
- **opencc-python-reimplemented** (0.1.7) - 中文繁简转换
- **lxml** (6.0.2) - XML处理（ebooklib依赖）

### 虚拟环境
- 位置：`~/.openclaw/epub_venv`
- Python版本：3.14+
- 自动创建和管理

### 转换流程
```
输入EPUB → 检查依赖 → 读取结构 → 转换文本 → 修复TOC → 保存输出
```
