# WSL Chrome CDP - 快速入门

**让 WSL2 无缝访问 Windows Chrome 浏览器**

**全自动模式：无需手动操作！**

---

## 🚀 30 秒开始

### **安装技能**

```bash
# 从 ClawHub 安装（待发布）
openclaw skills install wsl-chrome-cdp

# 或手动安装
git clone https://github.com/openclaw/wsl-chrome-cdp.git ~/.openclaw/workspace/skills/wsl-chrome-cdp
```

### **使用**

**在 OpenClaw 对话中直接说：**

```
打开百度
```

**就这么简单！**

---

## 🔧 手动启用（可选）

# 一键启用浏览器
