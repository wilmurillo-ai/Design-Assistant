# A股组合优化参考手册

## 1. 均值方差模型 (Markowitz Mean-Variance Optimization)

### 基本框架
Harry Markowitz (1952) 提出的现代投资组合理论核心：

**输入**：
- mu: N×1 预期收益率向量
- Sigma: N×N 收益率协方差矩阵
- Rf: 无风险利率

**目标函数**（最大夏普比）：
```
max  (mu^T w - Rf) / sqrt(w^T Sigma w)
s.t. sum(w) = 1
     w >= 0          (做多约束)
     w_i <= w_max    (个股上限)
```

等价转化为二次规划：
```
min  w^T Sigma w
s.t. mu^T w >= target_return
     sum(w) = 1
     0 <= w_i <= w_max
```

### 有效前沿 (Efficient Frontier)
- 在收益-波动(mu-sigma)平面上，所有最优组合构成的曲线
- 前沿上方无可行组合，前沿下方为非最优组合
- 切线组合：从Rf出发与有效前沿相切的点 = 最大夏普比组合

## 2. 最小方差组合 (Minimum Variance Portfolio)

**优势**：不需要估计预期收益率（mu估计误差大，协方差相对稳定）

```
min  w^T Sigma w
s.t. sum(w) = 1
     w >= 0
```

无约束解析解：w* = Sigma^{-1} 1 / (1^T Sigma^{-1} 1)

**实践价值**：
- 学术研究表明最小方差组合长期表现往往优于MVO（因为避免了mu估计误差）
- 适合风险厌恶型投资者
- A股波动大，最小方差策略有实际意义

## 3. 最大夏普比组合 (Maximum Sharpe Ratio Portfolio)

```
max  (mu^T w - Rf) / sqrt(w^T Sigma w)
s.t. sum(w) = 1, w >= 0
```

转化技巧（Cornuejols & Tutuncu）：令 y = w / (mu^T w - Rf)
```
min  y^T Sigma y
s.t. (mu - Rf)^T y = 1
     y >= 0
```
然后 w = y / sum(y)

**注意**：高度依赖mu估计，对输入极其敏感

## 4. 风险平价 (Risk Parity / Equal Risk Contribution)

### 原理
使每个资产对组合总风险的贡献相等：

组合风险: sigma_p = sqrt(w^T Sigma w)

资产i的风险贡献:
```
RC_i = w_i * (Sigma w)_i / sigma_p
```

目标: RC_i = sigma_p / N  (所有资产风险贡献相等)

### 数值求解
```
min  sum_{i=1}^{N} (RC_i - sigma_p/N)^2
s.t. sum(w) = 1, w >= 0
```

或等价：
```
min  sum_{i<j} (w_i * (Sigma w)_i - w_j * (Sigma w)_j)^2
```

### 特点
- 不依赖预期收益率估计
- 高波动资产分配低权重，低波动资产分配高权重
- 桥水基金全天候策略的核心思想
- 结果通常介于等权和最小方差之间

## 5. Black-Litterman 模型

### 框架
Fischer Black & Robert Litterman (1992)，结合市场均衡与投资者主观观点。

**Step 1: 均衡预期收益（逆优化）**
```
pi = delta * Sigma * w_mkt
```
- delta: 风险厌恶系数（通常取市场夏普比^2 / sigma_mkt）
- w_mkt: 市场权重（市值加权）

**Step 2: 投资者观点**
- P: K×N 观点矩阵（K个观点，N个资产）
- Q: K×1 观点收益向量
- Omega: K×K 观点不确定性矩阵

例：观点"贵州茅台未来1年收益率15%"
```
P = [0, 0, ..., 1, ..., 0]  (茅台位置为1)
Q = [0.15]
Omega = diag(confidence_adjusted_variance)
```

**Step 3: 融合**
```
mu_BL = [(tau*Sigma)^{-1} + P^T Omega^{-1} P]^{-1} * [(tau*Sigma)^{-1} pi + P^T Omega^{-1} Q]
Sigma_BL = [(tau*Sigma)^{-1} + P^T Omega^{-1} P]^{-1} + Sigma
```
- tau: 标量，通常取 1/T（T为样本量）或 0.025

**Step 4: 基于 mu_BL 做标准MVO**

### 优势
- 从市场均衡出发，避免极端权重
- 允许融合主观判断但有量化约束
- 无观点时退化为市值加权

## 6. 协方差矩阵估计方法

### 样本协方差 (Sample Covariance)
```
Sigma_sample = (1/(T-1)) * X^T X
```
- X: T×N 去均值收益率矩阵
- 问题：N大T小时，矩阵可能奇异或条件数大

### Ledoit-Wolf 收缩估计
```
Sigma_shrink = alpha * F + (1-alpha) * Sigma_sample
```
- F: 结构化目标矩阵（如对角矩阵、等相关矩阵）
- alpha: 收缩强度（由数据驱动的公式确定）
- 优势：减少估计误差，保证正定，提升组合外样本表现

### 因子模型协方差
```
Sigma_factor = B * Sigma_f * B^T + D
```
- B: N×K 因子载荷矩阵
- Sigma_f: K×K 因子协方差
- D: N×N 特质风险对角矩阵
- 大幅降低待估参数量

## 7. 约束条件详解

### A股常用约束
| 约束 | 表达式 | 说明 |
|------|--------|------|
| 做多约束 | w_i >= 0 | A股融券困难，默认不允许做空 |
| 权重和为1 | sum(w) = 1 | 满仓约束 |
| 个股上限 | w_i <= 0.40 | 防止单一股票过度集中 |
| 个股下限 | w_i >= 0.02 | 有时设置最小持仓（可选） |
| 行业上限 | sum(w_j, j in sector) <= 0.30 | 行业集中度限制 |
| 换手约束 | sum(|w_new - w_old|) <= turnover_limit | 控制交易成本 |
| 持仓数量 | count(w_i > 0) <= K | 基数约束（NP-hard，需整数规划或启发式） |

### 交易成本考虑
- A股印花税: 卖出 0.05%（2024年后）
- 佣金: 双向约 0.025%
- 冲击成本: 小盘股流动性差，需考虑滑点
- 总成本约 0.1%-0.3%/单边

## 8. A股实操注意事项

### 数据质量
- **停牌处理**：停牌期间无真实价格变动，需剔除或用前值填充
- **涨跌停**：A股10%/20%涨跌停限制，影响收益率分布（截断）
- **新股效应**：上市初期波动大，建议排除上市不足60个交易日的股票
- **复权处理**：必须使用前复权/后复权价格计算收益率

### 估计窗口选择
- 太短（<6个月）：协方差矩阵不稳定
- 太长（>3年）：市场结构可能已变化
- 推荐：1年日频数据（约250个交易日）
- 可选：加权方案（指数衰减加权，近期数据权重大）

### A股特殊性
- **散户占比高**：波动率和相关性结构可能不同于成熟市场
- **政策驱动**：板块轮动快，历史相关性可能不稳定
- **T+1制度**：影响实际再平衡的执行
- **板块效应强**：同行业股票相关性很高，注意分散行业
- **核心-卫星策略**：建议核心仓位用最小方差/风险平价，卫星仓位可用MVO

### 回测注意
- 避免用未来数据（前视偏差）
- 考虑交易成本后的净收益
- 样本外检验比样本内优化更重要
- 定期再平衡频率：月度或季度
