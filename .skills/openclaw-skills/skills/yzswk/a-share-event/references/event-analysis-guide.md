### 常见事件影响分析模板

#### 增发/配股
- **关键要素**：增发价格 vs 当前股价、增发规模/总市值、用途（项目投资/补充流动资金/收购）、锁定期
- **EPS 摊薄** = 增发股数 / (原股本 + 增发股数) × 100%
- **正面信号**：增发价接近市价、用于优质项目、大股东参与
- **负面信号**：大幅折价增发、用途模糊、仅补充流动资金

#### 回购
- **关键要素**：回购金额区间、回购价格上限、回购目的（注销/股权激励/市值管理）、回购期限
- **影响评估**：回购金额/市值比 > 1% 才有实质意义
- **正面信号**：大额回购+注销（减少股本=EPS提升）
- **一般信号**：回购用于股权激励（不减少股本）

#### 大股东增减持
- **增持**：金额、比例、是否有增持计划后续
- **减持**：减持原因（个人需要/基金退出/质押平仓）、减持比例
- **解禁**：解禁规模/流通盘、解禁股东类型（IPO前股东/定增股东）
- **关键判断**：减持比例 > 5% 为重大减持；解禁量 > 流通盘 20% 需关注

#### 股权激励
- **关键要素**：激励方式（股票期权/限制性股票/股票增值权）、行权价/授予价、考核条件、覆盖人员范围
- **正面解读**：考核条件有挑战性（暗示管理层对增长有信心）、覆盖面广（激励面大）
- **一般解读**：考核条件过低（"白送"）、仅覆盖少数高管

#### 并购重组
- **关键要素**：标的估值（PE/PB）、支付方式（现金/股份/混合）、业绩承诺、商誉
- **影响评估**：
  - 对 EPS 的影响 = 标的利润 / 收购后总股本 - 摊薄
  - 商誉风险 = 收购价 - 标的净资产
- **正面信号**：现金收购（不摊薄）、标的PE < 收购方PE（增厚）、业绩承诺有保障
- **负面信号**：高溢价+股份支付+模糊的业绩承诺

#### 政策事件
- **分析框架**：
  1. 政策力度：鼓励/引导 < 要求 < 强制
  2. 影响范围：个别公司 < 子行业 < 整个行业 < 跨行业
  3. 执行节奏：立即执行 vs 过渡期
  4. 补贴/处罚：有无具体金额
- **常见政策类型**：产业扶持、集采降价、环保限产、反垄断、新能源补贴退坡

### 事件影响持续性判断
| 持续性 | 特征 | 示例 |
|--------|------|------|
| 一次性 | 不改变公司基本面 | 一次性资产处置、短期政策补贴 |
| 短期(1-3月) | 影响当季度业绩 | 产品涨价/降价、季节性订单 |
| 中期(3-12月) | 影响当年业绩趋势 | 产能投放、新产品上市、行业政策 |
| 长期(>1年) | 改变公司竞争格局 | 重大并购、技术突破、行业重塑政策 |
