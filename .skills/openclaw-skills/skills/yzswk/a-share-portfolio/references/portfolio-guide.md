### 组合分析框架

#### 集中度参考标准
| 指标 | 保守型 | 均衡型 | 激进型 |
|------|--------|--------|--------|
| 持仓数量 | 15-30只 | 8-15只 | 3-8只 |
| 前5大占比 | <50% | 50-80% | >80% |
| 单只最大 | <10% | 10-20% | 20-40% |
| 行业最大 | <25% | 25-40% | >40% |

#### 风格因子参考
| 因子 | 价值型组合 | 成长型组合 | 均衡型组合 |
|------|----------|----------|----------|
| PE(TTM) | <15x | >25x | 15-25x |

| ROE | >15% | >15% | >12% |
| 收入增速 | <15% | >25% | 15-25% |
| 股息率 | >3% | <1.5% | 1.5-3% |

#### 简化风险指标
```
# 集中度指标
HHI = Σ(wi²)  其中 wi = 第 i 只股票的权重
HHI < 0.1 = 高度分散
HHI 0.1-0.2 = 适度集中
HHI > 0.2 = 高度集中

# 组合加权估值
组合PE = Σ(wi × PEi)
组合PB = Σ(wi × PBi)
组合ROE = Σ(wi × ROEi)
组合股息率 = Σ(wi × 股息率i)
```

#### 再平衡触发条件
| 触发条件 | 说明 |
|---------|------|
| 任一持仓偏离目标 > 5% | 个股再平衡 |
| 任一行业偏离目标 > 10% | 行业再平衡 |
| 组合PE偏离基准 > 30% | 风格漂移 |
| 季度末 | 定期再平衡 |
| 重大事件后 | 事件驱动再平衡 |

#### 常见组合问题
1. **过度集中**：单只占比>30% → 建议分批减至20%以下
2. **假分散**：10只股票但都是同一板块 → 行业真分散才有效
3. **风格漂移**：原本价值组合买入高PE成长股 → 检查是否有意为之
4. **忽略现金仓位**：满仓 = 放弃灵活性 → 建议保持 5-20% 现金
5. **追涨杀跌**：盈利股减仓+亏损股加仓（或反之）→ 需要有逻辑支撑
