# 多因子模型参考指南

## Barra CNE5/CNE6 模型概览

Barra 中国股票模型是 MSCI 开发的多因子风险模型，将股票收益分解为：
- **国家因子**：市场整体收益
- **行业因子**：31 个申万一级行业
- **风格因子**：10 个风格因子
- **特质收益**：个股特有的残差收益

### 10 大风格因子

| 因子 | 英文 | 计算方法 | A股数据来源 |
|------|------|---------|------------|
| 市值 | Size | ln(总市值) | quote |
| Beta | Beta | 对市场收益回归，252日窗口 | kline |
| 动量 | Momentum | 过去504日收益（去最近21日） | kline |
| 残差波动率 | Residual Volatility | 对市场回归残差的标准差 | kline |
| 非线性市值 | Non-linear Size | Size³ 对 Size 回归的残差 | quote |
| 账面市值比 | Book-to-Price | 净资产/总市值 | finance+quote |
| 流动性 | Liquidity | ln(过去21/63/252日平均换手率) | kline |
| 盈利收益率 | Earnings Yield | E/P + CF/P | finance+quote |
| 成长 | Growth | 营收增速 + 净利润增速 | finance |
| 杠杆 | Leverage | 资产负债率 + D/E + D/A | finance |

## 截面回归方法

### WLS（加权最小二乘）
- 权重 = sqrt(市值)，减少小盘股噪声影响
- 每期回归模型：R_i = Σ(f_k × X_ik) + ε_i
  - R_i: 股票 i 的收益率
  - X_ik: 股票 i 对因子 k 的暴露
  - f_k: 因子 k 的收益率（回归系数）
  - ε_i: 特质收益

### 因子收益统计检验
- t 统计量 = f_k / se(f_k)
- |t| > 2 表示因子显著
- 年化因子收益 = mean(f_k) × 252
- 因子 IR = mean(f_k) / std(f_k) × sqrt(252)

## 协方差矩阵估计

### 因子协方差（F）
- 使用指数加权移动平均（EWMA）
- 半衰期 τ = 90 天：w_t = 2^(-t/τ)
- F_kl = Σ w_t × (f_k,t - μ_k)(f_l,t - μ_l) / Σ w_t

### 特质风险（Δ）
- 对角矩阵，每只股票一个特质波动率
- σ²_i = var(ε_i)，同样使用 EWMA
- Newey-West 调整以处理自相关

### 总风险
- V = X × F × X' + Δ
  - V: 股票协方差矩阵 (N×N)
  - X: 因子暴露矩阵 (N×K)
  - F: 因子协方差矩阵 (K×K)
  - Δ: 特质风险对角矩阵 (N×N)

## 因子暴露标准化

1. **去极值**：MAD 法
   - median = median(x)
   - MAD = median(|x - median|)
   - 截断至 [median - 3×1.4826×MAD, median + 3×1.4826×MAD]

2. **标准化**：Z-score
   - z = (x - mean) / std

3. **行业/市值中性化**：
   - 对市值和行业哑变量回归，取残差作为中性化后的因子值

## A 股实践要点

1. **数据频率**：日频 K 线 + 季度财务，因子月度调仓
2. **行业分类**：申万一级（31 行业），部分研究用中信一级（29 行业）
3. **股票池**：剔除 ST、上市<60 天、停牌、涨跌停
4. **收益计算**：使用后复权价格
5. **因子滞后**：财务数据至少滞后一个季度，避免前视偏差
