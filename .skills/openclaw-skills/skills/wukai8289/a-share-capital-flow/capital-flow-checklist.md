# 资金流向预测 一页纸速查表

## 口诀
先看宏观定水位，再看政策定方向，资金连续才有效，量价共振不被骗。

## 每日3分钟速查

### 看水 10秒
- 北向连续3日流入 -> 偏多
- 北向连续3日流出 -> 谨慎

### 看风 30秒
- 当前最强政策主线: 新质生产力/科技/高端制造等

### 看量 1分钟
- 上涨放量回调缩量 = 真流入
- 上涨缩量下跌放量 = 假流入

### 看榜 1分钟
- 主力资金连续净流入 + 站上20日线 = 趋势成立

## 短线专用 1-5天

有效信号:
- 主力资金连续2日净流入
- 板块热度前排 个股涨停大涨
- 放量突破 次日不缩量大跌

离场信号:
- 高位放量滞涨
- 利好兑现高开低走
- 游资净卖出

## 中线专用 1-4周

有效信号:
- 资金连续5日净流入
- 股价站稳20日60日均线
- 业绩/行业景气向上

离场信号:
- 高位量价背离
- 机构连续减仓
- 板块热度明显退潮

## 资金真假对照表

| 现象 | 性质 | 短线 | 中线 |
|------|------|------|------|
| 连续3日净流入+温和上涨 | 真实建仓 | 可追 | 可拿 |
| 单日暴拉巨量+次日跳水 | 游资诱多 | 不追 | 回避 |
| 高位放量不涨 | 主力出货 | 快跑 | 减仓 |
| 股价新高资金不跟进 | 顶背离 | 离场 | 离场 |
| 下跌缩量拒绝放量 | 洗盘 | 可拿 | 持有 |

## 量化阈值

- 短线: 连续2日资金同向 -> 有效
- 中线: 连续5日资金同向 -> 有效
- 单日资金波动小于成交额1% -> 忽略
- 换手率突然翻倍无利好 -> 警惕出货
