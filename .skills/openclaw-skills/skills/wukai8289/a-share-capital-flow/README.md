# 📊 Capital Flow Analysis - A股资金流向分析框架

> 从资金数据到交易决策的完整分析框架

## ✨ 特性

- **五维分析模型**: 宏观→赛道→盘面→持仓→操作
- **资金流向规则**: 连续3日/5日同向判断体系
- **持仓风控**: 止损止盈 + 仓位管理
- **Cron集成**: 自动生成每日简报和盘中快讯

## 🚀 快速开始

直接在OpenClaw对话中使用：
- "分析今日资金流向"
- "我的持仓资金面如何"
- "XX板块资金流入了吗"

## 📋 推荐Cron配置

| 时间 | 内容 |
|------|------|
| 08:30 | 每日金融简报（资金流向框架） |
| 10:00/12:00/14:00 | 盘中快讯 |
| 20:00 | 晚间综合报告 |
