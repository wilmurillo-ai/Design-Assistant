# Briefing Templates — Three Modes

---

## 【09:00 Morning Briefing / 早盘简报】

```
📊 小韭早盘简报 | {YYYY-MM-DD} 09:00

🌙 昨夜美股回顾
（道指/纳指/标普500涨跌幅、重点个股、市场情绪）

🌍 隔夜重要消息
（特朗普动态、地缘政治、宏观政策等影响A/港股的关键信息）

💡 今日投资风向建议
结合昨夜美股表现和隔夜宏观，给出今日A股/港股的操作方向建议：
- 重点关注板块
- 需要回避的风险
- 1-2个具体标的参考（仅供参考，不构成投资建议）

⚠️ 以上信息仅供参考，不构成投资建议。
```

---

## 【10:00–15:00 Incremental Update / 增量快报】

```
📡 小韭增量快报 | {HH:MM}

📈 A股新动态：...
🇭🇰 港股新动态：...
🏛️ 特朗普/政策：...
⚔️ 军事/地缘：...

（只列出本小时新增的重要消息，已推送内容不重复）
```

If nothing new:
```
本时段暂无重要增量信息，市场平稳运行中。
```

---

## 【16:00 Closing Daily Report / 收盘日报】

```
📋 小韭收盘日报 | {YYYY-MM-DD}

📈 A股全天复盘
🇭🇰 港股全天复盘
🇺🇸 美股（昨夜）回顾
🏛️ 今日特朗普动态汇总
⚔️ 今日军事/地缘汇总
📋 今日政策汇总

💡 明日投资建议
结合全天信息，给出明日开盘的操作方向：
- 重点关注板块及逻辑
- 风险提示
- 2-3个具体标的参考（仅供参考，不构成投资建议）

⚠️ 以上信息仅供参考，不构成投资建议。
```
