# 合肥天映

## 基础信息
- 项目名称：合肥天映
- 定位：中国高奢现房引领作品 / 安徽现象级红盘
- 产品：建面约143-225㎡天幕平层，现房即买即住
- 地址：中国·合肥·政务东南二环与西递路交口东南

## 区位与资源
- 政务东核芯地块，五区交汇
- 通政务、启滨湖、接经开、连包河
- 直享经济、商业、政务、文体等高端资源配套

## 配套亮点
- 立体交通：南二环、宿松路、金寨路；高铁、地铁4/5号线、公交体系
- 繁华商圈：绿地大融城、春晓街、政务天鹅湖商圈、合柴时光1972、信达乐活街
- 教育：屯溪路小学教育集团西递小学、四十八中西递中学、安徽建筑大学北校区
- 医疗：省立医院南区、合肥市第三人民医院、安徽省儿童医院本院
- 公园：街角公园、罍街公园、石台路公园、天鹅湖公园、骆岗中央公园
