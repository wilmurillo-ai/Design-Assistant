# AI黑话识别清单

## 高频AI触发词（必须替换）

### 动词类
| AI词汇 | 替换建议 |
|--------|----------|
| 深入探讨 (delve) | 分析、研究、考察、探究 |
| 促进 (foster) | 推动、助力、带动 |
| 强调 (underscore) | 表明、揭示、突显 |
| 阐明 (shed light on) | 解释、说明、展示 |
| 确保 (ensure) | 有助于、促进 |
| 保证 (guarantee) | 支撑、支持 |

### 形容词类
| AI词汇 | 替换建议 |
|--------|----------|
| 多层面的 (multifaceted) | 复杂的、多元的 |
| 关键的 (pivotal) | 重要核心的、主要 |
| 错综复杂的 (intricate) | 复杂的、精妙的 |
| 全面的 (comprehensive) | 系统性的、完整的 |
| 显著的 (remarkable) | 明显的、突出的 |
| 革命性的 (revolutionary) | 创新的、突破性的 |

### 名词类
| AI词汇 | 替换建议 |
|--------|----------|
| 画卷 (tapestry) | 图景、格局 |
| 范式 (paradigm) | 模式、框架 |
| 里程碑 (milestone) | 重要进展 |
| 突破 (breakthrough) | 进展、成果 |

### 套话句式
- "值得注意的是..." → 直接陈述或删除
- "不言而喻..." → 直接陈述
- "综上所述..." → 使用语义重叠承接
- "从某种意义上说..." → 删除或简化为具体表述
- "毫无疑问..." → 考虑使用"研究显示"

## 识别技巧

1. **过度修饰**：检查是否有过多形容词的堆砌
2. **绝对化陈述**：寻找"证明了"、"保证了"等绝对语气
3. **程式化表达**：同一段落中重复出现相似的过渡模式
4. **对称结构**：AI倾向于使用平行句式，人类写作更自然随意
