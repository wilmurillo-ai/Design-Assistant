# 行业智能体厂商目录（Industry Intelligent Agent Vendors）

基于IDC MarketGlance《中国AI Agent行业应用与开发平台市场概览》，覆盖11个垂直行业子段的技术厂商清单。

**数据口径**：以IDC MarketGlance最新季度发布为准，每季度末核对一次。

> **Agent调用规则**：主Skill诊断完成后，先输出"COMPASS维度→供应商类型→MarketGlance子段"三层链路。用户追问具体厂商时，给出对应子段下所有候选厂商的完整清单，由用户自行筛选。不在对话中挑出单个厂商推荐。
