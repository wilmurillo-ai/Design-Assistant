# 工具：发送飞书消息给使用者

> **注意**：约面试流程中的时间确认，应直接在当前对话（Openclaw Channels）中回复用户，无需调用此工具。
> 此工具适用于需要**主动推送通知**的场景（用户不在当前对话中，需要从飞书 IM 触达时）。

**用途**：主动向使用者推送通知（非对话中确认）

## 使用者信息

使用者姓名和 open_id 见 **SKILL.md 常量表**。

## 命令（bot 身份发送）

```bash
lark-cli im +messages-send \
  --as bot \
  --user-id <YOUR_OPEN_ID> \
  --text "<消息内容>"
```

> `+messages-send` 必须显式加 `--as bot`，否则会报 identity 错误。

## 典型消息模板

### 约面试时间确认
```
【招聘助手】候选人 <姓名> 对约面试有意向，已查看您的日程。
建议面试时间：
• 选项1：<日期> <时间>
• 选项2：<日期> <时间>
请确认哪个时间OK，确认后我会发日程给对方。
```

### 通知面试已约好
```
【招聘助手】已与 <姓名> 约好面试：
时间：<日期 时间>
形式：视频会议
日程已添加到您的飞书日历。
```
