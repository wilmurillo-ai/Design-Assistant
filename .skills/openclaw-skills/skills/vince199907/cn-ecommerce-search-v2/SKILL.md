# SKILL.md - 中国电商搜索

用于在淘宝、天猫、小红书等中国平台搜索商品和获取产品信息。

## 触发条件

当用户提到以下内容时激活：
- 淘宝、天猫、京东、1688、咸鱼、拼多多、小红书、抖音商品搜索
- 中国供应商查找
- 中国平台价格查询
- 分享了淘宝/天猫/京东/1688/咸鱼/拼多多/小红书/抖音链接

## 重要规则（必须遵守）

### 1. 必须搜索全部8个平台

每次搜索商品价格，**必须**查询以下全部平台：
1. 淘宝
2. 天猫
3. 京东
4. 拼多多
5. 咸鱼
6. 小红书
7. 抖音
8. 1688

⚠️ **不能只搜索部分平台就输出结果！**

### 2. 链接要求

输出的每个商品**必须**是具体的商品页面链接，不是搜索页链接！

✅ 正确：
- https://item.jd.com/53336639517.html （京东具体商品）
- https://detail.tmall.com/item.htm?id=931605930758 （天猫具体商品）
- https://www.goofish.com/item?id=1030262585596 （咸鱼具体商品）

❌ 错误：
- https://search.jd.com/Search?keyword=商品关键词 （京东搜索页）
- https://s.taobao.com/search?q=商品关键词 （淘宝搜索页）

### 3. 输出流程

1. 先打开所有8个平台的搜索页面
2. 收集每个平台的商品信息和具体链接
3. **全部查完后再输出完整的价格对比表**
4. 不能查1-2个平台就输出结果

## 功能

1. **商品搜索** - 在指定平台搜索产品
2. **链接解析** - 提取商品链接中的关键信息
3. **价格对比** - 比较不同平台的价格
4. **详情获取** - 获取商品基本信息（价格、销量、店铺等）

## 支持平台

- 淘宝
- 天猫
- 京东
- 1688（阿里巴巴批发）
- 咸鱼（二手）
- 拼多多
- 小红书
- 抖音

## 使用方式

用户可以：
- 直接发商品链接让你分析
- 描述想找的产品进行搜索
- 询问某产品的价格

## 注意事项

- 支持中文交流
- 需要使用浏览器或网页工具访问这些平台
- 小红书/抖音内容可能需要特殊处理
- 如果某个平台没有结果，明确标注"无"
- 如果需要登录才能搜索，提醒用户登录
