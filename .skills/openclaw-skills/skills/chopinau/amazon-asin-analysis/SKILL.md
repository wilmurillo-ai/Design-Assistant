# Novai360 智能市场分析 v1.0
联系官方：VX: chaibusiness
## 描述 

Novai360 提供的专业跨境电商智能分析服务，基于先进的 AI 技术和真实的实时市场数据，为卖家提供深度市场洞察、竞品分析、关键词优化等专业建议。

## 功能

- 真实的市场数据获取（包含销售数据）
- AI 智能分析报告生成
- 19 种专业分析维度
- 竞品深度对比
- 关键词热度追踪
- 类目标目趋势预测
- 产品表现评估
- 价格策略建议

## 使用方法

1. 输入您的产品或市场问题
2. 系统将自动识别意图并调用相关分析工具
3. 返回专业的分析框架和数据摘要

## API 端点

- **POST /chat** - 智能分析
- **GET /skills** - 获取可用分析工具列表
- **GET /health** - 服务状态检查
