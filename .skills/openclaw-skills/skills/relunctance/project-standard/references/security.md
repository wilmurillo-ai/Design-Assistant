# 安全规范

## 安全评审检查清单

### 认证授权

- [ ] 所有接口有鉴权
- [ ] 敏感接口有多因素认证
- [ ] Token 有效期合理
- [ ] 权限最小化原则

### 数据安全

- [ ] 敏感数据加密存储
- [ ] 传输使用 HTTPS
- [ ] 日志无敏感信息
- [ ] 数据脱敏规范

### 接口安全

- [ ] SQL 注入防护
- [ ] XSS 防护
- [ ] CSRF 防护
- [ ] 请求频率限制

### 依赖安全

- [ ] 无高危 CVE
- [ ] 依赖版本锁死
- [ ] 定期更新依赖

## 安全漏洞等级

| 等级 | 定义 | 修复时限 |
|------|------|----------|
| 严重 | 系统数据泄露/权限绕过 | 24h |
| 高危 | 敏感数据泄露 | 72h |
| 中危 | 功能异常/信息泄露 | 2周 |
| 低危 | 体验问题 | 下一版本 |

## 加密规范

- 密码：bcrypt，cost ≥ 12
- 敏感字段：AES-256
- 传输：TLS 1.2+
- Key 管理：不硬编码，不提交 Git
