# 运维标准

## 监控告警标准

### 核心指标

| 指标 | 告警阈值 | 恢复阈值 |
|------|----------|----------|
| CPU 使用率 | > 80% | < 70% |
| 内存使用率 | > 85% | < 75% |
| 磁盘使用率 | > 80% | < 70% |
| 错误率 | > 0.5% | < 0.3% |
| 响应时间 P99 | > 1s | < 800ms |

### 告警处理

- P0：立即处理，电话通知
- P1：30分钟内响应
- P2：2小时内响应
- P3：下一工作日

## 日志规范

- 格式：JSON
- 级别：DEBUG/INFO/WARN/ERROR
- 保留：生产 7 天，测试 3 天
- 禁止记录：密码、Token、敏感个人信息

## 变更管理

- 变更申请：提前 1 个工作日
- 变更审批：P0 变更需 CTO 审批
- 变更窗口：工作日 10:00-16:00
- 变更记录：完整记录操作步骤
