# 指标体系标准

## 核心指标

### 产品指标

| 指标 | 定义 | 目标 |
|------|------|------|
| DAU | 日活跃用户数 | |
| MAU | 月活跃用户数 | |
| 转化率 | 完成目标用户/访问用户 | |
| 留存率 | 次日/7日/30日留存 | |

### 质量指标

| 指标 | 定义 | 目标 |
|------|------|------|
| 响应时间 | P50/P95/P99 | P99 < 1s |
| 错误率 | 失败请求/总请求 | < 0.1% |
| 可用性 | 正常运行时间 | ≥ 99.9% |

## 指标定义规范

- 口径清晰，有明确计算公式
- 数据来源明确
- 更新频率定义
- 异常阈值定义
