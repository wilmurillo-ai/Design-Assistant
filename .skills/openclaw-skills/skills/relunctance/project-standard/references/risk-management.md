# 风险管理标准

## 风险识别

### 常见风险类别

| 类别 | 常见风险 |
|------|----------|
| 人员 | 关键人员离职、技能不足 |
| 技术 | 技术选型失误、架构缺陷 |
| 进度 | 需求变更、依赖延迟 |
| 资源 | 预算超支、人力不足 |
| 外部 | 政策变化、供应商风险 |

## 风险评估矩阵

|  | 影响低 | 影响中 | 影响高 |
|--|--------|--------|--------|
| **概率高** | 中 | 高 | 高 |
| **概率中** | 低 | 中 | 高 |
| **概率低** | 低 | 低 | 中 |

## 风险应对策略

| 策略 | 适用场景 |
|------|----------|
| 规避 | 风险高且概率高，改变计划 |
| 转移 | 超出能力范围，保险/外包 |
| 缓解 | 降低概率或影响 |
| 接受 | 风险低或不可避免 |

## 风险跟踪

- 风险清单每周更新
- 高风险项每日检查
- 风险升级：48h 未解决自动升级
