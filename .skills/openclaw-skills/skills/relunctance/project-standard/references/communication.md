# 沟通协同标准

## 会议标准

| 会议 | 频率 | 时长 | 参与人 |
|------|------|------|--------|
| 每日站会 | 每天 | 15min | 团队 |
| 周会 | 每周 | 30min | 团队+干系人 |
| 迭代评审 | 迭代末 | 1h | 团队+产品 |
| 回顾会 | 迭代末 | 1h | 团队 |

## 信息同步标准

### 同步内容

- 项目进度：每周同步
- 风险预警：发现即同步
- 变更通知：变更后 24h 内
- 里程碑：提前 1 周预警

### 同步渠道

| 内容 | 渠道 |
|------|------|
| 日常沟通 | 即时通讯 |
| 项目周报 | 邮件 |
| 紧急问题 | 电话/视频 |
| 决策记录 | 文档 |

## 决策机制

- 技术决策：技术负责人
- 产品决策：产品经理
- 跨团队：项目负责人
- 高风险：评审会

## 文档标准

- 决策文档：每次决策后 24h 内
- 会议纪要：会议后 24h 内
- 技术文档：上线前完成
- API 文档：接口开发时同步
