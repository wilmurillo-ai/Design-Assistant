# 发布标准

## 发布前检查

### 功能检查

- [ ] 测试用例 100% 通过
- [ ] 核心流程回归通过
- [ ] 性能测试达标
- [ ] 兼容性测试通过

### 安全检查

- [ ] 安全扫描通过
- [ ] 无敏感信息泄露
- [ ] 配置审核通过

### 运维检查

- [ ] 监控告警配置完成
- [ ] 回滚方案准备完毕
- [ ] 变更公告已发送
- [ ] 值班人员已通知

## 发布流程

1. **预发布验证**：灰度 5% → 30% → 100%
2. **发布时间窗口**：工作日 10:00-16:00
3. **发布观察期**：发布后 30 分钟监控
4. **回滚标准**：错误率 > 1% 或响应时间 > 5s

## 回滚标准

立即回滚：
- P0 缺陷影响核心功能
- 安全漏洞被利用
- 数据损坏

可观察后回滚：
- 错误率上升超过阈值
- 响应时间明显下降
