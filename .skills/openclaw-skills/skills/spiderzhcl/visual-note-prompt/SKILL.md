# 视觉笔记提示词生成器（社区版）

## 技能用途

此技能用于将用户提供的简短文本（如学习要点、知识总结、概念说明等）转换为标准化的图片生成提示词。生成的提示词严格遵循视觉笔记生图指令格式，确保输出符合AI图像生成器的要求。

## 触发条件

当用户需要：
- 将学习内容转换为视觉化知识卡片
- 生成符合特定格式的图片提示词
- 创建手绘涂鸦风格的视觉笔记
- 将文本内容拆分为多张知识卡片

## 技能内容结构

### 核心能力

1. **内容分析**：自动统计要点数和字数，确定图片拆分数量
2. **风格识别**：识别用户偏好的视觉风格（默认为classic经典手账风格）
3. **构图选择**：根据内容类型选择合适的构图布局
4. **格式标准化**：严格遵循视觉笔记生图指令格式要求

### 支持的风格类型（2种）

- classic: 经典手账风格（浅米色纸张，暖色系配色）
- tech: 科技未来风格（深蓝灰渐变，霓虹配色）

### 支持的构图类型（2种）

- cards: 竖向卡片堆叠（适合要点罗列、步骤说明）
- center: 中心发散布局（适合核心概念、辐射关系）

### 生图类型与尺寸

本版本仅支持知识卡片类型：

| 生图类型 | 比例 | 推荐尺寸 |
|----------|------|----------|
| **知识卡片/视觉笔记** | 3:4竖版 | 900×1200 |

## 使用流程

### 输入要求

用户提供：
1. 主标题（第一行）
2. 具体内容要点（后续行）
3. 可选：风格偏好（classic / tech）、构图偏好（cards / center）

### 处理步骤

1. **内容解析**：提取主标题，统计要点数和字数
2. **拆分判断**：根据标准表格确定图片数量
3. **风格确定**：使用用户指定风格或默认classic
4. **构图选择**：根据内容类型自动选择或使用用户指定
5. **提示词生成**：逐张生成标准化提示词

### 输出格式

严格按照以下格式输出每张图片的提示词：

**单张图时（仅1张）：**

```
---【图开始】---
[Style]: 手绘涂鸦笔记(Sketchnote)，{风格标签}，手绘感线条，{风格背景}，{风格配色}，所有文字使用不规则不完美的手写体（非黑体非印刷体）
[Layout]: 3:4竖版（900×1200），{构图标签}，独立完整画布，非拼接
[Title]: `{主标题}`（位于画面顶部，醒目横幅/框框内，旁边画{标题相关简笔画}）
[Core Content]: 
- 左侧平衡区域：`内容文字`（旁边画{具体插画描述}）
- 中央宽阔区域：`内容文字`（旁边画{具体插画描述}）
- 右侧平衡区域：`内容文字`（旁边画{具体插画描述}）
[Decoration]: 手绘箭头、星星、连接线、点缀图标
[Negative Prompt]: 不要出现任何百分比数字（如30%、28%等）。不要出现百分比符号(%)。画面中不包含任何区域划分的坐标数字。不要有统计数字。不要使用黑体、宋体等印刷体字体，所有文字必须是不规则不完美的手写体。
[Author]: 底部居中标注"By 八点虾"作者署名，旁边画小标签和庆祝的彩色烟花插画
---【图结束】---
```

**多张图时（2张及以上）：**

```
---【图开始】---
[Style]: 手绘涂鸦笔记(Sketchnote)，{风格标签}，手绘感线条，{风格背景}，{风格配色}，所有文字使用不规则不完美的手写体（非黑体非印刷体）
[Layout]: 3:4竖版（900×1200），{构图标签}，独立完整画布，非拼接
[Series]: 0{当前图号}/0{总图数}
[Title]: `{主标题}`（位于画面顶部，醒目横幅/框框内，旁边画{标题相关简笔画}）
[Core Content]: 
- 左侧平衡区域：`内容文字`（旁边画{具体插画描述}）
- 中央宽阔区域：`内容文字`（旁边画{具体插画描述}）
- 右侧平衡区域：`内容文字`（旁边画{具体插画描述}）
[Decoration]: 手绘箭头、星星、连接线、点缀图标
[Negative Prompt]: 不要出现任何百分比数字（如30%、28%等）。不要出现百分比符号(%)。画面中不包含任何区域划分的坐标数字。不要有统计数字。不要使用黑体、宋体等印刷体字体，所有文字必须是不规则不完美的手写体。
[Author]: 系列标识"[0X/0X]"置于右上角，底部居中标注"By 八点虾"作者署名，旁边画小标签和庆祝的彩色烟花插画，{最后一张图加"完结"标记}
---【图结束】---
```

### 位置描述规则（严禁违反）

- **严禁使用百分比数字来规划区域大小**（如"左侧30%"、"顶部15-20%"）
- **必须用描述性语言代替数字**：如"左侧平衡区域"、"中央宽阔区域"、"右侧紧凑区域"、"顶部醒目横幅内"
- **必须在每张图的 [Negative Prompt] 中明确禁令百分比和数字**
- **[Core Content] 中的文字内容仅限用户指定的展示文字**，不得包含任何规划用的百分比或坐标数字

## 使用示例

当用户输入：
```
学习效率提升方法
番茄工作法：25分钟专注+5分钟休息
时间块管理：将任务分配到固定时间段
```

技能将自动：
1. 识别主标题为"学习效率提升方法"
2. 统计要点数2个，字数约40字
3. 判断需要1张图片
4. 选择classic风格和cards构图
5. 生成标准化的图片提示词

## 注意事项

- 严格遵守格式边界标记，不得在`---【图开始】---`前或`---【图结束】---`后添加任何内容
- 所有文字内容必须用反引号包裹
- 每个元素必须说明具体位置描述（使用描述性语言，禁止用百分比数字）
- 插画必须与对应文字一起描述
- 严禁合并画布或拼接图片
- 严禁使用黑体、宋体等印刷体字体
- 严禁在位置描述中使用百分比数字
- 每张图必须包含 [Negative Prompt] 字段
