# Visual Note Prompt - 社区版

> 🆓 免费开源的视觉笔记提示词生成器，将文本转换为高质量知识卡片图片提示词。
> 适用于 WorkBuddy / OpenClaw 等 AI 助手平台的 Skill 插件。

## ✨ 特性

- 🎨 **2种精选风格**：经典手账 + 科技未来
- 📐 **2种核心构图**：卡片堆叠 + 中心发散
- 📏 **固定知识卡片比例**：3:4 竖版（900×1200）
- 🖊️ **手绘涂鸦风格**：所有文字使用不规则手写体
- 🔤 **严格格式规范**：标准化字段结构，确保AI图像生成器正确解析
- 📄 **自动内容拆分**：根据要点数量和字数智能拆分为多张卡片

## 📁 文件结构

```
visual-note-prompt/
```

## 🚀 安装方法

### 方式一：直接下载
1. 下载本仓库全部文件
2. 放入 `~/.workbuddy/skills/visual-note-prompt/` 目录
3. 重启 WorkBuddy，技能自动加载

### 方式二：Git Clone
```bash
git clone https://github.com/spiderzhcl/visual-note-prompt ~/.workbuddy/skills/visual-note-prompt
```

## 📖 使用方法

安装完成后，在 WorkBuddy 中直接输入你的知识要点即可，例如：

```
学习效率提升方法
番茄工作法：25分钟专注+5分钟休息
时间块管理：将任务分配到固定时间段
```

技能会自动生成标准化的图片提示词，可直接用于 Midjourney / DALL·E / 通义万相 / Hunyuan 等 AI 绘图工具。

## 🎨 支持的风格

| 风格 | 标签 | 特点 | 适用场景 |
|------|------|------|----------|
| 经典手账 | `classic` | 浅米色纸张，暖色系配色 | 学习笔记、生活记录 |
| 科技未来 | `tech` | 深蓝灰渐变，霓虹配色 | 技术概念、产品介绍 |

## 📐 支持的构图

| 构图 | 标签 | 特点 | 适用场景 |
|------|------|------|----------|
| 竖向卡片堆叠 | `cards` | 卡片从上到下排列 | 要点罗列、步骤说明 |
| 中心发散 | `center` | 核心居中，周围环绕 | 核心概念、辐射关系 |

## 📋 提示词输出格式

每张图片的提示词包含以下标准化字段：

| 字段 | 说明 |
|------|------|
| `[Style]` | 视觉风格描述（手绘涂鸦 + 风格标签 + 背景 + 配色） |
| `[Layout]` | 布局与比例描述 |
| `[Series]` | 系列标识（仅多张图时出现） |
| `[Title]` | 标题区域描述 |
| `[Core Content]` | 核心内容 + 插画描述 |
| `[Decoration]` | 装饰元素描述 |
| `[Negative Prompt]` | 负面提示词（禁令百分比数字、印刷体字体等） |
| `[Author]` | 作者署名信息 |
