# openclaw-ticket-assistant

OpenClaw 工单助手 Skill，供 AI Agent 根据客服对话截图自动创建工单。

## 功能

| 功能 | 说明 |
|------|------|
| 截图分析 | 识别客服对话截图中的客户、反馈人、问题内容和平台类型 |
| 客户匹配 | 根据截图信息自动匹配客户列表中的客户 |
| 创建工单 | 自动打开工单系统并填写表单创建工单 |

## 使用方式

发送客服对话截图给 AI，说：

- "帮我创建一个工单"
- "提个工单"
- "根据这个截图创建工单"

## 目录结构

```
openclaw-ticket-assistant/
├── SKILL.md              # Skill 主文件，定义执行流程
├── README.md             # 说明文档
├── references/           # 引用文档（子流程说明等）
├── scripts/              # 辅助脚本
└── resources/            # 资源文件
    └── customers.json    # 客户列表
```

## 工作原理

1. **截图分析**：AI 分析用户提供的客服对话截图
2. **客户识别**：根据截图中的账号名、客户名称匹配 `resources/customers.json` 中的客户
3. **信息提取**：识别客户提问内容、平台类型（Android/iOS/鸿蒙）、反馈人
4. **确认信息**：向用户确认识别结果，无法识别的信息询问用户
5. **创建工单**：打开工单系统，填写表单并提交

## 输出格式

```
工单号：000F4N4G08
站点：广发银行
反馈人：罗曦冉
产品：鸿蒙
问题描述：请教一下鸿蒙原生在使用页面嵌入模式来展示H5时，看起来页面容器不是透明的，有办法设置透明吗？
```

## 依赖

| 依赖 | 说明 | 用途 |
|------|------|------|
| **浏览器 MCP** | 提供浏览器自动化工具 | 打开工单系统、填写表单 |
| **域账号登录** | 阿里内网域账号 | 访问工单系统需要登录 |

## 前置条件及限制

- 需要通过浏览器登录域账号才能访问工单系统
- 截图应清晰可读，包含完整的对话内容
- 无法识别的信息会询问用户确认
