# Women's Day Support Skill

A skill to provide resources, support, and celebration for International Women's Day (March 8th) and beyond.

## Features
- Provides information about local women's rights organizations and support services
- Offers inspirational quotes and stories from women leaders
- Helps users find women-owned businesses to support
- Provides educational resources about women's history and achievements
- Offers self-care and wellness tips specifically for women
- Connects users with community events celebrating Women's Day

## Usage
- `/womensday resources` - Get local support resources
- `/womensday quotes` - Receive inspirational quotes from women leaders
- `/womensday businesses` - Find women-owned businesses to support
- `/womensday history` - Learn about women's historical achievements
- `/womensday selfcare` - Get wellness and self-care recommendations
- `/womensday events` - Discover local Women's Day events

## Configuration
The skill can be configured with location data to provide relevant local resources.

## Privacy
All user data is handled with strict privacy controls. Location data is only used to provide relevant local resources and is not stored.
