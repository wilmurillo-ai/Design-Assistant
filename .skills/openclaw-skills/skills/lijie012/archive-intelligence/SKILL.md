# 档案行业情报收集

## 功能概述

帮助用户定时获取档案行业的最新情报，包括：
- 政策法规更新
- 行业新闻动态
- 技术发展趋势
- 学术研究成果
- 标准规范发布

## 使用方法

### 1. 单次获取情报

直接请求获取最新情报：
```
获取档案行业最新情报
搜索档案数字化相关政策
查找档案管理新技术
```

### 2. 设置定时任务

设置定时收集任务：
```
每天上午9点获取档案行业新闻
每周一获取本周档案政策汇总
每月1号生成档案行业月报
```

## 执行步骤

### 单次情报收集

1. **确定收集范围**
   - 明确情报类型（政策/新闻/技术/学术）
   - 确定时间范围
   - 选择信息来源

2. **搜索情报**
   - 使用网络搜索工具
   - 访问官方网站
   - 查询学术数据库

3. **整理汇总**
   - 按类别分类
   - 提取关键信息
   - 生成摘要

4. **输出结果**
   - 使用标准格式
   - 提供原文链接
   - 给出趋势分析

### 定时任务设置

1. **创建任务配置**
   ```json
   {
     "task_name": "档案日报",
     "schedule": "0 9 * * *",
     "sources": ["政策", "新闻", "技术"],
     "keywords": ["档案数字化", "电子档案"],
     "output_format": "markdown",
     "notification": true
   }
   ```

2. **设置执行计划**
   - 使用系统定时任务
   - 配置执行环境
   - 设置通知方式

3. **验证任务运行**
   - 测试任务执行
   - 检查输出结果
   - 调整配置参数

## 注意事项

1. **信息时效性**
   - 优先获取最新信息
   - 标注发布时间
   - 定期更新旧信息

2. **来源可靠性**
   - 优先官方渠道
   - 验证信息准确性
   - 标注信息来源

3. **内容合规性**
   - 遵守版权规定
   - 不传播涉密信息
   - 尊重知识产权

4. **任务稳定性**
   - 设置错误重试
   - 记录执行日志
   - 定期检查任务状态
