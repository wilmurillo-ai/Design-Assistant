## 示例 2：设置每日定时任务

**用户请求：**
```
每天早上9点自动获取档案行业新闻
```

**执行步骤：**
1. 创建任务配置文件 `daily-config.json`
2. 设置定时任务（Windows任务计划程序或Linux cron）
3. 配置执行脚本

**配置文件示例：**
```json
{
  "task_name": "档案日报",
  "schedule": "0 9 * * *",
  "sources": ["政策", "新闻", "技术"],
  "keywords": ["档案管理", "档案数字化", "电子档案"],
  "output_format": "markdown",
  "output_dir": "./reports/daily",
  "notification": true
}
```

**Windows 定时任务设置：**
```powershell
# 创建定时任务
schtasks /create /tn "档案日报" /tr "node C:\Users\PC\.qoder\skills\archive-intelligence\scripts\collect-intelligence.js C:\Users\PC\.qoder\skills\archive-intelligence\examples\daily-config.json" /sc daily /st 09:00
```

## 示例 5：设置月度汇总任务

**用户请求：**
```
每月1号生成上个月的档案行业月报
```

**执行步骤：**
1. 创建月报配置文件 `monthly-config.json`
2. 设置每月1号执行的定时任务
3. 配置月报生成逻辑

**配置文件示例：**
```json
{
  "task_name": "档案月报",
  "schedule": "0 9 1 * *",
  "sources": ["政策", "新闻", "技术", "学术"],
  "keywords": [
    "档案管理",
    "档案数字化",
    "电子档案",
    "档案信息化",
    "智慧档案馆",
    "档案安全",
    "档案法"
  ],
  "output_format": "markdown",
  "output_dir": "./reports/monthly",
  "max_results_per_source": 30,
  "notification": true
}
```

## 使用技巧

### 1. 关键词优化
- 使用核心关键词获取全面信息
- 使用专业关键词获取深度内容
- 组合多个关键词提高精准度

### 2. 来源选择
- 政策类：优先官方渠道
- 新闻类：选择权威媒体
- 技术类：关注行业期刊
- 学术类：使用学术数据库

### 3. 定时任务管理
- 日报：适合跟踪最新动态
- 周报：适合趋势分析
- 月报：适合深度研究

### 4. 报告定制
- 调整 `max_results_per_source` 控制信息量
- 修改 `keywords` 定制关注领域
- 配置 `notification` 接收完成提醒
