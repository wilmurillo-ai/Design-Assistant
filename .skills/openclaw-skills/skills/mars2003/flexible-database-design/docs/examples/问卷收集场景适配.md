# 问卷收集场景适配

> **应用案例**：基于 Flexible Database Skill 构建的表单/问卷收集系统。  
> 验证场景：员工满意度、培训反馈等，千条级规模。

---

## 一、场景特征

| 维度 | 说明 |
|------|------|
| **主干字段** | form_id、form_version、respondent_id、submitted_at |
| **软字段** | 题目 id → 答案（q1_satisfaction、q2_department、q3_improvement 等） |
| **题型** | 数值、单选、填空、多选、日期 |

---

## 二、从 Skill 继承

- 完整三层模型：records + dynamic_data + 业务视图
- 目录结构：references/ + scripts/ + data/
- 批量导入：import_batch 兼容 CSV

---

## 三、关键约定

**答案展平**：题目答案需放在 `extracted` **顶层**，而非嵌套在 `answers` 对象内，否则 `_flatten_to_dynamic` 无法按题目查询和统计。

```json
// ✅ 推荐：顶层
{"form_id":"survey_2025_q1","q1_satisfaction":4,"q2_department":"研发"}

// ❌ 不推荐：嵌套（无法按题目统计）
{"form_id":"survey_2025_q1","answers":{"q1_satisfaction":4,"q2_department":"研发"}}
```

---

## 四、交付物

- schema_template（source_type 含 form）
- gen_survey_csv.py：生成测试数据
- import_batch.py：批量导入
