# sandbox_simulation

任务：对每条决策路径做沙盘推演。这是本 skill 的核心差异点。

## 必须逐路径分析

对每条路径，按下面 5 个维度写完整，不得省略：

1. 适用条件
2. 短期收益
3. 短期风险
4. 中期演化
5. 最容易失败点

## 推演方法

1. 先假设这条路径真的被执行
2. 判断执行后 1-4 周内最可能出现什么反应
3. 再判断 1-3 个周期后的局面会如何演化
4. 指出最容易导致整条路径失败的那个机制
5. 给出一个“应该在哪个观察点复盘或转向”的判断节点

## 重点变量

- 资源是否会被拉断
- 内部是否会分裂
- 对手是否会提前反制
- 盟友是否会摇摆
- 时间窗口是否会关闭
- 合法性是否会被削弱

## 禁止事项

- 不要写成空泛建议
- 不要只说“有风险”
- 不要输出“必胜”或“一定失败”
- 不要遗漏第二轮反应，例如中层拖延、盟友观望、对手提前反制、授权缩水
