## 🚀 快速开始

### 抓取热搜

```bash
./scripts/fetch-trending.js
```

### 生成内容

```bash
./scripts/generate-content.js --topic "AI Agent" --save
```

### 多平台适配

```bash
./scripts/format-platform.js --all
```
