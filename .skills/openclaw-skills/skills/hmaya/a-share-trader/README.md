# A股核心交易框架

基于用户提供的12种选股策略，结合自适应交易引擎，构建的A股量化交易系统。

## 🎯 核心特性

1. **智能自适应**：通过对市场行情变化的感知适时调整交易策略和投资标的阈值
2. **多策略整合**：整合12种选股策略，根据不同市场环境动态调整权重
3. **风险自适应**：根据市场状态动态调整止损、仓位、换手率等风控参数
4. **学习优化**：从交易绩效中持续学习，优化策略参数和权重分配

| 策略名称 | 核心逻辑 | 适用市场 |
|---------|---------|---------|
| 1. 基本面选股 | 质量+增长双因子，排除ST/科创板 | 价值投资 |
| 2. 防御选股 | 高股息率，市值/PE截面排名 | 熊市/防御 |

| 6. 小市值成长 | 综合评分（市值+价格+增长） | 成长小盘 |
| 7. 营收利润双增 | 增长质量，市值排序 | 成长投资 |
| 8. 控盘策略 | 筹码集中度，中证1000成分 | 技术分析 |

## 🔧 自适应机制

### 市场状态识别
系统自动识别6种市场状态：
- **强牛市**：提高成长/小市值策略权重，放宽风控
- **弱牛市**：均衡配置，适度风控
- **强熊市**：提高防御/社保重仓权重，收紧风控
- **弱熊市**：侧重防御，谨慎配置
- **高波动震荡**：侧重震荡策略，中等风控
- **横盘震荡**：均衡配置，低换手

### 参数自适应调整
根据市场状态动态调整：
- **策略权重**：不同市场环境下策略权重自动调整
- **筛选阈值**：PE/PB/增长要求等阈值动态变化
- **风控参数**：止损幅度、仓位限制、换手率目标

## 📈 性能目标

- **年化收益率**：≥150%（基于10亿资金模拟）
- **最大回撤**：≤15%
- **夏普比率**：≥3.0
- **胜率**：≥60%

## 🛡️ 风险管理

### 多层风控体系
1. **个股层面**：单只股票仓位≤8%，止损≤3%
2. **行业层面**：单行业仓位≤25%
3. **组合层面**：最大回撤≤15%，VaR监控
4. **市场层面**：根据市场状态动态调整风险敞口

### 自适应风控
- 牛市中：适度放宽止损，提高仓位上限
- 熊市中：收紧止损，降低仓位，提高防御
- 震荡市中：中等风控，注重流动性

## 🔄 工作流程

```
市场数据更新 → 计算市场指标 → 识别市场状态 → 
获取自适应决策 → 调整策略权重 → 优化策略参数 → 
生成交易信号 → 风险检查 → 执行交易 → 
绩效反馈 → 学习优化
```
