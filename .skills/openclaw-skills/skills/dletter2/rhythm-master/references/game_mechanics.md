# 节奏大师 - 游戏机制说明

## 游戏概述

《节奏大师》是一款键盘节奏游戏，玩家需要在正确的时机按下对应的按键（↑↓←→空格），根据准确度和连击数获得分数。

## 核心玩法

### 按键系统

#### 基础按键
- **↑** (上箭头) - 对应方向 "UP"
- **↓** (下箭头) - 对应方向 "DOWN"
- **←** (左箭头) - 对应方向 "LEFT"
- **→** (右箭头) - 对应方向 "RIGHT"
- **Space** (空格键) - 对应 "HIT"

#### 按键显示方式
```

║  ←    ↑    SPACE    ↓    →          ║
║  ○    ○     ○      ○    ○           ║
║                                      ║
║         ██████████████████          ║  ← 判定线
║                                      ║
╚══════════════════════════════════════╝
```

### 游戏流程

#### 1. 开始界面
- 显示游戏标题和 Logo
- 选择游戏模式
- 查看排行榜
- 设置选项

#### 2. 游戏准备
- 3秒倒计时
- 显示即将开始的提示
- 玩家准备手位（建议：左手 ←↓，右手 →↑，拇指空格）

#### 3. 游戏进行
- 音符从上方落下
- 经过判定线时按下对应按键
- 实时显示判定结果和分数
- 显示连击数

#### 4. 游戏结束
- 显示最终得分
- 显示统计信息（Perfect/Great/Good/Miss 数量）
- 显示最大连击数
- 输入玩家名字保存分数
- 查看排行榜排名

## 判定系统

### 判定等级

| 判定 | 时间窗口 | 得分倍率 | 效果 |
|------|----------|----------|------|
| **PERFECT** | ±50ms | 100% | 完美命中，连击+1 |
| **GREAT** | ±100ms | 80% | 很好，连击+1 |
| **GOOD** | ±150ms | 50% | 一般，连击+1 |
| **BAD** | ±200ms | 20% | 较差，连击中断 |
| **MISS** | >200ms 或未按 | 0% | 错过，连击中断 |

### 判定显示
- **PERFECT**: 彩虹色闪烁 + "PERFECT!" 文字
- **GREAT**: 金色闪烁 + "GREAT!" 文字
- **GOOD**: 绿色闪烁 + "GOOD" 文字
- **BAD**: 灰色闪烁 + "BAD" 文字
- **MISS**: 红色闪烁 + "MISS" 文字 + 屏幕轻微震动

## 得分系统

### 基础得分公式
```
单音符得分 = 基础分(100) × 判定倍率 × 连击加成 × 难度系数
```

### 连击加成
| 连击数 | 加成倍率 |
|--------|----------|
| 1-10 | 1.0x |
| 11-30 | 1.1x |
| 31-50 | 1.2x |
| 51-100 | 1.5x |
| 101-200 | 2.0x |
| 200+ | 2.5x |

### 难度系数
| 难度 | 下落速度 | 音符密度 | 系数 |
|------|----------|----------|------|
| 简单 (EASY) | 慢 | 稀疏 | 1.0x |
| 普通 (NORMAL) | 中等 | 中等 | 1.2x |
| 困难 (HARD) | 快 | 密集 | 1.5x |
| 专家 (EXPERT) | 极快 | 极密 | 2.0x |
| 大师 (MASTER) | 极速 | 疯狂 | 3.0x |

### 评级系统

| 评级 | 得分百分比 | 称号 |

## 游戏模式

### 1. 经典模式 (Classic)
- 标准节奏游戏玩法
- 随机生成音符序列
- 固定时长（60秒/120秒/完整歌曲）
- 适合练习和熟悉操作

### 2. 无尽模式 (Endless)
- 无限进行直到 MISS 数达到上限（3次）
- 速度会逐渐加快
- 挑战最高连击和分数
- 适合高手挑战极限

### 3. 挑战模式 (Challenge)
- 特定的高难度谱面
- 预设的音符序列
- 完成特定目标（如：全连击、不准有MISS等）
- 解锁成就和奖励

### 4. 对战模式 (VS Mode)
- 两名玩家同屏对战
- 相同音符序列
- 分数高者获胜
- 支持本地双人（键盘左右分区）

### 5. 练习模式 (Practice)
- 可调节速度（0.5x - 2.0x）
- 可暂停和回放
- 显示判定线提示
- 适合新手学习

## 音符类型

### 基础音符
- **单音符 (Single)**: 普通按键，按一次即可

### 高级音符
- **长音符 (Long)**: 按住不放直到音符结束
- **连音符 (Chain)**: 快速连续按下同一按键多次
- **组合音符 (Combo)**: 同时按下多个按键（如：↑+→）

### 特殊音符
- **奖励音符 (Bonus)**: 金色音符，击中获得额外分数
- **生命音符 (Heal)**: 绿色音符，恢复失误次数
- **危险音符 (Danger)**: 红色音符，错过会扣除大量分数

## 生命值系统

### 经典模式
- 初始生命：5颗心
- BAD：-0.5心
- MISS：-1心
- 生命归零：游戏结束

### 无尽模式
- 初始生命：3次失误机会
- 任何非 PERFECT/GREAT：消耗 1 次机会
- 机会归零：游戏结束

## 排行榜系统

### 本地排行榜
- 存储在浏览器 LocalStorage
- 保存前 100 名分数
- 包含：排名、玩家名、分数、连击、难度、日期

### 联网排行榜

## 视觉效果

### 判定线
- 发光效果
- 判定成功时闪烁对应颜色
- 判定失败时红色警告

- 不同按键用不同颜色区分
- 音符带有光晕效果
- 接近判定线时放大提醒

### 背景效果
- 动态背景随节奏变化
- 连击数高时背景更加华丽
- 判定结果有粒子特效

### UI 界面
- 现代科技感设计
- 霓虹灯风格配色
- 流畅的动画过渡

## 音效系统

### 按键音效
- 每个按键有不同的音高
- 形成简单的音乐旋律
- 判定结果有不同的音效反馈

### 背景音乐
- 多首不同风格的 BGM
- 根据节奏自动播放
- 可调整音量或静音

### 音效反馈
- PERFECT: 清脆的高音
- GREAT: 中音
- GOOD: 低音
- BAD/MISS: 闷响或警告音

## 操作技巧

### 手位建议
```
左手区          空格          右手区
┌───┬───┐                    ┌───┬───┐
│ ← │ ↓ │       [SPACE]      │ ↑ │ → │
└───┴───┘                    └───┴───┘
小指  无名指      双手拇指       食指  中指
```

### 进阶技巧
1. **预判**: 提前看后面的音符序列
2. **节奏感**: 跟着背景音乐的节拍按键
3. **肌肉记忆**: 熟练后形成条件反射
4. **连击保持**: 宁可牺牲准确度也要保持连击
5. **放松**: 紧张会影响反应速度

## 难度曲线

### 简单 (EASY)
- BPM: 80-100
- 单音符为主
- 间隔较长 (>1秒)
- 适合初学者

### 普通 (NORMAL)
- BPM: 100-130
- 偶尔出现双音符
- 间隔中等 (0.5-1秒)
- 适合有一定基础的玩家

### 困难 (HARD)
- BPM: 130-160
- 常见双音符和短连击
- 间隔较短 (0.3-0.5秒)
- 需要较好反应速度

### 专家 (EXPERT)
- BPM: 160-200
- 频繁出现组合音符
- 间隔很短 (0.2-0.3秒)
- 需要精准操作

### 大师 (MASTER)
- BPM: 200+
- 复杂的音符组合
- 超高密度
- 极限挑战
