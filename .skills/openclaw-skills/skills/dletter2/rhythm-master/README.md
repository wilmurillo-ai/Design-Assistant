# 🎵 节奏大师 - Rhythm Master

一款类似 QQ炫舞 的键盘节奏游戏 Skill！

## 🎮 游戏简介

《节奏大师》是一款基于网页的键盘节奏游戏，玩家需要在正确的时机按下方向键（↑↓←→）和空格键，跟随音乐节奏击打落下的音符。

### 核心特色

- 🎯 **5键操作**：← ↓ SPACE ↑ → 简单易上手
- 🌟 **精准判定**：PERFECT/GREAT/GOOD/BAD/MISS 五级判定
- 🔥 **连击系统**：连击越高，得分倍数越高（最高 2.5x）
- 📊 **多难度等级**：简单/普通/困难/专家 四个难度
- 🏆 **排行榜系统**：本地存储 + 可扩展联网排行榜
- 🎨 **霓虹科技风**：炫酷的视觉特效和动画
- 📱 **响应式设计**：支持桌面端和移动端

## 🎯 操作说明

### 按键布局
```
左手区          空格          右手区
┌───┬───┐                    ┌───┬───┐
│ ← │ ↓ │       [SPACE]      │ ↑ │ → │
└───┴───┘                    └───┴───┘
小指  无名指      双手拇指       食指  中指
```

### 游戏流程
1. 选择难度等级
2. 3秒倒计时准备
3. 音符从上方落下
4. 音符到达判定线时按下对应按键
5. 根据准确度获得不同判定和分数
6. 保持连击获得更高分数
7. 生命归零或时间结束游戏结束

## 📊 判定系统

| 判定 | 时间窗口 | 得分 | 效果 |
|------|----------|------|------|
| 🌟 PERFECT | ±50ms | 100分 | 完美命中 |
| ✨ GREAT | ±100ms | 80分 | 很好 |
| 👍 GOOD | ±150ms | 50分 | 不错 |
| 😕 BAD | ±200ms | 20分 | 较差，断连击 |
| ❌ MISS | >200ms | 0分 | 错过，扣生命 |

## 🌐 联网排行榜

### 已实现的本地排行榜

### 扩展为联网排行榜

#### 方案一：Firebase（推荐，免费）
```javascript
// 在 rhythm-game.html 中添加 Firebase 配置
const firebaseConfig = {
  // 你的 Firebase 配置
};
firebase.initializeApp(firebaseConfig);

// 保存分数
firebase.database().ref('leaderboard').push(scoreData);

// 获取排行榜
firebase.database().ref('leaderboard')
  .orderByChild('score')
  .limitToLast(20)
  .once('value', callback);
```

## 💡 操作技巧

### 新手建议
1. **从简单难度开始** - 熟悉节奏和按键
2. **保持手位固定** - 不要移动手的位置
3. **眼睛看判定线** - 余光看音符
4. **宁可GOOD不断连** - 连击加成非常重要

### 进阶技巧
1. **预判音符** - 提前看后面的音符序列
2. **听节奏** - 跟着背景音乐节拍按键
3. **肌肉记忆** - 熟练后形成条件反射
4. **放松心态** - 紧张会影响反应速度
