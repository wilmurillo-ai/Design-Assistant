# Agent Development Toolkit

完整的 AI Agent 开发工具包，帮助你快速构建、自动化和变现 AI Agent。

## 快速开始

### 安装

```bash
# 安装 ClawHub CLI
npm install -g clawhub

# 登录
clawhub login

# 安装工具包
clawhub install agent-dev-toolkit
```

### 使用示例

```bash
# 创建新的 agent
openclaw agent create my-agent --toolkit agent-dev-toolkit

# 使用浏览器自动化
openclaw browser navigate "https://example.com"

# 创建钱包
openclaw wallet create --chain base

# 生成文档
openclaw docs generate --skill my-skill
```

## 包含技能

1. **Agent Builder** - 设计和生成 agent
2. **Agent Browser Core** - 浏览器自动化
3. **Agent Wallet** - 加密钱包管理
4. **Agent Development** - 开发最佳实践
5. **Agent Docs** - AI 优化的文档编写
