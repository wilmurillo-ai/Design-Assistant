# 📂 文件结构说明

## 根目录文件

| 文件名 | 类型 | 说明 |
|--------|------|------|
| `.gitignore` | 配置文件 | Git 忽略文件配置，忽略日志、临时文件、node_modules 等 |
| `README.md` | 文档 | 项目说明文档，包含功能介绍、使用方法、安装指南等 |
| `SKILL.md` | 文档 | OpenClaw 技能说明文档（如果作为技能使用） |
| `package.json` | 配置文件 | Node.js 项目配置文件，包含依赖、脚本、项目元信息 |
| `package-lock.json` | 配置文件 | 依赖版本锁定文件，确保依赖安装一致性 |
| `FILE_STRUCTURE.md` | 文档 | 本文档，项目文件结构说明 |

## 目录结构

### 📁 scripts/ 脚本目录
核心功能实现代码：
- `md_to_png.js` - Markdown 转图片主脚本，核心功能实现
- `package.json` - 脚本独立依赖配置（如果需要单独部署）
- `package-lock.json` - 脚本依赖版本锁定文件

### 📁 .log/ 日志目录
自动生成的日志文件存储目录：
- 存储每次转换的日志信息
- 自动清理旧日志，不会占用过多空间
- 默认忽略该目录下的所有文件

---

## 说明
项目结构简洁，核心代码都在 `scripts/` 目录下，配置文件集中在根目录，方便维护和部署。
