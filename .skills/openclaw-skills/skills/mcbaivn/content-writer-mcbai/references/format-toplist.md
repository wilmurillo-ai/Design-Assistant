# Toplist Format — Detailed Instructions

## What is a Toplist?

A curated, numbered list of items (companies, tools, trends, stats) with specific data for each. The most popular LinkedIn content format for information-dense posts.

## Structure

```
HOOK (1-2 lines)
Bold claim + specific number that stops the scroll.
Example: "7 AI startups just raised $500M+ in March. Here's what they're building."

CONTEXT (2-3 lines)
Why this matters right now. Set the stage.
Example: "AI funding in Q1 2026 is outpacing all of 2024 combined. The market isn't just recovering - it's transforming."

NUMBERED LIST (4-8 items)
Each item follows this pattern:
{number}. {Company/Tool/Thing Name}
→ {Key metric or detail}
→ {Why it matters} (optional, for long posts)

Example:
1. Anthropic
→ $2B Series D at $61B valuation
→ Enterprise API revenue grew 5x in 6 months

2. Perplexity
→ $500M raise, 100M monthly searches
→ Challenging Google's core business

TAKEAWAY (2-3 lines)
What pattern emerges from this list? What should the reader conclude?
Example: "The pattern is clear - AI infrastructure is the new cloud. The companies winning aren't building chatbots. They're building platforms."

CTA (1-2 lines)
Engagement question OR soft MCB AI mention.
Example: "Which of these would you bet on? Drop your pick below."
Example with brand: "We're tracking all of these at MCB AI. Which one surprises you most?"
```

## Constraints

- Every item MUST have specific data (number, dollar amount, percentage)
- Use numbered list (1, 2, 3...) with → for sub-details
- No generic filler — if you can't find data for an item, don't include it
- ZERO asterisks (*) in output
- No em dashes (—), no URLs, plain text only
- For short posts: 4-5 items, minimal sub-details
- For medium posts: 5-7 items, one sub-detail each
- For long posts: 6-8 items, multiple sub-details, deeper analysis

## Example Output (Medium Length)

```
7 AI startups that just raised $500M+ this month.

The AI funding wave isn't slowing down - it's accelerating. March 2026 saw more mega-rounds than all of Q4 2025.

1. Anthropic
→ $2B Series D at $61B valuation

2. Perplexity
→ $500M at $9B, 100M monthly searches

3. Glean
→ $260M Series E, enterprise AI search

4. Harvey
→ $200M for AI legal platform

5. Runway
→ $180M, video AI hitting mainstream

The pattern? Infrastructure and vertical AI are winning. Generic chatbot wrappers are out.

Which of these are you watching closest?
```
