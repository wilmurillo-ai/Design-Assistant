# Social Content Planner - 社交媒体内容排期器

## 快速开始

```bash
cd scripts
python planner.py --help
```

## 目录结构

```
social-content-planner/
├── SKILL.md
├── README.md
├── scripts/
│   └── planner.py
└── output/
```

## 主要功能

- 热点追踪（调用搜索引擎）
- 选题策划
- 内容生成（多平台模板）
- 日历排期
