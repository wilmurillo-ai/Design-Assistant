# 投放数据分析技能

基于《数据分析基础概念和逻辑v3.md》文档开发的标准化投放数据分析技能。

## 🚀 快速开始

### 安装依赖
```bash
pip install -r requirements.txt
```

### 基本使用
```bash
python main.py --date-range "2026-01-01:2026-01-31" --data-dir "/Users/zhouhao/Documents/投放数据"
```

### 高级使用
```bash
# 指定特定指标
python main.py --date-range "2026-01-01:2026-01-31" \
               --metrics "ROI,观看成本,订单成本,加购成本" \
               --output-format "html,csv"

# 自定义输出目录
python main.py --date-range "2026-01-01:2026-01-31" \
               --output-dir "/Users/zhouhao/Desktop/自定义报告"
```

## 📊 功能特性

### 核心功能
- ✅ **自动文件识别** - 自动识别超级直播、淘宝直播、财务三类数据文件
- ✅ **编码自动处理** - 支持GBK、UTF-8等多种编码格式
- ✅ **字段智能映射** - 基于文档的字段定义和计算公式
- ✅ **跨报表关联分析** - 支持多数据源关联计算
- ✅ **数据质量检查** - 自动检测数据质量问题

### 输出内容
- 📈 HTML可视化报告
- 📊 CSV格式数据汇总
- 🔍 数据质量报告
- 💡 优化建议

## 🔧 配置说明

### 数据目录配置
默认数据目录：`/Users/zhouhao/Documents/投放数据`

### 文件识别规则
- **超级直播文件**: 包含"超级直播"关键词
- **淘宝直播文件**: 包含"淘宝直播"关键词  
- **财务文件**: 包含"财务"或"财务报表"关键词

### 字段映射
基于《数据分析基础概念和逻辑v3.md》文档中的字段定义，支持自动字段识别和映射。

## 🛠️ 开发说明

```
主程序 (main.py)
```

1. 在 `config.py` 中添加新的字段映射
2. 在 `calculator.py` 中添加新的指标计算公式
3. 在 `reporter.py` 中添加新的报告格式

## ❓ 常见问题

### Q: 文件编码识别错误怎么办？
A: 系统会自动尝试多种编码格式，如仍无法识别，请检查文件格式。

### Q: 字段名称不匹配怎么办？
A: 在 `config.py` 中的 `FIELD_MAPPING` 配置中添加字段映射。

### Q: 如何添加新的指标计算？
A: 在 `calculator.py` 中添加新的计算方法，并在 `config.py` 中配置相关字段。
