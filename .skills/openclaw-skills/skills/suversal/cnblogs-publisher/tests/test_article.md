# 测试文章

这是一篇通过 OpenClaw 自动保存到博客园草稿箱的测试文章。

## 功能介绍

这个 skill 可以通过 MetaWeblog API 管理博客园文章：

- ✅ 保存草稿
- 📋 获取文章列表
- ✏️ 更新草稿
- 🚀 发布文章

## 技术栈

- Python 3
- xmlrpc.client
- MetaWeblog API

## 使用方法

```bash
# 保存草稿
python save_draft.py "文章标题" "content.md"

# 获取文章列表
python list_drafts.py
```
