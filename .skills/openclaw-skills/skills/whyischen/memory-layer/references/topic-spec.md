# Topic 层规范

## 位置

`memory/topics/{领域}-{主题}.md`

---

## 格式规范

```markdown
# 主题名称

> 📌 最后更新：2026-04-02 | 领域：investment | 版本：v1.0

---

## 元数据
- created: 2026-03-21
- importance: 0.9
- references: [memory/transcripts/2026-04/2026-04-02.log#L15]
- version: 1.0

---

## 核心事实（不可推导，必须存储）
- **配置版本**: v1.0
- **执行时间**: 工作日 18:52
- **主要标的**: ETF1, ETF2, ETF3

---

## 决策规则
| 标的 | 🟡 黄色 | 🔴 红色 |
|------|--------|--------|
| 标的 A | >15% | >25% |
| 标的 B | >8% | >15% |

---

## 动态状态（每次执行后更新）
| 字段 | 值 | 更新 |
|------|-----|------|
| 最后检查 | 2026-04-02 18:52 | 2026-04-02 |
| 状态 | 🟢 正常 | 2026-04-02 |

---

## 待解决问题（矛盾标记）
- [ ] ⚠️ 检测到矛盾 (2026-03-25)

---

*相关 Transcripts: memory/transcripts/2026-04/*.log*
```

---

## 标准章节

| 章节 | 说明 | 必填 |
|------|------|------|
| 元数据 | 元数据（创建时间/重要性/引用） | 是 |
| 核心事实 | 不可推导的长期知识 | 是 |
| 决策规则 | 阈值/条件/流程 | 可选 |
| 动态状态 | 执行后更新的状态 | 可选 |
| 待解决问题 | 矛盾标记/待决事项 | 可选 |

---

## 约束

| 规则 | 值 | 违规处理 |
|------|-----|----------|
| 文件大小 | ≤50KB | 建议拆分 |
| 版本控制 | 可选 | 关键 Topic 启用 |
| 引用 | 必填 | 每次更新时添加 |
| 矛盾标记 | 必填 | 检测到时标记 |

---

## 命名约定

```
memory/topics/{领域}-{主题}.md

领域前缀：
- investment-  (投资)
- project-     (项目)
- asset-       (资产)
- knowledge-   (知识)
- health-      (健康)
- social-      (社交)

示例：
- memory/topics/investment-dca.md
- memory/topics/project-tool.md
- memory/topics/asset-property.md
```

---

## 不存储原则

以下内容**不应**存储在 Topic 中：

- ❌ 每日价格数据（实时从 API 获取）
- ❌ 完整对话日志（存储在 Transcript）
- ❌ 新闻摘要（从搜索获取）
- ❌ 可从当前状态推导的信息
