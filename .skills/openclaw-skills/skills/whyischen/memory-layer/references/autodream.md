# autoDream 夜间整理

## 执行时间

- **时间**：每日 23:00（可配置）
- **触发**：OpenClaw Cron
- **通知**：默认关闭（`notifyOnComplete: false`）

---

## 首次运行建议

**启用 Cron 前，先手动测试**：

```bash
# 1. 干跑模式（仅查看，不执行）
# 检查当日 Transcripts，确认逻辑正确

# 2. 手动执行一次
# 观察行为，确认无意外修改

# 3. 确认无误后启用 Cron
openclaw cron add "0 23 * * *" "memory-system auto-dream"
```

**默认配置已禁用通知**，避免首次运行时消息轰炸。

## 去重算法

### 步骤

1. **标准化文本**：移除空白、标点、大小写
2. **计算语义哈希**：SimHash 算法
3. **检查重复**：哈希碰撞验证
4. **保留更高重要性**：删除较低重要性重复项

### 相似度阈值

| 阈值 | 处理方式 |
|------|----------|
| >0.95 | 视为重复，删除 |
| 0.8-0.95 | 视为相似，建议合并 |
| <0.8 | 保留 |

## 矛盾处理

### 检测逻辑

1. 按 `topic:key` 分组事实
2. 检查组内值是否冲突
3. 标记矛盾（不强制解决）

### 标记格式

```markdown
## 待解决问题

- [ ] ⚠️ 矛盾：VIX >25 vs >30
```

### 通知方式

- 飞书消息（配置 `notifyChannel: "feishu"`）
- 写入报告文件（`memory/heartbeat/auto-dream-report.md`）
