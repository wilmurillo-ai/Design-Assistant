`MEMORY.md`（工作区根目录）

## 格式规范

```markdown
# MEMORY.md - OpenClaw 记忆索引

> 📌 规则：仅指针，≤150 字符/行，≤25KB 总计

---

## 元数据

- last_dream: 2026-04-02T23:00:00+08:00
- total_topics: 12

## Topics
| 领域 | 主题 | 路径 | 更新 | 摘要 (≤50 字符) | 标签 | 重要性 | 热度 |
```

---

## 字段规范

### Topics 表格

| 字段 | 说明 | 约束 |
|------|------|------|
| 领域 | 领域分类 | 必填 |
| 主题 | 主题名称 | 必填，≤20 字符 |
| 路径 | Topic 文件路径 | 必填，相对路径 |
| 更新 | 最后更新日期 | 必填，YYYY-MM-DD |
| 摘要 | 内容摘要 | 必填，≤50 字符 |
| 标签 | 逗号分隔标签 | 可选 |
| 重要性 | 0.0-1.0 分数 | 自动计算 |
| 热度 | 🔥 表示高频 | 可选 |

### 元数据字段

| 字段 | 说明 |
|------|------|
| version | Index 格式版本 |
| last_dream | 上次 autoDream 执行时间 |
| total_topics | Topic 总数 |

---

## 约束

| 规则 | 值 | 违规处理 |
|------|-----|----------|
| 行长 | ≤150 字符 | 自动截断摘要 |
| 文件大小 | ≤25KB | 归档最旧 20% 行 |
| 更新顺序 | 先 Topic，后 Index | 写入 API 强制执行 |
| LRU 容量 | Top 5 | 自动淘汰最旧 |
| 重要性范围 | 0.0-1.0 | 自动归一化 |

---

## 归档机制

当 Index 超过 25KB 时：

1. 计算最旧 20% 行
2. 移动到 `memory/index-archive.md`
3. 更新元数据统计

### 归档格式

```markdown
# MEMORY.md - 历史归档

> 归档时间：2026-04-02

## 已归档内容
| 领域 | 主题 | 路径 | 更新 | 摘要 | 原因 |
|------|------|------|------|------|------|
| 项目 | 旧项目 A | memory/topics/old-project-a.md | 2026-02-15 | 已结项 | 过期 |
```
