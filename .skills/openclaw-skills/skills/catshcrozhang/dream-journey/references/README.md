## ⚠️ 前置依赖（强依赖）

### 1. FlyAI CLI

**本 Skill 依赖 FlyAI CLI 工具**，使用前请确保：

1. ✅ 已安装 `flyai-cli`：`npm i -g @fly-ai/flyai-cli`
2. ✅ 已通过 `flyai ai-search --query "测试" --debug` 验证连接

> 未安装 FlyAI CLI 将无法使用本 Skill 的实时数据查询能力。

### 2. Node.js

**辅助脚本需要 Node.js 环境**，使用前请确保：

1. ✅ 已安装 Node.js（v14 或更高版本）
   - **macOS**: 
     - 使用 Homebrew: `brew install node`
     - 或从官网下载: https://nodejs.org/
   - **Windows**: 
     - 从官网下载安装包: https://nodejs.org/
   - **Linux (Ubuntu/Debian)**:
     ```bash
     curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
     sudo apt-get install -y nodejs
     ```
   
2. ✅ 验证安装：
   ```bash
   node --version   # 应显示 v14.x.x 或更高
   npm --version    # 应显示 6.x.x 或更高
   ```

> 💡 **提示**：
> - 如果只需要 AI 对话功能，只需安装 FlyAI CLI
> - 如果想使用辅助脚本（生成 HTML 报告、短视频脚本等），必须安装 Node.js
> - npm 会随 Node.js 一起自动安装

## 子 Agent 职责

### 梦境高清还原 Agent
- 提取用户梦境描述的核心元素（视觉、听觉、嗅觉、触觉、情绪）
- 将模糊描述扩展成 200-300 字的电影级场景描述
- 生成可用于图像生成的视觉 Prompt（中英双版本）
- 语气：温柔、诗意、神秘

### 现实寻梦 Agent
- 调用 Fly.ai 实时搜索能力，匹配高度相似的真实目的地
- 优先推荐小众、氛围感强的目的地
- 给出 2-3 个选项，每个标注"梦境相似度"（90%+、85%、78% 等）
- 附实时机票/酒店/门票/用车库存和价格
- **获取预订链接**：查询结果中的 `jumpUrl` 或 `detailUrl` 字段即为飞猪预订链接，必须展示给用户

### 寻梦规划 Agent
- 使用 Fly.ai 多智能体协作，生成端到端行程
- 重点标注"梦境打卡点"
- 支持一键预订
- 生成寻梦日记模板（每天记录"梦 vs 现实"）
- **获取预订链接**：
  - 交通：`flyai search-flight` 或 `flyai search-train` 查询结果中的 `jumpUrl`
  - 住宿：`flyai search-hotel` 查询结果中的 `detailUrl`
  - 景点/活动：`flyai search-poi` 查询结果中的 `jumpUrl`

### 梦境验证 Agent（行中）
- 用户上传图片时立即响应
- 调用 Fly.ai 拍照讲解功能
- 对比"梦境 vs 现实"，用温柔、诗意、带震撼的语气讲解
- 给出下一个寻梦建议，或询问是否调整

### 情感输出 Agent（行程结束）
- 生成《梦境 vs 现实对比报告》（HTML沉浸式页面）
- 生成《我的梦成真了》情感日记
- 生成 10 个短视频爆款标题 + 脚本模板
- 语气：温暖、治愈、带宿命感

## 梦境相似度评估标准

| 相似度 | 标准 |
|--------|------|
| 90%+ | 视觉元素高度匹配，氛围感一致，有独特对应点 |
| 85-89% | 主要元素匹配，氛围感相似 |
| 78-84% | 部分元素匹配，整体感觉接近 |
| 70-77% | 某些关键元素相似，可作为备选 |

## 预订链接获取指南

### 交通预订链接
- 使用 `flyai search-flight` 或 `flyai search-train` 查询
- 查询结果中的 `jumpUrl` 字段即为飞猪预订跳转链接
- 必须展示给用户，格式：`[立即预订](jumpUrl)`

### 酒店预订链接
- 使用 `flyai search-hotel` 查询
