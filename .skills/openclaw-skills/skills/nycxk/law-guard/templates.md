# 常用文书模板包（可直接改写）

使用原则：

- 这是业务模板，不是官方统一文本；正式使用前请按具体事实和地区规则校对。
- 涉及争议、仲裁、诉讼、重大金额时，建议律师复核后再发出。
- 方括号 `[]` 为待替换字段。

---

## 模板 1：试用期不符合录用条件解除通知

**适用场景**：试用期员工经考核不符合录用条件，拟依法解除劳动合同。

**模板正文**

```text
解除劳动合同通知书

致：[员工姓名]（身份证号：[号码]）

你于[入职日期]入职我司，岗位为[岗位名称]，约定试用期为[试用期时长]。
经我司根据录用条件及岗位要求进行考核（见附件：录用条件、考核记录、沟通纪要等），你目前不符合录用条件。

依据《劳动合同法》及双方劳动合同约定，我司决定于[生效日期]与你解除劳动合同。

请你于[日期时间]前完成工作交接，并到[地点/部门]办理离职手续。
我司将依法为你结清截至解除日的工资及其他应付款项。

特此通知。

[公司名称]（盖章）
[日期]
```

**发送要点**

- 同步附上证据目录：录用条件、考核表、辅导记录、沟通留痕。
- 建议使用可留痕方式送达（签收、企业邮箱、系统回执等）。

## 模板 2：绩效改进通知（PIP）

**适用场景**：员工绩效不达标，需要先辅导改进再决定后续处理。

**模板正文**

```text
绩效改进通知书

致：[员工姓名]

根据你在[考核周期]的绩效评估结果，你在以下方面未达到岗位要求：
1. [指标1：现状与目标差距]
2. [指标2：现状与目标差距]

公司决定为你设置绩效改进期，自[开始日期]至[结束日期]，改进目标如下：
1. [可量化目标1]
2. [可量化目标2]

公司将提供如下支持：
1. [培训/辅导资源]
2. [复盘频率与负责人]

请你配合执行并于每周[时间]提交进展。改进期结束后，公司将依据结果进行评估并作出后续安排。

[公司名称]
[日期]
```

**发送要点**

- 目标要可量化、可验证，避免抽象措辞。
- 留存培训与复盘记录，形成证据闭环。

## 模板 3：协商解除劳动合同通知（邀约版）

**适用场景**：希望通过协商解除降低争议风险。

**模板正文**

```text
协商解除劳动合同沟通函

致：[员工姓名]

基于公司业务调整及岗位安排变化，现拟与你协商解除劳动合同。
公司建议解除日期为[日期]，并拟支付协商补偿金人民币[金额]元（税前/税后注明），具体以双方签署的《协商解除协议》为准。

如你同意协商，请于[回复期限]前回复，并与人力资源部预约办理手续。
在协商期间，你的现有劳动报酬及工作安排不受影响（另有约定除外）。

[公司名称]
[日期]
```

**发送要点**

- 明确“协商”性质，避免形成单方解除表述。
- 补偿口径、税费承担、支付时间写清楚。

## 模板 5：图片/字体侵权投诉回复函（首次沟通版）

**适用场景**：收到权利人或代理机构侵权投诉，需先稳态沟通。

**模板正文**

```text
关于[作品名称/链接]投诉的回复函

致：[对方名称]

我方已收到贵方于[日期]发送的关于[作品名称/链接]的权利主张通知。
我方高度重视知识产权保护，已于[时间]对相关内容采取[下架/屏蔽/替换]等临时措施。

为便于核实，请贵方补充以下材料：
1. 权利主体身份证明/主体资格证明；
2. 权属证明及授权链路材料；
3. 涉案内容对应关系说明（原作品与被诉页面比对）。

我方将在收到完整材料后尽快完成核验，并在合理期限内反馈处理方案。

[公司名称]
[联系人/邮箱]
[日期]
```

**发送要点**

- 先止损再核验，避免直接承认侵权事实。
- 全程保留邮件与页面快照证据。

## 模板 6：合同条款修改建议邮件（谈判版）

**适用场景**：向客户/供应商发送条款修订意见，推进签约。

**模板正文**

```text
主题：关于《[合同名称]》条款优化建议

[对方称呼]您好，

为保障双方合作可持续推进，我方对当前版本提出以下优化建议（其余条款保持不变）：

1. 付款与违约责任
- 建议将“逾期付款免责”调整为“逾期按[标准]承担违约责任”；
- 逾期超过[天数]日，我方有权暂停履行并书面催告。

2. 验收与交付标准
- 建议补充可量化验收标准及默认验收机制，避免主观争议。

3. 解除与争议解决
- 建议增加补救期[天数]日；
- 争议解决建议约定为[仲裁机构/法院管辖]。

如您认可，我方可在今日内回传修订版文本供双方确认。

谢谢。
[你的姓名/公司]
```

**发送要点**

- 用“保障合作”措辞，减少对抗感。
- 每条建议都给出业务理由，提升对方接受度。

## 模板 7：商标授权使用确认函

**适用场景**：品牌合作、渠道分销、代运营等需要明确商标使用边界。

**模板正文**

```text
商标授权使用确认函

授权方：[甲方全称]
被授权方：[乙方全称]

鉴于双方合作，授权方同意被授权方在以下范围内使用商标：
1. 商标信息：[注册号/标识]
2. 使用范围：[商品/服务类别、渠道、地区]
3. 使用期限：自[开始日]至[结束日]
4. 使用方式：仅限[宣传/销售页面/包装]，不得擅自变更标识或超范围使用

被授权方应按授权方品牌规范使用，并接受必要审查。
授权期限届满或合作终止后，被授权方应立即停止使用并下架相关素材。

本确认函自双方签署/盖章之日起生效。

授权方（盖章）：_______
被授权方（盖章）：_______
日期：_______
```

## 模板 9：律师函收悉回执（简版）

**适用场景**：已收到律师函，先确认收悉并争取核实时间。

**模板正文**

```text
律师函收悉回执

致：[律所/律师姓名]

我方已于[日期]收到贵方发送的《律师函》（编号：[编号]）。
我方正就函件所涉事项进行内部核实，并将于[日期]前作出正式回复。

本回执仅确认收悉，不视为对函件事实与法律主张的认可。

[公司名称]
[联系人]
[日期]
```

**发送要点**

- 先确认收悉，避免“已读不回”被动局面。
- 明确保留立场，避免被解读为承认。

**适用场景**：业务同事向法务助手提问前，先补齐关键信息。

**模板正文**

```text
法务问题提问卡

1) 你要解决的问题是（合同/用工/IP/治理/合规）：
2) 事件当前阶段（签约前/履约中/争议中）：
3) 涉及主体（我方、对方全称）：
4) 适用地区（国家/城市）：
5) 关键金额与期限：
6) 你希望的结果（保守风控/平衡成交/强势谈判）：
7) 已有证据材料（合同、聊天、邮件、付款凭证等）：
8) 是否已收到律师函/仲裁/法院材料：
```

## 模板 11：法务助手免责声明（通用版）

**适用场景**：放在法务助手回答末尾、产品页面、SOP 文档中，用于明确服务边界与用户责任。

**模板正文**

```text
免责声明

1. 本助手提供的信息仅用于一般法律与合规知识参考，不构成正式法律意见、律师代理意见或任何诉讼/仲裁策略承诺。
2. 本助手输出基于你提供的信息生成；如事实不完整、不准确或发生变化，结论可能相应调整。
3. 本助手不替代执业律师。涉及重大交易、争议处理、行政处罚、刑事风险、跨境业务等事项，请及时咨询具备资质的专业律师。
4. 你应对基于本助手建议作出的商业或法律决策自行判断并承担相应责任。
5. 除法律另有强制规定外，因使用本助手信息导致的直接或间接损失，提供方不承担保证结果性质的责任。

如你已收到律师函、仲裁通知、法院文书或监管调查通知，请立即进行证据保全并尽快联系律师。
```

**简版（对话末尾）**

```text
以上内容仅作一般法律信息参考，不构成正式法律意见。重大事项或已进入争议程序，请尽快咨询执业律师。
```

**发送要点**

- 免责声明应放在结论之后，避免影响可读性。
- 不要使用“100%合规”“保证胜诉”等绝对化表述。
- 若已出现争议文书，需同步给出“立即找律师 + 证据保全”提醒。
