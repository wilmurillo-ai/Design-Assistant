# 使用示例

## 示例 1：项目交付报告

### 用户输入
```
生成一个项目交付报告 HTML，包含以下内容：

**标题：** XX 银行零售风控体系建设项目交付报告

**章节：**
1. 监管政策与市场趋势
   - 宏观监管环境
   - 微观合规要求
2. 应对策略建议
   - 客群分层策略
   - 风险定价策略
3. 风控系统建设
   - 决策引擎架构
   - 实施路径

**风格：** 企业级、专业、紫色渐变主题
```

### 输出
调用 ruitian-html skill，生成完整 HTML 代码，包含：
- 导航栏（紫色渐变）
- Hero Section（标题 + 副标题 + 徽章）
- 目录（可点击跳转）
- 三个章节，每章包含卡片、表格、流程图
- 页脚（项目信息）

## 示例 2：技术方案文档

### 用户输入
```
创建一个技术方案 HTML 文档：

**标题：** 智能风控平台技术方案

**内容要点：**
- 系统架构图（用流程图展示）
- 核心模块说明
- 技术选型对比表格
- 实施时间线
```

### 输出
生成 HTML，使用：
- 流程图组件展示架构
- 表格对比技术选型
- 代码块展示实施步骤
- 卡片组织各模块说明

---

## 示例 3：内部培训材料

### 用户输入
```
生成培训材料 HTML：

**主题：** 信贷风控基础知识培训

**内容：**
- 风控核心概念
- 常见风险类型（用表格）
- 风控流程（用流程图）
- 案例分析
```

### 输出
生成 HTML，使用：
- 警告框强调重点概念
- 表格列出风险类型
- 流程图展示风控流程
- 卡片组织案例内容

---

## 快速开始

1. **确定文档结构** - 标题、章节、小节
2. **提供内容要点** - 每节的核心信息
3. **指定可视化需求** - 表格/流程图/代码块
4. **调用 skill** - 生成完整 HTML 代码

## 自定义选项

| 选项 | 默认值 | 可调整 |
|------|--------|--------|
| 主题色 | 紫色渐变 | 蓝色/绿色/红色渐变 |
| 圆角 | 16px | 8px/12px/20px |
| 阴影 | 中等 | 无/轻/重 |
| 动画 | 淡入上浮 | 无/淡入/滑动 |
| 字体 | 苹方/微软雅黑 | 系统默认 |
