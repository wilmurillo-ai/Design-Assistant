# ruitian-html - 瑞天 HTML 企业文档生成器

## 🎨 设计特点

- **紫色渐变主题** - 专业、高端的视觉效果
- **卡片式布局** - 清晰的信息层级
- **响应式设计** - 手机/平板/电脑完美适配
- **离线友好** - 支持 Base64 嵌入图片
- **动画效果** - Hover 上浮、淡入出场

## 📦 文件结构

```
ruitian-html/
├── SKILL.md           # 技能说明
├── README.md          # 使用指南（本文件）
├── template.html      # HTML 模板
└── examples/
    └── usage-example.md  # 使用示例
```

## 🚀 快速使用

### 方式 1：直接生成

告诉我要生成的文档内容，我直接输出完整 HTML 代码。

**示例：**
> 生成一个项目汇报 HTML，标题是"XX 系统建设方案"，包含 3 章：背景、方案、计划

### 方式 2：使用模板

复制 `template.html`，替换 `{占位符}` 为你的内容。

### 方式 3：参考示例

查看 `examples/usage-example.md` 了解完整用法。

## 📋 核心组件

| 组件 | 用途 | 示例 |
|------|------|------|
| `.card` | 内容卡片 | 章节内容容器 |
| `.table` | 数据表格 | 对比、列表 |
| `.alert` | 提示框 | 注意、警告、信息 |
| `.code-block` | 代码块 | 流程、命令 |
| 流程图 | 步骤展示 | 实施流程、架构 |

## 🎨 颜色规范

| 颜色 | 色值 | 用途 |
|------|------|------|
| 主渐变 | `#667eea → #764ba2` | 导航、表头 |
| 主色 | `#0d6efd` | 链接、图标 |
| 成功 | `#198754` | 正向状态 |
| 警告 | `#ffc107` | 注意事项 |
| 危险 | `#dc3545` | 风险、错误 |
| 信息 | `#0dcaf0` | 提示、说明 |

## 💡 最佳实践

1. **结构清晰** - 每章用 `<section>`，标题用 `.section-title`
2. **卡片化** - 每个子主题用 `.card` 包裹
3. **图标增强** - 使用 Bootstrap Icons
4. **间距统一** - `mb-4`（24px）为标准间距
5. **响应式** - 表格用 `.table-responsive` 包裹
