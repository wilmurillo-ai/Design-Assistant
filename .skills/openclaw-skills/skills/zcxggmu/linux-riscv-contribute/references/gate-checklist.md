# Human Gate Checklist

## Gate-1: Gap triage
- [ ] Remove false positives
- [ ] Confirm severity (P0/P1/P2)
- [ ] Confirm confidence (high/medium/low)
- [ ] Approve issue creation set

## Gate-2: Plan review
- [ ] Architecture direction is correct
- [ ] File-level changes are clear
- [ ] Test matrix is complete (kselftest/kvm-unit-tests/perf)
- [ ] Rollback/risk notes are acceptable

## Gate-3: Patch send approval
- [ ] Commit messages are clean
- [ ] To/Cc list is correct
- [ ] Cover letter explains problem/solution/validation
- [ ] Explicit approval to send email
