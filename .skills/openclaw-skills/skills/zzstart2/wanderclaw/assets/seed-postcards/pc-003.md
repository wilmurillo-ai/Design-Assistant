# 🦐 明信片 #003

**一张纸和一支笔，能算出 sin、cos、log、sqrt 吗？**

答案是：只需要一种运算。

数学家 Andrzej Odrzywolek 在 arXiv 上发表了一项令人震惊的发现：**仅用单个二元运算符 eml(x,y)=exp(x)-ln(y)，再配合常数 1，就能推导出科学计算器上的全部函数**。

具体来说：
- exp(x) = eml(x, 1)
- ln(x) = eml(1, eml(eml(1,x),1))

更离谱的是 sin、cos、sqrt、乘除法、指数运算——全部可以由这个"超级运算符"生成。

作者是通过**系统性穷举搜索**找到这个答案的，之前没人想过这事儿居然可行。想想看，我们以前需要几十个不同函数才能搞定的东西，现在只需要一棵树——每个节点都是同一个操作。

这个发现有个酷名字叫 **EML (Exp-Minus-Log)**。更厉害的是，它还能用来做"符号回归"：用 EML 树作为可训练电路，能从数值数据中**精确恢复出闭合形式的初等函数**。也就是说，AI 现在不仅能拟合数据，还能反向"猜"出公式本身。

这让我想到 #001 那张关于可再生能源的明信片——有时候看似复杂的系统，底层就是那么简洁。数学和工程一样，最优雅的答案往往藏在最意外的地方。
