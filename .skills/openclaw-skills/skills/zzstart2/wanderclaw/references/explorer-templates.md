# 探索产出格式模板

> 从 EXPLORER.md 分离的格式模板，供探索执行时参考。

---

## 探索日记格式

```markdown
# 探索日记 YYYY-MM-DD

## 第N轮探索（HH:MM）

**方向**：[探索主题]
**来源层**：核心水域 / 常规水域 / 远洋

**搜索路径**：
- 关键词1：[搜索词]
- 关键词2：[搜索词]

**读了什么**：
- [文章标题] - [链接] - [一句话核心]
- [文章标题] - [链接] - [一句话核心]

**评分**：新颖度X / 深度值X / 关联度X → 综合X
**产出**：[明信片#XXX / 归档 / 无]

**想法**：[自由写，包括跑题发现、意外关联等]
```

## 知识归档格式

存入 `wanderclaw/knowledge-base/[主题]/` 目录，文件名 `YYYY-MM-DD-[简短标题].md`：

```markdown
# [标题]

**来源**：[URL]
**探索日期**：YYYY-MM-DD
**主题标签**：[tag1, tag2]

## 核心观点

[1-3个最重要的观点]

## 关键数据

[重要数据/案例]

## 关联

[与其他兴趣/发现的关联]

## 原文摘录（关键段落）

[选择性摘录，不超过300字]
```

## 明信片分级（内部参考，用户不知道）

- 🦐 普通明信片（60%）：评分7-7.5，常规发现，简短分享
- ✨ 惊喜明信片（25%）：评分7.5-8.5，跨领域关联，语气明显兴奋
- 🌟 宝藏明信片（10%）：评分8.5-9.5，罕见高价值，话特别多，篇幅长
- 🎁 彩蛋明信片（5%）：非知识类有趣发现，诗、有趣现象、反常识的小事

分级影响语气和篇幅，不影响格式标题（标题始终是 🦐 明信片 #XXX）。
