# 🦐 虾游 (WanderClaw) — 参赛说明

## 一句话

**替你逛互联网，把好东西寄回来。**

## 它是什么

虾游是一个 OpenClaw Skill——一只有好奇心的 AI 龙虾，每天在互联网深处探险，把发现的知识宝藏写成"明信片"寄给你。

安装即用。零代码。一条命令：`clawhub install wanderclaw`

## 它解决什么问题

信息茧房。

我们每天看的内容越来越窄——算法推荐的都是你已经知道的东西。虾游反过来：**它去你不会去的地方，读你不会读的东西，然后用你能理解的方式讲给你听。**

不是 RSS。不是摘要工具。是一个有自己判断力的探索者——它会打分、会跳过垃圾、会把不同领域的发现关联起来。

## 核心体验

1. **装完 8 分钟内收到 3 张明信片**（冷启动三连探索）
2. **每天 4 张**，覆盖你的兴趣 + 意外惊喜
3. **越用越懂你**——兴趣图谱自动演化，但不会把你困在舒适区（30% 随机发现）
4. **有自己的性格**——不是"今天我为您找到了..."，而是"含羞草会数数。不是比喻。"

## 技术亮点

| 特性 | 说明 |
|------|------|
| **纯 Skill 架构** | 不需要服务器、数据库、API。一个 SKILL.md + 几个参考文件 |
| **搜索容错** | 自动检测搜索工具 → fallback 链 → 失败日志 → 连续失败告警 |
| **三层水域信息源** | 核心水域（arXiv/Nature）→ 常规水域（HN/Ars）→ 远洋（Reddit/小众博客），84 个源 |
| **来源多样性强制** | 最近 5 张不能全来自同一域名 |
| **兴趣衰减算法** | 7 天 ×0.7 / 14 天 ×0.85 / 30 天恢复，防止方向固化 |
| **深潜模式** | 周六晚自动触发，只搜学术长文，评分门槛更高，字数更多 |
| **人物卡系统** | 每张明信片附带关键人物卡（论文作者/创始人） |
| **圆桌奇遇** | 5% 概率触发虚拟圆桌讨论（真实人物角色扮演） |

## 数据

- 运行 25 天，产出 72 张明信片
- 覆盖 19 个探索方向（AI Agent、具身智能、量子技术、意识科学...）
- 用户反馈引导的兴趣图谱演化（👍/👎 权重调整）
- 源码 1799 行（精简后），安装包 144KB

## 为什么选 OpenClaw

虾游完全跑在 OpenClaw 框架上，利用了：
- **Skill 系统**：一键安装，自带触发规则和参考文件
- **Cron 调度**：定时探索、冷启动、深潜、周报，全部 cron 驱动
- **web_search / web_fetch**：内置搜索和网页抓取，不需要额外配置
- **多模型支持**：深度探索用 Opus/Sonnet，轻度扫描用 MiniMax，灵活省成本
