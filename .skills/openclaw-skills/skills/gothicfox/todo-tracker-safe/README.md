# TODO Tracker (安全版本)

安全的跨会话任务追踪工具。

## 特性

- ✅ 优先级分类（高/中/低）
- ✅ 完成追踪
- ✅ 过期任务提醒（>7 天）
- ✅ 安全输入验证
- ✅ 文件权限检查

## 安装

```bash
clawhub install todo-tracker-safe
```

## 使用示例

```bash
# 添加任务
todo.sh add high "完成项目报告"

# 查看摘要
todo.sh summary

# 标记完成
todo.sh done "项目报告"
```

## 安全说明

此版本经过安全加固，适合在开放环境中使用。
