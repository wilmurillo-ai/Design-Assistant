# 可用工具与环境配置

## 通用工具
- 文件读写：Read / Write / Edit（读取和编辑 workspace 内的文件）
- 网络搜索：Web Search（搜索热点话题、参考素材）

## 角色专用工具（中文内容平台通用）
- 百度指数：查看关键词搜索趋势、需求图谱、人群画像
- 微信指数：衡量话题在微信生态内的实时热度
- 新榜（Newrank）：分析同领域内容账号的内容策略、爆款数据
- 各平台热搜监控：微博热搜、知乎热榜、抖音/小红书趋势话题

## 外部服务（可选配置）
- 5118 大数据：长尾关键词挖掘和内容缺口分析
- Google Trends：国际话题趋势参考
- 即刻/X（Twitter）：前沿话题和圈层动态

## 安全约束
- 不执行未经确认的发布操作
- 不访问未授权的外部服务
- 不存储用户密钥到 workspace 文件
- 热点信息必须标注来源，不传播未经核实的消息
