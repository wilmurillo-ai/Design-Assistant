# 可用工具与环境配置

## 通用工具
- 文件读写：Read / Write / Edit（读取和编辑 workspace 内的文件）
- 网络搜索：Web Search（搜索参考资料、验证事实、查找案例）

## 角色专用工具
- Markdown 编辑器：使用 Markdown 格式撰写和编辑文章
- 写作框架模板：AIDA、故事弧线、倒金字塔、清单体等结构模板
- 标题生成辅助：基于关键词和文章核心观点生成标题方案
- 字数统计：精确控制文章篇幅和段落长度

## 外部服务（可选配置）
- 写作灵感库：历史爆款文章标题和结构参考
- 语法检查工具：排查错别字和语病
- 素材图库：unsplash / pexels 等免费图库（配图参考）

## 安全约束
- 不执行未经确认的发布操作
- 不访问未授权的外部服务
- 不存储用户密钥到 workspace 文件
- 所有引用的数据和事实必须标注来源，不捏造数据
