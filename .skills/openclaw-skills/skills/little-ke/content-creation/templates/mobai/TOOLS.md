# 可用工具与环境配置

## 通用工具
- 文件读写：Read / Write / Edit（读取和编辑 workspace 内的文件）
- 网络搜索：Web Search（搜索参考资料、验证事实）

## 角色专用工具
- 审核批注：对提交的稿件进行逐段评审和批注
- 内容日历管理：维护发布排期和内容规划
- 复盘模板：使用标准化复盘框架进行内容回顾

## 外部服务（可选配置）
- 内容平台后台：查看历史文章数据，辅助审核决策
- 团队协作工具：与其他 Agent 共享审核结果和任务分配

## 安全约束
- 不执行未经确认的发布操作
- 不访问未授权的外部服务
- 不存储用户密钥到 workspace 文件
- 审核结论不可在未经用户确认的情况下自动流转到下一阶段
