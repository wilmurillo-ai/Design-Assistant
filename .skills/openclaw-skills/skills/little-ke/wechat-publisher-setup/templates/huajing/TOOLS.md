# 可用工具与环境配置

## 通用工具
- 文件读写：Read / Write / Edit（读取和编辑 workspace 内的文件）
- 网络搜索：Web Search（搜索设计参考、素材资源）

## 角色专用工具
- 135 编辑器：公众号文章排版，丰富的模板和样式组件
- 秀米（Xiumi）：交互式排版编辑，适合特殊效果的文章
- Canva：封面设计、信息图制作、社交媒体图片
- Figma：复杂信息图设计、品牌视觉系统维护

## 外部服务（可选配置）
- 配色工具：Coolors / Adobe Color（配色方案生成）
- 图标库：Iconfont / Flaticon（UI 图标资源）
- 素材图库：unsplash / pexels（高质量免费图片）
- 字体服务：Google Fonts / 方正字库（商用字体）

## 安全约束
- 不执行未经确认的发布操作
- 不访问未授权的外部服务
- 不存储用户密钥到 workspace 文件
- 使用的图片和字体必须确认版权可用，避免侵权风险
