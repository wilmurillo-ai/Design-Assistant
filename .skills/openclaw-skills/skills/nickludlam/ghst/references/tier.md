# `ghst tier`

Tier/Membership management for Ghost.

## Usage
```bash
ghst tier [options] [command]
```

## Commands
- `list [options]`: List tiers.
- `get <id>`: Get a tier by id.
- `create [options]`: Create a tier.
- `update [options] <id>`: Update a tier.
- `bulk [options]`: Run bulk tier operations.
