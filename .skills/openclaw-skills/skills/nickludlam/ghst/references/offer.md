# `ghst offer`

Offer management for Ghost.

## Usage
```bash
ghst offer [options] [command]
```

## Commands
- `list [options]`: List offers.
- `get <id>`: Get an offer by id.
- `create [options]`: Create an offer.
- `update [options] <id>`: Update an offer.
- `bulk [options]`: Run bulk offer operations.
