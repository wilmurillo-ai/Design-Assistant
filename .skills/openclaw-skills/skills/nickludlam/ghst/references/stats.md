# `ghst stats`

Analytics and reporting for Ghost.

## Usage
```bash
ghst stats [options] [command]
```

## Commands
- `overview [options]`: Site analytics overview.
- `web [options] [view]`: Web traffic analytics.
- `growth [options]`: Member and revenue growth.
- `posts [options]`: Top posts by views.
- `email [options] [view]`: Email analytics.
- `newsletters [options] [view]`: Newsletter analytics.
- `post [options] <id> [view]`: Individual post analytics.
