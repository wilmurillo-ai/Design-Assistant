# `ghst migrate`

Migration utilities to import content into Ghost.

## Usage
```bash
ghst migrate [options] [command]
```

## Commands
- `wordpress [options]`: Import from WordPress XML export.
- `medium [options]`: Import from Medium export zip.
- `substack [options]`: Import from Substack export zip.
- `csv [options]`: Import from canonical CSV schema.
- `json [options]`: Import Ghost JSON/ZIP export.
- `export [options]`: Export site content as Ghost backup.
