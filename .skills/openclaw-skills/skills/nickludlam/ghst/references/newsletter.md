# `ghst newsletter`

Newsletter management for Ghost.

## Usage
```bash
ghst newsletter [options] [command]
```

## Commands
- `list [options]`: List newsletters.
- `get [options] <id>`: Get a newsletter by id.
- `create [options]`: Create a newsletter.
- `update [options] <id>`: Update a newsletter.
- `bulk [options]`: Run bulk newsletter operations.
