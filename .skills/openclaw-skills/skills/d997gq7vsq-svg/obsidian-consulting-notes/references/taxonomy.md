## 三、辅助标签规则

每条笔记选取**恰好 1 个**辅助标签，优先级从高到低：

| 辅助标签 | 适用场景 | 识别关键词 |
|---|---|---|
| `#话术模板` | 话术脚本、汇报 PPT 框架、邮件/报告模板 | 话术、模板、脚本、汇报、如何开口、提问清单 |
| `#实战案例` | 真实项目案例、客户案例片段 | 案例、项目背景、某公司、实战、复盘 |
| `#可复用工具` | 可跨项目直接使用的工具、表格、清单 | 工具、清单、表格、检查表、问卷、评估表 |
| `#理论框架` | 方法论、框架模型、理论介绍 | 框架、模型、理论、方法论、原理 |

## 六、命名前缀快速判断逻辑

```
if 含"诊断/根因/痛点/问题/为什么" → 前缀「诊断-」→ 01_问题诊断库
elif 含"方案/工具/步骤/落地/如何做/怎么做" → 前缀「方案-」→ 02_解决方案库
elif 含"模板/话术/报告/汇报/交付/示范" → 前缀「交付-」→ 03_交付输出库
else → 临时参考（不打标）
```
