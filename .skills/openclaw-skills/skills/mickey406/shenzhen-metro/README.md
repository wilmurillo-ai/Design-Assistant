# 🚌 深圳地铁查询 Skill

**一句话介绍**：查询深圳地铁线路、站点、票价、首末班车、换乘方案、实时客流的神器

---

## 🎯 适用场景

- 查某条地铁线有哪些站点
- 查两个站之间的票价
- 查首班车/末班车时间
- 查换乘怎么走
- 看哪个站点人多不多

---

## 📖 使用示例

```
"深圳地铁1号线有哪些站点？"
"从深圳北站到世界之窗多少钱？"
"4号线首班车几点？"
"福田站人多吗？"
"从机场东怎么换乘到罗湖？"
```

---

## 📊 数据来源

- **官网**：szmc.net（票价、线路）
- **高德/百度地图**（换乘、站点拥挤度）
- **深圳地铁微博**（客流数据）
- **深圳市政府数据开放平台**（官方站点数据）
