# Reviewer Response Letter Templates

## Structure of a Point-by-Point Response Letter

```
Dear Editor and Reviewers,

Thank you for your constructive feedback on our manuscript "[TITLE]" 
(Manuscript ID: [ID]). We have carefully addressed all comments and 
revised the manuscript accordingly. Below is our point-by-point response.

---

## Response to Reviewer 1

**Comment 1.1:** [Quote reviewer's exact comment]

**Response:** [Your response]

**Changes made:** [Describe specific changes, with line/section references]

---

[Repeat for each comment]

We hope the revised manuscript now meets the standards for publication.

Sincerely,
[Authors]
```

## Response Tone Guidelines

| Situation | Tone |
|-----------|------|
| Agree with reviewer | "We thank the reviewer for this insightful observation. We have revised..." |
| Partially agree | "We appreciate this comment. While we agree that..., we respectfully note that... We have addressed this by..." |
| Respectfully disagree | "We thank the reviewer for raising this point. However, we respectfully differ because [evidence/citation]. We have added a clarification in Section X to address this concern." |
| Clarification needed | "We apologize for the lack of clarity. We have rewritten Section X to clearly state..." |

## Common Reviewer Concerns & Response Patterns

### Sample Size / Statistical Power
> "The sample size seems insufficient."

Response template:
```
We acknowledge this concern. Our sample size of N=[X] was determined by 
power analysis (β=0.8, α=0.05), yielding sufficient power to detect 
the expected effect size of [Y] (see revised Methods, Section 2.3). 
We have added explicit power analysis details to the manuscript.
```

### Missing Related Work
> "The authors fail to discuss [paper/topic]."

Response template:
```
Thank you for bringing [topic] to our attention. We have added a 
discussion of [citation] in Section [X] (page Y, lines Z-Z), explaining 
how our work relates to and extends this prior work.
```

### Unclear Methodology
> "The methodology is not sufficiently described."

Response template:
```
We apologize for the insufficient detail. We have substantially expanded 
Section [X] to include [specific additions]. The revised section now 
provides complete reproducibility information including [details].
```

### Novelty Question
> "The contribution over existing work is unclear."

Response template:
```
We appreciate this feedback and have restructured our Introduction and 
Conclusion to more clearly articulate our contributions. Specifically, 
our work advances the state of the art by: (1)..., (2)..., (3)... 
This is distinct from [prior work] because...
```

## AI Prompt for Generating Responses

When the user provides a reviewer comment and asks for help responding, use this prompt structure:

```
Context: Academic paper titled "[TITLE]" in field "[FIELD]"
Reviewer comment: "[COMMENT]"
Our position: [agree/partially agree/disagree]
Key points to address: [bullet points]

Generate a professional, concise point-by-point response that:
1. Acknowledges the reviewer's concern respectfully
2. Clearly explains our position with evidence
3. Describes specific changes made to the manuscript
4. Is factual and avoids defensive language
```
