# Output Format Specification for amz-hot-keywords\n\nThe skill can emit results in **CSV** (default) or **JSON** formats. Both contain the same fields, ordered as described below.\n\n## CSV Output\n- **Filename pattern**: `amz123_hotwords_<keyword>_<YYYYmmdd_HHMMSS>.csv`\n- **Header row** (exact order, case‑sensitive):\n  ```\n  search_term,current_rank,last_rank,trend\n  ```\n- **Rows**: one per scraped keyword. Fields:\n  - `search_term` – The exact keyword string as shown on AMZ123.\n  - `current_rank` – Integer rank for the current week (1 = most popular).\n  - `last_rank` – Integer rank for the previous week; may be empty if the term is new.\n  - `trend` – One of `up`, `down`, `flat`, or `new` indicating the direction of change.\n\n### Example CSV\n```csv\nsearch_term,current_rank,last_rank,trend\n"dog bed",5,8,up\n"cat toy",12,10,down\n"yoga mat",1,,new\n```\n\n## JSON Output\n- **Filename pattern**: `amz123_hotwords_<keyword>_<YYYYmmdd_HHMMSS>.json`\n- **Structure**: a JSON array of objects, each object matching the CSV columns.\n\n### Example JSON\n```json\n[\n  {\n    "search_term": "dog bed",\n    "current_rank": 5,\n    "last_rank": 8,\n    "trend": "up"\n  },\n  {\n    "search_term": "cat toy",\n    "current_rank": 12,\n    "last_rank": 10,\n    "trend": "down"\n  },\n  {\n    "search_term": "yoga mat",\n    "current_rank": 1,\n    "last_rank": null,\n    "trend": "new"\n  }\n]\n```\n\n## Validation Rules\n- `current_rank` must be a positive integer.\n- `last_rank` is either a positive integer or `null`/empty.\n- `trend` must be one of the four allowed strings.\n- No duplicate `search_term` entries; if duplicates appear, keep the one with the better (lower) `current_rank`.\n\nThese rules are enforced by the script before writing the file; any violation aborts the run with an error message.\n\n---
