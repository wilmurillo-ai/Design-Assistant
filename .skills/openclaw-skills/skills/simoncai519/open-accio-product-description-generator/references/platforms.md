# Platform‑Specific Requirements\n\n## Amazon\n- **Title**: 150‑200 characters, primary keyword at the beginning.\n- **Bullet Points**: 5‑7, each 200‑250 characters, focus on benefits.\n- **Description**: 2000‑3000 characters, narrative flow with hook, features→benefits, social proof, FAQ, CTA.\n- **Backend Keywords**: 250 bytes, comma‑separated, no duplicates.\n- **Style**: Professional, data‑driven, concise.\n\n## Shopify\n- **Title**: ≤70 characters, SEO‑friendly.\n- **Description**: 300‑500 words, HTML allowed, brand voice consistent.\n- **Meta Description**: ≤155 characters, includes primary keyword.\n- **Handle/URL**: ≤75 characters, kebab‑case, keyword‑rich.\n- **Style**: Conversational, lifestyle‑oriented, visual cues.\n\n## eBay\n- **Title**: ≤80 characters, key specs and brand upfront.\n- **Description**: 500‑1000 words, HTML formatting, includes condition, warranty, shipping info.\n- **Item Specifics**: Fill all applicable fields (size, material, model).\n- **Style**: Direct, urgency‑focused, includes auction‑style calls‑to‑action.\n\n## Etsy\n- **Title**: ≤140 characters, front‑load primary keyword(s).\n- **Description**: 500+ words, storytelling about handmade process, materials, use‑case.\n- **Tags**: 13 tags, max 20 characters each, keyword‑rich.\n- **Materials**: Accurate list of all materials used.\n- **Style**: Personal, artisanal, emotional connection.\n\n## Custom/Other Platforms\n- Define `titleLength`, `descriptionLength`, `tagsCount`, and `tone` in the skill invocation.\n- Follow the generic template: Hook → Problem → Solution → Features→Benefits → Social Proof → FAQ → CTA.\n\n---\n\n*Adhere to each marketplace’s guidelines to avoid listing suppression and maximize conversion.*\n
