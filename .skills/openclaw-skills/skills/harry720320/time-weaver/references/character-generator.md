# 人物生成器

> 当新人物出现时，自动生成完整档案

---

## 生成模板

每遇到一个新人物，生成以下档案：

```markdown
## [人物名]

### 基本信息
- **年龄**：实际年龄 + 外貌年龄
- **性别**：
- **身份**：社会身份 + 家族背景
- **外貌**：关键特征（身高、体型、显著特征）

### 性格
- **表层性格**：外人眼中的形象
- **真实性格**：内心真实想法
- **性格关键词**：3-5个核心词汇

### 能力
- **技能**：擅长的技能或专业
- **武艺**：如有（拳脚/兵器/内功等）
- **其他**：特殊能力或知识

### 背景
- **来历**：如何来到主角身边
- **过去**：关键的人生经历（隐藏）
- **动机**：当前行为的深层原因

### 与主角关系
- **初始关系**：
- **潜在发展**：可能成为盟友/敌人/暧昧对象
- **关键节点**：未来可能改变关系的事件

### 语言特点
- **口头禅**：
- **说话风格**：文绉绉/口语化/粗犷等
- **特殊表达**：独特的说话方式
```

## 随机生成规则

当需要随机生成人物时，使用以下规则：

### 姓氏选择
- 贵族常用：姬、姜、赢、芈、司马、欧阳
- 中层常用：张、王、李、赵、刘、周
- 底层常用：阿、狗、驴等贱名

### 名字选择
- 单字：简洁有力，如"乾"、"坤"、"风"
- 双字：文雅好听，如"轻舟"、"明月"
- 字号：读书人常有字号，如"云中子"

### 背景选择
- 家族：父母是谁、家族势力如何
- 师承：如学武，是哪门哪派
- 经历：关键人生事件塑造当前性格

## 生成时机

- 主角遇到新NPC时立即生成
- 重要NPC需要完整档案
- 普通NPC可以简化（只填关键项）
- 每5章回顾一次，更新档案
