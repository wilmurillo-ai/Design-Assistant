# Era Generator - 年代生成器

## 生成流程

### 第一步：询问隐藏模式

> "请选择年代揭示模式：
> 
> **【隐藏模式】** ⭐推荐
> 你和主角都不知道具体年代，随情节逐步揭晓
> 通过服饰、建筑、货币等细节暗示时代
> 增加悬疑感和探索乐趣
> 
> **【公开模式】**
> 直接确定具体年代，主角知道自己在什么时代
> 可以利用历史知识主动谋划
> 
> 请回复 **隐藏** 或 **公开**"

### 隐藏模式说明

**后台操作**：AI根据用户大致偏好生成具体年代，但不告诉用户

**写作规则**：
- ❌ 不出现"明朝"、"嘉靖"等明确时代词
- ❌ 不出现具体历史人物名称
- ✅ 只用"这个时代"、"这个国度"等模糊描述
- ✅ 通过细节暗示：服饰纹样、建筑风格、货币形制、官制名称等
- ✅ 让读者和主角一起逐步拼凑真相

**逐步揭示计划**：

| 章节阶段 | 揭示内容 | 示例 |
|---------|---------|------|
| 第1-3章 | 只有模糊时代感 | 战乱、饥荒、朝廷腐败 |
| 第4-7章 | 通过物品暗示 | 发现圆形方孔钱、看到飞鱼服 |
| 第8-12章 | 接触关键信息 | 听到年号、了解政治格局 |
| 第13章+ | 真相大白 | 主角确认：这是嘉靖年间！ |

### 隐藏模式的特殊处理

如果是隐藏模式，需额外准备：

```yaml
隐藏线索设计:
  第1-3章:
    - 发现圆形方孔钱（暗示中国）
    - 看到飞鱼服（暗示明朝）
    - 听到"内阁"而非"丞相"（暗示明清）
  
  第4-7章:
    - 遇到倭寇袭击（暗示嘉靖时期）
    - 听说皇帝多年不上朝（暗示嘉靖/万历）
    - 看到《西游记》手抄本（暗示嘉靖年间）
  
  第8-12章:
    - 主角听到年号"嘉靖"
    - 确认时代：明朝嘉靖年间
    - 剧情进入新阶段：利用历史知识
```

## 使用方法

1. **询问隐藏模式** - 让用户选择是否隐藏年代
2. **选择时代类型** - 国内/国外/虚构/未来/末日
3. **细化具体年代** - 以历史事件为标记
4. **保存到 WORLD.md** - 作为后续写作依据
