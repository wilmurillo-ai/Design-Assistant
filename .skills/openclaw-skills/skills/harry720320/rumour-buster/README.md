# Rumour Buster - 谣言终结者

[![Version](https://img.shields.io/badge/version-0.3.0-blue.svg)](./SKILL.md)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](./LICENSE)

> 一个智能事实核查技能，通过多引擎交叉验证，帮你识别消息真假，追溯信息源头。

---

## ✨ 核心功能

- 🔍 **多引擎验证** - 中文 + 英文双引擎交叉验证
- 🔗 **来源溯源** - 追溯信息最早出处和传播路径
- 📊 **可信度评分** - 基于多维度分析给出评分
- 🌐 **全平台覆盖** - 支持文字消息、网页链接验证
- 💬 **对话式设置** - 简单交互即可完成配置

---

## 🚀 快速开始

### 安装

```bash
clawhub install rumour-buster
```

### 首次设置

安装后自动进入设置流程，或输入：
```
setup
```

按照对话提示完成配置：
1. 检查依赖（自动）
2. 配置 Tavily API（可选，推荐）
3. 选择搜索引擎
4. 完成！

### 使用示例

**验证文字消息**：
```
/验证 "柠檬水可以抗癌"
```

**验证网页链接**：
```
/验证 https://example.com/news/article
```

## 🔧 工作原理

### 双引擎验证架构

```
┌─────────────────────────────────────────┐
│  第1次：中文聚合搜索                      │
├─────────────────────────────────────────┤
│  • kimi_search（中文聚合）               │
│  • 搜狗（网页搜索）                       │
│  • 搜狗微信（公众号）                     │
│  • 头条搜索（自媒体）                     │
└──────────────────┬──────────────────────┘
                   │
                   ▼
         ┌─────────────────┐
         │   中文结果分析   │
         │  - 最早出处      │
         │  - 传播路径      │
         └────────┬────────┘
                  │
┌─────────────────────────────────────────┐
│  第2次：英文深度搜索                      │
├─────────────────────────────────────────┤
│  • Tavily（AI 深度搜索）                 │
│  • DuckDuckGo（隐私保护）                │
│  • Startpage（国际综合）                 │
└──────────────────┬──────────────────────┘
                   │
                   ▼
          ┌───────────────────────┐
          │   交叉验证 & 溯源分析   │
          │  - 可信度评分          │
          │  - 信息源头追溯         │
          │  - 综合判断            │
          └───────────────────────┘
```

### 搜索引擎配置

| 类型 | 引擎 | 特点 | 必需 |
|:---:|:---|:---|:---:|
| 中文 | **kimi_search** | 中文聚合搜索 | ❌ 可选 |
| 中文 | **搜狗** | 网页搜索 | ✅ 必需 |
| 中文 | **搜狗微信** | 公众号文章 | ✅ 必需 |
| 中文 | **头条** | 今日头条 | ✅ 必需 |
| 英文 | **Tavily** | AI 深度搜索 | ❌ 可选 |
| 英文 | **DuckDuckGo** | 隐私保护 | ✅ 必需 |
| 英文 | **Startpage** | 国际综合 | ✅ 必需 |

## 📋 输出示例

### 验证报告格式

```markdown
# 🔍 谣言终结者验证报告

## 待验证信息
"柠檬水可以抗癌，效果比化疗强一万倍"

## 🌐 使用的搜索引擎
中文：kimi_search, 搜狗, 搜狗微信, 头条
英文：Tavily, DuckDuckGo, Startpage

## 🔗 溯源结果
- 最早出处：2017年国外邮件谣言
- 原始声称来源：美国巴尔的摩健康科学研究所（已辟谣否认）
- 传播路径：邮件 → 社交媒体 → 中文自媒体 → 朋友圈

## 📊 可信度评分：5% - 虚假消息 ❌

## 📌 结论
该说法为虚假谣言。柠檬是健康水果，含抗氧化物质...

---
验证时间：2026-04-03 06:30 UTC
```

## 🛠️ 配置说明

### 依赖项

**必需**：
- `api-multi-search-engine` 技能

**可选**（推荐）：
- `kimi_search` 工具
- Tavily API Key（免费 1000次/月）

### Tavily API 申请

1. 访问 https://tavily.com/
2. 注册账号
3. 创建 API Key
4. 在 setup 中粘贴 Key
