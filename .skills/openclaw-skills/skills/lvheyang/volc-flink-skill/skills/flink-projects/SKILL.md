# Flink 项目管理技能

用于列举和查看火山引擎 Flink 项目信息，包括项目列表、项目详情、配额和使用情况等。

---

## 通用约定（必读）

本技能的基础约定与只读约定统一收敛在：

- `../../COMMON.md`
- `../../COMMON_READONLY.md`

本技能默认用于项目查询、项目详情和项目范围选择，不执行资源创建、删除、发布、启动等变更操作。

---

## 🎯 核心功能

### 0. 登录状态检测（从 flink-volc 借鉴）
**在执行任何操作前，先检测登录状态！**

**检测步骤**：
1. 尝试执行一个简单的命令（如 `volc_flink projects list`）
2. 如果提示"请先登录"，则提示用户需要登录
3. 引导用户使用 `flink-volc` 技能进行登录

**错误处理**：
- 如果检测到未登录，立即停止后续操作
- 友好提示用户需要先登录
- 引导使用 flink-volc 技能

---

### 1. 列举所有项目

使用 `volc_flink projects list` 列举所有项目。

**命令格式**：
```bash
volc_flink projects list
```

**输出内容包括**：
- 项目名称
- 项目 ID
- 项目描述
- 创建时间
- 状态
- 其他元数据

---

### 2. 查看项目详情

使用 `volc_flink projects detail` 查看项目详情。

#### 2.1 信息提取与智能选择
从用户提问中提取关键信息：
- **项目名** (project_name)

**如果用户没有明确提供项目名**：
1. 列出所有项目：`volc_flink projects list`
2. 让用户选择项目
3. 然后查看项目详情

#### 2.2 执行查看
**命令格式**：
```bash
# 查看项目详情
volc_flink projects detail <项目名称>
```

**输出内容包括**：
- 项目基本信息（名称、ID、描述等）
- 项目配额信息（CPU、内存、存储等）
- 资源使用情况
- 项目状态
- 创建和更新时间

---

### 3. 智能项目选择流程

当用户没有明确提供项目名时，按以下流程处理：

**流程步骤**：
1. 列出所有项目
2. 展示项目列表供用户选择
3. 用户选择后，执行具体操作

**展示格式**：
```
# 📋 可用项目列表

请选择一个项目：

1. [项目名 1] - [项目描述]
2. [项目名 2] - [项目描述]
3. [项目名 3] - [项目描述]
...

请输入项目编号或项目名：
```

## 工具调用顺序

### 列举项目
1. **检测登录状态** - 确认已登录
2. **列举项目** - `volc_flink projects list`

### 查看项目详情

2. **智能选择项目** - 如果用户没有提供，列出项目供选择
3. **查看项目详情** - `volc_flink projects detail <项目名>`
4. **展示项目信息** - 向用户展示项目详情

## 错误处理

### 常见错误及处理

#### 错误 1：未登录
**错误信息**：`❌ 请先登录`

**处理方式**：
- 友好提示："检测到未登录火山引擎账号，请先登录"
- 引导使用 flink-volc 技能："请使用 flink-volc 技能进行登录"
- 停止后续操作

#### 错误 2：项目不存在
**错误信息**：`Project not found`

**处理方式**：
- 提示："未找到指定的项目，请检查项目名是否正确"
- 列出可用项目：`volc_flink projects list`
- 让用户重新选择

## 注意事项

### 重要：项目选择

⚠️ **当用户没有明确提供项目名时**：
- 总是先列出所有项目供用户选择
- 不要假设默认项目
- 提供清晰的编号和项目信息

### 通用注意事项

1. **先检测登录状态**：在执行任何操作前，先检测是否已登录
2. **智能项目选择**：如果用户没有提供项目名，总是先列出项目供选择
3. **提供详细信息**：查看项目详情时，展示完整的配额和使用情况
4. **引导后续操作**：列举项目后，引导用户进行后续操作
5. **友好的错误处理**：如果操作失败，向用户说明失败原因，并提供解决方案

## 🎯 技能总结

### 核心功能
1. ✅ **列举项目** - 列出所有可用的 Flink 项目
2. ✅ **查看项目详情** - 查看项目的详细信息、配额和使用情况
3. ✅ **智能项目选择** - 交互式项目选择流程
4. ✅ **配额和使用情况** - 展示项目的资源配额和使用情况

### 与其他技能的关系
- **依赖 flink-volc** - 需要先登录才能查看项目
- **被其他技能依赖** - 很多技能都需要先选择项目
- **配合 flink-config** - 可以设置默认项目
- **配合其他技能** - 选择项目后可以使用其他技能进行操作

这个技能是项目操作的入口，让用户可以快速找到和选择自己的项目！
