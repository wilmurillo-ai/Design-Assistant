# Example: Competitive Analysis

## User Request
"Compare Stripe, Square, and Adyen — I need to understand their tech stacks, team sizes, and recent activity."

## What the Agent Does

1. **Matches all 3 companies** in a single batch
2. **Enriches with**: firmographics + technographics + workforce-trends
3. **Gets events**: Recent funding, hiring, partnerships for all 3
4. **Presents**: Side-by-side comparison table

## Expected Output

| Dimension | Stripe | Square | Adyen |
|---|---|---|---|
| Employees | 8,000+ | 5,000+ | 3,000+ |
| Revenue | $1B+ | $500M-1B | $1B+ |
| HQ | San Francisco | San Francisco | Amsterdam |
| Key Technologies | Ruby, React, AWS | Java, Kotlin, GCP | Java, Kafka, Azure |
| Recent Funding | Series I, $6.5B | Public (SQ) | Public (ADYEN) |
| Hiring Trend | +12% engineering | +8% all depts | +15% engineering |
