# 📬 email-monitor — OpenClaw Skill

> 让你的 AI 助手自动盯着邮箱，新邮件来了第一时间告诉你。

---

## ✨ 是什么？

**email-monitor** 是一个 OpenClaw Skill，帮你把邮件监控这件事彻底自动化。

你只需要告诉 AI：监控哪个邮箱、多久查一次、用什么格式通知你 —— 剩下的全交给它。

---

## 🚀 能做什么？

- 📥 **定时抓取新邮件**：每小时、每几小时、每天定时，自由设定
- 🔔 **主动推送通知**：新邮件直接发到你的聊天窗口，不用打开邮箱
- 📝 **智能摘要**：AI 帮你提炼关键信息，只看最重要的内容
- 📎 **附件自动下载**（可选）：指定目录，附件自动保存到本地
- 🔄 **增量拉取**：只处理新邮件，不重复，0 token 空跑浪费

---

## 📦 支持的邮件服务

| 服务商 | 支持情况 |
|--------|---------|
| Gmail | ✅ 完整支持（App Password） |
| Outlook / Hotmail | ✅ 支持 |
| QQ 邮箱 | ✅ 支持 |
| 163 / 126 邮箱 | ✅ 支持 |
| Yahoo Mail | ✅ 支持 |
| 任意 IMAP 邮箱 | ✅ 通用支持 |

## 🎯 使用场景

- 你订阅了很多 GitHub Issue / PR 通知，想让 AI 帮你筛选重要的
- 工作邮件太多，想让 AI 每天早上给你一份摘要
- 监控某个邮箱是否收到特定类型的邮件
- 懒得每隔一会儿就去刷邮箱

---

## 🔧 怎么用？

安装 Skill 后，直接对话就能启动引导流程：

```
我想监控我的 Gmail 邮箱
```

AI 会自动引导你完成 6 步配置：

```
第 1 步  告知要监控的邮箱地址
第 2 步  按邮件服务商给出配置方法（如何生成 App Password）
第 3 步  自动测试连接，确认可以正常读取
第 4 步  设定检查频率（每 1 小时 / 每 4 小时 / 每天 8 点…）
第 5 步  设定通知格式 + 是否下载附件
第 6 步  确认配置，自动创建定时任务 ✅
```

全程对话式，5 分钟内完成设置。

---

## 🏗️ 技术细节

- 使用标准 **IMAP over SSL** 协议，无需第三方服务
- 用 **UID 追踪**记录读取进度，重启不丢状态
- 无新邮件时 **LLM 完全不介入**，token 消耗为 0
- 配置文件存储在 `~/.openclaw/email-monitor/<账号>/`，多账号互不干扰

---

## 📥 安装方式

将 `email-monitor.skill` 文件拖入 OpenClaw → Skills 面板，或通过 CLI 安装：

```bash
openclaw skills install email-monitor.skill
```
