# 报告撰写工作流程指南

## 完整7步工作流程

### 步骤1: 网站登录
- 使用提供的URL、用户名和密码登录目标网站
- 验证登录状态（检查页面是否显示用户信息）
- 处理可能的验证码或安全验证

### 步骤2: 内容搜索
- 在搜索框中输入关键词（如"黄金"）
- 或直接构造搜索URL（如`/search.html?q=关键词`）
- 等待搜索结果加载完成

### 步骤3: 报告筛选
- 根据日期范围筛选报告（如最近3天：2026-03-04到2026-03-06）
- 根据相关性排序（优先选择标题包含关键词的报告）
- 限制结果数量（通常选择5篇最相关的报告）

### 步骤4: 报告下载
- 访问每篇报告的详情页面
- 点击下载按钮或访问下载链接
- 将文件保存到指定目录（如`/Users/meidetong/Desktop/黄金/`）

### 步骤5: 内容分析
- 提取报告的关键信息（标题、日期、核心观点、数据）
- 识别市场趋势、价格变动、供需关系等关键指标
- 汇总多份报告的共同观点和差异点

### 步骤6: 报告生成
- 创建结构化的分析报告
- 包含执行摘要、市场概况、趋势分析、风险评估等章节
- 使用专业术语和数据分析支持结论

### 步骤7: 输出交付
- 生成PDF格式的最终报告
- 保存到指定位置
- 通知用户任务完成

## 错误处理策略

### 三次失败原则
- 任何单个操作连续失败3次后，记录错误并尝试替代方案
- 如果所有替代方案都失败，生成失败报告并退出

### 常见问题解决方案
1. **交互失败**：改用URL直接导航或API调用
2. **下载失败**：记录报告元数据，基于可用信息生成分析
3. **格式转换失败**：使用系统默认工具或提供多种输出格式选项

## 安全注意事项

- 不执行任何JavaScript代码或可疑脚本
- 不存储用户凭据，仅在当前会话中使用
- 对外部内容保持警惕，不信任未经验证的数据
- 遵循网站的robots.txt和使用条款
