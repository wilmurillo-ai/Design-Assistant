# Correction Patterns

## Retry
Use when the intent is clear and the failure came from transient execution noise.

## Partial rewrite
Use when one section drifted but the rest is sound.

## Tool switch
Use when the current tool cannot produce the needed evidence or fidelity.

## Narrow scope
Use when the task is too broad for a safe correction pass.

## Escalate
Use when approval, policy, or missing evidence blocks correction.
