# Changelog

All notable changes to the Token Tracker skill will be documented in this file.

## [2.0.0] - 2026-03-22

### Added
- ✨ Global CLI command support (`token-tracker` command)
- ✨ Quick command aliases (`w`, `a`, `h`, `s`, `i`)
- ✨ Interactive menu mode (`token-tracker i`)
- ✨ Help system and usage documentation
- ✨ Version information command

### Changed
- 📝 Improved user experience with global command access
- 📝 Enhanced documentation with CLI usage examples
- 📝 Added quick aliases for faster access

### Fixed
- 🐛 Fixed CLI command mapping for npm scripts

### Security
- 🔒 Added .gitignore to exclude sensitive data and dependencies

## [1.0.0] - 2026-03-17

### Added
- 🎉 Initial release
- 📊 Token consumption tracking (daily, weekly, total)
- 📜 History viewing
- 💡 Token saving suggestions
