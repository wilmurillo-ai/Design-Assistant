# Scoring Template

Use this table when ranking topic candidates.

| Field | Meaning | Range |
|---|---|---|
| audience_fit | Match to target creator segment | 1-5 |
| freshness | Recency / momentum | 1-5 |
| monetization_fit | Likelihood of commercial payoff | 1-5 |
| execution_speed | How fast creator can ship | 1-5 |

Priority formula:

`(audience_fit * 0.35) + (freshness * 0.25) + (monetization_fit * 0.25) + (execution_speed * 0.15)`
