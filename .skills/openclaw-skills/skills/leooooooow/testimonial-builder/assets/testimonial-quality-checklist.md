# Testimonial Quality Checklist

---

## 1. Collection Process (7 items)

- [ ] Collected via official channel (email, in-app, contact form, review platform)
- [ ] Customer opt-in obtained (explicit agreement recorded)
- [ ] Submission date timestamped (MM/DD/YYYY in database)
- [ ] Customer contact available (email, phone, or account reference)
- [ ] Follow-up sent within 24 hours (thank you, next steps)
- [ ] Permission form signed (digital acceptance counts)
- [ ] Incentive disclosed to customer in writing (if applicable)

---

## 2. Source Verification (7 items)

- [ ] Verified purchase badge obtained (order # confirmed in system)
- [ ] Customer account in good standing (no fraud flags, no chargebacks)
- [ ] Purchase 3+ months prior to submission (established customer status)
- [ ] Product ownership verified (matches testimonial product, not different item)
- [ ] Customer identity verified (name + email match account)
- [ ] Not from competitor domain (customer company not competitor)
- [ ] No multiple submissions from same IP (max 3 testimonials per IP per 30 days)

---

## 3. Content Quality (7 items)

- [ ] No AI-detected language (reads authentically, not templated)
- [ ] Grammar/spelling acceptable (minor edits OK; major issues flag for revision)
- [ ] No profanity or crude language (family-friendly)
- [ ] Includes specific details (feature, metric, or use-case mentioned)
- [ ] Before/after narrative present (problem → solution → result)
- [ ] Emotional tone authentic (relief, joy, confidence evident)
- [ ] Credibility indicators present (verified purchase, timeline, media)

---

## 4. Platform Optimization (7 items)

- [ ] Character length appropriate for platform (within spec)
- [ ] Tone matches platform culture (pro for B2B, casual for B2C, enthusiastic for social)
- [ ] Terminology platform-appropriate (no jargon if consumer, no slang if B2B)
- [ ] Hashtags/emoji included if applicable (Instagram, TikTok, Twitter)
- [ ] Handles/mentions tagged correctly (if creator collaboration)
- [ ] CTA clear if applicable (next step: shop, learn more, sign up)
- [ ] Media dimensions correct (400x400px headshot min, 1080p video min)

---

## 5. Legal Compliance (6 items)

- [ ] Incentive disclosed (if gifted or discounted; clear disclosure)
- [ ] No false health/medical claims (non-medical product avoids FDA liability)
- [ ] No competitor bashing (focus on product benefits, not competitor criticism)
- [ ] No unsubstantiated claims (metrics tied to specific use-case, not universal)
- [ ] Usage rights obtained in writing (permission for ads, email, social, website)
- [ ] Legal review passed (approved by legal or compliance tool)

---

## 6. Formatting Standards (6 items)

- [ ] Customer name correct (full name, first+last initial, or first name; consistent)
- [ ] Location included (city, state, or country for context)
- [ ] Star rating displayed (5-star, 4-star, or verified badge)
- [ ] Headshot or media provided (professional quality, on-brand)
- [ ] Testimonial ID assigned (TST-[MONTH]-[SOURCE]-[INITIALS])
- [ ] Quotation marks correct (full testimonial in quotes; attribution outside)

---

## 7. Deployment Readiness (6 items)

- [ ] Manager/QA reviewer approval obtained (secondary sign-off)
- [ ] Brand style guide compliance verified (tone, terminology, formatting match)
- [ ] No confidential information exposed (no internal names, proprietary processes)
- [ ] Customer approval for publication obtained (confirm OK to publish publicly)
- [ ] Placement map defined (platform(s), position, target audience)
- [ ] Scheduled for publication (calendar entry created)

---

## 8. Performance Monitoring (5 items)

- [ ] Performance baseline established (initial CTR, conversion metrics recorded)
- [ ] A/B test variant created (alternative for comparison)
- [ ] Review schedule set (quarterly audit for performance assessment)
- [ ] Underperformance threshold defined (e.g., "pause if CTR < 2% after 6 months")
- [ ] Success metric identified (CTR, conversion, revenue, or engagement)

---

## 9. Media Quality (5 items)

- [ ] Headshot provided if required (400x400px min, professional quality)
- [ ] Headshot shows face clearly (not obscured, blurry, or AI-generated)
- [ ] Lifestyle image provided if applicable (product in use, on-brand)
- [ ] Video testimonial in good lighting if applicable (minimal shadows, clear audio)
- [ ] Media files named and organized (convention: [testimonial-id]_[type].jpg)

---

## 10. Metadata & Tracking (3 items)

- [ ] Logged in asset database (Airtable, spreadsheet, or CRM)
- [ ] Source documented (email, Amazon, in-app survey, TikTok, creator collab, etc.)
- [ ] Review date scheduled (6-month calendar reminder for performance re-assessment)

---

## Scoring Guide

**Per Category (Pass/Conditional/Fail):**
- Pass: 6-7 of 7 items checked ✓
- Conditional: 4-5 of 7 items checked (approve with notes if critical items covered)
- Fail: 0-3 items checked (request revision or reject)

**Overall Decision:**
- **APPROVE:** Pass in all 10 categories → ready for publication
- **APPROVE WITH EDITS:** Pass in 8-9 categories → minor fixes (grammar, formatting)
- **REQUEST REVISION:** Pass in 6-7 categories → significant changes needed
- **REJECT:** Pass in 0-5 categories → do not publish

---

## Common Issues & Solutions

| Issue | Solution | Checkbox |
|-------|----------|----------|
| Generic praise ("Great product!") | Request specific benefit; offer template | 3.4 |
| Missing timeline | Ask for usage duration; verify purchase date | 3.5 |
| Undisclosed incentive | Add disclosure to testimonial metadata | 5.1 |
| Competitor bashing | Refocus on your product benefits | 5.3 |
| AI-detected language | Request customer provide original words | 3.1 |
| Poor headshot quality | Request professional headshot | 9.2 |
| No verification | Request order confirmation | 2.1 |
| Wrong platform tone | Rewrite for platform culture match | 4.2 |

---

## Quick Review (1-minute check)

Before publishing ANY testimonial, verify:

1. ✓ Verified purchase confirmed in system
2. ✓ Specific benefit/metric mentioned (not generic)
3. ✓ Professional headshot or video provided
4. ✓ Incentive disclosed (if applicable)
5. ✓ Legal review approved
6. ✓ Tone matches platform
7. ✓ Metadata complete (ID, source, review date)
8. ✓ Manager approval obtained

**All 8 checked? APPROVE. Missing any? REQUEST REVISION.**
