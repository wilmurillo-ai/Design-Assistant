# Risk Scorecard

Rate each category from 1 (low) to 5 (high).

| Category | What to check |
|---|---|
| Policy/advertiser safety | Explicit harmful content, unsafe claims, restricted themes |
| Originality/reuse | Re-upload patterns, low-transform edits, template spam |
| Misleading claims | Unrealistic promises, unverifiable outcomes |
| Brand suitability | Tone mismatch for mainstream ad buyers |

Decision:
- Green: all categories <= 2
- Yellow: any category = 3
- Red: any category >= 4
