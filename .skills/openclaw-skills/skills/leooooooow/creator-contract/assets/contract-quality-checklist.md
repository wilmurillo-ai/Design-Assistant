# Creator Contract — Pre-Send Quality Checklist

Run through this before handing the draft to legal or to the creator. 36 items across 8 categories.

## Parties and Scope (4)

- [ ] Brand legal entity name and state of incorporation are correct
- [ ] Creator legal name (or LLC name if applicable) is correct
- [ ] Effective date is filled in
- [ ] Scope paragraph clearly references the Deliverables table

## Deliverables (5)

- [ ] Number of posts is specified
- [ ] Platform(s) named (TikTok / IG Reels / YT Shorts / etc.)
- [ ] Format and length spec included (e.g., 30–60s vertical video)
- [ ] Posting window has specific dates, not vague references like "Q2"
- [ ] Draft submission deadline is set N business days before posting

## Content Specs and Approvals (4)

- [ ] Required hashtags listed (e.g., #ad, #brandname)
- [ ] Required talking points listed
- [ ] Revision rounds capped (recommend 2)
- [ ] Approval SLA set for both sides (recommend 48 hours each)

## Usage Rights (5)

- [ ] Organic rights window stated
- [ ] Paid rights window stated
- [ ] Whitelisting language included if paid usage is in scope
- [ ] Repurposing channels enumerated
- [ ] Creator credit requirements stated where applicable

## Exclusivity (3)

- [ ] Exclusivity window has a specific number of days
- [ ] Scope is named-competitor or narrow-category, not "industry-wide"
- [ ] Named competitor list is filled in (if applicable)

## Compensation (5)

- [ ] Flat fee amount and currency specified
- [ ] Product value stated and ship date set
- [ ] Performance bonus tier table included (if applicable) with measurable metric
- [ ] Affiliate commission rate, attribution window, and payment cadence set (if applicable)
- [ ] Payment schedule is creator-fair (no 100% net 60+ post-publish)

## Compliance and Risk (5)

- [ ] FTC / ASA / equivalent disclosure clause included
- [ ] Originality representation included
- [ ] Music licensing representation included
- [ ] Kill fee schedule set
- [ ] Termination clause includes notice period and cure rights

## Final Pass (5)

- [ ] Jurisdiction and governing law set
- [ ] "Localize before signing" callout included for non-US creators
- [ ] Negotiation cheat sheet appended
- [ ] All bracket placeholders ([NAME], [DATE], etc.) removed or filled
- [ ] Disclaimer line ("draft, not legal advice") present

---

## Stop-Ship Conditions

If any of the following are true, do **not** send the draft to the creator:

1. Bracket placeholders remain unfilled
2. Compensation total is missing or inconsistent across sections
3. FTC / disclosure clause is missing
4. Usage rights window is unbounded ("perpetual paid" without compensation match)
5. Creator's legal name is not confirmed
