Output template for Spillover Estimator

Use this response shape when the user wants a clean final answer.

1. Executive summary
- Source channel reviewed
- Downstream channel reviewed
- Main estimate in plain language

2. Spillover estimate
- Estimated direction: positive / weak / unclear / negative
- Estimated strength: light / moderate / strong
- Timing overlap notes

3. Evidence blocks
- Source-channel change
- Downstream-channel change
- Timing alignment
- Confounders or uncertainty

4. Confidence and caveats
- High / medium / low confidence
- Missing data notes
- Why this is directional or stronger than directional

5. Recommended next step
- What to measure next
- Whether to scale, test, or hold
