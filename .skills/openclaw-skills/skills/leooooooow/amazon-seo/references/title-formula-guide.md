# Amazon Title Formula Guide by Category

Title structure requirements vary significantly by Amazon category. Use this guide to match the correct formula and character limit before writing.

---

## Universal Title Rules (All Categories)

- **No promotional phrases:** "Best", "#1", "Sale", "Free Shipping", "Limited Time"
- **No HTML tags** in the title field
- **No ALL CAPS** for the entire title (title case only; some caps in brand name is fine)
- **Numbers as digits:** "32oz" not "thirty-two ounce"
- **Category-specific style:** Check Amazon's style guide for your browse node — rules differ
- **Special characters:** Hyphens (–), commas (,), and parentheses allowed; avoid %, @, !

---

## Category-Specific Title Formulas

### Home & Kitchen / Kitchen & Dining
**Character limit:** 200 characters
**Formula:**
```
[Brand] [Product Type] [Key Feature] [Size/Quantity] – [Secondary Keyword] [Additional Feature] – [Material/Certification]
```
**Example:**
```
NaturalWare Bamboo Cutting Board with Juice Groove – Extra Large 18x12 Wooden Cutting Board for Kitchen – Organic Bamboo, Non-Slip Feet, 2-Pack
```

### Health, Household & Baby Care
**Character limit:** 200 characters
**Formula:**
```
[Brand] [Product Type] [Key Benefit] [Count/Size] – [Secondary Keyword] [Usage Context] – [Certification/Key Spec]
```
**Caution:** Avoid health claims in title — "cures", "treats", "prevents" trigger suppression

### Sports & Outdoors
**Character limit:** 200 characters
**Formula:**
```
[Brand] [Product Type] [Primary Keyword] [Size/Weight] – [Sport/Activity Context] – [Key Feature] [Secondary Keyword]
```

### Electronics & Accessories
**Character limit:** 200 characters
**Formula:**
```
[Brand] [Product Type] [Model] – [Primary Keyword] [Key Spec] – [Compatibility] [Feature]
```
**Note:** Model number matters for electronics — include it

### Beauty & Personal Care
**Character limit:** 200 characters
**Formula:**
```
[Brand] [Product Type] [Skin/Hair Type] [Size] – [Key Benefit] [Primary Keyword] – [Key Ingredient or Certification]
```
**Caution:** Drug claims trigger beauty-to-drug category crossover — avoid

### Toys & Games
**Character limit:** 200 characters
**Formula:**
```
[Brand] [Product Name] – [Age Range] [Primary Keyword] – [Key Feature] [Educational Benefit]
```

### Clothing, Shoes & Jewelry
**Character limit:** 150 characters
**Formula:**
```
[Brand] [Gender] [Product Type] [Style/Fit] [Primary Keyword]
```
**Note:** Size and color belong in variant attributes, not the title

### Books
**Character limit:** No standard limit (follows title + subtitle format)
**Formula:** Do not optimize book titles as product listings — different algorithm applies

## Title Keyword Placement Priority

Within any title, the ordering logic for keyword placement:

| Position | Content | Weight |
|---|---|---|
| First word | Brand name (if strong) or primary keyword (if brand unknown) | Highest CTR impact |
| Words 2-5 | Primary keyword if not in position 1 | High ranking weight |
| Words 6-12 | Key differentiator + secondary keyword | Medium ranking weight |
| Mid-title | Size, count, material | Conversion-relevant |
| After dash | Third keyword + additional feature | Lower but still indexed |

**Rule of thumb:** If a keyword is important enough to bid on in PPC, it's important enough to be in the first half of the title.

---

## Title Audit Scoring

Score your title 1-5 on each dimension before finalizing:

| Dimension | 1 (Poor) | 3 (OK) | 5 (Strong) |
|---|---|---|---|
| Primary keyword placement | Absent or buried in last 20% | In second half | In first 5 words |
| Character utilization | < 60% of limit used | 70-80% used | 85-100% used |
| Readability | Unreadable keyword string | Mostly readable | Flows naturally when read aloud |
| Policy compliance | Contains prohibited terms | Minor issues | Fully compliant |
| Conversion elements | No size, count, or features | 1-2 specs present | Size + count + key feature |

A title scoring 4+ on all dimensions is publish-ready.
