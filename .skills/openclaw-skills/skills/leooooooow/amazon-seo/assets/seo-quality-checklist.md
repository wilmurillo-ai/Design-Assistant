# Amazon SEO Quality Checklist

Use before publishing a new listing or completing a listing optimization audit.

---

## Title Checklist

### Keyword Placement
- [ ] Primary keyword (highest search volume) in first 5 words of title
- [ ] At least 2 secondary keywords included in title
- [ ] No critical keywords buried in last 20% of title

### Policy Compliance
- [ ] No promotional language: "Best", "#1", "Sale", "Free Shipping", "Amazon's Choice"
- [ ] No ALL CAPS (title case only)
- [ ] No HTML tags or special character violations
- [ ] No competitor brand names
- [ ] No health claims or prohibited claims for category

### Technical
- [ ] Character count at 85-100% of category limit (not wasting indexing space)
- [ ] Numbers as digits (32oz not "thirty-two ounce")
- [ ] Reads naturally aloud without sounding robotic

---

## Bullet Points Checklist

### Structure
- [ ] All 5 bullet slots used
- [ ] Each bullet starts with ALL CAPS BENEFIT HEADLINE
- [ ] Benefit comes before feature in each bullet (not "Made from bamboo — so it's durable" but "NATURAL DURABILITY — Crafted from...")
- [ ] Average bullet length: 180-250 characters

### Keyword Coverage
- [ ] Primary keyword appears in at least 2 of 5 bullets
- [ ] Secondary keywords distributed across bullets 1-3
- [ ] No keyword stuffing (keywords feel natural in context)

### Conversion Elements
- [ ] Bullet 1: Primary value proposition clearly stated
- [ ] Bullet 2: Key technical specification that drives decision
- [ ] Bullet 3: Compatibility or versatility addressed
- [ ] Bullet 4: Quality, material, or certification mentioned
- [ ] Bullet 5: Warranty or customer service / risk reversal present

---

## Backend Search Terms Checklist

### Duplication Audit (most important step)
- [ ] Every word in the title listed and excluded from backend
- [ ] Every word in all 5 bullets listed and excluded from backend
- [ ] Brand name not repeated in backend

### Coverage
- [ ] Long-tail phrases (3+ word combinations) included
- [ ] At least 5 common misspellings added
- [ ] Spanish equivalents added (US marketplace)
- [ ] Synonyms and related terms included
- [ ] Byte count is 240-250 bytes (using near-full field)

### Policy
- [ ] No competitor brand names
- [ ] No Amazon brand names ("Alexa", "Echo", etc.)
- [ ] No prohibited claims

---

## A+ Content Checklist (Brand Registered)

### Completeness
- [ ] At minimum 4 modules created (5 recommended)
- [ ] All modules using maximum allowed text fields
- [ ] Hero module present with primary keyword in headline

### Keyword and Conversion Coverage
- [ ] Primary keyword appears in at least 2 module headlines
- [ ] Comparison module present (vs. competitor category or older version)
- [ ] FAQ or objection-handling module present
- [ ] Technical spec module with key dimensions/materials

### Image and Copy Quality
- [ ] Each image has descriptive alt text with relevant keywords
- [ ] No lifestyle-only modules with no supporting copy
- [ ] Mobile rendering checked (Amazon previews desktop and mobile)

---

## Indexing Verification Checklist

- [ ] All target keywords searched on Amazon after 7 days of listing being live
- [ ] ASIN found in results for each target keyword (even if page 10+)
- [ ] Any keywords not indexed → identified and added to backend or a bullet

---

## Quarterly Refresh Triggers

Mark for refresh if any of the following apply:
- [ ] Listing hasn't been updated in 6+ months
- [ ] Organic rank for primary keyword dropped more than 20 positions
- [ ] A competitor updated their listing with keywords you're missing
- [ ] New product variants added that aren't reflected in title/bullets
- [ ] Customeq reviews surface new objections or use cases not in listing
- [ ] Platform updated category style guide requirements

---

## Final Pre-Publish Score

Rate each section 1-5:

| Section | Score | Notes |
|---|---|---|
| Title (keyword placement + compliance + length) | /5 | |
| Bullets (structure + keywords + conversion) | /5 | |
| Backend (zero duplicates + full coverage) | /5 | |
| A+ Content (completeness + keywords) | /5 | |

**Minimum to publish:** 4/5 on title and bullets, 3/5 on backend, A+ can be added post-launch
