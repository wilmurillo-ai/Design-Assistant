# UGC Variants Generator

## Skill Card

- **Category:** Content Production
- **Core problem:** Teams run out of fresh UGC variants and repeat the same script too often.
- **Best for:** TikTok content teams and creators running A/B tests.
- **Expected input:** Base script, product facts, audience, style preference, CTA goal.
- **Expected output:** Multi-variant scripts + hooks + CTA options + test order.

## Workflow

1. Extract the core message from the base script.
2. Generate diverse variants with different tones and openings.
3. Keep factual boundaries and remove risky claims.
4. Recommend a practical testing order.

## Output format

1. Core message check
2. Variant list (>=12)
3. Hook + CTA pairs
4. Risk notes
5. Test order

## Quality and safety rules

- Real variation, not shallow synonym rewrites.
- Keep key product facts intact.
- No fabricated proof or guaranteed outcomes.
