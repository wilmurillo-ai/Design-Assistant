# Creator CTA Optimizer

Improve weak, vague, or mistimed creator CTAs so the audience knows what to do next without the content feeling forced.

Use this skill when a user has a script, post, or creator video concept that needs a stronger call to action. It is most useful for TikTok Shop, short-form UGC, creator ads, live commerce clips, founder videos, and conversion-oriented social content where the CTA needs to match audience intent and platform context.

## Problem it solves
A lot of content loses conversion at the last moment because the CTA is too generic, too aggressive, too late, or mismatched to the audience stage. This skill helps adjust the CTA so it fits the content, the viewer’s readiness, and the platform behavior.

## Use when
- A script ends weakly
- The CTA sounds generic like “link in bio” or “buy now” with no nuance
- You need different CTAs by funnel stage
- A creator video needs softer, smarter conversion pressure
- You want multiple CTA options to test

## Do not use when
- The user needs a full script rather than CTA optimization
- There is no defined goal for the viewer action
- The content is purely informational with no next-step intent

## Inputs to ask for
- Existing script, post, or video concept
- Main goal: click, shop, DM, comment, follow, save, join live, etc.
- Funnel stage: awareness, consideration, conversion, retention
- Platform and format
- Product or offer context
- Tone preference: direct, soft, playful, urgent, community-led, proof-led

## Workflow
1. Diagnose why the current CTA underperforms
2. Identify the true audience intent and friction point
3. Match CTA style to the platform and funnel stage
4. Rewrite into multiple CTA variants with different pressure levels
5. Check that the CTA feels natural after the content body
6. Recommend the best variants to test first

## Output requirements
Return:
1. Short diagnosis of the current CTA
2. 5-10 rewritten CTA options
3. CTA variants labeled by tone or use case
4. Best stage/use recommendation for each CTA
5. Suggested top 2-3 to test first
6. Optional CTA ladder for soft -> medium -> hard conversion pressure

## Quality standards
- CTAs should sound like something a real creator would actually say
- Variants must be meaningfully different, not tiny edits
- CTA should fit the user’s actual goal and viewer readiness
- Avoid robotic, pushy, or lazy CTA language
- Recommendations should be immediately testable in content production

## Example prompts
- Fix the CTA at the end of this TikTok Shop script
- Give me better CTAs for a creator ad without sounding too salesy
- Rewrite this weak call to action into softer and stronger variants
