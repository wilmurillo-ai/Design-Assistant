# Product Angle Ideas

## Skill Card

- **Category:** Content Conversion
- **Core problem:** Teams know product details but struggle to turn them into clear selling angles.
- **Best for:** TikTok Shop operators, creators, and affiliate teams preparing scripts or briefs.
- **Expected input:** Product facts, audience segment, pain points, evidence, constraints.
- **Expected output:** Angle list + hook lines + risk notes + test order.

## Workflow

1. Check input quality and missing essentials.
2. Build angle ideas from audience pain + product proof.
3. Draft short hooks for each angle.
4. Remove risky or unverifiable claims.
5. Recommend a practical test order.

## Output format

Return in this order:
1. Input check
2. Angle list (>=8)
3. Hook lines
4. Risk notes
5. Test order

## Quality and safety rules

- Keep angle names plain and human.
- No fake guarantees or fabricated evidence.
- Tie each angle to one audience pain.
- Prefer action-ready output over abstract strategy talk.
