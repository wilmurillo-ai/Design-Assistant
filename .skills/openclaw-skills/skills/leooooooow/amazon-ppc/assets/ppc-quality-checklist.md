# Amazon PPC Quality Checklist

Use before launching campaigns and at each monthly optimization review.

---

## Pre-Launch Checklist

### Unit Economics (complete before any bid is set)
- [ ] Average selling price confirmed (current, not aspirational)
- [ ] Gross margin calculated after COGS, Amazon fees, and shipping
- [ ] Target ACoS set below gross margin
- [ ] Maximum CPC calculated using ACoS × CVR × ASP formula
- [ ] Break-even CPC calculated as a hard ceiling
- [ ] Estimated CVR documented with rationale (review count, category benchmarks)

### Campaign Structure
- [ ] All four tiers planned (Research, Performance, Brand Defense, Competitor)
- [ ] Campaign naming convention documented and applied consistently
- [ ] No duplicate campaigns targeting the same match type + keyword combination
- [ ] Each campaign has a clear stated purpose
- [ ] Daily budgets set with minimum 10× Max CPC floor

### Keywords
- [ ] Seed keyword list reviewed (5-20 terms minimum)
- [ ] Keywords organized by match type across tiers
- [ ] Day-1 negatives added (competitor brands, irrelevant category terms, informational terms)
- [ ] Exact keywords added as negatives in broad/auto campaigns to prevent overlap
- [ ] Negative match types correct (exact vs. phrase as appropriate)

### Bids
- [ ] All starting bids at or below calculated Max CPC
- [ ] Broad and auto bids set lower than exact bids (60-80%)
- [ ] Placement multiplier strategy documented (starting with +20% top-of-search or neutral)
- [ ] No campaigns with zero bid floor (must bid at least $0.02)

### Targeting Completeness
- [ ] Auto campaign running for initial discovery
- [ ] At least one manual exact match campaign for priority keywords
- [ ] Product targeting included (ASIN targeting for auto or Sponsored Display)
- [ ] Brand defense campaign created (even if low budget)

---

## 7-Day Review Checklist

### Data Availability
- [ ] Search term report downloaded (minimum 7 days of data)
- [ ] At least 100 clicks total across all campaigns (enough for early patterns)

### Negative Keyword Build
- [ ] All terms with 30+ clicks and 0 conversions added as negatives
- [ ] Irrelevant terms identified and blocked
- [ ] Competitor terms reviewed (add as negatives unless running conquest)

### Early Performance Signals
- [ ] CTR benchmarked (expected: 0.3-0.8% for product searches)
- [ ] Impression volume confirms campaigns are spending and serving
- [ ] Any campaigns at zero impressions investigated (bid too low, targeting issue)

---

## 30-Day Optimization Checklist

### Search Term Graduation
- [ ] All terms with 5+ conversions at target ACoS moved to dedicated exact match campaigns
- [ ] Graduated terms added as negatives in source auto/broad campaigns
- [ ] New exact campaigns set at calculated Max CPC bid

### Bid Optimization
- [ ] Bid adjustment formula applied: New Bid = Current Bid × (Target ACoS / Actual ACoS)
- [ ] No single bid adjustment exceeds ±20%
- [ ] Bids reviewed at keyword level, not just campaign level

### Budget Rebalancing
- [ ] Budget shifted from underperforming research campaigns to proven performance campaigns
- [ ] Total budget utilization reviewed (if campaigns capping, increase or add budget)
- [ ] Any campaigns with < 5 clicks/day reviewed for bid competitiveness

### Placement Review
- [ ] Placement report reviewed for ACoS by placement
- [ ] Top-of-search multiplier adjusted based on actual ACoS data
- [ ] Product page placement performance compared to category benchmarks

### Structural Health
- [ ] No campaigns with ACoS > 3× target for 30+ days (pause and restructure)
- [ ] Naming convention still consistent for new campaigns added
- [ ] Negative keyword list reviewed for gaps

## Monthly Health Scorecard

| Metric | Target | Actual | Status |
|---|---|---|---|
| Overall ACoS | ≤ [target]% | | |
| Top 5 keywords ACoS | ≤ [target]% | | |
| Wasted spend (0-conversion clicks) | < 15% of total | | |
| Search term graduation rate | 1+ per week | | |
| Budget utilization | 80-95% | | |
| Negative keyword additions | 5+ per review | | |
| Campaigns at < target ACoS | > 50% of budget | | |

---

## Red Flags — Act Immediately

- [ ] **ACoS above gross margin** → Pause or reduce bids on offending campaigns immediately
- [ ] **Budget capping before 8pm** → Increase budget or reduce bids to extend ad schedule
- [ ] **No impressions on exact campaigns** → Bids likely too low; increase 25-30%
- [ ] **Auto campaign spending > 80% of total budget** → Rebalance to manual campaigns
- [ ] **Single keyword consuming > 50% of campaign budget** → Create dedicated campaign for it
- [ ] **Negative search terms still generating impressions** → Check match type and add phrase-level negatives
