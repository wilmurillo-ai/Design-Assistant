# Trend Log — Quality Checklist and Performance Tracker

## Part 1 — Trend Adaptation Quality Checklist

Use before creating trend content to verify the adaptation is worth executing.

### Category 1 — Trend Timing (4 items)
- [ ] Trend discovery timing documented (Day 1 / 2 / 3 / etc.)
- [ ] Current use count checked (under 500k preferred)
- [ ] Trend not yet saturated in your specific product niche
- [ ] Production and posting deadline set to catch the window

**Score — __ / 4**

### Category 2 — Relevance Assessment (5 items)
- [ ] Genuine product connection identified (not forced)
- [ ] Audio emotional tone matches brand/product message
- [ ] Trend is appropriate for your brand (no sensitive associations)
- [ ] Target audience is using this trend (not just unrelated demographics)
- [ ] Product appears naturally — not as an obvious ad within a trend

**Score — __ / 5**

### Category 3 — Adaptation Brief Completeness (6 items)
- [ ] Trend mechanics understood (setup, payoff, why it works)
- [ ] Product connection point clearly stated
- [ ] Script or shot list written before filming
- [ ] Hook written (first 3 seconds specifically planned)
- [ ] CTA included in the brief
- [ ] Production notes provided (setting, lighting, filming style)

**Score — __ / 6**

### Category 4 — Production Standards (4 items)
- [ ] Multiple hook variants planned (2–3 options)
- [ ] Text overlays planned for accessibility (watching without sound)
- [ ] Audio timing mapped (key beat or moment identified for visual match)
- [ ] Native/UGC quality planned (not over-produced for a trend format)

**Score — __ / 4**

**Total pre-production score — __ / 19**
- **17–19**: Good to proceed
- **13–16**: Address flagged items first
- **Below 13**: Reconsider whether to participate in this trend
