# 改写质量自检清单

每次改写完成后，逐项检查：

---

## ✅ 自然度检查

- [ ] 读出声来像真人会说的话吗？
- [ ] 去掉了"不仅...而且"、"值得一提的是"等次板连接词了吗？
- [ ] 句子长短有变化吗？不全是 40-60 字的段落？
- [ ] 有具体细节/数据/场景吗？还是全是空泛形容？
- [ ] 开头 3 秒能抓住注意力吗？

## ✅ 商业意图检查

- [ ] 核心卖点保留了吗？
- [ ] CTA 清晰吗？用户知道下一步做什么？
- [ ] 产品利益点还在吗？没有为了"自然"而丢掉卖点？
- [ ] 目标受众能被这段文案打动吗？
- [ ] 与竞品的差异化表达够清楚吗？

## ✅ 平台适配检查

- [ ] 语气符合目标平台吗？（详情页 vs 评论区 vs 直播）
- [ ] 文案长度适合目标载体吗？（短视频字幕 ≤15 字/句）
- [ ] 用了该平台用户习惯的表达方式吗？
- [ ] 没有跨平台混写？（小红书风 ≠ 淘宝详情页风）

## ✅ 合规检查

- [ ] 没有绝对化用语？（最好、第一、唯一）
- [ ] 没有虚假功效承诺？
- [ ] 没有虚假稀缺/虚假促销？
- [ ] 没有未标注的商业合作？
- [ ] 没有引导站外交易？

## ✅ 整体评估

- [ ] 比原稿更可信？
- [ ] 比原稿更具体？
- [ ] 比原稿曵像真人写的？
- [ ] 能直接拿去用？不需要大改？
