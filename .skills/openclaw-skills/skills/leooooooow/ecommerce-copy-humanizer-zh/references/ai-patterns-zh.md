# 中文 AI 文案常见模式词库

当文案中出现以下模式时，大概率会被识别为 AI 生成。改写时应优先替换。

---

## 高频 AI 信号词

### 连接词/过渡词
| AI 模式 | 替换建议 |
|---|---|
| 值得一提的是 | 直接说事实，不需要引导 |
| 不仅...而且/更... | 拆成两个独立短句 |
| 与此同时 | 删掉，直接接下一句 |
| 在当今/在这个时代 | 删掉，从具体场景开始 |
| 让我们一起 | 换成直接的行动号召 |
| 综上所述 | 删掉，结论直接说 |
| 首先...其次...最后 | 打乱顺序，用口语化连接 |

### 营销空话
| AI 模式 | 替换建议 |
|---|---|
| 品质之选 | 说具体哪里好，用数据/细节 |
| 匠心打造 | 说具体工艺或材料 |
| 全方位 | 说具体覆盖哪几个方面 |
| 一站式解决方案 | 说具体解决了哪几个问题 |
| 赋能/焕新 | 用更口语的说法 |
| 臻选/甄选 | 说具体筛选标准 |
| 为您打造 | 删掉，直接说产品特点 |
| 尊享 | 除奢侈品外都应删掉 |
| 引领潮流 | 说具体的设计趋势或数据 |

### 过度承诺词
| AI 模式 | 风险等级 | 替换建议 |
|---|---|---|
| 最好的 | 🔴 高 | "XX 项指标达到行业前列" |
| 100% 有效 | 🔴 高 | "XX% 用户反馈..." |
| 绝对安全 | 🔴 高 | "通过 XX 认证" |
| 永久保障 | 🔴 高 | "提供 X 天售后" |
| 零风险 | 🟡 中 | "支持 X 天无理由退换" |
| 立竿见影 | 🟡 中 | "X 天内可感受到变化" |
| 秒杀同类 | 🟡 中 | "在 XX 方面做了不同选择" |

## 结构类 AI 信号

### 段落节奏
- **问题：** 每段 40-60 字，长度高度一致
- **修复：** 刻意制造长短句交错，关键信息用短句

### 列表格式
- **问题：** 每个 bullet 都是"XX + 动词 + 效果"格式
- **修复：** 混合问句、感叹句、短语、数据

### 开头模式
- **问题：** "您是否在为 XX 烦恼？"、"在这个 XX 的时代"
- **修复：** 从具体场景、用户痛点或事实数据开头

### 结尾模式
- **问题：** "赶快行动吧！"、"期待您的选择！"
- **修复：** 用具体的下一步动作（"现在下单 → 明天到"）

## 诊断速查

遇到一段文案，按以下顺序快速判断 AI 浓度：

1. **数一下"不仅...而且"出现几次** → ≥2 次基本确认 AI
2. **看开头是否有"在当今/随着..."** → 几乎 100% AI
3. **每段长度是否高度一致** → 强 AI 信号
4. **是否有具体数据/案例/用户场景** → 没有则大概率 AI
5. **读出声来像不像真人会这么说** → 最终检验标准
