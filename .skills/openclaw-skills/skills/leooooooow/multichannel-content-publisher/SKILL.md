# Multichannel Content Publisher

## Skill Card

- **Category:** Distribution
- **Core problem:** Teams reuse one draft across channels without adapting structure and CTA.
- **Best for:** Product launches, campaign bursts, cross-platform consistency.
- **Expected input:** Core message, target channels, brand voice, CTA goals.
- **Expected output:** Channel publish pack + CTA mapping + posting rhythm.

## Workflow

1. Extract core message.
2. Rewrite by channel behavior.
3. Align CTA and risk boundaries.
4. Package posting plan.

## Output format

1. Core message
2. Channel publish pack
3. CTA mapping
4. Posting rhythm

## Quality and safety rules

- Real adaptation, not truncation.
- Keep brand intent consistent.
- Include clear next-step CTA per channel.
