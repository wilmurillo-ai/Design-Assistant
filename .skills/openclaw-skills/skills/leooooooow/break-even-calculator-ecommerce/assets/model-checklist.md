# Break-even Model Quality Checklist

Before delivering results, verify each item:

---

## Cost completeness
- [ ] COGS / landed cost included?
- [ ] Outbound shipping included?
- [ ] Packaging + pick-pack included?
- [ ] Payment processing fees calculated on selling price?
- [ ] Platform / marketplace fees applied at correct rate?
- [ ] Return/refund cost allocated per unit?
- [ ] Fixed costs included if relevant to the decision?

## Calculation accuracy
- [ ] Variable vs fixed costs clearly separated?
- [ ] Contribution margin calculated correctly (price - all variable costs)?
- [ ] Break-even units = fixed costs / CM per unit?
- [ ] Break-even CPA = CM per unit (before ad allocation)?
- [ ] Break-even ROAS calculated correctly?
- [ ] Discount scenarios modeled through to new break-even?

## Assumption transparency
- [ ] Every input labeled with confidence level (✅ / ⚠️ / ❓)?
- [ ] Sources cited for non-obvious assumptions?
- [ ] Low-confidence inputs flagged in results?
- [ ] Sensitivity analysis covers the most uncertain inputs?

## Decision usefulness
- [ ] Results framed as launch / hold / scale / optimize?
- [ ] Specific actions recommended based on numbers?
- [ ] Risk flags highlighted (thin margins, uncertain inputs)?
- [ ] Clear about what would need to change if result is "hold"?

## Common error check
- [ ] Payment fees not calculated on COGS by mistake?
- [ ] Platform fees match actual platform rates?
- [ ] Return cost includes return shipping + restocking + loss?
- [ ] Discount impact modeled on CM, not just revenue?
- [ ] CPA scaled realistically (not assumed flat at higher spend)?
