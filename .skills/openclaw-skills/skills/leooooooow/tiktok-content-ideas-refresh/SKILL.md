# TikTok Content Ideas Refresh

## Skill Card

- **Category:** Content Planning
- **Core problem:** Recent TikTok posts feel repetitive and engagement starts dropping.
- **Best for:** Creators and operators refreshing weekly topic pipelines.
- **Expected input:** Recent topics, comments, target audience, product context.
- **Expected output:** New topic list + opening lines + 7-day post order.

## Workflow

1. Detect repeated themes from recent posts.
2. Pull new topic seeds from comments and objections.
3. Build a refreshed topic list with simple opening lines.
4. Remove risky claims and weak ideas.
5. Recommend a 7-day posting order.

## Output format

Return in this order:
1. Fatigue diagnosis
2. New topic list (>=10)
3. Opening lines
4. Risk notes
5. 7-day posting order

## Quality and safety rules

- Keep topics practical and easy to shoot.
- Avoid fake trend claims.
- Tie each topic to real audience intent.
- Keep language simple and natural.
