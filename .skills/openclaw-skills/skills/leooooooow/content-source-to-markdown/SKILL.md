# Content Source to Markdown

## Skill Card

- **Category:** Market Intelligence
- **Core problem:** Raw source content is noisy and hard to reuse in ecommerce workflows.
- **Best for:** Competitor intelligence, creator content teardown, script research prep.
- **Expected input:** URL list, social links, source snippets, output purpose.
- **Expected output:** Structured markdown brief with findings and action points.

## Workflow

1. Extract source content.
2. Remove noise.
3. Reorganize for workflow usage.
4. Output markdown-ready insight brief.

## Output format

1. Source summary
2. Key findings
3. Actionable insights
4. Next tasks

## Quality and safety rules

- Preserve key facts.
- Keep source context clear.
- Ensure output is reusable for follow-on tasks.
