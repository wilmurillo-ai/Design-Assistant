# 改写品質自己確認リスト

## コア品質チェック

### AIっぽさの除去
- [ ] AIに特有のテンプレート的な接続詞を削除した
- [ ] すべての段落が均一な長さになっていない
- [ ] 空虚な形容詞（「高品質」「プレミアム」等）を具体的な表現に置き換えた
- [ ] 過度な敬語（〜させていただく等の乱用）を自然な表現に修正した

### 商業的意図の維持
- [ ] 元の商品の強みと訴求ポイントを保持している
- [ ] CTAが明確で、変な場所に移動していない
- [ ] 価格・プロモーション情報を消していない
- [ ] ターゲット層が変わっていない

### プラットフォーム適合
- [ ] 対象プラットフォームの文体規範に合っているか確認した
- [ ] 文字数制限がある場合に適合しているか
- [ ] モバイル閲覧での読みやすさを考慮したか

### コンプライアンス
- [ ] 景表法に違反する絶対的表現がないか
- [ ] 薬機法に違反する効能・効果表現がないか
- [ ] SNS投稿の場合、PR表記が必要かを確認した
- [ ] 根拠のない比較・最上級表現を排除した

### 全体的な品質
- [ ] 日本人の実際のEC運営者が書いたように聞こえるか
- [ ] 読むと購買意欲が上がるか
- [ ] リスクある表現を指摘・代替案を提示したか
