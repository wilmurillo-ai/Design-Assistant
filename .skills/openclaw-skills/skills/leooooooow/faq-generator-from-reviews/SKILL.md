# FAQ Generator From Reviews

最好的 FAQ 不是凭空写出来的，而是从真实评论和疑问里长出来的。

## 解决的问题

很多商品页 FAQ 的问题是：
- 问题不真实，没打到用户犹豫点；
- 回答太官方，不像真人会问的；
- 明明评论区反复出现同一个顾虑，却没人把它整理到页面上。

这个 skill 的目标是：
**从评论、差评、问答和常见疑虑里提炼 FAQ，并输出更像用户语言的答案草稿。**

## 何时使用

- 商品页想补 FAQ；
- 评论区经常出现重复疑问；
- 客服和页面内容需要更一致。

## 输入要求

- 用户评论 / 差评 / 问答
- 商品事实与功能边界
- 常见顾虑（尺寸、效果、适用人群、物流、退换等）
- 可选：禁止表述或 claim 边界

## 工作流

1. 聚类重复出现的问题。
2. 区分购买前问题和购买后问题。
3. 生成 FAQ 问题标题。
4. 输出回答草稿与需要 proof 的位置。

## 输出格式

1. FAQ 问题列表
2. 答案草稿
3. 需要补证据 / 人工确认项
4. 建议放置位置

## 质量标准

- 问题要像真实用户会问的。
- 回答要清楚、简短、可信。
- 不要编造没有证据的功效回答。
- 输出应可直接进入 PDP 或帮助中心。
