# 재작성 품질 자가 확인 리스트

## 핵심 품질 체크

### AI 티 제거
- [ ] AI 특유의 틀에 박힌 접속사를 삭제했다
- [ ] 모든 단락의 길이가 균일하지 않다
- [ ] 공허한 형용사('최고 품질', '프리미엄' 등)를 구체적인 표현으로 교체했다
- [ ] 과도한 격식체(~드립니다 남발 등)를 자연스러운 표현으로 수정했다

### 상업적 의도 유지
- [ ] 원본 상품의 강점과 셀링포인트를 보존했다
- [ ] CTA가 명확하고, 이상한 위치로 이동하지 않았다
- [ ] 가격·프로모션 정보를 삭제하지 않았다
- [ ] 타깃 고객층이 변경되지 않았다

### 플랫폼 적합성
- [ ] 대상 플랫폼의 문체 기준에 맞는지 확인했다
- [ ] 글자 수 제한이 있는 경우 적합한지 확인했다
- [ ] 모바일 환경에서의 가독성을 고려했다

### 컴플라이언스
- [ ] 표시·광고법을 위반하는 절대적 표현이 없는가
- [ ] 의약품적 효능·효과 표현이 없는가
- [ ] SNS 게시물의 경우 광고 표기가 필요한지 확인했는가
- [ ] 근거 없는 비교·최상급 표현을 제거했는가

### 전체적인 품질
- [ ] 실제 한국 이커머스 운영자가 쓴 것처럼 들리는가
- [ ] 읽으면 구매 욕구가 생기는가
- [ ] 리스크 있는 표현을 지적하고 대안을 제시했는가
