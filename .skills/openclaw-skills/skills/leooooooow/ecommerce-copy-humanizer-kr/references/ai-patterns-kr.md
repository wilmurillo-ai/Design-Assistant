# AI 카피 패턴 사전 — 한국어 이커머스

## 전형적인 AI 카피 패턴

### 카테고리 1: 과도한 형용사 사용
- 최고 품질 / 최고급 소재
- 프리미엄 / 프리미엄 라인
- 혁신적인 / 혁신적 기술
- 탁월한 / 탁월한 성능
- 완벽한 / 완벽한 마감

### 카테고리 2: 공허한 약속
- 최상의 경험을 선사합니다
- 고객님을 위한 최선의 선택
- 안심하고 사용하실 수 있습니다
- 믿음과 신뢰의 ○○
- 업계를 선도하는 품질

### 카테고리 3: AI 특유의 연결 패턴
- 첫째~둘째~마지막으로
- 또한, ~뿐만 아니라, ~
- 더불어, ~라는 점에서도
- 한편으로는~반면에~
- 말할 필요도 없이~

### 카테고리 4: 과도한 격식체
- ~드립니다 (과도한 사용)
- ~해 드리겠습니다
- 귀하의 요구에 부응하여
- 진심으로 감사드리며
- 불편함을 드려 대단히 죄송합니다

### 카테고리 5: 공허한 약속형
- ~를 실현했습니다
- ~를 이루어 드립니다
- 새로운 라이프스타일을 제안합니다
- 편안한 일상을 지원합니다
- 더 나은 내일을 만들어 드립니다

### 카테고리 6: 과장된 차별화 표현
- 업계 1위
- 국내 최초
- 전례 없는
- 혁명적인
- 게임 체인저
