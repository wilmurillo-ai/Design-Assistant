# 리스크·컴플라이언스 체크리스트 — 한국어 이커머스

## 표시·광고 공정화법 주요 금지 사항

### 허위·과장 광고 (금지)
사실보다 현저히 우수하게 표시하는 광고:
- '국내 1위' '세계 최고' '업계 최초' (근거 없는 경우)
- '100% 효과' '반드시 만족' '무조건 개선'
- '유일한' '전례 없는' (사실과 다른 경우)

### 기만적 광고 (금지)
가격·조건에 대해 유리하게 오인시키는 광고:
- 허위 '정가' (스텔스 인상 후 할인)
- '한정 ○개' 허위 (실제 재고 충분)
- '오늘만' '지금 이 순간만' 허위

## 의약품·건강기능식품 관련 주의 사항

### 화장품·뷰티 제품 금지 표현
- '치료한다' '낫게 한다' '개선한다' (의약품적 효능)
- '○○ 질환에 효과' '질병 예방'
- '세포 재생' '유전자 레벨 개선'

### 식품·건강기능식품 금지 표현
- '○○를 치료한다' '효능·효과'
- '의사 추천' (허위인 경우)
- '식약처 인증' (사실과 다른 경우)

## SNS·인플루언서 관련

### 뒷광고 (스텔스 마케팅) 금지
- PR·광고 게시물에 #광고 #협찬 표기 필수
- '개인 의견입니다'만으로 불충분한 경우 있음
- 기업으로부터 제품 제공 받은 경우 반드시 명기

## 경쟁사 관련

### 비교 광고 규칙
- 사실에 근거한 명확한 비교는 OK
- 근거 없이 '○○보다 우수하다'는 불가
- 타사 브랜드명을 무단으로 사용한 비교는 상표권 침해 위험

## 안전 체크리스트 (재작성 후 확인)

- [ ] 최상급 표현에 근거가 있는가?
- [ ] 효과·효능은 증명 가능한가 (시험 데이터 등)?
- [ ] 가격 표시는 정확한가 (이중 가격 문제 없음)?
- [ ] '한정' '오늘만'은 사실인가?
- [ ] 의약품적인 효능·효과 표현이 없는가?
- [ ] SNS 게시물에 광고 표기가 필요한지 확인했는가?
- [ ] 경쟁사 비교는 사실에 근거하는가?
- [ ] 개인정보·프라이버시 관련 언급이 없는가?
