# 微信自动化部署步骤

## 1. 安装 Python

**方法 A：用 winget 安装（推荐）**
```powershell
winget install Python.Python.3.11
```

**方法 B：手动下载**
1. 访问 https://www.python.org/downloads/
2. 下载 Python 3.11
3. 安装时勾选 "Add Python to PATH"

## 2. 安装依赖

安装完 Python 后，打开**新的 PowerShell 窗口**（让 PATH 生效），然后运行：

```powershell
cd C:\Users\admin\.openclaw\workspace\wechat-automation-api
pip install -r requirements.txt
```

## 3. 配置文件

```powershell
# 复制配置文件
Copy-Item config.json.example config.json

# 编辑配置文件（设置 token）
notepad config.json
```

在 config.json 里修改 token：
```json
{
    "token": "你的秘密 token（随便设一个，记住就行）",
    "host": "127.0.0.1",
    "port": 8808,
    "message_interval": 1,
    "log_level": "INFO"
}
```

## 4. 启动服务

```powershell
# 确保微信 Windows 客户端已登录
run.bat
```

看到以下提示表示成功：
```
========================================
微信自动化 HTTP 服务启动
服务地址：http://127.0.0.1:8808
API 端点：POST http://127.0.0.1:8808/
========================================
```

## 5. 测试发送

```powershell
$body = @{
    token = "你的 token"
    action = "sendtext"
    to = @("文件传输助手")
    content = "你好，这是自动发送的消息"
} | ConvertTo-Json

Invoke-RestMethod -Uri "http://127.0.0.1:8808/" -Method Post -Body $body -ContentType "application/json"
```

---

**项目位置：** `C:\Users\admin\.openclaw\workspace\wechat-automation-api`

**注意事项：**
- 必须保持微信 Windows 客户端登录
- 微信窗口不能最小化到托盘（可以最小化到任务栏）
- 联系人名字必须和微信里显示的一模一样（区分大小写）
