# 建仓评估与策略参考

## 建仓条件判断框架

### 1. 多维度评估模型

对每个候选标的从以下维度评分（1-5分）：

| 维度 | 权重 | 评估标准 |
|------|------|----------|
| 估值水平 | 25% | PE/PB历史分位 |
| 短期跌幅 | 20% | 近1月/3月跌幅 |
| 技术面 | 20% | 均线支撑、MACD |
| 资金面 | 15% | 主力资金流向 |
| 基本面 | 20% | 业绩预期、行业景气度 |

**综合评级**:
- 4.5-5.0: 🟢 强烈建议建仓
- 3.5-4.4: 🟡 建议建仓
- 2.5-3.4: 🔵 继续观察
- <2.5: 🔴 暂不建议

### 2. 定量建仓条件

**基础条件**（必须满足）:
```
1. 单日跌幅 > 3% 或
2. 连续3日累计跌幅 > 5% 或
3. 触及历史估值低位（PE < 30%分位）
```

**加分条件**:
```
+ 大盘整体下跌（系统性风险）
+ 板块轮动调整（非个股问题）
+ 业绩预增但股价下跌（错杀）
+ 技术指标超卖（RSI < 30）
```

**减分条件**:
```
- 个股利空消息
- 业绩下滑预期
- 行业政策风险
- 资金持续流出
```

### 3. 分批建仓策略

**策略A: 等比例加仓**
```
首次建仓: 计划资金的 50%
跌3%加仓: 计划资金的 30%
跌8%加仓: 计划资金的 20%
```

**策略B: 金字塔加仓**（推荐）
```
首次建仓: 500元 (观察仓)
跌3%加仓: 1000元 (初步建仓)
跌8%加仓: 1500元 (完成建仓)
总计: 3000元
```

**策略C: 定投式建仓**
```
每周固定金额买入
不管涨跌持续建仓
适合: 宽基指数ETF
```

## 具体标的管理

### ETF建仓管理

**宽基ETF** (沪深300/中证500):
- 建仓时机: 大盘下跌3%以上
- 加仓策略: 每跌3%加仓一次
- 止盈策略: 持有为主，长期配置

**行业ETF** (电池/消费/科技):
- 建仓时机: 板块调整+估值低位
- 加仓策略: 分批建仓，控制仓位
- 止盈策略: 板块轮动时止盈

**跨境ETF** (恒生科技/纳指):
- 建仓时机: 汇率+估值双重低位
- 加仓策略: 考虑汇率因素
- 止盈策略: 汇率反向时减仓

### 个股建仓管理

**银行股**:
- 估值锚定: PB < 1.0
- 股息率: > 5%
- 建仓时机: 板块集体调整

**成长股**:
- 估值锚定: PE < 行业平均
- 业绩增速: > 20%
- 建仓时机: 业绩发布后错杀

**周期股**:
- 估值锚定: 周期底部PB
- 价格锚定: 商品价格低位
- 建仓时机: 周期反转初期

## 常见建仓误区

### 误区1: 追高建仓
**问题**: 股价上涨时才想起建仓
**解决**: 制定明确的建仓价位，严格执行

### 误区2: 一次性满仓
**问题**: 首次建仓即投入全部资金
**解决**: 分批建仓，保留弹药

### 误区3: 补仓变加仓
**问题**: 亏损股票不断补仓，越套越深
**解决**: 设置止损线，严格止损

### 误区4: 忽视仓位管理
**问题**: 单个标的仓位过重
**解决**: 单个标的仓位不超过总仓位20%

### 误区5: 情绪化决策
**问题**: 恐惧时不敢买，贪婪时追买
**解决**: 制定规则，机械执行

## 建仓检查清单

建仓前确认：
- [ ] 是否满足定量条件（跌幅/估值）
- [ ] 是否有加分因素支持
- [ ] 是否排除重大利空
- [ ] 仓位是否在控制范围内
- [ ] 资金是否充裕

建仓后确认：
- [ ] 记录建仓价格和数量
- [ ] 设定加仓线
- [ ] 设定止盈线
- [ ] 设定止损线
- [ ] 更新MEMORY.md

## 实战案例

### 案例1: 恒生科技ETF建仓

**背景**: 2026年4月，恒生科技指数持续下跌

**评估**:
| 维度 | 评分 | 说明 |
|------|------|------|
| 估值水平 | 5 | PE处于历史4%分位 |
| 短期跌幅 | 4 | 近1月跌幅超15% |
| 技术面 | 3 | 超跌但无明确支撑 |
| 资金面 | 3 | 南向资金持续流入 |
| 基本面 | 4 | 互联网政策回暖 |
| **综合** | **3.8** | 🟡 建议建仓 |

**执行**:
- 首次建仓: 4,000份 @ ¥0.640
- 加仓计划: ¥0.600加仓2,000份

**结果**: 建仓后上涨1.1%，持仓盈利

### 案例2: 人工智能ETF止盈

**背景**: 建仓后涨幅达5.9%

**决策**:
- 原本计划长期持有
- 但连续4日资金净流出
- 决定止盈清仓

**执行**:
- 清仓: 3,200份 @ ¥1.550
- 盈利: +¥277 (+5.9%)

**复盘**: 及时止盈规避后续回调，资金回笼等待更好机会
