### ✅ 5. 字幕样式可配置

**需求**: 支持自定义字幕样式

**实现**:
- [x] JSON 格式配置文件
- [x] 字体名称可配置
- [x] 字号可配置
- [x] 颜色可配置（BGR 格式）
- [x] 粗体/斜体/下划线
- [x] 边框样式（框/阴影）
- [x] 轮廓宽度
- [x] 阴影深度
- [x] 对齐方式（9 种位置）
- [x] 边距配置
- [x] 透明度配置

**配置文件**:
- `subtitle_style_template.json` - 完整模板
- `demo_style.json` - 黄色综艺字幕演示

**样式参数**（18 个）:
```json
{
  "font": "Arial",
  "fontsize": 24,
  "primary_color": "&H00FFFFFF",
  "outline_color": "&H00000000",
  "back_color": "&H00000000",
  "bold": 0,
  "italic": 0,
  "underline": 0,
  "strikeout": 0,
  "border_style": 4,
  "outline": 2,
  "shadow": 0,
  "alignment": 2,
  "margin_l": 10,
  "margin_r": 10,
  "margin_v": 50,
  "alpha": "00"
}
```

**使用方法**:
```bash
python scripts/subtitle_processor.py \
  -t "transcript_fixed.txt" \
  -v "video.mp4" \
  -o "./output" \
  -s "my_style.json"
```

**测试状态**: ✅ 已验证  
**演示样式**: 微软雅黑 28px 黄色粗体字幕

## 🚀 使用示例

### 场景 1: 快速处理

```bash
# 使用 base 模型快速处理
python scripts/video_processor.py -i "video.mp4" -m "base"
```

### 场景 2: 高精度处理

```bash
# 使用 medium 模型高精度处理
python scripts/video_processor.py -i "video.mp4" -m "medium"
```

### 场景 3: 修正错别字

```bash
# 1. 编辑 transcript.txt
# 2. 保存为 transcript_fixed.txt
# 3. 重新生成
python scripts/subtitle_processor.py \
  -t "transcript_fixed.txt" \
  -v "video.mp4" \
  -o "./final"
```

### 场景 4: 自定义样式

```bash
# 1. 创建样式配置
copy subtitle_style_template.json my_style.json
# 2. 编辑 my_style.json
# 3. 使用样式重新生成
python scripts/subtitle_processor.py \
  -t "transcript_fixed.txt" \
  -v "video.mp4" \
  -o "./final" \
  -s "my_style.json"
```

## 📝 常见错别字修正

| 识别错误 | 正确写法 | 出现频率 |
|----------|----------|----------|
| 博兰会 | 博览会 | 高 |
| 直观 | 乌镇 | 高 |
| 行榜 | 新榜 | 中 |
| 穿带式 | 穿戴式 | 中 |
| 外古隔 | 外骨骼 | 中 |
| 訪生手 | 仿生手 | 中 |
| 直光 | 之光 | 低 |
| 形容 | 应用 | 低 |
| 抢战 | 抢占 | 低 |
| 分口 | 风口 | 低 |

**建议**: 使用 medium 模型可大幅减少错别字
