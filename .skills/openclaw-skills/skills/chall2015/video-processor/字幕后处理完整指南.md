### 完整参数说明

```json
{
  "font": "Arial",              // 字体名称
  "fontsize": 24,               // 字号（像素）
  "primary_color": "&H00FFFFFF", // 文字颜色（BGR 格式）
  "outline_color": "&H00000000", // 轮廓颜色
  "back_color": "&H00000000",    // 背景颜色
  "bold": 0,                    // 粗体 (0/1)
  "italic": 0,                  // 斜体 (0/1)
  "underline": 0,               // 下划线 (0/1)
  "border_style": 4,            // 边框样式 (1=框，4=阴影)
  "outline": 2,                 // 轮廓宽度
  "shadow": 0,                  // 阴影深度
  "alignment": 2,               // 对齐方式 (2=中下，8=中上)
  "margin_l": 10,               // 左边距
  "margin_r": 10,               // 右边距
  "margin_v": 50,               // 垂直边距
  "alpha": "00"                 // 透明度 (00=不透明)
}
```

### 常用颜色代码

| 颜色 | 代码 |
|------|------|
| 白色 | `&H00FFFFFF` |
| 黑色 | `&H00000000` |
| 黄色 | `&H0000FFFF` |
| 红色 | `&H000000FF` |
| 绿色 | `&H0000FF00` |
| 蓝色 | `&H00FF0000` |

### 对齐方式

```
7  8  9  (顶部)
4     6  (中部)
1  2  3  (底部)

常用:
2 = 中下 (默认)
8 = 中上
```

## 💡 实用技巧

### 1. 批量查找替换

在编辑器中批量修正：
- `博兰会` → `博览会`
- `穿带式` → `穿戴式`
- `行榜` → `新榜`

### 2. 调整字幕显示时间

如果某段字幕太短：

**修改前**：
```
[00:03 - 00:05] 快速闪过的一句话
```

**修改后**：
```
[00:03 - 00:08] 快速闪过的一句话
```

### 3. 拆分长句

**修改前**：
```
[00:12 - 00:20] 这里是互联网博览会的现场我们看到了很多 AI 设备
```

**修改后**：
```
[00:12 - 00:16] 这里是互联网博览会的现场
[00:16 - 00:20] 我们看到了很多 AI 设备
```

### 4. 添加标点符号

**修改前**：
```
[00:03 - 00:12] 大家好今天我们分享一个有趣的话题
```

**修改后**：
```
[00:03 - 00:12] 大家好，今天我们分享一个有趣的话题！
```

## ⚠️ 注意事项

1. **时间戳格式**: 保持 `[MM:SS - MM:SS]` 格式
2. **文件编码**: 使用 UTF-8 编码保存
3. **视频路径**: 使用原始视频文件路径
4. **样式配置**: JSON 格式，注意引号和逗号
5. **字体名称**: 使用系统中已安装的字体
