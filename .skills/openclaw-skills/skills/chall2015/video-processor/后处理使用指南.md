# 字幕后处理使用指南

**版本**: v2.1  
**功能**: 修改文字稿后重新生成字幕和视频

---

## 🎯 使用场景

1. **修正错别字** - 语音识别有个别错误
2. **优化断句** - 调整字幕分段
3. **添加标点** - 完善文字稿标点符号
4. **自定义样式** - 调整字幕字体、颜色、位置

---

## 📝 工作流程

```
1. 初次处理视频
   ↓
2. 编辑 transcript.txt（修正错别字）
   ↓
3. 运行后处理器
   ↓
4. 生成新的字幕和视频
```

---

## 🚀 快速开始

### 步骤 1: 初次处理视频

```bash
python scripts/video_processor.py -i "video.mp4" -m "medium"
```

输出目录：`output_medium/video_20260324_103053/`

### 步骤 2: 编辑文字稿

打开 `transcript.txt`，修改错别字：

**修改前**：
```
[00:03 - 00:12] 这里是互联网直观博览会的现场
```

**修改后**：
```
[00:03 - 00:12] 这里是互联网乌镇博览会的现场
```

**格式说明**：
- 保留时间戳：`[MM:SS - MM:SS]`
- 只修改文字内容
- 可以调整时间戳
- 可以添加/删除段落

### 步骤 3: 重新生成

```bash
python scripts/subtitle_processor.py \
  -t "output_medium/video_20260324_103053/transcript.txt" \
  -v "video.mp4" \
  -o "./regenerated"
```

### 步骤 4: 使用自定义样式（可选）

```bash
# 1. 创建样式模板
python scripts/subtitle_processor.py --create-style-template

# 2. 编辑 subtitle_style_template.json

# 3. 使用自定义样式重新生成
python scripts/subtitle_processor.py \
  -t "output_medium/video_20260324_103053/transcript_modified.txt" \
  -v "video.mp4" \
  -o "./regenerated" \
  -s "subtitle_style.json"
```

## 🎨 字幕样式配置

### 常用样式示例

#### 1. 白色文字（默认）

```json
{
  "font": "Arial",
  "fontsize": 24,
  "primary_color": "&H00FFFFFF",
  "alignment": 2,
  "margin_v": 50
}
```

#### 2. 黄色综艺字幕

```json
{
  "font": "Microsoft YaHei",
  "fontsize": 28,
  "primary_color": "&H0000FFFF",
  "bold": 1,
  "border_style": 1,
  "outline": 3,
  "alignment": 2
}
```

#### 3. 顶部字幕

```json
{
  "font": "Arial",
  "fontsize": 24,
  "primary_color": "&H00FFFFFF",
  "alignment": 8,
  "margin_v": 50
}
```

#### 4. 电影效果（黑边白字）

```json
{
  "font": "Arial",
  "fontsize": 22,
  "primary_color": "&H00FFFFFF",
  "border_style": 1,
  "outline": 2,
  "outline_color": "&H00000000",
  "alignment": 2,
  "margin_v": 40
}
```

#### 5. 思源黑体（中文优化）

```json
{
  "font": "Source Han Sans CN",
  "fontsize": 26,
  "primary_color": "&H00FFFFFF",
  "border_style": 4,
  "shadow": 1,
  "alignment": 2,
  "margin_v": 50
}
```

---

## 📋 样式参数详解

| 参数 | 说明 | 示例值 |
|------|------|--------|
| `font` | 字体名称 | Arial, Microsoft YaHei, SimHei |
| `fontsize` | 字号（像素） | 24, 28, 32 |
| `primary_color` | 文字颜色（BGR） | `&H00FFFFFF` (白) |
| `outline_color` | 轮廓颜色 | `&H00000000` (黑) |
| `back_color` | 背景颜色 | `&H00000000` (透明) |
| `bold` | 粗体 | 0 (关闭), 1 (开启) |
| `italic` | 斜体 | 0, 1 |
| `border_style` | 边框样式 | 1 (框), 4 (阴影) |
| `outline` | 轮廓宽度 | 1, 2, 3 |
| `shadow` | 阴影深度 | 0, 1, 2 |
| `alignment` | 对齐方式 | 2 (中下), 8 (中上) |
| `margin_v` | 垂直边距 | 40, 50, 60 |

### 对齐方式

```
7  8  9  (顶部)
4     6  (中部)
1  2  3  (底部)

常用：
2 = 中下（默认）
8 = 中上
10 = 右上
```

### 颜色格式

**BGR 格式**（蓝 - 绿-红）：
- `&H00FFFFFF` = 白色
- `&H00000000` = 黑色
- `&H0000FFFF` = 黄色
- `&H00FF0000` = 蓝色
- `&H0000FF00` = 绿色
- `&H000000FF` = 红色

## 💡 实用技巧

### 1. 批量修正错别字

在编辑器中使用查找替换：
- `博兰会` → `博览会`
- `穿带式` → `穿戴式`
- `行榜` → `新榜`

### 2. 调整字幕分段

如果某段太长，可以拆分：

**修改前**：
```
[00:12 - 00:20] 这里是互联网博览会的现场我们看到了很多 AI 设备
```

**修改后**：
```
[00:12 - 00:16] 这里是互联网博览会的现场
[00:16 - 00:20] 我们看到了很多 AI 设备
```

### 3. 添加标点符号

**修改前**：
```
[00:03 - 00:12] 大家好今天我们分享一个有趣的话题
```

**修改后**：
```
[00:03 - 00:12] 大家好，今天我们分享一个有趣的话题！
```

### 4. 调整显示时间

如果字幕显示时间太短：

**修改前**：
```
[00:03 - 00:05] 快速闪过的一句话
```

**修改后**：
```
[00:03 - 00:08] 快速闪过的一句话
```

---

## ⚠️ 注意事项

1. **时间戳格式**：保持 `[MM:SS - MM:SS]` 格式
2. **文件编码**：使用 UTF-8 编码保存
3. **视频路径**：使用原始视频文件
4. **样式配置**：JSON 格式，注意引号和逗号

## 🎯 推荐工作流

```
1. 使用 medium 模型初次处理
   ↓
2. 快速浏览 transcript.txt
   ↓
3. 修正明显错别字（5-10 分钟）
   ↓
4. 应用自定义样式（可选）
   ↓
5. 重新生成最终视频
   ↓
6. 检查并输出
```

**总耗时**：约 10-15 分钟（2-3 分钟视频）
