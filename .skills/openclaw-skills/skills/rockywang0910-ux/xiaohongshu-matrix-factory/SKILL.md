# 小红书内容矩阵工厂

5分钟生成一周高质量内容，支持批量账号矩阵运营。

## 触发词

当用户提到以下内容时激活此skill：
- "做小红书内容"
- "批量生成小红书文案"
- "小红书矩阵"
- "生成一周内容"
- "生成小红书文案"
- "写小红书笔记"
- "批量生产内容"
- "MCN内容工厂"

---

## 核心能力

### 1. 爆款选题引擎

根据以下输入，自动生成当周高潜选题：
- 目标行业/产品
- 目标人群画像
- 竞品账号分析（可选）
- 当前热点话题

**输出：** 7个选题，附爆款概率评分 + 选题依据

### 2. AI文案工厂

**支持内容类型：**
- 产品种草文（突出卖点+真实感）
- 干货教程文（有用+可收藏）
- 痛点共鸣文（制造情绪共鸣）
- 品牌故事文（有温度+信任感）
- 热点蹭流文（借势热点+快速发布）
- 对比测评文（专业感+说服力）

**每篇文案包含：**
- 爆款标题（3个备选，含标题公式说明）
- 正文（平台原生风格，有情绪钩子）
- 互动引导（引导评论、收藏、关注）
- 推荐发布时间段
- 9个精准标签

### 3. AI配图Prompt工厂

每篇文案自动生成配套的5张配图Prompt：
- 封面图Prompt（高点击率）
- 内容图Prompt（内容相关）
- 九宫格排版建议
- 适用于Midjourney / Stable Diffusion / DALL-E

### 4. 批量矩阵模式

**输入：** 多个账号定位（如：美妆矩阵、母婴矩阵、职场矩阵）
**输出：** 每个账号独立的一周内容日历，相互差异化，避免内容重复

## 工作流程

### Step 1：收集输入（30秒）

使用上述模板收集信息。
信息越完整，内容质量越高。
如果用户未提供某项，使用该品类的最佳默认值。

### Step 2：选题生成（1分钟）

根据产品/行业，生成7个差异化选题，附：
- 选题类型（种草/干货/痛点/热点/品牌）
- 爆款潜力评分（1-10分）
- 预计互动率
- 为什么这个选题会火

### Step 3：文案生成（3-4分钟）

逐篇生成高质量文案，每篇包含：
- 标题（3个，含各自适合的封面风格）
- 正文（300-800字，平台原生风格）
- 标签（9个，精准长尾）
- 发布时间建议
- 配图Prompt（5张/篇，共35张）

### Step 4：汇总交付（30秒）

整合所有内容为完整周度内容包，格式：
- 内容日历表格
- 每篇内容的标题+正文+标签+配图Prompt
- 发布检查清单

## 爆款标题公式库

当用户需要为内容写标题时，自动应用以下公式：

```
公式1（痛点+解决）：[数字]个[人群]才知道的[痛点]解决方案
公式2（结果前置）：月薪3万的人，都在用[产品]
公式3（好奇冲突）：为什么[普遍认知]其实是错的？
公式4（身份认同）：[人群]不会告诉你的[行业真相]
公式5（热点借势）：[热点事件]背后，藏着什么[商业机会]？
公式6（数字清单）：[N个]你不知道的[领域]秘密
公式7（对比反差）：从[糟糕体验]到[完美体验]，只差[产品]
```

## 质量标准

- 文案必须符合小红书社区公约，不得包含违禁词
- 不得捏造虚假用户体验
- 种草内容需注明"含品牌合作"或"个人分享"标识
- 所有内容需有明显价值导向，不发纯广告
- 不得侵犯他人知识产权（配图需用AI生成或正版图库）
