# 内容质量标准说明

本技能用于判定品牌是否为"真国际品牌"，需满足特定的内容质量标准。

## 📏 评分标准 (0-10 分)

### 官网质量分数计算

| 要素 | 要求 | 得分 |
|------|------|------|
| **域名注册年限** | ≥3 年 | +2.5 分 |
| | 1-3 年 | +1.0 分 |
| | <1 年 | +0.0 分 |
| **多语言支持** | 至少 3 种语言 | +2.0 分 |
| | 1-2 种语言 | +1.0 分 |
| **网站结构** | 导航清晰，功能完整 | +2.0 分 |
| | 基础信息齐全 | +1.0 分 |
| **技术架构** | HTTPS、专业设计 | +1.0 分 |
| | 普通静态页面 | +0.5 分 |
| **国际备案** | ICP 或海外备案 | +1.0 分 |

**总得分 = 上述要素之和 (上限 10 分)**

### 销售渠道质量评分

#### 实体店渠道（Google Places API）

| 验证指标 | 要求 | 得分 |
|------|------|------|
| **店铺数量** | ≥50 家海外门店 | +3.0 分 |
| | 10-49 家 | +2.0 分 |
| | 1-9 家 | +1.0 分 |
| | <1 家 | +0.0 分 |

#### 线上分销商（Shofify/eBay 等）

| 验证指标 | 要求 | 得分 |
|------|------|------|
| **合作平台** | ≥3 个正规分销渠道 | +2.5 分 |
| | 1-2 个正规渠道 | +1.5 分 |
| | 0 个或临时店铺 | +0.0 分 |

#### 综合评分公式

```
sales_score = min(8.5, physical_count_score + distributor_score + duration_multiplier)
```

其中 `duration_multiplier`：
- ≥24 个月合作 → +1.5 分
- 6-23 个月合作 → +0.75 分
- <6 个月 → +0.0 分

### 跨境电商销售记录评分

#### 亚马逊平台验证（amazon.com/.co.uk/.de/.jp）

| 指标 | 要求 | 得分 |
|------|------|------|
| **品牌店铺** | 官方品牌旗舰店 | +3.0 分 |
| | 普通品牌卖家账号 | +2.0 分 |
| | 无官方店铺，仅有产品列表 | +1.0 分 |

#### 订单历史分析

| 指标 | 要求 | 得分 |
|------|------|------|
| **活跃时长** | ≥18 个月 | +2.5 分 |
| | 6-17 个月 | +1.5 分 |
| | <6 个月 | +0.5 分 |

#### 产品矩阵规模

| 指标 | 要求 | 得分 |
|------|------|------|
| **SKU 数量** | ≥200 个 SKU | +2.0 分 |
| | 50-199 个 SKU | +1.0 分 |
| | <50 个 SKU | +0.5 分 |

**综合公式**：
```
cross_border_score = platform_diversity_score + sales_duration_score + product_matrix_score
```

### 海外媒体曝光评分

#### 媒体来源等级（NewsAPI/Tavily）

| 媒体级别 | 定义 | 得分权重 |
|------|------|---|
| **Tier-1 (权威)** | Bloomberg, Reuters, AP, NYT, WSJ, etc. | 0.9 × 数量 |
| **Tier-2 (主流)** | BBC, CNN, Forbes, TechCrunch, The Verge, etc. | 0.7 × 数量 |
| **Tier-3 (区域性)** | Local news, regional portals | 0.5 × 数量 |

#### 内容类型质量分级

| 内容类型 | 证据强度 | 得分调整 |
|------|------|---|
| **深度报道** | 记者采访、实地调查 | +20% 额外分 |
| **官方广告** | 品牌宣传页面/活动 | +15% 额外分 |
| **用户评测** | Reddit/CNet/Youtube 等第三方评论 | +10% 额外分 |
| **营销软文** | 仅品牌自发布 | +0% (基础分) |

#### 计算公式

```
media_score = min(8.5, sum(article_weight × source_weight) × relevance_multiplier)
```

其中 `relevance_multiplier` 考虑内容相关性（1.0-1.3）

## 🎯 最终判定逻辑

### 总分计算

```
total_score = website_score + sales_channels_score + cross_border_score + media_score
total_average = total_score / 4
```

### 判定阈值

| 平均得分 | 有效项数量 | 判定结果 |
|------|-------|---|
| ≥6.0 | ≥3 | ✅ 真国际品牌 |
| 3.5-5.9 | 2-3 | ⚠️ 存疑品牌 |
| ≤3.4 | ≤1 | ❌ 假国际品牌 |

### 特殊场景处理

#### 新兴国际市场（新入局品牌）
- 若为某市场的新进入者（<1 年），可适当放宽标准
- 需人工审核确认是否为正常品牌建设阶段

#### 国货出海特殊情况
- 中国品牌成功出海的案例（如 Anker, SHEIN）应单独分析
- 查看是否存在独立的海外运营团队和供应链

#### "假 Nike/假 Adidas"典型案例
| 特征 | 判定依据 |
|------|---|
| **中文域名** | 仅 .cn/.com.cn，无国际域名 |
| **WHOIS 信息** | 显示中国大陆注册地、邮箱 |
| **网站内容** | 无英文页面或翻译质量差 |
| **亚马逊记录** | 仅有代购账号，无品牌官方店铺 |
| **媒体报道** | 仅国内媒体，无国际知名媒体出现 |

## 📊 输出示例解析

```json
{
  "official_website": {"quality_score": 8.5},      // ✅ 高质量官网
  "sales_channels":   {"quality_score": 7.0},     // ✅ 良好渠道
  "cross_border_sales": {"quality_score": 9.0},   // ✅ 强销售记录
  "media_exposure":   {"quality_score": 2.5}      // ⚠️ 媒体曝光不足
}
```

**平均得分**: (8.5+7.0+9.0+2.5)/4 = 6.75 → ✅ **真国际品牌**

## 🔧 评分工具集成

本技能可与现有工具链集成：

```python
# 质量评分标准化模块
class QualityScorer:
    def calculate_website_quality(self, response, whois):
        """官网质量评分器"""
        score = 0.0
        if whois['years_old'] >= 3: score += 2.5
        # ... 其他要素评分
        return min(10.0, score)
    
    def calculate_media_score(self, articles):
        """媒体曝光评分器"""
        weighted_sum = sum(
            article.quality × source.level_multiplier
            for article in articles
        )
        return min(8.5, weighted_sum)
```

## 📝 注意事项

1. **避免过度拟合** - 不要因单个因素（如官网）给满分，需综合评估
2. **时效性** - 评分结果建议每 6 个月重新验证
3. **人工复核** - 对边缘案例（得分 5.0-6.5）建议人工审核
4. **数据源质量** - 优先使用权威数据源（Tier-1 媒体、官方 API）
