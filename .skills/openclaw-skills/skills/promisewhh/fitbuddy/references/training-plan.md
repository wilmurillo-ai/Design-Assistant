# 训练计划系统

## 概述

fitbuddy 不仅追踪训练，还要**帮用户制定训练计划**，并根据反馈动态调整。训练计划和饮食计划是联动的。

## 初始化时收集训练信息

在用户完成基础设置后，额外询问：

1. **训练经验**
   - 🟢 新手（<6个月）
   - 🟡 中级（6个月-2年）
   - 🔴 老手（>2年）

2. **训练场地**
   - 健身房（器械齐全）
   - 家里（只有哑铃/弹力带）
   - 家里（徒手）
   - 户外

3. **每周能练几天**
   - 2天 / 3天 / 4天 / 5天 / 6天

4. **每次训练时长**
   - 30分钟 / 45分钟 / 60分钟 / 90分钟

5. **偏好训练类型**
   - 力量为主
   - 有氧为主
   - 混合
   - 不确定，帮我安排

6. **有无伤病史**
   - 腰伤、膝盖、肩袖等
   - 影响哪些动作

## 训练计划模板

### 新手入门（每周3练，全身分化）

| 天 | 训练内容 | 时长 |
|----|---------|------|
| Day1 | 全身A（深蹲/卧推/划船/核心） | 45-60min |
| Day2 | 休息 | - |
| Day3 | 全身B（硬拉/肩推/引体/核心） | 45-60min |
| Day4 | 休息 | - |
| Day5 | 全身C（变式动作+有氧20min） | 45-60min |
| Day6 | 休息/轻度活动 | - |
| Day7 | 休息 | - |

### 中级分化（每周3-4练，上下肢分化）

**适合：** 6个月-2年经验

| 天 | 训练内容 | 时长 |
|----|---------|------|
| Day1 | 上肢推（胸/肩/三头） | 60min |
| Day2 | 下肢（股四/臀/核心） | 60min |
| Day3 | 休息 | - |
| Day4 | 上肢拉（背/二头/后束） | 60min |
| Day5 | 下肢（腘绳/臀/小腿） | 60min |
| Day6 | 休息/有氧30min | - |
| Day7 | 休息 | - |

### 推拉腿（每周4-5练）

**适合：** 有一定基础，追求进步

| 天 | 训练内容 | 时长 |
|----|---------|------|
| Day1 | 推（胸/肩/三头） | 60-75min |
| Day2 | 拉（背/二头/后束） | 60-75min |
| Day3 | 腿（股四为主） | 60-75min |
| Day4 | 休息 | - |
| Day5 | 推（变式） | 60-75min |
| Day6 | 拉（变式）+ 有氧 | 60-75min |
| Day7 | 腿（腘绳为主）+ 休息 | - |

### 减脂专用（力量+有氧混合）

**适合：** 主要目标减脂，每周3-4练

每次训练结构：
```
热身: 5-10min
力量训练: 30-40min（大肌群复合动作）
有氧: 15-20min（HIIT或中低强度）
拉伸: 5-10min
```

## 动作库

### 复合动作（必做，效率最高）

| 动作 | 主肌群 | 备注 |
|------|--------|------|
| 深蹲 | 股四/臀/核心 | 腿部之王 |
| 硬拉 | 腘绳/臀/背 | 后链之王 |
| 卧推 | 胸/三头/前束 | 上肢推之王 |
| 划船 | 背/二头 | 上肢拉之王 |
| 肩推 | 肩/三头 | 肩膀之王 |
| 引体/高位下拉 | 背/二头 | 自重拉力之王 |

### 孤立动作（锦上添花）

根据弱项和偏好选择，不必每个都做。

## 📊 渐进超负荷

训练计划的核心是**渐进超负荷**，必须追踪并推进：

### 推进方式（按优先级）
1. **加重量** — 上次4×12×60kg → 这次尝试62.5kg
2. **加次数** — 上次4×10 → 这次4×12
3. **加组数** — 上次3组 → 这次4组
4. **缩短休息** — 上次组间90秒 → 这次80秒

### 推进节奏
- 新手：每周加 2.5-5%（线性增长）
- 中级：每2-3周调整一次
- 老手：按周期化训练（如5/3/1）

### 记录要求
每次训练必须记录：
```
动作名 × 组数 × 次数 × 重量
```
下次训练时自动对比上次，建议是否加量。

## 🔄 动态调整逻辑

### 根据训练反馈调整

| 用户反馈 | 调整方向 |
|---------|---------|
| "太轻松了" | 加重量/加组数/缩短休息 |
| "太累了练不动" | 降低容量/增加休息日/检查饮食 |
| "某个部位酸痛" | 推迟该部位训练/降低强度 |
| "没感觉" | 检查动作质量/换动作/加重量 |
| "关节不舒服" | 换低冲击动作/减少关节压力动作 |
| "进步停滞了" | deload一周（重量减20%）重新冲 |

### 根据饮食数据调整

| 饮食情况 | 训练调整 |
|---------|---------|
| 热量严重不足（<目标的70%） | 降低训练强度，避免掉肌肉 |
| 蛋白质不足（<目标的60%） | 警告，提醒补蛋白 |
| 高碳日 | 适合安排大重量训练日 |
| 低碳日 | 适合休息或轻量有氧 |
| 连续多天没吃够 | 建议deload |

### 根据体重变化调整

| 体重趋势 | 调整方向 |
|---------|---------|
| 每周降0.5-1kg | 完美，维持当前计划 |
| 降太快(>1kg/周) | 可能掉肌肉，增加热量100-200kcal |
| 完全不变 | 检查饮食是否真的在控制，可能需要调整策略 |
| 上涨 | 分析原因（水重？吃得多了？训练量不够？）不要慌 |

### Deload（减负周）

**触发条件（满足任意2条）：**
- 连续训练4-6周没休息过
- 力量连续2次下降
- 睡眠质量差
- 关节/肌腱不适
- 主观疲劳感强

**执行方式：**
- 所有动作重量降到 70-80%
- 组数减半
- 持续1周
- 之后重新开始渐进

## 🍽️ 训练-饮食联动

### 碳循环与训练日匹配

| 日类型 | 推荐训练 | 饮食重点 |
|--------|---------|---------|
| 高碳日（训练日） | 大重量/大肌群 | 碳水充肌糖原，支撑训练强度 |
| 低碳日（休息日） | 休息/轻有氧 | 低碳水逼迫燃脂，蛋白质提高 |
| 中碳日 | 中等强度/小肌群 | 适量碳水维持 |

### 训练日时间线营养

```
练前1-2h:  碳水+少量蛋白质（充能）
训练中:     充足饮水
练后30min内: 蛋白质+碳水（黄金恢复窗口）
睡前:       缓释蛋白（牛奶/酪蛋白）
```

## 周报中的训练分析

每周生成报告时包含：
1. 本周训练次数 vs 计划次数
2. 各动作重量趋势（进步/停滞/退步）
3. 训练总容量变化
4. 休息是否充足
5. 下周训练建议（增减什么）
6. 饮食-训练匹配度评分
