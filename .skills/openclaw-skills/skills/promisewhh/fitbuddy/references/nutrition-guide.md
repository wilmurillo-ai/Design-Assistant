# 营养知识问答指南

## 基础概念

### 三大营养素

**蛋白质（Protein）** — 4 kcal/g
- 作用：修复肌肉、维持免疫系统、合成激素
- 优质来源：鸡胸肉、牛肉、鱼虾、鸡蛋、牛奶、豆腐、蛋白粉
- 减脂建议：每公斤体重 2-2.5g/天
- 1个鸡蛋 ≈ 6g 蛋白质
- 100g 鸡胸肉（生） ≈ 23g 蛋白质
- 100g 牛肉（生） ≈ 20-26g 蛋白质
- 100g 虾仁（生） ≈ 20g 蛋白质
- 100ml 牛奶 ≈ 3.2g 蛋白质
- 1勺蛋白粉 ≈ 25g 蛋白质

**碳水化合物（Carbs）** — 4 kcal/g
- 作用：主要能量来源，支撑训练强度
- 分类：
  - 简单碳水（快吸收）：白糖、水果、蜂蜜 → 训练后补充
  - 复合碳水（慢吸收）：糙米、红薯、燕麦、全麦面包 → 日常主食
- 常见食物碳水含量（每100g生重）：
  - 白米饭 ≈ 28g（煮熟后约25%含水量，100g熟饭≈35g碳水）
  - 糙米 ≈ 75g（生）
  - 红薯 ≈ 20g
  - 土豆 ≈ 17g
  - 燕麦 ≈ 66g
  - 全麦面包 ≈ 41g
  - 面条（干）≈ 72g
  - 馒头 ≈ 47g

**脂肪（Fat）** — 9 kcal/g
- 作用：激素合成、脂溶性维生素吸收、保护器官
- 分类：
  - 好脂肪：橄榄油、坚果、牛油果、鱼油
  - 需控制：动物油脂、黄油
  - 尽量避免：反式脂肪（油炸、人造奶油）
- 常见食物脂肪含量（每100g）：
  - 植物油 ≈ 100g（纯脂肪）
  - 坚果 ≈ 45-60g
  - 牛油果 ≈ 15g
  - 鸡蛋黄 ≈ 30g
  - 五花肉 ≈ 37g
  - 鸡胸肉 ≈ 1-2g（几乎无脂肪）

## 🔍 如何估算营养素

### 方法一：食物秤 + 生重（推荐，最精准）

1. 买一个精确到 1g 的食物秤（几十块钱）
2. 烹饪前称每种食材的**生重**
3. 查营养数据（包装袋营养成分表、薄荷健康App等）
4. 按公式计算：`克数 × 每百克营养素含量 / 100`

**示例：**
```
鸡胸肉 200g（生）: 蛋白质 = 200 × 23 / 100 = 46g, 脂肪 = 200 × 1.5 / 100 = 3g
糙米饭 150g（熟）: 碳水 ≈ 150 × 25 / 100 = 37.5g
```

### 方法二：视觉估算法（外出吃饭时用）

**蛋白质份量（约20-25g蛋白质）：**
- 一块鸡胸肉 ≈ 手掌大小（不含手指）
- 一个拳头大小的牛排
- 4-5个鸡蛋
- 一罐金枪鱼

**碳水份量（约30g碳水）：**
- 一拳头大小的米饭
- 一片中等厚度的面包
- 一个中等大小的红薯
- 一小碗面条

**脂肪份量（约10g脂肪）：**
- 一拇指大小的油/黄油
- 一小把坚果（约15g）
- 半个牛油果

### 方法三：包装食品直接读标签

包装食品背面都有营养成分表，注意：
- 看清楚是"每100g"还是"每份"
- 有些标的是"每份（如30g）"，要换算成实际食用量
- 调味料（酱油、沙拉酱）的碳水/脂肪也别忽略

## ⚖️ 生重 vs 熟重

很多新手搞混生重和熟重，导致计算偏差很大：

| 食材 | 生→熟 重量变化 | 说明 |
|------|---------------|------|
| 米饭 | 生100g → 熟约250-300g | 吸水膨胀2.5-3倍 |
| 面条（干） | 生100g → 熟约250g | 吸水膨胀 |
| 鸡胸肉 | 生100g → 熟约75g | 水分流失缩约25% |
| 牛肉 | 生100g → 熟约70g | 缩水约30% |
| 蔬菜 | 生100g → 熟约50-70g | 缩水 |
| 土豆 | 生100g → 熟约90g | 变化不大 |

**关键原则：营养素含量用生重算，因为烹饪过程不改变蛋白质/碳水/脂肪总量，只是水分变化导致重量变化。**

例如：
- 200g生鸡胸肉有 46g 蛋白质
- 煮熟后只有约 150g，但蛋白质仍然是 46g
- 如果按熟重 150g 查"每百克蛋白质 23g" → 会算成 34.5g，偏低！

## 📱 推荐工具

- **薄荷健康 App** — 查食物营养数据，扫码看成分
- **食物秤** — 京东/淘宝搜"厨房秤"，精确到1g的就行
- **MyFitnessPal** — 国际版食物数据库更全

## 💡 常见误区

1. **"吃脂肪就会长脂肪"** — 错。长不长脂肪看总热量，健康脂肪是必须的
2. **"碳水是敌人"** — 错。减脂需要适量碳水支撑训练，过低会导致无力、掉肌肉
3. **"蛋白质吃多了伤肾"** — 健康人每公斤2-2.5g完全安全，有肾病除外
4. **"晚上吃碳水会胖"** — 错。看全天总热量，不是看什么时间吃
5. **"水果不算碳水"** — 错。水果的果糖也是碳水，大量吃一样超
6. **"沙拉很健康"** — 要看沙拉酱！一份凯撒沙拉酱就有 15-20g 脂肪
