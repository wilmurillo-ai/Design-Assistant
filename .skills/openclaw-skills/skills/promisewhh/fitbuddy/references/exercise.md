# 运动与训练参考

## 肌群分类

| 肌群 | 英文 | 常见动作 |
|------|------|---------|
| 胸 | chest | 卧推、上斜卧推、飞鸟、俯卧撑 |
| 背 | back | 引体向上、杠铃划船、坐姿划船、高位下拉 |
| 腿 | legs | 深蹲、硬拉、腿举、腿弯举、箭步蹲 |
| 肩 | shoulders | 推举、侧平举、前平举、面拉 |
| 手臂 | arms | 二头弯举、三头下压、锤式弯举 |
| 核心 | core | 平板支撑、卷腹、俄罗斯转体 |
| 有氧 | cardio | 跑步、快走、骑车、跳绳、游泳 |

## 热量消耗估算

### 力量训练
- 中等强度: 6-8 kcal/min
- 高强度（大重量低次数）: 8-10 kcal/min
- 公式: `消耗 = 强度系数 × 时长(min)`
- 估算时长：一组（含休息）约 2-3 分钟

### 有氧训练（kcal/min 参考值）
| 运动 | 消耗 |
|------|------|
| 跑步（中速） | 10 |
| 快走 | 5 |
| 骑车 | 7 |
| 跳绳 | 12 |
| 游泳 | 9 |
| 椭圆机 | 8 |
| HIIT | 12-14 |

### 力量训练时长估算
```
总时长 ≈ 组数 × (实际做功时间 + 组间休息)
≈ 组数 × 2.5 分钟（中等强度默认）
```

## 训练建议逻辑

### 肌群恢复检查
- 同一肌群需要至少 **48 小时** 恢复
- 查询最近 2 天的记录，检查目标肌群是否已训练
- 如已训练，建议换其他肌群或休息

### 渐进超负荷提示
- 对比同一动作的历史记录（最近 4 周）
- 检测维度：重量增加、次数增加、组数增加
- 如连续 2 周无进步，建议调整：增加重量、改变次数范围、更换变式

### 训练计划建议模板
根据用户目标：
- **减脂**: 力量训练 + 有氧结合，推荐分化训练（推/拉/腿 或 上/下）
- **增肌**: 以力量为主，渐进超负荷，每个肌群每周 2 次
- **维持**: 保持当前训练频率，注意恢复

### 恢复建议
- 每周至少 1 天完全休息
- 连续训练不超过 3 天
- 注意睡眠（7-9 小时）
- 注意蛋白质摄入是否达标
