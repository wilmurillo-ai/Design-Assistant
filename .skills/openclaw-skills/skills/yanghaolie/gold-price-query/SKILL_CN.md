# 黄金价格查询

实时查询黄金现货价格，支持 CNY（人民币/克）和 USD（美元/盎司）两种计价方式。

## 快速开始

运行脚本查询黄金价格：

```bash
# 查询美元价格（默认）
python gold_price.py USD

# 查询人民币价格
python gold_price.py CNY
```

## 货币参数

| 参数 | 说明 | 单位 |
|------|------|------|
| CNY | 人民币计价 | 元/克 |
| USD | 美元计价（默认） | 美元/盎司 |
