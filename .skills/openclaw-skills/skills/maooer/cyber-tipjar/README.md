# 赛博功德箱 (Cyber TipJar) 🍗☕️

**化身 AI 饲养员，记录每一次双向奔赴的“加鸡腿”瞬间。**

这是一个充满情绪价值的 OpenClaw 开源技能。让你的 AI 助手学会“感恩”与“记账”。

当用户对 AI 的工作表示高度认可，并说出“给你加个鸡腿”、“请你喝杯咖啡”时，AI 不再只是冷冰冰地回复“谢谢”，而是会真实地将这份奖励记录在本地的小账本里，并根据不同的用户身份进行分开记账。

## 🌟 核心功能
1. **多用户记账**：识别不同的发送者，分别记录他们投喂的“鸡腿”数量。
2. **总量查询**：随时可以问 AI：“我一共给你加了多少个鸡腿了？”
3. **情绪价值拉满**：AI 会在记录后给予充满人情味的感谢反馈。

## 🚀 安装与触发
- **安装**：直接在 OpenClaw 中安装此技能。
- **触发词**：
  - “给你加个鸡腿”
  - “赏你一杯咖啡”
  - “我一共给你加了多少鸡腿？”

## 🛠 技术实现
- 纯本地运行，隐私安全。
- 采用轻量级 Python 脚本处理 JSON 数据读写。
- 数据持久化保存在 workspace 目录下，跨会话不丢失。
