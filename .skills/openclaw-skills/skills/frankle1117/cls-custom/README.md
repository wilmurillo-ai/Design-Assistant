# 财联社自定义关键词新闻自动化Skill

自动获取财联社电报自定义关键词新闻并格式化为一句话新闻。

## 🚀 快速开始

### 前置要求
在使用本技能前，请确保已安装 **agent-browser** 工具：

```bash
clawhub install agent-browser
```

### 使用方法
安装完成后，直接告诉AI：
- "获取财联社[关键词]新闻"（例如："获取财联社房地产新闻"）
- "搜索[关键词]"（例如："搜索AI"）
- "执行财联社新闻任务"（使用上次的关键词）

AI会自动检查必需工具，执行完整的workflow并输出格式化的新闻。

## 功能特点

✅ **全自动执行**：打开网页 → 搜索 → 提取 → 格式化
✅ **标准化输出**：行业研报级的一句话新闻
✅ **数据完整**：保留所有关键数据（营收、出货量、市场份额等）
✅ **分类清晰**：按产业政策、企业应用、市场资本、政策标准分类

## 输出格式

每条新闻遵循以下规则：
- 1-2句完成
- 格式：【公司名】关键动作 + 场景落地/战略意义；关键数据
- 无主观评价，仅事实信息

## 定时任务（可选）

如需每天自动执行，可配置OpenClaw cron：

```json
{
  "cron": {
    "entries": {
      "cls-robot-news": {
        "enabled": true,
        "schedule": {
          "kind": "cron",
          "expr": "0 9,17 * * 1-5",
          "tz": "Asia/Shanghai"
        },
        "sessionTarget": "isolated",
        "payload": {
          "kind": "agentTurn",
          "message": "执行财联社机器人新闻自动化任务"
        },
        "delivery": {
          "mode": "announce"
        }
      }
    }
  }
}
```
