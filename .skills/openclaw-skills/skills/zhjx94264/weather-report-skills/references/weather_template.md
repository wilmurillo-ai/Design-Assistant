## 斜体使用规则

### 今天天气
- **已过去的时间段**：使用斜体 `*text*`
- **当前及未来的时间段**：正常显示

### 判断时间段的逻辑
1. 获取当前时间：`date '+%Y-%m-%d %H:%M'`
2. 判断用户询问的是哪天（今天/明天/后天）
3. 如果是**今天**：
   - 凌晨 0-6点：00:00-06:00，已过用斜体
   - 早上 6-9点：06:00-09:00，按当前时间判断
   - 上午 9-12点：09:00-12:00，按当前时间判断
   - 中午 12-15点：12:00-15:00，未到正常
   - 下午 15-18点：15:00-18:00，未到正常
   - 傍晚 18-21点：18:00-21:00，未到正常
   - 夜间 21-24点：21:00-24:00，未到正常
4. 如果是**明天或后天**：所有时段都不使用斜体

## 温度数据获取

```bash
curl -s "wttr.in/Tianjin?format=j1" | python3 -c "
import json, sys
d = json.load(sys.stdin)
w = d['weather'][0]  # 0=今天, 1=明天, 2=后天
print('Date:', w['date'])
print('Max:', w['maxtempC'], 'Min:', w['mintempC'])
for h in w['hourly']:
    print(f\"Time {h['time']}: {h['tempC']}C, feels {h['FeelsLikeC']}C, {h['weatherDesc'][0]['value'].strip()}, wind {h['windspeedKmph']}km/h, humidity {h['humidity']}%, sunshine {h['chanceofsunshine']}%\")
"
```
