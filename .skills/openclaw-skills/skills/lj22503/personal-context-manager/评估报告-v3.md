# 知识管理工作流 🎯

## 📋 功能描述

帮助用户沉淀知识、分类知识、让知识产生新内容，从完整工作流角度出发。

**适用场景：**
- 处理飞书文档/微信读书/网页文章
- 自动打标（主题 + 场景 + 行动）
- 知识发芽（灵光/心智模型/跨界/微习惯/潜意识）
- 生成公众号文章/周报/月报

**边界条件：**
- 不替代手动深度思考
- 需配合人工 review
