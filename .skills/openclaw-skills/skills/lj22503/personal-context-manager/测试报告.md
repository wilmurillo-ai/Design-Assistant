## 🧪 详细测试

### 测试 1：认知日志（每日 3 件触动事）

**输入**：
```python
cm.daily_log([
    {'content': '今天和朋友争论', 'judgment': '需要建立经历回溯'},
    {'content': '知识管理=上下文管理', 'judgment': '重要洞察'},
    {'content': 'FOMO 情绪', 'judgment': '需要警惕'}
])
```

**输出**：
```
✅ 2026-04-14: 3 条
标签：[['#关系锚点#'], [], []]
```

**效果**：
- ✅ 自动识别意义标签（#关系锚点#）
- ✅ 保存为 JSON 格式
- ✅ 日期索引
