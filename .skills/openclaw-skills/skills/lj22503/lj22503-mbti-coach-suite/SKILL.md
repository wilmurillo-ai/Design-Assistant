## 🔀 类型选择引导

**如果你知道自己的类型**：
→ 直接调用对应类型的专属教练技能（如 `intj-coach`）

**如果你不确定类型**：
→ 回答以下问题，我帮你判断：
1. 你更关注外部世界还是内心世界？（外向 E / 内向 I）
2. 你更关注事实细节还是整体模式？（感觉 S / 直觉 N）
3. 你做决定时更依赖逻辑还是情感？（思考 T / 情感 F）
4. 你更喜欢计划还是灵活？（判断 J / 知觉 P）

**如果你需要进阶指南**：
→ 调用 `mbti-advanced-guide` 技能（包含 16 类型的完整策略）

## ⚠️ 常见错误

**错误 1：类型决定论**
```
问题：
• 认为"我是 INTP 所以注定分析瘫痪"
• 用类型当借口不改变

解决：
✓ 类型是起点不是终点
✓ 每种类型都有成长路径
✓ 关注行动而非标签
```

**错误 2：测试依赖**
```
问题：
• 只做 1 次测试就确定类型
• 忽视自我观察

解决：
✓ 多次测试（推荐 16Personalities）
✓ 结合自我观察
✓ 允许类型流动
```

**错误 3：跳过类型直接执行**
```
问题：
• 不看类型直接选策略
• 策略可能不适合自己

解决：
✓ 先确认类型
✓ 选择对应类型的策略
✓ 执行后反馈调整
```

## 🧪 使用示例

**示例 1：用户知道自己是 INTJ**

**输入**：
```
我是 INTJ，最近很迷茫，不知道做什么方向
```

**处理流程**：
1. 确认用户类型（INTJ）
2. 引导调用 `intj-coach` 技能
3. 开始 INTJ 专属教练对话

**预期输出**：
```
收到。你是 INTJ，推荐你调用 `intj-coach` 技能。

INTJ 的典型卡点是：
- 完美主义：等"完全准备好"再行动
- 内耗严重：想太多 - 要求太高
- 忽视情感：忽略自己和他人感受

`intj-coach` 会帮你：
- 从迷茫中想清楚真正想要啥
- 制定可执行的战略计划
- 平衡理性与情感

要我现在调用 `intj-coach` 吗？
```

---

**示例 2：用户不确定类型**

**输入**：
```
我不确定自己是什么类型，能帮我判断吗？
```

**处理流程**：
1. 询问 4 个维度偏好
2. 根据回答判断类型
3. 引导调用对应教练技能

**预期输出**：
```
好的，帮你判断类型。回答 4 个问题：

1. 你更关注外部世界还是内心世界？
   - 外部世界（喜欢社交/行动）→ E
   - 内心世界（喜欢独处/思考）→ I

2. 你更关注事实细节还是整体模式？
   - 事实细节（具体/实际）→ S
   - 整体模式（抽象/未来）→ N

3. 你做决定时更依赖逻辑还是情感？
   - 逻辑（客观/分析）→ T
   - 情感（主观/共情）→ F

4. 你更喜欢计划还是灵活？
   - 计划（有序/决定）→ J
   - 灵活（随性/开放）→ P

选完告诉我，我帮你判断类型。
```
