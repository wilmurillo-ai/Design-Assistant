---

## 多种估值方法

### 方法 1：格雷厄姆公式
```
内在价值 = EPS × (8.5 + 2g) × 4.4 / Y
- EPS = 57 元
- g = 10%
- Y = 3%（AAA 企业债利率）

内在价值 = 57 × (8.5 + 2×10) × 4.4 / 3
         = 57 × 28.5 × 1.47
         = 2394 元
```

### 方法 2：资产价值法
```
流动资产价值 = 1800 亿 - 负债
             = 偏低（轻资产）
每股价值 = 不适用（轻资产公司）
```

### 方法 3：盈利价值法
```
盈利价值 = 正常化盈利 55 元 × 25（行业 PE）
         = 1375 元
```

### 方法 4：DCF 法
```
自由现金流折现 = 约 2000 元
```

## 安全边际

```
安全边际 = (1931 - 1700) / 1931 × 100%
         = 12%
```
