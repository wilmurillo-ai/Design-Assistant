## ⚠️ 背离事件（蕴含交易机会）

### 1. 油价回落 + SCFI 上涨 ⭐ 核心背离

**事件**：原油价格高位回落，但集装箱运价指数（SCFI）连续三周上涨

**数据**：
- 原油：高位回落（美伊谈判反复）
- SCFI：报 1890.77 点，连续三周走高
- 分化：美西/美东航线涨，欧洲/地中海跌

**背离逻辑**：
```
油价跌（成本下降）+ SCFI 涨（运价上涨）
        ↓
航运公司利润增厚（双击）
        ↓
类似中远海控的逻辑
```

**交易机会**：
- **利好**：航运公司（成本降 + 收入涨 = 利润增厚）
- **标的**：中远海控（集运 + 干散货双业务）
- **逻辑**：油价每跌 10 美元，利润增厚约 15 亿

### 2. 机器人量产提速 + 产能供不应求 ⭐ 核心背离

**事件**：人形机器人量产提速，但上半年产能已供不应求

**数据**：
- 智元机器人远征 A3：首批交付完成
- 订单：已全部锁定
- 产能：上半年供不应求
- 性质："量产能力不足"而非"需求不足"

**背离逻辑**：
```
需求远超供给
        ↓
产能扩张是必然
        ↓
上游零部件企业受益
```

**交易机会**：
- **利好**：机器人上游零部件（减速器/伺服电机/传感器）
- **逻辑**：产能扩张 → 零部件需求增加
- **关注**：有产能的零部件企业


**交易机会**：
