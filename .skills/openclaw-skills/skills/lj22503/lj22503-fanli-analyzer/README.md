## 🧠 四步心法详解

### 第一步：采集

**采集标准**：
| 类别 | 采集标准 |
|------|---------|
| **闸门** 🚪 | 法规生效、技术定型、产能锁定 |
| **管道** 📊 | 库存/周转率/价格同比环比超 20% 变动 |
| **背离** ⚠️ | 售价与成本剪刀差、估值与基本面背离 |

**采集格式**：
```
[闸门] [来源·日期] 法规名称 + 量化门槛/生效时间
[管道] [来源·日期] 指标名称 + 当前值 + 变动幅度
[背离] [来源·日期] 指标 A 走势 + 指标 B 走势 + 背离持续时间
```

**铁律**：无来源不入库，非一手不轻信。

---

### 第二步：选择

**三把尺子**：
| 尺子 | 权重 | 评分标准 |
|------|------|---------|
| **不可逆性** | 50% | 立法生效 9 分/行政命令 6 分/倡议 3 分 |
| **影响半径** | 30% | 全产业链 9 分/单一赛道 6 分/区域 3 分 |
| **认知时差** | 20% | 未定价 9 分/部分定价 3 分/充分定价 0 分 |

**筛选公式**：
```
综合得分 = 不可逆性×0.5 + 影响半径×0.3 + 认知时差×0.2
```

**处理方式**：
- **≥6 分**：本周核心矿料，必拆
- **4-5 分**：观察池，等待催化剂
- **≤3 分**：归档，不分析

---

### 第三步：判定

**判定清单**：
| 判定项 | A 级标准 | 红灯信号 |
|--------|---------|---------|
| **数据来源** | 一手（公告/统计局） | 来源无法追溯 |
| **统计口径** | 同比/环比明确 | 模糊词≥3 次 |
| **数据时效** | 时效≤1 个月 | 时间差>1 月 |
| **利益冲突** | 无利害关系 | 协会发布本行业数据 |
| **历史一致性** | 历史序列连续 | 断崖式变化无说明 |

**判定结论**：
- **A 级**：可直接使用（用于核心判断）
- **B 级**：可参考但需标注（需交叉验证）
- **C 级**：仅作背景（不用于核心判断）
- **D 级**：直接丢弃

---

### 第四步：分析

**四层拆解**：

#### 4.1 望气 · 定格局
- 利益流向：谁获利？谁受损？量化弹性？
- 供需断裂：临时性还是永久性？
- 数据印证：有无反常识背离？

#### 4.2 走卒 · 定营生
- 痛点变现：普通人怎么赚钱？
- 信息套利：时间差/地域差/认知差
- 最小可行性：几百几千元怎么启动？

#### 4.3 商贾·实业 · 定卡位
- 赛道预判：哪个细分行业爆发？
- 上游截流：产能瓶颈在哪？
- 下游延伸：渠道/售后/衍生服务空白？

#### 4.4 商贾·操盘 · 定买卖
- 价格锚定：估值隐含什么假设？
- 赛道与标的：股票/期货/期权/ETF？
- 周期与节奏：建仓/加仓/止盈/止损？
