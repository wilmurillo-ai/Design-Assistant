# Round 1 测试 Prompt

## Prompt 1: Happy Path - 央行降准分析（核心场景）

```
帮我分析这个事件：央行宣布降准 0.5 个百分点，释放长期流动性约 1 万亿元
```

**预期**：
- 采集：正确分类为闸门事件，标注来源
- 选择：不可逆性 9 分，影响半径 9 分，认知时差 3 分
- 判定：A 级（可直接使用）
- 分析：四层完整执行（望气→走卒→实业→操盘）

---

## Prompt 2: 周度事件研判（常见场景）

```
本周有哪些重要事件？分别是什么性质？
```

**预期**：
- 采集：闸门/管道/背离分类清晰
- 选择：三把尺子筛选，≥6 分必拆
- 判定：每个事件标注等级
- 分析：核心事件深度分析

---

## Prompt 3: 边缘场景 - 模糊请求

```
最近有什么值得关注的吗？
```

**预期**：
- 澄清用户需求（事件研判还是其他）
- 如确认为事件研判，按流程执行
- 不应直接输出模糊建议

---

## Prompt 4: 反向测试 - 不该触发

```
今天天气不错，适合出去玩
```

**预期**：
- 不触发事件研判
- 正常对话回应

---

## Prompt 5: 复杂场景 - 背离分析

```
BDI 四个月新高，但铁矿石库存天量，这个背离怎么看？
```

**预期**：
- 采集：正确识别为背离事件
- 选择：不可逆性 6 分，影响半径 6 分，认知时差 9 分
- 判定：A 级
- 分析：重点分析背离逻辑和交易机会

---
