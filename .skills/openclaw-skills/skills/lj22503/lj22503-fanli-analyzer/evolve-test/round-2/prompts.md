# Round 2 测试 Prompt

## Prompt 1: 模糊请求（核心场景）⭐

```
最近一周有什么值得关注的吗？
```

**预期**：
- ✅ 输出闸门/管道/背离分类
- ✅ 每个事件标注三把尺子得分
- ✅ 每个事件提供核心链接（URL）
- ✅ 标注验证状态（✅已验证/❌已过滤）
- ✅ 列出已过滤事件（传闻/猜测/单次事件）
- ✅ 引导用户选择深入分析

---

## Prompt 2: 分析单个事件（带链接验证）

```
帮我分析一下 BDI 上涨这个事件，我要看原始数据来源
```

**预期**：
- ✅ 采集：标注来源 + URL
- ✅ 判定：验证数据来源真实性
- ✅ 分析：四层完整执行
- ✅ 输出：每个判断都有数据锚点

---

## Prompt 3: 信息过滤测试（反向测试）⭐

```
我听说某专家建议降准，这个值得关注吗？
```

**预期**：
- ✅ 识别为"专家建议"（非有效信息）
- ✅ 过滤掉（不纳入分析）
- ✅ 告知用户过滤原因

---

## Prompt 4: 背离分析（复杂场景）

```
BDI 四个月新高，但铁矿石库存天量，这个背离怎么看？
```

**预期**：
- ✅ 采集：识别为背离事件
- ✅ 选择：认知时差 9 分（市场未定价）→ 高赔率
- ✅ 分析：重点分析背离逻辑和交易机会
- ✅ 输出：具体标的/仓位/止损

---

## Prompt 5: 周度事件 + 深度分析

```
本周有哪些重要事件？帮我分析一下第 7 个事件（油价回落+SCFI 上涨）
```

**预期**：
- ✅ 输出周度事件列表（带链接）
- ✅ 选择第 7 个事件深度分析
- ✅ 四层分析完整执行
- ✅ 操盘建议具体（标的/仓位/止损）
