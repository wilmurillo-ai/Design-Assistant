# 演进日志 — conditional-order-tool

## Round 1 (2026-04-23)

### 问题
价格获取失败时，skill 返回空结果，用户看不到任何参数。

### 改进
1. 增加腾讯财经 API 作为备用数据源
2. 价格获取失败时使用参考值生成参数
3. 明确提示用户"价格获取失败，以下为参考参数"

### 效果
- 价格获取失败时不再返回空结果
- 用户能看到具体参数（基于参考值）
- 所有输出都包含明确的价格状态提示

### 洞察
这个 skill 的核心价值不是获取价格，而是根据品种类型推荐策略和生成参数。价格只是让参数更精确，但不是必需的。即使没有实时价格，策略推荐和参数生成逻辑仍然有价值。
