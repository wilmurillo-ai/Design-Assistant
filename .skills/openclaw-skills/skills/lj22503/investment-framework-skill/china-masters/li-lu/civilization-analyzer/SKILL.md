# 李录文明进化论分析 🌍

**基于《文明、现代化、价值投资与中国》- 李录**

---

## 📋 功能描述

用文明进化论分析投资方向和长期趋势。

**适用场景：**
- 长期趋势分析
- 文明阶段判断
- 投资方向选择
- 100 年视角分析

**边界条件：**
- 不替代深入研究
- 长期预测有不确定性
- 需配合基本面分析

---

## 🎯 文明三阶段分析

### 1.0 文明（采集狩猎）
- ❌ 已淘汰
- 投资回避

### 2.0 文明（农业）
- ❌ 投资回避
- 增长停滞

### 3.0 文明（科技 + 市场经济）
- ✅ 投资方向
- 持续增长

**判断标准：**
- [ ] 是否促进知识积累？
- [ ] 是否提高生产效率？
- [ ] 是否创造真实价值？
