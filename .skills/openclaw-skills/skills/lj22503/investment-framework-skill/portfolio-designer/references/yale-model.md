## 再平衡策略

### 频率
**建议：** 每年 1 次（建议生日）

### 阈值
**建议：** 偏离>5% 时调整

### 方法
**建议：** 卖出高估资产，买入低估资产

---

## 定投策略

### 原则
1. **固定时间**（每月发薪日）
2. **固定金额**
3. **长期坚持**
4. **不止损**

### 优势
1. **平均成本**
2. **克服情绪**
3. **强制储蓄**
4. **复利效应**
