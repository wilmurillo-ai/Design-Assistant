## 📊 场景 1：个股价值分析

### 当你想分析一家公司是否值得投资时

**触发命令：**
```
@ant 分析贵州茅台，是否符合价值投资标准？
```

**使用技能：**
1. `value-analyzer` - 格雷厄姆标准分析
2. `moat-evaluator` - 护城河评估
3. `intrinsic-value-calculator` - 内在价值计算
4. `simple-investor` - A 股三要素分析

**输出内容：**
```
【价值分析】
- 防御型/积极型评估
- 7 大标准检查结果
- 安全边际计算

【护城河评估】
- 护城河类型（品牌/网络/转换成本/规模/特许）
- 强度评分（0-25 分）
- 持续性分析

【内在价值】
- 4 种估值方法结果
- 综合内在价值
- 安全边际百分比

【A 股三要素】
- 估值：便宜/合理/贵
- 品质：优秀/良好/一般/差
- 时机：买入/观察/回避

【综合建议】
建议：买入/观察/回避
理由：...
风险：...
```

**适用公司类型：**
- A 股上市公司
- 港股上市公司
- 美股上市公司

## 🏭 场景 3：行业研究与配置

### 当你想快速了解一个行业时

**触发命令：**
```
@ant 分析新能源汽车行业，现在值得投资吗？
```

**使用技能：**
1. `industry-analyst` - 行业生命周期分析
2. `future-forecaster` - 长期趋势预测
3. `cycle-locator` - 经济周期判断

**输出内容：**
```
【行业生命周期判断】
- 当前阶段：成长期
- 特征：市场快速扩大，增速 30%+

【行业空间分析】
- 市场规模：5000 亿
- 增长率：30%
- 天花板：远（渗透率仅 20%）

【竞争格局分析】
| 公司 | 市场份额 | 竞争优势 |
|------|---------|---------|
| 比亚迪 | 35% | 垂直整合 |
| 特斯拉 | 20% | 品牌技术 |
| 蔚来 | 10% | 服务换电 |

【投资价值评估】
| 维度 | 评分 |
|------|------|
| 行业空间 | 9/10 |
| 增长速度 | 9/10 |
| 竞争格局 | 7/10 |
| 盈利水平 | 6/10 |
| 进入壁垒 | 7/10 |

【综合评分】7.6/10

【配置建议】
- 优先配置：龙头公司
- 适度配置：二线公司
- 暂时回避：无优势公司
```

**适用行业：**
- 新兴行业（AI、新能源、生物科技）
- 传统行业（消费、金融、制造）
- 周期性行业（资源、地产）

## 🧠 场景 4：投资决策检查

### 当你想检查投资决策是否理性时

**触发命令：**
```
@ant 我想买入宁德时代，帮我检查这个决策
```

**使用技能：**
1. `decision-checklist` - 认知偏差检查
2. `second-level-thinker` - 第二层思维分析
3. `bias-detector` - 认知偏差检测

**输出内容：**
```
【能力圈检查】
| 问题 | 回答 | 评分 |
|------|------|------|
| 理解商业模式？ | 是 | 5 |
| 预测 10 年后格局？ | 是 | 4 |
| 有行业专业知识？ | 否 | 2 |
| 投资过类似公司？ | 是 | 4 |

能力圈评分：15/20（能力圈边缘）

【认知偏差检查】
| 偏差类型 | 是否存在 | 风险等级 |
|---------|---------|---------|
| 喜好偏见 | ✅ | 中 |
| 从众心理 | ✅ | 中 |
| 确认偏误 | ✅ | 中 |

偏差数量：3/8（中等风险）

【第二层思维分析】
| 问题 | 第一层回答 | 第二层回答 |
|------|-----------|-----------|
| 共识是否正确？ | 是 | 可能过度乐观 |
| 风险在哪里？ | 竞争加剧 | 技术路线风险 |

【综合建议】
建议：谨慎投资
理由：
1. 能力圈边缘（15/20 分）
2. 认知偏差中等（3/8）
3. 需警惕从众心理

【行动项】
1. 寻找反面证据
2. 关注技术路线变化
3. 设定明确止损位
```

**适用场景：**
- 大额投资前
- 追涨杀跌冲动时
- 市场狂热/恐慌时

## 🔄 场景 6：完整投资工作流

### 从分析到执行的完整流程

**触发命令：**
```
@ant 帮我分析贵州茅台，从价值分析到资产配置
```

**工作流：**
```
1. value-analyzer → 初步筛选（是否符合格雷厄姆标准）
   ↓
2. moat-evaluator → 护城河评估（是否有持续竞争优势）
   ↓
3. intrinsic-value-calculator → 价值计算（内在价值和安全边际）
   ↓
4. decision-checklist → 决策检查（避免认知偏差）
   ↓
5. asset-allocator → 资产配置（仓位控制）
   ↓
执行买入，定期复盘
```

**输出内容：**
```
【步骤 1：价值分析】
- 符合 X/7 格雷厄姆标准
- 安全边际：X%

【步骤 2：护城河评估】
- 护城河类型：品牌优势
- 强度：X/25

【步骤 3：内在价值】
- 综合估值：X 元
- 当前价格：X 元
- 安全边际：X%

【步骤 4：决策检查】
- 能力圈：内/边缘/外
- 认知偏差：低/中/高

【步骤 5：资产配置】
- 建议仓位：X%
- 配置比例：X%

【综合建议】
建议：买入/观察/回避
仓位：X%
理由：...
```

## 💡 最佳实践

### 1. 先学习理论
- 阅读相关投资书籍
- 理解技能背后的理论
- 不要盲目使用

### 2. 从简单开始
- 先用 value-analyzer 做初步筛选
- 再用 moat-evaluator 深入分析
- 最后用 decision-checklist 检查决策

### 3. 定期复盘
- 每月复盘投资决策
- 检查认知偏差
- 调整资产配置

### 4. 长期视角
- 关注长期趋势
- 忽视短期波动
- 坚持价值投资
