# 投资框架 Skill 包 📚

**基于 5 本经典投资书籍 + 凯文·凯利未来预测方法论**

---

## 📋 功能描述

基于经典投资理论的实战工具箱。

**适用场景：**
- 系统化投资分析
- 价值投资实践
- 资产配置规划
- 投资决策检查

**边界条件：**
- 不替代专业建议
- 需配合深入研究
- 个体情况需考虑
- 市场有风险

---

## 🎯 核心技能

### 1. value-analyzer（价值分析师）
**理论来源：** 《聪明的投资者》- 格雷厄姆
**功能：** 基于格雷厄姆原则分析股票价值
**触发：** "分析这只股票"、"是否符合格雷厄姆标准"

### 2. moat-evaluator（护城河评估师）
**理论来源：** 《巴菲特致股东的信》- 巴菲特
**功能：** 评估企业护城河类型和强度
**触发：** "评估护城河"、"这家公司有竞争优势吗"

### 3. asset-allocator（资产配置师）
**理论来源：** 《漫步华尔街》- 马尔基尔
**功能：** 根据生命周期设计资产配置方案
**触发：** "资产配置方案"、"我应该如何配置资产"

### 4. decision-checklist（决策清单）
**理论来源：** 《穷查理宝典》- 芒格
**功能：** 基于多元思维和认知偏差检查投资决策
**触发：** "检查这个投资想法"、"是否有认知偏差"

### 5. intrinsic-value-calculator（内在价值计算器）
**理论来源：** 《证券分析》- 格雷厄姆 & 多德
**功能：** 计算企业内在价值和安全边际
**触发：** "计算内在价值"、"安全边际是多少"

### 6. future-forecaster（未来预测师）
**理论来源：** 凯文·凯利《必然》《失控》
**功能：** 基于 KK 三思维模型和四层变化规律识别长期趋势
**触发：** "未来 10 年什么有机会"、"长期趋势分析"

## ⚠️ 常见错误

**错误 1：忽视安全边际**
```
问题：
• 好公司也要好价格
• 忽视估值风险
• 支付过高溢价

解决：
✓ 用 4 毛钱买 1 块钱的东西
✓ 等待好价格
✓ 保持耐心
```

**错误 2：能力圈外投资**
```
问题：
• 投资不懂的公司
• 盲目跟风
• 忽视能力圈边界

解决：
✓ 只投懂的公司
✓ 持续学习
✓ 承认无知
```

**错误 3：忽视资产配置**
```
问题：
• 全仓单一资产
• 忽视分散投资
• 忽视再平衡

解决：
✓ 全球分散配置
✓ 定期再平衡
✓ 长期坚持
```

**错误 4：认知偏差不检查**
```
问题：
• 过度自信
• 确认偏误
• 从众心理

解决：
✓ 使用决策清单
✓ 逆向思考
✓ 记录决策日志
```

**错误 5：忽视长期趋势**
```
问题：
• 只关注短期
• 忽视技术变革
• 忽视社会变化

解决：
✓ 关注长期趋势
✓ 四层变化分析
✓ 逆向布局
```

## 📚 核心理念

**关键洞察：**
1. 安全边际是核心
2. 只投能力圈内
3. 分散投资降低风险
4. 认知偏差需检查
5. 关注长期趋势

**健康公式：**
```
好投资 = 好公司 × 好价格 × 好时机 × (1 - 认知偏差)
```
