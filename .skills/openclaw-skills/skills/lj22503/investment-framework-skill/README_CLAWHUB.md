## 📋 简介

投资框架技能包是一个完整的 AI 辅助投资决策系统，包含 **35 个技能**，覆盖从价值分析、护城河评估、资产配置、决策检查到未来预测的全流程。

**核心理念：**
- 🛡️ **安全边际** - 价格必须低于价值
- 🏰 **护城河** - 只投资有持续竞争优势的公司
- 🧠 **理性决策** - 避免认知偏差，逆向思考
- 🌍 **长期视角** - 投资人类文明进步方向
- 🇨🇳 **中国机会** - 中国现代化是最大投资机会

## 🎯 适用场景

### 场景 1：个股分析

**当你想分析一家公司是否值得投资时：**

```
@ant 分析贵州茅台，是否符合价值投资标准？
```

**使用技能：**
1. `value-analyzer` - 格雷厄姆标准分析
2. `moat-evaluator` - 护城河评估
3. `intrinsic-value-calculator` - 内在价值计算
4. `simple-investor` - A 股三要素分析（估值/品质/时机）

**输出：**
- 防御型/积极型评估
- 护城河类型和强度
- 内在价值和安全边际
- 投资建议

## ⚠️ 常见错误

### 错误 1：忽视安全边际

**问题：**
- 好公司也要好价格
- 忽视估值风险
- 支付过高溢价

**解决：**
- 用 4 毛钱买 1 块钱的东西
- 等待好价格
- 保持耐心

### 错误 2：能力圈外投资

**问题：**
- 投资不懂的公司
- 盲目跟风
- 忽视能力圈边界

**解决：**
- 只投懂的公司
- 持续学习
- 承认无知

### 错误 3：忽视资产配置

**问题：**
- 全仓单一资产
- 忽视分散投资
- 忽视再平衡

**解决：**
- 全球分散配置
- 定期再平衡
- 长期坚持

### 错误 4：认知偏差不检查

**问题：**
- 过度自信
- 确认偏误
- 从众心理

**解决：**
- 使用决策清单
- 逆向思考
- 记录决策日志

### 错误 5：忽视长期趋势

**问题：**
- 只关注短期
- 忽视技术变革
- 忽视社会变化

**解决：**
- 关注长期趋势
- 四层变化分析
- 逆向布局
