# Usage (EN)

1. Upload this skill package to ClawHub.
2. Install and run:
```bash
bash scripts/deploy_project.sh
```
3. Logs:
- Backend: `/tmp/openclaw-agent-control-backend.log`
- Frontend: `/root/OpenClaw-Agent-Control/agent-monitor-ui/.run/agent-monitor-ui.log`

## Common env vars
- `PROJECT_DIR`
- `REPO_URL`
- `MONITOR_PORT`
- `PORT`
