# Official Writing 公文写作技能

中国党政机关公文格式国家标准 (GB/T 9704-2012) 全面指南。

## 适用场景

- 📝 撰写党政公文
- 📰 活动新闻稿
- 📊 工作汇报材料
- 🎤 领导讲话稿
- 📋 党建文章
- 📢 宣传稿件

## 核心功能

### 1. 格式规范
- 页面设置（页边距、行距）
- 字体字号标准
- 标题层级格式
- 页码规范

### 2. 模板库
- 通知模板
- 报告模板
- 讲话稿模板
- 新闻稿模板

### 3. 写作技巧
- 用词规范
- 表达规范
- 格式检查

## 格式规范详情

### 页面设置

| 参数 | 数值 |
|------|------|
| 上边距 | 3.7 cm |
| 下边距 | 3.5 cm |
| 左边距 | 2.8 cm |
| 右边距 | 2.6 cm |

### 字体字号

| 元素 | 字体 | 字号 |
|------|------|------|
| 公文标题 | 方正小标宋简体 | 二号 |
| 一级标题 | 黑体 | 三号 |
| 二级标题 | 楷体_GB2312 | 三号加粗 |
| 正文 | 仿宋_GB2312 | 三号 |

## 标题层级

```
一、一级标题（黑体三号）
  （一）二级标题（楷体三号加粗）
    1. 三级标题（仿宋三号）
      （1）四级标题（仿宋三号）
```

## 公文类型模板

### 通知模板
```
【标题】
关于开展XXX活动的通知

【正文】
一、活动目的
...

二、活动时间
X月X日至X月X日

三、主要内容
...

四、工作要求
（一）高度重视
（二）精心组织

【落款】
XXX单位
202X年X月X日
```

### 报告模板
```
【标题】
关于XXX情况的报告

【正文】
一、基本情况
...

二、主要做法
...

三、取得成效
...

四、下步打算
...

【落款】
XXX单位
202X年X月X日
```

## 使用方法

1. 确定公文类型
2. 选择对应模板
3. 按格式规范填写内容
4. 检查字体字号
