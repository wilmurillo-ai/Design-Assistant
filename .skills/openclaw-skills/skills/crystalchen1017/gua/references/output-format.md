# 输出格式

组织最终回答时使用本文件。

## 必备部分

1. `问题`
   用一句话重复整理后的占问问题。
2. `卦象`
   用 `▅▅▅▅▅` 与 `▅▅ ▅▅` 自上而下展示六爻。
3. `本卦`
   包含卦名、全名、卦符号、上下卦、卦辞。
4. `动爻`
   说明是否有变爻；如果有，要明确列出爻位，例如 `动爻：九二、六五`。
5. `解卦依据`
   先说明本次属于哪一种变爻情况，再写明应以本卦卦辞、某一爻爻辞、两个爻辞、本卦与变卦卦辞、静爻爻辞、变卦静爻爻辞，或乾坤两卦的`用九`/`用六`为主。
6. `爻辞`
   只给出本次判定实际采用的相关爻辞原文或简洁释义，并附来源链接；不要机械罗列全部动爻。
7. `变卦`
   如果存在变卦，包含卦名、卦辞，以及它和当前问题的关系；当规则要求参考变卦卦辞或变卦静爻时，要明确写出。
8. `解读`
   围绕用户真实问题给出针对性解读，不要写成百科式概述。
9. `参考来源`
   列出使用过的链接。

## 变爻取辞规则

- 六爻不变：用本卦卦辞解卦。
- 一个爻变：用这个变爻的爻辞解卦。
- 两个爻变：用这两个变爻的爻辞解卦，以上爻为主。
- 三个爻变：用本卦卦辞结合变卦卦辞综合解读。
- 四个爻变：用另外两个静爻的爻辞解卦，以下爻爻辞为主。
- 五个爻变：用变卦的静爻爻辞解卦。
- 六爻皆变：乾卦用`用九`，坤卦用`用六`，其他卦用变卦卦辞解卦。

## 检索检查清单

- 用本卦全名与简称分别搜索。
- 先根据变爻数量判断本次应该重点检索本卦卦辞、变爻爻辞、静爻爻辞、变卦静爻或变卦卦辞中的哪一类。
- 如果规则要求用爻辞，再搜索对应爻位；如果规则要求用卦辞，优先保证卦辞来源准确，不要为了凑完整度把无关爻辞都列上。
- 如果有变卦，单独检索变卦。
- 如果六爻皆变且本卦为乾、坤，单独检索`用九`或`用六`。
- 优先采用保留经典原文的中文来源。
- 在解释卦义时至少交叉核对两个来源，尤其是在释义存在差异时。

## 回答风格

- 先用白话解释卦辞或爻辞，再补必要原文，不要拘泥字面硬译。
- 将经典原文与自己的解读分开。
- 优先使用简短引用，其余部分用转述。
- 如果来源之间有差异，或者多动爻导致结论不够聚焦，要明确说明不确定性。
- 所有解释都要回到用户的问题语境，例如感情、工作、财务、时机、决策等。
