# 转换示例参考

本文档收集更多 Markdown 转换示例，供复杂场景参考。

---

## 示例 1：完整段落转换

**输入**：
```
今天给大家介绍一个新工具。这个工具叫WorkBuddy，是一款AI助手。它有三个主要功能：第一个是写作，可以帮你写文章、报告、邮件；第二个是数据分析，可以处理表格、生成图表；第三个是代码开发，可以写代码、调试bug。了解更多请访问 http://example.com
```

**输出**：
```markdown
# 全新 **AI 助手 WorkBuddy** 介绍

今天给大家介绍一个新工具——**WorkBuddy**，一款功能强大的 **AI 助手**。

## 三大核心功能

WorkBuddy 提供三个主要功能：

1. **写作助手**
   - 帮你写**文章**
   - 撰写**报告**
   - 处理**邮件**

2. **数据分析**
   - 处理**表格**
   - 生成**图表**

3. **代码开发**
   - 编写**代码**
   - 调试 **bug**

了解更多：[链接](http://example.com)
```

## 示例 9：术语定义

**输入**：
```
API是什么？API全称Application Programming Interface，是应用程序对外提供的编程接口。
SDK是什么？SDK全称Software Development Kit，是软件开发工具包，包含开发所需的工具和库。
```

**输出**：
```markdown
## 术语说明

**API**：Application Programming Interface，应用程序对外提供的编程接口。

**SDK**：Software Development Kit，软件开发工具包，包含开发所需的工具和库。
```
