# Manual AI - 手动模拟自动接口

当自动 API 调用失败（限速/无权限/不支持）或需要特定平台优势时，通过人机协作完成 AI 生成任务。

## 支持平台总览

| 平台 | 网址 | 需要登录 | 需要VPN | 免费 | 主要能力 |
|------|------|----------|---------|------|----------|
| **Gemini** | gemini.google.com | Google账号 | 国内需要 | 有额度 | 文本、图片、多模态、代码 |
| **Google AI** | google.com/search?udm=50 | ❌ 不需要 | 国内需要 | ✅ 完全免费 | 快速图片生成 |
| **ChatGPT** | chatgpt.com | OpenAI账号 | 不需要 | 免费版有限 | 文本(Plus有DALL-E生图) |
| **豆包** | doubao.com | 字节账号 | 不需要 | 有额度 | 文本、图片、联网 |
| **千问** | qianwen.com | 阿里账号 | 不需要 | ✅ 完全免费 | 文本、图片、多模态 |
| **NotebookLM** | notebooklm.google.com | Google账号 | 国内需要 | ✅ 完全免费 | 信息图、幻灯片、视频 |

## 平台详细指南

### 1. Gemini (gemini.google.com)

**已验证 ✅** — 国内需 VPN，全功能可用

#### 支持能力

| 功能 | 支持 | 说明 |
|------|------|------|
| 文本生成 | ✅ | 长上下文(1M tokens)，多轮对话 |
| 图片生成 | ✅ | 英文/中文提示词均可 |
| 代码生成 | ✅ | 代码能力强 |
| 图片理解 | ✅ | 上传图片分析 |
| PDF 分析 | ✅ | 上传文档总结 |
| 联网搜索 | ✅ | 实时信息 |

#### 登录方式
- Google 账号（Gmail）

#### 使用建议
- 图片生成：英文提示词效果更好，中文也可用
- 代码任务：质量高，推荐
- 长文档分析：支持大文件
- **提示词格式：** `Generate an image of [内容] + [风格/氛围描述]`

#### 操作步骤
1. 打开 https://gemini.google.com
2. 登录 Google 账号
3. 输入提示词或上传文件
4. 等待生成，复制/下载结果

### 2. Google AI 搜索生图 (google.com/search?udm=50)

#### 支持能力

| 功能 | 支持 | 说明 |
|------|------|------|
| 图片生成 | ✅ | 搜索框输入描述 |
| 无需登录 | ✅ | 最大优势 |
| 完全免费 | ✅ | 无任何限制 |

#### 使用建议
- 适合快速尝试，不适合高质量需求
- 质量中等，风格多样
- 速度最快，无需任何账号

#### 操作步骤
1. 打开 https://google.com/search?udm=50&q=[英文提示词]
2. 或直接打开网址，在搜索框输入图片描述
3. 切换到 Images 标签页
4. 选择喜欢的图片下载

#### URL 格式
```
https://google.com/search?udm=50&prompt=[英文提示词]
```

### 6. NotebookLM (notebooklm.google.com)

#### 支持能力

| 功能 | 支持 | 说明 |
|------|------|------|
| 文档分析 | ✅ | 深度理解 |

| 音频概览 | ✅ | 播客风格音频 |

- Google 账号

- 知识可视化 — 将文档转为信息图/幻灯片/视频

6. 选择生成类型：信息图/幻灯片/视频/音频

## 提示词模板

### 图片生成

#### 英文模板（Gemini/ChatGPT/Google AI）
```
Generate an image of [内容描述], [风格], [氛围], [技术要求]

示例：
Generate an image of a modern wireless headphone on white background, 
studio lighting, commercial photography style, 4K resolution
```

#### 中文模板（豆包/千问）
```
生成一张[内容描述]，[风格]，[氛围]，[技术要求]

示例：
生成一张现代无线耳机的产品照片，白色背景，影棚灯光，商业摄影风格，4K分辨率
```

### 文本生成

#### 文章写作
```
请写一篇关于[主题]的文章。

要求：
- 字数：[字数]字左右
- 风格：[正式/轻松/学术]
- 结构：[是否需要小标题]
- 目标读者：[读者群体]
```

#### 代码生成
```
用 [语言] 编写一个[功能描述]。

要求：
- 包含类型注解
- 包含注释
- 处理异常情况
```

## 故障排除

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| Gemini/NotebookLM 打不开 | 国内网络限制 | 使用 VPN |
| ChatGPT 无法生图 | 免费版不支持 | 升级 Plus 或换其他平台 |
| 豆包额度用完 | 免费额度限制 | 等第二天刷新或换平台 |
| 图片生成质量差 | 提示词不够详细 | 增加风格、氛围、技术细节 |
| 中文生图效果差 | 平台中文理解弱 | 换豆包/千问或用英文提示词 |
