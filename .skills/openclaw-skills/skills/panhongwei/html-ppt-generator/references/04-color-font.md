## 语义固定色（保留）

下列颜色有明确语义含义，不随 design-system 主题变化：

```css
--danger:  #ef4444;   /* 威胁、失败、高风险 */
--warning: #f59e0b;   /* 警告、中风险、注意 */
--success: #10b981;   /* 通过、安全、防御有效 */
--info:    #3b82f6;   /* 信息、中性数据 */
--neutral: #94a3b8;   /* 辅助文字、次级信息 */
```

这些变量应在每个 Tc 模板的 `:root` 中声明（值固定，不需根据风格调整）。
