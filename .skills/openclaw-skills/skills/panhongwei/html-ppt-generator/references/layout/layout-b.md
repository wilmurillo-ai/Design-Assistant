# 布局 B · 上下四行（每行双栏）

**适用：** 步骤流程、方法对比、四阶段分析
**结构基因：** 水平横切四行，每行左文右图

---

## 结构图

```
┌───────────────────────────────────┐
│  行1：左文字(58%)  右图表(42%)    │
├───────────────────────────────────┤
│  行2：左文字       右图表          │
├───────────────────────────────────┤
│  行3：左文字       右图表          │
├───────────────────────────────────┤
│  行4：左文字       右图表          │
└───────────────────────────────────┘
```

> ⚠ 每行必须是左文 + 右图双栏，**禁止纯文字行**

---

## 空间计算

```
padding-top  = 10px
每行高       = (580 - 10×3 - 10) ÷ 4 = 130px
左文字宽     = 967 × 58% ≈ 561px
右图表宽     = 967 × 42% ≈ 406px
正文可用     ≈ 130 - 标题 22px - padding 16px = 92px → 约 45–55 字
```

```css
}

  display: grid;
  grid-template-columns: 58% 42%;
  gap: 10px;
}
```

---

## 变形方向

- 行数可改为 3 行（每行更高，正文更充裕，约 75 字）
- 左右比例可改为 50/50（图文等重）或 65/35（文字主导）
- 某行可取消双栏改为全宽横幅（强调行，注意总高度不变）
