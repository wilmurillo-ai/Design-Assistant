# 布局 J · 品字形（上1宽 + 下2等）

**适用：** 总-分结构、问题陈述 + 双方案对比、背景铺垫 + 双维度展开
**结构基因：** 顶部全宽主格统领 + 底部均分双格展开

---

## 结构图

```
┌───────────────────────────────────────┐
│   ① 顶部全宽（背景 / 核心观点 / 图表）│  220px
├─────────────────────┬─────────────────┤
│   ② 左下细节         │   ③ 右下细节    │
└─────────────────────┴─────────────────┘
```

---

## 空间计算

```
padding-top  = 10px
顶行高        = 220px
gap          = 10px
底行高        = 580 - 10 - 220 - 10 = 340px
底行列宽      = (967 - 8) ÷ 2 ≈ 479.5px
正文可用（顶）≈ 220 - 标题 22px - padding 16px = 182px → 约 70–90 字
正文可用（底）≈ 340 - 标题 22px - padding 16px = 302px → 约 105–130 字
```

## CSS

```css
.layout-j {
  display: grid;
  grid-template-rows: 220px 1fr;
  gap: 10px;
  height: 100%;
  padding-top: 10px;
}
.lj-bottom {
  display: grid;
  grid-template-columns: repeat(2, 1fr);
  gap: 8px;
}
```

---

## 变形方向

- 顶行高度可调为 180–260px（视觉主图大小）
- 底部可改为三格（上1下3 结构，底行加 `repeat(3, 1fr)`）
- 顶行可放：大型 SVG 流程图、横向时间轴、大标题 + mini 数据条
- 底部两格可不等宽（如 60/40，强调某一方案）
