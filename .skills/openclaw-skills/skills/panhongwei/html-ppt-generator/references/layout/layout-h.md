# 布局 H · 三列中央突出（1:2:1）

**适用：** 核心概念展示、方案对比（中为推荐项）、前-中-后三阶段流程
**结构基因：** 两侧辅助列夹持中央主视觉列

---

## 结构图

```
┌──────────┬──────────────────────┬──────────┐
│          │                      │          │
│  ① 左辅  │   ② 中央主视觉       │  ③ 右辅  │
│  ~24%    │   ~52%               │  ~24%    │
│          │                      │          │
└──────────┴──────────────────────┴──────────┘
```

---

## 空间计算

```
padding-top  = 10px
中央列宽      = 967 × 52% ≈ 503px
左右侧列宽    = (967 - 503 - 8×2) ÷ 2 ≈ 224px
高度          = 580 - 10 = 570px
正文可用（中）≈ 570 - 标题 22px - padding 16px = 532px → 约 190–220 字
正文可用（侧）≈ 570 - 标题 22px - padding 16px = 532px → 约 85–110 字
```

---

## CSS

```css
.layout-h {
  display: grid;
  grid-template-columns: 1fr 2.2fr 1fr;
  gap: 8px;
  height: 100%;
  padding-top: 10px;
}
```

---

## 变形方向

- 中央列比例可调为 40–60%（权重越高越突出）
- 两侧可不等宽（如左 20% / 中 50% / 右 30%）
- 中央可放：大型 SVG 图、大数字 KPI、雷达图、核心流程箭头
- 任一侧列可内部分行（`display:grid; grid-template-rows: repeat(2,1fr)`）
