# 布局 L · 左三行 + 右大图（A 镜像）

**适用：** 分析文字 + 信息图印证、三步骤说明 + 架构图、三维度 + 可视化
**结构基因：** 左侧垂直三分 + 右侧主视觉（与 layout-a 水平镜像）

## 空间计算

```
右图宽        = 360px
gap          = 12px
左栏宽        = 967 - 360 - 12 = 595px
padding-top  = 12px
每行高        = (580 - 12×2 - 12) ÷ 3 ≈ 181px
正文可用      ≈ 181 - 标题 22px - padding 16px = 143px → 约 70–100 字
```

---

## CSS

```css
.layout-l {
  display: grid;
  grid-template-columns: 1fr 360px;
  gap: 12px;
  height: 100%;
  padding-top: 12px;
}
.ll-left {
  display: grid;
  grid-template-rows: repeat(3, 1fr);
  gap: 12px;
}
```

---

## 变形方向

- 右图宽可调为 300–420px（视觉权重决定）
- 左侧行数可改为 2 行（内容少）或 4 行（内容多，字号降至 10.5px）
- 与 layout-a 搭配使用，同报告内形成镜像多样性
- 右侧区域可替换为：竖向时间轴、大型 KPI 数字块、SVG 信息图
