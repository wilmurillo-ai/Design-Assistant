# 布局 F · 左导航栏 + 右侧双行

**适用：** 章节索引式内容、带分类标签的对比、功能模块说明
**结构基因：** 窄导航列 + 宽主内容区垂直二分

---

## 结构图

```
┌──────────┬───────────────────────────────┐
│          │   ① 上行主内容                │
│  左导航  ├───────────────────────────────┤
│  160px   │   ② 下行次内容                │
└──────────┴───────────────────────────────┘
```

---

## 空间计算

```
padding-top  = 10px
左栏宽        = 160px
gap（列间）   = 12px
右栏宽        = 967 - 160 - 12 = 795px
每行高        = (580 - 10 - 10) ÷ 2 = 280px
正文可用      ≈ 280 - 标题 22px - padding 16px = 242px → 约 95–125 字
左栏正文可用  ≈ 580 - 10 - 22 - 16 = 532px（整列通高）
```

## CSS

```css
.layout-f {
  display: grid;
  grid-template-columns: 160px 1fr;
  gap: 12px;
  height: 100%;
  padding-top: 10px;
}
.lf-right {
  display: grid;
  grid-template-rows: repeat(2, 1fr);
  gap: 10px;
}
```

---

## 变形方向

- 左栏宽可调为 120–200px（标签短则窄，内容多则宽）
- 右侧可改为三行（信息密度高，每行约 160px）
- 左栏可内置：数字步骤列表 / 竖向时间轴 / 分类标签块 / 章节导航
- 右侧行高可不等分（如上 40% + 下 60%，突出下方主内容）
