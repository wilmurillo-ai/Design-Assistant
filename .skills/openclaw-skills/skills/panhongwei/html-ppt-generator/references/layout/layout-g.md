## 变形方向

- 上行高度可调为 200–300px（图重降低，文重提高）
- 上行可改为三列（三维度汇入单一结论区）
- 下方全宽可放：SVG 时间轴、横向对比条、大型架构图、全宽雷达图
- 上行两列宽度可不等分（60/40 或 40/60，突出一侧）
