# 布局 D · 2×2 四格 + 右侧大图

**适用：** 执行摘要、综合展示、矩阵分析
**结构基因：** 左侧四小格 + 右侧跨行主视觉

---

## 结构图

```
┌──────────┬──────────┬──────────────┐
│  ① 格    │  ② 格    │              │
├──────────┼──────────┤  右侧大图    │
│  ③ 格    │  ④ 格    │  286px       │
└──────────┴──────────┴──────────────┘
```

---

## 空间计算

```
左侧总宽    = 967 - 8×2 - 286 = 665px
每小格宽    = (665 - 8) ÷ 2 ≈ 328px
每小格高    = (580 - 8 - 10) ÷ 2 = 281px
右侧图高    = 580 - 10 = 570px（跨两行）
正文可用    ≈ 281 - 标题 22px - padding 16px = 243px → 约 65–90 字
```

## 变形方向

- 右侧大图宽可调为 240–340px（跟随视觉权重）
- 左侧四格可改为三格（上方一行跨列横幅 + 下方两格）
- 右侧区域可替换为大标题文字块 / 大 KPI 数字 / 时间轴
- 可镜像为"左侧大图 + 右侧 2×2"结构
