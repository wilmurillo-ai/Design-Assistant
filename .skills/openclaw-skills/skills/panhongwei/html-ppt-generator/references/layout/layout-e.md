**适用：** 主题总述、成果亮点展示、章节概览

## 空间计算

```
padding-top  = 10px
横幅高        = 160px
gap          = 10px（横幅与下行之间）
下行高        = 580 - 10 - 160 - 10 = 400px
列宽          = (967 - 8×2) ÷ 3 ≈ 317px
正文可用（横幅）≈ 160 - 标题 22px - padding 16px = 122px → 约 45–65 字
正文可用（下行）≈ 400 - 标题 22px - padding 16px = 362px → 约 120–160 字
```

---

## 变形方向

- 横幅高度可调为 120–200px（内容量决定）
- 底部三列可改为二列（各更宽）或四列（信息密度高）
- 横幅可替换为大 SVG 图表、横向时间轴、横向对比条
- 横幅内部可再分为左文右图（`display:flex; justify-content:space-between`）
