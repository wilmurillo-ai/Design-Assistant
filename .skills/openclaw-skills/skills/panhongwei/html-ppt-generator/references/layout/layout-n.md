# 布局 N · 三横带（上窄 + 中宽 + 下窄）

**适用：** 背景-核心-结论三段式、问题-方案-效果、引言-分析-行动
**结构基因：** 三行高度不等，中行为主要展示区

---

## 结构图

```
┌───────────────────────────────────────┐
│   ① 上带（背景 / 引言 / 分类标签）    │  100px
├───────────────────────────────────────┤
│   ② 中带（核心内容 / 大图 / 主分析）  │  330px（1fr）
├───────────────────────────────────────┤
│   ③ 下带（结论 / 指标 / 行动项）      │  120px
└───────────────────────────────────────┘
```

---

## 空间计算

```
padding-top  = 10px
上带高        = 100px
gap          = 10px × 2 = 20px
下带高        = 120px
中带高（1fr） = 580 - 10 - 100 - 20 - 120 = 330px
正文可用（上）≈ 100 - padding 16px = 84px → 约 30–45 字（精炼引言）
正文可用（中）≈ 330 - 标题 22px - padding 16px = 292px → 约 105–130 字
正文可用（下）≈ 120 - padding 16px = 104px → 约 mini-row 或 3–5 行动项
```

---

## CSS

```css
.layout-n {
  display: grid;
  grid-template-rows: 100px 1fr 120px;
  gap: 10px;
  height: 100%;
  padding-top: 10px;
}
```

---

## 变形方向

- 各带内部可再分列（如中带分左右两列：`display:grid; grid-template-columns:60% 40%`）
- 上带可放：mini-row KPI 横排 / 分类标签 / 引言大字
- 下带可放：行动清单 / 迷你对比表 / 小结数据条
- 上下带高度可互换（小上大下 → 大上小下，`120px 1fr 100px`）
