## 空间计算

```
padding-top  = 10px
顶行高        = 210px
gap          = 10px
底行高        = 580 - 10 - 210 - 10 = 350px
顶行列宽      = (967 - 8) ÷ 2 ≈ 479.5px
正文可用（顶）≈ 210 - 标题 22px - padding 16px = 172px → 约 60–80 字
正文可用（底）≈ 350 - 标题 22px - padding 16px = 312px → 约 110–135 字
```

## 变形方向

- 顶行可改为三格（三因素汇聚为单一结论，`repeat(3, 1fr)`）
- 顶行高度可调为 160–250px
- 底部全宽可放：横向对比表、决策矩阵、路线图、大型 SVG 图
- 顶行两格可不等宽（如 45/55，一侧数据一侧解读）
