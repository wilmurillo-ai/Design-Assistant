## 空间计算

```
左图宽   = 366px
gap      = 12px
右侧宽   = 967 - 366 - 12 = 589px
右侧行高 = (580 - 12×2 - padding-top 12) ÷ 3 ≈ 181px
正文可用 ≈ 181 - 标题 24px - padding 16px = 141px → 约 70–100 字
```

---

## 变形方向

- 左图宽可调整为 300–420px（跟随内容）
- 右侧行数可改为 2 行（内容少）或 4 行（内容多，需降低字号）
- 左侧可换为 SVG 流程图 / 大数字 KPI 区 / 大标题文字块
