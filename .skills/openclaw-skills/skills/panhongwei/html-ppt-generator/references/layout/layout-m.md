# 布局 M · 四列等宽竖排

**适用：** 四要素并列展示、四阶段流程、四维度独立分析
**结构基因：** 全宽等分四列，每列内容纵向独立

---

│           │           │           │           │
└───────────┴───────────┴───────────┴───────────┘

---

## 空间计算

```
padding-top  = 10px
每列宽        = (967 - 8×3) ÷ 4 ≈ 235px
高度          = 580 - 10 = 570px
正文可用      ≈ 570 - 标题 22px - mini 40px - padding 16px = 492px → 约 180–220 字
```

---

## CSS

```css
.layout-m {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  gap: 8px;
  height: 100%;
  padding-top: 10px;
}
```

---

## 变形方向

- 列宽可不等分（重点列加宽：`30% 25% 25% 20%`）
- 某列可内置 mini 数据行（`display:grid; grid-template-rows: 1fr auto`）
- 列内顶部可加序号大字（Step 1/2/3/4）形成流程感
- 列间可加视觉箭头连接符（`::after` 或内嵌 SVG）
