# 🔴 HTML 生成强制检查清单

> **用途：** 生成任何 HTML 文件前，必须逐项打勾。任何一项未完成 = 禁止生成。

---

## 前置条件检查（对话级别）

```
□ [ ] 1. S0-1 联网核实数据已完成（带来源的数字）
□ [ ] 2. S0-2 主题定性已输出（3 句话）
□ [ ] 3. S0-3 逐页内容指令已输出（表格形式）
□ [ ] 4. S0-3.5 逐页内容扩写已完成（每页 4 项）
□ [ ] 5. S0-4 视觉规格声明已输出
□ [ ] 6. 规划表已输出（三元组：页码 | 主题 | 布局 | 配色 | Tc 变体 | 图表）
□ [ ] 7. 用户已明确确认（回复"继续"/"可以"/"开始生成"等）
```

**任何一项为否 = 立即停止，返回第零阶段补充。**

```

□ [ ] 2. 四区高度精确
       - Header: 72px ✓

       合计：720px ✓

       - height: calc(580px - padding-top) ✓
       - padding-top ≤ 8px ✓

□ [ ] 5. 卡片内容密度
       - 标题行 ✓
       - 正文段落（≥60 字，≥2 个数字）✓
       - 第三层组件（mini 卡片/SVG/数据网格）✓

□ [ ] 6. 75% 填充率
       - 内容高度 / 卡片可用高度 ≥ 75% ✓
       - 背景色面积 < 内容面积 ✓

□ [ ] 7. SVG 图表至少 1-3 个 ✓

□ [ ] 8. 摘要栏有实质内容（非空，≤40 字）✓

□ [ ] 9. 配色正确（来自规划表）✓

□ [ ] 10. 布局正确（来自规划表，不与上两页重复）✓
```

**任何一项为否 = 删除文件，重新生成。**

```
□ [ ] 6. 每页至少有 1 个 SVG 图表

       - 内容页≥18KB ✓
       - 特殊页≥8KB ✓
```

---

## 违规处理

**发现以下行为 = 删除全部 HTML，重新执行流程：**

1. ❌ 未输出 S0-3.5 扩写就直接生成 HTML
2. ❌ 未等待用户确认就生成 HTML
3. ❌ 文件体积<18KB 仍输出
4. ❌ 卡片正文<60 字或数字<2 个
5. ❌ 缺少第三层数据组件
6. ❌ 连续 3 页使用相同布局
