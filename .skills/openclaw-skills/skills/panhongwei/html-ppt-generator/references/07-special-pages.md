# 07 · 特殊页面规范（Design-System 版）

> 特殊页面继承激活的 **Tc 模板** CSS 变量与 design-system 风格，不使用独立 HTML 模板。
> 所有颜色、字体、背景纹理均从 Tc 变量派生，不得硬编码非语义色。

---

## 特殊页面识别规则

```
P00（可选）：封面页       — 第一印象，定义报告气质
P0N（可选）：目录页       — 内容导航，视觉梳理结构
章节节点处：  章节分隔页   — 每个主要章节前插入（长报告 ≥8 页时用）
最后一页：    结尾页       — 明确结束信号
```

---

## 一、封面页（Cover Page）

### 设计原则

```
- 文字极少（≤5 元素：主标题 + 副标题 + 日期/机构/总页数）
- 视觉装饰面积 ≥ 70%（背景渐变 + 光晕 + 几何线条）
- 气质必须与 Tc 模板一致，禁止另起炉灶
- 禁止卡片布局、数据图表
```

### 四区配置（封面专用）

```
方案 A 全屏沉浸：  Header 0px  + Content 700px + Summary 0px  + Footer 20px
方案 B 保留摘要：  Header 0px  + Content 650px + Summary 50px + Footer 20px
方案 C 标准四区：  Header 72px + Content 580px + Summary 48px + Footer 20px
```

### 必要元素与 Tc 变量映射

| 元素 | 规格 | CSS 变量来源 |
|------|------|-------------|
| 背景 | design-system 风格（噪点/光晕/渐变按 Tc 风格） | `var(--bg)` + Tc `body::before` |
| 顶部装饰线 | 2px 渐变横线 | `transparent → var(--p) → transparent` |
| eyebrow 眉标 | 9-10px uppercase letter-spacing 3px | `var(--p)`，两侧各 40px 细线 |
| 主标题 | 展示字体 44-54px，渐变裁剪 | `--font-display`，`var(--t) → var(--p)` 渐变 |
| 副标题 | 正文字体 12-14px light | `--font-body`，`var(--mt)` |
| 元信息行 | 机构 · 日期 · 总页数，10px | `var(--dt)`，圆点分隔 `var(--dt)` |
| 底栏 20px | CONFIDENTIAL + 进度条 + 页码标记 | `var(--dt)`，进度条 `var(--p)` |

### 结构布局

```
┌──────────────────────────── 1017px ─────────────────────────────┐
│ ▓ 顶部 2px 渐变装饰线（--p）                                    │
│                                                                  │

│                                                                  │
│              [ eyebrow · 主色 · 两侧细线 ]                       │
│                                                                  │
│              [ 主标题 · 44-54px · 展示字体 · 渐变裁剪 ]          │
│                                                                  │
│              [ 副标题 · 12-14px · --mt ]                         │
│              [ 机构 · 日期 · 页数 · --dt ]                       │
│                                                                  │
│ [CONFIDENTIAL · --dt]    [进度条 · --p]    [COVER · --dt]       │
└──────────────────────────────────────────────────────────────────┘
```

## 二、目录页（Table of Contents）

### 设计原则

```
- 清晰列出全部章节（≤12项），每项含序号 + 标题 + 页码
- 不需要三层内容结构，不需要 SVG 图表
- 使用 Tc 卡片样式（var(--card)、var(--bd)）
- 当前激活章节可高亮
```

### 必要元素与 Tc 变量映射

| 元素 | 规格 | CSS 变量来源 |
|------|------|-------------|
| 条目背景 | 圆角卡片，hover 无 | `var(--card)` + `var(--bd)` |
| 激活条目 | 高亮当前章节 | `var(--pm)` + `var(--p)` border |
| 序号 | 22-28px bold，低透明 | `var(--p)` opacity 0.35（激活 1.0） |
| 章节标签 | 9px uppercase eyebrow | `var(--dt)` |
| 章节标题 | 12px font-weight 600 | `var(--t)` |
| 页码 | monospace 右对齐 | `var(--dt)` |
| 布局 | 2列网格，gap 6-8px，padding-top 10px | height: calc(580px - 10px) |

## 三、章节分隔页（Chapter Divider）

### 设计原则

```
- 内容极少：序号 + 章节名 + 描述（≤30 字）
- 视觉冲击力强：大字号 / 大色块 / 满版装饰
- 不使用卡片布局，不使用 SVG 图表
- 继承 Tc 背景和字体
```

### 必要元素与 Tc 变量映射

| 元素 | 规格 | CSS 变量来源 |
|------|------|-------------|
| 装饰背景大字 | 280-320px，极低透明度背景 | `var(--t)` opacity 0.02-0.04 |
| eyebrow + 右侧细线 | 9px uppercase + 30px 线 | `var(--p)` |
| 章节序号 | 80px bold，渐变裁剪 | `var(--t) → var(--p)` 渐变 |
| 章节名 | 36px bold | `var(--t)` |
| 描述文字 | 12px line-height 1.6 | `var(--mt)` max-width 420px |
| 背景装饰 | 与封面同气质 | Tc `body::before` + 光晕 |

## 四、结尾页（End Page）

### 设计原则

```
- 明确结束信号（谢谢 / Q&A / 联系我们）
- 可选：关键结论一句话总结（≤30 字）
- 可选：联系方式/行动号召
- 风格与封面呼应，使用 Tc 背景和装饰
```

### 必要元素与 Tc 变量映射

| 元素 | 规格 | CSS 变量来源 |
|------|------|-------------|
| 主词 | 展示字体 60-70px，部分主色 | `var(--t)` + `var(--p)` |
| 副文 | 13px | `var(--mt)` margin-bottom 32px |
| 联系卡片 | 半透明圆角卡片，含圆形图标 | `rgba(var(--bg),0.04)` + `var(--bd)` |
| 图标圆点 | 14px 圆形 | `var(--p)` |
| 背景装饰 | 同封面风格，光晕+纹理 | Tc `body::before` |

---

## 特殊页面质量检查

```
□ 封面文字元素 ≤ 5 个？
□ 封面视觉装饰面积 ≥ 70%（非纯文字区域）？
□ 封面气质与后续页 Tc 模板一致（背景/字体/主色相同）？
□ 目录条目含序号 + 标题 + 页码三要素？
□ 章节分隔描述 ≤ 30 字，无卡片网格？
□ 结尾页是明确结束信号，非内容页突然截断？
□ 所有颜色来自 Tc CSS 变量（无硬编码非语义色）？
```
