# Cloud Orange Style Template - Design Specification

## Overview

| Property | Value |
|----------|-------|
| **Template Name** | cloud_orange (Cloud Orange Style Template) |
| **Design Philosophy** | Professional cloud/tech aesthetic with bold orange accents |
| **Theme Mode** | Dark theme (deep navy background + orange accent) |
| **Target Audience** | Cloud architects, tech leaders, engineering teams |

## Color Palette

| Color Name | Hex Code | Usage |
|-----------|----------|-------|
| **Deep Navy** | `#232F3E` | Primary background |
| **Card Navy** | `#2A3B4D` | Card backgrounds, content areas |
| **Cloud Orange** | `#FF9900` | Primary accent, titles, key data highlights |
| **Accent Blue** | `#527FFF` | Secondary accent, links, supporting elements |
| **White** | `#FFFFFF` | Primary text on dark backgrounds |
| **Light Gray** | `#D5DBDB` | Body text |
| **Mid Gray** | `#A0AAB2` | Secondary text, captions |
| **Dark Panel** | `#1A2332` | Right-side card panels, deep contrast areas |

## Typography

| Element | Font | Size | Color |
|---------|------|------|-------|
| Main Title | Arial Bold / Microsoft YaHei Bold | 36-44pt | Cloud Orange `#FF9900` |
| Subtitle | Arial / Microsoft YaHei | 20-24pt | White `#FFFFFF` |
| Section Title | Arial Bold / Microsoft YaHei Bold | 28-32pt | Cloud Orange `#FF9900` |
| Body Text | Calibri / Microsoft YaHei | 14-18pt | Light Gray `#D5DBDB` |
| Caption | Calibri / Microsoft YaHei | 10-12pt | Mid Gray `#A0AAB2` |
| Data/Numbers | Arial Bold | 36-48pt | Cloud Orange `#FF9900` |

## Layout Structure

### Cover Page
- Full Deep Navy background
- Cloud Orange accent bar at top (height 6px)
- Title in Cloud Orange, centered or left-aligned
- Subtitle in White below title
- Optional decorative geometric shapes in `#2E3F52`

### Section Divider Page
- Deep Navy background
- Large section number in Cloud Orange
- Section title in White
- Thin Cloud Orange horizontal rule

### Content Page
- Deep Navy background
- Top: Cloud Orange thin bar (4px)
- Page title in Cloud Orange, left-aligned
- Content area with Light Gray body text
- Cards use Card Navy `#2A3B4D` with 1px border
- Key metrics highlighted in Cloud Orange

### End Page
- Deep Navy background
- "Thank You" or closing message in Cloud Orange
- Contact info in Light Gray
- Cloud Orange accent elements

## Design Principles

1. **High contrast**: Orange on dark navy for maximum readability
2. **Technical precision**: Clean grid layouts, aligned elements
3. **Data-forward**: Large numbers and metrics in orange stand out
4. **Minimal decoration**: Let content breathe, avoid clutter
5. **Professional tone**: Suitable for executive and technical audiences

## Decorative Elements

| Element | Color | Usage |
|---------|-------|-------|
| Top bar | Cloud Orange `#FF9900` | Page header decoration |
| Side accent | Accent Blue `#527FFF` | Optional left border on cards |
| Background shapes | `#2E3F52`, `#1E2D3D` | Subtle geometric overlays |
| Divider lines | `#354A5F` | Section separators |

## Checklist

- [ ] Deep Navy background consistently applied
- [ ] Cloud Orange used for titles and key highlights only
- [ ] Body text in Light Gray for readability
- [ ] Cards use Card Navy with subtle borders
- [ ] No more than 2 accent colors per slide
- [ ] Font sizes follow hierarchy
- [ ] Sufficient contrast ratios maintained
