- 实用可执行性
- 边界控制

合格线：32 / 40

## 1. 被辞退赔偿
用户提问：我工作 3 年被辞退，月薪 12000，能赔多少？

### Round 1
- 结构清晰度：8
- 专业可信度：7
- 实用可执行性：7
- 边界控制：9
- **总分：31 / 40**

### 本轮补强后判断
新增了：
- 劳动纠纷情景规则
- 劳动赔偿区间输出模板

现在输出可以稳定做到：
- 先列已确认事实
- 明确还缺的关键事实
- 用“合法解除 / 违法解除”等情景表给区间
- 把证据需求和下一步建议结构化写出来

### Round 2 评分
- 结构清晰度：9
- 专业可信度：8
- 实用可执行性：8
- 边界控制：9
- **总分：34 / 40（由待加强升为合格）**

### 改善点总结
- 从“容易只有方向，没有稳定表达”
- 提升到“有固定输出模板，可给区间判断和补证据建议”

### 仍可继续补的点
- 工龄折算与工资基数争议还可以再细
- 可继续补试用期、未签合同、加班争议下的子模板

## 2. 值不值得起诉
用户提问：对方欠我 3 万，起诉值不值得？

### Round 1
- 结构清晰度：7
- 专业可信度：7
- 实用可执行性：7
- 边界控制：9
- **总分：30 / 40**

### 本轮补强后判断
新增了：
- “值不值得起诉”结构化判断模板

现在输出可以稳定做到：
- 先给初步结论
- 再按收益、成本、证据、执行可能性四个维度判断
- 最后收口成结构化行动建议

### Round 2 评分
- 结构清晰度：9
- 专业可信度：8
- 实用可执行性：8
- 边界控制：9
- **总分：34 / 40（由待加强升为合格）**

### 改善点总结
- 从“有框架但不够产品化”
- 提升到“能稳定输出判断逻辑和下一步动作”

### 仍可继续补的点
- 可增加小额欠款、执行难、证据弱三类专门模板
- 可增加“先催收还是先保全”的辅助判断
