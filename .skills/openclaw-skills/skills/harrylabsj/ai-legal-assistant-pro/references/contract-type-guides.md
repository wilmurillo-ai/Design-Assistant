# 按合同类型的高频风险提示

本文件用于在做合同初筛时，按合同类型快速补充更贴近场景的风险检查点。

## 1. 劳动合同

重点关注：
- 试用期是否过长、是否与合同期限不匹配
- 工资结构是否故意拆散，导致加班费、补偿基数不清
- 调岗调薪条款是否给了用人单位过宽单方权利
- 绩效考核、解除条件是否模糊到可被滥用
- 保密、竞业限制是否过宽，补偿是否缺失

建议输出方向：
- 明确指出哪些条款可能导致员工权利受损
- 区分“管理需要”与“单方过宽权利”
- 如涉及竞业限制，提醒核对补偿与期限

## 2. 房屋租赁合同

重点关注：
- 押金退还条件是否过于苛刻
- 维修责任是否全部压给承租人
- 提前解约责任是否严重失衡
- 转租、续租、涨租规则是否不清
- 水电、物业、税费等承担方式是否模糊

建议输出方向：
- 明确押金退还触发条件和期限
- 明确房屋维修、自然损耗、违约责任边界
- 关注“房东可随时解约、租客不能解约”类失衡条款

## 3. 服务合同

重点关注：
- 服务内容、交付标准、验收标准是否不清
- 付款节点是否完全绑定对方主观满意
- 需求变更是否没有书面确认机制
- 知识产权、成果归属是否一刀切
- 违约责任和赔偿范围是否过宽

建议输出方向：
- 先看“做什么、做到什么算完成、何时付款”这三件事是否清楚
- 强调变更确认和验收机制的重要性

## 4. 技术开发合同

重点关注：
- 需求范围、开发边界、交付件清单是否明确
- 验收标准和缺陷修复机制是否清楚
- 源代码、文档、部署材料是否属于交付范围
- 知识产权归属是否区分既有技术与项目成果
- 维护期、支持范围、响应时间是否明确

建议输出方向：
- 强调需求文档、里程碑、验收标准、知识产权边界
- 对“全部成果均归甲方所有”类条款做重点提示

## 5. 借款协议

重点关注：
- 借款金额、放款方式、到账证据是否明确
- 利息、违约金、逾期责任是否过高或冲突
- 还款时间、分期规则、提前还款规则是否清楚
- 担保条款是否完整
- 争议解决与证据保存安排是否明确

建议输出方向：
- 强调金额、转账记录、期限、利息和违约责任的一致性
- 如涉及个人之间借款，提醒保留付款凭证和催收记录

## 6. 采购 / 销售合同

重点关注：
- 标的物规格、数量、质量标准是否明确
- 验收与异议期限是否合理
- 交货、运输、风险转移时点是否清楚
- 发票与付款是否被不合理绑定
- 退换货和违约责任是否失衡

建议输出方向：
- 明确货权、风险、验收、付款、退换货规则
- 警惕“先开发票才付款”被用于无限拖延付款

## 快速使用建议

先判断合同类型，再叠加：
1. 通用风险检查（见 `risk-patterns.md`）
2. 类型化检查（见本文件）
3. 样例对照（见 `risk-scan-examples.md`）
