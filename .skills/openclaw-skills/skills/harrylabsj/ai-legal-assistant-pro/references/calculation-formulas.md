# 计算规则

本文件用于做**粗略、透明、可追溯**的第一轮估算。输出时要写清楚：使用了哪些事实、套用了什么公式、哪些地方仍不确定。

## 1. 中国民事案件诉讼费（常见财产案件）

按下列分段规则估算：

| 标的额 | 规则 |
|---|---|
| 不超过 10,000 元 | 50 元 |
| 10,001 - 100,000 元 | 标的额 × 2.5% - 200 |
| 100,001 - 200,000 元 | 标的额 × 2% + 300 |
| 200,001 - 500,000 元 | 标的额 × 1.5% + 1,300 |
| 500,001 - 1,000,000 元 | 标的额 × 1% + 3,800 |
| 1,000,001 - 2,000,000 元 | 标的额 × 0.9% + 4,800 |
| 2,000,001 - 5,000,000 元 | 标的额 × 0.8% + 6,800 |
| 5,000,001 - 10,000,000 元 | 标的额 × 0.7% + 11,800 |
| 10,000,001 - 20,000,000 元 | 标的额 × 0.6% + 21,800 |
| 超过 20,000,000 元 | 标的额 × 0.5% + 41,800 |

### 建议输出格式
- 标的额
- 使用公式
- 预估诉讼费
- 说明：不含律师费、保全费、差旅费、鉴定费等其他成本

## 2. 劳动经济补偿（高层级粗算）

仅适合做第一轮区间估算，不适合在事实不清时下最终结论。

### 常见简化逻辑
- 经济补偿通常会用 **N**（按工作年限）来估算
- 违法解除在常见场景下可能涉及 **2N**
- 部分合法解除场景可能涉及 **N+1** 或结合通知情况判断

### 最低必要事实
- 月工资基数
- 工作年限
- 解除日期
- 用人单位给出的解除理由
- 是否提前通知
- 是否签过书面劳动合同

### 安全输出规则
如果事实不完整，不要直接给唯一数字；优先给**区间**或**情景表**。

### 情景表示例
| 情景 | 依据 | 预估结果 |
|---|---|---|
| 如属于合法经济性解除 | N 或 N+1 逻辑 | X - Y 元 |
| 如属于违法解除 | 2N 逻辑可能适用 | A - B 元 |

## 3. 值不值得起诉的初步判断

当用户问“值不值得起诉”时，至少比较：
- 可能主张金额
- 预估诉讼费
- 律师 / 时间 / 证据成本
- 回款可能性与紧迫性

### 建议输出格式
- 潜在收益
- 直接程序成本
- 主要不确定性
- 初步建议：起诉 / 先谈判 / 先补证据

## 提醒

要始终说明：当地法院做法、最新规则、案件事实细节，都会显著影响最终结果。
