# 值不值得起诉：结构化判断模板

本文件用于回答“值不值得起诉”这类问题，避免只说“值得 / 不值得”，而是把判断过程结构化。

## 适用场景

- 对方欠我钱，要不要起诉
- 金额不大，打官司值不值
- 我现在该起诉还是先谈

## 推荐输出结构

### 1. 先给初步结论
可用表达：
- 初步看，值得考虑起诉，但建议先补齐证据后再决定。
- 初步看，更适合先催收 / 谈判，不宜马上起诉。
- 初步看，是否起诉主要取决于证据和执行可能性，而不只是金额大小。

### 2. 四个核心判断维度

#### （1）可能收益
- 可主张金额大概多少
- 除本金外，是否还可能主张违约金、利息、损失

#### （2）直接成本
- 诉讼费大概多少
- 是否还可能产生律师费、保全费、时间成本

#### （3）证据强弱
- 有没有合同 / 借条 / 转账记录 / 对账记录 / 催收记录
- 是否能证明债权债务关系和金额

#### （4）执行可能性
- 对方是否有财产线索
- 即便胜诉，是否有回款希望

### 3. 用结构化建议收口
| 情况 | 建议 |
|---|---|
| 证据强、金额尚可、对方有财产线索 | 可考虑起诉 |
| 金额不大但证据弱 | 先补证据 / 先催收谈判 |
| 证据尚可但回款希望很低 | 谨慎评估时间与执行成本 |

### 4. 给用户最直接的下一步动作
- 先整理证据清单
- 先发催款通知 / 律师函思路
- 先算诉讼费
- 再决定是否起诉

## 输出要求

- 不只看金额，不只看诉讼费
- 不给绝对化判断
- 一定要把证据与执行风险写进去
- 最后给出下一步行动建议，而不是只给态度

## 推荐结尾提醒

“值不值得起诉”本质上是收益、成本、证据和执行可能性的综合判断。当前回答仅供第一轮决策参考。
