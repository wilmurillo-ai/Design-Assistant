# 劳动纠纷赔偿估算输出模板

本文件用于把“被辞退赔偿多少”这类问题输出得更稳，不乱报唯一数字，同时让用户看清不同情景下的区间差异。

## 适用场景

- 被辞退赔偿多少
- 公司这样解除我要不要仲裁
- 合法解除和违法解除差多少

## 推荐输出结构

### 1. 先列已确认事实
- 月工资基数：
- 工作年限：
- 解除日期：
- 公司给出的解除理由：
- 是否提前通知：
- 是否有书面劳动合同：

### 2. 明确仍未确认的关键事实
- 是否属于合法解除情形
- 是否存在严重违纪、不能胜任工作等争议点
- 是否有完整通知、考核、规章制度证据

### 3. 用情景表输出，不直接拍唯一数字
| 情景 | 常见估算逻辑 | 预估区间 | 备注 |
|---|---|---|---|
| 如被认定为合法且需支付经济补偿 | N 或 N+1 | X - Y 元 | 视通知和具体情形 |
| 如被认定为违法解除 | 2N 方向 | A - B 元 | 需结合证据判断 |

### 4. 提醒影响结果的关键变量
- 工资基数是否存在争议
- 工龄如何计算
- 是否有调岗、降薪、绩效考核等前置事实
- 是否存在试用期、未签合同、社保等附加争议

### 5. 给下一步建议
- 先补证据
- 先谈判
- 先申请劳动仲裁

## 输出要求

- 不把“可能”说成“确定”
- 不在事实不清时给唯一金额
- 一定要把用户还缺什么事实说出来
- 用“情景区间 + 证据建议”代替拍脑袋结论

## 推荐结尾提醒

劳动纠纷金额判断高度依赖事实和证据。当前结果仅供第一轮估算，正式维权前应结合完整材料进一步核对。
