# Local Services and Entertainment

Use this file when the user asks about massage, spa, beauty, hair salons, KTV, escape rooms, gyms, parent-child venues, pet services, or similar local lifestyle merchants.

## Core rule

For services, “评分高” is much less important than:
- hidden cost risk
- booking friction
- upselling pressure
- cleanliness
- privacy / comfort
- whether reviews match the exact service type the user needs

## Massage / spa
Check for:
- hidden add-ons
- time缩水 complaints
- hygiene / disposable item issues
- aggressive card sales
- whether group-buy users are treated differently

## Beauty / hair salon
Check for:
- forced product sales
- haircut quality consistency by stylist
- whether the listed stylist is actually available
- whether low-price listings are only traffic bait

## KTV / entertainment
Check for:
- room equipment age
- sound / mic quality
- weekend reservation difficulty
- minimum spend or time restrictions
- extra charges for snacks / private room upgrades

## Escape rooms / immersive play
Check for:
- puzzle logic quality
- staff hint style
- forced group joining
- actual duration vs advertised duration
- whether horror intensity matches description

## Parent-child / pet / training
Check for:
- cleanliness
- staff patience
- safety complaints
- whether high scores come from aesthetics rather than substance

## Output pattern

> 结论：适合去 / 可去但要防坑 / 不建议优先  
> 关键原因：2–4 条  
> 风险提醒：推销、预约、隐形收费、卫生、体验波动

## Good caution example

> 这家店的分数不低，但更像“第一次去容易被价格吸引”的类型。  
> 真正要注意的是评论里反复出现的强推销和加价项。  
> 如果你只想安静做个项目，不想被推卡，建议换一家更稳的。
