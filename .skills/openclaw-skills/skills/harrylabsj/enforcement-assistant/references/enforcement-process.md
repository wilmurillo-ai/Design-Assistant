# 民事强制执行程序指南

## 一、执行申请条件

### 1. 申请执行的依据

以下法律文书可以申请强制执行：

| 文书类型 | 生效条件 | 管辖法院 |
|---------|---------|---------|
| 民事判决书 | 上诉期届满未上诉 | 一审法院或财产所在地法院 |
| 民事调解书 | 双方签收后生效 | 一审法院或财产所在地法院 |
| 民事裁定书（财产保全、先予执行等） | 送达即生效 | 作出裁定法院 |
| 支付令 | 15日异议期届满无异议 | 作出支付令法院 |
| 仲裁裁决书 | 作出即生效 | 被执行人住所地或财产所在地中级法院 |
| 公证债权文书 | 载明强制执行承诺 | 被执行人住所地或财产所在地法院 |
| 行政判决书、裁定书 | 上诉期届满 | 一审法院 |
| 刑事附带民事判决书 | 生效后 | 一审法院 |

### 2. 申请执行期限

**一般规定**：二年

- 起算时间：从法律文书规定履行期间的最后一日起计算
- 分期履行：从规定的每次履行期间的最后一日起计算
- 未规定履行期：从法律文书生效之日起计算

**期限中止、中断**：
- 适用法律有关诉讼时效中止、中断的规定
- 达成执行和解协议：期限重新计算

**⚠️ 超期后果**：法院可能裁定不予执行

### 3. 申请执行的条件

1. 申请人是生效法律文书确定的权利人或其继承人、权利承受人
2. 申请执行的法律文书已经生效
3. 申请执行人是生效法律文书确定的权利人
4. 义务人在生效法律文书确定的期限内未履行义务
5. 属于受申请执行的人民法院管辖

## 二、执行管辖

### 1. 级别管辖

| 法院级别 | 管辖范围 |
|---------|---------|
| 基层人民法院 | 一般执行案件 |
| 中级人民法院 | 仲裁裁决执行、涉外案件、重大复杂案件 |
| 高级人民法院 | 特别重大案件 |

### 2. 地域管辖

**一般原则**：
- 第一审人民法院
- 与第一审人民法院同级的被执行财产所在地法院

**特殊规定**：
- 仲裁裁决：被执行人住所地或财产所在地中级法院
- 公证债权文书：被执行人住所地或财产所在地法院

### 3. 管辖异议

- 当事人可在收到执行通知书之日起10日内提出
- 法院审查后作出裁定
- 对裁定不服可向上级法院申请复议

## 三、执行申请程序

### 1. 准备材料

**必备材料**：
1. 申请执行书
2. 生效法律文书副本
3. 申请执行人身份证明
4. 生效证明（部分法院要求）

**辅助材料**：
- 委托代理手续
- 被执行人财产线索
- 已采取保全措施的证明

### 2. 立案审查

**审查内容**：
- 是否符合受理条件
- 是否属于本院管辖
- 申请执行期限是否届满
- 法律文书是否生效
- 材料是否齐全

**审查期限**：7日内

**处理结果**：
- 符合条件：立案
- 不符合条件：裁定不予受理
- 材料不齐：通知补正

### 3. 执行立案

**立案后流程**：
1. 编立执行案号
2. 向被执行人发出执行通知
3. 财产调查
4. 采取执行措施

## 四、执行措施

### 1. 财产调查措施

**法院调查**：
- 网络执行查控系统查询
- 现场调查
- 责令被执行人报告财产

**财产报告令**：
- 收到执行通知后报告当前及前一年财产情况
- 包括：收入、不动产、动产、财产性权利等
- 虚假报告的法律后果

### 2. 财产控制措施

| 措施 | 适用对象 | 期限 |
|-----|---------|------|
| 冻结存款 | 银行账户 | 1年 |
| 查封不动产 | 房屋、土地 | 3年 |
| 扣押动产 | 车辆、设备 | 2年 |
| 冻结股权 | 公司股权 | 3年 |
| 冻结投资权益 | 投资收益 | 3年 |

**续行查封、扣押、冻结**：
- 期限届满前办理
- 续行期限不得超过前款规定期限

### 3. 财产处分措施

**拍卖**：
- 首选处置方式
- 网络司法拍卖为主
- 评估价或市价作为参考

**变卖**：
- 拍卖流拍后可变卖
- 当事人同意可直接变卖

**以物抵债**：
- 双方同意
- 不损害其他债权人利益
- 不违反法律规定

### 4. 行为执行措施

- 强制交付财物
- 强制迁出房屋或退出土地
- 强制履行行为义务
- 替代履行

### 5. 制裁措施

**限制消费**：
- 乘坐交通工具限制
- 高消费场所消费限制
- 子女就读高收费学校限制
- 其他非生活必需消费限制

**纳入失信被执行人名单**：
- 有履行能力而拒不履行
- 妨碍、抗拒执行
- 规避执行
- 违反财产报告制度
- 违反限制消费令

**罚款、拘留**：
- 拒不履行
- 妨碍执行
- 虚假报告财产

**刑事责任**：
- 拒不执行判决、裁定罪
- 非法处置查封、扣押、冻结财产罪

## 五、执行和解

### 1. 执行和解的达成

**形式**：
- 双方自行和解
- 法院主持和解

**内容**：
- 变更履行义务主体
- 变更履行标的
- 变更履行期限
- 变更履行方式

### 2. 执行和解协议

**效力**：
- 具有合同效力
- 不具有强制执行力

**救济**：
- 不履行协议：申请恢复执行原生效法律文书
- 或：就执行和解协议提起诉讼

### 3. 执行担保

**形式**：
- 被执行人提供担保
- 第三人提供担保

**效果**：
- 暂缓执行
- 逾期仍不履行：执行担保财产或担保人财产

## 六、执行中止与终结

### 1. 执行中止

**情形**：
- 申请人表示延期执行
- 案外人提出确有理由的异议
- 一方当事人死亡，等待继承人
- 法人终止，等待权利义务承受人
- 法院认为应当中止的其他情形

**效果**：
- 暂停执行程序
- 保全措施继续有效
- 原因消除后恢复执行

### 2. 终结本次执行程序

**情形**：
- 被执行人无财产可供执行
- 发现的财产不能处置

**效果**：
- 暂时结案
- 发现财产后可恢复执行
- 不受申请执行期限限制

### 3. 执行终结

**情形**：
- 申请人撤销申请
- 据以执行的法律文书被撤销
- 被执行人死亡无遗产
- 追索赡养费等权利人死亡
- 被执行人破产无财产分配
- 法院认为应当终结的其他情形

**效果**：
- 执行程序彻底结束
- 一般不可恢复

## 七、执行救济

### 1. 执行异议

**当事人、利害关系人异议**：
- 对执行行为提出异议
- 收到执行通知书后10日内提出
- 法院15日内审查并裁定
- 不服可10日内申请复议

**案外人异议**：
- 对执行标的提出异议
- 异议期间中止执行
- 法院15日内审查
- 成立：中止执行；不成立：驳回

### 2. 执行异议之诉

**案外人异议之诉**：
- 案外人异议被驳回后提起
- 诉讼期间不停止执行
- 但可提供担保请求停止

**申请执行人异议之诉**：
- 案外人异议成立后提起
- 请求继续执行

**债务人异议之诉**：
- 对执行依据的实体权利提出异议

### 3. 执行监督

**上级法院监督**：
- 指令执行
- 提级执行
- 督促执行

**检察院监督**：
- 提出检察建议
- 对违法行为监督

## 八、执行费用

### 1. 申请执行费

**原则**：
- 申请执行时无需预交
- 执行后由被执行人承担

**计算标准**：
- 无执行金额：50-500元
- 有执行金额：按财产案件标准计算（但不预交）

### 2. 实际支出费用

- 评估费
- 拍卖费
- 公告费
- 保管费
- 运输费

**承担**：
- 一般从执行款中扣除
- 必要时申请人垫付

## 九、特殊执行程序

### 1. 参与分配

**条件**：
- 被执行人为公民或其他组织
- 财产不能清偿所有债权

**程序**：
- 取得执行依据的债权人可申请
- 制作分配方案
- 债权人可提出异议

### 2. 执行转破产

**条件**：
- 被执行人为企业法人
- 财产不能清偿债务
- 申请执行人或被执行人同意

**效果**：
- 移送破产审查
- 中止执行程序

### 3. 刑事涉财产执行

- 罚金
- 没收财产
- 追缴违法所得
- 责令退赔

## 十、执行工作规范

### 1. 执行公开

- 执行流程公开
- 执行措施公开
- 财产处置公开
- 执行结果公开

### 2. 执行期限

- 一般案件：6个月内执结
- 非诉执行：3个月内执结
- 特殊情况可延长

### 3. 执行款管理

- 一案一账号
- 30日内发放
- 暂缓发放的情形

## 十一、常见问题解答

### Q1：执行不到钱怎么办？

A：
1. 积极提供财产线索
2. 申请将被执行人列入失信名单
3. 申请限制高消费
4. 关注被执行人动态
5. 符合条件时申请恢复执行
6. 考虑执行和解

### Q2：被执行人转移财产怎么办？

A：
1. 及时向法院报告
2. 申请撤销转移行为
3. 追究拒执罪刑事责任
4. 申请追加被执行人

### Q3：执行和解后对方不履行怎么办？

A：
1. 申请恢复执行原生效法律文书
2. 或：就执行和解协议起诉

### Q4：对执行行为不服怎么办？

A：
1. 提出执行异议
2. 对裁定不服申请复议
3. 案外人可提起执行异议之诉

### Q5：如何加快执行进度？

A：
1. 主动提供财产线索
2. 配合法院调查
3. 及时联系执行法官
4. 必要时申请上级法院监督
