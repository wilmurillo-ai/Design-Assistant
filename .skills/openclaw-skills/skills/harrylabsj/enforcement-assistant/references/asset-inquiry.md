# 被执行人财产调查指南

## 一、财产调查的重要性

执行案件能否顺利执结，很大程度上取决于能否找到被执行人的可供执行财产。财产调查是执行工作的核心环节。

## 二、法院财产调查措施

### 1. 网络执行查控系统

**查询范围**：
- 银行存款
- 车辆登记
- 不动产登记
- 股权登记
- 证券账户
- 支付宝、微信等网络资金

**查询流程**：
1. 立案后系统自动发起查询
2. 反馈查询结果
3. 对发现的财产采取控制措施

### 2. 现场调查

**调查内容**：
- 被执行人住所
- 经营场所
- 财产存放地
- 关联企业

### 3. 财产报告令

**报告内容**：
- 收入、银行存款、现金
- 不动产、动产
- 财产性权利
- 其他财产

**报告期限**：收到执行通知后

## 三、申请人可提供的财产线索

### 1. 银行存款线索

**常见信息**：
- 开户银行名称
- 银行账号
- 开户地点
- 账户余额（大致）

**获取途径**：
- 交易记录
- 转账凭证
- 合同文件
- 日常往来

### 2. 房产线索

**房产信息**：
- 房屋地址
- 产权证号
- 建筑面积
- 登记状态

**查询方式**：
- 不动产登记中心
- 房屋中介
- 小区物业
- 邻里了解

### 3. 车辆线索

**车辆信息**：
- 车牌号码
- 车辆品牌型号
- 车辆识别代码
- 登记地

**查询方式**：
- 车管所查询
- 停车场观察
- 日常出行记录

### 4. 股权/投资线索

**股权信息**：
- 持股公司名称
- 持股比例
- 认缴出资额
- 公司注册地

**查询方式**：
- 企业信用信息公示系统
- 公司章程
- 工商登记档案

### 5. 到期债权线索

**债权信息**：
- 债务人名称
- 债权金额
- 债权到期日
- 债权依据

**查询方式**：
- 合同文件
- 往来账目
- 业务关系

### 6. 知识产权线索

**知识产权类型**：
- 专利权
- 商标权
- 著作权
- 软件著作权

**查询方式**：
- 国家知识产权局
- 商标局网站
- 版权登记中心

### 7. 其他财产线索

- 应收账款
- 设备、存货
- 原材料
- 理财产品
- 保险单现金价值
- 住房公积金
- 养老金账户

## 四、财产线索收集方法

### 1. 日常观察

- 被执行人生活消费水平
- 出行工具
- 居住条件
- 经营规模

### 2. 社会关系调查

- 亲友了解
- 生意伙伴
- 同事员工
- 邻居

### 3. 公开信息查询

**企业信用信息**：
- 国家企业信用信息公示系统
- 天眼查、企查查等商业平台

**司法信息**：
- 中国裁判文书网
- 中国执行信息公开网
- 人民法院公告网

**知识产权**：
- 中国专利公布公告
- 中国商标网

**不动产**：
- 各地不动产登记中心网站

### 4. 委托调查

**律师调查**：
- 申请调查令
- 查询银行账户
- 查询工商档案
- 查询房产车辆

**专业机构**：
- 财产调查公司
- 征信机构

## 五、财产线索报告

### 报告内容

1. **财产类型**：明确是什么财产
2. **财产位置**：具体存放地或登记地
3. **财产权属**：证明属于被执行人
4. **财产价值**：大致价值范围
5. **证据材料**：支持线索的证据

### 报告方式

- 书面提交执行法院
- 通过执行法官提交
- 随财产线索清单提交

## 六、常见财产隐匿方式及应对

### 1. 转移存款

**方式**：
- 转至他人账户
- 取现藏匿
- 购买理财产品

**应对**：
- 查询交易明细
- 追查资金流向
- 申请撤销转移行为

### 2. 转移房产

**方式**：
- 虚假买卖
- 赠与亲友
- 低价转让

**应对**：
- 查询交易记录
- 申请撤销转让
- 确认无效

### 3. 转移车辆

**方式**：
- 过户他人
- 藏匿车辆
- 虚假抵押

**应对**：
- 查询过户记录
- 追查车辆下落
- 申请扣押

### 4. 隐匿股权

**方式**：
- 代持股权
- 虚假转让
- 注销公司

**应对**：
- 查询工商档案
- 追查实际股东
- 追加被执行人

### 5. 虚构债务

**方式**：
- 与关联公司虚构债务
- 制造虚假诉讼
- 虚假仲裁

**应对**：
- 申请撤销虚假诉讼
- 追加被执行人
- 追究刑事责任

## 七、财产保全与执行衔接

### 1. 诉前/诉中保全

**优势**：
- 提前控制财产
- 防止财产转移
- 提高执行成功率

**衔接**：
- 保全措施自动转为执行措施
- 无需重新办理查封手续

### 2. 保全到期处理

**续保**：
- 保全期限届满前申请续行
- 确保保全措施持续有效

## 八、执行不能的处理

### 1. 终结本次执行

**条件**：
- 确无财产可供执行
- 财产不能处置

**后果**：
- 暂时结案
- 发现财产可恢复执行

### 2. 持续关注

**措施**：
- 定期查询被执行人财产
- 关注被执行人动态
- 收集财产线索

### 3. 其他救济

- 执行和解
- 债权转让
- 以物抵债
- 参与分配

## 九、注意事项

### 1. 线索真实性

- 提供真实可靠的线索
- 虚假线索可能承担法律责任
- 不确定的线索应注明

### 2. 及时报告

- 发现线索及时报告
- 防止财产转移
- 把握执行时机

### 3. 配合调查

- 配合法院调查工作
- 提供必要的协助
- 补充完善线索

### 4. 合法取证

- 通过合法途径获取线索
- 不得侵犯他人合法权益
- 不得使用非法手段

## 十、相关网站和平台

### 官方查询平台

1. **中国执行信息公开网**
   - 网址：http://zxgk.court.gov.cn
   - 功能：查询被执行人、失信名单、限制消费

2. **国家企业信用信息公示系统**
   - 网址：http://www.gsxt.gov.cn
   - 功能：查询企业工商信息、股东信息

3. **中国裁判文书网**
   - 网址：https://wenshu.court.gov.cn
   - 功能：查询裁判文书、了解案件情况

4. **全国市场监管动产抵押登记业务系统**
   - 功能：查询动产抵押登记

5. **中国专利公布公告**
   - 网址：http://epub.cnipa.gov.cn
   - 功能：查询专利信息

6. **中国商标网**
   - 网址：https://sbj.cnipa.gov.cn
   - 功能：查询商标信息
