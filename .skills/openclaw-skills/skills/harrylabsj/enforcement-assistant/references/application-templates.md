# 执行申请文书模板

## 模板一：民事判决执行申请书（通用版）

```
执行申请书

申请人：[姓名]，[性别]，[出生年月]，[民族]，住[详细住址]，
身份证号码：[身份证号]，联系电话：[电话号码]

被执行人：[姓名]，[性别]，[出生年月]，[民族]，住[详细住址]，
身份证号码：[身份证号]，联系电话：[电话号码]

申请事项：

一、请求强制执行[法院全称]（[年份]）[案号]民事判决书；

二、请求被执行人向申请人支付：
    1. [款项性质，如：借款本金]人民币[金额]元；
    2. [款项性质，如：利息/违约金]人民币[金额]元（暂计算至[日期]，
       实际计算至清偿之日止，按[计算标准]计算）；
    3. 迟延履行期间的债务利息（按日万分之[1.75]计算）；

三、请求被执行人承担本案执行费用。

事实与理由：

申请人与被执行人[案由，如：民间借贷]纠纷一案，业经[法院全称]审理，
于[判决日期]作出（[年份]）[案号]民事判决书，判决如下：

"[完整引用判决主文内容]"

该判决书已于[生效日期]发生法律效力。根据判决，被执行人应于[履行期限]
前向申请人履行上述义务。履行期限届满后，被执行人未主动履行生效法律
文书确定的义务。申请人于[催告日期]向被执行人发出履行催告，被执行人
至今仍未履行。

为维护申请人合法权益，依据《中华人民共和国民事诉讼法》第二百四十七条
之规定，特向贵院申请强制执行，恳请依法准许。

此致

[执行法院全称]

                                    申请人：[签名]
                                    [日期]

附件：
1. 申请人身份证复印件
2. 生效法律文书复印件
3. 生效证明
4. 被执行人财产线索清单（如有）
```

## 模板二：法人/组织作为申请人

```
执行申请书

申请人：[公司全称]
住所地：[公司注册地址]
统一社会信用代码：[代码]
法定代表人：[姓名]，职务：[职务]
联系电话：[电话号码]

被执行人：[姓名/公司全称]
住所地：[地址]
身份证号码/统一社会信用代码：[号码]
联系电话：[电话号码]

申请事项：

一、请求强制执行[法院全称]（[年份]）[案号]民事调解书/判决书；

二、请求被执行人向申请人支付款项共计人民币[金额]元，具体包括：
    1. [款项1]：[金额]元；
    2. [款项2]：[金额]元；
    3. [款项3]：[金额]元；

三、请求被执行人加倍支付迟延履行期间的债务利息；

四、请求被执行人承担本案执行费用。

事实与理由：

申请人与被执行人[案由]纠纷一案，经[法院全称]主持调解/审理，于[日期]
作出（[年份]）[案号]民事调解书/判决书，确认/判决被执行人应向申请人
支付[义务内容]。

该法律文书已于[日期]生效，被执行人应于[日期]前履行义务。但被执行人
至今未履行，严重损害申请人合法权益。

为此，申请人依据《中华人民共和国民事诉讼法》相关规定，特申请强制执行，
请予准许。

此致

[执行法院全称]

                                    申请人：[公司盖章]
                                    法定代表人：[签名]
                                    [日期]

附件：
1. 营业执照复印件
2. 法定代表人身份证明
3. 生效法律文书复印件
4. 生效证明
5. 授权委托书（如有代理人）
```

## 模板三：仲裁裁决执行申请

```
执行申请书

申请人：[姓名/名称]
[身份信息和联系方式]

被执行人：[姓名/名称]
[身份信息和联系方式]

申请事项：

一、请求强制执行[仲裁委员会全称][案号]仲裁裁决书；

二、请求被执行人履行以下义务：
    1. [具体义务1]；
    2. [具体义务2]；
    3. [具体义务3]；

三、请求被执行人承担本案执行费用。

事实与理由：

申请人与被执行人[案由]纠纷一案，经[仲裁委员会全称]仲裁，于[日期]
作出[案号]仲裁裁决书，裁决：

"[裁决主文内容]"

该仲裁裁决已于[日期]作出并生效。根据裁决，被执行人应于[期限]内履行
义务。但被执行人至今未履行，申请人曾于[日期]催告，被执行人仍拒不履行。

根据《中华人民共和国民事诉讼法》第二百四十七条及《仲裁法》相关规定，
特向贵院申请强制执行，请依法准许。

此致

[中级人民法院全称]

                                    申请人：[签名/盖章]
                                    [日期]

附件：
1. 申请人身份证明
2. 仲裁裁决书复印件
3. 仲裁协议/合同中的仲裁条款复印件
4. 送达证明（如适用）
```

## 模板五：执行和解协议

```
执行和解协议

甲方（申请执行人）：[姓名/名称]
[身份信息和联系方式]

乙方（被执行人）：[姓名/名称]
[身份信息和联系方式]

鉴于甲乙双方[案由]执行一案（执行案号：[案号]），经友好协商，
就执行事宜达成如下和解协议：

一、债务确认
双方确认，乙方应向甲方支付的债务总额为人民币[金额]元（大写：[大写金额]），
包括[本金/利息/违约金等明细]。

二、履行方案
乙方按以下方式履行义务：
1. 于[日期]前支付人民币[金额]元；
2. 于[日期]前支付人民币[金额]元；
3. [其他履行安排]

三、担保条款（如有）
[担保人]自愿为乙方履行本协议提供连带责任保证/抵押担保/质押担保，
担保范围为[范围]，担保期限为[期限]。

四、违约责任
如乙方未按本协议约定履行任何一期付款义务，则：
1. 甲方有权申请恢复执行原生效法律文书；
2. 乙方应另行支付违约金人民币[金额]元；
3. [其他违约责任]

五、其他约定
1. [费用承担约定]
2. [执行费用处理]
3. [其他特别约定]

六、协议生效
本协议自双方签字（盖章）之日起生效。本协议一式[份数]份，双方各执
[份数]份，交执行法院[份数]份。

甲方（签字/盖章）：___________    乙方（签字/盖章）：___________
日期：___________                日期：___________

担保人（签字/盖章，如有）：___________
日期：___________
```

## 填写说明

### 通用注意事项

1. **信息准确**：确保所有当事人信息、案号、日期准确无误
2. **金额规范**：金额应同时注明阿拉伯数字和中文大写
3. **引用完整**：引用判决内容时应完整、准确
4. **签名盖章**：自然人签名，法人盖章并由法定代表人签名
5. **日期填写**：使用实际提交日期

### 提交份数

- 正本1份提交法院
- 副本按被执行人人数+1份（法院存档）
- 证据材料副本与申请书副本份数相同

### 附件要求

- 按清单顺序装订
- 复印件应清晰可辨
- 重要证据提供原件核对

### 联系方式

- 提供有效联系电话
- 如有代理人，注明代理人联系方式
- 地址变更应及时通知法院
