# Comparison Guide

Use this reference to compare Meituan stores or takeout options.

## Compare these fields first
- delivery fee
- minimum order
- estimated delivery time
- platform discounts
- store discounts
- rating and sales context
- basket fit for the user's scenario

## Scenario thinking

### One-person order
Prioritize:
- low delivery fee
- low minimum order
- small but usable discounts
- fast delivery

### Two-person or group order
Prioritize:
- threshold-friendly promotions
- bundle value
- dish variety
- delivery stability

### Late-night or urgent order
Prioritize:
- delivery time reliability
- realistic minimum order
- fewer conditions

## Final rule
The best store is the one with the best real checkout fit for the user’s scenario, not the one with the flashiest discount label.
