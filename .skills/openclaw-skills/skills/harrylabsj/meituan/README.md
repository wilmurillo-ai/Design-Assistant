# Meituan Skill

美团上最难的不是“选哪家”，而是“这一单现在该不该点”。

这个 skill 关注的是行动决策，不是菜单复述。它会把满减、配送费、包装费、配送时效、商家风险放到一起，直接给出建议。

## 适合什么问题

- “这两家美团哪家更值？”
- “为了满减再凑一个菜划算吗？”
- “这家更便宜，但要晚 25 分钟，值不值？”
- “这家店评论一般，要不要冒险试试？”
- “这一单现在点，还是换一家更稳？”

## 它会怎么帮用户

它默认会收敛到一个动作：
- 现在点这家
- 换另一家
- 不要为了门槛硬凑单
- 多花一点买更快更稳的配送
- 直接跳过这家店

## 它特别适合美团的地方

- 同类商家很多，适合快速横向比较
- 午饭、晚饭、通勤间隙这种“时间值钱”的场景很多
- 看起来便宜的店，最后常常被配送费和包装费吃掉

## 典型输出

- `Recommended Move`
- `Checkout Reality`
- `Risk Check`
- `Next Step`

## 安装

```bash
clawhub install meituan
```

## 适合放在列表页的短介绍

帮你判断这一单美团该不该点，是该为了便宜凑门槛，还是多花一点换更快更稳的配送。
