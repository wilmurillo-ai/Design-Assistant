# Skills Orchestrator

Orchestrate growth loops across the skill portfolio by connecting usage, feedback, improvement, and adoption.

## Quick Start

```bash
# Design a growth loop
./scripts/design-loop.sh --type viral --skill my-skill

# Analyze current loops
./scripts/analyze-loops.sh --portfolio

# Track growth metrics
./scripts/track-metrics.sh --dashboard
```

## Features

- **Loop Design**: Create viral, content, network, and engagement loops
- **Loop Analysis**: Measure and diagnose loop performance
- **Optimization**: Identify and fix bottlenecks
- **Metrics Tracking**: Monitor growth metrics over time
- **Simulation**: Test loops before implementation

## Loop Types

| Type | Mechanism |
|------|-----------|
| viral | Users bring new users |
| content | Usage creates shareable content |
| network | More users = more value |
| engagement | Usage drives more usage |
| feedback | Usage improves product |

## Key Metrics

- **Cycle Time**: Time for one loop iteration
- **Conversion Rate**: % completing the loop
- **Viral Coefficient**: New users per existing user
- **Retention**: % continuing in loop
