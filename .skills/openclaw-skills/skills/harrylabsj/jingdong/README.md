# Jingdong Shopping Skill v2.0

京东购物助手 Skill，支持浏览器自动化完成搜索、比价、看评论、加购物车等操作，支付环节留给用户控制。

## 功能特性

- 🔍 **商品搜索**：关键词搜索，支持筛选和排序
- 📖 **用户评价**：读取评论、评分、晒单图片
- 💰 **价格对比**：多店铺/多规格比价
- 🛒 **购物车操作**：登录态下添加商品
- 🎫 **优惠券**：自动检查并应用可用优惠券
- 📋 **订单预览**：生成完整订单信息（不含支付）

## 安全边界

| 操作 | Agent | 用户 |
|------|-------|------|
| 搜索/浏览 | ✅ | - |
| 比价/看评论 | ✅ | - |
| 加购/用券 | ✅ | - |
| **支付/提交订单** | ❌ | ✅ |

## 工作流程

```
发现 → 对比 → 选择 → 加购 → 用券 → 预览 → [用户支付]
```
