# elm Skill

饿了么上最容易让人误判的，不是红包有没有，而是红包到底是不是“真省钱”。

这个 skill 会把红包、满减门槛、配送费、包装费、商家可信度和场景适配一起看，最后告诉用户这一单该不该下。

## 适合什么问题

- “这张红包值得用吗？”
- “为了凑门槛多买一点划算吗？”
- “这家饿了么便宜，但看起来不太稳，要不要下单？”
- “夜宵、饮料、小超市补货这种单子该怎么选？”
- “是继续在饿了么点，还是换美团更合适？”

## 它会怎么帮用户

它会尽量直接给动作建议：
- 现在点
- 用这张红包
- 不要追这个门槛
- 换一家店
- 换平台
- 跳过这单

## 它特别适合饿了么的地方

- 红包和满减结构容易制造“看起来省了很多”的错觉
- 夜宵、饮料、零食、便利补货这类小单很多
- 小单场景里，简单和稳定往往比菜单花样更重要

## 典型输出

- `Recommended Move`
- `Savings Reality`
- `Risk Check`
- `Next Step`

## 安装

```bash
clawhub install elm
```
