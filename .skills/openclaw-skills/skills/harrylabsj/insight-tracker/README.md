# Insight Tracker

Track and manage insights, patterns, and observations discovered during OpenClaw sessions.

## Overview

This skill provides structured insight management for OpenClaw workflows. It captures observations, patterns, and learnings that emerge during sessions, making them searchable, referenceable, and actionable.

## Installation

Place this skill directory in your skills folder.

## Usage

See SKILL.md for detailed usage instructions.

## Scripts

- `scripts/add-insight.sh` - Add a new insight
- `scripts/list-insights.sh` - List insights with filtering
- `scripts/test.sh` - Run tests for this skill
