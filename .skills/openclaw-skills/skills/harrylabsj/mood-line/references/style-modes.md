# Style Modes

Use this reference when the user asks for a specific tone, use case, or emotional texture.

Default priority: **accuracy first, beauty second**.

## Core styles

### 1. 克制
Use for users who say: 不要鸡汤 / 简洁一点 / 冷静一点 / 克制一点.

Characteristics:
- short and clean
- little ornament
- emotionally restrained but still warm
- suitable for self-description and mature social posts

Pattern:
- fewer metaphors
- more direct truth
- avoid over-explaining

Example:
> 原创风格句（非名人原句）：我不是突然想开了，只是不想再把力气耗在旧事上。

### 2. 温柔
Use for users seeking comfort, healing, softness, or companionship.

Characteristics:
- gentle and receiving
- acknowledges pain without pressing positivity
- soothing, not preachy

Example:
> 原创风格句（非名人原句）：你不是太脆弱了，只是一个人撑了太久。

### 3. 文学
Use when the user asks for: 诗意一点 / 文艺一点 / 有画面感 / 高级一点.

Characteristics:
- richer imagery
- controlled rhythm
- elegant, but still understandable
- do not become purple prose

Example:
> 原创风格句（非名人原句）：心事像黄昏的潮水，退得很慢，却终究会把岸边还给夜色。

### 4. 清醒
Use for self-reflection, boundaries, reality checks, growth, and emotional clarity.

Characteristics:
- grounded
- sharp but not harsh
- suitable for整理思路、写日记、做决定

Example:
> 原创风格句（非名人原句）：你难过，不只是因为失去，而是因为你曾认真相信过。

### 5. 有力量
Use when the user wants energy,鼓劲,重整状态,但不要喊口号.

Characteristics:
- firm
- forward-looking
- avoids empty motivation slogans

Example:
> 原创风格句（非名人原句）：真正的恢复，不是立刻振作，而是把自己一点点接回来。

## Use-case variants

### 朋友圈版
Characteristics:
- shorter
- more memorable
- mild literary flavor
- low explanation

Target length:
- usually 12–28 Chinese characters

Example:
> 原创风格句（非名人原句）：我没有突然放下，只是终于把力气还给自己。

### 日记版
Characteristics:
- more intimate
- allows one more clause
- suitable for private writing

Example:
> 原创风格句（非名人原句）：今天终于承认，我不是不难过，只是在很努力地和难过相处。

### 发给朋友版
Characteristics:
- gentle
- second-person allowed
- supportive, not grandiose

Example:
> 原创风格句（非名人原句）：你已经做得很不容易了，不必连崩溃都要求自己体面。

## Selection guide

If the user specifies a style, follow it.
If not, use this default matching:

- 疲惫 / 委屈 / 想被理解 → 温柔 / 克制
- 迷茫 / 自责 / 复盘 → 清醒
- 孤独 / 想念 / 失落 → 文学 / 温柔
- 释然 / 放下 / 向前走 → 克制 / 有力量
- 想发圈 / 要一句短的 → 朋友圈版
- 想写日记 / 想整理自己 → 日记版

## Quality checks by style

Before finalizing, verify:
- Does the style fit the user’s emotional temperature?
- Is the sentence still natural Chinese, not translated-sounding?
- Is the line emotionally specific, not generic?
- Is it usable in the stated context?

## Anti-patterns

Avoid:
- empty motivational slogans
- exaggerated pain aesthetics
- too many metaphors in one sentence
- “高级感” that sacrifices clarity
- forcing elegance on blunt emotions like anger or exhaustion

## Compact fallback

If the user asks for “just one line”, skip long analysis and output:
1. one short emotional reading
2. one best-fit line

Even then, keep attribution safety rules from `quote-safety.md`.
