# Quote Whitelist

Use this file only when considering a real attributed quote.

Rule:
- Only use a quote from this whitelist when it clearly fits the user's emotion and context.
- When used, include the author and source.
- If none fits well, do not force a quote. Fall back to a natural unattributed expression.
- Prefer steady, well-known, low-dispute lines over clever or trendy ones.

## 1. 希望 / 微光

### Quote
> 黑夜无论怎样悠长，白昼总会到来。  
> —— 莎士比亚《麦克白》

Use when:
- the user is under strain but still wants a steady, classic line of hope

Avoid when:
- the user needs highly personal, intimate wording

## Selection notes

Before using any quote, verify all of the following:
1. the quote clearly matches the user's emotional core
2. the wording is reliable enough to present directly
3. the attribution is reliable enough to present directly
4. the source is included when available and stable

If any item is shaky, do not use the quote.

## Default behavior

Whitelist hit = quote may be used with source.  
No whitelist hit = output a natural `表达句：……` and do not force citation.
