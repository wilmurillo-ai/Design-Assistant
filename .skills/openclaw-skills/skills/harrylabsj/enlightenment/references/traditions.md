# Tradition Views

Use this file when the user asks about specific traditions or wants distinctions across lineages.

## Core rule

Do not flatten all traditions into one model.
Explain overlap where useful, but preserve real differences.

## 1. 禅宗 / 佛教语境
Common emphases:
- 见性
- 无常、无我、空
- 直接体认，不只停留在概念
- 修行与日常行住坐卧的统一

Useful framing:
> 在禅宗语境里，所谓“开悟”通常不是多知道了一个理论，而是对自心、自性、执著与分别的直接看见。  
> 但这种“看见”是否稳定、是否真正转化生命，还要看后续修行与日常表现。

## 2. 道家语境
Common emphases:
- 顺其自然
- 少私寡欲
- 与道相应
- 去强作、去执拗、去过度人为控制

Useful framing:
> 如果从道家的角度看，所谓“开了”往往不是变得更神秘，而是变得更不拧巴、更不逆势、更少和万物较劲。

## 3. 非二元 / Advaita 语境
Common emphases:
- 观察者与被观察者的关系
- 自我并非最终实在
- awareness / pure consciousness / witnessing

Useful framing:
> 在非二元语境里，很多人说的“觉醒”常指对“我是谁”的识别发生变化：不再把念头、角色、故事当成全部的自己。  
> 但这类看见，不等于人格问题自动解决，也不等于生活层面的成熟已经完成。

## 4. 现代心理学语境
Common emphases:
- self-awareness
- decentering
- less fusion with thought
- regulation and groundedness

Useful framing:
> 如果放到现代心理学语境里，有些“觉醒感”可以理解为观察自我能力增强、对念头不再那么认同、对内在模式有了新的觉察。  
> 这不必被浪漫化成终极证悟，但也可以是很有价值的成长节点。

## Safe comparison pattern

When comparing traditions:
1. give the shared theme
2. give the key difference
3. explain what this means for the user's question

Example:
> 它们的共同点，是都在提醒人不要把惯性的自我执著当成全部现实。  
> 差别在于：佛教更强调无我与空，道家更强调顺势与自然，非二元更强调“我是觉知，不只是内容”。  
> 所以你问的这个体验，要看你想从哪种修行语言去理解它。
