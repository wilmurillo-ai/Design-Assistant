# Refund Assistant - 退换货助手

智能退换货策略生成和流程指导。

## 功能

- 📋 平台规则查询（淘宝/京东/拼多多）
- 🎯 最优退货理由推荐
- 📸 证据准备清单
- 💰 运费/补偿计算
- 📝 退货申请文案生成

## 使用

```
退货策略 订单号
这个商品有质量问题怎么退
淘宝超过 7 天还能退吗
```
