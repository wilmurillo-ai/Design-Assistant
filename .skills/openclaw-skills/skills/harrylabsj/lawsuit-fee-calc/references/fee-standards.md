# 诉讼费用标准参考

## 一、财产案件受理费计算标准

根据《诉讼费用交纳办法》第十三条，财产案件根据诉讼请求的金额或者价额，按照下列比例分段累计交纳：

| 标的金额 | 费率 | 速算公式 |
|---------|------|---------|
| 不超过1万元 | 每件50元 | 50元 |
| 超过1万元至10万元的部分 | 2.5% | 标的额 × 2.5% - 200元 |
| 超过10万元至20万元的部分 | 2% | 标的额 × 2% + 300元 |
| 超过20万元至50万元的部分 | 1.5% | 标的额 × 1.5% + 1,300元 |
| 超过50万元至100万元的部分 | 1% | 标的额 × 1% + 3,800元 |
| 超过100万元至200万元的部分 | 0.9% | 标的额 × 0.9% + 4,800元 |
| 超过200万元至500万元的部分 | 0.8% | 标的额 × 0.8% + 6,800元 |
| 超过500万元至1000万元的部分 | 0.7% | 标的额 × 0.7% + 11,800元 |
| 超过1000万元至2000万元的部分 | 0.6% | 标的额 × 0.6% + 21,800元 |
| 超过2000万元的部分 | 0.5% | 标的额 × 0.5% + 41,800元 |

### 计算示例

**示例1：标的额5万元**
- 1万元部分：50元
- 1万至5万部分：40,000 × 2.5% = 1,000元
- 合计：1,050元
- 速算：50,000 × 2.5% - 200 = 1,050元 ✓

**示例2：标的额100万元**
- 速算：1,000,000 × 1% + 3,800 = 13,800元

**示例3：标的额500万元**
- 速算：5,000,000 × 0.7% + 11,800 = 46,800元

## 二、非财产案件受理费标准

### 1. 离婚案件
- 每件交纳50元至300元
- 涉及财产分割，财产总额不超过20万元的，不另行交纳
- 超过20万元的部分，按照0.5%交纳

**示例**：离婚案件，财产分割标的100万元
- 基础：300元
- 超额部分：(1,000,000 - 200,000) × 0.5% = 4,000元
- 合计：4,300元

### 2. 侵害姓名权、名称权、肖像权、名誉权、荣誉权等人格权的案件
- 每件交纳100元至500元
- 涉及损害赔偿，赔偿金额不超过5万元的，不另行交纳
- 超过5万元至10万元的部分，按照1%交纳
- 超过10万元的部分，按照0.5%交纳

### 3. 其他非财产案件
- 每件交纳50元至100元

### 4. 知识产权民事案件
- 没有争议金额或者价额的，每件交纳500元至1000元
- 有争议金额或者价额的，按照财产案件的标准交纳

### 5. 劳动争议案件
- 每件交纳10元

### 6. 行政案件
- 商标、专利、海事行政案件：每件交纳100元
- 其他行政案件：每件交纳50元

## 三、申请费标准

### 1. 申请执行（法院生效裁判）
- **无需交纳申请费**
- 执行后由被执行人承担

### 2. 财产保全申请费

| 保全金额 | 费用 |
|---------|------|
| 不超过1000元或不涉及金额 | 30元 |
| 超过1000元至10万元的部分 | 1% |
| 超过10万元的部分 | 0.5% |
| **最高限额** | **5000元** |

**示例**：申请保全标的80万元
- 1000元部分：30元
- 1000元至10万部分：99,000 × 1% = 990元
- 超过10万部分：700,000 × 0.5% = 3,500元
- 合计：4,520元

### 3. 支付令申请费
- 按照财产案件受理费标准的1/3交纳
- 不交纳申请费
- 如债务人提出异议，转入诉讼程序，补交剩余费用

**示例**：支付令标的10万元
- 正常受理费：10万 × 2.5% - 200 = 2,300元
- 支付令申请费：2,300 ÷ 3 ≈ 767元（实践中按整数收取）

### 4. 破产案件
按照破产企业财产总值依照财产案件收费标准计算，减半交纳，最高不超过10万元。

## 四、其他诉讼费用

### 1. 证人、鉴定人、翻译人员、理算人员在人民法院指定日期出庭发生的费用
- 交通费、住宿费、生活费和误工补贴
- 按照国家规定标准计算

### 2. 公告费用
- 公告送达：每次约200-800元（各地标准不同）
- 公告开庭、判决等

### 3. 鉴定费用
- 根据鉴定类型和机构收费标准
- 常见：笔迹鉴定、伤残鉴定、工程造价鉴定等

## 五、诉讼费减免政策

### 1. 免交情形
- 残疾人无固定生活来源
- 追索赡养费、扶养费、抚育费、抚恤金
- 最低生活保障对象
- 追索劳动报酬
- 见义勇为行为产生的民事权益

### 2. 减交情形
- 自然灾害等不可抗力
- 属于国家规定的优抚、安置对象
- 社会福利机构和救助管理站
- 确实需要减交的其他情形
- 减交比例不得低于30%

### 3. 缓交情形
- 追索社会保险金、经济补偿金
- 海上事故、交通事故、医疗事故等受害人请求赔偿
- 正在接受法律援助
- 确实需要缓交的其他情形

### 4. 申请流程
1. 提交书面申请
2. 提供证明材料（低保证明、残疾证等）
3. 法院审查决定
4. 准予减免的，在裁判文书中载明

## 六、诉讼费退还

### 1. 撤诉或按撤诉处理
- 减半退还案件受理费

### 2. 调解结案
- 减半收取案件受理费
