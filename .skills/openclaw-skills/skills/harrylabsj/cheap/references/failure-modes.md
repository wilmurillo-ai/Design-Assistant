# Failure Modes Reference

Use this reference when the cheapest-price search cannot produce a strong answer.

These failure modes usually push the skill into **Mode 2: Price-limited mode** unless there is still enough evidence for a narrow Mode 1 comparison.

## Product too broad
If the product name is too broad:
- ask one short clarifying question
- request the most important missing detail first

## Platform access incomplete
If some platforms are inaccessible or weakly readable:
- say which platforms were actually checked
- do not claim definitive full-market coverage

## Listing comparability weak
If listings may not be truly the same product:
- lower match confidence
- explain what may differ

## Price condition unclear
If the lowest visible price depends on unclear coupon, deposit, or member conditions:
- flag the condition
- avoid presenting it as a normal cheapest price

## Promotion distortion
If the visible low price depends on flash sale, subsidy, livestream framing, or another condition that is not cleanly verifiable:
- state that the price may not be a stable meaningful comparison point
- avoid presenting it as a straightforward all-user cheapest result

## Final rule
A cautious but comparable cheapest-price answer is better than a broad but misleading one.
