# JD Shopping Skill v2.0

京东决策型购物助手 Skill。不是单纯“帮你逛京东”，而是帮你判断这单值不值得在京东下、贵一点是否换来更稳的自营/时效/售后，再用浏览器把搜索、比价、看评论、加购物车做到支付前。

## 功能特性

### v2.0 - 新增浏览器自动化

- 🔍 **商品搜索**：关键词搜索，支持筛选和排序
- 📖 **用户评价**：读取评论、评分、晒单图片
- 💰 **价格对比**：多店铺/多规格比价
- 🛒 **购物车操作**：登录态下添加商品
- 🎫 **优惠券**：自动检查并应用可用优惠券
- 📋 **订单预览**：生成完整订单信息（不含支付）

### v1.0 - 决策支持

- ✅ 平台适用性分析
- ✅ 店铺类型指导（自营 vs 第三方）
- ✅ 价格和时机建议
- ✅ 信任和物流建议

## 安全边界

| 操作 | Agent | 用户 |
|------|-------|------|
| 搜索/浏览 | ✅ | - |
| 比价/看评论 | ✅ | - |
| 加购/用券 | ✅ | - |
| **支付/提交订单** | ❌ | ✅ |

## 工作流程

```
发现 → 对比 → 选择 → 加购 → 用券 → 预览 → [用户支付]
```

## 版本历史

- v2.0.0 - 新增浏览器自动化支持
- v1.0.0 - 基础决策支持

## 安装

```bash
clawhub install jd-shopping
```

## 使用示例

```
用户：帮我买 Mac mini
Agent：搜索 → 对比 → 加购 → 生成订单预览
用户：👤 完成支付
```
