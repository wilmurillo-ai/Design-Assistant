# Logistics Watcher - 物流异常预警

智能物流监控，自动识别异常并预警。

## 功能特性

- 📦 多快递支持：顺丰、京东、中通、圆通等
- ⚠️ 智能异常识别：停滞、退回、异常签收
- 🔔 主动预警：超时未更新、异常状态
- 📊 物流分析：预计到达时间、异常概率
- 🔄 自动追踪：定时检查状态

## 使用方式

### 添加快递
```
查快递 SF1234567890
追踪物流 JD000123456789
```

### 查看状态
```
我的快递
物流状态 SF1234567890
```

### 设置提醒
```
物流提醒 超时24小时
物流提醒 异常状态
```
