# Knowledge Connector Release Notes

## Short Description

把分散文档变成可导入、可回答、可视化、可行动的知识图谱结果。

## Marketplace Card Copy

Title:
- Knowledge Connector

Short description:
- 导入向导、跨文档答案页、关系可视化，并输出可操作的知识图谱结果

Install hook:
- 不是再造一个图谱工具，而是把知识从“存着”推进到“能查、能看、能行动”

## Announcement Copy

Knowledge Connector 这次不再只是“提取概念 + 建图”。

现在它更像一个完整产品入口：
- 可以先预览再导入
- 可以批量导入文档
- 可以做跨文档搜索
- 可以把搜索整理成更像答案页的结果
- 可以围绕单个概念生成可操作子图
- 可视化结果里会直接给下一步建议

这条线更应该继续做产品深度，而不是拆出更多相似 skill。
