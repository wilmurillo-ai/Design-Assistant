# VIP Skill v2.0

唯品会品牌折扣购物助手 Skill。核心不是“逛特卖”，而是判断这个品牌折扣现在值不值买、库存是否合适、超级VIP是否真的省钱，再把搜索、库存检查、加购推进到支付前。

## 功能特性

### v2.0 - 新增浏览器自动化

- 🔍 **商品搜索**：关键词搜索，支持筛选
- ⚡ **限时特卖**：浏览每日 rotating flash sales
- 💰 **价格对比**：原价 vs 特卖价 vs 超级VIP价
- 📦 **库存检查**：实时检查尺码/颜色 availability
- ✅ **品牌验证**：确认官方授权
- 🛒 **购物车操作**：登录态下添加商品
- 🎫 **优惠券**：自动检查优惠券、超级VIP折扣
- 📋 **订单预览**：生成完整订单信息（不含支付）

### v1.0 - 决策支持

- ✅ 平台适用性分析
- ✅ 限时特卖时机指导
- ✅ 品牌折扣策略
- ✅ 超级VIP权益说明
- ✅ 平台对比建议

## 安全边界

| 操作 | Agent | 用户 |
|------|-------|------|
| 搜索/浏览 | ✅ | - |
| 查看特卖 | ✅ | - |
| 库存检查 | ✅ | - |
| 加购/用券 | ✅ | - |
| **支付/提交订单** | ❌ | ✅ |

## 工作流程

```
发现 → 查看特卖 → 库存检查 → 选择 → 加购 → 用券 → 预览 → [用户支付]
```

## 安装

```bash
clawhub install vip
```

```
用户：帮我买唯品会的衣服
Agent：搜索 → 查看特卖 → 检查库存 → 加购 → 生成订单预览
用户：👤 完成支付
```
