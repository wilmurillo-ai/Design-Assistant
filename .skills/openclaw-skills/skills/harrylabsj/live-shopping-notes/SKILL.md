# Live Shopping Notes - 直播选品速记

直播购物快速记录和智能决策工具。

## 功能

- 🎥 直播会话管理
- ⚡ 快速商品记录（支持自然语言）
- 🏷️ 自动标签提取
- 📊 直播结束生成决策清单
- 💡 智能购买建议

## 使用

```
开始直播记录 李佳琦
记录 面霜 299 买一送一 划算
结束直播
查看直播清单
```
