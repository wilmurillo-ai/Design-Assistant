## 核心功能

### 1. 八字排盘 (BaZi Analysis)

**八字**，又称四柱，是根据出生年月日时推算的命理基础。

**排盘要素：**
- **年柱**：出生年份的天干地支
- **月柱**：出生月份的天干地支
- **日柱**：出生日期的天干地支（日主）
- **时柱**：出生时辰的天干地支

**天干**：甲、乙、丙、丁、戊、己、庚、辛、壬、癸
**地支**：子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥

**使用方法：**
1. 用户提供出生年月日时（注明阳历/阴历）
2. 系统自动转换为八字四柱
3. 分析日主强弱、五行分布
4. 给出基本性格特征和命理特点

### 2. 五行分析 (Five Elements)

**五行**：金、木、水、火、土

**五行相生**：木→火→土→金→水→木
**五行相克**：木克土→土克水→水克火→火克金→金克木

**分析维度：**
| 维度 | 说明 |
|------|------|
| 五行分布 | 八字中五行的数量分布 |
| 五行强弱 | 各元素的旺衰程度 |
| 喜用神 | 对命主有利的五行 |
| 忌神 | 对命主不利的五行 |
| 五行调和建议 | 颜色、方位、数字等建议 |

### 3. 运势预测 (Fortune Forecast)

**大运**：每十年一个大运周期
**流年**：每年的运势变化

**预测范围：**
- **事业运**：工作发展、升职机会
- **财运**：收入、投资、理财
- **感情运**：桃花、婚姻、人际关系
- **健康运**：身体状况、注意事项
- **学业运**：考试、进修、技能提升

### 4. 姓名学分析 (Name Analysis)

**五格剖象法**：
- **天格**：姓氏笔画数 + 1（单姓）
- **人格**：姓氏最后一字 + 名字第一字
- **地格**：名字笔画数总和
- **外格**：总格 - 人格 + 1
- **总格**：姓名全部笔画数

**分析内容：**
- 五格数理吉凶
- 三才配置（天、人、地）
- 五行属性分析
- 音律、字形、字义评价

### 5. 择日查询 (Auspicious Date Selection)

**适合查询的事项：**
- 搬家入宅
- 结婚嫁娶
- 开业开工
- 出行远行
- 签约交易
- 动土修建

**择日要素：**
- 黄道吉日
- 避开生肖冲日
- 结合八字喜用神
- 十二建星

### 6. 生肖配对 (Zodiac Compatibility)

**十二生肖**：鼠、牛、虎、兔、龙、蛇、马、羊、猴、鸡、狗、猪

**配对分析：**
- **六合**：最佳配对（鼠牛、虎猪、兔狗、龙鸡、蛇猴、马羊）
- **三合**：良好配对（猴鼠龙、虎马狗、蛇鸡牛、猪兔羊）
- **六冲**：避免配对（鼠马、牛羊、虎猴、兔鸡、龙狗、蛇猪）
- **六害**：不太理想（鼠羊、牛马、虎蛇、兔龙、猴猪、鸡狗）

## 使用示例

### 示例1：八字排盘
```
用户：帮我排一下八字，1990年3月15日下午3点出生

分析：
- 出生时间：1990年3月15日 15:00（阳历）
- 八字：庚午年 己卯月 辛巳日 丙申时
- 日主：辛金
- 五行分布：金2 木1 水0 火3 土2
- 日主偏弱，喜土金，忌木火
```

### 示例2：运势分析
```
用户：今年事业运怎么样？我是1992年出生的

分析：
- 生肖：猴
- 2026年（丙午年）运势：
  - 事业：有贵人相助，适合拓展新领域
  - 财运：正财稳定，偏财需谨慎
  - 注意：避免冲动决策，多听取建议
```

```
用户：分析一下"张伟"这个名字

分析：
- 天格：12（木）

- 总格：23（火）
- 三才配置：木金木（凶）
- 建议：可考虑中间加字调和五行
```

## 注意事项

1. **娱乐参考**：命理分析仅供娱乐参考，不应作为人生重大决策的唯一依据
2. **科学态度**：保持理性态度，命理是传统文化而非科学预测
3. **个人信息**：涉及出生时间等个人信息时注意隐私保护
4. **文化尊重**：命理是中国传统文化的一部分，应尊重其文化价值
