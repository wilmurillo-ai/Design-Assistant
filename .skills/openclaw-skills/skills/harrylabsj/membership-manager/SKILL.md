# Membership Manager - 会员权益管家

统一管理多平台会员权益。

## 功能

- 📱 多平台会员录入（淘宝/京东/拼多多/饿了么）
- ⏰ 到期提醒
- 🎁 未使用权益追踪
- 💰 会员价值计算
- 📊 权益使用分析

## 使用

```
添加会员 京东 PLUS 2025-12-31
我的会员列表
哪些权益快过期了
这个会员值不值得续费
```
