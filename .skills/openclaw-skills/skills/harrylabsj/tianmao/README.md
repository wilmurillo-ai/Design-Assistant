# Tianmao Skill v2.0

天猫品牌官方渠道购物助手 Skill。核心不是“搜商品”，而是判断官方旗舰店和 88VIP 权益是否值得、正品与售后是否值回价差，再把搜索、验店、加购推进到支付前。

## 功能特性

### v2.0 - 新增浏览器自动化

- 🔍 **商品搜索**：关键词搜索，支持筛选
- 🏪 **官方旗舰店**：筛选官方旗舰店、品牌直营店
- ✅ **品牌验证**：检查授权徽章、正品保障
- 📖 **用户评价**：读取评论、评分、晒单
- 💰 **价格对比**：原价 vs 活动价 vs 88VIP价
- 🛒 **购物车操作**：登录态下添加商品
- 🎫 **优惠券**：自动检查店铺券、平台券、88VIP折扣
- 📋 **订单预览**：生成完整订单信息（不含支付）

### v1.0 - 决策支持

- ✅ 平台适用性分析
- ✅ 官方旗舰店指导
- ✅ 品牌正品建议
- ✅ 88VIP权益说明
- ✅ 平台对比建议

## 安全边界

| 操作 | Agent | 用户 |
|------|-------|------|
| 搜索/浏览 | ✅ | - |
| 筛选官方店 | ✅ | - |
| 品牌验证 | ✅ | - |
| 加购/用券 | ✅ | - |
| **支付/提交订单** | ❌ | ✅ |

## 工作流程

```
发现 → 筛选官方店 → 品牌验证 → 选择 → 加购 → 用券 → 预览 → [用户支付]
```
