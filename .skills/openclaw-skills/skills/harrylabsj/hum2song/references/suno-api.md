## Suno API (External Service)

Suno is a third-party AI music generation platform. This skill does not integrate with Suno.

### If users want to use Suno separately:

- Website: https://www.suno.ai/
- Users must create their own account
- Users handle their own API keys and billing
- No integration with this skill
