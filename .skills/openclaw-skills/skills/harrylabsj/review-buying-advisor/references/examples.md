# Examples Reference

Use these examples to calibrate tone, structure, and verdict style.

Reuse the logic, not the exact wording.

The first group below reflects **Mode 1: Review-accessible mode**. Add a Mode 2 example when the platform does not expose enough public review content.

# Example 1: JD electronics accessory

## Input
"帮我看看 Best Buy 上的 Anker Soundcore Life Q30 值不值得买"

## Evidence pattern
- many positive comments on sound quality and comfort
- repeated praise for value
- repeated complaints about unstable Bluetooth connection
- several recent reviews mention weaker battery after short use

## Good answer

### Overall Verdict
Recommend with caveats

### Why
- Reviews support decent sound quality and comfort for the price.
- Buyers often feel the product is good value for casual use.
- The main concern is repeated connection instability, which affects the core experience.

### Best For
Budget-conscious buyers who mainly want casual listening.

### Main Positives
- Good sound for the price
- Comfortable enough for daily wear
- Good perceived value

### Main Risks
- Repeated issue: unstable Bluetooth connection
- Repeated issue: battery may decline faster than expected
- Occasional issue: average build quality

### Watch Before Buying
- Avoid if connection stability is critical
- Prefer a seller with a good return policy

### Final Advice
Reasonable as a budget buy, but not ideal if reliability matters more than price.

### Confidence
Medium
