# Product Identification Reference

Use this reference when the input is a product name plus platform.

Identify the intended product confidently enough to support review analysis. If identity is unclear, ask one short clarifying question.

## Product name + platform
If the user provides a product name and platform:
- locate the most likely listing on that platform
- match brand, model, category, and key specs
- verify visible variant context when possible
- avoid guessing if multiple plausible products exist

Generic names are more likely to need clarification.

## Confidence signals

### Strong
- exact platform match
- exact brand match
- exact model or series match
- matching size, capacity, or specification
- matching category and primary function

### Weak
- rough category match only
- similar appearance with no matching model details
- same brand but different generation or version

## Ambiguity handling

### Proceed
Proceed if there is one obvious match and no major competing interpretation.

### Proceed cautiously
Proceed with a brief caveat if the main product is clear but minor variant uncertainty remains.

### Ask
Ask one short clarifying question if:
- multiple listings fit the name
- multiple generations or variants are possible
- seller or variant likely changes the conclusion

If clarification is needed, ask for the single most decision-relevant detail first, such as brand, exact model, seller, or key variant.

## Variant contamination

Reviews may mix:
- size
- color
- material
- bundle contents
- storage or memory version
- hardware revision

If review mixing is likely:
- mention the risk
- avoid overconfident conclusions about a specific variant

## Seller vs product

Some categories vary a lot by seller, batch, support, or authenticity.

If seller quality materially affects outcomes:
- say so
- avoid giving a blanket product recommendation without that warning

## Failure handling

If you cannot confidently identify the intended product:
- do not give a firm purchase verdict
- ask for the exact model, seller, variant, or another distinguishing detail
