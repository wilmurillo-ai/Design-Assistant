## 核心功能

### 1. 起卦占卜 (Divination)

**起卦方法：**

#### 铜钱起卦法（推荐）
```
准备三枚铜钱，记录每次投掷结果：
- 三个字（老阴）：6 ⚋ 变爻
- 两个字一个背（少阴）：7 ⚊ 不变
- 两个背一个字（少阳）：8 ⚋ 不变
- 三个背（老阳）：9 ⚊ 变爻

投掷六次，从下到上记录，形成本卦和变卦
```

#### 数字起卦法
```
用户提供两个数字：
- 第一个数字 ÷ 8 的余数 = 上卦
- 第二个数字 ÷ 8 的余数 = 下卦
- 两数之和 ÷ 6 的余数 = 变爻

余数为0时取8（上卦/下卦）或6（变爻）
```

#### 时间起卦法
```
使用当前时间起卦：
- 年+月+日 = 上卦数
- 年+月+日+时 = 下卦数
- 年+月+日+时+分 = 变爻数

分别除以8或6取余数
```

### 3. 卦象解读 (Hexagram Interpretation)

**解读维度：**

| 维度 | 说明 |
|------|------|
| 卦辞 | 整体卦象的总断 |
| 彖辞 | 对卦辞的解释 |
| 象辞 | 象征意义的说明 |
| 爻辞 | 每一爻的具体解释 |
| 变爻 | 动爻的变化分析 |
| 互卦 | 内在变化趋势 |
| 错卦 | 阴阳全变，对立视角 |
| 综卦 | 上下颠倒，反向视角 |

**变爻分析：**
- 一个变爻：以变爻爻辞为主
- 两个变爻：以上爻为主，下爻为辅
- 三个变爻：以本卦卦辞为主，变卦参考
- 四个变爻：以变卦两不变爻为主，下爻优先
- 五个变爻：以变卦不变爻为主
- 六个变爻：以变卦卦辞为主（乾坤用九用六）

### 4. 易经哲学 (I Ching Philosophy)

**核心概念：**

#### 阴阳
```
阳 ⚊：刚、动、明、热、天、男
阴 ⚋：柔、静、暗、冷、地、女

阴阳关系：
- 阴阳互根：相互依存
- 阴阳消长：此消彼长
- 阴阳转化：物极必反
- 阴阳平衡：和谐之道
```

#### 三才
```
上爻：天
中爻：人
下爻：地

三才之道：天时、地利、人和
```

#### 时位
```
六爻位置：
初爻：开始、潜藏
二爻：显现、得中
三爻：发展、多凶
四爻：进退、犹豫
五爻：成功、得正
上爻：极盛、将变
```

### 5. 决策参考 (Decision Guidance)

**易经决策框架：**

1. **明确问题** - 具体、清晰的问题
2. **诚心起卦** - 专注、诚实地起卦
3. **分析卦象** - 理解当前状况
4. **查看变爻** - 了解变化趋势
5. **综合判断** - 结合实际情况
6. **采取行动** - 做出决策并执行

**注意事项：**
- 易经提供的是参考，不是绝对的预测
- 强调"自天佑之，吉无不利"（自助者天助）
- 重视道德修养和正确的心态
- 同一问题不宜反复起卦

## 使用示例

### 示例1：铜钱起卦
```
用户：帮我起一卦，问事业发展

操作：
1. 模拟铜钱投掷（或使用随机数）
2. 六次结果：7 8 9 6 7 8
3. 本卦：风火家人（上巽下离）
4. 变爻：第三爻（9→阴）、第四爻（6→阳）
5. 变卦：山火贲（上艮下离）

解读：
- 本卦风火家人：重视家庭内务，从基础做起
- 变卦山火贲：需要修饰、包装、展现
- 建议：先稳固内部，再向外发展，注重形象和品质
```

### 示例2：数字起卦
```
用户：用数字123和456起卦

计算：
- 上卦：123 ÷ 8 = 15余3 → 离卦 ☲
- 下卦：456 ÷ 8 = 57余0→8 → 坤卦 ☷
- 变爻：(123+456) ÷ 6 = 96余3 → 第三爻

结果：火地晋卦，第三爻变

解读：
- 晋卦：晋升、进步、光明
- 六三：众允，悔亡。得到众人认可，没有悔恨。
- 建议：当前适合积极进取，寻求晋升机会
```

## 占卜注意事项

1. **诚心正念**：起卦前静心，诚心发问
2. **一事一占**：同一问题不宜反复起卦
3. **理性看待**：易经是参考，不是绝对预测
4. **重在启示**：关注卦象的启示和指导意义
5. **知行合一**：占卜后要付诸行动
