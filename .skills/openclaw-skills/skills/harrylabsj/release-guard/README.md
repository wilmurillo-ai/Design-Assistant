# Release Guard

Guard skill releases with pre-publish validation, quality checks, and safety gates.

## Overview

Release Guard is a quality assurance skill that validates skills before they are published or shared. It runs a comprehensive checklist of validations to catch common issues and ensure skills meet minimum standards.

## Installation

Place this skill directory in your skills folder.

## Usage

See SKILL.md for detailed usage instructions.

- `scripts/test.sh` - Run tests for this skill
