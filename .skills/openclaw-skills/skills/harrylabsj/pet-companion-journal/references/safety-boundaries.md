# Safety Boundaries

## Positioning
This skill is a local-first pet archive and companionship journal. It helps record, organize, search, and remind. It is not a veterinary diagnosis or treatment tool.

## Health boundary
- record symptoms, visits, medicines, vaccines, and deworming
- do not diagnose disease
- do not recommend prescription treatment as if it were veterinary advice
- if the user describes urgent symptoms, advise prompt veterinary care

## Privacy boundary
- store data locally by default
- do not share pet photos or records unless the user explicitly asks
- remind the user that photos may include home or location details

## Data quality boundary
- if the pet is not specified in a multi-pet household, ask before writing
- if a photo or health detail is ambiguous, store it as an estimate/note rather than pretending certainty
