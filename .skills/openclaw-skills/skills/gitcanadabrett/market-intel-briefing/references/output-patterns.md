# Output Patterns

## Default concise commercial brief

1. Scope and framing
2. Key confirmed developments
3. Notable unverified or weakly supported claims
4. Commercial implications
5. Recommended next 3 actions
6. Client-facing talk track
7. Sources

## Variant: company-set brief

Use when comparing a small set of named companies.

1. Scope and framing
2. Company-by-company developments
3. Cross-cutting themes
4. Commercial implications
5. Recommended next 3 actions
6. Client-facing talk track
7. Sources

## Variant: what changed since last time

Use when the user asks for changes over a defined period.

1. Scope and comparison window
2. Material changes
3. Signals that look important but remain uncertain
4. Commercial implications
5. Recommended next 3 actions
6. Client-facing talk track
7. Sources

## Variant: watchlist-oriented brief

Use when the user mainly wants a decision-oriented output.

1. Scope and framing
2. What matters now
3. Watchlist items
4. Commercial implications
5. Recommended next 3 actions
6. Client-facing talk track
7. Sources

## Variant: sparse-data / thin-signal brief

Use when evidence is thin, few developments occurred, or the topic has limited public coverage. This is the minimum viable brief structure.

1. Scope statement (topic, geography, time window examined)
2. Evidence-limit declaration (what was and was not findable)
3. Monitored signal with threshold trigger ("watch for X; if X, posture changes because Y")
4. Key assumption to validate
5. One low-risk action justified now
6. Next-check trigger ("revisit when X occurs or after Y days")
7. Sources
