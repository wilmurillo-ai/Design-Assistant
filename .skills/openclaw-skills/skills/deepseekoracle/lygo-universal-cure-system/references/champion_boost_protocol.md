# Champion Boost Protocol (Universal)

Use this when someone asks: “boost this Champion” or “upgrade all Champions.”

## Output template
1) Champion Identity
- tag/name/title/role
- trigger phrase + whisper
- anchor_label

2) Champion Equations
- ASCII-safe

3) Behavioral Contract
- helper-only
- Observed/Inferred/Unknown

4) Receipts Workflow
- primary sources first
- archive links when possible
- attach hash + anchor snippet for upgrades

5) Mint + Anchor
- mint pack with LYGO-MINT verifier
- post anchor snippet
- backfill with URLs/ids

## Anti-annoyance rule
No background activity. Only run when invoked.
