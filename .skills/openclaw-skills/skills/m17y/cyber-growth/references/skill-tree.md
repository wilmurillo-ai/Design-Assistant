# Skill Tree - 皮层增强协议

各领域的升级曲线和解锁内容定义。

## 领域列表

| 领域 ID | 赛博名称 | 描述 |
|---------|----------|------|
| `api` | API Protocol | API 集成、数据接口、第三方服务 |
| `security` | Security Shell | 安全加固、权限管理、隐私保护 |
| `automation` | Automation Rig | 自动化脚本、定时任务、工作流 |
| `ops` | Ops Matrix | 运维监控、日志分析、故障排查 |
| `frontend` | Frontend Chrome | 前端开发、UI/UX、界面设计 |
| `backend` | Backend Core | 后端开发、数据库、服务架构 |
| `data` | Data Stream | 数据处理、分析、ETL |
| `ai` | Neural Link | AI/ML、模型调用、Prompt 工程 |
| `cloud` | Cloud Grid | 云服务、容器化、部署 |
| `docs` | Doc Codec | 文档写作、知识管理、Wiki |
| `social` | Signal Hub | 社区互动、分享、帮助他人 |
| `feishu` | Feishu Nexus | 飞书生态、多维表格、机器人 |

## 等级曲线

每级所需 XP = `50 × level × 1.1^(level-1)`

| 等级 | 所需 XP（累计） | 称号 |
|------|----------------|------|
| Lv.1 | 0 | 新手 Runner |
| Lv.2 | 55 | 街头黑客 |
| Lv.3 | 176 | 网格潜行者 |
| Lv.4 | 381 | 数据猎手 |
| Lv.5 | 697 | 协议专家 |
| Lv.6 | 1,155 | 系统渗透者 |
| Lv.7 | 1,795 | 网络幽灵 |
| Lv.8 | 2,667 | 暗网传奇 |
| Lv.9 | 3,834 | 神经领主 |
| Lv.10 | 5,376 | 超频核心 |

## 领域解锁条件

| 等级 | 解锁内容 |
|------|----------|
| Lv.1 | 基础协议激活，开始记录 XP |
| Lv.3 | 解锁该领域的 **中级协议** |
| Lv.5 | 解锁 **Exploit**（技巧/捷径识别） |
| Lv.7 | 解锁该领域的 **高级协议** |
| Lv.10 | **完全掌握** — 可以指导他人（Signal Broadcast XP 翻倍） |

## 总等级计算

总 Chrome Level = 所有领域 XP 总和对应的等级

但各领域独立追踪，鼓励多领域发展而非单一刷级。
