# React Component Generator

快速生成 React 组件代码。

## 功能

- ⚡ 一键生成组件
- 📝 支持 TypeScript
- 🎣 内置 Hooks
- 🎨 多种组件类型

## 组件类型

| 类型 | 说明 |
|------|------|
| function | 函数组件 |
| class | 类组件 |
| hook | 自定义 Hook |
| hoc | 高阶组件 |

## 使用方法

```bash
react-component-generator MyButton --type function --typescript
react-component-generator useMyHook --type hook
```
