# Git Hooks Generator

自动生成 Git hooks，提升开发效率。

## 功能

- ✅ pre-commit 格式化
- ✅ commit-msg 规范检查
- ✅ pre-push 测试检查
- ✅ 多种语言支持

## 使用方法

```bash
git-hooks-generator init
git-hooks-generator add --hook pre-commit --type lint
```
