# Role Memory Template

<!--
  使用说明:
  1. 此文件作为 .claude/teams/<team_name>/<role_name>/memory.md 的初始模板
  2. 执行过程中，角色主动将经验教训、踩坑记录、用户强调的内容追加到此文件
  3. 每条记录包含日期和结构化字段，便于 generate_prompts.py 读取注入
-->

## Lessons（经验教训）

<!-- 格式:
### YYYY-MM-DD: <简短标题>
- **场景**: <什么情况下>
- **洞察**: <学到了什么>
- **应用**: <下次怎么用>
-->

<!-- 示例:
### 2026-04-15: 增量构建比全量重建快得多
- **场景**: 实现 RAG 索引更新时
- **洞察**: 只处理变更文件的增量构建比全量重建快 10x
- **应用**: 下次遇到索引重建优先检查变更范围
-->

## Pitfalls（踩坑记录）

<!-- 格式:
### YYYY-MM-DD: <问题描述>
- **症状**: <表现是什么>
- **根因**: <根本原因>
- **修复**: <怎么解决的>
-->

<!-- 示例:
### 2026-04-15: Agent 无法识别新的 subagent_type
- **症状**: 实例化时报 "Agent type not found"
- **根因**: 创建 .claude/agents/ 后未重启 Claude Code
- **修复**: 创建角色配置后必须提示用户重启
-->
