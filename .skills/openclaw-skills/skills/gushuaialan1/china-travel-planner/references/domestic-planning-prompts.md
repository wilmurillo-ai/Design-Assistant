# Domestic planning prompt patterns

Use these example phrasings as triggers and as planning templates.

## Typical user requests

- 杭州周末两天一夜怎么安排
- 清明从上海去苏州玩三天，预算 2000 怎么规划
- 五一国内亲子游去哪儿比较合适
- 北京出发去成都，帮我做个 4 天攻略
- 三亚住哪个区域比较方便，顺便推荐酒店
- 西安 3 日游怎么排路线更顺
- 帮我比较杭州和南京，哪个更适合周末散心
- 预算 3000，两个人，国内去哪里玩性价比高

## Default plan templates

### 2D1N
- Day 1: depart + core attraction + night walk / food street
- Day 2: secondary attraction + lunch + return

### 3D2N
- Day 1: arrive + light sightseeing
- Day 2: full main sightseeing day
- Day 3: relaxed neighborhood / museum / brunch + return

### 4D3N+
- Split city core, landmark day, flexible buffer day, and departure day
- Avoid booking every hour too tightly

## Common domestic planning dimensions

- 出发方式：高铁 / 飞机 / 自驾
- 住宿区域：景区周边 / 市中心 / 交通枢纽附近
- 人群类型：情侣 / 朋友 / 家庭 / 亲子 / 独自
- 风格：省钱 / 舒适 / 打卡 / 松弛 / 吃喝为主
- 风险提示：节假日拥堵、天气、排队、跨城折腾
