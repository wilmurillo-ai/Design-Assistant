# Templates

This directory stores reusable page shells.

## Current template

- `trip-page-tailwind.html`
  - Tailwind-based single-page travel itinerary shell
  - Intended to render schema-driven travel content

## Template rules

- Do not hardcode destination-specific wording into the template.
- Keep reusable section order stable.
- Prefer rendering from structured JSON data.
- Shared card shapes should stay consistent across destinations.
