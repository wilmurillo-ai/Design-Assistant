# Examples

This directory contains real trip-page examples built on the shared schema and template conventions.

## Current examples

- `changsha-2026-03-31/`
  - A Changsha metro-coverage + side-trip itinerary example
  - Demonstrates hotels, metroCoverage, days, sideTrips, attractions, and tips

## Purpose

Examples should prove that the framework can support actual trip-planning scenarios, not just abstract demo content.

Each example should:
- keep trip content in `data/trip-data.json`
- keep presentation logic in `index.html`
- stay aligned with `schema/trip-schema.json`
