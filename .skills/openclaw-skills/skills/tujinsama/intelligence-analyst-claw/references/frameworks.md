# 分析框架库

## SWOT 分析
**适用**：企业/项目综合评估
- 优势（Strengths）：内部核心竞争力
- 劣势（Weaknesses）：内部短板
- 机会（Opportunities）：外部有利因素
- 威胁（Threats）：外部不利因素

## 波特五力模型
**适用**：行业竞争格局分析
- 供应商议价能力
- 买方议价能力
- 新进入者威胁
- 替代品威胁
- 现有竞争者竞争强度

## PEST 分析
**适用**：宏观环境扫描
- 政治（Political）：政策法规、监管环境
- 经济（Economic）：宏观经济、利率、汇率
- 社会（Social）：人口结构、消费趋势
- 技术（Technological）：技术成熟度、创新速度

## DCF 估值模型
**适用**：投资价值评估
- 预测未来自由现金流（FCF）
- 确定折现率（WACC）
- 计算终值（Terminal Value）
- 折现求和得企业价值

## 行业分析标准维度

| 维度 | 关注点 |
|------|--------|
| 市场规模 | TAM/SAM/SOM，增长率（CAGR） |
| 竞争格局 | CR4/CR8，HHI 指数，头部玩家份额 |
| 商业模式 | 收入来源，盈利模式，变现路径 |
| 技术壁垒 | 专利数量，研发投入，技术成熟度（TRL） |
| 政策环境 | 监管框架，政策支持/限制，合规要求 |
| 资本热度 | 融资轮次分布，投资机构偏好，估值倍数 |
