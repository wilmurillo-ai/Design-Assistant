# 信息源体系

## 公开数据源（免费）

### 政府与官方机构
- 国家统计局：https://www.stats.gov.cn — 宏观经济数据
- 工信部：https://www.miit.gov.cn — 工业/科技政策
- 证监会/港交所/SEC — 上市公司财报
- 中国人民银行：https://www.pbc.gov.cn — 金融数据

### 行业媒体
- 36氪：https://36kr.com — 科技/创投
- 虎嗅：https://www.huxiu.com — 商业分析
- 钛媒体：https://www.tmtpost.com — TMT 行业
- 晚点 LatePost：https://latepost.com — 深度报道

### 学术资源
- Google Scholar：https://scholar.google.com
- arXiv：https://arxiv.org — 技术论文
- 知网：https://www.cnki.net — 中文学术

### 投融资数据
- Crunchbase：https://www.crunchbase.com — 全球投融资
- IT桔子：https://www.itjuzi.com — 中国投融资
- 企查查/天眼查 — 企业工商信息

## 搜索策略

### 行业研究
```
搜索词模板：
- "[行业名] 市场规模 2024 研究报告"
- "[行业名] industry market size report 2024"
- "[行业名] 竞争格局 头部企业"
- site:iresearch.com.cn [行业名]
```

### 公司调研
```
搜索词模板：
- "[公司名] 财报 年报 2023"
- "[公司名] annual report investor relations"
- "[公司名] 融资 估值"
- "[公司名] 竞争对手 市场份额"
```

### 政策追踪
```
搜索词模板：
- "[政策关键词] 最新政策 2024"
- site:gov.cn [政策关键词]
- "[行业] 监管 新规"
```

## 可信度评级

| 等级 | 类型 | 示例 | 可信度 |
|------|------|------|--------|
| 一手 | 官方公告、财报、白皮书 | SEC 财报、政府公告 | ★★★★★ |
| 二手 | 行业报告、主流媒体 | 艾瑞报告、36氪 | ★★★★☆ |
| 三手 | 社交媒体、论坛 | 微博、知乎 | ★★★☆☆ |
