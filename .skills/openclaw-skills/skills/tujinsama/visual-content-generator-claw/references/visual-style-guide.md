# 视觉风格规范

账号视觉设计规范，确保所有生成素材风格统一。

## 主色调

| 色彩角色 | 色值 | 用途 |
|----------|------|------|
| 主色 | `#1A73E8`（科技蓝） | 主要视觉元素、标题 |
| 辅助色 | `#34A853`（活力绿） | 强调、CTA按钮 |
| 深色背景 | `#0D1117` | 科技感深色背景 |
| 浅色背景 | `#F8F9FA` | 简洁风格背景 |
| 禁用色 | 大红色 `#FF0000` | 避免单独大面积使用 |

> 可根据实际品牌色修改此文件。

## 字体规范

| 用途 | 字体 | 备注 |
|------|------|------|
| 中文标题 | 思源黑体 Bold / 阿里巴巴普惠体 | 醒目有力 |
| 中文正文 | 思源宋体 / 苹方 | 易读性好 |
| 英文标题 | Montserrat Bold | 现代感 |
| 装饰文字 | 站酷快乐体 / 庞门正道标题体 | 活泼风格 |

## 风格标签与 Prompt 关键词

| 风格 | 适用场景 | Prompt 关键词 |
|------|----------|---------------|
| 科技感 | AI/互联网/数字化内容 | `futuristic, digital, tech, blue purple gradient, particle effects` |
| 小清新 | 生活/美食/旅行内容 | `soft colors, natural light, minimalist, pastel, warm` |
| 商务 | B端/职场/培训内容 | `professional, corporate, clean, modern office, business` |
| 国潮 | 传统文化/国货/节日 | `Chinese traditional, ink painting, red gold, cultural` |
| 极简 | 品牌/高端/设计感 | `minimalist, white space, clean lines, elegant, simple` |

## 构图偏好

| 构图方式 | 适用场景 | 说明 |
|----------|----------|------|
| 居中构图 | 海报、封面 | 主体居中，视觉稳定 |
| 三分法 | 配图、背景 | 主体偏左或偏右，留白给文字 |
| 对称构图 | 科技感、正式场合 | 左右对称，庄重感 |
| 大留白 | 极简风格 | 大量空白，突出主体 |
