# 意向词库

## 强意向词（评分贡献 40-50 分）

直接表达购买行为或强烈需求：

- 求链接 / 求购买链接 / 发链接 / 链接发我
- 哪里买 / 在哪买 / 哪里有卖 / 怎么买
- 怎么下单 / 如何购买 / 购买渠道
- 有没有优惠 / 有折扣吗 / 有券吗 / 打折吗
- 我要买 / 我想买 / 准备入手 / 决定买了
- 已下单 / 已购 / 买了 / 入手了（确认购买，可用于竞品分析）
- 多少钱 / 价格是多少 / 报个价

## 中意向词（评分贡献 20-35 分）

表达兴趣或考虑购买：

- 好用吗 / 效果怎么样 / 值得买吗
- 有没有推荐 / 求推荐 / 推荐一下
- 求测评 / 有没有测评 / 谁用过
- 和 xxx 比哪个好 / 哪个更值
- 有没有同款 / 有没有类似的
- 想试试 / 想入手 / 考虑买
- 有点心动 / 感觉不错 / 看起来不错

## 弱意向词（评分贡献 5-15 分）

表达潜在需求或关注：

- 最近在找 xxx / 一直在找
- 想换一个 / 想换掉
- 感觉需要 / 好像需要
- 有没有更好的 / 有没有替代品
- 收藏了 / 先收藏

## 行业定制词库

### 美妆护肤
- 强意向：求色号 / 求小样 / 哪里有正品
- 中意向：适合什么肤质 / 敏感肌能用吗 / 孕期能用吗
- 弱意向：成分怎么样 / 和 xxx 成分一样吗

### 3C 数码
- 强意向：哪里有货 / 有没有现货 / 官网还是旗舰店
- 中意向：续航怎么样 / 和 xxx 比哪个好 / 值不值得升级
- 弱意向：参数怎么样 / 评测在哪看

### 服装
- 强意向：有没有 S 码 / 还有货吗 / 链接发我
- 中意向：面料怎么样 / 版型如何 / 显不显胖
- 弱意向：搭配什么好看 / 有没有同款

### 食品
- 强意向：哪里买得到 / 有没有网购渠道
- 中意向：口味怎么样 / 甜不甜 / 适合小孩吃吗
- 弱意向：配料表是什么 / 热量多少

## 排除词（命中则降低评分或排除）

- 不想买 / 不打算买 / 不会买
- 太贵了不买 / 买不起
- 已经有了 / 不需要了
- 广告 / 推广 / 恰饭（疑似营销号）
