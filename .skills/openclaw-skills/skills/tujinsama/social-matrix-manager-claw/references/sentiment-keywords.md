# 舆情关键词库

## 负面词汇库（高风险，需立即预警）

### 投诉类
质量差、假货、骗人、退款、维权、投诉、举报、欺骗、虚假宣传、货不对板

### 情绪类
垃圾、恶心、失望、后悔、坑、骗局、黑心、无良、差评、踩雷

### 风险类
封号、限流、违规、下架、删除、举报成功

## 正面词汇库（高价值信号）

### 种草类
好用、推荐、必买、回购、安利、种草、真香、绝了、yyds

### 购买意向类
求链接、在哪买、多少钱、怎么购买、有没有优惠、求同款

### 互动类
学到了、有用、收藏了、转发给朋友、@了某人

## 行业定制词库扩展

在 SKILL.md 中引导用户提供行业特定词汇，例如：
- 美妆行业：成分党、敏感肌、烂脸、过敏
- 食品行业：变质、异味、食品安全、拉肚子
- 教育行业：退课、退费、效果差、没用

## 情感分析规则

- 单条评论含 ≥2 个负面词 → 标记为高风险
- 同一内容负面评论占比 > 10% → 触发舆情预警
- 同一内容正面购买意向词出现 > 20次 → 标记为高转化内容
