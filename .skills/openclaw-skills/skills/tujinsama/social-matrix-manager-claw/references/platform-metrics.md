# 平台数据指标手册

## 核心指标定义

| 指标 | 计算方式 | 说明 |
|------|----------|------|
| 互动率 | (点赞+评论+分享) / 播放量 × 100% | 衡量内容吸引力 |
| 播放完成率 | 完播次数 / 播放次数 × 100% | 衡量内容留存 |
| 涨粉率 | 新增粉丝 / 原有粉丝 × 100% | 衡量账号增长 |
| 爆款系数 | 当日播放量 / 近30日日均播放量 | 识别爆款内容 |

## 各平台健康基准值

### 抖音
- 互动率正常范围：1%–5%（万粉以上账号）
- 播放完成率正常范围：30%–60%
- 日均涨粉率正常范围：0.1%–1%

### 小红书
- 互动率正常范围：3%–10%
- 收藏率正常范围：1%–5%
- 日均涨粉率正常范围：0.05%–0.5%

### 视频号
- 互动率正常范围：1%–4%
- 分享率正常范围：0.5%–2%

### B站
- 互动率正常范围：2%–8%
- 投币率正常范围：0.5%–3%

### 微博
- 互动率正常范围：0.5%–3%
- 转发率正常范围：0.1%–1%

## 异常阈值配置（可自定义）

```yaml
# 爆款预警：播放量超过近30日均值的倍数
viral_threshold: 3x

# 数据下滑预警：连续下降天数
decline_days: 3

# 负面舆情预警：负面评论占比
negative_ratio: 0.1  # 10%

# 账号风险：互动率骤降幅度
engagement_drop: 0.5  # 跌幅超过50%
```
