# 内容分类与标签体系

## 内容类型标签

| 类型 | 标签 | 特征描述 |
|------|------|----------|
| 干货教程 | `tutorial` | 步骤清晰、有实操价值 |
| 娱乐搞笑 | `entertainment` | 轻松幽默、情绪价值高 |
| 产品种草 | `product` | 展示产品、引导购买 |
| 活动促销 | `promotion` | 限时优惠、活动信息 |
| UGC互动 | `ugc` | 用户参与、话题挑战 |
| 知识科普 | `knowledge` | 专业内容、权威背书 |
| 生活记录 | `vlog` | 真实日常、情感共鸣 |

## 爆款内容特征提取规则

爆款内容通常具备以下特征（满足3条以上即为潜在爆款）：
1. 前3秒有强钩子（问题/冲突/反常识）
2. 标题含数字或对比词
3. 评论区有大量提问或求链接
4. 收藏率 > 互动率均值的2倍
5. 发布后6小时内播放量超过账号均值

## 跨账号内容复用建议逻辑

当某账号出现爆款内容时，按以下逻辑判断是否推荐复用：
1. 内容类型是否适合其他账号的定位
2. 目标受众是否有重叠（避免同质化竞争）
3. 爆款系数 > 5x 时，优先推荐同类型账号复制
4. 复用建议：改编角度、更换封面、调整文案风格
