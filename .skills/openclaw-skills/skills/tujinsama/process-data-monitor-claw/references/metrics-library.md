# 监控指标库

常见业务场景的预定义监控指标，可直接引用或作为配置参考。

## 电商场景

| 指标 | 说明 | 正常范围 | 告警阈值 | 告警级别 |
|------|------|----------|----------|----------|
| 订单转化率 | 下单/访客 | ≥ 2% | < 1% | warning |
| 支付成功率 | 支付成功/支付发起 | ≥ 95% | < 90% | urgent |
| 发货及时率 | 24h内发货/总订单 | ≥ 98% | < 95% | warning |
| 待发货超时订单数 | 超24h未发货订单 | 0 | > 0 | urgent |
| 退货率 | 退货/发货 | ≤ 5% | > 10% | warning |
| 库存周转率 | 销售量/平均库存 | 行业基准 | 连续3天下降 | warning |
| 库存差异量 | 线上线下库存差 | 0 | > 0 | urgent |
| 库存同步延迟 | 同步任务延迟时间 | < 5min | > 30min | warning |

## SaaS 场景

| 指标 | 说明 | 正常范围 | 告警阈值 | 告警级别 |
|------|------|----------|----------|----------|
| API 响应时间 | P99 响应时间 | < 500ms | > 2000ms | urgent |
| API 错误率 | 5xx/总请求 | < 0.1% | > 1% | urgent |
| 并发用户数 | 实时在线用户 | 容量 80% 以内 | > 90% 容量 | warning |
| 数据同步延迟 | ETL 任务延迟 | < 15min | > 60min | warning |
| 任务失败率 | 失败任务/总任务 | 0% | > 0% | urgent |
| 数据库连接数 | 活跃连接数 | < 最大连接 80% | > 90% | warning |

## 营销场景

| 指标 | 说明 | 正常范围 | 告警阈值 | 告警级别 |
|------|------|----------|----------|----------|
| 活动流量 | 实时 PV/UV | 预期范围内 | 突增 > 300% 或骤降 > 50% | urgent |
| 转化漏斗各环节 | 各步骤转化率 | 历史均值 ±20% | 偏差 > 50% | warning |
| 活动库存消耗速率 | 库存消耗/小时 | 预期速率 | 库存 < 10% | urgent |
| ROI | 收入/投入 | ≥ 1 | < 0.5 | warning |
| 用户留存率（次日） | 次日活跃/新增 | ≥ 30% | < 20% | warning |

## 数据管道场景

| 指标 | 说明 | 正常范围 | 告警阈值 | 告警级别 |
|------|------|----------|----------|----------|
| ETL 任务状态 | 任务执行结果 | success | failed/timeout | urgent |
| 数据延迟 | 数据到达延迟 | < 预期窗口 | > 2× 预期窗口 | warning |
| 数据量异常 | 实际/预期数据量 | 80%~120% | < 50% 或 > 200% | warning |
| 消息队列积压 | 未消费消息数 | < 1000 | > 10000 | urgent |
