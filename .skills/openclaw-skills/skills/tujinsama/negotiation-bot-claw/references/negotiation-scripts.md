# 谈判话术库

各场景标准话术模板，供 `email-negotiator.sh` 和 AI 生成邮件时参考。

---

## 首轮询价话术

**原则**：专业、明确需求、留议价空间（不透露目标价）

```
尊敬的 {{SUPPLIER_NAME}} 团队，

您好！我司正在进行 {{PRODUCT}} 的采购评估，计划采购数量约 {{QUANTITY}} 件/套。

为便于我司进行供应商评估，烦请贵司提供以下信息：
1. 产品报价（含税单价）
2. 最小起订量及批量折扣政策
3. 交货周期
4. 付款条件（账期）
5. 质保期及售后服务政策

期待贵司的回复，我司将综合评估后尽快给出合作意向。

此致
{{SENDER_NAME}}
采购部
```

## 压价话术

**原则**：委婉施压，暗示竞争，要求让步

### 场景 A：报价偏高，直接要求调整

```
尊敬的 {{SUPPLIER_NAME}} 团队，

感谢贵司的报价。经我司评估，贵司报价（{{QUOTED_PRICE}}）与我司预算及市场行情存在一定差距。

我司非常希望与贵司建立长期合作关系，但目前的报价难以满足我司的采购预算要求。

请问贵司是否有调整空间？如能在价格上给予支持，我司将优先考虑贵司作为本次采购的合作伙伴。

期待贵司的回复。

此致
{{SENDER_NAME}}
```

### 场景 B：竞价施压（暗示已有更低报价）

```
尊敬的 {{SUPPLIER_NAME}} 团队，

感谢贵司的报价。目前我司已收到多家供应商的报价，部分供应商的报价较贵司更具竞争力。

考虑到贵司的产品质量和服务口碑，我司希望给贵司一个再次报价的机会。

如贵司能在现有报价基础上进一步优化，我司将认真考虑与贵司优先合作。请于 3 个工作日内回复。

此致
{{SENDER_NAME}}
```

## 条款优化话术

**原则**：价格已接近目标，转向优化账期、质保等条款

```
尊敬的 {{SUPPLIER_NAME}} 团队，

感谢贵司的报价，价格方面基本符合我司预期。

为推进合作，我司希望就以下条款进一步沟通：
1. 账期：我司希望将付款账期延长至 60 天（Net 60）
2. 质保期：希望延长至 24 个月
3. 退换货政策：希望明确质量问题的退换货流程

如贵司能接受上述条款，我司可尽快推进合同签署。

期待贵司的回复。

此致
{{SENDER_NAME}}
```

## 拒绝话术

**原则**：礼貌拒绝过高报价，保持后续合作可能

```
尊敬的 {{SUPPLIER_NAME}} 团队，

感谢贵司参与本次采购评估，并提供详细报价。

经综合评估，本次采购我司选择了其他供应商合作。主要原因是价格因素，贵司报价超出了我司本次采购预算。

我司非常认可贵司的产品质量，希望未来有机会再次合作。如贵司日后有更具竞争力的报价方案，欢迎随时联系。

感谢贵司的支持与配合。

此致
{{SENDER_NAME}}
```

## 成交话术

**原则**：确认最终条款，推进合同签署

```
尊敬的 {{SUPPLIER_NAME}} 团队，

经过多轮沟通，我司决定与贵司就本次 {{PRODUCT}} 采购达成合作。

确认最终条款如下：
- 采购数量：{{QUANTITY}}
- 成交单价：{{FINAL_PRICE}}
- 交货周期：{{DELIVERY_DAYS}} 个工作日
- 付款条件：{{PAYMENT_TERMS}}
- 质保期：{{WARRANTY_PERIOD}}

请贵司安排合同起草，我司将在收到合同后 3 个工作日内完成审核和签署。

感谢贵司的配合，期待愉快合作！

此致
{{SENDER_NAME}}
```
