# 供应商画像库

记录供应商历史谈判行为，用于制定个性化谈判策略。

---

## 供应商分类

### 强硬型供应商
**特征**：
- 首轮报价接近底价，让步空间小（<5%）
- 不轻易回应竞价施压
- 通常是行业头部或垄断供应商

**应对策略**：
- 减少施压轮次，避免关系破裂
- 转向条款优化（账期、质保、服务）
- 引入替代供应商作为真实备选

### 灵活型供应商
**特征**：
- 首轮报价虚高20%-30%，让步空间大
- 对竞价施压敏感
- 重视长期合作关系

**应对策略**：
- 多轮施压，逐步逼近底价
- 强调长期合作潜力
- 竞价话术效果显著

### 急单型供应商
**特征**：
- 月末/季末明显更愿意让步
- 库存积压时报价更低
- 对付款速度敏感

**应对策略**：
- 选择月末发起谈判
- 提出预付款换折扣
- 快速决策，不拖延

## 谈判时机参考

| 时机 | 效果 | 说明 |
|-----|------|------|
| 月末最后3天 | ★★★★★ | 供应商冲业绩，让步幅度最大 |
| 季末最后一周 | ★★★★★ | 同上，效果更强 |
| 行业淡季 | ★★★★ | 供应商订单少，更愿意谈 |
| 供应商库存积压 | ★★★★ | 主动降价清库存 |
| 月初/季初 | ★★ | 供应商不急，让步有限 |
| 行业旺季 | ★ | 供应商订单充足，谈判筹码弱 |

## 常见话术应对

| 供应商说 | 实际含义 | 应对方式 |
|---------|---------|---------|
| "这是我们的底价" | 通常还有2%-5%空间 | 继续施压，或转向条款 |
| "需要请示领导" | 有让步空间 | 耐心等待，不要催促 |
| "我们成本就这么高" | 试图结束谈判 | 要求提供成本明细 |
| "其他客户都是这个价" | 标准话术 | 强调我司采购量和长期价值 |
| "再低就亏本了" | 接近底线 | 转向条款优化 |
