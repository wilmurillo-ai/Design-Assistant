# 文件分类标准

## 文件类型

| 类型代码 | 类型名称 | 存储路径 | 档案编号前缀 |
|---------|---------|---------|------------|
| LAW | 法律法规 | 法律法规库/ | LAW- |
| REG | 公司制度 | 公司制度库/ | REG- |
| STD | 行业规范 | 行业规范库/ | STD- |
| TPL | 合同模板 | 合同模板库/ | TPL- |
| MAN | 操作手册 | 操作手册库/ | MAN- |

## 公司制度子分类

| 子分类 | 存储路径 | 示例文件 |
|-------|---------|---------|
| 人事制度 | 公司制度库/人事制度/ | 员工手册、考勤制度、薪酬制度 |
| 财务制度 | 公司制度库/财务制度/ | 报销制度、采购制度、预算管理 |
| 业务制度 | 公司制度库/业务制度/ | 销售管理、客户服务规范 |
| IT制度 | 公司制度库/IT制度/ | 信息安全制度、数据管理规范 |
| 安全制度 | 公司制度库/安全制度/ | 安全生产制度、应急预案 |

## 命名规范

统一格式：`[类型代码]-[部门]-[文件名称]-v[版本号].[扩展名]`

示例：
- `REG-HR-员工手册-v3.0.pdf`
- `LAW-LEGAL-数据安全法实施细则-v1.0.pdf`
- `TPL-LEGAL-保密协议模板-v2.1.docx`

## 档案编号规则

格式：`[类型代码]-[年份]-[三位序号]`

示例：`REG-2026-042`

序号在同类型同年份内自增，从 001 开始。
