# 行业补货参数库

## 标准补货参数

| 品类 | 安全库存 | 补货周期 | 备注 |
|-----|---------|---------|------|
| 快消品（食品饮料、日化） | 7天销量 | 14天 | 高频次、短保质期 |
| 服装鞋帽 | 14天销量 | 30天 | 季节性强，提前备货 |
| 电子产品 | 21天销量 | 45天 | 供应链长，备货周期长 |
| 生鲜食品 | 3天销量 | 7天 | 保质期短，少量多次 |
| 家居家具 | 30天销量 | 60天 | 体积大，物流周期长 |
| 美妆护肤 | 14天销量 | 21天 | 中等周转 |
| 图书文具 | 30天销量 | 45天 | 低频次，稳定需求 |

## 补货计算公式

```
补货触发点 = 日均销量 × (供应商交货周期 + 安全库存天数)
建议补货量 = 日均销量 × 补货周期 + 安全库存 - 当前库存 - 在途库存
```

## 补货优先级

| 优先级 | 触发条件 | 建议行动 |
|-------|---------|---------|
| 🔴 紧急 | 当前库存 < 3天销量 | 立即下单，考虑加急运输 |
| 🟡 正常 | 当前库存 < 补货触发点 | 按计划下单 |
| 🟢 可延后 | 当前库存 > 补货触发点 × 1.5 | 暂不补货，监控趋势 |

## 自定义参数说明

- **安全库存天数**：可根据供应商稳定性调整（供应商不稳定 → 增加安全库存）
- **补货周期**：包含下单处理时间 + 物流时间 + 入库时间
- **在途库存**：已下单但未到货的库存，计入可用库存
