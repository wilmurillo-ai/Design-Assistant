# 预测模型库

## 模型选择指南

| 数据特征 | 推荐模型 |
|---------|---------|
| 平稳、无明显趋势 | 移动平均法 |
| 有趋势、无季节性 | 指数平滑法 |
| 有明显季节性 | Holt-Winters / 季节性分解 |
| 数据量充足（>180天）、多影响因素 | Prophet / XGBoost |

## 统计模型

### 移动平均法（MA）
- 适用：平稳型商品，无明显趋势
- 参数：窗口期 N（建议 7/14/30 天）
- 公式：`预测值 = 近N天销量之和 / N`

### 指数平滑法（ES）
- 适用：有趋势的商品
- 参数：平滑系数 α（0.1~0.3 保守，0.5~0.8 激进）
- 公式：`F(t) = α × A(t-1) + (1-α) × F(t-1)`

### Holt-Winters
- 适用：有趋势 + 季节性的商品
- 参数：趋势系数 β、季节系数 γ、季节周期 m
- 季节周期：周粒度 m=7，月粒度 m=12

### ARIMA
- 适用：时间序列平稳化后的预测
- 参数：(p, d, q)，建议用 auto_arima 自动选参

## 机器学习模型

### Prophet（Facebook）
- 适用：有节假日效应、多季节性的商品
- 优势：自动处理缺失值、异常值、节假日
- 安装：`pip install prophet`
- 最少数据量：90天

### XGBoost / LightGBM
- 适用：多特征输入（促销、天气、竞品等）
- 最少数据量：180天
- 特征工程：滞后特征、滚动统计、节假日编码

## 模型评估指标

| 指标 | 公式 | 说明 |
|-----|------|------|
| MAE | mean(|实际-预测|) | 平均绝对误差，直观 |
| MAPE | mean(|实际-预测|/实际) × 100% | 百分比误差，可比较 |
| RMSE | sqrt(mean((实际-预测)²)) | 对大误差更敏感 |

**目标**：MAPE < 20% 为良好，< 10% 为优秀
