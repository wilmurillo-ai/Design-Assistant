# 促销效应系数库

## 主要促销活动系数

| 活动类型 | 拉升系数范围 | 峰值天 | 备注 |
|---------|------------|-------|------|
| 双11（11.11） | 5x ~ 8x | 11月11日 | 全年最大促销节点 |
| 618（6.18） | 3x ~ 5x | 6月18日 | 年中大促 |
| 双12（12.12） | 2x ~ 3x | 12月12日 | 年末补货节点 |
| 38女王节 | 2x ~ 4x | 3月8日 | 美妆、服装类更高 |
| 99大促 | 1.5x ~ 2.5x | 9月9日 | 中等规模 |
| 日常满减 | 1.3x ~ 1.8x | 活动当天 | 常规促销 |
| 直播带货 | 2x ~ 4x | 直播当天 | 头部主播可达 5x+ |
| 限时秒杀 | 3x ~ 6x | 秒杀时段 | 持续时间短 |
| 新品首发 | 1.5x ~ 3x | 上线前3天 | 取决于推广力度 |

## 使用方法

```
活动预测销量 = 基准日均销量 × 拉升系数 × 活动天数
```

**注意**：
- 拉升系数取历史同类活动的实际数据，无历史数据时取区间中值
- 大促前预热期（活动前3-7天）通常有 1.2x ~ 1.5x 的预热效应
- 大促后通常有 1-3天的销量低谷（需求透支效应）

## 系数修正因子

| 因素 | 修正方向 | 修正幅度 |
|-----|---------|---------|
| 折扣力度比去年更大 | 上调 | +10% ~ +30% |
| 折扣力度比去年更小 | 下调 | -10% ~ -20% |
| 竞品同期大促 | 下调 | -5% ~ -15% |
| 独家活动/平台流量倾斜 | 上调 | +10% ~ +20% |
| 新品（无历史数据） | 参考同类品 | 取同类品系数 × 0.7 |
