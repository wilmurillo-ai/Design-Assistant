## 售后问题

### 退换货
**Q: 如何退货？**
A: 支持 7 天无理由退货（商品完好、未使用）。退货步骤：
1. 在订单页面申请退货
2. 等待审核通过（1-2 个工作日）
3. 按提示寄回商品
4. 收到商品后 3-5 个工作日退款

**Q: 如何换货？**
A: 换货流程：
1. 在订单页面申请换货，说明换货原因
2. 等待审核通过
3. 寄回原商品
4. 收到原商品后发出新商品

**Q: 商品有质量问题怎么办？**
A: 非常抱歉给您带来不便！质量问题可申请免费退换货，无需承担运费。请提供问题照片，我们会优先处理。

### 物流查询
**Q: 我的快递到哪了？**
A: 您可以通过以下方式查询物流：
1. 在订单详情页点击"查看物流"
2. 使用快递公司官网/APP 输入运单号查询
如有异常，请联系我们处理。

**Q: 快递很久没有更新怎么办？**
A: 如果物流超过 3 天没有更新，可能是快递延误。请提供您的订单号，我帮您联系快递公司查询。

### 发票
**Q: 如何开发票？**
A: 开票步骤：
1. 在订单详情页点击"申请发票"
2. 填写发票信息（个人/企业）
3. 提交后 3-5 个工作日内开具并发送到您的邮箱

## 账户问题

### 注册与登录
**Q: 如何注册账号？**
A: 注册步骤：
1. 点击"注册"按钮
2. 输入手机号/邮箱
3. 验证码验证
4. 设置密码完成注册

**Q: 忘记密码怎么办？**
A: 找回密码步骤：
1. 点击登录页面的"忘记密码"
2. 输入注册手机号/邮箱
3. 验证身份后重置密码

### 会员权益
**Q: 会员有什么权益？**
A: 会员权益包括：
- 专属折扣（9折起）
- 优先发货
- 专属客服通道
- 积分加速（2倍积分）
- 生日礼包

**Q: 如何查询积分？**
A: 登录后在"我的账户" → "积分中心"查看积分余额和明细。
