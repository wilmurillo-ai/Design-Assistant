# 数据异常规则库

## 异常判定标准

### 播放量/阅读量

| 规则 | 条件 | 严重程度 |
|------|------|---------|
| 播放量暴跌 | 日环比下降 > 30% | 🔴 高 |
| 播放量下滑 | 日环比下降 15%-30% | 🟡 中 |
| 播放量异常增长 | 日环比增长 > 200% | 🟡 中（可能刷量） |

### 涨粉数

| 规则 | 条件 | 严重程度 |
|------|------|---------|
| 涨粉异常（爆发） | 单日涨粉 > 近 7 日均值 × 3 | 🟡 中 |
| 掉粉预警 | 单日掉粉 > 100 | 🔴 高 |
| 掉粉轻微 | 单日掉粉 50-100 | 🟡 中 |

### 互动率

| 规则 | 条件 | 严重程度 |
|------|------|---------|
| 互动率极低 | 低于行业基准 50%（抖音 < 1.5%，小红书 < 2.5%） | 🔴 高 |
| 互动率偏低 | 低于行业基准 30% | 🟡 中 |

### 完播率（视频平台）

| 规则 | 条件 | 严重程度 |
|------|------|---------|
| 完播率极低 | < 15% | 🔴 高 |
| 完播率偏低 | 15%-25% | 🟡 中 |

### 数据缺失

| 规则 | 条件 | 严重程度 |
|------|------|---------|
| 数据采集失败 | 某平台数据为空或采集报错 | 🔴 高 |
| 数据延迟 | 数据时间戳与当前时间差 > 2 小时 | 🟡 中 |

---

## 行业基准参考值

| 平台 | 互动率基准 | 完播率基准 | 日均涨粉（万粉账号） |
|------|-----------|-----------|-------------------|
| 抖音 | 3%-5% | 35%-50% | 50-200 |
| 小红书 | 5%-8% | — | 30-100 |
| 视频号 | 2%-4% | 30%-45% | 20-80 |
| B站 | 4%-7% | 40%-60% | 30-100 |
| 微博 | 1%-3% | — | 20-60 |

## 异常处理建议

### 播放量暴跌
- 检查是否有内容被限流或删除
- 检查发布时间是否偏离最佳时段
- 对比同期竞品数据，判断是否平台整体下滑

### 掉粉预警
- 检查近期内容是否引发负面反馈
- 查看评论区是否有集中投诉
- 检查是否有批量取关行为（可能是买粉被清）

### 互动率极低
- 检查内容质量和话题相关性
- 检查发布时间和频率
- 考虑增加互动引导（提问、投票等）

### 数据采集失败
- 检查 API 密钥是否过期
- 检查 Cookie 是否失效（小红书、B站）
- 检查网络连通性和平台 API 状态
