# 爆款内容分析模型

基于大量爆款案例总结的内容分析框架，供 AI 分析热点时参考。

## 标题公式库（高点击率模板）

### 疑问式
- `为什么XXX的人越来越多？`
- `XXX真的有用吗？我测试了30天告诉你`
- `你还在用XXX？难怪效果差`

### 对比式
- `用了XXX之后，我的YYY提升了ZZZ%`
- `同样是XXX，为什么他能YYY而你不行？`
- `XXX vs YYY，用了3个月后我选择了...`

### 数字式
- `XXX个让你YYY的方法，第N个最有效`
- `我花了XXX小时，总结出这N条YYY规律`
- `N个月做到XXX，我只做对了这N件事`

### 揭秘式
- `XXX背后没人告诉你的真相`
- `做了N年XXX，我终于明白了...`
- `那些YYY的人，都在偷偷用XXX`

### 共鸣式
- `作为一个XXX，我想说...`
- `终于有人说出了XXX的真相`
- `看完这个，我哭了`

## 内容角度库（12种）

| 角度 | 说明 | 适用场景 |
|------|------|----------|
| 观点型 | 鲜明立场，引发讨论 | 热点事件、行业趋势 |
| 故事型 | 个人经历，情感共鸣 | 成长、转变、失败复盘 |
| 教程型 | 步骤清晰，实操性强 | 技能分享、工具使用 |
| 盘点型 | 合集整理，信息密度高 | 资源推荐、工具清单 |
| 测评型 | 亲测对比，有说服力 | 产品评测、方法对比 |
| 揭秘型 | 反常识，颠覆认知 | 行业内幕、冷知识 |
| 预测型 | 趋势判断，前瞻性 | 行业分析、未来展望 |
| 辟谣型 | 纠正误区，权威感 | 常见错误认知 |
| 案例型 | 真实案例，可信度高 | 成功/失败案例分析 |
| 对话型 | 问答形式，互动感强 | FAQ、用户问题解答 |
| 情绪型 | 强烈情绪，传播力强 | 社会议题、价值观 |
| 数据型 | 数据支撑，专业感强 | 行业报告、调研结果 |

## 情绪共鸣点（8种触发器）

| 情绪 | 触发方式 | 示例关键词 |
|------|----------|-----------|
| 焦虑 | 指出危机/差距 | 落后、被淘汰、错过、来不及 |
| 好奇 | 制造信息缺口 | 秘密、真相、没人知道、原来 |
| 愤怒 | 揭露不公/欺骗 | 骗局、套路、坑、不敢说 |
| 感动 | 温情故事/善意 | 坚持、逆袭、感谢、陪伴 |
| 认同 | 说出心声 | 终于、就是这样、太真实了 |
| 恐惧 | 风险提示 | 危险、警惕、千万别、后悔 |
| 希望 | 正向激励 | 改变、可能、做到、成功 |
| 惊喜 | 超出预期 | 没想到、竟然、原来还能 |

## 传播路径分析

典型爆款传播路径：
1. **首发平台**：通常是微博/知乎（文字类）或抖音/小红书（视觉类）
2. **二次传播**：截图/转发到微信群、朋友圈
3. **破圈路径**：被大V转发 → 登上热搜 → 全平台扩散
4. **长尾效应**：热点过后，相关内容仍有搜索流量

## 热点时效性判断

| 热点类型 | 时效窗口 | 建议响应时间 |
|----------|----------|-------------|
| 突发事件 | 24-48小时 | 2小时内 |
| 娱乐八卦 | 3-7天 | 当天 |
| 社会议题 | 1-2周 | 24小时内 |
| 行业趋势 | 1-3个月 | 3天内 |
| 节日节点 | 固定周期 | 提前1周 |
