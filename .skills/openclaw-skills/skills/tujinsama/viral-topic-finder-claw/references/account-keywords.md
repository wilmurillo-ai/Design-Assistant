# 账号定位关键词库

根据账号定位预设的关键词过滤规则。使用前请根据实际账号定位修改本文件。

## 使用说明

本文件是模板，需要根据实际账号定位填写。AI 在过滤热点时会参考这里的规则。

---

## 示例：科技/AI 类账号

### 核心关键词（必须匹配其中之一）
```
AI, 人工智能, ChatGPT, Claude, Gemini, 大模型, LLM, 机器学习, 深度学习,
科技, 互联网, 程序员, 开发者, 产品经理, 创业, 独立开发
```

### 相关关键词（加分项，提升推荐权重）
```
效率工具, 自动化, 编程, 代码, 算法, 数据, 云计算, SaaS, 出海
```

### 排除关键词（包含则过滤掉）
```
明星, 八卦, 娱乐, 综艺, 游戏, 体育, 美食, 旅游, 时尚, 美妆
```

### 高风险话题黑名单（绝对不碰）
```
政治敏感词, 宗教争议, 种族歧视, 违法内容
```
