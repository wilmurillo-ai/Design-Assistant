# 异常检测规则

## 金额异常检测

**规则**：单笔金额超过该费用类别历史平均值的 3 倍（3σ 原则）

```
历史均值 = 过去 90 天同类别支出的平均金额
异常阈值 = 历史均值 × 3
触发条件：本次申请金额 > 异常阈值
```

**白名单**：以下场景豁免金额异常检测
- 年度合同续签
- 大型活动预算（需提前标记）
- 设备采购（单次金额天然偏高）

## 频次异常检测

**规则**：滑动窗口内同一申请人高频申请

| 时间窗口 | 申请次数上限 |
|---------|------------|
| 1 天 | 5 次 |
| 7 天 | 15 次 |
| 30 天 | 40 次 |

超出上限触发黄灯预警，通知申请人直属上级。

## 类目异常检测

**规则**：费用类别与部门常规支出组合不符

常规组合白名单示例：
```
研发部 → 软件授权、差旅费、外包费、培训费
市场部 → 广告费、差旅费、活动费、礼品费
行政部 → 办公用品、差旅费、餐饮费、物业费
```

不在白名单的组合触发黄灯，提示人工核查。

## 异常等级

| 异常类型 | 默认等级 | 处理方式 |
|---------|---------|---------|
| 金额异常 | 🟡 黄灯 | 通知财务审核 |
| 频次异常 | 🟡 黄灯 | 通知直属上级 |
| 类目异常 | 🟡 黄灯 | 提示人工核查 |
| 多项异常叠加 | 🔴 红灯 | 拦截，需财务总监审批 |
