# 通知模板库

## 黄色预警模板

**飞书消息**：
```
【预算预警 🟡】
部门：{部门} | 项目：{项目（无则省略）}
费用类别：{费用类别}
本月预算：{预算总额}元 | 已使用：{已用金额}元（{使用率}%）
触发申请：{申请人} 申请 {申请金额}元
建议：请注意控制本月剩余支出，可用余额仅剩 {可用余额}元
```

## 红色预警模板

**飞书消息**：
```
【预算超支 🔴】
部门：{部门} | 项目：{项目（无则省略）}
费用类别：{费用类别}
本月预算：{预算总额}元 | 已使用：{已用金额}元
本次申请：{申请金额}元 → 超支 {超支金额}元
申请人：{申请人}
⚠️ 申请已被拦截，需升级至 {审批级别} 审批后方可通过
```

## 异常消费模板

**飞书消息**：
```
【异常消费预警 ⚠️】
异常类型：{金额异常 / 频次异常 / 类目异常}
申请人：{申请人} | 部门：{部门}
申请金额：{金额}元 | 费用类别：{费用类别}
异常说明：{具体异常描述}
请相关负责人核查此笔申请。
```

```
- {部门} {费用类别}：超支 {金额}元

🟡 预警项目（{N}项）：
- {部门} {费用类别}：使用率 {使用率}%
```
