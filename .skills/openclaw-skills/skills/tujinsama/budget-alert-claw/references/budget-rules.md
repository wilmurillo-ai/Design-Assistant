# 预算规则库

## 预警分级标准

| 级别 | 使用率阈值 | 颜色 | 处理方式 |
|------|-----------|------|----------|
| 正常 | < 80% | 🟢 绿灯 | 自动通过，无需通知 |
| 预警 | 80% ~ 99% | 🟡 黄灯 | 通知部门负责人，允许通过 |
| 超支 | ≥ 100% | 🔴 红灯 | 拦截申请，升级审批流程 |

默认阈值可在配置时覆盖，例如：`--warning-threshold 70 --block-threshold 95`

## 超支审批权限配置

| 超支金额 | 审批级别 |
|---------|---------|
| ≤ 5,000 元 | 部门经理审批 |
| 5,001 ~ 50,000 元 | 财务总监审批 |
| > 50,000 元 | CEO 审批 |

## 时间周期规则

- **月度预算**：每月 1 日 00:00 重置，跨月申请按申请日期归属
- **季度预算**：Q1(1-3月)、Q2(4-6月)、Q3(7-9月)、Q4(10-12月)
- **年度预算**：1月1日重置，支持按月均摊查看进度

## 预算余额计算公式

```
可用余额 = 预算总额 - 已审批支出 - 待审批金额
使用率 = (已审批支出 + 本次申请金额) / 预算总额 × 100%
```

## 兜底规则

未配置预算的维度处理方式（三选一，默认为"放行并记录"）：
1. **放行并记录**：允许通过，但记录到未配置预算日志
2. **黄灯预警**：触发黄灯，通知财务部门补充配置
3. **拦截**：直接拦截，要求先配置预算

## 预算配置表格式

```csv

研发部,,差旅费,30000,80,月度

```
