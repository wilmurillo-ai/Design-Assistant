# 维权函件模板库

## 通用格式规范

```
[发函方名称]
[发函方地址]
[日期]
函件编号：[YEAR]-[TYPE]-[SEQ]

[收函方名称]（以下简称"贵方"）：

【事由】
【违约事实陈述】
【法律依据】
【我方诉求】
【期限要求】
【法律后果告知】

特此函告，请贵方认真对待并及时回复。

[发函方名称]（盖章）
[签署人姓名/职务]
[日期]
```

---

## 模板1：催告函（迟延履行）

适用：对方到期未履行，但仍可履行，需设定宽限期。

```markdown
**催 告 函**
函件编号：{YEAR}-CG-{SEQ}

{受函方名称}：

一、事实陈述
我方与贵方于{合同签订日期}签订《{合同名称}》（合同编号：{合同编号}），约定贵方应于{约定履行日期}前{履行内容}。然而，截至本函发出之日（{发函日期}），贵方仍未履行上述义务，已构成迟延履行。

二、法律依据
根据《民法典》第577条、第579条，贵方应承担继续履行及赔偿损失的违约责任。

三、我方诉求
请贵方在收到本函后{宽限天数}日内（即{宽限截止日期}前）完成{履行内容}，并赔偿因迟延履行给我方造成的损失{损失金额}元。

四、法律后果
若贵方在上述期限内仍未履行，我方将依法解除合同，并保留追究贵方全部违约责任的权利，包括但不限于要求支付违约金、赔偿损失及承担诉讼/仲裁费用。

{发函方名称}（盖章）
{日期}
```

## 模板2：催款函（拖欠货款/服务费）

适用：对方拖欠应付款项。

```markdown
**催 款 函**
函件编号：{YEAR}-CK-{SEQ}

{受函方名称}：

一、欠款事实
根据双方签订的《{合同名称}》（合同编号：{合同编号}），贵方应于{付款截止日期}前向我方支付{款项名称}共计{欠款金额}元。截至本函发出之日，贵方已逾期{逾期天数}天，仍未支付上述款项。

二、法律依据
根据《民法典》第579条，贵方应立即支付上述款项，并依法承担逾期付款利息（按中国人民银行同期贷款利率计算）。

三、我方诉求
请贵方在收到本函后{宽限天数}日内支付欠款{欠款金额}元及逾期利息，合计约{合计金额}元。

四、法律后果
若贵方逾期仍未付款，我方将依法提起诉讼/申请仲裁，届时贵方还需承担诉讼/仲裁费用及律师费。

{发函方名称}（盖章）
{日期}
```

## 模板3：解除合同通知

适用：对方根本违约，我方行使法定解除权。

```markdown
**解除合同通知书**
函件编号：{YEAR}-JC-{SEQ}

{受函方名称}：

一、解除声明
我方与贵方于{合同签订日期}签订的《{合同名称}》（合同编号：{合同编号}），现依法通知贵方：我方决定解除上述合同。

二、解除依据
贵方存在以下违约行为：{违约事实描述}。上述行为已构成根本违约，致使合同目的无法实现。根据《民法典》第563条第（四）/（五）项，我方有权解除合同。

三、解除后处理
1. 贵方应于本通知送达之日起{天数}日内退还我方已支付的{预付款/定金}共计{金额}元；
2. 贵方应赔偿我方因此遭受的损失{损失金额}元；
3. 贵方应支付违约金{违约金金额}元（按合同约定）。

四、法律后果
若贵方拒绝履行上述义务，我方将依法提起诉讼/申请仲裁。

{发函方名称}（盖章）
{日期}
```

## 模板4：侵权警告函（知识产权）

适用：发现对方侵犯商标权、著作权、专利权等。

```markdown
**侵权警告函**
函件编号：{YEAR}-QJ-{SEQ}

{受函方名称}：

一、侵权事实
我方系{知识产权类型}（注册号/证书号：{编号}）的合法权利人。我方发现贵方于{发现日期}存在以下侵权行为：{侵权行为描述}，侵犯了我方的合法权益。

二、法律依据
根据{适用法律条款}，贵方上述行为构成侵权，应承担停止侵权、赔偿损失等法律责任。

三、我方诉求
1. 立即停止一切侵权行为；
2. 销毁侵权产品/材料；
3. 赔偿我方经济损失及合理维权费用，共计{索赔金额}元；
4. 在{媒体/平台}上公开道歉，消除影响。

四、期限与后果
请贵方在收到本函后{天数}日内书面回复并采取行动。否则，我方将依法提起诉讼，届时贵方将面临更高额的赔偿责任。

{发函方名称}（盖章）
{日期}
```

## 模板5：质量异议函

适用：对方交付的产品/服务存在质量问题。

```markdown
**质量异议函**
函件编号：{YEAR}-ZL-{SEQ}

{受函方名称}：

一、质量问题陈述
根据《{合同名称}》（合同编号：{合同编号}），贵方于{交付日期}向我方交付了{产品/服务名称}。经我方验收，发现存在以下质量问题：{质量问题描述}，不符合合同约定的{质量标准}。

二、损失情况
上述质量问题已造成我方损失：{损失描述及金额}。

三、我方诉求
1. 在{天数}日内免费修复/更换不合格产品；
2. 赔偿因质量问题造成的损失{损失金额}元；
3. 支付违约金{违约金金额}元。

{发函方名称}（盖章）
{日期}
```

## 模板6：保密协议违约函

适用：对方违反保密协议，泄露商业秘密。

```markdown
**保密协议违约通知函**
函件编号：{YEAR}-BM-{SEQ}

{受函方名称}：

一、违约事实
双方于{签订日期}签订《保密协议》（编号：{编号}），约定贵方对{保密内容}负有保密义务。我方发现贵方于{发现日期}存在以下违约行为：{泄露行为描述}，严重违反了保密协议约定。

二、法律依据
根据《反不正当竞争法》第9条及双方保密协议第{条款}条，贵方应承担停止侵权、赔偿损失等责任。

三、我方诉求
1. 立即停止泄露行为，召回已泄露的保密信息；
2. 赔偿我方损失{损失金额}元；
3. 支付保密协议约定的违约金{违约金金额}元。

{发函方名称}（盖章）
{日期}
```
