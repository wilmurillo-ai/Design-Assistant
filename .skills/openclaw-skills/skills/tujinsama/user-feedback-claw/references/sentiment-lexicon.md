# 情感分析词典（电商/产品场景）

## 强负面词（权重 3）
差、烂、骗人、垃圾、退货、投诉、坑、假货、劣质、恶心、气死、太差了、完全不行、强烈差评、绝对不买、后悔、浪费钱、坑人、欺骗

## 弱负面词（权重 1.5）
一般、还行、有点、不太、稍微、凑合、勉强、不够、差点意思、不是很、有些、略微、不太满意、有待改进

## 强正面词（权重 3）
超好、完美、强烈推荐、五星、必买、太棒了、非常好、超级满意、完全超出预期、神器、绝了、爱了、回购、值得

## 弱正面词（权重 1.5）
不错、可以、还好、满意、挺好、还不错、蛮好、凑合、说得过去、还行、可以接受

## 中性词（权重 1）
普通、正常、一般般、中规中矩、没什么特别

## 特殊场景词
- **物流相关负面**：慢、迟、延误、没到、等了好久、快递差
- **客服相关负面**：不理人、态度差、解决不了、推诿、没用
- **质量相关负面**：坏了、断了、掉了、不能用、失灵、故障
