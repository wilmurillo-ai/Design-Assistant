# 用户反馈问题分类体系

## 分类框架

| 一级分类 | 二级分类 | 典型关键词 |
|---------|---------|-----------|
| 产品质量 | 材质问题 | 掉色、变形、褪色、材质差 |
| 产品质量 | 功能缺陷 | 不能用、坏了、失灵、故障 |
| 产品质量 | 耐用性 | 用了几天就坏、不耐用、易损 |
| 产品质量 | 外观瑕疵 | 划痕、色差、做工粗糙 |
| 物流配送 | 发货速度 | 发货慢、等了很久、迟迟未发 |
| 物流配送 | 包装破损 | 包装烂了、压坏了、破损 |
| 物流配送 | 配送延误 | 快递慢、迟到、超时 |
| 物流配送 | 错发漏发 | 发错了、少发、漏发 |
| 客服体验 | 响应速度 | 不回复、等很久、无人理 |
| 客服体验 | 解决效率 | 问题没解决、推诿、踢皮球 |
| 客服体验 | 态度问题 | 态度差、不耐烦、冷漠 |
| 客服体验 | 售后政策 | 不退款、不换货、售后难 |
| 价格感知 | 性价比 | 贵、不值、性价比低 |
| 价格感知 | 促销规则 | 活动规则复杂、优惠没到账 |
| 价格感知 | 价格变动 | 买完就降价、价格不稳定 |
| 使用体验 | 操作复杂 | 难用、不会用、操作繁琐 |
| 使用体验 | 功能缺失 | 没有某功能、功能不够用 |
| 使用体验 | 兼容性问题 | 不兼容、连不上、不支持 |

## 优先级计算公式

```
优先级分数 = 提及频率 × 情感强度权重 × 评分影响系数

情感强度权重：强负面=3，弱负面=1.5，中性=1
评分影响系数：1-2星=3，3星=2，4星=1，5星=0.5
```

## 输出格式

分析结果按优先级排序，每类问题输出：
- 问题名称 + 数量 + 占比
- 代表性原文（1-3条）
- 建议改进方向
