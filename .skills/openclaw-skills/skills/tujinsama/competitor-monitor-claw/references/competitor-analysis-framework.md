# 竞品分析框架

## 价格策略分析

| 模式 | 特征 | 含义 |
|------|------|------|
| 高频调价 | 每周调价 ≥2 次 | 动态定价策略，跟随流量/库存 |
| 稳价策略 | 30天内变动 <3% | 品牌溢价维护，价格锚定 |
| 大促跟随 | 仅在节点调价 | 被动防御型，无主动定价能力 |
| 渐进降价 | 每月小幅下调 | 清库存或市场份额争夺信号 |

**解读原则**：
- 竞品连续降价 >10% → 可能面临库存压力或新品替代，观察是否跟进
- 竞品涨价 → 检查是否成本上涨（行业性）或供应收紧，评估跟涨空间
- 大促期间价格异常低 → 结合活动日历判断，避免误判为常态降价

## 上新节奏分析

- **高频上新**（每月 >5 SKU）：产品线快速扩张，可能在测款；重点关注哪些款留存
- **低频上新**（每季 <3 SKU）：精品策略，每款投入大；上新即重点竞品
- **季节性上新**：结合品类特性判断，非季节期上新说明有新战略意图

**判断竞品扩张方向**：连续3个月上新集中在某子品类 → 该方向是竞品战略重点

## 口碑对比分析

**评分基准**（电商平台）：
- ≥4.8 分：优质，用户满意度高
- 4.5–4.8 分：正常区间
- <4.5 分：存在明显问题，关注差评内容

**差评关键词优先级**：
1. 产品质量问题（功能缺陷、安全隐患）→ 高优先级，可作为己方差异化卖点
2. 物流/售后问题 → 中优先级，运营层面可优化
3. 价格/性价比抱怨 → 低优先级，主观感受为主

**竞品优势识别**：高频好评词 = 用户真实需求 → 若己方产品缺失，考虑补强

## 综合竞争态势评估

每月汇总时，对每个竞品输出：
1. **价格竞争力**：当前价格 vs 己方价格的差距百分比
2. **口碑健康度**：评分趋势（上升/平稳/下滑）
3. **活跃度**：上新频率 + 促销频率
4. **威胁等级**：高/中/低（综合以上三项判断）
