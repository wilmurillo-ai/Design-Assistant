# 费用分类规则

## 分类映射

| 费用类型 | 关键词/商家特征 | 发票类型 |
|---------|--------------|---------|
| 餐饮费 | 餐厅、饭店、酒楼、食堂、外卖、咖啡、茶饮、KFC、麦当劳、星巴克 | 餐饮发票、普通发票 |
| 交通费-机票 | 航空、航班、机票、行程单 | 机票行程单 |
| 交通费-火车 | 铁路、高铁、动车、火车 | 火车票 |
| 交通费-出租车 | 出租车、的士、滴滴、曹操、神州、T3 | 出租车票/网约车票 |
| 交通费-地铁 | 地铁、轨道交通、公交 | 交通票 |
| 住宿费 | 酒店、宾馆、旅馆、民宿、客栈、如家、汉庭、希尔顿 | 酒店发票 |
| 办公费 | 文具、耗材、打印、复印、办公用品、电脑、设备 | 普通发票/专用发票 |
| 通讯费 | 中国移动、中国联通、中国电信、话费、网费、宽带 | 普通发票 |
| 会议费 | 会议室、培训、会务 | 普通发票/专用发票 |
| 其他 | 无法匹配以上类别 | 任意 |

## 分类优先级

1. 先按发票类型判断（机票行程单 → 交通费-机票，火车票 → 交通费-火车）
2. 再按销售方名称关键词匹配
3. 再按商品/服务名称关键词匹配
4. 无法匹配时归为"其他"

## 特殊规则

- 同一张发票含多种费用（如酒店含餐饮）→ 按主要费用分类，备注说明
- 金额 > 500 元的餐饮费 → 标记为"需审核"
- 单次住宿 > 800 元/晚 → 标记为"需审核"（超标住宿）
