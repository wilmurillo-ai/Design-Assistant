# 发票类型识别规则

## 增值税专用发票
- 版面特征：左上角印有"增值税专用发票"字样，红色边框
- 关键字段：发票代码（10位）、发票号码（8位）、购买方税号、销售方税号、金额、税额
- 识别重点：税额单独列出，有"价税合计（大写）"行

## 增值税普通发票
- 版面特征：蓝色边框，"增值税普通发票"字样
- 关键字段：发票代码（12位）、发票号码（8位）、金额、税额
- 识别重点：与专票结构相似，但税号可能不完整

## 电子发票（全电发票）
- 版面特征：无纸质，PDF格式，有二维码
- 关键字段：同普通发票，但有电子签章
- 识别重点：直接解析PDF文本，无需图像OCR

## 机票行程单
- 版面特征：航空公司抬头，多行航班信息
- 关键字段：乘客姓名、航班号、出发/到达城市、日期、票价、燃油附加费、合计
- 识别重点：合计 = 票价 + 燃油附加费 + 机场建设费

## 火车票
- 版面特征：中国铁路12306，红色/蓝色底色
- 关键字段：乘客姓名、车次、出发站、到达站、日期、座位、金额
- 识别重点：金额在右下角，座位号格式如"05车08A号"

## 出租车/网约车票
- 版面特征：出租车公司名称，较小尺寸
- 关键字段：日期、金额、里程（部分）
- 识别重点：金额通常在中间大字显示

## 餐饮发票
- 版面特征：餐厅名称，消费明细
- 关键字段：商家名称、日期、金额、税额（部分有）
- 识别重点：注意区分小票（无税号）和正式发票（有税号）

## 酒店发票
- 版面特征：酒店名称，住宿日期范围
- 关键字段：酒店名称、入住/退房日期、房费、税额
- 识别重点：日期可能是范围格式"2026-03-10至2026-03-12"
