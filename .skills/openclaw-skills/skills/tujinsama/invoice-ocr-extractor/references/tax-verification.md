# 税务验真接口说明

## 适用范围

仅适用于增值税专用发票和增值税普通发票（含电子发票）。
机票、火车票、出租车票等不支持税务验真。

## 查验平台

国家税务总局全国增值税发票查验平台：
- 官网：https://inv.chinatax.gov.cn/
- API 文档：需申请企业账号后获取

## 查验所需字段

| 字段 | 说明 | 必填 |
|------|------|------|
| fpdm | 发票代码 | 是 |
| fphm | 发票号码 | 是 |
| kprq | 开票日期（YYYYMMDD） | 是 |
| je | 发票金额（不含税，精确到分） | 是 |
| jym | 校验码后6位（普票必填，专票可选） | 条件必填 |

## 返回结果

| 结果码 | 含义 |
|--------|------|
| 0001 | 查验一致（真实） |
| 0002 | 查无此票 |
| 0003 | 不一致（可能伪造） |
| 0004 | 超过查验次数 |
| 0005 | 已作废 |
| 0006 | 已冲红 |

## 频率限制

- 免费额度：每张发票每天可查验 5 次
- 超出后需付费或次日重试
- 批量验真时，建议每次请求间隔 1 秒

## 脚本调用

```bash
python3 scripts/extract-invoice.py verify \
  --code 3100192130 \
  --number 12345678 \
  --date 2026-03-15 \
  --amount 1000.00
```

## 注意事项

- 验真结果仅供参考，最终以税务局官网查验为准
- 已作废/已冲红的发票不能用于报销
- 查验失败时，建议提示用户手动登录官网查验
