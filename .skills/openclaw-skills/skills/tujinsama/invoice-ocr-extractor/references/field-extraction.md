# 字段提取规则

## 金额字段

### 识别格式
- 小写：`1,234.56` → `1234.56`
- 大写：`壹仟贰佰叁拾肆元伍角陆分` → `1234.56`
- 带符号：`¥1,234.56` → `1234.56`

### 大写金额转换表
```
零=0, 壹=1, 贰=2, 叁=3, 肆=4, 伍=5, 陆=6, 柒=7, 捌=8, 玖=9
拾=10, 佰=100, 仟=1000, 万=10000, 亿=100000000
元/圆, 角=0.1, 分=0.01
```

### 校验规则
- `amount + tax_amount = total_amount`（允许 ±0.01 误差）
- 金额不能为负数（红字发票除外）
- 单张发票金额上限参考：普票 10 万，专票无上限

## 日期字段

### 识别格式
- `2026年03月15日` → `2026-03-15`
- `2026-03-15` → `2026-03-15`
- `26.03.15` → `2026-03-15`（年份补全）
- `20260315` → `2026-03-15`

### 校验规则
- 开票日期不能晚于今天
- 开票日期不能早于 2000-01-01

## 发票代码

- 增值税专票：10位纯数字，前6位为省市代码
- 增值税普票：12位纯数字
- 全电发票：20位数字
- 正则：`^\d{10}$` 或 `^\d{12}$` 或 `^\d{20}$`

## 发票号码

- 8位纯数字
- 正则：`^\d{8}$`

## 税号（统一社会信用代码）

- 18位，字母+数字混合
- 正则：`^[0-9A-Z]{18}$`
- 第1位：登记管理部门代码（1-9,A-N,P-Y）
- 第2位：机构类别代码

## 公司名称

- 去除前后空格
- 保留括号内容（如"有限公司"、"股份有限公司"）
- 常见后缀：有限公司、股份有限公司、有限责任公司、集团、总公司

## 税率

- 常见值：0%, 1%, 3%, 5%, 6%, 9%, 13%
- 识别格式：`13%` 或 `0.13` → 统一存为 `13%`
- 免税发票：税率字段为"免税"或"不征税"
