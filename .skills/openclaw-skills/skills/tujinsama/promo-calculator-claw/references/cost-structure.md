# 成本结构参考模板

## 电商标品（3C配件、家居用品等）
- 采购成本：实际进货价
- 平台佣金：5-8%（按售价）
- 快递费：5-12元/件
- 退货率：5-10%（退货成本 = 退货率 × 采购成本 × 0.5）
- 营销成本：广告费按件分摊，通常5-20元/件

## 服装鞋帽
- 采购成本：实际进货价
- 平台佣金：5-10%
- 快递费：8-15元/件（含退换运费）
- 退货率：20-40%（退货成本较高，需重点核算）
- 营销成本：10-30元/件

## 食品生鲜
- 采购成本：实际进货价
- 冷链物流：15-30元/件
- 损耗率：5-15%（损耗成本 = 损耗率 × 采购成本）
- 平台佣金：3-6%
- 保质期风险：需考虑滞销损耗

## 数码电子
- 采购成本：实际进货价
- 平台佣金：3-5%
- 快递费：10-20元/件（含保价）
- 退货率：3-8%
- 保修成本：售价的1-3%
- 营销成本：20-50元/件

## 成本核算公式
```
总成本 = 采购成本 + 物流成本 + 平台佣金 + 营销成本 + 退货成本 + 仓储成本
退货成本 = 退货率 × (采购成本 + 物流成本)
平台佣金 = 售价 × 佣金率
```

## 常见漏算项
- 退货运费（买家退货的来回运费）
- 资金占用成本（大批量备货的利息）
- 包装耗材成本
- 客服人力成本分摊
