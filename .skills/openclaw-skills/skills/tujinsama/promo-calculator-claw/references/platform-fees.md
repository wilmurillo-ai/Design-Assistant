# 主流电商平台费率参考

## 淘宝/天猫
- 技术服务费：0.5-1%
- 佣金：3-8%（按类目）
  - 服装：5-8%
  - 3C数码：3-5%
  - 食品：3-5%
  - 家居：3-6%
- 天猫超市：另有仓储配送费

## 京东
- POP商家佣金：3-8%（按类目）
- 自营：不适用（京东自采）
- 物流费：POP商家自承担，或使用京东物流（按重量计费）
- 技术服务费：1000元/年起

## 拼多多
- 佣金：0.6%（极低，但推广费用高）
- 推广费：CPC模式，竞争激烈类目可达5-15%
- 百亿补贴：平台补贴，商家需配合降价

## 抖音电商
- 佣金：1-5%（按类目）
  - 服装：4-5%
  - 食品：3-4%
  - 3C：2-3%
- 直播坑位费：数千至数万元/场（需单独核算）
- 达人佣金：10-30%（如走达人带货）

## 小红书
- 佣金：5%起
- 笔记推广费：另计

## 费率使用说明
- 以上费率为参考区间，实际以平台最新公告为准
- 大促期间平台可能有费率优惠
- 新商家可能有费率减免政策
