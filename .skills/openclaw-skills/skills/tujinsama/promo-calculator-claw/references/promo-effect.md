# 促销效果系数参考

## 各类促销形式的销量拉升系数

| 促销类型 | 销量拉升系数 | 说明 |
|---------|------------|------|
| 直降5% | 1.1-1.3x | 感知弱，效果有限 |
| 直降10% | 1.3-1.8x | 常规促销效果 |
| 直降15% | 1.6-2.2x | 明显拉升 |
| 直降20%+ | 2.0-3.5x | 强力促销，注意利润 |
| 满减（门槛合理） | 1.5-2.5x | 提升客单价效果好 |
| 满减（门槛过高） | 1.1-1.4x | 门槛超过客单价150%效果差 |
| 赠品（高感知价值） | 1.2-1.6x | 赠品需有明显价值感 |
| 赠品（低感知价值） | 1.0-1.2x | 效果有限 |
| 限时秒杀 | 3.0-8.0x | 短时爆发，持续时间短 |
| 买赠（买一送一） | 2.0-4.0x | 清库存效果好 |
| 套装捆绑 | 1.3-2.0x | 提升连带率 |

## 大促节点系数（叠加基础系数）
- 双11：基础系数 × 1.5-3.0x（全年最大促销节点）
- 618：基础系数 × 1.2-2.0x
- 双12：基础系数 × 1.1-1.5x
- 日常大促（38女王节、99大促等）：基础系数 × 1.1-1.3x

## 使用说明
- 系数为参考区间，实际效果受品类、竞争、历史口碑影响
- 建议结合历史同类活动数据校准系数
- 悲观情景建议取系数下限，乐观情景取上限
- 首次参加大促建议用中性系数（区间中值）
