# 命名规范模板

## 通用格式

- 日期格式：`YYYY-MM-DD`（推荐）或 `YYYYMMDD`
- 分隔符：下划线 `_` 或连字符 `-`
- 避免空格和特殊字符

## 内置模板

| 场景 | 格式 | 示例 |
|------|------|------|
| 项目文件 | `项目名_模块_版本_日期` | `ProjectA_UI_v2_2026-04-20` |
| 照片 | `日期_地点_事件` | `2026-04-20_上海_会议` |
| 文档 | `类型_主题_作者_日期` | `报告_Q1总结_Wayne_2026-04` |
| 截图 | `Screenshot_YYYY-MM-DD_HHMMSS` | `Screenshot_2026-04-20_143000` |

## 批量重命名规则

支持变量替换：
- `{date}` → 文件修改日期
- `{name}` → 原文件名（不含扩展名）
- `{ext}` → 扩展名
- `{index}` → 序号（001, 002...）
- `{parent}` → 父目录名
