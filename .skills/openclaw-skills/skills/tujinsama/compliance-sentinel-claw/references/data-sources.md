# 数据源配置

## 免费数据源（内置）

### 1. 国家企业信用信息公示系统
- 查询地址：https://www.gsxt.gov.cn/
- 数据内容：工商注册信息、经营异常、行政处罚、股权信息
- 更新频率：T+1（工作日）
- 查询方式：公司名称 或 统一社会信用代码

### 2. 中国裁判文书网
- 查询地址：https://wenshu.court.gov.cn/
- 数据内容：法院判决文书、诉讼记录
- 更新频率：不定期（法院上传后可见）
- 查询方式：当事人名称

### 3. 全国法院失信被执行人名单
- 查询地址：https://zxgk.court.gov.cn/shixin/
- 数据内容：失信被执行人（老赖）名单
- 更新频率：实时
- 查询方式：公司名称 或 统一社会信用代码

### 4. 国家税务总局异常名录
- 查询地址：https://www.chinatax.gov.cn/
- 数据内容：税务异常、非正常户
- 更新频率：月度

## 商业数据源（可选，提升质量）

### 企查查 API
- 文档：https://openapi.qcc.com/
- 覆盖：工商、司法、舆情、知识产权、招投标
- 优势：数据更全、更新更快、支持批量查询
- 配置：在脚本中设置 `QCC_API_KEY` 环境变量

### 天眼查 API
- 文档：https://open.tianyancha.com/
- 覆盖：工商、司法、新闻舆情、经营风险
- 优势：舆情监控能力强
- 配置：在脚本中设置 `TYC_API_KEY` 环境变量

## 数据源优先级

```
商业数据源（企查查/天眼查）> 免费官方数据源
```

有商业 API Key 时优先使用商业数据源；无 API Key 时降级到免费数据源（覆盖范围有限）。

## 查询频率限制

| 数据源 | 免费限制 | 商业限制 |
|-------|---------|---------|
| 企查查 | 不适用 | 按套餐，通常 1000 次/天 |
| 天眼查 | 不适用 | 按套餐 |
| 国家信用系统 | 无明确限制，建议 < 100 次/小时 | — |
| 裁判文书网 | 无明确限制，建议 < 50 次/小时 | — |
