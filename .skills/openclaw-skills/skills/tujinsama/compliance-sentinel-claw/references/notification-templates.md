# 通知模板库

## 高风险预警模板

### 失信被执行人
```
【🔴 紧急预警】合规哨兵监控报告

合作方：{公司名称}
统一社会信用代码：{信用代码}
风险类型：失信被执行人
发现时间：{发现时间}

详情：{公司名称} 已被列入全国法院失信被执行人名单。

⚠️ 建议立即行动：
1. 暂停与该公司的新增业务往来
2. 评估现有合同的履行风险
3. 通知法务部启动风险评估
4. 如有应收款项，加快催收

档案链接：{档案链接}
```
