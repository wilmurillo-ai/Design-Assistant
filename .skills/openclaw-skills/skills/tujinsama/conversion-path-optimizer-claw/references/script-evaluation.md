# 话术效果评估框架

## 核心指标

| 指标 | 计算方式 | 说明 |
|------|----------|------|
| 话术转化率 | 使用该话术后成交数 / 使用总次数 | 核心指标 |
| 使用频率 | Top Sales 使用次数 | 验证话术的实战价值 |
| 平均成交周期 | 使用该话术后平均成交天数 | 越短越好 |
| 适用客户类型 | 按行业/规模/痛点分类统计 | 精准匹配场景 |

## A/B 测试设计

1. **分组**：随机将客户分为 A/B 两组，分别使用不同话术
2. **样本量**：每组至少 50 个有效对话，确保统计显著性
3. **控制变量**：确保两组客户画像相似（行业、规模、来源渠道）
4. **评估周期**：至少观察 2 周，避免短期波动干扰
5. **显著性检验**：使用卡方检验，p < 0.05 才认为差异显著

## 话术评分模型

综合评分 = 转化率 × 0.5 + 使用频率得分 × 0.3 + 成交周期得分 × 0.2

- 转化率得分：直接使用百分比
- 使用频率得分：归一化到 0-100
- 成交周期得分：100 - (实际周期/平均周期 × 100)，越短得分越高

## 时间衰减策略

话术效果会随市场变化而衰减：
- 每季度重新评估话术库
- 转化率下降 >10% 的话术标记为"待更新"
- 新增数据超过 200 条时触发自动重新排序
