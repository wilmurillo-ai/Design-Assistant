# 话术挖掘算法库

## 核心方法

### 1. 高频短语提取（TF-IDF）

从成交对话中提取高频关键短语：
- 计算每个短语在成交对话中的 TF-IDF 值
- 对比流失对话，找出成交对话中显著高频的短语
- 这些短语即为候选高转化话术

### 2. 对话结构分析

识别"问题-回答-反馈"模式：
```
客户提出异议 → 销售应对 → 客户反馈（积极/消极）
```
统计哪些应对方式获得积极反馈的比例最高。

### 3. 情感倾向判断

客户情感状态分类：
- **积极**：表达兴趣、认可、追问细节
- **犹豫**：要求对比、询问竞品、拖延决策
- **抗拒**：明确拒绝、价格异议、功能质疑

针对不同情感状态，推荐不同话术策略。

## 话术分类体系

| 场景 | 触发信号 | 推荐策略 |
|------|----------|----------|
| 开场白 | 首次接触 | 建立信任、快速定位需求 |
| 需求挖掘 | qualification 阶段 | SPIN 提问法 |
| 价值呈现 | demo 阶段 | 案例+ROI 量化 |
| 价格异议 | "太贵了" | 价值对比、分期方案、ROI 计算 |
| 竞品对比 | "我们在看 XX" | 差异化优势、客户证言 |
| 拖延决策 | "再考虑考虑" | 限时优惠、风险消除 |
| 临门一脚 | negotiation 阶段末期 | 小步骤承诺、试用方案 |

## 数据质量要求

- 建议最少 100 条成交记录才能产生有效统计
- 对话记录需包含：时间戳、发言方（销售/客户）、内容
- 去除测试账号、内部沟通记录
