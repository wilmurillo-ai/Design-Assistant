# 噪音识别模板

## HTML 噪音

常见需要清除的 HTML 元素：
- 标签：`<div>` `<span>` `<p>` `<a>` `<img>` `<script>` `<style>` `<table>` `<td>` `<tr>` 等
- 属性：`class=` `id=` `style=` `href=` `src=`
- HTML 实体：`&nbsp;` `&amp;` `&lt;` `&gt;` `&quot;` → 转换为对应字符或空格
- 处理方式：使用 BeautifulSoup 或正则 `<[^>]+>` 提取纯文本

## 广告文案特征

以下模式通常为广告噪音，可直接删除：
- "点击查看更多" / "查看详情" / "立即购买" / "限时优惠"
- "广告" / "推广" / "赞助"
- 含有 URL 的纯推广文本（非数据字段）
- "关注我们" / "扫码关注" / "微信公众号"

## 导航/页脚元素

网页抓取数据中常见的无效文本：
- 导航栏：首页、关于我们、联系我们、新闻中心、产品中心
- 页脚：版权所有、ICP备案、隐私政策、使用条款
- 面包屑：`首页 > 分类 > 详情`

## 无效字符

| 类型 | 示例 | 处理 |
|------|------|------|
| 表情符号 | 😀🎉❤️ | 删除或替换为空 |
| 零宽字符 | U+200B U+FEFF | 删除 |
| 控制字符 | \x00-\x1f（除 \n \t） | 删除 |
| 重复标点 | ！！！ …… | 压缩为单个 |
| 全角空格 | `　`（U+3000） | 替换为半角空格 |

## 敏感词/脱敏规则

- 手机号：保留前 3 位和后 4 位，中间替换为 `****`（如 `138****8000`）
- 身份证：保留前 6 位和后 4 位，中间替换为 `********`
- 银行卡：保留后 4 位，其余替换为 `****`
- 敏感词：替换为 `***`，不直接删除整行（避免数据丢失）
