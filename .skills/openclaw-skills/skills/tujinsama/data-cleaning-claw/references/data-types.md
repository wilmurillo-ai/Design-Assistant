# 数据类型识别库

## 日期识别

常见格式（按优先级排序）：

| 格式 | 示例 |
|------|------|
| `YYYY-MM-DD` | 2026-03-01 |
| `YYYY/MM/DD` | 2026/03/01 |
| `YYYY年MM月DD日` | 2026年3月1日 |
| `MM/DD/YYYY` | 03/01/2026 |
| `DD-MM-YYYY` | 01-03-2026 |
| `YYYYMMDD` | 20260301 |
| `YYYY-MM-DD HH:MM:SS` | 2026-03-01 14:30:00 |

- 使用 `pandas.to_datetime(errors='coerce')` 自动解析，失败则标记为异常
- 歧义格式（如 `01/02/03`）优先按 `YYYY/MM/DD` 解析，无法确定时标记为可疑

## 金额识别

正则：`[¥$€£]?\s*[\d,]+\.?\d*`

- 货币符号：¥ $ € £ ￥ USD CNY
- 千分位：`,`（如 `1,200,000`）
- 小数点：`.`
- 负数：`-` 前缀或 `()` 包裹（如 `(1200.00)` 表示负数）

## 联系方式识别

| 类型 | 正则 | 示例 |
|------|------|------|
| 手机号 | `1[3-9]\d{9}` | 13800138000 |
| 固话 | `\d{3,4}-\d{7,8}` | 010-12345678 |
| 邮箱 | `[\w.+-]+@[\w-]+\.[a-z]{2,}` | user@example.com |
| 网址 | `https?://[^\s]+` | https://example.com |

## 身份证识别

- 格式：18 位，前 17 位为数字，最后一位为数字或 X
- 正则：`[1-9]\d{5}(18|19|20)\d{2}(0[1-9]|1[0-2])(0[1-9]|[12]\d|3[01])\d{3}[\dX]`
- 校验：最后一位可通过加权求和验证（可选，性能敏感时跳过）

## 布尔值识别

以下值统一识别为布尔：
- True：`true` `True` `TRUE` `1` `yes` `Yes` `是` `Y`
- False：`false` `False` `FALSE` `0` `no` `No` `否` `N`
