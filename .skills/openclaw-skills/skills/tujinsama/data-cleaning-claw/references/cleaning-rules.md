# 数据清洗规则库

## 去重规则

### 完全去重
- 所有字段完全相同的行视为重复，保留第一条（或最新一条）

### 关键字段去重
- 指定关键字段（如手机号、邮箱、订单号），关键字段相同则视为重复
- 保留策略：优先保留字段最完整的记录；字段完整度相同时保留最新记录

### 模糊去重
- 文本字段去除空格/大小写后比较
- 姓名字段：全角/半角统一后比较

## 缺失值处理

| 字段类型 | 默认策略 | 备注 |
|---------|---------|------|
| 数值 | 填充均值或中位数 | 异常分布时用中位数 |
| 文本 | 填充"未知" | 关键字段建议删除该行 |
| 日期 | 标记为异常，不自动填充 | 避免引入错误时间 |
| 布尔值 | 填充 False/0 | 视业务语义决定 |

- 空字符串、"NULL"、"N/A"、"nan"、"-" 均视为缺失值

## 格式标准化

### 日期
- 目标格式：`YYYY-MM-DD` 或 `YYYY-MM-DD HH:MM:SS`
- 识别并转换：`MM/DD/YYYY`、`DD-MM-YYYY`、`YYYY年MM月DD日`、`2026/3/1` 等

### 金额
- 去除货币符号（¥ $ € £）
- 去除千分位逗号
- 统一为浮点数，保留 2 位小数
- 示例：`¥1,200.00` → `1200.00`

### 电话号码
- 手机号：统一为 11 位纯数字（去除 +86、空格、横线）
- 固话：统一为 `区号-号码` 格式
- 超出合法长度的号码标记为异常

### 文本
- 去除首尾空格（strip）
- 全角字符转半角（数字、字母、标点）
- 统一换行符为 `\n`

### 邮箱
- 转为小写
- 验证格式：`xxx@xxx.xxx`

## 异常值处理

| 字段 | 异常判断 | 处理策略 |
|------|---------|---------|
| 年龄 | <0 或 >150 | 标记为可疑 |
| 金额 | <0（非退款场景） | 标记为可疑 |
| 手机号 | 非 11 位 | 标记为异常 |
| 日期 | 未来日期（业务不允许时） | 标记为可疑 |

- 默认策略：标记（添加 `_flag` 列），不直接删除，保留人工复核
