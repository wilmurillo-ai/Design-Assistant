# 脚本结构模板库

## 模板一：知识科普型

适用场景：讲解知识点、科普概念、分析现象

```
【开场钩子 0-5s】
反常识观点 或 震惊数据
例："99%的人都用错了这个功能"

【问题引入 5-15s】
建立共鸣，点出痛点
例："你是不是也遇到过这种情况..."

【知识讲解 15-60s】
核心知识点，分3个层次递进
- 是什么
- 为什么
- 怎么用

【案例说明 60-120s】
1-2个具体案例，增强可信度
例："我有个朋友，用了这个方法之后..."

【总结升华 120-150s】
提炼核心观点，升华主题

【结尾CTA 最后10s】
引导关注/点赞/评论
```

## 模板二：故事叙事型

适用场景：个人经历、案例故事、情感共鸣

```
【悬念开场 0-5s】
直接抛出结果或冲突
例："那一天，我差点失去了一切..."

【背景交代 5-20s】
简短交代人物和背景

【冲突展开 20-80s】
核心矛盾和挑战，情绪递进

【高潮转折 80-120s】
关键转折点，情绪最高点

【结局反转 120-150s】
出乎意料的结局，引发思考

【结尾CTA 最后10s】
情感共鸣 + 互动引导
```

---

## 模板三：教程实操型

适用场景：工具教程、技能教学、操作指南

```
【痛点共鸣 0-5s】
直击目标用户的核心痛点
例："每次做PPT都要花3小时？"

【解决方案预告 5-10s】
一句话说明解决方案
例："今天教你一个5分钟搞定的方法"

【步骤拆解 10-120s】
3-5个清晰步骤，每步配操作画面
步骤1：...
步骤2：...
步骤3：...

【效果展示 120-140s】
前后对比，展示成果

【结尾CTA 最后10s】
保存/收藏引导 + 关注
```

---

## 模板四：观点输出型

适用场景：行业观点、趋势分析、价值观表达

```
【反常识观点 0-5s】
抛出与主流认知相反的观点
例："我认为努力工作是一种错误"

【论据支撑 5-60s】
3个有力论据，逻辑递进
论据1：数据/研究支撑
论据2：案例佐证
论据3：逻辑推演

【案例佐证 60-120s】
1-2个具体案例强化观点

【行动号召 120-150s】
给出具体的行动建议

【结尾CTA 最后10s】
引发讨论 + 互动引导
```

## 播客大纲模板

```
【开场钩子 0-2min】
- 本期核心问题/悬念
- 嘉宾/主题简介

【背景铺垫 2-5min】
- 话题背景和重要性
- 听众为什么要关注

【主体内容 5-40min】
- 核心观点1（配案例）
- 核心观点2（配案例）
- 核心观点3（配案例）
- 深度探讨/争议点

【互动环节 40-50min】
- 听众问题回答
- 嘉宾互动

【结尾CTA 50-60min】
- 本期核心总结（3点）
- 行动建议
- 下期预告
- 订阅/关注引导
```
