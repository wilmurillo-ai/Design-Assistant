# 分镜脚本格式规范

## 标准分镜表格格式

每个镜头包含以下字段：

| 字段 | 说明 | 示例 |
|------|------|------|
| 镜头编号 | 格式：SC-001 | SC-001 |
| 时间码 | 格式：开始-结束 | 00:00-00:05 |
| 景别 | 见景别说明 | 中景 |
| 运镜 | 见运镜说明 | 固定 |
| 画面描述 | 具体场景和动作 | 主播坐在书桌前，面对镜头 |
| 台词/旁白 | 完整台词文本 | "你知道吗，90%的人都..." |
| 字幕 | 屏幕显示文字 | 【震惊！90%的人都不知道】 |
| BGM/音效 | 背景音乐或音效 | 轻快节奏感背景音乐 |
| 特效/转场 | 后期特效建议 | 无 / 快切 / 淡入淡出 |

## 景别说明

- **特写（CU）**：面部或物体细节，强调情绪或重点
- **近景（MCU）**：胸部以上，适合讲话和表情展示
- **中景（MS）**：腰部以上，展示动作和环境
- **全景（FS）**：全身，展示完整场景
- **远景（LS）**：大场景，交代环境

## 运镜方式

- **固定**：镜头不动，稳定拍摄
- **推**：镜头向前推进，强调主体
- **拉**：镜头向后拉，展示环境
- **摇**：镜头左右/上下摇动
- **移**：镜头平行移动
- **跟**：跟随主体移动

## 分镜脚本示例

```
SC-001 | 00:00-00:05 | 近景 | 固定
画面：主播面对镜头，表情严肃
台词："你知道吗，每天有1000万人在用错误的方式工作"
字幕：【1000万人都在犯的错误】
BGM：无（静音开场，制造紧张感）
特效：无

SC-002 | 00:05-00:15 | 中景 | 缓慢推进
画面：主播站起来，走向白板
台词："今天我要告诉你，一个让效率翻倍的秘密"
字幕：【效率翻倍的秘密】
BGM：轻快背景音乐渐入
特效：无

SC-003 | 00:15-00:30 | 特写 | 固定
画面：白板上写下"番茄工作法"
台词："这个方法叫做番茄工作法，25分钟专注，5分钟休息"
字幕：番茄工作法 = 25分钟专注 + 5分钟休息
BGM：继续
特效：文字出现动画
```

## 字幕样式规范

- **重点词汇**：使用【】包裹，例：【效率翻倍】
- **数字强调**：数字单独成行或加粗显示
- **问句**：保留问号，增强互动感
- **字幕长度**：每行不超过15个汉字
- **停留时间**：与台词同步，每字约0.3秒

## BGM 选择建议

| 内容类型 | BGM 风格 | 节奏 |
|---------|---------|------|
| 知识科普 | 轻快、积极 | 中等节奏 |
| 故事叙事 | 情感、起伏 | 随情节变化 |
| 教程实操 | 简洁、专注 | 稳定节奏 |
| 观点输出 | 有力、激励 | 较快节奏 |
| 播客 | 轻柔、不干扰 | 慢节奏 |
