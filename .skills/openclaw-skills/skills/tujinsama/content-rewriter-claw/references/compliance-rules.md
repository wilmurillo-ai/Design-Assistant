# 查重与合规规则

## 重复率阈值

| 重复率 | 风险等级 | 建议 |
|--------|----------|------|
| < 10% | 安全 | 可直接发布 |
| 10-30% | 中风险 | 建议继续改写 |
| > 30% | 高风险 | 必须重新改写 |

## 各平台敏感词类别

通用敏感词（所有平台）：
- 绝对化用语：最、第一、唯一、100%、绝对
- 医疗健康：治愈、根治、特效、无副作用
- 金融承诺：保本、稳赚、无风险、保证收益
- 政治敏感：涉及政治人物、政策批评

平台差异：
- **抖音/视频号**：额外注意"引流"、"私信"、"加微信"等导流词
- **小红书**：注意"广告"、"推广"需标注，避免虚假种草
- **微信公众号**：注意版权声明，转载需授权

## 版权风险评估

低风险：
- 改写后重复率 < 10%
- 核心观点属于公共知识
- 案例已替换为自有案例

中风险：
- 使用了原文的独特数据/研究
- 保留了原文的独创性表达

高风险：
- 直接引用原文大段内容
- 使用了原文的独家采访/调研数据

## 合规检查流程

1. 自查重复率（目标 < 10%）
2. 检查绝对化用语
3. 检查平台特定敏感词
4. 确认案例/数据来源合法
5. 如有引用，注明出处
