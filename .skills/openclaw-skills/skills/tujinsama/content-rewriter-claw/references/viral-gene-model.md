# 爆款基因提炼模型

## 钩子公式（标题/开场）

| 类型 | 公式 | 示例 |
|------|------|------|
| 悬念型 | [反常识结论] + 原因揭秘 | "月薪3000的人，为什么比月薪3万的人更快乐？" |
| 痛点型 | [目标受众] + 最大痛点 + 解决方案暗示 | "做了5年运营，我才明白这件事" |
| 数字型 | 具体数字 + 结果/方法 | "用这3个方法，我的账号30天涨粉10万" |
| 反常识型 | 颠覆认知的结论 | "越努力越穷？这才是真相" |
| 共鸣型 | 描述目标受众的真实处境 | "每天加班到12点，却感觉什么都没做" |

## 情绪触发器

- **焦虑**：时间紧迫感、竞争压力、错过机会
- **好奇**：信息缺口、反常识、揭秘感
- **愤怒**：不公平、被欺骗、系统性问题
- **感动**：逆袭故事、真实经历、情感共鸣
- **共鸣**：说出了目标受众的心声

## 传播结构（起承转合）

```
开场（前3秒/前100字）：钩子 → 建立共鸣/制造悬念
铺垫（中段）：提供背景/论据/案例
转折（高潮）：反转/揭秘/核心干货
结尾：行动号召/情感升华/互动引导
```

## 价值密度评估

高价值内容特征：
- 每段都有可截图的金句或干货
- 读完有"学到了"或"说出了我的心声"的感觉
- 有具体可操作的方法/步骤
- 有真实案例或数据支撑

## 改写时保留的爆款基因

改写时必须保留：
1. 核心钩子类型（可换内容，不换类型）
2. 情绪触发点（焦虑/好奇/共鸣等）
3. 传播结构（起承转合节奏）
4. 价值密度（干货含量不能降低）
