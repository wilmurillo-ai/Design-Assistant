# 内容改写规则库

## 改写深度层级

### 浅层改写（shallow）
适合：快速产出、低风险内容
- 同义词替换（使用近义词、方言表达）
- 句式调整（主动→被动、陈述→疑问）
- 段落顺序微调
- 预期重复率：30-50%，风险较高，仅用于低敏感场景

### 中层改写（medium）
适合：大多数改编场景
- 段落重组（改变论证顺序）
- 案例替换（同类不同例，优先用国内案例）
- 数据更新（替换为更新或更贴近目标受众的数据）
- 表达方式转换（比喻/故事/对话）
- 预期重复率：10-30%

### 深度重构（deep）
适合：高风险内容、强竞争场景
- 角度转换（从不同立场/视角重新论证）
- 逻辑重建（改变因果关系、论证结构）
- 情绪基调调整（严肃→轻松、客观→主观）
- 人称视角转换（第三人称→第一人称）
- 预期重复率：<10%

### 完全原创（full）
适合：版权敏感、最高安全要求
- 仅保留核心观点/选题方向
- 全新案例、全新论证、全新表达
- 加入差异化观点和独家视角
- 预期重复率：<5%

## 改写操作清单

改写时逐项检查：
- [ ] 标题是否已重写（不能与原文相同）
- [ ] 开场前100字是否完全不同
- [ ] 核心案例是否已替换
- [ ] 数据来源是否已更新
- [ ] 句式结构是否有变化
- [ ] 人称/视角是否已调整
- [ ] 结尾/CTA是否已重写
