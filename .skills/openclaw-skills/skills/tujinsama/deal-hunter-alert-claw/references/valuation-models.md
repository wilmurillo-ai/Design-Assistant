# 估值模型库

## 房产估值

**核心方法：周边成交价对比法**
- 获取目标房源的位置、面积、楼层、建成年份
- 对比同小区/同商圈近3个月成交均价（单价）
- 折扣率 = (市场均价 - 标的单价) / 市场均价 × 100%
- 折扣率 ≥ 20% → 值得关注；≥ 30% → 高优先级预警

**法拍房特殊考量**
- 评估价通常已低于市场价，需对比评估价与市场价的差距
- 起拍价 = 评估价 × 70%（一拍）/ 评估价 × 49%（二拍）
- 关注点：是否有租约、是否有抵押、产权是否清晰

**快速判断口诀**
- 单价低于周边均价20%以上 → 触发预警
- 总价低于同类房源30%以上 → 高优先级

## 二手商品估值

**折扣率参考（相对全新价格）**
| 品相 | 合理折扣率 | 低于此价格值得关注 |
|------|-----------|-----------------|
| 全新未拆封 | 85-95折 | <8折 |
| 九成新 | 6-8折 | <5折 |
| 八成新 | 4-6折 | <3.5折 |
| 七成新及以下 | 2-4折 | <2折 |

**数码产品参考**
- iPhone：参考官网价格，二手九成新通常7-8折；低于6折需关注
- MacBook：参考官网价格，二手九成新通常6-7折；低于5折需关注
- 相机：参考当前市场新品价，二手通常5-7折

**奢侈品参考**
- 热门款（爱马仕、香奈儿等）：二手价可能高于官方零售价
- 普通款：二手通常5-8折，低于4折需关注

## 招标项目评估

**快速评分维度（满分10分）**
- 预算规模匹配度（1-3分）：预算在公司承接范围内
- 业务能力匹配度（1-3分）：业务范围与招标内容吻合
- 竞争难度（1-2分）：参与门槛、资质要求
- 利润空间（1-2分）：预算充足度、付款条件

**评分 ≥ 7分** → 高优先级，立即推送
**评分 5-6分** → 中优先级，汇总推送
**评分 < 5分** → 不推送
