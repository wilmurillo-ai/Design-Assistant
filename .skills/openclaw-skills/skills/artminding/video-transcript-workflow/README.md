# Video Transcript - 视频转录与多平台内容优化技能

> 视频/音频转录 → 优化润色 → 多平台发布稿的完整工作流

---

## 功能特性

- **智能转录**：基于 Whisper 模型，支持 GPU 加速与 CPU 自动回退
- **自动优化**：错误修正、口语润色、术语标注、金句提取
- **多平台输出**：一键生成知乎、微信公众号、小红书三个版本
- **智能推测**：从文件名/上下文推测主题、受众、风格，3分钟无响应自动继续
- **联网知识补充**：自动搜索专业术语，添加背景知识

---

## 安装要求

### 必需依赖

1. **Python 3.8+**
2. **Whisper**（OpenAI 开源语音识别模型）
   ```bash
   pip install openai-whisper
   ```

### GPU 加速（可选）

如需 GPU 加速转录，需安装：
- CUDA 11.x
- PyTorch GPU 版本

> 技能会自动检测 GPU 可用性，不可用时回退到 CPU

## 使用方法

### 1. 基础用法

```
用户：帮我转录这个视频：产品经理入门教程.mp4
```

### 2. 指定输出格式

```
用户：转录这个视频，只要知乎文章
用户：转录这个音频，输出全部版本
```

### 3. 自定义参数

```
用户：转录这个视频，主题是"AI教学"，受众是"高校教师"
```

## 风格说明

### 知乎文章
- **定位**：理性 + 经验 + 科学 + 专业
- **特点**：逻辑递进，论点+论据+案例
- **标题**：提问式或观点式

### 微信公众号
- **定位**：故事感 + 深度 + 专业
- **特点**：故事主线 + 知识穿插 + 情感共鸣
- **标题**：场景化或情感化

### 小红书
- **定位**：极简 + 视觉化 + 话题标签
- **特点**：要点清晰，emoji丰富
- **标题**：吸引眼球，带emoji

## 自动化规则

| 环节 | 自动处理 | 需用户确认 |
|------|----------|------------|
| 背景信息 | 推测 + 3分钟超时默认 | 用户在线时可修改 |
| 错误修正 | 高置信度自动修正 | 低置信度列表展示 |
| 知识补充 | 自动联网搜索 | 无需确认 |
| 金句提取 | 自动提取 | 无需确认 |
| 文章生成 | 自动生成 | 用户可要求修改 |

## 技术细节

### GPU → CPU 自动回退

当 GPU 显存不足或 CUDA 错误时，自动回退到 CPU 模式：

```
GPU (int8_float16) → GPU (int8) → CPU (int8)
```

### 错误修正置信度

**高置信度（自动修正）：**
- 常见技术术语拼写（TensorFlow, PyTorch, GitHub）
- 标准中文同音字（已经 → 已经，在 → 再）
- 常见品牌名（Apple, Microsoft, Google）

**低置信度（列表展示）：**
- 人名、地名、专业术语、数字/日期
