# 9部分报告结构详细规范

## 第一部分：简化结论

### 必须包含要素

| 要素 | 具体要求 | 示例 |
|------|----------|------|
| **投资者评级** | 买入/持有/观望/卖出 | 买入 |
| **交易员评级** | 积极/中性/谨慎/回避 | 积极 |
| **仓位建议** | 基于估值锚点的分档建议（具体百分比） | 建议仓位10-15% |
| **目标价区间** | 保守/中性/乐观三档（具体价格） | 保守35元/中性42元/乐观50元 |
| **硬性止损位** | 必须执行的价格点（具体价格） | 止损位28元 |
| **核心结论** | 3句话：①公司本质定位 ②当前核心矛盾 ③操作建议 | ①液冷龙头，AI算力受益标的 ②估值偏高但趋势强劲 ③分批建仓，严格止损 |
| **关键风险** | 3条，分级：🔴高风险 🟠中风险 🟡低风险 | 🔴估值过高回调风险 🟠行业竞争加剧 🟡原材料价格波动 |

### 格式要求

```markdown

| 要素 | 结论 |
|------|------|
| **投资者评级** | 买入 |
| **交易员评级** | 积极 |
| **仓位建议** | 建议仓位10-15% |
| **目标价区间** | 保守35元 / 中性42元 / 乐观50元 |
| **硬性止损位** | 28元 |

### 核心结论

① **公司本质定位：** 液冷龙头，AI算力基础设施受益标的
② **当前核心矛盾：** 估值偏高（PE 191倍），但趋势强劲，事件催化密集
③ **操作建议：** 分批建仓，严格止损，关注液冷订单进展

### 关键风险

🔴 **高风险：** 估值过高回调风险
🟠 **中风险：** 行业竞争加剧
🟡 **低风险：** 原材料价格波动
```

## 数字格式规范

- **百分比：** 保留1位小数（如：15.3%）
- **价格：** 保留2位小数（如：35.28元）
- **金额：** 
  - 亿元级别：保留2位小数（如：12.35亿元）
  - 万亿元级别：保留2位小数（如：1.25万亿元）
- **市盈率/市净率：** 保留1位小数（如：PE 25.3倍）

## 风险等级标识

- 🔴 **高风险：** 可能导致重大损失的因素
- 🟠 **中风险：** 需要关注但可控的因素
- 🟡 **低风险：** 影响较小的因素
- 🟢 **无风险：** 基本无影响的因素
