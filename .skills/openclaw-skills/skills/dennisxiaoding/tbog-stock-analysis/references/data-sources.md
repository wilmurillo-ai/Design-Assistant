# 数据源说明

## A股数据源

### 东方财富网
- **网址：** https://www.eastmoney.com/
- **数据类型：** 
  - 实时行情（股价、市值、PE/PB、换手率）
  - 财务数据（营收、净利、现金流）
  - 股东数据（股东户数、机构持仓、北向资金）
  - 公告信息（减持、增持、重大合同）
- **特点：** 数据全面，更新及时

### 同花顺
- **网址：** https://www.10jqka.com.cn/
- **数据类型：**
  - 技术指标（支撑位/压力位）
  - 龙虎榜数据
  - 大宗交易数据
- **特点：** 技术分析工具丰富

### 财联社
- **网址：** https://www.cls.cn/
- **数据类型：**
  - 新闻资讯
  - 快讯
  - 深度报道
- **特点：** 新闻时效性强

### 证券时报
- **网址：** https://www.stcn.com/
- **数据类型：**
  - 新闻资讯
  - 公司公告
  - 研报摘要
- **特点：** 官方媒体，可信度高

### 上海证券报
- **网址：** https://www.cnstock.com/
- **数据类型：**
  - 新闻资讯
  - 政策解读
- **特点：** 官方媒体，权威性强

### 巨潮资讯
- **网址：** http://www.cninfo.com.cn/
- **数据类型：**
  - 公司公告
  - 财务报告
  - 股东大会通知
- **特点：** 官方信息披露平台，最权威

## 数据验证规则

### 交叉验证
- 关键数据必须从至少2个来源验证
- 数据冲突时以官方平台为准

### 时效性检查
- 实时数据：不超过1天
- 财务数据：最新报告期
- 新闻数据：近30天

### 异常值处理
- 数据异常时标注"待确认"
- 不使用明显错误的数据
- 优先使用官方披露数据

## 注意事项

### 数据来源标注
- 每个数据点必须标注来源
- 格式：`数据项：XX（来源：东方财富）`

### 预测数据标注
- 所有预测数据必须标注"预测"
- 格式：`预测值：XX（来源：券商研报）`
