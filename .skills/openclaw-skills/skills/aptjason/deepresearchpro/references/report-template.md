# 调研报告模板

## 标准调研报告结构

### 1. 元数据信息
```
调研主题：[用户提出的问题]
调研时间：[执行调研的日期和时间]
调研时长：[实际耗时]
搜索平台：[使用的搜索引擎和网站]
信息源数量：[抓取和验证的网页数量]
```

### 2. 📌 核心摘要 (Executive Summary)
**字数限制**: 200 字以内

**内容要求**:
- 事件全貌或研究结论的高度概括
- 最重要的 2-3 个发现
- 关键数据或结论
- 不涉及细节，只呈现核心信息

**示例**:
> 2024 年 Q4 全球电动汽车电池成本下降 15%，主要得益于锂资源供应增加和生产工艺优化。宁德时代、比亚迪等头部企业推动能量密度提升至 300Wh/kg 以上。中国占据全球 60% 市场份额，但欧美通过《通胀削减法案》试图重建本土供应链。行业共识：2025 年固态电池将进入小规模量产阶段。

### 3. 🌍 多维视角深度分析

#### 3.1 国际动态
**内容**:
- 国际主流媒体的报道重点
- 不同国家/地区的立场差异
- 全球影响和趋势

**引用格式**: 每段末尾标注 `[1]`, `[2]` 等

**示例**:
> 据 Reuters 报道，全球电动汽车电池市场在 2024 年达到 250GWh 产能，同比增长 35%[1]。BBC 指出欧洲本土电池工厂建设滞后，依赖亚洲供应链的问题仍未解决 [2]。

#### 3.2 国内观察
**内容**:
- 中国权威媒体的报道
- 官方政策解读
- 国内产业现状

**示例**:
> 新华社报道，中国电动汽车电池出口量在 2024 年突破 100GWh，同比增长 40%[3]。财新分析认为，政策补贴退坡后，企业竞争力将更多依赖技术创新 [4]。

#### 3.3 社媒舆情
**3.3.1 B 站深度内容**
- UP 主观点摘要
- 视频播放量、点赞数等数据
- 评论区高赞观点

**示例**:
> B 站 UP 主"科技前沿"发布《2024 电池技术全景解析》视频，播放量 200 万+，点赞 15 万，指出固态电池技术突破是行业最大看点 [5]。

**3.3.2 小红书用户体验**
- 真实用户反馈
- 产品体验分享
- 种草/避雷笔记

**示例**:
> 小红书上大量电动汽车车主分享电池续航实测数据，冬季续航衰减普遍在 30-40% 之间，与厂商宣传存在差距 [6]。

**3.3.3 抖音/短视频趋势**
- 热门话题标签
- 传播数据
- 主要观点

**示例**:
> 抖音#电池技术#话题播放量突破 5 亿，主流观点聚焦在快充技术和安全性提升 [7]。

#### 3.4 学术/数据支撑
**内容**:
- 相关学术论文摘要
- 权威机构统计数据
- 研究结论和预测

**示例**:
> Nature Energy 期刊 2024 年 10 月发表研究，分析全球 50 家电池企业数据，证实能量密度提升与成本下降呈正相关 [8]。MIT 研究报告预测，2025 年固态电池成本将降至$100/kWh 以下 [9]。

### 4. ⚖️ 争议与不同声音

**内容**:
- 不同来源间的明显分歧
- 缺乏共识的领域
- 有待验证的说法

**格式**:
```
**争议点 1**: [简要描述]
- 观点 A: [来源及理由] [引用]
- 观点 B: [来源及理由] [引用]
- 分析：[你的判断或建议进一步验证]
```

**示例**:
> **争议点 1**: 固态电池量产时间表
> - 乐观派：宁德时代称 2025 年小规模量产 [10]
> - 保守派：彭博新能源财经预测 2027 年才能规模化 [11]
> - 分析：技术成熟度和成本控制是主要分歧点，需进一步跟踪


## 报告质量检查清单

### 完整性检查
- [ ] 核心摘要是否在 200 字以内
- [ ] 是否包含国际、国内、社媒、学术四个维度
- [ ] 是否调研了至少 10 个信息源
- [ ] 是否标注了所有引用的来源编号

### 引用规范性检查
- [ ] 每个关键信息是否都有引用标记
- [ ] 参考资料索引是否包含完整 URL
- [ ] 来源名称是否清晰明确
- [ ] 引用编号与正文是否对应

### 深度分析检查
- [ ] 是否进行了交叉验证
- [ ] 是否识别了争议点
- [ ] 是否提供了不同观点的对比
- [ ] 是否分析了事件背后的原因和影响

### 时效性检查
- [ ] 所有信息来源的发布日期是否明确
- [ ] 是否优先使用了最新信息
- [ ] 是否标注了调研执行时间

## 特殊场景调整

### 突发新闻调研
- 重点：时效性、事实核查
- 调整：减少学术支撑，增加多方信源交叉验证
- 时长：1-2 小时内完成

### 社会趋势分析
- 重点：社媒舆情、用户反馈
- 调整：增加 B 站、小红书、抖音内容分析
- 时长：2-3 小时

### 学术研究综述
- 重点：学术文献、数据支撑
- 调整：优先 Google Scholar、arXiv、CNKI
- 时长：3-5 小时

### 市场竞争分析
- 重点：行业报告、公司公告
- 调整：增加财新、36Kr、彭博等商业媒体
- 时长：2-4 小时
