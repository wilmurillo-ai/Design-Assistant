# OpenClaw Multi-Agent Architecture

## Overview
OpenClaw supports multiple isolated agents with independent workspaces.

## Key Concepts
- Main Agent (CEO): Coordinator with restricted tools
- Skill-Worker: Executor with full tool access
- Sub-agents: Spawned for specific tasks

## Configuration
See openclaw.json for agent definitions.
