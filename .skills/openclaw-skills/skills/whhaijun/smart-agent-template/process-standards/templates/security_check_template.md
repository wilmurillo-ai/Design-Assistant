**执行者:** [label]  
**时间:** [YYYY-MM-DD HH:MM]

---

## 安全检查清单（必须全部检查）

```markdown
- [ ] 是否涉及敏感数据？（API key、密码、token）
- [ ] 是否会对外发送？（git push、API 调用、消息发送）
- [ ] 是否不可逆？（删除、覆盖、发布）
- [ ] 是否涉及用户隐私？（姓名、手机、邮箱、地址）
- [ ] 是否需要用户确认？
```

---

## 安全级别判定

根据检查结果判定：

- 🟢 **安全** — 所有检查项都是"否" → 直接执行
- 🟡 **注意** — 1-2 项为"是" → 执行前说明影响
- 🔴 **危险** — 3 项以上为"是" → 必须用户确认

**本次操作级别:** 🟢 / 🟡 / 🔴
