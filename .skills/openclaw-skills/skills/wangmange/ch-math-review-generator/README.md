# 数学复习资料生成器

为初中数学章节自动生成复习指南 HTML 文件的 Skill。

## 功能特点

- 📐 **SVG 图形自动绘制**：坐标系、一次函数图像、几何图形等
- ✅ **内置图形校验**：自动检查直线截距点、对角线交点是否准确
- 🧮 **内置题目校验**：自动检测无解题、矛盾条件、判别式错误
- 📝 **知识要点总结**：章节核心概念与公式整理
- 📖 **选择题 + 填空题 + 解答题**：含完整答案与详细解析

## 使用方式

在任意对话中说：
- "帮我生成一次函数的复习资料"
- "出一份数学练习题"
- "生成几何图形复习指南"
- "帮我复习二次根式"

Skill 会自动加载并按照标准流程生成复习 HTML 文件。

## 校验机制

生成后自动执行双重校验：
1. **SVG 图形校验** - 确保图形坐标与标注点一致
2. **题目逻辑校验** - 确保每道题有正确解

如有错误自动修复，直到零错误才交付。

## 文件结构

```
math-review-generator/
├── SKILL.md                        # 主指令文件
├── references/
│   └── template_guide.md           # HTML模板 + SVG绘制规范
└── scripts/
    ├── validate_svg.py             # SVG图形校验脚本
    └── validate_questions.py        # 题目逻辑校验脚本
```

## 适用场景

- 初中数学各章节复习备考
- 课后练习与错题整理
- 数学知识点梳理与总结
