# 数学章节复习资料生成器

## 工作流程总览

生成一份完整的复习资料需经过以下 6 个阶段：

```
阶段1: 读取模板与章节内容
  ↓
阶段2: 生成 HTML（知识点 + SVG 图形 + 题目）
  ↓
阶段3: SVG 图形校验（运行 validate_svg.py）
  ↓ 有错误 → 返回阶段2修复
阶段4: 题目逻辑校验（运行 validate_questions.py）
  ↓ 有错误 → 返回阶段2修复
阶段5: 确认无错误后交付
阶段6: 写入工作记忆
```

**⚠️ 核心规则：阶段3和阶段4是强制性的，不允许跳过。即使只有1个图形或1道题也必须校验。**

---

## 阶段1: 读取模板

在开始生成前，读取模板参考文档以确保格式一致：

```
references/template_guide.md
```

该文件包含：
- HTML 结构和核心 CSS 样式
- SVG 几何图形的顶点坐标规范（平行四边形、矩形、菱形）
- 一次函数直线的坐标计算公式
- 题目设计规范（各类型题目的数量和难度分布）

---

## 阶段2: 生成 HTML

### 文件命名规范
- 格式: `{topic}_{YYYYMMDD}.html`
- 示例: `parallelogram_review_20260408.html`、`function_chapter_20260410.html`
- 保存路径: 用户当前工作目录

### 内容要求

每份复习资料必须包含以下全部模块：

#### A. 章节基本信息
- 标题：第X章 XXX 复习指南
- 包含版本日期

#### B. 知识点总结
- 每个图形/函数类型对应一个小节
- 包含所有判定定理、性质定理
- 包含记忆口诀（帮助学生快速记忆）
- 包含易错警示（中考最常见的失分点）

#### C. SVG 图形（必须内嵌，不可使用外部图片）
- 几何图形：平行四边形、矩形、菱形需用 SVG 绘制顶点、边、标注
- 函数图象：一次函数四种情况（k>0,b>0 / k>0,b<0 / k<0,b>0 / k<0,b<0）、反比例函数双曲线
- **图形必须精准**：直线截距点位置必须与标注吻合

#### D. 三类函数对比表（如涉及函数章节）
- 正比例函数 / 一次函数 / 反比例函数横向对比
- 包括：定义、k 的意义、b 的意义、图象特征、性质

#### E. 练习题（严格按数量要求）
| 题型 | 数量 | 要求 |
|------|------|------|
| 选择题 | 5道 | 覆盖核心概念、易错陷阱、判定定理 |
| 填空题 | 5道 | 含计算、多结论判断、综合填空 |
| 大题 | 3道 | 基础推导 → 综合计算 → 实际应用/创新变形 |

#### F. 答案与详解
- 每道题都附完整解析
- 解析放在可展开/收起的 answer-box 中
- 解析必须包含解题步骤，不能只给答案

## 阶段3: SVG 图形校验（强制）

### 校验方法
在 HTML 生成后，立即运行图形校验脚本：

**Windows 环境（需要设置 UTF-8 编码）：**
```bash
$env:PYTHONIOENCODING='utf-8'; python {skill_base}/scripts/validate_svg.py {html_file_path} --verbose
```

**Linux/macOS：**
```bash
python {skill_base}/scripts/validate_svg.py {html_file_path} --verbose
```

### 一次函数图形校验重点
- **k>0,b>0**：直线从左下到右上，y轴截距点必须在y轴**上半轴**（SVG y值 < 90）
- **k>0,b<0**：直线从左下到右上，y轴截距点必须在y轴**下半轴**（SVG y值 > 90）
- **k<0,b>0**：直线从左上到右下，y轴截距点必须在y轴**上半轴**
- **k<0,b<0**：直线从左上到右下，y轴截距点必须在y轴**下半轴**
- 截距点误差必须 < 5px

### 几何图形校验重点
- 平行四边形：对角线交于同一点（容差5px）
- 矩形：两组对边分别平行/相等，对角线交于同一点
- 菱形：四条边长度相等（容差5%），对角线交于中心

### 坐标计算公式（SVG 坐标系说明）
- viewBox="0 0 180 180"，原点 O(90, 90)
- **y轴向上为负**：y值越小，图形越高
- 直线 y = kx + b 的 SVG 截距点：cy = 90 - b_svg（b_svg 为数学y轴截距）

### 修复流程
如果校验报错：
1. 读取错误信息中的行号
2. 找到对应 SVG 块
3. 使用公式重新计算正确的直线端点坐标
4. 修改 SVG 代码
5. **再次运行校验**，确认通过
6. 只有校验通过后才进入阶段4

## 阶段4: 题目逻辑校验（强制）

### 校验方法
在 SVG 校验通过后，立即运行题目校验脚本：

**Windows 环境（需要设置 UTF-8 编码）：**
```bash
$env:PYTHONIOENCODING='utf-8'; python {skill_base}/scripts/validate_questions.py {html_file_path} --verbose
```

**Linux/macOS：**
```bash
python {skill_base}/scripts/validate_questions.py {html_file_path} --verbose
```

### 核心校验规则

#### 规则1: 正比例函数参数题必须有解
**常见错误模式（必须避免）：**
```
错误示例：若 y=(k-2)x+1 是正比例函数，则 k=____
错误原因：常数项固定为+1，永远无法满足 b=0

正确模式：若 y=(k-2)x+(k²-4) 是正比例函数，则 k=____
正确原因：常数项含k，可令 k²-4=0 得 k=±2，再排除 k≠2 得 k=-2
```

**校验逻辑：**
- 检测 `y=(k±数字)x+固定数` 形式
- 如果常数项是固定非零数 → 报错：题目无解
- 正确形式应在常数项中包含 k 的表达式

#### 规则2: 二次方程题判别式检查
- 如果题设暗示"有解"但判别式 Δ<0 → 报错
- 如果说"不相等实根"但 Δ=0 → 报错

#### 规则3: 答案与题设一致性
- 正比例函数：答案中 k≠0，且不能是题设中排除的值
- 反比例函数：答案中 k≠0

#### 规则4: 选择题选项互斥性
- 不能有两个完全相同的选项
- 不能有多个正确答案

### 修复流程
如果校验报错：
1. 读取错误类型和对应题目
2. 修改题目条件或答案
3. **再次运行校验**，确认通过
4. 循环此过程直到无错误

## 注意事项

1. **图形优先用内嵌 SVG**：不要依赖外部图片生成服务（可能超时），用 SVG 绘制的几何图形更精准
2. **所有题目必须原创**：不要从网上复制粘贴现成题目，确保每道题都是根据知识点设计的
3. **题目贴近中考风格**：难度适中，有区分度，避免过于简单或超纲
4. **解释要清晰**：大题的解析要写出完整步骤，不跳步，方便学生自学
5. **循环校验直到零错误**：这是 Skill 的核心要求，任何校验失败都必须修复后才能交付
