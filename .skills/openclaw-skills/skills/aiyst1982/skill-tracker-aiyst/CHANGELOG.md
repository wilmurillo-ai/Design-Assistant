## [1.2.0] - 2026-03-18

### ✨ 新增
- 文件锁机制，支持并发安全写入
- 原子写入操作，避免数据损坏
- 配置系统（数据目录、重试次数、日志级别等）
- 统一日志系统（DEBUG/INFO/WARNING/ERROR）
- 执行耗时统计（平均耗时）

### 🔧 改进
- Python 版使用线程锁 + 原子写入
- Node.js 版使用互斥锁 + 原子写入
- 日志格式统一，可配置级别
- 错误处理更完善，异常隔离

### 📖 文档
- 完整的 README.md
- API 参考文档
- 配置说明
- 数据格式说明

## 版本说明

### 语义化版本

- **主版本号 (Major)** - 不兼容的 API 变更
- **次版本号 (Minor)** - 向后兼容的功能性新增
- **修订号 (Patch)** - 向后兼容的问题修正

### 变更类型

- **新增 (Added)** - 新功能
- **改进 (Changed)** - 现有功能的变更
- **弃用 (Deprecated)** - 即将移除的功能
- **移除 (Removed)** - 已移除的功能
- **修复 (Fixed)** - Bug 修复
- **安全 (Security)** - 安全相关的修复
