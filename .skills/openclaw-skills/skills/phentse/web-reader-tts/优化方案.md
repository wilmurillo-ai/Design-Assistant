# Web Reader TTS 优化方案

## 当前问题分析

### 1. 正文提取不准确
**现状**: 直接使用 `document.body.innerText`
```python
content = page.evaluate('() => document.body.innerText')
```

**问题**:
- 包含导航栏、侧边栏、页脚、广告
- 包含"上一篇/下一篇"等导航链接
- 包含版权声明、备案信息
- 噪音占比可达 30-50%

### 2. 过滤规则过于简单
**现状**: 硬编码关键词过滤
```python
skip_keywords = ['上一篇', '下一篇', '版权所有', ...]
```

**问题**:
- 无法适应不同网站结构
- 容易误删正文内容
- 无法处理动态广告

### 3. 缺少结构化信息
**现状**: 只提取 title 和 content

**问题**:
- 没有提取作者、发布时间
- 没有识别文章结构（标题、段落、引用）
- 没有处理图片、代码块等特殊内容

## 优化方案

### 方案 1: 使用 Readability.js（推荐）

**优点**:
- Mozilla 开源，成熟稳定
- 专为正文提取设计
- 准确率高（90%+）
- 支持多语言

**实现**:
```python
from playwright.sync_api import sync_playwright

def extract_with_readability(url: str) -> dict:
    """使用 Readability.js 提取正文"""
    
    with sync_playwright() as p:
        browser = p.chromium.launch()
        page = browser.new_page()
        page.goto(url, wait_until='domcontentloaded')
        
        # 注入 Readability.js
        page.add_script_tag(path='Readability.js')
        
        # 提取正文
        result = page.evaluate('''() => {
            const article = new Readability(document).parse();
            return {
                title: article.title,
                text: article.textContent,
                html: article.content,
                excerpt: article.excerpt,
                byline: article.byline,  // 作者信息
                dir: article.dir
            };
        }''')
        
        browser.close()
        return result
```

**安装**:
```bash
# 下载 Readability.js
npm install @mozilla/readability
# 或使用 CDN
curl -O https://cdn.jsdelivr.net/npm/@mozilla/readability/Readability.js
```

### 方案 2: 使用 Trafilatura（推荐）

**优点**:
- Python 原生，无需 JS
- 专为新闻/博客文章设计
- 提取质量高
- 支持元数据提取

**实现**:
```python
import trafilatura

def extract_with_trafilatura(url: str) -> dict:
    """使用 Trafilatura 提取正文"""
    
    # 下载网页
    downloaded = trafilatura.fetch_url(url)
    
    # 提取正文
    text = trafilatura.extract(
        downloaded,
        include_comments=False,
        include_tables=True,
        favor_precision=True,  # 偏向精确
        favor_recall=False     # 不偏向召回
    )
    
    # 提取元数据
    metadata = trafilatura.extract_metadata(downloaded)
    
    return {
        'title': metadata.title if metadata else None,
        'author': metadata.author if metadata else None,
        'date': metadata.date if metadata else None,
        'text': text,
        'categories': metadata.categories if metadata else None,
        'tags': metadata.tags if metadata else None
    }
```

**安装**:
```bash
pip install trafilatura
```

### 方案 3: 使用 newspaper3k

**优点**:
- 专为新闻文章设计
- 支持 NLP（关键词、摘要）
- 多语言支持

**实现**:
```python
from newspaper import Article

def extract_with_newspaper(url: str) -> dict:
    """使用 newspaper3k 提取正文"""
    
    article = Article(url, language='zh')
    article.download()
    article.parse()
    article.nlp()  # NLP 分析
    
    return {
        'title': article.title,
        'text': article.text,
        'authors': article.authors,
        'publish_date': article.publish_date,
        'top_image': article.top_image,
        'keywords': article.keywords,
        'summary': article.summary
    }
```

**安装**:
```bash
pip install newspaper3k
```

### 方案 4: 混合方案（最稳健）

**策略**: 多引擎提取 + 质量评估

```python
def extract_with_fallback(url: str) -> dict:
    """多引擎提取，自动选择最佳结果"""
    
    results = {}
    
    # 尝试 Trafilatura
    try:
        results['trafilatura'] = extract_with_trafilatura(url)
    except Exception as e:
        print(f"Trafilatura 失败: {e}")
    
    # 尝试 Readability
    try:
        results['readability'] = extract_with_readability(url)
    except Exception as e:
        print(f"Readability 失败: {e}")
    
    # 尝试 newspaper3k
    try:
        results['newspaper'] = extract_with_newspaper(url)
    except Exception as e:
        print(f"newspaper3k 失败: {e}")
    
    # 选择最佳结果
    return select_best_result(results)


def select_best_result(results: dict) -> dict:
    """选择质量最好的提取结果"""
    
    scored = []
    for engine, data in results.items():
        if not data or not data.get('text'):
            continue
        
        score = 0
        text = data['text']
        
        # 评分标准
        score += min(len(text) / 1000, 10)  # 长度得分（最多 10 分）
        score += 5 if data.get('title') else 0  # 有标题 +5
        score += 3 if data.get('author') else 0  # 有作者 +3
        score += 2 if data.get('date') else 0  # 有日期 +2
        
        # 中文内容加分
        chinese_ratio = sum(1 for c in text if '\u4e00' <= c <= '\u9fff') / len(text)
        score += chinese_ratio * 5
        
        scored.append((engine, data, score))
    
    # 返回得分最高的
    if scored:
        best = max(scored, key=lambda x: x[2])
        print(f"✅ 选择 {best[0]} 引擎（得分: {best[2]:.1f}）")
        return best[1]
    
    return None
```
