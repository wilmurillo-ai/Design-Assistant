# 行前检查清单模板

本文件为 ski-assistant 的行前检查知识库，提供标准化 checklist 模板和动态生成规则。为 `modules/pre-departure.md` 生成检查清单时提供数据支撑。

---

## 基础检查清单（所有场景通用）

### 安全必备（不可省略）

| 物品 | 必带 | 可租 | 说明 |
|------|------|------|------|
| 头盔 | 必带 | 否 | CE/ASTM 认证，不可省略 |
| 滑雪专项险 | 必带 | 否 | 普通旅游险不含滑雪，国际需含救援 |
| 滑雪镜 | 必带 | 部分 | 按天气选镜片：晴天偏光，阴天增光 |
| 防晒霜 SPF50+ | 必带 | 否 | 雪地反射紫外线极强 |
| 身份证/护照 | 必带 | 否 | 国内身份证，国际护照+签证 |

### 服装基础

| 物品 | 1-2天 | 3-4天 | 5天+ | 说明 |
|------|-------|-------|------|------|
| 速干内衣 | 2件 | 3件 | 1件/天 | 聚酯纤维或美利奴羊毛，不穿棉质 |
| 滑雪袜 | 2双 | 3双 | 1双/天 | 长筒压缩款，不穿两双 |
| 滑雪外套 | 1件 | 1件 | 1-2件 | 防水 10000mm+ |
| 滑雪裤 | 1条 | 1条 | 1-2条 | 背带裤防雪更好 |
| 保暖中间层 | 1件 | 2件 | 2-3件 | 抓绒或轻薄羽绒 |
| 滑雪手套 | 1双 | 1双 | 1-2双 | 连指比五指更暖 |

## 按目的地气候适配

### 极寒模式（崇礼/东北，低于 -15°C）

**额外必带**：
| 物品 | 数量 | 说明 |
|------|------|------|
| 围脖/面罩 | 1 | 防风款，覆盖口鼻 |
| 暖宝宝 | 1-2包 | 贴手套内侧和鞋里 |
| 加厚保暖层 | 1件 | 厚抓绒或羽绒服 |
| 护手霜 | 1 | 防冻裂 |

**穿衣建议**：速干内衣 + 厚抓绒 + 防水外套 + 围脖 + 暖宝宝

### 寒冷模式（新疆/日本北海道，-15°C ~ -5°C）

**额外必带**：
| 物品 | 数量 | 说明 |
|------|------|------|
| 围脖/面罩 | 1 | 崇礼/东北同等级防风 |
| 暖宝宝 | 1包 | 可选，怕冷的人必带 |
| 防水套 | 1套 | 电子器件防水（粉雪区） |

### 温和模式（北京周边，-5°C ~ 0°C）

**可精简**：
- 无需厚重保暖层
- 暖宝宝可不带
- 围脖可选

### 室内雪场模式（恒温 -5°C ~ -3°C）

**可精简**：
| 可省略 | 仍需 |
|--------|------|
| 厚重保暖层 | 滑雪外套（必须） |
| 围脖/面罩 | 滑雪裤 |
| 暖宝宝 | 速干内衣 |
| 加厚手套 | 滑雪镜 |

## 按用户水平适配

### 初学者（beginner）

**额外必带**：
| 物品 | 说明 |
|------|------|
| 护腕 | 单板初学者必戴，大幅降低手腕骨折风险 |
| 护臀/小乌龟 | 减少摔跤痛苦，提高练习信心 |
| 护膝 | 单板摔膝盖概率高 |

### 中级（intermediate）

**推荐携带**：
| 物品 | 说明 |
|------|------|
| 护背 | 公园跳台或刻滑时保护脊椎 |
| 运动相机 | 记录动作，方便 AI 教练分析 |

### 高级/专家（advanced）

**野雪额外装备**：
| 物品 | 说明 |
|------|------|
| 雪崩搜救仪 | 野雪/非雪道必备 |
| 探杆 + 雪铲 | 搭配搜救仪使用 |
| 对讲机 | 团队通讯 |
| 急救包 | 含止血带、绷带 |

## 按板型适配

### 单板（snowboard）

| 额外必带 | 说明 |
|----------|------|
| 护腕 | 所有水平推荐，初学者必戴 |
| 护臀 | 初学者必带 |
| 护背 | 进阶以上推荐 |

### 双板（ski）

| 额外必带 | 说明 |
|----------|------|
| 雪杖 | 通常与板一起租 |
| 护膝 | 蘑菇/陡坡推荐 |

|----------|------|
| 护背 | 高速刻滑保护脊椎 |

| 推荐携带 | 说明 |
|----------|------|

| 必带 | 说明 |
|------|------|

| 急救包 | 偏远地区自救 |
| 宽板（100mm+） | 粉雪浮力 |

## 国际目的地额外清单

### 文档类

| 物品 | 说明 |
|------|------|
| 护照（有效期 6 月+） | 多数国家要求 |
| 签证/ETA | 提前办理 |
| 国际保险（含救援） | 必须覆盖滑雪和医疗转运 |
| 酒店预订单 | 打印或存手机 |
| 机票行程单 | 打印或存手机 |
| 雪票凭证 | 电子票或确认邮件 |
| 驾照国际认证 | 如计划自驾 |

### 通讯与支付

| 物品 | 说明 |
|------|------|
| 国际漫游/SIM卡 | 确认能接打电话 |
| 信用卡（Visa/MC） | 主流支付方式 |
| 少量当地现金 | 部分雪场只收现金 |
| 转换插头 | 确认目的地插座类型 |

### 装备托运

| 建议 | 说明 |
|------|------|
| 雪具包/板包 | 航空托运专用 |
| 雪鞋单独包 | 避免磨损其他行李 |
| 预留 2-3kg 行李额度 | 雪具通常超重 |

## 天气驱动的动态调整

### 大风（风力 4 级+）

| 额外携带 | 说明 |
|----------|------|
| 全脸面罩 | 防风防寒 |
| 护目镜（双层防雾） | 大风天易起雾 |
| 防风外套 | 外层必须防风 |

### 降雪（中雪/大雪）

| 额外携带 | 说明 |
|----------|------|
| 防水套 | 手机/相机防水 |
| 备用雪镜 | 镜片起雾可换 |
| 防水手套 | 比普通手套更防雪水 |

### 晴天（强紫外线）

| 额外携带 | 说明 |
|----------|------|
| 高倍防晒霜 | 补涂频率加倍 |
| 润唇膏 SPF30+ | 防风防紫外线 |
| 偏光镜片雪镜 | 减少雪面反光 |

### 极寒（低于 -20°C）

| 额外携带 | 说明 |
|----------|------|
| 暖宝宝 2 包+ | 手套、鞋、手机各贴 |
| 手机保暖套 | 低温自动关机 |
| 加厚滑雪袜 | 双层保暖 |
| 巴拉克拉法帽 | 全脸覆盖 |

## 交通方式适配

### 自驾

| 额外携带/确认 | 说明 |
|---------------|------|
| 防滑链 | 冬季山路可能要求 |
| 玻璃水（防冻型） | -20°C 以下用 |
| 满油出发 | 山区加油站少 |
| 停车位置确认 | 提前查雪场停车场 |

### 高铁/大巴

| 额外携带/确认 | 说明 |
|---------------|------|
| 雪具袋/板包 | 保护雪板 |
| 接驳交通确认 | 雪场到车站接驳车 |
| 轻装行李 | 公共交通不便携带过多 |

### 飞机

| 额外携带/确认 | 说明 |
|---------------|------|
| 航空雪具包 | 符合航司尺寸要求 |
| 提前 2 小时到机场 | 托运雪具需额外时间 |
| 雪具托运标签 | 标记"易碎/运动器材" |
