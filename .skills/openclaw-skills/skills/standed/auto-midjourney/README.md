This skill focuses on:

- submitting `imagine` jobs against `https://alpha.midjourney.com`
- Midjourney V8-first prompting
- browser-session result collection and PNG download
- conservative batch generation
- character-sheet and split-view prompt workflows

What is included:

- `SKILL.md` with the main workflow
- Python and Node helper scripts for submit, batch generation, prompt building, and browser-backed result retrieval
- prompt presets and quality profiles
- prompt-craft guidance for product shots, portraits, environments, and character sheets

What is not included:

- `.env`
- browser profile data
- outputs
- local state files

Recommended use cases:

- Midjourney Alpha web automation for one-off user-triggered generation
- batch prompt testing with conservative pacing
- character asset creation, split-view turnarounds, and design sheets
- local PNG downloads for downstream creative workflows
