# Reference Style Notes

这份参考说明用于抽象“超宽横版 PPT”常见的比例和构图信号。

后续重排时，优先学习“比例和气质”，而不是逐字逐框照搬某一份参考稿。

## 画布比例

- 参考画布可以是明显比 16:9 更扁、更长的超宽横版
- 常见目标是保留源稿高度，只继承参考稿的宽高比
- 这样通常比直接压成参考稿绝对尺寸更稳

## 视觉气质

- 深色背景常常更适合承接超宽留白和氛围感
- 左侧更适合作为主标题和信息主区
- 右侧更适合承接光效、留白、背景氛围或辅助图

## 色彩线索

- 正文主色应保证对比度和可读性
- 强调色适合用于标签、短横线、章节标识和底部信息
- 不要因为扩宽画布就随意新增过多颜色

## 构图线索

- 标题通常不写满整行，而是以块状构图出现
- 新增的横向空间更像“呼吸区”，不是“必须塞内容的空白区”
- 底部适合做时间、地点、组织名等收束信息

## 对实际任务的启发

如果源稿是常规 16:9：

- 默认不要压缩高度去贴参考稿的绝对尺寸
- 更稳妥的做法是保留源稿高度，只继承参考稿的超宽比例
- 这样可以避免正文段落在垂直方向被压得太狠
- 文字仍可通过字号、行数、文本框宽度适配，而不是靠拉伸
