# Report Templates

## Standard Report

```
🔒 [skill-name] Security Check

🟢 SAFE / 🟡 NEEDS REVIEW / 🔴 NOT SAFE

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Made by: [Author] ([Type])
Trust: [⭐ rating]
Code: [Yes/No - X scripts]
Issues: [✅ None / ⚠️ X found]

💡 [One sentence explanation]

🎯 [Clear action instruction]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Install? [yes/no]
(Type "why" for more info)
```

## Detailed View (on "why" or "details")

```
🔍 DETAILS

Source: [✅ Trusted / ⚠️ Unknown / ❌ Suspicious]
Code: [✅ Clean / ⚠️ Issues / ❌ Dangerous]

[If issues:]
⚠️ Found:
• [Issue 1 - description]
• [Issue 2 - description]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

💡 EXPLANATION
[2-3 sentences max]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🎯 RECOMMENDED ACTION
[Clear instruction]
```

## Special Cases

### Skill Already Installed

```
ℹ️ This skill is already installed.
Last checked: Never | Risk: Unknown

Options:
• [keep] - Keep, mark as checked
• [remove] - Uninstall now
• [check] - Run full security check
```

### Very High Risk (score > 50)

```
🔴 NOT SAFE

⚠️ STRONGLY NOT RECOMMENDED

Still install? [type 'install-at-own-risk' / no]
```

### Review Required

```
⚠️ Manual review recommended.

Found [X] concern(s):
• [List specific findings]

Options:
• [yes] - Install anyway (not recommended)
• [no] - Cancel installation
• [details] - Show complete analysis
```
