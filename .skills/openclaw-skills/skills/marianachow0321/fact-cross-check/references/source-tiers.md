# Source Credibility Tiers

| Tier | Confidence | Examples |
|------|-----------|----------|
| T1 | 95-100% | Official/primary sources, public records, peer-reviewed |
| T2 | 75-90% | Reputable media, technical analysis tools, analyst reports |
| T3 | 50-70% | Curated lists, blogs, newsletters |
| T4 | <50% | Forums, unsourced articles, content >2yr old |

## Confidence Calc

- 2+ T1 agree → 95-100%
- 1 T1 + 2+ T2 → 85-95%
- 3+ T2 agree → 80-90%
- Only T2-3 → 70-80%, mark Unverified
- T3-4 only → 50-70%, mark Uncertain
