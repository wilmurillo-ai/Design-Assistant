# Research Patterns

- **Single claim:** find claim → extract assertions → verify each independently → synthesize.
- **No obvious source:** launch multiple independent queries → find consensus → weight by tier → flag conflicts.
- **Suspicious claim:** generate alternative hypotheses → test each → pick best-supported.
- **May be outdated:** search current → search historical → build timeline → newer data wins.
- **Sources disagree:** higher tier wins → newer date wins → more specific wins → direct verify via `web_fetch`.
- **Multiple entities:** batch queries → rank by evidence quality → report individual confidence.

## Efficiency

- Official sites first, then technical tools, then articles.
- 2+ T1 agree → stop. Don't over-verify.
- Simple: 1 query. Standard: 2-3. Complex: 5+.
