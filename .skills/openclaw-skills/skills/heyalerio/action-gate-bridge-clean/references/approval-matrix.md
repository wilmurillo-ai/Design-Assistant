# Approval Matrix

Green:
- read
- classify
- summarize
- alert the user
- maintain ledgers

Yellow:
- prepare only
- draft a reply
- propose a booking choice
- prepare a decision summary

Red:
- send a message
- submit a form
- confirm a reservation
- connect a risky new integration
- spend money

Review model:
1. policy
2. scope
3. approval
4. reversibility

If any answer is weak or unclear, escalate.
