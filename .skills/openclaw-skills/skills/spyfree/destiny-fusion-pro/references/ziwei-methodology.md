# 紫微斗数解读方法（大众版）

本技能采用“由静到动”的四层解读结构：

1. 本命定调：先看命宫、身宫与主星组合。
2. 结构联动：看三方四正（命宫与对宫、三合宫位）。
3. 年度触发：看流年宫位与四化（禄权科忌）如何落点。
4. 运限验证：用大运阶段 + 流年节奏做现实落地建议。

## 使用原则
- 先结构、后断语：先看宫位关系，再给结论。
- 先趋势、后建议：先说走势，再给可执行方案。
- 先边界、后决策：明确出生时刻/时区/经度误差风险。
