# 紫微斗数二次解读框架

本技能输出的目标不是给出固定生活建议，而是提供可复核的解读骨架：

1. 命宫定调：看命宫主星、身宫落点、五行局。
2. 结构联动：看三方四正（本宫、三方、对宫）之间是否同向。
3. 运限触发：看大限宫与流年宫，再看四化落点。
4. 交叉验证：将运限结论回归本命结构，避免单点断语。

## 使用原则
- 先事实：先确认排盘口径和宫位事实。
- 后经验：再引入传统经验，不做单句绝断。
- 可复盘：边界时辰、经度、时区必须可重算。
