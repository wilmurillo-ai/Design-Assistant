# Agent Script Assets

These assets are examples, starter scaffolds, reusable snippets, and supporting metadata examples for Agent Script work.

## Directory Structure
- `agents/` — complete example agents
- `components/` — reusable snippets
- `patterns/` — validated patterns
- `metadata/` — supporting metadata examples

## Default usage
- Prefer direct `.agent` authoring in the target bundle.
- Use these files as examples or starting scaffolds when helpful.
- Do not treat this directory as a required template system.
