Track trip preparation progress with a Kanban-style board.

---

## Example Tasks by Category

### ✈️ Transport
- [ ] Research flight options
- [ ] Book flights
- [ ] Reserve seats
- [ ] Check baggage allowance
- [ ] Print/download boarding passes

### 🏨 Accommodation
- [ ] Research neighborhoods
- [ ] Compare hotels/Airbnbs
- [ ] Book accommodation
- [ ] Confirm reservation
- [ ] Save check-in instructions

### 📋 Documents
- [ ] Check passport validity
- [ ] Research visa requirements
- [ ] Apply for visa
- [ ] Get travel insurance
- [ ] Make copies of documents

### 🎒 Preparation
- [ ] Create packing list
- [ ] Check weather forecast
- [ ] Notify bank of travel
- [ ] Set up phone roaming
- [ ] Arrange pet/house sitting

### 🎯 Activities
- [ ] Research attractions
- [ ] Book tours/activities
- [ ] Make restaurant reservations
- [ ] Download offline maps
- [ ] Plan daily itinerary

---

## How to Use

1. **Copy** all tasks from above that apply to your trip
2. **Paste** them into the Kanban table
3. **Move** items between columns as you progress
4. **Update** regularly (weekly recommended)

**Pro Tip:** Mark items as "Blocked" when waiting for something (e.g., "Book hotel" blocked until "Confirm dates" is done).
