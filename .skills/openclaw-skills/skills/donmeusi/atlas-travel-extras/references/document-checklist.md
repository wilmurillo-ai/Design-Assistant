# Dokumenten-Checkliste - Vor jeder Reise

## 📄 Reisedokumente

### Pass / Personalausweis
- [ ] **Gültigkeit geprüft** → Mindestens 6 Monate über Reiseende hinaus?
- [ ] **Leere Seiten** → Mindestens 2-4 freie Seiten?
- [ ] **Kopie gemacht** → Digitale + physische Kopie vorhanden?
- [ ] **Foto aktuell** → Ähnelt das Passfoto noch dem aktuellen Aussehen?

### Visa (falls erforderlich)
- [ ] **Visa-Typ** → Tourist, Business, Transit?
- [ ] **Gültigkeitszeitraum** → Deckt die gesamte Reise ab?
- [ ] **Ein-/Ausreise** → Anzahl Einreisen korrekt (Single/Multiple)?
- [ ] **Max Aufenthalt** → Nicht überschreiten!
- [ ] **eVisa ausgedruckt** → Digital reicht oft nicht

### Reiseversicherung
- [ ] **Krankenversicherung** → Ausreichend Deckungssumme (min. 1 Mio €)?
- [ ] **Rücktrittsversicherung** → Bei Stornierung abgedeckt?
- [ ] **Gepäckversicherung** → Bei Verlust/Theft abgedeckt?
- [ ] **Versicherungskarte** → Mitgenommen?
- [ ] **Notfallnummer** → Hinterlegt bei Notfallkontakt?

---

## ✈️ Buchungsbestätigungen

### Transport
- [ ] **Flug** → Hin- UND Rückflug bestätigt?
- [ ] **Zug** → Tickets gekauft?
- [ ] **Bus** → Reservierung vorhanden?
- [ ] **Mietwagen** → Voucher + Versicherung?
- [ ] **Fähre/Schiff** → Reservierung bestätigt?

### Unterkunft
- [ ] **Hotel/Airbnb** → Buchungsbestätigung gespeichert?
- [ ] **Adresse** → Notiert für Taxi/Anfahrt?
- [ ] **Check-in Zeiten** → Passt zum Ankunftszeitpunkt?
- [ ] **Kontaktdaten** → Telefonnummer vor Ort?

---

## 💳 Finanzen

### Zahlungsmittel
- [ ] **Kreditkarte 1** → Gültig, limit ausreichend?
- [ ] **Kreditkarte 2** → Backup-Karte aktiv?
- [ ] **EC-Karte** → Gültig, für Notfälle?
- [ ] **Bargeld (Fremdwährung)** → Vor Ort vorhanden?
- [ ] **Bargeld (Heimat)** → Notfall-Cash?

### Bank informiert
- [ ] **Reisebank** → Nutzung im Ausland freigeschaltet?
- [ ] **PIN** → PINs überprüft, nicht nur notiert?
- [ ] **Kontakt** → Bank-Notfallnummer gespeichert?

---

## 🏥 Gesundheit

### Impfungen
- [ ] **Standardimpfungen** → Tetanus, Hepatitis A/B aktuell?
- [ ] **Reiseimpfungen** → Gelbfieber, Typhus, Cholera, etc.?
- [ ] **Impfpass** → Mitgenommen?
- [ ] **Malariaprophylaxe** → Falls nötig, Medikamente eingepackt?

### Medikamente
- [ ] **Rezeptpflichtige Medikamente** → Ausreichend + Rezept?
- [ ] **Reiseapotheke** → Schmerzmittel, Durchfallmittel, etc.?
- [ ] **Allergiemittel** → Falls nötig?
- [ ] **Brillen/Kontaktlinsen** → Ersatz vorhanden?

---

## 📱 Kommunikation

### Telefon
- [ ] **Roaming** → Aktiviert oder lokale SIM?
- [ ] **Ladegerät** → Adapter für Ziel-Land?
- [ ] **Powerbank** → Für unterwegs?

### Online-Zugang
- [ ] **Wichtige Accounts** → Passwörter erreichbar (Passwort-Manager)?
- [ ] **Cloud-Speicher** → Dokumente synchronisiert?
- [ ] **E-Mail** → Automatische Antwort eingerichtet (falls nötig)?

---

## 🏠 Zuhause

### Sicherheit
- [ ] **Fenster geschlossen** → Alle verschlossen?
- [ ] **Türen verriegelt** → Auch Kellertür, Balkon?
- [ ] **Alarmanlage** → Aktiviert?
- [ ] **Wasserhähne** → Alle geschlossen, Hauptventil?
- [ ] **Elektrogeräte** → Ausgesteckt (außer Kühlschrank)?
- [ ] **Heizung** → Runtergestellt oder aus?
- [ ] **Müll** → Rausgebracht?

### Nachbarschaft (optional)
- [ ] **Nachbarn informiert** → Bescheid gegeben?
- [ ] **Zeitungen/Post** → Jemand nimmt an?
- [ ] **Pflanzen** → Jemand gießt?
- [ ] **Haustiere** → Versorgung geregelt?

### Notfallkontakte hinterlegt
- [ ] **Familie/Freunde** → Wissen Bescheid?
- [ ] **Hotel** → Kontaktdaten geteilt?
- [ ] **Versicherung** → Notfallnummer bekannt?
- [ ] **Botschaft** → Standort registriert (falls Krisengebiet)?

---

## 📋 Last-Minute-Check (24h vor Abreise)

- [ ] Wetter geprüft → Packliste angepasst?
- [ ] Check-in online → Sitzplatz gewählt?
- [ ] Gepäckgewicht → Nicht überschritten?
- [ ] Handgepäck → Flüssigkeiten < 100ml?
- [ ] Boarding-Pass → Ausgedruckt oder App geladen?
- [ ] Ankunftszeit → Transfer/Unterkunft informiert?
