## 💡 Quick Actions

### If Passport Lost/Stolen:
1. Report to local police → get report
2. Contact embassy/consulate
3. Provide: photo ID, flight details, police report
4. Apply for emergency travel document

### If Wallet Lost/Stolen:
1. Cancel credit cards (numbers above)
2. File police report
3. Contact embassy (if ID stolen)
4. Request emergency funds from contact

### If Medical Emergency:
1. Call local emergency number
2. Show this card to medical staff
3. Contact insurance (number above)
4. Notify emergency contact
