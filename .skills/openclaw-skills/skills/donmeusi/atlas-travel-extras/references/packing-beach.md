# Beach Packing List

Beach-specific items to add to your main packing list.

## 👙 Swim & Beach
- [ ] Swimsuits (2-3)
- [ ] Beach cover-up / sarong
- [ ] Flip-flops / water shoes
- [ ] Beach towel (quick-dry)
- [ ] Beach bag / waterproof tote

## 🤿 Water Activities
- [ ] Snorkel gear (mask, snorkel, fins)
- [ ] Waterproof phone case
- [ ] Underwater camera / GoPro
- [ ] Floaties / inflatable raft
- [ ] Beach toys (ball, frisbee)

## ☀️ Sun Protection
- [ ] Sunscreen (SPF 30+)
- [ ] After-sun lotion / aloe vera
- [ ] Wide-brimmed hat
- [ ] UV-protection sunglasses
- [ ] Beach umbrella / sun shelter
- [ ] Rash guard / UV shirt

## 🏖️ Beach Comfort
- [ ] Beach mat / blanket
- [ ] Portable beach chair
- [ ] Cooler bag for drinks
- [ ] Waterproof bluetooth speaker
- [ ] Book / e-reader

## 💧 Practical
- [ ] Wet bag for damp clothes
- [ ] Baby powder (removes sand)
- [ ] Insulated water bottle
- [ ] Ziplock bags (for valuables)
- [ ] Portable phone charger

## 🩹 First Aid
- [ ] Jellyfish sting relief
- [ ] Aloe vera gel
- [ ] Blister plasters (for flip-flops)
- [ ] Rehydration salts
- [ ] Seasickness tablets

## 📱 Tech Protection
- [ ] Waterproof pouch for phone
- [ ] Dry bag for electronics
- [ ] Anti-fog spray (for snorkel mask)
