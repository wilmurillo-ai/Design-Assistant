## 1. 🍳 美食教程 —— 「3分钟搞定减脂早餐」

**平台：** 抖音 | **时长：** 60秒

```
【0-3秒 · 钩子】
口播：「早上起不来又想减脂？这个早餐3分钟搞定，好吃到哭！」
画面：成品特写，香气飘飘

【3-10秒 · 食材展示】
口播：「准备鸡蛋2个、燕麦30g、香蕉半根、蓝莓适量」
画面：食材依次入镜，每个1秒

【10-35秒 · 制作过程】
口播：「鸡蛋打散，加入燕麦搅拌，小火煎至两面金黄，摆上水果」
画面：快剪操作过程，BGM节奏配合动作

【35-50秒 · 成品展示】
口播：「看看这个颜值！热量只有300大卡，蛋白质20g+」
画面：成品多角度展示，叉子切开慢镜头

【50-60秒 · CTA】
口播：「减脂早餐合集持续更新，关注我不迷路！评论区告诉我你明天想吃什么」
画面：指向关注按钮
```

```
优点3条 + 缺点2条，文字弹出动效
「值不值得买？看你的需求——」

【75-90秒 · CTA】
「你觉得千元平板够用吗？弹幕告诉我！一键三连支持一下～」
```

## 通用模板总结

### 万能脚本公式
```
前3秒 = 钩子（抓注意力）
3-10秒 = 问题/痛点（引共鸣）
10-45秒 = 核心内容（给价值）
45-55秒 = 总结/案例（增信任）
55-60秒 = CTA（促行动）
```

### 各品类要点速查

| 品类 | 核心要素 | 钩子类型 |
|------|----------|----------|
| 美食 | 成品先行 + 步骤快剪 | 成品诱惑 |
| 科技 | 数据对比 + 实测画面 | 悬念对比 |
| 职场 | 痛点共鸣 + 解决方案 | 反面案例 |
| 家居 | 使用前后 + 价格标注 | 排行揭秘 |
| 健身 | 动作示范 + 时间标注 | 效果承诺 |
| 旅行 | 画面美感 + 费用明细 | 省钱诱惑 |
| 知识 | 反常识 + 数据支撑 | 颠覆认知 |
| 读书 | 金句 + 故事 | 情感共鸣 |
