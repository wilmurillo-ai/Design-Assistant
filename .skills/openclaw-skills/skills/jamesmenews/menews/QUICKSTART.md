## 🚀 超快速开始（2 步）

### 步骤 1：安装 Skill

```bash
# 使用 ClawHub CLI
clawhub install menews
```

向 OpenClaw 提问：

```
给我看看今天的市场摘要
```

## 💡 使用示例

### 获取市场摘要（推荐）

```
"给我看看今天的市场摘要"
"最新的市场资讯汇总"
"现在市场上有什么热点？"
"Show me today's market digest"
```

**一键获取完整市场全景**：当前热门、X热议、市场热点、预测事件、热议token

### 查询快讯

```
"最新的加密货币快讯有哪些？"
"今天有什么重要新闻？"
"Show me crypto flash news"
```

### 查询聚合热点

```
"最近有什么热点话题？"
"热门代币有哪些？"
"X 平台上在讨论什么？"
"Show me aggregated digest"
```

### 查询预测市场

```
"Polymarket 上有什么热门预测？"
"预测市场有什么新事件？"
"Show me prediction markets"
```

## ❓ 常见问题

### Q: 需要配置什么吗？

A: 不需要！完全公开 API，安装即用。

### Q: 需要付费吗？

A: 完全免费，无需注册。

### Q: 有使用限制吗？

A: 限流 20 次/分钟（每个 IP），正常使用完全够用。

### Q: Skill 没有响应？

A: 请检查：
1. Skill 是否已安装：`clawhub list`
2. 使用明确的触发词，如"最新快讯"
3. 检查网络连接
