## 📝 格式说明

**Daily 笔记自动生成规则：**

1. **每个会话独立小节**
   - 包含会话标题
   - 包含状态标注（✅活跃 / ❌删除 / 🔄重置）
   - 包含话题标签（#关键词）
   - **新增：** 时间标注支持更新标记（如"10:30-11:00（14:00 更新）"）

2. **内容要求**
   - 摘要 200-300 字
   - 关键决策/发现 1-3 个
   - 时间范围标注
   - **新增：** 活跃 session 支持"🔄 更新"区块，记录增量内容

3. **标签规范**
   - 状态标签：✅（活跃）、❌（删除）、🔄（重置）
   - 话题标签：#关键词 格式
   - 多个标签用空格分隔

4. **增量更新格式（新增）**
   - **🔄 更新时间：** 显示具体更新时间
   - **更新摘要：** 简述新增讨论内容
   - **新增决策：** 列出新的决策/发现
