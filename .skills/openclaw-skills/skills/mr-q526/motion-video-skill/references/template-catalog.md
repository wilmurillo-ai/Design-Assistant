# Template Catalog

当前 MVP 提供 4 套模板。

模板是主视觉配置。大多数情况下，theme 直接跟模板默认值走，不需要一开始把模板和 theme 分成两个问题来问。

## AI 引导规则

- 用户没有说风格时，先给 2 到 4 个模板选项，并明确推荐 1 个
- 不要一上来同时追问“模板 + theme + 配色 + 字体”
- theme 默认跟模板走；只有用户明确要改品牌色、明暗、氛围或 UI 质感时，才单独展开 theme
- 如果用户说“你来定”，可以直接选推荐模板和其默认 theme，但要在回复里说明你采用了哪个组合

## `signal-stage`

- 主题：`signal-ink`
- 定位：科技、工具、AI、产品讲解
- UI：深色玻璃舞台、发光背景、圆角信息卡
- 推荐问法：如果这是产品介绍、工具介绍或 AI 能力说明，优先推荐它

## `editorial-board`

- 主题：`sunset-board`
- 定位：教程、案例复盘、课程讲解
- UI：暖色编辑台、纸面线条、较扁的标签和卡片
- 推荐问法：如果这是教程拆解、课程总结或案例复盘，优先推荐它

## `creator-pop`

- 主题：`creator-flare`
- 定位：社媒亮点视频、短视频字幕讲解、节奏型内容
- UI：高饱和撞色、斜切舞台、明显的封面感
- 推荐问法：如果用户要更强的短视频感、更像字幕卡点或更抓眼的封面感，优先推荐它

## `mono-frame`

- 主题：`mono-slate`
- 定位：访谈摘要、理性观点、极简风表达
- UI：黑白框线、低装饰、收敛的卡片风格
- 推荐问法：如果内容偏观点表达、访谈摘要、理性分析，优先推荐它

## Theme 说明

### `signal-ink`

- 气质：深色科技、橙蓝对比、偏产品发布感
- 适合：AI 产品、工具介绍、技术功能讲解

### `sunset-board`

- 气质：暖色纸面、编辑台、课程笔记感
- 适合：教程、案例拆解、教学内容

### `creator-flare`

- 气质：高饱和、霓虹撞色、社媒封面感
- 适合：短视频、亮点摘录、节奏型讲解

### `mono-slate`

- 气质：黑白克制、理性、框架化表达
- 适合：访谈摘要、观点视频、极简信息表达

## 默认选择建议

- 不确定时优先使用 `signal-stage`
- 想更教程感，用 `editorial-board`
- 想更短视频感，用 `creator-pop`
- 想更克制极简，用 `mono-frame`

## 推荐问法

```text
这条视频我建议先从视觉模板里选一个。我先给你 3 个最合适的方向：signal-stage（产品/AI 介绍，推荐）、editorial-board（教程拆解）、creator-pop（短视频感更强）。你要我用哪个？
```

如果用户已经接受模板，但还想微调气质，再补一句：

```text
模板我先不换。如果你想只改配色和氛围，我可以在这个模板下再给你几个 theme 方向。
```
