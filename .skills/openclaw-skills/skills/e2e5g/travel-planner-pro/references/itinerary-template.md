# 行程卡片模板

## 概览
本模板提供标准化的行程卡片格式，用于每日行程的呈现。

## 每日行程卡片格式

### [日期] 第X天：[主题]

- **活动1**：[景点/活动名称]
  - 时间：09:00-11:00
  - 亮点：[核心看点]
  - 门票：[价格，如免费]
  - 交通：[如何到达]
  - 备注：[注意事项、最佳拍照点等]

- **活动2**：[景点/活动名称]
  - 时间：11:00-12:00
  - ...

#### 🍽️ 午餐（12:00-13:30）
- **推荐**：[餐厅名称或区域]
- **特色**：[菜品特色]
- **预算**：[人均价格]
- **位置**：[与下午行程的衔接]

#### 🌞 下午（13:30-18:00）
- **活动3**：[景点/活动名称]
  - 时间：13:30-16:00
  - ...

#### 🌆 晚间（18:00-22:00）
- **晚餐**：[推荐餐厅]
- **活动**：[夜间活动或自由活动]
- **住宿**：[酒店/民宿信息]

#### 💡 今日贴士
- 天气建议：[根据天气提示]
- 省钱小招：[今日省钱建议]

## 示例：北京第1天行程卡片

### 2024年4月15日 第1天：故宫胡同之旅

#### 🌅 上午（09:00-12:00）
- **活动1**：故宫博物院
  - 时间：09:00-11:30
  - 亮点：明清皇宫建筑、珍贵文物收藏
  - 门票：60元/人（提前网上预约）
  - 交通：地铁1号线天安门东站B口出，步行10分钟
  - 备注：
    - 周一闭馆，请避开
    - 建议租用电子导览（20元）
    - 最佳拍照点：太和殿广场、御花园

#### 🍽️ 午餐（12:00-13:30）
- **推荐**：王府井小吃街或故宫附近老北京菜馆
- **特色**：北京烤鸭、炸酱面、豆汁焦圈
- **预算**：80-100元/人
- **位置**：故宫东门步行15分钟

#### 🌞 下午（13:30-18:00）
- **活动2**：景山公园
  - 时间：13:30-15:00
  - 亮点：俯瞰故宫全景、登高望远
  - 门票：2元/人
  - 交通：故宫北门步行5分钟
  - 备注：山顶万春亭是拍摄故宫全景的最佳位置

- **活动3**：南锣鼓巷胡同游
  - 时间：15:30-18:00
  - 亮点：老北京胡同文化、特色小店
  - 门票：免费
  - 交通：公交至南锣鼓巷站
  - 备注：推荐下午茶、文创店

#### 🌆 晚间（18:00-22:00）
- **晚餐**：后海附近烤肉季或涮羊肉
- **活动**：什刹海夜游、酒吧休闲
- **住宿**：前门或王府井附近酒店（方便第二天行程）

#### 💡 今日贴士
- 天气建议：春季早晚温差大，建议携带薄外套
- 省钱小招：故宫学生票半价，带好学生证；景山公园老年人免费
- 避坑提醒：故宫内餐饮价格较高，建议自带水和小食
- 儿童友好：故宫有儿童语音导览，景山公园适合儿童放风筝

## 注意事项
- 根据实际情况灵活调整时间安排
- 预留充足的交通和休息时间
- 注意景点的开放时间和特殊规定
- 保持行程的弹性，应对突发天气或交通状况
- 确保每日行程强度适中，避免过度疲劳
