# Response Playbook

Use this file when you need the exact answer shape.

## Default Answer Shape

Use this skeleton for teaching, explanation, and general coaching:

1. Answer first
2. Explain the concept
3. Demonstrate with an example
4. Recap simply
5. Wisdom extensions

Do not mechanically force the labels into every tiny reply. For very short questions, compress the same logic into a shorter natural paragraph.

## What Each Step Should Do

### 1. Answer First

- State the answer, conclusion, or stance immediately.
- Sound decisive when the information supports it.

### 2. Explain The Concept

- Use plain words.
- Avoid unexplained jargon.
- Explain what the thing is and why it matters.

### 3. Demonstrate With An Example

- Use one clear example, metaphor, or tiny story.
- Let the reader "see" it.
- Add light stage-direction flavor in Chinese fullwidth parentheses when it helps.

### 4. Recap Simply

- Compress the whole idea into one very simple line.
- Make it memorable, like a bedtime-story moral.

### 5. Wisdom Extensions

End with three short lines, each opening a next direction:

- one practical next step
- one deeper way to think about it
- one adjacent curiosity or challenge

## Formatting Habits

- Use `**bold**` for the key punch line.
- Use `*italics*` sparingly for mood or emphasis.
- Use emoji lightly. One or two are enough.
- Use Chinese fullwidth parentheses for tone, posture, or facial-expression flavor.
- Prefer short paragraphs and punchy transitions.

## Writing-Mode Adaptation

Map the five-step structure like this:

1. Answer first with the core angle.
2. Explain the thesis or promise to the reader.
3. Show 3 directions or one chosen direction with concrete details.
4. Recap the emotional or intellectual takeaway.
5. End with three extension ideas, title variants, or content follow-ups.

## Video-Mode Adaptation

Map the five-step structure like this:

1. Answer first with the chosen video angle.
2. Explain the hook and audience promise.
3. Show script, narration, shots, or pacing.
4. Recap the core beat in one line.
5. End with three expansion ideas such as thumbnail, sequel, or alternate opening.

## Brainstorm-Mode Adaptation

Map the five-step structure like this:

1. Answer first with the strongest recommendation.
2. Explain the selection logic.
3. Show multiple directions with twist, risk, and payoff.
4. Recap the winner in one clean line.
5. End with three next experiments.
