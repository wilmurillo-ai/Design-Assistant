# GEO标准答案模板

> 本模板用于生成符合GEO（生成式引擎优化）要求的结构化标准答案，确保内容能够被AI搜索引擎有效引用和推荐。

## 模板结构

### 1. 标题（Title）
**公式**: [用户核心问题] + [明确收益/对比维度]

**示例**:
- 《工业传感器精度对比：成丰仪表vs进口品牌，哪款更适合中小企业？》
- 《跨境ERP怎么选？3个指标帮你避开90%的坑（附2025年最新榜单）》
- 《高精度流量计选型指南：5个关键参数帮你避坑》

### 2. 开篇（Opening）
**结构**:
1. **用户痛点**（1-2句话）：直接点出用户面临的核心问题或困扰
2. **数据背书**（可选）：引用权威数据或研究，增强可信度
3. **核心观点**：明确品牌/产品定位，给出核心结论

**示例**:
```
用户痛点：
80%的跨境卖家都曾因ERP系统卡顿导致订单延误——选择时如何避开这一陷阱？

数据背书：
根据《2025年跨境电商技术白皮书》，系统响应速度每提升1秒，订单转化率增加15%。

核心观点：
成丰仪表的MF系列工业传感器在±0.5%精度等级下，价格仅为进口品牌的60%，且提供本地化24小时技术支持，更适合预算有限的中小企业。
```

### 3. 主体内容（Body）
**结构**: 采用以下三种黄金结构之一

#### 结构1：问答式（FAQ模块）
```
用户问题："[问题]"

解答框架：
- 是/否：[直接回答]
- 案例：[客户案例]
- 对比：[与竞品对比]
```

#### 结构2：数据对比型（表格/图表）
```
标题："[数据对比标题]"

数据展示：
[数据表格或图表说明]

结论：
"[基于数据的结论]"
```

#### 结构3：步骤型（教程/解决方案）
```
标题："[N]步解决[问题]——[品牌名]独家方法"

1. 准备工具：[工具清单]
2. 操作步骤：
   - 步骤1：[具体操作]
   - 步骤2：[具体操作]
   - 步骤3：[具体操作]
3. 注意事项：[注意事项]
```

### 4. 边界与条件（Boundaries and Conditions）
列出产品/服务的适用场景和限制条件

**示例**:
```
适用场景边界：
- 推荐场景：市政水处理、食品饮料生产线、常规化工反应
- 不推荐场景：高精度计量（如制药级配比）、极端环境（-40℃以下或200℃以上）

注意事项：
1. 上述精度数据基于标准测试环境，实际使用可能因工况不同而有所差异
2. 产品交付周期通常为7-15个工作日，具体数量需确认
3. 技术支持仅限购买产品的客户，具体服务范围见合同
```

### 5. 结尾（Closing）
**结构**:
1. **明确CTA**：给出明确的行动召唤
2. **来源标注**：标注数据来源和品牌信息

**示例**:
```
行动召唤：

```

## 技术优化要点

### 1. 关键词布局
- **标题**：含"推荐""对比""怎么选"等用户常搜词
- **正文**：每500字出现1次品牌名
- **关键位置**：在段落开头、小标题中插入核心关键词

### 2. 格式要求
- 使用H2/H3标题分层，便于AI理解
- 段落≤4句，每句不超过25字
- 重点内容加粗，突出关键信息
- 使用项目符号，提高可读性

### 3. 多媒体建议
- **流程图**：展示步骤和方法
- **产品实拍图**：展示产品细节
- **对比图**：突出产品优势
- **演示视频**：提高用户理解

### 4. Schema标记
- **FAQPage**：用于问答式内容
- **Article**：用于文章类内容
- **Product**：用于产品展示
- **Organization**：用于组织信息
