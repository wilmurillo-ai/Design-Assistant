# 交付模板

需要稳定输出时，优先套下面的模板，而不是每次自由发挥。

## 决策备忘录

适合：

- 是否继续推进某方案
- 在多个选项中做选择

模板：

1. 问题定义
2. 关键约束
3. 选用专家与原因
4. 主要判断
5. 分歧与取舍
6. 推荐路径
7. 下一步动作
8. 风险与预警信号

## 方案对比

适合：

- A/B 路径选择
- 方案优先级排序

模板：

1. 比较维度
2. 方案 A
3. 方案 B
4. 方案 C
5. 哪类条件下各自更优
6. 最终建议

## 红队报告

适合：

- 高投入、高不确定性方案
- 需要故意找漏洞的场景

模板：

1. 核心假设
2. 最脆弱的点
3. 失败路径
4. 早期预警信号
5. 保护栏
6. 是否继续推进

## 专家会诊

适合：

- 多维度复杂问题
- 需要共识与分歧并呈现

模板：

1. 会诊目标
2. 专家名单与分工
3. 每位专家关键意见
4. 共识
5. 分歧
6. 综合判断
7. 行动建议

## 评审与改稿

适合：

- 已有方案、文章、脚本、文案、产品草图

模板：

1. 当前版本目标
2. 主要问题
3. 缺失项
4. 优先修改项
5. 修改方向
6. 下一版应达到什么标准

## 教学式解释

适合：

- 要把复杂问题讲给非专业用户

模板：

1. 一句话解释
2. 用大白话展开
3. 最容易误解的点
4. 一个例子
5. 如何自测是否真的理解
