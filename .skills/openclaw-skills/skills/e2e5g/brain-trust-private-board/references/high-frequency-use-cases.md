# 高频用途直达

优先处理这些最常见、最容易反复使用的场景。

## 1. 产品方向判断

- 推荐模式：`boardroom`
- 推荐专家：乔布斯 + 诺曼 + 克里斯坦森
- 推荐交付：决策备忘录
- 推荐风格：私董会风格
- 最好输入：目标用户、要解决的任务、已有方案、限制条件

## 2. 功能取舍

- 推荐模式：`compare`
- 推荐专家：乔布斯 + 拉姆斯
- 推荐交付：保留 / 删除 / 延后清单
- 最好输入：功能列表、核心目标、资源限制

## 3. 商业战略评估

- 推荐模式：`boardroom`
- 推荐专家：德鲁克 + 波特 + 巴菲特
- 推荐交付：战略 memo
- 推荐风格：私董会风格
- 最好输入：目标、竞争格局、资源、时间窗口

## 4. 风险和红队审查

- 推荐模式：`red-team`
- 推荐专家：塔勒布 + 卡尼曼 + 戴明
- 推荐交付：风险清单 + 预警信号 + 保护栏
- 最好输入：方案、关键假设、投入成本、失败代价

## 5. 现有方案评审

- 推荐模式：`critique`
- 推荐专家：按问题类型选 1-2 位，再补 1 位风险型专家
- 推荐交付：问题清单 + 优先修改项
- 最好输入：当前版本、目标、想解决的问题、你最担心什么

## 6. 技能 / Prompt / Workflow 设计评审

- 推荐模式：`boardroom` 或 `critique` 后接 `synthesis`
- 推荐专家：费曼 + 芒格 + 德鲁克
- 推荐交付：评审意见 + 重构方向
- 推荐风格：私董会风格 或 评审意见风格
- 最好输入：SKILL.md 或提示词、目标场景、触发方式、你觉得卡的地方

## 7. 文案 / 品牌表达优化

- 推荐模式：`compare`
- 推荐专家：奥格威 + 原研哉 + 费曼
- 推荐交付：表达方向对比 + 改写建议
- 最好输入：原文、受众、想要的气质、想促成的动作

## 8. 复杂问题讲清楚

- 推荐模式：`single`
- 推荐专家：费曼
- 可补专家：苏格拉底
- 推荐交付：教学式解释
- 最好输入：主题、受众水平、最难懂的点

## 9. 概念澄清与论证

- 推荐模式：`compare` 或 `panel`
- 推荐专家：苏格拉底 + 亚里士多德 + 康德
- 推荐交付：概念澄清稿 / 论证结构图
- 最好输入：概念、争议点、你当前的判断

## 10. 竞争与破局

- 推荐模式：`boardroom`
- 推荐专家：孙子 + 波特 + 格鲁夫
- 推荐交付：破局路径 + 节奏建议
- 推荐风格：私董会风格
- 最好输入：竞争对手、资源差距、窗口期、不能输的点

## 11. Workflow 故障排查

- 推荐模式：`boardroom` 或 `critique`
- 推荐专家：费曼 + 梅多斯 + 芒格
- 推荐交付：故障点 + 根因 + 重构建议
- 推荐风格：私董会风格 或 评审意见风格
- 最好输入：现有流程、出错位置、你观察到的异常表现
