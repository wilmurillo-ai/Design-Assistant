# 30-45秒口播稿模板

## 模板结构

```
【钩子 3-5s】用数字/反差抛问题
{引人注目的开场白，使用数字、反差或悬念}

【痛点 8-12s】引用热评/读者吐槽（1-2条）
{引用真实用户评论或普遍痛点，建立情感共鸣}

【验证 10-15s】列清单/对比/时间线+可复核来源
{提供清晰的对比、数据或事实，增强可信度}

【行动 8-10s】收藏/评论提问点/下期承诺
{明确的行动指令，引导互动和持续关注}

【合规】如含原创/转载/合作/AIGC→按平台标识
{必要的合规说明和标注}
```

## 使用指南

### 钩子（3-5秒）设计要点

#### 类型1：数字冲击
```
实测发现5个被忽略的真相！
3分钟搞定早餐，营养不输大厨！
90%的人都不知道的技巧！
```

#### 类型2：反差对比
```
苹果说是升级，实测却发现...
大家都说好，但我发现3个问题！
花1000和花10000的区别其实不大？
```

#### 类型3：悬念提问
```
你真的买对了吗？
为什么我建议你千万不要买？
这个方法为什么没人告诉你？
```

### 痛点（8-12秒）设计要点

#### 引用真实评论
```
很多粉丝问：
"到底值不值？"
"为什么我用就没用？"
"有没有更简单的方法？"
```

#### 展示普遍困境
```
你是不是也遇到过：
- 买了贵但不实用
- 学了很多但做不好
- 想省钱却花了更多
```

### 验证（10-15秒）设计要点

#### 清单式呈现
```
我对比了10款产品，发现3个关键点：
第一，XX功能其实很重要；
第二，XX参数不是越高越好；
第三，XX品牌性价比最高！
```

#### 对比式验证
```
测试了30天，结果很意外：
传统方法：成本1000元，效果60分
这个方法：成本200元，效果90分
```

### 行动（8-10秒）设计要点

#### 互动引导
```
想看完整数据？评论区扣"对比"
下期测XX，记得关注！
有问题的评论区见！
```

#### 承诺价值
```
明天分享：如何避开这些坑
关注我不迷路，带你少花冤枉钱！
点个赞，让更多人看到
```

## 时长控制

### 各部分时长分配

| 部分 | 最短 | 最长 | 建议 |
|------|------|------|------|
| 钩子 | 3s | 5s | 4s |
| 痛点 | 8s | 12s | 10s |
| 验证 | 10s | 15s | 12s |
| 行动 | 8s | 10s | 9s |
| 合规 | 2s | 3s | 2s |
| **总计** | **31s** | **45s** | **37s** |

### 语速参考

- 正常语速：约200-240字/分钟
- 快速语速：约260-300字/分钟
- 慢速语速：约160-200字/分钟

**计算示例**：
- 目标时长：37秒
- 语速选择：正常语速（220字/分钟）
- 字数计算：(37/60) × 220 ≈ 136字

## 制作注意事项

### 录制前准备

1. **内容审查**
   - 检查事实准确性
   - 确认数据来源可靠
   - 核实敏感词合规性

2. **语速测试**
   - 先通读一遍，计时
   - 调整字数到目标范围
   - 标注重点和停顿

3. **情绪把控**
   - 开头要吸引注意
   - 中间要传递价值
   - 结尾要引导行动

### 录制要点

1. **声音质量**
   - 使用麦克风，减少环境噪音
   - 保持稳定的音量
   - 避免呼吸声和嘴型音

2. **节奏控制**
   - 关键信息适当停顿
   - 避免语速过快
   - 保持自然流畅

3. **表情配合**
   - 视频录制时注意表情
   - 手势辅助表达
   - 眼神交流

### 后期处理

1. **音频优化**
   - 降低背景噪音
   - 调整音量平衡
   - 添加必要的BGM

2. **字幕制作**
   - 准确同步字幕
   - 重点信息突出显示
   - 字体清晰易读

3. **画面配合**
   - 配合内容切换画面
   - 添加关键信息字幕
   - 保持画面节奏一致

## 合规标注

### 必须标注的情况

| 情况 | 标注方式 | 示例 |
|------|----------|------|
| 转载内容 | 转载请注明来源 | 转载请注明来源 |
| 原创内容 | 无需标注 | （无） |
| 商业合作 | 本内容含商业推广 | 本内容含XX品牌推广 |
| AIGC生成 | 部分内容由AI生成 | 部分内容由AI生成 |
| 测评数据 | 实测数据/仅供参考 | 实测数据真实有效 |
| 医疗健康 | 内容仅供参考，请遵医嘱 | 内容仅供参考，请遵医嘱 |

### 标注位置

- 文末总结：放在口播稿最后2-3秒
- 字幕显示：同步显示在屏幕下方
- 文案说明：在视频简介或文案中详细说明

## 常见错误

### 内容错误

1. **钩子不够吸引**
   - ❌ "今天给大家分享一个产品"
   - ✅ "90%的人都买错了！"

2. **痛点不准确**
   - ❌ "你可能需要这个产品"
   - ✅ "你是不是也遇到过花冤枉钱的情况？"

3. **验证不充分**
   - ❌ "我觉得很好用"
   - ✅ "实测30天，对比10款产品"

### 结构错误

1. **时长超限**
   - ❌ 总时长超过45秒
   - ✅ 严格控制在30-45秒内

2. **结构混乱**
   - ❌ 钩子→验证→痛点→行动
   - ✅ 钩子→痛点→验证→行动

3. **行动不明确**
   - ❌ "大家觉得怎么样？"
   - ✅ "评论区扣'对比'，发你详细数据"

### 合规错误

1. **未标注来源**
   - ❌ 直接引用数据不标注
   - ✅ 数据来源：XX机构

2. **商业合作不说明**
   - ❌ 暗示推广是客观测评
   - ✅ 本内容含XX品牌推广

3. **医疗建议不当**
   - ❌ "这个方法能治好XX病"
   - ✅ "仅供参考，具体请遵医嘱"
