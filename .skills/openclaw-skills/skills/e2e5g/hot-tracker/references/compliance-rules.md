## 各平台规则差异

### 微信公众号

| 规则类型 | 要求 | 违规后果 |
|----------|------|----------|
| 原创声明 | 原创内容需声明 | 违规1次封号7天 |
| 转载规则 | 转载需授权并注明来源 | 违规删除+处罚 |
| 诱导分享 | 禁止诱导转发 | 违规删除 |
| 敏感话题 | 禁止政治敏感内容 | 永久封号 |
| 商业合作 | 需明确标注 | 违规封号 |

**合规要点**：
- 原创内容务必声明原创
- 转载内容必须获得授权并注明来源
- 商业推广内容需添加"广告"标识
- 避免使用诱导性语言

| 导流行为 | 禁止外链/二维码 | 违规限流 |

## 合规检测维度

### 检测维度与权重

| 维度 | 符号 | 权重 | 说明 |
|------|------|------|------|
| 原创合规 | P_original | 0.30 | 原创/转载标注 |
| 商业合规 | P_commercial | 0.25 | 广告/合作标注 |
| AIGC合规 | P_aigc | 0.25 | AI生成内容标注 |
| 版权合规 | P_copyright | 0.20 | 素材使用合规 |

### 原创合规检测

**检测规则**：

```python
if is_original == True:
    if is_repost == True:
        # 矛盾检测
        risk = "原创与转载标识矛盾"
    else:
        # 正常原创
        risk = None
elif is_repost == True:
    if source_attributed == False:
        # 违规转载
        risk = "转载未标注来源"
    else:
        # 合规转载
        risk = None
else:
    # 未明确标识
    risk = "未明确原创/转载状态"
```

**合规标准**：
- 原创内容：is_original = true
- 合规转载：is_repost = true AND source_attributed = true
- 违规转载：is_repost = true AND source_attributed = false

### 商业合规检测

**检测规则**：

```python
if has_ad == True or is_ad == True:
    if ad_marked == False:
        # 违规未标注
        risk = "商业合作未标注"
    else:
        # 合规
        risk = None
else:
    # 非商业内容
    risk = None
```

**常见违规类型**：
- 未标注"广告"或"推广"
- 软文未明确说明
- 隐含商业推广但未标注
- 优惠券/链接未说明利益相关

### AIGC合规检测

**检测规则**：

```python
if is_aigc == True:
    if aigc_marked == False:
        # 违规未标注
        risk = "AIGC内容未标注"
    else:
        # 合规
        risk = None
```

**平台要求**：
- 抖音：需标注"内容由AI生成"
- 小红书：建议标注"AI辅助创作"
- 知乎：需明确说明AI参与程度
- B站：建议标注AI使用情况

### 版权合规检测

**检测素材类型**：
1. 图片/照片：需确认拍摄者授权
2. 音乐/音效：需确认版权授权
3. 字体：需确认商业授权
4. 视频素材：需确认原创或授权
5. 表情包：需确认版权

**检测方法**：
- 检查素材来源是否标注
- 检查是否有版权声明
- 检查是否在授权范围内使用

## 风险识别与预警

### 风险等级划分

| 等级 | 风险描述 | 影响 | 处理方式 |
|------|----------|------|----------|
| 高风险 | 违规严重 | 可能封号 | 立即修改或删除 |
| 中风险 | 违规较重 | 限流降权 | 建议修改 |
| 低风险 | 轻微违规 | 影响较小 | 注意避免 |

### 常见风险场景

**高风险场景**：
1. 政治敏感内容
2. 虚假信息传播
3. 侵权严重
4. 诱导违法行为

**中风险场景**：
1. 商业合作未标注
2. 原创声明不准确
3. 轻微虚假宣传
4. 诱导关注/分享

**低风险场景**：
1. 格式不规范
2. 标识不明显
3. 建议性标注未添加

### 预警机制

**触发条件**：
- 内容包含敏感词
- 使用绝对化用语
- 涉及医疗/金融
- 包含外链/二维码

**预警内容**：
- 风险类型提示
- 违规原因说明
- 修改建议

## 合规创作建议

### 原创内容建议

1. **明确原创声明**
   - 务必勾选原创声明
   - 确保内容为原创
   - 引用他人内容需注明

2. **转载规范**
   - 获取授权后再转载
   - 转载时注明来源
   - 保留原文链接

### 商业合作建议

1. **明确标注**
   - 添加"广告"或"推广"标识
   - 在开头或结尾说明
   - 视频可在字幕中说明

2. **表述规范**
   - 避免绝对化用语
   - 数据来源需可靠
   - 观点需客观公正

### AIGC使用建议

1. **明确标识**
   - 声明AI参与程度
   - 标注"AI辅助创作"
   - 说明具体使用环节

2. **内容审核**
   - AI生成内容需人工审核
   - 确保内容准确性
   - 避免传播错误信息

### 平台特定建议

|------|----------|------|
