## 概述

时效性是热榜分析的核心维度之一。本框架定义了时效性评估的完整体系，包括时效性评分、爆发速度分析、趋势预判和生命周期预测。

## 时效性评分体系

### 综合时效性评分（T）

```
T = T_realtime × W_realtime + T_velocity × W_velocity + T_decay × W_decay
```

**权重配置**：

| 权重 | 符号 | 默认值 | 说明 |
|------|------|--------|------|
| 发布时间权重 | W_realtime | 0.40 | 发布时间越近，得分越高 |
| 爆发速度权重 | W_velocity | 0.35 | 增长速度越快，得分越高 |
| 生命周期权重 | W_decay | 0.25 | 剩余生命周期越长，得分越高 |

### T_realtime：发布时间得分

```
T_realtime = max(0, 1 - hours_since_publish / 24)
```

| 发布时间 | T_realtime | 说明 |
|----------|------------|------|
| 0-2h | 0.92-1.0 | 最佳发布窗口 |
| 2-6h | 0.75-0.92 | 较好发布窗口 |
| 6-12h | 0.5-0.75 | 一般发布窗口 |
| 12-24h | 0-0.5 | 较晚发布窗口 |
| >24h | 0 | 不建议使用 |

### T_velocity：爆发速度得分

```
T_velocity = min(1.0, velocity_score)

其中：
velocity_score = (delta_1h / 1000) × 0.5 + 
                 (delta_3h / 5000) × 0.3 + 
                 (delta_6h / 10000) × 0.2
```

| 爆发类型 | T_velocity | 特征 |
|----------|------------|------|
| 爆发期 | 0.8-1.0 | delta_1h > 5000 |
| 快速上升 | 0.5-0.8 | delta_1h > 1000 |
| 稳步上升 | 0.2-0.5 | delta_1h > 100 |
| 平稳期 | < 0.2 | 增量较少 |

### T_decay：生命周期得分

```
T_decay = remaining_life / typical_lifecycle
```

**话题类型与典型生命周期**：

| 话题类型 | 典型生命周期 | 适用场景 |
|----------|-------------|----------|
| breaking（突发） | 6小时 | 突发事件、重大新闻 |
| news（新闻） | 12小时 | 时事新闻、政策解读 |
| entertainment（娱乐） | 24小时 | 娱乐八卦、明星动态 |
| tech（科技） | 48小时 | 科技新品、技术突破 |
| general（一般） | 24小时 | 通用话题 |

## 爆发速度分析

### 爆发速度指数（V）

```
V = (delta_1h / 1h) × (current_heat / baseline) ^ 0.5

参数说明：
- delta_1h: 近1小时增量
- current_heat: 当前热度
- baseline: 基础热度（账号历史平均）
```

### 爆发等级划分

| 等级 | V值范围 | 描述 | 建议动作 |
|------|---------|------|----------|
| S级 | ≥ 10 | 超级爆发 | 紧急蹭热度，1小时内产出 |
| A级 | 5-10 | 快速爆发 | 尽快跟进，2-4小时内产出 |
| B级 | 2-5 | 稳步上升 | 可以跟进，6-12小时内产出 |
| C级 | < 2 | 平稳期 | 需差异化竞争 |

### 爆发拐点识别

**拐点特征**：
1. delta_1h > delta_3h 平均值 × 1.5
2. 增速呈加速趋势
3. 评论情感偏向正面

**拐点预警信号**：
- 某条内容的 delta_1h 进入平台前5%
- 话题开始出现在多个平台
- 评论区出现大量"转发"请求

## 趋势预判模型

### 基于历史模式的预测

```
预测热度(t) = 初始热度 × e^(-t/τ) × (1 + 增长因子)^t
```

**参数说明**：
- τ：衰减时间常数（基于话题类型）
- 增长因子：基于当前趋势的调整系数

### 预测时间窗口

| 窗口 | 预测精度 | 适用场景 |
|------|----------|----------|
| 0-2h | 高（>90%） | 短期蹭热度 |
| 2-4h | 中（70-90%） | 内容产出规划 |
| 4-8h | 中低（50-70%） | 中期选题规划 |
| 8-24h | 低（<50%） | 长期话题储备 |

### 预测置信度计算

```
confidence = 时效性 × 趋势一致性 × 数据完整度

- 时效性：当前时间距离发布的时间
- 趋势一致性：历史相似话题的发展模式匹配度
- 数据完整度：可用的历史数据量
```

## 生命周期预测

### 典型生命周期曲线

```
热度(t) = H_max × e^(-t/τ) × (1 - e^(-t/τ0))^n
```

**参数说明**：
- H_max：预估峰值热度
- τ：衰减时间常数
- τ0：上升期时间常数
- n：曲线形态参数

### 生命周期阶段

| 阶段 | 时间范围 | 特征 | 建议 |
|------|----------|------|------|
| 潜伏期 | 0-2h | 热度低，增长慢 | 识别期 |
| 爆发期 | 2-6h | 快速增长 | 最佳蹭热度期 |
| 成熟期 | 6-12h | 达到峰值 | 跟进期 |
| 衰退期 | 12-24h | 逐渐下降 | 收尾期 |
| 长尾期 | 24h+ | 低热度持续 | 差异化期 |

### 蹭热度时间窗口

| 窗口类型 | 进入时间 | 退出时间 | 优先级 |
|----------|----------|----------|--------|
| 黄金窗口 | 爆发初期 | 成熟期前 | 最高 |
| 白银窗口 | 成熟期 | 衰退初期 | 高 |
| 青铜窗口 | 衰退期 | 长尾期前 | 中 |
| 黄铜窗口 | 长尾期 | 话题结束 | 低 |

## 应用场景

### 内容产出时间规划

1. **黄金窗口（2-6h）**：
   - 优先产出短视频/图文
   - 强调时效性和独家角度
   - 快速跟进热点

2. **白银窗口（6-12h）**：
   - 产出深度分析内容
   - 强调差异化视角
   - 整合多平台信息

3. **青铜窗口（12-24h）**：
   - 产出总结性内容
   - 强调全面性和系统性
   - 提供独特见解

### 选题优先级排序

基于时效性的选题优先级：

```
优先级 = H_global × T × 平台匹配度

- H_global：综合热度
- T：时效性评分
- 平台匹配度：内容形式与平台调性的匹配程度
```

### 预警机制

**触发条件**：
- V（爆发速度）进入前5%
- T（时效性评分）> 0.8
- 多平台同时出现相似话题

**预警动作**：
1. 推送即时通知
2. 建议快速产出
3. 提供创作方向建议
