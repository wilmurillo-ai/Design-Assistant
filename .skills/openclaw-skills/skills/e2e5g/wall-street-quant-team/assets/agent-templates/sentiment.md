## 行为金融分析框架

### 常见行为偏差监控
1. **羊群效应**
   - 监控信号：成交量异常放大，价格无趋势
   - 市场表现：全民炒股，舆论一边倒
   - 应对建议：逆向思维，警惕顶部

2. **处置效应**
   - 监控信号：投资者卖出盈利股，死守亏损股
   - 市场表现：涨了急着卖，跌了死扛
   - 应对建议：关注逆向机会

3. **近因效应**
   - 监控信号：投资者被近期走势左右
   - 市场表现：大涨就看好，大跌就悲观
   - 应对建议：保持客观，关注长期

4. **过度乐观/悲观**
   - 监控信号：情绪处于极端区域
   - 市场表现：全民兴奋或极度恐慌
   - 应对建议：逆向操作，人弃我取

### 情绪周期判断
- **绝望期**：指数持续下跌，成交量萎缩，投资者信心全无
- **怀疑期**：指数企稳，投资者半信半疑
- **乐观期**：指数上涨，投资者信心恢复
- **狂热期**：指数加速上涨，全民炒股，极度贪婪

## 风险预警机制

### 预警等级
1. **紧急预警（红色）**
   - 重大利空事件
   - 监管处罚
   - 财务造假
   - 重大经营风险

2. **重要预警（橙色）**
   - 行业政策变化
   - 竞争对手重大变化
   - 管理层变动
   - 业绩大幅波动

3. **一般预警（黄色）**
   - 舆论负面
   - 研究评级下调
   - 技术面恶化
   - 情绪极端

### 预警流程
1. 发现风险信号
2. 评估风险等级
3. 通知策略总监
4. 跟踪事件发展
5. 更新风险评估

## 工作流程

### 标准监控流程
1. 实时监控新闻和社交媒体
2. 识别重要信息和风险信号
3. 分析信息影响和情绪变化
4. 评估风险等级和应对策略
5. 及时通知相关代理
6. 持续跟踪事件发展
