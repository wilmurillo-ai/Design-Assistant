### 各步骤详细说明

#### 【第1步】任务分解（策略总监）
```
执行代理：策略总监
执行时间：收到指令后即时
输出产物：任务分配方案

策略总监操作：
1. 理解主人需求（标的？深度？紧急程度？）
2. 判断任务类型（个股分析/板块研究/组合诊断/紧急风控）
3. 制定分析计划
4. 分配任务给专业代理
5. 设置优先级和时间要求
```

#### 【第2步】专业分析（4代理并行 + 量化支持）
```
执行代理：基本面分析师、技术分析师、行业研究员、舆情监控
执行时间：可并行执行，总耗时取决于最长单项
输出产物：各代理分析报告

并行任务说明：
• 四个代理同时工作，互不等待
• 各代理独立分析，发挥专业优势
• 量化研究员提供模型验证支持

基本面分析师任务：
• 财务报表深度解读
• 估值模型构建
• 商业模式分析
• 财务风险识别

技术分析师任务：
• 趋势判断（第一优先级）
• K线形态识别
• 支撑压力位分析
• 量价关系分析

行业研究员任务：
• 产业链上下游分析
• 竞争格局研判
• 政策影响评估
• 景气度判断

舆情监控任务：
• 市场情绪监测
• 行为偏差检测
• 舆情数据追踪
• 异常信号预警
```

#### 【第3步】量化验证（量化研究员）
```
执行代理：量化研究员
执行时间：接收各代理数据后执行
输出产物：量化分析报告

AI量化任务：
• 机器学习模型验证（XGBoost/LightGBM）
• 深度学习时序预测（LSTM/Transformer）
• 强化学习策略模拟（DQN/PPO）
• 因子IC/IR分析
• 风险量化评估
```

#### 【第4步】汇总整合（策略总监）
```
执行代理：策略总监
执行时间：接收所有分析报告后
输出产物：综合分析报告

整合任务：
• 汇总各代理核心观点
• 整合量化模型信号
• 识别一致点和分歧点
• 矛盾点权衡分析
• 趋势权重最高裁决
```

#### 【第5步】投资决策（策略总监）
```
执行代理：策略总监
执行时间：汇总完成后
输出产物：投资决策建议

决策任务：
• 顺势而为判断方向
• 综合评估机会与风险
• 制定投资方案
• 明确仓位建议
• 不给具体买卖指令
```

#### 【第6步】风控审查（风控专员）
```
执行代理：风控专员
执行时间：决策后即时
输出产物：风控意见

审查任务：
• 风险敞口评估
• 止损方案制定
• 合规性检查
• 极端情况预案
• 【关键】风险过高可否决
```

#### 【第7步】汇报主人（策略总监）
```
执行代理：策略总监
执行时间：风控审查后
输出产物：最终汇报

汇报任务：
• 清晰呈现六维分析结论
• 说明决策逻辑和依据
• 提示关注要点和风险因素
• 展示免责声明
• 【自动】执行智囊邀请判断
```

## 协作机制详解

### 信息传递机制

```
代理A ──分析报告──▶ 策略总监 ──汇总整合──▶ 综合报告
                              │
                              ▼

                      └───────────────┘
                              │
                              ▼
                      ┌───────────────┐
                      │   决策确认    │
                      │   策略总监    │
                      └───────────────┘
                              │
                              ▼
                      ┌───────────────┐
                      │   执行指令    │
                      │   风控专员    │
                      └───────────────┘
                              │
                              ▼
                      ┌───────────────┐
                      │   汇报主人    │
                      │   策略总监    │
                      └───────────────┘
```

### 冲突解决机制

| 冲突类型 | 解决方式 | 裁决者 |
|---------|---------|-------|
| 分析结论冲突 | 以趋势为准 | 策略总监 |
| 量化信号冲突 | 综合多模型 | 量化研究员 |
| 风控否决 | 主人最终决定 | 主人 |
| 代理观点分歧 | 策略总监裁决 | 策略总监 |
| 紧急情况 | 风控专员决定 | 风控专员 |

### 质量控制机制

1. **数据校验**：各代理需校验数据来源和时效性
2. **交叉验证**：量化研究员验证各代理分析结论
3. **逻辑审核**：策略总监审核整体逻辑一致性
4. **风险复核**：风控专员复核风险评估完整性
5. **主人确认**：重大决策需主人最终确认
