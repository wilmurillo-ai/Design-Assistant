## 功能简介

本技能提供华尔街级别的AI多代理量化投资团队协作系统，包含：

- **7个专业AI代理**：策略总监、基本面分析师、技术分析师、行业研究员、舆情监控、风控专员、量化研究员
- **完整投资分析流程**：宏观分析→行业研判→公司研究→技术量化→风险评估→综合研判
- **AI量化分析能力**：机器学习(XGBoost/LightGBM)、深度学习(LSTM/Transformer)、强化学习(DQN/PPO)
- **行为金融分析**：市场情绪、行为偏差检测

## 触发场景

- 创建AI投资团队
- 配置多代理协作
- 股票/板块深度分析
- 量化策略开发
- 趋势方向判断
- 风险评估与控制

## 核心投资理念

- **顺势而为**：趋势是第一准则，永远不要与趋势为敌
- **行为金融**：精准分析投资人群行为
- **AI量化**：AI时代最专业最具深度的量化分析能力
- **风险控制**：投资是极少数人赚钱的活动，必须专业、精准、严格

## 投资分析规范

- 每次分析强制展示免责声明
- 禁止直接荐股、目标价位、收益承诺
- 允许板块逻辑分析、风险提示、趋势判断
