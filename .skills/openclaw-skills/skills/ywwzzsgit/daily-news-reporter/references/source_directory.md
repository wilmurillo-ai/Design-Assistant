## 搜索技巧

### 精准搜索
```
# 官方声明
site:federalreserve.gov [topic]
site:un.org [topic]

# 特定日期
"March 20, 2026" [topic]
2026年3月20日 [主题]

# 专业数据
[commodity] price "March 2026"
[stock] "real time"
```

### 交叉验证策略
1. 同一事件至少2个独立来源
2. 包含不同地区视角
3. 区分事实陈述和观点分析
4. 注意信息时效性

## 可靠性评估标准

| 等级 | 标准 | 使用建议 |
|------|------|---------|
| 5/5 | 官方发布、一手数据 | 可直接引用 |
| 4/5 | 专业媒体、数据服务商 | 可信，可引用 |
| 3/5 | 主流媒体、有审核 | 基本可信，建议标注 |
| 2/5 | 行业媒体、分析文章 | 需交叉验证 |
| 1/5 | 自媒体、未验证 | 仅作参考，需多源确认 |
