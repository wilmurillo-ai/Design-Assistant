# 📈 缠论+基本面+估值 综合股票分析技能

> 输入一个股票代码，输出一份涵盖缠论技术面、基本面、估值面的专业投研报告，并自动生成排版精美的 PDF 文件。

---

## 功能概述

本技能通过四大维度对股票进行全面分析：

| 维度 | 内容 |
|------|------|
| 🎯 缠论技术分析 | 中枢、笔、线段、背驰、一/二/三类买卖点识别 |
| 📊 多指标共振 | MACD背驰 + 均线 + RSI/KDJ/CCI + 布林带 + 量价 14维度评分 |
| 📈 基本面分析 | 盈利能力、成长性、偿债能力、运营效率、杜邦分解 |
| 💰 估值分析 | PE/PB/EV/EBITDA历史百分位 + PEG + DCF三情景估值 |

**输出物**：
- ✅ Markdown 格式完整投研报告（含五维评分、操作建议、止损线）
- ✅ 专业排版 PDF 报告（A4，含封面摘要、页眉页脚、深蓝配色）

## 前置依赖

### 必需技能

安装并配置 `finance-data-retrieval` 技能：

```bash
clawhub install finance-data-retrieval
```

然后配置 Tushare Token：
```bash
export TUSHARE_TOKEN="your_tushare_token_here"
```

> 💡 获取 Tushare Token：前往 [tushare.pro](https://tushare.pro) 注册账号，在个人中心获取 Token。

### Python 依赖（运行时自动安装）

- `reportlab`：用于 PDF 生成，首次运行时自动安装

## 报告结构

生成的投研报告包含以下章节（**第一章为总结摘要，可直接查看结论**）：

```
📋 总体结论与投资建议（摘要）
  ├── 综合评级
  ├── 一句话结论
  ├── 五维评分（技术/基本面/估值/资金/情绪）
  ├── 后续操作建议（短/中/长期）
  └── 主要风险提示

一、公司概况
二、技术分析（缠论 + 多指标共振）
  ├── 2.1 缠论结构（日线+周线）
  ├── 2.2 趋势追踪指标（均线/MACD/一目云/ADX）
  ├── 2.3 动量振荡指标（RSI/KDJ/CCI/WR）
  ├── 2.4 波动率与空间（BOLL/ATR/乖离率）
  ├── 2.5 量价与资金（OBV/主力净流入）
  ├── 2.6 多指标共振评分（XX/23分）
  └── 2.7 关键价格位（支撑/压力/止损）
三、基本面分析
  ├── 3.1 盈利能力（毛利率/净利率/ROE）
  ├── 3.2 成长能力（3年CAGR/最新增速）
  └── 3.3 财务健康度（负债率/现金流/红旗信号）
四、估值分析
  ├── 4.1 相对估值（PE/PB/EV + 历史百分位）
  ├── 4.2 PEG分析
  └── 4.3 简化DCF估值（乐观/基准/悲观）
五、综合结论与操作建议
六、数据来源
```

## 注意事项

- ⚠️ **数据真实性**：所有行情和财务数据通过官方接口实时获取，不使用训练数据中的历史记忆值
- ⚠️ **缠论局限性**：缠论分析基于程序化规则，在复杂走势中存在主观性，结论仅供参考
- ⚠️ **免责声明**：本技能输出内容**不构成投资建议**，投资有风险，决策需谨慎
