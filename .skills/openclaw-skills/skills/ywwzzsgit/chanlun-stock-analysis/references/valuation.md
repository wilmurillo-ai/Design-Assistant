# 估值分析参考手册

## 一、估值方法体系

估值是将公司未来价值折现到当前的过程，应综合使用多种方法进行交叉验证。

## 二、相对估值法（市场对比）

### 1. 市盈率（P/E）
```
P/E = 当前市值 / 年化净利润（或最新EPS × 总股本）
静态P/E = 市价 / 上一年度EPS
动态P/E = 市价 / 预测EPS（前向P/E）
滚动P/E（TTM） = 市价 / 过去12个月EPS
```

**解读标准**（A股参考，行业差异大）：
| 行业类型 | 合理P/E区间 |
|----------|-----------|
| 消费白马（茅台、海天等） | 25~45× |
| 银行 | 5~10× |
| 地产 | 5~15× |
| 科技/互联网 | 30~80× |
| 医药生物 | 25~60× |
| 工业制造 | 15~30× |

**PEG = P/E / 净利润增速**
- PEG < 1：低估信号
- PEG = 1：合理估值
- PEG > 2：高估信号

### 2. 市净率（P/B）
```
P/B = 当前市值 / 净资产（账面价值）
```
- 适用：银行、保险、地产等资产密集型行业
- P/B < 1：可能破净，资产被低估（或净资产质量差）
- P/B > 3：溢价明显，需要ROE支撑

**关键公式**：合理P/B ≈ ROE / 要求回报率（科斯尔公式）

### 3. 市销率（P/S）
```
P/S = 当前市值 / 年营业收入
```
- 适用：亏损企业、早期成长公司、SaaS企业
- A股制造业参考：P/S < 1 较低，1~3 合理，>5 偏高
- SaaS/互联网参考：P/S 5~20 正常范围

### 4. EV/EBITDA
```
EV = 市值 + 有息负债 - 货币资金
EBITDA = EBIT + 折旧 + 摊销
EV/EBITDA = EV / EBITDA
```
- 适用：资本密集型行业（航空、电信、能源）
- 合理区间：8~15×（因行业差异较大）
- 剔除折旧摊销影响，适合跨国比较

### 5. 同行对比（Comps）
- 获取同行业5~10家可比公司的P/E、P/B、EV/EBITDA
- 计算行业中位数和均值
- 目标公司溢价或折价的合理解释

## 三、绝对估值法（DCF）

### 自由现金流折现模型（DCF）

**基本框架**：
```
内在价值 = Σ(FCFt / (1+WACC)^t) + 终值 / (1+WACC)^n

FCF（自由现金流） = EBIT × (1-税率) + 折旧 - 资本支出 - 营运资金增加
WACC（加权平均资本成本） = Ke × E/(D+E) + Kd × (1-t) × D/(D+E)

其中：
Ke = 股权成本 = 无风险利率 + Beta × 市场风险溢价
    A股参考：无风险利率≈2.5%~3.0%（10年国债），市场风险溢价≈6%
Kd = 债务成本（公司贷款利率）
```

**终值计算**：
```
永续增长法：TV = FCFn × (1+g) / (WACC - g)  （g通常取2%~4%）
退出乘数法：TV = EBITDAn × EV/EBITDA中位数
```

**敏感性分析**（必须做）：
- 对 WACC（±1%）和 永续增长率g（±1%）做二维表
- 展示估值区间而非单点估值

### DCF适用条件
- 现金流可预测（成熟型企业）
- 有正向FCF或可见的FCF转正路径
- 不适用：初创公司、高度不确定性行业

## 四、股息贴现模型（DDM）

适用于高股息、现金奶牛型公司（银行、公用事业、高股息消费）：
```
内在价值 = D1 / (Ke - g)

其中：
D1 = 下一年预期股息
Ke = 股权成本
g = 股息永续增长率
```

## 五、估值综合评分框架

### 多方法交叉验证
| 方法 | 权重 | 适用场景 |
|------|------|---------|
| P/E (TTM) | 30% | 盈利稳定型公司 |
| EV/EBITDA | 20% | 资本密集型公司 |
| DCF | 35% | 可预测现金流企业 |
| P/B | 15% | 金融/资产型企业 |

**综合估值 = Σ(各方法估值 × 对应权重)**

### 安全边际
- 买入区间：内在价值的60%~80%（安全边际20%~40%）
- 合理区间：内在价值的80%~110%
- 高估区间：超过内在价值110%

## 六、估值数据需求

```
# 实时行情（当前市值）
daily(ts_code)           → 最新收盘价、涨跌幅
daily_basic(ts_code)     → PE(TTM)、PB、市值、总股本

# 财务数据（最新报告期）
fina_indicator(ts_code)  → ROE、EPS、毛利率、每股净资产
income(ts_code)          → 营收、净利润、EBIT、EBITDA
balancesheet(ts_code)    → 总资产、净资产、有息负债、货币资金
cashflow(ts_code)        → 经营现金流、资本支出

# 同行业可比公司（Comps）
stock_basic(industry=XX) → 同行业股票列表
```

## 七、A股特殊考量

1. **商誉减值风险**：并购活跃公司需关注商誉占比，减值会直接影响估值
2. **股权激励摊销**：科技公司需从净利润中剔除非现金股权激励
3. **政策风险折价**：受监管行业（教育、互联网平台、地产）估值需加政策风险折价
4. **流通性溢价**：创业板/科创板相对主板有一定流通性溢价
5. **控股股东质押比例**：质押比例>60%需折价评估爆仓风险
