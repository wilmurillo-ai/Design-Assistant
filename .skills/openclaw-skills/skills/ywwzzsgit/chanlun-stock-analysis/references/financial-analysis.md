# 基本面与财报分析参考手册

## 一、财报数据获取原则

### 时效性规则（A股）
| 报告类型 | 发布截止时间 | 数据接口 |
|----------|-------------|---------|
| 年报（12月底财年） | 次年4月30日前 | `income`/`balancesheet`/`cashflow` (period=YYYYMMDD) |
| 一季报（3月底） | 4月30日前 | 同上 |
| 中报（6月底） | 8月31日前 | 同上 |
| 三季报（9月底） | 10月31日前 | 同上 |

**核心逻辑**：先确认当前最新可获取报告期，不默认拿年报
- 当前日期3月25日 → 最新可获取财报为**上一年度年报**（若已披露）或**三季报**
- 使用接口 `fina_indicator` 获取关键财务指标，`income`/`balancesheet`/`cashflow` 获取原始数据

### 数据接口映射
```
利润表:        income(ts_code, period)
资产负债表:    balancesheet(ts_code, period)
现金流量表:    cashflow(ts_code, period)
财务指标:      fina_indicator(ts_code, period)  # 包含ROE、毛利率等派生指标
审计意见:      fina_audit(ts_code, period)
```

## 二、盈利能力分析

### 关键指标

| 指标 | 计算公式 | 健康值参考 | 意义 |
|------|---------|-----------|------|
| **毛利率** | (营收-营业成本)/营收 | >30%优秀 | 产品竞争力 |
| **净利率** | 净利润/营收 | >10%良好 | 综合盈利能力 |
| **ROE** | 净利润/净资产 | >15%优秀 | 股东回报率 |
| **ROA** | 净利润/总资产 | >5%良好 | 资产利用效率 |
| **ROIC** | NOPAT/投入资本 | >WACC为佳 | 资本回报率 |
| **EPS** | 净利润/总股本 | 同比增长 | 每股收益 |

### 分析要点
- **毛利率趋势**：连续3年毛利率变化 > 5% 需深入分析原因
- **ROE杜邦分解**：ROE = 净利率 × 资产周转率 × 财务杠杆
  - 哪个维度驱动ROE？是否可持续？
- **扣非净利润**：关注扣除非经常性损益后的净利润，更真实反映主营盈利
- **营收质量**：营收增长 vs 应收账款增长，若应收增速 >> 营收增速，预警信号

## 三、成长能力分析

### 关键指标

| 指标 | 计算 | 优秀标准 |
|------|------|---------|
| 营收增速（YoY） | (本期-上期)/上期 | >20% 高成长 |
| 净利润增速（YoY） | 同上 | >20% 高成长 |
| 三年CAGR | (末值/初值)^(1/3)-1 | >15% |
| 研发投入占比 | 研发费用/营收 | 科技公司>10% |

### 分析要点
- **增长质量**：有机增长 vs 并购驱动
- **增长可持续性**：市场空间、竞争地位、行业趋势
- **研发转化效率**：研发投入 → 新产品 → 营收贡献

## 四、偿债能力分析

### 关键指标

| 指标 | 计算公式 | 安全值 |
|------|---------|--------|
| **流动比率** | 流动资产/流动负债 | 1.5~2.5 |
| **速动比率** | (流动资产-存货)/流动负债 | >1 |
| **资产负债率** | 总负债/总资产 | <60% |
| **有息负债率** | 有息负债/总资产 | <40% |
| **利息保障倍数** | EBIT/利息费用 | >3 |
| **净负债率** | (有息负债-现金)/净资产 | <100% |

### 分析要点
- 区分**有息负债**（需付息，实质债务压力）与**无息负债**（应付账款等，经营性）
- **货币资金质量**：大量货币资金 + 高有息负债 = 预警（可能存在资金受限或关联占用）
- **应付账款周转天数**：反映对上游供应商的话语权

## 五、现金流质量分析

### 核心原则：**净利润不等于真金白银**

| 分析维度 | 具体指标 | 说明 |
|----------|---------|------|
| **经营现金流 vs 净利润** | 经营现金流/净利润 > 1 | 盈利含金量高 |
| **自由现金流（FCF）** | 经营现金流 - 资本支出 | 真正可支配资金 |
| **FCF Margin** | FCF/营收 | >10%优质企业 |
| **分红可持续性** | 分红额/FCF | <80%为佳 |
| **资本支出强度** | 资本支出/折旧 | >1说明仍在扩张 |

### 现金流分类解读
| 类型 | 正负 | 典型含义 |
|------|------|----------|
| 经营现金流(+) 投资现金流(-) 融资现金流(-) | 成熟优质企业 |
| 经营现金流(+) 投资现金流(-) 融资现金流(+) | 高速扩张企业 |
| 经营现金流(-) 投资现金流(-) 融资现金流(+) | 初创/烧钱期 |
| 经营现金流(-) | 红色预警：主业不赚钱 |

## 六、运营效率分析

| 指标 | 计算 | 优秀标准 | 意义 |
|------|------|---------|------|
| 存货周转天数 | 365/(营收/存货) | 越短越好 | 库存管理效率 |
| 应收账款周转天数 | 365/(营收/应收账款) | 越短越好 | 回款效率 |
| 应付账款周转天数 | 365/(营业成本/应付账款) | 越长越好 | 上游谈判力 |
| **现金转换周期（CCC）** | 存货周转天数+应收周转天数-应付周转天数 | 越低越好 | 综合营运资金效率 |

## 七、财报质量红旗信号

⚠️ 以下信号表明财报可能存在质量问题：
1. 净利润增长但经营现金流持续下降
2. 应收账款增速远超营收增速
3. 存货大幅增加但毛利率下降
4. 商誉占净资产比例 > 30%（并购风险）
5. 关联交易占营收比例过高（>30%）
6. 审计意见非标准无保留意见
7. 货币资金充裕但大量借款
8. 营收增长但员工数/采购量不匹配

## 八、行业对比分析框架

分析任何公司财务数据，均需与行业均值对比：
- 毛利率 vs 行业均值：反映竞争壁垒
- ROE vs 行业均值：反映经营质量
- 成长速度 vs 行业增速：份额扩张或收缩

**数据接口**：
- `industry` 或 `stock_basic` 获取行业分类
- 对同行业多家公司运行相同财务分析进行对比
