# 缠论（缠中说禅）技术分析参考手册
## ——含多维辅助指标体系（专业增强版）

---

## 一、核心概念体系

### 1. K线处理规则（合并同向K线）
- **包含关系定义**：若相邻两根K线，一根的高低点均包含在另一根内（或相等），则发生"包含"
- **处理方向**：
  - 向上趋势中：取两K线的最高高点和最高低点，形成向上合并K线
  - 向下趋势中：取两K线的最低高点和最低低点，形成向下合并K线
- **目的**：消除不规则K线，还原真实的顶底结构

### 2. 分型（Fractal）
处理包含关系后，形成三根相邻K线的顶底结构：

| 类型 | 定义 | 含义 |
|------|------|------|
| **顶分型** | 中间K线高点高于两侧，低点高于两侧 | 局部价格高点 |
| **底分型** | 中间K线低点低于两侧，高点低于两侧 | 局部价格低点 |

### 3. 笔（Stroke）
- **定义**：由相邻的顶分型和底分型构成，中间必须有至少1根独立K线（不含端点）
- **上笔**：底分型 → 顶分型（价格上升）
- **下笔**：顶分型 → 底分型（价格下降）
- **有效性**：笔的最高点和最低点不能相互包含

### 4. 线段（Segment）
- **定义**：由至少3笔构成，且必须满足特征序列关系
- **特征序列**：上线段中取各上笔的顶，形成序列；若序列出现顶底特征，线段结束
- **线段方向**：决定了中枢的构建基础

### 5. 中枢（Center/Pivot）
中枢是缠论的核心概念，代表市场的"均衡区间"。

**中枢定义（严格版）**：
- 由至少3笔构成
- 3笔的高低点存在重叠区间：`ZG = min(各笔高点), ZD = max(各笔低点)`，且 ZG > ZD
- 中枢区间：[ZD, ZG]

**中枢级别**：
- 笔级别中枢 → 线段级别中枢 → 更高级别中枢
- 级别越高，代表趋势越显著

**中枢扩展**：后续进入/离开中枢的笔仍在中枢区间内，则中枢扩展

**中枢新生（升级）**：价格离开中枢后，在更高（低）位置形成新中枢，即趋势延续

### 6. 趋势（Trend）
通过中枢的位置关系判断趋势：

| 趋势 | 判断条件 |
|------|----------|
| **上涨趋势** | 后续中枢高于前序中枢（中枢整体向上移动） |
| **下跌趋势** | 后续中枢低于前序中枢（中枢整体向下移动） |
| **震荡** | 中枢在同一区间内反复（无方向性移动） |

### 7. 背驰（Divergence）
背驰是买卖点信号的核心依据：

**定义**：在同一趋势中，价格创出新高（低），但驱动力（力度）却减弱

**判断方法**：
- **MACD背驰**：趋势延伸中，MACD柱子（DIF-DEA面积）收缩，说明动能减弱
- **力度背驰**：后一段趋势的斜率/幅度小于前一段
- **均线背驰**：价格创新高但均线角度减小

**盘整背驰**：在中枢内部发生的背驰，级别较小，意义弱于趋势背驰

### 8. 买卖点（Trading Signals）

| 级别 | 定义 | 操作意义 |
|------|------|----------|
| **一买（1st Buy）** | 下跌趋势在某级别出现背驰后的底分型 | 最强买点，左侧抄底 |
| **一卖（1st Sell）** | 上涨趋势在某级别出现背驰后的顶分型 | 最强卖点，左侧高位卖出 |
| **二买（2nd Buy）** | 一买后上涨，回调不破一买低点，再次上涨 | 确认型买点，风险较小 |
| **二卖（2nd Sell）** | 一卖后下跌，反弹不破一卖高点，再次下跌 | 确认型卖点 |
| **三买（3rd Buy）** | 回调不进入中枢，构成新的中枢后向上突破 | 趋势延续买点 |
| **三卖（3rd Sell）** | 反弹不进入中枢，构成新的中枢后向下突破 | 趋势延续卖点 |

## 二、缠论分析流程

### 步骤1：确定分析级别
- 日线级别（适合中期持仓）
- 周线级别（适合长期趋势判断）
- 30分钟/60分钟（适合短线操作）
- **原则**：先看大级别方向，再找小级别入场点

### 步骤2：处理K线与分型
1. 对K线做包含处理
2. 标注顶分型（▼）和底分型（▲）

### 步骤3：划分笔和线段
1. 连接相邻顶底分型，确认笔
2. 根据特征序列划分线段

### 步骤4：识别中枢
1. 找出3笔以上重叠区间
2. 标注中枢区间 [ZD, ZG]
3. 判断中枢级别

### 步骤5：判断趋势与背驰
1. 观察中枢序列方向确定大趋势
2. 用MACD等工具检验背驰信号

### 步骤6：识别买卖点
1. 在背驰处寻找一买/一卖
2. 在二次确认处寻找二买/二卖
3. 中枢突破处确认三买/三卖

## 三、辅助技术指标体系（多维增强）

> **使用原则**：缠论结构为主框架，辅助指标做"多维共振验证"。
> 指标越多共振，信号越可靠；单一指标发出信号不宜轻举妄动。

---

### 📌 A类：趋势追踪指标

#### A1. 移动平均线系统（MA）
**参数配置（A股推荐）**：MA5 / MA10 / MA20 / MA60 / MA120 / MA250

| 均线 | 意义 | 实战用途 |
|------|------|----------|
| MA5 | 超短期（1周） | 短线进出参考 |
| MA10 | 短期（2周） | 短中线趋势 |
| MA20 | 月线（月均成本） | 中线趋势基准 |
| MA60 | 季线（季度成本） | 中长期趋势判断 |
| MA120 | 半年线 | 中长期支撑/压力 |
| MA250 | 年线 | 牛熊分界线 |

**信号规则**：
- **多头排列**：MA5 > MA10 > MA20 > MA60，上涨趋势强
- **空头排列**：MA5 < MA10 < MA20 < MA60，下跌趋势强
- **金叉**：短期均线上穿长期均线，买入信号
- **死叉**：短期均线下穿长期均线，卖出信号
- **均线粘合**：各均线聚拢，即将选择方向，等待突破

#### A2. MACD（Moving Average Convergence Divergence）
**参数**：快线EMA=12，慢线EMA=26，信号线=9

- **DIF**：EMA12 - EMA26，反映短期动能
- **DEA**：DIF的9日EMA，信号线
- **MACD柱**：2×(DIF - DEA)，柱子正负表示多空力度

**核心信号**：
- 金叉（DIF上穿DEA）→ 买入
- 死叉（DIF下穿DEA）→ 卖出
- **零轴金叉**：在零轴上方金叉，信号更强
- **背驰信号**（缠论核心）：价格新高/低而MACD柱面积收缩
- **顶背驰**：价格创新高，红柱面积 < 前一段 → 卖出预警
- **底背驰**：价格创新低，绿柱面积 < 前一段 → 买入预警

#### A3. 一目均衡表（Ichimoku Cloud，一目云）
**参数**：转换线9，基准线26，延迟线26，先行A/B（26/52）

| 组成 | 计算 | 意义 |
|------|------|------|
| **转换线（Tenkan）** | (9日最高+最低)/2 | 短期趋势方向 |
| **基准线（Kijun）** | (26日最高+最低)/2 | 中期趋势支撑/压力 |
| **先行带A** | (转换线+基准线)/2，前移26期 | 云的上沿 |
| **先行带B** | (52日最高+最低)/2，前移26期 | 云的下沿 |
| **延迟线（Chikou）** | 收盘价，后移26期 | 确认趋势 |

**信号解读**：
- 价格在云上方 → 多头市场
- 价格在云下方 → 空头市场
- 价格穿越云层 → 趋势转换信号
- 云厚 → 支撑/压力强；云薄 → 容易被突破
- 转换线上穿基准线 → 买入信号（TK交叉）

### 📌 B类：动量振荡指标（超买超卖）

#### B1. RSI（相对强弱指数）
**参数**：RSI6 / RSI12 / RSI24（短中长三周期）

**计算**：RSI = 100 - 100/(1 + RS)，RS = N日均涨 / N日均跌

| RSI区间 | 含义 | 操作参考 |
|---------|------|----------|
| > 80 | 严重超买 | 警惕见顶，考虑减仓 |
| 70~80 | 超买区 | 谨慎追涨 |
| 50~70 | 多头区 | 趋势向好，可持有 |
| 30~50 | 空头区 | 趋势向弱，观望 |
| 20~30 | 超卖区 | 关注反弹机会 |
| < 20 | 严重超卖 | 可能见底，考虑轻仓布局 |

**高级用法**：
- **RSI背离**：价格创新高而RSI未创新高 → 顶背离（卖出信号）
- **RSI底背离**：价格创新低而RSI未创新低 → 底背离（买入信号）
- **RSI多周期共振**：RSI6/12/24同步超卖，信号更强
- **50分水岭**：RSI站上50看多，跌破50看空

#### B2. KDJ（随机振荡指标）
**参数**：K=9，D=3，J=3（标准参数）

| 指标 | 含义 |
|------|------|
| **K线** | 快速随机值，对价格反应灵敏 |
| **D线** | K的3日均值，较平滑 |
| **J线** | 3K - 2D，超前指标，超买超卖更敏感 |

**信号规则**：
- J > 100：严重超买，注意回调
- J < 0：严重超卖，注意反弹
- K上穿D（金叉）：买入信号
- K下穿D（死叉）：卖出信号
- **KDJ底背离**：价格新低但KDJ未新低 → 买入信号
- **高位死叉（KDJ在80以上死叉）**：强烈卖出信号
- **低位金叉（KDJ在20以下金叉）**：强烈买入信号

#### B3. CCI（顺势指标）
**参数**：N=14 或 N=20

**计算**：CCI = (TP - MA) / (0.015 × 平均偏差)，TP = (高+低+收)/3

| CCI区间 | 信号 |
|---------|------|
| > +200 | 极度超买，高风险 |
| +100 ~ +200 | 超买，谨慎 |
| -100 ~ +100 | 正常波动区间 |
| -200 ~ -100 | 超卖，关注机会 |
| < -200 | 极度超卖，反转信号强 |

**特殊用法**：CCI从超买区回落至+100以下 → 卖出；从超卖区回升至-100以上 → 买入

#### B4. WR（威廉指标，Williams %R）
**参数**：N=14

**含义**：衡量收盘价在N日最高-最低区间中的相对位置

| WR值 | 含义 |
|------|------|
| 0 ~ -20 | 超买区（注意：WR值为负，-20以上） |
| -20 ~ -80 | 正常波动 |
| -80 ~ -100 | 超卖区 |

**与KDJ配合**：WR进入超卖区(-80以下)同时KDJ低位金叉 → 强买入信号

### 📌 C类：波动率指标（空间与风险）

#### C1. 布林带（Bollinger Bands，BOLL）
**参数**：中轨MA20，上轨=MA20+2σ，下轨=MA20-2σ

**核心规律**：
- 价格有95.4%的概率运行在上下轨之间
- **带宽收窄**（布林带收口）→ 即将大幅波动，等待突破方向
- **带宽扩大**（布林带开口）→ 趋势行情中，追势有效
- **价格触及上轨**：超买警示，若配合背驰可卖出
- **价格触及下轨**：超卖，若配合底分型可买入
- **价格突破上轨（走轨）**：强势上涨，可追多；突破下轨：强势下跌

**布林带与缠论结合**：
- 中轨（MA20）常与缠论中枢中点重合，可作为中枢中轴参考
- 上轨常是上涨笔的目标位；下轨常是下跌笔的目标位

#### C2. ATR（平均真实波幅）
**参数**：N=14

**计算**：真实波幅TR = max(高-低, |高-前收|, |低-前收|)；ATR = TR的14日均值

**用途**：
- **动态止损**：止损距离 = 1.5~2倍 ATR（避免被正常波动震出）
- **仓位管理**：单笔风险 = 账户资金 × 1~2%；仓位 = 单笔风险 / ATR
- **ATR扩张**：波动率上升，市场进入活跃期，趋势行情可能开始
- **ATR收缩**：波动率下降，市场平静，可能酝酿突破

**ATR止损公式**（专业版）：
```
止损价 = 买入价 - 2 × ATR(14)
目标价 = 买入价 + 3 × ATR(14)（风险收益比 1:1.5）
```

#### C3. 历史波动率（HV，Historical Volatility）
- 计算近20/60日收益率的标准差（年化）
- HV高 → 波动大，风险高，仓位宜轻
- HV低 → 波动小，可能酝酿趋势，适合布局
- HV与ATR配合，从两个角度衡量风险

### 📌 D类：量价关系指标（成交量分析）

> **核心原则**：量是价的先行指标，放量上涨确认趋势，缩量反弹是陷阱。

#### D1. OBV（能量潮，On Balance Volume）
**计算**：收盘价上涨日 +成交量；下跌日 -成交量；累计求和

**信号规则**：
- **OBV持续上升**：资金持续流入，看多
- **OBV持续下降**：资金持续流出，看空
- **价量背离**：价格新高而OBV未新高 → 上涨动能衰减（顶背离）
- **价格新低而OBV未新低** → 抛压减弱（底背离）
- OBV突破前期高点与价格同步 → 趋势有效确认

#### D2. 量比（Volume Ratio）
**计算**：当日成交量 / 过去5日日均成交量

| 量比 | 含义 |
|------|------|
| > 5 | 极度放量，极端行情（涨停/跌停概率高） |
| 2.5~5 | 大幅放量，强势信号或见顶信号 |
| 1.5~2.5 | 温和放量，趋势确认 |
| 0.8~1.5 | 平量，无明显异动 |
| 0.5~0.8 | 缩量，观望情绪浓厚 |
| < 0.5 | 极度缩量，可能见底或即将变盘 |

#### D3. 换手率（Turnover Rate）
**含义**：当日成交量 / 流通股本

| 换手率 | 含义 |
|--------|------|
| > 10% | 极高换手，庄家洗盘或出货 |
| 5~10% | 活跃，趋势行情 |
| 2~5% | 正常区间 |
| < 1% | 低迷，观望为主 |

**实战结合**：
- 底部区域 + 换手率突然放大 → 主力吸筹信号
- 高位区域 + 持续高换手 → 出货信号
- 缩量下跌 → 抛压减弱，可能见底
- 缩量上涨 → 上涨动力不足，需谨慎

#### D4. VWAP（成交量加权平均价）
**计算**：∑(价格×成交量) / ∑成交量

**用途**：
- **机构成本线**：大资金常以VWAP为交易基准
- 价格在VWAP上方：多头占优；下方：空头占优
- 常用于日内交易判断买卖时机
- **与均线区别**：VWAP考虑成交量权重，比普通均线更能反映真实成本

#### D5. MFI（资金流量指标，Money Flow Index）
**参数**：N=14

**含义**：结合价格和成交量的RSI变种，又称"量能版RSI"

- MFI > 80：超买，资金过度流入
- MFI < 20：超卖，资金过度流出
- **MFI背离**（与RSI背离配合）：价格新高而MFI未新高 → 双重顶背离警告

### 📌 E类：资金与筹码分析

#### E1. 北向资金（沪深港通北向）
- 追踪外资净流入/流出
- 连续多日净流入 + 价格上涨 → 外资认可，趋势可信
- 大幅净流出 + 价格下跌 → 外资撤离，下行压力大
- 单日净流入超过50亿 → 大资金异动，重点关注

#### E2. 主力资金流向（大单净流入）
- **超大单（>100万元/笔）**：机构/游资行为
- **大单（>20万元/笔）**：聪明钱行为
- **散单（<5万元/笔）**：散户行为

**信号**：
- 连续3日超大单净流入 → 主力在吸筹
- 价格上涨但大单净流出 → 主力出货，谨慎
- 价格下跌但大单净流入 → 主力逢低吸纳，关注买入机会

#### E3. 筹码分布（股权集中度）
- **套牢盘**：当前价格以上的成本密集区 → 形成压力位
- **获利盘**：当前价格以下的成本密集区 → 形成支撑位
- **筹码峰**：大量筹码集中的价格区域，是重要的支撑/压力参考
- **筹码转移**：低位筹码向高位转移，主力换手完成，上涨启动

### 📌 F类：趋势强度与市场结构

#### F1. ADX（平均趋向指标）
**参数**：N=14，+DI、-DI、ADX三线

| ADX值 | 趋势强度 |
|-------|---------|
| < 20 | 无明显趋势，震荡行情 |
| 20~25 | 趋势开始形成 |
| 25~40 | 趋势明确，可顺势操作 |
| 40~60 | 强趋势，顺势持有 |
| > 60 | 极强趋势，注意回调风险 |

**与缠论结合**：
- ADX > 25 时缠论的顺势买卖点成功率更高
- ADX < 20 时市场震荡，缠论的中枢震荡策略更适用

#### F2. DMI（趋向指标）
- +DI > -DI：多头占优
- -DI > +DI：空头占优
- +DI上穿-DI：买入信号
- -DI上穿+DI：卖出信号
- 与ADX配合：ADX上升 + +DI > -DI = 强势上涨确认

#### F3. 乖离率（BIAS）
**计算**：BIAS = (收盘价 - MA) / MA × 100%

| 乖离率 | 含义 |
|--------|------|
| MA20乖离 > +10% | 短期超买，可能回调 |
| MA20乖离 < -10% | 短期超卖，可能反弹 |

## 五、时间周期协同分析框架

**原则**：用"上下级别联动"确认方向，避免逆势操作。

```
月线（大趋势方向）
    ↓
周线（中期趋势）
    ↓
日线（实际操作级别）
    ↓
60分钟（精确入场点）
```

**分析步骤**：
1. 先看**月线**：判断长周期牛熊位置
2. 再看**周线**：确认中期方向，找大级别买卖点
3. 日线操作：在周线方向确认后，按日线买卖点入场
4. 60分钟确认：用小级别确认买入时机，控制入场成本

**共振要求**：月线、周线、日线方向一致时，操作胜率最高。
