# Platform Content Rules

## 微博 (Weibo)
- **Max length**: 2000 chars (display truncates at 140)
- **Images**: up to 9, JPG/PNG/GIF
- **Hashtags**: #话题# format (Chinese-style double hash)
- **Links**: auto-shortened by Weibo
- **Tone**: casual, emoji-friendly
- **Adaptation**: keep under 500 chars for best engagement; add 2-3 relevant #话题#

## 小红书 (Xiaohongshu)
- **Max length**: 1000 chars for caption
- **Images**: 1-9 required (cannot post text-only)
- **Hashtags**: #话题 format (single hash, no closing)
- **Title**: required, separate from body (≤20 chars recommended)
- **Tone**: personal, storytelling, "种草" style
- **Adaptation**: rewrite as personal experience; add emoji; split title from body; must have images
- **⚠️ No external links allowed** — remove all URLs, describe content instead

## 知乎 (Zhihu)
- **想法 (thoughts)**: ≤2000 chars, casual short post
- **文章 (article)**: unlimited, for long-form
- **Images**: supported in both
- **Hashtags**: add 话题标签 via tag selector
- **Tone**: informative, analytical
- **Adaptation**: add context and technical detail; use 想法 for <500 chars, 文章 for longer

## Twitter/X
- **Max length**: 280 chars per tweet
- **Thread**: chain tweets for longer content
- **Images**: up to 4 per tweet, JPG/PNG/GIF
- **Hashtags**: #hashtag (English style)
- **Links**: shortened by t.co
- **Tone**: concise, punchy
- **Adaptation**: English version; distill to key value prop + install command + link; use thread for full content

## Reddit
- **Title**: required, ≤300 chars
- **Body**: unlimited (markdown supported)
- **Images**: upload or link
- **Subreddit**: must specify (e.g. r/OpenClaw, r/AItools, r/MachineLearning)
- **Tone**: community-appropriate, not overtly promotional
- **Adaptation**: frame as sharing/discussion, not advertising; add context about why it's useful

## V2EX
- **Title**: required
- **Body**: markdown-like
- **Node**: must specify (e.g. 分享创造, 程序员)
- **Images**: upload via imgur or similar, paste link
- **Tone**: developer community, technical
- **Adaptation**: focus on technical details and pain points; keep it authentic

## LinkedIn
- **Max length**: 3000 chars
- **Images**: supported
- **Hashtags**: #hashtag (English)
- **Tone**: professional, industry-focused
- **Adaptation**: English; professional framing; emphasize innovation and metrics

## 豆瓣 (Douban)
- **广播 (status)**: ≤280 chars
- **Images**: supported
- **Tone**: literary, indie
- **Adaptation**: short, thoughtful phrasing
