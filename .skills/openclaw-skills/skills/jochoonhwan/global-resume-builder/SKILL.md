# 🌍 Global Resume/CV Builder

## Overview

Creates professional resumes optimized for international job markets. Handles ATS (Applicant Tracking Systems), regional formatting, and industry-specific templates.

## Supported Markets

| Region | Format Style | Key Features |
|--------|--------------|--------------|
| US/Canada | 1-page, Summary-led | ATS-optimized, Action verbs |
| UK | 2-page, Profile-led | Personal statement, UK spelling |
| EU | 2-page, Photo-optional | GDPR compliant |
| APAC | 1-2 page | Photo optional, English |

## Resume Types by Industry

EDUCATION

### 3. Marketing/Creative
```markdown

[Portfolio URL]

EXPERIENCE
Campaign results prominently displayed

Links to notable work
```

## ATS Optimization

### Keywords
- Match job description keywords exactly
- Use standard job titles
- Avoid: graphics, tables, headers/footers

### Formatting Rules
- .docx or .pdf (check job posting)
- Standard fonts: Arial, Calibri, Garamond
- No columns (some ATS can't read)
- Plain text file names: "John_Doe_Resume.pdf"

## Action Verbs by Category

| Category | Verbs |
|----------|-------|
| Leadership | Led, Directed, Managed, Mentored |
| Achievement | Achieved, Exceeded, Delivered, Transformed |
| Analysis | Analyzed, Evaluated, Researched, Identified |
| Communication | Presented, Negotiated, Collaborated, Influenced |
| Technical | Developed, Implemented, Built, Designed |

## Key Differences: US vs UK vs EU

### US Resume
- 1 page max (entry level 1, exec 2)
- "Resume" not "CV"
- No photo
- Summary first
- Dates in month/year
- Achievements with metrics

### UK CV
- 2 pages max
- "CV" or "Curriculum Vitae"
- Profile/personal statement first
- "Managed team of 5" not "Led"

### EU CV
- Photo optional (country-specific)
- Sometimes requires photo
- More personal details allowed
- Date format: DD/MM/YYYY

## Pro Tips

1. **Tailor every application** - Match keywords from job description
2. **Quantify everything** - "Increased sales by 25%"
3. **Use power verbs** - Start bullets with action verbs
4. **Keep it clean** - White space matters
5. **Proofread** - Zero typos allowed
