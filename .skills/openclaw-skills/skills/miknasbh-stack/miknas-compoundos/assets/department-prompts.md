## Customization Notes

**For each agent:**

1. **Fill in the brackets** with your specific business information
2. **Update capabilities** to match what the agent can actually do
3. **Adjust autonomy levels** based on your comfort with AI autonomy
4. **Add context file paths** for your actual file locations
5. **Customize workflow** to match your processes
6. **Test the agent** with real tasks and refine

**Common modifications:**
- Add platform-specific instructions (e.g., "Use OpenAI API for...")
- Add brand-specific voice guidelines
- Add tool-specific workflows
- Add compliance or legal requirements
- Adjust output format to match your systems
