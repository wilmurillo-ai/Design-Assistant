# Find Skills

🔍 Universal skill discovery across all AI agent skill marketplaces.

## Overview

This skill enables discovering, searching, and installing skills from 400K+ skills across multiple AI agent skill marketplaces using the SkillKit CLI. It supports browsing official partner collections (Anthropic, Vercel, Supabase, Stripe, and more) and community repositories.

## Quick Start

```bash
# Search for skills
npx skillkit@latest find <query>

# Browse the marketplace
npx skillkit@latest marketplace

# Install skills from GitHub
npx skillkit@latest install <owner/repo>

# Get project-based recommendations
npx skillkit@latest recommend
```

## Use Cases

- "Find a skill for React testing"
- "Is there a skill for Kubernetes?"
- "Browse available skills"
- "What skills are available for security?"

## Features

- Search 400K+ skills by domain or technology
- Install from official partner collections
- Browse community-maintained skill repositories
- Get personalized recommendations based on your project
