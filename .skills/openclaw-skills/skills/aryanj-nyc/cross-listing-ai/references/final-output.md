# Final Output

Use this file immediately before drafting the seller-facing response.

## Response Shape

Keep the final answer copy-paste friendly and conversational.

Recommended order:

1. one short summary sentence
2. price recommendation block
3. one section per selected marketplace
4. skipped marketplace notes, if any
5. one short revision prompt

## Price Recommendation Block

Include:

- suggested ask
- expected sale range
- confidence
- short comp summary

Keep it brief. Do not bury the seller in raw research notes.

## Marketplace Sections

For each selected marketplace, provide:

- marketplace name
- title
- description or copy block
- bullets or item specifics when useful
- seller notes

Keep the formatting simple enough to copy into marketplace forms.

## Revision Handling

If the seller corrects a detail:

- acknowledge the change briefly
- update the price only if the change affects value
- regenerate only the marketplaces affected by the change
