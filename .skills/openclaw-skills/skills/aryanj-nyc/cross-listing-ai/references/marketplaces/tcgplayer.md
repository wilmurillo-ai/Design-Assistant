# TCGPlayer

Use this file when generating TCGPlayer output.

## Hard Gate

Do not generate TCGPlayer output until these are known:

- card name
- game
- set

If any are missing, ask the seller. If the seller still cannot provide them, skip TCGPlayer and explain why.

## Listing Style

- Keep the title precise and card-specific.
- Include the exact card name, set, and any print or finish detail that matters.
- Use a condition description that matches the evidence and seller confirmation.

## Description

- Keep it factual and compact.
- Mention language, finish, rarity, card number, and notable defects when known.
- Avoid speculative authenticity or grade language.

## Seller Notes

- Confirm print, language, finish, and set match the card before posting.
- If the card is raw, avoid implying a professional grade.
