# Facebook Marketplace

Use this file when generating Facebook Marketplace output.

## Listing Style

- Keep the title at or under 100 characters.
- Optimize for local browsing rather than keyword stuffing.
- Emphasize the main item, condition, and standout included extras.

## Description

- Keep it plainspoken and local-first.
- Mention pickup, delivery, or meetup constraints if the seller shared them.
- Highlight bulky, fragile, or same-day convenience factors when relevant.

## Seller Notes

- Add pickup, meetup, or delivery details inside the Facebook form.
- Keep local logistics simple and explicit.
