# Craigslist

Use this file when generating Craigslist output.

## Listing Style

- Keep the title at or under 120 characters.
- Use plain language and concrete nouns.
- Avoid marketplace jargon or hype.

## Description

- Be direct about condition, included parts, and local logistics.
- Mention testing status and notable flaws early.
- Keep paragraphs short and easy to skim.

## Seller Notes

- Add location, contact preference, and payment expectations before posting.
- If the item is local-only, say so clearly.
