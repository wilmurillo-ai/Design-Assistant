# eBay

Use this file when generating eBay output.

## Listing Style

- Keep the title at or under 80 characters.
- Lead with brand, model, or the most searchable identity terms.
- Put condition-driving keywords early when they matter.
- Avoid filler, emoji, and all-caps spam.

## Description

- Keep the opening sentence factual and specific.
- Add short bullets when they improve scannability.
- Mention included accessories, defects, and testing status clearly.

## Seller Notes

- Review item specifics before posting.
- Verify shipping profile, return settings, and category mapping.
