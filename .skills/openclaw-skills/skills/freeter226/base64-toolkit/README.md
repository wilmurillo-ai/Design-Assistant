# Base64 Toolkit

Base64 encoding and decoding toolkit for developers.

## Features

- Encode/decode text to/from Base64
- URL-safe Base64 encoding
- Image to Base64 data URI conversion

## Quick Start

```bash
# Encode
python3 scripts/base64_toolkit.py encode --input "Hello"

# Decode
python3 scripts/base64_toolkit.py decode --input "SGVsbG8="

# Image to Base64
python3 scripts/base64_toolkit.py image-encode --input image.png
```
