# Example Article Template

Use this as a reference for article style and structure.

```markdown
# Article Title

*Context line — when tested, what hardware, source, etc.*

## Background

Brief context. Why this matters. 2-3 sentences max.

## Main Content

Use H2 sections to organize. Tables for comparisons:

| Thing | Metric A | Metric B |
|---|---|---|
| Option 1 | value | value |
| Option 2 | value | value |

Code blocks for commands:

\`\`\`bash
example-command --flag value
\`\`\`

## Key Findings

- Bullet points for takeaways
- Be specific — include numbers, not just "it was faster"
- Each bullet should be independently useful

## Setup / How-To (if applicable)

Step-by-step instructions if the article is procedural.

## See Also

- [Related Article](relative-path.md) — one-line description
```
