# iOS Components Reference

Quick reference for all iOS UIKit and SwiftUI components.

## Navigation
- **UINavigationBar / NavigationView** - Hierarchical navigation with title and back button
- **UITabBar / TabView** - 3-5 primary destinations at bottom
- **UISearchBar / .searchable()** - Search and filter content

## Views & Containers
- **UIScrollView / ScrollView** - Scrollable content area
- **UITableView / List** - Vertical list of rows
- **UICollectionView / LazyVGrid/LazyHGrid** - Grid layouts
- **UIStackView / VStack/HStack/ZStack** - Arrange views vertically, horizontally, or layered

## Controls
- **UIButton / Button** - Tappable action triggers
- **UISwitch / Toggle** - On/off binary state
- **UISlider / Slider** - Continuous value selection
- **UISegmentedControl / Picker(segmented)** - 2-5 mutually exclusive options
- **UIStepper / Stepper** - Increment/decrement values
- **UITextField / TextField** - Single-line text input
- **UITextView / TextEditor** - Multi-line text input

## Pickers
- **UIDatePicker / DatePicker** - Date and time selection (wheel, compact, graphical)
- **UIPickerView / Picker** - Multi-column wheel picker
- **Menu / Menu** - Dropdown selection (iOS 14+)

## Indicators
- **UIActivityIndicatorView / ProgressView(circular)** - Loading spinner
- **UIProgressView / ProgressView(linear)** - Progress bar
- **UIPageControl / PageIndicator** - Dot indicators for pages

## Information
- **UILabel / Text** - Display text
- **UIImageView / Image** - Display images
- **UIAlertController / .alert()** - Alert dialog with actions
- **.sheet() / .fullScreenCover()** - Modal presentation

See SKILL.md and ios.md for detailed usage patterns.
