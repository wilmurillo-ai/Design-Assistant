# tvOS Design Patterns

Designed for 10-foot viewing distance with focus-driven navigation.

## Core Principles
- **Cinematic:** Beautiful, immersive visuals
- **Focus-Driven:** Clear focus indicator
- **Simple:** Easy to understand from across room

## Layout
- **Large touch targets:** Minimum 250pt
- **Grid layouts:** Flow, columns
- **Focus engine:** Handles navigation automatically
- **Parallax:** Depth on focus (cards, buttons)

## Siri Remote
- **Directional pad:** Up/down/left/right
- **Touch surface:** Swipe gestures
- **Select (click):** Primary action
- **Menu:** Back navigation
- **Siri:** Voice commands

## Components
- **Tab Bar:** Top of screen (not bottom like iOS)
- **Collection Views:** Grid of content
- **Cards:** Hover effects, parallax
- **Buttons:** Large, clear labels

## Typography
- **SF Pro Display:** Large sizes for distance
- **Font sizes:** Much larger than iOS (48pt+ for titles)
- **High contrast:** Easy to read from distance

## Focus & Selection
- **Focus:** Clear border/glow around focused item
- **Parallax:** Focused item appears elevated
- **Sound:** Subtle audio feedback
- **Animation:** Smooth focus transitions
