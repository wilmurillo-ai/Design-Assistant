# 数据保护法规 (Data Protection Regulations)

## 全球数据保护法规概览

### 主要法规对比

| 地区 | 法规 | 生效时间 | 特点 |
|------|------|----------|------|
| 欧盟 | GDPR | 2018年5月 | 最严格、域外适用 |
| 美国加州 | CCPA/CPRA | 2020年/2023年 | 消费者权利导向 |
| 中国 | PIPL/DSL | 2021年/2021年 | 数据本地化要求 |
| 日本 | APPI | 2022年修订 | 宽松例外条款 |
| 韩国 | PIPA | 2020年修订 | 严格同意要求 |
| 新加坡 | PDPA | 2020年修订 | 风险导向方法 |
| 东盟 | ADPH (参考) | - | 框架指南 |

## 欧盟 GDPR

### 核心原则
1. **合法性、透明性**：合法基础处理、公开透明
2. **目的限制**：明确目的、不超范围处理
3. **数据最小化**：仅收集必要数据
4. **准确性**：保持数据准确
5. **存储限制**：期限内存储
6. **完整性与保密性**：安全保障
7. **问责制**：证明合规

### 合法处理基础
| 法律基础 | 适用场景 |
|----------|----------|
| 同意 (Consent) | 市场营销、非必要cookies |
| 合同履行 (Contract) | 服务提供、会员管理 |
| 法律义务 (Legal Obligation) | 税务申报、劳动合规 |
| 重大利益 (Vital Interests) | 医疗紧急情况 |
| 公共任务 (Public Task) | 政府职能 |
| 合法利益 (Legitimate Interests) | 安全防护、反欺诈 |

### 数据主体权利
- 知情权
- 访问权
- 更正权
- 删除权（被遗忘权）
- 限制处理权
- 数据可携带权
- 反对权
- 拒绝自动化决策权

### 违规处罚
- **一般违规**：最高1000万欧元或全球营业额2%
- **严重违规**：最高2000万欧元或全球营业额4%

### 跨境传输机制
| 机制 | 说明 |
|------|------|
| 充分性认定 | 欧委会认定国家具有充分保护水平 |
| 标准合同条款 (SCCs) | 合同保障措施 |
| 有约束力公司规则 (BCRs) | 集团内部传输 |
| 补充措施 | 技术、组织措施增强保护 |

## 美国 CCPA/CPRA

### 适用范围
- 年营业额超过2500万美元
- 处理超过10万消费者/家庭数据
- 50%以上收入来自销售消费者数据

### 消费者权利
- 知情权：收集、使用、共享的知情
- 删除权：要求删除个人信息
- 选择退出权：拒绝出售个人信息
- 不受歧视权：平等服务权

### CPRA 新增
- 敏感个人信息处理限制
- 扩大"出售"定义
- 消费者纠正权
- 限制使用个人信息权

## 中国数据保护法规

### 个人信息保护法 (PIPL)
- 域外适用效力
- 个人信息处理规则
- 跨境传输规则
- 数据本地化要求（关键信息基础设施）

### 数据安全法 (DSL)
- 数据分类分级
- 数据安全审查
- 重要数据出境评估

### 相关法规
| 法规 | 主要内容 |
|------|----------|
| 网络安全法 | 网络运营者安全义务 |
| 数据安全法 | 数据分类分级、安全审查 |
| 个人信息保护法 | 个人信息处理规则 |
| 儿童个人信息保护规定 | 未成年人特别保护 |
| 汽车数据安全管理若干规定 | 智能网联汽车数据 |

## 东南亚数据保护

### 新加坡 PDPA
- 通知义务
- 同意义务
- 目的限制
- 访问、更正权
- 准确、销毁义务
- 保护义务
- 转移限制

### 马来西亚 PDPA
- 同意、通知
- 收集、使用
- 处理、披露
- 数据主体权利

### 泰国 PDPA
- GDPR 风格立法
- 严格同意要求
- 数据保护官要求

### 越南 PDPA
- 2023年7月生效
- 严格同意要求
- 跨境传输限制

## 合规实施框架

### 数据保护合规步骤
```
1. 数据映射 → 2. 风险评估 → 3. 制度建立 → 4. 技术措施 → 5. 持续监控
```

### 数据保护影响评估 (DPIA)
适用场景：
- 系统性评估个人数据
- 大规模处理敏感数据
- 公共区域监控
- 跨境传输

### 数据保护官 (DPO)
要求场景：
- 核心活动涉及大规模系统监控
- 核心活动涉及大规模处理特殊类别数据
