# 税务合规规则 (Tax Compliance Rules)

## 跨境税务体系概览

### 主要税种

| 税种 | 说明 | 跨境要点 |
|------|------|----------|
| 企业所得税 | 对企业利润征税 | 常设机构、全球所得 |
| 预提税 | 支付给境外时预扣 | 税收协定优惠 |
| 增值税/GST | 消费税 | 跨境劳务/数字服务 |
| 个人所得税 | 对个人所得征税 | 税务居民判定 |
| 关税 | 进出口环节 | 原产地、配额 |
| 印花税 | 特定文件/交易 | 股权转让、合同 |

## 企业所得税

### 跨境利润征税规则

#### 常设机构 (Permanent Establishment - PE)
| 类型 | 定义 |
|------|------|
| 固定场所PE | 固定的营业场所（办公室、工厂等） |
| 代理PE | 有权签订合同的非独立代理人 |
| 建设PE | 建筑工地/安装项目超过12个月 |
| 服务PE | 提供服务累计超过12/18个月 |
| 依赖代理人PE | 惯常行使权力签订合同 |

#### 跨境所得类型
- **来源地所得**：在境内发生或实现的所得
- **全球所得**：税务居民的全部所得
- **资本利得**：股权转让、资产处置收益

## 转让定价 (Transfer Pricing)

### OECD 转让定价指南原则
1. **独立交易原则 (Arm's Length Principle)**
2. **可比性分析**
3. **最合适方法选择**
4. **文档要求**

### 转让定价方法
| 方法 | 适用场景 |
|------|----------|
| 可比非受控价格法 (CUP) | 有可比交易 |
| 再销售价格法 (RPM) | 分销商交易 |
| 成本加成法 (CPM) | 制造商交易 |
| 利润分割法 (PSM) | 高度整合业务 |
| 交易净利润法 (TNMM) | 利润指标分析 |

### 转让定价文档
| 层级 | 内容 | 门槛 |
|------|------|------|
| 国别报告 (CbCR) | 集团全球分配、税务 | 年收入>7.5亿欧元 |
| 主文档 (Master File) | 集团整体信息 | 各国要求不同 |
| 本地文档 (Local File) | 本地交易详情 | 各国要求不同 |

### 无形资产转让定价
- 成本分摊协议 (Cost Sharing Arrangement)
- 研发成本分摊
- 许可费定价

## 预提税

### 税收协定优惠
```
股息的预提税税率：
标准税率 → 协定税率（通常5%-15%）
条件：受益所有人、合格股东比例、实际管理机构等
```

### 协定网络覆盖的主要国家
| 地区 | 协定数量(约) | 典型优惠 |
|------|-------------|----------|
| 香港 | 40+ | 股息0%-10% |
| 新加坡 | 80+ | 股息0%-10% |
| 中国 | 100+ | 股息10% |
| 荷兰 | 90+ | 股息5%-15% |
| 卢森堡 | 80+ | 股息5%-15% |

### 受益所有人规则
- 必须是所得的实际受益人
- 导管公司/空壳公司可能不享受协定优惠
- 反滥用条款 (LOB, PPT) 限制

## 中国税务合规

### 企业所得税
- 基本税率：25%
- 小型微利：20%（实际10%）
- 高新技术企业：15%
- 重点扶持行业：15%

### 预提税
| 所得类型 | 标准税率 | 协定税率 |
|----------|----------|----------|
| 股息 | 10% | 5%-15% |
| 利息 | 10% | 7%-10% |
| 特许权使用费 | 10% | 6%-10% |

### 税收协定应用
- 受益所有人判定
- 导管公司防范
- 常设机构豁免
- 资料提交要求

### BEPS相关措施
- 37项行动计划
- 经济实质要求
- 数字服务税
- 国别报告交换
