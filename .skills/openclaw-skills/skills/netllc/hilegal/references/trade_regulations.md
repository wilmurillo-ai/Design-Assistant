# 贸易管制法规 (Trade Regulations)

## 国际贸易管制体系

### 主要管制框架

| 管制体系 | 管理机构 | 适用范围 |
|----------|----------|----------|
| 美国出口管制 (EAR) | 商务部工业安全局 (BIS) | 美国来源物项、美国成分超过门槛 |
| 美国制裁 (OFAC) | 财政部海外资产控制办公室 | 美国个人/实体、特定国家 |
| 欧盟出口管制 | 欧盟理事会、成员国主管机关 | 欧盟成员国、双重使用物项 |
| 中国出口管制 | 商务部、军委 | 中国来源物项、特定物项 |

## 美国出口管制 (Export Administration Regulations - EAR)

### EAR 分类体系 (ECCN)

| 类别 | 领域 | 示例 |
|------|------|------|
| 0 | 核与核相关 | 核材料、设施 |
| 1 | 材料/化学品/微生物 | 特殊材料、生物制剂 |
| 2 | 材料加工 | 机床、金属加工设备 |
| 3 | 电子 | 半导体、电路、测试设备 |
| 4 | 计算机 | 超算、高性能计算 |
| 5 | 通信/信息安全 | 加密、电信设备 |
| 6 | 激光/传感器 | 激光、导航设备 |
| 7 | 导航/航空电子 | 航空电子、惯性导航 |
| 8 | 海洋 | 船舶、潜艇设备 |
| 9 | 航天/推进 | 火箭、航空器 |

### 实体清单 (Entity List)
被列入实体清单的企业出口需要申请许可。

### 美国人说原则 (De Minimis Rule)
- **外国直接产品规则 (FDPR)**：使用美国技术/软件生产的特定产品
- **外国直接产品 (FDP)** 门槛：超过特定美国成分比例受EAR约束

## 欧盟出口管制

### 欧盟双重使用物项清单 (EU Dual-Use List)
- 附件I：国际管制清单
- 附件III：欧盟额外管制清单

### 欧盟制裁
- 欧盟理事会统一清单
- 针对性制裁措施
- 资产冻结清单

## 中国出口管制

### 管制物项
- 两用物项：核、生物、化学、导弹相关
-军品：武器装备及相关物资
- 特定物项：稀土、稀有金属等

### 出口许可要求
- 出口经营者登记
- 出口许可申请
- 最终用户/用途证明

## 原产地规则

### 非优惠原产地规则
- 完全获得标准
- 实质性改变标准（税则归类改变、加工增值、混合标准）

### 优惠原产地规则
| 协定 | 原产地标准 |
|------|------------|
| RCEP | 区域价值成分≥40% 或 税则归类改变 |
| CPTPP | 区域价值成分≥45% |
| 中国-东盟 | 区域价值成分≥40% |
| 中国-欧盟 | 区域价值成分≥45% |

## 关税政策

### 关税分类
- 从价税：按货值百分比
- 从量税：按数量单位
- 复合税：结合从价和从量
- 禁止性关税：高额保护关税

### 贸易救济措施
- 反倾销税
- 反补贴税
- 保障措施

## 合规建议

### 出口管制合规要点
1. 建立出口管制合规体系
2. 进行 ECCN 分类
3. 筛查交易对手方
4. 保存交易记录
5. 获取必要的出口许可

### 制裁合规要点
1. 全面筛查制裁名单
2. 了解"美国人说"原则
3. 实施尽职调查
4. 建立预警机制
5. 员工合规培训
