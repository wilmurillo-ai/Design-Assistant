### 示例七：数据表设计

```markdown
**用户会话表（user_session）**

| 字段名 | 字段中文名 | 数据类型 | 取值范围 | 是否必填 | 备注说明 |
|--------|------------|----------|----------|----------|----------|
| session_id | 会话ID | VARCHAR(36) | UUID格式 | 是 | 主键 |
| user_id | 用户ID | VARCHAR(36) | UUID格式 | 是 | 外键 |
| create_time | 创建时间 | DATETIME | 合法日期时间 | 是 | 自动记录 |
| expire_time | 过期时间 | DATETIME | 合法日期时间 | 是 | 自动计算 |
| status | 会话状态 | VARCHAR(20) | ACTIVE/EXPIRED | 是 | 默认ACTIVE |
```

## 快速检查清单

编写PRD时，确保包含：

- [ ] 产品概述（背景/目标/范围）
- [ ] 完整的用例图
- [ ] 每个用例包含前置/后置条件
- [ ] 基本事件流和备选事件流
- [ ] 数据字典/字段定义
- [ ] 交互流程图
- [ ] 时序图
- [ ] UI布局设计
- [ ] 非功能需求（性能/安全/兼容）
- [ ] 术语表
