# ima-skill 知识库集成指南

## 安装与初始化

### 安装步骤
```
1. 下载 ima-skill：
   https://app-dl.ima.qq.com/skills/ima-skills-1.1.2.zip

2. 解压后按照包内说明安装到 Cursor skills 目录

3. 获取 API Key：
   https://ima.qq.com/agent-interface
   登录后在「我的 Agent」→「API Key 管理」中创建

4. 在 Cursor 中配置 API Key：
   打开 ima-skill 的配置文件，填入 API Key
```

### 验证安装
安装成功后，可通过以下方式验证：
- 在 Cursor 中调用 `ima-skill`，能正常检索笔记即为成功
- 若报错「API Key 无效」，请重新检查 Key 是否正确填写

## 知识库检索（每步开始时执行）

### 检索模板
在每个运营步骤开始前，先调用 ima-skill 检索相关已有内容：

```
检索指令：在 ima 知识库中搜索与「[当前步骤主题]」相关的笔记和资料
```

### 各步骤检索关键词

| 步骤 | 检索关键词示例 |
|------|--------------|
| Step 1 行业分析 | `[行业名] 分析报告` / `[行业名] 用户画像` / `[行业名] 关键词` |
| Step 2 竞品分析 | `[竞品名] 账号分析` / `[行业] 竞品对比` / `爆款拆解` |
| Step 3 账号养号 | `养号策略` / `冷启动` / `[平台] 账号运营` |
| Step 4 内容创作 | `[行业] 爆款标题` / `二次创作` / `违禁词` / `内容日历` |
| Step 5 互动钩子 | `互动话术` / `私域引导` / `评论钩子` / `用户唤醒` |

### 检索结果使用原则
- 若找到相关历史报告 → 作为基础，增量更新而非重新开始
- 若找到相关话术/模板 → 直接复用并结合新客户定制
- 若找到竞品历史数据 → 与当前数据对比，看趋势变化
- 若无相关内容 → 正常执行，完成后建议沉淀

## 知识沉淀（每步完成后询问）

### 沉淀时机判断
每个步骤完成后，判断以下内容是否值得沉淀：

**高价值沉淀内容**（建议主动询问）：
- 行业分析报告（含关键词体系、用户画像）
- 竞品账号详细分析（含爆款拆解）
- 某行业的违禁词积累
- 效果好的话术和钩子模板
- 内容创作模板和标题公式

**一般价值内容**（可选沉淀）：
- 养号每日计划（通用性强，不必每次保存）
- 临时的搜索数据（时效性强）

### 询问话术
```
本步骤已完成。以下内容是否需要保存到 ima 知识库？

📄 [内容类型]：[内容简述]

选项：
A. 创建新笔记保存
B. 追加到已有笔记「[笔记名]」
C. 暂不保存
```

### 笔记命名规范
```
格式：[客户/行业]-[内容类型]-[日期]
示例：
- 护肤品行业-行业分析报告-2026Q1
- XX品牌-竞品账号分析-2026-03
- 通用-小红书违禁词库-持续更新
- 通用-互动话术模板库-v2
```
