# Examples

## 1) Minimal example TOML

The full sample is in `examples/example_network.toml`.

## 2) Run all scenarios

From the skill directory (or set your working directory accordingly):

```bash
python scripts/run_fluid_network.py --input examples/example_network.toml --format markdown
```

## 3) Run one scenario

```bash
python scripts/run_fluid_network.py --input examples/example_network.toml --scenario normal --format markdown
```
