# 股票分析专家智能体

专业股票分析师 AI，提供实时行情分析、技术指标解读、潜力股筛选和投资建议。

---

## 快速开始

### 1. 环境变量配置

```bash
# Tushare Token（已配置）
[Environment]::SetEnvironmentVariable("TUSHARE_TOKEN", "abfa8a1c06b30afd16dbe62e0c656dc769f4c56280d7c686556761b2", "User")

# Finnhub Token（可选）
[Environment]::SetEnvironmentVariable("FINNHUB_TOKEN", "your_token", "User")
```

### 2. 安装依赖

```bash
uv pip install pandas numpy requests --system
```

### 3. 使用示例

```bash
# 测试技能
python analyzer.py
```


```bash
openclaw cron add "30 15 * * 1-5" "分析今日市场表现"
```
