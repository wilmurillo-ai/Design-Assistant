## Gyakori szabályok

### 1. Egybeírás

```
Igei előtag + ige = egybe
  hozzáad, beleír, rámutat, elvisz

Határozószó + ige = külön
  oda megy, le ír, fel áll
```

### 2. Toldalékok

```
Birtokos személyjelek:
  házam, házad, háza, házunk, házatok, házuk

Többes szám:
  ház, házak

Esetjelek:
  ház (alany)
  házat (tárgyas -t)
  háznak (részes -nak/-nek)
  házban (inesszív -ban/-ben)
  házból (elativ -ból/-ből)
  házhoz (sublativ -hoz/-hez/-höz)
  háznál (adesszív -nál/-nél)
  házig (terminativ -ig)
```

### 3. Hangrendi illeszkedés

```
Mély hangrend: a, á, o, ó, u, ú → -ban, -nak, -hoz
Magas hangrend: e, é, i, í, ö, ő, ü, ű → -ben, -nek, -hez

Vegyes: illeszkedik az utolsó magánhangzóhoz
  ablak (a→o) → ablakban (mély)
  kert (e) → kertben (magas)
```

### 4. Deflexion (igei ragozás)

```
Határozatlan ragozás (általános):
  olvasok, olvasz, olvas, olvasunk, olvastok, olvasnak

Határozott ragozás (konkrét tárgy):
  olvasom, olvasod, olvassa, olvassuk, olvassátok, olvassák
```

### 5. Kötőjel használat

```
- Idegen tulajdonnevek: Shakespeare-dráma
- Betűszó + közszó: TCP/IP-protokoll
- Szám + főnév: 10-es szám
- Hasonlóság: New York-i
```
