## Stílus ajánlások

### 1. Rövid mondatok

- **Hosszú**: "Annak ellenére hogy eredetileg azt terveztük hogy a találkozót kedden tartjuk meg, végül úgy döntöttünk hogy szerdára halasztjuk."
- **Rövid**: "Eredetileg kedden terveztük a találkozót. Végül szerdára halasztottuk."

### 2. Aktív szerkezet

- **Passzív**: "A kód meg lett írva."
- **Aktív**: "Megírtam a kódot."

### 3. Konkrét kifejezések

- **Homályos**: "Valami probléma van a rendszerrel."
- **Konkrét**: "A rendszer lelassult a memóriahiány miatt."

### 4. Kerüld a töltelékszavakat

- **Töltelék**: "Úgy gondolom hogy talán esetleg lehet hogy..."
- **Tömör**: "Azt gondolom..." / "Lehetséges..."

## Gyakorlati minták

### Email / Üzenet minták

```
Kedves [Név]!

A [projekt/feladat] kapcsán a következőket szeretném jelezni:
[konkrét üzenet]

Kérdés esetén állok rendelkezésre.

Üdvözlettel,
[Név]
```

### Technikai leírás

```
A modul a következőket végzi:
1. Adatolvasás a forrásból
2. Feldolgozás a szabályok alapján
3. Eredmény írása a kimenetbe

Hibakezelés: ha a forrás nem elérhető, hibaüzenetet küld.
```

### Hibajelentés

```
Hiba: [rövid leírás]
Lépések:
1. [első lépés]
2. [második lépés]
3. [hiba megjelenik]

Várt eredmény: [mi lett volna helyes]
Tényleges eredmény: [mi történt valójában]
```

## Jegyzetek

- **Olvasd hangosan**: ha furcsán hangzik, valószínűleg hibás
- **Kérj visszajelzést**: mások is olvassák át
- **Gyakorold**: napi írás javítja a stílust
- **Olj jó magyar szövegeket**: irodalom, minőségi újságírás
