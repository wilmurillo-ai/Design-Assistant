1. 基础速查
1.1 天干五行与阴阳

天干对应五行：甲乙木、丙丁火、戊己土、庚辛金、壬癸水。

阴阳属性：奇数干为阳（甲、丙、戊、庚、壬），偶数干为阴（乙、丁、己、辛、癸）。

白话解释：天干体现外在的行为和性格，例如木象征生长与发散，火象征热情与扩散，土象征稳定与承载，金象征决断与规则，水象征流动与智慧。

1.2 地支五行与阴阳

地支对应五行：子（阳水）、丑（阴土）、寅（阳木）、卯（阴木）、辰（阳土）、巳（阴火）、午（阳火）、未（阴土）、申（阳金）、酉（阴金）、戌（阳土）、亥（阴水）。

白话解释：地支更代表时间节点、环境和具体事件。例如申酉金代表规则、决断、改革；寅卯木代表生发、开拓、协作。

1.3 地支藏干

每个地支中包含不同的天干，称为藏干，用来辅助判断五行强弱和十神。常见搭配：子藏癸，丑藏己癸辛，寅藏甲丙戊，卯藏乙，辰藏戊乙癸，巳藏丙戊庚，午藏丁己，未藏己丁乙，申藏庚壬戊，酉藏辛，戌藏戊辛丁，亥藏壬甲。

白话解释：藏干体现内在动机和潜藏的能量，如寅中藏甲代表潜在的木本能，申中藏庚体现内在的金属性。

1.4 五行生克

相生关系：木生火、火生土、土生金、金生水、水生木。可以理解为支持和滋养。

相克关系：木克土、土克水、水克火、火克金、金克木。可以理解为约束和消耗。

白话解释：生克关系帮助分析十神之间的互动。例如日主被克意味着承担压力，被生则代表得到支持。

2. 十神解释库

十神指以日主五行与其它天干的生克关系配合阴阳属性形成的十种角色，用于分析性格、行事风格与可能的事件倾向。

比肩：同元素同阴阳；象征伙伴、兄弟、自我意识。强调独立性与竞争心。

劫财：同元素异阴阳；象征竞争者、朋友。可能有夺财、竞争或合作关系。

正财：日主克同阴阳的元素；象征稳定的资源、财务、实际所得。与现实劳动回报相关。

偏财：日主克异阴阳的元素；象征流动财富、机会、投资、偏门收益。与冒险和开拓有关。

正官：克制日主且异阴阳；象征正当的权力、秩序、责任与法律。FateMaster网站指出正官代表合法的权力与责任，性格特点包括责任感强、注重规则和组织能力。

七杀（偏官）：克制日主且同阴阳；象征挑战、压力、竞争与突破。适度的七杀可激励人发展，过强则压力过大。

正印：生助日主且异阴阳；象征正统教育、知识、保护与助力。

偏印（枭神）：生助日主且同阴阳；象征异性或非正统的支持、学习、想象力与创意。

2.2 正官 / 七杀（权力、压力与责任）

含义与特征：正官代表合法权力、社会地位和责任感。有资料指出正官格局的人具有强烈的责任心、遵守规则、擅长管理与组织。七杀代表压力与挑战，可能带来竞争和冲动。

常见表现：正官旺者重视秩序与规则，容易获取职位与信誉；七杀旺者勇敢果断，易涉风险、突破常规。

风险点：正官过强可能固执守旧，缺乏灵活性；七杀过旺易招惹是非或压力过大。

建议：正官偏强者应适度放宽标准、学习灵活变通；正官偏弱者可通过遵循规章、树立公信力提升运势。七杀显著者宜合理调节压力，通过计划化方式应对挑战。

2.3 正财 / 偏财（资源与收益）

含义与特征：正财象征稳定收入、责任感和实际财富；偏财代表偏门收益、投资机会和冒险精神。

常见表现：正财旺者勤劳、务实、重视积累；偏财旺者敢于投资、有商业直觉、性格较外向。

风险点与建议：正财过旺可能保守、固执；宜适度学习理财与创新。偏财过旺可能冒进、风险过大；建议稳健理财、避免贪心。

2.4 食神 / 伤官（表达与才华）

含义与特征：食神象征创意、表达、享乐与口福，相关资料中指出其代表才华与享受；伤官代表表达欲强、挑战权威、创意外泄。

常见表现：食神旺者乐观开朗、擅长创造与表达；伤官旺者思维敏捷、善于突破旧框架，但也容易犯口舌是非。

建议：食神旺者可发展写作、艺术等领域；伤官旺者宜注意沟通方式，避免与权威对立。

2.5 正印 / 偏印（保护与学习）

含义与特征：正印象征传统学习、贵人助力和保护；偏印（枭神）象征灵感、创造、非正统知识。

常见表现：正印旺者学习力强、谨慎、重视精神层面；偏印旺者富于想象与灵性，但易陷入主观幻想。

建议：正印旺者应兼顾实践，避免过度保守；偏印旺者应落地行动，避免空想。

2.6 比肩 / 劫财（伙伴与竞争）

含义与特征：比肩代表同类同阵营的伙伴、自我意识、独立性；劫财代表异性力量的竞争或合作，有时意味着利益分享。

常见表现：比肩旺者性格自主、固执；劫财旺者善于合伙与人际互动，但也容易有竞争与财务压力。

建议：比肩旺者注意沟通合作；劫财旺者适度防范利益纠纷。

3. 身强弱与用神

身强弱判断和用神选择是八字分析的核心。以下内容仅提供一般思路，具体判断需结合盘面。

3.1 身强身弱的判断框架

月令力量：月支为一柱之根，若与日主同五行且得令，则日主易强。

通根与透干：日主在其他支藏干有根，或有比肩劫财透出，则有助增强日主；反之受克受制则弱。

十神平衡：食伤财官印的多少反映日主所受的生克泄耗，需综合评价。

白话解释：身强者本身能量充沛，抵抗外界冲克；身弱者则易受外界影响，需要依赖用神扶助。

3.2 身强倾向

表现：自信、主观性强、行动力强、喜欢主导。

风险点：易固执己见、不听建议。

建议：适度听取他人意见，合理安排精力，避免冲动。

3.3 身弱倾向

表现：缺乏安全感、易受外界影响、决策缓慢。

建议：增强自信，结交有助力的伙伴或贵人，强调学习与积累。

3.4 用神选择简述

选取思路：身强者多用克、泄、耗的五行（财官食伤）平衡；身弱者多用生助的五行（比劫印枭）扶助。具体需依据盘面季节、五行偏旺或偏枯来调整。

表达模板：用神出现时可描述为“有贵人扶助”“得到资源支持”；忌神出现时可描述为“需注意压力”“提防耗泄”。

4. 大运与流年

大运代表十年的背景环境，流年代表当年的具体事件与触发点。分析时先看大运，再看流年，最后结合原局关系。

4.1 大运总原则

大运干支显示一个十年的主线和环境。观察运干与运支对日主的生克关系、冲合刑害破以及与原局四柱的互动。

描述大运时可提炼3~5个关键词，如“学习、变动、机遇”或“压力、挑战、重塑”等；并给出适宜的行动策略和应避风险。

4.2 流年总原则

流年干支揭示某年主题。先看年支与大运支、原局支的冲合刑害破，再看年干与原局干的生克关系及十神。岁干岁支的力量主要用于解释具体事件的起因和性质。

叙述流年时可列出1~3条重点事件倾向，并给出实用建议，如“先稳再变”“注意健康”等。

4.3 岁运并临

含义：大运天干地支与流年天干地支完全相同，如壬寅大运遇到壬寅流年。华易网指出，岁运并临的年份被认为带来特殊运势，既可能出现重大机遇，也可能伴随挑战。

影响：当大运中的神为用神时并临可带来好运，如财星为喜用，则易有财运；若大运中的神为凶神，则需谨慎，可能带来压力或突发事件。

建议：在岁运并临之年需保持谨慎，抓住正面机会，同时做好应对计划，避免过度冒进。

4.4 岁运相冲

含义：大运地支与流年地支六冲，例如卯酉相冲。六冲体现对立、排斥和变化。指出，地支六冲代表变动、奔波与不安定。

常见表现：遇到六冲时，容易搬家、换工作、外出等，感情和事业会出现变化和冲击。

建议：适时调整心态，主动规划变动，避免僵硬抵抗。若原局有驿马星再逢冲，更宜预备外出或旅行机会。

4.5 日犯岁君

含义：指日柱日干与年柱年干互克，如日干克年干。华易网指出，这种情况需要结合其它柱看是否化解。

建议：如果日犯岁君，但命局有其他有利因素（用神或贵人），可化解不利；否则需谨慎处理人际关系，避免与长辈或权威冲突。

4.6 冲合刑害破的解释（六冲为主）

六冲：十二地支有六对互冲，分别是子午、丑未、寅申、卯酉、辰戌、巳亥。解释，六冲代表对立和排斥，形成强烈的冲击与动荡。

含义与表现：命局中六冲过多的人，性格好动、难安稳，运势反复，容易经历迁移、出差或感情破裂。

建议：逢冲则动，主动调整，利用动中求变；同时注意健康和安全，避免冲动行事。

4.7 犯太岁（值、刑、害、破）简介

值太岁：流年支与年柱支相同，说明今年可能聚焦家庭、祖辈或长辈事务。

冲太岁：流年支与年柱支六冲，主题围绕变动与突破。

刑太岁：刑代表不协调关系，需注意官非纠纷。

害太岁：害代表暗中损害或小人纠缠。

破太岁：破代表破坏、损失，需谨慎财务和关系。

建议：根据具体关系调节心态，适度调整计划，必要时求助专业人士。

5. 事业与财运建议
5.1 总原则

分析事业时需兼顾行业属性与十神主题。例如财星旺者宜从事财务、销售、商业等；官杀旺者适合管理、行政、公务等；食伤旺者适合创意、表达、教育等。应结合大运流年看机会与压力。

在不确定强弱时，保守建议以稳定为主，必要时先充电学习，再谋求突破。

5.2 常用事业信号

官杀显著：责任、考核增多，适合争取职位升迁；注意规避违法违规风险。

财星旺：资源增加或收益机会多，可规划投资与财务管理；避免过度贪婪。

食伤旺：创意和表达活跃，适合推广、教学、艺术，但需注意口舌是非。

印星旺：学习力和贵人助力增强，适合深造或做幕后支持工作。

比劫旺：竞争激烈，适合合作共赢，但需注意资源分配与互信。

5.3 财运词条

正财：稳定薪资收入与正当财源，强调责任与勤恳。建议稳中求进，合理理财。

偏财：投资或外财机会，宜审慎评估风险，避免赌博式行为。

合伙/分利：劫财或比肩与财星互动时，容易牵涉合作与利益分配，建议明确合同并做好风险管理。

是否适合换工作：先看大运流年是否为冲合变动之年；若岁运相冲或食伤流年，可考虑主动调整；否则建议积累实力再动。

8. 格局与结构

格局是指命局中某种组合达到一定条件时，五行力量偏向稳定或特殊结构，如“从格”“专旺格”等。由于流派众多，此处仅罗列常见格局解释，具体应用需谨慎。

8.1 常见格局判断口径

专旺格：某一五行极旺且其它五行极弱，日主顺从旺五行。若得生扶则吉，受克则凶。

从格：日主极弱无法承载五行时，顺从旺五行，如从财、从官格等。重点看日主是否完全不受根生。

伤官佩印：伤官旺而有印星化解，适合文艺、科研和言语表达。

8.2 常用格局条目（示例）

正官格：日主身旺，官星透出且有印星保护，忌伤官破格。表现为责任心强、名誉地位佳。

七杀格：七杀旺而有食神或印星化解，适合武职或压力激励型工作。忌七杀无制化，易凶险。

从财格：财星极旺日主极弱，顺从财富；适合经商、理财，但需防破财与健康耗损。

9. 统一话术模板
9.1 大运总结句式

“这一运以【关键词1】【关键词2】为主，意味着【简要解释】。建议【行动策略】，注意【风险提示】。”

9.2 流年总结句式

“今年主题是【重点1】与【重点2】；可能会发生【事件或变化】；建议【应对措施】。”

9.3 建议句模板

做：用动词短句表达，如“多积累资源”“主动学习”“增强沟通”。

不做：用否定句表达，如“避免过度冒进”“不要盲目投资”。

顺序：如果有先后次序，提示“先…再…”。
