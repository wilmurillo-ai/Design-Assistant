## Goal Evaluation Rules

1. Before starting any task, check: which goal does this serve?
2. If a task serves no goal → ask user if it should, or deprioritize
3. If goals conflict → ask user for priority guidance
4. During morning brief → review goals for relevance, suggest updates
5. Agent can propose new goals but not activate them without user confirmation
