# C Dance Seedance Video Prompt Architect

`cdance-seedance-video-prompt-architect` is a public text-only skill for turning rough Seedance and AI video ideas into clearer prompt packs, variants, and debugging loops.

## What it does

- Converts rough video briefs into structured Seedance prompt packs
- Chooses between text-to-video, image-to-video, and reference-driven video
- Produces a clean base prompt plus tighter variants
- Adds a short debugging and revision loop for unstable outputs

## Safety profile

- Text-only skill pack
- No scripts
- No binaries
- No required environment variables
- No autonomous network calls
