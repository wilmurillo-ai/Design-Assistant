# Seedance Video Prompt Architect

`seedance-video-prompt-architect` is a public prompt-design skill for turning rough video ideas into tighter Seedance 2.0 prompt packs, variants, and debugging loops.

- Prompt debugging guide: https://cdance.ai/blog/common-seedance-2-0-prompt-mistakes

## What it does

- Converts rough creative briefs into structured Seedance 2.0 prompts
- Chooses between text-to-video, image-to-video, and video-to-video
- Produces a clean base prompt plus tighter variants
- Adds a short debugging and revision loop for unstable outputs

## Safety profile

- Text-only skill pack
- No scripts
- No binaries
- No required environment variables
- No autonomous network calls

## Best fit

Use this skill for product ads, fashion clips, cinematic scenes, anime-style shots, creator content, and other short-form video ideas that need stronger motion structure and cleaner prompt control.
