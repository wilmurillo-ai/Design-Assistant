# Happy Horse Video Workflow Architect

`happy-horse-video-workflow-architect` is a public text-only skill for turning rough Happy Horse use cases into structured workflow plans, prompt packs, and revision loops.

## What it does

- Converts rough Happy Horse use cases into a clearer workflow
- Chooses between text-to-video, image-to-video, and reference-driven video
- Produces a clean base prompt plus tighter variants
- Adds a short testing and revision loop

## Safety profile

- Text-only skill pack
- No scripts
- No binaries
- No required environment variables
- No autonomous network calls
