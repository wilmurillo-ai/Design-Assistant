# My Research Lab 我的研究院

> Your personal research lab — 用户个人感兴趣的研究方向的专属私人研究院系统。

## 这是什么

一个**真正自动运转、自进化**的个人智库。

- 启动时自动创建定时任务，每天定时采集、质检、推送，不需要你盯着
- 深度脑爆 + 动手产出（建议制，你确认了才做）
- 在合适时机建议你写文章发公众号/小红书
- 每一次反馈都让研究院更懂你

启动方式：告诉院长你的研究方向，一个词就够。

## 核心能力

| 能力 | 说明 |
|------|------|
| 定时自动采集 | 冷启动自动创建 cron 任务，每天定时跑 |
| 渠道检测 | 启动时自动检测可用的推送渠道，缺什么引导配什么 |
| 内容质检 | 综合评分 ≥ 3.5 才入选日报 |
| 建议制动手 | 先建议再执行，可设为自动或每次确认 |
| 内容创作 | 建议写文章，支持公众号/小红书/邮件 |
| 自动更新 | 每次运行检查新版本 |
| 健康检查 | 每次对话时检查定时任务是否在跑 |
| 云端兼容 | 无桌面Chrome时自动降级为 web_search + web_fetch |

## 安装

```bash
npx clawhub@latest install my-research-lab
```

## 快速启动

```
"研究 AI 公益"
"帮我建一个美食研究院"
```

院长只问一个问题：**你的研究方向是什么？** 然后自动配置信源、检测渠道、创建定时任务、执行第一次采集。
