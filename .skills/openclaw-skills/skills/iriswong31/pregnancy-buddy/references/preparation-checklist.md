# 待产包与孕期物品清单

> 综合来源：小红书/知乎/公众号高赞经验帖交叉对比，标注"必备"为多个来源共同推荐
> 💡 建议准备时间：孕32-36周开始准备，36周前全部就绪

## 入院待产包

### 📋 简洁版（必备清单）

> 只买这些就够了，不踩坑不浪费

#### 妈妈用品
| 物品 | 数量 | 说明 |
|------|------|------|
| 产褥垫（大号60x90） | 10-20张 | 顺产4-6张够，剖宫产多备。检查、生产、术后都要用 |
| 卫生巾（计量型/产妇专用） | 2-3包 | 产后恶露用，夜用加长型也可以替代 |
| 一次性内裤 | 10-15条 | 产后恶露期省得洗，买大一码 |
| 哺乳内衣 | 2-3件 | 前开扣式，方便喂奶 |
| 月子服/睡衣 | 2套 | 开襟式方便哺乳，纯棉透气 |
| 拖鞋 | 1双 | 防滑、包跟的（月子里不建议露脚跟） |
| 吸管杯/弯头吸管 | 1个 | 产后躺着喝水神器 |
| 毛巾 | 2-3条 | 擦汗、擦身用 |
| 充电器+充电线 | 1套 | 别忘了这个！住院必需 |

| 物品 | 数量 | 说明 |

| 婴儿帽 | 1-2顶 | 出院戴，保暖 |

#### 证件资料
| 物品 | 说明 |
|------|------|

## 孕期分阶段物品准备

### 孕早期（1-12周）
| 物品 | 优先级 | 说明 |
|------|--------|------|
| 叶酸/复合维生素 | ⭐⭐⭐ | 越早吃越好 |
| 孕妇专用护肤品 | ⭐⭐ | 停用含A酸/水杨酸的产品 |
| 防吐小零食 | ⭐⭐ | 苏打饼干、酸梅、姜糖 |
| 舒适内衣（无钢圈） | ⭐⭐ | 乳房开始变大 |

### 孕中期（13-27周）
| 物品 | 优先级 | 说明 |
|------|--------|------|
| 孕妇装 | ⭐⭐⭐ | 肚子开始显怀 |
| 孕妇枕 | ⭐⭐⭐ | U型或C型，大幅改善睡眠质量 |
| 妊娠纹霜/油 | ⭐⭐ | 预防效果因人而异，但抹了心里踏实 |
| 平底鞋 | ⭐⭐ | 重心变了，穿平底更安全 |
| DHA补充剂 | ⭐⭐ | 如果鱼吃得少建议补 |
| 胎教音乐/故事书 | ⭐ | 23周后宝宝能听到了 |

### 孕晚期（28-40周）
| 物品 | 优先级 | 说明 |
|------|--------|------|
| 待产包（参考上方清单） | ⭐⭐⭐ | 36周前准备好 |
| 婴儿床/床中床 | ⭐⭐⭐ | 提前买好通风去味 |
| 婴儿推车 | ⭐⭐ | 可以等宝宝出生后再买，不着急 |
| 安全座椅 | ⭐⭐ | 出院就需要（如果开车回家） |
| 哺乳椅/靠垫 | ⭐⭐ | 月子里喂奶用 |
| 吸奶器 | ⭐⭐ | 建议开奶后再决定是否需要 |

| 婴儿浴盆 | ⭐⭐ | 提前准备好 |
| 恒温水壶 | ⭐⭐ | 冲奶粉用（设定40-45度） |

## 不建议买 / 鸡肋物品 ⚠️

| 物品 | 为什么不推荐 |
|------|-------------|
| 婴儿专用水温计 | 用手肘试水温就行，古法最管用 |
| 大量新生儿衣服 | 宝宝长太快了，每个尺码最多4-5套 |
| 定型枕 | 3个月内不需要枕头，且有窒息风险 |
| NB码大量囤纸尿裤 | 有些宝宝出生就能穿S码 |
| 婴儿手套 | 影响触觉发育，勤剪指甲更好 |
| 学步车 | 不利于正确学步姿势 |
| 大型消毒柜 | 普通锅煮沸消毒就够了 |
| 过多奶瓶 | 先看喂养方式再决定 |
| 高端婴儿秤 | 医院和体检时会称，不需要天天称 |

## 打包小贴士

### 打包方法
1. **分袋打包**：妈妈的一袋、宝宝的一袋、证件一袋、爸爸的一袋
2. **标签标注**：每袋外面贴个标签写清内容，紧急时好找
3. **放在显眼位置**：门口玄关最好，拎起就走
4. **让老公也知道里面有什么**：关键时刻主力可能是他！

### 准备时间建议
| 时间 | 准备内容 |
|------|---------|
| 32周 | 开始列清单、网购基础物品 |
| 34周 | 洗晒宝宝衣物、收纳整理 |
| 35-36周 | 完成打包、放到位、检查一遍 |
| 37周 | 确认证件齐全、确认去医院的路线 |
| 随时 | 保持手机满电！确认家人随时待命 |
