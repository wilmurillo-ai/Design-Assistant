# 孕期关键时间线

> 以预产期为锚点，反推整个孕期40周的关键节点
> 使用方法：Skill 获取用户的预产期或末次月经日期后，自动计算当前孕周，匹配对应阶段的提醒

---

## 时间线总览

```
孕早期(1-12周) ──→ 孕中期(13-27周) ──→ 孕晚期(28-40周)
   │                    │                    │
   ├─ 确认怀孕          ├─ 唐筛/无创DNA      ├─ 胎心监护
   ├─ 补叶酸            ├─ 大排畸(重要!)      ├─ GBS筛查
   ├─ 建档              ├─ 糖耐(重要!)        ├─ 准备待产包
   ├─ NT检查            ├─ 四维彩超           ├─ 确认分娩医院
   └─ 注意保胎          └─ 胎动监测开始       └─ 随时待发动
```

---

## 详细时间线

### 📌 孕4-5周：确认怀孕
- **要做的事**：
  - 用验孕棒确认（两条杠！）
  - 开始吃叶酸（如果之前没吃的话，现在开始也不晚）
  - 预约第一次产检
- **注意事项**：
  - 停止饮酒
  - 避免接触有毒化学物质
  - 告知医生正在服用的所有药物
- **情绪提醒**：刚确认怀孕可能又兴奋又紧张，这很正常！

### 📌 孕6-8周：第一次B超
- **要做的事**：
  - 去医院做第一次B超，确认宫内妊娠
  - 听宝宝心跳（可能是最感动的时刻！）
  - 了解建档所需材料
- **检查项目**：B超（确认胎心胎芽）、血HCG、孕酮
- **注意事项**：
  - 孕吐可能开始，别硬扛，吃不下就少量多餐
  - 充分休息，避免剧烈运动
  - 避免接触猫砂（弓形虫风险）

### 📌 孕8-12周：建档 + 全面体检
- **要做的事**：
  - 选择分娩医院，办理建档（有些热门医院需要抢号！）
  - 完成全套建档检查（血常规、肝肾功能、传染病筛查等）
  - 领取母子健康手册
- **检查项目**：血常规、尿常规、肝肾功能、血型、乙肝、HIV、梅毒、甲功、TORCH等
- **注意事项**：
  - 建档检查需要空腹，早上去
  - 带齐证件（身份证、结婚证等）
  - 了解产检费用和医保报销政策

### 📌 孕11-13+6周：NT检查 ⚡重要
- **要做的事**：
  - 做NT检查（颈项透明层超声）
  - 这是第一次重要的排畸筛查
- **检查项目**：NT超声
- **注意事项**：
  - ⚠️ 有严格时间窗口！错过就做不了
  - 提前预约，部分医院需要排期
  - NT正常值 <2.5mm

### 📌 孕13周：进入孕中期 🎉
- **里程碑**：度过了最不稳定的孕早期！
- **要做的事**：
  - 可以开始告诉亲朋好友好消息了
  - 开始关注孕妇装（肚子开始显了）
  - 可以开始适度运动（散步、孕妇瑜伽）
- **情绪提醒**：孕吐通常在这个时候减轻，精力恢复，是"蜜月期"的开始

### 📌 孕15-20周：唐氏筛查
- **要做的事**：
  - 做唐氏筛查（血清学筛查）或直接做无创DNA（NIPT）
  - 最佳时间：16-18周
- **检查项目**：唐筛（抽血）或 NIPT
- **注意事项**：
  - 唐筛需要空腹
  - 高龄孕妇（≥35岁）建议直接做NIPT或羊水穿刺
  - 结果"高风险"不要恐慌，进一步检查确认

### 📌 孕16-20周：可能感受到胎动
- **里程碑**：第一次感受到宝宝在动！
- **描述**：初产妇通常在18-20周感受到（像蝴蝶扇翅膀或小鱼冒泡）
- **要做的事**：
  - 记录第一次胎动的日期（写在手册上！）
  - 开始感受和宝宝的互动
- **情绪提醒**：这是很多妈妈觉得"宝宝真的存在"的时刻，会非常感动 ❤️

### 📌 孕20-24周：大排畸 ⚡⚡最重要的检查之一
- **要做的事**：
  - 做系统超声检查（大排畸/四维彩超）
  - 详细检查宝宝全身结构
- **检查项目**：系统超声/四维彩超
- **注意事项**：
  - ⚠️ 有严格时间窗口！必须提前预约
  - 检查前可以吃点甜食，让宝宝活跃一些配合检查
  - 如果宝宝不配合，可能需要走动后重做
  - 可以拍到宝宝的"人生第一张照片"（四维照片）

### 📌 孕24-28周：糖耐量检查 ⚡重要
- **要做的事**：
  - 做OGTT（口服葡萄糖耐量试验）
  - 筛查妊娠期糖尿病
- **检查项目**：OGTT（喝糖水后测三次血糖）
- **注意事项**：
  - 需要空腹（前一晚10点后禁食）
  - 检查前3天正常饮食，不要故意少吃
  - 糖水很甜——可以加冰块或柠檬汁
  - 带本书或手机，等待时间约2-3小时

### 📌 孕24周：开始数胎动
- **要做的事**：
  - 学会数胎动的方法
  - 建立固定的数胎动习惯
- **方法**：每天选一个固定时间段（如晚饭后），数1小时内胎动次数。≥3次为正常
- **注意事项**：
  - 如果2小时内胎动<6次，需要及时就医
  - 宝宝有自己的活跃周期，偶尔安静是正常的
  - 不要因为"数"而焦虑，感受大致规律就好

### 📌 孕28周：进入孕晚期
- **里程碑**：进入最后一个阶段！
- **变化**：
  - 产检变为每2周一次
  - 开始做常规胎心监护
- **要做的事**：
  - 了解分娩方式（顺产/剖宫产）的利弊
  - 开始了解分娩知识（呼吸法等）
  - 可以开始上孕妇课堂/分娩课

### 📌 孕30-32周：孕晚期B超 + 开始准备
- **要做的事**：
  - 做孕晚期B超（评估胎位、胎盘、羊水）
  - 如果胎位不正（臀位），医生可能建议矫正操
- **开始准备**：
  - 列待产包清单，开始网购
  - 预约月嫂（好月嫂要提前2-3个月！）
  - 了解母乳喂养知识
- **检查项目**：B超、血常规、尿常规

### 📌 孕32-34周：准备待产包
- **要做的事**：
  - 完成待产包采购和打包（参考待产包清单）
  - 洗晒宝宝衣物（新衣服都要先洗！）
  - 安装婴儿安全座椅（如果有车）
  - 确认去医院的路线（白天和夜间各规划一条）
- **注意事项**：
  - 不要坐长途车或飞机（一般32周后航空公司会限制）
  - 假性宫缩可能增多（不规律、不加剧=假的）

### 📌 孕35-37周：GBS筛查 + 最终准备
- **要做的事**：
  - 做GBS（B族链球菌）筛查
  - 产检变为每周一次
  - 最后一次全面评估（B超+胎心监护）
- **检查项目**：GBS、B超、胎心监护、血常规
- **准备事项**：
  - 待产包放在门口随时拎走
  - 确认家人联系方式和分工
  - 了解何时去医院（见红/破水/规律宫缩）
  - 手机保持满电、随身携带

### 📌 孕37-40周：随时准备 🎉
- **里程碑**：37周起宝宝已是足月！随时可能出生
- **关注信号**：
  - 🔴 **破水**：大量清亮液体流出 → 立即去医院
  - 🟠 **见红**：少量血性粘液 → 不急，但24-48小时内可能发动
  - 🟡 **规律宫缩**：每5分钟一次、每次持续60秒、持续1小时 → 去医院
- **日常建议**：
  - 适量散步有助于自然发动
  - 保持放松心态
  - 继续数胎动
  - 如果超过40周还没发动，医生会讨论催产方案

### 📌 产后即刻：新生命到来！
- **要做的事**：
  - 尽早母婴肌肤接触（金色一小时）
  - 尝试第一次母乳喂养
  - 新生儿筛查（足跟血、听力筛查）
  - 办理出生证明
- **证件办理**：
  | 证件 | 时限 | 在哪办 |
  |------|------|--------|
  | 出生医学证明 | 出生后尽快 | 分娩医院 |
  | 户口登记 | 1个月内 | 户籍所在派出所 |
  | 医保卡（新生儿） | 3个月内 | 社保中心 |
  | 疫苗接种证 | 出生时 | 分娩医院 |

## 提醒规则

Skill 应根据当前孕周，主动提醒以下内容：

### 提前提醒规则
| 事项 | 提前多久提醒 | 提醒频率 |
|------|------------|---------|
| NT检查 | 提前2周 | 周提醒 |
| 唐筛/NIPT | 提前2周 | 周提醒 |
| 大排畸预约 | 提前4周 | 周提醒直到预约成功 |
| 糖耐检查 | 提前2周 | 周提醒 |
| GBS筛查 | 提前1周 | 一次提醒 |
| 准备待产包 | 32周开始 | 周进度提醒 |
| 预约月嫂 | 24周开始 | 月提醒 |
| 学习分娩知识 | 28周开始 | 周推荐内容 |
| 数胎动 | 24周开始 | 每日提醒 |

### 语气规则
- 提醒不是催促，用轻松温暖的语气
- 例如："下周就到NT检查的最佳时间啦！记得提前预约哦~ 这是宝宝的第一次'面试'呢 😊"
- 避免制造焦虑，如果已经过了某个时间窗口，引导联系医生而非责备
