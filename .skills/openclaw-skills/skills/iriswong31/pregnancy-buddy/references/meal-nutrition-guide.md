# 孕期餐食营养指南

> 参考来源：《中国居民膳食指南（2022）》孕期妇女篇、协和医院营养科、WHO孕期营养建议
> ⚠️ 个人饮食请根据自身情况和医生建议调整

---

## 总体原则

1. **均衡为主**：不需要"一人吃两人份"，孕中期每天多增加300大卡、孕晚期多增加450大卡即可
2. **少量多餐**：每天5-6餐（3正餐+2-3加餐），避免暴饮暴食
3. **新鲜优先**：尽量吃新鲜烹制的食物，减少加工食品
4. **补充关键营养素**：叶酸、铁、钙、DHA是孕期四大"明星营养素"

---

## 孕早期（1-12周）：稳住别慌

### 核心营养目标
- **叶酸**：每天0.4-0.8mg（继续吃叶酸片！）
- 热量不需要额外增加，正常吃即可
- 应对孕吐为主，能吃下去就行

### 推荐食材
| 类别 | 推荐 | 说明 |
|------|------|------|
| 主食 | 全麦面包、小米粥、红薯、玉米 | 粗细搭配，B族维生素丰富 |
| 蛋白质 | 鸡蛋、鱼肉、豆腐、鸡胸肉 | 易消化的优质蛋白 |
| 蔬菜 | 菠菜、西兰花、番茄、胡萝卜 | 深色蔬菜叶酸含量高 |
| 水果 | 苹果、香蕉、葡萄、猕猴桃 | 补充维C和膳食纤维 |
| 坚果 | 核桃、腰果、杏仁（每天一小把） | 健脑+好脂肪 |

### 应对孕吐的饮食技巧
- 起床前先吃几块苏打饼干或面包干
- 少量多餐，不要空腹太久
- 姜茶、柠檬水可以缓解恶心
- 避免油腻和味道重的食物
- 实在吐得厉害吃不下→确保叶酸片能吃下就好，度过这段最重要

### 外卖推荐 🥡
- ✅ 清粥小菜、蒸饺、鱼肉馄饨、蔬菜三明治
- ✅ 日式便当（烤鱼+蒸蛋+蔬菜）
- ✅ 粤式蒸菜（清蒸鲈鱼、蒸水蛋）
- ❌ 避免：麻辣烫、烧烤、炸鸡、重口味火锅

## 孕中期（13-27周）：黄金营养期

### 核心营养目标
- **铁**：每天24-29mg（预防贫血！）
- **钙**：每天1000mg
- **DHA**：每天200mg
- 热量每天增加约300大卡（≈额外一杯酸奶+一把坚果）

### 推荐食材
| 类别 | 推荐 | 说明 |
|------|------|------|
| 补铁食材 | 牛肉、猪肝（每周1-2次）、鸭血、黑木耳 | 动物性铁吸收率高 |
| 补钙食材 | 牛奶（每天300-500ml）、酸奶、豆腐、小鱼干、芝麻 | 奶制品是最好的钙来源 |
| DHA来源 | 三文鱼、鲅鱼、沙丁鱼、海藻、亚麻籽 | 每周吃2-3次鱼 |
| 优质蛋白 | 鸡蛋（每天1-2个）、鸡肉、鱼虾、瘦猪肉 | 宝宝快速生长需要蛋白质 |
| 蔬菜 | 各种深色蔬菜为主，搭配浅色蔬菜 | 每天500g以上 |
| 水果 | 每天200-400g | 控制量，水果含糖高 |

### 一日食谱示例
| 餐次 | 内容 | 营养重点 |
|------|------|---------|
| 早餐 | 全麦面包+煎蛋+牛奶+几颗蓝莓 | 蛋白质+钙+维C |
| 上午加餐 | 一小把核桃+一个苹果 | 好脂肪+纤维 |
| 午餐 | 糙米饭+清蒸鱼+炒西兰花+紫菜蛋花汤 | DHA+铁+叶酸 |
| 下午加餐 | 酸奶+几颗红枣 | 钙+铁 |
| 晚餐 | 小米粥+牛肉炒芹菜+凉拌木耳+蒸南瓜 | 铁+膳食纤维 |
| 睡前 | 一杯温牛奶（饿的话） | 钙+助眠 |

### 外卖推荐 🥡
- ✅ **最佳选择**：
  - 蒸鱼/烤鱼套餐（不要太辣的）
  - 牛肉饭/牛腩饭（少油少盐版）
  - 鸡胸肉沙拉（加温蔬菜）
  - 日式三文鱼定食（三文鱼要全熟！）
  - 粤式烧腊饭（选烧鹅/烧鸭，少吃红烧肉）
  - 水饺/馄饨（荠菜鲜肉、虾仁）
- ✅ **凑合能吃**：
  - 麻辣香锅（少油少辣，多选蔬菜和豆腐）
  - 黄焖鸡米饭（要少油版）
  - 菌汤火锅（自选蔬菜和肉）
- ❌ **尽量避免**：
  - 炸鸡、汉堡（偶尔馋了吃一次没关系）
  - 重辣重油的菜（怕刺激肠胃）
  - 生鱼片、寿司里的生食
  - 奶茶（偶尔喝一杯解馋可以，但别当水喝）

### 外卖点餐小技巧
1. 备注"少油少盐"
2. 主食选粗粮饭/杂粮饭
3. 配一份蔬菜（很多外卖主菜蔬菜不够）
4. 自带一盒牛奶或酸奶补钙
5. 避免第一次吃的新店（卫生不确定）

## 孕晚期（28-40周）：冲刺储能

### 核心营养目标
- **热量**：每天增加约450大卡
- **铁**：每天29mg（血容量大增，更需要铁）
- **钙**：每天1000-1200mg
- **蛋白质**：每天增加30g
- 控制体重增长速度（每周0.3-0.5kg）

### 推荐食材
| 类别 | 推荐 | 说明 |
|------|------|------|
| 优质蛋白 | 鸡蛋、鱼、虾、瘦肉、豆制品 | 宝宝体重快速增加，蛋白需求大 |
| 补铁 | 猪肝、鸭血、红肉、黑芝麻 | 预防孕晚期贫血 |
| 补钙 | 牛奶、奶酪、豆腐、小鱼干 | 宝宝骨骼在硬化 |
| 润肠通便 | 红薯、燕麦、火龙果、酸奶 | 孕晚期便秘很常见 |
| 利水消肿 | 冬瓜、红豆、玉米须水 | 缓解水肿 |

### 特别注意
- **控制甜食**：孕晚期更容易血糖偏高
- **少吃多餐**：胃被子宫挤小了，一次吃太多会很撑
- **适量喝水**：每天1500-2000ml，不要因为怕跑厕所就少喝
- **补充维生素K**：多吃绿叶蔬菜，帮助宝宝凝血功能

## 必须避免的食物 🚫

### 绝对禁忌
| 食物 | 原因 | 风险等级 |
|------|------|---------|
| 酒精（任何含量） | 可能导致胎儿酒精综合征 | 🔴 严禁 |
| 生鱼片/寿司（生食） | 细菌和寄生虫风险 | 🔴 严禁 |
| 生肉/溏心蛋 | 细菌感染风险（沙门氏菌等） | 🔴 严禁 |
| 未消毒的奶制品 | 李斯特菌风险 | 🔴 严禁 |
| 含汞高的鱼（鲨鱼、旗鱼、金枪鱼大量） | 重金属影响神经发育 | 🔴 严禁 |

### 尽量少吃
| 食物 | 原因 | 建议 |
|------|------|------|
| 咖啡/茶 | 咖啡因 | 每天<200mg（约1杯小杯美式） |
| 冰激凌/甜品 | 高糖高脂 | 偶尔解馋可以 |
| 腌制食品 | 亚硝酸盐 | 偶尔吃一点没大问题 |
| 方便面 | 营养单一 | 偶尔吃一次可以，加个蛋加点菜 |
| 辣椒 | 刺激肠胃 | 不用完全戒，微辣可以 |

### 常见误区纠正
- ❌ "孕妇不能吃螃蟹" → ✅ 全熟的螃蟹可以吃，但别吃太多（高蛋白不好消化）
- ❌ "孕妇不能吃冰的" → ✅ 偶尔吃点冰的没事，不会导致流产
- ❌ "必须喝很多骨头汤" → ✅ 骨头汤含钙很少，主要是脂肪。喝牛奶补钙效率高得多
- ❌ "水果可以随便吃" → ✅ 水果含糖高，每天200-400g就够了
- ❌ "燕窝是孕期必备" → ✅ 燕窝蛋白质含量不如鸡蛋，不需要刻意花大价钱

## 关键营养素补充一览

| 营养素 | 每日需要量 | 最佳食物来源 | 是否需要吃补充剂 |
|--------|----------|-------------|----------------|
| 叶酸 | 0.4-0.8mg | 菠菜、西兰花、芦笋 | ✅ 建议整个孕期服用叶酸片 |
| 铁 | 24-29mg | 红肉、猪肝、鸭血 | 医生建议时补充（贫血者必须补） |
| 钙 | 1000-1200mg | 牛奶、酸奶、豆腐 | 饮食不足时补充钙片 |
| DHA | 200mg | 深海鱼、海藻 | 建议补充（鱼油或藻油） |
| 维生素D | 600IU | 日照、强化牛奶、鱼肝油 | 建议补充（尤其少晒太阳者） |
| 碘 | 230μg | 海带、紫菜、碘盐 | 一般饮食可满足 |
| 蛋白质 | 70-100g | 肉鱼蛋奶豆 | 饮食可满足 |

## 孕期体重管理

### 推荐增重范围

| 孕前BMI | 整个孕期建议增重 | 孕中晚期每周增重 |
|---------|----------------|----------------|
| <18.5（偏瘦） | 12.5-18kg | 0.44-0.58kg |
| 18.5-24.9（正常） | 11.5-16kg | 0.35-0.50kg |
| 25-29.9（超重） | 7-11.5kg | 0.23-0.33kg |
| ≥30（肥胖） | 5-9kg | 0.17-0.27kg |

### 增重分布
宝宝出生时约3-4kg，其余增重去了哪里？
- 胎盘：0.5-1kg
- 羊水：0.8-1kg
- 子宫增大：1kg
- 乳房增大：0.5-1kg
- 血液增加：1.5-2kg
- 体液增加：1.5-2kg
- 脂肪储备：2-4kg（产后会慢慢减掉的！）
