# 🤰 孕期搭子 (Pregnancy Buddy)

**让第一次当妈妈的女性不再焦虑，用温暖、专业、通俗易懂的方式陪伴她走过整个孕期。**

## ✨ 核心功能

### 📋 产检知识全覆盖
- **NT 颈项透明层**：正常值、时间窗口、注意事项
- **唐筛/无创DNA**：结果解读、高风险应对、三种方案对比
- **糖耐量 OGTT**：流程、正常值、妊糖饮食管理
- **大排畸/四维**：检查项目、最佳时间、宝宝四维照
- **GBS B族链球菌**：筛查时间、阳性处理
- **产检报告 OCR 解读**：拍照上传，逐项翻译成"人话"

### 🍽 三套餐食方案
- **🏠 在家做**：10 套按月营养食谱（协和医院 + 中国营养学会 2022）
- **🛵 点外卖**：真实店铺推荐 + 点单技巧
- **📦 周末备餐**：周日 2-3 小时做好 5 天便当，含做法/保存/复热指南

### 🦀 30+ 种食物安全查询
直接问"咖啡能喝吗""螃蟹能吃吗"——每种食物都有精准回答，附科学依据。

### 👶 宝宝发育可视化
4-40 周全覆盖，水果等比对照（芝麻→西瓜），宝宝拟人化描述。

### 🔔 智能提醒系统
PWA 浏览器通知：三餐提醒、加餐提醒、喝水提醒、产检提醒。

### 💬 Gemini 风格对话
基于 44000+ 字真实孕妈对话记录优化：
- 结构化但有温度（先共情 → 编号解读 → 💡追问）
- 情绪先行（负面情绪先 100% 共情再给知识）
- 宝宝拟人化（"TA在练拳击""在给你发勋章"）
- 每次回复末尾有具体追问引导下一轮对话

## 🚀 使用方式

安装后，对 AI 说以下任何一句话即可触发：
- "我怀孕了"
- "明天要去做 NT"
- "咖啡能喝吗"
- "宝宝现在多大了"
- "帮我看看产检报告"
- "今天吃什么好"
- "周末帮我备餐"
