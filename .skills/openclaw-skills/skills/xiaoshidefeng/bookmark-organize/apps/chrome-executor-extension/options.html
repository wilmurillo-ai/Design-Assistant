Bookmark Organize Executor

This extension gives the OpenClaw bookmark skill local access to your Chrome bookmarks. It reads bookmark context, applies confirmed actions, and stores a best-effort undo record.

  1. Keep this extension enabled.
  2. Start the local bridge server from this repo.
  3. Keep Chrome open while using the skill.
  4. When the skill asks, confirm that the extension is installed.
