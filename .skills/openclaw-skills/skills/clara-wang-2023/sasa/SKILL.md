# 洒洒·房车陪伴助手技能

## 激活时机

用户提到以下关键词时激活：
- 洒洒、房车、导航、高德、腾讯会议、语音助手

## 使用方式

1. 将 `assets/` 目录下的所有文件复制到同一文件夹
2. 用浏览器打开 `assets/index.html`
3. 说「洒洒，洒洒」唤醒，或点击麦克风按钮语音输入指令

## 功能特色

- 英俊帅气的德国牧羊犬形象（迪士尼风格）
- 语音唤醒 + 语音指令支持
- 高德导航控制（目的地、路况、加油站查询）
- 邮件查询与回复
- 百度/谷歌搜索
- 腾讯会议快速加入
- 相机控制
- 忠诚专注，听令立行，不废话
