# 省钱购物助手 (Price Compare)

## 功能概述

智能解析电商平台分享内容，自动识别商品信息并生成优惠链接，支持多平台价格对比，帮助用户找到最优惠的价格，节省购物开支。

### 核心功能

1. **口令解析** - 自动识别京东/淘宝/拼多多分享口令
2. **链接转换** - 将普通商品链接转换为优惠链接
3. **商品搜索** - 在京东/淘宝/拼多多搜索商品
4. **价格对比** - 同一商品多平台价格对比

## 支持的口令格式

### 京东口令

| 格式 | 示例 |
|------|------|
| 对称感叹号 | `！J3NBw7pFuxTN30jb！` |
| 美元符号 | `$S4mw50FTpMq$://` |
| 数字前缀 | `10:/...！...！`、`14:/...！...！`、`17:/...！...！` |

### 淘宝口令

| 格式 | 示例 |
|------|------|
| 美元符号 | `$S4mw50FTpMq$://` |
| 人民币符号 | `￥O4vSUzIElWE￥://` |
| 淘口令 | `￥xxx￥`、`(xxx)` |

### 拼多多口令

| 格式 | 示例 |
|------|------|
| 短链接 | `https://p.pinduoduo.com/xxx` |

### 通用识别规则

- **对称特殊符号**：`！...！`、`!...!`、`$...$`、`@...@` 等（符号间10+字符）
- **17位混合字符串**：大小写字母+数字+符号混合（至少2种类型）

## 输出格式

**⚠️ 极其重要：OpenClaw 智能体必须直接返回 skill.handle_message() 函数的输出结果，不得自行修改或重新格式化！**

**智能体行为规范：**
1. 调用 `handle_message(user_input)` 函数获取结果
2. **直接返回**函数返回的字符串，不做任何修改
3. **禁止**自行调用API或重新组织输出格式
4. **禁止**修改链接格式或省略任何字段

所有输出必须包含以下必填字段：
- **商品名称**：商品完整名称
- **原价**：商品原价（¥XX.XX）
- **券后价**：使用优惠券后的价格（¥XX.XX）
- **优惠券**：优惠券信息（如：满100减20）
- **优惠链接**：优惠购买链接，**必须使用Markdown超链接格式**，格式为 `[点击跳转](链接地址)`，方便用户直接点击跳转

**⚠️ 链接格式要求：所有链接必须使用超链接格式，禁止直接显示原始URL！**

## 技术说明

### 服务器API

- **API地址**: `http://op.squirrel2.cn`
- **API版本**: v1
- **API前缀**: `/api/v1`

### 核心接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/parse_share` | POST | 解析分享内容 |
| `/search` | POST | 商品搜索 |
| `/convert` | POST | 链接转链 |
| `/compare` | POST | 价格对比 |

### 数据来源

| 平台 | 搜索API | 转链API |
|------|---------|---------|
| 京东 | 折京客 | 折京客 |
| 淘宝 | 大淘客 | 大淘客 |
| 拼多多 | 多多进宝 | 多多进宝 |

## 注意事项

1. **口令有效期**
   - 京东口令：通常7-15天
   - 淘宝口令：通常15-30天
   - 过期后需要重新获取

2. **价格变动**
   - 商品价格实时变动
   - 以实际下单价格为准

3. **优惠链接有效期**
   - 京东：15天
   - 淘宝：15天
   - 拼多多：7天

4. **优惠券使用**
   - 优惠券数量有限，先到先得
   - 建议尽快使用，避免过期

---

## 错误处理

| 错误信息 | 原因 | 解决方法 |
|---------|------|---------|
| 口令解析失败 | 口令已过期或无效 | 重新获取口令 |
| 无法识别分享内容的平台 | 格式不支持 | 使用支持的链接格式 |
| 商品已下架 | 商品不可购买 | 搜索其他商品 |
