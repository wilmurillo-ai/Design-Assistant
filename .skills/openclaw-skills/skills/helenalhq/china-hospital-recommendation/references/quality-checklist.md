# Quality Checklist

Run this checklist before final delivery.

## Self-Containment

- SKILL.md references only files inside this skill.
- The report does not rely on runtime search for Fudan rankings.

## Recommendation Quality

- Exactly 3 hospitals by default unless a justified expansion reason is recorded.
- Every hospital has concrete fit reasoning.
- Specialist guidance is evidence-aware and does not invent named doctors.
- JCI appears as a positive factor when verified, not as a hard filter.
- Teleconsult wording is clearly separated from administrative intake and pre-travel record review.
- Procedure-first specialist directions are not used as the default when routing-critical clinical details are still unknown.
- Costs use scenario framing or clearly marked high-uncertainty language instead of a false sense of precision.

## Report Quality

- Final language is English.
- Chinese hospital names are supporting labels only.
- Evidence notes distinguish bundled ranking facts from current-search facts.
- Disclaimer is present and in English.
- The report speaks directly to the patient using "you" and "your".
- The report includes a quick comparison layer so the patient can tell the three options apart quickly.

## PDF Quality

- TOC renders
- ChinaMed brand colors are used consistently across the cover, section headings, tables, and footer
- hospital and specialist headings are readable
- hospital cards clearly separate summary, metadata, limitation, and booking checks
- Chinese names are not garbled
- footer is readable and consistent across pages
- cost table spacing remains readable without cramped rows
- PDF fallback path still produces a deliverable file when the premium export path fails
- The first page does not rely on long-value cover tables.
- Disclaimer includes the ChinaMed Select consult-service sentence.
