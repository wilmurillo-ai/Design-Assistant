# 接口返回字段说明

## 接口信息

- **地址**：`https://tz.smxqx.tech/api/stock/valuation/detail`
- **方法**：POST
- **Content-Type**：`application/json`
- **版本头**：`v: 2.5`

## 返回字段详解

### 基础信息字段

| 字段名 | 类型 | 示例值 | 说明 |
|--------|------|--------|------|
| `code` | string | `600519.SH` | 股票完整代码（含市场后缀） |
| `symbol` | string | `600519` | 股票简洁代码（不含市场后缀） |
| `name` | string | `贵州茅台` | 股票中文名称 |
| `exchange` | string | `SSE` | 交易所代码（见下方对照表） |
| `industry` | string | `白酒` | 所属行业 |
| `trade_date` | string | `12月27日` | 最近交易日期 |

### 行情数据字段

| 字段名 | 类型 | 示例值 | 说明 |
|--------|------|--------|------|
| `price` | float | `1528.97` | 当前股价（元/港元） |
| `daily_price_change` | string | `0.08%` | 当日涨跌幅（正为涨，负为跌） |
| `yearly_price_change` | string | `-11.42%` | 年初至今涨跌幅 |
| `total_mv` | string | `19206.9` | 当前总市值（单位：亿元） |
| `total_share` | float | `12.56` | 总股本（单位：亿股） |

### 估值指标字段

| 字段名 | 类型 | 示例值 | 说明 |
|--------|------|--------|------|
| `pe_ttm` | float | `11.63` | 市盈率（TTM，滚动12个月） |
| `pb` | float | `9.25` | 市净率（Price-to-Book） |
| `ps_ttm` | float | `11.63` | 市销率（TTM，滚动12个月） |
| `dv_ratio` | string | `2.94%` | 股息率（近12个月股息/当前股价） |

### 综合估值判断字段

| 字段名 | 类型 | 示例值 | 说明 |
|--------|------|--------|------|
| `sum_p` | float | `1938.89` | 综合合理股价（元/港元） |
| `sum_v` | float | `24356.36` | 综合合理市值（亿元） |
| `margin` | string | `26.81%` | 安全边际（合理价 vs 当前价差距） |
| `tag` | string | `低估` | 估值标签：`低估` / `合理` / `高估` |
| `desc` | string | — | 估值解析文字说明 |

### 聚合对象字段

| 字段名 | 类型 | 说明 |
|--------|------|------|
| `sum` | object | 估值概述对象，包含综合汇总描述 |
| `list` | list | 各估值方法汇总列表，每项含 `name`、`sum_p`、`sum_v`、`margin`、`tag` |

## 交易所代码对照

| exchange 值 | 含义 |
|-------------|------|
| `SSE` | 上海证券交易所（沪市） |
| `SZSE` | 深圳证券交易所（深市） |
| `HKEX` | 香港联合交易所（港股） |

## 估值标签说明

| tag 值 | 含义 | 参考安全边际 |
|--------|------|-------------|
| `低估` | 当前股价低于合理估值 | 安全边际 > 0% |
| `合理` | 当前股价接近合理估值 | 安全边际接近 0% |
| `高估` | 当前股价高于合理估值 | 安全边际 < 0% |

> **注意**：安全边际为负数表示当前价格已超出合理估值，即高估状态。

## 安全边际计算说明

```
安全边际 = (合理股价 - 当前股价) / 合理股价 × 100%
```

- 正值：低估，数值越大说明低估越深
- 负值：高估，绝对值越大说明高估越严重

## list 字段（估值方法汇总）

`list` 数组中每个元素包含：

| 字段名 | 类型 | 说明 |
|--------|------|------|
| `name` | string | 估值方法名称（如 PE估值法、DCF估值法等） |
| `sum_p` | float | 该方法计算的合理股价 |
| `sum_v` | float | 该方法计算的合理市值（亿元） |
| `margin` | string | 该方法下的安全边际 |
| `tag` | string | 该方法下的估值判断 |
