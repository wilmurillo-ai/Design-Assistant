# 股票代码格式说明

## 市场后缀对照表

| 市场 | 后缀 | 说明 | 示例 |
|------|------|------|------|
| 上交所（沪市主板）| `.SH` | 上海证券交易所 | `600519.SH` |
| 深交所（深市/创业板）| `.SZ` | 深圳证券交易所 | `000001.SZ`、`300750.SZ` |
| 港交所（港股）| `.HK` | 香港联合交易所 | `00700.HK` |

## 代码规则

### A 股（沪市 `.SH`）
- **主板**：6 位数字，以 `6` 开头，如 `600519.SH`（贵州茅台）
- **科创板**：6 位数字，以 `68` 开头，如 `688981.SH`（中芯国际）

### A 股（深市 `.SZ`）
- **主板**：6 位数字，以 `0` 开头，如 `000001.SZ`（平安银行）
- **中小板**：6 位数字，以 `002` 开头，如 `002594.SZ`（比亚迪）
- **创业板**：6 位数字，以 `3` 开头，如 `300750.SZ`（宁德时代）

### 港股（`.HK`）
- 接口要求 **5 位**数字，不足 5 位需左侧补零
- 如腾讯控股代码为 `700`，输入时补齐为 `00700.HK`
- 如美团代码为 `3690`，输入时补齐为 `03690.HK`

## 注意事项

1. **大小写不敏感**：`600519.sh` 和 `600519.SH` 等价
2. **港股补零**：港股代码必须左侧补零到 **5 位**（如 `700` → `00700.HK`，`3690` → `03690.HK`）
3. **A+H 股**：同一公司 A 股和 H 股为不同标的，估值可能不同
