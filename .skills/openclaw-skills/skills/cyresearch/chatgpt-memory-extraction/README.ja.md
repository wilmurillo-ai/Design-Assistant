# ChatGPT メモリ抽出 🧠✨

[OpenClaw](https://github.com/openclaw/openclaw) のスキルです。長年 ChatGPT と話してきた会話を、**整理された、検索できる個人メモリアーカイブ**に変えてくれます～

## このツールを作ったきっかけ

ChatGPT と3年以上チャットしてきて、500以上の会話、6万件以上のメッセージが溜まりました。読んだ論文、学んだこと、下した決断、嬉しかったことや辛かったこと、出会った人たち——全部が巨大な JSON ファイルの中に閉じ込められていて、振り返りたくても探せない状態でした 😢

なので、このツールを作りました。ChatGPT のエクスポートデータを、タイムラインに沿ったメモリアーカイブに整理してくれます。人間関係や学びの軌跡、大事な出来事なども一緒に抽出してくれます。

しかも、整理したアーカイブは振り返り用だけじゃなくて——OpenClaw や他の AI アシスタントに読み込ませれば、**初日からあなたのことをちゃんと理解してくれる**ようになります。新しい AI を使うたびに一から自己紹介するの、本当に大変ですよね 😅

## できること

- 📦 **自動抽出**：Python スクリプトが JSON を読めるテキストファイルに変換、年・四半期ごとに整理
- 📖 **丁寧な読解**：AI にタイトルだけ見て要約させるんじゃなくて、一つ一つの会話をちゃんと読ませて、構造化された四半期タイムラインを作成
- 👥 **人物プロファイル**：会話に登場する人を抽出して、それぞれのプロファイルを作成
- 🎯 **テーマ整理**：学問の成長、感情の出来事、生活の変化、スキル習得——テーマごとにまとめ

## 正直に言うと

AI って大量のテキストを処理するとき、**本当に手を抜きがち**なんです 😅 これは何度も痛い目にあって学んだ教訓です。なので、このスキルには AI がよくやるサボりパターンに対応する品質ルールが組み込まれています（最初の数メッセージだけ読んで要約したり、「XX件のメッセージでいろいろ話した」みたいな中身のない要約を書いたり、「ファイル更新しました」と言いながら実は何も変えてなかったり……）。

ただ、このルールがあっても、**AI の出力は自分の目で確認するのがおすすめ**です。四半期ごとにチェックして、浅いところがあったらやり直してもらう。一番いい結果が出るのは、AI に丸投げするんじゃなくて、一緒に作業するときです～

## 使い方

### 1. インストール

```bash
# ClawHub 経由
npx clawhub install chatgpt-memory-extraction

# または GitHub から手動インストール
git clone https://github.com/cyresearch/chatgpt-memory-extraction.git ~/.openclaw/workspace/skills/chatgpt-memory-extraction
```

### 2. ChatGPT のデータをエクスポート

[ChatGPT](https://chat.openai.com) → 設定 → データコントロール → データをエクスポート → 確認

> ⏳ エクスポートには**数時間〜1、2日**かかることがあります。データが多いともっとかかるかも。準備ができたら OpenAI からメールが届きます～

### 3. 整理スタート！

OpenClaw アシスタントに伝えてください：

> 「ChatGPT のデータをエクスポートしたよ、メモリアーカイブに整理して」

手順を一つずつ案内してくれます：抽出 → 四半期ごとに読解 → タイムライン作成 → あなたがレビュー → 人物・テーマの抽出 🌱

## 出力イメージ

```
memory/
├── timeline/{year}/Q{1-4}.md   # 四半期タイムライン（一番大事なファイル～）
├── people/{category}/*.md       # 人物プロファイル
├── topics/*.md                  # テーマ別の記録
├── meta/extraction-log.md       # 進捗管理
└── raw/{year}_Q{n}/             # 元の会話テキスト（確認用）
```

## 必要なもの

- [OpenClaw](https://github.com/openclaw/openclaw)
- Python 3.8+
- ChatGPT のデータエクスポートファイル（OpenAI からダウンロードした `.zip`）
