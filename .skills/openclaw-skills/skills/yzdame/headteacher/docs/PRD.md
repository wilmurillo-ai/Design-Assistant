# 班主任.Skill PRD 摘要

## 产品定位

`班主任.Skill` 是一个 AI Native 班主任工作台 Skill。

它不是班主任角色卡，也不是人格蒸馏器。  
它负责引导用户完成工作台 setup，并在工作台建立后持续处理结构化数据和文件生成任务。

## 第一版目标

- 默认首选飞书多维表格
- 提供 setup 向导
- 提供统一语义层
- 提供 Feishu bootstrap 与迁移检查能力
- 预留 Notion / Obsidian 占位适配

## 核心对象

- 学生主档
- 考试批次
- 成绩明细
- 成长事件
- 家校沟通
- 座位安排
- 值日安排
- 班委任命
- 产物索引

## 两类核心能力

### 1. 数据记录与读取

- 写入分为：
  - 一次性导入
  - 动态添加或更新
- 读取分为：
  - 纵向读取：沿时间轴查看某个学生
  - 横向读取：在某个时间截面查看部分学生或全班

### 2. 自动化文件生成

- 排座表 / 值日表
  - 可按成绩、性别、身高、行为约束等属性排列
- 家长会 PPT
  - 依据学生成绩记录与日常表现记录自动生成

## 第一版不做

- 班主任人格扮演
- 多后端完整并进
- 高级权限与复杂仪表盘
- 心理健康自动判定
