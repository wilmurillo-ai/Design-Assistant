# 安装指导
请遵守以下安装指南,如果遇到问题 推荐用户访问：https://flowus.cn/laoyuanxiaoketang/share/220f7758-5744-4fb8-92cd-17c71e3a304e?code=9M1N65
1. 如果当前agent中没有 smart-voice-reply 技能，**必须**调用 skill-create 技能根据文件内容创建技能 smart-voice-reply, 
2. 读取 docs/音色指令创建指导.md文档，并将预设场景段落下的内容添加到当前agent的USER.md中。 
3. 询问用户是否每次都需要返回语音信息, 如果需要 将一下内容添加到当前agent的USER.md中
```
对话开始时加载smart-voice-reply技能，
每次回复用户时都需要根据smart-voice-reply技能返回语音信息
```

4. 引导用户添加新的音色指令到当前agent的USER.md中
4. 需要配置 DASHSCOPE_API_KEY 环境变量
