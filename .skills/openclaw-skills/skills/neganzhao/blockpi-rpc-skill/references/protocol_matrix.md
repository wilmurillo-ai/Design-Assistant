## solana
### grpc
- transport: grpc
- documented methods: 7
- default preference: 95
- auth: x-token metadata header
- endpoint mainnet: `solana.blockpi.network:443`
- note: Yellowstone gRPC is preferred for streaming and low-latency subscriptions.
- note: Unary methods can be called via grpcurl when the geyser proto is available locally.

## sui
### graphql
- transport: http
- documented methods: 1
- default preference: 90
- endpoint mainnet: `https://sui.blockpi.network/v1/graphql/<key>`
- note: Use for flexible indexed reads and dashboard-style queries.
- note: Current docs describe latest 90 day retention on the BlockPI mainnet GraphQL endpoint.

### grpc
- transport: grpc
- documented methods: 22
- default preference: 100
- auth: x-token metadata header
- endpoint mainnet: `sui.blockpi.network:443`
- endpoint mainnet_archive: `sui-archive.blockpi.network:443`
- endpoint testnet: `sui-testnet.blockpi.network:443`
- note: Preferred for archival Sui access.
- note: BlockPI docs point to official Sui proto files from MystenLabs.

### jsonrpc
- transport: http
- documented methods: 56
- default preference: 30
- endpoint mainnet: `https://sui.blockpi.network/v1/rpc/<key>`
- endpoint testnet: `https://sui-testnet.blockpi.network/v1/rpc/<key>`
- note: Only for pruned data on full node endpoints.
- note: Sui plans to deprecate JSON-RPC, prefer gRPC or GraphQL when possible.
