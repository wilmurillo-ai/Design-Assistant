# 审美积累指南 & 模板

> 本文件是审美积累文档的模板和指南。
> 使用这个 Skill 时，你需要在项目中创建自己的 `aesthetic-library.md`，按本模板的格式记录。

---

## 为什么要维护审美积累文档

- 每次配图前**不用从零找灵感**，翻文档就有
- **避免审美退化**：不积累 → 每次输出都是 AI 默认审美
- **风格多样性**：积累越多，封面越不会千篇一律
- **团队协作**：新人接手也能快速了解审美标准

## 推荐的风格分类

根据 5 轮小红书实战，以下是常见且有效的封面风格：

### 1. 3D Clay / 黏土风格
- **关键词**：暖色、圆角、黏土质感、柔和光影、玩具感
- **适合**：科技主题但要亲和力的场景
- **要诀**：哑光材质 + 暖色调 + 头大身小的卡通比例

### 2. 产品截图 + 标注流
- **关键词**：真实截图、手写标注、箭头圈注
- **适合**：教程帖、工具推荐、实战记录
- **要诀**：截图必须真实 + 标注不超过 3 处 + 暖色背景

### 3. 权威人物 / 真人出镜流
- **关键词**：人脸、3D 风格化、利益点大字
- **适合**：观点帖、趋势帖、权威背书
- **要诀**：优先 3D 风格化（规避版权）+ 表情生动 + 人物占 60%

### 4. 纯文字卡 / 极简设计
- **关键词**：大号字体、留白、一句话核心
- **适合**：金句帖、方法论帖
- **要诀**：圆角无衬线体 + 暖色/品牌色背景 + 装饰不喧宾夺主

### 5. Logo/工具矩阵
- **关键词**：多 Logo 整齐排列、干净背景
- **适合**：工具合集、盘点类
- **要诀**：间距统一 + 品牌色调协调 + 标题简洁

## 每次调研的最低要求

| 维度 | 最低标准 |
|------|---------|
| 作品浏览量 | ≥ 30 个 |
| 精选记录数 | ≥ 10 条 |
| 记录格式 | 链接/截图 + 风格标签 + 借鉴点 + 适合度 |
| 新增分类 | 如果发现新风格，及时新增分类 |

---

## 通用反面案例

> 这些是经过实战验证的"绝对不要做"的封面类型。

| 反面案例 | 原因 |
|---------|------|
| 左右对比 / 左右分栏 | 没辨识度，像 AI 模板 |
| 概念文字堆砌 | 数据证明无人关心 |
| 新闻感封面（logo + 放大镜） | 像营销号 |
| 品牌官方宣传图做主视觉 | 小红书限流风险 |
| 裸露器官（大脑/心脏/眼球） | 信息流中生理不适 |
| "技术概念"直线联想 | AI 最缺乏创意的默认路径 |
| 所有帖子用同一模板 | 模板化 = 没有辨识度 |
