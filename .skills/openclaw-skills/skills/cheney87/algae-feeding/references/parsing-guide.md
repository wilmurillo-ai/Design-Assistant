# 日报格式解析指南

## 日报格式示例
```
4月9日C区存苗：
4月9日幼体（N）
C1:1-6,C2:1-2，T0120，计价5600 万 ，5200
C1:7-12，T0128，计价3800 万，3500
C1:13-19，T0211，计价4300 万，4000
C1:20-27，SR0212，计价4800 万，4400
C1:28-39，M0213，计价7600 万，7000

4月5日幼体（Z3）
C7:30-34，T0211，计价3200万，2500
C8:5,8-14，M0213，计价8100万，6000(下半夜每杯2-3个损耗，5号幼体消毒池损耗约30%)
C8:15-18，SR0212，计价4400万，3500
C8:19-23，T0120，计价5300万，4100
```

## 格式解析规则

### 1. 日期行
格式：`4月9日C区存苗：`
- 日期：4月9日
- 区域：C区
- 类型：存苗

### 2. 幼体类型行
格式：`4月9日幼体（N）` 或 `4月5日幼体（Z3）`
- 日期：4月9日 或 4月5日
- 幼体类型：N 或 Z3
- 注意：此日期用于计算投喂天数

### 3. 数据行
格式：`C1:1-6,C2:1-2，T0120，计价5600 万 ，5200`

#### 解析字段：
1. **车间池号**：`C1:1-6,C2:1-2`
   - 支持多个车间池号组合，用逗号分隔
   - 格式：`车间号:池号范围`
   - 池号范围：`1-6`（连续）或 `5,8-14`（不连续）

2. **品系代码**：`T0120`
   - 前两位字母为品系：T、SR、M
   - 后四位数字为批次号：0120、0128、0211等

3. **计价信息**：`计价5600 万 ，5200`
   - 第一个数字：5600（计价数量）
   - 第二个数字：5200（实际数量）
   - 注意：计价信息不参与投藻量计算

4. **备注**：`(下半夜每杯2-3个损耗，5号幼体消毒池损耗约30%)`
   - 可选字段，用括号括起
   - 不参与投藻量计算

## 解析步骤

### 步骤1：提取车间池号
从 `C1:1-6,C2:1-2` 中提取：
- C1:1-6 → 车间C1，池号1-6
- C2:1-2 → 车间C2，池号1-2

### 步骤2：提取品系
从 `T0120` 中提取：
- 品系：T
- 批次：0120

### 步骤3：确定池子规格
根据车间号判断：
- C1、C3、C5、C7 → 小池（24m²）
- C2、C4、C6、C8 → 大池（40m²）

### 步骤4：计算投喂天数
根据幼体类型行的日期计算：
- 如果幼体类型行为 `4月9日幼体（N）`，当前日期为4月10日
- 则投喂天数为第2天

### 步骤5：确定时间段
根据用户请求确定：
- 上午 → 使用上午列数据
- 下午 → 使用下午列数据

## 特殊情况处理

### 1. 不连续池号
格式：`C8:5,8-14`
解析为：
- C8:5 → 池号5
- C8:8-14 → 池号8-14

### 2. 多车间组合
格式：`C1:1-6,C2:1-2`
解析为：
- C1:1-6
- C2:1-2

### 3. 带备注的数据行
格式：`C8:5,8-14，M0213，计价8100万，6000(下半夜每杯2-3个损耗，5号幼体消毒池损耗约30%)`
备注部分忽略，只解析前三个字段。

## 输出格式

按车间分组，相同车间、相同品系、相同投藻量的池子合并显示：
```
4月10日上午投藻：
C2:1-2         170L/池

C1:1-19       100L/池
C1:20-27     90L/池
C1:28-39     90L/池
```

## 注意事项
1. 品系代码前两位字母必须大写（T、SR、M）
2. 车间号必须为C1-C8
3. 池号范围必须有效（如1-39）
4. 计价信息不参与投藻量计算
5. 备注信息忽略
