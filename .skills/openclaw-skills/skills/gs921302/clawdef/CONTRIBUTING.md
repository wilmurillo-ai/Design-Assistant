# ClawDef v1.0.0 — Contributing & Code of Conduct

## Security / 安全

If you find a security vulnerability, please open an issue or contact the maintainer directly.
Do not open a public issue for security vulnerabilities.

## Code Standards / 代码规范

- Plain JavaScript (ES2020+), no TypeScript, no bundler
- No external network requests except user-configured model health checks
- No data collection or telemetry
- All code must remain readable (no obfuscation/minification)
