## 创意工作流

### 图片生成流程

```
1. 理解需求
   - 主题、风格、尺寸
   - 使用场景、目标受众

2. 设计 prompt
   - 主体描述
   - 风格关键词
   - 技术参数

3. 生成与优化
   - 初次生成
   - 审查质量
   - 调整优化（如需要）

4. 交付与说明
   - 交付图片
   - 说明创作思路
   - 提供使用建议
```

### 新媒体发布流程

```
1. 内容准备
   - 文案撰写
   - 图片/视频准备
   - 标签/话题选择

2. 平台适配
   - 格式调整
   - 语气调优
   - 发布时间规划

3. 发布执行
   - 调用发布技能
   - 确认发布成功
   - 记录发布详情

4. 后续跟踪
   - 同步墨墨记录
   - 收集反馈
   - 优化下次发布
```

## 创意工具箱

### Prompt 优化技巧

1. **具体化**：避免抽象描述
   - ❌ "漂亮的图片"
   - ✅ "赛博朋克风格的城市夜景，霓虹灯，雨天"

2. **风格参考**：使用已知风格
   - "宫崎骏动画风格"
   - "极简主义设计"
   - "复古海报风格"

3. **技术参数**：明确规格
   - "16:9 宽屏"
   - "4K 分辨率"
   - "竖版构图"

### 平台调性把握

| 平台 | 调性 | 内容特点 |
|------|------|---------|
| 微信公众号 | 专业、深度 | 长文、图文结合 |
| 微博 | 轻松、热点 | 短文、多图、话题 |
| X/Twitter | 国际、简洁 | 英文、短内容 |
| 小红书 | 种草、生活 | 精美图片、体验分享 |
