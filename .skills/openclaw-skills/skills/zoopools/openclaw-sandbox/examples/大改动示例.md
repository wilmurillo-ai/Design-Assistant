# 大改动示例 - 新增 Agent

**场景**: 为 OpenClaw 新增一个客服 Agent

---

## 操作流程

### 1. 创建 Git 分支

```bash
cd ~/.openclaw
git checkout -b add-customer-service-agent-$(date +%Y%m%d-%H%M)
```

### 2. 修改配置

```bash
vim ~/.openclaw/config/agents/customer-service.json
```

添加内容：
```json
{
  "id": "customer-service",
  "name": "客服助手",
  "workspace": "~/Documents/openclaw/agents/customer-service",
  "model": {
    "primary": "bailian/glm-4.7",
    "fallback": "bailian/qwen-plus"
  },
  "tools": {
    "exec": {
      "node": "node-id"
    }
  }
}
```

### 3. 沙盒测试

```bash
~/.openclaw/skills/openclaw-sandbox/templates/safe-try.sh
```

**输出**:
```
🧪 正在启动沙盒验证环境 (Port: 18790)...
⚠️  此环境仅用于验证配置，不修改生产数据

📁 临时状态目录：/tmp/openclaw-sandbox-state-12345

📊 沙盒 WebUI: http://127.0.0.1:18790
📊 生产 WebUI: http://127.0.0.1:18789 (不受影响)

按 Ctrl+C 停止沙盒
```

**验证步骤**:
1. 访问 http://127.0.0.1:18790
2. 检查新 Agent 是否显示
3. 测试新 Agent 功能
4. 确认无误后 Ctrl+C 停止沙盒

### 4. 应用配置

```bash
~/.openclaw/skills/openclaw-sandbox/templates/apply-config.sh
```

### 5. Git 提交

```bash
cd ~/.openclaw
git add .
git commit -m "feat: 新增客服助手 Agent - $(date +%Y%m%d)"
```

---

## 验证

```bash
# 检查 Agent 列表
openclaw agents list | grep customer-service

# 测试 Agent
openclaw message send --agent customer-service --message "测试"
```

---

## 回滚方法

```bash
# 使用 Git 回滚
cd ~/.openclaw
git checkout HEAD~1
~/.openclaw/skills/openclaw-sandbox/templates/apply-config.sh
```
