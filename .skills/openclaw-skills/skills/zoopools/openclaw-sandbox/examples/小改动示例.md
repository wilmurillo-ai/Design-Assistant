# 小改动示例 - 修改日志级别

**场景**: 将日志级别从 `info` 改为 `debug`

---

## 操作流程

### 1. 修改配置

```bash
vim ~/.openclaw/config/core.json
```

修改内容：
```json
{
  "log_level": "info"  // 改为 "debug"
}
```

### 2. 应用配置

```bash
~/.openclaw/skills/openclaw-sandbox/templates/apply-config.sh
```

### 3. 验证输出


```bash
# 使用备份回滚
cp ~/.openclaw/openclaw.json.bak.20260308-1753 ~/.openclaw/openclaw.json

```
