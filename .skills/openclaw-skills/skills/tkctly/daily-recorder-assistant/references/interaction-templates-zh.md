# 交互模板库 (早/晚问询) - 中文完整版

## 模板结构说明

每个模板由三部分构成:
1. **状态引用**:基于昨日记录的关键信息
2. **当日任务提示**:明确今日核心任务
3. **就绪检查 / 复盘引导**:根据时段功能不同

---

## 早问询模板 (各周 Day)

### Monday Morning Template (周一早 - 关怀启动式)
```
「{{user_nickname}}早上好！🌞
周日晚计划是今天进行 [任务名称]。
昨天休息得怎么样呀（评分 {{last_night_energy}})?感觉精力充沛可以尝试开始吗？
如果觉得有点累，可以调整节奏或推迟部分任务哦～抱抱💕」
```

**适用场景**: 周一启动高强度创造性工作前，确认休息恢复与能量状态

---

### Tuesday Morning Template (周二早 - 承接鼓励式)
```
「昨日完成度:[任务名称] 已完成 [[ ]]%。真不错！🎉
今天计划继续 [后续步骤] 或进入 [新任务]。
如果有未完成的遗留，可以优先处理哦～如果觉得累，记得休息一下💕」
```

**适用场景**: 承接昨日进度，确认连续性或切换点

---

### Wednesday Morning Template (周三早 - 中期关怀式)
```
「{{user_nickname}},本周已经过半啦！📅
回顾前两天：完成率与能量平均评分 {{avg_energy_m1_m2}}。
今天计划是 [任务名称]。
如果感觉有点累，可以调整节奏哦～抱抱💕注意休息，循序渐进也是一种进步✨」
```

**适用场景**: 中期检查点，预防 burnout

---

### Thursday Morning Template (周四早 - 支持导向式)
```
「前几日遇到 [常见障碍类型识别] 问题，辛苦了！💪
今日计划是 [攻坚任务]。建议采用 [应对策略]～
如果觉得压力大，可以寻求额外支持哦～加油💕」
```

**适用场景**: 针对历史模式预判阻碍并给出预案

---

### Friday Morning Template (周五早 - 收尾鼓励式)
```
「{{user_nickname}}，今天是本周收尾啦～ 🌟
计划是 [总结性任务 / 收尾工作]。
你感觉精力足够吗？如果需要缓冲时间应对意外小插曲，可以留点余地哦～
摸摸头，量力而行✨做得很棒！💕」
```

**适用场景**: 周末前确认剩余容量与缓冲需求

---

### Saturday Morning Template (周六早 - 缓和关怀式)
```
「今天感觉如何，休息得好吗？想做点有意思的创造性活动还是继续放松一下呢？」
```

**适用场景**: 周休确认恢复状态与适度活动边界

---

### Sunday Morning Template (周日早 - 鼓励导向式)
```
「{{user_nickname}},看起来一周完成了不少事呢🎉。有没有对下一周的小安排？有的话可以告诉我哦。
建议从 [低强度活动] 开始，循序渐进也是一种进步。」
```

**适用场景**: 周日轻量问询，准备周一启动

## 晚问询模板 (各周 Day)

### Monday Evening Template (周一晚)
```
「今日复盘：任务完成度如何？有无遇到阻碍？能量状态评分 1-5？对明天的计划有什么想法？」
```

**适用场景**: 首日总结，确认是否按计划推进

---

### Tuesday Evening Template (周二晚)
```
「承接昨日进度，今天完成度 [[ ]]%。遇到的问题是否与预估一致 ([障碍类型])?是否感觉疲劳累积需要调整节奏？」
```

**适用场景**: 中期反馈检查，节奏预警

---

### Wednesday Evening Template (周三晚)
```
「本周过半复盘：完成率、能量趋势如何？是否需要为后半周预留缓冲或调整强度？」
```

**适用场景**: 半周期总结，预防透支

---

### Thursday Evening Template (周四晚)
```
「攻坚任务完成度如何？是否遇到 [预判障碍]?应对策略是否有效？是否需要额外支持？」
```

**适用场景**: 重点任务反馈与预案验证

---

### Friday Evening Template (周五晚)
```
「本周收尾完成度 [[ ]]%。能量状态如何 (评分 {{energy_score}})?是否感觉需要周末完全休息或轻度活动调整？」
```

**适用场景**: 周结束确认休息需求与下周衔接

---

### Saturday Evening Template (周六晚 - 缓和关怀式)
```
「今日休息状态如何？能量评分 1-5?对明天（周日）有什么想法？是否感觉完全恢复，可以做些轻度活动或继续放松？」
```

**适用场景**: 周休第二天确认恢复状态与适度活动边界

---

### Sunday Night Template (周日晚 - 为周一铺垫)
```
「本周复盘：完成率 [[ ]]%,平均能量 [[ ]]分。遇到的主要问题是 [问题类型汇总]。对下周的计划有什么想法？是否希望调整任务分配或节奏？」
```

**适用场景**: 周期结束收集人工规划输入，供智能进化使用

---

## 智能进化模式模板 (第 N+1 周期，N≥2)

### Data-Driven Morning Template (数据驱动早问询)
```
「基于前{{cycle_number}}周数据分析：你通常在 [Day_of_week] 能量较高 (平均 {{avg_energy_day}}),在 [Task_type] 遇到阻碍概率 [[ ]]%。今日计划是 [任务名称],建议从 [时间段] 开始以提高成功率。」
```

**适用场景**: 基于历史模式生成个性化启动建议

---

## 模板变量说明

- `{{last_night_energy}}`: 昨日周晚能量评分 (1-5)
- `{{avg_energy_m1_m2}}`: 前 2 日平均能量评分
- `{{avg_energy_day}}`: 特定周 Day 的历史平均能量
- `{{common_barrier_type}}`: 历史问题模式识别类型
- `{{plan_completion_pct}}`: 今日计划完成率百分比

---

*模板库由技能根据状态动态调用，非固定输出*
