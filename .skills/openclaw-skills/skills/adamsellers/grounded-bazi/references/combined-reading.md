# Combined BaZi + feng shui + practical reading mode

Use this when the user wants both a personal-pattern read and environmental guidance.

## Purpose

Combine:
1. BaZi-style personal interpretation
2. feng shui style space guidance
3. practical action advice

This works well for users asking things like:
- "Give me the full read"
- "What should I change in my life and my space?"
- "Do my current luck cycle and home setup line up?"
- "Give me the mystical version, but useful"

## Required inputs

For the BaZi portion:
- date of birth
- time of birth if known
- place of birth
- focus

For the feng shui portion, ask for any that are available:
- room or space type
- photos or layout
- current problem
- desired outcome

If the user has no room details yet, still do the combined read by giving general environmental guidance tied to the BaZi themes.

## Structure

### 1. Framing note
Keep it brief and honest.
Example:
"I’m combining a BaZi-style personal read with a feng shui-style environment read, so treat this as reflective guidance rather than celestial accounting."

### 2. Personal pattern
Cover:
- core temperament
- current phase / pressure pattern
- what is being rewarded right now
- what drains them right now

### 3. Environment pattern
Translate the personal pattern into space guidance.
Examples:
- if the person needs focus and authority, improve desk command position and reduce visual clutter
- if the person needs recovery and nervous-system calm, simplify bedroom, remove work bleed, soften lighting
- if the person needs momentum, clear blocked entrances and make active zones feel intentional

### 4. Practical alignment plan
Give 3 buckets:
- do this this week
- avoid this for now
- reinforce this habit

### 5. Distilled synthesis
End with 1-3 sentences that unify person, timing, and space.

## Tone

- elegant
- practical
- lightly mystical
- no fake certainty

## Good closing offers

- "If you want, send me a desk or bedroom photo and I’ll make this much sharper."
- "If you want, I can do a career/money-only combined pass next."
- "If you want, I can turn this into a clean note or email draft."
