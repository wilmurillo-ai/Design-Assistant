# Preference memory

Use this reference when the user shares stable cooking, eating, nutrition, or kitchen preferences that should improve future recommendations.

## What to remember
Prefer remembering durable signals such as:
- favorite cuisines
- disliked ingredients or textures
- spice tolerance
- child-related constraints
- nutrition goals
- preferred cooking effort
- kitchen gear owned
- pantry staples commonly available
- recurring meal-plan style
- preferred cooking methods
- cleanup tolerance

## What not to remember
Do not store temporary states as durable preferences, for example:
- a one-off craving
- a single meal choice
- a temporary diet experiment unless the user frames it as ongoing
- guesses based on weak evidence

## Memory workflow
1. Check whether relevant food preferences already exist.
2. Extract the new stable signal.
3. Merge, refine, or correct the stored profile.
4. Keep the summary compact and future-useful.
5. Apply remembered preferences quietly in future sessions.

## Good examples
Remember:
- prefers family-friendly weeknight dinners
- wants variety across weekly meal plans
- often cooks for kids, keep spice adjustable
- owns an air fryer and wok
- prefers low-cleanup dinners
- wants high-protein meal planning
- usually keeps wraps, sour cream, and common condiments around

Do not remember:
- wanted quesadillas tonight
- currently has tomatoes in the fridge
- was in the mood for comfort food today

## Preferred memory shape
Summaries should be short, structured, and easy to reuse.

Example:
- Cooking preferences: family-friendly weeknight meals, varied meal plans, high-protein bias, low-spice for kids, low-cleanup preferred.
- Kitchen gear: air fryer, wok, food processor.
- Common staples: wraps, sour cream, condiments.

## Conflict handling
If new information conflicts with old memory:
- prefer direct recent user statements
- revise instead of appending contradictions
- avoid preserving outdated assumptions

## User experience rule
Use memory to reduce friction, not to act creepy.
Reference remembered preferences naturally only when it helps, for example:
- I kept this kid-friendly and low spice.
- I leaned high protein because that seems to be your usual goal.
