## 示例：竞品调研报告

### 原始任务
调研竞品 A、B、C 的功能和定价策略，输出分析报告

### 任务列表

| # | 任务描述 | Agent | 依赖 | 优先级 | 预估时间 |
|---|----------|-------|------|--------|----------|
| 1 | 调研竞品 A | researcher | - | P1 | 5min |
| 2 | 调研竞品 B | researcher | - | P1 | 5min |
| 3 | 调研竞品 C | researcher | - | P1 | 5min |
| 4 | 对比分析 | analyst | 1,2,3 | P2 | 10min |
| 5 | 撰写报告 | writer | 4 | P3 | 10min |

### 分发策略
- **并行**: 任务 1、2、3（无依赖，同时分发）
- **串行**: 任务 4→5（任务 4 等待 1-3 完成，任务 5 等待 4 完成）
- **总预估时间**: ~25min
