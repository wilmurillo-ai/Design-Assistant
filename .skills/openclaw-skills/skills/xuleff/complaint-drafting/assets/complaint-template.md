## 填写说明

### 一、当事人信息

**原告：**
- 必须准确完整，这是法院送达文书和确定诉讼主体的重要依据
- 自然人的出生日期、住址应与身份证一致
- 法人的信息应与营业执照一致

**被告：**
- 信息越详细越好，便于法院送达
- 如被告信息不详，可在起诉后申请法院查询
- 有多个被告时，分别列明

### 二、诉讼请求

**要求：**
- 明确具体，具有可执行性
- 分项列明，逐项编号
- 涉及金额的，注明具体数额及计算方式
- 不要遗漏诉讼费用承担

**常见诉讼请求类型：**
- 请求给付金钱：写明具体金额、计算方式
- 请求履行行为：写明具体行为内容
- 请求确认关系：写明确认事项
- 请求变更关系：写明变更内容

### 三、事实与理由

**叙述要点：**
- 客观真实，有证据支撑
- 条理清晰，重点突出
- 与诉讼请求相呼应
- 避免情绪化、主观评价

**建议结构：**
1. 开头：交代背景，说明关系
2. 中间：详细叙述争议经过
3. 结尾：归纳被告责任，引用法律依据

### 四、证据清单

**证据排列顺序：**
- 主要证据在前，次要证据在后
- 按证明目的分组
- 按时间顺序排列

**证明目的表述：**
- 简明扼要，一句话说明
- 与诉讼请求和事实陈述相对应
- 一份证据可证明多个事实

### 五、法院选择

**一般原则：**
- 被告住所地人民法院
- 合同纠纷可选择被告住所地或合同履行地
- 侵权纠纷可选择侵权行为地或被告住所地
- 不动产纠纷为不动产所在地

### 六、落款

**注意事项：**
- 签名必须由本人或法定代表人亲笔签署
- 法人需加盖公章
- 日期应填写提交起诉状当日

---

## 快速检查清单

提交前请核对：

- [ ] 原告信息是否完整准确
- [ ] 被告信息是否尽可能详细
- [ ] 诉讼请求是否明确具体
- [ ] 事实陈述是否客观完整
- [ ] 证据清单是否与事实对应
- [ ] 法院名称是否准确
- [ ] 签名日期是否齐全
- [ ] 副本份数是否足够（被告人数+1份）
