# 小红书内容合规检查表

> 发布前必读。违规内容将导致笔记被限流、删除，账号被封禁。本表适用于图文和视频笔记。

---

## 🚨 一、绝对禁止内容（零容忍）

以下内容一经发现直接封号，无申诉通道：

| 类别 | 具体内容 |
|------|---------|
| **政治** | 任何国家领导人、政党、军警、国旗相关；历史虚无主义内容 |
| **色情低俗** | 裸露、性暗示、擦边内容；低俗色情小说/影视剪辑 |
| **暴力血腥** | 暴力行为、血腥画面、虐待动物 |
| **赌博彩票** | 彩票预测、赌博平台推广、赌术教学 |
| **毒品** | 毒品制作、贩卖、使用；吸毒相关内容 |
| **诈骗** | 杀猪盘、刷单诈骗、仿冒品牌、虚假宣传 |
| **违禁品** | 枪支、管制刀具、危险化学品买卖 |
| **邪教迷信** | 宗教极端、邪教组织、封建迷信活动 |
| **未成年人** | 涉及未成年人不良内容；未成年人保护相关违规 |

## ⚠️ 二、平台限流内容（软性限制）

以下内容不会封号，但会被限流或推荐降权：

| 类别 | 说明 | 优化建议 |
|------|------|---------|
| **医疗养生** | 不能说"治愈"等绝对化词汇；未经审核的医疗建议 | 用"改善/缓解"；加"请遵医嘱" |
| **金融理财** | 不能推荐具体股票/基金；不能承诺收益 | 只做知识科普，不给投资建议 |
| **品牌对比** | 不得贬低竞品；对比需有依据 | 客观陈述差异，不说"吊打/完败" |
| **政治隐喻** | 任何看似政治隐喻的梗、图案、数字 | 避免政治相关联想内容 |
| **外链二维码** | 笔记中不能出现其他平台外链/二维码 | 引导至评论区或私信 |
| **联系方式** | 手机号、微信号等直接联系方式 | 通过小红书号或私信功能 |
| **侵权内容** | 未授权的品牌 logo、剧照、截图、表情包 | 使用原创或正版授权素材 |
| **虚假信息** | 夸大宣传、虚假数据、伪科普 | 引用正规来源，标注数据出处 |

## 📋 三、标题合规规范

### ✅ 合格标题示例
```
"这款面霜用了3个月，皮肤真的变好了｜真实测评"
"上海周边1小时直达｜适合周末遛娃的小众农场"
"手把手教你做3道简单家常菜，新手也能成功"
```

### ❌ 违规标题示例
```
"抹平皱纹的神药！三天见效！" → 绝对化医疗宣传
"LV、Gucci、Chanel 哪个最垃圾" → 恶意品牌对比
"只需一招，白嫖平台全额退款" → 教唆欺诈
```

### 标题用词红线
```
绝对化词：100%、绝对、保证、必须、神器、神药
欺诈类词：免费、白嫖、薅羊毛、漏洞
比较类词：最垃圾、吊打、完败、碾压（除非客观数据）
```

## 📝 四、正文合规规范

### 敏感话题处理
| 话题 | 风险点 | 安全写法 |
|------|-------|---------|
| **医美/整形** | 不得描述手术过程；不得对比效果承诺 | "本人体验，不构成医疗建议" |
| **减肥/瘦身** | 不得出现具体减重数字承诺 | "配合运动，效果因人而异" |
| **金融理财** | 不得出现具体收益数字；不得推荐具体产品 | "本内容仅供科普，不构成投资建议" |
| **教育** | 不得出现具体分数承诺；不得贬低其他教育方式 | "个人经验分享，理性参考" |
| **宗教** | 不得宣传特定宗教；不得贬低宗教信仰 | 完全避免宗教话题 |

### 违禁词清单（部分）
```
最、第一、顶级、极品、奇迹、神效、特效
立即、马上、马上见效
退款保证、100%退款
全网最低价、最便宜
根治、永不复发
```

### 正确标注格式
```markdown
## 数据来源声明（合规必备）
- 本文数据来源：[权威平台名称+链接]
- 个人体验分享，实际效果因人而异
- 本文不构成医疗/投资/法律建议
- 如有疑问请咨询专业人士
```

## 🖼️ 五、图片/视频合规

| 类型 | 合规要求 |
|------|---------|
| **人像** | 必须获得本人授权；未成年人需监护人同意 |
| **品牌 logo** | 避免完整展示；用马赛克遮盖部分 |
| **影视剧截图** | 不得超过总时长的 20%；需标注来源 |
| **地图** | 需使用正规导航截图；禁止使用军事设施照片 |
| **二维码/水印** | 禁止任何第三方平台二维码/水印 |
| **国旗国徽** | 禁止在背景/装饰中出现 |
| **野生动物** | 禁止保护动物制品展示 |

## 📌 六、评论互动合规

| 场景 | 风险 | 合规做法 |
|------|------|---------|
| **引导私信** | "私信我获取"可能被判定为引流 | "点击主页链接" |
| **评论回复** | 不能留微信号/手机号 | 引导至评论区或小红书群 |
| **私信回复** | 不能发送外链 | 仅小红书站内内容 |
| **置顶评论** | 不能置顶引流内容 | 置顶正面互动内容 |

## ✅ 七、发布前检查流程

### 强制检查项（发布前必须全部通过）

```markdown
## 发布前合规自检

### 标题检查
- [ ] 无绝对化词汇（最/第一/绝对/保证/神器）
- [ ] 无欺诈词汇（免费/白嫖/退款保证）
- [ ] 无恶意品牌对比（吊打/完败/最垃圾）
- [ ] 字数在 20 字以内

### 正文检查
- [ ] 无违禁词（见违禁词清单）
- [ ] 无具体医疗/投资承诺
- [ ] 有数据来源标注（若有数据）
- [ ] 有"个人体验/效果因人而异"声明
- [ ] 无外链/二维码/联系方式

### 图片检查
- [ ] 无他人人像（除非获得授权）
- [ ] 无品牌完整 logo
- [ ] 无二维码/其他平台水印
- [ ] 无国旗/军警元素
- [ ] 图片清晰度达标（≥1080p）

### 标签检查
- [ ] 无品牌硬植入话题
- [ ] 话题数量 3-10 个
- [ ] 无政治相关话题

### 互动设置
- [ ] 开启/关闭评论（根据内容决定）
- [ ] 开启/关闭收藏（根据内容决定）
- [ ] 开启/关闭转发（根据内容决定）
```
