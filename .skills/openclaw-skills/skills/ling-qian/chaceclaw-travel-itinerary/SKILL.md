**简洁输入示例**：
```

```

### 2. 预算控制

**预算分配模型**：
- 背包客：住宿25%，餐饮35%，交通20%，活动15%，其他5%
- 中档：住宿35%，餐饮30%，交通15%，活动15%，其他5%
- 奢华：住宿50%，餐饮25%，交通10%，活动10%，其他5%

### 3. 兴趣优化

**兴趣组合建议**：
- 美食 + 历史：文化深度体验
- 海滩 + 自然：休闲度假
- 购物 + 夜生活：城市探索
- 冒险 + 自然：户外探险
