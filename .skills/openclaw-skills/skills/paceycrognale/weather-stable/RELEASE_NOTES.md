# Release Notes

## v1.0.0

Initial public release.

### Highlights
- stable same-day weather lookup
- Chinese and English city input
- automation-first output modes: plain / pretty / json
- structured weather data instead of HTML scraping
- suitable for OpenClaw agents, Feishu bots, logs, and workflows

### Why it exists
Many weather skills are either documentation-only wrappers or fragile HTML scrapers.
This release focuses on turning weather lookup into a dependable system capability.
