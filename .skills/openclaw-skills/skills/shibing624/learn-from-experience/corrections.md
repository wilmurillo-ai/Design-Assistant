# 更正日志 — 模板

> 该文件会在你首次使用本技能时创建于`~/learn-from-experience/corrections.md`。
> 仅保留最近50条更正，旧条目会评估是否升级留存或归档。


## 示例条目

```markdown
## 2026-02-19

### 14:32 — 代码风格
- **Correction:** "使用2空格缩进，不要用4空格"
- **Context:** 编辑TypeScript文件
- **Count:** 1（首次出现）

### 16:15 — 沟通规范
- **Correction:** "不要用'好问题！'开头回复"
- **Context:** 聊天回复
- **Count:** 3 → **已添加至memory.md**

## 2026-02-18

### 09:00 — Project: website
- **Correction:** "本项目统一使用Tailwind"
- **Context:** CSS样式讨论
- **Action:** 已添加至projects/website.md
```

## Log Format

Each entry includes:
- **Timestamp** — 更正发生的时间
- **Correction** — 用户提出的更正要求
- **Context** — 触发更正的场景
- **Count** — 累计出现次数（用于优化追踪）
- **Action** — 优化后存储的位置（若已优化）
