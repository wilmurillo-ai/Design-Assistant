# 中国税法要点摘要

## 一、企业所得税

### 1.1 应纳税所得额计算

**基本公式**：
```
应纳税所得额 = 收入总额 - 不征税收入 - 免税收入 - 各项扣除 - 允许弥补的以前年度亏损
```

**会计利润与应纳税所得额的差异**：
- 永久性差异：免税收入、不得扣除项目
- 暂时性差异：资产减值准备、折旧年限差异等

### 1.2 收入确认时点

#### 销售货物收入
- 托收承付方式：办妥托收手续时
- 预收款方式：发出商品时
- 需要安装和检验：购买方接受商品并安装检验完毕时
- 支付手续费方式委托代销：收到代销清单时

#### 提供劳务收入
- 安装费：根据安装完工进度确认
- 广告的制作费：根据制作广告的完工进度确认
- 软件费：为特定客户开发软件的根据开发的完工进度确认
- 服务费：含在商品售价内可区分的服务费，在提供服务期间分期确认

#### 特殊收入
- 利息收入：合同约定的债务人应付利息的日期
- 租金收入：合同约定的承租人应付租金的日期
- 特许权使用费收入：合同约定的应付特许权使用费的日期
- 接受捐赠收入：实际收到捐赠资产的日期

### 1.3 税前扣除项目

#### 限额扣除项目

| 项目 | 扣除限额 | 计算基数 |
|------|----------|----------|
| 职工福利费 | 14% | 工资薪金总额 |
| 工会经费 | 2% | 工资薪金总额 |
| 职工教育经费 | 8% | 工资薪金总额 |
| 业务招待费 | 发生额60% vs 收入5‰孰低 | 销售(营业)收入 |
| 广告费和业务宣传费 | 15%（特殊行业30%） | 销售(营业)收入 |
| 公益性捐赠 | 12% | 年度利润总额 |
| 手续费及佣金 | 5%（保险企业18%） | 收入金额 |

#### 不得扣除项目
- 向投资者支付的股息、红利等权益性投资收益款项
- 企业所得税税款
- 税收滞纳金
- 罚金、罚款和被没收财物的损失
- 超过规定标准的捐赠支出
- 赞助支出
- 未经核定的准备金支出
- 与取得收入无关的其他支出

### 1.4 税收优惠政策

#### 高新技术企业
- 税率：15%
- 条件：拥有核心知识产权、产品属于《国家重点支持的高新技术领域》等

#### 小型微利企业
- 应纳税所得额≤300万元：减按25%计入应纳税所得额，按20%税率缴纳（实际税负5%）
- 从业人数：≤300人
- 资产总额：≤5000万元

#### 研发费用加计扣除
- 一般企业：100%加计扣除
- 集成电路、工业母机企业：120%加计扣除
- 预缴时可享受，汇算清缴时补充资料

#### 固定资产加速折旧
- 单位价值不超过500万元的设备器具：允许一次性计入当期成本费用
- 缩短折旧年限：不低于规定年限的60%

### 1.5 特别纳税调整

#### 关联交易管理
- 关联方认定标准：持股25%、借贷资金占实收资本50%以上等
- 同期资料准备：主体文档、本地文档、特殊事项文档
- 转让定价方法：可比非受控价格法、再销售价格法、成本加成法等

#### 受控外国企业（CFC）
- 设立在实际税负低于12.5%的国家（地区）
- 非出于合理经营需要
- 不分配或减少分配利润

## 二、增值税

### 2.1 纳税义务发生时点

#### 一般规定
- 发生应税销售行为，为收讫销售款项或者取得索取销售款项凭据的当天
- 先开具发票的，为开具发票的当天
- 进口货物，为报关进口的当天

#### 具体情形
- 直接收款方式：收到销售款或取得索取销售款凭据的当天
- 托收承付和委托银行收款方式：发出货物并办妥托收手续的当天
- 赊销和分期收款方式：书面合同约定的收款日期当天
- 预收货款方式：货物发出的当天（生产销售生产工期超过12个月的大型机械设备等除外）
- 委托其他纳税人代销：收到代销单位的代销清单或者收到全部或者部分货款的当天

### 2.2 进项税额抵扣

#### 准予抵扣的进项税额
- 从销售方取得的增值税专用发票上注明的增值税额
- 从海关取得的海关进口增值税专用缴款书上注明的增值税额
- 购进农产品，按收购发票或销售发票注明的买价和9%扣除率计算
- 国内旅客运输服务（注明旅客身份信息的票据）

#### 不得抵扣的情形
- 用于简易计税方法计税项目、免征增值税项目、集体福利或个人消费
- 非正常损失的购进货物及相关劳务、交通运输服务
- 非正常损失的在产品、产成品所耗用的购进货物、劳务、交通运输服务
- 国务院规定的其他项目

#### 进项转出
- 已抵扣进项税额的购进货物（不含固定资产）、劳务、服务，发生不得抵扣情形的
- 计算公式：不得抵扣的进项税额＝已抵扣进项税额×不动产净值率

### 2.3 视同销售

以下情形视同销售货物：
- 将货物交付其他单位或者个人代销
- 销售代销货物
- 设有两个以上机构并实行统一核算的纳税人，将货物从一个机构移送其他机构用于销售（同一县市除外）
- 将自产、委托加工的货物用于集体福利或者个人消费
- 将自产、委托加工或者购进的货物作为投资、分配给股东或投资者、无偿赠送

### 2.4 税率及征收率

| 类型 | 税率/征收率 | 适用范围 |
|------|-------------|----------|
| 一般纳税人 | 13% | 销售货物、劳务、有形动产租赁 |
| 一般纳税人 | 9% | 交通运输、邮政、基础电信、建筑、不动产租赁、销售不动产、转让土地使用权、销售或进口特定货物 |
| 一般纳税人 | 6% | 金融服务、生活服务、现代服务（除租赁）、增值电信服务 |
| 一般纳税人 | 0% | 出口货物（除国务院另有规定） |
| 小规模纳税人 | 3% | 征收率（2027年底前减按1%） |
| 简易计税 | 5% | 销售不动产、不动产租赁等特定业务 |

## 三、个人所得税

### 3.1 居民个人综合所得

**应纳税所得额计算**：
```
应纳税所得额 = 年度收入额 - 6万元 - 专项扣除 - 专项附加扣除 - 其他扣除
```

**收入额确定**：
- 工资薪金所得：收入全额
- 劳务报酬所得：收入减除20%费用后的余额
- 稿酬所得：收入减除20%费用后再减按70%计算
- 特许权使用费所得：收入减除20%费用后的余额

### 3.2 专项附加扣除

| 项目 | 扣除标准 | 备注 |
|------|----------|------|
| 子女教育 | 2000元/月/孩 | 3岁至博士阶段 |
| 继续教育 | 400元/月或3600元/年 | 学历或职业资格 |
| 大病医疗 | 最高80000元/年 | 个人负担超过15000元部分 |
| 住房贷款利息 | 1000元/月 | 首套住房贷款 |
| 住房租金 | 800-1500元/月 | 按城市规模分档 |
| 赡养老人 | 3000元/月 | 60岁以上父母 |
| 3岁以下婴幼儿照护 | 2000元/月/孩 | 2022年起实施 |

### 3.3 经营所得

**适用税率表**（5级超额累进）：
- 不超过30000元：5%
- 超过30000至90000元：10%
- 超过90000至300000元：20%
- 超过300000至500000元：30%
- 超过500000元：35%

## 四、其他重要税种

### 4.1 印花税

**应税凭证及税率**：
- 借款合同：借款金额的0.05‰
- 融资租赁合同：租金的0.05‰
- 买卖合同：价款的0.3‰
- 承揽合同：报酬的0.3‰
- 建设工程合同：价款的0.3‰
- 产权转移书据：价款的0.5‰（商标专用权、著作权、专利权、专有技术使用权转让书据为0.3‰）
- 营业账簿：实收资本+资本公积的0.25‰

### 4.2 房产税

**计税依据**：
- 从价计征：房产原值×(1-扣除比例)×1.2%
- 从租计征：租金收入×12%（个人出租住房4%）

**房产原值确定**：
- 包括与房屋不可分割的各种附属设备或一般不单独计算价值的配套设施
- 纳税人对原有房屋进行改建、扩建的，要相应增加房屋的原值

### 4.3 城镇土地使用税

**计税依据**：实际占用的土地面积

**税额标准**：
- 大城市：1.5-30元/平方米/年
- 中等城市：1.2-24元/平方米/年
- 小城市：0.9-18元/平方米/年
- 县城、建制镇、工矿区：0.6-12元/平方米/年

## 五、税务稽查重点

### 5.1 高风险行业
- 房地产、建筑安装
- 医药、医疗器械
- 医美、教育培训
- 互联网平台企业
- 高收入人群（主播、明星等）

### 5.2 重点稽查领域
- 隐瞒收入（私户收款、阴阳合同）
- 虚开发票（无真实交易、资金回流）
- 虚列成本（虚假采购、虚构劳务）
- 关联交易转让定价
- 骗取出口退税
- 违规享受税收优惠

### 5.3 税务合规建议
- 建立完善的税务内控制度
- 确保业务真实性，留存完整证据链
- 定期进行税务健康检查
- 重大交易前进行税务影响评估
- 关注税收政策变化，及时调整
