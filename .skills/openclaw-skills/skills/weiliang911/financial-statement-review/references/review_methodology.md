# 财务报表审查方法论

## 一、审查方法论框架

### 1.1 风险导向审计方法

**核心思想**：识别和评估财务报表重大错报风险，设计和实施进一步审计程序

**实施步骤**：
1. **了解被审计单位及其环境**
   - 行业状况、监管环境
   - 性质、目标、战略
   - 财务业绩衡量和评价
   - 内部控制

2. **识别和评估重大错报风险**
   - 财务报表层次风险
   - 认定层次风险
   - 特别风险
   - 仅通过实质性程序无法应对的风险

3. **设计和实施进一步审计程序**
   - 控制测试（如适用）
   - 实质性程序
   - 细节测试
   - 实质性分析程序

### 1.2 分析程序方法

#### 趋势分析法
**适用场景**：连续多期财务数据比较

**分析维度**：
- 绝对金额变动
- 变动百分比
- 定基比率
- 环比比率

**异常信号识别**：
- 变动幅度超过预期阈值（通常±20%或±30%）
- 变动方向与行业趋势相反
- 第四季度异常波动

#### 比率分析法

**偿债能力指标**：
| 指标 | 计算公式 | 正常范围 |
|------|----------|----------|
| 流动比率 | 流动资产/流动负债 | 1.5-2.5 |
| 速动比率 | (流动资产-存货)/流动负债 | 0.8-1.2 |
| 资产负债率 | 负债总额/资产总额 | 40%-60% |
| 利息保障倍数 | EBIT/利息费用 | >3 |

**盈利能力指标**：
| 指标 | 计算公式 | 分析要点 |
|------|----------|----------|
| 毛利率 | (收入-成本)/收入 | 行业差异大，关注异常波动 |
| 净利率 | 净利润/营业收入 | 持续下降需关注 |
| ROE | 净利润/净资产 | 杜邦分析核心指标 |
| ROA | 净利润/总资产 | 资产运用效率 |

**营运能力指标**：
| 指标 | 计算公式 | 异常信号 |
|------|----------|----------|
| 应收账款周转天数 | 365/(收入/平均应收账款) | 持续上升可能虚增收入 |
| 存货周转天数 | 365/(成本/平均存货) | 异常上升可能少转成本 |
| 应付账款周转天数 | 365/(采购/平均应付账款) | 异常延长可能资金紧张 |
| 现金周期 | 存货天数+应收天数-应付天数 | 负值通常有利 |

#### 结构分析法

**资产负债表结构**：
- 资产结构：流动资产占比、固定资产占比
- 资本结构：负债占比、权益占比
- 关注：轻资产vs重资产、高杠杆vs低杠杆

**利润表结构**：
- 成本费用结构：营业成本率、期间费用率
- 关注：费用率异常变动、收入成本配比

### 1.3 财务造假识别方法

#### 收入舞弊识别

**常见手段**：
1. 虚构收入（空转、关联交易非关联化）
2. 提前确认收入（发出商品即确认）
3. 利用完工百分比法调节
4. 退货不入账

**识别信号**：
- 收入大幅增长但现金流未同步增长
- 应收账款周转天数异常延长
- 收入集中在年末或季末
- 毛利率异常高于行业水平
- 客户集中度异常（特别是关联方）

**验证程序**：
- 收入与增值税申报表核对
- 收入与银行流水核对
- 主要客户背景调查
- 期后退货检查

#### 成本费用舞弊识别

**常见手段**：
1. 少结转成本（调节毛利率）
2. 费用资本化
3. 推迟确认费用
4. 利用会计估计调节（折旧、坏账）

**识别信号**：
- 毛利率异常波动
- 存货周转天数异常延长
- 资本性支出突增
- 费用率异常下降
- 非经常性损益占比过高

**验证程序**：
- 存货盘点和计价测试
- 资本性支出与收益性支出区分
- 费用截止测试
- 折旧和摊销重新计算

#### 资产舞弊识别

**常见手段**：
1. 虚构资产（虚假采购、虚增在建工程）
2. 少计提减值准备
3. 关联方资金占用

**识别信号**：
- 资产结构异常（预付、其他应收款突增）
- 在建工程长期挂账不转固
- 商誉占净资产比例过高
- 其他应收款余额异常

**验证程序**：
- 大额预付款项背景调查
- 在建工程进度核实
- 资产减值测试复核
- 关联方往来识别

## 二、报表间勾稽关系验证

### 2.1 基本勾稽关系

**资产负债表与利润表**：
```
未分配利润期末数 = 未分配利润期初数 + 净利润 - 利润分配
```

**资产负债表与现金流量表**：
```
现金及现金等价物期末数 - 期初数 = 现金流量净增加额
货币资金（剔除受限资金）≈ 现金及现金等价物
```

**利润表与现金流量表**：
```
经营活动现金流量净额 ≈ 净利润 + 折旧摊销 ± 营运资金变动
```

### 2.2 详细勾稽关系检查

#### 资产负债表内部勾稽
```
资产总计 = 负债合计 + 所有者权益合计
流动资产合计 = 各项流动资产之和
非流动资产合计 = 各项非流动资产之和
```

#### 利润表内部勾稽
```
营业利润 = 营业收入 - 营业成本 - 税金及附加 - 期间费用 ± 其他收益等
利润总额 = 营业利润 + 营业外收入 - 营业外支出
净利润 = 利润总额 - 所得税费用
```

#### 现金流量表内部勾稽
```
现金及现金等价物净增加额 = 经营+投资+筹资活动现金流量净额
期初现金及现金等价物 + 净增加额 = 期末现金及现金等价物
```

#### 跨表勾稽关系

**货币资金**：
```
货币资金（资产负债表）≈ 现金及现金等价物（现金流量表）+ 受限资金
```

**长期资产**：
```
固定资产原值变动 = 购置 + 在建工程转入 - 处置/报废
固定资产净值 = 原值 - 累计折旧 - 减值准备
```

**往来款项**：
```
应收账款变动 = 收入（赊销）- 回款
应付账款变动 = 采购（赊购）- 付款
```

**存货**：
```
存货变动 = 采购 + 人工制造费用 - 营业成本
```

## 三、重点科目审查程序

### 3.1 货币资金审查

**审查目标**：存在、完整性、权利和义务、计价和分摊

**关键程序**：
1. 获取银行对账单，与账面余额核对
2. 编制银行余额调节表
3. 函证银行存款余额
4. 检查受限资金（保证金、冻结资金）
5. 抽查大额资金收支

**风险提示**：
- 定期存款质押未披露
- 体外资金循环（虚增收入）
- 关联方资金占用
- 坐支现金

### 3.2 应收账款审查

**审查目标**：真实性、可回收性、坏账准备充分性

**关键程序**：
1. 账龄分析，关注长期挂账款项
2. 期后回款检查
3. 函证重要客户
4. 坏账准备计提测试
5. 关联方往来识别

**风险提示**：
- 虚构收入导致应收虚增
- 坏账准备计提不足
- 关联方资金 disguised as 应收

### 3.3 存货审查

**审查目标**：存在、完整性、计价准确性

**关键程序**：
1. 存货监盘（观察、抽盘）
2. 计价测试（先进先出、加权平均）
3. 跌价准备测试
4. 出入库截止测试
5. 周转分析

**风险提示**：
- 虚构存货（虚假采购）
- 少转成本调节利润
- 呆滞存货未计提跌价

### 3.4 收入审查

**审查目标**：发生、完整性、准确性、截止、分类

**关键程序**：
1. 收入确认政策复核
2. 截止测试（资产负债表日前后）
3. 主要合同条款审查
4. 毛利率分析（分产品、分客户、分月份）
5. 收入与成本配比分析

**风险提示**：
- 提前确认收入
- 虚构收入（空转、关联交易）
- 收入成本不配比

### 3.5 关联方交易审查

**审查目标**：完整性、公允性、披露充分性

**关键程序**：
1. 关联方识别（全面、穿透）
2. 关联交易定价公允性分析
3. 决策程序合规性检查
4. 披露完整性审查

**风险提示**：
- 隐瞒关联方关系
- 非公允定价转移利润
- 关联方资金占用
- 关联方担保未披露

## 四、税务合规审查程序

### 4.1 增值税审查

**审查要点**：
1. 收入确认与增值税纳税义务发生时点差异
2. 视同销售处理合规性
3. 进项税额抵扣合规性
4. 进项转出处理准确性
5. 留抵税额确认

**风险提示**：
- 应视同销售未计提销项税
- 不得抵扣进项已抵扣
- 虚开发票风险

### 4.2 企业所得税审查

**审查要点**：
1. 收入确认税会差异调整
2. 成本费用税前扣除合规性
3. 税收优惠适用条件
4. 资产损失税前扣除
5. 关联交易转让定价

**风险提示**：
- 限额扣除项目超标未调整
- 不得扣除项目已扣除
- 税收优惠条件不满足
- 特别纳税调整风险

### 4.3 个人所得税审查

**审查要点**：
1. 工资薪金代扣代缴
2. 劳务报酬、股息红利个税
3. 专项附加扣除真实性
4. 股权激励税务处理

**风险提示**：
- 私户发放工资未代扣个税
- 利用个独/合伙企业避税
- 股权激励税务处理错误

## 五、审计工作底稿编制

### 5.1 底稿基本要素

**每份底稿应包含**：
1. 审计程序说明
2. 审计证据
3. 审计结论
4. 索引编号
5. 编制人和复核人签名

### 5.2 底稿索引体系

**常用索引分类**：
- A - 初步业务活动
- B - 风险评估
- C - 控制测试
- D - 实质性程序（资产类）
- E - 实质性程序（负债类）
- F - 实质性程序（权益类）
- G - 实质性程序（损益类）
- H - 完成阶段

### 5.3 重大事项记录

**应记录的重大事项**：
- 重大会计估计判断
- 重大审计判断
- 与治理层沟通事项
- 审计范围受限
- 专家工作利用
