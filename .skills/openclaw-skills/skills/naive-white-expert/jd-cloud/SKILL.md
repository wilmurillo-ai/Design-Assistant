# JD Cloud (京东云)

京东云计算服务平台，为企业提供稳定高效的云基础设施。

## 云服务器 ECS

### 实例类型
- 通用型
- 计算型
- 内存型
- GPU型
- 高IO型

### 实例管理
- 创建实例
- 配置规格
- 选择地域
- 选择镜像
- 网络配置
- 安全组设置

## 云存储 OSS

### 存储类型
- 标准存储
- 低频存储
- 归档存储

### 存储功能
- Bucket管理
- 文件上传下载
- 访问权限控制
- 生命周期管理
- 防盗链设置

### CDN服务
