## 🎯 使用示例

### 分析APK加固类型
```bash
# 英语环境
android-armor-breaker analyze --apk app.apk --language en-US

# 中文环境  
android-armor-breaker analyze --apk app.apk --language zh-CN
```

### 执行智能脱壳
```bash
# 自动选择最佳策略
android-armor-breaker --package com.example.app --verbose

# 指定Root内存提取策略
android-armor-breaker --package com.example.app --strategy root --verbose

# 指定Frida动态脱壳
android-armor-breaker --package com.example.app --strategy frida --verbose
```
