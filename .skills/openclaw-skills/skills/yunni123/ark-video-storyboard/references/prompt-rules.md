# Prompt Rules

## Character Rule

If any human character appears and the user does not specify ethnicity, default to East Asian appearance.

## Prompt Priorities

Write prompts in this order of importance:

1. Subject identity
2. Action or motion
3. Camera angle / shot language
4. Environment
5. Lighting and mood
6. Ending state of the shot

## Good Prompt Characteristics

- Specific visual action
- Clear camera perspective
- Clear lighting behavior
- Coherent emotional tone
- Enough continuity with previous/next segment

## Avoid

- Keyword soup with no scene logic
- Contradictory lighting instructions
- Changing the character’s identity across segments
- Abrupt style jumps unless explicitly requested
- Unnecessary technical jargon
