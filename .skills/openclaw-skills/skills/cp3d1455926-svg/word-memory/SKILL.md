# 📚 Word Memory - 单词记忆助手

## 触发规则

**关键词：**
- 背单词、记单词、单词记忆
- 每日单词、单词打卡
- 艾宾浩斯、记忆曲线
- 四级、六级、考研、托福、雅思
- 测试我、默写

**场景：**
- 用户想背单词但需要计划
- 用户需要每日单词推送
- 用户想测试记忆效果
- 用户查询单词释义/例句

## 功能描述

### 核心功能

1. **艾宾浩斯记忆曲线**
   - 自动安排复习时间
   - 5 个复习节点：5min、30min、12h、1d、2d、4d、7d、15d
   - 智能调整难度

2. **词库选择**
   - 四级词汇（4500 词）
   - 六级词汇（6000 词）
   - 考研词汇（5500 词）
   - 托福词汇（8000 词）
   - 雅思词汇（8000 词）
   - GRE 词汇（12000 词）
   - 自定义词库

3. **每日单词推送**
   - 定时推送（可设置时间）
   - 每日新词 + 复习词
   - 数量可配置（10/20/50/100）

4. **单词测试**
   - 英译中测试
   - 中译英测试
   - 拼写测试
   - 听力测试（可选）

5. **单词详情**
   - 释义（多义项）
   - 音标（英式/美式）
   - 例句（3-5 句）
   - 词根词缀
   - 近义词/反义词
   - 常用搭配

### 学习模式

- 📖 **学习模式**：新词学习，显示完整信息
- 📝 **测试模式**：隐藏释义，测试记忆
- ⚡ **快速模式**：只记核心释义
- 🎯 **重点模式**：只复习错误率高的词

## 权限需求

- [x] 网络访问（词典 API）
- [x] 文件读写（学习进度存储）
- [x] 定时任务（每日推送）

## 数据源

- 有道词典 API
- 柯林斯词典
- 剑桥词典
- 开源词库（CEDICT 等）

1. She is persistent in her efforts.
2. The problem is persistent.

🌳 词根：per-(始终) + sist(站) + -ent → 始终站着 → 坚持的
