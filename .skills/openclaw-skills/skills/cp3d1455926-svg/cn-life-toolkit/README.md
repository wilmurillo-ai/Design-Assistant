# 🇨🇳 CN Life Toolkit - 中国生活服务工具包

## 📖 功能说明

支持天气查询（街道级）、限行提醒、油价查询、快递物流跟踪、公交换乘查询等。

## 🚀 使用方法

### 在 OpenClaw 中使用

```
北京天气怎么样
今天限行吗
查一下油价
快递单号 123456789
从北京到上海怎么坐公交
```
