## 📖 功能说明

根据场景、心情推荐音乐，支持歌词获取、歌单管理、热门榜单查询。

### 核心功能
- 🎧 **场景推荐** - 学习/工作/睡眠/运动/通勤/派对
- 😊 **心情推荐** - 开心/难过/放松/兴奋/治愈/emo
- 📝 **歌词获取** - 网易云歌词实时查询
- 🔥 **热门榜单** - 飙升榜/新歌榜
- ❤️ **收藏管理** - 收藏喜欢的歌曲
- 📋 **歌单管理** - 创建和管理个人歌单

## 🔧 配置说明

### 网易云音乐 API

默认使用官方 API，无需配置即可使用基础功能。

如需更稳定的服务，可部署自己的 API 服务：

1. 参考：https://github.com/Binaryify/NeteaseCloudMusicApi
2. 部署后修改 `netease_api.py` 中的 `NETEASE_BASE_URL`

### 缓存配置

- 缓存文件：`netease_cache.json`
- 缓存过期：1 小时
- 自动重试：最多 3 次
