# 💻 Coding Lite - 轻量级编码助手

## 📖 功能说明

支持 Python 脚本生成与执行、Excel/WPS 函数自动化、小程序代码生成、SQL 查询助手。

## 🚀 使用方法

### 在 OpenClaw 中使用

```
帮我写个 Python 脚本读取 Excel

```
