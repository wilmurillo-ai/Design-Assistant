# Memory-Master 测试脚本

## 快速测试

```bash
# 1. 测试记忆分类
node scripts/test-classifier.js

# 2. 测试时间树
node scripts/test-time-tree.js

# 3. 测试检索功能
node scripts/test-retrieve.js

# 4. 完整测试套件
npm test
```
