# Plot Management Guide

## Story Structure

### Three-Act Structure
1. **Act I: Setup** (25%)
   - Introduction to world and characters
   - Inciting incident
   - First plot point

2. **Act II: Confrontation** (50%)
   - Rising action
   - Midpoint twist
   - Complications and obstacles
   - Second plot point

3. **Act III: Resolution** (25%)
   - Climax
   - Falling action
   - Resolution

### Chapter Structure
Each chapter should have:
- **Goal**: What happens
- **Conflict**: What goes wrong
- **Action**: What characters do
- **Result**: Outcome
- **Hook**: Reason to continue

## Plot Tracking

### Main Plot Arc
Track the central storyline:
- **Setup**: Initial situation
- **Inciting Incident**: What starts the story
- **Rising Action**: Building tension
- **Climax**: Peak moment
- **Falling Action**: Aftermath
- **Resolution**: How it ends

### Subplots
Track secondary storylines:
- **Romance subplot**: Relationship development
- **Character subplot**: Personal growth
- **Mystery subplot**: Unanswered questions
- **Conflict subplot**: Ongoing tensions

### Plot Points
Key moments in the story:
1. **Opening**: First impression
2. **Inciting Incident**: Call to action
3. **First Plot Point**: Commitment
4. **Midpoint**: Revelation or reversal
5. **Second Plot Point**: Darkest moment
6. **Climax**: Final confrontation
7. **Resolution**: New normal

## Timeline Management

### Chronology
Track when events happen:
- **Story Time**: Day 1, Day 2, etc.
- **Flashbacks**: Past events
- **Flash-forwards**: Future hints
- **Parallel Timelines**: Multiple perspectives

### Pacing
Control story speed:
- **Fast Pacing**: Action scenes, short chapters
- **Slow Pacing**: Reflection, description, dialogue
- **Varied Pacing**: Mix for engagement

## Plot Consistency Checklist

- [ ] Timeline makes sense
- [ ] Cause and effect are clear
- [ ] Character actions drive plot
- [ ] Stakes escalate appropriately
- [ ] Subplots connect to main plot
- [ ] Foreshadowing pays off
- [ ] No plot holes
- [ ] Satisfying resolution

## Common Plot Issues

1. **Plot Holes**: Logic gaps
2. **Deus Ex Machina**: Convenient solutions
3. **Lost Threads**: Unresolved subplots
4. **Rushed Ending**: Too fast conclusion
5. **Slow Middle**: Dragging second act
6. **Coincidences**: Too convenient
