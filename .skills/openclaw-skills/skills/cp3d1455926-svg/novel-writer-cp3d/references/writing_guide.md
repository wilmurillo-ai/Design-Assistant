# Novel Writing Style Guide

## Formatting Standards

### Chapter File Naming
```
XX-第X章 章节标题.md
```
Examples:
- `01-第一章 觉醒的边缘.md`
- `10-第十章 摊牌.md`
- `23-第二十三章 回声.md`

### Chapter Structure
```markdown
# XX-第X章 章节标题.md

## 第X章：章节标题

## 作者：作者名

## 日期：YYYY年MM月DD日

## 日期说明（可选）

---

正文内容

---

**第X天/章，结束。**

**关键信息总结**
**但……**

**后续预告/思考**
```

## Writing Style

### Paragraph Structure
- **Keep paragraphs manageable**: 3-8 sentences typically
- **Vary paragraph length**: Mix short and long for rhythm
- **One idea per paragraph**: Focus on single point
- **Use dialogue breaks**: Start new paragraph for each speaker

### Dialogue Formatting
```markdown
"This is dialogue," he said.

She replied, "This is the response."

"Multiple sentences in one speech."
"Still the same speaker, new paragraph."
```

### Pacing Guidelines
- **Action scenes**: Short sentences, short paragraphs
- **Emotional scenes**: Longer paragraphs, introspection
- **Dialogue scenes**: Back and forth, quick exchanges
- **Description scenes**: Slower, more detail

## Content Guidelines

### Chapter Length
- **Minimum**: 1000 words
- **Ideal**: 2000-4000 words
- **Maximum**: 6000 words (consider splitting)

### Scene Elements
Each scene should have:
1. **Setting**: Where and when
2. **Characters**: Who's present
3. **Goal**: What they want
4. **Conflict**: What's stopping them
5. **Action**: What happens
6. **Result**: Outcome
7. **Transition**: Lead to next scene

### Consistency Checks
- [ ] Character names spelled correctly
- [ ] Timeline is clear
- [ ] Locations are consistent
- [ ] Technology level matches setting
- [ ] Tone matches genre
- [ ] Point of view is consistent

## Common Issues to Avoid

1. **Head-hopping**: Switching POV mid-scene
2. **Info-dumping**: Too much exposition
3. **Telling not showing**: Stating instead of demonstrating
4. **Purple prose**: Overly flowery language
5. **Repetition**: Repeating information
6. **Passive voice**: Overuse of "was" and "were"
7. **Filter words**: "He saw", "she felt" instead of direct description

## Editing Checklist

Before finalizing a chapter:
- [ ] Spelling and grammar checked
- [ ] Formatting is consistent
- [ ] Dialogue tags are varied
- [ ] No unnecessary adverbs
- [ ] Show, don't tell
- [ ] Chapter advances plot or character
- [ ] Satisfying ending hook
