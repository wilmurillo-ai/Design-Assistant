## 🚀 使用方式

### 文案生成
```bash
# 生成技能分享文案
python scripts/xhs_copywriter.py --type skill --topic "效率工具"

# 生成科技新闻文案
python scripts/xhs_copywriter.py --type news --topic "AI 大模型"

# 生成学习内容文案
python scripts/xhs_copywriter.py --type study --topic "Python 学习"
```

### 标签推荐
```bash
# 推荐标签
python scripts/tag_recommender.py --topic "效率工具" --count 15

# 分析热门标签
python scripts/tag_recommender.py --analyze trending --category tech
```

### 内容日历
```bash
# 生成 30 天计划
python scripts/content_calendar.py --days 30 --output calendar.md

# 查看本周计划
python scripts/content_calendar.py --view week
```

### 数据分析
```bash
# 生成数据报告
python scripts/analytics_report.py --days 7 --output report.md

# 竞品分析
python scripts/analytics_report.py --competitor --compare 3
```

## 📊 功能详情

### 智能文案生成

**输入**：
- 主题/话题
- 内容类型（技能/新闻/学习）
- 目标受众
- 语气风格

**输出**：
- 标题（5 个备选）
- 正文（500-1000 字）
- 标签（10-15 个）
- 封面建议

**示例**：
```
主题：效率工具
类型：技能分享

输出：
标题：《3 个效率神器，打工人必备！》
正文：[完整文案]
标签：#效率工具 #打工人 #神器 #软件推荐 ...
封面：紫色渐变背景 + 大字标题
```

## 📝 示例工作流

### 发布一篇笔记的完整流程

```bash
# 1. 生成文案
python scripts/xhs_copywriter.py --type skill --topic "效率工具" --output draft.md

# 2. 推荐标签
python scripts/tag_recommender.py --topic "效率工具" --count 15 --output tags.txt

# 3. 查看内容日历
python scripts/content_calendar.py --view week

# 4. 编辑完善文案
# 手动编辑 draft.md

# 5. 生成封面建议
python scripts/cover_advisor.py --topic "效率工具" --style modern

# 6. 发布笔记
python scripts/cdp_publish.py --reuse-existing-tab publish \
  --title "3 个效率神器，打工人必备！" \
  --content-file draft.md \
  --images cover.jpg,image1.jpg,image2.jpg

# 7. 追踪数据
python scripts/analytics_report.py --note-id [笔记 ID] --days 7
```

## 💡 爆款公式
