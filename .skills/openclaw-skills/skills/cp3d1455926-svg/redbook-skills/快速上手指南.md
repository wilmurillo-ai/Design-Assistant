## 🎯 使用教程

### 1️⃣ 生成文案

**生成技能分享文案**：
```bash
python scripts/xhs_copywriter.py --type skill --topic "效率工具" --output draft.md
```

**生成科技新闻文案**：
```bash
python scripts/xhs_copywriter.py --type news --topic "AI 大模型" --output news_draft.md
```

**生成学习内容文案**：
```bash
python scripts/xhs_copywriter.py --type study --topic "Python 学习" --output study_draft.md
```

**输出示例**：

### 2️⃣ 推荐标签

**基础推荐**：
```bash
python scripts/tag_recommender.py --topic "效率工具" --count 15
```

**指定策略**：
```bash
# 激进策略（多用热门标签）
python scripts/tag_recommender.py --topic "科技" --strategy aggressive

# 保守策略（少用热门标签）
python scripts/tag_recommender.py --topic "学习" --strategy conservative
```

**导出标签**：
```bash
python scripts/tag_recommender.py --topic "效率工具" --output tags.txt
```

**输出示例**：
```
🏷️ 标签推荐（主题：效率工具）
==================================================
策略：balanced
数量：15 个

推荐标签：
#效率工具 #打工人 #神器 #软件推荐 #职场 #干货 
#实用 #免费 #学生党 #宝藏 #安利 #好物分享 
#教程 #科技 #效率工具
```

**生成 30 天计划**：

```bash
python scripts/content_calendar.py --days 30 --start 2024-03-25 --output march_calendar.md
```

## 📋 完整工作流

### 发布一篇笔记的完整流程

```bash
# 1. 生成文案草稿
python scripts/xhs_copywriter.py --type skill --topic "效率工具" --output draft.md

# 2. 推荐标签
python scripts/tag_recommender.py --topic "效率工具" --count 15 --output tags.txt

# 3. 查看内容日历（确认发布时间）
python scripts/content_calendar.py --view week

# 4. 编辑完善文案
# 用文本编辑器打开 draft.md，手动完善内容

# 5. 准备图片/封面
# 使用 Canva/稿定设计等工具制作封面

# 6. 发布笔记（使用原有的 Xiaohongshu Assistant）
python scripts/cdp_publish.py --reuse-existing-tab publish \
  --title "3 个效率神器，打工人必备！" \
  --content-file draft.md \
  --images cover.jpg,image1.jpg,image2.jpg

# 7. 追踪数据（发布后 7 天）
python scripts/analytics_report.py --note-id [笔记 ID] --days 7
```

## 🎯 运营建议

### 新手期（0-1000 粉）
- 📝 每周 3-5 篇笔记
- 🎯 垂直领域内容
- 💬 积极互动回复
- 📊 分析数据优化

### 内容比例
```
技能分享：60% （核心内容）
科技新闻：30% （引流内容）
学习内容：10% （个人成长）
```

### 最佳发布时间
- 🌅 早上：7:00-9:00（通勤）
- 🍽️ 中午：12:00-14:00（午休）
- 🌆 晚上：19:00-22:00（睡前）⭐ 最佳
