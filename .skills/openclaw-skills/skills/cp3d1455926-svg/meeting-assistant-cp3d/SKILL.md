# 📋 Meeting Assistant - 会议助手

## 触发规则

**关键词：**
- 会议纪要、会议记录、会议总结
- 待办事项、行动计划
- 安排会议、日程

**场景：**
- 用户需要整理会议纪要
- 用户需要提取待办事项
- 用户需要安排会议日程

## 功能描述

### 核心功能

1. **会议纪要生成**
   - 从录音/文字生成纪要
   - 提取关键决策
   - 总结讨论要点

2. **待办事项提取**
   - 识别行动项
   - 分配负责人
   - 设置截止日期

3. **日程安排**
   - 会议时间协调
   - 会议室预订
   - 提醒通知

4. **会议模板**
   - 站立会模板
   - 周会模板
   - 项目评审模板

## 权限需求

- [x] 文件读写（纪要存储）
- [x] 定时任务（提醒）
- [ ] 外部 API（日历集成）

## 示例对话

```
用户：帮我整理这个会议纪要...
助手：✅ 已生成会议纪要：

📋 **会议纪要** - 项目周会
📅 2026-03-16 14:00-15:00
👥 参会：Jake, Alice, Bob

📝 **讨论要点**
1. 项目进度：前端完成 80%，后端完成 60%
2. 问题：API 接口延迟，需要协调
3. 决定：下周三前完成接口对接

✅ **待办事项**
1. [ ] Jake - 完成前端页面（截止：3/20）
2. [ ] Alice - 修复 API bug（截止：3/18）
3. [ ] Bob - 准备测试数据（截止：3/19）

📅 **下次会议**
- 时间：2026-03-23 14:00
- 主题：项目评审

需要我发送提醒给相关人员吗？
```
