# 🚀 Proactive Agent - 主动式代理

## 📖 功能说明

将 AI 从任务执行者转变为主动伙伴，支持 WAL Protocol、Working Buffer、自主定时任务。

## 🚀 使用方法

```
设置主动检查
查看主动任务
配置 WAL 协议
```
