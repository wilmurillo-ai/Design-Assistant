## 📖 功能说明

根据心情、类型推荐电影，支持豆瓣评分查询、观影记录管理、Top250 榜单。

### 核心功能
- 😊 **心情推荐** - 开心/难过/放松/刺激/烧脑/累
- 🎭 **类型推荐** - 科幻/爱情/悬疑/喜剧/治愈...
- ⭐ **豆瓣评分** - 实时查询电影评分和详情
- 🏆 **Top250 榜单** - 豆瓣经典榜单
- 📝 **观影记录** - 记录看过的电影和评分
- 📌 **想看清单** - 收藏想看的电影

---

## 🚀 快速开始

### 在 OpenClaw 中使用

```
# 心情推荐
今天有点累，推荐电影
最近很兴奋，有什么电影推荐
心情不好，推荐治愈的电影

# 类型推荐
推荐科幻电影
想看悬疑片
推荐喜剧电影

# 查询评分
查一下星际穿越的评分
盗梦空间豆瓣评分多少

# 榜单
看看 Top250
豆瓣热门电影

# 观影记录
我看了星际穿越，评分 5 分
查看我的观影记录
把盗梦空间加入想看
```

### Python 脚本调用

```python
from movie_recommender import main

# 心情推荐
result = main("今天有点累，推荐电影")
print(result)

# 类型推荐
result = main("推荐科幻电影")
print(result)

# 查询评分
result = main("查一下星际穿越的评分")
print(result)

# Top250
result = main("看看 Top250")
print(result)
```

## 🔧 配置说明

### 豆瓣 API（可选）

默认使用网页爬虫方案，无需 API Key。

如需更稳定的服务，可申请豆瓣 API Key：

1. 申请地址：https://developers.douban.com/
2. 编辑 `douban_api.py`，填入 API Key：
   ```python
   DOUBAN_API_KEY = "your_api_key"
   ```

### 缓存配置

- 缓存文件：`douban_cache.json`
- 缓存过期：1 小时
- 自动重试：最多 3 次
