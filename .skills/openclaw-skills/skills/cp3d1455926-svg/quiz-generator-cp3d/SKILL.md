# 测试/问卷生成器

## 核心功能

### 1. 测试题生成
- 根据主题自动生成测试题
- 支持多种题型（单选、多选、量表）
- 自动平衡选项分布

### 2. 评分逻辑配置
- 维度划分与权重设置
- 计分规则配置
- 结果匹配算法

### 3. 结果页面生成
- 多种结果类型模板
- 个性化解读文案
- 分享引导设计

### 4. 问卷设计
- 调查问卷模板
- 用户反馈问卷
- 市场调研问卷

## 评分逻辑

### 简单计分法
```
每个选项对应一个类型
统计各类型选项数量
数量最多的类型即为结果
```

### 权重计分法
```
每个选项有维度权重
如：A→外向 +2, 内向 -1
累加各维度得分
得分最高的维度为结果
```

### 阈值判定法
```
设置多个阈值区间
总分在不同区间对应不同结果
如：0-10 分→类型 A, 11-20 分→类型 B
```

## 题目设计原则

### 1. 题目数量
- 快速测试：5-8 题（1-2 分钟）
- 标准测试：10-15 题（3-5 分钟）
- 深度测试：20-30 题（8-10 分钟）

### 2. 选项设计
- 每个选项长度相近
- 避免明显"正确"答案
- 选项之间互斥
- 覆盖所有可能性

### 3. 语言风格
- 口语化、易理解
- 避免专业术语
- 情境具体化
- 情绪共鸣

### 4. 信度效度
- 题目间内部一致性
- 维度划分清晰
- 结果可解释性强
- 避免引导性问题
