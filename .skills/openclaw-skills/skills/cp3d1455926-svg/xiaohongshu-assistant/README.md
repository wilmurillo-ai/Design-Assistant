## 📖 功能说明

小红书文案生成助手！支持 6 种文案风格，**安全模式**运行，零封号风险。

### 核心功能

#### 🎨 文案创作
- ✍️ **文案生成** - 6 种风格（casual/professional/story/小红书/知乎/微博）
- 🎯 **标题优化** - 爆款标题生成（5 个备选）
- 🏷️ **话题推荐** - 智能话题标签推荐
- 😊 **emoji 推荐** - 根据内容自动推荐 emoji
- ⏰ **发布时段建议** - 根据内容类型推荐最佳发布时间

#### 🛡️ 安全说明 ⭐ 重要
- ⚠️ **自动发布已禁用** - 为避免封号风险，仅支持文案生成
- ✅ **人工发布** - 生成文案后手动复制粘贴发布
- ✅ **零封号风险** - 纯文案生成，不涉及自动化操作
- ✅ **内容可控** - 人工审核修改，保证质量

## 🚀 快速开始

### 使用方法

#### 1. 文案生成

```
# 生成文案
用小红书风格写一篇好物分享
用 professional 风格写一篇评测
用 story 风格写一个使用体验
```

#### 2. 人工发布（安全）⭐

**步骤**：

1. **生成文案**
   ```
   用小红书风格写一篇宣传文案
   ```

2. **复制文案**
   - 复制生成的标题
   - 复制生成的正文

3. **打开小红书**
   - 访问：https://creator.xiaohongshu.com/
   - 点击"发布笔记"

4. **粘贴发布**
   - 粘贴标题
   - 粘贴正文
   - 上传配图
   - 添加话题标签
   - 人工确认后发布

## ⚠️ 安全使用指南

### 为什么禁用自动发布？

**小红书平台规则**：
- ❌ 不允许自动化发布工具
- ❌ 不允许批量操作
- ❌ 不允许营销号行为

**封号风险**：
- 🔴 自动化发布 → 中等风险
- 🔴 频繁发布 → 高风险
- 🔴 多账号操作 → 高风险
- 🔴 AI 生成不审核 → 中等风险

### 安全使用建议

#### ✅ 推荐做法

1. **人工审核**
   - 审核 AI 生成的文案
   - 修改不自然的内容
   - 添加个人风格

2. **控制频率**
   - 每天最多 3 篇
   - 间隔至少 2 小时
   - 夜间不发布（23:00-8:00）

3. **内容质量**
   - 使用真实图片
   - 添加个人体验
   - 避免纯营销内容

4. **账号维护**
   - 完善账号资料
   - 正常互动（点赞/评论）
   - 避免只发布不互动

#### ❌ 避免操作

- 一天发布 10+ 篇
- 几分钟内连续发布
- 深夜频繁操作
- 纯 AI 生成不修改
- 明显营销广告

## 🎯 使用技巧

### 文案风格选择

| 风格 | 适用场景 | 示例 |
|------|----------|------|
| **casual** | 日常分享 | 好物推荐、生活碎片 |
| **professional** | 专业评测 | 产品测评、技术分析 |
| **story** | 情感故事 | 使用体验、心路历程 |
| **小红书** | 爆款文案 | 种草笔记、开箱测评 |
| **知乎** | 深度解答 | 问题分析、经验分享 |
| **微博** | 短小精悍 | 快讯、短消息 |

### 发布时段建议

| 内容类型 | 最佳时段 | 说明 |
|----------|----------|------|
| 职场干货 | 8:00-9:00 | 通勤时间 |
| 美食探店 | 11:00-13:00 | 午饭时间 |
| 生活分享 | 19:00-21:00 | 下班放松 |
| 学习成长 | 21:00-23:00 | 睡前学习 |

### 话题标签策略

**数量**：5-10 个为佳  
**组合**：
- 2-3 个大流量标签（#AI #效率工具）
- 3-5 个精准标签（#OpenClaw #技能分享）
- 1-2 个长尾标签（#打工人摸鱼神器）
