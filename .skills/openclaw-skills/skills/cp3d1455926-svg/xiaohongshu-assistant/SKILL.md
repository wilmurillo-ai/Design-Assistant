# 小红书运营助手

## 核心功能

### 1. 文案生成
- 根据产品/主题自动生成小红书文案
- 支持多种风格：种草、测评、教程、分享
- 自动添加 emoji 点缀

### 2. 标题优化
- 生成 10 个爆款标题供选择
- 包含数字、情绪、痛点等元素
- 符合小红书标题规范（20 字内）

### 3. 话题标签推荐
- 根据内容推荐 10-15 个热门标签
- 包含大流量词 + 精准长尾词
- 避免违规敏感词

### 4. 发布时间建议
- 根据目标用户群体推荐最佳发布时间
- 考虑工作日/周末差异
- 提供流量高峰时段参考

## 使用示例

### 生成文案
```
用户：帮我写一篇城市性格测试的小红书文案
助手：生成完整文案（包含标题、正文、标签）
```

### 优化标题
```
用户：优化这个标题"城市性格测试"
助手：生成 10 个优化版本
```

### 推荐标签
```
用户：心理测试类笔记用什么标签
助手：推荐 15 个相关标签
```

```
# [吸引眼球的标题]

[使用感受/效果描述]！！😭

[产品亮点 1]
[产品亮点 2]
[产品亮点 3]

适合 [目标人群]
[行动引导]

#标签 1 #标签 2 #标签 3
```

```
# [产品名] 真实测评！不吹不黑

[购买原因/使用时长]

✅ 优点：
- [优点 1]
- [优点 2]

❌ 缺点：
- [缺点 1]
- [缺点 2]

```

### 教程模板
```
# [教程主题]｜[数字] 步搞定！

[效果展示/前后对比]

步骤 1️⃣：[具体操作]
步骤 2️⃣：[具体操作]
步骤 3️⃣：[具体操作]

[注意事项/小贴士]

[行动引导]

#标签 1 #标签 2 #标签 3
```

## 标题公式

### 数字型
- "[数字] 个 [主题]，[效果]"
- "[数字] 分钟搞定 [主题]"
- "月入 [数字] 万的 [方法]"

### 情绪型
- "我直接 [情绪反应]！"
- "[情绪词]！[主题] 太 [形容词] 了"
- "后悔没早 [动作]！"

### 痛点型
- "[痛点问题]？试试这个"
- "别再 [错误做法] 了！"
- "[人群] 必看！[解决方案]"

### 悬念型
- "为什么 [现象]？真相是…"
- "[数字]% 的人都不知道的 [主题]"
- "[动作] 后，我 [结果]"

## 注意事项

### 违规词避免
- 最、第一、顶级等绝对化用语
- 医疗功效宣称
- 诱导私信/加微信
- 价格敏感词

### 限流避免
- 不要频繁修改已发布笔记
- 不要刷赞刷评论
- 不要搬运抄袭
- 不要过度营销
