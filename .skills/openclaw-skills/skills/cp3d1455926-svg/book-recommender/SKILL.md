# 📚 Book Recommender - 书籍推荐助手

## 触发规则

**关键词：**
- 推荐书籍、书单推荐、看什么书
- 豆瓣评分、书籍评分
- 读书笔记、读后感

**场景：**
- 用户想找书看但不知道选什么
- 用户查询某本书的评分
- 用户想记录读书笔记

## 功能描述

### 核心功能

1. **书籍推荐**
   - 根据兴趣推荐
   - 根据阅读历史推荐
   - 热门书单

2. **豆瓣对接**
   - 评分查询
   - 书评查看
   - 作者信息

3. **读书笔记**
   - 记录阅读进度
   - 摘录金句
   - 写读后感

4. **书单管理**
   - 想读清单
   - 已读清单
   - 在读清单

## 权限需求

- [x] 网络访问（豆瓣 API）
- [x] 文件读写（笔记存储）
