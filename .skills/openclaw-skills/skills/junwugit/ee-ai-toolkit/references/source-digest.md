3.1 What Is Prompt Engineering?

Prompt engineering is the process of designing clear and structured instructions to guide artificial intelligence systems to produce useful and accurate results.

Modern AI assistants such as ChatGPT respond based on the input prompt provided by the user. The quality of the prompt significantly affects the quality of the output.

In engineering applications, prompts should include:

context

constraints

technical parameters

expected output format

Example 1: Weak vs Strong Prompt

Weak prompt:

Explain transformers.

This prompt is vague and produces a general explanation.

Strong prompt:

Explain the working principle of a three-phase power transformer for electrical engineering students, including equations, losses, and practical applications.

This prompt provides:

audience

technical scope

expected depth

Therefore, the AI response will be more useful.

3.2 Structure of an Effective Engineering Prompt

An effective engineering prompt typically includes four components:

1 Context

Provide background information.

Example:

“Assume you are an electrical engineer analyzing a distribution system.”

2 Task

Define the required task clearly.

Example:

“Calculate the voltage drop in a feeder line.”

3 Constraints

Include technical parameters.

Example:

voltage level

power rating

conductor length

4 Output Format

Specify how the result should be presented.

Example:

step-by-step solution

table

code

Example 2: Structured Prompt

Prompt:

Calculate the power consumption of a three-phase motor rated at 400 V with a current of 15 A and power factor of 0.9. Show step-by-step calculations.

Solution:

Three-phase power formula:

P = √3 × V × I × PF

Substitute values:

P = 1.732 × 400 × 15 × 0.9

P ≈ 9353 W

Answer:

Power ≈ 9.35 kW

Example 3: Voltage Drop Calculation

Prompt:

Calculate voltage drop in a single-phase feeder supplying 20 A over a 50 m copper cable with resistance of 0.02 ohms per meter.

Solution:

R = 1 ohm

Voltage drop formula:

Vdrop = I × R

Vdrop = 20 × 1
Vdrop = 20 V

Answer:

Voltage drop = 20 volts

4.3 AI-Generated Python Script for Electrical Calculations

Using Python, engineers can automate calculations.

Example script:

```python
import math
V = float(input("Voltage (V): "))

```

This script calculates three-phase electrical power.

10.3 Example: Minimizing Power Loss

Resistance = 0.4 Ω

Ploss = I²R

Ploss = 3600 × 0.4
