## Topic Routing

- AI basics, benefits, limits, validation responsibility: read `source-digest.md` sections for Lecture 1 and Introducing AI.
- AI tools, Python/MATLAB/ChatGPT workflow, documentation, spreadsheets, CSV/Excel/JSON handling: read Essential AI Tools and Tools/Data Handling sections.
- Prompt engineering, prompt patterns, reusable prompt templates: read `prompt-library.md` first, then matching sections in `source-digest.md`.
- Electrical calculations and design: use Ohm law, voltage drop, power triangle, transformer, cable sizing, protection, power factor, grounding and battery sections; check matching Python scripts.
- Power systems and smart grids: load forecasting, fault detection, voltage profiles, losses, renewable penetration, smart meters, demand response and predictive maintenance.
- Design automation: automate reports, circuit/system data, ETAP/MATLAB prompts, single-line data, versioned prompt workflows.
- Data analysis and visualization: load curves, voltage trends, harmonics, anomalies, power quality dashboards, correlations and interpolation.
- Optimization and decision-making: economic dispatch, capacitor placement, battery dispatch, renewable mix, transformer balancing, voltage setpoint, multi-criteria decisions and maintenance scheduling.
- Career toolkit: resume keywords, skills gaps, interview questions, portfolio/certification/job trackers, salary comparisons and LinkedIn optimization.
- Full 100 script appendix: read `python-script-catalog.md`; exact extracted code is under `assets/python-scripts/`.

## Use Policy

- Treat examples as educational starting points, not certified engineering designs.
- Preserve units, assumptions, formulas, and validation steps in answers.
- For safety-critical electrical work, tell the user to validate with applicable standards, licensed engineering review, and field measurements.
