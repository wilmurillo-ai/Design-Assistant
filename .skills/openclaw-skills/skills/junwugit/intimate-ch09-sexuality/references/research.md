# Research

## 使用方式

- 用户问研究者、年份、经典研究或百分比时读本文件。
- 只使用本章材料中出现的信息。
- 没记录的细节不要补写。

## 性态度和文化

- Wells 与 Twenge（2005）：如今只有不到 25% 的人认为婚前性行为"总是或几乎总是错误的"。
- Sprecher 等人（2006）：提出以情感为标准的性认可态度。
- Petersen 与 Hyde（2010）：男性平均在性态度上更宽容，但差异随时代减小且取决于具体态度。
- Galperin 等人（2011）：女性更可能后悔发生一夜情，男性更可能后悔错过性机会。
- Crawford 与 Popp（2003）：性双重标准变得更微妙。
- Conley（2011）、Marks 与 Fraley（2005）、Smith 等人（2008）、Jonason 与 Marks（2009）：讨论对男女不同性行为的评价差异。
- Widmer 等人（1998）：美国对婚前性行为、婚外性行为和同性关系的态度比多个国家更保守。

## 同性恋态度

- Saad（2010）：43% 美国成年人认为成人同性性关系道德败坏，52% 认为男同和女同关系道德容许。
- Saad（2007）：图 9.1 显示对同性恋起源的看法与宽容度相关。
- Langstrom 等人（2010）：同性性行为部分基于基因。
- Herek 等人（2010）：绝大多数同性恋者认为自己对性取向别无选择。
- Jones（2010）：53% 美国人反对同性伴侣正式婚姻。
- Myers 与 Scanzoni（2005）：没有实证依据支持剥夺同性伴侣法律权益。
- Balsam 等人（2008）、Biblarz 与 Stacey（2010）、Riggle 等人（2010）：支持美国心理学会关于同性关系、养育与婚姻益处的立场。

## 第一次性行为和频率

- Finer（2007）：几乎所有人，约 95%，结婚前有过性行为。
- National Center for Health Statistics（2007）：第一次性交平均年龄约 17 岁；到 20 岁，15% 仍未有过性行为。
- Tanner（2008）：美国超过四分之一少女感染性传播疾病；18% 年轻女性感染 HPV。
- de Graaf 等人（2009）：多数青少年第一次性行为发生在稳定、情感重视的关系中。
- National Center for Health Statistics（2004）：第一次性伙伴中 21% 只是相识的人或一般朋友。
- Higgins 等人（2010）、Metts（2004）、Mercer 等人（2006）、Oswalt 等人（2005）：第一次性行为结果受爱意承诺、年龄差、避孕套和情境影响。
- Willetts 等人（2004）：同居情侣平均每周约 3 次性行为，已婚夫妻约 2 次。
- Herbenick 等人（2010b）：2009 年，25-30 岁美国青年 86% 去年有性行为；50 多岁男性 58%、女性 51%；70 多岁男性 43%、女性 22% 有过阴道性交。
- Blumstein 与 Schwartz（1983）：图 9.2 比较不同关系类型和持续时间中的性频率。

## 不贞和社会性性行为取向

- Tafoya 与 Spitzberg（2007）：总结 47 项调查，超过 58,000 名参与者；21% 女性和 32% 男性至少一次性出轨。
- Blow 与 Hartnett（2005）：男性更可能为性新奇而追求伴侣外性行为，女性更可能寻找情感联系。
- Peplau 等人（2004）、Hoff 与 Beougher（2010）、Peplau（2003）：讨论男同伴侣中的伴侣外性行为模式。
- Simpson 与 Gangestad（1991）：测量社会性性行为取向。
- Simpson 等人（2004）：开放取向者更喜欢调情并寻找新性伴侣。
- Ostovich 与 Sabini（2004）：开放取向者性伴侣更多，也更可能欺骗伴侣。
- Seal 等人（1994）：当前恋爱的开放取向大学生中 36% 输入约会对象图片，保守取向者为 4%。
- Pillsworth 与 Haselton（2006）：优质基因假说。
- Anderson（2006）：67 项父子关系研究元分析显示，全世界平均 2% 的"父亲"不知道孩子亲生父亲另有其人。
- Blickstein（2005）：美国每 400 对异卵双生子中约 1 对有不同生父。

## 性欲望和安全性行为

- Regan 与 Atkins（2006）：年轻男性每周 37 次性欲望经历，女性 9 次；男性每周约 60 次想到性行为，女性 15 次。
- Klusmann（2002）：有固定性伴侣的男性近一半每周手淫一次以上，女性为 16%。
- Gerressu 等人（2008）：英格兰 16-44 岁男性 73% 上个月手淫过，女性 37%。
- Pitts 等人（2004）：澳大利亚 23% 男性至少一次为性行为付费。
- Paul 等人（2008）：约四分之三大学生有过 hookup；若发生性交，大约一半情形使用避孕套。
- Grello 等人（2006）：37% hookup 双方不认识或不太了解。
- Padgett（2007）：740 名寻找男友的线上约会女性中，30% 第一次见面发生性行为；第一次性行为时 77% 不使用避孕套。
- Linville 等人（1993）：单次无保护性接触 HIV 风险低，但重复暴露会提高累积风险。
- Wildsmith 等人（2010）：检测阳性的年轻成人中，22% 注意到近一年症状，28% 认为自己可能感染。
- Oncale 与 King（2001）：30% 男性和 41% 女性的性伙伴试图说服他们不要使用避孕套。

## 性满足、沟通和胁迫

- Lutfey 等人（2009）：波士顿大型研究中，条件具备时只有 6% 妇女整体上不满意性生活；51% 近一个月有性行为；性活跃者 39% 遇到问题。
- Crary（2010）：美国 45 岁以上者中 43% 目前有满意性生活。
- Blumstein 与 Schwartz（1983）：每周 3 次或更多性行为的夫妻中 89% 性满足；每月一次者为 32%。
- Deci 与 Ryan（2000）和 Smith（2007）：自我决定理论可用于理解性满足。
- Impett 等人（2005, 2008）：接近动机与更好性结果有关，回避动机与消极结果有关。
- Masters 与 Johnson（1970）：比较异性恋者、男同和女同的性经历，强调良好性沟通。
- Abbey（1982）：经典研究显示男性可能把女性友善行为误读为性兴趣。
- Jacques-Tiura 等人（2007）：54% 男性至少一次误解女性性意图。
- Holmberg 等人（2010）、Yucel 与 Gassanov（2010）：性满足与关系满意度相互关联。
- DeGue 与 DiLillo（2005）：图 9.4 的性侵犯分类。
- Gidycz 等人（2008）：6 位女大学生中每 2 个月就有 1 位遭遇某种形式的性胁迫。
- Crown 与 Roberts（2007）：56% 女性在大学生活中受到这类骚扰伤害。
