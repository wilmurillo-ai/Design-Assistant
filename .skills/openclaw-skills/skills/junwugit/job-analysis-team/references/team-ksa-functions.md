## Team KSA approach

当工作由小团队完成、岗位灵活且动态时，单靠具体任务推导个人 KSA 可能不够。Stevens 和 Campion 提出一组跨团队情境有用的 teamwork KSAs，并开发了用于测量这些 KSAs 的情境判断测验。

两大类：

- **Interpersonal KSAs**：冲突解决、协作解决问题、沟通。
- **Self-management KSAs**：目标设定与绩效管理、计划与任务协调。

## 14 个 teamwork KSAs

### I. Interpersonal KSAs

**A. Conflict Resolution KSAs**

1. 识别并鼓励有益冲突，同时抑制有害冲突。
2. 识别团队冲突的类型和来源，并选择合适的解决策略。
3. 使用整合式、双赢谈判，而不是传统的赢输式分配策略。

**B. Collaborative Problem Solving KSAs**

4. 识别需要参与式团队问题解决的情境，并使用合适的参与程度和类型。
5. 识别协作式问题解决的障碍，并采取合适纠正行动。

**C. Communication KSAs**

6. 理解沟通网络，并在可能时使用去中心化网络提升沟通。
7. 公开、支持性地沟通，使信息具体、前后一致、能确认他人、能接上对话并承担个人表达责任。
8. 非评价性倾听，并正确使用主动倾听技术。
9. 让非语言和语言信息保持一致，并识别、解释他人的非语言信息。
10. 理解寒暄、仪式化问候和闲谈在团队关系中的作用。

### II. Self-Management KSAs

**D. Goal Setting and Performance Management KSAs**

11. 帮助建立具体、有挑战性且被团队接受的目标。
12. 监控、评价并反馈团队整体绩效和个人成员绩效。

**E. Planning and Task Coordination KSAs**

13. 协调并同步成员之间的活动、信息和任务互依。
14. 帮助建立个人成员的任务和角色期望，并确保团队工作量平衡。

## Teamwork test 的证据与限制

Stevens 和 Campion 的 teamwork test 使用现实团队情境和多个行动选项，让被试选最有效做法。课本例子是团队争论谁做讨厌但例行的任务，最佳处理是轮换安排，让负担公平分摊。

证据与限制：

- 有研究支持它能帮助识别更好的团队成员。
- 但分数与一般认知能力测验高度相关，可能很大程度上是在测"聪明"。
- 它没有直接测 agreeableness 等人格特质。
- 团队绩效还受 personality traits 和 social skills 影响；team KSAs 只是团队成功的一部分。

## Team functions 五大类

Nieva、Fleishman 和 Reick 提出团队普遍需要完成的五类功能：

1. **Orientation functions**：让成员知道目标、资源、环境特征、任务顺序和约束。
2. **Resource distribution functions**：把人分配到任务，使工作量平衡，并让能力与任务要求匹配。
3. **Timing functions**：安排团队和个人活动的节奏、顺序和时间模式。
4. **Coordination functions**：让成员行动在正确模式中衔接起来。
5. **Motivational functions**：管理努力水平、冲突、绩效规范和团队奖励。

这些 function scales 曾被用于区分不同军事团队的需求 profile，但在企业团队中的应用没有那么广。它们的价值在于提供团队层面的 descriptor：团队必须完成什么功能。

## KSA / Function / Design 的分工

| 问题 | 更适合的框架 |
|---|---|
| 团队成员需要会什么？ | Team KSAs |
| 团队作为整体必须做什么？ | Team functions |
| 团队结构和过程条件是否支持有效性？ | Team design characteristics |
| 要把训练内容落到任务和 KSAO？ | MAP system |

## 对话使用提示

用户问"团队成员要具备什么能力"时，先给两大类：interpersonal + self-management。再根据场景挑 2-3 个最相关 KSA。

用户问"团队为什么失败"时，不要只讲成员能力。追问失败发生在：

- orientation：目标/资源/环境判断不清？
- resource distribution：人和任务不匹配？
- timing：节奏和先后顺序错？
- coordination：行动衔接失败？
- motivation：努力、规范、冲突或奖励出问题？
