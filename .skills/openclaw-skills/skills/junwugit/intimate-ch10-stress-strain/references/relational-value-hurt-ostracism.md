# Relational Value Hurt And Ostracism

## 使用方式

- 用户问关系价值、感情创伤、排斥或沉默对待时读本文件。
- 若需要具体实验数据，交叉读取 `research.md`。
- 回答时把"关系价值下降"作为主线。

## 关系价值

- 关系价值（relational value）指他人认为与我们的关系珍贵、重要或亲近的程度。
- 亲密关系越重要，我们越在意伴侣是否重视这段关系。
- 当感知到的关系价值低于期待，痛苦就会出现。
- 这种痛苦不只来自被拒绝，还来自"没有被重视到自己期待的程度"。

## 接纳和拒绝

- Leary 把接纳和拒绝看成连续体。
- 最大接纳是别人主动寻求我们的陪伴。
- 被动接纳是别人不主动邀请，但允许我们加入。
- 矛盾情感是别人不在乎我们是否出现。
- 被动拒绝是别人希望我们在别处。
- 主动拒绝是别人故意避开我们。
- 最大拒绝是别人明确要求我们离开。

## 感情创伤

- 感情创伤（hurt feelings）是关系价值下降时的独特体验。
- 它可伴随悲哀、愤怒和羞辱。
- 本章强调它与身体疼痛有相似反应系统。
- 依恋焦虑高、低自尊的人更容易受伤。
- 回避亲密高的人在被回避时可能体验到较小痛苦。

## 关系贬值实验

- Buckley 等人的实验通过虚假的连续评价制造接纳或拒绝。
- 稳定接纳让人更快乐、自我感觉更好。
- 稳定拒绝令人痛苦。
- 从接纳变成拒绝尤其痛苦，因为它显示关系价值正在下降。
- 这类下降甚至比稳定拒绝引发更多消极行为反应。

## 伴侣排斥

- 排斥（ostracism）指受到身边人的忽视和冷眼对待。
- 在亲密关系中，常见形式是沉默对待。
- 排斥者可能认为这是惩罚、避免对抗或冷静处理。
- 被排斥者通常不知道原因，只感到被拒绝。
- 他们体验到的更多是愤怒、沮丧和伤害，而不是内疚。

## 排斥威胁的需要

- 排斥威胁归属需要。
- 排斥会降低自我价值感。
- 排斥会削弱控制感。
- 排斥会让人感到社会存在被否定。
- 初始反应可能是情感迟钝、糟糕决策和心理倦怠。

## 排斥后的行为反应

- 如果归属需要受威胁，人们可能努力重新获得伴侣尊重。
- 如果关系可恢复，人们可能变得顺从、配合。
- 他们也可能寻找新的、更友好的伙伴。
- 如果排斥被体验为不公平，人们可能愤怒、敌对甚至攻击。
- 自尊高的人更可能离开排斥者；自尊低的人更可能留下但心怀怨恨。

## 答题提醒

- 不要把沉默对待自动解释成冷静。
- 不要只看客观拒绝强度，还要看关系期待和意义。
- 讲感情创伤时要回到关系价值，而不是泛泛说"情绪不好"。
