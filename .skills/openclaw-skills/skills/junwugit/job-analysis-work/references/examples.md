# 课本原型案例（第 2 章）

用于教学锚定——尽量调用这些原书案例而不是编造。每例标出它示范的是哪种方法 / 概念。

## Time-and-Motion Study 类

### 铺砖工（Bricklayer / Frank Gilbreth）
- 砌砖的一套动作："把灰浆从灰板抹到已砌好的砖面上，用瓦刀；然后将砖放到灰浆上，轻敲定位，再用瓦刀刮掉多余灰浆。"
- 用于讲 **motion study**——改进可能包括：减少每块砖的瓦刀击数、缩短砖的搬运距离；单块砖省一点、整墙省很多。

### 钢铁公司的铲子（Taylor）
- Taylor 重新设计铲子使每铲都恰好 ~21.5 磅，不论是灰、煤还是铁矿；年节省约 $78,000（以 1990 年前后美元计算，折合如今 ≈ $2 百万）。
- 用于讲 **scientific management**、Taylor 的效率观、以及劳资共享收益承诺为何常落空。

### 医院护工送餐（Orderly / Figure 2.1–2.2）
- 原方法：把餐车推到房间中央，逐一送到 17 张床。总行走 436 m、23.42 分钟。
- 改良方法：餐车随行，每次拿两盘送两床。总行走 197 m、16.98 分钟。
- 用于讲 **flowcharts**、**motion study**；经典图示是"扇形路径图 + flow process chart"。

### 银行 Teller（Table 2.1）
- 一位观察员 Murphy 在若干时点（8:29、8:57、10:15、12:40、1:50、3:10）记录每个 teller 在 8 类活动中的哪一类：1 支票兑现 / 2 存款 / 3 解答 / 4 汇票 / 5 计算 / 6 动作转换 / 7 结账 / 8 闲置。
- 用于讲 **work sampling**。

### 钻孔时间 & 拉链缝合时间
- 一次性"钻一个孔"、"缝一条拉链"要多少秒——作 **standard time** 示例。
- 用于讲 stopwatch time study。

### 木工车间的"industry standard data"
- 多年积累让大量特定机器加工任务的标准时已知；新 job 可由已知 task 的标准时合成。
- 用于讲 **industry standard data**。

### "把交响乐剪到 20 分钟"（Anonymous, 1955）
- 工程师吐槽经典段落："号器重复太多可剪；2 小时可压到 20 分；有的演奏者一手闲着本可再添一活。"
- 用于讲 **criticism of time-and-motion study**。

### 丰田生产方式（Parker, Morgeson, & Johns, 2017）
- 将 T&M 的方法融入现代制造系统。
- 用于反驳"T&M 方法已过时"的错觉。

### FJA 句：驯兽师（Table 2.7）
已写全："Trains wild animals to entertain audience."
- Verb = Trains
- Immediate Object = wild animals
- Infinitive (work field) = to entertain
- Object of infinitive (MPSMS) = audience

### FJA 句：正片检验员（Table 2.7）
3 条任务句覆盖 Data（Compares）、People（Signals）、Things（Handles）三个功能。

### Painting work field #262（Table 2.6）
- Methods verbs：Blanking out / Coloring / Drawing / Inking / Lettering / Rubbing / Shading / Sketching / Spotting out / Spraying / Tinting / Touching up / Wiping。
- MTEWA：机器=Lettering machine, Spray machine；工具=Air brush, Burnishing tools, Charcoal, Handtools, Knives, Pencils, Pens, Spray guns；设备=Camera, Projectors；work aids=Canvas, Charts, Diagrams, India ink, Letters, Magnifying glasses, Oil paints, Plastics, Reducing glasses, Tempera, Templates, Watercolors。

### Welding 模糊句（Fine, 1988）
- "Welds steel pipes." — 不够具体。用于讲"目标 vs worker 行为"的区分。

### 填补职位空缺的 HR（Fine, 1988, p. 1027）
- 原：Determines qualifications that employees must possess…
- 改：Reviews/analyzes job description data (e.g., training manuals…) drawing on experience and psychological background in order to determine employee qualifications…
- 用于讲 **what gets done vs what the worker does** 的改写示范。

### 社工劝客户穿正装面试（Levine, 1983）
- data/people/things 占比 ≈ 35%/60%/5%。
- 用于讲 Fine's **orientation**（百分比）概念。

### 高中咨询师
- 课本 FJA 示例：与 data / people 显著相关，与 things 不相关；其 data/people/things 近似层级 = analyzing / persuading / handling。

## Task Inventory 类

### 电话线安装主管（Table 2.8）
- 7 点 difficulty 量表；任务例：
  - "Assign personnel to installation or repair projects."
  - "Brief personnel on unit security or safety rules."
  - "Complete manhour accounting forms for work crews."
  - "Conduct supervisory orientation of newly assigned personnel."

### 药房 Drug Clerk（Table 2.9，Customer Service 职责节选）
- 三列：X if Done / Time Spent / Difficulty to Learn。
- 任务样例：
  - Answer customer questions about products and services.
  - Call patient about scrip not picked up after 7 days.
  - Make refunds.
  - Recommend products to customers.
  - Refer medical questions to pharmacist.
  - Ring up merchandise and prescriptions on register.

### 电话杆维护主管（Table 2.11，WPSS 输出）
- N = 113（全公司），子样本 C&P N=5、Illinois N=6。
- 任务例 "Analyze office costs/expenses for budgeting purposes."：PROP 0.97、MEAN 3.67、SD 1.71。
- Illinois 样本在同任务上 MEAN 偏低（2.80）——演示**按子群对比**的分析。
- 用于讲 **data analysis** 输出形式。

## CIT 类

### 警察灭车火（Table 2.12 well-written 版）
- Context：加油站加油的车突然引擎起火。
- Behavior：巡警开过，注意到火，停车，取出巡逻车上的化学灭火器迅速扑灭。
- Consequence：避免大火与爆炸。

### 文员纠错（Table 2.12 well-written 版）
- Context：某封信里有一项看着不对的内容。
- Behavior：文员注意到，核查，发现与原意相反，改正。
- Consequence：（可推断：避免传达错误信息。）

### 系主任忘排老师（Table 2.12 well-written 版）
- Context：新学期第一次课。
- Behavior：系主任没分配授课教师。
- Consequence：课堂上无老师，学生无法确认是否在班。

### 加错水的混凝土工（章末 tips 示例）
- Context：铺人行道浇混凝土。
- Behavior：工人在搅拌时多加了水。
- Consequence：混凝土未能正确凝固。

### 外科医生分心（章末 tips 示例）
- 讨论如何把 SME 的推断（"外科医生分心"）拉回到行为（"医生做了什么让你这样认为？"）。

### "showed good initiative"
- 典型的**不可接受**写法。CIT 的对立面示例。

## 跨方法出现的背景场景

- **Professor 的 work sampling**：讲课、出题、批改、分析数据、写报告、在桌前打盹（课本玩笑）。
- **Clock industry / 木加工 industry**：industry standard time data 的常见发源地。
- **拨电话 → 接纠纷报警 → 平息冲突 → vice activities**：element → activity → task → duty 示范（此例来自 Ch.1，但第 2 章写任务时可回扣）。
- **Dear friend's bakery rolls**：讨论 elemental motions 的意义——说明为何自己未必能复制（来自 Ch.1 例子，但在 Ch.2 也可做 micromotion 参考）。
