# 核心术语速查

## 法律与执法术语

| 术语 | 学习解释 |
|---|---|
| **EEO** (Equal Employment Opportunity) | 平等就业机会。核心思想是就业机会应基于与工作有关的资格和表现，而不是受保护类别身份。 |
| **EEOC** (Equal Employment Opportunity Commission) | Civil Rights Act 1964 创建的联邦机构，负责执行多项 EEO 法律，并使用 Uniform Guidelines 判断选拔程序问题。 |
| **OFCCP** (Office of Federal Contract Compliance Programs) | 监管联邦承包商的机构，特别涉及政府合同、非歧视和 affirmative action 要求。 |
| **Protected group / protected class** | 受法律保护的群体类别。Title VII 涵盖 race、color、sex、religion、national origin；其他法律还保护年龄、残障等。 |
| **Reverse discrimination** | 多数群体成员提出自己遭到非法歧视的主张。课本把第十四修正案与这类主张联系起来。 |
| **Affirmative action** | 积极行动。课本在 Executive Order 11246 下讨论，要求某些联邦承包商采取目标和时间表，使劳动力结构接近劳动力市场构成。 |

## 歧视与抗辩术语

| 术语 | 学习解释 |
|---|---|
| **Disparate treatment** | 差别待遇。雇主因 race、sex 等受保护因素故意区别对待个人。 |
| **Adverse impact / disparate impact** | 不利影响。表面中立的做法排除了更高比例的某个受保护群体成员。课本强调结果本身就可能触发法律问题。 |
| **Job relatedness** | 与工作相关。选拔要求、测验或程序必须能够连接到具体岗位所需的任务、KSAO 或绩效。 |
| **Business necessity** | 业务必要性。在 Title VII 相关 adverse impact 讨论中，雇主需要证明做法与业务和岗位需求有必要联系。 |
| **RFOA** (Reasonable Factors Other Than Age) | "年龄以外的合理因素"。课本在 ADEA disparate impact 案件中介绍，认为比 business necessity 更容易证明。 |
| **BFOQ** (Bona Fide Occupational Qualification) | 真正职业资格。第 6 章不是重点，但若用户提到，可解释为极窄情况下允许基于某些受保护特征作职业资格限制。 |

## 测验与效度术语

| 术语 | 学习解释 |
|---|---|
| **Selection procedure** | 用于招聘、晋升或筛选的程序，包括申请表、面试、纸笔测试、背景或推荐检查、工作样本、模拟等。 |
| **Validation** | 收集证据，判断某测验或程序是否适合用于某个雇佣决策。课本强调 validity 是一个由证据和理论支持的论证，不是测验本身的固定属性。 |
| **Criterion** | 标准或效标。工业/组织心理学中指希望预测或解释的工作绩效指标，如主管评分、产量、投诉数等。 |
| **Criterion-oriented evidence** | 把测验分数和绩效标准联系起来的效度证据。 |
| **Content-oriented evidence** | 证明测验内容代表工作重要内容的效度证据，如打字工作样本代表岗位中的打字任务。 |
| **Internal structure evidence** | 检查测验题项、组成部分、量表与构念之间关系的证据。它有帮助，但不能单独证明测验能预测工作绩效。 |
| **Validity generalization** | 汇总多个效度研究，估计测验与某类效标之间更稳定的总体关系。 |
| **Construct** | 测验意图测量的心理特质或能力，如批判性思维、坚持性、机械理解等。 |
| **KSAO** | Knowledge、Skills、Abilities、Other characteristics。工作分析常用来识别岗位所需知识、技能、能力和其他特征。 |

## ADA 术语

| 术语 | 学习解释 |
|---|---|
| **Disability** | 残障。课本按 ADA/ADAAA 介绍，指限制主要生活活动的身体或心理损伤，也包括有损伤史或被视为有损伤的人。 |
| **Qualified individual with a disability** | 有残障但合格的个体。关键是此人能在有或没有 reasonable accommodation 的情况下完成 essential functions。 |
| **Essential functions** | 岗位的主要或最重要功能。ADA 应用中，工作分析要帮助区分 essential 与非 essential。 |
| **Reasonable accommodation** | 合理住宿。包括调整工作环境、提供辅助人员、调整日程、改造测试和培训材料等，前提是不造成 undue hardship。 |
| **Undue hardship** | 过度困难。住宿是否合理的限制概念，课本提醒法律故意保持一定模糊性。 |

## 易混概念

| 对比 | 区分 |
|---|---|
| **Equal Pay vs comparable worth** | Equal Pay Act 要求相同或实质相同工作的男女同酬；课本说明它不覆盖"不同但价值相当的工作"的 comparable worth。 |
| **Fairness vs selection bias** | Standards 把 fairness 视为社会/伦理判断；selection bias 是技术问题，关心同分群体是否对应不同绩效水平。 |
| **Validity evidence vs valid test** | 严格说，validity 是对某用途的论证，不是测验一成不变的属性。 |
| **Essential function vs process detail** | ADA 下应描述岗位目标或功能，避免把某种执行方式写死，除非该方式本身确实必要。 |
