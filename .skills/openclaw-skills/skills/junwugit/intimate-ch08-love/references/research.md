# Research

## 使用方式

- 用户问"谁做的研究"、"哪一年"、"有什么数据"时读本文件。
- 只使用本章材料中出现的研究者、年份和数据。
- 不确定的细节不要补写。

## 爱情与婚姻观念

- Kephart（1967）：在 1967 年，76% 的美国女性和 35% 的美国男性表示愿意与"各方面都合适但自己并不爱"的人结婚。
- Simpson 等人（1986）：到 20 世纪后期，多数美国年轻人把浪漫爱情看成结婚的必要条件。
- Hunt（1959）：早期材料展示不同文化对爱情和婚姻的不同态度。
- Hatfield 与 Rapson（2008）：讨论北美把爱情作为婚姻基础的观念及其历史文化差异。
- Buunk 等人（2010）：指出在许多地方，因爱情结婚仍可能被视为荒唐。

## Sternberg 和三角理论

- Sternberg（1987, 2006）：提出爱情三角理论。
- Aron 与 Westbay（1996）：支持亲密、激情和承诺都是爱情的重要组成。
- Diamond（2004）：指出性欲涉及的脑区不同于依附和承诺相关脑区。
- Whitley（1993）：报告亲密、激情和承诺之间存在相关。
- Fehr（2006）：提醒三角理论并未完整覆盖所有爱情形式。

## 生理系统和脑成像

- Fisher（2006）：提出爱情相关的性欲、吸引和依附系统。
- Aron 等人（2005）：用脑成像研究浪漫爱情相关活动。
- Xu 等人（2011）：在中国和美国样本中研究浪漫伴侣照片引发的脑活动。
- Aron 等人（2008）：讨论多巴胺、血清素与浪漫吸引。
- Younger 等人（2010）：发现观看恋人照片能降低实验性热痛体验。

## 浪漫激情与唤醒

- Berscheid 与 Walster（1974）：提出激情吸引的二因素解释。
- Dutton 与 Aron（1974）：温哥华悬桥研究，19-35 岁男性在危险桥和安全桥后对女性访问者反应不同。
- White、Fishbein 与 Rutstein（1981）：运动唤醒会强化男性对女性录像对象的反应。
- 同一研究线索还使用负性、正性和中性唤醒材料，说明唤醒会放大评价。
- Myers 与 Berscheid（1997）以及 Regan（2008）：强调性吸引在浪漫爱情中的重要性。

## 量表和认知

- Hatfield 与 Sprecher（1986）：建构 Passionate Love Scale。
- Rubin（1973）：建构 Love Scale 和 Liking Scale。
- Goodwin 等人（2002）：约会期待会影响男性对女性工作表现的评价。
- Gonzaga 等人（2008）：思考伴侣之爱有助于抑制对有吸引力替代对象的想法。
- Maner 等人（2008）：回忆浪漫之爱会减少对有吸引力替代对象的注意。
- Aron 与 Aron（2006）：自我延伸模型。
- Aron 等人（1995）：跟踪年轻恋人 10 周，观察自我概念扩展和自尊变化。

## 相伴之爱与同情之爱

- Grote 与 Frieze（1994）：描述相伴之爱并发展 Friendship-Based Love Scale。
- Lauer 与 Lauer（1985）：研究至少结婚 15 年的伴侣，发现"配偶是最好的朋友"和"喜欢配偶这个人"是常见维持理由。
- Hendrick 与 Hendrick（1993）：44% 的未婚年轻人说浪漫伴侣也是最亲近的朋友。
- MacDonald 与 MacDonald（2010）：讨论催产素与相伴之爱。
- Fehr 与 Sprecher（2009b）：界定同情之爱。
- Sprecher 与 Fehr（2005）：发展 Compassionate Love Scale。
- Neff 与 Karney（2009）：同情之爱建立在对伴侣优缺点较准确的理解上。

## 文化、依恋和性别

- Hatfield 等人（2007）：浪漫爱情具有跨文化普遍性。
- Shaver 等人（2001）：东西方都能区分浪漫之爱和相伴之爱。
- Levine 等人（1995）：个人主义文化中的人更可能坚持爱情是婚姻理由。
- Zhang 与 Kline（2009）：中国大学生比美国大学生更可能听从父母择偶意愿。
- Mikulincer 与 Shaver（2007）：安全型依恋与更好的亲密、性体验和承诺有关。
- Levenson 等人（1994）：比较 60 岁和 40 岁左右夫妻的爱情体验。
- Sprecher 与 Metts（1989）：男性更常表现出浪漫态度。
- Hatfield 与 Sprecher（1986）：男性更相信一见钟情，也更快坠入爱河。

## 持久性和变化

- Sprecher 与 Regan（1998）：浪漫之爱婚后会下降，相伴之爱更稳定。
- Tucker 与 Aron（1993）：Passionate Love Scale 分数随时间下降。
- Gupta 与 Singh（1982）：印度包办婚姻与恋爱婚姻的爱情变化比较。
- Huston 与 Chorost（1994）：婚后两年，表达性亲昵约为新婚水平的一半。
- Fisher（1995）：全球范围内，婚后第四年是常见离婚高峰期。
- Dewsbury（1981）：Coolidge effect 研究用于说明新奇与性兴趣。
- Strong 与 Aron（2006）：新奇、兴奋的共同活动会增强情侣的爱意。
- Acevedo 等人（2011）：婚后 10 年仍深深爱着配偶的人，看到配偶照片时出现奖赏系统活动。
