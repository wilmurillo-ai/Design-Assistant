# Research

## 使用方式

- 本文件用于查第 6 章出现的研究者、年份和课本数据。
- 只列课本中明确出现的信息，不补充外部细节。
- 回答时按用户问题选择少量研究，不要整表倒出。
- 具体百分比和数量只使用本文件列出的课本数据。

## 相互依赖和社会交换

- Thibaut 与 Kelley（1959）：提出本章重点采用的社会交换观点。
- Kelley 与 Thibaut（1978）：发展后来被称为相互依赖理论的框架。
- Rusbult 等人（2001）：支持相互依赖理论和投入模型的核心综述来源。
- Sedikides 等人（1994）：讨论亲密关系中代价的心理负担。
- Edwards 等人（2011）：用于说明痛苦关系中是否离开与替代选择有关。
- Ellis 等人（2002）：用于说明当前关系优于替代选择时，人们更依赖伴侣。
- Rusbult 等人（1994）：投入是决定关系是否维持的重要因素。
- Goodfriend 与 Agnew（2008）：投入包括有形资源和无形心理收益。
- Swann 与 Bosson（2010）：自尊影响人们对替代伴侣前景的判断。
- Ciarrochi 与 Heaven（2008）：习得性无助影响对替代关系前景的估计。
- Rusbult 与 Martz（1995）：信息获取和受虐妻子是否逃离丈夫都用于说明 CLalt 与投入。
- Miller（2008）：关注替代恋人的大学生更频繁更换恋人。
- Albrecht 与 Kunz（1980）：离婚发展中 CLalt 最终超过当前结果。

## 期望和满意度变化

- Soons 等人（2009）：追踪荷兰 5,500 多名年轻人 18 年；约会、同居、结婚带来幸福上升，但 14 年后不比遇到伴侣前更幸福。
- Lucas（2007）：德国研究追踪 3 万多人 18 年；婚姻带来的快乐大多在两年后减少。
- Amato 等人（2007）：讨论期望攀升和婚姻质量变化。
- Glenn（1998）：课本用来说明美国婚姻平均不如 30 年前幸福的可能原因。
- South 等人（2001）：女性参与劳动大军会影响离开不幸福关系的能力。
- Putnam（2000）：社会流动性增加影响替代选择。
- Berscheid 与 Lopes（1997）：离婚障碍降低。
- Rodgers 等人（1999）：无过错离婚立法与离婚变化。
- Cherlin（2009）：描述“永远可得到”的婚姻市场状态。

## 奖赏代价和负面事件

- Le 等人（2010）：奖赏、交换和投入模型均得到支持。
- Berg 与 McQuinn（1986）：恋爱开始时的慷慨、关爱和自我表露可预测 4 个月后是否仍在一起。
- Averill（1982）：任何一天，44% 的人可能被爱人或好友惹恼。
- Perlman（1989）：恋爱大学生每周平均 8.7 次恼人的吵闹。
- McGonagle 等人（1992）：婚姻配偶每月一般一两件不愉快争执。
- Vincent 等人（1975）：人们与配偶解决问题时比与陌生人更无礼。
- Caughlin 等人（2000）：情绪化行为会给婚姻带来危险。
- Renshaw 等人（2010）：公开敌意更有害。
- Amato 与 Rogers（1997）：离婚者会列举反复发生的痛苦和愤怒事件。
- Kahneman 与 Tversky（1982）：得失不对称例子。
- Baumeister 等人（2001）：坏的比好的更有力量。
- Gottman 与 Levenson（1992）：5:1 正负互动比例；高风险组 4 年后 56% 分居或离婚，低风险组 24% 分手。
- Howard 与 Dawes（1976）：93% 已婚夫妻报告做爱次数远高于争吵次数，痛苦夫妻没有一对如此。
- Gable 等人（2003）：四周追踪中，伴侣注意到对方正面行为的比例不到四分之一。
- Carmichael 等人（2003）：疏离或恐惧型依恋者更可能漏看伴侣积极行为。

## 动机、沉闷和发展

- Gable 等人（2008）：接近动机和回避动机在关系中分别运作。
- Cooper 等人（2008）：性爱中的接近和回避动机例子。
- Cacioppo 与 Gardner（1999）：愉悦和痛苦可独立运作。
- Reis 与 Gable（2003）：接近/回避二维框架。
- Fincham 与 Beach（2010）：丰盛关系和接近/回避过程。
- Gable 与 Poore（2008）：高激情同时高风险的关系感受会波动。
- Harasymchuk 与 Fehr（2011）：关系沉闷的定义和特征。
- Tsapelas 等人（2009）：结婚数年后感到单调乏味者，9 年后更不幸福。
- Gable（2006）：接近和回避动机存在个体差异。
- Strachman 与 Gable（2006）：动机影响人们注意伴侣积极或烦心行为。
- Impett 等人（2005）：牺牲的动机影响关系结果。
- Impett 等人（2010b）：关注奖赏比关注伤害更有长期益处。
- Aron 与 Aron（2000）：自我延伸模型。
- Aron 等人（2000）：新奇活动和关系快乐。
- Aune 与 Wong（2002）、Graham（2008）、Lee（2008）：幽默、娱乐和共同笑声用于对抗沉闷。
- Knobloch 与 Solomon（2004）：关系紊乱模型。
- Knobloch 与 Theiss（2010）：不确定性与关系混乱。
- Solomon 等人（2010）：随意约会到严肃投入的过渡。
- Kurdek（1999）：538 对新婚夫妻，大多追踪 10 年，婚姻满意度平均下降。
- Lavner 与 Bradbury（2010）：结婚 4 年后的满意度轨迹。
- Anderson 等人（2010）：结婚 20 年后的满意度轨迹；约六分之一夫妻急剧下降。
- Roberts（2007）：多数婚姻不会持续 25 年。
- McNulty 与 Karney（2004）：82 对新婚夫妻追踪 4 年；现实期望者更幸福，4 年后离婚者超过五分之一。

## 公平、共有和承诺

- Davis 等人（2007）：职业女性往往承担丈夫两倍的家庭杂务。
- Arriaga 与 Agnew（2001）：承诺量表。
- Le 与 Agnew（2003）：投入模型的元分析支持。
- Kurdek（2008）：投入模型适用于异性恋和同性恋伴侣。
- Johnson（1999）：个人承诺、强迫承诺、道德承诺。
- Rusbult 等人（1998）：承诺促进顺应性行为。
