# Equity And Communal Relationships

## 使用方式

- 本文件用于回答交换关系、共有关系、公平、家务分配和获益不足问题。
- 先问用户是在讨论“关系规范”还是“贡献和结果是否成比例”。
- 不把公平简化为“每件事都一人一半”。
- 需要研究年份和数据时可补读 `research.md`。

## 为什么亲密关系不只是贪婪

- 相互依赖理论承认人们追求高奖赏低代价。
- 但亲密关系中，伴侣的幸福会影响自己的长期结果。
- 如果让伴侣满意能维持珍贵关系，慷慨就符合自身利益。
- 因此，交换逻辑也能解释牺牲、体贴和宽容。
- 关键不是“人性是否自私”，而是相互依赖让双方利益交织。

## 交换关系

- 交换关系（exchange relationship）由外在公平交换规范调节。
- Clark 与 Mills（1993）区分了交换关系和共有关系。
- 交换关系中，人们付出时期待同等回报。
- 接受帮助会让人觉得有义务回报。
- 双方不喜欢欠人情。
- 共同努力时会区分各自贡献。
- 只有认为自己有所得时，才更关注对方需要。
- 陌生人和一般熟人关系更常接近交换关系。

## 共有关系

- 共有关系（communal relationship）重视伴侣幸福。
- 伴侣不要求即时或同等回报。
- 即使没有直接好处，也会关注对方需要。
- 帮到伴侣本身就能带来良好感受。
- Beck 与 Clark（2010b）强调共有关系中的关照和支持。
- Clark 与 Grote（1998）指出，共有关系中小牺牲和互助与更高关系质量相关。
- Clark 等人（2010）报告，伴侣越关注和慷慨，婚姻越幸福。
- Mills 等人（2004）提出共有强度（communal strength）量表。
- 共有强度越高，人们越愿意为伴侣做牺牲，婚姻满意度也更高。

## 共有关系仍有交换

- 共有关系不是完全不计得失。
- Clark（1981）指出，针锋相对的交换也可出现在共有关系中，只是时间更长、形式更多样。
- 亲密伴侣可等待更久，也接受不同形式的回报。
- Levinger（1979）提出“盈余经济”思路：关系顺利时，人们较少细算。
- 当结果恶化时，即使共有关系中的伴侣也会开始关注交换。
- Grote 与 Clark（2001）指出，不满意时公平问题会更凸显。

## 公平理论

- 公平理论（equity theory）强调成比例的公正。
- Hatfield（1983）认为，当每个伴侣的结果与贡献成比例时，人们最满意。
- 公平公式关注双方“结果/贡献”的比率是否相似。
- 公平不要求双方获得同样多。
- 如果一方贡献更多并得到更多，也可能公平。
- 如果双方结果相等但贡献不同，也可能不公平。

## 获益不足和过度获益

- 获益不足（underbenefited）指相对于贡献得到太少。
- 过度获益（overbenefited）指相对于贡献得到太多。
- 公平理论预测，两者都可能引起紧张。
- 获益不足通常带来愤怒、怨恨和被剥削感。
- 过度获益可能带来负疚感，但研究支持较不一致。
- Sprecher（2001）等纵向研究显示，过度获益并不总是降低满意。
- 相互依赖理论更强调总体奖赏数量对满意度的预测力。

## 恢复公平

- 第一种方法是改变实际贡献或结果。
- 人们可以要求更好待遇，也可以减少自己的贡献。
- 也可能试图降低伴侣得到的结果。
- 第二种方法是恢复心理公平。
- 人们可重新解释伴侣为什么值得更多，或重新解释自己应得多少。
- 最后，个体可能离开关系，或在别处寻找公平。

## 家务和照料中的公平

- 第 6 章强调，公平在家务劳动和照料小孩方面尤其重要。
- Davis 等人（2007）报告，职业女性往往承担着丈夫两倍的家庭杂务。
- Coltrane 与 Shih（2010）指出，同居情侣、男同和女同在杂务分担上通常更公平。
- Claffey 与 Mickelson（2009）指出，家务不公平会给关系造成压力。
- 因此，公平不总是决定一切，但在高负担、反复出现的任务上特别关键。
