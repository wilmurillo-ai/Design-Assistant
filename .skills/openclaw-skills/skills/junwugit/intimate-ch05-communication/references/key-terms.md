# Key Terms

## 使用方式

- 本文件用于回答术语定义、英文原词和易混概念辨析。
- 只覆盖 Miller《Intimate Relationships》第 6 版第 5 章“沟通”。
- 首次讲解时保留英文原词，后续可用中文简称。
- 不把其他章节的亲密、依恋或冲突理论混入本章术语。

## 核心术语

- 沟通（communication）：把信息、意图、情绪或态度传递给他人，并由对方解读和回应的过程。
- 意图（intention）：发送者想表达或达成的心理目标。
- 编码（encoding）：把意图转换成语言、表情、动作、语调等可被他人接收的信号。
- 解码（decoding）：接收者解释信号含义的过程。
- 影响（impact）：接收者实际体验到的效果，可能不同于发送者原本意图。
- 人际隔阂（interpersonal gap）：发送者的意图与接收者感受到的影响之间的差距。
- 非言语沟通（nonverbal communication）：不用词语本身传达意义的沟通线索。
- 面部表情（facial expressions）：通过脸部动作表达情绪和态度的线索。
- 表现规则（display rules）：文化或情境要求人们如何调节外显表情的规则。
- 微表情（microexpressions）：非常短暂、难以完全控制的情绪表露。
- 注视行为（gazing）：通过看向、避开或持续注视他人来传递注意、兴趣、地位或亲密度。
- 视觉支配比（visual dominance ratio）：说话时看对方与倾听时看对方的相对比例。
- 身体动作（body movement）：姿势、手势、动作节奏和身体朝向等线索。
- 身体接触（touch）：握手、亲吻、拥抱、按摩等通过身体接触传递关系含义的行为。
- 人际距离（interpersonal distance）：互动者之间保持的物理距离。
- 亲密区（intimate zone）：Hall（1966）区分的人际距离之一，约 0-46 厘米。
- 个人区（personal zone）：Hall（1966）区分的人际距离之一，约 46 厘米-1.2 米。
- 社交区（social zone）：Hall（1966）区分的人际距离之一，约 1.2-3.7 米。
- 公共区（public zone）：Hall（1966）区分的人际距离之一，约 3.7 米以上。
- 副语言（paralanguage）：语速、音高、停顿、叹息、笑声等词语之外的声音线索。
- 非言语敏感性（nonverbal sensitivity）：准确发送和准确解读非言语线索的能力。
- 自我表露（self-disclosure）：向他人透露关于自己的私人信息。
- 广度（breadth）：自我表露涉及的话题范围。
- 深度（depth）：自我表露的私人性、情绪性或脆弱程度。
- 互惠（reciprocity）：一方表露后，另一方也作出相近程度表露的倾向。
- 社会渗透理论（social penetration theory）：Altman 与 Taylor（1973）提出的关系发展模型，用广度和深度描述亲密增加。
- 亲密过程模型（interpersonal process model of intimacy）：Reis 与 Shaver（1998）强调表露、回应和被感知应答性共同形成亲密。
- 应答性（responsiveness）：伴侣表现出理解、关心、接纳和重视的程度。
- 秘密（secrets）：有意不让特定他人知道的信息。
- 禁忌话题（taboo topics）：关系中被回避、难以公开讨论或被认为危险的话题。
- 直言倾向（blirtatiousness）：Swann 与 Rentfrow（2001）描述的快速、直接说出想法和感受的倾向。
- 工具性沟通（instrumental communication）：以解决问题、给建议、完成任务为主的沟通。
- 表达性沟通（expressive communication）：以表达情感、理解和支持为主的沟通。
- 读心（mindreading）：没有检验就假定自己知道对方的想法或动机。
- 翻旧账（kitchen-sinking）：一次争论中把许多旧问题一起倒出来。
- 跑题（off-beam）：偏离原本讨论的具体问题。
- 退避（stonewalling）：停止回应、沉默或退出互动。
- 是的但是（yes-butting）：表面承认对方，随后立即否定或反驳。
- 交叉抱怨（cross-complaining）：不回应对方抱怨，而是立刻提出自己的抱怨。
- 精确表述（being specific）：描述具体行为和情境，而不是贴人格标签。
- XYZ 陈述（XYZ statement）：在 Y 情境下你做 X 时，我感到 Z。
- 积极倾听（active listening）：用复述和检核来确认自己是否理解对方。
- 复述（paraphrasing）：用自己的话重述对方意思。
- 知觉检核（perception checking）：把自己的解读当作假设提出，请对方确认。
- 确认（validation）：承认对方观点或感受有其可理解性。

## 易混辨析

- 自我表露不是毫无保留：第 5 章强调适当、渐进和被回应的表露，而不是把所有隐私立刻说出。
- 秘密不等于禁忌话题：秘密是信息被隐藏；禁忌话题是某类话题被关系双方回避。
- 积极倾听不等于同意：复述和检核是为了确认理解，不要求立刻接受对方结论。
- 确认不等于认输：确认对方感受可理解，不代表放弃自己的立场。
- 非言语线索不等于“真实真相”：它们很重要，但仍需结合情境和语言，避免单线索过度解读。
