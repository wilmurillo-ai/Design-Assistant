# Nonverbal Communication

## 使用方式

- 本文件用于回答表情、眼神、动作、触摸、距离、声音和非言语敏感性问题。
- 若用户问“他是不是在说谎/不爱我”，先提醒不能靠单一线索定论。
- 回答应优先围绕第 5 章的线索类型和研究发现。
- 具体研究者和年份可从 `research.md` 补充。

## 面部表情

- 面部表情（facial expressions）是情绪沟通的重要线索。
- 本章引用的研究显示，人们能很快识别一些基本情绪表情。
- Tracy 与 Robins（2008）报告，美国大学生能在 3/4 秒或更短时间内识别快乐、悲伤、厌恶和惊讶。
- 表现规则（display rules）说明文化和情境会要求人们调节真实表情。
- 常见调节方式包括强化、弱化、中和和掩饰。
- 假笑和真实笑容可能不同，但日常互动中并不总容易准确区分。
- 微表情（microexpressions）短暂出现，可能透露被压抑的情绪。
- 解释表情时要结合语境，避免把一次表情当作稳定人格证据。

## 注视行为

- 注视行为（gazing）能表达注意、兴趣、亲密、回避和地位。
- 课本给出的常见模式是：倾听者看说话者约 60% 的时间，说话者看倾听者约 40% 的时间。
- 高地位者的视觉支配比可能反转，形成说话时看得更多、倾听时看得更少的模式。
- 持续注视可能增加亲密感，也可能在不同情境下显得侵犯或威胁。
- 解释眼神时必须考虑文化、地位、关系亲密度和谈话主题。

## 身体动作

- 身体动作（body movement）包括姿势、手势、身体朝向和动作节奏。
- 教师短片、人格判断和海关观察的研究都说明身体线索能快速影响他人判断。
- 身体有时比脸部更难完全控制，因此可能泄露紧张、回避或偏好。
- 手势含义有文化差异，不能把某个动作当作跨文化固定符号。
- 高地位或自信常与较开放、不对称的姿势有关，但不是绝对诊断。

## 身体接触

- 身体接触（touch）可以表达亲密、支持、支配、安慰或礼貌。
- 握手、亲吻、拥抱、按摩等在不同文化和关系中含义不同。
- 第 5 章提到，亲密增加时接触也常增加。
- 接吻和按摩研究被用来说明亲密触摸可能与压力和健康指标相关。
- 触摸是否积极，取决于双方关系、同意、情境和文化规则。

## 人际距离

- Hall（1966）把人际距离划分为四个区域。
- 亲密区（intimate zone）：约 0-46 厘米。
- 个人区（personal zone）：约 46 厘米-1.2 米。
- 社交区（social zone）：约 1.2-3.7 米。
- 公共区（public zone）：约 3.7 米以上。
- 亲密伴侣的距离通常更近，但冲突或不满意可能让距离增大。
- 文化、性别、地位和情境都会影响人们觉得“舒服”的距离。

## 副语言

- 副语言（paralanguage）包括音高、音量、语速、停顿、笑声、叹息等。
- 人们能从词语之外的声音线索判断情绪。
- 声音吸引力在本章中与面部、身体吸引力和亲密行为研究联系在一起。
- 语气可以改变同一句话的关系含义。
- 因此，学习沟通时不能只看文字内容，也要看声音如何让信息被接收。

## 线索结合与模仿

- 非言语线索常一起工作，而不是孤立出现。
- 当语言和动作冲突时，接收者常转向非言语线索判断真实意图。
- 模仿（mimicry）可以增加喜欢和互动顺畅感。
- 但模仿要自然、轻微；明显操控式模仿可能适得其反。

## 非言语敏感性

- 非言语敏感性（nonverbal sensitivity）包括发送线索和读取线索两方面。
- Noller 的研究显示，不满意的伴侣可能发送更多混合信息，也更容易误读彼此。
- 亲密并不保证准确理解；有些痛苦伴侣理解陌生人反而更好。
- 女性在许多非言语解码任务上表现更好，但动机和任务情境会影响性别差异。
- 训练、练习和对具体伴侣的熟悉都可能改善非言语理解。
