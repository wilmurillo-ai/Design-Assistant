# AET 与 JCI: 设备、工具与人体工学取向

## 为什么单独放一组

第 3 章在 trait-based 方法后介绍两种特别关注设备与工具的 worker-oriented 方法：

- **AET**: 从 ergonomics 角度看工作如何设计得更适合人。
- **JCI**: 用大量工具/设备条目帮助描述入门岗位，服务职业指导与培训。

两者都不只是列任务，而是问：工人与工具、设备、环境之间的关系会要求人具备什么，或会给人带来什么负荷？

## AET

全称：**Arbeitswissenschaftliches Erhebungsverfahren zur Tätigkeitsanalyse**。课本翻译为 **ergonomic task analysis data collection procedure**。

AET 来自德国，由 Rohmert 等人发展。它从 ergonomics 视角出发：在最大化绩效质量与数量的同时，最小化人的 stress 与 strain。

关注点：

- 完成工作所用 equipment。
- 工作条件与环境。
- 人与机器/工具互动时的负担。
- 是否可以重新设计工作，让它对工人更友好。

## AET 的典型用途

| 用途 | 示例 |
|---|---|
| 工作再设计 | 把需要反复弯腰搬车轮的流程改成让车轮沿坡道滚到车轴高度 |
| 人体工学改良 | 改善 hand tools、界面、控制台、电脑系统 |
| 残障便利 | 移除某个不必要任务，或提供 work aid/tool |
| 降低身体负荷 | 改变搬运、伸手、站立、视觉辨认等工作条件 |

课本例子：

- 强壮的人不代表应该整天把车轮从地上抬到车轴上。设计坡道可以减少重复背部 strain。
- 如果某合格员工视力差，可提供放大镜和强光。
- 如果工作偶尔要开车，而这对某人构成障碍，可以把该任务移给他人，让其完成岗位核心任务。

## AET 与合理便利的关系

当用户问 ADA、disability accommodation 或 job redesign 时，可以用 AET 给直觉，但提醒法律细节在后续章节。AET 的本章价值是说明：worker-oriented 并不只等于选人，也可以通过改变设备与环境降低对人的不必要要求。

## Job Components Inventory (JCI)

**JCI** 由 Banks 在英国发展，用于描述广泛的 entry-level jobs。它面向青少年职业指导与培训，让年轻人理解岗位实际是什么、需要如何准备。

JCI 最突出的部分是工具/设备：

- 包含约 **220** 个 tools and pieces of equipment。
- 工具类型包括 marking/drawing、measuring、holding/securing 等。

例子：

| 工具类型 | 示例 |
|---|---|
| Marking or drawing | scribes、dividers |
| Measuring length/angles/size/levels | micrometer、protractor |
| Holding or securing objects | clamp、tweezers |

## JCI 的用途

| 用途 | 说明 |
|---|---|
| Vocational guidance | 帮学生看到岗位实际工具与工作内容 |
| Training | 根据工具/设备使用要求设计入门培训 |
| Entry-level job description | 把岗位从抽象名称落到实际操作对象 |

## AET vs JCI

| 维度 | AET | JCI |
|---|---|---|
| 核心视角 | ergonomics、人机工效、stress/strain | 工具/设备清单、入门岗位实际内容 |
| 主要用途 | 工作再设计、便利、降低负荷 | 职业指导、培训 |
| 典型问题 | 能否重新设计工具/环境，让人更少受伤或更高效？ | 这份工作会用到哪些工具，学习者需准备什么？ |
| 与 worker-oriented 的关系 | 关注工作对人的身体/认知适应要求 | 关注工具使用带来的技能需求 |

## 教学提示

如果用户把 AET/JCI 当成 work-oriented 方法，可以这样解释：

> 它们确实很关心工具和工作环境，所以看起来像 work-oriented。但本章把它们放在 worker-oriented，是因为它们的重点是这些工具/设备/环境如何转化为对人的要求、负荷、适应与训练需求。

## 回答模板

当用户问"什么时候选 AET"：

> 当目的不是单纯选人，而是要改造工作、降低身体负荷、提高人与设备的匹配，或为残障员工设计便利时，AET 比 JEM/PAQ 更贴合。

当用户问"JCI 有什么特别"：

> JCI 的特别之处是它用大量工具/设备条目把 entry-level jobs 讲具体。它原本用于职业指导，但也能帮助设计培训，因为训练往往要围绕实际工具展开。
