# TTAS、ARS、ORP

## 放在一起的原因

这三种方法都用**标准化的人类特质/能力/需要清单**来描述岗位要求或岗位回报。

| 方法 | 描述对象 | 关键词 | 典型用途 |
|---|---|---|---|
| TTAS | traits: can do + will do | Lopez, 33 traits | selection、training、job description、job evaluation |
| ARS | generic human abilities | Fleishman, tests, job families | selection、能力测验、岗位族 |
| ORP | occupational reinforcers | Borgen, needs/rewards | vocational guidance |

教学时先问：你想知道"人要有什么"还是"岗位能满足人什么需要"？前者偏 TTAS/ARS，后者偏 ORP。

## Threshold Traits Analysis System (TTAS)

**TTAS** 由 Felix Lopez 于 1970 年发展。目标是提供一个理论一致、trait-oriented、多用途且法律上更可辩护的 job analysis 方法。

TTAS 包含多个组件：

- selection 用 trait description。
- training 用组件。
- job description 用组件。
- 也可用于 job evaluation 等用途。

核心是 **TTA (threshold traits analysis)**，基于 33 个标准 traits。

## TTAS 的 can do / will do

| 类别 | 含义 | 子类/例子 |
|---|---|---|
| can do | 作为 abilities 的 traits，覆盖体力、心智、可习得能力 | physical exertion 下有 strength 与 stamina；information processing 下有 comprehension |
| will do | 作为 attitudes 的 traits，覆盖动机与社会特质 | adaptability to change、adaptability to repetition、dependability、tolerance、influence、cooperation |

例子：

- **Strength**: lift、push、pull objects 的能力。
- **Stamina**: 长时间消耗体力的能力。
- **Comprehension**: 理解口头与书面 ideas 的能力。

TTAS 的一个实用价值是防遗漏：它用一套广泛 trait list 提醒 job analyst 不要只看眼前显眼的任务能力。

## Ability Requirements Scales (ARS)

**ARS** 来自 Ed Fleishman 及其同事。它评估每种 generic human ability 对岗位的需求程度。

关键点：

- 每种 ability 都有描述，并可链接到一个或多个心理测验。
- 结果可用于 selection。
- 通用 ability 语言可以帮助建立 **job families**，即能力需求相似的岗位簇。
- 缺点是很多 ability 名称对非心理学背景用户不直观。

## Table 3.6: ARS 能力样例

| 能力类别 | Ability | 描述 |
|---|---|---|
| Cognitive | Oral comprehension | Understands spoken words |
| Cognitive | Mathematical reasoning | Reason with mathematical symbols |
| Cognitive | Speed of closure | Combine bits into meaningful patterns |
| Cognitive | Spatial orientation | Tell where you are in relation to an object |
| Psychomotor | Control precision | Operate a vehicle |
| Psychomotor | Multilimb coordination | Coordinate movements of two or more limbs |
| Psychomotor | Finger dexterity | Make skilled movements of the fingers |
| Psychomotor | Speed-of-limb movement | Move limbs quickly |
| Physical | Static strength | Exhibit push or pull strength |
| Physical | Dynamic flexibility | Repeatedly bend, stretch, and twist |
| Physical | Gross body equilibrium | Demonstrate balance |
| Physical | Stamina | Continue working over time |
| Sensory/Perceptual | Night vision | See in the dark |
| Sensory/Perceptual | Hearing sensitivity | Hear loudness and pitch |
| Sensory/Perceptual | Speech recognition | Understand speech |

教学提示：ARS 的表格适合让学生练习把岗位任务翻译为能力需求。例如"夜间巡逻"可能涉及 night vision、spatial orientation、decision-making 等。

## Occupational Reinforcer Pattern (ORP)

**ORP** 的理论基础是个体 needs profile 的差异。人们从工作中想得到的东西不同：有人重视 recognition，有人不喜欢独处，有人偏好正式或随意环境。

ORP 描述的是：某个 occupation 能提供哪些 rewards/reinforcers。它主要服务于 **vocational guidance**。如果个人 needs 测验结果与某岗位提供的 reinforcers 相似，这个岗位可能更适合他/她。

## Table 3.7: ORP reinforcers

| Reinforcer | 含义 |
|---|---|
| Ability utilization | 使用自己的能力 |
| Activity | 一直保持忙碌 |
| Authority | 指挥他人 |
| Compensation | 公平报酬 |
| Creativity | 尝试自己的想法 |
| Moral values | 避免工作与自我价值冲突 |
| Security | 稳定就业 |
| Social status | 有声望的工作 |
| Variety | 工作经常变化 |
| Autonomy | 很少监督下规划工作 |

## 三者如何区分

| 问题 | 方法 |
|---|---|
| 这份工作需要哪些广泛 traits，不想漏掉 will do 特质？ | TTAS |
| 这份工作需要哪些可用心理测验测量的通用 abilities？ | ARS |
| 这个人想要的工作回报与某 occupation 提供的回报是否匹配？ | ORP |

## 和 JEM/PAQ 的差别

- JEM 从岗位 SME 语言出发，元素更岗位特定。
- PAQ 用标准化 job elements 描述工作要求，覆盖面广但较抽象。
- TTAS/ARS/ORP 用预设 trait/ability/reinforcer 清单，便于系统扫描与匹配，但可能离具体任务更远。

## 回答模板

当用户问"ARS 和 TTAS 有什么区别"：

> TTAS 的 traits 更广，包含 can do 与 will do，所以会覆盖动机和社会特质；ARS 更聚焦 generic human abilities，并且每种 ability 可链接到心理测验，因此更直接服务于 selection 与 job families。

当用户问"ORP 是不是 worker requirement"：

> ORP 更准确地说不是问工人需要具备什么，而是问工作能提供什么 reinforcers。它通过比较个人需要与岗位回报来服务职业指导，所以和 TTAS/ARS 的方向相反。
