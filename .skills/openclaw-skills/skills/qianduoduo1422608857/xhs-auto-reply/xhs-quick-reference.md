# 小红书快速自查卡

> 发布前30秒快速检查

## ❌ 绝对禁止

- "免费上网"、"蹭网"、"破解"
- "翻墙"、"VPN"
- "加微信"、"扫码进群"

## ⚠️ 尽量避免

- "自动"、"自动化"、"脚本"
- "教程"(改为"分享")
- "免费"(改为"配额"、"额度")
- "赚钱"(改为"收入")
- 外部工具名

## ✅ 推荐表达

| 危险词 | 安全替代 |
|--------|----------|
| 自动化 | 智能、便捷 |
| 教程 | 分享、经验 |
| 免费 | 配额、额度 |
| 上网 | 联网、在线 |
| 必须 | 建议试试 |

## 🎯 标题公式

**危险:** "XXX自动XXX教程"
**安全:** "我的XXX使用经验"

**危险:** "免费XXX神器"
**安全:** "好用的XXX分享"

## 📋 发布前检查

1. 标题有敏感词? → 改!
2. 像营销号? → 改!
3. 技术感太强? → 改!

**原则:** 像朋友聊天,不像老师上课
