# ✅ CHECKLIST: Preparación de Entrevista McKinsey

## 📅 1-2 Días Antes

### Investigación del Cliente
- [ ] Revisar sitio web completo
- [ ] Analizar redes sociales (LinkedIn, Instagram, etc.)
- [ ] Buscar noticias/referencias en medios
- [ ] Revisar perfiles de equipo clave en LinkedIn
- [ ] Investigar competidores directos
- [ ] Revisar reviews/comentarios de clientes (si aplica)

### Análisis de Industria
- [ ] Identificar tendencias del sector
- [ ] Revisar métricas estándar de la industria
- [ ] Identificar regulaciones relevantes
- [ ] Revisar casos de éxito de IA en la industria

### Preparación de Hipótesis
- [ ] Completar Template de Análisis Preliminar
- [ ] Definir 3 hipótesis principales a validar
- [ ] Preparar preguntas para cada hipótesis
- [ ] Identificar métricas clave a explorar

## 📅 Día de la Entrevista (Mañana)

### Revisión Final
- [ ] Revisar Template de Análisis Preliminar (10 min)
- [ ] Repasar preguntas clave (5 min)
- [ ] Verificar demo funcional (si aplica, 5 min)
- [ ] Confirmar stakeholders que asistirán

### Preparación Logística
- [ ] Cargar laptop al 100%
- [ ] Verificar conexión internet (backup hotspot)
- [ ] Probar equipo de audio (si es virtual)
- [ ] Confirmar dirección/hora (si es presencial)
- [ ] Preparar materiales físicos (notebook, tarjetas)

### Mental Preparation
- [ ] Recordar objetivo: ESCUCHAR más que hablar (80/20)
- [ ] Enfocarse en entender, no en vender
- [ ] Prepararse para pivotear si hipótesis son incorrectas
- [ ] Recordar técnica "5 Whys" para causa raíz

## 📅 30 Minutos Antes

### Setup Final
- [ ] Llegar 15 min antes (presencial) / Conectar 10 min antes (virtual)
- [ ] Configurar grabación (con permiso explícito)
- [ ] Abrir Template de Análisis Preliminar
- [ ] Abrir notebook para notas
- [ ] Silenciar notificaciones

### Última Revisión
- [ ] Nombre correcto de empresa y personas
- [ ] Pronunciación correcta de nombres
- [ ] Contexto específico de este cliente
- [ ] Objetivos específicos de esta entrevista

## 🎯 Durante la Entrevista

### Apertura (Primeros 5 min)
- [ ] Agradecer por el tiempo
- [ ] Confirmar agenda y tiempo disponible
- [ ] Explicar objetivo: entender sus desafíos
- [ ] Pedir permiso para grabar/tomar notas
- [ ] Preguntar: "¿Hay algo específico que quieras cubrir?"

### Exploración (Minutos 5-40)
- [ ] Empezar con preguntas abiertas
- [ ] Usar técnica "Cuéntame más sobre..."
- [ ] Validar/invalidar hipótesis preparadas
- [ ] Buscar datos concretos (no opiniones)
- [ ] Preguntar "¿Por qué?" 5 veces para causa raíz
- [ ] Tomar notas de métricas mencionadas

### Cuantificación (Minutos 40-50)
- [ ] Preguntar por impacto económico de problemas
- [ ] Buscar números concretos: horas, porcentajes, dinero
- [ ] Preguntar: "¿Cuánto te cuesta esto actualmente?"
- [ ] Preguntar: "¿Cuál sería el valor de resolverlo?"

### Cierre (Últimos 10 min)
- [ ] Resumir hallazgos principales
- [ ] Validar resumen con cliente: "¿Capté correctamente?"
- [ ] Preguntar: "¿Qué no pregunté que debería haber preguntado?"
- [ ] Definir next steps claros
- [ ] Agradecer nuevamente
- [ ] Confirmar seguimiento (fecha/hora)

## 📝 Inmediatamente Después

### Síntesis Rápida (Primeros 15 min post-entrevista)
- [ ] Transcribir notas clave mientras están frescas
- [ ] Identificar 3-5 hallazgos principales
- [ ] Notar emociones/tono (no solo contenido)
- [ ] Identificar contradicciones o áreas confusas

### Análisis Inicial
- [ ] ¿Se validaron las hipótesis iniciales?
- [ ] ¿Cuáles fueron las sorpresas?
- [ ] ¿Qué dolor fue más intenso/urgente?
- [ ] ¿Qué métricas se mencionaron?
- [ ] ¿Cuál es el ROI potencial estimado?

### Acciones Inmediatas
- [ ] Enviar email de agradecimiento (mismo día)
- [ ] Incluir resumen breve de lo discutido
- [ ] Confirmar next steps acordados
- [ ] Agendar seguimiento si se acordó

## 📊 24 Horas Después

### Síntesis Completa
- [ ] Completar Memo de Diagnóstico McKinsey
- [ ] Priorizar oportunidades identificadas
- [ ] Estimar ROI preliminar para cada oportunidad
- [ ] Identificar riesgos y barreras

### Preparación de Next Steps
- [ ] Preparar propuesta de diagnóstico detallado (si aplica)
- [ ] Definir entregables específicos para el cliente
- [ ] Estimar timeline y recursos necesarios
- [ ] Preparar carta de intención (letter of intent)

### Aprendizaje y Mejora
- [ ] ¿Qué funcionó bien en la entrevista?
- [ ] ¿Qué podría mejorar para la próxima?
- [ ] ¿Qué preguntas fueron más efectivas?
- [ ] ¿Qué información faltó investigar de antemano?

## 🎯 Métricas de Calidad de Entrevista

### Durante la Entrevista
- [ ] Ratio hablar/escuchar: 20/80 (ideal)
- [ ] Preguntas de causa raíz: ≥3 por problema
- [ ] Métricas cuantificadas obtenidas: ≥5
- [ ] Next steps claros definidos: ✅

### Post-Entrevista
- [ ] Síntesis completada en <24h: ✅
- [ ] Hallazgos validados con cliente: ✅
- [ ] ROI preliminar estimado: ✅
- [ ] Propuesta preparada (si aplica): ✅
