# 更新日志 - Tavily Search

## v1.0.0 (2026-04-15)

### 🚀 主要改进

#### 功能增强
1. **缓存系统**
   - 添加文件缓存支持
   - 可配置TTL（默认5分钟）
   - 自动清理过期缓存
   - 支持缓存清除命令

2. **错误处理优化**
   - 自定义异常类
   - 详细的错误消息
   - 网络超时处理
   - API密钥验证改进

3. **新输出格式**
   - `text`: 简单文本格式
   - 增强的Markdown格式
   - 保持与Brave搜索兼容

4. **配置选项**
   - 支持环境变量配置
   - 自定义缓存目录
   - 可配置超时时间
   - 详细模式输出

#### 命令行参数
- `--timeout`: 请求超时时间
- `--cache-ttl`: 缓存生存时间
- `--no-cache`: 禁用缓存
- `--clear-cache`: 清除所有缓存
- `--verbose`: 详细输出模式

#### 代码质量
- 模块化设计
- 类型提示
- 更好的代码注释
- 错误处理改进

### 📖 文档更新
- 完整的SKILL.md文档
- 使用示例
- 故障排除指南
- 最佳实践建议
