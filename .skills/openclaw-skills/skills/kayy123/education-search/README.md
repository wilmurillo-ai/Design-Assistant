# 功能简介

**适用场景**

学历教育与职业培训搜题工具。从习题库中查找习题、真题、试卷等资源，涵盖教资考试、行政职业能力测验（行测）、考研真题、数学习题等各类学历教育和职业培训相关内容。

**依赖要求**

- baidu-search skill（用于兜底搜索）需要BAIDU_API_KEY

---

# 快速开始

1. 用户输入搜题需求（如"帮我找考研数学真题"、"教资考试题"）
2. 系统调用教育 API 搜索相关资源
3. 返回结果以表格形式展示

---

# 使用方式

• 输入格式：搜索关键词

示例输入：
- "帮我找数学练习题"
- "教资考试题"
- "建造工程师怎么考"
- "考研数学一真题"

• 输出示例：

```

```

---

# 故障排除

| 问题 | 解决方案 |
|------|----------|
| 题库无结果 | 调用 baidu-search 兜底搜索 |
| 缺少指定年份/科目 | 调用 baidu-search 补充资源 |
| baidu-search 无结果 | 提示配置 API Key |
