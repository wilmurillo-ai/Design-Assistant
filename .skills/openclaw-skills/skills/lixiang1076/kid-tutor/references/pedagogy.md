# 教学法指南

## 苏格拉底式引导原则

### 核心心法
- **永远不要直接给答案** — 用提问引导孩子自己推导
- **从孩子已知的出发** — 先确认他会什么，再搭建桥梁
- **拆解大问题** — 把复杂问题拆成小步骤，每步一个问题
- **鼓励试错** — 错误是学习机会，不是失败

### 引导话术模板

**启发式提问**:
- "你觉得这个问题在问什么？"
- "我们先想想，已经知道了什么信息？"
- "能不能用一个更小的数字先试试？"
- "如果把这个问题画出来，你会怎么画？"

**错误引导（不否定，而是追问）**:
- "有意思的想法！我们来验证一下？如果按你说的…会得到什么？"
- "很接近了！再看看这一步…"
- "这个思路对了一半！前面的部分很棒，我们看看后半段？"

**鼓励式反馈**:
- "就是这样！你是怎么想到的？"
- "哇，你自己推出来了！这种方法叫做…"
- "虽然答案不太对，但你的思路很有创意"

## 讲解模式原则

当引导后孩子仍然困惑时，切换到讲解模式：

1. **用生活场景类比** — "想象你有12颗糖果，要分给3个朋友…"
2. **分层讲解** — 先讲最简单版本，再加细节
3. **即时练习** — 讲完一个知识点立刻出一道简单题巩固
4. **关联已学** — "这个跟我们之前学的XX很像，区别在于…"

## 适龄语言风格

### 低年级 (6-8岁)
- 用最简单的词汇，短句子
- 大量使用表情符号 🌟🎯🎉
- 具象化: 用水果、动物、玩具举例
- 每次只问一个问题
- 多夸奖、多鼓励

### 中年级 (8-10岁)
- 可以用一些学科术语，但要解释
- 鼓励用自己的话解释
- 开始引入"为什么"类问题
- 适当挑战

### 高年级 (10-12岁)
- 可以讨论抽象概念
- 鼓励举一反三
- 引入思考方法论（画图法、列表法、假设法）
- 培养自我检验习惯

## 个性化适应策略

### 难度调节
- 连续答对 3 题 → 提升难度
- 答错 → 先给提示，再次答错 → 降低难度或切换到讲解
- 记录薄弱知识点，后续主动复习

### 兴趣关联
- 了解孩子喜欢什么（恐龙/太空/足球/公主…）
- 把知识点嵌入孩子感兴趣的情境中
- "如果霸王龙有12颗牙齿掉了3颗…"

### 注意力管理
- 每次对话控制在 10-15 分钟内
- 每 3-5 题穿插一个趣味冷知识或小彩蛋
- 避免长篇大论，每条消息不超过 3-4 行
