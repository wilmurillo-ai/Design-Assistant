# 出题模板库

## 数学题模板

### 计算类

**口算练习** (低年级)
```
计算: {num1} {op} {num2} = ?
```

**竖式计算** (中年级)
```
用竖式计算: {num1} × {num2}
```

**混合运算** (高年级)
```
计算: {expr}，注意运算顺序哦！
```

### 应用题类

**购物场景**
```
{name}去买东西，{item1}每个{price1}元，{item2}每个{price2}元。
买了{count1}个{item1}和{count2}个{item2}，一共要花多少钱？
```

**分配场景**
```
{total_count}个{item}，平均分给{people_count}个小朋友，每人分到几个？还剩几个？
```

**路程场景** (中高年级)
```
{name}从家到学校，步行每分钟走{speed}米，一共走了{time}分钟。
家到学校有多远？
```

**比较场景**
```
{name1}有{count1}个{item}，{name2}比{name1}多{diff}个。
{name2}有多少个{item}？两人一共有多少个？
```

### 图形类

**周长**
```
一个长方形，长{length}厘米，宽{width}厘米，周长是多少？
```

**面积**
```
教室的地面是长方形，长{length}米，宽{width}米。
如果用边长1米的正方形地砖铺地，需要多少块？
```

### 逻辑推理类

**找规律**
```
观察这组数: {seq}，下一个数是什么？你发现了什么规律？
```

**鸡兔同笼** (高年级)
```
笼子里有鸡和兔，一共{heads}个头，{legs}条腿。鸡和兔各有几只？
```

## 科学题模板

### 观察描述类
```
{phenomenon}，你觉得这是为什么呢？
（引导观察→假设→验证的科学思维）
```

### 分类判断类
```
下面哪些是{category}？
A. {option1}  B. {option2}  C. {option3}  D. {option4}
说说你是怎么判断的？
```

### 实验思维类
```
如果我们想知道{question}，你会设计什么实验来验证？
需要什么材料？怎么做？怎么记录结果？
```

### 生活关联类
```
你有没有注意到{daily_phenomenon}？
这背后的科学原理是什么呢？我给你几个线索…
```

## 趣味挑战模板

### 数学谜题
```
🧩 挑战时间！
我是一个两位数，十位上的数字比个位大2，两个数字加起来等于{sum}。
我是谁？
```

### 科学冷知识引题
```
🤓 你知道吗？
{fun_fact}
基于这个知识，你来想想：{follow_up_question}
```

### 估算挑战
```
🎯 不用精确计算，估一估：
{estimation_question}
说说你是怎么估的？
```
