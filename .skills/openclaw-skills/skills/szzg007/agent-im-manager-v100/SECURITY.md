# 🔒 安全说明

## 本技能的安全承诺

### ✅ 安全特性

1. **无硬编码凭证**
   - 所有 Token 由用户自行配置
   - 使用 `config.example.json` 作为示例
   - `config.json` 已添加到 `.gitignore`

2. **无外部 API 调用**
   - 所有数据在本地处理
   - 不发送数据到第三方服务器
   - 对话历史存储在浏览器 localStorage

3. **无数据外传**
   - 不收集用户信息
   - 不上传使用数据
   - 不跟踪用户行为

4. **代码完全开源**
   - 所有代码可查看
   - 无混淆或加密
   - 欢迎安全审查

### 📁 访问的资源

本技能需要访问以下本地资源：

| 路径 | 用途 | 权限 |
|------|------|------|
| `~/.openclaw/agents` | 读取 Agent 配置 | 只读 |
| `~/.openclaw/workspace-*` | Agent 工作区 | 读写 |
| `~/.openclaw/devices/paired.json` | 读取 Operator Token | 只读 |

### ⚠️ 使用建议

1. **审查代码**
   - 使用前请审查 `server-gemini.js` 和 `index.html`
   - 确保信任作者后再使用

2. **保护 Token**
   - 不要分享 `config.json` 文件
   - 不要将 Token 提交到 Git

3. **定期更新**
   - 关注 ClawHub 上的更新
   - 及时安装安全补丁

### 🛡️ 安全最佳实践

**用户应做到:**
- ✅ 定期更改 Operator Token
- ✅ 不在公共场合分享配置
- ✅ 使用防火墙保护 Gateway 端口
- ✅ 定期备份对话历史

**技能开发者承诺:**
- ✅ 及时修复安全漏洞
- ✅ 透明公开所有变更
- ✅ 不添加任何恶意代码
- ✅ 接受社区安全审查
