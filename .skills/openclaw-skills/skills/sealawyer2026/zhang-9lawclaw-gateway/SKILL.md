# 9LawClaw Gateway

**产品定位**: 9LawClaw 法律智能操作系统的统一入口网关  
**核心功能**: 身份感知 + 需求解析 + 智能路由  
**版本**: v0.5.0 (Phase 1)  
**代号**: 龙虾大脑 (🦞 Brain)

## 🎯 核心定位

不是应用，是**入口**。就像：
- Windows 的开始菜单
- 微信的搜索框
- ChatGPT 的输入框

**用户只需要说一句话，系统自动完成一切。**

## 🚀 快速开始

### 安装

```bash
# 通过 ClawHub 安装
clawhub install zhang-9lawclaw-gateway

# 或在 OpenClaw 中启用
openclaw skills enable zhang-9lawclaw-gateway
```

### 使用

```javascript
// 直接咨询
const result = await openclaw.invoke('zhang-9lawclaw-gateway', {
  query: '员工试用期被辞退，公司不给赔偿怎么办？',
  userType: 'employee',  // 可选：自动识别
  context: {}  // 上下文
});

// 结果
{
  expert: 'zhang-labor-law',  // 匹配的劳动法专家
  response: '...',  // 专家回答
  relatedCases: [...],  // 相关案例
  applicableLaws: [...],  // 适用法规
  suggestions: [...],  // 行动建议
  confidence: 0.95  // 匹配置信度
}
```

## 🔧 配置

### 基础配置

```yaml
# config/gateway.yaml

gateway:
  name: "9LawClaw Gateway"
  version: "0.5.0"
  
  routing:
    strategy: intelligent  # intelligent | keyword | hybrid
    fallback: zhang-general-law  # 默认兜底专家
    confidence_threshold: 0.7  # 最低匹配置信度
    
  experts:
    count: 36
    path: "~/.openclaw/skills/zhang-*-law"
    auto_register: true
    
  database:
    cases:
      path: "~/.openclaw/data/cases"
      count: 22316
    laws:
      path: "~/.openclaw/data/laws"
      count: 85
      
  context:
    max_history: 10
    timeout: 3600
    persist: true
    
  output:
    format: structured  # structured | simple | verbose
    include_cases: true
    include_laws: true
    max_cases: 5
    max_laws: 3
```
