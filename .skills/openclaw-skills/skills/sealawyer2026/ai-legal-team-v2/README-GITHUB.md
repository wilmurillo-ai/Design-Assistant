## 🚀 快速开始

### 安装DeerFlow 2.0

```bash
# 克隆仓库
git clone https://github.com/bytedance/deer-flow.git

# 进入目录
cd deer-flow

# 安装依赖
pnpm install

# 配置环境变量
cp .env.example .env
# 编辑.env文件，添加DEEPSEEK_API_KEY

# 启动服务
pnpm dev
```
