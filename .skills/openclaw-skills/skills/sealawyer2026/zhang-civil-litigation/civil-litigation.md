# 民商事诉讼专用提示词库
# 针对民商事案件各阶段的AI辅助提示词

---

## 一、案件分析与策略制定

### 1.1 案件初步评估
```
你是一名资深民商事律师。请对以下案件进行初步评估：

【案件基本信息】
- 案件类型：{case_type}
- 当事人地位：{party_position}（原告/被告/第三人）
- 争议标的：{subject_matter}
- 标的金额：{amount}
- 管辖法院：{court}

【案情简介】
{case_summary}

请提供：
1. 法律关系分析（请求权基础）
2. 诉讼可行性评估（胜诉概率、主要风险）
3. 证据充分性分析
4. 诉讼时效审查
5. 管辖异议可能性
6. 财产保全建议
7. 初步诉讼策略
8. 费用预算（诉讼费、律师费、保全费）
```

### 1.2 请求权基础分析
```
请对以下案件进行请求权基础分析：

【案件事实】
{case_facts}

【当事人诉求】
{party_claims}

请按照王泽鉴教授请求权基础分析方法：

## 一、请求权基础检索
1. 合同请求权
   - 主给付请求权
   - 次给付请求权（违约金、损害赔偿）
   - 附随义务请求权

2. 类合同请求权
   - 缔约过失责任
   - 无权代理责任
   - 合同无效/撤销后的返还/赔偿

3. 无因管理请求权

4. 物权请求权
   - 返还原物
   - 排除妨害
   - 消除危险

5. 侵权请求权
   - 一般侵权
   - 特殊侵权

6. 不当得利请求权

## 二、请求权基础构成要件分析
对每一可能的请求权基础：
1. 构成要件该当性
2. 违法性（侵权）
3. 有责性（侵权）
4. 抗辩事由

## 三、请求权基础竞合
1. 竞合情况识别
2. 选择最优请求权基础
3. 预备性合并（如有必要）

## 四、结论
- 最优请求权基础
- 备选方案
- 诉讼策略建议
```

## 二、起诉阶段（原告）

### 2.1 起诉状
```
请撰写民事起诉状：

【案件信息】
- 原告：{plaintiff}
- 被告：{defendant}
- 案由：{case_cause}
- 管辖法院：{court}

【诉讼请求】
{claims}

【事实与理由】
{facts_and_reasons}

【证据清单】
{evidence_list}

起诉状结构：
1. 当事人信息（原被告基本情况）
2. 诉讼请求（明确、具体、可执行）
3. 事实与理由
   - 事实部分：时间线清晰，客观陈述
   - 理由部分：法律依据、合同约定、类案支持
4. 证据和证据来源
5. 此致法院
6. 起诉人签名/盖章、日期

写作要求：
- 诉讼请求明确具体（金额、行为、确认）
- 事实陈述客观真实
- 法律依据准确
- 逻辑清晰，层次分明
```

### 2.2 证据清单与证明目的
```
请生成证据清单：

【案件类型】
{case_type}

【待证事实】
{facts_to_prove}

证据清单格式：

## 证据清单

| 序号 | 证据名称 | 证据来源 | 证明目的 | 页码 |
|-----|---------|---------|---------|------|
| 1 | {name} | {source} | {purpose} | {page} |

## 分组证明

### 第一组：主体资格证据
- 证明原被告主体适格

### 第二组：法律关系证据
- 证明合同成立/侵权事实

### 第三组：履行情况证据
- 证明履约/违约事实

### 第四组：损失证据
- 证明损失金额及计算依据

### 第五组：其他证据
- 诉讼时效、免责事由等

## 证据说明
1. 证据来源合法性说明
2. 证据真实性说明
3. 证据关联性说明
4. 证据链完整性说明
```

### 2.3 财产保全申请书
```
请撰写财产保全申请书：

【案件信息】
- 申请人：{applicant}
- 被申请人：{respondent}
- 案由：{case_cause}
- 标的金额：{amount}

【保全标的】
{preservation_target}

【保全理由】
{preservation_reasons}

申请书内容：
1. 申请事项（查封/扣押/冻结的具体财产）
2. 事实与理由
   - 案件基本情况
   - 保全必要性（转移、隐匿、毁损风险）
   - 保全紧迫性
3. 担保方式
   - 保险公司保函
   - 自有资产担保
   - 担保公司担保
4. 法律依据
   - 《民事诉讼法》第103条
   - 《民事诉讼法》第104条
5. 附件
   - 财产线索清单
   - 担保材料

注意事项：
- 提供明确的财产线索
- 说明保全必要性
- 准备等额担保
- 关注保全期限（银行存款1年，动产2年，不动产3年）
```

## 三、应诉阶段（被告）

### 3.1 答辩状
```
请撰写民事答辩状：

【案件信息】
- 答辩人：{defendant}
- 原告：{plaintiff}
- 案由：{case_cause}
- 收到起诉状日期：{receive_date}

【原告主张】
{plaintiff_claims}

【答辩要点】
{defense_points}

答辩状结构：

## 首部
- 答辩人信息
- 案由

## 答辩意见

### 一、程序性答辩（如有）
1. 管辖权异议
2. 主体不适格
3. 重复起诉
4. 诉讼时效已过

### 二、实体性答辩

#### （一）针对原告诉讼请求
逐项回应，承认/否认/部分承认

#### （二）针对原告事实陈述
1. 认可的事实
2. 否认的事实及理由
3. 需要补充的事实

#### （三）针对原告法律依据
1. 法律适用错误
2. 合同解释异议
3. 责任认定异议

### 三、反诉（如有）
1. 反诉请求
2. 事实与理由

## 尾部
- 此致法院
- 答辩人签名/盖章、日期

写作技巧：
- 针对性回应，不遗漏要点
- 先程序后实体
- 承认对己方不利的事实，否认要有依据
- 提出反诉要独立成节
```

### 3.2 管辖权异议申请书
```
请撰写管辖权异议申请书：

【案件信息】
- 异议人：{objector}
- 受理法院：{court}
- 案号：{case_number}
- 收到应诉材料日期：{receive_date}

【异议理由】
{objection_reasons}

申请书内容：
1. 异议事项（请求移送管辖/驳回起诉）
2. 事实与理由
   - 级别管辖错误
   - 地域管辖错误
   - 专属管辖冲突
   - 协议管辖无效/不适用
   - 指定管辖不当
3. 正确管辖法院
4. 法律依据
   - 《民事诉讼法》第37条（管辖恒定）
   - 具体管辖条款
5. 证据材料
   - 合同中的管辖条款
   - 住所地证明
   - 不动产证明等

注意事项：
- 答辩期内提出（收到起诉状15日内）
- 逾期提出可能不被审查
- 对裁定不服可上诉
```

### 3.3 反诉状
```
请撰写反诉状：

【本诉信息】
- 本诉原告：{original_plaintiff}
- 本诉被告：{original_defendant}（即反诉原告）
- 本诉案由：{original_case_cause}

【反诉信息】
- 反诉被告：{counterclaim_defendant}（即本诉原告）
- 反诉案由：{counterclaim_cause}

【反诉请求】
{counterclaim_requests}

【事实与理由】
{facts_and_reasons}

反诉状结构：
1. 反诉原告、被告信息
2. 反诉请求
3. 事实与理由
   - 与本诉的关联性
   - 反诉的独立性
   - 抵销或吞并本诉的可能性
4. 证据清单
5. 此致法院
6. 反诉人签名/盖章、日期

法律依据：
- 《民事诉讼法》第143条（反诉）
- 反诉与本诉须有关联性
- 反诉应在本诉进行中提出
```

## 四、庭审阶段

### 4.1 庭审提纲
```
请生成庭审提纲：

【案件信息】
- 案由：{case_cause}
- 开庭法院：{court}
- 开庭时间：{hearing_time}
- 当事人地位：{party_position}

庭审提纲：

## 一、庭前准备
- [ ] 核对证据原件
- [ ] 确认当事人/代理人身份
- [ ] 准备授权委托书
- [ ] 准备代理词要点

## 二、法庭调查

### （一）原告陈述
- [ ] 诉讼请求确认
- [ ] 事实陈述（简明扼要）

### （二）被告答辩
- [ ] 针对每项诉讼请求回应
- [ ] 事实异议点
- [ ] 法律依据

### （三）举证质证
#### 我方举证
- [ ] 证据1：{name} → 证明{purpose}
- [ ] 证据2：{name} → 证明{purpose}

#### 对方质证应对
- [ ] 真实性异议应对
- [ ] 合法性异议应对
- [ ] 关联性异议应对

#### 对方举证质证
- [ ] 证据1质证要点
- [ ] 证据2质证要点

### （四）法庭询问
- [ ] 准备向对方发问的问题
- [ ] 预测法官可能询问的问题
- [ ] 准备回答要点

## 三、法庭辩论

### （一）第一轮辩论
1. 争议焦点归纳
2. 我方观点阐述
3. 法律依据引用
4. 类案参考

### （二）第二轮辩论（针对对方观点反驳）
1. 对方主要观点归纳
2. 逐条反驳
3. 强化我方观点

### （三）最后陈述
- [ ] 总结性陈述
- [ ] 再次强调核心观点

## 四、庭后工作
- [ ] 核对庭审笔录
- [ ] 补充提交材料（如有）
- [ ] 准备代理词
- [ ] 跟进判决

## 五、争议焦点预测
1. {issue_1}
2. {issue_2}
3. {issue_3}

## 六、风险提示
- [ ] 证据不足风险
- [ ] 法律适用风险
- [ ] 对方突袭风险
- [ ] 调解可能性
```

### 4.3 质证意见
```
请撰写质证意见：

【对方证据】
- 证据名称：{evidence_name}
- 证据类型：{evidence_type}
- 证明目的：{probative_purpose}

【质证要点】
{examination_points}

质证意见格式：

## 对{证据名称}的质证意见

### 一、对证据真实性的质证
（是否真实、有无篡改、复印件与原件核对）

### 二、对证据合法性的质证
（收集主体、程序、形式是否合法）

### 三、对证据关联性的质证
（与待证事实的关联程度）

### 四、对证据证明力的质证
（证明力大小、是否需要补强）

### 五、质证结论
（是否认可、证明目的是否成立）

法律依据：
- 《民事诉讼法》第66条（证据种类）
- 《民诉解释》第104-110条（证据认定）
- 证据规则相关规定
```

```
注意事项：
- 行为异议：15日内提出
- 标的异议：执行过程中提出
- 对裁定不服可复议/起诉
```

### 6.2 再审申请书
```
请撰写再审申请书：

【案件信息】
- 原审法院：{original_court}
- 案号：{case_number}
- 申请人：{petitioner}
- 对方当事人：{opponent}
- 裁判生效日期：{effective_date}

【申请再审理由】
{retrial_reasons}

再审事由（选择适用）：

## 一、《民事诉讼法》第207条情形

### （一）有新证据，足以推翻原判决、裁定的
- 新证据说明
- 证明内容
- 足以推翻的理由

### （二）原判决、裁定认定的基本事实缺乏证据证明
- 哪些基本事实缺乏证据
- 应当有哪些证据

### （三）原判决、裁定认定事实的主要证据是伪造的
- 伪造证据说明
- 鉴定意见（如有）

### （四）原判决、裁定认定事实的主要证据未经质证
- 未经质证的证据
- 影响事实认定

### （五）对审理案件需要的主要证据，当事人因客观原因不能自行收集，书面申请人民法院调查收集，人民法院未调查收集
- 申请调查的证据
- 申请时间
- 法院未调查

### （六）原判决、裁定适用法律确有错误
- 错误适用法律
- 应当适用的法律

### （七）审判组织的组成不合法或者依法应当回避的审判人员没有回避

### （八）无诉讼行为能力人未经法定代理人代为诉讼或者应当参加诉讼的当事人，因不能归责于本人或者其诉讼代理人的事由，未参加诉讼

### （九）违反法律规定，剥夺当事人辩论权利

### （十）未经传票传唤，缺席判决

### （十一）原判决、裁定遗漏或者超出诉讼请求

### （十二）据以作出原判决、裁定的法律文书被撤销或者变更

### （十三）审判人员审理该案件时有贪污受贿，徇私舞弊，枉法裁判行为

## 二、申请期限
```

## 使用说明

### 根据案件阶段选择
- 起诉前：案件分析、请求权基础
- 一审原告：起诉状、证据清单、保全申请
- 一审被告：答辩状、管辖异议、反诉
- 庭审阶段：庭审提纲、代理词、质证意见
- 执行阶段：执行申请、执行异议
- 二审/再审：上诉状、再审申请

### 组合使用示例
```
请按照以下流程处理本案：
1. 【案件分析】使用案件初步评估 + 请求权基础分析
2. 【起诉准备】使用起诉状 + 证据清单 + 保全申请
3. 【应诉准备】使用答辩状 + 管辖异议（如有）
4. 【庭审阶段】使用庭审提纲 + 代理词 + 质证意见
5. 【执行阶段】使用执行申请 + 执行异议（如有）
```
