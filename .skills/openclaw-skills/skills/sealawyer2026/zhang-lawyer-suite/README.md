# 张律师法律AI中台

> 中国首个开源法律AI技能库 | 专业律师团队打造 | 全领域法律实践支持

## 🌟 项目简介

本项目由执业律师张律师发起，旨在为法律从业者提供**开箱即用的AI法律助手**，涵盖刑事辩护、民商事诉讼、合同审查等执业全流程。

## 📦 技能包清单

| 技能包 | 说明 | 安装命令 |
|--------|------|----------|
| `zhang-lawyer-suite` | 综合套装（包含全部） | `clawhub install zhang-lawyer-suite` |
| `zhang-criminal-defense` | 刑事辩护全流程 | `clawhub install zhang-criminal-defense` |
| `zhang-civil-litigation` | 民商事诉讼全流程 | `clawhub install zhang-civil-litigation` |
| `zhang-contract-review` | 合同审查清单 | `clawhub install zhang-contract-review` |
| `zhang-legal-polish` | 法律文书润色 | `clawhub install zhang-legal-polish` |

## 🚀 快速开始

```bash
# 安装全套技能
clawhub install zhang-lawyer-suite

# 或单独安装
clawhub install zhang-criminal-defense
clawhub install zhang-civil-litigation
clawhub install zhang-contract-review
clawhub install zhang-legal-polish
```
