## 🚀 快速开始

### 步骤1：学习剪映工具
1. 访问剪映官网：https://www.capcut.cn/
2. 下载并安装剪映
3. 创建账号并登录
4. 学习基础操作（导入素材、剪辑、添加特效）
5. 学习高级功能（模板、音乐、字幕）

### 步骤2：申请API密钥
1. 登录剪映开放平台：https://open.capcut.cn/
2. 申请剪映API密钥
3. 记录AppID和AppSecret
4. 配置到技能中

### 步骤3：视频号配置
1. 访问视频号开放平台：https://open.xishu.com/
2. 申请视频号API密钥
3. 记录AppID和AppSecret
4. 配置到技能中

### 步骤4：抖音配置
1. 访问抖音开放平台：https://open.douyin.com/
2. 申请抖音API密钥
3. 记录AppID和AppSecret
4. 配置到技能中

## 📋 工作流程

### 流程1：生成视频内容
1. 读取法律文章
2. 生成视频脚本（基于抖音脚本模板）
3. 生成画面描述
4. 生成配音文案
5. 生成字幕内容

### 流程2：制作视频
1. 导入素材到剪映
2. 使用模板创建草稿
3. 添加画面描述（文字、图片、视频）
4. 添加音乐
5. 添加字幕
6. 导出最终视频

### 流程3：发布视频
1. 上传视频到视频号
2. 发布视频到抖音
3. 记录发布历史
4. 监控发布效果
