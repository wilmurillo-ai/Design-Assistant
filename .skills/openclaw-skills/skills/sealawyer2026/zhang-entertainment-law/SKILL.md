## 典型合同模板

### 模板1：演员聘用合同（主演）

```
核心条款：
1. 授权范围
   - 影片名称：__________
   - 角色名称：__________
   - 授权权利：表演权、肖像权、姓名权
   - 使用范围：全球/中国大陆/院线+网络
   - 使用期限：影片版权存续期间

2. 酬金与支付
   - 总酬金：人民币____万元（含税）
   - 支付方式：签约30% / 开机40% / 杀青30%
   - 税务承担：含税/不含税
   - 发票类型：演艺服务费

3. 拍摄安排
   - 拍摄周期：____年____月____日至____年____月____日
   - 工作日：预计____个工作日
   - 定妆、宣传：额外____个工作日
   - 超期处理：每日额外酬金____%

4. 署名条款
   - 署名位置：领衔主演/主演/特别出演
   - 署名顺序：第____位
   - 字体大小：不小于片名____%
   -  trailers、海报署名

5. 道德条款
   - 艺人保证无负面新闻影响影片
   - 如发生丑闻，片方有权解约、删减戏份
   - 违约金：已支付酬金的____倍
```

### 模板2：MCN机构与创作者合作协议

```
核心条款：
1. 合作期限
   - 合同期限：____年，自____年____月____日起
   - 续约条件：到期前____日书面通知
   - 优先续约权：MCN享有同等条件下的优先权

2. 账号归属
   - 签约前已有账号归属：创作者/MCN共有
   - 签约后新建账号归属：____
   - 解约后账号处理：____
   - 粉丝数据归属约定

3. 收益分成
   - 平台流量收益：创作者____%，MCN____%
   - 广告接单收益：创作者____%，MCN____%
   - 直播打赏收益：创作者____%，MCN____%
   - 结算周期：月结/季度结

4. 运营支持
   - MCN承诺提供的资源：
     * 运营团队____人
     * 每月商单不少于____个
     * 流量扶持/DOU+投放
     * 内容策划支持

5. 竞业限制
   - 合同期内不得与其他MCN合作
   - 合同期满____个月内不得从事竞争业务
   - 限制区域：中国大陆/全球
   - 补偿金：每月____元
```

## 风险警示

### ⚠️ 高风险业务

1. **阴阳合同**
   - 法律后果：逃税罪，最高7年有期徒刑
   - 案例：某女星逃税案，补缴税款+罚款超8亿元

2. **虚假流量**
   - 刷量行为可能构成诈骗罪、非法经营罪
   - 平台处罚：限流、封号、永久禁止入驻

3. **版权侵权**
   - 未经授权使用音乐、视频素材
   - 二创内容可能构成侵权
   - 平台连带责任风险

4. **艺人丑闻**
   - 代言人负面事件导致品牌损失
   - 影视剧因演员问题无法播出
   - 建议：完善道德条款、购买保险

## 使用示例

### 示例1：影视投资协议审查

**用户需求**：审查一份网络电影联合投资协议

**服务流程**：
1. **投资结构分析**
   - 确认投资方份额比例
   - 分析优先回款条款
   - 评估保底条款合法性

2. **收益分配审查**
   - 分账顺序：宣发费→投资款→收益
   - 分配比例：按投资比例/约定比例
   - 审计权利：确保透明度

3. **风险提示**
   - 对赌条款的合理性
   - 发行代理权约定
   - 版权归属与后续开发权

### 示例2：主播跳槽纠纷处理

**案情**：某游戏主播与MCN解约后加入竞品平台

**处理策略**：
1. **合同效力审查**
   - 竞业限制条款是否有效
   - 补偿金是否已支付
   - 限制范围是否合理

2. **违约责任评估**
   - 违约金金额：500万元
   - 实际损失举证
   - 违约金调减可能性

3. **诉讼/仲裁方案**
   - 管辖：约定仲裁/法院
   - 证据：直播数据、收益流水
   - 保全：冻结新平台账户
