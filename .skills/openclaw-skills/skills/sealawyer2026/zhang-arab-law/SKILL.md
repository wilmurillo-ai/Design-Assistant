# 九章-阿拉伯/中东法律专家

伊斯兰法与现代商业合规

## 功能

- 🕌 伊斯兰法（Sharia）基础
- 🇦🇪 阿联酋投资与自贸区
- 🇸🇦 沙特2030愿景合规
- 🏦 伊斯兰金融合规
- ⚖️ 中东争议解决

## 核心内容

### 1. 伊斯兰法基础

**法律渊源**
- 古兰经
- 圣训
- 公议（Ijma）
- 类比（Qiyas）

**与现代法关系**
- 沙里亚作为补充法
- 民法法系为主（埃及/阿联酋）
- 普通法（迪拜DIFC）
- 混合法系

**禁止事项**
- 利息（Riba）
- 投机（Gharar）
- 赌博（Maysir）
- 违禁品（猪肉/酒精）

### 2. 阿联酋

**自贸区**
- 迪拜DMCC
- 迪拜DIFC（普通法）
- 阿布扎比ADGM
- 杰贝阿里JAFZA

**公司设立**
- 自贸区公司：100%外资
- 本地公司：需本地保人（51%）
- 设立时间：1-2周

**税务**
- 无企业所得税（2023前）
- 2023起：9%（利润超37.5万迪拉姆）
- 无个人所得税
- 增值税5%

### 3. 沙特阿拉伯

**2030愿景**
- Vision 2030改革
- 经济多元化
- 外商投资开放

**SAGIA/MISA**
- 投资部审批
- 负面清单缩减
- 100%外资许可

**劳工改革**
- Kafala制度废除
- 外籍劳工权益
- 本地化成比例

### 4. 伊斯兰金融

**基本原则**
- 风险共担
- 资产支持
- 禁止利息

**主要工具**
- Murabaha（成本加利润销售）
- Musharaka（合伙）
- Mudaraba（资金+管理合伙）
- Ijara（租赁）
- Sukuk（伊斯兰债券）
