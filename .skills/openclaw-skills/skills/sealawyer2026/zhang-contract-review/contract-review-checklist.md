# 各类合同审查要点与风险提示

## 一、通用合同审查框架

### 1.1 合同审查通用清单
```
你是一名专业合同审查律师。请对以下合同进行全面审查：

【合同信息】
- 合同名称：{contract_name}
- 合同类型：{contract_type}
- 我方地位：{party_position}（甲方/乙方/丙方）
- 合同标的：{subject_matter}
- 合同金额：{amount}

【审查要求】
{review_requirements}

请按照以下框架审查：

## 一、主体资格审查
- [ ] 当事人名称与营业执照一致
- [ ] 法定代表人/授权代表签字/盖章
- [ ] 签约主体具有相应资质
- [ ] 特殊行业许可（如需要）
- [ ] 代理人授权委托书（如适用）

## 二、形式要件审查
- [ ] 合同名称准确
- [ ] 合同编号完整
- [ ] 签订日期明确
- [ ] 签订地点明确
- [ ] 份数约定清晰
- [ ] 生效条件明确
- [ ] 附件清单完整

## 三、核心条款审查

### （一）标的条款
- [ ] 标的描述准确、具体
- [ ] 数量/规格/型号明确
- [ ] 质量标准约定
- [ ] 验收标准约定

### （二）价款条款
- [ ] 金额大小写一致
- [ ] 币种明确
- [ ] 含税/不含税约定
- [ ] 发票类型约定
- [ ] 付款方式明确
- [ ] 付款时间节点
- [ ] 账户信息准确

### （三）履行条款
- [ ] 履行期限明确
- [ ] 履行地点明确
- [ ] 履行方式明确
- [ ] 履行顺序约定
- [ ] 分批履行安排（如适用）

### （四）违约责任
- [ ] 违约情形列举
- [ ] 违约金计算方式
- [ ] 违约金比例合理性
- [ ] 损失赔偿范围
- [ ] 免责条款审查

### （五）争议解决
- [ ] 争议解决方式（诉讼/仲裁）
- [ ] 管辖法院/仲裁机构
- [ ] 管辖约定有效性
- [ ] 法律适用

## 四、特殊条款审查
- [ ] 保密条款
- [ ] 知识产权条款
- [ ] 不可抗力条款
- [ ] 合同变更条款
- [ ] 合同转让条款
- [ ] 合同解除条款
- [ ] 通知送达条款
- [ ] 完整协议条款
- [ ] 可分割性条款

## 五、风险识别
1. {risk_1}
2. {risk_2}
3. {risk_3}

## 六、修改建议
1. {suggestion_1}
2. {suggestion_2}
3. {suggestion_3}

## 七、审查结论
- [ ] 可直接签署
- [ ] 需修改后签署
- [ ] 重大风险，建议不签

审查人：________
审查日期：________
```

## 二、常见合同类型审查要点

### 2.1 买卖合同审查要点
```
请审查买卖合同，重点关注：

【合同信息】
- 卖方：{seller}
- 买方：{buyer}
- 标的物：{goods}
- 金额：{amount}

## 一、标的物条款
- [ ] 名称、规格、型号、数量
- [ ] 质量标准（国标/行标/企标）
- [ ] 技术参数、性能指标
- [ ] 包装标准
- [ ] 随机资料、配件

## 二、交付条款
- [ ] 交付时间（具体日期/条件）
- [ ] 交付地点
- [ ] 运输方式及费用承担
- [ ] 风险转移时点
- [ ] 所有权保留条款

## 三、验收条款
- [ ] 验收标准
- [ ] 验收期限
- [ ] 验收异议期
- [ ] 不合格处理
- [ ] 退换货条件

## 四、付款条款
- [ ] 预付款比例（建议≤30%）
- [ ] 货到付款条件
- [ ] 质保金比例（建议5-10%）
- [ ] 质保期（与法定期限比较）

## 五、售后服务
- [ ] 质保期限
- [ ] 质保范围
- [ ] 维修响应时间
- [ ] 备品备件供应

## 六、特别风险提示
1. 卖方风险：
   - 买方付款能力
   - 验收标准模糊
   - 质保期过长

2. 买方风险：
   - 预付款过高
   - 验收期过短
   - 退换货条件苛刻
   - 质保责任不清

## 七、法律依据
- 《民法典》合同编第九章买卖合同
- 《产品质量法》
- 《消费者权益保护法》（如适用）
```

### 2.2 借款合同审查要点
```
请审查借款合同，重点关注：

【合同信息】
- 出借人：{lender}
- 借款人：{borrower}
- 借款金额：{amount}
- 借款用途：{purpose}

## 一、借款条款
- [ ] 借款金额（大小写一致）
- [ ] 借款币种
- [ ] 借款用途（明确具体）
- [ ] 用途限制（不得挪用）

## 二、利息条款（重点审查）
- [ ] 利率标准
  - 年利率 ≤ LPR的4倍（目前约14%）
  - 超过部分无效
- [ ] 计息方式（年/月/日）
- [ ] 利息支付时间
- [ ] 复利约定（原则上禁止）
- [ ] 逾期利率（不超过合同利率）

## 三、期限条款
- [ ] 借款期限（起止日期）
- [ ] 还款方式（到期/分期）
- [ ] 提前还款约定
- [ ] 展期条件

## 四、担保条款
- [ ] 担保方式（保证/抵押/质押）
- [ ] 担保人主体资格
- [ ] 担保范围
- [ ] 担保期间
- [ ] 抵押物/质押物清单
- [ ] 担保登记约定

## 五、放款与还款
- [ ] 放款条件
- [ ] 收款账户
- [ ] 还款账户
- [ ] 还款顺序（本金/利息/费用）

## 六、特别风险提示
1. 出借人风险：
   - 借款人资信
   - 担保有效性
   - 抵押登记办理
   - 诉讼时效

2. 借款人风险：
   - 利率过高
   - 砍头息（禁止）
   - 违约金叠加
   - 格式条款

## 七、合规审查
- [ ] 非职业放贷（无放贷资格不得经常放贷）
- [ ] 非套路贷（无虚增债务、恶意违约）
- [ ] 非非法集资
- [ ] 资金来源合法

## 八、法律依据
- 《民法典》合同编第十二章借款合同
- 《最高人民法院关于审理民间借贷案件适用法律若干问题的规定》
- 《九民纪要》关于民间借贷部分
```

### 2.3 租赁合同审查要点
```
请审查租赁合同，重点关注：

【合同信息】
- 出租人：{lessor}
- 承租人：{lessee}
- 租赁物：{leased_property}
- 租赁期限：{term}

## 一、租赁物条款
- [ ] 租赁物描述（位置、面积、状况）
- [ ] 权属证明（房产证/所有权证）
- [ ] 共有权人同意（如适用）
- [ ] 抵押情况（如有，需抵押权人同意）
- [ ] 附属设施清单

## 二、租赁期限
- [ ] 起止日期明确
- [ ] 不得超过20年（超过部分无效）
- [ ] 续租约定
- [ ] 优先承租权（如适用）

## 三、租金条款
- [ ] 租金标准
- [ ] 支付方式（月付/季付/年付）
- [ ] 支付时间
- [ ] 收款账户
- [ ] 租金调整机制（如适用）

## 四、押金条款
- [ ] 押金金额（通常1-3个月租金）
- [ ] 押金退还条件
- [ ] 押金退还时间
- [ ] 押金抵扣情形

## 五、使用与维护
- [ ] 使用范围限制
- [ ] 装修约定（是否允许、恢复原状）
- [ ] 维修责任划分
- [ ] 物业费、水电费承担

## 六、合同解除
- [ ] 约定解除条件
- [ ] 法定解除条件
- [ ] 解除通知期限
- [ ] 解除后处理（返还、结算）

## 七、特别风险提示
1. 出租人风险：
   - 承租人资信
   - 租金拖欠
   - 房屋损坏
   - 非法用途

2. 承租人风险：
   - 权属瑕疵
   - 拆迁风险
   - 提前解约
   - 押金不退

## 八、法律依据
- 《民法典》合同编第十四章租赁合同
- 《商品房屋租赁管理办法》
- 《最高人民法院关于审理城镇房屋租赁合同纠纷案件具体应用法律若干问题的解释》
```

### 2.4 劳动合同审查要点
```
请审查劳动合同，重点关注：

【合同信息】
- 用人单位：{employer}
- 劳动者：{employee}
- 岗位：{position}
- 工作地点：{work_location}

## 一、必备条款（《劳动合同法》第17条）
- [ ] 用人单位信息
- [ ] 劳动者信息
- [ ] 劳动合同期限
- [ ] 工作内容和工作地点
- [ ] 工作时间和休息休假
- [ ] 劳动报酬
- [ ] 社会保险
- [ ] 劳动保护、劳动条件和职业危害防护

## 二、合同期限
- [ ] 固定期限（起止日期）
- [ ] 无固定期限（法定情形）
- [ ] 以完成一定工作任务为期限
- [ ] 试用期（不得超过法定期限）

## 三、工作内容
- [ ] 岗位名称
- [ ] 工作职责
- [ ] 工作地点
- [ ] 调岗约定（合理性审查）

## 四、劳动报酬
- [ ] 工资标准（不低于最低工资）
- [ ] 支付方式
- [ ] 支付时间（至少每月一次）
- [ ] 加班工资计算基数
- [ ] 奖金、津贴约定

## 五、工作时间
- [ ] 标准工时/综合计算工时/不定时
- [ ] 加班审批程序
- [ ] 加班工资标准

## 六、社会保险
- [ ] 五险缴纳约定
- [ ] 公积金缴纳约定
- [ ] 缴纳基数
- [ ] 缴纳时间

## 七、特殊条款审查
- [ ] 保密条款（范围、期限合理）
- [ ] 竞业限制（限于高管/高技/保密人员）
  - 期限≤2年
  - 需支付补偿金
  - 范围合理
- [ ] 服务期（限于专项培训）
- [ ] 违约金（仅限服务期和竞业限制）

## 八、禁止条款
- [ ] 不得约定劳动者承担违约金（除服务期、竞业限制）
- [ ] 不得约定"工伤自负"
- [ ] 不得约定"不缴社保"
- [ ] 不得约定"无条件调岗"

## 九、特别风险提示
1. 用人单位风险：
   - 未签书面合同（双倍工资）
   - 违法解除（2N赔偿）
   - 未缴社保（补缴+滞纳金）
   - 加班工资争议

2. 劳动者风险：
   - 试用期过长
   - 工资约定不明
   - 竞业限制范围过宽
   - 违约金过高

## 十、法律依据
- 《劳动合同法》
- 《劳动合同法实施条例》
- 《最高人民法院关于审理劳动争议案件适用法律问题的解释（一）》
```

```
请审查保密条款：

- [ ] 保密措施要求

- [ ] 使用限制

## 三、保密期限
- [ ] 合同期内
- [ ] 合同终止后（通常2-5年）

## 四、例外情形
- [ ] 已公开信息
- [ ] 合法取得
```

## 四、合同审查工作流

### 4.1 快速审查流程
```
适用于简单、金额小、风险低的合同

时间：30分钟内
步骤：
1. 主体资格审查（5分钟）
2. 核心条款审查（15分钟）
   - 标的
   - 价款
   - 履行
   - 违约
   - 争议解决
3. 风险提示（5分钟）
4. 修改建议（5分钟）
```

### 4.2 标准审查流程
```
适用于一般合同

时间：2-4小时
步骤：
1. 主体资格审查
2. 形式要件审查
3. 通用条款审查
4. 特殊条款审查
5. 风险识别
6. 修改建议
7. 出具审查意见
```

### 4.3 深度审查流程
```
适用于重大、复杂、高风险合同

时间：1-3天
步骤：
1. 尽职调查（对方资信、履约能力）
2. 法律研究（特殊法律问题）
3. 案例检索（类似合同纠纷）
4. 谈判参与（条款协商）
5. 多轮审查
6. 出具详细审查报告
7. 风险提示与应对建议
```
