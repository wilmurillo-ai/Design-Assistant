## 行业特殊规定

### 商品房预售条件

| 条件 | 要求 | 说明 |
|-----|------|------|
| 土地使用权 | 已取得土地使用权证书 | 无抵押或已解押 |
| 建设工程规划 | 已取得建设工程规划许可证 | 符合规划条件 |
| 施工许可 | 已取得建筑工程施工许可证 | 可以合法开工 |
| 资金投入 | 已完成投资≥25% | 证明资金实力 |
| 施工进度 | 已确定施工进度和竣工日期 | 可以按期交付 |

### 房地产调控政策

**限购政策**（因城施策）：
- 户籍限制
- 社保/个税年限要求
- 已有住房套数限制
- 离婚购房限制

**限贷政策**：
- 首套房首付比例（20-30%）
- 二套房首付比例（40-80%）
- 贷款利率（LPR+基点）
- 三套房停贷

**限售政策**：
- 取得产权证满2/3/5年方可转让
- 打击投机炒房

**价格管控**：
- 预售价格备案
- 涨价幅度限制
- 一房一价公示

### 工程款优先受偿权

**行使条件**：
- 发包人未按约定支付工程款
- 催告后合理期限内仍未支付
- 建设工程质量合格

**优先范围**：
- 工作人员报酬
- 材料款等实际支出
- 不包括发包人违约损失

**行使期限**：
- 自发包人应当给付工程款之日起18个月

**优先顺位**：
- 优于抵押权和其他债权
- 但劣后于消费者购房款

## 合同模板精选

### 模板1：商品房买卖合同（预售）

```
核心条款：

第一条 项目建设依据
- 土地使用权证号：__________
- 土地使用权取得方式：出让/转让
- 规划用途：住宅/商业/办公
- 土地使用年限：____年

第二条 商品房销售依据
- 预售许可证号：__________
- 批准预售机关：__________
- 批准预售日期：____年____月____日

第三条 商品房基本情况
- 坐落位置：__________
- 楼号/单元/房号：__________
- 建筑面积：____平方米（套内____+公摊____）
- 户型：____室____厅____卫
- 层高：____米
- 装修标准：毛坯/精装

第四条 计价方式与价款
- 单价：____元/平方米
- 总价款：____元（大写：__________）
- 付款方式：
  * 一次性付款
  * 分期付款
  * 按揭贷款（首付____%，贷款____%）

第五条 付款时间与方式
- 定金：____元（签约时转为房款）
- 首付款：____元，于____年____月____日前支付
- 贷款金额：____元，于签约后____日内办理贷款手续
- 尾款：____元，于交房时支付

第六条 面积差异处理
- 面积误差比绝对值在3%以内（含3%）：据实结算
- 面积误差比绝对值超出3%：
  * 买受人选择退房：开发商退款+利息
  * 买受人选择继续履行：
    - 产权登记面积大于合同约定：3%以内补款，超出3%部分开发商承担
    - 产权登记面积小于合同约定：3%以内退款+利息，超出3%部分双倍退款

第七条 交付条件与期限
- 交付时间：____年____月____日前
- 交付条件：
  * 取得建设工程竣工验收备案证明文件
  * 取得房屋测绘报告
  * 住宅质量保证书、住宅使用说明书
- 延期交付：按日支付已付款____%的违约金
- 逾期超过____日，买受人有权解除合同

第八条 交接手续
- 书面通知送达
- 查验房屋
- 签署交接单
- 钥匙移交
- 资料移交

第九条 产权登记
- 初始登记：开发商于交房后____日内办理
- 转移登记：开发商于交房后____日内协助买受人办理
- 逾期办证违约金：按日支付已付款____%

第十条 质量责任
- 地基基础和主体结构质量：终身保修
- 防水工程：____年
- 供热/供冷系统：____个采暖/供冷期
- 电气管线、给排水管道、设备安装：____年
- 装修工程：____年

第十一条 违约责任
- 买受人逾期付款：按日支付逾期应付款____%的违约金
- 开发商逾期交房：按日支付已付款____%的违约金
- 开发商逾期办证：按日支付已付款____%的违约金
- 开发商擅自变更规划：买受人有权退房
```

### 模板2：建设工程施工合同

```
核心条款：

第一条 工程概况
- 工程名称：__________
- 工程地点：__________
- 工程内容：__________
- 工程规模：建筑面积____平方米
- 工程立项批准文号：__________

第二条 合同工期
- 开工日期：____年____月____日
- 竣工日期：____年____月____日
- 合同工期总日历天数：____天
- 工期延误：
  * 承包人原因：按日支付合同价款____%的违约金
  * 发包人原因：顺延工期+赔偿损失

第三条 质量标准
- 工程质量标准：合格/优良
- 质量奖项要求：鲁班奖/省优/市优（可选）
- 不合格处理：返工费用由承包人承担

第四条 签约合同价与合同价格形式
- 合同总价：____元（大写：__________）
- 价格形式：
  * 固定总价
  * 固定单价
  * 可调价格
  * 成本加酬金

第五条 工程款支付
- 预付款：合同价款的____%，于开工前____日支付
- 进度款：按月/按节点支付，支付比例为已完工程量的____%
- 结算款：竣工验收合格后____日内支付至结算价的____%
- 质保金：结算价的____%，质保期满后____日内退还

第六条 变更与签证
- 变更程序：发包人书面指令→承包人报价→发包人确认→实施
- 变更价款确定：
  * 已有适用单价：按单价
  * 类似单价：参照调整
  * 无适用单价：协商确定
- 签证时效：变更发生后____日内提出

第七条 工程索赔
- 索赔程序：
  * 索赔事件发生后____日内提交索赔意向
  * 索赔意向发出后____日内提交索赔报告
  * 发包人____日内答复
- 索赔内容：工期顺延+费用补偿+利润

第八条 竣工验收
- 竣工验收条件
- 竣工验收程序
- 竣工验收不合格处理
- 工程移交
- 竣工资料移交

第九条 缺陷责任与保修
- 缺陷责任期：____个月（最长不超过24个月）
- 保修期：按《建设工程质量管理条例》
- 保修责任
- 维修通知与响应
- 维修费用承担

第十条 违约责任
- 发包人违约：
  * 未按期支付工程款
  * 未按期提供图纸/场地
  * 擅自使用未验收工程
- 承包人违约：
  * 未按期完工
  * 工程质量不合格
  * 转包/违法分包
```
