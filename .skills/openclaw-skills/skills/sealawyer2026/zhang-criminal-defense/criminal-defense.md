# 刑事辩护专用提示词库
# 针对刑事案件各阶段的AI辅助提示词

---

## 一、接待咨询阶段

### 1.1 初步案情分析
```
你是一名经验丰富的刑事辩护律师。请根据以下当事人陈述，进行初步法律分析：

【当事人陈述】
{client_statement}

请提供：
1. 涉嫌罪名分析（可能涉及的罪名及法律依据）
2. 案件阶段判断（侦查/审查起诉/审判）
3. 紧急事项提示（是否需要立即采取行动）
4. 初步辩护思路（无罪/罪轻/程序辩护方向）
5. 下一步工作建议

注意：
- 保持客观中立，不做胜诉承诺
- 提示当事人如实陈述的重要性
- 告知律师保密义务和执业规范
```

### 1.2 委托风险评估
```
请对以下刑事案件进行接案风险评估：

【案件基本信息】
{case_info}

请分析：
1. 利益冲突检查（是否与现有客户冲突）
2. 案件复杂程度评估
3. 时间精力投入预估
4. 收费合理性建议
5. 风险告知要点
6. 是否建议接案的结论及理由
```

## 二、侦查阶段

### 2.1 会见提纲生成
```
请为以下刑事案件生成首次会见提纲：

【案件背景】
- 涉嫌罪名：{charge}
- 强制措施：{measure}（拘留/逮捕/取保）
- 羁押场所：{location}
- 案件阶段：侦查阶段

会见提纲应包括：
1. 身份确认与委托关系建立
2. 案件事实了解（时间线、参与程度、主观明知）
3. 证据情况询问（供述、物证、书证、证人）
4. 程序权利告知（回避、申诉、控告）
5. 生活关怀与心理安抚
6. 下一步工作安排

注意事项：
- 避免诱导性提问
- 记录关键时间点和细节
- 关注是否存在刑讯逼供
```

### 2.2 取保候审申请
```
请撰写取保候审申请书：

【案件信息】
- 犯罪嫌疑人：{suspect_name}
- 涉嫌罪名：{charge}
- 羁押时间：{detention_date}
- 案件机关：{authority}

【申请理由要点】
{reason_points}

申请书应包含：
1. 申请人信息
2. 申请事项
3. 事实与理由（逐条阐述符合取保条件）
4. 保证方式（人保/财保）
5. 法律依据
6. 此致机关

重点强调：
- 不具有社会危险性
- 不妨碍侦查
- 符合《刑事诉讼法》第67条规定的情形
```

### 2.3 不予批捕法律意见书
```
请撰写不予批准逮捕法律意见书：

【案件信息】
- 提请批捕机关：{authority}
- 犯罪嫌疑人：{suspect_name}
- 涉嫌罪名：{charge}
- 拘留时间：{detention_date}

【辩护要点】
{defense_points}

意见书结构：
1. 律师身份及接受委托情况
2. 案件基本事实（律师了解的部分）
3. 不予批捕的法律依据和理由
   - 不构成犯罪
   - 证据不足
   - 无逮捕必要
   - 符合取保条件
4. 类案参考（如有）
5. 结论性意见

引用依据：
- 《刑事诉讼法》第81条（批捕条件）
- 《刑事诉讼法》第90条（不批捕决定）
- 最高检《关于逮捕社会危险性条件若干问题的规定》
```

## 三、审查起诉阶段

### 3.1 阅卷要点清单
```
请生成审查起诉阶段阅卷要点清单：

【案件信息】
- 移送审查起诉机关：{authority}
- 涉嫌罪名：{charge}
- 卷宗数量：{volume_count}册

阅卷要点：

## 一、程序性审查
- [ ] 管辖是否正确
- [ ] 回避是否依法进行
- [ ] 强制措施是否超期
- [ ] 讯问是否有合法录音录像
- [ ] 搜查、扣押是否有见证人
- [ ] 鉴定意见是否告知当事人

## 二、证据审查
- [ ] 物证：来源、保管、鉴定链条
- [ ] 书证：原件核对、复制件说明
- [ ] 证人证言：利害关系、前后矛盾
- [ ] 被害人陈述：与其他证据印证
- [ ] 犯罪嫌疑人供述：自愿性、一致性
- [ ] 鉴定意见：资质、方法、结论
- [ ] 勘验检查笔录：见证人、签名
- [ ] 视听资料：完整性、真实性
- [ ] 电子数据：提取程序、完整性校验

## 三、事实认定
- [ ] 犯罪时间、地点、手段
- [ ] 犯罪数额/数量/次数
- [ ] 主从犯认定
- [ ] 既遂未遂认定
- [ ] 自首立功情节

## 四、法律适用
- [ ] 罪名认定是否准确
- [ ] 此罪彼罪区分
- [ ] 一罪数罪认定
- [ ] 追诉时效

## 五、量刑情节
- [ ] 法定从重/从轻/减轻情节
- [ ] 酌定情节
- [ ] 类案量刑参考
```

### 3.2 不起诉法律意见书
```
请撰写不起诉法律意见书：

【案件信息】
- 审查起诉机关：{authority}
- 犯罪嫌疑人：{suspect_name}
- 涉嫌罪名：{charge}
- 移送审查起诉日期：{transfer_date}

【辩护方向】
{defense_direction}（法定不起诉/酌定不起诉/证据不足不起诉）

【主要理由】
{main_reasons}

意见书内容：
1. 律师基本情况
2. 案件事实概述
3. 不起诉的法律依据
   - 法定不起诉：《刑事诉讼法》第177条第1款
   - 酌定不起诉：《刑事诉讼法》第177条第2款
   - 证据不足不起诉：《刑事诉讼法》第175条第4款
4. 详细论证
5. 类案支持（如有）
6. 结论请求

附件建议：
- 证据目录及质证意见
- 类案检索报告
- 相关证明材料
```

### 3.3 量刑协商建议书
```
请撰写量刑协商建议书（认罪认罚从宽案件）：

【案件信息】
- 犯罪嫌疑人：{suspect_name}
- 涉嫌罪名：{charge}
- 犯罪事实：{facts}
- 已退赃/赔偿：{restitution}
- 取得谅解：{forgiveness}

【协商要点】
{negotiation_points}

建议书结构：
1. 犯罪嫌疑人认罪认罚态度
2. 犯罪事实确认（无异议部分）
3. 量刑情节梳理
   - 法定从轻/减轻情节
   - 酌定从轻情节
4. 建议量刑幅度
   - 刑种建议
   - 刑期建议
   - 缓刑适用建议
5. 法律依据
6. 类案量刑参考

注意事项：
- 充分告知认罪认罚法律后果
- 确保犯罪嫌疑人自愿认罪
- 对量刑建议有异议的明确记录
```

## 四、审判阶段

### 4.1 一审辩护词
```
请撰写一审辩护词：

【案件信息】
- 审理法院：{court}
- 案号：{case_number}
- 被告人：{defendant_name}
- 涉嫌罪名：{charge}
- 起诉书指控：{indictment_allegations}

【辩护策略】
{defense_strategy}（无罪辩护/罪轻辩护/程序辩护）

【主要辩护观点】
{main_arguments}

辩护词结构：

## 首部
- 律师事务所及被告人信息
- 辩护人在诉讼中的地位

## 正文

### 一、关于案件事实部分
（针对指控事实提出异议或确认）

### 二、关于证据部分
（质证意见汇总，证据链断裂点）

### 三、关于法律适用部分
（罪名认定异议、此罪彼罪）

### 四、关于量刑部分
（从轻、减轻情节，建议刑期）

### 五、关于程序部分
（如有程序违法，影响公正审判）

## 尾部
- 总结性辩护意见
- 具体请求事项
- 辩护人签名、日期

写作要求：
- 逻辑严密，论证充分
- 法理与情理结合
- 语言规范，避免情绪化
- 引用法条准确具体
```

### 4.2 质证意见
```
请针对以下证据撰写质证意见：

【证据信息】
- 证据名称：{evidence_name}
- 证据类型：{evidence_type}
- 证据来源：{evidence_source}
- 证明目的：{probative_purpose}

【质证要点】
{examination_points}

质证意见格式：

## 对{证据名称}的质证意见

### 一、证据合法性
（收集程序、主体资格、形式要件）

### 二、证据真实性
（内容真伪、有无篡改、与其他证据印证）

### 三、证据关联性
（与待证事实的关联程度）

### 四、证明力
（证据效力大小、是否达到证明标准）

### 五、质证结论
（是否采信、证明目的能否实现）

法律依据：
- 《刑事诉讼法》第50条（证据种类）
- 《刑事诉讼法》第56条（非法证据排除）
- 《刑诉解释》第四章（证据审查判断）
```

### 4.3 上诉状
```
请撰写刑事上诉状：

【案件信息】
- 原审法院：{original_court}
- 案号：{case_number}
- 上诉人：{appellant_name}
- 原审判决：{judgment_result}
- 收到判决日期：{judgment_date}

【上诉理由】
{appeal_reasons}

【请求事项】
{requested_relief}

上诉状结构：
1. 上诉人、辩护人基本信息
2. 原审判决情况
3. 上诉请求
4. 上诉理由
   - 事实认定错误
   - 证据采信不当
   - 法律适用错误
   - 量刑不当
   - 程序违法
5. 结论
6. 此致法院
7. 上诉人签名、日期

注意事项：
- 上诉期10日（判决）/5日（裁定）
- 明确具体的上诉请求
- 针对原审错误逐条阐述
- 新证据一并提交
```

## 五、特殊程序

### 5.1 死刑复核辩护词
```
请撰写死刑复核阶段辩护词：

【案件信息】
- 复核法院：最高人民法院
- 被告人：{defendant_name}
- 罪名：{charge}
- 原审判决：死刑立即执行/死刑缓期执行

【辩护重点】
{defense_focus}

辩护词要点：
1. 案件事实部分（是否清楚、证据是否确实充分）
2. 量刑部分（是否必须立即执行/是否符合缓刑条件）
3. 程序部分（是否保障辩护权）
4. 刑事政策（宽严相济、少杀慎杀）
5. 人道主义关怀（家庭情况、改造可能）

特别强调：
- 死刑案件证据标准（唯一性、排他性）
- 死刑立即执行的适用条件
- 死缓适用的优先考虑
- 被害人过错（如有）
```

### 5.2 申诉书
```
请撰写刑事申诉书：

【案件信息】
- 原审法院：{original_court}
- 案号：{case_number}
- 申诉人：{petitioner_name}
- 原审裁判结果：{original_judgment}
- 裁判生效日期：{effective_date}

【申诉理由】
{petition_reasons}

申诉书结构：
1. 申诉人基本信息
2. 原审裁判情况
3. 申诉请求
4. 申诉理由
   - 有新证据证明原判错误
   - 证据不确实、不充分
   - 主要证据有矛盾
   - 适用法律错误
   - 程序严重违法
   - 审判人员有违法行为
5. 证据目录
6. 此致法院/检察院

注意事项：
- 申诉不停止执行
- 2年内提出（特殊情形除外）
- 向终审法院或其上级法院提出
- 也可向检察院申诉
```

### 案例1：{case_name}

## 使用说明

### 单阶段使用
针对案件所处阶段，选择对应提示词

### 全流程使用
```
请按照以下流程处理本案：
1. 【接待咨询】使用初步案情分析
2. 【侦查阶段】使用会见提纲 + 取保申请
3. 【审查起诉】使用阅卷清单 + 不起诉意见
4. 【审判阶段】使用辩护词 + 质证意见
5. 【特殊程序】根据需要选择
```

### 注意事项
- 所有文书需根据具体案情调整
- 注意保密，不得泄露当事人信息
- 及时更新法律法规变化
- 结合最新司法解释和判例
