# Token审计员

[![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)](https://github.com/sealawyer2026/skill-token-auditor)

> Token消费诊断与优化专家

## 🎯 简介

Token审计员分析AI Token消费模式，检测浪费和异常，生成优化建议报告，帮助企业节省成本。

## ✨ 功能特性

- 📊 **消费分析** - 模式识别与趋势分析
- ⚠️ **异常检测** - 自动发现消费突增
- 💸 **浪费识别** - 4种浪费类型自动检测
- 💡 **优化建议** - 具体可执行的节省方案
- 📈 **健康评分** - 0-100分直观评估

## 🚀 快速开始

```bash
# 安装
pip install -r requirements.txt

# 使用示例数据审计
python main.py audit --sample --days 30

# 导出详细报告
python main.py audit --sample --export report.md

# 生成JSON格式报告
python main.py export --format json --output report.json
```

## 📊 检测的浪费类型

| 类型 | 说明 | 节省潜力 | 检测方式 |
|------|------|----------|----------|
| 提示词过长 | 提示词>2000字符 | 25-30% | 长度分析 |
| 模型选择不当 | 简单任务使用GPT-4等大模型 | 60-70% | 任务类型vs模型成本 |
| 重复请求 | 相同输入多次调用 | 15-20% | 请求去重 |
| 高温度参数 | 输出Token远超输入 | 5-10% | I/O比例分析 |

## 🏥 健康评分

| 分数 | 等级 | 说明 |
|------|------|------|
| 80-100 | 🟢 优秀 | 消费健康，优化空间小 |
| 60-79 | 🟡 良好 | 有优化空间，建议关注 |
| 40-59 | 🟠 一般 | 存在明显浪费，需要优化 |
| 0-39 | 🔴 较差 | 严重浪费，立即行动 |

## 🔗 生态集成

- **Token Master** - 压缩提示词，解决"提示词过长"问题

- **Token预算管家** - 设置预算上限，防止异常消费
- **Token智能监控** - 实时监控，及时告警
