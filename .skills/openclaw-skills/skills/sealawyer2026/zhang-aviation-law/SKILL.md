## 典型案例与判例

### 案例1：马航MH370失联事件
**法律问题**：
- 承运人责任限额：约17.5万SDR/人（约150万元人民币）
- 能否突破责任限额：证明承运人故意或重大过失
- 失踪宣告：民法典第46条，满4年可宣告死亡
- 赔偿标准：参照国内空难标准，约100-200万元

### 案例2：伊春空难（2010）
**法律问题**：
- 机长责任：重大飞行事故罪
- 河南航空破产重组
- 赔偿标准：每位遇难者96万元
- 保险赔付：机身险+责任险

### 案例3：航班超售纠纷
**典型判决**：
- 美联航强制拖拽乘客事件（2017）
- 国内法院判例：超售需提前告知，否则构成欺诈
- 赔偿标准：改签费用+延误损失+精神损害

## 赔偿标准参考

### 国内航空运输赔偿

| 损害类型 | 赔偿标准 | 法律依据 |
|---------|---------|---------|
| 旅客死亡 | 按实际损失计算，无上限 | 民用航空法第128条（已修改） |
| 旅客受伤 | 医疗费+误工费+残疾赔偿金 | 民法典侵权责任编 |
| 行李丢失 | 每公斤100元人民币 | 国内航空运输承运人赔偿责任限额规定 |
| 航班延误 | 无强制赔偿，依运输条件 | 民航局指导意见 |
| 货物损坏 | 每公斤100元人民币 | 同上 |

### 国际航空运输赔偿（蒙特利尔公约）

| 损害类型 | 赔偿限额 | 说明 |
|---------|---------|------|
| 旅客死亡/受伤 | 128,821 SDR（约123万元人民币） | 严格责任，无过错 |
| 延误损失 | 5,346 SDR（约5.1万元人民币） | 每位旅客 |
| 行李/货物 | 1,288 SDR/件（约1.2万元） | 每公斤约30SDR |
| 突破限额 | 需证明承运人故意或重大过失 | 举证责任倒置 |

> SDR（特别提款权）汇率每日变动，请以实际汇率计算

## 合同模板精选

### 模板1：航空运输总条件（乘客版）

```
关键条款：

第X条 航班超售
1. 超售定义：为避免座位虚耗，航班可能超售
2. 自愿放弃：提供补偿+改签服务，乘客自愿放弃座位
3. 非自愿拒载：
   - 优先顺序：特殊旅客→高端旅客→普通旅客
   - 补偿标准：延误4小时内票价30%，4小时以上50%
   - 食宿安排：延误超4小时提供免费食宿

第X条 航班延误/取消
1. 延误定义：实际到港时间晚于计划15分钟以上
2. 免责事由：天气、空中交通管制、突发事件
3. 补偿标准：
   - 延误2-4小时：提供饮料、餐食
   - 延误4-8小时：现金补偿200元或等值服务
   - 延误超8小时：现金补偿400元+免费改签
4. 取消处理：全额退票或免费改签

第X条 行李运输
1. 免费行李额：头等舱40kg，公务舱30kg，经济舱20kg
2. 逾重行李费：每公斤按经济舱全价票1.5%计算
3. 声明价值：可办理声明价值，最高8000元
4. 丢失赔偿：
   - 托运行李：每公斤100元
   - 随身携带：最高3000元
   - 证明实际损失的，按实际损失赔偿
```

## 使用示例

### 示例1：航空事故赔偿谈判

**案情**：国内航班紧急迫降，乘客受伤

**谈判策略**：
1. **责任分析**
   - 是否属于承运人责任（机械故障vs天气原因）
   - 适用国内法还是国际公约
   - 是否存在第三方责任

2. **损失计算**
   - 医疗费：实际票据
   - 误工费：收入证明
   - 残疾赔偿金：伤残等级×当地标准
   - 精神损害抚慰金：通常1-10万元

3. **谈判要点**
   - 强调承运人安全保障义务
   - 援引类似案例赔偿金额
   - 媒体曝光压力
   - 快速解决vs诉讼成本

### 示例2：飞机租赁退租争议

**争议焦点**：退租时飞机状态不符合要求

**处理方案**：
1. **合同解释**
   - 退租条件的具体约定
   - "正常磨损"的定义
   - 大修记录的完整性

2. **技术评估**
   - 委托第三方技术评估
   - 维修成本估算
   - 残值影响评估

3. **争议解决**
   - 协商调解
   - 专家仲裁
   - 诉讼保全（扣押保证金）
