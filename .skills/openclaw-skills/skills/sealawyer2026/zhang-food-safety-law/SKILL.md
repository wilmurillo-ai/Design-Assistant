## 食品安全法律责任

### 民事责任

| 违法行为 | 赔偿标准 | 法律依据 |
|---------|---------|---------|
| 生产不符合安全标准食品 | 价款10倍或损失3倍，不足1000元按1000元 | 食品安全法第148条 |
| 经营明知不符合安全标准食品 | 同上 | 同上 |
| 欺诈 | 价款3倍，不足500元按500元 | 消费者权益保护法第55条 |
| 人身损害 | 医疗费、误工费、残疾赔偿金、死亡赔偿金 | 民法典侵权责任编 |

### 行政责任

| 违法行为 | 处罚措施 | 法律依据 |
|---------|---------|---------|
| 无证生产经营 | 没收违法所得、罚款（货值金额10-20倍） | 食品安全法第122条 |
| 用非食品原料生产 | 没收、罚款（货值金额15-30倍）、吊销许可证 | 食品安全法第123条 |
| 超范围使用添加剂 | 没收、罚款（货值金额5-10倍） | 食品安全法第124条 |
| 标签不符合规定 | 没收、罚款（5000-5万元） | 食品安全法第125条 |
| 未按规定查验 | 警告、罚款（5000-5万元） | 食品安全法第126条 |

### 刑事责任

| 罪名 | 情形 | 量刑 |
|-----|------|------|
| 生产、销售不符合安全标准的食品罪 | 足以造成严重食物中毒或其他严重食源性疾病 | 3年以下有期徒刑或拘役，并处罚金 |
| 生产、销售有毒、有害食品罪 | 掺入有毒有害非食品原料 | 5年以下有期徒刑，并处罚金；致人死亡或特别严重情节，10年以上有期徒刑、无期徒刑或死刑 |
| 生产、销售伪劣产品罪 | 销售金额5万元以上 | 根据销售金额量刑 |

## 合规要点

### 食品生产合规清单

- [ ] 取得食品生产许可证（SC证）
- [ ] 建立食品安全管理制度
- [ ] 配备食品安全管理人员
- [ ] 建立进货查验记录制度
- [ ] 建立生产过程控制制度
- [ ] 建立出厂检验记录制度
- [ ] 建立不合格品管理制度
- [ ] 建立食品安全自查制度
- [ ] 建立从业人员健康管理制度
- [ ] 建立食品安全追溯体系
- [ ] 产品标签符合GB 7718要求
- [ ] 产品标准符合食品安全国家标准

### 食品经营合规清单

- [ ] 取得食品经营许可证
- [ ] 经营场所符合卫生要求
- [ ] 配备与经营规模相适应的设备设施
- [ ] 建立食品安全管理制度
- [ ] 建立进货查验记录制度
- [ ] 建立销售记录制度（批发企业）
- [ ] 从业人员持有效健康证明
- [ ] 定期开展食品安全自查

## 合同模板

### 食品委托生产合同

```
核心条款：

第一条 委托生产产品
- 产品名称：__________
- 产品规格：__________
- 执行标准：__________
- 生产数量：__________

第二条 双方责任
委托方责任：
- 提供合法有效的产品配方、工艺
- 提供包装材料及标签样稿
- 负责产品销售及售后服务
- 承担产品质量安全主体责任

受托方责任：
- 按照委托方提供的配方工艺生产
- 确保生产过程符合食品安全要求
- 出厂检验合格
- 建立生产记录

第三条 质量要求
- 符合食品安全国家标准
- 符合产品执行标准
- 出厂检验合格
- 检验报告提供

第四条 标签标识
- 委托方名称地址
- 受托方名称地址
- 产品名称、规格、生产日期
- 保质期、配料表
- 生产许可证编号（受托方SC证）

第五条 知识产权
- 产品配方知识产权归委托方
- 不得将配方用于其他客户
- 保密义务

第六条 违约责任
- 质量不符合约定
- 未按期交货
- 知识产权侵权
```
