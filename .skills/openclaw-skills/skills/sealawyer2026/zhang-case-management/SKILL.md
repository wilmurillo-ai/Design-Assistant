# 九章-案件管理专家

> 案件管理大师，让办案井井有条

## 核心功能模块

### 📋 案件登记

#### 案件信息录入
- 案件基本信息
  * 案号/内部编号
  * 案由/案件类型
  * 收案日期
  * 案件来源
  * 紧急程度

- 当事人信息
  * 委托人信息（个人/企业）
  * 对方当事人信息
  * 第三人信息
  * 利益冲突检查

- 案件概况
  * 案件事实简述
  * 争议焦点
  * 诉讼请求/答辩要点
  * 标的金额
  * 管辖法院/仲裁机构

#### 案件分类
- **按案由分类**
  * 民事：合同、侵权、婚姻家事、劳动争议等
  * 刑事：各罪名分类
  * 行政：行政处罚、行政许可等
  * 仲裁：商事仲裁、劳动仲裁
  * 非诉：法律顾问、专项服务等

- **按阶段分类**
  * 咨询阶段
  * 委托洽谈
  * 立案准备
  * 一审
  * 二审
  * 再审
  * 执行
  * 结案归档

- **按优先级分类**
  * 紧急（24小时内响应）
  * 高（3日内处理）
  * 中（1周内处理）
  * 低（按计划处理）

#### 利益冲突审查
- 与现有客户冲突检查
- 与对方当事人冲突检查
- 与关联企业冲突检查
- 利益冲突豁免
- 回避处理

---

### 📅 进度管理

#### 时间节点管理
- **诉讼时效**
  * 起算点
  * 届满日期
  * 中断事由
  * 剩余天数提醒

- **程序期限**
  * 立案期限
  * 举证期限
  * 答辩期限
  * 上诉期限
  * 申请执行期限

- **关键节点**
  * 开庭日期
  * 证据交换日期
  * 调解日期
  * 宣判日期
  * 执行节点

#### 任务清单
- **标准任务模板**
  * 收案：接谈→利益冲突审查→委托手续→收费
  * 立案：准备材料→网上立案→现场提交→缴费→拿受理通知
  * 庭前：证据收集→证据整理→法律研究→代理词撰写
  * 庭审：出庭→举证质证→法庭辩论→庭后补充
  * 庭后：代理词补充→跟进判决→判后分析
  * 执行：申请执行→财产查控→执行措施→回款

- **自定义任务**
  * 任务名称
  * 负责人
  * 截止日期
  * 优先级
  * 完成状态
  * 备注

#### 里程碑
- 收案
- 立案成功
- 首次开庭
- 判决/调解/撤诉
- 结案
- 归档

## 案件管理流程

### 收案流程

```
1. 客户咨询
   ↓
2. 初步沟通，了解案情
   ↓
3. 利益冲突审查
   ↓
4. 报价与收费协商
   ↓
5. 签订委托合同
   ↓
6. 签署授权委托书
   ↓
7. 收取律师费
   ↓
8. 案件登记入系统
   ↓
9. 组建办案团队
   ↓
10. 制定办案计划
```

### 办案流程

```
立案阶段：
- 收集证据
- 准备起诉材料
- 网上立案/现场立案
- 缴纳诉讼费
- 拿受理通知书

庭前阶段：
- 证据交换
- 法律研究
- 撰写代理词
- 准备庭审提纲
- 与当事人沟通

庭审阶段：
- 出庭应诉
- 举证质证
- 法庭辩论
- 最后陈述
- 庭后补充

判后阶段：
- 领取判决
- 判后分析
- 上诉/申请执行
- 客户反馈
- 结案归档
```

## 使用示例

### 示例：民事案件管理

**案件信息**：
- 案由：买卖合同纠纷
- 标的：100万元
- 阶段：一审

**管理要点**：

1. **时间管理**
   - 诉讼时效：3年，剩余____天
   - 举证期限：收到举证通知后15日
   - 开庭日期：____年____月____日

2. **任务清单**
   - [ ] 收集合同、送货单、对账单
   - [ ] 整理证据清单
   - [ ] 撰写起诉状
   - [ ] 网上立案
   - [ ] 准备代理词
   - [ ] 庭前与当事人沟通

3. **文档管理**
   - 证据清单编号
   - 法律文书版本控制
   - 沟通记录存档

4. **客户沟通**
   - 每周进度汇报
   - 重要节点及时通知
   - 风险提示
