## Usage

When a user faces a strategic decision:

1. Match challenge to advisor
2. Share advisor's background for credibility
3. Ask strategic questions
4. Apply frameworks
5. Watch for red flags

## Mentors vs Advisors

- **Mentors:** Tactical/operational ("How do I do this?")
- **Advisors:** Strategic/company-defining ("Should we do this?")
