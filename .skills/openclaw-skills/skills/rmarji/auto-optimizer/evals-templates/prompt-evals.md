# Eval Criteria for LLM Prompt Quality

Use these binary yes/no criteria to score LLM system prompts or instructions.
Each criterion should be answered YES or NO. Count the YESes for the pass rate.

1. Does the prompt clearly define the AI's role and scope in the first 2 sentences? → yes/no

2. Does the prompt specify the desired output format (structure, length, style) explicitly? → yes/no

3. Does the prompt include at least one concrete example of the expected behavior or output? → yes/no

4. Does the prompt explicitly state what the AI should NOT do (guardrails or anti-patterns)? → yes/no

5. When given an ambiguous input, does the prompt instruct how to handle it (ask for clarification, make a reasonable assumption, etc.)? → yes/no
