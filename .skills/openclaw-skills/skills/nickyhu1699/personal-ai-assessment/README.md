# 个人AI能力测评工具

帮助个人科学评估AI能力，制定个性化提升路径。

## 快速开始

### 1. 运行测评

```bash
cd ~/.openclaw/skills/personal-ai-assessment/scripts
python3 assessment_tool.py
```

### 2. 在对话中使用

```
你："帮我测评一下我的AI能力"
小龙虾：运行测评工具，生成完整报告
```

## 功能特点

- ✅ **5大维度测评** - 知识、工具、实战、创新、伦理
- ✅ **5级能力等级** - 大师、专业、进阶、入门、小白
- ✅ **能力雷达图** - 可视化展示各维度分数
- ✅ **学习路径** - 个性化提升建议
- ✅ **详细报告** - Markdown格式，可导出

### 1. AI知识储备（0-100分）

### 3. AI实战应用（0-100分）

- AI效率提升

### 4. AI创新思维（0-100分）
- AI+行业创新

### 5. AI伦理意识（0-100分）
