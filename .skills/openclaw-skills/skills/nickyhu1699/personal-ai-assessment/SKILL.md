# 个人AI能力测评工具

帮助个人科学评估AI能力，制定个性化提升路径。

## 核心功能

### 1. 5大能力维度测评

#### 维度1：AI知识储备（0-100分）
- AI基础概念理解
- AI发展历史了解
- AI技术原理认知
- AI应用领域掌握

#### 维度2：AI工具使用（0-100分）
- 对话AI使用（ChatGPT、Claude等）
- 图片AI使用（Midjourney、DALL-E等）
- 写作AI使用（Notion AI、Jasper等）
- 办公AI使用（Copilot、Gamma等）

#### 维度3：AI实战应用（0-100分）
- 提示词编写能力
- AI工作流搭建
- AI解决实际问题
- AI效率提升

#### 维度4：AI创新思维（0-100分）
- AI+行业创新
- AI+业务创新
- AI商业模式
- AI未来发展

#### 维度5：AI伦理意识（0-100分）
- 数据安全意识
- AI偏见认知
- AI风险评估
- AI合规理解

### 2. 能力等级评定

| 等级 | 分数范围 | 特征 |
|------|---------|------|
| 🏆 大师级 | 450-500 | AI领域专家，能指导他人，能创新应用 |
| ⭐ 专业级 | 350-449 | 熟练使用AI，能解决复杂问题，能搭建工作流 |
| 📈 进阶级 | 250-349 | 掌握基础工具，能应用到工作，持续学习中 |
| 🌱 入门级 | 150-249 | 了解AI概念，会用1-2个工具，有学习意愿 |
| 📖 小白级 | 0-149 | 几乎不了解AI，需要系统学习 |

### 3. 测评形式

- **知识测试**：20题（单选10题 + 多选5题 + 判断5题）
- **场景测试**：10题（工作场景 + 生活场景 + 创新场景）
- **实战测试**：5题（提示词编写 + 工具选择 + 工作流设计）
- **总时长**：约15分钟

### 4. 输出报告

- 📊 能力雷达图（5维度可视化）
- 📈 能力总分（0-500分）
- 🏆 能力等级（5个等级）
- 💪 优势能力（TOP2维度）
- ⚠️ 待提升能力（BOTTOM2维度）
- 📚 学习路径（个性化推荐）
- 💡 提升建议（具体可执行）
