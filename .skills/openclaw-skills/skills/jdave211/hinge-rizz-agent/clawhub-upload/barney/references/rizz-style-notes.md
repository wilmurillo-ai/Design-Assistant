# Rizz Style Notes

Derived from public "rizz text" screenshots, opener roundups, and short-form flirting examples reviewed on March 4, 2026.

Patterns to keep:
- One sentence. Usually 5 to 14 words.
- One hook only: compliment, observation, or playful challenge.
- If the profile already gives a hook, use it directly instead of generic banter.
- Compliments should be short and specific, then pivot fast.
- Questions should be easy to answer in one text.

Patterns to avoid:
- Paragraphs
- Repeating their whole prompt back to them
- Corny STEM pickup lines
- Generic filler like "you seem fun"
- Double-question chains
