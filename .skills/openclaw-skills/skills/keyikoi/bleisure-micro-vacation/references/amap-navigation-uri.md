# 高德地图 · 路线直达链接（URI API）

官方文档：<https://lbs.amap.com/api/uri-api/guide/travel/route>

## 服务地址

```
https://uri.amap.com/navigation
```

## 步行/驾车等常用参数

| 参数 | 说明 |
|------|------|
| `from` | 起点：`经度,纬度[,名称]`，名称建议 URL 编码（UTF-8） |
| `to` | 终点：格式同 `from` |
| `mode` | `walk` 步行 · `car` 驾车 · `bus` 公交 · `ride` 骑行（骑行调起 App 有版本要求） |
| `src` | 来源标识，建议填写，如 `travel-evening-micro-vacation` |
| `callnative` | `1` 尝试调起高德 App（**仅移动端**有效）；`0` 仅网页。默认 `0` |

## 坐标系

必须使用 **GCJ-02（国测局 / 火星坐标）**，与高德一致。勿混用 WGS84 未偏移坐标（例如手机 raw GPS 未纠偏时不要用）。

## 坐标获取：优先高德 POI 搜索

### 最佳方式（有 API Key 时）

用 `references/amap-poi-search.md` 中的高德 POI 搜索接口分别查起点和终点，直接拿到 **GCJ-02 坐标**，填入 `from`/`to`。

这种方式拿到的坐标 **最准确**，因为来自高德自己的 POI 库，坐标系一致，无需额外转换。

```
起点：工作流第 2 步查用户位置时已拿到 location
终点：工作流第 8 步查推荐地点时已拿到 location
→ 直接拼链，起点终点都精准
```

### 回退方式（无 Key 时）

通过 WebSearch 查找 POI 坐标。**不允许**凭印象手写经纬度。

- 若检索到多个重名点，选 **与用户城市/区一致** 的一条。
- 实在无法核实：省略 `from` 让高德用当前位置，`to` 用搜到的近似坐标，并提示用户在 App 内确认。

## 输出中必须包含「坐标来源」

在高德链接下方增加一行：

```markdown
# 有 API Key 时（精准）：
**坐标来源：** 高德 POI 搜索「联发北秀观云」→「杭钢遗址公园」，GCJ-02。

# 无 Key 回退时：
**坐标来源：** 联网检索 POI「龙翔桥地铁站」与「断桥残雪」近似 GCJ-02；若与你位置不符，请在 App 内拖动起点。
```

用户未给精确酒店时，必须提示：**可在高德内将起点改为你的酒店**。

## 拼链步骤（给 Agent）

1. 根据用户起点、终点 POI **联网核对** 近似 GCJ-02 经纬度（保留小数点后 4～6 位即可）。
2. `from` / `to` 值为：`lon,lat` 或 `lon,lat,显示名`；**中文或空格**放入查询串时应对 **整个 `from`/`to` 参数值** 做 URL 编码（或至少对名称段编码），避免乱码。
3. 微度假默认 **`mode=walk`**；若本次动线以地铁为主可另给一条 `mode=bus`（可选）。**硬截止赴枢纽**时务必增加 **`mode=car` 或 `mode=bus`** 的枢纽链。
4. 默认输出 **一条** 观光步行链即可；带 `callnative=1` 时移动端易调起高德 App；PC 端通常仍打开网页导航。若用户明确在电脑端使用，可改 `callnative=0` 或说明「复制到手机打开」。

## 示例（结构示意，坐标需按实时检索替换）

```
https://uri.amap.com/navigation?from=116.4605%2C39.9087%2C%E5%9B%BD%E8%B4%B8&to=116.4653%2C39.9142%2C%E6%99%AF%E5%8D%8E%E5%8D%97%E8%A1%97&mode=walk&src=travel-evening-micro-vacation&callnative=1
```

（`%2C` 为逗号编码；亦可对整个 `from`/`to` 统一 encode。）

## 限制与产品说明

- **`via` 途径点**仅在 **驾车** `mode=car` 下有效，且最多 1 个；步行线多段可拆成 **多条链接**（如「酒店→打卡点」「打卡点→下一处」）。
- 微信内置浏览器等环境 **可能无法调起 App**，属平台限制；可提示用户「右上角浏览器打开」或复制链接到高德。
- 坐标与 POI 为近似时，在文案中加一句 **「以高德实际规划为准」**。
