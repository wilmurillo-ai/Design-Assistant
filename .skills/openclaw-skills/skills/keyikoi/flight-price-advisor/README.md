# Price Summary Skill

为普通用户生成简洁、易懂的航班价格总结和购买建议。

## 应用场景

当用户询问航班价格、购买时机时，使用此技能生成文字版的价格分析。

### 典型流程

```
用户询问：这个航班贵吗？/ 现在买划算吗？
    ↓
AI 调用价格数据 API
    ↓
AI 调用 price-summary 生成分析
    ↓
用户看到：简洁的价格总结 + 购买建议
```

## 快速开始

### 1. 数据输入

```javascript
const priceData = {
  currentPrice: 1299,
  priceHistory: [
    { date: "2026-01-01", price: 1450 },
    { date: "2026-01-02", price: 1380 },
    // ... 至少 7 天数据
  ],
  route: {
    origin: "上海",
    destination: "东京"
  }
};
```

### 2. 调用技能

```javascript
// 伪代码示例
const summary = await generatePriceSummary(priceData, {
  style: 'concise',  // 'concise' 或 'detailed'
  includeTable: true
});
```

### 3. 输出示例

```markdown
## 💰 价格分析

### 当前价格：¥1,299

**价格水位**：🟢 比均价便宜 12%

| 对比项 | 价格 |
|--------|------|
| 60 天最低 | ¥1,199 |
| 60 天平均 | ¥1,450 |
| 60 天最高 | ¥1,899 |

**走势**：📉 近 7 天下降

---

### 💡 建议：现在购买 ✅

当前价格比 60 天平均水平便宜 ¥151，处于较低水位。
如果您行程已确定，建议尽快下单。
```

## 价格水位判断

| 水位 | 条件 | Emoji | 建议 |
|------|------|-------|------|
| 🟢 低价 | 低于均价 15%+ | ✅ | 现在购买 |
| 🟡 中价 | 均价±15% | 🟡 | 可买可等 |
| 🔴 高价 | 高于均价 15%+ | 🔴 | 再等等 |

## 趋势判断

基于 7 日移动平均：

| 趋势 | 条件 | Emoji |
|------|------|-------|
| 下降 | 近 7 日均价 < 前 7 日均价 | 📉 |
| 上升 | 近 7 日均价 > 前 7 日均价 | 📈 |
| 平稳 | 差异 < 5% | ➡️ |

## 文案风格

### 语气
- 友好、像朋友给建议
- 避免冷冰冰的数据堆砌
- 用"建议"而非"一定"

### 结构
1. **结论先行**：第一句给出买/等
2. **数据支撑**：简单数字说明原因
3. **额外提示**：如"临近出发会涨价"

## 与 price-trend 的区别

| | price-summary | price-trend |
|---|---|---|
| 输出 | 纯文字 | 交互式图表 |
| 用户 | 普通 C 端 | 开发者/B 端 |
| 依赖 | 无 | 前端环境 |

## 注意事项

1. **数据不足时**：如果少于 7 天数据，标注"数据有限，仅供参考"
2. **极端价格**：如遇到异常值，需说明"可能受节假日影响"
3. **时效性**：临近出发日期（<7 天）建议更积极
