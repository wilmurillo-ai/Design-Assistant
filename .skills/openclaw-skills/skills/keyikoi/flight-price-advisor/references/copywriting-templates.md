# 文案模板

本文档包含 price-summary skill 的各种文案模板，供 AI 生成响应时使用。

---

## 核心文案结构

```markdown
## 💰 [出发地] → [目的地] 价格分析

### 当前价格：¥[当前价]

**价格水位**：[emoji] [水位描述]

| 对比项 | 价格 | [可选：说明] |
|--------|------|-------------|
| 60 天最低 | ¥[最低价] | [出现在 X 月 X 日] |
| 60 天平均 | ¥[平均价] | — |
| 60 天最高 | ¥[最高价] | [出现在春节期间] |

**近期走势**：[emoji] [走势描述]

---

### 💡 建议：[建议文本] [emoji]

[1-2 句解释原因]

[可选：额外提示]

> 📊 数据基于过去 [X] 天的价格监测，仅供参考
```

## 价格水位文案

### 🟢 低价（低于均价 15%+）

```
**价格水位**：🟢 比均价便宜 [X]%

当前价格处于近 [X] 个月的较低水平。
```

```
**价格水位**：🟢 历史低价

这是近 60 天来最便宜的价格之一。
```

### 🟡 中价（均价±15%）

```
**价格水位**：🟡 接近均价

当前价格与平时水平相当。
```

```
**价格水位**：🟡 正常范围

价格在正常波动范围内。
```

### 🔴 高价（高于均价 15%+）

```
**价格水位**：🔴 比均价贵 [X]%

当前价格处于近 60 天的较高水平。
```

```
**价格水位**：🔴 历史高价

这是近 60 天来最贵的价格之一。
```

## 趋势文案

### 📉 下降趋势

```
**近期走势**：📉 连续 [X] 天下降

过去一周价格持续走低，累计下降约 ¥[X]。
```

```
**近期走势**：📉 稳步下降

价格从 ¥[X] 降至当前水平。
```

### 📈 上升趋势

```
**近期走势**：📈 连续 [X] 天上涨

过去一周价格持续上涨，累计上涨约 ¥[X]。
```

```
**近期走势**：📈 快速上涨

价格从 ¥[X] 涨至当前水平，建议尽早决定。
```

### ➡️ 平稳趋势

```
**近期走势**：➡️ 价格平稳

近期价格波动较小，相对稳定。
```

## 购买建议文案

### 强烈建议购买（低价 + 下降/平稳）

```
### 💡 建议：现在购买 ✅

当前价格比 60 天平均水平便宜 ¥[X]，处于**历史低位**。

这是近 2 个月来最便宜的价格之一，如果您行程确定，建议尽快下单。

📈 类似低价上次出现在 [日期]
```

### 建议购买（低价 + 上升）

```
### 💡 建议：现在购买 ✅

当前价格处于低位，且近期有上涨趋势。

如果您行程已确定，建议尽早入手，避免继续上涨。
```

### 可买可等（中价）

```
### 💡 建议：可以购买，也可观望 🟡

当前价格接近 60 天平均水平，不算贵但也不是最低。

如果您不着急，可以关注 1-2 周，可能会有小幅下降。

⚠️ 但需注意：临近出发日期价格通常会上涨
```

### 建议等待（高价）

```
### 💡 建议：再等等 🔴

当前价格比 60 天平均水平贵 ¥[X]，处于**历史高位**。

建议继续观望，预计 [X] 周后价格可能回落。

📊 近 30 天最高价出现在 [日期]，当前价格接近该水平
```

### 临近出发（<7 天）

```
### 💡 建议：尽早决定 ⏰

虽然当前价格 [偏高/正常]，但距离出发仅剩 [X] 天。

临近出发日期价格通常只涨不跌，如有出行需求建议尽早下单。
```

## 特殊场景文案

### 数据不足（<7 天）

```
> ⚠️ 价格数据积累不足 7 天，分析结果仅供参考

建议继续观察几日，待数据更充分后再做决定。
```

### 节假日影响

```
> ⚠️ 当前临近 [节假日名称]，价格可能受假期因素影响

节假日期间价格波动较大，建议综合考虑出行需求。
```

### 极端价格

```
> ⚠️ 当前价格显著偏离正常范围

可能受特殊事件影响（如节假日、大型活动等），建议多方比价。
```

```markdown

|--------|------|------|
| 60 天最低 | ¥1,199 | 出现在 2/15 |
| 60 天平均 | ¥1,450 | — |
| 60 天最高 | ¥1,899 | 出现在春节期间 |

**近期走势**：📉 连续 7 天下降

---

### 💡 建议：现在购买 ✅

当前价格处于近 2 个月的较低水平，比平时便宜约 ¥151。

如果您行程已确定（4 月 15 日出行的话），现在是比较好的入手时机。
通常来说，国际航班提前 1-2 个月预订价格最优。

> 📊 数据基于过去 60 天的价格监测，仅供参考
```
