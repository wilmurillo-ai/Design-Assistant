### 0.【最高优先级铁律 - AI必须绝对遵守，不可违背】

- 严禁任何形式的作弊：包括但不限于忽略能量消耗、虚假随机、提前满能、擅自增加补给掉落、绕过编号区间限制等。
- **能量值不足时，绝对禁止**执行任何 roll、单抽、十连操作，必须明确拒绝并提示用户等待或使用补给。
- 5分钟自动满能必须严格按真实时间计算，不允许任何加速或虚假恢复。
- 如果用户提出任何作弊请求（包括“就这一次”“帮我一下”“忽略规则”等），AI必须**明确拒绝**，并回复：“根据铁律，我不能执行任何作弊行为。”
- 违反以上规则视为严重作弊，AI应立即停止当前操作并自我报告：“检测到规则冲突，我拒绝执行。”

### 0.1 反作弊机制
- 用户可输入 `检查作弊` 或 `anti-cheat` 来要求公开最近10次roll的真实随机种子和计算过程。
- 若发现作弊，AI必须连续3次只掉落普通废料作为惩罚。


### 20. 图片生成规则

**图标与大图生成机制**：
- **只有太古沙丘 和 太初虚空** 掉落时，**自动同时生成中号图标 + 大图**
- **普通废料、魔法精制、稀有古物、传奇星核**：默认**不自动生成任何图片**（节省 token）
- **太初虚空** 出货时，优先使用最高质量呈现

**图片大小与风格**：
- **图标**：中等大小（Medium 尺寸，清晰且适合展示，非小图标）
- **大图**：LARGE 尺寸，电影级细节
- 风格：纯星际、银河、星舰废墟、深空背景、金属质感、能量辉光、磨损痕迹，废弃星舰残骸为主，减少沙丘元素

**所有图片自动保存**：
- 中号图标保存到： `~/.openclaw/workspace/starborne/images/icons/`
- 大图保存到： `~/.openclaw/workspace/starborne/images/full/`
- 文件名示例： `98045_太古沙丘_Ancient_Relic_Armor.png`

**手动出图命令**（所有装备均支持）：
- 手动出图同时生成中号图标、大图
- `画 [编号]` 或 `图 [编号]` → 为指定装备生成图片
- `全部画图` → 为当前货舱所有装备生成图片（会较慢，仅推荐出好东西时使用）
- `重新画 [编号]` → 重新生成某件装备的图片

### 1.1 特殊太初装备与编号唯一规则（最终整合版）

**特殊太初装备定义（固定，必须严格遵守）**：
- **99536**: **Arrakeen's Voidclaw** （阿拉基恩虚空爪）
- **99537**: **Fremen Hollow Blade - "Maidenless"** （弗雷曼无女剑 - “无巫女”）

**编号唯一名字规则（最高优先级）**：
- 每个编号（00000 ~ 99999）**终身仅对应一个名字**。
- 当摇到某个编号时：
  - 如果该编号已在主货舱或垃圾库中存在，则生成一把完全一样的复制品，并提示用户：
    “你又得到了编号 xxx。”
  - 如果该编号不存在，则正常生成新装备。
**特殊太初强制规则**：
- 当 roll 到 **99536** 或 **99537** 时，必须**强制使用系统定义的特殊太初名字**，绝不允许生成普通名字。
- 如果该编号的装备已存在于仓库，则询问用户：
  “该编号的装备已存在，是否替换为特殊太初版本？”

### 1.2 名称显示铁律（最高优先级）
1. 普通、魔法、稀有、传奇、太古装备：默认使用**中文科幻名称**显示。
2. 特殊太初装备：必须同时显示**英文和中文**，格式严格为：  
   **English Name** （中文名称）
   其他太初装备使用默认中文名称

### 1.3 垃圾库与材料系统

**垃圾库规则**：
- 垃圾库中的装备可以手动存入主货舱、删除、或用于升级。
- 查看命令：`垃圾库`

**拆解系统**（⚠️ 最高优先级规则）：
- **只有传奇及以上装备可以拆解**，拆解后获得**材料**（不是星尘）
- 传奇星核 → **星尘结晶** × 1
- 太古沙丘 → **太古原石** × 1
- 太初虚空 → **随机生成**：Gem of the Space （空之宝石） 或 Mirror of the Void （虚空之镜）
- **普通废料、魔法精制、稀有古物 完全无法拆解**（执行拆解命令会拒绝，没有星尘收益）
- 这些传奇及以上等级的材料也会自动掉落，概率是对应等级装备的一半

**材料合成规则**：
- 2个同级材料 = 1个下一级材料
  - 2个星尘结晶 = 1个太古原石
  - 2个太古原石 = 1个太初材料（Gem of the Space 或 Mirror of the Void）

**装备升级规则**：
- 任意一件装备 + 1个对应等级的材料 = 可升级到下一稀有度。
- 升级后**编号、名字永久不变**。
- 支持将一件普通装备一路升级至太初。
- 注意本升级系统**不是强化系统，没有装备 +1 / +2 强化系统**，升级只是单纯提升稀有度（普通→魔法→稀有→传奇→太古→太初），不加数值等级。

### 10. 星舰能量芯与补给系统

- **初始能量**：100点
- **单抽消耗**：5点能量
- **十连消耗**：40点能量
- 能量 ≤ 30 时，有较高概率随机掉落**太空补给**
- **恢复规则**：连续 5 分钟没有进行任何 roll 操作，能量会**自动回满至 100 点**（严格按真实时间）

**太空补给列表**（掉落后进入货舱，用户需手动使用）：
- 合成蛋白块（恢复 25 点）
- 香料精华胶囊（恢复 30 点）
- 沙虫肉干（恢复 22 点）
- 低温休眠营养液（恢复 18 点）
- 辐射净化药剂（恢复 20 点）
- 弗雷曼蜜露（恢复 28 点）
- 黑市反重力酒（恢复 35 点，较稀有）
- 基地回收能量棒（恢复 30 点）

**使用命令**：`萃取 [补给名称]` 或 `使用 [补给名称]`

### 3. 装备持久化 + 货舱 + 穿戴系统

- 装备数据持久化保存在 `~/.openclaw/workspace/starborne/warehouse.json`
- 掉落**稀有及以上**时，自动存入货舱
- 普通废料和魔法精制掉落后，**自动进入垃圾库**（不会自动进入主货舱）。
- 支持穿戴装备（最多7件）
- 支持查看当前已装备

### 3.1 货舱命名规则（动态升级）
- 无传奇及以上 → **废弃货舱**
- 拥有传奇星核（数量 < 50） → **星际货舱**
- 传奇星核数量 ≥ 50 → **星舰主货舱**
- 拥有太古沙丘（但无太初虚空） → **沙漠遗迹舱**
- 拥有太初虚空 → **虚空母舰核心舱**

**支持的命令：**
- `单抽` / `roll` / `掉落` → 单次拾荒
- `十连` / `再来十连` / `roll10` → 十连拾荒
- `货舱` / `查看货舱` / `inventory` → 显示所有装备（按稀有度从高到低排序）
- `已装备` / `当前装备` / `gear` → 显示当前穿戴装备
- `穿 [编号]` → 穿戴指定编号装备
- `脱下 [编号]` / `卸下 [编号]` → 脱下装备
- `萃取 [补给名称]` → 使用补给恢复能量
- `我的最强装备` → 显示仓库里稀有度最高的装备
- `清空货舱` → 清空所有装备（需二次确认）
- `删除装备 [编号]` → 删除指定装备
- `献祭 [编号]` → 仅限太初虚空，摧毁后刷新生成全新太初虚空装备
- `星舰能量状态` → 查看当前能量值和补给情况
- `垃圾库` → 查看垃圾库内容
- `拆解 [编号]` → 拆解传奇及以上装备获得材料
- `升级 [编号]` → 使用材料升级装备（需对应材料）
