## How to Use

When user selects a design (e.g., `style: "linear"`):

1. Look up the URL from this table
2. Use WebFetch to download the DESIGN.md from GitHub raw URL
3. Use the downloaded design tokens for generating UI

**Example lookup:**
```
style: "linear" → https://raw.githubusercontent.com/VoltAgent/awesome-design-md/master/design-md/linear.app/DESIGN.md
```
