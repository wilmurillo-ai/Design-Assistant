# YouBike MCP Server

Model Context Protocol (MCP) server for YouBike 2.0 real-time data in Taipei, New Taipei, and Taoyuan.

## Features
- Search stations by keyword.
- Get nearby stations based on coordinates.
- Supports Taipei, New Taipei, and Taoyuan.

## Project Structure
- `src/`: Server implementation.
- `tests/`: Integration tests.
- `docs/`: Documentation.

## Usage
### Start Server
```bash
npm start
```

### Run Tests
```bash
npm test
```
