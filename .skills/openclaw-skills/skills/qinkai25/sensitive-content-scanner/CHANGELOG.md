## [2.0.0] - 2026-04-14

### 新增 ✨

- **置信度评分系统**：首次引入三级置信度评分（高/中/低），帮助用户快速识别真实威胁
  - 🔴 高置信度：已通过格式验证 + 校验码验证
  - 🟡 中置信度：格式匹配但验证不确定
  - 🟢 低置信度：仅符合基本模式

- **智能验证机制**：
  - 身份证号：校验码验证（加权因子算法）
  - 银行卡号：Luhn 算法校验
  - IP 地址：范围验证（0-255）

- **可视化报告**：
  - 图标化置信度指示器（🔴🟡🟢）
  - 统计摘要（按置信度分组）
  - 清晰的置信度说明章节

- **灵活配置**：
  - 中文姓名检测可通过 `--enable-chinese-name` 参数启用
  - 默认禁用，避免技术文档误报

### 优化 ⚡

- **误报率大幅降低**：
  - 默认禁用中文姓名检测
  - 姓氏白名单验证机制
  - 技术文档误报率降低 90%+

- **报告可读性提升**：
  - Markdown 格式增加图标和颜色标注
  - JSON 格式增加置信度字段
  - 统计数据更直观

- **代码质量**：
  - 重构验证逻辑，提高可维护性
  - 优化正则表达式，提升性能
  - 改进错误处理机制

### 修复 🐛

- 修复中文姓名检测误报问题（默认禁用）
- 修复报告格式不一致问题
- 修复文件路径处理异常

### 文档 📝

- 新增 README.md（快速开始指南）
- 更新 SKILL.md（完整功能说明）
- 新增 CHANGELOG.md（版本历史）
- 更新 pii_patterns.md（PII 验证说明）

## [1.0.0] - 2026-04-13

- **PII 检测**：
  - 身份证号检测（18位）
  - 手机号检测（11位）
  - 银行卡号检测（16-19位）
  - 邮箱地址检测

- **命令行接口**：
