# 用户操作指南

## 🎯 快速开始

### 第一步：安装

确保已安装 Python 3.8+，然后安装依赖：

```bash
pip install python-docx openpyxl python-pptx PyMuPDF requests
```

### 第二步：准备文件

将要处理的文档放在一个文件夹中。

### 第三步：选择处置方式

---

## 📋 两种处置方式

### 方式一：脱敏处理（推荐）

**适用场景**：想去除文档中的敏感信息，但仍想保留文档可读性。

**操作步骤**：

1. 打开终端，进入工具目录
2. 运行命令：
   ```bash
   python disposal.py 您的文档.docx --action redact
   ```
3. 处理后的文件会自动生成为 `您的文档_脱敏版.docx`

**高级选项**：
```bash
# 关键字替换
python disposal.py 文档.docx --action redact -k "密码,密钥"

# 覆盖原文件
python disposal.py 文档.docx --action redact --overwrite
```

---

### 方式二：文件加密

**适用场景**：需要完全保密文件，只有知道密码才能查看。

**操作步骤**：

1. 打开终端，进入工具目录
2. 运行命令：
   ```bash
   python disposal.py 您的文档.docx --action encrypt -p "您的密码"
   ```
3. 处理后的文件会自动生成为 `您的文档_加密版.docx.enc`
4. **请妥善保管密码！**

**发送通知**：
```bash
# 加密后发送到飞书
python disposal.py 文档.docx --action encrypt -p "密码" \
  --feishu-webhook "您的飞书Webhook地址"
```

## ❓ 常见问题

### Q: 脱敏和加密有什么区别？

**脱敏**：去除敏感信息，文档仍然可读，但敏感内容被替换为 `*`
- 例如：`13812345678` → `138****5678`

**加密**：用密码保护文件，没有密码无法查看内容
- 例如：文件变成 `.enc` 格式，需要密码才能打开

### Q: 密码忘记了怎么办？

抱歉，密码无法找回。请务必妥善保管密码。

### Q: 可以批量处理多个文件吗？

可以，将多个文件放在一个文件夹中，逐个处理。

### Q: 处理失败了怎么办？

检查错误信息，常见原因：
- 文件路径包含中文或特殊字符
- 没有文件写入权限
- 依赖库未正确安装

## 🔧 进阶配置

### 使用配置文件

创建 `config.json`：
```json
{
  "feishu_webhook": "https://open.feishu.cn/...",
  "wecom_webhook": "https://qyapi.weixin.qq.com/..."
}
```

然后使用：
```bash
python disposal.py 文档.docx --config config.json
```
