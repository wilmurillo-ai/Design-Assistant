## [1.2.0] - 2026-04-21

### 新增

- **PDF 格式支持**：新增 PDF 脱敏处理
  - 支持 PII 检测（身份证、手机号、银行卡、邮箱、IP）
  - 支持自定义关键词脱敏
  - 需要安装 PyMuPDF (`pip install pymupdf`)

### 测试验证

- ✅ PDF 脱敏：正常检测身份证、手机号、银行卡、邮箱
- ✅ scanner 非数据字典扫描：功能正常，不受数据字典影响

## [1.0.0] - 2026-04-20

### 新增

- **关键词脱敏功能**
  - 部分替换：保留首尾，中间用 `*` 替换
  - 关键字替换：完整替换为 `[REDACTED]`
  - 正则替换：按规则替换中间部分
- **PII 自动识别**
  - 身份证号（带校验码验证）
  - 手机号
  - 银行卡号（Luhn 算法校验）
  - 邮箱地址
  - IP 地址
- **文件加密功能**
  - 密码保护（PBKDF2 + XOR）
  - 支持解密
- **通知服务**
  - 飞书 Webhook 通知
  - 企业微信 Webhook 通知
- **多格式支持**
  - 文本文件：txt, md, json, xml, csv
  - Word：docx
  - Excel：xlsx
  - PPT：pptx
- **配置文件支持**（JSON 格式）
- **命令行接口**
