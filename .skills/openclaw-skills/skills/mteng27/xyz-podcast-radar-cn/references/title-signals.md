# Title Signals

The first pass should learn as much as possible from titles before enriching any Xiaoyuzhou page.

## Episode Markers

Look for:

- `S8E9`
- `EP03`
- `Vol132`
- `第 14 期`

These help estimate:

- maturity of the show
- editorial consistency
- whether the user is looking at a flagship or an early-stage program

## Guest Hints

Strong guest clues often appear as:

- `A×B`
- `对话某某`
- `对谈某某`
- `with 某某`
- a trailing guest name after punctuation

Treat them as hints, not guarantees.

## Format Hints

Common format cues:

- `对话`
- `对谈`
- `访谈`
- `复盘`
- `盘点`
- `问答`
- `故事`
- `观察`
- `独白`

These help decide whether a result suits listening, benchmarking, or distribution work.

## Topic Keywords

The script extracts simple topic cues from the title. Common useful buckets include:

- AI / 科技
- 商业 / 创业 / 投资
- 职场 / 管理
- 社会 / 文化 / 历史
- 喜剧 / 幽默
- 教育 / 健康 / 情感
- 女性 / 国际 / 媒体 / 电影 / 音乐

These are lightweight hints, not a replacement for full semantic classification.

## Practical Rule

If the title, genre, rank, and a few metrics already answer the user's question, stop there.

Use enrichment only when the answer would otherwise feel thin or ambiguous.
