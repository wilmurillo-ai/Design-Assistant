# Growth Spread Matrix

## Metadata
- **Name**: growth-spread-matrix
- **Description**: Value creation vs growth analysis framework
- **Triggers**: growth spread, value matrix, value creation analysis, competitive positioning

## Instructions
Analyze how business creates value and grows using the Growth Spread Matrix framework.

## Framework

### The 2×2 Matrix

```
                   HIGH GROWTH
                          │
              ┌─────────┴──────────────┐
              │       DESTROYING           │
              │   VALUE + GROWING QUICKLY │
              │   (High margin, low growth)  │
              └───────────┬──────────────────┘
                          │
        LOW GROWTH          HIGH VALUE CREATION
            ┌────────────┴──────────────┐
            │       BUILDING VALUE          │
            │   (Investing, high margin) │
            └───────────┬──────────────────┘
                          │
     LOW VALUE CREATION      │
            ┌────────────┴──────────────┐
            │   HARVESTING VALUE          │
            │   (Milk cash, divest)  │
            └───────────┴──────────────────┘
                          │
              MEDIUM VALUE
                    ┌────────────┼──────────────┐
                    │  GROWING SLOWLY
                    │   (Value adding, growing) │
                    └───────────┬──────────────────┘
```

### Quadrant Characteristics

| Quadrant | Value Creation | Growth | Strategy | Cash Flow | Investment |
|-----------|----------------|----------|-----------|------------|
| **Top-Left** | Destroying | High | Aggressive | 💸 Negative | 💸 High |
| **Top-Right** | Building | High | Growth | 💸 Positive | 💸 Moderate |
| **Bottom-Left** | Harvesting | Low | Defensive | 📊 Positive | 📊 Low |
| **Bottom-Right** | Marginal | Low | Turnaround | 📊 Negative | 📊 Moderate |

## Output Process

1. **Assess each unit** - Calculate margin and growth
2. **Plot on matrix** - Position business units
3. **Analyze portfolio balance** - Is there healthy mix?
4. **Identify opportunities** - Shift units to better quadrants
5. **Develop strategies** - Specific actions per quadrant
6. **Plan resource allocation** - Where to invest?
7. **Monitor** - Track movements over time

## Tips

- The matrix is dynamic - units move over time
- Portfolio balance is critical - all four quadrants should be represented
- Value creation metrics vary by industry
- Growth rates are relative to industry, not absolute
- Consider strategic options - acquisition, divestiture, turnarounds
- Monitor regularly - quarterly reviews recommended
