# Value Disciplines Model

## Metadata
- **Name**: value-disciplines
- **Description**: Treacy & Wiersema strategic positioning framework
- **Triggers**: value discipline, product leadership, operational excellence, customer intimacy

## Framework

Three generic value disciplines - companies must excel at one, be adequate in others:

| Discipline | Definition | Typical of |
|------------|------------|------------|
| **Product Leadership** | Best products, innovation | Apple, Tesla, Nike |
| **Operational Excellence** | Best price, convenience | Amazon, Walmart, McDonald's |
| **Customer Intimacy** | Best solution, relationship | Nordstrom, Ritz-Carlton, IBM consulting |
