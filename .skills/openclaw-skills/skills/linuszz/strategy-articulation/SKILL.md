# Strategy Articulation

## Metadata
- **Name**: strategy-articulation
- **Description**: Strategy statement development and communication
- **Triggers**: strategy statement, strategic vision, mission, strategic direction

## Instructions

Develop clear strategy articulation for $ARGUMENTS.

## Framework

### Strategy Statement Structure

```
┌─────────────────────────────────────┐
│         VISION                      │
│    Where we want to be              │
├─────────────────────────────────────┤
│         MISSION                     │
│    What we do and why               │
├─────────────────────────────────────┤
│         STRATEGIC OBJECTIVES        │
│    What we will achieve             │
├─────────────────────────────────────┤
│         STRATEGIC PRIORITIES        │
│    Where we will focus              │
├─────────────────────────────────────┤
│         KEY ACTIONS                 │
│    What we will do                  │
└─────────────────────────────────────┘
```

### Elements

| Element | Definition | Example |
|---------|------------|---------|
| **Scope** | Where we compete | "Enterprise software in healthcare" |
| **Advantage** | How we win | "Best-in-class user experience" |
| **Objectives** | What we achieve | "30% market share by 2027" |
| **Capabilities** | What we need | "World-class product team" |

## Output

```
## Strategy Articulation: [Business/Unit]

### Vision Statement
[Inspiring 10-15 word statement of future state]

### Mission Statement
[Clear statement of purpose and approach]

### Strategic Objectives

| Objective | Measure | Target | Timeline |
|-----------|---------|--------|----------|
| [Objective 1] | [Metric] | [Target] | [Date] |
| [Objective 2] | [Metric] | [Target] | [Date] |
| [Objective 3] | [Metric] | [Target] | [Date] |

### Strategic Priorities

1. **[Priority 1]**: [Description]
2. **[Priority 2]**: [Description]
3. **[Priority 3]**: [Description]

### Key Actions

| Priority | Action | Owner | Deadline |
|----------|--------|-------|----------|
| [P1] | [Action] | [Name] | [Date] |
| [P2] | [Action] | [Name] | [Date] |

### Strategy Summary

**Where we play**: [Scope]
**How we win**: [Advantage]
**What we'll achieve**: [Objectives]
```

## Tips
- Keep vision aspirational but achievable
- Make objectives SMART
- Limit to 3-5 priorities
- Test for clarity with stakeholders
