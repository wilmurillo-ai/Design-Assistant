## 阅读技巧

### 快速提取TXT内容
```
使用 read 工具读取文件
按章节分隔（通常以"第X章"或"Chapter X"标记）
```

### PDF处理
- 图片型PDF：需要OCR识别
- 文字型PDF：直接提取
- 使用工具：Python pdfplumber

### EPUB结构
```
/content.opf - 元数据
/Text/*.xhtml - 正文
/images/ - 图片
/toc.ncx - 目录
```
