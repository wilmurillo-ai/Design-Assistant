# AutoClaw - 自动化脚本工具箱

## 🤖 产品简介

AutoClaw 是一个自动化的系统健康和备份工具，专为 AI 助手和工作区设计。

**核心功能**:
- ✅ 自动健康检查
- ✅ 定时备份配置
- ✅ 进程守护监控
- ✅ 日志自动记录

---

## ✨ 主要功能

### 1. 健康检查
自动检查核心文件和目录：
- config.json (系统配置)
- MEMORY.md (长期记忆)
- docs/ (文档目录)
- memory_data/ (数据库)
- tasks/ (任务目录)

### 2. 自动备份
- 定时备份重要文件
- 自动清理旧备份 (保留最近 10 个)
- 备份位置可配置

### 3. 进程守护
- 监控关键进程状态
- 进程崩溃自动检测
- 可选自动重启

### 4. 日志记录
- 所有操作自动记录
- 带时间戳
- 便于问题排查

---

## 🚀 快速开始

### 安装
```bash
# 克隆技能
clawhub install autoclaw

# 或者直接运行
python autoclaw.py
```

### 配置
编辑 `autoclaw_config.json`:
```json
{
  "workspace": "C:\\Users\\YourName\\.openclaw\\workspace",
  "backup_dir": "backups",
  "max_backups": 10,
  "check_interval": 30,
  "auto_restart": true
}
```

### 运行
```bash
# 手动运行
python autoclaw.py

# 或者添加到定时任务
# Windows: 任务计划程序
# Mac/Linux: cron
```
