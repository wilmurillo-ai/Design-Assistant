# 飞书多维表格仪表盘设计

## 仪表盘概述

**仪表盘价值：**
- 📊 数据可视化
- 🎯 KPI监控
- 📈 趋势分析
- 🔍 洞察发现
- 🚀 决策支持

## 组件分类

### 1. 指标卡（最常用）

**适用场景：** 单个关键指标展示

**配置选项：**
- 数值格式（数字/百分比/货币）
- 环比/同比展示
- 目标值对比
- 趋势箭头

**示例：**
```
本月销售额
¥1,234,567
↑ 15.3% 环比增长
```

**最佳实践：**
- 一屏展示5-10个核心KPI
- 使用颜色区分正负增长
- 添加目标线

### 2. 柱状图

**适用场景：** 分类数据对比

**配置：**
- X轴：分类字段（单选/多选）
- Y轴：数值字段（求和/计数/平均值）
- 分组：第二个分类维度
- 堆叠/并列显示

**示例：**
```
各产品线销售额对比
  │
█ │     ██
█ │ ██  ██  ██
█ │ ██  ██  ██  ██
──┼──────────────
   A   B   C   D
```

### 3. 折线图

**适用场景：** 趋势分析

**配置：**
- X轴：时间字段（日期）
- Y轴：数值字段
- 多指标对比
- 平滑曲线

**示例：**
```
月度销售趋势
  │
█ │         ╱╲
█ │      ╱╲╱  ╲
█ │   ╱╲╱      ╲
█ │╱╱           ╲
──┼────────────────
   1月 2月 3月 4月 5月
```

### 4. 饼图

**适用场景：** 占比分析

**配置：**
- 分类字段
- 数值字段（自动计算占比）
- 显示百分比/数值
- 突出显示某扇区

**示例：**
```
客户行业分布
教育 25%
科技 30%
金融 20%
制造 15%
其他 10%
```

### 5. 漏斗图

**适用场景：** 转化率分析

**示例：**
```
销售漏斗
线索    1000
↘ 转化率 40%
机会    400
↘ 转化率 50%
报价    200
↘ 转化率 60%
成交    120
```

### 6. 排行榜

**适用场景：** Top N展示

**配置：**
- 排名字段（数值）
- 显示字段（名称）
- Top N数量
- 升序/降序

**示例：**
```
销售业绩排行
1. 张三  ¥892,000
2. 李四  ¥756,000
3. 王五  ¥623,000
4. 赵六  ¥534,000
5. 钱七  ¥421,000
```

### 7. 透视表

**适用场景：** 多维交叉分析

**配置：**
- 行维度
- 列维度
- 数值（汇总方式）
- 小计/总计

**示例：**
```
产品-区域销售透视
         北区   南区   总计
产品A    100    150    250
产品B    80     120    200
总计     180    270    450
```

### 8. 切片器

**适用场景：** 动态筛选

**功能：**
- 筛选仪表盘所有图表
- 多选/单选
- 快速切换维度

**示例：**
```
时间范围：[本月] [本季度] [本年]
产品线：  [全部] [产品A] [产品B]
区域：    [全部] [北区] [南区] [东区] [西区]
```

## 仪表盘设计流程

### Step 1: 确定KPI

**方法：**
1. 明确业务目标
2. 识别关键指标
3. 确定数据来源

**示例（销售系统）：**
- 核心KPI：
  - 本月销售额
  - 新客户数量
  - 订单转化率
  - 平均客单价

### Step 2: 选择组件

**原则：**
- **指标卡** → 单个数值KPI
- **柱状图** → 分类对比
- **折线图** → 时间趋势
- **饼图** → 占比分布
- **排行榜** → Top N
- **透视表** → 多维分析

### Step 3: 布局设计

**布局原则：**
```
┌────────────────────────────────────┐
│  指标卡（核心KPI，横向排列）        │
├────────────┬───────────────────────┤
│            │                       │
│  主要图表  │    次要图表           │
│  (大)      │    (中/小)            │
│            │                       │
├────────────┴───────────────────────┤
│  辅助图表/明细表格                  │
└────────────────────────────────────┘
```

**尺寸建议：**
- 指标卡：1x1（小方块）
- 主图表：2x2 或 3x2
- 次图表：2x1 或 1x2

### Step 4: 配置交互

**交互功能：**
- 切片器筛选
- 图表联动（点击图表筛选其他）
- 下钻（查看明细）
- 链接跳转

## 高级功能

### 1. AI创建仪表盘

**使用方式：**
- 输入："创建一个销售分析仪表盘"
- AI自动识别KPI和组件
- 生成推荐布局

### 2. 图表联动

**配置：**
- 点击图表A，自动筛选图表B
- 支持多图表联动
- 联动范围可配置

**示例：**
```
点击"区域分布"饼图的"北区"
→ 所有图表自动筛选为北区数据
```

### 3. 条件格式

**配置：**
- 根据数值自动着色
- 设置阈值和颜色规则
- 视觉预警

**示例：**
```
库存 < 100 → 红色
库存 100-500 → 黄色
库存 > 500 → 绿色
```

### 4. 数据刷新

**刷新方式：**
- 手动刷新
- 定时刷新（每小时/每天）
- 实时刷新（数据变更时）

## 性能优化

### 1. 数据量控制

**建议：**
- 单图表 ≤ 10,000条记录
- 使用筛选减少数据量
- 聚合后再展示

### 2. 组件数量

**建议：**
- 单仪表盘 ≤ 20个组件
- 避免过多复杂组件
- 合理使用切片器

### 3. 查询优化

**策略：**
- 使用索引字段
- 减少复杂计算
- 缓存常用查询
