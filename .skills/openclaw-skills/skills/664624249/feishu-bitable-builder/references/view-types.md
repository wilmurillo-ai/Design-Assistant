# 飞书多维表格视图类型指南

## 视图类型一览

| 视图类型 | 适用场景 | 核心特点 |
|---------|---------|---------|
| 表格视图 | 数据管理、批量操作 | 类似Excel，最常用 |
| 看板视图 | 状态流转、项目管理 | 可视化卡片，拖拽排序 |
| 日历视图 | 时间管理、日程安排 | 按日期展示，时间线 |
| 甘特图视图 | 项目规划、进度跟踪 | 时间线、依赖关系 |
| 画册视图 | 图片展示、产品管理 | 卡片式展示，突出图片 |
| 仪表盘视图 | 数据分析、可视化 | 图表组合，多维度分析 |
| 独立分享视图 | 对外分享、数据展示 | 独立链接，权限控制 |
| 保护视图 | 敏感数据保护 | 字段级权限控制 |

## 视图详细说明

### 1. 表格视图（最常用）

**适用场景：**
- 数据录入和管理
- 批量编辑和导入
- 数据筛选和排序
- 全局数据查看

**核心功能：**
- ✅ 行列操作（增删改）
- ✅ 筛选和排序
- ✅ 分组显示
- ✅ 冻结列
- ✅ 字段隐藏/显示
- ✅ 批量编辑

**配置建议：**
- 常用字段放在左侧
- 冻结关键字段（如名称）
- 设置默认排序规则
- 按角色定制筛选条件

**示例应用：**
- 客户列表
- 任务清单
- 产品目录
- 订单记录

### 2. 看板视图（项目管理神器）

**适用场景：**
- 状态流转管理
- 任务看板（待办→进行中→已完成）
- 销售漏斗（线索→机会→成交）
- 项目进度跟踪

**核心功能：**
- ✅ 按单选字段分组（状态、阶段）
- ✅ 卡片拖拽排序
- ✅ 自定义卡片显示字段
- ✅ WIP限制（在制品数量限制）
- ✅ 卡片计数统计

**配置建议：**
- 分组字段：使用单选字段（状态、阶段）
- 卡片内容：显示3-5个关键字段
- 颜色区分：按优先级或类型着色
- WIP限制：避免某个阶段任务过多

**示例应用：**
```
任务看板（按状态分组）：
┌─────────┬──────────┬─────────┐
│  待办   │ 进行中   │ 已完成  │
├─────────┼──────────┼─────────┤
│ 任务1   │ 任务3    │ 任务5   │
│ 任务2   │ 任务4    │ 任务6   │
│         │          │ 任务7   │
└─────────┴──────────┴─────────┘

销售看板（按阶段分组）：
┌─────────┬──────────┬─────────┬─────────┐
│  线索   │  机会    │  报价   │  成交   │
├─────────┼──────────┼─────────┼─────────┤
│ 线索A   │ 机会1    │ 报价X   │ 成交✓   │
│ 线索B   │ 机会2    │         │         │
└─────────┴──────────┴─────────┴─────────┘
```

### 3. 日历视图

**适用场景：**
- 时间管理
- 日程安排
- 截止日期跟踪
- 会议和活动管理

**核心功能：**
- ✅ 按日期字段展示
- ✅ 月/周/日视图切换
- ✅ 拖拽调整日期
- ✅ 颜色分组
- ✅ 全天事件/时间段

**配置建议：**
- 日期字段：选择开始日期或截止日期
- 显示字段：事件名称+关键信息
- 颜色：按类型或负责人着色

**示例应用：**
- 任务截止日历
- 会议安排
- 营销活动日历
- 客户拜访计划

### 4. 甘特图视图

**适用场景：**
- 项目规划和排期
- 任务依赖关系
- 进度跟踪
- 资源分配

**核心功能：**
- ✅ 时间线展示
- ✅ 任务依赖关系（前置任务）
- ✅ 进度条显示
- ✅ 里程碑标记
- ✅ 资源负载图
- ✅ 关键路径

**配置要求：**
- 必须有：开始日期字段、结束日期字段
- 可选：进度字段、前置任务字段（关联）

**配置建议：**
- 设置合理的任务粒度
- 明确任务依赖关系
- 定期更新进度
- 标记关键里程碑

**示例应用：**
```
项目计划：
任务1 ████████░░░░░░░ 60%  (03/01-03/15)
任务2 ░░░░██████████ 100%  (03/05-03/20)
任务3 ░░░░░░░░███████ 80%  (03/15-03/30) [依赖任务2]
```

### 5. 画册视图

**适用场景：**
- 图片为主的数据展示
- 产品目录
- 人员名册（含头像）
- 作品集展示

**核心功能：**
- ✅ 卡片式布局
- ✅ 图片预览
- ✅ 自定义显示字段
- ✅ 筛选和分组

**配置建议：**
- 主图片字段：附件字段（图片）
- 显示字段：名称+2-3个关键信息
- 排序：按时间或名称

**示例应用：**
- 产品展示（产品图片+名称+价格）
- 团队成员（头像+姓名+职位）
- 项目案例（项目图片+项目名称+客户）

### 6. 仪表盘视图（数据分析）

**适用场景：**
- 数据可视化
- KPI监控
- 趋势分析
- 决策支持

**组件类型：**

**基础图表（10种）：**
- 柱状图 - 分类对比
- 折线图 - 趋势变化
- 饼图 - 占比分析
- 条形图 - 排名对比
- 面积图 - 累积趋势
- 雷达图 - 多维评估
- 散点图 - 相关性分析
- 组合图 - 多指标对比
- 漏斗图 - 转化率分析
- 词云图 - 文本频率

**视图组件（5种）：**
- 表格视图 - 明细数据
- 看板视图 - 状态总览
- 日历视图 - 时间分布
- 甘特图视图 - 项目时间线
- 画册视图 - 图片展示

**数据组件（4种）：**
- 指标卡 - 单个关键指标
- 文本 - 说明文字
- 按钮 - 快捷操作
- 排行榜 - Top N展示

**高级组件（7种）：**
- NPS图 - 净推荐值
- 进度图 - 目标达成率
- 透视表 - 多维交叉分析
- 切片器 - 动态筛选
- 地图 - 地理分布
- 热力图 - 密度分析
- 关系图 - 关联关系

**配置建议：**
- 确定核心KPI（3-5个）
- 选择合适图表类型
- 使用切片器增强交互
- 合理布局（重要指标突出）

**参考：** [dashboard.md](dashboard.md)

### 7. 独立分享视图

**适用场景：**
- 对外展示数据
- 嵌入外部网站
- 公开数据发布
- 跨组织协作

**核心功能：**
- ✅ 生成独立链接
- ✅ 权限控制（只读/编辑）
- ✅ 字段级别控制
- ✅ 访问密码保护
- ✅ 访问统计

**配置建议：**
- 仅展示必要字段
- 设置访问密码
- 定期审查访问权限
- 监控访问日志

**示例应用：**
- 产品价格表（公开）
- 服务状态页（客户查看）
- 调查问卷结果（分享给参与者）

### 8. 保护视图

**适用场景：**
- 敏感数据保护
- 多角色权限管理
- 数据分级展示

**核心功能：**
- ✅ 字段级权限控制
- ✅ 角色视图
- ✅ 数据脱敏

**配置建议：**
- 按角色创建视图
- 隐藏敏感字段
- 限制编辑权限

## 视图组合策略

### 场景1：项目管理

**配置视图：**
1. **表格视图** - 任务清单（全局管理）
2. **看板视图** - 按状态分组（流转管理）
3. **甘特图** - 项目时间线（规划）
4. **日历视图** - 截止日期（时间管理）
5. **仪表盘** - 项目进度和统计（监控）

### 场景2：客户管理

**配置视图：**
1. **表格视图** - 客户列表（数据管理）
2. **看板视图** - 按客户阶段（销售漏斗）
3. **画册视图** - 客户名片（快速浏览）
4. **仪表盘** - 业绩分析（数据分析）

### 场景3：库存管理

**配置视图：**
1. **表格视图** - 库存清单（详细管理）
2. **看板视图** - 按库存状态（正常/预警/缺货）
3. **画册视图** - 产品展示（图片为主）
4. **仪表盘** - 库存概览（实时监控）

## 视图性能优化

### 数据量建议
- **表格视图：** 单视图 ≤ 10,000条记录
- **看板视图：** 单分组 ≤ 500条记录
- **日历视图：** 单视图 ≤ 1,000条记录
- **甘特图：** 单视图 ≤ 500条任务

### 优化技巧
1. **使用筛选：** 避免一次性加载所有数据
2. **字段精简：** 视图中仅显示必要字段
3. **分组合理：** 避免过多分组
4. **定期归档：** 历史数据定期归档
