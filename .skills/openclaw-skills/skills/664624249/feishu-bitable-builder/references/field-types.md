## 字段类型选择指南

### 1. 文本类字段

**文本 (1)**
- **适用场景：** 名称、描述、备注、标题
- **特点：** 自由输入，无格式限制
- **最佳实践：**
  - 重要字段设置必填
  - 长文本考虑字数限制
  - 需要格式校验使用公式

**电话 (13)**
- **适用场景：** 联系方式
- **特点：** 自动格式化，支持点击拨打
- **最佳实践：** 配合文本字段存储多个号码

**URL (15)**
- **适用场景：** 链接地址、文档链接
- **特点：** 可点击跳转，显示友好
- **最佳实践：** 配合文本字段添加链接说明

### 2. 数字类字段

**数字 (2)**
- **适用场景：** 数量、金额、百分比、评分
- **特点：** 支持数学运算、汇总统计
- **配置选项：**
  - 数字格式：整数、小数、百分比、货币
  - 小数位数：0-10位
  - 千分位分隔符
- **最佳实践：**
  - 金额使用货币格式
  - 百分比使用百分比格式
  - 设置合理的数值范围

**进度**（特殊数字字段）
- **适用场景：** 项目进度、任务完成度
- **特点：** 可视化进度条
- **显示方式：** 0-100% 进度条

### 3. 选择类字段

**单选 (3)**
- **适用场景：** 状态、级别、类型（只能选一个）
- **特点：** 预定义选项，下拉选择
- **最佳实践：**
  - 选项数量控制在10个以内
  - 使用颜色区分不同选项
  - 常用选项放在前面
- **示例：**
  - 任务状态：待办、进行中、已完成、已取消
  - 客户级别：A类、B类、C类、D类
  - 优先级：高、中、低

**多选 (4)**
- **适用场景：** 标签、分类（可选择多个）
- **特点：** 预定义选项，可多选
- **最佳实践：**
  - 选项数量控制在20个以内
  - 为常用组合创建选项组
- **示例：**
  - 技能标签：Python、JavaScript、设计、产品
  - 产品分类：硬件、软件、服务

**选项组**
- **功能：** 将相关选项分组管理
- **适用场景：** 选项较多需要分类
- **示例：**
  - 技术类：Python、Java、前端、后端
  - 设计类：UI、UX、平面、3D

### 4. 时间类字段

**日期时间 (5)**
- **适用场景：** 截止日期、创建时间、时间点
- **特点：** 日期选择器，支持时间
- **配置选项：**
  - 仅日期 / 日期+时间
  - 时区设置
- **最佳实践：**
  - 使用公式计算时间差
  - 配合自动化做提醒
  - 在日历视图中展示

**创建时间 (1001)、修改时间 (1002)**
- **适用场景：** 自动记录时间
- **特点：** 系统自动生成，不可手动修改
- **最佳实践：** 用于审计、统计分析

### 5. 人员类字段

**用户 (11)**
- **适用场景：** 负责人、审批人、成员
- **特点：** 选择飞书用户，支持多选
- **配置选项：**
  - 单选 / 多选
  - 默认值：当前用户
- **最佳实践：**
  - 负责人用单选
  - 团队成员用多选
  - 配合通知自动化

**创建用户 (1003)、修改用户 (1004)**
- **适用场景：** 自动记录操作人
- **特点：** 系统自动生成，不可手动修改

### 6. 关联类字段

**关联单选 (18)**
- **适用场景：** 一对一关系
- **特点：** 关联另一张表的一条记录
- **示例：**
  - 任务 → 关联项目
  - 订单 → 关联客户
- **最佳实践：**
  - 建立双向关联
  - 使用查找引用获取关联数据

**关联多选 (21)**
- **适用场景：** 一对多关系
- **特点：** 关联另一张表的多条记录
- **示例：**
  - 项目 → 关联多个成员
  - 文章 → 关联多个标签

**查找引用 (19)**
- **适用场景：** 引用关联记录的字段值
- **特点：** 自动获取关联数据，实时更新
- **示例：**
  - 任务表 → 引用项目表的"项目负责人"
  - 订单表 → 引用客户表的"联系方式"
- **最佳实践：**
  - 避免多层查找引用（性能问题）
  - 配合公式使用

### 7. 文件类字段

**附件 (17)**
- **适用场景：** 文件、图片、文档
- **特点：** 支持多文件上传，预览
- **文件类型：**
  - 图片：jpg、png、gif
  - 文档：pdf、doc、xls
  - 其他：zip、mp4 等
- **限制：** 单文件 ≤ 20MB

### 8. 特殊字段

**复选框 (7)**
- **适用场景：** 是/否判断、标记
- **特点：** 勾选/未勾选
- **示例：** 是否完成、是否重要、是否公开

**位置 (22)**
- **适用场景：** 地理位置、地址
- **特点：** 地图选点、地址搜索
- **示例：** 客户地址、门店位置

**群组 (23)**
- **适用场景：** 关联飞书群聊
- **特点：** 选择飞书群，支持群消息推送
- **示例：** 项目群、讨论组

**自动编号 (1005)**
- **适用场景：** 唯一编号
- **特点：** 系统自动生成，格式可配置
- **格式示例：**
  - `ORD-{YYYY}-{0001}` → ORD-2026-0001
  - `TASK-{MM}{DD}-{001}` → TASK-0312-001
- **最佳实践：**
  - 配置有意义的编号规则
  - 避免删除记录导致编号不连续

**公式 (20)**
- **适用场景：** 自动计算、条件判断
- **特点：** 基于其他字段计算，实时更新
- **常用函数：** IF、SUM、LOOKUP、CONCAT、DATEDIF
- **参考：** [functions.md](functions.md)

## 字段类型组合场景

### 客户管理
```
客户名称（文本）+ 联系人（文本）+ 电话（电话）+ 
客户级别（单选）+ 负责人（用户）+ 创建时间（创建时间）
```

### 任务管理
```
任务名称（文本）+ 负责人（用户）+ 状态（单选）+ 
优先级（单选）+ 截止日期（日期时间）+ 进度（进度）+ 
关联项目（关联单选）
```

### 订单管理
```
订单编号（自动编号）+ 客户（关联单选）+ 
订单金额（数字）+ 订单状态（单选）+ 
下单时间（创建时间）+ 相关文件（附件）
```

### 产品管理
```
产品名称（文本）+ SKU（文本）+ 分类（多选）+ 
价格（数字）+ 库存（数字）+ 图片（附件）+ 
供应商（关联单选）
```

## 性能优化建议

### 字段数量
- **单表字段数：** ≤ 100个
- **推荐：** 20-50个核心字段

### 字段类型选择
- **优先使用：** 单选/多选（优于文本）
- **避免过度：** 复杂公式、多层查找引用
- **合理使用：** 关联字段（影响加载速度）

### 数据类型
- **文本字段：** 避免过长内容（>1000字）
- **附件字段：** 控制文件大小和数量
- **关联字段：** 避免关联过多记录（>100条）
