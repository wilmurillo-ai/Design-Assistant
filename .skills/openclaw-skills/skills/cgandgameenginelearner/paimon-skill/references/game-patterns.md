# 游戏自动化模式参考

## 常见手游自动化模式

### 体力型（崩铁/原神类）

**模式**：消耗体力 → 战斗 → 领奖励 → 循环

**识别点**：
- 体力条/能量条 UI
- "开始挑战" 按钮
- "领取奖励" 弹窗
- 体力不足提示

**执行流程**：
```
1. 检查体力是否足够
2. 点击"开始挑战"
3. 等待加载进入战斗
4. 识别"自动战斗"按钮并点击（如有）
5. 等待"战斗结束"
6. 点击"领取奖励"
7. 循环直到体力耗尽
```

### 爬塔型（深渊/模拟宇宙类）

**模式**：逐层挑战 → 领奖励 → 下一层

**识别点**：
- 当前层数显示
- "开始挑战" / "再次挑战" 按钮
- "通关" / "失败" 状态
- 下一层按钮

**执行流程**：
```
1. 确认当前层数
2. 点击"开始挑战"
3. 战斗过程监控（如需）
4. 识别战斗结束状态
5. 点击"领取奖励"或"下一层"
6. 重复直到满星或层主
```

### 刷副本型（方舟/粥类）

**模式**：选择副本 → 部署干员 → 开始 → 结束

**识别点**：
- 副本选择界面
- 干员部署按钮
- 技能释放时机
- 结束判定

### 日常任务型

**模式**：接任务 → 完成任务 → 领奖励 → 交任务

**识别点**：
- 每日任务列表
- 任务npc位置
- 交任务npc

## 各游戏特征识别

### 崩坏星穹铁道

- 体力称为"开拓力"，右上角显示
- 忘却之庭有10层，每层有3间
- 委托任务在左上角小地图旁
- 常用按钮："开始挑战"、"再次挑战"、"领取"

### 原神

- 体力称为"原粹树脂"，右上角显示
- 圣遗物副本需要树脂
- 深渊需要打12层
- 常用按钮："开始挑战"、"重新挑战"、"领取奖励"

### 明日方舟

- 体力称为"理智"，左上角显示
- 剿灭作战在鸽笼下方
- 物资副本有4种类型
- 常用按钮："开始行动"、"结束行动"、"领取奖励"

## 图像识别技巧

### 按钮识别

常见按钮特征：
- 圆角矩形，有明显边框
- 文字居中，字体较大
- 颜色与背景有明显对比
- hover状态会有亮度变化

### 状态识别

- 战斗中：进度条、倒计时、敌方血条
- 结算中：奖励图标、翻牌动画
- 体力不足：灰色按钮、弹窗提示

### 异常处理

- 网络波动：识别重连提示，等待恢复
- 游戏更新：识别维护公告，停止操作
- 检测风险：随机延迟，点击间隔变化
