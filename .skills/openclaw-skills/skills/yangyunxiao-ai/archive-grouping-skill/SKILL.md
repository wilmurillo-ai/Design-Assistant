# 档案整理归档组卷技能

## 技能概述

本技能用于辅助档案工作人员对办理完毕的文件材料进行整理、分类、组卷和归档，遵循《高等学校档案实体分类法》和国家档案管理相关规范，按照事由原则将同一事由的文件材料组织成案卷。

## 适用场景

- 高校党政管理类文件材料的整理归档
- 教学类、科研类、基建类等各门类档案的组卷
- 年度归档文件的分类整理
- 档案工作人员的日常组卷工作指导

## 核心功能

### 1. 文件材料分析
- 识别文件的类型（批复、请示、报告、通知、会议纪要等）
- 判断文件的形成单位/部门
- 确定文件的年度归属
- 识别文件之间的关联关系

### 2. 分类定位
- 确定一级类目（党群DQ、行政XZ、教学JX、科研KY、基建JJ、设备SB、出版CB、外事WS、财会CK等）
- 确定二级类目（按机构或职能划分）
- 生成分类号

### 3. 事由识别与组卷
- 按事由原则识别同一事由的相关文件
- 判断文件的组卷归属
- 确定卷内文件排列顺序

### 4. 保管期限判定
- 根据文件内容和价值判定保管期限（永久、30年、10年）
- 区分不同保管期限的文件

### 5. 档号编制
- 按照标准模式编制档号：年度号 + 分类号 + 案卷号
- 示例：2024-XZ12-001

## 工作流程

### 第一步：接收文件清单
用户提交待整理的文件题目清单

### 第二步：逐件分析
对每份文件进行分析：
1. 判断文件类型
2. 确定形成部门
3. 确定年度归属
4. 判断事由归属

### 第三步：事由归类
将同一事由的相关文件归为一组，形成拟组卷方案

### 第四步：分类编号
为每个案卷确定分类号，编制档号

### 第五步：输出方案
生成完整的组卷方案，包括：
- 案卷标题
- 分类号
- 卷内文件目录
- 保管期限
- 档号

## 组卷原则

### 事由原则
同一事由的文件材料应归入同一案卷，保持文件的有机联系。

### 卷内文件排列
1. 批复在前，请示在后
2. 正件在前，附件在后
3. 印件在前，定稿在后
4. 按时间顺序排列（同类型文件）

### 年度归属
1. 一般文件归入形成年度
2. 跨年度的请示与批复，归入批复年度
3. 跨年度的规划，归入针对的第一年
4. 跨年度的总结，归入针对的最后一年
5. 跨年度的会议文件，归入会议开幕年度

### 保管期限区分
- 不同保管期限的文件应分别组卷
- 同一案卷内文件保管期限应相同或相近

## 保管期限判定参考

### 永久保管
- 学校重大决策、重要会议文件
- 机构设置、编制、人事任免文件
- 重要规章制度
- 学校发展规划、年度工作计划总结
- 重要合同协议
- 上级针对本校的重要文件

### 30年保管
- 一般性规章制度
- 常规工作计划、总结
- 人事考核、奖惩材料
- 统计报表
- 一般性请示批复

### 10年保管
- 事务性、参考性文件
- 一般性通知
- 普发性的上级文件

## 使用说明

1. **输入**：提供待整理的文件题目清单，可附带文件的基本信息（文号、日期、责任者等）

2. **处理**：技能将按照分类规则和组卷原则进行分析，输出组卷方案

3. **输出**：
   - 按案卷分组的文件清单
   - 每个案卷的分类号、保管期限、档号
   - 卷内文件目录

4. **人工确认**：组卷方案供档案工作人员参考，最终确定需人工审核

## 注意事项

1. 本技能依据《高等学校档案实体分类法》和《高等学校档案工作规范》设计
2. 各学校可根据本校实际情况调整二级类目设置
3. 保管期限判定需结合学校保管期限表和文件实际价值
4. 涉密文件需按保密规定单独处理
5. 电子档案、声像档案、实物档案有特殊整理要求
