## 📖 简介

通用互动游戏框架，支持多种游戏类型，可安装到任何 OpenClaw Agent。

### 支持的游戏类型

| 类型 | 说明 | 状态 |
|------|------|------|
| 文字冒险 | 多题材剧情游戏（武侠/都市/奇幻/科幻/历史） | ✅ 完成 |
| 猜谜游戏 | 传统谜语/脑筋急转弯/知识问答 | ✅ 完成 |
| 剧情生成 | 动态生成剧情、对话、场景 | ✅ 完成 |
| 存档系统 | JSON 格式存档/读档 | ✅ 完成 |

## 💻 代码示例

### Node.js 使用

```javascript
const { InteractiveGames } = require('./src/index');

const games = new InteractiveGames();

// 启动文字冒险游戏
const adventure = games.startGame('adventure', { theme: '历史穿越' });
console.log(adventure);

// 处理玩家输入
const result = games.handleInput('选择 2');
console.log(result);

// 保存游戏
games.saveGame('slot1');

// 读取游戏
games.loadGame('slot1');
```

### OpenClaw Agent 集成

```javascript
// 在 Agent 中调用
const games = new InteractiveGames();

// 响应用户命令
if (message.includes('玩游戏')) {
    const game = games.startGame('adventure');
    sendReply(game);
}
```

## 🎨 扩展开发

### 添加新游戏类型

1. 在 `src/` 目录创建新模块
2. 实现以下方法：
   - `start()` - 启动游戏
   - `processInput(input)` - 处理用户输入
   - `getState()` - 获取游戏状态

3. 在 `index.js` 中注册新游戏类型

### 添加新题材

修改 `adventure-game.js` 中的 `getIntro()` 方法，添加新题材介绍。
