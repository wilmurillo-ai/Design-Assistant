# Changelog
All notable changes to this skill will be documented in this file.

## v1.0.0 (2026-04-09)
### ✨ Features
- Initial release of Data Analysis for Feishu skill
- Support 12+ chart types: Line, Area, Bar, Stacked Bar, Pie, Donut, Gauge, Radar, Scatter, Funnel, Waterfall, Dual Axis
- Support 6+ data sources: Excel, CSV, Feishu Bitable, Feishu Sheet, Markdown Table, Raw data
- AI-powered intelligent features: auto chart recommendation, auto data cleaning, auto analysis report generation
- Ultra HD 2x Retina rendering, perfect Feishu compatibility
- Dual mode support: screenshot mode (100% compatible) and interactive card mode
- One-click test script, out-of-the-box experience
- Complete documentation and usage examples
- MIT License, open source free for commercial use

### 🚀 What's Next
- Feishu workflow integration (auto send to groups/documents, scheduled reports)
- Custom theme support
- More chart types: heatmap, tree chart, sankey diagram
- Advanced analysis: trend prediction, anomaly detection
- Multi-language support
