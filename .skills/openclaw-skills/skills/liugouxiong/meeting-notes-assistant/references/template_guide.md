# Word 模板规范

## 简洁版 (simple.docx)

### 结构
- 标题：会议纪要（居中）
- 会议信息：时间、主题、参会人
- 议题与结论：编号列表
- 待办事项： bullet list
- 关键词：逗号分隔
- Action Items：3列表格（任务、负责人、截止时间）

### 样式
- 标题：16pt, 宋体, 加粗
- 正文：12pt, 宋体
- 无背景色，无水印

---

## 专业版 (professional.docx)

### 结构
- 公司 Logo（顶部居中，宽度 1.5 inch）
- 标题：会议纪要（居中，18pt）
- 副标题：日期
- 会议信息卡片（带 emoji 前缀）
- 议题与结论（编号，结论加粗）
- 待办事项（checkbox 样式）
- 关键词（tag 样式）
- Action Items 表格（4列：序号、任务、负责人、截止时间）
- 页脚： Generated by Meeting Notes Assistant

### 样式
- 标题：18pt, 黑体, 加粗
- 副标题：12pt, 灰色
- 结论：加粗
- 表格：网格线，首行浅灰背景
- 页脚：9pt, 灰色

---

## 自定义模板 (custom.docx)

用户提供的 `.docx` 模板可直接保留原有版式，只需在文档中放入占位符：

| 占位符 | 说明 |
|--------|------|
| {{meeting_time}} | 会议时间 |
| {{meeting_theme}} | 会议主题 |
| {{meeting_title}} / {{title}} | 会议标题别名 |
| {{attendees}} | 参会人 |
| {{summary}} | 会议摘要 |
| {{topics}} | 议题与结论（多行文本） |
| {{todos}} | 待办事项（多行文本） |
| {{keywords}} | 关键词 |
| {{action_items}} | Action Items 三列表格 |
| {{action_items_text}} | Action Items 文本版（适合页眉/页脚等不方便放表格的位置） |
| {{raw_text}} | 原始转写文本 |

### 使用方式

```bash
python scripts/export_word.py notes.json --template custom --template-path my-template.docx --output custom-notes.docx
```

### 使用建议

1. `{{action_items}}` 建议单独放在正文段落或表格单元格中，脚本会替换成 3 列表格。
2. 若模板位置在页眉 / 页脚，建议改用 `{{action_items_text}}`，避免页眉页脚区域表格兼容性问题。
3. `{{topics}}` / `{{todos}}` 会按多行文本写入，适合放在普通段落中。
