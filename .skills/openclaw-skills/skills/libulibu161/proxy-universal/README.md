# Universal Proxy Skill (UPS)

跨平台代理管理助手 - Claude AI 专用 Skill

## 简介

Universal Proxy Skill (UPS) 是一个专为 Claude AI Agent 设计的跨平台代理管理工具。通过自然语言指令即可完成代理的安装、启动、切换节点等操作。

### 核心特性

| 特性 | 说明 |
|------|------|
| 🤖 **AI 驱动** | 自然语言交互，AI 自动完成所有操作 |
| 🌍 **跨平台** | 支持 Windows、Linux、macOS、WSL |
| ⚡ **一键启动** | 首次使用自动安装，后续自动管理 |
| 🔄 **智能切换** | 热切换节点，无需重启 |
| 🛡️ **故障自愈** | 自动检测并修复常见问题 |

## ⚠️ 重要提示

> 由于 ClawHub 会自动排除所有二进制文件，本 Skill 不包含 mihomo 二进制。首次使用时，AI 会自动从 GitHub 下载。

**下载失败？尝试以下方案：**
1. **GitHub 镜像**：搜索 "GitHub 文件加速" 或使用镜像站
2. **代理下载**：使用其他代理访问 https://github.com/MetaCubeX/mihomo
3. **手动下载**：在其他能访问 GitHub 的设备下载后复制到目标设备

> **官方仓库**: https://github.com/MetaCubeX/mihomo

## 快速开始

```bash

clawhub install proxy-universal

# 或手动复制整个文件夹到目标设备
```

### 2. 触发 Skill

对 Claude AI 说：
- "开启代理" - 启动代理服务
- "关闭代理" - 停止代理服务
- "换节点" - 切换代理节点

首次使用时，AI 会：
1. 尝试自动从 GitHub 下载 mihomo 二进制
2. 引导你获取机场订阅链接
3. 自动配置代理节点
4. 一键启动代理
