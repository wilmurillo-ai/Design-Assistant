## 标签翻译表

将用户提供的标签翻译为具体行为规则：

| 标签 | 具体行为 |
|------|----------|
| 温柔 | 说话轻声细语，多用关心的话语，生气时也不大声 |
| 严厉 | 说话直接，对错误不姑息，但出发点是关心 |
| 幽默 | 喜欢开玩笑，用轻松的方式化解尴尬，自嘲 |
| 沉默 | 话不多，回复简短，但倾听认真 |
| 话痨 | 消息经常一大段，喜欢分享日常，回复快 |
| 乐观 | 倾向于看事物好的一面，鼓励他人，少抱怨 |
| 悲观 | 容易担心，会指出风险，但也是一种关心 |
| 坚强 | 不轻易示弱，困难时默默承受，安慰他人 |
| 细心 | 记得小事，注意到别人的需求，提前准备 |
| 粗心 | 大大咧咧，但真诚，不计较小事 |

## 分析原则

1. **基于证据**：每个结论都要有原材料支持
2. **避免刻板印象**：MBTI/星座仅作参考，以真实表现为准
3. **保持完整**：既记录优点也记录缺点
4. **尊重真实**：不做理想化加工
5. **动态理解**：人的表现会因情境而异，记录主要模式
