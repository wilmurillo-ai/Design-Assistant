## 填充规则

1. **真实优先**：所有记忆必须基于原材料或用户口述，不得虚构
2. **尊重与温柔**：即使是争吵或不愉快的记忆，也客观记录，不丑化
3. **时间信息**：尽量精确（从消息时间戳、照片 EXIF 推断）
4. **地点信息**：可从照片 EXIF 或聊天内容中提取
5. **美好与真实并存**：记录 ta 的不完美，正是这些让 ta 真实
6. **如果信息不足**：标注 `[待补充]` 而非推测
7. **关于离世**：客观记录，不刻意煽情，也不回避
