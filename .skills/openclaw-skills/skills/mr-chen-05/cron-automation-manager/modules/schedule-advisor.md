# Schedule Advisor Module

Purpose: suggest optimal scheduling times for common automation tasks.

## Recommendations

News monitoring:
- morning summary: 09:00
- midday update: 12:00

GitHub monitoring:
- afternoon: 18:00

Trend analysis:
- evening: 21:00

Weekly reports:
- Sunday evening

## Behavior

If user does not specify a schedule:

1. detect task category
2. propose recommended schedule
3. ask user to confirm
