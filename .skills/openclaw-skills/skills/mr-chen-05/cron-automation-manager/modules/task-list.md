# Task Listing Module

Provides ability to inspect current automation jobs.

## Capabilities

- list all cron jobs
- show schedule
- show delivery channel
- show last run status

## Typical user requests

- "show my cron jobs"
- "what scheduled tasks do I have"
- "list automation"

## Output Format

Task Name
Schedule
Delivery Channel
Last Run Status

Example:

AI News Radar
0 12 * * *
Feishu
Last run: success
