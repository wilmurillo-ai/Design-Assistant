# Knowledge Index Template

Use this template when creating L2 knowledge index entries.

## Template Format

    ## Knowledge Index - [Domain/Folder Name]

    **Last Updated**: [ISO-8601 timestamp]
    **Total Files**: [count]

    ### File Mapping

    | File Path | Theme | Keywords | Size |
    |-----------|-------|----------|------|
    | path/to/file1.md | Brief theme | kw1, kw2 | S/M/L |
    | path/to/file2.md | Brief theme | kw3, kw4 | S/M/L |

    ### Quick Lookup

    - Topic A: file1.md, file3.md
    - Topic B: file2.md

## Size Indicators

- S (Small): < 100 lines
- M (Medium): 100-500 lines
- L (Large): > 500 lines

## Example

    ## Knowledge Index - AI Observability

    **Last Updated**: 2026-03-22T14:30:00Z
    **Total Files**: 8

    ### File Mapping

    | File Path | Theme | Keywords | Size |
    |-----------|-------|----------|------|
    | output/产品演进规划.md | Product roadmap | evolution, roadmap | M |
    | output/技术方案.md | Technical design | architecture, trace | M |

    ### Quick Lookup

    - Product Strategy: 产品演进规划.md
    - Technical Details: 技术方案.md
