## 关键 Insight — 2:30-3:30

[录屏：画一个简单示意图，或者用 Excalidraw]

**Z**: 这是整个视频最重要的一个点。

> **Whisper = 听写机器人，只听不懂**
> **Gemini = 一个懂编程的人在帮你记录**

**Z**: Whisper 是 ASR——自动语音识别。它做的是声波到文字的机械转换。所以 Claude 会变成 Cloud，OpenClaw 变成 Open Cloud。

Gemini 是 LLM——大语言模型。它理解 Claude Code 是一个编码工具，它知道 React 和 async 是什么。所以标点完美，术语精准，中英切换自然。

而且虾BB 有一个自定义词库——40+ AI 工具名称预置。你说 "Cursor"，它不会给你 "课色"。你说 "OpenClaw"，它不会变成 "Open Cloud"。
