## 关键 Insight — 为什么用 LLM 而不是 ASR (1:30-3:00)

[画面：白板/示意图]

**Z**: 这是整个视频最重要的一个 insight，你记住这一点就行。

[画面：大字幕]

**Whisper = 听写机器人，只听不懂**
**Gemini = 一个懂语言的人在帮你记录**

**Z**: Whisper 是 ASR（自动语音识别），它做的是声学匹配——把声波波形映射到文字。所以它没有标点，没有语义纠错，中英切换容易乱。

Gemini 是一个大语言模型。它不是在"识别"你说的话，它是在"理解"你说的话，然后输出文字。所以它有完美的标点，能纠正口音错误，中英文切换自然。

[画面：对比表]

| | Whisper | Gemini |
|---|---|---|
| 标点 | 没有 | 完美 |
| 错别字 | 多 | 少 |
| 中英混搭 | 一般 | 很好 |
| 模型大小 | 3GB | 0（云端） |
| 速度 | 3-5秒 | 1-2秒 |
| 价格 | 免费但要本地跑 | 免费250次/天 |

**Z**: 而且 Gemini 的免费 API，理论上支持最长 9.5 小时的单次音频。我自己测过好几分钟的连续录音，完全没问题。MacWhisper 长音频经常出错。

## 踩坑记录 — 真实的痛 (7:00-10:00)

[画面：Z 表情切换到"回忆痛苦"]

### 坑1：权限地狱 (1分钟)

**Z**: CGEventTap 需要 Accessibility 权限。但是！当你把 app 双击打开的时候，macOS 死活不给权限。不报错，不崩溃，就是——Globe 键没反应。

[画面：权限设置界面]

**Z**: 我调了快两个小时。最后发现，只有从 Terminal.app 启动的进程才能继承 Terminal 的 Accessibility 权限。所以我做了一个 hack——app 检测到没权限，就自动打开一个 Terminal 窗口来运行自己，然后把 Terminal 窗口最小化。

[画面：Z 苦笑]

**Z**: 对，就是这么 hacky。但它 works。

### 坑2：Event Tap 被系统偷偷关掉 (30秒)

**Z**: 更坑的是，macOS 如果觉得你的 Event Tap 回调太慢，会直接把它 disable 掉。不通知你。

解法：加了一个 watchdog，每5秒检查一次，死了就重新启用。

### 坑3：Flash-Lite 把中文翻译成英文 (30秒)

**Z**: 我一开始用的是 Gemini Flash-Lite 模型，因为便宜、额度多。结果我说中文，它给我翻译成英文输出。

[画面：错误输出截图]

**Z**: 换回 Flash，解决。有些钱不能省。

### 坑4：API Key 暴露在 URL 里 (30秒)

**Z**: 安全审计的时候发现——API Key 拼在 URL query string 里。这意味着任何代理、日志都能看到你的 key。

正确做法：放在 HTTP Header 里。这种细节 AI 帮你写代码的时候很容易忽略，一定要做 security review。
