# Changelog

## v1.3.0 (2026-04-02)

### 新功能 / New Features
- **Smart Modes 智能模式** — 录音时按快捷键切换 AI 处理模式
  - `Globe + `` — 翻译模式：说中文出英文，说英文出中文
  - `Globe + 1` — Prompt 优化：口述想法，自动生成专业 AI 提示词
  - `Globe + 2` — 邮件生成：口述内容，自动生成格式规范的邮件
  - `Globe + Esc` — 取消录音
  - 再按同一键可切换回普通听写模式
- **HUD 模式指示器** — 录音时右侧显示当前模式（emoji + 文字）
- **更新检查** — 启动时自动检查新版本，菜单栏可手动检查，有更新一键跳转下载
- **反馈入口** — 菜单栏"反馈意见"按钮，直接发邮件
- **API 错误友好提示** — 断网、超时、API Key 失效等错误显示中文提示，不再显示原始错误

### 改进 / Improvements
- Prompt 模式：忠实保留用户描述的主体信息（性别、对象、场景），不擅自添加或篡改细节
- Prompt 模式：强制保持输入语言，中文输入不会变成英文输出
- 所有模式支持自定义词典（Claude/OpenClaw 等专有名词不会被错误识别）
- HUD 文字过长时自动为模式标签留出空间，不会被文字覆盖

### 修复 / Bug Fixes
- 修复 HUD 模式徽章被窗口边界裁剪不可见的问题
- 修复模式切换按键（`, 1, 2）会输入到文本框的问题
