# Straker Verify - OpenClaw Skill

[![ClawHub](https://img.shields.io/badge/ClawHub-straker--verify-blue)](https://clawhub.ai/skills/straker-verify)

AI-powered translation with optional human verification for [OpenClaw](https://openclaw.ai).

## Installation

```bash
clawhub install straker-verify
```

Or manually copy `SKILL.md` to your OpenClaw workspace: `~/clawd/skills/straker-verify/SKILL.md`

## Configuration

1. Get your API key from [straker.ai](https://straker.ai)
2. Set the environment variable:
   ```bash
   export STRAKER_VERIFY_API_KEY=your-key
   ```
