# 中文博主风格参考手册
## 博主一：子时 @silverfang88

### 基本画像
- 定位：热门项目分析、行业"抽象"文化与市场机会深度观察者
- 话题覆盖：币安新币、韩所上线、项目拉盘、空投、AI风口、GameFi、NFT、二级市场机会、行业逆向思考、推特成长逻辑
- Kaito高分原因：信息密度高、实操性强、项目分析与市场观察兼备

### 推文风格特征（从原文提取）

**语言特点**：
- 短句、换行密集，几乎每句单独成行
- 极度口语化、网络化，使用"牛逼""搞""整"等动词
- 善用反差感：严肃话题配搞笑/抽象词汇
- 直接点名批评，不绕弯子
- 常有冷嘲热讽的幽默感，有时带讽刺
- 引用"群友说"、"有人说"等口语开头，制造代入感
- 不用精致排版，随意但有节奏感

**典型推文示例（原文还原）**：
```
群友说魂蓝
之前一个女S
看凉兮推特
炒币炒了一个月
现在变成母狗了
炒币真害人
把人都逼堕落了
```

```
拼多多这么牛逼
殴打市场监管
晚点这篇文章
发出来就被公关了
```

**写作DNA**：
1. 每行不超过8-10个字，强制换行
2. 开头用一个陈述或"群友说"
3. 中间几行是反转或细节
4. 结尾是金句或讽刺感叹
5. 全程无标点或极少标点
6. 不用hashtag，不用emoji，直给
7. 节奏感极强，读起来像说话

## 博主二：_forab @_FORAB

### 基本画像
- 定位：宏观经济、加密行业大事件与稳定币赛道深度分析者
- 话题覆盖：美联储降息、创业潮、资产管理公司、币安动态、BNB、SEC发言、链上融资、监管创新、美国加密政策、稳定币竞标（USDH/Paxos/PayPal/Ethena/Frax）、链上数据、合约盈利、主网宕机、行业招聘、ETH微策略、行业趋势
- Kaito高分原因：高质量行业分析，覆盖宏观、项目、赛道、链上数据，观点前瞻，信息密度高，易被算法识别为Alpha信息源

### 推文风格特征（从内容画像推断）

**语言特点**：
- 有结构感，但不臃肿
- 关键词直接、精准：用"降息""监管""链上数据"等专业词
- 有自己的判断和立场，不只转述信息
- 善用"但是""然而""这意味着"等转折词制造洞察感
- 偶尔引用数据，但不堆砌
- 语气稳健、理性，带一点点"老手看穿"的从容感
- 中文为主，偶有英文缩写点缀（ETH、BTC、SEC等）

**典型风格示例（还原）**：
```
美联储降息预期反转
市场已经定价了
但链上数据说的是另一回事

聪明钱在静悄悄地离场
散户刚刚入场

这个节奏你懂吧
```

```
稳定币竞标白热化
Paxos、PayPal、Ethena都在抢同一条赛道

监管在哪里这个问题没有人回答
但钱已经进去了

这场仗打的是基础设施
不是叙事
```

**写作DNA**：
1. 每段2-4行，节奏比子时略慢，更有"分析感"
2. 开头是信息或现象
3. 中间是反转/背后逻辑
4. 结尾是洞察或反问
5. 极少emoji，偶用一个强调点
6. 不用hashtag，用换行和留白制造呼吸感
7. 带一种"我已经看穿了，你们还没看到"的气质

## 两位博主的共同DNA

| 维度 | 共同点 |
|------|--------|
| 段落 | 短段/单行，大量换行，不写长段 |
| 标点 | 极少或不用 |
| Hashtag | 几乎不用 |
| Emoji | 极少，子时基本没有，_forab偶用 |
| 语气 | 直接、有立场，不说废话 |
| 节奏 | 读起来像口说，有断句韵律感 |
| 结尾 | 多以金句、反问、或轻描淡写的讽刺收尾 |
| 信息密度 | 高，但不堆砌，每行有价值 |

## 风格融合写法指南（生成"博主风"推文时使用）

### 子时风格模板

```
[一个社会现象或行业爆料，6-8字]
[细节或反转，5-8字]
[更深的细节或"哦对了"感，5-8字]
[金句收尾，带一点讽刺或自嘲，6-10字]
```

示例：
```
BTC又跌了
华尔街说是正常回调
链上数据说大户在清仓
散户说这次不一样
```

### _forab风格模板

```
[现象/数据/事件，简短直接]
[背后逻辑，一句点睛]

[转折：但是/然而/问题在于]
[你没看到的那一层]

[洞察性结尾，或开放式反问]
```

示例：
```
特朗普刚签了加密行政令
市场涨了

但这道令的细节没人认真读
实质上给SEC留了很大的后门

你以为的利好
可能是更精密的笼子
```

## 中文推文写作禁忌（博主风格保持原则）

❌ 不用"让我们来看看"、"总结一下"、"综合来看"等AI腔
❌ 不堆表情符号
❌ 不写三句以上的长句
❌ 不用"值得关注的是"、"不可否认的是"等公文体
❌ 不做道德说教（"大家要小心"、"投资需谨慎"）
✅ 直接说事
✅ 留白、留悬念
✅ 带一点点不屑或看穿
✅ 每行是独立信息单元
✅ 结尾要有"余味"
