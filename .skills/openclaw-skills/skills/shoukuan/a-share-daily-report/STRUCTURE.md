## 🎯 两版定位对比

| 维度 | 早报预测版 | 晚报复盘版 |
|------|-----------|-----------|
| **生成时间** | 盘前 8:00前 | 盘后 18:00后 |
| **核心** | 预测 + 策略 | 复盘 + 事实 |
| **重点** | 昨夜今晨数据 + 今日展望 | 今日完整数据 + 明日展望 |
| **特色** | 自选股简洁预测 + 期指指引 | 自选股复盘 + 事件日历 |
| **前3章节** | 30秒总览 → 自选股预测 → 核心决策 | 30秒总览 → 自选股复盘 → 今日复盘 |

---

## 📝 数据处理规则

1. **非交易日处理**：自动使用最后一个交易日的数据
2. **数据获取失败**：对应模块显示「数据获取失败」，不硬编码默认值
3. **关联影响分析**：基于真实龙头股涨跌幅动态生成
4. **自选股**：支持从配置文件读取真实持仓
5. **要闻数量**：明确10篇，保持稳定性
