# Changelog

All notable changes to this project will be documented in this file.

## [0.0.1] - 2026-03-11

### Added
- Initial release of Pollinations Sketch Note
- AI-powered sketch note generation with auto-search and summary
- 3 artistic styles (Minimalist, Cute Illustration, Cyberpunk)
- Cross-platform support (macOS, Linux, Windows)
- Environment variable configuration for API keys
- Bilingual documentation (Chinese & English)
- Automatic signature and timestamp on generated cards

### Features
- Auto-search from Wikipedia and Baidu Baike via Tavily API
- AI summary compression to 180-200 characters
- Standard 804×440 pixel output size
- Random style rotation for variety
- OpenClaw integration with voice triggers

### Security
- No hardcoded API keys
- Environment variable-based configuration
- Secure API key isolation

### Documentation
- Complete README.md with quick start guide
- Detailed INTRO.md with full usage instructions
- Multi-platform environment setup guides
- SEO-optimized bilingual descriptions
