# 封面图设计规范

> 版本：v1.0
> 使用工具：Python PIL

---

## 封面图标准

### 尺寸要求
- 分辨率：900x600px（3:2比例）
- 格式：PNG或JPEG
- 文件大小：不超过20MB

### 设计风格

#### 简约大气风格（推荐）
```
- 纯色背景 + 主题文字
- 配色：深色背景 + 亮色文字
- 边框装饰：2-3px描边
```

#### 渐变风格
```
- 渐变背景（从深到浅或双色渐变）
- 文字居中
- 装饰元素（圆形、线条等）
```

---

## 配色方案

### 情感类话题
- 背景：深蓝紫色 #1a1a2e
- 强调色：玫瑰粉 #e94560
- 文字：白色/金色

### 社会热点话题
- 背景：深灰黑 #1a1a1a
- 强调色：橙红色 #ff6b6b
- 文字：白色

### 职场/赚钱话题
- 背景：深蓝 #0a1628
- 强调色：金色 #ffd700
- 文字：白色

## Python代码示例

```python
from PIL import Image, ImageDraw, ImageFont

# 创建封面
img = Image.new('RGB', (900, 600), color=(26, 26, 46))
draw = ImageDraw.Draw(img)

# 渐变背景
for i in range(600):
    r = int(26 + (100 - 26) * (i / 600))
    g = int(26 + (80 - 26) * (i / 600))
    b = int(46 + (150 - 46) * (i / 600))
    draw.line([(0, i), (900, i)], fill=(r, g, b))

# 边框
```
