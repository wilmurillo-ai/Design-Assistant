## 各Agent职责

### Agent-Hunter（热点猎手）
- 监控微博热搜、抖音热榜、头条热榜、知乎热榜
- 采集阅读量、讨论量等数据
- 每日9:00输出选题清单，评分排序

### Agent-Writer（内容创作者）
- 根据选题创作微头条/文章
- 优化标题，生成标签建议
- 自检违禁词和合规性

### Agent-Designer（视觉设计师）
- 设计封面图（900x600px）
- 生成配图素材
- 使用Python PIL绘图

### Agent-Publisher（发布执行）
- 浏览器自动化发布
- 数据追踪（展现量、阅读量、点赞）
- 效果复盘

---

## 工作流程

```
06:00  Agent-Hunter 启动热点扫描
09:00  输出每日热点报告
10:00  Agent-Writer 开始创作
12:00  提交审核队列 → 通知用户
15:00  Agent-Designer 生成封面
16:00  Agent-Publisher 执行发布
18:00  数据追踪开始
次日   效果复盘 + 新一轮热点扫描
```

---

## 内容质量标准

- 标题：8-30字，有吸引力
- 正文：逻辑清晰，有观点
- 标签：3-5个相关标签
- 配图：高清、原创、符合内容
- 审核通过后才能发布
