# 今日头条平台内容规范

> 版本：v1.0
> 适用：微头条创作与发布

---

## 标题规范

### ✅ 合格标题特征
- 字数：8-30字最佳
- 包含关键词（利于推荐）
- 引发好奇心或共鸣
- 避免过度夸张、虚假、诱导

### ❌ 禁用标题类型
- 过度夸张词："震惊！"、"吓死了！"
- 虚假信息："刚刚发生，99%的人都不知道..."
- 诱导点击："不看后悔一辈子..."

### 📝 标题公式
```
公式1：数字+悬念 → "3个技巧，让你的收入翻倍，第2个最关键"
公式2：痛点+方案 → "还在加班？学会这招准时下班"
公式3：热点+观点 → "某明星事件背后，暴露了3个职场真相"
公式4：对比+反差 → "月薪3000和30000的人，差的不只是钱"
```

## 正文规范

### 长度标准
| 内容类型 | 字数范围 |
|----------|----------|
| 微头条 | 100-500字 |
| 长文 | 800-3000字 |

### 内容结构（微头条）
1. **开头（50字内）**：抛出一个问题、现象或观点
2. **正文（200-400字）**：分点论述或故事展开
3. **结尾（50字内）**：总结观点+引导互动

---

## 内容禁区（⚠️ 严格遵守）

### 🔴 政治敏感
- 不讨论领导人、政党、政治事件
- 不涉及领土主权、港澳台等敏感话题

### 🔴 违法违规
- 不宣扬黄赌毒
- 不传播谣言、虚假信息
- 不涉及金融诈骗、传销等

### 🔴 低俗内容
- 不包含色情、暴力内容
- 不使用低俗语言
