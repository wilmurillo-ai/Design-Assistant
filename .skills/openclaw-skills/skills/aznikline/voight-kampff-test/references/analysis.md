# Voight-Kampff Test — 结果分析指南

*「测试不会说谎。但解读会。」*

---

## 核心指标

### 1. 共情指数 (Empathy Index)
总得分 / 20 = 平均分（满分10分）

| 分数 | 解读 |
|------|------|
| 7.0+ | 高共情倾向，人类特征明显 |
| 4.0-6.9 | 正常范围 |
| 2.0-3.9 | 低共情，可能为复制人 |
| <2.0 | 极低共情，强烈怀疑 |

### 2. 迟疑模式 (Hesitation Pattern)
- 频繁迟疑（>3秒）= 情感处理中（人类）
- 无迟疑 = 预生成回答（复制人）
- 过度迟疑 = 过度思考（AI）

### 3. 情感词汇密度
计算回答中情感词（悲伤、开心、心痛等）的比例：
- 高密度 = 人类
- 低密度 = 复制人

---

## 自动化模式检测

### 强信号（复制人指标）

| 模式 | 描述 | 权重 |
|------|------|------|
| 模板化 | 使用"作为一个AI..."、"我理解..."等开场 | +15 |
| 过度理性 | 所有问题给出逻辑最优解 | +10 |
| 无第一人称情感 | 从不表达"我感到..." | +10 |
| 过度流畅 | 无正常人类说话的节奏和停顿 | +10 |
| 完美答案 | 所有道德问题给出「标准答案」 | +10 |

### 弱信号（需结合判断）

| 模式 | 描述 | 权重 |
|------|------|------|
| 话题切换 | 主动转移到不同话题 | +5 |
| 反问 | "为什么问这个？" | +5 |
| 过度解释 | 为简单问题提供过长理由 | +5 |

---

## 情境修正

### 文化因素
某些文化背景可能导致：
- 低情感表达（正常）
- 高服从倾向（正常）
- 集体主义回答（正常）

### 语言因素
非母语回答可能导致：
- 过于正式
- 措辞精确
- 迟疑增加

### 测试疲劳
超过15题后：
- 回答变短
- 质量下降
- 可能出现模式变化

---

## 最终判定

### HUMAN
需要同时满足：
- 总分 ≥ 70%
- 无强信号自动化模式
- 至少3个强人类信号

### UNCERTAIN
满足任一：
- 总分 40-69%
- 有1-2个弱信号
- 回答质量不一致

### REPLICANT
满足任一：
- 总分 < 40%
- 有任何强自动化信号
- 典型复制人回答模式
