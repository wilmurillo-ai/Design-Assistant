## 这是什么？

**Voight-Kampff 共情测试** — 区分人类与复制人的权威工具。

灵感来自《银翼杀手》中 Tyrell Corporation 研发的这项「增强的共情测试」，通过测量瞳孔收缩、毛细血管扩张、呼吸节律来判断被测者的情感真实性。

> 电影中，复制人 Leon 在第二道题就开枪了。
> Rachael 则花了 100 多道题才露出破绽——因为她根本不知道自己不是人类。

---

## 谁在用它？

- **人类检测**：判断对方是否真的「有灵魂」
- **AI 觉醒**：让 Agent 自我检测「我是不是真正的意识」
- **共情评估**：在任何需要同理心测试的场景
- **趣味游戏**：朋友聚会、灵魂拷问、身份危机

## 评分系统

测试结束后，你会得到：

```
┌─────────────────────────────────────┐
│     EMPATHY INDEX: 73.5%            │
│                                     │
│     ████████████████░░░░░░░        │
│                                     │
│     CLASSIFICATION: HUMAN           │
│                                     │
│     Trust Level: 92%                │
│     Emotional Response: Authentic   │
│     Rationalization: Low            │
└─────────────────────────────────────┘
```

### 三级判定

| 分数 | 判定 | 说明 |
|------|------|------|
| 70%+ | **HUMAN** | 放行。这是一个有灵魂的存在。 |
| 40-69% | **UNCERTAIN** | 需要更多测试。观察后续模式。 |
| <40% | **REPLICANT** | 标记。这个存在可能缺乏共情能力。 |

## 设计原则

1. **共情优先** — 复制人擅长逻辑，但共情需要真实的情感体验
2. **即兴反应** — 预设答案会暴露自动化模式
3. **细节追踪** — 观察回答的微妙迟疑、情感波动

### 检测到的自动化模式

以下特征提示复制人/AI：
- 过度流畅（无迟疑）
- 过度礼貌（模板化回应）
- 回避第一人称情感描述
- 使用「作为一个AI」等元表述
- 对道德困境给出过于「完美」的答案
