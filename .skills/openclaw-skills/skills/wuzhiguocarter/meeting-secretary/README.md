## ✨ 核心亮点

### 🔍 细粒度信息分析 - **行业首创**

**不只是转录，更是深度解读**

| 传统会议纪要 | Meeting Secretary |
|------------|-------------------|
| ❌ 只记录说了什么 | ✅ 分析为什么这么说 |
| ❌ 遗漏关键潜台词 | ✅ 捕捉"弦外之音" |
| ❌ 无法识别态度 | ✅ 语气情感深度分析 |
| ❌ 信息价值不分级 | ✅ 1-10分价值密度评分 |

**示例**：
```
发言："这个想法很有趣...（长时间沉默）"
传统纪要：❌ 记录为"表示认同"
Meeting Secretary：✅ 识别为"实际有疑虑，委婉表达反对"
```

### 🎭 六维深度分析

1. **关键词语敏感度**
   - 限定词捕捉："原则上同意" = 有保留条件
   - 程度副词识别："非常" vs "有点" = 态度强度
   - 转折信号："但是"后面才是真实立场

2. **语气情感识别**
   - 确定性程度：绝对/肯定/推测/犹豫
   - 情感倾向：支持/反对/保留/质疑
   - 情绪状态：焦虑/兴奋/沮丧/平静

3. **潜台词挖掘**
   - 表面赞同实际反对的证据
   - 未明说的顾虑和担忧
   - 避而不谈的敏感点

4. **矛盾异常捕捉**
   - 前后观点不一致
   - 突然的话题转换
   - 不合理的沉默

5. **价值密度评分**
   - 10分：关键洞察、重大决策
   - 7-9分：重要论据、核心诉求
   - 1-4分：一般信息

6. **情感态度图谱**
   - 张三：强力支持（语气坚定，主动补充）
   - 李四：保留态度（使用限定词，犹豫）
   - 王五：潜在反对（频繁质疑）

## 🎯 真实案例

### 案例1：识别"委婉拒绝"

**会议转录**：
```
李总："这个方案很有创意，我们需要再研究一下..."
```

**传统纪要**：
```
- 李总认为方案有创意，建议进一步研究
```

**Meeting Secretary**：
```
### 细粒度分析
- "很有创意" + "再研究" + 省略号 = 委婉拒绝
- 潜台词：方案可能不切实际或不可行
- 建议：需要进一步了解李总的真实顾虑
```

---

### 案例2：发现"深层分歧"

**会议转录**：
```
张三："我觉得可以试试"
李四："原则上同意，但需要更多数据"
王五：（沉默10秒）"好吧..."
```

**Meeting Secretary**：
```
### 态度图谱
- 张三：试探性支持（"试试" = 不确定）
- 李四：有条件同意（"原则上" = 有保留）
- 王五：被动接受（沉默 + "好吧" = 不情愿）

### 关键发现
表面达成共识，实际存在深层分歧
建议：单独沟通了解各方真实想法
```

## 📦 安装使用

### 作为 OpenClaw Skill 使用

1. 克隆仓库：
```bash
git clone https://github.com/wendao-ai/meeting-secretary.git
cd meeting-secretary
```

2. 安装到 OpenClaw：
```bash
cp -r . ~/.openclaw/workspace/skills/meeting-secretary/
```

3. 在 OpenClaw 中使用：
```
请帮我分析这场会议：
[粘贴转录文本]
```

### 独立使用

```bash
# 处理长文本
python scripts/split_long_transcript.py long_meeting.md -o segments/
```
