# 定积课程长图生成器 - 使用说明

## 🎯 Skill 概述

**Skill 名称**: dingji-long-image  
**中文名称**: 高等数学课程知识章节长图生成器  
**版本**: v2.0  
**触发词**: 生成课程长图、制作知识长图、章节总结图

---

## 📐 六个步骤流程

本 Skill 按照**教学设计六步法**生成高数知识长图：

### Step 1: 学习目标确定 🧠

根据章节内容，确定本章的学习目标层次：

| 层级 | 目标 | 示例 |
|------|------|------|
| 记忆/理解 | 基本概念、公式、定理 | 背诵极限定义、理解导数几何意义 |
| 应用 | 运用方法解决常规问题 | 用洛必达法则求极限 |
| 分析 | 识别结构、比较解法 | 比较不同积分方法优劣 |
| 评价/创造 | 判断解法、构造新题 | 设计新题型、评价证明严谨性 |

### Step 2: 知识点拆解 📚

将章节拆解为 5-7 个核心 PART：

| PART | 内容领域 | 说明 |
|------|---------|------|
| PART 1 | 概念建立 | 核心定义 + 几何直观 + 关键词 |
| PART 2 | 公式与定理 | 主要公式（LaTeX）+ 定理条件 |
| PART 3 | 计算方法 | 解题套路 + 步骤 + 适用条件 |
| PART 4 | 典型例题 | 2-3 道分级例题（基础/提高/综合）|
| PART 5 | 常见错误 | 错误类型 + 正确做法 |
| PART 6 | 核心口诀 | 记忆方法 + 顺口溜 |
| PART 7 | 进阶思考 | 开放问题 + 与前后章联系 |

### Step 3: 内容设计 ✍️

为每个 PART 撰写具体内容，确保：
- 符合同济七版高等数学教材
- 体现布鲁姆高阶认知（分析+评价+创造 ≥ 40%）
- 公式使用 matplotlib 支持的 LaTeX

### Step 4: 长图生成 🎨

调用图片生成脚本，输出 PNG 长图：
- 尺寸：1440px 宽，高度自适应
- 2x 高清分辨率
- 品牌配色：深海蓝 + 龙虾红

### Step 5: 质量检查 ✅

| 检查项 | 标准 |
|--------|------|
| 公式正确性 | LaTeX 正确渲染 |
| 内容完整性 | 定义+公式+例题+口诀 |
| 认知层次 | 高阶题目占比 ≥ 40% |
| 知识点准确 | 符合同济七版 |

### Step 6: 发布输出 📤

- 输出路径：`/root/.openclaw/media/outbound/{章节名}长图.png`
- 上传云存储，返回下载链接

---

## 🚀 如何使用

### 方式一：直接对话

```
用户：定积，生成第9章多元函数微分学的课程长图

定积执行六个步骤，输出长图文件 + 下载链接
```

### 方式二：指定内容

```
用户：制作关于不定积分计算的长图，包含换元法和分部积分

定积聚焦：不定积分计算方法，生成对应长图
```

### 方式三：指定参数

```
用户：生成第3章的长图，难度提高，水印用"代老师"

定积按要求生成，设置难度=提高，水印=代老师
```

---

## 📝 输入参数

| 参数 | 必填 | 说明 | 示例 |
|------|------|------|------|
| `chapter` | ✅ | 章节名称 | "第9章 多元函数" |
| `knowledge_points` | ✅ | 核心知识点列表 | ["偏导数", "全微分", "极值"] |
| `difficulty` | ❌ | 基础/提高/综合 | "综合" |
| `brand_name` | ❌ | 水印名称 | "代国兴" |
