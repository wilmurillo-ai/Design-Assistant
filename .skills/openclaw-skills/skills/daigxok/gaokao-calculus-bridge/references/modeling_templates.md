# 数学建模模板库

## 通用建模流程模板

### 模板1：优化问题建模

```markdown
## 问题类型：优化问题

### Step 1: 识别优化要素
- **决策变量**：需要确定的未知量（如x, y）
- **目标函数**：需要最大化或最小化的量（如利润、成本、时间）
- **约束条件**：限制条件（等式或不等式）

### Step 2: 建立数学模型
```
最小化/最大化：f(x)
约束条件：
  g_i(x) ≤ 0,  i = 1, ..., m
  h_j(x) = 0,  j = 1, ..., p
```

### Step 3: 选择求解方法
- 无约束优化：梯度下降法、牛顿法
- 等式约束：拉格朗日乘数法
- 不等式约束：KKT条件、内点法

### Step 4: 结果解释
- 最优解的实际意义
- 敏感性分析（参数变化的影响）
- 模型局限性讨论
```

### 模板2：变化率问题建模

```markdown
## 问题类型：变化率/动态问题

### Step 1: 识别动态要素
- **状态变量**：描述系统状态的量（如人口、温度、浓度）
- **时间尺度**：离散时间或连续时间
- **变化机制**：导致变化的原因（出生/死亡、流入/流出）

### Step 2: 建立微分/差分方程
连续时间：dX/dt = f(X, t, 参数)
离散时间：X_{t+1} = f(X_t, 参数)

### Step 3: 分析模型行为
- 平衡点：令dX/dt = 0，求解X*
- 稳定性：在X*附近线性化，分析特征值
- 长期行为：t → ∞时的渐近解

### Step 4: 参数估计与验证
- 利用历史数据估计参数
- 模型预测与实际数据对比
- 模型修正与改进
```

### 模板3：概率统计建模

```markdown
## 问题类型：随机现象建模

### Step 1: 识别随机结构
- **随机变量**：什么是不确定的？
- **分布类型**：离散（二项、泊松）或连续（正态、指数）
- **依赖关系**：变量间是否独立？

### Step 2: 建立概率模型
- 确定联合分布或条件分布
- 定义期望、方差等数字特征
- 建立假设检验框架（如需要）

### Step 3: 统计推断
- 参数估计：最大似然、贝叶斯方法
- 假设检验：确定显著性水平、计算p值
- 置信区间：估计的不确定性量化

### Step 4: 预测与决策
- 点预测与区间预测
- 风险分析（VaR等）
- 基于数据的决策建议
```

## 领域专用模板

### 物理建模模板

```markdown
## 物理问题建模

### 守恒律识别
- 质量守恒：连续性方程
- 动量守恒：牛顿第二定律/Navier-Stokes方程
- 能量守恒：热力学第一定律

### 本构关系
- 材料特性：胡克定律、状态方程
- 传输特性：傅里叶定律、菲克定律
- 电磁特性：麦克斯韦方程组

### 边界与初始条件
- 边界条件：Dirichlet/Neumann/Robin
- 初始条件：t=0时的状态
- 界面条件：不同介质交界处的连续性

### 无量纲化
- 识别特征尺度
- 引入无量纲变量
- 简化方程，识别主导项
```

## 建模假设检验清单

### 通用假设
- [ ] 线性假设是否合理？
- [ ] 独立性假设是否成立？
- [ ] 平稳性/时变性判断
- [ ] 空间均匀性假设

### 模型验证方法
1. **内部验证**：参数敏感性分析
2. **外部验证**：与独立数据集对比
3. **交叉验证**：K折验证、留一法
4. **专家验证**：领域专家评估合理性

### 模型改进策略
- 机制细化：增加过程细节
- 结构扩展：从单变量到多变量
- 随机性引入：确定性→随机性
- 尺度跨越：微观→宏观联系

## 常见建模陷阱

### 陷阱1：过度拟合
**症状**：训练数据表现好，新数据表现差
**对策**：正则化、简化模型、更多数据

### 陷阱2：单位不一致
**症状**：计算结果数量级明显错误
**对策**：无量纲化、单位检查清单

### 陷阱3：忽略约束
**症状**：数学解在实际中不可行
**对策**：显式列出所有约束条件

### 陷阱4：因果混淆
**症状**：相关性误认为因果性
**对策**：实验设计、工具变量、因果推断方法
