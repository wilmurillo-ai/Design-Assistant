上册第 1–7 章、下册第 8–12 章与 `knowledge-graph/data/skills-content-phase2.json` 中的 `skillId` 可作交叉引用（见主 SKILL「与知识图谱协同」）。

## 常用公式速记

### 导数公式

- (sin x)' = cos x
- (cos x)' = -sin x
- (e^x)' = e^x
- (ln x)' = 1/x
- (x^n)' = n x^{n-1}

### 积分公式

- ∫ e^x dx = e^x + C
- ∫ 1/x dx = ln|x| + C
- ∫ sin x dx = -cos x + C
- ∫ cos x dx = sin x + C
- ∫ x^n dx = x^{n+1}/(n+1) + C（n≠-1）

### 常见泰勒展开

- e^x = 1 + x + x^2/2! + x^3/3! + …
- sin x = x - x^3/3! + x^5/5! - …
- cos x = 1 - x^2/2! + x^4/4! - …
- ln(1+x) = x - x^2/2 + x^3/3 - …（|x|<1）

### 下册速记（条目级）

- 格林公式：∮_L P dx + Q dy = ∬_D (∂Q/∂x - ∂P/∂y) dσ
- 高斯公式（奥-高）：∯_Σ = ∭_Ω (∂P/∂x + ∂Q/∂y + ∂R/∂z) dV
- 斯托克斯公式：∮_Γ = ∬_Σ rot F · dS
