## 示例3: 错误模式预警

### 场景
学生多次在定积分换元时忘记变换积分限。

### 错误记录
```
第1次: 计算 ∫(0到1) √(1-x²) dx，换元x=sinθ后积分限仍写0到1 ❌
第2次: 计算 ∫(0到π/2) sin²x cosx dx，换元u=sinx后积分限仍写0到π/2 ❌  
第3次: 计算 ∫(0到2) x√(x²+1) dx，换元u=x²+1后积分限仍写0到2 ❌
      ↓ [Active Memory 识别模式]
错误模式: {
  "type": "integral_limit_transform",
  "frequency": 3,
  "confidence": 0.95,
  "context": "定积分换元场景"
}
```

### 下次自动预警
```
学生: 计算 ∫(0到π/4) tanx sec²x dx
AI: [检测到定积分换元场景]
    [查询记忆: integral_limit_transform 频率=3]
    [触发预警]

回复: 这道题可以用换元法，设u = tanx。

⚠️ **提醒**: 这是你的常见易错点——定积分换元时忘记变换积分限！

**换元检查清单**:
✓ 原积分限 x=0 → u=tan(0)=0
✓ 原积分限 x=π/4 → u=tan(π/4)=1  
✓ 新积分限应该是 0 到 1

请特别注意积分限的变换，继续计算吧。
```
