✨ Skill 功能特性

🧠 Active Memory (主动记忆系统)
   • 零触发记忆 - 无需学生说"记住这个"
   • 偏好识别 - 自动识别视觉/推导/练习型学习者
   • 掌握度追踪 - 实时追踪60+数学概念掌握状态
   • 错误预警 - 识别12种高数典型错误模式

🌙 Dreaming System (梦境系统)
   • 自动整理 - 每20分钟生成学习摘要
   • 知识图谱 - 自动构建概念依赖关系图
   • 断层检测 - 发现前置知识薄弱环节
   • 智能建议 - 生成个性化学习路径

📚 高等数学专项优化
   • 12个核心章节全覆盖
   • 极限、导数、积分、级数等60+概念
   • 积分限变换、链式法则等12种错误模式
   • 自适应难度调整
