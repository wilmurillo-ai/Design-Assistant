# 数学任务SOP模板库

## 一、计算类任务模板

### 1.1 极限计算任务

```markdown
## 极限计算任务

**任务ID**: LIM-{序号}
**难度等级**: ★到★★★★★
**适用年级**: 大一

### 标准化流程

#### Step 1: 类型识别
```
1.1 判断未定式类型
    - 0/0型 → 可尝试洛必达/泰勒/等价替换
    - ∞/∞型 → 可尝试洛必达/抓大头
    - 0·∞型 → 变形为0/0或∞/∞
    - ∞-∞型 → 有分母则通分，无分母则倒代换
    - 0⁰, ∞⁰, 1^∞型 → 取对数或利用重要极限

1.2 识别特殊结构
    - 根式差形式 → 有理化
    - 三角函数 → 利用重要极限或等阶替换
    - 幂指函数 → 指数化或取对数
```

#### Step 2: 方法选择矩阵
```
| 类型 | 首选手动 | 备选方法 | 警示点 |
|------|---------|---------|--------|
| 0/0多项式型 | 泰勒展开 | 洛必达(≤2次) | 展开阶数要够 |
| 0/0三角型 | 等价替换+洛必达 | 泰勒(正弦展开) | sinx~x只在乘除中安全 |
| ∞/∞指数型 | 抓大头/除最高次 | 洛必达 | e^x比x^n增长快 |
| 0·∞型 | 变除法 | 倒代换 | 选简单变量在分子 |
| 1^∞型 | 重要极限公式 | 取对数 | 验证底数趋于1 |
```

#### Step 3: 执行步骤模板
```
解: lim(x→x₀) [表达式]
    
    = lim [变形1]
    = lim [变形2]
    ...
    = lim [最终形式]
    = [数值/简单表达式]

检验: [可选 - 用另一种方法验证]
```

#### Step 4: 输出格式规范
```
1. 关键步骤编号
2. 关键转换用中文标注理由
3. 最终答案 \boxed{...}
4. 如有GeoGebra验证，附动态链接
```

#### Step 5: 常见错误警示
```
⚠️ 等价替换禁区:
   - 加减中慎用 (sinx - x ≠ 0 - x = -x)
   - 乘积中注意条件 (tanx~x, sinx~x 仅在 x→0)

⚠️ 洛必达禁区:
   - 0/0或∞/∞才可用
   - 微分后需检查是否更复杂
   - 最多用2-3次，无效则换方法
```

---

### 应用实例

**题目**: lim(x→0) (sinx - x)/(x³)

**执行**:
```
解: lim(x→0) (sinx - x)/x³

Step 1: 类型识别 → 0/0型

Step 2: 方法选择
   方案A: 洛必达 (需3次，可能复杂)
   方案B: 泰勒展开 (直接)

Step 3: 执行
   sinx = x - x³/6 + o(x³)

   = lim(x→0) (x - x³/6 + o(x³) - x)/x³
   = lim(x→0) (-x³/6 + o(x³))/x³
   = lim(x→0) (-1/6 + o(x³)/x³)
   = -1/6

\boxed{-\dfrac{1}{6}}

Step 4: 验证
   用洛必达验证: (-cosx-1)/3x² → sinx/6x → 1/6 → -1/6 ✓
```

## 1.2 导数计算任务

```markdown
## 导数计算任务

**任务ID**: DER-{序号}
**难度等级**: ★到★★★★
**适用年级**: 大一

### 标准化流程

#### Step 1: 函数类型识别
```
1. 基本函数 → 直接查表
2. 和差积商 → 分别求导后运算
3. 复合函数 → 链式法则
4. 隐函数 → 两边同时对x求导（注意y是x的函数）
5. 参数方程 → dy/dx = (dy/dt)/(dx/dt)
6. 反函数 → (f⁻¹)'(y) = 1/f'(x)
```

#### Step 2: 求导决策树
```
求导任务
   │
   ├── 基本初等函数？ → 直接查表
   │
   ├── 四则运算组合？
   │   └── f±g → f'±g'
   │   └── f·g → f'g + fg'
   │   └── f/g → (f'g - fg')/g²
   │
   ├── 复合函数？
   │   └── 分解为基本函数链
   │   └── 每层求导后相乘
   │
   ├── 特殊形式？
   │   └── 隐函数 → 两边对x求导
   │   └── 参数方程 → dy/dx = y'_t / x'_t
   │   └── 幂指函数 → 对数法或指数法
```

#### Step 3: 链式法则执行模板
```
已知: y = f(g(h(x)))

则: dy/dx = f'(g(h(x))) · g'(h(x)) · h'(x)

记忆口诀: "外到内，逐层剥，求导后，连乘过"
```

#### Step 4: 隐函数求导模板
```
对方程 F(x,y) = 0 两边同时对 x 求导:

d/dx[F(x,y)] = d/dx[0]

注意:
  - y 视为 x 的函数
  - 遇到 y 的项，使用 dy/dx
  - 解出 dy/dx（可能含 y）
```

#### Step 5: 高阶导数策略
```
1. 先求前几阶，寻找规律
2. 观察符号、阶乘、组合数模式
3. 用数学归纳法验证
4. 常用高阶导数公式:
   - (xⁿ)^{(m)} = n(n-1)...(n-m+1)·xⁿ⁻ᵐ
   - (e^{kx})^{(n)} = kⁿe^{kx}
   - (sinkx)^{(n)} = kⁿsin(kx + nπ/2)
   - (coskx)^{(n)} = kⁿcos(kx + nπ/2)
```

## 1.3 积分计算任务

### 不定积分SOP

```markdown
## 不定积分计算

**任务ID**: INT-{序号}
**难度等级**: ★到★★★★

### 积分策略决策流程

#### Phase 1: 快速识别
```
1. 是否为基本积分公式直接形式？
   → 是: 直接写出结果 + C

2. 被积函数形态:
   a) 多项式 × 指数/三角/对数
      → 分部积分法 (LIATE 选 u)

   b) 复合函数形式 f(ax+b)
      → 凑微分法

   c) 根式形式 √(a²-x²), √(x²+a²)
      → 三角换元

   d) 有理分式 P(x)/Q(x)
      → 部分分式分解
```

#### Phase 2: 分部积分执行 (LIATE法则)

```
决策规则: 选 LIATE 表中靠前的函数作为 u

| 优先级 | 函数类型 | 示例 |
|-------|---------|------|
| L (最高) | 对数函数 | ln x, log₂x |
| I | 反三角函数 | arctan x, arcsin x |
| A | 代数函数 | x², √x |
| T | 三角函数 | sin x, cos x |
| E (最低) | 指数函数 | eˣ, aˣ |

分部积分公式: ∫u dv = uv - ∫v du
```

#### Phase 3: 换元积分执行

```
三角换元规则:
┌─────────────────────────────────────────┐
│ 根式形式              │ 换元方式         │
├─────────────────────────────────────────┤
│ √(a²-x²)              │ x = a sin t     │
│ √(a²+x²)              │ x = a tan t     │
│ √(x²-a²)              │ x = a sec t     │
└─────────────────────────────────────────┘

换元步骤:
1. 设 x = g(t)，求 dx = g'(t)dt
2. 将被积函数中的 x 和 dx 替换
3. 在新变量 t 下积分
4. 用 t = g⁻¹(x) 回代
```

#### Phase 4: 有理分式分解

```
分解步骤:
1. 次数检查: 分子次数 < 分母次数? 否 → 长除法
2. 分母因式分解
3. 建立部分分式方程
4. 求系数: 待定系数法或代入法
5. 逐项积分

标准分解形式:
- 线性因子: A/(x-a)
- 重线性因子: A/(x-a)² + B/(x-a)
- 二次因子: (Ax+B)/(x²+px+q)
```

#### Phase 5: 结果验证

```
验证方法: 求导回代
如果 F'(x) = f(x)，则 ∫f(x)dx = F(x) + C
```

### 定积分SOP

```markdown
## 定积分计算

**任务ID**: DEF-{序号}
**难度等级**: ★到★★★★

### 额外检查项 (相对于不定积分)

#### 换元必检项
```
换元时必须同时改变积分上下限！

1. 计算新上下限: u₁ = g(a), u₂ = g(b)
2. 新积分: ∫[a,b] f(x)dx = ∫[u₁,u₂] f(g(t))g'(t)dt
3. 注意: u₁ 可能大于 u₂（此时调换顺序并变号）
```

#### 对称性检查
```
偶函数: ∫[-a,a] f(x)dx = 2∫[0,a] f(x)dx
奇函数: ∫[-a,a] f(x)dx = 0

注意: f(x) 必须是关于原点对称区间上的奇/偶函数
```

#### 周期函数检查
```
若 f(x) 周期为 T:
∫[0,nT] f(x)dx = n∫[0,T] f(x)dx
```

## 1.4 微分方程求解任务

```markdown
## 微分方程求解

**任务ID**: DEQ-{序号}
**难度等级**: ★到★★★★★

### 方程类型判定流程

#### Step 1: 阶数与形式识别
```
1. 确定阶数 (方程中最高阶导数的阶数)
2. 判断是否为标准形式 y', y'', 等
```

#### Step 2: 一阶方程判定

```
一阶方程 dy/dx = F(x,y)

├── 可分离变量?
│   └── dy/f(y) = g(x)dx → 分离后积分
│
├── 齐次方程? (F(x,y) 可化为 f(y/x))
│   └── 设 u = y/x → 化为可分离
│
├── 一阶线性?
│   └── dy/dx + P(x)y = Q(x)
│   └── 积分因子: μ = e^{∫Pdx}
│   └── 通解: μy = ∫μQdx + C
│
└── 伯努利方程?
    └── dy/dx + P(x)y = Q(x)yⁿ (n≠0,1)
    └── 令 z = y^{1-n} → 化为一阶线性
```

#### Step 3: 二阶方程判定

```
二阶方程 y'' = F(x, y, y')

├── 可降阶?
│   ├── 缺 y: y'' = f(x, y')
│   │   └── 令 p = y' → 降为一阶
│   │
│   └── 缺 x: y'' = f(y, y')
│       └── 令 p = y' → dy/dp · p = f(y,p)
│
├── 常系数齐次?
│   └── y'' + py' + qy = 0
│   └── 特征方程: r² + pr + q = 0
│       ├── 两个不等实根: y = C₁e^{r₁x} + C₂e^{r₂x}
│       ├── 两个相等实根: y = (C₁ + C₂x)e^{rx}
│       └── 共轭复根: y = e^{αx}(C₁cosβx + C₂sinβx)
│
└── 常系数非齐次?
    └── y'' + py' + qy = f(x)
    └── 通解 = 齐次通解 + 特解
        ├── f(x)=Pₙ(x)e^{λx} → 待定系数法
        ├── f(x)=e^{αx}[Pₙcosβx + Qₘsinβx] → 三角待定
        └── 一般情况 → 常数变易法
```

#### Step 4: 特解形式选择表

```
自由项 f(x) 形式              │ 待设特解形式
────────────────────────────────────────────────────────────
Pₙ(x) (n次多项式)             │ x^k · Qₙ(x) (k由λ决定)
e^{λx}·Pₙ(x)                   │ x^k · e^{λx}·Qₙ(x)
e^{αx}[Pₙcosβx + Qₘsinβx]     │ x^k·e^{αx}[Rₘcosβx + Sₘsinβx]

k 的取值:
  - λ (或 α±βi) 不是特征根 → k = 0
  - λ (或 α±βi) 是单特征根 → k = 1
  - λ (或 α±βi) 是重特征根 → k = 2
```

## 二、证明类任务模板

### 2.1 中值定理证明模板

```markdown
## 中值定理证明

**任务ID**: MVT-{序号}
**难度等级**: ★★★到★★★★★
**适用章节**: 第3章

### 证明策略库

#### 策略1: 辅助函数构造法

```
证明目标: 证明存在 ξ∈(a,b) 使得某个等式成立

步骤:
1. 化简目标等式，将ξ替换为x
2. 移项，使一边为0
3. 观察原函数形式 (通常是需要证明的等式来自某个函数的导数)
4. 逆用求导法则，构造辅助函数 F(x)

常用原函数模式:
┌────────────────────────────────────────────────┐
│ 目标形式                    │ 辅助函数 F(x)    │
├────────────────────────────────────────────────┤
│ f'(ξ) = 0                   │ f(x)             │
│ f'(ξ) = [f(b)-f(a)]/(b-a)  │ f(x) - 直线方程  │
│ f(ξ)g'(ξ) + f'(ξ)g(ξ) = 0 │ f(x)g(x)         │
│ f'(ξ)g(ξ) - f(ξ)g'(ξ) = 0 │ f(x)/g(x)        │
│ f''(ξ) + f(ξ) = 0          │ f(x)·sinx 等组合 │
└────────────────────────────────────────────────┘
```

#### 策略2: 端点值分析

```
步骤:
1. 计算端点值 f(a), f(b)
2. 分析 f(x) 在端点的性质 (如 f(a)=f(b) → 罗尔定理)
3. 确定使用的定理及所需条件
4. 应用定理得到结论
```

#### 策略3: 逆向分析

```
从结论出发:
1. 假设结论成立
2. 逆推需要什么条件
3. 寻找满足这些条件的点
4. 整理证明步骤 (从条件到结论)
```

### 拉格朗日中值定理证明模板

```
**证明目标**: 证明存在 ξ∈(a,b) 使得 f'(ξ) = [f(b)-f(a)]/(b-a)

**标准证明步骤**:

1. 构造辅助函数
   设直线 AB: y = f(a) + [(f(b)-f(a))/(b-a)](x-a)
   构造: F(x) = f(x) - [f(a) + ((f(b)-f(a))/(b-a))(x-a)]
   
   验证: F(a) = F(b) = 0

2. 验证条件
   - f(x) 在 [a,b] 连续 ✓
   - f(x) 在 (a,b) 可导 ✓
   - F(x) 由 f(x) 和直线方程线性组合，在 [a,b] 连续，(a,b) 可导 ✓

3. 应用罗尔定理
   由罗尔定理，存在 ξ∈(a,b) 使得 F'(ξ) = 0

4. 求导并整理
   F'(x) = f'(x) - [f(b)-f(a)]/(b-a)
   F'(ξ) = 0 ⟹ f'(ξ) = [f(b)-f(a)]/(b-a)

证毕 ✓
```

### 柯西中值定理证明模板

```
**证明目标**: 证明存在 ξ∈(a,b) 使得 [f(b)-f(a)]/[g(b)-g(a)] = f'(ξ)/g'(ξ)

**关键洞察**: 柯西定理是参数化的拉格朗日定理

**标准证明步骤**:

1. 构造辅助函数 (与拉格朗日类似)
   F(x) = f(x) - [f(a) + (f(b)-f(a))/[g(b)-g(a)] · (g(x)-g(a))]
   
   验证: F(a) = F(b) = 0

2. 验证条件
   - f(x), g(x) 在 [a,b] 连续 ✓
   - f(x), g(x) 在 (a,b) 可导 ✓
   - g'(x) ≠ 0 (保证分母非零) ✓

3. 应用罗尔定理
   存在 ξ∈(a,b) 使得 F'(ξ) = 0

4. 求导
   F'(x) = f'(x) - [f(b)-f(a)]/[g(b)-g(a)] · g'(x)
   F'(ξ) = 0 ⟹ f'(ξ)/g'(ξ) = [f(b)-f(a)]/[g(b)-g(a)]

证毕 ✓
```

## 三、几何应用任务模板

### 3.1 旋转体体积

```markdown
## 旋转体体积计算

**任务ID**: VOL-{序号}
**难度等级**: ★★到★★★★

### 方法选择决策

```
旋转体体积
   │
   ├── 绕 x 轴旋转?
   │   ├── 平面截面垂直于x轴 → 圆盘法/薄片法
   │   │   └── V = π∫[a,b] y²(x)dx
   │   │
   │   └── 平面截面垂直于y轴 → 壳法 (管壳法)
   │       └── V = 2π∫[a,b] |x|·y(x)dx
   │
   └── 绕 y 轴旋转?
       ├── 圆盘法: V = π∫[c,d] x²(y)dy
       └── 壳法: V = 2π∫[c,d] |y|·x(y)dy
```

### 圆盘法 vs 壳法选择指南

```
┌─────────────────────────────────────────────────────────┐
│           │ 圆盘法 (A)      │ 壳法 (B)                │
├─────────────────────────────────────────────────────────┤
│ 适用场景  │ 截面垂直旋转轴   │ 截面平行旋转轴          │
│ 函数形式  │ y=f(x) 已知     │ x=g(y) 易于积分        │
│ 积分变量  │ x               │ x (外半径)              │
│ 计算特点  │ 需平方           │ 需乘 2πx               │
├─────────────────────────────────────────────────────────┤
│ 选择建议: 优先尝试圆盘法；若需解复杂方程求 x=g(y)，    │
│          或被积函数复杂时，考虑壳法                      │
└─────────────────────────────────────────────────────────┘
```

### 完整解题模板

```
**题目**: 求 y=√x, x=1, y=0 绕 x 轴旋转所得旋转体体积

**Step 1: 可视化分析**
   - 区域: y=√x 从 (0,0) 到 (1,1)，x=1，y=0 围成
   - 旋转轴: x轴 (y=0)

**Step 2: 方法选择**
   - 截面 ⟂ x轴 (旋转轴) → 圆盘法适用
   - 体积元素: dV = π·(半径)²·(厚度) = π·y²·dx

**Step 3: 确定积分限**
   - x 从 0 到 1

**Step 4: 建立积分**
   V = ∫[0,1] π·(√x)²dx = π∫[0,1] x dx

**Step 5: 计算**
   = π · [x²/2]₀¹ = π/2

**Step 6: 答案**
   \boxed{V = \dfrac{\pi}{2}}
```

## 四、错题分析任务模板

```markdown
## 错题归因分析

**任务ID**: ERR-{序号}
**适用场景**: 学生作业/考试错题分析

### 错因分类体系

```
A. 概念性错误 (Conceptual Error)
   ├── A1: 基础概念理解错误
   ├── A2: 定理条件/结论记忆错误
   └── A3: 公式记忆错误

B. 方法性错误 (Methodological Error)
   ├── B1: 方法选择不当
   ├── B2: 方法应用步骤错误
   └── B3: 变换方向错误

C. 计算性错误 (Computational Error)
   ├── C1: 算术运算错误
   ├── C2: 符号处理错误
   └── C3: 漏项/重复

D. 理解性错误 (Comprehension Error)
   ├── D1: 题意理解偏差
   ├── D2: 条件误用
   └── D3: 忽略隐含条件

E. 前置知识缺失 (Prerequisite Gap)
   ├── E1: 基础知识不扎实
   └── E2: 相关章节内容遗忘
```

### 归因分析报告模板

```
┌─────────────────────────────────────────────────────┐
│ 错题编号: ERR-2026-001                              │
│ 原题: lim(x→0) sinx/x = ?                           │
│ 学生答案: 0                                         │
