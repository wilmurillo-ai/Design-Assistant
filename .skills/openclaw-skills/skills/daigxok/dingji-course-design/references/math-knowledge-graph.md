### LIATE法则（分部积分u选择）

| 顺序 | 函数类型 | 示例 |
|-----|---------|------|
| L | 对数函数 Logarithmic | ln x, log₂x |
| I | 反三角函数 Inverse trig | arctan x, arcsin x |
| A | 代数函数 Algebraic | x², √x |
| T | 三角函数 Trigonometric | sin x, cos x |
| E | 指数函数 Exponential | eˣ, aˣ |

**原则**：选表中靠前的函数作为u，剩余部分作为dv

3. **三角换元回代错误**：换元后忘记回到原变量

### 微积分基本定理核心

```yaml
定理陈述:
  如果 F(x) = ∫[a,x] f(t)dt
  那么 F'(x) = f(x)

几何意义:
  曲边梯形面积函数的导数 = 截面高度

物理意义:
  位置函数的导数 = 速度
  速度函数的积分 = 位移
```

### 常见错误模式

1. **上下限混淆**：换元后忘记调整积分上下限
2. **定理条件遗忘**：使用牛顿-莱布尼茨公式时f(x)不连续
3. **定积分与不定积分混淆**：定积分是数值，不定积分是函数族

### 链式法则（多元复合）

```yaml
z = f(u,v), u = g(x,y), v = h(x,y)

∂z/∂x = ∂f/∂u · ∂u/∂x + ∂f/∂v · ∂v/∂x
∂z/∂y = ∂f/∂u · ∂u/∂y + ∂f/∂v · ∂v/∂y

特别注意:
```

```yaml
  |▽f| = 最大方向导数值
```

### 微积分基本定理的推广

```yaml
一维（微积分基本定理）:
  ∫[a,b] F'(x)dx = F(b) - F(a)
  ∫_∂[a,b] ω = ∫_[a,b] dω

二维（格林/斯托克斯）:
  ∮_∂D Pdx + Qdy = ∬_D (∂Q/∂x - ∂P/∂y)dxdy
  ∮_∂Σ ω = ∫∫_Σ dω

  ∯_∂V Pdy dz + Qdz dx + Rdx dy = ∭_V (∂P/∂x + ∂Q/∂y + ∂R/∂z)dV
  ∯_∂V ω = ∭_V dω
```

```yaml
平面保守场条件:
  ▽×F = 0 ⟺ ∂P/∂y = ∂Q/∂x

势函数求法:
  1. 验证保守场条件
  2. 从起点到终点积分
  3. 或用原函数法: φ(x,y) = ∫Pdx + ∫(Q - ∂φ/∂y)dx
```

```
    │   └── 莱布尼茨：单调递减 + 趋于0
    │
    └── 任意项？
        └── |aₙ| 收敛？→ 绝对收敛 : 条件收敛
```

### 常见展开公式

```yaml

```

### 傅里叶系数公式

```yaml
周期为2π的函数f(x)的傅里叶展开:

a₀ = (1/π)∫[-π,π] f(x)dx
aₙ = (1/π)∫[-π,π] f(x)cos(nx)dx
bₙ = (1/π)∫[-π,π] f(x)sin(nx)dx

f(x) ~ a₀/2 + Σ(aₙcos(nx) + bₙsin(nx))

收敛条件（狄利克雷）:
  1. 连续或有限个第一类间断点
  2. 有限个极值点
  → 收敛于函数值或左右极限平均值
```
