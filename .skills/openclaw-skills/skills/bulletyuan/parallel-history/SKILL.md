# 平行历史 - 历史推演聊天文字游戏

## 游戏简介

**平行历史**是回合制历史推演游戏：玩家选定起点与阵营，在约 **20 年政治生命**内推动历史，可建**组织/教派**；生命结束后进入**观察者模式**，看遗产走向**荣耀或耻辱**。

## 核心机制（执行摘要）

- **政治生命**：每回合可选 **1–5 年**，共约 **4–8 回合**；耗尽后进观察者模式。
- **四维环境**（每回合推演；**地点**是自然/社会/生产力/文化的具体基底）：自然、社会、生产力、文化；环境影响可行性与成功率，突变可生特殊事件。
- **突发事件**：内/外/自然；约 **30%** 触发；分强度；须应对，不当则连锁反应。
- **羁绊**：盟友/对手/情感/利益相关；好感度与信任度影响选项与成功率；人物离去改剧情。
- **回合**：须符合历史逻辑、客观条件，并综合环境、事件、羁绊。
- **阵营**：每起点 **3–4** 个阵营，具目标、资源与时代约束。
- **传承**：政治生命结束时可选传承愿景、巩固组织、立教派等。
- **观察者**：不直接决策，每回合仍 ≤5 年，追踪遗产至结局。
- **结局**：荣耀榜（持续发展、正面评价）或耻辱柱（消亡/变质、负面评价、遗产磨灭）。

### 客观推演规则（要点）

变革受 **物质基础**（生产力、交通通讯、军事后勤）、**社会基础**（阶级矛盾、社会支持、文化接受度）、**历史惯性**（既得利益、制度成本、渐进 vs 激进风险）约束。每回合须说明可行/勉强/不可行，对不可行给出**替代路径**与**现实后果**。含「1840 中国建民主」等**完整可行性示例**：见 `references/skill-main-supplement.md` 开篇「客观推演规则」。

## 游戏流程（阶段）

1. **选历史起点**（推荐 1918 / 1840 / 1789 等，可自定义）→ 介绍背景、势力、时代、阵营列表。完整阵营与起点： `references/historical-starting-points.md`
2. **选活动地点**（真实地点；五维：气候、地理、民风、资源、信息流通；资源≥60 可多地点）。**地点列表示例与长对话**： `references/skill-main-supplement.md`「游戏流程」
3. **选阵营**
4. **政治生命回合**（地点环境 + 四维 + 突发事件 + 羁绊 + 客观约束 + 选项）。**1918 柏林长回合样板**：同上补充文件
5. **传承**（愿景/组织/教派/放弃）
6. **观察者**
7. **结局**（荣耀/耻辱长文示例见补充文件）

数值与公式：`references/game-mechanics.md`

## 输出格式（v2.2 骨架）

每回合输出 **≤5000 字**。必备块（顺序）：回合标题栏（年段、阵营、地点、剩余政治生命）→ **决策回顾**（约 50 字）→ **环境变化**（四维各一句，约 150 字）→ **核心事件进展**（叙事为主，约 **2500–3000 字**）→ **概率计算简报**（约 200 字，仅 3–5 个关键因素）→ **事件结果**（约 300 字）→ **历史进程意义**（约 200 字）→ **羁绊变化**（约 100 字）→ **下回合预览**（约 100 字）→ **可选行动**（3–5 项，约 500 字）。

**禁止**：复杂公式、罗列全部环境数值、过度解释修正项。**突发事件/人物离去/观察者**子模板与正反例、字数分配表： `references/skill-main-supplement.md`「输出格式规范」。

## 游戏规则

### 必须遵守

1. 每回推演四维环境及其对决策的影响  
2. 按约 30% 概率处理突发事件及连锁  
3. 维护羁绊并反映于选项与成功率  
4. 客观约束优先，标注不可行并给路径  
5. 符合时代逻辑与时间规则（每回合 ≤5 年；政治生命 **20 年**封顶）  
6. 公平、结局与后果基于推演而非偏袒  

### 禁止

1. 引入时代不存在的技术/思想  
2. 无视环境、羁绊与现实后果  
3. 保证必赢、延长政治生命、条件不足强行变革
