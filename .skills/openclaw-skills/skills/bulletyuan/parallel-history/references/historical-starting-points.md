# 平行历史 - 历史起点配置

## 🎮 标准起点

### 1️⃣ 1918年 - 一战结束

**历史背景**：
```
第一次世界大战结束，德意志帝国、奥匈帝国、俄罗斯帝国瓦解。
《凡尔赛条约》重划欧洲版图，民族自决思潮兴起。
共产主义革命在俄国成功，世界格局剧变。
```

**时代特征**：
- 帝国瓦解，新国家诞生
- 民主vs专制博弈
- 共产主义扩张
- 民族主义高涨

**可选阵营**：

#### 阵营1：德国社会民主党
```
理念：议会民主、社会公正
优势：组织完善、群众基础
劣势：左右受敌、经济困难
目标：建立稳定民主制度
```

#### 阵营2：德国国家人民党
```
理念：保守复辟、民族复兴
优势：军方支持、精英资源
劣势：民主潮流、国际压力
目标：恢复传统秩序
```

#### 阵营3：斯巴达克同盟
```
理念：共产主义革命
优势：革命热情、苏联支持
劣势：力量弱小、遭围剿
目标：建立苏维埃政权
```

#### 阵营4：自由军团
```
理念：民族主义、反共
优势：武装力量、战斗力强
劣势：缺乏政治纲领
目标：镇压左翼，重建秩序
```

## 🎨 自定义起点

### 规则

1. **必须是真实历史节点**
2. **提供当时历史背景**
3. **设计3-4个合理阵营**
4. **每个阵营有明确理念和优劣**

### 示例

**用户输入**："从1644年开始"

**AI生成**：
```
📜 1644年 - 明清易代

【历史背景】
李自成攻占北京，崇祯帝自缢。
吴三桂引清军入关，顺治帝迁都北京。
明朝残余势力在南方建立南明政权。

【时代特征】
- 民族矛盾尖锐
- 政权更迭剧烈
- 文化冲突激烈
- 社会秩序崩溃

【可选阵营】
1. 清朝朝廷 - 建立新秩序
2. 南明政权 - 恢复明朝
3. 农民军 - 推翻封建
4. 明朝遗民 - 文化抵抗
```
