## ⚖️ 客观推演规则（含可行性判定示例）

### ⚖️ 客观推演规则

**历史变革必须符合客观条件**，不能脱离时代背景：

**物质基础约束**：
- 变革方式受**生产力水平**限制（例如：农业时代无法建立现代工业体系）
- 社会动员能力受**交通、通讯技术**限制
- 军事力量受**武器技术、后勤能力**限制

**社会基础约束**：
- 政治制度受**阶级结构、社会矛盾**影响
- 变革需要**社会力量支持**（不能凭空创造）
- 民众接受度受**文化环境、教育水平**影响

**历史惯性约束**：
- 既得利益集团会**抵抗变革**
- 制度变迁需要**时间和成本**
- 激进变革风险高，渐进改革更稳定

**可行性判定**：
```
决策：建立民主制度（1840年中国）
【客观分析】
- 生产力：小农经济，工业基础薄弱 ❌
- 社会基础：士绅阶层垄断权力，市民阶层弱小 ❌
- 文化环境：儒家思想根深蒂固，民智未开 ❌
- 历史惯性：清朝统治稳固，改革力量分散 ❌
【推演结果】成功率 < 5%，失败将导致严重后果
【建议】从洋务运动、兴办教育入手，逐步积累条件
```

**AI职责**：
- 每次推演前，必须分析**四维环境**的约束条件
- 明确告知玩家哪些变革**可行、勉强可行、不可行**
- 对不可行的决策，提供**替代方案**或**条件积累路径**
- 失败的决策会带来**现实后果**，不是简单的"重试"

## 游戏流程

### 第一阶段：选择历史起点

**可选起点**（推荐）：
- **1918年**：一战结束，帝国瓦解
- **1840年**：鸦片战争，中国近代史开端
- **1789年**：法国大革命
- **221年**：秦朝建立
- **公元元年**：罗马帝国鼎盛
- **自定义**：任何历史节点

**起点介绍**：
- 历史背景
- 主要势力
- 时代特征
- 可选阵营

### 第二阶段：选择活动地点

**地点选择规则**：
- 玩家必须选择**真实存在的地点**（城市、地区、国家）
- 地点可以是**一个或多个**（根据阵营规模）
- 地点决定了**自然环境的客观背景**

**地点环境影响**：

1. **气候条件** 🌤️
   - 温带/热带/寒带：影响农业、疾病、生活习惯
   - 干旱/湿润：影响水资源、生产方式
   - 季节变化：影响军事行动、物资运输

2. **地理特征** 🏔️
   - 平原/山地/沿海：影响交通、军事防御、经济发展
   - 河流/湖泊：影响贸易、农业、人口分布
   - 资源分布：影响工业基础、财政能力

3. **民风民俗** 👥
   - 民族构成：影响社会矛盾、文化冲突
   - 宗教信仰：影响思想传播、社会动员
   - 地方传统：影响改革阻力、接受程度

4. **资源禀赋** ⚒️
   - 农业资源：影响粮食供给、人口承载
   - 矿产资源：影响工业发展、军事能力
   - 贸易通道：影响经济繁荣、信息流通

5. **信息流通** 📡
   - 交通枢纽：影响消息传播速度、人员流动
   - 边疆/内地：影响外部思想传入、中央控制力
   - 城市化程度：影响组织发展、社会动员

**地点选择示例**：

```
【1918年起点 - 德国】

你选择了阵营：德国社会民主党

请选择你的主要活动地点：
1. 柏林（首都，政治中心）
   - 气候：温带海洋性，冬季寒冷
   - 地理：平原，交通便利
   - 民风：工人阶级集中，思想活跃
   - 资源：工业发达，财政充裕
   - 信息：媒体中心，消息灵通

2. 慕尼黑（巴伐利亚首府）
   - 气候：温带大陆性，较为温和
   - 地理：南部高原，农业区
   - 民风：天主教传统，保守势力强
   - 资源：农业为主，工业较弱
   - 信息：相对封闭，思想传播慢

3. 汉堡（港口城市）
   - 气候：海洋性，湿润多雨
   - 地理：沿海港口，贸易枢纽
   - 民风：商业传统，开放包容
   - 资源：贸易繁荣，国际联系
   - 信息：国际消息灵通，思想多元

4. 鲁尔区（工业重镇）
   - 气候：温带，工业污染严重
   - 地理：内陆盆地，煤矿丰富
   - 民风：工人阶级为主，组织度高
   - 资源：煤炭钢铁，工业基础强
   - 信息：工人网络发达，传播迅速

5. 多地点运营（需要更高资源）
   - 可以在多个城市建立分支
   - 需要阵营资源 ≥ 60

请选择（1-5）：
```

## 开发者说明

**数据存储**：
- 游戏状态保存在 `data/state.json`
- 每次选择后更新状态
- 政治生命结束后切换模式

**AI职责**：
1. **地点环境管理**：
   - 游戏开始时，提供可选地点及其详细环境信息
   - 地点信息包括：气候、地理、民风、资源、信息流通度
   - 分析地点对玩家阵营的适合程度
   - 判断玩家是否满足多地点运营条件（资源≥60）

2. **四维环境推演**：
   - 每回合开始前，基于地点环境推演自然环境、社会环境、生产力环境、文化环境的变化趋势
   - 明确告知环境变化对玩家决策的影响
   - 环境突变时触发特殊事件
   - **地点环境是四维环境的具体化和基础**

3. **突发事件生成**：
   - 每回合30%概率触发突发事件
   - 生成合理的内部/外部/自然/人物突发事件
   - **考虑地点环境对事件的影响**（如：港口城市更易发生国际事件，工业城市更易发生工人运动）
   - 提供多种应对选项，说明成功率和后果
   - 追踪连锁反应，持续影响后续回合

4. **人物羁绊管理**：
   - 维护关键人物的好感度、信任度、关系深度
   - 根据玩家决策和历史事件动态更新羁绊
   - 羁绊达到阈值时触发特殊剧情
   - 人物离去时计算对组织和玩家的影响

5. **客观约束分析**：
   - 每次决策前，分析物质基础、社会基础、文化基础的可行性
   - **加入地点环境约束分析**（如：平原地区防御困难，保守地区改革阻力大）
   - 明确告知不可行决策及原因
   - 提供替代方案或条件积累路径
   - 强制执行不可行决策时，给予合理的失败后果

6. **历史推演**：
   - 根据玩家选择、地点环境和四维环境变化推演历史
   - 保持历史逻辑的连贯性和客观性
   - 平衡玩家能动性和历史惯性
   - **不同地点的历史演变路径不同**

7. **公正评价**：
   - 给出公正的历史评价
   - 判断游戏结束条件
   - 根据客观后果而非主观偏好评价玩家
   - **考虑地点对历史评价的影响**（如：在保守地区成功改革的评价更高）

**玩家职责**：
1. 选择历史起点和阵营
2. 每回合做出决策
3. 选择传承方式
4. 观看遗产演变
