# 平行历史 - 历史推演聊天文字游戏

## 简介

**平行历史**是一款回合制历史推演游戏。玩家选择历史起点，加入当时的历史阵营，在有限的20年政治生命中推动历史车轮，建立组织或教派传承自己的愿景。政治生命结束后，玩家进入观察者模式，观看自己的政治遗产如何在历史长河中演变——是荣耀加身，还是耻辱柱上？

## 快速开始

### 触发游戏

在对话中说以下任意内容：

- "玩平行历史"
- "推演历史"
- "我想改变历史"
- "从1918年开始"
- "从1840年开始"

### 游戏流程

1. **选择历史起点**：从推荐起点中选择或自定义
2. **选择阵营**：每个起点有3-4个可选阵营
3. **政治生命（20年）**：每回合1-5年，做出决策推动历史
4. **传承选择**：政治生命结束时，选择传承方式
5. **观察者模式**：观看组织/教派的后续发展
6. **结局判定**：荣耀榜或耻辱柱

## 核心机制

### 政治生命系统
- 玩家仅有**20年政治生命**
- 每回合**1-5年**
- 总计**4-8回合**

### 回合制推演
- 每回合显示历史背景
- 提供3-4个决策选项
- 支持自定义行动
- AI推演历史演变

### 阵营系统
- 每个起点3-4个阵营
- 阵营有明确的理念、优劣势
- 阵营会随历史演变

### 传承机制
- **传承愿景**：留下政治遗嘱
- **建立组织**：创建可持续组织
- **创立教派**：传播思想理念

### 观察者模式
- 政治生命结束后进入
- 观看组织/教派发展
- 每回合最多5年
- 直到组织消亡或游戏结束

### 结局系统
- **荣耀榜**：历史评价 ≥ 60
- **耻辱柱**：历史评价 ≤ -60
- **中性结局**：介于两者之间

## 历史起点

### 推荐起点
1. **1918年** - 一战结束
2. **1840年** - 鸦片战争
3. **1789年** - 法国大革命
4. **221年** - 秦朝建立
5. **公元元年** - 罗马帝国

### 自定义起点
可以输入任何历史节点，AI会自动生成：
- 历史背景
- 时代特征
- 可选阵营

## 文件结构

```
parallel-history/
├── SKILL.md                          # 主技能文档（短版，适配平台 embedding 8192 token 上限）
├── README.md                         # 本文件
├── scripts/
│   └── game_engine.py               # 游戏引擎（辅助工具）
├── references/
│   ├── skill-main-supplement.md     # 长示例、输出规范全文、开发者说明、更新日志
```

若上传技能平台时出现 **Embedding failed … maximum context length is 8192**，请确保平台**仅对短版 `SKILL.md` 做向量嵌入**，或将长文放在 `references/` 由模型按需读取；完整规则以 `references/skill-main-supplement.md` 为准。

## 使用建议

### 新手
1. 选择熟悉的历史起点（如1918年）
2. 优先建立组织基础
3. 注意历史趋势
4. 重视传承规划

### 进阶
1. 尝试创立教派（比组织更持久）
2. 注意理念纯度维护
3. 把握历史关键节点
4. 观察者模式可能逆转结局

### 高手
1. 自定义冷门历史节点
2. 尝试逆历史潮流
3. 追求极端结局（传奇或恶魔）
4. 多周目体验不同阵营
