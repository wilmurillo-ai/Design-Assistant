# 公文专业排版技能包 (Official Layout Pro)

本技能专为 OpenClaw 平台开发，用于自动化处理严格符合 GB/T 9704-2012 标准的公文文档。

## 🛠 功能亮点
* **双标宋区分**：红头使用“大标宋”，文件标题使用“小标宋”。
* **多模板支持**：用户可自定义无限个模板（如党委、行政、纪要等）。
* **中西文分设**：中文使用公文字体，所有数字和英文字符强制切换为 Times New Roman。
* **动态字号**：自动将发文字号插入模板预定位置。

## 📂 准备工作
1.  **字体安装**（至关重要）：
    请确保运行环境（Linux/Docker）安装了以下字体，或将它们放在 `fonts/` 文件夹并执行安装：
    - `方正大标宋简体`、`方正小标宋简体`
    - `仿宋_GB2312`、`楷体_GB2312`、`黑体`
    - `Times New Roman`
2.  **模板配置**：
    将您的 `.docx` 模板放入 `templates/` 文件夹。建议在模板中预设好红线和页码。

## 🚀 操作方法
直接在对话框中向 AI 下达指令，例如：
> “请使用 **行政模板** 排版这个 Word，发文字号是 **苏政办发〔2026〕12号**。”

## ⚠️ 注意事项
- 请确保输入文档的首行是文件的正式标题。
- 如果没有指定模板，系统将尝试调用 `default.docx`，若无此文件则创建一个空白公文。
