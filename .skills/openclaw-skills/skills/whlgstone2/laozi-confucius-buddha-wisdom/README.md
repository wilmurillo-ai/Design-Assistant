## 使用示例

### 查询核心概念
```
- 道家的"无为"是什么意思？
- 儒家的"仁"怎么理解？
- 佛教的"空"是什么？
```

### 对比分析
```
- 三圣怎么看待"欲望"？
- 老子和孔子在治国理念上有什么不同？
- 儒释道三家对"生死"的看法
```

### 应用指导
```
- 我工作压力大，三圣会给我什么建议？
- 如何用道家智慧处理人际关系？
- 面对失败，儒家和佛教的视角分别是什么？
```

### 语录查询
```
- 给我一句关于"智慧"的老子语录
- 孔子关于"学习"的名言
- 佛陀关于"放下"的教导
```
