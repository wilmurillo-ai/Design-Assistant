# 旅行路线规划建议模板

## 路线规划原则

### 1. 地理顺序优化
- 按照地理位置的自然顺序排列目的地
- 避免往返交叉，减少不必要的交通时间
- 考虑相邻城市或地区的组合

### 2. 交通方式选择
- **短距离**（<500km）：高铁、汽车
- **中距离**（500-1500km）：高铁、飞机
- **长距离**（>1500km）：飞机为主

### 3. 时间安排建议
- 每个目的地建议停留时间：
  - 大城市：2-4天
  - 中等城市：1-2天
  - 小城市/景点：半天-1天

## 常见路线模式

### 线性路线
```
起点 → 目的地1 → 目的地2 → 目的地3 → 终点
```
适用场景：单向旅行，如从北到南或从东到西

### 环形路线
```
起点 → 目的地1 → 目的地2 → 目的地3 → 起点
```
适用场景：围绕某个中心区域的旅行

### 放射状路线
```
        目的地2
           ↑
起点 → 目的地1
           ↓
        目的地3
```
适用场景：以某个城市为中心，向周边辐射

## 实用提示

### 季节考虑
- 春季（3-5月）：适合江南、西南地区
- 夏季（6-8月）：适合北方、高原地区
- 秋季（9-11月）：适合全国大部分地区
- 冬季（12-2月）：适合南方温暖地区或北方雪景

### 预算规划
- 交通费用：占总预算的30-40%
- 住宿费用：占总预算的20-30%
- 餐饮娱乐：占总预算的20-30%
- 门票购物：占总预算的10-20%

### 预订建议
- 机票：提前1-2个月预订
- 酒店：提前2-4周预订
- 热门景点门票：提前1周预订
- 高铁票：提前15天预订

## 技能使用指南

当用户使用此技能时，应该：

1. **调用travel技能**获取具体的旅行信息：
   - 机票价格和时刻
   - 酒店信息和价格
   - 景点门票和开放时间

2. **使用web_search工具**补充信息：
   - 当地天气情况
   - 交通路况信息
   - 旅游攻略和建议

3. **整合信息生成完整路线**：
   - 按时间顺序排列
   - 包含交通方式建议
   - 提供预算估算
   - 给出实用提示
