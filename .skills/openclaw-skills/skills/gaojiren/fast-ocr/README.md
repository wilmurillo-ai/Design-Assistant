# RapidOCR - 专业票据 OCR 识别工具

## 🚀 快速开始

### 1. 安装依赖
```bash
pip install rapidocr-onnxruntime
```

### 2. 使用命令

#### 识别发票/票据
```bash
call 票据识别 invoice.jpg
```

#### 文字识别
```bash
call 文字识别 image.jpg
```

#### 批量识别
```bash
call 批量识别 ./invoices/ output.txt
```

## 📊 提取字段

### 发票信息（30+ 字段）

**基本信息**
- 发票代码、发票号码
- 开票日期、发票类型
- 校验码、机器编号

**购买方信息**
- 名称、纳税人识别号
- 地址电话、开户行及账号

**销售方信息**
- 名称、纳税人识别号
- 地址电话、开户行及账号

**金额信息**
- 不含税金额、税额
- 价税合计、税率
- 商品明细（名称、规格、数量、单价）

**其他信息**
- 收款人、复核人、开票人
- 备注

## ⚠️ 已知问题

### v1.0.0 限制
- 金额小数点可能识别错误（v1.1.0 修复）
- 复杂背景票据识别率较低
- 手写文字不支持

### 建议
- 使用清晰的票据图片
- 确保光线充足、无遮挡
- 重要字段建议人工核对

## 🔧 故障处理

### 问题 1：识别结果为空
- 检查图片路径是否正确
- 确认图片格式支持（JPG/PNG/BMP）
- 检查图片是否过于模糊

### 问题 2：依赖安装失败
```bash
# 升级 pip
python -m pip install --upgrade pip

# 重新安装
pip install rapidocr-onnxruntime
```

### 问题 3：首次运行慢
- 首次运行会下载模型（约 100MB）
- 后续运行速度正常
