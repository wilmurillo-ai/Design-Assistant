# 世界各地风情人像摄影参考

## 使用说明

本参考提供了世界各地的地理位置和场景描述，用于构建具有地域特色的AI绘画/视频提示词。

## 常用场景描述模板

```

```

### 城市场景
```
在[城市名称]的[地标]附近，[城市特色描述]。[光线/天气描述]
```

### 自然场景
```
在[自然景观名称]中，[自然特色描述]。[季节/天气描述]
```

## 使用建议

1. **场景特色**: 突出地理位置的独特元素
2. **光线配合**: 根据地理位置选择合适的光线描述
3. **文化元素**: 添加当地文化或建筑特色
4. **季节考虑**: 考虑不同季节的景色变化
