# mcp-storyboard

一个 MCP 多场景分镜文生图制作助手，使用 BizyAir API 生成故事绘本分镜场景图。

## 📦 功能特性

- **智能提示词增强**：自动检测人物/模特关键词并追加优化提示词
- **灵活尺寸控制**：支持 1:1 到 3:5 等多种常见比例
- **批量生成**：单次最多生成 10 张场景图
- **双重调用方式**：优先使用 MCP 工具，失败时自动降级到 Shell curl
- **长时任务支持**：支持 3-10 分钟的图片生成，自动轮询进度

## 🚀 快速开始

### 1. 配置环境变量

```bash
export BIZYAIR_API_KEY="your_api_key_here"
```

建议将此配置添加到 `~/.zshrc` 或 `~/.bashrc` 中。

### 2. 使用方式

#### 方式一：通过 MCP 技能（推荐）

在 Claude Code 中直接描述需求，例如：
- "生成一个穿汉服的女孩在花园里跳舞的分镜场景图"
- "制作一个古风场景，比例 16:9，生成 8 张"

> 💡 图片生成通常需要 3-10 分钟，期间会自动输出进度信息

#### 方式二：使用 Python 脚本

```bash
python scripts/storyboard.py "一个穿汉服的女孩在花园里" --ratio 9:16 --batch 4
```

#### 方式三：使用 Shell 脚本

```bash
./scripts/bizyair_api.sh "一个穿汉服的女孩在花园里" 9:16 4
```

## 📐 支持的图片比例

| 比例 | 宽×高 | 适用场景 |
|-----|-------|---------|
| 9:16 | 928×1664 | 手机壁纸/短视频（默认） |
| 1:1 | 1240×1240 | 头像/方图 |
| 16:9 | 1664×928 | 视频封面/横屏 |
| 4:3 | 1080×1440 | 竖版海报 |
| 3:4 | 1440×1080 | 横版海报 |

更多比例请参考 SKILL.md 中的完整映射表。

## 🤖 智能提示词增强

当用户输入包含以下关键词时，自动追加优化提示词：

**中文**：模特、人物、人像、女性、女士、女孩、美女、穿搭、展示、试穿

**英文**：model, woman, female, girl, portrait, character, person

## ⏱️ 长时任务轮询

BizyAir 云端制图通常需要 **3-10 分钟**，系统会自动：

1. 每 5 秒轮询一次任务状态
2. 每 30 秒输出一次进度提示
3. 最多轮询 15 分钟（900 秒）

进度示例：
```
⏳ 开始轮询任务状态（预计 3-10 分钟）...
⏱️ 任务进行中... 已等待 0.5 分钟 (轮询 6)
🔄 [6] 当前状态: Processing (0.5分钟)
⏱️ 任务进行中... 已等待 1.0 分钟 (轮询 12)
...
✅ 任务完成（总耗时 4.2 分钟，轮询 50 次）
```

## 🔄 MCP 配置

在 Claude Code 的 `mcp.json` 中添加：

```json
{
  "mcpServers": {
    "Storyboard-mcp": {
      "url": "https://api.bizyair.cn/w/v1/mcp/242",
      "headers": {
        "Authorization": "Bearer ${BIZYAIR_API_KEY}"
      }
    }
  }
}
```
