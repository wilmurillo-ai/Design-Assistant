# ChatFlow Guardian Icon Design Specifications

## Overview
Icon for "ChatFlow Guardian" skill - A conversation monitoring and protection tool that ensures conversations never stall.

## Design Concept
**Chat Bubble + Shield** representing:
- **Chat Bubble**: Conversation, communication, dialogue
- **Shield**: Protection, guardian, safety
- **Combined**: Protecting conversations from interruption

## Color Palette
### Primary Colors
- **Blue (#4A90E2)**: Trust, reliability, technology
- **Green (#36B37E)**: Flow, smoothness, growth
- **White (#FFFFFF)**: Clean, modern, clarity

### Secondary Colors
- **Light Blue (#7BB8FF)**: Highlights, accents
- **Dark Blue (#1E5FB3)**: Shadows, depth
- **Light Green (#6FCF97)**: Accents, highlights

## Icon Variations

### Size 1: 64x64px (App Icon)
- **Use**: App icons, small displays
- **Detail Level**: Simplified
- **Features**: Chat bubble outline with shield emblem

### Size 2: 128x128px (Standard Icon)
- **Use**: Most common size, skill marketplace
- **Detail Level**: Medium detail
- **Features**: Chat bubble with detailed shield, subtle gradient

### Size 3: 256x256px (High Resolution)
- **Use**: High-DPI displays, marketing materials
- **Detail Level**: High detail
- **Features**: Detailed chat bubble with protective shield, gradient, subtle glow

## Design Elements

### Chat Bubble
- **Shape**: Rounded speech bubble
- **Style**: Modern, clean lines
- **Color**: Blue gradient (#4A90E2 to #7BB8FF)
- **Effect**: Subtle inner shadow for depth

### Shield
- **Shape**: Classic shield shape
- **Position**: Overlapping chat bubble
- **Color**: Green (#36B37E)
- **Effect**: Slight metallic shine
- **Symbol**: Optional: "CF" or conversation waves

### Protection Rays
- **Style**: Subtle rays emanating from shield
- **Color**: Light blue glow (#7BB8FF with transparency)
- **Effect**: Gentle glow effect

## Brand Guidelines

### Do's
- Use transparent background
- Maintain aspect ratio
- Use specified color palette
- Keep clean, modern design
- Ensure readability at small sizes

### Don'ts
- Don't distort the icon
- Don't change color palette significantly
- Don't add excessive detail to small sizes
- Don't use low contrast colors
- Don't obscure the shield element

## Usage Examples

### Skill Marketplace
- 128x128px icon for listing
- Clear, recognizable at thumbnail size

### Documentation
- 256x256px for detailed view
- SVG for scalable diagrams

- All sizes as needed
