## Screenshot Guidelines

### Visual Style
- **Theme**: Modern, clean, professional
- **Colors**: Blue/Green primary palette
- **Typography**: Clear, readable fonts
- **Spacing**: Ample whitespace
- **Consistency**: Same style across all screenshots

### Content Requirements
- Show real functionality (not just mockups)
- Include skill name and version
- Highlight key features
- Show configuration options
- Demonstrate user benefits

### Technical Requirements
- **Format**: PNG with transparency where needed
- **Quality**: High resolution, sharp text
- **Size**: 1920x1080px minimum
- **File Size**: <1MB per screenshot
- **Naming**: Clear, descriptive filenames

## Mockup Tools Recommendation
- **Figma** (Interactive prototypes)
- **Adobe XD** (UI design)
- **Sketch** (Mac design)
- **Canva** (Quick mockups)
- **ScreenFlow** (Screen recording)
