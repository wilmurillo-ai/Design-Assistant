# 热梗百科 Skill

## 用途

本 Skill 用于追踪和解析中文互联网（主要是B站、抖音）的热门梗/迷因，帮助用户：
- 快速了解当天/当周的热门网络梗
- 追溯热梗的来源和演变过程
- 理解热梗的含义和使用场景
- 收集相关视频素材和典型案例
- 生成结构化的热梗百科文档

## 适用平台

- **B站（bilibili）**：弹幕文化、鬼畜、二创梗
- **抖音**：短视频梗、挑战、BGM梗
- **微博**：热搜话题、明星梗
- **小红书**：生活方式梗、种草梗
- **知乎**：知识型梗、讨论梗

## 工作流程

### 第一步：明确需求

与用户确认以下信息：
1. **时间范围**：当天 / 本周 / 本月
2. **平台偏好**：B站 / 抖音 / 全平台
3. **梗类型**：全部类型 / 特定类型（如：鬼畜、BGM、挑战、语录等）
4. **数量要求**：Top 5 / Top 10 / 更多
5. **输出格式**：简要列表 / 详细百科 / 特定格式

### 第二步：搜索热梗信息

#### 2.1 B站热梗搜索

**热搜榜单**：
- 访问 B站热搜榜获取当前热门话题
- 关注热搜中的非新闻类话题（通常是梗的来源）

**搜索策略**：
- 关键词1：`B站热梗 当天/本周`
- 关键词2：`bilibili热门梗 合集`
- 关键词3：`鬼畜 热门 本周`
- 关键词4：`弹幕热词 梗`

**视频搜索**：
- 搜索近期（1-7天内）高播放量的梗合集视频
- 关注标题含"热梗"、"梗百科"、"梗指南"的视频
- 查看视频标签和评论区了解梗的传播情况

#### 2.2 抖音热梗搜索

**热搜榜单**：
- 访问抖音热搜榜获取热门话题
- 关注挑战榜、音乐榜中的热门内容

**搜索策略**：
- 关键词1：`抖音热梗 当天`
- 关键词2：`抖音挑战 热门`
- 关键词3：`抖音BGM 梗`
- 关键词4：`抖音热门话题`

#### 2.3 综合热梗搜索

**跨平台搜索**：
- 关键词1：`网络热梗 当天/本周`
- 关键词2：`互联网热词 梗`
- 关键词3：`{日期} 热梗`
- 关键词4：`梗百科 更新`

### 第三步：筛选和整理热梗

#### 3.1 热梗筛选标准

**热度判断**：
- 多平台同时出现
- 搜索指数高
- 相关视频播放量高
- 评论区讨论活跃

**时效性判断**：
- 优先选择1-7天内兴起的新梗
- 经典梗的复兴（周期性热梗）
- 排除过时的旧梗

#### 3.2 热梗分类

| 类型 | 说明 | 示例 |
|------|------|------|
| **语录梗** | 名人/网红的经典台词 | "家人们谁懂啊" |
| **BGM梗** | 配合特定音乐的视频 | "恐龙抗狼" |
| **挑战梗** | 特定动作/行为的模仿挑战 | "科目三" |
| **鬼畜梗** | 经过剪辑加工的搞笑素材 | 各种鬼畜视频 |
| **表情梗** | 特定表情包或表情 | "狗头"、"滑稽" |
| **剧情梗** | 影视/综艺中的经典片段 | "华强买瓜" |
| **知识梗** | 科普/知识类内容的梗 | "无穷小亮的科普" |
| **社会梗** | 社会事件衍生的梗 | 各类社会话题 |

### 第四步：追溯热梗来源

#### 4.1 来源追溯方法

**原始视频追踪**：
1. 搜索梗的关键词 + "出处"
2. 搜索梗的关键词 + "来源"
3. 搜索梗的关键词 + "原版"
4. 查看梗百科类视频的简介或置顶评论

**时间线梳理**：
1. 找到最早出现的视频/内容
2. 追踪传播路径（谁第一个发、谁带火的）
3. 记录关键时间节点（首次出现、爆发期、高峰期）

#### 4.2 来源信息收集

需要收集的信息：
- **原始出处**：视频标题、UP主/博主、发布时间
- **原始链接**：B站av号/BV号、抖音链接
- **背景故事**：梗产生的背景、前因后果
- **关键人物**：梗的创造者、传播者

### 第五步：解释热梗含义

#### 5.1 含义解析维度

**字面含义**：
- 梗的文字/表面意思
- 原始语境中的含义

**引申含义**：
- 在网络语境中的新含义
- 使用时的隐含意思

**使用场景**：
- 什么情况下使用这个梗
- 适合/不适合的场景

**情感色彩**：
- 搞笑 / 讽刺 / 吐槽 / 共鸣 / 其他

#### 5.2 使用示例

收集典型的使用案例：
- 正确的使用方式
- 常见的变体/改编
- 错误使用的反面教材

### 第六步：收集视频素材

#### 6.1 B站素材收集

搜索关键词：
- `{梗名称} 原版`
- `{梗名称} 出处`
- `{梗名称} 合集`
- `{梗名称} 鬼畜`

记录信息：
- 视频标题和BV号
- UP主名称
- 播放量/点赞数
- 视频链接

#### 6.2 抖音素材收集

搜索关键词：
- `{梗名称} 原版`
- `{梗名称} 挑战`
- `{梗名称} 合集`

记录信息：
- 视频标题
- 博主名称
- 点赞数/转发数
- 视频链接

## 热梗热度评估标准

### 热度指数（5星制）

| 星级 | 标准 | 特征 |
|------|------|------|
| ⭐⭐⭐⭐⭐ | 现象级 | 全平台爆发，人人皆知，持续一周以上 |
| ⭐⭐⭐⭐☆ | 高热 | 多平台热门，大量二创，持续3-7天 |
| ⭐⭐⭐☆☆ | 中热 | 单一平台热门，有一定传播，持续1-3天 |
| ⭐⭐☆☆☆ | 低热 | 小众圈子内传播，热度有限 |
| ⭐☆☆☆☆ | 微热 | 刚兴起或即将过气 |

### 热度判断依据

- **搜索量**：各平台搜索指数
- **播放量**：相关视频总播放量
- **讨论量**：评论、弹幕数量
- **二创数量**：衍生作品数量
- **传播范围**：跨平台传播情况

## 输出格式

默认输出为 Markdown 格式，可根据需要转换为：
- Word 文档（.docx）
- PDF 文档
- 网页（HTML）
- 图片（长图/信息图）

## 注意事项

1. **时效性**：热梗变化快，建议标注统计时间
2. **准确性**：来源追溯要准确，避免以讹传讹
3. **中立性**：客观描述，避免主观评价
4. **版权**：视频素材链接仅供引用，注意版权
5. **敏感性**：避免涉及政治、争议性社会事件

## 示例

### 示例请求
> "帮我整理一下今天B站和抖音的热梗，要Top 5，每个都要解释来源和含义"

### 示例输出流程
1. 搜索"B站热梗 当天"、"抖音热梗 当天"
2. 筛选出5个最热门的梗
3. 对每个梗追溯来源（搜索"{梗名} 出处"）
4. 解释每个梗的含义和使用场景
5. 收集相关视频素材
6. 按百科格式整理输出
