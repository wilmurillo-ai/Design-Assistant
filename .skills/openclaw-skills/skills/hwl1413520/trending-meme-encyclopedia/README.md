# Trending Meme Encyclopedia Skill

热梗百科 Skill，用于获取 B站、抖音等平台当天的热门梗/迷因，追溯来源、解释含义，生成结构化的热梗百科文档。

## 功能特点

- 🔥 **实时追踪**：获取当天/本周热门网络梗
- 🔍 **来源追溯**：追溯热梗的原始出处和背景故事
- 💡 **含义解析**：详细解释梗的字面意思和网络含义
- 📹 **视频素材**：收集B站、抖音相关视频链接
- 📊 **数据分析**：热度排行、类型分布、趋势分析
- 📝 **百科格式**：生成结构化的Markdown百科文档

## 支持平台

- **B站**：弹幕文化、鬼畜、二创梗
- **抖音**：短视频梗、挑战、BGM梗
- **微博**：热搜话题、明星梗
- **小红书**：生活方式梗、种草梗
- **知乎**：知识型梗、讨论梗

## 使用方法

### 基本用法

向AI助手描述你的需求：

```
帮我整理一下今天B站和抖音的热梗，要Top 5，每个都要解释来源和含义
```

```
最近有什么新的网络热梗？给我做个热梗百科
```

```
解释一下"科目三"这个梗的来源和含义
```

### 分析流程

1. **明确需求**：确认时间范围、平台、数量等
2. **搜索热梗**：搜索各平台热搜、热门视频
3. **筛选整理**：按热度筛选，分类整理
4. **追溯来源**：查找每个梗的原始出处
5. **解释含义**：分析字面意思和网络含义
6. **收集素材**：整理相关视频链接
7. **生成百科**：输出结构化百科文档

## 输出内容

生成的热梗百科包含：

- ✅ 热度总览和榜单
- ✅ 每个梗的详细来源追溯
- ✅ 含义解释和使用场景
- ✅ 经典台词和变体
- ✅ B站/抖音相关视频链接
- ✅ 热度趋势分析
- ✅ 类型和平台分布统计

## 注意事项

1. **时效性**：热梗变化快，建议标注统计时间
2. **准确性**：来源追溯要准确，避免以讹传讹
3. **中立性**：客观描述，避免主观评价
4. **版权**：视频素材链接仅供引用
5. **敏感性**：避免涉及政治、争议性社会事件
