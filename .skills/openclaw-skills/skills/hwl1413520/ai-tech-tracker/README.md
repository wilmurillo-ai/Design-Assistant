# AI技术动向追踪 Skill

获取最新AI技术动向，区分国内和国外市场，追踪各大模型的更新优化、重要技术迭代，识别划时代级产品，进行热度评级和专业分析。

## 功能特点

- 🌍 **国内外分离**：清晰区分国内和国外AI动态
- 🏢 **厂商全覆盖**：OpenAI、Google、Anthropic、百度、阿里、字节等
- 📊 **多维度分析**：模型更新、技术突破、产品发布
- 🔥 **热度评级**：5级热度评估体系
- 🏆 **重要级别**：划时代/重大/中等/一般四级分类
- 📈 **趋势分析**：技术趋势、模型对比、排行榜
- 📝 **专业报告**：生成结构化的AI技术周报/月报

## 使用方法

### 基本用法

向AI助手描述你的需求：

```
帮我整理一下本周AI领域的最新动态，区分国内国外
```

```
最近OpenAI和Google有什么新发布？
```

```
国内大模型这周有什么重要更新？
```

```
生成一份AI技术周报，重点关注模型能力对比
```

### 分析流程

1. **明确需求**：确认时间范围、关注区域、输出格式
2. **收集信息**：搜索国内外AI厂商官方公告和技术媒体
3. **筛选整理**：按重要级别和热度分类
4. **深度分析**：技术亮点、性能对比、影响分析
5. **热度评级**：综合多维度评估热度
6. **生成报告**：输出结构化报告

## 重要级别

| 级别 | 说明 | 示例 |
|------|------|------|
| 🔴 划时代 | 行业里程碑 | GPT-4发布、ChatGPT推出 |

## 注意事项

1. **时效性**：AI领域变化快，建议标注统计时间
2. **准确性**：优先采用官方渠道信息
3. **中立性**：客观分析，避免过度吹捧或贬低
4. **预测谨慎**：对未来趋势预测要标注不确定性
