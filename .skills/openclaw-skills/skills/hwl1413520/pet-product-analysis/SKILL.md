# 宠物科技用品专业分析 Skill

## 用途

本 Skill 用于生成专业的宠物科技用品分析文档，帮助用户：
- 全面了解目标产品的功能特性和技术参数
- 收集真实用户的使用体验和评价
- 识别并排除厂商软广内容
- 生成结构化的产品分析报告

## 适用产品类型

- 智能猫砂盆（自动清理、除臭、APP控制等）
- 自动喂食器（定时定量、远程控制、摄像头等）
- 智能饮水机（循环过滤、感应出水等）
- 宠物监控摄像头（双向语音、移动追踪等）
- 智能玩具（自动逗猫、远程互动等）
- 其他宠物智能设备

## 工作流程

### 第一步：明确分析目标

与用户确认以下信息：
1. **目标产品名称**（如：小佩智能猫砂盆 MAX）
2. **产品类型**（智能猫砂盆/自动喂食器/饮水机等）
3. **分析深度**（简要对比 / 深度单品评测）
4. **预算范围**（如有，用于推荐替代产品）
5. **特殊关注点**（如：多猫家庭适用性、噪音控制、安全性等）

### 第二步：收集产品基础信息

使用网络搜索获取以下信息：

#### 2.1 官方产品参数
搜索关键词：`[产品名称] 官方参数 规格 功能`

收集内容：
- 产品尺寸、重量、容量
- 核心功能和技术特点
- 材质和安全性认证
- 兼容的猫砂/猫粮类型
- 电源方式和续航
- APP功能和兼容性
- 保修政策和售后服务
- 官方售价

#### 2.2 竞品对比信息
搜索关键词：`[产品类型] 推荐 对比 测评 2024 2025`

收集3-5款主流竞品的基础信息，用于横向对比。

### 第三步：搜索真实用户评测

#### 3.1 小红书真实评测搜索

**搜索策略**：
- 关键词1：`[产品名称] 使用体验 真实`
- 关键词2：`[产品名称] 缺点 问题 避雷`
- 关键词3：`[产品名称] 测评 长期使用`
- 关键词4：`[产品类型] 推荐 真实分享`

**筛选真实用户的技巧**：
- ✅ 优先选择：粉丝数<1万的普通用户
- ✅ 内容特征：有具体使用场景、提到缺点、图片/视频真实
- ✅ 时间分布：发布时间跨度大（非集中发布）
- ❌ 排除信号：粉丝数>10万且内容过于完美、评论区大量统一话术、无缺点提及

#### 3.2 B站评测视频搜索

**搜索策略**：
- 关键词1：`[产品名称] 测评 体验`
- 关键词2：`[产品名称] 开箱 使用`
- 关键词3：`[产品类型] 横评 对比`

**筛选真实评测的技巧**：
- ✅ 优先选择：UP主粉丝数<5万、视频时长>5分钟、有优缺点分析
- ✅ 查看评论区：真实评测通常有用户互动讨论具体使用问题
- ❌ 排除信号：视频过于精美像广告、UP主历史全是品牌推广、评论区控评痕迹明显

**记录信息**：
- 视频标题和UP主名称
- 视频链接
- 核心观点摘要（优点/缺点各3-5条）
- 播放量/点赞数（反映关注度）

### 第四步：软广识别与排除

#### 4.1 软广内容特征

**文字内容信号**：
- 使用过多夸张形容词（"绝绝子"、"YYDS"、"闭眼入"）
- 通篇无缺点或缺点轻描淡写
- 评论区有大量相似话术
- 发布时间集中（如某产品刚上市时大量"测评"）

**视频内容信号**：
- 画面过于精美，像商业广告
- UP主历史内容全是品牌推广
- 视频下方有"赞助"或"合作"标识但未明确说明
- 评论区有大量"已买"、"种草"等统一回复

#### 4.2 真实内容特征

- 有具体的使用场景描述
- 提及产品的具体缺点和使用中的问题
- 图片/视频有生活气息，非专业拍摄
- 评论区有真实用户讨论使用问题
- 发布时间跨度大

### 第五步：分析框架

#### 5.1 智能猫砂盆分析维度

| 维度 | 分析要点 |
|------|----------|
| **清洁效果** | 铲屎干净程度、是否粘底、除臭效果 |
| **噪音控制** | 运行噪音分贝、夜间使用影响 |
| **安全性** | 防夹猫设计、传感器灵敏度、紧急停止 |
| **容量适配** | 适合猫咪体重、多猫家庭适用性 |
| **耗材成本** | 专用猫砂价格、滤网/除臭剂更换成本 |
| **APP体验** | 功能实用性、连接稳定性、数据准确性 |
| **清洁维护** | 拆卸清洗难度、部件耐用性 |
| **性价比** | 功能与价格匹配度、长期使用成本 |

#### 5.2 自动喂食器分析维度

| 维度 | 分析要点 |
|------|----------|
| **出粮精准度** | 定量准确性、是否卡粮 |
| **密封保鲜** | 防潮效果、粮食新鲜度保持 |
| **容量适配** | 适合猫咪数量、出粮频率 |
| **摄像头功能** | 画质清晰度、夜视效果、视角范围 |
| **APP体验** | 定时设置灵活性、远程控制稳定性 |
| **供电安全** | 断电保护、备用电池续航 |
| **清洁维护** | 拆卸清洗难度、死角清洁 |
| **性价比** | 功能与价格匹配度 |

#### 6.2 评分标准

- **9-10分**：强烈推荐，无明显短板
- **7-8分**：值得考虑，有小瑕疵但不影响使用
- **5-6分**：中规中矩，需权衡优缺点
- **3-4分**：不太推荐，有明显问题
- **1-2分**：强烈不推荐，存在严重缺陷

## 输出格式

默认输出为 Markdown 格式文档，可根据需要转换为：
- Word 文档（.docx）
- PDF 文档
- 结构化数据（JSON/CSV）

## 注意事项

1. **时效性**：产品信息和评测内容会随时间变化，建议标注分析日期
2. **主观性**：用户评测带有主观色彩，需综合多个来源形成客观判断
3. **个体差异**：不同猫咪的接受度可能不同，需考虑个体适应性
4. **价格变动**：关注促销节点（618、双11等），价格可能有较大波动
5. **售后政策**：不同渠道购买的售后政策可能不同，需特别说明
