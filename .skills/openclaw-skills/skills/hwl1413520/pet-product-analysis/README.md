# Pet Product Analysis Skill

宠物科技用品专业分析 Skill，用于生成智能猫砂盆、自动喂食器等宠物用品的详细评测报告。

## 功能特点

- 🔍 **智能搜索**：自动搜索产品参数、用户评测、对比信息
- 🎯 **真实内容筛选**：识别并排除厂商软广，聚焦真实用户体验
- 📊 **多维度分析**：基于专业框架进行全方位产品评估
- 📝 **结构化报告**：生成专业的Markdown格式分析报告
- 🔗 **来源可追溯**：所有评测内容标注来源链接

## 适用产品

- 智能猫砂盆（自动清理、除臭、APP控制）
- 自动喂食器（定时定量、远程控制、摄像头）
- 智能饮水机（循环过滤、感应出水）
- 宠物监控摄像头（双向语音、移动追踪）
- 智能玩具（自动逗猫、远程互动）

## 使用方法

### 基本用法

向AI助手描述你的分析需求：

```
帮我分析一下小佩智能猫砂盆MAX，主要是多猫家庭使用，
预算2000以内，关注噪音和除臭效果
```

### 分析流程

1. **明确需求**：确认产品名称、类型、关注维度
2. **信息收集**：搜索官方参数、竞品信息、用户评测
3. **内容筛选**：识别并排除软广内容
4. **深度分析**：基于专业框架进行多维度评估
5. **生成报告**：输出结构化分析报告

## 报告内容

生成的分析报告包含：

- ✅ 产品概述与官方参数
- ✅ 竞品横向对比
- ✅ 小红书真实用户评测汇总
- ✅ B站评测视频链接与摘要
- ✅ 多维度深度分析
- ✅ 购买建议与替代推荐
- ✅ 综合评分与总结

## 软广识别

本Skill内置软广识别机制，通过以下方式筛选真实内容：

- 账号特征分析（粉丝数、历史内容）
- 内容特征识别（关键词、表达方式）
- 评论区验证（互动真实性）
- 交叉验证（多平台对比）

## 注意事项

1. **时效性**：产品信息和评测会随时间变化，建议标注分析日期
2. **主观性**：用户评测带有主观色彩，需综合多个来源
3. **个体差异**：不同猫咪的接受度可能不同
4. **价格变动**：关注促销节点，价格可能有较大波动
