# 平台适配指南

各平台的详细适配说明和注意事项。

## 小红书

### 平台特点
- 主要用户群体：年轻女性
- 内容类型：图文笔记、短视频
- 推荐比例：3:4 竖版

### 登录方式
```yaml
login_type: qrcode
```
- 使用小红书 App 扫码登录
- Cookie 有效期约 30 天

### 内容限制
| 项目 | 限制 |
|------|------|
| 标题 | 20 字 |
| 正文 | 1000 字 |
| 图片 | 18 张 |
| 标签 | 10 个 |
| 视频 | 15 分钟 |

### 特殊注意
- 外链会被屏蔽
- 敏感词会触发审核
- 商业内容需标注

| 图片 | 35 张（图文模式） |
| 标签 | 5 个 |

## 适配策略

### 图片处理优先级

1. **小红书 3:4 → 抖音 9:16**
   - 首选：模糊填充
   - 备选：纯色填充
   - 不推荐：直接拉伸

2. **小红书 1:1 → 视频号 9:16**
   - 首选：上下加边
   - 备选：智能裁剪

3. **小红书 4:3 → 快手 9:16**
   - 首选：智能裁剪
   - 备选：模糊填充

## 常见问题

### Q: 视频号登录失败？

A: 视频号必须使用微信扫码登录：
1. 确保已安装微信桌面版
2. 打开微信并保持登录状态
3. 运行脚本时会弹出二维码
4. 使用手机微信扫码确认

### Q: 抖音发布被限流？

A: 可能原因：
- 发布频率过高
- 内容重复度太高
- 被判定为营销号

解决方案：
- 降低发布频率
- 修改标题和描述
- 增加内容原创度

### Q: 快手图片显示异常？

A: 快手对图片格式要求严格：
- 使用 JPEG 格式
- 文件大小不超过 50MB
- 建议使用 1080x1920 分辨率

## 最佳实践

1. **错峰发布**
   - 各平台间隔 30 分钟以上
   - 避免同一时间大量发布

2. **内容差异化**
   - 针对不同平台调整标题
   - 使用平台热门标签

3. **图片优化**
   - 使用高质量原图
   - 适当压缩文件大小
   - 保持清晰度

4. **互动管理**
   - 及时回复评论
   - 关注数据反馈
   - 调整发布策略
