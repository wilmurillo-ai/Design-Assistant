## 三、架构说明

### CPU架构

#### Intel Raptor Lake (13/14代)
- **制程**：Intel 7 (10nm Enhanced SuperFin)
- **特点**：性能核(P-Core) + 能效核(E-Core)混合架构
- **P-Core**：Raptor Cove架构，单核性能强
- **E-Core**：Gracemont架构，多线程效率高
- **缓存**：L3缓存最大36MB
- **内存**：支持DDR4-3200和DDR5-5600

#### AMD Zen 4 (Ryzen 7000)
- **制程**：TSMC 5nm (CCD) + 6nm (IOD)
- **特点**：全大核设计，IPC提升13%
- **缓存**：L3缓存最大96MB (3D V-Cache版本)
- **内存**：仅支持DDR5-5200
- **PCIe**：支持PCIe 5.0

#### AMD Zen 3 (Ryzen 5000)
- **制程**：TSMC 7nm
- **特点**：统一8核CCX设计，延迟降低
- **缓存**：L3缓存最大96MB (5800X3D)
- **内存**：支持DDR4-3200

### GPU架构

#### NVIDIA Ada Lovelace (RTX 40系列)
- **制程**：TSMC 4N (定制5nm)
- **特点**：
  - 第三代RT Core，光追性能翻倍
  - 第四代Tensor Core，DLSS 3支持帧生成
  - Shader Execution Reordering (SER)
  - 光流加速器
- **显存**：GDDR6X，最高24GB

#### NVIDIA Ampere (RTX 30系列)
- **制程**：Samsung 8nm
- **特点**：
  - 第二代RT Core
  - 第三代Tensor Core
  - 支持DLSS 2.0
- **显存**：GDDR6/GDDR6X

#### AMD RDNA 3 (RX 7000系列)
- **制程**：TSMC 5nm (GCD) + 6nm (MCD)
- **特点**：
  - Chiplet小芯片设计
  - 第二代Infinity Cache
  - 支持AV1编解码
  - 支持FSR 3帧生成
- **显存**：GDDR6

#### AMD RDNA 2 (RX 6000系列)
- **制程**：TSMC 7nm
- **特点**：
  - 第一代Infinity Cache
  - 支持硬件光追
  - 支持FSR
- **显存**：GDDR6

#### Intel Alchemist (Arc系列)
- **制程**：TSMC 6nm
- **特点**：
  - Xe HPG微架构
  - 支持硬件光追
  - 支持XeSS超采样
  - 支持AV1编解码
- **显存**：GDDR6
