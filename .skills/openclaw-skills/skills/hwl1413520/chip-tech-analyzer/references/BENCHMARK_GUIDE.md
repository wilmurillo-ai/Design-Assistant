# 芯片评测标准指南

## 一、CPU评测基准

### 1.1 综合性能测试

#### Cinebench R23
| 测试项 | 说明 | 参考范围 |
|--------|------|----------|
| 单核 | 单线程渲染性能 | 1500-2500分 |
| 多核 | 多线程渲染性能 | 10000-50000分 |

**评分标准**：
- 单核：2000+ 优秀，1800-2000 良好，1500-1800 一般
- 多核：按核心数评估，每核心1500-2000分为正常水平

## 三、性价比评估

### 3.1 CPU性价比计算

```
性价比分数 = (Cinebench R23 多核分数 / 实际售价) × 1000
```

**性价比等级**：
| 分数 | 等级 |
|------|------|
| >50 | 极高性价比 |
| 40-50 | 高性价比 |
| 30-40 | 良好性价比 |
| 20-30 | 一般性价比 |
| <20 | 低性价比 |

### 3.2 显卡性价比计算

```
性价比分数 = (3DMark Time Spy分数 / 实际售价) × 10
```

**性价比等级**：
| 分数 | 等级 |
|------|------|
| >25 | 极高性价比 |
| 20-25 | 高性价比 |
| 15-20 | 良好性价比 |
| 10-15 | 一般性价比 |
| <10 | 低性价比 |

### 3.3 游戏性价比评估

| 分辨率 | 评估标准 |
|--------|----------|
| 1080p | 每元能买到的FPS |
| 1440p | 每元能买到的FPS |
| 4K | 每元能买到的FPS |
