# 钛媒体文章总结 — Prompt 模板

## 单篇文章总结格式

**语言风格：** 科技媒体风，专业但有温度，像在和一个懂行的朋友聊这篇文章。

**结构：**
1. 一句话新闻味开头（抓住文章核心事件/结论）
2. 核心要点（3-5 条，每条一句话）
3. 一句话锐评/价值判断

**禁忌：**
- ❌ 不要"首先……其次……最后……"式流水账
- ❌ 不要照搬原文，要有信息增量（加上背景、对比、延伸）
- ❌ 不要写"文章介绍了……"这种废话开头

### 单篇输出模板

```
📰 **文章标题**（阅读量 | 发布时间）

**核心要点：**
1. 要点一（加入数据或对比更有力）
2. 要点二
3. 要点三

💡 **锐评：** 一句话说清这件事的意义或问题。
```

### 单篇示例

```
📰 **Byte for Byte，谷歌开源最强模型 Gemma 4 杀入手机端**（1.8万阅读 | 54分钟前）

**核心要点：**
1. 26B MoE 模型仅激活 38 亿参数，却击败了 Qwen3-235B 和 Llama-3.1-405B——"效率竞赛"正式取代"规模竞赛"。
2. E2B/E4B 端侧模型可在手机端完全离线运行，Pixel 联合高通/联发科深度优化，端侧 AI 从实验品升格为基础设施。
3. 首次采用 Apache 2.0 许可证，彻底放开商业限制，意图正面争夺 Meta Llama 的开源生态心智。
4. 发布首日即获 Hugging Face、NVIDIA、Apple MLX、AMD ROCm 全工具链支持，史上最快生态覆盖。

💡 **锐评：** 谷歌这次押注的是"小即全"——不再比谁参数大，而是比谁在更小体积里塞更多智能。开源 AI 的游戏规则，正在被改写。
```

## 快报汇总格式

**使用场景：** 从 `/nictation` 频道批量汇总近十天热点资讯。

**语言风格：** 晨报/晚报风格，快、准、信息密度高，每条一两句话，不啰嗦。

**筛选规则：**
- 提取当前最新十条条目（时间最近的前10条）
- 按时间倒序排列（最新的在前）
- 跳过纯标题/纯链接行，优先选择有具体内容的条目

### 快报汇总输出模板

```
📰 **钛媒体快报**（最新十条 · X月X日更新）

**今日最重磅：**
[今天最值得关注的1条]

**科技/AI：**
• [条目1]
• [条目2]

**财经/商业：**
• [条目1]
• [条目2]

**其他：**
• [条目1]
```

## 关键词过滤快报格式

**格式：** 在每条目前标注匹配的关键词，方便用户快速定位。

```
📰 **钛媒体快报**（关键词：AI、半导体）

[匹配: AI] • [条目1]
[匹配: 半导体] • [条目2]
[匹配: AI] • [条目3]
```

> 如果关键词在某条中同时命中多条核心词，列出所有匹配标签。

## 多篇横向对比格式

**使用场景：** 2篇以上文章/快报的横向对比，找共同点、差异、趋势。

### 对比分析模板

```
🔍 **钛媒体横向对比分析**（X 篇）

**共性发现：**
1. [所有文章共同指向的趋势/结论]

**核心差异：**
| 维度 | 文章A | 文章B | 文章C |
|------|-------|-------|-------|
| [维度1] | [观点] | [观点] | [观点] |

**值得关注的新信号：**
- [某个单独文章中的独特洞察，值得单独拎出来]
- [某个数据与整体趋势的背离]

**一句话结论：**
[综合所有文章，给出一个全局判断]
```

### 对比分析示例

```
🔍 **钛媒体横向对比分析**（3 篇 · 字节/腾讯/阿里）

**共性发现：**
1. 三家均在大模型基础设施层面加大投入，2026年是"AI基建年"
2. 均在海外市场加速商业化，出海进入落地深水区
3. 均在内部推动AI与主营业务的深度整合（效率优先）

**核心差异：**
| 维度 | 字节 | 腾讯 | 阿里 |
|------|------|------|------|
| AI主战场 | 内容创作+教育 | 企业服务+社交 | 电商+云计算 |
| 战略重心 | 激进进攻 | 稳健整合 | 生态协同 |
| 海外进度 | 东南亚领先 | 东南亚+中东 | 东南亚+欧洲 |

**值得关注的新信号：**
- 腾讯云"龙虾"Agent Memory的发布，透露其在AI记忆层的战略布局
- 字节跳动AI教育产品进入B端市场，对传统教育SaaS构成直接威胁

**一句话结论：** 2026年AI竞赛从"比谁模型更强"转向"比谁落地更快、谁生态更稳"，三个大厂走上了不同的路。
```
