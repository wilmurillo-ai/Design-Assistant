# 小红书文案模板

## 标题公式

1. **数字型**: `X个方法/Y种技巧/Z天见效`
   - 例: `7天瘦3斤！超实用减脂食谱`

2. **痛点型**: `为什么你总是.../别再...了`
   - 例: `为什么你总是瘦不下来？真相来了`

3. **悬念型**: `竟然是.../没想到...`
   - 例: `没想到这个方法这么有效！`

4. **干货型**: `保姆级教程/超详细攻略`
   - 例: `保姆级教程！从0到1做小红书`

## 正文结构

```markdown
[痛点引入] (1-2句，引发共鸣)

📍 核心内容要点：
- 要点1
- 要点2  
- 要点3

💡 为什么值得关注：
- 理由1
- 理由2

💬 互动引导：
你怎么看？评论区聊聊👇
```

## 新闻类模板

```markdown
今天国际局势又变了！

[新闻摘要]

📍 发生了什么？
- 事件要点1
- 事件要点2
- 事件要点3

💡 为什么值得关注？
- 影响1
- 影响2

💬 你怎么看？评论区聊聊👇
```

## 标签建议

通用标签组合:
- `#今日热点 #时事热点 #新闻早知道`
- `#干货分享 #实用技巧`
- `#生活记录 #日常分享`

内容相关标签:
- 根据主题添加 3-5 个垂直标签

## 避坑指南

❌ 不要用:
- 极限词: 最、第一、绝对、必须
- 夸大词: 惊人、震惊、不可思议
- 虚假信息
- 侵犯版权的图片

✅ 推荐用:
- 真实分享
- 有价值内容
- 原创图片
- 温和表达
