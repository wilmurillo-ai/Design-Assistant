# 弃程票风险评估

## 风险等级

| 风险类型 | 严重程度 | 发生概率 | 缓解措施 |
|---------|---------|---------|---------|
| 行李被运到最终目的地 | 高 | 100%（如托运） | **绝不托运**，只带手提行李 |
| 返程票被取消 | 高 | 100% | 仅用于单程或回程最后一段 |
| 常旅客资格受影响 | 中 | 中 | 不使用会员号预订，或接受风险 |
| 航班变动导致路线改变 | 中 | 低 - 中 | 购买时选择直飞段稳定的航班 |
| 被航空公司列入黑名单 | 低 | 低 | 偶尔使用，不要频繁 |

## 详细风险说明

### 1. 行李问题

**风险**：托运行李会直挂到机票显示的最终目的地。

**案例**：你买了北京→西安→长沙的票，想在西安下机。但你的行李箱会被运到长沙。

**缓解**：
- 只携带符合尺寸要求的手提行李
- 注意：航班满员时，手提行李可能被要求免费托运
- 如被要求托运，可尝试说明情况，但行李仍可能运到最终目的地

### 2. 后续航段取消

**风险**：放弃任何非最后一段航程，航空公司系统会自动取消所有后续航段。

**案例**：你买了北京→西安→长沙的往返票。去程在西安下机后，返程票（长沙→西安→北京）会被自动取消。

**缓解**：
- 仅用于单程票
- 或仅用于往返票的回程最后一段
- 不要购买包含多段的复杂行程

### 3. 常旅客计划

**风险**：航空公司可能：
- 取消本次飞行累积的里程
- 扣除账户中已有里程
- 取消会员资格
- 禁止未来乘坐该航空公司航班

**缓解**：
- 预订时不输入常旅客号码
- 不要频繁对同一家航空公司使用此策略
- 接受可能的损失（如里程价值不高）

### 4. 航班变动

**风险**：如原航班取消/延误，航空公司可能将你重新安排到不经过你实际目的地的航线。

**案例**：你买了北京→西安→长沙，想在西安下机。但航班取消后，航空公司可能改为你乘坐北京→成都→长沙，完全避开西安。

**缓解**：
- 选择航班稳定性高的航空公司
- 购买时查看历史准点率
- 预留备选方案

### 5. 法律与合同

**风险**：违反航空公司承运合同，但不违法。

**缓解**：
- 了解并接受风险
- 不要与航空公司工作人员发生冲突
- 如被询问，简单说明"有急事"即可

## 何时绝对不要使用

- ✗ 需要托运行李
- ✗ 购买了往返票且需要使用返程
- ✗ 行程中包含国际航班（可能导致入境问题）
- ✗ 对时间极其敏感（如参加重要会议、婚礼）
- ✗ 是航空公司高级会员且珍视该身份
- ✗ 带着儿童或需要特殊照顾的乘客

## 决策框架

```
1. 能省多少钱？
   - <100 元：不建议，风险/收益比不佳
   - 100-300 元：可考虑，但需评估其他因素
   - >300 元：值得认真考虑

2. 是否满足安全使用条件？
   - 单程或回程最后一段？✓
   - 无需托运行李？✓
   - 不介意可能的会员影响？✓
   - 行程不敏感？✓
   → 全部满足才可考虑

3. 备选方案？
   - 直飞价格是否可接受？
   - 其他中转方案是否更便宜？
   - 改天出行是否更便宜？
```
