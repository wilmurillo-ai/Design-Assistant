# 中国航空公司弃程票政策汇总

## 主要航空公司态度

### 中国国际航空 (Air China)
- **官方政策**：合同禁止 skip segment
- **实际执行**：相对宽松，偶尔使用一般无问题
- **会员计划**：凤凰知音，频繁使用可能影响里程
- **建议**：偶尔使用可行，不要使用会员号

### 东方航空 (China Eastern)
- **官方政策**：合同禁止
- **实际执行**：中等严格
- **会员计划**：东方万里行
- **特点**：上海、昆明、西安基地航班多，弃程票机会多
- **建议**：谨慎使用

### 南方航空 (China Southern)
- **官方政策**：合同禁止
- **实际执行**：相对严格
- **会员计划**：明珠俱乐部
- **特点**：广州基地，华南航线密集
- **建议**：不建议频繁使用

### 海南航空 (Hainan Airlines)
- **官方政策**：合同禁止
- **实际执行**：中等
- **会员计划**：金鹏俱乐部
- **特点**：旗下有多家廉航（祥鹏、西部、九元）
- **建议**：可使用其廉航品牌，风险较低

### 四川航空 (Sichuan Airlines)
- **官方政策**：未明确说明
- **实际执行**：相对宽松
- **会员计划**：金熊猫
- **特点**：成都基地，西南航线多
- **建议**：可考虑使用

### 春秋航空 (Spring Airlines)
- **官方政策**：廉航，票价已最低，弃程票意义不大
- **实际执行**：N/A
- **特点**：本身就是最低价，很少有更便宜的中转票
- **建议**：直接购买即可，无需弃程

### 厦门航空 (Xiamen Airlines)
- **官方政策**：合同禁止
- **实际执行**：中等
- **会员计划**：白鹭俱乐部
- **特点**：天合联盟成员，服务较好
- **建议**：谨慎使用

## 被发现的可能后果

### 轻微后果（偶尔使用）
- 本次飞行里程不计入
- 无其他影响

### 中等后果（多次使用）
- 账户中部分里程被扣除
- 会员等级降级
- 收到航空公司警告

### 严重后果（频繁使用/大量使用）
- 会员资格取消
- 被列入内部黑名单
- 禁止购买该航空公司机票
- 可能影响同一联盟的其他航空公司

## 降低风险的建议

1. **不要使用常旅客号码**
   - 预订时不输入会员号
   - 事后也不要求补登里程

2. **不要频繁对同一家航空公司使用**
   - 分散使用不同航空公司
   - 每年对同一家航空公司不超过 1-2 次

3. **不要购买高价舱位**
   - 经济舱最低折扣票即可
   - 避免商务舱/头等舱弃程（损失大且引人注意）

4. **不要团体使用**
   - 单人使用风险最低
   - 多人同时使用容易引起注意

5. **保持低调**
   - 不要主动告知工作人员
   - 如被询问，简单说明"有急事"
   - 不要与工作人员争论

## 国际航班警告

**强烈不建议对国际航班使用弃程票策略**，原因：

1. **入境问题**：可能涉及签证和入境记录问题
2. **海关风险**：行李和海关申报可能有问题
3. **后果严重**：可能影响未来签证申请
4. **价格差异小**：国际航班定价更合理，弃程票优势不明显

**仅限中国国内航班使用此策略。**
