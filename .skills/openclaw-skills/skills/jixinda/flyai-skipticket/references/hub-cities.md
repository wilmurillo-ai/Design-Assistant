# 中国国内热门中转枢纽城市

## 主要航空枢纽

### 北京 (PEK/PKX)
- **枢纽航司**：中国国际航空、海南航空
- **适合中转**：东北、西北、华北航线
- **特点**：航班密集，中转选择多

### 上海 (PVG/SHA)
- **枢纽航司**：东方航空、春秋航空
- **适合中转**：华东、华南、国际航线
- **特点**：春秋航空基地，廉价航班多

### 广州 (CAN)
- **枢纽航司**：南方航空
- **适合中转**：华南、东南亚航线
- **特点**：南航主基地，国内国际连接好

### 成都 (TFU/CTU)
- **枢纽航司**：四川航空、中国国际航空
- **适合中转**：西南、西北航线
- **特点**：西部重要枢纽，进藏航班多

### 昆明 (KMG)
- **枢纽航司**：东方航空、祥鹏航空
- **适合中转**：西南、东南亚航线
- **特点**：廉价航空多，中转免费住宿政策

### 西安 (XIY)
- **枢纽航司**：东方航空、海南航空
- **适合中转**：西北、西南航线
- **特点**：西北门户，航班价格常有优势

### 重庆 (CKG)
- **枢纽航司**：四川航空、西部航空
- **适合中转**：西南、西北航线
- **特点**：西部航空基地，廉价航班多

### 深圳 (SZX)
- **枢纽航司**：深圳航空、海南航空
- **适合中转**：华南、国际航线
- **特点**：商务航班多，价格相对较高

## 中转免费住宿政策

部分机场/航司提供中转免费住宿：

- **昆明长水机场**：经昆明中转的省内或暹粒机场旅客可享免费隔夜住宿
- **广州白云机场**：南航部分国际中转提供住宿
- **成都天府机场**：川航部分中转提供住宿

**注意**：使用弃程票时无法享受此服务，因为需要完成全部行程。

## 搜索技巧

1. **优先搜索廉航基地出发的航班**
2. **避开商务干线**（如京沪、广深）
3. **关注旅游城市**（如昆明、三亚）作为最终目的地
4. **考虑反向搜索**：从目的地反向搜索出发地
