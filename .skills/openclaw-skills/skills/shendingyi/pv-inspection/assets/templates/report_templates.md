## 导出说明

### 导出为 Word
1. 复制 Markdown 内容
2. 粘贴到 Word
3. 调整格式
4. 保存为 .docx

### 导出为 PDF
1. 使用浏览器打开 Markdown
2. 打印 → 另存为 PDF
3. 或使用 pandoc 转换

```bash
pandoc report.md -o report.pdf
```
