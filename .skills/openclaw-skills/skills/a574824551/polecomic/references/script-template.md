# 剧本格式规范参考

## 一、剧本结构（三幕式）

### 第一幕：起（Setup）—— 约占总时长 25%
- **钩子**（Hook）：前 30 秒抓住观众
- **世界观呈现**：用最少的信息量建立背景
- **主角登场**：展示性格与处境
- **触发事件**（Inciting Incident）：打破平衡的关键事件

### 第二幕：承转（Confrontation）—— 约占总时长 50%
- **上升行动**：主角追求目标，障碍逐渐增大
- **中点转折**（Midpoint）：局势反转或新信息揭露
- **危机加深**：主角陷入最低点（All Is Lost moment）

### 第三幕：合（Resolution）—— 约占总时长 25%
- **高潮对决**：最终冲突
- **结局**：目标达成/失败，情感落点
- **余韵**（可选）：为续集埋下伏笔

## 二、场景格式

```
═══════════════════════════════════════
【场景 01】室外·废弃工厂·傍晚
───────────────────────────────────────
[画面：橙红夕阳将工厂铁架拉出长影。锈蚀的铁门半掩。]
[音效：风声，远处城市嘈杂声渐渐淡出]

林晨（从铁门缝隙中侧身而入，手持手电筒，神色警惕）：
"来了三次了……每次都差一点。"

[画面：手电光扫过墙壁，定格在一组奇异符文上。]

系统音（机械感）：
「检测到能量残留——强度：S 级」

林晨（倒吸一口气，后退半步）：
"怎么可能……"

[音效：符文突然发出蓝色荧光，地板开始震动]
[情绪：震惊→恐惧→决意]
═══════════════════════════════════════
```

### 字段说明
| 字段 | 格式 | 说明 |
|------|------|------|
| 场景头 | `【场景 XX】地点·子地点·时间` | 时间可选：清晨/白天/黄昏/夜晚 |
| 画面描述 | `[画面：……]` | 摄影机看到的内容，客观描述 |
| 音效/音乐 | `[音效：……]` / `[BGM：……]` | 注明情绪功能 |
| 台词 | `角色名（动作/情绪）：「台词」` | 括号内为表演指示 |
| 情绪标注 | `[情绪：A→B→C]` | 帮助后续表情动画设计 |

## 三、台词写作原则

1. **每句话都有目的**：揭示性格、推进剧情、或建立关系
2. **少用解释性台词**（Avoid Exposition Dumps）：用动作代替说明
3. **每个角色有独特语气**：语速、用词、句式差异化
4. **动画台词宜短**：单句不超过 20 字为佳（配合口型动画）

### 反例 vs 正例
```
❌ "我叫林晨，我从小就在这座城市长大，三年前失去了双亲，
    所以我才会选择成为一名调查员，专门追查异常能量事件。"

✅ 林晨（看着手中的徽章，沉默片刻）：
   "三年了。"
   （将徽章收入口袋，转身走入工厂）
```

## 四、类型剧本要点速查

### 动作/热血
- 高频率景别切换（近景-全景交替）
- 动作描述精确到动作帧（"出拳→格挡→反击"）
- 台词短促有力，多感叹句

### 悬疑/推理
- 信息碎片化投放，留白大于说明
- 视角限制（不给观众主角不知道的信息）
- 音效强调关键线索

### 恋爱/日常
- 大量表情特写（分镜重点）
- 内心独白用斜体标注
- 对话节奏舒缓，注重"停顿"的戏剧性

### 奇幻/世界观
- 世界观说明书（放剧本前，供创作团队参考，不出现在画面）
- 名词表（专有名词统一用法）
- 能力/规则可视化描述
