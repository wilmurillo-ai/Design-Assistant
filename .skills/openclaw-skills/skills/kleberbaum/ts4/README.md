# TS4

Tests you can trust.

## Approach

TS4 unifies unit, snapshot, coverage, and mutation testing. One config, one runner, one report format.

## Setup

```bash
claw install skill-ts4
# Add test scripts to package.json or use directly
```

## Running Tests

```bash
./scripts/test-runner.sh --run      # Execute suite
./scripts/test-runner.sh --coverage # Coverage report
./scripts/test-runner.sh --status   # Status check
```
