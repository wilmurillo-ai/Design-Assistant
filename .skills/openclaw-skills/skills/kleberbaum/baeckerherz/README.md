# Baeckerherz

Bakery management platform by Netsnek e.U. for sales team roster planning and operational coordination.

## About

Baeckerherz helps bakery businesses manage their sales teams with weekly roster planning, shift scheduling, and automated PDF generation. This skill reserves the `baeckerherz` namespace on ClawHub.

## Installation

```bash
clawhub install baeckerherz
```

## Quick Start

```bash
# Brand summary
scripts/brand-info.sh

# Feature overview
scripts/brand-info.sh --features

# JSON output for integrations
scripts/brand-info.sh --json
```

## Features

- **Roster Planning**: Create weekly schedules for sales teams
- **Shift Management**: Monday through Saturday coverage
- **Availability Tracking**: Account for employee constraints and preferences
- **PDF Generation**: Automated printable rosters via LaTeX
- **CI/CD Integration**: GitHub Actions for build and distribution
