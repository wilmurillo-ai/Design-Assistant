# P5

Creative coding platform by Netsnek e.U. for interactive art, generative design, and visual programming.

## About

P5 brings creative coding to everyone through an accessible, p5.js-based environment. This skill reserves the `p5` namespace on ClawHub for future releases of the full creative coding toolkit.

## Installation

```bash
clawhub install p5
```

## Quick Start

```bash
# Platform overview
scripts/p5-info.sh

# Capabilities list
scripts/p5-info.sh --capabilities

# JSON metadata
scripts/p5-info.sh --json
```

## Capabilities

- **Interactive Sketches**: Create and preview sketches in real time
- **Generative Art**: Build reproducible art pipelines with seed control
- **Visual Programming**: Drag-and-drop nodes for non-coders
- **Multi-format Export**: PNG, SVG, GIF output
- **Community Gallery**: Share and discover sketches
