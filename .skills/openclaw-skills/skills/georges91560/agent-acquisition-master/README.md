## Ce que Agent fait

**5 canaux d'acquisition autonomes**
- Cold email personnalisé — séquence 8 touchpoints, signal-based
- LinkedIn outreach — view → like → connect → message
- Contenu organique — Twitter, LinkedIn, Reddit, Substack
- Funnel de capture — landing page → email → nurturing → vente
- Qualification — détecte les leads chauds, notifie Telegram immédiatement

**7 crons automatiques**
Agent prospecte, publie, envoie, relance et qualifie du lundi au vendredi.
Rapport de synthèse chaque lundi avec 3 ajustements proposés.

**Apprentissage continu**
Log immédiat après chaque action dans LESSONS.md, HOOKS.md, TEMPLATES_PERF.md.
Agent s'améliore toutes les semaines sans intervention.

**Closing avancé**
CLOSER Hormozi · SPIN Selling · Challenger Sale · Grand Slam Offer.
Préparation rdv automatique + Google Sheet tracker + alertes MRR Telegram.
