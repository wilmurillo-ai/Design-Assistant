# content-creator-pro

✍️ **Human-first autonomous content engine for agents.**

shark-mindset detects the signal. acquisition-master distributes.
This skill creates the content — written with authentic voice,
platform-native, and optimized for engagement.

---

## The Non-Negotiable

**AI-generated content is detectable. Detectable content destroys credibility.**

This skill is built around one principle: every piece of content
it produces must pass the stranger test — a person who doesn't know
you reads it and thinks "real person, real experience," not "generated."

---

## What It Does

- **Authentic voice system** — quality blacklist, structural patterns, editorial rewrite protocol
- **Hook mastery** — 10 archetypes, platform-specific rules, validated library
- **5 storytelling frameworks** — Moment of Change, BAB, Contrarian, Step-by-Step, Observation
- **Long-form article architecture** — lead, core argument, turn, close
- **Brand voice system** — reads voice.md before every piece
- **4 content modes** — signal-reactive, calendar-driven, on-demand, repurposing
- **Performance tracking** — logs every post, identifies top hooks, learns weekly
- **AUDIT.md logging** — viral alerts, weekly summary, voice creation events logged for agent core
