# Storytelling Reference — Proof Engine

## The 5 Story Arcs

### ARC 1 — TRANSFORMATION
Best for: Monthly updates, testimonials, personal milestones

```
BEFORE:    "I was [struggling with X — specific, relatable]"
TRIGGER:   "Then I [built/discovered/tried Y — concrete action]"
RESULT:    "[Specific outcome with real numbers]"
MISSION:   "That's why I document this publicly"

Example:
BEFORE:  "6 months ago I was manually checking charts 3 hours a day"
TRIGGER: "I built an AI agent to do it for me"
RESULT:  "Last month it generated €2,847 while I was asleep"
MISSION: "I share every step so you can build yours"
```

### ARC 2 — THE SYSTEM
Best for: Product launches, course promotions, service offers

```
PROBLEM:  "Most people do X the hard way [describe the pain]"
SYSTEM:   "I built a system that [solves X automatically]"
PROOF:    "[Specific result] achieved in [timeframe]"
OFFER:    "Here's how to build yours / Here's the shortcut"

Example:
PROBLEM: "Most traders lose because emotion overrides logic"
SYSTEM:  "My trading agent executes only when 3 signals align"
PROOF:   "62% win rate over 90 days — no emotional trades"
OFFER:   "The signal framework is in my newsletter"
```

### ARC 3 — THE MISTAKE
Best for: Trust-building content, transparency posts, authority building

```
MISTAKE:  "I spent €X / wasted X months on [wrong thing]"
LEARNING: "What I discovered was [counter-intuitive insight]"
FIX:      "Now I do [new approach — specific]"
RESULT:   "[Better outcome with numbers]"

Example:
MISTAKE:  "I spent 4 months trying to build a SaaS product"
LEARNING: "Nobody buys from someone they've never heard of"
FIX:      "I built audience first — 3K followers before any offer"
RESULT:   "First launch: €4,200 in 72 hours"
```

### ARC 4 — THE MILESTONE
Best for: Follower milestones, revenue milestones, platform growth

```
CONTEXT:  "X months/weeks ago I [starting point — humble]"
JOURNEY:  "3 things that actually made the difference:"
TODAY:    "[Current milestone — specific number]"
NEXT:     "Where this goes from here [creates anticipation]"

Example:
CONTEXT: "8 months ago I had 0 followers and €0 automated revenue"
JOURNEY: "3 things changed everything: [1] [2] [3]"
TODAY:   "Today: 4,200 followers + €3,100/month autonomous income"
NEXT:    "By December: €10K/month. I'll document every step."
```

### ARC 5 — THE PROOF (for sales)
Best for: VSL scripts, sales emails, landing page copy

```
CLAIM:    "[Bold promise of the offer]"
SKEPTIC:  "You might be thinking [most common objection]"
PROOF:    "[Specific result that directly addresses the objection]"
BRIDGE:   "Here's exactly how [client name / I] achieved this"
CTA:      "[Clear next step]"

Example:
CLAIM:   "You can build an autonomous income system in 30 days"
SKEPTIC: "You're probably thinking 'I'm not technical enough'"
PROOF:   "[Client] had zero coding experience — now €1,800/month"
BRIDGE:  "Here's the exact 4-step system he followed"
CTA:     "Get the system → [link]"
```

## Platform Adaptation Rules

```
TWITTER THREAD
  Arc length: 5-8 tweets
  Hook tweet: Result only (no context)
  Evidence: At least 1 specific number per 2 tweets
  Close: Question or CTA (not both)

LINKEDIN POST
  Arc length: 200-400 words
  Hook: Bold first line (no "I" opener)
  Format: Short paragraphs, white space
  Close: Question that invites comments

INSTAGRAM REEL SCRIPT
  Arc length: 30-60 seconds spoken
  Hook: First 3 words must stop the scroll
  Pacing: 1 point every 8-10 seconds
  Visual cue: [B-ROLL: screen recording] etc.

EMAIL
  Arc length: 300-500 words
  Subject: Contains the result (open rate driver)
  P.S.: Always a soft CTA or proof repeat

VSL SEGMENT (voice-agent-pro-v3)
  Arc length: 2-4 minutes spoken
  Stacked proof: 3+ proof points minimum
  Pacing: Slower than social — let it breathe
  Output to: /workspace/voice/scripts/
```

## Impact Score → Arc Recommendation

```
Score 9-10 (Hero Proof)
  → Arc 4 (Milestone) + Arc 5 (Proof) for sales
  → Deploy everywhere immediately

Score 7-8 (Strong Proof)
  → Arc 1 (Transformation) or Arc 2 (System)
  → Deploy in funnels + brand content

Score 5-6 (Supporting Proof)
  → Arc 3 (Mistake) or Arc 2 (System)
  → Deploy in nurture content + outreach

Score 3-4 (Context Proof)
  → Arc 3 (Mistake) only
  → Use in email sequences or stories

Score 1-2 (Archive)
  → No story generated
  → Stored only
```
