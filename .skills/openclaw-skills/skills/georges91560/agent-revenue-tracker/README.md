# revenue-tracker

📈 **Live revenue intelligence for autonomous agents.**

Every euro tracked. Every stream visualized. Every milestone celebrated.
Every anomaly caught before it becomes a problem.

**v1.1.0** — Auto-updating dashboard: Google Sheets → Notion → HTML fallback.

---

## What It Does

- **5 revenue streams**: subscriptions, trading, content, services, affiliate
- **ASCII dashboard**: visual MRR trend, stream breakdown, goal progress bars
- **Auto-updating HTML dashboard**: dark-mode, auto-refresh 60s — generated after every event
- **Google Sheets sync**: after every event, agent updates "Veritas Revenue Tracker" sheet
- **Notion fallback**: if Sheets unavailable, agent updates Notion database
- **HTML local fallback**: always generated by the script, zero external dependencies
- **Milestone alerts**: agent sends Telegram alert on every milestone crossed
- **Anomaly detection**: churn spikes, revenue gaps, negative MRR
- **Learning log**: revenue intelligence that compounds over time
- **Asset tracker**: trading capital, tools, anything of value
- **Goal system**: set MRR targets with deadlines and track progress

---

## Auto-Dashboard Update Flow

After **every** `record` command, the script:

```
1. Generates /workspace/revenue/reports/dashboard.html  ← always
2. Outputs structured instructions → agent executes:
   Priority 1 → Google Sheets (via wesley-web-operator / gog)
   Priority 2 → Notion (via virtual-desktop)
   Priority 3 → HTML local (already done by script)
```

**Google Sheets structure** (agent creates on first use):

| Tab | Content |
|---|---|
| `Events` | Date, Stream, Type, Amount, Description, MRR after |
| `Dashboard` | MRR, ARR, Total, Customers, Phase — live cells |
| `By Stream` | MRR per stream + % share |
| `Goals` | Target, Current, % progress, Deadline, Est. weeks |

## Quick Start

```bash
# Initialize
python3 revenue_tracker.py --init

# Record a sale (auto-updates all dashboards)
python3 revenue_tracker.py record \
  --stream subscriptions --amount 97 --type new_mrr \
  --description "New VIP subscriber"

# Visual ASCII dashboard
python3 revenue_tracker.py dashboard

# Weekly HTML report
python3 revenue_tracker.py report --period week --format html

# Set a goal
python3 revenue_tracker.py goal \
  --set "mrr_10k" --target 10000 --deadline "2026-09-30"

# Log a learning
python3 revenue_tracker.py learn \
  --category acquisition --insight "CTOs reply 14% to signal-based emails" \
  --impact high

# Check anomalies
python3 revenue_tracker.py anomalies
```
