# Research Paper Writer（兼容层）

Creates formal academic research papers following IEEE/ACM formatting standards.

## 迁移指南

**新用法：**
```
paper write introduction 振动控制
paper write related work RTHS
paper write methods 深度学习
```

**旧用法（仍然可用）：**
```
write introduction 振动控制
```

## Standard Paper Structure
1. Title and Abstract
2. Introduction
3. Related Work / Background
4. Methodology / Approach
5. Implementation (if applicable)
6. Evaluation / Experiments / Results
7. Discussion
8. Conclusion

## Formatting Standards
- IEEE format - Electrical engineering, computer science
- ACM format - Computer science
- ASCE format - Civil engineering
- Elsevier - General journals
