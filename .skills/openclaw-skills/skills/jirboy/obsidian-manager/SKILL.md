# Obsidian Manager（兼容层）

Obsidian 知识笔记管理技能，支持笔记创建、检索、双向链接。

## 迁移指南

**新用法：**
```
knowledge obsidian create 标题 方向
knowledge obsidian search 关键词
knowledge obsidian link 笔记 A 笔记 B
knowledge obsidian list 方向
```

**旧用法（仍然可用）：**
```
obsidian create 标题 方向
```
