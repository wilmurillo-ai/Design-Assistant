# 科研文献综述与迭代助手

## 功能

自动检索、结构化总结文献，并根据用户反馈迭代生成综述草稿，支持多轮优化。

**核心功能：**
- ✅ 多源检索（arXiv/PubMed/Google Scholar）
- ✅ 结构化总结（方法、创新点、结果、局限性）
- ✅ 相关性评估（1-10 分）
- ✅ 多轮迭代优化
- ✅ SCI 格式综述草稿
- ✅ 基金申请支持模块

## 安装

技能已放置在 `skills/research-review-assistant/`，OpenClaw 会自动加载。

## 使用方式

### 手动触发

对智能体说：

```
帮我做 RTHS 领域的文献综述
迭代我的论文背景部分
生成基金申请文献支持
帮我调研深度学习中时滞补偿方法
```

### 配置参数

- `max_papers` - 最大检索论文数（默认 30）
- `summary_length` - 总结长度（short/detailed/expert）
- `iteration_rounds` - 迭代轮次（默认 3）

### 定时任务

```bash
openclaw cron add --name "weekly-review" --cron "0 9 * * 1" --message "生成本周文献综述"
```

```markdown
作者：Author et al.
相关性：⭐⭐⭐⭐⭐ (9/10)
研究方法：...
核心创新点：...
建议：值得精读

📈 研究趋势分析
热点方向：...
研究空白：...

📚 参考文献（国标格式）

🔄 迭代日志
v1.0 → v1.1: 加强方法部分
```

## 注意事项

1. 所有事实来自检索结果，避免幻觉
2. 优先高 IF 期刊/高引用预印本
3. 用户可随时调整关键词
4. 支持多轮迭代优化
