# NSFC Writing Guide（兼容层）

NSFC 基金申请书协作撰写指南 - 采用人机协作模式撰写国家自然科学基金申请书。

## 迁移指南

**新用法：**
```
nsfc write 立项依据 振动台控制
nsfc write 研究内容 RTHS
nsfc write 摘要 面上项目
nsfc outline 面上项目 振动控制
```

**旧用法（仍然可用）：**
```
write 立项依据 振动台控制
outline 面上项目 振动控制
```

## 协作原则
- **先理解后写作**：所有修改前必须充分理解用户意图
- **逐步推进**：将申请书分解为可管理的小任务
- **持续反馈**：每个阶段完成后向用户确认
- **版本控制**：重要修改前创建备份
- **进度可见**：使用任务跟踪工具

## 质量保证原则
- **文献零编造**：所有引用必须通过检索验证
- **方法有据**：所有算法、方法必须引用文献支持
- **术语准确**：使用领域标准术语和规范表达

1. 立项依据 ⭐⭐⭐

3. 研究方案 ⭐⭐
4. 特色与创新 ⭐⭐⭐
