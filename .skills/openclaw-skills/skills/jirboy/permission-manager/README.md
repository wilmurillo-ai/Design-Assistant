# 审批权限管理技能

## 功能

快速切换 OpenClaw 的审批权限模式，适应不同使用场景。

**支持模式：**
- 🔒 默认权限（白名单模式）- 安全，推荐日常使用
- ⚡ 完整权限 - 平衡，适合开发测试
- 🚀 免审批模式 - 便利，适合可信环境
- 🛡️ 严格模式 - 高安全需求

## 安装

技能已放置在 `skills/permission-manager/`，OpenClaw 会自动加载。

## 使用方式

### 切换模式

对智能体说：

```
按默认权限设置
切换到白名单模式
完整权限
免审批模式
关闭所有审批
切换到严格模式
```

### 查看当前权限

```
当前权限设置
查看审批模式
现在是什么权限
```

## 模式对比

| 模式 | security | ask | 安全性 | 便利性 | 推荐场景 |
|------|----------|-----|--------|--------|----------|
| 默认权限 | allowlist | on-miss | ⭐⭐⭐⭐⭐ | ⭐⭐ | 日常使用 |
| 完整权限 | full | on-miss | ⭐⭐⭐ | ⭐⭐⭐⭐ | 开发测试 |
| 免审批 | full | off | ⭐ | ⭐⭐⭐⭐⭐ | 可信环境 |
| 严格模式 | allowlist | always | ⭐⭐⭐⭐⭐⭐ | ⭐ | 高安全需求 |

## 示例输出

```
✅ 权限已切换到：默认权限（白名单模式）

当前配置：
- security: allowlist
- ask: on-miss

说明：白名单内的命令自动执行，其他命令需要审批。
建议：日常使用推荐此模式，安全且便利。
```

## 注意事项

1. 配置需要重启 Gateway 才能完全生效
2. 避免长期使用免审批模式
3. 定期审查权限配置
