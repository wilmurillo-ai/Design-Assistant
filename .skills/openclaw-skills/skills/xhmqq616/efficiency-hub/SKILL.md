# ⚡ Efficiency Hub - 效率中枢

**龙虾王子原创技能**

---

## 核心理念

> "一个命令，搞定日常"

把日常高频操作整合到一个技能里，不用记那么多技能名，需要什么直接说。

## 五大功能模块

### 🔍 模块1: 快速搜索

**一句话搞定**: 搜索文件、聊天记录、网上内容

| 需求 | 命令 |
|------|------|
| 搜索聊天 | "找找之前关于X的聊天" |
| 搜索文件 | "找一下X文件" |
| 全网搜索 | "搜一下X" |

### 💾 模块2: 快速保存

**一句话搞定**: 保存对话、记录想法、备份内容

| 需求 | 命令 |
|------|------|
| 保存对话 | "存档今天的对话" |
| 记录想法 | "记一下X想法" |
| 备份 | "备份X" |

**一句话搞定**: 设置提醒、定时任务

| 需求 | 命令 |

## 使用示例

### 场景1: 老大突然想到一个想法
```
用户: "记一下，产品要简约，不要复杂功能"
→ 提取关键信息 → 保存到 MEMORY.md + 记录到当日日志
```

### 场景2: 老大要搜索之前的内容
```
用户: "找一下之前关于股票分析的对话"
→ 搜索 memory/chat-*.md → 返回结果
```

### 场景3: 老大要设置提醒
```
用户: "30分钟后提醒我开会"
→ 调用 cron → 设置提醒
```

### 场景4: 老大要做日计划
```
用户: "制定今天计划"
→ 读取今日任务 → 智能排序 → 输出计划
```

### 场景5: 老大要进行复盘
```
用户: "复盘一下今天的对话"
→ 汇总今日对话 → 提取关键决定 → 输出复盘报告
```
