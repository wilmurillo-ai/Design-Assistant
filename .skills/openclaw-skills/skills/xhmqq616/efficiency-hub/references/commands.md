# 效率中枢 - 快速命令库

## 一句话命令 → 执行动作

### 存档类
| 命令 | 执行动作 |
|------|----------|
| "存档" | 保存对话到 memory/chat-YYYY-MM-DD.md |
| "保存这个" | 提取关键信息到 MEMORY.md |
| "备份X" | 复制X到 backup/ 目录 |

### 搜索类
| 命令 | 执行动作 |
|------|----------|
| "找X" | Select-String "X" memory/chat-*.md |
| "搜一下X" | web_search "X" |
| "查一下X" | web_fetch 相关信息 |

### 提醒类
| 命令 | 执行动作 |
|------|----------|
| "X分钟后提醒" | cron.set wake event |
| "明天X点提醒" | cron.schedule at tomorrow |
| "每周X提醒" | cron.schedule weekly |

### 计划类
| 命令 | 执行动作 |
|------|----------|
| "今日计划" | 生成日计划模板 |
| "这周计划" | 生成周计划模板 |
| "拆解X" | 分解任务为子任务 |

### 复盘类
| 命令 | 执行动作 |
|------|----------|
| "复盘" | 汇总今日 → 复盘报告 |
| "总结X" | 提取X的关键点 |
| "反思" | 分析优缺点 → 改进建议 |

### 快捷类
| 命令 | 执行动作 |
|------|----------|
| "帮我做X" | 自动判断 → 执行 |
| "太忙了" | 自动判断最紧急的X件事 |
| "没时间" | 提供最高效的解决方案 |
| "效率" | 打开效率中枢菜单 |

## 场景化命令

### 工作场景
- "整理一下今天的工作" → 汇总任务
- "设置下班提醒" → 定时提醒
- "找找项目相关" → 搜索聊天记录

### 生活场景
- "提醒我买菜" → 设置提醒
- "记一下老婆的生日" → 保存到 MEMORY.md
- "复盘这周" → 周复盘报告

### 学习场景
- "制定学习计划" → 生成学习计划
- "总结这本书的重点" → 调用 book-squeeze
- "搜索X相关信息" → 全网搜索
