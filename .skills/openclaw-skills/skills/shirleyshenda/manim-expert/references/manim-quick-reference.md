# Manim社区版快速参考

## 常用类
- `Scene`: 基本场景类
- `MovingCameraScene`: 支持相机移动的场景
- `Mobject`: 所有图形对象的基类
- `Text`: 文本对象
- `MathTex`: LaTeX数学公式
- `VGroup`: 对象组合

## 常用动画
- `Create`: 创建对象
- `Write`: 书写文字/公式
- `FadeIn`: 淡入
- `FadeOut`: 淡出
- `Transform`: 变换
- `MoveToTarget`: 移动到目标位置

## 颜色系统
- 预定义颜色：RED, BLUE, GREEN, YELLOW等
- 十六进制颜色："#RRGGBB"
- RGB颜色：[R, G, B] (0-1范围)
