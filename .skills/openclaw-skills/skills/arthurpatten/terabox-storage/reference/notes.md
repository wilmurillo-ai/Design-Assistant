# TeraBox Storage Skill - Beta Notes

## Important Notice

This tool is in beta stage. Please note the following:

### Security Guidelines

1. **Data Backup** - Always backup important data; data loss risk exists during beta
2. **Behavior Review** - AI Agent behavior can be unpredictable; review command execution in real-time
3. **Environment Security** - NEVER authorize login in others', shared, or untrusted environments
4. **Token Protection** - Strictly protect config files and tokens; never expose in public

### Feature Limitations

1. **Sandbox Restriction** - All operations restricted to application sandbox directory
2. **API Limits** - Subject to TeraBox OpenAPI quota restrictions

### Known Issues

1. Large file uploads may be interrupted due to network fluctuations
2. Filenames with special characters may cause issues
3. Share downloads may fail for very large files

### Feedback

If you encounter issues, please provide:
- Command executed
- Error message
- Operating system and version
