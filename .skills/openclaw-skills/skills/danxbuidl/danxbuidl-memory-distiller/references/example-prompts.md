# Example Prompts

Use these examples when the user wants help invoking the skill or when you need
to pick the right output shape quickly.

## Extract stable preferences

```text
Use $memory-distiller to extract durable preferences from this conversation and
return only the items that look stable enough to reuse later.
```

## Convert outcomes into reusable rules

```text
Use $memory-distiller to turn these task outcomes into a short list of working
rules and anti-patterns I should reuse next time.
```

## Build a compact memory profile

```text
Use $memory-distiller to convert this raw history into a compact memory profile
with preferences, defaults, constraints, and avoid-rules.
```

## Produce a prompt-ready context block

```text
Use $memory-distiller to create a concise reusable context block from these
notes. Keep it short enough to drop into future prompts.
```

## Review an existing memory document

```text
Use $memory-distiller to review this memory document and split the result into
Keep, Rewrite, and Remove sections with short reasons.
```

## Conservative mode

```text
Use $memory-distiller in conservative mode. Exclude anything that is not clearly
durable or explicitly supported by the source material.
```

## Cleanup noisy memory

```text
Use $memory-distiller to clean this memory list by removing temporary details,
weak guesses, and one-off requests that should not become long-term memory.
```
