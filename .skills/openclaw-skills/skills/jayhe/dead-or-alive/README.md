# 安全签到助手

## 干什么？
每天飞书提醒打卡，7天不打卡自动打电话给紧急联系人。

## 怎么用？
1. 说"安全签到"激活
2. 回答3个问题：
   - 紧急电话
   - 飞书群ID
   - 提醒时间
3. 系统自动设置一切
4. 每天在飞书回复打卡

## 自动完成
- ✅ 自动保存配置
- ✅ 自动创建定时任务
- ✅ 自动发送提醒
- ✅ 自动检查打卡
- ✅ 自动打电话（7天未打卡）

## 文件
- `SKILL.md` - 流程约束
- `config/settings.json` - 配置
- `data/checkins.txt` - 打卡记录
- `data/alerts.txt` - 警报记录

## 核心
配置→提醒→打卡→监控→警报，全部自动。
