# 《豺狼的故事》结构学习示例

这个案例用于演示如何把一个“杀手接单刺杀”的单线故事，扩展成具备追查线、家庭线和工具人线的长篇剧集结构。

## 学习目标

- 看见单集冲突如何累积成整季张力
- 理解多条线如何共享危机、相互推进
- 学会在季终同时完成闭环与钩子

## 文件说明

- [season-map.md](season-map.md)：8 集季纲示意，展示四条线如何排布
- [episode-breakdown.md](episode-breakdown.md)：按单集 SOP 拆解前四集

## 使用方式

建议先读季纲，再读分集。阅读时重点关注：

- 哪条线在什么时候进入
- 哪个危机同时服务了两条以上故事线
- 主角的每次选择如何引发更大的代价

## 说明

本案例是基于公开讨论中的结构方法做教学性重组，不是对任何现成剧本、视频或台词的逐字还原。
