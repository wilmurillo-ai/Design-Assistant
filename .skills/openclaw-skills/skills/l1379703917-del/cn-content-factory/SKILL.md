# 🏭 中文自媒体内容工厂 (CN Content Factory)

一站式中文内容生产线。从选题到成稿，一条命令搞定多平台内容。

## 核心能力

### 🎯 智能选题
```
/选题 [行业/领域] [数量]
```
- 分析当前热点趋势
- 结合行业特征推荐选题
- 评估每个选题的流量潜力（高/中/低）
- 输出选题+角度+预估热度

### ✍️ 多平台内容生成

一个主题，自动适配4个平台的风格：

```
/内容 [主题] --platforms 小红书,公众号,抖音,知乎
```

| 平台 | 风格特征 | 字数 | 特殊要素 |
|---|---|---|---|
| 小红书 | 种草、口语化、emoji密集 | 300-800字 | 标签+封面标题建议 |
| 公众号 | 深度、有观点、结构化 | 1000-3000字 | 引导关注+阅读原文 |
| 抖音 | 口播脚本、节奏感强 | 150-300字 | 开头钩子+转场提示 |
| 知乎 | 专业、有理有据、干货 | 800-2000字 | 引用+数据+逻辑链 |

### 🔥 爆款标题生成器
```
/标题 [主题] [平台] [数量:5]
```
- 针对每个平台的算法偏好优化
- 小红书：数字+情绪+好奇心缺口
- 公众号：悬念+痛点+解决方案
- 抖音：冲突+反转+情绪价值
- 知乎：专业+经验+数据背书

### 🧹 去AI味润色
```
/润色 [文本]
```
- 消除AI生成的典型模式（"首先其次最后"、"值得注意的是"、"让我们"）
- 增加口语化表达、个人经历感
- 加入不完美表达（比如省略号、语气词）
- 保持信息密度的同时提升"人味"

### 📊 SEO优化（公众号/知乎）
```
/seo [文本] [目标关键词]
```
- 关键词密度优化（2-3%）
- 标题/小标题含关键词
- 首段包含核心关键词
- 内链建议

## 内容质量标准

### ✅ 合格标准
- 无事实错误
- 无明显AI痕迹
- 符合目标平台调性
- 信息密度达标
- 有具体案例/数据支撑

### ❌ 绝不输出
- 虚假数据/编造案例
- 违反平台规则的内容（如诱导关注、虚假宣传）
- 侵犯他人隐私的内容
- 政治敏感内容

## 去AI味核心策略

### 替换清单
| AI味表达 | 人味替换 |
|---|---|
| 首先...其次...最后 | 直接分点，不用连接词 |
| 值得注意的是 | 删掉，直接说 |
| 让我们来看看 | 删掉，直接开始 |
| 在当今社会/时代 | 具体年份或事件开头 |
| 综上所述 | 删掉或用"所以" |
| 不仅...而且 | 拆成两句短句 |
| 旨在/致力于 | 换成口语"就是想" |

### 增加人味
- 插入个人视角："我自己用了一个月的感受是..."
- 加入不确定性："可能不是每个人都适合，但..."
- 使用网络用语（适度）："真的绝了"、"救命"
- 段落长短参差不齐，不要太工整
