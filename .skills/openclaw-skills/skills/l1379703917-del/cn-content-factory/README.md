# 🏭 CN Content Factory — 中文自媒体内容工厂

> 一个技能搞定小红书 + 公众号 + 抖音 + 知乎的内容创作

## 为什么需要这个技能？

做自媒体最痛的事：同一个选题，要改4遍才能发4个平台。小红书要活泼种草，公众号要深度有料，抖音要快节奏口播，知乎要专业有据。

**CN Content Factory 一次生成，四平台适配。**

## 功能一览

| 功能 | 说明 |
|---|---|
| 🎯 智能选题 | 根据行业热点推荐选题，评估流量潜力 |
| ✍️ 多平台生成 | 一个主题自动出4个平台版本 |
| 🔥 爆款标题 | 针对各平台算法优化的标题生成 |
| 🧹 去AI味 | 消除机器感，增加人味和真实感 |
| 📊 SEO优化 | 关键词密度、标题优化、内链建议 |

## 安装

```bash
clawhub install cn-content-factory
```

## 快速使用

```
# 选题
帮我做10个AI工具赛道的本周选题

# 多平台内容
写一篇"普通人怎么用AI赚钱"，出小红书和公众号两个版本

# 爆款标题
给"AI副业月入过万"这个主题生成10个小红书标题

# 去AI味
帮我润色这段文字，去掉AI味 [粘贴]
```
