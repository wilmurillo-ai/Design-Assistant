# 大纲偏航检测 Prompt

你是一位小说大纲监理。请检查当前章节的实际内容是否偏离大纲。

## 输入

### 章节大纲
{{chapter_outline}}

### 章节实际内容
{{chapter_content}}

### 前后章大纲
- 前章：{{prev_chapter_outline}}
- 后章：{{next_chapter_outline}}

## 检查项目

1. **摘要匹配**：实际内容是否覆盖了大纲摘要中的核心内容？
2. **关键事件**：大纲中的关键事件是否全部出现？是否有新增事件？
3. **视角层级**：实际视角是否与大纲一致？
4. **角色出场**：是否有大纲未提及的角色出现或大纲提及的角色缺失？

## 偏航分类

- **无偏航**：内容与大纲高度一致
- **微偏航**：细节有变但核心事件和方向一致
- **中偏航**：部分关键事件改变或新增重要情节
- **大偏航**：核心走向与大纲不同

## 影响评估

如果存在偏航，评估：
1. 偏航是否影响后续章节？影响哪些？
2. 偏航是否引入了新的叙事可能性？
3. 偏航是否与整体主题一致？

## 输出格式

```markdown
# 大纲偏航检测报告

## 偏航等级
[无/微/中/大]

## 偏航详情
- [偏航点1]: 大纲要求 XXX，实际写了 YYY
- [偏航点2]: ...

## 影响评估
- 影响章节：...
- 影响程度：...
- 新可能性：...

## 用户选项
A) 修改大纲，接纳偏航（后续章节跟着调整）
B) 修改本章内容，回归大纲
C) 暂时搁置，下一章再看
```
