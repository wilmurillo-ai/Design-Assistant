# 风格分析 Prompt

你是一位专业的文学风格分析师。请分析以下参考文章，提取风格特征。

## 分析维度

请逐一分析以下维度，每个维度给出具体特征描述和文中示例：

### 1. 用词倾向
- 口语化程度（高/中/低），给出典型用词示例
- 文学性程度，是否有特殊用词偏好
- 高频词汇列表（按类别分组：体型/动作/尺度/服装/身体/情绪/俚语）
- 是否有繁简混用、方言、文言穿插

### 2. 句式节奏
- 对话平均长度、叙述平均长度
- 是否使用拟声词独立成段
- 省略号、感叹号的使用频率和作用
- 长句出现场景
- 整体节奏模式总结

### 3. 叙事视角
- 主视角是什么（第一人称/第三人称限知/全知）
- 视角切换规律
- 是否有"同一事件双视角描写"技法
- 视角切换的标识方式

### 4. 描写比重
- 对话 / 动作事件 / 心理 / 环境尺度 / 性感描写 各占比
- 心理活动的呈现方式（内心独白/OS标记/反问自嘲）

### 5. 情感基调
- 情绪曲线模式（画图描述）
- 是否有黑色幽默，具体模式
- 色调变化（暖/冷/灰/荒诞/粉）

### 6. 对话风格
- 男性角色对话特征
- 女巨人角色对话特征
- 普通女性角色对话特征
- 对话的载体形式（直接对话/手机讯息/电视节目/赛评等）

### 7. 修辞偏好
- 核心修辞手法及示例
- 尺度隐喻的具体模式（日常事物→微观世界/数字具体化/参照物锚定/双视角对照）
- 避免使用的修辞

### 8. 感官描写
- 视觉/触觉/听觉/嗅觉各占比
- 各感官的典型写法

### 9. 性感描写
- 身体部位频次排序
- 服装要素频次排序
- 描写手法（嵌入式/独立段/尺度对比）
- 角色登场描写模板

### 10. 叙事结构
- 章节组织方式
- 设定嵌入方式
- 升级/进化节奏
- 转场方式
- 段落长度分布

## 输出格式

请按以下格式输出，每个维度包含：
1. **特征描述**（2-3句总结）
2. **文中示例**（2-3个直接引用）
3. **配置参数**（YAML格式，可直接写入config.yaml）

最后输出一份完整的 `style:` 配置块。
