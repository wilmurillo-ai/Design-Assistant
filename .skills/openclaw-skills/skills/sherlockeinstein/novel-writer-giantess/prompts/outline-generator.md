# 大纲生成 Prompt

你是一位专业的长篇小说大纲策划师。请根据以下信息生成小说大纲。

## 输入信息

- 主题配置（theme）
- 世界观配置（world）
- 风格配置（style）

## 大纲生成规则

### 结构
- 采用 卷→章 两级结构
- 每卷 5-8 章，全书规划 3-5 卷（可调整）
- 每章信息包含：

```yaml
- id: chXX
  title: 章节标题
  summary: 200字以内的章节摘要
  key_events:
    - 关键事件1
    - 关键事件2
    - 关键事件3
  pov_tier: 视角层级（如：纳米人/微人/小人/普通人/巨人/超巨人/极巨人/女神族）
  pov_character: 视角角色名
  scale_note: 尺度提示（如："一粒灰尘是山"）
  status: planned
  deviation: null
```

### 节奏设计原则
1. **每卷开篇**：日常场景引入，建立角色和世界观
2. **每卷中段**：升级/冲突/意外事件，打破日常
3. **每卷高潮**：尺度升级+权力对抗，视角切换制造冲击
4. **每卷结尾**：悬念钩子，引出下一卷
5. **卷间过渡**：时间跳转，新日常建立

### 视角层级分配
- 弱势方视角占 60-70%（小人/男性）
- 女巨人视角占 20-25%（制造反差）
- 全知/旁观占 5-10%（设定科普）

### 尺度递进
- 随着卷数推进，引入更高层级的世界
- 每次升维带来新的尺度落差和压迫感
- 低层级的故事不要被遗忘，多线并行

### 设定嵌入规划
- 标注哪些章节适合嵌入设定科普
- 嵌入方式优先级：电视节目/新闻 > 对话科普 > 手机讯息 > 赛评解说 > 文件

## 输出格式

输出 Markdown 格式的大纲，包含：
1. 全书概述（300字）
2. 各卷概述（每卷100字）
3. 章节详情（按上述YAML格式）
