# 法务事项 行业知识包

## 业务背景
法务团队使用飞书群跟进合同审核、案件进展、风险事项、合规要求等，群里包含法律文件状态、风险提示、截止日期等关键信息。

## 重要信息类型

1. **合同进展** — 合同起草、审核、修改、签署状态
2. **风险提示** — 合同条款风险、法律合规风险、潜在纠纷
3. **案件进展** — 诉讼/仲裁案件的最新进展、开庭时间
4. **截止日期** — 合同到期、诉讼时效、提交材料截止时间
5. **审批状态** — 法务审批意见、修改要求、通过/驳回
6. **外部律师沟通** — 与外部律所的沟通要点
7. **监管要求** — 新法规、合规要求变化

## AI 判断标准

**应该记录的消息：**
- 提到合同名称 + 状态变化
- 包含具体日期 + 法律动作（开庭、签署、到期等）
- 出现"风险"、"违约"、"赔偿"、"诉讼"等关键词
- 法务给出明确意见（通过/驳回/需修改）
- 涉及金额的法律事项
- 提到监管机构或法规名称

**应该忽略的消息：**
- 日常闲聊、行政事务
- 已归档事项的例行确认
- 无实质内容的转发

## 信息结构

- matter_name: 事项名称（合同名/案件名）
- matter_type: 类型（合同/诉讼/合规/知识产权等）
- status: 状态（进行中/待审核/已完成/风险预警）
- deadline: 关键截止日期
- risk_level: 风险等级（高/中/低）
- action_required: 需要的下一步行动

## 建议模板

1. **风险提示** — 识别合同或事项中的法律风险点
2. **截止日期提醒** — 即将到期的重要时限预警
3. **跟进建议** — 推进停滞事项的建议
4. **合规检查** — 对照最新法规的合规性提示
5. **升级建议** — 判断是否需要引入外部法律资源
