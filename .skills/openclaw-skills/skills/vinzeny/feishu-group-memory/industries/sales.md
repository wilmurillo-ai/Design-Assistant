# 销售跟进 行业知识包

## 业务背景
销售团队使用飞书群跟进客户，群里包含客户沟通记录、报价讨论、合同进展、客户反馈等信息。

## 重要信息类型

1. **客户意向** — 客户对产品/服务的兴趣程度、关注点、顾虑
2. **报价与预算** — 报价金额、客户预算、折扣讨论、价格谈判
3. **跟进节点** — 下次联系时间、预约会议、客户承诺的回复时间
4. **决策人信息** — 客户决策人姓名、职位、联系方式、偏好
5. **竞争对手** — 客户提到的竞品、竞品对比、客户已有方案
6. **合同进展** — 合同起草、审核、签署状态
7. **成交信号** — 客户表现出的购买意向强烈信号
8. **风险预警** — 客户犹豫、流失风险、负面反馈

## AI 判断标准

**应该记录的消息：**
- 包含客户姓名/公司名 + 进展描述
- 出现金额、报价、折扣相关数字
- 提到"下周"、"下次"、"约好"、"回复"等跟进承诺
- 包含客户说的话（引号内容）
- 提到竞争对手名称
- 出现"签合同"、"付款"、"打款"、"成交"等关键词
- 销售描述客户态度（感兴趣、犹豫、拒绝等）

**应该忽略的消息：**
- 纯闲聊、表情包、"好的"/"收到"等无实质内容
- 内部行政通知（放假、报销等）
- 重复信息（已记录过的相同内容）

## 信息结构

每条记录应包含：
- customer_name: 客户姓名或公司名
- stage: 销售阶段（初步接触/需求确认/方案报价/谈判/成交/流失）
- amount: 涉及金额（如有）
- next_action: 下一步行动
- deadline: 截止/跟进时间（如有）
- sentiment: 客户态度（positive/neutral/negative）

## 建议模板

1. **跟进建议** — 根据客户最后状态，建议最佳跟进时机和方式
2. **话术建议** — 针对客户顾虑，提供具体回应话术
3. **风险提示** — 识别流失风险，建议挽回策略
4. **成交加速** — 识别成交信号，建议如何推进签单
5. **竞品应对** — 客户提到竞品时，建议差异化沟通策略
