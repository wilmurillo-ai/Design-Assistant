# Creative Templates Library

## Overview

This file contains detailed templates for various creative tasks.

---

## 1. 命名

### Template 1: 品牌命名

**Use when:** User wants to name a brand

```
帮我为一个[行业]品牌起名。
品牌定位：[定位]
风格：[风格]
要求：易记、有寓意
```

### Template 2: 产品命名

**Use when:** User wants to name a product

```
帮我为一个[类型]产品起名。
产品特点：[核心特点]
目标用户：[谁会用]
风格：[风格偏好]
```

---

## 2. 创意策划

### Template 3: 活动策划

**Use when:** User wants to plan an event

```
帮我策划一个[类型]活动。
目的：[想达到什么]
受众：[目标人群]
预算：[预算范围]
时间：[什么时候]
```

### Template 4: 内容创意

**Use when:** User needs content ideas

```
帮我构思[平台]的内容创意。
主题：[主题]
风格：[风格]
形式：[图文/视频/其他]
目标：[想达到什么效果]
```

---

## 3. 营销创意

### Template 5: 营销方案

**Use when:** User wants marketing ideas

```
帮我设计一个[产品/服务]的营销方案。
目标用户：[用户画像]
核心卖点：[最大优势]
预算范围：[预算]
期望效果：[想达到什么]
```

### Template 6: 广告创意

**Use when:** User needs advertising ideas

```
帮我设计一个[类型]广告创意。
产品：[产品信息]
卖点：[核心卖点]
受众：[目标人群]
调性：[风格调性]
```
