# Learning Templates Library

## Overview

This file contains detailed templates for various learning tasks.

---

## 1. 概念解释

### Template 1: 简单解释

**Use when:** User wants to understand a concept simply

```
用最简单的话解释[概念]，让[人群]也能听懂。
要求：用比喻，避免专业术语
```

### Template 2: 深度解析

**Use when:** User wants deep understanding

```
请深度解析[概念]。
要求：
1. 定义和核心原理
2. 应用场景
3. 与相关概念的区别
4. 实际案例
```

---

## 2. 学习规划

### Template 3: 学习路径

**Use when:** User wants to plan learning journey

```
我想学习[技能]，帮我制定学习路径。
当前水平：[零基础/有基础]
学习目标：[想达到什么水平]
学习时间：[每天/每周多少时间]
```

### Template 4: 学习计划

**Use when:** User needs detailed study plan

```
帮我制定一个[技能]学习计划。
时长：[几周/几月]
目标：[具体目标]
每周时间：[可用时间]
偏好：[学习方式偏好]
```

---

## 3. 知识整理

### Template 5: 知识梳理

**Use when:** User wants to organize knowledge

```
帮我梳理[领域]的知识体系。
范围：[具体范围]
深度：[入门/进阶/专业]
格式：思维导图结构
```

### Template 6: 读书笔记

**Use when:** User wants to summarize a book

```
帮我整理《[书名]》的读书笔记。
要求：
1. 核心观点提炼
2. 重点章节摘要
3. 实践启示
4. 推荐指数
```

---

## 4. 问题解答

### Template 7: 问题分析

**Use when:** User has a specific problem

```
我遇到了[问题描述]，请帮我分析原因。
背景：[相关背景]
已尝试：[做过什么]
期望：[想要的结果]
```

### Template 8: 方案建议

**Use when:** User needs solution suggestions

```
请针对[问题]给出解决方案。
现状：[当前情况]
限制条件：[有什么限制]
期望结果：[想要什么]
```
