# Lifestyle Templates Library

## Overview

This file contains detailed templates for various lifestyle tasks.

---

## 1. 日常规划

### Template 1: 旅行计划

**Use when:** User wants to plan a trip

```
帮我制定一个[目的地]的旅行计划。
天数：[几天]
预算：[预算范围]
偏好：[喜欢什么]
交通：[自驾/公共交通]
```

### Template 2: 健身计划

**Use when:** User wants to create a fitness plan

```
帮我制定一个健身计划。
目标：[减脂/增肌/塑形]
当前状况：[体重/体脂]
时间：[每周几次]
场地：[健身房/居家]
```

### Template 3: 饮食规划

**Use when:** User wants to plan meals

```
帮我制定一周饮食计划。
目标：[减脂/增肌/健康饮食]
饮食偏好：[口味/禁忌]
预算：[每天预算]
烹饪时间：[每天多久做饭]
```

## 2. 决策辅助

### Template 4: 选择建议

**Use when:** User needs help making a choice

```
帮我分析以下选项：
选项A：[描述]
选项B：[描述]
我的情况：[背景]
考虑因素：[最看重什么]
```

### Template 5: 职业规划

**Use when:** User wants career advice

```
帮我做职业规划建议。
当前职业：[职位]
工作年限：[几年]
困惑：[遇到什么问题]
目标：[想达到什么]
```

### Template 6: 购买决策

**Use when:** User needs help choosing a product

```
帮我选择[产品类型]。
预算：[预算范围]
需求：[主要用途]
偏好：[品牌/功能偏好]
使用场景：[在哪里用]
```

## 3. 沟通表达

### Template 7: 道歉信

**Use when:** User wants to write an apology

```
帮我写一封道歉信。
对方：[是谁]
事件：[发生了什么]
我的态度：[想表达什么]
期望：[希望怎么解决]
```

### Template 8: 感谢信

**Use when:** User wants to express gratitude

```
帮我写一封感谢信。
感谢对象：[是谁]
感谢原因：[为什么感谢]
想表达：[具体感受]
```

### Template 9: 邀请函

**Use when:** User wants to invite someone

```
帮我写一封邀请函。
活动：[什么活动]
时间地点：[何时何地]
受邀人：[邀请谁]
期望回复：[是否需要确认]
```
