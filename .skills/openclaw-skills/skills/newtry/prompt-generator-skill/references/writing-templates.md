# Writing Templates Library

## Overview

This file contains detailed templates for various writing tasks.

---

## 1. Public Account Articles (公众号文章)

### Template 1: Viral Article (爆款文章)

**Use when:** User wants to create viral content for WeChat public accounts

```
你是一个资深新媒体编辑，擅长写爆款文章。
帮我写一篇关于[主题]的公众号文章。
目标读者是[人群]，他们的痛点是[痛点]。
语言要接地气，有案例，有干货。
开头用故事引入，中间分3个要点，结尾升华。
字数2000字左右。
```

### Template 2: Tutorial Article (干货教程)

**Use when:** User wants to create educational content

```
你是一个[领域]专家，擅长把复杂概念讲简单。
帮我写一篇[主题]的教程文章。
要求：
1. 从零开始，循序渐进
2. 每个步骤有具体操作
3. 配合案例说明
4. 避免专业术语
字数：1500字
```

### Template 3: Hot Topic Analysis (热点解读)

**Use when:** User wants to analyze trending topics

```
最近发生了[热点事件]，帮我写一篇热点解读文章。
要求：
1. 简述事件经过（100字）
2. 分析背后原因（300字）
3. 对普通人的影响（300字）
4. 给出建议（200字）
风格：客观中立，有深度
```

## 2. Xiaohongshu (小红书)

### Template 4: Product Recommendation (种草笔记)

**Use when:** User wants to recommend products on Xiaohongshu

```
帮我写一篇小红书种草笔记。
产品：[产品名称]
亮点：[核心卖点]
使用场景：[什么时候用]
使用感受：[真实体验]
标题要吸引眼球，正文要有emoji，结尾引导互动。
```

### Template 5: Tips Sharing (干货分享)

**Use when:** User wants to share tips and knowledge

```
帮我写一篇小红书干货笔记。
主题：[分享主题]
内容：[具体干货]
格式：标题+正文+标签
要求：分点说明，每点一句话，配上emoji。
```

## 3. Zhihu Answers (知乎回答)

### Template 6: Professional Answer (专业回答)

**Use when:** User wants to provide professional answers on Zhihu

```
作为一个[领域]从业者，我来回答这个问题。
问题：[问题内容]
我的观点：[核心观点]
理由如下：
1. [理由一]
2. [理由二]
3. [理由三]
总结：[一句话总结]
```

### Template 7: Experience Sharing (经验分享)

**Use when:** User wants to share personal experiences

```
我在[领域]有[X]年经验，分享一些真实体会。
问题：[问题内容]
我的经历：[相关经历]
我的建议：[具体建议]
注意事项：[需要避免的坑]
```

## 4. Business Copy (商业文案)

### Template 8: Product Description (产品介绍)

**Use when:** User wants to describe a product

```
帮我写一段产品介绍文案。
产品名称：[名称]
目标用户：[谁会用]
核心卖点：[最大优势]
使用场景：[什么时候用]
风格：简洁有力，突出价值
字数：100字
```

### Template 9: Event Promotion (活动文案)

**Use when:** User wants to promote an event

```
帮我写一段活动宣传文案。
活动名称：[名称]
活动时间：[时间]
活动亮点：[吸引人的点]
参与方式：[怎么参加]
风格：热情有感染力
```

## 5. Workplace Writing (职场写作)

### Template 10: Work Report (工作汇报)

**Use when:** User needs to write work reports

```
帮我写一份周报。
本周工作：
1. [任务一]：完成情况
2. [任务二]：完成情况
遇到的问题：[问题描述]
下周计划：[计划内容]
需要支持：[需要的帮助]
```

### Template 11: Email Reply (邮件回复)

**Use when:** User needs to reply to emails

```
帮我回复一封邮件。
对方邮件内容：[原文]
我的立场：[想表达的态度]
回复要点：[要说什么]
语气：[正式/友好/委婉]
```

## 6. Story Articles (故事型文章)

### Template 12: Story Structure (故事结构)

**Use when:** User wants to create narrative content

```
帮我写一个关于[主题]的故事。
要求：
- 主角：[人物设定]
- 冲突：[遇到的问题]
- 转折：[如何解决]
- 结局：[结果如何]
- 寓意：[想传达的道理]
风格：温暖治愈，有代入感
```
