# Office Templates Library

## Overview

This file contains detailed templates for various office tasks.

---

## 1. PPT制作

### Template 1: PPT大纲

**Use when:** User wants to create a presentation outline

```
帮我设计一个关于[主题]的PPT大纲。
受众：[听众是谁]
目的：[想达到什么效果]
时长：[演讲时长]
要求：结构清晰，逻辑连贯
```

### Template 2: PPT内容

**Use when:** User wants detailed PPT content

```
帮我写一份PPT的内容。
主题：[主题]
页数：[几页]
每页要点：[每页要讲什么]
风格：[正式/活泼/简约]
```
