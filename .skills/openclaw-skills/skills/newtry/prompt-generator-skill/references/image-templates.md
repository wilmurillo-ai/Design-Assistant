# Image Generation Templates Library

## Overview

This file contains detailed templates for various image generation tasks using AI tools like Midjourney, DALL-E, and Stable Diffusion.

---

## 1. Portrait (人物肖像)

### Template 1: Realistic Portrait (写实人像)

**Use when:** User wants to create realistic human portraits

```
一位[年龄]岁的[性别]，[外貌特征]，[表情]，[服装]，[背景]，写实风格，高清摄影，自然光线，8k --ar 3:4
```

**Example:**
```
一位30岁的女性，长发，自信微笑，穿着深蓝色西装外套，简约办公室背景，专业形象，高清摄影，自然光线，8k --ar 3:4
```

### Template 2: Anime Character (动漫人物)

**Use when:** User wants to create anime-style characters

```
一个[特征]的动漫角色，[发型]，[服装]，[表情]，[动作]，[场景]，日系动漫风格，高清 --ar 3:4 --niji 6
```

**Example:**
```
一个蓝发的动漫少女，双马尾，穿着校服，开心微笑，站在樱花树下，日系动漫风格，高清 --ar 3:4 --niji 6
```

### Template 3: Professional Headshot (职业头像)

**Use when:** User needs professional profile pictures

```
一位[职业]的职业照，[性别]，[年龄]岁，[着装]，[背景]，专业摄影，柔和光线，高清 --ar 1:1
```

**Example:**
```

```

## 5. Style Reference (风格参考)

### Common Art Styles

| Style | Description | Example Tags |
|-------|-------------|--------------|
| 写实 | Realistic photography | `写实风格，高清摄影，8k` |
| 动漫 | Anime/Manga style | `日系动漫风格，--niji 6` |
| 油画 | Oil painting | `油画风格，[艺术家名字]风格` |
| 水彩 | Watercolor | `水彩风格，淡雅` |
| 赛博朋克 | Cyberpunk | `赛博朋克风格，霓虹灯，高科技` |
| 蒸汽朋克 | Steampunk | `蒸汽朋克风格，机械，齿轮` |
| 极简主义 | Minimalist | `极简风格，简约设计` |
| 复古 | Retro/Vintage | `复古风格，怀旧色调` |

### Common Parameters

| Parameter | Description | Example |
|-----------|-------------|---------|
| `--ar` | Aspect ratio | `--ar 16:9` (landscape), `--ar 3:4` (portrait), `--ar 1:1` (square) |
| `--niji` | Anime mode (Midjourney) | `--niji 6` |
| `--v` | Version (Midjourney) | `--v 6` |
| `--q` | Quality | `--q 2` (high quality) |
| `--s` | Stylize | `--s 750` (more artistic) |

### Lighting Types

| Lighting | Description | Chinese |
|----------|-------------|---------|
| Natural light | 自然光 | `自然光线` |
| Golden hour | 黄金时刻 | `金色阳光，黄昏` |
| Soft light | 柔光 | `柔和光线` |
| Dramatic lighting | 戏剧性光线 | `戏剧性光线，强对比` |
| Neon light | 霓虹灯 | `霓虹灯光` |
| Backlighting | 逆光 | `逆光，轮廓光` |
