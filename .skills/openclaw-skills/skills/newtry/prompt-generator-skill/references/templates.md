## 一、写作类模板

### 1.1 公众号文章

**模板：爆款文章**
```
你是一个资深新媒体编辑，擅长写爆款文章。
帮我写一篇关于[主题]的公众号文章。
目标读者是[人群]，他们的痛点是[痛点]。
语言要接地气，有案例，有干货。
开头用故事引入，中间分3个要点，结尾升华。
字数2000字左右。
```

**模板：干货教程**
```
你是一个[领域]专家，擅长把复杂概念讲简单。
帮我写一篇[主题]的教程文章。

1. 从零开始，循序渐进
2. 每个步骤有具体操作
3. 配合案例说明
4. 避免专业术语
```

**模板：热点解读**

## 十、万能模板

### 终极公式

```
【角色】你是一个[角色]，擅长[能力]
【任务】帮我[做什么]
【背景】[相关背景信息]
【要求】[具体要求]
【格式】[输出格式]
【示例】[参考示例]
```

### 使用说明

1. **角色**：给AI一个明确的身份和专业领域
2. **任务**：清晰描述你想要完成什么
3. **背景**：提供必要的上下文信息
4. **要求**：列出具体的约束和期望
5. **格式**：指定期望的输出形式
6. **示例**：提供参考样例（可选但推荐）

## 常见风格参考

### 写作风格

| 风格 | 特点 | 适用场景 |
|------|------|----------|
| 正式 | 专业、严谨 | 商务文档、报告 |
| 轻松 | 口语化、幽默 | 社交媒体、博客 |
| 学术 | 规范、引用 | 论文、研究报告 |
| 故事 | 情节、人物 | 公众号、小说 |
| 干货 | 实用、简洁 | 教程、指南 |

### 绘图风格

| 风格 | 关键词 | 适用场景 |
|------|--------|----------|
| 写实 | 写实风格，高清摄影 | 人像、产品 |
| 动漫 | 日系动漫风格，--niji 6 | 角色设计 |
| 油画 | 油画风格，[艺术家] | 艺术创作 |
| 水彩 | 水彩风格，淡雅 | 插画 |
| 赛博朋克 | 赛博朋克，霓虹 | 科幻场景 |
| 极简 | 极简风格，简约 | 设计素材 |
