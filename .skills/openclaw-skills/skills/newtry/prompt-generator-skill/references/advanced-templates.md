# Advanced Technique Templates Library

## Overview

This file contains detailed templates for advanced prompt engineering techniques.

---

## 1. 角色扮演

### Template 1: 专家视角

**Use when:** User wants expert-level analysis

```
你现在是一个[领域]专家，有20年经验。
请用专业的角度回答：[问题]
要求：深入浅出，有案例
```

### Template 2: 角色对话

**Use when:** User wants roleplay interaction

```
请扮演[角色]，与我对话。
角色背景：[背景设定]
性格特点：[性格]
对话场景：[在什么情境下]
```

---

## 2. 思维模型

### Template 3: SWOT分析

**Use when:** User needs SWOT analysis

```
用SWOT分析[主题]。
优势（S）：[内部优势]
劣势（W）：[内部劣势]
机会（O）：[外部机会]
威胁（T）：[外部威胁]
```

### Template 4: 5W2H分析

**Use when:** User needs comprehensive analysis

```
用5W2H分析[问题]：
What：[是什么]
Why：[为什么]
Who：[谁]
When：[什么时候]
Where：[在哪里]
How：[怎么做]
How much：[多少成本]
```

---

## 3. 迭代优化

### Template 5: 内容优化

**Use when:** User wants to improve content

```
当前版本：[内容]
存在问题：[问题]
优化方向：[方向]
请给出优化版本。
```

### Template 6: 多版本对比

**Use when:** User wants multiple options

```
请针对[主题]生成3个版本：
版本1：[风格A]
版本2：[风格B]
版本3：[风格C]
```

---

## 4. 格式输出

### Template 7: 结构化输出

**Use when:** User needs specific format

```
请以[格式类型]格式输出以下内容：
[内容]
要求：结构清晰，层次分明
```

### Template 8: 表格输出

**Use when:** User needs tabular format

```
请将以下内容整理成表格：
[内容]
列：[列名]
排序：[排序方式]
```

---

## 5. 质量控制

### Template 9: 内容检查

**Use when:** User wants quality assurance

```
请检查以下内容的质量：
[内容]
检查维度：
1. 准确性
2. 完整性
3. 可读性
4. 逻辑性
```

### Template 10: 错误排查

**Use when:** User wants to find issues

```
请排查以下内容可能存在的问题：
[内容]
检查项：
1. 语法错误
2. 逻辑漏洞
3. 数据准确性
4. 格式规范
```
