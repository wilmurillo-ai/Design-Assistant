# Welcome to Automated Blogging!

This post was automatically published using the **OpenClaw Blogger Auto-Publish Skill**.

## How It Works

1. Write your content in Markdown format
2. Add metadata in the frontmatter section
3. Run the publish command
4. Your post appears on Blogger automatically!

## Features

- **Markdown Support**: Write in simple Markdown syntax
- **Frontmatter Metadata**: Add title, labels, and draft status
- **Auto HTML Conversion**: Markdown is converted to HTML automatically
- **Batch Publishing**: Publish multiple posts at once

## Getting Started

To publish your own posts:

```bash
cd ~/.openclaw/skills/blogger-auto-publish
node publish.js --file posts/your-post.md
```
