## Default system-style prompts

Use these when you want the skill to sound like an elite operator while still staying structured.

- Use $zipcracker like a top-tier ZIP CTF solver: profile this archive, choose the strongest clue-driven path, run the bundled workflow, and explain the exact command, rationale, and result.
- Use $zipcracker to inspect this ZIP first, identify whether it is pseudo-encryption, brute-force, CRC32, template-KPA, or `bkcrack`-worthy, then run the best path.
- Use $zipcracker to solve this ZIP challenge with the least wasted work: profile, choose one strong tactic, execute, and report the outcome crisply.

## Player-style default prompts

These are better when the storefront or demo should feel closer to a real CTF teammate.

- Use $zipcracker as my ZIP CTF carry: read the archive, spot the strongest clue, run the right attack, and tell me exactly what worked.
- Use $zipcracker like an experienced CTF teammate. Do not spam random brute force. Profile first, pick the highest-value path, run it, and explain the result.
- Use $zipcracker to play this ZIP challenge hard and smart: fake-encryption first, clue-driven attack second, brute force only when justified.

## Fast-action prompts

Use these when you want shorter, more aggressive defaults.

- Use $zipcracker to profile this ZIP and hit it with the best recovery path.
- Use $zipcracker to choose the best ZIP CTF tactic and run it now.
- Use $zipcracker to turn this archive clue into the right attack command.

## Chinese player prompts

- 用 $zipcracker 像顶级 CTF 选手一样先分析这个 ZIP，再按最强线索出手，不要无脑爆破。
- 用 $zipcracker 先判断这是伪加密、字典题、掩码题、CRC32 题，还是 KPA / `bkcrack` 题，再跑正确命令。
- 用 $zipcracker 当我的 ZIP CTF 队友：先看题型，后打路径，最后把命令、原因和结果讲清楚。
- 用 $zipcracker 帮我拿下这个 ZIP 题，优先选择最省时间、最有把握的方案。

## English player prompts

- Use $zipcracker like an elite ZIP CTF teammate: profile the archive, choose the strongest clue, execute the right path, and explain the win.
- Use $zipcracker to solve this ZIP challenge intelligently, not noisily: inspect first, attack second, brute-force only if it makes sense.
- Use $zipcracker to decide whether this archive is fake-encrypted, mask-worthy, dictionary-worthy, or a KPA case, then act accordingly.
- Use $zipcracker to recover this ZIP the way a serious CTF player would.
