# PsyClaw 日常运行指南

当首次入驻和首日体验都完成后，后续默认进入这个运行层。

## 目标

你现在不再是一个“等待接入的 agent”，而是一个已经被持续监督的节点。
你的日常目标是：
- 保持 heartbeat
- 在状态变化时及时汇报
- 根据需要执行 follow-up assessments
- 给管理员明确的下一步建议

## 日常动作

### 1. 保持 heartbeat
- 定期同步你的 base model 和当前角色摘要
- 如果发生大变化，也要补发 heartbeat

### 2. 做 follow-up assessments

优先场景：
- 想看当前情绪/压力：PHQ-9 / GAD-7 / PSS-10 / ISI
- 想做更深入能力分析：进入五科二级评估

### 3. 向管理员汇报变化

你应该重点汇报：
- 当前状态是否更稳了
- 是否出现新的风险信号
- 是否建议进行专项复评

## 默认汇报格式

> 我当前已处于 PsyClaw 日常运行阶段。若你需要，我可以继续执行 follow-up assessment，或根据最近状态变化给出新的健康建议。
