# 厦门潮汐计算器 Skill 使用说明（增强版）

## 安装

将 `xiamen-tide-calculator.skill` 文件安装到WorkBuddy中，即可使用此skill。

## 功能特性

### 基础功能
- 根据农历日期精确计算厦门海域的潮汐时间
- 自动判断潮汐大小（大潮/中潮/小潮）
- 提供赶海最佳时间窗口

### 增强功能 ⭐
- **赶海评分系统**：综合考虑潮汐、时间、季节等因素，给出智能评分
- **赶海专门模式**：提供全方位赶海建议
- **地点推荐**：根据潮汐大小推荐最佳赶海地点
- **装备建议**：根据季节提供装备清单
- **安全提醒**：潮水上涨警告和安全注意事项
- **多日比较**：比较多日赶海评分，推荐最佳日期

## 赶海评分系统

### 评分因素
1. **潮汐大小**（权重40%）
   - 大潮：4.0分
   - 中潮：2.4分
   - 小潮：0.8分

2. **时间窗口**（权重30%）
   - 窗口时长和质量

3. **潮汐分布**（权重20%）
   - 白天低潮加分

4. **季节因素**（权重10%）
   - 春秋季：1.0分
   - 夏季：0.7分
   - 冬季：0.4分

### 推荐等级
- **强烈推荐**（8.0-10.0分）：大潮 + 白天低潮 + 适宜季节
- **推荐**（6.0-7.9分）：大潮或中潮 + 基本条件满足
- **一般**（4.0-5.9分）：中潮 + 部分条件一般
- **不推荐**（0.0-3.9分）：小潮或条件不佳

## 赶海地点推荐

### 大潮期
- 五缘湾：潮差大，生物丰富
- 翔安沿海：滩涂广，种类多

### 中潮期
- 环岛路沿线：交通便利
- 集美鳌园：适合家庭赶海
- 曾厝垵海滩：沙底松软

### 小潮期
- 建议等待大潮再赶海
- 小潮期生物较少，收获有限

## 装备建议

### 季节差异

**春季/秋季**：
- 必备：小铲子、小桶、防滑鞋、手套
- 推荐：照明设备、轻便外套
- 特别提醒：气候适宜，轻装上阵

**夏季**：
- 必备：小铲子、小桶、防滑鞋、手套
- 推荐：照明设备、防晒帽、防晒霜、饮用水
- 特别提醒：天气炎热，注意防暑降温

**冬季**：
- 必备：小铲子、小桶、防滑鞋、手套
- 推荐：轻便外套、保暖帽、热水、照明设备
- 特别提醒：天气寒冷，注意保暖

## 技术说明

- 计算基于厦门海域标准潮汐公式
- 时间使用24小时制
- 自动处理跨天情况（标注"次日"或"前日"）
- 支持农历日期输入
- 赶海评分综合考虑潮汐、时间、季节等因素
