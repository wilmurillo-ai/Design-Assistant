# Project Scaffolder

A ClawHub skill for generating project boilerplate from templates.

## Features

- Next.js, Express, Python CLI, Static HTML templates
- Auto-initialize Git with proper .gitignore
- Extensible to any framework
- No dependencies beyond the target framework's CLI

## Example Prompts

- "Create a new Next.js project called my-app"
- "Scaffold an Express API with TypeScript"
- "Set up a Python CLI tool"
