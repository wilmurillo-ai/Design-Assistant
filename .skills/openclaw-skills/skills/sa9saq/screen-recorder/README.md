# Screen Recorder

A ClawHub skill for screen recording using ffmpeg.

## Features

- Full screen and region capture
- Optional audio recording
- X11 and Wayland support
- Convert to GIF, compress recordings
- No dependencies beyond ffmpeg

## Example Prompts

- "Record my screen for a demo"
- "Record a 1280x720 region of my screen"
- "Convert my recording to a GIF"
