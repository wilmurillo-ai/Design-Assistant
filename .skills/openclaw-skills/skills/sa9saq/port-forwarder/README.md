# Port Forwarder

A ClawHub skill for managing SSH tunnels and port forwarding.

## Features

- Local, remote, and dynamic (SOCKS) port forwarding
- List and kill active tunnels
- Connection testing
- No dependencies beyond OpenSSH

## Example Prompts

- "Forward local port 8080 to remote port 80 on myserver"
- "List all active SSH tunnels"
- "Set up a SOCKS proxy through my jump host"
