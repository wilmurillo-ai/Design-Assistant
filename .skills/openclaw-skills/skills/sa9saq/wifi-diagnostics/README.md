# Wi-Fi Diagnostics

A ClawHub skill for diagnosing Wi-Fi connectivity issues.

## Features

- Signal strength and link quality analysis
- Channel scanning and congestion detection
- Speed testing via curl (no extra tools needed)
- DNS and latency diagnostics

## Example Prompts

- "Check my Wi-Fi signal strength"
- "Which Wi-Fi channel has the least congestion?"
- "Run a speed test"
