# Example Skills for Reference

This directory contains real-world examples of skills following the Nima Skill Creator framework.

## Examples

### 1. PDF Processing
- **Purpose**: Extract text and tables from PDFs
- **Resources**: `scripts/extract_pdf.py`, `references/pdf-libraries.md`
- **Pattern**: Script-based processing

### 2. Image Editor
- **Purpose**: Resize, compress, and convert images
- **Resources**: `scripts/edit_image.py`, `assets/templates/`
- **Pattern**: Script-based workflow

### 3. Notion Integration
- **Purpose**: Connect Claude to Notion workspace
- **Resources**: `scripts/notion_api.py`, `references/notion-schema.md`
- **Pattern**: API integration with schema

### 4. Document Generator
- **Purpose**: Create professional documents from templates
- **Resources**: `assets/templates/`, `references/style-guide.md`
- **Pattern**: Template-based generation

## Learning from Examples

Each example demonstrates different patterns:
- Script-based processing
- API integration
- Template-based generation
- Multi-step workflows

 review these examples when creating your skill.
