### 4. 问题解决

#### 问题 1: URL 格式错误
**现象**: 获取的数据是其他平台的数据

**根因**: URL 格式不正确，点点数据使用 line-{channel_id} 格式

**解决**: 
- 获取正确的 URL 格式
- 更新所有平台的 URL 映射表

#### 问题 2: 下载文件是旧数据
**现象**: 下载的文件是上一次导出的数据

**根因**: 导出后没有等待文件生成

**解决**:
- 点击导出后等待 5 秒
- 下载后验证文件内容
- 不匹配则等待 2 秒重新下载

#### 问题 3: 页面标题大小写不匹配
**现象**: OPPO 平台页面标题显示"oppo"（小写）

**解决**: 验证逻辑改为忽略大小写

#### 问题 4: 下架榜单 URL 生成
**现象**: 下架榜单 URL 格式错误

**根因**: URL 参数索引错误

**解决**:
- 上架榜：line-{id}-0-0-75-0-3-0
- 下架榜：line-{id}-1-0-75-0-3-0
- 动态生成 URL

#### 问题 5: 下架榜单消息文字
**现象**: 下架榜单消息显示"上架"文字

**解决**:
- 添加 is_offline 参数
- 动态调整消息文字
- 下架榜单只显示总计统计

#### 问题 6: 下架榜单统计不完整
**现象**: 下架榜单有"重新上架"但没统计

**解决**:
- 修改 parse_excel_data 函数
- 下架榜单统计"下架"和"重新上架"
- 消息显示下架、重新上架、总计
