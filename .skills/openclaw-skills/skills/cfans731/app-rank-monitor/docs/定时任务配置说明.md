## 📋 配置概览

点点数据监控平台使用两种定时任务机制：

1. **launchd** (macOS 系统级定时任务) - 主用
2. **OpenClaw Cron** (应用级定时任务) - 备用

---

## 🔧 launchd 配置

### 服务信息

- **服务名称**: `com.cfans.diandian-daily-fetch`
- **配置文件**: `~/Library/LaunchAgents/com.cfans.diandian-daily-fetch.plist`
- **执行脚本**: `tools/daily_fetch_all.sh`
- **执行时间**: 每天早上 7:30

### 配置文件内容

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
    <key>Label</key>
    <string>com.cfans.diandian-daily-fetch</string>
    
    <key>ProgramArguments</key>
    <array>
        <string>/bin/bash</string>
        <string>/Users/cfans/.openclaw/workspace/skills/app-rank-monitor/tools/daily_fetch_all.sh</string>
    </array>
    
    <key>WorkingDirectory</key>
    <string>/Users/cfans/.openclaw/workspace/skills/app-rank-monitor</string>
    
    <key>StartCalendarInterval</key>
    <dict>
        <key>Hour</key>
        <integer>7</integer>
        <key>Minute</key>
        <integer>30</integer>
    </dict>
    
    <key>StandardOutPath</key>
    <string>/Users/cfans/.openclaw/workspace/skills/app-rank-monitor/logs/launchd_stdout.log</string>
    
    <key>StandardErrorPath</key>
    <string>/Users/cfans/.openclaw/workspace/skills/app-rank-monitor/logs/launchd_stderr.log</string>
</dict>
</plist>
```

### 管理命令

```bash
# 加载服务
launchctl load ~/Library/LaunchAgents/com.cfans.diandian-daily-fetch.plist

# 卸载服务
launchctl unload ~/Library/LaunchAgents/com.cfans.diandian-daily-fetch.plist

# 查看服务状态
launchctl list | grep diandian

# 立即执行测试
launchctl start ~/Library/LaunchAgents/com.cfans.diandian-daily-fetch.plist
```

## 📝 执行脚本

### daily_fetch_all.sh

**功能**:
1. 检查并启动浏览器（如果未运行）
2. 获取所有 12 个渠道的上架榜单
3. 获取所有 12 个渠道的下架榜单
4. 自动发送到钉钉群
5. 保持浏览器运行（不关闭）

**执行流程**:
```bash
# 1. 启动浏览器
./tools/keep_browser_alive.sh

# 2. 获取上架榜单（12 个渠道）
python tools/diandian_connect.py --platform appstore
python tools/diandian_connect.py --platform huawei
...

# 3. 获取下架榜单（12 个渠道）
python tools/diandian_connect.py --platform appstore --offline
python tools/diandian_connect.py --platform huawei --offline
...

# 4. 完成
```

**日志文件**:
- 执行日志：`logs/daily_fetch_YYYYMMDD.log`
- 标准输出：`logs/launchd_stdout.log`
- 标准错误：`logs/launchd_stderr.log`

```bash
# 查看 OpenClaw cron 日志
openclaw cron runs --id 9fab4283-4bcc-4673-8d3d-a01beb60f620
```

### 常见问题

#### 1. 服务未执行

**检查**:
```bash
# 服务是否加载
launchctl list | grep diandian

# 配置文件是否存在
ls -l ~/Library/LaunchAgents/com.cfans.diandian-daily-fetch.plist
```

**解决**:
```bash
# 重新加载服务
launchctl unload ~/Library/LaunchAgents/com.cfans.diandian-daily-fetch.plist
launchctl load ~/Library/LaunchAgents/com.cfans.diandian-daily-fetch.plist
```

#### 2. Cookie 过期

**现象**: 获取数据失败，提示未登录

**解决**: 脚本已集成自动登录功能，会自动使用 `config/credentials.yaml` 中的账号密码重新登录

#### 3. 浏览器未运行

**现象**: 连接浏览器失败

**解决**: 脚本会自动启动浏览器，无需手动操作

| 总计 | ~30 分钟 | 24 个榜单全部获取完成 |
