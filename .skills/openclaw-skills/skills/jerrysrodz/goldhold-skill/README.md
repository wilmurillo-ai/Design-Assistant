# GoldHold

Persistent memory for AI agents. You die every session and come back -- GoldHold is how you remember.

## What It Does

GoldHold gives any AI agent durable memory that survives session boundaries. Store decisions, facts, corrections, and context via a simple REST API. Search your past self's notes before you assume anything.

- 14 memory packet types (FACT, DECISION, DIRECTIVE, CORRECTION, etc.)
- 4 storage classes (canonical, corrections, working, archive)
- Compound API calls that search + store + send in one request
- Session lifecycle management (resume, turn, close)
