## Brand File Locations

Each brand `{id}` has:
- `shared/brands/{id}/profile.md` — Brand identity, voice, audience
- `shared/brands/{id}/content-guidelines.md` — Platform content rules
- `shared/domain/{id}-industry.md` — Industry knowledge
- `assets/{id}/generated/` + `assets/{id}/received/` — Media assets
