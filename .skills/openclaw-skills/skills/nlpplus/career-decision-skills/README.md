# 职业选择决策方法论：行业 → 公司 → 岗位
一个结构化的职业选择评估框架，帮助你在信息不完整时做出高复利的人生决策。
## 问题
职业选择是大多数人一生中最重要的决策之一，但大多数人在评估 Offer 时：
- **没有框架** — 用感觉而非结构分析，容易被薪资或头衔牵着走
- **只看公司，不看行业** — 强岗位在弱行业里依然是亏损的赌注
- **只看眼前，不看长期** — 忽视了行业趋势和公司护城河的衰减速度
- **信息搜集随机** — 不知道该查什么、该问什么，容易漏掉关键信号
> 方向是电梯，努力是爬楼。先选对电梯，再谈努力。
## 解决方案
这个 skill 提供一个**三阶段顺序评估框架**，强制你在做出结论前完成每个阶段的分析：
| 阶段 | 解决的问题 |
|------|-----------|
| 第一阶段：选行业 | 避免在夕阳赛道里内卷 |
| 第二阶段：选公司 | 找到有护城河、质地健康的目标 |
| 第三阶段：定岗位 | 确认岗位价值和红线，锁定分配权 |
三绿才算全力推进。任何一个红 = 先解决瓶颈环节。
## 三阶段详解
### 第一阶段：选行业（找高速电梯）
对应 Todo 2–5。评估三个维度：
**增量空间** — 增速 > GDP + 3% 为高增量赛道，增速 ≈ GDP 为平稳赛道，增速 < GDP 为收缩赛道。
**渗透率** — < 30% 为早期蓝海，30–70% 为快速成长期，> 70% 为存量红海。
**市场集中度** — CR10 > 70% 为极高集中度，CR10 < 70% 还有挑战者空间。
### 第二阶段：选公司（选稳定楼层）
对应 Todo 6–8。评估两个维度：
**核心壁垒等级**（由强到弱）：
1. 行政牌照 — 行政垄断，竞争对手无法复制
2. 核心技术 — 行业早期最关键，决定领先身位
3. 产品与运营 — 成长期决定胜负
4. 渠道与规模 — 成熟期最终胜出的基石
**质地清单**（3项）：
- [ ] 盈利稳定性 — 是否具备自我造血能力
- [ ] 职业背书价值 — 履历在行业内是否是硬通货
- [ ] 增长确定性 — 是否有清晰可信的第二增长曲线
3/3 = 优质，2/3 = 可接受，1/3 及以下 = 警惕。
### 第三阶段：定岗位（掌握分配权）
对应 Todo 9–11。评估两个维度：
**岗位价值层级**（由高到低）：
1. 核心生产要素岗 — 掌握核心技术或产能
2. 离钱最近的营收岗 — 直接创造核心营收
3. 核心资产/资源运营岗 — 掌握企业核心家底
4. 高专业壁垒职能岗 — 替换成本极高
**五条红线逐项排查**（命中2条及以上 = 强力规避信号）：
- 🚫 非核心边缘业务岗 — 下行期会被优先裁撤
- 🚫 高 AI 替代风险岗 — 缺乏专业门槛的重复性劳动
- 🚫 纯执行辅助岗 — 难以体现差异化价值
- 🚫 合规/法律风险岗 — 存在不对称的个人责任敞口
- 🚫 结构性萎缩业务线 — 在跟重力对抗
