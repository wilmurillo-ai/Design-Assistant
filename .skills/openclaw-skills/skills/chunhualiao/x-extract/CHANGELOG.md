# Changelog

All notable changes to the x-extract skill will be documented in this file.

## [1.0.0] - 2026-02-16

### Added
- Initial release of x-extract skill
- Browser-based tweet content extraction without API credentials
- Support for x.com and twitter.com URLs
- Extract tweet text, author, timestamp, media, and engagement metrics
- Structured markdown output format
- Optional media download functionality
- Thread detection and extraction support
- Comprehensive error handling for failed page loads
- Reference documentation for X.com selectors

### Features
- No Twitter API credentials required
- Works with public tweets
- Extracts metadata (likes, retweets, replies)
- Media URL extraction and download
- Thread context detection
