## 三、主要空间设计说明

### 3.1 客厅

**设计要点：**
1. 沙发区面向采光面，保证充足自然光
2. 电视背景墙采用简约设计，局部使用木饰面或岩板点缀
3. 地面通铺灰色系地砖/地板，延伸空间感

**材料选择：**
- 地面：品牌地砖 800x800 或实木复合地板
- 墙面：乳胶漆 + 木饰面/岩板
- 天花：石膏板吊顶 + 射灯

### 3.2 餐厅

**设计要点：**
1. 餐桌面靠墙，节省空间
2. 餐边柜提供收纳和展示功能
3. 吊灯位置对准餐桌中心，营造用餐氛围

### 3.3 厨房

**设计要点：**
1. U型布局，最大化操作台面
2. 采用集成灶或分体灶+油烟机
3. 吊柜+地柜提供充足收纳

**材料选择：**
- 台面：石英石，耐用易清洁
- 柜体：多层实木或金刚板
- 墙面：爵士白瓷砖或钢化玻璃

### 3.4 主卧

**设计要点：**
1. 床头背景墙作为视觉焦点
2. 衣柜采用嵌入式设计，节省空间
3. 灯光采用暖色调，营造温馨睡眠氛围

- **60% 主色**：墙面、天花大面积色（浅灰/米白）
- **30% 辅色**：家具、窗帘、地毯（中灰/木色）
- **10% 点缀色**：装饰画、抱枕、灯具（品牌色/金属色）
