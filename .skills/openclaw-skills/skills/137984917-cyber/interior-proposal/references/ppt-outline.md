# PPT大纲模板

## 标准10页框架

### Page 1: 封面
```
项目名称
风格定位 | 面积㎡
公司名称 | 设计师
日期
```

### Page 2: 目录
```
01 项目概述
02 设计理念
03 空间设计
04 效果展示
05 材质说明
06 预算方案
07 设计流程
08 感谢聆听
```

### Page 3: 项目概述
```
【项目信息】
- 项目名称：
- 建筑面积：
- 设计风格：
- 项目类型：住宅/商业/酒店

【客户需求】
- 功能需求：
- 风格偏好：
- 预算区间：

【设计目标】
- 目标1
- 目标2
- 目标3
```

### Page 4: 设计理念（3个关键词卡片）
```
LESS IS MORE | 简约主义
QUALITY LIFE | 品质生活  
TIMELESS | 历久弥新

理念说明文字...
```

### Page 5-7: 空间设计（按空间分页）
```
【客厅设计方案】
- 布局思路
- 色彩搭配
- 材质选择
- 灯光设计

【餐厅设计方案】
...

【卧室设计方案】
...
```

### Page 8: 效果展示
```
多图网格布局（2x2 或 1+3）
最多展示4张效果图
```

### Page 9: 材质说明
```
【地面材料】
【墙面材料】
【柜体材料】
【色彩比例】60-30-10
```

### Page 10: 预算方案
```
【三档预算参考】

基础档 | 1500-2000元/㎡
品质档 | 2500-3500元/㎡
豪华档 | 4500元/㎡以上
```

### Page 11: 设计流程
```
Step 1 方案设计：7-10天
Step 2 施工图深化：10-15天
Step 3 施工阶段：60-90天
Step 4 软装摆场：7-10天
```

### Page 12: 封底
```
感谢聆听
期待为您打造理想居所

公司名称
联系方式
```
