# SketchUp-PPT生成 - 温州隐室空间设计自研技能

## 📋 技能说明

一键将SketchUp模型生成客户汇报PPT，按照工作室标准15页框架自动生成，预留效果图位置，只需你在SketchUp导出截图插入即可。

**适用场景：**
- 客户方案汇报
- 快速出PPT框架
- SketchUp模型转PPT

## 🎯 标准15页框架

完全匹配工作室输出规范：
1. **封面** - 项目名称 + 风格 + 工作室名称
2. **任务回顾** - 客户需求 + 核心目标
3. **场地分析** - 采光/动线/视线分析
4. **设计概念** - 灵感来源 + 概念推导
5. **功能布局** - 平面规划 + 动线标注
6-8. **效果展示** - 预留3个空间效果图位置（客厅/餐厅/主卧）
9. **材质搭配** - 主要材料 + 色彩搭配
10. **打卡点分析** - 出片位置 + 预期效果
11. **关键节点** - 特殊工艺/收口细节
12. **灯光设计** - 点位 + 场景模式
13. **软装方案** - 主要家具选型 + 采购建议
14. **时间与预算** -  timeline + 预算分配
15. **封底** - 感谢聆听 + 联系方式
