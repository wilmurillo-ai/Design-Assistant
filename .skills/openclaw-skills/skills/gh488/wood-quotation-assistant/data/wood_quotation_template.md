# 核算模板说明

## 模板结构

### Sheet1: 报价明细

| 列号 | 列名 | 说明 |
|-----|------|------|
| A | 项目名称 | 材料/工艺名称 |
| B | 规格/型号 | 尺寸规格 |
| C | 单价 | 元/单位 |
| D | 数量 | 用量 |
| E | 金额 | 单价×数量 |
| F | 备注 | 损耗说明等 |

### Sheet2: 成本汇总

- 材料成本合计
- 工艺成本合计
- 损耗金额
- 总成本
- 报价金额（含利润率）

## 计算逻辑

1. 板材用量 = 展开面积 ÷ 单张面积 × (1 + 损耗率)
2. 材料成本 = Σ(单价 × 数量)
3. 工艺成本 = Σ(工艺单价)
4. 损耗金额 = (材料成本 + 工艺成本) × 损耗率
5. 总成本 = 材料成本 + 工艺成本 + 损耗金额
6. 报价金额 = 总成本 × (1 + 利润率)

## 利润率配置

- 默认利润率：30%
- 可调整范围：20%-50%
- 修改位置：`scripts/calculate_cost.py` 中的 `PROFIT_MARGIN`
