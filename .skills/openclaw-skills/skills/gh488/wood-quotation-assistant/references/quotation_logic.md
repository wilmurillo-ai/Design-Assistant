## 三、板材核算逻辑

### 3.1 标准板材规格

| 板材类型 | 规格（mm） | 厚度 | 适用场景 |
|---------|-----------|------|---------|
| 密度板 | 1220×2440 | 18mm | 吸塑/烤漆基材 |
| 颗粒板 | 1220×2440 | 18mm | 性价比柜体 |
| 多层板 | 1220×2440 | 18mm | 中高端柜体 |
| 实木板 | 定制 | 20mm | 高端定制 |
| 欧松板 | 1220×2440 | 18mm | 载奢风格 |

### 3.2 板材用量计算公式

```
展开面积（m²）= (宽×深 + 深×高 + 宽×高) × 2 × 展开系数
理论张数 = 展开面积 ÷ 单张板材面积（2.98m²）
含损耗张数 = 理论张数 × 1.08（8%加工损耗）
```

**展开系数参考**：
- 衡柜/书柜：4.5
- 橱柜：4.8
- 酒柜/展示柜：5.2
- 异形柜体：5.5

## 四、损耗计算

### 4.1 综合损耗率

```python
综合损耗率 = Σ(各工艺损耗率)
```

### 4.2 损耗金额

```python
损耗金额 = (材料成本 + 工艺成本) × 综合损耗率
```

---

## 五、成本汇总

### 5.1 计算顺序

```
1. 板材用量 → 2. 材料成本 → 3. 工艺成本 → 4. 损耗金额 → 5. 总成本
```

### 5.2 总成本公式

```python
总成本 = 材料成本 + 工艺成本 + 损耗金额
```

### 5.3 报价金额公式

```python
报价金额 = 总成本 × (1 + 利润率)
默认利润率 = 30%
```
