## How to Use

1. Sum the user's correct answers across all 3 passages (total out of 40)
2. Look up the raw score in the table above
3. If the score falls on the boundary between two bands (e.g., 29 could be 6.5, 30 could be 7.0), show as a range: `6.5-7.0`
4. Always specify this is an **estimate** — actual IELTS scoring uses a more complex algorithm

## Notes

- This conversion is for **Academic Reading only** (General Training has a different scale)
- Cambridge practice tests tend to be slightly easier than the real exam
- Use this for progress tracking, not as a guarantee of actual test performance
- Encourage users who are close to a boundary (within 2-3 marks) — small improvements yield band jumps
