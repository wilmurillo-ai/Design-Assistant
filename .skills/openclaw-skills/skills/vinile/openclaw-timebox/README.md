# openclaw-timebox-cpr

> 基于潘农菲翻译的《时间盒》，为 OpenClaw 打造的全天任务执行系统。

你有没有过这样的感受：每天忙完一天，却说不清楚做了什么、卡在哪里、明天该从哪里继续？

这个 Skill 把经过验证的**时间盒工作法**带进你的 AI 工作流：把每件事关进固定时长的"盒子"，强制聚焦；每天结束时生成结构化复盘，帮你看清全天轨迹和明日方向。

## 工作方式

```
早晨规划 → 逐盒执行 → 快速收集 → 全天复盘
```

- **PLAN**：一段对话收集全天任务，AI 帮你讨论优先级、锁定执行顺序和时间盒时长
- **RUN**：启动时间盒后 AI 完全沉默，不打扰、不催促，专注时间归你自己
- **CHECK**：每盒结束 30 秒收集——完成了吗？卡在哪？有新任务吗？
- **REVIEW**：全天结束，AI 基于完整记录生成每日复盘报告

## 核心特性

- 📅 **多平台日历自动占位**：苹果日历、飞书日历、Google 日历、企业微信日历
- ✅ **滴答清单双向同步**：规划时自动创建任务、CHECK 时追加结果、REVIEW 时标记完成，一次 OAuth 授权永久有效
- 📝 **本地日志完整留存**：规划 + 执行记录 + 每日总结 + 全天复盘
- 🔁 **每日总结自动同步**：REVIEW 完成后推送摘要到 Flomo、Notion、飞书文档等，不产生噪音
- 🌐 **多语言支持**：中文 / English

## 滴答清单集成

首次配置时选择同步到滴答清单后，Skill 会引导完成一次 OAuth 授权：

```bash
python3 ~/.config/timebox/ticktick_auth.py
```

授权完成后：
- PLAN 确认时间安排 → 自动在滴答清单创建对应任务（含时间、优先级）
- CHECK 完成收集 → 自动更新任务内容，追加实际用时与结果
- REVIEW 复盘完成 → 已完成任务自动打勾，未完成任务保留供次日处理
