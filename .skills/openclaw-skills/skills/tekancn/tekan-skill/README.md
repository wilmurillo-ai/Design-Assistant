# 特看视频 AI Skill

一个 AI Agent 技能，让你用自然语言描述即可生成专业视频、图片、数字人和音频——无需手动操作，零学习成本。

本技能遵循 [Agent Skills 规范](https://agentskills.io/specification)，兼容任何支持该规范的 AI 编程助手（Cursor、Claude Code 等）。

## 你能做什么？

安装技能后，直接用自然语言告诉 AI 你需要什么。以下是一些示例：

### 一句话创作

| 你说… | 你得到… |
|------------|------------|
| "用不同风格生成 5 张产品图" | 5 张 AI 生成的图片，并行批量生产 |
| "做一个 10 秒的猫在日落沙滩散步的视频" | 一段 AI 生成的视频片段 |
| "用这张照片做一个我介绍新产品的视频" | 一段带口型同步的数字人视频 |
| "把这瓶水放到时尚模特身上" | 一张模特手持你产品的合成图 |
| "用温柔的女声朗读这段话" | 一段文字转语音音频文件 |
| "用这段录音克隆我的声音" | 一个可用于未来任何配音的自定义声音 |
| "去掉这张产品照片的背景" | 一张干净的抠图 PNG |
| "把这张照片的背景换成热带沙滩" | 一张 AI 编辑后的图片 |
| "做一个模仿这段参考视频风格的新视频" | 一段风格迁移视频 |

### 组合工作流

真正的强大之处在于将这些能力串联在一起。描述一个目标，AI 就会为你编排整条流水线：

**电商产品发布套件** — "我有一张产品照片，帮我做一整套营销素材。" AI 会去除背景、生成多个姿势的商品模特展示图、创建一段数字人口播视频介绍产品，并批量生成不同宽高比的社交媒体海报。一次对话，一套完整的营销素材库。

**教育视频系列** — "把这篇关于气候变化的文章做成视频系列。" AI 为每个段落编写脚本、为关键概念生成插图、创建数字人主持人视频，并用选定的声音制作配音。一篇文章生成多段视频。

**多语言内容** — "把这份产品演示脚本做成英文、日文和西班牙文版本。" AI 用各语言合适的声音生成文字转语音，然后用相同的人像但不同的音轨创建数字人视频。一份脚本即刻生成多语言内容。

**故事板转视频** — "我要一个 30 秒的动画广告，包含这 4 个场景。" AI 为每个场景生成一张图片、将每张图片转换为视频片段，交付一组按顺序排列、可直接剪辑的视频。从文字描述到动态画面，一次会话搞定。

**品牌风格统一批量内容** — "生成本周 10 条社交媒体帖子，全部用我们的品牌风格。" AI 先创建一张确定视觉风格的参考图，然后用相同风格批量生产不同主题的变体。整个内容日历风格一致。

**数字代言人** — "我想当频道的门面，但不想每期都拍摄。" 上传一张人像并克隆你的声音，之后只需提供每期的脚本。AI 会生成你的形象和声音的数字人视频，配以 AI 生成的插图作为画面素材。

**带旁白的产品演示** — "用这些截图演示这个 App 的功能。" AI 根据你的功能描述生成旁白、将每张截图转换为视频片段，并创建数字人开场/结尾片段。从静态截图生成精美的产品演示。

### 使用方式

1. **安装** — `npx skills add topviewai/skill`
2. **登录** — 运行 `python scripts/auth.py login`（仅需一次）
3. **创作** — 用自然语言告诉 AI 你想要什么。它会自动选择合适的工具、模型和参数。

**你的想象力是唯一的限制** — 以上示例只是起点。你可以自由组合图片生成、视频创作、数字人、声音克隆、去除背景等能力，构建你能想到的任何工作流。用你自己的话描述创意愿景，AI 会找到实现它的方法。

## 可用模块

技能底层使用以下模块。你不需要直接调用它们——只需描述你想要什么，AI Agent 会自动选择合适的工具。

| 模块 | 脚本 | 说明 |
|--------|--------|-------------|
| [Avatar4](references/avatar4.md) | `scripts/avatar4.py` | 从照片生成数字人视频 |
| [Video Gen](references/video_gen.md) | `scripts/video_gen.py` | 图片转视频、文字转视频、Omni 参考视频生成 |
| [AI Image](references/ai_image.md) | `scripts/ai_image.py` | 文字生图和 AI 图片编辑（10+ 模型） |
| [Remove BG](references/remove_bg.md) | `scripts/remove_bg.py` | 去除图片背景 |
| [Product Avatar](references/product_avatar.md) | `scripts/product_avatar.py` | 商品模特展示图，支持模板浏览 |
| [Text2Voice](references/text2voice.md) | `scripts/text2voice.py` | 文字转语音音频 |
| [Voice](references/voice.md) | `scripts/voice.py` | 声音列表/搜索、声音克隆、删除自定义声音 |
| [Board](references/board.md) | `scripts/board.py` | 看板管理 — 整理和在网页上查看结果 |
| [User](references/user.md) | `scripts/user.py` | 积分余额和使用记录 |

## 配置

需要 Python 3.10+ 和特看视频账号。通过设备授权流程登录：

```bash
pip install -r scripts/requirements.txt
python scripts/auth.py login
```

登录后凭证自动保存到本地，所有模块自动加载。
