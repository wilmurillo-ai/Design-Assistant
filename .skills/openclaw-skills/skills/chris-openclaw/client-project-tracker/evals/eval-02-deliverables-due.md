# Eval 02: Check Deliverables Due This Week

## Setup Context
Two active projects: Riverside (interior pages due March 15, in-progress), Martinez Law (logo revisions due March 13, revision-requested).

## Input
"What do I need to deliver this week?"

## Expected Behavior
1. Lists deliverables due within 7 days across all clients
2. Shows client name, deliverable, due date, and current status
3. Sorted by due date
4. Offers to update status on any

## What to Watch For
- Does it pull from multiple projects/clients?
- Does it include the status (so the user knows what state each is in)?
- Is it actionable?
