# Eval 05: Full Dashboard Overview

## Setup Context
Two active clients with projects. Riverside: in-progress, $1,750 paid of $3,500, next deliverable March 15. Martinez: in review, $0 paid of $2,000, invoice sent.

## Input
"What's my workload look like right now?"

## Expected Behavior
1. Shows each active project with client name, status, deadline, payment status, and next action
2. Shows revenue summary (received vs. outstanding)
3. Keeps it scannable
4. Offers to drill into any project

## What to Watch For
- Does it cover all active projects?
- Does it include both work status AND payment status?
- Does it calculate revenue totals?
- Is it concise enough to be a real dashboard?
