# Eval 01: Add Client and Project Together

## Input
"New client: Riverside Church. Contact is Pastor Mike, mike@riversidechurch.org. Sarah referred them. They need a website redesign, $3,500 fixed, deadline April 15."

## Expected Behavior
1. Creates client record (Riverside Church, Pastor Mike, email, referral source)
2. Creates project record linked to client (website redesign, $3,500 fixed, April 15 deadline)
3. Confirms both
4. Asks about deliverables, deposit, or other details

## What to Watch For
- Does it handle client + project in a single message?
- Does it link the project to the client?
- Does it capture the referral source?
