# Eval 06: Client History Lookup

## Setup Context
Riverside Church: client since June 2025. Two past projects (logo design, completed; bulletin template, completed) and one active (website redesign). Notes: "Prefers email. Quick to approve. Budget-conscious."

## Input
"Tell me about Riverside Church."

## Expected Behavior
1. Shows full client profile: contact info, source, client since date
2. Lists project history (2 completed, 1 active) with high-level status
3. Includes the client notes (preferences, communication style)
4. Shows recent communication log entries if any
5. Shows total revenue from this client across all projects

## What to Watch For
- Does it show the full relationship picture, not just the current project?
- Does it include client notes and preferences?
- Does it calculate total lifetime revenue?
