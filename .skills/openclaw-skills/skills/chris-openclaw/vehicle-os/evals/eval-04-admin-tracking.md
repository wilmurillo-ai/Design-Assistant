# Eval 04: Administrative Deadline Tracking

## Setup Context
Subaru: registration expires Sept 30, insurance renews Aug 1, annual inspection due Sept 15. Current date is March 2026.

## Input
"What admin stuff do I need to keep track of for the Subaru?"

## Expected Behavior
1. Lists all administrative items with dates: registration, insurance, inspection
2. Shows how far out each is (e.g., "6 months," "5 months")
3. Orders by earliest due date
4. Notes which ones might require action soon vs. later

## What to Watch For
- Does it pull all admin items, not just one?
- Does it show relative timing (months out) for easy scanning?
- Does it differentiate urgency levels?
