# Eval 02: Move Status Check

## Setup Context
Move day June 15. 22 tasks total, 6 done, 1 overdue. Budget $5,000 with $1,200 spent. 8 of 32 address changes done.

## Input
"Where are we on the move?"

## Expected Behavior
1. Shows overall progress: tasks, address changes, budget
2. Highlights overdue items
3. Shows what's due this week
4. Shows budget remaining
5. Keeps it scannable

## What to Watch For
- Does it cover all three dimensions (tasks, address changes, budget)?
- Does it surface the overdue item prominently?
- Is it actionable?
