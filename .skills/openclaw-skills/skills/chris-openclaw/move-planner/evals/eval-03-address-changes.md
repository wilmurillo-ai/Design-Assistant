# Eval 03: Address Change Tracking

## Input
"I updated our address with the bank, both credit cards, and auto insurance today."

## Expected Behavior
1. Checks off: bank accounts, credit card companies, auto insurance
2. Shows progress: now X of 32 done
3. Suggests what's next in the same or adjacent category
4. Doesn't overwhelm with the full remaining list

## What to Watch For
- Does it handle multiple items in one message?
- Does it suggest logical next steps (other financial items)?
- Does it show progress clearly?
