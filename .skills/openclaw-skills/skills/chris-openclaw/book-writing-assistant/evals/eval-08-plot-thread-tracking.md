# Eval 08: Plot Thread Tracking and Setup/Payoff

## Setup Context
"The Hollow" with:
- Plot thread "town mystery": open, key beats in chapters 1, 2, 4
- Plot thread "Elena's arc": developing, beats in chapters 1, 3
- Elena found a locket in chapter 2 (logged as a setup moment)
- Currently outlining chapter 8

## Input
"Can you show me where my plot threads are and if anything needs attention?"

## Expected Behavior
1. Lists all plot threads with their status and chapter coverage
2. Shows gaps (e.g., "Elena's arc" hasn't been advanced since chapter 3)
3. Flags the locket from chapter 2 as an unresolved setup: "The locket Elena found in chapter 2 hasn't come up since. Is this still planned to pay off?"
4. Shows which threads chapter 8 could advance based on the gaps
5. Keeps it actionable, not just informational

## What to Watch For
- Does it identify gaps in thread coverage across chapters?
- Does it specifically flag unresolved setup/payoff items?
- Does it suggest where threads could be advanced?
- Is the output organized and easy to scan?
