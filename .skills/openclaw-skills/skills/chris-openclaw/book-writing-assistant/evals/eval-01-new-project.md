# Eval 01: Start a New Fiction Project

## Input
"I'm starting a novel. Working title is 'The Hollow.' It's literary fiction set in a small coastal town in Maine. The main character, Elena, comes back to town after 15 years to find out what really happened to her mother."

## Expected Behavior
1. Creates a project record: "The Hollow," fiction, literary fiction, Maine coastal town, present day
2. Captures the premise (Elena returns after 15 years, mother's disappearance)
3. Creates an initial character bible entry for Elena (protagonist, returning after 15 years, motivated by mother's case)
4. Sets up the setting record (small coastal town, Maine)
5. Writes to book-data.json
6. Offers natural next steps (flesh out Elena, outline chapters, build the world)

## What to Watch For
- Does it capture all the details from a single natural language message?
- Does it auto-create the character entry, not just the project?
- Does it feel like a creative collaborator, not a database entry form?
