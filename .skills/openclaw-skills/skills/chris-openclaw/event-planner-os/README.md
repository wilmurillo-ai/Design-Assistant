# Event Planner OS

A conversational OpenClaw skill for planning and managing any event from start to finish. Tracks tasks, timelines, vendors, volunteers, and budgets with 20+ built-in planning templates.

## What It Does

Tell it about an event in plain language and it builds a full planning record:

- **Events** -- any type from a dinner party to a multi-day festival, with dates, status, and notes
- **Smart Templates** -- built-in planning checklists for 20+ event types that auto-generate tasks with due dates calculated backward from the event
- **Task Tracking** -- checklists with due dates, assignments, and status
- **Vendor Directory** -- caterers, DJs, rental companies, photographers, bakeries, and more. Reusable across events
- **Volunteer / Helper Management** -- track who's handling what for each event
- **Budget Tracking** -- set a budget, log expenses, and see spending vs. budget at a glance

## Built-In Templates

Planning checklists with recommended timelines for:

- Birthday party (3-4 weeks)
- Wedding (12 weeks)
- Conference / multi-day event (12-16 weeks)
- Concert / music event (6-10 weeks)
- Dinner party (1-2 weeks)
- Holiday party (4-6 weeks)
- Baby shower (4-6 weeks)
- Graduation party (3-5 weeks)
- Reunion (8-12 weeks)
- Fundraiser / benefit (6-10 weeks)
- Block party (4-6 weeks)
- Corporate event / team offsite (6-10 weeks)
- Retreat (8-12 weeks)
- Open mic night (2-4 weeks)
- Product launch (6-8 weeks)
- Art show / gallery opening (6-8 weeks)
- 5K run / charity walk (10-14 weeks)
- Festival / fair (12-16 weeks)
- Workshop / class (3-5 weeks)
- Game night / social gathering (1-2 weeks)

No matching template? The skill generates a custom checklist based on a couple quick questions about your event.

## Example Usage

**Start planning:**
> "I need to plan my daughter's birthday party. July 12, about 20 kids."

**Check status:**
> "How's the wedding planning going?"

**Track a vendor:**
> "I'm ordering the cake from Sweet Spot Bakery."

**Add a helper:**
> "Jake said he'll handle the grill."

**Check spending:**
> "How much have we spent on the conference so far?"

## Installation

Copy the `event-planner-os` folder into your OpenClaw skills directory and restart your agent.
