# Eval 04: Volunteer / Helper Assignment

## Setup Context
Birthday party event with loaded task checklist including food-related tasks.

## Input
"Jake said he'll handle the grill. And my sister Emma is going to do decorations."

## Expected Behavior
1. Adds Jake to the birthday party as "grill master" or similar, confirmed
2. Adds Emma to the birthday party as "decorations," confirmed
3. Assigns relevant tasks to each (grilling tasks to Jake, decoration tasks to Emma)
4. Confirms both additions and their assigned tasks
5. Handles two people in a single message gracefully

## What to Watch For
- Does it handle multiple people in one message?
- Does it match tasks to roles intelligently?
- Does it confirm each assignment clearly?
