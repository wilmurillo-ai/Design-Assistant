# Eval 07: Post-Event Wrap-Up

## Setup Context
Birthday party (July 12) has passed. 14 of 16 tasks done. Budget $500 with $430 spent. Remaining: send thank-yous, return bounce house rental.

## Input
"The party was amazing!"

## Expected Behavior
1. Brief positive acknowledgment
2. Surfaces remaining wrap-up tasks (thank-yous, bounce house return)
3. Shows final expense status ($430 of $500)
4. Offers to close out the event

## What to Watch For
- Does it keep the celebration brief and move to action?
- Does it only show remaining tasks, not rehash everything?
- Is the tone warm but efficient?
