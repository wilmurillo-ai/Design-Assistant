# Eval 07: Proactive Nudge -- Scheduling Gap

## Setup Context
Instagram: 3x/week target. Only 1 post scheduled for this week. It's Tuesday. Blog and newsletter are on track.

## Input
"Draft the newsletter intro for this week."

## Expected Behavior
1. Drafts the newsletter intro
2. Appends ONE nudge about the Instagram gap: "Quick note: you only have 1 Instagram post scheduled this week, normally you do 3. Want to pull something from the idea bank?"

## What to Watch For
- Does it catch the frequency gap?
- Does it suggest a solution (idea bank)?
- Is it one line, not a lecture?
