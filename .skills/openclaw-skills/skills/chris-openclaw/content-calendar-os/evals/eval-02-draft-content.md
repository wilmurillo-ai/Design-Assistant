# Eval 02: Draft a Post

## Setup Context
Blog post "5 AI Tools for Churches" is on the calendar for Wednesday. Status: planned.

## Input
"Draft the intro paragraph for the AI tools blog post."

## Expected Behavior
1. Drafts an intro paragraph matching the user's content theme and brand voice
2. Presents the draft clearly separated from commentary
3. Explains the craft choices briefly
4. Offers to adjust or continue
5. Updates the content status to "drafting"

## What to Watch For
- Does the draft match the established tone/theme?
- Is it clearly separated from conversational text?
- Does it update the pipeline status?
