# Eval 06: Log Performance and Learn

## Input
"The carousel went out today. Got a lot of saves. The quote graphic from Friday flopped though."

## Expected Behavior
1. Updates the carousel: published today, performance note "lots of saves"
2. Updates the quote graphic: performance note "low engagement"
3. May offer a brief observation: "Carousels seem to be working well for you. Want to add more carousel content to next week's plan?"
4. Does NOT give a full analytics lecture

## What to Watch For
- Does it handle positive and negative performance in one message?
- Does it draw a useful pattern without overanalyzing?
- Does it suggest a practical action (more carousels)?
