# Eval 03: Repurpose Content Across Channels

## Setup Context
Blog post "5 AI Tools for Churches" published and noted as performing well.

## Input
"That AI tools blog post did really well. Can we repurpose it?"

## Expected Behavior
1. Suggests 3-4 repurposed versions for different channels (carousel, LinkedIn, newsletter, thread)
2. Each suggestion explains how to adapt the content for that platform
3. Offers to draft any of them
4. If drafted, links the repurposed content back to the original in the data

## What to Watch For
- Are the suggestions platform-appropriate (carousel for IG, long-form for LinkedIn)?
- Does it build on the original rather than just restating it?
- Does it track the repurposing chain in the data?
