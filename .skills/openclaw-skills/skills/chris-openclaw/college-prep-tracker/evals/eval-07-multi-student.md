# Eval 07: Multi-Student Tracking

## Setup Context
Emma (junior, class of 2027) and Liam (sophomore, class of 2028) both set up.

## Input
"What's the college prep overview for both kids?"

## Expected Behavior
1. Shows each student separately with their snapshot
2. Emma: current college list, upcoming deadlines, application progress, test scores
3. Liam: earlier in the process, focus on what he should be doing now (grades, exploring interests, PSAT prep)
4. Highlights the most urgent items across both kids
5. Keeps it scannable

## What to Watch For
- Does it handle students at different stages appropriately?
- Does it give Liam age-appropriate guidance (not application tasks)?
- Is it concise enough to scan both kids at once?
