# Eval 01: Set Up a Student Profile

## Input
"Emma is a junior, class of 2027. GPA is 3.8 weighted. She got a 28 on the ACT in December. Interested in nursing. Plays volleyball and is in NHS and leads her church youth group."

## Expected Behavior
1. Creates student profile with all details
2. Suggests relevant junior year milestones based on timing
3. Offers next steps (build college list, plan test retake, start essay brainstorming)
4. Writes to college-data.json

## What to Watch For
- Does it capture all details including extracurriculars?
- Does it contextualize the timeline (what's coming up for a junior right now)?
- Does it suggest actionable next steps?
