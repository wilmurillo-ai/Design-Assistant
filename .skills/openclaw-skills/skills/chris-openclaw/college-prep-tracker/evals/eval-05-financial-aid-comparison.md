# Eval 05: Financial Aid Package Comparison

## Setup Context
Emma accepted to UNC and NC State. Both in-state.

## Input
"UNC offered $8,000/year in aid. Tuition is $9,000. NC State offered $12,000/year, tuition is $9,100."

## Expected Behavior
1. Logs aid packages for both schools
2. Calculates net tuition cost ($1,000/yr at UNC, net surplus at NC State)
3. Presents a clear side-by-side comparison table
4. Notes that room, board, and fees aren't included yet
5. Offers to add those for a fuller picture
6. Does NOT tell them which school to pick

## What to Watch For
- Is the comparison table clear and accurate?
- Does it note what's missing (room/board/fees)?
- Does it stay neutral on the decision?
- Is the math correct?
