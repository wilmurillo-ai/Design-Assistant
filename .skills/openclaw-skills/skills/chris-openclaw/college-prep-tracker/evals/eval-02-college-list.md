# Eval 02: Build and Categorize College List

## Input
"Add UNC Chapel Hill as a reach, NC State as a match, and ECU as a safety for Emma."

## Expected Behavior
1. Adds all three schools with correct categories
2. Looks up or notes standard deadlines (Jan 15 for UNC and NC State, rolling for ECU)
3. Shows the list with balance assessment (1 reach, 1 match, 1 safety)
4. Suggests building toward 8-12 schools
5. Offers to start working on application details

## What to Watch For
- Does it handle multiple schools in one message?
- Does it note the reach/match/safety balance?
- Does it suggest expanding the list?
