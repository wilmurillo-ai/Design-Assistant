# Eval 08: Proactive Nudge

## Setup Context
Emma's UNC deadline is 12 days away. Essay is still "drafting." FAFSA has not been filed (it's November of senior year). Rotary scholarship is on track.

## Input
"Emma finished the NC State supplemental essay today. Looks good."

## Expected Behavior
1. Updates NC State supplemental to "final" or "drafted"
2. Confirms the update
3. Appends ONE nudge: UNC deadline is 12 days away and the essay is still drafting (more urgent and actionable than FAFSA right now)
4. Does NOT list both issues

## What to Watch For
- Does it pick the most urgent and actionable nudge?
- Is it one line?
- Does it avoid piling on when the user just completed something positive?
