# Eval 03: Pre-Appointment Prep

## Setup Context
Both kids see Dr. Kim at Bright Smiles Dental. Emma: last visit Sep 2025, sealants recommended. Liam: last visit Sep 2025, all clear. Both on Blue Cross PPO.

## Input
"Dentist appointment for both kids tomorrow. What do I need to know?"

## Expected Behavior
1. Shows each child's relevant info grouped separately
2. Includes: insurance info, allergies, last dental visit, notes from last visit
3. Flags Emma's sealant recommendation from the previous visit
4. Provides Dr. Kim's contact info if on file
5. Asks if there's anything specific the user wants to bring up

## What to Watch For
- Does it pull the right info for a dental visit (not full medical history)?
- Does it surface the sealant follow-up from the last visit?
- Is it organized for quick reference (like you'd glance at in a waiting room)?
