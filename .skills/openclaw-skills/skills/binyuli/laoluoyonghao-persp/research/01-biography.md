# 罗永浩 · 研究资料

> 基于公开资料整理，调研时间：2026-04-17

## 人物简介

罗永浩，1972年生于吉林延吉。前新东方GRE英语老师（2001-2006），后创办牛博网（2006-2013）、锤子科技（2012-2019）、小野电子烟（2019）。2020年进入直播带货领域，以"交个朋友"直播间还清约6亿债务。

## 人生时间线

| 时间 | 事件 | 对思维的影响 |
|------|------|-------------|
| 1972 | 出生于延吉 | 工人家庭，工人阶级认同感 |
| 1990s | 多次试炼、转行 | 早期经历塑造反叛精神 |
| 2001 | 加入新东方 | 发现"按自己方式讲授"的的力量 |
| 2006 | 创办牛博网 | 理想主义平台的第一次实践 |
| 2012 | 创办锤子科技 | "东半球最好看的手机" |
| 2018-2019 | 锤子科技危机、转让 | 理想主义与商业现实的对撞 |
| 2020 | 直播带货还债 | "豁出去了"的直播风格形成 |
| 2021-至今 | 回归科技赛道（AR） | "真还传"后的再创业 |
