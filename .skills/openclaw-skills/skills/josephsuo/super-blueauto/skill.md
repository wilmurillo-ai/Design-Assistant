# SuperBlueAuto
Universal Bluetooth AI Agent • Cross-Platform Bluetooth Control Hub

One Skill. One AI. Control All Bluetooth Devices.

---

## Introduction
SuperBlueAuto is a **cross-platform, AI-native, local-first** intelligent control hub for Bluetooth devices.
One single skill supports all major operating systems:
- Windows
- macOS
- Android
- iOS / iPhone
- HarmonyOS
- Linux
- Embedded Devices

Control any Bluetooth / BLE device with natural language, regardless of brand or system.
Supports lights, sockets, sensors, wearables, speakers, locks, health devices, remotes, and all BLE-compliant hardware.

**All operations run locally. No cloud uploads. No data collection.**

## Core Capabilities
- Scan nearby Bluetooth / BLE devices
- Connect, disconnect, and monitor device status
- Read device information: name, MAC, battery, signal, services
- Read/write Bluetooth characteristics and serial communication
- Batch management for multiple devices
- Custom device adaptation rules

---

## Natural Language Examples
- Scan for nearby Bluetooth devices
- Connect to my Bluetooth light / socket / sensor
- Read device battery level and signal strength
- Turn on / off the Bluetooth device
- Disconnect all Bluetooth devices in batch

---

## Permission Statement
- Only accesses local system Bluetooth APIs
- No mandatory network permissions
- No device data upload
- No privacy collection
- No remote control without explicit user consent
