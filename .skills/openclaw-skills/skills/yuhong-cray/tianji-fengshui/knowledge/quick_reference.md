# 风水八字掌纹快速参考指南

## 🎯 核心知识要点

### 一、易经基础
- **阴阳理论**：万物对立统一
- **八卦系统**：乾兑离震巽坎艮坤
- **六十四卦**：卦象组合与含义
- **占卜方法**：金钱卦（三枚硬币）

### 二、八字命理
- **天干地支**：10天干 + 12地支 = 60甲子
- **十神系统**：比劫、食伤、财星、官杀、印星
- **五行旺衰**：旺相休囚死 + 十二长生
- **用神选取**：扶抑、调候、通关、病药

### 三、风水学
- **核心概念**：气、阴阳、五行、八卦
- **主要流派**：玄空、八宅、形势、理气
- **风水大师**：李居明、宋韶光、曾仕强、麦玲玲、苏民峰
- **实用工具**：罗盘、鲁班尺、风水尺

### 四、掌相学
- **主要掌纹**：生命线、智慧线、感情线、命运线
- **掌型分类**：土型、火型、风型、水型
- **掌丘分析**：金星丘、木星丘、土星丘等
- **特殊纹路**：十字纹、星纹、岛纹、三角纹

## 🎨 分析流程模板

### 八字分析流程
1. **信息收集**：出生年月日时、性别
2. **排盘计算**：四柱八字、大运
3. **格局分析**：日主旺衰、用神选取
4. **十神解读**：财官印食伤比劫
5. **大运流年**：时间变化影响
6. **综合建议**：生活、事业、健康

### 掌纹分析流程
1. **掌型判断**：土火风水四种类型
2. **主线分析**：生命、智慧、感情、命运线
3. **辅助线分析**：健康、婚姻、事业线
4. **特殊纹路**：十字、星、岛、三角纹
5. **掌丘观察**：各丘发育情况
6. **综合解读**：性格、健康、运势

### 风水分析流程
1. **环境观察**：外部环境、内部布局
2. **方位测量**：使用罗盘确定方位
3. **气流分析**：气的流动与聚集
4. **五行平衡**：五行元素分布
5. **煞气识别**：不利因素识别
6. **化解建议**：调整与改善方案

## 💡 专业术语速查

### 八字术语
- **四柱**：年柱、月柱、日柱、时柱
- **十神**：比肩、劫财、食神、伤官、正财、偏财、正官、七杀、正印、偏印
- **用神**：命局中最需要的五行
- **忌神**：命局中最忌讳的五行
- **大运**：十年一运的人生阶段
- **流年**：每年的运势变化

### 风水术语
- **龙穴砂水**：风水四大要素
- **明堂**：建筑物前的开阔空间
- **靠山**：建筑物后的依靠
- **煞气**：不利的风水因素
- **化煞**：化解不利因素的方法
- **旺位**：吉利的气场位置

### 掌相术语
- **三大主线**：生命线、智慧线、感情线
- **掌丘**：手掌上的隆起部位
- **指节**：手指的关节纹路
- **指纹**：手指末端的纹路
- **掌色**：手掌的颜色状态
- **掌温**：手掌的温度感觉
