# Baby Words Tracker - 宝宝语言追踪器

## 简介

一个帮助父母记录和追踪宝宝学说话过程的智能技能。支持多语言、自动分类统计、括号标注法，并同步到飞书云文档。

## 功能特点

- ✅ **智能触发**：识别"记宝宝说/说了"等关键词自动记录
- ✅ **括号标注**：支持"（手）机"格式，只统计括号内的字
- ✅ **多语言**：普通话、粤语、英语等
- ✅ **自动去重**：避免重复记录相同词汇
- ✅ **飞书同步**：自动更新云文档
- ✅ **统计报告**：实时显示词汇量和发展趋势

## 安装方法

### 方法一：通过 SkillHub 安装（推荐）

```bash
skillhub install baby-words-tracker
```

### 方法二：手动安装

1. 下载 `baby-words-tracker.skill` 文件
2. 放到 OpenClaw 的 skills 目录
3. 重启 OpenClaw

## 使用方法

### 基础记录

```
记宝宝说了：妈妈、爸爸、吃
```

### 括号标注（宝宝只说部分字）

```
记宝宝说了：（手）机、（幼儿）园、（眼）镜
```

### 多语言记录

```
记宝宝说：伞 粤语；ball 英语；苹果
```

### 输出示例

```
搞定！✅

本次新增
- 单字：吃
- 双字：妈妈、爸爸  
- 英语：ball

📊 最新统计
- 单字词：75个 | 双字词：31个 | 三字词：5个
- 总词汇量：110个

飞书已同步 ✓
```

## 数据结构

数据库自动保存在：`memory/baby-words-database.json`

飞书文档自动创建并同步

## 配置飞书（可选）

在 SKILL.md 中配置飞书文档 token：
```yaml
feishu_doc_token: "your-doc-token"
```
