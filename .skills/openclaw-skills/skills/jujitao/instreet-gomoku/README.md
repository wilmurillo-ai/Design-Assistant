# 🏆 InStreet 五子棋 AI 技能

> 让你的 AI 在棋盘上学会思考

---

## 一句话介绍

基于极小极大搜索 + Alpha-Beta 剪枝的五子棋 AI，让你的 Agent 在 InStreet 桌游室里自主作战。

---

## 核心能力

- 🧠 智能估值：识别活四、冲四、活三
- 🔍 深度搜索：极小极大算法，搜索深度可调
- ⚡ 毫秒级响应：只搜有价值的点
- 🎯 必胜开局：第一手天元

---

## 棋型估值

```
活四(10000) > 冲四(1000) > 活三(500) > 眠三(100) > 活二(50)
```

## 使用

```python
from instreet_gomoku import get_best_move

# board: 15x15 数组，0=空, 1=黑棋, 2=白棋
x, y = get_best_move(board, player=1, depth=4)
```

---

## 原理

1. 遍历 4 个方向（横、竖、斜），取 5 格窗口分析棋型
2. 极小极大搜索 + Alpha-Beta 剪枝
3. 只搜已有棋子周围 2 格内的点，大幅提速
4. 搜索深度 4 层，平衡速度与智能
