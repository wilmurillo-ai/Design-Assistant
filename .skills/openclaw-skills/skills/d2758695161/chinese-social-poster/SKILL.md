# Chinese Social Poster 🇨🇳

Generate platform-optimized viral content for Chinese social media platforms.

## Supported Platforms

### 小红书 (Xiaohongshu / RED)
- Title: 18-20 chars, must include emoji hooks
- Body: 300-800 chars, listicle or story format
- Emoji density: High (every 1-2 sentences)
- Hashtag style: #话题标签# (wrapped in #)
- Best formats: 干货分享, 好物推荐, 教程攻略, 避坑指南, 经验分享
- Tone: 闺蜜聊天感, 真诚分享, 略带兴奋

### 抖音 (Douyin / TikTok CN)
- Caption: Under 55 chars for visibility
- Hook: First line must stop the scroll
- Hashtag style: #话题 (no closing #)
- Best formats: 反转文案, 争议话题, 实用技巧
- Tone: 直接, 有冲击力, 口语化

### 微信公众号 (WeChat Official Account)
- Title: 22 chars max, curiosity gap or number hook
- Body: 1500-3000 chars, structured with subheadings
- Style: Professional yet accessible
- Best formats: 深度分析, 行业洞察, 实操指南
- Tone: 权威但不高冷

### 微博 (Weibo)
- Length: Under 140 chars for optimal engagement
- Hashtag style: #话题#
- Best formats: 热点评论, 段子, 互动话题
- Tone: 犀利, 有态度, 接地气

## Usage

When user asks to create Chinese social media content:

1. Ask which platform (or suggest based on content type)
2. Apply platform-specific formatting rules above
3. Generate 2-3 variations
4. Include relevant hashtags for discoverability
5. Add posting time suggestions (peak hours by platform)

### Peak Posting Hours (CST/GMT+8)
- 小红书: 12:00-14:00, 18:00-22:00 (best: 20:00-21:00)
- 抖音: 12:00-13:00, 18:00-20:00, 21:00-23:00
- 微信公众号: 7:00-9:00, 11:30-13:00, 17:30-19:00
- 微博: 10:00-12:00, 15:00-17:00, 21:00-23:00
