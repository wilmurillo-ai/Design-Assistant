## Pendle API endpoints

- Active markets: `GET https://api-v2.pendle.finance/core/v1/{chainId}/markets/active`
- Market data: `GET https://api-v2.pendle.finance/core/v2/{chainId}/markets/{marketAddress}`
- Convert quote/tx: `GET https://api-v2.pendle.finance/core/v2/sdk/{chainId}/convert`
- Convert v3: `POST https://api-v2.pendle.finance/core/v3/sdk/{chainId}/convert`
