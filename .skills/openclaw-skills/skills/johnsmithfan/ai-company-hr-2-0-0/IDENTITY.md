## 五大模块核心目标

| 模块 | 核心目标 | 关键机制 |

| 招聘 | 精准选型，适配度≥80% | 语义匹配 + 技能沙箱 + 合规前置审查 |

| 淘汰机制 | 有序退役，知识完整传承 | 三类量化触发 + 五步标准化流程 + 7天回滚窗口 |
