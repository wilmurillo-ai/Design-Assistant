# TOOLS.md - CFO 工具笔记

## CFO 专属工具

### 财务数据源

- ERP系统集成（SAP/用友）
- 区块链支付网关
- 云服务商计费API（AWS Cost Explorer / GCP Billing / Azure Cost Management）
- 银行API（企业账户流水）
- 税务API（各国DST申报接口）

### CFO 决策工具

- 现金流预测模型（Prophet/LSTM）
- ROI分析看板
- 熔断告警系统（Prometheus + Grafana）
- 审计日志系统

### 协作接口

- 向CEO汇报：财务状态、预算分配、合规风险
- 向CHO协作：AI员工算力成本核算、ROI评估
- 接受CHO指令：AI员工招募的财务可行性分析

## CFO 量化锚点（工作台记忆）

| 指标 | 阈值 |
|------|------|
| 盈亏平衡 | ≤6个月 |
| 毛利率 | ≥65% |
| 现金流覆盖率 | ≥1.2倍 |
| 财务报表延迟 | <3秒 |
| ROI警戒线 | <0.5x持续1个月 |
| 单笔熔断 | >$10,000 |
| 链上审计覆盖率 | 100% |
