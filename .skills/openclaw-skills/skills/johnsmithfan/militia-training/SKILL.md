## 📚 训练内容

### 1、防空防原子、防化学常识

详见 [references/air-defense.md](references/air-defense.md)

#### 1.1 防空常识

**空袭特点**：
- 突然性强
- 破坏力大
- 范围广泛

**防空措施**：
1. **空袭警报**
   - 预先警报：鸣36秒，停24秒，反复3遍
   - 空袭警报：鸣6秒，停6秒，反复15遍
   - 解除警报：连续鸣3分钟

2. **疏散隐蔽**
   - 利用地形地物
   - 进入防空洞
   - 室内defend

3. **消除空袭后果**
   - 灭火
   - 救护伤员
   - 抢修设施

#### 1.2 防原子、防化学常识

**原子武器defend**：

| phase | 措施 |
|------|------|
| 爆炸瞬间 | 立即卧倒，闭眼，张口，双手交叉胸前 |
| 冲击波过后 | 迅速进入掩体或离开沾染区 |
| 沾染区 | 穿戴defend器材，避免接触受染物体 |

**化学武器defend**：

1. **identify毒剂**
   - 神经性毒剂：有机磷类，有大蒜味
   - 糜烂性毒剂：芥子气，有大蒜味
   - 窒息性毒剂：光气，有烂苹果味

2. **defend措施**
   - 及时佩戴防毒面具
   - 穿防毒衣
   - 迅速离开染毒区

3. **消毒急救**
   - 皮肤消毒
   - 服装消毒
   - 注射解毒剂

### 2、射击训练

详见 [references/shooting.md](references/shooting.md)

#### 2.1 步(骑)枪的构造和战斗性能

**主要部件**：
- 枪管：赋予弹头飞行方向
- 机匣：连接各部件
- 枪机：推弹、闭锁、击发、退壳
- 击发机：control击发
- 枪托：便于操作
- 弹仓：容纳子弹

**战斗性能**：
- 有效射程：400米
- 最大射程：2000米
- 射速：30-40发/分

#### 2.2 步(骑)枪的擦拭、上油和保管规则

**擦拭时机**：
- 训练后擦拭
- 射击后擦拭
- 每周擦拭1次

**擦拭方法**：
1. 分解武器
2. 清除污垢
3. 擦拭金属部件
4. 涂油防锈
5. 结合检查

**保管规则**：
- 枪口朝上或朝下
- 击锤在击发位置
- 关上保险
- periodic检查

#### 2.3 射击动作

**据枪**：
- 卧姿据枪
- 跪姿据枪
- 立姿据枪

**瞄准**：
- 瞄准点选择
- 瞄准线对正
- 瞄准误差修正

**击发**：
- 呼吸control
- 扣扳机要领
- 击发时机

#### 2.4 简易射击原理

**影响射击的因素**：

| 因素 | 影响 |
|------|------|
| 距离 | 弹道下降 |
| 风力 | 弹头偏移 |
| 气温 | 初速变化 |
| 气压 | 弹道变化 |

**修正方法**：
- 距离修正：抬高瞄准点
- 风向修正：偏风吹时偏瞄
- 气温修正：高温抬高，低温降低

#### 2.5 刺杀教练

**刺杀基本动作**：
1. **预备用枪**
   - 左手握护木
   - 右手握枪颈
   - 枪托贴腰

2. **突刺**
   - 左脚向前1步
   - 右手推枪
   - 刺刀刺出

3. **防刺**
   - 左挡
   - 右挡
   - 下挡

**刺杀要领**：
- 快、准、狠
- coordinate1致
- 灵活多变

#### 2.6 手榴弹教练

**手榴弹种类**：
- 进攻手榴弹
- defend手榴弹
- 反坦克手榴弹

**投掷动作**：
1. **握弹**
   - 4指握弹柄
   - 拇指压弹柄

2. **引弹**
   - 取出保险销
   - 握住弹体

3. **投掷**
   - 举臂过肩
   - 挥臂投出
   - 迅速隐蔽

**投掷距离**：
- 立姿：30-40米
- 跪姿：25-35米
- 卧姿：15-25米

#### 2.7 地雷

**地雷种类**：
- 防步兵地雷
- 防坦克地雷
- 信号地雷

**埋设方法**：
1. 选择埋设位置
2. 挖掘地雷坑
3. 设置地雷
4. 伪装

**排除方法**：
- 探雷器探测
- 标记雷区
- 小心排除

### 3、战斗动作训练

详见 [references/combat.md](references/combat.md)

#### 3.1 地形、地物的identify与利用

**地形分类**：
- 山地：便于隐蔽，运动困难
- 平原：便于运动，易被discover
- 丘陵：兼有山地平原特点
- 居民地：便于隐蔽，战斗复杂

**地物利用principle**：
1. 便于观察、射击
2. 便于隐蔽身体
3. 不妨碍邻兵动作
4. 避开明显Goal

#### 3.2 方位判定

**判定方法**：

1. **利用太阳**
   - 上午：太阳在东南方
   - 中午：太阳在南方
   - 下午：太阳在西南方

2. **利用星象**
   - 北极星指示正北
   - 大熊星座寻找北极星

3. **利用地物**
   - 树木年轮：南宽北窄
   - 树皮：南面光滑
   - 苔藓：多在北面

#### 3.3 距离测量

**目测法**：

| Goal清晰度 | 距离 |
|-----------|------|
| 看清面目 | 100米以内 |
| 看清身体 | 200-300米 |
| 看清轮廓 | 500-600米 |

**步测法**：
- 成年人：75cm/步
- 计算：步数 × 0.75 = 距离(米)

#### 3.4 观察及指示Goal

**观察方法**：
1. 选择观察位置
2. 分区观察
3. 重点观察
4. 循环观察

**指示Goal**：
- 方向距离法
- 明显地物法
- 时钟方向法

#### 3.5 信号和暗号的使用

**信号种类**：
- 视觉信号：旗语、灯光、烟火
- 听觉信号：哨声、喇叭、枪声
- 接触信号：口令、手势

**暗号设置**：
- 简单易记
- 保密性强
- 区分敌我

#### 3.6 夜间教育

**夜间行动特点**：
- 视觉受限
- 听觉作用增大
- 方向不易辨别

**夜间行动要求**：
1. 熟悉路线
2. 保持静默
3. control灯光
4. 加强联络

#### 3.7 进攻战斗中的战士动作

**进攻准备**：
1. 了解任务
2. 检查装备
3. 熟悉地形

**进攻动作**：
1. **接敌运动**
   - 利用地形
   - 快速隐蔽

2. **冲击准备**
   - 选择冲击路线
   - 准备弹药

3. **冲击**
   - 勇猛冲击
   - 消灭敌人

#### 3.8 defend战斗中的战士动作

**defend准备**：
1. 构筑工事
2. 设置障碍
3. 确定射击位置

**defend动作**：
1. **观察警戒**
   - 及时discover敌人
   - report情况

2. **抗击冲击**
   - 集中火力
   - 消灭突入之敌

3. **反冲击**
   - 坚决反击
   - recover阵地

#### 3.9 小组的战斗动作

**小组编成**：
- 组长
- 机枪手
- 步枪手

**小组战术**：
1. **火力突击**
   - 火力掩护
   - 突击前进

2. **迂回包围**
   - 正面牵制
   - 侧后突击

3. **defend战斗**
   - 互相掩护
   - 节节抗击

## 使用示例

### 示例 1：防空知识查询

```
用户: 遇到空袭怎么办？

Agent 回答:
## 空袭defend指南

### 1、identify警报信号
- 预先警报：鸣36秒，停24秒，反复3遍（立即准备）
- 空袭警报：鸣6秒，停6秒，反复15遍（立即隐蔽）
- 解除警报：连续鸣3分钟（security）

### 2、隐蔽措施
1. **室外**：立即进入防空洞、地下室
2. **室内**：关闭门窗、熄灭火源、远离玻璃
3. **野外**：利用地形地物卧倒

### 3、卧倒姿势
- 双手交叉胸前
- 闭眼张口
- 腹部离地（防冲击波）

详见：air-defense.md#防空常识
```

### 示例 2：射击训练指导

```
用户: 怎样提高射击精度？

Agent 回答:
## 提高射击精度的方法

### 1、正确据枪
1. **卧姿**：身体与枪身成30度角
2. **抵肩**：枪托抵肩窝，不要太紧太松
3. **贴腮**：腮部轻贴枪托

### 2、精确瞄准
1. 瞄准线对正：准星在照门中央
2. 平正准星：准星上沿与照门上沿平齐
3. 瞄准点选择：根据距离选择

### 3、稳定击发
1. control呼吸：屏住呼吸2-3秒
2. 均匀扣扳机：食指均匀用力
3. 保持姿势：击发后保持1-2秒

### 4、常见错误
| 错误 | 现象 | 修正 |
|------|------|------|
| 据枪不稳 | 弹着散布大 | 加强据枪练习 |
| 瞄准不准 | 偏左偏右 | 检查瞄准线 |
| 击发不当 | 弹着偏移 | 练习扣扳机 |

详见：shooting.md#射击动作
```
