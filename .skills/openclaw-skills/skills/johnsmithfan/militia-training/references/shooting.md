# 射击训练指南

## 一、步(骑)枪的构造和战斗性能

### 1.1 主要部件

**枪管**：
- 作用：赋予弹头飞行方向
- 结构：内有膛线
- 特点：耐高温、高压

**机匣**：
- 作用：连接各部件
- 结构：内有闭锁机构

**枪机**：
- 作用：推弹、闭锁、击发、退壳
- 操作：拉枪机动作

**击发机**：
- 作用：控制击发
- 组成：扳机、击锤、阻铁

**枪托**：
- 作用：便于操作
- 类型：木质、金属折叠

**弹仓**：
- 作用：容纳子弹
- 容量：5-10发

### 1.2 战斗性能

| 项目 | 参数 |
|------|------|
| 口径 | 7.62mm |
| 有效射程 | 400米 |
| 最大射程 | 2000米 |
| 战斗射速 | 30-40发/分 |
| 弹仓容量 | 10发 |

## 二、步(骑)枪的擦拭、上油和保管

### 2.1 擦拭时机

- 训练后擦拭
- 射击后擦拭
- 潮湿天气后擦拭
- 每周定期擦拭

### 2.2 擦拭方法

**分解武器**：
1. 取出弹匣
2. 拉枪机检查膛内
3. 卸下枪机
4. 分解其他部件

**擦拭步骤**：
1. **清除污垢**
   - 用布擦拭金属部件
   - 清除灰尘、泥沙

2. **擦拭枪膛**
   - 用通条、布片
   - 从后向前擦拭
   - 反复多次

3. **涂油防锈**
   - 薄涂枪油
   - 不要涂太多

4. **结合检查**
   - 按顺序结合
   - 检查动作

### 2.3 保管规则

**保管要求**：
- 枪口朝上或朝下
- 击锤在击发位置
- 关上保险
- 弹匣取出

**定期检查**：
- 每周检查一次
- 雨季增加检查次数
- 发现锈蚀及时处理

## 三、射击动作

### 3.1 据枪

#### 卧姿据枪

**动作要领**：
1. 身体与射击方向成30-40度角
2. 左肘前伸，右手握枪颈
3. 枪托抵肩窝
4. 腮部轻贴枪托
5. 全身放松

**注意事项**：
- 身体不要过度紧张
- 呼吸自然
- 姿势舒适稳定

#### 跪姿据枪

**动作要领**：
1. 右腿跪地
2. 左腿屈膝
3. 左肘放在左膝上
4. 身体稍前倾

#### 立姿据枪

**动作要领**：
1. 双脚分开与肩同宽
2. 身体侧对目标
3. 左手托枪
4. 右手握枪颈

### 3.2 瞄准

#### 瞄准原理

**三点一线**：
- 照门、准星、目标

**瞄准要领**：
1. **平正准星**
   - 准星上沿与照门上沿平齐
   - 准星在照门中央

2. **瞄准线对正**
   - 瞄准线指向目标
   - 保持平正准星

3. **瞄准点选择**
   - 100米：瞄准点在下沿
   - 200米：瞄准点在中央
   - 300米：瞄准点在上沿

#### 常见错误

| 错误 | 现象 | 修正 |
|------|------|------|
| 准星偏左 | 弹着偏左 | 调整准星 |
| 准星偏右 | 弹着偏右 | 调整准星 |
| 准星过高 | 弹着偏高 | 降低准星 |
| 准星过低 | 弹着偏低 | 提高准星 |

### 3.3 击发

#### 击发要领

1. **呼吸控制**
   - 自然呼吸
   - 屏气2-3秒
   - 此时击发

2. **扣扳机**
   - 食指均匀用力
   - 不要突然扣击
   - 自然击发

3. **保持姿势**
   - 击发后保持1-2秒
   - 不要立即松懈

#### 击发常见错误

| 错误 | 后果 | 修正 |
|------|------|------|
| 扣扳机过猛 | 弹着散布大 | 均匀用力 |
| 击发后松开 | 影响精度 | 保持姿势 |
| 眨眼 | 影响瞄准 | 保持睁眼 |

## 四、简易射击原理

### 4.1 弹道知识

**弹道形成**：
- 弹头出膛后受重力影响
- 弹道呈抛物线
- 逐渐下降

**弹道要素**：
- 起点：枪口
- 最高点：弹道顶点
- 落点：命中点

### 4.2 影响射击的因素

#### 距离影响

| 距离 | 弹道下降 | 修正 |
|------|---------|------|
| 100米 | 很小 | 瞄准下沿 |
| 200米 | 较小 | 瞄准中央 |
| 300米 | 明显 | 瞄准上沿 |
| 400米 | 较大 | 抬高瞄准点 |

#### 风力影响

| 风力 | 影响 | 修正 |
|------|------|------|
| 微风 | 小 | 不需修正 |
| 和风 | 中等 | 偏风吹时偏瞄 |
| 强风 | 大 | 明显修正 |

**风力判断**：
- 微风：旗帜轻动
- 和风：旗帜飘动
- 强风：旗帜大飘

#### 气温影响

- 高温：初速增加，弹道偏高
- 低温：初速降低，弹道偏低

### 4.3 射击修正

**修正口诀**：
- 远则高，近则低
- 风偏修正：左风右瞄，右风左瞄

## 五、刺杀教练

### 5.1 基本动作

#### 预备用枪

**动作要领**：
1. 左手握护木
2. 右手握枪颈
3. 枪托贴腰右侧
4. 刺刀向前

#### 突刺

**动作要领**：
1. 左脚向前一步
2. 右手推枪向前
3. 左手握护木
4. 刺刀刺出

**要领口诀**：
- 快：动作迅速
- 准：刺点准确
- 狠：用力到位

#### 防刺

**左挡**：
- 枪身向左拨
- 同时准备反刺

**右挡**：
- 枪身向右拨
- 同时准备反刺

**下挡**：
- 枪身向下压
- 防敌刺下

### 5.2 刺杀要领

**进攻要领**：
1. 抓住时机
2. 突然快速
3. 连续进攻

**防守要领**：
1. 及时判断
2. 快速反应
3. 防守反击

## 六、手榴弹教练

### 6.1 手榴弹种类

| 类型 | 特点 | 用途 |
|------|------|------|
| 进攻手榴弹 | 破片少 | 进攻作战 |
| 防御手榴弹 | 破片多 | 防御作战 |
| 反坦克手榴弹 | 穿甲能力强 | 打坦克 |

### 6.2 投掷动作

#### 立姿投掷

**动作要领**：
1. **握弹**
   - 四指握弹柄
   - 拇指压弹柄

2. **引弹**
   - 取出保险销
   - 举臂过肩

3. **投掷**
   - 挥臂向前
   - 用力投出
   - 迅速隐蔽

**投掷距离**：30-40米

#### 跪姿投掷

**动作要领**：
- 右腿跪地
- 左腿屈膝
- 投掷动作同立姿

**投掷距离**：25-35米

#### 卧姿投掷

**动作要领**：
- 卧姿据枪姿势
- 投掷时起上身
- 用力投出后迅速卧倒

**投掷距离**：15-25米

### 6.3 投掷注意事项

1. 确认保险销已取出
2. 投掷后迅速隐蔽
3. 听到爆炸声后再行动
4. 不要投在硬物上（可能反弹）

## 七、地雷

### 7.1 地雷种类

| 类型 | 特点 | 用途 |
|------|------|------|
| 防步兵地雷 | 破片或爆破 | 杀伤人员 |
| 防坦克地雷 | 装药量大 | 破坏坦克 |
| 信号地雷 | 发光发响 | 报警 |

### 7.2 埋设方法

**选择位置**：
- 敌人必经之路
- 地形隐蔽处
- 不易发现处

**埋设步骤**：
1. 挖掘地雷坑
2. 放入地雷
3. 调整引信
4. 覆土伪装

**注意事项**：
- 记录埋设位置
- 设置标记
- 撤退时通知

### 7.3 排除方法

**探测方法**：
- 探雷器探测
- 目视观察
- 侦察兵报告

**排除要领**：
1. 标记位置
2. 小心挖掘
3. 判断类型
4. 安全排除

**安全要求**：
- 由专业人员排除
- 严禁擅自操作
- 保持安全距离

## 安全规范

### 射击安全

1. 严禁枪口对人
2. 不射击时打开保险
3. 装弹前检查枪膛
4. 射击后清点弹药

### 手榴弹安全

1. 未经许可不得使用
2. 检查保险销状态
3. 投掷后迅速隐蔽
4. 哑弹不得触碰
