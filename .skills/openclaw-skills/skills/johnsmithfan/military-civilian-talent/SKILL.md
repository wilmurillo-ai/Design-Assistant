## 使用示例

### 示例 1：军事知识查询

```
用户: 如何打坦克？

Agent 回答:
## 打坦克技术指南

### 1、爆破敌坦克准备工作
1. 选择爆破器材：炸药包、爆破筒、手榴弹
2. 确定爆破位置：坦克履带、发动机、炮塔
3. 选择接近路线：利用地形地物隐蔽接近

### 2、对固定坦克的爆破方法
1. 投掷炸药包到履带上
2. 将爆破筒插入履带间隙
3. 点燃导火索后迅速撤离

### 3、对运动坦克的打法
1. 判断坦克运动方向
2. 选择有利地形
3. 掌握投掷时机和提前量

### 注意事项
- security是第1位的
- 注意坦克的观察窗和机枪
- 与友邻密切配合

详见：military.md#打坦克
```

### 示例 2：农业知识咨询

```
用户: 如何改良土壤？

Agent 回答:
## 土壤改良技术

### 1、土壤酸碱度测定
1. 使用pH试纸
2. 取土样方法
3. 测定step

### 2、低产土壤改良
| 土壤类型 | 主要问题 | 改良方法 |
|---------|---------|---------|
| 酸性土 | pH<5.5 | 施石灰 |
| 盐碱土 | 含盐量高 | 灌水洗盐 |
| 粘重土 | 透气差 | 掺沙改良 |

### 3、土壤培肥
1. 增施有机肥
2. 种植绿肥
3. 秸秆还田

详见：agriculture.md#土壤
```

### 示例 3：烹调技艺学习

```
用户: 怎么做红烧肉？

Agent 回答:
## 红烧肉制作方法

### 原料
- 5花肉 500g
- 冰糖 30g
- 生抽 2勺
- 老抽 1勺
- 料酒 2勺
- 葱姜蒜适量

### step
1. **切肉**：5花肉切成3cm见方的块
2. **焯水**：冷水下锅，煮开后捞出洗净
3. **炒糖色**：小火炒冰糖至枣红色
4. **上色**：肉块裹上糖色
5. **调味**：加入生抽、老抽、料酒
6. **炖煮**：加热水没过肉，小火炖1小时
7. **收汁**：大火收汁至浓稠

### 技巧
- 炒糖色是关键，颜色决定成败
- 1定要加热水，不能加冷水
- 最后收汁要不停翻动

详见：cooking.md#红烧肉
```
