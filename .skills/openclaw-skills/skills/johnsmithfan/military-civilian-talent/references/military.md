# 军事知识指南

## 一、进攻战斗中的战士

战士在进攻战斗中，通常在班(组)内行动，以火力、手榴弹、刺刀和爆破器材打炸敌坦克，消灭敌步兵。

### 战士必须做到

1. **贯彻近战歼敌思想**
   - 发扬勇敢战斗、不怕牺牲的作风
   - 不怕疲劳、连续战斗

2. **灵活利用地形地物**
   - 防敌原子、化学武器和火力袭击
   - 迅速荫蔽地接近敌人

3. **勇猛冲击**
   - 与友邻战士密切协同
   - 坚决消灭敌人，完成战斗任务

---

## 二、利用地形地物

### 目的
利用地形地物的目的在于荫蔽身体、发扬火力，只有充分地发扬火力，消灭敌人，才能有效地保存自己。

### 地形地物的概念

- **地形**：地貌和地物的总称
  - 地貌：地面高低起伏的状态（山地、平原、凹地等）
  - 地物：地面上的固定物体（居民地、道路、土堆、江河、树木等）

### 利用地形地物的要求

1. **便于观察、射击、荫蔽身体**
2. **不妨碍班(组)长指挥、火器射击和邻兵动作**
3. **不要几个人拥挤在一起，以免增大伤亡**
4. **尽量避开独立、明显的物体和难以通行的地方**

### 特殊兵种注意事项

**火箭筒、单兵火箭手**：
- 应有良好的射界
- 在火箭弹飞行的路线上不得有草木等障碍物
- 筒(箭)后三十米内，不能有人，以免伤害自己

## 三、敌火下运动

### 运动原则
- 荫蔽迅速
- 利用有利地形
- 避开敌火密集区

### 运动方法

1. **跃进**
   - 快速短距离移动
   - 利用地形掩护

2. **匍匐前进**
   - 低姿势前进
   - 减少暴露面积

3. **滚动**
   - 利用地形起伏
   - 快速改变位置

## 四、打坦克

### 一、爆破敌坦克的准备工作

**选择爆破器材**：
- 炸药包
- 爆破筒
- 手榴弹
- 反坦克地雷

**确定爆破位置**：
- 履带
- 发动机
- 炮塔座圈

### 二、对固定坦克的爆破方法

1. **观察判断**
   - 确认坦克是否真的固定
   - 观察坦克乘员活动情况

2. **选择接近路线**
   - 利用地形地物隐蔽接近
   - 选择坦克观察死角

3. **实施爆破**
   - 将炸药包放在履带上
   - 将爆破筒插入履带间隙
   - 点燃导火索后迅速撤离

### 三、对运动坦克的打法

1. **判断坦克运动方向**
   - 观察履带痕迹
   - 判断速度

2. **选择有利地形**
   - 有掩护的位置
   - 便于撤离

3. **掌握投掷时机**
   - 计算提前量
   - 把握最佳时机

### 四、步兵班进攻战斗中打敌坦克

**战斗编组**：
- 反坦克组
- 火力掩护组
- 突击组

**武器器材携带**：
- 火箭筒
- 炸药包
- 爆破筒
- 手榴弹

### 五、苏美军主要装甲车辆

**主要强点**：
- 装甲防护
- 火力强大
- 机动性好

**主要弱点**：
- 观察死角大
- 近距离防御能力弱
- 对地雷敏感

**薄弱和要害部位**：
- 履带
- 发动机散热窗
- 炮塔座圈
- 观察设备

## 五、打飞机

### 一、同敌机作斗争的几种方法

1. **隐蔽伪装**
   - 利用地形
   - 设置假目标

2. **积极打击**
   - 轻武器射击
   - 高射机枪

3. **规避机动**
   - 快速转移
   - 利用地形

### 二、对敌机的识别

**判断飞机类型**：
- 战斗机
- 轰炸机
- 运输机
- 直升机

**识别特征**：
- 外形特征
- 声音特征
- 飞行姿态

### 三、判断飞机的距离

| 观察特征 | 大约距离 |
|---------|---------|
| 能看清机徽 | 500米以内 |
| 能看清机身 | 800-1000米 |
| 能看清轮廓 | 1500-2000米 |

### 四、打敌机要练的本领

1. **快速识别**
   - 判断敌我
   - 判断机型

2. **准确判断**
   - 距离
   - 高度
   - 速度

3. **把握时机**
   - 飞机俯冲时
   - 飞机转弯时
   - 直升机悬停时

### 五、打敌机的有利时机

- 飞机俯冲投弹时
- 飞机低空飞行时
- 直升机悬停时
- 飞机起飞降落时

### 六、打飞机时的提前量

**计算方法**：
- 根据飞机速度
- 根据子弹飞行时间
- 根据射击距离

**实践经验**：
- 对俯冲机：瞄准机头前方
- 对水平飞行：根据速度留提前量

## 六、防御战斗中的战士

### 做好防御准备

1. **构筑工事**
   - 散兵坑
   - 掩体
   - 交通壕

2. **设置障碍**
   - 铁丝网
   - 地雷
   - 陷阱

3. **组织火力**
   - 步枪
   - 机枪
   - 火箭筒

### 担任观察员时的动作

1. **选择观察位置**
   - 视野开阔
   - 隐蔽安全

2. **观察方法**
   - 分区观察
   - 重点观察
   - 循环观察

3. **报告情况**
   - 准确
   - 及时
   - 简明

### 防敌原子、化学和火力袭击

**防原子武器**：
- 发现闪光立即卧倒
- 利用地形掩护
- 保护暴露部位

**防化学武器**：
- 及时佩戴防毒面具
- 利用地形避风
- 快速通过染毒区

**防火力袭击**：
- 进入工事
- 保护头部
- 听从指挥

### 抗击敌坦克和步兵冲击

1. **远距离打击**
   - 火箭筒
   - 反坦克导弹

2. **近距离爆破**
   - 炸药包
   - 爆破筒

3. **消灭步兵**
   - 机枪
   - 步枪
   - 手榴弹
