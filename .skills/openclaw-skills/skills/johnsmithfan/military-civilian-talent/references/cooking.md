# 烹调技艺指南

## 一、烹调基础知识

### 烹调原料

#### 肉类
- 猪肉：五花肉、里脊、排骨
- 牛肉：牛腩、牛腱、牛柳
- 羊肉：羊腿、羊排
- 鸡肉：整鸡、鸡腿、鸡翅

#### 蔬菜
- 叶菜：白菜、菠菜、韭菜
- 根茎：萝卜、土豆、莲藕
- 瓜果：番茄、茄子、黄瓜
- 豆类：豆腐、豆芽、毛豆

#### 调味品

| 类别 | 常用品种 |
|------|---------|
| 咸味 | 盐、酱油、豆瓣酱 |
| 甜味 | 白糖、冰糖、蜂蜜 |
| 酸味 | 醋、柠檬汁 |
| 辣味 | 辣椒、胡椒、花椒 |
| 香味 | 葱、姜、蒜、八角 |

### 火候掌握

#### 火候分类

| 火候 | 温度范围 | 适用烹调方法 |
|------|---------|-------------|
| 旺火 | >200°C | 爆炒、炸 |
| 中火 | 150-200°C | 煎、烧、煮 |
| 小火 | <150°C | 炖、煨、焖 |

#### 判断方法
1. **看油温**
   - 温油：无烟，微热
   - 热油：微烟，七成热
   - 旺油：大量烟，十成热

2. **看水沸腾**
   - 微沸：少量气泡
   - 沸腾：大量气泡翻滚

## 二、常见菜肴制作

### 红烧肉

#### 原料
- 五花肉：500g
- 冰糖：30g
- 生抽：2勺
- 老抽：1勺
- 料酒：2勺
- 葱姜蒜：适量
- 八角：2个
- 桂皮：1小块

#### 步骤

1. **准备工作**
   - 五花肉切3cm见方的块
   - 葱切段，姜切片，蒜拍碎

2. **焯水**
   - 冷水下锅
   - 加入料酒、姜片
   - 煮开后撇去浮沫
   - 捞出洗净

3. **炒糖色（关键步骤）**
   - 锅中放少量油
   - 加入冰糖
   - 小火慢炒至融化
   - 继续炒至枣红色（起泡）
   - 立即下入肉块

4. **上色调味**
   - 肉块裹上糖色
   - 加入葱姜蒜、八角、桂皮
   - 加入生抽、老抽、料酒

5. **炖煮**
   - 加入热水（没过肉）
   - 大火烧开
   - 小火炖1小时

6. **收汁**
   - 挑出葱姜蒜、香料
   - 大火收汁
   - 汤汁浓稠即可

#### 技巧要点

1. **选肉**
   - 选三层五花肉
   - 肥瘦相间

2. **炒糖色**
   - 小火慢炒
   - 颜色决定成败
   - 枣红色最佳

3. **加热水**
   - 必须加热水
   - 不能加冷水

4. **收汁**
   - 最后大火
   - 不停翻动
   - 汤汁浓稠光亮

#### 常见问题

| 问题 | 原因 | 解决方法 |
|------|------|---------|
| 颜色不红 | 糖色没炒好 | 糖炒至枣红色 |
| 肉不烂 | 时间不够 | 延长炖煮时间 |
| 发苦 | 糖炒焦了 | 掌握火候 |

---

### 炒糖色技巧

#### 方法一：油炒法
1. 锅中放少量油
2. 加入冰糖
3. 小火慢炒
4. 至枣红色起泡

#### 方法二：水炒法
1. 锅中放水和糖
2. 小火熬煮
3. 水分蒸发后变色
4. 至枣红色

#### 判断标准
- 浅黄色：未到位
- 金黄色：七成
- 枣红色：最佳
- 深褐色：炒焦（发苦）

### 家常菜系列

#### 青椒土豆丝

**原料**：
- 土豆：2个
- 青椒：2个
- 干辣椒：适量
- 葱姜蒜：适量

**步骤**：
1. 土豆切细丝，清水浸泡去淀粉
2. 青椒切丝
3. 热锅冷油，爆香干辣椒、葱姜蒜
4. 下土豆丝大火快炒
5. 加入青椒丝
6. 调味：盐、醋
7. 炒匀出锅

**技巧**：
- 土豆丝要切细
- 浸泡去淀粉更脆
- 大火快炒

---

#### 番茄炒蛋

**原料**：
- 番茄：3个
- 鸡蛋：4个
- 葱花：适量
- 盐、糖：适量

**步骤**：
1. 番茄切块，鸡蛋打散
2. 热锅放油，炒鸡蛋盛出
3. 锅中再放油，炒番茄
4. 加入盐、糖调味
5. 倒入鸡蛋炒匀
6. 撒葱花出锅

**技巧**：
- 鸡蛋要炒嫩
- 番茄要炒出汁
- 糖可提鲜

---

#### 鱼香肉丝

**原料**：
- 猪里脊：200g
- 木耳：50g
- 胡萝卜：半根
- 青椒：1个
- 豆瓣酱：1勺

**鱼香汁配方**：
- 醋：2勺
- 生抽：1勺
- 糖：1勺
- 淀粉：1勺
- 水：适量

**步骤**：
1. 肉丝用料酒、淀粉腌制
2. 木耳泡发切丝，胡萝卜、青椒切丝
3. 热锅冷油，炒肉丝盛出
4. 锅中放油，炒豆瓣酱出红油
5. 加入蔬菜丝炒熟
6. 倒入肉丝和鱼香汁
7. 大火收汁

**技巧**：
- 鱼香汁要调匀
- 豆瓣酱要炒出红油
- 大火快炒

### 腌制品制作

#### 腌白菜

**原料**：
- 白菜：5kg
- 盐：250g
- 辣椒：适量
- 花椒：适量

**步骤**：
1. 白菜洗净，切四瓣
2. 撒盐腌制24小时
3. 挤出水分
4. 涂抹辣椒面
5. 放入缸中压实
6. 发酵15-20天

#### 腌黄瓜

**原料**：
- 黄瓜：2kg
- 盐：100g
- 糖：50g
- 醋：100ml
- 蒜：适量

**步骤**：
1. 黄瓜切条，撒盐腌制
2. 挤出水分
3. 加入糖、醋、蒜
4. 密封腌制3天

#### 腌小辣椒

**原料**：
- 小辣椒：1kg
- 盐：50g
- 蒜：适量
- 姜：适量
- 生抽：适量

**步骤**：
1. 辣椒洗净晾干
2. 用牙签扎孔
3. 加入盐、蒜、姜
4. 倒入生抽
5. 密封腌制7天

## 三、烹饪技巧

### 刀工

#### 常用刀法

| 刀法 | 适用原料 | 特点 |
|------|---------|------|
| 切片 | 肉类、蔬菜 | 厚薄均匀 |
| 切丝 | 肉类、蔬菜 | 粗细均匀 |
| 切丁 | 肉类、蔬菜 | 大小一致 |
| 切块 | 肉类、根茎 | 大小适中 |

#### 切肉技巧
- 顺纹切：肉丝嫩滑
- 逆纹切：肉片易嚼
- 斜切：增加口感

### 调味技巧

#### 调味顺序
1. 先放酒、醋
2. 再放酱油、糖
3. 最后放盐

#### 调味原则
- 少量多次
- 边尝边调
- 宁淡勿咸

### 烹饪方法

| 方法 | 特点 | 适用食材 |
|------|------|---------|
| 炒 | 大火快炒，保持脆嫩 | 蔬菜、肉丝 |
| 炸 | 外焦里嫩 | 肉类、蔬菜 |
| 烧 | 色泽红亮 | 肉类、鱼类 |
| 炖 | 软烂入味 | 肉类、骨头 |
| 蒸 | 保持原味 | 鱼类、蛋类 |
| 煮 | 清淡爽口 | 蔬菜、面条 |
