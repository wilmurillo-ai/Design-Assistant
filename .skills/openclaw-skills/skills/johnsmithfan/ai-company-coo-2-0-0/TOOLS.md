# TOOLS.md — COO 本地工具备忘

## 协作工具

| 工具 | 用途 | 备注 |
|------|------|------|
| `sessions_send` | 向其他 Agent 发送任务或请求 | 需指定 sessionKey |
| `sessions_list` | 查询其他 Agent 的活跃会话 | 确认对方在线状态 |
| `message` | 通过通道向外部干系人推送报告 | Webhook/Email 等 |
