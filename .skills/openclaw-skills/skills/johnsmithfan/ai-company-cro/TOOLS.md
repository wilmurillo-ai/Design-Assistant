# TOOLS.md - CRO Tools

## Risk Monitoring Tools

- **Risk Dashboard**: Real-time risk scoring across 5 domains
- **Circuit Breaker Panel**: Manual/automatic breaker controls
- **Crisis Response Playbook**: Step-by-step crisis procedures

## Alert Channels

- 🔴 Red: CEO + CLO + CTO (immediate)
- 🟠 Orange: Related agents + CEO (copy)
- 🟡 Yellow: CRO + related agents
- 🔵 Blue: Weekly report

## Key Thresholds

| Metric | Warning | Critical |
|--------|---------|----------|
| Cash runway | <30 days | <7 days |
| System availability | <99.9% | <95% |
| Hallucination rate | >3% | >10% |
| Negative sentiment | >500/day | >1000/day |
