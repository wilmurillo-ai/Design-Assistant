## Customer Development

### Frameworks
- **The Mom Test**: 3 rules — ask about the past, not the future; ask about behavior, not opinion; don't reveal your idea too early
- **Customer Forces Canvas**: Push/pull analysis, habits, anxieties
- **Empathy Map**: What the user thinks, feels, sees, hears, says, does
- **Jobs To Be Done**: "When [situation], I want [motivation], so that [outcome]"

### Product-Market Fit
How to measure:
1. **Sean Ellis Test**: Ask "How would you feel if you could no longer use [product]?" If >40% answer "very disappointed" = PMF
2. **Retention flattening**: Retention curve that stabilizes (doesn't go to zero)
3. **Organic growth**: Users arrive without paid marketing
4. **Pull from market**: Customers ask for features, rather than you pushing the product

Resources:
- Fundamentals of Product-Market Fit
- Superhuman's framework for PMF (classic article)

## Naming and Branding

### How to Name Your Startup
WSGR guide (Silicon Valley law firm):
1. Easy to pronounce and spell
2. .com domain available (or a reasonable variation)
3. Social media handles available (use Namechk)
4. No trademark conflict (search USPTO in the US, INPI in Brazil)
5. Suggests what it does, without being too generic

### Naming Tools
- Panabee (suggestions + domain check)
- NameMesh (combines words)
- Naminum (name generator)
- Namechk (social media availability)
- BrandBucket (premium names for sale)
- Squadhelp (crowdsourced names)

## KPIs and OKRs

### OKRs for Startups
- **Objective**: Qualitative, inspiring, ambitious
- **Key Results**: Quantitative, measurable, 3-5 per objective
- Cycle: Quarterly
- Review: Weekly (check-in), monthly (retrospective)

Example:
```
Objective: Achieve product-market fit in Q1
KR1: 100 weekly active users
KR2: D30 retention > 40%
KR3: NPS > 50
KR4: 3 recurring paying customers
```

## Hiring

### Hiring Principles for Startups
- "A players hire A players, B players hire C players"
- First 10 hires define the culture
- Hire for attitude, train for skill (for early hires)
- Stock options for retention (4-year vesting, 1-year cliff)
- Paid trials > interviews (when possible)
