# wechat-writer-kit

**公众号文章写作全流程助手（通用版）**

适用于任何公众号方向，支持多账号管理。你的账号画像和风格保存在本地 workspace，skill 本身不包含任何账号信息，下载即用。

---

## 功能

- 🎯 **账号画像管理**：引导填写账号定位、读者画像、内容风格，自动保存，随时更新
- 📰 **多账号支持**：可管理多个公众号，写作前一键切换
- 💡 **每日选题**：结合账号定位和当日热点，推荐 5 个选题
- ✍️ **文章创作**：按账号风格写作，内置反 AI 腔规范
- 📁 **自动存档**：文章存入企业微信文档，本地维护索引
- 🔄 **风格学习**：标记优质文章，自动提炼写作规律，越用越懂你

---

## 快速开始

安装后直接说：

```
帮我写公众号
```

或者：

```
今天写什么
```

首次使用时，skill 会引导你填写公众号画像（3个问题，一次回答）。

---

## 多账号使用

有多个公众号？没问题。

```
新建公众号     → 创建新账号画像
切换到育儿账号  → 切换当前写作账号
```

每次写作前，skill 会确认当前账号，或在有多个账号时询问你用哪个。

---

## 风格学习

写完文章觉得效果好？告诉我：

```
这篇文章写得好，学习一下
```

skill 会自动提炼这篇文章的写作亮点，追加到你的账号画像，下次写得更像你。

---

## 文件结构

安装后，skill 会在你的 workspace 中管理以下文件：

```
references/
├── profiles/
│   ├── account-健身.md    ← 你的账号画像（自动生成）
│   └── account-育儿.md    ← 多账号示例
├── active-account.md      ← 当前激活账号
├── article-index.md       ← 文章存档索引
├── writing-rules.md       ← 通用写作规范（可自行修改）
└── profile-template.md    ← 画像填写模板
```
