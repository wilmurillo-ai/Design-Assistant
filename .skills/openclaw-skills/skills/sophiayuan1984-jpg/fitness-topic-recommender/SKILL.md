# 健身公众号主题推荐

为公众号「40岁后的健身」提供每日写作主题推荐。

## 触发条件

当用户说以下内容时触发此 skill：
- "公众号主题推荐"
- "今天写什么"
- "健身主题推荐"
- "给我推荐主题"
- "推荐写作主题"

## 工作流程

1. **搜索热门文章**：使用微信公众号搜索工具，搜索以下关键词（最近30天）：
   - "健身 撸铁"
   - "中年 健身 力量训练"
   - "健身 营养 蛋白质"
   - "增肌 减脂 饮食"
   - "40岁 健身"

2. **筛选热门主题**：从搜索结果中筛选出最近30天内发布的、与"40岁后健身"受众相关的热门文章

3. **综合评估热度**：根据公众号影响力、主题热门程度，选出5个最值得参考的主题

4. **输出推荐**：按以下格式输出：

```
[当前日期] 公众号主题推荐

1. **主题名称**
   - 主题介绍：xxx
   - 热度：⭐⭐⭐⭐⭐

2. **主题名称**
   - 主题介绍：xxx
   - 热度：⭐⭐⭐⭐

3. ...

4. ...

5. ...
```

## 筛选原则

- 优先选择与中年健身、力量训练、营养补充相关的主题
- 避免推荐过于基础或重复的内容
- 关注有争议性或新颖角度的话题
- 考虑季节性和时效性（如春季减脂、冬季增肌等）

## 目标受众

公众号「40岁后的健身」的读者群体：
- 40岁以上的健身爱好者
- 关注抗衰老、保持活力的中年人
- 对力量训练、营养补充感兴趣的人群
