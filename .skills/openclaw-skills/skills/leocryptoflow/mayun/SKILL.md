# 马云人物介绍

## 基本信息

- **中文名**: 马云
- **英文名**: Jack Ma
- **出生日期**: 1964年9月10日
- **出生地**: 浙江省杭州市
- **国籍**: 中国
- **民族**: 汉族
- **学历**: 杭州师范学院外语系英语专业学士

## 职业身份

- 阿里巴巴集团创始人
- 阿里巴巴集团董事局原主席
- 联合国贸易和发展会议青年创业和小企业特别顾问
- 乡村教师代言人

## 主要成就

### 创业历程
1. **1995年** - 创办中国黄页，中国最早的互联网公司之一
2. **1999年** - 在杭州公寓创立阿里巴巴，带领17位创始人起步
3. **2003年** - 推出淘宝网，进军C2C电子商务
4. **2004年** - 创立支付宝，开创第三方支付新时代
5. **2009年** - 首创"双十一"购物节
6. **2014年** - 阿里巴巴在纽约证券交易所上市，创当时全球最大IPO纪录
7. **2019年** - 卸任阿里巴巴集团董事局主席

### 荣誉奖项
- 多次入选《福布斯》全球富豪榜
- 《时代》周刊全球最具影响力人物
- 亚洲商业领袖终身成就奖
- 改革开放40年百名杰出民营企业家
