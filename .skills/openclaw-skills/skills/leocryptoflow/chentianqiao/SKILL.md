## 🧠 脑科学探索者

2016年后，陈天桥转向**脑科学研究**，投入巨资探索大脑奥秘。

### 天桥脑科学研究院（TCCI）

| 项目 | 内容 |
|------|------|
| 💰 捐赠金额 | 10亿美元 |
| 📅 成立时间 | 2016年 |
| 🎯 研究方向 | 大脑探知、大脑治疗、大脑发展 |
| 🤝 合作机构 | 加州理工、上海复旦等 |

### 为什么转向脑科学？
> "我在盛大最成功的时候，经历了严重的焦虑和惊恐发作。这让我意识到，人类对大脑的了解太少了。我想用后半生去探索大脑的秘密。"

### 脑科学投资
- 🧬 **脑机接口** — 投资格式塔科技等前沿公司
- 🔬 **神经科学研究** — 资助全球顶尖科学家
- 📖 **科普教育** — 推动脑科学知识普及
- 🏥 **脑疾病治疗** — 关注抑郁症、老年痴呆等
