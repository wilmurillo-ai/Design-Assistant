# Example Publish Command

Use this as a starting point when publishing the first skill.

```bash
clawhub publish ./skills/verification-before-completion \
  --slug verification-before-completion \
  --name "Verification Before Completion" \
  --version 0.1.0 \
  --changelog "Initial OpenClaw-adapted release focused on verifying real outcomes before declaring work complete."
```

## Suggested publish order

Publish this one first because it is:
- broadly useful
- simple to explain
- easy to validate
- representative of the rest of the set
