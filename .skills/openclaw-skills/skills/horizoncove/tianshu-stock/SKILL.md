# 天枢股票分析系统 (TianShu Stock Analysis)

## 概述

**名称**: 天枢股票分析系统  
**版本**: 1.0  
**作者**: Sharon  
**功能**: AI驱动的智能股票诊断系统，基于七大哲学理论

---

## 理论基础

### 七大理论框架

| 理论 | 应用 |
|------|------|
| **场域论** | 资金流向、市场情绪、场域强度 |
| **分形论** | 多周期结构分析（日/周/月） |
| **连续论** | 趋势连续性判断 |
| **协同论** | 多维度共振分析 |
| **统一论** | 核心规律（价值、周期、均值回归） |
| **显现论** | 信号捕捉（买入/卖出信号） |
| **进化论** | 动态演化分析 |

---

## 诊断维度

### 多维度评分体系

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 基本面 | 30% | 营收增长、利润率、ROE、负债 |
| 技术面 | 25% | 趋势、均线、K线形态、量价关系 |
| 行业面 | 15% | 行业周期、景气度、竞争格局 |
| 消息面 | 10% | 政策、业绩公告、催化剂 |
| 资金面 | 10% | 主力资金流向、成交量、换手率 |
| 估值面 | 10% | PE、PB、PEG、相对估值 |

### 综合评分

| 分数区间 | 评级 |
|----------|------|
| 8.0-10.0 | 优秀 |
| 7.0-8.0 | 良好 |
| 6.0-7.0 | 中等 |
| 5.0-6.0 | 及格 |
| <5.0 | 风险 |

## 数据获取

### API配置

使用腾讯财经API作为主数据源：

```
URL: https://qt.gtimg.cn/q={股票代码}
示例: https://qt.gtimg.cn/q=sh603127
```

### 数据字段

- 当前价格、涨跌、涨幅
- 开盘、最高、最低价
- 成交量、成交额
- 总市值、市盈率、市净率
- 换手率
