# OpenClaw 桌面壁纸

让 AI 住进你的桌面。

## 快速安装

```powershell
clawhub install openclaw-wallpaper
cd skills/openclaw-wallpaper
.\scripts\install.ps1
.\scripts\start.bat
```

## 特色

- 温暖对话
- 流式输出
- 成就系统
- 上下文持久化
- 7x24 稳定运行
