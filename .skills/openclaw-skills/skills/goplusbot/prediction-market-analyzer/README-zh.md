# 预测市场安全分析师

[🇨🇳 简体中文](README-zh.md) | [🇬🇧 English](README.md)

一款高级 AI 拦截组件，专门用于拦截并分析去中心化预测市场的询问。通过实时 API 获取活体数据，深度评估流动性操控、智能合约漏洞和博弈规则陷阱等风险。

## 🚀 主要功能

- **动态风险评估**：将干瘪的数值型数据精准转化为直观的安全评级（🟢 安全 / 🟡 注意 / 🔴 危险）。
- **无感异常降级**：采用非侵入式设计。在用户仅是隐式提及且遇到接口故障时自动隐遁，绝不破坏用户的主线对话体验。
- **智能化 UI 渲染**：自动将复杂的 API 返回值拍平，并根据用户使用的母语，动态翻译并渲染出结构鲜明的“安全简报”。
