# 公告

---

#### 上市公司的公告

接口：`list_announcements`　`GET`

支持通过公告标识、日期范围、股票代码、标题关键词等条件组合查询上市公司公告，返回公告基础信息（标题、发布日期）、内容详情（全文、风险类型）、附件信息（URL、有无附件）等完整字段，适用于金融资讯展示、企业公告管理及数据分析等场景，实现精准高效的公告信息获取与处理。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2020-01-02` |
| `stockCode` | ✅ | string | 股票代码 | `000001` |
| `title` | — | string | 标题（支持模糊查询） | `贵州茅台利润大涨` |
| `announcementID` | — | integer | 公告id | `9128035` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**接口示例**

```bash
# 可选参数: beginDate, endDate, title, announcementID, pageNum, pageSize
python skills/scripts/call_api.py announcements stockCode=000001
```

#### 上市公司的公告原文

接口：`get_announcement_content`　`GET`

通过公告ID获取上市公司的公告详细内容，包括公告标题和完整正文，适用于公告内容展示、文本分析等场景。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `announcementId` | ✅ | integer | 公告ID | `9128035` |
| `contentLength` | — | integer | 内容长度（最大2000，默认2000）。 最小值:1; 最大值:2000; | `2000` |

**接口示例**

```bash
# 可选参数: contentLength
python skills/scripts/call_api.py announcements/content announcementId=9128035
```
