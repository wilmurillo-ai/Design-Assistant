####  echarts转图片Base64(标准)

接口：`generate_chart`　**`POST`**

将ECharts配置选项转换为Base64编码的图片，用于将动态图表数据快速生成为静态图片以便嵌入报告或分享。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `options` | ✅ | string |  | `{"xAxis":{"type":"category","data":["Mon","Tue","Wed","Thu","Fri","Sat","Sun"]},"yAxis":{"type":"value"},"series":[{"data":[120,200,150,80,70,110,130],"type":"bar"}]}` |

**接口示例**

```bash
python skills/scripts/call_api.py generate-chart --method POST options={"xAxis":{"type":"category","data":["Mon","Tue","Wed","Thu","Fri","Sat","Sun"]},"yAxis":{"type":"value"},"series":[{"data":[120,200,150,80,70,110,130],"type":"bar"}]}
```

#### echarts转图片Base64(组件)

接口：`generate_nex_chart`　**`POST`**

根据提供的ECharts图表配置选项（options）和唯一标识符（id），生成对应的图表图片，并以Base64编码的字符串形式返回，便于大模型在需要可视化数据时快速生成和嵌入图表。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `id` | ✅ | string |  | `nex-line-bar` |
| `options` | ✅ | string |  | ` {"xAxis":{"type":"category","data":["Mon","Tue","Wed","Thu","Fri","Sat","Sun"]},"yAxis":{"type":"value"},"series":[{"data":[120,200,150,80,70,110,130],"type":"bar"}]}` |

**接口示例**

```bash
python skills/scripts/call_api.py generate-nex-chart --method POST id=nex-line-bar options= {"xAxis":{"type":"category","data":["Mon","Tue","Wed","Thu","Fri","Sat","Sun"]},"yAxis":{"type":"value"},"series":[{"data":[120,200,150,80,70,110,130],"type":"bar"}]}
```
