#### 业绩报告披露时间表

接口：`list_stk _report_schema`　`GET`

通过股票代码和日期范围查询沪深京股票的定期报告（如年报、季报）预披露日程，返回包括股票代码、名称、预计披露日、报告期、实际披露日、首次发布日期和最新预计披露日等关键日程信息，帮助用户跟踪上市公司财报发布计划。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `002594` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**接口示例**

```bash
# 可选参数: beginDate, endDate, pageNum, pageSize
python skills/scripts/call_api.py stock/report-schema stockCode=002594
```
