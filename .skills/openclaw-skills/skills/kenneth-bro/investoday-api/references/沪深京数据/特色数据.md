#### 股票盈利能力的最新指标

接口：`get_stock_finance_profit_ability`　`GET`

通过股票代码获取公司盈利能力的最新指标，包括毛利率、营业利润率、净利率、股本回报率(ROE)、资产回报率(ROA)、资本回报率、杜邦分析指标等核心财务数据，并提供各项指标的当前排名和历史排名（0-100），便于大模型进行财务分析和投资决策。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | — |

**接口示例**

```bash
python skills/scripts/call_api.py stock/finance/profit-ability stockCode=<stockCode>
```

#### 行情技术指标(数据挖掘)

接口：`list_stock_market_ind_dm`　`GET`

根据股票代码和日期范围（格式为yyyy-MM-dd）查询指定沪深京股票在特定时间段内的行情技术指标数据，包括成交量与开盘价相关性、成交量与收盘价排序、开盘价收益率排序、最高价与成交量相关性排序、收盘价收益率、收盘价波动性排序、价量多重排序、最高价波动性排序、最高价与成交量排名相关性、收盘价市值中性化因子、收盘位置因子、迭代收盘相对位置因子以及市值排名迭代收益率因子等核心量化因子，用于股票量化分析和策略研究。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `002594` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**接口示例**

```bash
# 可选参数: beginDate, endDate, pageNum, pageSize
python skills/scripts/call_api.py stock/market-technical-indicators stockCode=002594
```
