# 沪深京数据 / 基础信息

---

#### 股票基本信息

接口：`get_stock_basic_info`　`GET`

通过股票代码查询股票的基本信息，包括股票名称、行业代码、发行日期、上市日期、证券标识、上市状态、板块代码、退市日期、外资行业分类、交易所行业分类、今日投资行业分类、申万行业分类、中信行业分类、通达信行业分类等多级行业分类信息，适用于股票信息查询和行业分析。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `000001` |

**接口示例**

```bash
python skills/scripts/call_api.py stock/basic-info stockCode=000001
```

#### 证券特别提示

接口：`list_stock_special_notice`　`GET`

通过股票代码和日期范围查询沪深京股票的特别提示信息，包括股票名称、上市状态、特别处理状态、变动原因、相关日期等核心公告数据，用于监控股票重大事件和风险提示。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `000001` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**接口示例**

```bash
# 可选参数: beginDate, endDate, pageNum, pageSize
python skills/scripts/call_api.py stock/special-notices stockCode=000001
```

#### 全市场股票列表

接口：`list_stock_all`　`GET`

获取全市场A股和B股股票的基本信息列表，包含股票代码、名称、所属交易所、上市板块、上市状态、上市日期、退市日期、股票类型、公司内部ID及公司成长阶段等核心数据，支持分页查询，适用于需要批量获取股票基础信息进行筛选、分析或构建股票池的场景。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**接口示例**

```bash
# 可选参数: pageNum, pageSize
python skills/scripts/call_api.py stock/all
```
