# 大模型语料

#### 上市公司互动问答

接口：`list_stk_consultations`　`GET`

根据股票代码和日期范围查询上市公司在互动平台上的问答记录，包括咨询类型、提问人、问题详情、回复内容及传闻核实情况，支持分页获取，便于分析公司沟通与市场传闻。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `002594` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**接口示例**

```bash
# 可选参数: beginDate, endDate, pageNum, pageSize
python skills/scripts/call_api.py stock/consultations stockCode=002594
```

#### 上市公司经营评述

接口：`list_stock_op_reviews`　`GET`

根据股票代码和报告期截止日期范围查询上市公司的经营述评信息，返回包括股票代码、股票名称、发布日期、报告期截止日期和经营述评内容等核心数据，用于分析公司经营状况和业绩展望。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `002594` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**接口示例**

```bash
# 可选参数: beginDate, endDate, pageNum, pageSize
python skills/scripts/call_api.py stock/operating-reviews stockCode=002594
```
