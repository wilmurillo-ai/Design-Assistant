# 指数 / 基础行情

---

#### 指数历史日行情

接口：`list_index_quotes`　`GET`

通过指数代码和日期范围查询指数的历史日行情数据，包括开盘价、最高价、最低价、收盘价、成交量、成交金额等核心指标，适用于市场分析和指数表现追踪。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `indexCode` | ✅ | string | 指数代码 | `000300` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2025-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**接口示例**

```bash
# 可选参数: beginDate, endDate, pageNum, pageSize
python skills/scripts/call_api.py index/quotes indexCode=000300
```

#### 指数最新实时日行情

接口：`get_index_realtime_quotes`　**`POST`**

使用指数代码获取指定指数的最新实时行情数据，包含指数当前价格、涨跌幅、开盘价、最高价、最低价、成交量、成交金额等核心数据，便于大模型进行即时的市场监测和指数分析。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `indexCodes` | ✅ | array | 指数代码 | `['000001', '399006']` |

**接口示例**

```bash
python skills/scripts/call_api.py index-quote/realtime --method POST indexCodes=000001 indexCodes=399006
```
