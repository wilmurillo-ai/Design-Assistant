# 指数 / 行情衍生数据

---

#### 指数区间涨幅

接口：`get_index_range_gains`　`GET`

通过指数代码查询指定指数的多时间区间涨跌幅数据，返回包括指数代码、名称以及近一日、近一周、近一月、近三月、近半年、近一年和今年以来等多个维度的涨跌幅百分比，用于分析指数的短期、中期和长期表现趋势。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `indexCode` | ✅ | string | 指数代码 | `000300` |

**接口示例**

```bash
python skills/scripts/call_api.py index/range-gains indexCode=000300
```
