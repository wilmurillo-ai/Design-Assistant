# 指数 / 指数资料

---

#### 指数基本信息

接口：`get_index_basic_info`　**`POST`**

通过指数代码（单个或批量）查询指数的基本信息，包括指数名称、全称、交易市场、发布机构、编制机构、基期、基点、启用日期、停用日期、加权方式、调整频率、指数简介、币种、发布状态和成分证券市场等核心元数据，适用于获取指数的元数据和分类信息。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `indexCode` | — | string | 指数代码【与indexCodes组成多选一参数，必须且只能传递其中一个】 | `399300` |
| `indexCodes` | — | array | 指数代码列表【与indexCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000300']` |

**接口示例**

```bash
# 可选参数: indexCode, indexCodes
python skills/scripts/call_api.py index/basic-info --method POST
```
