# 板块 / 衍生行情

---

#### 行业及关联成分股的最新实时日行情

接口：`get_industry_stock_realtime_quote`　`GET`

输入行业代码，获取该行业的最新整体行情（包括指数、涨跌幅、成交量、总市值、成分股数量、涨跌幅排名、领涨股等）以及所有关联成分股的实时行情明细（包括股票代码、名称、开盘价、当前价、涨跌幅、最高/最低价等），用于全面分析行业及其成分股的最新市场表现。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `industryCode` | ✅ | string | 行业代码 | `330000` |

**接口示例**

```bash
python skills/scripts/call_api.py industry-quote/stock-realtime industryCode=330000
```

#### 概念及关联成分股的最新实时日行情

接口：`get_concept_stock_realtime_quote`　`GET`

获取指定概念板块的最新实时行情，包括概念名称、涨跌幅、一周涨跌幅、总市值、涨跌幅排名、成分股数量、涨停/上涨/下跌/平盘股数、领涨股信息等，以及该概念下所有成分股的实时行情（如股票代码、名称、市场、开盘价、当前价、涨跌幅、最高/最低价等），用于快速洞察概念整体及其成分股的最新市场表现。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `conceptType` | ✅ | string | 概念类型(jy-聚源、cls-财联社)" | `jy` |
| `conceptCode` | ✅ | string | 概念代码 | `14060061` |

**接口示例**

```bash
python skills/scripts/call_api.py concept-quote/stock-realtime conceptType=jy conceptCode=14060061
```

#### 行业换手率

接口：`list_idu_turnover_rates`　**`POST`**

通过行业代码和日期范围查询指定行业的历史换手率数据，返回行业代码、名称、日期、换手率及其近一年百分位，适用于分析行业流动性及市场活跃度。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `industryCode` | ✅ | string | 行业代码 | `640000` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**接口示例**

```bash
# 可选参数: beginDate, endDate, pageNum, pageSize
python skills/scripts/call_api.py industry/turnover-rates --method POST industryCode=640000
```

#### 行业行情统计

接口：`get_industry_market_stats`　**`POST`**

通过行业代码（可选）查询指定行业（默认为全部行业）的行情统计数据，包括行业代码、名称、近一日至近五年等多个时间维度的涨跌幅、近一日至近二十日的主力资金净流入（万元），以及PE、PB、PS的近五年历史百分位，用于行业表现分析和资金流向监控。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `industryCode` | ✅ | string | 行业代码 | `640000` |

**接口示例**

```bash
python skills/scripts/call_api.py industry/market-stats --method POST industryCode=640000
```
