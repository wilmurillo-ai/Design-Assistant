#### 股票所属概念

接口：`list_stock_concept_classifications`　`GET`

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `000001` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2025-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**接口示例**

```bash
# 可选参数: beginDate, endDate, pageNum, pageSize
python skills/scripts/call_api.py stock/concept-classifications stockCode=000001
```

#### 行业列表

接口：`list_industries`　`GET`

通过行业名称（支持模糊匹配）或行业类型（指定行业体系）查询行业列表，返回行业代码、名称、类型、等级及对应的行业指数代码，便于构建行业分类体系或进行行业分析。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `industryCode` | — | string | 行业代码 | `740000` |
| `industryName` | — | string | 行业名称(模糊匹配) | `煤炭` |
| `industryType` | — | string | 行业类型(指定行业体系) | `INDUS4_CL` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**接口示例**

```bash
# 可选参数: industryCode, industryName, industryType, pageNum, pageSize
python skills/scripts/call_api.py industries
```
