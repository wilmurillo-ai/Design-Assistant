# 基金 / 基金投资组合

---

#### 基金的持仓股票

接口：`list_fund_holdings`　**`POST`**

通过基金代码查询基金的持仓股票信息，包括持仓股票代码、名称、数量、市值、占基金净值比例等核心数据，适用于分析基金投资组合和股票持仓情况。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `000006` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000006']` |

**接口示例**

```bash
# 可选参数: fundCode, fundCodes
python skills/scripts/call_api.py fund/portfolio-stock-holdings --method POST
```

#### 行业持仓的基金列表

接口：`list_industry_hold_fund`　**`POST`**

通过行业代码查询持有该行业股票的基金列表，返回基金代码、名称、类型及行业持仓比例等核心数据，适用于分析基金行业配置及投资策略。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `industryCode` | — | string | 行业代码【与industryCodes组成多选一参数，必须且只能传递其中一个】 | `370000` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |
| `industryCodes` | — | array | 行业代码(批量)【与industryCode组成多选一参数，必须且只能传递其中一个】 | `['370000', '74000']` |

**接口示例**

```bash
# 可选参数: industryCode, pageNum, pageSize, industryCodes
python skills/scripts/call_api.py fund/industry-hold-fund --method POST
```

#### 概念持仓的基金列表

接口：`list_concept_hold_fund`　**`POST`**

通过概念代码查询持有该概念的基金列表，返回基金代码、名称、类型及概念相关信息，包括概念代码、名称、类别以及基金在不同报告期（最新报告、季报、年报或半年报）中的概念持有比例，适用于分析基金对特定概念的持仓情况。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `conceptCode` | — | string | 概念代码 | `CLS82591` |
| `conceptCodes` | — | array |  | `['CLS80457', 'CLS82591']` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**接口示例**

```bash
# 可选参数: conceptCode, conceptCodes, pageNum, pageSize
python skills/scripts/call_api.py fund/concept-hold-fund --method POST
```
