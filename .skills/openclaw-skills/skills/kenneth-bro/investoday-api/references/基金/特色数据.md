# 基金 / 特色数据

---

#### 概念持仓的基金列表（批量）

接口：`list_concept_hold_fund_bh`　**`POST`**

根据概念代码、匹配类型和阈值查询持有该概念的基金列表，返回基金代码、名称、类型、概念代码、名称、类别以及基于不同报告期的持有比例和结束日期，适用于筛选特定概念持仓比例的基金。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `conceptCodes` | ✅ | array | 概念代码 | `['CLS82591','CLS82582']` |
| `matchType` | ✅ | integer | 匹配类型 | `1` |
| `threshold` | ✅ | integer | 阈值 | `40` |
| `isETF` | — | boolean | 是否ETF | `True` |

**接口示例**

```bash
# 可选参数: isETF
python skills/scripts/call_api.py fund/concept-hold-fund/batch --method POST conceptCodes=CLS82591 matchType=1 threshold=40
```

#### 行业持仓的基金列表（批量）

接口：`list_industry_hold_fund_b`　**`POST`**

通过行业代码批量查询持有该行业的基金列表，返回基金代码、名称、类型及行业持仓比例等核心数据，适用于分析基金行业配置及投资组合研究。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `industryCodes` | ✅ | array | 行业代码 | `['640000', '740000']` |
| `matchType` | ✅ | integer | 匹配类型 | `1` |
| `threshold` | ✅ | number | 阈值 | `40.0` |
| `isETF` | — | boolean | 是否ETF | `True` |

**接口示例**

```bash
# 可选参数: isETF
python skills/scripts/call_api.py fund/industry-hold-fund/batch --method POST industryCodes=640000 industryCodes=740000 matchType=1 threshold=40.0
```

#### 基金持仓股票及行业涨幅

接口：`list_fund_holdings_perf`　**`POST`**

根据基金代码查询其持仓股票及对应行业信息，包括基金代码、名称、持仓发布日期、截止日期、股票代码、名称、股票今年以来的涨幅、股票所属行业代码、名称以及行业今年以来的涨幅，并按持仓排名排序，适用于分析基金持仓结构及股票与行业的关联表现。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `002594` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000006']` |

**接口示例**

```bash
# 可选参数: fundCode, fundCodes
python skills/scripts/call_api.py fund/holdings-stocks-industries --method POST
```
