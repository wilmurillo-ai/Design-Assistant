#### 基金的现任基金经理及回报

接口：`list_fund_cur_mgr_rets`　**`POST`**

通过基金代码查询基金的现任基金经理信息，包括基金经理ID、姓名、任职开始时间、任期回报、擅长类型、擅长行情、获奖情况、从业年限、研究领域及从业平均年化表现等核心数据，适用于基金管理和投资分析场景。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `000300` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000006']` |

**接口示例**

```bash
# 可选参数: fundCode, fundCodes
python skills/scripts/call_api.py fund/current-manager-returns --method POST
```
