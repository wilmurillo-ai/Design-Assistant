# 基金 / 基金持有人

---

#### 基金持有人结构信息

接口：`list_fund_hold_structures`　**`POST`**

通过基金代码和日期范围查询基金的持有人结构信息，包括持有人户数、户均持有份额、机构、个人、其他投资者及基金从业人员的持有份额与占比，以及报告日期、信息发布来源等，用于分析基金的投资人构成和集中度。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `000001` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000006']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd） | `2025-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**接口示例**

```bash
# 可选参数: fundCode, fundCodes, beginDate, endDate, pageNum, pageSize
python skills/scripts/call_api.py fund/holder-structures --method POST
```

#### 基金份额变动

接口：`list_fund_shares`　**`POST`**

通过基金代码和日期范围查询基金份额变动明细，返回包括基金基本信息、变动期间、期初期末份额、以及申购、赎回、转入、转出、红利再投资、拆分、扩募、折算等各类变动金额的详细数据，适用于分析基金规模变化、资金流动和投资者行为。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `000001` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000006']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd） | `2025-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**接口示例**

```bash
# 可选参数: fundCode, fundCodes, beginDate, endDate, pageNum, pageSize
python skills/scripts/call_api.py fund/shares-changes --method POST
```
