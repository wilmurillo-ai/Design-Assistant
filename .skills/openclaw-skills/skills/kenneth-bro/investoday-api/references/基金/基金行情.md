# 基金 / 基金行情

---

#### 基金未复权日行情

接口：`list_fund_daily_quotes`　**`POST`**

根据基金代码和指定的日期范围（格式为 yyyy-MM-dd HH:mm:ss）查询基金的历史日行情数据，返回包括基金代码、名称、交易日期、开盘价、最高价、最低价、收盘价、成交量、成交金额等核心指标，支持分页查询，适用于分析基金的历史价格走势和交易活跃度。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `000001` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000006']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd） | `2025-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**接口示例**

```bash
# 可选参数: fundCode, fundCodes, beginDate, endDate, pageNum, pageSize
python skills/scripts/call_api.py fund/daily-quotes --method POST
```

#### ETF最新实时日行情

接口：`get_fund_quote_realtime`　`GET`

通过基金代码获取ETF基金的最新实时行情数据，包括基金代码、名称、市场类型、开盘价、昨日收盘价、当前价格、涨跌幅、最高价、最低价、数据时间和系统时间等核心数据，适用于实时监控ETF基金的市场表现和价格变动。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | ✅ | string | 基金代码 | `159001` |

**接口示例**

```bash
python skills/scripts/call_api.py fund-quote/realtime fundCode=159001
```

#### 基金技术指标

接口：`list_fund_tech_indicators`　**`POST`**

通过基金代码和日期范围查询指定基金的技术指标数据，包括压力位和支撑位，用于基金的技术分析和投资决策。入参包括基金代码、开始日期、结束日期、页码和页大小；出参包括基金代码、基金ID、基金名称、交易日期、压力位和支撑位。日期格式为yyyy-MM-dd。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `000001` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001','000006']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd） | `2025-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**接口示例**

```bash
# 可选参数: fundCode, fundCodes, beginDate, endDate, pageNum, pageSize
python skills/scripts/call_api.py fund/technical-indicators --method POST
```
