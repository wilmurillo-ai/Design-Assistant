# 基金 / 基金财务数据

---

#### 基金主要财务指标

接口：`list_fund_fin_inds`　**`POST`**

通过基金代码和报告期截止日期范围（格式为yyyy-MM-dd HH:mm:ss），查询指定基金在特定期间内的主要财务指标，包括基金基本信息、报告类型、发布及截止日期、以及本期净值增长率、加权平均净值收益率、可分配基金收益、期末单位基金资产净值、本期基金净值收益率、扣公允变动后本期利润、期末基金资产净值、基金累计净值收益率、本期利润、本期公允价值变动损益、加权平均份额本期利润、加权平均份额本期利润率、期末份额累计净值、期末基金资产净值合计等核心财务数据，用于基金财务分析和绩效评估。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `000001` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000006']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd） | `2025-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**接口示例**

```bash
# 可选参数: fundCode, fundCodes, beginDate, endDate, pageNum, pageSize
python skills/scripts/call_api.py fund/financial-indicators --method POST
```

#### 基金主要财务指标(季度）

接口：`list_fund_fin_inds_q`　**`POST`**

根据基金代码和报告期截止日期范围（格式为yyyy-MM-dd HH:mm:ss）查询指定基金的季度主要财务指标，包括基金基本信息、报告类型（Q1/Q2/Q3/Q4）、发布与报告日期、是否合并披露，以及本期净值增长率、加权平均净值收益率、可供分配基金收益、期末单位基金资产净值、本期基金净值收益率、扣公允价值变动后本期利润、期末基金资产净值、基金累计净值收益率、本期利润、本期公允价值变动损益、加权平均份额本期利润、加权平均份额本期利润率、期末份额累计净值、期末基金资产净值合计等核心财务数据，用于分析基金的季度财务状况和盈利能力。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `000001` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000006']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd） | `2025-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**接口示例**

```bash
# 可选参数: fundCode, fundCodes, beginDate, endDate, pageNum, pageSize
python skills/scripts/call_api.py fund/financial-indicators-q --method POST
```
