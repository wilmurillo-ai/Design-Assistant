# 港股 / 基础数据

---

#### 港股基本信息

接口：`get_hk_stock_basic_info`　`GET`

通过股票代码查询香港股票的基本信息，包括股票代码、名称（中文全称、简称、英文简称）、上市状态、上市板块、交易货币、买卖单位、上市日期、摘牌日期、公司代码、交易市场编码以及是否为港股通标的等核心数据，便于大模型进行股票基本信息的查询和分析。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `00001` |

**接口示例**

```bash
python skills/scripts/call_api.py hk-stock/basic-info stockCode=00001
```

#### 港股简称变更

接口：`get_hk_stock_name_change`　`GET`

通过港股股票代码查询其公司名称变更历史，返回包括变更前后的中英文简称、公告日期和生效日期等关键信息，便于大模型进行公司历史沿革分析或数据核对。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `00001` |

**接口示例**

```bash
python skills/scripts/call_api.py hk-stock/name-change stockCode=00001
```

#### 港股股票转板

接口：`get_hk_stock_transfers`　`GET`

通过股票代码查询港股股票转板信息，返回包括转板前后的股票代码、股票名称、公告日期、主板上市日期和创业板最后交易日等关键数据，用于追踪港股市场板块变动情况。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `00001` |

**接口示例**

```bash
python skills/scripts/call_api.py hk-stock/transfers stockCode=00001
```
