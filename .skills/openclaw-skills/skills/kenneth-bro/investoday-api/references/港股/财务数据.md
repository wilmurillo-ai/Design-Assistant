# 港股 / 财务数据

---

#### 港股现金流量表

接口：`list_hk_stock_cash_flows`　`GET`

通过股票代码、报告期起止日期（格式为yyyy-MM-dd）等条件，查询港股上市公司最新发布的现金流量表数据，包括经营活动、投资活动、融资活动产生的现金流量，以及现金及现金等价物的期初、期末余额等核心财务指标，用于分析公司的现金创造能力和财务健康状况。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `002594` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-03-30` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**接口示例**

```bash
# 可选参数: beginDate, endDate, pageNum, pageSize
python skills/scripts/call_api.py hk-stock/cash-flows stockCode=002594
```
