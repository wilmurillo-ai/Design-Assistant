# 港股 / 公司行为

---

#### 公司分红派息信息

接口：`list_hk_stock_dividends`　`GET`

通过股票代码查询香港上市公司的分红派息信息，包括每股派息金额、派息日、除净日、货币单位等核心数据，支持按公告日期范围筛选，便于分析公司的分红政策和历史记录。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `00001` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**接口示例**

```bash
# 可选参数: beginDate, endDate, pageNum, pageSize
python skills/scripts/call_api.py hk-stock/dividends stockCode=00001
```

#### 港股股本变动

接口：`list_hk_stock_cap_changes`　`GET`

通过股票代码、起始日期和结束日期查询港股公司的股本变动历史记录，返回包括股票代码、名称、变动日期、各类别股份数量（如优先股、普通股、H股、大陆股等）及总股本等详细信息，适用于分析公司资本结构变化和股权历史。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `00001` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | — |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**接口示例**

```bash
# 可选参数: beginDate, endDate, pageNum, pageSize
python skills/scripts/call_api.py hk-stock/cap_changes stockCode=00001
```
