# 新闻与观点 / 基础数据

---

#### 新闻与资讯

接口：`list_news`　`GET`

通过时间范围、标题关键词、情绪倾向、新闻类型、新闻等级、股票代码、行业代码或概念代码等多维度条件筛选新闻资讯，返回包括新闻ID、发布时间、类型、标题、摘要、关键点、影响分析、投资机会与风险、情绪得分与分析、主题代码及新闻等级等核心信息，适用于市场监测、投资决策和舆情分析等场景。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `beginTime` | — | string | 开始时间（格式yyyy-MM-dd HH:mm:ss）。 最小值:2020-01-01 00:00:00; | `2020-01-01 08:00:00` |
| `endTime` | — | string | 结束时间（格式yyyy-MM-dd HH:mm:ss） | `2025-02-01 08:00:00` |
| `title` | — | string | 标题 | `人工智能` |
| `sentiment` | — | integer | 情绪偏向 | `1` |
| `newsType` | — | integer | 新闻类型 | `1` |
| `newsLevel` | — | integer | 新闻等级 | `1` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**接口示例**

```bash
# 可选参数: beginTime, endTime, title, sentiment, newsType, newsLevel, pageNum, pageSize
python skills/scripts/call_api.py news
```

#### 实体的相关新闻

接口：`list_entity_related_news`　`GET`

通过股票代码、行业代码或概念代码查询相关实体的新闻数据，支持按时间范围、新闻类型、新闻等级、情感倾向等多条件筛选，返回新闻标题、摘要、关键点、影响分析、投资机会与风险、情感得分及分析等核心信息，适用于市场监测、投资分析和舆情跟踪。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `beginTime` | — | string | 开始日期（格式yyyy-MM-dd HH:mm:ss）。 最小值:2020-01-01 00:00:00; | `2020-01-01 08:00:00` |
| `endTime` | — | string | 结束时间（格式yyyy-MM-dd HH:mm:ss） | `2025-02-01 08:00:00` |
| `title` | — | string | 标题 | `人工智能` |
| `newsType` | — | integer | 新闻类型 | `1` |
| `newsLevel` | — | integer | 新闻等级 | `1` |
| `sentiment` | — | integer | 情绪偏向 | `1` |
| `stockCode` | — | string | 股票代码 | `600839` |
| `industryCode` | — | string | 行业代码 | `370000` |
| `conceptCode` | — | string | 概念代码 | `CLS81936` |
| `minRelevance` | — | integer | 实体的相关度下限。 最小值:1; 最大值:5; | `1` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**接口示例**

```bash
# 可选参数: beginTime, endTime, title, newsType, newsLevel, sentiment, stockCode, industryCode, conceptCode, minRelevance, pageNum, pageSize
python skills/scripts/call_api.py news/entity-related
```
