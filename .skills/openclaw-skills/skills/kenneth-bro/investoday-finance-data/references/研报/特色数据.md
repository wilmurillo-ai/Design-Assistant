# 研报 / 特色数据

---

## 研报舆情

接口路径：`research/sentiment`
请求方式：**`POST`**
tool_id：`list_research_sentiment`

接口说明：通过研报ID、标题模糊查询、股票代码、行业代码、概念代码、发布时间范围及情绪倾向等多条件组合筛选，获取研报的衍生信息，包括核心内容、分析观点、投资机会与风险、情绪分析等关键数据，适用于投资决策支持和市场情绪分析。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `guid` | — | string | 报告id | `0236EC2B-2416-455E-BF82-E4ED96E91DB0` |
| `title` | — | string | 标题(模糊查询) | `宏观` |
| `sentiment` | — | integer | 情绪偏向 | `1` |
| `beginTime` | — | string | 发布开始时间（格式:yyyy-MM-dd HH:mm:ss）。 最小值:2020-01-01; | `2020-01-01 08:00:00` |
| `endTime` | — | string | 发布截止时间（格式:yyyy-MM-dd HH:mm:ss） | `2025-01-02 08:00:00` |
| `stockCode` | — | string | 股票代码 | `000001` |
| `industryCode` | — | string | 行业代码 | `370000` |
| `conceptCode` | — | string | 概念代码 | `CLS81936` |
| `institutionCode` | — | integer | 机构代码 | `142` |
| `minRelevance` | — | integer | 实体的相关度下限。 最小值:1; 最大值:6; | `1` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `date` | 发布时间 | `2025-02-19 08:15:00` |
| `guid` | 报告id | `07DFC96B-2399-486C-A1D7-999562CD8695` |
| `title` | 标题 | `小米集团2024Q4前瞻：销量亮眼，看好2025年增长` |
| `coreContent` | 核心内容 | `小米手机2024Q4出货4270万台，同比增4.8%，市占率12.9%居全球前三。IoT业务有望随家电补贴政策增长，汽车2024年交付13.5万台，2025年目标30万台。预计2024 - 2026年收入、消费电子主业及汽车经调整净利有不同表现，重申 买入 评级。` |
| `analysisViewpoint` | 分析观点 | `1. **手机业务表现佳**：2024Q4出货4270万台，同比增长4.8%，市占率12.9%，稳居全球前三，展现了较强的市场竞争力。2. **IoT业务有望增长**：2024家电以旧换新补贴政策见成效，促使IoT业务有望快速增长。3. **汽车业务目标明确**：2024年首年交付13.5万台，2025年目标交付30万台，显示出积极的发展态势。4. **财务预期良好**：预计2024 - 2026年收入、消费电子主业经调整净利增长，汽车经调整净利逐步改善，重申 买入 评级。` |
| `investmentOpportunity` | 投资机会 | `- **小米集团投资机会**：小米手机销量增长、IoT业务有望提升、汽车业务发展前景良好，预计财务表现佳，重申 买入 评级，具有投资价值。` |
| `investmentRisk` | 投资风险 | `- **手机竞争风险**：手机竞争超预期或影响业绩。- **IoT进展风险**：IoT进展不及预期可能拖累业务。- **智能车竞争风险**：智能车竞争超预期会带来不利影响。` |
| `comScore` | 综合得分(值范围：-5～5) | `3.3` |
| `sentiment` | 情绪偏向 | `5` |
| `keyReason` | 关键事由 | ` 小米集团 2024Q4业绩 小米手机销量 小米汽车交付目标 ` |
| `sentimentAnalysis` | 情绪分析 | `小米24Q4业绩优，各业务前景好，虽有风险提示但整体利好。` |
| `analysisFramework` | 分析框架 | `分析小米2024Q4手机出货及市占率情况→考量IoT业务受补贴政策影响→关注小米汽车交付情况及目标→预测2024 - 2026年公司收入、消费电子主业和汽车经调整净利→得出重申买入评级结论` |
| `stockName` | 股票名称 | `平安银行` |
| `stockCode` | 股票代码 | `000001` |
| `minRelevance` | 关联度(范围1~6) | `1` |

**接口示例**

```bash
# 可选参数: guid, title, sentiment, beginTime, endTime, stockCode, industryCode, conceptCode, institutionCode, minRelevance, pageNum, pageSize
investoday-api research/sentiment --method POST
```
