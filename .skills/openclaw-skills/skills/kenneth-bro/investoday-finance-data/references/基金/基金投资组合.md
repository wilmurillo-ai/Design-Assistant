## 基金的持仓股票

| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000006']` |

**输出参数**

| 字段名 | 说明 | 示例 |

| `investmentType` | 投资类型 | `3` |
| `stockCode` | 基金持股股票代码 | `000002` |
| `rnk` | 基金持股占比名次 | `54` |
| `holdingShares` | 基金持股数量(股) | `23200.0` |
| `marketValue` | 基金持股股票市值(元) | `645192` |
| `navRatio` | 基金持股市值占基金净资产比例(%) | `0.47` |

**接口示例**

```bash
# 可选参数: fundCode, fundCodes
```
