# 基金 / 基金持有人

---

## 基金持有人结构信息

接口路径：`fund/holder-structures`
请求方式：**`POST`**
tool_id：`list_fund_hold_structures`

接口说明：通过基金代码和日期范围查询基金的持有人结构信息，包括持有人户数、户均持有份额、机构、个人、其他投资者及基金从业人员的持有份额与占比，以及报告日期、信息发布来源等，用于分析基金的投资人构成和集中度。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `000001` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000006']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd） | `2025-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `fundCode` | 基金代码 | `000006` |
| `fundId` | 今日投资内部编码 | `000006JK` |
| `fundName` | 基金名称 | `西部利得量化成长混合A` |
| `infoSourceCode` | 信息来源代码 | `3` |
| `infoSource` | 信息来源 | `中报` |
| `publishDate` | 信息发布日期 | `2019-03-20 00:00:00` |
| `reportDate` | 报告截止日期 | `2019-03-14 00:00:00` |
| `holderCount` | 持有人户数（户） | `12345.67` |
| `avgHoldingShare` | 户均持有份额（份） | `12345.67` |
| `instHoldingShare` | 机构持有份额（份） | `123456789.45` |
| `instHoldingRatio` | 机构持有比例（%） | `0.3547` |
| `retailHoldingShare` | 个人持有份额（份） | `12345.67` |
| `retailHoldingRatio` | 个人持有比例（%） | `0.325` |
| `otherHoldingShare` | 未明确投资者持有份额（份） | `12345.67` |
| `otherHoldingRatio` | 未明确投资者持有比例（%） | `0` |
| `staffHoldingShare` | 基金从业人员持有份额（份） | `2235407.2` |
| `staffHoldingRatio` | 基金从业人员持有比例（%） | `1.26` |

**接口示例**

```bash
# 可选参数: fundCode, fundCodes, beginDate, endDate, pageNum, pageSize
investoday-api fund/holder-structures --method POST
```

## 基金份额变动

接口路径：`fund/shares-changes`
请求方式：**`POST`**
tool_id：`list_fund_shares`

接口说明：通过基金代码和日期范围查询基金份额变动明细，返回包括基金基本信息、变动期间、期初期末份额、以及申购、赎回、转入、转出、红利再投资、拆分、扩募、折算等各类变动金额的详细数据，适用于分析基金规模变化、资金流动和投资者行为。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `000001` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000006']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd） | `2025-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `fundCode` | 基金代码 | `000006` |
| `fundId` | 今日投资内部编码 | `000006JK` |
| `fundName` | 基金名称 | `西部利得量化成长混合A` |
| `endDate` | 截止日期 | `2019-03-14 00:00:00` |
| `dataSource` | 信息来源 | `7` |
| `dataSourceDesc` | 信息来源描述 | `截止时点` |
| `startDate` | 开始日期 | `2023-11-15` |
| `initialTotalShare` | 基金合同生效日总份额 | `1000000000.5` |
| `beginShare` | 期初份额 | `10000.5` |
| `subscribeShare` | 本期申购份额 | `12345.67` |
| `redeemShare` | 本期赎回份额 | `12345.67` |
| `transferInShare` | 转入基金份额 | `12345.67` |
| `transferOutShare` | 转出基金份额 | `12345.67` |
| `dividendReinvestShare` | 红利再投资 | `0.035` |
| `splitShare` | 基金拆分 | `1.5` |
| `expansionShare` | 基金扩募 | `1.25` |
| `conversionShareChange` | 基金折算增减份额 | `-123.45` |
| `endShare` | 期末份额 | `177856344.09` |
| `totalEndShare` | 期末份额合计 | `123456.78` |
| `isConsolidated` | 是否合并披露 | `0` |

**接口示例**

```bash
# 可选参数: fundCode, fundCodes, beginDate, endDate, pageNum, pageSize
investoday-api fund/shares-changes --method POST
```
