## 概念持仓的基金列表（批量）

接口路径：`fund/concept-hold-fund/batch`
请求方式：**`POST`**
tool_id：`list_concept_hold_fund_batch`

接口说明：根据概念代码、匹配类型和阈值查询持有该概念的基金列表，返回基金代码、名称、类型、概念代码、名称、类别以及基于不同报告期的持有比例和结束日期，适用于筛选特定概念持仓比例的基金。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `conceptCodes` | ✅ | array | 概念代码 | `['CLS82591','CLS82582']` |
| `matchType` | ✅ | integer | 匹配类型 | `1` |
| `threshold` | ✅ | integer | 阈值 | `40` |
| `isETF` | — | boolean | 是否ETF | `True` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `fundCode` | 基金代码 | `000006` |
| `fundId` | 今日投资内部编码 | `000006JK` |
| `fundName` | 基金名称 | `西部利得量化成长混合A` |
| `endDate` | 截止日期 | `2024-03-28 00:00:00` |
| `fundType` | 基金类型代码 | `2` |
| `conceptCode` | 概念代码 | `CLS80005` |
| `conceptName` | 概念名称 | `中药` |
| `conceptCategory` | 概念类别 | `C01` |
| `holdingWeightLatest` | 基金持仓概念基于年报或半年报的持有比重(%) | `0.215` |
| `holdingWeightQuarterly` | 基金持仓概念基于季报的持有比重(%) | `2.215` |
| `holdingWeightAnnual` | 基金持仓概念基于最新报告的持有比重(%) | `2.3452` |

**接口示例**

```bash
# 可选参数: isETF
investoday-api fund/concept-hold-fund/batch --method POST conceptCodes=CLS82591 matchType=1 threshold=40
```

## 行业持仓的基金列表（批量）

接口路径：`fund/industry-hold-fund/batch`
请求方式：**`POST`**
tool_id：`list_industry_hold_fund_batch`

接口说明：通过行业代码批量查询持有该行业的基金列表，返回基金代码、名称、类型及行业持仓比例等核心数据，适用于分析基金行业配置及投资组合研究。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `industryCodes` | ✅ | array | 行业代码 | `['640000', '740000']` |
| `matchType` | ✅ | integer | 匹配类型 | `1` |
| `threshold` | ✅ | number | 阈值 | `40.0` |
| `isETF` | — | boolean | 是否ETF | `True` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `date` | 发布日期 | `2024-03-28 00:00:00` |
| `fundId` | 今日投资内部编码 | `001701JK` |
| `fundCode` | 基金代码 | `001701` |
| `fundName` | 基金名称 | `国联产业升级灵活配置混合` |
| `fundType` | 基金类型代码 | `2` |
| `industryStandard` | 行业体系 | `INDUS4_CL` |
| `industryCode` | 行业代码 | `C` |
| `industryName` | 行业名称 | `制造业` |
| `industryLevel` | 行业级别 | `1` |
| `holdingWeightLatest` | 基于最新报告的持有比重（占基金资产总值） | `43.0181` |
| `holdingWeightQuarterly` | 基于季报的持有比重（占基金资产总值） | `43.0181` |
| `holdingWeightAnnual` | 基于年报的持有比重（占基金资产总值） | `75.8681` |
| `holdingRatioLatest` | 基于最新报告的持有比重（占基金资产净值） | `44.18` |
| `holdingRatioQuarterly` | 基于季报的持有比重（占基金资产净值） | `44.18` |
| `holdingRatioAnnual` | 基于年报的持有比重（占基金资产净值） | `76.84` |
| `holdingRatioLatestPenetrated` | 基于最新报告的持有比重 | `0.125` |

**接口示例**

```bash
# 可选参数: isETF
investoday-api fund/industry-hold-fund/batch --method POST industryCodes=640000 industryCodes=740000 matchType=1 threshold=40.0
```

## 基金持仓股票及行业涨幅

接口路径：`fund/holdings-stocks-industries`
请求方式：**`POST`**
tool_id：`list_fund_holdings_perf`

接口说明：根据基金代码查询其持仓股票及对应行业信息，包括基金代码、名称、持仓发布日期、截止日期、股票代码、名称、股票今年以来的涨幅、股票所属行业代码、名称以及行业今年以来的涨幅，并按持仓排名排序，适用于分析基金持仓结构及股票与行业的关联表现。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `002594` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000006']` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `fundCode` | 基金代码 | `000001` |
| `fundId` | 今日投资内部编码 | `FINV20240315001` |
| `fundName` | 基金名称 | `华夏成长混合型证券投资基金` |
| `publishDate` | 发布日期 | `2023-08-15` |
| `endDate` | 截止日期 | `2024-12-31` |
| `rnk` | 名次 | `1` |
| `stockCode` | 股票代码 | `600519` |
| `stockName` | 股票名称 | `茅台` |
| `returnYtd` | 股票今年以来的涨幅(%) | `15.23` |
| `industryCode` | 行业代码 | `C01` |
| `industryName` | 行业名称 | `互联网信息服务` |
| `industryReturnYtd` | 行业今年以来的涨幅(%) | `12.5` |
| `marketCap` | 总市值（元） | `831696245316.3` |

**接口示例**

```bash
# 可选参数: fundCode, fundCodes
investoday-api fund/holdings-stocks-industries --method POST
```
