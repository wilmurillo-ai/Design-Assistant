## 基金未复权日行情

接口路径：`fund/daily-quotes`
请求方式：**`POST`**
tool_id：`list_fund_daily_quotes`

接口说明：根据基金代码和指定的日期范围（格式为 yyyy-MM-dd HH:mm:ss）查询基金的历史日行情数据，返回包括基金代码、名称、交易日期、开盘价、最高价、最低价、收盘价、成交量、成交金额等核心指标，支持分页查询，适用于分析基金的历史价格走势和交易活跃度。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `000001` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000006']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd） | `2025-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `fundCode` | 基金代码 | `000006` |
| `fundId` | 今日投资内部编码 | `000006jk` |
| `fundName` | 基金名称 | `华夏成长混合型证券投资基金` |
| `date` | 交易日期 | `2023-10-26` |
| `prevClose` | 前收盘价 | `12.75` |
| `openPrice` | 开盘价 | `12.35` |
| `highPrice` | 最高价 | `128.75` |
| `lowPrice` | 最低价 | `12.75` |
| `closePrice` | 收盘价 | `15.78` |
| `volume` | 成交量 | `12345.67` |
| `amount` | 成交金额 | `123456.78` |

**接口示例**

```bash
# 可选参数: fundCode, fundCodes, beginDate, endDate, pageNum, pageSize
investoday-api fund/daily-quotes --method POST
```

## ETF最新实时日行情

接口路径：`fund-quote/realtime`
请求方式：`GET`
tool_id：`get_fund_quote_realtime`

接口说明：通过基金代码获取ETF基金的最新实时行情数据，包括基金代码、名称、市场类型、开盘价、昨日收盘价、当前价格、涨跌幅、最高价、最低价、数据时间和系统时间等核心数据，适用于实时监控ETF基金的市场表现和价格变动。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | ✅ | string | 基金代码 | `159001` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `fundCode` | 基金代码 | `159616` |
| `fundName` | 基金名称 | `农牧ETF` |
| `marketType` | 市场类型(沪-sh、深-sz、京-bj) | `sz` |
| `openPrice` | 开盘价 | `0.855` |
| `closePriceYDay` | 昨日收盘价 | `0.855` |
| `currentPrice` | 当前价格 | `0.84` |
| `changeRatio` | 涨跌幅 | `-0.0175` |
| `highPrice` | 最高价 | `0.855` |
| `lowPrice` | 最低价 | `0.835` |
| `dataTime` | 数据时间 | `2025-11-04 15:09:00` |
| `sysTime` | 系统时间 | `2025-11-04 15:09:02` |
| `status` | 状态 | `0` |

**接口示例**

```bash
investoday-api fund-quote/realtime fundCode=159001
```

## 基金技术指标

接口路径：`fund/technical-indicators`
请求方式：**`POST`**
tool_id：`list_fund_tech_indicators`

接口说明：通过基金代码和日期范围查询指定基金的技术指标数据，包括压力位和支撑位，用于基金的技术分析和投资决策。入参包括基金代码、开始日期、结束日期、页码和页大小；出参包括基金代码、基金ID、基金名称、交易日期、压力位和支撑位。日期格式为yyyy-MM-dd。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `000001` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001','000006']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd） | `2025-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `date` | 交易日期 | `2023-10-26` |
| `fundId` | 今日投资内部编码 | `000006jk` |
| `fundCode` | 基金代码 | `000006` |
| `fundName` | 基金名称 | `华夏成长混合型证券投资基金` |
| `fundAsset` | 基金规模(元） | `1234568.88` |
| `pressure` | 压力位 | `12.75` |
| `support` | 支撑位 | `12.35` |

**接口示例**

```bash
# 可选参数: fundCode, fundCodes, beginDate, endDate, pageNum, pageSize
investoday-api fund/technical-indicators --method POST
```
