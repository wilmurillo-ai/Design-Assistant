# 基金 / 基金财务数据

## 基金主要财务指标

接口路径：`fund/financial-indicators`
请求方式：**`POST`**
tool_id：`list_fund_fin_inds`

接口说明：通过基金代码和报告期截止日期范围（格式为yyyy-MM-dd HH:mm:ss），查询指定基金在特定期间内的主要财务指标，包括基金基本信息、报告类型、发布及截止日期、以及本期净值增长率、加权平均净值收益率、可分配基金收益、期末单位基金资产净值、本期基金净值收益率、扣公允变动后本期利润、期末基金资产净值、基金累计净值收益率、本期利润、本期公允价值变动损益、加权平均份额本期利润、加权平均份额本期利润率、期末份额累计净值、期末基金资产净值合计等核心财务数据，用于基金财务分析和绩效评估。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `fundCode` | — | string | 基金代码【与fundCodes组成多选一参数，必须且只能传递其中一个】 | `000001` |
| `fundCodes` | — | array | 基金代码列表【与fundCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000006']` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd） | `2025-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `fundCode` | 基金代码 | `000001` |
| `fundId` | 今日投资内部编码 | `TZ20231015001` |
| `fundName` | 基金名称 | `华夏成长混合型证券投资基金` |
| `reportType` | 报告类型 | — |
| `publishDate` | 信息发布日期 | `2023-08-15` |
| `reportPeriodEnd` | 报告截止日期 | `2023-12-31` |
| `isConsolidated` | 是否合并披露 | `0` |
| `navChangePctCurrent` | 本期净值增长率 | `-0.0235` |
| `navReturnWeightedAvg` | 基金加权平均净值收益率 | `0.0457` |
| `distributableIncome` | 期末可供分配基金收益 | `1234567.89` |
| `distributableIncomePerShare` | 期末可供分配份额收益 | `12345.67` |
| `navPerShareEnd` | 期末单位基金资产净值 | `1.2567` |
| `navReturnCurrent` | 基金本期净值收益率 | `-0.0235` |
| `profitExclFairValueChange` | 扣公允价值变动后本期利润 | `-12345678.9` |
| `navEnd` | 期末基金资产净值 | `1234567890.12` |
| `navReturnCumulative` | 基金累计净值收益率 | `0.1523` |
| `profitCurrent` | 本期利润 | `-1234567.89` |
| `fairValueChangeIncome` | 本期公允价值变动损益 | `-1234567.89` |
| `profitPerShareWeightedAvg` | 加权平均份额本期利润 | `0.1523` |
| `profitMarginPerShareWeightedAvg` | 加权平均份额本期利润率 | `-0.0235` |
| `navAccAdjEnd` | 期末份额累计净值 | `1.3567` |
| `totalNavEnd` | 期末基金资产净值合计 | `1234567890.12` |

**接口示例**

```bash
# 可选参数: fundCode, fundCodes, beginDate, endDate, pageNum, pageSize
investoday-api fund/financial-indicators --method POST
```
