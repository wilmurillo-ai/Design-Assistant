# 市场数据

---

## 市场涨跌分布

接口路径：`market/change-ratio-status`
请求方式：`GET`
tool_id：`get_market_change_ratio_status`

接口说明：获取当前市场实时涨跌分布数据，包含上涨、下跌和平盘的股票数量，以及涨跌幅不同区间的股票具体数量，如涨停、跌停、涨跌幅超过8%、6%-8%、2%-6%、0%-2%等区间分布，便于大模型全面分析市场热度和情绪。

**输入参数**

_无参数_

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `dataTime` | 数据时间 | `2025-06-11 16:45:45` |
| `upAmount` | 上涨数量（涨幅>0%） | `3413` |
| `downAmount` | 下跌数量（涨幅<0%） | `1737` |
| `upTopAmount` | 涨停数量（主板涨幅=10%、创业板、科创板=20%、北交所=30%） | `71` |
| `upOver8Amount` | 上涨数量（8%,+∞） | `35` |
| `upBx68Amount` | 上涨数量（6%,8%] | `47` |
| `upBx26Amount` | 上涨数量（2%,6%] | `622` |
| `upBx02Amount` | 上涨数量（0%,2%] | `2638` |
| `bxAmount` | 平盘数量（涨幅=0%） | `244` |
| `downBx02Amount` | 下跌数量[-2%,0%) | `1467` |
| `downBx26Amount` | 下跌数量[-6%,-2%) | `226` |
| `downBx68Amount` | 下跌数量[-8%,-6%) | `16` |
| `downOver8Amount` | 下跌数量（-∞,-8%） | `21` |
| `downTopAmount` | 跌停数量（主板涨幅=-10%、创业板、科创板=-20%、北交所=-30%） | `7` |

**接口示例**

```bash
investoday-api market/change-ratio-status
```
