# 指数 / 基础行情

## 指数历史日行情

接口路径：`index/quotes`
请求方式：`GET`
tool_id：`list_index_quotes`

接口说明：通过指数代码和日期范围查询指数的历史日行情数据，包括开盘价、最高价、最低价、收盘价、成交量、成交金额等核心指标，适用于市场分析和指数表现追踪。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `indexCode` | ✅ | string | 指数代码 | `000300` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2025-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `date` | 发布日期 | `2015-05-26 00:00:00` |
| `indexCode` | 指数代码 | `000300` |
| `indexName` | 指数名称 | `沪深300` |
| `previousClosePrice` | 昨收盘价 | `3850.25` |
| `openPrice` | 开盘价 | `3860.15` |
| `highPrice` | 最高价 | `3880.45` |
| `lowPrice` | 最低价 | `3845.3` |
| `closePrice` | 收盘价 | `3875.6` |
| `volume` | 成交量 | `125000000.0` |
| `tradingAmountCny` | 成交金额（元） | `4850000000.0` |

**接口示例**

```bash
# 可选参数: beginDate, endDate, pageNum, pageSize
investoday-api index/quotes indexCode=000300
```

## 指数最新实时日行情

接口路径：`index-quote/realtime`
请求方式：**`POST`**
tool_id：`get_index_realtime_quotes`

接口说明：使用指数代码获取指定指数的最新实时行情数据，包含指数当前价格、涨跌幅、开盘价、最高价、最低价、成交量、成交金额等核心数据，便于大模型进行即时的市场监测和指数分析。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `indexCodes` | ✅ | array | 指数代码 | `['000001', '399006']` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `indexCode` | 指数代码 | `000001` |
| `industryName` | 指数名称 | `上证指数` |
| `openPrice` | 开盘价 | `3369.3662` |
| `closePriceYDay` | 昨日收盘价 | `3376.9961` |
| `currentPrice` | 当前价格 | `3385.1443` |
| `changeRatio` | 涨跌幅 | `0.00241285` |
| `highPrice` | 最高价 | `3385.5259` |
| `lowPrice` | 最低价 | `3368.8765` |
| `dealStockAmount` | 成交股数 | `335443869` |
| `dealMoney` | 成交金额 | `401388535808` |
| `dataTime` | 数据时间 | `2025-06-16 14:14:39` |

**接口示例**

```bash
investoday-api index-quote/realtime --method POST indexCodes=000001 indexCodes=399006
```
