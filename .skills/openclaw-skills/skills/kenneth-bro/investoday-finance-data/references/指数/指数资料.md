# 指数 / 指数资料

---

## 指数基本信息

接口路径：`index/basic-info`
请求方式：**`POST`**
tool_id：`get_index_basic_info`

接口说明：通过指数代码（单个或批量）查询指数的基本信息，包括指数名称、全称、交易市场、发布机构、编制机构、基期、基点、启用日期、停用日期、加权方式、调整频率、指数简介、币种、发布状态和成分证券市场等核心元数据，适用于获取指数的元数据和分类信息。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `indexCode` | — | string | 指数代码【与indexCodes组成多选一参数，必须且只能传递其中一个】 | `399300` |
| `indexCodes` | — | array | 指数代码列表【与indexCode组成多选一参数，必须且只能传递其中一个】 | `['000001', '000300']` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `indexCode` | 指数代码 | `000300` |
| `indexName` | 指数名称 | `000300` |
| `indexNameFull` | 指数全称 | `000300` |
| `exchangeCode` | 交易市场 | `000300` |
| `publisherName` | 发布机构 | `000300` |
| `compilerName` | 编制机构 | `000300` |
| `baseDate` | 基期 | `000300` |
| `basePoint` | 基点 | `300.0` |
| `launchDate` | 启用日期 | `000300` |
| `delistDate` | 停用日期 | `000300` |
| `weightingMethod` | 加权方式 | `000300` |
| `reviewFrequency` | 调整频率 | `300` |
| `indexDescription` | 指数简介 | `000300` |
| `currencyCode` | 币种 | `000300` |
| `publishStatus` | 发布状态 | `1` |
| `constituentMarket` | 成份证券市场 | `000300` |

**接口示例**

```bash
# 可选参数: indexCode, indexCodes
investoday-api index/basic-info --method POST
```
