## 行业的最新实时日行情

接口路径：`industry-quote/realtime`
请求方式：`GET`
tool_id：`get_industry_realtime_quote`

接口说明：输入行业代码，获取该行业的最新实时指数、涨跌幅、成交量、总市值、成分股数量、涨停/上涨/下跌/平盘/总股数、涨跌幅排名、领涨股等关键行情数据，用于实时跟踪行业整体表现。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `industryCode` | ✅ | string | 行业代码 | `330000` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `industryCode` | 行业代码 | `740000` |
| `industryName` | 行业名称 | `煤炭` |
| `price` | 行业指数 | `2549.1562` |
| `changeRatio` | 行业涨跌幅 | `-0.0121602` |
| `volume` | 行业成交量 | `9547379` |
| `changeRatio1W` | 行业一周涨跌幅 | `-0.00646898` |
| `totalValue` | 行业总市值 | `1756873387871.2708` |
| `ratioRank` | 行业涨跌幅排名 | `31` |
| `industryAmount` | 行业成分股数量 | `31` |
| `limitUpAmount` | 行业涨停股数量 | `0` |
| `stockUpAmount` | 行业上涨股数量 | `1` |
| `stockDownAmount` | 行业下跌股数量 | `32` |
| `stockBxAmount` | 行业平盘股数量 | `4` |
| `stockAmount` | 行业股票总数 | `37` |
| `leadUpStockCode` | 行业领涨股代码 | — |
| `leadUpStockName` | 行业领涨股名称 | — |
| `dataTime` | 数据时间 | — |

**接口示例**

```bash
investoday-api industry-quote/realtime industryCode=330000
```

## 行业的最新实时日行情V2

接口路径：`industry-quote/realtime-v2`
请求方式：**`POST`**
tool_id：`list_ind_real_quote_v2`

接口说明：通过指定行业代码（可批量，空数组表示全市场）、行业等级、行业分类（如申万SW）等条件，获取行业的最新实时行情数据，包括行业价格、涨跌幅、近一周涨跌幅、成交量、总市值、数据时间（格式为yyyy-MM-dd HH:mm:ss）、行业内股票涨跌家数、领涨股及ETF信息、行业排名等核心指标，并支持按指定字段（如涨跌幅）和排序方式（升序/降序）进行分页查询，适用于实时监测行业板块表现、分析市场热点和进行行业对比等任务。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `industryLevel` | — | integer | 行业等级(1、2、3)行业代码为空时必传 | `1` |
| `industryType` | — | string | 行业类型(默认: SW) | `SW` |
| `order` | — | string | 排序方式(asc、desc) | `desc` |
| `sortColumn` | ✅ | string | 排序字段(接口返回值) | `changeRatio` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |
| `industryCodes` | — | array | 行业代码(全市场为:[]) | — |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `industryCode` | 行业代码 | `740000` |
| `industryName` | 行业名称 | `煤炭` |
| `changeRatio` | 涨跌幅 | `0.04` |
| `changeRatio1W` | 近一周涨跌幅 | `0.12` |
| `dataTime` | 数据时间 | `2025-01-01 10:10:10` |
| `industryAmount` | 行业总数 | `30` |
| `leadUpFundCode` | 领涨ETF基金代码 | `159590` |
| `leadUpFundName` | 领ETF基金名称 | `软件ETF` |
| `leadUpStockCode` | 领涨股代码 | `002594` |
| `leadUpStockName` | 领涨股名称 | `比亚迪` |
| `limitUpAmount` | 涨停家数 | `10` |
| `price` | 价格 | `12` |
| `ratioRank` | 同行业排名 | `1` |
| `stockAmount` | 股票家数 | `30` |
| `stockBxAmount` | 0%家数 | `1` |
| `stockDownAmount` | 下跌家数 | `2` |
| `stockUpAmount` | 上涨家数 | `3` |
| `totalValue` | 总市值 T日 | `15343870.0` |
| `volume` | 成交量 | `1354808410.0` |

**接口示例**

```bash
# 可选参数: industryLevel, industryType, order, pageSize, industryCodes
investoday-api industry-quote/realtime-v2 --method POST sortColumn=changeRatio
```
