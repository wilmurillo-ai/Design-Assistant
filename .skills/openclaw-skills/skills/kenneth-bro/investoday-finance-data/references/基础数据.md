## 交易日历衍生

接口路径：`trade-calender/special-date`
请求方式：`GET`
tool_id：`get_trade_special_date`

接口说明：根据指定交易日期，返回A股市场相关的交易日衍生日期，包括前后N个交易日、近一周、近一月、近一年、以及当前周/月/年的首尾交易日，便于大模型进行时间维度的数据分析和策略计算。返回日期统一为yyyy-MM-dd HH:mm:ss格式。 上述时间范围均以指定的tradeDate作为基准日期推算得出。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `tradeDate` | — | string | 交易日期(格式yyyy-MM-dd) | `2025-05-19` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `pubDate` | 基准日期(格式yyyy-MM-dd HH:mm:ss) | `2025-05-19 00:00:00` |
| `before1Day` | 上一个交易日(格式yyyy-MM-dd HH:mm:ss) | `2025-05-16 00:00:00` |
| `before2Day` | 前2个交易日(格式yyyy-MM-dd HH:mm:ss) | `2025-05-15 00:00:00` |
| `before3Day` | 前3个交易日(格式yyyy-MM-dd HH:mm:ss) | `2025-05-14 00:00:00` |
| `before4Day` | 前4个交易日(格式yyyy-MM-dd HH:mm:ss) | `2025-05-13 00:00:00` |
| `before1Week` | 1周前交易日(格式yyyy-MM-dd HH:mm:ss) | `2025-05-12 00:00:00` |
| `before2Week` | 2周前交易日(格式yyyy-MM-dd HH:mm:ss) | `2025-04-30 00:00:00` |
| `before1Month` | 1月前首个交易日(格式yyyy-MM-dd HH:mm:ss) | `2025-04-18 00:00:00` |
| `before2Month` | 2月前首个交易日(格式yyyy-MM-dd HH:mm:ss) | `2025-03-19 00:00:00` |
| `before3Month` | 3月前首个交易日(格式yyyy-MM-dd HH:mm:ss) | `2025-02-19 00:00:00` |
| `before6Month` | 6月前首个交易日(格式yyyy-MM-dd HH:mm:ss) | `2024-11-19 00:00:00` |
| `before1Year` | 1年前首个交易日(格式yyyy-MM-dd HH:mm:ss) | `2024-05-17 00:00:00` |
| `before2Year` | 2年前首个交易日(格式yyyy-MM-dd HH:mm:ss) | `2023-05-19 00:00:00` |
| `before3Year` | 3年前首个交易日(格式yyyy-MM-dd HH:mm:ss) | `2022-05-19 00:00:00` |
| `firstCtWeek` | 本周首个交易日(格式yyyy-MM-dd HH:mm:ss) | `2025-05-19 00:00:00` |
| `lastCtWeek` | 本周最后交易日(格式yyyy-MM-dd HH:mm:ss) | `2025-05-23 00:00:00` |
| `firstCtMonth` | 本月首个交易日(格式yyyy-MM-dd HH:mm:ss) | `2025-05-06 00:00:00` |
| `lastCtMonth` | 本月最后交易日(格式yyyy-MM-dd HH:mm:ss) | `2025-05-30 00:00:00` |
| `firstCtYear` | 本年首个交易日(格式yyyy-MM-dd HH:mm:ss) | `2025-01-02 00:00:00` |
| `lastCtYear` | 本年最后交易日(格式yyyy-MM-dd HH:mm:ss) | `2025-12-31 00:00:00` |
| `after1Day` | 后1个交易日(格式yyyy-MM-dd HH:mm:ss) | `2025-05-20 00:00:00` |
| `after2Day` | 后2个交易日(格式yyyy-MM-dd HH:mm:ss) | `2025-05-21 00:00:00` |
| `after3Day` | 后3个交易日(格式yyyy-MM-dd HH:mm:ss) | `2025-05-22 00:00:00` |

**接口示例**

```bash
# 可选参数: tradeDate
investoday-api trade-calender/special-date
```

## 交易日历列表

接口路径：`trade-calender/special-date/cn/list`
请求方式：`GET`
tool_id：`get_cn_trade_calender_list`

接口说明：返回指定周期内的A股市场交易日列表，支持近一个月（M1）、近三个月（M3）、近六个月（M6）和近一年（Y1，默认），日期格式统一为yyyy-MM-dd。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `cycle` | ✅ | string | 周期 | `Y1` |

**接口示例**

```bash
investoday-api trade-calender/special-date/cn/list cycle=Y1
```

## 实体识别

接口路径：`entity-recognition`
请求方式：**`POST`**
tool_id：`entity_recognition`

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `input` | ✅ | string | 待识别的自然语言 | `贵州茅台怎么样？` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `entities` |  | — |

**接口示例**

```bash
investoday-api entity-recognition --method POST input=贵州茅台怎么样？
```

## 综合标的搜索

接口路径：`search`
请求方式：`GET`
tool_id：`search`

接口说明：通过关键字和指定的搜索类型（如沪深京A股、基金、ETF、港股、行业、概念等），分页搜索并返回匹配的金融产品列表，包含产品代码、简称、全称、拼音、市场类型、行业级别、风险等级等核心信息，便于大模型进行金融产品的快速查找和筛选。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `key` | — | string | 关键字 | `比亚迪` |
| `type` | ✅ | string | 搜索类型多个用,隔开 | `11` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `code` | 代码 | `002594` |
| `shortName` | 简称 | `比亚迪` |
| `fullName` | 全称 | `比亚迪` |
| `first` | 拼音首字母 | `B` |
| `pinyin` | 拼音简拼 | `BYD` |
| `quanPin` | 简称全拼 | `BiYaDi` |
| `type` | 搜索分类类型 | `11` |
| `id` | id | `002594` |
| `mkt` | 市场类型 | `SZ` |
| `fundSaleFlag` | 是否代销基金 | `0` |
| `industryLevel` | 行业级别 | `1` |
| `riskLevel` | 风险等级 | `1` |

**接口示例**

```bash
# 可选参数: key, pageNum, pageSize
investoday-api search type=11
```
