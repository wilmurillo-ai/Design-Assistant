**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `002594` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `stockCode` | 股票代码 | `600622` |
| `stockName` | 股票名称 | `光大嘉宝` |
| `reportDate` | 报告期 | `1990-12-31 00:00:00` |
| `actualPublishDate` | 实际披露日期 | `1992-05-18 00:00:00` |
| `nextReportPublishDate` | 下一报告期披露日期 | `1992-05-18 00:00:00` |
| `prevReportPeriod` | 上一报告期 | `1992-08-18 00:00:00` |
| `samePeriodLastYear` | 去年同期 | `2023-07-15` |

**接口示例**

```bash
# 可选参数: beginDate, endDate, pageNum, pageSize
investoday-api stock/report-schema stockCode=002594
```
