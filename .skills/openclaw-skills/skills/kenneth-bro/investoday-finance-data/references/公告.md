# 公告

---

## 上市公司的公告

接口路径：`announcements`
请求方式：`GET`
tool_id：`list_announcements`

接口说明：支持通过公告标识、日期范围、股票代码、标题关键词等条件组合查询上市公司公告，返回公告基础信息（标题、发布日期）、内容详情（全文、风险类型）、附件信息（URL、有无附件）等完整字段，适用于金融资讯展示、企业公告管理及数据分析等场景，实现精准高效的公告信息获取与处理。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2020-01-02` |
| `stockCode` | ✅ | string | 股票代码 | `000001` |
| `title` | — | string | 标题（支持模糊查询） | `贵州茅台利润大涨` |
| `announcementID` | — | integer | 公告id | `9128035` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `date` | 发布日期（格式yyyy-MM-dd） | `2025-01-15` |
| `announcementId` | 公告标识 | `9128035` |
| `stockCode` | 股票代码 | `000001` |
| `stockName` | 股票名称 | `平安银行` |
| `title` | 标题 | `同享科技:董事会决议公告` |
| `hasAttachment` | 有无附件 | `1` |
| `attachmentUrl` | 附件URL | `https://www.bse.cn/disclosure/2025/2025-01-15/f3973168ca8e4c87a5c434c4a858b9e4.pdf` |
| `riskType` | 风险类型 | `0` |
| `announcementSource` | 公告来源 | `北交所` |
| `announcementContentType` | 公告内容类别代码 | `040301` |
| `guid` | 公告编号 | `B761EA4B-7146-436E-9C4E-03CE948A6DD4` |

**接口示例**

```bash
# 可选参数: beginDate, endDate, title, announcementID, pageNum, pageSize
investoday-api announcements stockCode=000001
```

## 上市公司的公告原文

接口路径：`announcements/content`
请求方式：`GET`
tool_id：`get_announcement_content`

接口说明：通过公告ID获取上市公司的公告详细内容，包括公告标题和完整正文，适用于公告内容展示、文本分析等场景。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `announcementId` | ✅ | integer | 公告ID | `9128035` |
| `contentLength` | — | integer | 内容长度（最大2000，默认2000）。 最小值:1; 最大值:2000; | `2000` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `date` | 发布日期（格式yyyy-MM-dd） | `2025-01-15` |
| `announcementId` | 公告ID | `9128035` |
| `title` | 公告标题 | `"同享科技:董事会决议公告"` |
| `content` | 公告内容 | `...` |
| `guid` | 公告编号 | `B761EA4B-7146-436E-9C4E-03CE948A6DD4` |

**接口示例**

```bash
# 可选参数: contentLength
investoday-api announcements/content announcementId=9128035
```
