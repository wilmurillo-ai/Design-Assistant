## 上市公司互动问答

接口路径：`stock/consultations`
请求方式：`GET`
tool_id：`list_stk_consultations`

接口说明：根据股票代码和日期范围查询上市公司在互动平台上的问答记录，包括咨询类型、提问人、问题详情、回复内容及传闻核实情况，支持分页获取，便于分析公司沟通与市场传闻。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `002594` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `stockCode` | 股票代码 | `600519` |
| `stockName` | 股票名称 | `茅台` |
| `date` | 咨询日期 | `2025-12-30` |
| `consultId` | 咨询序号 | `1` |
| `consultType` | 咨询互动类型 | `1` |
| `infoSource` | 信息来源 | `深圳互动易` |
| `questioner` | 提问人 | `张三` |
| `question` | 问题情况 | `公司的经营状况怎么样` |
| `replyDate` | 回复日期 | `2026-01-01` |
| `reply` | 问题回复 | `非常好` |
| `isRumorTrue` | 传闻是否属实 | `1` |

**接口示例**

```bash
# 可选参数: beginDate, endDate, pageNum, pageSize
investoday-api stock/consultations stockCode=002594
```

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `002594` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `stockCode` | 股票代码 | `600519` |
| `stockName` | 股票名称 | `茅台` |
| `reportPeriodEnd` | 发布日期 | `2025-12-30` |
| `publishDate` | 截止日期 | — |
| `announcementId` | 公告id | `8596420` |
| `operationalReview` | 经营述评 | — |

**接口示例**

```bash
# 可选参数: beginDate, endDate, pageNum, pageSize
investoday-api stock/operating-reviews stockCode=002594
```

## 基金公司的公告

接口路径：`fund/announcements`
请求方式：**`POST`**
tool_id：`list_fund_announcements`

接口说明：支持通过基金代码（必填）、公告ID、日期范围（格式yyyy-MM-dd）、标题（支持模糊查询），结合页码、页长进行分页查询基金公司公告，可返回公告发布日期、公告标识、基金代码及名称、标题、风险类型、公告来源、公告内容类别代码等信息，适用于金融资讯展示、企业公告管理及数据分析等场景，实现精准高效的公告信息获取与处理。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2020-01-02` |
| `fundCodes` | ✅ | array | 基金代码 | `['000001', '000004']` |
| `title` | — | string | 标题（支持模糊查询） | `贵州茅台利润大涨` |
| `announcementID` | — | integer | 公告id | `9128035` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `date` | 发布日期（格式yyyy-MM-dd） | `2025-01-15` |
| `announcementId` | 公告标识 | `9128035` |
| `fundCode` | 基金代码 | `000001` |
| `fundName` | 基金名称 | `平安银行` |
| `title` | 标题 | `同享科技:董事会决议公告` |
| `riskType` | 风险类型 | `0` |
| `announcementSource` | 公告来源 | `北交所` |
| `announcementContentType` | 公告内容类别代码 | `040301` |

**接口示例**

```bash
# 可选参数: beginDate, endDate, title, announcementID, pageNum, pageSize
investoday-api fund/announcements --method POST fundCodes=000001 fundCodes=000004
```
