## 公司分红派息信息

接口路径：`hk-stock/dividends`
请求方式：`GET`
tool_id：`list_hk_stock_dividends`

接口说明：通过股票代码查询香港上市公司的分红派息信息，包括每股派息金额、派息日、除净日、货币单位等核心数据，支持按公告日期范围筛选，便于分析公司的分红政策和历史记录。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `00001` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-01-01` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `announceDate` | 最新公告日 | `1999-01-08 00:00:00` |
| `stockCode` | 股票代码 | `00027` |
| `stockName` | 股票名称 | `银河娱乐` |
| `fiscalYear` | 财政年度 | `2024-01-01` |
| `dividendYearStart` | 分配年度起始日 | `2024-01-01` |
| `dividendYearEnd` | 分配年度截止日 | `2024-12-31` |
| `dividendType` | 分红方式 | `RS` |
| `reportType` | 报告期类别 | `年报` |
| `cashDividendPerShareOrig` | 每股派息(原币) | `0.75` |
| `currencyCode` | 货币代码 | `USD` |
| `cashDividendPerShare` | 每股派息 | `0.75` |
| `physicalDeliveryPerShare` | 每股实物派送 | `10股送2股` |
| `exDate` | 除净日 | `1999-01-08 00:00:00` |
| `finalTransferDate` | 最后过户日 | `2023-08-15` |
| `transferStartDate` | 截止过户起始日 | `2024-12-31` |
| `transferEndDate` | 截止过户终止日 | `2024-12-31` |
| `payDate` | 派息日 | `2024-06-20` |

**接口示例**

```bash
# 可选参数: beginDate, endDate, pageNum, pageSize
investoday-api hk-stock/dividends stockCode=00001
```

## 港股股本变动

接口路径：`hk-stock/cap_changes`
请求方式：`GET`
tool_id：`list_hk_stock_capital_changes`

接口说明：通过股票代码、起始日期和结束日期查询港股公司的股本变动历史记录，返回包括股票代码、名称、变动日期、各类别股份数量（如优先股、普通股、H股、大陆股等）及总股本等详细信息，适用于分析公司资本结构变化和股权历史。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `00001` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | — |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `PUBDT` | 发布日期 | `2023-11-15` |
| `f2000` | 股票代码 | `00001` |
| `F2004` | 中文简称 | `长和` |
| `F1000` | 公司代码 | `LA00001` |
| `F1100` | 变动日期 | `2024-06-15` |
| `F1101` | 优先股股数 | `5000000.0` |
| `F1102` | 普通股股数 | `1000000.0` |
| `F1103` | H股股数 | `1000000.0` |
| `F1104` | 其他股数（大陆股数） | `10000.0` |
| `F1105` | 特殊股数 | `10000.0` |
| `F1106` | 香港股数部分 | `1500.5` |
| `F1107` | 大陆股数部分 | `1000000.0` |
| `F1108` | 总股数 | `1000000000.0` |

**接口示例**

```bash
# 可选参数: beginDate, endDate, pageNum, pageSize
investoday-api hk-stock/cap_changes stockCode=00001
```
