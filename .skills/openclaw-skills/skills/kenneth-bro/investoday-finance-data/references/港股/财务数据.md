## 港股现金流量表

接口路径：`hk-stock/cash-flows`
请求方式：`GET`
tool_id：`list_hk_stock_cash_flows`

接口说明：通过股票代码、报告期起止日期（格式为yyyy-MM-dd）等条件，查询港股上市公司最新发布的现金流量表数据，包括经营活动、投资活动、融资活动产生的现金流量，以及现金及现金等价物的期初、期末余额等核心财务指标，用于分析公司的现金创造能力和财务健康状况。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `002594` |
| `beginDate` | — | string | 开始日期（格式yyyy-MM-dd）。 最小值:2020-01-01; | `2020-01-01` |
| `endDate` | — | string | 结束日期（格式yyyy-MM-dd） | `2025-03-30` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `stockCode` | 股票代码 | `000001` |
| `stockName` | 股票名称 | `贵州茅台` |
| `publishDate` | 发布日期 | `2023-08-15` |
| `reportDate` | 报告期 | `2025-09-30` |
| `reportType` | 报告类型 | `Q1` |
| `fiscalPeriod` | 会计期间 | `9` |
| `currencyCode` | 货币代码 | `CNY` |
| `operatingCashFlowAmount` | 经营业务之现金流量额(元) | `12345667.11` |
| `interestReceived` | 已收利息(元) | `12345667.11` |
| `interestPaid` | 已付利息(元) | `12345667.11` |
| `dividendPaid` | 已派股息(元) | `12345667.11` |
