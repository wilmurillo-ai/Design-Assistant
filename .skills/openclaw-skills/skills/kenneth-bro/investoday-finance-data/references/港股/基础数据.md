## 港股基本信息（GET）

接口路径：`hk-stock/basic-info`
请求方式：`GET`
tool_id：`get_hk_stock_basic_info`

> 📌 GET 版本，兼容历史调用，仅支持单标的查询。推荐使用 POST 版本进行批量查询。

接口说明：通过股票代码查询香港股票的基本信息，包括股票代码、名称（中文全称、简称、英文简称）、上市状态、上市板块、交易货币、买卖单位、上市日期、摘牌日期、公司代码、交易市场编码以及是否为港股通标的等核心数据，便于大模型进行股票基本信息的查询和分析。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `00001` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `stockCode` | 股票代码 | `00001` |
| `stockName` | 股票名称 | `长和` |
| `stockNameFull` | 证券全称 | `长江和记实业有限公司` |
| `exchangeCode` | 交易市场编码 | `HK` |
| `stockNameEn` | 证券英文简称 | `CKH HOLDINGS` |
| `listStatus` | 上市状态 | `正常` |
| `boardName` | 上市板块 | `主板` |
| `currencyCode` | 交易货币 | `HKD` |
| `lotSize` | 买卖单位(股/手) | `500.0` |
| `listDate` | 上市日期 | `1972-11-01 00:00:00` |
| `delistDate` | 摘牌日期 | `2023-08-15` |
| `companyCode` | 公司代码 | `LH00001` |
| `isConnect` | 港股通标记 | `1` |

**接口示例**

```bash
investoday-api hk-stock/basic-info stockCode=00001
```

## 港股简称变更

接口路径：`hk-stock/name-change`
请求方式：`GET`
tool_id：`get_hk_stock_name_change`

接口说明：通过港股股票代码查询其公司名称变更历史，返回包括变更前后的中英文简称、公告日期和生效日期等关键信息，便于大模型进行公司历史沿革分析或数据核对。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `00001` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `announceDate` | 公告日期 | `2023-08-15` |
| `stockCode` | 股票代码 | `000001.SZ` |
| `stockName` | 变更后中文简称 | `华泰证券` |
| `stockNamePrev` | 更名前中文简称 | `华泰证券` |
| `stockNameEn` | 变更后英文简称 | `ABC Corp.` |
| `stockNameEnPrev` | 更名前英文简称 | `ABC Corp.` |
| `effectiveDate` | 变更生效日期 | `2024-03-15` |

**接口示例**

```bash
investoday-api hk-stock/name-change stockCode=00001
```

## 港股股票转板

接口路径：`hk-stock/transfers`
请求方式：`GET`
tool_id：`get_hk_stock_transfers`

接口说明：通过股票代码查询港股股票转板信息，返回包括转板前后的股票代码、股票名称、公告日期、主板上市日期和创业板最后交易日等关键数据，用于追踪港股市场板块变动情况。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `00001` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `announceDate` | 公布日期 | `2014-08-25 00:00:00` |
| `stockCode` | 转板后交易代码 | `00048` |
| `stockName` | 股票名称 | `中国汽车内饰` |
| `stockCodePre` | 转板前股票代码 | `08321` |
| `mainBoardListDate` | 主板上市日期 | `2014-08-25 00:00:00` |
| `gemLastTradeDate` | 创业板最后交易日 | `2014-08-22 00:00:00` |

**接口示例**

```bash
investoday-api hk-stock/transfers stockCode=00001
```

## 港股基本信息（POST）

接口路径：`hk-stock/basic-info`
请求方式：**`POST`**
tool_id：`get_hk_stock_basic_info`

> 📌 POST 版本，支持批量查询（可传多个标的）。

接口说明：通过股票代码（可批量）查询香港股票的基本信息，包括股票代码、名称（中文全称、简称、英文简称）、上市状态、上市板块、交易货币、买卖单位、上市日期、摘牌日期、公司代码、交易市场编码以及是否为港股通标的等核心数据，便于大模型进行股票基本信息的查询和分析。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCodes` | — | array | 股票代码 | `['00001', '00004']` |
| `pageNum` | — | integer | 页码。 最小值:1; | `1` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |

**输出参数**

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `stockCode` | 股票代码 | `00001` |
| `stockName` | 股票名称 | `长和` |
| `stockNameFull` | 证券全称 | `长江和记实业有限公司` |
| `exchangeCode` | 交易市场编码 | `HK` |
| `stockNameEn` | 证券英文简称 | `CKH HOLDINGS` |
| `listStatus` | 上市状态 | `正常` |
| `boardName` | 上市板块 | `主板` |
| `currencyCode` | 交易货币 | `HKD` |
| `lotSize` | 买卖单位(股/手) | `500.0` |
| `listDate` | 上市日期 | `1972-11-01 00:00:00` |
| `delistDate` | 摘牌日期 | `2023-08-15` |
| `companyCode` | 公司代码 | `LH00001` |
| `isConnect` | 港股通标记 | `1` |

**接口示例**

```bash
# 可选参数: stockCodes, pageNum, pageSize
investoday-api hk-stock/basic-info --method POST
```
