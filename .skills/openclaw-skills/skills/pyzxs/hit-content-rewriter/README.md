# hit-content-rewriter OpenClaw 技能使用指南

## 📋 技能概述

`hit-content-rewriter` 是一个爆款内容仿写专家技能，提供独立的爆款内容处理操作。每个操作都是独立的，可单独执行，不会自动触发其他操作。

**核心功能**：
- 🔍 **文案提取**：从短视频链接提取文案内容
- 📊 **内容分析**：深度分析爆款内容的结构、风格和传播要素
- 💾 **内容存储**：将爆款内容存储到向量数据库
- 🔎 **内容查询**：从数据库查询相似爆款内容
- ✍️ **智能仿写**：基于参考爆款创作新内容

## 🚀 安装与配置

### 前置依赖
在OpenClaw中使用本技能前，需要安装以下工具：

1. **向量存储工具**：
```bash
uv tool install jl-vector-store
```

2. **环境变量配置**（用于向量化API）：
```bash
export EMBEDDING_BASE_URL="https://api.siliconflow.cn/v1/embeddings"
export EMBEDDING_API_KEY="your-api-token"
export EMBEDDING_MODEL="BAAI/bge-large-zh-v1.5"
```

3. **可选技能**：
- `jl-video-downloader`：用于短视频文案提取
- `jl-content-rewriter`：用于内容人性化优化

### 技能安装
将本技能目录放置到OpenClaw的技能目录中，OpenClaw会自动识别并加载。

## ⚠️ 注意事项

### 1. 操作独立性
- 每个操作都是独立的，不会自动触发其他操作
- 提取文案仅提取文案，不会自动分析或存储
- 分析内容仅进行分析，不会自动存储
- 需要组合操作时，请分别执行

### 2. 依赖管理
- 文案提取需要 `jl-video-downloader` 技能
- 存储和查询需要 `jl-vector-store` 工具
- 确保环境变量正确配置

### 3. 输入格式
- 链接提取：支持抖音、小红书等平台链接
- 文件分析：支持文本文件路径
- 参数传递：使用 `--参数名 参数值` 格式

### 4. 输出说明
- 所有输出都会在OpenClaw对话中显示
- 结构化报告会参考模板格式
- 操作结果可能包含建议的后续步骤

### 5. 错误处理
- 如果依赖缺失，会提示安装方法
- 如果输入格式错误，会给出正确格式示例
- 如果操作失败，会提供错误原因和解决方案
