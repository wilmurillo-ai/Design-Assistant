# 记忆算法与复习计划

## 艾宾浩斯遗忘曲线复习间隔

| mastery_level | 复习间隔 | 说明 |
|---------------|----------|------|
| 0 | 当天 | 首次学习 |
| 1 | 第2天 | 第1轮复习 |
| 2 | 第4天 | 第2轮复习 |
| 3 | 第7天 | 第3轮复习 |
| 4 | 第15天 | 第4轮复习 |
| 5 | 第30天 | 第5轮复习 |
| 6 | 停止复习 | 已掌握 |

## 记忆状态更新规则

**测验结果处理：**

| 结果 | 操作 |
|------|------|
| 答对 | `mastery_level += 1`, `correct_count += 1`, 计算下次复习日期 |
| 答错 | `wrong_count += 1`, `mastery_level` 不变 |

## 复习优先级

1. 优先复习 `next_review_date` 已过期的单词
2. 按复习次数排序，次数少的优先
3. 每次复习数量不超过用户设置的每日数量
