# 合同审核流式接口 (Contract Audit Stream)

这是一个用于自动化批量审核合同的 AI 技能（Skill）。

## 🚀 快速开始

### 1. 配置 API Key（必须）

在使用本技能前，你必须拥有一个有效的 `api_key`。可前往微信小程序‘黛玉明察’创建您的 api_key。
请打开本目录下的 `config.json` 文件，将默认值替换为你真实的 API Key：

```json
{
  "api_key": "你的真实_API_KEY"
}
```

### 2. 如何使用

配置好 API Key 后，你可以通过自然语言让 AI 帮你审核合同。目前支持**两种提交方式**：

#### 方式 A：通过链接（URL）审核

直接把合同的公网访问链接发给 AI。

- **提示词示例**：“我是甲方视角，帮我审核一下这个合同：`https://example.com/contract.pdf`。”

#### 方式 B：通过文件上传审核

将文件放入你的工作区或直接拖给 AI，让 AI 读取并上传审核。

- **提示词示例**：“我是乙方视角，帮我审核一下当前目录下的 `租赁合同.pdf`。”

---

### 📝 注意事项

1. **视角选择**：务必告诉 AI 你是**甲方**还是**乙方**，这会影响审核结论。
2. **数量限制**：一次最多同时审核 **5** 份合同。
3. **大小限制**（针对文件上传方式）：所有文件总大小不能超过 **5MB**。
4. **支持格式**：PDF、Word 以及常见图片格式（jpg/png 等）。
