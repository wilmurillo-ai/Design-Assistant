# Voyage Itinerary Constraints

## Constraint Types
- Budget: daily and total budget targets
- Dietary: restrictions applied to all food recommendations
- Pace: relaxed/moderate/packed — controls activity density
- Walking: maximum daily walking distance
- Transit: preferred modes (walk, train, taxi, car)
- Hours: respect opening hours and reservation windows

## Optimization Rules
- Cluster activities by geography to minimize transit
- Schedule meals near activity locations
- Allow buffer time between activities
- Front-load high-priority items
- Flag items with uncertain availability
