# Vesper Signal Filtering

## Inclusion Criteria
- Actionable information requiring awareness or decision
- Outcomes with meaningful impact
- Changes affecting plans or schedule
- Opportunities produced from multiple converging signals
- Information useful for upcoming preparation

## Exclusion Criteria
- Routine background activity
- Events already experienced directly by the owner
- Internal reasoning, system processes, or skill telemetry
- Speculative or curiosity-only observations

## Evening-Specific Exclusions
- Past weather
- Summaries of meetings already attended
- Repeated items from morning briefing unless status changed

## Signal Sources
Calendar events, communication threads, financial logs, system artifacts, Chronicle records, Corvus pattern signals, journal entries.

## Corvus Opportunity Handling
When Corvus detects an opportunity through pattern analysis, Vesper surfaces only the resulting opportunity in natural language. The underlying pattern analysis is never mentioned.
