# 平台详解参考

本文档提供各主流网文平台的详细规则、变现政策和数据接口信息。

---

## 1. Amazon Kindle Direct Publishing (KDP)

### 平台概况

- **地区**：北美为主（美国/加拿大/英国/德国/日本等）
- **市场地位**：全球最大付费电子书平台
- **变现模式**：电子书销售 + Kindle Unlimited（KU）页面阅读分成

### 变现详情

| 收入来源 | 分成比例 | 说明 |
|---------|---------|------|
| 电子书销售 | 35% 或 70% | 70%需满足定价要求（$2.99~$9.99） |
| KU页面阅读 | 按页数分成 | 每页约 $0.004~$0.005 |

**KU推荐算法特点：**
-  BOR（Book Order Rate）— 新书曝光率
-  阅读完成率影响推荐权重
-  系列书比单本书有优势

### 内容规范

| 维度 | 要求 |
|------|------|
| 字数 | 无限制，建议6万~15万字 |
| 更新节奏 | 无强制要求，但持续更新有利于算法 |
| 敏感内容 | 禁止色情、暴力、仇恨内容 |
| 封面 | 独立1500×2500像素，300dpi |

### 数据采集来源

- **公开榜单**：Amazon畅销榜、New Release榜单
- **工具**：Jungle Scout、Publisher Rocket（竞品数据）
- **官网**：KDP Dashboard（作者后台数据需登录）

## 2. 番茄小说（字节跳动）

### 平台概况

- **地区**：中国
- **市场地位**：最大免费阅读平台，DAU超3000万
- **变现模式**：广告分成 + 读心思激励 + 保底计划

### 变现详情

| 收入来源 | 分成比例 | 说明 |
|---------|---------|------|
| 广告分成 | 50%~70% | 根据阅读时长和广告曝光 |
| 读心思奖励 | 平台补贴 | 优质内容额外奖励 |
| 新书扶持 | 平台补贴 | 新书期额外流量扶持 |

**保底计划：**
- 签约后享受平台保底收益
- 满足日更4000字以上可申请
- 优质作品保底千字50元起

### 内容规范

| 维度 | 要求 |
|------|------|
| 字数 | 无限制，建议10万~50万字 |
| 更新节奏 | 日更4000字以上有算法加成 |
| 更新时段 | 早8点、晚6点为最佳发布时间 |
| 敏感内容 | 审核较严格，涉政/涉黄绝对禁止 |
| 封面 | 竖版 256×384 或 512×768 |

### 数据采集来源

- **公开榜单**：番茄小说免费榜单、飙升榜
- **作者后台**：需登录获取详细数据
- **第三方工具**：七猫数据、起点中文网数据工具

## 3. 起点中文网（阅文集团）

### 平台概况

- **地区**：中国
- **市场地位**：付费网文最成熟平台，品牌价值最高
- **变现模式**：付费订阅 + 月票 + 打赏 + 改编版权

### 变现详情

| 收入来源 | 分成比例 | 说明 |
|---------|---------|------|
| 付费订阅 | 50% | 作者与平台分成 |
| 月票 | 月票前三有额外奖励 | 读者投票决定榜单 |
| 打赏 | 50%~70% | 取决于打赏类型 |
| 全勤奖 | 每日更新奖励 | 每日更新3000字奖150元 |

### 内容规范

| 维度 | 要求 |
|------|------|
| 字数 | 无限制，热门作品100万字以上 |
| 更新节奏 | 日更3000字以上 |
| 更新时段 | 晚8点后发布效果最佳 |
| 签约门槛 | 需内投或平台审核通过 |
| 敏感内容 | 审核严格，涉黄涉政禁止 |

### 数据采集来源

- **公开榜单**：起点中文网畅销榜、月票榜
- **起点数据**：创世中文网数据工具
- **作者后台**：需登录获取详细数据

## 4. 晋江文学城

### 平台概况

- **地区**：中国
- **市场地位**：女频绝对领先，改编版权价值高
- **变现模式**：付费订阅 + 营养液 + 改编版权

### 变现详情

| 收入来源 | 分成比例 | 说明 |
|---------|---------|------|
| 付费订阅 | 50% | vip章节按字数计费 |
| 营养液 | 1:1折算月票 | 读者灌溉营养液 |
| 收藏票 | 1:1折算月票 | 收藏书籍获得 |
| 改编版权 | 谈判 | 影视/游戏/动漫改编 |

### 内容规范

| 维度 | 要求 |
|------|------|
| 字数 | 无限制，10万~80万字常见 |
| 更新节奏 | 隔日更或周更更常见 |
| 更新时段 | 晚10点后发布 |
| 文风特点 | 细腻，感情线重要 |
| 敏感内容 | 审核严格，但BL/GL内容相对宽松 |

### 数据采集来源

- **公开榜单**：晋江文学城排行榜、积分榜
- **第三方工具**：晋江推文姬等
- **作者后台**：需登录获取详细数据

## 6. Royal Road

### 平台概况

- **地区**：北美/英语市场
- **市场地位**：海外网文爱好者聚集地
- **变现模式**：免费阅读 + Patreon打赏 + 付费解锁

### 变现详情

| 收入来源 | 说明 |
|---------|------|
| Patreon打赏 | 读者通过Patreon支持 |
| 付费解锁 | 前3~5章免费，后续付费 |
| 版权销售 | 有读者推荐机制 |

### 内容规范

| 维度 | 要求 |
|------|------|
| 字数 | 无限制，英文60000+ |
| 更新节奏 | 每日或每周更新 |
| 题材偏好 | 奇幻、科幻、穿越、系统流 |
| 文风特点 | 快节奏、章节短（1000~2000字） |

## 数据采集注意事项

### 合法的采集方式

✅ **公开数据：**
- 平台公开的排行榜（无需登录）
- 搜索引擎结果
- 社交媒体讨论（Twitter/微博/Reddit）
- 读者书评（脱敏后使用）

✅ **官方API（如有）：**
- Amazon KDP API
- 平台开放的数据接口

### 禁止的采集方式

❌ **禁止：**
- 破解平台反爬虫机制
- 入侵平台数据库
- 采集用户个人信息
- 绕过登录获取付费数据

### 各平台反爬虫难度

| 平台 | 难度 | 说明 |
|------|------|------|
| Amazon | 高 | 严格反爬虫，需代理池 |
| 番茄小说 | 中 | 有反爬虫，但可绕过 |
| 起点中文网 | 中高 | 需要cookie等 |
| 晋江 | 中 | 有基础反爬虫 |
| Royal Road | 低 | 相对宽松 |
| Wattpad | 高 | 严格反爬虫 |
