## 🔧 核心功能

### 1. Markdown 转微信公众号 HTML

```javascript
markdownToWechatHtml(markdown)
```

**转换规则：**
- ✅ `# 标题` → H1 居中
- ✅ `## 标题` → H2 左侧边框
- ✅ `### 标题` → H3 普通
- ✅ `**粗体**` → 加粗
- ✅ `` `代码` `` → 行内代码
- ✅ ```代码块``` → 代码块（带边框）
- ✅ `- 列表` → 圆点列表
- ✅ `|表格|` → HTML 表格
- ✅ `> 引用` → 引用块

**智能过滤：**
- ✅ `**text**` → `text`（移除粗体标记）
- ✅ `---` → 空（移除分割线）
- ✅ `--` → 空（移除多余符号）
- ✅ `>` → 空（移除引用标记）

## 💡 常见问题

### Q1: 草稿箱链接无法点击？
**A:** 这是微信公众号草稿箱的限制，编辑模式下链接无法点击。需要正式发布后才能点击导航链接。

### Q2: 如何预览效果？
**A:** 使用 `check-wechat-format.js` 生成预览文件，在浏览器模拟手机效果。

### Q3: 文档没有 chapters 目录怎么办？
**A:** 支持扁平结构！所有 `.md` 文件放在同一层即可，会自动按序号排序。

### Q4: 如何跳过某些文件？
**A:** 自动跳过以下文件：
- 包含"规划"的文件
- `collect*.js` 脚本
- `README.md`
- 非 `.md` 文件
