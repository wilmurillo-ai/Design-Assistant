## 📖 快速开始

### 1. 安装技能
```bash
openclaw skill install wechat-publisher
```

### 2. 配置公众号
```bash
openclaw skill config wechat-publisher
```

需要配置：
- `app_id`: 公众号 AppID
- `app_secret`: 公众号 AppSecret

### 3. 设置发布时间
```bash
openclaw schedule wechat-publisher 06:00
```

### 4. 发布测试
```bash
openclaw skill run wechat-publisher --test
```

## 📋 使用前准备

### 1. 公众号权限配置

**IP 白名单配置：**

1. 登录 https://mp.weixin.qq.com/
2. 设置 → 公众号设置 → 功能设置
3. 找到 IP 白名单
4. 添加当前服务器出口 IP

**获取出口 IP：**
```bash
curl http://ip-api.com/json/
```

### 2. 二维码图片上传

**上传到素材库：**

1. 登录公众号后台 → 素材管理 → 图片
2. 上传公众号二维码图片
3. 命名为 `qrcode.jpg`
4. 系统自动使用此图片

**图片要求：**
- 格式：JPG
- 尺寸：建议 200×200px 或更大
- 内容：清晰的公众号二维码

## 🚀 使用流程

### 日常发布（自动）

技能安装配置完成后，每天 06:00 自动发布：

1. **05:55** - 自动启动收集新闻
2. **06:00** - 自动发布到草稿箱
3. **06:01** - 写入结果文件
4. **06:05** - 汇报 DraftID

### 手动发布（测试）

```bash
# 发布测试
openclaw skill run wechat-publisher --test

# 查看发布结果
openclaw skill status wechat-publisher
```

### 查看草稿

发布成功后，登录公众号后台查看：
- 草稿箱 → 找到最新草稿
- 预览 → 发送到手机预览效果
- 编辑 → 可手动调整（一般不需要）

| 优先级 | 新闻源 | 类型 |

| High | TechCrunch AI | 国外 AI |

| Medium | Bloomberg Technology | 国际科技 |
| Low | 36 氪 AI | 国内新闻 |
| Low | 机器之心 | 国内新闻 |

### 去重规则

- **对比范围：** 30 天内发布的历史新闻
- **对比方式：** 标题 + 内容双重校验
- **去重逻辑：** 全量逐条对比（非一一对应）
- **缓存位置：** `D:\news\AI_早报_*.md`

### 字数要求

- 每条新闻内容 ≥80 字
- 全部中文，无英文
- 自动翻译英文新闻

## ⚠️ 常见问题

### 1. API 调用失败

**错误：** `40164 invalid ip`

**原因：** IP 白名单未配置

**解决：**
1. 获取当前出口 IP
2. 添加到公众号 IP 白名单
3. 等待 5 分钟后重试

### 2. 二维码不显示

**原因：** 素材库无 qrcode.jpg 或 URL 错误

**解决：**
1. 登录公众号后台 → 素材管理 → 图片
2. 上传二维码图片，命名为 `qrcode.jpg`
3. 重新发布

### 3. 新闻重复

**原因：** 缓存文件损坏或去重逻辑异常

**解决：**
1. 检查 `D:\news` 缓存文件
2. 清理损坏的缓存文件
3. 重新发布

### 4. 字体太小

**说明：** V2.0.0 已整体加大 2 号字体

**解决：** 升级到 V2.0.0
```bash
openclaw skill update wechat-publisher
```
