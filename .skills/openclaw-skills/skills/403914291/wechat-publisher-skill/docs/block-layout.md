## 🖼️ 二维码规范

### 图片要求
- **名称：** qrcode.jpg
- **位置：** 公众号素材库
- **尺寸：** 200×200px
- **格式：** JPG

### 样式要求
- 白色背景框
- padding: 15px
- border-radius: 8px
- box-shadow: 0 2px 8px rgba(0,0,0,0.15)

### URL 格式
```
https://mmbiz.qpic.cn/mmbiz_jpg/{media_id}/0?wx_fmt=jpeg
```

## 📝 内容规范

### 新闻数量
- 今日摘要：8 条要点
- 国外 AI：8 条
- 国际动态：8 条
- 国内大厂：8 条
- 其他科技：8 条
- **总计：32 条**

### 字数要求
- 每条新闻内容 ≥80 字
- 全部中文，无英文

### 去重规则
- 与 D:\news 缓存对比
- 30 天内不重复
- 标题 + 内容双重校验

## 🚀 发布流程

1. **收集新闻**：6 个新闻源实时抓取
2. **合并翻译**：英文新闻翻译 + 去重
3. **语义检查**：30 天历史对比
4. **生成 HTML**：块布局 v3.0 模板
5. **发布**：创建草稿 + 缓存
