# 模板说明 - 块布局 v3.0

**模板名称：** block-v3  
**版本：** v3.0（固定版）  
**状态：** ✅ 官方唯一推荐模板  

---

## 📋 模板概述

**块布局 v3.0** 是微信公众号发布技能的官方固定模板，所有用户统一使用此版本。

### 设计理念

- **统一视觉：** 所有用户使用相同模板，保证品牌一致性
- **优化阅读：** 字体加大 2 号，提升阅读体验
- **强化关注：** 底部二维码加大，方便用户关注
- **智能去重：** 30 天历史对比，避免重复新闻

### 新闻数量

| 板块 | 数量 | 说明 |
|------|------|------|
| 今日摘要 | 8 条 | 要点列表 |
| 国外 AI | 8 条 | 投融资、大模型、研究 |
| 国际动态 | 8 条 | 政策、投资、人才 |
| 国内大厂 | 8 条 | 百度、阿里、腾讯等 |
| 其他科技 | 8 条 | 手机、汽车、航天等 |
| **总计** | **32 条** | - |

### 字数要求

- 每条新闻内容 ≥80 字
- 全部中文，无英文
- 自动翻译英文新闻

### 去重规则

- 对比范围：30 天内发布的历史新闻
- 对比方式：标题 + 内容双重校验
- 对比逻辑：全量逐条对比
- 缓存位置：`D:\news\AI_早报_*.md`

## 🖼️ 二维码规范

### 图片要求

| 项目 | 要求 |
|------|------|
| 名称 | qrcode.jpg |
| 位置 | 公众号素材库 |
| 格式 | JPG |
| 尺寸 | 建议 200×200px 或更大 |
| 内容 | 清晰的公众号二维码 |

### 上传步骤

1. 登录公众号后台 → 素材管理 → 图片
2. 上传二维码图片
3. 命名为 `qrcode.jpg`
4. 系统自动使用此图片

### URL 格式

```
https://mmbiz.qpic.cn/mmbiz_jpg/{media_id}/0?wx_fmt=jpeg
```

## 📝 使用示例

### 安装模板

```bash
# 安装技能（自动使用块布局 v3.0）
openclaw skill install wechat-publisher

# 配置技能
openclaw skill config wechat-publisher

# 设置发布时间
openclaw schedule wechat-publisher 06:00
```

### 发布测试

```bash
# 手动发布测试
openclaw skill run wechat-publisher --test

# 查看发布状态
openclaw skill status wechat-publisher
```

### 查看结果

发布成功后，登录公众号后台：
1. 草稿箱 → 找到最新草稿
2. 预览 → 发送到手机预览效果
3. 编辑 → 可手动调整（一般不需要）

## ⚠️ 注意事项

### 1. 模板固定

- 块布局 v3.0 是官方固定模板
- 不建议修改模板样式
- 如需自定义，请联系技术支持

### 2. 二维码图片

- 必须上传到素材库
- 命名为 `qrcode.jpg`
- 尺寸建议 200×200px 或更大
- 保证二维码清晰可扫描

### 3. 字体大小

- V2.0.0 已整体加大 2 号
- 不建议再调整字体大小
- 影响阅读体验

### 4. 新闻数量

- 固定 32 条（4 类×8 条）
- 不建议调整数量
- 影响版面美观
