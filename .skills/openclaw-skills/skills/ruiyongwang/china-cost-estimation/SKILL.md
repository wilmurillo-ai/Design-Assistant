# 中国建设工程快速估算系统

## 技能定位

本技能基于最新国家标准和地方规定，提供专业的建设工程投资估算和费用计算服务：

### 主要依据标准

| 标准编号 | 标准名称 | 实施日期 | 适用范围 |
|---------|---------|---------|---------|
| GB/T 50500-2024 | 建设工程工程量清单计价标准 | 2025年9月1日 | 全国 |
| 建标[2013]44号 | 建筑安装工程费用项目组成 | 现行有效 | 全国 |
| 苏建价〔2024〕 | 江苏省建设工程费用定额 | 2024年 | 江苏省 |
| 苏建价〔2025〕第12号 | 江苏省建设工程费用组成规则 | 2026年6月1日 | 江苏省 |

```
建设项目总投资

│   ├── 工程费用

│   │   ├── 安装工程费
│   │   └── 设备购置费
│   ├── 工程建设其他费用
│   │   ├── 建设用地费
│   │   ├── 建设管理费
│   │   ├── 前期工作咨询费
│   │   ├── 勘察设计费
│   │   ├── 监理费
│   │   ├── 招标代理费
│   │   ├── 工程造价咨询费
│   │   └── 其他专项费用
│   └── 预备费
```

### 二、建筑安装工程费用组成（建标[2013]44号）

#### 按费用构成要素划分

| 费用项目 | 计算基础 | 江苏省2024参考费率 | 说明 |
|---------|---------|-------------------|------|
| **人工费** | 工日消耗量×人工单价 | - | 含工资、奖金、津贴、社保 |
| **材料费** | 材料消耗量×材料单价 | - | 含原价、运杂费、损耗、采购保管费 |
| **施工机具使用费** | 台班消耗量×台班单价 | - | 含折旧费、检修费、人工费、燃料动力费 |
| **企业管理费** | 人工费+机械费 | 25%-35% | 管理人员工资、办公费、差旅费等 |
| **利润** | 人工费+机械费 | 12%-15% | 施工企业完成工程获得的盈利 |
| **规费** | 人工费 | 25%-30% | 五险一金等法定费用 |
| **税金** | 税前造价 | 9% | 增值税销项税额 |

#### 按造价形成划分

| 费用项目 | 计算基础 | 参考比例 | 说明 |
|---------|---------|---------|------|
| **分部分项工程费** | ∑(工程量×综合单价) | 70%-80% | 实体项目费用 |
| **措施项目费** | 分部分项工程费×费率 | 5%-15% | 技术措施、组织措施 |
| **其他项目费** | 按实际发生 | 2%-5% | 暂列金额、暂估价、计日工 |
| **规费** | 人工费×费率 | 3%-5% | 法定费用 |
| **税金** | 税前造价×税率 | 9% | 增值税 |

## 执行逻辑与工作流

### 工作流一：建设项目投资估算

**触发场景**：用户需要进行项目投资估算

**执行步骤**：
1. 识别项目类型和规模
2. 确定估算指标（参考数据库或用户提供）
3. 计算工程费用 = 建筑面积 × 综合估算指标
4. 计算工程建设其他费 = 工程费用 × 费率（通常15%-25%）
5. 计算基本预备费 = (工程费用+其他费) × 费率（5%-8%）
6. 汇总建设投资

**计算公式**：
```
建设投资 = 工程费用 + 工程建设其他费 + 预备费
工程费用 = 建筑面积 × 综合估算指标
工程建设其他费 = 工程费用 × (15% ~ 25%)
基本预备费 = (工程费用 + 工程建设其他费) × (5% ~ 8%)
```

### 工作流二：建筑安装工程费计算

**触发场景**：用户需要计算建安工程费

**执行步骤**：
1. 读取 `references/cost-composition-rules.json`
2. 根据工程类别确定取费标准
3. 计算分部分项工程费
4. 计算措施项目费
5. 计算其他项目费
6. 计算规费
7. 计算税金
8. 汇总建安工程造价

**计算公式**：
```
建安工程造价 = 分部分项工程费 + 措施项目费 + 其他项目费 + 规费 + 税金
分部分项工程费 = ∑(工程量 × 综合单价)
综合单价 = 人工费 + 材料费 + 机械费 + 企业管理费 + 利润 + 一定范围的风险费用
```

### 工作流三：费用构成分析

**触发场景**：用户需要分析造价构成

**执行步骤**：
1. 读取费用组成规则
2. 分解各项费用占比
3. 对比行业标准
4. 识别异常项目
5. 给出优化建议

## 江苏省特殊规定

### 一、费用定额适用范围

江苏省建设工程费用定额（2024版）适用于：
- 江苏省行政区域内新建、改建、扩建的建设工程
- 与建筑工程配套的电气、给排水、通风空调等安装工程
- 市政道路、桥涵、管网、园林绿化等市政工程

### 二、计费程序

```
第一步：计算分部分项工程费
第二步：计算措施项目费
第三步：计算其他项目费
第四步：计算规费
第五步：计算税前造价 = 一+二+三+四
第六步：计算税金 = 五 × 税率
第七步：计算工程造价 = 五+六
```

### 三、不可竞争费用

根据江苏省规定，以下费用为不可竞争费用：
- 安全文明施工费
- 规费（社会保险费、住房公积金、环境保护税）
- 税金
