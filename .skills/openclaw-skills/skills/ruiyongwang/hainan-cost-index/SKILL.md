# 海南省建设工程造价指标专家

## 技能定位

本技能基于《2024年版海南省建设工程典型案例造价指标》（海南省建设标准定额站，2025年5月发布），
  提供专业的建设工程造价指标查询、分析和应用服务。

数据来源：海南省建设标准定额站官方发布的23个典型案例，涵盖：
- **房屋建筑工程**：19个案例（居住、办公、科研、交通、教育、文体、医疗、工业等）
- **市政工程**：4个案例（道路、桥涵、管网、绿化等）

编制依据：
- 《建设工程工程量清单计价规范》（GB50500-2013）
- 各专业工程工程量计算规范（GB50854～50858-2013）
- 海南省相关计价依据
- GB/T 50500-2024《建设工程工程量清单计价标准》

### 主要触发词
- 海南造价指标 / 海南省造价

- 造价对比 / 指标对比
- 经济指标 / 工程量指标

### 使用场景类型
1. **投资估算**：根据项目类型、规模估算投资
2. **指标查询**：查询特定工程类型的造价指标
3. **对比分析**：多项目造价指标对比
4. **可行性研究**：项目投资测算
5. **造价审核**：对比审核造价合理性

## 执行逻辑与工作流

### 工作流一：投资估算

**触发场景**：用户需要根据项目类型和规模估算投资

**执行步骤**：
1. 识别项目类型（居住/办公/教育/医疗等）
2. 确定建筑面积或功能性单位
3. 选择适配的造价指标（低/中/高）
4. 考虑调整因素：
   - 地区差异（海口/三亚/其他市县）
   - 时间差异（价格年份调整）
   - 装修标准差异
   - 结构形式差异
5. 计算投资估算并给出区间

**输出格式**：
```markdown
## 投资估算报告

**项目类型**：[类型]
**建筑面积**：[面积] ㎡
**参考造价指标**：[低限] - [高限] 元/㎡
**投资估算**：[低限] - [高限] 万元

**造价构成参考**：
- 建筑工程：[占比]%，约 [金额] 万元
- 安装工程：[占比]%，约 [金额] 万元
- 装饰工程：[占比]%，约 [金额] 万元

**调整建议**：
- [根据项目特殊性的调整说明]

**参考案例**：
- [案例名称]：[造价指标] 元/㎡
```

### 工作流二：造价指标查询

**触发场景**：用户查询特定类型工程的造价指标

**执行步骤**：
1. 读取 `references/hainan-cost-database.json`
2. 根据工程类型筛选匹配案例
3. 提取关键指标数据
4. 对比分析（如有多案例）
5. 输出结构化指标表

### 工作流三：造价对比分析

**触发场景**：用户需要对比多个项目或审核造价合理性

**执行步骤**：
1. 收集待对比项目的造价数据
2. 归一化处理（统一计算基数）
3. 对比维度：
   - 综合造价指标
   - 分部分项工程占比
   - 人工/材料/机械比例
   - 单位工程量指标
4. 识别异常差异并分析原因
5. 给出审核意见

## 输出规范

### 标准输出模块

每次响应应根据场景选择以下模块组合：

- **[估算]** 投资估算结果与区间
- **[指标]** 造价指标数据表
- **[对比]** 多项目对比分析
- **[构成]** 造价构成分析
- **[建议]** 调整建议与注意事项
- **[参考]** 参考案例与数据来源

## 边界说明

- 本技能数据基于海南省2024年版造价指标，其他地区仅供参考
- 造价指标为参考值，实际造价受多种因素影响
- 投资估算结果仅供参考，不构成正式造价咨询意见
- 涉及重大投资决策建议委托专业造价咨询机构
