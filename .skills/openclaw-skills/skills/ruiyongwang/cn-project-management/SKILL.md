# 中国建设工程项目管理

## 概述

本技能帮助项目经理、工程管理人员实现建设工程项目全过程管理，基于**《建设工程项目管理规范》GB/T 50326** 及相关法规。

**核心能力：**
- 项目管理规划
- 进度管理（关键路径法、挣值管理）
- 质量管理（PDCA循环）
- 安全管理
- 合同管理
- 风险管理

## 第二部分：进度管理

### 2.1 关键路径法（CPM）

**基本概念：**

```
网络计划参数计算：

ES（最早开始时间）：紧前工作全部完成后，本工作可以开始的最早时刻
EF（最早完成时间）：ES + 工作持续时间
LS（最迟开始时间）：不影响总工期的前提下，本工作最迟必须开始的时刻
LF（最迟完成时间）：LS + 工作持续时间
TF（总时差）：LS - ES 或 LF - EF
FF（自由时差）：不影响紧后工作最早开始时间的前提下，本工作可推迟的时间

关键路径判定：TF = 0 的工作组成的路径
```

**计算示例：**

```markdown
## XX项目网络计划计算

工作清单：
| 工作代号 | 工作名称 | 持续时间(天) | 紧前工作 |
|:--------:|----------|:------------:|----------|
| A | 施工准备 | 15 | - |
| B | 土方开挖 | 30 | A |
| C | 基础施工 | 45 | B |
| D | 主体结构 | 120 | C |
| E | 装饰装修 | 90 | D |
| F | 安装工程 | 60 | D |
| G | 室外工程 | 30 | E,F |
| H | 竣工验收 | 15 | G |

参数计算：
| 工作 | 持续时间 | ES | EF | LS | LF | TF | 关键工作 |
|:----:|:--------:|:--:|:--:|:--:|:--:|:--:|:--------:|
| A | 15 | 0 | 15 | 0 | 15 | 0 | ✓ |
| B | 30 | 15 | 45 | 15 | 45 | 0 | ✓ |
| C | 45 | 45 | 90 | 45 | 90 | 0 | ✓ |
| D | 120 | 90 | 210 | 90 | 210 | 0 | ✓ |
| E | 90 | 210 | 300 | 210 | 300 | 0 | ✓ |
| F | 60 | 210 | 270 | 240 | 300 | 30 | |
| G | 30 | 300 | 330 | 300 | 330 | 0 | ✓ |
| H | 15 | 330 | 345 | 330 | 345 | 0 | ✓ |

关键路径：A → B → C → D → E → G → H
总工期：345天
```

### 2.2 挣值管理（EVM）

**核心指标：**

```
基本参数：
PV（计划值）：计划完成工作的预算成本
EV（挣值）：实际完成工作的预算成本
AC（实际成本）：实际完成工作的实际成本

绩效指标：
SV（进度偏差）= EV - PV
    SV > 0：进度超前
    SV < 0：进度滞后

CV（成本偏差）= EV - AC
    CV > 0：成本节约
    CV < 0：成本超支

SPI（进度绩效指数）= EV / PV
    SPI > 1：进度超前
    SPI < 1：进度滞后

CPI（成本绩效指数）= EV / AC
    CPI > 1：成本节约
    CPI < 1：成本超支

预测指标：
BAC（完工预算）：项目总预算
EAC（完工估算）= BAC / CPI
ETC（完工尚需估算）= EAC - AC
VAC（完工偏差）= BAC - EAC
```

**应用示例：**

```markdown
## XX项目挣值分析报告

项目基本信息：
- 合同价款：11,256.80万元
- 计划工期：540天
- 当前时间：第270天（完成50%）

挣值数据：
| 指标 | 数值（万元） | 说明 |
|------|-------------:|------|
| PV | 5,628.40 | 计划完成产值 |
| EV | 5,180.00 | 实际完成产值（预算） |
| AC | 5,450.00 | 实际发生成本 |

绩效分析：
| 指标 | 公式 | 计算 | 结果 | 评价 |
|------|------|------|------|------|
| SV | EV-PV | 5,180-5,628.40 | -448.40万元 | 🔴 滞后 |
| CV | EV-AC | 5,180-5,450 | -270.00万元 | 🔴 超支 |
| SPI | EV/PV | 5,180/5,628.40 | 0.92 | 🔴 滞后8% |
| CPI | EV/AC | 5,180/5,450 | 0.95 | 🔴 超支5% |

预测分析：
| 指标 | 公式 | 计算 | 结果 |
|------|------|------|------|
| EAC | BAC/CPI | 11,256.80/0.95 | 11,849.26万元 |
| ETC | EAC-AC | 11,849.26-5,450 | 6,399.26万元 |
| VAC | BAC-EAC | 11,256.80-11,849.26 | -592.46万元 |

结论：
1. 进度滞后8%，需采取措施追赶
2. 成本超支5%，预计最终超支592万元
3. 建议优化施工方案，控制后续成本

纠偏措施：
1. 增加人员和设备投入，追赶进度
2. 优化采购方案，降低材料成本
3. 加强现场管理，减少返工和浪费
```

### 3.2 质量控制要点

**分部分项工程质量控制：**

| 分部工程 | 控制重点 | 验收标准 |
|---------|---------|---------|
| 地基基础 | 地基承载力、桩基检测 | GB 50202 |
| 主体结构 | 混凝土强度、钢筋保护层 | GB 50204 |
| 装饰装修 | 面层平整度、阴阳角 | GB 50210 |
| 屋面工程 | 防水层施工、排水坡度 | GB 50207 |
| 安装工程 | 管道试压、电气测试 | GB 50242/50303 |

**质量检查程序：**

```
三检制度：

班组自检 → 质量员专检 → 监理验收

1. 班组自检
   - 施工完成后，班组长组织自检
   - 发现问题及时整改
   - 自检合格后报专检

2. 质量员专检
   - 项目质量员进行检查
   - 填写检验批质量验收记录
   - 专检合格后报监理

3. 监理验收
   - 监理工程师验收
   - 签署验收意见
   - 不合格项整改后复验
```

### 3.3 质量问题处理

```markdown
## 质量问题处理流程

1. 发现质量问题
   └── 现场检查、监理巡视、检测发现

2. 问题定性
   ├── 一般质量缺陷：不影响结构安全和使用功能
   └── 严重质量问题：影响结构安全或使用功能

3. 分析原因
   ├── 人：操作不规范、技能不足
   ├── 机：设备故障、精度不够
   ├── 料：材料不合格、进场检验不严
   ├── 法：工艺不当、方案缺陷
   └── 环：环境条件不满足要求

4. 制定处理方案
   ├── 返修：修复至合格
   ├── 返工：重新施工
   ├── 让步接收：经设计同意，降级使用
   └── 报废：拆除重做

5. 实施处理
   └── 按批准方案实施

6. 验收确认
   └── 监理验收合格后进入下道工序

7. 记录归档
   └── 质量问题处理记录存档
```

### 5.2 索赔管理

**索赔程序：**

```markdown
## 工程索赔流程（28天规则）

承包人索赔程序：
1. 索赔事件发生后28天内，向发包人提交索赔意向通知书
2. 发出索赔意向通知书后28天内，提交正式索赔报告
3. 索赔报告内容包括：
   - 索赔理由
   - 索赔证据
   - 索赔金额计算
   - 工期顺延天数

发包人处理：
1. 收到索赔报告后28天内予以答复
2. 未答复视为认可
3. 如有异议，28天内通知承包人

常见索赔情形：
| 索赔原因 | 索赔内容 | 证据材料 |
|---------|---------|---------|
| 发包人延迟提供场地 | 工期+费用 | 通知函、会议纪要 |
| 图纸提供延迟 | 工期+费用 | 收发记录 |
| 工程变更 | 工期+费用 | 变更单、签证 |
| 材料价格上涨 | 费用 | 价格信息、采购合同 |
| 停水停电 | 工期+费用 | 施工日志、记录 |
| 不可抗力 | 工期 | 气象记录、政府文件 |
```
