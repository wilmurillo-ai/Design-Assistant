# 建设工程商事调解协议模板

---

## 使用说明

本模板适用于建设工程领域商事调解达成一致后起草正式调解协议。
使用前请核对：
- [ ] 当事方名称、法人代表信息准确
- [ ] 争议金额已明确
- [ ] 付款方式、期限已约定
- [ ] 各方代表均有充分授权
- [ ] 协议条款无歧义

---

## 模板一：工程款支付调解协议

```
建设工程施工合同纠纷调解协议书

申请方（承包人）：【全称】
住所：【注册地址】
法定代表人：【姓名】，职务：【职务】
委托代理人：【姓名】，【律师事务所/单位名称】

被申请方（发包人）：【全称】
住所：【注册地址】
法定代表人：【姓名】，职务：【职务】
委托代理人：【姓名】，【律师事务所/单位名称】

调解机构：【调解机构全称】
调解员：【姓名】

一、争议基本情况

申请方与被申请方于【日期】签订《【合同名称】》，合同编号：【编号】，
工程项目为：【项目名称】，合同价款为人民币【金额】元整（¥【数字】）。

双方因【争议简要描述：如工程款支付、变更签证等】产生纠纷，
申请方主张：【简述申请方诉求及金额】；
被申请方主张：【简述被申请方立场】。

二、调解结果

经【调解机构】调解员主持调解，双方当事人在自愿、平等的基础上，
达成如下协议：

（一）被申请方同意向申请方支付工程款（含结算款、违约金等）
共计人民币【金额】元整（¥【数字】），具体付款安排如下：

第一期：人民币【金额】元，于【具体日期】前支付至申请方指定账户；
账户信息：
    户名：【户名】
    开户行：【银行名称及支行】
    账号：【账号】

第二期：人民币【金额】元，于【具体日期】前支付至上述账户；
……

（二）申请方确认，上述款项支付完毕后，双方就本合同项下的所有
争议一次性了结，双方互不追究其他责任。

（三）如被申请方未按上述时间节点付款，申请方有权就未付款项
按照年利率【利率，如LPR+2%】计收逾期违约金，自逾期之日起
至实际支付之日止。

三、其他事项

1. 本协议自双方签署之日起生效。
2. 双方均有权申请【调解机构所在地】人民法院对本协议进行司法确认。
3. 本协议一式四份，申请方、被申请方各执一份，调解机构留存两份，
   具有同等法律效力。

申请方代表签字：_____________ 日期：_______年___月___日
（加盖公章）

被申请方代表签字：___________ 日期：_______年___月___日
（加盖公章）

调解机构（盖章）：___________
调解员签字：_________________
日期：_______年___月___日
```

## 模板二：工期/质量争议调解协议（要素版）

**核心要素清单**（在模板一基础上调整以下条款）：

**工期违约金争议**：
```
（一）双方确认，涉案工程实际竣工日期为【日期】，
合同约定竣工日期为【日期】，延误【N】日历天。

（二）经调解，双方同意就工期延误事项处理如下：
    1. 被申请方豁免申请方工期违约金计人民币【金额】元；
       或：被申请方收取工期违约金计人民币【金额】元，
       由申请方于【日期】前支付。
    2. 双方就本合同项下工期相关争议一次性了结。
```

**质量缺陷修复争议**：
```
（一）双方确认，工程存在以下质量缺陷：
    1. 【缺陷部位及描述】；
    2. 【缺陷部位及描述】。

（二）经调解，双方同意采用以下方式处理：
    □ 申请方（承包人）负责修复，修复完成时间：【日期】；
      发包人确认修复合格后，支付剩余工程款人民币【金额】元。
    
    □ 由第三方【单位名称】负责修复，修复费用预估为人民币【金额】元，
      由【责任方】承担，于【日期】前支付至第三方修复单位账户。
    
    □ 双方同意对工程价款作如下调整：在原合同结算价基础上
      扣减人民币【金额】元，作为质量瑕疵折价处理，双方对
      质量争议不再追究。
```

## 模板三：调解终止证明（调解不成功时使用）

```
调解终止证明书

本调解机构受理【申请方名称】与【被申请方名称】就
【工程项目名称】建设工程施工合同纠纷调解一案，案号：【案号】。

调解机构于【日期】至【日期】期间，组织双方进行了共【N】次调解。
因双方分歧较大，未能达成调解协议，本调解机构特此出具终止证明。

双方当事人可依据相关法律规定，向有管辖权的人民法院提起诉讼
或向约定的仲裁机构申请仲裁。

调解机构（盖章）：__________
日期：_______年___月___日
```

---

## 司法确认申请要点

调解协议签署后，当事人如需申请司法确认，应向以下法院申请：
- 申请调解的调解组织所在地基层人民法院
- 需在协议生效后30日内提出申请

**申请材料**：
1. 司法确认申请书（双方共同提交）
2. 调解协议原件
3. 当事人身份证明/营业执照
4. 授权委托书（委托他人申请的）
5. 调解机构资质证明
