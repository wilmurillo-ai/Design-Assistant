# 建设工程争议类型详细分类与特征

## 一、合同类争议

### 1.1 工程款支付争议

**特征**：发包人拒付或拖付工程款，是最常见的工程纠纷类型。

**常见原因**：
- 竣工结算未完成
- 质量异议导致拒付
- 资金链断裂
- 发包人恶意拖延
- 审计程序未完成

**关键证据**：
- 施工合同（含付款条款）
- 竣工验收报告
- 结算报告及审计意见
- 催款函件往来记录
- 工程量清单与计价书

**法律依据**：
- 《民法典》第807条：建设工程的价款就该工程折价或拍卖的价款优先受偿
- 《民法典》第779条：承包人交付工作成果后，定作人应当支付报酬
- GF-2017-0201 第12条：合同价款与支付

**调解要点**：
- 优先确定已完工程量及对应价款
- 区分争议金额与无争议金额，先推动无争议部分支付
- 评估质量异议的合理性

---

### 1.2 变更签证争议

**特征**：施工过程中因设计变更、现场签证引发的价款争议。

**常见原因**：
- 变更指令未书面确认
- 签证金额双方分歧
- 变更后价格未协商一致
- 监理拒绝签认

**关键证据**：
- 变更指令单（书面/口头记录）
- 现场签证单
- 监理日志
- 往来邮件/微信记录
- 变更前后工程量对比

**法律依据**：
- GF-2017-0201 第10条：变更
- GF-2017-0201 第13条：索赔
- GB/T 50500-2024 第9章：合同价款调整

**调解要点**：
- 还原变更指令的发出过程（谁授权、何时、如何传达）
- 按计价规则重新核算变更价款
- 区分有争议变更和无争议变更

---

### 1.3 索赔争议

**特征**：承包人主张因发包人原因造成的费用增加和工期损失。

**常见索赔原因**：
- 图纸延误或图纸错误
- 业主提供的材料设备延误
- 暂停施工指令
- 恶劣气候（超过合同约定标准）
- 不可预见地质条件

**法律依据**：
- GF-2017-0201 第13.1款：承包人的索赔
- GF-2017-0201 第13.2款：发包人的索赔

**调解要点**：
- 核实索赔通知的时效性（28天通知要求）
- 评估事件与损失之间的因果关系
- 区分可索赔费用和不可索赔费用

---

## 二、工期类争议

### 2.1 工期延误责任认定

**特征**：工程未按合同约定工期竣工，各方互相推责。

**责任方识别框架**：

| 延误原因 | 责任归属 | 处理原则 |
|---------|---------|---------|
| 设计图纸延误 | 发包人 | 工期顺延+费用补偿 |
| 甲供材料延误 | 发包人 | 工期顺延+费用补偿 |
| 承包人自身原因 | 承包人 | 支付逾期违约金 |
| 不可抗力 | 共担 | 工期顺延，费用各自承担 |
| 分包商原因 | 承包人 | 内部追偿 |

**关键分析工具**：关键路径法（CPM）分析，识别影响竣工日期的关键线路延误。

**法律依据**：
- GF-2017-0201 第7条：工期和进度
- GF-2017-0201 第17条：不可抗力

**调解要点**：
- 重建施工进度计划（投标vs实际）
- 识别各延误事件的责任方及影响工期
- 计算并发延误时的责任分担

---

### 2.2 工期赶工费用争议

**特征**：发包人要求赶工但未支付赶工费，或赶工费金额争议。

**调解要点**：
- 确认赶工指令的合法性和书面性
- 核算实际赶工措施费（加班费、设备增加等）
- 参照市场行情评估赶工费合理性

---

## 三、质量类争议

### 3.1 工程缺陷责任争议

**保修期标准**（《建设工程质量管理条例》第40条）：

| 工程部位 | 最低保修期 |
|---------|----------|
| 地基基础 | 设计文件规定的工程合理使用年限 |
| 主体结构 | 设计文件规定的工程合理使用年限 |
| 屋面防水 | 5年 |
| 供热供冷系统 | 2个采暖期、供冷期 |
| 给排水管道 | 2年 |
| 电气管线 | 2年 |
| 装修工程 | 2年 |

**归因分析**：

| 缺陷表现 | 可能责任方 |
|---------|----------|
| 结构裂缝 | 设计/施工/材料 |
| 防水渗漏 | 施工工艺/材料质量 |
| 装饰脱落 | 施工/维护使用 |
| 设备故障 | 设备供应商/安装 |

**调解要点**：
- 委托第三方检测机构出具鉴定报告
- 区分施工缺陷、设计缺陷、材料缺陷
- 评估修复方案及费用

---

## 四、造价类争议

### 4.1 清单计价偏差争议

**常见偏差类型**：
- 工程量清单漏项
- 综合单价组成争议
- 暂估价的调整
- 材料价格涨跌调差

**法律依据**：
- GB/T 50500-2024 第5章：合同价款的约定
- GB/T 50500-2024 第9章：合同价款调整
- GB/T 50500-2024 第11章：竣工结算与支付

**调解要点**：
- 对照清单规范核查工程量计算规则
- 复核综合单价组成要素
- 按合同约定的价格调整方式处理材料价差

---

## 五、争议复杂度评估矩阵

| 评估维度 | 简单（1分） | 中等（3分） | 复杂（5分） |
|---------|-----------|-----------|-----------|
| 争议金额 | <100万 | 100-1000万 | >1000万 |
| 争议焦点数 | 1-2个 | 3-5个 | >5个 |
| 当事方数量 | 2方 | 3-4方 | >4方 |
| 证据完整性 | 齐全 | 部分缺失 | 严重缺失 |
| 关系紧张度 | 友好 | 一般 | 对立 |
| 时效紧迫性 | 充裕 | 适中 | 紧迫 |

**总分判断**：
- 6-12分：简单争议，促进式调解，1-2次会议
- 13-20分：中等争议，混合式调解，3-5次会议
- 21-30分：复杂争议，评估式调解，需专家支持
