## 🚀 快速开始

### 安装

```bash
# 通过 SkillHub 安装
skillhub install qiaomai-skills

# 或通过 OpenClaw 安装
openclawmp install qiaomai-skills
```

### 初始化配置

```bash
# 一键配置向导
qiaomai init

# 或使用默认配置
qiaomai init --default
```

### 使用示例

```python
# 自然语言交互
qiaomai "帮我生成一份项目报告"

# 智能执行
qiaomai.execute("分析数据并生成图表", mode="smart")

# 记忆检索
qiaomai.memory.retrieve("上次的项目方案")

# 知识图谱查询
qiaomai.kg.query("建设工程合同相关法规")
```
