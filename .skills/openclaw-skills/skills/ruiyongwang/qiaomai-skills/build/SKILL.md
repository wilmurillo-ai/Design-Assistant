## 🚀 快速开始

### 安装

```bash
# 一键安装
openclawmp install qiaomai-skills

# 或使用 Asset ID
openclawmp install skill/{asset_id}
```

### 初始化

```bash
# 交互式配置向导
qiaomai init

? 选择使用场景: (Use arrow keys)
❯ 研究分析
  软件开发
  日常办公
  全部功能

? 是否启用高级功能: Yes
✅ 配置完成！已为您优化默认设置
```

### 使用

```bash
# 自然语言指令
qiaomai "帮我研究一下AI发展趋势"

# 参数化指令
qiaomai research "AI发展趋势" --depth=5 --format=report

# 启动交互模式
qiaomai chat

# 启动可视化仪表盘
qiaomai dashboard
```

## 💡 核心能力

### 1. 自然语言交互

```bash

$ qiaomai "分析一下这份合同的风险"
```

## 🛠️ 高级功能

### 智能报告生成

```bash
# 自动生成多格式报告
qiaomai report generate \
  --source "research_result.json" \
  --template "academic" \
  --format "pdf,markdown,html"

# 输出:
# - report.pdf (学术格式，带引用)
# - report.md (Markdown，便于编辑)
# - report.html (交互式网页)
```

### 类案检索

```bash
# 法律/工程/医疗类案检索
qiaomai case search \
  --domain "construction" \
  --query "工程款纠纷 工期延误" \
  --top 5

```

## 🔧 配置

### 配置文件

```yaml
# ~/.qiaomai/config.yaml
version: "1.0.0"

# 用户偏好
user:
  name: ""
  default_mode: "balanced"  # fast/balanced/thorough
  language: "zh-CN"

# 功能开关
features:
  memory: true
  knowledge_graph: true
  visualization: true
  prediction: true

# 性能设置
performance:
  cache_size: "1GB"
  max_workers: 10
  timeout: 30

# 集成配置
integrations:
  openclaw:
    enabled: true
  perplexity:
    enabled: true
    fallback: true
  grok:
    enabled: true
```

### 环境变量

```bash
# API Keys
export QIAOMAI_OPENCLAW_TOKEN=""
export QIAOMAI_PERPLEXITY_KEY=""
export QIAOMAI_GROK_KEY=""

# 性能调优
export QIAOMAI_WORKERS=10
export QIAOMAI_CACHE_SIZE="1GB"
```
