## 🛠️ 自动构建

### 运行构建脚本

```bash
cd ~/.openclaw/skills/qiaomai-skills
python scripts/publish_skillhub.py
```

构建完成后，会在 `dist/` 目录生成：
- `qiaomai-skills-1.0.0.zip` - 发布包
- `manifest.json` - 发布清单

## 📤 手动发布步骤

### 步骤 1: 访问开发者中心

打开浏览器访问：
```
https://skillhub.cn/developer
```

### 步骤 2: 登录账号

使用 dlh365 账号登录

### 步骤 3: 创建新技能

点击 **"发布新技能"** 或 **"上传技能"**

### 步骤 4: 填写基本信息

| 字段 | 填写内容 |
|------|----------|
| **技能名称** | 荞麦饼 Skills |
| **英文名称** | QiaoMai Skills |
| **版本号** | 1.0.0 |
| **简介** | 下一代智能体操作系统，从 oclaw-hermes v5.0.0 进化而来，通过八大维度全面优化，实现从"工程师工具"到"人人可用"的质变。 |
| **详细描述** | （复制 README_SKILLHUB.md 内容） |

### 步骤 5: 上传文件

上传以下文件：
- ✅ `SKILL.md` - 技能主文档
- ✅ `README_SKILLHUB.md` - SkillHub 专用 README
- ✅ `skillhub.json` - 元数据文件
- ✅ `core/` 目录 - 核心模块

或上传打包好的：
- ✅ `qiaomai-skills-1.0.0.zip`

### 步骤 6: 设置分类

选择分类：
- [x] 平台工具
- [x] 生产力
- [x] 研究开发

### 步骤 7: 添加标签

添加标签（建议）：
- 智能体
- 知识图谱
- 记忆系统
- 可视化
- 报告生成
- 类案检索
- 易用性
- 荞麦饼

### 步骤 8: 设置入口

- **入口文件**: `SKILL.md`
- **执行命令**: `qiaomai`

### 步骤 9: 提交审核

点击 **"提交审核"** 按钮
