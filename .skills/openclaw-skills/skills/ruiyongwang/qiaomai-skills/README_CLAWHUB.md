## ✨ 简介

**荞麦饼 Skills** 是从 [oclaw-hermes](https://openclawmp.stepfun.com) v5.0.0 进化而来的下一代智能体操作系统。

通过**八大维度全面优化**，实现了从"工程师工具"到"人人可用"的质变。

### 🎯 设计理念

> "让智能像荞麦饼一样，营养全面、易于消化、百搭实用。"

- **营养全面**：八大核心能力，覆盖智能体全生命周期
- **易于消化**：自然语言交互，零学习成本上手

## 📦 核心特性

### 1️⃣ 易用性优化
- 自然语言交互 - 用大白话描述需求
- 一键配置向导 - 5步完成初始化
- 智能提示系统 - 实时操作建议

### 2️⃣ 智能执行优化
- 自适应引擎 v2.0 - 自动选择执行策略
- 预测性执行 - 预判用户需求
- 智能任务分解 - 复杂任务自动拆解

### 3️⃣ 智能数据库优化
- 混合架构 - 向量 + 图 + 关系数据库
- 智能索引 - 自动选择最优策略
- 查询优化 - AI 驱动的查询计划

### 4️⃣ 智能记忆体优化 (OctoMemory)
八层记忆架构：
1. 瞬时记忆 (Instant) - 毫秒级
2. 工作记忆 (Working) - 秒级
3. 短期记忆 (Short) - 分钟级
4. 会话记忆 (Session) - 小时级
5. 技能记忆 (Skill) - 天级
6. 专家记忆 (Expert) - 周级
7. 图谱记忆 (Graph) - 月级
8. 归档记忆 (Archive) - 年级

### 5️⃣ 报告系统优化
- 多格式输出 - Markdown / PDF / Word / HTML / Excel
- 自适应模板 - 自动选择最佳模板
- 智能排版 - 自动分页、目录、图表

### 6️⃣ 类案检索优化
- 语义检索 - 理解查询意图
- 规则检索 - 精确匹配专业规则
- 混合排序 - 综合排序算法

### 7️⃣ 知识拓扑优化 (DynamicKG)
- 动态知识图谱 - 实时构建知识关联
- 多维度关联 - 实体、概念、规则、案例
- 知识推理 - 基于图谱的逻辑推理

### 8️⃣ 可视化优化
- 交互式仪表盘 - 实时数据可视化
- 3D 知识空间 - 沉浸式知识探索
- 智能图表 - 自动选择最佳图表类型

## 📝 快速开始

```python
# 初始化荞麦饼 Skills
from qiaomai import QiaoMaiSkills

skills = QiaoMaiSkills()

# 自然语言交互
result = skills.ask("帮我生成一份项目报告")

# 智能执行
result = skills.execute("分析数据并生成图表")

# 类案检索
result = skills.search_cases("工程合同纠纷")

# 生成报告
result = skills.generate_report("项目总结", format="pdf")

# 知识图谱查询
result = skills.query_kg("建设工程合同相关法规")
```
