### 3.2 核心算法

#### 3.2.1 精度控制算法

```python
class PrecisionController:
    """
    高精度估算引擎 v3.0
    目标精度：±3%
    方法：GB/T 50500-2024 + 规范计算 + 交叉验证
    """
    
    # 各分项精度目标
    PRECISION_TARGETS = {
        'concrete': 0.01,      # 混凝土 ±1%
        'rebar': 0.02,         # 钢筋 ±2%
        'formwork': 0.03,      # 模板 ±3%
        'masonry': 0.03,       # 砌体 ±3%
        'finishing': 0.05,     # 装饰 ±5%
        'mep': 0.05,           # 机电 ±5%
    }
    
    # 综合精度（误差传播公式）
    # σ_total = sqrt(Σ(σ_i² × w_i²))
    # w_i 为各分项造价权重
    
    @staticmethod
    def calculate_total_precision(estimates: dict) -> float:
        """
        计算综合精度（误差传播）
        """
        weights = {
            'concrete': 0.12,
            'rebar': 0.15,
            'formwork': 0.05,
            'masonry': 0.06,
            'finishing': 0.25,
            'mep': 0.20,
            'other': 0.17
        }
        
        variance_sum = 0
        for item, weight in weights.items():
            precision = estimates.get(item, {}).get('precision', 0.1)
            variance_sum += (precision * weight) ** 2
        
        return math.sqrt(variance_sum)
```

#### 3.2.2 指标合理性校验

```python
class ReasonablenessValidator:
    """
    指标合理性校验器
    数据来源：住建部、中价协、各省造价站发布的官方指标
    """
    
    # 官方发布的造价指标参考区间（2024年）
    OFFICIAL_INDICATORS = {
        '办公_框架-核心筒_高层': {
            'concrete_index': (0.45, 0.65, 'm³/㎡'),  # 地下室含量更高
            'rebar_index': (55, 90, 'kg/㎡'),        # 含钢量
            'formwork_index': (2.5, 4.5, '㎡/㎡'),
            'masonry_index': (0.15, 0.35, 'm³/㎡'),
            'unit_cost': (4500, 6500, '元/㎡'),      # 建安单方造价
        },
        '住宅_剪力墙_高层': {
            'concrete_index': (0.38, 0.55, 'm³/㎡'),
            'rebar_index': (45, 75, 'kg/㎡'),
            'formwork_index': (2.8, 4.2, '㎡/㎡'),
            'masonry_index': (0.10, 0.25, 'm³/㎡'),
            'unit_cost': (3500, 5000, '元/㎡'),
        },
    }
    
    @staticmethod
    def validate_quantity(building_type, structure_type, item, value) -> ValidationResult:
        """
        校验工程量指标的合理性
        返回：是否通过 + 偏离程度 + 建议
        """
        key = f"{building_type}_{structure_type}"
        if key not in OFFICIAL_INDICATORS:
            return None
        
        indicator = OFFICIAL_INDICATORS[key].get(item)
        if not indicator:
            return None
        
        low, high, unit = indicator
        if value < low:
            return ValidationResult(
                passed=False,
                deviation=(low - value) / low,
                suggestion=f"指标偏低，可能存在漏计或损耗系数取值不当"
            )
        elif value > high:
            return ValidationResult(
                passed=False,
                deviation=(value - high) / high,
                suggestion=f"指标偏高，可能设计用量过大或存在重复计算"
            )
        else:
            return ValidationResult(
                passed=True,
                deviation=(value - (low+high)/2) / ((low+high)/2),
                suggestion="指标在合理区间内"
            )
```

## 七、结论

要实现**±3%的估算精度**，必须：

1. **从配比估算升级到规范计量**：每个数量必须有规范条文支撑
2. **建立完整的数据溯源体系**：引用权威数据源
3. **实现三层校验机制**：输入/计算/输出全流程验证
4. **透明化误差传播**：让用户清楚知道不确定性的来源
5. **实事求是地声明精度**：根据实际输入条件合理承诺

**度量衡智库的承诺**：
- ✅ 每个数量都有规范依据
- ✅ 每个单价都标注数据来源
- ✅ 每项结论都经过交叉验证
- ✅ 估算精度透明可追溯
