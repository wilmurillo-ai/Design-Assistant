# 交通运输工程造价咨询预算编制典型案例

## 案例一：某高速公路全过程造价咨询

### 项目基本信息

| 项目 | 内容 |
|------|------|
| 项目名称 | 某高速公路工程 |
| 工程规模 | 10亿元 |
| 服务类型 | 全过程造价咨询（D1） |
| 工程专业 | 公路专业 |
| 服务期限 | 3年 |

### 预算编制过程

#### 第一步：确定基准

```
咨询服务类型：D1（全过程造价咨询）
基准系数：1.00
```

#### 第二步：确定调整系数

| 系数类型 | 取值 | 说明 |
|---------|------|------|
| 专业调整系数 | 1.00 | 公路工程为基准 |
| 规模调整系数 | 1.20 | 10亿规模适中 |
| 阶段调整系数 | - | 全过程按各阶段 |

#### 第三步：各阶段预算分配

| 阶段 | 咨询服务 | 预算占比 | 金额估算 |
|------|---------|---------|---------|
| 前期 | D2-1投资估算 | 10% | 100万 |
| 设计 | D2-2设计概算 | 15% | 150万 |
| 设计 | D3-3施工图预算 | 20% | 200万 |
| 招标 | D3-4清单预算 | 15% | 150万 |
| 实施 | D3-6工程结算 | 25% | 250万 |
| 竣工 | D3-7竣工决算 | 15% | 150万 |
| **合计** | - | **100%** | **1000万** |

## 案例二：某水运工程结算审核

### 项目基本信息

| 项目 | 内容 |
|------|------|
| 项目名称 | 某港口码头工程 |
| 工程规模 | 5亿元 |
| 服务类型 | 合同结算审核（D3-6） |
| 工程专业 | 水运专业 |
| 服务周期 | 6个月 |

### 预算编制过程

#### 确定系数

| 系数类型 | 取值 | 说明 |
|---------|------|------|
| 专业调整系数 | 1.30 | 水运工程复杂度高 |
| 规模调整系数 | 0.90 | 5亿规模适中 |
| 阶段调整系数 | 0.50 | 结算阶段 |

#### 计算预算

```
综合系数 = 1.00 × 1.30 × 0.90 × 0.50 = 0.585

基本预算 = 5亿 × 0.585% ≈ 292.5万元
```

#### 工作量分配

| 工作内容 | 占比 | 金额 |
|---------|------|------|
| 收集资料 | 5% | 15万 |
| 工程量审核 | 25% | 73万 |
| 单价审核 | 30% | 88万 |
| 报告编制 | 20% | 59万 |
| 配合工作 | 20% | 59万 |

## 案例三：某铁路隧道专项咨询

| 项目 | 内容 |
|------|------|
| 项目名称 | 某高铁隧道工程 |

| 工程专业 | 铁路隧道专业 |
| 服务周期 | 2个月 |

### 预算编制过程

#### 确定系数

| 系数类型 | 取值 | 说明 |
|---------|------|------|
| 专业调整系数 | 1.40 | 隧道工程高复杂度 |
| 规模调整系数 | 1.30 | 8km长大隧道 |
| 专项调整系数 | 0.40 | 成本测算基准 |

#### 计算预算

```
综合系数 = 1.00 × 1.40 × 1.30 × 0.40 = 0.728

基本预算 = 12亿 × 0.728% ≈ 87.4万元
```

## 案例四：驻场造价管理服务

### 项目基本信息

| 项目 | 内容 |
|------|------|
| 项目名称 | 某市政道路工程 |
| 服务类型 | 驻场造价管理 |
| 服务方式 | 附加工作-驻场服务 |
| 驻场人数 | 2人 |
| 驻场周期 | 12个月 |

### 预算编制

#### 人工成本计算

| 人员 | 月人工成本 | 人数 | 月合计 | 年度合计 |
|------|----------|------|-------|---------|
| 高级造价工程师 | 3万元/月 | 1 | 3万 | 36万 |
| 造价员 | 1.5万元/月 | 1 | 1.5万 | 18万 |
| **小计** | - | - | **4.5万** | **54万** |

#### 附加成本

| 项目 | 金额 | 说明 |
|------|------|------|
| 差旅补贴 | 5万/年 | 往返、住宿 |
| 办公费用 | 2万/年 | 办公设备、耗材 |
| 降效损失 | 3万/年 | 现场条件影响 |
| 管理费 | 5万/年 | 15%管理费 |
| **附加合计** | **15万/年** | - |

#### 总预算

```
年度驻场服务预算 = 54万（人工）+ 15万（附加）= 69万元

12个月服务 = 69万元
```

## 案例五：调整概算咨询

| 项目 | 内容 |
|------|------|
| 调整类型 | 设计变更导致的概算调整 |
| 原概算 | 8亿元 |
| 调整幅度 | 增加2亿元 |
| 服务类型 | D4-11调整概算 |

### 预算编制

#### 确定系数

| 系数类型 | 取值 | 说明 |
|---------|------|------|
| 调整时期 | 设计阶段 | 系数较高 |
| 基础系数 | 0.20 | - |
| 调整幅度系数 | 1.50 | 调整幅度>20% |

```
综合系数 = 0.20 × 1.50 = 0.30

```
