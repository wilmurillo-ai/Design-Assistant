## 工作内容分类

### 基本工作结构

```
基本工作
├── 通用工作（必选）
│   ├── 收集整理基础资料
│   ├── 编制造价咨询服务计划
│   ├── 了解建设条件
│   ├── 档案收集整理归档
│   └── 配合报审报批
│
├── 阶段专项工作（按需）
│   ├── 编制/审核
│   ├── 计价
│   └── 核对分析
│
├── 可选工作（合同约定）
│   └── 各项专项服务
│
└── 附加工作（单独签订）
    ├── 协调工作
    ├── 驻场服务
    └── 其他附加
```

## 预算编制方法

### 规模计价法

```
M = Q × P × K

其中：
M - 咨询服务预算费用
Q - 服务规模（万元）
P - 基准单价（元/万元）
K - 综合调整系数
```

### 工日计价法

```
M = Σ(D × R)

其中：
D - 各专业服务工日
R - 对应人工单价
```

## 合规要点

### 三级复核制度

```
第一级：专业工程师自检
    ↓
第二级：项目负责人复核
    ↓
第三级：技术负责人审核
```

### 档案管理要求

| 要求 | 说明 |
|------|------|
| 完整性 | 过程文件、成果文件齐全 |
| 及时性 | 随项目进度同步归档 |
| 可追溯 | 建立索引、便于查阅 |
| 保密性 | 按规定做好保密管理 |
