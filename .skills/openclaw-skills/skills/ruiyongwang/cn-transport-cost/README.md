## 基础概念

### 三种计价方法

#### 1. 规模计价法（推荐）
```
预算费用 = 服务规模 × 基准单价 × 综合调整系数
```
适用于：可量化服务规模的项目

#### 2. 工日计价法
```
预算费用 = 服务工日 × 人工单价 × 综合预算系数
```
适用于：难以量化规模或需精细化管理的项目

#### 3. 比例计价法
```
预算费用 = 咨询服务对应的工程造价 × 费率
```
适用于：与工程造价挂钩的咨询服务

## 实战步骤

### 第一步：确定咨询服务类型

```
1. 是全过程咨询还是分阶段咨询？
   ├─ 全过程 → D1
   └─ 分阶段 → D2 → 确定具体阶段
       ├─ 前期阶段 → D2-1（投资估算）
       ├─ 设计阶段 → D2-2（概算）/ D3-3（预算）
       └─ 实施阶段 → D3（招标/结算）
```

### 第二步：确定工程专业

| 专业 | 主要工程类型 |
|------|-------------|
| 公路 | 高速公路、普通公路、桥梁、隧道 |
| 铁路 | 高速铁路、普速铁路、轨道 |
| 水运 | 港口、航道、疏浚 |
| 通用 | 征地、拆迁、迁改、其他费 |

### 第三步：计算调整系数

```
综合调整系数 = 咨询分类系数 × 专业调整系数 × 规模调整系数 × 阶段调整系数
```

**典型系数示例**：
| 系数组合 | 结果 | 说明 |
|---------|------|------|
| 公路+概算 | 1.0×1.0 = 1.0 | 基准情况 |
| 水运+概算 | 1.0×1.3 = 1.3 | 水运复杂度高 |
| 公路+隧道专项 | 1.5×1.2 = 1.8 | 隧道工程复杂 |

### 第四步：分配预算比例

```
总预算 = Σ(各阶段工作量 × 费率)

| 阶段       | 典型占比  |
|------------|----------|
| 准备工作   | 3%~10%   |
| 基本工作   | 50%~60%  |
| 可选工作   | 15%~25%  |
| 配合工作   | 5%~12%   |
| 档案管理   | 2%~4%    |
```

## 常见问题

### Q1：投资估算和设计概算有什么区别？
- **投资估算**：项目建议书/可研阶段，精度要求较低
- **设计概算**：初步设计阶段，精度要求较高

### Q2：可选工作可以不选吗？
可以。可选工作基于市场自由供给原则，由委托人根据需要选择。

### Q3：驻场服务如何计费？
驻场服务属于附加工作，需单独签订补充协议，按驻场人员×驻场天数×综合单价计算。

### Q4：多个咨询人联合如何处理？
需指定一家作为总体负责方，统一技术标准、协调进度、汇总成果。总体协调工作按附加工作单独计费。

## 合规要点

### 必须完成的工作（基本工作）

1. 收集并整理基础资料，审核完整性
2. 编制造价咨询服务计划和大纲
3. 了解项目所在地的建设条件
4. 完成档案收集、整理和归档
5. 配合委托人完成报审、报批等工作

### 特别注意

- 基本工作不可删减，是咨询服务的核心
- 可选工作须在合同中明确约定
- 附加工作（协调、驻场）须单独签订协议

## 典型案例

### 某高速公路造价咨询项目

**项目概况**：
- 工程规模：10亿元
- 服务类型：全过程造价咨询（D1）
- 工程专业：公路专业

**预算编制要点**：
```
1. 确定基准费率（全过程咨询）
2. 专业调整：公路 ×1.0
3. 规模调整：10亿 × 规模系数
4. 综合计算后按阶段分配
```
