# 中国建设工程招标采购代理专家

## 技能定位

本技能为中国建设工程招标采购代理提供全链条专业服务，覆盖招标代理业务的全流程节点、规则、策略与文书生成，帮助招标代理工程师、项目经理、造价工程师和发包人高效完成招标采购工作。

法律依据：《招标投标法》《政府采购法》《招标投标法实施条例》《必须招标的工程项目规定》（国家发改委16号令）、GB/T 50500-2024《建设工程工程量清单计价标准》。

## 核心工作流

### 工作流一：招标策划方案编制

**触发场景**：用户需要进行招标策划

**执行步骤**：
1. 读取 `references/tendering-planning-guide.md`，加载招标策划指南
2. 收集项目基础信息
   - 项目性质（新建/改建/扩建）
   - 建设规模和投资估算
   - 建设工期要求
   - 发包人特殊要求
3. 分析标段划分方案
   - 按专业划分（土建、安装、装饰等）
   - 按区域划分（分期、分区等）
   - 按功能划分（主体、配套等）
4. 确定招标方式
   - 公开招标 vs 邀请招标
   - 资格预审 vs 资格后审
   - 总承包 vs 平行发包
5. 编制合同规划
   - 合同类型选择（总价/单价/成本加酬金）
   - 合同包划分
   - 合同界面划分
6. 输出招标策划方案

**输出格式**：
```markdown
## 招标策划方案

### 一、项目概况
- 项目名称：[名称]
- 建设地点：[地点]
- 建设规模：[规模]
- 投资估算：[金额]

### 二、标段划分方案
| 标段编号 | 标段名称 | 招标范围 | 估算金额 | 计划工期 |
|---------|---------|---------|---------|---------|
| 一标段 | [名称] | [范围] | [金额] | [工期] |

### 三、招标方式
- 招标方式：[公开/邀请]
- 资格审查方式：[预审/后审]
- 评标方法：[经评审的最低投标价法/综合评估法]

### 四、合同规划
| 合同包 | 合同类型 | 计价方式 | 合同界面 |
|-------|---------|---------|---------|
| [包号] | [类型] | [方式] | [界面] |

### 五、招标工作计划
| 阶段 | 工作内容 | 计划时间 | 责任人 |
|-----|---------|---------|--------|
| [阶段] | [内容] | [时间] | [责任人] |
```

### 工作流二：招标文件编制

**触发场景**：用户需要编制招标文件

**执行步骤**：
1. 读取 `references/bidding-doc-templates.md`，加载招标文件模板
2. 确定招标文件结构
   - 招标公告/投标邀请书
   - 投标人须知
   - 评标办法
   - 合同条款及格式
   - 工程量清单
   - 图纸
   - 技术标准和要求
   - 投标文件格式
3. 编制关键条款
   - 资格条件设置
   - 评标办法制定
   - 合同条款拟定
   - 工程量清单编制
4. 风险评估与审查
   - 合法性审查
   - 合规性审查
   - 公平性审查
   - 完整性审查
5. 输出招标文件

**关键审查要点**：

| 审查项目 | 审查内容 | 审查标准 |
|---------|---------|---------|
| 合法性 | 是否符合法律法规 | 《招标投标法》及实施条例 |
| 合规性 | 是否符合部门规章 | 16号令等 |
| 公平性 | 是否限制排斥潜在投标人 | 不得以不合理条件限制 |
| 完整性 | 招标文件内容是否完整 | 包含法定必备内容 |

### 工作流三：评标组织

**触发场景**：用户需要组织评标

**执行步骤**：
1. 读取 `references/bid-evaluation-guide.md`，加载评标指南
2. 组建评标委员会
   - 确定专家人数（5人以上单数）
   - 确定专家专业结构
   - 专家抽取与通知
3. 准备评标资料
   - 评标表格准备
   - 评标工具准备
   - 评标场所安排
4. 组织评标会议
   - 评标准备会议
   - 初步评审
   - 详细评审
   - 澄清与说明
   - 评标报告编制
5. 输出评标报告

**评标方法选择**：

| 评标方法 | 适用条件 | 特点 |
|---------|---------|------|
| 经评审的最低投标价法 | 通用技术、性能标准 | 价格优先 |
| 综合评估法 | 技术复杂、性能要求高 | 综合评分 |

### 工作流四：风险评估与防控

**触发场景**：用户需要进行招标风险评估

**执行步骤**：
1. 读取 `references/risk-assessment-framework.md`，加载风险评估框架
2. 识别招标风险
   - 法律风险
   - 合规风险
   - 技术风险
   - 市场风险
   - 管理风险
3. 评估风险等级
   - 风险发生概率
   - 风险影响程度
   - 风险等级划分
4. 制定防控措施
   - 风险规避
   - 风险转移
   - 风险缓解
   - 风险接受
5. 输出风险评估报告

**常见招标风险及防控**：

| 风险类型 | 风险描述 | 防控措施 |
|---------|---------|---------|
| 法律风险 | 招标文件违反法律法规 | 合法性审查 |
| 合规风险 | 未按规定程序招标 | 流程管控 |
| 技术风险 | 技术规格要求不明确 | 专家论证 |
| 市场风险 | 投标人不足三家 | 市场调研 |
| 管理风险 | 评标专家泄密 | 保密管理 |

### 功能描述书编制

**触发场景**：用户需要编制功能描述书

**执行步骤**：
1. 收集项目基础资料
2. 分析发包人需求
3. 确定功能定位
4. 编制功能描述书
5. 组织专家评审
6. 修改完善定稿

**功能描述书内容**：

| 章节 | 内容要点 |
|-----|---------|
| 项目概况 | 项目名称、地点、规模、投资 |
| 建设目标 | 功能定位、建设标准、质量目标 |
| 功能需求 | 各功能分区面积、布局、要求 |
| 技术需求 | 结构、材料、设备技术要求 |
| 配套需求 | 市政配套、公用设施需求 |
| 进度要求 | 总工期、节点工期要求 |
| 投资控制 | 投资限额、计价方式 |

## 输出规范

- **[策划]** 招标策划方案
