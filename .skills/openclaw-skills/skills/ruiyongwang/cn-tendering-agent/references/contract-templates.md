# 建设工程合同模板库

## 一、合同类型与适用范围

| 合同类型 | 适用范围 | 计价方式 | 风险分担 |
|---------|---------|---------|---------|
| 固定总价合同 | 工程量明确、工期短 | 总价一次包死 | 承包商承担量、价风险 |
| 固定单价合同 | 工程量有变化 | 单价固定，量据实 | 量风险发包人，价风险承包商 |
| 可调价合同 | 工期长、材料价格波动大 | 价格可按约定调整 | 共同承担价格风险 |
| 成本加酬金合同 | 紧急工程、新技术工程 | 实际成本+酬金 | 发包人承担大部分风险 |

## 二、建设工程施工合同关键条款

### 2.1 合同价格条款

**固定单价合同（推荐）**：
```
合同价格条款
第一条 合同价格形式：固定单价合同
第二条 合同价格：本合同签约合同价为（大写）___元整（¥___元）。
第三条 价格调整：
  1. 工程量清单综合单价固定，不得调整，但法律法规变化引起的税金调整除外。
  2. 出现下列情形时，综合单价可作调整：
     a）法律法规变化；
     b）工程变更引起工程量超过15%；
     c）招标文件规定的特殊情形。
  3. 主要材料价格调整：当钢材、混凝土、机电设备等主材价格上涨/下降超过合同签订时
     价格的±5%时，超出部分由发包人承担/享有。
第四条 暂列金额：本合同暂列金额为___元，由发包人指令后使用，不使用退回发包人。
```

### 2.2 工程变更条款

```
工程变更与价款调整
第一条 变更范围：下列情形为工程变更：
  1. 增减合同中任何工作；
  2. 改变合同中任何工作的质量标准；
  3. 改变合同中任何工作的施工时间和顺序；
  4. 发包人要求的其他变更。

第二条 变更程序：
  1. 发包人提出变更：发出变更指令→承包商提交变更报价→双方协商确认→继续施工
  2. 承包商建议变更：提交变更建议书→发包人审批→发出变更指令

第三条 变更价款确定原则（优先顺序）：
  1. 合同中已有适用综合单价的，按合同单价；
  2. 合同中有类似综合单价的，参照确定；
  3. 合同中没有的，由承包商提出，经发包人确认。

第四条 变更结算：每月25日前承包商提交当月变更汇总，发包人在收到后14天内确认。
```

### 2.3 付款条款

```
工程款支付
第一条 预付款：
  签约合同价的10%（不超过500万元），在开工后7天内支付。
  预付款在工程进度款中按比例扣回，扣完为止。

第二条 进度款：
  每月末承包商提交当月计量申请→发包人14天内确认→
  确认后7天内按确认金额的80%支付。
  （质量缺陷期间不支付尾款）

第三条 竣工结算款：
  竣工验收合格后，承包商提交结算申请→发包人56天内完成审核→
  结算完成后30天内支付结算款（留5%质保金）。

第四条 质量保证金：
  合同价格的3%（不超过3%），竣工验收后保留，
  缺陷责任期（12-24个月）届满后14天内退还（有质量缺陷的除外）。
```

### 2.4 工期条款

```
工期与延期
第一条 合同工期：本合同工程工期为___日历天，计划开工日期___，计划竣工日期___。

第二条 工期延误（发包人原因）：
  以下情形造成工期延误，承包商有权顺延工期并索赔费用：
  1. 发包人未按合同约定提供施工条件；
  2. 发包人变更；
  3. 设计错误；
  4. 不可抗力；
  5. 其他发包人责任事件。
  承包商应在事件发生后7天内提出通知，28天内提交索赔报告。

第三条 工期延误（承包商原因）：
  非发包人原因造成的工期延误，承包商需支付误期损失赔偿：
  每延误1天：合同价格的___/千（一般0.5‰-2‰，最高不超过合同价5%）。

第四条 节点工期：
  | 节点名称 | 计划日期 | 奖罚标准 |
  |---------|---------|---------|
  | 主体结构完成 | | 提前每天奖___元，延误每天罚___元 |
  | 竣工验收 | | 同上 |
```

### 2.5 质量与验收条款

```
质量标准与验收
第一条 质量标准：本工程质量目标为___（合格/优良/鲁班奖）。
  达到___目标奖励承包商___万元；未达合格标准，承包商整改至合格，并承担整改费用。

第二条 材料设备质量：
  承包商须提供材料进场报告、合格证及检测报告。
  发包人有权对材料质量抽检，不合格材料退场处理。

第三条 隐蔽工程验收：
  承包商应提前48小时通知发包人，发包人应按时参加验收。
  无故不参加或不签字，视为认可。

第四条 竣工验收：
  工程完工后承包商提交竣工验收申请→发包人在14天内组织验收→
  验收合格签署竣工验收证书→移交使用。
```

## 三、常用合同专用条款参考

### 3.1 资金支付保障条款（重要）

```
为保障承包商资金安全：
1. 发包人须在签约时开设专用资金账户；
2. 工程款支付须经承包商确认；
3. 逾期付款利息按合同利率（参考同期LPR+2%）计算；
4. 发包人有权通过银行见索即付保函代替预付款担保。
```

### 3.2 争议解决条款

```
争议解决
第一条 双方争议应先协商解决，协商不成提交监理/总监裁决。

第二条 友好协商期为28天，期满仍未解决的，任何一方可通过以下方式解决（选择一种）：
  □ 向___仲裁委员会申请仲裁（仲裁裁决为终局裁决）
  □ 向工程所在地人民法院提起诉讼

第三条 争议期间，除发包人指令外，承包商应继续履行合同义务。
```

## 四、合同审查重点清单

### 4.1 施工合同必审10项

| 序号 | 审查项目 | 风险点 | 建议 |
|-----|---------|-------|------|
| 1 | 合同价格类型 | 价格类型不明确 | 明确固定单价/总价 |
| 2 | 工程量变化处理 | 大量变更无价格调整机制 | 约定变更超15%可调价 |
| 3 | 付款条件与时限 | 无明确付款时间 | 约定具体天数，逾期计息 |
| 4 | 工期与节点 | 工期不明确 | 明确日历天、节点日期 |
| 5 | 质保金比例 | 超过3%或不退还 | 坚持3%，缺陷期满退还 |
| 6 | 签证变更程序 | 口头变更无据 | 变更须书面，7日申报 |
| 7 | 索赔时效 | 时效过短（<28天） | 最少28天提通知 |
| 8 | 争议解决方式 | 未约定或约定矛盾 | 选择仲裁或诉讼 |
| 9 | 不可抗力范围 | 范围过宽 | 限定法律意义上的不可抗力 |
| 10 | 保险安排 | 未约定保险 | 明确险种和受益人 |
