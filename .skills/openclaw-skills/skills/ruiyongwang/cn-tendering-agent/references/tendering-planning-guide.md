# 招标策划指南

## 一、招标策划总体原则

### 1.1 法规依据
- 《中华人民共和国招标投标法》（2017年修正）
- 《招标投标法实施条例》（国务院613号令，2017年修正）
- 《必须招标的工程项目规定》（国家发改委16号令，2018年）
- 《政府采购法》及《政府采购法实施条例》
- GB/T 50500-2024《建设工程工程量清单计价标准》

### 1.2 必须招标的工程项目（16号令）

| 项目类型 | 金额标准 |
|---------|---------|
| 施工单项合同估算价 | ≥ 400万元 |
| 重要设备、材料等货物采购 | 单项合同估算价 ≥ 200万元 |
| 勘察、设计、监理等服务 | 单项合同估算价 ≥ 100万元 |
| 项目总投资额（使用国有资金） | ≥ 3000万元 |
| 国家融资项目（施工） | ≥ 400万元 |

## 二、标段划分方案

### 2.1 标段划分原则
1. **专业性原则**：按专业特点划分，有利于专业化施工
2. **经济性原则**：合理规模，促进竞争
3. **可管理性原则**：便于发包人管理
4. **风险分配原则**：合理分配合同风险

### 2.2 标段划分常见模式

#### 模式一：按专业划分（平行发包）
```
项目总承包
├── 土建工程标（含结构、建筑装修）
├── 机电安装标（给排水、暖通、强电）
├── 弱电智能化标（安防、BA、信息化）
├── 幕墙装饰标（外幕墙、精装修）
├── 景观绿化标（园林景观、广场铺装）
└── 市政配套标（道路、管线）
```

#### 模式二：按区域划分（分区发包）
```
项目总承包
├── A区工程标（1-5号楼）
├── B区工程标（6-10号楼）
├── C区工程标（配套设施）
└── 地下室工程标（整体地下室）
```

#### 模式三：EPC总承包
```
EPC总承包标（设计+采购+施工）
├── 设计分包
├── 主体施工分包
└── 设备采购
```

### 2.3 标段划分决策矩阵

| 考量因素 | 平行发包 | 总承包 | EPC |
|---------|---------|-------|-----|
| 发包人管理能力 | 强 | 弱 | 弱 |
| 项目技术复杂度 | 中低 | 中 | 高 |
| 工期要求 | 宽松 | 中等 | 紧 |
| 投资控制 | 精细 | 中等 | 总价固定 |
| 风险承担 | 发包人多 | 均衡 | 承包人多 |

## 三、招标方式选择

### 3.1 公开招标 vs 邀请招标

| 对比项 | 公开招标 | 邀请招标 |
|-------|---------|---------|
| 适用条件 | 一般工程 | 特殊/专业工程 |
| 竞争程度 | 充分竞争 | 有限竞争 |
| 时间成本 | 较长（45-60天） | 较短（30-45天） |
| 行政审批 | 不需要 | 需报批（国有资金） |
| 招标人自主性 | 低 | 高 |

**邀请招标适用情形（《招标投标法》第11条）**：
1. 项目技术复杂，只有少量潜在投标人可供选择
2. 受自然地理环境限制
3. 涉及国家安全、国家秘密
4. 应采用邀请招标的其他情形

### 3.2 资格审查方式

| 审查方式 | 适用场景 | 优点 | 缺点 |
|---------|---------|------|------|
| 资格预审 | 技术复杂、潜在投标人多 | 降低废标风险 | 延长时间 |
| 资格后审 | 一般工程 | 时间短 | 废标风险高 |

## 四、评标方法选择

### 4.1 经评审的最低投标价法
**适用条件**：具有通用技术、性能标准或者招标人对其技术、性能没有特殊要求的招标项目

**评标流程**：
1. 初步评审（资格、符合性审查）
2. 算术修正
3. 价格调整（偏差）
4. 商务澄清
5. 按评标价由低到高排序
6. 通过详细评审的最低价投标人为第一中标候选人

### 4.2 综合评估法
**适用条件**：技术复杂或者无法精确拟定技术规格的项目

**评分构成参考**：

| 评分项目 | 分值区间 | 说明 |
|---------|---------|------|
| 价格（商务标） | 40-60分 | 评标基准价计算 |
| 技术方案 | 20-40分 | 施工方案、工期、质量 |
| 信誉业绩 | 10-20分 | 类似业绩、信用评价 |
| 项目管理 | 5-15分 | 项目班子、机械配置 |

**评标基准价计算方式**（常用三种）：
1. 有效投标报价去掉最高最低后的算术平均值
2. 招标控制价下浮一定比例
3. 有效报价平均值×下浮系数

## 五、招标工作计划编制

### 5.1 工作计划模板

| 阶段 | 工作内容 | 计划用时 | 法定时限 |
|-----|---------|---------|---------|
| 招标文件编制 | 编制招标文件、报批 | 10-15天 | - |
| 发售招标文件 | 公告发布至截止发售 | ≥20天 | 5个工作日内发售 |
| 投标人答疑 | 现场踏勘、投标预备会 | 3-5天 | 澄清在截止前15天 |
| 投标准备期 | 投标文件编制时间 | ≥20天 | 公告到截止≥20天 |
| 开标 | 开标会议 | 1天 | 截止时间同时开标 |
| 评标 | 评标委员会评审 | 3-7天 | - |
| 公示 | 中标候选人公示 | 3天 | 3个工作日 |
| 定标 | 发出中标通知书 | 1-2天 | 30日内确定 |
| 合同签订 | 合同谈判签订 | 5-10天 | 30日内签约 |

### 5.2 总工期参考
- **简单工程**（1个标段）：40-60天
- **中等工程**（2-3个标段）：60-90天
- **复杂工程**（多标段、资格预审）：90-150天
