# 招标采购风险评估框架

## 一、风险识别体系

### 1.1 风险分类

```
招标采购风险
├── 法律风险
│   ├── 项目不符合必须招标条件而强制规避
│   ├── 招标文件存在违法条款
│   ├── 程序违法（时间不足、未公告等）
│   └── 合同不合法
├── 合规风险
│   ├── 评标委员会组建不合规
│   ├── 评标程序违规
│   ├── 信息公开不规范
│   └── 保密管理失控
├── 技术风险
│   ├── 工程量清单漏项
│   ├── 技术规格描述不清
│   ├── 质量标准设置不合理
│   └── 施工方案可行性不足
├── 市场风险
│   ├── 投标人不足三家
│   ├── 市场价格大幅波动
│   ├── 恶性低价竞争
│   └── 串通投标
└── 管理风险
    ├── 专家保密管理失控
    ├── 评标结果被质疑投诉
    ├── 合同履约风险
    └── 廉政风险
```

## 二、风险评估矩阵

### 2.1 风险评级方法

```
风险等级 = 发生概率 × 影响程度

概率评分：
  高（3分）：60%以上概率
  中（2分）：30%-60%概率
  低（1分）：30%以下概率

影响评分：
  重大（3分）：导致招标失败或重大违规
  一般（2分）：影响招标进度或质量
  轻微（1分）：影响较小、可纠正
```

### 2.2 风险热力图

| | 影响重大(3) | 影响一般(2) | 影响轻微(1) |
|--|-----------|-----------|-----------|
| **概率高(3)** | 🔴 极高风险(9) | 🟠 高风险(6) | 🟡 中风险(3) |
| **概率中(2)** | 🟠 高风险(6) | 🟡 中风险(4) | 🟢 低风险(2) |
| **概率低(1)** | 🟡 中风险(3) | 🟢 低风险(2) | 🟢 低风险(1) |

## 五、招投标异常情形识别

### 5.1 串通投标识别特征

- 多家投标人报价相近（差异<1%）
- 投标文件错误相同
- 多家投标人同一IP地址提交文件
- 报价规律性明显（如等差数列）
- 多家使用相同计算机制作标书
- 不同投标人文件内容实质相同

### 5.2 弄虚作假识别特征

- 业绩合同无法核实
- 拟派人员无实际在册记录
- 资质证书颜色、字体异常
- 财务报告数据异常
- 印章疑似伪造

**处理程序**：
1. 暂停评标，保留证据
2. 书面记录并报告行政监督部门
3. 等待调查结论
4. 视情形否决投标或移送司法机关

## 六、投诉处理指南

### 6.1 投诉时限

| 事项 | 时限 |
|-----|-----|
| 投标人提出投诉 | 知道或应知之日起10日内 |
| 行政监督部门处理 | 受理后30工作日内 |
| 不符合条件不予受理 | 5工作日内告知 |

### 6.2 招标人应对投诉措施

1. 保留完整招标档案
2. 评标委员会成员及时联系
3. 提供完整证明材料
4. 配合行政部门调查
5. 不干预监督部门处理

### 6.3 质疑投诉预防

- 招标文件公示足够时间（3个工作日）
- 评标结果公示规范
- 招标程序全程留痕
- 资格审查标准客观明确
- 评标过程录音录像
