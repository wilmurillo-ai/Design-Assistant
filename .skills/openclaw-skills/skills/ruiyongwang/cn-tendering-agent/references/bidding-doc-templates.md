# 招标文件编制模板

## 一、招标文件结构

### 1.1 标准招标文件组成（依据住建部、九部委示范文本）

```
招标文件
├── 第一册 招标公告（或投标邀请书）
├── 第二册 投标人须知
│   ├── 投标人须知前附表
│   └── 投标人须知正文
├── 第三册 评标办法
│   ├── 评标办法前附表
│   └── 评标办法正文
├── 第四册 合同条款及格式
│   ├── 通用合同条款（GF-2017-0201）
│   ├── 专用合同条款
│   └── 合同附件格式
├── 第五册 工程量清单
│   ├── 总说明
│   ├── 分部分项工程量清单
│   ├── 措施项目清单
│   ├── 其他项目清单
│   └── 规费、税金项目清单
├── 第六册 图纸
├── 第七册 技术标准和要求
└── 第八册 投标文件格式
```

### 2.3 评标办法编制

#### A、综合评分法模板

**评标办法前附表**：

| 评分项目 | 权重/分值 |
|---------|---------|
| 商务标（投标报价） | 50分 |
| 技术标 | 35分 |
| 信誉业绩标 | 15分 |
| **合计** | **100分** |

**商务标评分（50分）**：
```
评标基准价 = 有效投标报价算术平均值 × 下浮系数K
（K值 = 0.97-0.99，招标文件中明确）

投标报价得分：
- 投标报价 = 评标基准价：50分
- 偏差每上升1%：扣2分
- 偏差每下降1%：扣1分
- 最高扣至0分
```

**技术标评分（35分）**：

| 评分项目 | 分值 | 评分标准 |
|---------|-----|---------|
| 施工方案 | 15分 | 专家评分 |
| 施工进度计划 | 8分 | 网络图、关键路径 |
| 质量保证措施 | 7分 | 质量体系、工艺 |
| 安全文明施工 | 5分 | 安全措施完整性 |

**信誉业绩标（15分）**：

| 评分项目 | 分值 | 评分标准 |
|---------|-----|---------|
| 类似工程业绩 | 8分 | 近5年同类业绩，1个4分，最多8分 |
| 企业信用评价 | 4分 | AAA：4分；AA：3分；A：2分 |
| 工程质量奖项 | 3分 | 鲁班奖/国优金奖3分，省优2分 |

#### B、经评审的最低投标价法模板

适用于通用建筑安装工程。

**评标流程**：
1. 初步评审：资格文件、符合性审查（Pass/Fail）
2. 算术核实：修正计算错误
3. 商务澄清：不重大偏差澄清
4. 评标价计算：报价±偏差调整
5. 排名：按评标价从低到高排列

## 三、工程量清单编制要点

### 3.1 工程量清单总说明

必须包含以下内容：
1. 工程概况（建筑规模、结构形式、建设地点）
2. 工程承包范围
3. 计价依据（定额、费率、材料价格信息）
4. 工程量清单编制说明
5. 其他需说明的问题

### 3.2 暂列金额设置

| 设置类型 | 金额范围 | 说明 |
|---------|---------|------|
| 暂列金额 | 分部分项工程费的5%-10% | 应对变更 |
| 暂估价 | 实际价格不确定项目 | 材料、专业工程 |
| 计日工 | 视项目情况 | 零星工作 |
| 总承包服务费 | 分包合同额的1%-3% | 协调配合费 |

### 3.3 工程量清单风险提示

⚠️ **常见风险**：
- 工程量漏项导致索赔
- 特征描述不清导致争议
- 暂估价设置不合理
- 清单与图纸不符

## 四、招标文件审查清单

### 4.1 合法性审查

| 审查项目 | 审查内容 | 审查结论 |
|---------|---------|---------|
| 招标范围 | 是否属于必须招标项目 | □合格 □不合格 |
| 招标方式 | 是否符合条件 | □合格 □不合格 |
| 发布媒介 | 是否在指定媒介发布 | □合格 □不合格 |
| 投标有效期 | 是否满足法定要求 | □合格 □不合格 |
| 投标保证金 | 金额是否超标准 | □合格 □不合格 |

### 4.2 合规性审查

| 审查项目 | 审查内容 | 审查结论 |
|---------|---------|---------|
| 资质要求 | 是否与项目匹配、无超越 | □合格 □不合格 |
| 业绩要求 | 是否合理、无地域限制 | □合格 □不合格 |
| 评标方法 | 是否符合规定 | □合格 □不合格 |
| 定标程序 | 是否符合规定程序 | □合格 □不合格 |

### 4.3 公平性审查

| 审查项目 | 审查内容 | 审查结论 |
|---------|---------|---------|
| 资格条件 | 是否含有歧视性条件 | □合格 □不合格 |
| 技术标准 | 是否指向特定品牌 | □合格 □不合格 |
| 评分标准 | 是否量化、客观 | □合格 □不合格 |
| 合同条款 | 是否公平合理 | □合格 □不合格 |
