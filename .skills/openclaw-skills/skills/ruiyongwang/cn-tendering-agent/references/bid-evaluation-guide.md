# 评标组织指南

## 一、评标委员会组建

### 1.1 评标委员会法定要求

依据《招标投标法》第37条、《评标委员会和评标方法暂行规定》（七部委第12号令）：

| 要求 | 规定内容 |
|-----|---------|
| 人数 | 5人以上单数 |
| 招标人代表 | 不超过1/3 |
| 技术、经济专家 | 不少于2/3 |
| 抽取方式 | 从省级以上专家库随机抽取 |
| 保密原则 | 开标前不得泄露专家名单 |
| 回避原则 | 与投标人有关系须回避 |

### 1.2 专家专业配置建议

| 项目类型 | 建议专家配置 |
|---------|------------|
| 房屋建筑工程（5人委员会） | 建筑2名+结构1名+设备1名+造价1名 |
| 市政道路工程（5人） | 道路2名+造价2名+结构1名 |
| 机电安装工程（5人） | 机电2名+暖通1名+电气1名+造价1名 |
| 综合体工程（7人） | 建筑2名+结构1名+设备2名+造价2名 |

## 二、评标准备工作

### 2.1 评标前准备清单

| 序号 | 准备事项 | 责任人 | 完成时限 |
|-----|---------|-------|---------|
| 1 | 评标场地安排（独立、保密） | 招标代理 | 开标前2天 |
| 2 | 评标资料印制（评标表格） | 招标代理 | 开标前1天 |
| 3 | 评标工具准备（计算器、文具） | 招标代理 | 开标当天 |
| 4 | 投标文件密封检查 | 招标人+监督 | 开标时 |
| 5 | 专家签到与资料分发 | 招标代理 | 评标开始前 |
| 6 | 纪律宣讲与保密协议 | 招标人代表 | 评标开始前 |

### 2.2 评标所需资料

- 招标文件（含工程量清单、图纸）
- 所有投标文件
- 评标办法
- 评标工作表（打分表、汇总表）
- 相关法律法规
- 工程量清单控制价分析
- 主要材料设备价格参考

## 三、评标程序

### 3.1 评标流程

```
评标流程
├── 第一步：评标准备（0.5-1小时）
│   ├── 宣布评标纪律
│   ├── 专家签署保密协议
│   ├── 宣读招标文件评标办法
│   └── 分发投标文件
├── 第二步：初步评审（1-2小时）
│   ├── 形式审查（装订、签章、密封）
│   ├── 资格审查（资质、人员、业绩）
│   ├── 响应性审查（实质性响应）
│   └── 算术修正（报价计算修正）
├── 第三步：详细评审（2-4小时）
│   ├── 价格评审
│   ├── 技术评审
│   ├── 商务评审
│   └── 需要时：澄清和说明
├── 第四步：评分汇总（0.5-1小时）
│   ├── 收回评分表
│   ├── 计算综合得分
│   ├── 排定名次
│   └── 确定推荐候选人
└── 第五步：编制评标报告（1-2小时）
    ├── 评标委员会签字
    └── 提交招标人
```

### 3.2 初步评审要点

**形式要求审查**：
- 投标文件按要求密封、签章
- 投标文件格式符合要求
- 授权委托书有效
- 投标保证金已缴纳
- 提交要求的所有材料

**资格审查要点**：

| 审查项目 | 审查方法 | 合格标准 |
|---------|---------|---------|
| 营业执照 | 核查有效期 | 在有效期内 |
| 施工资质证书 | 核查等级 | 满足招标文件要求 |
| 安全生产许可证 | 核查有效期 | 在有效期内 |
| 项目经理资格 | 核查注册证 | 符合要求 |
| 类似工程业绩 | 核查合同+竣工证明 | 满足数量要求 |
| 信用评级 | 系统查询 | 无不良记录 |

### 3.3 澄清与说明规定

依据《评标委员会和评标方法暂行规定》第19条：
- 评标委员会**可以**要求投标人澄清的情形：
  - 投标文件中存在含义不明确的内容
  - 同类问题表述不一致
  - 有明显文字和计算错误
- 评标委员会**不得**接受的澄清：
  - 改变投标价格
  - 改变技术方案实质性内容
  - 补充提交投标文件缺少的材料

## 四、评标报告编制

### 4.1 评标报告必备内容

依据《评标委员会和评标方法暂行规定》第40条：
1. 基本情况和数据表（投标人名单、价格）
2. 评标委员会成员名单
3. 否决投标情况说明
4. 评标标准、方法或者因素的说明
5. 经评审的投标人排序
6. 推荐的中标候选人名单及理由
7. 澄清、说明、补正事项
8. 评标委员会成员签字盖章

### 4.2 中标候选人推荐

- 推荐数量：一般推荐1-3名
- 排名方式：按评分由高到低（综合评估法）或评标价由低到高（最低价法）
- 第一中标候选人：评标结果最优者

### 4.3 否决投标情形

| 情形 | 法律依据 |
|-----|---------|
| 投标文件未按要求密封、签章 | 招标文件规定 |
| 未按要求提交投标保证金 | 《招标投标法》第28条 |
| 与招标文件有重大偏差 | 招标文件规定 |
| 提供虚假材料 | 《招标投标法实施条例》第51条 |
| 串通投标 | 《招标投标法》第32条 |
| 低于成本价竞争 | 《招标投标法》第33条 |

## 五、开标程序

### 5.1 开标程序流程

1. 主持人宣布开标会纪律
2. 确认投标人代表身份
3. 检查投标文件密封情况（由投标人代表确认）
4. 当众开封（按投标文件提交时间逆序）
5. 宣读投标人名称、投标报价、工期承诺、质量承诺
6. 开标记录表签字（投标人代表+主持人+监督员）
7. 投标文件移交评标委员会

### 5.2 开标异常处理

| 异常情形 | 处理方式 |
|---------|---------|
| 密封不严 | 记录，提交评标委员会决定 |
| 缺席投标人 | 代为开封，记录 |
| 发现串通嫌疑 | 及时报告监督部门 |
| 撤销投标文件 | 宣读前可退回，宣读后不予退回 |
