## 二、关键法律条文

### 2.1 《招标投标法》关键条款

**第3条** 在中国境内进行下列工程建设项目包括项目的勘察、设计、施工、监理以及与工程建设有关的重要设备、材料等的采购，必须进行招标：
1. 大型基础设施、公用事业等关系社会公共利益、公众安全的项目；
2. 全部或者部分使用国有资金投资或者国家融资的项目；
3. 使用国际组织或者外国政府贷款、援助资金的项目。

**第10条** 招标分为公开招标和邀请招标。

**第11条** 国务院发展计划部门确定的国家重点项目和省、自治区、直辖市人民政府确定的地方重点项目不适宜公开招标的，经国务院发展计划部门或者省、自治区、直辖市人民政府批准，可以进行邀请招标。

**第22条** 招标人不得向他人透露已获取招标文件的潜在投标人的名称、数量以及可能影响公平竞争的有关招标投标的其他情况。

**第28条** 投标人应当在招标文件要求提交投标文件的截止时间前，将投标文件送达投标地点。招标人收到投标文件后，应当签收保存，不得开启。投标人少于3个的，招标人应当依照本法重新招标。

**第32条** 投标人不得相互串通投标报价，不得排挤其他投标人的公平竞争，损害招标人或者其他投标人的合法权益。

**第33条** 投标人不得以低于成本的报价竞标，也不得以他人名义投标或者以其他方式弄虚作假，骗取中标。

**第37条** 评标由招标人依法组建的评标委员会负责。依法必须进行招标的项目，其评标委员会由招标人的代表和有关技术、经济等方面的专家组成，成员人数为5人以上单数，其中技术、经济等方面的专家不得少于成员总数的三分之二。

**第46条** 招标人和中标人应当自中标通知书发出之日起三十日内，按照招标文件和中标人的投标文件订立书面合同。

### 2.2 《招标投标法实施条例》关键条款

**第27条** 招标文件不得要求或者标明特定的生产供应者以及含有倾向或者排斥潜在投标人的其他内容。

**第39条** 禁止投标人相互串通投标。有下列情形之一的，属于投标人相互串通投标：
1. 投标人之间协商投标报价等投标文件的实质性内容；
2. 投标人之间约定中标人；
3. 投标人之间约定部分投标人放弃投标或者中标；
4. 属于同一集团、协会、商会等组织成员的投标人按照该组织要求协同投标；
5. 投标人之间为谋取中标或者排斥特定投标人而采取的其他联合行动。

**第51条** 有下列情形之一的，评标委员会应当否决其投标：
1. 投标文件未经投标单位盖章和单位负责人签字；
2. 投标联合体没有提交共同投标协议；
3. 投标人不符合国家或者招标文件规定的资格条件；
4. 同一投标人提交两个以上不同的投标文件或者投标报价，但招标文件要求提交备选投标的除外；
5. 投标报价低于成本或者高于招标文件设定的最高投标限价；
6. 投标文件没有对招标文件的实质性要求和条件作出响应；
7. 投标人有串通投标、弄虚作假、行贿等违法行为。

## 三、投标保证金规定

### 3.1 国家规定

依据《招标投标法实施条例》第26条：
- 投标保证金不得超过招标项目估算价的**2%**
- 最高不得超过**80万元人民币**

依据《招标投标法》第44条：
- 招标人收取保证金后，应在中标候选人确定后**5日内**退还未中标者保证金
- 中标人在签订合同后**5日内**退还其保证金

### 3.2 履约保证金

依据《招标投标法实施条例》第58条：
- 履约保证金不得超过中标合同金额的**10%**

## 四、招标时限汇总

| 环节 | 时限要求 | 法律依据 |
|-----|---------|---------|
| 资格预审公告发布 | 距截止提交申请≥5天 | 条例第16条 |
| 招标文件发售时限 | 自发出之日起≥5个工作日 | 条例第21条 |
| 投标准备时间 | 自发售日到截止日≥20天 | 招标法第24条 |
| 投标文件澄清时限 | 投标截止前≥15天 | 招标法第23条 |
| 联合体资格预审 | 联合体各方均须满足 | 条例第37条 |
| 中标通知书发出 | 评标完成后≤30天 | 招标法第45条 |
| 合同签订 | 中标通知书发出后≤30天 | 招标法第46条 |
| 中标候选人公示 | ≥3个工作日 | 条例第54条 |
| 投诉提出时限 | 知道之日起10日内 | 条例第22条 |

## 五、政府采购特别规定

### 5.1 政府采购方式

| 采购方式 | 适用条件 |
|---------|---------|
| 公开招标 | 货物或服务≥120万元 |
| 邀请招标 | 特殊专业性较强、供应商少 |
| 竞争性谈判 | 技术复杂、无法确定详细规格 |
| 单一来源采购 | 只有唯一供应商、紧急情况 |
| 询价 | 采购货物规格标准统一 |
| 竞争性磋商 | PPP项目、政府购买服务 |

### 5.2 政府采购特殊要求

- **国内采购优先**：进口产品需报批
- **中小企业优惠**：预留30%份额
- **节能环保要求**：优先采购节能环保产品
- **信息公开**：采购意向、招标公告、结果公告均须公开

## 六、GB/T 50500-2024 新规定要点

### 6.1 主要变化

1. **适用范围扩大**：从强制性标准改为推荐性标准，适用于建设工程全过程计价
2. **费用项目更新**：增加绿色建筑、装配式、BIM等新内容
3. **电子招标投标**：明确电子招标投标计价要求
4. **工程计量**：细化工程计量规则

### 6.2 工程量清单五要素

依据GB/T 50500-2024，工程量清单五要素：
1. **项目编码**：12位数字组成
2. **项目名称**：按规范规定名称
3. **项目特征**：影响价格的主要特征
4. **计量单位**：按规范规定
5. **工程量**：按规范计算规则计算

### 6.3 计价方式

| 计价方式 | 说明 |
|---------|------|
| 工程量清单计价 | 综合单价×清单工程量 |
| 定额计价 | 定额工程量×定额综合单价 |
| 参考指标 | 工程费用参考指标×建筑面积 |
