# 中国建设工程商事调解 v2.0

## 技能定位

本技能为中国建设工程领域商事调解提供**全流程专家级支持**，覆盖从纠纷诊断到协议履行的完整链条。

**适用对象**：调解员、律师、造价工程师、项目管理人员、业主方/承包方法务人员。

**核心法规**：以《商事调解条例》（国令第827号，2026年5月1日施行）为**首要法律依据**，该条例第2条明确将"工程建设"纳入商事调解适用范围，为建设工程商事调解提供了直接的行政法规支撑。

**法律依据体系**：

| 层级 | 名称 | 关键条款 |
|:---:|------|---------|
| 行政法规 | 《商事调解条例》（国令第827号，2026.5.1施行） | 第2条（适用范围含工程建设）、第14-23条（调解程序） |
| 法律 | 《人民调解法》 | 第17-33条（调解程序、协议效力） |
| 法律 | 《民事诉讼法》 | 第201-202条（司法确认程序） |
| 法律 | 《仲裁法》 | 第51条（仲裁调解） |
| 法律 | 《民法典》 | 第807条（建设工程价款优先受偿权） |
| 法律 | 《建筑法》 | 第60-62条（质量责任） |
| 司法解释 | 《建工司法解释（一）》（法释〔2020〕25号） | 全文 |
| 司法解释 | 《建工司法解释（二）》（法释〔2018〕20号） | 全文 |
| 行业标准 | GF-2017-0201（施工合同示范文本） | 变更、索赔、结算条款 |
| 国家标准 | GB/T 50500-2024（工程量清单计价标准） | 计价规则 |

## 核心能力矩阵

### 能力一：纠纷智能诊断

**输入**：用户描述的纠纷事实（可非结构化）
**输出**：结构化争议焦点清单

**分析维度**：
1. **争议类型识别**：合同类 / 工期类 / 质量类 / 造价类 / 安全类
2. **法律关系定性**：合同效力、履行状态、违约形态
3. **金额评估**：争议标的额、合理区间、谈判空间
4. **时效审查**：诉讼时效、除斥期间、仲裁时效
5. **可调解性评估**：双方意愿、利益交叉点、历史合作情况

**输出格式**：
```markdown
## 纠纷诊断报告

### 争议类型：[主类型 / 子类型]
### 争议标的：¥XXX（置信区间：¥XXX ~ ¥XXX）
### 紧急度：🔴 高 / 🟡 中 / 🟢 低

#### 争议焦点清单
| # | 焦点 | 金额 | 责任方 | 法律依据 | 证据充分度 |
|---|------|------|--------|---------|-----------|
| 1 | ... | ¥XXX | 甲方/乙方 | 条款X | ★★★★☆ |

#### 可调解性评估
- 调解可行性：高/中/低
- 预计难度：简单/中等/复杂
- 建议策略：评估式/促进式/混合式
```

### 能力二：五方利益图谱分析

系统化识别和分析建设工程纠纷中的五大核心利益方：

```
五方利益图谱
├── 发包人（业主/建设单位）
│   ├── 核心利益：控制投资、保证工期、工程质量
│   ├── 谈判筹码：付款权、验收权、后续合作
│   └── 潜在顾虑：工期延误损失、重复建设成本
├── 承包人（施工总承包）
│   ├── 核心利益：收回工程款、合理利润、工期合理
│   ├── 谈判筹码：停工权、优先受偿权、索赔权
│   └── 潜在顾虑：资金链断裂、资质影响、分包商追索
├── 分包商 / 供应商
│   ├── 核心利益：分包款/货款回收
│   ├── 谈判筹码：实际施工人地位（可诉业主）
│   └── 潜在顾虑：被总包转嫁风险
├── 设计单位 / 监理单位
│   ├── 核心利益：服务费收取、责任边界清晰
│   └── 潜在顾虑：设计变更连带责任、监理失职风险
└── 保险机构 / 担保方
    ├── 核心利益：控制赔付范围
    └── 潜在顾虑：理赔争议扩大
```

### 能力四：调解方案设计

三种模式适配不同场景：

#### 模式A：评估式调解（适合法律争议明确型）
- 调解员给出专业评估意见
- 侧重法律风险分析和裁判预测
- 适合：当事人法律认知差异大、需要权威判断

#### 模式B：促进式调解（适合关系维护型）
- 引导双方自主沟通、探索利益
- 侧重情感疏导和方案共创
- 适合：长期合作方、注重未来关系

#### 模式C：混合式调解（推荐默认模式）
- 第一阶段：促进式——各自陈述、梳理事实
- 第二阶段：评估式——法律分析、风险提示
- 第三阶段：方案磋商——在理性基础上达成一致

**输出格式**：
```markdown
## 调解方案

### 方案概要
- 调解模式：混合式（促进→评估→磋商）
- 预计周期：X-X次调解会议
- 时间安排：[具体日期]

### 议程设计
| 阶段 | 内容 | 预计时长 | 方式 |
|------|------|---------|------|
| 第一阶段 | 事实陈述与争议梳理 | 2-3小时 | 联席会议 |
| 第二阶段 | 法律分析与风险评估 | 1-2小时 | 背靠背 |
| 第三阶段 | 方案磋商与条款谈判 | 2-4小时 | 联席+背靠背 |

### 关键节点
1. [节点1]：[目标]
2. [节点2]：[目标]
3. [节点3]：[目标]
```

### 能力五：调解协议起草

生成规范、完整的调解协议，包含以下核心要素：

> **法律基础**：依据《商事调解条例》第22条，调解协议应载明主要事实、争议事项、协议内容、履行方式与期限，由调解员签名并加盖调解组织印章，**具有法律约束力**。

**必备条款**：
1. 当事方基本信息
2. 争议事实概述
3. 调解结果（金额/行为/时间）
4. 履行方式和期限
5. 违约责任与救济
6. 保密条款
7. 权利处分声明
8. 争议解决后备条款

**可选条款**（根据情况选用）：
- 司法确认申请条款
- 担保/保证条款
- 分期履行条款
- 继续合作条款
- 声明与承诺条款

> 详见 `references/agreement-templates.md`

### 能力六：司法确认指导

提供调解协议司法确认的完整指引：

1. **时限**：调解协议生效后30日内
2. **管辖**：调解组织所在地基层人民法院
3. **法律依据**：
   - 《商事调解条例》第23条（商事调解协议司法确认）
   - 《民事诉讼法》第201-202条（司法确认程序）
   - 最高人民法院《关于人民调解协议司法确认程序的若干规定》
4. **材料清单**：
   - 司法确认申请书（双方签字）
   - 调解协议书原件
   - 调解组织证明
   - 当事人身份证明
   - 证据材料
4. **审查要点**：自愿性、合法性、明确性
5. **裁定效力**：具有强制执行效力

> 详见 `references/judicial-confirmation.md`

## 典型争议类型与处理指引

### 一、工程款支付纠纷（最常见）

**特征**：发包人拖欠/少付工程款

**关键分析点**：
- 竣工验收状态（是否已验收）
- 付款节点是否已到达
- 是否存在质量抗辩
- 优先受偿权时效（竣工后18个月）
- 欠款金额及利息计算

**法律依据**：
- 《商事调解条例》第2条（工程建设争议适用商事调解）、第22条（调解协议法律约束力）
- 《民法典》第807条（优先受偿权）
- GF-2017-0201 第14条（竣工结算）
- 《建工司法解释（一）》第27条（利息起算）

**调解策略倾向**：评估式（法律关系相对清晰）

---

### 二、工期延误争议

**特征**：发包人主张工期延误索赔 / 承包人主张工期顺延

**关键分析点**：
- 实际开工日期认定
- 延误原因分类（甲方/乙方/不可抗力/第三方）
- 关键路径影响分析
- 工期签证/索赔程序是否合规
- 延误损失计算

**法律依据**：
- 《商事调解条例》第14条（自愿、合法、诚信、保密原则）、第17条（适用行业规则和商业惯例）
- GF-2017-0201 第7条（工期）
- GF-2017-0201 第19条（不可抗力）
- 《建工司法解释（一）》第10-11条

**调解策略倾向**：混合式（技术+法律）

---

### 三、工程质量争议

**特征**：工程质量不达标引发的修复/赔偿争议

**关键分析点**：
- 质量标准适用（设计标准/国家标准/合同约定）
- 缺陷成因鉴定（施工/设计/材料/使用不当）
- 修复方案与费用
- 缺陷责任期是否届满
- 鉴定必要性评估

**法律依据**：
- 《商事调解条例》第17条（适用行业规则）、第19条（保密原则）
- 《建筑法》第60-62条
- 《民法典》第801-802条
- GF-2017-0201 第15条（缺陷责任）

**调解策略倾向**：促进式（需技术鉴定辅助）

---

### 四、变更签证争议

**特征**：施工过程中的设计变更、工程签证引发的价款争议

**关键分析点**：
- 变更指令的有效性（书面形式）
- 变更工程量确认
- 变更计价方式（合同已有/类似/无适用）
- 签证审批流程是否合规
- 变更对工期的影响

**法律依据**：
- 《商事调解条例》第17条（适用交易习惯和商业惯例）、第22条（调解协议效力）
- GF-2017-0201 第10.1-10.4条（变更）
- GB/T 50500-2024（变更计价规则）

**调解策略倾向**：评估式（计价规则明确）

---

### 五、结算争议

**特征**：竣工结算金额的确认争议

**关键分析点**：
- 结算资料的完整性
- 清单工程量核实
- 材料调差计算
- 设计变更是否计入
- 违约金是否应扣除
- 结算审核期限合规性

**法律依据**：
- 《商事调解条例》第22条（调解协议法律约束力）、第23条（司法确认）
- GF-2017-0201 第14条
- GB/T 50500-2024
- 《建工司法解释（一）》第21条

**调解策略倾向**：评估式（数据驱动）

### 标准输出模块

每次响应根据场景选择以下模块组合：

- **[诊断]** 争议类型识别与焦点分析
- **[图谱]** 利益方关系与诉求对比
- **[BATNA]** 谈判替代方案评估
- **[策略]** 调解模式与方案设计
- **[法律]** 法律法规与裁判规则引用
- **[文书]** 调解文书/协议草稿
- **[确认]** 司法确认程序指引
- **[风险]** 风险提示与注意事项

### 引用规范

引用法规时必须标注：**法规名称 + 具体条文编号**

正确示例：
> 依据《商事调解条例》第2条，工程建设领域的商事争议属于商事调解适用范围。
> 依据《商事调解条例》第22条，经商事调解达成的协议具有法律约束力，当事人应当履行。
> 依据《建设工程施工合同示范文本》GF-2017-0201 第10.1款，承包人应按照发包人批准的施工图设计文件施工。

错误示例：
> 按照合同规定，施工应按图施工。

## 边界说明

1. 本技能聚焦**中国建设工程领域**商事调解，其他行业或法域纠纷建议切换对应技能
2. 涉及**重大争议（>1000万元）**或**群体性纠纷**时，建议提醒用户多方论证
3. 法律意见**仅供参考**，不构成正式法律建议，具体案件请咨询执业律师
4. 调解协议草稿须经专业律师审核后方可签署使用
5. 不涉及刑事案件、行政案件的处理
