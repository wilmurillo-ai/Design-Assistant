# 司法确认全流程指引

## 一、概述

调解协议达成后，当事人可向人民法院申请**司法确认**，经法院裁定确认后，该调解协议即具有**强制执行效力**——一方不履行时，另一方可直接向法院申请强制执行，无需再经过诉讼。

**法律依据**：
- 《民事诉讼法》第201-202条
- 《人民调解法》第33条
- 最高人民法院《关于人民调解协议司法确认程序的若干规定》（法释〔2011〕5号）

---

## 二、申请条件

### 2.1 积极条件

1. 调解协议经依法设立的调解组织调解达成
2. 双方当事人共同申请
3. 自调解协议生效之日起30日内提出
4. 向调解组织所在地的基层人民法院提出

### 2.2 消极条件（不予确认的情形）

1. 违反法律、行政法规强制性规定的
2. 侵害国家利益、社会公共利益的
3. 侵害案外人合法权益的
4. 损害社会公序良俗的
5. 内容不明确，无法确认和执行的
6. 当事人在意思表示不真实的情况下达成的（胁迫、欺诈等）
7. 当事人不具备民事行为能力的
8. 调解组织、调解员有索贿受贿、徇私舞弊、枉法调解行为的

---

## 三、申请时限

- **30日**：自调解协议生效之日起计算
- **不可延长**：超过30日未申请，只能通过诉讼方式解决
- **起算点**：通常为双方签署调解协议的日期

> ⚠️ **重要提示**：务必在调解协议签署后立即准备司法确认材料，确保在30日内提交。

---

## 四、管辖法院

- **调解组织所在地**的基层人民法院
- 不适用级别管辖和地域管辖的一般规定
- 常见受理法院：区/县人民法院

---

| 序号 | 材料 | 份数 | 要求 |

## 六、审查程序

### 6.1 立案

- 向法院立案窗口提交材料
- 法院应当在**收到申请之日起3日内**决定是否受理
- 法院应当向双方当事人发送受理通知书

### 6.2 审查方式

- **书面审查为主**：法院主要审查书面材料
- **听证审查为辅**：法院认为有必要时，可通知双方当事人到场
- **审查重点**：
  - 调解协议是否自愿达成
  - 调解协议内容是否合法
  - 调解协议内容是否明确可执行

### 6.3 审查期限

- 法院应当自受理之日起**15日内**作出裁定
- 有特殊情况需要延长的，经批准可延长

### 6.4 听证要点

如法院组织听证，可能涉及：

| 审查项 | 法官关注点 | 当事人应准备 |
|--------|-----------|-------------|
| 自愿性 | 是否有胁迫、欺诈 | 如实陈述调解过程 |
| 身份核实 | 申请人是否适格 | 身份证明、授权文件 |
| 协议内容 | 是否合法、明确 | 协议原件、合同依据 |
| 事实基础 | 协议是否有事实支撑 | 合同、签证、验收资料等 |

## 七、裁定结果

### 7.1 确认裁定

**裁定书内容**：
- 确认某年某月某日达成的调解协议有效
- 一方拒绝履行时，对方可直接向法院申请执行

**效力**：
- 赋予调解协议**强制执行力**
- 与法院判决书具有同等执行效力
- 不服裁定的，可向上一级法院申请复议（**不可上诉**）

### 7.2 不予确认裁定

**可能原因**：
- 调解协议内容违法
- 非自愿达成
- 内容不明确

**救济途径**：
- 申请人可通过诉讼方式解决争议
- 如对不予确认裁定不服，可申请复议

---

## 八、强制执行

### 8.1 申请执行条件

- 对方不履行经司法确认的调解协议
- 向作出确认裁定的法院申请
- 申请执行期限：**2年**（自履行期限届满之日起算）

### 8.2 执行措施

- 查封、扣押、冻结财产
- 扣划银行存款
- 拍卖变卖财产
- 纳入失信被执行人名单
- 限制高消费

```
调解协议签署
    │
    ├──── 0-30日 ────→ 提交司法确认申请
    │                      │
    │                      ├──── 3日内 ────→ 法院决定是否受理
    │                      │                      │
    │                      │                      ├──── 15日内 ────→ 法院作出裁定
    │                      │                      │
    │                      │                      ├──→ 裁定确认 → 调解协议具有强制执行力
    │                      │                      └──→ 裁定不予确认 → 可另行诉讼
    │                      │
    │                      └──→ 不予受理 → 补正或另行诉讼
    │
    └──── 超过30日 ──→ 丧失司法确认资格，只能诉讼
```

## 十、常见问题

### Q1：司法确认收费吗？

**不收费**。司法确认案件不收取案件受理费。

### Q2：一方不愿申请司法确认怎么办？

司法确认需要**双方共同申请**。如一方不愿，另一方只能通过诉讼方式解决。建议在调解协议中预先约定双方共同申请司法确认的义务。

### Q3：司法确认裁定能上诉吗？

**不能**。但当事人对裁定不服，可以向作出裁定的法院申请复议一次。

### Q4：经司法确认后，还能再起诉吗？

**不能**。调解协议经司法确认后，就同一纠纷再起诉的，法院不予受理。

### Q5：商业调解组织达成的协议能申请司法确认吗？

根据2022年《民事诉讼法》修正及各地实践，商事调解组织达成的调解协议也可申请司法确认。建议提前了解当地法院的具体实践。

## 十一、建议在调解协议中的约定

为确保司法确认顺利进行，建议在调解协议中增加以下条款：

```
第X条 司法确认
    双方同意，在本协议签署后XX日内，共同向[调解组织所在地名]
    基层人民法院申请对本协议进行司法确认。双方承诺积极配合
    司法确认程序，及时提供所需材料。
```

这样约定可以：
1. 明确双方共同申请的义务
2. 设定申请时限（提醒双方及时办理）
3. 在一方怠于配合时，为另一方提供合同依据
