# 调解协议模板库

## 一、建设工程调解协议（标准版）

```
                        调解协议书

编号：[调解组织缩写]调字〔XXXX〕第XX号

甲方（发包人）：[名称]
统一社会信用代码：[代码]
法定代表人：[姓名]
联系地址：[地址]

乙方（承包人）：[名称]
统一社会信用代码：[代码]
法定代表人：[姓名]
联系地址：[地址]

（如有丙方/丁方，依次列明）

调解组织：[名称]
调解员：[姓名]

鉴于：
1. 甲方与乙方于XXXX年XX月XX日签订了《[合同名称]》（合同编号：XXX），
   约定由乙方承建甲方[项目名称]工程（以下简称"本工程"）。
2. 本工程于XXXX年XX月XX日开工，[已/未]竣工验收。
3. 在合同履行过程中，双方因[争议事由]发生争议。
4. 双方于XXXX年XX月XX日共同向[调解组织]申请调解。
5. 经调解员主持调解，双方本着平等、自愿、诚信原则，经充分协商，
   现就争议事项达成如下协议：

第一条 争议事实确认
    双方共同确认以下事实：
    1.1 [事实确认1]
    1.2 [事实确认2]
    1.3 [事实确认3]

第二条 争议解决方案
    2.1 甲方同意于XXXX年XX月XX日前向乙方支付人民币[金额]元
       （大写：[中文金额]）。
    2.2 付款方式：[一次性/分期]
       （如分期：第一次于XX日前支付XX元，第二次于XX日前支付XX元...）
    2.3 付款账户：
       户名：[名称]
       开户行：[银行名称]
       账号：[账号]
    2.4 乙方同意[乙方让步内容]。

第三条 工程后续安排
    3.1 [如有未完工程、保修义务等，约定后续安排]

第四条 违约责任
    4.1 如甲方未按本协议第二条约定付款，每逾期一日，应按未付金额的
       [万分之X]向乙方支付违约金。
    4.2 如乙方未按本协议第三条约定履行义务，应承担[具体违约责任]。
    4.3 任何一方违约，守约方有权要求违约方继续履行、采取补救措施，
       并赔偿因此造成的损失。

第五条 保密条款
    5.1 双方对本协议内容及调解过程中知悉的对方商业秘密负有保密义务。
    5.2 未经对方书面同意，任何一方不得向第三方披露。
    5.3 保密期限自本协议签署之日起至[时间/争议公开之日]止。

第六条 权利处分声明
    6.1 双方确认，基于本协议的签署，双方就本协议约定的争议事项
       互不追究对方其他责任。
    6.2 除本协议另有约定外，双方在本协议签署前因本工程产生的一切
       债权债务关系即告了结。

第七条 争议解决
    7.1 本协议在履行过程中发生的争议，由双方协商解决。
    7.2 协商不成的，任何一方可向[管辖法院/仲裁机构]提起诉讼/仲裁。

第八条 其他约定
    8.1 本协议自双方签字（盖章）之日起生效。
    8.2 本协议一式[X]份，甲方执[X]份，乙方执[X]份，
       [调解组织]执[X]份，具有同等法律效力。
    8.3 本协议未尽事宜，由双方另行签订补充协议。

（以下无正文）

甲方（盖章）：                     乙方（盖章）：

法定代表人/授权代表：              法定代表人/授权代表：

签字：                            签字：

日期：XXXX年XX月XX日               日期：XXXX年XX月XX日

调解员（签字）：
日期：XXXX年XX月XX日

调解组织（盖章）：
日期：XXXX年XX月XX日
```

## 二、分期付款调解协议（工程款纠纷专用）

**特殊条款**：

```
第二条 分期付款安排
    2.1 甲方同意按以下方式向乙方分期支付工程款：

    ┌──────┬──────────┬──────────┬──────────┐
    │ 期数 │ 付款日期  │ 付款金额  │  累计金额  │
    ├──────┼──────────┼──────────┼──────────┤
    │ 第一期│ XXXX/XX/XX│ ¥XXX   │ ¥XXX    │
    │ 第二期│ XXXX/XX/XX│ ¥XXX   │ ¥XXX    │
    │ 第三期│ XXXX/XX/XX│ ¥XXX   │ ¥XXX    │
    └──────┴──────────┴──────────┴──────────┘

    2.2 逾期付款违约金：每逾期一日，按该期未付金额的万分之三计算。
    2.3 逾期超过XX日的，乙方有权要求甲方一次性支付全部剩余未付款项。
    2.4 甲方支付完毕全部款项后，乙方应在XX日内配合甲方完成[工程验收/
       资料移交/备案登记]等手续。

第三条 担保条款（可选）
    3.1 [担保人名称]自愿为甲方履行本协议项下的付款义务提供连带保证
       担保，保证期间为最后一期付款期限届满之日起两年。
    3.2 担保范围：主债权、逾期付款违约金及乙方实现债权的费用。
```

    (1) [缺陷描述1]，位于[位置]

    3.1.1 [修复方案1]，预计费用¥XXX，工期XX日历天

    3.2 修复质量标准：符合[设计要求/国家规范标准]
    3.3 修复期限：自本协议签署之日起XX日内完成

第四条 修复验收
    4.1 修复完成后，由双方共同组织验收。
    4.2 如验收合格，甲方应在XX日内确认。

## 七、调解协议起草注意事项

### 7.1 必备条款检查清单

- [ ] 当事方基本信息完整准确
- [ ] 争议事实陈述清楚
- [ ] 调解结果具体明确（金额精确到元）
- [ ] 履行方式和期限确定
- [ ] 违约责任清晰
- [ ] 保密条款（建议包含）
- [ ] 权利处分声明
- [ ] 双方签字盖章
- [ ] 调解员签字
- [ ] 调解组织盖章
- [ ] 协议签署日期

### 7.2 常见错误规避

| 错误 | 后果 | 正确做法 |
|------|------|---------|
| 金额表述模糊（如"约XXX万"） | 履行争议 | 写明精确金额（精确到分） |
| 付款时间不确定（如"尽快"） | 履行争议 | 写明具体日期或触发条件 |
| 遗漏部分争议事项 | 残留纠纷 | 一次性解决所有关联争议 |
| 条款相互矛盾 | 解释争议 | 起草后全文审查逻辑一致性 |
| 缺少权利处分条款 | 纠纷复发 | 明确"本协议签署后互不追究" |

### 7.3 司法确认审查要点

协议必须满足三个条件才能通过司法确认：

1. **自愿性**：双方系自愿达成，无胁迫、欺诈
2. **合法性**：内容不违反法律、行政法规强制性规定
3. **明确性**：协议内容具体、确定，可执行

> 特别注意：涉及工程款分期支付的，建议在协议中约定"如任何一期未按约支付，守约方有权要求一次性支付全部剩余款项"的加速到期条款。
