# 建设工程争议类型分类与处理指引

## 一、合同类争议

### 1.1 工程款支付争议

**适用场景**：发包人未按合同约定支付工程款，是最常见的工程纠纷类型。

**常见原因**：
- 竣工结算未完成
- 质量异议导致拒付
- 资金链断裂
- 发包人恶意拖延
- 审计程序未完成

**核心争议点**：
- 付款条件是否成就（验收/结算/审计）
- 应付金额（含变更签证、索赔）
- 欠付利息起算时间
- 优先受偿权是否成立

**关键证据**：
- 施工合同（含付款条款）
- 竣工验收报告
- 结算报告及审计意见
- 催款函件往来记录
- 工程量清单与计价书

**法律依据**：
| 条款 | 内容 | 关键要点 |
|------|------|---------|
| 《民法典》第779条 | 工作成果报酬 | 承包人交付工作成果后，定作人应当支付报酬 |
| 《民法典》第807条 | 建设工程价款优先受偿权 | 建筑工程价款包括人工、材料等实际支出，不含利润、违约金 |
| 《建工司法解释（一）》第27条 | 欠付工程款利息 | 当事人对欠付工程价款利息计付标准有约定的，按约定；无约定的，按同期LPR |
| 《建工司法解释（一）》第36条 | 优先受偿权范围 | 不包括预期利润、违约金、损害赔偿 |
| 《建工司法解释（一）》第41条 | 优先受偿权期限 | 自应当给付工程价款之日起18个月 |
| GF-2017-0201 第14.2条 | 竣工付款 | 发包人应在收到竣工结算申请后28天内核实并支付 |

**事实查明清单**：
- [ ] 合同约定的付款节点和条件
- [ ] 实际完成进度和验收情况
- [ ] 已支付金额和付款时间
- [ ] 欠付金额计算依据
- [ ] 优先受偿权时效状态
- [ ] 是否存在质量抗辩
- [ ] 区分争议金额与无争议金额

**调解策略**：
- 优先确定已完工程量及对应价款
- 区分争议金额与无争议金额，先推动无争议部分支付
- 评估质量异议的合理性
- 考虑分期付款方案缓解发包人资金压力
- 优先受偿权作为谈判筹码
- 利息可作为让步空间

---

### 1.2 变更签证争议

**适用场景**：施工过程中因设计变更、现场签证引发的价款/工期争议

**常见原因**：
- 变更指令未书面确认
- 签证金额双方分歧
- 变更后价格未协商一致
- 监理拒绝签认

**核心争议点**：
- 变更指令的有效性（书面形式要求）
- 变更工程量的确认
- 变更计价方法
- 签证审批程序的合规性

**关键证据**：
- 变更指令单（书面/口头记录）
- 现场签证单
- 监理日志
- 往来邮件/微信记录
- 变更前后工程量对比

**法律依据**：
| 条款 | 内容 | 关键要点 |
|------|------|---------|
| GF-2017-0201 第10.1条 | 变更范围 | 经批准的施工图设计文件、变更导致工程量/质量/工期变化 |
| GF-2017-0201 第10.2条 | 变更程序 | 发包人指示 → 承包人执行（紧急情况除外） |
| GF-2017-0201 第10.4条 | 变更估价 | 已有适用→按原价；类似→参照适用；无适用→协商/按定额 |
| GB/T 50500-2024 第9.3条 | 工程变更计价 | 变更导致措施费调整 |

**事实查明清单**：
- [ ] 变更是否经发包人/监理书面批准
- [ ] 紧急情况下的口头指令是否有事后确认
- [ ] 变更工程量的签证记录
- [ ] 合同中关于变更计价的约定
- [ ] 变更对工期的影响

**调解策略**：
- 还原变更指令的发出过程（谁授权、何时、如何传达）
- 按计价规则重新核算变更价款
- 区分有争议变更和无争议变更

---

### 1.3 索赔争议

**适用场景**：一方主张合同外的额外费用或工期延长

**常见索赔原因**：
- 图纸延误或图纸错误
- 业主提供的材料设备延误
- 暂停施工指令
- 恶劣气候（超过合同约定标准）
- 不可预见地质条件

**核心争议点**：
- 索赔事件是否属于合同约定范围
- 索赔程序是否合规（28天通知期）
- 索赔金额/工期的合理性
- 监理是否签认

**法律依据**：
| 条款 | 内容 | 关键要点 |
|------|------|---------|
| GF-2017-0201 第19.1条 | 索赔程序 | 知道/应知道索赔事件后28天内提出索赔意向通知书 |
| GF-2017-0201 第19.2条 | 索赔报告 | 索赔意向后28天内提交详细索赔报告 |
| GF-2017-0201 第19.5条 | 索赔时效 | 未在28天内提出视为放弃索赔权 |

**调解策略**：
- 核实索赔通知的时效性（28天通知要求）
- 评估事件与损失之间的因果关系
- 区分可索赔费用和不可索赔费用

---

### 1.4 合同解除争议

**适用场景**：一方主张解除施工合同引发的善后处理

**核心争议点**：
- 解除权的行使条件是否成就
- 已完工程的价款结算
- 解除后的损失赔偿范围

**法律依据**：
| 条款 | 内容 | 关键要点 |
|------|------|---------|
| 《民法典》第563条 | 法定解除 | 不可抗力/预期违约/迟延履行/根本违约 |
| GF-2017-0201 第16.2条 | 发包人解除 | 承包人转包/违法分包/明确表示不履行 |
| GF-2017-0201 第16.3条 | 承包人解除 | 发包人未按约支付/提供主要材料/场地 |
| 《建工司法解释（一）》第11条 | 合同解除后结算 | 已完工程经验收合格的，按合同约定结算价款 |

## 二、工期类争议

### 2.1 工期延误责任认定

**适用场景**：工程未按合同约定工期竣工，各方互相推责。

**责任方识别框架**：

| 延误原因 | 责任归属 | 处理原则 |
|---------|---------|---------|
| 设计图纸延误 | 发包人 | 工期顺延+费用补偿 |
| 甲供材料延误 | 发包人 | 工期顺延+费用补偿 |
| 承包人自身原因 | 承包人 | 支付逾期违约金 |
| 不可抗力 | 共担 | 工期顺延，费用各自承担 |
| 分包商原因 | 承包人 | 内部追偿 |

**关键分析工具**：关键路径法（CPM）分析，识别影响竣工日期的关键线路延误。

**核心争议点**：
- 延误原因归属（发包人/承包人/第三方/不可抗力）
- 关键路径影响分析
- 延误天数计算
- 工期索赔程序合规性

**分析方法**：
1. **实际工期对比**：合同工期 vs 实际工期
2. **延误事件梳理**：按时间线列出所有延误事件
3. **原因分类**：
   - 甲方原因：设计变更、场地交付迟延、甲供材迟延、图纸交付迟延
   - 乙方原因：人员不足、管理不善、安全事故
   - 不可抗力：自然灾害、政策调整、疫情管控
   - 第三方：政府部门审批、相邻关系影响
4. **关键路径分析**：延误事件是否影响关键路径
5. **顺延天数计算**：仅计入影响关键路径的甲方/不可抗力原因

**法律依据**：
| 条款 | 内容 | 关键要点 |
|------|------|---------|
| GF-2017-0201 第7.5条 | 工期顺延 | 顺延理由和天数需经发包人/监理确认 |
| GF-2017-0201 第7.9条 | 工期奖励 | 提前竣工奖励按合同约定 |
| GF-2017-0201 第19条 | 不可抗力 | 不可抗力导致工期延误的顺延 |

**调解策略**：
- 重建施工进度计划（投标 vs 实际）
- 识别各延误事件的责任方及影响工期
- 计算并发延误时的责任分担

---

### 2.2 工期赶工费用

**适用场景**：发包人要求缩短合同工期，承包人主张赶工费；或发包人要求赶工但未支付赶工费。

**核心争议点**：
- 赶工指令的合法性和书面性
- 实际赶工措施费核算
- 赶工费金额合理性

**法律依据**：
| 条款 | 内容 | 关键要点 |
|------|------|---------|
| GF-2017-0201 第7.8条 | 赶工 | 发包人要求提前竣工的，应承担承包人由此增加的合理费用 |

**调解策略**：
- 确认赶工指令的合法性和书面性
- 核算实际赶工措施费（加班费、设备增加等）
- 参照市场行情评估赶工费合理性

## 三、质量类争议

### 3.1 工程缺陷责任

**适用场景**：工程质量不达标引发的修复/赔偿争议

**保修期标准**（《建设工程质量管理条例》第40条）：

| 工程部位 | 最低保修期 |
|---------|----------|
| 地基基础 | 设计文件规定的工程合理使用年限 |
| 主体结构 | 设计文件规定的工程合理使用年限 |
| 屋面防水 | 5年 |
| 供热供冷系统 | 2个采暖期、供冷期 |
| 给排水管道 | 2年 |
| 电气管线 | 2年 |
| 装修工程 | 2年 |

**归因分析**：

| 缺陷表现 | 可能责任方 |
|---------|----------|
| 结构裂缝 | 设计/施工/材料 |
| 防水渗漏 | 施工工艺/材料质量 |
| 装饰脱落 | 施工/维护使用 |
| 设备故障 | 设备供应商/安装 |

**核心争议点**：
- 质量标准适用
- 缺陷成因（施工/设计/材料/使用不当）
- 修复方案与费用
- 缺陷责任期是否届满

**法律依据**：
- 《建筑法》第60-62条
- 《民法典》第801-802条
- GF-2017-0201 第15条（缺陷责任）

**处理要点**：
- 区分"缺陷"与"正常使用损耗"
- 质量争议通常需要鉴定
- 修复方案需经设计单位确认
- 委托第三方检测机构出具鉴定报告
- 评估修复方案及费用

**调解策略**：
- 委托第三方检测机构出具鉴定报告
- 区分施工缺陷、设计缺陷、材料缺陷
- 评估修复方案及费用

### 3.2 竣工验收争议

**核心争议点**：

**法律依据**：

## 四、造价类争议

### 4.1 清单计价偏差

**适用场景**：工程量清单编制与实际施工的差异引发的价款争议

**常见偏差类型**：
- 工程量清单漏项
- 综合单价组成争议
- 暂估价的调整
- 材料价格涨跌调差

**法律依据**：
- GB/T 50500-2024 第5章：合同价款的约定
- GB/T 50500-2024 第9章：合同价款调整
- GB/T 50500-2024 第11章：竣工结算与支付

**调解策略**：
- 对照清单规范核查工程量计算规则
- 复核综合单价组成要素
- 按合同约定的价格调整方式处理材料价差

---

### 4.2 材料价格调差

**适用场景**：施工期间材料价格波动超出合同约定风险范围

**核心争议点**：
- 合同是否约定调价条款
- 风险范围（通常±5%）
- 调价依据和时点

**法律依据**：
- GB/T 50500-2024 第3.4条（物价变化）
- 《建工司法解释（一）》第19条

---

参见工程款支付争议部分。

## 争议复杂度评估矩阵

| 评估维度 | 简单（1分） | 中等（3分） | 复杂（5分） |
|---------|-----------|-----------|-----------|
| 争议金额 | <100万 | 100-1000万 | >1000万 |
| 争议焦点数 | 1-2个 | 3-5个 | >5个 |
| 当事方数量 | 2方 | 3-4方 | >4方 |
| 证据完整性 | 齐全 | 部分缺失 | 严重缺失 |
| 关系紧张度 | 友好 | 一般 | 对立 |
| 时效紧迫性 | 充裕 | 适中 | 紧迫 |

**总分判断**：
- 6-12分：简单争议，促进式调解，1-2次会议
- 13-20分：中等争议，混合式调解，3-5次会议
- 21-30分：复杂争议，评估式调解，需专家支持

---

## 调解优先级规则

1. **可调解优先**：金钱纠纷 > 工期纠纷 > 质量纠纷 > 合同解除纠纷
2. **时效优先**：诉讼时效剩余 < 6个月时，加速推进
3. **关系维护**：长期合作方优先促进式调解
