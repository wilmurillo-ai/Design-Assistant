# 调解策略与技巧库

## 一、三种调解模式

### 1.1 评估式调解（Evaluative Mediation）

**核心理念**：调解员作为"中立专家"，对争议给出专业评估意见。

**适用场景**：
- 当事人法律认知差异大
- 争议焦点清晰，法律适用明确
- 双方需要权威第三方判断
- 当事人期望了解"如果上法院会怎样"

**主要技法**：

| 技法 | 说明 | 示例 |
|------|------|------|
| **风险提示法** | 客观分析各方的诉讼风险 | "根据类案检索，类似案件中承包人全部索赔被支持的比例约为60%" |
| **成本算账法** | 计算诉讼/仲裁的直接和间接成本 | "如果进入诉讼，仅诉讼费、律师费、鉴定费预计就要XX万元" |
| **案例引导法** | 引用类似案件的裁判结果 | "最高人民法院在XX案中认定..." |
| **权威引用法** | 引用法律法规和司法解释 | "依据《建工司法解释（一）》第XX条..." |

**操作要点**：
- 评估必须客观中立，不可偏袒
- 同时向双方提示各自的风险
- 评估意见仅供参考，不构成法律建议
- 避免过度施压导致当事人感受被胁迫

---

### 1.2 促进式调解（Facilitative Mediation）

**核心理念**：调解员作为"过程引导者"，帮助双方自主沟通和探索。

**适用场景**：
- 长期合作关系
- 争议涉及情感因素
- 双方希望维持友好关系
- 争议需要创造性解决方案

**主要技法**：

| 技法 | 说明 | 示例 |
|------|------|------|
| **积极倾听** | 复述当事人陈述，表达理解 | "我理解您的核心关切是..." |
| **重新框架** | 将对立立场转化为共同问题 | "不是A对B错，而是双方都希望项目顺利完成" |
| **利益探索** | 引导当事人表达深层利益 | "除了金额，您最希望获得什么？" |
| **方案共创** | 引导双方共同设计解决方案 | "有没有一种方案能让双方都能接受？" |
| **利益细分** | 将大问题拆分为可协商的小问题 | "我们可以先解决XX这个小问题" |

**操作要点**：
- 不评价、不判断，保持绝对中立
- 给双方充分表达的机会
- 关注情绪管理
- 适时总结共识点

---

### 1.3 混合式调解（Hybrid Mediation）★推荐

**核心理念**：先促进后评估，兼顾关系维护和法律风险意识。

**三阶段流程**：

```
第一阶段：促进（建立信任、梳理事实）
    ↓
第二阶段：评估（法律分析、风险揭示）
    ↓
第三阶段：磋商（理性基础上的方案谈判）
```

**各阶段详解**：

#### 第一阶段：促进阶段（1-2小时）
- **联席会议**：各方陈述事实和诉求
- **调解员角色**：倾听、记录、确认理解
- **目标**：建立沟通基础、梳理事实全貌
- **输出**：事实清单、各方诉求对比表

#### 第二阶段：评估阶段（背靠背，各1小时）
- **与甲方**：分析乙方主张的合理性、甲方的诉讼风险
- **与乙方**：分析甲方可能的抗辩、乙方的举证风险
- **目标**：让双方理性认识各自的风险和收益
- **输出**：各方风险评估报告

#### 第三阶段：磋商阶段（2-4小时）
- **方案提出**：调解员提出建议方案或引导双方提出
- **方案比较**：多方案对比分析
- **条款细化**：将共识转化为具体条款
- **目标**：达成具体、可执行的协议
- **输出**：调解协议草案

## 二、会议组织技巧

### 2.1 联席会议（Joint Session）

**适用**：事实陈述、方案磋商、协议签署

**座位安排**：
```
调解员
┌─────────────┐
│             │
├──────┬──────┤
│ 甲方 │ 乙方 │
│+律师 │+律师 │
└──────┴──────┘
```

**注意事项**：
- 开场时设定规则（轮流发言、不打断）
- 控制情绪化发言
- 及时总结共识
- 注意体态和语气

### 2.2 背靠背会议（Caucus）

**适用**：利益探索、风险评估、方案试探

**操作要点**：
- 保证另一方的信息不被泄露（除非获得明确授权）
- 传递信息时进行"建设性转述"
- 用于试探底线和创造方案空间
- 每方建议控制在45-60分钟

**信息传递技巧**：

| 甲方实际意思 | 建设性转述 |
|------------|-----------|
| "最多给500万，多一分不给" | "甲方有较大的诚意，愿意积极解决问题，目前在资金安排上存在一定限制" |
| "乙方狮子大开口" | "甲方认为目前的金额差距较大，希望能进一步沟通合理的解决方案" |

## 三、常见困境与对策

### 困境1：双方立场差距过大

**对策**：
1. 拆分争议为多个子问题，逐个解决
2. 引入利益交换（非金钱利益）
3. 使用"逐步靠近法"：先就原则达成一致，再谈具体数字

### 困境2：一方拒绝调解/消极应对

**对策**：
1. 强调调解的保密性（不像诉讼会公开）
2. 分析诉讼/仲裁的实际成本和风险
3. 利用时间压力（诉讼时效即将届满）
4. 通过背靠背了解真实顾虑

### 困境3：当事人情绪激动

**对策**：
1. 充分倾听，表达理解
2. 暂停讨论，给予冷静时间
3. 将讨论转向事实层面
4. 通过背靠背缓解对方面对面压力

### 困境4：代理人权限不足

**对策**：
1. 提前确认各方的决策权限
2. 要求决策者参与关键会议
3. 给代理人带回沟通的方案框架

### 困境5：涉及多方当事人

**对策**：
1. 先解决核心两方争议
2. 对第三方进行单独调解
3. 设计多方共赢方案
4. 必要时进行联合调解会议

## 四、调解话术参考

### 开场话术

> 各位好，我是调解员XXX。今天我们在这里，是为了帮助各方就XX争议寻求一个双方都能接受的解决方案。调解的原则是自愿、平等、保密。请各位放心，今天讨论的所有内容，未经各方同意不会向外透露。调解不成功不影响各方依法行使诉讼权利。

### 事实梳理话术

> 感谢甲方的陈述。让我确认一下我理解的是否正确：甲方认为...是这样吗？乙方对甲方的陈述有什么补充或不同意见？

### 风险提示话术

> 我从法律角度给大家做一个客观分析。根据《XX法》第X条规定，这类争议如果进入诉讼，通常的裁判趋势是...同时需要考虑的是，诉讼周期大约X个月，费用大约X万元。

### 方案磋商话术

> 基于刚才的讨论，我理解双方的核心关切分别是...有一种可能的思路是...不知各位觉得这个方向是否值得进一步探讨？

### 促成协议话术

> 我们今天已经取得了很大的进展，在X个问题上达成了共识。剩余的X个问题，差距其实并不大。如果双方各退一步，我们今天就有可能达成一个完整的解决方案。毕竟相比诉讼的成本和时间，今天达成协议对双方都是更有利的选择。
