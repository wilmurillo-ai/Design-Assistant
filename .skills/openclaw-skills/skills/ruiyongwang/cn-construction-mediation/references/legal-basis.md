# 法律法规速查表

## 一、行政法规

### 《商事调解条例》（国令第827号，2026年5月1日起施行）

> ⭐ **核心法规**：这是我国首部商事调解行政法规，建设工程纠纷商事调解直接适用。

| 条款 | 内容 | 适用场景 |
|------|------|---------|
| 第1条 | 立法目的：规范商事调解活动，有效解决商事争议 | 调解适用性判断 |
| 第2条 | 适用范围含**工程建设**领域商事争议 | 建设工程商事调解的法律基础 |
| 第8条 | 商事调解组织设立条件（30万资产、5名以上调解员） | 调解组织资质审查 |
| 第12条 | 商事调解员任职条件（法律职业资格/律师/专业技术等） | 调解员资格审查 |
| 第14条 | 商事调解原则：**自愿、合法、诚信、保密** | 调解程序基本原则 |
| 第15条 | 当事人自愿申请，一方拒绝不得调解 | 调解启动 |
| 第16条 | 商事调解组织可收取调解费用，标准应公开 | 调解费用 |
| 第17条 | 可适用行业规则、商业惯例、交易习惯 | 行业惯例适用 |
| 第18条 | 鼓励运用AI、大数据等技术，线上调解与线下同等效力 | 在线调解/数字化调解 |
| 第19条 | 调解不公开，调解员有保密义务 | 保密原则 |
| 第20条 | 利益冲突披露与回避制度 | 调解员中立性 |
| 第21条 | 调解终止情形（无法达成协议/当事人不同意/非法目的等） | 调解终止 |
| 第22条 | 调解协议的**制作、签名、盖章**要求；协议不得违法，**具有法律约束力** | 调解协议效力 |
| 第23条 | 商事调解协议可申请**司法确认**（《民事诉讼法》规定） | 司法确认程序 |
| 第24条 | 支持境外设业务机构；自贸区/海南自贸港可引入境外调解组织 | 涉外商事调解 |
| 第28条 | 未经批准擅自开展商事调解的处罚（10-30万罚款） | 合规要求 |
| 第33条 | 施行日期：2026年5月1日 | 时间效力 |
