# 中国建设工程全过程造价管控

## 概述

本技能帮助造价工程师、项目经理实现建设工程全过程造价管控，覆盖**投资决策→设计→招投标→施工→竣工**全生命周期。

**核心能力：**
- 投资估算编制
- 设计概算审查
- 施工图预算审核
- 施工阶段造价控制
- 竣工结算审核
- 动态成本监控

## 第一阶段：投资估算

### 1.1 估算编制依据

| 依据类型 | 具体内容 |
|---------|---------|
| 法律法规 | 《建设项目投资估算编审规程》CECA/GC 1 |
| 计价依据 | 估算指标、概算指标、类似工程资料 |
| 市场信息 | 材料价格、人工成本、设备价格 |
| 技术资料 | 项目建议书、可行性研究报告 |

### 1.2 估算编制方法

**方法一：生产能力指数法**
```
C₂ = C₁ × (Q₂/Q₁)^n × f

其中：
C₂：拟建项目投资
C₁：已建类似项目投资
Q₂：拟建项目规模
Q₁：已建类似项目规模
n：生产能力指数（0.6-0.9）
f：价格调整系数
```

**方法二：设备系数法**
```
C = E × (1 + f₁P₁ + f₂P₂ + ...) + I

其中：
C：项目投资
E：设备购置费
P₁、P₂：各专业工程费用系数
f₁、f₂：价格调整系数
I：工程建设其他费用
```

**方法三：单位面积估算法**
```
C = S × P × f

其中：
C：项目投资
S：建筑面积
P：单位面积造价指标
f：调整系数
```

## 第二阶段：设计概算

### 2.1 概算编制原则

```
设计概算控制原则：

1. 不得突破投资估算
   - 超过投资估算10%以上：重新报批可行性研究
   - 超过投资估算5%-10%：说明原因，调整概算

2. 概算编制深度
   - 初步设计概算：分解到单位工程
   - 施工图概算：分解到分部分项工程

3. 费用组成
   - 建筑安装工程费
   - 设备及工器具购置费
   - 工程建设其他费用
   - 预备费
   - 建设期利息
```

### 2.2 概算审查要点

| 审查项目 | 审查内容 | 常见问题 |
|---------|---------|---------|
| 编制依据 | 定额、取费标准、价格信息 | 使用过期定额、费率错误 |
| 工程量 | 与设计图纸一致性 | 工程量计算错误、漏项 |
| 设备价格 | 市场价格合理性 | 价格虚高、重复计价 |
| 取费标准 | 费率适用性 | 错套费率、重复取费 |

### 2.3 概算对比分析

```markdown
## 概算与估算对比分析表

| 费用项目 | 投资估算 | 设计概算 | 差额 | 差率 | 差异原因 |
|---------|---------:|---------:|-----:|-----:|---------|
| 建筑工程 | 8,500 | 8,780 | +280 | +3.3% | 设计调整 |
| 安装工程 | 2,100 | 2,256 | +156 | +7.4% | 标准提高 |
```

### 3.3 综合单价分析

```
综合单价组成：

综合单价 = 人工费 + 材料费 + 施工机具使用费 + 管理费 + 利润 + 风险费

其中：
人工费 = ∑(工日消耗量 × 人工单价)
材料费 = ∑(材料消耗量 × 材料单价)
机械费 = ∑(机械台班消耗量 × 台班单价)
管理费 = (人工费 + 机械费) × 管理费率
利润 = (人工费 + 机械费) × 利润率
风险费 = 综合单价 × 风险系数
```

### 4.3 材料价格调整公式

**基于 GB/T 50500-2024 的调价公式：**

```
ΔP = P₀ × [A + Σ(Bᵢ × Ftᵢ/F₀ᵢ) - 1]

其中：
ΔP：价格调整差额
P₀：承包人签约合同价中的可调整部分金额
A：定值权重（固定系数）
Bᵢ：第i种可调因子的变值权重
Ftᵢ：第i种可调因子的现行价格指数
F₀ᵢ：第i种可调因子的基准价格指数

可调因子通常包括：
- 人工费
- 材料费（钢材、木材、水泥、混凝土等）
- 施工机具使用费
```

## 第五阶段：竣工结算

### 5.1 结算审核流程

```
竣工结算审核流程：

1. 资料接收与审核
   ├── 审核资料完整性
   ├── 审核资料有效性
   └── 列出缺失资料清单

2. 工程量核对
   ├── 现场踏勘
   ├── 图纸核对
   └── 工程量确认

3. 综合单价确认
   ├── 合同内项目：执行合同单价
   ├── 变更项目：协商确定
   └── 新增项目：重新组价

4. 费用计算
   ├── 分部分项工程费
   ├── 措施项目费
   ├── 其他项目费
   ├── 规费、税金
   └── 造价调整

5. 出具审核报告
   ├── 审核依据
   ├── 审核过程
   ├── 审核结果
   └── 争议处理建议
```

### 5.2 结算审核要点

**工程量审核：**

| 项目 | 审核方法 | 注意事项 |
|------|---------|---------|
| 土方工程 | 现场测量、图纸核对 | 放坡、工作面是否重复 |
| 混凝土工程 | 按图纸计算 | 扣减洞口、构件重叠 |
| 钢筋工程 | 按图纸计算+变更 | 锚固、搭接是否合理 |
| 装饰工程 | 现场测量 | 扣减门窗洞口 |

**单价审核：**

| 项目 | 审核方法 | 注意事项 |
|------|---------|---------|
| 合同内项目 | 执行合同价 | 核对项目特征是否一致 |
| 材料价格 | 信息价+市场询价 | 关注价格波动期 |
| 新增项目 | 重新组价 | 参照类似项目单价 |

### 5.3 结算争议处理

```markdown
常见结算争议及处理方法：

1. 工程量争议
   - 协商：双方现场核对
   - 鉴定：委托造价咨询机构
   - 测量：第三方测量机构

2. 单价争议
   - 合同解释：按合同约定解释顺序
   - 市场询价：参考同期市场价格
   - 协商确定：双方认可的价格

3. 变更签证争议
   - 程序审查：是否符合签证程序
   - 实质性审查：是否实际发生
   - 金额认定：工程量×单价

4. 材料价格调整争议
   - 按合同约定的调价公式
   - 参考造价信息价格
   - 协商确定调整幅度
```
