# 东方智慧调解典故集

## 一、经典典故与现代应用

### 1. 六尺巷

**历史背景**：清朝康熙年间，大学士张英的家人因与邻居争地，写信求助。张英回信曰："千里修书只为墙，让他三尺又何妨？万里长城今犹在，不见当年秦始皇。"家人主动让出三尺，邻居见状也让出三尺，形成六尺巷。

**原文**：
```
千里修书只为墙，
让他三尺又何妨？
万里长城今犹在，
不见当年秦始皇。
```

**工程调解应用**：

| 场景 | 传统思维 | 六尺巷思维 |
|------|---------|-----------|
| 工程款争议 | 争到最后一分钱 | 各让一步，快速解决 |
| 工期延误 | 追讨全部违约金 | 合理分担，共渡难关 |
| 质量问题 | 拒付全部款项 | 修复为主，适度扣款 |
| 边界纠纷 | 寸土必争 | 退让三尺，赢得人心 |

**调解话术**：
> "您看这'六尺巷'的故事，张家让了三尺，吴家也让了三尺，最后皆大欢喜。咱们今天争的这点利益，比起邻里和睦、项目顺利，哪个更重要？"

### 2. 负荆请罪

**历史背景**：战国时期，蔺相如因"完璧归赵"被封为上卿，位在廉颇之上。廉颇不服，欲羞辱蔺相如。蔺相如以国家为重，处处忍让。后廉颇得知，亲自负荆条向蔺相如请罪。

**原文**：
> "顾吾念之，强秦之所以不敢加兵于赵者，徒以吾两人在也。今两虎共斗，其势不俱生。"

**工程调解应用**：

| 要素 | 典故内容 | 调解应用 |
|------|---------|---------|
| 主动方 | 廉颇负荆请罪 | 责任方主动认错 |
| 包容方 | 蔺相如宽容大度 | 受损害方接受道歉 |
| 结果 | 将相和，国家兴 | 双方和解，合作继续 |

**适用场景**：
- 一方明显违约，但愿意承担责任
- 双方关系对项目继续至关重要
- 过错方有诚意修复关系

---

### 3. 将相和

**历史背景**：蔺相如与廉颇的故事，体现"以国为重"的大局意识。

**核心精神**：
```
个人恩怨 < 集体利益
争强好胜 < 和谐共处
面子得失 < 事业成败
```

**调解话术**：
> "您看蔺相如和廉颇，一个是文臣，一个是武将，本来水火不容。但蔺相如说了：'强秦之所以不敢加兵于赵者，徒以吾两人在也。'咱们今天也一样，项目能不能做好，就看咱们能不能和衷共济。"

---

### 4. 退避三舍

**历史背景**：春秋时期，晋文公重耳曾流亡楚国，楚成王善待之。晋文公许诺：若两国交战，晋军将退避三舍（九十里）以报楚王之恩。后城濮之战，晋军果然退避三舍，后发制人，大败楚军。

**原文**：
> "晋文公避君三舍，义也。"

**工程调解应用**：

| 策略 | 典故智慧 | 现代应用 |
|------|---------|---------|
| 主动退让 | 退避三舍以报恩 | 主动让步换取信任 |
| 后发制人 | 退中求进 | 暂时让步赢得长远合作 |
| 信守承诺 | 说了就要做到 | 调解达成的协议必须履行 |

---

### 5. 三顾茅庐

**历史背景**：刘备三次拜访诸葛亮，终于请出卧龙先生辅佐汉室。

**核心精神**：
```
诚心诚意   —— 态度决定一切
坚持不懈   —— 好事多磨
礼贤下士   —— 尊重是和解的前提
```

**调解应用**：
- 调解前的充分准备
- 对当事人的尊重
- 用诚意打动对方

---

### 6. 田忌赛马

**历史背景**：孙膑为田忌出谋划策，用下等马对上等马、上等马对中等马、中等马对下等马的策略，以弱胜强。

**核心智慧**：
```
优劣是相对的
策略比蛮力更重要
换个思路，天地宽
```

**调解方案设计应用**：

| 传统方案 | 田忌赛马式方案 |
|---------|---------------|
| 零和博弈 | 寻求增量 |
| 你死我活 | 各取所需 |
| 硬碰硬 | 错位竞争 |
| 一次性解决 | 分步推进 |

### 7. 愚公移山

**历史背景**：愚公决心移走挡在家门口的太行、王屋二山，虽年迈体弱，但意志坚定，感动天帝，派神仙移走二山。

**核心精神**：
```
坚持不懈
团结合作
感动人心
终成正果
```

**长期调解应用**：
- 复杂纠纷需要耐心
- 各方需要团结协作
- 时间换空间

---

### 8. 塞翁失马

**历史背景**：边塞老翁的马跑到了胡地，人皆吊之；后来马带回了胡地良马，人皆贺之；其子骑马摔断腿，人又吊之；后来战争征兵，其子因瘸腿免于征兵而存活。

**原文**：
> "祸兮福所倚，福兮祸所伏。"

**调解心态引导**：
```
眼前的损失，可能是未来的收益
暂时的让步，可能是长远的获益
争到的不一定是好的
失去的可能是包袱
```

## 二、历史上的著名调解案例

### 1. 周处改过

**背景**：晋朝周处年轻时横行乡里，与南山虎、水中蛟并称"三害"。后经陆机、陆云教导，改过自新，成为有用之人。

**调解智慧**：
```
发现优点 → 给予希望
指出方向 → 指明出路
持续引导 → 陪伴成长
```

**应用**：引导过错方改过自新，而非一棒子打死。

---

### 2. 萧何追韩信

**背景**：韩信不被项羽重用，投奔刘邦后也不被重视。萧何月下追韩信，并向刘邦力荐，最终成就大汉江山。

**调解智慧**：
```
识人之才 → 发现各方优点
敢于推荐 → 为各方说公道话
创造机会 → 提供展示平台
```

**应用**：为弱势方说话，帮助其展示真实价值。

---

### 3. 张良拾履

**背景**：张良在桥下为老人捡鞋，老人考验其诚意后，传授《太公兵法》。

**调解智慧**：
```
考验诚意 → 确认当事人态度
传道授业 → 提供专业指导
成就大事 → 引导正确方向
```

## 三、古代调解制度

### 1. 息讼制度

**历史背景**：中国古代地方官常以"息讼"为政绩，劝导百姓不要轻易兴讼。

**息讼理念**：
```
讼则终凶 —— 诉讼代价高昂
和为贵   —— 和解利益更大
礼让先   —— 退让是美德
```

**现代转化**：
- 强调调解的成本优势
- 说明诉讼的时间成本
- 展示和解的关系价值

---

### 2. 乡贤调解

**历史背景**：宗法社会中，乡绅、族长主持调解纠纷，维护地方秩序。

**乡贤特质**：
```
公正无私 —— 道德威望高
熟悉人情 —— 了解当事人背景
德高望重 —— 双方都信服
```

**现代转化**：
- 引入行业专家参与调解
- 借助权威人士的影响力
- 发挥"老娘舅"的作用

---

### 3. 牌坊表彰

**历史背景**：古代通过立牌坊表彰"义夫节妇"、"孝子贤孙"，引导社会风气。

**表彰智慧**：
```
正面引导 → 树立榜样
道德激励 → 内化自律
社会监督 → 形成风气
```

**现代转化**：
- 对和解成功的案例进行表彰
- 建立调解员荣誉制度
- 形成"和为贵"的社会风尚
