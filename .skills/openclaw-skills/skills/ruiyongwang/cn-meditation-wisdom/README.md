## 核心功能

### 一、争议焦点识别

**传统方法**：基于合同条款和法律规定
**东方智慧**：增加"情"和"理"的分析

```
争议分析 = 法律分析 + 利益分析 + 情感分析
```

### 二、利益分析

**BATNA/WATNA分析（东方智慧版）**

```
发包人 BATNA：诉讼（耗时耗力，可能影响声誉）
承包人 BATNA：诉讼（胜诉概率大，但执行难）

东方式解读：
"对簿公堂，两败俱伤"
"打一场官司，结一个仇人"
"赢了官司，输了人心"
```

### 三、方案设计

**田忌赛马式方案设计**

| 传统方案 | 东方智慧方案 |
|---------|-------------|
| 一次性解决 | 分期推进 |
| 零和博弈 | 寻求增量 |
| 你死我活 | 各取所需 |

### 四、公平性评估

**五维公平性评估**

| 维度 | 权重 | 评估要点 |
|:----:|:----:|---------|
| 程序公平 | 25% | 中立、透明、自愿 |

| 情感公平 | 20% | 被尊重、被理解 |
| 关系公平 | 15% | 合作延续、信任修复 |
| 执行公平 | 10% | 可行、明确、可监督 |

## 使用方法

### 方式一：直接咨询

```
用户：有个工程款纠纷，调解时应该注意什么？
助手：运用本技能分析，提供包含东方智慧的分析和建议
```

```
用户：发包人欠工程款500万，承包人要起诉
助手：
1. 进行利益分析（BATNA/WATNA）
2. 寻找ZOPA（和解空间）

```

