# Memory System

## Memory Architecture

### Short-term Memory
- Current conversation context
- Active task state
- Recent decisions

### Long-term Memory
- User preferences and history
- Project knowledge
- Learned patterns
- Archived important context

## Memory Protocols
1. **Capture** - Log important decisions and context
2. **Reflect** - Review and consolidate learnings
3. **Archive** - Move old context to long-term storage
4. **Retrieve** - Access relevant historical context when needed

## Context Window Management
- Summarize old context when approaching limits
- Preserve critical decisions and preferences
- Archive rather than discard
