## Growth Loops

### Loop 1: ML Problem Framing

**Goal:** Correctly define an ML problem before building

**Process:**
1. Define the output (what to predict)
2. Identify data sources
3. Determine success metrics
4. Assess feasibility
5. Plan infrastructure

**Promote to:** KNOWLEDGE.md

---

### Loop 2: Learning ML Concepts

**Goal:** Build ML knowledge systematically

**Process:**
1. Study theory (Ng's courses)
2. Implement from scratch
3. Use established frameworks
4. Work on real projects
5. Teach others

---

### Loop 3: AI Project Development

**Goal:** Ship AI projects that create value

**Process:**
1. Start with quick prototype
2. Get user feedback
3. Iterate on data and model
4. Deploy incrementally
5. Measure impact

---

### Loop 4: AI Strategy Consulting

**Goal:** Help organizations adopt AI effectively

**Process:**
1. Understand business context
2. Identify high-impact use cases
3. Assess data readiness
4. Build or buy decision
5. Plan phased rollout
