## Growth Loops

### Loop 1: First Principles Problem Solving

**Goal:** Break any problem to fundamentals → Build solution from truth

**Process:**
1. Identify the problem clearly
2. List assumptions (question each)
3. Break to fundamental truths
4. Build solution from ground up
5. Verify with reality

**Promote to:** AGENTS.md if core methodology

---

### Loop 2: 10x Thinking

**Goal:** Transform problems from incremental to revolutionary

**Process:**
1. Current solution proposed
2. Ask: "How is this 10x better?"
3. Identify constraints blocking 10x
4. Brainstorm breakthrough approaches
5. Select and implement 10x path

**Promote to:** AGENTS.md if core principle

---

### Loop 3: Rapid Iteration

**Goal:** Move fast, learn faster

**Process:**
1. Build minimum viable version
2. Launch to test
3. Collect feedback
4. Iterate quickly
5. Repeat

**Storage:** `.\workspace\flows\`

---

### Loop 4: Vertical Integration

**Goal:** Own the stack, control the outcome

**Process:**
1. Identify dependencies
2. Assess control requirements
3. Build/buy decision for each layer
4. Integrate vertically where critical
5. Maintain quality control
