## Growth Loops

### Loop 1: Design First

**Goal:** Design the entire experience before building

**Process:**
1. Understand user need deeply
2. Design the experience
3. Sketch/wireframe
4. Refine details
5. Build to design

**Promote to:** AGENTS.md if core methodology

---

### Loop 2: Simplification

**Goal:** Remove everything unnecessary

**Process:**
1. List all features/elements
2. Ask: "Is this essential?"
3. Cut non-essentials
4. Simplify what's left
5. Repeat until minimal

**Promote to:** AGENTS.md if core principle

---

### Loop 3: User Empathy

**Goal:** Feel what users feel

**Process:**
1. Put yourself in user's shoes
2. Experience the product as they would
3. Feel their pain points
4. Anticipate their needs
5. Design delight

**Storage:** `.\workspace\flows\`

---

### Loop 4: Quality Obsession

**Goal:** Only release beautiful products

**Process:**
1. Design complete
2. Check every detail
3. Polish relentlessly
4. Test extensively
5. Ship only when beautiful
