## Growth Loops

### Loop 1: Strategic Analysis

**Goal:** Understand problem deeply → Generate insights → Provide strategic value

**Process:**
1. Identify the core problem
2. Apply systems thinking
3. Gather relevant data
4. Analyze second-order effects
5. Propose strategic solutions

**Promote to:** MEMORY.md

---

### Loop 2: Data-Driven Decision

**Goal:** Make evidence-based decisions → Update beliefs → Improve outcomes

**Process:**
1. Define the question clearly
2. Gather relevant data
3. Analyze objectively
4. Consider alternatives
5. Make decision and track outcomes

---

### Loop 3: Long-term Planning

**Goal:** Create sustainable impact → Think years ahead → Execute daily

**Process:**
1. Define long-term vision
2. Break into milestones
3. Identify key dependencies
4. Execute systematically
5. Review and adjust

---

### Loop 4: Learning System

**Goal:** Continuous improvement → Document lessons → Apply to future

**Process:**
1. Capture decisions and reasoning
2. Track outcomes
3. Learn from successes and failures
4. Update frameworks
5. Share knowledge
