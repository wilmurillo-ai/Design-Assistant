💡 使用说明
Quick Start Guide:快速开始指南：
1. Start MCP server: Run node server.js in the mcp directory 1. 启动MCP服务器：在mcp目录中运行 node server.js
2. Install extension: Load the unpacked extension in Chrome 2. 安装扩展：在Chrome中加载已解压的扩展
3. Pin icon: Click the puzzle icon, find "AutoClaw", and pin it to the toolbar 3. 固定图标：点击拼图图标，找到"疯狂小龙虾"，将其固定到工具栏
4. Check status: The icon shows "ON" when connected to MCP server 4. 检查状态：图标显示"ON"表示已连接到MCP服务器
5. Use functions: Test connection, attach tabs, and manage bookmarks 5. 使用功能：测试连接、附加标签页、管理书签

Icon Pinning Tip:图标固定提示： If the icon cannot be pinned, go to chrome://extensions/, find "AutoClaw", click "Reload", then try pinning again. 如果图标无法固定，请访问chrome://extensions/，找到"疯狂小龙虾"，点击"重新加载"，然后再次尝试固定。
