
# 📋 A股市场事件分类体系

## 一、政策事件类（POLICY）

### 1.1 宏观经济政策
| 事件类型 | 说明 | 典型影响 |
|---------|------|---------|
| 降准 | 央行降低存款准备金率 | 🟢 利好：银行、地产、周期股 |
| 降息 | 央行降低基准利率 | 🟢 利好：整体市场，特别是高负债行业 |
| 汇率调整 | 人民币升值/贬值 | 🟡 双向影响：进口/出口行业分化 |
| 财政政策 | 财政支出、税收调整 | 🟢 利好：相关受益行业 |

### 1.2 行业监管政策
| 事件类型 | 说明 | 典型影响 |
|---------|------|---------|
| 行业新规 | 新的行业监管规定 | 🔴 或 🟢 取决于具体内容 |
| 准入限制 | 行业准入门槛变化 | 🟢 利好：现有龙头企业 |
| 环保政策 | 环保要求提高 | 🔴 利空：高污染行业 |
| 价格管制 | 产品价格管制 | 🔴 或 🟢 取决于管制方向 |

### 1.3 产业扶持政策
| 事件类型 | 说明 | 典型影响 |
|---------|------|---------|
| 补贴政策 | 财政补贴支持 | 🟢 利好：受补贴行业 |
| 税收优惠 | 税收减免 | 🟢 利好：相关企业 |
| 产业规划 | 五年规划、产业政策 | 🟢 利好：规划重点产业 |
| 政府采购 | 政府购买支持 | 🟢 利好：中标企业 |

## 二、公司事件类（COMPANY）

### 2.1 业绩公告
| 事件类型 | 说明 | 典型影响 |
|---------|------|---------|
| 业绩超预期 | 实际业绩好于预期 | 🟢 利好：股价通常上涨 |
| 业绩低于预期 | 实际业绩差于预期 | 🔴 利空：股价通常下跌 |
| 业绩预告 | 提前发布业绩预告 | 🟢 或 🔴 取决于预告内容 |
| 业绩快报 | 初步业绩数据 | 🟢 或 🔴 取决于数据 |

### 2.2 资本运作
| 事件类型 | 说明 | 典型影响 |
|---------|------|---------|
| 并购重组 | 收购其他公司或资产 | 🟢 通常利好（视具体情况） |
| 资产注入 | 大股东注入资产 | 🟢 通常利好 |
| 分拆上市 | 子公司独立上市 | 🟢 通常利好 |
| 回购股份 | 公司回购自家股票 | 🟢 利好：显示信心 |
| 减持股份 | 股东减持 | 🔴 利空：供应增加 |

### 2.3 股权激励
| 事件类型 | 说明 | 典型影响 |
|---------|------|---------|
| 股票期权 | 授予期权 | 🟢 利好：绑定管理层利益 |
| 限制性股票 | 授予限制性股票 | 🟢 利好：长期激励 |
| 员工持股计划 | 员工持股 | 🟢 利好：全员激励 |

### 2.4 重大合同
| 事件类型 | 说明 | 典型影响 |
|---------|------|---------|
| 大额订单 | 签订重大销售合同 | 🟢 利好：业绩保障 |
| 战略合作 | 签订战略合作协议 | 🟢 利好：长期合作 |
| 项目中标 | 重大项目中标 | 🟢 利好：新业务拓展 |

### 2.5 分红送转
| 事件类型 | 说明 | 典型影响 |
|---------|------|---------|
| 现金分红 | 派发现金股利 | 🟢 中性偏利好 |
| 送股 | 股票股利 | 🟡 中性：数字游戏 |
| 转增股本 | 公积金转增 | 🟡 中性：数字游戏 |
| 高送转 | 10送转10以上 | 🟢 通常有炒作行情 |

## 三、行业事件类（INDUSTRY）

### 3.1 供需变化
| 事件类型 | 说明 | 典型影响 |
|---------|------|---------|
| 原材料涨价 | 原材料价格上涨 | 🔴 利空：下游行业 |
| 产品提价 | 产品价格上涨 | 🟢 利好：相关企业 |
| 库存变化 | 行业库存增减 | 🟢 或 🔴 取决于方向 |
| 需求爆发 | 需求大幅增长 | 🟢 利好：全行业 |

### 3.2 技术突破
| 事件类型 | 说明 | 典型影响 |
|---------|------|---------|
| 新技术应用 | 新技术商业化 | 🟢 利好：技术领先企业 |
| 新产品发布 | 重要新产品上市 | 🟢 利好：发布企业 |
| 行业标准 | 新标准出台 | 🟢 利好：符合标准的企业 |

## 四、舆情事件类（SENTIMENT）

### 4.1 媒体报道
| 事件类型 | 说明 | 典型影响 |
|---------|------|---------|
| 正面报道 | 主流媒体正面报道 | 🟢 利好：提升市场信心 |
| 负面报道 | 媒体曝光负面新闻 | 🔴 利空：可能引发抛售 |
| 澄清公告 | 公司澄清不实报道 | 🟡 取决于澄清效果 |

### 4.2 分析师观点
| 事件类型 | 说明 | 典型影响 |
|---------|------|---------|
| 上调评级 | 券商上调投资评级 | 🟢 利好：通常有正面反应 |
| 下调评级 | 券商下调投资评级 | 🔴 利空：可能引发调整 |
| 目标价调整 | 调整目标价 | 🟢 或 🔴 取决于调整方向 |

## 🎯 使用说明

1. **快速定位**：根据事件类型在本表中快速查找分类
2. **影响预判**：参考"典型影响"栏初步判断利好/利空
3. **历史对照**：结合历史类似事件的市场反应
4. **综合判断**：结合公司基本面、技术面综合分析
