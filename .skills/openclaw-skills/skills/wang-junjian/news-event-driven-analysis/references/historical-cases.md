## 二、公司事件案例

### 2.1 贵州茅台提价事件
| 项目 | 详情 |
|------|------|
| **事件时间** | 多次提价（2017、2018、2021等） |
| **事件内容** | 贵州茅台提高出厂价和零售价 |
| **市场反应** | 提价后股价通常上涨10%-30% |
| **持续时间** | 提价行情通常持续1-3个月 |
| **对标公司** | 五粮液、泸州老窖等高端白酒 |

**关键启示**：
- 具有定价权的公司提价是重大利好
- 高端白酒品牌力强，提价能力突出
- 提价往往开启新一轮业绩增长周期

### 2.2 中国中车合并案例
| 项目 | 详情 |
|------|------|
| **事件时间** | 2014年12月-2015年6月 |
| **事件内容** | 中国南车与中国北车合并为中国中车 |

## 五、事件投资策略总结

### 5.1 如何把握事件驱动机会
1. **快速反应**：事件驱动强调时效性
2. **深入分析**：区分实质性影响和情绪性影响
3. **精选标的**：选择真正受益的行业龙头
4. **控制仓位**：单笔仓位不超过总资金的10%
5. **严格止损**：设置止损位，控制风险

### 5.2 需要规避的陷阱
1. **利好出尽**：消息兑现后往往是行情终点
2. **虚假利好**：警惕概念炒作，关注实质影响
3. **过度追高**：避免在涨幅过大后追高
4. **单票重仓**：事件驱动不确定性大，避免单票重仓
5. **长期持有**：事件驱动通常是中短期行情，不适合长期持有

### 5.3 成功的关键要素
- ✅ 对政策和行业有深入理解
- ✅ 快速反应和果断决策
- ✅ 严格的风险控制
- ✅ 区分短期情绪和长期逻辑
- ✅ 从历史案例中学习经验教训
