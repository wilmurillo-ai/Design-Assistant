
# 🚀 消息面与事件驱动分析 - 快速开始

## 30秒上手

### ⚠️ 第一步：先搜索最新新闻！
**无论分析什么，必须先搜索相关新闻！**

### 最简单的使用方式
只需要说：
```
分析【事件】：【事件描述】
```

### 示例
```
分析政策事件：央行宣布降准0.5个百分点
分析公司公告：贵州茅台发布年报，净利润增长20%
分析行业事件：新能源汽车补贴退坡30%
分析【股票名称】股票消息面与事件驱动分析
```

### 新闻搜索关键词
- A股：`[公司名称] [股票代码] 最新消息 202X年X月`
- 港股：`[公司名称] [股票代码] 最新动态 202X年X月`
- 美股：`[公司名称] [股票代码] news March 202X`

### 搜索后处理
- ✅ 搜索成功：基于新闻内容分析
- ⚠️ 搜索失败：说明情况，基于历史框架分析，并标注"不包含实时新闻"

## 一、常见事件类型速查

### 🎯 政策事件（最常见）
| 关键词 | 事件类型 | 典型影响 |
|--------|---------|---------|
| 降准、降息 | 货币政策 | 🟢 利好整体市场 |
| 补贴、税收优惠 | 产业政策 | 🟢 利好相关行业 |
| 新规、监管 | 行业政策 | 🟢 或 🔴 取决于内容 |

### 🏢 公司事件
| 关键词 | 事件类型 | 典型影响 |
|--------|---------|---------|
| 业绩、年报、季报 | 业绩公告 | 🟢 或 🔴 取决于数据 |
| 并购、重组、收购 | 资本运作 | 🟢 通常利好 |
| 回购、增持 | 股份变动 | 🟢 利好 |
| 减持、限售 | 股份变动 | 🔴 利空 |

### 🏭 行业事件
| 关键词 | 事件类型 | 典型影响 |
|--------|---------|---------|
| 涨价、提价 | 价格变动 | 🟢 利好 |
| 缺货、供不应求 | 供需变化 | 🟢 利好 |
| 新技术、新产品 | 技术突破 | 🟢 利好 |

### 📰 舆情事件
| 关键词 | 事件类型 | 典型影响 |
|--------|---------|---------|
| 媒体报道、曝光 | 媒体舆情 | 🟢 或 🔴 取决于内容 |

## 二、5步事件分析法

### 第1步：判断性质
```
这是利好？利空？还是中性？
```

### 第2步：评估影响
```
影响有多大？短期还是长期？
```

### 第3步：寻找标的
```
哪些公司最受益？哪些受损？
```

### 第4步：历史对照
```
历史上类似事件怎么走的？
```

### 第5步：操作建议
```
买不买？买多少？什么时候买？
```

## 五、注意事项

### ✅ 应该这样做
1. **快速响应**：事件驱动强调时效性
2. **深入分析**：不只是看表面，要分析实质影响
3. **分散投资**：事件驱动不确定性大，不要单票重仓
4. **严格止损**：设置止损位，控制风险
5. **及时止盈**：事件驱动通常是中短期行情

### ❌ 不要这样做
1. **盲目追高**：涨幅过大后不要追高
2. **单票重仓**：不要把所有钱都押在一只股票上
3. **长期持有**：事件驱动不适合长期持有
4. **忽略风险**：只看利好，不看风险
5. **频繁交易**：不要每个事件都参与

## 六、进阶技巧

### 技巧1：如何判断"预期差"
- 市场预期是什么？
- 实际情况比预期好还是差？
- 预期差越大，行情越大

### 技巧2：如何把握"时间窗口"
- 事件发生前：提前布局（如果能预判）
- 事件发生时：快速反应
- 事件发生后：警惕"利好出尽"

### 技巧3：如何选择"弹性标的"
- 小市值 vs 大市值：小市值弹性大
- 纯受益 vs 概念受益：纯受益更可靠
- 龙头 vs 跟风：龙头更稳健，跟风弹性大
