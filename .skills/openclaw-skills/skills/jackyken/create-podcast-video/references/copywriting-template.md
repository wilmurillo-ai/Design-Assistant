# 爆款文案模板参考

## 结构框架（5段式）

### 1. 开头金句（1句话定调）
- 直接给核心结论，一句话抓住注意力
- 例："记住一句话：改变潜意识，就是改变命运！"

### 2. 钩子（制造紧迫感）
- "千万别划走"、"逆天改命"、"有缘者"
- 强调内容稀缺性和重要性
- 预告干货："我会分享X大核心法则，尤其最后一个..."

### 3. 痛点共鸣（连续提问）
- 用"你是否..."连续提问，让用户代入
- 引出问题的根源所在
- 例："为什么总是陷入相同困境？为什么明知不好却难以改变？"

### 4. 核心内容（数字法则+比喻）
- 用"三大法则"或"五个方法"等数字框架组织内容
- 用具象比喻解释抽象概念（魔法师、大树、根系/树干/枝叶）
- 层层递进，最后一条最有力

### 5. 收尾升华（排比+情绪高潮）
- 三连排比句推高情绪："如钢铁般坚不可摧、如火山般喷薄而出、如泉水般源源不断"
- 回扣开头金句，首尾呼应
- 赋能结尾："改变的力量就在你手中"

## 写作技巧

1. **悬念前置** — 开头就告诉用户"尤其最后一个"，留住注意力
2. **比喻具象化** — 抽象概念一定要找到生活中的对应物
3. **排比三连** — 情绪推高时用三个以上排比句
4. **数字法则** — 核心内容用3-5个要点，清晰有力
5. **痛点→方案** — 先戳痛再给解药，不要直接给答案
6. **重复核心句** — 关键金句在开头和结尾重复出现
