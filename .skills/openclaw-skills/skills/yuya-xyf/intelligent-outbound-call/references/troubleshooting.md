# 常见问题排查

## 1. 租户没有绑定号码 ⚠️ 最常见

**错误**：

```
❌ 租户下无绑定号码，无法进行外呼。
请先在阿里云晓蜜控制台申请并绑定外呼号码。
```

**解决**：Load references/phone-number-setup.md，根据用户情况引导选择合适的号码申请方式。

---

## 2. 环境变量未配置

**错误**：`请配置阿里云 AK/SK 环境变量`

**解决**：

```bash
export ALIBABA_CLOUD_ACCESS_KEY_ID="your-key"
export ALIBABA_CLOUD_ACCESS_KEY_SECRET="your-secret"
```

---

## 3. 电话号码格式错误

**错误**：`电话号码格式不正确`

**解决**：确保号码是中国大陆手机号（1开头的11位数字）

---

## 4. 任务创建失败

**错误**：`创建任务组失败`

**可能原因**：实例状态异常、话术脚本未发布、账号权限不足、并发数超限

**解决**：检查阿里云晓蜜控制台的实例和脚本状态

---

## 绑定号码检查机制

- ✅ 租户有绑定号码 → 自动绑定到实例，继续执行
- ❌ 租户没有绑定号码 → 终止流程，提示用户配置
