# 外呼号码申请指南

> 来源：[阿里云智能外呼机器人 - 线路申请流程](https://help.aliyun.com/zh/outboundbot/product-overview/apply-for-a-communication-line)

外呼号码有两种接入方式，根据用户情况引导选择：

---

## 方式一：合作伙伴号码（推荐新用户）

适合没有自有号码的用户，直接在控制台申请。

**步骤：**

1. 进入 [阿里云晓蜜控制台](https://outboundbot.console.aliyun.com/)
2. 选择 **号码申请** > **合作伙伴线路申请**
3. 按需选择号码类型（固话 / 手机号）和线路供应方
   - 号码价格、业务限制可在供应方的"购买须知"中查看
4. 点击"申请号线"，填写号码信息和公司信息
   - 联系信息建议填写**钉钉号**，审核通过后运营人员会通过钉钉联系完成后续对接
5. 点击"申请"提交

**审核时间**：一般 **3 个工作日**

---

## 方式二：自带线路接入

适合已有自有号码、希望接入外呼平台的用户。

**前提条件**：号码必须支持 SIP 协议对接（可联系号码运营商确认）

**步骤：**

1. 确认己方号码支持 SIP 中继协议
2. 让运营商建立自建 trunk 进行对接
3. 添加钉钉群（群号：**29130028336**），联系群内管理员
4. 提供以下 SIP 信息给管理员：
   - 用于 SIP 对接的 IP、媒体 IP、端口
   - 主叫号码（格式：`sip:号码@ip:端口`）

---

## 引导话术

当用户遇到"租户下无绑定号码"错误时，询问用户：

- "您是否已有自有号码（如公司座机或已购买的手机号）？"
  - 有 → 引导走**方式二**（自带线路）
  - 没有 → 引导走**方式一**（合作伙伴号码申请）
