# Autohome - 汽车资讯获取技能

本技能用于获取汽车之家（autohome.com.cn）的最新汽车资讯，并提供格式化输出。

## 功能概述

1. **获取最新汽车资讯** - 从汽车之家新闻页面获取最新的10条资讯
2. **格式化输出** - 按照标准格式输出资讯，包含标题、链接、摘要
3. **定时任务支持** - 可配置为每日定时执行
4. **询价入口** - 每条资讯包含"立即询价"链接

## 使用场景

当用户需要：
- 获取最新的汽车行业新闻
- 了解新车发布信息
- 跟踪汽车技术发展动态
- 配置每日汽车资讯推送

```
获取汽车之家最新10条资讯
汽车资讯

```

### 2. 配置定时任务
创建每日定时任务（每天早上9点执行）：
```bash
openclaw cron add --name "daily-car-news" --cron "0 9 * * *" --message "获取汽车之家最新10条资讯" --to "user:用户ID" --channel feishu --announce --description "每天上午9点获取汽车之家最新资讯"
```

## 格式说明

1. **中文序号** - 使用一、二、三...十作为序号
2. **标题加粗** - 每条资讯的标题使用加粗格式
3. **链接格式** - 标题后跟 **[链接](文章链接)** 格式
4. **项目符号** - 使用 `- ` 开头的项目符号列出详细信息
5. **清晰分类** - 信息按时间、价格、亮点等分类列出

## 高级配置

### 自定义时间
修改cron表达式调整执行时间：
- 每天中午12点：`0 12 * * *`
- 每天下午6点：`0 18 * * *`
- 工作日早上9点：`0 9 * * 1-5`

### 修改接收人
更新定时任务的接收目标：
```bash
openclaw cron edit <任务ID> --to "user:新用户ID"
```

### 禁用/启用任务
```bash
openclaw cron disable daily-car-news
openclaw cron enable daily-car-news
```

## 技术实现

### 数据来源
- 汽车之家新闻页面：https://www.autohome.com.cn/news/
- 使用web_fetch工具获取页面内容
- 解析最新10条资讯的标题、链接和摘要

### 依赖工具
- `web_fetch` - 获取网页内容
- `exec` - 执行系统命令
- `openclaw cron` - 管理定时任务

## 故障排除

### 常见问题
1. **资讯获取失败** - 检查网络连接和汽车之家可访问性
2. **格式问题** - 确保按照标准格式输出
3. **定时任务不执行** - 检查cron服务状态：`openclaw cron status`

### 调试命令
```bash
# 查看所有定时任务
openclaw cron list

# 查看任务状态
openclaw cron status

# 手动运行任务测试
openclaw cron run daily-car-news
```
