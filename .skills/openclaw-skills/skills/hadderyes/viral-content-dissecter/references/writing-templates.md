# 长文写作模板

> 供 SKILL.md Step 3 调用。按需加载。

## 公众号长文模板（1200-1500字）

```
## [小标题1：引入/场景/痛点]

[用读者能感同身受的场景或问题开头，200-300字]

## [小标题2：核心观点/分析]

[展开分析，用数据/案例/逻辑支撑，400-500字]
- 分论点1
- 分论点2
- 分论点3

## [小标题3：深度/延伸/反常识]

[提供独特的视角或更深层的洞察，300-400字]

## [小标题4：总结/行动指南]

[给出可执行的建议，100-200字]

---
[互动引导：你怎么看？评论区聊聊]
```

## 小红书长文模板（800-1200字）

```
[emoji] [一句话吸引注意力的开头]

[emoji] 为什么说这个？/ 发生了什么？
[100-200字背景/故事引入]

[emoji] 重点来了👇
[要点1 - 加粗核心信息]
[要点2 - 加粗核心信息]
[要点3 - 加粗核心信息]
[300-500字干货内容]

[emoji] 我的经验总结
[200-300字个人观点/建议]

[emoji] 最后说两句
[100-150字收尾+互动引导]
```

## SEO标题生成规则

### 公众号标题
- 15-30字
- 必须包含核心关键词（用户可能搜索的词）
- 结构：关键词+价值点+情绪词
- 示例模板：
  - "[关键词]：[数字]个[结果]，第[N]个[情绪]"
  - "为什么[关键词]？[反常识结论]"
  - "[权威/经验]+[关键词]+[量化成果]"

### 小红书标题
- 15-25字
- 必须有1-2个emoji
- 必须包含搜索词
- 结构：情绪词+场景+结果
- 示例模板：
  - "终于！[结果]，[场景]也能做到"
  - "[感叹词][结果]，后悔没早知道"
  - "亲测[结果]，[对比/数据]"

## 配图建议规范

每条配图建议必须包含以下要素：
1. **位置**: 封面/文中/结尾
2. **类型**: 实拍图/插画/数据图/截图/金句图
3. **内容**: 具体画面描述
4. **风格**: 色调、构图、字体建议
5. **文案**: 图上需要加的文字（如果有）

示例：
> 封面图 | 插画风格 | 一个人在书桌前被各种手机APP包围，表情焦虑，暖色调 | 图中加粗体白字"你被算法绑架了吗？"
