# 爆款拆解框架

> 供 SKILL.md Step 2 调用。按需加载。

## 六维拆解模型

### 1. 选题定位
评估选题是否命中以下要素（至少命中2个即为好选题）：
- **痛点**: 解决了一个真实存在的困扰
- **热点**: 与当前热门话题/事件关联
- **情绪**: 击中焦虑、共鸣、愤怒、惊喜等强情绪
- **好奇心**: 制造信息差，让人想知道答案
- **社交货币**: 转发后能让分享者显得有品位/有见识

### 2. 标题公式识别
对照以下公式分类：

| 公式 | 结构 | 示例 |
|------|------|------|
| 数字+结果 | N个方法/技巧+量化成果 | "3个习惯让我瘦了20斤" |
| 反常识 | 打破认知+揭秘 | "你以为的XX其实是错的" |
| 对比 | A vs B+意外结论 | "月薪3千和3万的人区别在这一点" |
| 好奇留白 | 制造悬念+省略 | "我终于想通了这件事，原来..." |
| 情绪共鸣 | 痛点+情绪词 | "打工人的辛酸，句句扎心" |
| 权威背书 | 经验/数据+观点 | "做了10年XX，这3条最值钱" |
| 利益驱动 | 方法+收益 | "学会这招，效率翻10倍" |

### 3. 开头钩子分析
常见钩子类型：
- **提问式**: 用问题引发思考（"你有没有想过..."）
- **场景式**: 描绘一个读者熟悉的画面
- **数据式**: 用震撼数据开头（"99%的人不知道..."）
- **故事式": 直接进入一个小故事
- **冲突式**: 抛出一个矛盾/争议

判断标准：前3秒（视频）/ 前50字（图文）能否让人停下来。

### 4. 内容结构评估
优秀内容的常见结构：
- **总-分-总**: 观点→论据→总结（最通用）
- **问题-方案**: 痛点→分析→解决方案（教程类）
- **时间线**: 起因→经过→结果（故事类）
- **对比式**: 现象A vs 现象B→深层原因（观点类）
- **递进式**: 现象→原因→本质→行动（深度分析）

### 5. 情绪曲线
分析内容是否包含情绪递进：
```
起步（共鸣/好奇）→ 加深（信息增量）→ 高潮（金句/反转/震撼）→ 收尾（行动召唤）
```

没有情绪高潮的内容很难爆。

### 6. 互动诱因
是什么让用户产生互动行为：
- **点赞触发**: 金句、认同感、被治愈
- **评论触发**: 开放性问题、争议观点、@好友
- **转发触发**: 社交货币、实用价值、身份认同
- **收藏触发**: 干货密度高、步骤清晰、值得反复看
- **关注触发**: 承诺系列内容、展示专业度
