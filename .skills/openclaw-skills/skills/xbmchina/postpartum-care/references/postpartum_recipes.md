# 月子餐详细食谱

## 第一周（排恶露）— 清淡易消化

### 早餐
**小米红枣粥**
```
材料：小米50g、红枣5颗、红糖适量
做法：
1. 小米洗净，加水煮粥
2. 煮至米开花时加入红枣
3. 出锅前加少许红糖调味
功效：健脾养胃、补气血
```

**红糖姜茶**
```
材料：红糖20g、生姜3片
做法：生姜加水煮沸，加红糖即可
功效：暖宫排恶露（喝3-5天即可，不宜过长）
```

### 午/晚餐
**莲藕排骨汤**
```
材料：莲藕1节、排骨200g、姜片3片
做法：
1. 排骨焯水去血沫
2. 莲藕切块
3. 全部放入砂锅，大火煮沸转小火炖1.5小时
4. 加少许盐调味
功效：补铁、促进恶露排出
```

**麻油菠菜猪肝汤**
```
材料：菠菜200g、猪肝100g、芝麻油少许、姜片
做法：
1. 猪肝切片，用清水浸泡30分钟去血水
2. 菠菜焯水备用
3. 锅中加水烧开，放入猪肝煮熟
4. 加入菠菜，淋入少许芝麻油
功效：补铁、补维生素A
```

**山药小米粥**
```
材料：山药100g、小米50g
做法：
1. 山药去皮切块
2. 与小米同煮成粥
功效：健脾养胃、促进消化
```

## 第二周（补气血、排水肿）

### 早餐
**红豆薏米粥**
```
材料：红豆30g、薏米30g、红枣5颗
做法：
1. 红豆和薏米提前浸泡2小时
2. 加水煮粥，快熟时加入红枣
功效：排水肿、健脾祛湿
```

**红枣鸡蛋糕**
```
材料：鸡蛋2个、红枣5颗、红糖10g
做法：
1. 鸡蛋打散
2. 红枣去核切碎
3. 加温水（蛋液的1.5倍）上锅蒸15分钟
功效：补气血、优质蛋白
```

### 午/晚餐
**当归乌鸡汤**
```
材料：乌鸡半只、当归10g、黄芪15g、红枣5颗、枸杞5g、姜片
做法：
1. 乌鸡焯水
2. 所有药材洗净，与乌鸡同炖1.5小时
3. 加少许盐调味
功效：补气血、暖宫（恶露干净后喝更好）
```

**番茄牛腩**
```
材料：番茄2个、牛腩200g、姜片、葱段
做法：
1. 牛腩切块，焯水去血沫
2. 番茄切块，炒出汁
3. 加水炖牛腩1.5小时
功效：补铁、促进伤口恢复
```

**菠菜猪血汤**
```
材料：菠菜200g、猪血100g、姜片
做法：
1. 猪血切块焯水
2. 菠菜焯水去草酸
3. 加水煮开，放入猪血和菠菜
4. 加少许盐和香油
功效：补铁、清理肠道
```

## 第三周（催乳、养气血）

### 催乳食谱
**通草鲫鱼汤**
```
材料：鲫鱼1条、通草5g、姜片
做法：
1. 鲫鱼煎至两面金黄
2. 加水，放入通草和姜片
3. 大火煮沸转小火炖30分钟
4. 喝汤吃鱼肉
功效：催乳通乳
```

**黄豆猪蹄汤**
```
材料：猪蹄1只、黄豆50g、花生30g、姜片
做法：
1. 猪蹄焯水去毛
2. 黄豆提前浸泡1小时
3. 全部放入砂锅，大火煮沸转小火炖2小时
4. 喝汤前去掉浮油
功效：催乳、补充胶原蛋白
```

**花生红枣排骨汤**
```
材料：排骨200g、花生50g、红枣5颗
做法：
1. 排骨焯水
2. 全部放入砂锅炖1.5小时
功效：催乳、补气血
```

**无花果炖鸡**
```
材料：鸡肉200g、无花果5颗、红枣3颗
做法：
1. 鸡肉切块焯水
2. 无花果、红枣洗净
3. 全部炖1小时
功效：天然催乳、补气血
```

### 午/晚餐
**清蒸鲈鱼**
```
材料：鲈鱼1条、姜丝、葱丝、蒸鱼豉油
做法：
1. 鲈鱼洗净，两面划刀
2. 铺上姜丝，上锅蒸10-12分钟
3. 出锅后放葱丝，淋上热蒸鱼豉油
功效：优质蛋白、补DHA、促进伤口恢复
```

**山药炒木耳**
```
材料：山药200g、木耳50g、胡萝卜少许
做法：
1. 山药和胡萝卜切片
2. 木耳泡发
3. 快火翻炒，加少许盐
功效：润肺、补气血、通便
```

## 第四周（固本培元、全面恢复）

### 早餐
**桂圆红枣粥**
```
材料：桂圆肉10g、红枣5颗、糯米50g
做法：同煮成粥
功效：补心血、安心神
```

**红枣核桃糊**
```
材料：红枣5颗、核桃3个、大米30g
做法：全部打成糊煮熟
功效：补气血、补脑
```

### 午/晚餐
**枸杞炖羊肉**
```
材料：羊肉200g、枸杞10g、姜片、当归5g
做法：
1. 羊肉焯水
2. 全部放入砂锅炖1.5小时
功效：温补肾阳、固本培元（适合秋冬季节）
```

**香菇乌鸡汤**
```
材料：乌鸡半只、香菇5朵、红枣5颗、姜片
做法：
1. 乌鸡焯水
2. 香菇泡发
3. 全部炖1.5小时
功效：补气养血、增强免疫力
```

**黑芝麻糊**
```
材料：黑芝麻100g、糯米50g、冰糖适量
做法：
1. 黑芝麻炒香
2. 和糯米一起打成糊
3. 加水煮熟，加冰糖
功效：补肾、乌发、润肠
```

## 月子餐注意事项

1. **少食多餐**：一天5-6餐，避免一次吃太饱
2. **少盐少油**：每天盐<5g，油<25g
3. **温热为主**：不吃冰的，从冰箱拿出来要温热
4. **多样化**：每天食材种类>15种
5. **催奶汤去油**：喝前去掉浮油，防堵奶
