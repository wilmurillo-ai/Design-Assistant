# 辅食添加完整指南

## 辅食添加总原则

- **时间**：6个月开始（不早于4个月、不晚于8个月）
- **信号**：能坐稳、顶舌反应消失、对食物感兴趣
- **顺序**：高铁米粉 → 蔬菜泥 → 水果泥 → 肉泥 → 蛋黄 → 全蛋
- **原则**：每次一种新食物，观察3天，无过敏再添加下一种

---

## 6-7个月：吞咽期

### 第一周：高铁米粉

**冲调方法**：
- 1勺米粉（约5g）+ 30ml温水
- 静置30秒后搅拌成糊状
- 刚开始每天1次，在两顿奶之间喂

**观察**：有无呕吐、腹泻、皮疹

### 第2-4周：蔬菜泥

| 顺序 | 食物 | 添加方法 | 注意事项 |
|---|---|---|---|
| 1 | 南瓜泥 | 蒸熟打泥 | 从稀到稠 |
| 2 | 胡萝卜泥 | 蒸熟打泥 | 胡萝卜素丰富 |
| 3 | 土豆泥 | 蒸熟打泥 | |
| 4 | 西兰花泥 | 焯熟打泥 | 花朵部分 |
| 5 | 山药泥 | 蒸熟打泥 | 轻微过敏体质先试 |
| 6 | 油菜泥 | 焯熟打泥 | 补钙 |

### 7个月：水果泥

| 食物 | 添加方法 | 特点 |
|---|---|---|
| 苹果泥 | 生刮或蒸熟 | 收敛止泻 |
| 香蕉泥 | 直接刮泥 | 容易便秘可选 |
| 牛油果泥 | 直接刮泥 | 优质脂肪 |
| 梨泥 | 生刮或蒸熟 | 清热润肺 |
| 红薯泥 | 蒸熟打泥 | 防便秘 |

---

## 8-9个月：蠕嚼期

### 食物质地升级

从细腻泥糊 → 粗颗粒/小碎末

### 推荐食谱

**番茄猪肝泥**
```
1. 猪肝切片，用清水浸泡30分钟去血水
2. 番茄划十字，开水烫过去皮切块
3. 两者一起蒸15分钟
4. 打泥，可加少量温水和滴几滴核桃油
```

**鱼肉泥**
```
1. 选择刺少的鱼（鳕鱼/三文鱼/鲈鱼）
2. 蒸10分钟，去皮去刺
3. 打泥，可加少量温水和蔬菜泥
```

**青菜肉末粥**
```
1. 粥煮烂，加入瘦肉末
2. 快熟时加入青菜碎
3. 可加少量盐或不加
```

### 每日辅食安排

| 时间 | 内容 |
|---|---|
| 上午10点 | 辅食（高铁米粉+肉/菜泥） |
| 下午3点 | 水果泥 |
| 晚上6点 | 辅食（粥/烂面条+肉/菜） |

---

## 10-11个月：细嚼期

### 食物质地升级

小碎末 → 小块状（指甲盖大小）

### 推荐食谱

**迷你小馄饨**
```
1. 馄饨皮切成4小块
2. 猪肉/牛肉+蔬菜做馅
3. 煮到馄饨皮透明即可
```

**蔬菜鸡蛋饼**
```
1. 胡萝卜/西葫芦擦丝
2. 加鸡蛋和少量面粉搅匀
3. 小火煎至两面金黄
4. 切成小块喂食
```

**小米饼**
```
1. 小米煮烂
2. 加鸡蛋和蔬菜末
3. 平底锅煎成小饼
```

### 手指食物（BLW自主进食法）

10个月开始可引入：
- 蒸软的手指食物（西兰花小朵、胡萝卜条）
- 软烂的水果条（香蕉、牛油果）
- 烤面包条
- 小馄饨/饺子

---

## 12个月+：咀嚼期

### 饮食接近成人

- 可以加少量盐（<1g/天）
- 可以喝全脂牛奶（每天300-500ml）
- 仍需单独烹饪，少糖少盐

### 推荐一日三餐

| 餐次 | 内容 |
|---|---|
| 早餐 | 粥/面条/鸡蛋饼 + 牛奶 |
| 午餐 | 米饭 + 肉/鱼 + 蔬菜 |
| 下午点心 | 水果 + 酸奶/小饼干 |
| 晚餐 | 面条/饺子/粥 + 肉/菜 |
| 睡前 | 牛奶200ml |

## 辅食添加禁忌

### ❌ 不能吃的食物

| 食物 | 原因 |
|---|---|
| 蜂蜜 | 1岁内可能含肉毒杆菌孢子 |
| 整颗坚果 | 窒息风险 |
| 整颗葡萄/小番茄 | 窒息风险（要切半或切碎） |
| 生鸡蛋 | 沙门氏菌风险 |
| 整颗珍珠奶茶料 | 窒息风险 |
| 腌制品/腊肉 | 高盐高亚硝酸盐 |
| 整颗草莓 | 窒息风险，可切小块 |
| 茶/咖啡 | 含咖啡因 |

### ⚠️ 需谨慎的食物

| 食物 | 注意事项 |
|---|---|
| 蛋清 | 先少量尝试，观察3天 |
| 海鲜 | 选新鲜，观察过敏 |
| 花生 | 花生酱可以，整颗不行 |
| 芒果/菠萝 | 可能过敏，少量试 |

## 常见问题

### Q：宝宝不爱吃辅食怎么办？

**方法**：
1. 不强迫，饿了就吃了
2. 变换食物性状和颜色
3. 大人表演吃得很香的样子
4. 每次只提供新食物，不和熟悉的混在一起

### Q：过敏了怎么办？

**轻度过敏**（轻微红疹）：继续观察，停该食物3天后再试
**中度过敏**（大面积红疹、呕吐）：停该食物，3个月后再试
**重度过敏**（呼吸困难、嘴唇肿胀）：立即就医，之后严格回避

### Q：辅食和奶怎么安排？

6-7个月：辅食1次/天，主要还是奶
8-9个月：辅食2次/天，奶量开始减少
10-12个月：辅食3次/天，奶为辅
1岁后：辅食为主，奶为辅（300-500ml/天）
