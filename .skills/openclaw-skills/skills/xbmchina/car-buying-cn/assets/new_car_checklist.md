# 新车验车检查清单（打印版）

## 验车前准备

- [ ] 带身份证、银行卡、购车合同
- [ ] 带手机（拍照记录问题）
- [ ] 带懂车朋友（可选但建议）
- [ ] 预约上午验车（光线好，有问题当天解决）

---

## 一、文件核查（最重要！）

**必须齐全**：
- [ ] **购车发票**（四联：发票联、抵扣联、报税联、注册登记联）
- [ ] **车辆合格证**（上牌必需，没有无法上牌！）
- [ ] **三包凭证**（质保依据，写明3年/10万公里）
- [ ] **使用说明书**（随车工具清单）
- [ ] **保养手册**（首保免费凭证）
- [ ] **保险单**（交强险+商业险，生效日期正确）
- [ ] **临时牌照**（有效期内，通常15天）

---

## 二、外观检查

**绕车一周，仔细查看**：
- [ ] **车漆**：有无划痕、凹陷、色差（重点：车门边缘、保险杠、后视镜）
- [ ] **玻璃**：前挡/侧窗/后挡，有无裂纹、气泡、划痕
- [ ] **车灯**：大灯、尾灯、转向灯、刹车灯、雾灯全试一遍
- [ ] **轮胎**：
  - 4个轮胎品牌型号一致
  - 生产日期（4位数字，如2523=2023年第25周）
  - 轮胎日期应早于整车生产日期
- [ ] **车门**：4个车门开关顺畅，无异响，密封条完好
- [ ] **后备箱**：开关顺畅，液压杆正常，备胎/工具齐全
- [ ] **底盘**（要求升起查看）：
  - 无漏油、漏液
  - 排气管无异常锈蚀
  - 底盘护板完好

---

## 三、内饰检查

**座椅**：
- [ ] 皮质无破损、无划痕
- [ ] 电动调节正常（前后/上下/靠背）
- [ ] 座椅加热/通风正常（如有）
- [ ] 安全带拉出顺畅，锁止正常

**仪表中控**：
- [ ] 里程数 ≤ 50km（正常范围）
- [ ] 各指示灯正常（发动机故障灯、ABS灯等熄灭）
- [ ] 中控屏幕无坏点，触控灵敏
- [ ] 车机系统流畅，不卡顿
- [ ] 倒车影像清晰，雷达正常

**空调音响**：
- [ ] 空调制冷/制热正常，风量可调
- [ ] 出风口无异响
- [ ] 音响各喇叭都响，无杂音
- [ ] 蓝牙/USB连接正常

**天窗**：
- [ ] 开关顺畅，无异响
- [ ] 关闭后密封良好（不漏水）
- [ ] 遮阳帘正常

---

## 四、功能检查

**钥匙**：
- [ ] 两把钥匙都试（遥控解锁/上锁）
- [ ] 机械钥匙可开门
- [ ] 无钥匙进入/启动正常（如有）

**车窗**：
- [ ] 四门玻璃升降正常
- [ ] 一键升降功能正常
- [ ] 防夹功能有效（用手测试）

**后视镜**：
- [ ] 电动调节正常
- [ ] 加热功能正常（如有）
- [ ] 电动折叠正常（如有）

**其他**：
- [ ] 雨刷各档位正常，喷水正常
- [ ] 内外后视镜清晰
- [ ] 点烟器/12V电源正常
- [ ] USB/Type-C充电口正常

---

## 五、发动机舱检查

- [ ] 发动机表面无漏油、漏液
- [ ] 各油液液位正常（机油、冷却液、刹车油）
- [ ] 电瓶接线牢固，无腐蚀
- [ ] 发动机启动顺畅，无异响
- [ ] 怠速稳定，不抖动
- [ ] 踩油门响应正常，无异响

---

## 六、新能源特有检查

- [ ] 电池电量显示正常
- [ ] 充电口盖板顺畅开启/关闭
- [ ] 充电桩连接正常（如有条件现场测试）
- [ ] 续航里程显示合理
- [ ] 能量回收功能正常

---

## 七、随车工具核查

**必须齐全**：
- [ ] 备胎（或补胎工具）
- [ ] 千斤顶
- [ ] 轮胎扳手
- [ ] 三角警示牌
- [ ] 反光背心
- [ ] 拖车钩
- [ ] 急救包（部分车型）
- [ ] 充电桩（新能源车）

---

## 八、问题记录

**发现问题立即记录**：
- [ ] 拍照/录像留存证据
- [ ] 要求4S店书面承诺解决时间
- [ ] 小问题：要求送保养/礼品补偿
- [ ] 大问题：拒绝提车，要求换车

---

## 九、提车确认

**付款前最后确认**：
- [ ] 所有文件齐全且信息正确
- [ ] 车辆无质量问题
- [ ] 承诺的赠品/精品已交付
- [ ] 首保时间/里程确认
- [ ] 4S店售后电话留存

**恭喜提车！**

---

## 验车小贴士

1. **不要急着交钱**：先验车，没问题再付款
2. **带朋友去**：两个人看比一个人看得细
3. **白天去**：光线好，容易发现问题
4. **不要怕麻烦**：有问题当场提，出了门就难维权
5. **保留证据**：所有问题拍照，书面确认解决方案
