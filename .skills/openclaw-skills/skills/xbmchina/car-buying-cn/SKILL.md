## 一、预算规划

### 买车总成本构成

```
落地价 = 裸车价 + 购置税 + 保险费 + 上牌费 + 精品/装潢费

养车年成本 = 油费/电费 + 保险费 + 保养费 + 停车费 + 过路费 + 洗车美容
```

### 预算比例建议

| 项目 | 建议比例 | 说明 |
|---|---|---|
| 车价（裸车） | 年收入的 0.5-1 倍 | 月入1万→买6-12万的车 |
| 首付 | 车价的 30-50% | 贷款首付最低20% |
| 月供 | 月收入 ≤ 20% | 避免影响生活质量 |
| 应急储备 | 车价的 10% | 维修、事故备用 |

### 全款 vs 贷款对比

| 方式 | 优点 | 缺点 | 适合人群 |
|---|---|---|---|
| **全款** | 无利息、议价空间大、手续简单 | 一次性占用资金 | 资金充裕、理财收益<贷款利率 |
| **贷款** | 资金压力小、可买更贵的车 | 有利息、手续复杂 | 资金紧张、理财收益>贷款利率 |

## 二、选车决策框架

### 燃油车 vs 新能源

| 维度 | 燃油车 | 新能源（纯电） | 插电混动 |
|---|---|---|---|
| **购车价** | 同配置低2-5万 | 同配置高2-5万 | 介于两者之间 |
| **使用成本** | 油费 0.5-0.8元/km | 电费 0.1-0.2元/km | 市区用电长途用油 |
| **续航** | 500-800km | 400-700km | 800-1200km |
| **补能** | 3分钟加油 | 快充30-60分钟 | 可油可电 |
| **保养** | 5000-10000km/次 | 20000km/次 | 介于两者之间 |
| **保值率** | 3年60-70% | 3年40-50% | 3年50-60% |
| **适合场景** | 长途多、充电不便 | 市区通勤、充电方便 | 全能型 |

### 轿车 vs SUV

| 维度 | 轿车 | SUV |
|---|---|---|
| **油耗/电耗** | 低 10-20% | 高（风阻大） |
| **操控** | 更好（重心低） | 一般（重心高） |
| **空间** | 后备箱小 | 后备箱大、头部空间好 |
| **通过性** | 一般 | 好（离地间隙高） |
| **视野** | 一般 | 好（坐姿高） |
| **适合人群** | 城市通勤为主 | 有娃家庭、经常出游 |

### 新车 vs 二手车

| 维度 | 新车 | 二手车 |
|---|---|---|
| **价格** | 高（含购置税） | 低（省购置税+折旧） |
| **质量** | 有保障 | 有风险（需检测） |
| **质保** | 3年/10万公里 | 剩余质保或无 |
| **选择** | 配置/颜色可选 | 现车有限 |
| **心理** | 全新体验 | 可能有心理障碍 |
| **适合人群** | 预算充足、不懂车 | 预算有限、会验车 |

## 三、贷款方案

### 贷款渠道对比

| 渠道 | 年利率 | 优点 | 缺点 |
|---|---|---|---|
| **厂家金融** | 0-5% | 审批快、手续简单、常有免息活动 | 额度有限、绑定保险 |
| **银行车贷** | 4-8% | 利率透明、额度高 | 审批慢、要求多 |
| **信用卡分期** | 6-12% | 手续最简单 | 利率高、额度低 |
| **第三方金融** | 8-15% | 门槛低 | 利率高、套路多 |

### 厂家金融常见套路

- **免息陷阱**："2年免息"但收高额手续费（相当于利息）
- **绑定消费**：强制店内买保险、买精品
- **GPS费**：收2000-3000元GPS安装费（可谈）
- **金融服务费**：收3000-5000元（可谈，可拒）

## 四、砍价谈判

### 价格构成拆解

```
裸车价 = 厂商指导价 - 现金优惠
落地价 = 裸车价 + 购置税 + 保险费 + 上牌费 + 精品费

购置税 = 裸车价 ÷ 1.13 × 10%（新能源车免征）
保险费 = 交强险 + 商业险（车损+三者+座位+盗抢）
上牌费 = 自己办 120元 / 4S代办 500-2000元
```

### 4S店砍价技巧

1. **多家比价**：至少跑3家4S店，拿A店报价压B店
2. **月底/季末去**：销售冲业绩，折扣最大
3. **全款威胁**：先说全款，拿到底价后再说贷款（贷款利润高，销售更愿意降价）
4. **分项砍价**：
   - 裸车价：压到最低
   - 购置税：国家规定，无法砍价
   - 保险费：可自己买，返点自己拿
   - 上牌费：自己办120元，4S收500-2000可谈
   - 精品费：脚垫/贴膜/记录仪，可送可砍
5. **不要表露急迫**："还在对比"、"下周再决定"
6. **要求送保养**：比现金优惠更实在

### 常见加价项目及应对

| 项目 | 4S店报价 | 实际成本 | 应对策略 |
|---|---|---|---|
| 金融服务费 | 3000-5000元 | 0 | **可拒**，厂家金融不收 |
| GPS费 | 2000-3000元 | 200元 | **可拒**，银行不强制 |
| 上牌费 | 500-2000元 | 120元 | **自己办**或砍到300 |
| 精品装潢 | 3000-8000元 | 1000元 | **不要**， aftermarket更便宜 |
| 延保 | 3000-6000元 | — | 视品牌而定，国产可买，合资慎重 |

## 五、验车检查

### 新车PDI检查清单

**外观检查**
- [ ] 车漆：全车绕一圈，看有无划痕、凹陷、色差
- [ ] 玻璃：前挡/侧窗/后挡，看有无裂纹、气泡
- [ ] 轮胎：生产日期（4个数字，如2523=2023年第25周），应早于整车生产日期
- [ ] 车门：开关顺畅，密封条完好
- [ ] 车灯：远近光、转向灯、刹车灯、雾灯全试一遍

**内饰检查**
- [ ] 座椅：皮质无破损，电动调节正常
- [ ] 仪表：里程≤50km（正常），各指示灯正常
- [ ] 中控：屏幕无坏点，触控灵敏，车机流畅
- [ ] 空调：制冷/制热正常，无异响
- [ ] 音响：各喇叭都响，无杂音
- [ ] 天窗：开关顺畅，无异响，不漏水

**功能检查**
- [ ] 钥匙：两把钥匙都试，遥控/机械都正常
- [ ] 车窗：四门升降正常，防夹功能有效
- [ ] 后视镜：电动调节/加热/折叠正常
- [ ] 雨刷：各档位正常，喷水正常
- [ ] 充电口（新能源）：充电正常，盖板顺畅

**底盘检查**
- [ ] 升起底盘，看有无漏油、漏液
- [ ] 排气管：无异常锈蚀
- [ ] 底盘护板：完好无损

**文件核查**
- [ ] 购车发票（四联：发票联、抵扣联、报税联、注册登记联）
- [ ] 车辆合格证（上牌必需）
- [ ] 三包凭证（质保依据）
- [ ] 使用说明书、保养手册
- [ ] 保险单（交强险+商业险）
- [ ] 临时牌照（有效期内）

### 二手车深度检测

**必查项目**
- [ ] 事故记录：查保险出险记录、4S维修记录
- [ ] 泡水痕迹：座椅滑轨锈蚀、安全带水渍、地毯异味
- [ ] 火烧痕迹：发动机舱线束、内饰焦味
- [ ] 调表嫌疑：轮胎/刹车片磨损与里程是否匹配
- [ ] 手续齐全：登记证书、行驶证、购置税完税证明、保险单
- [ ] 抵押/查封：车管所查询车辆状态
- [ ] 过户次数：登记证书看过户记录

**建议找第三方检测**
- 查博士、268V、检车家等，费用300-800元
- 出具检测报告，有问题可拒买或砍价

## 六、上牌流程

### 购置税缴纳
- **燃油车**：裸车价÷1.13×10%，税务局或网上缴纳
- **新能源车**：2025年底前免征

### 保险购买
**必买**：交强险（950元/年）
**强烈建议买**：
- 车损险（赔自己车）
- 三者险（建议200万-300万）
- 座位险（赔车上人员）
- 医保外用药责任险（补充三者）

### 上牌步骤
1. **临牌**：4S店代办或车管所申请，有效期15天
2. **验车**：车管所查验车辆识别代号、发动机号
3. **选号**：现场50选1或网上自编自选
4. **制证**：行驶证、登记证书
5. **领牌**：现场领取或邮寄到家

**费用**：自己办约120元，4S代办500-2000元
