# 房产税费全解析

## 新房税费（购入时一次性）

### 契税（买家承担）
**计税依据：** 合同成交价格（网签价）

| 类型 | 面积 | 税率 |
|---|---|---|
| 首套房 | ≤ 90㎡ | **1%** |
| 首套房 | > 90㎡ | **1.5%** |
| 二套房 | ≤ 90㎡ | **1%** |
| 二套房 | > 90㎡ | **2%** |
| 三套房及以上 | 不限 | **3%** |

### 维修基金（新房）
- **有电梯：** 当地建安成本×2-8%（各地标准不同）
  - 北京：每平米120元
  - 上海：每平米50元（多层）/ 高层每平米100元
  - 广州：每平米75元
- **无电梯：** 当地建安成本×2%（约每平米20-50元）

### 产权登记费
- 住宅：80元/件
- 非住宅：550元/件

### 其他可能费用
- 预告登记费：住宅80元
- 抵押登记费：住宅80元
- 物业费（预收）：通常1-3个月
- 暖气/燃气等开口费（部分城市已取消）

## 二手房税费（详细版）

### 契税（买家承担）
（同新房标准）

### 增值税及附加（卖家承担，俗称"营业税"）

| 持有年限 | 是否普通住宅 | 计税方式 |
|---|---|---|
| **不满2年** | — | 全额征收：网签价×5.3% |
| **满2年** | 普通住宅 | **免征** |
| **满2年** | 非普通住宅 | 差额征收：（现价-原价）×5.3% |

**普通住宅认定标准（各地略有不同）：**
- 单套建筑面积≤120（或144）㎡
- 容积率≥1.0
- 成交价格≤同级别土地上新建商品住房平均价格的1.2（或1.44）倍

### 个人所得税（卖家承担）

| 情况 | 计税方式 |
|---|---|
| **满5唯一住房**（家庭唯一住房且满5年） | **免征** |
| 普通情况 | 差额征收20% 或 全额征收1%（可选择） |
| 特殊：无法提供原值凭证 | 全额1%征收 |

**差额计算：**（现成交价 - 原购置成本 - 合理费用）× 20%
**全额计算：** 成交价 × 1%（更常用，卖家通常选择此方式）

### 中介费（买家承担）
- 链家/贝壳等：通常 **成交价×2%**（可谈）
- 建议谈至 **1-1.5%**，大中介也要有竞争压力
- 注意：问清是否包含权证办理、贷款服务等费用

### 其他费用
- 评估费：成交价×0.3-0.5%（部分城市已取消）
- 担保费：若有担保公司赎楼，收取0.5-1%

## 税费省钱实操技巧

### 1. 核实"满二"、"满五唯一"
这是省钱关键，签合同前务必核实：

**查证方法：**
- 要求卖家提供房产证，看登记日期
- 让中介拉产调，看产权年限
- "唯一"：卖家本人及家庭成员名下的住房查询

### 2. 税费谁承担要写进合同
口头约定的税费承担不算数。
常见写法：
```
"本次交易所产生的所有税费（包括但不限于契税、增值税、个人所得税），
均由买方承担。"
```
或者：
```
"卖方承担的税费为：___；买方承担的税费为：___"
```

### 3. 网签价可以合理规划
网签价（过户时的申报价格）可以在一定范围内调整：
- **拉低网签价**：降低税费（但银行贷款按评估价，不能低于此值）
- **拉高网签价**：多贷款（有风险，且受评估价限制）

### 4. 巧用公积金冲抵
如果卖家有房贷未还清：
- 用买家的首付帮卖家赎楼（资金监管，确保安全）
- 或者找担保公司赎楼（收取担保费）

## 税费计算实例

**情景：** 购入总价300万、100㎡二手普通住宅，首套，网签价=成交价，卖家持有多年的情况

| 税费项 | 计算 | 金额 |
|---|---|---|
| 契税（买家） | 300万×1.5%（首套>90㎡） | **45,000元** |
| 增值税 | **满2年，免征** | **0元** |
| 个人所得税 | **满5唯一，免征** | **0元** |
| 中介费 | 300万×2% | **60,000元** |
| **合计** | | **105,000元** |

**情景：** 同上，但卖家持房不满2年，且非唯一

| 税费项 | 计算 | 金额 |
|---|---|---|
| 契税（买家） | 300万×1.5% | 45,000元 |
| 增值税（卖家） | 300万×5.3% | **159,000元** |
| 个人所得税（卖家） | 300万×1% | **30,000元** |
| 中介费 | 300万×2% | 60,000元 |
| **合计** | | **294,000元** |

> ⚠️ 注意：卖家承担的税费，最终往往通过提高成交价转嫁给买家。
> 购入前要将所有税费计入总成本。
