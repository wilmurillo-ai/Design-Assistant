# 贷款全攻略

## 贷款基础知识

### 几个关键概念

- **LPR（贷款市场报价利率）：** 央行每月20日公布，是房贷利率的"基准锚"
- **基点（BP）：** 1BP = 0.01%。房贷利率 = LPR ± 基点
- **评估价：** 银行委托评估公司对房产的估值，通常为成交价的80-90%
- **可贷成数：** 评估价 × 成数 = 可贷金额（首套通常70-80%）

### 现行利率区间（2025-2026年参考）

| 类型 | 利率区间 | 说明 |
|---|---|---|
| 首套公积金 | 3.1%（5年+）/ 2.75%（5年以下） | 历史最低 |
| 首套商贷 | 3.6-4.1% | 城市/银行差异大 |
| 二套商贷 | 4.1-4.6% | LPR+20BP以上 |
| 经营贷（违规） | 2.8-3.5% | ❌ 灰色地带，不要碰 |

## 公积金贷款

### 额度上限（各地不同，以当地为准）

| 城市 | 单人上限 | 夫妻双人上限 |
|---|---|---|
| 北京 | 60万 | 120万 |
| 上海 | 50万 | 100万 |
| 广州 | 60万 | 100万 |
| 深圳 | 50万 | 90万 |
| 成都 | 40万 | 70万（省直）|
| 杭州 | 50万 | 100万 |

### 公积金贷款条件
1. 连续缴存公积金满6-12个月（各地不同）
2. 名下无未还清的公积金贷款
3. 信用记录良好
4. 购房行为真实

### 公积金贷款优点
- ✅ 利率显著低于商贷（3.1% vs 3.9%，100万30年省约15万利息）
- ✅ 提前还款无违约金
- ✅ 月供压力更小

### 公积金贷款缺点
- ❌ 额度有限，大房子不够用
- ❌ 组合贷流程长，放款慢（比纯商贷慢2-4周）
- ❌ 对开发商/卖家不够友好（回款慢），部分新房不接受公积金

## 商业贷款

### 贷款成数

| 情况 | 最高可贷成数 |
|---|---|
| 首套普通住宅 | 80%（评估价） |
| 首套非普通住宅 | 70% |
| 二套（大多数城市） | 60-70% |
| 二套（限购严格城市如北京） | 40-60% |

**非普通住宅标准：** 容积率<1.0，或面积>144㎡，或价格明显高于同区域均价

### 贷款年限

- 最长：**30年**（房龄+贷款年限≤50-70年，各银行不同）
- 贷款年限受年龄限制：男性≤65岁，女性≤60岁（有的银行至65岁）
- 实际能贷多少年 = min(30年, 50年房龄后, 年龄限制)

### 流水要求

- 流水金额 ≥ 月供 × 2（银行审查的核心）
- 流水认定：税后打卡工资为主，微信/支付宝转账通常不认
- 父母作为共同还款人：可合并计算流水

## 提前还款策略

### 何时提前还款最划算？

**等额本息：**
- 在还款周期的前1/3时间段内提前还款，能省更多利息
- 例如30年贷款，在前10年还清效果最好
- **超过贷款年限一半后，等额本息已还的大部分是利息，提前还款意义不大**

**等额本金：**
- 每月还的本金固定，提前还款时机没那么严格
- 越早还越好

### 提前还款方式

| 方式 | 说明 |
|---|---|
| **月供不变，缩短年限** | ⭐推荐，省利息最多 |
| 月供减少，年限不变 | 压力小，但省息少 |
| 部分还本金 | 减少剩余本金 |

### 操作建议
- 提前还款尽量选年初或利率调整后（利息节省更多）
- 公积金贷款提前还款无违约金，随时可还
- 商贷提前还款通常需预约（提前1个月申请）

## 贷款被拒的常见原因

| 原因 | 解决方案 |
|---|---|
| 流水不够 | 提供其他资产证明，父母作为共同还款人 |
| 征信有逾期 | 修复征信后再申请（通常2年后逾期影响消除） |
| 负债率过高 | 还清其他贷款后再申请 |
| 收入不稳定 | 提供租金/分红等辅助收入证明 |
| 小额贷款未还 | 还清并提供结清证明 |
| 信用卡使用率高 | 降低信用卡使用额度（<70%） |

## 组合贷款实操

**情景：** 房子总价300万，评估价270万，首付30%自付，需要贷款210万

**公积金额度计算：**
- 夫妻双人公积金上限：假设100万
- 公积金能贷：100万（3.1%）
- 剩余商贷：210-100 = 110万（假设4.0%）

**月供对比（30年）：**

| 方式 | 月供 | 总利息 |
|---|---|---|
| 纯商贷210万 | 10,009元 | 150万 |
| 公积金100万+商贷110万 | 9,192元 | 121万 |
| 纯公积金100万（不够用） | — | — |

> **组合贷比纯商贷每月少还约817元，30年省约29万利息**
