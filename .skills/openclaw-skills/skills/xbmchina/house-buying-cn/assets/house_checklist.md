# 买房看房检查清单（打印版）

## 新房看房检查清单

### 五证核查（必须看原件）

```
□ 国有土地使用证
□ 建设用地规划许可证
□ 建设工程规划许可证
□ 建筑工程施工许可证
□ 商品房预售许可证（最重要，可查当地住建局官网核实）
```

### 户型核查

```
□ 确认非承重墙位置（问销售或看图纸）
□ 厨房/卫生间是否有直接采光窗（无窗为暗厨/暗卫）
□ 每个房间尺寸是否合理（主卧≥12㎡，次卧≥8㎡）
□ 得房率多少（写在合同里）
□ 楼层层高多少（低于2.8米会压抑，标准≥2.9米）
□ 电梯配置（两梯四户还是两梯两户，梯户比影响等待时间）
□ 公摊面积占比（高层公摊通常22-28%）
□ 户型图与样板间是否一致（注意非交付标准差异）
□ 阳台/飘窗/设备平台面积（是否计容影响实际使用）
□ 空调外机位预留
□ 烟道/排污管位置（厨房/卫生间正上方）
```

### 小区规划核查

```
□ 人车分流设计（人车分流更安全更安静）
□ 地上停车位比例（地上车位多说明车位不足）
□ 绿化率（≥35%为佳）
□ 容积率（≤3.0为佳，越低越舒适）
□ 楼间距（参考：楼间距≥前楼高度×0.7）
□ 配套公示（不利因素：垃圾房/配电房/加油站/焚烧站等）
□ 会所/底商规划（可能带来噪音）
□ 物业品牌（万科/龙湖/中海等品牌物业更有保障）
□ 物业费标准（住宅通常2-3.5元/㎡/月）
□ 交付时间（具体到年月日）
□ 违约金条款（≥总房价万分之一/天为合理）
□ 装修标准（精装修要逐项核对品牌型号，写进合同）
□ 延期交付违约条款
```

### 实地看房必查

```
□ 采光：上午/中午/下午各看一次
□ 噪音：站在窗户边感受外部噪声
□ 通风：感受穿堂风效果
□ 防水：顶楼重点看天花板有无水渍
□ 地势：小区是否地势低洼（排水问题）
□ 周边不利因素：高压线/工厂/殡仪馆/焚烧站/变电站
□ 交通：到最近地铁站实际走一遍，测时间
□ 周边配套：超市/菜市场/医院/学校距离
```

## 二手房看房检查清单

### 进门前核查

```
□ 要求看房产证原件（拍照留存）
□ 权利人信息（姓名/证件号，与合同是否一致）
□ 登记时间（计算是否"满二"、"满五"）
□ 建筑面积（与实际是否一致）
□ 是否有抵押/查封（现场要求查档，或自己查）
□ 房屋地址是否一致（避免地址造假）
□ 土地性质（住宅70年产权 / 公寓40-50年）
□ 土地到期后处理（目前到期可续期，费用很低）
```

### 进屋检查

```
□ 墙面/天花板有无水渍、渗水痕迹
  （重点：墙角、窗户边框、厨卫顶部）
□ 墙角发霉/发黄（说明有渗水历史）
□ 门/窗开关是否顺畅，密封性
□ 所有插座通电测试（带个手机充电器）
□ 大功率电器测试（空调/热水器）
□ 厨房下水道是否反味
□ 卫生间是否漏水（看楼下天花板）
□ 水压测试（打开水龙头，感受水压）
□ 燃气是否开通（问物业）
□ 暖气是否可用（北方城市）
□ 采光（上午+下午各看一次）
□ 噪音（站在窗户边听）
□ 通风（每个房间感受）
□ 装修情况（决定要不要敲掉重装）
□ 家具家电是否留下（写在合同里）
□ 前业主留下的费用是否结清（物业费/水电燃气）
```

### 小区环境检查

```
□ 楼道是否干净整洁
□ 电梯是否维护良好（有无异味/损坏）
□ 小区绿化维护情况
□ 停车位情况（地上/地下，租金多少）
□ 物业服务质量（问门卫/保洁态度）
□ 邻居情况（友好度/是否群租）
□ 电动车停放/充电管理
□ 安保情况（门禁是否严）
□ 周边不利因素（与新房相同）
```

### 产权特别核查

```
□ 产权人是否都同意出售（多人共有需全部同意）
□ 是否有租约（买卖不破租赁，提前告知租客）
□ 是否有户口未迁出（影响落户）
□ 学位是否被占用（到教育局核实）
□ 是否为凶宅（邻居打听+网上搜索）
□ 是否有违建（封闭阳台/搭阁楼等）
□ 是否是"老赖"房产（最高法网站可查）
□ 卖家婚姻状态（已婚需配偶同意出售公证）
```

## 合同签订前最终核查

```
□ 产调已完成，确认无查封冻结
□ 资金监管协议已签（首付款打进监管账户）
□ 补充协议已签（五证/税费/交房/迁户等）
□ 定金收据/付款凭证齐全
□ 中介费已谈好，写明金额和支付时间
□ 违约条款公平（违约金通常为总价5-20%）
□ 争议解决条款（约定仲裁或起诉）
□ 附上房产证复印件和房屋照片
□ 卖家已提供赎楼授权（如有未还贷款）
□ 确认贷款银行和时间安排
□ 约定交房时的状态（清空/保留家具/物业结清）
□ 确认水电气账户过户时间
□ 预留物业押金（通常5000-10000元）
□ 房产证领取时间约定
□ 卖方户口迁出时间（若有）
□ 学位占用情况（写入合同）
□ 签署前已完整阅读全文，不明白的条款已要求解释
□ 全部口头约定均已写入书面合同
```
