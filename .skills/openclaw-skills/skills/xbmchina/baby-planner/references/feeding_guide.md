# 辅食添加完整指南

## 何时添加辅食

**时间：** 满6月龄（出生180天）开始，不早于4月龄，不晚于8月龄

**添加信号（必须同时满足）：**
1. ✅ 能独坐或靠坐（靠坐也行）
2. ✅ 顶舌反射消失（舌头不再把勺子顶出来）
3. ✅ 对食物感兴趣（大人吃东西时张嘴/伸手抓）
4. ✅ 体重达到出生时2倍（早产儿需矫正月龄）

## 第一口辅食

**首选：高铁米粉**
- 原因：6月龄后母乳铁含量骤降，婴儿自身铁储备开始消耗
- 不要用：家庭自制米粉（缺铁）、奶米粉（含糖）
- 冲调：温水/40°C，先水后米粉，稀糊状（能用勺子舀起来又流下去）

**添加顺序（无绝对标准）：**
```
米粉(1-2周) → 菜泥(根茎类：土豆/胡萝卜 → 叶菜类：菠菜/西兰花)
→ 果泥 → 肉泥 → 蛋黄 → 手指食物
```

**排敏原则：**
- 每种新食物连续吃3天，观察过敏
- 过敏症状：皮疹、呕吐、腹泻、嘴巴肿
- 出现过敏：该食物停3个月后再试
- 常见高致敏食物：**鸡蛋、牛奶、花生、坚果、海鲜、大豆、小麦**

## 各阶段食物形态

### 6-7月龄：吞咽期
**形态：** 细腻的泥糊状
**每日安排：**
- 辅食：1次/天，在两顿奶之间
- 奶量：600-800ml/天（减少不明显）
- 时间：上午10-11点（便于观察过敏）

**每日示例：**
```
早餐：母乳/配方奶
午餐：高铁米粉1-2勺 + 土豆泥
下午：母乳/配方奶
晚餐：母乳/配方奶
睡前：母乳/配方奶
```

### 7-8月龄：蠕嚼期
**形态：** 稠糊状 → 带小颗粒的碎末
**变化：**
- 辅食：2次/天
- 引入肉泥（猪肉/牛肉/鸡肉），补铁效果好
- 引入蛋黄（从1/4开始，观察3天）
- 可以吃猪肝粉（补铁）

**每日示例：**
```
早餐：母乳/配方奶
午餐：米粉 + 猪肉泥 + 胡萝卜泥
下午：水果泥（苹果/香蕉，用勺子刮着吃）
晚餐：母乳/配方奶
```

### 8-9月龄：细嚼期
**形态：** 小颗粒/碎末，手指食物开始
**变化：**
- 辅食：2次/天，逐渐替代1顿奶
- 手指食物：蒸软的手指条（西兰花小朵、蒸南瓜条、香蕉）
- 可以喝少量水（用杯子，不是奶瓶）

**每日示例：**
```
早餐：母乳/配方奶
午餐：碎面/烂粥 + 鱼肉泥 + 菠菜碎
下午水果：苹果薄片/香蕉
晚餐：高铁米粉 + 猪肝粉 + 西兰花碎
```

### 9-12月龄：咀嚼期
**形态：** 软烂的家庭食物，切碎/切小块
**变化：**
- 辅食：3次/天，接近家庭用餐时间
- 奶量：500-600ml/天
- 引入杯子喝水（1岁后戒奶瓶，防龋齿）
- 可以吃全蛋、酸奶（不过敏的前提下）

**手指食物参考：**
- 蒸蔬菜：胡萝卜条、西兰花朵、南瓜块
- 水果：苹果薄片、熟梨条、熟桃块
- 蛋白质：煮烂的鸡腿肉条、豆腐块、鱼肉条
- 碳水：蒸糕、发糕、松饼

## 辅食工具清单

| 工具 | 用途 | 推荐 |
|---|---|---|
| 辅食勺 | 初期喂食 | 硅胶软勺（不伤牙龈） |
| 辅食碗 | 盛装 | 吸盘碗（防打翻） |
| 围兜 | 防脏 | 硅胶围兜（大兜型） |
| 料理棒/辅食机 | 打泥 | 博朗/小熊料理棒 |
| 辅食格 | 分装冷冻 | 硅胶冰格 |
| 餐椅 | 培养用餐习惯 | 可调节高度/倾斜 |

## 常见问题

**Q：宝宝不爱吃辅食怎么办？**
A：正常，给10-15次尝试机会，不要强迫。可以在宝宝饿的时候先给辅食再给奶。

**Q：可以加调味品吗？**
A：1岁前不加盐、糖、酱油。天然食材的味道足够。

**Q：辅食可以冷冻吗？**
A：可以。肉泥、蔬菜泥冷冻保存3天，水果泥建议现做现吃。分装到辅食格冷冻。

**Q：奶粉还是母乳为主？**
A：6-12月龄，奶仍是主要营养来源（占50%以上），辅食是补充。
