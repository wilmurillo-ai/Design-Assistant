# 公众号内容模板库

## 一、孕期周记模板

### 标题公式
```
[孕周]+[情绪/事件]+[悬念]：如"孕20周|我第一次感受到胎动，那种感觉太神奇了"
[数字]+[痛点]：如"孕期最后30天，我最后悔没做这3件事"
对比型：如"怀孕前 vs 怀孕后，我的生活发生了哪些变化"
```

### 正文结构模板

```markdown
## [孕X周] [日期]

### 🌡️ 本周身体记录
| 项目 | 数据 |
|---|---|
| 体重 | [X]kg（较上周+/-[X]kg）|
| 腹围 | [X]cm |
| 胎动 | [有/没有，开始于XX周] |
| 身体感受 | [如：孕吐减轻，耻骨有点疼] |

### 📝 本周发生的事
[记录1-2件本周发生的重要事件，如产检、胎动体验、身体变化]

### 💭 本周心情
[真实地记录情绪，可以是焦虑、期待、幸福或困惑]

### 👶 宝宝现在长这样
[根据孕周描述胎儿发育]
- 身长：[X]cm（像[水果大小]）
- 体重：[X]g
- 本周发育重点：[胎儿本周的变化]

### 📋 下周计划
- [ ] [产检项目]
- [ ] [需要准备的东西]
- [ ] [想对宝宝说的话]

### 💬 评论区话题
你们现在是孕几周？最期待的事情是什么？
```

## 二、顺产日记模板（爆款）

### 标题公式
```
悬念型：[进产房前 X 小时]，我不知道会这样……
故事型：我以为生孩子是[错误认知]，没想到[真实情况]
数字型：[时间]，[X]斤，顺产全过程记录
痛点型：产房里发生的那些事，没人提前告诉我
```

### 正文结构模板

```markdown
## 我是如何顺产出[X]斤宝宝的｜完整记录

### 前情
[交代背景：孕周、有无特殊状况、为什么选择顺产/剖宫产]

### 第一产程：宫口扩张
[从见红/破水/规律宫缩开始记录]
- 时间节点：[几点几分] 发生了什么
- 痛感描述：[用比喻：像痛经的X倍 / 像被卡车碾过 等，写得真实]
- 应对方式：[呼吸法、无痛针、瑜伽球等]
- 情绪：[害怕/崩溃/坚持]

### 第二产程：胎儿娩出
[从宫口开全到宝宝出生]
- [用力和呼吸的配合]
- [侧切/撕裂情况，有无]
- [宝宝出来的瞬间感受]
- [宝宝出生时间、性别、体重]

### 第三产程：胎盘娩出
[轻描淡写，但要说清楚]

### 产后72小时
- 第一次下床：[什么感觉]
- 第一次喂奶：[开奶/含乳痛/宝宝吸吮]
- 住院期间最难受的事：[如实写]

### 我最后悔的事 / 最庆幸的事
[给读者2-3个实用建议]

### 写给准妈妈的话
[情感收尾，不煽情但真实]
```

## 三、辅食教程模板

### 标题公式
```
效果型：[X]个月宝宝抢着吃的辅食，我做了[X]次
对比型：超市[X]元 vs 自制[X]元，我选择自己在家做
清单型：辅食添加第一口，看这一篇就够了（[月龄]月龄）
```

### 正文结构模板

```markdown
## [X]月龄辅食｜[食物名称]这样做，宝宝爱吃还补[X]

### 适合月龄
[X]-[X]月龄，对[相关食材]不过敏的宝宝

### 营养价值
- [营养成分1，如高铁]
- [营养成分2，如维生素]
- 何时引入：[月龄]

### 准备食材
- [食材1] [X]g
- [食材2] [X]g
（食材越少越好，初期1-2种）

### 制作步骤
1. [步骤1，如清洗、浸泡]
2. [步骤2，如去皮、切块]
3. [步骤3，如蒸/煮]
4. [步骤4，如打泥/碾压]
5. [步骤5，如加米粉混合]

### 保存方法
- 冷藏：[X]天内吃完
- 冷冻：3天内吃完，[推荐用辅食格分装]

### 喂养Tips
- [实用技巧，如：从1勺开始尝试]
- [注意事项，如：第一次吃要观察3天过敏]

### 延伸做法
- [变体1]
- [变体2]
```

## 四、育儿避坑指南模板

### 标题公式
```
教训型：我最后悔的[3]件事，[宝宝X岁]才明白
数字型：[X]年带娃经验，告诉你这[6]件事不能做
人群型：[职场妈妈/新手爸爸/老人带娃]最容易踩的[X]个坑
```

```markdown
## [数字]年带娃，告诉你真正要避开的[数字]个坑

### 坑1：[具体坑的名称]
**❌ 我踩过的：**
[真实经历/错误做法]

**✅ 正确做法：**
[正确做法 + 原因]

### 坑2：...
### 坑3：...

### 写在最后
[情感升华 + 鼓励]
```

## 六、小红书图文脚本

### 封面公式
```
[数字]+[效果/痛点]：如"10款辅食|宝宝连吃3天不腻"
[对比]：自制 vs 买的，省了X元
[人群+场景]：职场妈妈|10分钟搞定宝宝早餐
[情绪词]：救命！这个辅食真的绝
```

### 封面文字排版建议
- 主标题：白色大字，黑色描边
- 副标题：浅色，字号小于主标题
- 背景：纯色或模糊生活照
- 品牌色统一：选择一个颜色作为全系列主色

### 正文结构（小红书图文）
```markdown
[开场图：成品图+文字"X款宝宝爱吃的辅食！"]

📌 适合月龄：[X]-[X]月龄
⏱️ 制作时间：[X]分钟
🧊 保存方式：冷藏[X]天/冷冻[X]天

【目录】
① [食物1]
② [食物2]
③ [食物3]
...

[每个食物的结构：]
### 🥦 [食物名称]
✅ 适合：[月龄]
🌟 营养：[营养价值]
📝 做法：[3-5步]

[结尾引导]
👉 关注我，收藏备用！
💬 评论区告诉我，你们家宝宝最爱吃什么辅食？
#宝宝辅食 #辅食添加 #[月龄]月龄辅食 #育儿干货 #辅食教程
```
