# A股分析方法论参考

## 基本面分析方法

### 财务指标体系

| 指标类别 | 核心指标 | 评判标准 |
|----------|----------|----------|
| **盈利能力** | 毛利率、净利率、ROE | 持续高于行业平均 |
| **成长能力** | 营收增速、净利润增速 | 连续3年>15% |
| **偿债能力** | 资产负债率、流动比率 | 行业合理区间 |
| **运营效率** | 存货周转、应收账款周转 | 趋势向好 |

### 估值方法

1. **相对估值法**
   - PE估值：适用于成熟稳定行业
   - PB估值：适用于金融、周期行业
   - PS估值：适用于成长初期企业

2. **绝对估值法**
   - DCF模型：现金流折现
   - 股息贴现模型：高分红企业

### 估值分位数标准

| 分位数区间 | 估值状态 |
|------------|----------|
| <30% | 相对低估 |
| 30%-70% | 合理区间 |
| >70% | 相对高估 |
| >90% | 泡沫区间 |

## 技术面分析方法

### 趋势判断标准

| 趋势类型 | 判断条件 |
|----------|----------|
| **上升趋势** | 股价>MA20>MA60，MA20向上 |
| **下降趋势** | 股价<MA20<MA60，MA20向下 |
| **震荡趋势** | 股价在均线间反复穿越 |

### MACD指标信号

| 信号类型 | 判断标准 |
|----------|----------|
| **金叉** | DIF上穿DEA，红柱出现 |
| **死叉** | DIF下穿DEA，绿柱出现 |
| **顶背离** | 价格新高，MACD未新高 |
| **底背离** | 价格新低，MACD未新低 |

### KDJ指标信号

| 信号类型 | 判断标准 |
|----------|----------|
| **超买** | KDJ>80 |
| **超卖** | KDJ<20 |
| **金叉** | K线上穿D线 |
| **死叉** | K线下穿D线 |

## 资金面分析方法

### 主力资金流向解读

| 净流入率 | 资金信号 |
|----------|----------|
| >3% | 强势买入 |
| 1%-3% | 温和买入 |
| -1%-1% | 观望 |
| -3%~-1% | 温和卖出 |
| <-3% | 强势卖出 |

### 北向资金分析

1. **持仓变化**：反映外资对个股看法
2. **持股比例**：与流通股占比对比
3. **持续性**：短期异动vs持续流入

## 综合评分体系

### 评分维度与权重

| 维度 | 权重 | 评估要点 |
|------|------|----------|
| **基本面** | 40% | 业绩稳定性、成长性、估值合理性 |
| **技术面** | 30% | 趋势方向、量价配合、技术位置 |
| **资金面** | 30% | 主力动向、外资持仓、市场情绪 |

### 评分标准

| 得分区间 | 投资建议 |
|----------|----------|
| 80-100 | 强烈推荐买入 |
| 60-80 | 建议持有/适度买入 |
| 40-60 | 建议观望 |
| 20-40 | 建议减仓 |
| 0-20 | 建议回避 |

## 风险识别框架

### 财务风险信号

| 风险类型 | 预警信号 |
|----------|----------|
| **业绩变脸** | 连续2季度增速下滑 |
| **资金压力** | 经营现金流持续为负 |
| **存货积压** | 存货周转天数持续上升 |
| **应收账款风险** | 应收账款增速远高于营收 |

### 技术风险信号

| 风险类型 | 预警信号 |
|----------|----------|
| **均线死叉** | MA5下穿MA20 |
| **破位下跌** | 跌破重要支撑位 |
| **量价背离** | 上涨缩量、下跌放量 |
| **MACD顶背离** | 高位反复背离 |

### 舆情风险信号

| 风险类型 | 预警信号 |
|----------|----------|
| **监管问询** | 收到交易所问询函 |
| **诉讼风险** | 重大诉讼未决 |
| **产品质量** | 食品安全/产品安全事故 |
| **业绩造假嫌疑** | 财务数据异常、毛利率异常 |
