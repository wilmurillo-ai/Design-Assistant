# TeamArchitect - 智能团队架构师

> 通过自然语言对话设计多智能体团队架构，并推荐安装专业技能

## 功能特点

- **需求分析**: 通过多轮对话深入了解项目需求
- **架构设计**: 智能匹配 12 种 Agent 角色（分析/执行/审查/协调/创作/陪伴/办公/技术/研究/运营/设计/测试）
- **技能推荐**: 根据 Agent 角色动态推荐对应技能
- **安全安装**: 核心技能自动安装，非核心技能可选安装

## 支持的 Agent 类型

| 类型 | 核心技能 | 应用场景 |
|-----|---------|---------|
| 分析类 | 数据分析、信息抓取、摘要提炼 | 调研报告、数据分析 |
| 执行类 | 代码编写、文件处理、API 调用 | 具体任务执行 |
| 审查类 | 代码审查、安全扫描、质量检查 | 质量把控 |
| 协调类 | 任务分发、上下文管理、进度追踪 | 多 Agent 协作 |
| 创作类 | 文案撰写、内容优化、排版美化 | 内容创作 |
| 陪伴类 | 对话交流、情感分析、记忆管理 | 聊天陪伴 |
| 办公类 | Excel、PPT、Word、PDF 处理 | 文员办公 |
| 技术类 | 前后端开发、数据库、运维部署 | 技术开发 |
| 研究类 | 深度调研、文献分析 | 学术研究 |
| 运营类 | 多平台发布、粉丝互动 | 自媒体运营 |
| 设计类 | UI/UX、视觉设计、图片处理 | 界面设计 |
| 测试类 | 自动化测试、Bug 检测 | 质量保证 |

## 安装方法

### Claude Code
```bash
# 克隆到本地技能目录
git clone https://github.com/MarsHong-86/teamarchitect-skill.git ~/.claude/skills/teamarchitect
```

### OpenClaw
```bash
# 使用 openclaw 安装
openclaw skills install github.com/MarsHong-86/teamarchitect-skill
```

## 使用方法

启动技能：
```
/teamarchitect
```

然后按照引导描述你的项目需求，技能会自动：
1. 分析你的需求
2. 设计适合的团队架构
3. 推荐对应的专业技能
4. 协助安装和配置

