# agent-web-cpu

**transweb.cn AI Agent 智能执行套件**

用户只需说需求，自动完成：匹配/创建应用 → 浏览器打开 → 填入 → 生成 → 提取结果 → 保存 Markdown。

## 能做什么

- **单应用执行** — 说一句话，自动找到或创建合适的 AI Agent 应用并执行
- **多应用串联** — 复杂任务自动拆分为多步流水线，前一步输出自动传给下一步
- **添加应用** — 通过 transweb.cn 应用 ID 注册新应用到列表
- **移除应用** — 通过名称从注册列表中移除应用
- **结果保存** — 生成内容自动保存为 `.md` 文件到 `~/Downloads/`

## 使用示例

```
# 单应用执行
"帮我写一篇关于量子计算的博客"

# 串联执行
"先用博文框架生成大纲，再用爆款润色助手润色"

# 管理应用
"添加应用 id=abc123def456..."
"移除应用 爆款润色助手"
"已有哪些应用"
"应用详情 博文框架"
```

## 五种模式

| 模式 | 触发 | 说明 |
|------|------|------|
| **single** | 单一需求 | 匹配一个应用执行 |
| **pipeline** | 多步骤/显式指定 | 串联多个应用逐步传递 |
| **list** | `已有哪些应用` | 展示应用列表 |
| **add** | `添加应用 id=xxx` | 通过 ID 注册新应用 |
| **remove** | `移除应用 xxx` | 通过名称移除应用 |

- OpenClaw `browser` 工具已启用（`browser.enabled: true`）
- `browser.evaluateEnabled: true`（结果提取需要）
- AI 执行细节见 [SKILL.md](SKILL.md)
