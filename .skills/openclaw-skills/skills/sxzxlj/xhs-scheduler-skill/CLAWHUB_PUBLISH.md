## 🚀 发布步骤

### 1. 初始化 ClawHub 配置

```bash
# 安装 ClawHub CLI
npm install -g clawhub

# 登录 ClawHub
clawhub login

# 初始化项目配置
clawhub init
```

## 📝 发布文案

### 标题
小红书智能排期发布器 - 让内容发布自动化

### 简介
告别手动发布！一键管理内容队列，智能推荐最佳发布时间，自动完成小红书笔记发布。

### 详细描述

**核心功能：**

📅 **内容队列管理**
- 批量添加待发布内容
- 支持图文和视频
- 灵活调整发布顺序

🎯 **智能排期推荐**
- 基于粉丝活跃时间分析
- 自动推荐最佳发布时段
- 提高内容曝光率

⏰ **定时自动发布**
- 按计划自动执行
- 支持多账号管理
- 无需人工值守

📊 **数据追踪**
- 实时查看发布状态
- 历史记录查询
- 发布成功率统计

🔔 **多渠道通知**
- 飞书/钉钉/Telegram
- 发布成功/失败提醒
- 每日统计报告

**适用场景：**
- 小红书个人博主
- MCN机构运营
- 品牌内容团队
- 自媒体创作者

**定价：**
- 免费版：5篇/天
- 基础版：$29/月（20篇/天）
- 专业版：$49/月（无限篇）

需要配合 xiaohongshu-mcp 服务使用
