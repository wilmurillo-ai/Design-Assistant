# Security Policy

## Reporting Vulnerabilities

If you discover a security vulnerability, please report it responsibly:

1. **Do NOT** open a public issue
2. Email: bcwilson@bcwilson.com
3. Include: description, reproduction steps, impact assessment

## Scope

This skill contains:
- Markdown documentation (SKILL.md, references, docs)
- A Bash validation script (scripts/validate.sh)
- No runtime dependencies
- No data storage

The skill itself does not handle credentials — it documents how to configure them via environment variables or secret managers.
