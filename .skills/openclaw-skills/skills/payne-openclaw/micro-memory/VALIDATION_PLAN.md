# Micro Memory V3.0 有效性验证计划

## 验证目标
验证语义关联和间隔复习功能是否真正提升记忆保留效率。

---

## 📊 验证指标

### 1. 间隔复习有效性
| 指标 | 测量方法 | 目标 |
|------|----------|------|
| 复习完成率 | 完成复习数 / 到期复习数 | ≥80% |
| 记忆保留率 | 30 天后仍为 Strong/Stable 的记忆比例 | ≥70% |
| 衰退速度 | 未复习记忆从 Stable→Decaying 的平均天数 | >7 天 |

### 2. 语义关联有效性
| 指标 | 测量方法 | 目标 |
|------|----------|------|
| 推荐准确率 | 接受的推荐数 / 总推荐数 | ≥60% |
| 检索效率提升 | 使用关联检索 vs 关键词检索的时间比 | <50% |
| 网络密度 | 有链接的记忆 / 总记忆数 | ≥40% |

### 3. 整体系统健康
| 指标 | 测量方法 | 目标 |
|------|----------|------|
| 健康分数 | `memory health` 得分 | ≥85/100 |
| 孤立记忆比例 | 无关联的记忆 / 总记忆数 | ≤30% |
| 用户满意度 | 主观评分 (1-5) | ≥4 |

## 🔧 验证命令

```powershell
# 基线快照
memory health > validation/baseline-health.txt
memory list --show-strength > validation/baseline-strength.txt
memory analyze > validation/baseline-network.txt

# 每周检查
memory health
memory strength
memory review --today

# 关联分析
memory suggest-links
memory analyze

# 复习统计
memory consolidate
```

## ⚠️ 注意事项

1. **数据备份**: 每周备份 `store/` 目录
2. **不要删除测试数据**: 保留完整的验证数据集
3. **记录主观感受**: 功能是否真的帮助记忆？
4. **调整阈值**: 根据实际效果调整 Jaccard 阈值 (当前 0.5)
5. **间隔优化**: 如果保留率低，考虑缩短间隔
