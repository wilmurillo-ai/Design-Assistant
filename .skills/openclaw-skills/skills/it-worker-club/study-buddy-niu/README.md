# Study Buddy 📚

**AI 驱动的个性化备考助手** - 让学习像聊天一样简单！

> OPC 极限挑战赛参赛作品 | 作者：牛宝华

---

## ✨ 核心功能

### 🎯 智能出题
- **日语 N2**：语法、词汇、阅读理解全覆盖
- **软考架构师**：历年真题、考点精准推送
- **难度分级**：简单/中等/困难，循序渐进
- **批量模式**：一次 5 题/10 题，高效刷题

### ⚡ 秒级判分
- 提交答案后立即解析
- 详细语法说明 + 例句对比
- 错误选项逐一分析

### 📝 智能错题本
- 自动收录答错题目
- 定期推送复习提醒
- 薄弱知识点针对性强化

### 📅 学习计划生成
- 根据考试日期倒推每日任务
- 科学分配各科目的复习时间
- 动态调整计划进度

## 🛠️ 技术架构

- **运行环境**: OpenClaw Skill Framework
- **数据存储**: 飞书 Bitable 云数据库
- **对话平台**: 飞书原生对话体验
- **核心引擎**: 
  - `question-engine.js` - 智能出题引擎
  - `bitable-client.js` - 云数据库客户端
  - `schedule-gen.js` - 学习计划生成器
  - `wrong-answers.js` - 错题本管理
  - `progress-tracker.js` - 进度追踪
