## 🔧 技术实现

### 核心改动

**1. 批量出题**
```javascript
// 一次获取多道题目
const questions = await getRandomQuestions(config, filters, count);

// 保存到会话
answerSessions.set(userId, {
  questions: [...],  // 多道题目
  answers: {},       // 收集用户答案
  mode: 'batch'      // 批量模式
});
```

**2. 答案收集**
```javascript
// 支持多种格式
parseBatchAnswers("1.A 2.B 3.C")  // → {0:'A', 1:'B', 2:'C'}
parseBatchAnswers("A,B,C")        // → {0:'A', 1:'B', 2:'C'}
parseBatchAnswers("1A 2B 3C")     // → {0:'A', 1:'B', 2:'C'}
```

**3. 统一判分**
```javascript
// 全部答完后，一次性判分
session.questions.forEach(q => {
  const userAnswer = session.answers[q.id];
  if (userAnswer === q.answer) correctCount++;
});
```

**4. 成绩报告**
```javascript
// 显示整体正确率 + 错题解析
formatBatchResultMessage(session, correctCount, accuracy, wrongQuestions);
```
