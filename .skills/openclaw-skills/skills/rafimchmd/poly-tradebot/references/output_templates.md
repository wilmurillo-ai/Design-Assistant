## File Naming Convention

- **Geopolitics:** `memory/geopolitics-YYYY-MM-DD-cnbc-<topic>.md`
- **Macroeconomics:** `memory/macro-YYYY-MM-DD-cnbc-<topic>.md`

Example:
- `memory/geopolitics-2026-03-17-cnbc-uae-energy-infrastructure.md`
- `memory/macro-2026-03-17-cnbc-treasury-yields-fed.md`
