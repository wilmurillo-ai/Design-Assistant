# Official Sources

Open these pages at the start of every run:

- `https://docs.openclaw.ai/gateway/configuration`
- `https://docs.openclaw.ai/gateway/configuration-reference`
- `https://docs.openclaw.ai/cli/config`
- `https://docs.openclaw.ai/cli/doctor`

Use them to confirm:

- config file location and JSON5 behavior
- allowed keys and enum values
- validation and repair commands
- whether a proposed fix is documented

Prefer CLI evidence and these docs over memory. If the docs and the local CLI disagree, report the mismatch and stay conservative.
