# 党建党政季度汇总分析 skill

## 功能

自动收集最近一个季度（三个月）习近平总书记重要讲话、重要文章、中央重要会议精神，汇总整理成清晰的分析报告，要求：
1. **依赖：** 需要系统安装 `summarize` CLI 工具（skillhub: summarize-1.0.0），用于提取总结《求是》文章核心内容
2. 使用 summarize 对《求是》杂志发表的总书记重要文章进行内容总结
3. 所有重要讲话、文章、会议都必须标注**来源链接**，保证有依据
4. 突出总书记原文引用和核心论述
5. 最后给出核心风向分析
6. 格式严格遵循以下模板：

---
