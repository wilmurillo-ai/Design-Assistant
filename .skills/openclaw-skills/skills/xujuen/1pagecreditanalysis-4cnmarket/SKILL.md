### **信评一页纸：结构化分析与撰写框架**

#### **一、 核心理念**

1.  **结论驱动：** 所有内容必须服务于最终信用结论。
2.  **证据支撑：** 定性结论必须有定量数据或具体事实支撑。
3.  **聚焦关键：** 仅分析最相关的核心驱动因素和重大风险。
4.  **动态分析：** 结合历史趋势、行业周期及未来风险事件进行分析。
5.  **标准化输出：** 使用统一的评级符号和风险等级关键词。
6.  **数据优先：** 优先使用 iFinD 官方数据作为分析素材，确保数据权威性与可追溯性。

---

#### **二、 文档模板**

**文档标题：** `[发行人全称] ([代码]) — 信用评估摘要 | 分析日期：YYYY-MM-DD`

---

**核心观点**

*   **信用观点：** 客观描述发行人的核心信用特征。
    *   示例：`"区域性公用事业运营商，收入稳定性较强，但盈利能力受上游成本波动影响显著。"`
*   **信用支撑点：** 1-2句话概括主要支撑点（尽量量化）。
    *   示例：`"区域专营地位稳固，近三年经营现金流/EBITDA维持在85%以上。"`
*   **核心风险提示：** 1-2句话指出关键风险（包含量化指标）。
    *   示例：`"债务期限结构存在一定错配，未来12个月到期债务占总债务45%，货币资金/短期债务为0.6x。"`
*   **未来12个月展望：** `稳定` / `正面` / `负面`。

---

**发行人情况**

*   **业务定位：** 一句话定义其在产业链中的角色。
    *   示例：`"华东地区领先的城市燃气运营商。"`
*   **基本情况：**
    *   成立时间与注册资本
    *   主要业务范围与板块贡献
    *   地理覆盖与运营规模
    *   行业地位与竞争格局

---

**股东背景与公司治理**

*   **股权结构：** 明确至最终控制方。
    *   示例：`"省国资委（持股60%）。"`
*   **股东支持：** 对国企/平台分析财政实力、历史支持记录、战略定位；对民企分析实控人经营状况、股权质押情况。
*   **公司治理：** 决策机制、财务透明度、过往信用记录。

---

**区域经济与财政实力（城投公司适用）**

*   **区域经济基本面：**
    *   区域定位与行政层级
    *   经济规模与增速（GDP、固定资产投资）
    *   产业结构与多元化程度
*   **财政实力与债务压力：**
    | 指标 | 数值 | 信用含义 |
    | :--- | :--- | :--- |
    | 一般公共预算收入 | XXX亿元 | 财政收入规模与地位 |
    | 财政收入增速 | X.X% | 收入可持续性 |
    | 财政自给率 | X.X% | 对转移支付依赖度 |
    | 政府性基金收入 | XXX亿元 | 土地财政依赖度 |
    | 政府债务率 | X.X% | 债务覆盖能力 |
    | 显性债务余额 | XXX亿元 | 债务累积速度 |
*   **区域信用环境：** 历史违约记录、区域战略地位、融资环境。

---

**主体信用深度分析**

*分析要点：根据发行人特点，选择2-3个核心维度深度分析，其他维度简要概述。*

**A. 主营业务分析**
*   行业周期与监管环境
*   商业模式与收入稳定性
*   市场地位与竞争优势
*   业务风险

**B. 资产质量分析**
*   资产结构与主要构成
*   资产流动性与变现能力
*   关联方交易与资产占用

**C. 负债与偿债能力分析**
*   债务结构与期限分布
*   杠杆水平（资产负债率、净债务/EBITDA）
*   短期偿债能力（货币资金/短期债务）
*   债务到期压力

**D. 盈利能力分析**
*   核心盈利指标（毛利率、净利率、EBITDA）
*   盈利驱动因素
*   利润质量评估

**E. 现金流分析**
*   经营现金流生成能力
*   营运资本管理效率
*   投资与筹资匹配性

---

**债券市场表现**

*   **存量债券概况：**
    | 项目 | 数值 |
    | :--- | :--- |
    | 存量债券只数 | XX只 |
    | 存量债券余额 | XX亿元 |
    | 未来12个月到期余额 | XX亿元 |
    | 当前主体评级 | AAA/AA+/AA |

*   **一级市场表现：**
    | 项目 | 数值 |
    | :--- | :--- |
    | 近12个月发行次数 | XX次 |
    | 平均发行利率 | X.XX% |
    | 平均认购倍数 | X.Xx |

*   **二级市场表现：**
    | 项目 | 数值 |
    | :--- | :--- |
    | 当前估值收益率 | X.XX% |
    | 信用利差 | XXbp |
    | 日均成交量 | XX万元 |

---

**关键风险与跟踪事项**

*   **重大风险事件：** 列出已知重大不确定性。
*   **预警指标跟踪：**
    | 指标类别 | 具体指标 | 预警阈值 | 风险等级 |
    | :--- | :--- | :--- | :--- |
    | 财务指标 | 净债务/EBITDA | 6.0x | 高/中/低 |
    | 财务指标 | 货币资金/短期债务 | 0.5x | 高/中/低 |
    | 融资指标 | 银行授信使用率 | 95% | 高/中/低 |
    | 市场指标 | 信用利差变化 | +50bp | 高/中/低 |

*   **下次评审触发点：** 具体化，如：发布年度财报后 / 触发预警指标时。

---

#### **三、 撰写流程**

1.  **数据提取：** 收集财报、募集说明书、行业数据，计算关键比率。
2.  **框架选择：** 判定城投/非城投，选择相应分析框架。
3.  **核心维度识别：** 根据发行人特点识别2-3个重点分析维度。
4.  **框架填充：** 将数据和分析填入模板各部分。
5.  **结论提炼：** 逆向撰写"核心观点"，确保结论有依据。
6.  **精炼输出：** 删除重复和次要信息，确保逻辑完整。

---

**最终产出是一份高度结构化、信息密度高、结论明确、重点突出的信用评估摘要。**
