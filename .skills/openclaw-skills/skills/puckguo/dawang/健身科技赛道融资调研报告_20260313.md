## 二、死亡/失败案例分析（近6个月）

### 2.1 Lululemon Mirror - 最惨烈的失败

| 项目 | 详情 |
|------|------|
| **收购价** | 5亿美元（2020年） |
| **关闭时间** | 2023年10月 |
| **损失** | 5亿美元收购价 + 运营投入 |
| **失败原因** | 1. 居家健身需求退潮<br>2. 内容订阅转化率低<br>3. 硬件成本高毛利低 |
| **教训** | 纯C端健身镜模式不可持续 |

### 2.2 FITURE裁员调整

| 项目 | 详情 |
|------|------|
| **累计融资** | 4亿美元+ |
| **裁员时间** | 2022年7月、2023年持续 |
| **调整方向** | 从C端转向B端（酒店、健身房） |
| **现状** | 艰难维持，估值大幅缩水 |

### 2.3 Tonal估值暴跌

| 项目 | 详情 |
|------|------|
| **高点估值** | 19亿美元 |
| **2023年估值** | 下调10亿美元 |
| **原因** | 销售不及预期，烧钱过快 |
| **现状** | 新任CEO制定EBITDA盈利计划 |

### 2.4 风险总结

```
健身科技赛道核心风险：
├── 需求风险：居家健身是疫情特殊需求，非常态
├── 模式风险：硬件+订阅模式获客成本高、留存率低
├── 竞争风险：传统健身房回归，免费内容冲击
├── 成本风险：硬件研发、供应链、库存压力大
└── 团队风险：硬件+软件+内容，需要复合团队
```

## 三、单人创始人项目投资条款惯例

### 3.1 分期放款结构（Milestone-Based）

**典型三期结构**:

| 期数 | 比例 | 放款条件 | 时间 |
|------|------|----------|------|
| **首期** | 30-40% | 签约即放，完成公司注册 | T+0 |
| **二期** | 30-40% | 核心团队到位（CTO/硬件负责人） | T+3~6月 |
| **三期** | 20-40% | 产品原型验证/首批客户签约 | T+9~12月 |

**郭总条款建议**:
- 首期争取40%（500-600万中的200-240万）
- 二期条件：CTO到位 + 硬件原型机完成
- 三期条件：3家健身房POC签约

### 3.2 团队到位作为放款条件

**常见条款表述**:

```
"第二期投资款（人民币XXX万元）的支付以下列条件
先决条件为准：
1. 公司已与CTO（或硬件负责人）签署正式劳动合同
2. 该核心成员已全职入职并缴纳社保
3. 创始人确认该成员具备完成产品原型的能力"
```

**郭总应对策略**:
- 提前与意向CTO签署意向书
- 明确"到位"定义（全职入职 vs 签署合同）
- 争取"关键人到位"而非"完整团队"

### 3.3 对赌/估值调整机制（Ratchet）

**常见对赌指标**:

| 类型 | 指标 | 调整方式 |
|------|------|----------|
| **团队对赌** | 核心成员未到位 | 估值下调20-30% |
| **里程碑对赌** | 产品延期 | 估值下调10-20% |
| **业绩对赌** | 营收/用户未达标 | 股权补偿或估值下调 |

**郭总条款建议**:
- 接受团队对赌，但限制下调幅度（不超过20%）
- 拒绝业绩对赌（早期硬件项目不确定性高）
- 要求对赌条款双向（超预期时估值上调）

### 3.4 董事席位分配

**单人创始人项目典型结构**:

| 席位 | 代表 | 投票权 |
|------|------|--------|
| 董事1 | 创始人 | 1票 |
| 董事2 | 投资人 | 1票 |
| 董事3 | 独立董事/双方认可 | 1票 |

**关键条款**:
- 重大事项需2/3以上董事同意
- 重大事项包括：融资、并购、预算超支、核心人员变动
- 创始人保留日常经营决策权

**郭总建议**:
- 争取3人董事会（创始人+投资人+独立董事）
- 明确"重大事项"清单，避免过度干预
- 保留产品、技术、运营日常决策权

### 3.5 其他风险控制条款

| 条款 | 目的 | 郭总应对 |
|------|------|----------|
| **Key-Man条款** | 创始人离职触发回购 | 限制触发条件（非因过错离职除外） |
| **竞业限制** | 3-5年不得从事同类业务 | 限制地域和业务范围，给予补偿 |
| **拖售权** | 投资人可强制出售公司 | 设置估值底线（不低于本轮估值） |
| **领售权** | 大股东可强制小股东一起出售 | 设置估值底线和时间限制 |
| **优先清算权** | 投资人优先拿回投资款 | 争取1倍非参与式优先 |

## 五、调研前质疑点回应

### 5.1 健身科技赛道是否还有投资价值？

**质疑**: 2023年后估值大幅回调，Keep上市后表现一般

**回应**:
- ✅ 赛道整体回调，但B端健身房场景仍有空间
- ✅ 配件模式比整机模式更轻、风险更低
- ✅ AI+硬件是趋势，pukup定位准确

### 5.2 单人创始人是否真能做硬件？

**质疑**: 硬件创业需要供应链、结构、电子、软件多维度能力

**回应**:
- ⚠️ 确实是核心风险，需要快速补强团队
- ✅ 建议：首期40%，CTO到位后放二期
- ✅ 利用9年医疗器械行业经验（供应链、合规）

### 5.3 pukup的差异化是否足够？

**质疑**: 现有案例已覆盖智能健身场景，配件定位是否有市场？

**回应**:
- ✅ 配件模式降低客户决策成本（vs 整机）
- ✅ 存量健身房改造市场 > 新建智能健身房
- ⚠️ 需验证：健身房付费意愿、安装复杂度

## 六、结论与建议

### 6.1 核心结论

1. **赛道**: 健身科技整体回调，但B端+AI方向仍有投资机会
2. **估值**: 4000万估值在合理区间（对标速境Pre-A轮）
3. **条款**: 分期放款+团队对赌是行业惯例，可接受但需限制幅度
4. **风险**: 单人团队是硬伤，需在term sheet签署前明确补强计划

### 6.2 谈判建议

| 条款 | 建议 |
|------|------|
| **金额** | 500-600万可接受，争取首期40% |
| **估值** | 4000万合理，不接受团队对赌下调超过20% |
| **分期** | 三期：签约40% + CTO到位30% + POC完成30% |
| **对赌** | 接受团队对赌，拒绝业绩对赌 |
| **董事** | 3人董事会，创始人+投资人+独立董事 |
| **退出** | 接受拖售权，设置估值底线（不低于本轮） |
