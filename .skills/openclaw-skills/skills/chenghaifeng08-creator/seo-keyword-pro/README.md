# 🔍 SEO Keyword Pro

AI 驱动的 SEO 关键词研究工具 - 发现低竞争高流量关键词，生成内容大纲

---

## 🚀 快速开始

```bash
# 安装技能
clawhub install seo-keyword-pro

# 关键词研究
/seo keywords --query "best crypto trading bot" --country US

# 生成内容大纲
/seo outline --keyword "crypto trading bot" --intent informational
```
