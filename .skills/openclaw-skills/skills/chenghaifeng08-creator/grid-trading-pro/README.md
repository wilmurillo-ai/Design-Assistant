专业网格交易策略 - 高级参数优化、多币种管理、收益最大化

## 🚀 快速开始

```bash
# 安装技能
clawhub install grid-trading-pro

# 创建网格
/grid create --symbol BTCUSDT --upper 70000 --lower 60000 --grids 20

# 查看收益
/grid profit --symbol BTCUSDT
```
