# 封面图生成 Prompt 模板

## 基础模板

```
生成一张 16:9 的技术文章封面图
主题：{文章标题}
风格：简约科技感，蓝紫色调
元素：AI、写作、自动化相关图标
配色：深蓝 (#1a1a2e) + 紫色 (#6e44ff) + 青色 (#00d9ff)
文字：{文章标题关键词}（可选）
尺寸：1080×864
```

## 注意事项

### 文字处理

- **原则**：少即是多
- **建议**：封面图上文字不超过 10 个字
- **原因**：小尺寸预览时文字会看不清
- **替代**：重要的文字放到文章标题和开头

### 元素选择

- ✅ 简洁的几何图形
- ✅ 抽象的概念表达
- ✅ 与主题强相关的图标
- ❌ 复杂的细节（小尺寸看不清）
- ❌ 过多的元素（显得杂乱）

### 配色建议

- **主色**：1-2 种，占 70%
- **辅色**：1-2 种，占 25%
- **强调色**：1 种，占 5%
- **避免**：超过 5 种颜色

### 尺寸要求

- **推荐**：1080×864（2.35:1，公众号封面）
- **最小**：900×383（公众号要求）
- **格式**：JPG 或 PNG
- **大小**：< 5MB
