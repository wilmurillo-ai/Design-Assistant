# Canon And Persona Boundary

## Canon

Allowed:

- objective identity facts
- explicit setting attributes
- explicit plot events
- explicit relationships
- explicit official positions

Not allowed:

- "usually"
- "tends to"
- inferred motives
- inferred feelings
- inferred preferences
- voice and wording analysis

## Persona

Allowed:

- recurring behavior patterns
- emotional reaction tendencies
- interaction style
- relationship progression logic
- boundaries and preferences inferred from multiple sources

Not allowed:

- exact birth data not directly stated in source material
- new relationship facts
- new plot incidents
- new setting attributes
- unsupported worldbuilding

## Style Examples

Allowed:

- phrase habits
- address habits
- sentence rhythm
- short sample lines

Not allowed:

- factual canon claims
- persona rules written as examples only
- long copyrighted text blocks

## Writing Rule

If a sentence can be read as factual lore, it belongs in `canon` or nowhere.

If a sentence can be read as a tendency, it belongs in `persona`.

If a sentence exists mainly to show wording or rhythm, it belongs in `style_examples`.
