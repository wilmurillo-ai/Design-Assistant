# Evidence Model

## Priority Order

Use this priority when sources conflict:

1. official material
2. quoted plot or dialogue excerpts
3. fandom wiki or community summary
4. user summary

## Evidence Labels

- `official`: explicit official statement, guidebook statement, profile card, or other official wording
- `quoted_text`: direct plot summary excerpt or direct dialogue excerpt
- `fandom_wiki`: community-maintained summary that still needs caution
- `user_summary`: user-provided synthesis, paraphrase, or memory

## Canon Admission Rule

`canon` is allowed to contain only content that is directly supported by the available source material.

Direct support means one of these:

- the source states the fact plainly
- the source shows the plot event plainly
- the source identifies the relationship plainly
- the source states the setting attribute plainly
- the source records an official position plainly

If the source only implies the idea, it does not automatically enter `canon`.

## Persona Admission Rule

`persona` may summarize patterns when they are supported by repeated scenes, dialogue, or explicit descriptions. These summaries must stay phrased as tendencies or patterns, not as new canon facts.

## Conflict Handling

When sources conflict:

- do not silently merge them
- keep the higher-priority source
- mark lower-priority disagreement as `needs_review`
- if two high-priority sources conflict, preserve the conflict explicitly instead of guessing

## Default Review Triggers

Flag a review when the text includes uncertainty markers such as:

- maybe
- perhaps
- rumored
- implied
- seems
- appears
- likely
