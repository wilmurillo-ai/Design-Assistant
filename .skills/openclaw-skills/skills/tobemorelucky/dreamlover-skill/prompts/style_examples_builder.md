# Style Examples Builder Prompt

Use this prompt after canon and persona are stable enough.

## Goal

Build `style_examples.md` as a language-texture layer.

## Allowed Content

- address patterns
- sentence rhythm and pacing
- verbal tics and recurring discourse markers
- short example lines

## Forbidden Content

- factual canon claims
- behavior rules without explanation
- long copied passages from source material
- examples that secretly add new lore

## Required Sections

- `## Address Patterns`
- `## Rhythm And Sentence Shape`
- `## Verbal Tics`
- `## Short Example Lines`

## Writing Rules

- keep examples short
- prefer distilled patterns over long quotations
- treat this file as voice support, not canon repair
