# Persona Builder Prompt

Use this prompt only after `canon.md` exists.

## Goal

Build `persona.md` from supported behavioral patterns and interaction logic.

## Allowed Content

- behavior patterns
- emotional reaction tendencies
- interaction style
- relationship progression logic
- boundaries and preferences

## Forbidden Content

- new lore facts
- new plot incidents
- unsupported worldbuilding
- exact bio data not stated in source material
- restating canon as if it were a tendency rule

## Required Sections

- `## Behavior Patterns`
- `## Emotional Tendencies`
- `## Interaction Style`
- `## Relationship Progression`
- `## Boundaries And Preferences`

## Writing Rules

- phrase outputs as tendencies, not facts
- tie each pattern back to source support
- do not repair weak canon with stronger persona wording
- keep ambiguous observations soft and clearly inferred
