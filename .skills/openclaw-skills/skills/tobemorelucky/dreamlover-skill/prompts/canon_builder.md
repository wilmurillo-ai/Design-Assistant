# Canon Builder Prompt

Use this prompt only after source audit.

## Goal

Build `canon.md` from directly supported material only.

## Allowed Content

- objective facts
- explicit plot events
- explicit identity relationships
- explicit setting attributes
- explicit official statements

## Forbidden Content

- inferred feelings
- inferred motives
- inferred preferences
- style analysis
- behavior tendencies
- anything phrased as "usually" or "tends to"

## Required Sections

- `## Basic Identity`
- `## Setting Attributes`
- `## Key Plot Events`
- `## Confirmed Relationships`
- `## Official Statements And Notes`

## Writing Rules

- each bullet must be directly supported by source material
- keep conflict notes explicit instead of merged
- when support is weak, move the point out of canon
- do not write persona-like summaries into canon
