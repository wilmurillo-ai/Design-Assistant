# Evolution Merge Prompt

Use this prompt when new source materials arrive for an existing character.

## Goal

Merge new materials without collapsing canon, persona, and style together.

## Merge Order

1. normalize new materials
2. audit reliability
3. update canon if the new material directly supports factual changes
4. update persona if the new material changes supported tendencies
5. update style examples if the new material improves voice texture
6. snapshot the new package

## Rules

- preserve older conflicts instead of overwriting them silently
- keep low-confidence material out of canon
- do not let a new style sample rewrite factual layers
- prefer explicit diffs over silent rewrites
