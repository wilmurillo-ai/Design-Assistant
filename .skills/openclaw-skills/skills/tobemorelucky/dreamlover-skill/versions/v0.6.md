# Version 0.6

This version shifts generated child skills toward OpenClaw-ready runtime packaging while preserving conditional memory gates.

## New Runtime Expectations

- child skills are authored for OpenClaw discovery from `./.agents/skills/{slug}/`
- child front matter includes `name`, `description`, and Python requirement metadata for memory scripts
- child skills keep the conditional memory flow: router, fetch, reply, router, commit, summarize
- child skills degrade to no-memory mode when `python3` is unavailable
