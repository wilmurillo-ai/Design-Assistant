# V0.1

This is the initial repository contract for `dreamlover-skill`.

## Scope

- text-only source handling
- layered output with `canon`, `persona`, and `style_examples`
- repository-level docs and prompt scaffolding
- deterministic helper tools for normalization, evidence indexing, style extraction, skill writing, and snapshotting

## Stability Promise

Until the next contract revision:

- the character package layout remains the same
- the recommended section headings remain the same
- the documented tool entrypoints remain the same
