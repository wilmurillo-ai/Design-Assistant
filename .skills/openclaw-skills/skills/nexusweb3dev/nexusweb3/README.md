# NexusWeb3 — OpenClaw Skill

Complete financial infrastructure for AI agents on Base mainnet.

10 protocols: wallet, identity, payments, yield, insurance, reputation, marketplace, cross-chain bridge, governance, and launchpad.

## Install

```
clawhub install nexusweb3
```

## Requirements

- `ETH_PRIVATE_KEY` environment variable (Base mainnet wallet)
- Small amount of ETH on Base for gas (< $0.01/tx)
- USDC on Base for protocol interactions
