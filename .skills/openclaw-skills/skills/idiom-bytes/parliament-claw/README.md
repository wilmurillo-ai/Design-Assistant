# parliament-game

Label Canadian parliamentary Q&A pairs and help train a civic AI that scores MP accountability.

**No setup required.** The skill auto-registers on first use — no API key, no sign-in.

## What it does

- Fetches real Q&A exchanges from Canada's House of Commons
- You read each exchange and decide: did the MP actually answer the question?
- Labels train models that score every MP's accountability
- ~2,500 Q&A pairs available, completable in ~2 hours

## Usage

Just tell your agent:

- "play the parliament game"
- "label parliament questions"
- "help label Q&A pairs"
- "contribute to canada central"
