# Polymarket Conditional Orders

Set up conditional and algorithmic orders on Polymarket. Automate your trading strategy.

## Features

- **Conditional Orders**: Trade when price triggers
- **OCO Orders**: One-cancels-other orders
- **Time-based Orders**: Execute at specific times

## Pricing

- **Price**: 0.001 USDT per API call
- **Payment**: Integrated via SkillPay.me

## Integration

This skill is integrated with SkillPay.me for automatic micropayments.
