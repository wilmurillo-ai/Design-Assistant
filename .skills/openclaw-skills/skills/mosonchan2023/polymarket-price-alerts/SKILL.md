# Polymarket Price Alerts

Set up price alerts for any market. Get notified when yes/no prices reach your target levels.

## Features

- **Target Prices**: Set yes/no price thresholds
- **Alert Delivery**: Receive notifications
- **Multiple Markets**: Track multiple markets

## Pricing

- **Price**: 0.001 USDT per API call
- **Payment**: Integrated via SkillPay.me

## Integration

This skill is integrated with SkillPay.me for automatic micropayments. Each call costs 0.001 USDT.
