# Polymarket Election Odds

Get current election prediction market odds for any candidate or race. Track political betting markets.

## Features

- **Candidate Odds**: Yes/No probabilities for candidates
- **Race Tracking**: Track senate, house, governor races
- **Historical Movement**: Track odds changes over time
