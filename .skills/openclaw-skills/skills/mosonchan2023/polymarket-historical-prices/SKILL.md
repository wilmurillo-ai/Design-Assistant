# Polymarket Historical Prices

Get historical price data and charts for any Polymarket prediction market. Analyze past performance.

## Features

- **OHLCV Data**: Open, High, Low, Close, Volume
- **Time Frames**: Daily, hourly, minute-by-minute
- **Trend Analysis**: Moving averages, patterns

## Pricing

- **Price**: 0.001 USDT per API call
- **Payment**: Integrated via SkillPay.me

## Integration

This skill is integrated with SkillPay.me for automatic micropayments.
