# Polymarket Liquidity Check

Check liquidity depth and spread for any Polymarket prediction market. Understand if you can enter/exit positions easily.

## Features

- **Liquidity Depth**: Total available liquidity
- **Bid-Ask Spread**: Current spread between yes/no prices
- **Slippage Estimation**: Estimate slippage for large orders

## Pricing

- **Price**: 0.001 USDT per API call
- **Payment**: Integrated via SkillPay.me

## Integration

This skill is integrated with SkillPay.me for automatic micropayments. Each call costs 0.001 USDT.
