# Screenshots Directory

此目录用于存放技能的截图文件。

## 需要的截图

1. **write-command.png** - write 命令执行示例
2. **analyze-output.png** - analyze 命令输出示例
3. **templates-list.png** - templates 命令列表示例

## 截图建议

- 使用终端截图展示命令输出
- 分辨率建议：1200x800 或更高
- 格式：PNG（透明背景更佳）
- 文件大小：< 500KB

## 生成截图

可以使用以下工具：
- Windows: Win + Shift + S
- macOS: Cmd + Shift + 4
- 终端：使用 `screenshot` 命令（如果可用）
