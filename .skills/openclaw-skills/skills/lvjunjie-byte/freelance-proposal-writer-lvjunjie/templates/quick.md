# 快速响应提案模板

简洁快速的投标提案，适合紧急项目。

## 结构

1. 直接开场
2. 核心能力
3. 快速交付承诺

## 模板内容

```
Hi {{clientName}},

I can help you with {{projectTitle}}.

✓ {{skill1}}
✓ {{skill2}}
✓ {{skill3}}

Ready to start immediately. Can deliver within {{timeline}}.

Let's chat!
{{yourName}}
{{contactInfo}}
```

## 变量说明

| 变量 | 说明 |
|------|------|
| {{clientName}} | 客户姓名 |
| {{projectTitle}} | 项目标题 |
| {{skill1-3}} | 核心技能（3 个） |
| {{timeline}} | 交付时间 |
| {{yourName}} | 你的姓名 |
| {{contactInfo}} | 联系方式 |

## 使用建议

- 保持极简（50-100 字）
- 适合预算<$500 的项目
- 强调即时可用性
- 快速响应是关键
- 适合简单明确的项目
- 用符号✓增加可读性

## 最佳场景

- 紧急项目
- 小额预算
- 简单任务
- 需要快速决策
- 竞争激烈的热门项目
