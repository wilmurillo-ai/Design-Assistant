# YOLO历史与版本

## 核心信息
- **YOLO**：You Only Look Once，实时目标检测算法
- **最新版本**：YOLO26系列（Ultralytics维护）
- **特点**：速度快，精度高，支持多任务

## 版本演进要点

### 关键版本里程碑
- **YOLOv1 (2015)**：开创性单阶段检测
- **YOLOv3 (2018)**：引入多尺度预测
- **YOLOv5 (2020)**：Ultralytics接手，易用性提升
- **YOLOv8 (2023)**：统一框架，支持多任务
- **YOLO26 (最新)**：端到端NMS-free，边缘优化

### 当前推荐版本
**YOLO26系列**：
- `yolo26n`：最快，最小
- `yolo26s`：平衡性能  
- `yolo26m`：中等精度
- `yolo26l`：高精度
- `yolo26x`：最高精度

## 许可说明
- **AGPL-3.0**：开源使用，要求项目开源
- **企业许可**：商业用途需要授权
- **重要**：使用前确认许可合规性

## 技术特点
1. **端到端推理**：无需NMS后处理
2. **多任务支持**：检测、分割、分类、姿态、跟踪
3. **边缘优化**：适合移动和嵌入式设备
4. **简单API**：易于使用和部署

## 使用建议
- **新项目**：直接使用YOLO26系列
- **学习研究**：参考YOLOv8文档和示例
- **生产部署**：确认许可要求，测试性能
