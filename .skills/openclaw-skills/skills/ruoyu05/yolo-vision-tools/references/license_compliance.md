# 许可合规指南

## 核心要求
**使用YOLO/SAM前必须确认许可合规性**

## 许可类型
- **YOLO**：AGPL-3.0（开源使用），企业许可（商业用途）
- **SAM3**：研究使用（需引用），商业授权（需Meta授权）

## 使用场景
### 允许（开源/研究）
- 学术研究（需引用）
- 个人非商业项目（需开源）
- 教育用途

### 需要授权（商业）
- 商业软件/服务
- SaaS/云服务
- 企业内部工具
- 盈利性项目

## 合规步骤
1. 确认用途：商业还是非商业
2. 选择许可：AGPL或企业许可
3. 遵守要求：开源项目或购买授权
4. 保留记录：许可文件和引用

## 重要提醒
- 首次使用必须提醒用户许可要求
- 商业用途必须获取正式授权
- 违反许可可能导致法律后果
