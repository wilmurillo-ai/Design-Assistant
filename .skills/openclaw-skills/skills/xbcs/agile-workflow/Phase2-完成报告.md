## 🎯 技术实现

### 后端技术栈

| 组件 | 技术 | 说明 |
|------|------|------|
| **框架** | Express | Web 服务器 |
| **实时** | Socket.IO | WebSocket 服务 |
| **数据** | 内存缓存 | 实时数据存储 |
| **采集** | 定时器 | 1 秒采集一次 |

### 前端技术栈

| 组件 | 技术 | 说明 |
|------|------|------|
| **框架** | 原生 JS | 零依赖 |
| **通信** | WebSocket | 实时数据推送 |
| **样式** | CSS3 | 响应式布局 |
| **图表** | CSS 进度条 | 轻量级可视化 |

### API 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/overview` | GET | 系统概览 |
| `/api/tasks` | GET | 任务列表 |
| `/api/tasks/:id` | GET | 任务详情 |
| `/api/agents` | GET | Agent 列表 |
| `/api/agents/:name` | GET | Agent 详情 |
| `/api/agents/:name/start` | POST | 启动 Agent |
| `/api/agents/:name/stop` | POST | 停止 Agent |
| `/api/alerts` | GET | 告警列表 |
| `/api/metrics` | GET | 性能指标 |

### WebSocket 事件

| 事件 | 方向 | 说明 |
|------|------|------|
| `connect` | 客户端→服务器 | 连接建立 |
| `disconnect` | 客户端→服务器 | 连接断开 |
| `init` | 服务器→客户端 | 初始数据 |
| `update` | 服务器→客户端 | 数据更新 |
| `ping/pong` | 双向 | 心跳检测 |

```bash

# 安装依赖
```
