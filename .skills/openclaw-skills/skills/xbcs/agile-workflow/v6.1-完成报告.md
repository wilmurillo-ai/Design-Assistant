## 🎯 核心功能

### LRU 缓存

```javascript
const cache = new LRUCache({
  maxSize: 1000,
  ttl: 300000 // 5 分钟
});

cache.set('key', 'value');
const value = cache.get('key');
```

### 多级缓存

```javascript
const multiCache = new MultiLevelCache({
  l1MaxSize: 100,
  l1Ttl: 60000,
  l2MaxSize: 1000,
  l2Ttl: 300000,
  fileCache: './cache'
});

const value = multiCache.get('key');
```

## 📦 使用说明

### 基本使用

```javascript
const { LRUCache } = require('./cache-manager.js');

const cache = new LRUCache({
  maxSize: 1000,
  ttl: 300000
});

// 设置缓存
cache.set('key', 'value');

// 获取缓存
const value = cache.get('key');

// 查看统计
const stats = cache.getStats();
console.log(`命中率：${stats.hitRate}`);
```

### CLI 使用

```bash
# 运行测试
node cache-manager.js test

# 查看统计
node cache-manager.js stats

# 性能基准测试
node cache-manager.js benchmark
```
