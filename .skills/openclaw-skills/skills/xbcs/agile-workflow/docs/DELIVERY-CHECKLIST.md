## 🔧 使用示例

### 快速开始
```javascript
const ConcurrentExecutor = require('./core/concurrent-executor-v2');

const executor = new ConcurrentExecutor({
  workspace: '/tmp/my-workspace',
  maxConcurrency: 3
});

// 提交任务
executor.submitTask('task-1', async (domainPath) => {
  return { result: 'Task 1' };
});

executor.submitTask('task-2', async (domainPath) => {
  return { result: 'Task 2' };
}, ['task-1']); // 依赖 task-1

// 执行
await executor.executeAll();

// 合并结果
const merged = executor.mergeResults(['task-1', 'task-2'], 'append');
```

## 🎓 知识转移

### 核心概念
1. **写入域隔离**: 每个 Agent 独占的写入空间
2. **DAG 依赖图**: 有向无环图管理任务依赖
3. **合并策略**: 预定义的结果整合规则
4. **并发执行器**: 整合所有模块的协调器

### 设计原则
1. 第一性原理：从本质出发分析问题
2. MECE 拆解：相互独立，完全穷尽
3. 零数据污染：安全优先
