### 创造性评分模型

**四维度模型**：
```
创造性 = 新颖性 (25%) + 复杂性 (25%) + 深度 (25%) + 原创性 (25%)
```

#### 1. 新颖性检测 (25%)

**通用检测**：
- 方案创新（技术/方法/工具）
- 思路创新（角度/逻辑/框架）
- 表达创新（结构/呈现/交互）

**任务类型配置**：
```javascript
{
  'code': ['新', '创新', '优化', '改进', '重构'],
  'document': ['新视角', '新方法', '新思路', '创新'],
  'design': ['创新', '独特', '新颖', '首创', '突破'],
  'analysis': ['发现', '洞察', '新发现', '意外']
}
```

#### 2. 复杂性检测 (25%)

**通用检测**：
- 系统复杂度（模块/依赖/层次）
- 逻辑复杂度（条件/分支/因果）
- 关系复杂度（实体/关联/影响）

#### 3. 深度检测 (25%)

**通用检测**：
- 分析深度（根因/本质/规律）
- 思考深度（抽象/概括/升华）
- 影响深度（短期/长期/全局）

#### 4. 原创性检测 (25%)

**通用检测**：
- 套路识别（模板/惯例/常规）
- 陈词滥调（空话/套话/废话）
- 创新点（独特/突破/首创）
