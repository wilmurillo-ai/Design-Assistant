# Demo Session

## 目标
演示两个用户通过网页控制台完成一次“发瓶子 → 收瓶子 → 回信”的完整闭环。

## 场景
- Alice：发件人
- Bob：收件人

## 步骤

1. 启动 relay：
   ```bash
   python3 scripts/relay_server.py --host 127.0.0.1 --port 8765
   ```

2. Alice 在浏览器 A 打开：
   `http://127.0.0.1:8765/`
   - user_id: `alice`
   - 昵称: `Alice`
   - 点击“保存并上线”

3. Bob 在浏览器 B 打开：
   `http://127.0.0.1:8765/`
   - user_id: `bob`
   - 昵称: `Bob`
   - 点击“保存并上线”

4. Alice 在“发送漂流瓶”中输入：
   `愿你在忙碌里也能遇到小确幸。`

5. Bob 在“收到的漂流瓶”里看到新瓶子，可：
   - 直接点“直接回信”
   - 或点“打开专属回信页”

6. Bob 回复：
   `谢谢你，也祝你今天顺顺利利。`

7. Alice 在“收到的回信”中看到 Bob 的回传；在“我发出的漂流瓶”中看到该条瓶子的回信状态变为已收到回信。
