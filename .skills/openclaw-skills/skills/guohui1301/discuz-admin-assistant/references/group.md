圈子管理
除了创建者可以在前台管理自己的圈子外，管理员也可以在后台对所有已有的圈子进行管理操作，可以对圈子的各项参数进行设置。
首先通过搜索条件找到需要进行管理的圈子，搜索条件包括圈子名称、圈子ID、圈子分类、成员数、主题数、回复数、创建时间、最后更新时间、创建者、创建者 UID。
