## 📋 技能介绍

**industry-compare** 是一个专业的行业对比分析 CLI 工具。

**核心功能：**
- 📊 多公司财务数据对比
- 📈 行业地位评估
- 🎯 竞争优势分析
- 💡 投资建议生成

---

## 🚀 快速开始

### 安装

```bash
clawhub install industry-compare
```

### 使用

**1. 多公司对比**
```bash
python scripts/compare.py --stocks 600519.SH,000858.SZ,002304.SZ
```

**2. 行业地位评估**
```bash
python scripts/compare.py ranking --stock 600519.SH --industry 白酒
```
