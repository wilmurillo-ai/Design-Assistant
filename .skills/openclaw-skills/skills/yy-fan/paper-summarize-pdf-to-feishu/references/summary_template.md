## 数据精度要求

### 1. 百分比格式
- ✅ 正确：`28.7%`、`+15.3%`、`-28.7%`
- ❌ 错误：`28.7000000001%`、`约 29%`

### 2. P 值格式
- ✅ 正确：`P < 0.001`、`P = 0.005`、`P = 0.37`
- ❌ 错误：`p<0.001`、`p 值小于 0.001`

### 3. 置信区间格式
- ✅ 正确：`95% CI [1.23, 4.56]`
- ❌ 错误：`CI: 1.23-4.56`

### 4. 样本量格式
- ✅ 正确：`n=691`、`2,069 人`
- ❌ 错误：`691 个样本`、`2069 个人`

### 5. 均值±标准差格式
- ✅ 正确：`47.6 ± 14.6 岁`、`3.14 ± 2.25 分钟`
- ❌ 错误：`47.6+/-14.6`、`mean=47.6, sd=14.6`

## 强制署名块要求

文档末尾**必须**包含以下署名块（不可省略）：

```markdown
## 📝 文档信息

| 项目 | 信息 |
|------|------|
| 撰写人 | <AI 名称>（<模型名称>） |
| 审核状态 | ⏳ 待审核 / ✅ 已审核 / ⏭️ 已跳过 |
| 审核模型 | <审核模型名称> |
| 生成时间 | <YYYY-MM-DD HH:MM> |
| 文档版本 | <v1.0（初稿/最终版）> |
```

**审核状态说明**：
- `⏳ 待审核`：初稿完成，等待审核
- `✅ 已审核`：审核完成，已应用修改
- `⏭️ 已跳过`：用户选择跳过审核
