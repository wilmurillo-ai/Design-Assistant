# 家庭旅行检查清单

## 出发前准备

### 证件类
- [ ] 护照（有效期至少6个月以上）
- [ ] 签证（根据目的地要求）
- [ ] 身份证
- [ ] 儿童出生证明（如有需要）
- [ ] 机票/酒店预订确认单
- [ ] 旅行保险单

### 儿童用品
- [ ] 婴儿食品/奶粉
- [ ] 尿布/湿巾
- [ ] 儿童常用药品
- [ ] 儿童防晒霜
- [ ] 玩具/书籍（长途旅行娱乐）
- [ ] 儿童耳机
- [ ] 换洗衣物（多备几套）

### 健康与安全
- [ ] 常用药品（退烧药、感冒药、肠胃药等）
- [ ] 急救包
- [ ] 防晒霜
- [ ] 驱蚊用品
- [ ] 儿童安全座椅（如自驾游）

## 行李打包

### 随身携带
- [ ] 证件包
- [ ] 手机/充电器/充电宝
- [ ] 现金/信用卡
- [ ] 儿童零食
- [ ] 换洗衣物一套（以防行李延误）

### 托运行李
- [ ] 衣物（根据天气准备）
- [ ] 洗漱用品
- [ ] 转换插头
- [ ] 雨具

## 目的地准备

### 住宿
- [ ] 确认酒店是否有婴儿床
- [ ] 确认是否提供儿童餐
- [ ] 了解附近医院位置

### 交通
- [ ] 机场接送安排
- [ ] 当地交通卡/租车确认
- [ ] 儿童推车（如需）

## 紧急联系
- [ ] 当地中国领事馆电话
- [ ] 保险公司紧急电话
- [ ] 酒店联系方式
- [ ] 国内紧急联系人
