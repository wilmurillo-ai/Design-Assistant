# Travel Manager Skill

## 核心能力
- 国际旅行规划
- 多目的地行程创建
- 家庭旅行事务管理
- 成本优化
- 旅行文件管理

## 工作流程步骤（针对中国公民出国旅行）

### 第一阶段：目的地规划
1. **目的地分析**
   - 确认目的地对中国公民的签证政策（免签/落地签/需提前申请）
   - 查询目的地安全状况和旅行警告
   - 了解当地气候和最佳旅行季节
   - 确认是否有特殊的健康要求（疫苗、核酸检测等）

2. **路线优化**
   - 设计合理的多目的地路线，避免走回头路
   - 考虑中转城市和停留时间
   - 评估不同交通方式（直飞/转机/高铁等）
   - 预留充足的转机时间（建议国际转机至少3小时）

### 第二阶段：预算与预订
3. **成本计算**
   - 机票价格对比（多平台比价）
   - 住宿费用估算
   - 当地交通和餐饮预算
   - 景点门票和活动费用
   - 预留应急资金（建议总预算的10-15%）

4. **预订协调**
   - 机票预订（确认退改签政策）
   - 酒店/民宿预订（确认取消政策）
   - 当地交通（租车/包车/公共交通卡）
   - 景点门票（热门景点提前预订）
   - 购买旅行保险（含医疗和意外险）

### 第三阶段：出行准备
5. **文件准备**
   - **护照检查**：有效期需6个月以上，足够的空白页
   - **签证办理**：根据目的地要求准备材料，预留充足时间
   - **机票/酒店预订单**：打印或保存电子版
   - **旅行保险单**：随身携带保险单和紧急联系电话
   - **证件备份**：护照、身份证复印2份，分开放置
   - **电子备份**：扫描件存储在云端和邮箱

6. **行前检查**
   - 确认航班状态和时间
   - 检查行李限制（重量和尺寸）
   - 准备当地货币或确保银行卡可用
   - 下载离线地图和翻译应用
   - 记录紧急联系人（国内+目的地中国领事馆）

### 第四阶段：旅行中
7. **入境准备**
   - 填写入境卡（准备酒店地址和联系方式）
   - 准备好往返机票和酒店预订单备查
   - 了解海关申报规定
   - 保存好入境章页照片

8. **当地管理**
   - 每日行程确认和调整
   - 费用记录和管理
   - 保持与家人/朋友的联系
   - 关注当地安全信息

## 家庭旅行关键注意事项
- **适合儿童的路线**：避免过长的飞行时间，选择有儿童设施的航司
- **中途停留舒适度**：选择适合家庭休息的中转机场
- **行李需求**：儿童用品（食品、药品、玩具）充足准备
- **年龄相关的旅行需求**：婴儿摇篮、儿童餐、推车等
- **证件特殊要求**：未成年人出行需准备出生证明、父母同意书等
- **紧急医疗**：了解目的地医疗设施，携带常用药品
- **通讯保障**：确保随时能与国内联系，考虑购买当地电话卡

## 使用示例

### 示例 1：完整家庭旅行规划
**用户输入**："规划一次7月份去日本和韩国的家庭旅行，两大一小，预算3万元"

**执行流程**：
1. 查询 Web 获取日韩签证政策（日本需提前申请，韩国济州岛免签）
2. 查询 Web 获取7月日韩天气预报
3. 查询 Web 搜索北京-东京-首尔-北京的多程机票价格
4. 参考 [airline-matrix.md](references/airline-matrix.md) 推荐适合家庭的航司
5. 查询 Web 搜索家庭友好型酒店
6. 查询 Web 获取适合儿童的景点推荐
7. 整合信息，输出完整行程方案

### 示例 2：文件准备检查
**用户输入**："我要去泰国旅行，帮我检查需要准备什么文件"

**执行流程**：
1. 查询 Web 确认泰国对中国免签政策（永久免签）
2. 参考 [travel-documents.md](references/travel-documents.md) 文件要求
3. 参考 [family-travel-checklist.md](references/family-travel-checklist.md) 检查清单
4. 输出文件清单和注意事项

### 示例 3：成本优化
**用户输入**："寻找春节期间去东南亚最具性价比的路线"

**执行流程**：
1. 查询 Web 对比泰国、越南、新加坡等目的地的机票价格
2. 查询 Web 获取各国汇率和消费水平
3. 查询 Web 比较各国住宿成本
4. 参考 [airline-matrix.md](references/airline-matrix.md) 低成本航空信息
5. 综合分析，推荐性价比最高的路线
