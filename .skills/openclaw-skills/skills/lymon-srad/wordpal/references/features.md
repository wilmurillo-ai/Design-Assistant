# WordPal · 功能说明

当用户问「有什么功能 / 能做什么 / 支持什么 / 哪些能调」时，读取本文件回答。

## 简短介绍

WordPal 主要有 4 类能力：学新词、复习到期词、看学习报告、调整学习参数。  
其中参数类可以让 LLM 帮忙改，词库底层状态不能直接手改。

## 功能表

| 功能     | 说明                                   | 是否可让 LLM 调整 |
| -------- | -------------------------------------- | ----------------- |
| 学新词   | 按目标、难度和上下文出新词题           | 部分可以          |
| 到期复习 | 复习今天该回顾的词                     | 部分可以          |
| 学习报告 | 看进度、趋势和风险词                   | 可以              |
| 学习参数 | 调整难度、每日目标、报告风格、推送时间 | 可以              |
| 词库状态 | 词的状态、复习间隔、下次时间           | 不可以直接改      |

## 一句话边界

可以直接让 LLM 帮你调的是难度、每日目标、报告风格和推送时间；不能直接手改某个词的状态或复习时间。
