# 报警维权示例案例

## 案例背景

赵女士在网上被诈骗分子以"投资理财"为由骗取人民币50,000元。发现被骗后，赵女士立即到当地派出所报案，但遭遇以下问题：
1. 派出所民警称"网络诈骗归网警管，我们管不了"，拒绝受理
2. 赵女士多次交涉后，民警才勉强登记，但拒绝出具受案回执
3. 赵女士通过12389投诉后，派出所才出具受案回执
4. 一周后，派出所电话通知赵女士"不够立案标准，不予立案"，但未出具书面通知书
5. 赵女士要求出具书面不予立案通知书，民警推诿不给

---

## Step 1：侵权诊断

**分析**：
- 派出所拒绝受理报案 → 违反《公安机关办理刑事案件程序规定》第166条（"都应当立即接受"）
- 不出具受案回执 → 违反第171条（"应当制作受案登记表和受案回执"）
- 不出具不予立案通知书 → 违反第178条（"应当制作不予立案通知书，并在三日以内送达控告人"）

**结论**：公安机关存在多项程序违法行为，赵女士的报案权和监督权受到侵害。

---

## Step 2：权益确认

**被侵犯的权益**：
| 权益 | 法律依据 | 侵权事实 |
|------|---------|---------|
| 报案权 | 刑诉法第110条 | 派出所拒绝受理报案 |
| 知情权 | 办案程序规定第171条 | 不出具受案回执 |
| 救济权 | 办案程序规定第178条 | 不出具不予立案通知书 |

**维权类型**：报警维权（刑事报案）

**法律依据**：
- 《刑事诉讼法》第110条（报案权）
- 《刑事诉讼法》第112条（不予立案复议权）
- 《刑事诉讼法》第113条（检察院立案监督权）
- 《公安机关办理刑事案件程序规定》第166条（立即接受义务）
- 《公安机关办理刑事案件程序规定》第171条（出具回执义务）
- 《公安机关办理刑事案件程序规定》第178条（不予立案程序）
- 《公安机关办理刑事案件程序规定》第179条（复议复核）
- 《公安机关督察条例》（警务督察）

---

## Step 3：证据整理

**已有证据**：
| 证据 | 类型 | 证明内容 |
|------|------|---------|
| 诈骗转账记录 | 电子证据 | 被骗金额、对方账户、转账时间 |
| 与诈骗分子的聊天记录 | 电子证据 | 诈骗过程、诈骗手段 |
| 诈骗平台/APP截图 | 电子证据 | 虚假投资平台 |
| 第一次报案通话录音 | 电子证据 | 派出所拒绝受理 |
| 12389投诉记录 | 电子证据 | 投诉派出所不出回执 |
| 派出所最终出具的受案回执 | 书证 | 确认已受案 |
| 派出所电话通知不予立案的录音 | 电子证据 | 口头告知不予立案 |

**缺失证据**（建议补充）：
- 要求派出所出具书面《不予立案通知书》（如仍拒绝，记录拒绝过程）
- 收集其他受害者信息（如涉及同一诈骗团伙，人数多更容易立案）

**案件定性分析**：
- 诈骗金额50,000元，已达到诈骗罪立案标准（一般3,000-10,000元即构成）
- 属于电信网络诈骗，应由公安机关立案侦查
- 派出所以"不够立案标准"为由不予立案，理由不成立

---

## Step 4：维权路径规划

### 路径A：要求出具不予立案通知书 + 复议 + 复核

1. **书面要求**派出所出具《不予立案通知书》
2. 收到后**7日内**向该派出所所属的区/县公安局申请**复议**
3. 对复议不服，**7日内**向市公安局申请**复核**

### 路径B：直接向检察院申请立案监督（推荐，最有效）

1. 向**区/县人民检察院**（第一检察部/侦查监督部门）提交《立案监督申请书》
2. 附材料：
   - 立案监督申请书
   - 受案回执（证明已报案）
   - 不予立案通知书（如有）
   - 全套证据材料（转账记录、聊天记录等）
   - 派出所拒绝受理/不出文书的证据
3. 检察院审查后，要求公安机关7日内说明不立案理由
4. 检察院认为理由不成立的，通知公安机关立案
5. 公安机关接到通知后**必须立案**

### 路径C：继续向警务督察投诉

1. 拨打**12389**或通过**12389.gov.cn**网上举报
2. 投诉内容：
   - 派出所拒绝受理报案
   - 不出具受案回执
   - 不出具不予立案通知书
   - 不作为

**建议**：路径B和路径C**同时进行**，双管齐下。

---

## Step 5：生成文书

### 文书1：立案监督申请书（向检察院提交）

```
立案监督申请书

申请人：赵女士，女，XXXX年XX月XX日生，
        身份证号：XXX，住址：XXX，电话：XXX

被申请监督机关：XXX公安局XXX派出所

申请事项：
请求贵院依法对XXX公安局XXX派出所不予立案的行为进行监督，
通知该所对赵女士被诈骗案依法立案侦查。

事实和理由：

申请人于XXXX年XX月XX日到XXX派出所报案，
称被他人以"投资理财"为由诈骗人民币50,000元。
申请人提交了转账记录、聊天记录等证据材料。

然而，该派出所：
1. 最初以"网络诈骗归网警管"为由拒绝受理，
   违反了《公安机关办理刑事案件程序规定》第166条
   关于"都应当立即接受"的规定；
2. 经申请人多次交涉并拨打12389投诉后才出具受案回执，
   违反了该规定第171条关于"应当制作受案回执"的规定；
3. 后电话通知申请人"不够立案标准，不予立案"，
   但至今未出具书面《不予立案通知书》，
   违反了该规定第178条关于"应当制作不予立案通知书，
   并在三日以内送达控告人"的规定。

根据《中华人民共和国刑法》第266条，诈骗公私财物，
数额较大的，处三年以下有期徒刑。申请人被骗50,000元，
已远超当地诈骗罪立案标准。

根据《中华人民共和国刑事诉讼法》第113条，
被害人认为公安机关对应当立案侦查的案件而不立案侦查，
向人民检察院提出的，人民检察院应当要求公安机关说明
不立案的理由。人民检察院认为公安机关不立案理由不能
成立的，应当通知公安机关立案。

综上所述，申请人恳请贵院依法履行立案监督职责，
通知XXX派出所对本案依法立案侦查，维护申请人的
合法权益。

此致
XXX区/县人民检察院

申请人（签名）：
XXXX年XX月XX日

附件：
1. 受案回执复印件
2. 诈骗转账记录打印件
3. 与诈骗分子的聊天记录打印件
4. 诈骗平台截图
5. 12389投诉记录
6. 其他相关证据材料
```

## 本案例涉及的关键法条速查

| 法条 | 内容 | 适用 |
|------|------|------|
| 刑诉法第110条 | 报案权 | 任何公民有权向公安机关报案 |
| 刑诉法第112条 | 不予立案复议 | 控告人可在7日内申请复议 |
| 刑诉法第113条 | 检察院立案监督 | 被害人可向检察院申请监督立案 |
| 刑法第266条 | 诈骗罪 | 诈骗公私财物，数额较大 |
| 办案程序规定第166条 | 立即接受义务 | 公安机关必须接受报案 |
| 办案程序规定第171条 | 出具回执义务 | 必须制作受案登记表和受案回执 |
| 办案程序规定第178条 | 立案/不予立案 | 立案三要件 + 不予立案须出具通知书 |
| 办案程序规定第179条 | 复议复核 | 7日内复议，不服7日内复核 |
| 公安机关督察条例 | 警务督察 | 投诉民警不作为、违法违纪 |

## 报警维权注意事项

1. **报案时**：务必要求制作笔录并签名，务必索要**受案回执**
2. **被拒绝时**：拨打110记录报警过程（110录音本身就是证据）
3. **不予立案时**：务必索要书面**不予立案通知书**
4. **救济途径**：可不经复议，直接向检察院申请立案监督（更有效）
5. **证据固定**：所有与公安机关的沟通尽量录音、录像
6. **网络诈骗**：及时联系银行冻结对方账户，增加追回资金的可能性
7. **多人受害**：如涉及同一诈骗团伙，联合其他受害者一起报案，更容易立案
