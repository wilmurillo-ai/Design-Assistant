# 互联网维权示例案例

## 案例背景

陈先生在使用某外卖APP时遭遇以下问题：
1. APP强制要求获取位置信息、通讯录、相册等权限，拒绝授权则无法使用
2. 发现该APP在后台频繁读取位置信息，即使未使用APP
3. 陈先生的手机号和点餐记录被泄露，频繁收到推销电话和短信
4. 向APP客服投诉，客服回复"这是系统默认设置，无法修改"
5. 陈先生要求注销账号，APP不提供注销入口，客服表示"账号一旦注册无法注销"

---

## Step 1：侵权诊断

**分析**：
- 强制索权、拒绝授权则无法使用 → 违反《个人信息保护法》第6条（最小必要原则）和第13条（知情同意原则）
- 后台频繁读取位置信息 → 违反《个人信息保护法》第6条（最小必要原则）
- 个人信息泄露 → 违反《个人信息保护法》第44条和《网络安全法》第42条
- 不提供账号注销渠道 → 违反《个人信息保护法》第47条

**结论**：多项互联网/个人信息权益受到侵害。

---

## Step 2：权益确认

**被侵犯的权益**：
| 权益 | 法律依据 | 侵权事实 |
|------|---------|---------|
| 知情权 | 个人信息保护法第13条 | 未充分告知收集目的和范围 |
| 决定权 | 个人信息保护法第44条 | 强制索权，拒绝则无法使用 |
| 删除权 | 个人信息保护法第47条 | 不提供账号注销渠道 |
| 个人信息安全权 | 个人信息保护法第44条 + 网络安全法第42条 | 个人信息被泄露 |

**维权类型**：互联网/通信维权

**法律依据**：
- 《个人信息保护法》第5条（合法正当必要原则）
- 《个人信息保护法》第6条（最小必要原则）
- 《个人信息保护法》第13条（知情同意规则）
- 《个人信息保护法》第24条（自动化决策）
- 《个人信息保护法》第44条（个人权利）
- 《个人信息保护法》第47条（删除权）
- 《网络安全法》第41条（收集使用规则）
- 《网络安全法》第42条（不得泄露篡改毁损）
- 《网络安全法》第43条（删除更正权）

---

## Step 3：证据整理

**已有证据**：
| 证据 | 类型 | 证明内容 |
|------|------|---------|
| APP权限申请截图 | 电子证据 | 强制索要通讯录、相册等权限 |
| 手机系统权限记录 | 电子证据 | APP后台频繁读取位置信息 |
| 推销电话/短信截图 | 电子证据 | 个人信息被泄露 |
| 与APP客服聊天记录 | 电子证据 | 客服拒绝处理、不提供注销 |
| APP无注销入口截图 | 电子证据 | 不提供账号注销渠道 |
| APP名称、版本号 | 信息 | XX外卖 v3.5.2，应用宝下载 |

**缺失证据**（建议补充）：
- APP隐私政策截图（查看是否明示收集目的）
- 手机权限管理页面的完整截图
- 推销电话录音（如接到电话）
- 向运营商查询的个人信息查询记录

---

## Step 4：维权路径规划

### 路径A：向企业投诉（前置步骤）

1. 通过APP内反馈渠道提交投诉
2. 拨打APP官方客服电话投诉
3. 保留投诉记录和工单号
4. 等待15日或对结果不满意

### 路径B：向监管部门投诉（推荐，可多渠道并行）

**渠道1：12321 / 互联网信息服务投诉平台**
- 网址：https://hlwtsxt.miit.gov.cn
- 投诉类别：个人信息保护
- 提交APP信息、侵权行为描述、截图证据

**渠道2：12321举报中心（APP专项举报）**
- 微信公众号"12321受理中心" → 其他投诉 → 手机应用
- 或微信公众号"工信微报" → 我要投诉 → 手机应用软件

**渠道3：12377网信办举报**
- 网址：https://www.12377.cn
- 举报类别：个人信息泄露 / APP违法违规

### 路径C：诉讼（如投诉后问题未解决）

- **管辖**：互联网法院（北京/杭州/广州）
- **案由**：个人信息权益纠纷
- **诉讼请求**：
  1. 判令被告停止违法收集使用个人信息
  2. 判令被告删除违法收集的个人信息
  3. 判令被告提供账号注销渠道并完成注销
  4. 判令被告赔偿因个人信息泄露造成的损失
  5. 判令被告公开道歉

**投诉内容模板**：

```

投诉内容：

   通讯录、相册等权限，拒绝授权则无法使用任何功能，
   违反了《个人信息保护法》关于最小必要原则和

   向客服反映后被告知"账号一旦注册无法注销"，
   违反了《个人信息保护法》第47条的规定。
```

## 本案例涉及的关键法条速查

| 法条 | 内容 | 适用 |
|------|------|------|
| 个人信息保护法第5条 | 合法正当必要原则 | APP收集信息的基本原则 |
| 个人信息保护法第6条 | 最小必要原则 | 不得过度收集，限于最小范围 |
| 个人信息保护法第13条 | 知情同意规则 | 应取得个人明示同意 |
| 个人信息保护法第24条 | 自动化决策 | 不得仅通过自动化决策方式向用户推送 |
| 个人信息保护法第44条 | 个人权利 | 知情权、决定权、查阅复制权、删除权等 |
| 个人信息保护法第47条 | 删除权 | 处理目的已实现等情形应主动删除 |
| 网络安全法第41条 | 收集使用规则 | 应明示目的方式和范围，经被收集者同意 |
| 网络安全法第42条 | 不得泄露篡改 | 未经同意不得向他人提供 |
| 网络安全法第43条 | 删除更正权 | 用户有权要求删除或更正个人信息 |

---

## 互联网维权注意事项

1. **证据先行**：发现侵权行为后第一时间截图录屏，APP可能随时更新
2. **多渠道并行**：可同时向12321、12377、互联网投诉平台投诉，提高处理效率
3. **前置投诉**：向监管部门投诉前，建议先向企业投诉并保留记录
4. **通信服务**：涉及运营商问题（扣费、套餐等），须先向运营商投诉，再向12300申诉
5. **个人信息保护**：要求企业删除个人信息时，建议通过书面方式（邮件/信函）提出，保留证据
6. **互联网法院**：涉及网络纠纷可向北京/杭州/广州互联网法院在线起诉，全流程网上办理
7. **联合维权**：如涉及大量用户同一问题，可考虑联合维权或向消费者协会反映
