# 交通事故维权案例：电动车闯红灯被撞 + 保险拒赔

## 用户描述

> "我骑电动车正常行驶通过绿灯路口，一辆小轿车闯红灯撞了我，我摔倒受伤，电动车损坏。交警认定对方全责。我住院治疗了15天，花了医疗费3.2万元。现在对方保险公司只愿意赔医疗费，不赔误工费和护理费，说需要我提供完税证明。我是个体户，没有固定工资流水，怎么办？"

---

## Step 1：侵权诊断

### 侵权分析

| 侵权行为 | 侵权类型 | 严重程度 |
|---------|---------|---------|
| 轿车闯红灯导致事故 | 交通事故侵权 | ⭐⭐⭐⭐ |
| 保险公司拒赔合理费用 | 保险理赔纠纷 | ⭐⭐⭐ |

### 关键判断
- 对方全责，依法应承担全部赔偿责任
- 保险公司以"无完税证明"拒赔误工费不合理
- 个体户可提供其他收入证明材料

---

## Step 2：权益确认

### 被侵犯的权益
1. **健康权** — 因交通事故受伤
2. **财产权** — 电动车损坏
3. **获得赔偿权** — 对方全责应全额赔偿

### 维权类型：交通事故维权 + 保险维权

### 主要法律依据

| 法律条文 | 内容 |
|---------|------|
| 《道路交通安全法》第76条 | 机动车与非机动车之间事故，由机动车一方承担赔偿责任 |
| 《民法典》第1179条 | 侵害他人造成人身损害的，应赔偿医疗费、护理费、交通费、误工费等 |
| 《民法典》第1208条 | 机动车发生交通事故造成损害的，依照道路交通安全法和本法的有关规定承担赔偿责任 |
| 《最高人民法院关于审理人身损害赔偿案件适用法律若干问题的解释》第7条 | 误工费根据受害人的误工时间和收入状况确定 |

---

## Step 3：证据整理

### 已有证据
- [x] 交通事故认定书（对方全责）
- [x] 住院病历、诊断证明
- [x] 医疗费发票（3.2万元）

### 需补充证据
- [ ] 营业执照（证明个体户身份）
- [ ] 近期营业收入证明（银行流水、微信/支付宝收款记录）
- [ ] 同行业平均收入证明（当地同行业收入标准）
- [ ] 护理人员身份证明及收入证明
- [ ] 电动车维修/报废评估报告
- [ ] 交通费票据

---

## Step 4：协商/投诉

### 与保险公司协商

**要点**：
1. 个体户误工费不要求完税证明，可提供营业执照+银行流水证明收入状况
2. 如无法证明最近三年平均收入，可参照**当地相同或相近行业上一年度职工的平均工资**计算
3. 护理费按护理人员实际收入或当地护工标准计算
4. 如保险公司仍拒绝，向12378投诉

### 预估赔偿金额

| 赔偿项目 | 金额 | 依据 |
|---------|------|------|
| 医疗费 | 32,000元 | 实际发票 |
| 误工费（15天） | 约3,000-5,000元 | 参照当地行业平均工资 |
| 护理费（15天） | 约2,000-3,000元 | 当地护工标准或家属误工 |
| 住院伙食补助 | 1,500元 | 100元/天 × 15天 |
| 营养费 | 500-1,000元 | 医疗机构意见 |
| 交通费 | 500元 | 实际发生 |
| 电动车损失 | 2,000-5,000元 | 评估报告 |
| **合计** | **约41,000-47,500元** | |

---

## Step 5-6：升级维权

如协商不成：
1. 向12378投诉保险公司
2. 向当地银保监局投诉
3. 准备起诉

---

## Step 7：准备起诉

### 管辖法院
- 事故发生地法院（最方便）
- 对方车主住所地法院

### 被告
1. 对方车主（侵权人）
2. 对方保险公司（交强险+商业三者险）

### 诉讼请求
1. 判令被告赔偿医疗费32,000元
2. 判令被告赔偿误工费X元
3. 判令被告赔偿护理费X元
4. 判令被告赔偿住院伙食补助费1,500元
5. 判令被告赔偿营养费X元
6. 判令被告赔偿交通费X元
7. 判令被告赔偿电动车损失X元
8. 本案诉讼费由被告承担

---

## 维权要点总结

1. **个体户误工费**：不需要完税证明，营业执照+银行流水即可证明收入
2. **交强险先行赔付**：医疗费1.8万+财产损失2000元由交强险优先赔付
3. **商业三者险补充**：超出交强险部分由商业三者险按责任比例赔付
4. **协商优先**：起诉前尽量通过协商解决，节省时间和费用
5. **诉讼时效3年**：从事故发生之日起计算
