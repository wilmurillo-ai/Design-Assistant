# 出行维权案例：航班取消 + 旅行社行程变更

## 用户描述

> "我报了一个5天4晚的云南旅行团，费用5800元。出发当天航班被航空公司取消了，旅行社说只能改到第二天出发，行程压缩为4天3晚，但不愿退还差价。到了云南后，导游又带我们去购物店，不买东西就不让上车。回来后我找旅行社理论，他们说合同里写了'行程可能因不可抗力调整'，拒绝赔偿。"

---

## Step 1：侵权诊断

### 侵权分析

| 侵权行为 | 侵权类型 | 严重程度 |
|---------|---------|---------|
| 航班取消后压缩行程且不退差价 | 旅游合同违约 | ⭐⭐⭐ |
| 导游强制购物、不买不让上车 | 强制消费（违反旅游法） | ⭐⭐⭐⭐ |
| 以"不可抗力"条款规避责任 | 不合理免责 | ⭐⭐⭐ |

### 关键判断
- 航班取消属于**不可抗力**，但旅行社应按《旅游法》第六十七条处理合同变更，退还减少的费用
- 强制购物属于**明确违法**，与航班取消无关，旅行社不能以此为由推卸责任
- "行程可能因不可抗力调整"条款不能免除旅行社的**法定义务**

---

## Step 2：权益确认

### 被侵犯的权益
1. **公平交易权** — 被强制购物
2. **知情权** — 未被告知行程变更后的费用退还方案
3. **人身自由权** — 不购物不让上车
4. **合同履约权** — 旅行社未按合同约定提供服务

### 维权类型：出行/交通维权 + 消费者维权

### 主要法律依据

| 法律条文 | 内容 |
|---------|------|
| 《旅游法》第三十五条 | 旅行社不得组织、诱导、强迫购物；强迫购物的赔偿团费20%惩罚性赔偿金 |
| 《旅游法》第四十一条 | 导游不得诱导、欺骗、强迫消费 |
| 《旅游法》第六十七条 | 因不可抗力合同变更的，增加费用旅游者承担，减少的费用退还旅游者 |
| 《旅游法》第七十条 | 旅行社不履行合同义务的，应承担继续履行、补救措施或赔偿损失等违约责任 |
| 《消费者权益保护法》第十条 | 消费者享有公平交易权 |

## Step 3：证据整理

### 已有证据
- [x] 旅游合同（含"不可抗力"条款）
- [x] 付款凭证（5800元转账记录）
- [x] 航班取消通知（短信/APP截图）
- [x] 旅行社通知行程变更的聊天记录

### 需补充证据
- [ ] 购物店的消费凭证（如有）
- [ ] 同行旅客证言（证明强制购物）
- [ ] 导游强制购物的录音/录像
- [ ] 行程压缩后实际提供服务的对比清单

---

## Step 4：投诉举报

### 投诉方案

**第一步：向旅行社正式投诉**
- 要求退还行程压缩的差价（约1天费用）
- 要求退还强制购物金额
- 要求支付团费20%的惩罚性赔偿金（1160元）

**第二步：行政投诉（如旅行社不配合）**
- 拨打 **12345** 转文旅部门
- 或向当地**文化和旅游局**提交书面投诉
- 附上所有证据材料

**第三步：消费者协会投诉**
- 拨打 **12315** 或通过全国12315平台在线投诉

---

## Step 5：升级维权

如行政投诉后旅行社仍不配合：
- 向**文化和旅游部**投诉（https://www.mct.gov.cn）
- 通过**12301**旅游投诉热线（部分地区已并入12345）

---

## Step 6-9：调解/诉讼

### 预估赔偿金额

| 赔偿项目 | 金额 | 依据 |
|---------|------|------|
| 行程压缩差价退还 | 约1000-1200元 | 《旅游法》第六十七条 |
| 强制购物退款 | 实际购物金额 | 《旅游法》第三十五条 |
| 惩罚性赔偿金 | 1160元（团费×20%） | 《旅游法》第三十五条 |
| **合计** | **约2160元 + 购物退款** | |

### 诉讼建议
- 如金额较小，建议通过**旅游投诉平台**或**消协调解**
- 如金额较大或调解失败，可向**旅行社所在地法院**提起诉讼
- 诉讼请求：退还差价 + 退还购物款 + 惩罚性赔偿金 + 诉讼费

---

## 维权要点总结

1. **航班取消 ≠ 旅行社免责**：不可抗力只免除违约责任，但减少的费用必须退还
2. **强制购物是明确违法**：无论合同怎么写，都不能免除《旅游法》的强制性规定
3. **保留证据是关键**：录音录像、消费凭证、聊天记录
4. **投诉渠道**：12345（文旅） > 12315（消协） > 法院诉讼
