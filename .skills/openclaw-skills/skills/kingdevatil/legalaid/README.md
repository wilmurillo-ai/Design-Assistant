# LegalAid / 维权助手

一个全流程维权辅助 Skill，帮助用户识别侵权行为、整理证据、生成法律文书、规划维权路径，并提供从"意识到被侵权"到"法庭判决/上诉"的完整维权指导。

## 特性

- **9步核心流程**：从侵权诊断到判决上诉，覆盖维权全链路
- **14大维权类型**：消费者、劳动、租房、行政、报警、互联网、人身伤害、物业、邻里/社区、上学、出行/交通、医疗医药、保险、交通事故
- **交叉类型支持**：自动识别多类型交叉场景，同时引用多个知识库
- **未知场景适配**：未覆盖的类型通过流程框架+网络搜索+类比处理
- **管辖权分析**：起诉阶段智能分析管辖法院，帮助用户选择最优法院避免异地维权成本
- **被告选择策略**：根据场景推荐最优被告组合，提高胜诉概率和执行效果
- **官方权威数据**：所有法律条文、文书模板、渠道信息均来源于政府官网
- **判例查询辅助**：集成裁判文书网、人民法院案例库等权威判例平台
- **多格式文书生成**：支持 Word（起诉状/投诉信）、Excel（证据清单）、PDF（分析报告）
- **7个完整示例案例**：消费者、劳动、租房、报警、互联网、出行、交通事故维权
- **11个模板 + 11类话术**：覆盖投诉信、起诉状、仲裁申请书、信访材料等
- **16个渠道章节**：含扫黑除恶(12337)、消防监督、税务监督(12366)

## 安装与使用

### OpenClaw

**方式 A：在对话中直接安装**

在 OpenClaw 对话中粘贴仓库地址，它会自动克隆并识别 `SKILL.md`：
```
Clone https://github.com/KingDevatil/LegalAid-skill.git
```

**方式 B：手动安装**
1. 克隆仓库：`git clone https://github.com/KingDevatil/LegalAid-skill.git`
2. 将 `LegalAid-skill` 文件夹复制到 `~/.openclaw/skills/` 目录
3. 重启 OpenClaw

### Hermes Agent

Hermes 原生支持 `SKILL.md` 格式，可直接安装：
```bash
hermes skills install https://github.com/KingDevatil/LegalAid-skill
```

### CoPaw

CoPaw 支持通过 GitHub 仓库 URL 导入 Skill：
1. 复制本仓库的 URL：`https://github.com/KingDevatil/LegalAid-skill`
2. 在 CoPaw 中进入 Skills 管理页面
3. 选择「导入 Skill」，粘贴 URL
4. 确认安装即可

### WorkBuddy

1. 打开 WorkBuddy 左侧「技能」菜单
2. 选择「上传技能包」或「导入自定义 Skill」
3. 选择克隆下来的 `LegalAid-skill` 文件夹（包含 `SKILL.md` 的目录）
4. 安装完成后在「已安装」区域启用

### Trae / Cursor 等 AI 编辑器

1. **克隆仓库**
   ```bash
   git clone https://github.com/KingDevatil/LegalAid-skill.git
   ```

2. **放置到 Skill 目录**
   
   | 平台 | Skill 目录位置 |
   |------|---------------|
   | Trae | `~/.trae/skills/` 或设置中配置的 Skill 路径 |
   | Cursor | `~/.cursor/skills/` |

3. **启用 Skill** 后即可在对话中使用

## 维权核心流程

```
Step 1  侵权诊断     → 帮助用户意识到被侵权
Step 2  权益确认     → 定性并归类维权类型，匹配法律条文（支持交叉类型）
Step 3  证据整理     → 梳理证据链，明确诉求，预估赔偿金额
Step 4  投诉举报     → 生成投诉信，指引政府职能部门投诉渠道
Step 5  升级举报     → 举报职能部门不作为，生成信访/举报材料
Step 6  调解评估     → 评估行政调解/处罚方案
Step 7  准备起诉     → 管辖权分析 + 被告选择 + 诉讼请求优化 + 生成起诉状
Step 8  庭审准备     → 生成庭审材料（代理词/答辩状/陈述提纲）
Step 9  判决分析     → 分析判决结果，生成上诉状
```

## 法律知识库

知识库涵盖以下法律法规的核心条款和维权要点：

- **《消费者权益保护法》《电子商务法》** — 知情权、七天无理由退货、欺诈赔偿（退一赔三）、电商平台维权、直播带货
- **《劳动法》《劳动合同法》** — 书面合同、试用期、加班费、经济补偿、违法解除赔偿金、社保
- **《民法典》** — 租赁合同、买卖不破租赁、侵权责任、格式条款、精神损害赔偿、相邻权
- **《行政复议法》《行政诉讼法》《信访工作条例》** — 行政复议、行政诉讼、信访流程
- **《刑事诉讼法》《公安机关办理刑事案件程序规定》** — 报案受案、立案条件、不予立案救济、警务督察、检察院立案监督
- **《个人信息保护法》《网络安全法》《数据安全法》** — 个人信息权利、APP治理、通信服务申诉
- **《道路交通安全法》《道路交通事故处理程序规定》** — 事故处理、责任认定、赔偿标准
- **《民用航空法》《航班正常管理规定》** — 航班延误/取消旅客权益、退改签
- **《铁路旅客运输规程》** — 退票改签、列车晚点
- **《巡游出租汽车经营服务管理规定》《网约车管理办法》** — 拒载、绕路、多收费
- **《旅游法》《旅行社条例》《在线旅游经营服务管理暂行规定》** — 行程变更、强制消费、OTA平台纠纷
- **《医疗纠纷预防和处理条例》《医疗事故处理条例》** — 医疗损害责任、鉴定程序、赔偿标准
- **《药品管理法》《医疗器械监督管理条例》** — 假药劣药认定、药品安全维权
- **《保险法》《银行保险机构消费者权益保护管理办法》** — 理赔规则、免责条款、销售误导、退保权益
- **《物业管理条例》** — 物业服务质量、物业费、公共区域管理
- **《教育法》《义务教育法》《未成年人保护法》** — 教育资源分配、学籍管理、校园霸凌
- **《消防法》** — 消防隐患举报、消防安全责任
- **《税收征收管理法》** — 税收违法举报、纳税人权益
