# 上学维权法律知识库

> 来源：《中华人民共和国教育法》《中华人民共和国义务教育法》《中华人民共和国未成年人保护法》（2020年修订）《未成年人学校保护规定》（2021年）
> 官方查询：http://www.npc.gov.cn / http://www.moe.gov.cn

---

## 一、教育法（2021年修正）

### 第9条：公民受教育权
中华人民共和国公民有受教育的权利和义务。公民不分民族、种族、性别、职业、财产状况、宗教信仰等，依法享有平等的受教育机会。

### 第44条：受教育者权利
受教育者享有下列权利：
- 参加教育教学计划安排的各种活动，使用教育教学设施、设备、图书资料
- 按照国家有关规定获得奖学金、贷学金、助学金
- 在学业成绩和品行上获得公正评价，完成规定的学业后获得相应的学业证书、学位证书
- 对学校给予的处分不服向有关部门提出申诉，对学校、教师侵犯其人身权、财产权等合法权益，提出申诉或者依法提起诉讼
- 法律、法规规定的其他权利

### 第45条：受教育者义务
受教育者应当履行下列义务：
- 遵守法律、法规
- 遵守学生行为规范，尊敬师长，养成良好的思想品德和行为习惯
- 努力学习，完成规定的学习任务
- 遵守所在学校或者其他教育机构的管理制度

### 第80条：校园秩序
结伙斗殴、寻衅滋事，扰乱学校及其他教育机构教育教学秩序或者破坏校舍、场地及其他财产的，由公安机关给予治安管理处罚；构成犯罪的，依法追究刑事责任。

## 二、义务教育法（2015年修正）

### 第2条：义务教育制度
国家实行九年义务教育制度。义务教育是国家统一实施的所有适龄儿童、少年必须接受的教育，是国家必须予以保障的公益性事业。

### 第4条：平等入学
凡具有中华人民共和国国籍的适龄儿童、少年，不分性别、民族、种族、家庭财产状况、宗教信仰等，依法享有平等接受义务教育的权利，并履行接受义务教育的义务。

### 第5条：保障入学
适龄儿童、少年的父母或者其他法定监护人应当依法保证其按时入学接受并完成义务教育。各级人民政府及其有关部门应当履行本法规定的各项职责，保障适龄儿童、少年接受义务教育的权利。

### 第12条：免试就近入学
适龄儿童、少年免试入学。地方各级人民政府应当保障适龄儿童、少年在户籍所在地学校就近入学。

### 第22条：均衡发展
县级以上人民政府及其教育行政部门应当促进学校均衡发展，缩小学校之间办学条件的差距，不得将学校分为重点学校和非重点学校。学校不得分设重点班和非重点班。

### 第25条：不得开除
学校不得违反国家规定开除、变相开除适龄学生。

### 第27条：不得违规收费
学校不得违反国家规定收取费用，不得以向学生推销或者变相推销商品、服务等方式谋取利益。

### 第28条：不得体罚
教师应当尊重学生的人格，不得歧视学生，不得对学生实施体罚、变相体罚或者其他侮辱人格尊严的行为，不得侵犯学生合法权益。

### 第55条：学校违规责任
学校或者教师在义务教育工作中违反教育法、教师法规定的，依照教育法、教师法的有关规定处罚。

## 三、未成年人保护法（2020年修订）

### 第27条：学校、幼儿园的教职员工不得对未成年人实施体罚、变相体罚或者其他侮辱人格尊严的行为。

### 第39条：校园欺凌防控
学校应当建立学生欺凌防控工作制度，对教职员工、学生等开展防治学生欺凌的教育和培训。
学校应对学生欺凌，应当立即制止；通知实施欺凌和被欺凌未成年学生的父母或者其他监护人参与欺凌行为的认定和处理；对相关未成年学生及时给予心理辅导、教育引导；对其父母或者其他监护人给予必要的家庭教育指导。
对实施欺凌的未成年学生，学校应当根据欺凌行为的性质和程度，依法加强管教。对严重的欺凌行为，学校不得隐瞒，应当及时向公安机关、教育行政部门报告，并配合相关部门依法处理。

### 第77条：网络欺凌
任何组织或者个人不得通过网络以文字、图片、音视频等形式，对未成年人实施侮辱、诽谤、威胁或者恶意损害形象等网络欺凌行为。

### 第100条：学校违法责任
学校、幼儿园的教职员工对未成年人实施体罚、变相体罚或者其他侮辱人格尊严行为的，由其所在单位或者上级机关责令改正；情节严重的，依法给予处分或者解除聘用关系。

## 四、未成年人学校保护规定（2021年）

### 第2条：适用范围
普通中小学、中等职业学校（以下简称学校）的未成年人保护，适用本规定。

### 第8条：不得公开排名
学校不得公开学生个人的考试成绩、名次，不得对外宣传学生升学情况。

### 第10条：不得歧视
学校应当平等对待每个学生，不得因学生及其父母的民族、种族、性别、户籍、职业、宗教信仰、教育程度、家庭状况、身心健康状况等歧视学生。

### 第18条：欺凌防控机制
学校应当落实法律规定建立学生欺凌防控和预防性侵害、性骚扰等专项制度，建立对学生欺凌、性侵害、性骚扰行为的零容忍处理机制和受伤害学生的关爱、帮扶机制。

### 第19条：欺凌治理组织
学校应当成立由校内相关人员、法治副校长、法律顾问、有关专家、家长代表、学生代表等参与的学生欺凌治理组织，负责学生欺凌行为的预防和宣传教育、组织认定、实施矫治、提供心理辅导和帮助。

### 第23条：手机管理
学校应当告知学生和家长，原则上不得将个人手机带入校园。学生确有将手机带入校园需求的，须经学生家长同意、书面提出申请，进校后应将手机交由学校统一保管，禁止带入课堂。

### 第24条：不得要求家长批改作业
学校应当与未成年学生的父母或者其他监护人互相配合，合理安排未成年学生的学习时间，保障其休息、娱乐和体育锻炼的时间。学校不得占用国家法定节假日、休息日及寒暑假期，组织未成年学生集体补课，加重其学习负担。学校不得要求或者变相要求家长为学生作业批改。

### 第33条：校园安全管理
学校应当完善校园安全管理制度，保障学生的人身和财产安全。学校应当建立学生体质健康监测制度，发现学生出现体格异常或者有特异体质、特定疾病或者异常心理状况的，应当及时告知学生父母或者其他监护人。

## 六、上学维权路径

```
发生学校/教育纠纷
    ↓
【Step 1】向学校反映
    → 向班主任、年级组长、教务处逐级反映
    → 保留书面投诉记录
    ↓
【Step 2】向教育局投诉
    → 拨打12345，转教育局
    → 或直接向当地教育局投诉
    → 教育局应当在规定时限内回复
    ↓
【Step 3】向上级部门反映
    → 对教育局处理不满 → 向上级教育局或市政府投诉
    → 涉及违法 → 向纪检监察部门举报
    → 涉及犯罪 → 向公安机关报案
    ↓
【Step 4】司法途径
    → 行政复议（对教育局行政行为不服）
    → 行政诉讼（起诉教育局不作为/违法作为）
    → 民事诉讼（起诉学校侵权，如人身损害、名誉权等）
```

## 七、校园霸凌专项处理流程

```
发现校园霸凌
    ↓
【Step 1】立即制止
    → 通知老师、班主任、学校安保
    → 如有身体伤害 → 立即送医，保留诊断证明
    ↓
【Step 2】固定证据
    → 拍摄伤情照片/视频
    → 保留聊天记录、恐吓信息等电子证据
    → 寻找目击证人
    → 保留心理咨询记录
    ↓
【Step 3】向学校正式投诉
    → 书面提交投诉信
    → 要求学校启动欺凌防控机制
    → 要求学校通知双方家长参与认定
    → 学校不得隐瞒，必须处理
    ↓
【Step 4】向教育局投诉
    → 学校不处理或处理不满意 → 向教育局投诉
    → 要求教育局依法调查处理
    ↓
【Step 5】报警
    → 如涉及殴打、恐吓、勒索 → 报警110
    → 如涉及网络欺凌 → 向网信办（12377）举报
    ↓
【Step 6】法律途径
    → 民事诉讼：要求施暴者及其监护人赔偿
    → 行政诉讼：追究学校管理责任
```
