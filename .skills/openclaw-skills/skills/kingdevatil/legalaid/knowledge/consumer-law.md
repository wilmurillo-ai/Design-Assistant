# 消费者权益保护法要点

> 来源：《中华人民共和国消费者权益保护法》（2013年修订）
> 官方查询：http://www.npc.gov.cn

---

## 一、消费者的权利（第二章）

### 第7条：安全保障权
消费者在购买、使用商品和接受服务时享有人身、财产安全不受损害的权利。消费者有权要求经营者提供的商品和服务，符合保障人身、财产安全的要求。

### 第8条：知情权
消费者享有知悉其购买、使用的商品或者接受的服务的真实情况的权利。消费者有权根据商品或者服务的不同情况，要求经营者提供商品的价格、产地、生产者、用途、性能、规格、等级、主要成份、生产日期、有效期限、检验合格证明、使用方法说明书、售后服务，或者服务的内容、规格、费用等有关情况。

### 第9条：自主选择权
消费者享有自主选择商品或者服务的权利。消费者有权自主选择提供商品或者服务的经营者，自主选择商品品种或者服务方式，自主决定购买或者不购买任何一种商品、接受或者不接受任何一项服务。消费者在自主选择商品或者服务时，有权进行比较、鉴别和挑选。

### 第10条：公平交易权
消费者享有公平交易的权利。消费者在购买商品或者接受服务时，有权获得质量保障、价格合理、计量正确等公平交易条件，有权拒绝经营者的强制交易行为。

### 第11条：获得赔偿权
消费者因购买、使用商品或者接受服务受到人身、财产损害的，享有依法获得赔偿的权利。

### 第14条：受尊重权
消费者在购买、使用商品和接受服务时，享有其人格尊严、民族风俗习惯得到尊重的权利，享有个人信息依法得到保护的权利。

### 第25条：七天无理由退货权（网购）
经营者采用网络、电视、电话、邮购等方式销售商品，消费者有权自收到商品之日起七日内退货，且无需说明理由，但下列商品除外：
- 定作的
- 鲜活易腐的
- 在线下载或者消费者拆封的音像制品、计算机软件等数字化商品
- 交付的报纸、期刊

## 二、经营者的义务（第三章）

### 第18条：安全保障义务
经营者应当保证其提供的商品或者服务符合保障人身、财产安全的要求。

### 第20条：真实信息义务
经营者向消费者提供有关商品或者服务的质量、性能、用途、有效期限等信息，应当真实、全面，不得作虚假或者引人误解的宣传。

### 第23条：质量保证义务
经营者应当保证在正常使用商品或者接受服务的情况下其提供的商品或者服务应当具有的质量、性能、用途和有效期限。

### 第24条：退货、更换、修理义务
经营者提供的商品或者服务不符合质量要求的，消费者可以依照国家规定、当事人约定退货，或者要求经营者履行更换、修理等义务。

### 第26条：格式条款限制
经营者不得以格式条款、通知、声明、店堂告示等方式，作出排除或者限制消费者权利、减轻或者免除经营者责任、加重消费者责任等对消费者不公平、不合理的规定，不得利用格式条款并借助技术手段强制交易。

## 三、争议解决（第六章）

### 第39条：争议解决途径
消费者和经营者发生消费者权益争议的，可以通过下列途径解决：
1. 与经营者协商和解
2. 请求消费者协会或者依法成立的其他调解组织调解
3. 向有关行政部门投诉
4. 根据与经营者达成的仲裁协议提请仲裁机构仲裁
5. 向人民法院提起诉讼

### 第40条：赔偿责任主体
- 消费者在购买、使用商品时，其合法权益受到损害的，可以向销售者要求赔偿
- 销售者赔偿后，属于生产者的责任或者属于向销售者提供商品的其他销售者的责任的，销售者有权向生产者或者其他销售者追偿
- 消费者或者其他受害人因商品缺陷造成人身、财产损害的，可以向销售者要求赔偿，也可以向生产者要求赔偿

### 第44条：网络购物责任
- 消费者通过网络交易平台购买商品，其合法权益受到损害的，可以向销售者要求赔偿
- 网络交易平台提供者不能提供销售者的真实名称、地址和有效联系方式的，消费者也可以向网络交易平台提供者要求赔偿

## 四、法律责任（第七章）

### 第55条：欺诈赔偿（退一赔三）
经营者提供商品或者服务有欺诈行为的，应当按照消费者的要求增加赔偿其受到的损失，增加赔偿的金额为消费者购买商品的价款或者接受服务的费用的**三倍**；增加赔偿的金额不足**五百元**的，为**五百元**。法律另有规定的，依照其规定。

### 第51条：精神损害赔偿
经营者有侮辱诽谤、搜查身体、侵犯人身自由等侵害消费者或者其他受害人人身权益的行为，造成严重精神损害的，受害人可以要求精神损害赔偿。

## 五、常见消费者维权场景

| 场景 | 适用法条 | 维权要点 |
|------|---------|---------|
| 商品质量问题 | 第24条 | 要求退货、更换、修理 |
| 虚假宣传/误导消费 | 第20条 + 第55条 | 可主张欺诈赔偿（退一赔三） |
| 网购商品不满意 | 第25条 | 七天无理由退货 |
| 格式条款霸王条款 | 第26条 | 条款无效 |
| 网络平台购物纠纷 | 第44条 | 可向平台追责 |
| 人身损害 | 第7条 + 第11条 + 第40条 | 可向销售者或生产者索赔 |
| 个人信息泄露 | 第14条 + 第29条 | 经营者不得泄露消费者个人信息 |
| 预付卡消费纠纷 | 第53条 | 经营者应按约定提供商品/服务 |
| 食品安全问题 | 第55条 + 《食品安全法》第148条 | 可主张价款十倍或损失三倍赔偿（最低1000元） |

## 六、电商平台维权场景

> 专门针对淘宝、京东、拼多多、抖音电商、快手电商等网络购物平台的维权指引。

### 6.1 主要法律依据

| 法规名称 | 发布机构 | 关键内容 |
|---------|---------|---------|
| 《中华人民共和国电子商务法》 | 全国人大常委会 | 平台经营者义务、消费者权益保护、争议解决 |
| 《网络购买商品七日无理由退货暂行办法》 | 原工商总局（现市场监管总局） | 七天无理由退货实施细则 |
| 《中华人民共和国消费者权益保护法》 | 全国人大常委会 | 网络购物责任（第44条）、七天无理由退货（第25条） |
| 《网络交易监督管理办法》 | 市场监管总局 | 网络交易行为规范、平台责任 |

### 6.2 七天无理由退货

**核心依据**：《消费者权益保护法》第25条、《网络购买商品七日无理由退货暂行办法》

- 消费者自收到商品之日起**七日内**可退货，**无需说明理由**
- 退回商品的运费由**消费者承担**（经营者另有约定的除外）
- **不适用七天无理由退货的商品**：
  - 定作的商品
  - 鲜活易腐的商品
  - 在线下载或拆封的数字化商品（音像制品、软件等）
  - 交付的报纸、期刊
  - 其他根据商品性质不宜退货的商品（商家需提前确认并消费者同意）
- **重要**：消费者因检查商品需要拆封查验，**不影响商品完好**的，商家不得以"已拆封"为由拒绝退货

### 6.3 假冒伪劣商品

**核心依据**：《消费者权益保护法》第55条、《电子商务法》第38条

- 购买到假冒伪劣商品，可主张**退一赔三**（最低500元）
- 平台**不能提供**销售者真实名称、地址和联系方式的，消费者可向**平台索赔**
- 平台明知或应知商家售假，未采取必要措施的，承担**连带责任**
- **维权步骤**：
  1. 保存商品实物、开箱视频、订单截图
  2. 向平台投诉举报假冒商品
  3. 向12315投诉（全国12315平台可在线提交）
  4. 涉嫌犯罪的，向公安机关报案

### 6.4 虚假宣传/刷单炒信

**核心依据**：《电子商务法》第17条、《反不正当竞争法》第8条

- 商家不得以虚构交易、编造用户评价等方式进行虚假宣传
- 发现刷单炒信、虚假好评的，可向**市场监管部门**举报
- 因虚假宣传导致误购的，可主张**退一赔三**

### 6.5 价格欺诈

**核心依据**：《消费者权益保护法》第55条、《价格法》第14条

- 常见价格欺诈行为：
  - 虚构原价（标高原价再打折）
  - 低价促销但实际不发货
  - "仅限今日"等虚假限时促销
  - 不同用户不同价格（大数据杀熟）
- 可主张**退一赔三**

### 6.6 预售/定金纠纷

**核心依据**：《电子商务法》第49条、《消费者权益保护法》第26条

- 预售定金规则须**明确告知**消费者
- 商家单方面取消预售订单的，应**双倍返还定金**
- 预售商品实际与宣传严重不符的，消费者有权**取消订单并退款**
- "定金不退"等格式条款不得排除消费者权利

### 6.7 平台不作为/客服推诿

**核心依据**：《电子商务法》第38条、第61条、第62条

- 平台有义务**协助消费者维权**
- 平台应提供**原始合同和交易记录**
- 平台未对商家资质进行审核造成消费者损害的，承担**相应责任**
- **维权途径**：平台投诉 → 12315 → 互联网法院

### 6.8 直播带货维权

**核心依据**：《电子商务法》、《消费者权益保护法》、《广告法》

- 直播间销售商品适用**七天无理由退货**
- 直播主播对商品作虚假宣传的，消费者可向**主播和商家**追责
- 平台（抖音/快手等）未对直播间经营者进行资质审核的，承担**相应责任**
- **证据保存**：直播录屏、商品链接截图、交易记录、聊天记录

### 6.9 维权渠道

| 渠道 | 电话/网址 | 适用场景 |
|------|----------|---------|
| 平台内投诉 | 各平台APP客服中心 | 首选，先走平台流程 |
| 全国12315平台 | 12315 / https://www.12315.cn | 平台不处理或处理不满意 |
| 市场监管局 | 12345转市场监管 | 商品质量、虚假宣传、价格欺诈 |
| 互联网法院 | 各地互联网法院 | 金额较大或平台不配合 |
| 消费者协会 | 12315 | 调解消费纠纷 |

### 6.10 证据保存要点

1. **下单前截图**：商品页面、价格、宣传内容、商家承诺（页面可能被修改或删除）
2. **开箱视频**：收到商品时录制开箱全过程（尤其贵重商品）
3. **聊天记录**：与商家/客服的沟通记录
4. **交易记录**：订单详情、支付凭证、物流信息
5. **商品实物**：保留原包装、商品实物、鉴定报告（如涉及假货）
