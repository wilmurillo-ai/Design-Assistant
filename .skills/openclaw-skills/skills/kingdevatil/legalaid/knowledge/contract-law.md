# 合同法/民法典要点

> 来源：《中华人民共和国民法典》（2021年1月1日施行），合同编、物权编、侵权责任编
> 官方查询：http://www.npc.gov.cn

---

## 一、合同编通则

### 第469条：合同形式
当事人订立合同，可以采用书面形式、口头形式或者其他形式。

### 第471条：合同订立方式
当事人订立合同，可以采取要约、承诺方式或者其他方式。

### 第490条：合同成立
当事人采用合同书形式订立合同的，自当事人均签名、盖章或者按指印时合同成立。

### 第496条：格式条款
格式条款是当事人为了重复使用而预先拟定，并在订立合同时未与对方协商的条款。采用格式条款订立合同的，提供格式条款的一方应当遵循公平原则确定当事人之间的权利和义务，并采取合理的方式提示对方注意免除或者减轻其责任等与对方有重大利害关系的条款，按照对方的要求，对该条款予以说明。

### 第497条：格式条款无效
有下列情形之一的，该格式条款无效：
- 提供格式条款一方不合理地免除或者减轻其责任、加重对方责任、限制对方主要权利
- 提供格式条款一方排除对方主要权利

### 第500条：缔约过失责任
当事人在订立合同过程中有下列情形之一，造成对方损失的，应当承担赔偿责任：
- 假借订立合同，恶意进行磋商
- 故意隐瞒与订立合同有关的重要事实或者提供虚假情况
- 有其他违背诚信原则的行为

### 第509条：合同履行原则
当事人应当按照约定全面履行自己的义务。当事人应当遵循诚信原则，根据合同的性质、目的和交易习惯履行通知、协助、保密等义务。

## 二、租赁合同（第三编第十四章）

### 第703条：租赁合同定义
租赁合同是出租人将租赁物交付承租人使用、收益，承租人支付租金的合同。

### 第705条：租赁期限
租赁期限不得超过**二十年**。超过二十年的，超过部分无效。

### 第711条：承租人合理使用义务
承租人未按照约定的方法或者未根据租赁物的性质使用租赁物，致使租赁物受到损失的，出租人可以解除合同并请求赔偿损失。

### 第714条：承租人保管义务
承租人应当妥善保管租赁物，因保管不善造成租赁物毁损、灭失的，应当承担赔偿责任。

### 第716条：转租
承租人经出租人同意，可以将租赁物转租给第三人。

### 第722条：租金支付
承租人无正当理由未支付或者迟延支付租金的，出租人可以请求承租人在合理期限内支付；承租人逾期不支付的，出租人可以解除合同。

### 第725条：买卖不破租赁
租赁物在承租人按照租赁合同占有期限内发生所有权变动的，不影响租赁合同的效力。

### 第726条：优先购买权
出租人出卖租赁房屋的，应当在出卖之前的合理期限内通知承租人，承租人享有以同等条件优先购买的权利。

### 第730条：不定期租赁
当事人对租赁期限没有约定或者约定不明确，依据本法第五百一十条的规定仍不能确定的，视为不定期租赁。当事人可以随时解除合同，但是应当在合理期限之前通知对方。

## 三、租房维权常见问题

### 押金退还
- 合同到期或解除后，出租人应退还押金（扣除合理费用后）
- 出租人不得以"房屋自然损耗"为由扣押金
- 如出租人无正当理由拒绝退还，承租人可起诉

### 房屋质量问题
- 出租人应保证租赁物符合约定用途（第708条）
- 租赁物危及安全健康的，承租人可随时解除合同（第731条）
- 出租人未履行维修义务的，承租人可以自行维修，维修费用由出租人负担（第713条）

### 违规出租
- 擅自改变房屋结构（如违规隔断）
- 不符合安全标准（如消防不合格）
- 群租房超过规定人数
- 可向住建部门投诉

### 提前退租
- 如合同约定违约金，按合同执行
- 如违约金过高（超过实际损失30%），可请求法院适当减少
- 建议先与房东协商，争取减少损失

### 押金纠纷赔偿标准
- 押金一般不超过一个月至三个月租金
- 出租人无正当理由不退还押金，可主张：
  - 退还全部押金
  - 押金占用期间的利息
  - 因押金纠纷产生的其他实际损失

## 四、侵权责任编（第七编）

### 第1165条：一般侵权责任
行为人因过错侵害他人民事权益造成损害的，应当承担侵权责任。

### 第1167条：侵权行为方式
侵权行为危及他人人身、财产安全的，被侵权人有权请求侵权人承担停止侵害、排除妨碍、消除危险等侵权责任。

### 第1179条：人身损害赔偿
侵害他人造成人身损害的，应当赔偿医疗费、护理费、交通费、营养费、住院伙食补助费等为治疗和康复支出的合理费用，以及因误工减少的收入。造成残疾的，还应当赔偿辅助器具费和残疾赔偿金；造成死亡的，还应当赔偿丧葬费和死亡赔偿金。

### 第1182条：财产损害赔偿
侵害他人人身权益造成财产损失的，按照被侵权人因此受到的损失或者侵权人因此获得的利益赔偿。

### 第1183条：精神损害赔偿
侵害自然人人身权益造成严重精神损害的，被侵权人有权请求精神损害赔偿。

### 第1184条：财产损失计算
侵害他人财产的，财产损失按照损失发生时的市场价格或者其他合理方式计算。

### 第1194条：网络侵权责任
网络用户、网络服务提供者利用网络侵害他人民事权益的，应当承担侵权责任。

### 第1202条：产品责任
因产品存在缺陷造成他人损害的，生产者应当承担侵权责任。

## 五、常见合同/侵权维权场景

| 场景 | 适用法条 | 维权要点 |
|------|---------|---------|
| 租房押金不退 | 第703条 + 合同约定 | 协商→投诉住建局→起诉 |
| 房屋质量问题 | 第708条 + 第713条 | 出租人维修义务，承租人可自修后追偿 |
| 违规隔断出租 | 地方租赁条例 | 向住建部门投诉举报 |
| 霸王条款 | 第497条 | 格式条款无效 |
| 产品质量缺陷 | 第1202条 | 可向生产者或销售者索赔 |
| 网络侵权 | 第1194条 | 通知平台删除，平台不作为承担连带责任 |
| 精神损害 | 第1183条 | 严重精神损害可主张赔偿 |
| 人身损害 | 第1179条 | 医疗费+误工费+护理费+残疾/死亡赔偿金 |
| 缔约欺诈 | 第500条 | 可主张缔约过失责任赔偿 |
| 违约责任 | 第577条 | 继续履行/采取补救措施/赔偿损失 |
