# 交通事故维权知识库

> 本文件涵盖交通事故责任认定、赔偿标准、保险理赔、伤残鉴定、诉讼等场景的维权法律依据和操作指引。
> 所有法规均来源于中国政府网（gov.cn）及主管部门官网。

---

## 二、事故现场处理

### 2.1 发生事故后的正确处理流程

1. **立即停车**：开启危险报警闪光灯
2. **设置警示**：在车后放置三角警示牌（常规道路50-100米，高速公路150米外）
3. **抢救伤员**：如有人员受伤，立即拨打120急救
4. **报警**：拨打122交通事故报警（有人伤或财产损失较大必须报警）
5. **保护现场**：不要随意移动车辆和物品（抢救伤员除外）
6. **拍照取证**：全方位拍摄事故现场（车辆位置、碰撞部位、路面标线、交通标志等）
7. **交换信息**：记录对方姓名、电话、车牌号、保险信息
8. **寻找证人**：记录目击者联系方式

### 2.2 可自行协商的情形

**同时满足以下条件的轻微事故可自行协商**：
- 仅造成轻微财产损失
- 双方对事实和成因无争议
- 车辆可以移动

**操作**：拍照后立即移至不妨碍交通的地点，填写《机动车交通事故快速处理协议书》或使用"交管12123"APP处理。

### 2.3 必须报警的情形

- 造成人员伤亡
- 车辆无法移动
- 对事实或责任有争议
- 一方无驾驶证、饮酒、服用管制药品
- 撞击公共设施
- 对方逃逸

## 三、责任认定

### 3.1 责任划分标准

**核心依据**：《道路交通事故处理程序规定》第六十条

| 责任类型 | 适用情形 |
|---------|---------|
| 全部责任 | 一方过错导致事故 |
| 主要责任 | 两方以上过错，一方过错为主 |
| 同等责任 | 双方过错程度相当 |
| 次要责任 | 两方以上过错，一方过错为辅 |
| 无责任 | 无过错 |

### 3.2 交通事故认定书

- 公安机关交通管理部门应自现场调查之日起**10日内**制作交通事故认定书（需检验鉴定的，鉴定结论确定后5日内）
- 认定书载明：事故发生经过、原因分析、责任划分
- **对认定书不服的救济**：
  - 认定书属于证据，不是行政行为，**不能申请行政复议或行政诉讼**
  - 可在收到认定书之日起**3日内**提出书面复核申请
  - 复核申请向上一级公安机关交通管理部门提出
  - 在诉讼中可对认定书提出异议，法院可重新认定责任

### 3.3 常见责任认定争议

| 场景 | 一般认定规则 |
|------|------------|
| 追尾事故 | 后车全责 |
| 变道事故 | 变道方全责或主责 |
| 十字路口事故 | 未让行方主责或全责 |
| 酒驾/醉驾 | 酒驾方全责，且承担刑事责任 |
| 逃逸 | 逃逸方全责 |
| 行人闯红灯 | 行人承担主要责任，机动车承担次要责任 |
| 非机动车与机动车 | 机动车承担无过错责任（有证据证明行人有过错的减轻） |

## 四、赔偿标准

### 4.1 交强险赔偿限额

**核心依据**：《机动车交通事故责任强制保险条例》

| 赔偿项目 | 有责任限额 | 无责任限额 |
|---------|-----------|-----------|
| 死亡伤残赔偿 | 18万元 | 1.8万元 |
| 医疗费用赔偿 | 1.8万元 | 1800元 |
| 财产损失赔偿 | 2000元 | 100元 |

### 4.2 商业三者险

- 交强险赔偿不足部分，由商业三者险按责任比例赔偿
- 无商业三者险的，由侵权人自行承担

### 4.3 人身损害赔偿项目

**核心依据**：《民法典》第1179条、最高法院人身损害赔偿司法解释

| 赔偿项目 | 计算方式 |
|---------|---------|
| 医疗费 | 实际发生的医疗费用（含后续治疗费） |
| 误工费 | 误工天数 × 日工资（需提供收入证明） |
| 护理费 | 护理天数 × 护理人员日工资 |
| 住院伙食补助费 | 住院天数 × 当地出差伙食补助标准（一般100元/天） |
| 营养费 | 根据伤残情况和医疗机构意见 |
| 交通费 | 实际发生的交通费用 |
| 住宿费 | 实际发生的住宿费用 |
| 残疾赔偿金 | 上年度人均可支配收入 × 伤残系数 × 赔偿年限 |
| 死亡赔偿金 | 上年度人均可支配收入 × 20年 |
| 被扶养人生活费 | 根据扶养人丧失劳动能力程度 |
| 精神损害抚慰金 | 根据伤残等级确定（一般5000-50000元） |
| 丧葬费 | 上年度职工月平均工资 × 6个月 |
| 残疾辅助器具费 | 按照普通适用器具的合理费用 |
| 车辆维修费 | 实际维修费用 |
| 停运损失 | 合理的停运期间损失（营运车辆） |

### 4.4 伤残等级与赔偿系数

| 伤残等级 | 赔偿系数 |
|---------|---------|
| 一级伤残 | 100% |
| 二级伤残 | 90% |
| 三级伤残 | 80% |
| 四级伤残 | 70% |
| 五级伤残 | 60% |
| 六级伤残 | 50% |
| 七级伤残 | 40% |
| 八级伤残 | 30% |
| 九级伤残 | 20% |
| 十级伤残 | 10% |

## 五、伤残鉴定

### 5.1 鉴定时机

- 一般在治疗终结后进行
- 骨折等损伤需等待骨折愈合、功能恢复稳定后
- 通常在事故后**3-6个月**进行

### 5.2 鉴定机构

- 向交警部门或法院委托的司法鉴定机构申请
- 选择具有法医临床鉴定资质的司法鉴定机构
- 可在司法部官网查询鉴定机构名录

### 5.3 鉴定材料

- 事故认定书
- 住院病历、出院小结、手术记录
- 影像学检查报告（X光、CT、MRI）
- 误工证明、收入证明
- 护理证明

## 六、保险理赔

### 6.1 理赔流程

```
1. 报案（48小时内通知保险公司）
2. 现场勘查 / 定损
3. 提交理赔材料
4. 保险公司审核
5. 赔付
```

### 6.2 理赔材料

- 交通事故认定书
- 驾驶证、行驶证
- 保险单
- 医疗费发票、费用清单
- 诊断证明、病历
- 伤残鉴定报告（如涉及）
- 误工证明、收入证明
- 车辆维修发票、定损单

### 6.3 理赔争议

- 保险公司定损金额过低：可申请第三方评估或司法鉴定
- 保险公司拒赔：索要书面拒赔通知书，向12378投诉或提起诉讼
- 对方保险公司拖延：可向法院起诉对方车主和保险公司

## 七、诉讼维权

### 7.1 管辖法院

- **事故发生地**法院
- **被告住所地**法院
- **车辆登记地**法院
- 原告可选择最方便的法院起诉

### 7.2 被告确定

| 情形 | 被告 |
|------|------|
| 对方有保险 | 对方车主 + 对方保险公司（交强险+商业三者险） |
| 对方无保险 | 对方车主 |
| 借用车辆 | 驾驶人 + 车主（车主有过错的承担连带责任） |
| 雇员驾驶 | 驾驶人 + 用人单位 |
| 挂靠车辆 | 挂靠人 + 被挂靠企业 |
| 套牌车辆 | 套牌使用者 + 套牌出借人 |

### 7.3 诉讼时效

- 人身损害赔偿诉讼时效：**3年**（《民法典》第188条）
- 从知道或者应当知道权利被侵害之日起计算
- 建议在事故认定书作出后尽快起诉

## 八、肇事逃逸

### 8.1 肇事逃逸的法律后果

- **行政处罚**：罚款200-2000元，吊销驾驶证，终身禁驾（致人死亡）
- **刑事责任**：交通肇事逃逸的，处3年以上7年以下有期徒刑；因逃逸致人死亡的，处7年以上有期徒刑
- **民事责任**：逃逸方承担全部责任，交强险先赔付，不足部分由逃逸方承担

### 8.2 被逃逸后的处理

1. 立即报警122，提供逃逸车辆信息（车牌号、车型、颜色、逃跑方向）
2. 寻找目击证人
3. 查看周边监控（道路监控、商铺监控、行车记录仪）
4. 如受伤，先救治后追责
5. 无交强险或逃逸车辆未查获的，可申请**道路交通事故社会救助基金**先行垫付抢救费用

## 十、证据保存要点

1. **事故现场**：全方位拍照录像（车辆位置、碰撞部位、路面情况、交通标志）
2. **事故认定书**：妥善保管原件
3. **医疗记录**：所有就诊病历、检查报告、费用票据
4. **收入证明**：劳动合同、工资流水、单位误工证明
5. **鉴定报告**：伤残鉴定书原件
6. **维修票据**：车辆维修发票、定损单
7. **沟通记录**：与对方、保险公司的沟通记录
