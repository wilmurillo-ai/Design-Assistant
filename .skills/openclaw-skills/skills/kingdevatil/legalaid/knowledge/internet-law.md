## 一、个人信息保护法（2021年11月1日施行）

### 第4条：个人信息定义
个人信息是以电子或者其他方式记录的与已识别或者可识别的自然人有关的各种信息，不包括匿名化处理后的信息。

**个人信息包括但不限于**：
- 个人基本信息：姓名、性别、年龄、身份证号码、电话号码、住址等
- 账户信息：网银账号、支付账号、社交账号、邮箱账号等
- 通信信息：通讯录、通话记录、短信记录、聊天记录等
- 个人生物识别信息：指纹、人脸、声纹、虹膜等
- 行踪信息：位置信息、行踪轨迹等
- 网络行为信息：浏览记录、搜索记录、购物记录等

### 第5条：处理原则
处理个人信息应当遵循合法、正当、必要和诚信原则，不得通过误导、欺诈、胁迫等方式处理个人信息。

### 第6条：最小必要原则
处理个人信息应当具有明确、合理的目的，并与处理目的直接相关。采取对个人权益影响最小的方式。收集个人信息应当限于实现处理目的的最小范围，不得过度收集个人信息。

### 第13条：知情同意规则
处理个人信息应当取得个人的同意。同意应当由个人在充分知情的前提下自愿、明确作出。

### 第28条：敏感个人信息
敏感个人信息是一旦泄露或者非法使用，容易导致自然人的人格尊严受到侵害或者人身、财产安全受到危害的个人信息，包括：
- 生物识别信息
- 宗教信仰
- 特定身份
- 医疗健康
- 金融账户
- 行踪轨迹
- 不满十四周岁未成年人的个人信息

### 第29条：单独同意
处理敏感个人信息应当取得个人的**单独同意**。

### 第24条：自动化决策
通过自动化决策方式向个人进行信息推送、营销，应当同时提供不针对其个人特征的选项或提供便捷的拒绝方式。

### 第44条：知情权和决定权
个人对其个人信息的处理享有**知情权**和**决定权**。

### 第45条：查阅权和复制权
个人有权向个人信息处理者查阅、复制其个人信息。

### 第46条：更正权和补充权
个人发现其个人信息不准确或者不完整的，有权请求个人信息处理者更正、补充。

### 第47条：删除权
有下列情形之一的，个人信息处理者应当主动删除个人信息；个人信息处理者未删除的，个人有权请求删除：
- 处理目的已实现、无法实现或者不再必要
- 个人信息处理者停止提供产品或者服务，或者保存期限已届满
- 撤回同意
- 个人信息处理者违反法律、行政法规或者违反约定处理个人信息
- 法律、行政法规规定的其他情形

### 第48条：解释说明权
个人有权请求个人信息处理者对其个人信息处理规则进行解释说明。

### 个人其他权利
- **可携带权**
- **拒绝自动化决策权**

### 第62条：投诉举报
个人认为其个人信息权益受到侵害的，有权向履行个人信息保护职责的部门进行投诉、举报。收到投诉、举报的部门应当依法及时处理。

### 第66条：违法处罚
违反本法规定处理个人信息，侵害他人民事权益的，应当依法承担民事责任。违反本法规定，构成违反治安管理行为的，依法给予治安管理处罚；构成犯罪的，依法追究刑事责任。

## 二、网络安全法（2017年6月1日施行）

### 第22条：网络产品和服务安全
网络产品、服务应当符合相关国家标准的强制性要求。网络产品、服务的提供者不得设置恶意程序；发现其网络产品、服务存在安全缺陷、漏洞等风险时，应当立即采取补救措施。

### 第40条：网络运营者个人信息保护义务
网络运营者应当对其收集的用户信息严格保密，并建立健全用户信息保护制度。

### 第41条：收集使用个人信息规则
网络运营者收集、使用个人信息，应当遵循合法、正当、必要的原则，公开收集、使用规则，明示收集、使用信息的目的、方式和范围，并经被收集者同意。

### 第42条：不得泄露篡改毁损
网络运营者不得泄露、篡改、毁损其收集的个人信息；未经被收集者同意，不得向他人提供个人信息。

### 第43条：删除更正权
个人发现网络运营者违反法律、行政法规的规定或者双方的约定收集、使用其个人信息的，有权要求网络运营者**删除**其个人信息；发现网络运营者收集、存储的其个人信息有错误的，有权要求网络运营者**更正**。

### 第44条：不得非法买卖
任何个人和组织不得窃取或者以其他非法方式获取个人信息，不得非法出售或者非法向他人提供个人信息。

### 第46条：禁止发送骚扰信息
任何个人和组织应当对其网络行为负责，不得设立用于实施诈骗、传授犯罪方法、制作或者销售违禁物品、管制物品等违法犯罪活动的网站、通讯群组，不得利用网络发布涉及实施诈骗，制作或者销售违禁物品、管制物品以及其他违法犯罪活动的信息。

### 第64条：违法处罚
窃取或者以其他非法方式获取、非法出售或者非法向他人提供个人信息，尚不构成犯罪的，由公安机关没收违法所得，并处违法所得一倍以上十倍以下罚款，没有违法所得的，处一百万元以下罚款。

## 三、数据安全法（2021年9月1日施行）

### 第27条：开展数据处理活动应依法
开展数据处理活动应当依照法律、法规的规定，建立健全全流程数据安全管理制度。

### 第32条：任何组织个人有权投诉举报
任何组织、个人有权对违法开展数据处理活动进行投诉、举报。收到投诉、举报的部门应当及时依法处理。

## 四、电信用户权益保护

### 4.1 电信条例

#### 第31条：电信服务标准
电信用户有权知悉使用电信服务费用的有关情况。电信业务经营者应当按照国家有关电信服务的标准提供电信服务。

#### 第40条：交费和查询
电信用户应当按照约定的时间和方式及时、足额地向电信业务经营者交纳电信费用；电信用户有权查询和要求电信业务经营者提供电信费用清单。

#### 第41条：退网自由
电信用户有权要求电信业务经营者提供电信费用清单。电信用户对交纳电信费用有异议的，电信业务经营者应当予以答复。

### 4.2 电信用户申诉处理办法

#### 申诉条件
- 已向电信业务经营者投诉且对其处理结果不满意
- 或者向电信业务经营者投诉后15日内未得到答复
- 才能向工信部或通信管理局申诉

#### 申诉渠道
- 电话：**12300**
- 网站：https://dxss.miit.gov.cn
- 工信部政务服务平台：https://ythzxfw.miit.gov.cn
- 微信小程序：搜索"工信部12300"

#### 申诉流程
1. **先向运营商投诉**：拨打运营商客服电话（10086/10010/10000）
2. **等待处理**：运营商应在15日内答复
3. **不满意或未答复**：向工信部/通信管理局申诉
4. **工信部转办**：转交属地通信管理局处理
5. **处理反馈**：一般15-30个工作日内反馈结果

## 五、APP侵害用户权益治理

### 5.1 工信部APP专项整治

工信部持续开展APP侵害用户权益专项整治行动，重点整治以下问题：

| 整治类别 | 具体问题 |
|---------|---------|
| **违规收集个人信息** | 未明示收集目的、超范围收集、强制收集 |
| **超范围使用个人信息** | 未按约定使用、未经同意向第三方提供 |
| **违规索取权限** | 强制索权、频繁索权、拒绝授权限制功能 |
| **欺骗误导用户** | 误导下载、强制跳转、虚假宣传 |
| **账号注销难** | 不提供注销渠道、注销条件苛刻 |
| **过度索取权限** | 与功能无关的权限申请 |
| **频繁弹窗骚扰** | 弹窗无法关闭、诱导点击 |
| **APP强制捆绑** | 强制安装、捆绑安装其他应用 |

### 5.2 APP举报渠道

| 渠道 | 方式 | 说明 |
|------|------|------|
| **工信微报**微信公众号 | 底部服务栏→我要投诉→手机应用软件 | 工信部官方渠道 |
| **12321受理中心**微信公众号 | 底部服务栏→其他投诉→手机应用 | 工信部下属举报中心 |
| **互联网信息服务投诉平台** | https://hlwtsxt.miit.gov.cn | 中国互联网协会运营 |
| **12321举报中心** | https://www.12321.cn | 网络不良信息举报 |
| **12300电话** | 拨打12300 | 电信用户申诉 |

### 5.3 举报准备材料
- APP名称、版本号、下载平台
- 侵权行为的具体描述
- 截图或录屏证据
- 手机型号和操作系统版本

## 六、互联网信息服务投诉平台

**网址**：https://hlwtsxt.miit.gov.cn
**运营**：中国互联网协会
**上线时间**：2019年7月

### 受理范围
- **个人信息保护**：APP违规收集使用个人信息、信息泄露等
- **服务功能**：互联网服务功能缺陷、服务中断等
- **客服渠道**：企业投诉渠道不畅通

### 不受理范围
- 网络交易纠纷、网购假冒伪劣 → 向12315投诉
- 虚假广告、欺诈性商业行为 → 向12315投诉
- 金融产品和服务的财产安全 → 向12378投诉
- 版权问题 → 向版权主管部门投诉

### 投诉流程
1. 访问平台首页，点击"我要投诉"
2. 填写投诉信息（可注册登录或游客投诉）
3. 选择投诉类别和具体问题
4. 描述问题并上传证据
5. 提交后等待处理反馈

## 八、常见互联网维权场景速查表

| 场景 | 适用法律 | 首选渠道 | 维权要点 |
|------|---------|---------|---------|
| APP强制索权/过度收集信息 | 个人信息保护法第6、13条 | 12321/互联网投诉平台 | 截图录屏+APP信息 |
| 个人信息泄露 | 个人信息保护法第44、45条 + 网络安全法第42条 | 12377网信办/12321 | 保留泄露证据+要求删除 |
| 大数据杀熟/算法歧视 | 个人信息保护法第24条 | 12377/12315 | 对比截图+投诉 |
| APP账号注销难 | 个人信息保护法第47条 | 互联网投诉平台/12321 | 要求提供注销渠道 |
| 骚扰弹窗/强制跳转 | 个人信息保护法 + 网络安全法 | 12321/互联网投诉平台 | 截图录屏 |
| 手机扣费异常/套餐纠纷 | 电信条例 + 电信用户申诉办法 | 先运营商→12300 | 话费清单+投诉记录 |
| 宽带故障不修 | 电信用户申诉办法 | 先运营商→12300 | 报修记录+投诉记录 |
| 携号转网受阻 | 携号转网服务管理规定 | 12300 | 运营商拒绝记录 |
| 网络诈骗 | 刑法第266条 | 110报警 | 转账记录+聊天记录 |
| 网络暴力/人肉搜索 | 民法典第1032-1034条 + 网络安全法 | 110报警/12377 | 证据固定+报警 |
| 虚假宣传/误导下载 | 消费者权益保护法 + 反不正当竞争法 | 12315/互联网投诉平台 | 截图+宣传页面 |
| 平台不作为（如电商、社交平台） | 民法典第1194-1197条 | 12315/互联网法院 | 通知平台记录+平台不作为证据 |
| 网络内容侵权（版权、名誉等） | 民法典 + 著作权法 | 12377/互联网法院 | 侵权内容截图+权属证明 |
| AI生成内容侵权 | 个人信息保护法 + 民法典 | 12377/互联网法院 | AI生成内容证据+损害证明 |

## 九、互联网维权全流程

```
发现互联网/通信权益受损
    ↓
【Step 1】固定证据
    → 截图、录屏、保存页面、导出数据
    → 记录APP名称、版本号、下载平台
    → 保存通话记录、聊天记录、话费清单
    ↓
【Step 2】向企业投诉
    → APP内投诉渠道 / 客服电话 / 官方邮箱
    → 运营商客服（10086/10010/10000）
    → 保留投诉记录和工单号
    ↓
【Step 3】向监管部门投诉
    → 根据问题类型选择渠道：
    ├─ 通信服务问题 → 12300（需先向运营商投诉）
    ├─ APP侵权问题 → 12321 / 互联网信息服务投诉平台
    ├─ 个人信息问题 → 12377网信办
    ├─ 网络交易问题 → 12315
    └─ 金融安全问题 → 12378
    ↓
【Step 4】等待处理
    → 工信部/通信管理局：一般15-30个工作日
    → 网信办：依法及时处理
    → 12315：7个工作日内决定是否受理
    ↓
【Step 5】不满意则升级
    → 向上级主管部门反映
    → 向互联网法院提起诉讼
    → 涉嫌犯罪的向公安机关报案
```
