# 医疗医药维权知识库

> 本文件涵盖医疗服务纠纷、医疗事故、药品安全、医疗器械、美容医疗等场景的维权法律依据和操作指引。
> 所有法规均来源于中国政府网（gov.cn）及主管部门官网。

---

## 一、医疗纠纷维权

### 1.1 主要法律依据

| 法规名称 | 发布机构 | 关键内容 |
|---------|---------|---------|
| 《中华人民共和国民法典》第七编侵权责任 | 全国人大常委会 | 医疗损害责任（第1218-1228条） |
| 《医疗纠纷预防和处理条例》（国务院令第701号） | 国务院 | 医疗纠纷预防、处理途径、鉴定、赔偿 |
| 《医疗事故处理条例》（国务院令第351号） | 国务院 | 医疗事故分级、鉴定程序、赔偿标准 |
| 《中华人民共和国医师法》 | 全国人大常委会 | 医师执业规范、权利义务 |

### 1.2 患者的权利

| 权利 | 内容 | 法律依据 |
|------|------|---------|
| 知情同意权 | 知情病情、治疗方案、风险、替代方案、费用 | 《民法典》第1219条 |
| 隐私权 | 病历信息、个人隐私受保护 | 《民法典》第1226条、《医师法》第31条 |
| 病历查阅复制权 | 患者有权查阅和复制病历资料 | 《民法典》第1225条 |
| 拒绝治疗权 | 患者有权拒绝不必要的治疗（需签署知情同意书） | 《民法典》第1224条 |
| 投诉举报权 | 对医疗服务不满意有权投诉举报 | 《医疗纠纷预防和处理条例》第22条 |

### 1.3 医疗损害责任的认定

**核心依据**：《民法典》第1218-1228条

- **过错归责原则**：患者在诊疗活动中受到损害，医疗机构或医务人员有过错的，由医疗机构承担赔偿责任
- **推定过错情形**（第1222条），患者有损害且下列情形之一的，**推定医疗机构有过错**：
  - 违反法律、行政法规、规章以及其他有关诊疗规范的规定
  - 隐匿或者拒绝提供与纠纷有关的病历资料
  - 遗失、伪造、篡改或者违法销毁病历资料
- **免责情形**（第1224条）：
  - 患者或其近亲属不配合医疗机构进行符合诊疗规范的诊疗
  - 医务人员在抢救生命垂危的患者等紧急情况下已经尽到合理诊疗义务
  - 限于当时的医疗水平难以诊疗

### 1.4 医疗纠纷解决途径

**核心依据**：《医疗纠纷预防和处理条例》第22条

| 途径 | 适用场景 | 操作方式 |
|------|---------|---------|
| 双方协商 | 争议较小、事实清楚 | 与医院医务科/医患办直接协商 |
| 人民调解 | 大多数医疗纠纷（推荐首选） | 向当地**医疗纠纷人民调解委员会**申请 |
| 行政调解 | 协商不成、不愿起诉 | 向当地**卫生健康行政部门**申请 |
| 诉讼 | 调解不成、争议较大 | 向法院提起**医疗损害责任纠纷**诉讼 |

### 1.5 医疗事故鉴定

- **首次鉴定**：由设区的市级地方医学会和省、自治区、直辖市直接管辖的县（市）地方医学会负责
- **再次鉴定**：对首次鉴定不服的，自收到鉴定书之日起15日内向省、自治区、直辖市医学会申请
- **司法鉴定**：诉讼中可申请法院委托司法鉴定机构进行医疗损害鉴定

### 1.6 医疗损害赔偿项目

**核心依据**：《民法典》第1179条、《医疗事故处理条例》第五十条

| 赔偿项目 | 计算方式 |
|---------|---------|
| 医疗费 | 实际发生的医疗费用（含后续治疗费） |
| 误工费 | 误工天数 × 当日工资 |
| 护理费 | 护理天数 × 护理人员日工资 |
| 住院伙食补助费 | 住院天数 × 当地出差伙食补助标准 |
| 交通费 | 实际发生的交通费用 |
| 住宿费 | 实际发生的住宿费用 |
| 营养费 | 根据伤残情况和医疗机构意见确定 |
| 残疾赔偿金 | 伤残等级 × 当地上年度人均可支配收入 × 赔偿年限 |
| 死亡赔偿金 | 当地上年度人均可支配收入 × 20年 |
| 被扶养人生活费 | 根据扶养人丧失劳动能力程度确定 |
| 精神损害抚慰金 | 根据伤残等级或损害后果确定 |
| 丧葬费 | 当地上年度职工月平均工资 × 6个月 |

## 二、药品安全维权

### 2.1 主要法律依据

| 法规名称 | 发布机构 | 关键内容 |
|---------|---------|---------|
| 《中华人民共和国药品管理法》（2019年修订） | 全国人大常委会 | 药品生产经营、假药劣药认定、处罚标准 |
| 《中华人民共和国疫苗管理法》 | 全国人大常委会 | 疫苗研制、生产、流通、接种管理 |
| 《最高人民法院关于审理食品药品惩罚性赔偿纠纷案件适用法律若干问题的解释》 | 最高法院 | 食品药品惩罚性赔偿裁判标准 |

### 2.2 假药、劣药的认定

**核心依据**：《药品管理法》第98条

**假药**：
- 药品所含成份与国家药品标准规定的成份不符
- 以非药品冒充药品或者以他种药品冒充此种药品
- 变质的药品
- 药品所标明的适应症或者功能主治超出规定范围

**劣药**：
- 药品成份的含量不符合国家药品标准
- 被污染的药品
- 未标明或者更改有效期的药品
- 未注明或者更改产品批号的药品
- 超过有效期的药品
- 擅自添加防腐剂、辅料的药品

### 2.3 药品维权赔偿

**核心依据**：《药品管理法》第144条

- 购买到**假药、劣药**的：
  - 有权要求**退还价款**
  - 可主张**价款十倍或者损失三倍**的赔偿金；增加赔偿的金额不足一千元的，为一千元
  - 造成健康损害的，可要求赔偿**医疗费、误工费、精神损害抚慰金**等
- **明知是假药劣药仍然销售**的，构成欺诈，消费者可按《消法》第55条主张**退一赔三**

### 2.4 药品不良反应

- 用药后出现不良反应，应立即停药并就医
- 向药品不良反应监测中心报告（可通过医院或直接报告）
- 不良反应 ≠ 质量问题，需通过鉴定区分
- 如因药品质量问题导致损害，可向药品生产企业或销售者索赔

### 2.5 维权渠道

- **12315**：药品质量问题、药店销售假药劣药
- **12345**：转卫生健康部门或药品监管部门
- **药品监督管理局**：当地药监局投诉举报
- **国家药监局**：https://www.nmpa.gov.cn

## 三、医疗器械维权

### 3.1 主要法律依据

| 法规名称 | 发布机构 | 关键内容 |
|---------|---------|---------|
| 《医疗器械监督管理条例》（国务院令第739号） | 国务院 | 医疗器械注册、生产、经营、使用管理 |
| 《医疗器械经营监督管理办法》 | 市场监管总局 | 医疗器械经营规范 |

### 3.2 常见侵权场景

- 购买到**未注册/未备案**的医疗器械
- 医疗器械**质量不合格**导致损害
- 医疗器械**虚假宣传**（夸大功效）
- 医院使用**过期或不合格**医疗器械

### 3.3 维权要点

- 因医疗器械缺陷造成损害的，可向**生产者**或**医疗机构**索赔
- 向国家药监局查询医疗器械注册信息：https://www.nmpa.gov.cn
- 投诉渠道同药品安全维权

## 四、美容医疗维权

### 4.1 主要法律依据

| 法规名称 | 发布机构 | 关键内容 |
|---------|---------|---------|
| 《医疗美容服务管理办法》 | 卫健委 | 医疗美容机构、人员资质管理 |
| 《医疗纠纷预防和处理条例》 | 国务院 | 医疗美容纠纷处理 |
| 《消费者权益保护法》 | 全国人大常委会 | 生活消费中的美容服务 |

### 4.2 常见侵权场景

| 场景 | 性质 | 维权依据 |
|------|------|---------|
| 生活美容机构违规开展医疗美容（如注射、激光） | 非法行医 | 《医疗美容服务管理办法》第23条 |
| 医生无执业资质开展美容手术 | 非法行医 | 《医师法》 |
| 美容产品/药品为假药劣药 | 药品违法 | 《药品管理法》 |
| 美容效果与宣传严重不符 | 虚假宣传/欺诈 | 《消法》第55条 |
| 美容手术造成人身损害 | 医疗损害 | 《民法典》第1218条 |
| 美容机构拒绝提供病历 | 侵犯知情权 | 《民法典》第1225条 |

### 4.3 维权要点

1. **区分生活美容和医疗美容**：注射、手术、激光治疗等属于医疗美容，需有《医疗机构执业许可证》
2. **核实资质**：确认机构和医生是否有相应资质（卫健委官网可查）
3. **术前签署知情同意书**：仔细阅读风险告知内容
4. **保留证据**：术前术后照片、合同、付款凭证、聊天记录
5. **投诉渠道**：12315（消协）+ 12345（卫健委）+ 12320（卫生热线）

## 五、通用医疗维权要点

### 5.1 证据保存

- **病历资料**：及时要求复印和封存病历（住院病历、门诊病历、处方、检查报告等）
- **费用凭证**：医疗费发票、费用清单
- **现场证据**：药品包装、医疗器械实物、照片
- **沟通记录**：与医生/医院的沟通记录、录音
- **鉴定报告**：医疗事故鉴定书、司法鉴定书

### 5.2 病历封存（重要）

- 发生医疗纠纷后，应**第一时间**要求复印并封存病历
- 封存病历需医患双方在场，共同签字确认
- 如医院拒绝封存或疑似篡改病历，可向卫健委投诉
- 篡改/伪造病历的，**推定医疗机构有过错**（《民法典》第1222条）

### 5.3 维权流程建议

1. **固定证据**：复印封存病历、保存药品/器械实物
2. **协商解决**：与医院医务科/医患办协商
3. **第三方调解**：申请医疗纠纷人民调解委员会调解（免费、高效）
4. **行政投诉**：向当地卫健委投诉（12320 / 12345）
5. **医疗鉴定**：申请医疗事故鉴定或司法鉴定
6. **诉讼**：向法院提起医疗损害责任纠纷诉讼

### 5.4 注意事项

- 医疗损害赔偿**诉讼时效为3年**（《民法典》第188条）
- 医疗纠纷人民调解委员会调解**免费**
- 涉及死亡的，应在48小时内进行**尸检**（冷冻条件下可延长至7日）
- 医疗纠纷中，**医疗机构承担举证责任**（部分情形下适用举证责任倒置）
