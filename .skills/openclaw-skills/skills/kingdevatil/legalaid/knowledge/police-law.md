# 报警维权法律知识库

> 来源：《中华人民共和国刑事诉讼法》《中华人民共和国治安管理处罚法》《公安机关办理刑事案件程序规定》《公安机关办理行政案件程序规定》《公安机关办理刑事复议复核案件程序规定》《公安机关督察条例》
> 官方查询：http://www.npc.gov.cn / https://www.mps.gov.cn

## 一、报案与受案

### 1.1 刑事案件报案

#### 报案权利
任何单位和个人发现有犯罪事实或者犯罪嫌疑人，有权也有义务向公安机关报案（刑事诉讼法第110条）。

#### 公安机关受案义务
《公安机关办理刑事案件程序规定》第166条：公安机关对于公民扭送、报案、控告、举报或者犯罪嫌疑人自动投案的，**都应当立即接受**，问明情况，并制作笔录，经核对无误后，由扭送人、报案人、控告人、举报人、自动投案人签名、捺指印。必要时，应当录音或者录像。

#### 受案回执
《公安机关办理刑事案件程序规定》第171条：公安机关接受案件时，应当制作**受案登记表**和**受案回执**，并将受案回执交报案人、控告人、举报人、扭送人。

> **重要**：报案后，公安机关**必须**出具受案回执。如果不出具，属于违规。

### 1.2 行政案件报案

#### 受案义务
《公安机关办理行政案件程序规定》第60条：公安机关对报案、控告、举报、群众扭送或者违法嫌疑人投案，以及其他国家机关移送的案件，**应当及时受理**并按照规定进行网上接报案登记。

#### 受案回执
《公安机关办理行政案件程序规定》第61条：对属于本单位管辖范围内的案件，应当立即调查处理，制作**受案登记表**和**受案回执**，并将受案回执交报案人、控告人、举报人、扭送人。

#### 不属于公安机关管辖的处理
对不属于公安机关职责范围的事项，在接报案时能够当场判断的，应当**立即口头告知**报案人、控告人、举报人、扭送人向其他主管机关报案或者投案；当场无法判断的，应当在三日内书面告知。

## 二、立案

### 2.1 刑事案件立案条件

《刑事诉讼法》第112条 + 《公安机关办理刑事案件程序规定》第178条：

立案需同时满足**三个法定条件**：

| 条件 | 内涵 | 判断要点 |
|------|------|---------|
| **有犯罪事实** | 客观上存在危害社会的行为，且该行为符合某一犯罪构成要件 | 不是普通的民事纠纷，而是涉嫌犯罪 |
| **需要追究刑事责任** | 依法应当追究行为人的刑事责任 | 不存在免责情形（如追诉时效已过、嫌疑人死亡等） |
| **属于自己管辖** | 案件属于公安机关管辖范围 | 非自诉案件、非检察机关/法院直接受理案件 |

### 2.2 立案程序

1. **接受案件** → 制作受案登记表和受案回执
2. **审查** → 询问、勘验、鉴定等（一般应在3日内完成审查，复杂可延长）
3. **审批** → 经县级以上公安机关负责人批准
4. **决定** → 予以立案 或 不予立案

### 2.3 立案后的法律效果

- 侦查程序正式启动
- 可依法采取询问、讯问、勘验、检查、搜查、扣押、查询、冻结、技术侦查等措施
- 符合法定条件的可采取拘留、逮捕等强制措施
- 报案人可凭立案回执/立案通知书了解案件进展

## 三、不予立案

### 3.1 不予立案的条件

《公安机关办理刑事案件程序规定》第178条：

有下列情形之一的，不予立案：
- **没有犯罪事实**（不构成犯罪，属于民事纠纷等）
- **犯罪事实显著轻微**，不需要追究刑事责任
- 具有**其他依法不追究刑事责任**的情形：
  - 情节显著轻微、危害不大，不认为是犯罪的
  - 犯罪已过追诉时效期限的
  - 经特赦令免除刑罚的
  - 依照刑法告诉才处理的犯罪，没有告诉或者撤回告诉的
  - 犯罪嫌疑人、被告人死亡的
  - 其他法律规定免予追究刑事责任的

### 3.2 不予立案的程序要求

《公安机关办理刑事案件程序规定》第178条第二款：

对有控告人的案件，决定不予立案的，公安机关应当制作**不予立案通知书**，并在**三日以内**送达控告人。

> **重要**：公安机关不予立案的，**必须**出具书面不予立案通知书。如果不出具，属于违规。

### 3.3 不予立案的救济途径

控告人对不予立案决定不服的，可以在收到不予立案通知书后**七日以内**向作出决定的公安机关申请**复议**；公安机关应当在收到复议申请后**三十日以内**作出决定，并将决定书交控告人。

控告人对不予立案的复议决定不服的，可以在收到复议决定书后**七日以内**向**上一级公安机关**申请**复核**；上一级公安机关应当在收到复核申请后**三十日以内**作出决定。

> **法律依据**：《刑事诉讼法》第112条 + 《公安机关办理刑事案件程序规定》第179条 + 《公安机关办理刑事复议复核案件程序规定》

### 3.4 检察院立案监督（终极手段）

《刑事诉讼法》第113条：

> 人民检察院认为公安机关对应当立案侦查的案件而不立案侦查的，或者**被害人认为公安机关对应当立案侦查的案件而不立案侦查，向人民检察院提出的**，人民检察院应当要求公安机关说明不立案的理由。人民检察院认为公安机关不立案理由不能成立的，应当通知公安机关立案，**公安机关接到通知后应当立案**。

**操作要点**：
- 向**同级人民检察院**（侦查监督部门/第一检察部）提交立案监督申请书
- 提交材料：申请书 + 不予立案通知书（或复议决定书）+ 全套证据材料
- 检察院审查后，要求公安机关7日内书面说明不立案理由
- 检察院认为理由不成立的，出具《通知立案书》
- 公安机关接到通知后**必须立案**

> **注意**：申请检察院立案监督**不需要**先经过公安机关的复议/复核程序，可以直接申请。

## 四、受案/立案维权常见问题

### 4.1 报案后公安机关不受理、不出回执

**问题**：去派出所报案，民警说"不归我们管""不够立案标准""你去找别的部门"，不登记、不出回执。

**法律依据**：
- 《公安机关办理刑事案件程序规定》第166条：**都应当立即接受**
- 《公安机关办理刑事案件程序规定》第171条：**应当制作受案登记表和受案回执**

**维权路径**：
1. **当场要求**出具受案回执（这是法定义务）
2. 如被拒绝，拨打 **110** 报警记录（110报警记录本身就是证据）
3. 向**警务督察**投诉举报（12389）
4. 向**检察院**申请立案监督

### 4.2 应当立案而不立案

**问题**：明显涉嫌犯罪，但公安机关不予立案。

**维权路径**：
1. 要求出具**不予立案通知书**（法定必须出具）
2. 收到后7日内申请**复议**
3. 对复议不服，7日内向上一级公安机关申请**复核**
4. 或直接向**同级检察院**申请**立案监督**（推荐，最有效）
5. 向**警务督察**投诉（12389）

### 4.3 不应当立案而立案（被刑事手段插手民事纠纷）

**问题**：明明是民事纠纷，公安机关却刑事立案。

**维权路径**：
1. 向**同级检察院**申请立案监督（要求撤销案件）
2. 向**警务督察**投诉（12389）
3. 聘请律师提出申诉

### 4.4 立案后不侦查、不作为

**问题**：已经立案，但公安机关迟迟不侦查、不处理。

**维权路径**：
1. 向办案机关了解案件进展（凭立案回执）
2. 向**警务督察**投诉不作为（12389）
3. 向**检察院**反映，要求监督侦查进度
4. 向**信访部门**反映

## 五、警务督察

### 5.1 什么是警务督察

警务督察是公安机关内部监督机制，负责监督公安机关及其人民警察依法履行职责、行使职权和遵守纪律。

**法律依据**：《公安机关督察条例》

### 5.2 警务督察受理范围

- 公安机关及其人民警察**不履行法定职责**（如应当受案不受案、应当立案不立案）
- **违法行使职权**（如违规立案、违规采取强制措施）
- **不遵守纪律**（如态度恶劣、吃拿卡要、泄露案情）
- **其他违法违纪行为**

### 5.3 举报渠道

| 渠道 | 方式 | 说明 |
|------|------|------|
| **12389** | 电话举报 | 全国公安机关警务督察举报电话 |
| **12389举报平台** | 网上举报 | https://www.12389.gov.cn |
| **信函举报** | 邮寄 | 寄送至当地公安局警务督察部门 |
| **当面举报** | 现场到访 | 当地公安局警务督察支队/大队 |

### 5.4 举报注意事项

- **实名举报**优先处理，需提供本人身份信息
- 尽量提供**具体事实和证据**（时间、地点、涉事民警信息、相关文书等）
- 举报后可要求反馈处理结果
- 举报人信息依法保密
- 如对督察处理结果不满意，可向上级督察部门申诉

## 六、报案维权全流程图

```
发现犯罪事实/违法行为
    ↓
【Step 1】报案
    → 到派出所/拨打110/网上报案
    → 要求制作笔录
    ↓
【Step 2】索要受案回执
    → 公安机关必须出具受案回执
    → 如不出具 → 向警务督察投诉(12389)
    ↓
【Step 3】等待审查结果
    → 刑事案件：一般3日内审查完毕
    → 行政案件：及时受理
    ↓
    ├─→ 予以立案 → 进入侦查/调查程序
    │     → 凭立案回执了解案件进展
    │     → 如立案后不作为 → 向督察/检察院反映
    │
    └─→ 不予立案
          → 必须出具《不予立案通知书》（3日内送达）
          ↓
    【Step 4】不予立案救济
          ├─→ 路径A：7日内申请复议 → 不服 → 7日内申请复核
          ├─→ 路径B：直接向同级检察院申请立案监督（推荐）
          └─→ 路径C：向警务督察投诉(12389)
          ↓
    【Step 5】检察院立案监督
          → 检察院要求公安说明不立案理由
          → 理由不成立 → 通知公安立案 → 公安必须立案
          ↓
    【Step 6】如公安仍不立案
          → 向检察院申请监督执行
          → 向上级公安机关反映
          → 向纪检监察部门举报
```

## 七、报案材料准备清单

### 刑事案件报案材料
| 材料 | 说明 |
|------|------|
| 身份证原件及复印件 | 报案人身份证明 |
| 报案陈述（书面） | 详细描述案件经过（时间、地点、人物、经过、损失） |
| 证据材料 | 书证、物证、电子证据、录音录像等 |
| 证据清单 | 列明证据名称、类型、证明内容 |
| 损失清单 | 经济损失明细及相关凭证 |

### 行政案件报案材料
| 材料 | 说明 |
|------|------|
| 身份证原件及复印件 | 报案人身份证明 |
| 报案陈述（书面） | 详细描述违法事实经过 |
| 证据材料 | 书证、物证、电子证据等 |
| 损失清单 | 如有经济损失，列明明细 |

## 八、常见报警维权场景速查表

| 场景 | 案件类型 | 关键法条 | 维权要点 |
|------|---------|---------|---------|
| 报案后不出回执 | 程序违法 | 办案程序规定第166、171条 | 要求出具，拒不出具向督察投诉 |
| 应立案不立案 | 刑事案件 | 刑诉法第112、113条 | 复议→复核或直接检察院监督 |
| 不该立案而立案 | 刑事插手民事 | 刑诉法第113条 | 向检察院申请撤销案件监督 |
| 立案后不侦查 | 不作为 | 督察条例 | 向督察/检察院反映 |
| 民警态度恶劣/吃拿卡要 | 纪律违法 | 督察条例 | 向12389警务督察举报 |
| 案件管辖推诿 | 程序违法 | 办案程序规定第61条 | 要求出具书面不予受理决定并告知管辖机关 |
| 证据被销毁/不收集 | 侦查违法 | 刑诉法第56条（非法证据排除） | 向检察院申诉 |
| 违规采取强制措施 | 程序违法 | 刑诉法相关条款 | 向检察院申诉/向督察投诉 |
| 泄露报案人信息 | 侵犯隐私 | 督察条例+刑法 | 向督察举报，涉嫌犯罪的追究刑事责任 |
