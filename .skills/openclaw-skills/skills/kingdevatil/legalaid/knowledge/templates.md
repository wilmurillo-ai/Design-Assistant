# 维权文书模板与话术指引

> 本文件包含各类维权场景的文书模板和投诉/举报话术，供生成正式法律文书时参考。
> 模板格式来源于最高人民法院诉讼文书样式、国家信访局、市场监管总局等官方渠道。

---

## 一、消费者投诉书模板

**来源**：市场监管总局《市场监督管理投诉举报处理暂行办法》

```
消费者投诉书

一、投诉人信息
   姓名：XXX        性别：X        联系电话：XXX
   通讯地址：XXX                    邮编：XXX

二、被投诉人信息
   名称（姓名）：XXX
   地址：XXX
   联系电话：XXX（如有）

三、投诉请求
   1. （具体请求：退货、退款、赔偿等）
   2. （如有多项请求，逐条列出）

四、消费者权益争议事实
   （购买/服务的时间、地点、方式、商品/服务信息）
   （争议经过：发现问题→与商家沟通→商家处理态度）
   （权益受损情况及具体损失）

五、法律依据
   （引用相关法律条文，如《消费者权益保护法》第XX条）

六、附件材料
   1. 购物凭证/合同复印件
   2. 商品照片/问题截图
   3. 与商家沟通记录
   4. 其他证据材料

   投诉人（签名）：XXX
   日期：XXXX年XX月XX日
```

## 六、投诉/举报话术指引

### 6.1 消费者投诉话术（12315）

**电话投诉**：
> "您好，我要投诉[商家名称]。我在[时间]在[地点/平台]购买了[商品/服务]，花费[金额]元。发现问题后，[描述问题]。我已与商家协商但[商家态度/处理结果]。我的诉求是[退货/退款/赔偿]。我有[购物凭证/聊天记录/照片]等证据。"

**要点**：
- 说清：谁、什么时间、在哪里、买了什么、花了多少钱
- 说清：问题是什么、造成了什么损失
- 说清：已与商家沟通的结果
- 说清：具体诉求（退款多少、赔偿多少）

### 6.2 劳动维权投诉话术（12333）

> "您好，我要反映劳动权益问题。我在[公司名称]工作，入职时间[XXXX年XX月]，岗位是[XXX]。公司存在[拖欠工资/未签合同/违法辞退/不缴社保]等问题。具体情况是[描述]。我的诉求是[支付工资/双倍工资/赔偿金/补缴社保]。"

### 6.3 报警维权话术

**报案时**：
> "我要报案。在[时间]，在[地点]，[嫌疑人]对我实施了[具体行为]，造成了我[具体损害]。我有[证据描述]。请依法受理并出具受案回执。"

**报案被拒时**：
> "根据《刑事诉讼法》第112条，公安机关对于报案材料应当迅速进行审查，认为有犯罪事实需要追究刑事责任的时候，应当立案。请依法受理并出具受案回执。如不予立案，请出具不予立案通知书。"

**警务督察投诉（12389）**：
> "我要投诉[公安机关名称]。我在[时间]到该单位报案，但[具体不作为情况]。该行为违反了[具体法律规定]。请依法调查处理。"

### 6.4 行政投诉话术（12345）

> "您好，我要投诉[部门名称]。我在[时间]向该部门反映[事项]，但该部门[不作为/乱作为/超时未处理]。具体情况是[描述]。我的诉求是[依法处理/限期答复]。"

### 6.5 退保投诉话术（12378）

> "您好，我要投诉[保险公司名称]。我在[时间]购买了[保险产品]，保费[金额]元。当时销售人员[描述误导行为]。我发现后要求退保，但保险公司[拒绝/拖延/扣费过高]。我的诉求是[全额退保/退还保费]。"

## 七、补充模板

### 7.1 检察院立案监督申请书

**来源**：《刑事诉讼法》第113条、《人民检察院刑事诉讼规则》

```
立案监督申请书

申请人：XXX，性别X，XXXX年XX月XX日出生，
        身份证件号码：XXX，住址：XXX，联系电话：XXX

被申请机关：XXXX公安局/派出所，地址：XXX

申请事项：
请求贵院依法对XXXX公安局不予立案一案进行立案监督，督促其依法立案侦查。

事实和理由：
一、报案情况
   申请人于XXXX年XX月XX日向XXXX公安局报案，报案内容为[简述案件事实]。
   该局于XXXX年XX月XX日出具了不予立案通知书（文号：XXX）。

二、不予立案理由不成立
   被申请机关以[不予立案的理由]为由不予立案。但根据[法律条文及事实依据]，
   该案[存在犯罪事实/需要追究刑事责任]，不予立案理由不能成立。

三、应当依法立案
   根据《刑事诉讼法》第112条、第113条之规定，申请人认为该案符合立案条件，
   恳请贵院依法进行立案监督。

此致 XXXX人民检察院

申请人（签名）：XXX
XXXX年XX月XX日

附件：
1. 不予立案通知书复印件
2. 报案材料及证据
3. 身份证明复印件
```

## 八、补充话术

### 8.1 民航投诉话术（12326）

> "您好，我要投诉[航空公司名称]。我的航班号是[XXX]，日期[XXXX年XX月XX日]，从[出发地]到[目的地]。航班因[原因]被[延误X小时/取消]。航空公司[未提供餐食/未安排住宿/未给予合理补偿]。我的诉求是[退款/赔偿/安排后续航班]。"

### 8.2 出租车/网约车投诉话术（12328）

> "您好，我要投诉[出租车公司/网约车平台]。在[时间]，我乘坐[车牌号XXX]，从[起点]到[终点]。司机存在[拒载/绕路/多收费/甩客]行为。[描述具体情况]。我的诉求是[退还多收费用/处罚司机]。我有[支付记录/行程截图/录音]等证据。"

### 8.3 医疗纠纷投诉话术（12320）

> "您好，我要反映医疗纠纷问题。我/我的家属[患者姓名]于[时间]在[医院名称][科室]就诊/住院。医院在诊疗过程中存在[描述问题，如误诊/延误治疗/手术失误]。造成了[描述损害后果]。我已与医院沟通但[处理结果]。我的诉求是[赔偿/道歉/追究责任]。"

### 8.4 通信服务申诉话术（12300）

> "您好，我要申诉[运营商名称]。我的手机号是[XXX]。运营商存在[乱扣费/未经同意开通业务/信号问题不处理/注销号码被拒]等问题。[描述具体情况]。我已向运营商投诉但[未解决/处理不满意]。我的诉求是[退还费用/取消业务/解决信号问题/办理注销]。"

## 九、文书生成注意事项

1. **当事人信息完整**：姓名、性别、出生日期、住址、联系方式必须准确
2. **诉讼请求明确**：每项请求具体、可执行，避免模糊表述
3. **事实简明**：按时间顺序叙述，重点突出，避免冗长
4. **法律依据准确**：引用具体法律条文（条款编号）
5. **证据清单对应**：每项主张都有对应证据支持
6. **格式规范**：使用 A4 页面，标题黑体二号，正文仿宋三号
7. **签名日期**：起诉状/申请书必须手写签名并注明日期
