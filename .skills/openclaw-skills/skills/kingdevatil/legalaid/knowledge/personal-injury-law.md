## 一、人身损害赔偿（民法典侵权责任编）

### 第1179条：人身损害赔偿范围
侵害他人造成人身损害的，应当赔偿医疗费、护理费、交通费、营养费、住院伙食补助费等为治疗和康复支出的合理费用，以及因误工减少的收入。造成残疾的，还应当赔偿辅助器具费和残疾赔偿金；造成死亡的，还应当赔偿丧葬费和死亡赔偿金。

### 第1180条：因同一侵权行为造成多人死亡
因同一侵权行为造成多人死亡的，可以以相同数额确定死亡赔偿金。

### 第1181条：死亡赔偿金
被侵权人死亡的，其近亲属有权请求侵权人承担侵权责任。被侵权人为组织，该组织分立、合并的，承继权利的组织有权请求侵权人承担侵权责任。

### 第1182条：侵害人身权益造成财产损失的赔偿
侵害他人人身权益造成财产损失的，按照被侵权人因此受到的损失或者侵权人因此获得的利益赔偿。

### 第1183条：精神损害赔偿
侵害自然人人身权益造成严重精神损害的，被侵权人有权请求精神损害赔偿。因故意或者重大过失侵害自然人具有人身意义的特定物造成严重精神损害的，被侵权人有权请求精神损害赔偿。

### 第1184条：财产损失计算
侵害他人财产的，财产损失按照损失发生时的市场价格或者其他合理方式计算。

## 二、殴打/故意伤害

### 2.1 治安管理处罚法第43条

殴打他人的，或者故意伤害他人身体的，处**五日以上十日以下拘留**，并处**二百元以上五百元以下罚款**；情节较轻的，处五日以下拘留或者五百元以下罚款。

有下列情形之一的，处**十日以上十五日以下拘留**，并处**五百元以上一千元以下罚款**：
- 结伙殴打、伤害他人的
- 殴打、伤害残疾人、孕妇、不满十四周岁的人或者六十周岁以上的人的
- 多次殴打、伤害他人或者一次殴打、伤害多人的

### 2.2 刑法第234条：故意伤害罪

| 伤害等级 | 量刑标准 |
|---------|---------|
| **轻伤** | 三年以下有期徒刑、拘役或者管制 |
| **重伤** | 三年以上十年以下有期徒刑 |
| **致人死亡或者以特别残忍手段致人重伤造成严重残疾** | 十年以上有期徒刑、无期徒刑或者死刑 |

### 2.3 伤情鉴定

#### 鉴定流程
1. 公安机关出具**法医鉴定委托书**
2. 到指定鉴定机构进行鉴定
3. 鉴定机构出具**鉴定意见书**
4. 如对鉴定结果不服，可申请**重新鉴定**

#### 伤情等级标准

| 等级 | 标准 | 法律后果 |
|------|------|---------|
| **轻微伤** | 皮肤软组织挫伤、擦伤、浅表伤口等 | 治安处罚（拘留+罚款）+ 民事赔偿 |
| **轻伤二级** | 骨折、较深刀伤、脑震荡等 | 刑事立案（故意伤害罪）+ 民事赔偿 |
| **轻伤一级** | 较重骨折、器官轻度损伤等 | 刑事立案（量刑更重） |
| **重伤二级** | 严重骨折、器官功能丧失等 | 刑事立案（3-10年） |
| **重伤一级** | 极重伤残、植物人状态等 | 刑事立案（10年以上至死刑） |

#### 鉴定时机
- **以损伤后果为主**的鉴定：在伤情稳定后进行（一般3-6个月）
- **以损伤当时情况为主**的鉴定：越早越好（如面部创口长度）

### 2.4 被殴打维权全流程

```
被殴打
  ↓
【Step 1】立即报警110
  → 要求出警到场
  → 要求制作笔录并签名
  → 务必索要受案回执
  ↓
【Step 2】就医验伤
  → 去正规医院就诊，说明被殴打
  → 保存全部病历、诊断证明、检查报告
  → 拍摄伤情照片（多角度、带时间）
  → 要求公安机关开具法医鉴定委托书
  ↓
【Step 3】伤情鉴定
  → 到指定机构鉴定
  → 等待鉴定结果（一般15-30个工作日）
  ↓
【Step 4】根据鉴定结果分流
  ├─ 轻微伤 → 治安案件
  │   → 公安机关调解或行政处罚
  │   → 调解不成 → 行政处罚 + 民事诉讼索赔
  │
  ├─ 轻伤 → 刑事案件
  │   → 公安机关立案侦查
  │   → 可提起刑事附带民事诉讼
  │
  └─ 重伤 → 刑事案件（更重刑罚）
      → 同上，刑期更长
  ↓
【Step 5】民事赔偿
  → 医疗费 + 误工费 + 护理费 + 交通费
  → 营养费 + 住院伙食补助费
  → 伤残赔偿金（如构成伤残）
  → 精神损害赔偿（严重时）
```

### 2.5 证据清单

| 证据 | 如何获取 | 重要性 |
|------|---------|--------|
| 报警记录 | 110通话记录 + 受案回执 | ⭐⭐⭐⭐⭐ |
| 笔录 | 派出所制作，核对后签名 | ⭐⭐⭐⭐⭐ |
| 伤情照片 | 手机拍摄，多角度 | ⭐⭐⭐⭐⭐ |
| 监控录像 | 要求公安机关调取 | ⭐⭐⭐⭐⭐ |
| 证人证言 | 目击者联系方式 | ⭐⭐⭐⭐ |
| 病历/诊断证明 | 医院出具 | ⭐⭐⭐⭐⭐ |
| 法医鉴定 | 公安机关委托 | ⭐⭐⭐⭐⭐ |
| 医疗费票据 | 医院出具 | ⭐⭐⭐⭐ |
| 误工证明 | 单位出具 + 工资流水 | ⭐⭐⭐ |

## 三、交通事故

### 3.1 道路交通安全法第76条：赔偿责任

机动车发生交通事故造成人身伤亡、财产损失的，由保险公司在机动车第三者责任强制保险责任限额范围内予以赔偿；不足的部分，按照下列规定承担赔偿责任：
- 机动车之间发生交通事故的，由有过错的一方承担赔偿责任；双方都有过错的，按照各自过错的比例分担责任
- 机动车与非机动车驾驶人、行人之间发生交通事故，非机动车驾驶人、行人没有过错的，由机动车一方承担赔偿责任

### 3.2 交通事故处理流程

```
发生交通事故
  ↓
【Step 1】保护现场
  → 开启危险报警闪光灯
  → 设置警告标志（常规道路50-100米，高速公路150米外）
  → 人员转移到安全地带
  ↓
【Step 2】报警/报案
  → 有人伤：立即拨打122（交警）+ 120（急救）
  → 无人伤轻微事故：拍照后快速撤离，自行协商或报交警
  → 同时拨打保险公司电话报案
  ↓
【Step 3】交警处理
  → 交警到场勘查、制作事故认定书
  → 事故认定书一般10日内出具
  → 如对认定有异议，3日内申请复核
  ↓
【Step 4】治疗与鉴定
  → 就医治疗，保存全部票据
  → 伤情稳定后进行伤残鉴定
  ↓
【Step 5】赔偿协商
  → 先由交强险赔付（死亡伤残18万、医疗费1.8万、财产损失2000元）
  → 不足部分由商业三者险赔付
  → 仍不足由侵权人承担
  ↓
【Step 6】诉讼（如协商不成）
  → 向事故发生地法院提起诉讼
  → 可同时起诉肇事方和保险公司
```

### 3.3 伤残等级与赔偿

| 伤残等级 | 赔偿系数 | 说明 |
|---------|---------|------|
| 一级伤残 | 100% | 最严重 |
| 二级伤残 | 90% | |
| 三级伤残 | 80% | |
| 四级伤残 | 70% | |
| 五级伤残 | 60% | |
| 六级伤残 | 50% | |
| 七级伤残 | 40% | |
| 八级伤残 | 30% | |
| 九级伤残 | 20% | |
| 十级伤残 | 10% | 最轻微 |

伤残赔偿金 = 受诉法院所在地上一年度城镇居民人均可支配收入 × 20年 × 伤残系数

### 3.4 交通事故赔偿项目

| 项目 | 说明 |
|------|------|
| 医疗费 | 按实际发生计算 |
| 误工费 | 误工天数 × 日工资 |
| 护理费 | 护理天数 × 护理人员日工资 |
| 住院伙食补助费 | 住院天数 × 当地标准（一般100元/天） |
| 营养费 | 按医嘱或鉴定意见 |
| 残疾赔偿金 | 城镇居民人均可支配收入 × 20年 × 伤残系数 |
| 残疾辅助器具费 | 按实际需要 |
| 精神损害赔偿 | 构成伤残时可主张 |
| 交通费 | 就医、处理事故实际发生 |
| 财产损失 | 车辆维修费、施救费等 |
| 死亡赔偿金 | 城镇居民人均可支配收入 × 20年 |

## 四、工伤

### 4.1 工伤认定条件（《工伤保险条例》第14条）

职工有下列情形之一的，应当认定为工伤：
- 在工作时间和工作场所内，因工作原因受到事故伤害的
- 工作时间前后在工作场所内，从事与工作有关的预备性或者收尾性工作受到事故伤害的
- 在工作时间和工作场所内，因履行工作职责受到暴力等意外伤害的
- 患职业病的
- 因工外出期间，由于工作原因受到伤害或者发生事故下落不明的
- 在上下班途中，受到非本人主要责任的交通事故或者城市轨道交通、客运轮渡、火车事故伤害的
- 法律、行政法规规定应当认定为工伤的其他情形

### 4.2 视同工伤（第15条）

- 在工作时间和工作岗位，突发疾病死亡或者在48小时之内经抢救无效死亡的
- 在抢险救灾等维护国家利益、公共利益活动中受到伤害的
- 职工原在军队服役，因战、因公负伤致残，已取得革命伤残军人证，到用人单位后旧伤复发的

### 4.3 工伤认定流程

```
发生事故伤害
  ↓
【Step 1】及时就医
  → 告知医生是工伤，在病历中注明
  → 保存全部医疗记录
  ↓
【Step 2】单位申报（30日内）
  → 用人单位应在事故发生之日起30日内提出工伤认定申请
  → 如单位不申请：
  ↓
【Step 3】个人申报（1年内）
  → 工伤职工或其近亲属可在1年内直接向社保行政部门提出申请
  → 向当地人社局/劳动保障行政部门提交
  ↓
【Step 4】工伤认定
  → 社保行政部门60日内作出工伤认定决定
  → 对决定不服，可申请行政复议或提起行政诉讼
  ↓
【Step 5】劳动能力鉴定
  → 伤情稳定后，向劳动能力鉴定委员会申请
  → 鉴定委员会60日内作出鉴定结论
  ↓
【Step 6】享受工伤保险待遇
  → 医疗费、康复费 → 工伤保险基金支付
  → 停工留薪期工资 → 用人单位支付（一般不超过12个月）
  → 伤残补助金 → 工伤保险基金支付
  → 伤残津贴 → 一至四级由工伤保险基金按月支付
```

### 4.4 工伤保险待遇

| 待遇项目 | 支付主体 | 标准 |
|---------|---------|------|
| 医疗费 | 工伤保险基金 | 按实际发生 |
| 停工留薪期工资 | 用人单位 | 原工资福利待遇不变，一般不超过12个月 |
| 一次性伤残补助金 | 工伤保险基金 | 一级27个月~十级7个月本人工资 |
| 伤残津贴 | 工伤保险基金 | 一级90%~四级60%本人工资（按月） |
| 一次性工伤医疗补助金 | 工伤保险基金 | 解除劳动合同时支付（各省标准不同） |
| 一次性伤残就业补助金 | 用人单位 | 解除劳动合同时支付（各省标准不同） |

### 4.5 单位不申请工伤认定的应对

- 用人单位未在30日内申请的，工伤职工可在**1年内**自行申请
- 用人单位未缴纳工伤保险的，**全部工伤待遇由用人单位承担**
- 可同时向劳动监察投诉（12333）+ 申请工伤认定 + 申请劳动仲裁

## 五、医疗事故/医疗损害

### 5.1 法律依据

- 《民法典》第1218条：患者在诊疗活动中受到损害，医疗机构或者其医务人员有过错的，由医疗机构承担赔偿责任。
- 《医疗事故处理条例》（国务院令第351号）
- 《医疗纠纷预防和处理条例》（2018年）

### 5.2 医疗损害维权流程

```
怀疑医疗事故/医疗损害
  ↓
【Step 1】保全证据（最重要）
  → 要求医院封存病历（主观病历+客观病历）
  → 封存实物证据（药品、血液、器械等）
  → 保存所有缴费票据
  ↓
【Step 2】与医院协商
  → 向医院医务科/医患关系办公室投诉
  → 要求医院说明情况
  ↓
【Step 3】医疗鉴定
  → 途径A：医学会医疗事故技术鉴定
  → 途径B：司法鉴定机构医疗损害鉴定（诉讼中常用）
  → 鉴定结论是认定医疗机构过错的关键证据
  ↓
【Step 4】行政投诉
  → 向当地卫健委投诉（12320或12345转卫健委）
  → 卫健委可对医疗机构进行行政处罚
  ↓
【Step 5】诉讼
  → 向医院所在地法院提起医疗损害责任纠纷诉讼
  → 案由：医疗损害责任纠纷
  → 诉讼时效：自知道或应当知道受到损害之日起3年
```

### 5.3 医疗事故分级

| 等级 | 说明 |
|------|------|
| 一级医疗事故 | 造成患者死亡、重度残疾 |
| 二级医疗事故 | 造成患者中度残疾、器官组织损伤导致严重功能障碍 |
| 三级医疗事故 | 造成患者轻度残疾、器官组织损伤导致一般功能障碍 |
| 四级医疗事故 | 造成患者明显人身损害的其他后果 |

### 5.4 医疗损害赔偿项目

| 项目 | 说明 |
|------|------|
| 医疗费 | 原发病医疗费 + 后续治疗费 |
| 误工费 | 误工天数 × 日工资 |
| 护理费 | 护理天数 × 护理人员日工资 |
| 住院伙食补助费 | 住院天数 × 当地标准 |
| 残疾赔偿金 | 伤残等级对应赔偿 |
| 死亡赔偿金 | 城镇居民人均可支配收入 × 20年 |
| 精神损害赔偿 | 造成严重后果时可主张 |
| 丧葬费 | 受诉法院所在地上一年度职工月平均工资 × 6个月 |

## 七、人身伤害通用证据清单

| 证据类型 | 说明 | 适用场景 |
|---------|------|---------|
| 报警/受案回执 | 公安机关出具 | 殴打、交通事故、高空抛物 |
| 病历/诊断证明 | 医院出具 | 所有人身伤害 |
