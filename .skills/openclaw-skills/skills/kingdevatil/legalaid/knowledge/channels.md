## 二、12315 消费者投诉举报

### 基本信息
- **全称**：全国12315平台
- **主办**：国家市场监督管理总局
- **电话**：12315
- **官网**：https://www.12315.cn
- **平台**：https://app.12315.cn
- **微信小程序**：搜索"12315"
- **支付宝小程序**：搜索"12315"
- **APP**：全国12315平台（各大应用商店下载）

### 管辖范围
- 商品质量问题、虚假宣传、假冒伪劣
- 服务消费纠纷（餐饮、旅游、美容等）
- 食品安全问题
- 价格违法行为
- 知识产权侵权
- 电子商务消费纠纷

### 投诉流程
1. **登记投诉**：通过网站、电话、小程序、APP等渠道登记
2. **审批分流**：市场监管部门审批后分流转至辖区市场监管所
3. **受理决定**：7个工作日内作出受理或不予受理决定
4. **调解处理**：采用调解方式处理
5. **结案归档**：调解成功或终止后结案

### 所需材料
1. 投诉人信息：姓名、电话号码、通讯地址
2. 被投诉人信息：名称（姓名）、地址
3. 具体投诉请求
4. 消费者权益争议事实
5. 相关凭证：购物发票、合同、聊天记录、照片、视频、支付凭证等

### 管辖规则
- 由被投诉人实际经营地或住所地县级市场监管部门处理
- 电子商务平台内经营者由其实际经营地或平台经营者住所地县级市场监管部门处理

### 时效限制
- 自知道或应当知道权益受到侵害之日起不超过**三年**

### 法律依据
- 《市场监督管理投诉举报处理暂行办法》（总局令第20号）
- https://www.gov.cn/zhengce/2022-10/08/content_5723506.htm

## 三、12345 政务服务便民热线

### 基本信息
- **全称**：12345政务服务便民热线
- **电话**：12345（24小时服务）
- **微信公众号**：各地"12345"公众号（如"广州12345"）
- **网站**：各地政府门户网站的12345专区

### 功能
- 倾听群众和企业诉求
- 推动解决实际问题
- 涵盖政策咨询、办事指南、投诉举报、意见建议、求助等
- 几乎覆盖所有政府职能领域

### 使用流程
1. 拨打12345或通过互联网渠道提交诉求
2. 话务员记录诉求内容，分类转办至相关职能部门
3. 职能部门在规定时限内处理并反馈
4. 回访评价

### 适用场景
- 不确定具体归哪个部门管时，12345是最通用的入口
- 会自动分转至对应部门（住建、人社、市场监管等）

### 政策依据
- 国务院办公厅《关于进一步规范和提升12345热线服务的意见》（国办函〔2025〕66号）

## 五、劳动仲裁

### 基本信息
- **全称**：劳动人事争议仲裁
- **管辖**：劳动合同履行地或用人单位所在地的劳动人事争议仲裁委员会

### 仲裁时效
- 一般情况：自知道或应当知道权利被侵害之日起**1年**内
- 拖欠劳动报酬：劳动关系存续期间不受限制；终止后1年内

### 申请流程
1. 准备材料（仲裁申请书、身份证明、证据材料）
2. 向有管辖权的仲裁委员会提交书面申请及副本
3. 仲裁委5日内决定是否受理
4. 开庭审理
5. 一般45日内结案（复杂可延长15日）

### 所需材料
1. 仲裁申请书（按被申请人人数提交副本）
2. 身份证明（身份证原件）
3. 证据材料（劳动合同、工资条/银行流水、工作证、考勤记录、解除/终止劳动合同通知书、聊天记录等）
4. 委托代理需提交授权委托书

### 法律依据
- 《中华人民共和国劳动争议调解仲裁法》
- 《劳动人事争议仲裁办案规则》（人社部令第33号）
- https://www.mohrss.gov.cn/SYrlzyhshbzb/ztzl/ldrszytjzc/fgzc/202207/t20220714_457861.html

## 七、行政复议

### 基本信息
- **全国行政复议服务平台**：https://xzfy.moj.gov.cn
- **示范文本**：https://xzfy.moj.gov.cn/c/2020-09-17/486112.shtml

- 自知道或应当知道行政行为之日起**60日**内

5. 一般60日内作出复议决定（可延长30日）

### 所需材料
1. 行政复议申请书
2. 身份证明（身份证复印件/营业执照副本）
3. 被诉行政行为文书复印件
4. 相关证据材料
5. 委托代理需提交授权委托书

## 十一、警务督察举报

### 基本信息
- **全称**：公安机关警务督察
- **举报电话**：12389
- **网上举报平台**：https://www.12389.gov.cn
- **法律依据**：《公安机关督察条例》

### 受理范围
- 公安机关及其人民警察**不履行法定职责**（如应当受案不受案、应当立案不立案、立案后不侦查）
- **违法行使职权**（如违规立案插手民事纠纷、违规采取强制措施、刑讯逼供）
- **违反纪律**（如态度恶劣、吃拿卡要、泄露案情、泄露报案人信息）

### 举报方式
1. **电话举报**：拨打12389
2. **网上举报**：登录https://www.12389.gov.cn 填写举报信息
3. **信函举报**：邮寄至当地公安局警务督察部门
4. **当面举报**：到当地公安局警务督察支队/大队

### 举报注意事项
- **实名举报**优先处理，需提供本人身份信息
- 尽量提供**具体事实和证据**（时间、地点、涉事民警警号/姓名、相关文书等）
- 举报后可要求反馈处理结果
- 举报人信息依法保密
- 如对督察处理结果不满意，可向上级督察部门申诉

### 法律依据
- 《公安机关督察条例》
- 《公安机关办理刑事案件程序规定》
- 《刑事诉讼法》第113条（检察院立案监督）

## 十二、工信部/通信管理局维权渠道

### 12.1 电信用户申诉（12300）

**基本信息**
- **全称**：电信用户申诉处理中心
- **电话**：12300
- **申诉网站**：https://dxss.miit.gov.cn
- **政务服务平台**：https://ythzxfw.miit.gov.cn
- **微信小程序**：搜索"工信部12300"

**前置条件（重要）**
- 必须先向电信运营商投诉（10086/10010/10000）
- 对运营商处理结果不满意，或投诉后15日内未得到答复
- 才能向工信部/通信管理局申诉

**申诉流程**
1. 拨打运营商客服投诉，保留工单号
2. 等待15日或对结果不满意
3. 通过12300电话或网站提交申诉
4. 工信部转交属地通信管理局处理
5. 一般15-30个工作日内反馈结果

**适用场景**
- 手机号莫名扣费、套餐强制捆绑
- 宽带故障迟迟不修
- 电信虚假宣传（免费流量有隐藏条件）
- 携号转网受阻
- 信号质量长期不达标
- 运营商未经同意开通增值业务

## 十五、消防监督举报渠道

### 基本信息
- **全称**：消防安全违法行为及火灾隐患举报
- **举报电话**：12345（96119已全国并入12345）
- **火警电话**：119（仅限火灾报警，非举报用途）
- **安全生产举报**：12350（重大安全隐患）
- **主办**：消防救援支队/大队

### 受理范围
- 消防通道堵塞、安全出口锁闭
- 消防设施损坏（灭火器过期、消防栓无水、报警器故障）
- 违规使用明火、私拉电线
- 电动自行车违规停放充电
- 建筑物未经消防验收擅自投入使用
- 人员密集场所违反消防规定（如网吧、商场、宾馆等）
- 物业公司未履行消防安全职责

### 举报方式
1. **电话举报**：拨打12345，说明隐患地点和具体问题
2. **现场举报**：到当地消防救援支队或大队
3. **网络举报**：通过当地12345微信公众号或小程序
4. **12350**：重大安全生产隐患（含消防安全）可拨打

### 举报注意事项
- 119是火警电话，发现**正在发生的火灾**拨打119
- 发现**消防隐患**拨打12345
- 提供准确的隐患地点、具体问题描述
- 可拍照或录像作为证据
- 举报人信息依法保密

### 法律依据
- 《中华人民共和国消防法》
- 《消防安全责任制实施办法》

## 十六、税务监督投诉举报渠道

### 基本信息
- **全称**：纳税服务热线/税收违法行为举报
- **服务电话**：12366
- **举报网站**：https://12366.chinatax.gov.cn
- **主办**：国家税务总局

### 受理范围
- **涉税咨询**：税收政策、征管程序等咨询
- **税收违法举报**：偷税漏税、虚开发票、骗取出口退税等
- **发票违法举报**：假发票、非法代开发票
- **服务投诉**：对税务机关工作人员服务态度、服务质效不满意
- **权益侵害投诉**：认为税务机关及其工作人员侵害合法权益

### 举报方式
1. **电话举报**：拨打12366（人工服务：工作日8:30-17:30，自动语音：7×24小时）
2. **网上举报**：登录 https://12366.chinatax.gov.cn
3. **现场举报**：到当地税务局稽查局/举报中心
4. **信函举报**：邮寄至当地税务局税收违法案件举报中心

### 举报注意事项
- 提倡实名举报，税务机关依法保密
- 尽量提供被举报人名称、统一社会信用代码、违法事实等
- 税收违法举报经查实后，可按规定获得奖励
- 发票违法举报可通过全国增值税发票查验平台核验

### 法律依据
- 《中华人民共和国税收征收管理法》
- 《税收违法行为检举管理办法》（国家税务总局令第49号）
- 《中华人民共和国发票管理办法》
