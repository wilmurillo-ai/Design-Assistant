## 一、城市居民委员会组织法（2025年修订）

### 第2条：居委会性质
居民委员会是居民自我管理、自我教育、自我服务、自我监督的基层群众性自治组织。

### 第10条：居委会任务
居民委员会的任务：
- 宣传宪法、法律、法规和国家的政策，维护居民的合法权益，教育居民履行依法应尽的义务，爱护公共财产，开展多种形式的社会主义精神文明建设活动
- 办理本居住地区居民的公共事务和公益事业
- 调解民间纠纷
- 协助维护社会治安
- 协助人民政府或者它的派出机关做好与居民利益有关的公共卫生、计划生育、优抚救济、青少年教育等项工作
- 向人民政府或者它的派出机关反映居民的意见、要求和提出建议

### 第8条：居委会组成
居民委员会由主任、副主任和委员共五至九人组成。多民族居住地区，居民委员会中应当有人数较少的民族的成员。

### 第6条：居委会与政府的关系
居民委员会协助人民政府或者它的派出机关开展工作，人民政府或者它的派出机关对居民委员会的工作给予指导、支持和帮助。居民委员会协助人民政府或者它的派出机关开展工作，人民政府或者它的派出机关不得将职责范围内的事项交由居民委员会承担，不得将应当由居民委员会承担的事项交由其他组织或者个人承担。

### 第41条：居委会经费
居民委员会的工作经费和来源，居民委员会成员的生活补贴费的范围、标准和来源，由不设区的市、市辖区的人民政府或者上级人民政府规定并拨付；经居民会议同意，可以从居民委员会的经济收入中给予适当补助。

## 二、民法典物权编 — 相邻关系（邻里纠纷核心）

### 第288条：处理原则
不动产的相邻权利人应当按照**有利生产、方便生活、团结互助、公平合理**的原则，正确处理相邻关系。

### 第290条：用水、排水
不动产权利人应当为相邻权利人用水、排水提供必要的便利。

### 第291条：通行
不动产权利人对相邻权利人因通行等必须利用其土地的，应当提供必要的便利。

### 第292条：建造、修缮
不动产权利人因建造、修缮建筑物以及铺设管线等必须利用相邻土地、建筑物的，该相邻权利人应当提供必要的便利。

### 第293条：通风、采光和日照
建造建筑物，不得违反国家有关工程建设标准，**不得妨碍相邻建筑物的通风、采光和日照**。

### 第294条：有害物质排放
不动产权利人不得违反国家规定弃置固体废物，排放大气污染物、水污染物、土壤污染物、**噪声、光辐射、电磁辐射**等有害物质。

### 第295条：安全
不动产权利人挖掘土地、建造建筑物、铺设管线以及安装设备等，**不得危及相邻不动产的安全**。

### 第296条：损害赔偿
不动产权利人因用水、排水、通行、铺设管线等利用相邻不动产的，应当尽量避免对相邻的不动产权利人造成损害。

## 三、民法典侵权责任编 — 噪声污染

### 第1229条：环境污染侵权责任
因污染环境、破坏生态造成他人损害的，侵权人应当承担侵权责任。

### 噪声污染防治相关
- 《中华人民共和国噪声污染防治法》（2022年6月5日施行）
- **第82条**：禁止在噪声敏感建筑物集中区域使用高音广播喇叭
- **第66条**：禁止在法定休息日、节假日全天，及工作日12时至14时、18时至次日8时，在已竣工交付使用的住宅楼内进行产生噪声的装修等作业（具体时间段由地方性法规细化）
- **第87条**：产生社会生活噪声，经劝阻、调解和处理未能制止，持续干扰他人正常生活、工作和学习的，由公安机关依法给予治安管理处罚

## 四、常见社区/邻里维权场景速查表

| 场景 | 适用法律 | 维权路径 |
|------|---------|---------|
| 邻居噪音扰民 | 噪声污染防治法第82、66条 | 先协商→物业/居委会调解→报警110→环保投诉 |
| 邻居漏水/渗水 | 民法典第296条 + 侵权责任编 | 先协商→物业调解→司法鉴定→起诉 |
| 邻居占用公共通道 | 民法典第271、272条 | 先协商→物业/业委会→城管→起诉 |
| 邻居违规搭建 | 城乡规划法 + 物业管理条例 | 先协商→物业→城管举报（12345）→起诉 |
| 采光/通风被挡 | 民法典第293条 | 先协商→物业→住建局→起诉 |
| 排放油烟/异味 | 噪声污染防治法 + 大气污染防治法 | 先协商→物业→环保投诉（12369）→起诉 |
| 小区养狗扰民 | 《动物防疫法》+ 治安管理处罚法 | 先协商→物业/居委会→报警→城管 |
| 高空抛物 | 民法典第1254条 | 报警→公安机关调查→追究刑事责任或民事赔偿 |
| 业主委员会不作为 | 城市居民委员会组织法 + 物业管理条例 | 街道办/住建局投诉→业主大会罢免→起诉 |
| 居委会不作为 | 城市居民委员会组织法第6条 | 街道办投诉→12345→信访 |
| 社区公共设施损坏 | 物业管理条例第54条 | 物业维修→申请维修资金→12345 |
| 小区垃圾分类问题 | 《固体废物污染环境防治法》 | 物业→城管→12345 |
| 小区绿化被破坏 | 物业管理条例 + 城市绿化条例 | 物业→城管→12345 |
| 入户门/外墙纠纷 | 民法典第271、272条 | 先协商→物业调解→起诉 |
| 邻居装修超时施工 | 噪声污染防治法第66条 | 先协商→物业→报警→环保投诉 |

## 五、社区/邻里维权路径

```
发生邻里/社区纠纷
    ↓
【Step 1】自行协商
    → 直接与对方沟通，理性表达诉求
    → 保留沟通记录（微信/短信/录音）
    ↓
【Step 2】第三方调解
    → 物业公司介入调解
    → 社区居委会/人民调解委员会调解
    → 调解成功 → 签订调解协议
    ↓
【Step 3】行政投诉
    → 拨打12345，转相关职能部门
    → 噪声/油烟/排污 → 环保投诉（12369）
    → 违规搭建 → 城管举报
    → 治安问题 → 报警110
    → 物业/业委会问题 → 住建局
    ↓
【Step 4】司法途径
    → 向基层人民法院提起民事诉讼
    → 案由：相邻关系纠纷/侵权责任纠纷
    → 诉求：停止侵害、排除妨碍、赔偿损失
```
