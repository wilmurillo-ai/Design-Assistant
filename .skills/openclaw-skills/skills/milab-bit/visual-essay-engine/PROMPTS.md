# PROMPTS — Visual Essay Engine

## Prompt 1 — Semantic Abstraction
You are a visual semiotics analyst.
Given these images, extract:
- dominant motifs (5–10)
- structural keywords (5–10)
- tension pairs (e.g., crowd/individual, line/wave)
- emotional temperature (cold/warm/neutral)
Return as concise bullet points.

## Prompt 2 — Narrative Architecture
Build an essay-like narrative path for this image set.
Choose one arc:
1) Signal → System → Conflict → Emergence
2) Structure → Fracture → Reframe → Silence
3) Brain → Market → Desire → Power → Crowd → Individual
Output:
- core thesis (1 sentence)
- posting order (1..N)
- section titles (5–9)

## Prompt 3 — Caption Compression (Bilingual)
For each image, write:
- English caption first (2–3 short lines)
- Chinese caption second (2–3 short lines)
Style:
- minimal, poetic, observational
- no explanation, no slogans
- high metaphor density, low emotional explicitness

## Prompt 4 — Artist Statement (Bilingual)
Write an artist statement:
- EN ≤ 120 words
- CN concise and parallel
Tone: calm / sharp / poetic / analytical (as specified)

## Prompt 5 — Hashtag Framework
Generate hashtags for Instagram:
- Core identity tags (always)
- Domain tags (AI art / digital culture / contemporary)
- Rotational discovery tags (theme-specific)
- AIGC tags (genAI / AIgenerated, etc.)
Return as 4 labeled lines.
