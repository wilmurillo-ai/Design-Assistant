# 小红书发布器技能使用指南

## 前提条件
1. **小红书创作者账号**：确保已登录小红书创作者平台
2. **浏览器自动化**：OpenClaw浏览器服务已启动
3. **准备好的内容**：标题和正文内容已经准备好

## 基本使用

### 通过OpenClaw CLI调用
```bash
# 安装技能（如果尚未安装）
openclaw skill install xiaohongshu-publisher

# 运行技能发布内容
openclaw skill run xiaohongshu-publisher --title "你的标题" --content "你的正文内容"
```

### 通过Agent直接调用
在对话中提供标题和内容：
- "使用小红书发布器发布以下内容：[标题和正文]"
- "帮我把这段内容发布到小红书：[内容]"

## 发布流程步骤

技能会自动执行以下操作流程：

1. **导航到创作平台** - 访问 `https://creator.xiaohongshu.com`
2. **点击"写长文"** - 选择长文发布模式
3. **点击"新的创作"** - 开始新笔记创作
4. **填充标题** - 在标题输入框中填入提供的标题
5. **填充正文** - 在正文区域填入提供的内容
6. **点击"下一步"** - 进入发布设置页面
7. **点击"发布"** - 完成最终发布操作
8. **确认成功** - 验证发布是否成功

## 故障排除

### 发布失败
1. 检查小红书账号是否已登录
2. 确认浏览器自动化服务是否正常运行
3. 验证网络连接是否正常

### 元素找不到
1. 小红书界面可能已更新，需要调整元素引用
2. 检查页面加载是否完成
3. 确认是否在正确的页面上下文中

## 最佳实践

1. **内容格式**：确保内容符合小红书的内容规范
2. **字符限制**：注意标题64字符、正文1000字符的限制
3. **预览检查**：发布前可在小红书预览功能中检查效果
4. **标签添加**：在正文中适当添加话题标签以提高曝光
