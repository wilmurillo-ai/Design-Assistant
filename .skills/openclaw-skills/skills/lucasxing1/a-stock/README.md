## 📖 简介

**A股智投大师** 是一个专注于 A 股市场的智能投研助手，基于东方财富数据，整合了：

- 📊 实时行情分析
- 📈 基本面深度分析
- 🎯 智能选股
- ⭐ 自选股管理
- 📰 舆情监控
- 🔔 监控预警

---

## 🎯 核心能力

### 1. 行情分析
- 实时行情、资金流向
- K线形态分析
- 涨跌幅、成交量监控

### 2. 基本面分析
- PE / PB / ROE 估值
- 财务报表分析
- 盈利预测

### 3. 智能选股
- 按条件筛选股票
- 行业、概念、资金面多维度筛选

### 4. 自选股管理
- 查看自选股列表
- 批量查询涨跌幅
- 一键更新自选股

### 5. 舆情监控
- 新闻、研报、公告
- 实时跟踪上市公司动态

### 6. 监控预警
- 成本百分比监控
- 均线金叉/死叉
- RSI超买超卖
- 成交量异动
- 跳空缺口
- 动态止盈

## 📝 使用示例

### 查询行情
```
用户：查一下贵州茅台现在多少钱
→ 调用 mx_data 查询 600519 最新价

用户：今天A股涨跌幅前10
→ 查询涨幅榜/跌幅榜
```

### 查看自选股
```
用户：我的自选股今天怎么样
→ 批量查询自选股涨跌幅

用户：添加宁德时代到自选
→ 调用 mx_selfselect 添加
```

### 智能选股
```
用户：帮我选一下PE<20的银行股
→ 条件筛选：PE<20 + 银行板块

用户：最近资金流入的科技股
→ 筛选：资金净流入 + 科技板块
```

### 深度分析
```
用户：分析一下宁德时代的投资价值
→ 1. 调用 mx_data 获取基本面
→ 2. 调用 mx_data 获取资金流向
→ 3. 调用 mx_search 获取最新新闻
→ 4. 综合分析给出建议
```

### 设置监控
```
用户：帮我监控宁德时代，成本价是400元
→ 添加成本价监控规则

用户：设置MA5金叉提醒
→ 添加均线金叉监控
```

## 🔧 依赖

本技能依赖以下 OpenClaw Skills：

- **mx-data** - 妙想金融数据（行情、财务、估值）
- **mx-search** - 妙想资讯搜索（新闻、研报）
- **mx-select-stock** - 妙想智能选股
- **mx-selfselect** - 妙想自选股管理
- **stock-monitor-skill** - 股票监控预警

### 安装依赖

```bash
clawhub install mx-data mx-search mx-select-stock mx-selfselect stock-monitor-skill
```

### 配置 API Key

1. 打开**东方财富 APP**
2. 搜索「**东方财富 skills**」
3. 点击「我的」→「API Key」
4. 复制 Key 并配置到环境变量
