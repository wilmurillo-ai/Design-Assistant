# Startup Researcher

An AI-powered research assistant skill that automates the generation of comprehensive market intelligence reports for startups.

## Overview

This skill is designed to:
- Research target companies from a predefined `watchlist.yaml`.
- Synthesize individual company profiles including fundamental facts, recent developments, products, and hiring.
- Compile a category-level macro-overview and competitive matrix.
- Generate a professional, styled PDF briefing.

## Quick Start

Install the skill:

```bash
npx skills install https://github.com/hxy9243/skills/tree/main/startup-researcher
```

## Usage

This skill is orchestrated by an AI assistant. To use it, simply ask your assistant to run the `startup-researcher` skill.

```bash
"Run the startup researcher skill on the Open Source AI category"
"Can you use startup-researcher to profile Groq?"
```
