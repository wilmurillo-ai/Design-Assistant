# mommy-role-research-brief

A research-and-editorial OpenClaw skill for tracking discourse around mommy-type characters, mommy issues, daddy issues, mothering ideology, gendered care, and digital intimacy, then converting it into Chinese-language writing material.

## Best use cases

- Weekly research briefs
- Topic discovery for essays and newsletters
- Source pack generation
- Chinese article angle generation
- Pop-culture + feminist criticism synthesis
- Turning scattered English material into structured Chinese writing plans

## Example prompts

- 请按周报模式，追踪最近 7 天关于“妈妈系角色 / 母职意识形态 / 数字亲密”的新讨论，并生成中文选题包。
- 请做一个专题研究包：为什么“妈妈系角色”会把照料重新编码成女性义务？
- 帮我找近三年关于 daddy issues、依恋和亲密关系的高质量论文，并整理成中文综述。
- 把这些英文来源转成一篇中文长文的论点、结构和标题。

## Notes

This skill is designed to be primarily research-oriented and prompt-driven.
It should avoid opaque shell instructions and should prefer transparent, source-based outputs.
