# 🎬 Seedance 2.0 分镜向导 - 即梦 (Jimeng) 电影级提示词导演

[English](README.md) | [中文](README_CN.md)

专为 **即梦 (Jimeng) Seedance 2.0** 打造的专业分镜与提示词 (Prompt Engineering) 向导。本技能旨在帮助用户将简单的创意火花，转化为“电影级”的视频生成提示词，充分释放 Seedance 的多模态能力（图生视频、视频生视频）。

## ✨ 核心亮点

-   **🎯 搜索与提示词深度优化**：针对 **即梦 (Jimeng)** 和 **Seedance 2.0** 的分词逻辑精准调优。
-   **双模驱动工作流**：支持 **基础 (Basic)** 分镜脚本与 **高级 (Advanced)** 导演 breakdown，满足从快速创意到高精度复刻全场景需求。
-   **🛠️ 多模态深度优化**：精通 `@图片`、`@视频`、`@音频` 引用语法。
-   **🎥 高精度动作/运镜复刻**：支持 1-to-1 视频还原。
-   **🩹 视频延长与编辑**：提供精准指令引导，用于平滑延长视频或“颠覆”已有剧情。

## 🚀 使用方法

直接输入您的创意构想即可唤醒技能：

> “/seedance-guide 一个温暖的除夕夜团圆饭视频。”

技能将作为您的 **创意导演**，通过询问风格、镜头要求及可用素材，一步步引导您生成最终的专业提示词。

## 📝 Seedance 2.0 基础知识

Seedance 2.0 支持最多 **12 个文件** 作为输入：
-   **图片 (≤9)**：用于角色参考、风格参考、首/尾帧。
-   **视频 (≤3)**：用于动作参考、运镜参考、视频延长 (2-15s)。
-   **音频 (≤3)**：用于背景音乐或对口型参考。
