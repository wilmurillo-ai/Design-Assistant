# 剧本示例

## 对白正反对照（核心参考）

以下示例展示同一场景下"错误写法"与"正确写法"的差异，任何模型输出前必须对照检查。

### 对照 1：截获情报

**错误（直给、无潜台词、角色无攻防）：**
```text
沈无衣：字被刮掉了。三十六个名字，一个不剩。
密使：什么？
沈无衣：有人不想让这些名字被看见。
密使：我不知道是谁干的。
沈无衣：那你知道名字都有谁？
密使：我不敢说。
```
问题：角色在交换信息，不在争夺主动权；每句话只有字面意思。

**正确（绕着说、每句争主动权、日常语气谈危险事）：**
```text
沈无衣（翻完册子，又翻一遍）：好纸。
密使：什么？
沈无衣：八百里水路没泡烂，就是字没了。
密使：字……怎么会没了？
沈无衣：你抱了八百里，你不知道里面有没有字？
密使：朝廷的东西——
沈无衣：朝廷的东西你不敢看，我的刀你敢不敢看？
密使（退半步）：……册子你拿走。
沈无衣：你倒大方。空的给我，全的给谁了？
```
要点：沈无衣从"夸纸"切入（不着调），密使每次想搬出"朝廷"挡驾都被堵回去，权力关系在 6 句话里翻了两次。

### 对照 2：谈判场景

**错误（解释性对白、角色替观众总结）：**
```text
白会长：你今天来是想跟我合作对付县长吧？
马三刀：是的，我知道你手里有账本。
白会长：我确实有，但我不信任你。
马三刀：那你需要什么才能信任我？
```
问题：角色在"开会"，不在"过招"；信息全靠嘴说，没有行动暴露。

**正确（试探→刺探→反转，信息通过行动暴露）：**
```text
马三刀：白会长的茶不错，几泡了？
白会长：三泡。头泡洗杯，二泡敬人，三泡——
马三刀：三泡见底？
白会长：三泡看对面的人渴不渴。
（马三刀端杯不喝，搁下。）
马三刀：不渴。我就是路过，听说这条街的铺子最近换了不少东家。
白会长：换东家跟你一个卖马的有什么关系？
马三刀：马要吃草，草长在谁的地上，就跟谁有关系。
（白会长把茶壶转了个方向，壶嘴对着马三刀。）
白会长：那你到底是来买草的，还是来量地的？
```
要点：全程没提"合作""账本"，但两人都知道在谈什么；茶壶方向的动作暴露了白会长从客气变警惕。

### 对照 3：黑色幽默

**错误（刻意搞笑、网络段子味）：**
```text
杀手：你知道我来干什么的吧？
老板：不会是来吃面的吧哈哈。
杀手：面也行，加个卤蛋。反正你也吃不上了。
```
问题：像网络短视频台词，幽默靠反差梗硬凑。

**正确（荒诞来自角色的真诚，幽默来自立场冲突）：**
```text
杀手（坐下，看菜单）：牛肉面还是手擀的？
老板：手擀的。
杀手：粗的细的？
老板：您看着选。
杀手：细的吧。粗的不好消化。——对了，你欠孙老板的钱，他让我来问问。
老板：面先下？还是先谈钱？
杀手：一起吧，面坨了不好吃。
```
要点：杀手是认真在点面，不是"装"在点面；谈命和谈面用同一个语气，荒诞感是观众自己感受到的，不是角色表演给观众看的。

## 示例 1：故事梗概（节选）

城里新来的税务督办说要“清账”，三天后便在戏楼里被人当众揭了底：他自己就是假账最大的受益人。一个自称卖马的外乡人借机做局，想把三方势力引到同一张赌桌上。县长要面子，商会要活路，地头蛇要规矩，人人都说自己讲道理，人人都在偷偷改筹码。最后真正决定胜负的不是枪，而是谁先承认自己怕。

## 示例 2：人物小传（节选）

- 马三刀（男，38）
  - 表层身份：走南闯北的马贩子
  - 真实动机：查清兄长当年“意外坠河”真相
  - 关系张力：与县长互相利用；与商会会长惺惺相惜但立场对立
  - 语言特征：比喻多、话锋快、从不正面回答

- 白会长（女，34）
  - 表层身份：商会代理会长
  - 真实动机：保住码头税权，摆脱县衙控制
  - 关系张力：对马三刀有合作需求又高度戒备
  - 语言特征：句子短、停顿长、关键句常压低音量

## 示例 3：分场大纲（节选）

- S02-县衙偏厅-晨-县长安抚商会代表-税单被当场指出一处年份错误
- S03-河埠头-黄昏-马三刀与线人交易账册-线人临时改口并引来巡防队

## 示例 4：分场剧本（节选）

【场次】S02
【地点/时间】县衙偏厅 / 晨
【人物】县长、白会长、师爷
【场面目标】县长要稳住商会；白会长要确认县长是否参与假账。
【动作与调度】
- 县长始终坐主位不动，白会长站着说话。
- 师爷在门口与桌案之间来回，像计时器。
- 说到“旧账”时，外面晨鼓响三下。
【对白】
县长：账嘛，旧的不去，新的不来。
白会长：那得先说清，旧账是谁的旧。
师爷：会长，县里也是一片好心。
白会长：好心我见得多，好账我还没见着。
【场尾钩子】
师爷递错一本账册，封皮上写着“军需”。县长伸手太快，露了怯。

## 示例 6：近未来（分场剧本节选）

【场次】S04
【地点/时间】记忆归档中心 / 凌晨
【人物】沈岚、审计员K、值班工程师
【场面目标】沈岚要找回被删记忆片段；K要确认她是否非法篡改记录。
【动作与调度】
- 沈岚始终不靠近主控台，只在玻璃墙反光区移动。
- K站在门禁旁，保持退路，手指敲门禁节拍。
- 工程师在两人中间切换屏幕，形成视觉遮挡。
【对白】
K：你要的是记忆，还是证据？
沈岚：证据会过期，记忆不会。
工程师：两位，系统只认权限。
沈岚：权限是给守规矩的人准备的。
【场尾钩子】
屏幕弹出“归档者签名：沈岚（七年前）”，她第一次失去表情。
