## ✨ 核心功能

### 1. 自动简历筛选
- **智能解析**：自动提取简历中的姓名、经历、技能、项目
- **匹配评分**：0-100分，基于硬性要求、核心技能、经验相关性
- **分级推荐**：🟢 强烈推荐 (80+) / 🟡 建议考虑 (60-79) / 🔴 不太匹配 (<60)

### 2. 定制化面试问题
- **个性化设计**：根据面试官性格偏好生成问题风格
- **多维度考察**：技术能力、项目深挖、软实力、风险点澄清
- **期望回答**：每道题附带期望回答要点，便于评估

### 3. 可视化报告
- **交互式HTML**：候选人排序表，点击展开详情
- **可打印**：面试时可随身携带问题清单
- **可分享**：团队成员可共享查看

## 📁 文件结构

```
jobs/                              ← 工作目录
├── personalprefer.txt             ← 面试官偏好配置（全局共享）
│                                    （可选，用于个性化问题）
│
├── 产品经理-增长/                  ← 职位文件夹A
│   ├── JD.txt                     ← 职位A描述
│   └── 简历/                       ← 简历存放
│       ├── 张三.pdf
│       ├── 李四.docx
│       └── ...
│
└── 后端开发-GO/                   ← 职位文件夹B
    ├── JD.txt                     ← 职位B描述
    └── 简历/
        └── ...

输出文件：
└── 面试报告-YYYY-MM-DD.html       ← 生成的报告
```

## 🚀 使用方式

### 方式一：批量筛选
```
用户：帮我筛选 jobs/产品经理-增长 的简历

AI：
✓ 读取 JD.txt
✓ 解析简历/文件夹内的所有简历
✓ 生成 面试报告-2026-03-05.html
```

### 方式二：单个准备
```
用户：帮我准备 jobs/后端开发-GO/简历/张三.pdf 的面试问题

AI：
✓ 只针对张三的简历生成详细面试清单
```

---

## 💡 为什么好用？

| 传统方式 | 招聘助手 |
|---------|---------|
| 人工阅读每份简历 10-15 分钟 | AI 批量处理，秒级出结果 |
| 主观判断，标准不一 | 结构化评分，客观可对比 |
| 面试问题临时想 | 针对每份简历定制问题 |
| 团队评估标准难统一 | personalprefer.txt 共享配置 |
