# 人格元数据框架 (Persona Metadata Framework)

基于荣格心理学的人格分析 Skill，用于结构化地拆解和描述人物或人格类型的多维度特征。

## 核心能力

本框架提供了一套完整的**五维人格元数据体系**，可对任意人物或人格类型进行系统化分析：

| 维度 | 层级数 | 核心概念 |
|:---|:---:|:---|
| 结构层次 | 3 | 意识自我、个人潜意识、集体潜意识 |
| 原型组件 | 4 | 人格面具、阿尼玛/阿尼姆斯、阴影、自性 |
| 类型与功能 | 4+ | 态度类型、功能类型、八维功能、十六型人格 |
| 动力与发展 | 3 | 心理能、个体化进程、人生两半场 |
| 人际与关系 | 3 | 面具互动、投射机制、影响力原则 |

## 快速开始

### 资料准备

使用框架前需收集分析对象的基础资料：

| 等级 | 必需性 | 内容 |
|:---:|:---:|:---|
| **L1** | 必须 | 出生背景、教育经历、职业轨迹、代表性言论、公众评价、争议事件 |
| L2 | 推荐 | 演讲/访谈中的行为模式 |
| L3 | 理想 | 与他人的互动模式 |
| L4 | 最佳 | 原始视频/音频素材 |

**L1 必需信息清单**：
```
□ 出生背景（年代、地域、家庭阶层）
□ 教育经历（专业、学校类型）
□ 职业轨迹（关键转折点、核心成就）
□ 代表性言论（3-5句核心观点）
□ 公众形象标签（他人如何描述）
□ 争议/冲突事件（如何应对）
```

### 使用流程

```
1. 收集资料 → 2. 结构定位 → 3. 原型识别 → 4. 功能推断 → 5. 整合输出soul.md
```

### 示例分析

**输入**：亚里士多德的哲学立场和行为特征

**输出**（部分）：

```
态度类型：外倾型（E）
主导功能：Te（外倾思维）
辅助功能：Si（内倾感觉）
人格类型：ESTJ/ENTJ
核心特征：务实的条理性
阴影：神秘主义、超越性体验
自性整合：以"具体化"和"折中"实现
```

## 适用场景

- **人物分析**：历史人物、公众人物、文学角色的心理画像
- **自我觉察**：个人人格类型识别与发展路径探索
- **关系分析**：人际互动模式诊断与优化

## 框架哲学

> 本框架将人格视为一个动态流转的生态系统，而非孤立概念的堆砌。每个维度相互关联、互相影响，共同构成完整的人格图谱。
