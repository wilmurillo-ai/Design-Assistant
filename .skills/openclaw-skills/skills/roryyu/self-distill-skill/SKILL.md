# Self-Distill Skill 人格元数据框架

## 1. 概述

### 1.1 框架起源
本框架深度融合了荣格心理学、智力发展轨迹，提供一个多层次、动态关联的全景式人格拆解图谱。

### 1.2 框架核心
通过结构化的人格元数据体系，将人物或人格类型分解为可量化、可分析的多维度组件。

---

## 2. 核心维度体系

### 2.1 结构层次（三层模型）

| 层级 | 概念 | 定义 | 分析要点 |
|:---:|:---|:---|:---|
| 1.1 | **意识自我（Ego）** | 感知、思考、决策的中心，构成人格的表层 | 主导认知倾向、决策模式 |
| 1.2 | **个人潜意识** | 被遗忘、压抑的个人经验，形成"情结" | 内在冲突、情感模式 |
| 1.3 | **集体潜意识** | 人类世代相传的深层心理遗产，核心是"原型" | 文化烙印、普遍心理模式 |

### 2.2 原型组件（四元模型）

| 原型 | 定义 | 分析要点 |
|:---|:---|:---|
| **人格面具（Persona）** | 展示给外部世界的"面孔"，适应社会的工具 | 社会角色、身份认同 |
| **阿尼玛/阿尼姆斯** | 异性特质的内在桥梁与投射源 | 情感模式、关系倾向 |
| **阴影（Shadow）** | 被压抑、否认的面向（个人+集体） | 内在张力、潜在力量 |
| **自性（Self）** | 心灵的整体性与终极中心 | 整合方式、个性化路径 |

### 2.3 类型与功能（四维模型）

#### 态度类型
- **内倾（I）**：力比多内收，定向于内在主观世界
- **外倾（E）**：力比多外投，定向于外在客观世界

#### 功能类型（四大认知）
- **思维（T）**：逻辑判断、因果推理
- **情感（F）**：价值评估、情感判断
- **感觉（S）**：感官感知、具体经验
- **直觉（N）**：整体洞察、可能性捕捉

#### 八维心理功能矩阵
| 功能 | 描述 | 表现特征 |
|:---|:---|:---|
| Se | 外倾感觉 | 具身参与、即时反应 |
| Si | 内倾感觉 | 经验传承、稳定性 |
| Ne | 外倾直觉 | 可能性探索、联想发散 |
| Ni | 内倾直觉 | 模式洞察、深度理解 |
| Te | 外倾思维 | 系统组织、效率优先 |
| Ti | 内倾思维 | 原理追索、逻辑自洽 |
| Fe | 外倾情感 | 社交和谐、价值共鸣 |
| Fi | 内倾情感 | 内在道德、个体价值 |

#### 十六型人格（MBTI简化）
基于主导+辅助功能组合形成的类型画像。

### 2.4 动力与发展（三维模型）

| 维度 | 概念 | 分析要点 |
|:---|:---|:---|
| **心理能（Libido）** | 驱动人格运作的能量，流向决定态度 | 能量分布、关注焦点 |
| **个体化进程** | 实现完整自我的毕生旅程 | 整合路径、成长轨迹 |
| **人生两半场** | 前半场适应社会，后半场回归整合 | 阶段重心、角色转换 |

### 2.5 人际与关系（三维模型）

| 维度 | 描述 | 分析要点 |
|:---|:---|:---|
| **面具互动** | 不同场合扮演角色以换取社会报酬 | 角色切换、社会适应 |
| **投射机制** | 将内在未发展部分投射到他人 | 关系模式、冲突来源 |
| **影响力原则** | 通过理解人性需求赢得合作 | 领导风格、人际策略 |

## 3. 分析方法论

### 3.1 最小资料准备框架

#### 资料分级体系

| 等级 | 资料类型 | 必需性 | 说明 |
|:---:|:---|:---:|:---|
| **L1** | **基础公开信息** | 必须 | 身份背景、主要成就、社会评价 |
| L2 | **行为模式记录** | 推荐 | 演讲/访谈/著作中的表达模式 |
| L3 | **关系互动描述** | 理想 | 与他人的互动模式、他人评价 |
| L4 | **原始素材** | 最佳 | 视频/音频/一手访谈（非二手转述） |

#### L1 必需信息清单

```
□ 出生背景（年代、地域、家庭阶层）
□ 教育经历（专业、学校类型）
□ 职业轨迹（关键转折点、核心成就）
□ 代表性言论（3-5句核心观点）
□ 公众形象标签（他人如何描述）
□ 争议/冲突事件（如何应对）
```

#### 资料质量判断

| 质量 | 特征 | 可信度 |
|:---:|:---|:---:|
| **优秀** | 原始视频/音频、本人著作 | 高 |
| **良好** | 深度访谈、一手报道 | 较高 |
| **一般** | 二手转述、概括性描述 | 中 |
| **较差** | 猜测、标签化评价 | 低 |

> **重要**：资料越接近原始来源越好。二手转述可能扭曲原意，应交叉验证。

#### 资料分析示例（以张雪峰为例）

**收集到的L1资料：**
```
□ 出生背景：1984年，黑龙江齐齐哈尔，普通家庭
□ 教育经历：郑州大学给排水工程专业
□ 职业轨迹：2008年讲台受挫 → 2010年找到风格 → 2016年爆红
□ 代表性言论：
  - "讲的不奇葩不行"
  - "热搜能带来流量，流量能让更多人听到我的内容"
  - "这个专业出来一般去哪"
□ 公众形象："接地气"、"敢说真话"、"功利"
□ 争议事件：替朋友还房贷算错账、直播中情绪激动指人
```

**L1→推断分析：**
| 资料 | 可推断内容 |
|:---|:---|
| 普通家庭出身 + 非热门专业 | 对"稳定"的渴望，身份焦虑 |
| 早期被学生否定 | 不安全感的根源，面具膨胀后的补偿 |
| "讲的不奇葩不行" | Ne功能驱动，差异化竞争策略 |
| "热搜带来流量" | Te功能主导，外倾型能量模式 |
| 替人还贷算错账 | Se冲动的表现，Fi薄弱的证据 |

### 3.2 五步分析流程

```
第一步：收集资料 → 第二步：结构定位 → 第三步：原型识别 → 第四步：功能推断 → 第五步：整合输出
```

1. **收集资料**：获取分析对象的行为记录、著作、言论、历史背景（参考L1-L4分级）
2. **结构定位**：判断意识层级的优先倾向（内倾/外倾）
3. **原型识别**：识别人格面具、阴影、自性特征
4. **功能推断**：确定主导功能与辅助功能组合
5. **整合输出**：形成完整的人格元数据描述

### 3.3 推断逻辑规则

| 推断来源 | 可推断目标 | 推断依据 |
|:---|:---|:---|
| 态度类型（I/E） | 八维主导 | 内倾→思维/感觉/直觉内倾功能；外倾→外倾功能 |
| 主导功能 | 辅助功能 | 通常为同一态度倾向的不同功能 |
| 面具特征 | 阴影特征 | 面具的对立面往往是阴影所在 |
| 外在行为 | 内在动机 | 行为模式背后的心理需求 |

## 6. 框架逻辑关联

```
┌─────────────────────────────────────────────────────────────────┐
│                        集体潜意识                                │
│  （原型：智者、秩序创造者等普遍心理模式）                          │
└─────────────────────────────────────────────────────────────────┘
                              ↑
                    向下渗透/向上整合
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│                        个人潜意识                                │
│  （情结：被压抑的经验、情感冲突）                                  │
└─────────────────────────────────────────────────────────────────┘
                              ↑
                    向外表达/向内觉察
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│                        意识自我（Ego）                            │
│  （主导态度类型 + 主导功能 = 人格类型）                           │
└─────────────────────────────────────────────────────────────────┘
                              ↑
                    面具互动/现实检验
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│                        外部世界                                  │
│  （社会角色、人际关系、成就表现）                                  │
└─────────────────────────────────────────────────────────────────┘

动力系统：
- 心理能（力比多）驱动各层流动
- 个体化进程推动自性整合
- 人生两半场决定发展重心
```

---

## 7. 注意事项

1. **推断而非断言**：人格分析基于行为证据推断，非绝对判定
2. **动态而非静态**：人格随时间和经历发展变化
3. **整合而非评判**：阴影不是负面标签，而是待整合的能量
4. **文化背景**：原型表达受文化语境影响
