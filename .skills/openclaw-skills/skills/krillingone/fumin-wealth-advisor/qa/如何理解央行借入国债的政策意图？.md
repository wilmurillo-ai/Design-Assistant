# 如何理解央行借入国债的政策意图？

- 事件回顾 - 2024年7月1日，央行公告将开展国债借入操作，引发市场关注。

- 国债借入操作 
  
  - 定义：央行向一级交易商借入国债，再在二级市场卖出，压低价格，推升收益率。 
  - 类比：类似资本市场的“融券交易”。 
  - 预计：央行将“借长卖长”，引导长期国债收益率上升。

- 央行借入国债的意图 
  
  - 背景：央行多次提示长债收益率风险，担心收益率过低。 
  - 目的： 
    - 引导长期国债收益率回归合理区间（2.5%-3%）。 
    - 降低市场利率风险，避免类似“硅谷银行”事件发生。 
    - 引导市场预期，保持投资正向激励作用。

- 央行买卖国债的影响 
  
  - 缓和M2回落压力。 
  - 有助于央行资产负债表扩张。

- 债市投资策略 
  
  - 市场逻辑未变，降息可能性仍存，利率下行趋势未变。 
  - 做平曲线，做窄利差需适度。 
  - 短期策略：哑铃型，短端下沉信用票息，长端适度考虑利率久期。
