# Source Selection for Car-Sleep Guides

Use the source that matches the decision.

## Hard data

Use maps, official scenic pages, or structured listings for:

- origin → overnight parking distance and duration
- overnight parking → scenic distance and duration
- scenic → scenic transfer time
- tolls
- scenic latest entry / opening hours
- charger existence
- public parking existence
- public toilet existence when the night plan depends on it

If the itinerary depends on sleeping in the car, parking / charging / toilet checks are schedule-critical, not optional detail.

## Parking and charging

Treat these as hard-data decisions:

- public parking vs scenic-dedicated parking
- charger operator / location context if visible
- whether the parking point is in a town center, transport hub, scenic access point, or service area
- distance from the overnight anchor to the next day's first activity
- whether any source explicitly says overnight parking is allowed, restricted, or prohibited

Do not assume a generic public charger parking lot is the best overnight anchor. Prefer, in order:

1. official campsite / campground
2. repeatedly mentioned car-sleep / overnight point
3. service area with strong facilities
4. generic public parking + charger

A public charger parking lot may still be useful as a recharge point or temporary fallback, but it should not outrank a stronger overnight signal source.

## Soft signal

Use notes, forums, and review posts for:

- whether the place feels quiet enough to sleep
- whether a lot is usually crowded
- whether a county-town overnight stop feels safer or simpler
- scenic vibe and避坑
- which exact place names are repeatedly treated as camping / overnight points

When Xiaohongshu detail pages are blocked, search-result clustering is still useful. If multiple results independently converge on the same place name as a camping / overnight point, treat that as meaningful soft signal—but not as formal permission.

Do not use these as the only source for distances, chargers, or opening hours.

## When the user removes lodging mid-plan

Immediately re-check:

- the new night anchor for each day
- anchor → scenic legs
- charger / toilet coverage
- whether the original scenic order still makes sense

Removing lodging is a structural itinerary change, not a cosmetic revision.
