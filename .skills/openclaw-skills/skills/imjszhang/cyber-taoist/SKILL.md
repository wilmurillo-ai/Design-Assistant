# cyber-taoist 进化学应用指南

> **版本**: v1.0.0 | **基于**: [宪章 CONSTITUTION.md](CONSTITUTION.md)

宪章定义了"是什么"，本文告诉你"怎么用"。
以下方法适用于任何尺度——从个人职业选择到文明兴衰，框架相同，填入内容不同。

---

## 一、五元分析框架（N/R/T/EC/NI）

用七步拆解你关心的任意情境。每一步附一个引导问句，帮你启动思考。

**第 1 步：定主体**
你要分析谁？一个人、一家公司、一个行业、还是一个文明？
> 主体确定后，后续所有概念都围绕它展开。

**第 2 步：辨层次**
你的主体处于哪个尺度？微观（个体）、中观（组织/物种）、还是宏观（生态系统/文明）？
> 同一框架在不同尺度自相似地重复——参见宪章第十五条。

**第 3 步：识天道（N）**
当前有哪些不可逆的客观趋势正在改变游戏规则？
> 天道不可直接观测，只能通过后果感知：顺之则存，逆之则亡——参见第二条。

**第 4 步：审法则（R）**
主体当前遵循的规则、制度、惯例、技术范式是什么？它们是否已滞后于天道？
> 法则是防火墙：保护你免于直面天道，但滞后则成牢笼——参见第三条。

**第 5 步：观交易（T）**
主体当前的主要交互模式是什么？交易空间是否正被法则挤压？
> 法则繁则交易稀，生态脆——参见第八条。如果你感到"什么都不能做"，这是危险信号。

**第 6 步：定生态位（NI）**
主体在现行法则框架下的位置如何？资源获取能力强还是弱？
> 警惕：高生态位 ≠ 安全。NI 只衡量"适人之道"，非"适天道"——参见第十条。

**第 7 步：判阶段**
对照以下五阶段模型（参见第十三条），判断你的主体当前处于哪个阶段：

1. **常规阶段** — 法则内交易顺畅，生态位稳步提升
2. **感知滞后** — 天道已变，法则现偏差，交易反馈开始模糊
3. **试探性突破** — 部分主体冒险越过法则边界，试探天道
4. **成败筛选** — 契合天道者存活，触犯天道者灭亡
5. **规则更新** — 成功经验沉淀为新法则，新一轮循环开始

## 二、决策启发式：守与破

完成七步分析后，你面临的核心决策是：**在法则内优化（守），还是突破法则（破）？**

| 信号 | 判断 | 行动 | 依据 |
| ---- | ---- | ---- | ---- |
| R 与 N 偏差小，T 反馈清晰 | 法则仍有效 | **守**：在 R 内优化 T，提升 NI | 第十二条·常规进化 |
| R 严重滞后，T 反馈模糊，灭亡风险积聚 | 防火墙已失效 | **破**：冒险突破 R，直接感应 N | 第九条、第十二条·突破性进化 |
| R 过度繁复，T 空间被压缩 | 生态正在僵化 | **预警**：生态脆弱，需简化 R 或寻找新 EC | 第八条 |
| NI 很高，但 R 可能已滞后 | 高位≠安全 | **警惕**：高 NI 只反映适 R，天道变时首当其冲 | 第十条 |

核心原则：适者非最强者，乃能判断何时守、何时破，并在破后成功与新天道相容者——参见第十四条。

## 三、分形分析模板

将上述框架落到纸面。你可以复制下表，在任意尺度填入具体内容：

| 分析维度 | 填写内容 |
| -------- | -------- |
| 主体 | _______ |
| 分析尺度 | 微观 / 中观 / 宏观 |
| 所处生态（EC） | _______ |
| 现行法则（R） | _______ |
| 主要交易（T） | _______ |
| 当前生态位（NI） | _______ |
| 天道趋势（N） | _______ |
| R 与 N 的偏差程度 | 低 / 中 / 高 / 严重滞后 |
| 当前进化阶段（1-5） | _______ |
| 建议策略 | 守 / 破 / 预警 |

**分形提示**：同一事件可在不同尺度各填一张表。例如分析"AI 对程序员的冲击"，你可以分别以个人（微观）、软件行业（中观）、全球劳动力市场（宏观）为主体，各做一次分析。三张表的结论会相互印证、相互补充——这就是第十五条所说的分形自相似。

## 四、典型应用场景

以下示例展示如何套用框架，重在演示思路而非完整分析。

**场景 A：职业决策**
你（主体）是一名传统行业从业者。行业惯例和职业路径是你的法则（R），AI 与自动化浪潮是天道（N）的变化。如果你发现日常工作中的反馈越来越"失真"——努力提升却看不到回报——说明你可能处于"感知滞后"阶段（阶段 2）。此时应评估：是在现有 R 内深耕（守），还是跨领域学习新技能以突破 R（破）？

**场景 B：创业与产品**
你的公司（主体）处于某个市场生态（EC）中。市场规范、用户习惯、竞争格局构成法则（R），底层技术变革是天道（N）。如果监管越来越繁复，创新空间越来越小（第八条信号），说明生态正在僵化。你可以选择：在现行 R 内精细运营（守），或开辟全新品类绕过旧 R（破）。

**场景 C：历史与文明**
以罗马帝国（主体）为例。帝国的法律、行省制度是法则（R），蛮族迁徙与气候变化是天道（N）。当帝国法则日益繁复却无法应对新威胁时（阶段 2→3），边缘行省率先"突破"——脱离帝国自行应对。最终旧法则崩塌（阶段 4），中世纪封建制度作为新法则诞生（阶段 5）。

**场景 D：道德经解读**
用 N/R/T/NI 框架映射古典概念：老子所言"道"对应天道（N），"德"对应主体与天道的相容度，"无为"是减少法则（R）对交易（T）的过度约束以保持生态灵活性。
