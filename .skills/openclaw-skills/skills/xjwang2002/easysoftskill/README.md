# 易软数据查询（EasySoft Reporter）

[![Skill Version](https://img.shields.io/badge/version-1.0.0-blue)]()

> ✅ **跨平台兼容**：基于 OpenClaw Skill 标准开发，可安装运行于 QClaw、OpenClaw 及所有兼容 OpenClaw Skill 协议的小龙虾平台。

物业管理数据报表查询工具。通过自然语言对话，自动调用易软系统接口，查询和导出各类经营数据报表。

---

## 功能特点

- **零配置**：首次使用引导式配置，后续自动加载凭证
- **多报表支持**：欠费报表、收缴率、应收、实收、月报等
- **智能参数匹配**：自动识别小区名称、收费科目、时间范围
- **数据自动美化**：千分位格式化、空值智能显示、列名自动映射
- **凭证安全**：AES-256 加密，机器绑定，不可拷贝滥用
- **跨平台**：QClaw / OpenClaw 及所有小龙虾平台通用

---

## 快速开始

### 安装

在 QClaw/OpenClaw 中输入：

```
skillhub install easysoftskill
```

或由 AI 助手自动提示安装。

### 首次配置

安装后首次使用时，AI 会分步引导你完成配置：

1. 输入物业公司 Key（由易软公司提供）
2. 输入易软系统账号
3. 输入易软系统密码
4. 系统自动验证并加密存储凭证

### 查询示例

> 查询 2026 年 3 月的欠费情况

```
帮我查一下2026年3月的欠费报表
```

AI 自动引导你选择小区、科目、报表类型，确认后返回结果。

> 查询某小区的收缴率

```
查一下A小区的收缴率
```

## 数据展示

查询结果自动美化处理：

- 数值自动添加千分位（如 `88,256.78`）
- 零值智能显示（显示 `-`）
- 百分比自动格式化
- 多层级表头自动拆解
- 查询后自动推荐同组其他报表
