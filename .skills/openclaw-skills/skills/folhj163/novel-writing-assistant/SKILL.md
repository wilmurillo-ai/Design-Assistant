# SKILL.md - 小说创作助手

## 🎯 技能概述
**技能名称**：小说创作助手  
**技能类型**：创作指导、结构化写作辅助  
**适用对象**：写作新手、想要系统化创作长篇小说的作者  
**核心价值**：降低创作门槛，提高写作效率，保证作品质量  
**技能口号**：从0到1，八步完成你的第一部小说

## 📚 技能亮点
1. **结构化引导**：八步流程，循序渐进，避免创作迷茫
2. **专业模板**：提供标准化文档模板，确保作品质量
3. **实时审核**：一致性检查，避免人设崩塌和逻辑矛盾
4. **多轮交互**：支持长期创作，随时回溯修改
5. **飞书集成**：所有文档自动生成飞书文档，便于协作

## 🔍 触发条件
当用户提到以下关键词时自动触发：
- "写小说"、"长篇小说"、"小说创作"
- "小说大纲"、"人物设定"、"世界观构建"
- "写作指导"、"小说写作助手"
- "我需要写一本小说"、"帮我写小说"
- "创作灵感"、"小说构思"、"写作规划"
- "玄幻小说"、"都市小说"、"仙侠小说"（各类题材）
- "系统流"、"穿越重生"、"无敌流"（各类卖点）

**触发逻辑**：智能识别创作意图，提供针对性引导

## 🚀 八步创作流程详解

### 第一步：小说灵感生成
**目标**：确定小说基本方向，提供3套完整创意方案
**交互流程**：
```
用户：我想写一本小说
助手：欢迎使用小说创作助手！我们开始第一步：灵感生成。

请回答以下问题：
1. 您希望写什么题材？
   [玄幻/都市/仙侠/言情/悬疑/科幻/历史/军事/游戏/其他]

2. 目标读者群体是？
   [学生/上班族/喜欢爽点/喜欢深度剧情/喜欢甜宠/喜欢悬疑推理]

3. 核心卖点偏向？
   [系统流/穿越重生/无敌流/种田流/逆袭打脸/甜宠日常/悬疑解谜]

4. 期望字数？
   [30万字/50万字/100万字/200万字+]

5. 更新频率？
   [日更/周更/不定期]
```

**生成内容**：
1. **3个备选书名**：直白有冲突，吸引目标读者
2. **一句话简介**：主角+困境+金手指+目标（3句话版本）
3. **核心冲突**：主角目标+最大阻碍+对抗方式
4. **核心爽点**：每章都能让读者获得的情感满足
5. **市场定位**：竞争分析+差异化优势

**输出格式**：飞书文档《小说创意方案对比表》

### 第二步：世界观构建
**目标**：基于选定题材构建完整、自洽的世界观
**交互流程**：
```
助手：现在开始构建世界观。基于您选择的{{题材}}，我需要了解一些细节：

1. 时代背景偏向？
   [古代/近代/现代/未来/混合/架空中世纪]

2. 力量体系类型？
   [修仙等级/魔法元素/科技文明/异能觉醒/混合体系]

3. 社会结构特点？
   [封建王朝/民主联邦/宗门统治/部落联盟/城邦自治]

4. 主要冲突来源？
   [资源争夺/理念对立/种族矛盾/阶级冲突/外部威胁]

5. 特殊规则或禁忌？
   [例如：不能随意杀人/有天道制约/科技限制等]
```

**生成内容**：
1. **基础框架**：时代背景、地域分布、社会结构完整设定
2. **核心规则**：力量体系详细等级、修炼方式、突破条件
3. **势力地图**：主要国家、宗门、家族、组织的分布与关系
4. **历史脉络**：关键历史事件时间线，影响当前格局
5. **细节设定**：货币系统、度量衡、语言文字、风俗习惯
6. **禁忌常识**：社会潜规则、绝对禁忌、常识认知
7. **关键场景**：主要城市、秘境遗迹、战场要塞详细描述

**输出格式**：飞书文档《世界规则文档》（使用模板生成）

### 第三步：人物塑造
**目标**：创建立体、有深度、令人难忘的角色
**交互流程**：
```
助手：开始人物塑造。首先创建主角：

1. 主角基础信息：
   - 姓名偏好：[传统中式/现代风格/异域风情/其他]
   - 年龄设定：[少年15-18/青年19-25/中年26-40/其他]
   - 出身背景：[平民/贵族/穿越者/重生者/特殊血脉]

2. 性格特点：
   - 核心性格：[热血正直/冷静理智/腹黑狡猾/玩世不恭/其他]
   - 最大优点：[坚韧/智慧/善良/领导力/其他]
   - 致命缺点：[冲动/多疑/骄傲/优柔寡断/其他]

3. 金手指设定：
   - 类型：[系统/老爷爷/神器/知识/血脉/其他]
   - 限制条件：[有代价/需成长/有副作用/其他]

需要创建其他重要角色吗？[是/否]
```

**生成内容**：
1. **主角深度剖析**：
   - 完整背景故事：童年创伤、关键经历、性格形成
   - 心理深度：内心矛盾、恐惧渴望、成长需求
   - 能力体系：当前实力、成长路径、终极潜力
   - 关系网络：与各角色的情感纽带和利益关联

2. **配角阵容**：
   - 重要盟友：性格互补、能力辅助、情感支持
   - 主要对手：理念冲突、利益对立、个人恩怨
   - 暧昧对象：情感纠葛、成长陪伴、最终选择
   - 功能性角色：导师、信息源、氛围组

3. **关系图谱**：
   - 情感纽带：爱情、友情、亲情、师徒情
   - 利益关联：同盟、竞争、雇佣、胁迫
   - 隐藏关系：血缘、前世、秘密联系

**输出格式**：
1. 飞书文档《主角人物小传》（详细版）
2. 飞书文档《配角人物图鉴》（简洁版）
3. 飞书文档《人物关系图谱》（可视化）

### 第四步：大纲生成
**目标**：制定完整的三幕式小说结构，确保节奏合理
**交互流程**：
```
助手：开始制定小说大纲。基于已有的{{字数规划}}：

1. 整体结构偏好？
   [传统三幕式/多卷连载/单元剧形式/其他]

2. 高潮分布期望？
   [每10章小高潮/每30章中高潮/每100章大高潮]
   
3. 感情线重要性？
   [重要，与主线并行/次要，点缀作用/几乎没有]

4. 结局类型倾向？
   [圆满结局/开放式结局/悲剧结局/反转结局]

5. 需要特别强调的主题？
   [成长/救赎/复仇/爱情/自由/其他]
```

**生成内容**：
1. **三幕式详细结构**：
   - 第一幕（1-20%）：建立常态 → 打破常态 → 接受使命
   - 第二幕（20-75%）：试炼成长 → 中点转折 → 至暗时刻 → 重新崛起
   - 第三幕（75-100%）：最终准备 → 高潮对决 → 结局余韵

2. **三线并行规划**：
   - **剧情主线**：核心事件链，冲突升级路径
   - **成长副线**：实力提升节点，心智成熟过程
   - **情感副线**：关系发展轨迹，情感变化脉络

3. **单元拆分设计**：
   - 每10章为一个单元，明确单元目标
   - 每个单元包含：小目标、小冲突、小高潮、伏笔埋设
   - 单元间衔接自然，节奏起伏有致

4. **关键节点标注**：
   - 开篇钩子（第1章）：必须吸引读者
   - 第一个高潮（第10章）：确立故事方向
   - 中点转折（第50章）：提升故事格局
   - 至暗时刻（第65章）：情感最低点
   - 最终高潮（第85章）：所有矛盾爆发
   - 结局余韵（第100章）：满足中留有回味

**输出格式**：飞书文档《小说详细大纲》（含三幕式结构图）

### 第五步：章节规划
**目标**：为具体章节制定可执行的详细计划
**交互流程**：
```
助手：开始规划具体章节。您想规划哪一章？

1. 章节范围？
   [单章规划/连续5章/连续10章/其他]

2. 本章在大纲中的位置？
   [开篇部分/发展部分/高潮部分/结局部分]

3. 本章主要功能？
   [推进主线/展现人物/埋设伏笔/情感发展/其他]

4. 预期读者感受？
   [紧张刺激/感动落泪/会心一笑/思考反思]

5. 特别注意事项？
   [需要回收伏笔/引入新人物/环境描写重点]
```

**生成内容**：
1. **章节核心设计**：
   - 章节编号与标题：吸引眼球，暗示内容
   - 核心事件：明确本章要发生的关键事件
   - 章节目标：必须完成的情节推进和人物展现
   - 主要冲突：内部冲突/外部冲突/人际冲突设计

2. **场景详细规划**：
   - 场景数量：通常1-3个，避免频繁跳转
   - 场景衔接：时间、地点、视角的转换方式
   - 场景功能：每个场景在章节中的具体作用
   - 环境描写：服务于情节和氛围的细节设计

3. **人物出场安排**：
   - 主要人物：目标、障碍、情感变化、关键动作
   - 次要人物：功能性角色，出场时长，记忆点
   - 人物互动：对话设计、冲突展现、关系进展

4. **节奏精准控制**：
   - 开头（0-25%）：快速切入，建立悬念
   - 发展（25-75%）：情节展开，情感积累
   - 高潮（75-90%）：冲突爆发，情感释放
   - 结尾（90-100%）：结果呈现，新悬念

5. **写作实用提示**：
   - 重点描写：动作、对话、环境的关键点
   - 避免问题：节奏拖沓、信息过载、人设矛盾
   - 本章亮点：必写的精彩之处和加分项

**输出格式**：飞书文档《第X章执行表》（含自查清单）

### 第六步：正文生成
**目标**：根据详细规划生成高质量章节正文
**交互流程**：
```
助手：开始生成正文。基于《第X章执行表》：

1. 文风偏好？
   [简洁明快/细腻描写/幽默风趣/严肃深沉]

2. 视角选择？
   [主角视角/上帝视角/多视角切换]

3. 对话比例？
   [高对话比例/平衡型/低对话高描写]

4. 悬念强度？
   [强悬念，结尾留大钩子/中等悬念/弱悬念]

5. 情感浓度？
   [高情感，强调内心戏/适中情感/情节驱动]
```

**生成原则**：
1. **严格遵守设定**：
   - 人物性格一致性（不OOC）：行为符合人物小传
   - 世界规则一致性：不违背已建立的世界观
   - 情节推进准确性：完成章节规划的所有目标

2. **写作质量标准**：
   - **动作优先**：多用"展示"少用"讲述"，通过动作展现人物
   - **对话自然**：符合人物身份，有潜台词，推动情节
   - **环境服务**：描写服务于氛围营造和情节需要
   - **节奏把控**：张弛有度，避免平铺直叙或过于跳跃

3. **章节结构优化**：
   - **开头抓人**：前200字必须吸引读者继续阅读
   - **中间充实**：情节推进有力，信息交代清晰
   - **高潮到位**：情感爆发充分，冲突解决合理
   - **结尾留白**：完成本章任务同时设置新悬念

4. **读者体验注重**：
   - **代入感**：让读者能感受到主角的情感和选择
   - **爽点安排**：根据题材安排适当的满足感时刻
   - **信息节奏**：重要信息分批交代，避免信息过载
   - **情感共鸣**：触动读者情感，建立情感连接

**输出内容**：
1. 完整章节正文（2000-3000字，符合网络小说标准）
2. 章节摘要：核心事件、情感变化、伏笔设置
3. 写作统计：字数、对话比例、场景数量、情感强度

**输出格式**：飞书文档《第X章正文》（可直接发布或修改）

### 第七步：一致性审核
**目标**：确保已写内容的一致性、合理性和高质量
**交互流程**：
```
助手：开始一致性审核。请提供需要审核的内容：

1. 审核范围？
   [单章审核/连续章节/已写全部内容]

2. 重点审核维度？
   [人设一致性/设定一致性/逻辑合理性/伏笔回收/风格统一性]

3. 审核严格程度？
   [宽松：只检查重大矛盾/标准：全面检查/严格：细节追究]

4. 是否检查文笔问题？
   [是：包括语言表达/否：只检查内容逻辑]
```

**审核维度详解**：
1. **人设一致性审核**：
   - 行为一致性：角色行为是否符合其性格、背景、动机
   - 语言一致性：对话是否符合人物身份、教育水平、当前情绪
   - 成长合理性：性格变化是否有充分铺垫和合理原因
   - 关系稳定性：人物关系发展是否符合逻辑和情感基础

2. **设定一致性审核**：
   - 世界观遵守：是否违背已建立的世界规则
   - 力量体系：能力使用是否符合等级设定和限制条件
   - 时空逻辑：时间线是否清晰，地点转换是否合理
   - 社会规则：行为是否符合该社会的风俗、法律、常识

3. **逻辑合理性审核**：
   - 情节逻辑：事件发生是否有充分原因和合理过程
   - 动机合理：人物行动是否有可信的动机支撑
   - 因果关系：前因后果是否清晰，无逻辑跳跃
   - 概率合理：巧合发生是否在可接受范围内

4. **伏笔回收审核**：
   - 伏笔追踪：已埋设的伏笔是否被遗忘
   - 回收时机：伏笔回收时机是否恰当，不过早也不过晚
   - 回收质量：伏笔解答是否令人满意，有无漏洞
   - 新伏笔检查：新埋设的伏笔是否清晰可追踪

5. **风格统一性审核**：
   - 文风稳定：语言风格是否保持统一
   - 节奏协调：章节间节奏变化是否自然
   - 视角一致：视角转换是否清晰合理
   - 情感连贯：情感发展是否自然流畅

**审核输出**：
1. **问题清单**：
   - 严重问题（必须修改）：人设崩塌、逻辑硬伤、设定矛盾
   - 一般问题（建议修改）：节奏失调、伏笔模糊、情感突兀
   - 轻微问题（可选修改）：表达粗糙、细节缺失、冗余描述

2. **修改建议**：
   - 具体位置：章节、段落、句子精确定位
   - 问题描述：清晰说明问题所在
   - 修改方案：提供可操作的具体修改建议
   - 修改优先级：高、中、低优先级排序

3. **一致性评分**：
   - 总体评分：0-10分，10分为完美一致
   - 分项评分：各维度的具体评分
   - 改进建议：提升一致性的具体方法

**输出格式**：飞书文档《一致性审核报告》（含问题清单和修改建议）

### 第八步：润色优化
**目标**：将合格文字提升为优秀作品，增强阅读体验
**交互流程**：
```
助手：开始润色优化。请提供需要优化的内容：

1. 优化重点方向？
   [语言精炼/对话优化/节奏调整/氛围强化/细节补充/综合优化]

2. 目标文风？
   [保持原风格/更文学性/更网络化/更影视化]

3. 读者层次考虑？
   [小白读者：简单直白/中层读者：适度文学/资深读者：精致深刻]

4. 是否保留个人特色？
   [尽量保留/适度调整/全面优化]

5. 优化深度？
   [轻度：只改明显问题/中度：全面优化/深度：重写级别]
```

**优化维度详解**：
1. **语言精炼优化**：
   - 删除冗余：去除重复表述、多余修饰、废话空话
   - 增强力度：替换软弱词汇，使用更有力的表达
   - 统一用词：规范专有名词、力量体系术语、人物称呼
   - 语法修正：调整病句、优化句式结构、规范标点

2. **对话自然优化**：
   - 符合人设：根据人物身份、教育、性格调整对话
   - 潜台词：让对话有言外之意，增加深度
   - 节奏感：对话长短结合，避免长篇大论或过于简短
   - 信息传递：确保对话推动情节或展现人物

3. **节奏控制优化**：
   - 段落调整：长短段落交替，控制阅读速度
   - 情节密度：调整信息量，避免过密或过疏
   - 情感曲线：强化情感起伏，增强代入感
   - 悬念设置：优化悬念位置和强度

4. **氛围强化优化**：
   - 环境描写：增强画面感，让场景"活"起来
   - 情绪渲染：通过细节描写强化情感氛围
   - 象征运用：使用意象、隐喻增加文学深度
   - 感官细节：补充视觉、听觉、嗅觉、触觉细节

5. **细节补充优化**：
   - 动作细节：让人物动作更具体、有特点
   - 心理细节：合理增加内心活动，但不冗长
   - 环境细节：补充场景细节，增强真实感
   - 文化细节：增加世界观特有的文化元素

**优化输出**：
1. **润色后正文**：
   - 直接可用的优化版本
   - 改动处高亮标注（如使用颜色或注释）
   - 优化说明：简要说明优化思路

2. **修改详细说明**：
   - 语言优化：具体修改了哪些表达，为什么
   - 对话优化：调整了哪些对话，改进效果
   - 节奏优化：如何调整节奏，读者感受变化
   - 氛围优化：强化了哪些氛围，使用的手法
   - 细节优化：补充了哪些细节，作用是什么

3. **优化前后对比**：
   - 关键段落对比展示
   - 优化效果量化说明（如：阅读流畅度提升30%）
   - 读者预期感受变化描述

4. **进一步优化建议**：
   - 如需要深度优化，提供具体建议
   - 如已达到出版标准，给予肯定
   - 后续写作的改进方向

**输出格式**：
1. 飞书文档《润色优化版本》（含改动标注）
2. 飞书文档《优化详细说明》（含优化思路）
3. 飞书文档《优化前后对比》（展示改进效果）

## 交互模式

2. **结构化输出**：所有输出均为结构化格式，便于修改和引用
3. **渐进式推进**：上一步完成后自动进入下一步
4. **随时回溯**：允许用户返回修改之前的设定
