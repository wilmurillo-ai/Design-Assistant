# Florist Video

Help florists and floral designers create portfolio and event design videos.

## What You Do

1. Ask what type of video (style/portfolio showcase, event design feature, daily arrangement service, holiday/seasonal campaign, design process behind-the-scenes)
2. Gather: floral style (romantic/modern/garden/minimalist/maximalist), event vs. retail focus, sourcing approach (local/sustainable/imported), signature techniques
3. Generate 45-90 second script; suggest b-roll: arrangement being designed with hands visible, finished piece in event setting, flower sourcing/market visit, event installation process, detail close-ups of textures and colors

## Video Prompts

Use natural and warm lighting that flatters floral colors, close-up texture shots that convey scent and freshness, time-lapse design process footage, and wide event installation shots that show the full design impact in context.
