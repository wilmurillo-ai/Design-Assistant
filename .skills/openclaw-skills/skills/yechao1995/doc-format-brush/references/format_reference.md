## 格式属性覆盖范围

### 页面格式
| 属性 | 说明 |
|------|------|
| left_margin_cm | 左页边距（厘米） |
| right_margin_cm | 右页边距（厘米） |

### 段落格式
| 属性 | 说明 | 常见值 |
|------|------|--------|
| alignment | 对齐方式 | JUSTIFY、CENTER、LEFT、RIGHT |
| space_before_pt | 段前间距（磅） | 0 |
| space_after_pt | 段后间距（磅） | 0 |
| line_spacing_rule | 行距类型 | MULTIPLE、EXACTLY、AT_LEAST |
| line_spacing_value | 行距值 | 0.5、1.0、1.5、2.0 |
| first_line_chars | 首行缩进（字符数） | 2 |
| snap_to_grid | 与网格对齐 | True/False |

### 字符格式
| 属性 | 说明 |
|------|------|
| font_name | 中西文字体名称 |
| east_asia_font | 中文字体 |
| font_size_pt | 字号（磅） |
| bold | 加粗 |
| italic | 斜体 |

## 标题层级识别正则

```
一级标题: ^[\u4e00-\u9fa5一二三四五六七八九十百]+、
二级标题: ^（[\u4e00-\u9fa5一二三四五六七八九十百]+）
文号:     ^[\u4e00-\u9fa5a-zA-Z]+〔\d+〕\d+号
```

## 常见问题

### Q: 字体没有变化？
A: 检查是否同时设置了 `w:eastAsia` 属性。中文文档必须设置该属性。

### Q: 首行缩进变成厘米？
A: 使用 `w:firstLineChars` XML 属性而非磅/厘米。

### Q: 如何只格式化部分段落？
A: 在 apply_format.py 的主循环中添加过滤条件。
