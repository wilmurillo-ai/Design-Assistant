# PDF转Word格式参考

## 支持的格式元素

| 元素 | 支持情况 | 说明 |
|-----|---------|------|
| 字体名称 | ✅ 支持 | 自动映射到Word可用字体 |
| 字号 | ✅ 支持 | 精确保留pt值 |
| 粗体 | ✅ 支持 | |
| 斜体 | ✅ 支持 | |
| 下划线 | ✅ 支持 | |
| 删除线 | ✅ 支持 | |
| 文本颜色 | ✅ 支持 | RGB颜色 |
| 上标/下标 | ✅ 支持 | |

### 段落格式
| 元素 | 支持情况 | 说明 |
|-----|---------|------|
| 对齐方式 | ✅ 支持 | 左/中/右/两端/分散 |
| 行间距 | ✅ 支持 | 单倍/1.5倍/双倍/固定值 |
| 段前间距 | ✅ 支持 | |
| 段后间距 | ✅ 支持 | |
| 首行缩进 | ✅ 支持 | 字符级缩进 |

### 页面元素
| 元素 | 支持情况 | 说明 |
|-----|---------|------|
| 文本框 | ✅ 支持 | |
| 表格 | ✅ 支持 | 结构完整保留 |

## 限制说明

以下情况转换效果可能不理想：
1. **扫描件PDF** - 需要OCR识别，建议使用专业OCR工具
2. **复杂艺术字** - 艺术字效果可能丢失
3. **3D图表** - 3D效果可能无法保留
4. **加密PDF** - 需要解密后才能转换
5. **纯图片PDF** - 建议使用OCR工具

## 转换质量建议

- 使用 `--dpi 150` 加速大批量转换
- 复杂布局使用默认参数以获得最佳效果
- 转换后检查Word文档，必要时手动调整
