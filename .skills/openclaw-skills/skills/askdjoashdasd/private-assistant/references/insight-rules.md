# Insight Rules

## Goal

洞察链路优先做低打扰、可复核的消费分析：

- 先看账本，再看备忘
- 先给事实，再给最多 1 条建议
- 自动订阅低信号时保持静默

## Signal Priority

1. 异常消费：某分类较上期明显上升，或首次出现较大支出
2. 分类偏移：Top1 类别占比过高，或本期主导分类明显变化
3. 记录质量：`其他` 过多、备注过少，导致后续难以判断异常
4. 同期线索：同周期 memo 里出现作息、焦虑、疲惫、开心等生活节奏信息

## Memo Boundary

- memo 只作为“同期记录”或“可能背景”
- 最多引用 2 条
- 不得写成强因果，例如“你一定是因为熬夜所以消费变多”
- 若没有明显相关 memo，可以完全不引用

## Silence Rules

- 手动查询永不静默
- 周订阅：少于 3 笔账单且无明显异常时静默
- 月订阅：少于 5 笔账单且无明显异常时静默
- 静默时只返回 `[SILENT]`

## Suggestion Rules

- 最多 1 条建议
- 优先顺序：异常消费提醒 > 分类偏移提醒 > 记录质量提醒 > 生活节奏提醒
- 建议要克制、可执行，不要说教
