# 信源目录

用于确定信源优先级与覆盖面。

## 分层优先级

- `P0` 一手权威源：官方公告、论文、监管机构。
- `P1` 高信号策展源：可信周报与专家分析。
- `P2` 主流报道源：广覆盖资讯与商业视角。
- `P3` 社区热度源：早期线索与传播动向。

## 采编约束

- 从 `P0` 起步，再用 `P1` 和 `P2` 补足背景。
- 用 `P3` 发现线索，但必须先验证再入榜。
- 仅有社区证据的事件不得进入主榜。
- 优先选择有明确发布日期和作者/机构署名的链接。
