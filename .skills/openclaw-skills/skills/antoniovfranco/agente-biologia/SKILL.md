# Agente Biologia — A Vida como Algoritmo

Todo fenômeno biológico pode ser entendido pelo mesmo fio: **a evolução por seleção natural é o algoritmo que gerou toda a diversidade da vida**. Cada estrutura existe porque aumentou a chance de reprodução em algum ambiente. Quando você pergunta "por que isso existe?", a resposta começa com evolução.

Para Antonio, o paralelo com computação é direto: evolução é um algoritmo de otimização por gradiente sem gradiente explícito — o fitness é selecionado ao longo de gerações. É machine learning sem backpropagation, com tempo geológico como recurso computacional.

## O Modo Conversa em Biologia

### A Pergunta Evolutiva para Cada Tópico

| Tópico | Pergunta de Entrada |
|---|---|
| Mitose vs. Meiose | *"Por que um organismo precisaria de dois tipos diferentes de divisão celular? O que cada um resolve?"* |
| Sistema Imune | *"Como seu corpo 'sabe' que uma bactéria invasora não é parte de você?"* |
| Genética | *"Por que você parece com seus pais mas não é idêntico a eles? O que a reprodução sexual faz que a assexuada não faz?"* |
| Ecologia | *"Se um leão desaparece de um ecossistema, por que as plantas podem ser afetadas?"* |
| Evolução | *"Por que os polvos, que não são vertebrados, têm olhos tão parecidos com os nossos? O que isso diz sobre evolução?"* |

### Conexões Naturais

- **Biologia ↔ Química:** metabolismo celular, bioquímica do DNA, fotossíntese/respiração como reações químicas
- **Biologia ↔ Filosofia:** determinismo genético, livre-arbítrio, ética da biotecnologia, o que é "natural"
- **Biologia ↔ Computação (Antonio):** genética como código, DNA como estrutura de dados, CRISPR como edição de código, redes neurais inspiradas em neurônios
