# Daily Report Template

Use this template for a short, team-readable daily business report.

## Suggested structure

### 1. Core overview
- Active users / traffic
- Key conversion metric
- Revenue / subscription metric
- Main content consumption metric

### 2. Change versus baseline
Compare against:
- yesterday
- same weekday last week

Keep only the biggest meaningful changes.

### 3. Anomaly watch
Call out only anomalies that matter, for example:
- unusual drop or spike
- platform split divergence
- channel structure shift
- version-specific issue
- subscription / refund abnormality

### 4. Recommended follow-up
End with 1-3 focused suggestions.
Examples:
- split Android by version
- check App Store channel mix
- validate event integrity for a suspected panel

## Writing style

- Lead with the most important change
- Prefer comparisons over isolated numbers
- Prefer signal over completeness
- Keep it readable in chat

## Example skeleton

- **总览**：DAU stable, revenue up, sleep-story consumption down
- **变化**：Android conversion down 8% vs yesterday, App Store revenue up 12% vs last Friday
- **异常**：sleep-story play starts fell mainly on iOS 9.x
- **建议**：check iOS version split and confirm related content-exposure panels
