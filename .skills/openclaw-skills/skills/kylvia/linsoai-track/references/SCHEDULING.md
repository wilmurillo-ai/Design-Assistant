# 调度频率速查表

## Cron 表达式格式

```
分  时  日  月  周
│   │   │   │   │
│   │   │   │   └── 周几 (0=周日, 1=周一, ..., 6=周六)
│   │   │   └────── 月份 (1-12)
│   │   └────────── 日期 (1-31)
│   └────────────── 小时 (0-23)
└────────────────── 分钟 (0-59)
```

## 常用 Cron 表达式

| 频率 | 表达式 | 命令参数 |
|------|--------|----------|
| 每天 9:00 | `0 9 * * *` | `--cron "0 9 * * *"` |
| 每天 9:00 和 18:00 | `0 9,18 * * *` | `--cron "0 9,18 * * *"` |
| 工作日 9:00 | `0 9 * * 1-5` | `--cron "0 9 * * 1-5"` |
| 每周一 10:00 | `0 10 * * 1` | `--cron "0 10 * * 1"` |
| 每周五 17:00 | `0 17 * * 5` | `--cron "0 17 * * 5"` |
| 每月 1 号 8:00 | `0 8 1 * *` | `--cron "0 8 1 * *"` |
| 每月 15 号 8:00 | `0 8 15 * *` | `--cron "0 8 15 * *"` |
| 每季度首日 | `0 9 1 1,4,7,10 *` | `--cron "0 9 1 1,4,7,10 *"` |
| 每天 0:00 (午夜) | `0 0 * * *` | `--cron "0 0 * * *"` |
| 每天凌晨 3:00 | `0 3 * * *` | `--cron "0 3 * * *"` |

## 间隔执行 (--every)

| 频率 | 命令参数 |
|------|----------|
| 每 15 分钟 | `--every 15m` |
| 每 30 分钟 | `--every 30m` |
| 每小时 | `--every 1h` |
| 每 2 小时 | `--every 2h` |
| 每 3 小时 | `--every 3h` |
| 每 6 小时 | `--every 6h` |
| 每 12 小时 | `--every 12h` |

## 一次性执行 (--at)

```bash
# ISO 8601 格式
--at "2026-03-15T14:00"

# 搭配时区
--at "2026-03-15T14:00" --tz "Asia/Shanghai"
```

## 自然语言到参数映射

| 用户说法 | 参数 |
|----------|------|
| "每天早上9点" | `--cron "0 9 * * *"` |
| "每天" (未指定时间) | `--cron "0 9 * * *"` (默认 9:00) |
| "每周一" | `--cron "0 9 * * 1"` |
| "每周一三五" | `--cron "0 9 * * 1,3,5"` |
| "工作日" | `--cron "0 9 * * 1-5"` |
| "周末" | `--cron "0 9 * * 0,6"` |
| "每月1号" | `--cron "0 9 1 * *"` |
| "每3小时" | `--every 3h` |
| "半小时一次" | `--every 30m` |
| "明天下午2点" | `--at "明天日期T14:00"` |
| "下周一" | `--at "下周一日期T09:00"` |
