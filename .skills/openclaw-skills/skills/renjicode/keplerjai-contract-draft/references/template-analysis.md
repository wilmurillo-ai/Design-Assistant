# 模板说明

这份文档只负责描述当前示例模板的结构特点，以及它对 workflow 的影响。

## 示例模板

当前 skill 自带的示例模板是一份接近完整的战略合作协议，而不是空白表单。

它通常包含：

1. 合同标题
2. 鉴于 / 背景
3. 合作原则
4. 合作内容
5. 权利与义务
6. 合作机制
7. 知识产权
8. 保密条款
9. 法律适用与争议解决
10. 签署页

## 对执行的影响

由于模板正文语义较重：

1. 不应从头重写整份合同
2. 也不能只填封面和合作内容标题
3. 当合作主题变化时，正文相关段落通常也需要联动改写

## 应优先检查的段落

当合作主题、主体身份或行业背景变化时，优先检查：

1. 鉴于 / 背景
2. 合作原则
3. 甲方权利与义务
4. 乙方权利与义务
5. 合作机制
6. 主体承继或连续性条款
7. 知识产权前置归属
8. 签署页主体名称
