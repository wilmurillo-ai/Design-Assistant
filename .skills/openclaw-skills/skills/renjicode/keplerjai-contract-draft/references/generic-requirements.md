# 通用原则

这份文档只负责说明合同起草 skill 的通用质量要求。

## 基本原则

1. 优先保证模板忠实度，而不是追求风格重写。
2. 保留章节顺序、条款顺序、表格、签署页和原有版式。
3. 缺失信息必须清晰可见，不要伪造关键商业或法律事实。
4. 假设说明、未决事项说明尽量放在合同正文之外。
5. 输出应视为业务和法务审阅草稿，而不是正式法律意见。
6. 甲乙方介绍应尽量基于公开信息形成短摘要，避免空泛套话。

## 适用边界

这份 skill 适合：

1. 基于已有 `.docx` 模板生成草稿
2. 在保留模板结构前提下做定向改写
3. 输出可继续审阅和迭代的 Word 草稿

这份 skill 不适合：

1. 在没有模板时直接从零定稿正式法律文本
2. 为了“看起来完整”而补造未提供的关键事实
