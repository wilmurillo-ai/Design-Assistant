# 钉钉权限申请

## 当前状态

- ✅ Token 获取成功

## 需要开通的权限

根据钉钉 API 返回，你的应用需要开通以下权限：

### 1. 通讯录权限
- 权限码：`qyapi_manage_addresslist`
- 用途：获取部门列表、用户列表

### 2. 考勤权限
- 权限码：`qyapi_attendance` (可能需要)
- 用途：获取打卡记录、考勤报表

---

## 申请方式

### 方式 1: 点击链接直接申请

在应用权限页面直接申请：

```
https://open-dev.dingtalk.com
```

登录后进入你的应用详情页，在「权限管理」中添加权限并提交审批。

### 方式 2: 手动在后台申请

1. 访问 https://open-dev.dingtalk.com
2. 进入「应用开发」→「企业内部开发」
3. 找到你的应用（按应用名称搜索）
4. 点击「权限管理」
5. 点击「添加权限」
6. 搜索并添加：
   - 通讯录管理
   - 考勤管理
7. 提交审批

---

## 审批流程

1. 提交申请后，需要**企业管理员**审批
2. 审批通过后，权限立即生效
3. 重新获取 access_token 即可使用

---

## 验证

权限开通后，运行测试脚本验证：

```bash

```

成功标志：

## 注意事项

- access_token 有效期 2 小时，权限变更后请重新获取
- 审批可能需要几分钟时间生效
- 确保使用企业管理员账号审批
