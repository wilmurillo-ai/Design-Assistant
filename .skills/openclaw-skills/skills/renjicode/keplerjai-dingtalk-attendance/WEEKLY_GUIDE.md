# 考勤周报使用指南

## 📋 功能说明

现在支持两种模式获取考勤数据：

| 模式 | 命令 | 说明 |
|------|------|------|
| 日报 | `node index.js` | 获取昨天的考勤数据 |
| 周报 | `node index.js weekly` | 获取上一周的考勤数据（上周一到上周日） |

---

## 🚀 使用方法

### 方式 1: 命令行运行

**获取日报（昨天）：**
```bash
cd keplerjai-dingtalk-attendance
node index.js
```

**获取周报（上周）：**
```bash
cd keplerjai-dingtalk-attendance
node index.js weekly
```

### 方式 2: 双击运行（推荐）

**日报：** 双击 `run.bat`（如果创建了）  
**周报：** 双击 `weekly.bat`

## ⏰ 定时任务配置

### 日报定时任务

每天早上 9 点自动获取昨天的考勤数据：

```json
{
  "name": "钉钉考勤日报",
  "schedule": {
    "kind": "cron",
    "expr": "0 9 * * *"
  },
  "payload": {
    "kind": "systemEvent",
    "text": "node index.js daily"
  },
  "sessionTarget": "main"
}
```

### 周报定时任务

每周一早上 9 点自动获取上周的考勤数据：

```json
{
  "name": "钉钉考勤周报",
  "schedule": {
    "kind": "cron",
    "expr": "0 9 * * 1"
  },
  "payload": {
    "kind": "systemEvent",
    "text": "node index.js weekly"
  },
  "sessionTarget": "main"
}
```

## 🔧 自定义日期范围

如果需要获取特定日期范围的数据，可以修改 `index.js`：

```javascript
// 在 main() 函数中修改日期范围
const startDate = '2026-04-01';  // 开始日期
const endDate = '2026-04-30';    // 结束日期
```

## 📝 注意事项

1. **周末和节假日**：周末和节假日可能没有打卡记录，属正常现象
2. **请假人员**：请假人员会显示在"请假记录"工作表中
3. **文件命名**：Excel 文件带时间戳，避免覆盖
4. **数据量**：周报数据量较大，生成需要 2-3 分钟
