# Platform Guides — Academic Social Publishing

## ResearchGate

**Best for:** Academic networking, paper discovery, citation boost
**Character limit:** ~3,000 (but 500-800 is optimal for engagement)
**Post frequency:** 1-2x per paper, once at publication, once at conference/citation

### Tips
- Use academic hashtags: #PublicationName #ResearchTopic #Methodology
- Include direct link to paper
- Tag co-authors in the post
- Ask a question to drive engagement
- Post in groups relevant to your field (Formation Control, Control Theory, Multi-Agent Systems)

### Hashtag strategy
```
Primary: #[SpecificTopic] (e.g., #FormationControl)
Secondary: #EventTriggered #MultiAgentSystems #UAV #Robotics
Tertiary: #AcademicTwitter #Research #PhDlife
```

## LinkedIn

**Best for:** Professional visibility, industry connections, collaborators
**Character limit:** ~3,000 (but 150-300 for optimal feed reach)
**Post frequency:** 1x per major paper/result

### Tips
- First line is everything — make it a hook, not a title
- Use line breaks liberally (every 2-3 sentences)
- Emoji: 1-3 max, use meaningfully
- End with a question or CTA to drive comments
- Tag people and pages when relevant
- Add a image/card visual if possible

### Post Structure Template
```
[Hook — surprising stat, question, or bold claim]

This is what we found / built / discovered in our latest paper.

Why it matters: [1 sentence on impact]

The approach: [Brief methodology highlight]

Thoughts on what this means for the field? 👇

[Link to paper]

# hashtags
```

## WeChat 公众号

**Best for:** Chinese audience, lab/group communication, broader public outreach
**Character limit:** No hard limit, but 800-1500 characters optimal for WeChat reader
**Post frequency:** Per major paper, or monthly digest

### Tips
- Catchy title is critical (use numbers, contrast, or questions)
- Write for educated non-experts when possible
- Technical terms: introduce with brief plain-language explanation
- Use 2-3 relevant emoji per section
- Include 1-2 visual elements (figure, diagram reference)
- End with a personal note or call to follow

### Title Formulas
```
公式一：[数字/对比] + 核心发现 + [人群]
公式二：问句式 — 你知道吗/为什么...
公式三：热点结合 + 领域词
```

## Video Introduction

**Platform:** Suitable for WeChat Video, B站, YouTube Shorts, or as slide audio
**Duration:** 60-90 seconds (150-200 word narration)
**Style:** Conversational, excited but clear

### Script Template
```
[Scene 1: 0-5s] — Opening hook
"你知道多无人机编队控制最大的难题是什么吗？"

[Scene 2: 5-15s] — Problem context
"在多无人机系统中，通信和计算资源是有限的..."

[Scene 3: 15-40s] — Core contribution
"我们的方法通过...实现了...，在不牺牲控制精度的前提下..."

[Scene 4: 40-60s] — Why it matters + demo/visualization
"这意味着在实际应用中..."

[Scene 5: 60-75s] — Key results (numbers if available)
"实验表明，准确率提升X%，通信频率降低Y倍"

[Scene 6: 75-90s] — Closing
"如果你对这个方向感兴趣，论文链接在描述栏 👇"
```
