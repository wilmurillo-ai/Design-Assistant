## IEEE Transactions on Automatic Control (TAC)

| Field | Value |
|-------|-------|
| **IF (2024)** | ~5.8 |
| **Review Speed** | Slow (12-18 months) |
| **Acceptance Rate** | ~15-20% |
| **Scope** | Pure theory, fundamental control |
| **Focus** | Strong mathematical rigor, stability, optimality proofs |

## Journal Selection Quick Guide

| If Your Paper Is... | Consider... |
|---------------------|-------------|
| Pure theory, strong proofs | TAC, Automatica |
| Theory + motivation from application | Automatica, TCST |
| Control + industrial application | TIE, TCST, IET |
| Control + learning/AI | TNNLS, IEEE TAI |
| Robotics + hardware | TRO, ICRA, IROS |
| Incremental theory, concise | Systems & Control Letters |
| Need fast publication | CDC, ACC, ICRA (conference first) |
| Chinese community target | Asian Journal of Control, 自动化学报 |

---

## Decision Tree

```
Is there strong hardware/robotics experiments?
  YES → ICRA/IROS/TRO/TIE
  NO → Continue

Is the paper pure theory with new proofs?
  YES → TAC/Automatica
  NO → Continue

Is the paper learning-based with theory?
  YES → TNNLS/IEEE TAI
  NO → Continue

Is the paper applied without extensive experiments?
  YES → TCST/IET/Asian Journal of Control
  NO → Continue

Need fast publication?
  YES → CDC/ACC conference
  NO → Target mid-tier journal
```
