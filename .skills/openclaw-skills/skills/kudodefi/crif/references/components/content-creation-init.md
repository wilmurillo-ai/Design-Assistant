## STEP 2: TEMPLATE & OUTPUT TYPE SELECTION

Present options:

```
Format(s):
1. Blog Article
2. X Thread
3. TikTok Script
4. YouTube Script

Output type:
A. brief
B. standard
C. comprehensive

Examples: 1B → Blog standard | 2A → X Thread brief | 1B 3A → Blog + TikTok

Select (e.g., '1B' or '1B 3A'): ___
```

- Valid → apply configuration
- Default/Enter → 1B (Blog Article, standard)
- Invalid → retry once, then defaults

Read ONLY selected template(s):
- 1 → blog-article | 2 → x-thread | 3 → tiktok-script | 4 → youtube-script

> STOP if template missing → report error.
