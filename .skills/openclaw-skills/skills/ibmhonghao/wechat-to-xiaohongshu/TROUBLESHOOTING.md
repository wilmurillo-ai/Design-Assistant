# 故障排查指南 - WeChat to Xiaohongshu

这个文档列出运行时可能遇到的具体问题和解决方案。

## 常见错误信息

### ❌ "Tab not found" 或 "浏览器标签页未找到"

**症状：** 
```
Error: Could not find browser tab for WeChat / Xiaohongshu
```

**原因：**
- OpenClaw Browser Relay 连接断开
- 标签页被关闭
- 扩展失效

**解决：**

```
步骤 1：检查标签页是否还开着
  → 微信后台标签页是否还在？
  → 小红书标签页是否还在？
  → 如果关了，重新打开并登录

步骤 2：重新激活 Relay
  → 点击浏览器工具栏上的 OpenClaw 扩展图标
  → 确认看到绿色 ON 标志
  → 如果没看到，可能需要重新安装扩展

步骤 3：刷新页面
  → 在两个标签页上分别按 F5 刷新
  → 等待 3 秒让页面完全加载

步骤 4：重新试一次
  → 现在应该可以了
```

**如果还是不行：**
- 重启 EasyClaw（关闭再打开）
- 重启浏览器
- 检查是否有浏览器更新（Chrome 自动更新可能需要重启）

### ❌ "Import timeout" 或 "导入超时"

**症状：**
```
Error: WeChat article import to Xiaohongshu timeout after 30 seconds
```

**原因：**

可能的原因从**最常见**到**最少见**：

| 排序 | 原因 | 概率 | 解决时间 |
|------|------|------|--------|
| 1️⃣ | 网络不稳定（微信→小红书 链接转移慢） | 40% | 1-2 分钟 |
| 2️⃣ | 小红书服务器繁忙 | 30% | 5-10 分钟 |
| 3️⃣ | 微信文章链接过期 / 无法访问 | 20% | 重发文章 |
| 4️⃣ | 文章包含不支持的媒体（如视频） | 7% | 手动处理 |
| 5️⃣ | 浏览器缓存问题 | 3% | 清空缓存 |

**解决方案：**

**方案 A：等待重试（最常见的原因）**
```
1. 等待 5 分钟
2. 检查网络速度（打开 Speedtest 或类似工具）
3. 重新执行技能
```

**方案 B：检查微信文章是否公开**
```
1. 复制微信文章链接
2. 在新浏览器标签页打开试试
3. 如果看不了，说明文章不公开或链接失效
4. 重新在微信后台发表一遍
```

**方案 C：清空浏览器缓存**
```
1. 按 F12 打开开发者工具
2. 右键 → 清空所有 Cookie 和缓存数据
3. 关闭 F12
4. 刷新两个标签页（F5）
5. 重新登录微信和小红书
6. 重试
```

**方案 D：手动导入（备选方案）**
```
如果自动导入一直超时，可以手动做：
1. 在微信复制文章链接
2. 在小红书手动点 "导入链接" 按钮
3. 粘贴链接，点 "一键排版"
4. 等待完成（通常快得多）
5. 技能只负责添加话题和发布
```

### ❌ "Publish button disabled" 或 "发布按钮灰色"

**症状：**
```
Error: Publish button is disabled, cannot proceed
```

**原因：**

常见原因（按可能性排序）：

1. **页面还没加载完** (50%) — 图片/文本还在处理
2. **缺少必填字段** (30%) — 标题、描述、封面之一未填
3. **图片上传还未完成** (15%) — 小红书在处理导入的图片
4. **权限问题** (5%) — 账号无发布权限

**解决方案：**

**立即检查清单：**
- [ ] 标题是否显示？
- [ ] 描述/话题是否显示？
- [ ] 封面图是否加载完？
- [ ] 页面右下角有没有加载指示器（转圈）？

**如果发现缺失：**

```
问题                  | 解决方法
---------------------|--------
标题为空              | 重新导入，或手动添加标题
描述为空              | 手动添加描述/话题（至少 10 个字）
封面未显示            | 等待 10-15 秒，小红书在生成
图片还在上传          | 等待，右下角加载完才能发
```

**等待足够长的时间：**

```
第一次导入通常需要：
- 微信导入解析：10-15 秒
- 图片上传和处理：15-20 秒
- 总计：30-40 秒

所以请耐心等待！
```

**如果还是不行：**
```
1. 手动检查一遍内容是否完整
2. 点击其他字段，再点回发布按钮（刷新 UI）
3. 如果还是灰色，可能是权限问题
   → 检查账号是否新注册（新账号需 7 天审核期）
   → 检查是否被限流（过度发布）
```

### ❌ "Content format corrupted" 或 "排版乱掉了"

**症状：**
```
导入完成后，看到文章排版混乱、段落错乱、图片位置不对
```

**原因：**

微信和小红书的 HTML 渲染不同，有些格式无法完美转换：

| 微信格式 | 小红书支持 | 备注 |
|--------|---------|------|
| 标题/正文 | ✅ 完全 | 1:1 转换 |
| 加粗/斜体 | ✅ 完全 | 保留 |
| 链接 | ⚠️ 部分 | 外链可能被去掉 |
| 表格 | ❌ 不支持 | 会转成文本 |
| 代码块 | ✅ 支持 | 但格式可能变 |
| 图片 | ✅ 支持 | 位置和大小可能变 |
| 视频 | ❌ 不支持 | 只保留链接 |
| 音频 | ❌ 不支持 | 无法转移 |

**解决方案：**

**快速修复（5 分钟）：**
```
小红书有编辑功能，发布前可以改：

1. 在"草稿箱"中打开文章
2. 点 "编辑" 按钮
3. 调整排版（拖动段落、删除空行、调整图片位置）
4. 保存，然后发布
```

**预防未来的排版问题：**
```
1. 微信文章写的时候，尽量用简单格式
2. 避免过度嵌套、颜色、大小写混合
3. 表格改成列表形式
4. 视频单独处理（不要混在文章里）
```

**如果排版实在太乱：**
```
1. 不发这个版本
2. 在小红书手动复制粘贴文案
3. 重新格式化（麻烦，但质量更好）
```

### ❌ "Article link expired" 或 "文章链接失效"

**症状：**
```
微信文章链接无法在小红书打开，或被提示"链接已删除"
```

**原因：**

微信公众号文章有以下几种情况会"失效"：

| 情况 | 表现 | 解决方法 |
|-----|------|--------|
| 文章被删除 | 链接 404 | 重新发表文章 |
| 文章被停更 | 链接打不开 | 恢复发布状态 |
| 链接不公开 | 显示"需要登录" | 改成公开可见 |
| 分享链接失效 | 小红书无法识别 | 用直接链接而非分享链接 |
| 账号冻结 | 所有文章无法访问 | 联系微信恢复账号 |

**解决方案：**

```
步骤 1：在微信后台检查文章状态
  → 打开 mp.weixin.qq.com
  → 去 "素材管理" → "已发表"
  → 找到这篇文章
  → 检查是否显示为"已发表"（不是"草稿"或"已删除"）

步骤 2：检查分享设置
  → 点文章旁边的"分析"或"预览"
  → 确认可以用外网链接访问

步骤 3：如果还是不行，重新发表一遍
  → 有时微信链接会变
  → 完全重新发表比修改更可靠
```

### ❌ "Account not logged in" 或 "账号未登录"

**症状：**
```
Error: WeChat / Xiaohongshu account not logged in or session expired
```

**原因：**

- 长时间未操作，Session 过期（通常 30-90 天）
- 被迫登出（比如在其他地方登录了）
- 浏览器 Cookie 被清除

**解决方案：**

```
最快的办法：

步骤 1：打开微信后台标签页
  → 检查右上角是否显示昵称
  → 如果显示 "登录" 按钮，说明掉线了

步骤 2：重新登录
  → 点 "登录" 或 "账号登录"
  → 扫二维码或输入账密
  → 等待加载完成

步骤 3：同样方式重新登录小红书

步骤 4：重新激活 Relay
  → 点 OpenClaw 扩展图标
  → 确认显示绿色 ON

步骤 5：重试技能
```

**避免频繁掉线：**
```
- 在登录界面勾选 "30 天记住我"
- 不要用无痕/隐私窗口
- 定期刷新页面（每周一次），确保 Session 保活
- 避免在多个地方同时登录同一账号
```

## 网络相关问题

### 🌐 "Connection timeout" 或 "连接超时"

**症状：**
```
无法连接到 mp.weixin.qq.com 或 creator.xiaohongshu.com
```

**原因：**

| 原因 | 表现 | 诊断方法 |
|-----|------|--------|
| 国际网络 | 完全无法连接 | 在国外但没 VPN |
| DNS 问题 | "无法解析域名" | `nslookup mp.weixin.qq.com` |
| 防火墙 | 连接被拒 | Windows Defender / 第三方防火墙 |
| ISP 限流 | 连接极慢 | 其他网站正常，只有这两个慢 |
| DNS 污染 | 解析到错误 IP | 用 8.8.8.8 或 1.1.1.1 DNS |

**解决方案：**

**快速诊断：**
```
打开命令行（Win+R，输入 cmd）：

ping mp.weixin.qq.com
  → 如果超时，说明网络无法到达
  → 如果有回复，网络正常

nslookup mp.weixin.qq.com
  → 检查是否解析正确
  → 应该返回一个有效 IP 地址
```

**解决方案（按优先级）：**

```
1️⃣ 检查是否在国外
   → 如果是，需要 VPN 回国内

2️⃣ 检查防火墙
   → Windows 防火墙 → 允许应用通过
   → 检查是否有第三方防火墙阻止 Chrome

3️⃣ 清空 DNS 缓存
   → cmd: ipconfig /flushdns
   → 然后重启浏览器

4️⃣ 修改 DNS
   → Windows 网络设置 → DNS
   → 改为 8.8.8.8 或 1.1.1.1
   → 重启浏览器
```

### 🐢 "Very slow" 或 "特别慢"

**症状：**
```
技能执行时间超过 3-5 分钟，或某一步卡很久
```

**原因：**

通常是网络瓶颈，具体可能是：

- 微信链接解析慢
- 小红书图片处理慢
- 网络延迟高（国际连接）

**解决方案：**

```
步骤 1：测试网速
  → 打开 https://speedtest.net
  → 测速，看下载/上传/延迟
  → 如果延迟 > 200ms，可能太慢了

步骤 2：清除浏览器缓存
  → F12 → Storage → Clear All
  → 这会清除过期的缓存，可能加速

步骤 3：关闭其他应用
  → 特别是下载、视频播放、P2P
  → 这些会占用带宽

步骤 4：换个时间试
  → 夜晚网络可能快些（低峰期）
  → 避免工作时间中午（高峰期）
```

## 权限和账号问题

### ⛔ "Account has no permission" 或 "账号没有权限"

**症状：**
```
Error: Your account doesn't have permission to publish on Xiaohongshu
```

**原因：**

| 原因 | 表现 | 解决时间 |
|-----|------|--------|
| 新账号 | 账号注册 < 7 天 | 等待 7 天 |
| 创作者认证 | 普通账号无法发笔记 | 1-3 天完成认证 |
| 限流惩罚 | 被判定为垃圾账号 | 1-7 天自动解除 |
| 违规封禁 | 发过违规内容 | 联系小红书客服 |
| 登录位置 | 在异常地点登录 | 验证身份即可 |

**解决方案：**

```
步骤 1：检查账号年龄
  → 如果 < 7 天，只能等
  → 期间可以编辑草稿，准备内容

步骤 2：检查是否需要创作者认证
  → 打开 creator.xiaohongshu.com
  → 看右上角有无 "认证" 或 "升级" 提示
  → 如果有，点进去完成认证（通常 5 分钟）

步骤 3：检查是否被限流
  → 查看 "笔记管理" 中最近的发布
  → 如果最近 24 小时内发了 > 5 篇，可能被限流
  → 解决方法：减少发布频率

步骤 4：如果都没问题，可能被封禁
  → 需要联系小红书客服
  → 官网底部有 "联系我们"
```

## Browser Relay 相关问题

### 🔌 "Extension not found" 或 "扩展未安装"

**症状：**
```
Error: OpenClaw Browser Relay extension not found
```

**解决方案：**

```
步骤 1：检查扩展是否已安装
  → Chrome：菜单 → 更多工具 → 扩展程序
  → 找 "OpenClaw Browser Relay"
  → 如果没有，说明没装

步骤 2：安装扩展
  → 打开 EasyClaw UI（通常 http://localhost:3000）
  → 找到 "浏览器扩展" 或 "Browser" 菜单
  → 点 "安装浏览器扩展"
  → 按提示完成安装

步骤 3：验证安装
  → 打开一个网页
  → 右上角工具栏应该看到 OpenClaw 图标
  → 点一下，确认显示 "ON" 状态
```

---

### ❌ "Extension ON but not responding" 或 "扩展在线但无响应"

**症状：**
```
看到绿色 ON 状态，但技能仍报 "tab not found" 或"无法通信"
```

**解决方案：**

```
步骤 1：刷新扩展
  → 点扩展图标
  → 点 "刷新" 或 "重新连接" 按钮

步骤 2：刷新网页
  → F5 刷新微信和小红书标签页

步骤 3：检查 EasyClaw 是否还活着
  → Help → Status
  → 看是否显示 "Connected"

步骤 4：重启 EasyClaw
  → 关闭 EasyClaw
  → 等 5 秒
  → 重新打开

步骤 5：重启浏览器
  → 完全关闭 Chrome（包括所有标签页）
  → 重新打开，重新登录
  → 重新激活扩展
```

## 无法分类的问题

### 🤔 "Something went wrong" 或 "发生了什么"

**症状：**
```
技能给出模糊错误信息，无法判断具体问题
```

**诊断步骤：**

```
1️⃣ 看错误日志
   → Help → Logs（如果有）
   → 查看完整错误信息

2️⃣ 逐个排查环境
   → 检查网络（ping 测试）
   → 检查 Relay（扩展是否 ON）
   → 检查登录状态（能否打开网站）
   → 检查浏览器缓存（清空）

3️⃣ 试试最小化重现
   → 关掉所有其他应用
   → 只打开微信和小红书两个标签
   → 确保都已登录
   → 再试一次技能

4️⃣ 还是不行？
   → 重启 EasyClaw 和浏览器
   → 如果重启后还是不行，可能是系统问题
   → 可以反馈（见底部联系方式）
```
