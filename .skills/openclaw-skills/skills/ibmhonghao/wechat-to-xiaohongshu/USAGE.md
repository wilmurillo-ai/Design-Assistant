# 使用指南 - WeChat to Xiaohongshu

## 快速开始

当你在 EasyClaw 中想要把微信公众号的文章发到小红书时，直接说：

```
用 wechat-to-xiaohongshu 技能把今天的新文章搬到小红书
```

或者更具体的场景：

- "把最新的公众号文章导入小红书，加上相关话题"
- "帮我在小红书上发布今天的微信文章"
- "一键同步微信到小红书"

## 前置条件

**必须满足：**

1. ✅ **WeChat 公众号后台已登录**
   - 网址：https://mp.weixin.qq.com
   - 确保能访问"已发表"的文章列表

2. ✅ **小红书创作平台已登录**
   - 网址：https://creator.xiaohongshu.com/
   - 确保有发布笔记的权限

3. ✅ **OpenClaw 浏览器 Relay 已激活**
   - 两个标签页上都要看到 Relay 图标（绿色 ON 标志）
   - 如果没看到，点一下 OpenClaw Toolbar 按钮激活

## 工作流程（简化版）

| 步骤 | 说明 | 预计时间 |
|------|------|--------|
| 1️⃣ | 系统找到微信最新文章 | 5秒 |
| 2️⃣ | 抓取文章链接 | 3秒 |
| 3️⃣ | 导入到小红书（"导入链接" + "一键排版"） | 30秒 |
| 4️⃣ | 自动添加话题标签 | 5秒 |
| 5️⃣ | 发布 | 3秒 |

**总耗时：** 约 50 秒（90% 自动）

## 如果出了问题

### 问题 1：提示"找不到浏览器标签页"

**原因：** OpenClaw Relay 断开连接了

**解决：**
1. 切回浏览器
2. 点一下 OpenClaw 工具栏图标（刷新连接）
3. 等等看绿色 ON 标志出现
4. 重新执行技能

### 问题 2：导入超时（30秒还没完成）

**原因：** 
- 微信文章链接失效或无法访问
- 文章包含小红书不支持的媒体格式（如视频）

**解决：**
- 检查微信文章是否公开发表
- 如果包含视频，可能需要手动提取视频链接另外上传
- 尝试"手动复制粘贴"的方式（备选方案）

### 问题 3：发布按钮灰色（无法点击）

**原因：** 
- 页面内容还没加载完成
- 少了标题或描述
- 图片还在处理中

**解决：**
- 等待 10-15 秒让页面完全加载
- 检查标题、描述、封面是否完整
- 手动点一下"发布"（如果确认内容没问题）

### 问题 4：文章导入后格式乱了

**原因：** 微信排版复杂，小红书 HTML 解析有差异

**解决：**
- 这很正常——小红书会自动调整排版
- 发布后预览一下，如果内容基本完整就可以
- 如果排版太乱，手动编辑一下

## 高级用法

### 批量发布（手动方式）

目前技能是"一篇一篇"发。如果你想连续发多篇：

```
把最新的3篇文章都搬到小红书
```

系统会循环执行，每篇间隔 1-2 分钟。

## 注意事项

- 🔐 **登录状态：** 技能依赖浏览器的登录状态，注销了就无法工作
- 📱 **移动端：** 目前只支持桌面版，手机版小红书无法识别
- ⏰ **发布时间：** 文章发布后会立即显示在小红书，建议避免半夜发（小红书算法）
- 📸 **封面：** 小红书会自动从微信文章内容生成 7 张封面选项，你可以手动选
- 🔄 **自动保存：** 小红书会每隔几分钟自动保存草稿，意外关闭了也不会丢
