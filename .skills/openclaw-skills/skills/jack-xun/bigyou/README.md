# 比优 · bigyou

> **先画标准，再量选项。**

把「凭感觉纠结」变成「有结构地决策」的选择工具。

---

## 核心方法

```
需求调研 → 目标画像 → 备选池筛选 → 量化打分 → 决策
    ↑            ↑              ↑
 维度确定    底线+加分权重    打分表
```

**精髓**：先不比较选项，先定义「好」的标准。

- 🔴 **底线维度**：不满足就直接淘汰，没有例外
- 🟡 **加分维度**：满足越多越好，按权重打分排序

---

## 使用方式

### 直接描述选择困境

```
帮我分析：现在有两个工作机会，A薪资高但方向不喜欢，B方向喜欢但薪资低20%，怎么选？
```

### 任意选择场景

```
选房：市中心房子贵但方便，还是郊区便宜但远？
选工具：Notion vs Obsidian，选哪个？
选offer：这家创业公司值得去吗？
```

---

## 比优 vs 其他决策工具的区别

| 维度 | 比优 | 打分法 | 感觉法 |
|------|------|--------|--------|
| 维度来源 | 先定标准再打分 | 先打分再定权重 | 凭感觉 |
| 底线思维 | ✅ 明确底线否决 | ❌ 没有 | ❌ 没有 |
| 风险推演 | 最坏情况自问 | ❌ 没有 | ❌ 没有 |
| 可复用性 | 画像记录可复用 | 一次性 | 一次性 |
