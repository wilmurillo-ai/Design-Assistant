# 四角度框架详解

## 一、逻辑角度

**五维分析路径**：结构 → 系统 → 规律 → 模型 → 生命周期

### 1. 结构

**核心问题**：组成部分有哪些？各部分之间的关系是什么？边界在哪里？

**分析方法**：
- 穷举法列出所有组成部分
- 识别部分之间的连接关系（串联/并联/网络）
- 明确系统的边界：什么是系统内，什么是外部环境

**检查要点**：
- [ ] 系统边界是否清晰？
- [ ] 各部分是否相互依存？
- [ ] 是否有"枢纽节点"断了会导致系统崩溃？

---

### 2. 系统

**核心问题**：各部分如何构成一个有机整体？子系统有哪些？系统与环境的交互是什么？

**分析方法**：
- 识别子系统及其分工
- 分析子系统之间的协作机制
- 识别系统与外部环境（其他系统）的输入/输出关系
- 识别系统中的"涌现性"——整体是否大于部分之和？

**系统三问**：
1. 这个系统由哪些子系统构成？
2. 子系统之间如何协同？
3. 系统与外部环境如何交互？

**示例**：
- 医院系统：门诊子系统、住院子系统、医技子系统、行政子系统；通过患者流、数据流、资金流串联成一个有机整体
- 环境交互：医院与医保局（支付）、药企（采购）、患者（服务）、政府（监管）持续交互

---

### 3. 规律

**核心问题**：系统的什么行为/规律在反复出现？什么在驱动它持续运转？

**分析方法**：
- 历史复盘：找到3个以上相似案例，提炼共同规律
- 识别周期性规律（时间维度）
- 识别条件反射：特定输入 → 特定输出的稳定映射
- 识别反模式（表面规律下的反直觉真相）

**常见规律类型**：
- 二八法则（关键少数驱动大部分效果）
- 正反馈循环（马太效应、富者愈富）
- 负反馈调节（稳态回归、平衡机制）
- 边际效用递减（投入增加但边际收益下降）

**检查要点**：
- [ ] 过去发生的事能否预测未来？（规律是否稳定）
- [ ] 特定条件下系统是否有固定反应模式？
- [ ] 规律背后的驱动机制是什么？

---

### 4. 模型

**核心问题**：能否抽象为一个可描述的框架/模型？核心变量是什么？变量之间的关系是什么？

**分析方法**：
- 识别系统的核心变量（3-7个为宜）
- 画出变量之间的关系图（因果关系/相关关系）
- 找到主导变量（杠杆效应最大的变量）

**常用模型类型**：
- 供需模型（供方-需方-价格-数量）
- 增长模型（获客-激活-留存-变现-推荐）
- 竞争模型（五力分析：现有竞争者/潜在进入者/替代品/供应商议价/买方议价）
- 生命周期模型（引入-成长-成熟-衰退）

**检查要点**：
- [ ] 能否用简洁的语言描述系统的核心逻辑？
- [ ] 核心变量之间的关系是否清晰？
- [ ] 模型是否能解释历史，也能预测未来？

---

### 5. 生命周期

**核心问题**：系统从哪里来？现在处于什么阶段？未来会如何演变？

**分析方法**：
- 回溯起点：系统是如何诞生的？什么条件催生了它？
- 判断现状阶段：引入期/成长期/成熟期/转型期/衰退期？
- 预判演变：下一阶段的主要矛盾是什么？什么因素会触发系统性变革？

**示例**：
- 医院体系的生命周期：单位医务室 → 街道/乡镇卫生院 → 县级医院 → 三甲医院 → 医疗集团/县域医共体
- 判断：当前处于"分级诊疗深化 + 城市医院集团化整合"阶段，DRG改革是触发转型的关键变量

**检查要点**：
- [ ] 系统当前处于生命周期哪个阶段？
- [ ] 什么因素会推动系统进入下一阶段？
- [ ] 系统会消亡还是会转型？转型的触发条件是什么？

---

## 二、本质角度

**核心句式**：从 XX 角度看，XX 的本质是 XX

### 六大常用视角

| 视角 | 核心问题 | 适用场景 |
|------|---------|---------|
| **信息视角** | 信息如何流动？谁有信息优势？ | 中介行业、平台经济 |
| **激励视角** | 各方的激励机制是什么？利益如何驱动行为？ | 制度改革、政策设计 |
| **博弈视角** | 各方如何策略互动？谁赢谁输？ | 竞争分析、谈判 |
| **权力视角** | 谁有决定权？权力如何分布和传导？ | 组织分析、政治分析 |
| **时间视角** | 长期 vs 短期，节奏和时机？ | 投资决策、战略规划 |
| **成本视角** | 显性/隐性成本结构是什么？成本由谁承担？ | 商业模式分析 |

### 分析技巧

- 选择视角时，优先选择**直觉最不明显的视角**
- 每换一个视角，尝试说出"从X角度看，这个东西的本质是Y"
- 如果说不出新东西，说明视角选得不够深

### 检查要点

- [ ] 换一个角度看，对象是否还是原来的东西？
- [ ] 它的不可替代性到底在哪里？
- [ ] 哪个视角能最有力地解释它的存在价值？

---

## 三、数据角度

**分析路径**：核心KPI → 驱动因子 → 数据特征 → 阈值效应

### 1. 核心KPI

衡量系统表现的核心指标体系：

**三层指标**：
- **数量指标**：规模、总量、速度
- **质量指标**：效率、满意度、质量稳定性
- **财务指标**：收入、成本、利润、资产效率

### 2. 驱动因子

识别最能解释结果变化的变量：

**方法**：
- 拆解法：从结果倒推，拆到不可再拆
- 相关性分析：什么变量和结果强相关
- 敏感性分析：改变什么，对结果影响最大

### 3. 数据特征

- 历史趋势（增长/下降/周期性/随机波动）
- 季节性（是否有固定的时间模式）
- 异常值识别（离群点意味着什么？）

### 4. 阈值效应

- 哪些因子存在"临界点"？
- 超过临界点后，系统行为是否会发生质变？
- 阈值是固定还是可以迁移的？

### 检查要点

- [ ] 核心KPI是否被各方认可（避免局部优化伤害整体）？
- [ ] 驱动因子的变化是否可控？
- [ ] 各因子之间是否存在交互效应（一个因子改变会联动其他因子）？

---

## 四、领域视角

### 社会问题矛盾图谱

**分析方法**：
1. 识别系统涉及的所有参与方
2. 找出每个参与方的核心诉求
3. 绘制矛盾矩阵：哪些诉求之间存在根本冲突？
4. 找到现有解决方案，分析其局限性

**矛盾类型**：
- 效率 vs 公平（做大蛋糕 vs 分蛋糕）
- 短期 vs 长期（当下利益 vs 未来代价）
- 局部 vs 整体（子系统优化 vs 系统整体最优）
- 创新 vs 稳定（颠覆 vs 渐进）

### 经济产业分析

**产业链分析**：
- 上游：谁是供给方？供给集中度如何？
- 中游：这个对象在产业链中的位置？
- 下游：谁是需求方？需求弹性如何？
- 价值链：利润如何在各环节分配？

**竞争格局分析**：
- 现有竞争者：市场集中度、差异化程度
- 潜在进入者：壁垒高低、进入成本
- 替代品：替代程度、替代成本
- 五力模型综合评估

**政策监管分析**：
- 主管部委是谁？核心监管逻辑是什么？
- 政策是支持还是约束？政策周期如何？
- 合规红线在哪里？

---

## 组合分析顺序建议

| 顺序 | 角度 | 作用 |
|------|------|------|
| 1 | 逻辑角度 | 建立基础认知，先"看懂"对象（结构→系统→规律→模型→生命周期层层递进） |
| 2 | 本质角度 | 打破表面，深入"本质"，换个角度看什么最重要 |
| 3 | 数据角度 | 找到关键"驱动因子"，用数据验证假设 |
| 4 | 领域视角 | 跳出对象，放在"大图"里，理解位置和关系 |

> **提示**：四个角度不必面面俱到。快速分析可只选2个角度（逻辑+本质），深度研究可四个角度全部展开。每个角度也不必死板地全部子维度都做，根据分析目的灵活裁剪。
