## 输出字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| policyID | string | 政策唯一标识符 |
| title | string | 政策标题 |
| publishDate | string | 发布时间，格式：YYYY-MM-DD HH:mm:ss |
| issuingOrg | string | 发布机构 |
| policyType | string | 政策类型（宏观政策、行业政策、区域政策等） |
| region | string | 地域范围（全国性、省级、市级等） |
| analysisResults | array | 分析维度结果数组 |
| analysisResults[].dimension | string | 分析维度名称 |
| analysisResults[].content | string | 该维度的分析内容（原文输出） |
| benefitIndustries | array | 受益行业列表 |
| affectedCompanies | array | 受影响企业列表 |

## 分析维度列表

常见的分析维度包括但不限于：

- 政策背景
- 核心内容
- 影响范围
- 实施时间
- 受益行业
- 风险提示
- 政策解读
- 实施效果
- 政策对比
- 国际经验

**注意：** 接口返回的所有分析维度均应保留，不应过滤或删减。

## 政策类型分类

- **宏观政策**：国家层面的总体性、指导性政策
- **行业政策**：针对特定行业的专项政策
- **区域政策**：针对特定区域的发展政策
- **财政政策**：涉及财政收支、税收等政策
- **货币政策**：涉及货币供应、利率等政策
- **产业政策**：涉及产业结构调整、升级的政策
- **科技政策**：涉及科技创新、研发的政策

## 地域范围分类

- **全国性**：国务院及各部委发布的全国性政策
- **省级**：省级政府及部门发布的政策
- **市级**：市级政府及部门发布的政策
- **区域级**：跨省区域（如长三角、粤港澳大湾区等）政策
