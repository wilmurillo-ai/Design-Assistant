## 产品筛选规则

### 数据准确性要求

**所有产品数据必须从聚源 MCP 实时查询，禁止使用示例数据或历史记忆数据！**

| 数据类型 | 查询工具 | 必须字段 |
|----------|----------|----------|
| 基金规模、收益率 | `FinQuery` 或 `FundMultipleFactorFilter` | 规模 (元)、近 1 年收益、近 3 年收益、基金经理 |
| 申购状态 | `FinQuery` | 申购状态、赎回状态 |
| 宏观经济数据 | `MacroIndustryData` | GDP、CPI、PMI、社融等 |
| 市场行情 | `FinQuery` | 指数点位、涨跌幅、成交量 |
| 研报观点 | `FinancialResearchReport` | 机构、作者、发布时间、核心观点 |

### 基金产品准入标准

| 检查项 | 标准 | 处理方式 |
|--------|------|----------|
| 申购状态 | 必须为"开放申购" | 如暂停申购，标注"⚠️ 暂停申购"并提供替代产品 |
| 存续规模 | ≥5000 万元 | 如<5000 万，标注"⚠️ 规模过小（XXX 万）"并提供替代产品 |
| 成立年限 | 建议≥1 年 | 如<1 年，标注"⚠️ 新基金（成立 X 个月）" |
| 数据准确性 | 必须来自聚源 MCP 实时查询 | 如无法查询到数据，不推荐该产品 |

### 固收基金异常收益检测

**检测规则：**
- 纯债基金年化收益 >8% → 触发预警，需检查是否大额赎回导致数据失真
- 固收 + 基金年化收益 >15% → 触发预警，需检查权益仓位是否过高

**处理方式：**
- 如确认大额赎回导致收益失真 → 不推荐该基金，标注"⚠️ 收益失真（大额赎回）"
- 如权益仓位过高导致波动大 → 标注"⚠️ 权益仓位高（XX%），波动可能较大"
- 提供 2-3 只替代产品
