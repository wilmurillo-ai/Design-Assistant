# 绘图工具调用指南

## 图表类型与使用场景

### 1. 叠加柱状图

**用途：** 营业总收入与归属净利润对比

**调用参数：**
```json
{
  "chart_type": "bar",
  "title": "营业总收入与归属净利润趋势",
  "x_axis": ["2024H1", "2024Q3", "2024H2", "2025Q1", "2025H1"],
  "series": [
    {
      "name": "营业总收入",
      "data": [10000, 11000, 12000, 13000, 14000],
      "unit": "百万元"
    },
    {
      "name": "归属净利润",
      "data": [2000, 2200, 2400, 2600, 2800],
      "unit": "百万元"
    }
  ],
  "output_format": "png"
}
```

### 2. 叠加折线图

**用途：** 同比YOY变化趋势

**调用参数：**
```json
{
  "chart_type": "line",
  "title": "营业总收入与归属净利润YOY对比",
  "x_axis": ["2024H1", "2024Q3", "2024H2", "2025Q1", "2025H1"],
  "series": [
    {
      "name": "营收YOY",
      "data": [15.2, 18.5, 20.1, 22.3, 25.6],
      "unit": "%"
    },
    {
      "name": "净利润YOY",
      "data": [12.1, 14.3, 16.8, 19.2, 21.5],
      "unit": "%"
    }
  ],
  "output_format": "png"
}
```

### 3. 饼图

**用途：** 主营业务收入占比

**调用参数：**
```json
{
  "chart_type": "pie",
  "title": "主营业务收入构成",
  "labels": ["产品A", "产品B", "产品C", "其他"],
  "data": [45.5, 30.2, 18.3, 6.0],
  "unit": "%",
  "output_format": "png"
}
```

## 图表样式规范

### 颜色方案

| 数据系列 | 推荐颜色 |
|----------|----------|
| 营业总收入 | #4A90E2 (蓝色) |
| 归属净利润 | #50C878 (绿色) |
| 营收YOY | #FF6B6B (红色) |
| 净利润YOY | #FFD93D (黄色) |
| 产品A | #4A90E2 |
| 产品B | #50C878 |
| 产品C | #FF6B6B |

### 尺寸规范

| 图表类型 | 宽度 | 高度 |
|----------|------|------|
| 柱状图 | 800px | 500px |
| 折线图 | 800px | 500px |
| 饼图 | 600px | 600px |

### 字体规范

- 标题：16px，加粗
- 轴标签：12px
- 图例：11px
- 数据标签：10px

## Markdown嵌入方式

生成的图表以如下方式嵌入报告：

```markdown
![营业总收入与归属净利润趋势](charts/revenue_profit_trend.png)

*图1：公司近5期营业总收入与归属净利润对比（单位：百万元）*
```

## 数据验证

在生成图表前验证：

1. 数据点数量一致（X轴与各系列）
2. 数值范围合理（无异常值）
3. 单位标注清晰
4. 图例与数据系列对应

## 常见问题

### 数据量级差异大

当营收和净利润量级差异大时：
- 使用双Y轴
- 或在图注中说明量级差异

### 负值处理

YOY数据可能为负：
- 折线图Y轴包含0点
- 负值区域用不同颜色标注
