# 金融数据API参考

## MCP工具调用指南

### 条件选股工具

**功能：** 根据用户设定的条件筛选股票

**调用参数：**
```json
{
  "conditions": [
    {
      "field": "pe_ratio",
      "operator": "lt",
      "value": 30
    },
    {
      "field": "roe",
      "operator": "gt",
      "value": 15
    },
    {
      "field": "revenue_growth",
      "operator": "gt",
      "value": 20
    }
  ],
  "market": "A股",
  "limit": 50
}
```

**常用筛选字段：**
- `pe_ratio` - 市盈率
- `pb_ratio` - 市净率
- `roe` - 净资产收益率
- `revenue_growth` - 营收增长率
- `profit_growth` - 净利润增长率
- `market_cap` - 总市值
- `dividend_yield` - 股息率

### 财务数据工具

**获取核心财务指标：**
```json
{
  "stock_code": "000001.SZ",
  "report_type": "quarterly",
  "periods": 5
}
```

**返回数据结构：**
```json
{
  "stock_code": "000001.SZ",
  "stock_name": "平安银行",
  "reports": [
    {
      "period": "2025H1",
      "revenue": 12345.67,
      "gross_profit": 5678.90,
      "net_profit": 3456.78,
      "deducted_net_profit": 3234.56
    }
  ]
}
```

### 估值数据工具

**获取实时估值指标：**
```json
{
  "stock_code": "000001.SZ"
}
```

**返回数据：**
- 最新价
- 总市值
- PE(TTM)
- PB(LF)
- PEG
- 股息率

### 机构预期工具

**获取一致预期：**
```json
{
  "stock_code": "000001.SZ",
  "forecast_periods": 4
}
```

### 舆情数据工具

**获取公司舆情：**
```json
{
  "stock_code": "000001.SZ",
  "sentiment": "all",
  "days": 30
}
```

## 数据单位规范

| 数据类型 | 单位 | 说明 |
|----------|------|------|
| 营收/利润 | 百万元 | 财务报表原始数据 |
| 比率指标 | % | 毛利率、净利率、ROE等 |
| 股价 | 元 | 人民币元 |
| 市值 | 亿元 | 总市值 |
| 持股数量 | 股 | 股东持股数 |
| 持股比例 | % | 占总股本比例 |

## 报告期命名规范

| 代码 | 含义 |
|------|------|
| 2025H1 | 2025年半年报 |
| 2025Q1 | 2025年一季报 |
| 2024H2 | 2024年年报 |
| 2024Q3 | 2024年三季报 |
| 2024H1 | 2024年半年报 |

## 数据质量检查

在生成报告前，验证数据完整性：

1. **财务数据：** 至少5个报告期的完整数据
2. **估值数据：** 实时数据，无缺失
3. **预期数据：** 至少未来2个季度的预测
4. **舆情数据：** 近30天的重要信息

如发现数据缺失，在报告中标注"数据暂缺"并继续生成其他部分。
