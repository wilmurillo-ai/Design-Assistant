# 【通用智能选股&诊股】

## 技能简介

基于恒生聚源（GilData）MCP 金融数据服务平台，提供从条件选股到深度个股诊断的全流程分析能力。

**核心能力：** 条件选股、财务分析、估值分析、技术分析、舆情监控、报告生成。

---

## 功能范围

| 场景 | 说明 |
|------|------|
| 条件选股 | 根据财务指标、技术指标、行业等条件筛选股票 |
| 个股诊断 | 对单只股票进行全面的基本面分析 |
| 多股对比 | 对多只股票进行横向对比分析 |
| 行业分析 | 分析特定行业的发展趋势、竞争格局 |
| 研报生成 | 输出标准格式的公司研究报告 |
| 财务诊断 | 分析公司财务状况、盈利能力、成长能力 |
| 估值分析 | 评估股票当前估值水平及合理区间 |

---

## 查询建议

**查询要素：**
1. **股票代码/名称**（必填）：如"贵州茅台"、"600519"、"00700.HK"
2. **分析类型**（可选）：如"财务分析"、"估值分析"、"完整报告"
3. **时间范围**（可选）：如"最新财报"、"2024 年报"
4. **对比标的**（可选）：如"与五粮液对比"

**查询写法：**
```
分析 [股票名称/代码]
筛选 [条件] 的股票并分析
对比 [股票 A] 和 [股票 B]
[股票名称] 的 [财务/估值/技术] 分析
```

---

## 查询示例

```
分析贵州茅台
筛选市盈率小于 20 且 ROE 大于 15% 的股票
对比贵州茅台和五粮液的财务状况
腾讯控股的估值分析
白酒行业发展趋势分析
```

## 核心工作流程

**工具调用尽量并发执行以提速。**

### 步骤 1：条件选股（如适用）

```bash
mcporter call jy-financedata-api.StockMultipleFactorFilter query="换手率大于 10% 的股票"
```

- 查询格式：自然语言描述筛选条件
- 如果结果超过 10 只，选择其中 5 只进行分析

### 步骤 2：数据采集

对每只股票采集以下数据（可并发执行）：

| 数据类别 | 工具名称 |
|----------|----------|
| 公司基本信息 | `CompanyBasicInfo` |
| 股权结构 | `Top10ShareHolders` |
| 财务指标 | `FinancialStatements` |
| 财务比率 | `FinancialRatioComparison` |
| 业务构成 | `MainOperIncData` |
| 估值指标 | `StockValueAnalysis` |
| 技术指标 | `StockQuoteTechIndex` |
| 行情数据 | `AShareLiveQuote` |
| 机构预期 | `ConsensusExpectation` |
| 新闻舆情 | `StockNewslist` |
| 行业数据 | `IndustryAnalysisViewpoints` |

### 步骤 3：图表绘制

1. 营业总收入和归属净利润叠加柱状图
2. 营业总收入和归属净利润同比 YOY 叠加折线图
3. 主营业务产品收入占比饼图

### 步骤 4：同行业对比

选取 3-5 家同行业公司，调用 `FinancialDataComparison` 进行对比分析。

### 步骤 5：报告生成

按照标准模板格式（见下方报告模板）生成完整的公司研究报告。

**申请邮箱：** datamap@gildata.com

**邮件标题：** 数据地图 KEY 申请-XX 公司 - 申请人姓名

**正文模板：**
```
• 姓名：
• 手机号：
• 公司/单位全称：
• 所属部门：
• 岗位：
• MCP_KEY 申请用途：

• 是否需要 Skill 安装包：（是，邮件提供/否，自行下载）
• 其他补充说明（可选）
```

申请通过后，恒生聚源将默认发送【工具版和接口版】KEY。

**Skill 下载：** https://clawhub.ai/

### MCP 服务配置命令

```bash
# 配置 jy-financedata-tool 服务
mcporter config add jy-financedata-tool --url "https://api.gildata.com/mcp-servers/aidata-assistant-srv-tool?token=你的 JY_API_KEY"

# 配置 jy-financedata-api 服务
mcporter config add jy-financedata-api --url "https://api.gildata.com/mcp-servers/aidata-assistant-srv-api?token=你的 JY_API_KEY"

# 验证配置
mcporter list
```

### OpenClaw 配置

**配置文件路径：**
- mcporter: `~/.config/mcporter.json` (Linux/MacOS) 或 `C:\Users\你的用户名\config\mcporter.json` (Windows)
- OpenClaw: `~/.openclaw/openclaw.json`

**编辑 openclaw.json：**
```json
{
  "skills": {
    "entries": {
      "mcporter": {
        "enabled": true,
        "env": {
          "MCPORTER_CONFIG": "/root/config/mcporter.json"
        }
      }
    }
  }
}
```

**重启 OpenClaw：**
```bash
openclaw gateway restart
```
