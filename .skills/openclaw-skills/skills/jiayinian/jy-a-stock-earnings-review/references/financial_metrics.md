# 财务指标定义与计算公式

## 盈利能力指标

### 毛利率 (Gross Margin)
```
毛利率 = (营业收入 - 营业成本) / 营业收入 × 100%
```
**含义：** 反映产品或服务的初始盈利能力

### 净利率 (Net Margin)
```
净利率 = 归母净利润 / 营业收入 × 100%
```
**含义：** 反映公司最终盈利能力

### 期间费用率
```
期间费用率 = (销售费用 + 管理费用 + 研发费用 + 财务费用) / 营业收入 × 100%
```

**分项费用率：**
- 销售费用率 = 销售费用 / 营业收入
- 管理费用率 = 管理费用 / 营业收入
- 研发费用率 = 研发费用 / 营业收入
- 财务费用率 = 财务费用 / 营业收入

## 成长能力指标

### 营收同比增速
```
营收同比增速 = (本期营业收入 - 上年同期营业收入) / 上年同期营业收入 × 100%
```

### 净利润同比增速
```
净利润同比增速 = (本期归母净利润 - 上年同期归母净利润) / 上年同期归母净利润 × 100%
```

## 营运能力指标

### 应收账款周转率
```
应收账款周转率 = 营业收入 / 平均应收账款余额
```

### 存货周转率
```
存货周转率 = 营业成本 / 平均存货余额
```

## 偿债能力指标

### 资产负债率
```
资产负债率 = 负债总额 / 资产总额 × 100%
```

### 流动比率
```
流动比率 = 流动资产 / 流动负债
```

### 速动比率
```
速动比率 = (流动资产 - 存货) / 流动负债
```

## 现金流指标

### 经营性现金流净额
**含义：** 经营活动产生的现金流量净额，反映主业造血能力

### 自由现金流 (FCF)
```
自由现金流 = 经营性现金流净额 - 资本性支出
```

## 估值指标

### 市盈率 (PE)
```
PE = 总市值 / 归母净利润
或 PE = 股价 / 每股收益 (EPS)
```

### 市净率 (PB)
```
PB = 总市值 / 净资产
```

### PEG
```
PEG = PE / 净利润增速
```

## 报告期说明

| 报告期类型 | 披露时间 | 数据期间 |
|-----------|---------|---------|
| 年报 | 次年 1-4 月 | 1 月 1 日 -12 月 31 日 |
| 中报 | 当年 7-8 月 | 1 月 1 日 -6 月 30 日 |
| 一季报 | 当年 4 月 | 1 月 1 日 -3 月 31 日 |
| 三季报 | 当年 10 月 | 1 月 1 日 -9 月 30 日 |

## 注意事项

1. **同比 vs 环比**
   - 同比：与上年同期比较，消除季节性影响
   - 环比：与上一期比较，反映短期趋势

2. **单季度数据计算**
   - Q1 单季度 = 一季报数据
   - Q2 单季度 = 中报数据 - 一季报数据
   - Q3 单季度 = 三季报数据 - 中报数据
   - Q4 单季度 = 年报数据 - 三季报数据

3. **TTM 数据**
   - TTM (Trailing Twelve Months) = 最近 4 个季度滚动加总
   - 用于计算动态估值指标
