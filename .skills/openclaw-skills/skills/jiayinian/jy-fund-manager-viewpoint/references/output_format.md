## 字段说明

| 字段 | 说明 |
|------|------|
| 行业 | 分析的目标行业名称 |
| 季度 | 对比的两个季度 |
| 时间 | 报告生成时间 |
| 数据源 | 数据来源说明 |
| 基金数 | 参与分析的基金数量 |
| 核心结论 | 季度间核心变化总结（200 字内） |
| 季度对比表格 | 情绪、主题、共识、分歧、观点详情对比 |
| 报告来源 | 所有引用报告的列表 |

## 共识/分歧格式

**共识格式：**
- 描述 _(支持基金 1、支持基金 2)_

**分歧格式：**
- **主题**: 观点 A vs 观点 B
