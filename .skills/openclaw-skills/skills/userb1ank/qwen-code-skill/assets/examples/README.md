# Qwen Code 示例代码

本目录包含 Qwen Code Skill 的使用示例。

## 示例列表

- `basic-task.example.sh` - 基本任务执行示例
- `code-review.example.sh` - 代码审查示例
- `ci-cd.example.yml` - CI/CD 集成示例
- `headless-mode.example.js` - Headless 模式示例

## 使用方法

```bash
# 查看示例
cat assets/examples/basic-task.example.sh

# 运行示例（需要 Qwen Code CLI）
bash assets/examples/basic-task.example.sh
```

## 注意事项

1. 示例需要预先安装 Qwen Code CLI
2. 部分示例需要配置 API Key 或 OAuth 认证
3. 生产环境使用前请仔细阅读安全说明
