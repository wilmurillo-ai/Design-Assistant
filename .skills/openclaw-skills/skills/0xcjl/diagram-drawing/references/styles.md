# Style Reference Guide

## Style 1 — Flat Icon (Default)
- **Background**: `#ffffff`
- **Primary text**: `#1f2937`
- **Node fill**: `#f3f4f6`
- **Node stroke**: `#374151`
- **Accent 1**: `#3b82f6` (blue)
- **Accent 2**: `#10b981` (green)
- **Accent 3**: `#f59e0b` (amber)
- **Accent 4**: `#ef4444` (red)
- **Font**: Helvetica, system-ui
- **Best for**: Blogs, slides, docs, presentations

## Style 2 — Dark Terminal
- **Background**: `#0f0f1a`
- **Primary text**: `#e5e7eb`
- **Node fill**: `#1f2937`
- **Node stroke**: `#6b7280`
- **Accent 1**: `#22d3ee` (cyan neon)
- **Accent 2**: `#a78bfa` (purple neon)
- **Accent 3**: `#34d399` (green neon)
- **Accent 4**: `#fb923c` (orange neon)
- **Font**: SF Mono, Fira Code, monospace
- **Best for**: GitHub README, dev articles

## Style 3 — Blueprint
- **Background**: `#0a1628`
- **Primary text**: `#94a3b8`
- **Node fill**: `#0f172a`
- **Node stroke**: `#1e40af`
- **Accent 1**: `#38bdf8` (cyan)
- **Accent 2**: `#818cf8` (indigo)
- **Grid lines**: `#1e3a5f` dashed
- **Font**: Courier New, monospace
- **Best for**: Architecture docs, engineering specs

## Style 4 — Notion Clean
- **Background**: `#ffffff`
- **Primary text**: `#37352f`
- **Node fill**: `#f7f6f3`
- **Node stroke**: `#e5e5e5`
- **Accent 1**: `#000000` (black)
- **Accent 2**: `#6b6b6b` (gray)
- **Arrow (all types)**: `#000000` single color
- **Font**: system-ui
- **Best for**: Notion, Confluence, wikis

## Style 5 — Glassmorphism
- **Background**: `#0d1117` to `#161b22` gradient (top to bottom)
- **Primary text**: `#f0f6fc`
- **Node fill**: `rgba(48,54,61,0.7)` with backdrop blur effect via opacity
- **Node stroke**: `rgba(240,246,252,0.1)`
- **Card glow**: `rgba(88,166,255,0.15)` border
- **Accent 1**: `#58a6ff` (blue)
- **Accent 2**: `#f778ba` (pink)
- **Accent 3**: `#7ee787` (green)
- **Font**: Inter, system-ui
- **Best for**: Product sites, keynotes, AI demos

## Style 6 — Claude Official
- **Background**: `#f8f6f3` (warm cream)
- **Primary text**: `#1a1a1a`
- **Node fill**: `#ffffff`
- **Node stroke**: `#e5e5e5`
- **Accent 1**: `#c96018` (Anthropic orange)
- **Accent 2**: `#82589F` (Anthropic purple)
- **Accent 3**: `#2a7f64` (teal)
- **Accent 4**: `#c94a1a` (warm red)
- **Font**: system-ui, -apple-system, sans-serif
- **Best for**: Anthropic projects, warm professional docs

## Style 7 — OpenAI Official
- **Background**: `#ffffff` (pure white)
- **Primary text**: `#1a1a1a`
- **Node fill**: `#fafafa`
- **Node stroke**: `#e5e5e5`
- **Accent 1**: `#10a37f` (OpenAI green)
- **Accent 2**: `#4f46e5` (indigo)
- **Accent 3**: `#6b7280` (gray)
- **Font**: system-ui
- **Best for**: OpenAI projects, clean modern docs
