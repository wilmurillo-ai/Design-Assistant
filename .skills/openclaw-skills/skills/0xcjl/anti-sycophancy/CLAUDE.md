# anti-sycophancy Skill Repository

This is a Claude Code / OpenClaw skill for deploying a three-layer sycophancy defense system.

## Quick Reference

- **SKILL.md** — The main skill file. Install with `/anti-sycophancy install`.
- **hooks/** — Layer 1 hook scripts (shell + Python). Copied to `~/.claude/hooks/` on install.
- **docs/INSTALL.md** — Installation guide
- **docs/DESIGN.md** — Design rationale
- **README.md** — Overview (English)
- **README.zh-CN.md** — Overview (中文)
