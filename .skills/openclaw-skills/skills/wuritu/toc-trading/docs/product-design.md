# TOC 模拟交易系统 - 产品设计

> **别名**：TOC 模拟交易 / 小悟操盘手 🎯

---

## 1. 产品定位

**目标用户**：苍苍子森（老板）—— 验证选股思路、练习交易策略

**核心价值**：
- 不真金白银试错，在模拟中积累经验
- 实时数据 + 历史回测，验证交易逻辑
- 每日复盘，逐步形成自己的交易体系

---

## 2. 功能模块

### 2.1 股票池

**功能**：管理你推荐的股票，展示实时行情

**数据结构**：
```json
{
  "代码": "600519.SH",
  "名称": "贵州茅台",
  "当前价": 1680.50,
  "推荐理由": "龙头稳健，业绩确定性强",
  "行业": "白酒",
  "添加时间": "2026-03-29",
  "备注": "目标价1800"
}
```

**操作命令**：
| 命令 | 说明 |
|------|------|
| 加一只 XXX | 添加股票到池子 |
| 去掉 XXX | 从池子移除 |
| 更新 XXX | 调整推荐理由/备注 |
| 搜索 XXX | 查找股票 |
| 按行业分组 | 按行业分类展示 |
| 按涨幅排序 | 今日涨跌幅排序 |

**展示形式**：
```
📈 今日股票池（白酒）
┌──────────┬───────┬───────┬─────────┐
│ 代码     │ 名称   │ 价格  │ 涨幅    │
├──────────┼───────┼───────┼─────────┤
│ 600519.SH│ 茅台  │ 1680  │ -0.32%  │
│ 000858.SZ│ 五粮液│ 145   │ +0.85%  │
└──────────┴───────┴───────┴─────────┘
```

### 2.2 持仓模拟

**功能**：记录模拟买入，实时计算盈亏

**数据结构**：
```json
{
  "股票代码": "600036.SH",
  "股票名称": "招商银行",
  "买入日期": "2026-03-20",
  "买入价": 15.60,
  "数量": 10000,
  "备注": "测试仓"
}
```

**操作命令**：
| 命令 | 说明 |
|------|------|
| 买 100 手 @ 15.6 元 招商银行 | 记录买入 |
| 卖 50 手 招商银行 | 记录卖出 |
| 持仓 | 查看当前持仓 |
| 清仓 招商银行 | 全部卖出 |
| 历史交易 | 查看成交记录 |

**展示形式**：
```
📊 当前持仓
┌─────────┬────────┬────────┬─────────┬──────────┬──────────┬────────┬────────┐
│ 股票    │ 买入价 │ 当前价 │ 数量    │ 持仓成本 │ 当前市值 │ 盈亏   │ 盈亏率 │
├─────────┼────────┼────────┼─────────┼──────────┼──────────┼────────┼────────┤
│ 招商银行│ 15.60  │ 16.23  │ 100手   │ 156,000  │ 162,300  │ +6,300 │ +4.04% │
│ 宁德时代│ 185.60 │ 192.30 │ 50手    │ 928,000  │ 961,500  │+33,500 │ +3.61% │
└─────────┴────────┴────────┴─────────┴──────────┴──────────┴────────┴────────┘

💰 总盈亏：+39,800 元 (+3.85%)
```

### 2.3 演练模式（假设交易）

**功能**：不实际下单，假设买入计算收益

**操作命令**：
| 命令 | 说明 |
|------|------|
| 如果昨天开盘买入招商银行 | 假设昨日开盘买入 |
| 3月20日买入宁德时代赚多少 | 指定日期演练 |
| 如果上周五收盘买现在赚多少 | 周线级别演练 |

**展示形式**：
```
🔍 演练结果
股票：招商银行（600036.SH）
买入假设：2026-03-28 开盘价 15.60
当前价格：16.23（实时）
演练收益：+6,300 元 (+4.04%)
持仓天数：1 天
```

### 2.4 监控推送

**功能**：定时推送股票动态，异常提醒

**推送时间**：
| 时间点 | 内容 |
|--------|------|
| 09:30 早盘 | 开盘前自选股竞价情况 |
| 13:00 午盘 | 上午收盘总结 |
| 15:30 收盘 | 今日持仓/股票池变动 |

**异动规则**：
| 异动类型 | 阈值 | 推送方式 |
|----------|------|----------|
| 单日涨跌幅 | > ±5% | 即时推送 |
| 成交量暴增 | 较昨日 > 100% | 即时推送 |
| 涨停/跌停 | 触及涨跌停 | 即时推送 |
| 破新低/新高 | 30日新低/新高 | 收盘推送 |

**推送格式**：
```
📢 异动提醒【招商银行】
⚠️ 单日跌幅 -5.23%
当前价格：15.80
原因：可能受板块拖累
建议：关注支撑位 15.50
```

### 2.5 🏆 AI股神挑战

**功能**：模拟实盘比赛，验证交易能力

**规则**：
| 项目 | 说明 |
|------|------|
| 初始资金 | 50,000 元 |
| 模拟周期 | 1 个月（自然月） |
| 决策方式 | 你自主决策，不问我 |
| 交易频率 | 每天最多 3 笔 |
| 止损线 | 单笔亏损 > 7% 强制止损 |

**展示功能**：
| 功能 | 说明 |
|------|------|
| 收益率排行 | 每日更新排名 |
| 收益曲线 | 资产变化折线图 |
| 交易统计 | 胜率、平均盈利、单笔最大盈利/亏损 |
| 连胜记录 | 最长连续盈利天数 |

**挑战命令**：
| 命令 | 说明 |
|------|------|
| 开启挑战 | 开始新一轮挑战 |
| 我的排名 | 当前排名和收益 |
| 挑战排行榜 | 查看所有参与者 |
| 挑战统计 | 查看个人交易统计 |

## 3. 数据来源

| 数据类型 | 接口 | 备注 |
|----------|------|------|
| 实时行情 | `pro_bar` | 股票当前价 |

## 4. 技术实现

### 4.1 存储方案

**JSON 文件存储**（简单场景）：
```
~/.openclaw/workspace/toc-trading/
├── config.json          # 系统配置
├── stock_pool.json      # 股票池
├── positions.json       # 持仓记录
├── trades.json          # 历史交易
├── challenge.json       # 挑战数据
└── cache/               # 缓存数据
```

### 4.2 命令解析

使用自然语言处理思路：
1. 提取关键词（买/卖/加/去掉/如果...）
2. 解析标的（股票代码/名称）
3. 提取参数（价格/数量/日期）
4. 执行对应操作

### 4.3 推送实现

- 使用 OpenClaw 心跳任务实现定时推送
- 飞书消息渠道推送
