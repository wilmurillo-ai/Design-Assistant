# 个人身份规划评估框架

## 规划前的自我评估

在做任何身份规划决策前，先回答以下核心问题。

---

## 一、明确核心目标（排序）

以下目标通常互斥，**需要排序而非全选**：

| 目标编号 | 目标描述 | 说明 |
|----------|----------|------|
| A | 提升护照免签能力 | 让出行更自由 |
| B | 获取欧盟居留权/公民身份 | 享受欧盟教育、医疗、生活圈 |
| C | 税务优化 | 降低全球税负，合法避税 |
| D | 资产安全/分散风险 | 应对地缘政治不确定性 |
| E | 子女教育 | 为下一代获取更好的教育通道 |
| F | 实际居住/生活 | 真正去另一个国家生活 |
| G | 退休养老 | 低成本高质量生活 |
| H | 职业发展 | 获取国际工作经验和人脉 |
| I | 商业拓展 | 进入新市场、降低商业壁垒 |

**排序方法**：将上述目标按 1-9 排序，1最重要。

## 二、个人画像评估

### 2.1 财务维度

```
可投资预算区间：
├─ < $100K      → 侧重 DIY 签证、数字游民、血统入籍
├─ $100K-300K   → 加勒比CBI、部分欧洲居留
├─ $300K-700K   → 欧洲黄金签证、土耳其房产CBI
├─ $700K-$2M    → 高端欧洲项目、阿联酋
└─ > $2M        → 几乎所有项目可选，关注质量和服务
```

### 2.2 时间维度

```
紧迫程度：
├─ 需要 6 个月内拿到    → CBI项目（加勒比）、瓦努阿图、部分DIY签证
├─ 6-18 个月           → 黄金签证、投资居留、数字游民
├─ 2-5 年              → 工作签证→PR、留学→PR
└─ 5 年以上            → 自然化归化、长期积累路径
```

### 2.3 职业维度

| 职业类型 | 推荐路径 | 说明 |
|----------|---------|------|
| 博士/研究人员 | 技术移民、学术签证、人才通道 | 多国有绿色通道 |
| IT/工程师 | 高技能签证、数字游民 | 全球需求量大 |
| 企业家/创业者 | 投资签证、创业签证 | 部分国家有创业孵化计划 |
| 自由职业/远程工作 | 数字游民签证、被动收入居留 | 新兴热门路径 |
| 退休人员 | 退休签证、MM2H | 需证明稳定被动收入 |
| 学生 | 留学→工作→PR | 最传统的路径 |

### 2.4 家庭维度

- 家庭人数（配偶、子女、父母）
- 子女年龄（是否即将超龄）
- 是否需要同时解决多个家庭成员
- 配偶是否有独立能力申请

### 2.5 语言与适应维度

- 英语水平（决定英语国家可行性）
- 是否愿意学习新语言
- 对不同文化的适应能力
- 是否愿意长期居住在目标国

## 四、成本全貌模板

任何项目评估都应包含以下成本项：

```
一、前期费用（一次性）
  ├─ 政府申请费
  ├─ 律师/顾问费
  ├─ 中介服务费（如果使用）
  ├─ 材料公证/翻译费
  ├─ 体检费
  └─ 背景调查费（CBI项目）

二、投资本金
  ├─ 投资/购买/捐赠金额
  └─ 投资持有成本（如房产管理费、基金管理费）

三、年度维护费用
  ├─ 居留续签费
  ├─ 年度税务申报费
  ├─ 保险费（如需要）
  └─ 房产持有成本（物业费、地税等）

四、隐性成本
  ├─ 飞行/差旅费
  ├─ 陪审/语言学习费
  ├─ 子女教育差价
  └─ 机会成本（资金的潜在回报）

五、退出成本
  ├─ 投资退出时的税费
  └─ 身份维持失败后的沉没成本
```

## 五、常见错误

1. **只看投资门槛，忽略持有成本**：希腊€250K房产看似便宜，但持有5年可能产生€50K额外费用
2. **忽略居住要求（移民监）**：买了居留身份却无法满足居住要求，身份到期无法续签
3. **混淆居留和公民**：拥有黄金签证不等于拥有护照，免签待遇不变
4. **忽视税务影响**：获得身份可能改变全球税务义务（尤其是 CRS 报税）
5. **只考虑自己**：配偶、子女、父母是否都能涵盖
6. **忽视退出策略**：5年后房产能否卖？能卖多少钱？居留是否可转让？
7. **盲目追求多国身份**：每个身份都有维护成本和税务义务，多不一定好
8. **被中介牵着走**：没有自己的研究基础就去见中介，容易被"专业话术"忽悠

## 六、信息搜集工作流

当需要深入研究一个项目时：

```
1. 查政府官网 → 获取官方项目要求、费用、处理时间
2. 查独立论坛 → 获取真实用户反馈（成功案例和失败案例）
3. 查新闻/行业网站 → 了解政策变动趋势
4. 对比2-3家中介报价 → 交叉验证费用结构
5. 咨询独立律师 → 获取无利益冲突的法律意见
6. 联系已成功申请人 → 最有价值的一手信息
7. 实地考察（如果预算允许）→ 亲眼看到的生活环境
```
