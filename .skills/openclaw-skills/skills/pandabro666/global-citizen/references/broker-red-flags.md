# 中介套路与风险防范指南

## 核心原则

> 所有身份规划信息都必须**交叉验证**，绝不盲信单一来源。政府官网是唯一权威来源。

---

## 一、常见中介话术套路

### 🔴 "包成功" / "100%获批"
- **现实**：没有任何政府项目是100%获批的
- **真相**：正常项目都有审核流程，拒签是正常现象
- **应对**：要求中介提供历史获批率数据，而非口头保证

### 🔴 "政策马上变动，现在不办就来不及了"
- **现实**：政策确实在变化，但通常是渐进式的
- **真相**：制造紧迫感是经典的焦虑营销，逼迫用户冲动签约
- **应对**：核实政府官网是否有公告，不要被"倒计时"驱使决策

### 🔴 "价格超低，别人卖50万我们只要30万"
- **现实**：低价背后可能隐藏各种附加费用
- **真相**：常见的手法是"低价签约" + "层层加码"（翻译费、公证费、政府费、律师费等）
- **应对**：要求出具**完整费用清单**，明确哪些包含、哪些不包含

### 🔴 "这个项目可以直接拿护照，不需要住"
- **部分正确**：CBI项目确实不需要居住，但黄金签证项目通常有转公民的居住要求
- **真相**：混淆居留权和公民身份概念，或者隐瞒转公民的难度
- **应对**：明确区分"居留权"和"公民身份"，问清楚每一步的具体要求

### 🔴 "我们的律师/顾问有政府关系"
- **现实**：正规移民国家（加拿大、澳洲、葡萄牙）政府流程透明，关系不等于加速
- **真相**：利用信息不对称制造信任感
- **应对**：核实律师是否持有该国政府认可的执业执照

### 🔴 "费用全付后才开始办理"
- **现实**：一次性全款支付风险极高
- **真相**：如果中介跑路或项目失败，追回款项极其困难
- **应对**：坚持分期付款，按节点支付（签约、材料递交、获批等）

## 二、财务风险识别

### 费用结构拆解（以黄金签证为例）

```
实际总成本 = 政府申请费 + 投资金额 + 法律费 + 中介服务费 + 附加费
                 ↓              ↓           ↓            ↓            ↓
              透明可控      常见隐形增价   需核实资质   最大水分区    变数最多
```

**常见隐形费用**：
- 材料翻译费（按页收费，累积惊人）
- 无犯罪记录公证费
- 体检费
- 保险费
- 背景调查费（CBI项目常见）
- 房产管理费（黄金签证房产项目）
- 维持身份的年费
- 家属附加费（很多人漏算）

### 💡 红线规则
- 如果总费用无法提供**书面报价单**，直接pass
- 如果不愿签订**退款条款**（拒签退款），需要高度警惕
- 如果投资金额比政府官网公示的**低很多**，基本是骗局

- **伪造学历/工作证明**：后果严重

## 四、中介资质核查清单

### 对任何中介，在签约前完成以下核查：

1. **资质认证**
   - [ ] 是否持有目标国家的执业执照？（美国USCIS认证律师、加拿大ICCRC持牌RCIC、澳洲MARA注册等）
   - [ ] 是否为该国移民局官网可查询的认证代理？
   - [ ] 是否是行业协会成员（如美国AILA、加拿大CAPIC）？

2. **透明度**
   - [ ] 是否提供完整费用清单（含所有附加费）？
   - [ ] 是否使用官方申请表格（而非自制模板）？
   - [ ] 是否允许客户直接与律师沟通？

3. **合同保障**
   - [ ] 合同是否包含拒签退款条款？
   - [ ] 费用是否分期支付，按节点绑定？
   - [ ] 是否有明确的办理时限承诺？

4. **口碑验证**
   - [ ] Google Reviews / Trustpilot 评分如何？
   - [ ] 是否有成功案例可以联系（付费可，但中介应能提供部分参考）？
   - [ ] 在移民论坛（如immigration subreddit）上口碑如何？

## 六、自保核心原则

1. **DIY可行性评估**：很多项目（D7、数字游民签证）可以完全DIY，省下中介费
2. **政府直申优先**：能直接向政府递交的，不要通过中间人
3. **资金安全第一**：投资资金必须进入自己的账户或政府监管账户，不要打给中介个人
4. **保留所有书面记录**：合同、收据、通信记录至少保存10年
5. **多个信息源交叉验证**：如果只有中介提供的信息，不做任何决定
6. **不要被"名额有限"催促**：即使政策真的在变，也需要冷静评估
