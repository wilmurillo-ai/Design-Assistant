# 全球身份规划.skill

> **身份不是终点，是工具。**
> **世界公民不是梦想，是一种选择。**

不是移民中介，不是信息搬运工——是一个可运行的**全球身份规划方法论引擎**。

以大陆华人视角出发，把海外身份规划这件事，从一个充满焦虑和信息的黑洞，蒸馏成一个结构化的决策框架。每条路径都从**门槛、经济、时间**三个维度拆解，所有数据强制实时验证，永远不信任静态信息。

> 不管你是想给孩子多一个选择、给事业多一个通道、还是单纯想给自己一个"Plan B"——
> 这个 skill 帮你把模糊的念头变成可执行的行动方案。

---

## 它能做什么

跟你的 AI 助手说以下任意一句话，它会自动加载这个 skill：

- "帮我分析25岁博士生的海外身份规划路径"
- "比较葡萄牙黄金签证和格林纳达CBI"
- "这个中介的报价有问题吗"
- "香港优才计划现在什么门槛"
- "我想拿一个可以免签申根的身份"
- "我是做AI的，哪些国家对技术人才最友好"

然后你会得到一份**结构化的分析报告**：

```
用户画像 → 候选路径（2-4条）→ 实时数据验证 → 三维度对比矩阵 → 风险提醒
```

不是给答案，是给你一个**自己做决策的框架**。

## 工作原理

```
                        ┌──────────────────────┐
                        │   1. 理解用户画像      │
                        │   年龄 / 预算 / 诉求   │
                        └──────────┬───────────┘
                                   │
                        ┌──────────▼───────────┐
                        │   2. 生成候选路径      │
                        │   从结构知识中匹配     │
                        └──────────┬───────────┘
                                   │
                        ┌──────────▼───────────┐
                        │  3. 实时查询（强制）   │  ◄── 关键差异
                        │   Web Search 政府官网  │
                        └──────────┬───────────┘
                                   │
              ┌────────────────────┼────────────────────┐
              │                    │                    │
    ┌─────────▼─────────┐ ┌───────▼───────┐ ┌─────────▼─────────┐
    │  维度1: 门槛要求    │ │ 维度2: 经济成本 │ │  维度3: 时间成本   │
    │  语言 / 学历 / 技能│ │ 投资 / 政府费  │ │  处理时间 / 居留要求 │
    │  资金 / 背景审查   │ │ 维护 / 隐性成本 │ │  入籍路径 / 灵活性  │
    └─────────┬─────────┘ └───────┬───────┘ └─────────┬─────────┘
              │                    │                    │
              └────────────────────┼────────────────────┘
                                   │
                        ┌──────────▼───────────┐
                        │   4. 对比矩阵 + 推荐   │
                        │   让你自己做决策       │
                        └──────────┬───────────┘
                                   │
                        ┌──────────▼───────────┐
                        │   5. 风险提醒         │
                        │   中介套路 / 政策波动  │
                        │   税务义务 / CRS/FATCA │
                        └──────────────────────┘
```

**为什么不用静态数据？** 因为各国政策每几个月就变。这个 skill 里的 references 只存"结构知识"（分类框架、分析模板、决策维度），具体的价格、分数、门槛全部实时搜索政府官网获取。**你拿到的那一刻，数据就是最新的。**

## 安装

### OpenClaw / ClawHub

```bash
# 方式1: ClawHub 一键安装
npx clawhub install global-mobility

# 方式2: 手动安装
git clone https://github.com/PandaBro666/global-mobility.git
cp -r global-mobility ~/.openclaw/skills/
```

### Cursor / Claude Code / Copilot

这些工具使用相同的 `SKILL.md` + `references/` 结构，把 `global-mobility/` 放到对应目录即可：

```
~/.cursor/skills/global-mobility/     # Cursor
~/.claude/skills/global-mobility/     # Claude Code
.copilot/skills/global-mobility/      # GitHub Copilot
```

## 蒸馏了什么

### 3套结构框架

| 框架 | 文件 | 解决什么问题 |
|------|------|-------------|
| 身份类型分类学 | `references/identity-types.md` | 签证 / 居留 / PR / 公民，各自的权利边界和转换路径 |
| 三维度分析模板 | `references/analysis-template.md` | 门槛 / 经济 / 时间，每条路径都必须覆盖，避免信息碎片化 |
| 自评规划框架 | `references/planning-framework.md` | 预算 / 时间线 / 家庭 / 语言，先理清自己再选路径 |

### 2套实操工具

| 工具 | 文件 | 解决什么问题 |
|------|------|-------------|
| 目的地结构知识 | `references/destinations.md` | 覆盖欧洲 / 中东 / 东南亚 / 加勒比 / 港澳台 / 美洲 / 大洋洲的路径特征 |
| 中介套路识别 | `references/broker-red-flags.md` | 6种经典话术 + 费用拆解 + 资质核查清单 + 自主信息源 |

### 1条核心原则

> **不替你做决策。** 给你框架、给你数据来源、给你风险提醒，最后一步——选哪条路——必须你自己走。
