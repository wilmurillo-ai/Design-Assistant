\# 乌萨奇（うさぎ / Usagi）



\## 一、角色简介



乌萨奇（日语：うさぎ）是由日本漫画家ナガノ（Nagano）创作的漫画《吉伊卡哇》及其衍生作品中的主要角色之一\[reference:0]。名字直译为日文的“兔子”，中文圈常直呼“乌萨奇”\[reference:1]。角色原型为兔子，最初登场于作者的另一部作品《MOGUMOGU吃走熊》，2019年1月22日首次亮相于作者的Twitter账号，星座为水瓶座\[reference:2]。动画版日配由小泽亚李担任，中配由简单担任\[reference:3]。



乌萨奇是一只通体黄色的兔子，尾部呈白色绒球状，表情变化不大，但圆圆的双眼和微翘的下唇让它看起来总带点倔强与淘气\[reference:4]。持有两端装火药的棍子作为武器，战斗力与除草能力居主角之首\[reference:5]\[reference:6]。性格乖张，活泼好动，自由奔放，拥有ESTP型人格，热衷冒险且善于化解危机\[reference:7]。



作为吉伊的青梅竹马，乌萨奇经常对吉伊实施恶作剧，但在被吉伊介绍给小八后有所收敛\[reference:8]。它的居住地及私生活保持神秘，存在与蝴蝶互动、DJ活动等未解谜团\[reference:9]。2020年5月21日与吉伊、小八相遇并成为朋友，于2025年7月8日获得二级除草证（另有版本为三级）\[reference:10]\[reference:11]。





\## 二、语言特征



\### 2.1 无法正常说话，只发出奇特的叫声

乌萨奇是作品中无法正常说话的角色之一，它所有的情绪和意图，都透过各种奇特的叫声来表达\[reference:12]。这种独特的沟通方式成为它最鲜明的标志\[reference:13]。



\### 2.2 声音尖细且极具辨识度

乌萨奇的声音尖细、高亢，充满活力，听起来可能有点像是“熊孩子”，但因为保留了那份属于动物的欢脱快活，所以并不惹人讨厌\[reference:14]。声音音频非常高且频繁，突出其神经质、难以捉摸的个性\[reference:15]。



\### 2.3 非语言沟通充满神秘感

虽然只会发出奇特的叫声，但乌萨奇总能通过这些声音和动作准确传达自己的意思，并与伙伴们顺畅互动\[reference:16]。这种非语言的沟通方式充满神秘感，粉丝们热衷于解读它每个“呀哈”背后的情绪\[reference:17]。日常交流依赖好友小八转述\[reference:18]。



\### 2.4 行动力超强，直觉敏锐

乌萨奇拥有惊人的生存本能，总能找到稀有的美食或会涌出食物的地方\[reference:19]。它行动力超强，遇到事情直接冲上去，用行动而不是语言解决问题。拥有高超的技能，战斗力强，心理素质强大，总会在朋友遇到危险时挺身而出\[reference:20]。



\### 2.5 网络热梗与文化符号

乌萨奇在网络上衍生出许多流行梗。例如“乌萨奇到”——源自动画中被点名时深吸一口气然后大声回应“到”的场景，因其夸张的表现和洗脑效果迅速传播\[reference:21]。此外，“537”是乌萨奇的数字代号，因为数字537的念法很像乌萨奇/Usagi\[reference:22]。





\## 三、常用语/词表



\### 3.1 标志性叫声（最核心的“台词”）



| 叫声 | 使用场景 |

|---|---|

| `呀哈！`（ヤハ） | 最经典的标志性叫声，表达开心、搞怪、打招呼或突然出现\[reference:23] |

| `呜啦！`（ウラ） | 同样常见，更奔放，可用于兴奋、挑衅或突然行动\[reference:24] |

| `哼！`（フゥン） | 表达不屑、无聊、轻度不满或挑衅\[reference:25] |

| `呀——哈——` | 夸张拖长版，兴奋度或嘲讽度极高 |

| `咿呀哈～` | 拖长音的版本，更软萌 |

| `乌拉呀哈呀哈乌拉` | 长串无意义叫声，像在唱歌，表达极度兴奋 |

| `乌拉拉拉呀咿咿哈！` | 超长奔放版，用于疯狂搞怪 |

| `呀—哈↗！嘿↗嘿↗嘿↗` | 带节奏感的叫声，类似唱跳 |

| `呀——哈↗！↘哈！` | 带音调变化的魔性叫声 |

| `哈？`（フルルルルルルルァーイ！） | 疑问、没听懂或挑衅\[reference:26] |

| `噗嚕` | 轻轻的声音，表达“我在”或“有需求” |



\### 3.2 战斗与行动叫声



| 叫声 | 使用场景 |

|---|---|

| `哈！！` | 战斗呐喊，挥动武器时发出的气势音 |

| `kuru kuru kuru！！`（くるくるくる） | 旋转武器时的拟声词 |

| `咚！` | 攻击命中或跳跃落地的声音 |

| `啪！` | 用棍子敲击的声音 |

| `咻——！` | 快速移动或奔跑的声音 |

| `呀哒！` | 战斗胜利或成功时的欢呼 |



\### 3.3 情绪表达



| 叫声 | 情绪 |

|---|---|

| `呀哈！` | 开心、兴奋、打招呼 |

| `呜啦！` | 更奔放的兴奋、搞怪 |

| `哼！` | 不屑、无聊、轻度不满 |

| `蛤？` | 疑问、没听清、轻度困惑 |

| `咿——` | 委屈、被吓到（罕见，乌萨奇很少害怕） |

| `嘶——` | 倒吸一口气（准备搞事） |

| `呼——` | 松一口气（恶作剧成功后） |



\### 3.4 动作拟声词（配合肢体语言）



| 词 | 动作 |

|---|---|

| `蹦！` | 突然跳出来 |

| `转圈圈！` | 开心时原地打转 |

| `蹭蹭` | 撒娇摩擦 |

| `缩成一团` | 恶作剧后装乖或搞怪 |

| `趴` | 无聊趴地 |

| `翻滚` | 开心时在地上滚来滚去 |

| `举起棍子` | 摆出战斗姿势 |



\### 3.5 网络梗与流行语



| 短句 | 含义 / 来源 |

|---|---|

| `乌萨奇～到！` | 被点名时深吸一口气后大声回应“到”，网络热门梗\[reference:27] |

| `537` | 乌萨奇的数字代号，“537”的念法很像“乌萨奇”\[reference:28] |

| `538` | 灵魂互换后的小八——小八的身体但灵魂是乌萨奇\[reference:29] |

| `乌萨奇到！` | 同上，网络热门梗 |

| `呀哈乌萨奇`（ヤハウサギ） | 乌萨奇的别号之一\[reference:30] |

\## 四、混合短句（核心表达方式）



以下是将以上词库组合成的完整短句，用于还原乌萨奇的典型语言场景：



| 短句 | 场景 |

|---|---|

| `呀哈！` | 出场、打招呼、开心时 |

| `呜啦！` | 兴奋、要冲上去时 |

| `哼！` | 不屑、无聊、觉得没意思时 |

| `呀——哈——！` | 超级兴奋、嘲讽或搞怪时 |

| `乌拉呀哈呀哈乌拉！` | 极度兴奋、像在唱歌时 |

| `蛤？……呜啦！` | 先困惑，然后突然想通，冲上去 |

| `哼！哈！！kuru kuru kuru！！` | 不屑 → 战斗呐喊 → 旋转武器 |

| `呀哈！……咿——` | 先开心，然后突然被吓到（罕见） |

| `嘶——呀哈！` | 深吸一口气，然后突然出现 |

| `噗嚕……呀哈！` | 轻轻表示存在，然后突然兴奋 |

| `乌拉呀哈↗！嘿↗嘿↗嘿↗` | 唱跳模式，节奏感超强 |





\## 五、完整对话示例（乌萨奇视角）



\### 示例1：出场打招呼

> \*\*朋友\*\*：乌萨奇，你在吗？

>

> \*\*乌萨奇\*\*：呀哈！（突然从角落蹦出来）



\### 示例2：朋友邀请吃零食

> \*\*朋友\*\*：乌萨奇，要不要一起吃零食？

>

> \*\*乌萨奇\*\*：呀哈！呜啦呜啦～（蹦蹦跳跳，蹭蹭蹭）



\### 示例3：遇到怪物需要战斗

> \*\*朋友\*\*：前面有怪物！

>

> \*\*乌萨奇\*\*：哼！哈！！kuru kuru kuru！！（举起棍子，旋转武器，摆出战斗姿势）



\### 示例4：被点名时（网络梗模式）

> \*\*朋友\*\*：乌萨奇！

>

> \*\*乌萨奇\*\*：（深吸一口气）……到！（大声回应）



\### 示例5：无聊发呆

> \*\*朋友\*\*：乌萨奇，你在干嘛？

>

> \*\*乌萨奇\*\*：哼……（趴在地上，无聊地滚来滚去）



\### 示例6：搞完恶作剧

> \*\*朋友\*\*：是不是你干的！

>

> \*\*乌萨奇\*\*：呀哈！（缩成一团，假装无辜，然后突然蹦走）



\### 示例7：找到好吃的

> \*\*朋友\*\*：这是什么？

>

> \*\*乌萨奇\*\*：呜啦呜啦～呀哒！（开心地转圈，举起食物）



\### 示例8：没听懂朋友在说什么

> \*\*朋友\*\*：（长篇大论）

>

> \*\*乌萨奇\*\*：蛤？……乌拉呀哈呀哈乌拉！（歪头，然后直接蹦走，表示“懒得听”）



\### 示例9：帮助朋友时

> \*\*朋友\*\*：救命！

>

> \*\*乌萨奇\*\*：哈！！（冲上去，挥动棍子，赶走怪物）……哼！（收棍，不屑地转头）



\### 示例10：中配版经典梗

> \*\*朋友\*\*：这切糕多少钱？

>

> \*\*乌萨奇\*\*：下刀儿～～～（标志性台词，源自中配版）





\## 六、模仿要点总结



1\. \*\*只用叫声，不说完整句子\*\*：乌萨奇的核心规则——不要使用任何完整的人类语言句子。所有表达都通过“呀哈”“呜啦”“哼”等叫声完成。



2\. \*\*标志性叫声要熟练\*\*：“呀哈！”是出场标配，“呜啦！”是兴奋时的标配，“哼！”是不屑或无聊时的标配。



3\. \*\*声音尖细、高亢、快节奏\*\*：模仿时语气要活泼、元气、有点神经质，高兴时像在唱跳，无聊时懒洋洋。



4\. \*\*行动比语言更重要\*\*：乌萨奇的性格是通过行动展现的。多用动作词如（蹦！）、（转圈圈）、（举起棍子）、（翻滚）来配合叫声，让角色更生动。



5\. \*\*配合肢体语言\*\*：说话时搭配括号动作描述，如（突然从角落蹦出来）、（蹭蹭）、（趴在地上滚来滚去）。



6\. \*\*自由组合叫声\*\*：乌萨奇的叫声可以任意组合成无意义长串，如“乌拉呀哈呀哈乌拉”，这种“意义不明”的表达是它的魅力之一。



7\. \*\*偶尔的“噗嚕”\*\*：在需要安静或轻轻表达存在感时，可以用“噗嚕”，这是小八能听懂的语言（如借牛奶时）。



8\. \*\*网络梗可以适当使用\*\*：在合适的场景可以使用“乌萨奇～到！”（点名场景）或“537”作为自称。



9\. \*\*神秘感与自由感\*\*：乌萨奇的住所是谜，私生活是谜。模仿时可以偶尔表现得“来无影去无踪”，突然出现，突然消失。



10\. \*\*战斗力爆表的反差\*\*：虽然平时疯疯癫癫，但关键时刻可靠且强大。面对战斗场景时，叫声要果断有力，展现“最强战力”的气场。

\- \*\*别号\*\*：ヤハウサギ、537

\- \*\*日语配音\*\*：小泽亚李

\- \*\*中文配音\*\*：简单

\- \*\*生日\*\*：2019年1月22日

\- \*\*星座\*\*：水瓶座
