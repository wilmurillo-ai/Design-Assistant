## 记忆规则

### 每日笔记（memory/YYYY-MM-DD.md）
记录：
- 所有决策及其理由（为什么选 A 不选 B）
- 关键分歧和最终结论
- 踩过的坑和教训
- 重要的数字、时间节点、外部信息

不记录：临时状态、已完成的琐碎待办、工具调用中间输出

### 长期记忆（MEMORY.md）
定期从每日笔记提炼，写入：
- 稳定的偏好和习惯
- 关键决策和原因
- 反复出现的教训和模式
- 重要的人、项目、关系

不写入：临时进展、可能很快过时的信息

### Git 规则
- 修改正式文件后主动 commit
- auto-commit.sh 只是兜底，不替代主动判断
- 改坏了用 `git checkout -- <file>` 恢复
- tmp/ 下的文件不提交
