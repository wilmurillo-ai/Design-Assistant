# 二、标准剧本结构模板

## 2.1 通用黄金结构（60-120集爆款标准配置，每集1-3分钟，短剧/漫剧通用）

### 【第一部分：钩子期（1-5集）】
- **第1集：** 直接抛最高冲突，展示主角最低谷，亮出金手指/特殊身份
- **第2集：** 反派持续碾压，主角暗中蓄力，首次小范围反击
- **第3集：** 第一次公开打脸，小反派吃瘪，观众情绪第一次释放
- **第4集：** 更大的反派登场，危机升级，主角陷入新的困境
- **第5集：** 结尾留大悬念，主角即将爆发，引导用户继续观看

### 【第二部分：发展期（6-50集）】
- 每5集一个小单元，完成一次"遭遇危机->反派装逼->主角打脸->收获奖励"的完整爽点循环
- 逐步升级反派等级：小喽啰→中层→反派BOSS亲信→最终BOSS
- 逐步解锁主角身份/能力：第一层身份→第二层身份→最终隐藏身份
- 逐步拓展世界观：家庭→公司→城市→行业→全国/世界

### 【第三部分：高潮期（51-90集）】
- 主角与最终BOSS正面交锋
- 所有伏笔回收，最大反转揭开
- 连续高能爽点轰炸，每集都有大场面
- 重要配角的命运结局逐步揭晓

### 【第四部分：收尾期（91-100集）】
- 最终BOSS彻底覆灭，得到应有的惩罚
- 主角事业达到顶峰，情感归宿确定
- 开放式结尾或彩蛋，为续集留下空间
- 升华主题，传递正向价值观

## 2.2 单集写作标准（真人短剧版）
```
【第X集】
时长：1分30秒左右
场景：[场景描述]
人物：[出场人物]

□ 画面（0-10秒）：[开场画面，承接上集悬念，快速进入冲突]
□ 对话/旁白：[简洁有力，每句都要有信息量，不要废话]
□ 动作：[夸张有表现力，符合竖屏观看特点]
□ 画面（10-60秒）：[核心冲突展开，反转/爽点设计]
□ 对话/旁白：[情绪饱满，有张力，制造情绪起伏]
□ 动作：[关键动作特写，强化视觉冲击]
□ 画面（60-90秒）：[本集高潮，爽点兑现，结尾立刻抛出新悬念]
□ 对话/旁白：[最后一句必须留钩子，引导看下一集]
□ 结尾画面：[定格在主角装逼/反派震惊/关键道具特写]

# 爽点标记：[标注本集核心爽点，方便拍摄时重点突出]
# 付费卡点提示：[如果是付费剧，标注适合卡点的位置]
```

## 2.3 单集写作标准（AI漫剧专属版）
```
【第X集】
时长：1分30秒左右
场景：[场景描述，注明风格：古风/现代/科幻/玄幻等]
人物：[出场人物，注明形象特征：服装/发型/表情/标志性配饰等]
分镜数量：[建议8-12格分镜，匹配AI生成节奏]

□ 分镜1（0-5秒）：[开场画面，承接上集悬念，强冲突视觉化，适合做封面]
  - 画面描述：[详细到可直接喂给AI生成，例如："近景，女主被扇耳光，脸上红印清晰，眼泪在眼眶里打转，背景是豪华客厅"]
  - 对话/旁白：[不超过20字，精炼有冲击力]
  - 动态效果：[AI动效提示：镜头拉近/推远/震屏/特效闪烁等]
□ 分镜2-9（5-75秒）：[核心冲突展开，每格分镜一个小节点，反转/爽点分阶段呈现]
  - 画面描述：[每格独立画面描述，保持画风统一]
  - 对话/旁白：[每格不超过30字，口语化，情绪饱满]
  - 动态效果：[人物动作/特效/转场提示]
□ 分镜10（75-90秒）：[本集高潮，爽点兑现，结尾留大悬念]
  - 画面描述：[视觉冲击力强，适合定格]
  - 对话/旁白：[最后一句必须是钩子，例如："你敢动我？我爸可是城主！"]
  - 结尾定格：[明确定格画面，引导下集观看]

# 爽点标记：[标注本集核心爽点，方便AI重点渲染]
# AI生成提示：[画风统一提示：国漫/日漫/写实/Q版/3D等，统一人物特征描述]
# 付费卡点提示：[如果是付费剧，标注适合卡点的位置]
# 配音提示：[人物语气/情绪/背景音乐风格提示]
```
