# 四、剧本产出流程（工业化标准流程）

## 4.1 前期对标阶段（成功率提升70%）
1. **类型确认：** 明确是真人短剧还是AI漫剧，选择对应赛道的对标作品
2. **对标筛选：** 选择上线≤45天的同题材爆款作为参考，要求：
   - **真人短剧**：播放数据达标：完播率≥40%，点赞率≥5%
   - **AI漫剧**：播放数据达标：完播率≥45%，点赞率≥6%
   - 用户画像匹配：核心受众年龄偏差≤5岁，性别比例差异≤15%
   - 商业化验证：已有广告植入或分账记录
3. **爆款拆解：** 拆解对标作品的核心结构、爽点频率、钩子设计、情感曲线，保留80%已验证框架
   - 漫剧额外拆解：画风类型、分镜节奏、AI动效设计、人物特征设定
4. **创新设计：** 加入≥20%的原创元素，避免同质化，保证内容相似度在30-70%法律安全区间

## 4.2 创作执行阶段
1. **需求确认：** 明确题材类型、内容形式（真人/漫剧）、集数、时长、目标受众、核心卖点、商业化方向
   - 漫剧额外确认：画风要求、AI生成工具适配要求、配音风格
2. **标签优化设计：** 提前确定核心题材关键词，在账号资料、文案、话题、评论引导中统一使用，提升标签匹配精准度，可将核心标签权重提升2-3倍
3. **设定创作：**
   - 通用：核心设定、人物小传、分阶段剧情走向，每5集一个爽点循环
   - 漫剧额外：人物标准形象描述（精确到发型、服装、配饰、标志性特征）、整体画风设定、场景风格库
4. **样稿写作：** 先写前5集样稿，重点优化开场钩子和每集结尾悬念，测试节奏符合要求，确保前3秒完播率≥70%，前30秒完播率≥40%
   - 漫剧样稿需包含完整分镜描述和AI生成提示，先测试3-5张图验证画风一致性
5. **互动节点设计：** 每集设置1个互动引导点（如"评论区猜下集剧情"、"喜欢的话点赞收藏"），提升互动质量评分，获得更高推荐权重
6. **完整创作：** 按照对应模板完成全部剧集创作，要求：
   - **真人短剧：**
     - 每集有效反转≥3次（都市情感类≥3.8次）
     - 甜宠剧"虐-甜"间隔≤5分钟
     - 单集成本控制在行业平均28万的60-120%区间
     - 避免"开头套路+内容注水"模式，硬广内容占比≤10%，采用自然植入的软广形式
   - **AI漫剧：**
     - 每集分镜8-12格，每格对应一个节奏点
     - 每格画面描述精确到可直接输入AI生成
     - 人物特征保持100%一致，避免AI生成崩坏
     - 单集成本控制在真人短剧的10-20%区间
7. **优化调整：** 根据反馈修改，强化爽点，优化节奏，确保通过广电总局"清风"系统检测
   - 漫剧额外优化：AI生成提示词，提升画面准确率和画风统一性

## 4.3 运营适配阶段
1. **互动设计：** 每10集设置一个互动点，引导评论区"求更新"，达到500条触发加更机制；重点优化第3-5天的互动引导内容，可使整体播放量提升2.5倍
2. **商业化预留：** 每集预留3个品牌植入点位，适配广告变现需求
3. **数据监测：** 上线后7天推流周期内每日监测完播率、点赞率、评论率数据，前3秒完播率低于70%立即调整开头内容
4. **算法驯化：** 保持垂直题材持续更新，30天内可进入平台"优质内容供给池"，获得算法加权推荐
5. **多平台适配：** 针对不同平台特点优化内容：
   - 短视频平台（抖音、快手）：重点优化前3秒钩子，提升完播率
   - 微短剧App：强化连续悬念，提升用户付费转化率
   - 长视频平台：提升内容深度，适配分账规则
   - 小程序平台：缩短爽点间隔，提升分享传播率
