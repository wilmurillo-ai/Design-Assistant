# 示例剧本：《龙婿入世》第1集（男频赘婿题材）

时长：1分20秒
场景：豪华酒店宴会厅
人物：陈平（赘婿，真实身份是龙殿主）、李倩倩（妻子）、王少（富二代）、众宾客

[0-10秒]
画面：富丽堂皇的宴会厅，正在举办李家寿宴，陈平穿着朴素，站在角落被所有人指指点点
李倩倩（一脸嫌弃）："陈平，你个废物来干什么？还嫌不够丢人的吗？"
王少（搂着美女，嘲讽）："这不是李家的上门废物吗？怎么，今天是来给岳母磕头祝寿的？"
众宾客哄堂大笑。

[10-50秒]
画面：陈平面无表情，没有理会众人的嘲讽，走到岳母面前，递出一个古朴的木盒
陈平："妈，这是我送你的寿礼，千年人参，价值千万。"
岳母（直接把盒子扔在地上，人参滚出来）："什么破东西！我看你是想咒我早死！王少送的是百万翡翠，你送个破人参也敢拿出来？"
王少（踩碎人参，张狂大笑）："什么千年人参，我看就是路边十块钱买的假货！废物就是废物，永远上不了台面！"

[50-80秒]
画面：陈平看着被踩碎的人参，眼神逐渐变冷，周身气压骤降
陈平（声音冰冷，一字一句）："你，踩碎了我送的寿礼，今天，打断你双腿，爬着出去。"
王少（愣住，随即狂笑）："你个废物敢跟我这么说话？我看你是活腻了！来人，给我打！"
画面定格在几个保镖冲向陈平，陈平嘴角勾起一抹冷笑。

# 爽点标记：身份反差、嘲讽积累、主角放狠话，情绪铺垫到位
