## v1.0.0 - 2026-01-28

### ✨ 首次发布

### ✨ 首次发布
- 完整实现**四层纵深安全防御体系**
  - **第一层**：静态代码扫描，检测恶意代码、硬编码密钥、危险系统调用
  - **第二层**：逻辑安全审计，基于 OWASP Top 10 检查代码漏洞和权限越权
  - **第三层**：运行时实时防护，检测并阻断提示注入、命令注入、SSRF 攻击
  - **第四层**：定期安全巡检，每日基线完整性检查、每周全量扫描

### Features
- 📋 标准化三层安装检查流程沉淀
- 🛡️ 开箱即用，零配置
- 📊 自动生成安全报告
- 🔒 完整性基线保护，检测文件篡改
