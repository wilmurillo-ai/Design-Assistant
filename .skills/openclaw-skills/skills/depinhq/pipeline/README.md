# Pipeline

> A pipeline's job is not to make you feel good about your forecast. Its job is to tell you the truth.

Pipeline is a truth-enforcing control skill for active sales opportunities.

## What it does

This skill helps you:
- detect stalled and fake-active deals
- spot stage pollution
- surface the few deals that need attention now
- improve forecast reliability
- clean up pipeline hygiene

## What it does NOT do

- find targets for you
- qualify one engaged lead in depth
- write proposals
- redesign your entire sales process from scratch

## Works Well With

- `@dpetcr/prospect` for filtering targets before outreach
- `@AGIstack/lead` for judging one engaged opportunity
- `@dpetcr/proposal` for formal deal progression
