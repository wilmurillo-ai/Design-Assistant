# Prospect

> Turn raw target lists into prioritized prospects worth pursuing.

Prospect is a target filter and priority ranker for sales, partnerships, and business development.

## What it does

This skill helps you:
- filter out poor-fit targets
- rank the remaining prospects
- identify missing information
- decide whether to pursue, research, watch, or drop

## What it does NOT do

- find targets for you
- handle active conversations
- write outreach copy

Use Prospect before contact.  
Use Lead after engagement begins.

## Works Well With

- `@AGIstack/lead` for qualifying engaged prospects after they reply
- `@dpetcr/pipeline` for managing active opportunities as a system
