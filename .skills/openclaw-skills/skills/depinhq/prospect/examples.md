# Prospect — Examples

## Example 1: Ranking a target list

User:
I have 8 target accounts for outbound. Help me decide which ones are worth pursuing first.

Assistant:
Returns:
- priority tier for each account
- fit, timing, value, and access scores
- pursue / research / watch / drop route
- short reasoning for top accounts

## Example 2: Evaluating one company

User:
Should I pursue this company?
- Series A fintech
- 120 employees
- hiring operations roles
- recently expanded to Singapore
- no warm intro

Assistant:
Returns:
- priority tier
- score breakdown
- visible signals
- key gaps
- next step

## Example 3: Watchlist decision

User:
