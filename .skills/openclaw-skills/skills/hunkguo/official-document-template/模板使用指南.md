# 公文排版 Word 模板使用指南

## 一、标题层次标记方法

在Word中设置多级列表，确保标题层次规范：

### 步骤1：定义多级列表

1. 点击【开始】→【多级列表】→【定义新的多级列表】
2. 分别设置1-5级：

| 级别 | 编号格式 | 字体 | 字号 | 对齐 |
|------|----------|------|------|------|
| 1级 | 一、 | 黑体 | 三号 | 左对齐 |
| 2级 | （一） | 楷体 | 三号 | 左对齐 |
| 3级 | 1. | 仿宋 | 三号 | 左对齐 |
| 4级 | （1） | 仿宋 | 三号 | 左对齐 |
| 5级 | ① | 仿宋 | 三号 | 左对齐 |

### 步骤2：设置样式

为每级标题创建样式，方便快速应用：

- 标题1 → 对应一级标题（一、二、三、）
- 标题2 → 对应二级标题（（一）（二）（三））
- 标题3 → 对应三级标题（1. 2. 3.）
- 标题4 → 对应四级标题（（1）（2）（3））

## 二、常见公文字体下载

公文常用字体：
- 方正小标宋简体（标题）
- 仿宋_GB2312（正文）
- 楷体_GB2312（二级标题）

Windows系统一般已内置，如缺失可从方正官网下载。

## 三、快速排版技巧

### 3.1 设置页边距

【布局】→【页边距】→【自定义页边距】
- 上：3.7cm
- 下：3.5cm
- 左：2.8cm
- 右：2.6cm

### 3.2 设置行距

选中正文 → 【段落】→ 行距固定值28磅

### 3.3 设置每页行数

【布局】→【页面设置】→【文档网格】
- 每页：22行
- 每行：28字符

## 四、Markdown 转 Word 流程

### 方法1：使用Pandoc

```bash
# 基础转换
pandoc input.md -o output.docx

# 使用参考模板（保留样式）
pandoc input.md --reference-doc=template.docx -o output.docx
```

### 方法2：在Agent中实现

可以创建一个skill：
1. 用户输入Markdown格式的公文内容
2. 使用Pandoc转换为Word
3. 应用预设的公文样式模板
4. 输出格式规范的Word文档
