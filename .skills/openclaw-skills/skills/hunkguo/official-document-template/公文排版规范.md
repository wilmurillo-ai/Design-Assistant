# 公文排版规范

按照《党政机关公文格式》（GB/T 9704-2012）国家标准制作公文。

## 一、标题层次

公文标题层次采用以下格式（从大到小）：

| 层级 | 格式 | 示例 |
|------|------|------|
| 第一级 | 一、二、三、 | 一、关于加强环境保护工作 |
| 第二级 | （一）（二）（三） | （一）明确工作目标 |
| 第三级 | 1. 2. 3. | 1.制定实施方案 |
| 第四级 | （1）（2）（3） | （1）细化任务分工 |
| 第五级 | ①②③ | ①落实责任主体 |

**注意事项：**
- 第一级标题后用顿号"、"，其他级别不用标点
- 标题末尾不加标点符号
- 标题一般独占一行，前后各空一行

# 五、标点符号规范

- 标题末尾不加标点
- 序号后：一级用"、"，其他不用标点
- 引用文件：《×××》用书名号
- 引用条款：根据《××》第×条规定
- 日期：2024年1月1日（不用2024.1.1）
- 数字：概数用汉字"三五个"，精确数用阿拉伯数字

# 六、常见错误示例

❌ 错误：一、加强领导。
✅ 正确：一、加强领导

❌ 错误：（一）、完善建设
✅ 正确：（一）完善建设

❌ 错误：1、制定工作计划
✅ 正确：1.制定工作计划

❌ 错误：2024.1.1
✅ 正确：2024年1月1日
