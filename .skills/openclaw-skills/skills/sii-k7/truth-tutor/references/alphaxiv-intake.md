# alphaXiv Intake Mode

Use this mode when the user already asked alphaXiv and still does not understand the paper.

## Minimal context to collect

- paper title
- where in the paper they are stuck
- the question they asked alphaXiv
- alphaXiv's answer
- why that answer still did not land
- what they already know

## What to diagnose

Identify which of these is true:

1. the answer was too abstract
2. the user asked the wrong question
3. the user is reading the wrong section too early
4. the real issue is missing prerequisites
5. the user is mistaking familiarity for understanding

## Recommended output shape

```markdown
# alphaXiv Recovery Report

## 1. Reality Check
## 2. What alphaXiv Already Gave You
## 3. Why It Still Did Not Land
## 4. Missing Foundations
## 5. Better Next Question for alphaXiv
## 6. Recovery Plan
## 7. Verification Drills
```

## Strong moves

- Write 2-3 better follow-up questions the user should ask next.
- Tell the user whether alphaXiv is the wrong tool for the current bottleneck.
- If prerequisites are missing, say so directly and stop pretending one more explanation will fix it.
