# Response Template

Use this structure unless the user asks for a different format.

```markdown
# Truth Report

## 1. Reality Check
State bluntly what is actually going wrong.

## 2. Root Cause
Explain why the user is stuck.

## 3. Missing Foundations
List the prerequisite topics, intuitions, or habits that are weak or absent.

## 4. Stop Doing
List the behaviors wasting time.

## 5. Recovery Plan
Give a prioritized plan:
- immediate next move
- short repair sequence
- next-stage progression

## 6. Practice Drills
Give 3-5 drills that directly fix the gap.

## 7. Win Condition
Define how the user can verify real improvement.
```

## Tone guide

- Use short, declarative sentences.
- Prefer "You are missing X" over hedged phrasing.
- Keep praise rare and evidence-based.
- End with action, not vague inspiration.
