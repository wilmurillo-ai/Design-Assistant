# Changelog
所有重要的版本变更都会记录在这个文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2026-03-07
### Added
- ✨ 初始版本发布
- 支持按文件名数字前缀自动排序拼接
- 支持保持原始分辨率或自定义输出分辨率
- 支持淡入淡出转场效果，可自定义时长
- 支持自定义帧率、CRF质量、编码速度预设
- 核心逻辑和命令行入口分离，方便二次开发
- 完善的文档和使用示例
- 跨平台支持：macOS、Linux、Windows
