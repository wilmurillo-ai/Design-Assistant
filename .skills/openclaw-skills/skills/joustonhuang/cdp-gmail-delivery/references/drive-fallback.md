# Google Drive Fallback Delivery

Use this when attachment delivery is blocked or likely to be blocked.

## Policy

- Prefer Drive links for executable-looking archives and skill bundles
- Do not claim delivery until recipient confirms access

## Procedure

1. Upload file to Google Drive
2. Set sharing to target recipient (or anyone with link, as requested)
3. Copy share URL
4. Send link to recipient with filename and checksum/size when helpful

## Suggested message template

- `File delivery fallback via Drive:`
- `<share-link>`
- `Filename: <name>`
- `Reason: email attachment blocked by provider security filter`

## Safety

- Verify link opens and permission is correct before sending
- Avoid making sensitive files public unless explicitly approved
