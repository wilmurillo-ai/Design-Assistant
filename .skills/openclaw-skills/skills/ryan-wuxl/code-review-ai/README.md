# 🔍 代码审查助手

自动代码审查，提升代码质量。

## ✨ 功能特点

- 🔍 **自动审查** - 分析代码结构和风格
- ⚡ **性能优化** - 识别性能瓶颈
- 🔒 **安全检测** - 发现潜在安全漏洞
- 📊 **质量报告** - 生成详细审查报告
- 🛠️ **多语言支持** - 支持 JS/Python/Go/Java 等

## 🚀 快速开始

### 安装

```bash
clawhub install code-review-assistant
```

### 审查单个文件

```bash
node scripts/review.mjs --file ./src/app.js
```

### 审查整个项目

```bash
node scripts/review.mjs --dir ./src --output ./review-report.md
```
