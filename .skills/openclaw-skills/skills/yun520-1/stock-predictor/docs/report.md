## 4. 模型架构

### 4.1 机器学习模型

#### Random Forest (RF)
- 树数量：80
- 最大深度：5
- 最小样本分割：10

#### Gradient Boosting (GB)
- 树数量：80
- 最大深度：3
- 学习率：0.1

#### Logistic Regression (LR)
- 正则化：L2
- 最大迭代：500

#### XGBoost
- 树数量：80
- 最大深度：3
- 学习率：0.1
- 子采样：0.8

#### LightGBM
- 树数量：80
- 最大深度：5
- 学习率：0.1
- 子采样：0.8

### 4.2 深度学习模型

#### LSTM
- 序列长度：10
- 隐藏层大小：32
- 层数：1
- Dropout: 0.1
- 批次大小：16
- 训练轮数：15

### 4.3 集成策略

简单平均集成：
```
最终概率 = (P_RF + P_GB + P_LR + P_XGB + P_LGB + P_LSTM) / 6
最终预测 = 最终概率 > 0.5 ? 涨 : 跌
```

1. **数据延迟**: API 数据有 15 分钟延迟
2. **因子有限**: 仅使用技术指标，未包含基本面
3. **市场变化**: 模型可能需要定期重新训练
