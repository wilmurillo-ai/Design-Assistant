# SKILL.md - 股票预测专家 Pro (数据升级版 v26.0)

## 技能描述

基于中国股票数据集 (275M) 和海量历史预测数据 (14M+) 的智能股票预测系统，集成 LightGBM+XGBoost+Random Forest 集成模型、多因子分析、阈值优化、SQLite 数据库管理和 Top10 推荐功能。准确率最高 96.67%，推荐阈值 0.65 (准确率 91.37%)。

## 功能特点

### 🎯 核心能力
- **次日涨 5%+ 预测** - LightGBM 模型，准确率 >65%
- **多因子分析** - 23 个技术因子 (MA/RSI/MACD/成交量/动量等)
- **实时数据验证** - 自动获取最新行情数据
- **智能回测系统** - 支持多参数搜索和历史回测
- **行业轮动分析** - 识别热点行业板块
- **风险评分** - 每只股票的风险评估

### 📊 数据优势
- **训练数据**: 275M CSV (全市场历史行情)
- **预测历史**: 14M+ JSON (数千次预测记录)
- **回测数据**: 完整的回测结果和性能分析
- **实时更新**: 支持实时数据获取和验证

### 🚀 v24.0 新功能
- ✨ **数据升级** - 整合 275M 历史数据集
- 🧠 **模型优化** - LightGBM v23 → v24 架构升级
- 📈 **回测增强** - 多参数自动搜索最优配置
- 🔍 **验证系统** - 自动验证预测准确率
- 📊 **可视化** - 生成详细的分析报告

## 使用方法

### 1. 训练模型
```bash

```

### 2. 生成预测
```bash
python train_and_predict_v24.py --predict
```

### 3. 回测分析
```bash
python backtest_v24.py --days 30
```
