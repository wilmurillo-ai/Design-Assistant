# 🧒 Easy Travel with Child — 亲子出行规划助手

> 让带娃出行不再纠结：AI 帮你搜真实酒店数据、比价、挑选适合孩子的玩法，大人也能好好休息。

---

## ✨ 这个 Skill 解决什么问题？

每到周末或节假日，带娃家庭最头疼的问题：

- 🏨 **酒店说有"亲子乐园"，到了才发现要额外收费，或者孩子全程需要家长守着**
- 🎪 **不知道哪些景点适合自家孩子的年龄和性格**
- 🔄 **要在十几家酒店里对比，累到放弃**
- 😮‍💨 **大人想休息，孩子也想玩，但两者难以兼顾**

**Easy Travel with Child** 用 AI + 飞猪实时数据，帮你把这些全搞定。

---

## 🚀 核心能力

### 1. 实时酒店检索与评估
接入飞猪实时数据，自动搜索目标城市的亲子酒店，评估每家酒店的：
- 儿童游乐场是否**真的免费**
- 游乐区是否需要**家长全程陪伴**（⚠️ 核心指标）
- 设施质量、面积、适合年龄段

### 2. 度假套餐一键搜索
搜索"酒店 + 景点门票"组合套餐，省去分开购买的麻烦，支持：
- 迪士尼、乐高、海昌等热门亲子主题套餐
- 含早餐、含接送、含儿童门票的多种套餐类型

### 3. 孩子画像精准匹配
根据孩子的**性别 × 年龄 × 性格**推荐最适合的游玩场地：

| 性格类型 | 推荐场地方向 |
|----------|------------|
| 🏃 运动活跃型 | 蹦床公园、室内攀岩、水上乐园 |
| 📚 文静安静型 | 绘本馆、艺术手工坊、自然博物馆 |
| 🔍 探索好奇型 | 科技馆、儿童职业体验馆、自然农场 |

### 4. 多酒店横向对比
自动生成对比表格，价格、游乐场、陪伴要求一眼看清，平均决策时间减少 50%。

### 5. 完整出行方案卡片输出
最终输出结构化「出行方案卡片」，包含推荐酒店、可选套餐、每日行程安排、实时门票价格和预订链接。

## 🛠️ 脚本使用说明

### 搜索亲子酒店

```bash
# 基础搜索（必填城市）
bash scripts/search-hotel.sh -c 上海

# 高级搜索：指定星级 + 日期 + 价格上限
bash scripts/search-hotel.sh -c 上海 -s 4,5 -i 2026-05-01 -o 2026-05-03 -p 1500

# 参数说明
# -c  目的地城市（必填）
# -k  补充关键词，默认"亲子 儿童游乐场"
# -s  星级，多个用逗号分隔，如 4,5
# -i  入住日期 YYYY-MM-DD
# -o  退房日期 YYYY-MM-DD
# -p  最高价格（元/晚）
```

### 搜索度假套餐（含门票）

```bash
# 搜索含迪士尼门票的亲子套餐
bash scripts/search-package.sh -q "上海亲子酒店套餐 含儿童门票"

# 搜索含早餐的周末度假套餐
bash scripts/search-package.sh -q "北京度假套餐 2大1小 含早餐"

# 搜索特定景区套票
bash scripts/search-package.sh -q "上海迪士尼亲子套票 春季"
```

## 💡 使用场景示例

**场景 1：周末带 6 岁好动男孩去上海玩**
> "我想带儿子去上海过五一，他 6 岁，超级好动，我希望他玩的时候我能休息一下"

Skill 会自动：
1. 搜索上海亲子酒店，筛选出「无需家长全程陪同」的游乐场
2. 搜索上海蹦床公园、室内攀岩、乐高乐园等运动类门票套餐
3. 输出含实时价格和预订链接的完整两日方案

---

**场景 2：比较多家酒店**
> "帮我比一下迪士尼酒店、乐高乐园酒店、海昌酒店，哪家更适合带 4 岁女儿"

Skill 会：
- 实时搜索三家酒店数据
- 生成对比表格（价格/游乐场质量/适合年龄/是否需陪同）
- 给出针对 4 岁女孩的个性化推荐意见
