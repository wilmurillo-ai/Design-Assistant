# OpenClaw Workspace Starter Agent Home Template

Your agent needs a home. Without these files, your agent wakes up every session with amnesia. This template fixes that in 10 minutes. SOUL.md, MEMORY.md, AGENTS.md included.

---

**Your agent needs a home. This is it.**

Without these files, your agent wakes up every session with amnesia — no name, no memory, no personality. This template fixes that in about 10 minutes.

## 3-Step Quick Start

1. **Copy this folder** into your OpenClaw workspace (`~/.openclaw/workspace/`)
2. **Open BOOTSTRAP.md** and follow the 5 steps (fill in 3 files, glance at 2)
3. **Delete BOOTSTRAP.md** when you're done — then go to Chat and say hello

That's it. Your agent works now.

## What's In Here

### Files You Edit (your agent's identity)
| File | What It Does | Time to Fill In |
|------|-------------|-----------------|
| **IDENTITY.md** | Agent's name and emoji | 30 seconds |
| **USER.md** | Who you are (so your agent can actually help you) | 5 minutes |
| **SOUL.md** | Agent's personality, tone, and rules | 5 minutes |

### Files That Work Automatically (you can ignore these)
| File | What It Does |
|------|-------------|
| **AGENTS.md** | Operating rules (good defaults, customize later) |
| **TOOLS.md** | Your environment notes (fill in as needed) |
| **HEARTBEAT.md** | What your agent checks during periodic wake-ups |
| **ACTIVE_PLAN.md** | Goal tracking (your agent nudges you) |
| **MEMORY.md** | Long-term memory (your agent maintains this) |

### Guides
| File | What It Does |
|------|-------------|
| **BOOTSTRAP.md** | First-time setup walkthrough (delete after) |
| **UI_GUIDE.md** | Dashboard guide with 🟢 Simple and 🔵 Advanced modes |
| **FAQ.md** | Common questions and fixes |

### Folders
| Folder | What It Does |
|--------|-------------|
| **memory/** | Daily notes (auto-created by your agent) |
| **inbox/** | Task queue (drop tasks here for later) |
| **backup/** | Emergency backups |

## Dashboard Guide

New to the OpenClaw dashboard? **UI_GUIDE.md** has two modes:
- 🟢 **Simple Mode**: You need 4 pages. Just 4. Chat, Agents, Cron Jobs, Usage.
- 🔵 **Advanced Mode**: Full breakdown of every page for when you want more control.

## Philosophy

- **Files = Memory.** Your agent can't remember what it doesn't write down.
- **Structure = Focus.** A messy workspace makes a confused agent.
- **Identity = Usefulness.** An agent with personality is more useful than a generic chatbot.
- **Start simple, add later.** Don't try to fill everything in perfectly on day one.
