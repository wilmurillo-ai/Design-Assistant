# Inbox

Drop task files here. When you're ready to work on something, tell your agent to check inbox/.

## Format

Each task is a markdown file describing what needs to be done:

```markdown
# Task Name

## What
[What needs to happen]

## Context
[Any relevant background]

## Done When
[How to know it's complete]
```

## Why This Exists

Writing tasks down means you don't forget them and your agent can reference them later. It's a simple queue — no fancy tooling needed.
