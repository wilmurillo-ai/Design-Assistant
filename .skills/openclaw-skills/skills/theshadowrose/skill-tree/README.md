# SkillTree — Learning Progress Tracker

Track learning like an RPG skill tree. Prerequisites, milestones, what to study next.

## Usage

```
You: "I just finished learning JavaScript basics"
Agent: "JavaScript Basics complete! 🎯
        Unlocked: DOM Manipulation, Node.js, React
        Suggested next: Node.js (prerequisite for 5 others)
        Progress: 12/47 skills (25%)"
```

## Features

- **Visual skill tree** — dependency graph
- **Prerequisites** — "learn X before Y"
- **Progress tracking** — percentage per branch
- **Smart suggestions** — "what should I learn next?"
- **Time estimates** — how long each skill takes
- **Custom trees** — build your own or use templates
