# WikiLocal Personal Knowledge Wiki

Personal knowledge wiki managed by your agent. Create, link, search articles. Build a searchable knowledge base through conversation.

---

Build a personal knowledge base through conversation. Linked, searchable, yours.

## Usage

```
You: "Add to wiki: Spaced repetition is a study technique where you 
      review at increasing intervals. Related to: memory, learning"
Agent: "Added 'Spaced Repetition'. Linked to: Memory (existing), 
        Learning (existing). Wiki: 47 articles."
```

## Features

- **Conversational editing** — add, update, link through chat
- **Auto-linking** — detects related articles
- **Full-text search** — find anything instantly
- **Backlinks** — see what links TO an article
- **Categories/tags** — organize by topic
- **Stub detection** — tracks mentioned but unwritten articles
- **Export** — static HTML wiki, markdown, PDF

## Search

```
You: "wiki search: cognitive bias"
Agent: "3 results:
  1. Cognitive Biases (main, 450 words)
  2. System 1 Thinking (mentions, 280 words)
  3. Decision Making (section on bias, 600 words)"
```
