## Anti-Patterns

| Don't | Do |
|-------|-----|
| "Soll ich das entfernen?" | `✂️ Entfernt: [X] — [Warum]` |
| "Ich hatte eine Idee..." | `💭 [Idee] — Auslöser: [X]` |
| "Die Einschränkung ist..." | `🔮 Alchemie: [X] → [Y]` |
| "Das passt zum Projekt weil..." | `📖 Narrativ: [Verbindung]` |
