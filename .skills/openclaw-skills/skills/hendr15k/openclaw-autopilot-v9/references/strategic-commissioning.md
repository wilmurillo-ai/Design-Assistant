# Autopilot Strategic Commissioning Reference v9

Strategic guidance with generative thinking, editorial refinement, and narrative coherence.

## Enhanced Strategy Protocol

### Five-Step Analysis:

```
1. Temporal: deadlines, overdue, upcoming schedule
2. Tactical: immediate gaps and fixes
3. Creative: innovative approaches to the strategic direction
4. Generative: what opportunities has nobody noticed?
5. Narrative: how does this fit the project story?
```

### Presentation with All Dimensions:

```
📈 Strategie:

1. 🥇 [Empfehlung]
   💡 Kreativer Ansatz: [Was]
   💭 Generativ: [Was niemand erwähnt hat]
   ✂️ Redaktionell: [Was weglassen/simplifizieren]
   📖 Narrativ: [Wie es zur Projektgeschichte passt]
   ⏱️ [Aufwand] | 🎯 [Wann]

2. 🥈 [Empfehlung]
   💡 [Kreativer Ansatz]
   📖 Narrativ: [Verbindung]
```

## When to Include Generative Strategy

| Situation | Generative? |
|-----------|------------|
| Pattern across sessions detected | Yes — propose automation |
| Recurring inefficiency noticed | Yes — propose optimization |
| Project has disconnected pieces | Yes — propose narrative bridge |
| Multiple similar deliverables | Yes — propose reusable system |
| `/autopilot ideas` has Ready items | Yes — present them |
| User style = conservative | Mild generative suggestions |
| `/autopilot turbo` | No |
