## 2️⃣ 快速使用 (3 分钟)

### 示例 1: 从对话中学习记忆

```python
from memory_api import HongNaoMemorySystem, MemorySystemConfig

# 初始化
config = MemorySystemConfig()
memory_system = HongNaoMemorySystem(config)

# 添加记忆
memory_system.add_memories_from_text(
    text="我叫唐锋，在燧弘华创担任执行总裁。我喜欢简洁的沟通风格。",
    source="chat"
)

# 检索记忆
result = memory_system.retrieve_memories("唐锋 燧弘华创")
print(result['context'])
```

### 示例 2: OpenClaw Session 同步

```python
from openclaw_integration import OpenClawMemorySync

sync = OpenClawMemorySync(memory_system)

# 同步 Session
result = sync.sync_session_to_memory(
    session_id="session_001",
    messages=[
        {"role": "user", "content": "我叫唐锋"},
        {"role": "user", "content": "我喜欢 Python 开发"}
    ],
    auto_extract=True
)

print(f"抽取了 {result['memories_extracted']} 条记忆")
```

### 示例 3: 获取用户画像

```python
from openclaw_integration import OpenClawTools

tools = OpenClawTools(memory_system)
profile = tools.get_user_preference()

print(f"事实：{len(profile['facts'])} 条")
print(f"偏好：{len(profile['preferences'])} 条")
```

## 3️⃣ 配置 (可选)

复制配置模板：

```bash
cp hongnao_memory_config.ini.example ~/.openclaw/workspace/hongnao_memory_config.ini
```

常用配置：

```ini

[preference_learning]
enable_preference_learning = true

```

## 4️⃣ 常见问题

### Q: 检索不到记忆？

```python
# 检查记忆库
stats = memory_system.get_stats()
print(f"总记忆数：{stats['total_memories']}")

# 降低检索阈值
result = memory_system.retrieve_memories("查询", min_score=0.1)
```

### Q: 偏好提取不准确？

```python
# 手动添加偏好
tools.add_memory(
    content="用户喜欢简洁",
    cell_type="preference",
    importance=7.0
)
```

### Q: 如何备份记忆？

```python
# 导出
json_data = memory_system.export_to_json()
with open('backup.json', 'w') as f:
    f.write(json_data)

# 导入
with open('backup.json', 'r') as f:
    memory_system.import_from_json(f.read())
```
