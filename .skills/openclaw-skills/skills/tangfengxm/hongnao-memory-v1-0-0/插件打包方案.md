## 🔧 安装流程

### 方法 1: 自动安装 (推荐)

```bash
cd /Users/tangfeng/.openclaw/workspace/hongnao_memory
python3 install_hongnao.py
```

**安装脚本执行步骤**:
1. ✅ 检查 Python 版本 (3.8+)
2. ✅ 检查依赖包 (chromadb, numpy, sentence-transformers)
3. ✅ 复制插件文件到工作区
4. ✅ 创建配置文件模板
5. ✅ 创建安装文档
6. ✅ 生成安装报告

### 方法 2: 手动安装

```bash
# 1. 安装依赖
pip3 install -r requirements.txt

# 2. 复制文件到 OpenClaw 工作区
cp *.py ~/.openclaw/workspace/hongnao_memory/

# 3. 创建配置文件
cp hongnao_memory_config.ini.example ~/.openclaw/workspace/hongnao_memory_config.ini

# 4. 编辑配置
vim ~/.openclaw/workspace/hongnao_memory_config.ini

# 5. 测试安装
python3 test_session_integration.py
```

## ⚙️ 配置文件

### 配置文件位置

```
~/.openclaw/workspace/hongnao_memory_config.ini
```

### 配置项说明

```ini
[memory]
# 记忆库路径
db_path = ~/.openclaw/workspace/hongnao_memory/hongnao.db
vector_store_path = ~/.openclaw/workspace/hongnao_memory/chroma_db

# 检索配置
retrieval_top_k = 10              # 默认检索数量
retrieval_min_score = 0.15        # 最低分数阈值

# 遗忘曲线
forgetting_half_life_days = 30.0  # 半衰期 (天)
auto_cleanup_days = 90            # 自动清理 (天)

[extraction]
# 记忆抽取配置
enable_extraction = true
enable_consolidation = true

[openclaw]
# OpenClaw 集成配置
enable_session_sync = true
auto_extract_from_sessions = true
sync_interval_minutes = 30        # 同步间隔 (分钟)

[preference_learning]
# 偏好学习配置
enable_preference_learning = true
auto_sync_to_memory = true
confidence_threshold = 0.6        # 置信度阈值
```

### 版本号规则

采用语义化版本：`MAJOR.MINOR.PATCH`

- **MAJOR**: 不兼容的 API 变更

## 📊 性能基准

### 测试环境

- CPU: Apple M2
- 内存：16GB
- Python: 3.11
- ChromaDB: 0.4.0

### 性能指标

| 操作 | 延迟 | 备注 |
|------|------|------|
| 添加记忆 (单条) | <10ms | 包含向量嵌入 |
| 检索记忆 (top_k=10) | <100ms | 混合检索 |
| Session 同步 (10 条消息) | <500ms | 包含抽取 + 巩固 |
| 偏好学习 (单条) | <50ms | 模式匹配 + 分类 |

### 优化建议

1. **批量操作**: 批量添加记忆优于单条添加
2. **索引优化**: 定期优化 ChromaDB 索引
3. **缓存**: 对频繁检索的查询使用缓存
4. **清理**: 定期应用遗忘曲线清理旧记忆
