## 🚀 快速开始

### 1. 安装插件

```bash
cd /Users/tangfeng/.openclaw/workspace/hongnao_memory
python3 install_hongnao.py
```

### 2. 安装依赖

```bash
pip3 install -r requirements.txt
```

### 3. 初始化记忆系统

```python
from memory_api import HongNaoMemorySystem, MemorySystemConfig

# 创建配置
config = MemorySystemConfig(
    enable_extraction=True,
    enable_consolidation=True,
    enable_retrieval=True,
    retrieval_top_k=10,
    retrieval_min_score=0.15
)

# 初始化系统
memory_system = HongNaoMemorySystem(config)
```

```python

    source="chat"
)

# 检索记忆
result = memory_system.retrieve_memories("用户偏好", top_k=5)
print(result['context'])
```

## 💡 核心概念

### 记忆单元 (MemCell)

记忆的基本单位，包含：

```python
{
    'id': '唯一标识',
    'content': '记忆内容',
    'memory_type': 'fact|preference|skill|emotion|constraint',
    'importance': 1-10,  # 重要性评分
    'tags': ['标签列表'],
    'source': '来源标识',
    'created_at': '创建时间',
    'updated_at': '更新时间'
}
```

### 记忆场景 (MemScene)

围绕主题/人物/任务形成的记忆集合：

```python
{
    'id': '场景 ID',
    'title': '场景标题',
    'scene_type': 'project|user|task|relationship',
    'memcell_ids': ['包含的记忆 ID 列表'],
    'created_at': '创建时间',
    'updated_at': '更新时间'
}
```

### 记忆类型

| 类型 | 说明 | 示例 |
|------|------|------|
| **FACT** (事实) | 客观事实信息 | "用户在燧弘华创工作" |
| **PREFERENCE** (偏好) | 用户喜好和习惯 | "喜欢简洁的沟通风格" |
| **SKILL** (技能) | 技能和能力 | "熟练使用 Python 和 FastAPI" |
| **EMOTION** (情感) | 情感和情绪 | "对 AI 技术充满热情" |
| **CONSTRAINT** (约束) | 限制和边界 | "不喜欢过多的废话" |

### 记忆流程

```
情景痕迹形成 → 语义巩固 → 重建式回忆

1. 抽取 (Extraction): 从文本中识别记忆
2. 巩固 (Consolidation): 压缩、关联、评分
3. 检索 (Retrieval): 向量 + 关键词混合检索
4. 更新 (Update): 处理新信息，应用遗忘曲线
```

## 🔧 API 参考

### HongNaoMemorySystem

#### 初始化

```python
from memory_api import HongNaoMemorySystem, MemorySystemConfig

config = MemorySystemConfig(
    enable_extraction=True,      # 启用抽取
    enable_consolidation=True,   # 启用巩固
    enable_retrieval=True,       # 启用检索
    enable_forgetting=True,      # 启用遗忘
    retrieval_top_k=10,          # 默认检索数量
    retrieval_min_score=0.15,    # 最低分数阈值
    forgetting_half_life_days=30.0,  # 遗忘半衰期
    auto_cleanup_days=90         # 自动清理天数
)

memory_system = HongNaoMemorySystem(config)
```

#### 核心方法

```python
# 从文本添加记忆
result = memory_system.add_memories_from_text(
    text="对话或文本内容",
    source="conversation|chat|document"
)

# 检索记忆
result = memory_system.retrieve_memories(
    query="查询文本",
    top_k=10,           # 覆盖配置
    min_score=0.15      # 覆盖配置
)

# 更新记忆 (处理新信息)
result = memory_system.update_memories(
    new_text="新的对话内容",
    source="conversation"
)

# 应用遗忘曲线
result = memory_system.apply_forgetting()

# 获取统计
stats = memory_system.get_stats()

# 导出为 JSON
json_str = memory_system.export_to_json()

# 从 JSON 导入
memory_system.import_from_json(json_str)
```

### OpenClawMemorySync

#### 初始化

```python
from openclaw_integration import OpenClawMemorySync

sync = OpenClawMemorySync(
    memory_system=memory_system,
    workspace_path="/Users/tangfeng/.openclaw/workspace"
)
```

#### 核心方法

```python
# 同步 Session 到记忆
result = sync.sync_session_to_memory(
    session_id="session_123",
    messages=[
        {"role": "user", "content": "你好"},
        {"role": "assistant", "content": "你好！有什么可以帮助你的？"}
    ],
    auto_extract=True  # 自动抽取记忆
)

# 结果
{
    'session_id': 'session_123',
    'message_count': 2,
    'memories_extracted': 1,
    'scene_created': True,
    'success': True
}
```

### OpenClawTools

#### 初始化

```python
from openclaw_integration import OpenClawTools

tools = OpenClawTools(memory_system)
```

#### 核心方法

```python

result = tools.search_memory(
    query="用户偏好",
    top_k=5
)

# 获取用户偏好画像
profile = tools.get_user_preference()
# 返回：{'facts': [], 'preferences': [], 'skills': [], 'constraints': []}

# 添加记忆
result = tools.add_memory(
    content="用户喜欢 Python",
    cell_type="preference",
    tags=["编程", "Python"],
    importance=7.0
)

# 列出记忆
result = tools.list_memories(
    cell_type="preference",  # 可选过滤
    limit=20
)

# 获取统计
stats = tools.get_stats()
```

## 📝 使用示例

### 示例 1: 完整的 Session 记忆同步

```python
from memory_api import HongNaoMemorySystem, MemorySystemConfig
from openclaw_integration import OpenClawMemorySync

# 初始化
config = MemorySystemConfig()
memory_system = HongNaoMemorySystem(config)
sync = OpenClawMemorySync(memory_system)

# 模拟 Session 对话
session_messages = [
    {"role": "user", "content": "我叫唐锋，在燧弘华创担任执行总裁"},
    {"role": "assistant", "content": "您好唐锋！很高兴为您服务"},
    {"role": "user", "content": "我喜欢简洁直接的沟通风格"},
    {"role": "user", "content": "我们公司在做 AI 平台产品"},
]

# 同步到记忆
result = sync.sync_session_to_memory(
    session_id="session_20260413_001",
    messages=session_messages,
    auto_extract=True
)

print(f"抽取了 {result['memories_extracted']} 条记忆")
```

### 示例 2: 跨 Session 记忆检索

```python
from openclaw_integration import OpenClawTools

tools = OpenClawTools(memory_system)

# Session 1: 学习用户信息
sync.sync_session_to_memory(
    session_id="session_001",
    messages=[{"role": "user", "content": "我叫唐锋，在燧弘华创工作"}]
)

# Session 2: 检索用户信息 (不同时间)
result = tools.search_memory("唐锋 燧弘华创", top_k=3)

print(f"找到 {result['count']} 条相关记忆")
for mem in result['memories']:
    print(f"  - {mem['content']}")
```

### 示例 3: 用户偏好自动学习

```python
from preference_learning import UserPreferenceLearner

learner = UserPreferenceLearner(memory_system)

# 从多轮对话中学习
conversations = [
    "我喜欢简洁直接的沟通风格，不喜欢太多废话",
    "我通常早上 8 点查看新闻和邮件",
    "我偏好使用 Python 进行开发，不太喜欢 Java",
    "我喜欢详细的报告，但回复消息要简洁",
    "我习惯在晚上 10 点前休息",
]

print("从对话中学习偏好:\n")
for conv in conversations:
    print(f"输入：{conv}")
    learned = learner.learn_from_conversation(conv, source="test")
    if learned:
        for pref in learned:
            print(f"  → [{pref.category}] {pref.content}")
    print()

# 导出报告
report = learner.export_report()
print(f"\n学习了 {report['total_count']} 条偏好")
print(f"分类统计：{list(report['by_category'].keys())}")
```

## 🎯 最佳实践

### 1. 记忆抽取

✅ **推荐**:
- 使用具体的来源标识 (`source="chat"`, `source="document"`)
- 启用自动巩固 (`enable_consolidation=True`)
- 设置合理的检索阈值 (`retrieval_min_score=0.15`)

❌ **避免**:
- 过长的文本一次性抽取 (建议 < 1000 字)
- 忽略来源标识 (不利于追溯)

### 2. 记忆检索

✅ **推荐**:
- 使用语义相关的查询词
- 调整 `top_k` 适应不同场景 (3-10)
- 结合关键词和向量检索

❌ **避免**:
- 过于宽泛的查询 ("所有记忆")
- 过高的 `top_k` (影响性能)

### 3. 偏好学习

✅ **推荐**:
- 定期从对话中提取偏好
- 设置合理的置信度阈值 (0.6+)
- 检测并解决冲突偏好

❌ **避免**:
- 盲目相信低置信度偏好
- 忽略偏好冲突

### 4. 性能优化

✅ **推荐**:
- 定期应用遗忘曲线清理旧记忆
- 使用批量操作而非单条添加
- 监控检索延迟 (<100ms)

❌ **避免**:
- 记忆库无限增长 (定期清理)
- 频繁重建索引

## 🔍 故障排查

### 问题 1: 检索不到记忆

**可能原因**:
- 记忆库为空
- 检索阈值过高
- 查询词不相关

**解决方案**:
```python
# 检查记忆库
stats = memory_system.get_stats()
print(f"总记忆数：{stats['total_memories']}")

# 降低检索阈值
result = memory_system.retrieve_memories(
    query="查询",
    min_score=0.1  # 降低阈值
)
```

### 问题 2: 偏好提取失败

**可能原因**:
- 模式匹配不准确
- 置信度过低

**解决方案**:
```python
# 检查提取的偏好
prefs = PreferencePatterns.extract_preferences(text)
print(f"提取到 {len(prefs)} 条偏好")

# 手动添加偏好
tools.add_memory(
    content="用户喜欢简洁",
    cell_type="preference",
    importance=7.0
)
```

### 问题 3: Session 同步失败

**可能原因**:
- 消息格式不正确
- 工作区路径错误

**解决方案**:
```python
# 检查消息格式
messages = [
    {"role": "user", "content": "..."},  # 正确
    {"role": "assistant", "content": "..."}
]

# 检查工作区路径
sync = OpenClawMemorySync(
    memory_system,
    workspace_path="/Users/tangfeng/.openclaw/workspace"  # 确保路径存在
)
```
