## 🔄 Step 2: 剧本 + 分镜脚本

### 思维流程（核心！）

```
┌─────────────────────────────────────────────────────────────┐
│                        剧本分析                              │
├─────────────────────────────────────────────────────────────┤
│  故事完整？→ 有开头/发展/结尾                                │
│  角色明确？→ 主角+配角性格/动机                              │
│  场景设定？→ 时间/地点/氛围                                  │
│  情感主线？→ 核心情感/主题                                   │
└─────────────────────────────────────────────────────────────┘
                        │
                        ▼
┌─────────────────────────────────────────────────────────────┐
│                     分镜脚本设计                              │
├─────────────────────────────────────────────────────────────┤
│  导演思维：每个镜头应用                                        │
│  ├─ 景别：远/全/中/近/特                                    │
│  ├─ 角度：平/仰/俯/侧                                       │
│  ├─ 焦段：14mm/35mm/85mm                                    │
│  ├─ 构图：三分/对称/负空间                                   │
│  ├─ 光影：光源/方向/效果                                     │
│  └─ 动态：运镜/动作/物理反馈                                 │
└─────────────────────────────────────────────────────────────┘
                        │
                        ▼
┌─────────────────────────────────────────────────────────────┐
│                     Clip表（镜头合并）                        │
├─────────────────────────────────────────────────────────────┤
│  相邻相似镜头 → 合并为Clip单元                               │
│  ├─ 每个Clip: 3-15秒                                         │
│  ├─ 标注参考素材                                             │
│  └─ 跨Clip追踪                                               │
└─────────────────────────────────────────────────────────────┘
                        │
                        ▼
┌─────────────────────────────────────────────────────────────┐
│                     素材清单（跨Clip）                        │
├─────────────────────────────────────────────────────────────┤
│  人物清单                                                     │
│  ├─ 女侠（已上传，涉及Clip1-11）                            │
│  └─ 锦衣卫首领（需生成，涉及Clip3-10）                       │
│                                                              │
│  场景清单                                                     │
│  ├─ 大漠-开阔沙地（需生成，涉及Clip1/11）                   │
│  └─ 大漠-龙门客栈前（需生成，涉及Clip2-10）                 │
│                                                              │
│  物品清单                                                     │
│  ├─ 棕色战马（需生成，涉及Clip2-3/11）                      │
│  └─ 佩剑（需生成，涉及Clip2-11）                            │
└─────────────────────────────────────────────────────────────┘
```

### 剧本检查清单

| 检查项 | 说明 | 通过标准 |
|--------|------|----------|
| 故事完整 | 有开头、发展、结尾 | ✓ |
| 角色明确 | 主角/配角的性格、动机清晰 | ✓ |
| 场景设定 | 时间、地点、氛围确定 | ✓ |
| 情感主线 | 核心情感/主题明确 | ✓ |

### 分镜检查清单

| 检查项 | 内容 |
|--------|------|
| 剧本符合度 | 每个场景有对应镜头 |
| 导演思维 | 景别/角度/焦段/构图/光影/动态 |
| 镜头连贯性 | 空间/动作/景别/色调衔接 |
| 动作时长 | 动作戏≤15秒 |

### Clip检查清单

| 检查项 | 内容 |
|--------|------|
| 合并合理性 | 相邻镜头是否适合合并 |
| 时长控制 | 每个Clip 3-15秒 |
| 素材标注 | 每个Clip有参考素材 |

### 素材清单检查

| 检查项 | 内容 |
|--------|------|
| 人物清单 | 所有角色已标注 |
| 场景清单 | 所有场景已标注 |
| 物品清单 | 所有物品已标注 |

## 🔄 Step 5: 首帧图片生成

### 思维流程

```
分镜脚本（逐镜头）
        │
        ▼
┌─────────────────────────────────────────────────────────────┐
│  每个镜头生成首帧图片                                         │
├─────────────────────────────────────────────────────────────┤
│  1. 上传参考图（角色+场景+物品）                            │
│  2. 编写提示词（模块化公式）                                 │
│     [基础风格] + [分镜内容] + [动态控制]                     │
│  3. 设置参数（图片4.0/21:9/2K）                             │
│  4. 生成图片                                                 │
│  5. 检查一致性                                               │
└─────────────────────────────────────────────────────────────┘
```

### 提示词模块化

```
[基础风格模块]
(8k, cinematic, 35mm lens, film grain, 1970s Shaw Brothers style)
        +
[分镜内容模块]
(A warrior in white robe, desert sunset, Three Jinwei appear)
        +
[动态控制模块]
(Camera push in, sand particles floating)
```

### 首帧检查清单

| 检查项 | 验证 |
|--------|------|
| 静态描述 | 景别/角度/色调明确 |
| 符合分镜 | 与分镜脚本一致 |
| 人物一致性 | 脸型/发型/服装与参考图一致 |
| 场景一致性 | 色调/物件/光线与场景底图一致 |
| 运镜预留 | 为图生视频预留运动空间 |

## 🔄 Step 6: 图生视频

### 思维流程

```
首帧图片
        │
        ▼
┌─────────────────────────────────────────────────────────────┐
│  每个镜头生成视频片段                                         │
├─────────────────────────────────────────────────────────────┤
│  1. 上传首帧图片                                             │
│  2. 编写动作提示词                                          │
│     ├─ 运镜：push forward/pull backward                    │
│     ├─ 动作：具体动作描述                                     │
│     └─ 物理反馈：风/沙/粒子                                  │
│  3. 设置参数（视频3.0 Fast/5s/运动3-4）                      │
│  4. 生成视频                                                 │
│  5. 检查连贯性                                               │
└─────────────────────────────────────────────────────────────┘
```

### 运镜方向表

| 分镜动态 | 即梦运镜关键词 |
|----------|----------------|
| 推近 | push forward |
| 拉远 | pull backward |
| 左移 | move left |
| 右移 | move right |
| 上升 | move up |
| 下降 | move down |
| 旋转 | rotate |
| 跟随 | follow |

### 视频检查清单

| 检查项 | 验证 |
|--------|------|
| 动态描述 | 运镜+动作+物理反馈 |
| 符合分镜 | 运镜方向与分镜设计一致 |
| 人物不变 | 人物无明显变形 |
| 色调统一 | 整体色调一致 |
| 首帧对齐 | 视频首帧与图片一致 |

## 🔄 Step 7: 音频/剪辑

### 思维流程

```
所有视频片段
        │
        ▼
┌─────────────────────────────────────────────────────────────┐
│  剪辑流程                                                     │
├─────────────────────────────────────────────────────────────┤
│  1. 导入素材 → 排列顺序                                      │
│  2. 粗剪 + 精剪                                              │
│  3. 调色（统一色调）                                          │
│  4. 添加音效（环境音+动作音）                                 │
│  5. 添加BGM（符合风格）                                      │
│  6. 配音（对话+独白）                                        │
│  7. 混音 → 导出                                              │
└─────────────────────────────────────────────────────────────┘
```

### 剪辑检查清单

| 检查项 | 验证 |
|--------|------|
| 结构完整 | 有开头/发展/结尾 |
| 节奏合适 | 快慢交替 |
| 音画同步 | 音频对齐视频 |
| 色调统一 | 整体调性一致 |
| 无技术问题 | 无卡顿/黑屏/杂音 |
