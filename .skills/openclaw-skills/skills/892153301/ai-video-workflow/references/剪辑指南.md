# 剪辑完全指南

## 剪辑流程

```
1. 导入素材
2. 排列顺序
3. 粗剪
4. 精剪
5. 调色
6. 加音效
7. 加BGM
8. 混音
```

## 剪辑检查

### 结构完整

- 开头：吸引观众
- 中间：叙述完整
- 结尾：升华/悬念

### 节奏合适

- 动作场景：快节奏
- 情感场景：慢节奏
- 转场：自然过渡

### 音画同步

- 动作音画同步
- 音乐节奏匹配
- 配音口型对齐

### 色调统一

- 整体调性一致
- 场景之间过渡自然
- 避免跳色

### 无技术问题

- 无卡顿
- 无黑场
- 无杂音

## 音画同步技巧

### BPM对齐

根据BGM的BPM调整剪辑点：
- 重音在剪辑点
- 动作在节拍上

### 动作匹配

- 剪辑点选择动作高潮
- 视线方向一致
- 运动方向一致

## 检查清单

- [ ] 音画同步
- [ ] 色调统一
