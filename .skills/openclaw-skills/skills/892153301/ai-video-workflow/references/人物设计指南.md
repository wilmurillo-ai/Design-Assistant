## 参考图生成

### 必须生成的视角

| 视角 | 用途 |
|------|------|
| 正面 | 人物正面形象 |
| 侧面 | 轮廓特征 |
| 背面 | 背部特征 |
| 表情（2-3种） | 喜怒哀乐 |

### 生成技巧

1. 固定Seed值
2. 使用--iw垫图
3. 记录满意的图片参数

## 一致性保障方法

### 1. Seed固定

```
1. 生成满意角色
2. 记录Seed值
3. 后续使用 --seed [数字]
4. 测试一致性
```

### 2. 视觉锚点

每次包含标志性特征：
- 特定首饰
- 伤痣/胎记
- 特色配色

### 3. 服装描述模板化

```
固定：[颜色Hex] [材质] [款式]
示例：#F5F5F5 真丝 白衣
```
