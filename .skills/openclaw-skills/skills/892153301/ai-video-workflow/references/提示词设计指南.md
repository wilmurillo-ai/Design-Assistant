# 提示词设计完全指南

## 模块化结构

```
[基础风格模块] + [分镜内容模块] + [动态控制模块]
     ↓                 ↓                  ↓
   不变              每次替换            每次替换
```

## 完整结构

```
[主体] + [动作] + [场景] + [光影] + [色调] + [构图] + [运镜] + [风格] + [参数]
```

## 基础风格模块（不变）

```
(8k, cinematic lighting, film grain, 35mm lens, hyper-realistic, detailed)
```

## 分镜内容模块（每次变）

```
A warrior in white robe standing in bamboo forest
```

## 动态控制模块（每次变）

```
Camera slow push in, leaves gently swaying
```

## 组合示例

```
(8k, cinematic lighting, film grain, 35mm lens) + 
(A warrior in white robe) + 
(bamboo forest, morning mist) + 
(soft volumetric light) + 
(jade green tones) + 
(rule of thirds) + 
(camera push in) + 
(gentle breeze, leaves swaying)
```

## 参数设置

### Midjourney参数

| 参数 | 用途 | 示例 |
|------|------|------|
| --ar | 长宽比 | --ar 16:9 |
| --v | 模型版本 | --v 6 |
| --stylize | 风格化 | --stylize 400 |
| --seed | 种子 | --seed 12345 |
| --iw | 垫图权重 | --iw 0.5 |
| --no | 负面 | --no text |

### 常用比例

| 比例 | 用途 |
|------|------|
| 16:9 | 电影/YouTube |
| 9:16 | 抖音/短视频 |
| 1:1 | Instagram |
| 4:3 | 传统电视 |
