# 一致性完全指南

## 一致性三大目标

1. **人物一致性** - 脸型、服装、道具
2. **场景一致性** - 色调、物件、光线
3. **风格一致性** - 质感、构图、光影

## 人物一致性

### 必须保持一致的要素

| 要素 | 检查方法 |
|------|----------|
| 脸型 | 对比各帧截图 |
| 发型 | 检查轮廓 |
| 眼睛 | 眼型/眼神 |
| 服装 | 颜色(Hex)/款式/材质 |
| 道具 | 是否缺失 |
| 肤色 | 无明显变化 |

### 保障方法

1. **固定Seed**
   ```
   1. 生成满意角色
   2. 记录Seed值
   3. 后续使用 --seed [数字]
   ```

2. **视觉锚点**
   每次包含标志性特征：首饰、伤痣、特色配色

3. **参考图垫图**
   生成：正面/侧面/背面/表情
   保存为参考图，视频生成时垫图

4. **服装描述模板化**
   ```
   固定：[颜色Hex] [材质] [款式]
   示例：#F5F5F5 真丝 白衣
   ```

## 场景一致性

### 必须保持一致的要素

| 要素 | 检查方法 |
|------|----------|
| 色调 | 截图对比 |
| 物件位置 | 检查道具摆放 |
| 光线方向 | 对比光源方向 |
| 天气氛围 | 无突变 |

## 风格一致性

### 必须保持一致的要素

| 要素 | 检查方法 |
|------|----------|
| 质感 | 画面质感是否一致 |
| 色调 | 整体色调是否协调 |
| 构图 | 构图方式是否一致 |

## 常见问题解决

### 变脸问题

| 原因 | 解决 |
|------|------|
| 没固定Seed | 固定Seed |
| 描述不一致 | 用统一预设 |
| 角度差异大 | 固定角度范围 |

### 服装问题

| 原因 | 解决 |
|------|------|
| 颜色变化 | 用Hex码 |
| 款式变化 | 固定描述 |
| 配饰丢失 | 强调配饰 |

### 色调问题

| 原因 | 解决 |
|------|------|
| 光线不同 | 统一光线关键词 |
| 风格变化 | 固定风格模块 |
| 饱和度变 | 明确饱和度 |
