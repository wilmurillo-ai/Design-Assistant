# 导演思维公式

## 六大要素

每个镜头必须包含：

| 要素 | 选项 |
|------|------|
| 景别 | 远/全/中/近/特 |
| 角度 | 平/仰/俯/侧/荷兰角 |
| 焦段 | 14mm/35mm/85mm/135mm |
| 构图 | 三分/对称/负空间/框架 |
| 光影 | 光源/方向/效果 |
| 动态 | 运镜/动作/物理反馈 |

## 四大高级公式

### 公式一：视点叙事

```
[视角身份] + [前景遮挡/框架] + [特定焦段]
示例：POV shot, framing through bamboo, spy lens
```

### 公式二：环境压抑

```
[极端远景] + [高俯角] + [负空间]
示例：Extreme long shot, high angle, subject as silhouette
```

### 公式三：时间停滞

```
[微距/特写] + [细节捕捉] + [高对比]
示例：Macro close-up, frozen motion, high shutter speed
```

### 公式四：心理外化

```
[双色温] + [侧逆光] + [体积光]
示例：Warm candlelight vs cold moonlight, rim lighting
```

## 运镜参数

| 运镜 | 英文 | 效果 |
|------|------|------|
| 推近 | push forward | 靠近主体 |
| 拉远 | pull backward | 远离主体 |
| 左移 | move left | 向左移动 |
| 右移 | move right | 向右移动 |
| 上升 | crane up | 向上升起 |
| 下降 | crane down | 向下降落 |
| 环绕 | orbit | 环绕主体 |
| 跟随 | follow | 跟踪主体 |

## 动态强度

| 等级 | 关键词 | 场景 |
|------|--------|------|
| 静态 | static, still | 对话 |
| 微动 | subtle, gentle | 抒情 |
| 轻动 | light, moderate | 叙事 |
| 强动 | dynamic, intense | 动作 |
