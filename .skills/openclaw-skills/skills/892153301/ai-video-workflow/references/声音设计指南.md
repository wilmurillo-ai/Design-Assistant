# 声音设计完全指南

## 声音三要素

| 要素 | 内容 |
|------|------|
| BGM | 背景音乐 |
| 音效 | 环境音/动作音 |
| 配音 | 人声 |

## BGM设计

### 模板

```
## BGM
- 类型：
- BPM：
- 乐器：
- 情绪：
```

### BPM参考

| 场景 | BPM |
|------|-----|
| 紧张 | 140-180 |
| 动作 | 120-140 |
| 叙事 | 80-100 |
| 抒情 | 60-80 |

### 情绪类型

- 紧张、悬疑
- 温暖、感人
- 史诗、宏大
- 轻松、愉快
- 悲伤、忧愁

## 音效设计

### 模板

```
## 音效
- 环境音：
- 动作音：
- 物件音：
```

### 类型

| 类型 | 示例 |
|------|------|
| 环境音 | 风声、水滴、鸟叫 |
| 动作音 | 脚步、格斗、呼吸 |
| 物件音 | 门响、枪声、钟声 |

## 配音设计

### 模板

```
## 配音
- 音色：
- 语速：
- 情感：
- 停顿点：
```

### 风格

| 风格 | 特点 |
|------|------|
| 冷淡 | 低沉、平静、少起伏 |
| 激昂 | 高亢、情绪饱满 |
| 温暖 | 柔和、亲切 |
| 神秘 | 轻声、气声 |
