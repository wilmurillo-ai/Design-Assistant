# AI视频制作工作流

> 版本：v5.0 精简版
> 基于：v4.2 完整版

## 触发场景

当用户有以下需求时自动执行：
- "帮我制作一个AI视频"
- "怎么做AI视频"
- "从剧本开始制作视频"
- "AI视频工作流"

## 核心架构

| 角色 | 功能 |
|------|------|
| **导演** | 审美准入，设计叙事，掌控情绪，决策风格 |
| **技术制片** | 提供提示词模板、监控进度、执行生成 |

## 七步制作流程

| 步骤 | 内容 | 输出 |
|------|------|------|
| 1 | 剧本/想法分析 | 主题确定 |
| 2 | 风格设计 | 视觉风格文档 |
| 3 | 分镜设计 | 分镜脚本（景别/角度/运镜/台词） |
| 4 | 生图 | 关键帧图片 |
| 5 | 图生视频 | AI生成视频片段 |
| 6 | 剪辑 | 剪映Pro剪辑成片 |
| 7 | 配音/音效 | 完整作品 |

## 导演四维度

1. **视听语言**：焦距塑造空间，光影叙事心理
2. **导演公式**：角色基准图 + 分镜脚本
3. **美学风格**：参考经典（胡金铨/塔可夫斯基等）
4. **剪辑思维**：Match Cut / 节奏控制
