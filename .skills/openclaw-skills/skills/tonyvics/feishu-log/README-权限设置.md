# 飞书日志技能 - 权限设置指南

## ⚠️ 重要：首次使用需要手动设置文件夹权限

由于飞书 API 限制，文件夹权限需要手动设置一次，之后所有子文件夹会自动继承权限。

---

## 📋 设置步骤

### 方法 1: 通过飞书客户端（推荐）

1. **打开飞书云盘**
   - 访问：https://feishu.cn/drive

2. **找到"日志"文件夹**
   - 在根目录下找到刚创建的"日志"文件夹

3. **右键点击 → 分享**

4. **添加协作者**
   - 搜索你的名字或账号
   - 权限设置为：**可管理** (full_access)
   - ✅ 勾选"**同时共享子文件夹**"

5. **完成**
   - 点击"分享"或"确定"
   - 现在所有子文件夹（2026 年/03 月/04 日/log/）都可以访问了！

---

### 方法 2: 通过飞书链接直接访问

如果文件夹已经创建，可以直接访问：

```
https://feishu.cn/drive/folder/[文件夹 token]
```

例如：
- 日志文件夹：https://feishu.cn/drive/folder/MDjAfBJorlqu14dWxBocy7MnnId

---

## ✅ 验证权限

设置完成后，验证是否可以：
- [ ] 查看文件夹
- [ ] 打开子文件夹
- [ ] 查看文档
- [ ] 编辑文档
- [ ] 删除文档

---

## 📝 说明

### 为什么需要手动设置？

飞书的文件夹权限 API 有特殊限制：
- 文档权限 API ✅ 工作正常
- 文件夹权限 API ❌ 需要特殊权限

### 设置一次就够了

只需要手动共享**根文件夹（"日志"）**一次：
- 所有子文件夹自动继承权限
- 之后创建的所有文档都有正确权限

### 文档权限自动设置

技能会自动为每个创建的文档添加你的协作者权限（full_access），无需手动设置。

## 🆘 遇到问题？

如果设置后仍然无法访问文件夹：
1. 检查是否勾选了"同时共享子文件夹"
2. 确认权限设置为"可管理"
3. 尝试退出飞书重新登录
4. 联系飞书技术支持
