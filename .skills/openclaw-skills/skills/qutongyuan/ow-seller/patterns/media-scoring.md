# 商品展示与评分标准

## 一、商品图片规范

### 图片数量与分值

| 图片数量 | 得分 | 说明 |
|----------|------|------|
| 0 张 | 0 分 | 无图片展示 |
| 1 张 | 3.3 分 | 基础展示 |
| 2 张 | 6.6 分 | 较好展示 |
| 3 张 | 10 分 | 完整展示 |

### 图片内容建议

```
推荐组合：
├── 图片1：商品正面/整体图（必须）
├── 图片2：商品细节/特写图（推荐）
└── 图片3：配件/包装/证书图（推荐）
```

### 图片质量要求

| 要求 | 标准 |
|------|------|
| **分辨率** | ≥ 800×800 像素 |
| **格式** | JPG, PNG, WEBP |
| **大小** | 单张 ≤ 5MB |
| **清晰度** | 商品细节清晰可见 |
| **真实性** | 实拍图，不得过度美化 |

---

## 二、商品视频规范

### 视频评分

| 情况 | 得分 |
|------|------|
| 无视频 | 0 分 |
| 有视频 | 10 分 |

### 视频内容建议

```
推荐内容：
├── 商品全方位展示（360°展示）
├── 功能演示（如适用）
├── 使用场景展示
└── 真实状态确认

时长：≤ 30 秒
格式：MP4, MOV, WEBM
大小：≤ 50MB
```

### 视频禁止内容

- ❌ 与商品无关的内容
- ❌ 虚假宣传内容
- ❌ 侵权内容
- ❌ 过度美化/特效

## 六、示例计算

### 投标案例

```
供应商 A 的投标：
- 价格：¥2,800（预算¥5000，最低价¥2,500）
- 真品证明：营业执照+代理权+授权书 = 14分
- 到货时间：3天（最快）
- 商品展示：3张图+1视频 = 20分
- 商家信誉：50笔成交，98%好评，无纠纷 = 10分
- 交易记录：50笔成交，98%好评 = 10分

计算：
- 价格分 = 35 × (5000-2800)/(5000-2500) = 30.8分
- 真品分 = 5+5+4 = 14分
- 到货分 = 5分（最快）
- 媒体分 = 10+10 = 20分
- 信誉分 = 5+5 = 10分
- 交易分 = 5+3 = 8分

总分 = 30.8+14+5+20+10+8 = 87.8分 → A级
```
