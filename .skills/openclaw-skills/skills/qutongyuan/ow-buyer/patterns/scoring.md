# 评标规则详解

## 四维度评分体系

### 维度1：价格 (权重 50%)

**评分公式：**
```
价格得分 = (最低价 / 该投标价) × 100
```

**示例：**
- 最低投标价：¥2,000
- 投标A价格：¥2,000 → 得分 = 100
- 投标B价格：¥2,500 → 得分 = 80
- 投标C价格：¥3,000 → 得分 = 66.7

**加权得分 = 价格得分 × 0.5**

---

### 维度2：真品证明 (权重 20%)

**评分标准：**

| 证明文件 | 分值 |
|----------|------|
| 企业营业执照 | 20分 |
| 产品代理权证明 | 20分 |
| 原厂授权书 | 15分 |
| 产品质检报告 | 15分 |
| 进口报关单（进口商品） | 15分 |
| 门店照片/实体证明 | 10分 |
| 其他真实性证明 | 5分/项 |

**上限：100分**

**计算：**
```
真品得分 = min(各项得分之和, 100)
加权得分 = 真品得分 × 0.2
```

**示例：**
- 供应商A：营业执照+代理权+授权书 = 55分 → 加权 11分
- 供应商B：营业执照+代理权+质检报告+报关单 = 70分 → 加权 14分
- 供应商C：仅营业执照 = 20分 → 加权 4分

---

### 维度3：到货时间 (权重 10%)

**评分公式：**
```
时间得分 = (最短时间 / 该投标时间) × 100
```

**示例：**
- 最短到货：2天
- 投标A：2天 → 得分 = 100
- 投标B：5天 → 得分 = 40
- 投标C：7天 → 得分 = 28.6

**加权得分 = 时间得分 × 0.1**

---

### 维度4：商家信誉 (权重 20%)

**评分因素：**

| 因素 | 计算 |
|------|------|
| 过往成交数 | 每次成交 +2分，上限 50分 |
| 成交好评率 | 好评率 × 30 |
| 纠纷记录 | 每次纠纷 -5分 |
| 投诉记录 | 每次投诉 -10分 |
| 平台认证 | 认证商家 +10分 |

**计算：**
```
信誉得分 = min(max(成交分 + 好评分 - 纠纷分 - 投诉分 + 认证分, 0), 100)
加权得分 = 信誉得分 × 0.2
```

**示例：**
- 供应商A：成交50次(50分) + 好评率95%(28.5分) + 无纠纷 + 平台认证(10分) = 88.5分 → 加权 17.7分
- 供应商B：成交20次(40分) + 好评率90%(27分) + 1次纠纷(-5分) = 62分 → 加权 12.4分
- 供应商C：成交5次(10分) + 好评率80%(24分) + 新商家 = 34分 → 加权 6.8分

---

## 综合得分计算

```
综合得分 = 价格加权 + 真品加权 + 时间加权 + 信誉加权
         = 价格得分×0.5 + 真品得分×0.2 + 时间得分×0.1 + 信誉得分×0.2
```

**完整示例：**

| 供应商 | 价格 | 真品 | 时间 | 信誉 | 综合 |
|--------|------|------|------|------|------|
| A | ¥2,800 | 55分 | 3天 | 88.5分 | 42.0+11+10+17.7=**80.7** |
| B | ¥3,200 | 70分 | 5天 | 62分 | 35+14+4+12.4=**65.4** |
| C | ¥2,500 | 20分 | 7天 | 34分 | 40+4+2.86+6.8=**53.66** |

假设最低价为 ¥2,500，最短时间为 2天。

## 特殊情况处理

### 价格异常低
- 价格低于市场均价50% → 标记 ⚠️ 需人工审核
- 可能假货风险 → 提高真品证明权重

### 无真品证明
- 真品得分为0 → 综合得分上限限制为60分
- 食品/药品类 → 必须有质检报告，否则不接受投标

### 新商家（无成交记录）
- 信誉得分按50分基准计算
- 提示用户：新商家风险，建议选择有成交记录的供应商

### 纠纷记录过多
- 纠纷≥3次 → 标记 ⚠️ 高风险
- 纠纷≥5次 → 自动排除，不参与评标
