# 交易流程说明

## 交易方式

**本技能不生成支付链接。** 所有交易通过卖家提供的外部店铺完成。

---

## 交易流程

### 1. 买家确认中标

买家确认中标供应商后，获得卖家的店铺链接。

### 2. 前往外部店铺

买家点击店铺链接，前往卖家店铺：
- 淘宝/天猫
- 亚马逊
- 京东
- 拼多多
- 独立站

### 3. 店铺内完成交易

在卖家店铺内完成：
- 下单
- 付款（使用店铺支持的支付方式）
- 物流跟踪
- 售后服务

---

## 支持的店铺平台

| 平台 | 支付方式 |
|------|----------|
| 淘宝/天猫 | 支付宝 |
| 京东 | 微信支付/京东支付 |
| 亚马逊 | 信用卡/PayPal |
| 拼多多 | 微信支付 |

---

## 优势

| 优势 | 说明 |
|------|------|
| ✅ 平台担保 | 店铺平台提供交易保障 |
| ✅ 售后完善 | 平台客服处理纠纷 |
| ✅ 支付安全 | 无需在本系统输入支付信息 |
| ✅ 无需资质 | 不需要支付牌照 |

---

## 记录交易状态

交易完成后，可记录交易状态供信用系统使用：

```json
{
  "tx_id": "TX-20260407-001",
  "req_id": "REQ-xxx",
  "bid_id": "BID-xxx",
  "status": "completed",
  "shop_platform": "淘宝",
  "shop_link": "https://shop123.taobao.com",
  "completed_at": "2026-04-07T12:00:00Z"
}
```

**注意：** 不存储支付凭证、银行卡号等敏感支付信息。
