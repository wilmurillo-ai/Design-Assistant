## 概述

大恩营销策划七步法是一套完整的房地产项目营销策略制定方法论，从问题界定到落地执行，七步完成系统性营销方案。

**核心原则：每个项目都有其独特性，七步法提供方法论框架，具体策略需根据项目实际情况灵活调整。**

---

## Step 1：核心营销课题

### 目的
明确项目面临的核心挑战，为后续策略制定提供方向。

### 课题界定方法

**第一步：诊断现状**
- 项目当前面临的最大障碍是什么？
- 与竞品相比的差距在哪里？
- 客户认知存在什么误区？

**第二步：归纳课题**
将诊断结果归纳为1-3个核心课题，每个课题包含：
- 课题描述
- 挑战本质
- 解决方向

### 常见课题类型

| 课题类型 | 典型问题 | 解决方向 |
|----------|----------|----------|
| 认知课题 | 区域认知度低、品牌势能不足 | 影响力构建 |
| 转化课题 | 价格抗性、竞品分流 | 必买逻辑体系化 |
| 差异化课题 | 产品同质化、价值模糊 | 核心竞争力提炼 |
| 信任课题 | 品牌新入、兑现能力质疑 | 信任背书构建 |

---

## Step 2：核心操盘思路

### 目的
制定总体策略框架，明确执行路径。

### 策略制定原则

**原则一：课题导向**
- 策略必须回应Step 1界定的核心课题
- 每个策略动作都要有明确目的

**原则二：资源匹配**
- 策略要与项目资源匹配
- 传播调性要与项目定位一致

**原则三：阶段递进**
- 先建立认知，再深化理解，最后促成转化
- 传播节奏要有层次感

### 策略框架模板

```
【总体策略】
[一句话概括核心策略方向]

【执行路径】
第一步：[阶段目标]
• [具体动作]

第二步：[阶段目标]
• [具体动作]

【传播调性】
[关键词1] [关键词2] [关键词3]
```

## Step 3：市场分析

### 目的
全面分析地段价值、竞品格局、目标客户。

### 一、地段价值分析

**分析框架：**

| 层面 | 分析维度 | 关键问题 |
|------|----------|----------|
| 宏观层面 | 城市/区域定位 | 项目在城市发展中的角色？ |
| 中观层面 | 板块价值 | 板块的核心价值是什么？ |
| 微观层面 | 配套资源 | 周边有哪些核心资源？ |

### 二、竞品格局分析

**竞品选择原则：**
- 选择直接竞品（同区域、同价位）
- 选择间接竞品（不同区域、同价位）
- 选择替代产品（不同产品类型）

**分析维度：**
- 位置对比
- 产品对比
- 配套对比
- 价格对比
- 品牌对比

### 三、目标客户画像

| 维度 | 分析要点 |
|------|----------|
| 基础画像 | 年龄、职业、收入、家庭结构 |
| 购买理由 | 为什么买？核心驱动因素、价值诉求 |
| 购买障碍 | 为什么不买？顾虑点、抗性点、决策阻力 |
| 需求痛点 | 核心诉求、购买动机 |
| 决策路径 | 信息获取渠道、决策因素 |
| 触媒习惯 | 媒体偏好、社交圈层 |

## Step 4：产品优劣势分析

### 目的
识别项目核心优势，建立竞争壁垒。

### SWOT分析

| 优势 Strengths | 劣势 Weaknesses |
|----------------|-----------------|
| [品牌优势] | [价格不确定性] |
| [产品优势] | [区域认知度] |
| [配套优势] | [配套成熟度] |

| 机会 Opportunities | 威胁 Threats |
|--------------------|--------------|
| [政策红利] | [竞品分流] |
| [市场窗口] | [市场波动] |

### 核心卖点提炼

**提炼原则：**
1. 选择真正有差异化的卖点
2. 卖点要可感知、可验证
3. 卖点要与客户需求匹配

**输出格式：**
```
核心卖点一：[卖点名称]
• [支撑点1]
• [支撑点2]

核心卖点二：[卖点名称]
• [支撑点1]
• [支撑点2]
```

## Step 5：项目定位

### 目的
确立项目市场地位，提炼核心传播口号。

### 定位构建

**定位公式：**
```
项目定位 = [区域/城市地位] + [品牌系/产品系] + [产品类型/生活方式]
```

**定位原则：**
- 要有差异化
- 要有支撑点
- 要能延展传播

### SLOGAN创作

**创作原则：**

| 原则 | 说明 |
|------|------|
| 相关性 | 与项目价值相关 |
| 差异性 | 与竞品区分 |
| 记忆点 | 朗朗上口 |
| 延展性 | 可延展传播 |

**输出结构：**
```
【SLOGAN】[主口号]

【释义】
[时代背景/价值内涵/情感共鸣]
```

## Step 6：故事线

### 目的
构建项目叙事框架，增强传播感染力。

### 叙事结构

**三幕式结构：**

**第一幕：宏观层面**
- 主题：时代/城市发展背景
- 内容：项目所在区域的战略意义
- 调性：宏大、前瞻

**第二幕：中观层面**
- 主题：区域/地段价值
- 内容：地段优势、配套资源
- 调性：权威、确信

**第三幕：微观层面**
- 主题：产品/生活方式
- 内容：产品价值、生活场景
- 调性：真诚、极致

### 输出模板

```
【叙事主题】

第一幕：[主题]
[内容...]

第二幕：[主题]
[内容...]

第三幕：[主题]
[内容...]
```

## Step 7：传播计划

### 目的
制定阶段性传播策略，有序推进市场引爆。

### 传播节奏设计

**三阶段模型：**

| 阶段 | 目标 | 关键动作 | 传播调性 |
|------|------|----------|----------|
| 阶段一 | 建立认知 | 亮相事件、品牌发声 | 宏大、引领 |
| 阶段二 | 深化理解 | 产品解读、样板展示 | 真诚、高级 |
| 阶段三 | 促成转化 | 稀缺营造、开盘活动 | 紧迫、稀缺 |

### 传播动作规划

**每个传播动作包含：**
1. 目的：要达成什么效果
2. 主题：传播的核心信息
3. 形式：具体的执行方式
4. 渠道：传播媒介选择
5. 预期效果：可衡量的指标

### 排期模板

## 重要提示

### ⚠️ 避免误区

1. **生搬硬套** - 每个项目都有独特性，需灵活调整
2. **跳过步骤** - 七步法环环相扣，跳过会导致逻辑断层
3. **脱离实际** - 策略要与项目资源和市场现状匹配
4. **忽视执行** - 再好的策略也需要扎实的落地执行

### ✅ 成功要点

1. **深入调研** - 充分了解项目、市场、客户
2. **逻辑闭环** - 每个Step的输出要能支撑下一步
3. **保持弹性** - 根据实际情况灵活调整策略
4. **聚焦核心** - 抓住主要矛盾，不要面面俱到
