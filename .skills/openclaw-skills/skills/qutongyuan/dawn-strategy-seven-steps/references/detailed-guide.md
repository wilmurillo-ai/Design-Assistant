# 七步法详细操作指南

## Step 1：核心营销课题 - 详细操作

### 课题界定的三个维度

**1. 市场维度**
- 区域认知度如何？
- 板块价值是否清晰？
- 品牌势能是否足够？

**2. 产品维度**
- 产品是否有代际优势？
- 差异化是否明显？
- 价值点是否可感知？

**3. 客户维度**
- 客户画像是否清晰？
- 购买动机是否明确？
- 决策路径是否顺畅？

### 课题界定方法

```
方法一：痛点倒推法
客户痛点 → 核心障碍 → 营销课题

方法二：竞品对标法
竞品优势 → 我方劣势 → 破局方向

方法三：价值透视法
项目价值 → 市场认知 → 传递课题
```

## Step 2：核心操盘思路 - 详细操作

### 策略制定的四个原则

**原则一：先占位后渗透**
- 先建立城市级认知
- 再渗透地缘客户

**原则二：先高端后大众**
- 先影响意见领袖
- 再扩散大众市场

**原则三：先价值后产品**
- 先传递核心价值
- 再展示产品细节

**原则四：先声量后转化**
- 先建立传播声量
- 再推动销售转化

### 必买理由的构建逻辑

```
必买理由 = 功能价值 + 情感价值 + 身份价值

功能价值：解决了什么问题
情感价值：满足了什么情感
身份价值：代表了什么身份
```

## 常见问题与解答

**Q: 七步法适用于所有项目吗？**
A: 七步法是通用方法论框架，但每个项目需要根据实际情况灵活调整。不要生搬硬套，要根据项目特点制定具体策略。

**Q: 七个步骤必须按顺序执行吗？**
A: 建议按顺序，但可根据项目情况灵活调整。核心是确保每个步骤都有明确输出，逻辑闭环。

**Q: 方案制定需要多长时间？**
A: 完整方案通常需要2-4周，包括调研、分析、策略制定、方案撰写。

**Q: 如何判断方案是否成功？**
A: 关键指标包括：市场认知度、客户到访量、销售转化率、品牌美誉度。

**Q: 可以跳过某些步骤吗？**
A: 不建议跳过。七步法环环相扣，跳过会导致逻辑断层。但如果项目已有明确的某个步骤结论，可以快速通过。
