## Step 1：核心课题

### 目的
明确项目面临的核心挑战，为后续策略制定提供方向。

### 课题界定方法

**第一步：诊断现状**
- 项目当前面临的最大障碍是什么？
- 与竞品相比的差距在哪里？
- 客户认知存在什么误区？

**第二步：归纳课题**
将诊断结果归纳为1-3个核心课题，每个课题包含：
- 课题描述
- 挑战本质
- 解决方向

### 常见课题类型

| 课题类型 | 典型问题 | 解决方向 |
|----------|----------|----------|
| 认知课题 | 区域认知度低、品牌势能不足 | 影响力构建 |
| 转化课题 | 价格抗性、竞品分流 | 必买逻辑体系化 |
| 差异化课题 | 产品同质化、价值模糊 | 核心竞争力提炼 |
| 信任课题 | 品牌新入、兑现能力质疑 | 信任背书构建 |

### 输出格式

```
【课题界定】

课题一：[课题名称]
• 挑战描述：______
• 解决方向：______

课题二：[课题名称]
• 挑战描述：______
• 解决方向：______
```

## Step 2：解决方案

### 目的
制定策略方案，明确项目定位和核心卖点。

### 解决方案框架

**一、策略方向**
- 总体策略一句话概括
- 核心执行路径
- 传播调性关键词

**二、市场定位**
- 地段/区域价值定位
- 目标客群定位
- 竞争差异化定位

**三、核心卖点**
- 核心卖点一：[卖点名称] + 支撑点
- 核心卖点二：[卖点名称] + 支撑点
- 核心卖点三：[卖点名称] + 支撑点

**四、项目定位**
- 项目定位语
- SLOGAN + 释义

### 目标客户画像

| 维度 | 分析要点 |
|------|----------|
| 基础画像 | 年龄、职业、收入、家庭结构 |
| 购买理由 | 为什么买？核心驱动因素 |
| 购买障碍 | 为什么不买？顾虑点、抗性点 |

### 输出格式

```

执行路径：______
传播调性：______、______、______

【市场定位】
地段定位：______
客群定位：______
差异化定位：______

【核心卖点】
核心卖点一：______
• 支撑点：______

核心卖点二：______
• 支撑点：______

【项目定位】
定位语：______
SLOGAN：______
释义：______
```

## Step 3：传播规划

### 目的
制定阶段性传播策略，有序推进市场引爆。

### 传播节奏设计

**三阶段模型：**

| 阶段 | 目标 | 关键动作 | 传播调性 |
|------|------|----------|----------|
| 阶段一 | 建立认知 | 亮相事件、品牌发声 | 宏大、引领 |
| 阶段二 | 深化理解 | 产品解读、样板展示 | 真诚、高级 |
| 阶段三 | 促成转化 | 稀缺营造、开盘活动 | 紧迫、稀缺 |

### 传播动作规划

**每个传播动作包含：**
1. 目的：要达成什么效果
2. 主题：传播的核心信息
3. 形式：具体的执行方式
4. 渠道：传播媒介选择
5. 预期效果：可衡量的指标

### 输出格式

```

目标：______
动作1：______
动作2：______

阶段二：[时间范围] - [阶段名称]
目标：______
动作1：______
动作2：______

阶段三：[时间范围] - [阶段名称]
目标：______
动作1：______
动作2：______
```

## 三步法 vs 七步法

| 对比项 | 三步法 | 七步法 |
|--------|--------|--------|
| 适用场景 | 专题策划、快速提案 | 完整项目策划 |
| 深度 | 聚焦核心 | 全面深入 |
| 灵活性 | 高 | 中 |

---

## 重要提示

### ⚠️ 避免误区

1. **课题界定不清** → 策略方向模糊
2. **定位过于宏大** → 与项目实际脱节
3. **传播节奏混乱** → 资源分散低效

### ✅ 成功要点

1. **聚焦核心** - 三步法贵在精简，抓住主要矛盾
2. **快速迭代** - 先有框架，后续可深化
3. **逻辑闭环** - 每步输出支撑下一步
