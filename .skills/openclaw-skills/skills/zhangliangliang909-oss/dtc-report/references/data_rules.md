## 零 -3、当月未结束时的收入/毛利折算规则

**适用场景：** 当月数据还未结束时（如 3 月只到 30 日）

### 折算规则总表

| 业务段 | 收入折算规则 | 毛利折算规则 |
|--------|-------------|-------------|
| **B 段** | 按上月单箱收入×当月箱量预估 | 按上月单箱毛利×当月箱量预估 |
| **C 段** | 按上月单箱收入×当月箱量预估 | 按上月单箱毛利×当月箱量预估 |
| **D 段** | 按天数折算 | 按天数折算 |
| **电商集拼** | 按天数折算 | 按天数折算 |

### 计算公式

**B/C 段（预估）：**
```
当月收入 = 上月单箱收入 × 当月箱量
当月毛利 = 上月单箱毛利 × 当月箱量
```

**D 段（折算）：**
```
当月收入 = 实际收入 × (当月总天数 / 已过的天数)
当月毛利 = 实际毛利 × (当月总天数 / 已过的天数)
```

### 示例（3 月截止到 30 日）

**已知：**
- 2 月 B 段单箱收入：6,733 元/箱
- 2 月 B 段单箱毛利：239 元/箱
- 3 月 B 段箱量：300 TEU
- 3 月 D 段实际收入：4,085 万元
- 3 月总天数：31 天
- 已过天数：30 天

**计算：**
```python
# B 段预估
mar_b_revenue = 6733 × 300 = 2,019,900 元 = 202 万元
mar_b_profit = 239 × 300 = 71,700 元 = 7.2 万元

# D 段折算
mar_d_revenue = 4085 × (31/30) = 4,221 万元
```

### Python 实现

```python
# 从文件名提取截止日期
filename = "DTC 业务费用根据客户仓库统计表_2025 年至 2026 年 3 月 30 日.xlsx"
match = re.search(r'(\d{4}) 年至 (\d{4}) 年 (\d{1,2}) 月 (\d{1,2}) 日', filename)
end_day = int(match.group(4))  # 30
month_days = 31  # 3 月总天数
ratio = month_days / end_day  # 31/30 = 1.0333

# D 段折算
d_revenue_adjusted = d_revenue_raw * ratio
d_profit_adjusted = d_profit_raw * ratio

# B/C 段预估（需要读取上月单箱数据）
# 从 2 月数据计算单箱收入/毛利
feb_b_revenue_per_box = feb_b_revenue / feb_b_volume
feb_b_profit_per_box = feb_b_profit / feb_b_volume

# 预估 3 月
mar_b_revenue = feb_b_revenue_per_box * mar_b_volume
mar_b_profit = feb_b_profit_per_box * mar_b_volume
```

## 零 -1、环比规则

**环比 = 与上一个周期对比**

| 报告期间 | 环比期间 | 说明 |
|---------|---------|------|
| **月度报告** | 上月 | 如 4 月环比=3 月 |
| **Q1 报告** | 去年 Q4 | 如 2026-Q1 环比=2025-Q4 |
| **Q2 报告** | 今年 Q1 | 如 2026-Q2 环比=2026-Q1 |
| **Q3 报告** | 今年 Q2 | 如 2026-Q3 环比=2026-Q2 |
| **Q4 报告** | 今年 Q3 | 如 2026-Q4 环比=2026-Q3 |
| **半年度（1-6 月）** | 去年下半年（7-12 月） | 如 2026 上半年环比=2025 下半年 |
| **半年度（7-12 月）** | 今年上半年（1-6 月） | 如 2026 下半年环比=2026 上半年 |
| **年度报告** | 去年全年 | 如 2026 年环比=2025 年 |

**环比增长率计算：**
```
环比增长率 = (本期数 - 上期数) / 上期数 × 100%
```

**Python 实现：**
```python
def get_comparison_period(period):
    """获取环比期间"""
    if period.upper().startswith('2026-Q'):
        quarter = int(period[-1])
        if quarter == 1:
            return '2025-Q4'  # Q1 环比=去年 Q4
        else:
            return f'2026-Q{quarter-1}'  # Q2/Q3/Q4 环比=上季度
    
    if period.startswith('2026-'):
        month_part = period[5:]
        if '-' in month_part and '月' in month_part:
            # 半年度：1-6 月或 7-12 月
            if '1-6' in month_part:
                return '2025-7-12 月'  # 上半年环比=去年下半年
            elif '7-12' in month_part:
                return '2026-1-6 月'  # 下半年环比=今年上半年
        else:
            current_month = int(month_part)
            if current_month == 1:
                return '2025-12'  # 1 月环比=去年 12 月
            else:
                return f'2026-{current_month-1:02d}'  # 其他月份环比=上月
    
    return '2025-Q4'  # 默认
```

## 四、天数折算规则

**当月还没结束时，需要按天数折算：**

| 数据项 | 折算规则 |
|--------|---------|
| **D 段箱量** | 实际天数 / 当月总天数 |
| **出库件数** | 实际天数 / 当月总天数 |
| **D 段收入/毛利** | 实际天数 / 当月总天数 |
| **B/C 段收入/毛利（预估）** | 上月单箱收入 × 本月箱量 |

**Python 示例：**
```python
# D 段箱量折算
actual_days = 30  # 当前日期
month_days = 31   # 3 月总天数
ratio = actual_days / month_days
d_volume_adjusted = d_volume_raw * ratio

# 出库件数折算
outbound_pieces_adjusted = outbound_pieces_raw * ratio
```

# 错误：取合并单元格第一行
