# 常用端口参考

## 推荐端口范围

OpenClaw 网关推荐使用 **18000-18999** 范围的端口，避免与常见服务冲突。

## 已知端口分配

| 端口 | 用途 | 说明 |
|------|------|------|
| 18789 | 本地虾 (openim) | 默认端口 |
| 18790 | 飞书机器人 (feishu) | 第二实例 |
| 18791 | 测试实例 | 临时测试 |
| 18888 | 备用实例 1 | 好记的端口 |
| 18889 | 备用实例 2 | 好记的端口 |
| 18890 | 备用实例 3 | 好记的端口 |

## 避免使用的端口

| 端口 | 占用服务 |
|------|---------|
| 18000-18010 | Beats (Elastic) |
| 18080 | 常见 HTTP 代理 |
| 18443 | 常见 HTTPS 代理 |
| 18980 | Zabbix agent |

## 端口选择建议

1. **开发/测试**: 18791-18799
2. **生产实例**: 18800-18899
3. **临时实例**: 18900-18999

## 检查端口占用

```bash
# 检查端口是否被占用
lsof -i :18888

# 查看所有 node 进程监听的端口
lsof -i -n -P | grep LISTEN | grep node
```
