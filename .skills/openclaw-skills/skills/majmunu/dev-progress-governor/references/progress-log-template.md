# Progress Log Template

Default file: `progress-log.md`

Append entries in this shape unless the user prefers another format.

```md
## 2026-03-09 14:30
- Phase: [planning / queue / execute / issue]
- Task: [task or issue name]
- Completed: [what finished]
- Files: [paths or areas]
- Commit: [hash or pending]
- Next: [single next step]
- Blockers: [none or concise blocker]
```

## Minimal variant

```md
## [timestamp]
- Completed: ...
- Files: ...
- Commit: ...
- Next: ...
- Blockers: ...
```
