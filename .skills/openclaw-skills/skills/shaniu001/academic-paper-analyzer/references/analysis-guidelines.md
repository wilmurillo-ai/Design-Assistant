# 学术论文分析参考指南

## 摘要类型识别

### 结构式摘要 (Structured Abstract)
通常包含明确的标签：
- **目的/Objective/Aim**
- **方法/Methods**
- **结果/Results**
- **结论/Conclusions**

### 非结构式摘要 (Unstructured Abstract)
- 连续段落形式
- 无明确标签
- 长度通常150-300词

### 半结构式摘要
- 部分有标签
- 部分连续叙述

## 论文结构模式

### 实证研究论文结构
```
1. 引言 (Introduction)
   1.1 研究背景
   1.2 文献综述
   1.3 研究问题/假设

2. 研究方法 (Methods)
   2.1 研究设计
   2.2 样本/数据
   2.3 变量测量
   2.4 分析方法

3. 研究结果 (Results)
   3.1 描述性统计
   3.2 假设检验
   3.3 补充分析

4. 讨论 (Discussion)
   4.1 主要发现
   4.2 理论意义
   4.3 实践意义
   4.4 研究局限
   4.5 未来方向

5. 结论 (Conclusion)

6. 参考文献
```

### 综述论文结构
```
1. 引言
   - 研究重要性
   - 综述范围和目的

2. 方法 (可选)
   - 文献检索策略

3. 正文
   - 主题1
   - 主题2
   - 主题3

4. 讨论
   - 综合评述
   - 研究空白
   - 未来方向

5. 结论

6. 参考文献
```

### 理论论文结构
```
1. 引言
   - 问题提出
   - 理论背景

2. 理论框架
   - 核心概念
   - 理论模型

3. 论证分析
   - 论点1
   - 论点2

4. 结论与启示

5. 参考文献
```

## 字数参考标准

### 国际期刊
| 期刊类型 | 正文字数 | 摘要字数 | 参考文献 |
|----------|----------|----------|----------|
| SCI/SSCI | 5000-8000 | 150-300 | 30-100 |
| 会议论文 | 4000-6000 | 150-250 | 20-50 |
| Letter/Note | 1500-3000 | 100-150 | 10-30 |

### 国内期刊
| 期刊类型 | 正文字数 | 摘要字数 | 关键词 | 参考文献 |
|----------|----------|----------|--------|----------|
| 核心期刊 | 8000-15000 | 200-400 | 3-8 | 30-80 |
| 普刊 | 5000-10000 | 150-300 | 3-8 | 20-50 |
| 学位论文 | 20000-50000 | 500-800 | 4-10 | 50-100+ |

## 引用风格检测

### APA格式特征
- 作者, A. A., & Author, B. B. (年份)
- (作者, 年份) 或 作者 (年份)
- 参考文献使用 hanging indent

### MLA格式特征
- Author. "Title." *Journal*, vol., no., year, pp.
- (作者 页码) 或 (作者)

### Chicago格式特征
- 脚注格式
- 参考文献列出所有作者

### GB/T 7714 (中国标准)
- 主要用于中文学术期刊
- 格式与国际惯例有差异

## 参考文献质量评估标准

### 数量评估
- 实证研究：30-50篇为基本要求
- 综述论文：100篇以上为佳
- 学位论文：50-100篇

### 时间分布
- 理想状态：近5年文献占40%以上
- 可接受：近5年占25%以上

### 语种分布
- 理工科：外文文献应占60%以上
- 社会科学：外文文献应占40%以上
- 人文学科：根据研究主题而定

### 来源类型
- 期刊文章：应占主导地位（60%+）
- 学位论文：适当引用（10-20%）
- 专著：经典理论引用（10-20%）
- 网络资源：限制使用（<10%）
