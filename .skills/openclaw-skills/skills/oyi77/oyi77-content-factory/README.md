# Content Factory

All-in-one YouTube content generator for OpenClaw.

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python content_factory.py "Your Topic"
python content_factory.py "https://youtu.be/VIDEO_ID" --shorts
python content_factory.py "/path/to/video.mp4" --shorts
```

## Features

- Regular YouTube videos from prompts
- YouTube Shorts from long videos
- YouTube Shorts from scratch
- 100% free tools (Edge TTS, Pexels, FFmpeg)
