# Google Flow Reference

## Model Capabilities

### Google Veo 3.1 - Fast

**Supported Features:**
- Text-to-video generation
- Portrait aspect ratio (9:16) - optimized for mobile/social
- Landscape aspect ratio (16:9) - standard video
- Square aspect ratio (1:1) - social media

**Video Specifications:**
- Duration: ~5-8 seconds
- Resolution: Depends on aspect ratio and credit tier
- Format: MP4

**Prompt Engineering Guidelines:**

**Effective prompts include:**
1. Subject description (what/who)
2. Action/motion (what they're doing)
3. Environment/setting (where)
4. Camera work (how it's filmed)
5. Lighting/atmosphere (mood)
6. Style modifiers (aesthetic)

**Example good prompts:**
```
"A sleek sports car drifting on a rainy mountain road at night, neon lights reflecting on wet asphalt, cinematic tracking shot, film noir style"

"A golden retriever playing fetch in a sunny meadow, slow motion shot, warm golden hour lighting, high quality"

"Aerial drone shot of a tropical beach with turquoise water, gentle waves lapping at white sand, palm trees swaying, documentary style"
```

**Camera movement keywords:**
- Static, handheld, gimbal smooth
- Tracking shot, dolly in/out, push in
- Pan left/right, tilt up/down
- Aerial, drone shot, overhead
- Close-up, wide shot, medium shot
- Slow motion, timelapse, hyperlapse

## Presets Reference

### Cinematic Preset
- Adds cinematic color grading
- Film-like motion blur
- Professional camera work
- Best for: dramatic scenes, storytelling

### Film Noir Preset
- High contrast black and white or desaturated
- Strong shadows
- Mysterious atmosphere
- Best for: dramatic, moody content

### Action Figure Preset
- Plastic/toy-like aesthetic
- Studio lighting
- Simplified environments
- Best for: product shots, playful content

## Credit System

**Understanding Credits:**
- Base cost scales with resolution and duration
- Additional output variations cost more
- Aspect ratios: 9:16 often optimized for mobile delivery
- Check credits before generating large batches

**Optimization Tips:**
- Start with lower settings for testing
- Use 1-2 outputs for concept validation
- Scale up for final production
- Monitor notification area for generation status

## Project Workflow Best Practices

**Organizing Projects:**
1. Create project per campaign/client
2. Use descriptive names: "Brand-Product-Date"
3. Delete test generations to save storage
4. Export final videos promptly

**Version Control:**
- Each generation creates new clip
- Previous versions remain in project
- Use search to find specific clips
- Star favorites for quick access

**Asset Management:**
- Projects can contain many clips
- Use view modes (cozy, grid, favorites) for navigation
- Search by date or custom filters
- Download final MP4 files for safekeeping
