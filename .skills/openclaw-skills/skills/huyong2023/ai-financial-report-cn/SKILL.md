**上传数据**：
```

营业成本,320000
销售费用,85000
管理费用,62000
财务费用,18000
```

**AI生成报告**：
```
【利润表】
一、营业收入：580,000元
二、营业成本：320,000元
    毛利润：260,000元（毛利率：44.8%）

三、期间费用：
    销售费用：85,000元
    管理费用：62,000元
    财务费用：18,000元
    合计：165,000元

四、营业利润：95,000元
五、净利润：71,250元（所得税25%）

【关键指标】
✅ 毛利率：44.8%（良好）
✅ 净利率：12.3%（健康）
⚠️ 费用率：28.4%（偏高）

【趋势分析】
📈 收入环比增长：15.2%
📉 成本环比下降：3.8%

【异常发现】
🚨 销售费用占比14.6%，高于行业平均（10%）
   建议：优化销售渠道，降低获客成本

【改进建议】
1. 控制销售费用，目标降至12%以下
2. 优化成本结构，提升毛利率至50%
3. 加快应收账款回收，改善现金流
```
