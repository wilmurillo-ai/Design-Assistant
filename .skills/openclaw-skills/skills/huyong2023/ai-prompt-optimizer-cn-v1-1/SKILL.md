# AI绘画提示词优化器 v1.1 🎨

智能优化AI绘画提示词，支持图片反推、参数调优、对比测试、批量生成。

## 🆕 v1.1 新增功能

### 🖼️ 图片反推提示词
```
用户: 分析这张图片的提示词
用户: 从图片生成提示词
用户: 这张图是怎么生成的
```

**反推功能**：
- 上传图片自动分析
- 识别风格、构图、光影
- 生成相似提示词
- 支持Midjourney/SD格式

### 💾 提示词收藏库
```
用户: 保存这个提示词
用户: 收藏优秀提示词
用户: 查看我的收藏
```

**收藏库功能**：
- 按风格分类收藏
- 搜索已收藏提示词
- 一键使用收藏
- 分享收藏库

### ⚙️ 参数精细调优
```
用户: 调整分辨率
用户: 修改采样器参数
用户: 优化生成参数
```

**可调参数**：
- 分辨率 (512-4096)
- 采样器 (Euler/DPM/DDIM)
- 步数 (20-150)
- CFG Scale (1-30)
- 种子值
- 负面提示词

### 🔄 提示词对比
```
用户: 对比这两个提示词
用户: A/B测试
用户: 哪个效果更好
```

**对比功能**：
- 并排对比效果
- 差异高亮显示
- 效果评分对比
- 推荐最优版本

### 🎨 风格混合
```
用户: 混合动漫和油画风格
用户: 赛博朋克+水彩
用户: 创建风格组合
```

**混合功能**：
- 多风格权重调整
- 实时预览混合效果
- 保存混合方案
- 风格推荐
