# 🔔 Report Reminder

> 日报提醒助手

## 功能简介

在工作日自动检查日报是否已提交，未提交时自动发送提醒。

## ✨ 特性

- ⏰ **定时检查** - 按设定时间自动检查日报提交状态
- ✅ **内容校验** - 检查日报是否为有效内容（不只是空标题）
- 👥 **自动@** - 未提交时自动@相关人员提醒
- 📅 **工作日识别** - 自动识别周一至周五为工作日，周末不强制提醒

## 使用方式

建议配合 Cron 定时任务使用，自动检查日报提交状态。

## ⏰ 定时配置（必选）

日报提醒需要定时触发，建议配置：

```bash
# 每天下午5点检查日报（周一至周五工作日）
cron add \
  --name "日报提醒-每日17点" \
  --schedule '{"kind":"cron","expr":"0 17 * * 1-5"}' \
  --payload '{"kind":"systemEvent","text":"请检查日报是否已提交"}' \
  --sessionTarget main
```

### Cron 表达式说明

| 字段 | 含义 | 取值范围 |
|------|------|----------|
| 第1个 | 分钟 (minute) | 0-59 |
| 第2个 | 小时 (hour) | 0-23 |
| 第3个 | 日期 (day) | 1-31 |
| 第4个 | 月份 (month) | 1-12 |
| 第5个 | 星期 (weekday) | 0-6 (0=周日) |

### 常用示例

```bash
# 每天早上9点检查
"0 9 * * *"

# 每天下午5点检查（常用）
"0 17 * * *"

# 每天下午5点检查（仅工作日）
"0 17 * * 1-5"

# 每天上午10点半检查
"30 10 * * *"

# 每周一至周五下午6点检查
"0 18 * * 1-5"
```

## 检查规则

视为"未提交"的情况：
- 📄 文件不存在
- 📄 文件为空
- 📄 仅有标题，没有正文
- 📄 正文只有"待补充"占位内容

## 提醒文案

### 未提交时
```
@xxx 提醒：今天的日报还未提交，请尽快补充今日工作内容。
```

### 已提交时
```
今日日报已提交，无需提醒。
```

## 📌 注意事项

- 不会伪造日报内容
- 不会自动编写完整日报
- 仅在确认缺失时发送提醒
- 周末不触发催报，但接受用户主动提交
