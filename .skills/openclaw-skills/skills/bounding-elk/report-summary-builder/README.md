# 📊 Report Summary Builder

> 工作汇总助手

## 功能简介

基于已有的日报数据，自动汇总生成周报和月报。

## ✨ 特性

- 📅 **周报生成** - 自动汇总上周工作完成情况
- 🗓️ **月报生成** - 自动汇总本月重点工作成果
- 🔍 **智能合并** - 合并重复事项，去重整理
- ⚠️ **缺失提示** - 日期缺失时显式标注
- 📈 **专业输出** - 企业级汇报语言，简洁有力

## 使用方式

当用户输入以下指令时触发：
- "生成周报"
- "生成月报"
- "汇总上周工作"
- "汇总本月工作"

## ⏰ 定时配置（可选）

如需自动生成汇总报告，可配合 Cron 使用：

```bash
# 每周一早上9点自动生成周报
cron add \
  --name "周报生成-每周一9点" \
  --schedule '{"kind":"cron","expr":"0 9 * * 1"}' \
  --payload '{"kind":"systemEvent","text":"请生成上周周报"}' \
  --sessionTarget main

# 每月1号早上9点自动生成月报
cron add \
  --name "月报生成-每月1号9点" \
  --schedule '{"kind":"cron","expr":"0 9 1 * *"}' \
  --payload '{"kind":"systemEvent","text":"请生成上月月报"}' \
  --sessionTarget main
```

### Cron 表达式说明

| 字段 | 含义 | 取值范围 |
|------|------|----------|
| 第1个 | 分钟 (minute) | 0-59 |
| 第2个 | 小时 (hour) | 0-23 |
| 第3个 | 日期 (day) | 1-31 |
| 第4个 | 月份 (month) | 1-12 |
| 第5个 | 星期 (weekday) | 0-6 (0=周日) |

### 常用示例

```bash
# 每周一早上9点
"0 9 * * 1"

# 每周五下午5点
"0 17 * * 5"

# 每月1号早上9点
"0 9 1 * *"

# 每月最后一天下午6点
"0 18 28-31 * *"

# 每周一至周五早上9点半
"30 9 * * 1-5"
```

## 📌 注意事项

- 只基于真实日报内容汇总，不编造
- 语言专业简洁，避免空话
- 缺日报时必须显式提示用户
