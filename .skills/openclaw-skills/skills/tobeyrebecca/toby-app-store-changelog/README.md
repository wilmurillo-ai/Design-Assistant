# App Store Changelog - OpenClaw Skill

OpenClaw skill for app-store-changelog functionality.

## Installation

### Via ClawHub
```bash
clawhub install toby-app-store-changelog
```

### Manual Installation
```bash
git clone https://github.com/TobeyRebecca/app-store-changelog.git
cp -r app-store-changelog ~/.openclaw/skills/app-store-changelog
```
