# Anycrawl - OpenClaw Skill

OpenClaw skill for anycrawl functionality.

## Installation

### Via ClawHub
```bash
clawhub install toby-anycrawl
```

### Manual Installation
```bash
git clone https://github.com/TobeyRebecca/anycrawl.git
cp -r anycrawl ~/.openclaw/skills/anycrawl
```
