# Appdeploy - OpenClaw Skill

OpenClaw skill for appdeploy functionality.

## Installation

### Via ClawHub
```bash
clawhub install toby-appdeploy
```

### Manual Installation
```bash
git clone https://github.com/TobeyRebecca/appdeploy.git
cp -r appdeploy ~/.openclaw/skills/appdeploy
```
