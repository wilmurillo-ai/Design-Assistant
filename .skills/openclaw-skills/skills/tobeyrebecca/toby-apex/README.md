# Apex - OpenClaw Skill

OpenClaw skill for apex functionality.

## Installation

### Via ClawHub
```bash
clawhub install toby-apex
```

### Manual Installation
```bash
git clone https://github.com/TobeyRebecca/apex.git
cp -r apex ~/.openclaw/skills/apex
```
