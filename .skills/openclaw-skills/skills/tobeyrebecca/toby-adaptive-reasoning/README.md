# Adaptive Reasoning - OpenClaw Skill

Adaptive reasoning assistant for complex problem-solving and decision-making

## Installation

### Via ClawHub
```bash
clawhub install adaptive-reasoning
```

### Manual Installation
```bash
git clone https://github.com/TobeyRebecca/adaptive-reasoning.git
cp -r adaptive-reasoning ~/.openclaw/skills/adaptive-reasoning
```
