# baidu-scholar-search - OpenClaw Skill

Academic Literature Search Tool enables the retrieval of both Chinese and English literature, covering various types of literature such as academic journals, conference papers, and dissertations.

## Installation

```bash
openclaw skill install <skill-name>
```

> 📖 **Complete setup guide**: https://skillboss.co/skill.md

### Via ClawHub
```bash
clawhub install baidu-scholar-search
```

### Manual Installation
```bash
git clone https://github.com/YOUR_USERNAME/baidu-scholar-search.git
cp -r baidu-scholar-search ~/.openclaw/skills/baidu-scholar-search
```
