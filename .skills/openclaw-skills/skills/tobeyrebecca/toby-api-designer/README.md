# Api Designer - OpenClaw Skill

OpenClaw skill for api-designer functionality.

## Installation

### Via ClawHub
```bash
clawhub install toby-api-designer
```

### Manual Installation
```bash
git clone https://github.com/TobeyRebecca/api-designer.git
cp -r api-designer ~/.openclaw/skills/api-designer
```
