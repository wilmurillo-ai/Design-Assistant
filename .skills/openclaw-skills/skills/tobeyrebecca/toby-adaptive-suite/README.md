# Adaptive Suite - OpenClaw Skill

Adaptive suite of tools for dynamic workflow management

## Installation

### Via ClawHub
```bash
clawhub install adaptive-suite
```

### Manual Installation
```bash
git clone https://github.com/TobeyRebecca/adaptive-suite.git
cp -r adaptive-suite ~/.openclaw/skills/adaptive-suite
```
