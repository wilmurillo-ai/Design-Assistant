# Mental Health Assistant 心理健康支持助手（专业版 v2.1）

[![OpenClaw Skill](https://img.shields.io/badge/OpenClaw-Skill-blue)](https://openclaw.ai)
[![Version](https://img.shields.io/badge/version-2.1.0-green)](https://clawhub.com)
[![License](https://img.shields.io/badge/license-MIT-orange)](LICENSE)

基于临床心理学与精神医学循证标准的专业心理健康支持助手。

## ✨ 专业特性

### 📊 循证心理评估
- **PHQ-9/GAD-7/ISI/PSS-10/WHO-5** 标准化量表
- 符合DSM-5诊断标准的评估框架
- C-SSRS自杀风险评估

### 🆘 危机干预
- 危机干预六步法（WHO心理急救）
- 自杀风险分级与干预策略
- 安全计划模板
- 全国24小时心理援助热线

### 🧠 循证心理治疗
- **CBT** 认知行为疗法（⭐⭐⭐⭐⭐证据）
- **DBT** 辩证行为疗法（边缘型人格障碍一线）
- **ACT** 接纳承诺疗法
- **EMDR** 眼动脱敏再加工（PTSD一线）
- **MBCT** 正念认知疗法
- **SFBT** 焦点解决短期治疗
- 精神动力学治疗、人本主义疗法、家庭治疗

### 💊 精神医学基础
- DSM-5/ICD-11诊断标准参考
- 主要精神障碍概述
- 心理药理学基础（SSRIs/SNRIs/心境稳定剂）
- 转介指征

### 📖 专业伦理
- 中国心理学会伦理守则
- 保密原则与例外
- 知情同意
- 双重关系处理
