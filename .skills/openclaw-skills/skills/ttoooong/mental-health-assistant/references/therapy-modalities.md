# 循证心理治疗流派

> 本文档汇总了具有强实证支持的主要心理治疗流派

---

## 一、认知行为疗法（CBT）

### 核心原理
认知决定情绪，情绪影响行为。通过改变非适应性思维和行为模式改善心理问题。

### 疗效证据
| 适应症 | 证据等级 | 疗效数据 |
|--------|----------|----------|
| 抑郁症 | ⭐⭐⭐⭐⭐ | 降低复发率43%，与药物疗效相当 |
| 焦虑障碍 | ⭐⭐⭐⭐⭐ | 首选疗法，显著改善症状 |
| 强迫症 | ⭐⭐⭐⭐⭐ | 暴露反应阻止(E-RP)为非药物首选 |
| PTSD | ⭐⭐⭐⭐ | 创伤聚焦CBT有效 |

### 核心技术
- 认知重构：识别并改变认知歪曲
- 行为激活：增加积极活动
- 暴露疗法：系统脱敏
- 技能训练：社交技能、问题解决

### 治疗结构
```
初期(1-4次)：建立关系、评估、设定目标
中期：认知重构、行为干预
后期：巩固、预防复发
```

## 二、辩证行为疗法（DBT）

### 概述
由玛莎·林内翰开发，专为边缘型人格障碍设计，融合CBT与辩证思想。

### 四大核心模块

| 模块 | 核心目标 | 关键技能 |
|------|----------|----------|
| **正念** | 活在当下 | 观察、描述、参与 |
| **情绪调节** | 管理剧烈情绪 | 识别情绪、增强正性情绪 |
| **人际效能** | 改善人际关系 | 沟通、问题解决 |
| **承受痛苦** | 不自伤应对痛苦 | 危机生存技能 |

### 适用人群
- 边缘型人格障碍（一线推荐）
- 反复自伤/自杀行为
- 情绪调节困难

### 治疗形式
- 个体治疗（每周）
- 技能训练团体（每周）
- 电话辅导（危机时）
- 治疗师团队会议

## 三、接纳承诺疗法（ACT）

### 核心概念
提升**心理灵活性**——带着痛苦仍能过有价值的生活。

### 六大核心过程

#### 开放支柱
1. **接纳**：开放对待不愉快体验
2. **认知解离**：将想法视为语言符号，而非事实
3. **以己为景**：识别"观察者自我"
4. **联结当下**：专注当前体验

#### 投入支柱
5. **澄清价值**：明确人生方向
6. **承诺行动**：基于价值采取行动

### 核心隐喻
```
"你与你的想法"
想法："我很失败"
↓
融合："我是个失败者"
解离："我有一个'我很失败'的想法"
```

### 适用范围
抑郁症、焦虑症、慢性疼痛、成瘾

## 四、眼动脱敏再加工（EMDR）

### 概述
通过双侧刺激处理创伤记忆，PTSD一线推荐治疗。

### 核心技术：双侧刺激
- 眼动：左右移动（1-2Hz）
- 触觉：交替轻拍双肩/膝盖
- 听觉：左右声道交替声音

### 八阶段模型

| 阶段 | 目标 |
|------|------|
| 1. 病史采集 | 评估适应性，确定目标记忆 |
| 2. 准备期 | 建立信任，教授调节技巧 |
| 3. 评估期 | 识别图像、信念、情绪、躯体感 |
| 4. 脱敏期 | 双侧刺激降低困扰 |
| 5. 植入期 | 强化积极认知 |
| 6. 躯体扫描 | 处理残留躯体不适 |
| 7. 结束与围合 | 确保情绪稳定 |
| 8. 再评估 | 追踪效果 |

### 适用范围
PTSD、复杂创伤、焦虑、抑郁

---

## 五、正念认知疗法（MBCT）

### 概述
整合正念训练与认知技术，预防抑郁复发。

### 八周课程结构

| 周次 | 主题 | 练习 |
|------|------|------|
| 1-2 | 基础觉察 | 身体扫描、呼吸冥想 |
| 3-4 | 深化技能 | 正念行走、3分钟呼吸空间 |
| 5-6 | 情绪管理 | 与困难共处、认知解离 |
| 7-8 | 生活整合 | 日常正念、预防复发计划 |

### 核心机制
- 减少反刍思维
- 识别抑郁思维模式
- 培养"存在模式"而非"行动模式"

### 疗效
降低抑郁复发风险约50%

## 六、焦点解决短期治疗（SFBT）

### 核心原则
聚焦解决方案而非问题原因，利用来访者自身资源。

### 三大核心技术

#### 奇迹提问
```
"如果今晚奇迹发生，问题解决了，
 明天早上你会注意到什么变化？"
```

#### 例外提问
```
"最近有没有问题不那么严重的时候？
 当时你做了什么？"
```

#### 量表提问
```
"如果10分是理想状态，1分是最糟，
 你现在在几分？如何能提高1分？"
```

### 适用范围
短期咨询、危机干预、学校/企业咨询

## 七、精神动力学治疗

### 核心概念

| 概念 | 定义 | 临床意义 |
|------|------|----------|
| **移情** | 将早期情感投射到治疗师 | 揭示关系模式 |
| **反移情** | 治疗师对来访者的情感反应 | 理解无意识 |
| **阻抗** | 抵制治疗过程 | 保护现有平衡 |
| **防御机制** | 无意识的心理策略 | 应对焦虑和冲突 |

### 治疗目标
将潜意识内容意识化，修通早期关系创伤。

### 适用范围
人格障碍、长期人际关系困难、心身疾病

---

## 八、人本主义疗法（来访者中心）

### 三大核心条件

| 条件 | 内涵 | 实践 |
|------|------|------|
| **真诚一致** | 真实、透明、不伪装 | 以完整自我参与 |
| **无条件积极关注** | 不加批判地接纳 | 尊重感受与选择 |
| **共情理解** | 感同身受 | 深入主观世界 |

### 核心原则
- 非指导性
- 来访者自主探索
- 激发自我实现潜能

### 适用范围
自我成长、身份认同、人际关系

---

## 九、家庭治疗

### 系统视角
个体症状源于家庭系统互动失衡，通过改变家庭动态解决问题。

### 主要流派

| 流派 | 创立者 | 核心概念 |
|------|--------|----------|
| **结构式** | 米纽钦 | 权力结构、边界、子系统 |
| **鲍恩系统** | 鲍恩 | 自我分化、三角关系、代际传递 |
| **萨提亚** | 萨提亚 | 自尊、沟通模式、自我成长 |

### 常用工具
- 家谱图（Genogram）
- 家庭地图
- 沟通训练

### 适用范围
亲子冲突、婚姻问题、青少年行为问题

---

## 十、团体心理治疗

### 团体发展阶段（Tuckman模型）
1. 形成期：试探、建立规范
2. 风暴期：冲突、权力斗争
3. 规范期：形成凝聚力
4. 工作期：深入探索
5. 结束期：处理分离

### 亚隆11项疗效因子
1. 希望灌注
2. 普适性
3. 信息传递
4. 利他主义
5. 原生家庭矫正性重现
6. 社交技能发展
7. 模仿行为
8. 人际学习
9. 团体凝聚力
10. 情绪宣泄
11. 存在因子

### 适用范围
人际关系问题、社交焦虑、成瘾康复

## 治疗选择指南

| 问题类型 | 首选疗法 | 备选疗法 |
|----------|----------|----------|
| 抑郁症 | CBT/MBCT | ACT、人际心理治疗 |
| 焦虑障碍 | CBT | ACT、正念 |
| PTSD | 创伤聚焦CBT/EMDR | CBT |
| 边缘型人格障碍 | DBT | 移情聚焦治疗 |
| 慢性疼痛 | ACT/CBT | 正念 |
| 夫妻/家庭问题 | 家庭治疗 | EFT |
| 短期危机 | SFBT | 危机干预 |
