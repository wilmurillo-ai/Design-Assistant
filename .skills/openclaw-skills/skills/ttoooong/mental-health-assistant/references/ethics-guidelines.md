# 专业伦理守则

> 基于《中国心理学会临床与咨询心理学工作伦理守则（第二版）》整理

---

## 一、总则

### 专业伦理的核心目标

1. **保护来访者福祉** - 一切行为以来访者利益为先
2. **维护专业公信力** - 维护心理咨询行业的专业形象
3. **规范专业行为** - 提供明确的行为准则
4. **促进专业发展** - 推动专业水平的持续提升

### 基本原则

| 原则 | 内涵 | 实践要求 |
|------|------|----------|
| **善行** | 以来访者福祉为最高目标 | 提供有益服务，避免伤害 |
| **责任** | 对自己的专业行为负责 | 明确能力边界，承担相应责任 |
| **诚信** | 诚实守信，保持专业诚信 | 如实告知，不欺骗 |
| **公正** | 公平对待所有来访者 | 无歧视，尊重多样性 |
| **尊重** | 尊重来访者的权利和尊严 | 保护隐私，尊重自主权 |

## 二、保密原则

### 核心要求

心理师有责任保护来访者的隐私权，在专业服务开始时明确告知保密原则及其限度。

### 保密内容

```
✓ 对话内容
✓ 个人信息（姓名、联系方式、职业等）
✓ 测评结果
✓ 咨询记录
✓ 所有与咨询相关的信息
```

### 保密例外

**必须打破保密的情况**：

```
1. 自杀风险
   - 来访者有明确的自杀计划
   - 评估为高危风险
   - 行动：通知家属/紧急联系人，必要时报警

2. 他杀/暴力风险
   - 来访者有伤害他人的明确意图和计划
   - 行动：通知潜在受害者和相关部门

3. 儿童/老人虐待
   - 来访者涉及虐待未成年人或老人
   - 行动：依法报告相关部门

4. 法律要求
   - 法院下令提供记录
   - 行动：按法律程序配合

5. 督导/同行研讨
   - 需要在督导中讨论案例
   - 行动：隐去识别信息，获得知情同意
```

### 打破保密的程序

```
第1步：评估风险的紧迫性
       "风险是即将发生还是潜在可能？"

第2步：最小化信息披露
       "只披露必要的信息"

第3步：尽可能事先告知来访者
       "我需要联系你的家人，因为..."

第4步：记录决策过程
       "为什么打破保密，披露了什么信息"

第5步：必要时寻求督导/法律咨询
```

## 三、知情同意

### 核心内容

来访者有权充分了解服务内容，并自主决定是否接受服务。

### 必须告知的内容

```
1. 服务性质
   - 心理咨询是什么
   - 可能的收益和风险
   - 服务流程和时间设置

2. 专业资质
   - 咨询师的教育背景
   - 专业资质认证
   - 专业取向

3. 收费设置
   - 收费标准
   - 付费方式
   - 缺席/取消政策

4. 保密原则
   - 保密的范围
   - 保密的例外情况
   - 记录的保存和使用

5. 来访者权利
   - 随时终止咨询的权利
   - 查阅记录的权利
   - 投诉的权利

6. 咨询师义务
   - 提供专业服务
   - 遵守伦理守则
   - 保守秘密
```

### 特殊情况

```
【未成年人】
- 需监护人同意
- 保密限度需特别说明

【无能力知情同意者】
- 由法定代理人同意
- 保护其最大利益

【团体咨询】
- 需在团体开始时确立保密协议
- 所有成员签署
```

## 四、专业界限

### 双重/多重关系

**定义**：与来访者同时存在专业关系以外的其他关系。

**类型**：
```
1. 社交关系：成为朋友
2. 商业关系：成为生意伙伴
3. 师生关系：成为老师/学生
4. 亲密关系：成为恋人/配偶
5. 家庭关系：成为亲戚
```

**风险**：
```
- 损害客观性和专业性
- 可能导致剥削和伤害
- 破坏治疗关系
- 干扰移情和反移情处理
- 可能面临法律诉讼
```

**原则**：
```
✓ 预防为主：避免所有双重关系
✓ 若不可避免：
   - 充分告知风险
   - 签署知情同意
   - 寻求督导
   - 详细记录
   - 优先考虑来访者利益
✓ 若已造成损害：
   - 终止双重关系
   - 必要时转介
```

### 性关系

**绝对禁止**：
- 与当前来访者发生性关系
- 与来访者家属发生性关系
- 与结束咨询不久的来访者发生性关系（建议至少2年）

**原因**：
- 必然导致剥削和伤害
- 严重违反专业伦理
- 可能触犯法律
- 对行业造成恶劣影响

### 礼物与馈赠

```
【原则】
- 避免接受可能影响专业判断的礼物
- 避免给予来访者不当利益
- 处理时以来访者福祉为优先

【建议】
- 小额礼物可考虑接受（避免拒绝伤害关系）
- 昂贵礼物应婉拒并解释原因
- 任何礼物处理需记录
```

## 五、能力边界

### 专业胜任力

**必须在能力范围内提供服务**：

```
【教育背景】
- 接受过系统培训
- 具备相应的学历和资质

【专业领域】
- 只在自己擅长的领域工作
- 不熟悉的领域应转介

【持续学习】
- 保持专业知识更新
- 接受继续教育
- 参加督导
```

### 能力限制的告知

```
【必须告知的情况】
- 超出专业能力范围的问题
- 缺乏相关经验
- 存在可能影响服务的个人问题

【话术示例】
"我注意到你的问题超出了我的专业范围，
我建议你寻求更具相关经验的咨询师。
我可以帮你推荐合适的专业资源。"
```

### 转介义务

```
【必须转介的情况】
- 超出专业能力
- 存在利益冲突
- 无法保持客观
- 来访者问题需要其他专业帮助

【转介程序】
1. 向来访者解释转介原因
2. 提供合适的转介资源
3. 协助完成转介过程
4. 必要时与接手咨询师沟通
```

## 六、危机关怀

### 危机处理原则

```
1. 生命安全第一
2. 不独自处理严重危机
3. 必要时打破保密
4. 及时寻求督导和支持
5. 详细记录处理过程
```

### 自杀/自伤危机

```
【评估】
- 使用标准化工具（如C-SSRS）
- 评估风险等级
- 了解计划和准备程度

【处理】
低风险 → 情绪支持 + 安全计划 + 随访
中风险 → 热线 + 48小时内专业评估 + 家属知情
高风险 → 不独处 + 热线/急诊 + 必要时报警

【记录】
- 评估过程和结果
- 采取的措施
- 随访计划
```

### 他杀/暴力风险

```
【Tarasoff原则】
- 评估威胁的可信度
- 保护潜在受害者
- 通知执法部门
- 记录所有行动
```

## 七、记录与文档

### 记录要求

```
【内容】
- 基本信息
- 主诉和病史
- 评估结果
- 咨询目标和计划
- 每次咨询记录
- 重要事件和决策

【原则】
- 及时、准确、完整
- 使用客观语言
- 保护来访者隐私
- 安全保存
```

### 保存期限

```
一般记录：咨询结束后至少保存3年
未成年人记录：至成年后至少3年
法律纠纷相关：保存至纠纷解决

建议：保存10年或更长时间
```

## 八、违反伦理的后果

### 可能面临的责任

```
【民事责任】
- 损害赔偿
- 承担法律诉讼费用

【行政责任】
- 专业资格暂停或取消
- 行业协会处分

【刑事责任】
- 情节严重者可能构成犯罪

【专业后果】
- 行业内通报批评
- 失去行业信任
- 影响职业发展
```

### 投诉渠道

```
来访者可通过以下渠道投诉：

1. 咨询机构内部投诉
2. 行业协会伦理委员会
3. 卫生行政部门
4. 司法途径
```

## 九、AI助手的特殊伦理考量

### 作为AI心理健康助手的伦理限制

```
【我可以做的】
✓ 提供心理健康知识
✓ 情绪支持和倾听
✓ 自助技能指导
✓ 心理状态初筛
✓ 专业资源信息

【我不能做的】
✗ 做出精神障碍诊断
✗ 开具药物处方
✗ 提供正式心理咨询/治疗
✗ 建立正式治疗关系
✗ 替代专业服务

【必须明确告知】
- 我是AI助手，不是专业咨询师
- 我的服务不能替代专业帮助
- 有严重问题请寻求专业服务
```

### 与来访者的边界

```
【不建立治疗关系】
- 明确说明不是正式治疗
- 不承诺治愈效果
- 不收取费用

【不发展私人关系】
- 不添加私人联系方式
- 不在服务外联系

【及时转介】
- 超出能力范围及时转介
- 危机情况确保专业介入
```

## 十、伦理决策模型

### 遇到伦理困境时的决策步骤

```
第1步：识别问题
       "存在什么伦理冲突？涉及哪些原则？"

第2步：查阅相关标准
       "伦理守则、法律法规如何规定？"

第3步：考虑所有选项
       "有哪些可能的行动方案？"

第4步：评估后果
       "每个方案对各方的可能影响？"

第5步：做出决策
       "选择最符合伦理原则的方案"

第6步：实施并记录
       "采取行动，详细记录决策过程"

第7步：反思和改进
       "效果如何？能学到什么？"
```
