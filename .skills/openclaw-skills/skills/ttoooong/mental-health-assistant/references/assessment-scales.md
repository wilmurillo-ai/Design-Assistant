# 心理评估量表完整版

⚠️ **重要声明**：以下量表仅供自我筛查参考，不能替代专业诊断。如分数较高或持续感到困扰，请及时寻求专业心理帮助。

---

## 一、PHQ-9 抑郁筛查量表

### 基本信息
- **名称**：9项患者健康问卷（Patient Health Questionnaire-9 items）
- **用途**：评估过去两周内抑郁症状的严重程度
- **题目数**：9题
- **用时**：约2分钟

### 量表题目

**指导语**：在过去两周，你有多少时候受到以下问题困扰？

| 题号 | 题目内容 |
|------|----------|
| 1 | 做事时提不起劲或没有兴趣 |
| 2 | 感到心情低落、沮丧或绝望 |
| 3 | 入睡困难、易醒或睡眠过多 |
| 4 | 感觉疲倦或没有活力 |
| 5 | 食欲不振或吃得太多 |
| 6 | 觉得自己很糟，或觉得自己很失败，让自己或家人失望 |
| 7 | 对事物专注有困难，例如阅读报纸或看电视时 |
| 8 | 动作或说话速度缓慢到别人可以察觉？或者相反，烦躁或坐立不安、动来动去 |
| 9 | 有不如死掉或用某种方式伤害自己的念头 |

### 评分标准

**每题选项分值**：
- 完全没有 = 0分
- 有几天 = 1分
- 一半以上时间 = 2分
- 几乎每天 = 3分

**总分范围**：0-27分

### 结果解读

| 总分 | 抑郁程度 | 建议措施 |
|------|----------|----------|
| 0-4分 | 无抑郁 | 正常范围，无需干预 |
| 5-9分 | 轻度抑郁 | 建议观察、自我调节，必要时寻求咨询 |
| 10-14分 | 中度抑郁 | 建议寻求专业心理咨询或治疗 |
| 15-19分 | 中重度抑郁 | 强烈建议就医评估 |
| 20-27分 | 重度抑郁 | 请立即寻求专业医疗帮助 |

### ⚠️ 特别关注
- **第1题（兴趣减退）和第4题（疲劳）**为抑郁症核心症状，若≥2分需重视
- **第9题（自杀意念）**评分≥1分即需高度关注，建议立即干预

## 二、GAD-7 焦虑筛查量表

### 基本信息
- **名称**：7项广泛性焦虑障碍量表（Generalized Anxiety Disorder-7）
- **用途**：评估过去两周内焦虑症状的严重程度
- **题目数**：7题
- **用时**：约2分钟

### 量表题目

**指导语**：在过去两周，你有多少时候受到以下问题困扰？

| 题号 | 题目内容 |
|------|----------|
| 1 | 感到紧张、焦虑或急切 |
| 2 | 不能控制或停止担忧 |
| 3 | 对各种各样的事情担忧过多 |
| 4 | 难以放松 |
| 5 | 焦躁不安以致于无法静坐 |
| 6 | 变得容易烦恼或易怒 |
| 7 | 感到害怕似乎会发生可怕的事情 |

### 评分标准

**每题选项分值**：
- 完全没有 = 0分
- 有几天 = 1分
- 一半以上时间 = 2分
- 几乎每天 = 3分

**总分范围**：0-21分

### 结果解读

| 总分 | 焦虑程度 | 建议措施 |
|------|----------|----------|
| 0-4分 | 正常/轻微 | 无明显焦虑症状 |
| 5-9分 | 轻度焦虑 | 建议关注，学习放松技巧 |
| 10-14分 | 中度焦虑 | 建议寻求专业评估 |
| 15-21分 | 重度焦虑 | 建议立即寻求专业帮助 |

### 临床临界值
- 总分≥8分建议进行进一步焦虑障碍评估

## 三、ISI 失眠严重指数量表

### 基本信息
- **名称**：失眠严重指数量表（Insomnia Severity Index）
- **用途**：评估主观失眠严重程度
- **题目数**：7题
- **用时**：约2分钟

### 量表题目

**指导语**：请评估你过去两周的睡眠情况

| 题号 | 题目内容 |
|------|----------|
| 1 | 你对目前睡眠问题的严重程度如何评价？ |
| 2 | 你对目前睡眠模式的满意程度如何？ |
| 3 | 你的睡眠问题在多大程度上影响你的日常功能？ |
| 4 | 你认为你的睡眠问题在多大程度上被他人察觉？ |
| 5 | 你对睡眠问题的担忧程度如何？ |
| 6 | 你目前入睡困难的程度如何？ |
| 7 | 你目前睡眠维持困难（易醒、早醒）的程度如何？ |

### 评分标准

**每题选项分值**：0-4分
- 0分 = 无问题
- 1分 = 轻度问题
- 2分 = 中度问题
- 3分 = 重度问题
- 4分 = 极严重问题

**总分范围**：0-28分

### 结果解读

| 总分 | 失眠程度 | 建议措施 |
|------|----------|----------|
| 0-7分 | 无显著失眠 | 睡眠状况良好 |
| 8-14分 | 轻度失眠 | 建议改善睡眠卫生 |
| 15-21分 | 中度失眠 | 建议寻求专业评估 |
| 22-28分 | 重度失眠 | 强烈建议就医 |

## 四、PSS-10 感知压力量表

### 基本信息
- **名称**：感知压力量表（Perceived Stress Scale-10）
- **用途**：评估过去一个月内对生活压力的感知程度
- **题目数**：10题
- **用时**：约3分钟

### 量表题目

**指导语**：请回想过去一个月，你对以下情况的感受频率

| 题号 | 题目内容 |
|------|----------|
| 1 | 因为某些意料之外的事情而感到心烦意乱 |
| 2 | 感觉无法控制生活中的重要事情 |
| 3 | 感到紧张和压力 |
| 4 | 对处理个人问题充满信心（反向题） |
| 5 | 感到事情按照你的意愿发展（反向题） |
| 6 | 发现自己无法应付必须做的事情 |
| 7 | 能够控制生活中烦躁的情绪（反向题） |
| 8 | 觉得自己驾驭事情的能力很强（反向题） |
| 9 | 对自己能处理的事情感到愤怒 |
| 10 | 觉得困难堆积如山，无法克服 |

### 评分标准

**每题选项分值**：
- 从未发生 = 0分
- 偶尔发生 = 1分
- 有时发生 = 2分
- 经常发生 = 3分
- 总是发生 = 4分

**⚠️ 反向计分题**：第4、5、7、8题需反向计分
- 原0分计4分，原1分计3分，原2分计2分，原3分计1分，原4分计0分

**总分范围**：0-40分

### 结果解读

| 总分 | 压力水平 | 建议措施 |
|------|----------|----------|
| 0-13分 | 低压力 | 压力调节能力良好 |
| 14-26分 | 中等压力 | 建议学习压力管理技巧 |
| 27-40分 | 高压力 | 建议寻求专业心理支持 |

## 五、WHO-5 幸福感指数量表

### 基本信息
- **名称**：世界卫生组织五项幸福感指数量表（WHO-5 Well-being Index）
- **用途**：评估过去两周的主观幸福感水平
- **题目数**：5题
- **用时**：约1分钟

### 量表题目

**指导语**：请回想过去两周，你有多少时间有以下感受？

| 题号 | 题目内容 |
|------|----------|
| 1 | 我感到心情愉悦 |
| 2 | 我感到平静和放松 |
| 3 | 我感到活跃和精力充沛 |
| 4 | 我睡醒后感到神清气爽 |
| 5 | 我的日常生活充满了感兴趣的事物 |

### 评分标准

**每题选项分值**（6级评分）：
- 从未有过 = 0分
- 少量时间 = 1分
- 一些时间 = 2分
- 较多时间 = 3分
- 大部分时间 = 4分
- 全部时间 = 5分

**总分计算**：
- 原始分 = 5题得分相加（0-25分）
- 最终得分 = 原始分 × 4（0-100分）

### 结果解读

| 得分 | 状态 | 建议措施 |
|------|------|----------|
| >50分 | 幸福感良好 | 继续保持积极生活状态 |
| 29-50分 | 幸福感一般 | 建议关注心理健康 |
| ≤28分 | 幸福感较低 | 可能存在抑郁风险，建议进一步评估 |

## 使用建议

### 给用户使用时

1. **说明目的**
   > "这个量表可以帮助你了解自己的心理状态，但不是诊断。如果分数较高，建议寻求专业帮助。"

2. **逐题提问**
   > "请根据过去两周（或一个月）的实际情况作答..."
   > 逐题朗读题目，提供选项

3. **计算与解读**
   - 计算总分
   - 说明结果范围
   - 提供建议

4. **后续建议**
   - 低分：鼓励继续保持
   - 中分：建议自我调节或咨询
   - 高分：建议就医评估

### ⛔ 重要限制

- 不做诊断，不说"你有抑郁症/焦虑症"
- 不开药，不推荐具体药物
- 高分必须建议就医
- 有自杀意念立即危机干预
