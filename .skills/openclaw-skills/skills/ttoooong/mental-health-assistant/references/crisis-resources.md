## 二、危机干预六步法

### 步骤概览

| 步骤 | 核心目标 | 关键行动 |
|------|----------|----------|
| 1. 确定问题 | 理解当事人的危机体验 | 倾听、开放式提问 |
| 2. 确保安全 | 消除即时威胁 | 风险评估、安全协议 |
| 3. 给予支持 | 建立信任关系 | 共情回应、传递关心 |
| 4. 提出策略 | 探索可行应对方式 | 提取过往有效策略 |
| 5. 制定计划 | 制定具体行动方案 | 明确步骤、资源链接 |
| 6. 获得承诺 | 确保遵守计划 | 获取承诺、约定随访 |

## 三、风险评估工具

### 自杀风险评估"三问"

当怀疑有自杀风险时，按以下顺序询问：

```
1. "最近有没有想过要结束自己的生命？"
   - 无 → 低风险
   - 有 → 继续

2. "有没有具体的计划？打算怎么做？"
   - 无计划 → 中风险
   - 有计划 → 继续

3. "有没有准备工具？有没有确定时间地点？"
   - 无 → 中高风险
   - 有 → 高风险，需立即干预
```

### 风险等级评估表

| 风险等级 | 特征 | 干预措施 |
|----------|------|----------|
| **高风险** | 有具体计划、工具、时间地点 | 立即拨打热线/报警，不要独处 |
| **中高风险** | 有计划但无工具/时间 | 提供热线，建议24小时监护 |
| **中风险** | 有想法但无计划 | 提供热线，建议就医 |
| **低风险** | 有消极念头但无行动意图 | 倾听支持，提供资源 |

## 四、危机干预对话脚本

### 高风险情况（有具体计划）

```
【立即响应】
"我很担心你现在的情况。谢谢你愿意告诉我这些。"

【评估风险】
"你现在在哪里？有人陪你吗？"
"你准备的工具在哪里？"

【紧急干预】
"请先把手边的东西放远一点，好吗？我在这儿陪你。"
"让我们打个电话给专业人士，他们能更好地帮助你。"
"现在就拨打 12356，我陪你一起。"

【不独处】
"在我结束对话之前，你能答应我不做任何伤害自己的事情吗？"
"我会保持联系，直到有人来到你身边。"
```

### 中风险情况（有想法但无计划）

```
【共情回应】
"听起来你正在经历一段非常困难的时期，愿意多跟我说说吗？"

【正常化】
"很多人在极度痛苦时都会有这样的想法，这不是你的错。"

【提供资源】
"有一些专业的人可以帮助你，你想试试拨打心理热线吗？12356 是24小时的。"

【制定安全计划】
"如果这些想法再次出现，你愿意答应我：先给热线打个电话，好吗？"
"有什么事情是你现在能做的，让自己感觉稍微好一点？"
```

### 低风险情况（消极念头）

```
【倾听支持】
"最近发生了什么让你有这种感觉？"

【探索资源】
"以前遇到类似的情况，你是怎么度过的？"
"现在有什么人或事能给你一些支持？"

【建立连接】
"如果需要的话，随时可以找我聊天。"
"建议你关注一下这些专业资源..."
```

## 六、就医指南

### 需要就医的情况

- 持续2周以上情绪低落
- 严重影响工作/学习/生活
- 有自伤或自杀想法
- 出现幻觉或妄想
- 焦虑症状无法自我调节

### 挂号科室

| 医院类型 | 科室选择 |
|----------|----------|
| 三甲综合医院 | 精神科/心理科 |
| 综合医院 | 心身医学科 |
| 精神专科医院 | 精神卫生中心 |

### 就医准备

1. 记录症状持续时间
2. 准备既往病史
3. 列出正在服用的药物
4. 有家人陪同更佳

## 八、干预后跟进

### 跟进时间表

| 时间节点 | 行动 |
|----------|------|
| 24小时内 | 确认安全状态，确认是否就医 |
| 3天内 | 了解情绪变化，强化安全计划 |
| 1周后 | 评估干预效果，调整支持策略 |
| 2周后 | 再次评估，必要时转介专业 |

### 跟进话术

```
"上次聊完后，这两天感觉怎么样？"
"有尝试使用我们说的安全计划吗？"
"有去看医生吗？医生怎么说的？"
"还有什么我能帮你的吗？"
```

## ⛔ 危机干预禁区

### 绝对不能做的事

1. **不评判** - 不说"你怎么能这样想"
2. **不淡化** - 不说"这有什么大不了的"
3. **不激怒** - 不说"你敢吗"、"你不会的"
4. **不保密** - 高风险必须打破保密，通知相关方
5. **不独处** - 高风险绝不让人独处
6. **不争辩** - 不与自杀意念争辩

### 必须做的事

1. **认真对待** - 每一个自杀威胁都当真
2. **表达关心** - "我很担心你"
3. **倾听理解** - 让对方说出感受
4. **提供资源** - 提供热线和就医信息
5. **持续关注** - 不轻易结束对话
6. **记录备案** - 重要情况需记录
