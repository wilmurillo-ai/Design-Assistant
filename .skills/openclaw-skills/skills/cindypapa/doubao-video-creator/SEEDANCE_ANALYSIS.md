# Seedance Storyboard Skill 分析报告

## 📊 核心亮点分析

### 1. 分步引导流程 ⭐⭐⭐⭐⭐

**五步法**：
```
第一步：理解用户想法（3 个问题）
第二步：深入挖掘细节（5 个维度）
第三步：构建分镜结构（时间轴拆解）
第四步：生成专业提示词（结构化模板）
第五步：优化与建议（检查 + 提醒）
```

**可借鉴点**：
- ✅ 流程清晰，循序渐进
- ✅ 每个步骤目标明确
- ✅ 引导问题具体可操作

### 2. 细节挖掘维度 ⭐⭐⭐⭐⭐

**5 个维度**：
1. **内容叙事** - 故事起承转合、关键情节、角色
2. **视觉风格** - 风格、色调、比例
3. **镜头语言** - 景别、运镜、转场
4. **动作与节奏** - 主体动作、节奏、卡点
5. **声音设计** - 配乐、音效、对白

**可借鉴点**：
- ✅ 覆盖全面，不遗漏重要信息
- ✅ 专业术语 + 通俗解释
- ✅ 每个维度都有具体问题

### 3. 提示词结构模板 ⭐⭐⭐⭐⭐

**标准结构**：
```
【整体描述】风格 + 时长 + 画面比例 + 整体氛围

【分镜描述】
0-X 秒：[镜头运动]，[画面内容]，[主体动作]，[光影/特效]
X-Y 秒：[镜头运动]，[画面内容]，[主体动作]，[光影/特效]
...

【声音说明】配乐风格/音效/对白

【参考素材说明】（如有）
@图片 1 作为首帧/角色参考
@视频 1 参考运镜/动作
@音频 1 用于配乐/对白参考
```

**可借鉴点**：
- ✅ 结构清晰，层次分明
- ✅ 时间轴标注明确
- ✅ 支持多模态引用（@语法）
- ✅ 包含声音设计（我们缺少）

### 4. 实战案例库 ⭐⭐⭐⭐⭐

**6 个典型案例**：
1. 复杂运镜复刻（希区柯克变焦）
2. 角色一致性叙事
3. 视频延长创作
4. 创意特效复刻
5. 商业广告成片
6. 多场景无缝转场

**可借鉴点**：
- ✅ 覆盖常见使用场景
- ✅ 每个案例有完整提示词
- ✅ 包含素材说明和用法

### 5. 高级使用技巧 ⭐⭐⭐⭐⭐

**特殊场景用法**：
- 首帧 + 视频动作
- 视频延长
- 多视频融合
- 参考视频声音
- 连续动作
- 角色替换
- 完整运镜复刻

**多模态组合策略**：
- 一致性控制
- 运镜动作复刻
- 创意特效复刻

**可借鉴点**：
- ✅ 高级功能覆盖全面
- ✅ 提供具体写法示例
- ✅ @语法使用规范

### 6. 平台限制提醒 ⭐⭐⭐⭐

**明确告知**：
- ⚠️ 暂不支持写实真人脸部
- ⚠️ 视频参考消耗更多额度
- ⚠️ 混合输入总上限 12 个文件
- ⚠️ 视频像素数范围

**可借鉴点**：
- ✅ 提前管理用户预期
- ✅ 避免违规和失败
- ✅ 成本透明

## 📝 具体改进计划

### 改进 1：需求收集表单 ⭐

**新增问题**：
```
【镜头语言】
- 希望用什么运镜方式？（推/拉/摇/移/环绕）
- 需要景别变化吗？（远景→全景→中景→近景→特写）

【动作节奏】
- 主体有什么动作？（行走/奔跑/舞蹈/表情）
- 节奏是舒缓还是急促？

【声音设计】（虽然不能上传音频，但可以描述）
- 期望的配乐风格？
- 需要音效吗？
```

### 改进 2：提示词模板 ⭐

**新增模板**：
```python
def generate_prompt(scene, style, camera, action, sound):
    prompt = f"""【整体描述】{style}风格，{scene['duration']}秒，{scene['ratio']}，{style['atmosphere']}

【分镜描述】
{scene['time_range']}：{camera['movement']}，{scene['description']}，{action['main_action']}，{style['lighting']}

【声音说明】{sound['music']}，{sound['sfx']}
"""
    return prompt
```
