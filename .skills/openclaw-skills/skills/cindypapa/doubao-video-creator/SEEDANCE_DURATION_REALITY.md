## 💡 推荐策略

### 统一使用 5 秒
```python
# 推荐配置
duration = 5  # 固定 5 秒
```

**原因**：
1. ✅ **稳定性最好** - 几乎不会失败
2. ✅ **质量最高** - 动作流畅，画面稳定
3. ✅ **生成最快** - 1-2 分钟完成
4. ✅ **成本最优** - 单次成本最低

### 长视频方案：多场景合成
```
总时长 30 秒 = 6 个场景 × 5 秒
总时长 60 秒 = 12 个场景 × 5 秒
```

**优势**：
- 每个场景都稳定
- 可以逐个确认质量
- 不满意只重做那个场景
- 总体成功率更高

### 场景规划逻辑
```python
def calculate_scenes(total_duration):
    """
    根据总时长计算场景数
    
    Args:
        total_duration: 用户希望的总时长（秒）
        
    Returns:
        scene_count: 场景数量
        scene_duration: 固定 5 秒
    """
    scene_duration = 5  # 固定 5 秒
    scene_count = math.ceil(total_duration / scene_duration)
    
    return scene_count, scene_duration
```
