# 镜头语言与提示词模板库

## 📐 镜头语言词库

### 1. 景别（Shot Size）

| 景别 | 说明 | 适用场景 |
|------|------|---------|
| **远景** (Extreme Long Shot) | 展示大场景，人物很小 | 建立场景、环境氛围 |
| **全景** (Long Shot) | 人物全身，场景完整 | 展示人物与环境关系 |
| **中景** (Medium Shot) | 人物腰部以上 | 对话、动作展示 |
| **近景** (Close-Up) | 人物胸部以上 | 表情、情感 |
| **特写** (Extreme Close-Up) | 局部细节（眼睛、手） | 强调细节、情感高潮 |

**使用示例**：
```
远景缓慢推近，展示城市天际线
全景环绕镜头，人物站在场景中央
中景跟随移动，展示人物动作
近景聚焦面部，捕捉微妙表情
特写手部动作，展示产品细节
```

### 2. 运镜方式（Camera Movement）

| 运镜 | 说明 | 效果 |
|------|------|------|
| **推** (Dolly In) | 镜头向前推进 | 强调主体、紧张感 |
| **拉** (Dolly Out) | 镜头向后拉远 | 展示环境、结束感 |
| **摇** (Pan) | 镜头左右摇动 | 展示横向空间 |
| **移** (Truck) | 镜头横向移动 | 跟随主体、流畅感 |
| **跟** (Follow) | 镜头跟随主体 | 沉浸感、参与感 |
| **环绕** (Arc) | 镜头绕主体旋转 | 展示全貌、戏剧性 |
| **升降** (Pedestal) | 镜头上下移动 | 揭示、转换视角 |
| **希区柯克变焦** (Dolly Zoom) | 推拉 + 变焦 | 眩晕、紧张效果 |

**使用示例**：
```
镜头从下往上缓缓推近，聚焦产品
镜头缓慢拉远，展示完整场景
镜头左右摇摄，展示环境全貌
镜头横向跟随移动，保持主体居中
镜头环绕主体旋转，展示 360 度视角
镜头从低角度升降到高角度
希区柯克变焦，营造眩晕紧张感
```

### 3. 转场方式（Transition）

| 转场 | 说明 | 适用场景 |
|------|------|---------|
| **硬切** (Cut) | 直接切换 | 常规转场 |
| **渐变** (Fade) | 渐隐渐显 | 时间流逝、情绪转换 |
| **匹配剪辑** (Match Cut) | 形状/动作匹配 | 创意转场、建立联系 |
| **特效转场** | 特殊效果 | 科幻、奇幻风格 |
| **无缝渐变** | 自然过渡 | 一镜到底效果 |

**使用示例**：
```
硬切到下一个场景
渐变到黑色，再渐显
匹配剪辑：圆形物体切换到圆形物体
特效转场：粒子消散、碎片重组
无缝渐变：海浪变成飘动的窗帘
```

### 4. 特殊效果（Special Effects）

| 效果 | 说明 | 示例 |
|------|------|------|
| **一镜到底** | 无剪辑连续拍摄 | 长镜头、连贯叙事 |
| **慢动作** | 降格拍摄 | 强调瞬间、戏剧性 |
| **延时摄影** | 加速时间流逝 | 云卷云舒、城市流动 |
| **粒子效果** | 粒子消散/聚集 | 魔法、科幻 |
| **光影流转** | 光线变化 | 氛围、情绪 |

## 🎯 提示词优化技巧

### 1. 时间轴清晰
```
✅ 好：0-5 秒：特写镜头...
❌ 差：开始是特写...
```

### 2. 镜头语言明确
```
✅ 好：中景环绕镜头，展示 360 度视角
❌ 差：镜头转一转
```

### 3. 动作描述具体
```
✅ 好：手指轻触屏幕，向上滑动切换功能
❌ 差：用手操作
```

### 4. 光影效果详细
```
✅ 好：自然光从左侧窗户射入，形成柔和侧光
❌ 差：有光线
```

### 5. 质量词加持
```
✅ 好：高清写实，电影级画质，4K 细节
❌ 差：清晰的
```
