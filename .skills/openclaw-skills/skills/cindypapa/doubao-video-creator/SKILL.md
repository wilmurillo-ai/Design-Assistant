# doubao-video-creator - 豆包视频创作助手 v2.0

## 📋 技能描述

使用火山引擎豆包（Doubao Seedance）视频生成模型，帮助用户创作专业视频。支持**项目化配置管理**、**分场景生成方式选择**、**完整状态追踪**。

   - 产品图片
   - 参考场景

   - 具体需求说明

### 阶段 2：脚本规划
1. 分析参考资料
2. 生成视频场景规划脚本（默认 12 秒）
3. 发送脚本给用户确认
4. 根据反馈调整脚本

### 阶段 2.5：关键元素生成
1. **提取关键元素**：从脚本中识别人物、场景、物品
2. **生成参考图**：用通义万相生成关键元素图片
3. **用户确认**：发送参考图给用户确认
4. **调整优化**：根据反馈重新生成参考图

### 阶段 3：分场景生成方式确认（新增！）✅

**对每个场景，单独确认生成方式**：

```
🎬 场景 1 确认（0-4 秒）：开场展示

请选择生成方式：
  A) 文生视频 - 直接用文字提示词生成
  B) 图生视频 - 需要先生成场景/人物原型图

当前场景描述：手表特写，镜头从下往上扫过
```

**如果选择 B) 图生视频**：
- 自动生成该场景的人物/场景原型图
- 用户确认原型图
- 使用原型图 + 提示词生成视频

**如果选择 A) 文生视频**：
- 直接使用文字提示词生成视频

### 阶段 4：提示词确认（关键！）✅

**⚠️ 重要：必须先确认提示词，才能生成视频！**

对每个场景：
1. 生成视频提示词（基于脚本和参考图）
2. 发送提示词给用户确认
   - 展示完整提示词
   - 说明将生成的画面内容
   - 显示预计成本
   - 提醒确认后无法撤销
3. **用户确认提示词后**才进入下一步
4. 如用户要求修改，调整提示词后重新确认

**为什么这步很重要？**
- ✅ 避免生成不符合预期的视频
- ✅ 减少反复生成的成本浪费
- ✅ 确保每个场景一次成功
- ✅ 提高整体制作效率

### 阶段 5：分场景生成（逐个进行！）✅

**⚠️ 重要：逐个场景生成，不要批量！**

对每个场景按顺序执行：
1. **检查提示词是否已确认**
   - 未确认 → 返回阶段 4
   - 已确认 → 继续

2. **调用豆包 API 生成视频**（4 秒/场景）
   - 文生视频：只用提示词
   - 图生视频：提示词 + 原型图

3. **等待生成完成**（约 1-2 分钟）

4. **发送视频给用户确认**
   - 展示生成的视频
   - 询问是否满意

5. **用户确认**
   - 满意 → 标记为 confirmed，继续下一个场景
   - 不满意 → 记录反馈，修改提示词后重新生成

6. **完成一个场景后再开始下一个**
   - 不要同时生成多个场景
   - 确保每个场景都确认后再继续

**为什么逐个生成？**
- ✅ 及时发现问题，避免全部生成后才发现问题
- ✅ 减少浪费，一个场景不满意只重生成这一个
- ✅ 用户参与度高，可以随时调整方向
- ✅ 降低总体成本

### 阶段 6：视频合成
1. 将所有确认的场景视频按顺序合成
2. 添加转场效果
3. 添加背景音乐（可选）
4. 发送最终视频给用户

## ⚙️ 配置参数

### 首次使用配置（v2.0 新增）

**全局配置文件**: `~/.openclaw/workspace/doubao-config.json`

```json
{
  "default_api_key": "YOUR_API_KEY",
  "default_text_to_video_model": "doubao-seedance-2-0-pro",
  "default_image_to_video_model": "doubao-seedance-1-5-pro",
  "last_updated": "2026-03-31T12:00:00",
  "is_configured": true
}
```

**配置项说明**:
- `default_api_key`: 火山引擎豆包 API Key
- `default_text_to_video_model`: 文生视频默认模型
- `default_image_to_video_model`: 图生视频默认模型

**可用模型列表**:
| 模型 ID | 版本 | 推荐 | 说明 |
|--------|------|------|------|
| `doubao-seedance-2-0-pro` | 2.0 | ✅ | 最新版本，效果最佳 |
| `doubao-seedance-1-5-pro` | 1.5 | ✅ | 稳定版本，广泛使用 |
| `doubao-seedance-1-0-pro` | 1.0 | ❌ | 基础版本，兼容性好 |

### 图片生成参数（通义万相）
- **分辨率**: 1024x1024
- **风格**: 写实/插画/3D（根据视频风格）
- **数量**: 每个元素生成 2-3 个版本供选择

### 视频生成参数
- **时长**: 每个场景 2-12 秒（推荐 4 秒）
- **分辨率**: 480p / 720p / 1080p（推荐 720p）
- **比例**: 16:9 / 9:16 / 1:1（推荐 16:9）
- **帧率**: 24 fps
- **格式**: MP4

### 生成方式选择
- **文生视频**: 直接使用文字提示词生成
- **图生视频**: 使用原型图 + 提示词生成（画面更可控）
