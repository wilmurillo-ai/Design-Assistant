# 豆包视频创作助手 v2.1 - 快速参考

## 🎯 核心改进（3 个）

### 1. 专业提示词生成器 ⭐⭐⭐⭐⭐
```python
from prompt_generator import PromptGenerator

generator = PromptGenerator()
prompt = generator.generate_scene_prompt(
    scene=scene,
    style=style,
    camera=camera,  # 镜头语言
    action=action,  # 动作描述
    sound=sound     # 声音设计
)
```

**输出结构**：
```
【整体描述】风格 + 时长 + 比例 + 氛围
【分镜描述】0-5 秒：[景别][运镜]，[内容]，[动作]，[光影]，[质量]
【声音说明】配乐/音效
【参考素材】@图片 1 作为...
```

### 2. 镜头语言词库 ⭐⭐⭐⭐⭐
**景别**：远景、全景、中景、近景、特写  
**运镜**：推、拉、摇、移、跟、环绕、升降、希区柯克变焦  
**转场**：硬切、渐变、匹配剪辑、特效转场  
**质量词**：高清写实、电影级画质、4K 细节  

### 3. 时长自动计算 ⭐⭐⭐⭐
```
场景数量 = 总时长 ÷ 5（固定）

30 秒 → 6 个场景
60 秒 → 12 个场景
```

## 📋 标准流程（10 步）

```
1. 首次配置 → API Key + 模型
2. 需求收集 → 主题 + 资料
3. 技术参数 → 总时长/比例/画质
4. 脚本规划 → 五步法（内容/视觉/镜头/动作/声音）
5. 关键元素 → 参考图生成
6. 生成方式 → 文生/图生选择
7. 提示词生成 ⭐ → 使用 prompt_generator
8. 提示词确认 ⚠️ → 必须确认
9. 分场景生成 ✅ → 逐个确认
10. 视频合成 → 交付
```

## ⚠️ 关键检查点

### 提示词检查清单
- [ ] 时间轴清晰（0-5 秒，5-10 秒...）
- [ ] 镜头语言明确（特写/推镜头/环绕）
- [ ] 动作描述具体（避免"操作"等模糊词）
- [ ] 光影效果详细（侧光/逆光/自然光）
- [ ] 质量词加持（高清/电影级/4K）

### 生成前检查
- [ ] 提示词已确认（用户回复 A）
- [ ] 参考图已确认（图生视频模式）
- [ ] 参数正确（时长/比例/画质）
- [ ] 成本已告知（¥0.15-0.50/场景）

## 💡 快速技巧

### 1. 快速选择配置

### 2. 提示词优化
```
太暗 → 替换为"明亮" + "光线充足"
不清晰 → 添加"超清 4K 画质，细节清晰"
节奏快 → 替换"快速"为"缓慢"
```

```

```
