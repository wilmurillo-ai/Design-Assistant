# 摘要模板（通用版 V2）

适用于：推文系列、教程系列、报告系列、小说系列

---

## 第N段摘要

### 必须包含

#### 1. 关键内容
- 核心观点/事件：[本段要传达的核心]
- 新增信息：[本段引入的新内容]
- 结论/转折：[本段的收尾]

#### 2. 状态快照

| 对象 | 当前状态 | 变化 |
|------|---------|------|
| [主角/主题] | [当前状态] | [+/-变化] |

#### 3. 关键数据追踪
- 时间节点：[当前时间]
- 资金/数量：[具体数字]
- 其他指标：[需要追踪的数据]

#### 4. 未收伏笔
- [ ] 本段埋下的线索：[具体内容]
- [ ] 前文伏笔待收：[具体内容]

#### 5. 关键细节（需后文呼应）
- [细节1]：[具体描述]
- [细节2]：[具体描述]

#### 6. 下段预告
- 预期方向：[下段计划写什么]
- 待解决问题：[需要回答的问题]
- 建议切入点：[下段可以从哪里开始]

---

## 质量标准

**必须包含**：
- [x] 关键内容
- [x] 状态快照
- [x] 未收伏笔
- [x] 下段预告

**字数要求**：300-500字

**禁止**：歧义表述、遗漏关键信息

---

## 连贯性验证

加载摘要后必须确认：
- [ ] 核心对象状态理解正确
- [ ] 时间线连续
- [ ] 数据一致性
- [ ] 伏笔未遗漏

---

**使用方式**：段落完成后，按此模板生成摘要。
