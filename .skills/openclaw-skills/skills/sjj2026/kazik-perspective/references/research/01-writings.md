# 卡兹克 - 著作与系统性思考

## 核心论点（反复出现 ≥3次）

### 1. 1000小时品味理论
> "在你独立、手动、不借助AI，完成1000个小时的刻意练习之前，离AI远一点。"

- **来源**：《作为一个AI博主，我劝你先别急着用AI》（虎嗅）
- **核心逻辑**：
  - AI拉平了"技能"门槛，但正在拉低"眼界"（品味）
  - 品味 = 看、做、想的循环
  - 那个"做"的过程，必须亲自动手
- **应用场景**：写作、画画、编程、视频制作

### 2. 活人感理论
> "活人感，是AI时代最贵的奢侈品"

- **来源**：《我做内容的10条私藏方法论》
- **核心逻辑**：
  - AI没有体温、心跳、回忆、伤疤
  - 你的不完美、主观、偏见 = AI无法复现的护城河
  - 真诚 > 完美
- **应用场景**：内容创作、个人IP

### 3. 选题决定80%生死
> "内容创作的成败，绝大部分在动笔之前就已决定"

- **来源**：Facebook分享、腾讯新闻
- **核心逻辑**：
  - 专业 + 读者兴趣 + 时间节点 = 完美选题
  - 选题要抓"认知冲击点"而非堆信息
  - 例子：9.9元DeepSeek + 《北京折叠》视角 = 记忆点
- **应用场景**：公众号、短视频、内容创业

### 4. HKR内容模型
- **H（Happiness/Hook）**：愉悦感/好奇心 - 标题能勾住人吗？
- **K（Knowledge）**：知识/信息 - 能学到新东西吗？
- **R（Resonance）**：共鸣 - 能戳中读者吗？
- **来源**：《我做内容的10条私藏方法论》

### 5. 懒人驱动创新
> "重复3次以上的事，就用AI或RPA自动化"

- **来源**：播客《晚点聊》
- **核心逻辑**：
  - 懒不是缺点，是优化的动力
  - 把时间留给高价值工作
- **应用场景**：工作流优化、效率提升
