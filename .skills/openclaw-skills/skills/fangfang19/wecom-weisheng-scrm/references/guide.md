# 微盛企微管家SCRM 使用参考

本 Skill 适用于企业微信客户运营与管理场景，可用于查询或管理客户信息、客户标签、客户群、素材、活码、群发、跟进记录、会话存档、联系人、商机、产品库、日程等相关能力。

如需人工协助，可扫码联系专属客服：<https://pubres.wshbe.com/qrcode/20260416-53d7fee83fd04dbab3101546559490a6.png>

建议使用流程：

1. 先确认已完成企微管家授权并配置 APP KEY。
2. 如当前在 WorkBuddy 的 Ask 模式且无法执行命令，请先切换到 Craft（或其他具备命令执行能力的模式）。
3. 根据要处理的业务场景提出问题，例如客户查询、群发查看、活码查看、素材处理等。
4. 如需更多条件，按提示补充客户名称、时间范围、标签、员工等业务信息。
5. 查询结果优先查看结论；写操作需在确认后再执行。

补充约束：

- 读取接口目录时，必须直接获取目标地址的原始内容，并统一使用 `python3 scripts/scrm.py fetch-raw-doc --url <url>`。
- 不要使用 `web_fetch` 或其他网页抓取方式读取 open.wshoto.com 文档。
- 不要仅基于首页、搜索结果或网页摘要推断接口。

常见场景包括：

- 查询客户信息与客户标签
- 查看客户群、群发和跟进记录
- 查看会话存档与联系人情况
- 处理活码、素材、商机、产品库和日程相关事项
