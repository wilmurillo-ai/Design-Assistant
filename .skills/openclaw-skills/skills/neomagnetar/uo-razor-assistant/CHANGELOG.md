# Changelog

## 1.0.0 - 2026-04-12

- Initial release of the **UO Razor Assistant** skill.
- Provides slash-invocable guidance for Ultima Online Razor macros, scripts, hotkey bindings, profile XML, and Outlands differences.
- Distinguishes between macro files, Razor scripts, and profile XML, and advises on the proper artifact type for each use case.
- Cautions against inventing internal hotkey IDs and encourages in-client verification when mappings are unclear.
- Highlights Outlands-specific commands and PvP automation restrictions where relevant.
- Provides copy-paste-ready artifacts and validation guidance for user changes.
- Emphasizes accuracy, validation, and honest handling of version and shard uncertainty.
