# ChatDecode
> AI message decoder — reveals the real meaning behind any text message.

An [OpenClaw](https://github.com/openclaw/openclaw) Skill that analyzes tone, subtext, emotional state, and hidden intentions in text conversations.

## Features

- **🔍 Subtext Analysis** — What they really mean vs. what they wrote
- **🎭 Emotional State Mapping** — Detect frustration, anxiety, distance, or genuine engagement
- **📊 Confidence Scoring** — Know how certain the interpretation is
- **👥 Group Chat Dynamics** — Read who responds to whom, who gets ignored, clique patterns, and passive-aggressive group energy
- **👁️ Read Receipt Analysis** — Distinguish "read but no reply" from genuine busyness using timing, context, and reply patterns
- **🌐 Cultural Decode** — Chinese texting norms (嗯 vs 嗯嗯, 哦 vs 哦哦) and English patterns (k. vs K vs ok!)
- **📱 Platform Aware** — WeChat, WhatsApp, iMessage, Telegram, Slack, DMs

## Quick Start

```bash
clawhub install chatdecode
```

Then paste any message and ask what it means.
