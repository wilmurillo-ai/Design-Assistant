# Batch Video Creator

> Produce dozens of unique short videos from a single source in one run — with automated caption generation, hook rotation, color grading, voiceover variation, and platform-specific formatting. The content factory for teams that need volume without sacrificing quality.

## What This Skill Does

Batch Video Creator is the mass-production engine behind content-heavy marketing strategies. Feed it one hero video and a set of parameters, and it outputs 10, 20, or 50+ unique variants — each with different hooks, captions, color treatments, and pacing. Every video passes platform deduplication checks thanks to a 6-layer uniqueness system.

### Core Capabilities

1. **One-to-Many Generation** — 1 source → 10-50+ unique outputs with configurable variation depth
2. **Hook Library** — Rotate through proven hook templates (question, statistic, controversy, story)
3. **Caption Automation** — AI-written captions with blue-word strategy, emoji placement, and hashtag optimization
4. **Voiceover Variants** — Multiple TTS voices and pacing styles for the same script
5. **Color Grade Rotation** — Warm, cool, vintage, high-contrast — each variant gets a different treatment
6. **Video Deduplication** — Prompt mutation + color shift + speed jitter + crop variation + subtitle style + fingerprint perturbation
7. **Batch Export** — All variants exported with platform-correct specs, named and organized

### Production Scale

| Tier | Input | Output | Time |
|------|-------|--------|------|
| Quick | 1 video | 5 variants | ~10 min |
| Standard | 1 video | 15 variants | ~30 min |
| Campaign | 3 videos | 50+ variants | ~2 hours |
