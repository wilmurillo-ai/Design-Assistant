# SocialCoach
> AI social skills coach — handle any social situation with grace.

An [OpenClaw](https://github.com/openclaw/openclaw) Skill for navigating friendships, family dynamics, and everyday social interactions. Decline invitations gracefully, comfort friends genuinely, handle toxic people, and never be awkward again.

## Features

- **💬 5-Style Replies** — Polite / warm / funny / honest / casual
- **🆘 Awkward Situations** — Decline invites, dodge money requests, exit boring convos
- **💔 Comfort Mode** — Actually helpful responses when someone is hurting (not "everything happens for a reason")
- **🎭 Conflict Resolution** — Friends, family, acquaintances, online drama
- **🌐 Cultural Awareness** — Chinese social norms (面子, 客气, 催婚) and Western etiquette

## Quick Start

```bash
clawhub install socialcoach
```

Describe any social situation and get actionable advice with ready-to-send messages.
