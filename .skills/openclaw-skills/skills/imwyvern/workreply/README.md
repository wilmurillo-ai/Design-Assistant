# WorkReply
> AI workplace communication assistant — never send the wrong email again.

An [OpenClaw](https://github.com/openclaw/openclaw) Skill for crafting professional replies to bosses, clients, and colleagues. Handles difficult emails, salary negotiation, declining tasks, and delivering bad news.

## Features

- **💬 5-Style Replies** — Safe / concise / diplomatic / assertive / strategic
- **🛡️ Difficult Situations** — Decline tasks, deliver bad news, handle blame, negotiate salary
- **📊 Hierarchy Aware** — Adapts tone for boss, peer, report, client, or HR
- **🌐 Bilingual** — Chinese workplace culture (面子, 委婉) and Western corporate norms
- **⏰ Deadline Management** — Buy time, push back, escalate professionally
- **🎤 Meeting Speaking Templates** — Ready-to-say scripts for status updates, pushback, clarification, and disagreement

## Quick Start

```bash
clawhub install workreply
```

Paste any work message and get 5 professionally crafted reply options.
