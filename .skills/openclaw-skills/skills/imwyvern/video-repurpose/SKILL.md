# Video Repurpose

> Transform one video into multiple platform-optimized versions — remix, resize, re-caption, and re-voice for TikTok, Instagram Reels, YouTube Shorts, Xiaohongshu, and Douyin. Turn a single piece of content into a full cross-platform distribution strategy.

## What This Skill Does

Video Repurpose takes any source video and creates platform-native variants. It handles aspect ratio conversion (16:9 → 9:16 smart crop), caption restyling, hook rotation, color grade variation, and voiceover localization — all while maintaining a 6-layer deduplication system so each variant passes platform uniqueness checks.

### Core Capabilities

1. **Smart Aspect Ratio Conversion** — Intelligent cropping that follows the action (not just center-crop)
2. **Hook Rotation** — Generate multiple opening hooks for the same content to test what converts
3. **Caption Restyling** — Different subtitle styles (burned-in, floating, karaoke, minimal) per platform
4. **Voiceover Localization** — Replace narration in different languages while preserving BGM
5. **Color & Speed Variation** — Subtle shifts that make each variant unique to platform algorithms
6. **Anti-Duplicate Engine** — 6 layers: prompt mutation, color shift, speed jitter, crop variation, subtitle style rotation, fingerprint perturbation

### Supported Platforms

| Platform | Aspect | Max Length | Caption Style |
|----------|--------|-----------|---------------|
| TikTok | 9:16 | 60s | Bold, emoji-rich |
| Instagram Reels | 9:16 | 90s | Clean, branded |
| YouTube Shorts | 9:16 | 60s | Minimal |
| Xiaohongshu | 3:4 / 9:16 | 60s | Chinese blue-word |
| Douyin | 9:16 | 60s | Trending format |
