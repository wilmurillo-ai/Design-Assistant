### 藏干力量分配原则

**本气（60-100%）：**
- 地支的主要五行属性
- 力量最强，主导地支特性
- 决定地支的基本象意

**中气（30-40%）：**
- 地支的次要五行
- 力量中等，影响地支变化
- 丰富地支的内涵

**余气（10%）：**
- 上一季节的残留之气
- 力量最弱，但有特殊作用
- 常用于应期判断

## 五、藏干应用法则

### 1. 透干法则

**透干：** 地支藏干出现在天干

```
例：地支有寅，天干见甲
- 甲木透干，寅木力量显现
- 甲木代表的事物会明显

例：地支有巳，天干见庚
- 庚金透干，巳中庚金力量显现
- 庚金代表的事物会明显
```

**不透干：** 藏干深藏，力量内敛

```
例：地支有辰，天干不见戊乙癸
- 藏干深藏，力量内敛
- 需要大运流年引动
```

### 2. 通根法则

**通根：** 天干在地支有同五行支撑

```
例：天干甲木，地支有寅卯
- 甲木通根，力量强
- 甲木代表的事物稳固

例：天干庚金，地支有申酉
- 庚金通根，力量强
- 庚金代表的事物稳固
```

**不通根：** 天干虚浮，力量弱

```
例：天干甲木，地支无寅卯亥未
- 甲木虚浮，力量弱
- 甲木代表的事物不稳定
```

### 3. 藏干取用法

**取本气：** 一般情况以本气为主

```
例：日支辰土
- 本气戊土为主导
- 配偶性格稳重包容
```

**取中气：** 特殊格局或中气透干

```
例：日支辰土，乙木透干
- 乙木力量显现
- 配偶性格灵活有生机
```

**取余气：** 应期判断或特殊组合

```
例：日支辰土，流年癸水
- 癸水为余气，被引动
- 该年有水相关应事
```

## 十、注意事项

1. **藏干力量**：本气>中气>余气
2. **透干优先**：藏干透干则力量显现
3. **通根重要**：天干通根则力量强
4. **库墓特殊**：库墓需开才能用
5. **综合判断**：不可单凭藏干断事
