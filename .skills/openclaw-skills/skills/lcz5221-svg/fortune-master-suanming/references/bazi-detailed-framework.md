# 八字精细化分析框架（六经汇通版）

## 一、基础框架要素（命盘根基）

### 1. 阴阳五行
- 分析八字中阴阳五行分布情况
- 五行强弱与平衡状况
- 相生相克关系对命主的影响
- 寒暖燥湿的平衡状态（参考《穷通宝鉴》调候理论）

### 2. 天干地支
- 天干透出与地支藏干分析
- 天干五合、地支六合、三合、三会
- 天干地支之间的相互作用

### 3. 四柱八字
- 年柱：祖上、少年、根基、早年运势
- 月柱：父母、青年、提纲（月令用神）、中年运势
- 日柱：自身（日主）、配偶、中年、夫妻宫
- 时柱：子女、晚年、归宿、未来运势

### 4. 地支藏干
- 精确分析每个地支的本气、中气、余气
- 藏干对十神配置的影响
- 藏干透出时的动态变化

### 5. 节气与旺衰
- 准确按节气划分月令
- 日主在各月令中的旺衰状态
- 五行在不同季节的休囚旺相

## 二、核心分析要素（论命关键）

### 1. 十神关系
- 以日主为中心构建十神体系
- 正官：权威、事业、领导、夫星（女命）
- 偏官：威权、竞争、男命子女、女命丈夫
- 正财：财富、妻财、物质享受
- 偏财：意外之财、父缘、男命妻缘
- 正印：学识、母亲、保护、男命妻子
- 偏印：学术、继母、男命妾缘、女命父缘
- 食神：才华、福气、男命女儿、快乐
- 伤官：才华、口舌、女命儿子、聪明
- 比肩：兄弟、朋友、助力
- 劫财：劫夺、竞争、兄弟

### 2. 格局分析
- **正格**（参考《子平真诠》格局理论）：
  - 正官格、偏官格
  - 正财格、偏财格
  - 正印格、偏印格
  - 食神格、伤官格
  
- **特殊格局**（参考《滴天髓》《神峰通考》）：
  - 专旺格：从旺格、从强格
  - 从格：从财格、从杀格、从儿格
  - 其他变格

### 3. 用神体系
- **用神**：平衡命局的关键五行，分格局用神、调候用神、通关用神
- **喜神**：辅助用神发挥作用的五行
- **忌神**：破坏命局平衡的五行
- **闲神**：中性作用，可转化为喜用
- **仇神**：助长忌神的五行

### 4. 调候与通关
- **调候**（《穷通宝鉴》核心）：
  - 寒暖调候：冬月需暖，夏月需寒
  - 燥湿调候：四季土旺需润
- **通关**：
  - 解决五行相战（木土战、火金战、土水战等）
  - 引导五行流通，化敌为友

## 三、辅助判断要素（细节补充）

### 1. 刑冲合会
- **地支六冲**：子午冲、丑未冲、寅申冲、卯酉冲、辰戌冲、巳亥冲
- **地支六害**：子未害、丑午害、寅巳害等
- **地支三刑**：寅巳申、丑戌未、子卯相刑、辰午酉亥自刑
- **天干五合**：甲己合、乙庚合、丙辛合等
- **地支三合**：申子辰、亥卯未、寅午戌、巳酉丑
- **地支半合**：拱合、拱会
- **三会局**：寅卯辰、巳午未、申酉戌、亥子丑

### 2. 神煞应用
- **吉神**：天德贵人、月德贵人、文昌贵人、天乙贵人等
- **凶神**：羊刃、七杀、孤辰寡宿、亡神等
- **特异神煞**：驿马、桃花、华盖、太极贵人等
- **参考《三命通会》**中各类神煞应用，但不为主

### 3. 大运与流年
- **大运**：十年一换，影响人生阶段
- **流年**：一年之运，与命局、大运三方作用
- **小运/流月**：短期运势变化
- **岁运并临**：天干地支相同，力量加强
- **冲克大运**：运势转折点

### 4. 六亲定位
- **父亲**：偏财（男命）、食伤（女命）
- **母亲**：正印（男女皆同）
- **配偶**：正官/七杀（女命）、正财/偏财（男命）
- **子女**：食神、伤官（男命子女看官杀）
- **兄弟姐妹**：比肩、劫财

### 5. 宫位与限运
- **四柱宫位**：年月日时各代表不同人生层面
- **十二宫位**：参考紫微斗数宫位理念（可选）
- **分限运**：不同年龄阶段的运势重点

## 四、分析流程

### 1. 命局总览
- 整体格局分析
- 五行分布比例
- 阴阳平衡状况

### 2. 旺衰判断
- 日主强弱分析
- 用神喜忌确定
- 调候需求判断

### 3. 十神配置
- 十神组合分析
- 六亲关系推断
- 性格特征分析

### 4. 命运走向
- 大运流年配合分析
- 吉凶祸福预测
- 时机把握建议

## 五、注意事项

1. **融汇各家**：结合《渊海子平》、《三命通会》、《子平真诠》、《滴天髓》、《穷通宝鉴》、《神峰通考》的精髓
2. **灵活应用**：不可拘泥一法，需活法用神
3. **客观公正**：避免宿命论色彩，注重启发指导
4. **分寸把握**：不涉及健康疾病、生死时限等医学范畴
5. **人文关怀**：注重心理疏导和人生指导
