# 八字格局分析框架

## 一、格局判定流程

### 第一步：定月令格局

**月令取格法则：**

| 月支 | 藏干 | 取格优先级 |
|------|------|-----------|
| 寅 | 甲丙戊 | 甲木→丙火→戊土 |
| 卯 | 乙 | 乙木 |
| 辰 | 戊乙癸 | 戊土→乙木→癸水 |
| 巳 | 丙庚戊 | 丙火→庚金→戊土 |
| 午 | 丁己 | 丁火→己土 |
| 未 | 己丁乙 | 己土→丁火→乙木 |
| 申 | 庚壬戊 | 庚金→壬水→戊土 |
| 酉 | 辛 | 辛金 |
| 戌 | 戊辛丁 | 戊土→辛金→丁火 |
| 亥 | 壬甲 | 壬水→甲木 |
| 子 | 癸 | 癸水 |
| 丑 | 己癸辛 | 己土→癸水→辛金 |

**取格原则：**
1. 月令本气透干 → 取本气格
2. 本气不透，中气透 → 取中气格
3. 本气中气都不透 → 取本气格（藏格）
4. 月令被冲克 → 格局有破

---

### 第二步：辨格局真假

**真格条件：**
1. 用神得月令
2. 用神透干或有根
3. 无严重冲克破格
4. 相神有力配合

**假格条件：**
1. 用神不得月令
2. 用神虚浮无根
3. 冲克破格严重
4. 相神无力或缺失

**破格因素：**
- 用神受冲克
- 相神受损
- 忌神猖獗
- 刑冲破害严重

---

### 第三步：论格局层次

**上格：**
- 用神有力，相神有情
- 无破格因素
- 气势流通
- 调候到位

**中格：**
- 用神有力，但有瑕疵
- 小有破格，但有救应
- 气势基本流通
- 调候部分到位

**下格：**
- 用神无力
- 破格严重，无救应
- 气势阻滞
- 调候不到位

---

## 二、八大正格详解

### 1. 正官格

**成格条件：**
- 月令正官，官星透干
- 身旺能任官
- 有印星化官生身（官印相生）
- 或有财星生官（财官双美）

**破格因素：**
- 伤官见官（无财通关）
- 官杀混杂
- 身弱不胜官
- 官星受冲克

**格局层次：**
- 上格：官印相生，身旺官旺
- 中格：官星有根，身稍弱
- 下格：官杀混杂，伤官见官

**应事：**
- 上格：贵气天成，仕途顺利
- 中格：有职位，但压力大
- 下格：官非口舌，事业波折

---

### 2. 七杀格

**成格条件：**
- 月令七杀，杀星透干
- 身旺能任杀
- 有食神制杀（食神制杀）
- 或有印星化杀（杀印相生）

**破格因素：**
- 杀重身轻
- 无制无化
- 财生杀旺
- 杀星受冲克

**格局层次：**
- 上格：杀印相生，身杀两停
- 中格：有制化，但力量不足
- 下格：杀重身轻，无制无化

**应事：**
- 上格：权柄在手，武职贵格
- 中格：有压力但能扛
- 下格：灾祸不断，身体差

---

### 3. 正印格

**成格条件：**
- 月令正印，印星透干
- 印星有力
- 有官杀生印（官印相生）
- 或有比劫帮身

**破格因素：**
- 财星坏印
- 印星过旺（母慈灭子）
- 印星受冲克
- 身旺印旺无泄

**格局层次：**
- 上格：官印相生，印旺身旺
- 中格：印星有根，身稍弱
- 下格：财坏印，印星受损

**应事：**
- 上格：学业有成，贵人扶持
- 中格：学习好，但助力有限
- 下格：学业受阻，长辈缘浅

---

### 4. 偏印格

**成格条件：**
- 月令偏印，印星透干
- 印星有力
- 有食神吐秀
- 或有官杀生印

**破格因素：**
- 枭神夺食
- 印星过旺
- 财星坏印
- 印星受冲克

**格局层次：**
- 上格：食神吐秀，印旺身旺
- 中格：有食神，但力量不足
- 下格：枭神夺食，印星过旺

**应事：**
- 上格：钻研有成，技术专家
- 中格：有才华但难展现
- 下格：孤独贫困，怀才不遇

---

### 5. 正财格

**成格条件：**
- 月令正财，财星透干
- 身旺能担财
- 有食伤生财（食伤生财）
- 或有官星护财（财官双美）

**破格因素：**
- 比劫夺财
- 身弱不胜财
- 财星受冲克
- 财多身弱

**格局层次：**
- 上格：身旺财旺，食伤生财
- 中格：财星有根，身稍弱
- 下格：比劫夺财，财多身弱

**应事：**
- 上格：富裕之命，财运亨通
- 中格：有财运但辛苦
- 下格：贫困潦倒，财来财去

---

### 6. 偏财格

**成格条件：**
- 月令偏财，财星透干
- 身旺能担财

**破格因素：**
- 比劫夺财

**格局层次：**
- 上格：身旺财旺，食伤生财

---

### 7. 食神格

**成格条件：**
- 月令食神，食神透干
- 食神有力
- 有财星泄食（食伤生财）
- 或有比劫帮身

**破格因素：**
- 印星克食（枭神夺食）
- 食神过旺（泄身过度）
- 食神受冲克

**格局层次：**
- 上格：食神生财，身旺食旺

**应事：**

- 中格：有才华但难展现
- 下格：贫困潦倒，才华受阻

---

### 8. 伤官格

**成格条件：**
- 月令伤官，伤官透干
- 伤官有力
- 有财星泄伤（伤官生财）
- 或有印星制伤（伤官配印）

**破格因素：**
- 伤官见官（无财通关）
- 伤官过旺（泄身过度）
- 伤官受冲克
- 身弱伤旺

**格局层次：**
- 上格：伤官生财/伤官配印

**应事：**

- 下格：是非不断，贫困潦倒

## 三、特殊格局详解

### 从格

**从杀格：**
- 条件：官杀旺极，日主极弱
- 喜：财星、官杀
- 忌：印星、比劫
- 应事：行财官运发财发贵

**从财格：**
- 条件：财星旺极，日主极弱
- 喜：食伤、财星
- 忌：印星、比劫
- 应事：行食财运发财

**从儿格：**
- 条件：食伤旺极，日主极弱
- 喜：食伤、财星
- 忌：印星、官杀
- 应事：行食伤运才华展现

**从强格：**
- 条件：印比旺极，日主极强
- 喜：印星、比劫
- 忌：财星、官杀
- 应事：行印比运顺利

---

### 化气格

**化气条件：**
1. 天干五合
2. 化神得月令
3. 化神透干或有根
4. 无克制化神的五行

**化气类型：**
- 甲己化土
- 乙庚化金
- 丙辛化水
- 丁壬化木
- 戊癸化火

**应事：**
- 真化：富贵之格
- 假化：起伏之命
- 不化：平常之人

---

### 特殊贵格

**魁罡格：**
- 条件：壬辰、庚戌、庚辰、戊戌日
- 特点：性格刚强，适合军警司法
- 应事：不贵则富

**金神格：**
- 条件：乙丑、己巳、癸酉时
- 特点：刚毅果断，有决断力
- 应事：武职贵格

**六阴朝阳格：**
- 条件：六阴日主，时支见子
- 特点：阴柔聪明，有贵人
- 应事：文职贵格

**子午双包格：**
- 条件：子午年午子时，或午年子时
- 特点：贵气非凡
- 应事：大贵之格

## 四、格局应事推断

### 格局成败应事

**成格：**
- 事业：顺利，有成就
- 财运：好，能积累
- 感情：稳定，和睦
- 健康：好，少病

**破格：**
- 事业：波折，难成功
- 财运：差，难积累
- 感情：不顺，多争吵
- 健康：差，多病

### 格局层次应事

**上格：**
- 事业：大成就，社会地位高
- 财运：大富，财富自由
- 感情：美满，配偶优秀
- 健康：长寿，少病

**中格：**
- 事业：中等成就，稳定工作
- 财运：小康，有积蓄
- 感情：一般，偶有争吵
- 健康：一般，偶有小病

**下格：**
- 事业：难成功，工作辛苦
- 财运：贫困，难积蓄
- 感情：不顺，多波折
- 健康：差，多病

## 五、格局与大运配合

### 用神大运

**成格大运：**
- 格局得到加强
- 事业财运好转
- 应吉事

**破格大运：**
- 格局受损
- 事业财运波折
- 应凶事

### 大运应事

| 格局 | 用神大运 | 忌神大运 |
|------|---------|---------|
| 正官格 | 印运、财运 | 伤官运、杀运 |
| 七杀格 | 食神运、印运 | 财运、官杀运 |
| 正印格 | 官杀运、比劫运 | 财运、食伤运 |
| 偏印格 | 食神运、官杀运 | 财运、印运 |
