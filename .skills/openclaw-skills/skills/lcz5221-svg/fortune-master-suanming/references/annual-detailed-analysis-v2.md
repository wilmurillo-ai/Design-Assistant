# 年度详细分析框架 v2（具体年份版）

## 一、分析原则

**核心要求：**
1. 每个维度必须指出**具体年份**
2. 每个维度必须说明**具体变动/事件**
3. 必须结合：八字五行、十神、格局、调候、大运、刑冲合害绝
