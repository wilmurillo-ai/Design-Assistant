# 推理说明框架（增强版）

## 一、核心原则

**每个分析结论后必须说明：**
1. **为什么** - 推理逻辑
2. **依据什么** - 经典依据/命理规则
3. **如何推导** - 分析过程

---

## 二、分析格式模板

```
【分析结论】XXXX

【为什么】
1. 命理规则：XXXX
2. 原局情况：XXXX
3. 推导过程：XXXX

【经典依据】
《书名·卷 X》：「原文」

【实际应事】
具体事件：XXXX
```

## 三、完整分析示例

### 示例 1：格局分析

```
【分析结论】偏印格真格，上格（90 分）

【为什么】
1. 命理规则：
   - 月令藏干透干取格（本气不透取中气）
   - 真格条件：用神得月令、透干、有根、无破格
   - 格局层次评分：用神力量 + 相神配合 + 无破格

2. 原局情况：
   - 月支巳火，藏干丙火（本气）、庚金（中气）、戊土（余气）
   - 丙火（本气）不透
   - 庚金（中气）透年干 ✅
   - 庚金有日支申金为根 ✅
   - 相神辛金透月干配合 ✅
   - 乙木伤官被庚金合，不破格 ✅

3. 推导过程：
   月支藏干 → 本气不透 → 取中气庚金 → 庚金透干有根 → 
   偏印格真格 → 用神得月令 (8 分) + 透干 (10 分) + 有根 (10 分) + 
   相神配合 (9 分) + 无破格 (8 分) = 45/50 = 90 分 → 上格

【经典依据】
《子平真诠·卷二·论偏印格》：「偏印格，要身旺印旺，主聪明多艺，技术成名。」

【实际应事】
- 性格：聪明、钻研、有悟性
- 事业：专业领域有成就，技术成名
- 层次：中上等成就，行业内有影响力
```

### 示例 2：调候分析

```
【分析结论】调候中等偏上（庚金到位，癸水不足）

【为什么】
1. 命理规则：
   - 《穷通宝鉴》调候法则：夏水需金发源，需水帮身
   - 五月壬水，取庚为君（第一用神），取癸为臣（第二用神）
   - 调候层次：两用神都到位=最佳，一到位=中等偏上，都不到位=不足

2. 原局情况：
   - 壬水生于巳月（五月近午月），火旺水弱
   - 年干庚金透干 ✅（第一用神到位）
   - 庚金有日支申金为根 ✅
   - 癸水不透，仅藏于申中 ⚠️（第二用神不足）

3. 推导过程：
   壬水生于巳月 → 火旺水弱 → 需庚金发源、癸水帮身 → 
   庚金透干有根（第一用神到位）+ 癸水藏不透（第二用神不足）→ 
   一等用神到位，二等用神不足 → 中等偏上

【经典依据】
《穷通宝鉴·五月壬水》：「五月壬水，丁旺壬弱，取庚为君，取癸为臣。」

【实际应事】
- 庚金到位：一生有贵人扶持，逢凶化吉
- 癸水不足：需岁运补充，逢金水流年运势最佳
- 调候中等偏上：人生成就中上等，非大富大贵但富足安康
```

## 五、推理说明注意事项

1. **逻辑清晰：** 推理过程要清晰，一步一步推导
2. **依据充分：** 每个结论都要有命理规则或经典依据
3. **过程透明：** 让用户知道是怎么推导出来的
4. **语言通俗：** 白话解释要通俗易懂
5. **多因素分析：** 综合考虑原局、大运、流年配合
