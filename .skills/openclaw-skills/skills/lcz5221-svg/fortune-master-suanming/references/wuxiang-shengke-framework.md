# 五行生克刑冲合害框架

## 一、五行相生相克基础

### 五行相生（母子关系）

```
木 → 火 → 土 → 金 → 水 → 木（循环）

相生关系：
- 木生火：木燃火旺（如甲木生丙火）
- 火生土：火烬土成（如丙火生戊土）
- 土生金：土藏金出（如戊土生庚金）
- 金生水：金凝水聚（如庚金生壬水）
- 水生木：水润木荣（如壬水生甲木）
```

**相生象意：**
- 生我者为**印星**（保护、学习、长辈）
- 我生者为**食伤**（才华、表达、子女）
- 相生有情：关系和谐，助力明显
- 相生无力：有心无力，助力有限

### 五行相克（制约关系）

```
木 → 土 → 水 → 火 → 金 → 木（循环）

相克关系：
- 木克土：木疏土松（如甲木克戊土）
- 土克水：土阻水流（如戊土克壬水）
- 水克火：水灭火焰（如壬水克丙火）
- 火克金：火熔金化（如丙火克庚金）
- 金克木：金伐木折（如庚金克甲木）
```

**相克象意：**
- 克我者为**官杀**（压力、规则、上级）
- 我克者为**财星**（欲望、控制、财富）
- 相克有情：制约得当，成就格局
- 相克无情：克制过度，造成伤害

## 二、五行生克力量等级

### 生克力量评估

| 等级 | 条件 | 力量 | 象意 |
|------|------|------|------|
| **特强** | 得月令 + 多柱帮扶 | ★★★★★ | 主导命局，影响力最大 |
| **强** | 得月令或有根 + 帮扶 | ★★★★☆ | 重要影响力 |
| **中** | 有根但无帮扶 | ★★★☆☆ | 平衡状态 |
| **弱** | 失令无根 | ★★☆☆☆ | 影响力有限 |
| **极弱** | 失令无根 + 受克 | ★☆☆☆☆ | 几乎无影响力 |

### 生克优先级

1. **月令优先**：月支五行力量最强（占 40%）
2. **日支次之**：日支为夫妻宫，力量占 25%
3. **年时再次**：年支、时支各占 15%
4. **天干最弱**：天干各占 5%（但透干为显）

## 三、天干五合

### 天干合化关系

| 合化 | 条件 | 化气 | 象意 |
|------|------|------|------|
| **甲己合** | 甲 + 己 | 化土 | 中正之合，诚信稳重 |
| **乙庚合** | 乙 + 庚 | 化金 | 仁义之合，刚柔并济 |
| **丙辛合** | 丙 + 辛 | 化水 | 威严之合，权威显达 |
| **丁壬合** | 丁 + 壬 | 化木 | 仁寿之合，温和长寿 |
| **戊癸合** | 戊 + 癸 | 化火 | 无情之合，老少配 |

### 合化条件

**真化：**
- 化神得月令（如甲己合，生于辰戌丑未月）
- 化神透干或有强根
- 无克制化神的五行

**假化：**
- 化神不得月令
- 化神无力
- 有克制化神的五行

**合而不化：**
- 只合不化，以绊住论
- 双方力量都受限

### 合化象意

**甲己合（中正之合）：**
- 性格：诚信、稳重、可靠
- 事业：适合管理、公职
- 感情：婚姻关系稳定

**乙庚合（仁义之合）：**
- 性格：刚柔并济、有原则
- 事业：适合法律、外交
- 感情：配偶互补

**丙辛合（威严之合）：**
- 性格：有威严、领导力
- 事业：适合管理、军警
- 感情：配偶有能力

**丁壬合（仁寿之合）：**
- 性格：温和、善良、长寿
- 事业：适合教育、医疗
- 感情：感情和睦

**戊癸合（无情之合）：**
- 性格：现实、功利
- 事业：适合商业、金融
- 感情：年龄差距大或利益结合

## 十、生克应事判断

### 生克应事原则

1. **得令者应吉**：得月令的生克关系力量最大
2. **透干者应显**：天干透出的关系更明显
3. **有根者应实**：地支有根的关系更实在
4. **逢冲者应动**：逢冲的关系会有变动
5. **逢合者应和**：逢合的关系会有和解

### 生克应事时间

**大运应事：**
- 大运引动原局生克关系
- 大运十年一个大主题

**流年应事：**
- 流年引动原局或大运关系
- 流年一年一个主题

**流月应事：**
- 流月引动原局、大运、流年关系
- 流月一月一个主题

### 生克应事类别

**事业应事：**
- 官杀生印：晋升、考证
- 食伤生财：创业、赚钱
- 比劫帮身：合作、竞争

**感情应事：**
- 财官相生：结婚、恋爱
- 食伤见官：感情纠纷
- 比劫争财：感情竞争

**健康应事：**
- 五行受克：对应身体部位注意
- 五行过旺：对应身体部位亢进
- 五行过弱：对应身体部位虚弱

## 十二、注意事项

1. **生克有情无情**：同样是生克，有情则吉，无情则凶
2. **力量对比**：生克双方力量对比决定结果
3. **位置关系**：远近、上下影响生克力量
4. **合冲优先**：合冲关系优先于生克关系
5. **综合判断**：不可单凭一个关系断事
