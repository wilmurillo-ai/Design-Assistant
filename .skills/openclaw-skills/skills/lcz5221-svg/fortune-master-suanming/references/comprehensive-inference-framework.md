# 八字综合推断框架

## 一、推断核心思路

### 推断逻辑链

```
用户八字信息
    ↓
1. 排盘（含真太阳时）
    ↓
2. 五行生克分析 → 确定喜用神
    ↓
3. 地支藏干分析 → 十神配置
    ↓
4. 天干刑冲克害 → 关系网络
    ↓
5. 格局分析 → 命局层次
    ↓
6. 神煞分析 → 特殊标志
    ↓
7. 六亲分析 → 人际关系
    ↓
8. 象法分析 → 直观象意
    ↓
9. 大运分析 → 十年趋势
    ↓
10. 流年分析 → 具体应期
    ↓
综合判断 → 具体应事
```

---

## 二、综合分析步骤

### 第一步：排盘与基础信息

**输入：**
- 出生年月日时（农历/公历）
- 性别
- 出生地

**输出：**
```
四柱八字：
年柱：XX（纳音 XX）
月柱：XX（纳音 XX）
日柱：XX（纳音 XX）
时柱：XX（纳音 XX）

日主：X 水（阳 X/阴 X）
节气：XX 后第 X 天
真太阳时校正：XX
```

---

### 第二步：五行生克分析

**分析内容：**
1. 五行分布（含藏干）
2. 五行力量等级（特强/强/中/弱/极弱）
3. 相生关系
4. 相克关系
5. 天干五合
6. 地支六合/六冲/六害/三合/三会/相刑

**输出示例：**
```
五行力量：
金：★★★★☆ 35%（强）
木：★★☆☆☆ 15%（弱）
水：★★★★☆ 30%（强）
火：★★★☆☆ 20%（中）
土：★☆☆☆☆ 10%（弱）

相生关系：
- 金生水：庚辛生壬，力量强
- 水生木：壬生甲，但木弱不受生

相克关系：
- 金克木：庚辛克甲，木受克严重
- 土克水：戊克壬，但土弱克力小

天干五合：
- 丁壬合木：化气成功，木力量增加

地支关系：
- 子午冲：水火相冲，情绪波动
- 寅亥合：木局成，木力量增强
```

**推断要点：**
- 找出最旺的五行 → 主导命局特性
- 找出最弱的五行 → 需要注意的方面
- 找出受克的五行 → 对应身体部位/六亲注意

---

### 第三步：地支藏干分析

**分析内容：**
1. 各地支藏干（本气/中气/余气）
2. 藏干透干情况
3. 藏干通根情况
4. 藏干十神转换

**输出示例：**
```
年支 XX 藏干：
- 本气 X（60%）：XX 十神
- 中气 X（30%）：XX 十神
- 余气 X（10%）：XX 十神

透干情况：
- X 透干：力量显现，应 XX 事
- X 不透：力量内敛，待运年引动

通根情况：
- 日主通根：有根，力量强
- 财星通根：有根，财运稳固
```

**推断要点：**
- 藏干透干 → 该十神代表的人事会明显
- 藏干通根 → 力量实在，不是虚浮
- 财库/官库 → 逢冲开时发财/晋升

---

### 第四步：天干刑冲克害分析

**分析内容：**
1. 天干相克（20 组）
2. 天干四冲（甲庚冲、乙辛冲、丙壬冲、丁癸冲）
3. 天干相害
4. 天干组合象意

**输出示例：**
```
天干相克：
- 庚克甲：金克木，甲木受克严重
  - 力量等级：强克（庚得令）
  - 象意：肝胆注意、事业压力大

天干四冲：
- 甲庚冲：金木相战
  - 象意：内心矛盾、事业变动
  - 应事：工作变动、搬家

天干组合：
- 甲 + 丙 + 戊：顺生组合
  - 象意：气势流通，运势顺畅
```

**推断要点：**
- 天干受克 → 对应十神的人事有波折
- 天干相冲 → 变动、冲突
- 天干组合 → 气势流向

---

### 第五步：格局分析

**分析内容：**
1. 确定主格局（月令 + 透干）
2. 辅助格局
3. 格局层次（上/中上/中/下）
4. 格局喜忌

**输出示例：**
```
主格局：正官格

格局特点：
- 性格：正直、守规矩、有责任感
- 适合：公职、管理、稳定工作

格局层次：中上等

理由：
✅ 官星得令
✅ 官印相生
✅ 身旺能任官
⚠️ 有小瑕疵（伤官见官）

喜用神：
- 喜：印星、财星
- 忌：伤官、刑冲
```

**推断要点：**
- 格局真 → 应事明显
- 格局假 → 应事轻微
- 格局破 → 人生波折

---

### 第六步：神煞分析

**分析内容：**
1. 吉神（天乙贵人、文昌、禄神等）
2. 凶煞（羊刃、桃花、驿马等）
3. 特殊神煞（魁罡、华盖等）
4. 神煞位置（年/月/日/时）

**输出示例：**
```
吉神：
- 天乙贵人（年柱）：祖辈有贵人
- 文昌贵人（月柱）：学习好，有才华
- 禄神（日支）：中年有俸禄

凶煞：
- 羊刃（日支）：婚姻不顺，易争吵
- 桃花（时柱）：晚年风流，子女漂亮

特殊神煞：
- 魁罡（日柱）：性格刚强，适合军警
```

**推断要点：**
- 吉神到位 → 应吉事
- 凶煞到位 → 应凶事
- 神煞逢冲 → 力量减弱或增强

---

### 第七步：六亲分析

**分析内容：**
1. 父母（偏印/正印）
2. 兄弟（比肩/劫财）
3. 配偶（男命财星/女命官杀）
4. 子女（食神/伤官）
5. 六亲宫位（年/月/日/时）

**输出示例：**
```
父母：
- 父亲（偏印）：旺，有能力，关系好
- 母亲（正印）：弱，身体差，缘分浅

兄弟：
- 比肩：2 个，兄弟 2 人
- 关系：比肩帮身，兄弟助力大

配偶：
- 男命正财：旺，配偶有能力
- 婚姻宫：日支受冲，婚姻有波折

子女：
- 食神：1 个，子女 1 人
- 子女宫：时柱好，子女有出息
```

**推断要点：**
- 十神旺 → 该六亲有能力/缘分深
- 十神弱 → 该六亲能力差/缘分浅
- 宫位受冲 → 该六亲有波折

---

### 第八步：象法分析

**分析内容：**
1. 干支象意
2. 十神象法
3. 组合象法
4. 多派别汇通（盲派、滴天髓、神峰通考等）

**输出示例：**
```
干支象意：
- 甲木：大树、首领、开创
- 丙火：太阳、热情、光明
- 甲 + 丙：木火通明，才华横溢

十神象法：
- 官印相生：贵气、有地位
- 食伤生财：才华赚钱

组合象法：
- 金木相战：筋骨注意、事业变动
- 水火相冲：心血管注意、情绪波动

盲派直断：
- 「官星见禄，必掌权」
- 「财库逢冲，能发财」
```

**推断要点：**
- 象意直观 → 快速判断
- 组合象法 → 具体应事
- 口诀直断 → 点睛之笔

---

### 第九步：大运分析

**分析内容：**
1. 大运排法（起运时间、顺逆排）
2. 大运十年主题
3. 大运与原局关系（生/克/冲/合）
4. 大运力量评估

**输出示例：**
```
当前大运：XX（35-44 岁）

大运主题：事业发力、责任重

大运与原局关系：
- 生克：大运 X 生原局 X，助力大
- 冲合：大运 X 冲原局 X，变动大
- 力量：用神大运，力量强

十年运势：
- 事业：晋升、掌权
- 财运：财运好，收入增加
- 感情：稳定
- 健康：注意压力

关键流年：
- 2026 年：晋升机会
- 2028 年：财运高峰
- 2030 年：注意健康
```

**推断要点：**
- 用神大运 → 十年顺利
- 忌神大运 → 十年波折
- 大运冲原局 → 该十年变动大

### 第十步：流年分析

**分析内容：**
1. 流年干支
2. 流年与原局关系
3. 流年与大运关系
4. 流年应期判断

**输出示例：**
```
2026 丙午年（36 岁）：

流年：丙午（火旺）

流年与原局关系：
- 丙火：偏财，与原局 X 关系
- 午火：正官，与原局 X 关系
- 丙午：财官相生

流年与大运关系：
- 大运 X，流年 Y，X 生 Y
- 关系：助力大，应吉事

综合判断：
- 事业：晋升机会，把握
- 财运：财运好，加薪
- 感情：女命婚恋，男命一般
- 健康：压力大，注意心血管

断语：
「财官相生运势好，晋升加薪在此年；
若是女命婚恋动，男命注意身体安。」
```

**推断要点：**
- 用神流年 → 应吉事
- 忌神流年 → 应凶事
- 流年冲原局 → 变动
- 流年合原局 → 和解/合作

## 三、综合推断法则

### 1. 多层次验证

```
原局显示 + 大运显示 + 流年显示 = 高概率应事

例：
- 原局：财库逢冲，能发财
- 大运：财运旺
- 流年：冲开财库
→ 该年发财概率极高
```

### 2. 轻重缓急判断

**特强应事（90%+ 概率）：**
- 原局 + 大运 + 流年都显示
- 用神到位，无克制
- 多个神煞同时应验

**强应事（70-90% 概率）：**
- 原局 + 大运显示
- 或原局 + 流年显示
- 用神到位，有小克制

**中应事（50-70% 概率）：**
- 单一层面显示
- 力量中等
- 有克制但有救应

**弱应事（30-50% 概率）：**
- 力量弱
- 克制多
- 救应少

### 3. 应事类别判断

**事业应事：**
- 官杀透干 + 得令 → 晋升
- 印星透干 + 得令 → 考证、学习
- 财星透干 + 得令 → 加薪、赚钱
- 食伤透干 + 得令 → 才华展现、创业
- 比劫透干 + 得令 → 竞争、合作

**财运应事：**
- 财星旺 + 身旺 → 发财
- 财库逢冲 → 发财
- 食伤生财 → 才华赚钱
- 比劫夺财 → 破财

**感情应事：**
- 男命财星年 → 恋爱、结婚
- 女命官杀年 → 恋爱、结婚
- 桃花年 → 异性缘好
- 婚姻宫动年 → 感情变化

**健康应事：**
- 五行受克 → 对应身体部位注意
- 羊刃年 → 外伤、手术
- 冲克日主年 → 整体健康注意

### 4. 应期精确判断

**精确到年：**
- 用神到位年
- 忌神到位年
- 冲合年

**精确到月：**
- 流月引动原局
- 流月与大运流年组合

**精确到日：**
- 特殊日子（结婚、开业等）
- 需结合择日

## 五、实战案例

### 案例：正官格 35-44 岁大运推断

**原局：**
```
年柱：庚午（劫财/七杀）
月柱：辛巳（比肩/正官）
日柱：壬申（日主/偏印）
时柱：乙巳（伤官/偏财）
```

**分析：**
1. 五行：金旺、火旺、水中、木弱、土弱
2. 格局：正官格，中上等
3. 喜用：金、水
4. 忌神：火、土

**当前大运：甲申（35-44 岁）**
- 甲木：食神，泄秀
- 申金：偏印，生身
- 大运主题：才华展现、贵人扶持

**流年推断：**

**2026 丙午年（36 岁）：**
- 丙火：偏财，忌神
- 午火：正官，忌神
- 丙午：财官旺，但为忌
- 综合：压力大，但有晋升机会
- 断语：「财官虽旺为忌神，压力之中有机会；谨慎把握晋升路，注意身体莫太累。」

**2027 丁未年（37 岁）：**
- 丁火：正财，忌神
- 未土：正官，忌神
- 综合：压力持续，但稳定
- 断语：「正财正官压力在，稳扎稳打莫急躁；等待好时机来到，明年运势会更好。」

**2028 戊申年（38 岁）：**
- 戊土：七杀，忌神
- 申金：偏印，用神
- 综合：杀印相生，贵人提携
- 断语：「杀印相生贵人来，事业转折在此年；把握机会勇向前，晋升加薪可见。」

**2029 己酉年（39 岁）：**
- 己土：正官，忌神
- 酉金：正印，用神
- 综合：官印相生，运势佳
- 断语：「官印相生运势好，事业顺利步步高；财运也有好转机，家庭和睦乐陶陶。」

**2030 庚戌年（40 岁）：**
- 庚金：偏印，用神
- 戌土：七杀，忌神
- 综合：印化杀，有变动
- 断语：「印化杀星有变动，工作可能有调整；谨慎决策莫冲动，平稳过渡保安宁。」

## 六、注意事项

1. **综合判断**：不可单凭一个层面断事
2. **力量对比**：原局、大运、流年力量要对比
3. **灵活应用**：不可死板套用，需灵活运用
4. **尊重隐私**：涉及隐私需谨慎
5. **安全边界**：不可断生死、疾病、绝对事件
