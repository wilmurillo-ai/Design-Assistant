### 相克力量等级

**特强克：** 得月令 + 多柱帮扶
```
例：丙火日主，生于午月，天干透庚
- 丙火得月令，力量特强
- 丙克庚，克力特强
- 庚金受制严重
```

**强克：** 得月令或有根
```
例：甲木日主，生于寅月，天干透戊
- 甲木得月令，力量强
- 甲克戊，克力强
- 戊土受制明显
```

**中克：** 有根但无帮扶
```
例：乙木日主，地支有卯，天干透己
- 乙木有根，力量中等
- 乙克己，克力中等
- 己土受制一般
```

**弱克：** 失令无根
```
例：庚金日主，生于午月，天干透甲
- 庚金失令，力量弱
- 庚克甲，克力弱
- 甲木受制轻微
```

## 四、天干相害

### 天干害关系

天干无直接相害，害主要在地支。但天干可通过以下方式产生害的效果：

**1. 合害：** 天干合而地支害
```
例：天干甲己合，地支子未害
- 表面和谐，内部矛盾
- 外合内害，表里不一
```

**2. 克害：** 天干克而地支害
```
例：天干甲克戊，地支子未害
- 克制加害，雪上加霜
- 压力更大，困难更多
```

**3. 冲害：** 天干冲而地支害
```
例：天干甲庚冲，地支子未害
- 冲害交加，动荡不安
- 变动大，影响深
```

## 五、天干应事判断

### 应事原则

**1. 透干应显：** 天干透出的关系更明显
```
例：甲庚冲透干
- 冲突明显，容易看到
- 应事快速，影响直接
```

**2. 得令应强：** 得月令的天干力量强
```
例：甲木生于寅月，甲庚冲
- 甲木得令，冲力强
- 庚金受制严重
```

**3. 有根应实：** 地支有根的天干实在
```
例：甲木地支有寅，甲庚冲
- 甲木有根，冲力实
- 不是虚张声势
```

**4. 多现应重：** 多次出现的关系重要
```
例：八字两个甲木，一个庚金
- 甲木多，庚金受制重
- 应事更明显
```

### 应事时间

**大运应事：**
- 大运天干引动原局关系
- 十年一个大主题

**流年应事：**
- 流年天干引动原局或大运关系
- 一年一个主题

**流月应事：**
- 流月天干引动原局、大运、流年关系
- 一月一个主题

### 应事类别

**事业应事：**
- 官杀透干受克：工作压力、职位变动
- 财星透干受克：财运波动、投资失利
- 印星透干受克：学习受阻、证书难拿
- 食伤透干受克：才华受制、表达受阻
- 比劫透干受克：朋友矛盾、竞争失利

**感情应事：**
- 男命财星受克：感情波折、破财
- 女命官杀受克：感情不顺、婚姻波动
- 食伤受克：子女缘分浅、表达受阻
- 印星受克：与长辈关系紧张

**健康应事：**
- 木受克：肝胆、筋骨、神经系统
- 火受克：心血管、眼睛、炎症
- 土受克：脾胃、皮肤、肌肉
- 金受克：肺部、呼吸系统、皮肤
- 水受克：肾脏、泌尿系统、血液

## 七、实战应用

### 1. 性格分析

```
例：天干甲庚冲

分析：
- 甲木：刚直、向上
- 庚金：刚毅、变革
- 金木相战：内心矛盾

断语：
命主性格刚强，有冲劲，但内心矛盾，
既想坚持又想改变，容易冲动决策。
```

### 2. 事业分析

```
例：天干丙壬冲，丙为食神，壬为偏财

分析：
- 食神克偏财：才华赚钱
- 水火相冲：波动大

断语：
命主适合用才华赚钱，但事业起伏大，
需要平衡热情与冷静。
```

### 3. 感情分析

```
例：女命天干丁癸冲，丁为正官，癸为伤官

分析：
- 伤官见官：感情波折
- 水火相冲：情绪波动

断语：
命主感情细腻，但容易与伴侣产生误会，
需要注意沟通，控制情绪。
```

### 4. 健康分析

```
例：天干甲庚冲

分析：
- 木受克：肝胆注意
- 金受克：肺部注意
- 金木相战：筋骨注意

断语：
命主需注意肝胆、肺部、筋骨健康，
避免过度劳累，定期体检。
```

## 八、注意事项

1. **力量对比**：天干力量取决于得令、有根、帮扶
2. **位置关系**：远近影响生克力量
3. **合冲优先**：合冲关系优先于生克关系
4. **通关重要**：有通关五行可化解冲突
5. **综合判断**：不可单凭天干关系断事
