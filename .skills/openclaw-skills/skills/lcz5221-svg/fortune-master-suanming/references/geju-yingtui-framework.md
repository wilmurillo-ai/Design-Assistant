# 八字格局应事推断框架

## 一、格局应事基础

### 推断原则

**核心思路：**
```
原局格局（命） + 大运（运） + 流年五行（岁） = 具体应事
```

**推断步骤：**
1. 确定原局格局类型
2. 分析格局喜忌
3. 看大运对格局的影响
4. 看流年五行对格局的触发
5. 综合判断具体应事

---

## 二、十大格局应事详解

**格局特点：**
- 月令正官，官星透干或得地
- 性格：正直、守规矩、有责任感
- 适合：公职、管理、稳定工作

- 喜：印星（官印相生）、财星（财生官）

**具体断语：**

## 四、应期判断法则

### 1. 用神到位应吉

```
例：用神为木，逢甲乙寅卯年
- 用神到位，运势好
- 应吉事：晋升、发财、婚恋
```

### 2. 忌神到位应凶

```
例：忌神为火，逢丙丁巳午年
- 忌神到位，运势差
- 应凶事：破财、是非、健康问题
```

### 3. 冲合应事

```
例：日支逢冲年
- 婚姻宫动
- 应事：结婚、离婚、感情变化
```

### 4. 伏吟应事

```
例：流年与日柱相同
- 伏吟年
- 应事：重大变化、重复之事
```

### 5. 反吟应事

```
例：流年与日柱天克地冲
- 反吟年
- 应事：剧烈变动、吉凶明显
```
