# 经典著作引用框架（增强版）

## 一、引用原则

**核心要求：**
1. 每个重要判断必须标注**经典出处**
2. 引用原文 + 白话解释 + 实际应事
3. 多书互证，增强可信度

---

## 二、八字核心理论经典引用

### 2.1 《渊海子平》引用

**核心理论：** 十神体系、格局论命、六亲定位

**引用格式：**
```
【经典引用】《渊海子平·卷 X·XX 章》
原文：「XXXX」
白话：XXXX
应用：XXXX
```

**示例：**
```
【经典引用】《渊海子平·卷二·论正官》
原文：「正官者，乃克我之神，要身旺逢之，主贵。」
白话：正官是克日主的五行，需要日主身旺才能担，主贵气、地位。
应用：此造壬水日主，得申金长生，身旺能担官，月令巳中戊土正官，主事业有地位。
```

---

### 2.2 《三命通会》引用

**核心理论：** 纳音五行、神煞系统、干支象意

**示例：**
```
【经典引用】《三命通会·卷 X·论天乙贵人》
原文：「天乙贵人者，三命中最吉之神也。凡人命遇之，主聪明智慧，出入近贵。」
白话：天乙贵人是八字中最吉利的神煞，命带此贵人主聪明、有贵人扶持。
应用：此造月支巳、日支申、时支巳，三重天乙贵人，一生贵人扶持，逢凶化吉。
```

---

### 2.3 《子平真诠》引用

**核心理论：** 格局成败、用神选取、格局层次

**示例：**
```
【经典引用】《子平真诠·卷 X·论偏印格》
原文：「偏印格，要身旺印旺，主聪明多艺，技术成名。」
白话：偏印格需要日主旺、印星旺，主聪明、有技艺，靠技术成名。
应用：此造偏印格真格，庚金透干有根，壬水得申金长生，身印两旺，主技术成名。
```

---

### 2.4 《滴天髓》引用

**核心理论：** 气势流通、清浊判断、体用关系

**示例：**
```
【经典引用】《滴天髓·通神论》
原文：「壬水通河，刚中之德；周流不滞，化则有情。」
白话：壬水如江河，有刚柔并济之德，流动不滞，能化育万物。
应用：此造壬水日主，得庚辛金生，申金为源，金水相涵，主智慧深远，事业顺利。
```

---

### 2.5 《穷通宝鉴》引用

**核心理论：** 月令调候、寒暖燥湿

**示例：**
```
【经典引用】《穷通宝鉴·五月壬水》
原文：「五月壬水，丁旺壬弱，取庚为君，取癸为臣。」
白话：五月（午月）火旺水弱，壬水需要庚金发源（第一用神），癸水帮身（第二用神）。
应用：此造壬水生于巳月（近午月），火旺，年干庚金透干为第一用神到位，日支申中癸水为第二用神不足，调候中等偏上。
```

---

### 2.6 《神峰通考》引用

**核心理论：** 病药说、五行平衡

**示例：**
```
【经典引用】《神峰通考·病药说》
原文：「有病方为贵，无伤不是奇；格中如去病，财禄两相随。」
白话：命局有病症但有药方制化，反而是贵格；无病无伤只是平常命。
应用：此造火旺为病，庚金为药，有病有药，主中年后发福。
```

## 三、象法经典引用

### 3.1 《玉照定真经》引用

**核心理论：** 干支象意、组合象法

**示例：**
```
【经典引用】《玉照定真经》
原文：「庚金午火，火炼真金。」
白话：庚金坐午火，如同火炼真金，早年磨砺，中年成才。
应用：此造年柱庚午，火炼真金之象，主早年有磨砺，中年后成才。
```

```
应用：此造壬水日主，主聪明智慧，适应力强。
```

```
应用：此造当前甲申大运（30-39 岁），申金为用神到位，主这十年贵人扶持，事业起步。
```

---

### 5.2 《滴天髓·论流年》引用

## 八、引用注意事项

1. **原文准确：** 引用原文需准确，不可杜撰
2. **白话清晰：** 白话解释要清晰易懂
3. **应用贴切：** 实际应用要与原文匹配
4. **多书互证：** 重要判断多书互证，增强可信度
5. **适度引用：** 不可过度引用，影响阅读体验
