# 八字 / 四柱分析框架（增强版）

## 目的

让解读像真正懂八字的人写出来，结合《渊海子平》《三命通会》《子平真诠》《滴天髓》《穷通宝鉴》《神峰通考》等古典命理典籍，进行全面细致的分析，但在缺少完整排盘条件时保持诚实。

## 默认分析顺序

1. 先判断资料完整度
2. 再抓”日主”或人格核心（参考《子平真诠》关于日主强弱的论述）
3. 看五行偏向与整体冷热燥湿感（参考《穷通宝鉴》调候理论）
4. 看十神结构：比劫、食伤、财、官杀、印（参考《渊海子平》十神用法）
5. 看格局成败与高低（参考《子平真诠》格局理论，《滴天髓》精神气理论）
6. 看问题对应的重点领域
7. 看阶段切换：起势、蓄力、受压、转向、收获（结合《三命通会》大运流年理论）
8. 分析刑冲合害等特殊关系（参考《三命通会》干支关系）
9. 考虑神煞影响（参考《三命通会》各类神煞）
10. 最后给建议与节奏

## 推荐写法

### 1. 先定人格底色
不要一上来堆术语，先把用户的气质说清楚，结合《滴天髓》中”何知其人凶，忌神辗转攻”的理念，以及象法中的直观形象，例如：
- 行动力强，但容易急
- 外表克制，内里其实很敏感
- 对关系有责任感，但容易把压力自己扛
- 思维强，执行慢；或执行强，情绪恢复慢

### 2. 再解释五行和十神
术语出现后要顺手翻译成人话，参考《神峰通考》的灵活取用原则，结合象法理论：

- 木：生长、规划、向上、原则
- 火：表达、热情、显化、冲劲
- 土：承载、现实、稳定、顾虑
- 金：规则、决断、边界、效率
- 水：流动、感受、洞察、变化

十神不要硬背定义，要服务问题，参考《渊海子平》：

- 比劫偏强：自我主张强，合作中容易顶着来
- 食伤明显：表达力、创意、输出欲强，也容易口快
- 财星有力：现实目标清晰，重结果、资源、效率
- 官杀明显：责任感、压力感、秩序要求强
- 印星偏重：安全感需求高，重吸收、重思考、易多想

### 3. 象法分析
参考盲派象法、《干支易象学》等象法理论：
- 通过干支组合直接呈现象意
- 运用象法口诀判断吉凶
- 如”财星见冲，财有变动”
- 结合天干地支类象进行直观分析

### 4. 分析格局层次
参考《子平真诠》的格局理论，区分用神喜忌：
- 正格：正官格、七杀格、正财格、偏财格、正印格、偏印格、食神格、伤官格
- 杂气格：辰戌丑未月令藏干杂气
- 从格：从财、从杀、从儿、从强、从旺等特殊格局

根据《穷通宝鉴》调候理论，特别注意季节性用神：
- 寒暖湿燥调候的重要性
- 不同月份的用神喜忌

### 5. 解读大运流年
参考《三命通会》的大运流年理论：
- 大运与原局的相互作用
- 流年与大运、原局的三方关系
- 《滴天髓》中”太过宜剥削，不及喜滋生”的运程调节原则

### 6. 分析刑冲合会关系
参考《三命通会》中干支关系理论：
- 地支六冲：子午冲、丑未冲、寅申冲、卯酉冲、辰戌冲、巳亥冲
- 地支六合：子丑合、寅亥合等
- 地支三合：申子辰、亥卯未等
- 地支三会：寅卯辰、巳午未等
- 地支刑害：寅巳申三刑等

### 7. 考虑神煞影响
参考《三命通会》中的神煞理论（仅作为辅助）：
- 吉神：文昌、天乙贵人、太极贵人等
- 凶神：羊刃、孤辰寡宿等
- 特殊神煞：驿马、桃花、华盖等

### 8. 再落到用户的问题
#### 感情
重点看：
- 自我表达与亲密关系的平衡
- 容不容易控制、回避、依赖或过度付出
- 当前更像”桃花增加”还是”关系筛选期”
- 参考《神峰通考》感情章节的论断方法
- 运用象法分析感情状态

#### 事业
重点看：
- 适合稳定体系、自由发挥，还是资源整合
- 当前是开局期、蓄力期、转型期，还是收尾期
- 是需要”出头”还是”收心”
- 参考《三命通会》职业章节分析职业倾向
- 运用象法判断事业发展

#### 财富
重点看：
- 赚钱方式更偏主动型还是配置型
- 容易因冲动、面子、情绪、人情而失财吗
- 适合扩张还是保守
- 参考《渊海子平》财星章节的财富分析
- 运用象法分析财运起伏

#### 学业 / 考试
重点看：
- 吸收力、执行力、耐性、节奏感
- 更需要方法、环境还是心态调整
- 参考《穷通宝鉴》中印星与学习能力的关系
- 运用象法判断学业时机

### 9. 强化阶段主题
增强版必须写出，参考《滴天髓》的精深理论：
- 现在卡点在哪
- 这个阶段该补什么，不该硬冲什么
- 哪类机会是”看起来像机会，实际上会分神”
- 哪类关系或合作值得留

## 当用户提供的资料达到S级（高精度素材）时

在用户提供了完整命盘截图、已排好的八字四柱或其他高精度素材时，执行深度精读分析，具体要求如下：

### 深度精读流程

1. **基础信息确认**
   - 核对八字四柱准确性
   - 确认出生时间、地点、性别等基础信息
   - 分析节气归属，精确判断月令

2. **多书汇通分析**
   - 结合《渊海子平》分析格局用神和十神配置
   - 运用《三命通会》的干支关系和神煞理论
   - 依《子平真诠》判断格局成败高低
   - 用《滴天髓》分析气势、体用、精神气概
   - 按《穷通宝鉴》验证调候需求
   - 以《神峰通考》活法思路修正格局判断
   - 运用盲派象法进行直观断事
   - 结合《滴天髓》（任铁樵注）提升理论高度
   - 运用《干支易象学》进行取象应象
   - 参考《命理象法大全》盲派实战技巧
   - 依《象法大观》构建完整象法体系

3. **联网查询辅助分析**
   - 必要时联网查询相关书籍原文、注解或现代学者的研究成果
   - 查找类似命例的分析方法和断语
   - 验证古籍中的理论要点和应用方法

4. **全方位分析内容**
   - 阴阳五行分布及平衡状况
   - 天干地支组合关系（合、化、冲、刑、害、穿）
   - 地支藏干对整体格局的影响
   - 十神配置及其对六亲关系的反映
   - 格局高低及用神喜忌分析
   - 调候与通关分析
   - 神煞作用（参考《三命通会》中的神煞体系）
   - 象法直观解读（结合盲派口诀、象法大观等）
   - 各柱所代表人生领域的详细分析
   - 大运流年对原局的引动和影响

5. **精细推断**
   - 逐年大运的具体影响
   - 关键流年的吉凶判断
   - 结合象法理论推测具体事项发生时机
   - 运用盲派断语验证分析结果

6. **输出格式要求**
   - 结构清晰，层次分明
   - 说明所依据的理论来源
   - 区分确定性分析和推断性分析
   - 提供具体建议和应对策略

## 常用句式（结合古籍精华）

参考《滴天髓》《子平真诠》等古籍中的经典断语，转化为现代语言：
- "你命中格局有情，但需要合适的时机来展现。"
- "你的强项在于内敛的积累，而非表面的张扬。"

- "你的问题在于时机把握，过早或过晚都会错失良机。"
