# 八字经典著作框架（六经汇通）

## 六大经典简介

### 《渊海子平》
- 宋代徐子平著，子平术之奠基作
- 重点：十神用法、格局高低、用神概念
- 特点：理论扎实，实战性强，奠定了正格格局体系

### 《三命通会》
- 明代万民英著，八字集大成之作
- 重点：五行生克、神煞运用、干支关系
- 特点：内容全面，包含大量实例和口诀，是百科全书式作品

### 《子平真诠》
- 清代沈孝瞻著，格局理论集大成
- 重点：格局成败、用神变化、真假喜忌
- 特点：逻辑严谨，条理分明，格局理论最为精深

### 《滴天髓》
- 传说宋代京图撰，明代刘基注，清代任铁樵疏证
- 重点：精神气概、体用关系、通关调和
- 特点：文字简奥，哲理深刻，强调高层次的命理思维

### 《穷通宝鉴》
- 传说明代刘基著，清代余春台辑，民国徐乐吾评注
- 重点：调候理论、月令用神、季节喜忌
- 特点：以调候为核心，突出五行在不同季节的应用

### 《神峰通考》
- 明代谢廷珠著
- 重点：病药理论、格局真假、活法用神
- 特点：打破格局执念，提倡灵活分析，强调变通之道

## 象法流派著作

### 盲派八字象法
- 特点：注重象形、象意，通过干支组合直观呈现人事现象
- 实战性强，口诀丰富，如"财星见冲，财有变动"
- 重视干支类象和直观断事技巧

### 《滴天髓》（任铁樵注）
- 高阶象理，气势、流通、清浊的象法思维
- 拔升理论高度，超越格局论命

### 《八字命理学基础教程》（陆致极）
- 现代白话，干支类象、十神象意讲得通俗，配有实例

### 《四柱八字预测学》（邵伟华）
- 入门经典，基础象法 + 简单断事，适合快速上手

### 《干支易象学》（贺云飞）
- 象法核心著作，打通干支与八卦象
- 讲取象、应象、象变，实战性极强

### 《命理象法大全》（盲派体系）
- 盲派象法精华，干支类象、取象原则、应事实例
- 侧重实战断事

### 《象法大观》（陈立强）
- 系统讲八字象法体系，涵盖天干地支象、十神象、格局象、流年象

## 经典融合分析法

### 格局分析（《子平真诠》+《渊海子平》）
1. 首先确定月令用神
2. 判断格局成败高低
3. 看用神是否透干，以及其卫护情况
4. 分析格局真假（《神峰通考》）

### 调候理论（《穷通宝鉴》）
1. 冬月夏月重调候，调候第一
2. 春秋月令需观其深浅，配合适当用神
3. 寒暖湿燥配合格局使用

### 神煞运用（《三命通会》）
1. 在格局基础上参考神煞
2. 以格局成败为主，神煞为辅
3. 不可过分依赖神煞判断

### 体用通关（《滴天髓》）
1. 看八字全局的气势流向
2. 寻找通关环节，化解矛盾
3. 重视精神气概，不拘泥于形式

### 活法用神（《神峰通考》）
1. 不拘泥于固定格局
2. 根据实际情况灵活取用
3. 病药理论：有病有药为佳

### 象法应用（盲派、干支易象学等）
1. 从干支组合直观判断人事象
2. 运用象法口诀进行快速断事
3. 通过象意联系现实生活

## 实际应用技巧

### 综合分析步骤
1. 按《子平真诠》方法确定格局用神
2. 按《穷通宝鉴》调候理论检验合理性
3. 按《滴天髓》理念看整体气势流通
4. 按《神峰通考》活法思路修正格局判断
5. 按《三命通会》丰富细节和神煞信息
6. 按《渊海子平》理论总结十神喜忌
7. 运用象法理论进行直观判断（盲派象法、干支易象学等）
8. 结合现代象法著作深化理解（象法大观、命理象法大全等）

### 不同问题侧重点
- 感情：重视印枭食伤关系（《滴天髓》关于情性），结合象法判断感情状态
- 事业：看官杀与财星的配合（《渊海子平》），运用象法断事业起伏
- 财富：财星与格局成败关系（《穷通宝鉴》调候财星），用盲派口诀判断财运
- 健康：五行平衡与调候（《神峰通考》），但不涉及医疗范畴
- 学业：印星与食伤的关系（《三命通会》），结合象法判断学习运势

## 经典断语现代化
- 《滴天髓》："何知其人凶，忌神辗转攻" → 当忌神在局中得到连续生扶时，易有困顿
- 《子平真诠》："用神如有损，格局有高低" → 用神受制，则格局降低，成就有限
- 《穷通宝鉴》："春月不宜见水" → 春季木旺，过多见水反致漂浮
- 《三命通会》："财官印食俱全，定是富贵神仙" → 八字搭配得宜，必有福泽
- 盲派口诀："财星见冲，财有变动" → 财星逢冲，钱财易有波动

## 注意事项
- 不可孤立看待任何一部经典，应当六经汇通
- 《神峰通考》虽提倡活法，但仍需格局基础
- 《滴天髓》虽讲精神，但需五行实体支撑
- 融合象法理论，提升直观断事能力
- 保持古籍中劝善向上的人生指导意义
- 实战中结合现代象法著作，提高断事准确性
