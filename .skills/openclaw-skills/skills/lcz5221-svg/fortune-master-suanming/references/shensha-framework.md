# 神煞详解框架

## 一、吉神详解

### 天乙贵人

**查法：**
```
甲戊庚牛羊（丑未）
乙己鼠猴乡（子申）
丙丁猪鸡位（亥酉）
壬癸兔蛇藏（卯巳）
六辛逢马虎（午寅）
```

**象意：**
- 最大的贵人星，解难呈祥
- 有贵人扶持，逢凶化吉
- 聪明智慧，人缘好

**应事：**
- 事业：贵人提携，晋升机会
- 财运：贵人帮助得财
- 感情：良缘，配偶有能力
- 健康：逢凶化吉，病有良医

**位置含义：**
- 年柱：祖辈有贵人
- 月柱：父母、兄弟有贵人
- 日柱：配偶是贵人
- 时柱：子女是贵人，晚福好

## 二、凶煞详解


## 四、神煞应用法则

### 1. 神煞力量评估

**特强：**
- 得月令 + 多柱出现
- 主导命局特性

**强：**
- 得月令或有根
- 明显影响力

**中：**
- 有根但无帮扶
- 一般影响力

**弱：**
- 失令无根
- 轻微影响力

### 2. 神煞组合

**吉神 + 吉神：**
- 吉上加吉
- 福气更厚

**吉神 + 凶煞：**
- 吉凶参半
- 有救应

**凶煞 + 凶煞：**
- 凶上加凶
- 灾祸重

### 3. 神煞与十神

**吉神 + 用神：**
- 吉力增强
- 应吉明显

**凶煞 + 忌神：**
- 凶力增强
- 应凶明显

### 4. 神煞应期

**流年逢吉神：**
- 该年应吉事
- 贵人帮助

**流年逢凶煞：**
- 该年应凶事
- 需注意

## 五、实战应用

### 1. 性格分析

```
例：命带文昌 + 华盖

分析：
- 文昌：聪明，有才华
- 华盖：孤傲，有悟性

断语：
命主聪明有才华，但性格孤傲，
适合从事文化、艺术、宗教工作。
```

### 2. 事业分析

```
例：命带将星 + 禄神

分析：
- 将星：领导才能
- 禄神：有俸禄

断语：
命主有领导才能，适合管理岗位，
有固定收入，生活富足。
```

### 3. 感情分析

```
例：命带红鸾 + 桃花

分析：
- 红鸾：婚姻喜庆
- 桃花：异性缘好

断语：
命主异性缘好，容易恋爱结婚，
但需注意感情专一。
```

### 4. 健康分析

```
例：命带羊刃 + 劫煞

分析：
- 羊刃：容易受伤
- 劫煞：意外灾祸

断语：
命主需注意意外伤害，
避免危险活动，定期体检。
```

## 六、注意事项

1. **神煞为辅**：神煞是辅助，不可单凭神煞断事
2. **组合判断**：神煞需与五行、十神、格局结合
3. **力量评估**：神煞力量有强弱，不可一概而论
4. **吉凶转化**：吉神不一定吉，凶煞不一定凶
5. **综合判断**：需综合原局、大运、流年
