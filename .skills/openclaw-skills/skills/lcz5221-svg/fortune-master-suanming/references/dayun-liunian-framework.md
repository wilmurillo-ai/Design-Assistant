# 大运流年详解框架

## 一、大运基础

### 大运排法

**起运时间计算：**
```
阳男阴女：顺数到下一个节气
阴男阳女：逆数到上一个节气

三天为一岁，一天为四月，一时辰为十天

例：阳男，顺数 15 天到下一个节气
15 天 ÷ 3 = 5 岁起运
```

**大运排列：**
```
以月柱为基准，顺排或逆排

例：月柱甲子，阳男顺排
大运：乙丑、丙寅、丁卯、戊辰、己巳、庚午...

例：月柱甲子，阴男逆排
大运：癸亥、壬戌、辛酉、庚申、己未、戊午...
```

### 大运十年周期

| 大运阶段 | 年龄 | 特点 |
|---------|------|------|
| **第一步大运** | 5-14 岁 | 童年、学习、成长 |
| **第二步大运** | 15-24 岁 | 青年、学业、初恋 |
| **第三步大运** | 25-34 岁 | 事业起步、成家 |
| **第四步大运** | 35-44 岁 | 事业发力、责任重 |
| **第五步大运** | 45-54 岁 | 事业巅峰、收获 |
| **第六步大运** | 55-64 岁 | 守成、准备退休 |
| **第七步大运** | 65-74 岁 | 晚年、享福 |
| **第八步大运** | 75 岁后 | 高寿、安康 |

### 大运力量评估

**特强大运：**
- 用神大运 + 得月令
- 改变命局层次
- 十年黄金期

**强大运：**
- 用神大运或有根
- 明显助力
- 十年发展期

**中大运：**
- 平运，无大起大落
- 平稳发展
- 积累期

**弱大运：**
- 忌神大运
- 压力较大
- 守成为主

**特弱大运：**
- 忌神大运 + 冲克原局
- 困难较多
- 需谨慎

## 二、大运与原局关系

### 大运生原局

**大运生身：**
```
例：日主壬水弱，大运庚辛金

象意：
- 印星生身，贵人扶持
- 学习进步，事业顺利
- 身体健康，心情好

应事：
- 考证、升学
- 晋升、加薪
- 买房、置业
```

**大运生财：**
```
例：财星弱，大运食伤

象意：
- 食伤生财，才华赚钱
- 财运好转，收入增加
- 投资机会多

应事：
- 创业、投资
- 加薪、奖金
- 意外之财
```

**大运生官：**
```
例：官星弱，大运财星

象意：
- 财生官旺，事业上升
- 职位晋升，权力增大
- 名声提升

应事：
- 晋升、提拔
- 考公、入职
- 获奖、成名
```

### 大运克原局

**大运克身：**
```
例：日主壬水弱，大运戊己土

象意：
- 官杀克身，压力大
- 工作繁重，身体累
- 需谨慎行事

应事：
- 工作压力
- 健康问题
- 口舌是非

建议：
- 保守发展
- 注意健康
- 低调做人
```

**大运克财：**
```
例：财星弱，大运比劫

象意：
- 比劫克财，破财
- 竞争激烈，赚钱难
- 朋友借钱不还

应事：
- 破财、投资失利
- 借钱不还
- 竞争失利

建议：
- 不宜投资
- 不借钱给他人
- 保守理财
```

**大运克官：**
```
例：官星弱，大运食伤

象意：
- 食伤克官，工作变动
- 与上级关系紧张
- 女命感情波折

应事：
- 换工作、辞职
- 与领导矛盾
- 感情分手

建议：
- 谨慎跳槽
- 控制情绪
- 注意沟通
```

### 大运冲原局

**大运冲年柱：**
```
象意：
- 祖业变动
- 搬家、移民
- 长辈健康注意

应事：
- 搬家、换城市
- 祖辈健康问题
- 出国、远行
```

**大运冲月柱：**
```
象意：
- 事业变动
- 工作环境变化
- 父母健康注意

应事：
- 换工作、跳槽
- 部门调整
- 父母健康问题
```

**大运冲日柱：**
```
象意：
- 感情波动
- 婚姻宫动
- 自己健康注意

应事：
- 结婚、离婚
- 感情波折
- 自己健康问题
```

**大运冲时柱：**
```
象意：
- 子女缘分动
- 事业结果变化
- 晚年计划调整

应事：
- 怀孕、生子
- 子女升学、工作
- 退休计划变化
```

## 三、流年详解

### 流年力量

**流年力量等级：**
- 流年占 30% 影响力
- 引动原局和大运
- 一年一个主题

**流年应事特点：**
- 快速、直接
- 明显、具体
- 可预测、可准备

### 流年与原局关系

**流年生身：**
```
例：日主壬水，流年庚辛金

象意：
- 印星生身，贵人助力
- 学习进步，事业顺利
- 有考证、升学机会

应事：
- 考试顺利
- 晋升机会
- 贵人帮助
```

**流年克身：**
```
例：日主壬水，流年戊己土

象意：
- 官杀克身，压力大
- 工作繁重
- 需注意健康

应事：
- 工作压力
- 身体疲惫
- 口舌是非
```

**流年冲日主：**
```
例：日主壬水，流年丙火

象意：
- 水火相冲，情绪波动
- 事业变动
- 感情波折

应事：
- 换工作
- 感情变化
- 搬家
```

### 流年与大运关系

**流年生大运：**
```
例：大运甲木，流年壬癸水

象意：
- 水生木，大运得助
- 该年运势好
- 事半功倍

应事：
- 事业顺利
- 财运好
- 感情和睦
```

**流年克大运：**
```
例：大运甲木，流年庚辛金

象意：
- 金克木，大运受制
- 该年压力大
- 需谨慎

应事：
- 工作阻力
- 财运波动
- 健康问题
```

**流年冲大运：**
```
例：大运甲木，流年庚金

象意：
- 金木相冲，变动大
- 该年变化多
- 需准备

应事：
- 工作变动
- 搬家
- 感情变化
```

## 四、大运流年组合分析

### 大运好 + 流年好

```
例：用神大运 + 用神流年

象意：
- 十年最好的年份
- 事业财运双丰收
- 重大机遇

应事：
- 创业成功
- 晋升提拔
- 结婚生子
- 买房置业

建议：
- 把握机遇
- 大胆发展
- 但不可过度
```

### 大运好 + 流年差

```
例：用神大运 + 忌神流年

象意：
- 大环境好，小环境差
- 有阻力但能克服
- 先难后易

应事：
- 年初困难，年末顺利
- 有小人但能化解
- 财运波动但总体好

建议：
- 年初保守
- 年中发力
- 年末收获
```

### 大运差 + 流年好

```
例：忌神大运 + 用神流年

象意：
- 大环境差，小环境好
- 短暂好转
- 抓住机会

应事：
- 该年有机会
- 但不可贪多
- 见好就收

建议：
- 把握流年机会
- 不宜长期投资
- 积累实力
```

### 大运差 + 流年差

```
例：忌神大运 + 忌神流年

象意：
- 十年最差的年份
- 压力最大
- 需谨慎

应事：
- 工作困难
- 财运差
- 健康问题
- 感情波折

建议：
- 保守发展
- 注意健康
- 低调做人
- 等待时机
```

## 六、流年应期判断

### 应期原则

**1. 用神到位应吉：**
```
例：用神为木，逢甲乙寅卯年
- 用神到位，运势好
- 应吉事
```

**2. 忌神到位应凶：**
```
例：忌神为火，逢丙丁巳午年
- 忌神到位，运势差
- 应凶事
```

**3. 冲合应事：**
```
例：日支逢冲年
- 婚姻宫动
- 感情变化
```

**4. 伏吟应事：**
```
例：流年与日柱相同
- 伏吟年
- 重大变化
```

**5. 反吟应事：**
```
例：流年与日柱天克地冲
- 反吟年
- 剧烈变动
```

### 应事类别判断

**事业应期：**
- 官杀流年：工作变动、晋升
- 印星流年：考证、学习
- 财星流年：财运变化、投资
- 食伤流年：才华展现、创业
- 比劫流年：竞争、合作

**感情应期：**
- 男命财星年：恋爱、结婚
- 女命官杀年：恋爱、结婚
- 桃花年：异性缘好
- 婚姻宫动年：感情变化

**健康应期：**
- 五行受克年：对应身体部位注意
- 冲克日主年：整体健康注意
- 伏吟反吟年：重大健康变化

**财运应期：**
- 财星流年：财运变化
- 食伤流年：赚钱机会
- 比劫流年：破财可能
- 财库冲开年：发财机会

## 八、注意事项

1. **大运为重**：大运决定十年大趋势
2. **流年为应**：流年决定具体应事时间
3. **原局为本**：原局决定层次和潜力
4. **综合判断**：原局 + 大运 + 流年综合看
5. **灵活应用**：不可死板套用
