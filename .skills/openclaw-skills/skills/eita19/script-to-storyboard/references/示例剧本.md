# 示例剧本：热带海岸沙滩派对夜景

## 场景概述

热带海岸沙滩派对夜景，史诗级非洲部落风格。20多名非洲土著人围着篝火载歌载舞，头戴华丽羽毛头饰，身穿精致草裙，脸上有彩色部落图腾彩绘。天空各种绚丽多彩的烟花不停绽放，有人弹奏非洲鼓和尤克里里。

## 详细剧本

### 场景1：开篇
热带海滩夜色，椰树剪影，篝火点燃，人群开始聚集。海风轻拂，远处传来海浪声。

### 场景2：篝火特写
篝火燃烧特写，火焰随风狂舞，火星向四周飞溅。火焰越来越大，突然爆发火势窜天，火苗粒子满屏飘洒。

### 场景3：集体舞蹈
多人围着篝火多层舞蹈队形。有人单脚站立旋转身体，草裙展开成圆盘状；有人面对面拍手对跳，身体前倾后仰幅度超过90度；有人高高跃起空中转体，羽毛头饰狂扬；有人大幅度摇摆身体呼喊"哈！哈！哈！"。人物表情开心激昂。

### 场景4：乐器演奏
侧面多人弹奏非洲鼓和尤克里里，身体随音乐大幅度摇摆。弹奏乐器特效粒子飞溅。

### 场景5：人物特写
面部彩绘特写，眉头紧皱眼神狂热，嘴巴大张呼喊，汗水从额头呈弧线飞溅。天空升起绚烂烟花，红黄蓝绿紫多色绽放，火光照亮人物脸庞。

### 场景6：高光时刻
3人跃起身体后仰成C字形最高点，其他人围着篝火继续大幅度跳舞。满天各种绚丽多彩的烟花不停绽放，火树银花照亮夜空，烟花碎片如流星雨倾泻而下。篝火火焰扶摇直上与烟花交织成璀璨光幕。部落狂欢与烟花盛典达到沸点。

## 人物服装细节

- 羽毛头饰：彩色羽毛层层叠叠展开，孔雀蓝绿色、鸵鸟黑色白色、红色黄色点缀，羽毛长度30-50厘米
- 草裙：棕榈叶编织而成，长度到膝盖，呈A字形展开，草裙上均匀缝制白色贝壳片和黄色琥珀珠子
- 面部彩绘：额头、脸颊、下巴绘制部落图腾，红黄白三色几何线条和圆点图案
- 配饰：手腕戴贝壳手链，脖子戴多圈贝壳项链，脚踝系着铃铛
