# 输出格式

## 输出类型

### ⚡ 今日信号（0-2 条）

判定标准：同时满足"需要行动"+"三天后仍有意义"+"有具体所以呢"

没有同时满足三条的 = 留空。空是正常结果。

格式：
```
⚡ [事件] 一句话事实陈述
  所以呢：跟用户的关系
  行动：具体可执行的建议（P0/P1/P2）
  时效：🔴24h紧急/🟡7天中期/🟢30天长期
  影响分：1-10分
  来源：URL
  置信度：高/中
```

### 🚫 避坑提醒（0-2 条）

判定标准：明确告知用户不该做什么，避坑价值大于机会价值

格式：
```
🚫 [事件] 一句话事实陈述
  避坑原因：为什么不要做
  建议：具体的规避动作
  来源：URL
  置信度：高/中
```

### 🔍 值得留意（2-5 条）

判定标准：满足"具体性门控"，但不需要满足"持久性门控"。跟用户有关联，但可能三天后就不重要了。

格式：
```
🔍 [简述] — 所以呢：一句话
  来源：URL | 置信度：高/中/低
```

### 🧊 冰山一角（0-2 条）

判定标准：必须有明确的三步推理链条。缺一步就不报。

推理结构：大家关注 X → X 会导致 Y → Y 对用户意味着 Z

格式：
```
🧊 [表面] 大家都在关注什么
  [冰山] 真正值得关注的是什么
  [推理] X → Y → Z
  置信度：高/中
```

### 📡 待观察（0-3 条）

判定标准：置信度低但潜力大的早期信号，连续2天有佐证升级为正式信号，7天无进展移除

格式：
```
📡 [事件简述] — 当前佐证：[来源]
  后续关注：需要验证什么信息
```

### 🔇 无信号输出

当前前四类有效信号都是 0 条时的默认输出。

```
🔇 今日无关键信号。

以下为背景信息供了解：
- 一句话快讯 1
- 一句话快讯 2
- 一句话快讯 3
```

这不是失败，是高质量筛选的结果。

## 完整输出示例

```
📡 AI 信号 | 2026-04-12

⚡ DeepSeek V3 开源
  所以呢：可以本地部署做评测裁判，你的横向评测成本降为零
  行动：下载 V3 权重，搭建本地评测环境，替代当前付费 API 裁判
  来源：https://github.com/deepseek-ai/DeepSeek-V3
  置信度：高

🔍 MiniMax M3 发布 — 所以呢：你评测列表里的 M2.7 过时了，下次横评换 M3
  来源：https://minimax.ai/blog | 置信度：中

🔍 豆包 Seed 2.0 Pro 降价 40% — 所以呢：如果你用豆包做 API 代理，成本结构变化
  来源：https://volcengine.com/docs | 置信度：高

🧊 [表面] 大家都在关注 DeepSeek V3 的跑分
  [冰山] 真正值得关注的是开源模型追上闭源后，基于 API 差价的服务模式会萎缩
  [推理] V3 追上闭源 → 中小企业可本地部署 → API 代理差价模式萎缩 → 你需要调整变现方向
  置信度：中

---
💬 反馈：回复 +1 有用 / 没用 / 深入 / 不想看这类 / 想看更多关于 XX
```
