# 行业板块列表 

> 分类: 股票基础数据 | 目录: `股票基础数据` | 索引见 `SKILL.md`


### 描述

查询指定交易日期的行业板块列表，默认查询最新一天板块数据，可指定查询日期，默认按照涨跌幅排序

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `str` | 是 | "F7" | 接口ID |
| `tradeDate` | `str` | 否 | - | 交易日日期，示例：2026-03-18 |
| `pageNum` | `int` | 否 | `0` | 分页偏移量，从 0 开始（第 1 页为 0，第 2 页为 pageSize） |
| `pageSize` | `int` | 否 | `10` | 每页条数 |

### 返回参数

ResultData.data 为 List，元素为行业板块数据

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `tradeDate` | `str` | 否 | 交易日日期 |
| `industryCode` | `str` | 否 | 行业代码 |
| `nameLevelOne` | `str` | 否 | 一级板块名称 |
| `nameLevelTwo` | `str` | 否 | 二级板块名称 |
| `nameLevelThree` | `str` | 否 | 三级板块名称 |
| `industryChgPct` | `str` | 否 | 板块涨跌幅 |

### 调用示例

```python
from equal_data import EqualDataApi  
api = EqualDataApi('your API_KEY') 
# 调用接口
data = api.query_equal_data(
    interfaceId="F7",
    tradeDate=None,  # str  # 默认: 
    pageNum=None,  # int  # 默认: 0
    pageSize=None,  # int  # 默认: 10
)
```
