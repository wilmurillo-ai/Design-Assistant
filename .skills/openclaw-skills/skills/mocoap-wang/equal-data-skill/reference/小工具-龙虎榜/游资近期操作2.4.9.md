# 游资近期操作2.4.9

> 分类: 龙虎榜 | 目录: `小工具-龙虎榜` | 索引见 `SKILL.md`


### 描述

查询游资在指定交易日附近的近期操作明细

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "15" | 接口ID |
| `hotMoneyName` | `String` | 是 | - | 游资名称 |
| `tradeDate` | `String` | 是 | - | 交易日期 |

### 返回参数

游资近期操作汇总以及个股明细

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `hotMoneyName` | `String` | 是 | 游资名称 |
| `listNum` | `Integer` | 否 | 上榜次数 |
| `buyNum` | `Integer` | 否 | 买入个股数量 |
| `sellNum` | `Integer` | 否 | 卖出个股数量 |
| `buyAmountClean` | `BigDecimal` | 否 | 净买入金额 |
| `children` | `List<HotMoneyOperateVo.DataVo>` | 否 | 近期操作个股明细列表 |

**`children`** 数组元素结构：

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `stockCode` | `String` | 否 | 股票代码 |
| `stockName` | `String` | 否 | 股票名称 |
| `buyAmount` | `BigDecimal` | 否 | 买入金额 |
| `sellAmount` | `BigDecimal` | 否 | 卖出金额 |
| `buyAmountClean` | `BigDecimal` | 否 | 净买入金额 |
| `pctChg` | `BigDecimal` | 否 | 当日涨跌幅 |
| `periodDays` | `Integer` | 否 | 连续N个交易日 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="15",
    requestType="GET",
    hotMoneyName=None,  # String  # 默认: 
    tradeDate=None,  # String  # 默认: 
)
```
