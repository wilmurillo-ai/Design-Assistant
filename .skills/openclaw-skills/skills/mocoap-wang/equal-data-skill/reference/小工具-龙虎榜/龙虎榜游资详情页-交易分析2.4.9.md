# 龙虎榜游资详情页-交易分析2.4.9

> 分类: 龙虎榜 | 目录: `小工具-龙虎榜` | 索引见 `SKILL.md`


### 描述

获取指定游资的交易分析数据

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "13" | 接口ID |
| `hotMoney` | `String` | 是 | - | 游资名称 |

### 返回参数

游资交易分析数据，包括最佳上涨概率、平均涨幅及图表数据

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `odds` | `BigDecimal` | 否 | 最佳上涨概率 |
| `zdfAvg` | `BigDecimal` | 否 | 最佳平均涨幅 |
| `riseTrackDay` | `Integer` | 否 | 上涨概率跟踪天数 |
| `zdfAvgTrackDay` | `Integer` | 否 | 涨跌幅跟踪天数 |
| `listNum` | `Integer` | 否 | 上买榜次数 |
| `chartDataList` | `List<DragonTradeAnalysisVO.AnalysisChartData>` | 否 | 图表数据列表 |

**`chartDataList`** 数组元素结构：

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `tradeDateDesc` | `String` | 否 | 交易日描述（如T、T+1等） |
| `riseRatio` | `BigDecimal` | 否 | 上涨概率 |
| `riseNum` | `Integer` | 否 | 上升个股数 |
| `fallNum` | `Integer` | 否 | 下跌个股数 |
| `zdfAvg` | `BigDecimal` | 否 | 平均涨幅 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="13",
    requestType="GET",
    hotMoney=None,  # String  # 默认: 
)
```
