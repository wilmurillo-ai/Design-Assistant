# 龙虎榜胜率排行榜2.4.9

> 分类: 龙虎榜 | 目录: `小工具-龙虎榜` | 索引见 `SKILL.md`


### 描述

获取龙虎榜游资胜率排行榜

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "9" | 接口ID |
| `sortFlag` | `Integer` | 否 | `0` | 排序标记（0 默认） |
| `listNum` | `Integer` | 否 | `10` | 榜单数量，默认10 |
| `isNew` | `Integer` | 否 | `0` | 是否仅看新上榜（0 否） |

### 返回参数

龙虎榜游资胜率排行榜列表

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `infoId` | `String` | 否 | 游资infoId |
| `infoType` | `Integer` | 否 | 信息类型 |
| `hotMoney` | `String` | 是 | 游资名称 |
| `sort` | `Integer` | 否 | 排序序号 |
| `riseOdds` | `BigDecimal` | 否 | 上涨概率 |
| `avgOdds` | `BigDecimal` | 否 | 平均涨幅 |
| `trackDayNum` | `Integer` | 否 | 跟踪天数 |
| `traceMonth` | `Integer` | 否 | 跟踪月数 |
| `listNum` | `Integer` | 否 | 上榜次数 |
| `tradeDay` | `String` | 否 | 交易日期 |
| `tradeDayDesc` | `String` | 否 | 交易日描述（今日/昨日/具体日期） |
| `tradeRecordList` | `List<HotMoneyListSearchVo>` | 否 | 操作买入的股票信息列表 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="9",
    requestType="GET",
    sortFlag=None,  # Integer  # 默认: 0
    listNum=None,  # Integer  # 默认: 10
    isNew=None,  # Integer  # 默认: 0
)
```
