# 营业部操作明细

> 分类: 龙虎榜 | 目录: `小工具-龙虎榜` | 索引见 `SKILL.md`


### 描述

查询指定营业部的历史操作明细

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "6" | 接口ID |
| `deptName` | `String` | 是 | - | 营业部全称 |
| `sortName` | `Integer` | 否 | - | 排序字段 1-交易日期 2-净买额 |
| `sortType` | `Integer` | 否 | - | 排序类型 1-倒序 2-正序 |
| `pageIndex` | `Integer` | 是 | - | 当前页从0开始 |
| `pageSize` | `Integer` | 是 | - | 每页显示记录数 |

### 返回参数

营业部操作明细列表

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `stockCode` | `String` | 是 | 股票代码 |
| `stockName` | `String` | 否 | 股票名称 |
| `tradeDate` | `String` | 否 | 交易日期 |
| `buyAmount` | `BigDecimal` | 否 | 买入金额 |
| `sellAmount` | `BigDecimal` | 否 | 卖出金额 |
| `buyAmountClean` | `BigDecimal` | 否 | 净买入金额 |
| `periodDays` | `Integer` | 否 | 连续N个交易日 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="6",
    requestType="GET",
    deptName=None,  # String  # 默认: 
    sortName=None,  # Integer  # 默认: 
    sortType=None,  # Integer  # 默认: 
    pageIndex=None,  # Integer  # 默认: 
    pageSize=None,  # Integer  # 默认: 
)
```
