# 龙虎榜单-营业部榜2.4.9

> 分类: 龙虎榜 | 目录: `小工具-龙虎榜` | 索引见 `SKILL.md`


### 描述

获取指定交易日的龙虎榜营业部榜单

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "12" | 接口ID |
| `tradeDate` | `String` | 是 | - | 交易日期 |

### 返回参数

指定交易日的营业部榜数据列表

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `deptName` | `String` | 是 | 营业部名称 |
| `hotMoneyNames` | `List<String>` | 否 | 关联游资名称列表 |
| `buyStockList` | `List<DeptListVo.StockInfo>` | 否 | 买入股票列表 |
| `sellStockList` | `List<DeptListVo.StockInfo>` | 否 | 卖出股票列表 |
| `buyNum` | `Integer` | 否 | 买入个股数量 |
| `sellNum` | `Integer` | 否 | 卖出个股数量 |
| `buyAmount` | `BigDecimal` | 否 | 买入金额 |
| `sellAmount` | `BigDecimal` | 否 | 卖出金额 |
| `buyAmountClean` | `BigDecimal` | 否 | 净买入金额 |

**`buyStockList`** 数组元素结构：

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `stockCode` | `String` | 否 | 买入股票代码 |
| `stockName` | `String` | 否 | 买入股票名称 |
| `periodDays` | `Integer` | 否 | 买入连续N日标签 |

**`sellStockList`** 数组元素结构：

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `stockCode` | `String` | 否 | 卖出股票代码 |
| `stockName` | `String` | 否 | 卖出股票名称 |
| `periodDays` | `Integer` | 否 | 卖出连续N日标签 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="12",
    requestType="GET",
    tradeDate=None,  # String  # 默认: 
)
```
