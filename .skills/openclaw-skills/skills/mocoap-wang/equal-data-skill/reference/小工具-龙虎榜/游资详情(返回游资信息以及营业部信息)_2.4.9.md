# 游资详情(返回游资信息以及营业部信息) 2.4.9

> 分类: 龙虎榜 | 目录: `小工具-龙虎榜` | 索引见 `SKILL.md`


### 描述

查询游资详情，返回游资信息及相关营业部信息

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "7" | 接口ID |
| `hotMoneyName` | `String` | 是 | - | 游资名称 |

### 返回参数

游资详情数据，包含游资统计与相关营业部信息

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `hotMoneyName` | `String` | 是 | 游资名称 |
| `buyAmountClean` | `BigDecimal` | 否 | 净买入金额 |
| `children` | `List<HotMoneyListVo.DataVo>` | 否 | 游资操作个股明细列表 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="7",
    requestType="GET",
    hotMoneyName=None,  # String  # 默认: 
)
```
