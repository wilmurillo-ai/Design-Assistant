# 龙虎榜日期列表

> 分类: 龙虎榜 | 目录: `小工具-龙虎榜` | 索引见 `SKILL.md`


### 描述

获取龙虎榜可用的历史交易日期列表

### 请求参数

*无参数*

### 返回参数

龙虎榜日期列表，按时间倒序或业务规则排序

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `tradeDates` | `List<String>` | 是 | 可查询的龙虎榜交易日期列表 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="3",
    requestType="GET",
)
```
