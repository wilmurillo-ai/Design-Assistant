# 龙虎榜单-游资榜2.4.9

> 分类: 龙虎榜 | 目录: `小工具-龙虎榜` | 索引见 `SKILL.md`


### 描述

获取指定交易日的龙虎榜游资榜单

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "11" | 接口ID |
| `tradeDate` | `String` | 是 | - | 交易日期 |

### 返回参数

指定交易日的游资榜数据列表

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `hotMoneyName` | `String` | 是 | 游资名称 |
| `buyAmountClean` | `BigDecimal` | 否 | 净买入额 |
| `children` | `List<HotMoneyListVo.DataVo>` | 否 | 游资操作个股明细 |
| `jumpUrl` | `String` | 否 | 跳转URL |

**`children`** 数组元素结构：

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `stockCode` | `String` | 否 | 个股代码 |
| `stockName` | `String` | 否 | 个股名称 |
| `pctChg` | `BigDecimal` | 否 | 个股涨跌幅 |
| `buyAmount` | `BigDecimal` | 否 | 买入金额 |
| `sellAmount` | `BigDecimal` | 否 | 卖出金额 |
| `buyAmountClean` | `BigDecimal` | 否 | 净买入金额 |
| `buyAmountCleanNumber` | `BigDecimal` | 否 | 净买入金额原始值 |
| `periodDays` | `Integer` | 否 | 连续N个交易日 |
| `hotMoneyList` | `List<String>` | 否 | 该股票其他游资操作列表 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="11",
    requestType="GET",
    tradeDate=None,  # String  # 默认: 
)
```
