# 龙虎榜日期列表（近1年/近1月）2.4.9

> 分类: 龙虎榜 | 目录: `小工具-龙虎榜` | 索引见 `SKILL.md`


### 描述

获取龙虎榜近1年或近1月的交易日期列表

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "14" | 接口ID |
| `flag` | `Integer` | 是 | - | 日期列表范围 0：近1年 1：近1月 |

### 返回参数

指定范围内的龙虎榜交易日期列表

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `tradeDates` | `List<String>` | 是 | 交易日期列表 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="14",
    requestType="GET",
    flag=None,  # Integer  # 默认: 
)
```
