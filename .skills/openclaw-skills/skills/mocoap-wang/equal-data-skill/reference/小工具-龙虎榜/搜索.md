# 搜索

> 分类: 龙虎榜 | 目录: `小工具-龙虎榜` | 索引见 `SKILL.md`


### 描述

根据名称搜索游资或营业部

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "8" | 接口ID |
| `name` | `String` | 是 | - | 游资或营业部名称 |

### 返回参数

搜索结果列表，包括游资/营业部及相关股票信息

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `name` | `String` | 是 | 名称（游资或营业部） |
| `shortName` | `String` | 否 | 简称 |
| `type` | `Integer` | 否 | 类型 1-游资 2-营业部 |
| `stockCode` | `String` | 否 | 股票代码 |
| `stockCodeShow` | `String` | 否 | 展示用股票代码 |
| `infoId` | `String` | 否 | infoId |
| `dataType` | `Integer` | 否 | 数据类型 1-游资或营业部 2-上市公司 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="8",
    requestType="GET",
    name=None,  # String  # 默认: 
)
```
