# 首页-龙虎榜（前一个版本）

> 分类: 龙虎榜 | 目录: `小工具-龙虎榜` | 索引见 `SKILL.md`


### 描述

首页-龙虎榜（前一个版本），返回龙虎榜首页数据（兼容旧版）

### 请求参数

*无参数*

### 返回参数

龙虎榜首页数据（旧版本兼容接口），包含首页展示所需的汇总信息与列表信息

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `homepageSummary` | `object` | 否 | 首页概览统计信息（如成交额、上榜数量等） |
| `homepageSections` | `array` | 否 | 首页各模块数据列表，如热门个股榜、热门游资榜等 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="1",
    requestType="GET",
)
```
