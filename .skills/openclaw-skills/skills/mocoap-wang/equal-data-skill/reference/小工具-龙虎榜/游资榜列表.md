# 游资榜列表

> 分类: 龙虎榜 | 目录: `小工具-龙虎榜` | 索引见 `SKILL.md`


### 描述

根据交易日期获取游资榜列表

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "4" | 接口ID |
| `tradeDate` | `String` | 是 | - | 交易日期 |

### 返回参数

指定交易日的游资榜列表

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `hotMoneyName` | `String` | 是 | 游资名称 |
| `buyAmountClean` | `BigDecimal` | 否 | 游资净买入额（原值） |
| `children` | `List<HotMoneyListVo.DataVo>` | 否 | 该游资当日操作的个股明细列表 |
| `jumpUrl` | `String` | 否 | 跳转URL |

**`children`** 数组元素结构：

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `children.stockCode` | `String` | 否 | 个股代码 |
| `children.stockName` | `String` | 否 | 个股名称 |
| `children.pctChg` | `BigDecimal` | 否 | 个股涨跌幅 |
| `children.buyAmount` | `BigDecimal` | 否 | 买入金额 |
| `children.sellAmount` | `BigDecimal` | 否 | 卖出金额 |
| `children.buyAmountClean` | `BigDecimal` | 否 | 净买入金额 |
| `children.buyAmountCleanNumber` | `BigDecimal` | 否 | 净买入金额原始值 |
| `children.periodDays` | `Integer` | 否 | 连续N个交易日 |
| `children.hotMoneyList` | `List<String>` | 否 | 该股票其他游资操作列表 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="4",
    requestType="GET",
    tradeDate=None,  # String  # 默认: 
)
```
