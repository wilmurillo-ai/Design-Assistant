# 龙虎榜单-个股榜2.4.9

> 分类: 龙虎榜 | 目录: `小工具-龙虎榜` | 索引见 `SKILL.md`


### 描述

获取指定交易日的龙虎榜个股榜单

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "10" | 接口ID |
| `tradeDate` | `String` | 是 | - | 交易日期 |

### 返回参数

指定交易日的个股龙虎榜数据列表

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `jumpUrl` | `String` | 否 | 跳转链接 |
| `stockCode` | `String` | 是 | 股票代码 |
| `stockName` | `String` | 否 | 股票名称 |
| `pctChg` | `BigDecimal` | 否 | 今日涨幅 |
| `buyMoneyClean` | `BigDecimal` | 否 | 净买入额 |
| `tradeDay` | `String` | 否 | 交易日 |
| `hotMoneyNameDesc` | `String` | 否 | 知名游资(空格隔开) |
| `periodDays` | `Integer` | 否 | 连续N日的标签 |
| `totalCount` | `Integer` | 否 | 统计的总交易日数 |
| `limitUpCount` | `Integer` | 否 | 涨停交易日数量 |
| `hotMoneyList` | `List<String>` | 否 | 游资相关信息列表 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="10",
    requestType="GET",
    tradeDate=None,  # String  # 默认: 
)
```
