# 游资近1月操作日期列表2.4.9

> 分类: 龙虎榜 | 目录: `小工具-龙虎榜` | 索引见 `SKILL.md`


### 描述

获取游资近1月有操作记录的交易日期列表

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "16" | 接口ID |
| `hotMoney` | `String` | 是 | - | 游资名称 |

### 返回参数

游资近1月操作日期列表

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `tradeDates` | `List<String>` | 是 | 交易日期列表 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="16",
    requestType="GET",
    hotMoney=None,  # String  # 默认: 
)
```
