# 究竟策略最新交易明细

> 分类: 究竟策略 | 目录: `小工具-究竟策略` | 索引见 `SKILL.md`


### 描述

获取究竟策略最新交易明细

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "43" | 接口ID |
| `strategyId` | `String` | 否 | - | 策略ID |

### 返回参数

究竟策略最新交易明细列表

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `orderBookId` | `String` | 否 | 订单ID（关联买卖） |
| `tradingDay` | `String` | 否 | 交易日 |
| `stockCode` | `String` | 是 | 股票代码 |
| `stockName` | `String` | 否 | 股票名称 |
| `side` | `String` | 否 | 交易方向 buy：开仓 sell：平仓 |
| `avgPrice` | `BigDecimal` | 否 | 成交均价 |
| `quantity` | `Long` | 否 | 成交数量 |
| `isBonus` | `Integer` | 否 | 是否分红 0-否 1-是 |
| `summaryReturns` | `BigDecimal` | 否 | 累计盈亏（sell 时取值） |
| `holdCost` | `BigDecimal` | 否 | 单笔持仓成本（buy 时取值） |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="43",
    requestType="GET",
    strategyId=None,  # String  # 默认: 
)
```
