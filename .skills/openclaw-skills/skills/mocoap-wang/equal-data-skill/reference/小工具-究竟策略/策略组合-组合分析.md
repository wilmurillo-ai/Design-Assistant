# 策略组合-组合分析

> 分类: 究竟策略 | 目录: `小工具-究竟策略` | 索引见 `SKILL.md`


### 描述

获取策略组合分析数据

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "48" | 接口ID |
| `strategyId` | `String` | 是 | - | 策略ID |
| `periodType` | `Integer` | 是 | `0` | 周期类型 0-近1年 2-近2年 |

### 返回参数

策略组合分析数据列表

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `strategyName` | `String` | 是 | 策略名称 |
| `strategyId` | `Integer` | 是 | 策略Id |
| `volatility` | `BigDecimal` | 否 | 波动率 |
| `sharpeRatio` | `BigDecimal` | 否 | 夏普率 |
| `drawdown` | `BigDecimal` | 否 | 最大回撤 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="48",
    requestType="GET",
    strategyId=None,  # String  # 默认: 
    periodType=None,  # Integer  # 默认: 0
)
```
