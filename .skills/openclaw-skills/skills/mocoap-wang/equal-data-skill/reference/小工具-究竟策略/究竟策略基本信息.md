# 究竟策略基本信息

> 分类: 究竟策略 | 目录: `小工具-究竟策略` | 索引见 `SKILL.md`


### 描述

获取指定策略的基本信息

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "38" | 接口ID |
| `strategyId` | `String` | 是 | - | 策略ID |

### 返回参数

究竟策略基本信息

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `strategyName` | `String` | 是 | 策略名称 |
| `strategyId` | `String` | 是 | 策略ID |
| `tradingDay` | `String` | 否 | 交易日 |
| `recent1Month` | `BigDecimal` | 否 | 近1月收益 |
| `recent3Month` | `BigDecimal` | 否 | 近3月收益 |
| `recent6Month` | `BigDecimal` | 否 | 近6月收益 |
| `recent1Year` | `BigDecimal` | 否 | 近1年收益 |
| `recent3Year` | `BigDecimal` | 否 | 近3年收益 |
| `recent5Year` | `BigDecimal` | 否 | 近5年收益 |
| `yearToDay` | `BigDecimal` | 否 | 今年以来收益 |
| `totalRate` | `BigDecimal` | 否 | 累计收益率 |
| `holdingCount` | `Integer` | 否 | 持仓个数 0-空仓 |
| `createDateNum` | `Integer` | 否 | 创建天数 |
| `createDate` | `String` | 否 | 策略创建日期 |
| `unitNetValue` | `BigDecimal` | 否 | 净值 |
| `dailyReturns` | `BigDecimal` | 否 | 日收益率（组合） |
| `buyNum` | `Integer` | 否 | 买入股票数量 |
| `sellNum` | `Integer` | 否 | 卖出股票数量 |
| `signalDate` | `String` | 否 | 调仓信号最新日期 |
| `bestRatio` | `BigDecimal` | 否 | 最好看的收益指标 |
| `bestRatioType` | `Integer` | 否 | 最好看的收益指标类型 |
| `selectedShowStatus` | `Integer` | 否 | 近3年、近5年展示状态 |
| `rule` | `String` | 否 | 调仓规则 |
| `investRule` | `String` | 否 | 投资规则 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="38",
    requestType="GET",
    strategyId=None,  # String  # 默认: 
)
```
