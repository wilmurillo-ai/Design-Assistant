# 策略组合-调仓信号

> 分类: 究竟策略 | 目录: `小工具-究竟策略` | 索引见 `SKILL.md`


### 描述

获取策略组合调仓信号

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "45" | 接口ID |
| `strategyId` | `String` | 是 | - | 策略ID |

### 返回参数

策略组合调仓信号数据

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `tradingDay` | `String` | 否 | 交易日 |
| `timeFlag` | `Integer` | 否 | 时间标记 0：今天 1：明日 2：最新 |
| `nextNearestDay` | `String` | 否 | 最近交易日 |
| `buyList` | `List<StockInfoVo>` | 否 | 买入股票列表 |
| `sellList` | `List<StockInfoVo>` | 否 | 卖出股票列表 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="45",
    requestType="GET",
    strategyId=None,  # String  # 默认: 
)
```
