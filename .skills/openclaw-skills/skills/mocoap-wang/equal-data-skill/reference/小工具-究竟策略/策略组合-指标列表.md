# 策略组合-指标列表

> 分类: 究竟策略 | 目录: `小工具-究竟策略` | 索引见 `SKILL.md`


### 描述

获取策略组合指标列表

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "47" | 接口ID |
| `strategyId` | `String` | 是 | - | 策略ID |

### 返回参数

策略组合指标列表

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `indexCode` | `Integer` | 是 | 指标编码 |
| `indexName` | `String` | 是 | 指标名称 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="47",
    requestType="GET",
    strategyId=None,  # String  # 默认: 
)
```
