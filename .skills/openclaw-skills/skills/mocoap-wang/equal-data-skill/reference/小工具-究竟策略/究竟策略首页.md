# 究竟策略首页

> 分类: 究竟策略 | 目录: `小工具-究竟策略` | 索引见 `SKILL.md`


### 描述

获取究竟策略首页数据

### 请求参数

*无参数*

### 返回参数

究竟策略首页数据

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `strategyName` | `String` | 是 | 策略名称 |
| `strategyId` | `String` | 是 | 策略ID |
| `bestRatio` | `BigDecimal` | 否 | 最好看的收益指标 |
| `netVal` | `BigDecimal` | 否 | 最新净值 |
| `moduleRightsBaseInfo` | `RightsConstant.ModuleRightsBaseInfo` | 否 | 权益内容 |
| `buyNum` | `Long` | 否 | 购买数量 |
| `sellNum` | `Long` | 否 | 卖出数量 |
| `tradingDate` | `String` | 否 | 交易日期 |
| `nextNearestDay` | `String` | 否 | 最近交易日 |
| `dailyReturns` | `BigDecimal` | 否 | 当日涨跌 |
| `timeFlag` | `Integer` | 否 | 时间标记 0:今日 1:明日 2:最新 |
| `indexCode` | `String` | 否 | 指标编码 |
| `indexName` | `String` | 否 | 指标名称 |
| `redisKey` | `String` | 否 | 缓存key |
| `tradeDetialUrl` | `String` | 否 | 交易明细跳转链接 |
| `strategyDetialUrl` | `String` | 否 | 策略详情跳转链接 |
| `trendChartUrl` | `String` | 否 | 走势图地址 |
| `trendDataList` | `List<String>` | 否 | 趋势数据列表 |
| `tradeRecord` | `String` | 否 | 交易记录原始串 |
| `dealRecordList` | `List<StrategyHoldingDetialVo>` | 否 | 交易明细列表 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="37",
    requestType="GET",
)
```
