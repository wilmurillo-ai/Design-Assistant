# 策略组合当前持仓

> 分类: 究竟策略 | 目录: `小工具-究竟策略` | 索引见 `SKILL.md`


### 描述

获取策略组合当前持仓列表

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "41" | 接口ID |
| `strategyId` | `String` | 否 | - | 策略ID |
| `tradeDate` | `String` | 否 | - | 交易日期 |
| `pageIndex` | `Integer` | 是 | - | 当前页从0开始 |
| `pageSize` | `Integer` | 是 | - | 每页显示记录数 |

### 返回参数

策略组合当前持仓列表

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `stockCode` | `String` | 是 | 股票代码 |
| `stockName` | `String` | 否 | 股票名称 |
| `holdingRatio` | `BigDecimal` | 否 | 持仓比例 |
| `marketValue` | `BigDecimal` | 否 | 持仓最新市值 |
| `holdingCost` | `BigDecimal` | 否 | 持仓总成本 |
| `holdingCostNumber` | `BigDecimal` | 否 | 持仓总成本（未转换） |
| `dailyPnl` | `BigDecimal` | 否 | 当日盈亏 |
| `quantity` | `Long` | 否 | 持仓数量 |
| `quantityNumber` | `Long` | 否 | 持仓数量（未转换） |
| `holdingDays` | `Integer` | 否 | 持仓天数 |
| `holdingAvgPrice` | `BigDecimal` | 否 | 持仓成本均价 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="41",
    requestType="GET",
    strategyId=None,  # String  # 默认: 
    tradeDate=None,  # String  # 默认: 
    pageIndex=None,  # Integer  # 默认: 
    pageSize=None,  # Integer  # 默认: 
)
```
