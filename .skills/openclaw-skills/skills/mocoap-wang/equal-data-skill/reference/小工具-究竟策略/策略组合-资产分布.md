# 策略组合-资产分布

> 分类: 究竟策略 | 目录: `小工具-究竟策略` | 索引见 `SKILL.md`


### 描述

获取策略组合资产分布数据

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "46" | 接口ID |
| `strategyId` | `String` | 是 | - | 策略ID |
| `dateStr` | `String` | 是 | - | 交易日期 |

### 返回参数

策略组合资产分布数据

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `marketValue` | `BigDecimal` | 否 | 持仓市值 |
| `marketValueRatio` | `BigDecimal` | 否 | 持仓市值占比 |
| `cash` | `BigDecimal` | 否 | 现金 |
| `cashRatio` | `BigDecimal` | 否 | 现金占比 |
| `totalValue` | `BigDecimal` | 否 | 总资产 |
| `totalValueRatio` | `BigDecimal` | 否 | 总资产占比 |
| `todayAddMoney` | `BigDecimal` | 否 | 今日转入资金 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="46",
    requestType="GET",
    strategyId=None,  # String  # 默认: 
    dateStr=None,  # String  # 默认: 
)
```
