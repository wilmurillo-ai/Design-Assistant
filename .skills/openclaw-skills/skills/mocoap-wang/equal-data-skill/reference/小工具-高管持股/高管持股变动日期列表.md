# 高管持股变动日期列表

> 分类: 高管持股 | 目录: `小工具-高管持股` | 索引见 `SKILL.md`


### 描述

获取高管持股变动日期列表

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "57" | 接口ID |
| `stockCode` | `String` | 是 | - | 股票代码 |
| `changeType` | `Integer` | 是 | - | 变动类型（增持：1，减持：-1） |

### 返回参数

高管持股变动日期列表

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `tradeDates` | `List<String>` | 是 | 高管持股变动日期列表 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="57",
    requestType="GET",
    stockCode=None,  # String  # 默认: 
    changeType=None,  # Integer  # 默认: 
)
```
