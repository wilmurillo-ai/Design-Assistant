# 个股详情/动态详情--增减持操作明细数据

> 分类: 高管持股 | 目录: `小工具-高管持股` | 索引见 `SKILL.md`


### 描述

获取个股增减持操作明细数据

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "52" | 接口ID |
| `pageIndex` | `Integer` | 是 | - | 当前页从0开始 |
| `pageSize` | `Integer` | 是 | - | 每页显示记录数 |
| `changeType` | `Integer` | 否 | `0` | 变动类型（增持：1，减持：-1），0或不传表示全部 |
| `stockCode` | `String` | 是 | - | 股票代码 |
| `managerName` | `String` | 否 | - | 高管名称 |
| `changeDate` | `String` | 否 | - | 变动日期 |

### 返回参数

个股增减持操作明细列表

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `stockCode` | `String` | 是 | 股票代码 |
| `stockName` | `String` | 否 | 股票名称 |
| `totalQuantity` | `Long` | 否 | 增减持数量 |
| `totalAmount` | `BigDecimal` | 否 | 总金额 |
| `ratioOfTotal` | `BigDecimal` | 否 | 占总股本比 |
| `ratioOfCirculate` | `BigDecimal` | 否 | 占流通股比 |
| `actualChangeQuantity` | `Long` | 否 | 净增持/减持数量 |
| `avgPrice` | `BigDecimal` | 否 | 增减持均价 |
| `chgPct` | `BigDecimal` | 否 | 当日涨跌幅 |
| `changeType` | `String` | 否 | 持股变动类型 增持/减持 |
| `mangerName` | `String` | 否 | 高管姓名 |
| `changeName` | `String` | 否 | 变动人姓名 |
| `position` | `String` | 否 | 高管职位 |
| `relationDesc` | `String` | 否 | 变动人与高管关系 |
| `zdOdds` | `BigDecimal` | 否 | 上涨概率 |
| `trackDayNum` | `Integer` | 否 | 跟踪天数 |
| `changeDate` | `String` | 否 | 变动日期 |
| `changeReason` | `String` | 否 | 变动原因（增减持方式） |
| `kinshipNum` | `Integer` | 否 | 是否有人物关系图 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="52",
    requestType="GET",
    pageIndex=None,  # Integer  # 默认: 
    pageSize=None,  # Integer  # 默认: 
    changeType=None,  # Integer  # 默认: 0
    stockCode=None,  # String  # 默认: 
    managerName=None,  # String  # 默认: 
    changeDate=None,  # String  # 默认: 
)
```
