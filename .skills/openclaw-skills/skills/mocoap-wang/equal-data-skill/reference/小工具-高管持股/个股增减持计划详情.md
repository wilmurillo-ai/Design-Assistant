# 个股增减持计划详情

> 分类: 高管持股 | 目录: `小工具-高管持股` | 索引见 `SKILL.md`


### 描述

获取个股增减持计划详情数据

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "53" | 接口ID |
| `pageIndex` | `Integer` | 是 | - | 当前页从0开始 |
| `pageSize` | `Integer` | 是 | - | 每页显示记录数 |
| `stockCode` | `String` | 是 | - | 股票代码 |
| `managerName` | `String` | 否 | - | 高管名称 |

### 返回参数

个股增减持计划详情列表

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `stockCode` | `String` | 是 | 股票代码 |
| `stockName` | `String` | 否 | 股票名称 |
| `companyName` | `String` | 否 | 公司全称 |
| `type` | `String` | 否 | 类型：增持/减持 |
| `announcementDate` | `String` | 否 | 公告日期 |
| `pdfUrl` | `String` | 否 | 公告链接 |
| `progress` | `String` | 否 | 进度(进行中、已完成) |
| `holderPositionDesc` | `String` | 否 | 股东名称以及职位 |
| `transactionRatio` | `BigDecimal` | 否 | 增减持数量占总股本比例 |
| `managerInfoList` | `List<ManagerSharesPlanDetailVo.ManagerInfo>` | 否 | 计划涉及的高管信息列表 |

**`managerInfoList`** 数组元素结构：

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `personEsId` | `Integer` | 否 | 高管ESId |
| `personInfoId` | `String` | 否 | 高管InfoId |
| `managerName` | `String` | 否 | 高管姓名 |
| `position` | `String` | 否 | 职务 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="53",
    requestType="GET",
    pageIndex=None,  # Integer  # 默认: 
    pageSize=None,  # Integer  # 默认: 
    stockCode=None,  # String  # 默认: 
    managerName=None,  # String  # 默认: 
)
```
