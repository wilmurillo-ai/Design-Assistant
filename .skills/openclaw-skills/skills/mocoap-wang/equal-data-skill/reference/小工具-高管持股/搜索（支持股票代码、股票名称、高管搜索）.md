# 搜索（支持股票代码、股票名称、高管搜索）

> 分类: 高管持股 | 目录: `小工具-高管持股` | 索引见 `SKILL.md`


### 描述

高管持股搜索（支持股票代码、股票名称、高管搜索）

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "54" | 接口ID |
| `keyword` | `String` | 是 | - | 股票代码、股票名称或高管名称 |

### 返回参数

高管持股搜索结果汇总

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `stockList` | `List<SearchTotalVo>` | 否 | 股票搜索结果列表 |
| `peopleList` | `List<SearchTotalVo>` | 否 | 人物搜索结果列表 |
| `agencyList` | `List<SearchTotalVo>` | 否 | 机构搜索结果列表 |
| `fundList` | `List<SearchTotalVo>` | 否 | 基金搜索结果列表 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="54",
    requestType="GET",
    keyword=None,  # String  # 默认: 
)
```
