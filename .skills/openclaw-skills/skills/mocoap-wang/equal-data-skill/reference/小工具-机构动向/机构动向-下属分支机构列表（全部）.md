# 机构动向-下属分支机构列表（全部）

> 分类: 机构动向 | 目录: `小工具-机构动向` | 索引见 `SKILL.md`


### 描述

机构动向-下属分支机构列表（全部）

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "35" | 接口ID |
| `agencyOnlyId` | `String` | 是 | - | agencyOnlyId |

### 返回参数

机构动向-下属分支机构列表（全部） 返回数据

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `holderName` | `String` | 否 | 分支机构名称 |
| `infoId` | `String` | 否 | 分支机构infoId |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="35",
    requestType="GET",
    agencyOnlyId=None,  # String  # 默认: 
)
```
