# 机构本期持股列表

> 分类: 机构动向 | 目录: `小工具-机构动向` | 索引见 `SKILL.md`


### 描述

机构本期持股列表

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "27" | 接口ID |
| `agencyOnlyId` | `String` | 是 | - | 机构唯一标识  |
| `holderName` | `String` | 否 | - | 分支机构-股东名称(全部不用传)() |
| `branchInfoId` | `String` | 否 | - | 分支机构-infoId |
| `pageIndex` | `Integer` | 是 | - | 当前页从0开始 |
| `pageSize` | `Integer` | 是 | - | 每页显示记录数 |
| `sort` | `Integer` | 是 | `0` | 排序 0-默认（按照公告日期降序） 1-持股占比 降序 2-持股占比 升序 |

### 返回参数

机构本期持股列表 返回数据

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `stockCode` | `String` | 是 | 股票代码 |
| `stockName` | `String` | 否 | 股票名称 |
| `marketValue` | `BigDecimal` | 否 | 现市值 |
| `holderNum` | `Long` | 否 | 现持股数 |
| `holderNumOld` | `Long` | 否 | 原持股数 |
| `changeNum` | `Long` | 否 | 主动变动股数 |
| `changeNumStr` | `String` | 否 | 变动股数（单位自适应） |
| `holderNumStr` | `String` | 否 | 现持股数（单位自适应） |
| `holdingCost` | `BigDecimal` | 否 | 持股成本 |
| `endTermPrice` | `BigDecimal` | 否 | 期末价格 |
| `priceAvg` | `BigDecimal` | 否 | 均价 |
| `reportDate` | `String` | 否 | 变动期（自匹配季度） |
| `rateRatio` | `BigDecimal` | 否 | 预估收益率 |
| `isBeforeList` | `Integer` | 否 | 是否上市前就持有公司股票 0-否 1-是 |
| `holdingAvgPrice` | `BigDecimal` | 否 | 持仓成本均价 |
| `publicDate` | `String` | 否 | 公告日期 |
| `changeDirect` | `Integer` | 否 | 变动方向 |
| `dataDate` | `String` | 否 | 数据日期 |
| `holderName` | `String` | 否 | 股东名称 |
| `agencyOnlyId` | `String` | 否 | 机构唯一ID |
| `holderOnlyStr` | `String` | 否 | 股东唯一标识串 |
| `agencyType` | `Integer` | 否 | 机构类型 |
| `type` | `Integer` | 否 | 比例类型 0-总股本 1-流通股 |
| `ratio` | `BigDecimal` | 否 | 持股比例 |
| `holderCode` | `String` | 否 | 股东编码 |
| `showHolderName` | `String` | 否 | 展示用股东名称 |
| `holderInfoId` | `String` | 否 | 股东infoId |
| `fundStockQuarter` | `String` | 否 | 基金持仓季度 |
| `fundPositionRatio` | `String` | 否 | 基金持仓比例（占净值比例） |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="27",
    requestType="GET",
    agencyOnlyId=None,  # String  # 默认: 
    holderName=None,  # String  # 默认: 
    branchInfoId=None,  # String  # 默认: 
    pageIndex=None,  # Integer  # 默认: 
    pageSize=None,  # Integer  # 默认: 
    sort=None,  # Integer  # 默认: 0
)
```
