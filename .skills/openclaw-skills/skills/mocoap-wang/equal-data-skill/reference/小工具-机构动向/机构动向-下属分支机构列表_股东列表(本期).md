# 机构动向-下属分支机构列表/股东列表(本期)

> 分类: 机构动向 | 目录: `小工具-机构动向` | 索引见 `SKILL.md`


### 描述

机构动向-下属分支机构列表/股东列表(本期)

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "34" | 接口ID |
| `agencyOnlyId` | `String` | 是 | - | agencyOnlyId |

### 返回参数

机构动向-下属分支机构列表/股东列表(本期) 返回数据

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `showHolderName` | `String` | 否 | 展示用股东名称 |
| `holderOnlyStr` | `String` | 否 | 股东唯一标识串 |
| `holderName` | `String` | 否 | 股东名称 |
| `infoId` | `String` | 否 | 客户端要传的值 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="34",
    requestType="GET",
    agencyOnlyId=None,  # String  # 默认: 
)
```
