# 机构搜索

> 分类: 机构动向 | 目录: `小工具-机构动向` | 索引见 `SKILL.md`


### 描述

机构搜索

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "30" | 接口ID |
| `name` | `String` | 是 | - | 机构名字 |

### 返回参数

机构搜索 返回数据

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `agencyName` | `String` | 是 | 机构名称 |
| `agencyOnlyId` | `String` | 是 | 机构唯一ID |
| `agencyType` | `Integer` | 否 | 机构类型 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="30",
    requestType="GET",
    name=None,  # String  # 默认: 
)
```
