# 机构风云榜-榜单详情数据2.5.0

> 分类: 机构动向 | 目录: `小工具-机构动向` | 索引见 `SKILL.md`


### 描述

机构风云榜-榜单详情数据2.5.0

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "20" | 接口ID |
| `pageIndex` | `Integer` | 是 | - | 当前页从0开始 |
| `pageSize` | `Integer` | 是 | - | 每页显示记录数 |
| `agencyType` | `Integer` | 是 | - | 机构类型 2007-牛散 3001-公募 1003-私募 1006-信托公司 1007-证券公司 <br>1009-保险公司 1015-外资 8000-国家队 1201-知名机构  9999-热门榜单  |
| `reportDate` | `String` | 是 | - | 报告期 yyyy-MM-dd格式，如 2025-09-30 |
| `direc` | `Integer` | 是 | `1` | 榜单类型 1-买入榜 2-卖出榜 34-跌破国家队持仓榜 35-高管增持榜 36-高管减持榜 |
| `sortDesc` | `Integer` | 是 | `0` | 变动比例排序  0-降序 1-升序 |
| `sourceId` | `Integer` | 否 | `2` | 是否走缓存  1-风云榜列表（走缓存） 2-榜单详情（默认；不走缓存） |

### 返回参数

机构风云榜-榜单详情数据2.5.0 返回数据

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `stockCode` | `String` | 是 | 股票代码 |
| `stockName` | `String` | 否 | 股票名称 |
| `productNum` | `Integer` | 否 | 基金数/私募家数... |
| `changeQuantity` | `Long` | 否 | 累计变动股数 |
| `holdRatio` | `BigDecimal` | 否 | 当前持股占比 |
| `thisChangeRatio` | `BigDecimal` | 否 | 持股变动比例 |
| `changeDire` | `String` | 否 | 持股变动方向 |
| `changeDireLabel` | `String` | 否 | 是否含新进/退出标签 |
| `changeMarketValue` | `BigDecimal` | 否 | 持股变动市值 |
| `reportDateFormat` | `String` | 否 | 格式化后的报告期 |
| `reportDate` | `String` | 否 | 报告期 |
| `totalShareHoldNum` | `Long` | 否 | 本期持股总数 |
| `totalShareHoldNumOld` | `Long` | 否 | 上期末持股总数 |
| `cycleRatio` | `String` | 否 | 环比变动 |
| `changeMoney` | `BigDecimal` | 否 | 变动金额 |
| `secIndustryName` | `String` | 否 | 二级行业 |
| `industryCode` | `String` | 否 | 二级板块Id |
| `concept` | `String` | 否 | 概念 |
