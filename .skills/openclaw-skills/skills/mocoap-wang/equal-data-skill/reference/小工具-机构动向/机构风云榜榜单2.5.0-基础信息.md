# 机构风云榜榜单2.5.0-基础信息

> 分类: 机构动向 | 目录: `小工具-机构动向` | 索引见 `SKILL.md`


### 描述

机构风云榜榜单2.5.0-基础信息

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "19" | 接口ID |
| `agencyType` | `Integer` | 否 | - | 机构类型 2007-牛散 3001-公募 1003-私募 1006-信托公司 1007-证券公司 <br>1009-保险公司 1015-外资 8000-国家队 1201-知名机构  全部不传值  |
| `reportDate` | `String` | 是 | - | 报告期 yyyy-MM-dd格式，如 2025-09-30 |
| `direc` | `Integer` | 是 | `1` | 榜单类型 1-买入榜 2-卖出榜 |

### 返回参数

机构风云榜榜单2.5.0-基础信息 返回数据

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `agencyId` | `Integer` | 是 | 机构ID |
| `agencyType` | `Integer` | 是 | 榜单机构类型 |
| `agencyName` | `String` | 是 | 机构名称 |
| `reportDate` | `String` | 是 | 报告期 yyyy-MM-dd |
| `transDirect` | `Integer` | 是 | 操作类型  1-买入榜单 2-卖出榜单 |
| `reportYear` | `String` | 否 | 报告期年 |
| `reportQuarter` | `String` | 否 | 报告期季度 |
| `staticPicUrl` | `String` | 否 | 静态图片地址 |
| `sourceId` | `Integer` | 否 | 来源 1-机构动向 2-股东协同 |
| `firstCategory` | `Integer` | 否 | 一级分类 0-热门榜单 1-买入榜 2-卖出榜 3-协同榜 |
| `title` | `String` | 否 | 类型标题 |
| `iconUrl` | `String` | 否 | 榜单类型对应的 icon地址 |
| `jumpUrl` | `String` | 否 | 跳转页面地址（已拼接参数） |
| `updateDate` | `String` | 否 | 数据更新日期 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="19",
    requestType="GET",
    agencyType=None,  # Integer  # 默认: 
    reportDate=None,  # String  # 默认: 
    direc=None,  # Integer  # 默认: 1
)
```
