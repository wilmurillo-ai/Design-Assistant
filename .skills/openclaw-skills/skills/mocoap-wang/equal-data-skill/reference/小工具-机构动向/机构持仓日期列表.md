# 机构持仓日期列表

> 分类: 机构动向 | 目录: `小工具-机构动向` | 索引见 `SKILL.md`


### 描述

机构持仓日期列表

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "25" | 接口ID |
| `agencyOnlyId` | `String` | 否 | - | 机构唯一标识  |

### 返回参数

机构持仓日期列表 返回数据

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `data` | `List<String>` | 是 | 机构持仓日期列表 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="25",
    requestType="GET",
    agencyOnlyId=None,  # String  # 默认: 
)
```
