# 机构动向-往期市值规模

> 分类: 机构动向 | 目录: `小工具-机构动向` | 索引见 `SKILL.md`


### 描述

机构动向-往期市值规模

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "31" | 接口ID |
| `agencyType` | `Integer` | 否 | - | 机构类型（1015-外资，8000-国家队）  |

### 返回参数

机构动向-往期市值规模 返回数据

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `id` | `Integer` | 否 | 记录ID |
| `agencyType` | `Integer` | 否 | 机构类型代码 |
| `agencyOnlyId` | `String` | 否 | 机构唯一id |
| `dataDate` | `String` | 否 | 数据日期（报告期） |
| `marketValue` | `BigDecimal` | 否 | 市值 |
| `createTime` | `Date` | 否 | 创建时间 |
| `updateTime` | `Date` | 否 | 更新时间 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="31",
    requestType="GET",
    agencyType=None,  # Integer  # 默认: 
)
```
