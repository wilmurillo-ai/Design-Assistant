# 机构风云榜-产品列表 2.5.0

> 分类: 机构动向 | 目录: `小工具-机构动向` | 索引见 `SKILL.md`


### 描述

机构风云榜-产品列表 2.5.0

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "22" | 接口ID |
| `agencyType` | `Integer` | 否 | - |  机构类型 3001-公募 1003-私募 1006-信托公司 1007-证券公司 <br>1009-保险公司 1015-外资 8000-国家队 1201-知名机构  全部不传值  |
| `reportDate` | `String` | 是 | - | 报告期 yyyy-MM-dd格式，如 2025-09-30 |
| `stockCode` | `String` | 是 | - | 股票代码 |
| `direc` | `Integer` | 是 | - | 榜单类型 1-买入榜 2-卖出榜 |

### 返回参数

机构风云榜-产品列表 2.5.0 返回数据

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `stockCode` | `String` | 是 | 股票代码 |
| `stockName` | `String` | 否 | 股票名称 |
| `agencyOnlyId` | `String` | 是 | 唯一Id（页面跳转使用） |
| `changeDire` | `String` | 否 | 持股变动方向 |
| `changeDireDesc` | `String` | 否 | 持股变动方向-展示 |
| `productName` | `String` | 否 | 产品名称 |
| `showProductName` | `String` | 否 | 显示的产品名字 |
| `agencyName` | `String` | 否 | 主体名称（机构） |
| `shareholderName` | `String` | 否 | 股东名称 |
| `reportDate` | `String` | 否 | 报告期 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="22",
    requestType="GET",
    agencyType=None,  # Integer  # 默认: 
    reportDate=None,  # String  # 默认: 
    stockCode=None,  # String  # 默认: 
    direc=None,  # Integer  # 默认: 
)
```
