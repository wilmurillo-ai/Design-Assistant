# 机构历史个股追踪

> 分类: 机构动向 | 目录: `小工具-机构动向` | 索引见 `SKILL.md`


### 描述

机构历史个股追踪

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "26" | 接口ID |
| `agencyOnlyId` | `String` | 否 | - | 机构唯一标识  |
| `yearNum` | `Integer` | 否 | - | 年份-前yearNum年(全部不用传) |
| `holderName` | `String` | 否 | - | 分支机构-股东名称(全部不用传)() |
| `branchInfoId` | `String` | 否 | - | 分支机构-infoId |

### 返回参数

机构历史个股追踪 返回数据

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `num` | `Integer` | 否 | 跟踪总次数 |
| `changeDirect` | `Integer` | 否 | 持股变动方向 1-新进 2-退出 3-增加 4-减少 5-重仓 |
| `riseNumTwenty` | `Integer` | 否 | 20日上涨次数 |
| `fallNumTwenty` | `Integer` | 否 | 20日下跌次数 |
| `riseNumForty` | `Integer` | 否 | 40日上涨次数 |
| `fallNumForty` | `Integer` | 否 | 40日下跌次数 |
| `riseNumSixty` | `Integer` | 否 | 60日上涨次数 |
| `fallNumSixty` | `Integer` | 否 | 60日下跌次数 |
| `trackDayNum` | `Integer` | 否 | 概率最高的天数 |
| `publicDate` | `String` | 否 | 最近统计日期 |
| `isAdvantage` | `Boolean` | 否 | 是否亮点 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="26",
    requestType="GET",
    agencyOnlyId=None,  # String  # 默认: 
    yearNum=None,  # Integer  # 默认: 
    holderName=None,  # String  # 默认: 
    branchInfoId=None,  # String  # 默认: 
)
```
