# 机构风云榜-季度筛选列表 2.5.0

> 分类: 机构动向 | 目录: `小工具-机构动向` | 索引见 `SKILL.md`


### 描述

机构风云榜-季度筛选列表 2.5.0

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `String` | 是 | "23" | 接口ID |
| `agencyType` | `Integer` | 是 | - |  机构类型 3001-公募 1003-私募 1006-信托公司 1007-证券公司 <br>1009-保险公司 1015-外资 8000-国家队 1201-知名机构  全部不传值  |

### 返回参数

机构风云榜-季度筛选列表 2.5.0 返回数据

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `reportDate` | `String` | 是 | 报告期 |
| `year` | `String` | 否 | 年份 |
| `showQuarter` | `String` | 否 | 显示字段（季度） |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="23",
    requestType="GET",
    agencyType=None,  # Integer  # 默认: 
)
```
