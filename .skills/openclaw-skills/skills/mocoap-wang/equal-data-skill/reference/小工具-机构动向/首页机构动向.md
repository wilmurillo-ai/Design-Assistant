# 首页机构动向

> 分类: 机构动向 | 目录: `小工具-机构动向` | 索引见 `SKILL.md`


### 描述

首页机构动向

### 请求参数

*无参数*

### 返回参数

首页机构动向 返回数据

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `showAgencyName` | `String` | 否 | 显示的机构名称 |
| `agencyName` | `String` | 否 | 机构名称 |
| `fundCompany` | `String` | 否 | 基金公司 |
| `companyShortName` | `String` | 否 | 公司简称 |
| `stockCode` | `String` | 是 | 股票代码 |
| `stockName` | `String` | 否 | 股票名称 |
| `publicDate` | `String` | 否 | 更新日期 |
| `holderNum` | `Long` | 否 | 持股数量 |
| `holderCode` | `String` | 否 | 股东编码 |
| `holderName` | `String` | 否 | 股东名称 |
| `jumpUrl` | `String` | 否 | 跳转链接 |
| `changeNum` | `Long` | 否 | 变动股数 |
| `changeDirect` | `Integer` | 否 | 持股变动方向 1-新进 2-退出 3-增持 4-减持 |
| `ratio` | `BigDecimal` | 否 | 持股比例 |
| `agencyType` | `Integer` | 否 | 机构类型 |
| `agencyOnlyId` | `String` | 否 | 机构唯一id |
| `type` | `Integer` | 否 | 比例类型 0-总股本 1-流通股 |
| `typesArr` | `JSONArray` | 否 | 类型数组 |

### 调用示例

```python
from kjj import KjjApi 
api = KjjApi('bear your token') 

# 调用接口
data = api.query_kjj_data(
    interfaceId="24",
    requestType="GET",
)
```
