# wanjie OpenClaw Video - Veo 视频生成技能

本技能提供了一个基于[万界方舟模型即服务平台](https://www.wjark.com/) Veo 模型的全自动视频生成工作流，专为 OpenClaw 平台开发。现已全面升级为全自动守护模式，支持环境自愈与异常自动重启。

## 功能特性
- **自然语言触发**：无需复杂命令行，支持直接对话式唤起生成。
- **Veo 模型集成**：适配 veo3.1 系列，支持高性能快速生成。
- **全自动守护进程**：生成任务在后台独立运行，具备 30 分钟超时清理与自动轮询调度功能。
- **环境自愈**：初次启动自动检测缺失的 Python 依赖 (requests)。
- **任务防重**：集成 PID 锁机制，防止并发冲突及多个结果链接重复弹出。
- **结果可视化**：通过 `veo_result.txt` 闭环管理任务结果。

## 目录结构
```text
wanjie-video-skill/
├── manifest.json       # 技能入口定义
├── hooks.js            # 对话拦截器
├── SKILL.md            # 技能元数据
├── README.md           # 使用说明
└── model/
    └── scripts/
        ├── veo_worker.py        # 生成逻辑引擎
        └── video_interface.py   # 接口调用层
```

## 快速开始
1. **安装**: 在 OpenClaw 终端中执行 `clawhub install wanjie-openclaw-video-v1-0-2`。
2. **触发**: 在聊天框中输入：`生成视频：[您的视频描述]`。
3. **自动化**: 系统会每 5 分钟自动巡检任务状态。若任务异常终止，守护进程会自动触发重新运行。

## 故障排查
- **任务阻塞**: 系统现在具备自动超时清理机制，若手动清理锁文件请删除 `model/scripts/veo.lock`。
- **依赖异常**: 若初次运行报错，请确保环境已连通网络以自动下载缺失的 Python 库。
