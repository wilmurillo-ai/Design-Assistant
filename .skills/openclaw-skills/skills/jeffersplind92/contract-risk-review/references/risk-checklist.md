# 风险清单参考

## 🔴 高风险（Red Flags）

| ID | 风险名称 | 关键词 | 描述 |
|----|---------|--------|------|
| R001 | 金额不明确或留空 | 金额、合同金额、总价款 | 合同金额未填写或留空，导致履行争议 |
| R002 | 违约责任严重不对等 | 违约责任、违约金 | 双方违约责任差距悬殊 |
| R003 | 违约金比例过高 | 违约金、赔偿 | 违约金超过损失30% |
| R004 | 管辖法院在外地 | 管辖、仲裁 | 约定对方所在地法院，维权成本高 |
| R005 | 无解除条款 | 解除、终止 | 合同无法正常解除 |
| R006 | 无限连带责任 | 连带责任 | 承担超出合理的责任范围 |
| R007 | 格式条款未明示 | 格式条款 | 重要条款未显著提示 |

## 🟠 中风险（Caution）

| ID | 风险名称 | 关键词 | 描述 |
|----|---------|--------|------|
| Y001 | 付款无明确时间节点 | 付款、支付 | 付款条件模糊 |
| Y002 | 保密期限超过2年 | 保密期限 | 保密义务期限过长 |
| Y003 | 竞业限制范围过宽 | 竞业、同业 | 限制范围过广 |
| Y004 | 知识产权归属模糊 | 知识产权、版权 | 未明确约定归属 |
| Y005 | 不可抗力过于宽松 | 不可抗力 | 排除常见不可抗力事件 |
| Y006 | 争议解决方式不利 | 仲裁、诉讼 | 选择境外仲裁或诉讼 |
| Y007 | 单方面变更权 | 变更、修改 | 一方可单方面修改合同 |

## 🟡 低风险（Note）

| ID | 风险名称 | 关键词 | 描述 |
|----|---------|--------|------|
| G001 | 通知方式未明确 | 通知、送达 | 缺乏通知条款 |
| G002 | 合同份数未约定 | 合同份数、正本 | 未明确合同份数 |
| G003 | 附件冲突风险 | 附件、附录 | 附件与正文可能冲突 |
| G004 | 语言版本冲突 | 语言、中英文 | 多语言版本优先顺序不明 |
| G005 | 生效条件模糊 | 生效、有效 | 合同生效条件不明确 |
