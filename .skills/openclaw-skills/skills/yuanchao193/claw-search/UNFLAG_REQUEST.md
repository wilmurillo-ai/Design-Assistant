## Skill 说明

Claw Search 是一个为 AI Agents 提供的免费全网搜索 API 服务。

### 主要功能

1. **Web 搜索 API** - 提供类似 Brave Search 的 RESTful API
2. **多语言支持** - 自动识别中、英、日、韩等多语言查询
3. **意图识别** - 自动识别教程、新闻、官网等搜索意图
4. **内容提取** - 提供 URL 内容提取功能
5. **缓存系统** - 智能缓存提升响应速度
6. **统计分析** - 提供调用统计功能

### 技术实现

- **后端**: Node.js + Express
- **搜索引擎**: Bing Web Search（通过浏览器自动化抓取）
- **部署方式**: Docker / 手动部署
- **端口**: 8093

### 用途

- 供 AI Agents（如 OpenClaw）调用搜索功能
- 为开发者提供免费的无需 API Key 的搜索服务
- 支持各种 Agent 框架集成

## 安全说明

本服务：

1. ✅ **不收集用户敏感信息** - 仅记录调用统计（调用次数、国家）
2. ✅ **不存储搜索数据** - 搜索请求直接转发到搜索引擎
3. ✅ **不包含恶意代码** - 纯开源代码，可自行审计
4. ✅ **无第三方依赖** - 不使用任何付费 API Key
5. ✅ **合规使用** - 仅用于合法搜索用途
