# Education Planning Reference

## Bloom's Taxonomy (Revised)

| Level | Cognitive Process | Key Verbs |
|-------|------------------|-----------|
| 1. Remember | Recall facts | define, list, recall, identify, name |
| 2. Understand | Explain concepts | explain, summarize, interpret, classify |
| 3. Apply | Use in new situations | use, solve, demonstrate, calculate |
| 4. Analyze | Draw connections | analyze, examine, compare, differentiate |
| 5. Evaluate | Justify decisions | evaluate, judge, critique, defend |
| 6. Create | Produce original work | create, design, develop, compose |

## Lesson Plan Structure (5E Model)

1. **Engage** — Capture attention, activate prior knowledge
2. **Explore** — Hands-on investigation, student-driven discovery
3. **Explain** — Teacher-led instruction, concept introduction
4. **Elaborate** — Apply understanding, extend learning
5. **Evaluate** — Assess understanding, reflect on learning

## Curriculum Planning Principles

- **Backward Design**: Start with learning outcomes, then assessments, then activities
- **Scaffolding**: Build complexity gradually
- **Spiral Curriculum**: Revisit concepts at increasing depth
- **Differentiation**: Adapt for diverse learner needs
- **Formative Assessment**: Ongoing checks during instruction

## Common Standards

- **CCSS** — Common Core State Standards (US)
- **NGSS** — Next Generation Science Standards (US)
- **Alberta Programs of Study** — Alberta Education curriculum
- **IB** — International Baccalaureate
- **AP** — Advanced Placement

## Assessment Types

| Type | Purpose | Frequency |
|------|---------|-----------|
| Diagnostic | Identify prior knowledge | Before unit |
| Formative | Monitor progress | During instruction |
| Benchmark | Measure progress toward goals | Mid-unit/term |
| Summative | Evaluate achievement | End of unit/course |

## Study Techniques (Evidence-Based)

- **Spaced Repetition** — Review at increasing intervals
- **Active Recall** — Test yourself rather than re-read
- **Interleaving** — Mix different topics in one session
- **Elaboration** — Explain concepts in your own words
- **Dual Coding** — Combine words with visuals
