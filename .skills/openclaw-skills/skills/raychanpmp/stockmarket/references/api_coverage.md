## Key Metrics

| Metric | What It Measures |
|--------|-----------------|
| P/E Ratio | Price / Earnings — valuation |
| PEG Ratio | P/E / Growth Rate — value vs growth |
| Beta | Volatility vs market (1.0 = market avg) |
| EPS | Earnings Per Share |
| Market Cap | Share price × total shares |
| Dividend Yield | Annual dividend / share price |
| Moving Average | Smoothed price trend |
| RSI | Relative Strength Index (0-100) |
| MACD | Moving Average Convergence Divergence |
| Bollinger Bands | Price volatility envelope |
