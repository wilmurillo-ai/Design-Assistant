# 贡献指南

感谢你对 feishu-send-file 的兴趣！

## 如何贡献

1. Fork 本仓库
2. 创建你的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交你的改动 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 打开一个 Pull Request

## 报告问题

如果你发现问题或有功能建议，请通过 GitHub Issues 报告。

## 代码规范

- Shell 脚本使用 `#!/bin/bash`  shebang
- Python 代码遵循 PEP 8 规范
- 提交信息使用清晰的描述

## 安全提醒

永远不要提交包含真实凭证的 `config.json` 文件！
