# SEO Audit Scoring Rubric

## Score Ranges
- **90-100 (A):** Excellent — minor optimizations only
- **80-89 (B):** Good — a few improvements needed
- **70-79 (C):** Average — noticeable gaps to address
- **60-69 (D):** Below average — significant issues
- **0-59 (F):** Failing — critical problems

## On-Page Checks

### Title Tag (0-100)
- 100: Present, 50-60 chars, contains target keyword
- 70: Present but too long (>65) or too short (<30)
- 0: Missing

### Meta Description (0-100)
- 100: Present, 150-160 chars, compelling with CTA
- 60: Present but wrong length
- 0: Missing

### H1 Tag (0-100)
- 100: Exactly one, contains primary keyword
- 60: Multiple H1s
- 0: Missing

### Header Hierarchy (0-100)
- 100: Proper H1→H2→H3 nesting, no gaps
- 60: Minor gaps (H2→H4)
- 30: Major structural issues

### Images (0-100)
- Score = 100 × (images with alt / total images)
- 100: All images have descriptive alt text
- 0: No images have alt text

### Word Count (0-100)
- 100: 800+ words (comprehensive)
- 70: 300-799 words (adequate)
- 30: <300 words (thin content)

### Schema Markup (0-100)
- 100: Valid JSON-LD present
- 0: No structured data

## Technical Checks

### Page Speed (0-100)
- 100: <1s response time
- 60: 1-3s
- 20: >3s

### Core Web Vitals (0-100)
- Directly from PageSpeed Insights score (0-100)

### Robots.txt (0-100)
- 100: Present and properly configured
- 50: Missing
- 0: Blocking important content

### Sitemap (0-100)
- 100: Valid XML sitemap found
- 0: No sitemap

## GEO Checks

### Q&A Structure (0-100)
- 100: 3+ Q&A patterns or FAQ section
- 60: 1-2 Q&A patterns
- 20: No Q&A structure

### Data Points (0-100)
- 100: 5+ statistics/data points
- 70: 2-4 data points
- 30: 0-1 data points

### Schema Richness (0-100)
- 100: 3+ GEO-relevant schema types
- 60: 1-2 schema types
- 0: No schema

### E-E-A-T Signals (0-100)
- 25 points each: author info, about page, dates, external sources
