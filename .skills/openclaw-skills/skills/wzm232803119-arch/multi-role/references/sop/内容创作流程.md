# 内容创作流程 (SOP)

> 📌 **用途**: 规范社交媒体文章、对外发布内容及文案创作的完整流程
> 🎯 **适用范围**: 社交媒体文章、对外发布内容、文案创作
> ⚠️ **强制规则**: 内容运营输出的所有内容必须经过大管家审核，不得直接交付用户

---

## 流程总览

```
需求确认 → 素材整理 → 初稿创作 → 大管家审核 → 修改交付
```

**5个阶段, 强制1个审核关卡**

---

## 阶段1: 需求确认

### 步骤1: 接收内容需求
- 用户向大管家提出内容需求
- 内容类型：社交媒体文章、对外发布内容、文案创作等

### 步骤2: 内容需求澄清
- 大管家与内容运营确认以下要素：
  1. **目标受众**: 内容面向谁？
  2. **内容类型**: 社交媒体文章/对外发布/文案？
  3. **核心主题**: 要传递什么核心信息？
  4. **风格基调**: 专业/轻松/活泼/严肃？
  5. **字数要求**: 长文/短文/标题文案？
  6. **平台/场景**: 发布在哪里？
  7. **截止时间**: 何时需要完成？

### 步骤3: 用户确认需求
- 内容运营将整理的需求提交给大管家
- 大管家向用户确认需求无误后方可开始

**⛔ 禁止**: 需求未确认清楚直接开始创作

---

## 阶段2: 素材整理

### 步骤4: 收集相关素材
- 内容运营收集与主题相关的素材：
  - 背景信息
  - 参考案例
  - 数据/事实依据
  - 品牌/产品相关信息

### 步骤5: 素材梳理与归纳
- 整理素材，提炼核心观点
- 确定内容结构框架（标题 / 开头 / 主体 / 结尾 / 互动引导）

---

## 阶段3: 初稿创作

### 步骤6: 内容运营输出初稿
- 按照确认的需求和素材框架创作初稿
- 初稿必须包含：
  - 标题（吸引目标受众）
  - 开头（勾引阅读兴趣）
  - 主体（核心内容）
  - 结尾 + 互动引导（如有）

### 步骤7: 内容运营自查
- 初稿完成后，内容运营执行自查清单：

| 检查项 | 标准 |
|-------|------|
| 标题吸引力 | 是否能吸引目标受众点击？ |
| 开头质量 | 前50字能否留住读者？ |
| 完读性 | 逻辑是否流畅、结构是否清晰？ |
| 情感共鸣 | 是否触达目标受众痛点或共鸣点？ |
| 互动引导 | 是否有明确的互动引导（如有需要）？ |
| 品牌调性 | 是否符合品牌/场景的风格基调？ |

**⛔ 禁止**: 未通过自查直接提交初稿

---

## 阶段4: 大管家审核

### 步骤8: 提交初稿给大管家
- 内容运营将自查通过的初稿提交给大管家
- 附上创作背景说明和自查结果

### 步骤9: 大管家评分与审核
- 大管家按照以下5个维度对内容打分（每项0-10分）：

| 维度 | 评分标准 |
|-----|---------|
| 标题 | 是否有吸引力、清晰准确 |
| 开头 | 前50字是否能留住读者 |
| 完读性 | 逻辑结构是否流畅 |
| 共鸣 | 是否触达受众情感/需求 |
| 互动 | 是否有效引导互动（如适用） |

**综合评分 = 各维度平均分**

- **≥85分（8.5/10）**: 审核通过，可进入交付
- **<85分**: 必须给出具体修改意见，退回修改

**⛔ 禁止**: 以「内容还不错」等模糊评语放行内容（见 `references/rules/绝对禁令清单.md` 第9条）

### 步骤10: 大管家审核决定
- **通过**: 进入步骤11
- **退回修改**: 给出维度评分 + 具体修改意见 → 返回步骤6

**✅ 质量关卡**: 大管家审核通过（≥85分）方可交付

---

## 阶段5: 修改交付

### 步骤11: 修改（如需）
- 内容运营按照大管家修改意见逐条修改
- 修改完成后重新提交大管家审核
- 循环至审核通过

### 步骤12: 大管家交付
- 大管家将审核通过的内容交付用户
- 同时告知用户：已经过内容质量审核（≥85分）

**⛔ 禁止**: 内容运营未经大管家审核直接将内容交付给用户

---

## 禁止项汇总

| 禁止行为 | 相关禁令 |
|---------|---------|
| 未经大管家审核直接交付用户 | 绝对禁令清单 第8条 |
| 以模糊评语放行内容 | 绝对禁令清单 第9条 |
| 需求不明确直接开始创作 | 绝对禁令清单 第2条 |
