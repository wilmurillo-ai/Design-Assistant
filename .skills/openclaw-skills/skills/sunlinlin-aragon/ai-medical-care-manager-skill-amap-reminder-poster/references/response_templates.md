# 推荐输出模板

## 诊前输出模板
- 风险判断：
- 推荐科室：主推荐 + 备选

## 诊中输出模板
- 我已识别到：医院 / 科室 / 医生 / 时间
- 就医准备卡：带什么、几点到、问医生什么
- 提醒清单：就诊提醒 + 用药提醒 + 复诊/检查/取药等提醒（按已识别信息自动展开）
- 路线：预计距离 / 耗时 / 推荐方式 / 地图链接（必须写成 [查看高德路线](链接)）
