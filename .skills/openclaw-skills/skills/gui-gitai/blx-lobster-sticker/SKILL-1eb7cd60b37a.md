## 使用方式

使用 `image` 工具读取 `skills/blx-lobster-sticker/assets/` 目录下对应的图片文件发送给用户。

示例：
```
用户说："加油你最棒！"
→ 情绪：加油/鼓励
→ 发送 assets/036_cheer.png
```

## 注意事项

- 每次只发送一张最匹配的情绪图片
- 如果无法判断情绪，可以不发送
- 图片路径：`skills/blx-lobster-sticker/assets/[文件名].png`
- 风格：可爱卡通小龙虾，LINE贴纸风格，厚白色边框，统一色调
- 角色特征：红色外壳、大眼睛、粉色腮红、白色厚边框
