# 学分查询技能

## 技能概述

- **技能名称**: 学分查询
- **技能版本**: 0.0.2
- **技能描述**: 用于查询学生各科目成绩的OpenClaw技能
- **作者**: bc96

## 功能列表

| 功能 | 说明 |
|-----|------|
| 查询单科成绩 | 根据姓名和科目查询特定成绩 |
| 查询全部成绩 | 查询某学生的所有科目成绩 |
| 科目名称转换 | 支持口语化转标准科目名 |
