# ROI 计算实际案例

## 案例 1: Tech With Tim YouTube 合作 (P0-3)

**背景**: 与 YouTube 技术博主 Tech With Tim 合作推广 ClawHub 技能

### 输入数据
```
预期收益:
- 首单佣金：$87,906 × 30% = $26,372
- 续费佣金：$87,906 × 15% × 3 年 = $39,558
- 总收益：$65,930 (约¥475,000)

投入成本:
- 佣金支出：$26,372 (首单) + $13,186/年 (续费)
- 时间成本：5 小时 × ¥50 = ¥250 ($35)
- 总成本：$26,407 (约¥190,000)

成功率：60% (基于频道影响力和历史合作数据)
时间周期：长期 (3 年)
```

### 计算结果
```
短期 ROI (首年):
= (首单佣金 + 首年续费 - 总成本) / 总成本
= ($26,372 + $13,186 - $26,407) / $26,407
= $13,151 / $26,407
= 0.50x (50%)

长期 ROI (3 年):
= (总收益 - 总成本) / 总成本
= ($65,930 - $26,407) / $26,407
= $39,523 / $26,407
= 1.50x (150%)

期望值:
= 总收益 × 成功率
= $65,930 × 60%
= $39,558 (约¥285,000)
```

### 决策建议
🟡 **谨慎评估 (长期 ROI=1.50 刚好达到阈值)**

### 关键洞察
- 首年 ROI 仅 0.50x，未达 1.5 阈值
- 需要 3 年才能实现 1.5x ROI
- 建议谈判提高首单佣金比例至 40-50%
- 或要求保底费用降低风险

### 优化方案
```
方案 A: 提高首单佣金
- 首单佣金：40% = $35,162
- 短期 ROI: ($35,162 + $13,186 - $35,197) / $35,197 = 0.37x
- 仍不推荐

方案 B: 保底 + 佣金
- 保底费用：$20,000
- 佣金比例：20% 首单 + 10% 续费
- 总成本：$20,035
- 短期 ROI: ($17,581 + $8,791 - $20,035) / $20,035 = 0.32x
- 风险降低，但 ROI 仍低

方案 C: 放弃，转向更高 ROI 渠道
- ClawHub 技能上架：ROI=21.1x
- 优先执行高 ROI 项目
```

**最终决策**: ❌ 建议放弃，优先执行 ClawHub 技能上架 (ROI=21.1x)
