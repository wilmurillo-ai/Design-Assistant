# ROI 计算预设模板 (10+ 行业)

## 模板使用方法

1. 选择对应行业的模板
2. 填入实际数据
3. 自动计算 ROI 和期望值
4. 根据阈值做出决策

## 模板 1: 渠道合作 (KOL/ influencer)

**适用场景**: YouTube 合作/播客赞助/Newsletter 推广

```
预期收益:
- 短期：首单佣金 $X (通常 30% 首单)
- 长期：续费佣金 $Y/月 (通常 15% 续费) × 12 月
- LTV: $Y × 平均留存月数

投入成本:
- 佣金支出：首单 30% + 续费 15%
- 内容制作：$0 (合作方自行制作)
- 时间成本：N 小时 × ¥50/小时
- 总成本：佣金 + 时间成本

成功率:
- 头部 KOL (10 万 + 粉丝): 30-50%
- 中部 KOL (1-10 万粉丝): 50-70%
- 微型 KOL (<1 万粉丝): 70-90%

阈值参考:
- ROI ≥ 1.5: 推荐执行
- ROI < 1.5: 重新谈判佣金比例
```

**案例**: Tech With Tim YouTube 合作
- 预期收入：$87,906 (首年)
- 佣金成本：30% 首单 + 15% 续费 = $26,372
- 时间成本：5 小时 × ¥50 = ¥250 ($35)
- 总成本：$26,407
- ROI: ($87,906 - $26,407) / $26,407 = 2.33x ✅

## 模板 2: 社群推广 (Discord/Slack)

**适用场景**: 官方社群/付费社群/地区社群

```
预期收益:
- 转化用户数：社群规模 × 曝光率 × 转化率
- 单用户价值：¥X (首单 + 续费)
- 总收益：转化用户数 × 单用户价值

投入成本:
- 社群费用：$0 (免费) / $29/月 (付费)
- 内容制作：N 小时 × ¥50/小时
- 运营时间：N 小时/周 × ¥50/小时 × 周数
- 总成本：费用 + 时间成本

成功率:
- 官方社群 (Discord): 60-80% (高信任度)
- 付费社群 (Skool): 70-90% (高质量用户)
- 地区社群 (Facebook): 40-60% (地域限制)

阈值参考:
- ROI ≥ 1.5: 推荐执行
- ROI < 1.5: 优化内容或选择其他社群
```

**案例**: OpenClaw Lab Skool (Florian Darroman)
- 社群规模：265+ 创始人
- 预期转化：265 × 50% 曝光 × 5% 转化 = 6-7 用户
- 单用户价值：¥4,000 (ClawHub 技能)
- 总收益：¥24,000-28,000
- 投入成本：$29/月 + 2 小时×¥50 = ¥310
- ROI: (24000 - 310) / 310 = 76.4x ✅

## 模板 10: 股权投资/早期投资

**适用场景**: 初创公司投资/股权置换/战略合作

```
预期收益:
- 股权价值：投资额 × 预期回报倍数
- 分红收益：年利润 × 股权比例
- 战略价值：合作带来的间接收益
- 总收益：股权 + 分红 + 战略

投入成本:
- 投资金额：¥X
- 尽调时间：N 小时 × ¥50/小时
- 机会成本：资金其他用途收益
- 总成本：投资 + 时间 + 机会成本

成功率:
- 早期项目：10-30% (高风险)
- 成长期项目：40-60%
- 成熟期项目：70-90%

阈值参考:
- ROI ≥ 5.0: 推荐执行 (考虑风险)
- ROI 2.0-5.0: 谨慎评估
- ROI < 2.0: 建议放弃 (风险不匹配)
```

**案例**: OpenClaw 生态项目投资
- 投资金额：¥50,000
- 预期回报：5-10x (3-5 年)
- 预期收益：¥250,000-500,000
- 尽调时间：10 小时 × ¥50 = ¥500
- ROI (5x): (250000 - 50500) / 50500 = 3.95x ✅
- 风险调整：30% 成功率 → 期望值 = 3.95 × 0.3 = 1.19x 🟡
