# Business ROI Calculator - 使用说明

## 产品概述

**Business ROI Calculator** 是为商务人士打造的实时投资回报率计算工具。输入预期收益和投入成本，自动计算短期/长期 ROI，判断合作是否值得执行 (阈值>1.5)。

准备以下信息：
- **预期收益**: 短期 (首单/首月) + 长期 (续费/LTV)
- **投入成本**: 人力 + 时间 + 资金
- **成功率**: 预估成交概率 (如无数据用 50%)

## 核心功能

### 1. ROI 实时计算

**公式**:
```
短期 ROI = (短期收益 - 投入成本) / 投入成本
长期 ROI = (长期收益 × 成功率 - 投入成本) / 投入成本
期望值 = 长期收益 × 成功率
```

**阈值标准**:
- ROI ≥ 1.5: ✅ 推荐执行
- ROI 0.5-1.5: 🟡 谨慎评估
- ROI < 0.5: ❌ 建议放弃

### 2. 10+ 行业预设模板

内置模板覆盖：
1. 渠道合作 (KOL/ influencer)
2. 社群推广 (Discord/Slack)
3. 技能培训 (ClawHub 上架)
4. 商务咨询服务
5. 内容创作 (教程/博客)
6. 邮件外展 (Cold Email)
7. 联盟营销 (Affiliate)
8. 模板销售 (数字产品)
9. API 合作 (Tavily/MCP)
10. 股权投资 (早期投资)

### 3. 决策建议

基于 ROI 阈值自动给出：
- ✅ 强烈推荐 (ROI ≥ 1.5)
- 🟡 谨慎评估 (ROI 0.5-1.5)
- ❌ 建议放弃 (ROI < 0.5)

### 4. 关键洞察

提供深度分析：
- 风险承受能力 (成功率下降 X% 后的 ROI)
- 盈亏平衡点 (ROI=0 时的成功率)
- 优化建议 (如何提高 ROI)

## 最佳实践

### 1. 数据准确性

- 使用历史数据而非猜测
- 参考行业基准 (如邮件回复率 5-15%)
- 保守估计收益，充分估计成本

### 2. 隐性成本

注意包含：
- 机会成本 (时间用于其他项目的收益)
- 维护成本 (售后支持/更新迭代)
- 风险成本 (失败的可能性)

### 3. 动态调整

- 每周回顾实际 ROI
- 调整预测模型参数
- 优化低 ROI 活动

### 4. 组合策略

- 高 ROI + 低投入：立即执行 (如 ClawHub 技能)
- 高 ROI + 高投入：谨慎评估 (如大型合作)
- 低 ROI + 低投入：可接受 (如品牌建设)
- 低 ROI + 高投入：避免 (如无效广告)

## 常见问题

### Q: ROI 阈值为什么是 1.5？

A: 1.5x 意味着收益是成本的 1.5 倍，即 50% 利润率。这是商务合作的基本门槛，低于此值不值得投入时间。

### Q: 如何估算成功率？

A: 
- 参考历史数据 (如过往合作转化率)
- 使用行业基准 (如邮件回复率 5-15%)
- 保守估计 (宁可低估不可高估)
- 无数据时用 50% 作为默认值

### Q: 长期 ROI 如何计算？

A: 长期 ROI = (长期收益 × 成功率 - 投入成本) / 投入成本。关键是要考虑续费/LTV 和成功率。

### Q: 如何处理多阶段项目？

A: 分阶段计算 ROI：
- 阶段 1: 投入¥X, 预期收益¥Y, 计算阶段 ROI
- 阶段 2: 追加投入¥A, 预期收益¥B, 计算累计 ROI
- 决策：每阶段独立评估，阶段 1 达标才进入阶段 2
