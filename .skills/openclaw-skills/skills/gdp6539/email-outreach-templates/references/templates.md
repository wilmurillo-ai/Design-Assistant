# 50+ 邮件外展模板

## 首次联系 (F1-F10)

**F1: 价值导向 (科技)**
```
主题：{公司} x {渠道} 合作机会

{姓名}您好，

我是{公司}的{职位}，关注{渠道}已久。

我们专注于{核心价值}，已帮助{案例}实现{成果}。

是否有兴趣 15 分钟电话聊聊？

{签名}
```

**F2: 案例导向 (电商)**
```
主题：{类似公司} 实现{成果}的案例

{姓名}您好，

{类似公司}通过我们的{产品}实现了{成果}。

考虑到{渠道}的受众特点，相信类似方案也能帮助您。

附件是详细案例。

{签名}
```

## 跟进模板 (F11-F40)

**F11: 价值提醒 (1-3 天)**
```
主题：Re: {原主题}

补充一点：{新价值点}。

{签名}
```

**F21: 问题引导 (7-14 天)**
```
主题：关于{话题}的想法

{姓名}好，

最近在思考{行业问题}，想听听您的看法。

{签名}
```

**F31: 最后通牒 (30 天+)**
```
主题：最后一次联系

如果目前不合适，我理解。

如果还有兴趣，请告诉我时间。

{签名}
```

## 谈判/成交 (F41-F50)

**F41: 条款确认**
```
主题：合作条款确认

根据讨论，总结如下：
1. {条款 1}
2. {条款 2}

请确认。

{签名}
```

**F46: 成交确认**
```
主题：合作确认

确认合作！

下一步：
1. {步骤 1} - {日期}

{签名}
```
