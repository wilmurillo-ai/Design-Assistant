# 示例视频目录

这里放置示例视频文件，用于演示和测试 AutoClip Pro 的功能。

## 支持的格式

- `.mp4` (推荐)
- `.mov`
- `.avi`
- `.mkv`
- `.webm`

## 使用方法

1. 将你的视频文件放入 `input` 文件夹
2. 运行 `run.bat`
3. 处理完成后，查看 `output` 文件夹

## 测试视频建议

为了快速测试，建议使用：
- 时长: 10-30 秒
- 分辨率: 720p 或 1080p
- 大小: 10MB 以内
