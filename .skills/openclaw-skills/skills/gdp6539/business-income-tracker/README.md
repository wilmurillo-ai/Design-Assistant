## 快速开始

1. 每日 09:00 记录收入/支出
2. 自动计算周/月汇总
3. 追踪渠道状态
4. 查看 ROI 统计

## 记录模板

**收入**:
```

金额：¥350
备注：ROI 计算器 x1
```

**支出**:

## ROI 计算

```
ROI = (总收入 - 总支出) / 总支出
```
