# 渠道跟进模板库 (50+)

## 模板分类

### 首次联系 (10 个)

**模板 F1: 价值导向 (科技行业)**
```
主题：{公司名} x {渠道名} 合作机会

{姓名}您好，

我是{公司}的{职位}，关注{渠道}已久。

我们专注于{核心价值}，已帮助{案例}实现{成果}。

相信双方合作能为您带来{具体价值}。

是否有兴趣 15 分钟电话聊聊？

祝好，
{签名}
```

**模板 F2: 案例导向 (电商行业)**
```
主题：{类似公司} 通过{产品}实现{成果}的案例

{姓名}您好，

{类似公司}最近通过我们的{产品}实现了{具体成果}。

考虑到{渠道名}的受众特点，相信类似方案也能帮助您的用户。

附件是详细案例，欢迎查看。

有时间聊聊如何复用到{渠道名}吗？

{签名}
```

### 跟进 1-3 天 (10 个)

**模板 F3: 价值提醒**
```
主题：Re: {原主题}

{姓名}好，

补充一点：{新价值点}。

这对{渠道名}的意义是{具体说明}。

期待您的反馈。

{签名}
```

**模板 F4: 案例分享**
```
主题：{相关行业}最新案例

{姓名}好，

刚完成一个{相关行业}案例，{成果}。

想到{渠道名}可能也感兴趣，分享给您。

{案例链接}

{签名}
```

### 跟进 7-14 天 (10 个)

**模板 F5: 问题引导**
```
主题：关于{具体话题}的想法

{姓名}好，

最近在思考{行业问题}，想听听您的看法。

{渠道名}在这个问题上有什么策略吗？

{签名}
```

### 跟进 30 天 + (10 个)

**模板 F6: 最后通牒**
```
主题：最后一次联系 - {合作机会}

{姓名}好，

这已经是我第{N}次联系了。

如果目前不合适，我理解并会停止跟进。

如果还有兴趣，请告诉我一个合适的时间。

无论结果如何，感谢关注。

{签名}
```

### 谈判阶段 (5 个)

**模板 F7: 条款确认**
```
主题：合作条款确认

{姓名}好，

根据讨论，总结合作条款如下：

1. {条款 1}
2. {条款 2}
3. {条款 3}

请确认是否有遗漏或需要调整。

{签名}
```

### 成交/失败 (5 个)

**模板 F8: 成交确认**
```
主题：合作确认 - 下一步

{姓名}好，

确认合作！

下一步：
1. {步骤 1} - {日期}
2. {步骤 2} - {日期}

期待合作成功。

{签名}
```

**模板 F9: 失败保持联系**
```
主题：保持联系

{姓名}好，

理解目前不合适。

希望未来有机会合作。

我会继续分享{有价值内容}，保持联系。

{签名}
```

---

## A/B 测试说明

**A 版 (价值导向)**: 强调合作带来的具体价值
**B 版 (案例导向)**: 用成功案例证明能力

建议：
- 科技行业：A 版优先
- 电商行业：B 版优先
- 教育行业：A/B 交替
- 金融行业：B 版优先
