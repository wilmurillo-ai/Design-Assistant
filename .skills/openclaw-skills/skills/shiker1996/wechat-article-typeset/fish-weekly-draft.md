# 技术摸鱼周报 Vol.88｜大厂裁员与 GitHub 神器的冰火两重天

**导语：** 本周的 GitHub  Trending 比大厂 HR 的邮件还刺激——有人开源了 SuperAgent 框架狂揽 7 万 star，也有人收到了"毕业大礼包"。来，看看这周的代码世界发生了什么。

## 🔧 本周 GitHub 热门

### 1. obra/superpowers - 71k star，本周 +9k
**是什么：** 一个"能用"的 Agent 技能框架和软件开发方法论  
**怎么玩：** 如果你正在构建多 Agent 系统，这个仓库提供了一套经过实战验证的技能框架。作者声称这是"真正能用"的方法论，不是纸上谈兵。  
**地址：** [github.com/obra/superpowers](https://github.com/obra/superpowers)

### 2. anthropics/claude-code - 74k star，本周 +3.6k
**是什么：** Anthropic 官方的终端 AI 编程助手  
**怎么玩：** 直接在终端里跟 Claude 对话，让它帮你写代码、解释复杂逻辑、处理 Git 工作流。比起打开网页，这种"假装在写代码"的方式更适合摸鱼。  
**地址：** [github.com/anthropics/claude-code](https://github.com/anthropics/claude-code)

### 3. ruvnet/RuView - 27k star，本周 +19k
**是什么：** 用 WiFi 信号做人体姿态估计和生命体征监测  
**怎么玩：** 不需要摄像头，仅凭 WiFi 信号就能检测房间里有没有人、姿势如何、甚至呼吸频率。老板站在你背后的时候，这个技术可能帮不上忙，但用来做无接触式摸鱼检测系统倒是不错。  
**地址：** [github.com/ruvnet/RuView](https://github.com/ruvnet/RuView)

### 4. moeru-ai/airi - 26k star，本周 +6k
**是什么：** 自托管的 AI 伴侣，支持语音聊天、玩 Minecraft 和 Factorio  
**怎么玩：** 想要一个 24 小时在线的赛博老婆/老公？这个项目让你能部署自己的 AI 伴侣，还能一起玩游戏。比真人靠谱，至少不会突然离职。  
**地址：** [github.com/moeru-ai/airi](https://github.com/moeru-ai/airi)

## 📰 本周大厂热点

### 阿里 Qwen 团队离职潮
据传阿里通义千问团队近期出现离职潮，多位核心成员选择离开。有网友调侃："Qwen 没走，写 Qwen 的人走了。"也有人说这是"大厂优化 2.0"——不是裁员，是"人才流动"。

**摸鱼点评：** 开源项目最怕的就是核心贡献者突然"毕业"。所以啊，还是自己的 Side Project 最靠谱，至少不会被 HR 约谈。

### 字节绩效季又来了
一年一度的字节绩效评定季，熟悉的配方，熟悉的味道。375 的拿奖金，325 的拿 N+1。有员工在脉脉上吐槽："绩效面谈比相亲还紧张，至少相亲对象不会决定你能不能继续还房贷。"

**摸鱼点评：** 绩效这东西，就像 GitHub 的 star——看着挺重要，其实也就是个数字。真正重要的是你写了什么代码、学到了什么技术、以及摸鱼时有没有被抓到。

### MacBook Neo 曝光
苹果被曝正在研发一款名为"MacBook Neo"的新品，定位介于 MacBook Air 和 MacBook Pro 之间。传闻价格更亲民，性能更均衡，适合"既想要 Pro 的性能，又只有 Air 的预算"的打工人。

**摸鱼点评：** 苹果这是要细分收割啊。不过说实话，对于大部分程序员来说，买电脑的钱够买好几年的摸鱼时间了——这才是真正的 ROI 计算。
