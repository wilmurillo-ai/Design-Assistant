# 七维分析框架深度股票分析 Agent

基于七维分析框架的深度股票分析系统，整合多数据源提供全方位的股票投资分析。

## 功能特性

✅ **七维分析框架**
- 数据收集与验证
- 基本面分析（盈利能力、资产负债、现金流）
- 估值分析（相对估值、估值风险）
- 行业与竞争分析（生命周期、竞争格局、护城河）
- 技术面分析（趋势、支撑压力、资金流向）
- 风险识别（财务风险、行业风险、估值风险）
- 综合结论（评分卡、投资建议矩阵）

✅ **多数据源支持**
- eFinance（实时行情、基础数据）
- AKShare（深度财务数据）
- QVeris（动态API调用、交叉验证）

✅ **灵活的报告格式**
- Markdown 详细报告
- JSON 结构化数据
- 简报摘要

## 快速开始

### 安装依赖

```bash
pip install efinance akshare pandas numpy
```

### 环境变量

```bash
export QVERIS_API_KEY=your_qveris_api_key
```

### 基础使用

```bash
# 基础分析（默认维度）
python scripts/analyze.py 600519

# 完整七维分析
python scripts/analyze.py 600519 --full

# 指定维度分析
python scripts/analyze.py 600519 --dimensions fundamental,valuation

# 指定数据源
python scripts/analyze.py 600519 --sources efinance,akshare

# 输出到文件
python scripts/analyze.py 600519 --output-file report.md
```

## 分析维度

### 1. 数据收集与验证
- 实时行情数据
- 财务报表数据
- 公司基本信息

### 2. 基本面分析
- 盈利能力分析（ROE、毛利率、净利率）
- 资产负债分析（资产负债率、流动比率、资产质量）
- 现金流分析（经营现金流、自由现金流）

### 3. 估值分析
- 相对估值法（PE、PB、PS）
- 估值风险评估
- 历史估值区间比较

### 4. 行业与竞争分析
- 行业生命周期判断
- 竞争格局分析
- 护城河识别

### 5. 技术面分析
- 趋势判断
- 支撑压力位
- 资金流向分析

### 6. 风险识别
- 财务风险清单
- 行业风险
- 估值风险

### 7. 结论输出
- 综合评分（0-100分）
- 投资评级（⭐⭐⭐⭐⭐ ~ ⭐）
- 投资建议矩阵
- 监控要点

## 评分系统

| 分数 | 评级 | 含义 | 建议 |
|------|------|------|------|
| 85-100 | ⭐⭐⭐⭐⭐ 强烈推荐 | 重仓买入，长期持有 |
| 70-84 | ⭐⭐⭐⭐ 推荐 | 适量买入，中期持有 |
| 55-69 | ⭐⭐⭐ 中性 | 观望，等待更好买点 |
| 40-54 | ⭐⭐ 谨慎 | 减仓，或短线博弈 |
| <40 | ⭐ 回避 | 远离，或做空 |

## 技术架构

```
7d-stock-analyzer/
├── scripts/
│   ├── analyze.py              # 主入口
│   ├── analyzers/              # 分析器模块
│   │   ├── base.py            # 基础分析器
│   │   ├── data_collector.py  # 数据收集
│   │   ├── fundamental.py     # 基本面分析
│   │   ├── valuation.py       # 估值分析
│   │   ├── industry.py        # 行业分析
│   │   ├── technical.py       # 技术面分析
│   │   ├── risk.py            # 风险识别
│   │   └── conclusion.py      # 结论生成
│   ├── adapters/              # 数据适配器
│   │   ├── efinance_adapter.py
│   │   ├── akshare_adapter.py
│   │   └── qveris_adapter.py
│   └── reporters/             # 报告生成器
│       ├── markdown_reporter.py
│       └── json_reporter.py
└── SKILL.md                   # 技能说明
```

## 使用场景

1. **个股深度分析**：全面了解一只股票的投资价值
2. **投资决策支持**：为买入/卖出/持有提供数据支撑
3. **投资组合管理**：定期分析持仓股票
4. **股票筛选**：快速排除不符合要求的股票

## 优势特点

- 🚀 **并行分析**：多个维度并行执行，提高效率
- 💰 **成本控制**：复用数据接口，降低 API 调用成本
- ✅ **交叉验证**：多数据源验证，提高准确性
- 🔧 **模块化设计**：每个维度独立，易于扩展
- 📊 **结构化输出**：便于程序化处理和集成
