# 程序员客栈技能

## 1. 平台简介

程序员客栈是一个专注于程序员外包接单的平台，提供远程工作、兼职外包、技术众包等服务。开发者可以在平台上展示技能、接取项目、赚取收入。

**网址**: https://www.proginn.com

---

## 2. 注册/登录方式

### 注册方式
- **手机号注册**：输入手机号 + 验证码
- **邮箱注册**：输入邮箱 + 密码
- **第三方登录**：支持微信、Github 登录

### 登录入口
- 右上角「登录/注册」按钮
- 支持微信扫码登录

## 3. 完善资料流程

### 第一步：基础资料
1. 登录后进入「个人中心」→「我的简历」
2. 填写：姓名、头像、学历、工作年限、城市
3. 填写：专业技能标签（最多10个）

### 第二步：工作经历
1. 进入「工作经历」模块
2. 点击「添加工作经历」
3. 填写：公司名称、职位、在职时间、工作描述
4. 建议至少添加2段经历

### 第三步：作品展示
1. 进入「我的作品」模块
2. 点击「添加作品」
3. 填写：
   - 作品名称
   - 作品链接（GitHub / 演示地址）
   - 作品描述（技术栈、职责）
   - 封面图
4. 建议上传3-5个代表性项目

### 第四步：认证
1. 完成实名认证（身份证 + 支付宝芝麻信用）
2. 完成技能认证（可选，但能提高接单率）

---

## 4. 发布服务/接单流程

### 方式一：完善简历接单（被动）
1. 完善简历后，设置为「可接单」状态
2. 等待需求方主动邀请
3. 收到邀请后，查看需求详情报价

### 方式二：投标接单
1. 进入「全部需求」页面
2. 筛选：技能匹配、预算、周期
3. 点击「报名投标」
4. 填写：项目报价、交付周期、解决方案简述
5. 等待需求方审核

### 方式三：技能服务（主动）
1. 进入「发布服务」页面
2. 选择服务类型：整包项目 / 驻场开发 / 远程兼职
3. 填写服务详情：
   - 服务标题（简洁明了）
   - 服务描述
   - 技能标签
   - 价格区间
   - 交付周期
4. 提交审核

## 6. 注意事项

1. **资料完整度影响排名**：简历完整度越高，接单机会越多
2. **报价要合理**：参考平台上同类型项目的报价，避免过高或过低
3. **沟通要及时**：需求方通常会同时联系多人，响应快者得
4. **合同保障**：大额项目务必签订平台合同，避免私下交易
5. **先收定金**：建议预收30%-50%定金再开始开发
6. **交付物要规范**：代码要写注释，文档要清晰，方便交接

## 7. 常见问题

**Q: 接单需要收费吗？**
A: 平台会收取服务费，比例约为10%-20%，具体看服务类型。

**Q: 如何提高接单率？**
A: 完善简历、提高技能认证评分、快速响应需求、积累好评。

**Q: 款项多久到账？**
A: 需求方确认验收后，款项会进入平台账户，可随时提现到支付宝/银行卡。

**Q: 可以拒绝需求方的邀请吗？**
A: 可以，但频繁拒绝会影响接单权重，建议谨慎操作。
