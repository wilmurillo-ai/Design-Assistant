# 闲鱼发布技能

## 1. 平台简介

闲鱼是阿里巴巴旗下的二手交易平台，用户可以发布闲置物品、技能服务进行交易。也可以发布「付费服务」类目，用技能变现。

**网址**: https://2.taobao.com / 闲鱼App

---

## 2. 发布闲置服务流程

### 步骤一：下载并登录
1. 下载「闲鱼」App（iOS / Android）
2. 使用淘宝账号或支付宝账号登录
3. 完成实名认证（必须）

### 步骤二：发布内容
1. 点击右下角「+」号
2. 选择发布类型：
   - 「发布闲置」（实物商品）
   - 「发布服务」（技能服务）
3. 选择「发布服务」

### 步骤三：填写信息
1. **标题**：简洁，突出关键词（建议15-25字）
2. **描述**：详细说明服务内容、流程、价格
3. **价格**：设置服务价格区间
4. **配图**：上传1-9张图片（服务场景/案例/成果图）
5. **分类**：选择对应分类，如「IT服务 > 软件开发」
6. **标签**：添加热门标签，增加曝光

### 步骤四：发布
1. 确认信息无误
2. 点击「发布」

### 商品配图建议
- 首图：文字海报图，突出服务卖点
- 后续图：案例截图/成果展示/流程说明/资质证明
- 建议尺寸：1:1 正方形或 3:4 竖图
- 避免：模糊图、过多文字、水印遮挡

## 4. 注意事项

1. **标题关键词要精准**：用户通过搜索找到你，「Python开发」「小程序」等词要出现
2. **首图决定点击率**：设计一张吸引眼球的主图，突出「快速」「低价」「专业」
3. **价格要透明**：提前说明是否有额外费用，避免纠纷
4. **响应要快**：闲鱼用户习惯快速咨询，回复慢会被跳过
5. **避免违规词**：不要出现「最」「第一」「国家级」等极限词
6. **实物+服务混合**：如果有实物交付（如软件工具），可以同时发布「闲置商品」
7. **芝麻信用很重要**：信用分高会获得更多曝光和买家信任

## 5. 常见问题

**Q: 闲鱼发布服务需要收费吗？**
A: 发布本身免费，成交后闲鱼会收取一定比例的服务费（约1%-5%）。

**Q: 如何让更多人看到我的服务？**
A：优化标题关键词、上传清晰配图、提高芝麻信用分、保持在线、及时回复。

**Q: 买家付款后钱什么时候到账？**
A：买家确认收货后，资金自动转入支付宝账户。

**Q: 可以发布虚拟物品吗？**
A：可以，但要注意避免违规（如盗版软件、违规资源）。

**Q: 出现纠纷怎么办？**
A：保留聊天记录作为证据，可申请闲鱼小法庭裁决。
