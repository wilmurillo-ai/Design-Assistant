# JSON Diff Tool

免费的在线 JSON 对比工具，支持路径级别差异分析。

## 功能

- 🔍 路径级别差异对比
- 📊 统计：新增/删除/未变数量
- 🎨 语法高亮
- ⚡ 纯前端，数据不上传服务器
- 📱 响应式设计
- ⌨️ Ctrl+Enter 快捷对比

## 本地运行

```bash
# 克隆
git clone https://github.com/shenghoo123-png/json-diff-tool.git
cd json-diff-tool

# 直接用浏览器打开 index.html
# 或启动本地服务器
python3 -m http.server 8080
# 访问 http://localhost:8080
```

## 快速开始

1. 粘贴两个 JSON 到左右两侧
2. 点击「对比」或按 Ctrl+Enter
3. 查看差异结果
