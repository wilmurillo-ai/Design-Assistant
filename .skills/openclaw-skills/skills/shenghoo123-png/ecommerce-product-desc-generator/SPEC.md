# 电商产品描述批量生成器 - 设计文档

## 1. 概念与愿景

为电商卖家提供一站式、高质量的产品描述批量生成工具。支持亚马逊、淘宝、拼多多、TikTok Shop、Shopify 五大平台，一键生成平台适配的标题、五点描述（bullet points）和详情页文案，大幅降低卖家编写文案的时间和门槛。

## 2. 目标用户

- 跨境电商卖家（亚马逊、TikTok Shop、Shopify）
- 国内电商卖家（淘宝、拼多多）
- 独立站卖家（Shopify）
- 有批量上新需求的电商团队

## 3. 功能列表

### 核心功能
- **多平台生成**：一次输入，同时生成5个平台的产品描述
- **平台适配**：每个平台使用对应的文案风格和格式
- **批量生成**：支持 CSV 导入，批量生成多产品描述
- **多格式导出**：Markdown / TXT / CSV

### 支持平台及风格

| 平台 | 语言 | 标题风格 | 描述风格 |
|------|------|----------|----------|
| 亚马逊 | 英文 | SEO关键词堆砌，200字符内 | 5条bullet points + 详情段落 |
| 淘宝 | 中文 | 热词+属性词，30字内 | 卖点多条+详情图描述 |
| 拼多多 | 中文 | 低价引流词+核心卖点 | 极致简洁，爆点前置 |
| TikTok Shop | 英文 | 口语化，含Emoji | 短视频风格，种草安利 |
| Shopify | 英文 | 品牌感，SEO友好 | 产品特点+使用场景 |

### 输入参数
- 产品名称（必填）
- 品类/类目（必填）
- 关键词（选填，多个用逗号分隔）
- 目标价格（选填）
- 品牌名（选填）

### 输出格式
- `--format markdown`：每个平台一个 Markdown 区块
- `--format txt`：纯文本，每个平台用 `===` 分隔
- `--format csv`：CSV表格，列为平台名

## 7. 核心算法

使用"模板槽填充 + 词汇池随机组合"策略：

1. **词汇池**：每个平台维护词库（开头词、卖点词、品质词、场景词）
2. **模板系统**：每个平台有3-5个标题模板和描述模板
3. **关键词注入**：用户提供的关键词优先插入标题
4. **多样性保证**：每次生成随机选择不同模板，避免重复
