📖 使用说明 ▼
$.key
获取顶层对象的 key 值
$.name → Alice
$.key.nested
嵌套访问，查询深层属性
$.address.city → Beijing
$.array[index]
访问数组指定索引，从 0 开始
$.friends[0].name → Bob
$.array[*]
通配符 * 匹配数组所有元素
$.books[*].title → [所有书名]
$.key.*
通配符 * 匹配对象所有子属性
$.address.* → {所有属性}
$.a[0].b[*].c
组合使用：数组索引 + 通配符
$.users[0].orders[*].id
@.price>10
类型过滤（仅表格视图支持）
@.price>100 · @.age>=18
