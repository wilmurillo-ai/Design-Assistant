# Digital Product Creator

Complete workflow for building, validating, and selling digital products — with zero upfront cost and no physical inventory.

## The 5-Phase System

### Phase 1 — Validate Before You Build

**Do this before writing a single word.**

Ask: Is there existing demand for this?
1. Search Etsy (etsy.com/search) for the category + keyword. Are there 10+ shops selling this? → Demand confirmed.
2. Search Reddit (reddit.com/r/Entrepreneur, r/digitalnomad) for the problem. Do people complain about it? → Pain confirmed.
3. Search Amazon Kindle for similar ebooks. Are there books with 50+ reviews? → Market confirmed.

If all 3 confirm → Build it.
If 1-2 confirm → Pivot the angle.
If 0 confirm → Choose a different product.

**Key insight:** Products that solve confirmed pain (people already paying for alternatives) sell faster than products that create new categories.

### Phase 3 — Build It Fast

**AI Prompt Pack structure (battle-tested):**
- File 1: Index with table of contents
- File 2: [Category Name] Prompts (20-50 prompts per category)
- File 3: Usage guide + 5 example sessions
- File 4: Bonus prompts for early buyers

**Ebook structure (5-day build):**
- Day 1: Chapter outline (use AI to draft, you edit)
- Day 2-3: Write chapter by chapter (AI first draft, your expertise shapes depth)
- Day 4: Design cover on Canva (or hire on Fiverr for $5)
- Day 5: Export to PDF, write sales page

**Use AI as accelerator, not replacement:**
- AI: First drafts, research synthesis, formatting
- You: Industry expertise, editing, strategic decisions

---

### Phase 4 — Set Up Sales Infrastructure

**Option A — Gumroad (fastest, 5 minutes):**
1. Go to gumroad.com → Start selling
2. Create product: Name + Description + Price + Upload file
3. Set URL: gumroad.com/l/[your-product-slug]
4. Embed on Carrd site or Notion page

**Option B — Etsy (organic traffic, longer setup):**
1. Etsy shop → "Sell a digital item"
2. 10 listings minimum for algorithm visibility
3. Title: [What it is] — [Who it's for] — [Key benefit]
4. Tags: 13 tags, mix of broad + specific
5. Photos: Canva mockup showing the product open

**Pricing psychology:**
- $9–$14: Impulse buy zone — first-time buyers
- $19–$29: Value zone — serious buyers
- $47+: Premium — bundling required
- Launch discount: List at 2x, launch at 0.5x (e.g., $29 listed → $14 launch)

---

### Phase 5 — Launch Without an Audience

**Pre-order validation (zero risk):**
1. Write a 1-page landing page (Carrd.co, free)
2. Describe the product and collect emails
3. Send to your network, Reddit, relevant communities
4. 50+ emails signed up → Validated, build it
5. < 50 emails → Pivot or improve the pitch

**Launch sequence:**
- Week -1: Set up product page, pre-order open
- Day 0: Post on Reddit (r/Entrepreneur, r/digitalnomad, relevant niche)
- Day 1: Post on Twitter/X with demo
- Day 3: Email collected list
- Day 7: Reminder post — "Pre-order closes Friday"
- Day 14: Send to Gumroad followers (if you have any)

## Recommended Resources

For creating your digital product files, use these tools:
- **Canva** (canva.com) — Free cover and mockup design
- **Notion** (notion.so) — Template creation
- **ChatGPT/Claude** — Content generation and editing
- **Gumroad** (gumroad.com) — Payment processing and delivery
- **Carrd.co** — Free one-page landing pages

## Files to Deliver

For AI Prompt Packs, deliver as `.zip` containing:
- `README.md` — Product overview and how to use
- `[Category-1].md` — Prompts organized by category
- `[Category-2].md` — Additional categories
- `examples.md` — Example sessions showing output quality

For Ebooks, deliver as `.pdf`:
- Professional cover (Canva template)
- Formatted interior (Amazon KDP compatible)
- Bonus appendix or worksheets

For Templates, deliver as `.zip`:
- Working Notion template link or `.zip` of `.canva` files
- Setup instructions
- Customization guide
