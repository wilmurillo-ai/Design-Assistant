# 30-Day Content Calendar Template

## How to Use This Template

1. Choose ONE anchor content piece per week
2. Fill in the calendar below
3. Repurpose from one format to 4-6 outputs per anchor
4. Batch create all content on Monday

## Weekly Anchor System

| Week | Anchor Content Type | Platforms Active |
|---|---|---|
| Week 1 | How-to article | LinkedIn, Twitter, Email, Blog |
| Week 2 | Opinion/Hot take | LinkedIn, Twitter, Instagram |
| Week 3 | Case study | LinkedIn, Email, Twitter |
| Week 4 | Tools/Resources list | LinkedIn, Twitter, Blog |

## Content Creation Batch Workflow (90 min/week)

1. **[10 min]** Read/select anchor content
2. **[20 min]** Generate LinkedIn post + Twitter thread (AI prompts)
3. **[15 min]** Edit AI outputs — add personal voice
4. **[15 min]** Write email version
5. **[10 min]** Create Instagram caption
6. **[20 min]** Schedule everything (Buffer/Later)
