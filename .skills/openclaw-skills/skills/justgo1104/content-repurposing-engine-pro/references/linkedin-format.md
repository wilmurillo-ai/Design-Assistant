## 5 Hook Templates That Stop the Scroll

### Type 1: The Bold Number
```
[Number] is the new [old thing].
[One-sentence explanation]
Example: "Remote work is the new 9-to-5. Here's why most people can't handle it."
```

### Type 2: The Counter-Intuitive Claim
```
Most [profession/experts] get this wrong.
[Contrarian statement]
Example: "Most freelance writers charge by the hour. That's the slowest path to $100K."
```

### Type 3: The Story Opener
```
I made [$X / achieved Y] in [timeframe] by doing [unusual thing].
[One sentence on what it taught]
Example: "I turned down a $50K client last month. It was the best decision I made all year."
```

### Type 4: The "Nobody Talks About" Frame
```
Nobody tells you this about [industry/topic].
[One sentence that validates the reader's frustration]
Example: "Nobody tells you that most 'productivity tips' actually make you slower."
```

### Type 5: The Question Hook
```
What's the difference between [A] and [B]?
[One-sentence challenge]
Example: "What's the difference between a freelancer and a business owner? One word."
```

## LinkedIn Post Template

```
[HOOK — 1-3 sentences, first line must be max 150 chars]

[BODY — 3-5 short paragraphs, each max 2 sentences]

Here's my question for you: [specific question that drives comments]

[2-3 specific hashtags]

---
Save this post if you found it useful.
```

## LinkedIn Best Practices

- **Post timing:** Tuesday–Thursday, 7–9am or 5–7pm (poster's timezone)
- **First 50 words matter most** — algorithm reads this for relevance
- **Questions get 3x more comments** than statements
- **"Save this" phrasing increases saves without seeming spammy**
- **Hashtags:** Exactly 3, specific (not #business or #success)
