# Twitter/X Thread Format — Complete Reference

## Thread Structure

### The Opening Hook (Tweet 1)
Rules:
- No emoji in the hook
- Bold claim, surprising stat, or counter-intuitive statement
- Creates tension or curiosity gap
- Max 250 characters
- Should make reader NEED to read tweet 2

### The Middle (Tweets 2–4)
- Each tweet: one idea, one example
- Can use emoji (sparingly, and NOT in tweet 1)
- Each tweet should work stand-alone
- Read time: ~30 seconds for full thread

### The Closer (Tweet 5+)
- Key takeaway (the action or insight)
- Soft CTA: follow, reply, share
- Can include a relevant emoji here

## Hook Library (50 Examples)

### For Opinion/Takes
1. "Hot take: The best time to start was 3 years ago. The second best time is today."
2. "Controversial opinion: Working more hours doesn't make you more productive."
3. "Nobody is talking about this, but..."

### For Tips/How-To
4. "The 80/20 of [skill] in one thread:"
5. "The biggest mistake [profession] make (and how to fix it):"
6. "[Number] things I wish I knew about [topic]:"

### For Case Studies
7. "I made $[X] in [time] using [method]. Here's what worked:"
8. "I interviewed [X] successful people. This is what they all had in common:"
9. "The exact system I used to [achieve result]:"

### For Data/Research
10. "[X]% of [group] don't know this. That changes everything."
11. "New data on [topic] confirms what experts suspected:"
12. "The numbers behind [common assumption] are shocking."

## Thread Writing Rules
- Never use cliffhangers — each tweet must deliver value on its own
- Read the thread aloud — if it sounds robotic, rewrite
- Numbers work: specific > vague ("$4,732" > "thousands of dollars")
- Use "you" and "I" — not "one" or "people"
- Write tight: every word must earn its place
