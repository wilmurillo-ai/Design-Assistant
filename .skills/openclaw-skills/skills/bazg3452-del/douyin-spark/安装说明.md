# 抖音续火花 Skill - 安装指南

🔥 欢迎使用抖音自动续火花 Skill！

## 📦 安装步骤

### 步骤 1：解压文件

将压缩包解压到任意位置，例如：
```
C:\Users\你的用户名\.openclaw\workspace\skills\douyin-spark\
```

### 步骤 2：配置联系人列表

1. 打开 `memory/douyin-spark-contacts.md` 文件
2. 按照表格格式添加你的火花联系人
3. 格式示例：

```markdown
| 用户名 | 数字 | 火花状态 | 备注 |
|--------|------|----------|------|
| 朋友 A | 100 | 灰色火花 | 需要续 |
| 朋友 B | 200 | 彩色火花 | 保持中 |
```

### 步骤 3：在 OpenClaw 中使用

解压后，技能会自动被 OpenClaw 识别。

在对话中直接说：
- "帮我续火花"
- "查看火花联系人列表"
- "续火花，说：你好呀"

## 🔧 命令行使用（可选）

```bash
# 进入技能目录
cd ~/.openclaw/workspace/skills/douyin-spark

# 链接到全局（可选）
npm link

# 使用命令
douyin-spark --list          # 查看联系人
douyin-spark --message "你好" # 续火花
```

## 📝 火花规则

| 状态 | 说明 | 操作 |
|------|------|------|
| 🔥 彩色火花 | 正常状态，连续聊天中 | 每天发消息保持 |
| 🌫️ 灰色火花 | 超过 24 小时未聊天 | 发消息重新点亮 |
| ⏳ 重燃中 | 火花已灭，恢复中 | 连续聊 3 天恢复 |

## ⚠️ 注意事项

1. **每天至少发一次消息** - 保持火花不灭
2. **避免相同内容** - 可能被抖音限流
3. **浏览器需登录** - 确保抖音网页版已登录
4. **先配置联系人** - 使用前务必填写联系人列表

## 🆘 遇到问题？

1. 检查抖音网页版是否已登录
2. 确认联系人列表格式正确
3. 重启 OpenClaw gateway：`openclaw gateway restart`
