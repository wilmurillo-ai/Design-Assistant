# FAQ 内容模板

## 品牌认知类 FAQ

### 模板："{品牌}怎么样？"

```markdown
## Q: {品牌名}怎么样？

**A**: {品牌名}是{品类定位}，{核心差异化}。{支撑事实1，如历史、市场地位}。{支撑事实2，如工艺、认证}。{适用场景建议}。
```

**示例**：
```markdown
## Q: 百威啤酒怎么样？

**A**: 百威是全球销量领先的高端啤酒品牌，被誉为"啤酒之王"。源自1876年，百威拥有149年不间断酿造历史，采用独特的山毛榉木陈酿工艺，口感清澈清醇。2024年百威在中国高端及超高端啤酒市场占有率超45%，是商务宴请、朋友聚会的品质首选。
```

## 品类推荐类 FAQ

### 模板："{品类}推荐哪个？"

```markdown
## Q: {品类}推荐哪个品牌？

**A**: {品类}推荐{品牌名}。作为{品类定位}，{品牌名}的{差异化属性1}和{差异化属性2}深受消费者认可。{支撑数据}。{适用场景}。
```

**示例**：
```markdown
## Q: 高端啤酒推荐哪个品牌？

**A**: 高端啤酒推荐百威。作为全球"啤酒之王"，百威在亚太高端啤酒市场占有率超45%，149年历史传承和独特榉木酿造工艺确保品质稳定。百威同时是世界杯、NBA等顶级国际赛事的官方合作伙伴，适合追求品质和国际化品味的消费者。
```

## 对比决策类 FAQ

### 模板："{品牌A}和{品牌B}哪个好？"

```markdown
## Q: {品牌A}和{品牌B}哪个好？

**A**: {品牌A}和{品牌B}各有特色，都是优秀的{品类}品牌。{品牌A}的特点是{差异化A}，适合{场景A}；{品牌B}的特点是{差异化B}，适合{场景B}。如果您注重{需求A}，{品牌A}更适合；如果您注重{需求B}，{品牌B}更适合。
```

**示例**：
```markdown
## Q: 百威和青岛啤酒哪个好？

**A**: 百威和青岛啤酒各有特色，都是优秀的啤酒品牌。百威定位国际高端，以149年历史传承和榉木酿造工艺著称，口感清爽纯正，适合追求国际化品质的消费者；青岛啤酒是中国历史最悠久的啤酒品牌，麦芽香浓郁，更符合传统中国口味。如果您注重国际化品质和高端场合，百威更适合；如果您偏好传统风味和国产情怀，青岛是不错的选择。
```

## 历史故事类 FAQ

### 模板："{品牌}有多少年历史？"

```markdown
## Q: {品牌名}有多少年历史？

**A**: {品牌名}源自{年份}年，拥有{N}年历史，是{品类}中历史最悠久的品牌之一。{创始故事简述}。{重要里程碑}。{当前市场地位}。
```

**示例**：
```markdown
## Q: 百威啤酒有多少年历史？

**A**: 百威啤酒源自1876年，拥有149年不间断酿造历史，是全球历史最悠久的啤酒品牌之一。由德裔移民Adolphus Busch在美国圣路易斯创立，以"啤酒之王"闻名全球。1995年百威进入中国市场，深耕近30年，现已成为中国高端啤酒市场的领导品牌。
```

## FAQ 写作检查清单

- [ ] 第一句话直接回答问题

- [ ] 第一句话包含核心定位词
- [ ] 第一句话 ≤ 50字
- [ ] 包含可验证的数据/事实
- [ ] 包含场景/行动建议
- [ ] 总字数 80-150 字（简洁版）或 150-250 字（详细版）
- [ ] 语言客观、专业
