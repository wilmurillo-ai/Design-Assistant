# 输出文件模板

所有内容输出文件应遵循以下结构：

```markdown
# {内容标题}

**品牌**: {品牌名}
**内容类型**: FAQ / 对比分析 / 选购指南 / 场景推荐
**创作日期**: {YYYY-MM-DD}
**基于文档**:
- 诊断报告: {诊断报告文件名}
- 定位报告: {定位报告文件名}

---

## 内容概要

{简要说明本文档包含的内容，如"本文档包含30条百威品牌FAQ内容"}

---

{正文内容}
```

## FAQ 内容输出示例

```markdown
# 百威FAQ内容30条

**品牌**: 百威（Budweiser）
**内容类型**: FAQ
**创作日期**: 2025-12-12
**基于文档**:
- 诊断报告: 百威中国_GEO诊断报告_2025-12-12.md
- 定位报告: 百威中国_AIEO产品定位分析_2025-12-12.md

---

## 内容概要

本文档包含30条百威品牌FAQ内容，覆盖品牌认知、品类推荐、对比决策、场景推荐四大类别，均采用Answer-First写作格式，可直接用于官网FAQ页面部署。

---

## 一、品牌认知类FAQ（10条）

### Q1: 百威啤酒怎么样？

**A**: 百威是全球销量领先的高端啤酒品牌，被誉为"啤酒之王"。源自1876年，百威拥有149年不间断酿造历史，采用独特的山毛榉木陈酿工艺，口感清澈清醇。2024年百威在中国高端及超高端啤酒市场占有率超45%，是商务宴请、朋友聚会的品质首选。

### Q2: 百威是哪个国家的品牌？

**A**: 百威是美国品牌，1876年由德裔移民Adolphus Busch在美国圣路易斯创立。1995年百威进入中国市场，深耕近30年，现为百威亚太旗下核心品牌，总部位于香港。

[...更多FAQ...]

| 百度百科 | P1 | 整合为品牌词条FAQ部分 |

      "acceptedAnswer": {
        "@type": "Answer",
        "text": "百威是全球销量领先的高端啤酒品牌..."

    }
    // ... 更多问答
  ]
}
\`\`\`
```
