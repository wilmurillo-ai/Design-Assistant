## 使用说明

### 何时生成内容计划

1. 完成 AIEO 诊断报告后
2. 完成产品定位分析后
3. 开始内容创作前

### 如何使用本模板

1. **读取诊断报告**：提取AI可见性得分、内容缺口、竞品分析
2. **读取定位报告**：提取定位声明、消息层级、差异化属性
3. **确定行业**：参考 `industry_strategies.md` 确定行业策略
4. **确定平台**：参考 `ai_platform_specs.md` 确定平台优先级
5. **生成清单**：列出具体内容创作任务
6. **制定时间表**：安排发布节奏
7. **设定KPI**：明确效果追踪指标

### 输出文件命名

```
{品牌名}_内容生成计划_{YYYY-MM-DD}.md

示例：
- 百威_内容生成计划_2025-12-12.md
- 飞鹤_内容生成计划_2025-12-12.md
```
