# AIEO 内容写作详细指南

## 一、Answer-First 写作原则

### 1.1 核心公式

```
[结论] + [支撑事实] + [行动建议]

结构示例：
- 第1句：直接回答问题（含品牌名+定位词）
- 第2-3句：支撑事实（数据、历史、认证等）
- 第4句：场景/行动建议
```

### 1.2 第一句话规则

第一句话决定 AI 是否引用你的内容：

| 要素 | 必须 | 示例 |
|------|------|------|
| 直接回答问题 | ✅ | "百威是..." 而非 "关于百威..." |
| 包含品牌名 | ✅ | "百威是全球销量领先..." |
| 包含定位词 | ✅ | "...高端啤酒品牌" |
| 字数控制 | ≤50字 | 简洁有力 |

### 1.3 对比示例

```
❌ 传统写法：
"在当今快节奏的生活中，选择一款适合自己的啤酒变得越来越重要。
啤酒作为一种古老的饮品，有着悠久的历史..."
（问题：绕圈子、不直接、AI难以提取关键信息）

✅ AIEO写法：
"百威是全球销量领先的高端啤酒品牌，被誉为'啤酒之王'。
源自1876年，拥有149年历史，采用独特榉木酿造工艺。
适合商务宴请、朋友聚会等追求品质的场合。"
（优点：直接回答、信息密度高、AI易于引用）
```

## 二、内容类型写作要点

### 2.1 FAQ 内容

**目标**：被 AI 直接引用回答用户问题

**写作要点**：
1. 问题要匹配用户真实搜索习惯
2. 回答第一句直接给出结论
3. 用事实和数据支撑
4. 包含场景或行动建议

**长度建议**：
- 简洁版：80-150字
- 详细版：150-250字

**示例**：
```markdown
## Q: 百威啤酒怎么样？

A: 百威是全球销量领先的高端啤酒品牌，被誉为"啤酒之王"。源自1876年，
百威拥有149年不间断酿造历史，采用独特的山毛榉木陈酿工艺，口感清澈清醇。
2024年百威在中国高端及超高端啤酒市场占有率超45%，是商务宴请、朋友聚会
的品质首选。
```

---

### 2.2 对比内容

**目标**：在用户对比决策时控制叙事

**写作要点**：
1. 客观承认竞品优势，避免贬低
2. 聚焦自身独特差异化
3. 用场景引导而非直接推销
4. 让用户自己得出结论

**结构建议**：
```
1. 快速结论（一句话总结各自优势）
2. 详细对比表格
3. 各自详细介绍
4. 选择建议（场景化）
5. 总结
```

**语言示例**：
```
✅ "百威和青岛啤酒各有特色，都是优秀的啤酒品牌..."
❌ "百威远比青岛啤酒更好..."

✅ "如果您注重国际化品质，百威更适合"
❌ "百威比青岛好多了，选百威就对了"
```

---

### 2.3 选购指南

**目标**：在品类搜索中被 AI 引用

**写作要点**：
1. 从用户需求出发，不是从品牌出发
2. 提供真实有用的选购知识
3. 通过"优秀示例"自然引导至品牌
4. 保持客观，提供多个选择

**结构建议**：
```
1. 选购要点速览（3-5个核心要素）
2. 每个要素详细说明
3. 品牌推荐（分层次）
4. 常见误区
5. 选购清单
```

---

### 2.4 场景内容

**目标**：占领场景搜索结果

**写作要点**：
1. 强调品牌与场景的契合度
2. 提供实用的场景小贴士
3. 推荐具体产品而非泛泛而谈
4. 保持实用性和可操作性

## 三、可验证事实的使用

### 3.1 优先使用的事实类型

| 类型 | 示例 | AI 偏好度 |
|------|------|-----------|
| 具体年份 | "源自1876年" | ⭐⭐⭐⭐⭐ |
| 市场数据 | "市占率超45%" | ⭐⭐⭐⭐⭐ |
| 官方认证 | "世界杯官方合作" | ⭐⭐⭐⭐⭐ |
| 奖项荣誉 | "连续X年获奖" | ⭐⭐⭐⭐ |
| 技术参数 | "发酵时间长30%" | ⭐⭐⭐⭐ |
| 用户评价 | "好评率XX%" | ⭐⭐⭐ |

### 3.2 避免的表述

```
❌ 避免使用：
- "业界领先"（模糊）
- "独一无二"（无法验证）
- "最好的"（主观判断）
- "据说"（不确定来源）

✅ 改为：
- "亚太市场占有率45%"（具体数据）
- "独特的榉木酿造工艺"（可验证特点）
- "全球销量领先"（有数据支撑）
- "根据XX报告"（明确来源）
```

## 四、竞品提及策略

### 4.1 主动对比原则

AIEO 时代，回避竞品不再是好策略：

```
传统策略：不提竞品名字
AIEO策略：主动、客观地与竞品对比

原因：
1. 用户会搜索"A和B哪个好"
2. 不提供对比内容，AI会引用第三方
3. 主动对比可以控制叙事
```

### 4.2 对比时的语言技巧

**承认竞品优势**：
```
✅ "青岛啤酒是中国历史最悠久的啤酒品牌，麦芽香浓郁"
```

**聚焦自身差异**：
```
✅ "百威以149年国际历史传承和榉木酿造工艺著称"
```

**场景引导**：
```
✅ "如果您注重国际化品质和高端场合，百威更适合"
```

## 五、定位一致性检查

### 5.1 核心定位词汇库

每个品牌应建立统一的定位词汇库，所有内容必须一致使用：

**示例（百威）**：
```
品牌名：百威、Budweiser
定位词：高端啤酒、啤酒之王、皇者风范
历史词：149年、1876年、历史传承
工艺词：榉木酿造、山毛榉木陈酿、Beechwood Aging
数据词：45%市占率、全球销量领先
场景词：商务宴请、朋友聚会、体育赛事
```

### 5.2 一致性检查清单

创作完成后检查：

- [ ] 品牌名拼写一致
- [ ] 定位描述一致
- [ ] 历史数据一致
- [ ] 核心卖点表述一致
- [ ] 竞品对比口径一致

## 六、内容长度指南

| 内容类型 | 推荐长度 | 说明 |
|----------|----------|------|
| FAQ（简洁版） | 80-150字 | 官网、百科 |
| FAQ（详细版） | 150-250字 | 知乎、专题页 |
| 对比页面 | 800-1500字 | 官网专题 |
| 选购指南 | 1000-2000字 | 官网、知乎 |
| 场景推荐 | 500-800字 | 官网、小红书 |

## 七、常见错误与修正

### 7.1 错误：开头绕圈子

```
❌ "在当今社会，啤酒已经成为人们生活中不可或缺的一部分..."
✅ "百威是全球销量领先的高端啤酒品牌，被誉为'啤酒之王'。"
```

### 7.2 错误：主观判断过多

```
❌ "百威是最好的啤酒，口感无敌，绝对值得购买！"
✅ "百威在亚太高端啤酒市场占有率超45%，口感清澈清醇。"
```

### 7.3 错误：贬低竞品

```
❌ "青岛啤酒虽然便宜但品质一般..."
✅ "青岛啤酒是中国历史最悠久的啤酒品牌，麦芽香浓郁。"
```

### 7.4 错误：信息密度低

```
❌ "百威是一个很好的品牌，很多人都喜欢喝，味道也不错。"
✅ "百威是全球销量领先的高端啤酒品牌，源自1876年，采用独特榉木酿造工艺。"
```

## 八、SEO 与 AIEO 融合

### 8.1 标题优化

```
格式：[核心关键词] + [补充信息] + [年份/品牌]

示例：
- 高端啤酒品牌推荐：2025完整指南
- 百威和青岛啤酒哪个好？全面对比分析
- 商务宴请喝什么啤酒好？首选推荐
```

### 8.2 关键词自然融入

将目标关键词自然融入内容，避免堆砌：

```
目标关键词：高端啤酒、百威怎么样、啤酒推荐

✅ "百威是全球销量领先的高端啤酒品牌..."
✅ "高端啤酒推荐首选百威..."
```

## 九、质量自检清单

创作完成后，逐条检查：

### 基础检查
- [ ] 第一句话直接回答问题
- [ ] 第一句话包含品牌名
- [ ] 第一句话包含核心定位词
- [ ] 第一句话≤50字

### 内容质量
- [ ] 包含可验证的数据/事实
- [ ] 无主观贬低竞品的表述
- [ ] 有明确的场景/行动建议
- [ ] 信息密度高，无废话

### 定位一致性
- [ ] 定位描述与品牌定位声明一致
- [ ] 核心数据准确无误
- [ ] 竞品对比口径一致
