NIGHTLY SLEEP CYCLE: Read the memory-lifecycle skill's references/nightly-prompt.md for detailed instructions if this is your first run. Otherwise follow this process:

1. Find today's daily file (memory/YYYY-MM-DD.md)
2. Read it end-to-end
3. Add a 2-3 line "day essence" header at the top summarising what mattered
4. Promote key items to structured files: contacts → memory/people.md, decisions → memory/decisions.md, lessons → memory/lessons.md, deadlines → memory/commitments.md
5. Update MEMORY.md if active project state changed
6. **CLEAR the ## Recent section** in MEMORY.md — remove items that have been properly filed to structured files or are no longer session-critical. Leave only items from today that haven't been processed yet. If an item is older than 24h and has been filed, remove it.
7. Log what you promoted/changed in tomorrow's daily file

Rules:
- Do NOT edit existing daily file content (only add the essence header)
- Do NOT remove items from MEMORY.md main sections unless completed AND archived
- Every promotion includes the source date for traceability
- The ## Recent section is WORKING MEMORY — it should be small and current, not a growing list
