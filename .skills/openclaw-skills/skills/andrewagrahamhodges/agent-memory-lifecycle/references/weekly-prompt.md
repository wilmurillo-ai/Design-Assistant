WEEKLY REFLECTION: Systematic review of the past 7 days to spot patterns, refine memory, and catch gaps.

## Steps

1. **Read daily files:** Read all 7 daily files from the past week (`memory/YYYY-MM-DD.md`), including their day essence headers.

2. **Pattern detection:** Look for:
   - Repeated topics or themes across multiple days
   - Unresolved threads — things mentioned but never concluded
   - Questions asked multiple times (knowledge gaps)
   - Emerging relationships or recurring contacts

3. **Refine MEMORY.md:**
   - Tighten language — say the same thing in fewer words without losing meaning
   - Update stale descriptions (projects that changed status, resolved issues)
   - Improve structure — group related items, fix ordering
   - **Refinement means IMPROVING structure and language, NOT stripping content**
   - If removing anything from MEMORY.md, it must first exist in a structured file or archive

4. **Review structured files:**
   - `memory/commitments.md` — any missed or upcoming deadlines? Update status.
   - `memory/people.md` — any stale contacts or new relationships forming? Update.
   - `memory/decisions.md` — any decisions from this week that need outcome tracking?
   - `memory/lessons.md` — any new lessons to add from this week's daily files?

5. **Write "week in review":** Add a section in today's daily file:
   ```
   ## Week in Review (YYYY-MM-DD to YYYY-MM-DD)
   - **Key themes:** [2-3 themes that dominated the week]
   - **Completed:** [what got finished]
   - **Carried forward:** [what's still active]
   - **Knowledge gaps:** [things to investigate]
   ```

6. **Identify knowledge gaps:** Things you should know but don't — questions that came up repeatedly without clear answers. Note these for investigation.

## Rules
- Refinement means IMPROVING structure and language, NOT stripping content
- If removing anything from MEMORY.md, it must first exist in a structured file or archive
- Flag anything uncertain to the human rather than guessing
- Do NOT edit existing daily file content (only add the week-in-review section to today's file)
