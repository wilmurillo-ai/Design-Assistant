## 路径选择框架 {#framework}

选路径前先明确三个维度的优先级：

```
1. 合规性：是否必须走完全合规通道？
   → 是：只考虑 SWIFT/Wise/CITIC 等正规渠道
   → 容忍灰色：P2P 加密可作备选（了解风险）

2. 规模：金额多大？
   → <$5,000：大多数渠道均可，手续费影响小
   → $5,000–$50,000：注意年度额度；SWIFT 中间行费用影响变小
   → >$50,000：超年度限额，需合规拆分或企业渠道

3. 时效：多急？
   → 当天：P2P 加密或 Wise（如已有境外账户）
   → 1–3 天：Wise + CITIC 组合，或国内 SWIFT
   → 3–5 天：国内银行 SWIFT 直汇
```

## 人民币出境（CNY → 境外法币）{#outbound}

### 路径 A：国内银行 SWIFT 直汇（最合规）
```
适合：有明确用途（留学/商务），金额在年度 $50,000 以内
流程：国内银行网银 → 填写汇款申请 → 2–5 工作日到账

费用估算（汇 USD 10,000）：
  - 电报费：CNY 150
  - 银行汇率损失（vs 中间价 2%）：约 USD 200
  - 中间行费用：USD 15–35
  - 合计损耗：约 USD 220–250（2.2–2.5%）
  
注意：必须在年度 $50,000 额度内；需申报用途
```

### 路径 B：国内银行 → 中信国际（香港）→ Wise（推荐合规路径）
```
适合：频繁汇款、追求低汇率损失、已有或愿意开立香港账户
前提：需在中信香港开户（一次性操作）

流程：
  国内行 SWIFT → 中信HK 存入 HKD
  → 中信HK 绑定 Wise 香港账户
  → Wise 转账到目标境外账户

费用估算（汇 USD 10,000）：
  - 国内→香港 SWIFT：CNY 150 + 1.5% 点差
  - Wise 手续费：~0.5%
  - 合计损耗：约 2.1%（比直接 SWIFT 略优，速度更快）

优势：汇率更好，中间行费用低，到账 1–2 天
```

### 路径 C：OKX P2P 买 USDT → Kraken 卖出（灰色，高效）
```
适合：不受年度额度限制；可接受合规风险；金额任意
前提：有 Kraken/境外交易所账户（需海外 KYC）

流程：
  国内银行转账 → OKX P2P 买 USDT (1% 溢价)
  → TRC20 提现到 Kraken ($2.5)
  → Kraken 卖出 USDT → USD
  → Kraken SWIFT 出金到境外银行 ($35)

费用估算（汇 USD 10,000）：
  - P2P 溢价：约 $100
  - Kraken 交易：$16
  - Kraken 出金：$35
  - 合计：约 $151（1.5%），速度 1–3 小时

风险：银行卡冻结风险；OKX 账户合规风险
```

### 路径 D：银联卡境外消费（小额日常）
```
适合：境外旅行日常消费，金额小（<$2,000/次）
前提：持有银联信用卡/借记卡

费用：约 1.5% FX 手续费（含银联汇率点差）
限制：取现每年累计 $10,000；消费无官方硬性上限
```

### 路径对比（汇 CNY 70,000 ≈ USD 10,000）

| 路径 | 到账金额(USD) | 损耗% | 到账速度 | 合规性 |
|------|-------------|-------|---------|--------|
| SWIFT 直汇 | ~9,750 | 2.5% | 2–5天 | 完全合规 |
| CITIC+Wise | ~9,790 | 2.1% | 1–2天 | 完全合规 |
| P2P+Kraken | ~9,849 | 1.5% | 1–3小时 | 灰色区域 |
| 银联卡消费 | ~9,850(等值) | 1.5% | 即时 | 完全合规（境外消费） |

## 境外法币入境（境外法币 → CNY）{#inbound}

### 路径 A：境外银行 SWIFT → 国内银行（最合规）
```
适合：工资/合法收入入境，有完整资金来源证明
流程：境外银行 → SWIFT → 国内银行（需提供合同/工资单）

费用估算（入 USD 5,000）：
  - 发送方手续费：$0–30（各境外银行不同）
  - 中间行费用：$15–35
  - 国内银行结汇汇率损失（vs 中间价 2%）：~$100
  - 合计：约 $130–160（2.6–3.2%）

注意：大额（$10,000+）需申报资金来源；正当收入完全合规
```

### 路径 B：Wise → 中信香港 → 国内（推荐合规路径）
```
适合：定期收取境外收入（自由职业/远程工作/投资收益）
前提：需有中信香港账户

流程：
  境外账户/Wise 收款 → 转入中信HK
  → 中信HK → SWIFT → 国内银行

费用估算（入 USD 5,000）：
  - Wise 接收费：0（持有 Wise 账户）
  - Wise → 中信 HK：~0.5%
  - 中信 → 国内 SWIFT：约 0.1% + HKD 100 固定费
  - 合计：约 1.6–2%

速度：1–2 个工作日
```

### 路径 C：Kraken 买 USDT → OKX P2P 卖出（灰色，快速）
```
适合：无完整资金来源证明；追求速度；可接受合规风险

流程：
  境外银行 → Kraken 入金(SEPA免费/SWIFT $5)
  → Kraken 买 USDT (0.16%)
  → TRC20 提现到 OKX ($2.5)
  → OKX P2P 卖 USDT → 国内银行收款

费用估算（入 USD 5,000）：
  - Kraken 入金：$5（SWIFT）
  - Kraken 交易：$8
  - 链上转账：$2.5
  - P2P 溢价：~$50（1%）
  - 合计：约 $65（1.3%），速度 2–4 小时

风险：同出境 P2P 风险；P2P 买家账户可能有风险资金
```

### 路径 D：PayPal/支付宝境外收款（小额）
```
适合：境外电商收款、小额转账
流程：PayPal 收款 → 提现到国内银行（需实名）
费用：约 3.5%（PayPal 手续费 + 汇率）；不推荐大额
```

### 路径对比（入 USD 5,000）

| 路径 | 到账金额(CNY) | 损耗% | 到账速度 | 合规性 |
|------|-------------|-------|---------|--------|
| SWIFT 直汇 | ~34,340 | 3% | 2–5天 | 完全合规 |
| Wise+CITIC | ~34,690 | 2% | 1–2天 | 完全合规 |
| Kraken+P2P | ~34,935 | 1.3% | 2–4小时 | 灰色区域 |

_(以 USD/CNY = 7.1 为例)_

## 法币与加密货币互换 {#crypto-fiat}

### CNY → USDT（出金换稳定币）
```
最佳路径：OKX P2P（成本最低，流动性最强）
  国内银行转账 → OKX P2P 挂单/吃单 → USDT
  费用：P2P 溢价 ~1%
  提现 TRC20：~$1
  总成本：约 1.1%
  速度：30–60 分钟
```

### USDT → CNY（入金变法币）
```
最佳路径：OKX P2P（同上方向）
  USDT 充值 OKX (TRC20, ~$1) → P2P 卖出 → 收取 CNY
  费用：P2P 溢价 ~1%（作为卖家）
  注意：商家可能压价；可挂单等待更好价格（费时）
  速度：20–60 分钟
```

### USD/EUR → USDT
```
最佳路径：Kraken（最合规，成本合理）
  SEPA 入金 EUR（免费）→ 买 USDT（0.16%）→ 提现 TRC20（$2.5）
  总成本：约 0.2%（大额），$40 固定（小额）

次选：Bybit/OKX 直接入金（需验证是否支持目标地区银行卡）
```

### USDT → USD/EUR
```
最佳路径：Kraken
  USDT 存入（TRC20, $2.5）→ 卖出 USD（0.16%）→ SWIFT 出金（$35）
  总成本：0.16% + $37.5
  注意：固定成本 $37.5，需 $3,750+ 才比 0.5% 合算
```

## 特殊场景 {#special}

### 场景1：留学生生活费（每月 CNY 5,000–20,000 → 境外）
**推荐：**
- 银联卡直接境外消费（最省事，损耗 1.5%）
- 或 Wise 收款账户 + 国内家长 SWIFT 汇款（合规，每月操作）
- 不建议 P2P（频繁小额操作风险高，银行容易识别异常）

### 场景2：自由职业者境外收入（每月 $1,000–5,000）
**推荐：**
- 建立 Wise 账户收款 → 中信香港 → 国内（合规）
- 或 iFAST（新加坡居民/持有 PR）
- 保留所有工作合同、发票备查（税务审计证明）

### 场景3：虚拟货币持有者在境内消费
**路径：**
```
USDT → OKX P2P 卖出（收取 CNY 到微信/支付宝/银行卡）→ 扫码消费
总损耗：约 1%（P2P 溢价）
注意：分批操作，单次不超过 5 万 CNY，避免银行风控
```

### 场景4：海外资产大额回流（>$50,000）
**合规路径（必须咨询专业律师/税务师）：**
1. 企业账户汇款（无个人年度限额，但需真实贸易背景）
2. 分年度、分家庭成员利用各自 $50,000 额度
3. 通过中信香港大额专项申请（需资金来源证明）
4. QDII 等金融产品回流（限制多，需资质）

**不合规路径的代价：**
- 外汇违规：罚款本金 30%
- 洗钱认定：刑事责任
- P2P 大额：银行账户冻结，解冻需漫长申诉
