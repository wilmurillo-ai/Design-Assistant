## OKX（欧易）{#okx}

| 项目 | 说明 |
|------|------|
| 类型 | 注册地：塞舌尔；面向全球含中文市场 |
| P2P 手续费 | 0%（挂单/吃单均免手续费） |
| P2P 价差 | 约中间价 +0.8–1.5%（CNY↔USDT，随市场波动） |
| 现货交易费 | Maker: 0.08%；Taker: 0.1% |
| USDT 提现（TRC20）| ~1 USDT |
| USDT 提现（ERC20）| ~5–15 USD（以太坊 Gas 费） |
| P2P 到账速度 | 15–30 分钟（等待对手方放币） |
| KYC 要求 | 初级：手机号；高级（法币）：身份证 + 人脸识别 |
| 中文人民币支持 | ✅ CNY P2P 最流动市场 |

**P2P 支付方式：**
- 收款：银行卡转账、支付宝、微信支付
- 注意：使用支付宝/微信更快但银行识别风险略高

**OKX P2P 买入 USDT 流程：**
1. 完成 KYC → 进入 P2P 市场
2. 选择"买入 USDT，支付 CNY"的挂单（优先选信誉 ≥ 98%、成交量高的商家）
3. 下单 → 按商家指定方式付款（银行转账/支付宝）
4. 商家确认收款 → USDT 到账 OKX 钱包
5. 提现到外部钱包（如 Bybit、Kraken 或个人冷钱包）

**风控建议：**
- 单笔不超过 CNY 5万（降低银行风控触发）
- 避免整点转账（系统自动风控）
- 备注填写"朋友借款/生活费"等（避免被标记为涉赌）
- 使用专用银行卡，不混入日常收支

| P2P 手续费 | 0% |
| P2P 价差 | 约中间价 +1–1.5%（略高于 OKX） |
| 现货交易费 | Maker: 0.1%；Taker: 0.1% |

**与 OKX 对比：**
- Bybit P2P 流动性略低于 OKX（大额成交可能需拆单）

**Bybit 特色场景：**

- 适合海外 USDT 转入后 → 境外法币出金

## Kraken {#kraken}

| 项目 | 说明 |
|------|------|
| 类型 | 美国持牌交易所（FinCEN 注册，多州 MTL 牌照） |
| 交易费 | Maker: 0.0%–0.16%；Taker: 0.1%–0.26% |
| SEPA 存款 | 免费（EUR，欧元区，T+1） |
| SWIFT 存款 | USD 5 固定费 |
| SWIFT 取现 | USD 35 固定费（主要成本！） |
| SEPA 取现 | EUR 0.09 |
| USDT 提现（TRC20）| ~2.5 USDT |
| KYC 要求 | Tier 2（法币）：政府证件 + 地址证明 + 资金来源证明 |
| 中文人民币支持 | ❌ 不支持 CNY |
| 适合场景 | 境外 USD/EUR ↔ USDT/BTC；合规出金通道 |

**Kraken 典型用法：**
```
场景A：海外法币 → USDT
USD → SWIFT 入金 Kraken ($5) → 买 USDT (0.16%) → 提现 TRC20 (~$2.5) → Bybit/OKX → P2P 卖 CNY

场景B：USDT → 海外法币
USDT → 存入 Kraken ($2.5) → 卖 USDT 换 USD (0.16%) → SWIFT 出金 ($35)
注：SWIFT 固定 $35 费用，金额越大越合算（$5000 以上才合理）
```

**重要限制：**
- 中国大陆居民开户需使用海外手机号/地址（监管限制）
- Tier 2 KYC 需提供资金来源证明（大额建议准备）

## P2P 交易机制详解 {#p2p}

### 工作原理
P2P 交易所扮演**托管中介**角色：
1. 卖家将 USDT 锁定在交易所托管账户
2. 买家向卖家银行账户转账人民币
3. 买家确认付款 → 交易所释放 USDT
4. 全程无法抵赖（链上记录）

### P2P 价格构成
```
P2P USDT/CNY 价格 = 中间价 × (1 + P2P溢价)
溢价来源：
  - 商家盈利：0.3–0.5%
  - 风险补偿（银行冻卡风险）：0.3–0.5%
  - 市场供需：牛市溢价升至 2–3%，熊市可能倒挂
```

### 安全 P2P 实践
| 风险 | 防范措施 |
|------|----------|
| 银行冻卡 | 使用专用银行卡；避免来路不明资金混入 |
| 商家跑路 | 只选认证商家（≥500笔，信誉≥98%）|
| 汇率损失 | 大额分批成交；对比 OKX 和 Bybit 价格 |
| 洗钱误判 | 转账备注填写合理用途；保留聊天记录截图 |

### 哪种货币对效率最高？
- **USDT-TRC20** 最推荐：低手续费（~$1），5分钟确认，流动性最强
- **USDT-ERC20** 不推荐：Gas 费高（$5–50），慢
- **BTC/ETH** 不适合作为价值转移工具（波动大，手续费高）

## 虚拟货币线下/实体消费 {#offline}

### 场景1：USDT 换取人民币现金（离线消费）
```
路径：USDT → OKX/Bybit P2P 卖出 → 微信/支付宝收款 → 扫码消费
特点：准实时到账；全程数字；需手机网络
```

### 场景2：加密货币直接线下支付（境外）
- **Crypto.com Visa 卡**：USDT/BTC 充值，境外刷卡消费（汇率约中间价 +1%）
- **Binance Card（部分地区）**：与 Crypto.com 类似
- **Bitpay 商户**：部分境外商家直接接受 BTC/USDT
- **限制**：中国大陆境内几乎无商家直接接受加密货币

### 场景3：USDT 预付卡（境外消费）
- 购买 Gift Card 类产品（Amazon、Steam 等）可通过加密货币购买平台兑换
- 部分 VISA 预付卡服务商支持 USDT 充值（Revolut/Wirex 等）

### 场景4：海外购物返回中国
```
路径：境外电商（USD结算）→ 用 Wise/N26 扣款 → 转运寄回国内
或：朋友代购 → Wise 转账 USD 给朋友 → 朋友换人民币
```
