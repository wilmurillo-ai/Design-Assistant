## Wise {#wise}

| 项目 | 说明 |
|------|------|
| 类型 | 英国持牌电子货币机构（FCA 监管） |
| 手续费 | 约 0.35%–2%（按货币对浮动） |
| 汇率 | 实时中间价，**无额外点差** |
| 速度 | 多数主流货币 0–24 小时 |
| KYC | 护照/ID 证件 + 地址证明 |
| 支持货币 | 50+ 种（含 EUR, USD, GBP, HKD, AUD, JPY, SGD） |
| 中文人民币 | ❌ 无法从中国大陆直接出金；可通过香港中信 CNH 账户入金 |
| 适合场景 | 海外→国内（经 HK 中转），或纯境外货币互转 |

**具体手续费（2025参考）：**
- USD→EUR: ~0.43%
- GBP→USD: ~0.35%
- EUR→CNH (HK): ~0.6%
- USD→CNH (HK): ~0.55%

**开户流程：**
1. App 注册 → 上传证件 → 验证地址
2. 人民币收款需添加 HK 本地账户（Sort code）
3. 首次汇款需小额验证

**局限：**
- 中国大陆居民无法直接绑定国内银行卡出金
- 部分货币对有每日/每月限额

## N26 {#n26}

| 项目 | 说明 |
|------|------|
| 类型 | 德国银行执照（BaFin 监管） |
| 手续费 | SEPA 区内免费；国际汇款通过 Wise 完成（Wise 收费） |
| 汇率 | Wise 中间价（国际）；ECB 汇率（欧元区） |
| 速度 | SEPA 即日；国际 1–3 天 |
| KYC | 欧盟居民优先；需视频验证 |
| 支持货币 | EUR 主要；通过 Wise 拓展 |
| 中文人民币 | ❌ 不支持 |
| 适合场景 | 欧元区收款，EUR↔USD 中转 |

**注：** N26 不直接支持中文用户开户（需欧盟地址）。

## iFAST Financial {#ifast}

| 项目 | 说明 |
|------|------|
| 类型 | 新加坡 MAS 持牌证券/基金平台，附带银行账户（iFAST Bank） |
| 手续费 | 出境 SWIFT: SGD 10 固定费 + 约 0.1% |
| 汇率 | 约中间价 +0.5% 点差 |
| 速度 | 1–3 个工作日 |
| KYC | 新加坡/香港居民；护照 + 证明文件 |
| 支持货币 | USD, EUR, HKD, SGD, CNH（离岸人民币） |
| 中文人民币 | ✅ 支持 CNH（离岸人民币），不支持 CNY（在岸） |
| 适合场景 | 新加坡/香港居民中转；CNH↔其他货币 |

**与 CNY 的关系：**
- CNH ≠ CNY：iFAST 处理的是离岸人民币，需先将 CNY 通过正规渠道转换为 CNH（如通过中信香港）

## 中信国际银行（香港）{#citic}

| 项目 | 说明 |
|------|------|
| 类型 | 香港持牌银行（HKMA 监管），中信集团旗下 |
| 手续费 | 汇款手续费约 0.1%，最低 HKD 100 |
| 汇率 | 约中间价 +1.5% 点差（CNY↔HKD 较优） |
| 速度 | 港澳通即日；SWIFT 国际 1–2 个工作日 |
| KYC | 内地居民：护照 + 港澳通行证 + 大陆银行流水证明 |
| 支持货币 | CNY, HKD, USD, EUR 等主流 |
| 中文人民币 | ✅ **CNY↔HKD 核心优势，桥接内地与境外** |
| 适合场景 | 内地→香港→境外；最合规的 CNY 出境通道之一 |

**开户要求：**
- 亲赴香港分行（部分支持预约）
- 内地居民需存款证明；开户门槛约 HKD 10,000–50,000
- 港卡可绑定境外 Wise/iFAST 账户

**跨境人民币业务：**
- 个人每日 CNY↔HKD 上限：约 CNY 80,000（香港政策）
- 可作为大额出金的分批通道

## 中国境内银行 SWIFT 汇款 {#domestic-swift}

主要银行：中国银行（BOC）、工商银行（ICBC）、建设银行（CCB）、招商银行（CMB）

| 项目 | 说明 |
|------|------|
| 汇出手续费 | 约 CNY 150–300（电报费）+ 约 0.1% |
| 汇率 | 银行挂牌汇率，约中间价 +1.5–2% |
| 收款方中间行费用 | USD 15–35（由发送方承担或到账方扣除） |
| 速度 | 2–5 个工作日 |
| 年度限额 | **每人每年 $50,000 USD 等值**（外汇管理局规定） |
| 申报要求 | 需填写汇款用途（教育/旅游/商务等） |
| 大额申报 | 单笔 $10,000+ 需额外文件 |
| 中文人民币 | ✅ 直接从 CNY 账户操作 |

**最佳实践：**
- 使用"中国银行"：国际汇款网络最广、中间行最少
- 选择 BEN 模式（收款方承担费用）避免国内多扣
- 提前核实收款行 SWIFT 代码（错误无法退款）

**年度额度管理：**
- $50,000 年度额度按自然年重置（1月1日）
- 家庭成员名义可各自使用额度（需实际本人操作）
- 超额需通过企业账户或其他合规渠道

## 银联卡境外使用 {#unionpay}

| 项目 | 说明 |
|------|------|
| 境外消费 | 银联汇率，约中间价 +1.5% |
| 境外取现 | ATM 手续费约 1–3%（当地 ATM 可能额外收费） |
| 取现年度限额 | 等值 $10,000 USD/年（累计） |
| 消费限额 | 无官方限制（但银行可能设单日上限） |
| 速度 | 即时（消费）；1–3 天（账单结算） |
| 适合场景 | 短期出境消费；小额日常支出 |

## 常见监管问题 {#regulations}

**中国外汇管制要点：**
- 个人年度购汇额度：$50,000 USD 等值（含所有用途）
- 境外汇款需申报用途（可选：留学/医疗/旅行/商务/移民/家庭赡养）
- 单笔 $5,000 USD 以上需填写对外付款申请书
- 向个人汇款需提供受益人关系证明（非贸易类）

**香港跨境人民币：**
- 个人每日 CNY↔HKD 互换上限：CNY 80,000
- 在港个人人民币存款可自由汇出（无额度限制，但需 KYC）

**常见合规路径顺序（由高到低）：**
1. 国内银行 SWIFT 汇款（完全合规，有额度限制）
2. 中信香港 + Wise（合规，需开设香港账户）
3. 境外身份直接 Wise/iFAST（合规，需境外居留资格）
4. P2P 加密（法律灰色，存在银行账户冻结风险）
