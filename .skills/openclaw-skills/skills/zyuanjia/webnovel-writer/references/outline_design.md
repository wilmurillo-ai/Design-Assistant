# 大纲设计方法论

## 一、从零开始设计大纲

### 第1步：确立核心卖点
- 一句话概括故事（不超过30字）
- 核心冲突是什么
- 为什么读者会追下去（悬念/爽感/情感）

### 第2步：确定类型和风格
- 类型：都市/奇幻/悬疑/言情/玄幻/科幻...
- 风格：轻松/沉重/热血/催泪/恐怖...
- 平台适配：番茄小说风格（快节奏、短句、高密度钩子）

### 第3步：设计核心设定
- 主角：身份、性格、核心困境、成长弧线
- 核心机制：系统/金手指/特殊能力/核心悬念
- 世界观：背景规则、限制条件
- 核心冲突：主角vs什么（人/制度/命运/自己）

### 第4步：分卷规划
- 每卷主题（这一卷解决什么问题）
- 每卷关键事件（3-5个转折点）
- 每卷结尾高潮（让读者想继续看下一卷）
- 卷与卷之间的升级逻辑（难度递增、格局扩大）

### 第5步：角色设定
- 主角档案（背景、动机、性格、说话风格、弱点）
- 核心配角（每人的功能定位、与主角的关系弧线）
- 反派/对手（动机要合理，不能脸谱化）
- 角色关系图

### 第6步：章节列表
- 每章标题 + 一句话概述
- 标注关键节点（转折、高潮、揭秘）
- 控制每章节奏（不能连续几章都是铺垫）

## 二、分卷设计模板

```markdown
## 第X卷：卷名

- **主题：** 本卷核心主题
- **起因：** 什么事件开启了本卷
- **经过：** 3-5个关键事件
- **高潮：** 本卷最大冲突/揭秘
- **结局：** 本卷如何收尾，留什么悬念给下一卷
- **角色变化：** 主角在本卷的成长/变化
- **章节数：** XX-XX章（约XX章）

### 章节列表
| 章节 | 标题 | 核心事件 | 关键度 |
|------|------|----------|--------|
| XX1 | 标题 | 一句话描述 | ★/★★/★★★ |
```

## 三、大纲调整原则

1. **大纲是活的** — 写作中发现更好的方向可以调整
2. **调整要双向同步** — 改了大纲要更新追踪表，发现正文好想法要补进大纲
3. **重大调整要记录** — 为什么改、改了什么、影响哪些后续章节
4. **不轻易推翻核心设定** — 核心冲突和主角弧线是地基，可以改装修不能改地基

## 四、节奏设计

### 单章节奏
```
钩子开头（抓注意力）
→ 铺垫/发展（推进情节）
→ 小高潮/反转（中段爆点）
→ 收束 + 悬念结尾（让读者点下一章）
```

### 单卷节奏
```
开篇（1-2章）— 建立本卷背景和悬念
发展（占60%）— 逐层推进，每3-5章一个小高潮
高潮（2-3章）— 本卷最大冲突
收尾（1章）— 解决+留下一卷钩子
```

### 全书节奏
- 前30章：必须让读者入坑（核心机制展示+第一个大高潮）
- 中段：持续升级，不能水（每卷都要有推进）
- 后期：收线，前面埋的伏笔逐个引爆
