# Skill 使用示例文档

本文档给出 novel-writer skill 的4个典型使用场景，包含具体命令和预期输出示例。

---

## 场景1：开始写新章节前，检查一致性

**何时使用：** 每章写完后或写下一章前，确保正文没有设定矛盾、时间线错误、AI高频词等问题。

**命令：**

```bash
python3 scripts/consistency_check.py 正文/ --tracking 追踪表.md --dict 设定词典.md --character-states character_states.json
```

**预期输出示例：**

```
正在扫描正文目录: 正文/
加载追踪表: 追踪表.md
加载设定词典: 设定词典.md
加载角色状态: character_states.json

扫描完成: 38 个文件

===== 一致性检查报告 =====

[A类-设定一致性] ✅ 通过
[B类-时间线] ⚠️ 发现 1 个问题
  - 第025章: "三天后" 与 追踪表记录的时间线偏差1天（预期: 第14天, 实际: 第15天）

[C类-逻辑一致性] ✅ 通过
[D类-风格质量] ⚠️ 发现 3 个问题
  - 第023章: AI高频词 "不禁" 出现 2 次
  - 第024章: AI高频词 "微微" 出现 3 次
  - 第025章: 对话超过150字无打断/动作，建议拆分

[E类-大纲合规] ✅ 通过
[角色状态机] ✅ 通过
[角色消失] ⚠️ "林小美" 已连续 6 章未出场
[跨章重复] ✅ 未发现重复段落
[章节字数] ✅ 所有章节在 2000-3000 字范围内

===== 总结 =====
通过: 8/11  |  警告: 3  |  错误: 0
建议修正上述 3 个警告后再继续写作。
```

**解读：** 检查通过率较高，需修正时间线偏差、替换AI高频词、拆分过长对话。修正后可安心写下一章。

## 场景2：写完章节后，用 rhythm_check 检查节奏

**何时使用：** 批量审校阶段，或感觉最近几章"读起来太平""节奏不对"时。

**命令：**

```bash
python3 scripts/rhythm_check.py 正文/ --suggest --emotion-track
```

**预期输出示例：**

```
节奏检测: 正文/ (38 章)

===== 章节紧张度 =====
第021章: ★★☆☆☆ (2/5) 日常过渡
第022章: ★★★☆☆ (3/5) 信息铺垫
第023章: ★★★★☆ (4/5) 矛盾升级
第024章: ★★★★★ (5/5) 高潮冲突
第025章: ★★☆☆☆ (2/5) 冷却消化
第026章: ★★★☆☆ (3/5) 新线索
第027章: ★★★☆☆ (3/5) 调查推进
第028章: ★★★☆☆ (3/5) 调查推进  ← ⚠️ 连续3章紧张度相同

===== 钩子密度 =====
第021章: ✅ 章末钩子 (悬念型)
第022章: ✅ 章末钩子 (信息型)
第023章: ❌ 缺少500字内钩子

第027章: ✅ 章末钩子 (反转型)
第028章: ✅ 章末钩子 (悬念型)

===== 情绪曲线 =====
平静 ████████░░
紧张 ██████░░░░
悲伤 ████░░░░░░
愤怒 ███░░░░░░░
好奇 █████████░  ← 好奇情绪占比偏高

===== 问题摘要 =====
1. ⚠️ 第026-028章紧张度连续3章为3/5，节奏单调，建议插入一个高/低谷
2. ⚠️ 第023章缺少中段钩子

  第027章: 提升至4/5（发现关键证据，危机感）
  第028章: 4/5→5/5过渡（逼近真相，章末反转）
```

**解读：** 发现连续3章节奏平坦的问题，按推荐模板调整即可恢复张力起伏。

## 场景3：用 tension_forecast 规划接下来的章节

**何时使用：** 大纲规划阶段或中期节奏调整，想知道接下来几章的张力应该怎么排。

**命令：**

```bash
python3 scripts/tension_forecast.py 正文/ --count 7 --outline 大纲.md --character-states character_states.json
```

**预期输出示例：**

```
张力预测: 正文/ (基于已写 38 章)

===== 已写章节张力走势 =====
[021] ██  [022] ███  [023] ████  [024] █████  [025] ██  
[026] ███  [027] ███  [028] ███

===== 未来7章张力预测 =====
第029章: █████ (5/5) 高潮  ← 大纲第29章为"真相揭露"
第030章: ████  (4/5) 余波  
第031章: ███   (3/5) 新篇铺垫
第032章: ████  (4/5) 小高潮 ← 角色弧线转折点（林远山信念动摇）
第033章: ██    (2/5) 情感沉淀
第034章: ███   (3/5) 蓄势
第035章: █████ (5/5) 大高潮 ← 大纲第35章为"终极对决"

===== 弃书风险预判 =====
第030-031章: ⚠️ 高潮后低谷连续2章，建议第031章加入小悬念维持兴趣
第033章: ⚠️ 情感沉淀章容易拖节奏，建议控制在2000字以内

===== 建议写作节奏 =====
29(爆) → 30(收) → 31(钩) → 32(升) → 33(沉) → 34(蓄) → 35(爆)
```

**解读：** 预测了未来7章的张力走势，并标记了2个弃书风险点。写第031章和第033章时需特别注意节奏控制。

## 场景4：用 outline_drift 检查是否偏离大纲

**何时使用：** 写了较长一段后，怀疑正文可能偏离大纲规划时。

**命令：**

```bash
python3 scripts/outline_drift.py --outline 大纲.md --chapters 正文/ --fix-suggest
```

**预期输出示例：**

```
大纲偏差检测: 正文/ vs 大纲.md

===== 标题匹配度 =====
匹配: 32/38 (84.2%)
偏离章节:
  - 第019章: 大纲标题"林远山的抉择" → 正文标题"深夜对话" (关键词偏差)
  - 第023章: 大纲标题"线索浮现" → 正文标题"意外的发现" (关键词偏差)
  - 第026章: 大纲标题"内鬼现身" → 正文标题"新的方向" (关键词偏差)

===== 关键词覆盖率 =====
第019章: 60%  ⚠️ 缺少大纲关键词: "抉择"、"挣扎"
第023章: 75%  ✅
第026章: 45%  ❌ 严重偏离，缺少: "内鬼"、"暴露"、"对峙"

===== 角色出场偏差 =====
大纲预期但未出场:
  - 第020章: "赵局长" 应出场但未出现
  - 第026章: "老钱" 应出场但未出现

正文出现但大纲未规划:
  - 第022章: "小王" 额外出场（不影响主线，可保留）

===== 综合偏差评分 =====
第019章: 3.2/10 (轻微偏离)
第023章: 2.1/10 (基本符合)
第026章: 7.8/10 ⚠️ 严重偏离

===== 修复建议 =====
[第026章 - 严重偏离]
  问题: 大纲要求"内鬼现身"，正文改为"新的方向"，核心剧情未展开
  建议: 
    1. 在第026章末尾增加内鬼线索暗示（不完全揭露，留悬念）
    2. 或在第027章尽早补上"内鬼"剧情线
    3. 如有意调整大纲，请在大纲.md中同步修改第026章规划并记录修改日志

[第019章 - 轻微偏离]  
  问题: 缺少"抉择"主题关键词
  建议: 在第019章对话中增加林远山面临两难选择的内心刻画
```

**解读：** 第026章严重偏离大纲，需要补回"内鬼"剧情线或同步修改大纲。第019章轻微偏离，补充抉择主题即可。

---

## 小结

| 场景 | 命令 | 何时用 |
|------|------|--------|
| 一致性检查 | `consistency_check.py` | 每章写完后 |
| 节奏检测 | `rhythm_check.py --suggest` | 感觉节奏不对时 |
| 张力预测 | `tension_forecast.py --count N` | 规划未来章节时 |
| 大纲偏差 | `outline_drift.py --fix-suggest` | 怀疑偏离大纲时 |
