# 章节质量检查清单

每章写完或审稿时逐项检查。

## A类：设定一致性

- [ ] A1 角色名字/称呼是否与设定集一致
- [ ] A2 角色年龄、身份、职业是否与设定一致
- [ ] A3 角色外貌描写是否与设定一致（发型、体型、特征）
- [ ] A4 角色能力/技能是否与当前进度匹配（不能提前展现）
- [ ] A5 角色关系是否正确（谁认识谁、亲疏程度）
- [ ] A6 公司/组织/地点名称是否统一
- [ ] A7 专有名词是否与设定词典一致
- [ ] A8 数值是否正确（金额、人数、距离、时间长度）

## B类：时间线一致性

- [ ] B1 故事内日期/天数是否与追踪表一致
- [ ] B2 事件先后顺序是否正确
- [ ] B3 角色行动时间是否合理（不可能同时出现在两个地方）
- [ ] B4 季节/天气是否与时间线匹配
- [ ] B5 提到的历史事件是否与世界观设定一致

## C类：逻辑一致性

- [ ] C1 角色是否知道他应该知道的信息（不泄露未来剧情）
- [ ] C2 角色是否不知道他不应该知道的信息
- [ ] C3 物品持有是否正确（不能凭空出现或消失）
- [ ] C4 因果关系是否成立（结果必须有合理原因）
- [ ] C5 角色动机是否充分（关键行为必须有铺垫）

## D类：风格质量（去AI味）

- [ ] D1 是否避免了AI高频词（不禁、缓缓、淡淡、微微、竟然、居然）
- [ ] D2 外貌描写是否与前面章节重复（不能每章用同一套描写）
- [ ] D3 章末是否用了不同的收尾方式（不能每章都是悬念结尾）
- [ ] D4 对话是否自然口语化（不能书面腔）
- [ ] D5 弹幕/群聊发言顺序是否有变化（不能每章固定顺序）
- [ ] D6 是否避免了每章相同结构（开头环境→对话→结尾）
- [ ] D7 情绪是否靠行为和细节传达（不靠形容词堆叠）
- [ ] D8 是否有足够的钩子密度（每500字一个）

## E类：大纲合规

- [ ] E1 本章内容是否在大纲规划范围内
- [ ] E2 是否提前泄露了后续章节的重大信息
- [ ] E3 是否遗漏了本章应该出现的关键事件
- [ ] E4 角色行为是否符合当前阶段的性格发展
- [ ] E5 结束点是否与写作卡一致
