# Unicode Font Character Maps

Reference file for the text-repeater skill's font conversion feature.
Each table maps standard ASCII characters to their Unicode styled equivalents.

---

## Bold (Mathematical Bold)

```
A→𝐀 B→𝐁 C→𝐂 D→𝐃 E→𝐄 F→𝐅 G→𝐆 H→𝐇 I→𝐈 J→𝐉 K→𝐊 L→𝐋 M→𝐌
N→𝐍 O→𝐎 P→𝐏 Q→𝐐 R→𝐑 S→𝐒 T→𝐓 U→𝐔 V→𝐕 W→𝐖 X→𝐗 Y→𝐘 Z→𝐙
a→𝐚 b→𝐛 c→𝐜 d→𝐝 e→𝐞 f→𝐟 g→𝐠 h→𝐡 i→𝐢 j→𝐣 k→𝐤 l→𝐥 m→𝐦
n→𝐧 o→𝐨 p→𝐩 q→𝐪 r→𝐫 s→𝐬 t→𝐭 u→𝐮 v→𝐯 w→𝐰 x→𝐱 y→𝐲 z→𝐳
0→𝟎 1→𝟏 2→𝟐 3→𝟑 4→𝟒 5→𝟓 6→𝟔 7→𝟕 8→𝟖 9→𝟗
```

Unicode offset: Uppercase A=U+1D400, Lowercase a=U+1D41A, Digits 0=U+1D7CE

## Double Struck (Blackboard Bold)

```
A→𝔸 B→𝔹 C→ℂ D→𝔻 E→𝔼 F→𝔽 G→𝔾 H→ℍ I→𝕀 J→𝕁 K→𝕂 L→𝕃 M→𝕄
N→ℕ O→𝕆 P→ℙ Q→ℚ R→ℝ S→𝕊 T→𝕋 U→𝕌 V→𝕍 W→𝕎 X→𝕏 Y→𝕐 Z→ℤ
a→𝕒 b→𝕓 c→𝕔 d→𝕕 e→𝕖 f→𝕗 g→𝕘 h→𝕙 i→𝕚 j→𝕛 k→𝕜 l→𝕝 m→𝕞
n→𝕟 o→𝕠 p→𝕡 q→𝕢 r→𝕣 s→𝕤 t→𝕥 u→𝕦 v→𝕧 w→𝕨 x→𝕩 y→𝕪 z→𝕫
0→𝟘 1→𝟙 2→𝟚 3→𝟛 4→𝟜 5→𝟝 6→𝟞 7→𝟟 8→𝟠 9→𝟡
```

Note: C, H, N, P, Q, R, Z use dedicated codepoints (ℂ ℍ ℕ ℙ ℚ ℝ ℤ).

## Small Caps

Small caps uses a mix of dedicated Unicode letters and standard characters.
Lowercase letters map to small capital variants; uppercase stays standard.

```
a→ᴀ b→ʙ c→ᴄ d→ᴅ e→ᴇ f→ꜰ g→ɢ h→ʜ i→ɪ j→ᴊ k→ᴋ l→ʟ m→ᴍ
n→ɴ o→ᴏ p→ᴘ q→q r→ʀ s→s t→ᴛ u→ᴜ v→ᴠ w→ᴡ x→x y→ʏ z→ᴢ
```

Note: q, s, x pass through unchanged (no small cap equivalent).
Uppercase letters remain unchanged in Small Caps style.

```
A→𝓐 B→𝓑 C→𝓒 D→𝓓 E→𝓔 F→𝓕 G→𝓖 H→𝓗 I→𝓘 J→𝓙 K→𝓚 L→𝓛 M→𝓜
```

Unicode offset: Uppercase A=U+1D4D0, Lowercase a=U+1D4EA

## Conversion Algorithm

When converting text to a Unicode style:

1. Iterate over each character in the input string.
2. If the character is an ASCII letter (A-Z or a-z), look up its Unicode replacement from the table above.
3. If the character is a digit (0-9) and the style supports digit mapping, replace it.
4. All other characters (spaces, punctuation, emoji, non-Latin scripts) pass through unchanged.
5. Concatenate all characters to produce the styled output.

### Python reference implementation

```python
def to_bold(text):
    result = []
    for ch in text:
        if 'A' <= ch <= 'Z':
            result.append(chr(0x1D400 + ord(ch) - ord('A')))
        elif 'a' <= ch <= 'z':
            result.append(chr(0x1D41A + ord(ch) - ord('a')))
        elif '0' <= ch <= '9':
            result.append(chr(0x1D7CE + ord(ch) - ord('0')))
        else:
            result.append(ch)
    return ''.join(result)
```

Apply the same pattern for other styles by changing the Unicode offsets.
