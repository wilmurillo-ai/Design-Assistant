## 快速开始

### 第一步：配置企业微信 Webhook

1. 打开企业微信群聊
2. 点击群设置 → 群机器人 → 添加机器人
3. 复制 Webhook 地址
4. 发送命令：
   ```
   配置webhook=https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=你的key
   ```

### 第二步：配置抖音 Cookie（可选，提高价格准确性）

1. 用浏览器打开 [抖音商城](https://haohuo.jinritemai.com/)
2. 登录账号
3. 按 F12 打开开发者工具 → Network → 任意请求 → Headers → Cookie
4. 复制完整 Cookie 值
5. 发送命令：
   ```
   配置cookie=你的完整Cookie
   ```

### 第三步：开始监控

```
监控抖音商品 https://v.douyin.com/xxx/ 低于100元告警
```
