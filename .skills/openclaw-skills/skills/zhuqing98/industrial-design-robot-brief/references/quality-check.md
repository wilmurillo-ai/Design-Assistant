# Quality Check

Run this checklist before finalizing each daily brief.

## Per-Paper Checks

For each of the `5` papers, verify:

- [ ] arXiv ID or DOI is real and clickable
- [ ] English title exactly matches the source
- [ ] Chinese title is design-friendly, not a stiff word-for-word translation
- [ ] first author and affiliation are verifiable or clearly marked uncertain
- [ ] publication date is correct
- [ ] all numeric claims trace back to the paper text
- [ ] uncertain numbers are marked `待核实` or `论文未公开该数据`
- [ ] `设计师必读` contains at least `3` specific points
- [ ] each design insight points to a body part, surface, layout zone, or interaction behavior
- [ ] no design insight is so generic that it fits any robotics paper
- [ ] all `4` score dimensions are filled with one-line justifications
- [ ] links section is complete or explicitly marked `未找到` / `未开源`
- [ ] `产品定义一句话` is present

## Daily-Level Checks

- [ ] exactly `5` papers, or fewer with an explicit note
- [ ] papers cover at least `3` different design-relevant topic areas
- [ ] papers are ordered by design impact score, highest first
- [ ] editorial summary is present
- [ ] `5` keywords are listed
- [ ] no two papers repeat the exact same narrow point without justification
- [ ] date and weekday in the header are correct
- [ ] today's section is inserted directly below the pinned header
- [ ] older daily sections remain intact

## Anti-Hallucination Checks

- [ ] no fake arXiv ID
- [ ] no invented author names
- [ ] no guessed institution without qualifier
- [ ] no extrapolated performance number
- [ ] if browsing was unavailable, this is made clear and only verified materials are used

## Weekly Checks

- [ ] weekly trend summary is appended when needed
- [ ] high-frequency directions are counted accurately
- [ ] the `3 things designers should know` are not repetitive filler
