# Template

Use this template when generating daily content for `industrial-design-robot-brief`.

## Daily Header

```md
# 🤖 Robot Paper Daily Brief｜[YYYY.MM.DD] [星期X]

> **今日编辑按语**
>
> [3-5 句话总结今天的 5 篇论文整体趋势。]
> - 今天最值得工业设计师关注的是哪一篇？为什么？
> - 今天的论文整体偏向哪个方向？
> - 有没有新的 UX、信任或产品定义信号？
>
> **今日关键词：** `关键词1` `关键词2` `关键词3` `关键词4` `关键词5`
>
> **设计影响力最高分：** Paper [NN]（[分数]/20）— [一句话原因]
```

## Per-Paper Structure

Use this exact order:

```md
## 📄 Paper [NN]｜[YYYY.MM.DD]

### [中文标题]

**原文标题：** [exact English title]
**arXiv 链接：** [exact URL, or `arXiv ID 未确认`]
**机构：** [verified affiliation, or `作者信息待核实`]
**发布日期：** [YYYY-MM-DD]
**实验平台：** [robot name + key specs, or `纯仿真` or `未公开`]

### ⚡ 一句话人话
[不超过 40 个中文字。尽量用“这篇论文让机器人能___”的句式。]

### 🔍 核心内容（3 分钟读完）

**解决什么问题：**
> [2-3 句话，日常语言]

**怎么解决的：**
1. **感知层：** [...]
2. **决策层：** [...]
3. **执行层：** [...]

**关键数据：**
- [指标1]：[数值 + 对比基准]
- [指标2]：[数值 + 对比基准]
- [指标3]：[数值 + 对比基准]
- 实验平台：[机器人名称]，[关键信息]

### 🎨 设计师必读

#### 1. [具体设计影响点]
[论文事实]
→ **设计影响：** [具体影响]

#### 2. [具体设计影响点]
[论文事实]
→ **设计影响：** [具体影响]

#### 3. [具体设计影响点]
[论文事实]
→ **设计影响：** [具体影响]

### 🧭 产品定义一句话
[一句话说明它更像什么产品、适合什么场景或角色。]

### 📊 设计影响力评分

| 维度 | 评分 | 说明 |
|------|------|------|
| 外观形态 | ★×N | [一句话理由] |
| 硬件布局 | ★×N | [一句话理由] |
| 用户体验与信任 | ★×N | [一句话理由] |
| 产品定义 | ★×N | [一句话理由] |
| **总分** | **N/20** | |

### 🔗 延伸资料
- 论文 PDF：[link]
- 演示视频：[link, or `未找到`]
- 代码仓库：[link, or `未开源`]
- 项目主页：[link, or `未找到`]
```
