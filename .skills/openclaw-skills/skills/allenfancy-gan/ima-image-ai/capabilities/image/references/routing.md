# Image Routing

- no input image -> `text_to_image`
- one or more input images -> `image_to_image`
- reference or continuity language without image input -> clarify before execution

Examples:

- “画一张海报” -> `text_to_image`
- “把这张图改成油画风格” + image -> `image_to_image`
- “继续上一张那个角色” without image -> clarification
