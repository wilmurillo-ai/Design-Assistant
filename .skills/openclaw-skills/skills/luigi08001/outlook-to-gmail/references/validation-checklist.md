# Post-Migration Validation Checklist

Per-user checklist. Copy for each migrated user.

## User: _______________  |  Date: _______________

### Email
- [ ] Can receive new emails on Gmail (test from external address)
- [ ] Can send emails from Gmail (check From: address is correct)
- [ ] Historical emails present — spot check:
  - [ ] Oldest email visible
  - [ ] Email from 6 months ago visible
  - [ ] Recent email (last week) visible
- [ ] Folder structure → Labels mapping correct
- [ ] Attachments accessible on migrated emails
- [ ] No items >25MB missing (check source for large items)

### Contacts
- [ ] Contacts imported to Google Contacts
- [ ] Contact count matches source (±5% acceptable)
- [ ] Contact groups / distribution lists recreated
- [ ] No duplicate contacts (run Google merge if needed)

### Calendar
- [ ] Past events visible
- [ ] Future events present
- [ ] Recurring events intact (check weekly + monthly patterns)
- [ ] Shared calendars re-shared
- [ ] Meeting room bookings transferred (if applicable)

### Configuration
- [ ] Email signature set up
- [ ] Vacation responder configured (if needed)
- [ ] Filters / rules recreated (manual — no auto migration)
- [ ] Delegate access re-configured

### Devices
- [ ] Desktop: Gmail web or Outlook reconfigured
- [ ] Mobile: Gmail app or mail app reconfigured
- [ ] Tablet (if applicable)

### Sign-off
- [ ] User confirms migration complete
- [ ] User trained on Gmail basics (if needed)
