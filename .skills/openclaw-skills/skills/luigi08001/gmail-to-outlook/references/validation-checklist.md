# Post-Migration Validation Checklist

## Email Functionality

- [ ] **Test inbound email** — Send from external account, verify receipt in Outlook
- [ ] **Test outbound email** — Send from Outlook to external account, verify delivery
- [ ] **Historical email present** — Check oldest and newest emails migrated
- [ ] **Email count validation** — Spot-check folder counts vs. original Gmail
- [ ] **Search functionality** — Test searching for old emails
- [ ] **Attachments intact** — Open email with attachment, verify it downloads

## Folder/Label Structure

- [ ] **Inbox migrated** — All Gmail inbox items present
- [ ] **Sent items migrated** — Sent folder populated from Gmail
- [ ] **Custom labels → folders** — Gmail labels converted to Outlook folders
- [ ] **Nested folder structure** — Multi-level Gmail labels properly nested
- [ ] **Spam/Junk folder** — Junk email migrated if applicable
- [ ] **Archive functionality** — Archive behavior configured

## Contacts

- [ ] **Contact count** — Compare total contacts: Gmail vs. Outlook
- [ ] **Contact details** — Spot-check names, emails, phone numbers
- [ ] **Contact groups** — Gmail contact groups migrated as Outlook categories
- [ ] **Profile photos** — Contact photos migrated (where supported)
- [ ] **Special contacts** — Check VIP contacts are properly flagged

## Calendar

- [ ] **Event count** — Compare calendar entries: Google vs. Outlook
- [ ] **Recurring events** — Weekly/monthly meetings working correctly
- [ ] **Meeting invitations** — Can send/receive calendar invites
- [ ] **Shared calendars** — Access to shared/team calendars
- [ ] **Time zones** — Events display in correct local time
- [ ] **Reminders** — Pop-up and email reminders functioning

## Client Configuration

### Desktop Outlook
- [ ] **Account added** — Microsoft 365 account configured
- [ ] **Sync working** — Email/calendar/contacts syncing
- [ ] **Signature** — Email signature configured
- [ ] **Rules** — Critical Outlook rules recreated from Gmail filters
- [ ] **Archive settings** — Auto-archive configured as needed

### Mobile Device (iOS/Android)
- [ ] **Outlook app installed** — Microsoft Outlook app
- [ ] **Account configured** — Signed in with Microsoft 365 account
- [ ] **Push notifications** — Email notifications working
- [ ] **Calendar sync** — Calendar events showing in phone's calendar app
- [ ] **Contacts sync** — Contacts accessible from phone's contacts app

### Web Access
- [ ] **Outlook Web Access** — Can log in to outlook.office.com
- [ ] **All features working** — Email, calendar, contacts accessible online
- [ ] **Browser compatibility** — Works in user's preferred browser

## Security & Compliance

- [ ] **Two-factor authentication** — MFA enabled for Microsoft 365 account
- [ ] **App passwords** — Any IMAP clients updated with new credentials
- [ ] **Gmail access** — Gmail account secured or access removed as per policy
- [ ] **Shared mailbox access** — Permissions to shared mailboxes verified
- [ ] **Distribution lists** — Membership in relevant distribution lists
