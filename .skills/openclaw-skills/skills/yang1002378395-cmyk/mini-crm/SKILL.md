# Mini-CRM Skill

轻量级客户管理系统，个人/小团队适用。

## 功能

- 客户信息管理
- 跟进记录
- 成交追踪
- 数据分析
- 提醒通知

## 使用方式

```
添加新客户：张总，微信 zhang123，预算 ¥50,000
```

```
查看张总的跟进记录
```

```
生成本月客户报告
```
