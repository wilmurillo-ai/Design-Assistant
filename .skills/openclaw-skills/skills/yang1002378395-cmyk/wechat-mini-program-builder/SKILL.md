## 🚀 快速开始

### 安装
```bash
npm install -g miniprogram-cli
```

### 创建项目
```bash
mp-builder create --template shop --name my-shop
cd my-shop
mp-builder dev
```

### AI 生成页面
```bash
mp-builder generate --type page --name product-detail --prompt "商品详情页，展示图片、价格、规格选择、加入购物车按钮"
```
