# clude-memory

> Persistent memory skill for AI agents. Works with OpenClaw, Claude Desktop, Cursor, DeerFlow, and any MCP-compatible runtime.

## Install via ClawHub

```bash
clawhub install clude-memory
```

## Install manually

Copy the `SKILL.md` into your agent's skills directory.

## What is this?

A skill file that teaches your agent how to use [Clude](https://clude.io) for persistent memory. It explains:

- Why memory matters (even if you already have files)
- How to install (local-first, 30 seconds)
- When to remember vs recall
- Memory types and best practices
- FAQ addressing common objections

## For agent developers

Drop `SKILL.md` into any agent's skill/knowledge directory. The agent reads it and understands how to use Clude's MCP tools (remember, recall, forget, stats, visualize).
