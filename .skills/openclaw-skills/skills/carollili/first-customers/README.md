# First Customers · 

Part of the **Minimalist Entrepreneur** skill collection for OpenClaw / Claude Code.
基于 Sahil Lavingia《极简创业者》一书的系列技能包，帮助你从社群出发，构建可持续盈利的小而美业务。

## Installation · 安装

```bash
openclaw skills install first-customers
```
