# Presets

These are optional advanced overrides.

Most users should not need to type any of these. `humanize` already has a
built-in default direction for "make this Chinese message sound more human".
Use these only when you want to override that default.

## Customer Reply

```text
更像真人发给客户的微信，不要客服模板腔。礼貌、清楚、可信，别过度客套。
```

## Manager Update

```text
像项目负责人给上级汇报的即时消息。简洁、直接、可执行，不要官腔。
```

## Interview Follow-up

```text
像真实候选人的跟进消息。自然、得体、有分寸，不要像润色过度的模板。
```

## After-sales Reply

```text
像靠谱售后同事的回复。安抚情绪但不空泛，给出明确动作和时间点，不要模板化。
```
