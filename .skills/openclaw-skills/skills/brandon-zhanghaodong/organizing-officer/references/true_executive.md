# 真高管 (True Executive)

在首席组织官的语境下，“真高管”并非指职级高的人，而是指那些能够真正承担起组织责任、创造核心价值的领导者。

### 真高管的定义与衡量标准

真高管是指那些在所辖领域，对内可以领导一号位及高管团队，对外可以代表公司竞争力的核心人才。其核心特点是**职责/角色四合一**：

| 角色 | 核心职责 | 关键要求 |
| :--- | :--- | :--- |
| **操作者 (Operator)** | 在需要的时候，可以亲自做简单重复性的事。 | 保持对一线的敏感度，不脱离实际。 |
| **专业技术人士 (Professional)** | 承担规划设计相关的专业技术职责。 | 在所辖领域具备深厚的专业功底。 |
| **领导者 (Leader)** | 激发他人，塑造文化，指引方向。 | 具备感召力和前瞻性。 |
| **管理者 (Manager)** | 建立系统，优化流程，确保结果。 | 具备系统思维和执行力。 |

### 真高管的三大导向

1.  **责任导向**：不仅关注个人KPI，更关注组织的整体成败。
2.  **贡献导向**：以创造价值为荣，而非以拥有权力为荣。
3.  **价值创造导向**：致力于提升组织的长期竞争力，而非短期利益。

### 领导一号位

真高管的一个重要标志是“对内领导一号位”。这意味着高管不应仅仅是CEO的执行者，而应在自己的专业领域内成为CEO的顾问甚至导师，能够挑战CEO的决策并提供更优的组织策略。
