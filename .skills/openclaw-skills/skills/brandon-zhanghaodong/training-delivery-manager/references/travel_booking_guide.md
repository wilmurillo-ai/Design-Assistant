# 讲师差旅预订指南

本指南用于规范讲师差旅预订流程及标准，确保讲师出行顺利、体验良好。

## 预订标准

| 项目 | 标准 |
| :--- | :--- |
| 机票 | **经济舱** |
| 酒店 | **四星级或同等级别商务酒店** |
| 预算 | 在满足以上标准前提下，选择性价比高的选项 |

## 预订流程

1.  **信息确认**：
    *   与讲师确认姓名、身份证号、联系电话等个人信息。
    *   确认培训时间与地点，预留充足的在途时间。

2.  **机票预订**：
    *   首选直飞航班，避免中转。
    *   航班时间应在培训开始前一天抵达，培训结束后可于当天或次日返程。
    *   通过携程或其他指定平台进行预订。

3.  **酒店预订**：
    *   酒店应位于市中心或交通便利的区域，距离培训场地不宜过远。
    *   预订时确认酒店包含早餐、网络等基本服务。
    *   以讲师本人名义预订，并提前支付房费。

4.  **信息同步**：
    *   预订完成后，及时将机票、酒店的详细信息（航班号、时间、酒店地址、入住凭证等）发送给讲师。
    *   提醒讲师携带有效身份证件。

5.  **费用报销**：
    *   保留所有交通和住宿的电子发票或行程单，用于后续报销。
