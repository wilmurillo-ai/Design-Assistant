# 培训后宣传内容创作指南

本指南用于指导培训结束后宣传稿和海报的制作，旨在提升品牌形象和客户满意度。

## 宣传原则

- **及时性**：培训结束后 1-2 个工作日内发布。
- **客户导向**：内容和设计应体现客户公司的文化和元素。
- **价值聚焦**：突出培训为学员和企业带来的核心价值。

## 宣传稿 (推文)

宣传稿的结构应参考 `templates/promotional_article_template.md` 模板。核心要素包括：

1.  **一个吸引人的标题**：结合培训主题和客户名称，如“创效未来·人资护航”。
2.  **精彩的开篇**：概括培训的基本信息（时间、地点、主题、参与人员），并点明培训的核心目标。
3.  **核心内容提炼**：
    *   系统性地介绍培训的核心模块和知识点。
    *   可采用“精准识人”、“梯队建设”等小标题，使结构更清晰。
4.  **实战与成果导向**：
    *   强调课程的实战性，如案例分析、实操演练等。
    *   展示学员的成果，体现培训的价值。
5.  **升华与展望**：
    *   引用高层领导的讲话或期望，提升文章高度。
    *   以充满正能量的语句结尾，展望未来。

## 宣传海报

海报设计应参考 `templates/poster_reference.jpg` 的风格。关键设计元素：

- **主色调**：通常采用客户公司的品牌色或红色等积极、热烈的颜色。
- **核心标题**：醒目地展示培训主题，如“卓越引领，战略笃行”。
- **客户 Logo**：在海报的显著位置放置客户公司的 Logo。
- **核心信息**：简要介绍培训的核心内容和亮点。
- **现场照片**：精选 3-5 张高质量的现场照片，包括讲师授课、学员互动、团队展示、大合影等，展现积极的培训氛围。
- **设计元素**：可适当添加简约的几何图形或商务风格的背景，提升设计感。
