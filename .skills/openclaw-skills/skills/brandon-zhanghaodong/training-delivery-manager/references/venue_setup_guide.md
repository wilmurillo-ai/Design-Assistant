# 培训场地布置指南

本指南旨在规范培训场地的布置流程，为讲师和学员创造一个专业、舒适的培训环境。

## 布置原则

- **专业性**：场地布置应体现公司和培训项目的专业形象。
- **互动性**：鼓励学员间的互动和交流，座位安排不宜过于拥挤。
- **便利性**：确保所有物料摆放有序，方便讲师和学员取用。

## 布置清单与流程

1.  **主视觉区域**：
    *   [ ] **讲台/主席台**：确保讲台整洁，放置讲师桌牌和饮用水。
    *   [ ] **背景板/横幅**：在讲台后方的醒目位置悬挂培训主题横幅或背景板。
    *   [ ] **易拉宝/海报**：在会场入口或签到处放置欢迎海报或课程介绍易拉宝。

2.  **学员区域**：
    *   [ ] **座位安排**：根据培训形式（如U型、岛屿式）摆放桌椅，确保每位学员都有良好的视野。
    *   [ ] **物料摆放**：
        *   在每个座位上摆放**学员手册/课件**、**签字笔**和**座位卡**。
        *   确保电源插座可供需要使用电脑的学员使用。

3.  **互动与展示区域**：
    *   [ ] **白板/白板纸**：放置在教室前方或侧方，方便讲师和学员使用。旁边备好各色白板笔和板擦。
    *   [ ] **小组讨论区**：如空间允许，可设置专门的小组讨论区域，并放置白板纸。

4.  **签到与服务区域**：
    *   [ ] **签到处**：在会场入口设置签到台，摆放**签到表**和签字笔。
    *   [ ] **茶歇区**：如提供茶歇，请将茶点、饮品摆放整齐。

5.  **最终检查**：
    *   [ ] **音响与投影设备**：开课前与讲师一同测试麦克风、音响和投影仪，确保一切正常。
    *   [ ] **空调与照明**：调节好室温和灯光，创造舒适的环境。
    *   [ ] **清场**：检查场地是否整洁，无多余杂物。
