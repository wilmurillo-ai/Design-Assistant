# 培训物料准备清单

本清单用于确保所有培训所需物料准备齐全，保障培训顺利进行。

## A. 课件与文件类

- [ ] **培训课件**
    - [ ] 与讲师最终确认课件版本（PPT/PDF）。
    - [ ] 与客户确认课件封面设计（是否需要加入客户 Logo）。
    - [ ] 确认打印份数（通常为学员人数 + 讲师 + 备用 1-2 份）。
    - [ ] 完成课件的打印和装订。
- [ ] **签到表**
    - [ ] 准备印有公司 Logo 和培训名称的签到表。
- [ ] **满意度调查问卷**
    - [ ] 准备纸质或电子版问卷。

## B. 场地布置类

- [ ] **横幅**
    - [ ] 确认横幅内容、尺寸和材质，并完成制作。
- [ ] **海报/易拉宝**
    - [ ] 设计并制作欢迎海报或课程介绍易拉宝。
- [ ] **座位卡/桌牌**
    - [ ] 为学员和讲师制作座位卡。
- [ ] **分组牌**
    - [ ] 如有分组活动，提前准备分组标识牌。

## C. 文具与工具类

- [ ] **签字笔**
    - [ ] 为每位学员准备一支签字笔。
- [ ] **白板和白板笔**
    - [ ] 确认场地提供白板，并准备多种颜色的白板笔和板擦。
- [ ] **白板纸 (Flip Chart)**
    - [ ] 准备至少 2-3 本白板纸，供小组讨论和展示使用。
- [ ] **便利贴/N次贴**
    - [ ] 准备不同颜色和尺寸的便利贴。
- [ ] **激光笔/翻页器**
    - [ ] 为讲师准备。

## D. 其他物料

- [ ] **奖品/纪念品**
    - [ ] 为优秀学员或小组准备小礼品。
- [ ] **茶歇用品**
    - [ ] 确认茶歇安排，如需自行准备，请采购咖啡、茶叶、点心等。
- [ ] **摄影摄像设备**
    - [ ] 安排专人负责拍照或录像。
