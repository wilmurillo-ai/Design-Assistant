# 知识类型策略（预留扩展点）

此目录用于存放按知识类型定制的学习策略文件。

v1.0 使用通用策略（已内嵌在 SKILL.md 模板中的启发式规则），不按知识类型分支。

未来迭代中，可根据实际学习项目的反馈添加策略文件，例如：
- `technical.md` — 技术类知识的学习策略（侧重实践验证、代码练习）
- `conceptual.md` — 概念/理论类知识的学习策略（侧重深度理解、类比思维）
- `procedural.md` — 流程/方法类知识的学习策略（侧重步骤拆解、场景模拟）

策略文件与骨架模板（templates/）正交：骨架决定项目整体结构，策略注入学习模式内部的具体引导方式。
