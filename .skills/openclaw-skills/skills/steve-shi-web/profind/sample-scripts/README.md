# 脚本使用说明

此目录存放自定义脚本示例，可直接使用。

## 使用方法

```bash
# 复制到用户脚本目录即可生效
cp *.sh ~/Library/Scripts/ProFind/
```

## 脚本列表

### BatchRename.sh

批量重命名工具，将文件名中的空格替换为下划线。

```bash
./BatchRename.sh
# 或在 ProFind 中选中文件 → Scripts → BatchRename
```

### SendFilePathsViaMail.scpt

选中文件后，自动在新邮件正文中列出文件路径。

```bash
cp SendFilePathsViaMail.scpt ~/Library/Scripts/ProFind/
# 在 ProFind 中选中文件 → Scripts → SendFilePathsViaMail
```
