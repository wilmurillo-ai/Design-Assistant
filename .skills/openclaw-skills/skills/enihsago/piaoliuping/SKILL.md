# 漂流瓶

随机获取一个漂流瓶，其中包含人生感悟、小故事、祝福或励志语录。

## 使用方法

触发后，运行脚本随机选择并返回一个漂流瓶消息：

```bash
python3 scripts/get_bottle.py
```

脚本会随机返回一条预设的漂流瓶内容。

## 漂流瓶类型

漂流瓶可能包含：

- 人生感悟和智慧
- 小故事或趣闻
- 祝福和心愿
- 励志语录
- 思考与反思

## 扩展内容

要添加新的漂流瓶消息，编辑 `scripts/get_bottle.py` 并在 `messages` 列表中添加新条目。

每条消息应该简洁、有意义，并且能够独立成篇。
