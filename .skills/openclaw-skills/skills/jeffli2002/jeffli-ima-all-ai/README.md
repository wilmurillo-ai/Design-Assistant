# IMA All AI

一站式 IMA 创作技能：图像、视频、音乐、TTS 统一入口。

## GitHub 安装（推荐）

> 重要：安装 `ima-all-ai` 时，请务必同时安装 `ima-knowledge-ai`，用于模型选择与工作流优化。

```bash
# 先安装知识库（必装）
git clone https://github.com/imastuido/ima-knowledge-ai.git

# 再安装全能技能
git clone https://github.com/imastuido/ima-all-ai.git

# 可选：单独安装 TTS 增强
git clone https://github.com/imastuido/ima-tts-ai.git
```

## 为什么要同时安装 ima-knowledge-ai

- 复杂任务（MV、多镜头、跨媒体）需要工作流拆解。
- 视频任务需要区分 `image_to_video` 与 `reference_image_to_video`。
- 多张图/多镜头任务需要一致性策略。
- 模型成本和质量可按场景做更优选择。
