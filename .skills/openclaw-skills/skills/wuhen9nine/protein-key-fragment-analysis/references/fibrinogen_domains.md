# 纤维蛋白原 (Fibrinogen) 已知功能域参考

## 纤维蛋白原结构概述
纤维蛋白原由三条多肽链组成六聚体 (AαBβγ)₂：
- FGA (Aα链) - UniProt: P02671
- FGB (Bβ链) - UniProt: P02675  
- FGG (γ链) - UniProt: P02679

## 关键功能片段

### 1. 纤维肽 (Fibrinopeptides) - 凝血酶切位点
| 名称 | 位置 | 序列特征 | 功能 |
|------|------|----------|------|
| 纤维肽A (FpA) | Aα链 N端 (1-16) | AGPT...RG... | 凝血酶切割后释放，启动纤维蛋白聚合 |
| 纤维肽B (FpB) | Bβ链 N端 (1-14) | ...凝血酶识别位点 | 凝血酶切割，促进侧向聚合 |

### 2. RGD 序列 - 细胞黏附
| 名称 | 位置 | 序列 | 功能 |
|------|------|------|------|
| RGD motif | Aα链 (95-97) | RGDS | 整合素结合位点，介导血小板聚集和细胞黏附 |

### 3. γ链交联位点 - 因子XIIIa底物
| 名称 | 位置 | 序列特征 | 功能 |
|------|------|----------|------|
| γ交联位点 | γ链 C端附近 | Lys-Gln底物序列 | 因子XIIIa介导的共价交联，稳定纤维蛋白凝块 |

### 4. 纤维蛋白原C端结构域 (Fibrinogen C-terminal domain)
| 链 | 位置 | 特征 | 功能 |
|----|------|------|------|
| 所有三条链 | C端约220-250aa | 含保守Cys残基形成二硫键 | 聚合位点，D区/E区相互作用 |

### 5. 聚合相关位点
| 名称 | 位置 | 功能 |
|------|------|------|
| "a"位点 | γ链C端 | 与D区"A"位点结合 |
| "b"位点 | Bβ链C端 | 与D区"B"位点结合 |
| D-二聚体位点 | γ链 | 纤维蛋白降解产物标志 |

## 功能预测线索

| 特征序列 | 功能推断 |
|----------|----------|
| N端信号肽 (M...L/FC/V) | 分泌导向 |
| RGDS/RGDF | 整合素结合，细胞黏附 |
| 高保守Cys富集区 | 二硫键网络，结构稳定 |
| 凝血酶识别位点 (Lys/Arg附近) | 酶原激活位点 |
| YGGGWWYN...CQA | γ链特征序列，聚合功能 |
| GHLNGVYYQGGTYSK | γ链保守区，交联相关 |
