# 氨基酸功能分类体系

> ⚠️ 此分类方案已与文献验证对齐，**严格固定，不得修改**。

---

## 参与统计的6个类别（共17种氨基酸）

| 类别 | 英文名 | 单字母 | 化学性质 |
|------|--------|--------|---------|
| 疏水性 | Hydrophobic | **V、L、I、M** | 非极性脂肪族侧链，倾向埋藏于蛋白质内核 |
| 亲核性 | Nucleophilic | **S、T、C** | 含羟基/巯基，极性，多暴露于表面 |
| 芳香性 | Aromatic | **F、Y、W** | 含芳香环，具π电子系统 |
| 酰胺类 | Amide | **N、Q** | 含酰胺基，极性不带电 |
| 酸性 | Acidic | **D、E** | 侧链在生理pH带负电 |
| 阳离子性 | Cationic | **H、K、R** | 侧链在生理pH带正电 |

## 排除不统计的字符

| 字符 | 含义 | 排除原因 |
|------|------|---------|
| X | 低保守性位点 | MSA中该位置无主导氨基酸 |
| A | 丙氨酸 | 用户指定排除 |
| G | 甘氨酸 | 用户指定排除（无侧链，结构特殊） |
| P | 脯氨酸 | 用户指定排除（亚氨基酸，破坏二级结构） |

## 代码实现

```python
FUNCTIONAL_CLASSES = {
    'Hydrophobic':  ['V', 'L', 'I', 'M'],
    'Nucleophilic': ['S', 'T', 'C'],
    'Aromatic':     ['F', 'Y', 'W'],
    'Amide':        ['N', 'Q'],
    'Acidic':       ['D', 'E'],
    'Cationic':     ['H', 'K', 'R'],
}
EXCLUDE = set('XAGP')
```

## 注意事项

- F/Y/W 归入 Aromatic，**不归入** Hydrophobic
- A/G/P 与 X 处理方式完全相同：从序列中删除，不参与任何统计
