# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for this project, following the [MADR](https://adr.github.io/madr/) format.

## What is an ADR?

An Architectural Decision Record captures a significant architectural decision made along with its context and consequences. ADRs are immutable once accepted -- if a decision is reversed, a new ADR supersedes the old one.
