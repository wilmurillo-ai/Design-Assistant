## 溯源标注概述

### 源源标注的目的
1. **可追溯性**：确保所有关键论点都有明确来源
2. **可信度**：通过可信来源提高报告的可信度
3. **可验证性**：方便读者验证论点的真实性

### 基本原则
- 所有关键论点必须标注来源
- 优先使用高可信度来源
- 标注格式统一规范
- 避免重复标注同一来源

### 关键论点界定
行业定义报告中必须溯源的关键论点包括：
1. **技术参数**：如CAS号、纯度、尺寸、熔点、沸点、电压、转化率、视场角、亮度、分辨率、刷新率等理化参数和性能参数
   - **简洁性约束**：技术原理部分必须压缩为一句话概括，仅保留最核心的技术特征和关键参数（如CAS号、熔点、核心性能指标），避免详细罗列
2. **法规标准引用**：如GB/T编号、监管定义、行业标准、药典版本、专利号等
3. **行业共识性结论**：如"主流技术路线是XX"、"行业普遍认为"、"核心技术为XX"等结论性陈述

## 标注格式规范

### 标准格式
定义内容中使用编号标记，溯源来源部分单独列出：

```
定义内容：
HUD的视场角为10度×5度，虚像距离为2.5米，亮度为8000尼特[1]。

溯源来源：
[1] 专利CN112345678：https://patents.google.com/patent/CN112345678
```

### 格式要求
- 在定义内容中使用编号标记（如[1][2][3]）标注关键论点
- 在"溯源来源"部分单独列出所有来源的链接
- 编号与来源列表一一对应
- 确保URL完整且可访问

### 标注示例
```
HUD的视场角为10度×5度，虚像距离为2.5米，亮度为8000尼特[1]。
阿司匹林的CAS号为50-78-2，分子量为180.16，熔点为135℃[2]。
根据国家标准GB/T 18384-2020，HUD是一种车载光学显示装置[3]。
主流技术路线是激光投影[4]。

### 溯源来源
[1] 专利CN112345678：https://patents.google.com/patent/CN112345678
[2] 中国药典2020版：https://www.chp.org.cn/
[3] 国家标准GB/T 18384-2020：https://openstd.samr.gov.cn/
[4] 汽车工业协会白皮书2023：https://www.caam.org.cn/
```

## 可信来源分级

### 一级来源（最高可信度）- 权威信源
- 政府机构文件
- 国家标准和法规
- 国际标准组织文件
- 药典（国家药典、国际药典）
- 专利文件

### 二级来源（高可信度）- 行业信源
- 行业协会官方文件
- 券商研报（头部券商）
- 顶尖咨询机构报告（麦肯锡、BCG、贝恩）

### 三级来源（中等可信度）- 学术信源
- 学术论文（同行评审）
- 学术数据库
- 研究机构报告

### 不可信来源（禁止使用）
- 未经验证的社交媒体内容
- 匿名来源
- 商业广告和营销内容
- 企业宣传资料（非技术文档）
- 未经验证的第三方商业咨询机构（如智研咨询、艾瑞咨询、易观分析等）

## 标注位置与频率

### 标注位置
- 在定义内容中，每个关键论点后使用编号标记（如[1][2][3]）
- 在"溯源来源"部分单独列出所有来源的链接
- 同一来源用于多个论点时，可使用相同编号

### 标注频率
- 技术参数：必须标注（100%）
- 法规标准引用：必须标注（100%）
- 行业共识性结论：必须标注（100%）
- 支撑性论据：建议标注（≥80%）
- 背景描述：可选标注（≥50%）

### 标注示例（正确）
```
HUD是一种车载光学显示装置，通过将关键驾驶信息投影到驾驶员视野前方，区别于传统仪表盘和中控屏[1]。

HUD的视场角为10度×5度，虚像距离为2.5米，亮度为8000尼特，刷新率为60Hz[2]。

数据显示，使用HUD可降低30%的事故率[3]。

### 溯源来源
[1] GB/T 18384-2020：https://openstd.samr.gov.cn/bzgk/gb/newGbInfo?hcno=18384-2020
[2] 专利CN112345678：https://patents.google.com/patent/CN112345678
[3] 汽车工业协会白皮书2023：https://www.caam.org.cn
```

### 标注示例（错误）
```
HUD是一种车载光学显示装置。[来源：GB/T 18384-2020]其技术原理基于光学投影技术。

（错误：使用了旧的链接格式，且标注在段落中间）

HUD的视场角为10度×5度。

（错误：技术参数未标注）

根据某研究报告，主流技术路线是激光投影。

（错误：来源不具体）
```

## 常见错误

### 错误1：来源不具体
**错误示例**：
```
HUD的视场角为10度×5度。[来源：某报告]
```
**正确示例**：
```
HUD的视场角为10度×5度[1]。

### 溯源来源
[1] 专利CN112345678：https://patents.google.com/patent/CN112345678
```

### 错误2：格式不规范
**错误示例**：
```
HUD的视场角为10度×5度（来源：专利CN112345678）
```
**正确示例**：
```
HUD的视场角为10度×5度[1]。

### 溯源来源
[1] 专利CN112345678：https://patents.google.com/patent/CN112345678
```

### 错误3：重复标注同一来源
**错误示例**：
```
HUD的视场角为10度×5度[1]，虚像距离为2.5米[2]。

### 溯源来源
[1] 专利CN112345678：https://patents.google.com/patent/CN112345678
[2] 专利CN112345678：https://patents.google.com/patent/CN112345678
```
**正确示例**：
```
HUD的视场角为10度×5度，虚像距离为2.5米[1]。

### 溯源来源
[1] 专利CN112345678：https://patents.google.com/patent/CN112345678
```

### 错误4：使用低可信度来源
**错误示例**：
```
HUD技术遥遥领先。[来源：某企业宣传册]
```
**正确示例**：
```
HUD技术基于光学投影原理，视场角为10度×5度，亮度为8000尼特[1]。

### 溯源来源
[1] 专利CN112345678：https://patents.google.com/patent/CN112345678
```

### 错误5：来源与内容不匹配
**错误示例**：
```
主流技术路线是激光投影[1]。

### 溯源来源
[1] 专利CN112345678：https://patents.google.com/patent/CN112345678
```
**正确示例**：
```
HUD的视场角为10度×5度[1]。
主流技术路线是激光投影[2]。

### 溯源来源
[1] 专利CN112345678：https://patents.google.com/patent/CN112345678
[2] 汽车工业协会白皮书2023：https://www.caam.org.cn
```

## 引用示例

### 完整示例
```markdown
## HUD 定义报告

**分类层级**：L4 – 具体产品
**统计口径**：终端总消费规模

### 定义内容
HUD（抬头显示器）是一种车载光学显示装置，通过将关键驾驶信息投影到驾驶员视野前方，区别于传统仪表盘和中控屏[1]。其技术基于光学投影和反射机制，核心参数包括视场角10度×5度、亮度8000尼特、刷新率60Hz[2]。HUD主要应用于汽车、航空和工业设备领域。数据显示，使用HUD可降低30%的事故率，双向提升驾驶体验和安全保障[3]。本定义中，包含汽车前装HUD、汽车后装HUD、航空HUD，不包含传统仪表盘、中控屏，市场规模按终端全渠道消费总金额计算[3]。

### 溯源来源
[1] GB/T 18384-2020：https://openstd.samr.gov.cn/bzgk/gb/newGbInfo?hcno=18384-2020
[2] 专利CN112345678：https://patents.google.com/patent/CN112345678
[3] 汽车工业协会白皮书2023：https://www.caam.org.cn

---
*报告生成时间：2025-06-17*
```

### 技术原理简洁性约束示例

**正确示例（符合简洁性约束）**：
```
其技术基于光学投影和反射机制，核心参数包括视场角10度×5度、亮度8000尼特[1]。
```

**错误示例（过于详细罗列）**：
```
其技术原理基于光学投影技术，具体来说是通过微型显示器产生的光学图像经过光学系统放大和反射，最终在驾驶员前方的挡风玻璃上形成虚像，核心技术参数包括视场角10度、虚像距离2.5米、亮度8000尼特、刷新率60Hz、分辨率1280×720、对比度1000:1、响应时间10ms[1]。
```
