## 四维结构概述

### 定义
四维结构是指行业定义报告必须包含的四个核心维度，确保定义的完整性、准确性和可操作性。

### 四个维度
1. **核心本体界定**：明确词条在广义分类学中的归属（属概念）和差异化核心特征（种差）
2. **技术原理与物理化学属性（简洁性约束）**：压缩为一句话概括，仅保留最核心的技术特征和关键参数
3. **病理/功能与下游应用场景**：遵循"疾病成因/痛点-靶向受众-解决手段"的逻辑闭环
4. **边界划定与统计口径**：使用"包含"与"不包含"进行排他性界定，明确市场规模计算方式

### 融合输出格式
将四个维度整合为一个流畅的自然段落，逻辑连贯，表述清晰。

## 四维结构融合写作指南

### 写作原则
1. **逻辑连贯**：四个维度之间使用过渡词连接，形成完整的逻辑链条
2. **信息完整**：确保每个维度的关键信息都包含在段落中
3. **表述流畅**：避免生硬的分段，用自然语言串联各个维度
4. **溯源分离**：在定义内容中使用编号标记（如[1][2][3]），对应后续的溯源来源列表
5. **技术简洁**：技术原理部分必须压缩为一句话，仅保留最核心的技术特征和关键参数

### 写作顺序
1. **开头**：核心本体界定（属概念+种差）
2. **承接**：技术原理与理化属性（一句话概括，仅核心特征+关键参数）
3. **转折**：病理/功能与下游应用场景（痛点+受众+方案）
4. **收尾**：边界划定与统计口径（包含+不包含+计算方式）

### 过渡词使用
- 从本体界定到技术原理："其技术基于"、"其核心参数包括"
- 从技术原理到应用场景："主要应用于"、"针对"、"服务于"
- 从应用场景到边界划定："包含"、"不包含"、"其市场规模计算方式"

### 技术原理简洁性约束

**核心原则**：
- 技术原理部分必须压缩为一句话
- 仅保留最核心的技术特征
- 仅保留关键参数（如CAS号、熔点、核心性能指标）
- 避免冗长的技术细节描述

**正确示例**：
```
✅ 其技术基于锂离子嵌入脱嵌机制，核心参数包括标称电压3.7V、能量密度150-260Wh/kg。
```

**错误示例**：
```
❌ 其技术原理基于锂离子在正负极之间的嵌入和脱嵌过程，具体来说是在充电时锂离子从正极材料中脱出，经过电解质和隔膜嵌入负极材料中，放电时则相反，锂离子从负极脱出嵌入正极，通过这种可逆的电化学反应实现电能的存储和释放。
```

### 溯源标注分离格式

**格式要求**：
- 在定义内容中使用编号标记（如[1][2][3]）标注关键论点
- 在"溯源来源"部分单独列出所有来源的链接
- 编号与来源列表一一对应
- 确保URL完整且可访问

**示例**：
```
定义内容中：
HUD的视场角为10度×5度，虚像距离为2.5米[1]。主流技术路线是激光投影[2]。

溯源来源部分：
[1] 专利CN112345678：https://patents.google.com/patent/CN112345678
[2] 汽车工业协会白皮书2023：https://www.caam.org.cn/
```

## 融合写作模板

### 通用模板
```
[词条名]是[广义分类学中的属概念]，通过[差异化核心特征]区别于[其他类似概念]。其技术基于[核心技术]，核心参数包括[关键参数1]、[关键参数2]、[关键参数3][1]。[词条名]主要应用于[应用场景1]、[应用场景2]和[应用场景3]，针对[疾病成因/痛点]，服务于[靶向受众]，通过[解决手段]实现应用价值[2]。本定义中，包含[包含内容1]、[包含内容2]和[包含内容3]，不包含[不包含内容1]和[不包含内容2]，其市场规模按[计算方式]计算，[具体说明][3]。

### 溯源来源
[1] [来源名称]：URL
[2] [来源名称]：URL
[3] [来源名称]：URL
```

### 简化模板（参数较少时）
```
[词条名]是[广义分类学中的属概念]，通过[差异化核心特征]区别于[其他类似概念]。其技术基于[核心技术]，核心参数为[关键参数][1]。[词条名]针对[痛点]，服务于[受众]，通过[解决手段]实现价值，主要应用于[应用场景][2]。包含[包含内容]，不包含[不包含内容]，市场规模按[计算方式]计算[3]。

### 溯源来源

[2] [来源名称]：URL
[3] [来源名称]：URL
```

## 写作技巧

### 技巧1：使用分号分隔长列表
当包含/不包含的内容较多时，使用分号分隔：
```
包含：汽车前装HUD、汽车后装HUD、航空HUD；不包含：传统仪表盘、中控屏、手机导航
```

### 技巧2：合并相似参数
将相关的核心参数合并为一句话：
```
核心参数包括CAS号50-78-2、分子量180.16、熔点135℃、纯度≥99.5%。
```

### 技巧3：简化应用场景描述
使用概括性语言描述应用场景：
```
主要应用于汽车、航空和工业设备领域
```

### 技巧4：自然连接边界划定
使用自然语言连接包含/不包含：
```
本定义中，包含汽车前装HUD、汽车后装HUD、航空HUD，不包含传统仪表盘、中控屏、手机导航
```

### 技巧5：技术原理一句话概括
使用"其技术基于...，核心参数包括..."的句式，将技术原理压缩为一句话：
```
其技术基于光学投影和反射机制，核心参数包括视场角10度×5度、亮度8000尼特。
```

## 完整示例

### 示例1：HUD（抬头显示器）定义报告

## HUD 定义报告

**分类层级**：L4 – 具体产品，面向终端消费，符合产品级命名规范
**统计口径**：终端总消费规模（金额）或终端总销量（台数）

### 定义内容
HUD（抬头显示器）是一种车载光学显示装置，通过将关键驾驶信息投影到驾驶员视野前方实现虚实叠加显示，区别于传统仪表盘和中控屏等固定位置显示设备。其技术基于光学投影和反射机制，核心参数包括视场角10度×5度、亮度8000尼特、刷新率60Hz[1]。HUD主要应用于汽车、航空和工业设备领域，针对驾驶员视线频繁转移的痛点，服务于汽车驾驶员，通过将关键信息投影到视野前方实现提高驾驶安全性的应用价值[2]。本定义中，包含汽车前装HUD、汽车后装HUD、航空HUD、工业显示HUD，不包含传统仪表盘、中控屏、手机导航、独立显示屏，其市场规模按终端全渠道消费总规模/总销量计算，具体包括整车厂前装配套的出厂价值（按最终用户提车时配置的硬件价值折算）以及后装零售市场的终端销售额，不包括上游零部件（如芯片、光学镜片）的单独产值[3]。

### 示例2：阿司匹林定义报告（医疗场景）

## 阿司匹林 定义报告

**分类层级**：L4 – 具体产品，化学药品
**统计口径**：终端总消费规模（金额）或终端总销量（片数/克数）

### 定义内容
阿司匹林是一种非甾体抗炎药，通过抑制环氧合酶-1和环氧合酶-2活性，区别于其他抗炎药物。其技术基于乙酰化环氧合酶抑制机制，核心参数包括CAS号50-78-2、分子量180.16、熔点135℃、纯度≥99.5%[1]。阿司匹林针对血小板聚集的疾病成因，服务于心脑血管疾病患者，通过抑制血小板血栓素A2的生成实现预防血栓形成的应用价值，主要应用于心肌梗死预防、脑卒中预防和缺血性心脏病治疗，具有预防和治疗双向价值[2]。本定义中，包含阿司匹林片剂、阿司匹林肠溶片、阿司匹林泡腾片，不包含阿司匹林复方制剂、其他非甾体抗炎药、布洛芬，其市场规模按终端全渠道消费总规模/总销量计算，具体包括处方药渠道（凭处方购买，含医保统筹支付）和非处方药渠道（零售直接购买）的终端销售额，以及低剂量心血管预防用途和高剂量解热镇痛用途的所有口服及外用剂型，不包括上游原料药、中间体、复合制剂的产值[3]。

### 示例3：锂离子电池定义报告（新能源场景）

## 锂离子电池 定义报告

**分类层级**：L4 – 具体产品，储能装置
**统计口径**：终端总消费规模（金额）或终端总销量（千瓦时/只数）

### 定义内容
锂离子电池是一种二次电池，通过锂离子在正负极之间嵌入和脱嵌实现电能存储和释放，区别于铅酸电池、镍氢电池等传统电池。其技术基于锂离子嵌入脱嵌机制，核心参数包括标称电压3.7V、能量密度150-260Wh/kg、循环寿命500-2000次[1]。锂离子电池针对传统电池能量密度低、循环寿命短的痛点，服务于消费电子、新能源汽车和储能系统等领域，通过高能量密度和长循环寿命实现清洁能源存储的应用价值，主要应用于智能手机、电动汽车、储能电站等场景[2]。本定义中，包含消费类锂离子电池、动力类锂离子电池、储能类锂离子电池，不包含一次性锂电池、镍氢电池、铅酸电池，其市场规模按终端全渠道消费总规模/总销量计算，具体包括消费电子、新能源汽车、储能系统等终端应用领域的电池销售额，不包括上游锂矿、正负极材料、电解液等原材料的产值[3]。
