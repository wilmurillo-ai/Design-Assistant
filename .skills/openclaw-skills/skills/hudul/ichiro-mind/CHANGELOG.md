# Ichiro-Mind Changelog

## [1.0.0] - 2026-03-07

### Added
- Initial release of Ichiro-Mind
- 4-layer memory architecture (HOT, WARM, COLD, ARCHIVE)
- Neural graph with spreading activation (WARM layer)
- Semantic vector search (COLD layer)
- Experience tracking and learning
- Smart routing between layers
- MCP protocol integration
- CLI tool for management
- Bilingual documentation (EN/ZH)

### Features
- **HOT Layer**: SESSION-STATE.md with WAL protocol
- **WARM Layer**: Neural graph with Hebbian learning
- **COLD Layer**: Vector search with LanceDB
- **ARCHIVE Layer**: Persistent long-term storage
- **Hygiene Engine**: Auto-cleanup and optimization
- **Learning Engine**: Decision tracking and lesson learning
