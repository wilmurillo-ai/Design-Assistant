# 通用分析框架速查

> 在规划/活动/服务体系/汇报任何场景下，按需调用以下框架。

---

## 框架一：增长路径漏斗模型

```
[用户池总量] × [触达率%] = [触达人数]
[触达人数] × [参与率%] = [参与人数]
[参与人数] × [转化率%] = [成交人数]
[成交人数] × [客单价] = [GMV]

每层拆解时：
- 当前转化率是多少？
- 行业/历史基线是多少？
- 如果提升X个点，产出是多少？
```

**使用场景**：任何需要从「人群」推算「产出」的场景，都用漏斗模型。先建模，再优化哪层转化率。

---

## 框架二：人物场分析

```
人（执行层/团队）
├── 岗位配置：[角色 | 人数 | 主要职责]
├── 能力现状：[擅长什么 / 缺什么]
└── 人效：[每人服务/产出量]

物（工具/资源层）
├── 现有工具：[在用的 + 效率评估]
├── 内容资产：[沉淀了什么 / 缺什么]
└── 预算资源：[余额 / 可调配量]

场（场景/触点层）
├── 主要服务/转化场景：[列举]
├── 各场景现状：[有效/低效/缺失]
└── 用户旅程覆盖：[入学/服务/续费/流失各节点]
```

**使用场景**：现状诊断必用。拆解完人物场，才能定位问题在哪一层，方法才能命中病因而非症状。

---

## 框架三：目标合理性分析法

```
历史基线：[上期实际值]
行业基准：[对标数据（如有）]
↓
增量来源拆解：
- 确定性增量：[已有路径的自然增长，数字]
- 新路径贡献：[新策略预计产出，数字]
- 不确定折扣：[新路径测试期打折比例，如×70%]
↓
目标公式：
[确定性增量] + [新路径贡献×折扣率] = [合理目标值]

目标区间：
- 保守：[确定性增量 only]
- 中性：[+新路径×50%]
- 乐观：[+新路径×100%]
```

**使用场景**：在「目标确定」节，用这个公式论证目标合理性，避免拍脑袋或过度承诺。

---

## 框架四：Q1-Q4 战略解码结构

```
年度北极星指标：[一个核心数字]

Q1（1-3月）：[主题词]
- 核心任务：[1-2件]
- 关键结果：[数字目标]
- 资源重点：[人/预算/技术]

Q2（4-6月）：[主题词]
- 同上结构

Q3（7-9月）：[主题词]
- 同上结构

Q4（10-12月）：[主题词]
- 同上结构

各季度节奏建议：
- Q1：建基础（体系/工具/数据）
- Q2：跑路径（主路径验证+优化）
- Q3：规模化（复制有效路径）
- Q4：冲目标（续费/GMV冲刺+年度复盘）
```

**使用场景**：年度规划必用。把年度目标落到每季度具体任务，避免「年初立flag年尾看数据」的断层。

## 框架五：预算ROI测算表

```
收入侧：
[项目] | [计算方式] | [预计金额]
GMV：[成交人数] × [客单价] = [XX万]
毛利：[GMV] × [毛利率XX%] = [XX万]
↓ 总收入：[XX万]

成本侧：
[项目] | [金额]
人力成本：[人数×时间×日薪]
工具/平台费：[XX]
内容/物料制作：[XX]
投放/渠道费：[XX]
↓ 总成本：[XX万]

ROI公式：
ROI = (毛利 - 总成本) / 总成本 × 100%

健康ROI参考：
- 拉新活动：ROI ≥ 150%（需覆盖LTV）
- 促活/转化：ROI ≥ 300%
- 续费活动：ROI ≥ 500%（边际成本低）
```

**使用场景**：活动方案/预算申请必用。有ROI测算才能争取资源，有历史基线才能论证可行性。

## 框架六：四层方法路径检查表

在输出任何运营方案时，至少覆盖以下4层中的2层：

| 层级 | 覆盖内容 | 核心问题 |
|------|---------|---------|
| **体系层** | 流程/机制/规范/SLA | 什么规则保证服务/执行的一致性？ |
| **产品层** | 功能/界面/用户体验改造 | 工具/产品端能做什么来提效？ |
| **技术层** | 数据采集/自动化/系统开发 | 什么可以自动化，不依赖人工？ |
| **运营层** | 人工干预/活动/内容/社群 | 人工的核心价值在哪里，如何配置？ |

**使用提示**：续费率低的方案，如果只做运营层（加强沟通），而不做体系层（服务SLA）和技术层（预警系统），改善有限且不可持续。

---

## 快速选框架

| 需要做什么 | 优先用哪个框架 |
|----------|------------|
| 推算预期产出 | 框架一（漏斗模型） |
| 诊断团队/系统问题 | 框架二（人物场） |
| 论证目标是否合理 | 框架三（目标合理性分析） |
| 年度/季度规划拆解 | 框架四（Q1-Q4解码） |
| 申请预算/评估活动 | 框架五（ROI测算） |
| 自检方案是否完整 | 框架六（四层路径检查） |
