# AI PPT - Professional Presentation Creator

A sophisticated PPT creation skill that produces elegant, modern presentations with refined typography, balanced layouts, and professional visual design.

## When to Use

- Create business presentations, project proposals, or reports
- Design portfolio or product showcases
- Build training materials or educational content
- Generate meeting presentations with clean aesthetics
- Any request involving PowerPoint, slides, or presentation creation

## Core Principles

1. **Minimalist Elegance** - Clean layouts with purposeful white space
2. **Professional Typography** - Clear hierarchy with 2-3 complementary fonts
3. **Balanced Composition** - Asymmetric balance for visual interest
4. **Refined Color Palette** - Understated elegance, never garish
5. **Content Clarity** - Information hierarchy that guides the viewer

## Design Guidelines

### Typography
- Headings: Bold, larger size (28-36pt for titles)
- Body: Regular weight, comfortable reading size (16-18pt)
- Use 2-3 font families maximum throughout

### Color Scheme
- Primary: Deep navy (#1a365d) or charcoal (#2d3748)
- Accent: Muted gold (#d69e2e) or teal (#319795)
- Background: Pure white or soft cream (#faf5eb)
- Text: Dark gray (#4a5568) for body, black for headings

### Layout Principles
- Title slides: Centered, minimal, impactful
- Content slides: Left-aligned text, visual breathing room
- Use consistent margins and alignment
- 6x6 grid system for element placement

## Output Format

```markdown
## Slide Title

### Content Point 1
Brief description or details

### Content Point 2
Brief description or details

### Visual Element
[Description of chart/image/graphic]
```

## Quality Checklist

- [ ] Clear visual hierarchy on every slide
- [ ] Consistent font usage throughout
- [ ] Adequate white space between elements
- [ ] Professional color combinations
- [ ] No cluttered or overwhelming layouts
- [ ] Readable at standard projection distances
