# 飞猪旅游攻略规划技能

## ✅ 技能简介

**技能名称：** `fliggy-travel-planner`

自动从小红书获取旅游攻略，并整合飞猪机票信息，生成完整的行程规划！

---

## 🚀 快速使用

### 方式 1：直接使用（推荐）

在聊天中直接说：

```
帮我规划一下去大理的旅行，看下小红书的攻略和机票价格
```

技能会自动：
1. ✅ 从小红书搜索大理旅游攻略
2. ✅ 提取攻略中的关键信息（景点、美食、住宿、预算）
3. ✅ 查询飞猪机票价格（需要你指定出发地和日期）
4. ✅ 生成完整的行程规划报告

### 方式 2：脚本调用

```bash
# 搜索攻略
python scripts/search_xiaohongshu.py \
  --destination 大理 \
  --limit 10

# 生成报告
python scripts/generate_report.py \
  --travel-info travel_info.json \
  --flight-info flight_info.json \
  --from-city 北京 \
  --output 大理旅行规划.md
```

## 🎯 核心功能

### 1. 小红书攻略搜索
- 🔍 搜索目的地相关高赞笔记
- 📊 提取点赞、收藏、评论数据
- 📝 解析笔记正文内容

### 2. 信息结构化处理
- 🤖 使用大模型提取关键信息
- 📋 生成结构化旅行数据
- 🎯 识别景点、美食、住宿、预算

### 3. 飞猪机票查询
- ✈️ 查询往返机票价格
- 💰 提取最低价航班
- 🔗 生成预订链接

### 4. 整合报告生成
- 📄 生成 Markdown 格式报告
- 📊 包含攻略 + 机票 + 预算
- 🗓️ 推荐行程安排

## ⚠️ 注意事项

1. **攻略时效性** - 小红书攻略可能有过期信息，注意筛选最新内容
2. **价格波动** - 机票价格实时变动，查询结果仅供参考
3. **登录状态** - 首次使用需要登录各平台，后续会保持会话
4. **验证码处理** - 遇到验证码需要人工介入
5. **内容合规** - 攻略内容需遵守平台规范，避免侵权

## 🐛 故障排查

### 问题 1：小红书搜索结果为空
- 检查关键词是否准确
- 尝试使用更通用的目的地名称
- 检查浏览器登录状态

### 问题 2：机票查询失败
- 检查飞猪账号密码是否正确
- 确认日期是否已过期
- 检查航线是否存在

### 问题 3：攻略信息提取不准确
- 调整大模型提取 prompt
- 增加参考笔记数量
- 手动校验关键字段
