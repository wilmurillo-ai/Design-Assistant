# 监控任务配置示例

## 单程监控

### 基本配置

```json
{
  "name": "机票监控 - 北京 - 三亚 03-25",
  "schedule": {
    "kind": "cron",
    "expr": "0 9 * * *",
    "tz": "Asia/Shanghai"
  },
  "payload": {
    "kind": "agentTurn",
    "message": "查询北京到三亚 3 月 25 日机票价格，低于 1500 元通知用户"
  },
  "sessionTarget": "isolated"
}
```

## 往返监控

```json
{
  "name": "机票监控 - 北京 - 三亚 往返",
  "schedule": {
    "kind": "cron",
    "expr": "0 9,18 * * *",
    "tz": "Asia/Shanghai"
  },
  "payload": {
    "kind": "agentTurn",
    "message": "查询北京到三亚往返机票，去程 3 月 25 日，返程 3 月 30 日，总价低于 2500 元通知用户"
  },
  "sessionTarget": "isolated"
}
```

## Cron 表达式说明

| 表达式 | 说明 |
|--------|------|
| `0 9 * * *` | 每天 9:00 |
| `0 */6 * * *` | 每 6 小时 |
| `0 9,18 * * *` | 每天 9:00 和 18:00 |
| `0 9 * * 1-5` | 周一到周五 9:00 |
| `0 9 1 * *` | 每月 1 号 9:00 |
