# Social Media Manager - 社交媒体管理助手

## 功能描述
一站式社交媒体管理工具，支持多平台内容发布、数据分析、粉丝互动。

## 使用场景
- 多平台内容同步发布
- 社交媒体数据分析
- 粉丝互动管理
- 热点追踪
- 内容排期管理
- KOL/KOC 数据分析

## 命令

### 多平台发布
```
发布内容 内容=文章.md 平台=微博，公众号，知乎 定时=2026-03-29 09:00
```

### 数据分析
```
获取数据 平台=微博 账号=@xxx 时间范围=7 天
```

### 粉丝互动
```
回复评论 平台=微博 时间范围=1 小时 情感=正面优先
```

### 热点追踪
```
追踪热点 关键词=AI 平台=微博，知乎 时间=24 小时
```

## 支持平台
