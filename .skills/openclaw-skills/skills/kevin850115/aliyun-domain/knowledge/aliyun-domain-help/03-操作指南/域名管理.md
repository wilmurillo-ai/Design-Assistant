## 核心场景操作指南

### 场景一：域名生命周期管理（续费、赎回与到期）

一个典型的域名生命周期包括正常期、续费宽限期、高价赎回期和等待删除期。

| 阶段 | 说明 | 操作建议 |
|------|------|----------|
| **域名到期前** | 域名处于正常状态 | 配置**自动续费**。系统会在域名到期前 15 天自动扣除下一年费用，这是相对安全、低成本的策略 |
| **续费宽限期** | 到期后 0-30 天（.hk 域名为 28 天） | 域名解析可能被暂停，但仍可以按**正常价格手动续费** |
| **高价赎回期** | 到期后 31-60 天（.cn 等为 31-44 天） | 必须支付**远高于正常续费价格的赎回费用**才能恢复域名 |
| **等待删除期** | 赎回期结束后 | 域名将被注册局删除并重新向公众开放注册，原持有者将无法恢复域名 |

**操作入口**：

| 操作 | 路径 |
|------|------|
| **手动续费** | 阿里云域名控制台 → 域名列表 → 找到目标域名 → 单击操作列的续费 |
| **自动续费** | 域名列表页面 → 勾选希望开启自动续费的域名 → 单击下方更多 > 设置自动续费 |
| **批量续费** | 域名列表页面 → 勾选多个域名 → 单击下方的续费 |
| **代续费** | 如无法登录域名所在账号或需由其他阿里云账号支付续费，可使用域名代续费 |

### 场景二：域名持有者与信息变更

当域名所有权发生转移（如交易、公司业务调整），或持有者联系信息变更时，需及时更新域名信息。

| 操作类型 | 目的与场景 |
|----------|------------|
| **域名持有者信息修改（过户）** | 变更域名持有人。适用于域名买卖、个人域名转至企业名下等场景 |
| **域名其他信息修改** | 仅更新域名持有者的联系方式（如邮箱、电话、地址），或变更域名的管理者、付费者、技术者等联系信息，不改变域名所有权 |

### 场景三：域名解析与 DNS 服务器管理

域名解析是将域名指向网站服务器 IP 地址的关键步骤。

**概念区分**：

| 操作 | 说明 |
|------|------|
| **修改解析记录** | 在当前 DNS 服务商的平台修改域名指向的网站服务器 |
| **修改 DNS 服务器** | 将域名的解析权整体从一个服务商迁移到另一个服务商 |

> **重要**: 修改域名 DNS 服务器前，必须确保所有解析记录（如 A 记录、CNAME、MX 记录等）已在**新 DNS 服务商侧完整配置并验证无误**。否则变更 DNS 服务器可能导致网站服务中断。

### 场景四：域名转移

根据业务需求，在不同注册商或不同阿里云账号之间转移域名的管理权。

| 转移类型 | 目的与场景描述 |
|----------|----------------|
| **转入阿里云** | 域名当前在其他注册商，希望转到阿里云进行管理 |
| **从阿里云转出** | 域名在阿里云，希望转到其他注册商进行管理 |
| **账号间转移** | 域名在阿里云 A 账号下，希望转移到阿里云 B 账号下管理（例如，从个人账号转到公司账号） |

## 高级功能与自动化

### 权限管理（RAM）

为实现企业内部的多人协作和安全管控，可使用**RAM**（访问控制）对域名管理权限进行精细化授权。

例如，可授权 RAM 用户管理某个特定域名的权限，但禁止其管理其他域名。

**详情**: [授权 RAM 用户管理域名](https://help.aliyun.com/zh/dws/user-guide/authorize-ram-users-to-manage-domains)

### 域名安全

| 安全措施 | 说明 |
|----------|------|
| **域名安全锁** | 在域名控制台开启"禁止更新锁"和"禁止转移锁"，可有效防止域名因账号被盗等原因被恶意修改信息或转移 |
| **注册局安全锁** | 对安全性要求较高的域名，可选择购买开通注册局安全锁 |
| **DNSSEC** | 为域名启用 DNSSEC（域名系统安全扩展），可防止 DNS 欺骗和缓存污染，提升域名解析的安全性 |
