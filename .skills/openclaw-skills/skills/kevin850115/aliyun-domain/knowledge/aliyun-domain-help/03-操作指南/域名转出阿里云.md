当您因业务需求需要将域名更换服务商时，本文指导您如何从阿里云将域名转移至其他注册商。

如果需要将域名转移到另一个阿里云中国站账号，请参见 [域名转移至其他阿里云账号](https://help.aliyun.com/zh/dws/user-guide/online-transfer)。

## 转出规则

请对照以下清单，检查您的域名能否转出阿里云：

| 检查项 | 说明与操作指引 |
|--------|----------------|
| **域名注册商为阿里云** | 在域名基本信息页面的"注册商"一栏确认 |
| **域名注册/转入已满 60 天** | 根据 ICANN 转移政策要求，域名注册或转入阿里云需满 60 天，且距离最后一次成功转移注册商也需大于 60 天 |
| **域名状态正常** | - **关闭禁止转移锁**: 确保禁止转移锁（clientTransferProhibited）已关闭<br>- **无异常状态**: 域名状态不能为 serverTransferProhibited（注册局锁定）或 clientHold（注册商暂停解析）等 |
| **无纠纷** | 域名未处于任何纠纷、仲裁或法院限制转出的状态 |
| **域名持有者邮箱有效** | 转移码和所有关键通知都将发送至域名持有者邮箱，请确认邮箱地址正确且可正常收发邮件 |
| **域名有效期充足** | CNNIC 注册局下的域名，在域名到期前 15 天内不能转出。建议在域名到期 30 天前办理转出 |
| **续费/赎回已满 45 天** | 如果域名近期有续费或赎回操作，建议在续费/赎回操作 45 天后再提交转出申请。在 45 天内转出将导致您的续费年限失效 |
| **完成实名认证** | 域名已完成实名认证 |

> **重要**: 提交修改时，请务必勾选关闭"域名信息修改成功后 60 天内禁止转出阿里云"选项，否则操作成功后 60 天内域名将无法转出阿里云。

## 操作流程

> **重要**: 如需迁移域名解析记录，建议在转出前将域名的 DNS 修改为新服务商的 DNS 并做好解析，避免域名转出后无法正常解析。

### 步骤一：获取域名转移码

也称**域名授权码**、**转移密码**、**Auth Code**、**EPP Code**、**Auth Info**

1. 登录**域名控制台域名列表页面**，找到您希望转出的域名，点击右侧操作列的**管理**
2. 在左侧导航栏中，选择**域名转出**
3. 根据系统引导确认域名持有者邮箱，并完成手机号码验证
4. 验证通过后，单击**下一步**
5. 系统会将包含转移密码的邮件发送到该域名的持有者邮箱

> **说明**: 
> - ".hk"域名不支持线上转出
> - 域名转移码是授权域名转移的唯一凭证，**有效期为 15 天**，请在获取后尽快使用

### 步骤二：在新注册商平台提交转入申请

1. 登录您选择的新注册商的官方网站，找到"域名转入"或"Transfer Domain"相关功能入口
2. 按照新注册商的指引，输入您要转入的域名和上一步获取的转移码
3. 完成支付（此步骤通常会涉及支付一年域名续费的费用）
4. 检查您的邮箱，接收转入注册商的确认邮件

### 步骤三：等待或确认域名转移

在新注册商处提交申请后，阿里云会收到转移请求。

| 转出方式 | 预计耗时 | 关键操作 |
|----------|----------|----------|
| **普通转出（建议）** | 5~7 个工作日 | 获取转移密码、在新服务商处提交申请后，无需任何操作，等待系统自动完成 |
| **快速转出（可选）** | 约 1 小时 | 在标准流程基础上，需要返回阿里云控制台主动确认，并完成邮件二次确认 |

#### 快速转出操作

> **重要**: 此操作将导致域名被立即转移且无法撤销，转出后域名将不再属于阿里云平台。为确保域名安全，防范域名欺诈，请您谨慎操作。

1. 返回控制台域名列表，当目标域名状态变为**正在转出**后，再次进入**域名转出**页面
2. 点击**立即转出**按钮，并按提示发送确认邮件
3. 登录域名持有者邮箱，找到标题为"阿里云转出中域名立即确认转出邮件通知"的邮件，单击邮件中的**确认链接**
4. 在打开的页面中，最终单击**确认转出**

域名通常约 1 小时内完成转移，如时间过长可咨询新注册商。

## 后续操作

### 确认转出状态

您可以随时通过**域名信息查询**（WHOIS）工具检查域名状态。

| 状态 | 说明 |
|------|------|
| **pendingTransfer** | 域名待转移 |
| **注册商显示为非阿里云** | 域名已成功转出 |

此后，请登录新注册商的平台管理您的域名。

### 取消转出

如果您在转出过程中改变主意，只要域名状态仍是"正在转出中"，且您未进行"快速转出"的邮件确认，可通过以下方法取消域名转出：

1. 登录阿里云域名控制台，单击待转出的域名，进入基本信息页面
2. 在左侧菜单栏单击**域名转出 > 取消转出**

> **说明**: 如果域名转出页面没有取消转出的按钮，说明此域名已经转出成功。一旦转出完成，则无法撤销，需等待 60 天后才能再次将域名转入阿里云。

## ".hk"域名转出

.hk 域名的转出流程不由 ICANN 统一管理，需要线下人工处理。

### 操作步骤

1. **准备材料**：
   - 下载并填写《.hk 域名转出申请表》，需由域名持有者和域名所在阿里云账号的持有者共同签字、盖章

2. **准备身份证明材料的原件彩色照片**：
   | 持有人类型 | 材料要求 |
   |------------|----------|
   | **个人** | 身份证正反面 |
   | **企业** | 营业执照副本 |
   | **事业或政府单位** | 组织机构代码证 |

   > **拍照要求**: 为满足注册局对资料真实性的审核要求，请确保您的照片能证明是对证件原件的真实拍摄（例如，将证件放在桌面上，照片中带有部分桌面背景）。不支持扫描、复印、打印等形式。

3. **发送邮件**：
   - 将填写好的申请表和身份证明材料照片，发送至邮箱：`cndomain-notice@service.alibaba.com`

4. **后续操作**：
   - 阿里云在收到合格的材料后，会在 3 个工作日内通过邮件回复您后续的操作方法

## 常见问题

### 域名转出后如何管理域名？

域名转出后，您需要在新的注册商平台进行域名管理，请确保您已获得新平台的登录信息。

### 我收不到获取转移密码的邮件怎么办？

请按以下顺序排查：
1. 确认您在控制台看到的"域名持有者邮箱"地址是否准确无误
2. 检查该邮箱的垃圾邮件或广告邮件文件夹
3. 如果邮箱确认无误，可以在获取转移密码的页面尝试点击"重新获取转移码"

### 提交的授权码无效怎么办？

请检查：
- 授权码是否已超过 15 天有效期。如果是，请重新获取
- 复制授权码时是否包含了多余的空格或字符，区分字符如 0 和 O、全角/半角符号。请确保完整、精确复制
