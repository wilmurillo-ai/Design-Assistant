## 场景一：域名持有者信息修改（过户）

### 何时使用此操作

- 当需要将域名所有权转让给另一个组织或个人时
- 当发现当前持有者的姓名、地址、联系方式等信息有任何错误，需要更正时
- 如果仅需修改持有者信息（如手机号码、通讯地址等），与过户操作相同（但持有者保持不变）

### 核心影响

| 影响项 | 说明 |
|--------|------|
| **重新实名认证** | 新信息需提交审核，耗时 **3~5 个工作日** |
| **备案状态变更** | 过户完成后，若域名持有者信息将与工信部备案系统中的信息不一致，**必须立即登录备案系统提交变更备案申请**。若不及时处理，可能导致网站被管局关停 |
| **60 天禁止转出（可关闭）** | 过户成功后，域名将被锁定 60 天，无法转至其他注册商。如需转出请在操作时勾选关闭 |

### 操作步骤

参见：[域名持有者信息修改（过户）](https://help.aliyun.com/zh/dws/user-guide/modify-registrant-contact-information-or-transfer-domain-name-ownership)

## 场景二：修改域名其他联系人信息

此操作适用于仅变更域名的管理者、技术者或付费者的名称、联系方式（如邮箱、电话），不改变域名所有权。

### 核心影响

| 影响项 | 说明 |
|--------|------|
| **风险较低** | 不触发重新实名认证 |
| **不影响网站备案** | 无需变更备案 |
| **不会导致 60 天禁止转出** | 可正常转出域名 |
| **操作实时生效** | 修改后立即生效 |

## 场景三：修改域名 DNS 服务器

此操作适用于需要更换域名解析服务商的场景，例如从第三方 DNS 服务商迁移到阿里云云解析 DNS。

### 核心影响

DNS 服务器地址变更后，全球各地的递归 DNS 服务器需要时间（**最长可达 48 小时**）刷新缓存。若准备不充分，部分用户可能会访问到旧的服务器，导致服务中断或访问不稳定。

### 平滑迁移方案

为避免中断，操作前务必在**新 DNS 服务商处配置好所有必需的解析记录**，并确保与旧 DNS 服务商的记录一致。

### 操作步骤

参见：[修改 DNS 服务器](https://help.aliyun.com/zh/dns/manage-dns-servers)

## 概念辨析："DNS 服务器"与"解析记录"

这是两个极易混淆但性质完全不同的操作：

| 操作 | 说明 | 比喻 |
|------|------|------|
| **修改 DNS 服务器** | 更换为域名提供解析服务的"导航服务商" | 更换导航软件（如从百度地图换到高德地图） |
| **修改 DNS 解析记录** | 在当前"导航服务商"处，修改域名指向的具体"目的地"（如服务器 IP 地址） | 在同一个导航软件中修改目的地地址 |

> **注意**: 修改解析记录不涉及服务商的变更。如需修改解析记录，请参见 [添加网站解析](https://help.aliyun.com/zh/dns/quick-start)。

## 不可修改项说明

**域名名称本身不支持任何形式的修改。**

如果因拼写错误注册了不正确的域名，该操作无法撤销，也无法直接修改为正确的域名。

### 解决方法

1. **重新注册**一个正确的域名
2. 已注册的错误域名在到期后不续费即可

### 减少损失

- 若域名刚注册，可尝试**域名付费删除**
- 也可考虑将域名进行**出售**
