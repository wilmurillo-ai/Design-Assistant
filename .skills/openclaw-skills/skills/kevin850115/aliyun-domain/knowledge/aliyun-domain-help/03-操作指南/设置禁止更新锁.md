## 📄 文档内容

为防止您的域名因账号被盗、操作失误等原因，导致域名注册信息（如 DNS 服务器、联系人信息等）被恶意篡改或转移，您可以通过开启**禁止更新锁**来锁定域名信息，保障域名安全。

---

## 功能说明

开启禁止更新锁后，您的域名将被置于锁定状态，以下关键注册信息将**无法通过控制台或 API 进行修改**：

| 锁定项 | 说明 |
|--------|------|
| **域名持有者信息** | 持有者姓名、证件号、联系方式等 |
| **域名管理者、付费者、技术者信息** | 各类联系人信息 |
| **DNS 服务器** | 域名解析服务器设置 |

**状态查询**: 成功开启后，通过 WHOIS 查询时，域名的状态将显示为"**注册商设置禁止更新**"（clientUpdateProhibited）。

> **注意**: 禁止更新锁仅保护域名的注册信息，**不影响**您在云解析 DNS 控制台对具体的解析记录（如 A 记录、CNAME 等）进行增加、删除、修改等操作。

---

## 开启或关闭禁止更新锁

您可以根据业务需求，随时开启或关闭域名的禁止更新锁。

> **重要**: 在进行域名持有者信息修改（过户）、修改 DNS 服务器等操作前，**必须先关闭此锁**。操作完成后，为保障安全，建议您尽快重新开启禁止更新锁。

### 操作步骤

1. 登录**阿里云域名控制台**
2. 在域名列表页面，找到目标域名，单击操作列下的**管理**
3. 在左侧导航栏，单击**安全设置**
4. 在**禁止更新锁**区域，单击**开启**或**关闭**
5. 在弹出的确认对话框中，完成安全验证：
   - 单击**获取验证码**
   - 输入收到的验证码
   - 单击**确认**

## 常见问题

### 开启禁止更新锁后，想修改域名信息怎么办？

开启禁止更新锁后，如果想修改域名的信息，需要**先关闭禁止更新锁服务**。域名信息修改完成后，建议尽快重新开启禁止更新锁。

### 域名过期后，禁止更新锁是否有效？

禁止更新锁服务依赖于域名的有效状态。**域名到期后，禁止更新锁会随之失效**，此时域名信息将面临被修改的风险。

**建议**: 及时为域名续费，并在续费或赎回成功后，尽快重新开启禁止更新锁。
