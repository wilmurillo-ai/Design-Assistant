## 📄 文档内容

根据中国内地监管要求，所有域名必须完成实名认证，才能正常解析和使用。若未完成认证，域名将被注册局锁定（域名状态为 serverHold），导致网站无法访问。

本文档指导如何在阿里云为域名完成实名认证。**在阿里云平台完成域名实名认证不收取费用。**

域名实名认证包含两个关键步骤：
1. **创建信息模板并通过认证**：创建并提交包含域名持有者身份资料的信息模板以供审核
2. **将模板关联至具体域名**：将审核通过的信息模板关联到一个或多个域名

## 第一步：创建信息模板

"信息模板"是用于管理域名持有者信息的身份档案。其价值在于"一次创建，多处复用"，你名下所有域名都可关联至此模板，便于统一管理。

### 前提条件

在注册域名前请确认完成阿里云账号实名认证。

### 操作步骤

1. 登录阿里云域名控制台 - 信息模板页面，单击**创建新信息模板**
2. 根据界面提示，完成域名持有者信息填写

#### 企业/组织填写参数

| 参数 | 说明 |
|------|------|
| 统一社会信用代码 | 与营业执照上完全一致的证件号 |
| 域名持有者单位名称 | 必须与上传的证件上的名称完全一致，包括标点符号 |
| 所属区域 | 与证件上的注册地址保持一致 |
| 通讯地址 | 与营业执照上一致 |
| 邮编 | 企业所在地的邮政编码 |
| 电子邮箱 | 填写真实有效的邮箱并完成邮箱验证 |

#### 个人填写参数

| 参数 | 说明 |
|------|------|
| 域名持有者姓名 | 必须与上传的身份证明文件上的姓名完全一致 |
| 所属区域 | 选择中国内地真实有效的通讯地址所在区域 |
| 通讯地址 | 填写中国内地真实有效的通讯地址 |
| 邮编 | 通讯地址所在地的邮政编码 |
| 手机号码 | 国家代码：中国内地区域填写 86 |
| 电子邮箱 | 填写真实有效的邮箱并完成邮箱验证 |

3. 确认信息无误后，单击**提交**

### 审核时间

- 信息模板审核通常需要**1 个工作日**，部分可能需要**3～5 个工作日**
- 在信息模板页面查看模板状态是否为**模板实名成功**

---

## 第二步：将信息模板关联至域名

当信息模板状态显示为"模板实名成功"后，须将其关联至域名，才能完成整个认证流程。

### 场景 1：为新注册的域名认证

在域名购买的订单确认页面，选择已实名成功的模板即可。支付后，系统会自动为您提交认证。

### 场景 2：为已有的域名认证

1. 在域名列表页找到未实名认证的域名
2. 单击目标域名
3. 在**域名持有者实名认证 > 快速认证**页，选择已通过认证的模板进行关联

---

## 第三步：查询域名实名认证状态

关联操作提交后，系统会立即向注册局同步信息。

### 审核时间

- 域名关联模板后的实名认证审核，通常在**1 天内**完成，部分可能需要**3～5 个工作日**

### 验证方法

返回域名列表页面，观察目标域名的状态。如果状态显示为**正常**，表示域名已成功完成实名认证，可进行备案和解析配置。

---

## 域名状态说明

| 域名状态 | 含义 | 常见原因及解决方案 |
|----------|------|-------------------|
| **clientHold** | 注册商锁定 | 通常由阿里云设置。原因可能是域名涉及违规或争议。解决方案：检查阿里云站内信或邮件通知，按要求处理 |
| **serverHold** | 注册局锁定 | 最常见的原因是未通过实名认证。也可能因域名指向非法网站或被要求强化实名。解决方案：提交实名认证，认证成功后状态会自动解除（可能会有 1 个工作日的延迟） |

## 常见问题

### 为什么我的域名状态为未实名？

请根据以下步骤进行检查：
1. 信息模板是否已通过认证？→ 信息模板页面查看模板状态是否实名成功
2. 域名是否已成功关联上这个模板？→ 域名列表页点击目标域名进行关联

### 实名认证审核需要多长时间？

- 信息模板通常在**1 个工作日内**完成审核，部分可能需要**3～5 个工作日**
- 域名关联模板后的审核，通常在**1 天内**可完成审核，部分可能需要**3～5 个工作日**

### 信息模板实名认证失败了怎么办？

在信息模板页面，单击模板右侧查看，根据系统提示的失败原因，重新上传资料，待模板审核通过后，再为域名提交认证。

### 域名实名认证后，网站仍无法访问或无法解析？

1. 域名未完成备案、以及设置域名解析：请先完成域名的备案和解析设置
2. 已设置域名解析：查看域名状态、域名解析 DNS 服务器设置是否正确
