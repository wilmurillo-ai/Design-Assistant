## 各防护能力简介

### 多因素认证（MFA）

**MFA**（Multi-Factor Authentication，多因素认证）是一种提升账号安全性的最佳实践，它能够在用户名和密码之外再额外增加一层安全保护。

**启用 MFA 后，登录阿里云时需完成以下两步验证**：

| 验证步骤 | 说明 |
|----------|------|
| **第一重验证** | 输入您的账号名和密码 |
| **第二重验证** | 虚拟 MFA 每 30 秒自动生成的 6 位动态验证码 |

通过双重认证，即使密码泄露，未持有您手机的人也无法登录您的账号，有效防止账号被盗，极大提升安全性。

**具体操作**: [配置账号的 MFA](https://help.aliyun.com/zh/account/user-guide/configure-mfa)

### 禁止转移锁

开启后域名将被置为**注册商禁止转移状态**（clientTransferProhibited），避免您的域名被恶意转出阿里云。

> **说明**: 如需获取域名转移密码，需先关闭禁止转移锁。

**具体操作**: [设置禁止转移锁](https://help.aliyun.com/zh/dws/user-guide/enable-the-transfer-prohibition-lock)

### 禁止更新锁

开启后可防止您的域名注册信息（域名联系人、电话、地址、传真、电子邮箱）、域名 DNS 服务器被恶意篡改。

**支持的后缀**：
- `.com`、`.net`、`.org`、`.info`、`.biz`
- `.mobi`、`.asia`、`.me`、`.so`、`.cc`
- `.tv`、`.name`、`.tel`、`.cn`、`.中国`
- `.公司`、`.网络` 等

**具体操作**: [开启禁止更新锁](https://help.aliyun.com/zh/dws/user-guide/enable-the-update-prohibition-lock)

### 注册局安全锁

注册局安全锁是目前**最高等级的域名安全保护措施**，由注册局在根服务器层面操作，禁止域名被恶意转移、篡改及删除。

**支持的后缀**：
- `.com`、`.cn`、`.net`、`.cc`、`.tv`
- `.name`、`.中国`、`.gov.cn` 等

**开启后的锁定状态**：

| 状态 | 说明 |
|------|------|
| **serverDeleteProhibited** | 注册局设置禁止删除 |
| **serverTransferProhibited** | 注册局设置禁止转移 |
| **serverUpdateProhibited** | 注册局设置禁止更新 |

如果需要对域名做任何状态的变更及信息更改，需先解除对应的锁定状态。

**具体操作**: [使用注册局安全锁](https://help.aliyun.com/zh/dws/user-guide/use-registry-security-lock)

### DNSSEC

**域名系统安全扩展**（DNS Security Extensions，简称 DNSSEC）是用于确定源域名可靠性的数字签名。

通过在域名中添加 DNSSEC 记录，可以增强对 DNS 域名服务器的身份认证，有效防止 DNS 缓存污染等攻击。

**具体操作**: [配置 DNSSEC](https://help.aliyun.com/zh/dns/configure-dnssec)
