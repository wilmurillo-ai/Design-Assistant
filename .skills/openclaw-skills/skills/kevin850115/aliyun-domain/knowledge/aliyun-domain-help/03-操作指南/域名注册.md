## 注册前的准备

### 创建信息模板

在阿里云域名控制台信息模板页面，提前创建并等待信息模板审核通过。信息模板记录域名持有者的身份资料（姓名/企业名、证件号等），域名注册时要求将域名关联到信息模板上。

> **提示**: 提前创建信息模板并完成审核认证，可加速注册过程。

## 域名注册流程

注册一个标准域名通常遵循以下三个核心步骤：

```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│  步骤一：查询   │ →  │  步骤二：支付   │ →  │  步骤三：验证   │
│  并选择域名     │    │  并完成支付     │    │  与后续操作     │
└─────────────────┘    └─────────────────┘    └─────────────────┘
```

### 步骤一：查询并选择域名

1. 访问域名注册平台，在搜索框中输入您想注册的域名（如"wangwang.cc"），单击查询域名
2. 根据查询结果，判断域名的可用状态：

| 状态 | 说明 | 操作 |
|------|------|------|
| **可注册** | 域名右侧有立即注册按钮 | 直接注册 |
| **已注册** | 域名已被他人持有 | 单击委托询价 |
| **可购买** | 域名右侧有立即购买按钮 | 直接购买 |

> **重要**: 根据注册局 CNNIC 相关规定，若域名包含"中国"、"国家"、"中华"、"全国"、"china"、"chn"、"sino"等字样，则该域名必须由相关权益主体申请注册。

### 步骤二：确认订单并完成支付

在确认订单页面，完成以下配置：

| 配置项 | 说明 |
|--------|------|
| **购买年限** | 选择注册时长，可选 1-10 年。对于品牌核心域名，建议一次性注册多年（如 5-10 年） |
| **信息模板** | 电子邮箱需已验证，且模板实名成功 |
| **优惠口令（可选）** | 可享受促销优惠 |

阅读并同意《域名服务条款》，点击立即购买并完成支付。

> **重要**: 支付成功不等于注册成功。完成支付仅代表您的注册请求已成功提交至阿里云。部分域名需要经过命名审核或资质审核，此过程可能需要 1-3 个工作日。只有当域名状态最终变为"正常"时，才代表注册彻底成功。

### 步骤三：验证注册结果与后续操作

#### 查看域名是否注册成功

在域名控制台域名列表页面，域名状态为**正常**，则代表域名注册成功可正常使用。

#### 注册成功后的关键后续步骤

1. **设置域名解析**：域名注册后，需通过设置解析将域名指向服务器的 IP 地址，用户才能通过该域名访问网站或应用
2. **完成 ICP 备案**：根据中国内地法律法规，任何在中国内地提供服务的网站都必须先完成 ICP 备案才能上线

## 注册特殊类型域名

### ".商标"域名

**前提条件**: 注册".商标"域名前，请确保您已持有符合要求的商标注册证书。

**操作步骤**:
1. 登录".商标"商标域名注册页面，输入您想要注册的域名名称，单击查询
2. 在查询结果中，单击立即注册
3. 选择域名的购买年限和信息模板后，发起支付
4. 完成支付后，域名注册订单会短暂的显示为处理中状态，待订单状态变为交易完成，域名即注册成功

### 党政机构".gov.cn"域名

**.gov.cn 域名仅限我国党政机关及依法行使党政机关职能的单位注册**。

#### 注册所需材料

| 材料 | 说明 |
|------|------|
| **域名持有者身份证明材料** | 组织机构代码证或统一社会信用代码证原件彩色扫描件 |
| **联系人身份证明材料** | 身份证正反两面的原件彩色扫描件，拼接为一张图片文件 |
| **《域名注册信息申请表》** | 下载并填写，加盖公章后上传至系统 |

#### 注册步骤

1. 登录阿里云域名注册控制台
2. 输入您想要注册的域名前缀，并选择".gov.cn"后缀，单击查询域名
3. 单击相应域名后的立即申请，进入".gov.cn"域名申请资料提交流程
4. 按照页面资料项目要求填写并上传".gov.cn"注册初审材料，并单击提交审核
5. 等待初审资料合格后，系统会自动生成《.gov.cn 域名注册申请表》
6. 下载并打印申请表，完成其他信息填写、加盖申请单位公章
7. 拍照或扫描已完成信息填写和加盖公章的申请表，并上传至系统
8. 确认域名信息无误后，勾选同意《域名服务条款》并提交，完成支付
9. 域名订单提交后，域名注册局（CNNIC）在审核资料的同时，会电话核实注册意愿，请保持联系电话畅通
10. 注册局审核通过后，域名注册成功

## 费用说明

### 注册费用

域名的价格由后缀和注册年限决定，不同后缀的价格差异巨大。

### 注册错误付费删除

如果您注册了错误的域名，部分后缀支持在"删除宽限期"内进行有偿删除。此期限通常为注册后的 3-5 天内，具体视后缀而定。

> **重要**: 删除操作会产生一笔不可退还的手续费，剩余的注册费用将被退还。并非所有后缀都支持，且手续费各不相同。

## 常见问题

### 我已经付款了，为什么域名列表里找不到我的域名？

访问企业服务产品订单页面，查看购买域名的订单状态是否为成功。阿里云当日会对处理中的订单进行审核，如果审核发现注册信息填写有误，会协助您修改后重新提交注册局审核。

### 如何查看域名的购买订单信息？

企业服务产品订单页面可查看购买域名的订单信息：包含购买域名的价格、订单状态、域名购买时间等。

## 附录：域名命名规则

### 中文域名

- 不能是纯英文或数字，需至少包含 1 个汉字
- 其余部分可包含英文字母（a~z，不区分大小写）、数字（0~9）以及连接符（-）
- 开头和结尾不能有非法字符
- 域名长度不超过 63 个字符

### 英文域名

- 只能使用英文字母（a~z，不区分大小写）、数字（0~9）以及连接符（-）
- 不支持使用空格及以下字符：!$&?等
- 连接符（-）不能单独注册，也不能放在开头和结尾
- 域名长度不超过 63 个字符

### ".hk"域名

- **二级域名（***.hk）**：不分中国香港、中国内地或海外，任何单位和个人均可申请注册二级域名，无需提供证明文件（除银行、保险行业外）
- **三级域名（***.com.hk 等）**：暂不支持注册
