## 详细解答

### 域名转入

#### 什么是域名转入？

域名转入阿里云，是指将您当前在其他域名注册商（非阿里云）注册的域名，转移到阿里云进行管理。转移成功后，阿里云将作为您的域名注册商，为您提供域名续费、信息管理、安全设置等相关服务。

#### 使用中的域名如何实现平滑转入？

**前提条件**: 关闭 DNSSEC（如已配置）

**操作步骤**:

1. **添加域名**: 在云解析 DNS 控制台添加要迁移的域名
2. **添加解析记录**: 
   - 导出原解析记录
   - 填写导入模板
   - 导入解析记录到云解析 DNS
3. **修改 DNS 服务器地址**: 
   - 检查并关闭禁止更新锁
   - 在域名注册商处修改 DNS 服务器为阿里云解析 DNS 地址
   - 等待 48 小时全网生效

> **警告**: 修改 DNS 服务器 48 小时内，请尽量不变更解析记录。

#### 域名转入规则

| 规则项 | 要求 |
|--------|------|
| **支持转入的后缀** | 在域名价格总览页面转入列显示价格 |
| **注册时间** | 必须大于 60 天 |
| **到期时间** | 距离到期日必须大于 15 天，小于 10 年 |
| **域名状态** | 不能处于禁止转移、暂停解析等异常状态 |
| **法律状态** | 未处于任何纠纷、仲裁或法院限制转出的状态 |
| **续费状态** | 续费年限未达到上限（10 年） |
| **续费时间** | 近期续费的，请在续费成功 45 天后再转入 |

#### 域名转入收费说明

- **转入操作**: 免费
- **续费要求**: 根据 CNNIC 规定，域名转移注册商必须将域名的到期日延续一年
- **转入价格**: 通常为域名续费一年的价格

> **说明**: ".hk"域名转入不收费，域名到期日不延续。

#### 域名转入后到期时间没有延长一年的原因

**可能原因一**: 域名过期后已成功续费，但距离成功续费不满 45 天便转移了注册商

**处理方法**: 联系原域名注册商退还域名续费费用

**可能原因二**: 原注册商还未在注册局进行续费，域名就已转入阿里云

**处理方法**: 联系原注册商向注册局核实域名转移前的真实到期日，并退还续费费用

#### 域名被原注册商锁定，转入后如何解锁？

**准备资料**:
1. 域名解锁的免责声明
2. 域名持有者的身份证明材料

**发送邮箱**: `cndomain-notice@service.alibaba.com`

阿里云收到资料后，会在三个工作日内处理。

### 域名转出

#### 域名转出规则

| 规则项 | 要求 |
|--------|------|
| **注册商** | 必须是阿里云 |
| **注册时间** | 距离域名申请日 60 天以上 |
| **到期时间** | 距离到期日大于 15 天 |
| **转移间隔** | 距离最后一次成功转移注册商大于 60 天 |
| **域名状态** | 正常，不能是禁止转出状态 |
| **法律状态** | 不欠费、不处于任何仲裁及法律程序中 |
| **续费时间** | 最后一次续费 45 天后 |

#### 域名转出可以终止吗？

如果您操作了域名转出，且域名转出申请还未通过，可取消转出：

1. 登录阿里云域名控制台
2. 单击待转出的域名，进入基本信息页面
3. 在左侧菜单栏单击**域名转出 > 取消转出**

> **重要**: 如果域名转出页面没有取消转出的按钮，说明此域名已经转出成功，无法取消。

#### 如何查看域名转出状态？

通过**WHOIS 查询**查看域名的注册商信息。当域名的注册商显示为非阿里云，则转出成功。

### 域名转移密码

#### 什么是域名转移密码？

域名转移密码又称为**域名转移码**、**授权码**（Authorization code）或**域名 EPP 代码**，在国外又称 EPP Code 或 Auth Code 等，每个域名都有自己的转移密码。

#### 转移密码有效期

域名转移密码的有效期为**15 天**，获取后需尽快在新注册商处提交域名转入。
