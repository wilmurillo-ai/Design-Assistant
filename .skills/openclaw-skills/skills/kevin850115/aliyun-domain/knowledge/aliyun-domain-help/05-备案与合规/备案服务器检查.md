## 📄 文档内容

根据《非经营性互联网信息服务备案管理办法》规定，网站/App是否需要备案，取决于服务器所在地域。**若使用中国内地服务器，您必须在服务器所属接入商平台完成备案。**

## 使用须知

在阿里云备案必须关联或购买阿里云中国内地节点服务器：

- **如果您的网站/App使用的阿里云云资源位于中国内地**：阿里云作为网络接入服务者会为您提供备案服务
- **如果云资源位于中国香港、海外等节点**：则无需备案
- **如果使用其他厂商服务器**（如"自建机房"、"华为"、"腾讯"等）：请咨询对应厂商进行备案

## 备案服务器与备案服务码

备案服务码是阿里云备案系统提供的一串唯一的随机码，用于标识**备案服务器与备案网站/App的关联关系**。

### 如何获取备案服务码

请您先确保备案服务器已满足可备案服务器要求，若没有服务器或不满足要求，请及时续费或购买。

| 是否为同一账号 | 获取服务码方式 | 下一步 |
|:---|:---|:---|
| **备案账号与服务器所在账号是同一阿里云账号** | 备案时无需提前获取，系统会自动生成服务码 | 进入备案流程，填写备案信息 |
| **备案账号与服务器所在账号不是同一阿里云账号** | 在备案前，由服务器所在的企业认证账号授权，获取服务码<br><br>**说明**: 个人认证账号不能给他人授权服务码。请将个人认证变更为企业认证后再授权服务码 | 接收企业账号授权的服务码后，进入备案流程，填写备案信息 |

### 查看备案服务码状态&数量

使用服务器所在阿里账号登录 [服务码管理控制台](https://bsn.console.aliyun.com/#/bsnManagement) 查看服务码状态&数量。

- **服务器类型不同服务码数量不同**，建议您在可备案实例管理查看免费备案服务码数量。若已达上限，可购买服务码并绑定实例以扩充备案数量，单独购买服务码无法完成备案操作
- **本账号生成的服务码**：通过本阿里云账号购买的云产品生成的服务码、购买的收费服务码、以及其他途径直接生成的服务码
- **他人授权给我的服务码**：通过企业认证账号授权给本账号的服务码

### 服务码状态说明

| 服务码状态 | 您的操作&注意事项 |
|:---|:---|
| **可使用** | 未用于ICP备案，您可用于ICP备案 |
| **未绑定实例** | 该服务码未绑定云服务（仅收费的服务码会有此状态） |
| **备案中** | 该备案服务码已用于ICP备案，当前正在备案中 |
| **已备案** | 该备案服务码已备案成功，无法退还，不可再重复使用 |
| **已退费** | 该备案服务码已失效，无法用于ICP备案（仅收费服务码会存在此状态） |
