## 📄 文档内容

根据《工业和信息化部关于规范互联网信息服务使用域名的通知》，阿里云ICP代备案管理系统会对域名注册信息进行核验，**如果您未能通过域名核验，阿里云将按照通信管理部门的要求停止为您提供接入服务**。本文为您介绍域名核验的范围和要求。

---

## 域名核验的范围

首次备案、变更备案等不同备案类型下，会核验不同范围的域名，在核验范围内的域名，都必须满足核验要求。

| ICP 备案类型 | 域名核验范围 | 备注 |
|:---|:---|:---|
| **首次备案** | 核验全部域名 | 无 |
| **新增互联网信息服务** | 核验新增的全部域名 | 无 |
| **变更备案** | 核验对应主体下 2018 年 1 月 1 日后备案成功的域名 | 部分省份管局会核验您的域名后缀在 2018 年 1 月 1 日后是否获得工信部批复 |
| **注销主体** | 不进行域名核验 | 注销备案后，如果重新提交域名备案，需通过备案域名核验 |
| **注销互联网信息服务** | 不进行域名核验 | - |
| **取消接入** | 不进行域名核验 | 取消接入后，再接入暂不进行域名核验 |

## 域名核验要求

ICP备案的域名需同时满足以下要求，方可完成备案申请。若域名仅解析至非中国内地的服务器，则无需进行备案。

### 要求一：域名后缀（顶级域名）获得工信部批复

访问 [工信部网站](https://domain.miit.gov.cn/) 的**中国互联网域名体系**页面，查看已批复的域名后缀（顶级域名），未列入列表中的域名无法进行ICP备案。

**常见可备案域名后缀：**
- **国家顶级域名**: .cn, .com.cn, .net.cn, .org.cn 等
- **国际通用域名**: .com, .net, .org, .xyz, .top, .club, .vip 等
- **中文顶级域名**: .公司，.网络，.商城，.网址等

### 要求二：域名的注册商为已批复机构

1. 通过 [域名信息查询（WHOIS）](https://whois.aliyun.com/) 页面获取当前域名的注册商信息
2. 前往 [工信部网站](https://domain.miit.gov.cn/) 的**域名注册服务机构审批**情况页面，查询该注册商的批复状态
   - **当域名注册商未获批复时**：需要将域名转移至已获批复的域名注册商
   - **当域名注册商获批复时**：可直接进行备案

**说明**: 阿里云国际站域名服务（alibabacloud.com）为国际域名注册商。在此站点注册的域名需转入中国内地域名注册商，并完成域名实名认证才可以进行ICP备案。

### 要求三：域名持有人信息与备案主体信息一致

**1. 域名实名认证同步时间**
- **阿里云注册的域名**：可直接提交备案申请，阿里云备案系统会自动延期并提交
- **非阿里云注册的域名**：需要等待 2~3 天，再提交备案申请

**2. 域名有效期要求**
用于备案的域名有足够长的有效期，至少确保域名距离到期日期**大于 45 天**。如果不满足要求请您先域名续费。

**3. 域名实名信息和主办者信息一致**

**个人备案**：域名实名信息（持有人名称、证件类型、证件号码）需与主办者信息保持一致。

**企业备案**：域名实名信息（持有人名称、证件类型、证件号码）需与主办单位信息或主体负责人一致。且**实名认证证件类型**必须与**主办单位证件类型**吻合。

**常见信息不一致情况及解决方法：**

| 问题类型 | 解决方法 |
|:---|:---|
| 域名认证信息为个人，无法为企业备案 | 1. 先创建域名信息模板<br>2. 将域名持有者信息修改（过户）后再进行备案申请<br>**说明**: 认证信息修改后需等待 2~3 个工作日用于信息同步 |
| 域名认证信息没有证件类型和证件号码 | 同上 |
| 域名认证信息有证件类型，但和备案主体的证件类型不一致 | 同上 |
| 域名持有者（中文）的单位名称中使用了英文括号 | 修改为中文括号 |

**说明**: 在进行ICP备案时，只需对主域名（如 `example.com`）进行备案。备案完成后，其下的所有子域名（如 `www.example.com`）无需额外备案即可正常使用。
