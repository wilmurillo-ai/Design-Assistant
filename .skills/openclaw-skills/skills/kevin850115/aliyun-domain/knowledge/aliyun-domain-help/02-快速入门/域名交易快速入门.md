## 📄 文档内容

当您无法通过常规注册途径获取心仪的域名时，可通过域名交易选择购买他人注册的、尚未到期的域名。

**阿里云域名交易平台**（mi.aliyun.com）是阿里云基于支付宝担保交易打造的域名交易系统，为您提供域名抢注、一口价、竞价等多种交易类型，为买卖双方提供安全、便捷的域名交易服务。

### 前提条件

已完成域名实名认证。

## 我要买域名

### 场景一：购买一口价域名

**什么是一口价（严选）域名**

一口价（严选）是阿里云域名交易平台中的一种交易类型，一口价（严选）域名的出售信息由阿里云域名用户直接发布，阿里云全程参与交易过程，并为买家开具发票。

**交易须知**

| 角色 | 说明 |
|------|------|
| **卖家** | 一口价（严选）仅限卖家发布其在阿里云注册的域名或者卖家在其他平台注册管理的域名。卖家发布域名前，阿里云账户需绑定支付宝，且完成支付宝的实名认证 |
| **买家** | 通过一口价（严选）购买的是已注册的二手域名，其价格高于未被注册域名的价格。生成订单后您需及时付款，未及时付款可能导致域名被其他人购买 |

**购买流程**

1. 进入域名交易首页，选择一口价
2. 找到想要购买的域名，单击直接购买
3. 选择实名认证成功的信息模板，并勾选相关协议后，单击立即购买
4. 选择一种支付方式，单击支付，完成域名购买

> **说明**: 购买一口价（严选）域名并完成付款后，订单会显示处理中状态。当域名购买成功后，域名的到期时间不会增加一年，您需关注域名的到期时间并及时续费。

**批量购买**

1. 进入域名交易首页，选择一口价
2. 勾选所有想要购买的域名，在页面下方单击批量购买
3. 选择实名认证成功的信息模板，并勾选相关协议后，单击立即购买
4. 选择一种支付方式，单击支付，完成域名购买

### 场景二：域名回购

**什么是域名回购**

域名回购是由阿里云及其合作方根据用户提出的具体域名的购买需求，为用户提供询价、议价、分析、谈判、域名收购、域名转移等的一系列服务的统称。

**回购流程**

```
申请 → 沟通 → 谈判 → 成交
```

1. **提交域名回购申请**：登录阿里云域名控制台，单击域名回购 > 发布域名回购需求
2. **域名专家与买家沟通**：阿里云域名专家在收到买家的域名回购需求后，会在 2 个工作日内与买家电话联系确认域名购买需求
3. **域名专家与卖家谈判**：域名专家将尽力为买方寻求一切域名回购机会
4. **域名回购成功，完成交易**：域名专家将确保提供一个可靠的域名过户流程，保证买卖双方的交易快捷、安全

### 场景三：域名抢注

**什么是域名抢注**

域名抢注是由阿里云提供的、经由合作方渠道的过期待删除域名的预订、竞价服务。

**交易须知**

- 域名预订成功后无法取消
- 如果您的域名交易功能被冻结，则此阿里云账号无法参与域名预订
- 竞拍时间为 3 天，竞拍结束前 5 分钟内，如果有新的出价系统会自动延长 5 分钟

**抢注流程**

1. 进入域名交易首页
2. 单击抢注域名查询或抢注预定页签
3. 找到需要预订的域名，单击预订
4. 根据界面提示，完成保证金冻结后域名进入已预订状态
5. 域名预订成功，等待域名开始竞拍
6. 竞拍结束后，得标者需在 72 小时内完成订单支付

> **注意**: 如果域名竞拍成功，买家在规定时间内未完成付款将造成违约。违约用户会被没收相应的已冻结的保证金。违约行为超过三次，将被暂停域名交易 30 天。

## 我要卖域名：发布域名在售信息

域名持有者可以通过域名控制台发布域名一口价售卖信息，阿里云支持域名持有者通过域名控制台发布阿里云注册商域名和非阿里云注册商域名。

### 前提条件

您的阿里云账号需先绑定已实名的支付宝账户。

### 发布前检查

#### 阿里云注册商域名

| 检查项 | 要求 |
|--------|------|
| **域名账号** | 域名需在当前登录的阿里云账号下，域名不可以限制账号间转移 |
| **域名类型** | 微站域名、".gov.cn"域名、微海报".cn"域名等特殊要求的域名后缀，均不能发布域名在售信息 |
| **到期日** | 域名未过期且距离域名到期日大于 7 天 |
| **域名状态** | 域名如果处于 serverhold 状态，需先完成域名实名认证或解除 serverhold 状态 |
| **安全锁** | 域名开启了禁止更新锁的将不能正常过户和交易，请先关闭禁止更新锁 |
| **业务状态** | 域名下没有未完成的业务，例如，域名在过户中、转出中等 |

#### 非阿里云注册商域名

| 检查项 | 要求 |
|--------|------|
| **域名账号** | 确保域名已注册且您对域名有控制权，域名未上架，不在出售列表中 |
| **注册时长** | 域名已注册超过 60 天且可正常从其他注册商转出，并正常转入阿里云 |
| **域名状态** | 域名有禁止更新状态、禁止解析状态的将无法转入或完成交易 |

### 操作步骤

#### 1. 填写域名

登录域名控制台，选择**我是卖家 > 发布交易**，在发布域名页面，输入域名信息，然后单击下一步。

#### 2. 完善出售信息

| 域名类型 | 操作 |
|----------|------|
| **阿里云注册商域名** | 输入收款价格、停售时间和域名简介，然后单击确认发布 |
| **非阿里云注册商域名** | 输入收款价格、停售时间和域名简介，缴纳保证金后单击确认发布 |
| **入库其他平台的阿里云域名** | 输入收款价格、停售时间和域名简介，缴纳保证金后单击确认发布 |

#### 3. 验证域名所有权（非阿里云域名专用）

| 场景 | 验证方式 |
|------|----------|
| **传统验证**（单域名或<50 个批量发布） | 阿里云每日自动验证 DNS 解析记录（需为每个域名配置解析），7 天内未完成则自动下架 |
| **DNS 简化验证**（≥50 个批量发布 + 资格满足） | 系统随机指定一个域名进行 DNS 验证，验证通过后，全部域名自动上架 |
