## 操作步骤

### 第一步：注册并认证域名

域名是网站在互联网上的唯一地址（例如 aliyun.com），是用户访问您网站的入口，也是品牌标识的一部分。

**操作内容**：
1. **选择域名**：新注册域名 或 购买已有域名
2. **完成实名认证**：提交所有权人（个人或企业）的身份证明材料

**审核时间**：
- 域名信息模板的审核过程通常需要 **1~3 个工作日**
- 域名实名认证资料的审核通常需要 **1 个工作日**

> **提示**: 为保护品牌资产，可在注册域名的同时，通过申请商标专用权获得法律保护。

### 第二步：购买并配置服务器

购买并配置 ECS 服务器（Elastic Compute Service）实例，为网站提供满足运行需求且符合安全规范的计算资源。

**选择建站方式**：

| 方式 | 适合人群 | 说明 |
|------|----------|------|
| **自助建站** | 开发者 | 完全自主地控制网站的功能与设计 |
| **模板建站** | 无技术基础用户 | 万小智 AI 建站，提供预制模板，快速上线 |
| **定制官网** | 企业用户 | 云·企业官网，提供个性化设计服务 |
| **功能定制开发** | 复杂业务需求 | 定制解决方案，满足复杂业务需求 |

**购买 ECS 实例配置要点**：

| 配置项 | 说明 |
|--------|------|
| **付费类型** | 如需 ICP 备案，请选择包年包月、且购买时长需大于 3 个月（含续费累计） |
| **地域** | 选择靠近目标用户的地域以降低访问延迟。如需 ICP 备案，地域必须选择中国内地 |
| **实例规格** | 根据预估的访问量和应用负载选择，小型网站通常从经济型或通用型入门规格开始 |
| **公网 IP** | 如需 ICP 备案，需开通公网访问。请勾选分配公网 IPv4 地址 |
| **安全组** | 作为虚拟防火墙，用于控制实例的入站和出站流量。配置时需遵循最小权限原则 |

> **说明**: 为节省时间，您可在购买 ECS 实例后同步搭建网站，并在等待 ICP 备案管局审核期间同步完善网站内容。

### 第三步：完成 ICP 备案

根据工信部规定，托管在中国内地提供服务的网站都必须先完成 ICP 备案，否则网站无法开通访问。

**备案服务选择**：

| 服务 | 费用 | 说明 |
|------|------|------|
| **自助备案** | 免费 | 个人网站备案快速入门 |
| **备案管家服务** | 付费 | 专业备案管家协助 |
| **备案智能助理** | 付费 | AI 智能助理辅助备案 |

**备案流程与耗时**：

```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│  阿里云初审     │ →  │  工信部短信核验 │ →  │  管局审核       │
│  1-2 个工作日   │    │  24 小时内完成   │    │  1-20 个工作日  │
└─────────────────┘    └─────────────────┘    └─────────────────┘
```

> **提示**: 在等待管局审核期间，可并行进行网站内容部署。

### 第四步：部署网站内容

将网站的应用程序、代码和静态文件上传并配置到 ECS 服务器。部署完成后，可通过服务器 IP 地址进行内部访问和测试。

具体部署方法取决于网站的技术栈（如 LAMP、LNMP、Node.js、Java 等）。

### 第五步：配置域名解析

域名解析的作用是将用户易于记忆的域名（如 aliyun.com）转换为服务器的 IP 地址（如 192.0.2.1）。没有正确的解析，用户输入域名后将无法找到您的网站。

**操作指南**：配置域名解析

**生效时间**：
- 首次添加：**实时生效**
- 修改或删除：生效时间取决于 TTL 设置，通常为 **2-10 分钟**

### 第六步：部署 SSL 证书

SSL 证书用于启用 HTTPS 协议，可在用户浏览器和您的服务器之间建立一条加密通道，以保护数据传输安全并验证网站身份。

**操作指南**：
- [使用个人测试证书（免费版）为 Web 站点开启 HTTPS 访问](https://help.aliyun.com/zh/ssl-certificate)
- [使用正式证书为 Web 站点开启 HTTPS 访问](https://help.aliyun.com/zh/ssl-certificate)

完成以上步骤后，网站即可通过 `https://yourdomain.com` 安全访问。
