# 域名行业咨询 RAG 检索使用指南

## 📚 功能说明

本 RAG（检索增强生成）工具基于阿里云官方帮助文档（`knowledge/aliyun-domain-help`），为域名注册、交易、建站、备案等问题提供精准答案。

## 🎯 使用场景

### 1. 域名注册咨询
- 如何注册域名？
- 注册域名需要什么材料？
- 域名实名认证流程？
- 个人和企业注册有什么区别？

### 2. 域名交易咨询
- 一口价域名是什么？
- 如何购买二手域名？
- 域名 push 流程？
- 域名回购服务？

### 3. 建站咨询
- 注册域名后如何建站？
- 如何绑定域名到服务器？
- DNS 解析如何配置？
- 万小智 AI 建站是什么？

### 4. 备案咨询
- ICP 备案流程？
- 个人备案需要什么材料？
- 企业备案流程？
- 备案需要多长时间？

### 5. 域名管理咨询
- 域名如何过户？
- 如何修改域名持有者信息？
- DNS 服务器如何修改？
- 如何开启域名锁定？

### 6. 续费赎回咨询
- 域名续费价格？
- 域名过期了怎么办？
- 域名赎回流程？
- 赎回价格是多少？

## 🔧 使用方法

### 方法 1：命令行使用

```bash
# 基本查询
python3 scripts/domain_knowledge_rag.py "域名注册流程"

# 备案咨询
python3 scripts/domain_knowledge_rag.py "备案需要什么材料"

# 建站咨询
python3 scripts/domain_knowledge_rag.py "如何建站"

# 域名交易
python3 scripts/domain_knowledge_rag.py "一口价域名"
```

### 方法 2：Python 代码集成

```python
from domain_knowledge_rag import answer_with_rag, DomainKnowledgeRAG

# 简单用法：直接获取答案
query = "域名注册需要什么流程？"
answer = answer_with_rag(query)
print(answer)

# 高级用法：获取更详细的信息
rag = DomainKnowledgeRAG()

# 搜索相关文档
sections = rag.get_relevant_sections("备案材料", max_sections=3)

for section in sections:
    print(f"📄 标题：{section['title']}")
    print(f"📁 分类：{section['category']}")
    print(f"📍 文件：{section['file_path']}")
    print(f"📝 摘要：{section['excerpt'][:200]}...")
    print("-" * 60)
    
    # 如需完整内容
    # full_content = rag.get_document_content(section['file_path'])
    # print(full_content)
```

```
用户提问

    ↓

    ↓
计算文档相关性得分

    ↓
```

## 🎓 最佳实践

### 1. 提问技巧
- ✅ **具体明确**："个人域名备案需要什么材料"
- ✅ **包含关键词**："域名注册流程"、"ICP 备案步骤"
- ❌ **过于宽泛**："域名"、"备案"

### 2. 答案验证
- 检查引用的文档来源
- 查看文档更新时间
- 对比多个相关文档
- 必要时访问原文确认

### 3. 时效性注意
- 帮助文档可能随时更新
- 价格和政策会调整
- 建议以阿里云官网最新版本为准
- 回答中应标注文档更新时间

## ⚠️ 注意事项

1. **知识库覆盖范围**：当前知识库主要覆盖阿里云域名服务，其他注册商信息可能不包含
2. **文档时效性**：帮助文档会更新，回答中应标注文档更新时间
3. **地域差异**：备案等政策可能因地区而异，需提醒用户咨询当地管局
4. **价格变动**：域名价格会调整，具体以下单时为准

## 🆘 故障排查

### 问题：检索结果为空
**原因**：
- 知识库中没有相关文档
- 查询词过于特殊

**解决**：
- 尝试使用更通用的关键词
- 检查知识库是否已更新

### 问题：检索结果不相关
**原因**：
- 查询词模糊
- 关键词提取不准确

**解决**：
- 使用更具体的问题描述
- 包含领域关键词（如"域名"、"备案"、"ICP"等）
