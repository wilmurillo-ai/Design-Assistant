## 建议的展示场景

### 场景 1：AI 写作 + 一键发布
```
用户：帮我写一篇关于 ChatGPT 的文章，用科技蓝模板发布到公众号

AI：正在生成文章...
文章已生成！请选择排版模板：
[模板卡片列表]

用户：选择「科技蓝」

AI：已选择模板：科技蓝
请选择要发布的公众号：
[公众号列表]

用户：选择「技术前沿」

AI：⏳ 正在发布... (35%)
🎨 应用模板样式
📋 上传图片 (2/5)

AI：✅ 发布成功！
[🔗 查看草稿] [📝 再写一篇]
```

### 场景 2：仅排版
```
用户：帮我把这段内容排版后发布到公众号：
# Markdown 内容...

AI：收到！检测到 Markdown 格式
请选择排版模板...
```

### 场景 3：错误处理
```
用户：发布到公众号

AI：你还没有绑定任何公众号。
请先完成授权：
1. 访问 open.tyzxwl.cn
2. 点击「授权公众号」
3. 扫码授权

[去授权] [查看帮助]
```
