# 常见问题解答

关于 OpenClaw Skill「微信公众号排版与发布」的常见问题。

---

## 安装配置

### Q1: 如何安装这个 Skill？

将 `openclaw-skill` 文件夹复制到 OpenClaw Gateway 的 skills 目录即可：
```bash
cp -r openclaw-skill /path/to/openclaw/gateway/skills/
```

### Q2: API 密钥在哪里获取？

1. 登录 [排版平台](https://open.tyzxwl.cn)
2. 进入「设置」→「API 管理」
3. 点击「生成新密钥」
4. 复制生成的 `wemd_xxxxxx` 格式密钥

### Q3: SaaS 平台地址是什么？

如果你使用的是官方平台，地址是：`https://open.tyzxwl.cn`

## 使用问题

### Q4: 发布后在哪里查看文章？

发布成功后，会生成微信草稿箱链接，点击即可在微信后台查看和编辑。

### Q5: 支持哪些模板？

Skill 内置 6 种模板：
- 📝 默认主题（通用）
- 💻 科技蓝（技术/产品）
- 📖 优雅简约（散文/阅读）
- 💼 商务专业（报告/分析）
- 🍜 美食探店（餐饮/餐厅）
- 🎓 教育培训（课程/知识）

此外，Web 端模板市场提供 36+ 种模板（涵盖新闻媒体、科技数码、创意设计等分类），你可以在模板市场 Fork 到私有模板库后，Skill 会自动展示你的自定义模板。

### Q6: 如何选择模板？

有两种方式：
1. **自然语言**：告诉 AI "用科技蓝模板"
2. **交互选择**：让 AI 展示模板列表，包含你的自定义模板和内置模板，点击选择

### Q6.5: 什么是用户自定义模板？

你可以在 Web 端模板市场 Fork 官方模板，或在模板设计器中自行创建模板。创建后的模板会自动出现在 Skill 的模板选择列表中（排在内置模板之前），标记为「🎨 我的模板」。

### Q7: 封面图是如何处理的？

当前发布链路采用 5 层兜底策略：
1. 用户指定的封面图
2. AI 自动生成匹配封面
3. 文章第一张图片
4. 默认封面图
5. 自动生成的占位图

## 付费问题

### Q8: 这个 Skill 是免费的吗？

基础功能免费，但有使用额度限制：

| 套餐 | 价格 | 发布次数 | 公众号数 | 自定义模板数 |
|------|------|----------|----------|-------------|
| 免费版 | ¥0/月 | 10次/月 | 1个 | 3个 |
| 基础版 | ¥29/月 | 100次/月 | 3个 | 20个 |
| 专业版 | ¥99/月 | 500次/月 | 10个 | 不限 |
| 企业版 | ¥299/月 | 不限 | 不限 | 不限 |

你可以随时使用 `/quota` 命令或问 AI「查询额度」来查看当前使用情况。

### Q9: 如何升级套餐？

1. 登录排版平台
2. 进入「个人中心」→「套餐管理」
3. 选择需要的套餐并支付
4. 支付成功后立即生效

## 技术问题

### Q10: 发布失败怎么办？

常见原因及解决方法：

| 错误 | 原因 | 解决方法 |
|------|------|----------|
| 图片上传失败 | 图片超过 1MB | 压缩图片后重试 |
| API 连接失败 | 配置错误 | 检查 API Key 是否正确 |
| 未授权公众号 | 未绑定公众号 | 先在平台授权公众号 |
| 额度已用完 | 超出套餐限制 | 升级套餐或等待下月重置 |

如果是套餐限制触发，Skill 会透传服务端返回的限制信息，便于你在 OpenClaw 侧直接提示用户升级或管理账号。

### Q11: 可以发布到多个公众号吗？

可以！支持绑定多个公众号，发布时可以选择目标账号。

### Q12: 文章会自动同步到公众号吗？

目前是发布到**草稿箱**，需要你手动在微信后台审核后发布。

2. 在对话中问「查询额度」「还能发几篇」「我的额度」

使用 `/preview` 命令或在对话中说「预览一下」，AI 会生成一个带内嵌样式的预览链接，确认效果后再发布。
