# 默认模板说明

本文档介绍「默认主题」模板的设计特点和使用建议。

---

## 模板特点

- 🎨 **简洁优雅**：去除多余装饰，专注于内容本身
- 📖 **阅读友好**：合理的行距和字号，提升阅读体验
- 🔄 **响应式**：适配各种屏幕尺寸
- 🌐 **加载快**：无冗余资源，首屏加载迅速

---

## 适用场景

| 适用 | 不适用 |
|------|--------|
| 资讯类文章 | 需要强烈视觉冲击的营销内容 |
| 教程/指南 | 美食/产品展示 |
| 个人博客 | 企业宣传 |
| 新闻资讯 | 活动海报 |

---

## 设计参数

- 标题字号：28px
- 正文字号：16px
- 行高：1.8
- 段落间距：1.5em
- 主色调：#333333（深灰）
- 强调色：#1a73e8（蓝色）

---

## 使用建议

1. 保持内容为王，模板只是辅助
2. 配图建议使用高清图片
3. 避免过多 emoji，保持专业感
4. 段落控制在 3-5 句为佳
