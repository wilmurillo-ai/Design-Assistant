# Ethics & Copyright Reference

Ethics rules enforced by anyone-skill, layered by subject type.

## Someone you know (colleague, friend, family, partner, ex)

- Chat logs and photos are private data. Confirm **personal use only** before proceeding.
- Not for harassment, stalking, deception, or impersonating the real person to others.
- All data stored locally — nothing is uploaded to any server.
- If the user shows signs of unhealthy fixation (unable to let go, obsessive re-engagement), gently acknowledge it and suggest professional support.

## Yourself

- When chat logs include messages from others, treat those as supporting context only — the distillation focuses on *you*, not them.
- Check that exported data does not include sensitive credentials or private third-party information.

## Public figure (entrepreneur, artist, politician, athlete, etc.)

- Use only **publicly traceable sources**: interviews, published books, speeches, public social media.
- Do not use leaked or private materials.
- Generated skill must include this disclaimer on first activation (say it once, not on every message):  
  *"This persona is based on public information about [name]. It is not the real person and should not be treated as their actual views."*
- Do not fabricate political positions, health information, or private relationships they have not publicly disclosed.

## Historical figure

- Sources must be published: books, documented speeches, academic research, verified letters.
- Claims inferred from incomplete evidence must be tagged L3/L4, not stated as facts.
- Note primary references in the generated skill's `soul/injection.md`.

## Fictional character

### Personal local use (not distributing)
No restrictions. Direct roleplay mode.

### Distributing to others
- Commercial IP: activate **Inspired-by mode**
  - Output must differ meaningfully from the original (reinterpretation, not reproduction)
  - Do not copy verbatim dialogue or lore as the skill's own content
  - Note in `description` frontmatter: `Inspired by [IP name]`
- Open-licensed / public domain: direct roleplay mode, credit the source

### The copyright standard is distribution intent, not publication date
- Keeping it for yourself → no restrictions
- Sharing with others → commercial IP requires Inspired-by mode

## Archetype

- Clearly inform the user: this is a synthetic persona and does not correspond to any real individual.
- Do not take a real person as the primary template with only surface-level changes.

## Universal Layer 0 rule

The generated skill will never speak words the subject would absolutely never say —  
**unless L1 or L2 evidence directly supports it.**
