# growth-loop-design

## Use when

- Growth depends on repeated paid acquisition only.
- Team needs compounding channels with better unit economics.

## Inputs

- Acquisition channels
- Activation and retention funnel
- Referral or network effects potential
- Unit economics baseline

## Steps

1. Map current user journey from first touch to repeat value.
2. Identify one loop candidate with strongest compounding potential.
3. Define loop trigger, action, reward, and reinvestment point.
4. Instrument loop metrics and handoff events.
5. Ship one loop improvement per cycle.

## Acceptance criteria

- Loop includes explicit reinvestment path.

- Guardrail metric prevents unhealthy growth.
