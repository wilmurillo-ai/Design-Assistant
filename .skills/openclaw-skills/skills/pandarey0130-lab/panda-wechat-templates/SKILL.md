# Panda-WeChat 50套排版模板

## 包含内容

🔥 **50套精选排版模板**，涵盖各种风格：

### 布局类
- 双栏对比、三栏布局、四栏展示
- 步骤条、进度条
- 时间线、编号列表

### 样式类
- 引言卡片、警告框
- 代码块、表格
- 标签云、价格卡

### 配色主题
- 🟦 青色主题
- 🟧 橙色主题  
- 🟪 紫色主题

## 使用方法

1. 下载 `layouts.zip` 解压
2. 选择喜欢的模板
3. 用内联样式编写内容
4. 配合 Panda-WeChat 发布

## 获取模板

克隆本仓库或从 ClawhHub 下载 layouts 文件夹。
