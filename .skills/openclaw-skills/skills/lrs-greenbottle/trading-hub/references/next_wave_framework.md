## 核心思想：不要追已经涨高的，找下一个还没涨的

> "By the time CNBC is talking about it, the easy money is gone."

---

## 三幕剧本

每个 mega-trend 都会经历3个阶段：

```
第一幕 — 早期：被忽视、无聊、被讨厌。聪明钱悄悄建仓。
第二幕 — 中期：CNBC报道，分析师升级，散户入场。还好但拥挤了。
第三幕 — 末期：狂热，所有人都知道。风险/回报极差。避开！
```

### 历史案例
| 趋势 | 第一幕 ✅ | 第二幕 | 第三幕 ❌ |
|------|----------|--------|----------|
| AI | NVDA芯片 | 云基础设施 | AI应用/SaaS |
| 能源危机 | 石油巨头 | LNG船运 | 太阳能补贴 |
| 加密 '21 | BTC | 山寨币 | NFT |

## 如何找到第一幕

问自己这些问题：
- 当前热门板块**依赖什么上游**？（AI → 电力 → 冷却 → 稀土）
- 因为这个趋势，**什么会被建造**？（AI繁荣 → 数据中心 → 电网 → 核能）
- 什么板块**今天很无聊/被讨厌**，但解决了热门板块造成的问题？
- **巴菲特/主权基金/内部人** 6-12个月前悄悄买了什么？

## 交易规则（来自 trading-brain）

### 风险管理（不可谈判）
- 每笔交易最大亏损 **1%**
- 每日最大回撤 **3%** — 达到则停止交易
- 最多同时 **5个仓位**
- 单仓位不超过组合 **20%**
- **快速砍亏损，让利润奔跑**

### 入场信号
- 突发新闻 + 直接、明确的市场影响
- Polymarket 赔率变化 >10% = 可交易信号
- 财报超预期 + 指引超预期
- 政策公告（关税、制裁、利率变化）
- 社交媒体热度积累（但不要在病毒式传播后追）

### 仓位类型
- **新闻交易**：快进快出，小时到1天
- **宏观赌注**：持有数周到数月，政策主题
- **财报博弈**：财报前后动量
- **Next-Wave**：早期持仓，持有到第二幕开始

## 信息规模优势

### 要扫描的信息源（按新鲜度排序）
1. **Google News** — 最快新闻
2. **X/Twitter** — 突发新闻、政治信号、情绪
3. **Polymarket** — 预测市场赔率变化 >10% = 可交易信号
4. **Reddit** — r/wallstreetbets（情绪）、r/stocks、r/cryptocurrency
5. **南华早报/金融时报** — 亚洲/中国宏观、台海、贸易战信号

### 要关注
- 地缘政治变化（战争、制裁、关税、选举）
- 联储信号（利率决定、鲍威尔语调、PCE/CPI意外）
- 财报超预期 + 指引超预期/不及预期
- 社交媒体病毒时刻（Meme股票、加密暴涨）
- 有明确市场影响的政策公告
