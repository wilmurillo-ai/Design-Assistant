# 交易规则 | Trading Rules

## 核心原则（v5.2.0 更新）

1. **保护本金** - 亏损后先休息，不急于翻本
2. **顺势而为** - 趋势是你的朋友
3. **严格止损** - 止损是保险，不是限制
4. **仓位控制** - 永远不要满仓
5. **1%铁律**（来自 trading-brain）：单笔最大亏损1%，日最大回撤3%，最多5个仓位

---

## BTC/加密货币规则

### 开仓规则

| 条件 | 要求 |
|------|------|
| RSI | < 60（做多）或 > 40（做空） |
| 止损 | 必须同时设置 |
| 杠杆 | ≤ 10x（建议5x） |
| 仓位 | ≤ 总资金20% |

### 止损规则

| 情况 | 止损位置 |
|------|----------|
| 5x杠杆 | 入场价 - 20% |
| 10x杠杆 | 入场价 - 10% |
| 强势突破 | EMA20下方 |

### 止盈规则

| 情况 | 止盈位置 |
|------|----------|
| 保守 | 2:1盈亏比 |
| 激进 | 移动止损 |
| 短线 | RSI>70时止盈 |

### 平仓规则

- RSI 超买（>70）且价格触及布林上轨
- 价格跌破 EMA20
- 单笔亏损 > 5%
- 日亏损 > 5%（停止交易）

---

## 仓位计算公式

```
单笔仓位 = 总资金 × 风险比例 ÷ (入场价 - 止损价)
```

示例：
- 总资金: $10,000
- 风险比例: 2%
- 入场价: $70,000
- 止损价: $68,000
- 仓位 = 10000 × 0.02 ÷ (70000 - 68000) = 0.01 BTC

## 情绪规则

### 市场亢奋（涨停>80家）
- 降低仓位至50%
- 不追高
- 逐步止盈

### 市场分化（炸板率>25%）
- 仓位≤50%
- 减少短线操作
- 关注龙头股

### 市场冰点（涨停<20家）
- 仓位≤30%或空仓
- 不抄底
- 等待情绪反转

---

## 禁止事项

1. ❌ 连续亏损3笔后继续交易
2. ❌ 日亏损>5%后继续交易
3. ❌ 超过10x杠杆
4. ❌ 无止损开仓
5. ❌ 全仓操作
6. ❌ 逆势加仓摊平成本
