## 决策树

```
收到报价请求
  ↓
1. 判断业务类型
  ├─ 全屋智能方案 → 读产品库 → 生成飞书文档报价单
  ├─ 单品询价 → 查询产品库 → 计算价格
  └─ 服务费计算 → 设备总价 × 25%
  ↓
2. 生成飞书文档 → 推送链接
  ↓
+ 定时任务：每周一22点行业学习，方案复盘
```

## 🚀 安装定时任务

安装后说"配置技能"，我会自动执行以下命令注册定时任务：

```bash
# 行业学习+方案复盘（每周一22:00）
openclaw cron add --name "smart-home_learn" --cron "0 22 * * 1" --session isolated --wake now --deliver announce --channel feishu --message "执行 smart-home 每周任务：搜索智能家居行业新动态、新产品、新价格（行业学习），同时分析本周报价案例和方案模板优化（方案复盘），汇总汇报给主人。"
```

## 📦 核心功能

### Part 1: 智能家居全屋方案

**触发词：**
- "帮我做个智能家居方案"
- "设计一套全屋智能"
- "出个报价"
- "智能家居多少钱"

**工作流程：**
```
需求采集 → 品牌确认 → 产品选型 → 开关计算 → 报价生成 → 交付存档
```

### Part 2: 服务/项目报价

**触发词：**
- "做服务报价"
- "报价单"
- "项目估算"

**工作流程：**
```

```
