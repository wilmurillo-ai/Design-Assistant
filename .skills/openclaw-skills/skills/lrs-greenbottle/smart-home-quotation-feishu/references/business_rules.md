# 智能家居业务规则

## 开关配置规则

### 核心原则
- **每路灯 = 必须配1个开关**，除非客户明确不要开关
- 开关分为单键/双键/三键，分别控制1/2/3路灯
- 三键开关最多控制3路灯，超过用多个开关

### 开关数量计算
```
客厅有9路灯：
方案：三键×1 + 双键×2 + 单键×1 = 3+4+1 = 8路 ❌
方案：三键×2 + 双键×1 + 单键×1 = 3+3+2+1 = 9路 ✅
```

### 双控规则
- 双控 = 两个位置控制同一路灯
- **两个开关必须是同样型号**（都是单键或都是双键）
- 实现方式：有线开关 + 无线开关（无需额外布线）

### 中控屏替代规则
- 小米智能中控屏 = 3路开关 + 语音助手 + 蓝牙网关
- 床头/玄关等位置可用中控屏替代普通开关

## 窗帘配置规则

### 窗帘数量
- **一套 = 一个窗户**（含电机+轨道）
- 布帘和纱帘算两套（两个电机）
- 次卧只有布帘 = 1套

## 服务费规则

- **服务费 = 设备总价 × 25%**
- 服务内容包括：设计、安装指导、调试、售后

## 报价单格式规范

### 必须包含内容
1. 项目名称和日期
2. 客户需求描述
3. 各房间设备清单（型号/数量/单价/小计）
4. 设备合计
5. 服务费
6. 最终总价

### 备注说明
1. 以上报价不含开槽布线、不含耗材、无税金
2. 电动窗帘包含电机和轨道
3. 双控方案采用两个同型号开关实现
4. 设备质保按厂家标准执行
