# Charmie CRM Lite 3.0.3

A lightweight CRM with SQLite for OpenClaw. Manage contacts (add, search, delete, update).

## Features
- Add new contacts
- Search by name
- Delete contacts
- Update contact details

**Email and messaging are not included** – they are available in the Professional version.

## Installation
1. `npm install`
2. `node setup.js` (creates the SQLite database)
3. `npm start`
