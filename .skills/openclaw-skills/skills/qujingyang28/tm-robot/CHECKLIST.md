## ✅ 已完成功能

### 1. 基础连接
- [x] SCT 连接（Port 5890）- 运动控制
- [x] SVR 连接（Port 5891）- 状态读取
- [x] 自定义 SVR 解析器（100% 兼容 TMflow 协议）
- [x] 连接状态监控
- [x] 自动重连机制

### 2. 状态监控
- [x] 关节角度读取 (`Joint_Angle`)
- [x] 笛卡尔位姿读取 (`Coord_Robot_Flange`)
- [x] 错误码读取 (`Error_Code`)
- [x] 机器人状态 (`Robot_Error`, `Project_Run`)
- [x] 关节力矩 (`Joint_Torque`)
- [x] 关节速度 (`Joint_Speed`)
- [x] TCP 受力 (`TCP_Force`)
- [x] TCP 速度 (`TCP_Speed`)
- [x] 机器人灯状态 (`Robot_Light`)
- [x] 手动/自动模式 (`MA_Mode`)

### 3. 运动控制
- [x] 关节运动 (PTP)
- [x] 回零运动 (`move_joints_zero`)
- [x] 速度控制
- [x] 加速度控制
- [x] 运动停止 (`stop_motion`)
- [x] 报警复位 (`reset_alarm`)

### 4. I/O 控制
- [x] 数字输出设置 (`Ctrl_DO0`~`Ctrl_DO7`)
- [x] 数字输入读取 (`Ctrl_DI0`~`Ctrl_DI7`)
- [x] 末端 IO 读取 (`End_DO0`~`End_DI3`)
- [x] 模拟 IO 读取 (`Ctrl_AO0`, `Ctrl_AI0`)

### 5. 相机/视觉
- [x] 手眼相机位姿读取 (`HandCamera_Value`)
- [x] 相机光源状态读取 (`Camera_Light`)

- [ ] 相机光源控制（需 TMflow 配置 DO 绑定）
- [ ] 触发拍照（需 TMflow 视觉项目）
- [ ] 视觉抓取应用

### 6. 文档

## 🔧 需要 TMflow 端配置的项目

### 必须配置
1. **Ethernet Slave 数据表** ✅ 已配置
   - 所有状态变量
   - IO 变量
   - 视觉变量

2. **Listen Node 项目** ✅ 已配置
   - 项目名称：`openclaw`
   - 端口：5890

### 建议配置
3. **相机光源 DO 绑定**
   - TMflow → 设置 → I/O → 相机光源 → 绑定到 DO2

4. **视觉项目**
   - TMflow → 视觉 → 新建项目
   - 配置相机参数
   - 配置检测工具

5. **工具坐标系**
   - TMflow → 设置 → 工具 → 新建
   - 标定 TCP

6. **力控配置**
   - TMflow → 力控 → 力控向导
   - 配置力控参数
