# 数据集结构详解

## 目录结构

```
box_dataset/
└── box_20260401_103100/           # 数据集目录（按时间命名）
    ├── camera_intrinsic.json      # 相机内参（全局唯一）
    ├── dataset_config.json        # 数据集配置
    ├── capture_stats.json         # 采集统计
    └── raw_data/                  # 原始数据目录
        ├── 000001_Color.png       # RGB 彩色图
        ├── 000001_Depth.raw       # 深度图 (16位 uint16)
        ├── 000002_Color.png
        ├── 000002_Depth.raw
        └── ...
```

## 文件详解

### camera_intrinsic.json

相机内参文件，全局唯一，用于坐标转换：

```json
{
  "fx": 391.14,      # X 方向焦距（像素）
  "fy": 391.14,      # Y 方向焦距（像素）
  "ppx": 320.0,      # 主点 X 坐标（像素）
  "ppy": 240.0,      # 主点 Y 坐标（像素）
  "width": 640,      # 图像宽度
  "height": 480      # 图像高度
}
```

### dataset_config.json

数据集配置信息：

```json
{
  "dataset_name": "box_20260401_103100",
  "box_type": "box",
  "capture_date": "2026-04-01T10:31:00",
  "camera": "Intel RealSense D455"
}
```

### capture_stats.json

采集统计信息：

```json
{
  "total_frames": 30
}
```

### raw_data/ 目录

| 文件类型 | 格式 | 大小 | 说明 |
|----------|------|------|------|
| `XXX_Color.png` | PNG 8位 RGB | ~100-300KB | 彩色图像 |
| `XXX_Depth.raw` | uint16 raw | 614,400 bytes | 深度数据 (640×480×2) |

## 深度数据说明

### 编码格式

- **数据类型**：uint16（无符号 16 位整数）
- **单位**：毫米（mm）
- **分辨率**：640 × 480
- **文件大小**：614,400 bytes (640 × 480 × 2)

### 有效范围

- **最小值**：0（无效深度）
- **最大值**：65535（约 65 米）
- **有效范围**：D455 为 0.1m - 6m（超过此范围深度值不可靠）

### 特殊值

- **0**：无效深度（太近、太远、反射表面等）
- **65535**：饱和深度

## 箱体尺寸分类

| 类型 | 尺寸范围 (长×宽×高 m) |
|------|----------------------|
| 料箱 | 0.15-0.25 × 0.10-0.20 × 0.05-0.15 |
| 料盘 | 0.25-0.40 × 0.20-0.30 × 0.15-0.25 |
| 料盒 | 0.40-0.50 × 0.30-0.40 × 0.25-0.35 |
