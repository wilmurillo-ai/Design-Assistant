| 关节 | 范围 | 正方向 | 负方向 | 实际执行 |

| **J4** | -180°~180° | ✅ | ✅ | 部分执行 |
| **J5** | -180°~180° | ✅ | ✅ | 执行良好 |
| **J6** | -360°~360° | ✅ | ✅ | 执行良好 (±29.9°) |

**重要发现：**
1. **所有关节命令都返回成功 (代码 0)**

3. **J3 可能触发限位保护**
4. **J6 执行精度最高**
