## [1.0.0] - 2026-03-23

### ✨ 新增功能
- 情绪分析核心功能：基于 DeepSeek LLM 识别 6 种情绪
- 天气映射系统：情绪 → 天气图标 + 治愈文案
- 多语言支持：中英文密语触发（mood/心情/情绪）
- 自动重试机制：网络异常时重试 3 次，指数退避
- 健康检查命令：`mood --healthcheck` 一键诊断
- 帮助系统：`mood --help` 显示详细用法
- OpenClaw 技能集成：支持自然语言触发

### 🔧 技术改进
- 添加 OpenClaw 门控机制（requires.bins + requires.env）
- 完善错误处理与用户提示
- 添加运行时健康检查功能
- 优化 SKILL.md 文档结构

### 📝 文档
- 创建完整的 README.md
- 添加 MIT 开源协议
- 完善 package.json 元数据
- 创建 CHANGELOG.md

### 🧪 测试
- 基础集成测试脚本（test/test.sh）
- 支持 4 种情绪场景测试

- **主版本号（Major）**：不兼容的 API 变更
- **次版本号（Minor）**：向后兼容的功能新增
- **修订号（Patch）**：向后兼容的问题修复
