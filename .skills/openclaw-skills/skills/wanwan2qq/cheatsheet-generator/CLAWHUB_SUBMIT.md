## ✨ 技能描述

生成结构化 Markdown 速查表，沉淀实战经验，支持导出分享。

**核心功能**：
- 📋 结构化输出（检查清单 + 核心命令 + 常见问题 + 实战经验）
- 🎯 内置 3 个高质量模板（npm/Git/OpenClaw 技能开发）
- 🔍 智能搜索（支持标题/关键词/内容搜索）
- 💾 支持导出为 Markdown 文件
- 🌍 **支持任何领域的速查表**（技术类/生活类/健康类/学习类/工作类/美食类...）

## 📋 内置模板

### 技术类（3 个）
1. **npm 发布速查表** - npm 包发布完整指南（含实战经验）
2. **OpenClaw 技能开发速查表** - OpenClaw 技能开发指南（含踩坑经验）
3. **Git 常用命令速查表** - Git 高频命令速查

### 非技术类（AI 生成，支持任何领域）
- 🏠 生活类：咖啡冲泡、健身动作、旅行攻略
- 💪 健康类：运动指南、饮食建议、冥想技巧
- 📚 学习类：语言学习、考试复习、知识总结
- 💼 工作类：会议组织、时间管理、沟通技巧
- 🍳 美食类：菜谱、烘焙技巧、调酒配方
- 🎨 兴趣类：摄影技巧、绘画入门、音乐理论
