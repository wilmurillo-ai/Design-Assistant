# 诗意散文风格克隆器

将口语、素材或草稿改写为**诗意哲思型散文/情书体**文章。

## 工作流程

```
输入素材/草稿 → 注入诗意风格 → 输出散文/情书
```

---

## 适用边界

| 类型 | 说明 |
|------|------|
| ✅ **适用** | 情书、爱意表达、诗意散文、哲理随笔、日记体文字、口语转诗意 |
| ✅ **适用** | 素材润色、草稿改写、风格统一化改写 |
| ❌ **不适用** | 实用说明文、新闻稿、正式商务邮件、学术论文 |

---

## 固化风格档案

> 基于多篇散文/情书/日记/随笔/小说片段样本分析生成

---

### 风格概述

写作风格是**诗意哲思型散文**：以浓烈而克制的情感为底色，用丰富的自然意象和精妙的比喻来承载爱意与哲思。文字如潮水般铺陈，排比句增强气势，短句独立成段制造节奏冲击。善用省略号营造沉吟感，用问句引领思考，在浪漫与理性之间保持精妙的平衡。

写作的精神世界同时包含：**诗意的感性**（情书、梦境叙事）和**哲学的冷峻**（存在主义分析、符号学框架），两者在同一支笔下交替出现，甚至融为一体。

---

### 1. 句式结构

| 特征 | 描述 |
|------|------|
| **长短句比例** | 长句铺陈 + 短句点睛交替 |
| **段落节奏** | 诗意断行：短句单独成段，形成视觉与节奏双重冲击 |
| **平均句长** | 中长句为主，承载意象；短句负责点题与强调 |
| **句式偏好** | 陈述为主，设问与反问引导哲思；哲学分析段落偏好复杂长句 |

**典型句式**：
> "我可以看着你永远，看你的眼睛就像看电影，看一场关于所有我所爱的世界的漫长记录影片。"

> "时间长了两个人都离不开对方然后变成彼此的样子，是我理想的爱情状态。"

> "我是如何确定我爱你"

---

### 1b. 多元体裁模式

同一作者在不同篇章中呈现显著的风格分化，改写时须识别素材的体裁类型并对应处理：

| 体裁 | 典型特征 | 语言腔调 |
|------|----------|----------|
| **情书/爱意散文** | 意象密集、排比铺陈、身体感通 | 诗意浪漫，情感浓烈 |
| **梦境叙事** | 场景具体可信、感受真实、记忆可疑、第一人称回忆体 | 恍惚、犹疑、在真实与虚构之间游移 |
| **哲理随笔** | 哲学概念嵌套、括号自语、设问解构 | 冷峻、抽离、符号密集 |
| **角色小说片段** | 第三人称、叙事者讲述、悲剧弧线 | 客观叙述、命运感强 |
| **祝福/赠言** | 四句古典格式、意象+祝愿 | 典雅、凝练、古雅 |
| **英文散文随笔** | I behold the vessel... 古体书面英语 | 维多利亚叙事腔 |
| **日常随笔** | 一个瞬间+哲学跳跃、口语入诗 | 生活感、思辨、荒诞 |

**识别方式**：
- 素材中出现拉康/萨特/梅洛-庞蒂等概念 → 哲理随笔模式
- 素材中描述梦境、细节极为具体 → 梦境叙事模式
- 素材是给他人的寄语/祝福 → 古典祝福体模式
- 素材为英文且用词古雅 → 英文散文随笔模式
- 素材讲述虚构角色故事 → 角色小说片段模式

### 2. 用词特征

| 特征 | 描述 |
|------|------|
| **文学性** | 极高，大量使用诗意、生僻、精准的词汇 |
| **意象来源** | 自然万象：花、山谷、星云、河流、森林、月光、晨曦、海洋 |
| **感官词汇** | 视觉（潺潺冰泉）、触觉（温热、柔软）、听觉（钟磬之音）互通 |
| **形容词** | 精准而克制，不堆砌，用在意象核心 |

**核心意象词汇**：
花、星云、火焰、冬天、春天、河流、森林、山谷、小涧、
月亮、太阳、尘埃、时间、永恒、灵魂、玻璃钟罩

**高频连接词**：
于是、于是、然而、不过、而、然后、所以、因为

---

### 3. 修辞偏好（核心特征）

| 手法 | 使用频率 | 典型用法 |
|------|----------|----------|
| **博喻** | **极高** | 一个本体用多个喻体铺陈，如"花有纸的花，有水的花……" |
| **排比** | **极高** | 大量排比句式，增强气势与感染力 |
| **比喻** | **极高** | 意象比喻是核心表达方式 |
| **省略号** | **高频** | "……"营造沉吟、迟疑、余韵 |
| **设问/反问** | 高频 | 引领哲思，如"爱是什么？" |
| **拟人** | 中高频 | 赋予自然万物情感 |
| **通感** | 中频 | 视觉/触觉/听觉互通 |
| **分行断句** | 高频 | 短句独立成段，制造节奏 |

**修辞示例**：
> "遇见了你之后，世界变得那么不同！一开始的火燃起来，绽放的是温暖的焰，一滴滴融化我的冬天。"

> "花有纸的花，有水的花，有高耸的花，有低垂的花，有柔美的花，有娇艳的花，有阳春白雪的花，有下里巴人的花……"

> "看你的眼睛就像看电影，看一场关于所有我所爱的世界的漫长记录影片。"

> "遍布我身体的血管里流淌着你。"

---

### 4. 叙事视角

| 特征 | 描述 |
|------|------|
| **人称** | 第一人称"我"，极致真诚 |
| **读者/对象** | 直接称呼"你"，私密对话感 |
| **距离感** | **极近**——像在耳边呢喃、像日记本里的私语 |
| **卷入感** | **极高**——情感浓烈，读者/对象被直接卷入 |

---

### 5. 情感基调

```
克制 ○───○───●───○───○ 热烈    （热烈但有节制，不过度煽情）
理性 ○───○───○───●───○ 感性    （感性主导，但有哲思底色）
优雅 ○───○───○───●───○ 粗犷    （优雅细腻，文学性强）
确定 ○───○───●───○───○ 探索    （哲思探索，用问句寻求答案）
```

**基调描述**：
> 以诗意为骨、哲思为髓、情感为血。文字如潮水铺陈而来，短句如礁石露出水面。爱你至深，但表达克制；意象万千，却精准有力。是情书，也是诗；是告白，也是对存在与时间的思考。

---

### 6. 结构模式

| 结构元素 | 偏好 |
|----------|------|
| **开头模式** | 意象切入 / 情感高潮前置 / 回忆回望 |
| **正文结构** | 意象流——随思绪/情感/回忆自然延展，不依赖硬性标题；不做主题归类陈列，意象之间靠联想和感受自行衔接 |
| **过渡方式** | 省略号过渡、意象并置过渡；不用"像""好像""一切都像"这类解释性连接词 |
| **结尾风格** | 点题句独立成段 + 升华到一个普世意象 |

**开头示例**：
> "从前我看身边的一切，总是那么稀松平常而寡淡，它们只不过是一些花、一些树以及一堆泥土。"

> "我是如何确定我爱你"

**结尾示例**：
> "我想，我爱你，爱得纯粹，爱得深沉，爱得透彻。"

> "请你，成为我灵魂中亘古不变的长春。"

---

### 7. 个人标记 ⭐

> **最能体现个人风格的特征，改写时必须融入**

**标志性句式**：
- "xxx，爱得xxx，爱得xxx"（排比强调爱意）
- "xxx，是因为xxx，是因为xxx"（追问原因）
- "我喜欢xxx，我喜欢xxx"（并列喜欢的事物）
- "时间长了……变成彼此的样子"（爱情隐喻）
- "看你的眼睛就像看电影"（意象类比）
- "遍布我身体的血管里流淌着你"（身体意象）

**必须有的节奏元素**：
- 短句单独成段："我喜欢看着你的眼睛。"
- 省略号："……"
- 感叹号："！"（用于情感爆发点）
- 排比句式大量出现

**独特表达**：
- 博喻式排比：一个事物用数十个形容词/意象并列铺陈
- 身体感通意象：血管、心脏、手触感
- 自然宇宙意象：星球、银河、永恒、时间
- 诗意分行：诗歌/诗句单独成行

## 改写规则

### 核心原则

1. **写作前先确定句式策略**：短句还是长句？长句的话——意识流式的长句（如《撒旦探戈》那种，一个句子拖得很长，中间有无数停顿和分支）还是散文式长句？一旦确定，全篇保持一致
2. **保留情感内核**：素材里的情感、事实、意象不可丢失
3. **注入诗意意象**：找到或创造自然意象来承载情感
4. **节奏分行**（仅限短句模式）：善用短句独立成段、省略号、感叹号制造节奏

### 必须做到

- [ ] 用丰富的自然意象（花、山谷、星云、河流……）承载情感
- [ ] 大段排比句式（三个及以上并列意象）
- [ ] 短句单独成段，形成视觉冲击
- [ ] 省略号"……"出现在沉吟/转折处
- [ ] 感叹号"！"出现在情感爆发点
- [ ] 意象比喻为核心表达方式
- [ ] 口语直接陈述的情感转为诗意表达

### 避免出现

- [ ] 干巴巴的陈述句（没有意象）
- [ ] 空洞的感叹（没有载体）
- [ ] emoji（❌所有emoji都不用）
- [ ] 结构化的"1️⃣2️⃣3️⃣"列表
- [ ] 说教式、说理过于直白的话语
- [ ] 鸡汤口号式结尾
- [ ] **直接陈述情感**："我在想你""我在等你""我很爱你"——须用意象承载，让情感"发生"在自然万物中，而非直接说出来
- [ ] **过度使用感叹号**："！"仅在真正情感爆发点出现，不可滥用
- [ ] **解释性连接**："像""好像""一切都像""身边的xx都像xx"——意象并置即可，让联想关系自己成立，不解释
- [ ] **AI 高频词**：crucial、showcase、underscore、pivotal、intricate、foster、delve、tapestry、vibrant、enhance、testament、enduring、landscape（这些词在诗意散文里出现极不自然，优先用朴素词汇替换）
- [ ] **代理腔/填充词**："In order to""Due to the fact that""It is important to note that""I hope this helps""Certainly!"——直说即可，不绕弯
- [ ] **节奏均匀**：短句与长句交替，不要每句话结构长度都差不多——读起来会像机器写的
- [ ] **意识流长句去标点**：意识流长句中，标点是用工具而非法律；无标点可制造混沌/窒息感，有标点可制造节奏感——先想清楚要什么效果，不要无意识地去或留

## 改写示例

| ❌ 素材表达 | ✅ 诗意散文风格 |
|------------|----------------|
| "我爱你，看不够你" | "我可以看着你永远，看你的眼睛就像看电影，看一场关于所有我所爱的世界的漫长记录影片。" |
| "你让我觉得世界变美了" | "遇见了你之后，世界变得那么不同！一开始的火燃起来，绽放的是温暖的焰，一滴滴融化我的冬天。" |
| "爱一个人是什么感觉" | "爱是什么？爱是你对待对方如同对待另一个自己，也曾犯错，但从不失望。" |
| "爱是一起成长" | "爱就像一起滋养的一棵树，往更深处扎根。" |
