# writER

模仿 ER 个人写作风格的 skill。将口语、素材或草稿改写为富有诗意与哲思的散文/情书体文章。

---

## 安装方法

### 方法一：手动复制

将整个 `writER` 目录复制到 agent 的 skills 目录下：

```bash
# 复制到所有 agent 共用目录
cp -r writER/ /path/to/agents/skills/

# 或复制到特定 agent 的工作空间
cp -r writER/ /path/to/agent/workspace/skills/
```

### 方法二：让 Agent 自动安装（推荐）

在对话中告诉 Agent：

> "请安装 writER skill，路径：https://clawhub.ai/jewelroam/er-writer-skill"

Agent 会通过 clawhub CLI 自动安装到默认 skills 目录。

如果 Agent 已有 clawhub CLI，也可让它运行：

```bash
clawhub install writER
```

## 适用场景

- 情书、爱意表达、诗意散文、哲理随笔
- 梦境叙事、角色小说片段、意识流随笔
- 口语/素材诗意化改写、风格统一化改写

## 不适用场景

- 实用说明文、新闻稿、正式商务邮件、学术论文

---

## 使用方法

### 基本用法

给 AI 发送素材，并说明期望的体裁/句式（可选），例如：

```
按 writER 风格改写这段素材，用短句分行：
[粘贴素材]
```

```
按 writER 风格，用意识流长句写一段关于失眠的文字
```

### 指定体裁

| 体裁 | 触发方式 |
|---|---|
| 情书/爱意散文 | 直接粘贴素材，说"按我的风格改写" |
| 梦境叙事 | 说明"这是一段梦境" |
| 哲理随笔 | 说明"用意识流长句"或"哲理随笔" |
| 角色小说片段 | 说明"用第三人称写一个故事" |
| 祝福/赠言 | 说明"写一段祝福" |
| 英文随笔 | 直接粘贴英文素材 |

## 工作流程

```
写作前：确定体裁 + 确定句式
输入素材/想法
↓
对照风格档案（SKILL.md）注入诗意
↓
输出：按选定句式呈现
```

**可指定句式**：
1. 短句分行（情书、日记、诗意散文）
2. 意识流长句（哲理随笔、意识流小说）
3. 散文长句（叙事散文、角色小说片段）

---

## 核心原则

1. **意象承载情感**：所有情感都要有自然意象作为载体，不直接陈述
2. **克制陈述**："我在想你""我很爱你"不出现，让情感在意象中自己浮现
3. **意识流并置**：意象之间靠联想自然衔接，不做主题归类陈列
4. **标点是工具**：意识流长句中，标点控制呼吸，不是律法
5. **思与叙同时在场**：叙事中自然带出哲思，不强行分开

## 风格快照

- **意象来源**：花、星云、火焰、河流、山谷、血管、眼睛、月亮、梦
- **节奏**：长句铺陈 + 短句点睛；或意识流长句跨段落
- **基调**：诗意浪漫，但冷峻哲思并存；热烈但克制
- **独特句式**：博喻排比、身体感通意象、省略号沉吟、意识流长句
