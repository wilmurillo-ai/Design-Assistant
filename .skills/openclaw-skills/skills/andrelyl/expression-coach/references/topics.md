# 即兴话题库

## 职场类（权重高）

### 汇报与展示
- 向领导汇报一个项目延期了，但你有补救方案
- 用30秒向跨部门同事介绍你们团队在做什么
- 在周会上总结本周最重要的一个成果
- 你的方案被领导否了，请用2分钟重新说服他
- 给新来的同事介绍你们部门的核心业务

### 沟通与协作
- 同事交付的工作质量不达标，你怎么反馈
- 两个部门对一个需求有分歧，你怎么协调
- 你的下属最近状态不好，你怎么和他聊
- 客户提了一个不合理的需求，你怎么拒绝又不伤关系
- 领导在会上点名问你意见，但你没准备，怎么回答

### 职业发展
- 用1分钟介绍你自己的职业优势
- 你想申请调岗/升职，怎么和领导谈
- 面试中被问"你最大的缺点是什么"

## 社交类（权重高）

### 破冰与闲聊
- 在一个不认识人的聚会上，怎么开始第一段对话
- 电梯里遇到大领导，只有30秒，你说什么
- 朋友介绍了一个新朋友给你，你怎么自我介绍
- 饭局上别人聊的话题你完全不懂，怎么参与

### 关系维护
- 很久没联系的朋友突然找你借钱，怎么应对
- 你想拒绝一个饭局邀约，但不想得罪人
- 同事升职了你没升，你怎么恭喜他（真诚地）
- 亲戚问你工资多少/有没有对象，你怎么回答

### 冲突处理
- 朋友说了让你不舒服的话，你怎么表达
- 你和室友/同事在一个问题上意见完全相反
- 有人在公共场合让你很尴尬，你怎么化解

## 思维表达类

### 观点阐述
- "远程办公比到岗办公更高效"——你同意吗？请论述
- "AI会让大多数人失业"——你怎么看？
- "加班文化是中国企业的竞争力还是负担？"
- "年轻人应该先追求稳定还是先追求梦想？"
- "社交媒体让人更孤独还是更连接？"

### 即兴描述
- 描述你昨天做过的最有意思的一件事
- 用1分钟推荐一本你最近看的书/电影
- 向一个完全不懂技术的人解释你的工作
- 描述一个你克服困难的经历

### 创意表达
- 如果你是你们公司的CEO，你会改变什么
- 你有100万但只能用于改善一件事，你改善什么
- 如果时间可以倒流一年，你会做什么不同的选择

## 话题难度标签

- 🟢 入门：有明确立场即可（适合刚开始练习）
- 🟡 进阶：需要多角度思考和具体例子
- 🔴 挑战：需要即兴应变、处理情绪、高压场景
