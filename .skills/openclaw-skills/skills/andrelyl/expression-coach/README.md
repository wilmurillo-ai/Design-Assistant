# Expression Coach — 表达力训练教练

AI 驱动的个人口语表达训练工具，帮助你在职场和社交场合中更自信地开口说话。

## ✨ 核心功能

### 1. 即兴话题练习 🎯
- 36 个精选话题，涵盖职场、社交、思维表达
- 三档难度：🟢入门 / 🟡进阶 / 🔴挑战
- **语音回答** + AI 六维评分（结构/逻辑/表达力/简洁/说服力/流畅度）
- 具体反馈 + 口语化示范改写

### 2. 场景模拟 🎭
- 10 个预设场景（汇报、谈判、面试、社交、投诉处理等）
- AI 角色扮演，真实施压（不是走过场）
- 支持自定义场景
- 完整复盘：评分 + 亮点 + 改进 + 关键时刻改写

### 3. 框架速查 📋
- 8 大实用表达框架（PREP/STAR/SCQA/NVC/三明治法等）
- 常用话术模板（会议发言、表达不同意、请求、拒绝、安慰）
- 针对具体场景的口语化示范

### 4. 自定义话题 📝（需配置 Bitable）
- 添加个人话题到题库
- AI 智能生成针对性话题
- 追踪每道题的练习次数和最高分

### 5. 进步追踪与数据分析 📈（需配置 Bitable）
- 综合分走势、各维度雷达图
- 填充词 Top 3 和频率趋势
- 最强/最弱维度识别 + 针对性建议

### 6. 每日表达力 Tips 📬
- 35 条实用表达技巧，按顺序循环推送
- 每条附带 30 秒微练习任务
- 通过 cron job 定时推送

## 🎤 语音优先

这是**口语训练**，不是写作训练：
- 用户发语音消息回答，AI 用 Whisper 转写 + 分析
- 智能停顿分析：区分正常换气停顿 vs 真正的卡壳犹豫
- 填充词统计（嗯、那个、就是、然后...）
- 支持文字回答，但会温柔提醒语音练习更有效

## 📦 安装

```bash
clawhub install expression-coach
```

## 🔧 依赖

- **OpenAI Whisper**（必需）：语音转写
  ```bash
  pip install openai-whisper
  ```

## 🚀 快速开始

安装后直接对 AI 说：
- **"来个即兴话题"** — 开始练习（无需任何配置）
- **"模拟一下面试"** — 进入场景模拟
- **"PREP框架怎么用"** — 查看框架速查

核心功能**开箱即用**，无需配置。

## 📊 可选：飞书 Bitable 集成

配置后解锁进阶功能（练习记录、自定义话题、数据分析）：

1. 在飞书创建一个多维表格
2. 对 AI 说"设置表达力训练的数据库"，发送多维表格链接
3. AI 自动配置表结构并创建 `config.json`

详见 `config.json.example`。
