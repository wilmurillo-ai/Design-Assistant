# 川麻（血战到底）牌理参考

## 基本概念

### 牌种（共108张）
- **万子**：1万-9万，每种4张（36张）
- **筒子**：1筒-9筒，每种4张（36张）
- **条子**：1条-9条，每种4张（36张）

### 面子类型
- **顺子**：同花色连续3张（如 1万2万3万）
- **刻子**：3张相同（如 5筒5筒5筒）
- **杠子**：4张相同（明杠/暗杠/加杠）

### 胡牌条件
4组面子 + 1对雀头 = 14张

### 特殊牌型
| 牌型 | 条件 | 番数 |
|------|------|------|
| 平胡 | 4顺子+1雀头 | 1番 |
| 碰碰胡 | 4刻子+1雀头 | 2番 |
| 清一色 | 全部同一花色 | 4番 |
| 七对 | 7对子 | 4番 |
| 龙七对 | 七对中有四张一样 | 8番 |
| 金钩钓 | 4杠+1单钓 | 8番 |
| 十八罗汉 | 4暗杠+1对 | 极品 |

## 向听数

向听数 = 距离听牌还需要换几张有效牌
- **-1向听**：已胡牌
- **0向听（听牌）**：再摸一张有效牌就胡
- **1向听**：需要换1张牌才能听牌
- **N向听**：需要换N张牌

## 出牌优先级原则

1. **打孤张** — 不跟任何牌有关联的单牌
2. **打边张** — 1或9的搭子（只听一头）
3. **打嵌张** — 间隔搭子（如 13 只听 2）
4. **保两面** — 两面搭子最好（如 45 听 3和6）
5. **保对子** — 对子可做雀头或碰成刻子

## 川麻特殊规则

### 血战到底
- 一人胡了继续打，剩余玩家继续
- 最后一个没胡的"包三家"

### 换三张
- 开局每人选3张同花色跟对面交换
- 策略：把不要的花色集中换出去

### 定缺
- 选择一门花色为"缺"，胡牌时手上不能有缺门的牌
- 策略：选张数最少的花色定缺

## 实战建议

### 听牌张数越多越好
- 优先选择听牌面最宽的打法
- 例：听 3和6（两面，8张）优于听 5（嵌张，4张）

### 做清一色的时机
- 起手同花色 ≥8张 → 可以考虑做清一色
- 同花色 ≥6张 + 好搭子 → 边打边看

### 安全牌判断
- 已出4张的牌 → 绝对安全
- 上家刚打的牌 → 相对安全（不会被上家碰/胡）
- 已出3张的牌 → 只可能被明杠
