# 挪威国家石油 · Equinor ASA

> "We are proud to power the world" — Equinor企业口号

Equinor ASA（原名Statoil）是挪威最大的能源公司，成立于1972年。作为全球领先的海上油气生产商和欧洲最大的能源公司之一，Equinor在北海、巴伦支海、巴西湾和墨西哥湾等区域拥有丰富的油气资产。近年来，公司积极向可再生能源转型，是全球海上风电增长最快的能源公司之一。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1972** | 挪威议会通过法案成立Statoil |
| **1979** | 北海Ekofisk油田投产，成为挪威经济支柱 |
| **1981** | 发现Troll气田，欧洲最大的天然气田 |
| **1990s** | 扩张至北海、巴伦支海等北极区域 |
| **2001** | 在纽约和奥斯陆双重上市 |
| **2007** | 收购Norsk Hydro的石油天然气业务 |
| **2012** | 开始投资海上风电 |
| **2018** | 正式更名为Equinor，标志转型决心 |
| **2020** | Hywind Scotland成为全球首个浮动式海上风电 |
| **2022** | 成为欧洲最大天然气供应商之一 |
| **2023** | 海上风电装机超过2GW |
| **2024** | 年营收超过900亿美元，加速可再生能源投资 |

北海、巴伦支海、巴西湾等地的海上油气勘探开发

欧洲最大的天然气供应商之一

## 值得了解

- Equinor的前身Statoil是挪威政府为管理北海石油资源而创建的

- Hywind是全球首个商业化运营的浮动式海上风电场

- Equinor为欧洲提供了约20%的天然气，是俄乌冲突后的关键供应方

- 公司的油气田位于世界上最恶劣的海洋环境之一

- Equinor在巴西的深海油田是全球最先进的深水开采项目之一

- 公司更名为Equinor时引发了内部和外部的巨大争议

- Equinor的天然气产量足以供应约1.2亿欧洲家庭的年度需求

- 公司是首家在北极圈进行大规模商业油气开发的西方公司
