# 达能 · Danone

---

## 发展历程

| 时期 | 关键事件 |
|:----:|--------|
| **1919 年** | Isaac Carasso 在巴塞罗那创立——最初做酸奶 |
| **1970s** | 与 Gervais 合并进入法国 |
| **1990s-2000s** | 全球扩张 |
| **2020s** | 向植物基和健康食品转型 |

---

## 核心特点

基础乳制品（酸奶）+ 特殊营养（婴儿/医疗）+ 饮用水（Evian/Volvic）

## 核心优势

全球酸奶第一 + 饮用水（Evian/Aqua）+ 特殊营养
