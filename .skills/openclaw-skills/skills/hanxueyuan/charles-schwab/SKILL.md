# 嘉信理财 · The Charles Schwab Corporation

> "投资不应该是富人的专利。"

嘉信理财（Charles Schwab）是美国最大的折扣券商和财富管理公司之一，由 Charles R. Schwab 于 1971 年创立。公司以推动零佣金交易和投资者教育而闻名，目前管理资产超过 9 万亿美元，服务超 3,400 万客户。2020 年收购 TD Ameritrade 后进一步巩固行业地位。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1971** | Charles R. Schwab 在旧金山创立公司，最初是传统的折扣券商 |
| **1975** | SEC 取消固定佣金制度，Schwab 抓住机会推出折扣经纪服务 |
| **1995** | 推出网上交易，成为最早拥抱互联网的券商之一 |
| **1999** | 纽交所上市（SCHW） |
| **2019** | 率先推出零佣金交易，迫使所有竞争对手跟进 |
| **2020** | 宣布收购 TD Ameritrade，约 260 亿美元 |
| **2024** | 管理资产超 9 万亿美元，客户超 3,400 万 |

---

## 业务结构分析

### 折扣经纪

股票、ETF、期权、共同基金交易，2019 年引领零佣金革命

### 财富管理

个人投资咨询、退休规划、信托服务

### 嘉信银行

高收益储蓄和支票账户，存款超 3,000 亿美元

### 机构服务

为投资顾问 RIA 提供托管和技术平台，管理超 4 万亿美元

---

## 🔑 核心护城河

```
零佣金吸引客户 → 资产规模增长 → 利息和管理费收入 → 更多服务投资 → 更多客户
```

1. 零佣金模式先驱，品牌认知度极高
2. 超 3,400 万客户的资产粘性极高，迁移成本很大
3. 嘉信银行提供低成本资金来源，支持利息收入
4. RIA 托管业务管理超 4 万亿美元，是独立投资顾问的首选平台
