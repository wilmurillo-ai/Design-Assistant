# 法国巴黎银行 · BNP Paribas

> "The bank for a changing world" —— BNP Paribas

法国巴黎银行是欧元区最大的银行和全球顶级金融服务集团，由巴黎国民银行（1848年创立）和巴黎巴银行（1872年创立）于2000年合并而成。集团业务涵盖零售银行、企业银行、投资银行和资产管理，在全球65+国家拥有近18万名员工。BNP Paribas是欧洲企业金融的领导者，也是全球可持续金融的先锋。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1848** | 巴黎国民银行（BNP前身）成立，服务法国国家经济建设 |
| **1872** | 巴黎巴银行（Paribas前身）成立，专注国际贸易融资 |
| **2000** | BNP与Paribas合并，成立法国巴黎银行 |
| **2001** | 收购法国农业信贷银行的零售银行网络，巩固法国市场 |
| **2006** | 收购意大利BNL银行，成为意大利主要银行之一 |
| **2009** | 收购富通银行（Fortis Bank），成为比利时最大银行 |
| **2020** | 启动2022战略计划，聚焦数字化和可持续金融 |
| **2024** | 可持续金融承诺超3,000亿欧元，绿色贷款增长超40% |

## 业务结构分析

### 法国零售银行
法国最大的零售银行之一，服务超1,300万客户。Fortis银行在比利时市场领先，BNL在意大利排名前五。

### 企业与机构银行
全球顶级的企业银行，提供贷款、贸易融资、现金管理等服务。欧洲企业贷款市场份额第一。

### 投资解决方案
全球第十大投行，衍生品和证券服务业务突出。衍生品清算和结构性产品欧洲领先。

### 资产管理与保险
BNP Paribas Asset Management管理超7,000亿欧元资产，是欧洲最大的资管公司之一。

## 值得了解

- BNP Paribas的LOGO是一颗绿色的星星，象征"指引方向"——这也是其可持续金融战略的视觉表达
- 德银双塔在法兰克福，而BNP Paribas的总部大楼在巴黎第九区，是一座融合了历史和现代风格的建筑
- BNP Paribas是全球最大的网球赛事赞助商之一，赞助法网公开赛长达50年
- 2014年因违反美国制裁规定，BNP Paribas被罚款89亿美元，是当时银行史上最大罚单
