# 加州大学伯克利分校 · UC Berkeley

---

## 发展历程

| 时期 | 关键事件 |
|:----:|--------|
| **1868 年** | 加州大学成立——伯克利是旗舰校区 |
| **1940s** | 曼哈顿计划——Glenn Seaborg 在此发现 16 种元素 |
| **1960s** | 言论自由运动——Free Speech Movement 从这里开始 |
| **1970s-2000s** | 硅谷崛起——伯克利成为最大的人才输送基地 |
| **2020s** | AI/ML 研究重镇——多名诺贝尔奖和图灵奖得主 |

---

## 核心特点

本科 + 研究生（工程/计算机/商学/理科/文科）+ 研究

## 核心优势

公立大学中学术实力最强（诺贝尔奖数量全球前五）+ 硅谷地理位置 + 低廉学费（对本州学生）

---

## 关键数据

| 指标 | 数值 |
|------|------|
| 学生数 | 约 45000 人 |
| 诺贝尔奖 | 110 位（关联） |

## 值得了解
