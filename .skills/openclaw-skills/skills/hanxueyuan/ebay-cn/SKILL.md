# eBay 易趣技能

## 技能描述
eBay 是全球知名的 C2C 电商平台，支持拍卖和一口价购买。本技能帮助用户搜索商品、参与拍卖、管理订单。

## 核心功能
- 🔍 **商品搜索**：按关键词、类别搜索全球商品
- 🔨 **拍卖参与**：出价、竞拍、拍卖提醒
- 💰 **价格查询**：商品价格、运费、关税估算
- 📦 **订单管理**：查看订单、物流追踪
- ⭐ **卖家评价**：查看卖家信誉、买家评价
- 🌍 **海淘指导**：国际运输、清关流程

## 使用场景
- 搜索海外商品
- 参与拍卖竞价
- 查看订单状态
- 获取海淘建议

## 注意事项
- 需要 eBay 账户才能购买
- 国际运输可能需要清关
- 注意卖家信誉和退换政策
