# 道奇 · Dodge (Stellantis)

> "Brotherhood of Muscle." — 品牌标语

Dodge 由 Dodge 兄弟于 1900 年创立，最初为福特生产零部件，后发展为独立汽车品牌。Dodge 是美国肌肉车文化的代名词，Charger（轿车）和 Challenger（跑车）是品牌灵魂。作为 Stellantis 旗下品牌，Dodge 在 2023 年宣布停产燃油肌肉车，全面转向电动化。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1900** | John 和 Horace Dodge 兄弟在底特律创立 Dodge Brothers Company |
| **1914** | 推出首款 Dodge 汽车，以可靠性和耐用性著称 |
| **1928** | 被克莱斯勒收购，成为其旗下品牌 |
| **1960s** | Charger 和 Challenger 诞生，肌肉车黄金时代的代表 |
| **1970s** | 石油危机导致肌肉车市场萎缩，Dodge 转向小型车 |
| **1990s** | Viper（蝰蛇）发布，重新定义美国超级跑车 |
| **2006** | Charger 和 Challenger 回归，重启肌肉车传统 |
| **2015** | Hellcat 版本发布（707 马力），肌肉车性能巅峰 |
| **2021** | Demon 170 发布（1,025 马力），最后的大排量燃油肌肉车 |
| **2023** | 最后一批燃油 Charger/Challenger 下线，品牌进入电动化转型期 |
| **2024** | Charger Daytona SRT 概念车亮相，纯电肌肉车时代开启 |

## 业务结构分析

### 肌肉车系列

Charger（四门轿车）和 Challenger（双门跑车）是 Dodge 的灵魂车型。SRT Hellcat 和 Demon 版本拥有 700-1,025 马力，是美国最强大的量产车之一。

### SUV 与跨界

Durango 是 Dodge 的 SUV 产品，提供 V8 发动机选项，是市场上少数提供 V8 动力的 SUV。

### 电动化转型

Dodge 正在从大排量燃油车转向电动性能车。Charger Daytona 纯电版预计 2024-2025 年上市，保留了肌肉车的外观和性能基因。

### Mopar 售后

Mopar 是 Stellantis 的售后和性能零部件品牌，为 Dodge 车主提供原厂改装件，是品牌社区文化的重要组成部分。

## 值得了解

- Dodge 兄弟最初是为福特生产 Model T 的零部件供应商，后来因为与亨利·福特意见不合而创立自己的品牌

- Charger 这个名字来源于 1960 年代的"charger horse"（战马），象征着力量和速度

- Demon 170 的 1,025 马力需要使用 E85 乙醇燃料才能达到——使用普通汽油时约 807 马力

- Dodge 的 Viper（蝰蛇）最初是由 Chrysler 工程师在午餐时间构思的概念车

- Mopar 这个名字来源于 "MOtor PARts" 的组合，是 Chrysler 体系内所有零部件的总称

- 2023 年最后一批燃油 Charger 下线仪式上，工厂工人为这些车举行了"告别仪式"
