# IMF · 国际货币基金组织

---

## 发展历程

| 时期 | 关键事件 |
|:----:|--------|
| **1944 年** | 布雷顿森林会议决定成立 IMF 和世界银行 |
| **1947 年** | IMF 正式开始运作 |
| **1971 年** | 布雷顿森林体系瓦解（美元与黄金脱钩）——IMF 职能转向金融危机救助 |
| **1997 年** | 亚洲金融危机——向韩国、泰国、印尼提供救助 |
| **2008 年** | 全球金融危机——大规模救助计划 |
| **2010s** | 欧债危机——希腊、葡萄牙、爱尔兰救助 |
| **2020s** | 新冠疫情和俄乌冲突后的全球经济救助 |

---

## 核心特点

金融救助贷款、经济政策建议、技术援助、全球金融监测

## 核心优势

全球唯一拥有 190 个成员国的多边金融救助机构

---

## 关键数据

| 指标 | 数值 |
|------|------|
| 成员国 | 190 个 |
| 贷款承诺 | 超过 2000 亿美元 |
| 总部 | 华盛顿 |
| 总裁 | Kristalina Georgieva |

## 值得了解

IMF 的投票权按份额分配——美国拥有最大份额（约 17%），拥有事实上的否决权（重大事项需 85% 通过）
