# 🥋 ONE 冠军赛 · ONE Championship

ONE Championship 是亚洲最大的综合格斗赛事组织，总部位于新加坡。融合 MMA、泰拳、踢拳和巴西柔术，在亚太地区拥有庞大的粉丝基础。

## 📜 历史沿革

- **2011 年**：由 Chatri Sityodtong 在新加坡创立
- **2014 年**：首次实现盈利
- **2018 年**：与 Turner Sports 签约，进入美国市场
- **2021 年**：与 Amazon Prime Video 签约全球转播

## 🏟️ 赛制与规则

- 多规则：MMA、泰拳、踢拳、降服缠斗
- 圆形笼（Circle），而非八角笼
-  hydration 测试：赛前禁止脱水，确保选手安全
- 3 回合（普通）/ 5 回合（冠军战）

## ⭐ 知名选手/球队

- **Demetrious Johnson**："Mighty Mouse"，ONE 蝇量级冠军
- **Stamp Fairtex**：女子泰拳与 MMA 双料选手
- **Rodtang Jitmuangnon**：泰拳"铁人"，蝇量级踢拳冠军
- **唐凯（Tang Kai）**：中国首位 ONE 世界冠军
