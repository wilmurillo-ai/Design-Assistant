# 墨西哥城 · Mexico City / Ciudad de México

> "La ciudad de los palacios." — 宫殿之城

Mexico City 是墨西哥联邦首都，建立在阿兹特克帝国首都特诺奇提特兰的遗址上。作为拉美最大、世界第六大都市区，拥有超过 2,200 万人口。墨西哥城不仅是政治中心，也是拉美最重要的金融枢纽之一，贡献了墨西哥约 15% 的 GDP。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1325** | 阿兹特克人建立特诺奇提特兰，在特斯科科湖的人工岛上建城 |
| **1521** | 科尔特斯攻陷特诺奇提特兰，在原址上建立墨西哥城 |
| **1810** | 墨西哥独立战争爆发，城市成为独立运动中心 |
| **1821** | 墨西哥独立，墨西哥城成为新国家首都 |
| **1910** | 墨西哥革命爆发，城市经历了重大社会变革 |
| **1968** | 举办夏季奥运会，成为首个举办奥运的拉美城市 |
| **1985** | 8.0 级大地震造成约 10,000 人死亡，城市重建 |
| **1990s** | NAFTA 签署后成为北美自由贸易区的重要商业中心 |
| **2016** | 政治地位从"联邦区"改为"墨西哥城"，获得更大自治权 |
| **2021** | 地铁 12 号线部分坍塌后启动全面安全升级 |
| **2023** | 入选 UNESCO 创意城市网络，设计之都 |
| **2024** | 人口超 2,200 万，GDP 约 4,000 亿美元，拉美经济引擎 |

## 业务结构分析

### 金融与商业

墨西哥证券交易所（BMV）是拉美第二大交易所。改革大道沿线集中了 Citibanamex、BBVA Mexico 等银行总部和跨国公司拉美总部。

### 文化与旅游

拥有 150+ 座博物馆，包括世界级的国家人类学博物馆。历史中心区和 Xochimilco 水上花园是 UNESCO 世界遗产，年接待游客超 1,000 万人次。

### 制造业与工业

大都会区是墨西哥制造业核心，覆盖汽车、航空航天、电子产品和制药。得益于 USMCA，大量制造企业在此设立。

### 科技创新

近年来成为拉美创业中心之一，金融科技、电子商务和物流初创公司快速增长。2024 年科技投资超 20 亿美元。

## 值得了解

- 墨西哥城正以每年约 10 厘米的速度下沉——因为它建立在排干水的湖床上

- 墨西哥城有超过 400 个公园，比世界上任何城市都多

- 阿兹特克首都特诺奇提特兰在 1519 年时人口约 20-30 万，当时比伦敦或巴黎都大

- 墨西哥城的地铁系统是世界上第三大，部分车站展示了阿兹特克遗址

- 墨西哥城是拉美第一个举办夏季奥运会（1968）和世界杯（1970、1986）的城市

- 城市建在 Lake Texcoco 的干涸湖床上，这导致了沉降和地震放大的问题
