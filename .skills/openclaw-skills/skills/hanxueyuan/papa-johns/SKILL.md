# 棒约翰 · Papa John's

Papa John's 以 **"Better Ingredients. Better Pizza."（更好的原料，更好的披萨）** 为核心承诺，曾是美国增长最快的披萨连锁品牌。它的故事充满戏剧性——从一个印第安纳州酒吧的门洞里起步，登顶美国第三大披萨连锁，又因创始人的争议言论跌入低谷，随后踏上艰难的品牌重塑之路。

## 📖 品牌起源故事

**1984 年**，23 岁的 **John Schnatter**（约翰·施纳特）在印第安纳州杰斐逊维尔市（Jeffersonville），把自己酒吧壁橱改造成了一个简易披萨厨房。他变卖了 1,600 美元买来的旧比萨设备，开始了创业之路。

Schnatter 的父亲告诉他："如果你想成功，就要做出比周围所有店都好吃的披萨。" 于是 John 坚持使用新鲜的番茄酱、手工揉制的面团和真正的马苏里拉奶酪——在当时以冷冻面团和预制酱料为主的披萨行业，这种坚持堪称"奢侈"。

他的披萨大受欢迎。1985 年，第一家正式的 Papa John's 门店开业。1993 年，公司上市（NASDAQ: PZZA），随后开启了疯狂的扩张。2000 年代初，Papa John's 超越了 Little Caesars 成为美国第三大披萨连锁，门店数量一度超过 5,000 家。

**经典营销**：Papa John's 的广告强调"高品质原料"，对比 Domino's 的"30 分钟送达"和 Pizza Hut 的"家庭聚餐"定位，Papa John's 占据了"品质"的心智高地。

**2018 年危机与转折**：

2018 年 7 月，创始人 John Schnatter 在一次电话会议中使用种族歧视性语言被曝光，引发轩然大波。他被迫辞去董事会主席职务，公司随后宣布与他"彻底切割"——禁止他在任何门店出现，甚至将官网上的照片全部撤下。品牌销量应声暴跌，股价大幅下挫。

**后 Schnatter 时代**：
Papa John's 在危机后进行了全面品牌重塑：启用新 Logo、更新门店形象、推出全新产品（如 Epic Stuffed Crust 馅料披萨）、加大数字化投资。到 2022 年，公司业绩开始恢复增长。

## 🍕 招牌产品与配方坚持

| 产品 | 特色 |
|------|------|
| **Original Crust Pizza** | 经典手工面团，冷发酵工艺 |
| **The Works** | 意大利辣香肠、香肠、洋葱、青椒、蘑菇、黑橄榄、 jalapeño 全料披萨 |
| **Epic Stuffed Crust** | 馅料填充卷边披萨，2021 年重磅推出 |
| **Papadia** | 披萨面团制成的卷饼，内陷丰富 |
| **Garlic Sauce（蒜蓉酱）** | 品牌标志性蘸酱，每份披萨必配 |
| **Pizza Dipping Sauce** | 独家配方番茄蘸酱 |
| **Boneless Wings** | 无骨鸡翅，多种口味 |
| **Chocolate Chip Dessert Pizza** | 巧克力碎片甜披萨 |

**原料承诺**：Papa John's 坚持使用不含人工色素、香精、防腐剂的原料。其番茄酱使用 vine-ripened 藤熟番茄，奶酪使用 100% 真正的马苏里拉。

## 🌍 全球版图

Papa John's 在全球 **45+ 个国家**拥有超过 **5,500 家**门店。

**市场分布**：
- **美国**：约 3,300 家门店，为最大市场
- **英国**：第二大市场，200+ 家门店
- **中东 & 北非**：快速增长区域，沙特阿拉伯、阿联酋、埃及等
- **墨西哥**：拉美最大市场
- **中国**：曾进入但经营困难，后调整策略
- **韩国、菲律宾、印度**：亚洲重点市场

**数字化创新**：
- Papa John's 是最早推出手机 App 订购的披萨品牌之一
- "Papa Rewards" 忠诚度计划
- 与 DoorDash、Uber Eats、Grubhub 深度合作
- 支持语音订购（Alexa、Google Home）
