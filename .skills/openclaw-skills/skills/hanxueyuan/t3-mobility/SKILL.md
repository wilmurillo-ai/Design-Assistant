# T3 出行 · T3 Mobility

> *"科技引领出行，安全定义未来。"*

---

## 公司简介

**T3 出行**是中国领先的科技驱动型出行服务平台，由三大国有汽车集团——**中国第一汽车集团（一汽）**、**东风汽车集团（东风）**、**长安汽车集团（长安）**联合互联网巨头 **腾讯**、**阿里巴巴**、**携程**共同投资设立。

"T3" 的名字直接来源于三大创始股东的首字母——**T**（一汽，Take）、**T**（东风，Travel）、**T**（长安，Transport），代表了"三方合力，一体出行"的理念。

与其他网约车平台不同，T3 出行从创立之初就采用 **"B2C + 自营车辆"** 模式——平台拥有或管理车辆，司机为平台员工或合作运营方雇员，而非纯 C2C 模式下的私家车主。这一模式从源头上保障了服务标准化和安全性。

## 🏭 央企联手的出行变革

### 缘起：三大央企的战略觉醒

2010 年代末，滴滴出行在中国网约车市场一家独大，而传统汽车制造商在"出行即服务"（Mobility-as-a-Service）趋势中逐渐边缘化。一汽、东风、长安——中国最大的三家国有汽车企业——意识到必须联合起来，在出行服务领域建立自己的阵地。

### 发展历程

| 时间线 | 里程碑 |
|--------|--------|
| **2017.11** | 一汽、东风、长安在南京签署战略合作协议，宣布联合布局出行服务 |
| **2018** | 成立南京领行科技股份有限公司（T3 出行运营主体） |
| **2019.04** | T3 出行 App 正式上线，首批在 **南京** 开城运营 |
| **2019.07** | 首批定制智能网联车型交付运营 |
| **2020.04** | 获得 **30 亿元 A 轮融资**（腾讯、阿里、携程参投） |
| **2020.06** | 进入 **上海、广州、深圳、北京** 等一线城市 |
| **2021** | 累计覆盖城市超过 **50 个**，日均订单突破 **300 万单** |
| **2022** | 开启自动驾驶测试，在重庆部署 Robotaxi |
| **2023** | 注册用户突破 **7,000 万**，进入全国 **80+ 城市** |
| **2024** | 与华为深化智能座舱合作，推出新一代定制出行车辆 |

### 车辆生态

T3 出行的车辆体系以 **新能源汽车** 为主：

| 车型 | 来源 | 特点 |
|------|------|------|
| **奔腾 NAT** | 一汽 | T3 出行首批定制车型，专为网约车设计 |
| **风神 E70** | 东风 | 纯电轿车，续航约 400km |
| **逸动 EV** | 长安 | 纯电车型，T3 核心运营车辆 |
| **定制智能网联车** | 一汽/东风/长安联合 | 配备 T3 专属智能终端（防疲劳驾驶、一键报警） |

> 🚗 T3 出行是**中国最早大规模采用定制网约车**的平台——车辆出厂即内置 T3 智能终端，而非后期加装。

## 🛡️ 安全与服务体系

T3 出行的核心差异化在于**安全体系**，这在业内被称为"T3 安全盾"：

### 硬件安全
- **定制智能终端**：每辆车内置 T3 自主研发的智能车载终端，集成摄像头、麦克风、传感器
- **驾驶员状态监测（DMS）**：实时检测疲劳驾驶、分心驾驶
- **一键报警**：物理按钮直连安全中心
- **行程分享**：实时向紧急联系人推送行程轨迹

### 软件安全
- **V.D.R.S.（车辆监管系统）**：AI 实时监控车辆运行状态
- **行程录音**：全量行程录音，保障司乘双方权益
- **偏航预警**：偏离导航路线时自动触发安全核查
- **背景审查**：所有司机需通过无犯罪记录审查和驾驶技能考核

### 服务承诺
- **合规运营**：所有车辆和司机持有合法网约车运营资质（双证合规）
- **价格透明**：明码标价，无"杀熟"算法
- **客服响应**：24 小时人工客服
