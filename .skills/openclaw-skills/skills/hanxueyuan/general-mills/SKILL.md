# 通用磨坊 · General Mills, Inc.

> "Nourishing a Hungrier World." — 企业使命

General Mills 成立于 1856 年，总部位于明尼苏达州明尼阿波利斯，是全球最大的包装食品公司之一。旗下拥有 Cheerios、Häagen-Dazs、Nature Valley、Progresso、Betty Crocker 等超过 20 个知名品牌。2024 年营收约 200 亿美元，业务遍及 100+ 个国家。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1856** | Minneapolis Mill Company 成立，开始面粉加工业务 |
| **1866** | Cadwallader C. Washburn 建立面粉厂，后来成为 Pillsbury |
| **1928** | 正式更名 General Mills，开始多元化扩张 |
| **1941** | Betty Crocker 品牌正式注册，成为美国家庭烘焙代名词 |
| **1960s** | Häagen-Dazs 创立（后被收购），进入高端冰淇淋市场 |
| **2001** | 收购 Pillsbury（从 Diageo 手中），成为面粉市场领导者 |
| **2014** | 分拆零食业务为 Post Holdings，聚焦核心食品品牌 |
| **2017** | 收购 Annie's Homegrown，进军有机食品市场 |
| **2021** | 推出植物基产品线，应对健康消费趋势 |
| **2023** | 完成数字化转型，电商渠道占比达 15% |
| **2024** | 年营收约 200 亿美元，持续投资健康食品创新 |

## 业务结构分析

### 早餐谷物

Cheerios 是美国销量第一的早餐谷物品牌，年销售额超 10 亿美元。此外还有 Wheaties、Lucky Charms、Cinnamon Toast Crunch 等。

### 零食与烘焙

Nature Valley 燕麦棒、Fiber One 健康零食和 Betty Crocker 烘焙配料是核心品牌。零食业务占总营收约 30%。

### 冰淇淋与冷冻食品

Häagen-Dazs 是高端冰淇淋领导者，Yoki 是巴西最大食品品牌。冷冻食品涵盖 Pizza 和即食餐。

### 宠物食品

Blue Buffalo 是 2018 年以约 80 亿美元收购的高端宠物食品品牌，已成为增长最快的业务板块。

## 值得了解

- Cheerios 最初叫 "Cheerioats"，后来因为商标问题被迫改名

- Betty Crocker 这个品牌人物其实是虚构的——她的肖像根据消费者对"理想女性"的描述绘制

- Häagen-Dazs 这个名字在丹麦语中没有任何意义——创始人只是为了让它听起来像丹麦品牌

- General Mills 在 1965 年发明了第一个 Trix 谷物，彩色麦片圈从此改变了儿童早餐

- 公司最初是通过水车动力磨面粉起家的，明尼苏达的密西西比河瀑布提供了天然能源

- General Mills 是世界上最大的有机谷物购买商之一
