# 劳氏 · Lowe's Companies

> 永不打烊，永不放弃。

Lowe's Companies, Inc. 是美国第二大家居建材零售商，仅次于Home Depot。公司在北美经营约1700家门店，年营收约860亿美元。传统上以DIY家庭消费者为主要客户群，但近年来大力拓展专业承包商业务以缩小与Home Depot的差距。Lowe's以更好的购物体验、更明亮的门店布局和更友好的客户服务著称。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1921年** | 卢修斯·史密斯在北卡罗来纳州开设一家五金杂货店 |
| **1946年** | 卡尔·贝尔接手经营，开始转型为现代家居建材店 |
| **1955年** | 贝尔兄弟收购史密斯的股份，公司更名为Lowe's |
| **1961年** | 在纽约证券交易所上市 |
| **1980年代** | 开始向大店模式转型，引入品类管理 |
| **2005年** | 进入加拿大市场，收购Rona部分股份 |
| **2018年** | Marvin Ellison出任CEO，启动全面转型计划 |
| **2019年** | 宣布17亿美元投资供应链和数字化升级 |
| **2021年** | 专业客户业务全面提速，推出专项服务计划 |
| **2024年** | 关闭部分低效门店，聚焦高产出地点，推进全渠道战略 |

## 业务结构分析

### DIY零售

Lowe's的核心业务，面向家庭消费者提供建材、工具、电器、园艺和装修产品。相比Home Depot，Lowe's门店更注重购物体验，通道更宽、照明更好、商品陈列更有序。

### 专业客户（Pro）

增长最快的业务板块，面向承包商和装修专业人士。推出Lowe's for Pros计划，提供批量定价、专属客户经理和快速配送服务。

### 电器与家居设备

Lowe's是全美最大的家电零售商之一，提供冰箱、洗衣机、烤箱等大型家电的销售、配送和安装服务，这是区别于Home Depot的重要差异化优势。

### 数字商务与全渠道

Lowe's大力投资电商平台和全渠道体验，支持线上下单门店提货、当日配送和AR可视化等数字化功能，2023年线上销售占比约13%。

## 🔑 核心护城河

```
  购物体验优化 → 家庭客户偏好 → 品牌忠诚
       ↓                          ↗
  供应链升级 → 履约效率 → 全渠道整合
       ↓
  Pro业务拓展 → 大额采购 → 收入多元化
```

1. 购物体验差异化：Lowe's以更好的购物环境著称，通道更宽、照明更亮、商品陈列更有条理，对DIY家庭消费者尤其友好
2. 家电品类优势：Lowe's在全美家电零售市场占据领先地位，提供一站式家电选购、配送和安装服务，构成独特的品类护城河
3. 供应链后发优势：2019年开始的17亿美元供应链投资使Lowe's在履约效率和库存管理上快速追赶Home Depot
4. 品牌心智定位：在DIY消费者心中，Lowe's常被认为比Home Depot更友好，对女性消费者和新手的吸引力更强

## 值得了解

- Lowe's标志性的蓝色标志和门店配色是刻意为之，目的是在视觉上与Home Depot的橙色形成对比

- Lowe's是全美最大的家电零售商之一，每年销售超过500万台大型家电，远超大多数竞争对手

- Lowe's的吉祥物是一只名为Louie的猎犬，出现在其多支广告中

- 公司创始人卢修斯·史密斯最初开的是一家五金杂货店，卖钉子、绳索和农用工具，与现代Lowe's的大店形象截然不同
