# 百威英博 · Anheuser-Busch InBev S.A.

> "Dream Big" — ABI企业口号

Anheuser-Busch InBev S.A.是全球最大的啤酒酿造商，总部位于比利时鲁汶。公司通过一系列激进并购（Interbrew收购AmBev、AB收购InBev等）建立了全球最大的啤酒品牌帝国，拥有超过500个啤酒品牌，包括百威、科罗娜、时代啤酒、福佳等。ABI在全球约50个国家运营，年啤酒销量超过4亿百升。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1366** | Stella Artois品牌在比利时鲁汶创立(起源) |
| **1876** | Adolphus Busch在美国创立百威啤酒 |
| **1977** | Interbrew在比利时成立 |
| **1999** | AmBev在巴西成立 |
| **2004** | AmBev收购Interbrew，成立InBev |
| **2008** | InBev以520亿美元收购Anheuser-Busch |
| **2013** | 与3G资本合作，激进的成本削减文化形成 |
| **2016** | 以超过1000亿美元收购SABMiller，史上最大并购 |
| **2019** | 品牌组合优化，聚焦核心高端品牌 |
| **2021** | Beyond Beer战略，布局硬苏打和精酿 |
| **2023** | 全球啤酒销量超过4亿百升 |
| **2024** | 年营收超过570亿美元，高端化战略持续推进 |

精酿啤酒、硬苏打和无酒精啤酒

## 值得了解

- 百威英博是全球首家年啤酒销量超过4亿百升的啤酒公司

- 3G资本的零基预算文化使ABI成为全球效率最高的啤酒公司

- Bud Light曾经是美国最畅销的啤酒，连续数十年保持第一

- Corona在墨西哥生产但被ABI收购后成为全球品牌

- Stella Artois的名字来源于1926年的圣诞酿造传统

- ABI在巴西的市场份额超过70%，几乎垄断

- 公司每年的啤酒花采购量超过全球总产量的15%

- ABI的无酒精啤酒品牌是增长最快的细分品类
