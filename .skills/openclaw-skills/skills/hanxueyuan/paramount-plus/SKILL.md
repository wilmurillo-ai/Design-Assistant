# Paramount+ 流媒体服务 · Paramount+

Paramount+ 是派拉蒙全球（Paramount Global）旗下的订阅制流媒体平台，前身为 CBS All Access（2014–2021）。平台整合了 CBS、派拉蒙影业、Comedy Central、MTV、尼克儿童频道（Nickelodeon）、史密森尼频道等多个品牌的内容资源，提供电影、电视剧、原创节目、体育直播和新闻等综合服务。

---

## 品牌演进时间线

```
CBS All Access (2014)
    │
    ├── 2018 年更名为 CBS All Access 国际版
    │
    ▼
Paramount+ (2021年3月4日 美国首发)
    │
    ├── 2021 年扩展至拉丁美洲、北欧
    ├── 2022 年进入澳大利亚、韩国、东南亚
    ├── 2022 年 6 月登陆英国、爱尔兰
    ├── 2023 年覆盖 190+ 国家和地区
    │
    ▼
用户突破 7000 万（2023 年底）
```

## 核心内容生态

### 🎬 电影库
- **派拉蒙百年片库**：数千部电影，包括《教父》《阿甘正传》《泰坦尼克号》
- **院线同步**：部分电影在院线上映后 30–45 天上线平台
- **独家电影**：原创电影如《蛇眼》《宠物联盟》

### 📺 剧集内容
| 品牌来源 | 代表剧集 |
|----------|----------|
| CBS | 《星际迷航》系列、《傲骨之战》、《犯罪现场调查》系列 |
| Showtime | 《亿万》《国土安全》《黄蜂》 |
| Comedy Central | 《南方公园》《每日秀》 |
| Nickelodeon | 《海绵宝宝》《忍者神龟》衍生剧 |
| MTV | 《真实世界》经典合集 |

### 🏈 体育直播
- UEFA 欧冠联赛（美国独家流媒体）
- NFL on CBS 赛事
- NCAA 大学篮球锦标赛（"疯狂三月"）
- PGA 高尔夫锦标赛

### 📰 新闻
- CBS News 24/7 直播
- 60 Minutes 完整版
- 本地 CBS 附属频道直播

## 市场定位

在流媒体红海竞争中，Paramount+ 采取了差异化策略：

| 竞争者 | 核心优势 |
|--------|----------|
| Netflix | 海量原创内容和全球布局 |
| Disney+ | 漫威、星战、皮克斯等顶级 IP |
| HBO Max/Max | HBO 优质剧集和华纳电影库 |
| **Paramount+** | **CBS 新闻+体育直播+派拉蒙经典电影+南方公园** |

Paramount+ 的独特卖点在于：
1. **体育直播**：NFL 和欧冠直播吸引体育迷
2. **新闻整合**：CBS News 提供实时新闻覆盖
3. **合家欢**：Nickelodeon 内容覆盖儿童市场
4. **经典 IP**：《星际迷航》系列在平台持续扩展
