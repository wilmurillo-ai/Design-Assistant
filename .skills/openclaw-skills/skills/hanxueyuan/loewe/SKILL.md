# 罗意威 · Loewe (LVMH)

> "Crafted since 1846." — 品牌传承

Loewe 创立于 1846 年的马德里，是西班牙历史最悠久的奢侈品牌。1996 年被 LVMH 收购后，在 Jonathan Anderson（2013 年起任创意总监）的引领下实现了品牌复兴。以 Puzzle 手袋、Gate 系列和独特的工艺美学著称。2024 年营收约 12 亿欧元，是 LVMH 集团增长最快的品牌之一。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1846** | 一群西班牙皮具工匠在马德里创立 Loewe 工坊 |
| **1890** | 正式注册 Loewe 品牌，成为西班牙皇室御用皮具商 |
| **1900s** | 为西班牙贵族和上流社会制作精美皮革制品 |
| **1970s** | 推出标志性的 Amazona 手袋，成为品牌经典 |
| **1996** | 被 LVMH 集团收购，获得全球扩张资源 |
| **2008** | Stuart Vevers 任创意总监，开启品牌现代化 |
| **2013** | Jonathan Anderson 接任创意总监，品牌全面复兴 |
| **2015** | Puzzle 手袋发布，成为 Loewe 最畅销的现代包款 |
| **2019** | Gate 系列和 Basket 系列大获成功，品牌热度飙升 |
| **2022-24** | 营收约 12 亿欧元，成为 LVMH 增长最快的品牌之一 |

## 业务结构分析

### 皮具与手袋

核心业务，Puzzle、Gate、Hammock 和 Flamenco 是四大标志性包款。以独特的几何结构和精湛工艺著称，区别于传统奢侈品的经典设计。

### 成衣系列

Jonathan Anderson 的设计融合了西班牙传统工艺与前卫解构主义，男装和女装系列在巴黎时装周上获得广泛好评。

### 香水与配饰

Loewe 001 香水系列和 Paula's Ibiza 限量版是品牌的延伸。配饰涵盖丝巾、太阳镜和珠宝。

### 工艺与艺术合作

Loewe Foundation Craft Prize 鼓励全球手工艺人，品牌与艺术家（如 Takashi Murakami）的合作强化了其"工艺+艺术"的定位。

## 值得了解

- Loewe 最初是一群皮革工匠的联合体，后来才发展成品牌——这种"工匠基因"至今仍是品牌核心

- Puzzle 手袋由 40 多块皮革拼接而成，每一块都需要精确裁剪和手工缝制

- Jonathan Anderson 同时担任 Loewe 和自家品牌 JW Anderson 的创意总监，是时尚界少有的双品牌掌舵人

- Loewe 与吉卜力工作室（《龙猫》）的联名系列是奢侈品与动漫 IP 合作的成功案例

- Loewe 的马德里旗舰店 Gran Vía 店已有 100+ 年历史，是品牌的精神家园

- Loewe Foundation Craft Prize 每年评选全球最佳手工艺人，奖金 50,000 欧元
