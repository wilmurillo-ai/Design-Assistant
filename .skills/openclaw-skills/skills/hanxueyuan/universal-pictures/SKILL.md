# 环球影业 · Universal Pictures

环球影业（Universal Pictures）是好莱坞最老牌的电影制片厂之一，由卡尔·莱姆勒（Carl Laemmle）于 1912 年创立。作为好莱坞"五大"之一，环球影业拥有《侏罗纪公园》《速度与激情》《神偷奶爸》《谍影重重》等全球知名 IP。环球影业是 NBC 环球集团（NBCUniversal）的核心电影业务部门，而 NBC 环球最终归属于康卡斯特（Comcast）。

---

## 历史沿革

| 年代 | 事件 |
|------|------|
| **1912** | 卡尔·莱姆勒在加州创立环球电影制造公司（Universal Film Manufacturing Company） |
| **1915** | 环球影城（Universal City）开放 — 全球首个综合性电影制片厂园区 |
| **1920s–30s** | 经典恐怖片黄金期：《科学怪人》《德古拉》《隐形人》 |
| **1930s** | 音乐剧和情节剧盛行 |
| **1940s–50s** | 黑色电影经典期；环球国际（Universal International）成立 |
| **1962** | 被 MCA（Music Corporation of America）收购 |
| **1990** | MCA 被日本松下电器收购 |
| **1995** | 加拿大 Seagram 收购 MCA/Universal |
| **2004** | NBC 与 Vivendi Universal 合并成立 NBC 环球 |
| **2011** | 康卡斯特收购 NBC 环球多数股权 |
| **2013** | 康卡斯特完成 100% 收购 |

## 超级 IP 家族

环球影业拥有商业价值极高的 IP 矩阵：

### 侏罗纪系列 🦖
- **《侏罗纪公园》**（1993）：斯皮尔伯格执导，开启 CGI 恐龙时代
- **《侏罗纪世界》**（2015）：全球票房 16.7 亿美元
- 全系列累计票房超过 **60 亿美元**
- 主题园区全球多个环球影城的核心景点

### 速度与激情 🏎️
- **《速度与激情》**（2001–2023）：从街头赛车发展为全球动作系列
- 全系列 11 部电影累计票房超过 **70 亿美元**
- 《速度与激情 7》（2015）全球 15.1 亿美元
- 《速度与激情 10》（2023）全球 7.04 亿美元

### 照明娱乐（Illumination）动画 🎈
- **《神偷奶爸》/《小黄人》**系列：全球累计超 45 亿美元
- **《爱宠大机密》**系列
- **《超级马力欧兄弟大电影》**（2023）：全球 13.6 亿美元，改编游戏电影票房纪录

### 其他重要 IP
