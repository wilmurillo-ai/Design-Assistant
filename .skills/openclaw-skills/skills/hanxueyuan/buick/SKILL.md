# buick

## 城市概况
别克（Buick）是美国通用汽车旗下历史最悠久的汽车品牌之一，1903 年创立。在中国市场由上汽通用生产，以君威、昂科威、GL8 等车型畅销。

官方网站：https://www.buick.com
