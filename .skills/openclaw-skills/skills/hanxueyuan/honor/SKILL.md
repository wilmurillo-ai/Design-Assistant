# 荣耀 (HONOR)

中国智能手机品牌，原华为子品牌，现独立运营。

## 品牌介绍

**荣耀**成立于 2013 年，原为华为旗下互联网手机品牌，2020 年独立运营。

### 基本信息

| 项目 | 详情 |
|------|------|
| **成立时间** | 2013 年 |
| **总部** | 中国深圳 |
| **母公司** | 深圳市智信新信息技术 |
| **品牌定位** | 年轻科技潮牌 |

### 热门产品

| 系列 | 类型 | 特点 |
|------|------|------|
| **Magic 系列** | 旗舰手机 | 高端配置 |
| **数字系列** | 中高端 | 时尚设计 |
| **X 系列** | 中端 | 性价比 |
| **Play 系列** | 入门 | 年轻用户 |
| **笔记本** | 电脑 | 轻薄本 |

### 技术特色

- 麒麟芯片（独立前）
- 影像技术
- 快充技术
- 智慧交互
