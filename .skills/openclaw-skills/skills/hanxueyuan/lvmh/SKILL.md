# LVMH · LVMH Moët Hennessy Louis Vuitton

> "奢侈品不是价格高昂的东西，而是无法被替代的东西。" — Bernard Arnault

LVMH 是全球最大的奢侈品集团，由 Bernard Arnault 通过一系列并购打造而成。集团旗下拥有 75 多个顶级品牌，涵盖时装皮具、香水美妆、腕表珠宝、精选零售和香槟烈酒五大领域。2024 年营收约 862 亿欧元，Bernard Arnault 曾多次登顶全球首富。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1987** | Louis Vuitton 与 Moët Hennessy 合并，正式成立 LVMH 集团 |
| **1989** | Bernard Arnault 通过资本市场运作获得 LVMH 控制权 |
| **1990s** | 陆续收购 Givenchy、Sephora、TAG Heuer 等品牌 |
| **2011** | 收购 Bulgari，加强珠宝腕表板块 |
| **2017** | 收购 Christian Dior 时装业务，整合品牌矩阵 |
| **2021** | 收购 Tiffany & Co.（162 亿美元），奢侈品史上最大并购 |
| **2023** | Bernard Arnault 个人财富一度超过 2,000 亿美元，成为世界首富 |

## 🔑 核心护城河

```
收购顶级品牌 → 保持独立运营 → 共享供应链/渠道 → 品牌溢价 → 更高利润 → 更多收购资金
```

1. 品牌组合的深度和广度无与伦比，覆盖从入门奢华到顶级高定全价格带
2. 每个品牌保持独立创意总监和运营，同时共享集团供应链和渠道优势
3. Bernard Arnault 的收购眼光和整合能力，40 年持续并购无人能及
4. 稀缺性和传统工艺构成的品牌护城河，新品牌需要百年才能建立同等声望
