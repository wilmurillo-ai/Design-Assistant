# 博世 · Robert Bosch GmbH

> "宁肯承受损失，也不违背信誉。"

博世（Bosch）是德国最大的工业集团之一，由罗伯特博世于 1886 年在斯图加特创立。公司是全球最大的汽车零部件供应商，在工业自动化、家电、安防和 IoT 领域也处于领先地位。2024 年营收约 916 亿欧元，全球员工约 429,000 人。公司独特的基金会所有制使其能进行长期战略规划。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1886** | 罗伯特博世在斯图加特开设精密机械和电气工程车间 |
| **1897** | 发明世界上第一台汽车用高压点火磁电机 |
| **1970s** | ABS 防抱死制动系统研发成功，革命性提升汽车安全 |
| **2000s** | 大规模扩张亚洲市场，中国成为第二大市场 |
| **2010s** | IoT 和自动驾驶转型，投资传感器和 AI 技术 |
| **2024** | 营收约 916 亿欧元，半导体业务投资 30 亿欧元在德累斯顿建厂 |

## 业务结构分析

### 汽车与智能交通

燃油系统、底盘控制系统、自动驾驶传感器，贡献约 55% 收入

### 工业技术

工业自动化、驱动与控制、包装技术

### 消费品

博世家电、电动工具，与西门子合资后独立运营

### 能源与建筑技术

暖通空调、安防系统、智能家居

## 🔑 核心护城河

```
汽车零部件技术 → 大规模生产 → 成本优势 → 更多客户 → 研发投入 → 更多技术领先
```

1. 基金会所有制使公司能进行 10-20 年的长期投资，不受短期股价压力影响
2. 汽车零部件领域的全球第一地位，几乎每辆车都有博世零件
3. ABS 等革命性发明定义了汽车行业安全标准
4. 德累斯顿半导体工厂是欧洲最大的车规芯片厂之一

## 值得了解

博世发明了汽车的 ABS 和 ESP 系统，这些安全配置现在几乎每辆车都有

公司 92% 由罗伯特博世基金会持有，基金会用分红资助科学研究和社会福利

博世在二战期间曾被要求为纳粹生产军用设备，战后进行了深刻反思

德累斯顿晶圆厂是欧洲最先进的 300mm 车规芯片工厂之一
