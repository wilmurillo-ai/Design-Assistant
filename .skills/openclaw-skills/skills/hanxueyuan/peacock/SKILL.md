# Peacock 流媒体 · Peacock

Peacock（孔雀）是 NBC 环球集团（NBCUniversal）旗下的流媒体服务平台，2020 年 7 月正式上线。平台命名源自 NBC 的标志性孔雀 Logo。Peacock 以独特的免费增值（Freemium）模式切入市场，提供免费、付费含广告和付费无广告三个层级，内容涵盖 NBC 节目库、环球影业电影、WWE 赛事、Premier League 英超联赛以及大量原创内容。

---

## 发展里程碑

| 时间节点 | 关键事件 |
|----------|----------|
| **2019** | NBC 环球宣布推出自有流媒体平台 |
| **2020.04** | 为康卡斯特 Xfinity 用户提前开放 |
| **2020.07.15** | 正式面向全美用户开放 |
| **2020.10** | 推出三层定价模式：免费 / Premium 含广告 / Premium Plus 无广告 |
| **2022.01** | 取消免费层级（新用户），保留付费模式 |
| **2022.10** | 宣布用户突破 2000 万 |
| **2023.08** | 与 Netflix 达成内容授权协议 |
| **2024** | 用户突破 3300 万，加强体育和 WWE 内容投入 |

## 内容版图

### NBC 电视遗产
Peacock 的核心吸引力来自 NBC 庞大而经典的节目库：
- **《办公室》**（The Office）— Peacock 独家流媒体版权，是平台最大的引流内容
- **《公园与游憩》**（Parks and Recreation）
- **《法律与秩序》系列**（Law & Order franchise）
- **《周六夜现场》**（SNL）最新和经典剧集

### 环球影业电影
- 环球影业院线电影在 Peacock 上线窗口缩短至 45 天
- 《侏罗纪世界》《速度与激情》《神偷奶爸》等系列
- 经典影片如《E.T.》《大白鲨》《回到未来》

## 商业模式分析

Peacock 采用了与传统流媒体不同的策略：

```
┌──────────────────────────────────────────────┐
│               Peacock 三层模式                │
├──────────────────────────────────────────────┤
│  Free (免费)          │ 广告 + 内容受限        │ ← 获客入口
│  Premium ($5.99/月)   │ 广告 + 完整内容库      │ ← 主力层级
│  Premium Plus ($11.99)│ 无广告 + 离线下载      │ ← 高端用户
└──────────────────────────────────────────────┘
```

**独特优势：**
1. **免费获客**：免费层级降低了用户门槛
2. **母公司生态**：可借助康卡斯特有线电视用户直接导流
3. **体育独占**：英超和 WWE 构成强大护城河
4. **《办公室》独家**：一部剧就为平台带来数百万用户
