# 瑞银集团 · UBS

> 当 2023 年 3 月的一个周日夜晚，瑞士政府在 48 小时内逼迫瑞银吞下瑞士信贷时，全球银行业的历史被改写了。

---

## 1862 年至今：跨越三个世纪的故事

UBS 的故事从**两家独立的瑞士银行**开始——它们最终合并，创造了全球最强大的财富管理机构。

### 两家创始银行

| 银行 | 创立年份 | 发源地 | 历史角色 |
|------|---------|--------|---------|
| **Bank in Winterthur** | 1862 | 温特图尔 | 为瑞士工业革命提供融资 |
| **Swiss Bank Corporation (SBC)** | 1872 | 巴塞尔 | 国际贸易融资与跨境银行 |

**1912 年**，Bank in Winterthur 收购了 Toggenburger Bank，更名为 **Union Bank of Switzerland**——"UBS" 这个名字从此诞生。

**1998 年**，Union Bank of Switzerland（UBS）与 Swiss Bank Corporation（SBC）完成**世纪合并**，新公司保留了 **UBS** 这个名字。

### 关键历史节点

| 年份 | 事件 |
|:----:|------|
| **1998** | UBS 与 SBC 合并，诞生现代 UBS |
| **2000** | 收购 PaineWebber——大举进军美国财富管理市场 |
| **2008** | 金融危机中遭受重创，接受瑞士政府 600 亿瑞郎救助 |
| **2012** | 任命 Sergio Ermotti 为 CEO，启动战略转型：聚焦财富管理 |
| **2014** | 退出固定收益交易等高风险业务 |
| **2020** | Ralph Hamers 接任 CEO |
| **2022** | 瑞士信贷危机开始发酵 |
| **2023.3.19** | **瑞士政府促成 UBS 以 30 亿瑞郎收购瑞士信贷**——167 年历史的瑞银一夜归零 |
| **2023-2025** | 整合瑞士信贷业务，裁员数千人，剥离部分资产 |

## 🏦 业务架构：四大支柱

### ① 全球财富管理（Global Wealth Management）

**这是 UBS 的核心灵魂。**

| 指标 | 数据 |
|------|------|
| 管理资产（AUM） | ~4.9 万亿美元（2024，含瑞信整合后） |
| 全球最大私人银行 | ✅ |
| 服务客户 | 高净值（HNWI）+ 超高净值（UHNWI）+ 家庭办公室 |
| 客户门槛 | 通常 CHF 500K 起，私行客户 CHF 5M+ |

UBS 的财富管理业务覆盖了全球超过 **50% 的十亿美元级富豪**。

### ② 个人与公司银行（Personal & Corporate Banking）

- 瑞士本土零售银行市场**领导者**
- 为瑞士个人客户和企业提供全面银行服务
- 在瑞士拥有约 **240 家分行**

### ③ 资产管理（Asset Management）

- 管理资产规模约 **1 万亿美元**
- 产品线涵盖 ETF、主动管理基金、另类投资
- 全球最大 ETF 提供商之一（通过与 ETF Securities 等合作）

### ④ 投资银行（Investment Bank）

- 2023 年后大幅缩减规模
- 保留瑞士信贷的某些优势领域（如大宗商品交易）
- 战略定位：服务财富管理客户的投行需求（Wealth-Driven IB）

## 💥 收购瑞士信贷：世纪交易

### 交易概要

| 项目 | 详情 |
|------|------|
| 收购价格 | **30 亿瑞郎**（全股票） |
| 原瑞信市值（交易前） | 约 60 亿瑞郎 |
| 原瑞信总资产 | ~1.5 万亿瑞郎 |
| 政府支持 | 瑞士国家银行提供 1,000 亿瑞郎流动性贷款 |
| 政府担保 | 瑞士政府提供约 90 亿瑞郎损失担保 |
| AT1 债券 | **170 亿瑞郎 AT1 债券被全部减记为零**（震惊全球债券市场） |
| 完成时间 | 2023 年 6 月 |

### 整合挑战

- **裁员**：全球裁员数千人（瑞士信贷约 25,000 员工中的大部分）
- **资产剥离**：剥离部分投行业务、退出部分市场
- **文化冲突**：UBS 的保守风险文化 vs 瑞信的高风险交易文化
- **品牌整合**：瑞士信贷品牌在大多数市场被逐步淘汰

## 🇨🇭 瑞士银行的全球地位

UBS 的存在与瑞士的独特地位密不可分：

| 维度 | 说明 |
|------|------|
| **银行保密传统** | 1934 年瑞士银行法确立保密制度，吸引全球富豪（2009 年后逐步放松） |
| **政治中立** | 瑞士的中立国地位提供了政治稳定性 |
| **法郎稳定** | 瑞士法郎是全球最主要的避险货币之一 |
| **金融中心** | 苏黎世和日内瓦是全球最重要的私人银行中心 |
| **监管环境** | 瑞士金融市场监管局（FINMA）监管严格但相对灵活 |
