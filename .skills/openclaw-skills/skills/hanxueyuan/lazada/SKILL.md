# Lazada

东南亚领先的电商平台。

## 平台介绍

**Lazada**成立于 2012 年，总部位于新加坡，是阿里巴巴集团旗下的东南亚电商平台。

### 基本信息

| 项目 | 详情 |
|------|------|
| **成立时间** | 2012 年 3 月 |
| **母公司** | 阿里巴巴集团 (2016 年收购) |
| **总部** | 新加坡 |
| **覆盖国家** | 6 |
| **月活用户** | 1.5 亿 + |
| **卖家数量** | 100 万 + |
