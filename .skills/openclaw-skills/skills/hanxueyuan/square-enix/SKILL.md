# 史克威尔艾尼克斯 · Square Enix

史克威尔艾尼克斯（Square Enix）是日本最具代表性的电子游戏开发商与发行商之一，由 Square 和 Enix 两家公司于 2003 年合并而成。公司以《最终幻想》（Final Fantasy）和《勇者斗恶龙》（Dragon Quest）两大 JRPG 系列闻名于世，在角色扮演游戏、动作冒险游戏领域拥有不可替代的地位。

---

## 合并前后

### Square 时代（1986–2003）
| 年份 | 事件 |
|------|------|
| **1986** | 电友社（Electro-Square）创立游戏开发部门 |
| **1987** | 首部《最终幻想》在任天堂 FC 平台发售，拯救公司免于破产 |
| **1994** | 《最终幻想 VI》被誉为 16-bit 时代 RPG 巅峰 |
| **1997** | **《最终幻想 VII》** 在全球引爆 JRPG 热潮，PS 平台销量超 1000 万 |
| **2001** | CG 电影《最终幻想：灵魂深处》票房惨败，导致 Square 财务危机 |
| **2002** | 与任天堂和解（FC 时代决裂后），为合并铺路 |

### Enix 时代（1975–2003）
| 年份 | 事件 |
|------|------|
| **1975** | Enix 前身"日本业余程序俱乐部"成立 |
| **1986** | **《勇者斗恶龙》** 首部发售，成为日本"国民 RPG" |
| **1990s** | 《勇者斗恶龙》系列成为日本销量最高的游戏系列之一 |

### 合并之后（2003–至今）
| 年份 | 事件 |
|------|------|
| **2003** | Square 与 Enix 正式合并为 Square Enix Holdings |
| **2005** | 收购 Taito（太东）公司 |
| **2009** | 收购 Eidos Interactive（《古墓丽影》《杀出重围》开发商） |
| **2020** | 《最终幻想 VII 重制版》发售，全球首周 350 万份 |
| **2022** | 出售部分西方工作室（包括《古墓丽影》IP）给 Embracer Group |
| **2024** | 《最终幻想 VII 重生》《龙之信条 2》等作品持续推动 |

## 旗舰 IP 图谱

### 《最终幻想》系列（1987–至今）
- 正传 16 部 + 大量衍生作品
- **《最终幻想 VII》**：JRPG 史上最具影响力的作品
- **《最终幻想 XIV》**：MMORPG，从失败重做后成为业界标杆
- **《最终幻想 XVI》**（2023）：转向动作化的最新正传

### 《勇者斗恶龙》系列（1986–至今）
- 日本"国民 RPG"，每部新作发售都被视为社会事件
- 鸟山明（《龙珠》作者）负责人设
- 最新作《勇者斗恶龙 XII》开发中

### 其他重要 IP
| IP | 类型 | 备注 |
|----|------|------|
| 《王国之心》（Kingdom Hearts） | 动作 RPG | 迪士尼角色联动，全球销量超 3600 万 |
| 《尼尔》系列（NieR） | 动作 RPG | 横尾太郎创作，以哲学叙事著称 |
| 《古墓丽影》（Tomb Raider） | 动作冒险 | 2009 年通过 Eidos 获得（2022 年已出售） |
| 《杀出重围》（Deus Ex） | 沉浸式模拟 | Eidos 旗下经典 IP（已出售） |

## 行业地位与贡献

Square Enix 在游戏史上的关键贡献包括：

1. **JRPG 全球化推手**：《最终幻想 VII》让日式 RPG 走向全球主流
2. **CG 动画技术**：早期投入大量资源开发 CGI 动画，推动了游戏过场动画的演进
3. **MMORPG 复兴**：《最终幻想 XIV：重生之境》（2013）是游戏史上最成功的翻盘案例
4. **跨媒介叙事**：《最终幻想》系列的电影、动画、音乐（植松伸夫作曲）构成庞大的文化生态
