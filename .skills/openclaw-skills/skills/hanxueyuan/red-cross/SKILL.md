# 红十字会 · Red Cross

---

## 发展历程

| 时期 | 关键事件 |
|:----:|--------|
| **1863 年** | 亨利·杜南在日内瓦创立国际红十字委员会 |
| **1901 年** | 亨利·杜南获得首届诺贝尔和平奖 |
| **1906-1949 年** | 日内瓦公约确立红十字标志的保护地位 |
| **2020s** | 在全球 192 个国家开展人道主义工作 |

---

## 核心特点

灾难救援 + 战地医疗 + 血液供应 + 人道主义教育

## 核心优势

全球最古老和最大的人道主义网络
