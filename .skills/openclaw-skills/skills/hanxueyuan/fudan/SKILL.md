# 复旦大学 · Fudan

---

## 发展历程

| 时期 | 关键事件 |
|:----:|--------|
| **1905 年** | 马相伯创立复旦公学 |
| **1917 年** | 迁至江湾校区 |
| **1941 年** | 抗战期间内迁重庆 |
| **2000 年** | 与上海医科大学合并 |
| **2020s** | QS 世界大学排名前 50——中国大学前三 |

---

## 核心特点

本科 + 研究生（文理/医学/商学/新闻/国际关系）

## 核心优势

中国最顶尖的综合性大学之一 + 上海地理位置优势

## 值得了解

复旦的名字取自《尚书》'日月光华，旦复旦兮'——寓意光明日复一日
