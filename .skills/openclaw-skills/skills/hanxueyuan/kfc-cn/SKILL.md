# KFC 肯德基技能

## 技能描述
KFC（肯德基）是全球知名快餐连锁品牌，以炸鸡、汉堡、早餐闻名。本技能帮助用户查询肯德基产品、优惠、门店信息。

## 核心功能
- 🍗 **产品查询**：炸鸡、汉堡、套餐
- 💰 **优惠信息**：优惠券、促销活动
- 🏪 **门店查询**：附近门店、营业时间
- 📱 **APP 功能**：手机点餐、会员积分
- 🍳 **早餐菜单**：早餐粥、油条
- 🎁 **儿童套餐**：玩具、儿童餐

## 使用场景
- 查询附近肯德基
- 获取优惠信息
- 了解产品菜单
- 手机点餐指导

## 注意事项
- 优惠可能因地区而异
- 营业时间可能调整
- 部分产品限时供应
