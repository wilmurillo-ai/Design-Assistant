# 💻 Cursor — AI 代码编辑器

**Cursor** 是一款 AI 驱动的代码编辑器，基于 VS Code 构建，深度集成大语言模型，支持代码生成、编辑、调试和重构。

## 核心信息

| 项目 | 详情 |
|------|------|
| **公司** | Cursor Inc. |
| **创始人** | Aman Sanger, Michael Truell, Arvid Lunnemark |
| **发布时间** | 2022 年 |
| **官网** | https://cursor.com |
| **定价** | 免费 + Pro $20/月 |
| **基于** | VS Code (兼容所有 VS Code 插件) |

## 核心功能

1. **AI 代码生成** — 根据注释或需求生成完整代码
2. **智能编辑** — 选中代码后让 AI 修改、优化、添加功能
3. **代码审查** — AI 自动检查代码问题并给出建议
4. **调试助手** — 分析错误信息，提供修复方案
5. **知识库** — 索引整个项目，理解上下文

## 与 GitHub Copilot 对比

| 特性 | Cursor | GitHub Copilot |
|------|--------|----------------|
| **编辑器** | 独立 (基于 VS Code) | VS Code 插件 |
| **上下文理解** | 项目级索引 | 文件级 |
| **编辑能力** | 直接修改代码 | 建议为主 |
| **定价** | $20/月 | $10/月 |
