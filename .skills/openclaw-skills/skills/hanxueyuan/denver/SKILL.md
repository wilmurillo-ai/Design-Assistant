# 丹佛 · Denver

> 丹佛不是终点，而是通往落基山脉的门户。

丹佛是美国科罗拉多州首府和最大城市，位于落基山脉东麓、海拔恰好一英里（1609米），因此得名"一英里之城"（Mile High City）。大都会区人口约300万，是美国西部增长最快的都市区之一。丹佛已从传统的采矿和农业经济转型为航空航天、科技、清洁能源和户外产业的区域中心。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1858年** | 派克峰淘金热引发，首批采矿者在南普拉特河畔建立定居点 |
| **1861年** | 科罗拉多领地成立，丹佛被选为领地首府 |
| **1876年** | 科罗拉多成为美国第38个州，丹佛正式成为州首府 |
| **1893年** | 白银价格崩盘引发经济危机，丹佛经历严重衰退 |
| **1902年** | 丹佛正式成为独立的市-县（consolidated city-county） |
| **1940-50年代** | 二战期间军事基地和军工厂建设推动城市工业化 |
| **1961年** | 丹佛国际机场选址确定（实际1995年建成通航） |
| **1970年代** | 石油危机推动能源产业繁荣，丹佛成为"能源首都" |
| **2012年** | 科罗拉多州成为全美首个合法化娱乐用大麻的州，丹佛引领产业 |
| **2020年代** | 科技产业爆发，谷歌、亚马逊等巨头在丹佛大规模扩张 |

## 业务结构分析

### 航空航天与国防

丹佛是美国航空航天业的重要枢纽，巴克利太空部队基地和众多国防承包商驻扎于此，洛克希德·马丁等公司在该区域有大量雇员。

### 科技产业

丹佛科技生态近年来快速增长，被称为"硅山"（Silicon Mountain），吸引大量加州科技公司设立分部，2023年科技行业就业人数超10万。

### 清洁能源

科罗拉多州是美国清洁能源领先州之一，丹佛聚集了大量太阳能、风能和储能技术公司，国家可再生能源实验室（NREL）位于大丹佛区。

### 户外与旅游

作为通往落基山脉的门户，丹佛本身是户外经济中心，拥有超过300家户外产业公司总部或重要办公室，包括REI、VF Corporation等。

## 🔑 核心护城河

```
  地理门户位置 → 户外产业生态 → 人才吸引力
       ↓                    ↗
  低成本优势（vs加州）→ 科技产业迁入
       ↓
  航空航天基础 → 国防合同稳定收入
```

1. 独特的地理位置：落基山脉门户位置赋予其户外经济不可替代的地理优势，同时高海拔气候吸引追求生活方式的人才
2. 生活成本相对优势：相比硅谷和西雅图，丹佛住房成本和生活费用低约30-40%，持续吸引加州科技人才和企业迁入
3. 多元化产业基础：不依赖单一产业，航空航天、科技、能源、医疗和旅游业形成互补的经济结构，抗风险能力强
4. 政策创新领先者：率先大麻合法化、清洁能源政策激进等创新政策吸引特定产业和年轻人口流入

## 值得了解

- 丹佛国际机场（DEN）比曼哈顿岛还大，是美国面积第二大的机场，占地135平方公里，超过旧金山整个城市的面积

- 丹佛国际机场的航站楼顶部设计形似落基山脉的雪峰，同时 conspiracy theorists 认为其地下隐藏着世界末日避难所和新世界秩序的标志

- 丹佛的"一英里之城"称号是精确的——科罗拉多州议会大厦第15级台阶的海拔恰好是海拔1英里（5280英尺）

- 丹佛是美国唯一一个拥有职业体育联盟所有四大联赛球队（NFL、MLB、NBA、NHL）且均使用相同配色方案的城市
