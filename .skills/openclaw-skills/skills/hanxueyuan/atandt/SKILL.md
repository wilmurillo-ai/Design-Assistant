# 美国电话电报 · AT&T

> "电信帝国的百年沉浮，从垄断到重组的传奇"

AT&T 是全球电信行业历史最悠久的公司之一，其前身贝尔电话公司由电话发明者亚历山大贝尔创立。经过百年发展和多次拆分重组，AT&T 仍然是美国最大的电信运营商之一，在移动宽带和企业通信服务领域占据重要地位。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1877** | 亚历山大贝尔创立贝尔电话公司 |
| **1885** | AT&T 作为贝尔系统的长途业务子公司成立 |
| **1984** | 反垄断拆分，贝尔系统分为 7 家小贝尔 |
| **1999** | 收购 Tele-Communic Inc 进入有线电视 |
| **2005** | 被 SBC Communications 收购，沿用 AT&T 名称 |
| **2015** | 以 485 亿美元收购 DirecTV 卫星电视 |
| **2018** | 以 850 亿美元收购时代华纳，进入媒体行业 |
| **2022** | 剥离 WarnerMedia 和 DirecTV，重新聚焦电信核心 |

---

## 业务结构分析

### 移动通信

美国第二大移动运营商，超过 2 亿用户

### 宽带互联网

光纤和 DSL 宽带服务，覆盖超过 2,000 万家庭

### 企业服务

5G 专网物联网和云通信解决方案

### 第一网络

FirstNet 政府应急通信网络，独家运营

## 值得了解

- AT&T 的前身贝尔电话公司是电话发明者亚历山大贝尔创立的
- 曾是全球最大公司，1984年因反垄断被拆分为七家"小贝尔"公司

- 贝尔实验室在其旗下诞生了晶体管、激光、UNIX系统和C语言等多项诺贝尔奖级发明

- 2022年再次拆分，将媒体业务（华纳媒体）独立，回归传统电信业务


- 1984 年的反垄断拆分创造了 7 家小贝尔公司，彻底改变了美国电信格局

- 公司名称 AT&T 经历了从贝尔系统到 SBC 再回到 AT&T 的多次变更

- FirstNet 是 9/11 事件后建立的全国性应急通信网络
