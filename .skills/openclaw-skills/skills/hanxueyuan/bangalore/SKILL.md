# 班加罗尔 · Bengaluru / Bangalore

> "The Silicon Valley of India." — 全球对班加罗尔的昵称

Bangalore（现官方名为 Bengaluru）是印度卡纳塔克邦首府，被誉为"印度硅谷"。从花园城市转型为全球 IT 外包和创业中心，拥有 Infosys、Wipro 等印度 IT 巨头总部。2024 年科技产业出口额超 500 亿美元，印度独角兽企业中约 40% 总部位于此。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1537** | Kempe Gowda I 建立 Bangalore 堡垒，城市雏形形成 |
| **1800s** | 英国殖民时期成为军事驻地和"花园城市" |
| **1947** | 印度独立后，设立公共部门企业（HAL、BEL）奠定工业基础 |
| **1970s** | 印度空间研究组织（ISRO）和电子工业在此发展 |
| **1981** | Infosys 成立，开启软件外包时代 |
| **1991** | 印度经济自由化改革，外资 IT 公司开始涌入 |
| **2000s** | 成为全球 IT 外包中心，"Bangalore model"成为行业标杆 |
| **2010s** | 创业爆发期，Flipkart 等本土独角兽涌现 |
| **2022** | 官方更名 Bengaluru，但 "Bangalore" 仍广泛使用 |
| **2023** | 科技出口超 450 亿美元，初创融资超 80 亿美元 |
| **2024** | 科技出口超 500 亿美元，拥有印度 40% 的独角兽企业总部 |
| **2025** | 半导体制造产业园启动建设，印度芯片战略核心 |

## 业务结构分析

### IT 服务与外包

Infosys、Wipro、TCS 等巨头总部，提供软件开发、业务流程外包和技术支持。全球超过 200 家财富 500 强公司在 Bangalore 设有研发中心。

### 科技研发中心

Google 在 Bangalore 有最大的海外研发中心之一（超 10,000 名员工），Microsoft、Amazon、Intel、Qualcomm 等也设有大规模团队。

### 创业与风险投资

Bangalore 是印度创业生态的心脏，拥有超 8,000 家初创公司。Flipkart（被 Walmart 以 160 亿美元收购）、Ola、Razorpay 等均诞生于此。

### 生物技术与航空航天

Biocon 是印度最大生物制药公司。ISRO 和 HAL 使 Bangalore 成为印度航空航天的心脏。

## 🔑 核心护城河

```

  [人才池] ──→ [IT 公司集聚]
       ↑              │
       │              ▼
  [高校资源] ←── [基础设施]
       │              │
       ▼              ▼
  [创业生态] ──→ [国际吸引力]

```

1. 人才蓄水池：拥有 IISc 等顶尖院校，每年输出数十万工程师，低成本高技能劳动力是核心竞争力
2. 先发锁定效应：作为印度最早发展 IT 的城市，已形成完整的产业生态和人才网络
3. 英语优势：相比中国或东欧的技术中心，英语普及率使欧美企业沟通成本更低
4. 时间窗口：与欧美有 9.5-12.5 小时时差，可实现"日落开发、日出交付"的 24 小时研发循环

## 值得了解

- Infosys 创始人 Narayana Murthy 在 1981 年用 10,000 卢比（约 120 美元）的启动资金创立了公司

- Bangalore 的交通拥堵全球闻名——从"花园城市"到"交通之城"是快速城市化的缩影

- 印度的第一个购物中心和第一家披萨店（Pizza Hut）都出现在 Bangalore

- Bangalore 的天气是"印度最好的"——全年温度在 15-35°C 之间，不需要空调的日子最多

- Lalbagh 植物园有一块 30 亿年历史的岩石，是地球上最古老的岩石之一

- Bangalore 是印度精酿啤酒之都，拥有超过 100 家精酿啤酒酒吧
