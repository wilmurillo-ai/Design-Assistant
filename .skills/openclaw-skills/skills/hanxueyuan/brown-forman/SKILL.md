# 布朗霍文 · Brown-Forman Corporation

> "Every time we pour, we pour our hearts into it" — BF品牌理念

Brown-Forman Corporation成立于1870年，是美国历史最悠久的家族控股烈酒公司之一。公司总部位于肯塔基州路易斯维尔，拥有Jack Daniels（全球销量第一的美国威士忌）、Woodford Reserve、Old Forester和Korbel等标志性品牌。Brown-Forman在全球超过180个国家销售产品，年营收超过35亿美元。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1870** | George Garvin Brown在肯塔基创立公司 |
| **1870** | 推出Old Forester，首款瓶装波旁威士忌 |
| **1889** | 家族成员加入，开启家族控股传统 |
| **1940s** | 收购Jack Daniel品牌（后出售再回购） |
| **1987** | 在纽交所上市，A/B双重股权结构 |
| **1990s** | 收购Korbel香槟和芬兰伏特加 |
| **2000s** | Woodford Reserve高端波旁上市 |
| **2016** | 重新获得Jack Daniel全球独家分销权 |
| **2018** | 推出Jack Daniel单桶系列，高端化升级 |
| **2022** | 出售部分品牌，聚焦核心威士忌业务 |
| **2023** | 中国高端威士忌市场增长强劲 |
| **2024** | 年营收超过35亿美元，高端化战略持续 |

## 值得了解

- Brown-Forman是首家将威士忌装在玻璃瓶中销售的公司，此前都是散装

- Jack Daniel的糖枫木炭过滤工艺是田纳西威士忌的独家标志

- 公司总部在路易斯维尔的威士忌酒窖历史超过150年

- Brown-Forman的双重股权结构使创始家族至今仍控制着公司

- Jack Daniel的酒窖火灾后，烟熏味意外创造了新口味的威士忌

- 公司的Korbel香槟是美国独立日庆祝活动的最畅销香槟

- Brown-Forman的Woodford Reserve被誉为最接近苏格兰工艺的波旁

- 公司的威士忌陈酿仓库温度波动是Kentucky威士忌风味形成的关键
