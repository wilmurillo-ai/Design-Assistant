# 吉普 · Jeep (Stellantis)

> "Go Anywhere. Do Anything." — Jeep 品牌精神

Jeep 起源于 1941 年的二战军用越野车（Willys MB），是少数从军用转民用并获得巨大成功的汽车品牌。"Jeep"一词已成为越野车的代名词。如今作为 Stellantis 集团旗下核心品牌，Jeep 在全球 SUV 市场占据独特定位，品牌估值超 300 亿美元。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1941** | Willys-Overland 为美军生产 MB 型军用越野车，Jeep 诞生 |
| **1945** | 推出首款民用 Jeep CJ-2A，开创 SUV 品类先河 |
| **1953** | Kaiser-Willys 收购品牌，"Jeep"正式注册商标 |
| **1963** | Wagoneer 发布，首款豪华 SUV 原型 |
| **1974** | Cherokee（切诺基）问世，被视为现代 SUV 鼻祖 |
| **1987** | 克莱斯勒收购 AMC（含 Jeep 品牌），投入大规模生产 |
| **1993** | Grand Cherokee（大切诺基）发布，开辟豪华 SUV 细分市场 |
| **2009** | 菲亚特-克莱斯勒合并（FCA），Jeep 全球化扩张加速 |
| **2021** | Stellantis 成立，Jeep 成为全球第四大汽车集团的核心品牌 |
| **2023** | 推出 4xe 插电混动车型系列，加速电动化 |
| **2024** | Recon 纯电车型发布，标志品牌全面电动化开始 |
| **2025** | Wagoneer S 纯电 SUV 上市，进入豪华电动市场 |

## 业务结构分析

### 越野 SUV 系列

Wrangler（牧马人）是品牌灵魂，保留了 1941 年以来的硬派越野基因，拥有可拆卸车门和车顶的独特设计。

### 城市 SUV 系列

Cherokee、Compass 和 Renegade 面向城市和郊区用户，将 Jeep 的四驱技术与日常实用性结合。

### 豪华 SUV

Grand Cherokee 和 Wagoneer 定位高端市场，配备空气悬架、高级内饰和先进驾驶辅助。

### 电动化转型

4xe 插电混动车型已覆盖 Wrangler、Compass 和 Grand Cherokee。纯电车型 Recon 和 Wagoneer S 标志着全面电动化。

## 值得了解

- "Jeep"这个名字的来源至今有争议——有人认为是 GP 的谐音，也有人认为是漫画角色

- Jeep Wrangler 的车门和车顶可以完全拆卸，是市面上唯一合法上路的可拆卸车身汽车

- 每年复活节期间 Jeep 车主聚会用泥巴互相"攻击"，形成独特的 Jeep Easter 传统

- Jeep 在二战期间为美军生产了超过 60 万辆军用车辆

- Jeep 的七槽进气格栅设计受法律保护，只有 Jeep 车型可以使用

- Jeep 曾出现在 1967 年电影《毕业生》中，是那个时代自由精神的象征
