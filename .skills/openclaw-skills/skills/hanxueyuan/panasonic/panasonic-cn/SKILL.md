# Panasonic 松下技能

## 技能描述
Panasonic（松下）是日本电子巨头，以家电、相机、电池、汽车电子闻名。本技能帮助用户了解松下产品、服务、技术支持。

## 核心功能
- 📷 **相机**：LUMIX 系列
- 🏠 **家电**：空调、冰箱、洗衣机
- 🔋 **电池**：Eneloop、车用电池
- 💰 **价格查询**：产品定价、促销
- 🔧 **技术支持**：保修、维修
- 📊 **技术规格**：能效、功能
