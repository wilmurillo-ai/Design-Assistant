# Microsoft 微软技能

## 技能描述
Microsoft（微软）是全球领先科技公司，以 Windows、Office、Azure、Xbox 等产品闻名。本技能帮助用户了解微软产品、服务、技术支持。

## 核心功能
- 💻 **产品查询**：Windows、Office、Surface、Xbox
- ☁️ **云服务**：Azure、OneDrive、Microsoft 365
- 🎮 **游戏**：Xbox 游戏、Game Pass
- 💰 **价格查询**：产品定价、订阅费用
- 🔧 **技术支持**：使用技巧、故障排查
- 📦 **企业服务**：商业解决方案

## 使用场景
- 查询微软产品信息
- 获取使用建议
- 了解订阅服务
- 技术支持帮助

## 注意事项
- 以微软官网信息为准
- 订阅服务可能随时间变化
- 企业服务需联系销售
