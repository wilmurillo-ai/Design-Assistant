# Patagonia

> "We're in business to save our home planet." —— Patagonia 使命宣言

Patagonia是全球最具环保意识的户外服装和装备品牌，1973年由Yvon Chouinard在加州创立。从攀岩装备起家，发展为年营收超15亿美元的全球户外品牌。Patagonia以其激进的环保行动闻名——2022年创始人将整个公司（价值约30亿美元）捐给了一个致力于对抗气候变化的信托基金，成为企业社会责任的标杆。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1957年** | Yvon Chouinard开始手工锻造攀岩岩钉，创立Chouinard Equipment |
| **1973年** | 创立Patagonia品牌，最初只卖攀岩装备 |
| **1993年** | 全面采用再生聚酯纤维制造抓绒衣，行业首创 |
| **1996年** | 将100%棉T恤改为100%有机棉，承担转换成本的损失 |
| **2011年** | "Don't Buy This Jacket"黑色星期五广告——呼吁消费者减少消费 |
| **2018年** | 起诉特朗普政府缩减国家保护区，胜诉 |
| **2022年** | 创始人将整个公司所有权捐给Holdfast Collective（环保信托基金） |
| **2024年** | Worn Wear二手服饰计划扩展，品牌持续增长 |

## 业务结构分析

### 户外服装
夹克、抓绒、T恤、裤子等，覆盖登山、滑雪、冲浪、跑步、飞钓等户外运动。

### 装备
背包、帐篷、攀岩装备、冲浪板等。以耐用和可修复著称。

### Worn Wear
二手服装回收和转售计划，鼓励消费者修补而非替换，是时尚行业最成功的循环经济案例。

### 环保行动
1% for the Planet（每年捐赠营收的1%给环保组织）、供应链碳足迹追踪、再生农业。

3. **道德品牌效应**：2022年捐出公司的决定让品牌价值暴涨而非下跌
4. **Worn Wear**：二手转售计划既环保又创造新收入来源

## 值得了解

- 2011年Patagonia在黑色星期五刊登整版广告"Don't Buy This Jacket"——结果当年销售额反而增长了30%
- 创始人Yvon Chouinard自称"不情愿的企业家"——他一生中最想做的是攀岩和冲浪
- 2022年Chouinard将价值30亿美元的公司捐给环保信托基金——这是商业史上最激进的环保行为
- Patagonia的衣服标签上会告诉你这件衣服的"环境成本"——包括碳排放、水消耗和废弃物
- 该领域的发展历程反映了行业变革的重要趋势

- 其战略决策经常被商学院作为经典案例研究

- 在专业领域内保持着持续的创新能力和市场领先地位

- 近年来的战略调整展现了强大的适应能力和前瞻性

- 在全球化布局方面正在加速拓展新兴市场
