# 全国广播公司 · NBC (National Broadcasting Company)

NBC 是美国历史最悠久的主要商业广播电视网，1926 年由美国无线电公司（RCA）创立。作为美国三大电视网之一，NBC 在新闻、体育、喜剧和晚间节目等领域具有深远影响。其标志性的三声音符（NBC Chimes）和孔雀标识（Peacock）已成为美国流行文化的符号。NBC 目前是康卡斯特（Comcast）旗下 NBC 环球集团（NBCUniversal）的核心资产。

---

## 历史长河

| 时期 | 关键事件 |
|------|----------|
| **1926** | RCA 在纽约创立 NBC，首次广播由威尔·罗杰斯主持 |
| **1928** | 开始实验性电视广播，是美国最早的电视先驱 |
| **1939** | 在纽约世界博览会上首次向公众展示电视 |
| **1953** | 首次播出全彩电视节目（玫瑰花车游行） |
| **1956** | 孔雀标志首次出现，1979 年定型为今天六色版本 |
| **1960s** | 新闻部崛起，亨特利-布林克利晚间新闻成为收视冠军 |
| **1975** | 《周六夜现场》（SNL）首播，成为美国喜剧的摇篮 |
| **1980s** | "Must-See TV" 时代，《考斯比秀》《干杯酒吧》称霸收视 |
| **1994–2004** | NBC 连续多年收视率第一，《老友记》《急诊室的故事》风靡全球 |
| **2004** | NBC 与维旺迪环球合并成立 NBC 环球 |
| **2011** | 康卡斯特完成对 NBC 环球的收购 |
| **2022** | NBC 环球重组为 Comcast 旗下独立运营部门 |

## 旗舰节目矩阵

### 情景喜剧
- **《老友记》**（Friends，1994–2004）：全球最具影响力的情景喜剧之一
- **《宋飞正传》**（Seinfeld，1989–1998）："关于什么都没有的节目"，彻底改变喜剧形态
- **《办公室》**（The Office，2005–2013）：伪纪录片式喜剧标杆
- **《公园与游憩》**（Parks and Recreation，2009–2015）

### 新闻与资讯
- **NBC 晚间新闻**（NBC Nightly News）：美国收视率最高的晚间新闻节目
- **《今天》**（Today Show）：晨间新闻节目先驱，1952 年开播至今
- **MSNBC**：NBC 与微软联合创立的有线新闻频道

### 体育与直播
- **奥运会转播权**：NBC 自 1964 年起多次转播夏奥会和冬奥会
- **NFL 周日之夜橄榄球**（Sunday Night Football）：收视率最高的周播节目
- **《美国达人》**（America's Got Talent）

### 喜剧遗产
- **《周六夜现场》**（Saturday Night Live，1975–至今）：培养了艾迪·墨菲、威尔·法瑞尔、蒂娜·菲、吉米·法伦等无数喜剧明星

## 影响力与地位

NBC 在美国电视史上的地位可以用几个"第一"概括：

1. **第一个全国性电视网**：1926 年通过无线电网络连接全美
2. **彩色电视先锋**：1953 年率先推出全彩广播
3. **晨间节目始祖**：《Today》开创了电视早间节目的形态
4. **喜剧帝国**：1990 年代凭借"Must-See TV"品牌统治周四晚间档
5. **奥运霸主**：NBC 持有美国地区奥运会转播权至 2036 年
