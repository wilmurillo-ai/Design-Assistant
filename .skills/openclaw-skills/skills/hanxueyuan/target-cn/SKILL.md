# Target 塔吉特技能

## 技能描述
Target（塔吉特）是美国知名零售连锁，以时尚家居、服装、日用品闻名。本技能帮助用户查询 Target 产品、优惠、门店信息。

## 核心功能
- 🛒 **产品查询**：服装、家居、食品
- 💰 **优惠信息**：Weekly Ad、优惠券
- 🏪 **门店查询**：附近门店、营业时间
- 📱 **APP 功能**：手机购物、Circle 会员
- 🎯 **自有品牌**：Good & Gather、Cat & Jack
- 🚚 **配送服务**：送货上门、Drive Up

## 使用场景
- 查询附近 Target
- 获取优惠信息
- 了解产品类别
- 会员服务查询

## 注意事项
- 优惠可能因地区而异
- Circle 会员免费注册
- 部分商品限时供应
