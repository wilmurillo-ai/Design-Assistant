# 美国运通 · American Express

---

## 发展历程

| 时期 | 关键事件 |
|:----:|--------|
| **1850 年** | 由三家快递公司的创始人合并成立——最初做快递业务 |
| **1891 年** | 发明 **旅行支票**——让国际旅行更安全 |
| **1958 年** | 推出第一张 **签账卡**（Charge Card）——不是信用卡，每月全额还款 |
| **1984 年** | 推出百夫长卡（Centurion / 黑卡）——邀请制，无额度上限 |
| **2000s** | 从封闭网络转向开放网络——扩大商户覆盖 |
| **2020s** | 高端旅行和生活方式定位持续强化 |

---

## 核心特点

信用卡/签账卡（会员费+交易手续费）+ 商户服务费 + 旅行服务

## 核心优势

高净值用户群体（持卡人平均消费是 Visa/MC 持卡人的 3 倍）+ 品牌溢价 + 封闭网络模式
