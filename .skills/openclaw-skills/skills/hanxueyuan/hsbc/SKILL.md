# 汇丰银行 · HSBC

---

## 发展历程

| 时期 | 关键事件 |
|:----:|--------|
| **1865 年** | 在香港和上海成立——The Hongkong and Shanghai Banking Corporation |
| **1980s** | 收购米特兰银行（Midland Bank）——进入欧洲 |
| **1990s** | 业务扩展到全球 60+ 国家 |
| **2020s** | 重心重新向亚洲倾斜 |

---

## 核心特点

商业银行 + 投资银行 + 财富管理 + 环球银行与市场

## 核心优势

亚洲金融门户 + 全球贸易融资第一 + 跨时区运营能力

---

| 指标 | 数值 |
|------|------|

## 值得了解

汇丰的名字缩写 HSBC 在全球任何机场都能找到——它的广告语是'环球金融 地方智慧'
