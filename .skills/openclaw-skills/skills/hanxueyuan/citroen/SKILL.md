# 雪铁龙 (Citroën)

法国知名汽车品牌，隶属于 Stellantis 集团。

## 品牌介绍

**雪铁龙**（Citroën）是法国著名汽车制造商，由安德烈·雪铁龙于 1919 年创立。以创新技术和独特设计闻名，是欧洲历史最悠久的汽车品牌之一。

### 基本信息

| 项目 | 详情 |
|------|------|
| **成立时间** | 1919 年 |
| **创始人** | 安德烈·雪铁龙 (André Citroën) |
| **总部** | 法国巴黎 |
| **母公司** | Stellantis 集团 |
| **品牌定位** | 大众化市场，舒适与创新 |

### 品牌特色

- **舒适悬挂**：首创前轮驱动和液压气动悬挂系统
- **创新设计**：独特的外观设计和人性化内饰
- **环保技术**：积极推进电动化和混合动力技术

## 购车建议

### 适合人群

- ✅ 追求舒适驾驶体验
- ✅ 注重车辆性价比
- ✅ 喜欢法系设计风格
- ✅ 城市通勤和家庭用车

### 注意事项

- ⚠️ 保值率相对较低
- ⚠️ 维修保养网点较少（相比德系/日系）
- ⚠️ 部分车型油耗偏高
