# Bungie — FPS 品类的定义者

## 历史时间线
- **1991**: Alex Seropian 和 Jason Jones 在芝加哥创立 Bungie
- **1993**: Pathways into Darkness 发布
- **2001**: Halo: Combat Evolved 发布，重新定义主机 FPS
- **2004**: Halo 2 发布，首日营收 $125 百万
- **2007**: Bungie 从微软独立，保留 Halo IP 但失去所有权
- **2014**: Destiny 发布——首个跨平台 MMO-FPS
- **2019**: 与动视分拆，完全拥有 Destiny IP
- **2022**: 索尼以 $36 亿收购 Bungie

## 商业模式
Games as a Service (GaaS)——通过赛季通行证、DLC、微交易持续变现。Destiny 2 的年收入约 $5 亿+。

## 护城河分析
FPS 品类的定义性作品——Halo 系列塑造了主机 FPS 的标准。Destiny 的跨平台在线生态系统。索尼收购后获得 PlayStation 生态的独家内容支持。

## 关键数据
- 被索尼以 $36 亿收购
- Destiny 2 月活跃用户约 2500 万
- Halo 系列累计销量超 8000 万
- 员工约 1000 人

## 有趣事实
Halo 最初的名字是 "Monkey Nuts"，后来改名为 "Blam!"，最后才定为 Halo。Bungie 从微软独立时，Halo IP 的所有权留在了微软——这导致 Bungie 创造了 Destiny，一个完全属于他们自己的宇宙。
