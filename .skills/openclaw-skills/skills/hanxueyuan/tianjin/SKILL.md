# 🇨🇳 天津 · Tianjin 城市指南

天津是中国四大直辖市之一，拥有 600 余年建城史。近代作为中国最早开放的通商口岸之一，留下了丰富的殖民时期建筑遗产。

## 🏛️ 历史与地标

- **五大道**：2,000 多栋近代洋楼，英法意德日等多国风格
- **意式风情区**：亚洲最大的意大利风格建筑群
- **天津之眼**：世界上唯一建在桥上的摩天轮
- **古文化街**：泥人张、杨柳青年画等非遗文化聚集地

## 🍽️ 美食与餐饮

- **煎饼馃子**：天津最具代表性的早餐，绿豆面 + 馃子（油条）
- **狗不理包子**：1858 年创办的百年老字号
- **耳朵眼炸糕**：天津三绝之一，糯米油炸甜点

## 🎭 文化与体验

- **相声**：天津是相声的发源地和重要传承地
- **杨柳青年画**：中国四大木版年画之一
- **滨海新区**：于家堡金融区和航母主题公园
