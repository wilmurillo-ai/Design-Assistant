# 卡特彼勒 · Caterpillar

> "Built for It." —— Caterpillar

卡特彼勒是全球最大的工程机械和矿山设备制造商，1925年由C.L. Best和Holt Manufacturing合并创立。黄色的CAT品牌设备遍布全球每一处大型工地和矿山。Caterpillar是道琼斯工业指数成分股，也是全球经济的"晴雨表"——其设备销售数据常被用来预测全球经济走势。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1904** | Holt Manufacturing在加州发明第一台商用履带拖拉机 |
| **1925** | Holt与C.L. Best合并成立Caterpillar Tractor Company |
| **1931** | 推出第一台柴油发动机，引领行业技术变革 |
| **1950s** | 参与美国州际公路建设，二战后全球重建 |
| **1986** | 推出液压挖掘机，进入全品类工程设备 |
| **2000s** | 中国市场快速扩张，成为卡特彼勒全球第二大市场 |
| **2020** | 加速电气化和自动驾驶矿山设备研发 |
| **2024** | 营收670亿美元，建筑和资源行业需求强劲 |

---

## 业务结构分析

### 建筑设备
液压挖掘机、推土机、装载机、平地机等。全球市场份额约17%，排名第一。

### 资源设备
大型矿用卡车、电铲、钻机等。在露天采矿领域占据全球30%+份额。

### 能源与运输
柴油和天然气发动机、发电机组、铁路机车。Cat发动机是全球最广泛使用的工业发动机。

### 金融服务
Cat Financial为客户提供设备融资和租赁服务，贡献约10%的利润。

3. **品牌信任**：在矿山和大型工地，"CAT"代表最高可靠性和最低停机时间
4. **二手残值**：CAT设备的二手残值率远超竞争对手，降低客户总拥有成本

## 值得了解

- CAT的黄色是行业标准——几乎所有大型工地都能看到标志性的"CAT黄"
- 卡特彼勒的设备参与了巴拿马运河、胡佛大坝、三峡工程等全球标志性基建项目
- CAT的二手设备在非洲和南亚市场极受欢迎——一台用了10年的推土机仍能卖到原价的40-50%
- 卡特彼勒股票被华尔街视为"经济晴雨表"——其设备订单量往往领先宏观经济指标3-6个月
