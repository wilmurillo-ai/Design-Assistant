# 雪碧 · Sprite

Sprite 是全球**最受欢迎的柠檬酸橙味（lemon-lime）汽水**，也是可口可乐公司的第二大汽水品牌。透明的瓶身、清新的口感、绿色的标志——Sprite 代表了夏日、运动场上大汗淋漓后的畅饮，以及"清凉解渴"的极致定义。

## 🍋 品牌起源故事

Sprite 的诞生，其实是一场"对标战争"的产物。

**1950 年代**，七喜（7 Up）作为柠檬酸橙味汽水的领导者，在美国市场占据主导地位。可口可乐公司意识到，它缺少一款能够直接竞争 7 Up 的产品。于是，德国的可口可乐装瓶厂（Fanta 的诞生地——没错，Fanta 也是二战时德国可口可乐装瓶厂发明的）率先开发了一款柠檬酸橙味汽水。

**1961 年**，可口可乐公司在美国正式推出 **Sprite**。

Sprite 这个名字来源于德语单词 **"Sprite"**（意为"精灵"、"小仙子"），暗示着轻盈、透明、清爽的感觉。最初的产品设计刻意采用了透明液体和绿色标签，与可乐的深色液体形成鲜明对比，传达"纯净、无负担"的品牌联想。

**关键转折点：1980-90 年代的崛起**

在推出后的前 20 年里，Sprite 一直屈居 7 Up 之后。真正的逆转发生在 **1980-90 年代**：

- 1980 年代，Sprite 的销量开始逼近 7 Up
- **1991 年**，Sprite 首次在销量上超越 7 Up，成为美国最受欢迎的柠檬酸橙味汽水
- 此后，Sprite 在全球范围内持续领先，至今仍是该品类的绝对王者

**嘻哈文化与 Sprite**：

Sprite 是**第一个深度拥抱嘻哈文化**的主流汽水品牌。1990 年代，"Obey Your Thirst"（服从你的渴望）广告战役中，Sprite 大量使用嘻哈音乐人和街头文化元素，这在当时是革命性的。此后，Sprite 持续与说唱歌手、篮球明星、街舞文化绑定，成为年轻流行文化的标志性品牌。

## 🥤 招牌产品与配方

| 产品 | 特色 |
|------|------|
| **Sprite 经典原味** | 标志性的柠檬酸橙味，碳酸感强，清爽解渴 |
| **Sprite Zero Sugar** | 零糖版本，保留原味口感 |
| **Sprite Cherry** | 樱桃味 Sprite |
| **Sprite Tropical Mix** | 热带水果混合口味 |
| **Sprite Ginger** | 姜味 Sprite（特定市场） |
| **Sprite Caffeine-Free** | 不含咖啡因版本 |
| **Sprite Tropical** | 番石榴+芒果等热带风味（亚洲市场） |
| **Sprite Remix 系列** | 多种混搭口味限定款 |

**配方秘密**：Sprite 不含咖啡因（caffeine-free），这是它与可乐类汽水的核心差异之一。其口感主要来自柠檬和青柠（lime）天然香精的组合，碳酸化程度较高，带来"杀口"的清爽感。

## 🌍 全球市场版图

Sprite 是可口可乐公司的**全球第三大汽水品牌**（仅次于 Coca-Cola 和 Fanta），年销售额约 **70 亿美元**。

**市场分布**：
- **美国**：柠檬酸橙味汽水市场占有率 **50%+**，绝对领先
- **中国**：2019 年推出全新"透心凉"系列，周杰伦等明星代言，年轻化战略成功
- **日本**：推出多种限定口味，如 Sprite 葡萄、Sprite 白桃
- **欧洲**：与 Fanta 并列为可口可乐公司两大非可乐汽水品牌
- **中东 & 非洲**：快速增长市场
