# 携程 · Trip.com Group

> *"携程在手，说走就走。"*

---

## 全球旅游科技的东方引擎

Trip.com Group（携程集团）是**亚洲最大、全球前三**的在线旅行服务公司。从 1999 年上海的一间小办公室起步，到如今覆盖全球 **200+ 国家/地区**、运营着多个旅行品牌，携程已发展成为涵盖机票、酒店、火车票、旅游度假、商旅管理和金融服务的综合性旅行科技巨头。

> 🌏 携程集团旗下拥有 **Trip.com**（国际品牌）、**携程旅行**（中国）、**Skyscanner**（天巡，全球机票搜索引擎）、**Qunar**（去哪儿）四大核心品牌，形成了一个覆盖全球消费者的旅行生态矩阵。

## 📈 从"携程四君子"到全球巨头

### 1999：创始故事

1999 年，四位联合创始人——**梁建章**（James Liang）、**沈南鹏**（Neil Shen）、**季琦** 和 **范敏**——在上海创立了携程。"携程四君子" 各具专长：梁建章（技术）、沈南鹏（金融）、季琦（运营）、范敏（旅游），这个组合奠定了携程"科技+金融+运营+行业"的独特基因。

> 💡 有趣的是，沈南鹏后来创立了**红杉中国**，成为中国最成功的风险投资人之一；季琦后来创办了**如家酒店**和**华住集团**。

### 关键时间线

| 年份 | 事件 | 意义 |
|------|------|------|
| **1999** | 携程在上海成立 | 中国最早的在线旅行平台之一 |
| **2000** | 收购现代运通 | 首次行业并购，整合酒店预订资源 |
| **2003** | 纳斯达克上市（CTRP） | 中国 OTA 第一股 |
| **2009** | 推出手机 App | 从 PC 向移动端转型 |
| **2010** | 推出携程旅行网 App | 全面移动互联网化 |
| **2012** | 推出国际品牌 **Trip.com** | 全球化战略起点 |
| **2014** | 战略投资 **MakeMyTrip**（印度最大 OTA）| 进军南亚市场 |
| **2015** | 与 **去哪儿（Qunar）** 合并 | 整合中国最大两大 OTA，市场份额超 60% |
| **2016** | 收购 **Skyscanner**（天巡），耗资约 **17.4 亿美元** | 获得全球机票搜索入口 |
| **2017** | 战略投资 **MakeMyTrip** 追加投资 | 深化印度市场布局 |
| **2019** | 推出 **Trip.com** 全新品牌（国际统一） | 替代原有的 Ctrip 国际品牌 |
| **2021** | 在 **港交所双重上市**（9961.HK）| 强化亚太资本布局 |
| **2022** | 推出"超级目的地"战略 | 从交易平台向内容+交易转型 |
| **2023** | 推出 AI 旅行助手 **TripGenie** | 进入 AI 旅行规划时代 |
| **2024** | 启动全球直播营销，CEO 梁建章亲自直播 | 内容电商模式在旅行领域的应用 |

## 🌐 品牌矩阵

携程集团通过多品牌战略覆盖不同市场和消费层级：

| 品牌 | 定位 | 核心市场 | 年活跃用户 |
|------|------|----------|------------|
| **携程旅行（Ctrip）** | 中国最大综合性 OTA | 中国大陆 | 4 亿+ |
| **Trip.com** | 全球化旅行平台（英文为主） | 全球 200+ 国家 | 1 亿+ |
| **Skyscanner（天巡）** | 全球机票比价搜索引擎 | 全球 77+ 语言版本 | 1 亿+ 月活 |
| **Qunar（去哪儿）** | 中国年轻消费者/价格敏感市场 | 中国大陆 | 1.5 亿+ |
| **Trip.com for Business** | 企业商旅管理 | 全球 | 企业客户数万 |

### Trip.com（国际品牌）

**Trip.com** 是携程面向全球市场的旗舰品牌，总部位于 **新加坡**，提供多语言（40+ 语言）、多币种支持。它主打：
- 全球酒店预订（200 万+ 酒店）
- 国际机票（覆盖 800+ 航司）
- 火车票（中国高铁 + 欧洲铁路 + 日本新干线）
- 旅行攻略和社区内容
- 目的地玩乐（门票、一日游、交通）
