# 安赛乐米塔尔 · ArcelorMittal S.A.

> "Building the future together" — ArcelorMittal愿景

ArcelorMittal S.A.是全球最大的钢铁生产商，由安赛乐(Arcelor)和米塔尔钢铁(Mittal Steel)于2006年合并而成。公司总部位于卢森堡，在全球约60个国家运营，年产钢约6000万吨。ArcelorMittal的产品广泛应用于汽车、建筑、家电、包装和能源行业，是全球工业供应链的关键环节。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1976** | Lakshmi Mittal在印尼建立第一家钢铁厂 |
| **1989** | Mittal Steel在伦敦成立，开始全球扩张 |
| **1990s** | 收购特立尼达、哈萨克斯坦等地钢铁厂 |
| **2002** | 米塔尔钢铁成为全球最大钢铁公司 |
| **2006** | 以250亿欧元收购Arcelor，创建ArcelorMittal |
| **2008** | 金融危机后全球钢铁需求大幅下滑 |
| **2012** | 启动资产优化计划，出售非核心资产 |
| **2019** | 宣布2050年碳中和目标 |
| **2021** | 绿色钢铁战略加速，投资氢能炼钢 |
| **2022** | 俄乌冲突影响乌克兰和哈萨克斯坦工厂运营 |
| **2023** | 电炉钢(EAF)产能扩张，降低碳排放 |
| **2024** | 年产钢约6000万吨，加速向绿色钢铁转型 |

## 业务结构分析

### 扁平材

热轧板、冷轧板和镀锌板，用于汽车和家电制造

### 长材

钢筋、型钢和线材，用于建筑和基础设施

### 管材

石油和天然气管道用钢管

### 采矿

铁矿石和煤炭开采，保障原材料供应

## 值得了解

- ArcelorMittal的创始人Lakshmi Mittal是印度首富，身家超过200亿美元

- 公司生产的钢铁足够建造500座埃菲尔铁塔

- 公司的铁矿石产量足以满足自身约70%的需求

- ArcelorMittal是全球首家承诺2050年实现碳中和的钢铁公司

- 公司在乌克兰拥有欧洲最大的铁矿之一

- ArcelorMittal的汽车用钢占全球市场份额的约5%

- 公司的XCarb绿色钢铁品牌是全球首个低碳钢铁认证品牌
