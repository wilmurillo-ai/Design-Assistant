# 动视暴雪 · Activision Blizzard

---

## 发展历程

| 时期 | 关键事件 |
|:----:|--------|
| **1979 年** | Activision 成立——第一方第三方游戏发行商 |
| **2008 年** | Activision 与 Vivendi Games 合并成立 Activision Blizzard |
| **2000s-2010s** | 使命召唤系列年销 2000 万+ 份 |
| **2022 年** | 微软宣布 690 亿美元收购——游戏史上最大交易 |
| **2023 年** | 收购完成——微软成为全球第三大游戏公司 |

---

## 核心特点

游戏发行（使命召唤/魔兽世界/Candy Crush）+ 电竞（OWL/CDL）

## 核心优势

使命召唤是史上最成功的 FPS 系列 + Candy Crush 是移动游戏收入之王
