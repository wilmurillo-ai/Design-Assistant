# 华纳兄弟 · Warner Bros.

华纳兄弟娱乐公司（Warner Bros. Entertainment, Inc.）是全球最大的电影和电视娱乐制作公司之一，由华纳四兄弟（Harry、Albert、Sam、Jack Warner）于 1923 年在加州伯班克创立。作为好莱坞"五大"制片厂之一，华纳兄弟拥有 DC 漫画、哈利波特、指环王等顶级 IP，并通过 HBO 和流媒体 Max 构建了覆盖院线、电视和流媒体的完整内容生态。

---

## 历史轴心

| 年代 | 关键节点 |
|------|----------|
| **1918** | 华纳兄弟开始电影制作，首部独立作品《My Four Years in Germany》 |
| **1923** | 华纳兄弟影业公司正式成立 |
| **1927** | **《爵士歌手》**（The Jazz Singer）— 史上第一部有声电影，改写电影史 |
| **1930s** | 经典黑帮片和歌舞片黄金期，《疤面人》《四十二街》等 |
| **1940s** | 《卡萨布兰卡》（1942）等不朽经典 |
| **1950s** | 电视业务起步，华纳兄弟电视部门成立 |
| **1967** | 被 Seven Arts Productions 收购 → Kinney National → 后并入 Time Inc. |
| **1990** | Time-Warner 合并，成为时代华纳（Time Warner） |
| **2001** | 《哈利·波特与魔法石》上映，开启跨 decade 的电影系列 |
| **2016** | DC 扩展宇宙（DCEU）启动，《蝙蝠侠大战超人》《自杀小队》上映 |
| **2022** | 华纳媒体与 Discovery 合并成立 **Warner Bros. Discovery（WBD）** |
| **2023** | 流媒体 HBO Max 更名为 **Max** |
| **2024** | DC 宇宙重启，詹姆斯·古恩（James Gunn）执掌 DC Studios |
