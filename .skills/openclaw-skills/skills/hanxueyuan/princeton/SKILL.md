# 普林斯顿大学 · Princeton University

---

## 发展历程

| 时期 | 关键事件 |
|:----:|--------|
| **1746 年** | 在新泽西伊丽莎白镇创立 |
| **1756 年** | 迁至普林斯顿 |
| **1900s** | 转型为研究型大学 |
| **1933 年** | 爱因斯坦加入高等研究院 |
| **2020s** | 本科教育排名常春藤第一 |
