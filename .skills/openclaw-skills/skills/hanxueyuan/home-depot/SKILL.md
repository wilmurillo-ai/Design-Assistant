# 家得宝 · The Home Depot

> 你可以做到，我们帮你。—— You Can Do It. We Can Help.

The Home Depot 是全球最大的家居建材零售商，在北美（美国、加拿大、墨西哥）经营超过2300家大型仓储式门店。公司服务于两大客户群体：DIY家庭消费者和专业承包商（Pro Customers）。2023财年营收超过1500亿美元，是全球最大的家装零售企业。近年来大力投资数字化和供应链基础设施，巩固其行业领导地位。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1978年** | 伯尼·马库斯和亚瑟·布兰克在亚特兰大开设前两家Home Depot门店 |
| **1981年** | 公司上市，门店扩张至5家 |
| **1989年** | 营收突破50亿美元，成为全美最大家居建材零售商 |
| **1994年** | 进入加拿大市场，开始国际扩张 |
| **2001年** | 收购Hughes Supply，强化专业承包商业务 |
| **2007年** | 营收突破800亿美元，达到阶段性峰值 |
| **2014年** | 推出"One Home Depot"战略，整合线上线下体验 |
| **2016年** | 收购Interline Brands（后更名为HD Supply），补充MRO产品线 |
| **2020年** | 疫情期间DIY需求爆发，营收创历史新高 |
| **2022年** | 宣布10年17亿美元投资供应链和配送网络升级 |

## 业务结构分析

### DIY零售（Consumer）

面向家庭消费者的建材、工具、园艺和家装产品零售，是Home Depot的核心业务。门店平均面积约10万平方英尺，每个门店SKU约4万种。

### 专业承包商（Pro）

面向建筑商、承包商和装修专业人士的销售，贡献约45%的总营收。提供批量定价、信用额度和专属服务，是增长最快的板块。

### 在线与数字商务

Home Depot在线平台是全美最大的家居建材电商，支持线上下单门店提货（BOPIS），2023年线上销售占比约16%，约230亿美元。

### 批发与MRO供应

通过HD Supply面向设施管理和工业客户提供MRO（维护、维修和运营）产品，服务超过100万个商业客户账户。

## 🔑 核心护城河

```
  大店模式 → 一站式选购 → 高客流量
       ↓                      ↗
  专业客户网络 → 大额采购 → 规模采购优势
       ↓
  供应链投资 → 履约效率 → 成本领先
```

1. 大店规模壁垒：单店平均10万平方英尺，需要大量资本投入和选址资源，新竞争者难以快速复制这种实体网络
2. 专业客户锁定：约45%营收来自专业承包商，这类客户转换成本高（依赖批量定价、信用账期、库存深度），形成稳定收入基础
3. 供应链基础设施：数十亿美元投资配送中心和快速配送网络（Flatbed、Store Delivery），在建材物流效率上建立显著优势
4. 品牌认知度：Home Depot在北美家装零售领域拥有极高的品牌认知，"要去装修，先去Home Depot"已成为消费者心智定式

## 值得了解

- Home Depot是全球最大的圣诞树零售商之一，每年圣诞季在门店外搭建的圣诞树销售棚是美国人的节日传统

- Home Depot的橙色围裙是其品牌标志，员工被称为"橙色围裙"（Orange Apron），公司提供"橙色围裙大学"内部培训项目

- 公司创始人伯尼·马库斯和亚瑟·布兰克被解雇后才创立Home Depot——他们之前在同一家家居建材公司工作，被解雇后决定自己创业

- Home Depot平均每个门店储存约3万-4万个SKU，但线上目录超过100万种产品，线下只是冰山一角
