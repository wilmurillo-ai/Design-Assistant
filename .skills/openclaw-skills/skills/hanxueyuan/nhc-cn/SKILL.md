# 中国卫健委技能

## 技能描述
国家卫生健康委员会主管全国卫生健康工作。本技能提供医疗政策、卫生健康、疾病防控等信息。

## 核心功能
- 🏥 **医疗政策**：医改政策、医疗法规
- 💊 **药品管理**：药品目录、用药指南
- 🦠 **疾病防控**：传染病、慢性病防控
- 📊 **健康数据**：统计公报、健康指标
- 👶 **妇幼健康**：妇幼保健、生育政策
- 🚨 **公共卫生**：应急响应、健康提醒

## 使用场景
- 查询医疗政策
- 了解疾病防控信息
- 获取健康指导
- 查询药品信息

## 注意事项
- 医疗建议请咨询专业医生
- 疫情信息以官方发布为准
- 紧急情况请拨打急救电话
