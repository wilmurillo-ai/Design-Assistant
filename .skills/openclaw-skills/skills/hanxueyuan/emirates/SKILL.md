# Emirates (阿联酋航空)

总部位于迪拜的国际航空公司，以豪华服务著称。

## 公司介绍

**阿联酋航空**成立于 1985 年，是中东地区最大的航空公司。

### 基本信息

| 项目 | 详情 |
|------|------|
| **成立时间** | 1985 年 |
| **总部** | 迪拜，阿联酋 |
| **枢纽机场** | 迪拜国际机场 (DXB) |
| **机队规模** | 250+ 架 |
| **目的地** | 150+ 个城市 |

### 舱位等级

| 舱位 | 特点 |
|------|------|
| **头等舱** | 私人套房、空中淋浴 |
| **商务舱** | 平躺座椅、专属休息区 |
| **经济舱** | 宽敞座椅、娱乐系统 |

### 服务特色

- **ice 娱乐系统** - 数千部电影和节目
- **机上美食** - 多国料理
- **空中酒廊** - 头等舱和商务舱专属
- **接送服务** - 部分城市提供免费接送
