# 沃尔沃 · Volvo

---

## 发展历程

| 时期 | 关键事件 |
|:----:|--------|
| **1927 年** | Assar Gabrielsson 和 Gustaf Larson 在瑞典哥德堡创立 |
| **1959 年** | 发明三点式安全带——无偿开放专利 |
| **2010 年** | 被中国吉利集团收购 |
| **2020s** | 全面电动化——2030 年只销售电动车 |
