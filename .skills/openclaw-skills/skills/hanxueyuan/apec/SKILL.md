## 📊 组织概况

### 基本信息

| 项目 | 详情 |
|------|------|
| **全称** | Asia-Pacific Economic Cooperation |
| **中文** | 亚太经济合作组织 |
| **成立时间** | 1989 年 11 月 |
| **秘书处** | 新加坡 |
| **秘书长** | 任期 3 年 |
| **成员 GDP 总量** | ~54 万亿美元 |
| **占全球 GDP** | ~60% |
| **占全球贸易** | ~50% |
| **覆盖人口** | ~29 亿 |

### 宗旨目标

**核心目标**：
1. 促进亚太地区自由贸易和投资
2. 推动区域经济一体化
3. 缩小成员发展差距
4. 促进可持续经济增长

**茂物目标（1994 年）**：
- 发达成员：2010 年实现贸易自由化
- 发展中成员：2020 年实现贸易自由化

## 📅 峰会机制

### 领导人非正式会议

**举办方式**：
- 每年一次
- 成员经济体轮流主办
- 领导人穿东道主传统服装合影

**历届峰会**：

| 年份 | 主办地 | 主题/成果 |
|------|-------|----------|
| 2023 | 美国旧金山 | 复苏、韧性、可持续 |
| 2022 | 泰国曼谷 | 开放、联通、平衡 |
| 2021 | 新西兰（线上） | 疫情后复苏 |
| 2020 | 马来西亚（线上） | 疫情应对 |
| 2019 | 智利（取消） | - |
| 2018 | 巴新莫尔兹比港 | 把握包容性机遇 |
| 2017 | 越南岘港 | 打造新动力 |
| 2016 | 秘鲁利马 | 高质量增长 |
| 2015 | 菲律宾马尼拉 | 包容性增长 |
| 2014 | 中国北京 | 共建面向未来的亚太伙伴关系 |
| 2013 | 印尼巴厘岛 | 活力亚太，全球引擎 |
| 2012 | 俄罗斯符拉迪沃斯托克 | 融合谋发展，创新促繁荣 |

### 部长级会议

- **时间**：每年一次（通常在领导人会议前）
- **参会者**：外交部长、经贸部长
- **职责**：为领导人会议做准备

### 高官会议（SOM）

- **频率**：每年 3-4 次
- **参会者**：各成员 APEC 高官（副部级）
- **职责**：协调政策、监督项目

## 🤝 合作领域

### 贸易投资自由化

**重点议题**：
- 降低关税和非关税壁垒
- 简化海关程序
- 促进服务贸易
- 投资便利化

**成果**：
- 区域内平均关税从 1989 年的 17% 降至 2023 年的 5%
- 贸易额增长超 10 倍

### 区域经济一体化

**倡议**：
- 亚太自贸区（FTAAP）：长期目标
- 跨太平洋伙伴关系（CPTPP）：部分成员参与
- 区域全面经济伙伴关系（RCEP）：15 个 APEC 成员参与

### 互联互通

**重点领域**：
- 基础设施：港口、公路、机场
- 数字连接：宽带、5G、数字贸易
- 人员流动：商务旅行卡、签证便利化

**APEC 商务旅行卡**：
- 覆盖 21 个成员
- 3-5 年有效期
- 免签证入境、快速通关

### 可持续发展

**议题**：
- 清洁能源
- 绿色增长
- 海洋保护
- 应对气候变化

**目标**：
- 2030 年区域可再生能源占比翻倍
- 降低能源强度
- 促进绿色技术创新

## ⚠️ 注意事项

1. **经济体 vs 国家**：APEC 成员称为"经济体"，包含港澳台
2. **非约束性**：APEC 决议无法律约束力，靠自愿执行
3. **共识决策**：所有决定需成员一致同意
4. **开放区域主义**：成果可惠及非成员
5. **信息时效**：数据可能更新，以官网为准
