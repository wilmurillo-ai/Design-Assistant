# 康卡斯特 · Comcast

> "美国最大的有线电视和互联网服务提供商，NBC环球的母公司"

Comcast 是美国最大的有线电视互联网和媒体集团，总部位于费城。通过 NBCUniversal 的收购和整合，Comcast 不仅是基础设施提供商，更是全球领先的影视娱乐内容生产商，拥有环球影业NBC和多个流媒体平台。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1963** | Ralph Roberts 在密西西比州创立 American Cable Systems |
| **1969** | 公司改名 Comcast，开始全国扩张 |
| **1988** | 收购美国最大的有线电视系统之一 |
| **2002** | 与 AT&T 宽带合并，成为美国最大有线电视运营商 |
| **2011** | 以 137 亿美元收购 NBC 控股权 |
| **2013** | 以 230 亿美元完全收购 NBCUniversal |
| **2018** | 以 390 亿美元收购英国天空广播 Sky |
| **2024** | Peacock 流媒体订阅用户突破 3,000 万 |

---

## 业务结构分析

### 有线电视

Xfinity 品牌有线电视服务，覆盖约 3,000 万家庭

### 宽带互联网

美国最大的有线宽带运营商，用户超过 3,200 万

### 媒体娱乐

NBCUniversal 环球影业和多个有线电视频道

### 流媒体

Peacock 流媒体平台和主题乐园业务

## 值得了解

- Comcast 是全球最大的有线电视运营商，也是最大的宽带互联网服务提供商之一
- 从单一有线电视系统起步，通过收购AT&T Broadcom成为全美最大有线电视运营商

- 旗下NBC环球拥有环球影城主题公园，年接待游客超过5000万人次

- 其Peacock流媒体平台凭借独家体育内容（奥运会、英超）快速积累用户


- NBCUniversal 拥有环球影城主题公园，在全球拥有 6 个园区

- Sky 收购案是欧洲媒体史上最大的并购交易之一

- 公司创始人 Ralph Roberts 在 2015 年去世前将公司从零做到千亿营收
