# Palantir Technologies

> "We build the software that powers the intelligence of nations." —— Alex Karp

Palantir是全球领先的大数据分析和AI决策平台公司，2003年由Peter Thiel、Alex Karp等人在硅谷创立。公司以《指环王》中"真知晶球"（Palantír）命名，意为"洞察一切的视觉"。核心产品Gotham服务于政府情报和国防，Foundry服务于企业数据决策，AIP（Artificial Intelligence Platform）则是最新AI原生平台。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **2003** | Peter Thiel、Alex Karp、Joe Lonsdale和Stephen Cohen在硅谷创立Palantir |
| **2004** | 获得CIA旗下In-Q-Tel投资，开始为情报机构开发反恐数据分析工具 |
| **2008** | 与美军合作，在伊拉克和阿富汗战场部署Gotham系统，用于IED预测和情报分析 |
| **2015** | 推出Foundry平台，从政府业务扩展到商业市场 |
| **2020** | 纽交所直接上市（DPO），首日市值约270亿美元 |
| **2022** | 俄乌冲突中，Palantir系统被用于军事目标识别和情报分析 |
| **2023** | 推出AIP（Artificial Intelligence Platform），将大模型引入企业决策 |
| **2024** | AIP推动营收加速增长，首次加入S&P 500指数 |

## 业务结构分析

### Gotham（政府与国防）
服务于CIA、FBI、国防部等情报和军事机构。提供多源数据融合、目标识别、任务规划等功能，是美军反恐行动的核心软件基础设施。

### Foundry（企业数据平台）
帮助企业整合分散的数据源，构建"操作系统级"的数据平台。客户包括空中客车、摩根士丹利、BP等全球500强企业。

### AIP（AI决策平台）
2023年推出的AI原生平台，将大语言模型与企业数据结合，实现自然语言数据查询、自动化决策、场景模拟等功能。

### 商业模式
采用"前置部署+持续运营"模式，平均合同金额超500万美元，客户续约率超95%。

## 值得了解

- 公司名Palantír来自《指环王》中的"真知晶球"——一种能看到远方的魔法水晶球，完美契合公司使命
- CEO Alex Karp是斯坦福哲学博士，以"硅谷最奇怪的CEO"著称——他住在新罕布什尔州的农场，不开车，每天跑步上班
- 创始人Peter Thiel是PayPal联合创始人、Facebook首个外部投资者，也是特朗普的支持者，政治立场鲜明
- 电影《钢铁侠》中托尼·斯塔克使用的AI系统"Jarvis"，其原型据说就参考了Palantir的数据分析理念
- 该领域的发展历程反映了行业变革的重要趋势

- 其战略决策经常被商学院作为经典案例研究

- 在专业领域内保持着持续的创新能力和市场领先地位

- 近年来的战略调整展现了强大的适应能力和前瞻性
