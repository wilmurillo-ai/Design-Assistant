# 诺基亚 · Nokia

> "Connecting People" — 一句广告语定义了手机行业的黄金年代

诺基亚起源于1865年芬兰的一家造纸厂，经过150多年转型，成为全球移动通信的标志性企业。其手机业务在2007年iPhone发布前占据全球40%以上的市场份额。如今诺基亚是仅次于华为的全球第二大电信网络设备供应商，同时在专利授权领域保持强大话语权。

## 业务结构分析

### 网络基础设施

5G基站、光网络和IP路由设备，服务全球130+国家的运营商客户

### 诺基亚贝尔实验室

诺贝尔奖级别的研发机构，拥有150多项诺贝尔奖相关成就

### 专利授权

拥有20,000+项专利族，通过专利许可获得稳定高利润收入

### 云与网络服务

为运营商提供云原生网络解决方案和数字化转型服务

## 值得了解

- 诺基亚3310被网友戏称为"能砸核桃的砖头"，其耐用性在互联网上成为传奇

- 诺基亚最初是造纸厂和橡胶厂，后来还生产过避孕套（通过子公司）

- "Connecting People"广告语中的握手图标是全球最知名的品牌标识之一

- 诺基亚曾研发过平板电脑和智能手表的概念原型，但从未量产

- 2007年诺基亚内部备忘录警告iPhone威胁，但高管层认为"iPhone没有物理键盘无法成功"

- 诺基亚铃声"Gran Vals"源自1892年西班牙吉他曲，每天被播放约18亿次

- 诺基亚在1990年代中期短暂生产过电视和显示器

- 芬兰总统府使用的安全通信系统由诺基亚提供

- 诺基亚的贝尔实验室还发明了太阳能电池、光纤通信和CDMA技术的基础理论

- 诺基亚手机业务被出售后，原诺基亚大厦被改造为创业孵化中心
