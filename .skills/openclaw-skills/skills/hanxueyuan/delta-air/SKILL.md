# 达美航空 · Delta Air Lines Inc.

> "Keep climbing" — Delta企业口号

Delta Air Lines是全球营收规模最大的航空公司之一，总部位于亚特兰大。作为美国三大航空公司之一（与美联航、美国航空并列），Delta以其卓越的正点率、客户满意度和强大的常旅客计划著称。Delta运营着超过900架飞机，每天执行5400+次航班，覆盖全球325个以上的目的地。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1928** | Collett E. Woolman在路易斯安那创立Huff Daland Dusters |
| **1929** | 更名为Delta Air Service，开始客运业务 |
| **1953** | 收购Chicago and Southern，成为全国性航空公司 |
| **1972** | 引入L-1011三星客机，提升长途航线 |
| **1987** | 收购Western Airlines，大幅扩展西海岸航线 |
| **2000** | 加入天合联盟(SkyTeam)，构建全球航空网络 |
| **2008** | 收购西北航空(Northwest)，创建全球最大航司 |
| **2013** | 投资维珍大西洋航空，强化跨大西洋航线 |
| **2016** | 推出Basic Economy票价，应对廉航竞争 |
| **2020** | 疫情冲击下裁员但保持机队优化 |
| **2022** | 国际航线恢复，营收突破疫情前水平 |
| **2024** | 年营收超过580亿美元，SkyMiles会员超1亿 |

---

## 业务结构分析

### 客运航空

国内和国际航班运营，覆盖全球325+目的地

### 货运业务

Delta Cargo提供全球航空货运和物流解决方案

### 常旅客计划

SkyMiles会员计划是美国最有价值的航空常旅客项目

### MRO维修

Delta TechOps是北美最大的航空公司维修部门

## 值得了解

- Delta的名字来源于密西西比河三角洲地区，公司最初在路易斯安那州创立

- 亚特兰大机场的Delta航站楼是全球最大的单一航站楼之一

- Delta是首家在所有飞机上提供免费WiFi的美国主要航空公司

- 公司的飞机维修部门Delta TechOps甚至为竞争对手提供服务

- Delta曾拥有自己的炼油厂，以对冲航油价格波动风险

- SkyMiles积分在美国被用作准货币，可以在无数合作商户消费

- Delta是世界上少数几家仍然运营波音757的主要航空公司之一

- 疫情期间Delta是首家要求所有乘客和员工佩戴口罩的航空公司
