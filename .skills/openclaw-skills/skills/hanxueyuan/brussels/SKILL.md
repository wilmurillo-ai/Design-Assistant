# brussels

## 城市概况
布鲁塞尔是比利时首都和最大城市，也是欧盟和北约总部所在地，被誉为'欧洲首都'。以撒尿小童雕像、布鲁塞尔松露巧克力和华夫饼闻名于世。

官方网站：https://www.visitbrussels.be

## 核心特色
- 🏛️ **政治中心**：欧盟委员会、欧盟理事会、北约总部\n- 🍫 **美食**：松露巧克力、华夫饼、啤酒\n- 🗿 **地标**：撒尿小童、原子塔、大广场\n- 🎨 **文化**：漫画之都、丁丁和蓝精灵故乡\n- 📊 **经济**：欧洲政治和外交中心
