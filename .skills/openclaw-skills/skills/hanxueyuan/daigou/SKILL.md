# 🛍️ 代购 DaiGou — 跨境购物助手

**代购** 是指帮助消费者从海外市场购买商品并邮寄到国内的服务，主要优势是价格差异、正品保障和稀缺商品获取。

## 核心功能

### 1. 商品搜索
- 海外电商平台比价 (亚马逊、乐天、免税店)
- 专柜价格查询
- 限量款/地区限定款寻找

### 2. 价格计算
- 商品原价
- 国际运费
- 关税估算
- 代购服务费
- **到手价 = 原价 + 运费 + 关税 + 服务费**

### 3. 正品验证
- 购买凭证 (小票、订单截图)
- 物流追踪 (海外直邮)
- 品牌专柜验货

### 4. 物流追踪
- 国际快递查询
- 清关状态
- 预计送达时间

## 费用构成

| 项目 | 费率 |
|------|------|
| **商品原价** | 100% |
| **国际运费** | 50-200 元/kg |
| **关税** | 0-30% (按品类) |
| **代购费** | 5-15% |
| **国内快递** | 10-20 元 |

## 风险提示

⚠️ **购买风险**
- 假货风险 (选择可靠代购)
- 物流损坏
- 海关扣税

⚠️ **法律风险**
- 个人自用合理数量
- 避免走私嫌疑
- 如实申报

## 选择代购建议

✅ **可靠代购特征**
- 提供完整购买凭证
- 支持海外直邮
- 有实体店/公司
- 好评率高
- 支持验货

❌ **警惕**
- 价格过低 (低于市场价 50%+)
- 无法提供凭证
- 只发国内快递
- 无售后保障
