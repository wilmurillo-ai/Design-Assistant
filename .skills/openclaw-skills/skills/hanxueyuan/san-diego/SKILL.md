# 圣地亚哥 · San Diego

> "America's Finest City." — 城市昵称

San Diego 位于加州南部，与墨西哥接壤，是美国第八大城市（人口超 140 万，都会区超 330 万）。以"全年 70°F（21°C）"的完美气候闻名，是美国海军太平洋舰队的母港，也是全球领先的生物科技产业集群。2024 年 GDP 约 2,800 亿美元，旅游年收入超 120 亿美元。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1542** | 葡萄牙探险家 Juan Rodríguez Cabrillo 首次抵达圣地亚哥湾 |
| **1769** | 西班牙方济各会士建立 Presidio 和 Mission，加州第一个欧洲定居点 |
| **1850** | 加州加入美国，圣地亚哥成为建制市 |
| **1880s** | 铁路到达，城市开始快速发展 |
| **1915** | 巴拿马-加州博览会举办，Balboa Park 建筑群落成 |
| **1920s-40s** | 美国海军基地建立，军事成为城市经济支柱 |
| **1950s** | 二战后海军扩张，圣地亚哥成为太平洋舰队最大母港 |
| **1990s** | 生物科技产业兴起，UCSD 和 Salk 研究所是核心驱动力 |
| **2000s** | 旅游和会议经济爆发，San Diego Convention Center 扩建 |
| **2024** | 生物科技集群全美第二，仅次于波士顿，GDP 约 2,800 亿美元 |

---

## 业务结构分析

### 生物科技与生命科学

圣地亚哥是全球第二大生物科技集群（仅次于波士顿），拥有 Illumina、Neurocrine 等上市公司和 700+ 家生物科技公司。UCSD 和 Salk 研究所是人才和研发核心。

### 军事与国防

美国海军太平洋舰队最大母港，拥有 2 艘航母母港和约 90,000 名军事人员。General Atomics（无人机制造商）和 Northrop Grumman 在此运营。

### 旅游与会展

Balboa Park、圣地亚哥动物园、海滩和墨西哥边境文化是核心旅游资源。年接待游客超 3,500 万人次，旅游收入超 120 亿美元。

### 高科技与创业

除了生物科技，圣地亚哥在通信（Qualcomm 总部）、半导体和海洋科技方面也有优势。创业生态在加州仅次于硅谷。

## 值得了解

- 圣地亚哥动物园是全球第一个成功的熊猫繁育项目的举办地，"白云"和"高高"在这里生活了 20+ 年

- 圣地亚哥是美国唯一一个拥有两艘航母母港的城市——USS Carl Vinson 和 USS Ronald Reagan

- Balboa Park 的面积（1,200 英亩）比纽约中央公园还大

- 圣地亚哥是"墨西哥卷饼"（California Burrito）的发源地

- Qualcomm 在圣地亚哥发明了 CDMA 技术，这是现代手机通信的基础

- 圣地亚哥的天际线有一个独特规定——所有建筑不能超过机场跑道的高度限制
