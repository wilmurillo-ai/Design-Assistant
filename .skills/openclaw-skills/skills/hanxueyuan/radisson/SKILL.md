# Radisson 丽笙

全球知名酒店集团，提供高品质住宿体验。

## 品牌简介

- **成立时间**: 1909 年
- **总部**: 比利时布鲁塞尔
- **行业**: 酒店、度假村
- **酒店数量**: 1,400+ 家
- **覆盖**: 120+ 国家
- **官网**: https://www.radissonhotels.com
