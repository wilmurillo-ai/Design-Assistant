# 霍尼韦尔 · Honeywell

> "We make what matters work." —— Honeywell

霍尼韦尔是全球领先的工业技术和制造集团，1906年在美国创立。公司业务涵盖航空航天、建筑科技、材料技术和安全生产力四大板块，产品从飞机发动机到楼宇自控、从特种化学品到仓库自动化，是工业4.0时代的关键赋能者。霍尼韦尔是道琼斯工业指数成分股，服务全球几乎所有行业和领域。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1906** | Mark C. Honeywell创立Honeywell Heating Specialty Company |
| **1927** | 发明第一台商用恒温器，奠定楼宇自控业务基础 |
| **1950s** | 进入航空航天领域，为军用和商用飞机提供控制系统 |
| **1999** | 与AlliedSignal合并，沿用Honeywell名称 |
| **2015** | 收购UOP（霍尼韦尔Process Solutions），强化材料和工艺 |
| **2018** | 分拆AdvanSix和Resideo，聚焦四大核心业务 |
| **2021** | 推出Forge工业物联网平台，加速数字化转型 |
| **2024** | 分拆Advanced Materials业务，聚焦航空航天和自动化 |

## 业务结构分析

### 航空航天
飞机发动机、辅助动力装置（APU）、航电系统、机舱系统。几乎所有商用飞机都使用霍尼韦尔产品。

### 建筑科技
楼宇自控、消防安防、暖通控制。全球超10亿栋建筑使用霍尼韦尔的控制系统。

### 材料技术
特种化学品、制冷剂、UOP炼化工艺。全球90%以上的汽油生产使用UOP工艺。

### 安全与生产力
仓库自动化（Intelligrated）、条码扫描、PPE个人防护设备、工业软件。

```
┌─────────────────────────────────────────┐
│        工业基础设施嵌入                   │
│                                         │
|   飞机/建筑/工厂 ─→ 长期使用 ─→ 替换成本  │
|       ↑                        ↓        │
|   持续服务 ←── 升级改造 ←── 客户粘性     │
|                                         │
|   + 飞机APU：全球40%+份额               │
|   + 炼油工艺：90%+汽油用UOP技术          │
|   + 楼宇自控：全球超10亿栋建筑使用       │
└─────────────────────────────────────────┘
```

1. **嵌入式基础设施**：飞机、建筑、工厂一旦使用霍尼韦尔设备，替换成本极高
2. **航空航天认证**：飞机部件需要数年认证周期，竞争者难以进入
3. **炼油工艺垄断**：全球90%+汽油生产使用UOP工艺，几乎不可替代
4. **工业物联网**：Forge平台连接数百万传感器，数据积累形成AI优化优势

## 值得了解

- 霍尼韦尔的产品几乎无处不在——你坐的飞机、住的酒店、工作的办公楼、加油的加油站都用到了它
- UOP炼油工艺是全球最不为人知的"基础设施"——你加的每一滴汽油，几乎都经过UOP技术处理
- 霍尼韦尔曾发明第一个商用恒温器，至今仍是楼宇自控领域的绝对领导者
- 公司历史上最著名的事件是1999年与AlliedSignal的"蛇吞象"合并——规模较小的Honeywell反而保留了名字
