# 英超技能

## 技能描述
Premier League（英超）是英格兰最高级别足球联赛，全球最受欢迎的足球联赛之一。本技能帮助用户了解英超球队、球员、赛程、比分等信息。

## 核心功能
- ⚽ **球队查询**：20 支球队信息
- 📅 **赛程比分**：比赛时间、实时比分
- 🏆 **积分榜**：排名、积分、净胜球
- 👤 **球员信息**：射手榜、助攻榜
- 📊 **数据统计**：进球、助攻、零封
- 📰 **新闻动态**：转会、伤病、资讯

## 使用场景
- 查询比赛信息
- 了解球队球员
- 获取比分数据
- 查看新闻动态

## 注意事项
- 赛程可能调整
- 数据以官方为准
- 时区为英国时间
