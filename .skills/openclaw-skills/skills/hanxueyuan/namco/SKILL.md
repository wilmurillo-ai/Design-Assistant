# 南梦宫 · Namco

南梦宫（株式会社ナムコ）是日本游戏产业历史上最具标志性的公司之一，以《吃豆人》（Pac-Man）这一全球最知名的游戏角色和开创性的街机游戏而闻名。公司成立于 1955 年，最初经营游乐设施，后转型为电子游戏巨头。2005 年，南梦宫与万代（Bandai）合并为万代南梦宫控股（Bandai Namco Holdings），成为日本最大的娱乐集团之一。

---

## 企业进化史

### 🎡 游乐设施时代（1955–1974）
| 年份 | 事件 |
|------|------|
| **1955** | 中村雅哉在横滨创立"中村制作所"（Nakamura Manufacturing Co.） |
| **1950s–60s** | 为百货公司屋顶游乐场制造骑乘游乐设备 |
| **1970** | 公司名称简化为 "Namco"（Nakamura Amusement Machine Manufacturing Company） |
| **1974** | 收购 Atari 在日本的分公司，正式进入游戏行业 |

### 🕹️ 街机黄金时代（1978–1990s）
| 年份 | 事件 |
|------|------|
| **1978** | 发布首款街机游戏《Gee Bee》 |
| **1979** | **《Galaxian》**（小蜜蜂）— 彩色位图图形先驱 |
| **1980** | **《Pac-Man》**（吃豆人）— 全球最知名的街机游戏 |
| **1981** | 《Ms. Pac-Man》《Galaga》相继推出 |
| **1983** | 南梦宫在东京证券交易所上市 |
| **1980s–90s** | 《铁拳》《山脊赛车》《王牌空战》等系列崛起 |

### 🎮 3D 游戏与合并（1994–2005）
| 年份 | 事件 |
|------|------|
| **1994** | 《山脊赛车》（Ridge Racer）推动 PlayStation 主机成功 |
| **1995** | **《铁拳》**（Tekken）3D 格斗游戏标杆 |
| **2000** | 《皇牌空战》系列确立 |
| **2005** | **万代与南梦宫合并**，成立万代南梦宫控股 |
| **2006** | 游戏部门整合为 Bandai Namco Games（现 Bandai Namco Entertainment） |

### 🏢 合并后（2005–至今）
- 《黑暗之魂》系列（FromSoftware 合作）
- 《艾尔登法环》（2022）— 全球销量超 2500 万，年度游戏
- 《铁拳 8》（2024）
- 持续运营街机业务和主题乐园

## 文化影响

**《吃豆人》的全球现象：**
- 全球街机收入超过 **100 亿美元**（经通胀调整）
- 首个拥有广泛主流认知度的电子游戏角色
- 1982 年 CBS 播出卡通片《Pac-Man》
- "吃豆人" 形象出现在 T 恤、麦片、玩具等无数商品上
- 影响了从流行音乐到街头艺术的广泛领域

**街机文化推动者：**
- 在日本推动街机厅（ゲームセンター）文化
- 创新性的游戏设计影响了整个行业
- Namco System 街机基板成为行业标准之一
