# 蒂姆霍顿斯 · Tim Hortons

在加拿大，Tim Hortons 不仅仅是一个咖啡品牌——它是一种**国家象征**。"Let's grab a Tim's"（我们去 Tim's 坐坐吧）是加拿大人最常说的话之一。这个以已故冰球运动员命名的咖啡连锁，拥有加拿大最大的快餐市场份额，门店密度超过了麦当劳、星巴克和 Dunkin' 的总和。

## 🏒 品牌起源故事

Tim Hortons 的故事，始于冰球场和甜甜圈。

**Tim Horton** 是加拿大冰球联盟（NHL）的明星球员，1949-1974 年间效力于多伦多枫叶队等球队，四次获得斯坦利杯冠军。他不仅是球场上的硬汉，还是个精明的商人。

**1964 年**，Tim Horton 与商业伙伴 **Ron Joyce**（一位汉密尔顿的警察）在安大略省汉密尔顿市开设了**第一家 Tim Horton Donut Shop**。

有趣的是，Tim 一开始并不打算离开冰球。他白天打比赛，晚上经营甜甜圈店。但随着生意的快速增长，他意识到甜甜圈的潜力不亚于冰球。

**1974 年**，悲剧降临——Tim Horton 在一场暴风雪中的车祸中不幸去世，年仅 44 岁。很多人以为 Tim Hortons 会随之消失。但 Ron Joyce 接过了重担，将品牌从一家地方小店发展为全国性连锁。

**1995 年**，Tim Hortons 与 **Wendy's** 合并（Joyce 用 Tim Hortons 的股权换取了 Wendy's 的大量股份），这一举动为 Tim Hortons 带来了国际化管理经验和资本支持。

**2006 年**，Tim Hortons 重新独立上市。2014 年，**Burger King 的母公司 3G Capital** 以 114 亿美元收购了 Tim Hortons，合并成立了 **Restaurant Brands International (RBI)**——这个交易让 Tim Hortons 与 Burger King、Popeyes、Firehouse Subs 成为同一集团下的兄弟品牌。

**"Double-Double"文化**：

在加拿大，"I'll have a double-double, please"（请来一杯 double-double）就像"给我来杯咖啡"一样自然。据估计，Tim Hortons 每天售出 **300 万+ 杯**咖啡——对于人口仅 4000 万的加拿大来说，这意味着几乎每 13 个人中就有一个人每天在 Tim Hortons 喝咖啡。

**Timbit 的由来**：

"Timbit"这个名字由一个小女孩在一次比赛中命名。Tim Hortons 为甜甜圈洞（donut holes）征集名字，最终 "Timbits" 从数百个提案中脱颖而出。如今 Timbits 已经成为加拿大的文化符号——甚至被用作"小事、小东西"的代名词（"He's no Timbit" ≈ "他不是小角色"）。

## 🌍 全球版图

Tim Hortons 在全球 **15+ 个国家**拥有超过 **5,700 家**门店。

**市场分布**：
- **加拿大**：大本营，4,000+ 家门店，覆盖从城市到偏远小镇的每个角落
- **美国**：约 700 家门店，主要集中在东海岸（纽约州、密歇根州、俄亥俄州等），正在加速扩张
- **中国**：2018 年进入，2019 年在上海开出首店，目前数百家门店
- **英国、墨西哥、西班牙、菲律宾、印度**：新兴国际扩张
- **沙特阿拉伯 & 中东**：通过 RBI 的特许经营网络

**中国市场策略**：
Tim Hortons 在中国采取了独特的本土化策略——将咖啡 + 甜甜圈升级为"咖啡 + 暖食"模式，增加了更多热食选择。同时以低于星巴克的定价和更快的出餐速度，瞄准"大众咖啡"市场。
