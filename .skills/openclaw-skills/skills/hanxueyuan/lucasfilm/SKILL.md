# 卢卡斯影业 · Lucasfilm

> *"A long time ago in a galaxy far, far away..."*

卢卡斯影业由乔治·卢卡斯于 1971 年创立，总部位于美国加利福尼亚州旧金山。这家公司凭借《星球大战》系列彻底改变了电影工业，在特效技术、音效设计和叙事宇宙构建等领域树立了全新标杆。2012 年，迪士尼以 40.5 亿美元收购卢卡斯影业，将其纳入庞大的娱乐帝国。

---

## 发展里程碑

| 年份 | 事件 |
|------|------|
| 1971 | 乔治·卢卡斯在加州马林县创立卢卡斯影业（Lucasfilm Ltd.） |
| 1977 | 《星球大战》（后更名为《星球大战IV：新希望》）上映，全球票房 7.75 亿美元 |
| 1978 | 成立工业光魔（ILM）特效部门，后独立为行业顶尖视觉特效公司 |
| 1980–1984 | 《帝国反击战》《绝地归来》《夺宝奇兵》系列相继成功 |
| 1999–2005 | 《星球大战》前传三部曲上映，推动数字电影技术发展 |
| 2012 | 迪士尼以 40.5 亿美元现金加股票完成收购 |
| 2015 | 《星球大战：原力觉醒》上映，全球票房 20.68 亿美元 |
| 2019 | 《星球大战：天行者崛起》为"天行者传奇"画上句号 |
| 2022 | 《曼达洛人》《安多》《欧比旺·克诺比》等剧集在 Disney+ 热播 |

## 代表作品

### 电影
- **《星球大战》正传三部曲**（1977–1983）：开创太空歌剧电影新纪元
- **《夺宝奇兵》系列**（1981–2023）：印第安纳·琼斯冒险故事，斯皮尔伯格执导
- **《星球大战》前传三部曲**（1999–2005）：探索银河共和国的衰落
- **《星球大战》后传三部曲**（2015–2019）：迪士尼时代续作

### 剧集（Disney+）
- **《曼达洛人》**（2019–至今）：采用虚拟制片技术的开创性剧集
- **《安多》**（2022–2025）：以谍战风格重构星战叙事
- **《欧比旺·克诺比》**（2022）：伊万·麦格雷戈回归主演

## 技术贡献

卢卡斯影业远不止是一家电影制作公司，它是多项电影技术革命的策源地：

- **工业光魔（ILM）**：全球最顶尖的视觉特效公司之一，参与过《侏罗纪公园》《阿凡达》等数百部影片的特效制作
- **THX 认证**：由卢卡斯影业制定的影院音质标准，至今仍是高品质音响的标志
- **剪辑软件**：早期开发的 EditDroid 为非线性数字剪辑奠定了基础
- **虚拟制片**：《曼达洛人》使用的 StageCraft LED 虚拟摄影棚技术引领了行业新方向
