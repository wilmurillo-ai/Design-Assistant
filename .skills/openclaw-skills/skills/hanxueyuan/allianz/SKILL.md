# 安联集团 · Allianz SE

> "Outperform. Outthink. Outlast." — 欧洲最大的保险和资管集团

安联集团创立于1890年的柏林，从一家海运保险公司发展为全球性的保险和资产管理巨头。其旗下的PIMCO是全球最大的债券投资基金管理公司之一。安联的多元化业务结构（财险、寿险、资管三驾马车）使其在欧洲保险集团中拥有最强的抗风险能力。

## 值得了解

- 安联的Logo是三道横条，象征着"安全、信任和力量"

- PIMCO的"Total Return Fund"曾是全球最大的共同基金，管理超3,000亿美元

- 安联在二战后几乎被完全摧毁，但仅用10年就重建了业务

- 安联球场的冠名权归安联所有，是拜仁慕尼黑的主场

- 安联在2008年金融危机中没有接受任何政府救助

- PIMCO的创始人Bill Gross被称为"债券之王"

- 安联在1894年承保了世界上第一份海运保险

- 安联在1906年旧金山大地震后支付了巨额理赔，一举建立了国际声誉

- PIMCO的Total Return Fund连续30年跑赢基准指数，是共同基金历史上的奇迹
