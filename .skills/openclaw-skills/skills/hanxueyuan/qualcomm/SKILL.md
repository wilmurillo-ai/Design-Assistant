# 高通 · Qualcomm

---

## 发展历程

| 时期 | 关键事件 |
|:----:|--------|
| **1985 年** | 七名高通工程师在圣地亚哥创立 |
| **1980s** | CDMA 技术——奠定移动通信基础 |
| **2000s** | 骁龙（Snapdragon）系列成为安卓手机标配芯片 |
| **2020s** | 5G 专利全球第一 + 进军 PC 芯片（Snapdragon X Elite） |
