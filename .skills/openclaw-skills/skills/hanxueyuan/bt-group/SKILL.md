# 英国电信 · BT Group

> "英国通信基础设施的守护者，从国有垄断到市场化竞争"

BT Group 是英国历史最悠久的电信公司，其历史可追溯到 1846 年的电气电报公司。作为英国最大的固定线路宽带和移动网络运营商，BT 拥有英国最广泛的通信基础设施，包括 Openreach 网络，为数百万家庭和企提供连接服务。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1846** | 电气电报公司成立，英国电信的历史起点 |
| **1981** | 英国电信从邮政局独立，成为国有企业 |
| **1984** | 在英国政府主导下私有化，在伦敦证券交易所上市 |
| **1997** | 以 150 亿美元收购 MCI 进入国际电信市场 |
| **2003** | 创立 Openreach 部门，管理铜线和光纤接入网络 |
| **2016** | 以 125 亿英镑收购 EE 移动运营商 |
| **2019** | Ofcom 要求 Openreach 独立运营 |
| **2024** | 英国 5G 覆盖超过 70% 人口，光纤用户突破 1,000 万 |

---

## 业务结构分析

### 固定宽带

BT 和 EE 品牌的固定宽带和固定电话服务

### 移动通信

EE 品牌移动网络，英国最大的 4G 和 5G 运营商

### Openreach

英国最大的电信基础设施网络，覆盖 3,100 万处地址

### 企业服务

全球网络服务和 IT 解决方案，服务 20,000 家企业

## 核心护城河

```
  英国电信基础设施
    |
    Openreach网络 --> 物理垄断优势
    |                                  |
    EE移动领先 --> 用户规模
    |                                  |
    光纤投资 --> 宽带升级
    |                                  |
    企业服务 --> B2B收入
```

1. Openreach 管理着英国最大的电信接入网络，覆盖 3,100 万处地址
2. EE 是英国首家推出 4G 和 5G 商业服务的运营商
3. 英国政府要求 Openreach 向竞争对手开放接入，形成独特监管模式
4. 光纤到户 FTTP 用户在过去 3 年增长了 300%

## 值得了解

- BT 的历史可以追溯到 1846 年，比电话的发明还要早
- 前身是1846年成立的Electric Telegraph Company，是世界上第一家电报公司

- 运营着连接英国与全球的海底光缆网络，承担约45%的英国国际带宽

- 曾拥有英超联赛转播权，是英国体育广播的重要参与者


- Openreach 的铜线网络是英国电信历史上最大的基础设施资产

- EE 品牌的橙色标识是英国最具辨识度的移动通信品牌

- BT Tower 是伦敦的地标建筑之一，建于 1964 年，曾是英国最高的建筑
