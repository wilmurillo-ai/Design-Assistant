# 亚德诺半导体 · Analog Devices

> "模拟信号是现实世界的语言，ADI 是它的翻译者"

Analog Devices Inc.是全球领先的模拟和混合信号芯片制造商，产品覆盖数据转换、信号处理、电源管理和射频技术。公司通过收购Maxim Integrated进一步强化了在汽车和工业领域的领导地位。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1965** | MIT 教授 Ray Stata 创立公司，首批产品为运算放大器 |
| **1969** | 在纳斯达克上市，成为早期半导体上市公司之一 |
| **1990** | DSP 数字信号处理器问世，拓展至通信市场 |
| **2001** | 初步布局高性能信号链产品线 |
| **2017** | 以 200 亿美元收购 Linear Technology，史上最大模拟芯片并购 |
| **2021** | 以 210 亿美元完成对 Maxim Integrated 的收购 |
| **2023** | 汽车电子营收占比超过 30%，成为最大终端市场 |
| **2024** | 推出新一代自动驾驶传感器芯片组 |

---

## 业务结构分析

### 汽车电子

为 ADAS 自动驾驶和电动车提供传感器接口芯片电源管理方案

### 工业自动化

高精度数据采集和工业物联网传感器信号链解决方案

### 通信基础设施

5G 基站射频收发器和高速数据转换器

### 医疗健康

便携式医疗设备和患者监护仪器的精密信号处理芯片

## 核心护城河

```
  研发创新
    |
    高精度模拟芯片 --> 客户粘性
    |                       |
    收购整合能力 --> 产品组合扩展
    |                       |
    长产品生命周期 --> 稳定现金流
```

1. 模拟芯片设计需要数十年积累的工艺know-how，新进入者难以复制
2. 产品生命周期长达10年以上，客户一旦采用极少更换供应商
3. 通过大规模并购整合形成最完整的模拟芯片产品组合
4. 在汽车和工业等高可靠性市场建立了严格的认证壁垒

## 值得了解

- ADI 的创始人 Ray Stata 99 岁时仍在担任董事会职务，是半导体行业最年长的活跃创始人
- 其信号处理芯片被用于NASA的火星探测器和哈勃太空望远镜

- 通过连续并购策略，过去十年完成了超过15次重要收购

- 在精密ADC（模数转换器）市场占有率超过60%，几乎没有直接竞争对手


- 公司总部大楼内设有全球最完整的模拟电路博物馆，收藏了自 1965 年以来的所有芯片

- 2021 年对 Maxim 的收购创造了当时模拟芯片行业最大并购纪录

- ADI 的运算放大器被阿波罗登月计划使用，至今仍用于 NASA 的太空任务
