# 纽约大学 · New York University

> 大学的城市，城市的大学。

New York University (NYU) 是美国最大的私立大学之一，成立于1831年，主校区位于纽约市曼哈顿格林威治村。拥有约5.9万名学生，是全球规模最大的私立非营利性大学之一。在纽约主校区之外，还在阿布扎比和上海拥有学位授予权的分校。在艺术、商学、法学和医学领域拥有顶尖排名。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1831年** | 阿尔伯特·加勒廷创立纽约大学，是美国第一所位于大都市的大学 |
| **1886年** | 华盛顿广场校区正式成为主校区 |
| **1914年** | Stern商学院前身成立 |
| **1935年** | Tisch艺术学院成立 |
| **1970年代** | 面临财务危机，后在校长领导下复兴 |
| **1990年代** | 开始全球化战略，在伦敦、巴黎、佛罗伦萨等地设立学术中心 |
| **2008年** | NYU Abu Dhabi校区正式运营 |
| **2012年** | NYU Shanghai校区启用，中美合办 |
| **2020年代** | Stern商学院排名进入全美前五 |
| **2024年** | 全球校区网络覆盖五大洲，在校生约5.9万人 |

## 业务结构分析

### 艺术与人文（Tisch）

全球顶尖的艺术学院之一，培养了无数奥斯卡获奖者和百老汇明星。电影制作、表演和视觉艺术是其最强领域。

### 商科（Stern）

全美前五的商学院，金融和创业方向尤其强势。得益于纽约华尔街的地理优势，学生在投行和咨询行业就业率极高。

### 法学与公共政策

NYU法学院是全美T14法学院之一，在国际法、税法和公法领域排名顶尖。

### 全球教育网络

拥有全球最完整的大学海外校区网络，阿布扎比和上海两所分校提供完整学位教育，11个学术中心支持学期交换。

## 🔑 核心护城河

```
  纽约地理优势 → 实习与就业 → 学生吸引力
       ↓                        ↗
  全球校区网络 → 国际化教育 → 品牌差异化
       ↓
  顶尖学院声誉 → 师资与生源 → 排名提升
```

1. 纽约地理位置：位于曼哈顿核心地带，提供无与伦比的实习、就业和文化体验机会，任何其他大学无法复制
2. 全球校区网络：拥有全球最完整的大学海外校区体系，阿布扎比和上海两所分校提供完整学位，这种全球化布局独一无二
3. 艺术与商学双强：Tisch和Stern分别在全美同类学院中排名顶尖，这种学科组合在顶级大学中非常罕见
4. 规模效应：约5.9万学生的规模使NYU在研究投入、设施建设和校友网络上具有显著优势

## 值得了解

- NYU没有传统意义上的封闭校园，教学楼与城市街区融为一体，被誉为大学就是城市的典范

- NYU Abu Dhabi录取率低于3%（低于哈佛），为录取学生提供全额奖学金约30万美元每年

- NYU校友包括超过30位奥斯卡奖得主和30多位普利策奖得主

- NYU主校区华盛顿广场拱门是纽约最具标志性的地标之一，也是无数电影的取景地
- 该领域的发展历程反映了行业变革的重要趋势

- 其战略决策经常被商学院作为经典案例研究

- 在专业领域内保持着持续的创新能力和市场领先地位
