# 意大利国家电力 · Enel S.p.A.

> "Open Power" — Enel企业口号

Enel S.p.A.成立于1962年，是意大利国有电力公司经过私有化后形成的全球能源巨头。作为欧洲最大的可再生能源运营商和全球第三大电力公司，Enel在全球28个国家运营，管理超过7300万客户。公司已从传统化石燃料发电全面转型为可再生能源领导者，运营超过54GW的可再生能源装机容量。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1962** | 意大利政府将全国电力行业国有化，成立Enel |
| **1999** | 开始私有化进程，在米兰和纽约上市 |
| **2001** | 电力市场自由化，Enel面临竞争 |
| **2007** | 收购Endesa，成为西班牙最大电力公司 |
| **2014** | 推出开放式可再生能源战略 |
| **2016** | 收购智利和哥伦比亚的可再生能源资产 |
| **2019** | 宣布2030年前淘汰所有煤电 |
| **2020** | 可再生能源装机超过50GW里程碑 |
| **2022** | 加速电网数字化和储能投资 |
| **2023** | 全球可再生能源装机突破54GW |
| **2024** | 年营收超过900亿欧元，绿色转型加速 |
| **2025** | 氢能试点项目在意大利和西班牙启动 |

## 业务结构分析

### 可再生能源

风能、太阳能、地热和水电，全球超过54GW装机

### 电网运营

意大利和西班牙等国的输配电网络运营

### 零售供电

全球超过7300万终端客户的电力和天然气供应

### 新兴业务

储能、电动汽车充电、绿色氢能和能源管理服务

## 值得了解

- Enel是全球最大的地热发电运营商，在意大利托斯卡纳运营历史悠久的地热电站

- 公司的Innovability理念将创新(Innovation)和可持续(Sustainability)合并为一个战略

- Enel在西班牙收购的Endesa是西班牙历史上最大的并购案之一

- 公司的分布式能源管理系统控制着全球超过100万个智能电表

- Enel是首家承诺2030年前完全淘汰煤电的全球主要电力公司

- 公司在智利的可再生能源投资使其成为南美最大的绿色能源运营商

- Enel的绿色债券发行规模在欧洲公用事业公司中排名第一

- 公司的创新孵化器Enel X已孵化超过500个能源科技项目
