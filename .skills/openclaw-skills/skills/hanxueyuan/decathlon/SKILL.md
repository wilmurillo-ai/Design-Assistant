# 迪卡侬 · Decathlon

> "Sport for All, All for Sport" —— Decathlon

迪卡侬是全球最大的体育用品零售商，1976年由Michel Leclercq在法国里尔创立。迪卡侬以"让每个人都能以最低价格享受运动的快乐"为使命，覆盖70+运动品类，几乎所有产品都是自有品牌。从2美元的T恤到2,000美元的山地车，迪卡侬是全球性价比最高的体育用品零售商。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1976年** | Michel Leclercq在法国里尔开设第一家迪卡侬门店 |
| **1986年** | 推出第一个自有品牌Quechua（登山），开启全自有品牌战略 |
| **1990s** | 全球扩张，进入西班牙、德国、中国等市场 |
| **2000s** | 每年推出200+新产品，自主研发+自有制造体系成型 |
| **2016年** | 成立40周年，全球门店超1,500家 |
| **2020年** | 疫情后运动需求暴涨，线上销售增长超80% |
| **2023年** | 全球门店超2,000家，60+国家 |
| **2024年** | 营收超150亿欧元，中国成为第二大市场 |

## 业务结构分析

### 全品类体育产品
覆盖70+运动：足球、篮球、登山、骑行、游泳、瑜伽、滑雪等。每个品类都有专属子品牌。

### 自有品牌矩阵
Quechua（登山）、Kipsta（团队运动）、Tribord（水上运动）、B'Twin（骑行）、Domyos（健身）等20+品牌。

### 零售体验
店内设运动体验区（试骑、试投、试穿），鼓励顾客"先体验再购买"。

### 二手与可持续
Second Life二手运动装备计划，推动体育用品循环经济。

3. **规模效应**：全球2,000+门店+60+国家，采购成本极低
4. **体验式零售**：店内运动体验区让顾客在决策前就能试用产品

## 值得了解

- 迪卡侬的LOGO是一个由不同运动器材组成的"太阳"——象征"运动属于所有人"
- 迪卡侬在中国已成为"性价比运动装备"的代名词——2美元速干衣和20元瑜伽垫是爆款
- 创始人Michel Leclercq至今仍保持着极简生活方式——他开的车是最便宜的Dacia
- 迪卡侬的自行车品牌B'Twin曾被拆分为独立品牌，但后来发现消费者更认迪卡侬——又合并回来
