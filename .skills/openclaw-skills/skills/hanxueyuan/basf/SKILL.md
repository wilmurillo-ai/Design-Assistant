# 巴斯夫 · BASF SE

> "We create chemistry" — BASF企业口号

BASF成立于1865年的德国路德维希港，是全球最大的综合性化工集团。公司业务涵盖化学品、材料、工业解决方案、表面处理技术、营养与护理、农业解决方案六大领域，为全球几乎所有行业提供化学基础材料。巴斯夫在40多个国家拥有约390个生产基地，员工超过11万人。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1865** | Friedrich Engelhorn在德国曼海姆创立BASF |
| **1878** | 发明合成靛蓝染料，开启工业染料时代 |
| **1909** | 发明哈伯-博施法合成氨，获得诺贝尔奖 |
| **1925** | 与拜耳等合并组建IG法本公司 |
| **1952** | 在路德维希港建立全球最大的化工厂 |
| **1985** | 基因工程部门成立，进军生物技术 |
| **1990** | 收购东德化工企业，两德统一后整合 |
| **2001** | 将制药业务剥离为BASF制药 |
| **2012** | Verbund一体化战略深化，提升协同效应 |
| **2018** | 湛江一体化基地在中国开工建设 |
| **2022** | 因俄乌冲突削减在俄业务，重组全球供应链 |
| **2024** | 年营收约680亿欧元，加速向可持续发展转型 |

## 业务结构分析

### 化学品

基础化学品、中间体和石化产品，是公司最大的业务板块

### 材料

高性能塑料、聚氨酯和特种化学品

### 工业解决方案

汽车涂料、催化剂和电子材料

### 表面处理技术

工业催化剂和汽车尾气处理

### 营养与护理

维生素、化妆品原料和动物营养

### 农业解决方案

种子处理、杀菌剂和除草剂

## 值得了解

- BASF的Verbund一体化概念是现代化工生产最核心的效率创新

- 哈伯-博施法合成氨被认为养活了全球一半的人口

- 路德维希港基地拥有自己的消防队和医院

- BASF是世界上最大的靛蓝染料生产商，牛仔裤产业依赖其供应

- 公司的催化剂产品帮助全球汽车减少约10亿吨尾气排放

- BASF在湛江的投资是中国历史上最大的外商独资化工项目

- 公司的电子材料部门为全球芯片制造提供关键化学品

- BASF的生物可降解塑料Ecovax已应用于包装和农业领域
