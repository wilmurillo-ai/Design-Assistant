# 旷视科技 · Megvii

> "Deep learning for the real world" — 计算机视觉AI技术的领跑者

旷视科技由印奇、唐文斌和杨沐三位清华姚班校友于2011年创立，是中国"AI四小龙"之一。其核心产品Face++是全球最大的开放人脸识别API平台之一。近年来旷视从人脸识别扩展至供应链物流机器人和城市物联网领域，构建了"算法+软件+硬件"的全栈AI能力。

## 值得了解

- 三位创始人都是清华"姚班"（计算机科学实验班）的校友，平均年龄不到25岁

- Face++这个名字意为"Face plus plus"，寓意超越表面的人脸识别

- 旷视的Brain++框架是全球少数几个自研的深度学习框架之一

- 印奇曾是微软亚洲研究院最年轻的实习生

- 旷视在2017年开始做物流机器人，是因为发现仓储自动化是中国制造业最大的痛点之一

- Face++平台曾被用于支付宝的人脸支付功能

- 旷视的Logo是一张抽象的人脸轮廓，象征着其人脸识别的核心业务

- 旷视科技是第一个在LFW人脸识别竞赛中达到99.5%准确率的公司

- 旷视的物流机器人能在10分钟内完成一个标准仓库的布局规划
