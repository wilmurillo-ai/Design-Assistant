# 家乐福 · Carrefour

> "大卖场模式的发明者，欧洲零售业的活化石"

Carrefour 是全球大卖场 Hypermarket 概念的发明者，也是欧洲最大的零售集团之一。公司覆盖食品零售、大卖场、便利店和电子商务，在法国西班牙巴西阿根廷等市场占据领先地位。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1959** | Marcel Fournier 等人在法国安锡创立家乐福 |
| **1963** | 在巴黎郊区开设全球首家大卖场 |
| **1970** | 在西班牙和意大利开设首批海外门店 |
| **1999** | 与 Promodes 合并，巩固法国市场领导地位 |
| **2005** | 进入中国零售市场，后战略调整 |
| **2018** | 推出全渠道转型战略 Carrefour 2022 |
| **2021** | 与 Tesco 建立欧洲采购联盟 |
| **2024** | 食品电商增速超过 15%，全渠道转型成效显著 |

---

## 业务结构分析

### 大卖场

Carrefour 品牌大卖场，食品和日用品一站式购物

### 超市

Carrefour Market 社区超市，覆盖日常购物需求

### 便利店

Carrefour Express 和 proximity 便利店网络

### 电商

全渠道电商平台，支持到店自提和送货上门

## 核心护城河

```
  全渠道零售飞轮
    |
    大卖场规模 --> 采购成本优势
    |                           |
    门店网络密度 --> 最后一公里配送
    |                           |
    食品专长 --> 高频消费粘性
    |                           |
    欧洲采购联盟 --> 更强议价
```

1. 大卖场业态的发明者，定义了现代零售的商业模式
2. 在欧洲食品零售市场占据约 15% 的份额
3. 通过采购联盟与 Tesco 合作，增强对供应商的议价能力
4. 在拉美市场尤其是巴西和阿根廷保持强劲增长

## 值得了解

- 家乐福发明的 Hypermarket 大卖场概念被沃尔玛等全球零售商效仿
- 1960年创立了世界上第一个"大卖场"（hypermarket）概念，将超市与百货合并

- 在巴西和阿根廷拥有超过1000家门店，是南美最大的零售商之一

- 率先在零售业推行区块链溯源系统，消费者可扫描二维码追踪食品来源


- 1963 年首家大卖场面积超过 2,500 平方米，在当时是革命性的

- 家乐福是全球最早推出会员卡制度的零售商之一

- 公司名称 Carrefour 在法语中意为十字路口，首家店确实开在一个十字路口旁
