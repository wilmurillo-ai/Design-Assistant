# Agoda (安可达)

亚洲领先的在线酒店预订平台。

## 平台介绍

**Agoda**成立于 2005 年，总部位于新加坡，是 Booking Holdings 集团旗下的在线酒店预订平台。

### 基本信息

| 项目 | 详情 |
|------|------|
| **成立时间** | 2005 年 |
| **总部** | 新加坡 |
| **覆盖国家** | 200+ |
| **房源数量** | 200 万 + |
| **支持语言** | 38+ |
| **母公司** | Booking Holdings |

### 核心优势

| 优势 | 介绍 |
|------|------|
| **亚洲资源** | 亚太地区酒店资源丰富 |
| **价格优惠** | 独家优惠和折扣 |
| **即时确认** | 预订即时确认 |
| **免费取消** | 大部分房源支持 |
| **真实评价** | 住客真实点评 |
| **多币种** | 支持多种货币支付 |

### 住宿类型

- **星级酒店**: 豪华、高端、中端酒店
- **度假村**: 海滨、滑雪、温泉度假村
- **公寓民宿**: 服务式公寓、民宿
- **特色住宿**: 别墅、树屋、船屋
- **青年旅社**: 经济型住宿
