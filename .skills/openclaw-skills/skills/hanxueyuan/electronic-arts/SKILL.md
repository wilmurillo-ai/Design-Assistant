# 艺电 · Electronic Arts Inc.

> "挑战极限，我们热爱游戏"——EA早期使命宣言

Electronic Arts是全球最大的互动娱乐软件公司之一，总部位于加州雷德伍德城。公司以体育游戏（EA Sports）和知名IP（模拟人生、战地、极品飞车）闻名。2024财年营收约76亿美元，拥有超过5亿注册用户，是全球游戏行业收入排名前五的发行商，其Ultimate Team模式开创了游戏内购收入的新范式。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1982** | Trip Hawkins创立EA，口号"我们看得见的人" |

| **2009** | 推出Ultimate Team模式，开启内购时代 |
| **2011** | Origin数字平台上线，对抗Steam |
| **2017** | Star Wars Battlefront II微交易争议 |
| **2020** | EA Play订阅服务与Xbox Game Pass整合 |
| **2022** | FIFA合作终止，更名EA Sports FC |
| **2024** | EA Sports FC 25首年营收创纪录 |

## 业务结构分析

### EA Sports

核心收入来源，FIFA/EA Sports FC、Madden NFL和NBA Live构成体育游戏矩阵，Ultimate Team模式贡献数十亿美元年收入

### EA Entertainment

模拟人生系列（全球最畅销PC游戏）、战地系列和Apex Legends等，覆盖模拟、射击和免费游戏市场

### EA Mobile

手游业务快速增长，EA Sports Mobile、Sims FreePlay等，占营收约20%，中国市场是重点

### EA Play订阅

游戏订阅服务，与Xbox Game Pass和PlayStation Plus合作，经常性收入模式

## 🔑 核心护城河

```
    [体育IP授权] --> EA Sports
           |                  |
    独家NFL/FIFA          玩家社区
           |                  |
           v                  v
    [Ultimate Team内购] --> [年复一年收入]
           |
           v
    [订阅+新IP] --> [生态扩张]
```

1. 体育IP垄断：拥有NFL独家授权，NBA和FIFA/FC深度合作关系难以替代
2. Ultimate Team模式：玩家投入时间和金钱形成沉没成本，每年续购新作
3. 多平台分发：覆盖主机、PC和移动端，EA Play订阅绑定用户长期消费
4. 全球发行能力：本地化团队覆盖50+国家，实时运营服务数十亿玩家

## 值得了解

- EA创始人Trip Hawkins创立公司时年仅28岁，是哈佛商学院最年轻的毕业生之一

- 模拟人生系列全球累计销量超2亿份，是有史以来最畅销的PC游戏系列

- EA在2017年因Star Wars Battlefront II的微交易争议被Reddit踩到历史最低分

- EA Sports FC是EA与国际足联终止20年合作后推出的替代品牌

- Apex Legends在2019年上线首周即吸引2,500万玩家，是EA最成功的免费游戏

- EA总部位于硅谷，但最大的工作室群分布在加拿大温哥华和瑞典斯德哥尔摩

- Madden NFL系列自1988年以来每年发行新版本，从未中断过

- EA的Frostbite引擎被广泛用于旗下多款游戏，但也因技术债务问题备受批评
