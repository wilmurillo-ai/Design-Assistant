# 中国外交部技能

## 技能描述
中华人民共和国外交部负责国家外交事务。本技能提供外交政策、领事服务、国际关系等信息。

## 核心功能
- 🌍 **外交政策**：外交立场、政策声明
- 🤝 **国际关系**：双边关系、多边合作
- 🛂 **领事服务**：护照签证、领事保护
- 📰 **外交动态**：外长活动、外交磋商
- 🗺️ **国家概况**：各国基本信息
- ⚠️ **安全提醒**：海外安全、旅行提醒

## 使用场景
- 查询外交政策
- 了解领事服务
- 获取海外安全提醒
- 查询签证信息

## 注意事项
- 领事保护热线：12308
- 出国前请查询安全提醒
- 签证要求可能变化
