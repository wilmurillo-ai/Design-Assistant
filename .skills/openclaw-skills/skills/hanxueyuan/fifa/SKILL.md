# 国际足联技能

## 技能描述
FIFA（国际足联）是世界足球运动的管理机构，负责组织世界杯、推广足球运动。本技能帮助用户了解 FIFA 赛事、球队、球员信息。

## 核心功能
- ⚽ **世界杯**：男足、女足世界杯
- 📅 **赛事日程**：比赛时间、赛程
- 🏆 **世界排名**：国家队、俱乐部
- 🌍 **成员协会**：211 个成员
- 🏟️ **主办信息**：历届、申办
- 📊 **数据统计**：进球、纪录

## 使用场景
- 查询世界杯信息
- 了解赛事安排
- 获取球队排名
- 查询足球数据

## 注意事项
- 世界杯每四年举办
- 赛程可能调整
- 以 FIFA 官网为准
