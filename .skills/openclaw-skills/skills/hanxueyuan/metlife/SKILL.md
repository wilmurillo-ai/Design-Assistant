# 大都会人寿 · MetLife

> "Together for what matters" — 全球最大人寿保险公司之一

大都会人寿创立于1868年，最初为美国内战后移民潮提供保险保障。经过150多年发展，已成为全球保险业的巨头之一。其独特的标志——史努比（Snoopy）和查理·布朗——自2001年起成为品牌代言人，使严肃的保险业务拥有了温暖亲切的形象。近年来积极布局亚洲市场和数字化保险平台。

## 业务结构分析

### 北美保险

美国人寿保险、年金和员工福利，核心收入来源

### 亚洲保险

日本、韩国、中国等亚洲市场的寿险和医疗险

### 拉丁美洲

墨西哥、巴西、智利等市场的保险业务

### 员工福利

全球企业员工福利和团体保险方案

## 值得了解

- MetLife的史努比形象是从United Feature Syndicate授权的，已使用了超过20年

- 大都会人寿大厦（MetLife Building）曾是纽约市最高的建筑之一

- 1909年推出的团体寿险是世界上第一个面向企业的员工保险方案

- MetLife在2001年9·11事件中支付了超过5亿美元的理赔

- 其日本子公司是MetLife海外最大的利润来源

- MetLife Stadium是NFL纽约巨人队和喷气机队的主场

- MetLife在2020年疫情初期向医护人员捐赠了超过1亿美元的保险保障

- 大都会人寿是第一个在广告中使用动画角色的保险公司

- MetLife在2019年宣布退出史努比代言合同，但在2021年又续签了
