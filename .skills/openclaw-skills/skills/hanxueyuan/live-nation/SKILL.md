# 现场娱乐 · Live Nation Entertainment

> "The world leader in live entertainment."

Live Nation Entertainment 是全球最大的现场娱乐公司，由 Live Nation 和 Ticketmaster 于 2010 年合并而成。公司运营着全球最大的演唱会推广网络，拥有超过 70 个音乐节品牌，每年举办超过 50,000 场演出，覆盖超过 40 个国家。旗下艺人包括 Madonna、U2、Jay-Z 等顶级明星。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **2005** | Live Nation 从 Clear Channel 分拆独立 |
| **2009** | 宣布与 Ticketmaster 合并 |
| **2010** | Live Nation Entertainment 正式成立，在 NYSE 上市 |
| **2012** | Citi 以约 4 亿美元收购 Live Nation 18% 股份 |
| **2013** | 收购 C3 Presents（Austin City Limits 主办方） |
| **2015** | 收购 AC Entertainment（Bonnaroo 主办方） |
| **2017** | 收购 BigChampagne 数据公司 |
| **2019** | 美国司法部对其提起反垄断诉讼 |
| **2021** | 疫情后全面恢复演出，创纪录营收 |
| **2022** | Taylor Swift 巡演门票销售事件引发反垄断审查 |
| **2023** | 年度营收突破 200 亿美元 |
| **2024** | 营收超 210 亿美元，举办演出超 50,000 场 |

## 值得了解

- Live Nation 最初是 Clear Channel 的现场娱乐部门，2005 年被分拆出来

- Ticketmaster 的历史可以追溯到 1976 年，是全球最古老的票务公司之一

- 2022 年 Taylor Swift 巡演门票销售崩溃事件引发了美国国会的反垄断听证

- 美国司法部对 Live Nation-Ticketmaster 合并的反垄断诉讼仍在进行中

- Live Nation 每年举办的音乐节吸引了超过 300 万名观众

- Lollapalooza 音乐节已扩展到智利、巴西、德国、法国和印度

- Live Nation 旗下的 House of Blues 场馆是美国最知名的音乐场馆品牌之一

- Citi 在 2012 年以 4 亿美元收购了 Live Nation 18% 的股份

- Live Nation 是最早推出虚拟演唱会和流媒体直播的公司之一

- 公司每年在营销上的支出超过 20 亿美元
