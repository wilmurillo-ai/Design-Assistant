# 法国液气集团 · Air Liquide S.A.

> "A company, a planet, a future" — Air Liquide愿景

Air Liquide S.A.成立于1902年，是法国跨国工业气体集团，与Linde并列为全球工业气体行业的领导者。公司专注于工业气体（氧气、氮气、氢气等）、医疗气体和电子特种气体的研发与供应。Air Liquide在全球80个国家运营，拥有超过67000名员工，为从钢铁冶炼到半导体制造的多个行业提供关键气体解决方案。

| 时期 | 关键事件 |
|:----:|--------|
| **1902** | Paul Delorme和Georges Claude在法国创立 |
| **1906** | 发明第一台工业制氧机 |
| **1938** | 开始医用氧气供应业务 |
| **1960** | 推出低温制气技术革命 |
| **1980** | 进入美国市场，建立北美业务 |
| **2001** | 收购中国多家气体公司，进入亚洲 |
| **2007** | 成为全球第二大工业气体供应商 |
| **2015** | 收购Airgas，大幅扩大北美业务 |
| **2018** | 推出氢能源战略，投资加氢站网络 |
| **2022** | 电子特种气体业务大幅增长 |
| **2023** | 小型模块化制氢设备商业化 |
| **2024** | 年营收超过270亿欧元，氢能和电子气体增长强劲 |

## 业务结构分析

### 工业气体

氧气、氮气、氢气等大宗气体的生产和供应

### 医疗保健

医用氧气、呼吸治疗设备和麻醉气体

### 电子/半导体

超高纯度电子特种气体，用于芯片制造

### 工程/服务

气体设备设计、安装和维护服务

## 值得了解

- Air Liquide的创始人Paul Delorme曾是一名冶金工程师

- 公司的电子特种气体可以精确到十亿分之一(ppb)级别的纯度

- Air Liquide为阿波罗登月计划提供宇航员呼吸用氧

- 公司在全球运营的加氢站数量超过150座，是全球最大的加氢网络

- Air Liquide的医疗气体业务为全球超过2000家医院提供服务

- 公司的氢能战略目标是到2030年成为全球氢能领导者

- Air Liquide在法国的工厂是世界上最大的空气分离装置之一

- 公司的电子气体部门为全球主要芯片制造商供货
