# 恩桥 · Enbridge

---

## 发展历程

| 时期 | 关键事件 |
|:----:|--------|
| **1949 年** | 作为 Interprovincial Pipe Line 成立 |
| **1950s** | 建设第一条跨加拿大输油管道 |
| **1998 年** | 更名为 Enbridge |
| **2020s** | 运营北美最长管道网络（超过 40000 公里） |

---

## 核心特点

管道运输（原油/天然气）+ 天然气配送 + 可再生能源

## 核心优势

北美能源运输的'收费公路'——管道网络覆盖美国和加拿大

## 关键数据

| 指标 | 数值 |
|------|------|
| 管道长度 | 超过 40000 公里 |
| 年收入 | 约 500 亿加元 |
| 员工数 | 约 10000 人 |
