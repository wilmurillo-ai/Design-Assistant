# 任天堂 · Nintendo

---

## 发展历程

| 时期 | 关键事件 |
|:----:|--------|
| **1889 年** | 山内房治郎在京都创立 **任天堂花札**——手工纸牌公司 |
| **1980s** | Game & Watch 掌机成功——Game Boy 后来成为最畅销掌机 |
| **1985 年** | **超级马里奥兄弟** 在 NES 上发布——拯救了崩溃的北美游戏市场 |
| **1989 年** | Game Boy 发布——全球销量 1.18 亿台 |
| **1996 年** | **宝可梦** 系列发布——后来成为全球最赚钱的媒体 IP |
| **2006 年** | **Wii** 发布——体感游戏革命，1.01 亿台销量 |
| **2017 年** | **Nintendo Switch** 发布——家用+掌机混合概念，超过 1.4 亿台销量 |
| **2024 年** | Switch 后继机型公布 |

---

## 核心特点

游戏硬件（Switch）+ 软件（第一方游戏：马里奥/塞尔达/宝可梦）+ IP 授权（电影/周边）

## 核心优势

全球最强第一方游戏 IP 库（马里奥、宝可梦、塞尔达、动森）+ 独特的游戏设计理念
