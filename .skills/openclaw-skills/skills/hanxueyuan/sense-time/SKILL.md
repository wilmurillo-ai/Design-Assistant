# 商汤科技 · SenseTime

> "Sense the world, create the future" — AI视觉技术的先行者

商汤科技由汤晓鸥教授于2014年在香港中文大学创立，是中国"AI四小龙"中估值最高的公司。其核心优势在于计算机视觉和深度学习算法，拥有全球最大规模之一的AI算力基础设施SenseCore。尽管面临商业化和盈利挑战，商汤在智慧城市、自动驾驶和生成式AI领域的技术实力处于全球第一梯队。

## 值得了解

- 创始人汤晓鸥教授是计算机视觉领域的世界级学者，曾在微软亚洲研究院工作

- 商汤的名字来源于"Sense"（感知）的音译和汤晓鸥教授的姓氏

- 商汤是第一个被美国列入实体清单的AI公司（2019年）

- SenseCore AI大装置的名称寓意"感知世界的核心"

- 商汤在2020年疫情期间开发了AI体温检测和口罩识别系统

- 商汤的投资方包括阿里巴巴、高通和银湖资本等国际知名机构

- 日日新SenseNova大模型的命名灵感来自中国古代典籍《大学》中的"苟日新，日日新"

- 商汤科技的日日新大模型在2024年通过了中国首个AI大模型备案

- SenseCore AI大装置的能耗管理使用了液冷技术，PUE值低至1.15
