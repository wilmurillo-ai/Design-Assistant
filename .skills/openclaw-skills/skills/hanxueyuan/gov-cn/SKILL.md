# 中国政府网技能

## 技能描述
中国政府网是国家级政务服务平台，提供政策发布、政务服务、信息公开等功能。本技能帮助用户查询政府信息、了解政策法规。

## 核心功能
- 📰 **政策发布**：国务院政策、部委文件
- 🏛️ **机构信息**：政府机构设置、职能介绍
- 📋 **政务服务**：办事流程、在线服务
- 📊 **数据公开**：统计数据、公报信息
- 🔍 **信息检索**：政策文件搜索
- 📞 **互动交流**：留言咨询、意见征集

## 使用场景
- 查询最新政策法规
- 了解政府机构设置
- 获取政务服务指南
- 查看官方统计数据

## 注意事项
- 信息以官方网站为准
- 政策文件请注意发布日期
- 地方政策请咨询当地政府
