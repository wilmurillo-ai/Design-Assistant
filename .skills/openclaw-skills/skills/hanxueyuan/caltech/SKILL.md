# 加州理工学院 · Caltech

---

## 发展历程

| 时期 | 关键事件 |
|:----:|--------|
| **1891 年** | Throop University 创立 |
| **1920s** | 转型为理工科研究型大学 |
| **1930s** | 吸引爱因斯坦/冯·卡门等顶尖学者 |
| **1940s** | NASA 喷气推进实验室（JPL）由 Caltech 运营至今 |

---

## 核心特点

本科 + 研究生（工程/物理/化学/生物/行星科学）

## 核心优势

师生比 1:3（全球最低）+ 所有学生必须完成科研项目

---

## 关键数据

| 指标 | 数值 |
|------|------|
| 学生数 | 约 2400 人 |
| 诺贝尔奖 | 70+ 位（关联） |
| 录取率 | 约 3% |

---

## 值得了解

Caltech 只有约 2400 名学生——却产生了 70+ 位诺贝尔奖，人均诺奖率全球第一
