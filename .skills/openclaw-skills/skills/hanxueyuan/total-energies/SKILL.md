# 道达尔能源 · TotalEnergies

> "我们不再是一家石油公司——我们是一家多元能源公司。" TotalEnergies 正在用一场从品牌名到业务结构的全面变革，重新书写能源巨头的定义。

---

## 身份：从三个缩写词到一家公司

TotalEnergies 的名字本身就是一部并购史：

```
1924  Compagnie Française des Pétroles (CFP)  ← 法国政府主导创立
  │
1985  Total                               ← 品牌简化，走向国际
  │
1999  TotalFina                           ← 收购比利时 Petrofina
  │
2000  TotalFinaElf                        ← 收购 Elf Aquitaine（法国石油巨头）
  │
2003  Total                               ← 品牌统一
  │
2021  TotalEnergies                       ← 品牌重塑，宣示能源转型决心
```

每一次更名都对应一次重大战略转型。2021 年从 **Total** 变为 **TotalEnergies** 是最具象征意义的一次——**"Energies"（复数）** 意味着公司不再只靠石油和天然气，而是要在太阳能、风能、电力、氢能等多个能源赛道同时竞争。

---

## 五大业务板块

与传统石油公司"上游-下游"的简单划分不同，TotalEnergies 采用**五大业务板块**架构：

| 板块 | 核心业务 | 关键市场 |
|------|---------|---------|
| **勘探与生产 (E&P)** | 全球油气勘探、开发、生产 | 中东、非洲、北海、巴西、美国 |
| **综合天然气、可再生电力与电力 (IGR&P)** | LNG 贸易、太阳能、风电、电池储能 | 全球（LNG 贸易量全球前三） |
| **炼油与化工** | 原油炼制、石化产品生产、生物燃料 | 欧洲、非洲、中东 |
| **营销与服务** | 全球加油站网络（约 15,000 座）、润滑油、航空燃油 | 全球（品牌名 TotalEnergies） |
| **电力与储能** | 可再生能源开发、电力交易、电池生产 | 欧洲、非洲、中东、亚洲 |

## 🌱 能源转型：与同行的差异化路径

TotalEnergies 的转型策略与 BP（先激进后回调）和 Shell（摇摆不定）形成鲜明对比——它走的是一条**"渐进式多元化"** 路线：

### 可再生能源投资目标

| 年份 | 目标 |
|------|------|
| 2023 | 可再生能源装机 ~15 GW |
| 2025 | 装机 35 GW |
| 2030 | 装机 **100 GW**（太阳能 + 风能） |

### 核心转型举措

- **太阳能**：收购 SunPower（美国太阳能公司）、投资印度 Adani Green Energy、在法国/西班牙/美国建设大型光伏电站
- **LNG 霸主**：TotalEnergies 是全球**第二大 LNG 交易商**（仅次于壳牌），天然气被视为"过渡能源"的核心
- **电动出行**：在欧洲加油站大规模部署电动车充电桩
- **生物燃料**：将部分炼厂改造为生物炼厂
- **2050 目标**：全球运营与能源产品实现净零排放
