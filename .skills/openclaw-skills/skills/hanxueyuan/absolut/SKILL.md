# Absolut Vodka

## 概述

瑞典标志性伏特加品牌，以长达数十年的标志性广告运动和瓶身设计闻名于世

## 历史时间线

- **1879年**: Lars Olsson Smith 在瑞典 Åhus 创立绝对伏特加酿造工艺
- **1979年**: Vin & Sprit 推出 Absolut 品牌进军国际市场
- **1985年**: 启动 'Absolut Perfection' 广告运动，开启 25 年标志性营销
- **2008年**: Pernod Ricard 以 €56 亿收购 Absolut
- **2010s**: 推出 flavored 系列（柑橘、覆盆子等）
- **2023年**: 全球销量超 2000 万箱，仍为顶级伏特加品牌


## 商业模式

高端伏特加核心产品线 + 风味系列。通过 Pernod Ricard 的全球分销网络覆盖 125+ 国家。品牌溢价策略，定位高于普通伏特加。

## 护城河分析

独一无二的瓶身设计（灵感来自瑞典药瓶）无法复制；25 年广告运动建立了深入人心的品牌形象；Pernod Ricard 全球分销；瑞典 Åhus 的纯净水源和冬小麦原料

## 关键数据

- **annual_sales**: ~20M+ cases
- **brand_value**: ~$50 亿+
- **owner**: Pernod Ricard
- **markets**: 125+


## 有趣事实

Absolut 最著名的广告运动 'Absolut [X]' 持续了 25 年，制作了超过 1,500 幅广告，全部以瓶身轮廓为核心视觉。这被认为是广告史上最长寿、最一致的品牌运动之一。
