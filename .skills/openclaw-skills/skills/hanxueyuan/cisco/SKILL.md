# 思科 · Cisco

---

## 发展历程

| 时期 | 关键事件 |
|:----:|--------|
| **1984 年** | 斯坦福大学的 Leonard Bosack 和 Sandy Lerner 创立 |
| **1990s** | 互联网繁荣期——路由器和交换机需求爆发 |
| **2000s** | 大量收购扩张 |
| **2020s** | 向软件和订阅模式转型 |
