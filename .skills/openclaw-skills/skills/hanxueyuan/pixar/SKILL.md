# 皮克斯 · Pixar

---

## 发展历程

| 时期 | 关键事件 |
|:----:|--------|
| **1979 年** | 乔治·卢卡斯在卢卡斯影业内创立计算机图形部门 |
| **1986 年** | Steve Jobs 以 1000 万美元收购并命名为 Pixar |
| **1995 年** | **玩具总动员** 发布——史上第一部全 CGI 动画长片 |
| **2000s** | 连续出品经典：《海底总动员》《超人总动员》《飞屋环游记》 |
| **2006 年** | 迪士尼以 74 亿美元收购 Pixar |
| **2020s** | 《心灵奇旅》《青春变形记》等持续创新 |

---

## 核心特点

动画电影制作 + 技术研发（RenderMan 渲染引擎）

## 核心优势

全球最成功的动画工作室——每部电影几乎都是票房和口碑双丰收
