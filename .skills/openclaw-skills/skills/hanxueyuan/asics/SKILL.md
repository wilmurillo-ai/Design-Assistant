# 亚瑟士 · ASICS Corporation

> "健全的精神寓于健全的身体。"

亚瑟士（ASICS）是日本最大的运动品牌，由鬼冢喜八郎于 1949 年在神户创立。品牌名称源自拉丁语 Anima Sana In Corpore Sano（健全的精神寓于健全的身体）。ASICS 以 Gel 缓震技术和专业跑鞋闻名，在全球跑鞋市场稳居前三。2024 年营收约 7,500 亿日元。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1949** | 鬼冢喜八郎在神户创立鬼冢株式会社（Onitsuka Tiger） |
| **1960s** | 日本国家田径队穿着鬼冢跑鞋参加奥运会 |
| **1977** | Onitsuka Tiger、GTO 和 JELENK 合并，成立 ASICS 公司 |
| **1986** | 推出 Gel 缓震技术，革新跑鞋缓冲性能 |
| **1990s** | 进军欧美市场，与 Nike 和 Adidas 竞争 |
| **2010s** | Kayano 系列跑鞋成为经典，累计销量超 5,000 万双 |
| **2024** | 营收约 7,500 亿日元，复古跑鞋系列在年轻群体中爆火 |

---

## 业务结构分析

### 专业跑鞋

Gel-Kayano、Gel-Nimbus、Gel-Cumulus 等系列，马拉松跑者首选

### 运动鞋

复古 Gel-1130、Gel-1090 等在街头时尚中走红

### 运动服饰

跑步服装、压缩衣和配件

### 球类运动

网球鞋、排球鞋，在大满贯赛事中有较高使用率

---

## 🔑 核心护城河

```
Gel 技术积累 → 跑者口碑 → 品牌声望 → 更多研发投入 → 产品创新 → 更多跑者
```

1. Gel 缓震技术 30+ 年的专利积累和持续迭代，在跑步圈的专业口碑无可替代
2. Kayano 系列是跑鞋历史上最长寿的产品线之一（1993 年至今，超 30 代）
3. 在马拉松和铁人三项赛事中的赞助覆盖率极高
4. 复古鞋款在社交媒体上的二次爆火打开了年轻消费者市场
