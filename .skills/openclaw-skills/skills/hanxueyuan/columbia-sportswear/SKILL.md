# 哥伦比亚运动 · Columbia Sportswear

> "Tested Tough."

Columbia Sportswear 是美国知名的户外运动服装和装备公司，总部位于俄勒冈州波特兰。旗下拥有 Columbia、Mountain Hardwear、SOREL 和 prAna 四大品牌，覆盖从休闲户外到专业登山的各个细分市场。公司以技术创新（如 Omni-Heat 保暖科技）和性价比著称。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1938** | Paul Lamfrom 在波特兰创立 Columbia Hat Company |
| **1960s** | Neal Boyle 接管公司，转向户外服装 |
| **1970s** | 推出 Bugaboo Parka，成为标志性产品 |
| **1980s** | Columbia Sportswear Company 正式成立 |
| **1997** | Columbia 品牌重新定位，Gert Boyle 主导产品创新 |
| **1998** | 在纳斯达克上市，股票代码 COLM |
| **2001** | 收购 Mountain Hardwear 专业户外品牌 |
| **2005** | 收购 SOREL 鞋类品牌 |
| **2010** | 收购 prAna 可持续瑜伽户外品牌 |
| **2015** | 推出 Omni-Heat Infinity 保暖技术 |
| **2020** | 疫情期间户外产品销售爆发式增长 |
| **2024** | 营收约 32 亿美元，产品覆盖 90+ 国家 |

---

## 业务结构分析

### Columbia 品牌

核心户外服装品牌，夹克、裤装、鞋类和配件

### Mountain Hardwear

专业登山和高海拔装备，面向高端户外爱好者

### SOREL

时尚户外鞋类品牌，冬季靴子市场占有率极高

### prAna

可持续瑜伽和户外生活方式品牌，强调环保材料

## 值得了解

- Columbia 最初是一家帽子制造公司，1938 年为当地伐木工人提供工作帽

- Gert Boyle 在丈夫去世后接管公司，被称为户外行业的铁娘子

- Omni-Heat 保暖技术最初是受宇航服保温层的启发开发的

- Columbia 的 Bugaboo Parka 在 1980 年代推出后成为最畅销的户外夹克之一

- Mountain Hardwear 的 Ghost Whisperer 超轻羽绒服是登山爱好者的首选

- SOREL 的 Caribou 雪地靴在北美冬季市场占有率超过 40%

- prAna 是最早使用有机棉和再生材料的户外品牌之一

- Columbia 是最早在户外服装中使用可回收材料的品牌之一

- Columbia 的 Titanium 系列是品牌最高端的技术产品线

- Gert Boyle 在 2019 年去世前一直担任公司首席品牌官
