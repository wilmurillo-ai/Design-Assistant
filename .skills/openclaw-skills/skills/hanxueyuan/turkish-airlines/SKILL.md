# 土耳其航空 · Turkish Airlines

> *"Widen Your World."*

---

## 全球飞得最远的国家航司

土耳其航空（Türk Hava Yolları）是土耳其的国家载旗航空公司，也是**全球飞往国家数量最多**的航空公司——其航线网络覆盖超过 **130 个国家和地区**，远超任何其他航司。以伊斯坦布尔这座横跨欧亚两大陆的城市为枢纽，土耳其航空天然拥有连接东西方的地理优势。

> ✈️ 土耳其航空是全球首个（也是唯一一个）航线网络覆盖 **联合国全部 197 个国家中超过 128 个** 的航空公司。

## 百年飞行的轨迹

### 黎明（1933–1956）

土耳其航空的故事始于 **1933 年**，当时它是土耳其国防部下属的一个小型机构，仅有 **5 架飞机** 和 **17 名员工**。最初只运营国内航线——伊斯坦布尔至安卡拉和埃斯基谢希尔。

| 日期 | 事件 |
|------|------|
| **1933.05.20** | 土耳其国家航空（Devlet Hava Yolları）正式成立 |
| **1933.09** | 首航：伊斯坦布尔 → 埃斯基谢希尔 → 安卡拉 |
| **1938** | 机队扩充至 12 架飞机 |
| **1956.02.20** | 重组为独立商业实体 **Türk Hava Yolları** |

### 国际扩张（1956–1990）

- **1957**：开通首条国际航线——伊斯坦布尔 → 尼科西亚（塞浦路斯）
- **1960s**：逐步进入欧洲市场——伦敦、巴黎、罗马
- **1971**：加入 **IATA**（国际航空运输协会）
- **1980s**：开通北美航线（纽约、洛杉矶），机队现代化（引入波音 737）

### 黄金时代（2000s–2010s）

这是土耳其航空真正崛起的十年：

| 年份 | 关键事件 |
|------|----------|
| **2006** | 加入 **星空联盟（Star Alliance）** |
| **2010** | 航线网络突破 **200 个目的地** |
| **2010** | 成为 **欧洲足球锦标赛** 和后续多项国际体育赛事官方合作伙伴 |
| **2011** | 被 Skytrax 评为"南欧最佳航空公司" |
| **2014** | 成为 **FC Barcelona** 官方赞助商（球衣胸前广告） |
| **2016** | 年运送旅客突破 **6,000 万** |
| **2018.10** | 伊斯坦布尔新机场（IST）一期启用——全球最大单体航站楼 |

### 新机场新时代（2018–至今）

**2018 年 10 月 29 日**（土耳其共和国国庆日），土耳其航空从阿塔图尔克机场（ISL）全面迁至全新的 **伊斯坦布尔机场（IST）**。这座机场最终规划年吞吐量达 **2 亿人次**，将超过北京首都机场成为全球最大航空枢纽。

> 🏗️ 新机场建设耗资超过 **120 亿欧元**，拥有 **6 条跑道**和一座面积超过 **140 万平方米**的航站楼。

## 🍽️ 服务与体验

### 飞行餐饮——"空中厨房"

土耳其航空的机上餐饮以 **"土耳其美食大使"** 为定位，在全球航司中独树一帜：

- **商务舱**：由名厨 **Doğuş Group** 设计的精致土耳其菜单
- **经济舱**：热餐标配——经典的土耳其烤肉（Köfte）、皮拉夫（Pilav）、果仁蜜饼（Baklava）
- **特殊餐**：超过 30 种特殊饮食选项（清真、素食、低钠等）
- **Flying Chefs**：长途航班上有专职"飞行厨师"现场准备餐食

> 🥘 Turkish Airlines 是 **APEC（亚太餐饮委员会）** 多次"最佳机上餐饮"奖项得主。

### 商务舱体验

| 项目 | 内容 |
|------|------|
| **座位** | 全平躺式商务舱座椅（180°） |
| **娱乐系统** | 4K IFE，超过 1,000 部电影/电视节目 |
| **休息室** | 伊斯坦布尔 IST 土耳其航空休息室——全球最大航司休息室之一 |
| **接送服务** | 商务舱免费提供伊斯坦布尔市区接送（部分航线） |
