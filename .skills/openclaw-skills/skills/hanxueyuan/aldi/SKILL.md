# 阿尔迪 · Aldi Group

> "简单就是极致的高效。"

阿尔迪（Aldi）是全球最大的折扣零售商之一，由阿尔布雷希特家族于 1946 年在德国创立。1960 年代兄弟分家后形成 Aldi Nord 和 Aldi Sud 两个独立运营集团。公司以极简 SKU（约 1,500-2,000 个）、自有品牌为主（90%+）和极致效率的运营模式闻名。2024 年全球年营收估计超过 1,300 亿欧元。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1913** | Anna Albrecht 在德国埃森开设第一家食品店 |
| **1946** | 兄弟 Karl 和 Theo Albrecht 接管家族生意 |
| **1961** | 兄弟分家，成立 Aldi Nord（北）和 Aldi Sud（南） |
| **1976** | Aldi Sud 进入美国市场 |
| **1990s** | 全球扩张加速，进入英国、澳大利亚、中国等市场 |
| **2024** | 全球门店超 12,000 家，计划在美国投资 50 亿美元扩店 |

---

## 业务结构分析

### 德国本土

Aldi Nord 和 Aldi Sud 合计约 4,000 家门店，德国折扣零售双巨头之一

### 美国市场

约 2,200 家门店，目标 2028 年达到 2,500 家，是增长最快的市场

### 欧洲其他国家

英国、法国、荷兰、西班牙、波兰等，英国约 970 家门店

### 澳大利亚

约 560 家门店，是澳大利亚第三大超市集团

---

## 🔑 核心护城河

```
精简SKU → 极致成本控制 → 超低价格 → 高客流 → 大规模采购 → 更强议价力
```

1. 极致精简的运营模式（约 1,500-2,000 个 SKU vs 沃尔玛 140,000 个）
2. 90% 以上自有品牌，绕过中间商直接对接供应商
3. 家族所有、不上市、不披露财务，可以长期布局
4. 硬折扣模式在通胀时期表现突出，2022-2024 年高通胀期增长最快

## 值得了解

Aldi 的购物车需要投币 1 欧元才能使用，这个设计减少了员工整理工作量

Aldi 的员工效率是行业平均水平的 2-3 倍

创始人 Theo Albrecht 曾被绑架 17 天，赎金 3,000 万德国马克

Trader Joes 最初是 Aldi 在美国创建的，后来被出售
