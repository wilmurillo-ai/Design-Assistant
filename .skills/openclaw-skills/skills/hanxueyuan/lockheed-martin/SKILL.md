# 洛克希德·马丁 · Lockheed Martin

---

## 发展历程

| 时期 | 关键事件 |
|:----:|--------|
| **1995 年** | 洛克希德与马丁·玛丽埃塔合并成立 |
| **1943 年** | 前身洛克希德的'臭鼬工厂'（Skunk Works）创立——研发 U-2/SR-71 |
| **2000s** | F-35 项目——全球最大武器采购项目 |
| **2020s** | 太空业务（猎户座飞船/导弹防御系统） |

---

## 核心特点

航空（F-35/F-22/C-130）+ 导弹与火控 + 太空系统 + 旋转任务（直升机）

## 核心优势

F-35 是全球最先进的战斗机——已被 18 个国家采购

|------|------|

| F-35 订单 | 超过 3600 架 |
| 员工数 | 约 122000 人 |

## 值得了解

洛克希德·马丁的'臭鼬工厂'是世界上最神秘的研发机构——U-2 侦察机和 SR-71 黑鸟都诞生于此
