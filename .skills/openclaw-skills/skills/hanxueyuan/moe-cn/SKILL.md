# 中国教育部技能

## 技能描述
中华人民共和国教育部主管全国教育工作。本技能提供教育政策、考试招生、学历认证等信息。

## 核心功能
- 📚 **教育政策**：教育改革、政策文件
- 🎓 **考试招生**：高考、考研、招生政策
- 🏫 **学校信息**：高校名单、学科建设
- 📜 **学历认证**：学历学位、认证服务
- 👨‍🏫 **教师队伍**：教师资格、职称评定
- 🌐 **国际教育**：留学、国际合作

## 使用场景
- 查询教育政策
- 了解考试招生信息
- 学历学位认证
- 查询学校信息

## 注意事项
- 招生政策以当年发布为准
- 学历认证需通过官方渠道
- 各地政策可能有差异
