# 亚特兰大 · Atlanta

> "The City Too Busy to Hate." — 民权运动时代的城市格言

Atlanta 是佐治亚州首府，美国南方的经济、文化和交通中心。作为全球旅客吞吐量最大的机场所在地，亚特兰大是连接全球的航空枢纽。这里是可口可乐、CNN、达美航空、家得宝等 Fortune 500 企业的总部，也是民权运动的历史圣地。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1837** | 作为 Western & Atlantic 铁路终点站建立，原名 Terminus |
| **1847** | 正式更名为 Atlanta（取自 Atlantic 的阴性形式） |
| **1864** | 南北战争中被谢尔曼将军焚毁，战后重建 |
| **1886** | John Pemberton 在亚特兰大发明可口可乐 |
| **1940s-50s** | 民权运动中心，马丁·路德·金在此领导非暴力抗议 |
| **1980** | CNN 开播，成为全球第一个 24 小时新闻频道 |
| **1990** | 获得 1996 年夏季奥运会主办权 |
| **1996** | 举办百年奥运会，城市基础设施大幅升级 |
| **2006** | 哈茨菲尔德机场年旅客量突破 1 亿人次，全球最繁忙 |
| **2010s** | 影视产业爆发（"南方好莱坞"），Marvel 等大量作品在此拍摄 |
| **2020s** | 科技企业大量迁入，经济多元化加速 |
| **2024** | GDP 超 4,000 亿美元，影视产业年超 40 亿美元 |

---

## 业务结构分析

### 航空与物流

哈茨菲尔德-杰克逊机场连续 25 年全球旅客量第一（约 1 亿人次/年）。UPS 的全球航空枢纽使亚特兰大成为北美物流核心节点。

### 媒体与影视

CNN 总部所在地。凭借 Georgia 州 30% 的影视税收优惠，Marvel、Netflix 等大量作品在此拍摄，被称为"南方好莱坞"。

### 企业总部

拥有 18 家 Fortune 500 企业总部（可口可乐、达美、家得宝、UPS 等），总部经济是美国城市中最密集的之一。

### 科技与创业

南方最大的科技创业中心之一，拥有 Georgia Tech 等人才来源。金融科技（PayPal、NCR 总部）和网络安全是重点领域。

## 值得了解

- 可口可乐在亚特兰大被发明时，发明者最初是把它当作止痛药水销售的

- 亚特兰大的 MARTA 地铁系统是世界上少数几个在车站里展示公共艺术的城市地铁

- 马丁·路德·金的出生地、教堂和墓地都在亚特兰大 Auburn Avenue"甜蜜奥本"历史区

- 亚特兰大在 1996 年奥运会期间举办了 271 个比赛项目，是当时规模最大的夏季奥运会之一

- 亚特兰大是" Gone with the Wind"（乱世佳人）的故事背景地

- 哈茨菲尔德机场每天处理超过 2,700 架次航班，相当于每 30 秒一架
