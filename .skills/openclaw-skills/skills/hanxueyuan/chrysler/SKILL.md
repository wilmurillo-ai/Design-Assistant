# 克莱斯勒 · Chrysler (Stellantis)

> "Imported from Detroit." — Eminem 超级碗广告语

Chrysler 由 Walter P. Chrysler 于 1925 年创立，曾是美国三大汽车制造商之一。公司历史上多次经历合并与重组——1998 年与戴姆勒合并、2009 年被菲亚特收购、2021 年成为 Stellantis 的一部分。如今 Chrysler 品牌主要聚焦 Pacifica 小型货车和少数车型。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1925** | Walter P. Chrysler 在底特律创立 Chrysler Corporation |
| **1930s** | Chrysler Airflow 革命性设计，引入空气动力学概念 |
| **1950s-60s** | V8 Hemi 发动机成为美国肌肉车的标志 |
| **1984** | 推出 Plymouth Voyager 和 Dodge Caravan，开创小型货车品类 |
| **1998** | 与戴姆勒-奔驰合并为 DaimlerChrysler，轰动全球 |
| **2007** | 戴姆勒分拆 Chrysler，出售给 Cerberus Capital |
| **2009** | 破产重组，菲亚特获得 20% 股份后逐步控股 |
| **2014** | 菲亚特-克莱斯勒（FCA）正式成立 |
| **2021** | Stellantis 成立（FCA 与 PSA 合并），Chrysler 成为旗下品牌 |
| **2024** | 品牌聚焦 Pacifica，电动化转型计划中 |

## 业务结构分析

### Pacifica 小型货车

Chrysler 目前的核心产品，是北美最畅销的小型货车之一。插电混动版 Pacifica Hybrid 是该品类唯一的新能源选项。

### 历史品牌遗产

Chrysler 历史上拥有 Dodge、Jeep、Ram、Plymouth 等子品牌，如今这些都已独立运营或停产，但 Chrysler 品牌的 Hemi 发动机遗产仍在延续。

### 电动化转型

Stellantis 计划将 Chrysler 品牌电动化，概念车 Airflow 预示了未来方向。预计 2025-2026 年推出纯电 SUV 和跨界车。

### 品牌定位

在 Stellantis 品牌矩阵中，Chrysler 定位高于 Dodge 但低于 Alfa Romeo，面向追求舒适和豪华的美国家庭用户。

## 值得了解

- Walter P. Chrysler 在创立公司之前是通用汽车的高管，因为与高层意见不合而离开

- Chrysler 在 1998 年与戴姆勒合并时被称为"世纪合并"，但最终被认为是失败的并购

- Eminem 的 "Imported from Detroit" 超级碗广告（2011）让 Chrysler 获得了巨大的品牌关注

- Chrysler Building（纽约克莱斯勒大厦）是装饰艺术风格的杰作，但公司与大厦没有直接关系

- Hemi 发动机的半球形燃烧室设计提供了更大的动力输出，是 NASCAR 早期胜利的关键

- Chrysler 曾拥有兰博基尼（1987-1994），但后来出售给了大众集团
