# 桑坦德银行 · Banco Santander, S.A.

> "Good is the enemy of great. We are never satisfied."

Banco Santander 是西班牙最大的银行集团，也是欧元区最具国际特色的银行之一。业务覆盖欧洲、南美和北美 10 个核心市场，拥有超过 1.5 亿客户。 Santander 在巴西、英国、墨西哥和西班牙等市场占据领先地位，是全球系统重要性银行之一。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1857** | 在西班牙桑坦德创立，最初为小商业银行 |
| **1909** | 开始在西班牙全国扩张 |
| **1950s** | 进入拉丁美洲市场（墨西哥、阿根廷、巴西） |
| **1980s** | 欧洲扩张，在英国和德国建立业务 |
| **1999** | 收购 Abbey National（英国第四大银行） |
| **2004** | 收购 Sovereign Bancorp（美国东北部银行） |
| **2007** | 与 RBS、苏格兰银行竞购 ABN AMRO |
| **2008** | 收购 Bradford & Bingley 的储蓄业务 |
| **2010** | 重组为四大业务区域 |
| **2014** | Santander UK 成为英国最大的抵押贷款银行 |
| **2017** | Ana Botin 就任执行主席，推动数字化转型 |
| **2024** | 营收约 620 亿欧元，客户超 1.5 亿 |

## 业务结构分析

### 零售银行

在西班牙、英国、巴西、墨西哥等 10 个市场的个人银行服务

### 企业银行

面向中小企业和大型企业的贷款、贸易融资和现金管理服务

### 财富管理

私人银行、投资管理和保险服务

### 数字银行

Openbank 数字银行和 Superdigital 等金融科技投资

## 值得了解

- Santander 是全球最大的西班牙语银行，业务遍及西班牙语和葡萄牙语国家

- Ana Botin 是欧洲最大的银行之一的执行主席，被誉为银行业最具影响力的女性之一

- Santander 在巴西的分支机构数量超过 3,000 家，是巴西最大的私营银行之一

- Santander 的 Openbank 是欧洲最早的纯数字银行之一

- Santander Arena 在美国宾夕法尼亚州是知名的娱乐场馆

- Santander 是最早投资 Fintech 的传统银行之一，投资了超过 30 家金融科技公司

- Santander UK 赞助了伦敦的 Santander Cycles 共享单车项目

- Santander 的 Superdigital 是巴西领先的数字钱包之一

- Santander 在 2020 年推出了全球首个基于区块链的贸易融资平台

- Santander 的 Formula 1 赞助使其成为全球最知名的银行品牌之一
