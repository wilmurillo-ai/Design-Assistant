# 3M Company

> "Science. Applied to Life." —— 3M

3M是全球最具创新力的多元化工业公司之一，1902年在美国明尼苏达州创立。从采矿企业起家，经过120多年发展，拥有60,000+种产品，覆盖医疗健康、电子材料、工业制造、消费品四大领域。便利贴、报事贴、N95口罩、透明胶带、光学膜……3M的产品几乎存在于每个人的日常生活中。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1902** | 五位创业者在明尼苏达州创立明尼苏达矿业制造公司（Minnesota Mining & Mfg） |
| **1925** | 收购Okashi公司，进入胶带行业 |
| **1930** | 发明 masking tape（美纹纸胶带），3M标志产品 |
| **1968** | 发明便利贴（Post-it），由 Spencer Silver的"失败"胶水意外诞生 |
| **1980s** | 扩展至医疗健康领域，收购多个医疗产品公司 |
| **2000s** | 六西格玛质量管理全面推行，效率大幅提升 |
| **2020** | N95口罩成为抗疫核心物资，3M产能全开 |
| **2024** | 剥离消费者业务（含Post-it），聚焦工业和医疗健康 |

---

## 业务结构分析

### 安全与工业
个人防护设备（PPE）、研磨材料、胶带、密封剂。N95口罩和工业安全产品全球领先。

### 医疗健康
医用胶带、伤口护理、牙科产品、健康信息管理系统。

### 电子与能源
光学膜（LCD/OLED显示屏核心材料）、电子胶带、绝缘材料。全球每部智能手机都含有3M材料。

### 消费（已剥离）
Post-it便利贴、Command挂钩、Scotch胶带等品牌。2024年已分拆为独立公司SOLV。

## 🔑 核心护城河

```
┌─────────────────────────────────────────┐
│        材料科学 + 创新文化                │
│                                         │
|   基础研究 ─→ 材料平台 ─→ 多领域应用     │
|       ↑                        ↓        │
|   专利壁垒 ←── 持续创新 ←── 15%规则      │
|                                         │
|   + 119,000+ 专利，全球领先              │
|   + 15%规则：员工可用15%时间做自主研究   │
|   + 60,000+ 产品：最多元化的工业公司     │
└─────────────────────────────────────────┘
```

1. **材料科学平台**：胶粘剂、研磨、过滤、反射等核心技术可跨领域应用
2. **创新文化**："15%规则"允许员工用15%工作时间做自主研究，便利贴就是产物
3. **专利护城河**：119,000+项专利，覆盖几乎所有工业领域
4. **多元化风险分散**：60,000+产品使得任何单一市场波动对公司影响有限

## 值得了解

- 3M的名字来源于"Minnesota Mining and Manufacturing"——三个M的缩写
- 便利贴（Post-it）的发明源自一次"失败"——Spencer Silver本想发明强力胶，结果做出了弱粘性胶
- 3M每年有约30%的营收来自过去4年内开发的新产品——这在百年老企中极为罕见
- 全球每部智能手机的屏幕都含有3M的光学增亮膜——你可能看不见它，但没有它屏幕不会亮
