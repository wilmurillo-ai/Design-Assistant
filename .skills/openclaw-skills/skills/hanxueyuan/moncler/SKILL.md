# Flipkart · 沃尔玛子公司

> "Empowering every Indian's dream" —— Flipkart

Flipkart是印度最大的本土电商平台，2007年由Sachin Bansal和Binny Bansal（无血缘关系）在班加罗尔创立。2018年被沃尔玛以160亿美元收购大部分股权。Flipkart在印度电商市场与亚马逊印度展开激烈竞争，通过本地化策略（现金支付、大促销日、本地语言支持）占据印度约40%的电商市场份额。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **2007** | 两位前亚马逊员工Sachin和Binny Bansal在班加罗尔创立Flipkart，最初只卖书 |
| **2010** | 扩展至电子产品和服装，推出"货到付款"适应印度支付习惯 |
| **2012** | 推出Flipkart App，移动优先策略 |
| **2014** | Big Billion Day大促诞生，对标亚马逊Prime Day |
| **2017** | 收购Myntra（时尚电商）和PhonePe（数字支付） |
| **2018** | 沃尔玛以160亿美元收购77%股权，是印度科技史上最大收购 |
| **2021** | PhonePe分拆独立运营，成为印度最大数字支付平台之一 |
| **2024** | 印度电商市场竞争白热化，Flipkart与Amazon India争夺第一 |

---

## 业务结构分析

### 电商市场
印度第二大电商（仅次于Amazon India），但通过Big Billion Day和独家品牌合作保持强势。GMV超200亿美元。

### Myntra（时尚）
印度最大的在线时尚平台，占据印度在线时尚市场超35%份额。

### PhonePe（数字支付）
从Flipkart分拆的支付平台，UPI交易量印度第一，超300月活用户。

### 物流
eKart自建物流网络，覆盖印度20,000+邮编区域，包括农村和偏远地区。

## 值得了解

- 两位创始人Sachin和Binny Bansal没有血缘关系，只是同姓——这在全球创业史上都是罕见的巧合
- Flipkart最初只卖书，因为两位创始人曾在亚马逊工作，深知从垂直品类切入的策略
- "货到付款"是Flipkart最大的创新——在印度只有不到10%的人有信用卡的时代，这是唯一可行的电商模式
- 2018年沃尔玛的160亿美元收购是印度科技史上最大的交易，超过软银对Flipkart之前所有投资的总和
