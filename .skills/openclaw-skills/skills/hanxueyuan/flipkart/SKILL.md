# Flipkart · Walmart India

> "Empowering every Indian's dream" —— Flipkart

Flipkart是印度最大的本土电商平台，2007年由Sachin和Binny Bansal（无血缘关系）在班加罗尔创立。2018年被沃尔玛以160亿美元收购。Flipkart通过本地化策略（货到付款、Big Billion Day大促、本地语言支持）占据印度约40%的电商市场份额，与亚马逊印度展开激烈竞争。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **2007** | 两位前亚马逊员工在班加罗尔创立Flipkart，最初只卖书 |
| **2010** | 扩展至电子和服装，推出"货到付款"适应印度支付习惯 |
| **2012** | 推出移动端App，移动优先策略 |
| **2014** | Big Billion Day大促诞生，对标亚马逊Prime Day |
| **2017** | 收购Myntra（时尚电商）和PhonePe（数字支付） |
| **2018** | 沃尔玛以160亿美元收购77%股权，印度科技史最大收购 |
| **2021** | PhonePe分拆独立运营，成为印度最大UPI支付平台 |
| **2024** | 与Amazon India争夺印度电商第一，IPO预期升温 |

---

## 业务结构分析

### 电商市场
印度最大本土电商，GMV超200亿美元，通过Big Billion Day和独家品牌合作保持竞争力。

### Myntra（时尚）
印度最大在线时尚平台，占在线时尚市场超35%份额。

### PhonePe（数字支付）
从Flipkart分拆，UPI交易量印度第一，月活用户超3亿。

### 物流（eKart）
自建物流网络，覆盖20,000+邮编区域，包括农村偏远地区。
