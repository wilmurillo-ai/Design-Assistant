# 🐴 Longchamp · 珑骧

> *"The Art of Travel"* —— 将旅行的艺术融入每一针每一线

## 关于品牌

Longchamp（珑骧）创立于 1948 年的巴黎，是法国最具代表性的皮具品牌之一。品牌以其标志性的 **Le Pliage 折叠尼龙包** 闻名全球，年销量超过数百万只，成为全球旅行者的经典选择。Longchamp 坚持家族独立经营，历经四代传承，将法式优雅与实用主义完美融合，产品涵盖手袋、小皮具、鞋履、成衣及旅行配件。

## 📜 品牌编年史

| 年份 | 里程碑事件 |
|------|-----------|
| **1948** | Jean Cassegrain 在巴黎创立 Longchamp，最初以烟草斗和烟斗皮套起家 |
| **1970s** | 第二代掌门人 Philippe Cassegrain 将业务扩展至手袋和旅行皮具领域 |
| **1993** | 传奇产品 **Le Pliage 折叠包** 诞生，由尼龙和皮革制成，可折叠收纳，迅速风靡全球 |
| **2000s** | 第三代传人 Jean Cassegrain 执掌，加速全球化扩张，进驻中国市场 |
| **2018** | 收购瑞士奢侈腕表品牌 **Eberhard & Co.**（后于 2021 年出售） |
| **2021** | 推出 **Longchamp Paris** 全新品牌定位，聚焦高级皮具和时装系列 |
| **2023** | 宣布在法国旺代省（Vendée）开设全新现代化工厂，投资超 1,000 万欧元 |

## 👜 经典产品矩阵

### Le Pliage 折叠系列
品牌的灵魂产品。采用轻质尼龙面料搭配意大利小牛皮手柄，可折叠成手掌大小，自重仅 200g 左右。自 1993 年问世以来已售出数千万只，有数十种颜色可选。

### Le Foulonné 压纹牛皮系列
采用全粒面小牛皮，表面有独特的压纹处理。代表了 Longchamp 在高端皮具领域的工艺水平，适合商务和正式场合。

### Roseau 竹节扣系列
灵感来源于竹节造型的金属扣饰，是 Longchamp 高级皮具线的代表作之一。以精湛的结构设计和优质皮革著称，定价定位品牌高端线。

### La Boîte
2023 年推出的高级手袋系列，以品牌档案馆中的经典箱形设计为灵感，强调法式手工艺的极致追求。

## 🔧 工艺与传承

Longchamp 的核心竞争力在于**家族传承的皮具工艺**与**现代工业的完美结合**。

- **生产布局**：在法国拥有多家工厂（Mazé、Châteaurenard、Hénin-Beaumont），同时在美国新罕布什尔州设有皮具工坊
- **Le Pliage 产地**：绝大多数 Le Pliage 在突尼斯和摩洛哥生产，法国本土工厂专注于高端皮具
- **皮革处理**：与意大利顶级鞣革厂合作，确保皮革的质感和耐用性
- **家族管理**：Cassegrain 家族四代经营，保持品牌独立性和决策灵活性
