# 尼曼百货 · Neiman Marcus

> "给顾客他们想要的，哪怕他们还不知道自己想要什么。"

尼曼百货（Neiman Marcus）是美国最负盛名的奢侈品百货连锁，由 Herbert Marcus、Abraham Lincoln Neiman 和 Carrie Marcus Neiman 于 1907 年在达拉斯创立。公司以高端时尚品牌、个性化服务和传奇的圣诞愿望目录闻名。2020 年经历破产重组后已恢复运营，目前拥有约 36 家全尺寸门店和约 32 家 Last Call 折扣店。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1907** | 在达拉斯创立，三位创始人是兄妹和合作伙伴 |
| **1920s** | 首创圣诞愿望目录，成为全球最奢侈的节日购物指南 |
| **1950s** | 推出 His and Hers 圣诞礼品，成为美国文化现象 |
| **2005** | 被 TPG 和 Warburg Pincus 以 60 亿美元杠杆收购 |
| **2020** | 新冠疫情冲击下申请破产重组，债务减免约 40 亿美元 |
| **2024** | 恢复运营，聚焦核心奢侈品客户和数字化渠道 |

## 业务结构分析

### 全尺寸门店

约 36 家，覆盖美国主要城市，高端设计师时装和配饰

### Last Call 折扣店

约 32 家，销售过季和折扣奢侈品

### 在线销售

NeimanMarcus.com 是全美最大的奢侈品电商平台之一

### 圣诞愿望目录

传奇的 Christmas Book，包含天价礼品

## 🔑 核心护城河

```
顶级品牌独家 → 高净值客户 → 个性化服务 → 品牌忠诚度 → 独家合作 → 更多顶级品牌
```

1. 100+ 年的奢侈品零售品牌信任，在高端消费者心中无可替代
2. 圣诞愿望目录是零售史上最具话题性的营销活动之一
3. 个人购物顾问服务是高净值客户的核心体验
4. 破产重组后债务大幅减轻，财务结构更健康

## 值得了解

圣诞愿望目录曾推出过活体骆驼、价值千万的私人飞机等天价礼品

尼曼百货最早是一家服装店，但创始人意识到奢侈品百货的潜力后转型

2020 年破产重组后，公司砍掉了部分 Last Call 门店聚焦核心全尺寸店

圣诞愿望目录的发行量约 150 万份，是美国最具话题性的营销材料之一
