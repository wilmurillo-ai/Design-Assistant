# EA · 艺电

---

## 发展历程

| 时期 | 关键事件 |
|:----:|--------|
| **1982 年** | Trip Hawkins 创立 Electronic Arts |
| **1990s** | FIFA 系列发布——后来成为全球最畅销体育游戏 |
| **2000s** | 收购大量工作室——BioWare/DICE/Respawn |
| **2013 年** | 转向'游戏即服务'模式——微收入引发争议 |
| **2020s** | EA Sports FC 取代 FIFA——继续足球游戏市场统治 |

## 核心特点

游戏发行（FIFA/ Madden/NHL/NBA Live）+ 第一方开发（Battlefield/Apex Legends/The Sims）
