# PlayStation · PlayStation（索尼互动娱乐）

PlayStation 是索尼集团（Sony Group）旗下的游戏品牌，由索尼互动娱乐（Sony Interactive Entertainment, SIE）运营。自 1994 年首台 PlayStation 主机在日本发售以来，PlayStation 已成为全球最成功的游戏品牌之一，累计售出超过 6.4 亿台主机，拥有超过 6 亿注册用户。从挑战任天堂和世嘉的后来者，到与微软 Xbox 和任天堂 Switch 三分天下的行业巨头，PlayStation 的游戏帝国历程堪称商业教科书。

---

## 主机世代演进

| 世代 | 主机 | 发售年份 | 全球销量 | 关键特征 |
|------|------|----------|----------|----------|
| **第一代** | PlayStation（PS1） | 1994 | ~1.02 亿台 | CD-ROM 介质，打破卡带垄断 |
| **第二代** | PlayStation 2（PS2） | 2000 | ~1.55 亿台 | **史上最畅销游戏主机**，内置 DVD 播放器 |
| **第三代** | PlayStation 3（PS3） | 2006 | ~8740 万台 | Cell 处理器、Blu-ray、PS Network |
| **第四代** | PlayStation 4（PS4） | 2013 | ~1.17 亿台 | x86 架构、社交功能、独占游戏黄金期 |
| **第五代** | PlayStation 5（PS5） | 2020 | ~7500 万台+ | SSD 超高速加载、光线追踪、DualSense 手柄 |

## 商业战略演变

```
PlayStation 商业演进
─────────────────────────────────
1994–2000 │ 硬件颠覆者
          │ CD 取代卡带，降低游戏开发门槛
          │
2000–2010 │ 生态霸主
          │ PS2 统一市场，DVD 功能成杀手锏
          │
2013–2020 │ 独占内容为王
          │ PS4 以强大独占游戏压制 Xbox One
          │ 第一方工作室大规模收购
          │
2020–至今 │ 多元化平台
          │ PS5 硬件 + PC 移植 + 影视改编
          │ PlayStation Studios 品牌独立运营
```

---

## 关键商业决策

- **CD-ROM 战略（1994）**：放弃卡带改用光盘，大幅降低游戏成本
- **PS2 兼容 DVD（2000）**：将游戏主机变成家庭娱乐中心
- **收购第一方工作室**：2015 年后收购 Insomniac、Bluepoint、Haven 等
- **PS Plus 订阅**：对标 Xbox Game Pass 的订阅服务
- **PC 移植战略**：2022 年起将独占游戏移植到 PC 平台
- **影视化**：《最后生还者》HBO 剧集、《神秘海域》电影
