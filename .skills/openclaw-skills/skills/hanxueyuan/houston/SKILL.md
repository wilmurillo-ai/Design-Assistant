# 休斯顿 · Houston

> "Space City." — 航天之城

Houston 位于德克萨斯州东南部，是美国第四大城市（按都会区人口计超 700 万）。作为全球"能源之都"，它是埃克森美孚、雪佛龙、康菲等能源巨头的总部所在地。2024 年 GDP 超 5,000 亿美元，医疗中心是全球最大的医疗研究综合体。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1836** | Sam Houston 将军命名城市，Allen 兄弟购买土地并规划城市发展 |
| **1901** | Spindletop 油田发现，开启德克萨斯石油繁荣 |
| **1914** | 休斯顿航道开通，使城市成为深水港 |
| **1961** | NASA 载人航天中心选址休斯顿 |
| **1969** | "休斯顿，这里是静海基地"——阿波罗 11 号登月 |
| **1970s** | 能源产业爆发，大量石油公司总部迁入 |
| **1980s** | 石油价格崩盘导致经济衰退，推动产业多元化 |
| **2000s** | 德州医学中心崛起，成为全球最大的医疗研究集群 |
| **2017** | 哈维飓风造成约 1,250 亿美元损失，推动防洪重建 |
| **2020s** | 能源转型加速，氢能和碳捕获投资激增 |
| **2023** | NBA 火箭队新球馆启用，城市体育设施升级 |
| **2024** | 氢能 Hub 获得联邦 12 亿美元资助，清洁能源转型加速 |

## 业务结构分析

### 能源产业

休斯顿是全球能源之都，拥有 5,000+ 家能源相关企业。能源行业就业超 20 万人，从勘探、炼化到贸易和金融服务全链条覆盖。

### 航空航天

NASA 约翰逊航天中心是载人航天任务的控制中心，也是国际空间站的地面指挥中枢。SpaceX 和 Blue Origin 在此设有设施。

### 医疗健康

德州医学中心（TMC）是全球最大的医疗综合体，拥有 100,000+ 名员工和 92 个成员机构。在癌症治疗和器官移植方面全球领先。

### 港口与物流

休斯顿港是美国国际贸易量第一的港口，处理超过 3.2 亿吨货物/年，是石化产品出口的关键通道。

```
       │              │
       ▼              ▼

```

1. 能源集群效应：全球能源公司在此形成完整的产业生态——从勘探、炼化到贸易和金融服务
2. 德州医学中心垄断：TMC 的规模和多样性全球独一无二，在特定医疗领域形成技术壁垒
3. 航天基础设施：NASA 约翰逊中心数十年积累的航天工程人才和设施，其他城市无法短期建立
4. 无州所得税优势：德克萨斯州不征收个人和州企业所得税，持续吸引企业和人才迁入

## 值得了解

- 休斯顿是美国种族最多元化的大城市之一——约 25% 拉美裔，25% 非裔，7% 亚裔

- "休斯顿，我们有麻烦了"实际上是 Jack Swigert 说的，原话是 "Okay, Houston, we have had a problem here"

- 休斯顿没有城市规划分区法（zoning laws），是美国最大的没有分区法的主要城市

- 休斯顿港在 2020 年取代纽约港，成为美国国际贸易量最大的港口

- 德州医学中心拥有全球最大的儿童癌症治疗中心，每年治疗超过 15,000 名患者

- 休斯顿的 NASA 控制中心是所有载人航天任务的"大脑"，阿波罗 13 号的救援指挥就是在这里完成的
