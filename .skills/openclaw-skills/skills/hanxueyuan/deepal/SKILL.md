# 深蓝汽车 (Deepal)

长安汽车旗下的新能源品牌，专注于智能电动汽车。

## 品牌介绍

**深蓝汽车**是长安汽车于 2022 年推出的新能源品牌，定位年轻化和智能化。

### 基本信息

| 项目 | 详情 |
|------|------|
| **成立时间** | 2022 年 |
| **所属公司** | 长安汽车 |
| **总部** | 中国重庆 |
| **品牌定位** | 智能电动，年轻科技 |

### 热门车型

| 车型 | 类型 | 续航 | 价格区间 |
|------|------|------|----------|
| **SL03** | 轿车 | 500-700km | 15-20 万 |
| **S7** | SUV | 500-650km | 15-20 万 |
| **G318** | 越野 SUV | 400-500km | 20-25 万 |

### 技术特点

- **超级增程** - 可油可电，无里程焦虑
- **智能座舱** - 高通 8155 芯片
- **自动驾驶** - L2+ 级辅助驾驶
- **快速充电** - 3C 超充技术
