# 苏黎世联邦理工学院 · ETH Zurich

> "想象力比知识更重要。" — Albert Einstein

苏黎世联邦理工学院（ETH Zurich）是欧洲大陆最负盛名的理工科大学，在全球大学排名中常年位居前 10。该校培养了 32 位诺贝尔奖得主，包括阿尔伯特爱因斯坦。学校在工程、计算机科学、自然科学和建筑领域享有世界级声誉。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1855** | 由瑞士联邦创立，最初名为联邦理工学院，旨在培养工程技术人才 |
| **1905** | 校友爱因斯坦发表狭义相对论（他 1900 年毕业于 ETH） |
| **1950s** | 计算机科学先驱在 ETH 推动计算机科学发展 |
| **2012** | 在 QS 世界大学排名中跻身全球前 10 |
| **2021** | DeepMind 联合创始人 Demis Hassabis 获 ETH 荣誉博士学位 |
| **2024** | QS 排名全球第 7，连续多年欧洲大陆第一 |

---

## 业务结构分析

### 工程科学

机械、电气、土木、材料工程全球顶尖，与工业界深度合作

### 自然科学与数学

物理、化学、数学基础研究实力雄厚，诺贝尔奖产出密集

### 计算机科学

AI、机器人学、网络安全领域欧洲领先，与 Google Zurich 紧密合作

### 建筑与环境科学

建筑系全球排名前五，可持续发展和城市规划研究领先

## 🔑 核心护城河

```
顶尖学者 → 科研突破 → 诺贝尔奖声望 → 吸引全球天才 → 更多突破
```

1. 32 位诺贝尔奖得主的学术传承，形成强大的学术品牌效应
2. 瑞士的高生活质量和科研投入（瑞士 R&D 支出占 GDP 约 3.4%）
3. 与 Google、IBM、Roche 等企业的紧密合作关系，科研转化能力强
4. 英语授课的国际化环境吸引全球顶尖学生（国际学生占比约 40%）

## 值得了解

爱因斯坦 1900 年毕业时并没有立刻获得教职，而是去专利局当了审查员

ETH 与苏黎世大学（UZH）共享校园和部分设施，两校学生可以交叉选课

ETH 的机器人实验室是欧洲最顶尖的之一，曾开发出能完成复杂操作的仿生手

ETH 的学生必须通过极其严格的入学考试，录取率约 25%
