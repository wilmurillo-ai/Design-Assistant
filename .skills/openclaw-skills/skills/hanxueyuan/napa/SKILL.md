# NAPA Auto Parts

> "The Knowledge. The Parts. The People." —— NAPA

NAPA（National Automotive Parts Association）是美国最大的汽车零部件零售连锁，1925年创立。NAPA为专业汽车维修技师和DIY车主提供超过50万种汽车零部件和配件，通过超6,000家门店（美国+加拿大+墨西哥）覆盖北美市场。NAPA是汽车后市场（Aftermarket）的标杆企业。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1925年** | 20家汽车配件经销商联合创立NAPA，共享采购和分销网络 |
| **1950s** | 随着美国汽车保有量激增，NAPA快速扩张至全国 |
| **1970s** | 推出DIY零售概念，普通车主也能在NAPA购买配件 |
| **1990s** | NAPA AutoCare Centers认证维修网络建立 |
| **2000s** | 电商转型，在线订购+到店取货模式成型 |
| **2010s** | 电动和混合动力车配件业务起步 |
| **2020s** | 加速数字化工具和移动应用 |
| **2024年** | 超6,000家门店，母公司Genuine Parts Company年营收超250亿美元 |

---

## 业务结构分析

### 零售门店
超6,000家门店，分为专业技师店（B2B）和DIY零售店（B2C）。每个门店平均库存约10,000个SKU。

### 批发与分销
NAPA Distribution Centers为门店提供次日补货，覆盖超50万种SKU。

### NAPA AutoCare
认证维修网络，超20,000家维修店使用NAPA配件和认证标志。

### 线上
NAPA.com支持在线查询、订购和到店取货。

## 🔑 核心护城河

```
┌─────────────────────────────────────────┐
│        分销网络 + 品牌信任                 │
│                                         │
|   门店密度 ─→ 即时供应 ─→ 技师首选       │
|       ↑                        ↓        │
|   采购规模 ←── SKU覆盖 ←── 供应商关系    │
|                                         │
|   + 6,000+ 门店：北美最密集的汽配网络    │
|   + 50万+ SKU：最完整的汽车配件覆盖       │
|   + AutoCare认证：20,000家维修店生态    │
└─────────────────────────────────────────┘
```

1. **门店密度**：北美每10英里内至少有一家NAPA，维修技师可以即时获取配件
2. **SKU覆盖**：50万+SKU，几乎覆盖所有车型和配件需求
3. **B2B关系**：与独立维修技师的长期合作关系是AutoZone和O'Reilly难以撼动的
4. **品牌信任**：近100年历史，NAPA是"汽车配件"的代名词

## 值得了解

- NAPA的LOGO是一个蓝色六边形——这是美国汽配行业最知名的标志之一
- NAPA是 NASCAR 的长期赞助商——NAPA Racing是赛事中最著名的车队之一
- "NAPA Know How"是NAPA最著名的营销口号，强调专业汽车配件知识
- NAPA的母公司Genuine Parts Company是标普500成分股和道琼斯运输指数成分股
