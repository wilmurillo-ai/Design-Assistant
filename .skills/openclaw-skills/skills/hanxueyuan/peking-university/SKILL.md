# 北京大学 · Peking University

---

## 发展历程

| 时期 | 关键事件 |
|:----:|--------|
| **1898 年** | 光绪帝批准设立 **京师大学堂**——中国近代第一所国立综合性大学 |
| **1912 年** | 更名为北京大学 |
| **1917 年** | 蔡元培出任校长——'思想自由，兼容并包'的办学理念 |
| **1919 年** | **五四运动** 从北大开始——中国现代史的转折点 |
| **1952 年** | 院系调整——成为以文理基础学科为主的综合性大学 |
| **2020s** | QS 世界大学排名前 20——中国大学第一梯队 |

---

## 核心特点

本科 + 研究生（文理/医学/法学/商学/工程）+ 研究

## 核心优势

中国最顶尖的高等学府 + 五四运动的精神遗产 + 强大的校友网络（政商学界）
