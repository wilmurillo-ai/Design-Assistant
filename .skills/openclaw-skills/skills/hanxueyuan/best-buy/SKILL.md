# 百思买 · Best Buy Co. Inc.

> "我们不是在卖产品，我们是在丰富生活。"

百思买是美国最大的消费电子产品零售商，经营约 1,000 家门店和强大的电商平台。在 2010 年代初一度被 Amazon 逼入绝境（Showrooming 现象），但在 Hubert Joly 领导下成功转型。2024 财年营收约 434 亿美元，被誉为实体零售反击战的经典案例。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1966** | Richard Schulze 在明尼苏达州创立 Sound of Music 音响店 |
| **1983** | 品牌更名为 Best Buy，首创超级卖场概念 |
| **2001** | Geek Squad 技术服务品牌收购，成为差异化服务 |
| **2012** | 被称作 Amazon 的 Showroom，股价跌至 12 美元 |
| **2012** | Hubert Joly 出任 CEO，启动 Renew Blue 转型计划 |
| **2019** | Joly 卸任，Corie Barry 成为新任 CEO |
| **2024** | 年营收 434 亿美元，Geek Squad 和全渠道服务成为增长引擎 |

---

## 业务结构分析

### 消费电子产品

电视、电脑、手机、游戏机等核心品类，贡献约 60% 收入

### 家电

大型家电安装和配送服务，差异化优势明显

### Geek Squad

技术支持和安装服务，约 10% 收入，高利润率

### 会员与健康

Totaltech 会员计划、GreatCall 老年人移动和监护服务

---

## 🔑 核心护城河

```
门店体验+Geek Squad服务 → 顾客信任 → 复购和会员 → 服务收入 → 更高利润率
```

1. Geek Squad 技术服务是 Amazon 难以复制的差异化优势
2. 大家电配送安装需要本地化运营，电商巨头难以替代
3. 价格匹配政策消除了 Showrooming 的动机
4. 门店转型为体验和服务中心而非纯销售点

## 值得了解

Hubert Joly 的《The Heart of Business》被列为商学院必读案例

Best Buy 的 Showrooming 危机被称为零售业最经典的 Amazon 杀手故事

Geek Squad 的制服是复古白衬衫和黑领带，灵感来自 1950 年代极客文化

Best Buy 疫情期间逆势增长了 20%，因为大家需要在家办公的设备
