# 哥本哈根 · Copenhagen

> 幸福是一种设计。

哥本哈根是丹麦王国首都和最大城市，位于西兰岛东部，与瑞典马尔默隔厄勒海峡相望。大都会区人口约140万。哥本哈根常年位居全球最宜居城市前三名，以自行车文化（超50%通勤者骑自行车）、可持续发展和高福利社会闻名。城市也是丹麦经济引擎，Novo Nordisk（诺和诺德）和乐高集团等全球知名企业的总部所在地。2025年被指定为欧洲绿色首都。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1167年** | 阿布萨隆大主教在现哥本哈根位置建立渔村和城堡 |
| **1254年** | 哥本哈根获得城市宪章 |
| **1443年** | 大火后重建，开始发展为北欧重要商业中心 |
| **1660年** | 成为丹麦-挪威王国的绝对君主制首都 |
| **1807年** | 英国海军炮击哥本哈根，城市严重受损 |
| **1850年代** | 城墙拆除，城市开始大规模扩张 |
| **1947年** | 手指规划（Finger Plan）确定城市沿铁路线扩展 |
| **2000年** | 厄勒海峡大桥通车，连接瑞典马尔默 |
| **2009年** | 主办联合国气候变化大会（COP15） |
| **2024年** | Novo Nordisk市值推动哥本哈根成为欧洲重要金融中心 |

## 业务结构分析

### 生命科学

哥本哈根是全球生命科学产业的重要集群，Novo Nordisk（胰岛素和GLP-1药物）和Genmab（抗体药物）等制药巨头总部在此，贡献丹麦大量GDP。

### 清洁能源与可持续

哥本哈根是全球绿色转型的标杆城市，拥有Vestas（风力涡轮机）和Orsted（海上风电）等清洁能源巨头总部，目标是2025年成为全球首个碳中和首都。

### 创意与设计

丹麦设计全球闻名，哥本哈根是乐高集团、Bang & Olufsen等品牌的精神家园。哥本哈根时装周和三年展提升了城市的创意影响力。

### 航运与物流

马士基（Maersk）全球最大航运集团的总部设在哥本哈根，哥本哈根港是波罗的海地区最重要的货运和客运港口。

## 🔑 核心护城河

```
  绿色基础设施 → 可持续品牌 → 国际吸引力
       ↓                        ↗
  生命科学集群 → 高附加值就业 → 经济繁荣
       ↓
  宜居城市声誉 → 人才流入 → 创新循环
```

1. 可持续发展标杆：哥本哈根是全球城市可持续发展的标杆，碳中和目标和自行车文化使其在环保政策制定中具有全球影响力
2. 生命科学产业集群：Novo Nordisk等制药巨头的总部效应，吸引了全球顶尖生物医药人才和研究机构入驻
3. 生活质量护城河：全球顶级宜居城市的声誉吸引跨国企业和高技能人才，形成良性循环
4. 北欧一体化优势：厄勒海峡大桥连接瑞典，哥本哈根-马尔默区域是北欧最大的跨国都市区，市场辐射范围扩大

## 值得了解

- 哥本哈根超过50%的市民每天骑自行车上班，城市拥有超过400公里的专用自行车道，是全球自行车友好城市之首

- Novo Nordisk的市值一度超过整个丹麦的GDP，其GLP-1减肥药Ozempic和Wegovy的火爆使公司成为欧洲市值最高的企业

- 哥本哈根的美人鱼雕像是丹麦最知名的标志，但实际尺寸远小于预期——仅1.25米高，许多游客第一次看到时都很惊讶

- 乐高集团的总部不在哥本哈根市区而是在比隆（Billund），但哥本哈根的Legoland和乐高之家是城市最受欢迎的景点之一
