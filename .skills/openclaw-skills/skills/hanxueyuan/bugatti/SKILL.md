# bugatti

## 城市概况
布加迪（Bugatti）是法国顶级超级跑车品牌，1909 年由埃托雷·布加迪创立。现为大众集团旗下品牌，以 Chiron、Veyron 等限量超跑闻名，是全球最昂贵的汽车品牌之一。

官方网站：https://www.bugatti.com

- 🏎️ **超级跑车**：Chiron、Veyron\n- 💰 **价格**：百万美元级限量超跑\n- 🏆 **历史**：1909 年创立的百年超跑品牌\n- 🔧 **技术**：W16 发动机、极速纪录
