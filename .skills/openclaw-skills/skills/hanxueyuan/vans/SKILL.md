# 万斯 · Vans

---

## 发展历程

| 时期 | 关键事件 |
|:----:|--------|
| **1966 年** | Paul Van Doren 在加州阿纳海姆创立 |
| **1970s** | 成为滑板社区的标配鞋 |
| **1990s** | Off The Wall 口号流行 |
| **2004 年** | 被 VF Corporation 收购 |
| **2020s** | 街头时尚回潮 |

---

## 核心特点

滑板鞋 + 运动鞋 + 服装 + 配件

## 核心优势

滑板文化的代名词 + 经典款式（Old Skool/Sk8-Hi/Authentic）永不过时
