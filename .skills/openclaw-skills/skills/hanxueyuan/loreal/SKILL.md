# L'Oreal 欧莱雅技能

## 技能描述
L'Oreal（欧莱雅）是法国知名化妆品公司，全球最大的化妆品制造商，旗下拥有多个知名品牌。本技能帮助用户查询欧莱雅产品、品牌信息。

## 核心功能
- 💄 **产品查询**：护肤品、彩妆、护发产品
- 🏪 **门店查询**：全球零售店、专柜
- 📅 **新品发布**：最新产品资讯
- 💰 **价格查询**：产品价格范围
- 🌟 **旗下品牌**：L'Oreal Paris、Lancome、YSL Beauty、Kiehl's 等
- 🔬 **科技研发**：护肤科技、成分研究
