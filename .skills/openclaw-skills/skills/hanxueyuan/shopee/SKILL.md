# Shopee (虾皮购物)

东南亚最大的电商平台。

## 平台介绍

**Shopee**成立于 2015 年，总部位于新加坡，是 Sea Group 旗下的领先电商平台。

### 基本信息

| 项目 | 详情 |
|------|------|
| **成立时间** | 2015 年 |
| **母公司** | Sea Group (冬海集团) |
| **总部** | 新加坡 |
| **覆盖国家** | 10+ |
| **月活用户** | 3 亿 + |
| **上市** | 纽交所 (Sea Group 2017 年) |
