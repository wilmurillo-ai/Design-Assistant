# 万代南梦宫 · Bandai Namco

> "日本娱乐帝国，从玩具到游戏的跨界巨头"

Bandai Namco Holdings 是日本最大的综合性娱乐集团之一，由万代和南梦宫于 2005 年合并而成。公司业务覆盖玩具电子游戏街机游乐园和动漫 IP 运营，拥有龙珠高达吃豆人和铁拳等全球知名 IP。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1950** | 岩山治夫创立万代，最初生产木质玩具 |
| **1955** | 中村雅哉创立中村制作所，后改名南梦宫 |
| **1980** | 南梦宫推出吃豆人 Pac-Man，全球最成功的街机游戏 |
| **1990s** | 万代推出高达模型和龙珠系列玩具 |
| **2005** | 万代与南梦宫合并，成立 Bandai Namco Holdings |
| **2010s** |  Elden Ring 和铁拳等游戏全球畅销 |
| **2022** | Elden Ring 全球销量突破 2,000 万份 |
| **2024** | 龙珠 IP 累计创造超过 300 亿美元收入 |

---

## 业务结构分析

### 玩具和 Hobby

高达模型龙珠手办和各类收藏玩具，全球市场份额领先

### 电子游戏

Elden Ring 铁拳龙珠格斗和吃豆人系列

### 街机游乐

日本最大的街机游戏运营商

### 主题乐园

南梦宫主题乐园和室内游乐设施

## 核心护城河

```
  IP 驱动飞轮
    |
    经典游戏IP --> 全球游戏收入
    |                             |
    动漫IP授权 --> 玩具和衍生品
    |                             |
    主题乐园 --> 实体娱乐体验
    |                             |
    跨媒体联动 --> IP 价值放大
```

1. 龙珠 IP 累计创造超过 300 亿美元，是全球最赚钱的媒体 IP 之一
2. Elden Ring 由宫崎英高和乔治 RR 马丁合作开发，获得年度游戏大奖
3. 吃豆人是电子游戏史上最具辨识度的角色之一
4. 万代的高达模型 Gunpla 年销量超过 1 亿个零件

## 值得了解

- 吃豆人的灵感来自一张缺少一角的披萨，设计者岩谷彻在 1980 年创造了这个游戏
- 由玩具制造商万代和游戏公司南梦宫于2005年合并，是日本娱乐业最大整合案

- 高达系列模型累计销量超过5亿个，被誉为"日本最具商业价值的IP"

- 其街机业务至今在日本保有超过50家大型游戏中心


- 万代南梦宫的 logo 是 2022 年更新的，将红色和黄色合并为统一标识

- Elden Ring 是 FromSoftware 首次与西方作家合作开发的游戏

- 南梦宫最初是在百货公司屋顶经营旋转木马游乐设施的
