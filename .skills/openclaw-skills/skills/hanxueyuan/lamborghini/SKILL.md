# Lamborghini 兰博基尼技能

## 技能描述
Lamborghini（兰博基尼）是意大利超级跑车制造商，以激进设计、V12 引擎、高性能闻名。本技能帮助用户了解兰博基尼产品、历史、购买信息。

## 核心功能
- 🏎️ **车型查询**：Aventador、Huracán、Urus
- 💰 **价格查询**：官方定价、限量款
- 🏪 **经销商**：授权经销商查询
- 🎨 **定制服务**：Ad Personam 定制
- 📚 **品牌历史**：创始人、经典车型
- 🔊 **V12 引擎**：引擎技术、声浪

## 使用场景
- 查询兰博基尼车型信息
- 了解超跑技术
- 获取价格参考
- 了解品牌历史

## 注意事项
- 部分车型限量发售
- 价格较高
- 需通过授权经销商购买
