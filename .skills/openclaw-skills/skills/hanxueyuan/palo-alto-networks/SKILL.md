# 派洛特网络 · Palo Alto Networks

> "Zero Trust is not a product, it is an architecture."

Palo Alto Networks 是全球领先的网络安全公司，提供下一代防火墙、云安全、终端保护和安全运营平台。2024 财年营收突破 77 亿美元，同比增长约 18%，在全球网络安全市场份额位居前列。公司通过 Cortex、Prisma 和 Strata 三大平台构建了完整的安全产品矩阵。

## 业务结构分析

### 下一代防火墙 (NGFW)

Strata 平台提供行业领先的下一代防火墙，集成应用识别、威胁防御和 URL 过滤

### 云安全

Prisma Cloud 提供多云和云原生应用的一体化安全保护，涵盖 CSPM、CWPP 和 CWP

### 安全运营

Cortex XDR/XSIAM 平台利用 AI 和自动化技术实现威胁检测、调查和响应的闭环

### SASE/零信任

Prisma Access 和 GlobalProtect 构建零信任网络访问和 SD-WAN 融合平台
