# MCM · MCM Worldwide

> "Modern Creation München." — 品牌全称

MCM（Modern Creation München）1976 年创立于慕尼黑，最初为旅行者和精英阶层制作高级皮具。品牌标志性的 Cognac Visetos 印花（月桂叶图案）是全球最具辨识度的奢侈品设计之一。2005 年被韩国 Sungjoo Group 收购后，MCM 在亚洲市场尤其是中国市场实现了爆发式增长。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1976** | Michael Cromer 在慕尼黑创立 MCM（Modern Creation München） |
| **1980s** | Cognac Visetos 印花设计诞生，成为品牌标志性元素 |
| **1990s** | 被全球精英和旅行者推崇，与 Louis Vuitton 竞争旅行皮具市场 |
| **1997** | Michael Cromer 去世，品牌一度陷入困境 |
| **2005** | 韩国 Sungjoo Group 收购 MCM，启动亚洲市场战略 |
| **2010s** | 在 K-pop 和亚洲年轻消费者中爆红，品牌年轻化转型成功 |
| **2013** | 聘请新创意总监，设计更加时尚和街头化 |
| **2017** | 全球门店超 600 家，亚洲市场占比超 60% |
| **2020s** | 疫情影响后重新调整产品组合，聚焦核心 Visetos 系列 |
| **2024** | 全球 400+ 家门店，亚洲市场持续主导，品牌估值超 10 亿美元 |

## 业务结构分析

### 皮具与手袋

核心业务，Cognac Visetos 印花的背包、手袋和旅行袋是品牌的标志性产品。定价带 $500-$2,000，在轻奢与顶级奢侈之间。

### 时尚与成衣

成衣系列融合了德国工艺和亚洲街头美学，背包和夹克是最受欢迎的品类。K-pop 明星的频繁穿着是品牌推广的关键。

### 配饰与小皮具

钱包、皮带、太阳镜和手机壳是品牌的入门级产品，价格带 $200-$500，是吸引新消费者的入口。

### 亚洲市场战略

MCM 是少数在亚洲（尤其是中国和韩国）比欧洲更成功的欧洲奢侈品牌。韩国 ownership 帮助品牌精准理解亚洲消费者。

## 值得了解

- MCM 最初是为"Modern Creation München"的缩写，但大多数人并不知道全称的含义

- Cognac Visetos 印花上的月桂叶图案象征着胜利和荣耀——这是古罗马的传统符号

- MCM 在被韩国收购之前几乎破产，2005 年的收购价格不到品牌巅峰时期价值的 10%

- BTS 成员 V（金泰亨）被拍到多次背着 MCM 背包，带动了品牌的全球搜索量激增

- MCM 的每一件产品都有一个独特的序列号铭牌，这是品牌自 1976 年以来的传统

- MCM 在 1990s 曾是欧洲皇室的旅行皮具首选，与 Louis Vuitton 直接竞争
