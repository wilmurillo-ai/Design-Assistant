# IBM (国际商业机器公司)

全球领先的科技和咨询服务公司。

## 公司介绍

**IBM** 成立于 1911 年，总部位于纽约，是全球最大的信息技术公司之一。

### 基本信息

| 项目 | 详情 |
|------|------|
| **成立时间** | 1911 年 6 月 16 日 |
| **总部** | 美国纽约阿蒙克 |
| **员工数** | 28 万+ |
| **昵称** | 蓝色巨人 (Big Blue) |
| **股票代码** | IBM (NYSE) |

### 核心业务

| 业务 | 说明 |
|------|------|
| **云计算** | 混合云解决方案 |
| **人工智能** | Watson AI 平台 |
| **咨询服务** | 企业数字化转型 |
| **软件** | 企业软件解决方案 |
| **基础设施** | 服务器、存储 |

### 技术创新

- 发明硬盘、软盘、ATM
- 开发 SQL 数据库
- 发明条形码
- 量子计算机研发
- 人工智能 Watson
