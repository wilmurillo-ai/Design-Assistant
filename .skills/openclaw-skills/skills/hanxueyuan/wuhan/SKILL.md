# 🇨🇳 武汉 · Wuhan 城市指南

武汉是湖北省省会，长江与汉江交汇处的"九省通衢"。由武昌、汉口、汉阳三镇组成，是中国中部的交通枢纽和科教中心。

## 🏛️ 历史与地标

- **黄鹤楼**："天下江山第一楼"，崔颢和李白诗篇的灵感来源
- **东湖**：中国最大的城中湖，33 平方公里的樱花海
- **湖北省博物馆**：曾侯乙编钟和越王勾践剑的收藏地
- **长江大桥**：1957 年建成的万里长江第一桥

## 🍽️ 美食与餐饮

- **热干面**：武汉早餐的灵魂，芝麻酱 + 碱水面
- **三鲜豆皮**：糯米、肉丁和蛋皮煎制的传统小吃
- **周黑鸭**：武汉特产卤味，甜辣口味的鸭脖和鸭锁骨

## 🎭 文化与体验

- **武汉大学樱花**：每年 3 月，3,000 多棵樱花盛开，全国最美校园之一
- **汉秀剧场**：中国最高水准的驻场演出
- **光谷**：中国最大的光电子产业基地
