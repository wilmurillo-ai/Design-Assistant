# Arm · 安谋

---

## 发展历程

| 时期 | 关键事件 |
|:----:|--------|
| **1990 年** | 从 Acorn Computers 分拆成立——ARM 代表 Advanced RISC Machine |
| **2016 年** | 软银以 320 亿美元收购 |
| **2023 年** | 纳斯达克上市——估值 545 亿美元 |
| **2024 年** | 全球 99% 的智能手机使用 Arm 架构 |

---

## 核心特点

芯片架构授权（不制造芯片，只设计 IP 授权给 Apple/Qualcomm/Samsung 等）

## 核心优势

Arm 架构占据全球移动端 CPU 99% 份额；低功耗设计使其在 IoT 和服务器市场也在扩张
