# 宝格丽 · Bulgari

> "珠宝是情感的载体，每一件都应该讲述一个故事。"

宝格丽（Bulgari）是意大利最著名的奢侈珠宝品牌，由希腊移民 Sotirios Bulgari 于 1884 年在罗马创立。品牌以大胆的彩色宝石设计、古罗马-inspired 元素和 Serpenti（灵蛇）系列闻名于世。2011 年被 LVMH 集团收购后，品牌在全球奢侈品市场的地位进一步巩固。

---

## 历史时间线

| 时期 | 关键事件 |
|:----:|--------|
| **1884** | 希腊银匠 Sotirios Bulgari 在罗马孔多蒂街开设第一家银器店 |
| **1905** | 孔多蒂街 10 号旗舰店开业，至今仍是品牌圣地 |
| **1930s** | 开创彩色宝石镶嵌工艺，打破传统法式珠宝设计规范 |
| **1948** | Serpenti 灵蛇系列首次亮相，成为品牌最具标志性的设计 |
| **1977** | Bulgari Bulgari 系列推出，将品牌名字刻在戒指上的创新设计 |
| **2001** | 推出首家宝格丽酒店（米兰），进军奢华酒店领域 |
| **2011** | 被 LVMH 集团以约 43 亿美元收购，成为集团珠宝核心品牌 |

## 业务结构分析

### 珠宝

Serpenti、B.zero1、Divas Dream 等系列，贡献约 70% 收入

### 腕表

Octo Finissimo 系列多次打破超薄腕表世界纪录

### 皮具与配饰

手袋、太阳镜、丝巾等，依托 LVMH 的皮具供应链

### 宝格丽酒店

全球 7 家宝格丽酒店（米兰、巴厘岛、伦敦、北京、上海、迪拜、罗马）

## 🔑 核心护城河

```
意大利设计传统 → 标志性产品 → 品牌溢价 → LVMH资源 → 全球扩张 → 更多品牌影响力
```

1. Serpenti 灵蛇系列拥有超过 75 年历史，是奢侈品界最持久的设计 IP 之一
2. Octo Finissimo 腕表 8 次打破超薄世界纪录，确立制表领域技术实力
3. LVMH 集团的全球零售网络和供应链支持，加速品牌国际化
4. 宝格丽酒店将品牌从珠宝延伸到生活方式，提升品牌体验

## 值得了解

创始人是希腊人，原姓 Voulgaris，品牌名字保留了希腊语拼写传统

伊丽莎白泰勒是宝格丽最著名的粉丝，在电影埃及艳后中佩戴宝格丽珠宝

Serpenti 灵蛇设计灵感来自古罗马神话，蛇象征智慧和重生

宝格丽酒店的浴室用品全是宝格丽调制的专属香氛
