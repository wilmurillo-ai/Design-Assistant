工作原理

三步打造
你的演示文稿

1

运行 /slide-creator

在 Claude Code 或 OpenClaw 中。说出你的主题和内容目标。

2

从 3 个视觉预览中选择

在提交前比较风格选项。凭感觉选择，不用猜测。

3

打开 HTML

独立文件。无需服务器。无需安装。作为文件共享或在任何地方托管。

两阶段工作流： 使用 --plan 先概述，然后 --generate 生成完整演示文稿。
