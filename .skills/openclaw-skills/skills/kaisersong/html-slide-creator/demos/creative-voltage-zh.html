流程

从想法到演示文稿只需几分钟

四个问题。三个预览。一个 HTML 文件。这就是整个工作流 — 快速构建，更快发布。

01
/slide-creator
回答 4 个问题：目的、时长、内容、图片。最小摩擦 — 最大清晰度。
02
选择风格
对 3 个视觉预览做出反应。指向那个感觉。不要描述它 — 只需感受它。
03
打开并呈现
箭头 · 空格 · 滑动 · 圆点。任何浏览器。永久有效。零依赖。
功能

你需要的一切。你不需要的什么都没有。

有意简化 — 每个功能
服务演示，而不是产品团队。

视口
视口适配
每张幻灯片恰好填充 100vh。溢出的内容被分割 — 而不是压缩。
编辑
内联编辑
按 E。点击文本。实时编辑。Ctrl+S 保存。无需代码编辑器。
导出
PPTX 导出
通过 Playwright + 你的 Chrome 使用 --export pptx。像素完美的 PowerPoint 输出。
工作流
两阶段流程
--plan 首先创建 PLANNING.md 大纲。--generate 呈现 HTML。渲染前审查。
