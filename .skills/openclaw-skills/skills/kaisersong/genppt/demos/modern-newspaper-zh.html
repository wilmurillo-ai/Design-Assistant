Slide Creator 方式

选择适合您受众的风格。每次都获得稳定、精美的输出。19 个设计预设，零妥协。

架构

单个 presentation.html 文件。内联 CSS 和 JavaScript。无外部依赖。在任何浏览器中运行，在线或离线均可。

通过电子邮件分享。放在 USB 上。在飞机上打开。无需 CDN 调用、npm 或构建工具。

如何使用
  • 01
    /slide-creator

    交互式 — 样式发现、内容收集、生成全在一个会话中完成

  • 02
    --plan

    分析 resources/，创建 PLANNING.md 大纲。在生成 HTML 前停止。

  • 03
    --generate

    读取 PLANNING.md 并生成 HTML。直接跳到生产环境。

  • 04
    --export pptx

    通过系统 Chrome 截屏每张幻灯片并组装 PowerPoint 文件。
