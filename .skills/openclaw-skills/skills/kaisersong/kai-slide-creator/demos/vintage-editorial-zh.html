01
复杂的工具链
npm install、webpack 配置、300MB node_modules——仅仅是为了制作幻灯片。太荒唐了。

通用化输出

03
没有所有权
封闭平台、导出限制、月度收费。你的幻灯片被扣为人质。
slide-creator§02 / 08
核心洞察

真正有效的风格发现

"你无法用言语描述设计偏好。"

slide-creator 生成 3 个视觉预览——你反应，不描述。没有色彩理论。没有字体名称。只是：就是这个。

I.
表达气质
告诉它你想要的氛围。温暖、锐利、活泼、严肃但不沉闷。
II.
生成 3 个迷你预览
三种不同的视觉风格，并排展示，瞬间生成。
III.
选择感觉对的那个
相信你的直觉。这就是全部要点。

工作流程

从想法到演示文稿，只需几分钟

I.
/slide-creator
回答几个问题。内容、气质、观众、语气。两分钟最多。
II.
从 3 个预览中选择
对三种视觉风格做出反应。不需要色彩理论——我们保证。
III.
打开你的 HTML
方向键、空格、滑动。在任何浏览器中运行。从不需要安装。

从开始到完成不超过十分钟。我们计过时间。如果超时了，责怪你的内容——不是工具。
