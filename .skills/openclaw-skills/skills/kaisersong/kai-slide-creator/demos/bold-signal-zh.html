零依赖 HTML 文件，21 种设计预设，一个文件全部搞定。不需要安装任何软件，浏览器打开即演示。

05
kai-slide-creator/核心模式
05
三种模式

播放、演讲、编辑 — 一个文件搞定

F5
播放
全屏缩放，控制栏隐藏
P
演讲
备注、计时器、同步
E
编辑
所见即所改，Ctrl+S 保存
无需播放器，无需 PowerPoint，浏览器就够了。
06
kai-slide-creator/使用方法
06
三个命令，一个流水线

从规划到生成，全程自动化

/slide-creator --plan
生成规划大纲
创建 PLANNING.md，可编辑后再生成
/slide-creator --generate
生成 HTML
从规划文件生成，执行质量校验
/slide-creator --review
检查与修复
对已有 HTML 执行检查点
或者直接给内容 + 风格，立即生成。
