What is slide-creator?

A skill that generates self-contained HTML presentations. Zero external dependencies. Runs in any browser. 21 distinct visual styles — each one intentional, none generic.

  • No frameworks, no build step
  • 21 design presets, not one generic theme
  • Playback mode — F5 fullscreen
  • Presenter mode — notes, timer, sync
  • Edit mode — E key, Ctrl+S save
