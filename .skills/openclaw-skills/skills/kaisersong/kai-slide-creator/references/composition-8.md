# Composition Guide — 8 页精简版（slide-creator 自身介绍）

用于 slide-creator 自身 demo 文件。所有 21 种预设的自我介绍演示均使用此 8 页结构。

---

## 叙事弧线（8 页）

| # | 叙事角色 | 布局分类 | 叙事目的 |
|---|---------|---------|---------|
| **1** | **Hero 封面** | 全屏宣告 | 产品名 + 核心主张 + 关键数据（0 依赖 / 21 预设 / ∞ 幻灯片） |
| **2** | **问题提出** | 分栏证据 | 现有工具的痛点：复杂工具链、通用 AI 美学、平台锁定 |
| **3** | **风格发现** | 流程步骤 | 看图选风格：3 个预览 → 反应而非描述 |
| **4** | **解决方案** | 推荐网格 | 21 种预设展示，强调"没有通用 AI 美学" |
| **5** | **工作流程** | 流程步骤 | 三个命令：/slide-creator → 选风格 → 打开演示 |
| **6** | **核心功能** | 网格检查点 | 视口适配 / 编辑模式 / PPTX 导出 / 两阶段流程 |
| **7** | **安装方式** | 证据列表 | Claude Code + OpenClaw 各一行命令 |
| **8** | **CTA** | 堆叠行动 | `/slide-creator` — 立即开始 |

---

## 布局定义

复用 `composition-guide.md` 的布局分类定义。每页必须使用 2-3 种不同组件类型，禁止连续两页相同布局。
