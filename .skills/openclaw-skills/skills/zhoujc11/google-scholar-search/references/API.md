# Semantic Scholar API Reference

## Base URL

```
https://api.semanticscholar.org/graph/v1
```

## Rate Limits

- **Free tier**: 5 requests per second, 100,000 calls per day
- No API key required for basic usage
- For higher limits, apply for an API key at https://www.semanticscholar.org/product/api

## Search Endpoint

### `GET /paper/search`

Query parameters:
- `query` (required): Search query
- `limit` (optional): Number of results (max 100)
- `year` (optional): Filter by year (e.g., "2023" or "2020-2023")
- `minCitationCount` (optional): Minimum citation count
- `publicationTypes` (optional): Type of publication ("JournalArticle", "Conference", "Book")
- `fields` (optional): Comma-separated fields to return

### Available Fields

**Basic:**
- paperId
- title
- abstract
- year

**Authors:**
- authors.name
- authors.authorId

**Citations:**
- citationCount
- influentialCitationCount

**Publication:**
- venue
- publicationVenue
- publicationDate
- journal

**Links:**
- url
- openAccessPdf.url
- openAccessPdf.status

**Advanced:**
- references
- citations
- fieldsOfStudy
- s2FieldsOfStudy

## Paper Details Endpoint

### `GET /paper/{paper_id}`

Get detailed information about a specific paper, including:
- Full references list
- Full citations list
- Author details
- Fields of study
- Tldr (Too Long; Didn't Read summary - generated by AI)

## Field of Study Codes

Common fields:
- Computer Science (cs)
- Medicine (med)
- Biology (bio)
- Physics (phys)
- Chemistry (chem)
- Mathematics (math)
- Engineering (eng)
- Psychology (psych)
- Economics (econ)

## Common Use Cases

### Finding Review Papers
```bash
python3 {baseDir}/scripts/search_papers.py "survey" --year 2020-2024
```

### Finding Papers by Topic
```bash
python3 {baseDir}/scripts/search_papers.py "deep learning computer vision" --limit 20
```

### Finding Recent Papers
```bash
python3 {baseDir}/scripts/search_papers.py "natural language processing" --year 2024 --limit 15
```

### Finding High-Impact Papers
```bash
python3 {baseDir}/scripts/search_papers.py "reinforcement learning" --min-citations 100
```
