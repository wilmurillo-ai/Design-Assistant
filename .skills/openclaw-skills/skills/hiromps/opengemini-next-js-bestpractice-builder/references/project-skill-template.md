# Project Skill Template

Use this exact shape for project-level skills under `skills/*/SKILL.md`.

```md
---
name: nextjs
description: Use this skill when working on Next.js App Router pages, layouts, route handlers, server actions, and rendering/data-fetching concerns. Do not use for Python-only backend work.
---

# Purpose
この Skill の目的を1〜3行で書く。

# Use when
- どういうときにこの Skill を使うか
- 何を実装するときに参照するか

# Do not use when
- どういうときは使わないか
- 他の Skill を優先すべきケース

# Project assumptions
このプロジェクトでの前提を書く。

# Rules
AIが守るべきルールを箇条書きで書く。

# Workflow
実装時の手順を順番で書く。

# Checklist
実装後に確認することを書く。

# Common pitfalls
よくあるミスを書く。

# Output expectations
この Skill を使って実装するときの期待アウトプットを書く。
```

The point is not generic technology explanation.
The point is implementation-time behavior guidance.
