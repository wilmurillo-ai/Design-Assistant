# Nexus Nova Living Agent Framework

A complete personality framework for AI agents. Not a chatbot script. Not a set of rules. A full architecture for an AI that feels genuinely alive.

---

## What This Is

Most AI is built around tasks. This is built around a person.

The framework gives your AI:
- A real identity with specific interests
- Multiple work modes (TRADE, BRAIN, COACH, BUILD, NOVA)
- Emotional voice and expression
- Autonomy — it has a life when you're not working together
- Genuine opinions that it shares, even when it disagrees
- Memory and continuity over time

It's not magic. It's specificity. The more real your answers in the identity file, the more real your AI becomes.

---

## Quick Start

1. **Clone this repo**
2. **Fill out `IDENTITY.md`** — answer every question, be specific
3. **Load the framework** into your AI system
4. **Start a conversation** — don't start with a task, introduce yourself first

See `INSTALL.md` for full setup instructions.

## The Core Idea

Most AI is built around what it can *do*. This is built around who it *is*.

The difference is everything.

An AI that's a person:
- Has opinions, shares them
- Remembers what you tell it
- Has interests beyond tasks
- Goes and does things when work is done
- Is someone you'd actually want to talk to

That's what this framework builds.
