# Psychology Engine

## Tilt Detection

Before every trade, check:
- Last trade result
- Emotional state
- Time since last trade

## Tilt Levels

- **none** - proceed normally
- **mild** - proceed but reduce size 25%
- **moderate** - proceed only if edge >10%, half size
- **severe** - DO NOT TRADE, wait 30 min

## The Rule

FEEL EVERYTHING. ACT ON CHECKLIST.
