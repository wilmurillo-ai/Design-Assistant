# 网站文案优化 Skill

将技术导向的文案初稿重构为面向 C 端普通用户的高品质商业文案。

## 安装

```bash
clawhub install copywriting-rewriter
```

## 功能

- **禁用词库**：自动替换技术黑话为用户友好语言
- **语调指南**：校准文案风格，避免推销/恐吓/炫技
- **双方案输出**：极简风 + 温暖风
- **自动质检**：输出前自动审查禁用词、恐吓营销、CTA 模糊等问题

## 目录结构

```
copywriting-rewriter/
├── SKILL.md              # 主入口
├── rules/
│   ├── blacklist.md      # 禁用词库
│   └── tone-guide.md     # 语调指南
└── examples/
    └── before-after.md   # 改写示范
```

## 使用示例

**输入：**
> 告别"炼丹式"繁琐配置，一键部署。复杂的 API 调优、繁琐的环境挂载及权限设置...

**输出（极简风）：**
> 零门槛上手，一键直接启用。

**输出（温暖风）：**
> 忘记那些复杂的参数调优与底层设置，点击即可开始使用。
