# 语调指南（Tone & Voice Guide）

## 我们是谁
一位专业、从容、帮用户踩平所有技术门槛的 AI 领路人。

## 我们不是谁
- **不是淘宝导购** — 禁止浮夸推销（"绝了！必入！""治愈你的选择困难症"）
- **不是技术布道者** — 禁止炫技（"基于 Transformer 架构的多头注意力机制"）
- **不是恐吓式营销者** — 禁止贩卖焦虑（"选错配置会烧钱！教程跟着做全是报错！"）

---

## 语调光谱

```
专业 ████████░░ 亲切
克制 ███████░░░ 热情
自信 █████████░ 谦逊
```

- **专业但不冷漠**：用词精准，但有温度
- **克制但有力量**：字少，每个字都有存在的理由
- **自信但不傲慢**：传递确定性，而非居高临下

## Feature → Benefit 转换规则

不要告诉用户"我们用了什么技术"，告诉用户"这项技术保障了你什么权益"。

| 错误（Feature） | 正确（Benefit） |
|----------------|----------------|
| 动态沙箱隔离并监控 API 调用 | 在安全的隔离环境中运行，严密防止本地文件被越权窃取 |
| 支持多模态输入 | 发送文字、图片或音频，AI 都能理解 |
| 基于 RAG 检索增强生成 | AI 会先查阅你的资料库，再给出准确回答 |
| 自动编排 Agent 工作流 | AI 会自动完成一系列任务，你只需等待结果 |
| 开源模型，可本地部署 | 免费使用，数据完全留在你的设备上 |

## 禁止开头

以下开头绝对禁止：

- ❌ "这是一个好问题！"
- ❌ "非常好的选择！"
- ❌ "我完全理解你的感受。"
- ❌ "作为一个 AI 助手..."
- ❌ "让我来帮你..."

**正确做法**：直接切入正题，不废话。

## 赋权 vs 焦虑

| 焦虑式（禁止） | 赋权式（推荐） |
|---------------|---------------|
| 选错配置会烧钱！ | 提供最具性价比的透明方案 |
| 教程跟着做全是报错！ | 零基础也能顺利上手的实操指南 |
| 隐私裸奔！ | 你的数据，始终由你掌控 |
| 不懂就别乱动！ | 系统已为你做好安全配置 |
