# 禁用词库（Jargon Blacklist）

以下词汇绝对禁止出现在面向 C 端用户的文案中。遇到时必须替换为右侧的"人话"版本。

## 基础黑话

| 禁用词 | 替换为 |
|---|---|
| 部署 | 启用 / 安装 / 开始使用 |
| API | 接口（或直接省略） |
| Token | 使用额度 / 消耗量 |
| 环境配置 | 前期设置 |
| 挂载 | 连接 / 加载 |
| 沙箱 | 隔离的安全环境 |
| MCP | 插件协议（或直接省略） |
| 代码投毒 | 恶意代码植入 |
| 依赖 | 底层组件 |
| 容器/镜像 | 运行环境 |
| 跑通 | 顺利完成 |
| 报错 | 技术错误 / 异常 |
| 炼丹 | 反复调试 |
| 开源模型 | 免费开放的 AI 模型 |

## 进阶黑话

| 禁用词 | 替换为 |
|---|---|
| 编译 | 生成 / 构建 |
| 实例化 | 创建 / 启动 |
| 智能体 | AI 助手 / AI 代理 |
| 编排 | 组织 / 安排 |
| 推理 | 运行 / 处理 |
| 微调 | 定制 / 个性化训练 |
| 向量化 | 转换为可理解的格式 |
| 嵌入 | 内置 / 集成 |
| 端侧 | 本地 / 设备上 |
| 多模态 | 支持文字、图片、音频等多种形式 |
| 上下文窗口 | 记忆容量 / 处理范围 |
| 幻觉 | 错误生成 / 不准确内容 |
| 提示词 | 指令 / 对话输入 |

---

## 心智模型类比（Jargon → Mental Model）

| 技术概念 | 用户心智模型 |
|---------|-------------|
| 一键部署智能体 | 像下载 App 一样添加你的 AI 助手 |
| 动态沙箱隔离 | 在安全的隔离房间里运行，外面的看不到里面 |
| API 调用 | 向服务器发送请求并获取结果 |
| Token 消耗 | 每次对话消耗的"额度" |
| 环境配置 | 前期准备工作 |
| 微调模型 | 给 AI 上私教课，让它更懂你的业务 |
| RAG 检索增强 | 让 AI 翻阅你的资料库再回答 |
| Agent 工作流 | AI 自动完成一系列任务 |

---

## 使用原则

1. **能用生活类比就不用技术术语**
2. **能省略就省略**（用户不需要知道的技术细节直接删除）
3. **转化视角**（从"系统怎么运作"转为"用户得到什么"）
