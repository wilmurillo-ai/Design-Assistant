# 第2章 信息系统项目管理

## 背诵重点

1. 项目定义：为创造独特的产品、服务或成果而进行的临时性工作
2. 项目特征：临时性、独特性、渐进明细
3. 项目与运营区别：临时性vs持续性、独特性vs重复性、目标导向vs过程导向
4. 项目管理定义：将知识、技能、工具和技术应用于项目活动，以满足项目要求
5. 项目管理五大过程组：启动过程组、规划过程组、执行过程组、监控过程组、收尾过程组
6. 项目管理十大知识领域：整体管理、范围管理、进度管理、成本管理、质量管理、资源管理、沟通管理、风险管理、采购管理、干系人管理
7. 项目生命周期：启动阶段、组织与准备阶段、执行阶段、收尾阶段
8. 项目生命周期类型：预测型生命周期（瀑布）、迭代型生命周期、增量型生命周期、适应型生命周期（敏捷）
9. 项目组织结构：职能型组织、项目型组织、矩阵型组织（弱矩阵、平衡矩阵、强矩阵）
10. 项目经理职责：计划、组织、协调、控制

## 补充说明

项目管理概述是软考高项的基础章节，需要掌握项目的基本概念、管理过程组、知识领域和组织结构等核心内容。
