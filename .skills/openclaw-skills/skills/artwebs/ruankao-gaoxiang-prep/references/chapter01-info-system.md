# 第1章 信息系统基础

## 背诵重点

1. 信息系统定义：由人、硬件、软件、数据和网络构成的，用于收集、处理、存储、传递和分发信息的系统
2. 信息系统基本功能：输入、处理、输出、反馈、控制
3. 信息系统发展阶段：电子数据处理系统（EDPS）、管理信息系统（MIS）、决策支持系统（DSS）、专家系统（ES）、办公自动化系统（OAS）
4. 信息特征：客观性、适用性、可传输性、共享性、时效性
5. 信息质量属性：准确性、完整性、一致性、及时性、安全性、可用性
6. 信息系统生命周期：立项、开发、运维、消亡四个阶段
7. 信息系统开发方法：结构化方法、原型法、面向对象方法、敏捷开发方法
8. 软件工程三要素：方法、工具、过程
9. 软件开发生存周期模型：瀑布模型、原型模型、螺旋模型、迭代模型、增量模型
10. 软件需求分类：功能需求、非功能需求、设计约束

## 补充说明

信息系统基础是软考高项的重要内容，需要掌握信息系统的基本概念、开发方法、生命周期等核心知识。
