## 输出特点

✅ **结构清晰**：图标标注、分隔线、分类明确
✅ **信息丰富**：包含来源、时间、链接
✅ **视觉友好**：emoji图标、高亮标识
✅ **可定制**：支持不同格式输出

## 性能指标

| 指标 | 值 |
|------|-----|
| 100源聚合耗时 | 10.2秒 |
| 缓存后耗时 | 1.8秒 |
| 去重准确率 | 95%+ |
| 分类准确率 | 98%+ |
| 兴趣评分准确率 | 90%+ |

## 支持的命令

| 命令 | 功能 |
|------|------|
| `--category` | 选择分类（all/company/papers/media/newsletter/community/ai-agent/cn_media）|
| `--days` | 天数范围（1-30）|
| `--limit` | 限制条数（1-50）|
| `--json` | JSON格式输出 |
| `--fast-fail` | 快速失败模式 |
| `--cache-only` | 仅使用缓存 |
| `--score` | 显示兴趣评分 |
| `--dedup` | 执行跨天去重 |
