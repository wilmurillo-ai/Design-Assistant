# 常见鱼种数据库

## 热带鱼

### 孔雀鱼
- 难度：⭐ 简单
- 水温：22-26°C
- pH值：6.5-7.5
- 体型：3-5cm
- 性格：温和
- 混养：适合

### 红绿灯
- 难度：⭐⭐ 中等
- 水温：24-28°C
- pH值：6.0-7.0
- 体型：2-3cm
- 性格：温和
- 混养：适合

### 斗鱼
- 难度：⭐ 简单
- 水温：24-28°C
- pH值：6.5-7.5
- 体型：5-7cm
- 性格：雄性好斗
- 混养：单养或雌性

### 神仙鱼
- 难度：⭐⭐⭐ 较难
- 水温：26-30°C
- pH值：6.5-7.5
- 体型：10-15cm
- 性格：温和
- 混养：需大空间
