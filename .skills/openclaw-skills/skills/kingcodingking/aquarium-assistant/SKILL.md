## 石头分类速查

| 分类 | pH影响 | 推荐场景 |
|------|--------|---------|
| 火山石 | 中性 | 生物过滤、所有鱼缸 |
| 青龙石 | 弱碱性 | 溪流缸、雨林缸 |
| 太湖石 | 强碱性 | 需要高pH的鱼缸 |
| 珊瑚石 | 强碱性 | 非洲慈鲷缸专用 |
| 河卵石 | 中性 | 底质、所有鱼缸 |

## 水草分类速查

| 分类 | 高度 | 光需求 | CO2需求 |
|------|------|--------|---------|
| 前景草 | 1-5cm | 中等 | 中等 |
| 中景草 | 5-15cm | 中等 | 低 |
| 后景草 | 15cm+ | 中等到强 | 低 |
| 浮草 | 水面 | 强 | 不需要 |
| 莫斯 | 附着 | 低到中等 | 低 |
