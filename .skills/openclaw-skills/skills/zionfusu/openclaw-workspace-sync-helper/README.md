# OpenClaw Workspace Sync Helper

这是一个「向导式」Skill，用于指导你
将 **云端 OpenClaw 工作区** 同步到本地电脑。

---

## 我需要准备什么？

在使用前，请确认你知道：
- 云端服务器操作系统（一般是 Linux）
- 本地电脑操作系统（Mac / Windows / Linux）
- 云端 OpenClaw 的工作目录路径

---

## 如何开始？

在对话中直接输入：
> 我想把云端 OpenClaw 工作区同步到本地

Skill 会一步步引导你完成配置，
你只需要按提示回答即可。

---

## 这个 Skill 能做什么？

- 推荐合适的同步方案（sshfs / rsync / syncthing）
- 生成可复制执行的示例命令
- 提醒常见错误和注意事项

---

## 这个 Skill 不能做什么？

- 不会自动执行命令
- 不会修改你的服务器
- 不会后台常驻运行

所有操作都需要你手动确认并执行。
