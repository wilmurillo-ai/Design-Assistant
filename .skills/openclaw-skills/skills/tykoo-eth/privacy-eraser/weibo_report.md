# 微博隐私投诉模板

## 投诉入口
1. 微博内容 → 右上角「...」→「投诉」→「人身攻击/隐私泄露」
2. 网页版: https://service.account.weibo.com/
3. 客服热线: 400-690-0000

## 投诉理由模板

### 选择类型
- [x] 泄露隐私
- [x] 人身攻击

### 投诉描述

```
被投诉微博链接：[URL]

投诉理由：
该微博未经本人授权，公开披露了本人的个人隐私信息：

【泄露的信息类型】
- 真实姓名：[是/否]
- 手机号码：[是/否]
- 身份证号：[是/否]
- 家庭住址：[是/否]
- 工作单位：[是/否]
- 照片：[是/否]
- 其他：[具体说明]

【具体侵权内容】
原文摘录："[引用侵权内容]"

【本人诉求】
1. 删除该条微博
2. 或删除/打码涉及本人的个人信息部分

本人承诺以上投诉内容真实有效。

联系方式：[手机号]
日期：[YYYY-MM-DD]
```

## 补充材料
- 身份证明（证明信息属于你）
- 侵权内容截图
- 授权证明（如代他人投诉）

## 处理时间
- 普通投诉：3-7 个工作日
- 严重情况：24-48 小时

## 如未处理
1. 再次投诉，选择「申诉」
2. 发私信给 @微博客服
3. 拨打客服热线：400-690-0000
4. 向网信办举报
