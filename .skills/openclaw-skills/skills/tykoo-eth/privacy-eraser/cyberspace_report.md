# 国家网信办举报 (中央网络安全和信息化委员会办公室)

## 举报入口
https://www.12377.cn/

## 适用情况
- 平台投诉无果时的升级渠道
- 严重的隐私侵权
- 涉及大规模个人信息泄露
- 平台不作为

## 举报类型选择
- 侵犯公民个人信息
- 违法和不良信息

## 举报模板

```
举报类型：侵犯公民个人信息

被举报网站/平台：[平台名称]
被举报内容链接：[URL]

举报内容描述：

一、基本情况
[时间]，[平台名称] 上发布的内容（链接如上）公开披露了本人的个人信息，具体包括：
- [姓名]
- [手机号码]
- [身份证号码]
- [家庭住址]
- [工作单位]
- [其他敏感信息]

二、已采取的措施
本人于 [日期] 向该平台提交了隐私投诉（投诉编号/截图：[附上]），但截至目前（已超过 [X] 天）：
☐ 平台未回复
☐ 平台拒绝处理
☐ 平台处理不当
☐ 内容仍未删除

三、违法情况
该行为违反了：
1. 《中华人民共和国个人信息保护法》第十三条——处理个人信息需取得同意
2. 《中华人民共和国网络安全法》第四十二条——不得泄露个人信息
3. 《中华人民共和国民法典》第一千零三十二条——隐私权保护

四、举报诉求
1. 责令平台删除侵权内容
2. 对平台的不作为行为进行处理
3. 保护本人的个人信息权益

举报人信息：
- 姓名：[真实姓名]
- 身份证号：[身份证号]
- 联系电话：[手机号]
- 电子邮箱：[邮箱]

本人承诺以上举报内容真实有效，愿意配合调查并承担相应法律责任。

日期：[YYYY-MM-DD]
```

## 附件准备
1. 身份证正反面照片
2. 侵权内容截图（含URL和时间）
3. 向平台投诉的记录/截图
4. 平台回复（如有）
5. 其他证据材料

## 处理流程
1. 提交举报 → 获得举报编号
2. 网信办受理 → 3-5 个工作日
3. 调查处理 → 视情况
4. 反馈结果 → 可查询处理进度

## 查询进度
https://www.12377.cn/index/jbcx.html
输入举报编号查询

## 注意事项
- 举报内容要具体、有证据
- 优先通过平台解决，网信办是升级渠道
- 保留所有证据原件
- 如实填写个人信息，虚假举报可能承担责任
