# 抖音隐私举报模板

## APP 内举报文案

```
该视频包含本人真实影像/个人信息，未经本人同意拍摄和发布，侵犯本人肖像权和隐私权。

请立即删除该视频。
```

## 侵权投诉平台

入口：https://www.douyin.com/draft/copyright/complaint

### 投诉材料准备

1. **主体证明**
   - 身份证正反面照片
   - 手持身份证照片（可选但加速）

2. **侵权证据**
   - 视频截图（标注侵权位置）
   - 视频链接/分享码

3. **投诉说明**
```
投诉类型：侵犯隐私/肖像权

投诉说明：
该视频于 [日期] 发布，视频中出现本人真实影像/个人信息：
- [第 X 秒处出现本人面部]
- [视频描述中提及本人姓名/单位]
- [评论区泄露本人联系方式]

本人从未授权该内容的拍摄和传播。根据《民法典》第一千零一十九条，未经肖像权人同意，不得制作、使用、公开肖像权人的肖像。

要求：立即删除该视频及相关评论。
```

## 私信博主模板（最快路径）

```
你好，我是视频中出现的当事人。

该视频包含我的 [面部/姓名/其他信息]，发布前未经我同意，侵犯了我的隐私权。

请在 24 小时内删除该视频，否则我将通过平台投诉和法律途径维权。

谢谢配合。
```

## 注意事项

1. 私信博主协商删除是最快路径
2. "未经同意拍摄"成功率较高
3. 视频比图文难删（审核人工+算法）
4. 播放量高的内容几乎删不掉，考虑其他策略
5. 处理时间：1-7天
