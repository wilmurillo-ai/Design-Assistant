# 小红书隐私举报模板

## APP 内举报文案

```
该笔记包含本人真实姓名/照片/个人信息，未经本人授权公开发布，严重侵犯本人隐私权。

根据《个人信息保护法》第四十七条，本人有权要求删除涉及本人的个人信息。

请立即删除该内容。
```

## 邮件举报模板 (jubao@xiaohongshu.com)

```
主题：隐私侵权投诉 - 请求删除笔记

小红书内容审核团队：

本人 [姓名] 发现以下笔记侵犯本人隐私权：

笔记链接：[URL]
笔记作者：[博主ID]
侵权内容：[具体描述哪些信息被泄露]

该笔记包含本人以下个人信息，未经本人同意公开：
- [ ] 真实姓名
- [ ] 照片/视频
- [ ] 联系方式
- [ ] 工作单位/学校
- [ ] 其他：________

根据《中华人民共和国个人信息保护法》第四十七条，个人有权要求个人信息处理者删除其个人信息。

请在收到本邮件后 72 小时内删除上述内容。

附件：
- 身份证正面照片（打码）
- [其他证明材料]

联系方式：[手机号/邮箱]

此致
[姓名]
[日期]
```

## 注意事项

1. 选择"人身攻击"类型比"隐私泄露"处理更快
2. 多个账号同时举报会加速处理
3. KOL/大V 内容难删，考虑先私信协商
4. 素人笔记举报成功率较高
