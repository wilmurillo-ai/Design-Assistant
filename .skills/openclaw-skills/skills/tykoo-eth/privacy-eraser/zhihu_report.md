# 知乎隐私投诉模板

## 投诉入口
1. 内容页面 → 右下角「举报」→「侵犯个人隐私」
2. 或访问: https://www.zhihu.com/report

## 投诉理由模板

### 选择类型
- [x] 侵犯隐私
- [x] 泄露个人信息

### 投诉描述

```
投诉内容链接：[URL]

本人 [姓名/知乎ID]，该内容未经本人同意公开了以下个人信息：

【被泄露的信息】
- [真实姓名]
- [手机号码/微信号]
- [工作单位/学校]
- [家庭住址]
- [其他敏感信息]

【信息出现位置】
- 文章第 X 段 / 回答中第 X 行
- 具体原文："[引用侵权内容]"

【诉求】
请删除该内容，或至少隐去本人个人信息。

本人从未授权发布上述信息，该行为侵犯了本人的隐私权和个人信息权益，违反了《个人信息保护法》。

联系邮箱：[email]
```

## 补充材料
- 身份证明（证明你是信息主体）
- 侵权内容截图
- 信息属于你的证明

## 预计处理时间
- 快速通道: 24-72 小时
- 普通: 7 个工作日

## 如未处理
1. 发邮件至 privacy@zhihu.com
2. 使用知乎盐选会员专属客服（如有）
3. 向网信办举报
4. 法律途径
