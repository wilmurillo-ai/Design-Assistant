# 微信公众号隐私投诉模板

## 投诉入口
1. **APP内**: 文章底部 → "投诉" → 选择"侵犯隐私"
2. **PC端**: 文章页右上角 "..." → 投诉
3. **专用通道**: https://weixin110.qq.com (隐私侵权)

## 投诉理由模板

### 选择类型
- [x] 侵犯隐私

### 投诉描述

```
本人 [真实姓名]，身份证号 [可选，脱敏]。

该文章未经本人授权，公开披露了本人的以下个人信息：
- [列出具体信息，如：真实姓名、手机号、工作单位、住址等]

上述信息的公开严重侵犯了本人的隐私权，违反了《中华人民共和国个人信息保护法》第十三条关于个人信息处理需取得个人同意的规定。

本人从未授权该公众号收集、使用或公开本人的个人信息。

请求：
1. 立即删除该文章或隐去本人个人信息
2. 防止该内容进一步传播

本人保留追究法律责任的权利。

联系方式：[手机号/邮箱]
日期：[YYYY-MM-DD]
```

## 补充材料
- 身份证正面照片（可打码部分数字）
- 证明该信息属于本人的材料
- 截图保存的侵权证据

## 预计处理时间
- 一般: 3-7 个工作日
- 复杂情况: 15 个工作日

## 如未处理
1. 再次投诉，附上首次投诉记录
2. 联系微信客服: 0755-83765566
3. 向网信办举报
4. 考虑法律途径

## 潜规则 & 技巧

1. **律师函/公证书** → 处理速度快很多
2. **侵权投诉**比"其他"类型处理优先级高
3. 普通用户投诉需要排队，会比较慢
4. 已过原创保护的文章更难删除
5. 处理时间：3-7天（普通），1-3天（有证据）
