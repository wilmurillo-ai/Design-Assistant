## 1. 设备概述

本文档描述郑州科威莱步进电机控制器的技术规格和操作方法。该设备通过 Modbus RTU 协议进行通讯，支持精确的位置控制和状态反馈。

### 1.1 主要功能

- ✅ Modbus RTU 通讯 (RS-485/RS-232)
- ✅ 精确位置控制 (增量模式)
- ✅ 机械原点设定 (反转限位)
- ✅ 限位保护功能
- ✅ 速度可调 (1-800 r/min)
- ✅ 加减速控制
- ✅ 实时位置反馈

### 1.2 应用场景

- 自动化设备位置控制
- 精密位移平台
- 线性模组控制
- 实验设备自动化

## 5. 机械参数

### 5.1 行程参数

| 参数 | 值 | 单位 |
|------|-----|------|
| 每圈行程 | 5 | mm |
| 每圈角度 | 360 | ° |
| 1mm 对应角度 | 72 | ° |
| 1° 对应行程 | 0.0139 | mm |

### 5.2 换算公式

```
毫米 → 角度：角度 = 毫米 × 72
角度 → 毫米：毫米 = 角度 ÷ 72
圈数 → 角度：角度 = 圈数 × 360
角度 → 圈数：圈数 = 角度 ÷ 360
```

### 5.3 示例计算

| 目标距离 | 目标角度 | 圈数 |
|----------|----------|------|
| 1mm | 72° | 0.2 圈 |
| 5mm | 360° | 1 圈 |
| 6mm | 432° | 1.2 圈 |
| 10mm | 720° | 2 圈 |

### 5.4 限位保护

- **最大行程**: 3600° (10 圈，50mm)
- **机械原点**: 反转限位位置 (寄存器 23=2)
- **限位使能**: 寄存器 13=1 (已启用)

### 7.5 CRC16 计算

Modbus CRC16 多项式：`x^16 + x^15 + x^2 + 1` (0xA001)

```python
def calculate_crc16(data):
    crc = 0xFFFF
    for byte in data:
        crc ^= byte
        for _ in range(8):
            if crc & 0x0001:
                crc = (crc >> 1) ^ 0xA001
            else:
                crc >>= 1
    return crc
```

## 9. 注意事项

### 9.1 硬件注意事项

1. **接线检查**
   - 确保 RS-485 接线 A+、B- 极性正确
   - 长距离通讯建议使用终端电阻 (120Ω)
   - 避免与大功率设备共用电源

2. **限位开关**
   - 确保限位开关接线正确
   - 定期检查限位开关功能
   - 寄存器 13 启用后，注意不要超过 3600°

3. **接地**
   - 设备应可靠接地
   - 避免静电干扰

### 9.2 软件注意事项

1. **通讯设置**
   - 通讯超时建议设置为 1-3 秒
   - 两次通讯间隔建议 ≥ 50ms
   - 注意字节序 (大端模式)

2. **寄存器 2 使用**
   - ⚠️ 运行中无法写入 0
   - ⚠️ 紧急停止请使用寄存器 3
   - 任务完成后自动归 0

3. **原点设定**
   - ⚠️ 必须通过反转限位确定机械原点
   - ⚠️ 设定原点后需等待 1 秒
   - 不要使用寄存器 10 代替机械原点

4. **位置保护**
   - ⚠️ 寄存器 22 不要超过 3600°
   - 定期执行回原点操作
   - 监控限位状态 (寄存器 23)

### 9.3 错误处理

| 现象 | 可能原因 | 解决方法 |
|------|----------|----------|
| 无响应 | 接线错误、地址错误、波特率错误 | 检查接线、地址、波特率 |
| CRC 错误 | 干扰、接线质量差 | 检查屏蔽、缩短距离 |
| 位置异常 | 未设定原点、溢出 | 重新设定机械原点 |
| 限位触发 | 超出行程 | 反向移动、检查程序 |
