# 部署指南

## 部署准备

### 硬件准备

1. QuecPython 开发板或设备
2. USB 数据线 (用于烧录)
3. SIM 卡 (已开通流量)
4. 4G 天线
5. RS485 传感器设备

### 软件准备

1. QuecPython 开发工具 (QPYcom)
2. 串口驱动
3. 代码文件 (本项目全部 `.py` 文件)

---

## 部署步骤

### 步骤 1: 连接设备

1. 使用 USB 线连接设备和电脑
2. 打开 QPYcom 工具
3. 选择正确的 COM 端口
4. 点击"连接"

### 步骤 2: 上传文件

将以下文件上传到设备的 `/usr` 目录：

```
config/
└── config.py           →  /usr/config.py

src/
├── main.py             →  /usr/main.py
├── led_control.py      →  /usr/led_control.py
├── modbus_rtu.py       →  /usr/modbus_rtu.py
├── sensor_device.py    →  /usr/sensor_device.py
├── network_manager.py  →  /usr/network_manager.py
├── system_manager.py   →  /usr/system_manager.py
└── data_reporter.py    →  /usr/data_reporter.py
```

**注意**: QuecPython 不支持子目录，所有文件必须放在同一目录

### 步骤 3: 配置文件

上传后，在设备上编辑 `/usr/config.py`，修改以下参数：

```python
# 修改为你的平台地址
URL_REPORT = "https://your-platform.com/api/data"
URL_OTA = "https://your-ota-server.com/ota"

# 修改设备信息
PROJECT_NAME = "your-project-name"
PROJECT_VERSION = "1.0.0"

# 如需修改 Modbus 参数
MODBUS_BAUDRATE = 9600  # 根据传感器调整
```

### 步骤 4: 运行测试

在 QPYcom 控制台执行：

```python
import main
main.main()
```

观察输出日志，确认：

- [✓] 网络连接成功
- [✓] Modbus 通信正常
- [✓] 数据上报成功

### 步骤 5: 设置开机自启

编辑设备的 `main.py` (系统启动文件)，添加：

```python
import usr.main
usr.main.main()
```

或者在 QPYcom 中设置"开机自动运行"。

## 验证部署

### 1. 检查文件

```python
import os
files = os.listdir('/usr')
print(files)
```

应看到所有 8 个 `.py` 文件。

### 2. 测试 Modbus

```python
from modbus_rtu import ModbusRTU
modbus = ModbusRTU(2, 9600, 8, 0, 1, 0)
result = modbus.read_registers(1, 0, 1)
print(result)
# 应返回：{'error': 0, 'data': [xxx]}
```

### 3. 测试网络

```python
from network_manager import NetworkManager
net = NetworkManager("test", "1.0")
if net.wait_connected(30):
    info = net.get_network_info()
    print("IMEI:", info['IMEI'])
    print("信号:", info['signal_power'])
```

### 4. 测试上报

```python

led = LED(36)
```

## 故障排查

### 问题 1: 文件上传失败

**原因**: 存储空间不足

**解决**:
```python
import os
# 删除无用文件
os.remove('/usr/old_file.py')
```

### 问题 2: 导入模块失败

**原因**: 文件未上传或命名错误

**解决**:
- 检查文件是否在 `/usr` 目录
- 检查文件名是否完全一致 (区分大小写)

### 问题 3: 运行时报错

**查看错误日志**:
```python
import sys
try:
    import main
    main.main()
except Exception as e:
    sys.print_exception(e)
```

## 升级维护

### OTA 升级

设备会自动检查 OTA 升级：

1. 每日上报时检查平台返回
2. 如有新版本，自动下载升级
3. 升级后自动重启

### 手动升级

1. 在平台上传新版本固件
2. 设备重启后会自动检查
3. 或等待次日自动检查

## 备份建议

部署成功后，建议备份以下配置：

1. `config.py` - 你的配置参数
2. 设备 IMEI、ICCID 等信息
3. 平台 URL 地址

```python
# 备份设备信息
import modem, sim
print("IMEI:", modem.getDevImei())
print("ICCID:", sim.getIccid())
```
