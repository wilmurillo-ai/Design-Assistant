# 快速部署脚本 - QuecPython 设备

## 使用说明

本脚本用于将所有代码文件快速部署到 QuecPython 设备的 `/usr` 目录。

---

## 方法 1: 使用 QPYcom 工具

### 步骤

1. 打开 QPYcom
2. 连接设备
3. 选择"文件管理"
4. 上传以下 8 个文件到 `/usr` 目录：

```
config.py              (从 config/config.py 复制)
main.py                (从 src/main.py 复制)
led_control.py         (从 src/led_control.py 复制)
modbus_rtu.py          (从 src/modbus_rtu.py 复制)
sensor_device.py       (从 src/sensor_device.py 复制)
network_manager.py     (从 src/network_manager.py 复制)
system_manager.py      (从 src/system_manager.py 复制)
data_reporter.py       (从 src/data_reporter.py 复制)
```

---

## 方法 2: 使用 Python 脚本批量上传

```python
# deploy.py - 在电脑上运行
import os
import serial

# 文件列表
files = [
    'config/config.py',
    'src/main.py',
    'src/led_control.py',
    'src/modbus_rtu.py',
    'src/sensor_device.py',
    'src/network_manager.py',
    'src/system_manager.py',
    'src/data_reporter.py'
]

# 串口配置
ser = serial.Serial('COM3', 115200)

for file_path in files:
    with open(file_path, 'r', encoding='utf-8') as f:
        content = f.read()
    
    # 发送文件内容到设备
    filename = os.path.basename(file_path)
    print(f"上传 {filename}...")
    ser.write(f"with open('/usr/{filename}', 'w') as f:\n".encode())
    for line in content.split('\n'):
        ser.write(f"    f.write('{line}\\n')\n".encode())
    
print("上传完成！")
```

## 方法 3: 手动复制 (推荐新手)

### 逐个文件复制

1. 打开 `config/config.py`
2. 复制全部内容
3. 在 QPYcom 中创建 `/usr/config.py`
4. 粘贴内容
5. 保存

重复以上步骤上传所有 8 个文件。

## 常见问题

### Q: 上传后找不到文件？
**A**: 确认文件上传到了 `/usr` 目录，不是 `/usr/bin` 或其他目录

### Q: 导入模块失败？
**A**: 检查文件名是否完全一致 (区分大小写)

### Q: 配置文件未生效？
**A**: 确认上传的是 `config/config.py` 的内容，保存为 `/usr/config.py`

## 回滚到旧版本

如果新版本有问题，可以：

1. 备份当前配置
2. 重新上传旧版本文件
3. 重启设备

```python
# 备份当前配置
import config
print("当前配置:")
print("URL_REPORT:", config.URL_REPORT)
print("PROJECT_VERSION:", config.PROJECT_VERSION)
```
