## 🔧 核心配置参数

### 必须修改 (5 个)

| 参数 | 说明 | 示例 |
|------|------|------|
| `PROJECT_NAME` | 项目名称 | gas-monitor-beijing |
| `PROJECT_VERSION` | 版本号 | 1.0.0 |
| `URL_REPORT` | 上报平台地址 | https://platform.com/api |
| `URL_OTA` | OTA 地址 | https://ota.com/firmware |
| `REPORT_INTERVAL_SEC` | 上报频率 | 60 (秒) |

### 按需修改 (5 个)

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `MODBUS_BAUDRATE` | 波特率 | 9600 |
| `SENSOR_MAX_COUNT` | 传感器数量 | 32 |
| `LED_NET_PIN` | 网络 LED 引脚 | 36 |
| `LED_MODBUS_PIN` | Modbus LED 引脚 | 44 |
| `DEBUG_ENABLE` | 调试开关 | True |
