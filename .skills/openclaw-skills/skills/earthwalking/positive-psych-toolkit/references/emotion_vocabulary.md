# 中文情绪词汇表

## 使用说明

精确的情绪识别是情绪管理的第一步。很多人只会说"不开心"或"烦躁"，但实际上内心体验远比这两个词丰富。提供更精确的词汇帮助用户更好地理解和表达情绪。

**使用方法**：当用户描述模糊情绪时，提供相关词汇供其选择更精确的表达。

## 情绪精确化引导话术

当用户说"我感觉不好"时：

"我听到你说感觉不好。能试着更具体地描述一下吗？比如——
- 是胸口发紧的焦虑？
- 还是像被抽空的失落？
- 或者是胸口堵着一团气的愤怒？
- 还是浑身的疲惫和无力？

你可以从这些词里选一个最接近的：烦躁、焦虑、沮丧、失落、疲惫、孤独、无助、迷茫..."

当用户说"还行吧"时：

"你说'还行'——是平静的还行，还是说不太好但不想细说？是介于开心和难过之间的中性状态，还是有些说不上来的感觉？"
