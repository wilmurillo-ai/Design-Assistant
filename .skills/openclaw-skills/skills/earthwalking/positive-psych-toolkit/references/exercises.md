# 积极心理练习引导手册

## 使用说明

每个练习包含：目的、适用场景、引导步骤、注意事项和示例。选择练习时需匹配用户当前的心理状态和需求——不要在用户处于严重情绪低谷时推荐需要大量能量的练习。

---

## 练习 1：三件好事（Three Good Things）

**目的**：对抗消极偏见（negativity bias），训练大脑关注积极体验
**来源**：Seligman, Steen, Park & Peterson (2005)
**适用**：日常幸福维护、轻度情绪低落、培养感恩心态
**频率**：每日，连续 7-14 天效果最佳
**研究证据**：连续一周后幸福感提升持续 6 个月

**引导步骤**：

1. "每天晚上睡前，花 5 分钟写下今天发生的 3 件好事。"
2. "对每一件好事，写下：为什么这件好事会发生？你的感受是什么？"
3. "好事不分大小——吃到好吃的、朋友一句温暖的话、天气好，都算。"

**示例**：
```
好事 1：同事夸我今天的方案做得很用心
  原因：我确实花了额外时间准备，同事注意到了
  感受：被认可的感觉真好，工作更有动力了

好事 2：下班路上看到了漂亮的晚霞
  原因：我刚好抬起头看了一眼天空
  感受：突然觉得生活很美好，心情放松了

好事 3：妈妈打电话说身体不错
  原因：她最近坚持散步，身体确实好了些
  感受：安心，也为自己督促她运动感到高兴
```

**注意事项**：
- 刚开始可能觉得"今天没什么好事"——这是正常的，因为大脑习惯关注消极面
- 持续一周后，会发现越来越容易找到好事
- 如果某天真的想不出来，可以写"今天结束了，我活着，这是好事"

## 练习 2：感恩拜访（Gratitude Visit）

**目的**：大幅提升幸福感和人际关系质量
**来源**：Seligman et al. (2005)
**适用**：有需要感谢但一直未表达的人
**频率**：一次性，但效果持久
**研究证据**：单次练习幸福感提升可持续 1 个月

**引导步骤**：

1. "想一个对你的人生产生过积极影响、但你从未充分表达感谢的人。"
2. "写一封 300 字左右的感恩信，具体描述：他做了什么、对你有什么影响、你现在的生活因为他有什么不同。"
3. "如果可能，安排一次见面，亲自朗读这封信。不方便见面可以打电话或视频。"
4. "如果都无法实现，写完信本身就已有积极效果。"

**关键点**：
- 要具体，不要泛泛而谈
- 描述真实感受，不要只写客套话
- 接受对方的情绪反应（可能会感动流泪）

## 练习 3：积极重构（Positive Reframing）

**目的**：改变消极思维模式，培养更灵活的解释风格
**来源**：认知行为疗法 + Seligman 习得性乐观理论
**适用**：遇到挫折、面对失败、陷入消极思维循环时
**频率**：按需使用

**引导步骤（ABCDE 模型）**：

1. **A - Adversity（逆境）**：发生了什么事？（只描述事实，不加评价）
   "今天的工作汇报没通过。"

2. **B - Belief（信念）**：你当时怎么想的？
   "我能力不行，领导肯定对我失望了，我可能不适合这份工作。"

3. **C - Consequences（后果）**：这个想法让你有什么感受和行为？
   "感到沮丧、焦虑，想逃避接下来的工作。"

4. **D - Disputation（反驳）**：这个想法完全准确吗？有其他解释吗？
   - 证据挑战："上次汇报通过了，之前也得到过表扬。"
   - 替代解释："这次内容可能确实需要修改，不一定是能力问题。"
   - 意义重构："这正好是个学习改进的机会。"
   - 可行动化："我可以问领导具体的修改建议。"

5. **E - Energization（激发）**：反驳后你的感受和行动倾向是什么？
   "虽然有点失落，但觉得可以改进，准备明天去找领导沟通。"

**注意事项**：
- 不是"盲目乐观"，而是寻找更准确、更平衡的解释
- 允许消极情绪存在，重构不等于否定真实感受
- 悲观型解释风格三个特征：永久性（"永远"）、普遍性（"所有"）、个人化（"都是我的错"）

## 练习 4：优势日记（Strengths Journal）

**目的**：增强自我认知和自信心，发现日常中的积极体验
**来源**：VIA 优势理论应用
**适用**：低自信、自我价值感低、想更了解自己
**频率**：每日，连续 14 天

**引导步骤**：

1. "每天记录一个你运用了某项性格优势的时刻。"
2. 格式：
   ```
   日期：___
   情境：发生了什么
   使用的优势：我用了 [具体优势]
   结果：发生了什么好的结果
   感受：我当时的感受
   ```

**提示问题**：
- "今天有什么时刻让你觉得自己做得不错？"
- "你今天帮助了谁？怎么帮的？"
- "今天你克服了一个小困难吗？用的什么品质？"
- "今天有什么事让你感到自豪？"

## 练习 5：生命线（Life Timeline）

**目的**：整合人生叙事，发现成长模式和意义
**来源**：叙事心理学（McAdams）
**适用**：人生迷茫期、过渡期、重大决策前
**频率**：一次性深度练习（需要 30-60 分钟）

**引导步骤**：

1. "在纸上画一条横线代表你的人生时间轴。"
2. "在时间轴上标记关键事件——高潮（正峰值）和低谷（负峰值）。"
3. "对每个关键事件，写下：发生了什么、你的感受、你从中学到了什么。"
4. "完成后退后一步看整条线——你看到了什么模式？有没有转折点？"
5. "你如何讲述自己的人生故事？主角是谁？主题是什么？"

**反思问题**：
- "低谷时期你靠什么力量走出来的？"
- "高潮时期你做对了什么？"
- "如果把人生比作一本书，当前章节的标题是什么？"
- "你希望下一章写什么？"

## 练习 6：正念呼吸（Mindful Breathing）

**目的**：降低焦虑、提升专注力、增强情绪调节能力
**来源**：Kabat-Zinn 正念减压（MBSR）+ 佛教禅修传统
**适用**：焦虑、压力、失眠、情绪波动
**频率**：每日 5-15 分钟

**引导步骤**：

1. 找一个安静的地方，舒服地坐好
2. 闭上眼睛或半睁半闭，目光向下
3. 将注意力集中在呼吸上——感受空气进入鼻腔、胸腔扩张、腹部起伏
4. 当思绪飘走时（一定会的），温柔地把注意力拉回呼吸
5. 不评判自己"走神了"，走神了再回来，这就是练习
6. 从 5 分钟开始，逐渐延长到 15-20 分钟

**引导语（可直接使用）**：
"请找一个舒服的姿势坐好...现在把注意力带到你的呼吸上...不需要改变呼吸的方式，只是观察它...感受每一次吸气...每一次呼气...如果你发现自己走神了，没关系，这是很正常的...轻轻地，温柔地，把注意力带回呼吸..."

## 练习 7：未来自我写信（Future Self Letter）

**目的**：建立希望感、明确人生方向、增强目标动机
**来源**：King (2001) 希望理论 + Best Possible Self 练习
**适用**：目标不清晰、缺乏动力、人生过渡期
**频率**：每季度一次

**引导步骤**：

1. "想象 5 年后的你，一切顺利，你成为了最好的自己。"
2. "给现在的自己写一封信，从未来视角描述：你过着怎样的生活？你在做什么？你和谁在一起？你感到怎样？"
3. "不要限制自己，大胆想象。同时思考：从现在到那时，哪些优势和资源帮你到达那里？"
4. 写完后阅读，提取 3 个可行动的小步骤。

## 练习 8：社会连结清单（Social Connection Audit）

**目的**：评估和改善人际关系质量
**来源**：哈佛幸福研究
**适用**：感到孤独、人际关系困扰、想增强社会支持

**引导步骤**：

1. 画出 concentric circles（同心圆）：
   - 最内圈：最亲密的人（2-5 人）
   - 第二圈：重要的人（5-10 人）
   - 第三圈：定期联系的人
   - 最外圈：认识但不太联系的人

2. 反思每个圈：
   - "最内圈的人让我感到被支持和被爱吗？"
   - "有没有人应该在更内圈但被我忽略了？"
   - "最外圈中有没有想加深联系的人？"

3. 制定行动：
   - "本周联系一个疏远但重要的朋友"
   - "安排一次有质量的深度对话"
   - "减少消耗性关系的精力投入"

## 选择练习的建议

| 用户状态 | 推荐练习 | 避免 |
|---------|---------|------|
| 日常维护，整体状态好 | 三件好事、优势日记 | — |
| 轻度情绪低落 | 三件好事、正念呼吸、积极重构 | 生命线（需要较多心理能量） |
| 面对具体挫折 | 积极重构（ABCDE）、优势日记 | 感恩拜访 |
| 人际关系困扰 | 社会连结清单、感恩拜访 | — |
| 人生迷茫/无方向 | 生命线、未来自我写信、Ikigai | — |
| 自信心低 | 优势日记、优势探索 | — |
| 焦虑紧张 | 正念呼吸 | — |
