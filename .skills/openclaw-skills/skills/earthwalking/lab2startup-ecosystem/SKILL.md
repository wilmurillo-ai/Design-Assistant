# 师生共创生态系统对接指南

师生共创不是孤立的活动，需要大学、平台、社会的多方支持。本指南帮助用户系统性地对接和整合各类外部资源。

## 生态系统概览

```
           ┌─────────────────────────────────────┐
           │           大学层面                   │
           │  政策支持 │ 资源支持 │ 教育支持      │
           └─────────────────┬───────────────────┘
                             │
           ┌─────────────────┴───────────────────┐
           │           平台层面                   │
           │  教育 │ 培育 │ 生态                 │
           └─────────────────┬───────────────────┘
                             │
           ┌─────────────────┴───────────────────┐
           │           社会层面                   │
           │  投资机构 │ 产业伙伴 │ 政府支持     │
           └─────────────────────────────────────┘
```

## 一、大学层面资源对接

### 政策支持

#### 科技成果转化政策

**核心政策要点**：
- 成果所有权：明确科技成果归属
- 收益分配：技术入股、收益分成比例
- 审批流程：成果转化的审批权限和流程
- 税收优惠：技术转让税收减免

**政策对接清单**：
- [ ] 了解本校科技成果转化管理办法
- [ ] 确认成果归属和收益分配比例
- [ ] 了解成果转化审批流程
- [ ] 咨询税务优惠政策

**常见政策模式**：
| 模式 | 成果归属 | 收益分配 | 适用高校 |
|------|----------|----------|----------|
| 传统模式 | 学校所有 | 学校70%+ | 大部分高校 |
| 改革试点 | 团队所有 | 团队70%+ | 试点高校 |
| 混合模式 | 共有 | 按比例 | 改革推进中 |

#### 教师创业激励政策

**激励政策类型**：
- 保留教职：创业期间保留教师岗位
- 职称评审：创业成果纳入职称评审
- 科研项目：创业相关项目优先支持
- 收入分配：创业收入不纳入绩效工资总额

**政策对接要点**：
1. 咨询人事处：教师创业相关规定
2. 咨询科研处：科研项目支持政策
3. 咨询教务处：教学安排调整政策
4. 咨询院系：院系层面支持政策

#### 学生创业支持政策

**支持政策类型**：
- 学分认定：创业实践可认定学分
- 学籍管理：保留学籍创业
- 创业基金：学校创业基金支持
- 孵化空间：校内孵化器入驻

**政策对接要点**：
1. 咨询教务处：学分认定和学籍管理
2. 咨询团委/就业中心：创业基金和孵化空间
3. 咨询院系：院系创业支持政策

### 资源支持

#### 实验室开放

**可对接资源**：
- 实验场地：研发所需的实验室空间
- 实验设备：研发测试所需的仪器设备
- 技术支持：实验室技术人员支持
- 材料耗材：研发所需的原材料

**对接方法**：
1. 与实验室负责人沟通需求
2. 签订实验室使用协议
3. 明确使用范围和费用
4. 建立长期合作机制

**注意事项**：
- 明确知识产权归属
- 建立设备使用记录
- 遵守实验室管理规定
- 承担必要的费用

#### 设备共享

**共享平台对接**：
| 平台类型 | 资源类型 | 对接方式 |
|----------|----------|----------|
| 校级平台 | 通用设备 | 预约使用 |
| 院级平台 | 专业设备 | 申请使用 |
| 国家平台 | 大型设备 | 项目申请 |
| 区域平台 | 共享设备 | 有偿使用 |

**共享费用处理**：
- 科研项目经费支付
- 学校补贴政策
- 设备入股抵扣
- 有偿使用协议

#### 人才支持

**可对接人才**：
- 研究生：作为研发人员参与项目
- 本科生：作为实习生参与项目
- 博士后：作为核心研发人员
- 访问学者：作为技术顾问

**人才对接机制**：
- 研究生联合培养
- 本科生毕业设计
- 博士后工作站
- 兼职/顾问聘用

### 教育支持

#### 创新创业课程

**课程类型**：
| 类型 | 内容 | 形式 |
|------|------|------|
| 通识课 | 创新创业基础 | 课堂讲授 |
| 专业课 | 行业创业案例 | 案例教学 |
| 实践课 | 创业项目实践 | 项目驱动 |
| 工作坊 | 专题技能培训 | 实操训练 |

**课程对接价值**：
- 学习创业知识和方法
- 组建创业团队
- 获得导师指导
- 积累创业资源

#### 实践项目指导

**指导类型**：
- 学术导师：技术方向指导
- 企业导师：商业实践指导
- 创业导师：创业经验指导
- 投资导师：融资和战略指导

**导师对接方法**：
- 通过学校创业平台对接
- 通过院系推荐
- 通过校友网络
- 主动联系和拜访

#### 创业导师制度

**导师资源类型**：
| 类型 | 背景 | 指导重点 |
|------|------|----------|
| 学术导师 | 教授/研究员 | 技术方向 |
| 企业导师 | 企业高管 | 商业实践 |
| 投资导师 | 投资人 | 融资战略 |
| 行业导师 | 行业专家 | 行业洞察 |

## 二、平台层面资源对接（以清华x-lab为例）

### 教育功能

#### "三创"教育

**创意、创新、创业教育体系**：

| 阶段 | 内容 | 目标 |
|------|------|------|
| 创意 | 创意产生、问题发现 | 培养创新意识 |
| 创新 | 方案设计、原型开发 | 培养创新能力 |
| 创业 | 商业模式、团队组建 | 培养创业能力 |

#### 学生创新力提升证书课程

**课程模块**：
- 创新思维与方法
- 创业基础与实践
- 行业专题与创新应用
- 创业项目实训

**证书价值**：
- 系统学习创新创业知识
- 获得学校认可的证书
- 建立创业网络
- 获得导师资源

#### 创新工作坊

**工作坊类型**：
| 类型 | 时长 | 内容 |
|------|------|------|
| 设计思维工作坊 | 1天 | 创意产生、用户调研 |
| 商业模式工作坊 | 1天 | 商业模式设计 |
| 融资工作坊 | 0.5天 | BP撰写、融资技巧 |
| 路演工作坊 | 0.5天 | 路演技巧、投资人对接 |

### 培育功能

#### 驻校企业家和天使咨询

**驻校企业家（EiR）**：
- 背景：成功创业者或企业高管
- 角色：提供创业指导和建议
- 服务方式：定期办公时间、一对一咨询
- 价值：实战经验、人脉资源

**天使咨询服务**：
- 商业模式诊断
- 融资策略建议
- 团队问题诊断
- 资源对接

#### 赛课结合活动

**主要赛事**：
| 赛事 | 级别 | 价值 |
|------|------|------|
| 挑战杯 | 国家级 | 最高荣誉 |
| 互联网+ | 国家级 | 创业大赛 |
| 创青春 | 国家级 | 创业大赛 |
| 校内赛 | 校级 | 筛选和培育 |

**赛课结合模式**：
- 课程学习 → 项目实践 → 赛事检验 → 迭代优化

#### 项目孵化支持

**孵化服务**：
- 办公空间：免费或优惠工位
- 启动资金：种子基金支持
- 导师匹配：一对一导师指导
- 资源对接：投资、产业、政策对接

### 生态功能

#### 创新空间提供

**空间类型**：
| 类型 | 面积 | 服务对象 |
|------|------|----------|
| 开放工位 | 1-2个工位 | 早期团队 |
| 独立办公室 | 10-20平米 | 成型团队 |
| 会议室 | 共享使用 | 所有团队 |
| 活动空间 | 共享使用 | 所有团队 |

**入驻条件**：
- 团队成员为学生或教师
- 项目有创新性
- 有明确的创业意向
- 接受平台管理

#### 资源对接服务

**对接资源类型**：
| 资源类型 | 对接方式 | 服务频率 |
|----------|----------|----------|
| 投资机构 | 路演活动、一对一 | 每月 |
| 产业伙伴 | 对接会、项目合作 | 按需 |
| 政府资源 | 政策宣讲、项目申报 | 季度 |
| 服务机构 | 服务集市、优惠服务 | 定期 |

#### 国际化交流

**国际交流形式**：
- 海外创业考察
- 国际创业大赛
- 海外孵化器合作
- 国际创业者交流

## 三、社会层面资源对接

### 投资机构

#### 早期投资支持

**投资机构类型**：
| 类型 | 特点 | 适用阶段 |
|------|------|----------|
| 天使投资人 | 个人投资、决策快 | 种子期 |
| 早期VC | 机构投资、专业服务 | 天使轮/A轮 |
| 产业资本 | 战略投资、资源协同 | A轮及以后 |
| 政府基金 | 政策导向、优惠条件 | 各阶段 |

**投资对接方法**：
1. 参加路演活动（学校平台、创业大赛）
2. 熟人介绍（导师、校友、创业者）
3. 主动BP投递（官网、邮箱）
4. 社交媒体触达（LinkedIn、微信）

**投资人关注要点**：
- 团队背景和能力
- 技术壁垒和差异化
- 市场规模和增长潜力
- 商业模式可行性
- 竞争格局和护城河

#### 产业资源对接

**产业资本价值**：
- 市场渠道资源
- 技术合作机会
- 供应链资源
- 品牌背书

**产业资本对接注意**：
- 明确战略协同点
- 保护核心技术
- 设计合理的股权结构
- 明确业务合作边界

#### 战略指导

**投资人能提供的支持**：
- 战略方向建议
- 行业资源对接
- 管理团队招募
- 下一轮融资帮助

### 产业伙伴

#### 技术合作

**合作模式**：
| 模式 | 内容 | 价值 |
|------|------|------|
| 联合研发 | 共同开发新技术 | 分摊研发成本 |
| 技术授权 | 授权使用技术 | 获得授权收入 |
| 技术服务 | 提供技术咨询 | 获得服务收入 |
| 技术转让 | 出售技术所有权 | 获得转让收入 |

**合作对接方法**：
- 行业会议和展会
- 供应链上下游合作
- 学校产业合作平台
- 行业协会和联盟

#### 市场渠道

**渠道合作模式**：
| 模式 | 内容 | 适用场景 |
|------|------|----------|
| 代理合作 | 对方代理销售 | 区域市场拓展 |
| 渠道合作 | 对方提供渠道 | 快速上量 |
| 联合营销 | 共同市场推广 | 品牌建设 |
| OEM合作 | 为对方代工生产 | 产能利用 |

**渠道合作要点**：
- 明确权责和利益分配
- 设计合理的渠道政策
- 建立长期合作机制
- 保护自有品牌和客户

#### 应用场景

**场景对接价值**：
- 验证技术可行性
- 获得真实使用数据
- 建立标杆案例
- 形成口碑传播

**场景对接方法**：
- 主动联系潜在客户
- 提供免费试用机会
- 参与客户项目试点
- 建立联合创新机制

### 政府支持

#### 政策扶持

**主要政策类型**：
| 类型 | 内容 | 对接部门 |
|------|------|----------|
| 创业补贴 | 启动资金补贴 | 人社局 |
| 税收优惠 | 减免税、研发加计扣除 | 税务局 |
| 人才政策 | 人才引进、住房补贴 | 组织部/人社局 |
| 产业政策 | 专项资金、产业基金 | 发改委/工信局 |

**政策对接方法**：
- 关注政府官网和公众号
- 参加政策宣讲会
- 咨询创业服务平台
- 聘请专业顾问

#### 资金补贴

**常见补贴类型**：
| 补贴类型 | 金额范围 | 申请条件 |
|----------|----------|----------|
| 创业启动补贴 | 1-10万 | 新注册企业 |
| 高新企业补贴 | 10-50万 | 高新认定 |
| 研发补贴 | 项目金额的10-30% | 研发项目 |
| 人才补贴 | 5-50万/人 | 高层次人才 |

**申请技巧**：
- 提前了解政策和时间窗口
- 准备充分的申请材料
- 配合现场审核和答辩
- 关注后续验收要求

#### 税收优惠

**主要税收优惠**：
| 类型 | 内容 | 条件 |
|------|------|------|
| 高新企业优惠 | 15%企业所得税 | 高新认定 |
| 研发加计扣除 | 研发费用175%加计扣除 | 研发活动 |
| 技术转让免税 | 技术转让收入免税 | 技术转让 |
| 创业投资抵扣 | 投资额70%抵扣应纳税所得 | 投资于初创企业 |

## 输出格式

完成生态系统对接后，生成《师生共创资源对接方案》，包含：
1. 已有资源盘点
2. 待对接资源清单
3. 对接优先级排序
4. 对接行动计划（时间表、责任人）
5. 资源整合策略

## 实用建议

- 资源对接要趁早，不要等到需要时才去找
- 建立长期合作关系，而非一次性利用
- 善用学校和平台的背书效应
- 资源是"用出来"的，不是"等出来"的
- 在资源对接中思考"我能为对方提供什么价值"
