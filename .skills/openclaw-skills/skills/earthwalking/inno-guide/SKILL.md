# 创赛指南

## Overview

本技能为中国国际大学生创新大赛（2025）参赛者提供全流程指导服务，包括：
1. 根据项目类型和赛道提供针对性参赛建议
2. 解读各赛道评审规则和评分要点
3. 整合PPT制作能力，为参赛项目生成专业参赛PPT

## 赛道类型识别

根据教育部通知，大赛设以下赛道：

### 1. 高教主赛道
- **创意组**：面向本专科生，尚未注册公司但有创意方案
- **创业组**：面向本专科生/研究生，已注册公司运营

### 2. 青年红色筑梦之旅赛道
- **公益组**：非营利性社会创新项目
- **创意组**：文旅或其他产业创新项目
- **创业组**：乡村振兴相关创业项目

### 3. 产业赛道
- **企业命题组**：解决企业实际技术需求
- **成果转化组**：已有科研成果转化

### 4. 萌芽赛道
- 面向高中生，科技创新创意项目

## 工作流程

### 步骤1：赛道确认

询问用户以下信息以确定参赛赛道：
1. 项目类型（创意/创业/公益/成果转化）
2. 团队成员构成（本科生/研究生/高中生）
3. 项目所属行业领域
4. 是否已有公司实体

### 步骤2：项目分析

根据用户提供的项目信息，分析：
1. 项目所属赛道是否合适
2. 项目的核心竞争力
3. 需要重点准备的材料

### 步骤3：针对性建议

根据评审要点，提供具体建议：
1. **个人成长维度**：展示项目如何体现产教融合、科教融汇
2. **项目创新维度**：明确创新点（产品/工艺/服务/商业模式）
3. **产业价值维度**：分析市场规模、竞争格局、商业模式
4. **团队协作维度**：展示团队分工、资源整合能力

### 步骤4：PPT制作

使用pptx技能，为参赛项目生成专业PPT。

#### 调用pptx技能

当用户请求制作参赛PPT时，执行以下步骤：

1. **加载pptx技能**：调用 `use_skill` 工具加载 pptx 技能
2. **收集项目信息**：向用户获取项目详细信息
3. **生成PPT结构**：根据赛道特点确定PPT内容结构
4. **生成PPT内容**：使用pptx技能创建PPT

#### 典型参赛PPT结构

根据不同赛道，PPT结构有所调整：

**高教主赛道-创意组（10-12页）**：
1. 封面页：项目名称、团队信息、参赛赛道
2. 项目概述：解决的问题、解决方案、核心价值
3. 创新点分析：技术创新/模式创新/服务创新
4. 市场分析：市场规模、目标客户、竞争分析
5. 商业模式：盈利模式、营销策略
6. 团队介绍：成员背景、核心能力、分工协作
7. 落地进展：原型/产品、测试数据、用户反馈
8. 未来规划：发展目标、资源需求
9. 结尾页：联系方式、感谢语

**高教主赛道-创业组（12-15页）**：
在前述基础上增加：
- 企业概况：公司基本信息、核心业务
- 经营绩效：营收、利润、市场份额
- 财务预测：营收预测、成本分析、融资计划
- 创新成果：专利、技术突破、行业认可

**青年红色筑梦之旅赛道**：
- 增加"传承红色基因"专题页
- 强调社会价值和社会影响
- 突出公益属性和实践成果

**产业赛道**：
- 强调命题解决思路
- 展示技术方案可行性
- 突出与企业命题的契合度

## 参赛建议模板

### 创意组建议框架

```
1. 项目定位
   - 解决什么社会/行业痛点
   - 目标用户群体是谁

2. 创新亮点
   - 技术创新：核心技术优势
   - 模式创新：独特的商业模式
   - 服务创新：差异化服务体验

3. 可行性论证
   - 市场调研数据
   - 原型/测试结果
   - 团队能力支撑

4. 商业价值
   - 目标市场规模
   - 盈利模式
   - 成长潜力
```

### 创业组建议框架

```
1. 企业概况
   - 公司基本信息
   - 核心业务
   - 发展阶段

2. 经营绩效
   - 营收/订单情况
   - 市场份额
   - 客户案例

3. 创新成果
   - 专利/软著
   - 技术突破
   - 行业认可

4. 发展战略
   - 市场拓展计划
   - 融资规划
   - 竞争优势
```

## 使用示例

### 示例1：赛道咨询

用户：我想参加创新大赛，但不知道选哪个赛道

回复：
1. 请问您的项目目前处于什么阶段？（有创意方案/已有公司运营）
2. 项目是做什么方向的？（科技创新/社会创新/乡村振兴）
3. 团队成员是什么学历层次？

根据回答，推荐合适赛道并说明理由。

### 示例2：PPT制作

用户：帮我制作参赛PPT

回复：
请提供以下信息：
1. 项目名称和核心定位
2. 所属赛道和组别
3. 项目创新点（技术/模式/服务）
4. 目标市场和商业模式
5. 团队成员构成
6. 目前项目进展（原型/产品/已有收入）

收集信息后，使用pptx技能生成PPT。
