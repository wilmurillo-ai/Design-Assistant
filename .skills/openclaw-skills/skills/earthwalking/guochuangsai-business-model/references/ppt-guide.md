## 1. PPT商业模式板块推荐结构

### 通用结构（5个模块）

| 模块 | 核心内容 | 建议页数 |
|------|---------|---------|
| **产业分析** | 市场规模、增速、竞争格局、产业趋势、政策支持 | 1-2页 |
| **目标市场** | 客户细分、用户画像、TAM/SAM/SOM、痛点分析 | 1页 |
| **商业模式** | 价值主张、收入模式、渠道通路、客户关系、BMC画布 | 1-2页 |
| **竞争分析** | 竞品对比、核心壁垒、SWOT分析 | 1页 |
| **发展规划** | 发展路径、财务预测（3年）、融资需求 | 1页 |

### 创意组PPT结构（10-12页总）

```
1. 封面页
2. 项目概述（痛点+解决方案）
3. 创新点分析
4. 产业分析（规模、趋势、政策）    ← 产业价值
5. 目标市场与用户画像              ← 产业价值
6. 商业模式（BMC画布+收入模式）    ← 产业价值
7. 竞争分析与壁垒                  ← 产业价值
8. 落地进展与社会影响              ← 产业价值
9. 团队介绍
10. 未来规划
11. 致谢页
```

### 创业组PPT结构（12-15页总）

```
1. 封面页
2. 企业概况
3. 项目概述（痛点+解决方案）
4. 创新点与创新成果
5. 产业分析                        ← 产业价值
6. 目标市场与商业模式              ← 产业价值
7. 经营绩效（营收、利润、客户）     ← 产业价值
8. 竞争分析与壁垒                  ← 产业价值
9. 财务规划与融资                  ← 产业价值
10. 发展战略                       ← 产业价值
11. 社会影响（就业、区域发展）      ← 产业价值
12. 团队介绍
13. 致谢页
```

## 3. 商业模式板块写作技巧

### 3.1 用数据说话

| 原则 | 具体做法 |
|------|---------|
| 市场规模量化 | "千亿级市场，CAGR 25%" 而非 "市场很大" |
| 财务预测表格化 | 用表格展示3年营收预测 |
| 用户调研可视化 | 用图表展示调研结果 |
| 数据来源标注 | 引用报告名称和发布年份 |

### 3.2 讲好商业故事

- 用一个典型案例或场景引入
- "假设你是XX行业的从业者..."
- 从痛点→方案→价值的逻辑链
- 让评委代入用户视角

### 3.3 展示验证成果

| 证据类型 | 呈现方式 |
|---------|---------|
| MVP测试数据 | 用户增长曲线、留存率数据 |
| 用户反馈 | 用户评价截图、NPS评分 |
| 试点合作 | 合作协议、合作方logo |
| 营收证明 | 订单截图、合同摘要 |

### 3.4 突出创新亮点

- 商业模式画布作为核心可视化工具
- 竞品对比用表格或矩阵
- 收入模式创新点用图表高亮
- 专利/技术壁垒列表展示

### 3.5 逻辑链清晰

```
市场痛点 → 解决方案 → 核心价值 → 收入模式 → 竞争壁垒 → 财务预测 → 社会影响
```

每个环节之间要有逻辑递进关系，不能跳跃。

## 4. 常见错误与规避方法

| 错误 | 表现 | 正确做法 |
|------|------|---------|
| 市场太空 | "面向全球XX市场" | 聚焦具体细分市场，用TAM/SAM/SOM |
| 模式不清 | 说不清楚怎么赚钱 | 明确收入模式，标注收费对象和方式 |
| 闭门造车 | 没有用户调研 | 展示调研过程和用户反馈数据 |
| 财务随意 | "预计营收1个亿" | 展示测算逻辑和关键假设 |
| 忽视竞品 | "我们没有竞争对手" | 用对比矩阵展示差异化优势 |
| 重技术轻商业 | 花80%时间讲技术 | 商业模式板块不少于总篇幅30% |
| 数据无来源 | "据不完全统计" | 标注具体报告来源 |
| 社会影响空话 | "助力乡村振兴" | 量化：带动X人就业，增加Y收入 |

## 5. PPT设计规范

### 排版原则

- **每页一个核心观点**，不超过5个要点
- **数据用图表**而非纯文字
- **关键数据**用大字号+强调色高亮
- **逻辑链**用流程图或箭头串联
- **页面风格**保持统一（配色、字体、版式）

### 视觉元素

| 元素 | 用途 | 建议 |
|------|------|------|
| 商业模式画布 | 展示整体商业模式 | 作为商业模式页核心图表 |
| 竞品对比矩阵 | 展示竞争优势 | 用表格或雷达图 |
| 漏斗图 | 展示市场规模 | TAM→SAM→SOM |
| 柱状图 | 展示财务预测 | 3年营收/利润增长 |
| 饼图 | 展示成本/收入结构 | 各项占比 |
| 时间线 | 展示发展规划 | 里程碑和关键节点 |
| Logo墙 | 展示合作伙伴 | 已签约伙伴的logo |

### 配色建议

- **专业稳重**：深蓝+灰+红（商务风）
- **科技感**：深蓝+青绿+白（科技风）
- **创新活力**：蓝紫+橙+白（创意风）
- 避免超过3种主色调
