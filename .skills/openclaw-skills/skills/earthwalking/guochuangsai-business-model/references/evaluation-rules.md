# 国创赛各赛道评审规则（商业模式相关）

> 本文件包含各赛道完整评审要点，重点关注与商业模式相关的"产业价值""发展前景"等维度。

---

## 一、高教主赛道·创意组

总分100分，商业模式相关占**25分**（产业价值）。

### 产业价值（25分）

| 子项 | 评审内容 |
|------|---------|
| **产业认知** | 对所在产业的产业规模、增长速度、竞争格局、产业趋势、产业政策等情况充分了解，形成完备、深刻的产业认知 |
| **市场定位** | 项目具有明确的目标市场定位，对目标市场的特征、需求等情况有清晰的了解，并据此制定合理的营销、运营、财务等计划，设计出完整、创新、可行的商业模式，展现团队的商业思维 |
| **落地前景** | 项目落地执行情况，对促进区域经济发展、产业转型升级的情况；已有盈利能力或盈利潜力情况。项目是否具备国际化发展的潜力 |
| **社会影响** | 项目直接或间接带动就业的数量和质量，对社会文明、生态文明、民生福祉等方面的积极推动作用 |

### 其他维度（简要）

- **个人成长（30分）**：立德树人、调研深入、逻辑正确、知识掌握与应用能力、人才培养成效
- **项目创新（30分）**：问题导向、目标导向、创新成效（产品/工艺流程/服务/商业模式创新）
- **团队协作（15分）**：团队精神、团队结构、团队效能、团队资源

## 二、高教主赛道·创业组

总分100分，商业模式相关占**30分**（产业价值）。

### 产业价值（30分）

| 子项 | 评审内容 |
|------|---------|
| **产业发展** | 充分掌握所在产业的产业规模、增长速度、竞争格局、产业趋势、产业政策等情况；具有明确的目标市场定位，充分掌握目标市场的特征、需求等情况；具有完整、创新、可行的商业模式 |
| **经营绩效** | 重点考察项目存续时间、营业收入（合同订单）现状、企业利润、持续盈利能力、市场份额、客户（用户）情况、税收上缴、投入与产出比等情况 |
| **经营管理** | 项目是否有清晰的企业发展目标；是否有完备的研发、生产、运营、营销等制度和体系；是否采用先进、科学的管理方法，以确保企业具有较强的竞争力 |
| **成长前景** | 项目是否有清晰、有效、全方位的企业发展战略，并拥有可靠的内外部资源（人才、资金、技术等方面）实现企业战略，以建立企业的持续竞争优势。项目是否具备国际化发展的潜力 |
| **财务管理** | 关注项目融资情况、获取资金渠道情况、企业经营的现金流情况、融资需求及资金使用情况是否合理 |
| **社会影响** | 项目直接或间接带动就业的数量和质量，对促进区域经济发展、产业转型升级、社会文明、生态文明、民生福祉等方面的积极推动作用 |

### 其他维度（简要）

- **个人成长（25分）**：立德树人、调研深入、逻辑正确、知识掌握与应用能力、人才培养成效
- **项目创新（30分）**：问题导向、目标导向、创新成效、发展前景（创新战略/流程/组织/制度与文化）
- **团队协作（15分）**：团队精神、团队结构、团队效能、团队资源
