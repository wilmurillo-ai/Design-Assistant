# 挑战杯小挑参赛指南

## Overview

本技能为"挑战杯"中国大学生创业计划竞赛（第十五届，2026年）参赛者提供全流程指导服务，包括：
1. 根据项目类型和赛道提供针对性参赛建议
2. 解读评审规则和五大评分维度
3. 整合PPT制作能力，为参赛项目生成专业参赛PPT

## 赛事概况

- **全称**："挑战杯"中国大学生创业计划竞赛（小挑）
- **举办周期**：偶数年举办（2026年第十五届）
- **主办单位**：共青团中央、教育部、中国科学技术协会等
- **参赛对象**：全日制非成人教育的各类高等院校在校专科生、本科生、硕士研究生（不含在职研究生）

## 赛道分类

根据第十五届小挑，参赛项目分为以下五大赛道：

### 1. 科技创新和未来产业
- 围绕人工智能、量子信息、集成电路、生命健康、脑科学、空天科技、深地深海等前沿领域
- 具有前瞻性、创新性的技术和产品

### 2. 乡村振兴和农业农村现代化
- 农业科技、乡村旅游、农产品电商
- 农村振兴解决方案、智慧农业

### 3. 城市治理和社会服务
- 智慧城市、公共服务、社区治理
- 教育、医疗、养老等社会服务创新

### 4. 生态环保和可持续发展
- 绿色技术、清洁能源、环境保护
- 碳中和、循环经济、可持续发展

### 5. 文化创意和区域合作
- 文化创新、非遗传承、文化数字化
- 国际合作、区域协调发展

## 评审五大维度

小挑评审围绕五大核心维度，总分100分：

| 评审维度 | 分值 | 核心要点 |
|---------|------|---------|
| **创新意义** | 30分 | 技术创新、模式创新、产品/服务创新 |
| **社会价值** | 25分 | 社会意义、可落地性、促进就业 |
| **实践过程** | 20分 | 调研情况、阶段性成果、原型测试 |
| **发展前景** | 15分 | 市场规模、盈利模式、竞争优势 |
| **团队协作** | 10分 | 团队构成、分工协作、能力互补 |

## 各维度详细解读

### 1. 创新意义（30分）

**评审要点**：
- 技术创新：核心技术的先进性、独特性、专利布局
- 模式创新：商业模式的创新性、可复制性、可持续性
- 产品/服务创新：差异化竞争优势、市场适应性

**备赛建议**：
- 明确项目的核心创新点是什么
- 展示技术壁垒和难以复制的优势
- 提供专利、论文、查新报告等支撑材料

### 2. 社会价值（25分）

**评审要点**：
- 项目对社会的积极贡献
- 解决社会痛点/市场需求
- 促进就业、带动产业发展
- 可落地性和可推广性

**备赛建议**：
- 深入调研社会痛点，展示项目的社会意义
- 量化社会效益（如带动就业数量、服务人群规模）
- 展示项目的可复制性和推广价值

### 3. 实践过程（20分）

**评审要点**：
- 深入社会、行业、实地调研
- 项目已有进展和阶段性成果
- 产品/服务原型及测试情况
- 用户反馈和市场验证

**备赛建议**：
- 展示完整的市场调研过程和数据
- 提供产品原型、设计图、测试报告
- 展示用户反馈、试点运行数据
- 强调"从0到1"的实践过程

### 4. 发展前景（15分）

**评审要点**：
- 市场规模和目标客户
- 盈利模式和财务预测
- 竞争优势和市场壁垒
- 未来发展规划和成长空间

**备赛建议**：
- 进行详细的市场分析，展示TAM/SAM/SOM
- 展示清晰的盈利模式和收入结构
- 分析竞争格局，突出差异化优势
- 提供3-5年的财务预测和发展规划

### 5. 团队协作（10分）

**评审要点**：
- 团队成员的互补性
- 分工明确、协作顺畅
- 团队执行力和发展潜力
- 团队成员的背景和经验

**备赛建议**：
- 展示团队成员的专业背景和能力互补
- 明确团队分工和协作机制
- 展示团队已取得的成果和进展

## 工作流程

### 步骤1：赛道确认

询问用户以下信息以确定参赛赛道：
1. 项目所属领域（科技/农业/城市/环保/文化）
2. 项目核心创新点
3. 项目的实践进展阶段
4. 团队成员构成

### 步骤2：项目分析

根据用户提供的项目信息，分析：
1. 项目所属赛道是否合适
2. 项目的核心竞争力和亮点
3. 需要重点准备的材料
4. 五大维度的得分潜力

### 步骤3：针对性建议

根据五大评审维度，提供具体建议：
- **创新意义**：明确技术创新点，准备专利等支撑材料
- **社会价值**：量化社会效益，展示可推广性
- **实践过程**：补充调研数据和原型测试
- **发展前景**：完善市场分析和财务预测
- **团队协作**：展示团队互补性和分工

### 步骤4：PPT制作

使用pptx技能，为参赛项目生成专业PPT。

#### 调用pptx技能

当用户请求制作参赛PPT时，执行以下步骤：

1. **加载pptx技能**：调用 `use_skill` 工具加载 pptx 技能
2. **收集项目信息**：向用户获取项目详细信息
3. **生成PPT结构**：根据赛道特点确定PPT内容结构
4. **生成PPT内容**：使用pptx技能创建PPT

#### 典型参赛PPT结构（8-12页）

**封面页**：项目名称、团队信息、参赛赛道

**项目概述**（1-2页）：
- 解决的问题和痛点
- 解决方案和核心价值
- 项目定位和目标市场

**创新亮点**（2-3页）：
- 技术创新：核心技术优势、专利布局
- 模式创新：独特商业模式
- 产品/服务创新：差异化特点

**社会价值**（1-2页）：
- 解决的社会问题
- 预期社会效益
- 可推广性和复制性

**市场分析**（1-2页）：
- 市场规模（TAM/SAM/SOM）
- 目标客户画像
- 竞争分析和竞争优势

**商业模式**（1-2页）：
- 盈利模式
- 定价策略
- 营销策略
- 财务预测

**实践进展**（1-2页）：
- 调研过程和成果
- 产品/服务原型
- 测试数据和用户反馈
- 阶段性成果

**团队介绍**（1页）：
- 团队成员背景
- 核心能力
- 分工协作

**未来规划**（1页）：
- 发展目标
- 里程碑计划
- 资源需求

**结尾页**：联系方式、感谢语

## 参赛建议模板

### 科技创新赛道建议框架

```
1. 技术创新
   - 核心技术是什么
   - 技术领先性（对比现有方案）
   - 专利布局和知识产权

2. 产品定位
   - 目标用户群体
   - 解决的核心需求
   - 产品/服务形态

3. 市场前景
   - 技术成熟度
   - 市场化路径
   - 潜在市场规模
```

### 乡村振兴赛道建议框架

```
1. 乡村问题洞察
   - 调研过程和方法
   - 发现的痛点问题
   - 解决的必要性

2. 解决方案
   - 针对乡村的具体方案
   - 模式的可行性
   - 已有实践案例

3. 社会价值
   - 带动就业
   - 促进农民增收
   - 可复制推广性
```

### 城市治理赛道建议框架

```
1. 城市痛点
   - 城市管理中的问题
   - 居民需求调研
   - 现有解决方案的不足

2. 创新方案
   - 技术/模式创新点
   - 方案可行性
   - 试点效果

3. 推广价值
   - 复制推广条件
   - 规模化潜力
   - 社会效益
```
