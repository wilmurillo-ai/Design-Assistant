# 14页标准PPT模板

## 模板使用说明

本模板提供14页标准融资路演PPT的页面布局建议和内容框架。根据项目类型（硬科技/消费互联网/社会企业等）适当调整。

---

## 第1页：使命页 (Mission)

### 布局建议
- 全屏背景图（与行业相关）
- 中央：公司Logo + 一句话使命宣言
- 底部：创始人姓名/日期

### 内容模板

```
[公司Logo]

一句话使命宣言：
"[具体行动] + [目标对象] + [核心价值]"

示例：
"将纺织废料转化为高韧性纤维，减少时尚行业碳排放"
"利用人工智能可持续地养活世界，最大限度地减少气候变化的影响"
```

### 设计要点
- 字体：大号（48-60pt），简洁有力
- 配色：公司主色
- 避免：过于空泛的表述（如"改变世界"）

---

## 第2页：问题页 (Problem)

### 布局建议
- 左侧：痛点数据可视化（图表/大数字）
- 右侧：1-3个核心问题描述
- 底部：客户引言

### 内容模板

```
行业痛点

[大数字] - 问题规模量化
  示例：全球发电行业每年水资源支出750亿美元

核心问题：
1. [问题1] - 具体描述
2. [问题2] - 具体描述  
3. [问题3] - 具体描述（可选）

客户引言：
"[真实客户的话，强调痛点紧迫性]"
- [客户姓名]，[职位]，[公司]
```

### 设计要点
- 使用图标增强视觉
- 数据要具体可信
- 叙事：从宏观到微观

---

## 第3页：解决方案页 (Solution)

### 布局建议
- 中央：解决方案架构图
- 四周：2-3个核心用例
- 底部：核心价值主张

### 内容模板

```
我们的解决方案

[解决方案名称]

核心创新点：
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  [输入]     │ →  │  [处理]     │ →  │  [输出]     │
│  [描述]     │    │  [AI/技术]  │    │  [价值]     │
└─────────────┘    └─────────────┘    └─────────────┘

用例1：[场景1] → [解决的问题] → [带来的价值]
用例2：[场景2] → [解决的问题] → [带来的价值]
用例3：[场景3] → [解决的问题] → [带来的价值]

核心价值：[一句话总结]
```

### 设计要点
- 流程图清晰展示输入-处理-输出
- 强调AI/技术的独特价值
- 使用对比色突出关键信息

## 第6页：市场时机页 (Why Now?)

### 布局建议
- 时间轴设计：过去 → 现在 → 未来
- 关键突破点高亮显示
- 底部：为什么我们已经准备好

### 内容模板

```
为什么是现在？

过去：[传统解决方式]
      • [方式1] - [局限性]
      • [方式2] - [局限性]

关键突破：[技术/政策/市场变革]
      • [突破点1]
      • [突破点2]
      • [突破点3]

现在：[新机会的出现]
      • [机会1]
      • [机会2]

我们已经准备好：
      ✓ [准备1]
      ✓ [准备2]
      ✓ [准备3]
```

### 设计要点
- 使用时间轴增强叙事
- 突出"突破性"时刻
- 证明时机恰到好处

## 第7页：市场规模页 (Market Size)

### 布局建议
- 左侧：自下而上计算过程
- 右侧：TAM/SAM/SOM金字塔图
- 底部：滩头阵地市场聚焦

### 内容模板

```
市场规模分析

自下而上计算：
目标市场终端用户：[X,XXX] 家
每家年营收贡献：$[XXX,XXX]
滩头阵地市场 = $[XX]M

市场层级：
      ┌─────────────┐
      │    TAM      │ $[X]B
      │  [总市场]    │
      ├─────────────┤
      │    SAM      │ $[X]B
      │  [可服务]    │
      ├─────────────┤
      │    SOM      │ $[X]M
      │  [可获得]    │
      └─────────────┘

滩头阵地市场：[具体描述]
• 目标客户：[描述]
• 市场需求：[证据]
• 扩展路径：[如何扩展到更大市场]
```

### 设计要点
- 展示计算逻辑，不只是数字
- 使用金字塔或漏斗图
- 强调滩头阵地市场的可攻击性

## 第8页：竞争分析页 (Competition)

### 布局建议
- 中央：2x2竞争矩阵
- 四周：主要竞争对手标识
- 底部：我们的差异化优势

### 内容模板

```
竞争格局

                    高
    ┌─────────────────────────────┐
    │                             │
高  │  [竞品A]      ★我们         │  解决
    │                             │  痛点
    │  [竞品B]                    │  能力
    │                             │
    ├─────────────────────────────┤
    │                             │
    │  [竞品C]      [竞品D]       │
低  │                             │  低
    └─────────────────────────────┘
         低        技术能力        高

主要竞争对手：
• [竞品A]：[特点] vs [我们的优势]
• [竞品B]：[特点] vs [我们的优势]

我们的护城河：
1. [壁垒1]
2. [壁垒2]
3. [壁垒3]
```

### 设计要点
- 诚实定位，不要虚假领先
- 选择客户真正关心的维度
- 清晰展示差异化优势

## 第14页：结尾页 (Closing)

### 布局建议
- 全屏背景图（与使命页呼应）
- 中央：强力收尾语
- 底部：联系方式和二维码

### 内容模板

```
[公司Logo]

"[强力收尾语，呼应使命]"

示例：
"我们相信，通过AI技术，我们能够为全球电力行业
节省百亿美元成本。期待与您携手，共同实现这一愿景。"

联系我们：
[姓名] - [职位]
📧 [邮箱]
📱 [电话]
🌐 [网站]

[LinkedIn二维码]

谢谢！
```

### 设计要点
- 与开场页呼应，形成闭环
- 高能量收尾，留下深刻印象
- 二维码便于快速连接
