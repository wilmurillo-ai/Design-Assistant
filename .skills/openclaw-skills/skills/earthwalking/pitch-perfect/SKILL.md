# Pitch Perfect - 创业融资路演PPT专家

## 描述

基于Oliver Pitch Course、MIT创业课程及实战案例的系统化创业融资路演PPT制作指南。帮助创业者制作专业、有说服力的融资路演PPT，提高融资成功率。

## 标签

pitch, fundraising, startup, venture-capital, presentation, entrepreneurship, deck, investment

## 工具

无需特殊工具，使用自然语言对话即可。

## 工作流

当用户需要制作创业融资路演PPT时，执行以下系统化流程：

### 步骤1：明确路演背景

首先了解用户的基本情况：

1. **路演类型**：
   - 融资路演（投资人办公室，深度沟通）
   - 展示路演（公开舞台，表演性质）

2. **融资阶段**：
   - 种子轮（15-16页，侧重使命、团队、产品原型）
   - A轮（17-19页，侧重商业模式、市场验证、财务预测）

3. **项目类型**：
   - AI/硬科技项目
   - 消费互联网项目
   - 企业服务/B2B项目
   - 社会企业/气候科技项目

4. **目标受众**：
   - VC风险投资机构
   - 天使投资人
   - 企业战略投资
   - 政府引导基金

### 步骤2：核心原则指导

向用户传达九大核心原则：

| 原则 | 关键要点 |
|------|---------|
| **简洁** | 30秒内抓住注意力，每页一个核心要点 |
| **独特性** | 提炼独特的商业故事与价值主张 |
| **逻辑** | 清晰阐述观点、背景及重要性 |
| **热情** | 用真实激情感染听众 |
| **信心** | 展现团队的专业与自信 |
| **品牌** | 保持视觉与色彩的一致性 |
| **相关性** | 契合投资人关注点 |
| **跟进** | 明确下一步行动 |
| **避免冷邮件** | 通过熟人引荐建立信任 |

**关键警告**：永远不要超过20页！50-60页的PPT会被直接删除。

### 步骤3：14页标准结构规划

根据用户情况，规划PPT的14页标准结构：

```
1. 使命页 (Mission)
2. 问题页 (Problem)
3. 解决方案页 (Solution)
4. 产品页 (Product)
5. 技术/IP页 (Technology)
6. 市场时机页 (Why Now?)
7. 市场规模页 (Market Size)
8. 竞争分析页 (Competition)
9. 商业模式页 (Business Model)
10. 团队页 (Team)
11. 业务进展页 (Traction)
12. 财务预测页 (Financials)
13. 融资需求页 (Fundraising Ask)
14. 结尾页 (Closing)
```

根据项目类型调整：
- **硬科技项目**：团队页前置（第2-3页）
- **故事型项目**：团队页后置（第10页）

### 步骤4：逐页内容设计

为每一页提供详细的内容框架和设计建议：

#### 第1页：使命页
- 一句话定义：你是谁、做什么、为客户带来什么
- 具体而非空泛（❌"解决世界饥饿" → ✅"将纺织废料转化为高韧性纤维，减少时尚行业碳排放"）
- AI初创：明确AI在解决方案中的核心作用

#### 第2页：问题页
- 聚焦1-3个核心问题
- 用数据量化问题规模
- 加入客户引言增强真实感
- 叙事技巧：从宏观危机到微观痛点

#### 第3页：解决方案页
- 展示创新方法如何连接问题与产品
- 2-3个技术用例
- AI带来的具体价值（降低成本/提高效率/缩短周期）

#### 第4页：产品页
- 流程图：输入 → 处理 → 输出
- 说明产品如何集成到客户工作流程
- 展示用户界面或原型

#### 第5页：技术/IP页
- 已申请专利清单
- 技术壁垒和防御性优势
- 未来2-3年技术路线图

#### 第6页：市场时机页
- 过去如何解决 → 关键突破点 → 新机会 → 我们已准备好
- 证明为什么不是太早也不是太晚

#### 第7页：市场规模页
- ❌ 错误：自上而下引用麦肯锡报告
- ✅ 正确：自下而上计算（终端用户数量 × 年营收贡献）
- 聚焦滩头阵地市场（$50M-$150M）

#### 第8页：竞争分析页
- 2x2竞争矩阵
- 诚实定位，展示差异化优势
- 选择客户真正关心的对比维度

#### 第9页：商业模式页
- 定价模型：成本导向 vs 价值导向
- 收入模式：SaaS/产品销售/订阅
- 关键指标：LTV/CAC > 3，回收期 < 12个月

#### 第10页：团队页
- 核心创始人（技术+商业双背景）
- 顾问团队
- 关键岗位招聘计划

#### 第11页：业务进展页
- 最有价值：销售收入 > 客户渠道 > 试点项目 > 意向书
- 其他：论文、媒体报道、行业奖项

#### 第12页：财务预测页
- 3-5年收入/成本/利润预测
- 关键假设说明
- 技术经济分析（硬科技必备）

#### 第13页：融资需求页
- 融资金额、出让股权、投前估值
- 资金用途分配
- 具体可衡量的里程碑

#### 第14页：结尾页
- 强力收尾语
- 联系方式和LinkedIn二维码
- 明确的行动号召

### 步骤5：叙事与情感设计

指导用户设计情感节拍：

| 环节 | 期望听众情绪 |
|------|-------------|
| 阐述问题 | 愤怒或震惊 |
| 展示方案 | 惊叹 |
| 介绍团队 | 信任 |
| 展示数据 | 信服 |
| 结尾号召 | 行动欲望 |

**推荐叙事结构**（漏斗式）：
```
全球危机 → 行业痛点 → 具体人物 → 解决方案 → 量化价值
```

### 步骤6：视觉设计指导

提供视觉设计建议：

**颜色选择**：
- 主色占60-70%，1-2个辅助色，1个强调色
- 深色背景用于标题/结尾页，浅色用于内容页

**布局原则**：
- 每页必须有视觉元素（图片/图表/图标）
- 避免纯文字页
- 使用大数字突出关键数据

**双版本策略**：
- **阅读版**：会前发送，包含足够文字上下文
- **演讲版**：现场演示，精简文字，依赖口述

### 步骤7：演讲技巧指导

提供演讲准备建议：

1. **保持真实激情**：像向朋友推荐最喜欢的东西
2. **极端重视练习**：台上几分钟，台下数十次演练
3. **时态选择**：尽量用现在时（We do）而非将来时（We will）
4. **视觉化表达**：能用图就别用字
5. **备战问答**：准备FAQ备份幻灯片

### 步骤8：整合PPT制作

当用户需要生成PPT文件时，调用pptx技能：

1. 使用 `use_skill` 工具加载 pptx 技能
2. 根据上述规划的内容结构
3. 生成专业的融资路演PPT

## 示例对话

**用户**：我要做一个种子轮融资的PPT，做AI客服机器人

**助手**：
1. 首先确认这是融资路演，种子轮，AI项目
2. 规划14页结构，团队页建议后置（AI应用型项目）
3. 重点突出：
   - 使命页：明确AI如何解决客服痛点
   - 问题页：传统客服成本高、响应慢、服务质量不稳定
   - 解决方案页：AI带来的效率提升和成本降低
   - 技术页：NLP能力、多语言支持、持续学习能力
   - 商业模式：SaaS订阅，按坐席数或对话量收费
4. 提供每页的具体内容建议
5. 最后询问是否需要生成PPT文件

## 注意事项

1. **页数控制**：严格控制在20页以内
2. **数据真实性**：所有数据必须有来源和假设支撑
3. **诚实定位**：竞争分析要诚实，VC会做调研
4. **平衡投入**：平衡技术投入和商业投入
5. **持续迭代**：根据投资人反馈不断优化
