### 方式 2：使用 Token 直接 API 上传

如果 Clawhub 提供了 API 上传端点，可以使用以下 PowerShell 脚本：

```powershell
$token = "clh_BXhiYXSolilafAFobvo269pTv60C02vOTPb1JbHPYIg"
$zipPath = "C:/Users/13600/.openclaw/workspace/clawhub-skill-search.zip"

# 读取 ZIP 文件
$file = [System.IO.File]::ReadAllBytes($zipPath)

# 创建 multipart/form-data 请求
$boundary = "----WebKitFormBoundary" + [System.Guid]::NewGuid().ToString("N")
$contentType = "multipart/form-data; boundary=$boundary"

# 构建请求体
$body = [System.Text.Encoding]::UTF8.GetBytes("--$boundary`r`n")
$body += [System.Text.Encoding]::UTF8.GetBytes("Content-Disposition: form-data; name=`"file`"; filename=`"clawhub-skill-search.zip`"`r`n")
$body += [System.Text.Encoding]::UTF8.GetBytes("Content-Type: application/zip`r`n`r`n")
$body += $file
$body += [System.Text.Encoding]::UTF8.GetBytes("`r`n--$boundary--`r`n")

# 设置请求头
$headers = @{
    "Authorization" = "Bearer $token"
}

# 发送请求（需要确认正确的 API 端点）
try {
    $response = Invoke-RestMethod -Uri "https://clawhub.ai/api/v1/skills" -Method Post -Headers $headers -ContentType $contentType -Body $body
    Write-Host "发布成功！" -ForegroundColor Green
    Write-Host "技能 URL: $($response.skill.url)" -ForegroundColor Green
} catch {
    Write-Host "发布失败：$($_.Exception.Message)" -ForegroundColor Red
}
```

**注意**: 需要确认 Clawhub 的正确 API 上传端点。当前尝试的 `/api/skills/upload` 返回 404。
