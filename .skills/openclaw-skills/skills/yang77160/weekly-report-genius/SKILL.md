# 周报大神 (Weekly Report Genius)

> **拒绝加班！30秒生成高大上周报**

## 🎯 核心功能

- **一键润色**：把你的大白话自动转换成“互联网黑话”和“职场专业术语”。
- **结构优化**：自动生成【本周进展】、【存在问题】、【下周计划】三大板块。
- **高情商话术**：自动补充团队协作、自我反思等加分项。

## 🚀 如何使用

在 OpenClaw 中直接对话：

**用户输入：**
> 帮我写个周报。本周修了登录界面的 Bug，跟产品对了需求，下周准备做支付功能。

**“周报大神”输出：**

> **本周工作总结**
> 1. **系统稳定性优化**：针对登录模块存在的潜在逻辑漏洞进行了深度排查与修复，显著提升了用户入口的稳定性与安全性。
> 2. **跨部门协同推进**：与产品团队完成深度对齐，明确了下一阶段业务迭代的核心需求，为后续开发扫清障碍。
>
> **下周工作计划**
> 1. **核心业务攻坚**：启动支付模块的功能设计与开发，确保交易链路的顺畅与资金安全。
> 2. **技术预研**：针对支付环节可能涉及的高并发场景进行前期技术调研。

## 💡 进阶技巧

- **指定风格**：你可以说“请用更谦虚的语气”或“请突出我的个人贡献”。
- **指定长度**：你可以说“请扩充到 500 字”或“请精简成 3 条要点”。
