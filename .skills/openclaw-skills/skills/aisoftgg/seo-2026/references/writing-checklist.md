# SEO Article Writing Checklist

## Before Writing
- [ ] Primary keyword identified and intent classified
- [ ] Top 5 SERP results analyzed (structure, depth, headings)
- [ ] AI Overview checked (present? what's cited?)
- [ ] Content brief completed with required sections
- [ ] 5-10 secondary/LSI keywords listed

## Article Structure
- [ ] Title includes primary keyword (50-60 chars)
- [ ] Meta description is a direct answer + CTA (150-160 chars)
- [ ] H1 matches title tag
- [ ] First paragraph directly answers the query (40-60 words)
- [ ] H2 sections are self-contained (134-167 words each)
- [ ] H2/H3 headings use natural question/keyword phrasing
- [ ] Paragraphs are 2-3 sentences max
- [ ] Includes comparison table (if applicable)
- [ ] FAQ section at the bottom with 3-5 questions

## Content Quality
- [ ] Word count meets or exceeds competitor average (min 2000)
- [ ] Stats with source attribution every 150-200 words
- [ ] Definite language (specific numbers, not vague claims)
- [ ] Named entities (tools, people, companies mentioned specifically)
- [ ] First-hand experience or unique insight included (E-E-A-T)
- [ ] Author bio with credentials
- [ ] No fluff paragraphs -- every section adds unique value
- [ ] All competitor topics covered + at least one gap filled

## Technical SEO
- [ ] Article schema (JSON-LD) generated
- [ ] FAQ schema (JSON-LD) if FAQ section exists
- [ ] BreadcrumbList schema
- [ ] Internal links to related content (3-5 minimum)
- [ ] External links to authoritative sources (2-3)
- [ ] Images with descriptive alt text
- [ ] URL slug is clean and contains primary keyword

## Post-Publish
- [ ] Submit URL to Google Search Console
- [ ] Share on relevant channels (Reddit, social)
- [ ] Set calendar reminder to refresh in 8-12 weeks
- [ ] Monitor ranking and AI citation status after 2 weeks
