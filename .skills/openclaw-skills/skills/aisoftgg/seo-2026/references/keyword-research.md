# Keyword Research Methodology

## Step-by-Step Process

### 1. Seed Keyword Discovery
Start with the core topic. Generate seed keywords using:
- web_search for "[topic] guide" / "[topic] tutorial" / "how to [topic]"
- Check "People Also Ask" boxes in Google results
- Look at competitor site navigation and blog categories
- Check Reddit/forum threads for how real people phrase questions

### 2. Keyword Expansion
For each seed keyword, expand with modifiers:
- **How-to**: "how to [keyword]", "[keyword] tutorial", "[keyword] step by step"
- **Comparison**: "[keyword] vs [alternative]", "best [keyword]", "[keyword] alternatives"
- **Problem**: "[keyword] not working", "[keyword] issues", "fix [keyword]"
- **Specific**: "[keyword] for beginners", "[keyword] 2026", "[keyword] examples"

### 3. Intent Classification
Classify each keyword:
- **Informational**: user wants to learn ("what is X", "how does X work")
- **Commercial**: user is comparing options ("X vs Y", "best X for Y")
- **Transactional**: user wants to buy/do ("buy X", "X pricing", "X download")
- **Navigational**: user wants a specific page ("X login", "X docs")

Match content format to intent:
- Informational -> comprehensive guide
- Commercial -> comparison/review post
- Transactional -> landing page / product page
- Navigational -> ensure your page is optimized for brand terms

### 4. Difficulty Assessment
Without paid tools, estimate difficulty by:
- **Low difficulty**: few results, forums/Reddit ranking, thin content in top 5
- **Medium difficulty**: established sites ranking but content is outdated or shallow
- **High difficulty**: major brands dominating, deep comprehensive content already exists

Target low-medium difficulty keywords first. Build authority before competing on high-difficulty terms.

### 5. AI Overview Check
For each target keyword, search it and check:
- Does an AI Overview appear?
- What sources does it cite?
- What format does the cited content use?
- What questions does the AI Overview answer?

If AI Overview appears: optimize for citation (structure, stats, authority)
If no AI Overview: optimize for traditional ranking (this is becoming rare for informational queries)

### 6. Topic Cluster Mapping
Group related keywords into clusters:

```
Pillar: "openclaw guide" (highest volume)
  Cluster: "openclaw setup" (how-to)
  Cluster: "openclaw memory" (feature-specific)
  Cluster: "openclaw vs hermes" (comparison)
  Cluster: "openclaw skills tutorial" (how-to)
  Cluster: "openclaw cost" (commercial)
```

Each cluster page targets a specific keyword while linking to the pillar page. This builds topical authority signals that both Google and AI systems use.

### 7. Prioritization Matrix

Score each keyword on:
- **Relevance** (1-5): how closely does it match our product/service?
- **Difficulty** (1-5): how hard is it to rank? (lower = easier = better)
- **Volume signal** (1-5): rough search volume based on competition density
- **AI citation opportunity** (1-5): is there a gap in what AI currently cites?

Priority = (Relevance * 2) + Volume + (5 - Difficulty) + AI Citation Opportunity

Start with the highest-scoring keywords.
