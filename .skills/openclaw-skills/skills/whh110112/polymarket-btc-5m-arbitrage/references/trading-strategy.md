# BTC 5分钟套利策略

## 市场特性

### 什么是 BTC 5分钟市场

- **系列**: btc-up-or-down-5m
- **时间**: 每5分钟一个市场
- **标的**: BTC/USD 价格涨跌
- **结果**: Up (涨) 或 Down (跌)
- **数据源**: Chainlink BTC/USD

### 交易时间窗口

- 每个市场持续5分钟
- 实时价格更新
- 适合高频交易

## 策略

### 1. 价差套利

当 `Bid + Ask < 1.0` 时:
- 买入 Bid (低价)
- 同时卖出 Ask (高价)
- 锁定利润: `1.0 - (Bid + Ask)`

### 2. 趋势跟踪

- 观察订单簿深度
- 顺势交易
- 5分钟窗口短持仓

### 3. 做市策略

- 同时挂 bid 和 ask
- 赚取价差收益
- 提供流动性

## 风险控制

- 最小流动性: $1000
- 最小订单量: 5
- 最大价差: 5%
- 单笔最大金额: $10

## 注意事项

1. 市场流动性充足时参与
2. 5分钟窗口短，注意时间
3. Chainlink 数据源为准
4. 考虑交易手续费
