# APP 调起支付

> 服务商模式与商户模式的调起支付代码完全一致，区别仅在于 `appId` 的取值：
> - 在服务商 APP 中调起支付时，`appId` 传 `sp_appid`
> - 在子商户 APP 中调起支付时，`appId` 传 `sub_appid`
>
> `appId` 必须与下单时传入的对应参数一致。
