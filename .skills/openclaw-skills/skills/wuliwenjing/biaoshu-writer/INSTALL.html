一、工具简介

biaoshu-writer 可以根据招标要求、评分标准自动生成技术标 Word 文档。

功能说明
📄 多格式解析支持 txt、pdf、docx、xlsx 招标文件
📊 智能大纲根据评分标准自动生成分章大纲
✍️ 内容扩充5倍法则充实，每章配套表格
🤖 AI去痕去除AI写作特征
📝 规范格式SimSun宋体，WPS/Office兼容
🎨 格式自定义支持自定义字体/字号/行距

三、第一步：安装 OpenClaw

1访问官网下载安装

https://openclaw.ai

2验证安装

openclaw --version
✅ 显示版本号 = 安装成功

四、第二步：安装 Python 库

打开终端（Terminal），运行：

pip install python-docx pdfplumber openpyxl PyPDF2

如果提示没有pip：

# macOS
python3 -m pip install python-docx pdfplumber openpyxl PyPDF2

# Windows
py -m pip install python-docx pdfplumber openpyxl PyPDF2
✅ 验证安装成功： 
python3 -c "import docx; print('python-docx OK')"

五、第三步：安装 SimSun 字体

⚠️ 重要：这一步必须做！
没有安装这个字体，生成的Word文档会显示"MS明朝"等错误字体。
🔄 正在检测字体...

安装方法：

方法1：从 Windows 复制（推荐）

  1. 找一台 Windows 电脑
  2. 打开 C:\Windows\Fonts
  3. 找到 SimSun.ttf（注意不是 SimSun.ttc）
  4. 复制到 Mac 的 ~/Library/Fonts/ 文件夹

方法2：网上下载

  1. 搜索 "SimSun.ttf下载"
  2. 下载得到 .ttf 文件
  3. 放入 ~/Library/Fonts/

验证字体：

ls -lh ~/Library/Fonts/SimSun.ttf
字体测试：这是宋体字体测试

六、第四步：安装技能

1下载 biaoshu-writer.skill 文件

2在终端运行：

openclaw skills install /path/to/biaoshu-writer.skill

3验证安装：

openclaw skills list | grep biaoshu

七、第五步：开始使用

  1. 打开 OpenClaw
  2. 发送招标文件（txt/pdf/docx/xlsx）给AI
  3. AI自动解析并生成技术标大纲
  4. 确认大纲（说"开始执行"）
  5. AI生成完整内容
  6. 获得符合投标规范的Word文档 ✅
📌 执行流程：

并发生成章节内容
       ↓

转换Word文档 ✅

八、常见问题

Q: pip不是内部命令？
# macOS
python3 -m pip install python-docx

# Windows
py -m pip install python-docx
Q: 生成的Word字体不对，显示MS明朝？
SimSun.ttf未正确安装。重新复制字体到 ~/Library/Fonts/ 并重启Word。
Q: PDF无法解析？
如果是扫描版PDF（没有文字层），需要先OCR识别。
推荐：Adobe Acrobat / 在线OCR
Q: 如何自定义Word格式？
在Markdown顶部加入格式指令： 
<!-- doc-format
font: SimSun
body-size: 32pt
line-spacing: 26pt
-->
