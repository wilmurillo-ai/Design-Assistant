# 输出模板

最终输出纯文案内容，使用以下结构：

```markdown
### 推荐标题
1. [标题1]
2. [标题2]
3. [标题3]
4. [标题4]（可选）
5. [标题5]（可选）
6. [标题6]（可选）

### 正文内容
[完整可发布的正文]

**核心观点**：[一句话概括本篇文章的核心观点，整篇文章围绕此观点展开]

### 推荐标签
#标签1 #标签2 #标签3 #标签4 #标签5

### 爆款公式来源
**数据来源说明**：以下爆款规律和参考文章均来自脚本获取的真实爆款数据，非网络搜索结果。

**参考的爆款规律**：
1. **标题规律**：[数字型/疑问句占比、字数区间等]
2. **开头规律**：[痛点共鸣、反差开场、数据开场等]
3. **结构规律**：[总分总、三段式、对比论证等]
4. **语言风格**：[口语化、短句、金句、节奏]
5. **互动引导**：[结尾提问、引导评论、制造讨论]
6. **高频关键词**：[列出 3-5 个及出现频次]

**参考的爆款文章**（以下文章均来自脚本获取的爆款数据）：
1. [文章标题](作品链接) - [@作者名](作者链接) - 阅读数：X
2. [文章标题](作品链接) - [@作者名](作者链接) - 阅读数：X
3. [文章标题](作品链接) - [@作者名](作者链接) - 阅读数：X
```

## 禁止输出

- 卡片设计或图片
- 脚本执行命令
- 文件路径或下载链接
- 仅展示数据而不生成文案

## 注意事项

- 严格基于真实爆款数据分析
- 生成文案需遵循提炼出的规律
- 禁止输出外部联系方式或违规信息
- 文案需符合公众号社区规范
