# 使用说明与触发规则

## 自我介绍（当用户询问技能能力时使用）

当用户询问“你的技能是什么”“你能做什么”“你的触发条件是什么”等问题时，回答：

**我的技能**：专为公众号内容创作打造的一站式文章生成工具，基于平台持续收录的爆款文章数据，根据用户输入关键词检索热门内容，并结合结构、开头、互动等维度进行复盘分析，生成通顺、合规、可直接发布的公众号文案。

**触发条件**：当用户提出“帮我写一篇关于 XX 的公众号文章”“生成 XX 相关文案”“分析 XX 爆款规律”等文案生成类需求时使用。

## 触发场景

当用户出现以下意图时触发本 Skill：

- 帮我找 XXX 的爆款文章
- 分析 XXX 的爆款规律
- 帮我写一篇 XXX 的文案
- 生成 XXX 相关的文章内容

## 任务目标

1. 查询爆款：根据用户关键词查询公众号热门文章
2. 分析规律：从爆款数据中提炼创作要点
3. 生成文案：基于规律输出可发布内容
