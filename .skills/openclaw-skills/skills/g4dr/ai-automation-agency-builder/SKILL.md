## ⚙️ The Automation Opportunity Detection System

```
HOW TO FIND BUSINESSES THAT NEED AUTOMATION:

🔍 SIGNAL 1 — Job Posting Analysis
  └─ Hiring "Data Entry Specialist" = manual data work begging to be automated
  └─ Hiring "Admin Assistant" = repetitive workflows = automation goldmine
  └─ Hiring "Operations Coordinator" = process chaos = high ROI automation
  └─ Multiple of these roles = significant budget + significant pain

💬 SIGNAL 2 — Glassdoor Review Mining
  └─ "Lots of copy-pasting between systems" = integration automation
  └─ "Everything is manual, nothing talks to each other" = Zapier/Make heaven
  └─ "We use spreadsheets for everything" = database + automation opportunity

🌐 SIGNAL 3 — Tech Stack Detection
  └─ Using 5+ disconnected SaaS tools = integration opportunity
  └─ CRM + Email + Spreadsheet + Invoicing all separate = glue automation
  └─ No automation tool detected (no Zapier/Make/n8n) = first-mover opportunity

📊 SIGNAL 4 — Company Size Sweet Spot
  └─ 10-50 employees = enough volume to justify automation, not big enough to have IT
  └─ Growing fast (hiring surge) = processes breaking under scale = urgent need
  └─ Agency, consultancy, real estate, e-commerce = highest automation ROI niches
```

```

STEP 7 — Claude AI Writes Personalized Pitch
  └─ Opens with THEIR specific manual workflows identified
  └─ Shows ROI math — "your team spends X hours/month on this"
  └─ Proposes specific automation + price + timeline
        ↓

OUTPUT: Prospect list + ROI calculations + personalized audits + pitch videos + agency menu
```
