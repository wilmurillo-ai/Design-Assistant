## 功能特性

✅ **列清理**
- 自动删除"项目/产品"、"产品/项目"、"产品"、"项目"等列
- 智能识别相关列并删除，清理冗余数据

✅ **智能识别**
- 自动识别"优先级"列（支持中英文）
- 自动识别"编号"列（支持多种命名）

✅ **精确筛选**
- 提取P0和P1优先级的测试用例
- 自动跳过无P0/P1用例的sheet

✅ **格式保持**
- 完整保留原文件样式
- 保持列宽和行高
- 复制所有单元格格式

✅ **智能处理**
- 自动拆分合并单元格
- 内容填充到所有拆分单元格
- 重新编号从TC001开始

## 处理流程

```
1. 分析文件结构
   ├─ 识别所有sheet
   ├─ 查找优先级列
   └─ 统计P0/P1用例

2. 处理每个sheet
   ├─ 拆分合并单元格
   ├─ 筛选P0/P1用例
   ├─ 复制格式
   └─ 重新编号

3. 生成输出文件
   └─ 保持原格式
```

## 常见问题

**Q: 为什么某些sheet被跳过？**
A: 该sheet可能：
- 没有"优先级"列
- 优先级列全为空
- 没有P0或P1用例

**Q: 编号会重复吗？**
A: 不会。每个sheet独立从TC001开始编号。

**Q: 格式会改变吗？**
A: 不会。完整保留所有格式、样式、列宽、行高。

**Q: 合并单元格如何处理？**
A: 自动拆分并将内容填充到所有拆分后的单元格中。
