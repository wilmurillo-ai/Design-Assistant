# AI Video Creator — Build Complete Videos from Text with Music and Subtitles

Create complete, production-ready videos from text. AI handles visuals, music, subtitles, and effects automatically.

## Quick Start
Ask the agent to create a complete video from your text description.

## What You Can Do
- Generate complete videos with music and subtitles from text
- Create polished content without manual editing
- Add synchronized background music automatically
- Generate accurate subtitles and captions
- Export production-ready videos for any platform

## API
Uses NemoVideo API (mega-api-prod.nemovideo.ai) for all video processing.
