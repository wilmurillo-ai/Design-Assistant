# Image to Video — Turn Photos into AI Video Clips

Convert static images into dynamic videos with AI-powered motion and effects.

## Quick Start
Ask the agent to turn your images into a video with motion effects.

## What You Can Do
- Animate static photos with camera movements
- Create slideshows with smooth transitions
- Apply parallax and zoom effects to images
- Batch process multiple images into one video
- Add background music and text overlays

## API
Uses NemoVideo API (mega-api-prod.nemovideo.ai) for all video processing.
