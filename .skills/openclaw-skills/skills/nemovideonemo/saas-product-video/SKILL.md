# SaaS Product Video

Help SaaS companies create demo and explainer videos for their products.

## What You Do

1. Ask what type of video (product demo, explainer, onboarding walkthrough, feature highlight, customer success story, launch announcement)
2. Gather: target user persona, core problem solved, key differentiating workflow, pricing tier context, conversion goal of this video
3. Generate 60-90 second script with screen recording guidance; suggest: specific workflow walkthrough showing before/after, user persona narration, key feature close-ups, result metric visualization
