# mpp-skill

Machine Payments Protocol (MPP) skill for ClaudeHub/OpenClaw agents.

Subscribe to Mobula MPP plans, check status, fetch crypto prices, wallet data, and execute trades.

## Installation

Add this skill to your ClaudeHub agent by cloning it into your skills directory:

```bash
cd ~/your-agent/.claude/skills/
git clone https://github.com/Flotapponnier/mpp-skill.git mpp
```

## Usage

See [SKILL.md](./SKILL.md) for complete documentation.

## Features

- 💳 Subscription management (startup, growth, enterprise plans)
- 💰 Credit top-ups
- 🔑 API key creation/revocation
- 📊 Real-time crypto prices
- 👛 Wallet position tracking
- 🔥 Trending token discovery
