# 已知错误记录

## 错误案例与教训

### 2026-03-23 错误1：凭空编造事实

**错误内容**：
在宝塔实业分析中写"重组已终止"

**实际情况**：
重组仍在进行中，搜索结果未提及终止

**教训**：
- 分析判断不能替代新闻事实
- 不确定时必须标注"搜索结果未提及"
- 禁止根据推测断言事实

**正确做法**：
```
❌ 错误：重组已终止
✅ 正确：搜索结果未提及重组是否终止
```

---

### 2026-03-23 错误2：财务数据自相矛盾

**错误内容**：
世纪华通 P/E 98倍，但 EPS -0.85（亏损）

**问题分析**：
- 亏损公司（EPS为负）不能有正的市盈率
- 数据来源可能混用了不同时期

**教训**：
- 亏损公司P/E必须标注为"亏损，不适用"
- P/E和EPS必须互相印证
- 禁止混用不同时期的数据

**正确做法**：
```
❌ 错误：P/E: 98倍, EPS: -0.85元
✅ 正确：P/E: 亏损，不适用（EPS为负）
```

---

### 2026-03-23 错误3：早报输出被截断

**错误内容**：
早报输出时，宝塔实业和07709港股缺失

**问题分析**：
- 单只股票内容过长（超过400字）
- 导致后续股票无法输出

**教训**：
- 每只股票严格控制在400字以内
- 必须确保5只股票全部输出（4只A股 + 1只港股）
- 输出前检查字数

**正确做法**：
- 新闻摘要精简到50字以内
- 投资建议控制在50-100字
- 删除冗余描述

## 预防措施

### 输出前检查清单

1. **事实核查**
   - [ ] 所有事实性陈述都有来源
   - [ ] 不确定信息标注"搜索结果未提及"

2. **数据一致性**
   - [ ] P/E与EPS互相印证
   - [ ] 亏损公司P/E标注为"亏损，不适用"
   - [ ] 所有数据来自同一时期

3. **格式规范**
   - [ ] 单只股票 ≤ 400字
   - [ ] 5只股票全部输出
   - [ ] 新闻都在7天内

4. **来源标注**
   - [ ] 每条新闻标注来源
   - [ ] 使用白名单内的新闻源
   - [ ] API数据标注API来源
