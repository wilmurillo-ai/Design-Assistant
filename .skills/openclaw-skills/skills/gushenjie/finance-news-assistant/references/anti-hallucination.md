# 防止幻觉（Hallucination）规则

**这是最高优先级规则，违者视为致命错误。**

## 核心原则

### 1. 严格区分"搜索结果"和"分析判断"

- **搜索结果中的事实** → 可直接引用，必须注明来源
- **分析判断** → 必须明确标注"基于上述新闻的分析"

### 2. 禁止凭空添加信息

- 如果搜索结果没有提到某件事，禁止断言其"有"或"没有"
- **正确表述**："搜索结果未提及此事"
- **错误表述**："该事件已终止"（无来源支持）

### 3. 不确定的信息必须标注

- 无法核实的内容 → 标注「⚠️ 搜索结果未提供」
- 不能当作确定性事实输出

## 财务数据校验规则

### 市盈率(P/E)规则

- **亏损公司（净利润为负）的P/E必须标注为"亏损，不适用"**
- 不能写具体数字（如"98倍"）

### 数据一致性检查

- P/E 和 EPS 必须互相印证：
  - 若 EPS 为负 → P/E 必须为"亏损，不适用"
  - 若 EPS 为正 → P/E 才能为正数

### 时间一致性

- 股价涨跌幅必须是同一天的数据
- 禁止混用不同日期的数据

### 缺失数据处理

- 搜索结果没有提供的数据 → 如实写"搜索结果未提供"
- 不要用"N/A"、"--"等模糊表述

## 违规案例（禁止）

### ❌ 案例1：凭空编造事实
```
错误：重组已终止
原因：没有新闻来源支持
正确：搜索结果未提及重组是否终止
```

### ❌ 案例2：数据自相矛盾
```
错误：P/E 98倍 + EPS -0.85
原因：亏损公司不能有正P/E
正确：P/E：亏损，不适用（EPS为负）
```

### ❌ 案例3：混淆分析与事实
```
错误：该公司业绩将大幅改善
原因：这是预测，不是事实
正确：基于上述新闻分析，公司业绩可能改善
```

## 合规表述模板

### 不确定信息
```
✅ "搜索结果未提及重组是否终止"
✅ "关于该事项，搜索结果未提供相关信息"
✅ "⚠️ 搜索结果未提供该数据"
```

### 亏损公司财务数据
```
✅ "P/E：亏损，不适用（EPS为负）"
✅ "市盈率：公司亏损，不适用"
```

### 分析判断
```
✅ "基于上述新闻的分析，该事件可能对股价产生正面影响"
✅ "根据搜索到的信息推测，..."
```

## 检查清单

在输出前，必须检查：

- [ ] 所有事实性陈述都有来源支持
- [ ] 亏损公司的P/E标注为"亏损，不适用"
- [ ] P/E和EPS数据互相印证
- [ ] 分析判断明确标注为"分析"而非"事实"
- [ ] 不确定信息标注"搜索结果未提及"
- [ ] 没有混用不同日期的数据
