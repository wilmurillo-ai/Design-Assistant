## 五档行情异常标注规则

### 卖盘压力异常
- 单档挂单量 > 平均值3倍 → 标注 ⚠️ 大单压盘
- 卖一价格远高于现价 → 标注 ⚠️ 跳空压力

### 买盘支撑异常
- 单档挂单量 > 平均值3倍 → 标注 ⚠️ 大单托盘
- 买一价格远低于现价 → 标注 ⚠️ 跳空支撑

## 新闻影响标注

### 🔴 利空
- 业绩下滑、亏损扩大
- 重大诉讼、监管处罚
- 高管离职、股东减持
- 行业政策利空

### ✅ 利好
- 业绩超预期、盈利增长
- 重大合同、战略合作
- 政策扶持、行业利好
- 股东增持、回购

### ⚪ 中性
- 常规公告
- 行业动态
- 市场评论

## 字数控制要点

1. **单只股票总字数 ≤ 400字**
2. **新闻摘要每条 ≤ 50字**
3. **投资建议 50-100字**
4. **整体仓位建议 ≤ 150字**

## 输出检查清单

- [ ] 5只股票全部输出（4只A股 + 1只港股）
- [ ] 每只股票字数 ≤ 400字
- [ ] 所有新闻都在7天内
- [ ] 所有数据都有来源（API或搜索结果）
- [ ] 亏损公司P/E标注为"亏损，不适用"
- [ ] 五档行情异常已标注
- [ ] 新闻影响已分类（🔴/✅/⚪）
- [ ] 整体仓位建议已输出
- [ ] 风险提示已列出
