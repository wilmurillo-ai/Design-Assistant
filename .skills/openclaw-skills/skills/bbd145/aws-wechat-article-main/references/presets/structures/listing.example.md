# 清单体结构

## 标准结构

```
标题
├── 摘要（80–128 字，概括核心价值与结论）
├── 开头（1–2 段，直给式：今天分享 N 个 XXX 技巧/步骤）
├── 正文（每节一个小标题 + 2–4 段，短段为主）
│   ├── 一、……
│   ├── 二、……
│   └── …
├── 结尾（1 段：总结 + 行动号召）
└── 文末区块（按 closing_block 预设）
```

## 要求

- 小标题简洁，不超过 15 字
- 每段 3–5 行，核心观点放段首
- 配图密度：每节一图或按 config `image_density`
