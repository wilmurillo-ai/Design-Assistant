# Seasonal Maintenance Schedule — Residential Rentals

## Spring (March-May)
- [ ] HVAC: Schedule AC tune-up before summer
- [ ] Gutters: Clean after spring rain/pollen
- [ ] Exterior: Check siding, paint, caulking for winter damage
- [ ] Landscaping: Mulch, trim, mow schedule begins
- [ ] Roof: Visual inspection for missing/damaged shingles
- [ ] Windows/screens: Replace damaged screens, check seals
- [ ] Smoke/CO detectors: Test all units, replace batteries

## Summer (June-August)
- [ ] HVAC: Replace filters (monthly in heavy use)
- [ ] Pest control: Preventive treatment
- [ ] Exterior: Power wash siding, decks, walkways
- [ ] Plumbing: Check outdoor faucets, irrigation
- [ ] Dryer vents: Clean to prevent fire hazard
- [ ] Inspect vacant units: Prevent standing water, mold

## Fall (September-November)
- [ ] Furnace: Schedule tune-up before winter
- [ ] Gutters: Clean leaves before freeze
- [ ] Weatherstripping: Check doors and windows
- [ ] Insulation: Check attic, pipes for winter prep
- [ ] Exterior faucets: Shut off and drain
- [ ] Smoke/CO detectors: Test all units, replace batteries (bi-annual)
- [ ] Tree trimming: Before ice/snow loads branches

## Winter (December-February)
- [ ] Snow/ice: Ensure walkways cleared (liability)
- [ ] Pipes: Monitor for freeze risk in cold snaps
- [ ] HVAC filters: Replace monthly
- [ ] Check heating: Respond quickly to any outages (habitability)
- [ ] Inspect for ice dams on roof
- [ ] Carbon monoxide: Extra vigilance with heating systems

## Ongoing (Monthly/Quarterly)
- HVAC filters: Monthly during heavy use, quarterly otherwise
- Fire extinguishers: Annual inspection
- Plumbing: Check for leaks under sinks quarterly
- Common areas: Monthly walkthrough
- Exterior lighting: Replace burned-out bulbs promptly (safety/liability)
