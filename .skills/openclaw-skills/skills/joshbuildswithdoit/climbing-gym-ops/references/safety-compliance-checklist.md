# Climbing Gym Safety Compliance Checklist

## Daily Checks (Opening Staff)
- [ ] Walk all climbing areas — visual inspection of mats, holds, walls
- [ ] Check auto-belay devices: webbing condition, retraction speed, carabiner gates
- [ ] Verify first aid kits stocked and accessible (one per climbing area minimum)
- [ ] Check AED battery/status indicator
- [ ] Confirm staff on shift are belay-certified and first aid/CPR current
- [ ] Inspect rental gear: shoes (sole integrity, heel tabs), harnesses (stitching, buckles), belay devices
- [ ] Temperature/humidity check — mats get slippery above 70% humidity

## Weekly Checks (Manager)
- [ ] Spin bolt audit: random sample of 10 holds per area — hand-torque test
- [ ] Auto-belay deep check: full extension/retraction cycle, webbing fraying, anchor hardware
- [ ] Mat condition: tears, compression, gaps between mats, wall-to-mat interface
- [ ] Emergency exit paths clear and marked
- [ ] Fire extinguisher inspection (visual)
- [ ] Log review: any incident reports from the week needing follow-up?

## Monthly Checks (Manager/Owner)
- [ ] Full hold audit: every hold in one section (rotate sections monthly so full gym covered quarterly)
- [ ] Anchor point inspection: bolts, hangers, chains, draws on lead walls
- [ ] Rope inspection: wear marks, sheath damage, core shots, retire per manufacturer spec
- [ ] Belay device inventory: GriGris, ATCs — check for wear, proper function
- [ ] Rental harness deep inspection: all stress points, buckles, tie-in loops
- [ ] Crash pad / bouldering mat compression test — replace if bottomed out
- [ ] Staff certification expiry check — flag anyone within 60 days of expiring

## Quarterly Checks (Owner + Qualified Inspector)
- [ ] Auto-belay manufacturer-recommended service interval (varies by brand: TruBlue = 6mo, Perfect Descent = annual)
- [ ] Structural wall inspection: T-nut integrity, panel attachment, framework
- [ ] HVAC system service — air quality matters for exertion sports
- [ ] Emergency procedure drill (evacuation, injured climber on wall, auto-belay failure)
- [ ] Insurance policy review — coverage still matches operations?

## Annual Requirements
- [ ] Full professional wall inspection by qualified routesetter/engineer
- [ ] Auto-belay factory service/recertification
- [ ] Insurance renewal — document any claims, changes in operations, new equipment
- [ ] Staff re-certification: belay tests, first aid/CPR renewal
- [ ] Emergency procedure manual review and update
- [ ] Waiver language legal review (state law changes)
- [ ] Workers comp audit
- [ ] Fire marshal inspection (if required by jurisdiction)

## Incident Reporting Protocol
1. **Immediate:** Secure the scene, attend to injured person, call 911 if needed
2. **Within 1 hour:** Complete incident report form (who, what, when, where, witnesses, actions taken)
3. **Within 24 hours:** Manager review, photo documentation, equipment quarantine if involved
4. **Within 48 hours:** Notify insurance carrier if injury required medical attention
5. **Within 1 week:** Root cause analysis, corrective action plan, staff briefing
6. **Monthly:** Review all incidents for patterns (same wall section? same time of day? same equipment?)

## Severity Classification
| Level | Description | Example | Action |
|-------|-------------|---------|--------|
| 1 - Minor | No injury or very minor | Scrape, small bruise | Log only |
| 2 - Moderate | First aid needed, no medical visit | Sprain, deeper scrape | Log + manager review |
| 3 - Significant | Medical visit required | Suspected fracture, concussion | Log + insurance notify + root cause |
| 4 - Severe | ER/ambulance required | Fall from height, head injury | Log + insurance + full investigation + corrective action |

## Staff Certification Tracking
| Certification | Required For | Renewal Period |
|--------------|-------------|----------------|
| Belay certification (in-house) | All floor staff | Annual |
| First Aid | All staff on shift | 2 years |
| CPR/AED | All staff on shift | 2 years |
| CWI (Climbing Wall Instructor) | Lead instructors | 3 years |
| Route setter certification | Setters | Varies by org |
