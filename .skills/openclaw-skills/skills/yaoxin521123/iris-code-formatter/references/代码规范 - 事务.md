# 代码规范 - 事务

# 事务

1. 严格禁止开放性事务。

反例：

```java
ClassMethod Save()
{
	tstart
}
```

正例：

```java
ClassMethod Save()
{
	ts
	
	tc //或 tro 
}
```

2. 事务`ts`、`tc`、`tro`的位置应保持近距离，在一屏幕范围内，避免位置过远。代码过多应当提取成方法，让事务看起来简洁、语义明朗。

反例：

```java
ClassMethod Save()
{
	ts
	
	/* 添加主表信息 */
	// todo 50行代码

	
	/* 添加明细信息 */
	// todo 100行代码

	tc
}
```

正例：

```java
ClassMethod Save()
{
	ts
	
	/* 添加主表信息 */
	s sc = ..InsMain()
	i $$$ISERR(sc) tro
	q:($$$ISERR(sc)) sc
	
	/* 添加明细信息 */
	s sc = ..InsDetail()
	i $$$ISERR(sc) tro
	q:($$$ISERR(sc)) sc
	
	tc
	
	q sc
	
}
```


3. 严格禁止**跨方法**提交事务。

反例：

```java
ClassMethod Save()
{
	ts
	s sc = ..InsMain()

}
ClassMethod InsMain()
{
	// todo
	tc
}

```

正例：

```java
ClassMethod Save()
{
	ts
	
	s sc = ..InsMain()

	tc
	
	q sc
}

ClassMethod InsMain() as %Status
{
	// todo
}

```

4. 为明确显示事务，事务命令需要**简写并且小写**。

反例：

```java
ClassMethod Save()
{
	TSTART
	...
	TCOMMIT
}
```

正例：

```java
ClassMethod Save()
{
	ts
	
	... // todo
	
	tc
}
```

5. 同一个方法内不应该出现事务嵌套的现象。

反例：

```java
ClassMethod Save()
{
	ts
    ts
    ... // todo
    tro
    
	tc
	
	... // todo
	ts
	
	tc
}

```

正例：

```java
ClassMethod Save()
{
	ts
	
	... // todo
	
	tc
}


```

6. 事务应在保存程序的最外层。

反例：

```java
ClassMethod Save()
{
	ts
	
    s sc = ..InsMain()
    
    s sc = ..InsDetail()
    
	tc
}

ClassMethod InsMain() as %Status
{
    ts
	// todo
	tc
}

ClassMethod InsDetail() as %Status
{
    ts
	// todo
	tc
}
```

正例：

```java
ClassMethod Save()
{
	ts
	
    s sc = ..InsMain()
	i $$$ISERR(sc) tro
	q:($$$ISERR(sc)) sc
    
    s sc = ..InsDetail()
	i $$$ISERR(sc) tro
	q:($$$ISERR(sc)) sc
	
	tc
}

ClassMethod InsMain() as %Status
{
	// todo
	q sc
}

ClassMethod InsDetail() as %Status
{
	// todo
	q sc
}
```

7. 单条`SQL`语句的数据保存不需要事务。

反例：

```java
ClassMethod Save()
{
	ts
    
    &&sql()
	
	tc
}

```

正例：

```java
ClassMethod Save()
{
    &&sql()
}


```


8. `ts`,`tc` 首尾添加`空行`，或者`注释`.。

反例：

```java
ClassMethod Save()
{
	TSTART
	s ret=..SaveMain()
	...
	s retItm=..SaveDetail()
	...
	TCOMMIT	
}

ClassMethod SaveMain()
{
	TSTART
	...
	TCOMMIT
}

ClassMethod SaveDetail()
{
	TSTART
	...
	TCOMMIT
}
```

正例：

```java
ClassMethod Save()
{
	ts
	
	/* 注释 */
	s ret = ..SaveMain()
	i ret < 0 tro q
	...
	
	/* 注释 */
	s ret = ..SaveDetail()
	i ret < 0 tro q
	...
	
	tc	
}

ClassMethod SaveMain()
{
	...
}

ClassMethod SaveDetail()
{
	...
}

```
