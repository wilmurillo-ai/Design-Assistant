# 代码规范 - 锁

# 锁

1. 禁止直接锁表结构的`Global`。

反例：

```java

l +^BS.IRIS.StudentD

```


2. 加解锁必须加`+`、`-`严格控制，必须成对出现。

反例：

```java

l +^BS.IRIS.Student(1)
l +^BS.IRIS.Student(1)

l -^BS.IRIS.Student(1)

```

正例：

```java
l +^BS.IRIS.Student(1):3
l -^BS.IRIS.Student(1):3

l +^BS.IRIS.Student(1):3
l -^BS.IRIS.Student(1):3
```

3. 加锁必须要带`+`，否则导致解锁进程内所有锁。

反例：

```java

l +^BS.IRIS.Student1(1)
l +^BS.IRIS.Student2(1)
l ^BS.IRIS.Student3(1)


```

正例：

```java
l +^BS.IRIS.Student1(1):3
l +^BS.IRIS.Student2(1):3
l +^BS.IRIS.Student3(1):3
```

4. 加锁必须写超时的退出，避免死锁，造成进程无限期阻塞。

正例：

```java
l +^BS.IRIS.Student:3
```

5. 自定义功能锁：`^产品组代码(产品线,规范代码:唯一标识)`（例如：单号）。

反例：

```java
l ^TMPOP(lockName)
```

正例：

```java
l +^IRIS("BIZ", "BOXNO:" _ boxNo):5 
```

6. 私有进程全局变量名不能用作锁名。

反例：

```java
l +^||BS.IRIS.Student(1):3
```

正例：

```java
l +^BS.IRIS.Student(1):3
```

7. 禁止单独在程序中使用无参数锁。


反例：

```java
lock
```

8. 使用锁时一定要下标节点。


反例：

```java

l +^BS.IRIS.Student

```

正例：

```java
l +^BS.IRIS.Student(1):3
```
