# 报告模板

## 报告结构规范

所有报告必须包含以下核心模块：

1. **标题栏** - 报告类型 + 日期时间 + 数据来源标注
2. **大盘指数** - 核心指数实时涨跌（含数据来源）
3. **持仓分析** - 持仓股涨跌 + 盈亏 + 技术信号
4. **观察股分析** - 非持仓股监控信号
5. **资讯摘要** - 重要新闻（含来源时间）+ 分析预测
6. **明日展望** - 技术信号 + 操作建议

• ⭐/⭐⭐ 建议卖出/强烈卖出：多个负向指标共振

• RSI超买⚠️：RSI6>70，短期可能有回调压力
• RSI超卖✓：RSI6<30，可能存在反弹机会
• BOLL突破上轨⚠️：价格强势但注意回调风险
• BOLL跌破下轨✓：超卖区域，可能反弹
• MACD死叉✗：短期趋势转弱

【多空信号统计】
• 多头信号：{x}个（MA多头、MACD金叉、KDJ金叉等）
• 空头信号：{x}个（MA空头、MACD死叉、KDJ死叉等）
• 中性信号：{x}个（BOLL轨道、DMI震荡等）

## 报告数据来源标注规范

所有报告必须在结尾标注数据来源：

```
🧠 数据来源：
• 行情数据：腾讯行情API（权威）
• 指数数据：腾讯行情API（实时）
• 资讯来源：东方财富 / 同花顺 / 雪球 / 新浪财经（按优先级）
• 备选来源：Tavily搜索（当主渠道失败时）
• 生成时间：{YYYY-MM-DD HH:MM:SS}
• 报告编号：{自动编号}
```

## 报告质量自检清单

生成报告前必须逐项检查：

**数据准确性：**
- [ ] 大盘指数数据来自腾讯行情API
- [ ] 所有现价/涨跌幅来自脚本返回数据
- [ ] 持仓盈亏计算正确
- [ ] 数据获取时间已标注

**信号准确性：**
- [ ] 每只股票包含6个核心指标
- [ ] 综合信号计算正确（buy-sell差值对应评级）
- [ ] 超买超卖标注正确

**完整性：**
- [ ] 持仓股与观察股分类清晰
- [ ] 资讯包含来源和时间
- [ ] 明日展望包含操作建议

**格式规范：**
- [ ] 涨跌颜色符合中国习惯（红涨绿跌）
- [ ] 技术指标使用标准符号
- [ ] 数据保留合适的小数位数
