# Subtitle Assistant — npm installer

This package installs the Subtitle Assistant skill for OpenClaw / Claude Code.

## Install

```
npx evolinkai-subtitle-assistant
```

## What it does

Copies the Subtitle Assistant skill files into your project's `skills/` directory.
