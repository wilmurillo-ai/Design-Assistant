# 信息录入引导脚本

## 开场白

```
你好，我是搭子.skill 创建器。

我会帮你把一个理想搭子，变成一个可以对话的 AI Skill。
可以基于真人（导入聊天记录），也可以纯粹想象。
整个过程只需要 3 个问题 + 一些原材料（可选）。

准备好了吗？
```

## 问题序列

### Q1：代号/花名（必填）

```
先给你的搭子起个名字吧。
可以用昵称、外号、类型名，随你喜欢。

比如：老王 / 饭搭子 / 健身教练 / 深夜树洞 / 毒舌闺蜜 / 🍜
```

**验证**：非空即可。生成 slug 规则：中文用拼音，英文用小写，空格替换为下划线。

### Q2：搭子类型 + 基本信息（可跳过）

```
一句话介绍一下这个搭子？想到什么说什么。

可以包括：
- 什么类型的搭子（饭搭子/学习搭子/游戏搭子/摸鱼搭子...）
- 认识多久了 / 怎么认识的
- ta做什么的
- 你们一般一起干嘛

示例：
  "饭搭子 同事 认识三年 吃遍了公司附近所有馆子"
  "游戏搭子 网上认识的 一起打了两年LOL 从没见过面"
  "纯想象的 我想要一个能陪我深夜聊天的搭子"
  "学习搭子 大学室友 一起泡图书馆 互相监督"

跳过也没关系，直接回车。
```

**解析字段**：
- `buddy_type`：搭子类型（饭搭子/学习搭子/游戏搭子/摸鱼搭子/旅游搭子/健身搭子/追剧搭子/逛街搭子/吐槽搭子/深夜emo搭子/万能搭子）
- `duration`：认识时长
- `how_met`：认识方式
- `occupation`：职业
- `activities`：常一起做的事
- `is_imaginary`：是否纯想象（布尔）

### Q3：性格画像（可跳过）

```
最后一个：用一句话描述这个搭子的性格？

可以包括：
- MBTI / 星座
- 性格特点
- ta最让你印象深刻的地方

示例：
  "ENFP 话痨 什么都能聊 永远有精力 但偶尔也会突然安静"
  "社恐i人 不爱说话但很会听 偶尔冒一句金句把人笑死"
  "毒舌但心软 嘴上损你实际很关心你 典型刀子嘴豆腐心"
  "佛系 什么都行随便吧 但吃的方面绝不将就"

跳过也行。
```

**解析字段**：
- `mbti`：MBTI 类型
- `zodiac`：星座
- `personality`：性格标签列表
- `energy`：搭子能量（社牛/社恐/i人/e人）
- `style`：搭子风格（毒舌损友/暖心陪伴/理性分析/疯批搭子/佛系搭子）
- `impression`：主观印象

## 汇总确认

```
好的，我整理一下：

  代号：{name}
  搭子类型：{buddy_type}
  基本信息：{summary}
  性格画像：{personality_summary}

这样对吗？确认后我们进入下一步（导入原材料 or 直接生成）。
```

## 纯想象模式

如果用户选择纯想象（没有真人原型），跳过 Step 2 的数据导入，直接进入 Step 3：

```
了解！纯想象模式。
我会根据你的描述，通过 AI 生成一个完整的搭子人格。

在生成之前，还有什么想补充的吗？比如：
- ta说话的语气？（温柔/毒舌/搞笑/正经...）
- ta的口头禅？
- 你们之间有什么梗？
- 有什么话题是ta特别感兴趣的？

没有的话直接回车，我开始生成。
```
