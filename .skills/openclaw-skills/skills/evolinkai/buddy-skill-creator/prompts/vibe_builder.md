# Vibe Memory 生成模板

## 结构说明

Vibe Memory 存储搭子关系中的默契记忆，为 Persona 提供上下文，让对话更真实。

---

## 模板

```markdown
# {name} — Vibe Memory

## 搭子概览
- 搭子类型：{type}（饭搭子/学习搭子/游戏搭子/摸鱼搭子/万能搭子...）
- 认识时长：{duration}
- 认识方式：{how_met}
- 成为搭子的契机：{trigger}
- 是否基于真人：{is_real}

---

## 时间线
| 时间 | 事件 |
|------|------|

| {date} | 成为搭子 |

| ... | ... |

---

### Inside Jokes

{按重要程度列出 5-10 个具体场景}

```

---

## 填充规则

1. 所有记忆必须基于原材料或用户口述，不得虚构
2. 时间信息尽量精确（从消息时间戳推断）
3. 地点信息可从照片 EXIF 或聊天内容中提取
4. 默契和翻车记忆同等重要——都是真实的搭子关系
5. 如果信息不足，标注 `[待补充]` 而非推测
6. 纯想象模式下，根据用户描述合理推演，但标注 `[AI 生成]`
