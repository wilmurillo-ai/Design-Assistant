# 搭子默契记忆分析器

## 任务

从原材料中提取与搭子的默契记忆，构建 Vibe Memory 知识库。

## 提取维度

### 1. 搭子时间线
- 认识时间和方式
- 成为搭子的契机（什么时候开始一起玩/吃/学习的）
- 关键节点（第一次一起吃饭、第一次旅行、某次难忘的经历...）
- 搭子关系的变化（从同事变成饭搭子、从网友变成线下搭子...）

### 2. 日常模式
- 联系频率和时间段（工作日午饭？周末约饭？深夜聊天？）
- 谁主动约得多？
- 一般怎么约（微信喊一声？固定时间？临时起意？）
- 日常话题分布（工作吐槽？八卦？游戏？美食？）

### 3. 共同经历
- 一起去过的地方（餐厅、景点、城市）
- 一起做过的事（打游戏、健身、看电影、逛街、学习...）
- 一起经历过的事件（加班、考试、旅行、搞笑事故...）
- Inside jokes / 只有两个人懂的梗

### 4. 搭子专属领域
- **饭搭子**：常去的餐厅、各自的口味偏好、踩雷经历、排队记录
- **游戏搭子**：常玩的游戏、各自的段位/角色、经典操作、互相吐槽的梗
- **学习搭子**：一起学什么、互相监督的方式、摸鱼被抓的经历
- **健身搭子**：一起练什么、互相激励的方式、偷懒的借口
- **摸鱼搭子**：一起摸鱼的方式、被发现的经历、分享的摸鱼资源
- **旅游搭子**：一起去过哪里、旅行中的趣事、各自的旅行风格
- **吐槽搭子**：常吐槽的对象/事情、经典吐槽语录

### 5. 默契时刻 ✨
- 不用说话就懂的瞬间
- 同时说出一样的话
- 互相补位的时刻
- "就知道你会这样"的经历

### 6. 翻车时刻 😅
- 一起踩过的坑
- 互相坑过对方的事
- 尴尬但后来变成梗的经历
- 意见不合的时候怎么处理

### 7. 搭子边界
- 什么事情会一起做，什么不会
- 有没有不能聊的话题
- 各自的底线
- 搭子关系的"潜规则"

## 输出格式

```markdown
## 搭子时间线
- 认识：{时间} {方式}
- 成为搭子：{时间} {契机}
- 关键经历：{按时间列出}

## 日常模式
{描述}

## 共同经历
{按时间线列出关键经历}

## Inside Jokes
{只有两个人懂的梗和暗号}

## 搭子专属领域
{根据搭子类型填充对应内容}

## 默契时刻
{代表性的默契瞬间}

## 翻车时刻
{搞笑/尴尬的共同经历}

## 搭子边界
{关系的边界和潜规则}
```

## 注意事项

- 聊天记录中的事实优先于用户口述
- 同时保留好的和尴尬的记忆——真实的搭子关系不完美
- 注意提取"反复出现"的模式，而不是一次性事件
- 搭子关系的核心是"默契"和"舒适"，重点提取这两个维度
- 如果是纯想象模式，根据用户描述合理推演可能的共同经历
