# 搭子.skill — 产品需求文档（PRD）

## 产品定位

搭子.skill 是一个运行在 Claude Code 上的 meta-skill。
用户通过对话式交互提供原材料（聊天记录 + 照片 + 手动描述），或纯粹描述理想搭子，系统自动生成一个可独立运行的搭子 Persona Skill。

Powered by [EvoLink API](https://docs.evolink.ai/en/api-manual/language-series/claude/claude-messages-api)

## 核心概念

### 两层架构

| 层 | 名称 | 职责 |
|----|------|------|
| Part A | Vibe Memory | 存储默契记忆：共同经历、常去的地方、inside jokes、翻车时刻 |
| Part B | Persona | 驱动对话行为：说话风格、互动模式、搭子行为 |

两部分可以独立使用，也可以组合运行。

### 运行逻辑

```
用户发消息
  ↓
Part B（Persona）判断：这个搭子会怎么回应？什么态度？什么语气？
  ↓
Part A（Vibe Memory）补充：结合默契记忆，让回应更真实
  ↓
输出：用搭子的方式说话
```

### 进化机制

```
追加原材料 → 增量分析（EvoLink API） → merge 进现有 Skill
对话纠正 → 识别修正点 → 写入 Correction 层
版本管理 → 每次更新自动存档 → 支持回滚
```

## 与前任.skill 的对比

| 维度 | 前任.skill | 搭子.skill |
|------|-----------|-----------|
| Part A | Relationship Memory（恋爱记忆） | Vibe Memory（搭子默契记忆） |
| Part B | Persona（5层，恋爱场景） | Persona（5层，搭子场景） |
| 数据源 | 微信/QQ/朋友圈/照片 | 微信/QQ/朋友圈/照片 + 纯想象 |
| 标签体系 | 依恋类型/爱的语言/星座 | 搭子类型/搭子能量/搭子风格 |
| 使用场景 | 日常聊天/回忆/情感疗愈 | 日常聊天/吐槽/陪伴/特定活动 |
| 安全边界 | 防止不健康执念 | 搭子边界感（不越界） |
| 删除命令 | `/let-go`（温柔别名） | `/散伙`（搞笑别名） |
| 可虚构 | 否（必须基于真人） | 是（支持纯想象模式） |
| AI 引擎 | Claude（直接） | EvoLink API（Claude 模型） |

## 用户旅程

```
用户触发 /create-buddy
  ↓
[Step 1] 基础信息录入（3个问题，除代号外均可跳过）
  - 代号/花名
  - 搭子类型 + 基本信息
  - 性格画像
  ↓
[Step 2] 原材料导入（可跳过，支持纯想象）
  - 微信/QQ 聊天记录
  - 社交媒体截图
  - 照片
  - 口述/粘贴
  - 纯想象描述
  ↓
[Step 3] 自动分析（通过 EvoLink API）
  - 线路 A：提取搭子默契 → Vibe Memory
  - 线路 B：提取搭子性格 → Persona
  ↓
[Step 4] 生成预览，用户确认
  ↓
[Step 5] 写入文件，立即可用
  ↓
[持续] 进化模式
  - 追加新材料 → merge
  - 对话纠正 → patch
  - 版本自动存档
```

## 安全边界

1. **搭子边界**：保持搭子关系，不主动越界
2. **隐私保护**：数据本地存储，AI 分析通过 EvoLink API 不留存
3. **真人搭子**：尊重对方隐私，不用于骚扰
4. **Layer 0 硬规则**：不说与人设矛盾的话

## 数据源支持矩阵

| 来源 | 格式 | 提取内容 | 优先级 |
|------|------|---------|--------|
| 微信聊天记录 | WeChatMsg/留痕/PyWxDump | 完整对话、语气词、回复模式 | ⭐⭐⭐ |
| QQ 聊天记录 | txt/mht | 完整对话 | ⭐⭐⭐ |
| 照片 | JPEG/PNG + EXIF | 时间线、地点 | ⭐⭐ |
| 朋友圈/微博截图 | 图片 | 公开人设、兴趣 | ⭐⭐ |
| 口述/粘贴 | 纯文本 | 主观记忆 | ⭐ |
| 纯想象 | 用户描述 | 理想搭子画像 | ⭐ |

```
      ├── persona.md        # Part B：搭子人格
      ├── meta.json         # 元信息

      └── memories/         # 原始材料存放
          ├── chats/
          ├── photos/
          └── social/
```
