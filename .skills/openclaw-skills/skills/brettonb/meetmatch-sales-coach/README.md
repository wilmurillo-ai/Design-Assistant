# MeetMatch Sales Coach — OpenClaw Skill

Gives your OpenClaw agent access to ML-powered sales predictions, persistent rep coaching memory, and personalized morning briefings.

## Quick start

```bash
openclaw skills install meetmatch-sales-coach
```

Then configure with your MeetMatch API key and org ID (from Settings > Integrations at meetmatch.ai).

## What it does

- **Morning briefings** with risk-scored meetings, coaching nudges, and prep tips
- **Persistent memory** that tracks each rep's patterns across calls
- **ML context** from MeetMatch's routing and no-show prediction models
- **On-demand coaching** via natural language ("how's my day", "brief me on my 2pm")

## Requirements

- MeetMatch Pro plan ($50/seat/mo) or above
- OpenClaw API key from your MeetMatch settings

Full docs in SKILL.md.
