# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2025-01-01

### Added

- Design Token System with colors, spacing, font weights, border radius, shadows, gradients, and responsive breakpoints
- Typography Engine with multi-language font stacks (EN, zh-CN, zh-TW, JA, KO) and Google Fonts fallbacks
- Copywriting Optimizer with Apple-style copy rules for English and Chinese
- Image Curator with Unsplash/Pexels recommendations and CSS gradient fallbacks
- Layout Patterns including Hero sections, product grids, feature cards, and scroll-animated blocks
- OpenClaw manifest for ecosystem compatibility
- Claude Code compatible entry file (CLAUDE.md)
- Three usage examples: product landing page, feature showcase, and pricing page
- Property-based and unit tests for prompt content validation
