# Screenshot - Page Capture

## Overview

Take screenshots of webpages.

## When to Use

- Need visual inspection of page
- Debugging UI issues
- Verifying page state
- Capturing layout

## Available Actions

| Action | Description |
|--------|-------------|
| `skill_screenshot` | Take screenshot on a web page with tab_id |
