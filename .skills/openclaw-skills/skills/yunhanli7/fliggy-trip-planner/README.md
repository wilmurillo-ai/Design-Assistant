# fliggy-trip-planner（智能行程规划师）

## 一句话描述

解决"想做旅行攻略但懒得跨平台查航班、比价酒店、筛选景点"的痛点，只需告诉它你想去哪、玩几天、预算多少，一键生成包含交通、住宿、景点、餐厅的完整行程方案。

## 解决的核心问题

- ❌ 不用在携程、去哪儿、小红书之间来回切换
- ❌ 不用自己计算景点之间的距离和路线合理性
- ❌ 不用担心漏掉必去景点或踩雷餐厅
- ❌ 不用手动整理预算表格

## 使用场景 & 例子

| 场景 | 用户说的话 | Skill 会做什么 |
|------|-----------|---------------|
| 家庭出游 | "暑假想带爸妈去北京玩5天，预算人均3000" | 推荐适合老人的慢节奏路线，避开暴走行程，标注无障碍设施 |
| 情侣度假 | "想跟女朋友去三亚过纪念日，要浪漫一点的" | 安排海景酒店、日落餐厅、私人海滩，推荐拍照打卡点 |
| 商务出差+游玩 | "去深圳开会，提前两天到，想顺便玩玩" | 根据会议地点推荐附近酒店，紧凑安排两天精华游 |
| 美食之旅 | "想去成都吃火锅，四天三晚" | 行程以美食为主线，穿插宽窄巷子、熊猫基地等景点 |
| 亲子游 | "带6岁娃去上海，三天，别太累" | 推荐迪士尼、科技馆、外滩，标注儿童友好餐厅和休息点 |
