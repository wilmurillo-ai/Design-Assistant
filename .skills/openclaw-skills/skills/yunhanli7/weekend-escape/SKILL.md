# 周末逃跑计划 (Weekend Escape)

专为忙碌都市人设计的周末短途旅行方案，周五下班出发，周日晚上返回，不用请假也能享受诗和远方。

## 触发场景

- 用户问"周末去哪玩"
- 用户说"想逃离城市"
- 用户需要"短途旅行推荐"
- 用户说"这周太累了，想出去透透气"
- 用户询问周边游、短途游

## 核心能力

### 1. 智能目的地推荐

根据出发城市和偏好，推荐 2-3 个目的地：

**距离分级**：
- **2小时圈**：高铁/自驾可达，适合放松（如北京→天津、上海→杭州）
- **3-4小时圈**：稍远但值得，适合探索（如北京→青岛、上海→黄山）
- **飞行圈**：1.5小时飞行，周末也能去（如北京→大连、上海→厦门）

**主题分类**：
- **自然治愈**：山水、海岛、森林、温泉
- **人文探索**：古镇、博物馆、艺术区
- **美食之旅**：地方特色、夜市小吃
- **活力运动**：滑雪、冲浪、徒步、骑行
- **躺平发呆**：民宿、咖啡馆、慢生活

### 2. 周末时间规划

标准周末 48 小时安排：

```
周五 Evening
  19:00 下班出发（高铁/飞机/自驾）
  22:00 抵达目的地，入住
  23:00 夜宵/休息

周六 Full Day
  09:00 自然醒，当地早餐
  10:00-12:00 核心景点 A
  12:00-14:00 特色午餐
  14:00-17:00 核心景点 B / 体验活动
  17:00-19:00 休息/咖啡馆
  19:00-21:00 特色晚餐
  21:00+ 夜景/酒吧/夜市

周日 Half Day
  09:00 早餐
  10:00-12:00 轻松活动（市集/公园/购物）
  12:00-14:00 告别午餐
  15:00 出发返程
  18:00-20:00 抵达出发城市
```

### 3. 数据获取流程

```
Step 1: 确定出发城市（询问或推断）

Step 2: 推荐 2-3 个目的地
  基于距离、季节、用户偏好筛选

Step 3: 对每个目的地搜索
  flyai search-flight --origin <出发地> --destination <目的地> --dep-date <本周五> --back-date <本周日> --sort-type 3
  flyai search-hotel --dest-name <目的地> --check-in-date <周五> --check-out-date <周日> --sort rate_desc --max-price <预算>
  flyai search-poi --city-name <目的地> --poi-level 4,5

Step 4: 生成对比方案
```

## 输出格式

### 1. 周末逃跑方案对比

| 目的地 | 交通方式 | 单程时间 | 人均预算 | 适合人群 | 亮点 |
|--------|----------|----------|----------|----------|------|
| 杭州 | 高铁 | 1.5h | ¥800 | 文艺青年 | 西湖、茶山 |
| 青岛 | 飞机 | 1.5h | ¥1200 | 海边控 | 啤酒、海鲜 |
| 黄山 | 高铁 | 3h | ¥1000 | 户外党 | 云海、温泉 |

### 2. 推荐方案详情

**推荐理由**：一句话说明为什么选这个目的地

**交通**：
- 去程：周五晚 XX:XX 出发
- 返程：周日晚 XX:XX 返回
- 预订链接

**住宿**：
- 酒店名称 + 图片
- 位置优势
- 价格 + 预订链接

**48小时行程**：
- 按时间线详细安排
- 每个时段包含地点、活动、预计花费
- 景点图片 + 链接

**美食地图**：
- 必吃清单（早餐/午餐/晚餐/夜宵）
- 推荐餐厅 + 招牌菜

### 3. 逃跑清单

```
周末逃跑 Checklist:
- [ ] 预订周五晚交通
- [ ] 预订周日晚返程
- [ ] 预订酒店
- [ ] 查看目的地天气
- [ ] 收拾行李（建议背包即可）
- [ ] 设置周五闹钟（别加班太晚）
- [ ] 充电宝、身份证、现金
```

### 4. Plan B

如遇天气不好/没订到票：
- 替代目的地推荐
- 或改为本地深度游方案

## 特色功能

### 随机盲盒模式
当用户说"随便推荐一个"：
- 从目的地库随机选择
- 制造惊喜感
- 附上"盲盒目的地揭秘"仪式感

### 季节限定推荐
- 春季：赏花、踏青
- 夏季：海岛、避暑
- 秋季：红叶、银杏
- 冬季：温泉、滑雪

### 社交友好方案
标注适合拍照打卡的地点，方便用户发朋友圈。

## 示例对话

**用户**：这周末想逃离北京，推荐个地方

**执行流程**：
1. 确认出发地=北京，时间=本周末
2. 推荐 3 个方案：天津（近）、青岛（海）、大同（古建）
3. 分别查询交通、酒店、景点
4. 生成对比表格和详细行程

**用户**：预算 1000 以内，想看海

**执行流程**：
1. 筛选海边目的地：北戴河、青岛、大连
2. 按预算排序推荐
3. 重点推荐北戴河（最近最省）

## 注意事项

1. 优先考虑交通便利性，周末时间宝贵
2. 酒店推荐靠近火车站/机场或市中心
3. 行程不要太满，留出发呆时间
4. 标注每个方案的"累人指数"（轻松/适中/暴走）
5. 提醒用户提前订票，周末热门目的地容易售罄
