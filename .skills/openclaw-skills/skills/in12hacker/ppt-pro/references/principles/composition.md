# 构图与留白

> 留白不是"浪费空间"，而是信息的呼吸。好的构图引导视线，差的构图让视线迷路。

## 格式塔原则

人脑会自动把视觉元素组织为有意义的整体：

| 原则 | 含义 | PPT 应用 |
|------|------|---------|
| **接近性** | 靠近的元素被视为一组 | 卡片内部元素间距明显小于卡片之间间距 -- 间距比例传达关系 |
| **相似性** | 外观相似的元素被视为同类 | 同级卡片用相同的 card_style 基因（但可以在 card_style 间灵动混合） |
| **连续性** | 视线沿线条/方向流动 | 时间线/流程图的节点用连线引导视觉路径 |
| **闭合性** | 人脑自动补全不完整的形状 | 卡片边界可以是暗示而非硬切（如 outline 的极淡描边） |
| **图底关系** | 前景(图)和背景(底)要清晰分离 | 不同 card_style 之间的存在感差异天然制造图底分离 |

## 三分法

将画布横纵各三等分，重要元素放在交叉点或线上：

```
┌───┬───┬───┐
│   │   │   │  交叉点是视觉强点
├───┼───┼───┤  标题放上 1/3 线
│   │   │   │  核心数据放中心或右上交叉点
├───┼───┼───┤
│   │   │   │
└───┴───┴───┘
```

- 不要所有内容都居中（除非是章节封面/金句页）
- 左上角是自然阅读起点，重要信息从这里开始
- **灵动应用**：三分法是参考线不是铁轨 -- 可以刻意让核心元素偏离交叉点，制造非对称的视觉张力

## 留白的三个层级

| 层级 | 位置 | 作用 |
|------|------|------|
| **宏观留白** | 页面边距（安全区） | 把内容框在画布安全区内，防止贴边 |
| **中观留白** | 卡片之间的间距 | 区分不同的信息组 |
| **微观留白** | 卡片内部元素间距 | 区分标题/正文/数据/图表的层级 |

**留白递增法则**：层级越高的间距越大。微观 < 中观 < 宏观。如果三者大小接近，层级感就消失了。

**灵动应用**：留白不必均匀。可以用极大的留白把某个核心元素"孤立"出来（如单一焦点版式中 120px 数字周围的大面积空白），这就是用留白制造"无声的戏剧性"。

## 自检

- 页面是否有明确的"呼吸空间"（不是每寸都塞满）？
- 去掉所有文字后，布局骨架本身是否清晰有序？
- 三个层级的留白是否递增（微观 < 中观 < 宏观）？
