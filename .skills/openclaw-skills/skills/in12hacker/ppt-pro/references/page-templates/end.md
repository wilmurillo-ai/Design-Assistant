# 结束页 -- 演讲的最后一个视觉印记

> 结束页是观众离开前看到的最后一帧。它决定了观众带走的是"印象深刻"还是"终于结束了"。

## 结束页的灵魂

结束页不是封面的复制粘贴版，而是它的**收束镜像** -- 观众看到结束页时应该无意识地回忆起封面的视觉印记，产生"首尾相衔"的完整感。

**一个好结束页只需要做到两件事**：
1. 让观众**记住最核心的一个结论**（不是 5 个要点，是 1 个）
2. 让观众知道**下一步该做什么**（行动号召/联系方式/思考题）

## 灵动设计指引

### 核心结论是唯一的"必须"
- 结束页必须有一个清晰的核心信息 -- 无论是总结标题、核心结论、行动号召还是一个发人深省的问题
- 这个核心信息应该是画面中视觉权重最大的元素

### 其他所有元素都是"可选的收束工具"
- **要点回顾** -- 不是必须的，也不该做成编号列表。可以是散落的关键词药丸、可以是一句精炼的结论占据画面大部分空间、甚至可以完全省略（核心结论本身就够了）
- **行动号召** -- 如果有，它应该是画面中最后的视觉停留点。可以用 accent 色块/底线/高亮让它"发光"
- **品牌信息** -- 与封面形成位置呼应但不必完全相同
- **装饰** -- 与封面的装饰系统呼应（同一套风格 DNA），但具体技法可以不同

### 与封面的呼应不是复制
"呼应"的灵动在于"似曾相识但别有洞天"：
- 色调呼应（相同的色温，但构图完全不同）
- 元素回声（封面的某个装饰元素以变体形式再现）
- 情绪闭环（封面是"宏大的开启"，结束页是"沉思的收束"）

### 绝对禁止
- **禁止** 做成封面的换皮版
- **禁止** 用编号列表堆砌 5 条以上的要点回顾（那是目录不是结束页）
- **禁止** 完全没有核心信息只放一个"谢谢"

## 唯一硬约束

- 所有颜色通过 CSS 变量引用
- 装饰用真实 DOM 节点
