# 章节封面页 -- 演讲中的呼吸

> 章节封面是"呼吸页"，它的存在不是为了传达信息，而是为了**让观众的大脑在信息密集区之间获得片刻松弛**。

## 章节封面的灵魂

章节封面是演讲节奏中的"逗号" -- 它告诉观众"刚才那个话题告一段落，接下来是新的篇章"。它的视觉重量应该是所有页面中最低的。

**一个好章节封面只需要做到一件事**：
让观众清晰地感知到"进入了新的章节"

## 灵动设计指引

### 章节标题是唯一的"必须"
- 必须有能代表新章节主题的标题
- 标题的大小、位置、方向完全自由

### 所有构图手法都是开放的
以下只是灵感种子，不是清单：
- 标题可以极度偏心 -- 贴靠画面的某个边缘，其余全是留白
- 编号可以超大（120px+）但极低透明度铺在背景，标题在其上浮现
- 导语可以极小极远 -- 与标题形成"大 vs 微"的尺度戏剧
- 配图做氛围底纹时应该是"一层呼吸的纱"（opacity 5-10%）
- 也可以什么都不加 -- 纯色背景 + 一行标题就是最极致的"呼吸"

### 章节封面之间必须变化
- 连续两个章节封面**绝对禁止**使用相同的构图
- 如果第一个是"标题居左 + 大编号背景"，下一个就应该是"标题居中 + 纯留白"或"标题底部 + 氛围配图"
- 变化的维度：标题位置、编号处理方式、留白分布、有无配图

### 绝对禁止
- **禁止** 所有章节封面都是"PART XX 居中 + 标题居中 + 导语居中"
- **禁止** 章节封面比内容页更复杂（它是呼吸不是高潮）

## 唯一硬约束

- 所有颜色通过 CSS 变量引用
- 装饰用真实 DOM 节点
