# 布局模式库 (Layout Pattern Library)

> 本文档定义 12 种经过验证的高质量布局模式。根据页面内容类型选择最匹配的模式，然后结合 prompt-4 中的高级组件三件套（顶线、insight 条、圆形图标）来实现。

## 布局选择指南（先查此表）

| 内容类型 | 推荐布局 | 备选 |
|---------|---------|------|
| A vs B 对比 | 1. 对比面板 | — |
| 3 个并列概念 | 2. 三列卡片 | 10. 三列子卡片 |
| 核心 + 周边关系 | 3. 中心辐射图 | — |
| 6-8 个模块/功能 | 4. 2x4 网格 | — |
| 数据分析 | 5. 图表+侧栏 | 8. 象限分析图 |
| 时间线/路线图 | 6. 三阶段时间线 | — |
| 系统状态展示 | 7. Dashboard | — |
| 能力矩阵 | 8. 象限分析图 | 5. 图表+侧栏 |
| 多维度关注点 | 9. 左列表+右概念图 | 12. 四象限卡片 |

## 12 种布局模式

### 1. 对比面板 (Comparison Panels)
**适用**：对比新旧/好坏/前后两种方案

```
┌──────────────────────────────────────────┐
│ 标题 + 副标题                              │
│ ┌──────────────┐  ┌──────────────────┐   │
│ │ 旧方案 [标签] │  │ 新方案 [标签] 蓝框 │   │
│ │ ○→○→○ 流程  │  │ ○→○→○ 流程      │   │
│ │ ──────────── │  │ ────────────     │   │
│ │ • 要点1       │  │ • 要点1 (hl)    │   │
│ │ • 要点2       │  │ • 要点2 (hl)    │   │
│ └──────────────┘  └──────────────────┘   │
│ [insight 条]                              │
└──────────────────────────────────────────┘
```

关键：面板内有**横向箭头流程图**（圆形图标 + → + 标签）；面板标题旁有标签徽章；右侧用蓝框强调。

### 2. 三列卡片 (Triple Column Cards)
**适用**：展示 3 个并列概念/阶段

```
┌──────────────────────────────────────────┐
│ 标题 + 副标题                              │
│ ┌──────────┐ ┌──────────┐ ┌──────────┐  │
│ │ [○图标]   │ │ [○图标]   │ │ [○图标]   │  │
│ │ 标题      │ │ 标题      │ │ 标题      │  │
│ │ 子问题列表 │ │ 子问题列表 │ │ 子问题列表 │  │
│ │ [底部标签] │ │ [底部标签] │ │ [底部标签] │  │
│ └──────────┘ └──────────┘ └──────────┘  │
│ 底部 2-3 行加粗洞察                        │
│ [insight 条]                              │
└──────────────────────────────────────────┘
```

关键：每列有圆形图标 + 标题 + 分隔线 + 底部标签（PAST/PRESENT/FUTURE 式）；第三列可用蓝框强调。

### 3. 中心辐射图 (Center Radial)
**适用**：展示一个核心概念与周围关联元素

```
┌──────────────────────────────────────────┐
│ 标题 + 副标题                              │
│         ○卫星1                            │
│   ○卫星2      ┌──────┐      ○卫星4       │
│               │ 中心   │                   │
│   ○卫星3      └──────┘      ○卫星5       │
│                                           │
│ • 要点列表              [关键点卡片]        │
│ [insight 条]                              │
└──────────────────────────────────────────┘
```

关键：中心用大图标+标题；卫星用圆形图标+标签；SVG `<line>` 连线；右下角放"落地关键点"卡片。

### 4. 2x4 网格 (Grid Cards)
**适用**：展示 6-8 个并列模块/功能

```
┌──────────────────────────────────────────┐
│ 标题 + 副标题                              │
│ [组1标题]      [组2标题]      [组3标题]     │
│ ┌────┐┌────┐ ┌────┐┌────┐ ┌────┐┌────┐  │
│ │○名 ││○名 │ │○名 ││○名 │ │○名 ││○名 │  │
│ │说明 ││说明 │ │说明 ││说明 │ │说明 ││说明 │  │
│ └────┘└────┘ └────┘└────┘ └────┘└────┘  │
│ ┌────┐┌────┐   (第二行)                    │
│ [insight 条]                              │
└──────────────────────────────────────────┘
```

关键：每个卡片有不同颜色的圆形图标 + 标题 + 描述；顶部有分组标题带颜色竖线。

### 5. 图表+侧栏列表 (Chart + Side List)
**适用**：数据驱动的分析页

```
┌──────────────────────────────────────────┐
│ 标题 + 副标题                              │
│ ┌───────────────────┐ ┌────────────────┐ │
│ │ [标签] SVG 图表     │ │ 白板结论        │ │
│ │ ── 正常 ── 异常    │ │ 信号洞察...     │ │
│ │ (折线/柱图/面积图)  │ │                 │ │
│ │                    │ │ [○] 要点1       │ │
│ │                    │ │ [○] 要点2       │ │
│ │ [X轴标签]          │ │ [○] 要点3       │ │
│ └───────────────────┘ └────────────────┘ │
│ [insight 条]                              │
└──────────────────────────────────────────┘
```

关键：图表用 SVG `<line>`/`<path>`；右侧列表每条带圆形图标；图表上方有标签说明。

### 6. 三阶段时间线 (Timeline)
**适用**：展示进展阶段、路线图

```
┌──────────────────────────────────────────┐
│ 标题 + 副标题                              │
│      [✓]━━━━━━━━[◎]━━━━━━━━[→]          │
│     阶段1        阶段2       阶段3        │
│ ┌──────────┐ ┌──────────┐ ┌──────────┐  │
│ │ 完成项    │ │ 进行项    │ │ 待办项    │  │
│ │ • ...    │ │ ✓ ...   │ │ • ...    │  │
│ └──────────┘ └──────────┘ └──────────┘  │
│ [insight 条]                              │
└──────────────────────────────────────────┘
```

### 7. Dashboard 模拟 (Dashboard)
**适用**：展示系统状态、实时监控

```
┌──────────────────────────────────────────┐
│ 标题                                      │
│ [● Active] [≡ Events] [▲ Alerts]        │
│ ┌────────────────────┐ ┌──────────────┐ │
│ │ 实时事件流           │ │ 2x2 状态卡片 │ │
│ │ 时间 [标签] 事件      │ │ (进度条)    │ │
│ │ ...                 │ │             │ │
│ └────────────────────┘ └──────────────┘ │
│ [insight 条]                              │
└──────────────────────────────────────────┘
```

### 8. 象限分析图 (Quadrant)
**适用**：二维分析（能力×体验、风险×影响 等）

```
┌──────────────────────────────────────────┐
│ 标题 + 副标题                              │
│ ┌──────────────────┐ ┌────────────────┐ │
│ │ SVG 象限图         │ │ [○] 核心要点1  │ │
│ │ (四象限 + 标注点)   │ │ [○] 核心要点2  │ │
│ │                   │ │ [○] 核心要点3  │ │
│ └──────────────────┘ └────────────────┘ │
│ [insight 条]                              │
└──────────────────────────────────────────┘
```

### 9. 左列表+右概念图 (Split Layout)
**适用**：多个关注点 + 核心概念图

```
┌──────────────────────────────────────────┐
│ 标题 + 副标题                              │
│ ┌────────────────────┐ ┌──────────────┐ │
│ │ [○] 关注点1 + 描述   │ │ 概念图/金字塔 │ │
│ │ [○] 关注点2 + 描述   │ │ (顶→中→底)  │ │
│ │ [○] 关注点3 + 描述   │ │             │ │
│ │ [○] 关注点4 + 描述   │ │             │ │
│ └────────────────────┘ └──────────────┘ │
│ [insight 条]                              │
└──────────────────────────────────────────┘
```

### 11. 三列递进+底部总结 (Progressive Cards)
**适用**：展示演进关系（能力→执行→治理）

```
┌──────────────────────────────────────────┐
│ 标题 + 副标题                              │
│ ┌──────────┐ > ┌──────────┐ > ┌────────┐│
│ │ 阶段A     │   │ 阶段B     │   │ 阶段C  ││
│ │ 内容      │   │ 内容      │   │ 内容   ││
│ └──────────┘   └──────────┘   └────────┘│
│                                          │
│ 底部 2x2 总结网格                          │
│ [insight 条]                              │
└──────────────────────────────────────────┘
```

### 12. 四象限卡片 (Quad Cards)
**适用**：展示 4 个独立但相关的概念

```
┌──────────────────────────────────────────┐
│ 标题 + 副标题                              │
│ ┌──────────────────┐ ┌────────────────┐  │
│ │ INSIGHT 01        │ │ INSIGHT 02      │  │
│ │ [○] 英文标题      │ │ [○] 英文标题    │  │
│ │ 中文说明          │ │ 中文说明        │  │
│ └──────────────────┘ └────────────────┘  │
│ ┌──────────────────┐ ┌────────────────┐  │
│ │ INSIGHT 03        │ │ INSIGHT 04      │  │
│ │ [○] 英文标题      │ │ [○] 英文标题    │  │
│ │ 中文说明          │ │ 中文说明        │  │
│ └──────────────────┘ └────────────────┘  │
│ [insight 条]                              │
└──────────────────────────────────────────┘
```

关键：每格有编号（INSIGHT 01）+ 圆形图标 + 英文标题 + 中文说明。
