# 资源菜单速查卡（策划时随时翻看）

> 本文件在每页策划时作为 `{{RESOURCE_MENU}}` 注入 prompt-3 上下文。
> 目的：防止上下文衰减导致 LLM 在策划后半程退化为只用三板斧。

---

## A. 布局工具箱（10 种 -- 全部平等，不是只用前 3 个）

| layout_hint | 适用场景 | 卡片数 | 容易被遗忘？ |
|-------------|---------|-------|------------|
| 单一焦点 | 1 个核心论点/数据独占全场 | 1 | |
| 50/50 对称 | 2 个对等概念碰撞 | 2 | |
| 非对称两栏 | 主概念 + 补充注解 | 2 | |
| 三栏等宽 | 3 个并列要素平等展示 | 3 | 常被忽略 |
| 主次结合 | 1 核心 + 2 佐证 | 3 | |
| 英雄式 | 综述 + 3-4 子展开 | 4-5 | |
| 混合网格 | 4-6 异构信息密集排布 | 4-6 | |
| **L 型** | 主论点 + 侧面佐证 + 底部总结 | 4 | **高频遗忘** |
| **T 型** | 总览 + 展开 + 侧面数据 | 4 | **高频遗忘** |
| **瀑布流** | 多条不等高的信息瀑布 | 4-6 | **高频遗忘** |

**硬约束**：相邻 content 页禁止相同 layout_hint / 全 PPT 任一布局占比 <= 30%

## B. 卡片类型工具箱（13 种 -- 复合类型不是"高级选项"，该用就用）

### 基础类型（prompt-4 内联定义，无需额外资源）

| card_type | 一句话用途 | 何时选 |
|-----------|-----------|-------|
| `text` | 展开论述/解释原理 | 需要段落文字时 |
| `data` | 数据冲击力（必须配 chart_type） | 有具体数字时 |
| `list` | 并列要点 >= 4 条 | 同级信息枚举时 |
| `process` | 步骤/流程/因果链 | 有先后顺序时 |
| `tag_cloud` | 关键词/标签一览 | 技术栈/覆盖范围时 |
| `data_highlight` | 超大数字占据视觉中心 | 封面/章节需要巨型数字时 |

### 复合类型（按需从 blocks/*.md 加载，推荐 transparent card_style）

| card_type | 一句话用途 | 何时选 | 容易被遗忘？ |
|-----------|-----------|-------|------------|
| `timeline` | 时间线/里程碑 | 有时间维度的叙事时 | 常被忽略 |
| `diagram` | 架构图/流程图/层次图 | 有拓扑/层级关系时 | **高频遗忘** |
| `quote` | 名言/金句/权威引用 | 有说服力引语时 | 常被忽略 |
| `comparison` | 双面板 VS 对比 | 有 A vs B 对比时 | 常被忽略 |
| `people` | 团队/人物介绍 | 有人物展示需求时 | |
| `image_hero` | 大图+叠加文字 | 需要视觉冲击力配图时 | |
| `matrix_chart` | 四象限矩阵 | 有二维分类评估时 | **高频遗忘** |

**自检**：如果整个 PPT 没用过任何复合类型 -> 大概率在偷懒，回头看看哪些页面的内容适合 timeline/diagram/comparison

## D. 卡片视觉变体（6 种 card_style -- 每页必须 >= 2 种）

| card_style | 视觉效果 | 适用场景 | 每页限制 |
|------------|---------|---------|---------|
| `filled` | 有背景 + 圆角 | 标准卡片 | 无限制 |
| `transparent` | 无背景无边框 | 大数字/金句/复合组件 | 无限制 |
| `outline` | 透明底 + 1px 描边 | 辅助信息 | 无限制 |
| `accent` | accent 渐变底 + 白字 | 核心强调（视觉锚点） | **<= 1 个** |
| `glass` | 半透明毛玻璃 | 有底图时叠加 | 无限制 |
| `elevated` | 明显阴影悬浮 | 视觉锚点 | **<= 1 个** |

**推荐组合**：accent + transparent + filled（强对比）或 elevated + outline + transparent（层次丰富）

## E. 装饰技法速查（3 层 x 5 种 = 每层选 1 种组合使用）

| 层 | 技法选项 |
|----|---------|
| 背景层 | 光晕(radial-gradient) / 网格点阵 / 斜向扫描线 / 噪点底纹 / 大面积色块 |
| 卡片层 | 左侧竖线 / 顶部色带 / 角标装饰 / 编号气泡 / 底边渐隐线 |
| 页面层 | 大号水印(超大字 opacity 0.04) / 渐隐分隔线 / 浮动几何碎片 / 品牌色块 / 数据残影 |

**硬约束**：相邻内容页的 decoration_hints 至少有 1 个维度不同
