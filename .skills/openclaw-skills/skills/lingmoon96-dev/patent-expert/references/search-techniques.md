# 专利检索技巧

## 检索策略

### 关键词检索

1. 提取核心概念词（中文 + 英文）
2. 扩展同义词/近义词
3. 使用 AND / OR / NOT 组合

示例：
- "机器学习 AND 图像识别 AND (实时 OR 在线)"
- "(deep learning OR neural network) AND (object detection OR visual recognition)"

### 分类号检索

- **IPC**（国际专利分类）— 最通用
- **CPC**（联合专利分类）— Google Patents 支持，更细粒度

常用CPC类目：
- G06F: 计算机技术
- G06N: 人工智能/机器学习
- H04L: 数据传输/网络
- G06T: 图像处理

### 申请人检索

用于竞品分析：
- 搜索特定公司名下所有专利
- 追踪竞争对手的专利申请趋势

## 检索式构建技巧

1. **由宽到窄** — 先宽泛检索了解领域全貌，再逐步缩小范围
2. **多库交叉** — 同一检索式在不同数据库跑一遍，结果互补
3. **时间限定** — 关注近5年的专利（技术迭代快时缩短到3年）
4. **引用追踪** — 找到核心专利后，查看其引用和被引用专利

## 结果筛选标准

1. **标题相关度** — 标题是否直接涉及目标技术
2. **摘要匹配度** — 摘要中的技术特征是否与本方案重叠
3. **权利要求对比** — 独立权利要求的技术特征对比（最关键）
4. **申请日期** — 在本方案之前申请的才构成现有技术
5. **法律状态** — 是否有效、是否已被驳回、是否已过期

## 输出要求

检索报告应包含：
- 检索目标与范围
- 使用的检索式和数据库
- 检索结果列表（专利号、标题、摘要、相关度评分）
- 核心专利的详细分析
- 新颖性/创造性影响评估
- 后续建议
