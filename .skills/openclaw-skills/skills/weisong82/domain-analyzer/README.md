# 域名价值分析 Skill

🔍 **Domain Value Analyzer** - 专业的域名投资评估工具

---

## 📖 功能说明

本 Skill 提供域名投资价值的专业分析，包括：

- ✅ 拼写检查（识别常见拼写错误）
- 📏 长度评估（字符数评分）
- 🌐 后缀评级（.com/.io/.xyz 等价值对比）
- 💡 含义分析（词根识别）
- 🧠 记忆度评估（是否易记顺口）
- ⚠️ 行业风险（加密/博彩/医疗等风险评估）
- 💰 价值估算（注册成本 vs 变现潜力）

---

## 🚀 使用方式

### 方式 1：自然语言触发

直接对话中提及域名即可：

```
tokensfab.com 如何？
这个域名值得注册吗：globaltoken.link
帮我分析下域名：gloabltoken.com
```

### 方式 2：命令行工具

```bash
cd ~/.openclaw/skills/domain-analyzer/scripts
python3 analyze_domain.py <域名>

# 示例
python3 analyze_domain.py tokensfab.com
python3 analyze_domain.py globaltoken.io
```

## 📊 评分体系

### 6 维评估模型

| 维度 | 权重 | 评分标准 |
|------|------|---------|
| **长度** | 20% | ≤6 字符=5 分，7-9=4 分，10-12=3 分 |
| **拼写** | 20% | 正确=5 分，错误=1 分 |
| **含义** | 20% | 清晰热门=5 分，模糊=3 分 |
| **记忆度** | 15% | 易记=5 分，难记=2 分 |
| **后缀** | 15% | .com=5 分，.link=1 分 |
| **行业** | 10% | 低风险=5 分，高风险=2 分 |

### 综合评级

| 总分 | 评级 | 建议 |
|------|------|------|
| 90-100 | 极品 | ✅ 立即注册 |
| 80-89 | 优质 | ✅ 值得注册 |
| 70-79 | 良好 | ⚠️ 谨慎考虑 |
| 60-69 | 普通 | ⚠️ 如有项目可用 |
| 40-59 | 一般 | ❌ 不建议 |
| <40 | 较差 | ❌ 放弃 |

## 💡 分析示例

### 示例 1：tokensfab.com

```
综合评分：90/100 ⭐⭐⭐⭐⭐
投资等级：极品
潜在变现：10 万 -100 万+
建议：✅ 值得注册
```

**优势：**
- 9 字符，短小精悍
- .com 后缀，认知度最高
- 含义清晰："代币工厂"
- 拼写正确，易记顺口

---

### 示例 2：gloabltoken.com

```
综合评分：66/100 ⭐⭐⭐
投资等级：普通
潜在变现：500-2000 元
建议：⚠️ 谨慎考虑
```

**缺陷：**
- 拼写错误：`gloabl` 应为 `global`
- 致命硬伤，用户输入正确拼写会访问不到

---

### 示例 3：globaltoken.link

```
综合评分：74/100 ⭐⭐⭐
投资等级：良好
潜在变现：2000-1 万
建议：✅ 值得注册（如有项目）
```

**评价：**
- 拼写正确，含义清晰
- .link 后缀认知度低，影响价值
- 有具体项目可使用，纯投资需谨慎

## 📈 域名投资基本原则

| 原则 | 说明 |
|------|------|
| **拼写必须正确** | 这是底线，没有例外 |
| **越短越好** | 6-10 字符最佳 |
| **含义清晰** | 一眼看懂是做什么的 |
| **避免数字/连字符** | 价值大打折扣 |
| **优先 .com** | 其他后缀价值低 50-90% |
| **警惕敏感行业** | 博彩/加密/医药监管风险高 |
