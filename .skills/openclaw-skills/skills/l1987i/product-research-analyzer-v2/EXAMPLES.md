### 示例 5：批量调研

#### 脚本
```bash
#!/bin/bash

products=("eilik" "芙崽" "小爱同学")

for product in "${products[@]}"; do
  echo "🔍 正在调研：$product"
  python3 scripts/research.py "{\"product_name\": \"$product\", \"target_project\": \"蝉小狗\"}"
  echo ""
done
```

#### 执行
```bash
chmod +x batch_research.sh
./batch_research.sh
```
