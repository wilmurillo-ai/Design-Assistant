## Day 16: 工具调用 (Function Calling)

### 学习内容

今天完成了 Python Agent 学习的 Day 16 - 工具调用。

#### 核心概念
工具调用是 AI Agent 从「对话」走向「行动」的关键技术，让 AI 能够：
1. 理解何时需要调用外部工具
2. 构造正确的参数
3. 执行工具
4. 将结果返回给模型继续处理

#### 实现的代码文件

1. **weather_tool.py** - 基础工具调用示例
   - Tool 基类设计
   - WeatherTool / CalculatorTool / SearchTool 实现
   - ToolRegistry 工具注册表
   - ToolAgent 核心 Agent 类

2. **advanced_tools.py** - 高级工具调用模式
   - 并行工具调用（asyncio + ThreadPoolExecutor）
   - 工具链模式（串行执行，上游输出作为下游输入）
   - 带钩子的工具（执行前后日志、验证）

3. **real_llm_integration.py** - 真实 LLM 集成
   - ToolDefinition 数据类
   - OpenAIClient / MockLLMClient
   - ToolCallingAgent 完整实现

#### 关键知识点

- 工具定义遵循 OpenAI Function Calling 规范
- JSON Schema 用于定义参数类型
- 工具返回结果应该是结构化 JSON
- 需要设置最大迭代次数防止无限循环

工具调用是构建实用 AI Agent 的核心能力。通过今天的学习，理解了如何让 AI 不只是回答问题，而是能够实际操作外部系统、调用 API。这是从 ChatBot 到 Agent 的关键一步。

## CLI-Anything Skill 创建

### 背景
主人提到 CLI-Anything 项目，希望创建一个 OpenClaw Skill 来包装它。

### CLI-Anything 简介
- 项目地址: https://github.com/HKUDS/CLI-Anything
- 功能: 让任何软件都能被 AI 控制，把 GUI 软件变成 CLI 工具
- 支持软件: GIMP、Blender、LibreOffice、ComfyUI、Ollama 等 11 个

### 创建的 Skill 文件

**位置**: `~/.openclaw/workspace/skills/cli-anything-wrapper/`

| 文件 | 说明 |
|------|------|
| `SKILL.md` | Skill 定义文档 |
| `run.sh` | 快捷入口脚本 |
| `scripts/run.py` | 主程序 (300+ 行) |
| `scripts/test.sh` | 测试脚本 |
| `references/USAGE.md` | 使用指南 |

### 功能特性
- 支持 11 个软件分类管理
- JSON 格式输出
- 模拟运行模式 (--dry-run)
- 自动安装 CLI-Anything
- 5分钟超时保护

### 使用命令
```bash
openclaw run cli-anything-wrapper --list      # 列出软件
openclaw run cli-anything-wrapper --info      # 显示信息
openclaw run cli-anything-wrapper --install   # 安装 CLI-Anything
openclaw run cli-anything-wrapper --app ollama --args "run llama3.2"
```

### 状态
Skill 框架已完成，等待安装 CLI-Anything 后即可使用。
