### 关键知识点

#### 1. Typer 现代 CLI 框架
- `@app.command()` - 注册命令
- `typer.Argument()` - 位置参数
- `typer.Option()` - 可选参数，支持 `-f`/`--forecast` 短长选项
- 自动类型转换和验证 (`min=1`, `max=7`)
- 自动生成美观的帮助信息

#### 2. 配置管理三优先级
```
环境变量 > 配置文件 > 默认值
```
- `os.getenv()` 读取环境变量
- `yaml.safe_load()` 解析配置文件
- `**data` 字典解包

#### 3. HTTP 客户端最佳实践
- `requests.Session()` - 复用TCP连接
- `raise_for_status()` - 自动检查HTTP错误
- 自定义异常类 `WeatherAPIError`
- 私有方法 `_parse_*` 约定

#### 4. 数据模型
- `@dataclass` - 自动生成 `__init__`, `__repr__`
- `@property` - 方法变属性 (`temp_fahrenheit`)
- `__str__` - 自定义字符串表示

#### 5. Rich 富文本输出
- `Panel` - 带边框面板
- `Table` - 表格展示
- 颜色样式 `[bold]`, `[red]`, `[green]`

#### 6. 测试
- `@pytest.fixture` - 测试装置
- `@patch` - Mock外部依赖
- `Mock()` - 创建假对象
