### 📚 学习笔记

#### 1. 记忆系统核心概念

**为什么需要记忆？**
- LLM 每次对话独立，没有记忆
- 需要记住：用户名字、偏好、对话历史

**两种记忆类型：**
| 类型 | 特点 | 存储方式 |
|------|------|---------|
| 短期记忆 | 当前对话，容量有限 | 列表/数组 |
| 长期记忆 | 跨会话持久化 | 向量数据库 |

#### 2. 向量与 Embedding

**Embedding = 文本 → 向量**
```
"苹果" → [0.2, -0.5, 0.8, ...]  # 128维或更高
```

**核心原理：**
- 语义相似的文本，向量距离近
- 用余弦相似度计算距离
- 范围：1.0(相同) → 0.0(无关) → -1.0(相反)

**实际 Embedding API：**
- OpenAI: `text-embedding-3-small`
- 百度千帆: Embedding-V1
- 智谱: embedding-2

#### 3. 记忆系统架构

```
AgentMemory (总控)
├── ShortTermMemory (短期记忆)
│   └── messages: List[Message]  # 最近10条对话
├── SimpleVectorDB (向量数据库)
│   └── records: Dict[id, VectorRecord]  # 持久化存储
└── EmbeddingClient (向量化)
    └── embed(text) → List[float]
```

#### 4. 核心代码流程

**存记忆：**
```python
# 1. 文本 → 向量
vector = embedding.embed("用户叫小明")

# 2. 创建记录
record = VectorRecord(id=..., text=..., vector=...)

# 3. 存入数据库
vector_db.add(record)
```

**取记忆：**
```python
# 1. 查询 → 向量
query_vector = embedding.embed("我是谁？")

# 2. 搜索相似向量
results = vector_db.search(query_vector, top_k=3)

# 3. 返回最相关的记忆
return [record.text for record in results]
```

**构建上下文：**
```python
def build_context(user_input):
    # 1. 检索相关长期记忆
    memories = recall(user_input)
    
    # 2. 取近期对话历史
    conversation = get_recent_messages()
    
    # 3. 拼接成完整提示
    return "相关背景 + 对话历史 + 当前输入"
```

#### 5. 关键类和方法

| 类/方法 | 作用 |
|---------|------|
| `ShortTermMemory` | 管理当前对话上下文 |
| `SimpleVectorDB` | 向量数据库存储和搜索 |
| `AgentMemory.remember()` | 存入长期记忆 |
| `AgentMemory.recall()` | 从长期记忆检索 |
| `AgentMemory.build_context()` | 构建 LLM 提示上下文 |
| `cosine_similarity()` | 计算向量相似度 |

### 💻 代码实践

✅ 已运行 `memory_system.py` 演示

**三个场景验证：**
1. 用户介绍自己 → 记住关键信息（名字、职业）
2. 同一会话询问 → 用短期记忆直接回答
3. 新会话询问 → 从长期记忆检索（即使短期记忆已清空）

---

### ✅ 今日总结

**核心公式：**
```
用户输入 → Embedding → 向量 → 语义搜索 → 相关记忆 + 对话历史 → LLM → 回复
```

**类比理解：**
- 短期记忆 = 工作台上的便利贴（临时用）
- 长期记忆 = 文件柜里的档案（永久存）
- 向量 = 语义指纹（相似内容指纹相似）
- 语义搜索 = 按指纹找相似档案

**收获：**
1. 理解了记忆系统在 AI Agent 中的重要性
2. 掌握了 Embedding 和向量相似度的核心原理
3. 学会了整合短期记忆和长期记忆的架构设计
4. 能看懂并修改记忆系统的代码实现

## Claude Code 源码泄露事件（2026-04-01 补充）
