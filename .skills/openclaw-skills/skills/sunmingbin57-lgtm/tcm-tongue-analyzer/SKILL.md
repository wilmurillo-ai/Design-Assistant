# 中医舌像分析技能

## 产品概述

**中医舌像分析**是一款专业的中医舌诊辅助工具，通过人工智能技术自动分析舌象照片，提供准确的中医辨证建议。

### 核心价值
- **专业准确**：基于经典中医理论和大量临床数据
- **操作简便**：一键上传，自动分析
- **快速高效**：30秒内完成全面分析
- **学习辅助**：详细解释分析结果

## 功能特性

### 1. 舌色自动分类
- **淡白舌**：阳虚、气血不足
- **淡红舌**：正常或轻度气虚
- **红舌**：热证、阴虚火旺
- **绛舌**：营血热盛
- **紫舌**：血瘀、寒凝

### 2. 舌形识别
- **胖大舌**：脾肾阳虚、水湿内停
- **瘦薄舌**：阴血亏虚
- **齿痕舌**：脾虚湿盛
- **裂纹舌**：阴液亏耗

### 3. 舌苔分析
- **苔色**：白、黄、灰、黑
- **苔质**：薄、厚、润、燥、腻、腐
- **分布**：均匀、剥落、地图舌

### 4. 特殊特征识别
- **瘀点瘀斑**：血瘀证
- **舌下络脉**：青紫、迂曲
- **舌乳头**：充血、萎缩

## 使用方法

### 基本命令
```bash
# 分析单张舌象照片
tcm-tongue analyze --image "舌象照片路径"

# 批量分析
tcm-tongue batch --folder "舌象照片文件夹"

# 生成详细报告
tcm-tongue report --image "照片路径" --output "报告文件"
```

### 参数选项
- `--image`：舌象照片路径（支持JPG、PNG格式）
- `--format`：输出格式（text/json/html）
- `--detail`：详细程度（basic/standard/detailed）
- `--compare`：对比分析多张照片

## 技术原理

### 图像处理流程
1. **预处理**：标准化光线、角度、大小
2. **舌体分割**：分离舌体与背景
3. **特征提取**：颜色、纹理、形状特征
4. **分类识别**：基于深度学习模型分类
5. **辨证推理**：结合中医知识库推理

### 准确率指标
- 舌色分类准确率：92%
- 舌形识别准确率：88%
- 舌苔分析准确率：85%
- 整体辨证准确率：80%
