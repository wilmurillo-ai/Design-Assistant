# Bundle & cross-sell playbook

For `cart-bundle-aov-upsell` when you need extra pairing logic or copy patterns.

---

## 1. Pairing logic by cart signal

| Cart signal | Complement type | Rationale angle |
|-------------|-----------------|-----------------|
| Single hero SKU | Accessory / care / refill | Complete the use case |
| Beauty shade chosen | Primer, remover, brush | Same routine, same look |
| Electronics main unit | Case, cable, warranty | Protection + peace of mind |
| Apparel size selected | Belt, socks, layer piece | Outfit completion |
| Low cart vs free ship threshold | Small add-on, samples | Shipping efficiency |
| New launch in catalog | Trial size + hero SKU | Launch trial bundle |

---

## 2. Discount rationale (not generic)

Prefer **specific** stories:

- "Ship one box" / combined fulfillment
- "First-time bundle" for launch SKUs
- "Routine saver" (step 1 + step 2)
- "Protection pack" (device + case + warranty)
- "Gift-ready" (wrap + card) — seasonal

Avoid empty "10% off" with no reason.

---

## 3. One-click copy patterns

- Button: **Add complete set — save $X** / **Add [Complement] — ships with this order**
- Subline: **Fits the [Main] you chose** / **One tap — no new checkout step**
- Urgency (light): **Adds to this shipment only**

Keep primary CTA under ~6 words when possible.

---

## 4. Checkout bounce "bait"

When bounce is high:

- **Threshold nudge**: item just above gap to free shipping
- **Low decision weight**: sample, sticker pack, mini — low cognitive load
- **Risk reversal**: free returns on add-on only if policy allows

---

## 5. What to avoid

- Substitute of same role as main (unless upsell tier)
- Below-margin doorbusters unless strategic
- Irrelevant category jumps without story
