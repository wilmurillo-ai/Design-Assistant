# 变更日志

所有重要的变更都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

## [3.1.0] - 2026-03-04

### 新增
- ✨ 配置向导：首次使用时自动提示配置 GitCode API Token
- ✨ 配置文件支持：使用独立配置文件 `config/gitcode.conf`（更安全）
- ✨ 环境变量支持：可通过 `GITCODE_API_TOKEN` 环境变量配置
- 📝 安装后脚本：安装完成后自动提示配置步骤
- 📝 配置模板：提供 `config/gitcode.conf.example` 模板文件
- 🔒 安全增强：配置文件权限自动设置为 600

### 变更
- 🔄 Token 配置从 TOOLS.md 迁移到技能内部配置文件
- 🔄 gitcode-api.sh 新增 `setup` 命令用于配置向导
- 📝 更新文档，强调配置步骤

### 移除
- ❌ 移除对 TOOLS.md 的依赖（Token 配置）

### 优化
- 🔒 Token 不再硬编码在脚本中
- 🔧 配置优先级：环境变量 > 配置文件 > 默认值

## [3.0.0] - 2026-03-04

### 新增
- ✨ 使用 GitCode API 替代浏览器自动化
- ✨ 新增 `gitcode-api.sh` 辅助脚本
- ✨ 新增 README.md 使用文档

### 变更
- 🔄 步骤 1：从浏览器打开页面改为 API 获取 PR 信息
- 🔄 步骤 2：从浏览器查看变更改为 API 获取文件列表
- 🔄 步骤 5：从浏览器输入评论改为 API 发布评论
- 🔄 步骤 6：从浏览器输入 LGTM 改为 API 发布
- 🔄 自动审查模式：从浏览器访问通知页改为 API 获取 PR 列表

### 移除
- ❌ 移除所有浏览器自动化相关内容
- ❌ 移除浏览器配置说明
- ❌ 移除浏览器相关常见问题（ref 不稳定、元素定位等）

### 优化
- 🚀 大幅提高稳定性和可靠性
- 🚀 减少执行时间（无需等待页面加载）
- 🚀 简化操作流程（无需处理浏览器元素）
- 📝 更新常见问题，聚焦 API 相关问题

### 修复
- 🐛 解决浏览器 ref 编号不稳定问题
- 🐛 解决评论输入到错误输入框问题
- 🐛 解决浏览器连接丢失问题
