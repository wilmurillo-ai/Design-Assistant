## 示例对比

### ❌ 原写法

### ✅ 优化写法
接口QPS提升200% | 负责XX系统核心模块开发，通过缓存优化和SQL优化提升系统性能
- 系统架构：设计并实现XX模块，提供X个RESTful API，支撑X万DAU
- 性能优化：
  - 引入Redis缓存热点数据，数据库查询减少70%，接口响应从200ms降至80ms
  - 优化慢查询SQL，添加联合索引，查询耗时从500ms降至50ms
  - 使用Kafka异步处理削峰，系统QPS从1000提升至3000
- 代码质量：编写单元测试（覆盖率80%+），参与Code Review，保证代码质量
