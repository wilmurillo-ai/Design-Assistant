**背景**：OGV的IP上线与宣发长期依赖运营基于经验手动完成搜索补词与投放策略拆解，存在漏配风险与重复劳动成本高的问题。

**我的职责**：

**成果**：
- 替代2-3名运营人力/周
- 补词覆盖率提升XX%
- 投放策略一致性显著改善

### 项目二：2025萌战
**背景**：萌战为B站核心二次元角色票选活动，停办七年后重启。

**我的职责**：

- 对活动组件进行结构化拆解以支持后续复用

**成果**：

- 相关稿件VV 2200w，超预期20%

**我的职责**：
- 统一主站与OTT预约数据口径，重构预约组件与数据合并逻辑
- 推动预约模块配置化与组件化

**成果**：

- 拜年纪总预约230w，同比增加1.3%
- 预约转化率增加4.3%

## 得物 国际事业部 产品实习生（2025.03-2025.06）

### 项目一：新个卖限制出价节点
**背景**：新卖家利用发货节点逻辑漏洞，在未真实到货情况下快速累积单量并逃避系统监管导致平台资损。

**我的职责**：

**背景**：美国区存在常规业务需多段物流与质检，整体履约流程耗时长、成本高。

**我的职责**：

**成果**：

## 使用建议

**输入技巧：**
1. 时间、公司、岗位尽量写清楚
2. 有数据的尽量写数据
3. 没数据的写具体做了什么
4. 可以分段输入，系统会合并

**系统会做的：**
1. 自动提取关键信息
2. 识别缺失信息并询问
3. 按时间排序
4. 合并相似项目
5. 根据岗位方向优化表述
