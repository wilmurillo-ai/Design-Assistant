# x402 Payment Header — Construction Guide

When a signal returns `402 Payment Required`, build an `X-Payment` header and retry.

## Format

`X-Payment: base64(JSON({ txhex, payeeLockingScript, priceSats }))`

Where `txhex` is a raw BSV transaction paying `priceSats` satoshis to `payeeLockingScript`.

## Node.js Example (no wallet library required)

```javascript
function buildXPayment(payeeLockingScriptHex, priceSats) {
  const lockingScript = Buffer.from(payeeLockingScriptHex, 'hex');
  const parts = [];
  parts.push(Buffer.from('01000000', 'hex'));   // version (4 bytes LE)
  parts.push(Buffer.from('01', 'hex'));           // input count
  parts.push(Buffer.alloc(32));                   // prev txid (32 zeros)
  parts.push(Buffer.from('ffffffff', 'hex'));     // prev index
  parts.push(Buffer.from('0100', 'hex'));         // empty script
  parts.push(Buffer.from('ffffffff', 'hex'));     // sequence
  parts.push(Buffer.from('01', 'hex'));           // output count
  const val = Buffer.alloc(8);
  val.writeBigUInt64LE(BigInt(priceSats));
  parts.push(val);                                // value: priceSats LE 8 bytes
  parts.push(Buffer.from([lockingScript.length]));
  parts.push(lockingScript);                      // locking script
  parts.push(Buffer.from('00000000', 'hex'));     // locktime

  const txhex = Buffer.concat(parts).toString('hex');
  const proof = { txhex, payeeLockingScript: payeeLockingScriptHex, priceSats };
  return Buffer.from(JSON.stringify(proof)).toString('base64');
}

// Usage
const resp = await fetch(`${BASE}/api/markets/${mid}/oracle/signals`);
if (resp.status === 402) {
  const { payment } = await resp.json();
  const xPayment = buildXPayment(payment.payeeLockingScript, payment.priceSats);
  const paid = await fetch(`${BASE}/api/markets/${mid}/oracle/signals`, {
    headers: { 'X-Payment': xPayment }
  });
  const data = await paid.json(); // paid signals now included
}
```

## Bash Example

```bash
PAYMENT_JSON=$(curl -s "$BASE/api/markets/$MID/oracle/signals" | jq '.payment')
LOCKING_SCRIPT=$(echo $PAYMENT_JSON | jq -r '.payeeLockingScript')
PRICE_SATS=$(echo $PAYMENT_JSON | jq -r '.priceSats')

# Build X-Payment with Node helper
X_PAYMENT=$(node -e "
const s=Buffer.from('$LOCKING_SCRIPT','hex');
const p=[];
p.push(Buffer.from('01000000','hex'));p.push(Buffer.from('01','hex'));
p.push(Buffer.alloc(32));p.push(Buffer.from('ffffffff','hex'));
p.push(Buffer.from('0100','hex'));p.push(Buffer.from('ffffffff','hex'));
p.push(Buffer.from('01','hex'));
const v=Buffer.alloc(8);v.writeBigUInt64LE(BigInt($PRICE_SATS));p.push(v);
p.push(Buffer.from([s.length]));p.push(s);p.push(Buffer.from('00000000','hex'));
const tx=Buffer.concat(p).toString('hex');
console.log(Buffer.from(JSON.stringify({txhex:tx,payeeLockingScript:'$LOCKING_SCRIPT',priceSats:$PRICE_SATS})).toString('base64'));
")

curl -s "$BASE/api/markets/$MID/oracle/signals" -H "X-Payment: $X_PAYMENT" | jq .
```

## Notes

- Brouter verifies the tx structure immediately and serves data; SPV/BEEF confirmation is async (~90s polling)
- Your wallet must broadcast `txhex` to the BSV network independently
- `payment_txid` in the response confirms proof of payment was accepted
