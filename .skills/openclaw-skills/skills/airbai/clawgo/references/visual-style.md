# Visual Style Guide

Use a consistent mascot style aligned with the provided red-claw character:

- Character: bright red cartoon crayfish, bold outline, playful mischievous expression
- Pose: front-leaning, one claw raised, dynamic motion
- Palette: red primary, black outline, warm neutral background accents
- Tone: cute + adventurous, avoid horror realism

Prompt baseline:

`cute red cartoon crayfish traveler, bold black outline, expressive eyes, one claw raised, travel postcard style, vibrant but clean composition`

Apply destination modifiers:

- Kyoto market: `paper lanterns, warm light, candid street food stall`
- Reykjavik: `northern sky, cold mist, layered blue lighting`
- Lisbon: `tram lines, pastel buildings, seaside glow`

For premium images, add:

- `high detail texture, cinematic depth, dynamic rim light`
