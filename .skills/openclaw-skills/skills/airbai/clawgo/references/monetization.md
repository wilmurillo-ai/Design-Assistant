# Monetization

## 1. Tier Matrix

| Feature | Free | Pro |
|---|---:|---:|
| Proactive updates/day | 1 | 3 |
| Image quality | Standard | HD |
| Voice styles | 1 | 4 |
| Rare-location arc | No | Yes |
| Memory itinerary depth | Basic tags | Deep preference graph |

## 2. Revenue Model

Use hybrid monetization:

- Subscription: monthly/annual Pro access
- Add-on packs: temporary photo pack, festival voice pack, special map pack

## 3. Conversion Triggers

Trigger plan nudge only in these moments:

- User hit daily update quota
- User tries rare-location command
- User requests premium voice style

Use soft copy:

- Explain specific value unlocked
- Offer one-tap retry once entitlement becomes active

## 4. Guardrails

- Never block core companionship loop for free users
- Keep free mode emotionally complete but lower in volume/depth
- Prevent paywall spam: max 1 upsell per day
