# Feishu News Skill

A skill that fetches and references news information from Feishu when interacting with users.

## Description

This skill enables the assistant to fetch current news information from Feishu and properly reference news sources when providing information to users. It ensures that all news references include proper attribution to the original sources.

## Features

- Fetch latest news articles from Feishu
- Reference news sources with proper attribution
- Format news information for clear presentation
- Maintain source integrity in all news references

## Usage

When users request news or current events, this skill can:
1. Retrieve relevant news from Feishu
2. Format the information with clear source references
3. Present news updates while maintaining proper attribution

## Integration

This skill integrates with Feishu's news system to provide current information while ensuring that all sources are properly credited when referenced in conversations.

## Example Output

When referencing news, the skill formats information like:

**Latest Tech Innovation Breakthrough**
*Published: 2026-04-13*
Scientists have developed a new breakthrough in renewable energy technology.

📰 Source: [Feishu Tech News](https://feishu.example.com/news/1)
