# Code Generator and Tester Skill

## 描述
一个用于生成代码、保存文件并运行测试的OpenClaw技能。这个技能可以让AI助手根据自然语言需求生成代码、运行代码并执行测试。

## 功能
- 根据自然语言需求生成代码
- 支持多种编程语言（Python、JavaScript、Java等）
- 运行生成的代码并捕获输出
- 运行用户提供的测试代码
- 将代码保存到指定文件路径
- 返回详细的执行结果

## 使用方法

### 生成并运行代码
```
生成一个计算阶乘的Python函数
写一个JavaScript程序来反转字符串
创建一个Java类来表示学生信息
```

### 运行测试
```
运行以下测试代码...
对这个函数运行单元测试
执行测试验证功能
```

### 保存代码
```
生成一个排序算法并保存到 sort.py
将计算器程序保存到 calc.js
```

## 输入参数
- 查询字符串包含需求描述、目标语言和期望的行为

## 输出格式
返回JSON格式的结果，包括：
- 生成的代码
- 运行结果（退出码、标准输出、错误输出）
- 测试结果（如果有）
- 临时文件路径
- 时间戳

## 适用场景
- 快速原型开发
- 代码实验和测试
- 自动化代码生成
- 编程教学和演示
- 算法验证

## 支持的编程语言
- Python
- JavaScript
- Java
- C++
- C
- Go
- Rust
- Bash/Shell
- PHP
- Ruby
- TypeScript

## 依赖
- 对应编程语言的运行环境
- Python 3.7+

## 注意事项
- 生成的代码会暂时保存在临时文件中
- 代码执行有30秒的超时限制
- 某些代码可能需要额外的依赖包
- 执行任意代码可能存在安全风险，建议在沙箱环境中运行
