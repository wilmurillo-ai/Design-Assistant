# Seedance 2.0 提示词撰写指南

## 核心原则

好的提示词是高质量视频生成的基础。Seedance 2.0 对自然语言理解能力很强，以下指南帮助写出更有效的提示词。

## 提示词结构建议

### 基础公式
```
[画面风格] + [主体描述] + [动作/运动] + [镜头语言] + [环境/背景] + [音效/配音]
```

### 时间轴分段描述（推荐）

对于有多个场景变化的视频，使用时间戳分段：
```
[整体风格和设定]
0-3 秒：第一个场景的详细描述；
4-6 秒：第二个场景的详细描述，注意转场方式；
7-10 秒：最终场景和结尾。
```

## 各维度详细说明

### 1. 画面风格关键词

| 风格类型 | 关键词 |
|---------|--------|
| 写实风格 | 写实、真实感、电影级画质、4K、超高清 |
| 动漫风格 | 二次元、日系动漫、赛璐珞风格、新海诚风格 |
| 电影感 | 电影质感、胶片感、宽银幕、电影调色 |
| 商业广告 | 广告质感、商业摄影、产品特写、高级感 |
| 艺术风格 | 油画风格、水彩、赛博朋克、复古怀旧 |

### 2. 镜头语言

**运镜方式：**
- 推镜头 / Dolly in：缓慢靠近主体
- 拉镜头 / Dolly out：逐渐远离
- 摇镜头 / Pan：水平移动视角
- 移镜头 / Track：跟随主体移动
- 环绕 / Orbit：围绕主体旋转
- 俯拍 / Top-down：从上往下拍摄
- 仰拍 / Low angle：从下往上拍
- 第一人称 POV：第一人称主观视角

**景别：**
- 特写 / Close-up：聚焦细节
- 中景 / Medium shot：半身或膝盖以上
- 全景 / Wide shot：全身+环境
- 远景 / Long shot：广阔场景

### 3. 运动与动态

- 明确描述物体的运动方向和速度
- 使用动词增强动感："飞舞"、"流淌"、"旋转"、"飘落"
- 描述相机与主体的相对运动
- 注意物理合理性：重力、惯性、碰撞反应

### 4. 光影与色彩

- 光源方向：侧光、逆光、顶光、自然光、柔光
- 时间氛围：黄金时刻、蓝色时刻、夜晚霓虹
- 色调：暖色调、冷色调、高对比度、低饱和度
- 特殊效果：镜头光晕、焦外光斑、体积光

### 5. 音效描述 (generate_audio=true 时)

在提示词中可以描述：
- 背景音乐风格：轻快鼓点、悠扬钢琴、电子节拍
- 环境音效：风声、雨声、脚步声、碰撞声
- 语音/旁白内容：直接用引号标注，如 `背景音：「鲜切现摇」`
- 音色要求：如"女生音色"、"低沉男声"

## 参考素材的使用说明

当提供参考图片/视频/音频时，应在提示词中明确说明：

- **参考图片**: "首帧为图片1"、"尾帧定格为图片2"
- **参考视频**: "全程使用视频1的第一视角构图"、"模仿视频1的运镜方式"
- **参考音频**: "全程使用音频1作为背景音乐"

## 最佳实践

1. **具体优于抽象**: "一只橘猫在阳光下伸懒腰" 比 "可爱的动物" 好 100 倍
2. **长度适中**: 100-500 字通常效果最佳，过短信息不足，过长可能互相矛盾
3. **避免负面描述**: 不要说"不要xxx"，而是正面描述想要什么
4. **物理一致性**: 确保描述的动作符合物理规律
5. **时间合理**: 在 duration 限制内合理安排场景数量（5秒建议1-2个场景，10秒建议2-4个场景）
6. **测试迭代**: 先用 fast 版本快速验证提示词效果，满意后再用标准版生成高质量版本

## 示例提示词

### 果茶宣传广告示例
```
全程使用视频1的第一视角构图，全程使用音频1作为背景音乐。第一人称视角果茶宣传广告，「苹苹安安」苹果果茶限定款；首帧为图片1，你的手摘下一颗带晨露的阿克苏红苹果，轻脆的苹果碰撞声；2-4 秒：快速切镜，你的手将苹果块投入雪克杯，加入冰块与茶底，用力摇晃，冰块碰撞声与摇晃声卡点轻快鼓点，背景音：「鲜切现摇」；4-6 秒：第一人称成品特写，分层果茶倒入透明杯，你的手轻挤奶盖在顶部铺展，镜头拉近看奶盖与果茶的分层纹理；6-8 秒：第一人称手持举杯，将果茶举到镜头前（模拟递到观众面前的视角），背景音「来一口鲜爽」，尾帧定格为图片2。背景声音统一为女生音色。
```

### 风景航拍示例
```
超高清航拍风光短片。清晨的喀纳斯湖面，晨雾缭绕在湖面上方，阳光从云层缝隙中洒下形成耶稣光。0-3 秒：无人机高空俯瞰，湖面如镜倒映着周围雪山，缓缓向前推进；3-6 秒：镜头下降掠过湖面，可以看到清澈见底的湖水和游弋的鱼群；6-9 秒：环绕拍摄岸边的白桦林，金黄的树叶随风飘落；9-10 秒：拉远至全景，展现整个山谷的壮丽景色。自然环境音效，宁静悠远。
```
