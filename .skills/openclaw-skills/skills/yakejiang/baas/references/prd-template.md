# 需求文档（只考虑页面功能逻辑和页面之前的跳转关系即可）

## 1. 页面列表

- **index.html** - 登录页面（应用入口文件）
- **workbench.html** - 主工作台页面
- **profile.html** - 个人中心页面

---

## 2. 页面跳转逻辑

- **启动应用** → 检测登录状态 → 已登录则跳转至 `workbench.html`，未登录则进入 `index.html`
- **index.html** → 登录成功 → `workbench.html`
- **workbench.html** → 点击底部导航【我的】→ `profile.html`
- **profile.html** → 点击【工作台】→ 返回 `workbench.html`（保持上次筛选状态和滚动位置）
- **profile.html** → 退出登录 → `index.html`

---

## 3. 页面功能设计

### 3.1 index.html（登录页面）

- **核心登录表单**：用户名输入框、密码输入框、登录按钮
- **注册功能**：点击【注册账号】链接 → 弹出注册弹框（包含用户名、邮箱、密码字段）→ 提交注册 → 关闭弹框并显示成功提示
- **登录流程**：输入用户名+密码 → 点击登录 → 成功则跳转 `workbench.html`，失败则页面内显示错误提示

---

### 3.2 workbench.html（主工作台页面）

- **顶部任务概览卡片**：
  - 今日待办任务数量（点击过滤今日任务）
  - 本周已完成任务统计（点击过滤已完成任务）
  - 高优先级任务Top5提醒（点击过滤高优先级任务）

- **中部任务管理区（主从视图）**：
  - **任务列表**：显示任务卡片（标题、优先级、截止时间、完成状态复选框），支持搜索、排序筛选、上滑加载更多
  - **任务详情区**：点击任务卡片后展示完整信息（标题、描述、截止日期、优先级、标签、创建时间、修改历史）
  - **操作按钮**：编辑（弹出编辑弹框）、归档（确认弹框）、删除（二次确认弹框）、标记完成/未完成（状态实时更新）

- **底部操作栏**：
  - 【工作台】：刷新当前页面或回到顶部
  - 【添加】：弹出新建任务弹框
  - 【我的】：跳转至 `profile.html`

---

### 3.3 profile.html（个人中心页面）

- **用户信息展示区**：显示头像、昵称、邮箱，【修改资料】按钮弹出编辑资料弹框（可修改头像、昵称、邮箱，保存后实时更新）
- **设置管理区**：账号与安全设置项
- **退出登录**：点击【退出登录】按钮 → 弹出确认弹框 → 确认后清除登录状态 → 跳转至 `index.html`
