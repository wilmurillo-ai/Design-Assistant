# Message patterns

Use these patterns to keep outgoing WhatsApp messages natural.

## Core rule

Lead with the actual reason for the message. Do not spend the first sentence warming up unless the relationship calls for it.

## Common patterns

### 1. Quick ask

Use for simple requests.

Pattern:
- short opener
- request
- easy response path

Example:
`Hey, can you send me the final PDF when you get a minute?`

### 2. Availability check

Pattern:
- direct availability question
- optional time range
- optional reason

Example:
`Hi, are you free sometime after 6 tomorrow? Need 10 minutes for a quick discussion.`

### 3. Follow-up nudge

Pattern:
- reference prior context
- nudge without guilt
- specific next step

Example:
`Hey, just following up on the documents I sent yesterday. Let me know if anything else is needed from my side.`

### 4. Professional share

Pattern:
- state what is being shared
- say what feedback or action is needed
- keep it compact

Example:
`Hi Riya, sharing the revised proposal here. Please review the pricing section when you get a chance.`

### 5. Reschedule

Pattern:
- acknowledge change
- propose new time
- keep apology brief

Example:
`Sorry, I need to push this by about 30 minutes. Would 4:30 work instead?`

### 6. Payment reminder

Pattern:
- polite reference
- amount or invoice context if needed
- clear request

Example:
`Hi, following up on invoice 1042 shared last week. Please let me know if there’s any issue, otherwise I’d appreciate the payment this week.`

## Style tuning

### Friend
- can be shorter
- can drop punctuation if it still reads clearly
- light humor is okay

### Coworker
- prefer full sentences
- keep the ask obvious
- avoid emojis unless already normal in the thread

### Client or recruiter
- professional and calm
- no slang unless they use it first
- respect time, keep it tight

## Final polish checklist

Before sending, check:
- Can the recipient understand the ask in one read?
- Does it sound like a person, not a template?
- Is the length justified?
- Is there a clear next step?
