## ✅ 本次优化内容

### 1. 简化封面设计 ✅

**优化前**: 
- 复杂装饰线
- 评分仪表图
- 多行标语
- 底部声明

**优化后**:
- 简洁标题
- 基本信息（名称/代码/评分/时间）
- 清晰分隔线
- 清爽布局

**效果**:
```
          A 股专业分析报告
        ====================
        
        股票名称：中国船舶
        股票代码：600150
        综合评分：7.0/10 (推荐)
        分析时间：2026-03-01 11:15:16
```

### 2. 优化段落布局 ✅

**章节结构**:
```
一、核心摘要
二、实时行情
三、技术分析
    1. 均线系统
    2. MACD 指标
    3. 其他指标
四、投资建议
五、历史分析回顾（如有）
六、风险提示
```

**段落间距优化**:
- 章节标题后：0.3cm
- 子标题后：0.2cm
- 表格之间：0.4cm
- 列表项之间：0.2cm

### 3. 统一表格样式 ✅

**表格规范**:
- 字体：宋体 10pt
- 行高：6pt 上下内边距
- 标签列：浅灰背景
- 边框：0.5pt 灰色
- 外框：1pt 灰色

**示例**:
```
┌─────────────┬─────────────┐
│   技术信号  │  bullish    │
│   趋势方向  │  上涨趋势   │
│   均线排列  │  多头排列   │
│   RSI 指标  │   75.64     │
└─────────────┴─────────────┘
```

### 4. 优化列表格式 ✅

**买入/卖出条件**:
- 使用项目符号 (•)
- 左缩进 20pt
- 行间距 4pt
- 清晰易读

**示例**:
```
买入条件参考:
  • 价格站稳支撑位上方
  • MACD 形成金叉信号
  • 成交量明显放大
  • 突破关键阻力位
```

### 5. 优化字体和字号 ✅

**字体规范**:
| 元素 | 字体 | 字号 |
|------|------|------|
| 主标题 | 黑体 | 22pt |
| 章节标题 | 黑体 | 15pt |
| 子标题 | 黑体 | 12pt |
| 正文 | 宋体 | 10.5pt |
| 表格 | 宋体 | 10pt |
| 列表 | 宋体 | 10.5pt |

**行距规范**:
- 正文：16pt
- 列表：16pt
- 表格：单倍行距

```

左边距：2.5cm
右边距：2.5cm
上边距：2cm
下边距：2cm
```

### 章节间距
```
章节标题前：15pt
章节标题后：12pt
子标题前：10pt
子标题后：8pt
表格前：5pt
表格后：5pt
```

### 表格规范
```
列宽：根据内容自适应
行高：最小 20pt
内边距：上下 6pt，左右 5pt
边框：0.5pt (内部), 1pt (外部)
背景：标签列浅灰 (#f8fafc)
```
