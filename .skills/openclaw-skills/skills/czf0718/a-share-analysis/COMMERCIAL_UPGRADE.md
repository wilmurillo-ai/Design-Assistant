## 🚀 使用方法

### 命令行

```bash
# 商业版报告
python analyze_stock_pro.py 600482 中国动力 --commercial

# 专业版报告（默认）
python analyze_stock_pro.py 600482 中国动力

# 添加新参数
--commercial, -c    生成商业版报告
```

### 程序调用

```python
from generate_report_commercial import AShareCommercialReport

generator = AShareCommercialReport()
report = generator.generate_report(data)
filepath = generator.save_report(report, '600482', '中国动力')
```
