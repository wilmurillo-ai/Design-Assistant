## 📐 技术细节

### 自定义组件

#### 1. ColorHeader（彩色标题）
```python
class ColorHeader(Flowable):
    """绘制彩色背景章节标题"""
    - 背景色块
    - 白色文字
    - 固定高度
```

#### 2. ScoreGauge（评分仪表）
```python
class ScoreGauge(Flowable):
    """绘制评分进度条"""
    - 圆角矩形背景
    - 彩色进度条
    - 分数显示
    - 等级标签
```
