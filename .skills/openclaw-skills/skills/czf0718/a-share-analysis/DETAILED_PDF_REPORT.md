## 📊 详细 PDF 报告结构

### 11 个专业章节

```
1. 专业封面
   ├── 中英文标题
   ├── 股票信息
   ├── 编制机构
   ├── 报告编号
   └── 保密等级

2. 重要声明（6 项）
   ├── 报告性质
   ├── 投资建议
   ├── 数据准确性
   ├── 风险提示
   ├── 使用限制
   └── 责任免除

3. 投资评级
   ├── 综合评分
   ├── 评级标签（中英文）
   ├── 目标价格
   ├── 止损价格
   └── 评级说明表

4. 核心摘要
   ├── 基本信息
   ├── 技术要点
   └── 关键位置

5. 实时行情
   ├── 6 项行情数据
   ├── 价格统计
   └── 盘面解读

6. 技术分析（超详细）⭐
   ├── 均线系统（4 条均线表格）
   ├── MACD 指标（DIF/DEA/MACD 柱）
   ├── RSI 指标（状态 + 解读）
   ├── 成交量分析（量比解读）
   ├── 支撑与阻力（2 支撑 +2 阻力）
   └── 综合技术评级（评分 + 评级）

7. 新闻情绪分析
   ├── 情绪概览
   ├── 情绪分布
   └── 情绪解读

8. 历史分析回顾
   ├── 分析统计
   ├── 历史观点
   └── 价格历史

9. 投资建议（详细）
   ├── 综合建议
   ├── 仓位建议
   ├── 价格目标表
   ├── 操作策略
   └── 关注要点（5 项）

10. 情景分析
    ├── 乐观情景（30%）
    ├── 基准情景（50%）
    ├── 悲观情景（20%）
    └── 风险收益比

11. 风险提示（详细）
    ├── 市场风险（3 项）
    ├── 技术风险（3 项）
    ├── 风险应对措施（5 项）
    └── 风险评级

12. 报告附录
    ├── 数据来源
    ├── 分析方法
    ├── 指标说明
    └── 报告信息
```

## 🎨 PDF 设计特点

### 专业配色方案

| 元素 | 颜色 | 用途 |
|------|------|------|
| 主色调 | #1e3a8a（深蓝） | 标题、表头 |
| 辅助色 | #0891b2（青色） | 子标题 |
| 背景色 | #f8fafc（浅灰） | 表格背景 |
| 边框色 | #e2e8f0 | 表格边框 |
| 文字色 | #1e293b（深灰） | 正文 |

### 表格样式

- **表头**: 深蓝色背景 + 白色文字
- **数据行**: 浅灰色背景交替
- **边框**: 0.5pt 细线
- **内边距**: 5-8pt

### 字体规范

| 元素 | 字体 | 字号 |
|------|------|------|
| 主标题 | 黑体 | 20pt |
| 章节标题 | 黑体 | 14pt |
| 子标题 | 黑体 | 11pt |
| 正文 | 宋体 | 10pt |
| 表格 | 宋体 | 9-10pt |

## 🚀 使用方法

### 自动生成

每次运行分析时，会自动生成详细 PDF 报告：

```bash
# 分析股票（自动生成详细 PDF）
python analyze_stock_pro.py 600619 海立股份

# 静默模式
python analyze_stock_pro.py 600619 海立股份 --quiet
```

### 输出位置

PDF 报告保存在：
```
C:\Users\fj\.openclaw\workspace\a-share-reports\
├── 600619_海立股份_20260301_122745_DETAILED.pdf
├── 600619_海立股份_20260301_122745_PRO.md
└── ...
```

## 🎯 使用场景

### 场景 1: 个人深度研究
```bash
python analyze_stock_pro.py 600519 贵州茅台
```
→ 获取 6-8 页详细 PDF 报告

### 场景 2: 投顾服务
```bash
python analyze_stock_pro.py 600482 中国动力
```
→ 生成专业 PDF 发送给客户

### 场景 3: 批量分析
```bash
python analyze_stock_pro.py --batch
```
→ 批量生成多只股票详细 PDF
