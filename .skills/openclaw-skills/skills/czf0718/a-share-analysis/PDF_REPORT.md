- RSI、量比
- 支撑位、阻力位

- 新闻总数

- 情绪评分

### 6. 历史分析回顾
- 分析次数
- 历史价格记录
- 主要建议

### 7. 投资建议
- 综合建议（强烈推荐/推荐/观望/谨慎/回避）
- 操作策略

- 6 项风险说明

## 🚀 使用方法

### 自动 PDF 生成

现在每次运行分析时，会自动生成 Markdown 和 PDF 两种格式：

```bash
python analyze_stock_pro.py 603258 电魂网络
```

**输出示例**:
```
[5/7] 生成专业报告...
[OK] Markdown 报告：...\603258_电魂网络_20260301_105156_PRO.md
[*] 正在生成 PDF 报告...
[OK] PDF 报告：...\603258_电魂网络_20260301_105156.pdf
```

### 单独生成 PDF

```python
from generate_pdf_report import ASharePDFReport

# 准备分析数据
data = {
    "stock_code": "603258",
    "stock_name": "电魂网络",
    "price": 19.11,
    "change_percent": 1.22,
    "technical": {...},
    "news_sentiment": {...},
    "memory_history": {...}
}

# 生成 PDF
generator = ASharePDFReport()
pdf_path = generator.generate_pdf(data)
print(f"PDF 已生成：{pdf_path}")
```

## ⚠️ 注意事项

### 字体依赖
PDF 生成使用 Windows 系统字体：
- `C:\Windows\Fonts\simsun.ttc` (宋体)
- `C:\Windows\Fonts\simhei.ttf` (黑体)

如果在其他系统运行，需要：
1. 安装中文字体
2. 或修改字体路径配置

### 依赖库
```bash
pip install reportlab
```

已预装库：
- reportlab 4.4.10
- pillow 12.1.1

### PDF 大小
- 典型大小：50-150 KB
- 页数：2-3 页
