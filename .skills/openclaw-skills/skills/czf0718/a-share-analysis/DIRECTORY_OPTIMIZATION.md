## 🔧 使用说明

### 自动生成
每次运行分析时，会自动创建股票代码目录并保存报告：

```bash
# 分析股票（自动创建 600519 目录）
python analyze_stock_pro.py 600519 贵州茅台

# 查看报告
ls a-share-reports/600519/
```

### 整理旧报告
```bash
# 运行整理脚本
cd scripts
python organize_reports.py
```

### 查看报告
```bash
# 查看所有股票目录
ls a-share-reports/

# 查看某支股票的所有报告
ls a-share-reports/600519/

# 统计报告数量
ls a-share-reports/*/*.md | wc -l
```
