## 🚀 使用方法

### 命令行

```bash
# 详细版报告（新增）
python analyze_stock_pro.py 600619 海立股份 --detailed

# 简写
python analyze_stock_pro.py 600619 海立股份 -d

# 专业版报告（默认）
python analyze_stock_pro.py 600619 海立股份

# 商业版报告
python analyze_stock_pro.py 600619 海立股份 --commercial
```

### 参数说明

| 参数 | 简写 | 说明 |
|------|------|------|
| `--detailed` | `-d` | 生成详细版报告 ⭐新增 |
| `--commercial` | `-c` | 生成商业版报告 |
| `--quiet` | `-q` | 静默模式 |

## 📊 详细版报告结构

### 12 个详细章节

```
1. 专业封面
   ├── 中英文标题
   ├── 股票信息
   ├── 编制机构
   └── 报告编号

2. 重要声明（6 项）
   ├── 报告性质
   ├── 投资建议
   ├── 数据准确性
   ├── 风险提示
   ├── 使用限制
   └── 责任免除

3. 投资评级
   ├── 综合评分
   ├── 评级标签（中英文）
   ├── 目标价格
   ├── 止损价格
   └── 评级说明

4. 核心摘要（详细）
   ├── 基本信息
   ├── 技术面要点
   ├── 核心观点
   ├── 操作建议
   └── 关键位置

5. 行情数据（详细）
   ├── 实时行情
   ├── 价格统计
   └── 盘面解读

6. 技术分析（超详细）⭐
   ├── 均线系统分析
   ├── MACD 指标分析
   ├── RSI 指标分析
   ├── 成交量分析
   ├── 支撑阻力分析
   └── 综合技术评级

7. 新闻情绪
   ├── 情绪概览
   ├── 情绪分布
   └── 情绪解读

8. 历史回顾
   ├── 分析统计
   ├── 历史观点
   └── 价格历史

9. 投资建议（详细）
   ├── 综合建议
   ├── 仓位建议
   ├── 价格目标
   ├── 操作策略
   └── 关注要点

10. 情景分析
    ├── 乐观情景
    ├── 基准情景
    ├── 悲观情景
    └── 风险收益比

11. 风险提示（详细）
    ├── 市场风险
    ├── 个股风险
    ├── 技术风险
    ├── 操作风险
    ├── 风险应对
    └── 风险评级

12. 报告附录
    ├── 数据来源
    ├── 分析方法
    ├── 指标说明
    ├── 报告生成信息
    └── 报告编号
```

## 🎯 使用场景

### 场景 1: 个人深度研究
```bash
python analyze_stock_pro.py 600619 海立股份 --detailed
```
→ 获取 5-6 页详细分析报告

### 场景 2: 投顾参考
```bash
python analyze_stock_pro.py 600519 贵州茅台 --detailed
```
→ 获取详细技术分析供客户参考

### 场景 3: 批量深度分析
```bash
python analyze_stock_pro.py --batch --detailed
```
→ 批量生成多只股票详细报告
