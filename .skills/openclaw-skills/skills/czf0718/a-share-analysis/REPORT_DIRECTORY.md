## ✨ 优化特点

### 1. 按股票代码分类
- 每支股票独立目录
- 目录名：股票代码（如 600519）
- 自动创建，无需手动管理

### 2. 所有版本集中存储
每个股票目录包含：
- ✅ 专业版报告 (.md)
- ✅ 详细版报告 (.md)
- ✅ 商业版报告 (.md)
- ✅ PDF 详细版报告 (.pdf)

### 3. 文件命名规范
```
{股票代码}_{股票名称}_{时间戳}_{版本}.md/pdf
```

**示例**:
```
600519_贵州茅台_20260301_120000_PRO.md
600519_贵州茅台_20260301_120000_DETAILED.md
600519_贵州茅台_20260301_120000_COMMERCIAL.md
600519_贵州茅台_20260301_120000_DETAILED.pdf
```

## 📋 使用说明

### 自动生成
每次运行分析时，会自动创建股票代码目录并保存报告：

```bash
# 分析股票（自动创建 600519 目录）
python analyze_stock_pro.py 600519 贵州茅台

# 查看报告
ls a-share-reports/600519/
```

### 查看报告
```bash
# 查看所有股票目录
ls a-share-reports/

# 查看某支股票的所有报告
ls a-share-reports/600519/

# 查看某支股票的 PDF 报告
ls a-share-reports/600519/*.pdf

# 查看某支股票的专业版报告
ls a-share-reports/600519/*_PRO.md
```

### 清理报告
```bash
# 删除某支股票的所有报告
rm -rf a-share-reports/600519/

# 删除所有报告
rm -rf a-share-reports/*
```
