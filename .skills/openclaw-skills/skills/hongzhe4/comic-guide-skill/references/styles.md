# 漫画风格 — AI 图像生成提示词模板

每种风格定义了：画风描述词、角色外观、配色方案、面板布局指导、对话气泡样式。
这些描述直接用于 AI 图像生成提示词（prompt），非 CSS。

## 11. custom — 自定义风格

用户通过自然语言描述风格，Agent 据此生成对应的画风提示词和角色设定。

用户描述示例：
- "赛博朋克风格，霓虹灯光，暗色背景"
- "宫崎骏水彩风，柔和渐变"
- "漫威美漫风格，英雄姿态"

Agent 处理：
1. 解析用户描述中的视觉关键词
2. 生成对应的画风提示词
3. 创建通用角色（导师/学习者/助手）设定
4. 确定配色方案和面板布局

---

## 通用画风修饰词

无论何种风格，图像生成提示词末尾均应附加：

```
High quality illustration, professional comic panel layout, speech bubbles
with text in Chinese characters, consistent character designs across panels,
clean readable text, educational content, no watermarks
```

## 面板布局描述词

| 布局 | 提示词描述 |
|------|----------|
| 2x2 四格 | `4 panels in a 2x2 grid layout, landscape 16:9` |
| 2x3 六格 | `6 panels in a 2x3 grid layout, landscape 16:9` |
| 竖版三格 | `3 panels stacked vertically, portrait 3:4 ratio` |
| 竖版四格 | `4 panels stacked vertically, portrait 3:4 ratio, webtoon style` |
| 标题大图+小图 | `1 large title panel on top, 3 smaller panels below in a row` |
