## 按情绪选音乐

| 情绪 | 音乐风格 | 场景 |
|------|----------|------|
| 治愈温暖 | 钢琴和弦乐 | 猫咪睡觉、互动、结尾 |
| 欢快活泼 | 轻快流行、尤克里里 | 玩耍、追蝴蝶、跑酷 |
| 宁静平和 | 自然声、轻吉他 | 窗台看雨、午后晒太阳 |

| 灰暗讽刺 | 低沉 Rap、节拍 | 职场吐槽、周一主题 |
| 搞笑反转 | 突然变调、滑稽音效 | 意外、跌倒、惊讶 |

## 音乐生成提示词（6要素）

```
[情感描述]的[音乐风格]，约[BPM] BPM，
主要乐器：[主奏+伴奏+节奏]，
人声：[描述]，
制作风格：[混音特点]
```

### 示例

**治愈风：**
```
温暖治愈的流行歌曲，80 BPM，
主要乐器：钢琴和弦乐，伴奏：吉他，
人声：女声温柔甜美，
制作风格：现代混音，温暖质感
```

**Rap风：**
```
灰暗讽刺的嘻哈说唱，90 BPM，
主要乐器：鼓点+贝斯，伴奏：钢琴采样，
人声：男声低沉有磁性，
制作风格：现代嘻哈，叙事感强
```
