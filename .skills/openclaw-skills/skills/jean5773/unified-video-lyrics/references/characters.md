# 角色模板库

## 猫咪

```yaml
灰白虎斑猫:
  外观: 灰白相间，大眼睛，长毛胖乎乎
  性格: 好奇、慵懒、偶尔脱线
  动作: 伸懒腰、摇尾巴、蹭爪子
  表情: 耳朵高低、尾巴摆动、眼睛睁闭

橘猫:
  外观: 橘色虎斑，大饼脸，胖
  性格: 贪吃、慵懒、黏人
  动作: 吃饭、睡觉、伸懒腰
  表情: 满足、慵懒、喵喵叫

奶牛猫:
  外观: 黑底白花，警长脸
  性格: 活泼、调皮、神经质
  动作: 跑酷、玩闹、突然静止
  表情: 好奇、惊讶、困惑
```
