# 视频提示词指南

## 基础结构

```
[主体] + [场景] + [动作] + [光照] + [风格] + [技术]
```

## 完整示例

```
A cute fluffy gray and white kitten with big expressive eyes,
ears twitching, whiskers moving, sitting on a wooden windowsill,
warm golden morning sunlight streaming through,
soft pastel pink and orange gradient sky,
dreamy magical healing atmosphere,
cinematic Douyin vertical video style,
highly detailed, 8k, professional lighting
```

## 风格关键词

| 风格 | 关键词 |
|------|--------|
| 治愈 | warm, healing, cozy, soft pastel |
| 灰暗 | dark, moody, cinematic, blue gray |
| 活泼 | playful, energetic, bright, dynamic |
| 日漫 | anime, 2D, Japanese animation |

## 光照关键词

- warm golden sunlight
- soft indoor lighting
- moonlight
- dramatic lighting
- bokeh background

## 角色描述（重要！）

### 猫咪
```
cute fluffy [颜色] cat, big expressive eyes, 
ears twitching, whiskers moving, nose twitching,
animated facial expressions, tail swaying
```

### 狗狗
```
cute [品种] puppy, big puppy eyes,
tongue out, tail wagging, head tilting,
animated expressions
```

1. 固定主角外观描述
2. 统一光照色调
3. 统一艺术风格
4. 保持场景连贯性
