# 歌词创作技巧

## 结构标签

| 标签 | 含义 | 位置 |
|------|------|------|
| [Intro] | 前奏 | 开头 |
| [Verse] | 主歌 | 叙事 |
| [Pre-chorus] | 预副歌 | 情绪上升 |
| [Chorus] | 副歌 | 高潮 |
| [Hook] | 钩子 | 记忆点 |
| [Bridge] | 桥段 | 转折 |
| [Outro] | 尾奏 | 结束 |

## 押韵技巧

| 韵脚 | 词例 |
|------|------|
| ang | 阳光、方向、翅膀、远方 |
| eng | 星星、精灵、心情、倾听 |
| an | 陪伴、温暖、安然、慢慢 |
| i | 调皮、回忆、甜蜜、好奇 |

## 情绪对应

| 画面情绪 | 歌词基调 | 用词 |
|----------|----------|------|
| 治愈温暖 | 柔软缓慢 | 阳光、拥抱、温暖 |
| 欢快活泼 | 轻快跳跃 | 奔跑、笑声、翅膀 |
| 忧伤孤独 | 缓慢低沉 | 雨滴、夜空、思念 |
| 灰暗讽刺 | 叙事自嘲 | 无奈、吐槽、现实 |

## 金句模板

- 治愈："有你在身边，就是最好的时光"
- 扎心："陪伴是最长情的告白"
- 职场："周一Plus，键盘敲到报废"
- 俏皮："管他呢，先玩再说"

## Rap 歌词特点

```
- 节奏感强，句子短
- 常用押韵（ang/eng/an）
- 自嘲、讽刺、叙事
- 重复副歌加强记忆
```

### Rap 示例

```
[Verse]
六点的地铁 人挤人
我只是一只猫 为什么要上班
工牌挂在胸前 像个笑话
喵生不易 周一Plus

[Chorus]
周一 周一 灰暗的天
周一 周一 喵的不想睁眼
周一 周一 还要继续
喵~ 这就是生活
```
