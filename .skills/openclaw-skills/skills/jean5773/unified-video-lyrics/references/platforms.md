# 多平台脚本指南

## 抖音 (Douyin)

### 特点
- 15-60秒为主
- 快节奏、反转多
- 音乐配合关键
- 黄金3秒法则

### 结构
```
[开头 - 3秒] 悬念/冲突/疑问
[中间 - 情绪递进]
[结尾 - 反转/共鸣/引导]
```

### 关键词
```
#抖音 #douyin #短视频 #剧情反转
```

## B站 (Bilibili)

### 特点
- 1-10分钟深度内容
- 弹幕互动
- 年轻化表达
- 知识性/娱乐性

### 结构
```
[开场 - 引入话题]
[起承转合 - 故事展开]
[高潮 - 核心亮点]
[结尾 - 互动引导]
```

### 关键词
```
#bilibili #B站 #UP主 #弹幕
```
