## 核心工作流

```
1. 上传/提供首个参考视频
2. 分析角色外观、画风、色调
3. 创作剧本+歌词
4. 生成6个统一风格镜头
5. 合并成完整视频
6. 生成背景音乐
```

## 首创功能：统一角色画风

### 工作流程

```
[首个参考视频] → [分析角色+画风] → [生成后续6个视频]
                      ↓
              保持一致：
              - 主角外观
              - 画风（2D日漫/3D/写实）
              - 色调（明亮/灰暗/暖色/冷色）
              - 光照风格
```

### 首个视频分析要点

收到参考视频后，使用 videos_understand 分析：

1. **主角外观** - 品种/眼睛/毛发/服装
2. **画风** - 2D动漫/3D/写实
3. **色调** - 明亮/灰暗/饱和度
4. **光照** - 室内/自然光/柔光

## 二次元萌系风格（赛璐璐）

### 风格特点

| 特点 | 描述 |
|------|------|
| 画风 | 日漫萌系 (Kawaii Anime) |
| 渲染 | 赛璐璐 (Cell Shading) - 清晰阴影边缘 |
| 线条 | 清晰均匀黑色勾线 |
| 色调 | 低饱和度灰白+蓝色 |
| 氛围 | 治愈、无辜、略带忧伤 |

### 角色设计

- 巨大圆眼睛
- 圆润脸庞
- 简化身体比例
- 拟人化（直立行走、穿戴服饰）

### 表情特点

- 巨大黑色瞳孔
- 略微下垂眉毛
- 倒V形嘴角
- 无辜委屈眼神

### 视频提示词模板

```
[角色描述], anime style, 2D Japanese animation,
cell shading, clean lineart,
[色调描述], [光照描述], cinematic
```

## 视频合并

```bash
# 合并视频
ffmpeg -i input1.mp4 -i input2.mp4 -filter_complex "[0:v][1:v]concat=n=2:v=1:a=0" output.mp4

# 添加音频
ffmpeg -i video.mp4 -i audio.mp3 -c:v copy -c:a aac output.mp4
```
