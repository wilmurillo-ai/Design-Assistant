## 📖 使用说明

本风格库按 **5 大风格大类** 分类，每个风格包含：
- 风格名称 + 英文标识
- 核心特征（3-5 关键词）
- 代表参数（色调/光影/质感）
- 适用场景
- 负面提示词建议

## 🎬 一、电影感风格 (Cinematic)

### 1.1 写实电影风 (Cinematic Realism)
**英文标识：** `cinematic realism, photorealistic`

**核心特征：**
- 胶片质感 / 自然光影 / 真实纹理 / 电影调色

**代表参数：**
- 色调：Kodak 2383 胶片模拟，轻微暖调
- 光影：自然光源优先，柔和过渡
- 质感：皮肤纹理/布料褶皱/环境细节清晰

**适用场景：** 人像写真、剧情短片、纪录片风格

**负面提示词：** `cartoon, anime, cgi, 3d render, plastic skin`

---

### 1.2 王家卫风格 (Wong Kar-wai Style)
**英文标识：** `wong kar wai style, stephen wong cinematography`

**核心特征：**
- 高饱和暖色 / 运动模糊 / 霓虹光影 / 情绪化构图

**代表参数：**
- 色调：橙青色调 (teal & orange)，高饱和
- 光影：霓虹灯/路灯/烛光，强对比
- 质感：轻微颗粒感，motion blur

**适用场景：** 情绪人像、都市夜景、浪漫剧情

**负面提示词：** `flat lighting, neutral colors, sharp focus, documentary`

---

### 1.3 Wes Anderson 风格 (Wes Anderson Aesthetic)
**英文标识：** `wes anderson style, symmetrical composition`

**核心特征：**
- 对称构图 / 粉彩配色 / 平面化 / 复古美学

**代表参数：**
- 色调：粉彩 pastel colors (粉/蓝/黄/绿)
- 光影：均匀柔光，低对比
- 质感：平面化，minimal shadows

**适用场景：** 创意广告、品牌视觉、艺术短片

**负面提示词：** `asymmetrical, dark shadows, high contrast, gritty`

---

### 1.4 诺兰暗黑风 (Nolan Dark Cinematic)
**英文标识：** `nolan style, dark cinematic, imax quality`

**核心特征：**
- 低照度 / 高对比 / 冷色调 / IMAX 质感

**代表参数：**
- 色调：冷蓝/青灰色，desaturated
- 光影：硬光，deep shadows，practical lights
- 质感：IMAX 胶片颗粒，ultra sharp

**适用场景：** 科幻、悬疑、动作大片

**负面提示词：** `bright, colorful, soft lighting, cheerful`

---

### 1.5 赛博朋克电影风 (Cyberpunk Cinematic)
**英文标识：** `cyberpunk, blade runner style, neon noir`

**核心特征：**
- 霓虹光影 / 高饱和对比 / 雨夜氛围 / 未来都市

**代表参数：**
- 色调：紫/蓝/粉霓虹，dark background
- 光影：neon signs, volumetric fog, wet surfaces
- 质感：反光表面，rain drops

**适用场景：** 科幻场景、概念设计、MV

**负面提示词：** `daylight, natural lighting, pastoral, clean`

## 💡 使用技巧

### 1. 风格叠加
可以叠加 2-3 个风格创造独特效果：
```
cinematic realism + wong kar wai style = 情绪化写实电影风
minimalist commercial + bauhaus = 理性商业美学
```

### 2. 风格强度控制
通过权重调整风格强度：
```
(wes anderson style:1.3) - 强化风格特征
(bauhaus:0.8) - 轻微影响
```

### 3. 避免风格冲突
不要同时使用矛盾风格：
```
❌ realistic + cartoon
❌ dark cinematic + pastel colors
✅ realistic + cinematic lighting
```
