## 🎬 一、基础运镜（10 种）

### 1. 推镜头 (Push In / Dolly In)

**功能描述：** 镜头逐渐向主体推进

**适用场景：**
- 强调细节
- 内心戏
- 情绪递进
- 聚焦注意力

**提示词示例：**
```
镜头缓慢向前推进，细雪匀速飘落
镜头推进，他的眼神开始坚定
快速推近到角色脸部特写
```

**实战案例（年兽）：**
```
镜号 1.1 | 时长 3s | 运镜：慢推 | 景别：远景
镜头缓慢向前推进，细雪匀速飘落，无其他动态元素，情绪压抑
```

---

### 2. 拉镜头 (Pull Out / Dolly Out)

**功能描述：** 镜头逐渐远离主体

**适用场景：**
- 揭示环境
- 孤独感
- 场景收尾
- 从细节到宏大

**提示词示例：**
```
镜头快速拉远，展现周围的环境
镜头拉远到腰部，他化作一道流光
缓慢后拉，展现宏大战场
```

**实战案例（剑来）：**
```
【镜头 2】00:02 - 00:05
镜头拉远到腰部，他化作一道流光，以极其凌厉的姿态向前突进。
背景一轮明月高悬。
```

---

### 3. 摇镜头 (Pan / Tilt)

**功能描述：** 固定支点水平/垂直转动

**适用场景：**
- 扫视场景
- 建立关联
- 跟随运动
- 揭示空间

**提示词示例：**
```
镜头快速向右摇，焦点落在...
镜头垂直上移，展现行军队伍
从左向右摇摄，展示完整阵容
```

**类型细分：**
- **水平摇（Pan）**：左右扫视
- **垂直摇（Tilt）**：上下扫视

---

### 4. 移镜头 (Truck / Pedestal)

**功能描述：** 镜头水平方向平稳移动

**适用场景：**
- 展现空间纵深
- 跟随平行运动
- 场景转换

**提示词示例：**
```
镜头垂直上移，展现行军队伍
镜头水平右移，展示完整战场
缓慢上移，从脚部到头部
```

**类型细分：**
- **水平移（Truck）**：左右移动
- **垂直移（Pedestal）**：上下移动

---

### 5. 跟镜头 (Follow Shot / Tracking)

**功能描述：** 镜头跟随运动主体

**适用场景：**
- 突出动态存在感
- 动作场景
- 追逐戏
- 第一人称跟随

**提示词示例：**
```
低角度手持跟拍镜头
镜头跟随角色移动
侧面跟随拍摄，保持角色在画面中心
```

**实战案例：**
```
跟拍镜头，角色快速奔跑，背景模糊产生速度感
低角度手持跟拍，增强临场感
```

---

### 6. 航拍镜头 (Aerial / Drone Shot)

**功能描述：** 空中全知视角

**适用场景：**
- 宏大景观
- 史诗感
- 场景建立
- 上帝视角

**提示词示例：**
```
航拍镜头，高空向前航拍
无人机航拍，俯瞰整个古村落
垂直上升航拍，展现全景
```

**实战案例（年兽）：**
```
航拍镜头，高空向前航拍，展现寒冬腊月的古村落全貌
屋顶积满厚雪，村口挂着枯萎的红色灯笼
```

---

### 7. 环绕镜头 (Orbit / Arc Shot)

**功能描述：** 360 度环绕主体

**适用场景：**
- 顿悟时刻
- 中心感
- 角色登场
- 360 度展示

**提示词示例：**
```
镜头以士兵为中心进行环绕拍摄
环绕角色 360 度旋转
顺时针环绕，展示角色全貌
```

**实战案例（剑来）：**
```
人物快速转身背对着镜头快速向上飞去，
镜头以人物为中心进行环绕拍摄，
周围散发着耀眼蓝白光芒。
```

---

### 8. 变焦镜头 (Zoom / Rack Focus)

**功能描述：** 改变光学焦距

**适用场景：**
- 戏剧性
- 惊觉
- 注意力转移
- 希区柯克变焦

**提示词示例：**
```
镜头焦点由前景转为后景
快速变焦到角色眼睛
希区柯克变焦效果（dolly zoom）
```

**类型细分：**
- **光学变焦（Zoom）**：焦距变化
- **焦点转移（Rack Focus）**：焦点从前景到后景

---

### 9. 主观镜头 (POV / First Person)

**功能描述：** 第一人称视角

**适用场景：**
- 身临其境体验
- 动作场景
- 恐怖/紧张
- 角色代入

**提示词示例：**
```
第一人称视角，人物快速匍匐前进
主观视角，双手伸向前方
POV 视角，看到对方向自己冲来
```

**实战案例：**
```
第一人称视角，人物快速匍匐前进，
双手撑地，前方是敌人的阵地
```

---

### 10. 固定镜头 (Fixed / Static Shot)

**功能描述：** 机位完全不动

**适用场景：**
- 戏剧张力
- 宿命感
- 对话场景
- 静态展示

**提示词示例：**
```
固定镜头，机位完全不动
镜头固定，情绪沉稳凝重
三脚架固定拍摄，稳定构图
```

**实战案例（年兽）：**
```
镜号 1.2 | 时长 2.5s | 运镜：固定 | 景别：中景
镜头固定，老者站在村口老槐树下，情绪沉稳凝重
```

## 🎬 二、进阶运镜组合（6 种经典模式）

### 1. 跟随 + 环绕

**镜头序列：** 侧面跟随→向左环绕→聚焦正面

**适用场景：** 捕猎/追逐场景

**提示词示例：**
```
镜头从侧面跟随角色奔跑，
然后向左环绕到角色正面，
最后聚焦在角色坚定的眼神上
```

---

### 2. 下移 + 环绕 + 上升 + 后拉

**镜头序列：** 面部特写→手部→环绕拉高→顶视俯拍→后拉

**适用场景：** 人物登场/氛围营造

**提示词示例：**
```
从角色面部特写开始，
镜头下移到手部细节，
然后环绕拉高到头顶，
顶视俯拍后缓慢后拉，
展现角色与环境的关系
```

---

### 3. 上升 + 前推 + 俯视

**镜头序列：** 过肩视角→前推聚焦→快速拉升→上帝视角

**适用场景：** 宏大场面揭示

**提示词示例：**
```
从过肩视角开始，
前推聚焦到关键物体，
然后快速拉升到空中，
俯视展现整个战场布局
```

---

### 4. 跟随 + 上摇 + 环绕 + 下移

**镜头序列：** 跟随→上摇→360 度环绕→拉近定格

**适用场景：** 温馨互动场景

**提示词示例：**
```
镜头跟随角色行走，
上摇展现角色表情，
360 度环绕展示周围环境，
最后下移拉近定格在牵手细节
```

---

### 5. 环绕 + 前推 + 后拉

**镜头序列：** 身后推进→穿过窗户→旋转 180 度→拉远

**适用场景：** 空间感展示

**提示词示例：**
```
从角色身后推进，
穿过窗户进入室内，
旋转 180 度面对角色，
然后拉远展现完整房间
```

---

### 6. 后拉 + 上升 + 环绕

**镜头序列：** 超低广角仰拍→上升移动→向右环绕→面部聚焦

**适用场景：** 英雄登场

**提示词示例：**
```
超低广角仰拍角色脚部，
快速上升移动，
向右环绕展示全身，
最后聚焦在角色面部特写
```

## 🎬 三、景别递进法则

### 正推（聚焦细节）

**序列：** 远景 → 全景 → 中景 → 近景 → 特写

**适用：** 从环境到人物，逐步聚焦情绪

**实战案例（年兽情绪递进）：**
```
镜 1.1：远景（古村落全貌）→ 压抑基调
镜 1.2：中景（老者站在村口）→ 引入角色
镜 1.3：近景（老者表情）→ 情绪传递
镜 1.4：特写（红色粗布）→ 关键道具
```

---

### 反推（揭示环境）

**序列：** 特写 → 近景 → 中景 → 全景 → 远景

**适用：** 从细节到宏大，揭示空间关系

**实战案例（剑来能量释放）：**
```
镜头 1：特写（男主握剑柄）→ 细节聚焦
镜头 2：近景（剑柄发光）→ 能量聚集
镜头 3：中景（男主突进）→ 动作展开
镜头 4：全景（化作流光）→ 宏大揭示
```

## 🎬 五、实战案例拆解

### 案例 1：年兽的由来（15 个镜头）

**视频结构：** 45 秒，15 个镜头，每镜 2-3 秒

**镜头运动统计：**
- 慢推：4 次（氛围营造）
- 固定：5 次（对话场景）
- 跟拍：2 次（动作跟随）
- 快速切入：2 次（转场强调）
- 黑场切换：2 次（场景转换）

**情绪递进设计：**
```
段 1（镜 1.1-1.4）：压抑静谧 → 建立基调

段 3（镜 3.1-3.4）：对抗激烈 → 村民反击
段 4（镜 4.1-4.2）：喜悦祥和 → 形成习俗
```

## 🎬 六、镜头选择指南

### 按场景快速选择

| 场景类型 | 推荐运镜 | 景别 | 时长 |
|----------|----------|------|------|
| 氛围营造 | 慢推、固定 | 远景/全景 | 3-5s |
| 对话场景 | 固定、轻微摇摄 | 中景/近景 | 2-3s |
| 动作场景 | 跟拍、快速切入 | 近景/特写 | 1-2s |
| 角色登场 | 环绕、后拉 + 上升 | 全景/中景 | 3-5s |
| 情绪特写 | 推近、固定 | 特写/大特写 | 2-4s |
| 场景转换 | 拉远、黑场、航拍 | 远景 | 2-3s |
| 宏大揭示 | 航拍、上升 + 俯视 | 远景 | 4-6s |

### 按情绪选择

| 情绪 | 推荐运镜 | 示例 |
|------|----------|------|
| 压抑 | 慢推、固定 | 镜头缓慢推进，细雪飘落 |
| 紧张 | 快速切入、手持跟拍 | 快速切入脸部特写 |

| 悲伤 | 拉远、固定 | 镜头拉远，角色孤独身影 |
| 激昂 | 快速推近、变焦 | 快速推近到坚定眼神 |

## 🎬 七、不推荐的写法

### ❌ 含糊词语
```
❌ "炫酷的镜头"
❌ "复杂的运镜"
❌ "快速切换"
✅ 应使用：具体运镜术语（推/拉/摇/移/环绕）
```

### ❌ 跨度过大
```
❌ "从地面直接飞到宇宙"
✅ 应使用：首尾帧锚定，中间过渡镜头
```

### ❌ 过多动作
```
❌ "同时升起 + 转圈 + 推进 + 下落 + 环绕"
✅ 应使用：2-3 个动作组合（如：上升 + 环绕）
```

### ❌ 复杂镜头语言直接写
```
❌ "希区柯克变焦效果"
✅ 应使用：组合模仿（推近 + 变焦）
```
