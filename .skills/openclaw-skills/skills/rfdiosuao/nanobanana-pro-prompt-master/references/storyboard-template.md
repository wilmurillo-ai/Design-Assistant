## 📖 使用说明

本模板基于 **年兽的由来** 实战案例整理，是 AI 视频生成的标准化分镜格式。

**核心优势：**
- ✅ 10 列完整结构（镜号/时长/运镜/景别/生图/生视频/对白/旁白/音乐/音效）

## 🎨 三、AI 生图提示词结构

### 标准结构

```
[风格]，[画质]，[场景/人物描述]，[色调/氛围]，[景别] 视角，[视角方向]
```

### 示例拆解（镜号 1.1）

```
国漫 3D 玄幻风格，8K 超清，寒冬腊月的古村落，屋顶积满厚雪，
烟囱无炊烟，村口挂着枯萎的红色灯笼，周围群山环绕，
天空飘着细雪，整体色调偏冷灰，氛围压抑静谧，
远景视角，平视
```

**结构分析：**
- **风格**：国漫 3D 玄幻风格
- **画质**：8K 超清
- **场景描述**：寒冬腊月的古村落（主场景）+ 细节元素（屋顶/烟囱/灯笼/群山/细雪）
- **色调/氛围**：冷灰，压抑静谧
- **景别**：远景
- **视角**：平视

## 🎬 四、AI 图生视频提示词结构

### 标准结构

```
[镜头运动描述]，[动态元素描述]，情绪 [情绪关键词]
```

### 示例拆解（镜号 1.1）

```
镜头缓慢向前推进，细雪匀速飘落，无其他动态元素，情绪压抑
```

**结构分析：**
- **镜头运动**：缓慢向前推进（慢推）
- **动态元素**：细雪匀速飘落
- **情绪**：压抑

### 动作行为分类

| 类型 | 结构 | 示例 |
|------|------|------|
| 单一动作 | 主体 + 单动作 | "老妪牵着孩童快速走向木屋" |
| 时序组合 | 动作 1 + 衔接词 + 动作 2 | "年兽头部快速向后退缩，然后转身逃跑" |

## 🔊 六、音效设计体系

### 音效分类

| 类型 | 示例 | 作用 |
|------|------|------|
| **自然环境音** | 风雪声、风声、火焰燃烧声、水流声 | 营造氛围 |
| **动作音效** | 脚步声、碰撞声、摩擦声、爆裂声 | 增强真实感 |
| **生物音效** | 人类声音（对话/呐喊/哭泣）、动物声音、怪兽声音 | 情绪传递 |
| **特殊音效** | 爆炸声、魔法音效、转场音效 | 戏剧张力 |

### 音量层级（默认）

```
对白（100%）> 音效（60-80%）> 音乐（30-50%）
```

### 音效时机

```
- 动作发生前 0.5 秒：预铺垫
- 动作发生时：同步
- 动作发生后 0.5 秒：余韵
```

### 音效空间感

| 景别 | 音效特征 |
|------|----------|
| 近景 | 清晰、响亮、细节丰富 |
| 中景 | 适中、平衡 |
| 远景 | 模糊、低沉、混响增加 |

## 📊 七、情绪递进设计

### 4 段式情绪曲线

```
段 1（开场）：压抑/平静 → 建立基调（25%）
段 2（发展）：紧张/不安 → 情绪递进（25%）
段 3（高潮）：对抗/激烈 → 情绪顶点（35%）
段 4（结局）：喜悦/祥和 → 情绪释放（15%）
```

### 情绪描述进阶

**基础版：**
```
恐惧、悲伤、嘲讽、害羞、喜悦、愤怒
```

**进阶版：**
```
眼露胆怯、放松欣慰、孤独发呆、抿嘴苦笑、偷看窃喜、害羞低头、
眉头紧锁、嘴角微扬、眼神坚定、呼吸急促
```
