# NanobananaPro 平台规格说明

> 平台参数详解 | NanobananaPro 生图 Skill 核心知识库

---

## 📖 平台概述

**NanobananaPro** 是专业级 AI 生图/视频生成平台，支持高质量图像和视频生成。

**核心能力：**
- 图像生成（单图/系列图）
- 视频生成（5s/10s/15s/30s）
- 首尾帧锁定（视频一致性）
- 风格迁移
- 高清放大

---

## ⚙️ 核心参数

### 1. 画幅比例 (--ar)

**支持比例：**

| 比例 | 值 | 适用场景 | 推荐平台 |
|------|-----|----------|----------|
| 正方形 | `1:1` | 头像、产品主图 | 全平台 |
| 竖屏视频 | `9:16` | 抖音、视频号、Reels | 抖音/视频号 |
| 横屏视频 | `16:9` | B 站、YouTube、院线 | B 站/YouTube |
| 电影宽屏 | `21:9` | 电影感、院线 | 院线/高端 |
| 照片比例 | `3:2` | 摄影作品 | 小红书/摄影 |
| 平板比例 | `4:3` | iPad 展示 | 教育/演示 |
| 小红书 | `3:4` | 小红书笔记 | 小红书 |

**语法示例：**
```
--ar 9:16
--ar 16:9
--ar 1:1
--ar 21:9
--ar 3:4
```

**推荐配置：**
- 抖音/快手：`--ar 9:16`
- 视频号：`--ar 9:16` 或 `--ar 1:1`
- 小红书：`--ar 3:4` 或 `--ar 1:1`
- B 站：`--ar 16:9`
- YouTube：`--ar 16:9`
- 院线：`--ar 21:9`

---

### 2. 风格模式 (--style)

**支持模式：**

| 模式 | 值 | 特征 | 适用场景 |
|------|-----|------|----------|
| 原始模式 | `raw` | 写实、少风格化 | 产品摄影、人像写真 |
| 默认模式 | `default` | 平衡、通用 | 大多数场景 |
| 表现模式 | `expressive` | 强风格化、艺术感 | 创意艺术、概念设计 |

**语法示例：**
```
--style raw
--style default
--style expressive
```

**推荐配置：**
- 电商产品：`--style raw`（真实质感）
- 人像写真：`--style raw`（自然肤色）
- 创意海报：`--style expressive`（艺术效果）
- 概念艺术：`--style expressive`（风格化）
- 日常使用：`--style default`（平衡）

---

### 3. 风格强度 (--s)

**取值范围：** `0-1000`

**强度分级：**

| 强度值 | 效果 | 适用场景 |
|--------|------|----------|
| 0-100 | 极简风格化 | 超写实摄影 |
| 100-200 | 轻微风格化 | 商业摄影、人像 |
| 200-300 | 中等风格化 | 创意摄影、艺术 |
| 300-500 | 强风格化 | 概念艺术、插画 |
| 500-1000 | 极强风格化 | 抽象艺术、实验 |

**推荐配置：**
- 产品摄影：`--s 150-200`
- 人像写真：`--s 150-250`
- 创意海报：`--s 250-350`
- 概念艺术：`--s 300-500`
- 二次元：`--s 250-400`

**语法示例：**
```
--s 200
--s 300
--s 500
```

---

### 4. 质量等级 (--q)

**支持等级：**

| 等级 | 值 | 渲染时间 | 画质 | 推荐场景 |
|------|-----|----------|------|----------|
| 草稿 | `0.25` | 最快 | 低 | 快速测试 |
| 标准 | `0.5` | 快 | 中 | 日常使用 |
| 高质量 | `1` | 中等 | 高 | 正式输出 |
| 超高质量 | `2` | 慢 | 最高 | 商业交付 |

**语法示例：**
```
--q 0.25
--q 0.5
--q 1
--q 2
```

**推荐配置：**
- 测试/草稿：`--q 0.25` 或 `--q 0.5`
- 日常使用：`--q 1`
- 商业交付：`--q 2`（强烈推荐）

## 🎬 首尾帧锁定

### 语法说明

**首帧锁定：**
```
[FIRST FRAME LOCK: 精准描述首帧画面]
```

**尾帧锁定：**
```
[LAST FRAME LOCK: 精准描述尾帧画面]
```

### 使用示例

**单镜头转场：**
```
[FIRST FRAME LOCK: Woman standing in modern office, facing camera, professional attire, confident smile]
[LAST FRAME LOCK: Same woman in same office, turning to look out window, contemplative expression]
```

**多镜头批量生成：**
```
镜头 1:
[FIRST FRAME LOCK: Product on white background, front view]
[LAST FRAME LOCK: Product on white background, 45-degree angle]

镜头 2:
[FIRST FRAME LOCK: Product on white background, 45-degree angle]
[LAST FRAME LOCK: Product on white background, side view]
```

### 首尾帧一致性法则

**必须保留 90% 以上固定元素：**
- ✅ 固定：人物、服装、场景、道具、光影
- ✅ 变化：姿态、表情、视角（仅 1 个核心变量）

**示例对比：**

❌ **错误示例**（多变量的）：
```
[FIRST FRAME LOCK: Woman in red dress, office, daylight]
[LAST FRAME LOCK: Man in blue suit, park, sunset]
```
→ 人物、服装、场景、时间全变了，会崩坏

✅ **正确示例**（单变量）：
```
[FIRST FRAME LOCK: Woman in red dress, office, daylight, facing camera]
[LAST FRAME LOCK: Woman in red dress, office, daylight, turning to side]
```
→ 仅姿态变化，其他元素锁定

## ⚖️ 权重语法

### 提升权重

**语法：** `(关键词：权重值)`

**权重范围：** `1.0-2.0`

**示例：**
```
(masterpiece:1.3)      # 轻微强化
(ultra-high quality:1.4)  # 中等强化
(best quality:1.5)     # 强强化
(8K UHD:1.6)          # 极强强化
```

**推荐权重：**
- 核心质量词：1.3-1.4
- 风格关键词：1.2-1.5
- 关键元素：1.4-1.6
- 负面词：1.5-1.7

### 降低权重

**语法：** `[关键词：权重值]`

**权重范围：** `0.5-0.9`

**示例：**
```
[blur:0.7]        # 轻微模糊效果
[soft focus:0.6]  # 柔焦效果
[dark:0.8]        # 降低暗度
```

### 组合使用

```
(masterpiece:1.3), (best quality:1.4), (8K:1.5), [blur:0.7], [noise:0.6]
```

## 🎨 平台特色功能

### 1. 高清放大

**放大倍数：** 2x / 4x / 8x

**语法：**
```
--upscale 2
--upscale 4
--upscale 8
```

**推荐场景：**
- 社交媒体：2x 足够
- 商业印刷：4x 推荐
- 大幅海报：8x

### 2. 风格迁移

**参考图上传：** 支持上传参考图进行风格迁移

**语法：**
```
--image [图片 URL]
--style-reference [风格参考图 URL]
--character-reference [角色参考图 URL]
```

**强度控制：**
```
--image-weight 0.5    # 参考图权重 0.5
--style-weight 0.7    # 风格权重 0.7
```

### 3. 局部重绘

**支持功能：**
- 指定区域重绘
- 保持其他区域不变
- 多次迭代优化

**语法：**
```
--inpaint [遮罩图 URL]
--inpaint-prompt [局部描述]
```

### 4. 高清修复

**功能：** 修复低清图的细节

**语法：**
```
--hd-fix
--hd-strength 0.7    # 修复强度
```

## 🚨 常见问题排查

### 问题 1：生成速度慢

**原因：** 质量参数过高

**解决：**
- 测试阶段：`--q 0.5`
- 正式输出：`--q 2`

### 问题 2：风格化不够

**原因：** 风格强度过低

**解决：**
- 提升 `--s` 值（200→300→400）
- 切换 `--style expressive`

### 问题 3：画面太假

**原因：** 风格化过强

**解决：**
- 降低 `--s` 值（300→200→150）
- 切换 `--style raw`

### 问题 4：视频闪烁

**原因：** 首尾帧不一致

**解决：**
- 使用 `[FIRST FRAME LOCK]` 和 `[LAST FRAME LOCK]`
- 确保 90% 元素固定
- 仅保留 1 个核心变量

### 问题 5：画面模糊

**原因：** 质量参数低或负面词缺失

**解决：**
- 设置 `--q 2`
- 添加负面词：`(blurry:1.4), (low quality:1.5)`
