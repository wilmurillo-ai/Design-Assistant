# 🎨 OpenClaw 内容创作者快速配置工具

> 帮助内容创作者快速配置 OpenClaw 环境，一键搭建自媒体创作工作流

**版本**: v3.0.0  
**作者**: 郑宇航  
**许可证**: MIT-0

---

## 📦 安装

```bash
clawhub install openclaw-creator-setup
```

或在飞书中对机器人说：
```
安装内容创作者工具
```

---

## 🚀 核心功能

### 1. 环境检测
- ✅ 检查飞书授权状态
- ✅ 检测已安装的 Skills
- ✅ 验证配置完整性

### 2. 工作流模板
- 每日内容创作
- 周报生成
- 会议记录

### 3. 平台配置
支持：微信公众号、微博、抖音、小红书、B 站、知乎

---

## 💡 使用方法

在飞书中对机器人说：

- `安装内容创作者工具` - 执行完整配置
- `检查创作者环境` - 查看状态
- `配置平台 wechat weibo` - 添加发布平台
- `创建工作流 每日内容` - 创建工作流
