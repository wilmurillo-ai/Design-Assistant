# Temp Heartbeat - 临时心跳任务

> 一次性定时任务，执行后自动删除

[![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)](https://clawhub.ai/skills/temp-heartbeat)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

---

## 🎯 Skill 定位

**临时心跳任务管理器** - 允许用户设置一次性临时心跳任务，在指定时间后执行，执行完成后自动删除

---

## ⚡ 快速开始

### 安装

```bash
clawhub install temp-heartbeat
```

### 触发词

- `/heart` - 标准触发
- `临时心跳` - 中文触发
- `定时任务` - 定时触发

### 使用示例

**创建临时心跳：**
```
/heart 10 分钟后提醒我开会
```

**查看任务列表：**
```
/heart list
```

**取消任务：**
```
/heart cancel task_123
```

---

## 🏗️ 核心功能

### 1. 创建临时心跳

```
/heart {时间} {任务内容}
```

**支持的时间格式：**
- `10 分钟后` - 相对时间
- `1 小时后` - 相对时间
- `今晚 8 点` - 绝对时间
- `明天上午 10 点` - 绝对时间

### 2. 查看任务列表

```
/heart list
```

显示所有待执行的临时心跳任务

### 3. 取消任务

```
/heart cancel {任务 ID}     # 取消单个任务
/heart cancel all           # 取消所有任务
```

## 🔧 技术特性

### 自动删除
- ✅ 执行后自动删除任务文件
- ✅ 不写入 HEARTBEAT.md
- ✅ 适合一次性任务

### 时间解析
- ✅ 相对时间（10 分钟后/1 小时后）
- ✅ 绝对时间（今晚 8 点/明天 10 点）
- ✅ 支持分钟/小时/天单位

### 任务管理
- ✅ 查看所有待执行任务
- ✅ 取消指定任务
- ✅ 一键取消所有任务
