# Context Usage Checker - 上下文使用率查询 Skill

快速查询当前会话的 Token 使用情况，支持进度条可视化和智能预警。

## 🚀 快速开始

### 安装
```bash
claw skill install context-usage-checker
```

### 使用
直接对话：
```
查一下上下文使用率
看看 token 用了多少
```

## ✨ 功能特性

- 📊 实时查询 Token 使用情况
- 📈 进度条可视化展示
- ⚠️ 智能预警（70%/90% 阈值）
- 🎯 支持多种模型
- ⚡ 快速响应（<10ms）
