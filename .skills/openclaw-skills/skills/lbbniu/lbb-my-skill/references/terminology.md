# 产品术语表

## 使用说明

本术语表提供了技术文档中常用术语的标准中英文对照。翻译时必须严格遵循此表，确保术语的一致性和准确性。

**原则**：
- 术语表中的术语必须按照指定译法翻译
- 首次出现时可以用"中文（英文）"的形式注释
- 后续出现保持译法一致
- 产品特定术语保持英文或使用品牌标准译法

## 使用建议

1. **术语优先级**：术语表 > 产品文档 > 行业惯例
2. **上下文适配**：某些术语在不同上下文下有不同译法，选择最合适的
3. **保持更新**：发现新术语或更好的译法时，及时更新术语表
4. **团队协作**：翻译团队应共同维护和遵守术语表
