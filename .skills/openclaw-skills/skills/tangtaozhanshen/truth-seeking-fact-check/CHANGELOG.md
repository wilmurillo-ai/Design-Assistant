# CHANGELOG - Truth (求真)

## v1.5.0 (2026-03-31)

### 新增功能
- **元认知置信度输出**：模型对核查结论输出自我置信度（0-100%），告诉用户对结论的确定程度，符合专家团建议
- **维度权重可配置**：用户可自定义各核查维度权重，关闭不需要的维度，灵活适配不同场景
- **可选定时核查能力**：默认关闭，用户配置后开启，支持定时重复核查内容，分数变化超过阈值告警
  - 最小间隔24小时，最大任务数10，单次最大并发2，完全适配2核2G资源约束
  - 仅当可信度分数变化≥2分时触发通知，减少噪音
- **完善隐私保护声明**：明确所有数据本地处理，不上传到外部服务器，100%隐私保护

### 改进
- 优化核心计算逻辑，几乎不增加额外资源占用
- 完全遵循专家团讨论共识，保持现有6层框架不变，只增量新增功能
- 严格控制资源，默认关闭高资源功能，用户按需开启

### 隐私保护
本版本严格遵循隐私保护原则：
1.  所有核查完全运行在用户自己的OpenClaw实例中
2.  用户核查内容**不会上传到任何外部服务器/开发者服务器**，完全本地处理
3.  技能不收集、不存储、不分享任何用户核查内容、个人数据
4.  开源可审计，代码完全开放，任何人可以验证隐私保护实现
