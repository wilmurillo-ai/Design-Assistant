# A 股盯盘技能 - 完整功能文档

## 📊 项目概述

**A 股盯盘技能**是一个功能完整的股票分析工具，提供实时行情、技术分析、自动化报告和智能投顾功能。

### 核心特性

✅ **完全免费** - 所有数据源均免费，无需 API Key  
✅ **多数据源** - 腾讯财经 + 新浪财经 + 东方财富，自动故障转移  
✅ **20+ 技术指标** - MA/RSI/MACD/KDJ/BOLL 等  
✅ **自动化报告** - 每日收盘报告、持仓日报、异动监控  
✅ **智能投顾** - 风险评估、止损止盈、估值分析、投资组合优化  

---

## 📈 功能模块

### 1. 基础功能

| 功能 | 指令 | 说明 |
|------|------|------|
| 实时查价 | `查询 600036 行情` | 获取实时股价、涨跌幅、成交量 |
| 价格预警 | `盯盘 600036 涨至 35 元` | 目标价触发提醒 |
| 风险提示 | `600036 风险提示` | 暴涨/暴跌/高波动预警 |
| 操盘建议 | `600036 操盘建议` | 基于规则的操盘参考 |

### 2. 技术分析

| 功能 | 指令 | 说明 |
|------|------|------|
| 技术分析 | `600036 技术分析` | 完整技术分析报告 |
| 查看指标 | `600036 指标` | 同上 |

**支持指标：**
- 趋势：MA/EMA (5/10/20/60 日)
- 动量：RSI、MACD、KDJ
- 波动：BOLL、ATR
- 成交量：OBV、量比

### 3. 自动化报告

| 功能 | 指令 | 说明 |
|------|------|------|
| 收盘报告 | `收盘报告` / `日报` | 市场概览 + 个股详情 |
| 持仓报告 | `持仓报告` | 持仓盈亏统计 |
| 添加自选 | `添加到自选 600036 招商银行` | 加入监控列表 |
| 设置持仓 | `设置持仓 002892 成本 10.5 数量 1000 科力尔` | 记录持仓成本 |
| 价格预警 | `设置预警 600036 涨至 40 元` | 目标价提醒 |

### 4. 智能投顾

| 功能 | 指令 | 说明 |
|------|------|------|
| 风险评估 | `风险评估` | 5 题测评风险画像 |
| 提交评估 | `风险评估 3,3,3,4,3` | 提交答案获取建议 |
| 止损止盈 | `止损止盈 入场价 10.5 当前价 11.2 仓位 10000` | 动态计算止损止盈位 |
| 估值分析 | `估值分析 600036 PE 15 行业 PE 20 增长 15` | PE 相对估值法 |
| 投资组合 | `投资组合` | 资产配置优化建议 |

## 🔧 技术架构

### 数据源

| 优先级 | 数据源 | 延迟 | 特点 |
|--------|--------|------|------|
| 主 | 腾讯财经 | 5-15 秒 | 数据完整（含最高/最低/成交额） |
| 备 | 新浪财经 | 5-20 秒 | 历史数据（60 天 K 线） |
| 降 | 东方财富 | 1-3 秒 | 实时性好（但数据不完整） |

### 模块结构

```
a-stock-watcher/
├── a_stock_watcher.py      # 主模块（行情 + 指令处理）
├── technical_analysis.py   # 技术分析（20+ 指标）
├── historical_data.py      # 历史数据（K 线获取）
├── auto_report.py          # 自动化报告
├── notification.py         # 消息推送
├── investment_advisor.py   # 智能投顾
├── stock_config.json       # 配置文件（自选股/持仓）
└── SKILL.md               # 技能文档
```
