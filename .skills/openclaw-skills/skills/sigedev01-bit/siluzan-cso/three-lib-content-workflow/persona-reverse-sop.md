# 代表作品反推人设卡 SOP

## 目标
通过少量代表作品，反推出一个账号的人设卡，包括：
- 账号定位
- 受众
- 核心任务
- 风格
- 选题边界
- 常用表达
- 禁忌
- 适合内容形态

## 输入要求
你给我的内容最好是：
- 1-5 篇代表作品
- 如果有，附上账号名称或用途
- 如果有，附上你对这个账号的直觉判断
- 如果有，附上你希望我重点关注的点

## 第一步：先读作品，不急着下结论
先看作品本身，找这些信息：
- 讲什么主题
- 面向谁
- 语气是什么
- 开头怎么起
- 中段怎么推
- 结尾怎么收
- 有没有固定套路
- 有没有强烈的情绪或判断

这一阶段不要先写人设，先**找共性**。

## 第二步：从作品里反推 7 个维度
### 1）账号定位
回答：这组作品整体在解决什么问题？它更偏认知、方法、情绪，还是转化？

### 2）目标受众
回答：它明显在写给谁看？是老板、业务员、外贸新人、创业者，还是消费者？受众的成熟度如何？

### 3）核心任务
回答：它是提供判断？提供方法？提供灵感？提供案例？提供安慰？提供成交？

### 4）核心风格
回答：是强势还是温和？是理性还是感性？是专业还是口语？是犀利还是稳重？是快节奏还是叙述型？

### 5）选题边界
回答：它常讲什么？它不讲什么？它适合哪些主题？它有哪些明显禁区？

### 6）常用表达
回答：有没有固定句式？有没有高频关键词？有没有标志性比喻？有没有喜欢用的转折结构？

### 7）适合的内容形态
回答：热点解读、方法教程、案例拆解、趋势判断、爆款口播、转化内容。

## 第三步：提取“作品证据”
不要只写主观判断，要写证据。
比如：
- 从作品 A 看，账号更像……
- 从作品 B 看，它常用……
- 从作品 C 看，它不太写……

这样写出来的人设卡更稳，不容易跑偏。

## 第四步：先写“临时人设卡”
如果作品还不多，不要强行写死。先写一个**临时版**，分三类：

### 已确认
作品已经非常明确表现出来的特征。

### 高概率
从多篇作品中能看出来，但还可以继续验证的特征。

### 待验证
目前只是推测，后面需要更多作品确认的部分。

## 第五步：整理成人设卡
推荐结构如下：
```markdown
# XXX 人设卡（由代表作品反推）

## 代表作品
- 作品1
- 作品2
- 作品3

## 作品共性
- ...

## 反推账号定位
- ...

## 反推受众
- ...

## 反推核心任务
- ...

## 反推核心风格
- ...

## 反推选题方向
- ...

## 反推常用表达
- ...

## 反推禁忌
- ...

## 适合的内容形态
- ...
```

## 第六步：如果要落地到写稿，还要补 3 项
人设卡不只是描述，还要能直接用。
所以建议再补：

### 1）开头偏好
- 反差开头
- 设问开头
- 结论先行
- 故事开头

### 2）中段推进方式
- 三点拆解
- 现象→本质
- 问题→解法
- 案例→判断

### 3）结尾收法
- 金句收尾
- 行动提醒
- 评论互动
- 下期预告
- 边界提醒

## 标准工作流
**拿作品 → 找共性 → 拆 7 维 → 找证据 → 写临时卡 → 补开头/中段/结尾偏好 → 输出人设卡**

## 最实用的判断原则
如果你只靠作品反推人设卡，最怕两件事：
1. **过度脑补**：看到一篇作品就把人设写死。
2. **只看风格，不看任务**：只写“犀利、专业、稳重”，但没写它到底服务什么目标。

所以一定要记住：**人设 = 定位 + 受众 + 任务 + 风格 + 边界**，不是只有风格。
