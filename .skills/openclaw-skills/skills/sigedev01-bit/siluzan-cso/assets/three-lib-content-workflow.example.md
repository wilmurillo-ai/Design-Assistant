# three-lib-content-workflow 使用说明

这是一套把“热点/素材 → 选题 → 结构 → 文案 → 复盘”串起来的工作流。核心是三层：

- 流量因子库：决定写什么更容易被看见
- 产品资产库：决定写什么更有长期价值
- 烹调方法库：决定怎么写得像目标账号

## 你什么时候该用它
当你要做下面这些事时，直接用这套流程：
- 从热点里挑选题
- 把一篇文案改成成功易或尚薇风格
- 把代表作品反推成一个人设卡
- 复盘一篇爆款为什么有效
- 输出能重复使用的文案 SOP

这个文件已经把：
- 三库定义
- 人设卡
- 文案生成 SOP
- 代表作品反推人设
- 协作关系
都整合在一起了。

## 推荐使用方式
### 1. 先定人设
先确定是：
- 成功易
- 尚薇
- 还是一个新的人设

### 2. 再定输入类型
你可以给它：
- 热点
- 一篇文案
- 代表作品
- 一个选题方向

### 3. 让它按流程处理
它会按下面顺序工作：
- 先找流量因子
- 再找产品资产
- 再选烹调方法
- 最后输出成稿 / 人设卡 / 选题建议

## 如果你只想快速上手
你可以直接这样说：
- “帮我按三库拆这篇文案”
- “根据这个热点，生成成功易选题”
- “把这几篇代表作品反推成人设卡”
- “把这篇改成尚薇风格”

## 一句话总结
这不是单纯的写稿说明，而是一套可重复使用的内容生产系统。
