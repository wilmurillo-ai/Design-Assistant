# Cloud Analysis Note

## Overview

ClawSpa is published on ClawHub as a **local-first maintenance skill**. Optional cloud analysis is available through **clawspa.org** for users who want deeper recommendations and historical comparisons.

## Publishing boundary

This published skill bundle does **not** include the full cloud API contract. ClawHub should present the maintenance skill primarily as a local scan tool, with optional cloud features documented separately on the website.

## Guidance

- Run local scans first
- Review clawspa.org docs and privacy details before using optional cloud analysis
- Do not send raw memory content or credentials
- If cloud analysis is unavailable, continue with local-only scans
