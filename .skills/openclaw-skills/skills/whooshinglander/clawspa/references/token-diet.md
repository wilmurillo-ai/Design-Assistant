# Token Diet — Add-on Treatment

## Overview

Token Diet is an optional ClawSpa add-on treatment powered by the companion skill **Where Am I Burning Tokens?**.

It audits OpenClaw token spend, identifies sinkholes like bloated sub-agents, noisy crons, or runaway logging, and recommends where to trim context calories.

## Companion Skill

- ClawHub: https://clawhub.ai/whooshinglander/whereamiburningtokens

## What it checks

- Token usage by session category
- Estimated cost by category
- High-volume, low-value sinkholes
- Expensive background work with low token share

## Outcome

Use Token Diet when the agent feels heavy, costly, or inefficient and you want a focused spend audit before changing prompts, crons, or model routing.
