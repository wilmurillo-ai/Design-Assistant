# 重命名文档命令

重命名思源笔记文档。

## 命令格式

```bash
siyuan rename <docId> <title>
```

## 参数说明

| 参数 | 类型 | 必填 | 说明 |
|-----|------|------|------|
| `<docId>` | string | ✅ | 文档 ID（位置参数） |
| `<title>` | string | ✅ | 新标题（位置参数） |

## 使用示例

### 基本用法

```bash
# 重命名文档
siyuan rename <docId> "新标题"

# 完整示例
siyuan rename 20260304051123-doaxgi4 "更新后的标题"
```

## 返回格式

```json
{
  "success": true,
  "data": {
    "id": "20260304051123-doaxgi4",
    "title": "更新后的标题",
    "notebookId": "20260227231831-yq1lxq2"
  },
  "message": "文档重命名成功",
  "timestamp": 1646389200000
}
```

## 注意事项

1. **文档ID格式**：文档 ID 格式为 14 位数字 + 短横线 + 7 位字母数字
2. **标题长度**：标题长度建议不超过 512 字符
3. **特殊字符**：标题中避免使用特殊字符，如 `/`、`\`、`:` 等
4. **权限限制**：需要相应的权限才能重命名文档
5. **重名检测**：如果同级已存在同名文档，会返回错误
