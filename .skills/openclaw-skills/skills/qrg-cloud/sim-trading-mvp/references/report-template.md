# Post-market report template

Use concise Chinese.

## Recommended sections

### Account overview
- equity
- daily return
- cumulative return
- benchmark comparison (SPY / QQQ)

### Current positions
- ticker / weight / cost / unrealized PnL when available

### Today's three decision windows
1. pre_or_open
2. intraday
3. near_close

For each window include:
- action: BUY / SELL / HOLD
- ticker if relevant
- reason

### Core judgment
- what mattered most today
- what the strategy was leaning into
- what it avoided

### Review
- what was right
- what may be wrong
- what to watch next

## Truthfulness reminder

If any figure is unavailable or estimated, label it clearly. Never present guessed values as retrieved market data.

Use the single primary market-data source (recommended: Finnhub) for prices, benchmark comparison, and account valuation whenever available.
Do not place API keys inside the report template or any public skill files.
