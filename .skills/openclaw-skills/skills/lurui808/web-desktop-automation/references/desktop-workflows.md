# Desktop Workflows

Use desktop automation for:
- Native apps
- File manager
- Cross-window copy/paste
- Keyboard shortcuts
- GUI-only workflows

## Preferred approach
- Focus the correct window first
- Prefer app-specific shortcuts
- Verify the active window before typing
- Minimize reliance on screen coordinates

## Common failure modes
- Window focus lost
- Resolution changes
- App moved between monitors
- UI theme or scaling changes
