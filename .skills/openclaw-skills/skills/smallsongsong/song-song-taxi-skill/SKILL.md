# 个人出AI服务 (Taxi Mobility)

通过 DiDi MCP Server API 实现完整的出行服务，支持打车、路线规划等功能。

## 核心功能

### 打车服务

| 功能 | 说明 |
|------|------|
| 实时叫车 | 需用户确认后下单 |
| 预约出行 | 托管叫车需求，到点后自动发起打车请求 |
| 订单跟踪 | 自动轮询订单状态 |
| 取消订单 | 需用户确认后执行 |
| 查询司机位置 | 逆地址编码 + 美化输出 |
| 价格预估 | 获取各车型价格 |

### 路线规划

| 功能 | 说明 |
|------|------|
| 驾车路线 | 规划小客车/轿车出行方案 |
| 公交地铁 | 综合公交、地铁通勤方案 |
| 步行路线 | 规划步行出行方案 |
| 骑行路线 | 规划骑行出行方案 |

### 周边搜索

| 功能 | 说明 |
|------|------|
| 附近POI搜索 | 根据位置坐标搜索周边餐饮、加油站等 |
