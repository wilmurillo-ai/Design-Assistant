### 压缩贡献分解

| 手段 | 贡献占比 | 说明 |
|------|---------|------|
| 结构拆分 | ~70% | 详细规则 → skill 引用（最大贡献） |
| 删除冗余 | ~15% | 删除 BOOTSTRAP.md、IDENTITY.md |
| 去重复 | ~8% | 跨文件唯一来源 |
| 去范例/说明 | ~5% | 比喻列、示例、模板残留 |
| 文言化 | ~2% | 边际贡献，天花板低 |

### Token 级效果

| 指标 | 瘦身前 | 瘦身后 |
|------|--------|--------|
| 注入 bytes | ~30,000 | ~6,000 |
| 注入 tokens | ~8,000 | ~1,800 |
| 系统指令 | ~15,000 | ~15,000 (不变) |
| 每轮基础 | ~23,000 | ~16,800 |
| **节省/轮** | — | **~6,200 tokens (-27%)** |

> 系统指令 (~15k tokens) 由 OpenClaw 平台注入，不可控。
> 我们只能优化工作空间注入部分 (~8k → ~1.8k tokens)。
