# 购票流程详细说明

本文档包含猫眼购票技能各分支的详细流程说明。

## 分支 1 - 查热映影片

**触发关键词：** "最近有什么电影"/"热映影片"/"现在什么好看"

**操作流程：**
1. 调用 `get-hot-movies` 获取热映列表
2. 展示 8-10 部影片（名称、评分、主演、场次信息）
3. 询问用户对哪部感兴趣，或是否需要查询特定影片

**示例对话：**
```
用户：最近有什么电影？
助手：最近热映的影片有：
1. 飞驰人生3 - 9.7分
2. 挽救计划 - 9.4分
...
想看哪部？
```

---

## 分支 2 - 查附近影院

**触发关键词：** "附近有什么影院"/"周边影院"/"附近能看电影吗"

**操作流程：**
1. 询问用户位置（地址或区域）
2. 调用 `get-nearby-cinemas`（支持 address 自动地理编码）
3. 展示附近影院（名称、距离、票价、特色）
4. 用户选择影院后，展示该影院的影片和场次

**示例对话：**
```
用户：附近有什么影院？
助手：请问您在哪个位置？
用户：北京朝阳三里屯
助手：附近有以下影院：
1. 英皇电影城（三里屯店）- 130m
2. 新影联东环影城 - 1.5km
...
```

---

## 分支 3 - 指定影片购票（最常用）

**触发关键词：** "帮我买飞驰人生3"/"我想看xxx"/"买张xxx的票"

**操作流程：**
1. 用 `search-movies` 确认影片信息（处理用户输入的片名）
2. **询问用户位置**："请问您在哪个城市/区域？"
3. 调用 `get-cinemas-by-movie`（传入 movieId + address + showDate）
4. 展示放映该影片的附近影院（按距离排序）
5. 用户选择影院后，展示该影院的场次

**示例对话：**
```
用户：帮我买飞驰人生3
助手：好的！《飞驰人生3》正在热映，9.7分。
请问您在哪个城市/区域？
用户：杭州西湖区
助手：西湖区附近放映《飞驰人生3》的影院有：
1. 完美世界影城 - 1.8km
2. 浙影时代影城 - 1.9km
...
```

---

## 分支 4 - 查特定影院

**触发关键词：** "至潮影城有什么片"/"万达影城"/"xxx影院"

**操作流程：**
1. **询问城市**（如果用户没提供）："请问影院在哪个城市？"
2. 调用 `get-cities` 获取城市列表，匹配 cityId
3. 调用 `search-cinemas`（传入 keyword + cityId）
4. 展示匹配的影院列表（如果有多个，让用户选择）
5. 用户选择影院后，调用 `get-showtimes` 展示该影院的所有影片和场次

**示例对话：**
```
用户：至潮影城有什么片？
助手：请问影院在哪个城市？
用户：杭州
助手：杭州的至潮影城有：
1. 至潮影城（星尚发展大厦店）- 拱墅区
2. 至潮影城（东兴天地店）- 拱墅区
...
```

---

## 分支 5 - 模糊购票

**触发关键词：** "我要买电影票"/"订电影票"/"看电影"

**操作流程：**
1. 调用 `get-hot-movies` 展示热映影片
2. 让用户选择感兴趣的影片
3. 进入分支 3 的后续流程

**示例对话：**
```
用户：我要买电影票
助手：好的！最近热映的影片有：
1. 飞驰人生3 - 9.7分
2. 挽救计划 - 9.4分
...
想看哪部？
```

---

## 错误处理

### 影片搜索不到
- 提示用户确认片名
- 提供相似影片推荐
- 或引导使用热映影片列表

### 影院搜索不到
- 提示用户确认影院名
- 询问具体城市
- 或推荐附近其他影院

### 城市匹配失败
- 提示用户输入完整城市名
- 展示热门城市列表供选择
