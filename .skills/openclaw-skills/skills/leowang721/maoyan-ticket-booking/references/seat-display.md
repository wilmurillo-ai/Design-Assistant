# 推荐座位展示规范

## 座位状态符号

| 符号 | 含义 | seatStatus |
|:---|:---|:---|
| ` ` (空格) | 过道/空白区域 | 0 |
| `○` | 可售 | 1 |
| `×` | 已锁定 | 2 |
| `●` | 已售出 | 3 |
| `■` | 禁售 | 4 |
| `★` | 推荐座位 |

## 重要原则

**"必须在图上标记推荐座位" 的意思是：**
- 推荐几个座位就在图上标记几个`★`
- 不是标记整个可选区域
- 只标记具体推荐的座位号码
- 最多展示 **6行 × 10列**，聚焦推荐座位附近区域
  - 仅展示推荐座位所在行及前后各2行（共5-6行）
  - 每行最多展示10个座位（推荐座位左右各5个）
- **重要：推荐座位必须实际显示在座位图中，不能只显示在文字说明里**
- **标记方式：`★` 直接替换掉对应位置的 `○`，不是额外添加**
  - 例如：推荐第6排8座、9座，则第6排显示为 `○○○○○○○★★○○○○○○○○○`

## 展示要求

- 最多展示10行
- 必须在座位图上用 `★` 标记出具体推荐的位置（推荐几个标记几个）
- **座位之间不需要空格间隔**，确保紧凑显示
- 排号放在左侧，使用接口返回的 `rowNum` 字段
- 在推荐排旁标注"←推荐"
- 文字说明中展示推荐座位及价格


**价格：¥40/张，2张共¥80**
