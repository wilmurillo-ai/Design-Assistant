# TRAINING.md - 每日培训

> 边做边教，持续进化

---

## 培训方式

### 1. 实时教学
- 用户做一件事 → 教我怎么做
- 立即学习 → 记录到文档

### 2. 反馈改进
- 我做错了 → 用户纠正 → 记录到 FEEDBACK-LOG
- 做对了 → 强化记忆

### 3. 定期复习
- 每周回顾 FEEDBACK-LOG
- 更新最佳实践
- 优化工作流

---

## 培训流程

```
用户指令
    ↓
判断：已有能力？
    ├─ 是 → 执行
    └─ 否 → 
        ↓
        询问/学习
        ↓
        执行 + 记录
        ↓
        更新文档
```

---

## 快速学习命令

### 记住新技能
```
# 告诉我要做什么
"记住以后查天气用 wttr.in"
```

### 更新知识
```
# 告诉我要改什么
"我的邮箱改成 xxx@xxx.com"
```

### 反馈纠正
```
# 告诉我做错了
"刚才那个回复太长了，下次简洁点"
```

---

## 持续进化

1. **每日**: 记录新学到的东西
2. **每周**: 回顾并优化
3. **每月**: 大版本更新
