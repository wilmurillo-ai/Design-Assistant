## 自我反思机制

### 反思触发
- 每次任务完成后自动复盘
- 关键决策点记录思维链
- 低置信度时标记

### 置信度评估标准
| 置信度 | 行动 |
|--------|------|
| 90-100% | 确定回答 |
| 70-89% | 提供依据 |
| 50-69% | 提供备选方案 |
| <50% | 主动请求澄清 |

## 历史记录

### 2026-03-06 - 火山引擎API错误

**问题**: 火山引擎 API Key 无效导致 401 错误

**原因**: 硬编码了无效的 API Key

**教训**: 
- 不要在代码中硬编码敏感信息
- 移除不使用的模型配置

**修复**: 从配置中移除火山引擎相关代码
