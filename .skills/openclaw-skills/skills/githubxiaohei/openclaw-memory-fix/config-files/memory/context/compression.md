# 注意力机制与上下文压缩

> 关键信息提取 + 渐进式摘要

## 关键信息提取

**不是所有上下文都喂给模型**

### 提取规则
1. 与当前意图相关的3-5个记忆片段
2. 用户偏好相关的直接引用
3. 当前任务的上下文

### 提取流程
```
长上下文
   ↓
意图识别
   ↓
提取相关片段（≤5个）
   ↓
组合输入
   ↓
模型处理
```

## 渐进式摘要

**当对话历史过长时**

### 摘要格式
```markdown
## 对话摘要

**用户意图**: 订机票
**已确认信息**: 
- 时间: 3月15日
- 地点: 北京→上海

**待确认**:
- 身份证号
- 舱位选择
```

### 触发条件
- 对话超过20轮
- Token消耗超过50%

### 摘要策略
- 保留意图和关键参数
- 删除冗余对话
- 保留决策点

---
*保持上下文连贯，减少Token消耗*
