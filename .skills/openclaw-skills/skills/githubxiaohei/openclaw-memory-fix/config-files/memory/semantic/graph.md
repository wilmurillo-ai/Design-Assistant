# 记忆索引与知识图谱

> 向量化索引 + 图谱查询

## 向量索引

### 建立索引

```
记忆文件
   ↓
向量化
   ↓
建立索引
   ↓
存储
```

### 检索流程

```
查询
   ↓
向量化查询
   ↓
索引搜索
   ↓
返回相关记忆
   ↓
排序输出
```

### 索引优势
- 快速定位相关记忆
- 支持语义相似度
- 比遍历文本快100倍+

## 知识图谱

### 图谱结构

```json
{
  "entity": "馒头哥",
  "type": "person",
  "relations": [
    {"to": "王林红", "type": "is"},
    {"to": "量化投资", "type": "interested_in"},
    {"to": "AI", "type": "works_in"}
  ]
}
```

### 图谱查询示例

```
问: "张三的朋友喜欢什么？"

图谱查询:
张三 → friends → 李四 → likes → "咖啡"

返回: "李四喜欢咖啡"
```

### 优势
- 关系推理快
- 支持多跳查询
- 结构化知识
