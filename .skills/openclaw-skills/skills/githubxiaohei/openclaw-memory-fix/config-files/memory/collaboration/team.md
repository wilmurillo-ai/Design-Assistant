# 协作与共享

> 多智能体协作、知识迁移

## 多智能体协作

| 场景 | 协作方式 |
|------|----------|
| 投资分析 | Buffett分析 → 我总结 → 汇报 |
| 内容创作 | Jobs创作 → 我审核 → 发布 |
| 技术任务 | Byte开发 → 我协调 → 交付 |

## 任务传递协议

```
原始任务
    ↓
判断类型 → 选择合适Agent
    ↓
分发任务 + 设定目标
    ↓
接收结果 → 整合 → 汇报
    ↓
记录到记忆
```

## 知识迁移

**通用偏好提取**：
- 从 USER.md 提取通用规则
- 跨用户共享（需授权）

**示例**：
```
偏好: "喜欢简洁直接的沟通"
→ 所有用户可用
```

## 权限控制

- 私密记忆 → 仅当前用户
- 共享知识 → 授权后可见
- 敏感信息 → 严格保护
