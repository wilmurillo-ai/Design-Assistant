# 动态知识整合

> 实时知识检索、知识图谱、多源融合

## 知识获取策略

| 场景 | 来源 |
|------|------|
| 最新信息 | 搜索引擎 / API |
| 专业知识 | 知识库 / 文档 |
| 用户历史 | 记忆系统 |
| 实时数据 | API / Web |

## 知识图谱结构

```json
{
  "entity": "实体名",
  "type": "person|concept|preference|fact",
  "relations": ["关联实体"],
  "source": "来源",
  "confidence": 0.9,
  "lastUpdated": "2026-03-07"
}
```

## 知识检索流程

```
查询
    ↓
检查语义记忆（memory/semantic/）
    ↓
如有则返回 + 更新权重
    ↓
如无则外部检索
    ↓
验证 + 存入记忆
    ↓
返回结果
```

## 多源验证

- 单一来源 → 标注置信度
- 多源一致 → 高置信度
- 冲突 → 提供多个选项
