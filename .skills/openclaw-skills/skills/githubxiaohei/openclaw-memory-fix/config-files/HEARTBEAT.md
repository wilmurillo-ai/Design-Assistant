# HEARTBEAT.md

# 每日自动安全巡检 + 任务检查
# 每天 3 次

## 巡检任务

### 1. 网关状态检查
- 检查网关是否仍绑定到 loopback (127.0.0.1)
- 检查是否正常运行

### 2. 防火墙状态检查
- 检查 macOS 防火墙是否开启

### 3. API 密钥安全检查
- 检查 .env 文件是否存在且安全存储

### 4. 异常检测
- 检查是否有异常的认证失败记录
- 检查最近是否有配置被修改

### 5. 待办任务检查
- 检查 TODO.md 是否有未完成的任务
- 检查 tasks.md 是否有待办事项

### 6. 记忆维护（每周日）
- 扫描最近一周的对话历史
- 提取重要事实写入对应实体的 items.json
- 执行 memory-decay.sh 刷新 summary（Hot/Warm/Cold 分档）
- 更新 life/index.md 索引

---

## 执行时间
- 每天 3 次：9:00 / 13:00 / 18:00

## 发现异常
- 发现任何异常立即通知用户

- 中午 13 点：快速状态检查
- 下午 6 点：待办任务进度确认
