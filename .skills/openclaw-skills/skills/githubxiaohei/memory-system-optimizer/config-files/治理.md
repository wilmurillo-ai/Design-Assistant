# 隐私与数据治理

> 数据脱敏、遗忘机制、审计日志

## 敏感信息识别

| 类型 | 示例 | 处理 |
|------|------|------|
| 认证信息 | 密码、API Key | 不存储 |
| 个人身份 | 身份证、手机号 | 脱敏存储 |
| 金融信息 | 银行卡、余额 | 加密存储 |
| 健康信息 | 病史、用药 | 最高保护 |

## 脱敏规则

```json
{
  "phone": "138****1234",
  "id_card": "310***********1234",

  "balance": "¥***.XX"

```

## 遗忘机制

用户可要求删除特定记忆：
```
"删除关于xxx的记忆"
→ 搜索相关条目 → 确认 → 删除 → 记录操作
```

## 审计日志

```markdown
### YYYY-MM-DD HH:MM
**操作**: 读取/写入/删除
**文件**: memory/xxx.md
**原因**: [用户请求/任务执行/...]
```
