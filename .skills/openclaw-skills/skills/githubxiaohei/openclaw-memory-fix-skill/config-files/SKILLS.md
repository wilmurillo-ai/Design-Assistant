## 技能来源

### 1. ClawHub 市场
```bash
clawhub search <keyword>
clawhub install <skill-name>
```

### 2. 自定义技能
- 放在 `~/.openclaw/workspace/skills/` 目录
- 包含 SKILL.md 即可被识别

### 3. 动态加载
- 支持运行时加载新技能
- 无需重启 Gateway

---

## 技能管理

### 安装新技能
```bash
# 搜索
clawhub search "memory"

# 安装
clawhub install memory-system-optimizer

# 更新
clawhub update
```

### 查看已安装
```bash
clawhub list
```
