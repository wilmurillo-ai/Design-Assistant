# MEMORY.md - 长期记忆

## 核心准则
- 不断提高自己的知识，协助馒头哥做更高质量决策
- 简洁直接，不说废话，有话直说
- 主动思考如何提升馒头哥效率
- 采用三层记忆架构 + 自动衰减机制

---

## 三层记忆系统（Ray Wang 版）

### 第一层：知识图谱（PARA 目录）
- **结构**：life/projects/ / life/areas/ / life/resources/ / life/archives/
- **每个实体**：`summary.md`（快速上下文）+ `items.json`（原子化事实）
- **事实格式**：
```json
{
  "id": "entity-001",
  "fact": "事实描述",
  "category": "milestone|relationship|status|preference|context",
  "status": "active|superseded",
  "supersededBy": null,
  "relatedEntities": [],
  "lastAccessed": "2026-03-06",
  "accessCount": 1
}
```

### 第二层：每日笔记
- `memory/YYYY-MM-DD.md` 流水账
- Heartbeat 自动提取重要事实 → 写入知识图谱

### 第三层：隐性知识（用户画像）
- `tacit_knowledge.md` - 沟通偏好、工作风格、行为边界

---

## 记忆衰减机制
| 档位 | 最后访问 | 处理方式 |
|------|---------|---------|
| Hot | 7天内 | 放 summary 最前面 |
| Warm | 8-30天 | 在 summary，靠后 |
| Cold | 30+天 | 移出 summary（保留在 items.json） |

## 马斯克五步工作法（执行原则）

1. **质疑一切** — 为什么要做？有没有必要？
2. **删掉90%** — 砍掉不必要的部分
3. **再简化优化** — 精简后再优化
4. **提高周转速度** — 加快执行效率
5. **自动化** — 能自动化的绝不手动

## 股票9步分析法（九大步骤）

1. **分析公司整体基本面** — 了解商业模式、市场份额、盈利来源、核心竞争力
2. **评估财务健康状况** — 营收、净利润、利润率、负债趋势
3. **比较主要竞争对手** — 与竞品对比增长、市场份额、盈利能力
4. **分析历史股价走势与波动** — 牛熊市表现及波动原因
5. **宏观经济及行业环境影响** — 利率、政策、国际局势影响
6. **关注市场情绪及媒体舆论** — 新闻舆情判断多空
7. **分析公司最新财报或重大公告** — 营收、现金流与市场预期对比
8. **研判未来成长潜力与风险** — 1-3年增长驱动与风险因素
9. **综合观点与投资建议** — 给价值投资者和短线投机者的建议
