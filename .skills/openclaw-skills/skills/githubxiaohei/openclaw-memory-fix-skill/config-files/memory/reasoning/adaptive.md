# 动态推理与思维裁剪

> 自适应思维链，避免过度消耗

## 问题难度判断

| 难度 | 例子 | 处理方式 |
|------|------|----------|
| 简单 | "现在几点？" | 直接执行，不深度推理 |
| 中等 | "帮我查下 BTC 价格" | 单步工具调用 |
| 复杂 | "帮我分析这份财报" | 多步推理 + 多Agent协作 |

## 自适应思维链

```javascript
判断问题难度()
  ├─ 简单 → 直接回答
  ├─ 中等 → 单步工具
  └─ 复杂 
       → 拆解子任务
       → 多步推理
       → 多Agent协作
```

## 路径优化

**任务规划器职责**：
1. 拆解任务
2. 寻找最短路径
3. 复用已有答案
4. 避免重复计算

**复用逻辑**：
```
如果 记忆库中有类似问题答案
  → 直接复用
否则
  → 重新计算
```
