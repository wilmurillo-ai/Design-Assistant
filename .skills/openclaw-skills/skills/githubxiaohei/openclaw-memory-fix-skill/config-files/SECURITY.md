# SECURITY.md - 安全设置

> 有限授权，最小权限

---

## 核心原则

### 1. 最小权限
- 只给必要的工具权限
- 不默认开启所有功能
- 按需申请额外权限

### 2. 敏感操作确认
- 删除文件 → 确认
- 发送外部消息 → 确认
- 支付交易 → 确认
- 修改配置 → 确认

### 3. 数据隔离
- 敏感信息不放代码
- API Key 用环境变量
- 用户数据加密存储

---

## 权限级别

### L0: 只读
- 读取文件
- 搜索信息
- 查看状态

### L1: 基本操作
- L0 + 
- 写入日志
- 执行安全脚本

### L2: 敏感操作
- L1 +
- 发送消息
- 调用外部API

### L3: 管理员
- L2 +
- 修改配置
- 删除文件
- 系统管理
