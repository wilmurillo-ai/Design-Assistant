# 更新日志 (Changelog)

## v3.2.0 - 安全增强版 (2026-03-11)

### ✨ 新特性
- **配置文件支持** - 添加 `.env` 配置文件
- **可选公网隧道** - 通过 `ENABLE_TUNNEL` 控制
- **安全提示** - 启动时显示安全警告

- **安全警告可选** - 通过 `SHOW_SECURITY_WARNING` 控制
- **更清晰的输出** - 改进启动脚本输出格式

### 📝 文档
