# 相机购买推荐智能体技能

这是一个智能相机推荐技能，可以根据用户需求、预算和使用场景，推荐合适的相机产品。

## 功能特点

- **需求分析**：深入了解用户的摄影需求、预算、技术水平
- **产品推荐**：根据用户条件推荐最适合的相机产品
- **对比分析**：提供多款相机的详细对比
- **购买建议**：给出专业的购买建议和注意事项

## 使用方法

在对话中直接提出你的相机购买需求，例如：
- "我想买一台相机，预算5000元左右，主要拍风景"
- "推荐一台适合初学者的单反相机"
- "我想买台微单，预算1万以内"

智能体会自动分析你的需求并给出专业建议。
