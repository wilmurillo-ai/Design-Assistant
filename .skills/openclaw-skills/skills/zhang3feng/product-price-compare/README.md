# 商品比价智能体技能 (product-price-compare)

跨平台商品比价智能体技能，支持淘宝、京东、拼多多等主流电商平台的价格自动抓取、优惠规则解析与等效到手价计算。

## 📋 功能特性

### 核心能力

| 功能 | 说明 |
|------|------|
| **跨平台比价** | 支持京东、天猫、拼多多、苏宁等主流电商平台 |
| **优惠解析** | 自动解析满减、折扣、赠品等复杂优惠规则 |
| **价格归一化** | 统一转换为等效到手价，实现公平对比 |
| **智能推荐** | 基于价格、配送、服务给出购买建议 |
| **历史价格** | 提供 90 天价格趋势分析 |
| **企业采购** | 支持批量采购比价和企业协议价查询 |

### 解决的问题

| 痛点 | 解决方案 |
|------|----------|
| 信息碎片化 | 统一采集多平台数据，标准化展示 |
| 操作繁琐 | 一句话指令自动完成全流程 |
| 优惠规则复杂 | 自动解析满减、赠品等非标优惠 |
| 实时性不足 | 实时抓取最新价格 |
| 缺乏决策依据 | 提供历史价格趋势与推荐建议 |

## 🔧 技术实现

### 工作流程

```
┌─────────────┐
│ 1. 意图解析  │ → 识别商品名、目标平台
└─────────────┘
       ↓
┌─────────────┐
│ 2. 商品定位  │ → 调用 web-search-lite
└─────────────┘
       ↓
┌─────────────┐
│ 3. 价格抓取  │ → 启动 automation-browser
└─────────────┘
       ↓
┌─────────────┐
│ 4. 优惠计算  │ → 解析满减、折扣、赠品
└─────────────┘
       ↓
┌─────────────┐
│ 5. 生成报告  │ → 输出结构化比价表格 + 推荐
└─────────────┘
```

### 价格归一化算法

```python
等效到手价 = (商品总价 - 满减 - 优惠券 + 运费 - 赠品折价) / 商品数量
```

### 商品匹配算法

通过多维特征哈希实现跨平台商品对齐：
- 品牌 + 型号 + SKU 编码（70% 权重）
- 主图视觉相似度（20% 权重）
- 标题关键词（10% 权重）

匹配度 ≥ 85% → 判定为同一商品

## ⚠️ 注意事项

### 价格时效性
- 抓取的是当前页面价格
- 实际下单可能因优惠券/库存变化
- 建议尽快下单锁定价格

### 平台访问限制
- 如遇平台反爬，可能跳过该平台
- 部分平台需要登录才能查看价格
- 建议使用企业采购渠道获取更优价格

### 商品匹配
- 跨平台商品名称可能不同
- 系统自动匹配但需用户确认
- 建议提供详细型号信息
