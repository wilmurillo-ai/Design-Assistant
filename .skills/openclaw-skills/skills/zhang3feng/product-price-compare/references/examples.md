| 京东 | ¥7,999 | ¥7,699 | 白条减 200 | 次日达 |
| 天猫 | ¥8,199 | ¥7,899 | 88VIP 95 折 | 2-3 天 |

**推荐**: 拼多多（当前最低价，省¥400）

**购买链接**: 
- [拼多多](https://...)
- [京东](https://...)
- [天猫](https://...)

### 示例 4：多规格比价

**用户指令**：
```

```

**预期输出**：
```markdown
## 多规格比价分析

**商品**: iPhone 16 Pro

| 容量 | 京东 | 天猫 | 拼多多 | 推荐平台 |
|------|------|------|--------|----------|
| 128GB | ¥6,999 | ¥7,199 | ¥6,799 | 拼多多 |
| 256GB | ¥7,999 | ¥8,199 | ¥7,599 | 拼多多 |
| 512GB | ¥9,999 | ¥10,199 | ¥9,599 | 拼多多 |

**容量性价比分析**:
- 128GB→256GB：+¥800（+11.4%）
- 256GB→512GB：+¥2,000（+26.3%）

**推荐**: 256GB 版本性价比最高，存储翻倍且价格适中
```

## 错误处理示例

### 商品未找到

**用户指令**：
```
比较 XYZ 不存在的商品 123 的价格
```

**预期输出**：
```markdown
⚠️ **未找到匹配商品**

在京东、天猫、拼多多均未找到"XYZ 不存在的商品 123"。

**建议**:
1. 检查商品名称是否准确
2. 提供更详细的型号信息
3. 尝试使用商品官方名称

**相似商品推荐**:
- 商品 A（¥XXX）
- 商品 B（¥XXX）
```

```
比较 iPhone 16 在京东、天猫、拼多多的价格
```

**预期输出**：
```markdown
## 商品比价报告（部分数据）

**商品**: iPhone 16 128GB

| 平台 | 当前价格 | 状态 |
|------|----------|------|
| 京东 | ¥5,999 | ✅ |
| 天猫 | ¥6,199 | ✅ |
| 拼多多 | - | ⚠️ 暂时无法访问 |

**说明**: 拼多多平台暂时无法访问，已展示其他平台价格。

**推荐**: 京东（当前可访问平台中最低）
```
