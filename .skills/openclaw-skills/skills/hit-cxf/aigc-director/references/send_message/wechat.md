# 微信发送消息

## 注意事项

微信不支持 Markdown 语法，**不要使用 Markdown 表格**！

如果需要发送表格内容，改用**带缩进的编号列表**格式：

```markdown
1. 选项1：说明内容
   - 详情1
   - 详情2
2. 选项2：说明内容
   - 详情1
   - 详情2
3. 选项3：说明内容
   - 详情1
   - 详情2
```

## 示例

❌ 错误（Markdown 表格，微信不显示）：
```
| 选项 | 说明 |
|------|------|
| A | xxx |
| B | yyy |
```

✅ 正确（缩进编号列表）：
```
1. 选项A：xxx
   - 详情1
   - 详情2
2. 选项B：yyy
   - 详情1
   - 详情2
```
