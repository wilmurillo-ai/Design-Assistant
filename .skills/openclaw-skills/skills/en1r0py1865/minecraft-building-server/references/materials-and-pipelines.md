# Materials, Farms, and Production Pipelines

## 1. When This Matters

This reference is most relevant for:
- survival build servers
- industrial build servers
- mixed build/community worlds where material logistics affect building throughput

Pure creative build servers usually need much less of this.

## 2. Core Material Sources

### Mining
- branch mining for targeted gathering
- strip mining for volume
- TNT-based clearing for bulk extraction
- beacon + haste for fast manual throughput

### Logging
- manual chopping for small scale
- semi-automatic tree farms for medium scale
- TNT / piston tree farms for large scale

## 3. Key Farms for Builders

| Farm | Output | Build relevance |
|------|--------|-----------------|
| Sugar cane | paper/books | bookshelves, trading, decoration |
| Cactus | green dye | dyed blocks |
| Bamboo | scaffolding / bamboo blocks | construction + materials |
| Wool farm | colored wool | pixel art, decoration |
| Iron farm | iron | tools, hoppers, building blocks |
| Mob farm | bone meal / drops | supports other production systems |
| Concrete converter | concrete | one of the most useful builder automations |

## 4. Villager Trading

Key professions for builders:
- Mason — terracotta, polished blocks, quartz
- Librarian — tool enchantment support
- Farmer — emerald generation
- Shepherd — wool and decorative items
- Cartographer — map-related items

Useful long-term pattern:
1. breeder
2. trading hall
3. discounted trades via cure mechanic
4. stable emerald income

## 5. Redstone Production Pipelines

Typical pipeline:
1. input collection
2. processing (smelting / conversion)
3. sorting
4. storage

Important builder-facing systems:
- auto-smelter arrays
- concrete conversion lines
- hopper item sorters
- bulk storage rooms

## 6. Practical Framing

These systems matter because they reduce build friction. The goal is not automation for its own sake, but getting materials into builders' hands faster and more reliably.
