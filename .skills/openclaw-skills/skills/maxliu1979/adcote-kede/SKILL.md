# 浦江科德高级中学融合部

## 学校基本信息

| 项目 | 内容 |
|------|------|
| **学校名称** | 浦江科德高级中学融合部（International Department of Adcote Zhezhong Campus） |
| **品牌** | 阿德科特（Adcote）系列 |
| **主办方** | 上海雅力教育集团 + 长住集团 |
| **成立时间** | 2020年9月22日奠基，2024年9月1日开学 |
| **位置** | 浙江省金华市浦江县，观岩山之南，春风长住别墅小镇 |
| **定位** | 多元、国际、前沿、品质 |
| **培养目标** | 传承、创新、独立、领袖 |
| **办学追求** | 面向全球、面向未来的全人教育 |

## 课程体系

### 学制
- **四年制**：Pre-IG → IG → AS → A2
- **三年制**：IG → AS → A2

### 三大学院
- **理学院**：数学、高等数学、化学、生物（STEM方向）
- **商学院**：经济学、会计学、创变商学院（创业教育）
- **文学院**：中文、俄语、阿拉伯语（跨文化语言特色）

### 评估体系
- **IGCSE**：Cambridge 剑桥国际考评
- **A-Level**：三大考试局战略合作伙伴（爱德思/CIE/AQA）

## 特色项目

### 亚洲国家本科直通车
- **目标国家**：新加坡、马来西亚
- **优势**：
  - 留学周期短（高中3年+本科2-3年）
  - 总费用60-70万（远低于英美50万/年）
  - 毕业生可优先直升新马名校
  - 雅力教育集团在马来西亚有自有学校（英国阿德科特马来西亚分校）提供在地保障

### 个人发展课程
Pre-IG至AS/A2各阶段均配备：
- 素质拓展、STEM实验、生涯规划
- 社会实践、公共演讲、领导力
- 学术讲座、名校游学、名企实习
- 艺术体验

### 课外活动
- 科技类、艺术体育类、中西文化融合类社团
- 中医养生、徒步越野等户外拓展
- 社区服务、实习项目、国际交流

## 咨询与联系

- **适合家长**：希望孩子接受国际教育、目标全球升学
- **课程特色**：A-Level体系 + 亚洲本科直通车（双轨选择）
- **语言特色**：中文 + 俄语 + 阿拉伯语（非英语二外）
- **学校地址**：浙江省金华市浦江县春风长住小镇
