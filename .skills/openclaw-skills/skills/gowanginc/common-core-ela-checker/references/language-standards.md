# Language Standards

## Conventions (L.1-2)
- L.K-12.1: Demonstrate command of grammar and usage
- L.K-12.2: Demonstrate command of capitalization, punctuation, spelling

## Knowledge of Language (L.3)
- L.K-12.3: Apply knowledge of language in different contexts

## Vocabulary Acquisition (L.4-6)
- L.K-12.4: Determine word meanings using context clues
- L.K-12.5: Demonstrate understanding of figurative language
- L.K-12.6: Acquire and use academic vocabulary
