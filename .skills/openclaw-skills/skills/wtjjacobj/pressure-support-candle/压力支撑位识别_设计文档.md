# 压力位支撑位识别系统设计文档

## 1. 系统概述

本系统用于自动识别股票/指数K线图中的压力位和支撑位，基于波峰波谷分析和成交量验证。

## 2. 核心算法

### 2.1 波峰波谷识别

**局部极值法**：
- 使用滑动窗口（默认N=20日）寻找局部最高点（波峰）和局部最低点（波谷）
- 波峰：窗口内最高点，且高于前后各N/2个交易日
- 波谷：窗口内最低点，且低于前后各N/2个交易日

**显著过滤**：
- 幅度阈值：>5%
- 幅度计算：`|价格 - 前一价格| / 前一价格 × 100%`
- 只保留幅度超过5%的波峰波谷，消除小幅震荡噪音

### 2.2 压力位/支撑位对应

| 类型 | 定义 | 强度依据 |
|------|------|----------|
| 压力位 | 波峰价格 | 触及次数越多越强 |
| 支撑位 | 波谷价格 | 触及次数越多越强 |

**强度计算**：
- 基础强度 = 触及次数
- 有效支撑/压力位：触及次数 ≥ 2

### 2.3 成交量验证

**放量阈值**：`成交量 > 20日均量 × 1.5`

| 场景 | 信号 | 判定 |
|------|------|------|
| 放量突破压力位 | 有效突破 | 阳线+放量 |
| 缩量突破压力位 | 假突破可能 | 阳线+缩量 |
| 放量跌破支撑位 | 有效跌破 | 阴线+放量 |
| 缩量跌破支撑位 | 诱空可能 | 阴线+缩量 |

## 3. 数据流程

```
akshare获取K线 → 计算波峰波谷 → 聚类生成支撑/压力位 → 成交量验证 → 输出结果+可视化
```

## 4. 输出格式

### 4.1 支撑/压力位列表
```python
{
    'type': 'support' | 'resistance',
    'price': float,
    'strength': int,  # 触及次数
    'touches': [dates],  # 触及日期列表
    'valid': bool  # 是否有效（触及≥2次）
}
```

### 4.2 突破信号
```python
{
    'type': 'breakout' | 'breakdown',
    'price': float,
    'volume_ratio': float,  # 量比
    'signal': '有效' | '假突破可能' | '诱空可能'
}
```

## 5. 可视化

- K线蜡烛图
- 压力位：红色水平线 + 标注
- 支撑位：绿色水平线 + 标注
- 触及点：圆点标记

## 6. 参数配置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| window | 20 | 波峰波谷识别窗口 |
| min_amplitude | 0.05 | 最小幅度阈值(5%) |
| volume_ma | 20 | 成交量均量周期 |
| volume_threshold | 1.5 | 放量阈值系数 |
