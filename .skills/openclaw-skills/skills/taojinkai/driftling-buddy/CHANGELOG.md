# Changelog

## 1.0.0

- Initial public release of `buddy_mode`.
- Added sidebar-ready terminal buddy cards with bilingual Chinese and English rendering.
- Added rarity tiers with distinct card frames and star counts.
- Added collectible themed pools that can emerge from ordinary user behavior.
- Added more than twenty base species across two species packs.
- Added non-human creature lore, side quests, hints, and unlock examples.
- Added portable rendering and smoke-test scripts.
- Structured the project so species data lives separately from rendering logic for future expansion.
