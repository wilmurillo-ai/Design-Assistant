## 使用方法

### 在 Obsidian 中使用模板

1. 安装 **Templater** 插件
2. 将模板保存到 `Templates/` 文件夹
3. 设置快捷键快速插入模板

### 快速创建笔记

```bash
# 使用模板创建新笔记
cp Templates/daily-note.md journal/$(date +%Y-%m-%d).md
```

### 自定义模板

根据个人需求修改模板内容，添加常用标签、字段等。
