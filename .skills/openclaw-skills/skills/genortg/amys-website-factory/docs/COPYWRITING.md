# Copywriting & SEO templates

Hero section (short):
- Headline (6–12 words) — problem + benefit
- Subtitle (1–2 sentences) — who / what / why
- Primary CTA (verb + target): e.g. "Start free trial" / "Book demo"

Features (3–5 bullets): each bullet with title, 1‑line description, and optional micro-CTA.

About / Trust section:
- Short paragraph, logos, testimonial template (quote, name, role).

SEO meta template:
- title: {{siteName}} — {{primaryBenefit}}
- description: One concise sentence, 120–155 chars.
- og:title, og:description, og:image

Tone guidance:
- Use the brief's tone (editorial / playful / luxury / etc.). Keep CTAs clear and active.

Microcopy rules:
- Button labels: 2–4 words.
- Form labels: short + example placeholder.
