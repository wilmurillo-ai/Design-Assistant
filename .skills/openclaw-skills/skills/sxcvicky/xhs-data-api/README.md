# 小红书数据分析 Skill

帮你用数据回答小红书营销中的关键问题——该做什么内容、找哪类博主、竞品在做什么。

---

## 能帮你做什么

**内容策略**
- "我想进入护肤赛道，现在什么话题最热？竞争大不大？"
- "帮我分析敏感肌这个方向，有没有差异化机会"

**博主合作**
- "帮我找几个适合合作的美妆博主，互动率高的"
- "这个博主的粉丝画像是什么？跟我的目标人群匹配吗？"

**竞品分析**
- "华为手机在小红书上的内容策略是什么？"
- "帮我对比一下这几个品牌在小红书的表现"

**市场洞察**
- "手机数码行业现在哪些关键词在飙升？"
- "美妆行业的市场供需情况怎么样，有没有蓝海？"

---

## 怎么用

直接向 AI 描述你的问题，Skill 会自动调取相关数据并给出分析结论。

**你需要准备**：一个 API Key（`XHS_API_KEY`），向 Skill 提供者申请。
