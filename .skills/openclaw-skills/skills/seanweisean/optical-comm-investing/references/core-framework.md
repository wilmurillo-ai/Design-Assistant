## 二、框架（边读边迭代）

### 2.1 光通信需求量化公式

**光模块数量 = f(GPU数量, 集群架构, 算法复杂度)**

- **NVLink** = 机柜内近距离高速互联（电/铜/PCB）→ 解决"内存墙"
- **InfiniBand** = 跨机柜大规模扩展（光模块）→ 决定集群上限
- NVLink 之于 IB = CPU Cache 之于内存条
- Blackwell NVL72：集群GPU>576时，**每GPU配7.5支800G + 4支1.6T光模块**；比Hopper **+50%**

### 2.2 产业节奏时间轴

**设备投资前置6-9个月** → 从设备订单推导下游放量时点。
**估值**：高成长看**PS**（30-50x），差异化看：全球核心 vs 国产替代 / 垄断 vs 红海 / 久期8+年 vs 3-5年。

### 2.3 三律共舞模型

| 律 | 代言人 | 方向 |
|---|------|------|
| Scaling-Law | OpenAI | 算力需求↑（不封顶） |
| Moore-Law | TSMC | 单位成本↓（5→3→2nm功耗仅降15%） |
| **Infra-Law** | DC基建 | 功率密度↑时capex指数级↑ |

**硬约束**：NVL72 = 250kW/rack → 全世界仅6% DC能满足。

### 2.4 TSMC出题、NVIDIA答题

- TSMC路线10年+1000亿$ → **长周期稳定**
- NVIDIA架构每次GTC可能reset → **快周期**
- 正确姿势：从NVIDIA映射 → 转向TSMC映射（先进封装）

### 2.5 铜缆NVL72 = 出道即巅峰

250kW/rack Infra反噬 + TTM不可控 → 无演进空间 → R100代必然回归光

### 2.6 GreenField → BrownField拐点

2026-2027更新换代 → 存量改造看重兼容性/模块化/低功耗 → 有利NPO/CPO/硅光

### 2.7 快周期资产DCF数学死刑

铜缆PE：情景1（2026底终结）= **2.4x**；情景2（2028底）= **4.5x**
**"能把未来算死的行业，博弈价值几乎为零"**

### 2.8 光进铜退加速

- AI模型规模每**3.5月翻倍**（远超GPU 18月翻倍）
- 光成本做到铜**3倍**即可商用（系统综合成本更低）
- 决定权：OpenAI控需求，TSMC控供给，其他被动接受

### 2.9 NVIDIA双护城河

**CUDA（软件）+ NVLink（硬件）** = 双护城河。软件:硬件=5:1，锁定6份资产。
A股：生态核心伙伴圈内=α，圈外=β。

### 2.10 CPO三质变 vs 铜缆

NVL512 vs NVL72 = **7x GPU密度** / 省电**70%** / GPU直出CPO已成熟
CPO对铜缆=绞杀；对可插拔光模块=替代威胁

### 2.11 ⭐⭐ o1范式：终极判决

**AI交互层级跃迁**：GPT-4=Human-Machine → **o1=Machine-to-Machine** → 集群通信10x+，内存语义集群规模10x+ → 铜缆物理不可能支撑

**Scaling-Law 2.0**：推理算力代偿数据+参数不足 → 推理算力需求爆炸

**产业链重洗牌**：铜缆1年寿命 / 光模块回退Cloud DC / **CPO主宰Scale-out** / **OIO主宰Scale-up(40x光模块)**
**剪刀差**：需求×10 / 供应×2 / Infra×1 → 架构必须大幅变革

### 2.12 CPO经济可行性量化

每支800G光模块基准下CPO节省**$450**（能耗$300+硬件$150）
CPO模组每8通道增加成本<$3,600即不吃亏
中际/新易盛800G≈$750 → CPO等效省**60%**

### 2.13 CPO vs OIO精确定位

## 四、2025-2026年框架补充

### 4.1 Scale-out/Scale-up渗透率量化

**Scale-out**：CPO渗透率上限**75%**（DCN交换机60%+Leaf侧15%），LPO=15%，相干=10%
**Scale-up**：先CPO再OIO，渗透率**100%**（全有或全无）
**Scale-up带宽 = Scale-out的18倍** → 光互联规模 = Scale-out的**6-9倍**
**技术关系**：CPO ≠ 光模块，CPO = 2.5D/3D先进封装；OIO = CPO + DWDM

### 4.2 电互连三道物理死刑

趋肤效应(56GHz铜面爬行仅几微米) + 介质损耗(22dB=信号仅剩2%) + 电磁耦合串扰 → 3.2T短距光模块无解
CPO本质：电信号域距离从0.5-5m缩到**5mm**（3D TSV到0.1mm）
**CPO官方数字**（NV官网）：成本-30% / 能效+3.5x / 可靠性+10x / 部署+1.3x

### 4.3 OCS光交换机

光域内直接交换（无光电转换）= 火车岔道（vs电交换机=到站换乘）
优势：延迟-50% / 能耗-40%/rack / 400×400端口无阻塞 / 软件定义拓扑重构
谷歌TPUv4已用OCS → 2025年200万颗TPU → 2.3万台OCS → **CAGR 50%到2031**
谷歌TPUv8改用CLOS架构（承认OCS搭3D Torus在MoE推理有bug）→对A股光模块+PCB更利好

### 4.4 国内路线：NPO/NPC

**OIF标准**：CPO=ASIC与光引擎共享基板；NPO=独立基板通过HDI PCB连接
国内选择：Scale-up → **NPC**（铜缆+连接器DSP-free）；Scale-out → **NPO**（80分方案）
原因：3D异质封装3-5年难普及 → NPO扬长（光通信）避短（封装）
华为+头部互联网 → 2026H2推NPO，2027规模上量
阿里磐久128每Rack高速Cable价值量100万 → 1万Rack = **100亿市场**

### 4.5 Scale-up硅光时代

2026主题词：Scale-up上光互联。谷歌AEC→LPO(30%→100%)；NV Rubin第二层NPO/CPO
硅光MZ调制器：线性宽/电反射低 → DSP-Free绝配 → 比EML成本功耗低20%，失效率低1/2
CW-DFB四大金刚：源杰/长光华芯/永鼎/仕佳 → 景气+2年

### 4.6 ⭐ NPO大和解

**CPO三版本**：NPO(2D,PCB上socket可插拔,难度1x) / 2.5D CPO(基板上,10x) / 3D CPO(Interposer,100x)
**现实**：英伟达/Intel/博通/阿里/工业富联所有"CPO"实际全是NPO！
Scale-up流量占AI集群**97%** → 可靠性要求≥10x Scale-out → 必须走稳妥路线
NPO = 所有人都有饭吃的现实最优解

### 4.7 ⭐ GTC后终极路线图

**GTC 2026**：黄仁勋官宣Rubin = Hybrid（一层铜+一层光），非纯CPO
**NV Scale-up CPO真实渗透率仅25%**（NVL144内部不需光）
**谷歌Scale-up光互联可逼近100%**（3D Torus + OCS）
**XPO横空出世**：系统级工程优化 > CPO，英伟达有更多选项
**投资策略修正**：从死盯CPO → **NPO/XPO/CPO全局最优**
**市场空间**：Scale-out未来三年年翻倍；NV Scale-up=1-2x；非NV Scale-up=**3-4x**
**需求公式**：光通信需求 = **GPU²**

### 4.8 二季度行动方针

**九字诀**：逻辑硬、预期足、业绩稳
**抱大西瓜**：中下游龙头(中际/新易盛PE10-20x) + 垂直一体化(东山精密) + 定价不充分(华懋科技+3x)
**扔小土豆**：价值量<1-2%的细分器件（千亿市值泡沫）
供应瓶颈：EML磷化铟 / 法拉第旋光片 / mSAP PCB → 持续紧张

## 六、产业知识（按主题整合，精炼版）

### 6.1 技术路线全景：CPO / NPO / XPO / OCS

**CPO代际定义**：
| 代际 | 封装 | 难度 | 时间线 |
|------|------|------|--------|
| CPOv1 = **NPO** | 2D, PCB上socket可插拔 | 1x | **2026-2028（当前量产的全是这个）** |
| CPOv2 | 2.5D, 基板上 | 10x | 2028+ |
| CPOv3 | 3D, Interposer | 100x | 2029+ |
| CPOv4 = OIO | 3D + DWDM | — | 2030+ |

**NPO vs CPO核心辩论**：
- 功耗差异极小（112T交换机NPO仅比CPO高~60W）
- 良率差距巨大：CPO 204T=72光引擎→0.98^72=**23.4%**；XPO 4模块→**92.24%**
- 当前单个CPO光引擎良率不足50%→实际更差
- **CPO增加而非降低TCO**（闷蜜反共识观点）
- **CPO被跳过概率**：Scale-out 98%否了；Scale-up 60%概率被跳过
- **本质博弈**：CPO=英伟达锁生态 vs NPO=云厂商夺回话语权

**XPO（eXtreme Pluggable Optics）**：
- CPC+XPO实现LPO能力(去oDSP)，集成度4倍(12.8T起步)
- 1024GPU超节点**缩小75%**，可延伸至400G/λ(25.6T)
- 制造/供应链/开放性**大幅超越NPO和CPO**

**OCS光交换机**：
- OCS≠交换机！="智能光纤配线架"
- 谷歌OCS：软件动态组装Cube(3D Torus)；需**2.5dB功率预算**→CPO满足不了
- 英伟达OCS：**DragonFly**拓扑，GPU>576→OCS必选项
- OCS需FR模块→需EML→EML不能用于CPO→**闭环：英伟达想搞NPO**
- 市场空间：**80亿美金**（CAGR 50%到2031）

**3.2T NPO三代际**：第一代32×100G(国内) / 第二代16×200G(海外主流) / 第三代8×400G(仅NV)
**⚠️ 国内无企业真正用800G→26年突然用3.2T存疑**

### 6.2 芯片层三国演义

| 材料 | 角色 | 调制带宽 | 400G/λ适用性 |
|------|------|----------|-------------|
| **InP** | 光源核心，无替代 | 一代EML60/二代90/三代110GHz | **第一阶段唯一选择** |
| **SiPhi** | "光的PCB"，集成效率 | 58-60GHz（需oDSP拐棍） | 纯硅光止步200G→必须异质集成 |
| **TFLN** | 性能上限，技术期权 | 180GHz+ | 量产最早2028年（四大工厂无工艺包） |

**奈奎斯特**：1.6T(224Gbps)需**78.4GHz** → 硅光不够 → 第二代EML轻松超过
**异质集成两路线**：有InP=EAM+SiPhi(成熟) / 无InP=TFLN+SiPhi(更远)
**硅光 vs EML = 铁锂 vs 三元锂**：硅光性价比/TTM优，EML适应性/健壮性优
- 硅光渗透率上限大概率止步50-60%
- **oDSP占光模块成本40%+功耗50%** → DSP-Free必然
- **产业节奏**：100G/λ(>20家硅光) → 200G/λ(少数头部, 2026-2028=1.6T黄金期) → 400G/λ(纯硅光止步)
- **炒作公式**："1年炒概念+1年炒业绩" → 2027H1=3.2T概念期，2028=业绩期

## 八、投资方法论（闷蜜投资哲学）

### 8.1 估值层级体系

| 层级 | PE范围 | 代表 |
|------|--------|------|
| 第一层：垂直一体化/IDM | 40-60 PE | 中际旭创、新易盛、东山精密 |
| 第二层：专业核心器件 | 30-50 PE | 天孚、源杰、炬光、光库科技 |

IDM享受更高溢价：研发周期3-6月(vs非IDM 6-12月)，头部毛利率35-40%，净利率30-35%

### 8.2 大西瓜 vs 小土豆

- **大西瓜**：价值量占比>10%的中下游集成商/IDM（10-20x PE，2027动态个位数）
- **小土豆**：价值量<1-2%的细分器件（CW/隔离器/滤镜/分光器/尾纤）→千倍PE泡沫

### 8.3 技术期权思维

科技股价值 = 现有现金流 + **技术期权**（成功→无限，失败→仅限研发成本）
传统PE/PB对未盈利科技股完全失效
押"**黄金赛道里的高潜小苗子**"=期权成本低+潜在收益高

### 8.4 信号验证体系

**模糊→强→硬→确定性**：
1. 老板豪言壮语 → 暴露战略野心
2. 员工持股/股权激励 → 利益绑定
3. 可量化产能/订单数据 → 硬信号
4. 业绩预告/季报 → 确定性信号 → 主升浪开启

**大级别行情 = 公司战略 + 产业周期完美共振**

### 8.5 "公因数"投资策略

不选边CPO vs NPO，押注两边都需要的公因数：华懋(全品类PCBA) / 沪电(交换机PCB) / 东山(InP芯片)

### 8.6 供给侧选股（2026切换）

从"吹订单"→"拼供应"：有硅光和没硅光→估值体系彻底分开
两类享受溢价：产能领先型 + 创造新供给型

### 8.7 二级市场买一级资产

三原则：高景气赛道 + 新资产成色硬 + 早参与早受益
案例：中际(300x)/罗博特科(20x)/东山(2x)/华懋

### 8.8 反共识捡活鱼

守好核心逻辑 + 反共识找安全边际 + 分清"暂时跌"和"彻底坏"

### 8.9 炒作公式与节奏

- **1年概念+1年业绩** → 2027H1=3.2T概念期，2028=业绩期
- **节奏指挥棒**：发布期(关注芯片)→落地期(跟踪组装)→放量期(量价齐升)
- 三者共振=主升浪；分化=根据风险偏好取舍

### 8.10 四族群定位

产业深耕者(最赚) > 交易博弈者 > 信息差猎手 > 情绪跟风者

### 8.11 投资纪律

- 选股：投小/投早/投硬科技
- 交易：做长线(季度审视)/慢行动(宁错过不踩错)/散开花(适度分散)
- VIX：不破前高，胆子大一点
- "风浪越大鱼越贵"
