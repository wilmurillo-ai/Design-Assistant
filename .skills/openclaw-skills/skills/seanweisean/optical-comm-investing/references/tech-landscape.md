# 光通信技术路线全景

## 一、需求驱动：为什么必须全光互联

### 1.1 需求量化公式

**光通信需求 = GPU² × (训练 + 推理 + LPU + CXL)**

- Scale-out（交换机互联，占AI集群流量3%）：1 GPU ≈ 4个光模块
- Scale-up（GPU间互联，占流量97%）：全转光后 1 GPU ≈ **148个光模块**
- Scale-up带宽 = Scale-out的18倍，光互联规模 = Scale-out的 **6-9倍**

### 1.2 三律共舞模型

| 律 | 方向 | 含义 |
|---|------|------|
| Scaling-Law | 需求×10 | AI模型规模每3.5月翻倍，远超GPU 18月翻倍 |
| Moore-Law | 供应×2 | 5→3→2nm功耗仅降15%，改善放缓 |
| Infra-Law | 拖拽×1 | 功率密度↑时capex指数级↑，NVL72=250kW/rack，全球仅6%DC满足 |

**剪刀差**：需求×10 / 供应×2 / Infra×1 → 架构必须大幅变革

### 1.3 o1范式终极判决

GPT-4=Human-Machine → o1=Machine-to-Machine → 集群通信需求10x+
Scaling-Law 2.0：用推理算力代偿数据+参数不足 → 推理算力需求爆炸

### 1.4 电互连三道物理死刑

- **趋肤效应**：56GHz铜面爬行仅几微米
- **介质损耗**：22dB=信号只剩2%
- **电磁耦合串扰**：3.2T短距光模块无解

CPO本质：电信号域距离从0.5-5m缩到**5mm**（3D TSV到0.1mm）→ 三道坎全消

### 1.5 铜缆数学死刑

铜缆PE：2026底终结=**2.4x**；2028底=**4.5x**
NVL72铜缆六大问题：CoWoS-L良率/掩模重流/连接器/132kW过热/UQD漏电/5000根铜缆cartridge良率
"能把未来算死的行业，博弈价值几乎为零"

## 二、模组形态：CPO / NPO / XPO / OCS

### 2.1 CPO代际定义

| 代际 | 封装 | 难度 | 时间线 |
|------|------|------|--------|
| CPOv1 = **NPO** | 2D, PCB上socket可插拔 | 1x | **2026-2028（当前量产的全是这个）** |
| CPOv2 | 2.5D, 基板上 | 10x | 2028+ |
| CPOv3 | 3D, Interposer | 100x | 2029+ |
| CPOv4 = OIO | 3D + DWDM | — | 2030+ |

**关键认知**：英伟达/Intel/博通/阿里/工业富联所有号称的"CPO"实际全是NPO。

### 2.2 NPO vs CPO 核心辩论

- **功耗差异极小**：112T交换机NPO仅比CPO高~60W（系统级可忽略）；NPO 1880W vs CPO 1791W（<5%差异）
- **良率差异巨大**：CPO 512路SerDes配16个光模组，0.95^16=**44%**；NPO单模组95%→整体95%。CPO要达同等良率需单模组99.7%
- **可维护性**：NPO面板热插拔；CPO需断电拔板维修（30分钟中断）
- CPO有两个空气光纤端面（FAU+面板MPO），端面脏污故障率比NPO多一倍
- **CPO增加而非降低TCO**（闷蜜反共识观点）
- **CPO被跳过概率**：Scale-out CPO 98%否了；Scale-up 60%概率被跳过
- **本质博弈**：CPO=英伟达锁生态工具 vs NPO=云厂商夺回话语权
- **判断方法**：光引擎订单给中际/天孚=NPO；给台积电/日月光=真CPO
- **NPO = 半固态电池，CPO = 全固态电池** → NPO先上量，CPO可能被跳过

### 2.3 XPO（eXtreme Pluggable Optics）

- CPC+XPO实现LPO能力(去oDSP)，集成度4倍(12.8T起步)
- 1024GPU超节点**缩小75%**，解决CPO/NPO MTTR痛点
- XPO可延伸至400G/λ(25.6T)：CPC+第二代EML(>90GHz)=线性直驱
- 制造/供应链/开放性**大幅超越NPO和CPO**
- 极大提升PCB/PCBA价值量

### 2.4 OCS光交换机

**本质**：光域内直接交换（无光电转换）="智能光纤配线架"，任务开始后光路静止

**两大阵营**：
- **谷歌OCS**：软件动态组装不同大小计算Cube(3D Torus)；需**2.5dB功率预算**→CPO满足不了；TPUv8改用CLOS架构（承认3D Torus在MoE推理有bug）→对光模块+PCB更利好
- **英伟达OCS**：DragonFly拓扑，GPU>576→OCS必选项，需求量=N倍于谷歌

**OCS需FR模块→需EML→EML不能用于CPO→闭环：英伟达想搞NPO**

**市场空间**：从20亿→40亿→**80亿美金**（CAGR 50%到2031）

**OCS爆发扩大EML需求的三个原因**：
1. 额外3dB插损 → 硅光功率不够（>13dBm出现双光子吸收非线性，>20dBm烧波导）
2. 必须WDM(FR4/FR8) → 硅光热光系数是InP的10倍，75℃全温区波长漂移5.6nm+工艺误差10nm，超出CWDM4信道范围
3. 硅光做CWDM4需4/8个不同波长DFB异构集成，耦合精度<±0.5μm，良率<70%

### 2.5 3.2T NPO三个代际

| 代 | 规格 | 厂商 | 说明 |
|----|------|------|------|
| 第一代 | 32×100G | 华工/光迅→国内互联网 | 4支800G LPO缩小版 |
| 第二代 | 16×200G | 海外主流 | — |
| 第三代 | 8×400G | 只有NV需要 | 年底 |

⚠️ 国内没有企业真正用800G（中际2025/9展会经理原话）→26年突然用3.2T存疑
"讨论国内NPO与海外NPO完全不是一个话语体系，不要混淆"

### 2.6 LPO在全光版图中的位置

| 距离 | 方案 | 成本(vs标准光模块) |
|------|------|-------------------|
| > 2km | 传统光模块/CPO | 100% |
| 10m-2km | CPO/LPO | — |
| < 10m Intra-Rack | LPO (DSP-Free+SiPhi) | 40%（AEC的70%） |
| Fabric GPU-GPU | OIO | — |
| Die-to-Die | oNOC | — |

### 2.7 调制器形态

- **MRR**：3D顶配CPO必须用（绑定NV/台积电）
- **MZM**：NPO/2.5D CPO首选（中际/博通/COHR/可川）
- 两者无优劣，只有场景适配

## 三、光芯片三国演义：InP vs 硅光 vs TFLN

### 3.1 材料特性对比

| 材料 | 角色 | 调制带宽 | 400G/λ适用性 |
|------|------|----------|-------------|
| **InP** | 光源核心，无替代 | EML一代60GHz/二代90GHz/三代110GHz | **第一阶段唯一选择** |
| **SiPhi** | "光的PCB"，集成效率 | 58-70GHz（十年未变，需oDSP拐棍） | 触物理天花板，纯硅光止步200G |
| **TFLN** | 性能上限，技术期权 | 理论300GHz+，实验室260GHz | 量产最早2028年 |

**奈奎斯特准则**：1.6T(224Gbps)需**78.4GHz** → 硅光不够 → 第二代EML轻松超过
**InP电子迁移率=硅的10倍+**，器件带宽可达200GHz

### 3.2 硅光 vs EML 量化对比（查尔斯数据）

| 维度 | 硅光MZM | InP EML |
|------|---------|--------|
| 商用带宽 | 65-70GHz(十年未变) | 90-110GHz |
| TDECQ | 3.0-3.4dB | 1-1.5dB |
| 性能差距 | — | 强1.5-2dB |
| InP晶圆消耗 | 1x（CW-DFB长腔高功率） | **2.3x**（EML短腔但每模组8颗） |
| IDM迭代点 | 2-3个(代工锁死) | 几十个(百花齐放) |
| 400G可行性 | ❌ 需找外援 | ✅ 唯一可靠平台 |

**硅光 vs EML = 铁锂 vs 三元锂**：
- 硅光：性价比/TTM优势，扩产容易，中际2026年75%、新易盛50%+
- EML：适应性/余量/健壮性优，CSPs愿溢价100-200$，扩产难格局稳定
- EML需求无上限，只要能供应；谷歌OCS需3dB→只能EML
- 硅光渗透率上限大概率止步50-60%

**硅光精简器件**：不需TEC + AWG自集成 + 隔离器减半 + 分立无源减少50-70%
EML 800G DR8需8颗独立EML vs 硅光仅需2-4颗CW → 封装良率差30%
oDSP占光模块成本40%+功耗50% → DSP-Free必然

### 3.3 异质集成两条路线

- 有InP技术 → EAM+SiPhi（成熟路线）
- 无InP技术 → TFLN+SiPhi（更远路线）

### 3.4 TFLN现实

- 卡点不在TFLN芯片而在与硅光平台异质集成，没有主流厂家搞定
- 四大代工厂（Tower/GF/ST/台积电）全未发布TFLN-on-SiPhi工艺包
- 400G时代TFLN调制带宽只有70GHz（商用水平远低于理论）→"只能做科研发论文"
- "做薄膜铌酸锂的公司都靠政府补贴在延续"
- **但**：Coherent采用TFLN的CPO方案斩获超大订单；索尔思+易缆微发布400G TFLN异质集成硅光芯片方案

### 3.5 400G五大InP方案（查尔斯梳理）

1. **标准EML**：索尔思110GHz
2. **D-EML**（差分EML）：COHR/BRCM/LITE，EAM上下端面分别施加正负电压实现2倍调制压差
3. **InP PIC**：索尔思（DFB+EAM分离集成）
4. **InP+SiPhi混合**：Openlight
5. **InP+SiN**：索尔思为NV定制（全单片高密度，不用外部透镜/隔离器）

→ 五大方案全部以InP为核心，东山精密占3/5条路线

### 3.6 产业节奏

| 阶段 | 时间 | 特征 |
|------|------|------|
| 100G/λ | 当前 | >20家硅光供应商 |
| 200G/λ | 2026-2028 | 少数头部，**1.6T黄金期** |
| 400G/λ | 2026底小批量→2027规模 | 纯硅光止步→必须异质集成 |

**炒作公式**："1年炒概念+1年炒业绩" → 2027H1=3.2T概念期，2028=业绩期
**Lumentum OFC判断**：400G时代EML成事实标准，硅光带宽满足不了400G/Lane

### 3.7 台积电硅光突破

一年内攻克：4行并行光纤垂直耦合器 / 氮化硅波导 / 嵌入式硅微透镜(FAU可插拔=类USB) / 良率97% / 光子计算DOC(512×512矩阵比GPU提高20倍)
**硅光只需65-45nm老工艺** → 成本=规模的负函数

### 3.8 最新渗透率数据

- 高盛4/17：硅光渗透率2024Q1=6%→2028Q4=45%
- 光模块毛利率→48-55%
- 800G硅光占比>50%，1.6T占比70-80%
- 硅光迭代周期从3-4年→2-3年（AI加速）

## 七、DSP与形态组合关系

- DSP三种模式：Full DSP（收发都有）、TX DSP（仅发送）、Linear Optics（无DSP）
- 形态与DSP正交：XPO也可带DSP（XPO-DPO/XPO-LRO），NPO也可带DSP（NPO-LRO）
- 带DSP模块仍是当前主流（占比>90%），DPO和LRO性能绝对优势
- **oDSP占光模块成本40%+功耗50%** → DSP-Free是长期趋势
- NPO/XPO/CPO都去掉oDSP，但LDD+TIA+LA不可或缺且强化 → **穿透所有技术路线之争**
