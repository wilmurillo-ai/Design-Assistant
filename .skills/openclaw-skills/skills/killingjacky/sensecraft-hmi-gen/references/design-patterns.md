# E-Ink Design Patterns - 墨水屏的艺术哲学

> 墨水屏不是缩小的网页，而是数字时代的纸张。

## 核心理念

### 墨水屏的本质
墨水屏是**静态媒介**，不是动态屏幕。它的美学更接近：
- 印刷海报
- 书籍内页
- 报纸版面
- 日本浮世绘

**设计时要问自己：** 如果这是一张要挂在墙上的海报，我会怎么设计？

---

## 三大设计流派

### 1. 文学派 (Literary) - 呼吸的文字

**哲学：** 文字需要呼吸空间，就像音乐需要休止符。

**视觉语言：**
- 衬线字体（Georgia, Merriweather）- 带有人文温度
- 80%+ 留白 - 让眼睛休息
- 居中对齐 - 仪式感
- 行高 1.8-2.0 - 优雅的节奏

**适合内容：** 名言、诗歌、冥想文本、每日一词

**灵感来源：**
- Kindle 屏保
- 日本禅宗书法
- 瑞士平面设计

**案例：** Naval Quotes, Stoic Quotes

---

### 2. 信息派 (Dashboard) - 秩序的美学

**哲学：** 信息密度不等于混乱，网格是秩序的基础。

**视觉语言：**
- 无衬线字体（Arial, Helvetica）- 清晰直接
- 网格系统 - 2x2, 3x2, 4x3
- 粗体数字 - 视觉锚点
- 反白区域 - 层次分明
- 图标 + 文字 - 快速识别

**适合内容：** 天气、日历、股票、赛事数据

**灵感来源：**
- 瑞士国际主义风格
- 机场信息牌
- 仪表盘设计

**案例：** Weather Dashboard, Formula 1

---

### 3. 艺术派 (Artistic) - 视觉的诗

**哲学：** 一图胜千言，让图像说话。

**视觉语言：**
- 大图占 60-80% - 主角明确
- 高对比度 - 黑白分明
- 极简文字 - 点到为止
- 装饰性元素 - 画龙点睛

**适合内容：** 漫画、艺术作品、地图、摄影

**灵感来源：**
- 日本浮世绘
- 包豪斯海报
- 极简主义艺术

**案例：** XKCD, Art of the Day

---

## 墨水屏的视觉语法

### 1. 对比度 - 黑白的力量

墨水屏只有黑白，这是限制也是优势。

**纯黑 #000 + 纯白 #FFF**
- 不要用灰色（#888）- 墨水屏会糊
- 用点阵纹理代替灰色
- 用留白代替淡色背景

**对比度层次：**
```
█████████  100% 黑 - 强调、标题
▓▓▓▓▓▓▓▓▓   75% 点阵 - 背景、装饰
░░░░░░░░░   25% 点阵 - 微妙分隔
         0% 白 - 留白、呼吸
```

---

### 2. 字体 - 可读性至上

**衬线 vs 无衬线：**
- **衬线（Georgia）** - 文学、优雅、长文本
- **无衬线（Arial）** - 信息、清晰、数据

**字号黄金比例：**
```
标题 : 正文 : 小字 = 3 : 2 : 1
例如：72px : 48px : 24px
```

**最小字号：24px**
- 墨水屏刷新慢，小字难读
- 宁可少说，不要说小

---

### 3. 留白 - 负空间的艺术

**留白不是浪费，是设计的一部分。**

**留白比例指南：**
- 文学派：80% 留白
- 信息派：40% 留白
- 艺术派：30% 留白

**Padding 原则：**
- 最小 30px
- 重要内容 50px+
- 超大标题周围 80px+

**日本美学：** 间（Ma）- 空间本身就是内容

---

### 4. 网格 - 隐形的骨架

**为什么需要网格？**
- 创造秩序
- 对齐元素
- 视觉平衡

**常用网格：**
```
2x2 - 天气四要素
3x2 - 周历
4x3 - 月历
1:2 - 黄金分割
```

**对齐原则：**
- 左对齐 - 自然、易读
- 居中对齐 - 正式、对称
- 右对齐 - 少用，特殊强调

## 高级技巧

### 1. 点阵纹理 (Dithering) - 灰度的替代

墨水屏没有真正的灰色，用点阵模拟：

```css
/* 25% 灰 */
background: repeating-linear-gradient(
  0deg, #000 0px, #000 1px, 
  transparent 1px, transparent 4px
);

/* 50% 灰 */
background: repeating-linear-gradient(
  45deg, #000 0px, #000 1px, 
  transparent 1px, transparent 2px
);
```

**用途：**
- 背景装饰
- 区域分隔
- 增加质感

---

### 2. 反白设计 (Inverted) - 视觉锚点

黑底白字，创造强烈对比：

```css
.inverted {
  background: #000;
  color: #FFF;
  padding: 20px;
}
```

**何时使用：**
- 侧边栏
- 标题栏
- 需要强调的区域
- 创造视觉层次

**注意：** 不要大面积使用，会刺眼

---

### 3. 卡片式布局 - 内容的容器

白色卡片 + 装饰背景 = 层次感

```css
.card {
  background: #FFF;
  padding: 50px;
  border: 3px solid #000;
  /* 或用阴影 */
  box-shadow: 8px 8px 0 #000;
}
```

**灵感：** 日本名片、瑞士海报

---

### 4. 装饰元素 - 画龙点睛

**极简装饰：**
- 细线分隔 `border-top: 1px solid #000`
- 小图标（24-32px）
- 引号 `" "`
- 日期标签

**原则：** 少即是多，一个页面最多 2-3 个装饰元素

---

## 排版的节奏

### 视觉层次（3 层原则）

```
第一层：主标题 - 72px 粗体
第二层：副标题 - 48px 常规
第三层：正文 - 32px 常规
```

**不要超过 3 层** - 层次太多 = 没有层次

---

### 行高与间距

**行高（Line Height）：**
- 标题：1.2-1.3
- 正文：1.6-2.0
- 长文本：1.8+

**段落间距：**
- 最小 20px
- 推荐 30-40px

**原则：** 墨水屏刷新慢，要让眼睛容易找到下一行

---

## 色彩心理学（黑白版）

### 黑色的力量
- **大面积黑** - 严肃、权威、神秘
- **细黑线** - 精致、优雅
- **粗黑字** - 强调、重要

### 白色的智慧
- **大面积白** - 纯净、呼吸、禅意
- **白色卡片** - 内容容器
- **反白文字** - 特殊强调

## 实战工作流

### 1. 确定内容类型
- 文字为主？→ 文学派
- 数据为主？→ 信息派
- 图像为主？→ 艺术派

### 2. 选择布局
参考 `layouts.md` 的 10 种布局

### 3. 设定视觉层次
- 什么最重要？→ 最大
- 什么次要？→ 中等
- 什么辅助？→ 最小

### 4. 调整留白
- 太挤？→ 增加 padding
- 太空？→ 增大字号

### 5. 测试对比度
在真实墨水屏上预览，不要只看电脑屏幕

---

## 常见错误

### ❌ 不要做
1. **字太小** - 最小 24px
2. **用灰色** - 墨水屏会糊
3. **太多层次** - 最多 3 层
4. **填满屏幕** - 留白是设计的一部分
5. **复杂渐变** - 墨水屏不支持
6. **动画效果** - 刷新太慢

### ✅ 要做
1. **大字体** - 32px 起步
2. **纯黑白** - #000 和 #FFF
3. **简单层次** - 3 层足够
4. **大胆留白** - 50% 以上
5. **高对比度** - 黑白分明
6. **静态设计** - 一屏一内容

## 最后的建议

**记住：墨水屏是纸，不是屏幕。**

设计时想象你在为一本书设计内页，为一张海报设计版面。

**少即是多。** 删除一切不必要的元素，直到不能再删为止。

**测试，测试，测试。** 在真实设备上看效果，不要只看设计稿。
