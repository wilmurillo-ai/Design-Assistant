## 🏗️ 系统架构

### 三层数据管理
1. **HOT层** - 实时信息（最近1天，最高重要性）
2. **WARM层** - 近期信息（最近7天，中等重要性）
3. **COLD层** - 历史信息（最近30天，参考重要性）

### 触发机制
- **内存触发**：Token使用率 > 70% 或消息数 > 50
- **时间触发**：每小时自动检查
- **事件触发**：会话开始/结束、任务完成
- **手动触发**：用户命令触发

## 🚀 快速使用

### 安装
```bash
# 复制到技能目录
cp -r context-compactor ~/.openclaw/workspace/skills/

# 安装依赖
pip install -r requirements.txt

# 测试技能
./test_skill.sh
```

### 启动
```bash
# 启动完整系统
./start_system.sh

# 检查状态
./check_status.sh

# 查看报告
python3 integration.py --report
```

## 🔧 集成到OpenClaw

### 1. 添加到技能列表
技能会自动出现在OpenClaw的技能列表中。

### 2. 配置心跳检查
编辑 `HEARTBEAT.md`：
```markdown
# 上下文压缩检查
- 检查压缩系统状态
- 查看最近压缩报告
- 优化配置参数
```

### 3. 设置定时任务
```bash
# 每小时检查一次
0 * * * * cd /path/to/skills/context-compactor && python3 integration.py --check
```
