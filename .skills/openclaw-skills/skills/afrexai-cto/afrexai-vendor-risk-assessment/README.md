# Vendor Risk Assessment

**By AfrexAI** | [clawhub.com](https://clawhub.com)

Assess third-party vendor risk for AI and SaaS products. Evaluates security posture, data handling, compliance, financial stability, and operational resilience.

## What It Does

Scores vendors across critical risk dimensions:

- **Security posture** — encryption, access controls, pen testing
- **Data handling** — retention, processing, cross-border transfers
- **Compliance** — SOC2, ISO 27001, GDPR, HIPAA
- **Financial stability** — funding, revenue, customer concentration
- **Operational resilience** — uptime, DR, support SLAs

## Output

- Scored risk report (Low / Medium / High / Critical)
- Dimension breakdown with evidence requirements
- Mitigation recommendations per risk area
- Vendor comparison matrix (multi-vendor evaluation)
- Review schedule and escalation triggers

## Usage

```
Assess vendor risk for:
Vendor: CloudAI Corp — AI-powered document processing
Use Case: Processing customer contracts (PII, financial data)
Our Industry: Financial services, SOC2 certified
Data Volume: 50K documents/month
Contract Value: $120K/year
```
