# RAG Production Engineering

The complete methodology for building, optimizing, and operating Retrieval-Augmented Generation systems — from architecture decisions through production monitoring.

## Install

```
clawhub install afrexai-rag-engineering
```

## What This Skill Does

Most RAG skills on ClawHub are API wrappers for specific tools. This skill teaches the **methodology** — how to design, build, evaluate, and operate RAG systems that actually work in production.

**12 phases covered:**
1. Architecture decision (Basic vs Advanced vs Agentic vs Graph RAG)
2. Document processing & chunking strategies
3. Embedding model selection with cost/quality tradeoffs
4. Vector database selection & indexing
5. Retrieval engineering (hybrid search, reranking, query transformation)
6. Context assembly & generation prompts
7. Evaluation framework (metrics, test datasets, tools)
8. Production operations (monitoring, alerting, freshness)
9. Advanced patterns (parent-child, GraphRAG, CRAG, multi-modal)
10. Security & access control
11. Cost optimization
12. Complete architecture patterns for 4 common use cases

**Includes:** Architecture brief YAML, chunking decision guide, embedding model comparison, vector DB selection matrix, retrieval pipeline design, evaluation dataset template, monitoring dashboard, cost calculator, 4 complete production patterns, 100-point quality rubric.
