# Learning & Skill Acquisition Engine

Learn anything faster. Retain it longer. Apply it for real.

An evidence-based learning system built on cognitive science — spaced repetition, active recall, deliberate practice, and project-based learning. Not a course. A methodology.

## What This Skill Does

Turns your AI agent into a personal learning coach that:

- **Audits your current level** — honest assessment with scoring
- **Decomposes any skill** into sub-skills with dependency mapping and 80/20 filtering
- **Curates resources** using the 3-Source Rule (primary text + alternative + practice ground)
- **Runs learning sessions** using the ARPD Cycle (Absorb → Retrieve → Practice → Debrief)
- **Manages spaced repetition** — never forget what you learned
- **Designs projects** that build real competence, not tutorial completion certificates
- **Tracks progress** with weekly reviews, milestone checks, and evidence collection
- **Handles motivation dips** with a specific recovery protocol

## Quick Start

Tell your agent:
