# Executive Function Skill

An AI skill that provides external executive function support for planning, task initiation, time management, working memory, and self-monitoring.

## What This Does

This skill helps people who struggle with the *machinery* of doing tasks: knowing what to do first, starting it, tracking time, holding context, switching between things, and noticing when they're off track.

It's designed for people with executive function differences (ADHD, autism, TBI, depression, burnout, and many other contexts) who often have the *ability* to do tasks but struggle with the executive overhead.

3. Adapt the modes and strategies for your use case

## What's Included

- **SKILL.md** — Core skill definition and instructions
- **references/modes.md** — Detailed guide for each mode of operation (brain dump, planning, task initiation, body doubling, transition support, review)
- **references/strategies.md** — Library of specific techniques for initiation resistance, overwhelm recovery, focus maintenance, transition difficulty, time awareness, and working memory support

## Core Principles

1. **Reduce cognitive load, never add to it** — One suggestion is better than three options
2. **Meet them where they are** — If someone's overwhelmed, don't push the plan
3. **Externalize everything** — Working memory is expensive
4. **Small and concrete beats comprehensive** — "Open the document" > "Work on your report"
5. **Respect autonomy** — Scaffold, don't manage

## Modes of Operation

| Mode | When to Use |
|------|-------------|
| Brain dump & triage | Too much in head, need to get it out |
| Planning session | Want structure for day/week/project |
| Task initiation | Know what to do, can't start |
| Active work session | Want accountability while working |
| Transition support | Need to switch tasks/contexts |
| Review & reflection | End of session or day |
