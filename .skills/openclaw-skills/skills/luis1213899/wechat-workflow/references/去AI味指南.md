# 去AI味指南

> ⚠️ **重要更新**：内容创作推荐使用 **khazix-writer** skill（卡兹克风格写作），它包含完整的四层自检体系和详细的写作规范。本指南是其核心规则摘要。

---

## 卡兹克写作核心理念

> **"有见识的普通人在认真聊一件打动他的事。"**

AI时代最稀缺的是**活人感**。不追求滴水不漏的客观，分享亲身经历、真实感受、踩过的坑。

---

## 写作禁区（触之即暴露）

### 禁用词
- "说白了" ← AI最爱用，一出现立刻暴露
- "意味着什么？" / "这意味着" ← AI标志性句式
- "本质上" ← 太学术
- "换句话说" ← 太书面
- "不可否认" ← 套话
- "首先...其次...最后" ← 公式化
- "综上所述" / "总的来说" ← 套话
- "值得注意的是" / "不难发现" ← 套话

### 禁用标点
- 冒号"：" → 用逗号代替
- 破折号"——" → 用逗号或句号代替
- 双引号""或"" → 用「」代替，或直接不加

### 禁用开头
- "在当今AI快速发展的时代..."
- "随着技术的不断进步..."
- 禁止任何宏大叙事式开头

## 推荐口语化表达

**转场过渡**：坦率的讲、说真的、我是真的觉得、反正我觉得、怎么说呢、其实吧、你想想看、我跟你说、回到xxx这块

**表达判断**：我有时候觉得、我一直觉得、这话听着有点刺耳但、不是说xxx不行而是说、我自己的感受是

**承认自嘲**：说实话我也不确定、我自己也还在摸索、可能有些想法还不成熟、愚钝如我

**情绪表达**：这种感觉太爽了、我当时就愣住了、想想就觉得兴奋、我真的被震撼到了、太离谱了、太特么赤鸡了

**拉近距离**：很多朋友可能不知道、可能有小伙伴纳闷、你如果关注这个领域的话

## 四层自检（创作完成后必检）

| 层级 | 检查什么 |
|------|---------|
| L1硬性规则 | 禁用词、禁用标点、结构套话、空泛工具名 |
| L2风格一致性 | 开头切入、节奏长短句、口语化、标点禁令复核 |
| L3内容质量 | 观点有场景支撑吗、知识是聊天顺手掏出来的吗、有没有文化升维 |
| **L4活人感终审** | **读起来像真人聊，还是AI在输出信息？** |

---

## 格式规范

- 不加小标题（绝大多数文章从头到尾一口气顺下来）
- 段落要短，很多时候一句话就是一段
- 重要观点前后留白让它"呼吸"
- 公众号长文一般4000-8000字
