# CSI 2000 股票分析器

专业的A股投资分析工具，整合新闻舆情分析、财务报表分析和技术指标分析，为中证2000成分股及全市场股票提供深度分析。

## ✨ 功能特性

### 📰 新闻舆情分析
- ✅ 检测是否存在机构明确做空报告
- ✅ 识别利好/利空消息（成本优化、研发、销售、合同、产品质量、法律诉讼等）
- ✅ 提取管理层对未来的盈利展望
- ✅ 监控机构和管理层增减持情况
- ✅ 分析管理层稳定性（高层离职、空降高管等）

### 💰 财务报表分析
- ✅ 管理层稳定性及未来发展评价
- ✅ 公司战略方向变化及市场评级
- ✅ 行业地位和产品护城河变化分析
- ✅ 营收季度/年度增长及分析师预测
- ✅ 净利润季度/年度增长及分析师预测
- ✅ ROE稳定性分析
- ✅ 扣非ROE表现分析
- ✅ 偿债能力及有息负债分析
- ✅ 净现比（现金流质量）分析
- ✅ 长期投资和研发支出分析
- ✅ PEG估值分析
- ✅ 自由现金流分析

### 📊 技术指标分析
- ✅ MACD指标分析（金叉/死叉、趋势判断）
- ✅ KDJ指标分析（超买/超卖、买卖信号）
- ✅ RSI指标分析（强弱判断、背离识别）
- ✅ EMA均线分析（多头/空头排列、趋势判断）
- ✅ 综合买卖点建议

## 🚀 快速开始

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 配置API密钥
在环境变量中设置Tavily API密钥（用于新闻搜索）：
```bash
export TAVILY_API_KEY="your_api_key_here"
```

或者在 `.env` 文件中配置：
```env
TAVILY_API_KEY=your_api_key_here
```

### 3. 运行分析
```bash
# 基本用法
python scripts/analyze_stock.py 股票代码或名称

# 指定分析天数
python scripts/analyze_stock.py 000001 --days 180

# 不包含新闻分析
python scripts/analyze_stock.py 贵州茅台 --no-news

# 保存报告到指定文件
python scripts/analyze_stock.py 比亚迪 -o 比亚迪分析报告.txt
```

## ⚙️ 配置说明

### 数据源配置
默认使用模拟数据进行演示。要接入真实数据源，请在 `config.yaml` 中配置：
- `market_data`: 行情数据源（支持tushare、akshare、baostock等）
- `news_source`: 新闻数据源（默认使用Tavily）
- `financial_source`: 财务数据源

### API配置
需要配置以下API密钥：
- `TAVILY_API_KEY`: 用于新闻搜索，可在 https://tavily.com 申请
- 可选：`TUSHARE_TOKEN`、`AKSHARE_KEY` 等

## 📝 注意事项

1. **免责声明**：本工具提供的所有分析结果仅供参考，不构成任何投资建议。投资者应独立判断，自负盈亏。
2. **数据延迟**：公开数据可能存在延迟，实际交易请以交易所实时数据为准。
3. **API限制**：使用第三方API时请注意调用频率限制，避免被封禁。
4. **缓存机制**：工具会自动缓存已获取的数据，减少重复API调用，提高分析速度。

## 🔧 功能扩展

本工具支持灵活扩展：

- 新增分析指标：在各分析模块中添加新的分析逻辑
- 新增输出格式：在 `generate_analysis_report` 方法中添加新的输出格式
- 接入交易接口：可以扩展对接实盘交易接口，实现自动化交易
