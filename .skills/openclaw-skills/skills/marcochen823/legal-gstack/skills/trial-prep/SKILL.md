# 庭审策略顾问 `/trial-prep`

## 功能
1. **庭审提纲** - 庭审流程、各方观点
2. **发问清单** - 对对方/证人/鉴定人的发问
3. **辩论要点** - 核心争议焦点、辩论意见
4. **庭审模拟** - 预判对方观点、准备应对

## 用法
```
/trial-prep [案件名]
/trial-prep [案件名] --role 原告/被告
```

## 输出位置
- `~/Documents/01_案件管理/进行中/[案件名]/庭审准备.md`
