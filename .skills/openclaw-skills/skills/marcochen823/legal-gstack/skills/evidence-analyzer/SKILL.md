# 证据分析师 `/evidence-analyzer`

## 功能
1. **证据三性分析** - 真实性、合法性、关联性
2. **证据清单生成** - 标准化证据目录
3. **质证意见** - 对对方证据的质证意见
4. **时间线梳理** - 案件事实时间线

## 用法
```
/evidence-analyzer [案件名]
/evidence-analyzer [案件名] --side 原告/被告
```

## 输出位置
- `~/Documents/01_案件管理/进行中/[案件名]/证据分析报告.md`
