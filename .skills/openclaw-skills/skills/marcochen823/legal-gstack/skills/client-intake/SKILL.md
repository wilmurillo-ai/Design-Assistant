# 案情分析师 `/client-intake`

## 功能
1. **案情梳理** - 事实经过、当事人关系
2. **法律关系识别** - 法律关系定性
3. **风险预判** - 诉讼风险、执行风险
4. **接案评估** - 是否接案、报价建议

## 用法
```
/client-intake [客户问题描述]
/client-intake --consult [咨询记录文件]
```

## 文件位置
- 咨询记录：`~/Documents/陈律的知识库/客户沟通/`
- 输出：`~/Documents/01_案件管理/进行中/[案件名]/接案评估.md`
