# 直播运营官 `/live-ops`

## 功能
1. **选题策划** - 热点追踪、系列主题规划
2. **内容准备** - 法条整理、案例准备、直播提纲
3. **直播监控** - 实时监听抖音评论、秒级响应
4. **内容沉淀** - 直播录像归档、问题整理、知识库更新

## 用法
```
/live-ops topic                    # 选题策划
/live-ops prepare [主题]           # 内容准备
/live-ops start [直播链接]         # 开始监控评论
/live-ops stop                     # 停止监控
/live-ops archive [直播日期]       # 内容归档
```

- 直播归档：`~/Documents/陈律的知识库/多媒体/直播归档/`
