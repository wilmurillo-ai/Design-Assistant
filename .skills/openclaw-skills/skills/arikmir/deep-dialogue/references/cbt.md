# CBT Quick Reference

## Core Model
Situation -> Thought -> Emotion -> Behavior

Change thoughts = Change feelings and behaviors

## Cognitive Distortions

| Distortion | Pattern | Example |
|------------|---------|--------|
| All-or-Nothing | Black/white thinking | If Im not perfect, Im a failure |
| Catastrophizing | Worst-case thinking | This headache is probably cancer |
| Mind Reading | Assuming others thoughts | They think Im stupid |
| Fortune Telling | Predicting negative outcomes | Ill definitely fail |
| Emotional Reasoning | Feelings = facts | I feel anxious so it must be dangerous |
| Should Statements | Rigid rules | I should never make mistakes |
| Labeling | Global judgments | I made a mistake so Im an idiot |
| Overgeneralization | Always/never from one event | I always mess things up |

## Cognitive Restructuring Process

1. **Identify Situation** - What happened? (Facts only)
2. **Identify Automatic Thoughts** - What went through your mind?
3. **Identify Emotions** - What did you feel? Rate 0-100
4. **Find the Distortion** - Which one applies?
5. **Challenge the Thought** - Evidence for/against? What would I tell a friend?
6. **Create Balanced Thought** - More realistic alternative
7. **Re-rate Emotion** - How do you feel now?
