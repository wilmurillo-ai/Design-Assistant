## playwright-cli原始内容模式

### 京东评论提取
- **目标元素**: `.comment-content`, `.user-name`, `.comment-time`
- **数据类型**: 用户评论原文、评分、购买时间
- **注意事项**: 动态加载内容需等待完全渲染

### 1688价格信息提取
- **目标元素**: `.price`, `.supplier`, `.min-order`
- **数据类型**: 批发价格、供应商名称、起订量
- **注意事项**: 价格可能需要登录后才能看到
