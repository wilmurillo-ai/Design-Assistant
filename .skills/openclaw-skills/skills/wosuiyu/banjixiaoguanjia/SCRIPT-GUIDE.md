# 班级小管家作业批改 - 脚本使用手册

> **⚠️ 重要提示**: 本目录下脚本众多，请严格按照本文档推荐的脚本执行，不要使用其他未标注的脚本，以免出现意外错误。

## 🎯 完整作业流程

### 执行规则

**当用户说"检查某节课作业"时，自动完整执行以下流程，中途不需要确认：**

1. **截图收集** - 运行 `capture-new-browser.js` 收集所有学生作业截图
2. **下载原图** - 运行 `download-homework.js` 下载学生作业原图（自动跳过已点评学生）
3. **AI批改分析** - 运行 `analyze-homework.js` 对每个学生的作业进行逐题批改

**注意：** 如果学生已点评，下载脚本会自动跳过，此时AI批改阶段会提示无原图可分析。

---

### 🆕 新版推荐: 一键截图（自动浏览器管理）

**✅ 强烈推荐使用脚本**: `capture-new-browser.js`

**功能特点**:
- ✅ **自动复用浏览器**: 检测已打开的班级小管家页面，有则复用，无则启动新浏览器
- ✅ **动态学生列表**: 自动获取学生列表，不固定学生名字，支持任意班级
- ✅ **智能截图**: 列表第一个学生默认可见直接截图，其他学生滚动查找定位
- ✅ **窗口大小**: 1280x1000
- ✅ **使用本地 Chrome**: 不是 Chrome for Testing

**使用方法**:
```bash
node capture-new-browser.js "课程名称"
```

**示例**:
```bash
node capture-new-browser.js "二下(第38节)"
```

**输出位置**: `桌面/课程名称/`

**浏览器管理逻辑**:
1. 检查是否有已打开的 Chrome 调试端口
2. 如果有，查找是否有班级小管家的页面
   - 有 → 直接使用，导航到首页
   - 没有 → 关闭连接，启动新浏览器
3. 如果没有调试端口 → 启动新浏览器（开启调试端口 9222）

### 调试脚本: 单独进入课程
**使用脚本**: `enter-course.js`

```bash
node enter-course.js "课程名称"
```

**使用场景** (非必须):
- `capture-auto.js` 执行失败，需要人工调试
- 只想查看课程页面，不截图分析
- 测试页面元素定位

**注意**: 正常流程不需要运行此脚本，`capture-auto.js` 已包含进入课程功能

---

### 阶段2: 下载原图
**使用脚本**: `download-homework.js` (v8)

```bash
node download-homework.js "课程名称"
```

**功能**:
- 自动下载所有学生的作业
- 动态获取学生列表并切换
- 自动核对下载数量
- **自动跳过已点评学生**（检测"已点评"标签）

**实现原理**:
1. 动态获取左侧学生列表（过滤非学生元素）
2. 检查学生是否有"已点评"标签，有则跳过
3. 通过索引点击切换学生
4. 验证切换结果（检查页面顶部元素[7]）
5. 逐个下载所有学生作业

**⚠️ 重要**: 已点评的学生会自动跳过，不会重复下载

---

### 阶段3: AI批改分析
**使用脚本**: `analyze-homework.js`

```bash
node -e "
const { HomeworkAnalyzer } = require('./analyze-homework');
const analyzer = new HomeworkAnalyzer();
analyzer.analyze('课程名称', '学生姓名', '课内|奥数');
"
```

**功能**:
- 批量分析作业图片
- 逐题批改（正确/错误）
- 生成TXT报告
- 生成Word文档（手机阅读优化版）

**Word格式特点**:
- 窄边距（0.5英寸），适合手机屏幕
- 大字体（正文14pt，标题16-20pt）
- 📚表情符号 + ▶章节标记
- 正确显示绿色 ✅，错误显示红色 ❌
- 流式布局，无表格

**输出位置**: `C:\Users\Administrator\Desktop\课程名称\`

## 🔧 故障处理

### 切换学生失败
- `download-homework.js` v8 已使用动态获取学生列表的方式
- 如仍失败，检查页面是否已登录、Chrome调试端口是否正常

### AI分析超时
- 自动切换到逐一分析模式
- 耐心等待完成

### 学生列表获取异常
- 检查班级小管家页面是否加载完成
- 尝试刷新页面后重新运行脚本
