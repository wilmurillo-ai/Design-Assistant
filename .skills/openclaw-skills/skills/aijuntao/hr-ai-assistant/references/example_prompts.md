# HRrule AI 示例提示词

本文档提供了各种 HR 场景的示例提示词,可以直接使用或根据需要进行修改。

## 使用方法

将以下提示词复制到 API 调用的 `content` 参数中,或直接在 HRrule AI 平台上使用。

```
帮我设计一份HR风险自测表。

评估维度:

1. 劳动合同管理
   □ 是否与所有员工签订劳动合同
   □ 合同条款是否完整合法
   □ 合同到期是否及时续签
   □ 劳动合同备案是否规范

2. 薪酬福利合规
   □ 薪酬是否按时足额发放
   □ 加班费计算是否符合规定
   □ 社保公积金是否按时缴纳
   □ 个税是否正确代扣代缴

3. 考勤休假管理
   □ 考勤记录是否完整准确
   □ 请假审批流程是否规范
   □ 年假计算是否符合规定
   □ 法定节假日是否合规安排

4. 员工关系管理
   □ 是否建立员工投诉渠道
   □ 劳动纠纷处理是否及时
   □ 员工沟通是否顺畅
   □ 员工满意度是否定期调研

5. 招聘合规
   □ 招聘过程是否公平公正
   □ 背景调查是否规范
   □ 试用期管理是否符合规定
   □ 入职体检是否按规定进行

6. 培训发展
   □ 培训计划是否制定
   □ 培训记录是否完整
   □ 培训费用是否合理
   □ 员工发展路径是否清晰

7. 安全健康
   □ 劳动保护措施是否到位
   □ 职业健康检查是否定期
   □ 工伤处理流程是否规范
   □ 安全培训是否定期开展

8. 信息保密
   □ 保密制度是否建立
   □ 商业秘密保护措施是否完善
   □ 信息安全管理是否到位
   □ 员工保密协议是否签订

评分标准:
- 是: 2分
```

## 提示词优化建议

### 1. 提供详细信息

好的提示词应该包含:
- 公司规模和行业
- 具体需求和要求
- 目标受众
- 时间周期
- 特殊要求

### 2. 结构化表达

使用清晰的结构:
```
1. 背景
2. 需求
3. 要求
4. 约束
5. 输出格式
```

### 3. 明确预期

说明您期望的输出:
- 文档类型
- 内容详略
- 格式要求
- 语言风格

### 4. 提供示例

如果可能,提供示例:
- 示例结构
- 示例内容
- 参考模板

### 5. 迭代优化

根据生成结果持续优化提示词:
- 补充缺失信息
- 调整表达方式
- 明确具体要求
