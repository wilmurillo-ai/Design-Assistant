# Example Prompts

## 1. Support writing
Compare GPT, Claude, and Gemini on this customer support task: write a calm, concise reply to a user asking for a refund after being double charged.

## 2. Coding
Run this task across three models and compare production readiness: write a Python function that retries an HTTP request with exponential backoff and proper error handling.

## 3. SEO/content
Compare a low-cost model and a premium model on this task: create a blog outline for "OpenAI-compatible API gateway" aimed at developers evaluating alternatives.

## 4. Decision memo
Compare two budget models and one premium model for summarizing a 2-page product requirement draft into an executive recommendation.

## Recommended instruction wrapper
For each model, ask for the output independently. Then produce a final report using the comparison rubric and recommend the best model for specific use cases.
