# 高效学习方法论

> 学习不在于花了多少时间，而在于用对了多少方法。

---

## 一、四大学习法

### 1. 费曼学习法（Feynman Technique）

**核心理念：** 如果你不能简单地解释一件事，说明你还没有真正理解它。

**四步走：**
1. **选择概念** → 写下要学的东西
2. **教给别人** → 用最简单的话解释，假装对方是10岁小孩
3. **找到差距** → 讲不清楚的地方就是你没懂的地方
4. **简化和类比** → 用生活中的事物做类比，重新组织语言

**使用命令：** `learn.sh feynman "概念"`

> 💡 费曼的精髓：教是最好的学。讲不出来 = 没理解。

---

### 2. 间隔重复法（Spaced Repetition）

**核心理念：** 在即将遗忘的时候复习，记忆效果最好。

**艾宾浩斯复习时间表：**

| 复习次数 | 间隔时间 | 记忆保留率 |
|----------|----------|------------|
| 第1次 | 学完当天 | 恢复到100% |
| 第2次 | 1天后 | 90%+ |
| 第3次 | 3天后 | 85%+ |
| 第4次 | 7天后 | 80%+ |
| 第5次 | 14天后 | 75%+ |
| 第6次 | 30天后 | 进入长期记忆 |

**工具推荐：**
- Anki（闪卡软件，自动安排复习时间）
- 使用 `learn.sh flashcard "主题"` 生成闪卡

> 💡 不是学了就完了，复习才是巩固记忆的关键。

---

### 3. 主动回忆法（Active Recall）

**核心理念：** 主动回忆比被动阅读有效10倍。

**实践方法：**
- 合上书本，尝试回忆刚才学的内容
- 用白纸写出知识点，看能写多少
- 做练习题而不是反复看笔记
- 向别人讲解学到的内容

**使用命令：**
- `learn.sh quiz "主题"` — 快速检测
- `learn.sh test "主题"` — 完整自测

> 💡 看书10遍不如做题1遍。被动输入 → 主动输出。

---

### 4. 交叉学习法（Interleaving）

**核心理念：** 混合学习不同主题比单一重复更有效。

**具体做法：**
- 不要一整天只学一个科目
- 交替学习相关但不同的主题
- 在不同科目之间建立联系
- 适当制造"困难"，迫使大脑深度思考

> 💡 大脑喜欢新鲜感。交叉学习让每次切换都激活不同的神经连接。

## 三、学习路线设计

### 通用学习路线模板

```
🌱 入门期（0-1个月）
   ├── 了解全貌：这个领域是什么？
   ├── 核心术语：掌握20个关键词
   ├── 入门资料：1本书 + 1个视频课
   └── 目标：能用自己的话概括

🌿 成长期（1-3个月）
   ├── 核心知识：深入理解原理
   ├── 动手实践：做3-5个小项目
   ├── 建立联系：和已有知识关联
   └── 目标：能解决基础问题

🌲 进阶期（3-6个月）
   ├── 深入专题：选择1-2个方向深耕
   ├── 解决难题：挑战复杂场景
   ├── 输出内容：写文章或教别人
   └── 目标：能独立解决复杂问题

🏔️ 精通期（6-12个月）
   ├── 融会贯通：形成知识体系
   ├── 创新应用：发展自己的方法
   ├── 持续学习：关注前沿发展
   └── 目标：能教给别人
```

**使用命令：** `learn.sh roadmap "领域"` / `learn.sh plan "主题"`

## 四、常见学习误区

### ❌ 误区 vs ✅ 正确做法

| 误区 | 为什么错 | 正确做法 |
|------|----------|----------|
| 反复阅读课本 | 被动输入，效率极低 | 主动回忆 + 做题 |
| 画满荧光笔 | 制造虚假的熟悉感 | 用自己的话做笔记 |
| 一次学很久 | 注意力下降，效率递减 | 番茄钟 25 分钟一轮 |
| 只看不练 | 知道 ≠ 会做 | 学一个练一个 |
| 追求速度 | 基础不牢地动山摇 | 确认理解再往下走 |
| 死记硬背 | 遗忘速度快 | 理解 + 间隔重复 |
| 从难题开始 | 受挫 → 放弃 | 从简单开始建立信心 |
| 不做总结 | 知识碎片化 | 每天/每周做知识总结 |

## 五、学习效果衡量

### 布鲁姆分类法（学习深度）

```
  6. 创造 ─── 能设计新方案
  5. 评价 ─── 能判断优劣
  4. 分析 ─── 能拆解结构
  3. 应用 ─── 能解决问题
  2. 理解 ─── 能用自己的话说
  1. 记忆 ─── 能复述出来
```

**自评方法：**
- 能给别人讲清楚 → 至少达到「理解」层
- 能用来解决实际问题 → 达到「应用」层
- 能分析比较不同方案 → 达到「分析」层
- 能设计创新的解决方案 → 达到「创造」层

### 学习效果检测命令

```bash
learn.sh quiz "主题"      # 快速4题检测
learn.sh test "主题"       # 完整试卷（选择+简答）
learn.sh feynman "概念"    # 费曼学习法深度理解
learn.sh flashcard "主题"  # 间隔重复记忆
```
