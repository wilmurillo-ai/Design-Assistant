# 微信公众号媒体下载器 WeChat Media Downloader

一个面向中文微信公众号场景的 OpenClaw Skill，用于下载公众号文章中的视频、音频和音乐卡片。

## 它解决什么问题

很多公众号文章里的视频、音频明明可以在浏览器里正常播放，但直接脚本抓取时常会被微信拦截，返回：

- 环境异常
- 去验证

这个 skill 不走“死磕反爬”的路线，而是采用更稳定的实战流程：

1. 先尝试普通抓取
2. 若被微信验证页拦截，则切换到可见 Chrome
3. 由用户手动完成验证
4. 通过 Chrome 远程调试连接真实页面
5. 抓取视频直链与音频 mediaid
6. 自动下载、提取标题、重命名并整理输出

## 适用场景

- 一篇公众号文章里有 1 个或多个视频
- 同时还有多个音频/音乐卡片
- 需要完整保存到本地
- 需要按文章顺序整理成可读文件名
- 主要面向中文用户

## 特点

- 专门适配微信公众号媒体页面
- 支持视频 + 多段音频
- 能处理微信“环境异常 / 去验证”拦截
- 优先用户级安装，尽量不要求 sudo
- 支持中文标题提取与重命名
- 最终输出成干净目录，便于归档

## 技术路线

- Playwright
- Chrome 远程调试（CDP）
- 微信页面 HTML 提取
- `res.wx.qq.com/voice/getvoice` 音频下载
- `mpvideo.qpic.cn` 视频直链下载

## 注意

请仅在你有权访问和保存这些媒体内容时使用本 skill。
