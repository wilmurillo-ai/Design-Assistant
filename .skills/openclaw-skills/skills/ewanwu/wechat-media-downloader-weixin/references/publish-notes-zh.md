# 发布简介（中文）

## 标题
微信公众号媒体下载器 WeChat Media Downloader

## 一句话简介
下载微信公众号文章中的视频、音频和音乐卡片，适配微信验证拦截场景。

## 详细简介
这是一个面向中文微信公众号场景的 OpenClaw Skill。

它解决的不是普通网页下载，而是一个更具体也更常见的问题：
**公众号文章里的视频和音频明明能在浏览器里播放，但直接脚本抓取常常会被微信拦截，返回“环境异常 / 去验证”。**

这个 skill 采用一条更可靠的流程：

1. 先尝试普通抓取
2. 若被微信验证页拦截，则切换到**可见 Chrome + 人工验证**
3. 通过 Chrome 远程调试连接已打开的真实页面
4. 抓取视频直链和音频 `voice_encode_fileid`
5. 自动下载媒体
6. 从文章中提取音频标题并重命名
7. 整理输出到干净文件夹

适合这些场景：
- 一篇公众号文章里有 1 个视频 + 多个音频/音乐
- 用户已经能在浏览器里正常打开文章
- 需要把媒体完整保存到本地
- 希望文件名可读、按文章顺序整理

## 特点
- 专门针对微信公众号媒体页面
- 支持视频 + 多段音频
- 能处理微信“环境异常 / 去验证”拦截
- 优先使用最小权限、最小安装
- 支持中文标题提取与重命名
- 适合中文用户直接使用
