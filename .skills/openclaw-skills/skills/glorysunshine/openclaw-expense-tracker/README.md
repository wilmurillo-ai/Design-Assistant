# OpenClaw 智能记账本

一个基于 OpenClaw 框架的智能记账技能，支持自然语言输入、自动分类和数据分析。

## 功能特性

- ✅ 自然语言记账（"今天午饭花了35元"）
- ✅ 自动分类识别（餐饮、交通、购物等）
- ✅ 多维度统计分析
- ✅ 数据可视化图表
- ✅ OpenClaw 原生集成
- ✅ 本地数据保护

## 安装使用

```bash
# 安装技能
skillhub install openclaw-expense-tracker

# 启动记账本
openclaw skill run expense-tracker
```

## 使用示例

- "记账：今天午饭花了35元"
- "记录：打车花了25元"
- "添加：超市购物120元"
- "查询：这个月花了多少钱"
- "统计：餐饮支出占比"

## 技术栈

- OpenClaw SDK 2026.3.2
- Node.js
- 本地 JSON 数据存储
