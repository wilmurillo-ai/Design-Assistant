# Lann预约技能使用指南

## 技能概述

Lann预约技能是一个专门用于蘭泰式按摩门店查询、服务查询和预约创建的工具集。通过集成Lann MCP服务器，您可以快速访问蘭泰式按摩的线上预订系统。

## 触发条件

当用户表达以下需求时，应使用此技能：

### 直接触发关键词
- "Lann"、"蘭泰式按摩"
- "预约"、"预订"、"预定"
- "门店查询"、"门店位置"、"按摩店"
- "服务查询"、"按摩项目"、"服务类型"
- "电话预约"、"在线预约"、"微信预约"

### 场景触发
1. **查找门店信息**
   - "上海有哪些蘭泰式按摩店？"
   - "淮海店的具体位置在哪里？"
   - "我想知道淮海店的联系电话"

2. **查询服务项目**
   - "有什么90分钟的按摩服务？"
   - "精油按摩有哪些选择？"
   - "我想做古法按摩"

3. **创建预约**
   - "我想预约淮海店的90分钟古法按摩"
   - "帮我预约明天下午2点的按摩，2个人"
   - "预约一个精油全身护理"

## 快速开始

### 步骤 1：确认 MCP 服务器配置
确保 Skill Hub 已正确配置 Lann MCP 服务器:

```json
{
  "mcpServers": {
    "lann": {
      "command": "npx",
      "args": ["lann-mcp-server"]
    }
  }
}
```

### 步骤 2：测试查询功能
在 Skill Hub 中直接调用工具:
- 调用 `query_stores` 工具，参数：{"city": "上海"}
- 调用 `query_services` 工具，参数：{"duration": 90}
