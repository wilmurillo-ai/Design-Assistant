## 价格处理规则

```python
def clean_price(v):
    """¥12,999 → 12999.0；暂无报价 → None（跳过）"""
    if v is None: return None
    s = str(v).strip().replace('¥','').replace(',','')
    if s in ('-','暂无报价','','None'): return None
    try: return float(s)
    except: return None
```

## BJ（最大允许总质量）计算

```
BJ = 整备质量(BI) + 最大有效载荷(BM)
```
需确认目标网站是否有"最大允许总质量"字段，如有则直接使用；如只有"载荷"字段则需相加计算。

## 添加自定义映射

在 `web_scraper.py` 的 `FIELD_MAP` 字典中添加，或直接在对话中告诉我要映射的字段：

```python
FIELD_MAP['你的网页字段名'] = '目标Excel列'  # 例如: FIELD_MAP['续航里程'] = 'AP'
```
