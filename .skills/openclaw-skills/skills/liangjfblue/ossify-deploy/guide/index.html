注册/登录阿里云

点击下方按钮打开阿里云官网，注册一个新账号或使用已有账号登录。

打开阿里云 →

开通 OSS 服务

  1. 1打开 OSS 控制台，如果是首次使用，会提示未开通，点击「立即开通」
OSS 管理控制台（已开通状态）
  1. 2确认订单，费用 ¥0.00（按量付费，开通免费），点击确认支付
确认开通，费用为 ¥0.00

OSS 按量付费，部署一个小网站每月费用约 ¥1-3，几乎可以忽略。

购买域名

  1. 1打开阿里云 域名注册页，搜索你想要的域名
阿里云域名注册首页
  1. 2输入想要的域名（如 tatadi.top），点击搜索查看是否可注册
搜索域名是否可注册
  1. 3选择可用的域名，加入购物车
找到可用域名，加入购物车
  1. 4填写域名信息，确认购买
填写域名信息并确认购买
  1. 5完成支付
完成支付
  1. 6购买成功！可以在域名管理列表中看到你的域名
域名购买成功
  1. 7购买后需要完成域名实名认证（上传身份证/营业执照），审核通常 1-3 个工作日
完成域名实名认证

.top 域名首年约 ¥6，非常便宜。购买后需完成实名认证才能正常使用。

创建 RAM 用户并获取 AccessKey

  1. 1打开 RAM 控制台 → 用户 → 创建用户
点击「创建用户」按钮
  1. 2填写登录名称（如 ossify），勾选「使用永久 AccessKey 访问」
填写用户名并勾选 AccessKey
  1. 3勾选确认，点击「确定」，完成安全验证（手机/扫脸）
勾选确认并点击「确定」
  1. 4立即复制保存 AccessKey ID 和 Secret！Secret 仅显示一次。
重要：AccessKey Secret 只在创建时显示一次！请立即复制保存。
立即复制 AccessKey ID 和 Secret

为 RAM 用户添加权限

勾选刚创建的用户，点击「新增授权」，依次添加三个权限策略：

  1. 1在列表中找到并勾选 AliyunOSSFullAccess
勾选 AliyunOSSFullAccess
  1. 2搜索 AliyunDNS，勾选 AliyunDNSFullAccess
搜索并勾选 AliyunDNSFullAccess
  1. 3搜索 AliyunCDN，勾选 AliyunCDNFullAccess
搜索并勾选 AliyunCDNFullAccess
  1. 4确认已选 3 个权限，点击「确认新增授权」
确认 3 个权限已选，点击确认
授权成功！3/3 权限绑定完成

回到终端粘贴凭证

$ ossify config
AccessKey ID:

现在回到 Claude Code 终端，将 AccessKey ID 和 AccessKey Secret 粘贴到终端中。

你的 AccessKey 只保存在你电脑本地（~/.ossify/auth.json），不会传输到任何外部服务器。
