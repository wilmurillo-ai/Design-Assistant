## 九、关联交易字段（需从附注获取）

Tushare无直接关联交易字段，需从财报附注文本或公司公告中提取。

关键指标：
- 关联采购金额 / 总采购
- 关联销售金额 / 总销售
- 关联方应收应付款项占比
- 对外担保金额 / 净资产

## 十、API调用频率限制

| 接口类型 | 限制 | 建议间隔 |
|---------|------|---------|
| 高频接口（实时行情） | ≤50次/分钟 | 1.2s |
| 基础接口（日线/财务） | ≤800次/分钟 | 0.3s |
| 美股日线 | ≤5次/天 ⚠️ | 保留配额 |

---

## 十一、字段缺失处理策略

| 缺失场景 | 处理方法 |
|---------|---------|
| 单年度缺失 | 使用最近年度替代，标注局限性 |
| 多年度缺失 | 标注"数据不足"，跳过该指标 |
| 部分字段缺失 | 使用已知字段计算，标注假设 |
| 行业对比缺失 | 使用历史趋势替代，标注局限性 |
