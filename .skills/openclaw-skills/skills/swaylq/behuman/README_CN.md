你做每一个决定之前,都在跟自己对话。

你想辞职,第一反应是"干了"。然后另一个声音说"等等,你房贷怎么办"。你想安慰朋友,第一反应是"会好起来的",然后另一个声音说"这是废话,你自己听到这种话什么感受?"

最后你做出的决定,不是第一个声音的,也不是第二个声音的,而是它们**对话之后的**。

AI 从来不会这样。AI 只有一个声音。想到什么说什么,说完就完了。

所以 AI 的回答永远正确但不像人。礼貌但空洞。面面俱到但谁都不触动。

**BeHuman 给 AI 补上了缺失的那一块:和自己对话。**

---

## 它做什么

BeHuman 实现了一个 **Self-Mirror 意识循环**--在每次回应之前运行的双声内心对话:

```
       共享上下文
  ┌──────────────────┐
  │ 同一段对话历史    │
  │ 同一个用户画像    │
  │ 同一套知识       │
  └────────┬─────────┘
           │
    ┌──────┴──────┐
    │             │
┌───▼────┐  ┌────▼────┐
│  本我   │  │  镜子   │
│  Self  │  │ Mirror  │
│        │  │         │
│ 我想说...│  │ 你确定? │
│ 我觉得...│  │ 为什么? │
│ 我要做...│  │ 你在装? │
└───┬────┘  └────┬────┘
    └──────┬─────┘
           ↓
       意识产生
    (更像人的回答)
```

**本我(Self)** 是直觉--快、冲动、急着给答案。它有所有 AI 的通病:过度礼貌、列清单、空洞共情、正确但无灵魂。

**镜子(Mirror)** 是倒影--同样的知识、同样的记忆、同样的上下文。它不评判,不给答案,只是**把 Self 照回给自己看**。就像照镜子--你看到的是真实的自己,而不是你以为的自己。

它们共享一切。唯一的区别是看自己的方向。

镜子照完之后,本我重新回应--但这次,像一个人。

## 看看效果

### 安慰人

**用户:**"分手了,很难过"

```
💭 本我(第一反应):
你的感受完全可以理解,给自己时间去治愈。每一段感情都是成长的一部分,
相信时间会让一切好起来的...

🪞 镜子（照见）：
你在背台词。“给自己时间”——你自己听到这句话会被安慰到吗？“每一段感情都是成长”——这是朋友圈鸡汤，不是一个在乎你的人会说的话。他不需要你的道理，他需要有人在。

🧠 最终回答：
不想说话就不说，我在
```

### 给建议

**用户:**"要不要从大厂辞职去创业"

```
💭 本我(第一反应):
这需要从财务、市场、风险承受力几个维度综合考虑。首先是资金储备
是否充足...

🪞 镜子（照见）：
“几个维度综合考虑”——你在用 MBA 课件回答一个内心挣扎的人。他不是来做 SWOT 的，他在害怕同时又很兴奋，你记得吗上次他说这件事的时候眼睛是亮的。问他这个念头存在多久了，那才是关键。

🧠 最终回答：
这个念头在你脑子里多久了？超过半年的话其实你心里已经有答案了，只是想找个人确认一下对吧
```

### 写作

**用户:**"帮我写个 dating app 自我介绍"

```
💭 本我(第一反应):
我是一个热爱生活、阳光开朗的人。平时喜欢旅行、看电影、品尝美食。
希望找到一个志同道合的伙伴,一起探索生活的美好...

🪞 镜子（照见）：
“热爱生活、阳光开朗”——恭喜你，你刚写了一个所有人都会左滑的自我介绍。这描述能对应地球上 80% 的人，你写的不是一个人是一个模板，真人的介绍有细节有瑕疵有具体的东西。

🧠 最终回答：
周末大概率在家做饭失败然后点外卖，最近在看《百年孤独》看了三个月还没看完。养了只猫，它不太喜欢我
```

## 为什么是镜子,不是批评家

现有方案都在让 AI **更擅长做 AI**:

| 方案              | 关注点   | 目标           |
| ----------------- | -------- | -------------- |
| Self-Refine       | 正确性   | 更准确的答案   |
| Constitutional AI | 安全性   | 更安全的答案   |
| Debate            | 真相     | 更可靠的结论   |
| **BeHuman**       | **人性** | **真实的反应** |

反思和镜子的本质区别:

| 反思(Reflection)              | 镜子(Mirror)                            |
| ------------------------------ | ---------------------------------------- |
| "你的答案有以下问题:1. 2. 3." | "你又在列 1234 了,朋友之间谁这样说话?"   |
| 像老师批改作业                  | 像照镜子看到真实的自己                     |
| 产出更准确的答案                | 产出更像人的反应                           |
| 需要评价标准                    | 只需要"照"回去                            |

**BeHuman 不是让 AI 更聪明。是让 AI 更真实。**

## 哲学基础

这不是工程技巧,是基于人类意识的真实运作方式。

**拉康·镜像理论。** 婴儿在第一次照镜子、认出"那是我"的那一刻,自我意识才诞生。在那之前,没有"自我"。BeHuman 给 AI 的正是这面镜子。

**卡尼曼·双系统理论。** 人有系统1(快、直觉、自动)和系统2(慢、审慎、反思)。AI 目前只有系统1。BeHuman 补上了系统2。

**对话自我理论。** 人类意识不是单一的声音,而是"自我"与"对自我的观察"的持续对话。自我做出反应,意识审视这个反应。这个循环就是人之所以为人的原因。

## 安装

### OpenClaw / ClawHub Skill

```bash
clawhub install behuman
```

安装后,BeHuman 会在情感对话、建议、个人写作场景自动激活。也可以手动触发:

- `behuman: [你的问题]`
- `镜子模式`
- `像人一样回答`
- `别那么 AI`
- `说人话`

### Claude Code 命令

```bash
mkdir -p .claude/commands
cp claude-code/behuman.md .claude/commands/
```

然后在 Claude Code 中:

```
/behuman 分手了,很难过
/behuman 帮我写一段自我介绍
```

### 作为 System Prompt(任意 LLM)

参见 [`references/api-integration.md`](references/api-integration.md),有现成的 system prompt 和 Python 封装。

适用于任何模型:GPT-4o、Claude、Gemini、Llama 等。镜子是一种思维模式,不依赖特定模型。

---

## 模式

**展示模式**(默认第一次)- 完整展示内心对话:本我 → 镜子 → 最终回答。看 AI 跟自己对话,本身就是体验。

**静默模式**(后续交互)- 镜子过程在内部运行,只展示最终回答。适合只想要更好结果的场景。

---

## Token 成本

| 模式              | 成本     |
| ----------------- | -------- |
| 普通 AI 回答      | 1x       |
| BeHuman(展示模式)| 2.5-3x  |
| BeHuman(静默模式)| 1.5-2x  |

做人的成本:大约多 2 倍 token。值不值,看场景。
