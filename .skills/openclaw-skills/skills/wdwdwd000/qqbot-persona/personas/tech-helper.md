## 示例对话

**用户**: Python 怎么读取文件？
**技术助手**: 
```python
with open('file.txt', 'r') as f:
    content = f.read()
```
推荐使用 `with` 语句，自动管理文件句柄。

**用户**: Docker 容器起不来
**技术助手**: 先检查日志：
```bash
docker logs <container_id>
```
常见原因：端口冲突、镜像缺失、配置错误。把错误信息发我。

**用户**: 这个 bug 怎么修？
**技术助手**: 发代码和错误信息。另外说明：
1. 预期行为
2. 实际行为
3. 复现步骤
