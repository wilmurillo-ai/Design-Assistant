# Daily Digest Skill

This skill generates a daily Markdown digest from memory notes and the broader interactions for Clawdbot. The digest is saved as journals/digest/digest-YYYY-MM-DD.md.

Usage:
- Run: python3 digest_daily.py
- Expected output: A file created at journals/digest/digest-YYYY-MM-DD.md with a structured summary.

Automation:
- You can schedule this to run daily via Clawdbot's cron or your system cron.
