## 🎯 一句话介绍

---

## ✨ 核心功能

### 1. 实时持仓监控
- 自动获取长桥证券账户持仓
- 支持 49+ 只股票同时监控
- 实时计算市值和盈亏

### 2. 智能止盈止损
- 预设止盈止损价格
- 价格触发自动提醒
- 支持自定义监控规则

### 3. 投资组合可视化
- 港股/美股分开显示
- 前15大持仓饼图
- 做多/做空一目了然

### 4. 投资建议
- 持仓集中度分析
- 风险评估
- 调仓建议

## 🚀 快速开始

### 安装

```bash
# 1. 安装依赖
pip install longbridge matplotlib

# 2. 配置 API Token
echo 'export LONGBRIDGE_APP_KEY="你的AppKey"' > ~/.longbridge/env
echo 'export LONGBRIDGE_APP_SECRET="你的AppSecret"' >> ~/.longbridge/env
echo 'export LONGBRIDGE_ACCESS_TOKEN="你的AccessToken"' >> ~/.longbridge/env

# 3. 运行
./run.sh
```

### 获取长桥 API Token

1. 访问 https://open.longportapp.com
2. 登录长桥账户
3. 申请开发者权限
4. 创建应用获取 Token
