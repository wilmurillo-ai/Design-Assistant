# Report Types

## `sprint-summary`

Inputs:

- project
- team
- optional iteration id or current sprint fallback

Outputs:

- raw records
- summary by state
- summary by assignee
- summary by type
- chart-ready totals

## `wiql`

Inputs:

- project
- query
- optional team

Outputs:

- raw records
- summary by state
- summary by assignee
- summary by type

## Chart ideas

Use these default charts in spreadsheets or chart images:

- Work items by state (bar/pie)
- Work items by assignee (bar)
- Work items by type (donut/bar)
- Original estimate by assignee (stacked bar)
- Remaining work by assignee (bar)
