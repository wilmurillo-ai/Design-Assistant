# A 股市场复盘 Skill

**收盘后专用** — 对当日 A 股市场表现进行全面总结，帮助快速把握市场脉络。

## 何时使用

✅ **使用场景：**
- "今日 A 股复盘" / "今日市场总结"
- "今天什么板块涨得好"
- "查询今日热门板块和龙头股"
- 每日收盘后市场回顾（15:00 后）
- 投资复盘与策略调整参考

❌ **不使用场景：**
- 盘中实时行情查询（数据可能滞后）
- 个股深度分析（需用专门股票分析工具）
- 历史数据回测
- 实时交易决策

## 数据来源

- **东方财富网** (eastmoney.com)
- 无需 API Key
- 行情数据（交易时段实时更新）

## 使用方法

### 基础用法

```bash
python3 scripts/a_stock_market_review.py
```

### 在对话中

直接请求：
- "今日 A 股复盘"
- "今天市场表现如何"
- "查询热门板块和龙头股"
- "A 股收盘总结"

## 输出内容

1. **大盘指数** — 上证指数、深证成指、创业板指涨跌幅
2. **🔥 热门板块 Top 10** — 按涨幅排序的概念板块
3. **🏆 板块龙头股** — 前 3 大热门板块的领涨个股
4. **🎯 创历史新高股票** — Top 20 列表
5. **📊 连续创新高股票** — 20 日/60 日新高股票
6. **市场简评** — 当日市场特征简要分析

## 依赖

- Python 3.7+
- requests 库

```bash
pip install requests
```

## API 接口说明

### 大盘指数
```
http://push2.eastmoney.com/api/qt/stock/get
参数：secid=1.000001 (上证指数)
```

### 板块排行
```
http://push2.eastmoney.com/api/qt/clist/get
参数：fs=m:90+t:3 (概念板块)
```

### 板块成分股
```
http://push2.eastmoney.com/api/qt/clist/get
参数：fs=b:BK1128 (具体板块代码)
```

### 创新高股票
```
http://push2.eastmoney.com/api/qt/clist/get
参数：fid=f109 (按历史新高排序)
```

## 注意事项

1. **数据更新时间**：交易日 9:30-15:00 实时更新，非交易时段显示最后收盘价
2. **最佳使用时机**：建议 15:00 收盘后使用，获取完整当日数据
3. **网络超时**：请求超时设置为 5 秒，失败时会提示
4. **免责声明**：数据仅供参考，不构成任何投资建议
