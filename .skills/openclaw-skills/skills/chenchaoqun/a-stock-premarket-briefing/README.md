# A 股盘前简报 Skill

📈 交易日专用 — 开盘前分析市场新闻，生成投资参考报告

## 快速开始

### 1. 安装依赖

```bash
pip install requests
brew install steipete/tap/summarize
```

### 2. 运行脚本

```bash
cd /home/node/.openclaw/workspace/skills/a-stock-premarket-briefing
python3 scripts/a_stock_premarket_briefing.py
```

### 3. 在对话中使用

直接请求：
- "今日 A 股盘前简报"
- "生成盘前分析报告"
- "今天市场主线是什么"

## 功能特性

✅ **新闻搜索** — 使用 multi-search-engine 搜索过去 12 小时关键新闻
✅ **多源验证** — 至少 5 个来源交叉验证
✅ **AI 分析** — 使用 summarize 生成结构化分析
✅ **主线叙事** — 3 条核心投资逻辑
✅ **市场偏向** — 多/空/震荡判断 + 置信度
✅ **板块预测** — 当日最强板块
✅ **观察清单** — 5 个可关注标的
✅ **来源溯源** — 每条结论附带链接

## 搜索策略

### 核心关键词
- A 股最新消息
- 美联储最新讲话
- 证监会新政
- 北向资金流入
- 美股昨夜涨跌

### 优先来源
- 官方：证监会、交易所、新华社
- 权威：财新、一财、证券时报
- 数据：东方财富、同花顺
