**WeryAI Image** へようこそ！このスキルは、[WeryAI](https://weryai.com) を利用して、OpenClaw エージェントに Generate high-quality AI images using WeryAI (Flux, SD3, etc.). の機能を提供します。

言語: [English](README.md) | [中文](README_zh.md) | [日本語](README_ja.md) | [Deutsch](README_de.md) | [Español](README_es.md)

## 🚀 クイックスタート

1. **インストール:** `clawhub install weryai-image`
2. **設定:** WeryAI API Key を `~/.openclaw/openclaw.json` に追加します。
3. **使い方:** エージェントに依頼するだけです！
