Welcome to the **WeryAI Media Editor** for OpenClaw! This skill empowers your agent with Advanced media editor: AI Upscaling, Background Removal, Image-to-Video (I2V), and Face Swapping. powered by [WeryAI](https://weryai.com).

Languages: [English](README.md) | [中文](README_zh.md) | [日本語](README_ja.md) | [Deutsch](README_de.md) | [Español](README_es.md)

## 🚀 Quick Onboarding

1. **Install:** `clawhub install weryai-media-editor`
2. **Setup:** Add your WeryAI API Key to `~/.openclaw/openclaw.json`.
3. **Usage:** Ask your agent to use it!
### Supported Models
You can check all the officially supported models at: [https://www.weryai.com/api/discovery](https://www.weryai.com/api/discovery)

To test it from CLI:
```bash
node weryai-media-editor.js models
```
