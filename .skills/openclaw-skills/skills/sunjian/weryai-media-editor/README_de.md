Willkommen beim **WeryAI Media Editor**! Dieser Skill verleiht Ihrem OpenClaw-Agenten die Fähigkeit: Advanced media editor: AI Upscaling, Background Removal, Image-to-Video (I2V), and Face Swapping., angetrieben von [WeryAI](https://weryai.com).

Sprachen: [English](README.md) | [中文](README_zh.md) | [日本語](README_ja.md) | [Deutsch](README_de.md) | [Español](README_es.md)

## 🚀 Schnellstart

1. **Installation:** `clawhub install weryai-media-editor`
2. **Setup:** Fügen Sie Ihren WeryAI API-Schlüssel zu `~/.openclaw/openclaw.json` hinzu.
3. **Nutzung:** Bitten Sie einfach Ihren Agenten, es zu benutzen!
