欢迎使用 **WeryAI Media Editor**！本插件为您的 OpenClaw 助手提供 Advanced media editor: AI Upscaling, Background Removal, Image-to-Video (I2V), and Face Swapping. 的能力，由 [WeryAI](https://weryai.com) 驱动。

语言: [English](README.md) | [中文](README_zh.md) | [日本語](README_ja.md) | [Deutsch](README_de.md) | [Español](README_es.md)

## 🚀 快速上手

1. **安装:** `clawhub install weryai-media-editor`
2. **配置:** 将 WeryAI API Key 写入 `~/.openclaw/openclaw.json`。
3. **使用:** 直接与您的助手对话即可！
