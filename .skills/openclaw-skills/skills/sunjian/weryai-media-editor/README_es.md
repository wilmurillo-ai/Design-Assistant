## 🚀 Inicio Rápido

1. **Instalación:** `clawhub install weryai-media-editor`
2. **Configuración:** Agregue su API Key de WeryAI a `~/.openclaw/openclaw.json`.
3. **Uso:** ¡Solo pídale a su agente que lo use!
