**WeryAI Media Editor** へようこそ！このスキルは、[WeryAI](https://weryai.com) を利用して、OpenClaw エージェントに Advanced media editor: AI Upscaling, Background Removal, Image-to-Video (I2V), and Face Swapping. の機能を提供します。

言語: [English](README.md) | [中文](README_zh.md) | [日本語](README_ja.md) | [Deutsch](README_de.md) | [Español](README_es.md)

## 🚀 クイックスタート

1. **インストール:** `clawhub install weryai-media-editor`
2. **設定:** WeryAI API Key を `~/.openclaw/openclaw.json` に追加します。
3. **使い方:** エージェントに依頼するだけです！
