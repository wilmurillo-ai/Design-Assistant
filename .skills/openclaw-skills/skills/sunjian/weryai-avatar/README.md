# WeryAI Avatar (Talking Heads)

Create stunning, lip-synced avatar videos from a single photo and an audio file! 

## 🚀 Quick Onboarding

1. **Install:** `clawhub install weryai-avatar`
2. **Setup:** Export your `WERYAI_API_KEY` in your environment variables.
3. **Usage:** 
   - "Make this image talk with this audio file."

To see it from CLI:
```bash
node weryai-avatar.js https://.../photo.png https://.../audio.mp3
```
