# iwencai-screener

> 基于i问财的智能选股技能 - 根据利好行业/概念自动选股

## 功能介绍

**iwencai-screener** 是一个智能选股技能，能够：
1. 分析输入内容（链接、文件、文本、关键词）
2. 识别利好行业、概念、题材、产业链上下游
3. 在i问财网站进行选股查询
4. 返回每项前5个结果（包含所有字段）
5. 生成Excel文件方便分析

## 触发方式

当用户具有以下意图时自动激活：
- "XX选股" - 如"新能源选股"、"AI选股"
- "XX利好哪些股票" - 如"芯片利好哪些股票"
- "分析利好哪些行业"
- "概念选股"、"行业选股"
- "根据XX选股"
- 输入链接/文本后要求分析并选股

## 核心特点

### 1. 聚焦原则
- 识别3-5个最核心的利好方向
- 优先选择与内容最直接相关的行业/概念
- 不泛泛而谈

### 2. 产业链逻辑
- 每个方向标注"利好逻辑"
- 标注产业链位置（上游/中游/下游）

### 3. 数据完整
- 表格包含所有字段：序号、股票代码、股票简称、现价(元)、涨跌幅(%)、概念解析、概念资讯、所属概念数量(个)、A股流通市值(元)、所属概念
- 生成Excel文件输出

## 使用示例

### 示例1：行业选股
```
用户：新能源选股
技能：分析新能源行业利好方向，查询相关股票
```

### 示例2：利好分析
```
用户：芯片利好哪些股票
技能：分析芯片产业链，查询上下游受益股票
```

### 示例3：链接分析
```
用户：https://news.example.com/xxx 选股
技能：获取链接内容，分析利好行业，查询相关股票
```

## 输出格式

### 文本回复
```
【核心利好方向分析】

根据对内容的分析，聚焦以下核心利好方向：

## 一、XX行业（最核心）
- **利好逻辑**：具体说明为什么利好
- **产业链位置**：上游/中游/下游

---

### 选股结果

## XX概念相关股票
| 序号 | 股票代码 | 股票简称 | 现价(元) | 涨跌幅(%) | ... |
|------|----------|----------|----------|-----------|-----|
| 1 | 301396 | 宏景科技 | 149.98 | 20.00 | ... |

---
📊 Excel文件已生成: 选股结果_20260312_171123.xlsx
```

### Excel文件
- 文件名：`选股结果_YYYYMMDD_HHMMSS.xlsx`
- Sheet：按行业/概念名称命名
- 列：包含所有原始字段
