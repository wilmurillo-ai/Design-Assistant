## 1. AUDIT SWEEP PHILOSOPHY

### Why Audits Matter After Publishing
Publishing is not the end of the quality lifecycle. Live assets decay:
- Prices become stale (competitors adjust, market shifts)
- Links break (external references stop working)
- Claims become inaccurate (statistics age, context changes)
- Design becomes dated (what looked fresh becomes dated)
- Listings lose ranking (keywords shift, algorithm changes)

Audits catch decay before it damages trust or revenue.

### Audit Frequency Guidelines
```
MONTHLY: Gumroad product listings, active email sequences, link checks
QUARTERLY: Full content audit, competitive positioning, pricing review
ANNUALLY: Brand audit, full catalog review, archive decisions
```

## 2. MONTHLY GUMROAD PRODUCT AUDIT

### Per-Product Audit (Run Every 30 Days)
```
LISTING HEALTH:
- [ ] Product is still active and purchasable
- [ ] Cover image loads correctly
- [ ] Description renders correctly (no broken formatting)
- [ ] Price still matches strategy (not accidentally changed)
- [ ] Category still accurate
- [ ] Tags still relevant

DOWNLOAD INTEGRITY:
- [ ] Download link tested (can still download)
- [ ] File opens without errors
- [ ] File is correct version (not accidentally overwritten)

PERFORMANCE REVIEW:
- [ ] Sales this month vs. last month (flagging unusual drops)
- [ ] Any negative reviews? (Address within 48 hours)
- [ ] Conversion rate from views → purchases (benchmark: 3-8%)

COMPETITIVE POSITIONING:
- [ ] Is our price still appropriate vs. comparable products?
- [ ] Are competitor products outranking us on similar searches?
```

### Audit Action Triggers
```
If ZERO sales in 30 days:
  → Review: Is listing visible? Price right? Description compelling?
  → Test purchase to verify flow
  → Consider: price experiment, description refresh

If conversion rate <2%:
  → Review listing copy (likely not converting)
  → Check cover image quality
  → Review first 50 words of description

If negative review:
  → Read and acknowledge (don't dismiss)
  → Respond publicly if appropriate
  → Consider product update if complaint is valid
```

## 3. MONTHLY EMAIL SEQUENCE AUDIT

### Active Sequence Health Check
```
For each active email sequence:
- [ ] Open rates still healthy (baseline: 20%+ for warm list)
- [ ] Click rates still healthy (baseline: 2%+ for warm list)
- [ ] Unsubscribe rate not spiking (flag if >0.5% per email)
- [ ] All links in sequence still working
- [ ] All personalizations tokens still filling correctly
- [ ] Sequence timing appropriate (not too fast/slow)

CONTENT FRESHNESS:
- [ ] References to "this year" or seasons still accurate
- [ ] Any mentioned products still exist and are priced correctly
- [ ] Any promotions referenced have been removed (expired offers)
- [ ] Any statistics mentioned are still current
```

## 4. QUARTERLY CONTENT AUDIT

### Full Catalog Review
```
Quarterly review of all published content:

GUMROAD PRODUCTS:
- [ ] Full product list reviewed (active/inactive appropriate)
- [ ] Archive products that are outdated or underperforming
- [ ] Update descriptions for products with 0 reviews but active sales
- [ ] Update cover images for products with declining traffic

WEBSITE/LANDING PAGES:
- [ ] All pages load without errors
- [ ] All forms work (test submission)
- [ ] All links to external resources work
- [ ] Contact information current
- [ ] Any "coming soon" pages that should now be live or removed

SOCIAL PROFILES:
- [ ] Bio/description current
- [ ] Profile/header image current
- [ ] Website link in bio correct
- [ ] Pinned posts still relevant
```

---

## 5. ANNUAL BRAND AUDIT

### Full Brand Consistency Review
```
Once per year: comprehensive brand audit across all assets

VISUAL CONSISTENCY:
- [ ] All product covers use current brand palette
- [ ] All covers use correct fonts
- [ ] Logo is current version everywhere it appears
- [ ] No severely dated design elements

VOICE CONSISTENCY:
- [ ] Old content doesn't represent voice we've moved past
- [ ] No content with positions we've changed
- [ ] AI-generated content review for quality standards

COMPETITIVE POSITION:
- [ ] Are we still competitively priced?
- [ ] Have competitors released similar products?
- [ ] Are our products still relevant to current market?

ARCHIVE DECISIONS:
- [ ] Products with < $50 total lifetime revenue → consider archiving
- [ ] Products with consistent negative feedback → update or archive
- [ ] Products that don't align with current positioning → evaluate
```

## 7. AUDIT CALENDAR

```
MONTHLY (first Monday of month):
  - Gumroad product audit
  - Active email sequence links check
  - Any reported issues from previous month

QUARTERLY (first Monday of Q2, Q3, Q4):
  - Full catalog review
  - Social profile audit
  - Email performance review
  - Pricing competitive check

ANNUALLY (first Monday of January):
  - Full brand audit
  - Archive decisions for underperforming products
  - Major copy refresh decisions
  - Platform strategy review
```
