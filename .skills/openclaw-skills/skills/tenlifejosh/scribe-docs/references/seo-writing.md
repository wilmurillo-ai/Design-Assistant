# SEO Writing — Reference Guide

Keyword integration, meta descriptions, alt text, semantic structure, and on-page
SEO for content that ranks without reading like it was written for robots.

---

## 1. SEO WRITING PHILOSOPHY

Write for humans. Optimize for search engines. In that order.
Google's algorithm has evolved to reward content that genuinely helps readers.
Content that's keyword-stuffed but shallow ranks poorly and converts worse.

---

## 2. ON-PAGE SEO ELEMENTS

### The SEO Element Hierarchy (Priority Order)
```
1. TITLE TAG: Most important for rankings
   - Contains primary keyword (front-loaded)
   - Under 60 characters
   - Compelling enough to get clicked
   - Format: "[Primary Keyword] — [Benefit/Context] | Site Name"
   
2. H1 HEADING: One per page, contains primary keyword
   - Can differ slightly from title tag
   - Compelling + keyword-forward
   
3. OPENING PARAGRAPH: Contains primary keyword in first 100 words
   - Naturally, not forced
   
4. H2/H3 HEADINGS: Contain secondary and related keywords
   - Semantic variations of main keyword
   
5. BODY COPY: Keyword density 1-2% maximum
   - Focus on comprehensiveness, not repetition
   
6. META DESCRIPTION: Click-through driver (not direct ranking factor)
   - 150-160 characters
   - Contains keyword + compelling reason to click
   
7. IMAGE ALT TEXT: Descriptive with relevant keywords
   - "family organization planner printable" not "image1.jpg"
```

## 3. KEYWORD INTEGRATION RULES

### Natural vs. Forced Keyword Use
```
NATURAL (good):
  "Family organization can be challenging. This planner addresses..."
  "When it comes to budgeting for families, most systems fail because..."

FORCED (bad):
  "Family organization planner PDF digital download for families who need 
  family organization planning is available here in our family organization store."
```

### Keyword Density Guideline
```
Primary keyword: Appears 2-4 times per 500 words
Secondary keywords: 1-2 times each
LSI (related) keywords: Naturally throughout

If you're counting keywords: stop. Just write comprehensively about the topic.
Comprehensive content naturally includes the right keywords.
```

---

## 4. META DESCRIPTION FORMULA

```
META DESCRIPTION = [Keyword phrase] + [Compelling benefit] + [CTA]
Length: 150-160 characters

EXAMPLES:
  "Family organization planner with 47 pages for weekly schedules, budgets, 
  and meals. Print once, stay organized all year. Download instantly."
  
  "Cold email templates for local service businesses. Get replies, book calls, 
  and close more clients. 10 proven templates — free to download."
```

## 5. HEADER STRUCTURE FOR SEO

### Correct Header Hierarchy
```
H1 — One per page — Primary keyword + main topic
  H2 — Major sections — Secondary keywords + semantic variations
    H3 — Subsections — Specific aspects of H2
      H4 — Only if needed — Very specific subsections

EXAMPLE for "Family Organization Planner" article:
H1: The Family Organization Planner That Actually Works
  H2: Why Most Family Planners Fail
  H2: What a Good Family Organization System Needs
    H3: The Weekly Schedule Component
    H3: Monthly Budget Tracking
    H3: Meal Planning Integration
  H2: How to Use FamliClaw
  H2: FAQs About Family Organization
```

## 6. SEO CONTENT STANDARDS

```
MINIMUM STANDARDS FOR RANKABLE CONTENT:
  - Primary keyword in H1
  - Primary keyword in first 100 words
  - 3-5 secondary/related keywords naturally included
  - Headers that outline the page's structure clearly
  - Internal links to related content
  - At least 1 image with descriptive alt text
  - Meta description under 160 characters
  - Page loads in < 3 seconds (technical SEO, not copy's job — but flag it)
  
CONTENT QUALITY FOR RANKINGS:
  - Completely answers the search query
  - More comprehensive than top 3 current results
  - Specific examples and actionable advice
  - Original insight, not just summarizing what others say
```
