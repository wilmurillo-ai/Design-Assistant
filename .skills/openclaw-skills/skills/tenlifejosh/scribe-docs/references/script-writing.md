# Script Writing — Reference Guide

Video scripts, podcast outlines, presentation scripts, and VSL scripts.
The structure and pacing for scripts that hold attention.

---

## 1. SCRIPT vs. COPY

Scripts are written to be HEARD, not read.
Sentences that look fine on paper sound weird when spoken.
Write for the ear:
- Shorter sentences (the pause replaces punctuation)
- Contractions (I'm, you're, don't — not I am, you are, do not)
- Conversational rhythm (how would you actually say this?)
- No complex nested clauses (listener can't reread)

## 2. THE VIDEO SCRIPT STRUCTURE

### Hook-Body-CTA Framework
```
HOOK (First 15-30 seconds):
  The only job: stop the scroll.
  State the most compelling thing first.
  Do NOT start with: "Hey everyone, welcome to my channel..."
  DO start with: The problem, the surprising fact, or the payoff

BODY:
  One clear point per minute of video (maximum)
  Example follows every concept (no abstract without concrete)
  Visual pacing: something new every 20-30 seconds
  Story elements woven through for emotional engagement

CTA:
  One clear action at the end
  Don't give them three options
  "Download the link below" or "Subscribe for more" — not both
```

### Short Video Script Template (60-90 second Reel/TikTok)
```
[0-5 seconds] PATTERN INTERRUPT:
  "Most family planners fail for the same reason."
  [Visual: hook image or kinetic text]

[5-20 seconds] THE PROBLEM:
  "They're designed to be maintained. Nobody maintains anything.
  So you use it for two weeks and quit. Again."

[20-45 seconds] THE SOLUTION:
  "FamliClaw is different. It's designed to require zero maintenance.
  You print it. Fill in the week. Refer to it. That's it.
  No app. No syncing. No notifications you ignore."

[45-60 seconds] THE PROOF + CTA:
  "47 pages. One print. One system.
  Link in bio if you want it."
```

## 3. VSL (VIDEO SALES LETTER) STRUCTURE

For longer promotional videos (5-15 minutes):

```
MINUTES 1-2: THE HOOK
  State the problem dramatically and specifically
  Create identification: "If you've ever..."

MINUTES 2-4: THE AGITATION
  Show what happens if nothing changes
  Paint the cost of the status quo

MINUTES 4-6: THE REVELATION
  Introduce the insight or solution
  Why everything else they've tried failed

MINUTES 6-10: THE PRODUCT PRESENTATION
  What it is
  What's inside
  Who it's for

MINUTES 10-12: PROOF
  Testimonials, results, social proof

MINUTES 12-14: CLOSE
  Price reveal
  Guarantee
  CTA

MINUTES 14-15: URGENCY
  Why now
  Final CTA
```

## 4. PODCAST OUTLINE FORMAT

### Episode Outline Template
```
EPISODE: [Title]
HOOK: One sentence that makes someone press play

INTRO (2-3 min):
  What this episode is about (30 seconds)
  Why it matters (30 seconds)
  What they'll learn (30 seconds)

SEGMENT 1 (5-8 min):
  Topic: [Specific point]
  Supporting idea 1
  Supporting idea 2
  Example/story
  Transition to segment 2

SEGMENT 2 (5-8 min):
  [Same structure]

SEGMENT 3 (5-8 min):
  [Same structure]

CLOSE (2-3 min):
  Synthesis (key takeaways — 3 max)
  Action step for listener
  CTA (subscribe, review, link)
```

## 5. SCRIPT WRITING STANDARDS

```
PACING:
  Read the script out loud before finalizing
  Time it — does it fit the format?
  Every section earns its time

LANGUAGE:
  Conversational, not formal
  Contractions used naturally
  No sentence over 20 words (when spoken)

TRANSITIONS:
  No announced transitions: "Now, let's move on to..."
  Natural pivots: "Here's the thing about that..."
  Story bridges between sections

ENERGY:
  Opening energy = high (earn their attention)
  Middle energy = varied (rising and falling, not flat)
  Close energy = high again (end strong)
```
