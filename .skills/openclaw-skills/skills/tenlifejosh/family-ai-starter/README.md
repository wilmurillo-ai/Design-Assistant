# Family AI Starter

**Your family's AI command center — set up in 30 minutes.**

The complete starter kit for families using OpenClaw. Built from FamliClaw, a production family AI system. Includes homework helper, meal planner, chore tracker, and family vault.

## What Your Agent Can Do After Setup

- 📚 **Homework Helper** — Guides kids through homework using the Socratic method. Never just gives the answer. Tracks each child's learning profile over time.
- 🍽️ **Meal Planner** — Weekly dinner planning with grocery lists. Respects dietary restrictions automatically. Suggests meals from what's in your fridge.
- ✅ **Chore Tracker** — Weekly chore assignments by child, age-appropriate. Track completion via chat. Allowance tracking built in.
- 🗄️ **Family Vault** — Stores insurance info, medical records, school docs, emergency contacts. Ask anything in plain English and get an instant answer.
- 📅 **Family Calendar** — Tracks events, appointments, pickups, and sends morning check-ins.

## Quick Start

1. Install this skill
2. Say "Family setup" to your agent
3. Answer ~10 questions (takes 10-15 min)
4. Done — your agent knows your family

## What Gets Created

After onboarding, your workspace will have:
- `family-config.json` — your family profile
- `SOUL-family.md` — family-optimized agent personality
- `HEARTBEAT-family.md` — morning/evening check-in routines
- `USER-template.md` — fill-in template for full family profile
