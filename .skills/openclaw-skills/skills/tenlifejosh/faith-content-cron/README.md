# Faith Content Cron

**Consistent faith-based social content, on autopilot.**

Generate daily scripture posts, devotional captions, and short-form video scripts for TikTok, Instagram, Twitter, and more — without burning out.

## What It Does

- Picks today's scripture based on day-of-week themes or your custom topic
- Generates platform-specific content: TikTok scripts, Instagram captions, Twitter threads, Facebook posts
- Auto-posts to configured channels (or generates for your review)
- Tracks post history so you never repeat
- Detects Christian calendar seasons (Advent, Lent, Easter, etc.) and adjusts content
- Weekly content calendar planning (Sunday planning session)

## Quick Start

1. Install this skill into your OpenClaw workspace
2. Your agent will ask a few config questions (platform, tone, frequency)
3. Say "Post today's content" and it handles the rest

## Configuration Options

| Setting | Options |
|---------|---------|
| Platforms | TikTok, Instagram, Twitter/X, Facebook, YouTube Shorts |
| Tone | Warm, Bold, Contemplative, Youth-focused |
| Frequency | Daily, Weekdays, MWF, Weekly |
| Bible version | NIV, ESV, KJV, NLT, NKJV |
| Auto-post | Yes (needs channel tool) / Generate only |
