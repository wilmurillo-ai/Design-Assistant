# Source Evaluation — Reference Guide

Credibility assessment, bias detection, cross-referencing, and data freshness standards.
Knowing which information to trust and how much.

---

## 1. SOURCE CREDIBILITY FRAMEWORK

### The CRAAP Test
```
CURRENCY: Is the information current enough?
  For market data: < 18 months (fast-moving markets: < 12 months)
  For statistics: < 3 years (unless clearly non-changing)
  For best practices: < 2 years

RELEVANCE: Does it directly address the research question?
  Is this actually about our market, our audience, our geography?
  Or are we extrapolating?

AUTHORITY: Who produced this and why should we believe them?
  Author credentials: Expert? Industry insider? Unknown?
  Publication reputation: Known and established? Unknown blog?
  Primary vs. secondary source?

ACCURACY: Can the claims be verified?
  Are sources cited within the source?
  Can the methodology be found?
  Do other sources corroborate the key claims?

PURPOSE: Why was this created?
  Educational → More likely neutral
  Commercial → Has an agenda (not disqualifying, but weight accordingly)
  Advocacy → Has an agenda (similar caution)
  Academic → Check methodology rigorously
```

## 2. BIAS DETECTION

### Common Source Biases
```
SELLER BIAS: Source is selling something related to the data
  Example: "Market for productivity tools is $50B" published by a productivity tool company
  Action: Find corroborating independent source before citing

SURVIVOR BIAS: Data only reflects successful examples
  Example: "Entrepreneurs who read this book doubled revenue"
  Action: What happened to those who read it and didn't succeed?

SELECTION BIAS: Sample doesn't represent population
  Example: "Survey of 500 LinkedIn users shows..."
  Action: Are LinkedIn users representative of your actual target audience?

RECENCY BIAS: Overweighting recent events
  Example: Citing COVID-era behavior changes as permanent trends
  Action: Compare to 5-year trends, not just last 12-18 months

GEOGRAPHIC BIAS: US data presented as universal
  Example: US market statistics applied to global opportunity
  Action: Specify geography; separate US vs. global if needed
```

## 3. CONFIDENCE LABELING STANDARDS

### How to Label Every Finding
```
🟢 HIGH CONFIDENCE:
  - Primary data or direct measurement (our own analytics)
  - Multiple independent sources corroborate (3+)
  - Established research methodology visible
  - Recent (< 12 months for market data)
  Use when: Making significant business decisions

🟡 MEDIUM CONFIDENCE:
  - Single credible source, well-established
  - Multiple lower-quality sources agree
  - Slightly outdated (12-36 months) but topic not fast-moving
  - Extrapolation from closely related data
  Use when: Directional decisions, can course-correct

🔴 LOW CONFIDENCE:
  - Single unverified source
  - Anecdotal (individual stories, n=1)
  - Data > 3 years old on fast-moving topic
  - Significant extrapolation required
  - Source has clear conflict of interest
  Use when: Hypothesis generation only, not for decisions
```

## 4. CROSS-REFERENCING PROTOCOL

### The Three-Source Rule
Critical claims require at least three independent sources before being labeled High Confidence.
"Independent" means:
- Different organizations
- Different data collection methods
- Different time periods (if freshness allows)

### When Sources Contradict
```
Source A says market is $X billion
Source B says market is $Y billion (significantly different)

RESOLUTION PROCESS:
1. Check methodology of each source
2. Check date of each source
3. Check whether definitions of "market" match
4. If still unclear: use the lower estimate (conservative = safer decisions)
5. Always note the contradiction in briefing: 
   "Market size estimates vary significantly: Source A ($X) vs. Source B ($Y) — 
   use midpoint with low confidence"
```
