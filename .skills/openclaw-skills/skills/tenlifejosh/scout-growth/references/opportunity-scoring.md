# Opportunity Scoring — Reference Guide

Ranking opportunities by potential, timing, effort, and strategic fit. The frameworks
that prevent chasing every shiny object while missing the real opportunities.

---

## 1. THE OPPORTUNITY SCORING IMPERATIVE

Every week brings new opportunities. Without a scoring system, you either chase everything
(spread too thin) or nothing (paralyzed by analysis). Opportunity scoring creates a
disciplined way to say yes to the right things and no to everything else.

---

## 2. THE RICE SCORING FRAMEWORK

### RICE: Reach × Impact × Confidence ÷ Effort

```
REACH: How many people does this affect in the relevant time period?
  - For product: estimated monthly buyers
  - For content: estimated monthly views
  - For feature: estimated monthly users
  Score: Actual number (e.g., 500 people/month)

IMPACT: How much does this move the needle per person?
  3 = Massive impact (game-changing)
  2 = High impact (significant improvement)
  1 = Medium impact (noticeable improvement)
  0.5 = Low impact (small improvement)
  0.25 = Minimal impact

CONFIDENCE: How confident are we in our estimates?
  100% = Verified data from tests
  80% = Strong evidence
  50% = Directional signals
  20% = Mostly guesses

EFFORT: Person-hours to complete
  Actual estimated hours/days

RICE SCORE = (Reach × Impact × Confidence%) ÷ Effort

INTERPRETATION:
  Higher score = higher priority
  Rank opportunities by RICE score
  Top 3 = focus
  Bottom 50% = parking lot
```

### RICE Example: Ten Life Creatives
```
OPPORTUNITY A: Build new journal product
  Reach: 200 buyers/month (estimated)
  Impact: 2 (good revenue impact per buyer)
  Confidence: 60% (based on existing product performance)
  Effort: 40 hours
  RICE = (200 × 2 × 0.60) / 40 = 6.0

OPPORTUNITY B: Refresh FamliClaw description
  Reach: 1000 visitors/month × 2% lift = 20 more buyers
  Impact: 1 (modest improvement per visitor)
  Confidence: 50%
  Effort: 2 hours
  RICE = (20 × 1 × 0.50) / 2 = 5.0

OPPORTUNITY C: Launch on Etsy
  Reach: 500 additional buyers/month (optimistic)
  Impact: 2 (new revenue channel)
  Confidence: 30% (haven't tested)
  Effort: 20 hours setup + 5/month ongoing
  RICE = (500 × 2 × 0.30) / 20 = 15.0

Ranking: C (15.0) > A (6.0) > B (5.0)
Action: Test Etsy first, then build new product, then refresh copy
```

## 3. THE ICE SCORING FRAMEWORK

### ICE: Impact × Confidence × Ease

Simpler than RICE — use when time for analysis is limited:

```
IMPACT: What will happen if this works? (1-10)
  10 = Game-changing, major revenue impact
  7 = Significant improvement
  5 = Noticeable but not transformational
  3 = Minor improvement
  1 = Negligible

CONFIDENCE: How likely is it to work? (1-10)
  10 = Very high confidence, proven approach
  7 = Good evidence this will work
  5 = 50/50
  3 = Mostly guessing
  1 = Very uncertain

EASE: How easy is it to execute? (1-10)
  10 = Trivial effort (1 hour)
  7 = Manageable effort (1 day)
  5 = Significant effort (1 week)
  3 = Major effort (1 month)
  1 = Enormous effort (months)

ICE SCORE = (Impact + Confidence + Ease) / 3

Prioritize: Highest ICE score first
```

## 4. OPPORTUNITY SCREENING MATRIX

### First Pass: Kill Questions
Before scoring, answer these kill questions.
If ANY answer is NO → stop, don't score:

```
KILL QUESTION 1: Does this align with our current focus?
  We have three priorities: GND, MFJ, Prayful
  Does this serve one of them? YES/NO

KILL QUESTION 2: Is there real evidence of demand?
  Not: "I think people would want this"
  Yes: Actual search volume, existing products selling, community discussion
  YES/NO

KILL QUESTION 3: Can we complete this in < 2 weeks?
  If longer: break it into phases, score first phase
  Can first phase deliver standalone value? YES/NO

KILL QUESTION 4: Is this better than our current top priority?
  Would this displace something we're already committed to?
  Is it worth the opportunity cost? YES/NO

If all YES → Score it
If any NO → Park it (not kill it — timing might change)
```

```

```

### Maintaining a Ranked Opportunity List
```

  1. [Opportunity] — [Status] — [Expected completion]
  2. [Opportunity] — [Status] — [Expected completion]
  
NEXT SPRINT (highest scored, ready to start):
  3. [Opportunity] — [Score] — [Owner]
  4. [Opportunity] — [Score] — [Owner]

  5. [Opportunity] — [Score] — [Blocker/Condition]
  6. [Opportunity] — [Score] — [Blocker/Condition]

  - [Opportunity] — [Why parked] — [Revisit date]
  
KILLED (not worth pursuing, archived):
  - [Opportunity] — [Why killed]
```

## 6. STRATEGIC FIT ASSESSMENT

### Beyond the Numbers: Does It Build the Business?
Score each opportunity on strategic value (1-5):

```
ASSET BUILDING:
  Does this create a lasting asset? (Content, tool, audience, relationship)
  5 = Creates major long-term asset
  1 = One-time benefit, no lasting asset

LEVERAGE:
  Does this multiply future opportunities?
  5 = Opens 10x more doors
  1 = Doesn't lead anywhere else

DEFENSIBILITY:
  Is this hard to copy?
  5 = Unique to us (our story, relationship, IP)
  1 = Easily copied by anyone

MOMENTUM:
  Does this build on our existing strengths?
  5 = Direct extension of what we do well
  1 = Requires completely new capability

REVENUE TIMING:
  5 = Revenue in < 2 weeks
  3 = Revenue in < 3 months
  1 = Revenue in > 6 months (avoid at zero runway)
```

## 7. URGENCY vs. IMPORTANCE CALIBRATION

### The Four Quadrants Applied to Opportunities
```
Q1 — URGENT + IMPORTANT: Do immediately
  Examples: Fix broken Gumroad link losing sales, responding to negative review

Q2 — NOT URGENT + IMPORTANT: Schedule it (strategic work)
  Examples: New product development, SEO improvements, audience building
  → This is where most revenue growth lives
  → Must protect time for Q2 work

Q3 — URGENT + NOT IMPORTANT: Delegate or minimize
  Examples: Platform notifications, minor formatting requests

Q4 — NOT URGENT + NOT IMPORTANT: Eliminate
  Examples: Exploring platforms we'll never use, meetings about meetings
```

## 8. OPPORTUNITY SCORING FOR TEN LIFE CREATIVES

### Our Specific Scoring Weights
Given Joshua's situation (zero runway, need revenue fast), weight accordingly:

```
Revenue speed: Weight 40% (most important — need money fast)
Effort required: Weight 30% (time is limited, everything competes)
Strategic fit: Weight 20% (builds toward long-term goals)
Market size: Weight 10% (nice to have large market, not required)

Apply to each opportunity:
  Revenue speed score × 0.4
  + Effort efficiency score × 0.3
  + Strategic fit score × 0.2
  + Market size score × 0.1
  = Weighted TLC Score

Rank by Weighted TLC Score, not raw RICE/ICE
```
