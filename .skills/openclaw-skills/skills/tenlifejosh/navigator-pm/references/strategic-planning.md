## 1. THE STRATEGY STACK

### From Mission to Daily Tasks
```
MISSION (Why we exist — permanent):
  "Build practical tools for builders, creators, and families"

VISION (What we're building toward — 3-5 year):
  "Ten Life Creatives is a recognized brand with a profitable catalog of
  digital products and a successful local web design service"

ANNUAL GOALS (What we achieve this year — set once/year):
  Goal 1: GND reaches $X MRR by December
  Goal 2: Launch 3 new digital products
  Goal 3: Prayful reaches beta with 50 users

QUARTERLY OKRS (What moves us toward annual goals — set each quarter):
  This quarter: "Build GND pipeline to 5 monthly leads minimum"

WEEKLY PRIORITIES (What actually happens — set every Monday):
  "Contact 10 Parker/Colorado Springs businesses with cold email"
  "Launch Budget Binder on Gumroad by Friday"

DAILY TASKS (Execution — tactical):
  Research targets, write emails, build product
```

## 2. THE 30/60/90 DAY PLAN

### Structure for Any New Initiative or Quarter Start
```
DAYS 1-30: LEARN & ESTABLISH
  Focus: Get the basics right. Remove uncertainty.
  Questions: What do we know? What's working? What's the baseline?
  Key outputs: 
    - First customers/sales if possible
    - Baseline metrics established
    - Process documented
  Measure of success: "We understand the situation clearly"

DAYS 31-60: BUILD & TEST
  Focus: Execute the core strategy. Test assumptions.
  Questions: What's working? What needs to change?
  Key outputs:
    - First meaningful results
    - At least one iteration based on learning
    - Bottlenecks identified
  Measure of success: "We're executing and learning"

DAYS 61-90: OPTIMIZE & ACCELERATE
  Focus: Double down on what works. Kill what doesn't.
  Questions: What's the highest-leverage next move?
  Key outputs:
    - Clear wins documented
    - Losing bets retired
    - Momentum activities identified and funded
  Measure of success: "We have a working model"
```

## 3. OKR FRAMEWORK

### OKR: Objectives + Key Results

**Objective**: Qualitative, inspiring direction
**Key Results**: Quantitative, measurable outcomes that prove the objective

```
OBJECTIVE: "Build GND into a consistent revenue stream"

KEY RESULTS:
  KR1: Acquire 5 new GND clients this quarter (currently 0)
  KR2: Generate $3,000 in GND MRR by end of quarter
  KR3: Send 100 cold outreach emails with 20%+ open rate

TEST: If all three KRs are achieved, was the Objective accomplished?
If YES → good OKR
If NO → the KRs don't actually prove the Objective

GRADING OKRS (at quarter end):
  1.0 = 100% of KR achieved
  0.7 = 70% achieved (this is GOOD — 70-80% is ideal)
  0.4 = 40% achieved (concerning)
  0.0 = No progress
  
  Target: 0.7-0.8 on KRs. If hitting 1.0 consistently — set harder targets.
  If hitting 0.3 consistently — either targets are wrong or execution is failing.
```

## 4. ANNUAL GOAL SETTING

### Year-Level Goal Structure
```
PROCESS (run once a year, December-January):

STEP 1: Review the past year
  What worked?
  What didn't?
  What was underestimated?
  What surprised us?

STEP 2: Set 3-5 annual goals (max)
  One per major business area
  Each must be measurable
  Each must be achievable (stretch but not fantasy)

STEP 3: Prioritize ruthlessly
  If resources constrained to one goal — which one?
  That's Goal #1. Everything else is conditional on #1.

GOAL FORMAT:
  "[Metric] [Direction] [Target] by [Date]"
  "GND revenue: grow to $5,000 MRR by December 31, 2025"
  "Digital product catalog: 12 active products by December 31, 2025"
  "Prayful: beta launch with 100 users by June 30, 2025"
```

## 5. QUARTERLY PLANNING

### The 90-Minute Quarterly Planning Session
```
PART 1 (30 min): Review
  How did last quarter go?
  What moved? What didn't? Why?
  What key learnings must inform this quarter?

PART 2 (30 min): Set
  Write OKRs for this quarter (max 3 per business unit)
  Identify the single most important thing per unit
  Identify the single biggest threat per unit

PART 3 (30 min): Plan
  What must happen in Month 1 for the quarter to succeed?
  What's the first week's priority?
  What's off the plate this quarter? (Must name what we're NOT doing)
```

## 6. PRIORITY FRAMEWORKS

### The Eisenhower Matrix (Daily Filtering)
```
          URGENT        NOT URGENT
IMPORTANT | DO NOW   | SCHEDULE IT
          | (Q1)     | (Q2 — most value)
NOT       | DELEGATE | ELIMINATE
IMPORTANT | (Q3)     | (Q4)

KEY INSIGHT: Most strategic work lives in Q2 (Not Urgent but Important).
Protect Q2 time aggressively. It's where the real value is built.
```

### The "Most Important Thing" Framework
```
Every week, identify ONE most important thing per active workstream.
If that ONE thing happens, the week was a success regardless of anything else.

EXAMPLE:
  GND: "Send 10 cold emails to Parker businesses"
  Products: "Launch Budget Binder"
  Prayful: "Build prayer prompt API"
  
  If all three happen → week wins regardless of everything else done.
```

## 8. STRATEGY FOR ZERO RUNWAY

### Special Rules When Revenue is Urgent
At zero runway, standard strategic planning needs modification:

**Standard rule**: Focus on best long-term opportunity
**Zero runway rule**: Focus on fastest path to revenue FIRST

```
ZERO RUNWAY STRATEGY FILTER:

QUESTION 1: Will this generate revenue in < 30 days?
  YES → Consider prioritizing
  NO → It's a Q2 project (schedule it, don't do it now)

QUESTION 2: What's the single highest-confidence revenue action available?
  Identify it. Do it first. Every day.

QUESTION 3: What existing asset is closest to generating revenue?
  A product that needs a description refresh?
  A client relationship that needs a follow-up email?
  Start there, not with new builds.

ZERO RUNWAY PRIORITY ORDER:
  1. Close or advance any open sales opportunities
  2. Improve conversion of existing products (copy, price, listing)
  3. Launch already-built products that aren't live yet
  4. New product development (only if <2 weeks to revenue)
  5. Platform expansion (only if established products are already live)
```

## 9. TLC STRATEGIC PLANNING CALENDAR

```
DAILY (5 min): Review top 3 priorities for the day. Did yesterday's happen?

WEEKLY (Monday, 30 min): Weekly review + set top priorities
  Run: weekly-review.md

MONTHLY (first Monday, 60 min): Monthly review + strategy check
  Are we on track for quarterly OKRs?
  What needs to change?

QUARTERLY (90 min): Full quarterly review + next quarter planning
  Run: full quarterly planning session

ANNUALLY (half day): Annual review + year goal setting
  What worked? What didn't?
  Set annual goals
```
