# Strategic Memos — Reference Guide

How to write actionable strategic documents, the Monday memo, and the formats
that communicate strategy clearly and drive decisions.

---

## 1. THE MEMO PHILOSOPHY

A strategic memo is not a report. It is not a summary. It is not a knowledge dump.
A strategic memo has one job: enable a specific decision or commitment.

Before writing any memo, answer:
- What decision does this enable?
- Who makes that decision?
- What do they need to read to make it confidently?

If you can't answer these, you don't need a memo.

## 2. MEMO FORMATS

### Format 1: Decision Memo (One-Pager)
```
Use when: A specific decision needs to be made
Length: 1 page maximum

STRUCTURE:
  DECISION NEEDED: [One sentence — what must be decided]
  
  CONTEXT: [2-3 sentences — what's happening that requires this decision]
  
  OPTIONS:
    A) [Option description] — Pros: / Cons: / Revenue impact: / Effort:
    B) [Option description] — Pros: / Cons: / Revenue impact: / Effort:
    
  RECOMMENDATION: [Option A/B] because [one reason — the strongest one]
  
  REQUIRED ACTION: [If Option A: who does what by when]
  
  DEADLINE FOR DECISION: [Date — don't leave it open-ended]
```

### Format 2: Status Memo (Weekly)
```
Use when: Weekly review and priority setting
Length: 1-2 pages
[See weekly-review.md for full format]
```

### Format 3: Strategy Reset Memo
```
Use when: Quarterly planning, after significant setback, or after major drift
Length: 2-3 pages

STRUCTURE:
  SITUATION: [Current state — honest, specific]
  ANALYSIS: [What's working, what isn't, why]
  RECOMMENDATION: [Specific strategic direction]
  PRIORITIES: [Top 3 for next quarter with success metrics]
  WHAT WE'RE STOPPING: [Explicit stop list]
  TIMELINE: [Key milestones]
  DECISION NEEDED: [Any founder input required]
```

## 3. MEMO WRITING PRINCIPLES

### What Makes a Memo Good
```
ONE RECOMMENDATION — Not "here are some options"
  Commit to a point of view. Present alternatives, but recommend one.

BOTTOM LINE FIRST — Lead with the conclusion
  Busy founders read memos in order of priority
  If the conclusion is on page 3, they won't get there

SPECIFIC NOT VAGUE — "Send 10 emails by Wednesday" not "do more outreach"

SHORT — If it takes more than 5 minutes to read, it's too long
  For most memos: 1 page or less

ACTION-ORIENTED — Every memo ends with a specific next step
  Who does what by when
  What decision must be made by when
```

### Red Flags in Strategic Memos
```
"Things to consider..." → Should be a recommendation
"As you know..." → Delete it
"In the current environment..." → Delete it
More than 2 pages → Too long, cut
No recommendation → Not a strategic memo, it's a report
No next step → Not actionable
```
