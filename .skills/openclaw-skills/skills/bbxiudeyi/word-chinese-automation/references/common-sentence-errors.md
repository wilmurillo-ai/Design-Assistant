# 常见病句类型

## 1. 语序不当

词语顺序不合理，影响表达效果。

**多项状语正确顺序：时间 → 地点 → 范围 → 程度 → 状态/方式 → 对象 → 动词**

| 顺序 | 位置 | 示例 |
|------|------|------|
| 1 | 时间 | 昨天、早上、每周一 |
| 2 | 地点 | 在学校、在办公室 |
| 3 | 范围 | 全班、全厂、都 |
| 4 | 程度 | 很、非常、特别 |
| 5 | 状态/方式 | 热情地、认真地、快速地 |
| 6 | 对象 | 同他、跟我、向他 |
| 7 | 动词 | 交谈、学习、工作 |

例：❌ 热情地在休息室里同他交谈 → ✅ 在休息室里热情地同他交谈

## 2. 搭配不当

词语搭配不符合语言习惯或逻辑。重点检查动宾搭配。

例：❌ 赔偿诉讼费 → ✅ 支付诉讼费

## 3. 成分残缺或赘余

缺少必要成分或多加不必要成分。

例：❌ 目的是为了 → ✅ 目的是

## 4. 结构混乱

句式杂糅，两种说法混在一起。

例：❌ 原因是由于... → ✅ 原因是... / 由于...

## 5. 表意不明

指代不明、词义不明确、修饰语歧义。

## 6. 不合逻辑

自相矛盾、否定失当、主客倒置。

例：❌ 基本上完全同意 → ✅ 基本上同意

---

## 快速检查清单

- [ ] **语序不当** — 多项状语顺序是否正确？
- [ ] **搭配不当** — 动宾、主谓搭配是否恰当？
- [ ] **成分残缺或赘余** — 是否缺主语/谓语/宾语？是否有重复？
- [ ] **结构混乱** — 是否句式杂糅？
- [ ] **表意不明** — 是否有歧义？
- [ ] **不合逻辑** — 是否自相矛盾？
