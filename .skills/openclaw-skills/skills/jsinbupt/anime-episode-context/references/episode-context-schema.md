## 5. 推荐使用顺序

按集处理时，优先遵守这个顺序：
1. 第 1 集 TXT -> 产出“第1集初始设定库” + “第1集结束后的继承摘要”
2. 第 2 集 TXT -> 输入“第1集继承摘要” + “第2集 TXT” -> 产出“第2集上下文包”
3. 第 N 集 TXT -> 输入“第 N-1 集继承摘要” + “第 N 集 TXT” -> 产出“第 N 集上下文包”

每一集处理完后，都要再产出一份新的“继承摘要”，供下一集继续使用。

## 6. 防漂移检查清单

在发给模型前，检查：
- 角色名字是否统一
- 发型、发色、服装是否和当前集状态一致
- 是否错误回退到角色默认服装
- 受伤、血迹、灰尘、破损是否被遗漏
- 手里拿的道具是否前后统一
- 人物关系是否沿用最新状态
- 未揭露的秘密是否被提前泄露
