# Statistics Templates

Ready-to-use templates for reports and statistics.

## Customer Project Statistics

```markdown
## Customer Project Statistics

| Status | Count | Projects |
|--------|-------|----------|
| Active | X | A, B, C |
| Blocked | Y | D, E |
| Pending | Z | F |
| Total Active | X+Y+Z | - |

### Tasks in Quadrants (Personal Execution)
| Quadrant | Customer Tasks | Internal Tasks |
|----------|----------------|----------------|
| Q1 | X | Y |
| Q2 | X | Y |
| Q3 | X | Y |
```

## Main Task List Statistics

```markdown
## Statistics

| Quadrant | Count | Task Numbers | Customer | Internal |
|----------|-------|--------------|----------|----------|
| Q1 | X | 1-X | A | B |
| Q2 | Y | (X+1)-(X+Y) | C | D |
| Q3 | Z | ... | E | F |
| Q4 | W | ... | G | H |
| Total | N | - | - | - |
```

## Progress Summary
