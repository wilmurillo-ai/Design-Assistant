## 16 项必备条款核查表

| # | 条款名称 | 关键检查点 | Pass | Fail | 备注 |
|---|---------|-----------|:----:|:----:|------|
| 1 | **研究目的说明** | ✓ 用通俗语言（非专业术语）说明<br>✓ 清晰表述研究性质（不是常规诊疗） | ☐ | ☐ | |
| 2 | **研究流程、步骤、时长、检查项目** | ✓ 详细步骤和顺序说明<br>✓ 每次访视内容和时长<br>✓ 总研究周期明确 | ☐ | ☐ | |
| 3 | **多中心信息** | ✓ 明确说明为多中心试验<br>✓ 参研单位总数<br>✓ 总受试者数<br>✓ 本中心计划入组数 | ☐ | ☐ | |
| 4 | **预期受益** | ✓ 说明可能的受益<br>✓ 未夸大受益（如：无"保证有效""最佳治疗"等表述）<br>✓ 未使用"一定""绝对"等承诺性语言 | ☐ | ☐ | |
| 5 | **可预见风险、不适、不良反应** | ✓ 列出所有可预见风险<br>✓ 列出可能的不适和不良反应<br>✓ 未淡化风险（如：无"无任何副作用"等表述） | ☐ | ☐ | |
| 6 | **替代治疗方案说明** | ✓ 说明当前可选的其他治疗方案<br>✓ 各方案的利弊比较 | ☐ | ☐ | |
| 7 | **自愿参加与退出权利** | ✓ 明确自愿参加<br>✓ 可随时退出且不惩罚<br>✓ 退出不影响正常医疗待遇<br>✓ 未使用"必须""不得"等强迫性语言 | ☐ | ☐ | |
| 8 | **数据隐私与保密措施** | ✓ 说明数据收集范围<br>✓ 保密措施明确<br>✓ 符合个人信息保护法要求<br>✓ 生物样本保密说明（如适用） | ☐ | ☐ | |
| 9 | **费用、补偿与报销** | ✓ 明确受试者相关费用（检查/用药等）<br>✓ 补偿金额或计算方式<br>✓ 报销流程说明 | ☐ | ☐ | |
| 10 | **研究相关损害的处理** | ✓ 发生与研究相关损害的治疗安排<br>✓ 医疗费用承担说明<br>✓ 赔偿/保险说明<br>✓ 联系人与联系方式 | ☐ | ☐ | |
| 11 | **紧急联系人** | ✓ 研究医生姓名<br>✓ 24小时联系电话<br>✓ 研究机构名称/地址 | ☐ | ☐ | |
| 12 | **伦理委员会联系方式** | ✓ 伦理委员会名称<br>✓ 联系电话<br>✓ 通讯地址 | ☐ | ☐ | |
| 13 | **数据与生物样本使用** | ✓ 样本保存地点与期限<br>✓ 二次使用范围与条件<br>✓ 样本销毁方式与时间节点<br>✓ 受试者撤回同意后样本处理 | ☐ | ☐ | |
| 14 | **研究结果反馈** | ✓ 说明研究结束后受试者可获取结果总结<br>✓ 获取方式与时间 | ☐ | ☐ | |
| 15 | **明确研究性质** | ✓ 明确注明"这是研究，不是常规治疗"<br>✓ 与常规医疗的区别说明清晰 | ☐ | ☐ | |
| 16 | **禁止不公平条款** | ✗ 无强迫性表述<br>✗ 无诱导性表述（如：夸大受益、淡化风险）<br>✗ 无免责不公平条款<br>✗ 无"自动参加""不得退出"等陷阱条款 | ☐ | ☐ | |
