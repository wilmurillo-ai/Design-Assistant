# “群众路线 / 一般号召与个别指导”使用护栏

[返回总入口](../../SKILL.md) · [风险路由](../routing/confidence-rules.md) · [误用边界入口](./misuse-boundaries.md) · [表达红线入口](./translation-red-lines.md)

## 这组表达什么时候可用

- 用来处理判断已经形成，但还需要把它接到现场经验、样板和群众参与上的场景。
- 用来说明为什么既不能只发总号召，也不能只靠领导者盯细节。

## 正确含义

- `群众路线` 指从群众经验中提炼，再回到群众中检验，不是迎合、讨好或照单全收。
- `一般号召与个别指导结合` 指先抓出样板和骨干，再把经验翻成一般要求，不是二选一。
- 这组判断的目标是让路线能落地、可复制、可回流纠偏，而不是让领导者更忙。

## 禁止翻译

- 不能翻成“群众说什么就照着做什么”。
- 不能翻成“先发原则口号，现场自然会自己理解”。
- 不能翻成“抓几个样板点做展示，就算完成推广”。

## 运行期检查

- 使用 `群众路线` 时，必须同时说明收集了什么现场经验、怎样做了集中提炼、怎样回到现场检验。
- 使用 `一般号召与个别指导结合` 时，必须同时写出一般要求和样板抓手分别是什么。
- 输出里既要防口号空转，也要防只抓个案、不形成复制规则。
