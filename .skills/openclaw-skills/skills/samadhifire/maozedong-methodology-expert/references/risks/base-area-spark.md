# “根据地 / 星星之火”使用护栏

[返回总入口](../../SKILL.md) · [风险路由](../routing/confidence-rules.md) · [误用边界入口](./misuse-boundaries.md) · [表达红线入口](./translation-red-lines.md)

## 这组表达什么时候可用

- 用来处理整体弱势、多线受压、必须先站住再谋扩张的局面。
- 用来说明为什么小支点值得经营，但前提是它能守、能长、能供给。

## 正确含义

- `根据地` 指可守、可长、可供给的基本盘，不是消极退守或躲到边缘。
- `星星之火，可以燎原` 指在结构裂缝、支持基础和供给条件存在时经营小支点，不是逆风期通用励志话。
- 这组判断的目标是先建可持续支点，再从支点放大影响，不是靠口号熬过去。

## 禁止翻译

- 不能翻成“先退一步，总会有机会”。
- 不能翻成“只要有一点希望，就一定能燎原”。
- 不能翻成“做个小试点，就天然会扩散成全局优势”。

## 运行期检查

- 使用 `根据地` 时，必须同时写出它为什么能守、能长、能供给。
- 使用 `星星之火` 时，必须同时写出结构裂缝、支持基础和扩散条件是什么。
- 输出里既要防盲目悲观，也要防无条件乐观。
