# “集中优势兵力 / 各个歼敌”使用护栏

[返回总入口](../../SKILL.md) · [风险路由](../routing/confidence-rules.md) · [误用边界入口](./misuse-boundaries.md) · [表达红线入口](./translation-red-lines.md)

## 这组表达什么时候可用

- 用来处理资源有限、必须形成局部压强的多线竞争局面。
- 用来说明为什么不能平均分兵，而要围绕关键节点分段收口。

## 正确含义

- `集中优势兵力` 指先选对象、再选薄弱点、再形成局部压强，不是单纯堆人堆钱。
- `各个歼敌` 指按关键节点分段收口、逐点改写局面，不是幻想一轮总清场。
- 这组判断的目标是制造局部优势并滚动改写棋盘，不是赌一次 all-in。

## 禁止翻译

- 不能翻成“把所有资源都压到最显眼的地方”。
- 不能翻成“只要集中资源，就一定能打穿”。
- 不能翻成“先赢一场大仗，其他问题自然都会消失”。

## 运行期检查

- 使用 `集中优势兵力` 时，必须同时说明对象是谁、薄弱点在哪里、为什么值得压主力。
- 使用 `各个歼敌` 时，必须同时写出哪些线主攻、哪些线牵制、哪些线守底盘。
- 输出里要保留后方供给和承压校验，防止前台胜利无法持续。
