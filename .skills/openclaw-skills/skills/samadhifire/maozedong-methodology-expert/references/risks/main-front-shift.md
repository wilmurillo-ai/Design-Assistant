# “主战场 / 工作重心转移”使用护栏

[返回总入口](../../SKILL.md) · [风险路由](../routing/confidence-rules.md) · [误用边界入口](./misuse-boundaries.md) · [表达红线入口](./translation-red-lines.md)

## 这组表达什么时候可用

- 用来判断当前资源最该集中的中心舞台。
- 用来判断阶段变化后，组织角色、能力结构和任务中心是否需要换挡。

## 正确含义

- `主战场` 指当前最该集中资源的中心舞台，不是把所有问题都战争化或紧急化。
- `工作重心转移` 指角色换挡、能力补课和组织接口重写，不是追热点式转向。
- 这组判断的目标是重排资源与组织动作，而不是制造气氛化口号。

## 禁止翻译

- 不能翻成“凡是重要的事，都是主战场”。
- 不能翻成“既然换重心，就把旧基础全部丢掉”。
- 不能翻成“只要换个口号，就算完成转移”。

## 运行期检查

- 使用 `主战场` 时，必须同时说明为什么其他战场当前不是资源中心。
- 使用 `工作重心转移` 时，必须同时写出哪些旧能力仍保留、哪些新能力必须补。
- 输出要能落到资源重排、接口重写或角色换挡，而不是停在口号层。
