# “宣传 / 统一口径”使用护栏

[返回总入口](../../SKILL.md) · [风险路由](../routing/confidence-rules.md) · [误用边界入口](./misuse-boundaries.md) · [表达红线入口](./translation-red-lines.md)

## 这组表达什么时候可用

- 用来处理需要公开解释、组织动员、内部对外口径协同的场景。
- 用来说明为什么表达既要能传播，也要能和事实、对象、行动接口对齐。

## 正确含义

- `宣传` 指公共解释、组织动员和教育，不是虚构、操控或情绪煽动。
- `统一口径` 指基于校准后的对外表达一致，不是禁止内部辨析和反馈进入。
- 这组判断的目标是让表达可传播、可执行、可纠偏，而不是只求声量整齐。

## 禁止翻译

- 不能翻成“宣传就是把话说得更会煽动”。
- 不能翻成“统一口径就是内部不许再讨论”。
- 不能翻成“只要大家说一样的话，就说明判断已经一致”。

## 运行期检查

- 使用 `宣传` 时，必须同时说明对象是谁、要解释什么、要接到什么行动或教育任务。
- 使用 `统一口径` 时，必须同时写出哪些事实和边界已经校准，哪些内部分歧仍允许继续辨析。
- 输出既要防套话和黑话，也要防压反馈造成的失真。
