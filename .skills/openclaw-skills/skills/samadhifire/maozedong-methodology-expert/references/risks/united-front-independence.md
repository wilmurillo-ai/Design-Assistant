# “统一战线 / 独立自主”使用护栏

[返回总入口](../../SKILL.md) · [风险路由](../routing/confidence-rules.md) · [误用边界入口](./misuse-boundaries.md) · [表达红线入口](./translation-red-lines.md)

## 这组表达什么时候可用

- 用来处理多方局势里围绕共同任务建立合作、但又不能丢掉主体和边界的场景。
- 用来说明为什么既要联合，又不能在联合中丢掉自己的判断和组织能力。

## 正确含义

- `统一战线` 指围绕共同任务建立有边界的合作，不是谁都拉进来的泛团结。
- `独立自主` 指合作中保留自己的判断、组织和行动空间，不是拒绝协作的单干。
- 这组判断的目标是让联盟可进入、可承接、可持续，而不是只停在表态层。

## 禁止翻译

- 不能翻成“只要目标差不多，内部差异都可以先不管”。
- 不能翻成“为了保持独立，就不要深度合作”。
- 不能翻成“只要达成高层共识，联盟自然会自己运转”。

## 运行期检查

- 使用 `统一战线` 时，必须同时说明共同任务是什么、边界画到哪里。
- 使用 `独立自主` 时，必须同时写出哪些判断、组织能力和行动权限不能交出去。
- 输出要能落到真实参与入口、权责安排或监督机制，而不是停在姿态层。
