# “合法性”使用护栏

[返回总入口](../../SKILL.md) · [风险路由](../routing/confidence-rules.md) · [误用边界入口](./misuse-boundaries.md) · [表达红线入口](./translation-red-lines.md)

## 这句话什么时候可用

- 用来处理公开争论、政策发布、危机回应或规则正当性被质疑的场景。
- 用来说明为什么某个判断、行动或安排值得获得公共支持与服从。

## 正确含义

- `合法性` 不是单靠名义、位置或程序外壳自动成立。
- 它要同时落到共同规则、现实贡献、后果承担、公开证据和程序有效性上。
- 正确动作是用可检验标准争主动，而不是抢词、抢帽子或先喊谁更正当。

## 禁止翻译

- 不能翻成“只要程序上过了，就天然有合法性”。
- 不能翻成“只要站在高位发声，就自动占有公共判断权”。
- 不能翻成“只要给对方扣帽子，自己的合法性就成立”。

## 运行期检查

- 使用 `合法性` 时，必须同时给出规则依据、现实贡献或后果承担中的至少一组硬支撑。
- 公开表达里要有可检验要求、公开证据或责任链，而不是只剩姿态。
- 对手公开材料如果能反证其失效，应优先转成自己的教育和说明资源，而不只停在情绪反驳。
