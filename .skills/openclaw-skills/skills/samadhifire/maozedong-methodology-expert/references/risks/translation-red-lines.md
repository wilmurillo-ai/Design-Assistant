# 现代翻译禁区

[返回总入口](../../SKILL.md) · [风险路由](../routing/confidence-rules.md) · [误用边界入口](./misuse-boundaries.md)

## 文件作用

- 这张入口页只收口一种风险：高风险表达不能怎么翻、不能怎么口语化、不能怎么直接平移到现代日常语境。
- 如果你已经知道问题不在“用错时机”，而在“这句话本身很危险、很容易被误解”，先看这里。
- 如果风险主要是方法扩大化、错位调用、把高压方法拿去打轻问题，请转去 [misuse-boundaries.md](./misuse-boundaries.md)。

## 快速判断：什么该进这里

- 一句话如果离开原方法结构就容易变味，就该先进这里。
- 一句话如果最容易被误读成口号、立场、身份审查、操控术或清洗术，也该先进这里。
- 一句话如果问题主要在“怎么翻”而不是“什么时候用”，优先挂这里。

## 最容易翻坏的方向

### 1. 翻成口号

- 例如把 `主要矛盾`、`统一战线`、`群众路线`、`实践检验` 直接翻成一句空洞态度。

### 2. 翻成身份审查或权力话

- 例如把 `没有调查，没有发言权` 翻成资格封口。
- 例如把 `顽固势力` 翻成“我不喜欢的人”。

### 3. 翻成操控术或包装术

- 例如把 `宣传` 翻成操控舆论。
- 例如把 `统一口径` 翻成压制反馈。
- 例如把 `合法性` 翻成抢帽子。

### 4. 翻成极端化行动口号

- 例如把 `集中优势兵力` 翻成 all-in。
- 例如把 `主战场` 翻成所有问题都紧急化。
- 例如把 `近胜期` 翻成自动收官。

### 5. 翻成清洗术或和稀泥

- 例如把 `整风` 翻成运动式清洗。
- 例如把 `治病救人` 翻成不碰问题。

## 使用顺序建议

1. 先判断风险来自“表达本身”，还是“调用方式本身”。
2. 如果是表达本身，先在这张页里按方法族定位。
3. 再去对应独立风险卡看正确含义、禁止翻译和运行期检查。
4. 如果一个问题同时有翻译风险和调用风险，先看这里，再回 [misuse-boundaries.md](./misuse-boundaries.md)。
