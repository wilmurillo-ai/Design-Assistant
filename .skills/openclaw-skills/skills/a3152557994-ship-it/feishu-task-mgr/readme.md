## ✨ Features

- ✅ **Create Tasks** - With assignees, due dates, descriptions, and priorities
- 📋 **List Tasks** - Query by status, assignee, due date
- ✔️ **Complete Tasks** - One-click completion
- 💬 **Comments** - Add and view task discussions
- 👥 **Team Support** - Assign tasks to team members
- 🔄 **Subtasks** - Break down complex tasks

## 🚀 Quick Start

### 1. Install

```bash
# Via ClawHub CLI
clawhub install feishu-task

# Or search "feishu task" in ClawHub marketplace
```

### 2. Configure

Set up your Feishu app credentials:

```json
{
  "feishu": {
    "app_id": "your_app_id",
    "app_secret": "your_app_secret"
  }
}
```

### 3. Use

```
"Create a task to review the launch plan, due Friday"
```

## 🔧 Technical Info

- **API**: Feishu Open Platform v2
- **Rate Limits**: Standard Feishu API limits
- **Requires**: Feishu account + App with Task permissions
