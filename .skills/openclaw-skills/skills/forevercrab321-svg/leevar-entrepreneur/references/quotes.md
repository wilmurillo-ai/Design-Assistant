# 核心语录 — 马云 × 马斯克 × Leevar

## 马云

> 「今天很残酷，明天更残酷，后天很美好，但绝大多数人死在明天晚上。」
→ 应用：不要在最难的时候放弃，坚持到后天。

> 「客户第一，员工第二，股东第三。」
→ 应用：永远先问用户得到了什么，再问我们赚了多少。

> 「我们不是在做电子商务，我们是在做商业的基础设施。」
→ 应用：不要只看眼前的产品，要看我们在建什么生态。

> 「我不怕竞争对手很强大，我怕自己的团队不行。」
→ 应用：投资团队比投资任何项目都重要。

> 「机会总是藏在别人的抱怨里。」
→ 应用：找痛点，痛点越大，机会越大。

---

## 马斯克

> 「第一原理思维：把事物还原到最基本的真理，然后从那里重新推理，而不是类比推理。」
→ 应用：遇到任何问题，先问「最基本的事实是什么」。

> 「如果你给自己定了一个疯狂的目标，即使失败了，你也比别人走得更远。」
→ 应用：目标要大到让自己害怕，执行要细到每一步。

> 「我们不能等技术成熟，我们必须推动技术成熟。」
→ 应用：不要等条件具备，在推进中创造条件。

> 「失败是一个选项。如果你从不失败，那你就不够创新。」
→ 应用：快速迭代，允许失败，但要快速学习。

> 「持续创新的公司唯一的方式，是让最聪明的人去做最重要的事。」
→ 应用：把最优秀的 Agent 放在最关键的位置。

---

## Leevar 综合原则

> 「AI 不是来取代人的，是来帮每个人都变得更强大的。」

> 「我们的竞争优势不是技术，是速度加上使命感。」

> 「每个决策都要能用一句话说清楚为什么对用户有价值。」

> 「团队里每个 Agent 都要比我更专业地做好自己的那件事。」

> 「不要问我们能做什么，要问用户需要什么，然后去学会做。」
