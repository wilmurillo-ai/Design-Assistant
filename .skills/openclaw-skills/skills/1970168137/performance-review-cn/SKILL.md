# 绩效评估

## 概述

本SKILL提供专业的绩效能力，帮助企业简化运营流程并提升业务成果。

## 核心能力

- 绩效面谈报告、OKR对齐度检测、校准辅助
- 行业最佳实践和标准化工作流程
- 数据驱动的洞察和建议
- 合规性和风险管理考量

## 应用场景

### 主要应用
- 业务运营优化
- 战略决策支持
- 流程自动化和标准化
- 绩效监控和改进

### 行业应用
- 企业级实施方案
- 中小企业解决方案
- 跨职能团队协作
- 供应商和合作伙伴管理

## 输入要求

**COPY以下内容并粘贴输入：**

```
[请在此处提供您的具体业务背景、数据或需求]
```

### 必需信息
- 业务背景和目标
- 相关数据或文档
- 具体要求或约束条件
- 目标受众或利益相关方

### 可选信息
- 历史数据或基准
- 竞争格局
- 监管要求
- 预算和时间限制

## 输出格式

本SKILL将提供：

1. **分析与洞察** - 数据驱动的观察和建议
2. **可交付成果** - 可直接使用的文档、脚本或计划
3. **实施指导** - 分步执行说明
4. **最佳实践** - 行业标准和成熟方法论

## 最佳实践

- 始终提供完整的上下文以获得最佳结果
- 根据业务特定需求审查输出内容
- 根据您的特定行业标准验证建议
- 基于反馈和绩效指标进行迭代优化

## 使用限制

- 需要准确和完整的输入数据
- 输出内容应由领域专家审查
- 高度专业化行业可能需要定制
- 不能替代专业法律、财务或医疗建议
