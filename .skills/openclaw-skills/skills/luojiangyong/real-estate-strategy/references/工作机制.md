# 产策方案撰写框架

## 标准五章结构（项目启动会汇报模板）

### 第一章：项目概况

**1.1 区位及配套**
- 区位定位（城市/板块/地块坐标，S/A/B/C 级判断）
- 上游板块 / 下游板块关系
- 周边配套成熟度（商业/医疗/教育/交通）
- 与同品牌现有项目的区位关系

**1.2 四至及规划**
- 地块四至边界
- 周边道路/规划条件
- 轨道交通（步行距离到最近站点）

**1.3 不利条件**
- 地质条件（是否有水坑/软土/特殊地质）
- 周边不利因素（噪音/高压线/污染设施）
- 场地标高与地下室关系

**1.4 土地价值判断**
- 外部属性总结（10大指标）
- 内部属性总结（规划/用地/现状）
- 外联机会 + 内拓潜力

---

### 第二章：经营策略及目标

**2.1 集团/区域经营策略**
- 该城市/区域战略定位
- 品牌布局（在售项目、待入市项目）
- 拿地目标与战略匹配度

**2.2 目标成本**
- 目标成本适配策略
- 地下室/基础方案成本处理方式
- 奖励政策争取情况

**2.3 开发节奏目标**
- 分批次开发计划
- 首开批次目标（去化率/利润要求）
- 关键节点（开工/取证/首开/续销）

---

### 第三章：客群分析及方案推导

**3.1 客群定位（三层）**

| 层次 | 内容 |
|------|------|
| 地缘客群 | 周边原有居民、板块内换房需求 |
| 辐射客群 | 可控通勤时间内可覆盖的外区客群 |
| 溢出客群 | 主城区低供高价板块外溢客户 |

**3.2 竞品客群分析**
- 标杆竞品客户来源（板块/年龄/职业/动机）
- 竞品成交客户特征标签

**3.3 目标客群画像**
- 家庭结构 / 职业 / 年龄段
- 核心需求标签（功能性需求 vs 情感性需求）
- 价格承受力（总价段 / 单价承受上限）

**3.4 客户语录法（定性研究具象化）**
收集真实客户原话，用于支撑产品设计判断：
- 客户对风格的期待（如"园林有层次感、有设计感"）
- 客户对抗性的在意点（如"等电梯太久"）
- 客户对配套的期待（如"会所实用，不想出门"）

---

### 第四章：项目整体策划

**4.1 产品规划方案**
- 产品类型组合（高层/小高/洋房/叠拼/合院等）
- 户型配比（面积段 × 房间数）
- 价格定位（均价 / 总价段）
- 差异化亮点

**4.2 户型设计原则**
- 功能实用性优先
- 面宽与功能空间的取舍原则（例：大面宽次卧 > 明卫）
- 关键空间尺度（厨房/主卧/客厅）
- 特色场景打造（如洄游动线、LDKB一体化）

**4.3 产品配置标准**
- 精装标准 / 公区配置
- 社区会所功能（泳池/健身/儿童/宠物区等）
- 园林风格定位

**4.4 商业设计**
- 商业规模及定位
- 街区商业 / 底商 / 独立商业
- 商业与住宅的动线关系

**4.5 社区底盘**
- 总图排布原则（楼栋朝向/间距/日照）
- 中央景观轴线
- 归家动线设计

## 竞品分析规范

### 竞品分层（价格/产品线）

| 层级 | 判断 | 代表竞品 |
|------|------|---------|
| 上游 | 地价更高，产品为高改/豪宅 | 典型高改项目 |
| 同级 | 地价相近，产品相似 | 主要竞争项目 |
| 下游 | 地价更低，以首置/刚改为主 | 拦截型项目 |

### 竞品分析维度

```
竞品名称 | 参拍地价 | 主力产品 | 主力面积段 | 近三月均价 | 月均流速
```

### 竞争格局判断
- 有效竞争范围（板块内 + 辐射板块）
- 差异化机会点
- 市场容量测算（年均成交量 / 去化周期）

---

## 六大风险排查（投前必查）

| 风险类型 | 关注要点 |
|---------|---------|
| 设计条件 | 容积率/限高/密度是否损容 |
| 用地风险 | 日照/通廊/退距是否制约排布 |
| 地质风险 | 是否有水坑/软土/特殊地质（成本影响大）|
| 信息跟踪 | 报建/施工/验收各环节政策差异 |
| 成本适配 | 地下室不利因素处理（清淤/桩基）是否充分考虑 |
| 奖励政策 | 装配式/绿建/海绵城市等奖励是否可争取 |
