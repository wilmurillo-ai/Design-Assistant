# 投前产策两大重点

## 重点一：排查风险

### 六大风险排查项

**设计条件审查**
| 风险项 | 说明 |
|--------|------|
| 可建设用地面积 < 3万㎡ | 影响产品排布 |
| 容积率（折算100米限高）> 6.0（湾区）/5.0（非湾区） | 可能损容 |
| 分区限高（60米）> 3.5（湾区）/3.0（非湾区） | 产品品质受限 |
| 24米限高 > 1.5（湾区）/1.2（非湾区） | 楼栋排布受限 |

**用地风险排查**
- 日照要求（大寒日三小时 > 冬至日一小时）
- 视觉通廊
- 建筑密度 + 绿地率 > 70%
- 楼栋排布受限、朝向认定规则

**信息跟踪**
- 工程报建：落实报批报建流程及要求，提前摸查审批力度、报建难度
- 施工限制：各城市政策差异（如杭州商业配套管控）
- 验收要求：住建/消防、国安/军事、交通/环保/水务/教育/公交/卫生等
