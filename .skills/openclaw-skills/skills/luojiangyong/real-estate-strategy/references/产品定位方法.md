# 产品定位四要素

**土地 + 市场 + 客户 = 产品方案**

## 一、读懂土地

### 基本分析要素

**外部属性**（8类）：人口 / 交通 / 商业 / 商务 / 教育 / 医疗 / 休闲 / 景观 / 产业 / 劣势

**内部属性**：规划条件 / 用地条件 / 现状条件

### 价值挖掘方法

- **外联**：联动积极要素，挖掘土地潜力
- **内拓**：通过产品打造，改变土地属性

---

## 二、读懂市场

### 三个层面

| 层面 | 核心问题 |
|------|---------|
| 城市层面 | 看发展水平 |
| 板块层面 | 看供需情况 |
| 竞品层面 | 看竞争格局 |

### 分析目标

- **了解趋势**：市场水平和产品趋势
- **寻找机会**：市场需求和产品机会

---

## 三、读懂客户

### 定性研究（发现）

观察 → 发现

### 定量研究（验证）

度量 → 验证

### 定性五步法

1. 找对人
2. 问对题
3. 问好题
4. 找对点
5. 用对点

---

## 四、产品方案

在土地、市场、客户三要素客观分析基础上，形成产品方案。
