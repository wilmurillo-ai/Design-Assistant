# 产业园类案例

**适用项目特征**：以产业功能为主的园区开发，包含创意产业园、科创园、工业设计园、文创综合体、孵化器等；目标客群是企业/创业者/创意工作者而非居住用户；叙事主轴是产业价值与城市使命，而非居住体验。

**常见场景**：工业遗存改造型产业园、新建科创园区、TOD产业综合体、文创聚落、创意办公园区。

**故事线写作重点**：
- 以「产业使命」切入，而非空间美学或硬件配置
- 强调与目标企业/人群的价值观对话
- 国际案例参照应选创意/科技产业地标（苏荷、巴特西、King's Cross等）
- 核心命题往往是：「这里，是一类人重新定义城市的地方」

**视觉符号偏好**：发条/引擎（驱动感）、工业美学符号（铁架、旧砖、锈迹）、创意爆发意象、人群聚集
