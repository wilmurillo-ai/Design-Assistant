# 企业品牌类案例

**适用项目特征**：以开发商/房企品牌整体为叙事主体，而非单一项目；目标是建立或升级品牌认知、价值观传递、品牌焕新；输出物常为品牌故事、品牌宣言、品牌主题片、年度品牌战役等。

**常见场景**：房企品牌升级/焕新、跨城市/跨产品线的品牌统一叙事、开发商进入新城市的品牌破冰战役、企业周年纪念叙事、开发商与城市的关系表达。

**故事线写作重点**：
- 叙事主体是「企业」，不是某个楼盘，语言更宏观、更具战略感
- 强调企业价值观与时代精神的共鸣，而非产品功能陈述
- 常以「我们相信……」「我们选择……」的主体性语言建立品牌人格
- 国际参照选品牌战役案例（苹果Think Different、耐克Just Do It的叙事逻辑）
- 核心命题往往是：「这家公司，为什么做这件事，为什么值得信任」

**视觉符号偏好**：企业标志性符号的意象化延伸、时间轴/历程感、城市全景、人与城市的关系
