# 城市或片区营造类案例

**适用项目特征**：以城市板块、片区、街区、新城为叙事主体，而非单一楼盘；开发商或政府以城市共建者角色出现；项目承载城市发展使命，叙事重心在于城市价值升维而非产品功能。

**常见场景**：城市更新片区、特色小镇、历史街区活化、新城开发启动区、滨水公共岸线、城市文化地标区域。

**故事线写作重点**：
- 以城市/片区为主角，项目是城市故事的载体而非主角
- 强调「时代节点感」——为什么是现在，为什么是这里
- 引用城市规划、政策文件、历史典故作为叙事锚点
- 核心命题往往是：「这座城市，在这里完成了某种蜕变」

**视觉符号偏好**：城市天际线、历史地图叠加、河流/脉络意象、灯光/夜景、人群流动
