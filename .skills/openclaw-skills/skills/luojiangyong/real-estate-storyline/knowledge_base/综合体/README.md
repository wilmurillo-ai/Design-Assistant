# 综合体类案例

**适用项目特征**：住宅+商业+办公+公寓等多业态混合的大型开发项目；单一叙事维度无法覆盖全部客群；需要建立统一的项目精神内核，再向各业态分别延伸表达。

**常见场景**：城市级大型综合体（含住宅/商业/写字楼/酒店）、TOD综合体、城市更新型综合体、超级大盘（含多个组团/业态）。

**故事线写作重点**：
- 先建立「城市母题」作为所有业态的精神公分母，再分别延伸
- 处理好「整体叙事」与「各业态差异化叙事」的层级关系
- 强调综合体对城市生态的整体贡献，而非各业态的简单相加
- 国际参照选超级综合体案例（Hudson Yards、新加坡Marina Bay、上海ifc）
- 核心命题往往是：「这里，是一座城市的缩影，也是未来生活的预演」

**视觉符号偏好**：城市切片、垂直维度（从地下到天空）、业态间的流动与连接、全天候生活场景
