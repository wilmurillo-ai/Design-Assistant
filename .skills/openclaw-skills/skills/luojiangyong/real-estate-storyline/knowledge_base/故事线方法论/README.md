# 故事线方法论

## 说明

本分支沉淀地产品牌故事线创作的核心方法论，供生成故事线时调用参照。与案例库不同——案例库提供「做了什么」的参照，方法论库提供「怎么想」的框架。

**调用时机**：在检索案例库之后、开始写作之前，读取与当前项目类型最相关的方法论文件，用于指导叙事结构与切入角度的选择。
