# 商业类案例

**适用项目特征**：以商业运营为主的地产项目，包含购物中心、商业街区、城市综合体商业部分、零售地产等；叙事目标既面向消费者（招揽客流）也面向招商对象（吸引品牌入驻）。

**常见场景**：区域级购物中心开业、商业街区品牌焕新、城市商业地标开幕、商业TOD配套。

**故事线写作重点**：
- 双向叙事：消费者向（体验、生活方式、向往感）+ 招商向（商业价值、客群质量、区位势能）
- 强调「消费场景」的具体可感，而非功能罗列
- 国际案例参照选消费地标（K11、新天地、太古里系列等）
- 核心命题往往是：「这里，重新定义了这座城市的消费美学」

**视觉符号偏好**：人流、光影、消费场景、城市生活切片、夜经济
