# NutriCoach - 奢华精致风格设计

## 设计方向

**风格**: 奢华精致 (Luxury/Refined)  
**特点**: 微妙动画、高级字体、克制配色、精致细节

## 色彩系统

```css
:root {
  /* 主色调 - 深墨绿 + 香槟金 */
  --bg-primary: #0d1f1c;      /* 深墨绿背景 */
  --bg-secondary: #152b26;    /* 次级背景 */
  --bg-card: #1a332e;         /* 卡片背景 */
  --bg-elevated: #223d37;     /* 悬浮背景 */
  
  /* 文字 */
  --text-primary: #f5f5f0;    /* 米白主文字 */
  --text-secondary: #a8b5b0;  /* 灰绿次文字 */
  --text-muted: #6b7b76;      /* 弱化文字 */
  
  /* 强调色 - 香槟金 */
  --accent: #d4af37;          /* 香槟金 */
  --accent-hover: #e5c158;    /* 亮金 */
  --accent-subtle: rgba(212, 175, 55, 0.1); /* 金色透明 */
  
  /* 功能色 */
  --success: #4ade80;
  --warning: #fbbf24;
  --danger: #f87171;
  
  /* 渐变 */
  --gradient-gold: linear-gradient(135deg, #d4af37 0%, #f4e5c2 50%, #d4af37 100%);
  --gradient-dark: linear-gradient(180deg, #0d1f1c 0%, #152b26 100%);
}
```

## 字体选择

**标题字体**: Playfair Display (衬线，优雅)  
**正文字体**: Plus Jakarta Sans (几何无衬线，现代)  
**数字字体**: DM Mono (等宽，精致)

## 布局原则

- **大量留白**: 元素间距 24px-48px
- **非对称平衡**: 左侧重，右侧轻
- **层次感**: 微妙的阴影和边框
- **网格系统**: 12列网格，最大宽度 1200px

## 动画效果

- **入场动画**: 淡入 + 轻微上移 (0.3s ease-out)
- **悬停效果**: 金色边框发光 (0.2s)
- **数字变化**: 平滑计数动画
- **图表动画**: 线条绘制效果

## 组件设计

### 卡片
- 背景: `--bg-card`
- 边框: 1px solid rgba(212, 175, 55, 0.1)
- 圆角: 12px
- 阴影: 0 4px 24px rgba(0, 0, 0, 0.2)
- 悬停: 边框变为金色

### 按钮
- 主按钮: 金色渐变背景，深色文字
- 次按钮: 透明背景，金色边框
- 圆角: 8px
- 内边距: 12px 24px

### 图表
- 线条: 金色渐变
- 填充: 金色透明渐变
- 网格: 极淡的金色线条
- 数据点: 金色圆点带发光

## 最近UI优化

### 编辑食材面板
- **分类+位置并排**: 两列网格布局，节省空间
- **生产日期+保质期并排**: 1.5:1 比例，更紧凑
- **使用区域简化**: 移除"记录使用"标签，直接显示输入框+按钮
- **删除按钮**: 底部红色危险按钮，带确认对话框

### 头部优化
- **标题+用户名并排**: 一行显示，更精致
- **用户徽章**: 金色边框标签，等宽字体
- **紧凑高度**: 减少padding，更简洁

### 食材列表
- **过期标签简化**: "已过期"/"今天"/"明天"/"X天"
- **自适应食材名**: 防止换行导致按钮错位
- **编辑按钮**: 金色渐变，精致悬停效果
