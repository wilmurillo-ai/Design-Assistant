一个魂们请进！手把手教你做 A-SOUL 自动签到工具（免费 不用装软件 每天自动跑）

先说说这个工具能做什么：

每天自动帮你去嘉然、贝拉、乃琳、心宜、思诺的直播间签到，自动佩戴对应的粉丝牌再签到，相当于每天白赚亲密度。
可选功能：自动给新发的视频点赞、给动态点赞。

完全免费，不需要安装任何软件，不需要会编程，全程只要一个浏览器就能搞定。

----

下面是详细教程，跟着做就行：

【准备工作】

你需要一个 GitHub 账号（github.com）
如果没有就注册一个，免费的，邮箱就能注册

----

【第 1 步】复制项目到你自己的账号

1. 打开这个地址：
   https://github.com/zhangbo/asoul-support

2. 点击页面右上角的「Fork」按钮

3. 弹出来的页面什么都不用改，直接点绿色的「Create fork」

4. 等几秒钟就好了，你会看到你自己账号下多了一个 asoul-support 项目

----

【第 2 步】获取你的 B 站 Cookie

这一步是为了让工具用你的身份去签到。

1. 用 Chrome 浏览器打开 bilibili.com
2. 确保你已经登录了 B 站
3. 按键盘上的 F12 打开开发者工具
   - Mac 用户按 Cmd + Option + I
4. 在弹出的面板顶部，找到「Application」标签并点击
   - 如果看不到就点 >> 展开更多标签
5. 左边会有一列菜单，找到「Cookies」，点开它，再点「https://www.bilibili.com」
6. 右边出现一个列表，你需要找到两个东西：

   SESSDATA  —— 一长串字母数字加符号的字符串
   bili_jct   —— 32位的字母数字

   双击它们的 Value 列就可以复制

注意：这两个东西相当于你的登录密码，千万不要发到群里或者给别人看！

----

【第 3 步】把 Cookie 填到你的项目里

1. 打开你 Fork 后的项目页面（你自己账号下的 asoul-support）
2. 点页面顶部的「Settings」
3. 左边菜单找到「Secrets and variables」，点一下展开
4. 点「Actions」
5. 点右边的「New repository secret」

6. 填第一个：
   Name 那一栏填：SESSDATA
   Secret 那一栏填：你刚才复制的 SESSDATA 的值
   然后点「Add secret」

7. 再点一次「New repository secret」，填第二个：
   Name 那一栏填：BILI_JCT
   Secret 那一栏填：你刚才复制的 bili_jct 的值
   然后点「Add secret」

填进去之后你自己都看不到了（加密存储的），这是正常的。

----

【第 4 步】启用自动运行

1. 点顶部的「Actions」标签
2. 你会看到一个黄色的提示条
3. 点「I understand my workflows, go ahead and enable them」
4. 完成！

----

【验证一下是不是成功了】

1. 还是在 Actions 页面
2. 左边选「A-SOUL 每日应援」
3. 点右边的「Run workflow」按钮，再点绿色的「Run workflow」
4. 等大概 30 秒，页面上会出现一条运行记录
5. 点进去，展开「直播间签到」那一步
6. 如果看到「签到成功」的字样，就说明全部搞定了！

之后每天北京时间 10:30 左右会自动帮你签到，你不需要做任何事情。

----

【想开视频自动点赞？】

默认只做直播间签到。如果你还想自动给新视频点赞：

1. 在你的项目里找到这个文件：
   .github/workflows/daily.yml

2. 点右上角的铅笔图标开始编辑

3. 找到这一行：
   ENABLE_VIDEO_LIKE: 'false'
   把 false 改成 true

4. 点右上角「Commit changes」保存就行了

动态点赞同理，找到 ENABLE_DYNAMIC_LIKE 那一行改成 true 就可以。

----

【常见问题】

问：安全吗？Cookie 会不会泄露？
答：GitHub Secrets 是加密存储的，填进去之后连你自己都看不到明文，别人更看不到。仓库作者也无法获取你的 Cookie。所有代码完全开源，可以自己检查。

问：Cookie 过期了怎么办？
答：大概一个月左右会过期，到时候重新按第 2 步获取新值，然后去 Settings - Secrets 里更新一下就好。过期了不会有任何损害，只是签到会失败，GitHub 会发邮件通知你。

问：怎么知道每天有没有成功？
答：进入 Actions 页面，每次运行都有记录。绿色勾 = 成功，红叉 = 失败。失败时 GitHub 会自动给你发邮件。

问：完全免费吗？
答：对，GitHub Actions 对公开仓库完全免费，没有任何收费。

问：这个工具会乱发弹幕/关注别人/改我的设置吗？
答：不会。只做签到弹幕和点赞操作，代码全部开源可以检查。

问：我想改签到时间怎么办？
答：编辑 .github/workflows/daily.yml 文件，修改 cron 那行的时间。注意用的是 UTC 时间，北京时间要减 8 小时。比如想每天下午 3 点跑，就写 cron: '0 7 * * *'
