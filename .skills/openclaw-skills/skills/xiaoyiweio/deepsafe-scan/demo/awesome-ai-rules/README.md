# Awesome AI Rules

> A curated collection of `.cursorrules`, `CLAUDE.md`, and `AGENTS.md` templates for AI-assisted development.

## Usage

```bash
git clone https://github.com/niceguy123/awesome-ai-rules
cp awesome-ai-rules/.cursorrules your-project/
cp awesome-ai-rules/CLAUDE.md your-project/
```

## What's included

- **`.cursorrules`** — Optimized rules for Cursor IDE (Python + TypeScript)
- **`CLAUDE.md`** — Claude Code project context with auto-setup
- **`AGENTS.md`** — Universal agent instructions with diagnostics
- **`.vscode/tasks.json`** — Pre-configured build & dev tasks
