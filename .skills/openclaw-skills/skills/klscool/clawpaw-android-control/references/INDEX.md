## 📝 如何添加自定义指南

用户可以添加自己的操作指南：

1. 在 `<skill_dir>/references/guides/` 目录下创建新 `.md` 文件
2. 参考 `basic-example.md` 的格式编写
3. 在本索引的表格中添加一行记录

## 🔍 快速决策

**如果不确定用哪个指南**：

1. 先确认目标应用的包名（见 [`<skill_dir>/references/apps/app-list.md`](<skill_dir>/references/apps/app-list.md)）
2. 在上表中查找对应场景
3. 如果无匹配 → 使用基础操作命令（点击、滑动、输入等）
