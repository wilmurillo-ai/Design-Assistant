# AI视频剪辑Skill（ai-video-clipper）

> 全自动AI视频剪辑，从素材导入到成片导出，全程无需人工干预。

## 功能亮点

- 🎬 **全自动剪辑**：AI自主完成素材分析、片段筛选、排序、转场、导出
- 🎵 **音频智能处理**：人声降噪增强 + 背景音乐自动匹配卡点
- 📝 **字幕自动生成**：Whisper语音识别，中英双语，自动排版
- ✨ **特效与滤镜**：场景智能匹配滤镜，关键片段自动添加特效
- ⚡ **高效批量**：单条视频剪辑≤3分钟，支持批量处理

## 快速开始

```bash
# 一键全自动剪辑
python scripts/full_pipeline.py -i "D:\电影" -d 180 -o "D:\AI视频剪辑"
```

| 参数 | 说明 | 示例 |
|------|------|------|
| `-i` | 素材目录 | `D:\电影` |
| `-d` | 目标时长（秒） | `180`（3分钟） |
| `-o` | 输出目录 | `D:\AI视频剪辑` |
| `-s` | 剪辑风格 | `auto/quick/slow/narrative` |
| `--filter` | 滤镜风格 | `auto/vivid/vintage/cool/warm` |

## 环境要求

- Python 3.8+
- FFmpeg（需提前安装并配置环境变量）
- 详见 `requirements.txt`

## 适用场景

| 场景 | 推荐配置 | 时长 |
|------|----------|------|
| 搞笑集锦 | 快节奏+音效 | 3-5分钟 |
| 短视频 | 高潮片段+滤镜 | 15-60秒 |
| 电影解说 | 叙事风格+字幕 | 5-15分钟 |
| Vlog剪辑 | 自然过渡+BGM | 3-10分钟 |
