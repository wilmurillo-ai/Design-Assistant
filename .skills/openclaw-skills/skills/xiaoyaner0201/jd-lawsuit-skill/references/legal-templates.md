# 维权文书模板

所有文书输出为 Markdown，保存在证据目录 `~/Downloads/dispute-evidence/{订单号}/` 下。
模板中 `{变量}` 从 `raw-data.json` 和用户输入中填充。

## 一、退款申请函

文件名：`refund-request.md`

```markdown
# 退款/退货申请

## 基本信息

| 项目 | 内容 |
|------|------|
| 交易平台 | {platform} |
| 订单号 | {orderId} |
| 商品名称 | {productName} |
| 订单金额 | ¥{paidAmount} |
| 下单时间 | {orderTime} |
| 商家名称 | {seller.name} |

## 申请类型

- [ ] 仅退款（未收到货 / 已拒收）
- [ ] 退货退款（已收到货）
- [ ] 部分退款（协商金额：¥___）

## 退款原因

{根据纠纷类型选择并填充}

**质量问题**：收到商品存在以下质量缺陷：{具体描述}。
根据《消费者权益保护法》第二十四条，经营者提供的商品不符合质量要求的，
消费者可以要求退货。

**描述不符**：商品页面/快照承诺"{snapshot.promises}"，
实际收到商品{具体差异}，构成描述不符。
根据《电子商务法》第十七条，电子商务经营者不得以虚构交易、
编造用户评价等方式进行虚假或者引人误解的商业宣传。

**未收到货**：订单显示已签收，但本人未收到商品。
物流记录显示{logistics.signedAt}签收，签收人非本人。

## 诉求

1. 全额退款 ¥{paidAmount}
2. 承担退货运费（如适用）
3. {其他诉求}

## 证据

详见 evidence-index.md
```

## 三、民事起诉状

文件名：`civil-complaint.md`

适用场景：12315 投诉未果，金额较大，需通过法院诉讼解决。

> 提醒用户：起诉状仅供参考，建议咨询专业律师。
> 小额诉讼（标的额不超过各省上年度就业人员年平均工资 30%）
> 可走简易程序，不请律师也能自行起诉。

```markdown
# 民事起诉状

## 原告

{姓名}，{性别}，{出生年月}，{民族}，
住址：{地址}，联系电话：{电话}。

## 被告

**被告一**：{seller.company}
住所地：{注册地址}
（工商信息来源：国家企业信用信息公示系统 / 天眼查 / 企查查）

**被告二**（如适用，平台连带责任）：{平台运营公司全称}
住所地：{平台注册地址}

## 诉讼请求

1. 判令被告退还原告货款人民币 {paidAmount} 元；
2. 判令被告支付三倍赔偿金人民币 {paidAmount × 3} 元
  （依据《消费者权益保护法》第五十五条）；
3. 判令被告承担本案诉讼费用。

以上合计人民币 {paidAmount × 4} 元。

## 事实与理由

{时间}，原告通过{platform}（{平台网址}）向被告购买
"{productName}"，支付价款人民币 {paidAmount} 元，
订单号 {orderId}。

{纠纷事实描述——时间线叙事}

原告认为，被告的上述行为违反了{具体法条}，
严重损害了原告作为消费者的合法权益。

## 法律依据

{根据纠纷类型引用，详见 legal-basis.md}

## 证据清单

{引用 evidence-index.md}

## 管辖法院

{原告住所地 / 收货地}基层人民法院
（依据民诉法第23条，网购合同纠纷可由合同履行地即收货地法院管辖）

此致

{法院名称}

起诉人：_______________
{当前日期}
```

## 四、赔偿金额计算

```
│ 退还货款:          ¥{paidAmount}              │
│                                              │
│ 【方案B：退一赔三（欺诈适用，消法55条）】       │
│ 退还货款:          ¥{paidAmount}              │
│ 三倍赔偿:          ¥{paidAmount × 3}          │
│ 最低赔偿保底:      ¥500                       │
│ 实际赔偿:          ¥{max(paidAmount×3, 500)}  │
│ 合计:              ¥{paidAmount + 实际赔偿}    │
│                                              │
│ 【方案C：退一赔十（食品安全适用，食安法148条）】 │
│ 退还货款:          ¥{paidAmount}              │
│ 十倍赔偿:          ¥{paidAmount × 10}         │
│ 最低赔偿保底:      ¥1000                      │
│ 合计:              ¥{paidAmount + max(×10,1000)}│

```
