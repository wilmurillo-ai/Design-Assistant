# Pokémon Pokopia Guide (宝可批攻略)

A comprehensive guide for **Pokémon Pokopia** (Switch 2, developed by Koei Tecmo / Game Freak).

## Quick Reference

| Topic | File | Use when |
|-------|------|----------|
| **Pokédex** | `references/pokedex.md` | Looking up a Pokémon by name (EN/ZH/JP), finding habitat, specialty, time/weather |
| **Habitats** | `references/habitats.md` | Building a specific habitat, finding what Pokémon it attracts |
| **Items** | `references/items.md` | Looking up furniture, materials, food, building kits |
| **Comfort** | `references/comfort.md` | Raising comfort levels, roommate pairing, preference groups |
| **Abilities** | `references/abilities.md` | Ditto transformations, Pokémon specialties, material production |
| **Walkthrough** | `references/walkthrough.md` | Story progression, area guides, endgame |

## How to Use

1. **Pokémon lookup**: Read `references/pokedex.md`, search by English name, Chinese name (中文名), or Japanese name (日本語名). Each entry has: Dex #, EN/ZH/JP names, habitat, specialty, time, weather.

2. **"What Pokémon can I find at night?"** → grep pokedex.md for `Night`

3. **"How do I get Pikachu?"** → Find Pikachu in pokedex.md → see required habitat → check habitats.md for build instructions

4. **"Best roommates?"** → Read `references/comfort.md` for preference groups and roommate pairing recommendations

5. **"What does Ditto learn?"** → Read `references/abilities.md` for all 14 Ditto transformations

## Trilingual Support (三语查询)

The Pokédex includes names in three languages:
- **English**: Official English names (primary key)
- **中文**: Official Simplified Chinese names from the Pokémon franchise
- **日本語**: Official Japanese names (katakana)

Search any language to find the Pokémon. Example:
- "Bulbasaur" = "妙蛙种子" = "フシギダネ" → Dex #1

## Key Game Concepts

- **You play as Ditto** transforming into Pokémon to learn skills
- **4 main areas**: Withered Wasteland → Bleak Beach → Rocky Ridges → Sparkling Skylands
- **Build habitats** to attract Pokémon; each Pokémon needs specific habitat + conditions
- **Pokémon have Specialties** (Build, Burn, Chop, Water, etc.) that help you gather materials
- **Comfort system**: Keep Pokémon comfortable by matching preferences and roommates
- **300 Pokémon total** (including 12 legendaries)
