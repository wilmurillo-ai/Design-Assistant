# Habitats Guide (栖息地图鉴)

Habitats are specific environments you build to attract Pokémon. Each Pokémon requires at least one specific habitat.

## How Habitats Work

1. **Build the habitat** using specific items/blocks/nature
2. **Meet additional conditions** (time of day, weather) if needed
3. **Pokémon will visit** once habitat + conditions are met
4. **Befriend them** by interacting, feeding, or completing their requests

## Basic Habitats (Withered Wasteland)

- **Items**: Any grass tile
- **Pokémon**: Bulbasaur, Charmander, Squirtle, Oddish (Night), Geodude
- **Notes**: Simplest habitat; the first one you'll build
