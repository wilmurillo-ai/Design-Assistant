# Walkthrough (主线攻略)

## Game Overview

- **Developer**: Koei Tecmo / Game Freak
- **Platform**: Nintendo Switch 2 exclusive
- **Genre**: Building/Simulation with Pokémon
- **Main story**: ~20-30 hours
- **Completionist**: 80+ hours
- **You play as**: Ditto, transforming into Pokémon to learn skills and rebuild the world

## Story Structure

4 main towns → Endgame → Post-game

```
Withered Wasteland → Bleak Beach ─┐
                                   ├→ Endgame → Post-game
                  → Rocky Ridges ──┘
                  → Sparkling Skylands
```

After Withered Wasteland, you can choose Bleak Beach or Rocky Ridges first. Sparkling Skylands is last.

## Area 1: Withered Wasteland (枯萎荒原)

**Main Quest**: "Yawn Up A Storm!" — Rescue Onix and restore rain.

### Key Steps
1. Meet Squirtle → learn **Water Gun**
2. Meet Bulbasaur → learn **Leafage**
3. Meet Charmander → learn **Burn** (via befriending)
4. Meet Scyther → learn **Cut**
5. Meet Hitmonchan → learn **Rock Smash**
6. Meet Drilbur → learn **Rototiller**
7. Meet Magikarp → learn **Jump**
8. Build habitats, increase humidity to 100%
9. Rescue Onix → rain returns

### Ditto Skills Learned
Water Gun, Leafage, Cut, Rock Smash, Rototiller, Jump

### Pokémon Center Rebuild
After clearing the story, rebuild the Withered Wasteland Pokémon Center with required materials and Pokémon.

## Area 2: Bleak Beach (暗淡海滩)

**Main Quest**: "Brighten Things Up!" — Wake up Mosslax and bring light back.

### Key Steps
1. Meet Lapras → learn **Surf**
2. Meet Zorua → learn **Camouflage**
3. Meet Paldean Wooper → learn **Suck Up Liquids**
4. Meet Peakychu (special Pikachu) → learn Illuminate concept
5. Clean up the S.S. Anne area
6. Wake up Mosslax → light returns to town

### Ditto Skills Learned
Surf, Camouflage, Suck Up Liquids

### After Clearing
- Rebuild Bleak Beach Pokémon Center
- Access more trading Pokémon

## Area 3: Rocky Ridges (岩石山脊)

**Main Quest**: "Time To Party!" — Throw a party in the ash-covered ruins.

### Key Steps
1. Meet Machoke → learn **Strength**
2. Meet Graveler → learn **Rollout**
3. Work with Chef Dente (Greedent) for cooking
4. Set up DJ Rotom for music
5. Increase area mood through party preparations
6. Learn about ores, ingots, and furnace

### Ditto Skills Learned
Strength, Rollout

### Key Features
- Smelting Furnace unlocked (ore → ingots)
- Cooking system expanded (Curry, party food)
- Music/DJ system with Rotom

## Area 4: Sparkling Skylands (闪耀天空岛)

**Main Quest**: "Rebuild The Huge Building!" — Help Tinkmaster rebuild a skyscraper.

### Key Steps
1. Meet Dragonite → learn **Glide**
2. Meet Gyarados → learn **Waterfall**
3. Navigate floating islands
4. Work with Tinkmaster (Tinkaton) on the skyscraper
5. Gather advanced materials (Pokémetal, Glass, etc.)

### Ditto Skills Learned
Glide, Waterfall

### Key Features
- Most complex building projects
- Pokémetal crafting
- Tinkagear production (via Tinkaton)

## Endgame

### Before Rolling Credits
1. Complete all 4 area main quests
2. Complete the **Team Initiation Challenge** — A game-spanning quest from early on
   - Requires delivering specific items across all beaches
   - Multiple deliveries needed
3. All Important Requests must be cleared

### Team Initiation Challenge
- Starts early in the game
- Involves building requests and item deliveries at Bleak Beach
- Required to beat the game
- Check quest log for remaining deliveries

### Magnet Rise (Final Transformation)
- Unlocked post-game
- Lets you fly up/down freely
- Place and absorb blocks from the air
- Game-changer for building

### Dream Islands
- Unlocked via Drifloon
- Solo or co-op with a friend
- Unique materials and dolls available
- Some yield legendary Pokémon encounters

### Mysterious Slates
- 27 slates (A-Z + ?!)
- Found via Dowsing Machine + Search Pokémon
- Collecting all 27 summons Mew

### Mysterious Feathers
- **Rainbow Feather** — Falls from sky when you hear a Pokémon roar → likely related to Ho-Oh
- **Silver Feather** — Falls similarly → likely related to Lugia

## Beginner Tips

1. **Save Life Coins** — Don't spend early; you'll need them later
2. **Build habitats ASAP** — More Pokémon = more Specialties = faster progress
3. **Get a Chop Pokémon early** — Lumber is needed constantly
4. **Get a Burn Pokémon early** — Bricks from Clay, Ingots from Furnace
5. **Community Box + Gather Pokémon** = passive material collection
6. **Plant crops** — Food for cooking, which upgrades Ditto's skills
7. **Talk to every Pokémon** — Some have requests that give rare items
8. **Don't ignore Litter Pokémon** — They drop critical materials automatically
9. **Search everything** — Use Dowsing Machine + Search Pokémon for hidden items
10. **Check weather** — Some rare Pokémon only appear in rain or sun
11. **Play at different times** — Night-exclusive Pokémon exist
12. **Rebuild Pokémon Centers** — They provide access to PC shops and more Building Kits
13. **Keep Recycle Pokémon near trash** — Free Iron Ore and Paper
14. **Snorlax/Mosslax effects** — Feed Mosslax for daily buffs to item rarity or spawn rates

## Which Area After Withered Wasteland?

After completing Withered Wasteland, you can go to either:

| Bleak Beach First | Rocky Ridges First |
|---|---|
| ✅ Learn Surf early (better exploration) | ✅ Learn Strength early (move boulders) |
| ✅ Beach Pokémon + Water types | ✅ Fire/Rock Pokémon + Furnace |
| ✅ Easier overall | ✅ More materials sooner |

**Recommendation**: Bleak Beach first for Surf, which opens up more exploration options.
