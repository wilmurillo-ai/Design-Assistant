# 🎮 Pokopia Guide — OpenClaw Skill

A comprehensive Pokémon Pokopia (宝可批/宝可森/ぽこポケ) guide as an [OpenClaw](https://github.com/openclaw/openclaw) Skill.

## Features

- 📖 **Full Pokédex** — 100+ Pokémon with English, Chinese (中文), and Japanese (日本語) names
- 🏠 **Habitat Dex** — All habitat build requirements and which Pokémon they attract
- 🛋️ **Item Database** — Furniture, materials, food, and building kits
- 😊 **Comfort System** — How to raise comfort levels + roommate pairing recommendations
- 🔄 **Ditto Abilities** — All 14 transformations and food upgrades
- 🗺️ **Walkthrough** — Full story progression for all 4 areas + endgame

## Trilingual Support (三语查询 / 三言語対応)

Search by name in any language:
- **English**: "Where do I find Pikachu?"
- **中文**: "皮卡丘在哪里？"
- **日本語**: "ピカチュウはどこにいる？"

## Install

```bash
# OpenClaw CLI
openclaw skill install pokopia-guide

# Or clone directly
git clone https://github.com/xiaoyaner0201/pokopia-guide.git ~/.openclaw/skills/pokopia-guide
```

## Usage

Once installed, just ask your OpenClaw agent about Pokopia:

- "How do I build the Hydrated Tall Grass habitat?"
- "What are Pikachu's comfort preferences?"

- "宝可批怪力推石头怎么完成？"
