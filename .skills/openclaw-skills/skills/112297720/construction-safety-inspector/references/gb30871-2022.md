# GB30871-2022 特殊作业安全规范 — 核心条款速查

> 本文件依据 GB30871-2022《危险化学品特殊作业安全规范》编制，作为图片安全分析的基准依据。

---

## 一、八大特殊作业分类

| 作业类型 | 定义要点 |
|---------|---------|
| 动火作业 | 焊接、切割、打磨等产生火花/火焰/赤热表面的作业 |
| 受限空间作业 | 进出口受限，可能缺氧或有毒气体积聚的空间 |
| 盲板抽堵作业 | 设备管道上加装/拆除盲板的作业 |
| 吊装作业 | 利用各种起重机械对重物进行吊运 |
| 临时用电作业 | 临时性的电气连接和使用 |
| 动土作业 | 挖土、打桩、地锚埋设等 |
| 高处作业 | 坠落高度基准面 2m 及以上 |
| 断路作业 | 道路临时断通 |

---

## 二、高处作业（重点）

### 等级划分
- I级：高处作业 2m ≤ 高度 ＜ 5m
- II级：5m ≤ 高度 ＜ 15m
- III级：15m ≤ 高度 ＜ 30m
- IV级：高度 ≥ 30m

### 关键要求
1. **安全带（安全帽）**：高处作业必须系挂安全带，安全带应高挂低用；所有人必须佩戴安全帽（下颌带系紧）
2. **脚手架**：脚手架搭设符合 GB 51210，验收合格后方可使用
3. **作业平台**：操作平台四周设置防护栏杆，踢脚板
4. **工具防坠落**：小件工具装入工具袋或用安全绳系牢
5. **禁止事项**：严禁抛掷材料、工具

---

## 三、动火作业（重点）

### 分级
- 特级动火：处于运行状态的易燃易爆生产装置、输送管道、储罐等区域的动火
- 一级动火：易燃易爆区域外的动火
- 二级动火：除特级、一级动火外的固定动火区

### 关键要求
1. **许可证**：特级动火需主管厂长/总工审批，一级需安全管理部门审批
2. **气体检测**：动火前30分钟内检测可燃气体、有毒气体浓度，合格后方可作业
3. **监护人**：特级动火须连续检测；一二级动火在异常情况时检测
4. **防火措施**：清理周围10米内可燃物；下方设置接火盆；配备灭火器
5. **焊接手套/面罩**：焊工必须穿戴防护手套、焊接面罩/护目镜
6. **瓶体间距**：乙炔瓶与氧气瓶间距 ≥ 5m，分开存放

---

## 四、受限空间作业（重点）

### 关键要求
1. **气体检测**：氧含量 19.5%~23.5%；可燃气体＜LEL 10%；有毒气体＜PC-TWA/PCTLV
2. **强制通风**：作业前通风换气≥30分钟，持续通风
3. **安全隔离**：电气设备断电、管道加盲板隔离
4. **监护人**：外部持续监护，准备安全绳/防毒面具
5. **应急联络**：配备救援三脚架、空气呼吸器

---

## 五、临时用电

1. **三级配电两级保护**：总配电箱→分配电箱→开关箱
2. **漏电保护**：开关箱漏电动作电流≤30mA，动作时间≤0.1s
3. **接地保护**：所有电气设备外壳必须接地
4. **电缆敷设**：架空或埋地，禁止拖地、浸水
5. **专业电工**：必须由持证电工操作

---

## 六、吊装作业

1. **十不吊**：超载/信号不清/安全装置失灵/埋地下/视线不清/斜拉斜拽/捆绑不牢/无证指挥/风≥6级/危险介质环境未采取安全措施
2. **吊具检查**：吊钩防脱装置完好，钢丝绳完好（断丝数≤规定值）
3. **作业区域警戒**：设置警戒区，无关人员禁止入内
4. **指挥人员**：佩戴明显标志，持证指挥

---

## 七、脚手架安全（参照 GB 51210）

1. **基础**：基础平整夯实，垫板/底座可靠
2. **杆件**：横杆、纵杆、剪刀撑齐全
3. **连墙件**：按设计要求设置，禁止随意拆除
4. **作业层**：脚手板满铺，踢脚板设置，防护栏杆≥1.2m
5. **验收挂牌**：使用前须验收，悬挂验收牌

---

## 八、盲板抽堵作业规范（补充）

1. **作业票办理**：同一盲板的抽、堵作业须分别办理盲板抽、堵安全作业票；一张票只能进行一块盲板的一项作业
2. **系统降压**：作业前应将系统管道压力降至安全范围，并切断所有物料来源
3. **盲板材质**：根据介质、温度、压力选用适当材质和规格
4. **气体检测**：在有毒有害介质的管道上作业时，须佩戴防毒面具
5. **现场监护**：属地单位须安排专人在现场监护

---

## 九、重要新规速递

### GB 46768-2025《有限空间作业安全技术规范》
- **实施日期**：2026年5月1日（强制性国家标准）
- **核心变化**：正式区分"有限空间"与"受限空间"两个概念
  - **有限空间**：封闭/半封闭、进出口受限、未被设计为固定工作场所、自然通风不良、易积聚有毒有害/易燃易爆物质或氧含量不足的空间
  - **注意**：新标准使用"有限空间"表述，与旧规中"受限空间"有所区别

### 建筑施工安全总监配备条件（2026新规）
三类项目须配备专职安全总监：
1. 施工现场管理人员及施工人员峰值超500人
2. 房屋建筑工程施工面积超10万平方米
3. 市政基建工程合同价款超4亿元，或隧道、城市轨道交通工程

### 特殊作业安全承诺公告制度
企业须每日向上级主管部门公告当日特殊作业数量，包括：特级/一级/二级动火作业、受限空间作业、盲板抽堵作业、高处作业、吊装作业、临时用电作业、动土作业、断路作业。

## 十、常见重大隐患（红线）

- 高处作业不系安全带
- 临边、洞口无防护
- 氧气乙炔瓶距离不足5米
- 焊接作业无接火措施
- 特种作业无证操作
- 配电箱违规使用
- 进入受限空间不检测气体
- 脚手架搭设不合格投入使用
- 交叉作业无隔离防护
- 动火作业无监火人
