# 石油石化论文写作格式规范

## 一、论文结构规范

### 摘要格式
- 中文摘要：200-400字
- 英文摘要（可选）：与中文对应
- 关键词：3-5个，用分号分隔

摘要结构（五要素）：
1. 研究背景（1-2句）
2. 研究目的（1句）
3. 研究方法（2-3句）
4. 主要结论（2-3句）
5. 研究贡献（1句）

### 正文结构

**常见论文结构（按研究类型）：**

#### 技术应用类
```
第一章  绪论
第二章  相关技术与理论基础
第三章  石油石化行业信息化现状分析
第四章  技术方案设计与实现
第五章  应用效果与评估
第六章  结论与展望
```

#### 问题诊断类
```
第一章  绪论
第二章  行业现状与问题分析
第三章  问题成因与影响机制
第四章  解决方案设计
第五章  实施效果分析
第六章  结论与建议
```

#### 技术对比类
```
第一章  绪论
第二章  技术原理与现状
第三章  国内外技术对比分析
第四章  应用场景与适配性分析
第五章  结论与建议
```

## 二、格式要求

### 字号与字体
| 位置 | 字体 | 字号 |
|-----|------|------|
| 论文标题 | 黑体 | 二号 |
| 章节标题 | 黑体 | 三号 |
| 正文 | 宋体/微软雅黑 | 小四号 |
| 摘要标题 | 黑体 | 四号 |
| 关键词 | 宋体 | 小四号 |
| 页眉 | 宋体 | 五号 |

### 行距
- 正文：1.5倍行距
- 摘要：1.5倍行距
- 参考文献：1.5倍行距

### 页码
- 摘要页单独编页
- 正文从第1页开始
- 页码居中或右下

## 三、参考文献格式（GB/T 7714）

### 期刊格式
```
[序号] 作者. 题名[J]. 刊名, 出版年, 卷(期): 起止页码.
```

### 学位论文
```
[序号] 作者. 题名[D]. 城市: 培养单位, 出版年.
```

### 会议论文
```
[序号] 作者. 题名[C]//会议名称. 出版地: 出版者, 出版年: 起止页码.
```

### 网络资源
```
[序号] 作者. 题名[EB/OL]. (更新日期)[引用日期]. URL.
```
