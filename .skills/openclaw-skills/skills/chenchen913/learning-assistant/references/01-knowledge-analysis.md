# 知识解析模块

## 概述
本模块定义了技术概念解析的结构化输出格式。
当用户要求“解释”、“分析”或“深入探讨”某个概念时触发。

## 输出维度

输出内容应根据请求的模式（快速/标准/深度）进行自适应调整。

| 维度 | 描述 | 内容指南 |
|---|---|---|
| **[DEF] 一句话定义** | 它是什么？ | 简洁、高层次的总结。尽可能不使用行话。 |
| **[USE] 应用场景** | 真实工程场景 | 在生产环境中实际用在哪里？ |
| **[HARD] 技术难点** | 常见陷阱 | 难点在哪里？大家通常在哪里卡住？ |
| **[LIST] 掌握清单** | 学习内容 | 必须掌握 vs. 了解即可 vs. 暂时跳过。 |
| **[PRE] 前置知识** | 依赖项 | 在此之前必须理解什么？ |
| **[LINK] 关联主题** | 知识图谱链接 | 链接到其他概念（用于学习路径）。 |
| **[PATH] 学习路径** | 如何学习 | 循序渐进的步骤。 |
| **[LAB] 实践建议** | 动手练习 | 具体的项目或练习建议。 |
| **[LEVEL] 掌握程度基准** | 级别 | 初级（基本使用）/ 中级（底层原理）/ 高级（优化与架构）。 |
| **[JOB] 面试视角** | 常见问题 | 与此相关的高频面试话题。 |
| **[TIME] 学习周期建议** | 时间投入 | 达到初级/中级掌握程度的预计时间。 |
| **[VER] 技术时效标注** | 相关性检查 | 适用版本（例如 React 18+, Python 3.10+）。 |

## 输出逻辑

```
IF 模式 == "快速":
    输出: 定义 + 应用场景 + 技术难点（简略）

IF 模式 == "标准" (默认):
    输出: 定义 + 应用场景 + 掌握清单 + 实践建议 + 面试视角

IF 模式 == "深度":
    输出: 所有维度 + 代码示例 + 原理推导
```
