# 过去维度：人生归档系统

## 核心定位

过去维度让 AI 真正理解用户是谁，而不是只知道用户"最近"是谁。

**两个核心问题：**
- 时间线：我经历了什么？
- 主题线：我是一个什么样的人？

---

## 一、时间线（按年归档）

**路径**：`/memory/past/timeline/YYYY.md`

**记录原则**：
- 越近越详细，越远越粗线条
- 关键词 + 一两句话足够，不需要完整叙述
- 实在记不清的年份，写几个印象最深的片段即可

**单年模板**：

```markdown
# YYYY 年

## 这一年的基调

[2-3 句话：这一年你整体在经历什么？]

## 重要事件

| 月份 | 事件 | 简述 |
|---|---|---|
| | | |

## 重要决定

[这一年做过的、对人生有影响的决定，以及当时的理由]

## 情绪底色

[这一年总体的心理状态：什么情绪最常在？为什么？]

## 重要的人际变化

[认识了谁、失去了谁、关系发生了什么变化]

## 年末状态

[这一年结束时，你带着什么走进下一年？]

## 最骄傲的事

[一件]

## 最遗憾的事

[一件]
```

### decisions.md — 重大决策记录

**⚠️ 这是未来维度做推演时最重要的参考文件。**

```markdown
# 人生重大决策记录

## 记录格式

每条记录包含：
- **决策**：做了什么决定
- **时间**：大约什么时候

## 决策列表

[按时间从近到远排列]

---

## 我的决策规律（由 AI 归纳，定期更新）

> 基于以上决策记录，AI 观察到的用户决策模式：
> [AI 在积累了足够数据后填写，例如："你在职业决策上倾向于……；在关系决策上容易……；
> 你做得最好的决定往往有一个共同特点……"]
```

### cognition.md — 认知演化记录

**⚠️ 这是整个过去维度里时间复利最高的文件。**

```markdown
# 认知演化轨迹

## 关于这个文件

记录的不是事件，而是你**思维方式的变化**。
写过的文章、读书笔记、重要对话、改变你看法的时刻，都是原材料。

## 记录格式

```
[时间] | [类型：文章/读书/对话/经历/顿悟]
来源/标题：[是什么]
核心观点：[当时认为什么是对的]
当时背景：[为什么当时这么想]
现在是否认同：[完全认同 / 部分认同 / 不再认同]
如果改变了：[为什么改变，现在认为什么是对的]
```

## 认知记录列表

[按时间从近到远]

## 重大认知转折点

格式：
- **时间**：
- **触发**：[什么书/人/事让你改变了看法]
- **转变前**：[之前相信什么]
- **转变后**：[现在相信什么]
- **影响**：[这个转变如何影响了你后来的行为或决策]

## 我目前还没想清楚的问题

[你还在探索中、还没有答案的重要问题——这些往往是最值得持续追踪的东西]
```

## 三、AI 辅助回忆模式

**触发**：用户说"帮我回忆过去"、"我不知道从哪里开始"、"陪我聊聊我自己"。

**原则**：每次只问一个问题，根据回答追问，把零散记忆整理成可归档的内容。

**引导问题库（按主题）**：

职业：
- "你做过的工作里，哪一份让你真正感到有意义？（不是最体面的，是最有意义的）"
- "有没有一次你觉得'这不是我想要的'的经历？它是什么？"
- "如果职业可以重来，你会在哪个节点做不同的选择？"

关系：
- "有没有一个人，对你的影响是你到现在都能感受到的？"
- "你人生中最好的友谊是什么样的？"
- "有没有一段关系，让你对自己有了新的认识？"

成长：
- "你哪个阶段变化最大？是什么触发了那些变化？"
- "你现在和 10 年前的自己相比，最大的不同是什么？"
- "有没有一句话，某本书或某个人说的，你一直记到现在？"

决策：
- "过去 5 年里，你最庆幸自己做的一个决定是什么？"
- "有没有一个决定，当时很难，现在庆幸做了？"
- "如果给 5 年前的自己一条建议，你会说什么？"

## 四、过去维度的更新时机

以下情况，AI 需要主动提示更新过去维度：

| 触发情况 | 建议更新 |
|---|---|
| diary 中出现重大事件 | 提示归档到 timeline 或对应主题文件 |
| 用户完成了一个重大决策 | 在 decisions.md 中补充结果和评价 |
| 用户说了一个新的重要认知 | 提示记录到 cognition.md |
| 每年年底 | 提示为当年建立 timeline 文件 |
| 用户的 profile 信息发生重大变化（换城市/换工作/状态大变）| 提示更新 profile.md |

---

## 五、冷启动建议（填写优先级）

1. `personality.md` — 最快帮 AI 理解你的文件，优先填
2. `decisions.md` — 最近 3 年的重大决策
3. `timeline/` — 最近 3-5 年
4. `career.md` — 如果职业是当前主要关注点
5. `cognition.md` — 如果有过往文章/日记/笔记
6. 其他文件 — 有时间慢慢补

哪怕只填了 `personality.md`，AI 对你的理解也会大幅提升。
