# 📚 Warm Academia (Style ID: warm-academia)

## Vibe
Intellectual, Nostalgic, Cozy, "Old Library".
Best for: Educational platforms, book clubs, literary magazines, personal essays, cultural events.

## 1. Color Palette (System)

| Token | Hex | Role |
|-------|-----|------|
| `bg-page` | `#F4ECD8` | Aged parchment background |
| `bg-surface` | `#FDFAF3` | Cream card surface |
| `text-primary` | `#2C2416` | Deep sepia (Almost black-brown) |
| `text-secondary` | `#7A6A52` | Warm taupe |
| `accent` | `#8B3A2A` | Burgundy / Dark crimson |
| `accent-gold` | `#C9A84C` | Antique gold for ornaments |
| `border` | `rgba(44,36,22,0.12)` | Subtle warm line |

## 2. Typography
- **Headings**: `Playfair Display`, `EB Garamond`, `Cormorant Garamond`. (Weight: 600/700, Tracking: 0em, Italic variants welcome).
- **Body**: `Lora`, `Georgia`, `Times New Roman`. (Weight: 400, Line-height: 1.8).
- **Accent / Captions**: `Cormorant SC` small caps or `Libre Baskerville`.

## 3. Visual Treatments
- **Radius**: `4px` (Nearly square — books have corners).
- **Shadows**:
    - `shadow-sm`: `0 1px 4px rgba(44,36,22,0.08)`
    - `shadow-lg`: `0 6px 20px rgba(44,36,22,0.14)`
- **Texture Overlay**: Subtle paper grain (`noise: 8%, opacity: 0.4`).
- **Dividers**: Thin ornamental rules (`─── ✦ ───`) instead of solid lines.

## 4. Layout Principles
- **Dense but Breathing**: Column widths inspired by book typography (`max-width: 680px` for body text).
- **Drop Caps**: First letter of sections enlarged (3–4 lines tall).
- **Margin Notes**: Secondary info placed in narrow side margins.
- **No Hard Geometry**: Avoid sharp grid boxes — prefer flowing, editorial layouts.
