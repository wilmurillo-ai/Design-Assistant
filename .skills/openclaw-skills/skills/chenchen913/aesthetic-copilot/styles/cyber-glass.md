# 🔮 Cyber Glassmorphism (Style ID: cyber-glass)

## Vibe
Futuristic, Immersive, Dark, "Living in the Interface".
Best for: Gaming UIs, crypto/Web3 projects, tech startups, music visualizers, dark-mode dashboards.

## 1. Color Palette (System)

| Token | Hex | Role |
|-------|-----|------|
| `bg-page` | `#080B14` | Deep space black |
| `bg-surface` | `rgba(255,255,255,0.06)` | Frosted glass panel |
| `text-primary` | `#E8EEFF` | Cool off-white |
| `text-secondary` | `#7B8DB0` | Muted slate |
| `accent-cyan` | `#00F5FF` | Electric cyan (primary glow) |
| `accent-violet` | `#BF5FFF` | Deep violet (secondary glow) |
| `border` | `rgba(0,245,255,0.18)` | Neon-tinted glass edge |

## 2. Typography
- **Headings**: `Space Grotesk`, `Rajdhani`, `Oxanium`. (Weight: 700, Tracking: 0.04em, ALL CAPS for hero text).
- **Body**: `Inter`, `DM Sans`. (Weight: 300/400, Line-height: 1.6).
- **Data / Mono**: `JetBrains Mono`, `Fira Code` for numeric readouts.

## 3. Visual Treatments
- **Radius**: `16px` (Smooth glass panels).
- **Glassmorphism**:
    - `backdrop-filter: blur(24px) saturate(180%)`
    - `background: rgba(255,255,255,0.05)`
    - `border: 1px solid rgba(0,245,255,0.18)`
- **Glow Effects**:
    - `box-shadow: 0 0 20px rgba(0,245,255,0.25), 0 0 60px rgba(0,245,255,0.08)`
- **Gradient Mesh**: Subtle radial gradients of cyan + violet in the far background.
- **Scanlines**: Optional very faint horizontal scanline overlay (opacity: 0.03).

## 4. Layout Principles
- **Dark Canvas First**: Background is almost entirely black — UI floats on it.
- **Layered Depth**: Use 3+ z-layers of glass panels at different blur levels.
- **Accent Sparingly**: Neon colors only on interactive or focal elements.
- **Data Density**: Compact spacing is acceptable — this aesthetic tolerates density.
