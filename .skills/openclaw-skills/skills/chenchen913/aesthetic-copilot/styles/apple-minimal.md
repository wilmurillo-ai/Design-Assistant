# 🍎 Apple Minimal (Style ID: apple-minimal)

## Vibe
Premium, Trustworthy, Airy, "Museum-Quality".
Best for: Professional portfolios, SaaS dashboards, Educational content, High-end e-commerce.

## 1. Color Palette (System)

| Token | Hex | Role |
|-------|-----|------|
| `bg-page` | `#F5F5F7` | Main background (Off-white gray) |
| `bg-surface` | `#FFFFFF` | Card background (Pure white) |
| `text-primary` | `#1D1D1F` | Main headings (Almost black) |
| `text-secondary` | `#86868B` | Subtitles (Warm gray) |
| `accent` | `#0071E3` | Action buttons (Apple Blue) |
| `border` | `rgba(0,0,0,0.05)` | Subtle separation |

## 2. Typography
- **Headings**: `SF Pro Display`, `Inter`, `-apple-system`. (Weight: 600/700, Tracking: -0.02em).
- **Body**: `SF Pro Text`, `Inter`. (Weight: 400, Line-height: 1.6).

## 3. Visual Treatments
- **Radius**: `20px` (Large, smooth continuous curves).
- **Shadows**:
    - `shadow-sm`: `0 2px 8px rgba(0,0,0,0.04)`
    - `shadow-lg`: `0 12px 30px rgba(0,0,0,0.08)`
- **Glassmorphism**: `backdrop-filter: blur(20px)` + `bg-white/70`.

## 4. Layout Principles
- **Generous Padding**: Minimum `24px` padding inside cards.
- **Center Focus**: Content often centered with max-width `980px` or `1200px`.
