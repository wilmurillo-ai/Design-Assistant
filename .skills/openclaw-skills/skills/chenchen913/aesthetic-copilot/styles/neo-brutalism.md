## Vibe
Playful, Bold, Raw, "Not Boring", Attention-Grabbing.
Best for: Creative agencies, Gen-Z brands, Event posters, Personal blogs.

## 1. Color Palette (System)

| Token | Hex | Role |
|-------|-----|------|
| `bg-page` | `#FFFDF5` | Cream/Paper background |
| `bg-surface` | `#FFFFFF` | Card background (or pastel variants) |
| `text-primary` | `#000000` | Pure Black (High Contrast) |
| `accent-1` | `#FF6B6B` | Red/Pink |
| `accent-2` | `#4ECDC4` | Teal |
| `accent-3` | `#FFE66D` | Yellow |
| `border` | `#000000` | Hard strokes (2px-4px) |

## 2. Typography
- **Headings**: `Clash Display`, `Space Grotesk`, `Archivo Black`. (Weight: 800+).
- **Body**: `DM Sans`, `Public Sans`. (Weight: 500).

## 3. Visual Treatments
- **Borders**: `border: 3px solid black`.
- **Shadows**: Hard shadows (No blur).
    - `box-shadow: 4px 4px 0px 0px #000000`
- **Radius**: Mix of `0px` (Sharp) and `12px` (Slightly rounded).

## 4. Layout Principles
- **Asymmetry**: Elements can overlap or be slightly misaligned.
- **Grid Lines**: Visible grid lines separating sections.
- **Marquee**: Scrolling text for announcements.
