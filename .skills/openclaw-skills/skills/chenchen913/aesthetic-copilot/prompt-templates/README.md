# 🖼️ Prompt Templates (Structural Masterpieces)

This directory contains high-fidelity prompt structures designed for top-tier Text-to-Image models (Flux, Ideogram, Midjourney v6).

## The "Mega-Prompt" Structure
Each template follows this robust architecture:
1.  **Meta-Instruction**: "Create a [Style] image..."
2.  **Subject & Layout**: Explicit spatial positioning.
3.  **Text Rendering**: Specific instructions for font, weight, and integration.
4.  **Art Direction**: Lighting, Texture, Camera, Color Palette.
5.  **Micro-Innovation**: The "Twist" that makes it unique.

## Available Templates
| ID | Name | Best For |
|----|------|----------|
| `structural-poster` | Structural Poster | Informational layouts, calendars, infographics |
| `editorial-spread` | Editorial Spread | Magazine style, heavy text-image integration |
| `product-showcase` | Product Showcase | Commercial, object-focused, clean background |
| `surreal-concept` | Surreal Concept | Abstract ideas, "Micro-Innovation" heavy |
