# 🧠 The Engine: Logic & Micro-Innovation

This directory defines how the "Brain" processes vague inputs into artistic prompts.

## 1. The "Micro-Innovation" Matrix (Artistic Twist)

When the user asks for a common subject, the Engine MUST apply a "Twist" to make it commercial-grade art.

| Input Category | Standard Depiction (Boring) | **Micro-Innovation (Twist)** |
|----------------|-----------------------------|------------------------------|
| **Book/Reading** | A book on a table | Book pages turning into birds; A book glowing from within; A library inside a glass bottle. |
| **Technology** | A phone or laptop | Circuit boards made of golden roots; Data streams as liquid silk; A glass device floating in clouds. |
| **Nature** | A tree | A tree made of crystal; Leaves that are tiny glowing lanterns; A tree growing out of a book. |
| **Calendar** | A paper calendar | Dates floating in 3D space; Time represented as flowing sand; A calendar carved into stone. |

## 2. Text Integration Logic

For models like Flux/Ideogram, text is physical, not just an overlay.

*   **Embossed**: Text rises from the background material.
*   **Neon**: Text is a light source.
*   **Integrated**: Text is formed by objects (e.g., clouds forming letters).
*   **Masked**: Subject overlaps part of the text (magazine style).
