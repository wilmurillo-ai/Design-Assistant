# 首尾模块模板库

AI 根据文章类型和内容智能推荐合适的模块组合。用户可自定义修改。

---

## 文首模块

### 1. 原创声明

适用：所有原创文章

```
本文为原创内容，未经授权禁止转载。如需转载请联系公众号后台获取授权。
```

### 2. 系列导读

适用：系列文章

```
这是「[系列名称]」系列的第 [N] 篇。

往期回顾：
- 第 1 篇：[标题]（链接）
- 第 2 篇：[标题]（链接）
```

### 3. 阅读提示

适用：长文（2000+ 字）或深度内容

```
全文约 [XXXX] 字，阅读约需 [X] 分钟。建议收藏后仔细阅读。
```

### 4. 背景说明

适用：时效性/事件性文章

```
本文写于 [XXXX 年 X 月]，基于 [XX 版本/XX 事件] 撰写。如有更新，请以最新信息为准。
```

### 5. 免责声明

适用：涉及投资、医疗、法律等敏感话题

```
声明：本文仅代表个人观点和经验分享，不构成任何 [投资/医疗/法律] 建议。请根据自身情况独立判断。
```

## 文末模块

### 1. 作者简介

适用：所有文章

模板 A（简洁版）：
```
作者：[姓名]，[一句话介绍]。
```

模板 B（详细版）：
```
关于作者
[姓名]，[领域]从业 [X] 年。专注于 [方向1] 和 [方向2]。
个人公众号：[公众号名称]
```

### 2. 转载说明

适用：希望被转载的文章

```
转载须知：本文欢迎转载，但请保留作者信息和原文链接。商业转载请联系公众号后台。
```

### 3. 关注引导

适用：所有文章

模板 A（轻量版）：
```
觉得有用？点个「赞」和「在看」，我会持续更新。
```

模板 B（引导关注版）：
```
如果这篇文章对你有帮助，欢迎：
- 点赞 + 在看，让更多人看到
- 关注公众号「[名称]」，第一时间获取更新
- 转发给需要的朋友
```

### 4. 相关推荐

适用：有关联文章时

```
你可能还想看：
- [相关文章标题 1]
- [相关文章标题 2]
- [相关文章标题 3]
```

### 5. 资源汇总

适用：文中提到了工具、链接、资源

```
本文提到的工具和资源：
- [工具名称 1]：[简要说明]（链接）
- [工具名称 2]：[简要说明]（链接）
```

### 6. 互动引导

适用：希望读者参与讨论

模板 A：
```
你的经验是什么？欢迎在评论区聊聊。
```

模板 B：
```
关于 [话题]，你有什么看法？留言区见。
```

## 智能推荐逻辑

AI 根据以下规则推荐模块组合：

| 文章类型 | 推荐的文首模块 | 推荐的文末模块 |
|---------|--------------|--------------|
| 教程类 | 阅读提示 | 资源汇总 + 关注引导 |
| 测评类 | 背景说明 | 资源汇总 + 互动引导 |
| 观点类 | 原创声明 | 作者简介 + 互动引导 |
| 故事类 | 无（直接开篇） | 作者简介 + 关注引导 |
| 系列文章 | 系列导读 + 阅读提示 | 相关推荐 + 关注引导 |
| 涉及敏感话题 | 免责声明 | 作者简介 + 转载说明 |

推荐后征求用户意见，用户可增删或自定义内容。
