# 媒体资源指南 (Phase 7)

本阶段处理文章中所有媒体资源：图片和视频。进入本阶段前先询问用户选择执行模式。

## 媒体生成模式选择

即将进入【Phase 7: 媒体资源（配图 + 配视频）】
- 作用：为文章标注并生成封面图、正文配图和视频资源
- 预计耗时：自动生成约 5-10 分钟，仅提示词约 2-3 分钟

请选择生成模式：
A) 自动生成 -- AI 直接生成图片/视频并嵌入文章（需要对应的生成 Skill）
B) 仅提示词 -- AI 只输出生成用的提示词，你自行用工具生成后放入文章
C) 混合模式 -- AI 先列出全部媒体资源清单，再逐项询问你选择自动生成还是仅提示词
D) 跳过本步骤
E) 我有补充信息，先听我说

**降级策略**：若用户选择模式 A 但 image-gen 或视频生成工具不可用，自动降级为模式 B（输出提示词），并告知用户原因。

## 7.1 封面图

- 推荐尺寸：大图 900x383（2.35:1），小图 1:1
- AI 给出设计建议（风格、核心元素、文字内容）
- 模式 A：调用 image-gen Skill 直接生成
- 模式 B：输出详细提示词，用户自行生成

## 7.2 正文配图

建议频率：每 500-800 字一张。

**三类标注**：

```
[用户提供] 第 X 段 -- 需要一张 XX 截图，用于说明 XX
  说明：此图必须由用户提供（截图/实拍/数据图表）

[AI 生成-图片] 第 Y 段 -- 需要一张 XX 氛围图
  提示词：[详细描述画面内容、风格、色调、构图]
  建议尺寸：XXXxXXX
  >> 模式 A：调用 image-gen Skill 生成并嵌入
  >> 模式 B：仅输出上方提示词，用户自行生成

[AI 生成-视频] 第 Z 段 -- 需要一段 XX 演示/案例视频
  （完整格式见 7.3 视频提示词格式）
  >> 模式 A：调用视频生成工具生成
  >> 模式 B：仅输出提示词，用户自行制作
```

## 7.3 视频资源

适用场景：教程操作演示、案例展示、数据可视化动画、产品功能展示等。

**视频标注规则**：
- AI 分析文章内容，识别适合用视频呈现的段落（比纯文字/图片更有表现力的地方）
- 不强制添加视频；仅当视频确实比图片更合适时才建议
- 每个视频标注包含：内容描述、建议时长（通常 5-30 秒）、风格说明、关键画面

**视频提示词格式**：

```
[AI 生成-视频] 第 X 段
  内容：[视频展示什么内容]
  时长：约 X 秒
  风格：[真实感/动画/屏幕录制/数据可视化]
  关键画面：
    1. 开头：[描述]
    2. 主体：[描述]
    3. 结尾：[描述]
  用途：[说明为什么这里用视频比图片更合适]
```
