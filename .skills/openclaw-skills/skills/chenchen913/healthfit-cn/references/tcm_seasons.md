## 春季养生（立春→谷雨）

### 一、立春（2 月 3-5 日）

**气候特点：** 阳气始生，万物复苏，但寒气未散

**养生原则：** 养肝护阳，春捂防寒

#### 通用建议

**起居：**
- 早睡早起（21:00-22:00 入睡，6:00-6:30 起床）
- 春捂：不要急于减衣，注意背部、腹部保暖
- 晨起后开窗通风 15-20 分钟

**运动：**
- 选择温和运动：散步、太极拳、八段锦
- 晨练时间不宜过早（等太阳出来后再出门）
- 运动强度：微微出汗即可

**情志：**
- 保持心情舒畅，避免生气发怒
- 多到户外接触自然，感受春意

#### 体质专项提醒

| 体质 | 特别提醒 |
|------|---------|
| 气虚质 | 注意保暖，避免受风感冒 |
| 阳虚质 | 继续保暖，可开始适度户外活动 |
| 阴虚质 | 避免过度出汗伤阴 |
| 痰湿质 | 增加运动量，促进代谢 |
| 湿热质 | 注意皮肤清洁，预防痤疮 |
| 气郁质 | 多到户外踏青，疏肝解郁 |

#### 代表性方案

**食疗方：韭菜炒鸡蛋**
```
材料：韭菜 200g，鸡蛋 3 个，盐适量
做法：韭菜切段，鸡蛋打散，先炒鸡蛋盛出，再炒韭菜，加入鸡蛋翻炒，加盐调味
功效：温阳护肝，助阳生发
频率：每周 2-3 次
```

**茶饮：玫瑰花茶**
```
材料：玫瑰花（干）10g，冰糖适量
做法：沸水冲泡，焖 5 分钟
功效：疏肝理气，活血养颜
频率：每日 1 杯
```

**穴位保健：太冲穴**
```
位置：足背侧，第 1、2 跖骨结合部之前凹陷处
方法：拇指按揉，每侧 5 分钟
频率：每日 2 次
功效：疏肝理气，平肝熄风
```

### 二、雨水（2 月 18-20 日）

**气候特点：** 降雨增多，空气湿润，湿气渐重

**养生原则：** 健脾祛湿，养护脾胃

#### 通用建议

**起居：**
- 保持室内干燥，必要时使用除湿机
- 避免淋雨，淋雨后及时更换干爽衣物
- 睡前泡脚（水温 40-45℃，20 分钟）

**饮食：**
- 少吃生冷食物，避免损伤脾胃
- 多吃健脾祛湿食物：山药、薏米、茯苓
- 适量吃辛温食物：生姜、葱、蒜（助阳化湿）

**运动：**
- 增加运动量，促进湿气排出
- 推荐：慢跑、快走、瑜伽
- 运动后及时擦干汗水，避免受风

#### 体质专项提醒

| 体质 | 特别提醒 |
|------|---------|
| 痰湿质 | 重点祛湿季节，加强运动，控制饮食 |
| 湿热质 | 清热利湿，避免辛辣油腻 |
| 阳虚质 | 注意保暖，避免寒湿侵袭 |
| 气虚质 | 健脾益气，增强运化功能 |

#### 代表性方案

**食疗方：山药薏米粥**
```
材料：山药 50g，薏米 50g，大米 50g
做法：薏米提前浸泡 2 小时，与山药、大米同煮成粥
功效：健脾祛湿，补肺益肾
频率：每周 4-5 次
```

**茶饮：陈皮茯苓茶**
```
材料：陈皮 5g，茯苓 15g
做法：沸水冲泡或煮 10 分钟
功效：健脾祛湿，理气化痰
频率：每日 1 剂
```

### 三、惊蛰（3 月 5-7 日）

**气候特点：** 春雷始鸣，万物苏醒，细菌病毒活跃

**养生原则：** 疏肝利胆，预防传染病

#### 通用建议

**起居：**
- 保持室内通风，每日至少 2 次，每次 30 分钟
- 注意个人卫生，勤洗手
- 避免到人群密集场所（传染病高发期）

**饮食：**
- 多吃清淡食物，少吃油腻
- 适当吃辛凉食物：梨、芹菜、菠菜
- 避免过量饮酒（伤肝）

**运动：**
- 增加户外活动，呼吸新鲜空气
- 推荐：登山、徒步、放风筝
- 运动强度：中等强度，微微出汗

**防疫：**
- 注意保暖，预防感冒
- 过敏体质者注意防护（花粉、尘螨）

#### 体质专项提醒

| 体质 | 特别提醒 |
|------|---------|
| 特禀质 | 过敏高发期，注意防护，随身携带药物 |
| 气郁质 | 疏肝解郁最佳时节，多到户外 |
| 湿热质 | 预防皮肤疾病，保持皮肤清洁 |

#### 代表性方案

**食疗方：芹菜炒百合**
```
材料：芹菜 200g，百合 50g，盐适量
做法：芹菜切段，百合洗净，快炒 3 分钟，加盐调味
功效：清热平肝，润肺安神
频率：每周 3-4 次
```

**茶饮：菊花枸杞茶**
```
材料：菊花 10g，枸杞 15g
做法：沸水冲泡，焖 5 分钟
功效：清肝明目，滋阴补肾
频率：每日 1 剂
```

### 四、春分（3 月 20-22 日）

**气候特点：** 昼夜平分，阴阳平衡，气候温和

**养生原则：** 调和阴阳，平衡饮食

#### 通用建议

**起居：**
- 保持规律作息，不熬夜
- 适当午休（30 分钟左右）
- 保持心情平和，避免情绪波动

**饮食：**
- 饮食均衡，不偏食
- 寒热食物搭配食用
- 多吃时令蔬菜：菠菜、韭菜、春笋

**运动：**
- 选择温和运动，避免过度
- 推荐：太极拳、八段锦、瑜伽
- 运动时间：上午 9-11 点或下午 15-17 点

#### 体质专项提醒

| 体质 | 特别提醒 |
|------|---------|
| 平和质 | 保持现状，继续均衡饮食 |
| 阴虚质 | 注意滋阴，避免过度出汗 |
| 阳虚质 | 继续温补，但避免过热 |

#### 代表性方案

**食疗方：菠菜猪肝汤**
```
材料：菠菜 200g，猪肝 100g，生姜 3 片，盐适量
做法：猪肝切片焯水，与生姜同煮 10 分钟，加入菠菜再煮 2 分钟，加盐调味
功效：养血补肝，滋阴润燥
频率：每周 2-3 次
```

**保健：阴阳平衡操**
```
动作：
1. 双手托天理三焦（八段锦第一式）× 6 次
2. 左右开弓似射雕（八段锦第二式）× 6 次
3. 调理脾胃须单举（八段锦第三式）× 6 次
频率：每日 1-2 次
```

### 五、清明（4 月 4-6 日）

**气候特点：** 气清景明，万物生长，肝气旺盛

**养生原则：** 养肝明目，踏青祭祖

#### 通用建议

**起居：**
- 早睡早起，顺应阳气生发
- 多到户外踏青，亲近自然
- 注意眼部休息，避免长时间用眼

**饮食：**
- 多吃绿色蔬菜（养肝）
- 适当吃酸味食物（收敛肝气）
- 避免过量饮酒（伤肝）

**运动：**
- 户外运动最佳时节
- 推荐：登山、徒步、骑行
- 运动强度：中等强度

**情志：**
- 保持心情舒畅
- 祭祖时避免过度悲伤

#### 体质专项提醒

| 体质 | 特别提醒 |
|------|---------|
| 气郁质 | 踏青郊游最佳时节，疏肝解郁 |
| 阴虚质 | 注意眼部保养，避免过度用眼 |
| 高血压患者 | 避免情绪激动，监测血压 |

#### 代表性方案

**食疗方：枸杞菊花粥**
```
材料：枸杞 20g，菊花 10g，大米 100g
做法：大米煮粥，快熟时加入枸杞、菊花，再煮 5 分钟
功效：养肝明目，滋阴补肾
频率：每周 3-4 次
```

**茶饮：决明子茶**
```
材料：决明子（炒）15g
做法：沸水冲泡，焖 10 分钟
功效：清肝明目，润肠通便
频率：每日 1 剂
```

### 六、谷雨（4 月 19-21 日）

**气候特点：** 雨生百谷，湿气加重，春季最后一个节气

**养生原则：** 健脾祛湿，预防过敏

#### 通用建议

**起居：**
- 保持室内干燥通风
- 避免淋雨，淋雨后及时更换衣物
- 注意防潮防霉

**饮食：**
- 多吃健脾祛湿食物：薏米、赤小豆、冬瓜
- 少吃油腻、甜食（助湿）
- 过敏体质者避免食用易致敏食物

**运动：**
- 增加运动量，促进湿气排出
- 推荐：慢跑、游泳、瑜伽
- 运动后及时洗澡，保持皮肤清洁

**防疫：**
- 过敏高发期，注意防护
- 花粉过敏者减少户外活动

#### 体质专项提醒

| 体质 | 特别提醒 |
|------|---------|
| 痰湿质 | 重点祛湿，控制饮食，加强运动 |
| 特禀质 | 过敏高发期，注意防护 |
| 湿热质 | 清热利湿，预防皮肤病 |

#### 代表性方案

**食疗方：赤小豆薏米汤**
```
材料：赤小豆 50g，薏米 50g，冰糖适量
做法：赤小豆、薏米提前浸泡 2 小时，同煮 1 小时，加冰糖调味
功效：健脾祛湿，清热消肿
频率：每周 4-5 次
```

**茶饮：玉米须茶**
```
材料：玉米须（干）15g
做法：沸水冲泡或煮 10 分钟
功效：利水消肿，清热祛湿
频率：每日 1 剂
```

## 夏季养生（立夏→大暑）

### 七、立夏（5 月 5-7 日）

**气候特点：** 暑气渐生，阳气旺盛，万物繁茂

**养生原则：** 养心安神，防暑降温

#### 通用建议

**起居：**
- 晚睡早起（22:00-23:00 入睡，5:30-6:00 起床）
- 适当午休（30-60 分钟）
- 保持室内通风，避免直吹空调

**饮食：**
- 多吃清淡食物，少吃油腻
- 适当吃酸味食物（收敛汗液）
- 多喝水，补充水分
- 推荐食物：苦瓜、黄瓜、西红柿、西瓜

**运动：**
- 避免在高温时段运动（10:00-16:00）
- 选择清晨或傍晚运动
- 推荐：游泳、晨跑、太极拳
- 运动强度：中等强度，避免大汗

**情志：**
- 保持心情平和，避免烦躁
- 静心养神，避免大喜大悲

#### 体质专项提醒

| 体质 | 特别提醒 |
|------|---------|
| 气虚质 | 避免大汗伤气，运动适度 |
| 阳虚质 | 可适当接受阳光，补充阳气 |
| 阴虚质 | 注意补充水分，避免过度出汗 |
| 湿热质 | 清热利湿，预防痤疮 |

#### 代表性方案

**食疗方：苦瓜炒鸡蛋**
```
材料：苦瓜 1 根，鸡蛋 3 个，盐适量
做法：苦瓜切片焯水（去苦味），鸡蛋打散炒熟，加入苦瓜翻炒，加盐调味
功效：清热解暑，明目解毒
频率：每周 3-4 次
```

**茶饮：酸梅汤**
```
材料：乌梅 15g，山楂 15g，甘草 5g，冰糖适量
做法：材料加水煮沸 15 分钟，加冰糖调味，放凉饮用
功效：生津止渴，收敛汗液
频率：每日 1-2 杯
```

### 八、小满（5 月 20-22 日）

**气候特点：** 麦粒渐满，气温升高，雨水增多

**养生原则：** 清热利湿，预防皮肤病

#### 通用建议

**起居：**
- 保持室内干燥通风
- 勤换衣物，保持皮肤清洁
- 避免长时间待在空调房

**饮食：**
- 多吃清热利湿食物：冬瓜、黄瓜、薏米
- 少吃辛辣油腻食物
- 避免过量饮酒

**运动：**
- 增加运动量，促进排汗
- 推荐：游泳、慢跑、羽毛球
- 运动后及时洗澡

**皮肤护理：**
- 保持皮肤清洁干燥
- 预防痱子、湿疹

#### 体质专项提醒

| 体质 | 特别提醒 |
|------|---------|
| 湿热质 | 重点清热利湿，预防皮肤病 |
| 痰湿质 | 加强运动，促进代谢 |
| 特禀质 | 预防皮肤过敏 |

#### 代表性方案

**食疗方：冬瓜薏米排骨汤**
```
材料：冬瓜 500g（带皮），薏米 50g，排骨 500g，生姜 3 片
做法：排骨焯水后与薏米、生姜同炖 1 小时，加入冬瓜再炖 30 分钟，加盐调味
功效：清热利湿，消肿解毒
频率：每周 2-3 次
```

### 九、芒种（6 月 5-7 日）

**气候特点：** 麦收时节，气温高，湿度大

**养生原则：** 防暑祛湿，调理脾胃

#### 通用建议

**起居：**
- 保持规律作息，避免熬夜
- 适当午休
- 避免长时间暴露在烈日下

**饮食：**
- 饮食清淡，易消化
- 多吃瓜果蔬菜
- 少吃油腻、生冷食物
- 推荐食物：绿豆、冬瓜、丝瓜

**运动：**
- 避免在高温高湿环境下运动
- 选择清晨或傍晚
- 推荐：游泳、太极拳

**防暑：**
- 多喝水，补充盐分
- 随身携带防暑药物（藿香正气水）

#### 体质专项提醒

| 体质 | 特别提醒 |
|------|---------|
| 气虚质 | 避免过度出汗，注意补气 |
| 痰湿质 | 重点祛湿，饮食清淡 |
| 阳虚质 | 避免过度贪凉 |

#### 代表性方案

**食疗方：绿豆粥**
```

```

## 四季养生总结

### 春养肝
- **原则：** 疏肝理气，助阳生发
- **饮食：** 多吃绿色蔬菜，适当吃酸
- **运动：** 温和运动，多到户外
- **情志：** 保持心情舒畅

- **原则：** 养心安神，防暑降温
- **饮食：** 清淡饮食，多吃瓜果
- **运动：** 避免高温时段，选择清晨傍晚
- **情志：** 保持心情平和

- **原则：** 养肺润燥，滋阴润肺
- **饮食：** 多吃滋阴润燥食物
- **运动：** 温和运动，避免大汗
- **情志：** 避免悲伤情绪

### 冬养肾
- **原则：** 补肾藏精，温阳散寒
- **饮食：** 温补为主，多吃温性食物

- **情志：** 保持心情平和
