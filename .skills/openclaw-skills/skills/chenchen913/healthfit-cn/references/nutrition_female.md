## 一、女性营养特点

### 生理特点

| 特点 | 营养影响 |
|------|---------|
| 基础代谢率低（比男性低 10-15%） | 热量需求较低 |
| 肌肉量小（比男性低 30-40%） | 蛋白质需求略低 |
| 体脂率高（健康范围 20-30%） | 需要足够脂肪支持激素 |
| 月经周期 | 铁需求高，营养需求随周期变化 |
| 骨骼密度较低 | 钙、维生素 D 需求高 |
| 妊娠期特殊需求 | 叶酸、铁、钙等需求增加 |

### 激素特点

**雌激素：**
- 促进脂肪储存（尤其臀部、大腿）
- 保护骨骼健康
- 影响食欲和代谢

**孕激素：**
- 黄体期升高，增加食欲
- 提高基础体温
- 影响水分潴留

### 营养目标优先级

| 年龄段 | 主要目标 | 营养重点 |
|--------|---------|---------|
| 18-25 岁 | 建立骨量峰值、规律月经 | 钙、维生素 D、铁、充足热量 |
| 26-35 岁 | 维持健康、备孕/孕期 | 叶酸、铁、均衡营养 |
| 36-45 岁 | 维持肌肉、预防骨质疏松 | 蛋白质、钙、维生素 D |
| 46-55 岁（围绝经期） | 缓解更年期症状、保护骨骼 | 钙、维生素 D、植物雌激素 |
| 55 岁以上（绝经后） | 预防骨质疏松、心血管健康 | 钙、维生素 D、Omega-3 |

## 二、月经周期营养

### 周期概述

**标准周期：** 28 天（21-35 天均属正常）

| 阶段 | 时间 | 激素特点 | 身体感受 |
|------|------|---------|---------|
| 月经期 | 第 1-5 天 | 雌激素、孕激素低 | 疲劳、腹痛、情绪低落 |
| 卵泡期 | 第 6-14 天 | 雌激素上升 | 精力充沛、状态好 |
| 排卵期 | 第 14-15 天 | 雌激素峰值 | 状态最佳 |
| 黄体期 | 第 15-28 天 | 孕激素上升，后期下降 | 食欲增加、水肿、情绪波动 |

### 各阶段营养策略

#### 月经期（第 1-5 天）

**营养重点：**
- 补充铁（弥补经血损失）
- 温热食物（缓解痛经）
- 充足热量（不要节食）

**推荐食物：**
- 富含铁：红肉、动物肝脏、菠菜、黑木耳
- 温热食物：姜茶、红糖水、热汤
- 富含镁：坚果、黑巧克力（缓解痛经）

**示例饮食：**
```
早餐：红糖姜茶 + 鸡蛋 + 全麦面包
午餐：牛肉炒菠菜 + 米饭 + 热汤
晚餐：鸡汤 + 蔬菜 + 米饭
加餐：坚果 + 红枣
```

**运动建议：**
- 轻度运动：散步、瑜伽、拉伸
- 避免高强度训练
- 听从身体感受，适当休息

---

#### 卵泡期（第 6-14 天）

**营养重点：**
- 利用代谢优势，适合减脂
- 支持训练表现
- 均衡营养

**热量调整：**
- 可比平时减少 100-200kcal（减脂期）
- 或维持 TDEE（增肌期）

**运动建议：**
- 状态最佳时期
- 适合安排高强度训练
- 尝试突破 PR

**示例饮食：**
```
早餐：燕麦 + 蛋白粉 + 水果
午餐：鸡胸肉 + 糙米 + 蔬菜
晚餐：鱼 + 红薯 + 蔬菜
加餐：希腊酸奶 + 坚果
```

---

#### 排卵期（第 14-15 天）

**营养重点：**
- 维持均衡饮食
- 充足水分
- 抗氧化食物

**运动建议：**
- 状态峰值
- 适合安排重要训练或比赛

---

#### 黄体期（第 15-28 天）

**营养重点：**
- 应对食欲增加
- 缓解水肿
- 稳定情绪

**策略：**
1. **增加热量（不要硬抗）：**
   - 黄体期基础代谢增加 100-300kcal/天
   - 可适当增加健康碳水和脂肪

2. **控制食欲：**
   - 增加蛋白质和膳食纤维
   - 选择低 GI 碳水
   - 少食多餐

3. **缓解水肿：**
   - 减少盐分摄入
   - 增加钾摄入（香蕉、土豆、菠菜）
   - 充足水分（ paradoxically 帮助排水）

4. **稳定情绪：**
   - 富含 Omega-3 食物
   - 富含 B 族维生素食物
   - 黑巧克力（适量）

**示例饮食：**
```
早餐：全麦面包 + 鸡蛋 + 牛油果
午餐：三文鱼 + 藜麦 + 蔬菜
晚餐：瘦牛肉 + 土豆 + 蔬菜
加餐：香蕉 + 坚果 + 黑巧克力（85%+）
```

**运动建议：**
- 降低训练强度
- 增加恢复时间
- 瑜伽、冥想帮助缓解情绪

---

### 周期营养模板

```
【月经期】（第 1-5 天）
热量：TDEE
蛋白质：1.6g/kg
碳水：3-4g/kg
脂肪：1.0g/kg
重点：补铁、温热食物

【卵泡期】（第 6-14 天）
热量：TDEE - 100kcal（减脂期）
蛋白质：1.8g/kg
碳水：3-4g/kg
脂肪：0.9g/kg
重点：利用代谢优势

【黄体期】（第 15-28 天）
热量：TDEE + 100-200kcal
蛋白质：1.8g/kg
碳水：3-4g/kg
脂肪：1.0g/kg
重点：应对食欲、缓解水肿
```

## 三、基础营养参数

### 热量需求计算

**基础代谢率（BMR）- Mifflin-St Jeor 公式：**

```
BMR = (10 × 体重 kg) + (6.25 × 身高 cm) - (5 × 年龄) - 161
```

**示例：** 28 岁女性，165cm，55kg
```
BMR = (10 × 55) + (6.25 × 165) - (5 × 28) - 161
    = 550 + 1031.25 - 140 - 161
    = 1,280 kcal/天
```

**每日总能量消耗（TDEE）：**

```
TDEE = BMR × 活动系数

活动系数：
- 久坐（几乎不运动）：1.2
- 轻度活动（每周 1-3 次运动）：1.375
- 中度活动（每周 3-5 次运动）：1.55
- 高度活动（每周 6-7 次运动）：1.725
- 极高活动（体力劳动 + 每天训练）：1.9
```

**示例：** 中度活动
```
TDEE = 1,280 × 1.55 = 1,984 kcal/天
```

### 宏量营养素配比

#### 蛋白质

| 目标 | 推荐量 | 说明 |
|------|--------|------|
| 增肌 | 1.6-2.0g/kg | 支持肌肉合成 |
| 减脂 | 1.8-2.2g/kg | 保护肌肉，增加饱腹感 |
| 维持 | 1.2-1.6g/kg | 基础需求 |
| 孕期 | 1.1g/kg（孕前体重） | 支持胎儿发育 |
| 哺乳期 | 1.3g/kg（孕前体重） | 支持乳汁分泌 |

#### 脂肪

**女性脂肪需求高于男性！**

| 目标 | 推荐量 | 说明 |
|------|--------|------|
| 增肌 | 1.0-1.2g/kg | 激素合成必需 |
| 减脂 | 0.8-1.0g/kg | 不低于 0.8g/kg |
| 维持 | 1.0-1.2g/kg | 激素健康 |
| 月经失调 | 1.2-1.5g/kg | 恢复月经 |

**⚠️ 警告：** 脂肪摄入过低（<0.6g/kg）会导致：
- 月经失调/闭经
- 雌激素水平下降
- 骨质疏松风险增加
- 皮肤干燥、脱发

#### 碳水化合物

```
碳水 = (总热量 - 蛋白质热量 - 脂肪热量) ÷ 4
```

| 目标 | 推荐量 | 说明 |
|------|--------|------|
| 增肌 | 4-6g/kg | 支持训练和恢复 |
| 减脂 | 2-3g/kg | 训练日高，休息日低 |
| 维持 | 3-5g/kg | 根据活动量调整 |

## 四、增肌营养方案

### 女性增肌特点

- 增肌速度比男性慢（睾酮水平低）
- 更容易储存脂肪（雌激素影响）
- 需要更耐心（每月增肌 0.25-0.5kg 已很好）

### 热量目标

```
增肌热量 = TDEE + 250 kcal

预期增重速率：0.25-0.5kg/月
（其中约 30-40% 为肌肉，60-70% 为脂肪）

女性增肌需要更小的热量盈余，
因为更容易储存脂肪
```

### 宏量营养素配比

```
蛋白质：1.6-2.0g/kg
脂肪：1.0-1.2g/kg（支持激素）
碳水：剩余热量（通常 4-5g/kg）

示例（55kg 女性，TDEE 2000kcal）：
- 目标热量：2250 kcal
- 蛋白质：88-110g（352-440 kcal，16-20%）
- 脂肪：55-66g（495-594 kcal，22-26%）
- 碳水：304-351g（1216-1404 kcal，54-62%）
```

### 增肌饮食模板（55kg 女性）

```
【目标热量：2250 kcal】
【蛋白质：95g | 脂肪：63g | 碳水：320g】

【早餐】（450 kcal）
- 燕麦 60g（210 kcal）
- 蛋白粉 1 勺（120 kcal）
- 香蕉 1 根（105 kcal）
- 杏仁 10g（60 kcal）

【加餐】（300 kcal）
- 希腊酸奶 150g（100 kcal）
- 混合坚果 20g（120 kcal）
- 苹果 1 个（80 kcal）

【午餐】（600 kcal）
- 鸡胸肉 150g（165 kcal）
- 糙米饭 150g（180 kcal）
- 西兰花 200g（70 kcal）
- 橄榄油 10ml（90 kcal）
- 混合蔬菜（100 kcal）

【训练前】（300 kcal）
- 全麦面包 2 片（160 kcal）
- 花生酱 1 勺（90 kcal）
- 香蕉 1/2 根（50 kcal）

【训练后】（200 kcal）
- 乳清蛋白 1 勺（120 kcal）
- 香蕉 1 根（105 kcal）

【晚餐】（500 kcal）
- 三文鱼 120g（250 kcal）
- 红薯 150g（135 kcal）
- 芦笋 200g（40 kcal）
- 橄榄油 5ml（45 kcal）
```

## 五、减脂营养方案

### 女性减脂特点

- 减脂速度比男性慢（激素影响）
- 黄体期体重波动正常（水分潴留）
- 不要过度节食（影响月经）

### ⚠️ 重要警告

**女性减脂最低热量：**
```
最低热量 = BMR × 1.2

低于此热量会导致：
- 月经失调/闭经
- 代谢下降
- 肌肉流失
- 脱发、皮肤干燥
- 情绪问题
```

### 热量目标

```
减脂热量 = TDEE - 300-500 kcal

预期减重速率：0.25-0.5kg/周

⚠️ 不要超过 500kcal 赤字！
女性过度节食风险更高
```

### 宏量营养素配比

```
蛋白质：1.8-2.2g/kg（保护肌肉）
脂肪：0.8-1.0g/kg（不低于 0.8g/kg！）
碳水：剩余热量（通常 2-3g/kg）

示例（55kg 女性，TDEE 2000kcal）：
- 目标热量：1600 kcal
- 蛋白质：99-121g（396-484 kcal，25-30%）
- 脂肪：44-55g（396-495 kcal，25-31%）
- 碳水：155-202g（620-808 kcal，39-50%）
```

### 减脂饮食模板（55kg 女性）

```
【目标热量：1600 kcal】
【蛋白质：100g | 脂肪：53g | 碳水：175g】

【早餐】（350 kcal）
- 燕麦 40g（140 kcal）
- 蛋白粉 1 勺（120 kcal）
- 蓝莓 50g（30 kcal）
- 杏仁 10g（60 kcal）

【午餐】（450 kcal）
- 鸡胸肉 120g（132 kcal）
- 糙米饭 100g（120 kcal）
- 西兰花 200g（70 kcal）
- 橄榄油 10ml（90 kcal）
- 混合蔬菜（50 kcal）

【加餐】（200 kcal）
- 希腊酸奶 100g（70 kcal）
- 苹果 1 个（80 kcal）
- 核桃 10g（50 kcal）

【晚餐】（500 kcal）
- 三文鱼 120g（250 kcal）
- 红薯 150g（135 kcal）
- 芦笋 200g（40 kcal）
- 橄榄油 5ml（45 kcal）
- 混合蔬菜（50 kcal）

【加餐】（100 kcal）
- 蛋白粉 1/2 勺（60 kcal）
- 小番茄 100g（20 kcal）
```

### 女性减脂注意事项

1. **不要过度节食**
   - 最低热量：BMR × 1.2
   - 脂肪不低于 0.8g/kg

2. **关注月经**
   - 月经失调 = 热量/脂肪过低
   - 立即增加热量和脂肪

3. **理解体重波动**
   - 黄体期体重增加 1-2kg 正常（水分）
   - 不要因此放弃

4. **耐心**
   - 女性减脂比男性慢
   - 每月减重 1-2kg 是健康的

## 六、女性专项营养素

### 铁（Iron）

**作用：**
- 血红蛋白组成，氧气运输
- 能量代谢
- 免疫功能

**推荐量：** 18mg/天（经期女性）
**绝经后：** 12mg/天

**女性为什么需要更多铁？**
- 月经失血（每月约 30-80mL 血液）
- 妊娠期需求增加
- 素食者吸收率低

**食物来源：**

| 食物 | 含量 (mg/100g) | 吸收率 |
|------|---------------|--------|
| 猪肝 | 22.6 | 高（血红素铁） |
| 牛肉（瘦） | 2.8 | 高（血红素铁） |
| 鸡肉 | 1.5 | 高（血红素铁） |
| 菠菜 | 2.9 | 低（非血红素铁） |
| 黑木耳 | 97.4 | 低（非血红素铁） |
| 红豆 | 7.4 | 低（非血红素铁） |

**提高铁吸收：**
- 与维生素 C 一起食用（橙子、猕猴桃、青椒）
- 避免与茶、咖啡同餐（单宁抑制吸收）
- 素食者需要更高摄入量

**缺乏症状：**
- 疲劳、乏力
- 脸色苍白
- 头晕
- 指甲脆弱
- 脱发

**补充建议：**
- 优先从食物获取
- 确诊缺铁性贫血才补充铁剂
- 铁剂剂量：18-27mg/天（遵医嘱）
- 与维生素 C 一起服用
- 避免与钙、茶、咖啡同服

---

### 钙（Calcium）

**作用：**
- 骨骼健康
- 肌肉收缩
- 神经传导
- 血液凝固

**推荐量：** 800mg/天

**女性为什么需要关注钙？**
- 骨骼密度低于男性
- 绝经后骨质疏松风险高
- 妊娠期需求增加

**食物来源：**

| 食物 | 含量 (mg/100g) |
|------|---------------|
| 牛奶 | 104 |
| 酸奶 | 118 |
| 奶酪 | 799 |
| 豆腐（石膏点） | 138 |
| 小鱼干 | 764 |
| 芝麻酱 | 1170 |
| 菠菜 | 66（草酸影响吸收） |
| 西兰花 | 47 |

**补充建议：**
- 优先从食物获取
- 补充剂剂量：500-600mg/天（如饮食不足）
- 分次服用（单次吸收上限约 500mg）
- 与维生素 D 一起服用
- 避免与铁剂同服

---

### 维生素 D

**作用：**
- 促进钙吸收
- 骨骼健康
- 免疫功能
- 情绪调节

**推荐量：** 10μg (400IU)/天

**实际建议：**
- 日照不足者：1000-2000 IU/天
- 绝经后女性：2000 IU/天
- 冬季或北方地区：建议补充

**食物来源：**

| 三文鱼 | 526 |
| 蛋黄 | 87 |
| 强化牛奶 | 40-50 |

**补充建议：**
- 维生素 D3（胆钙化醇）优于 D2
- 与脂肪一起服用（脂溶性）
- 定期检测血清 25(OH)D 水平

---

**作用：**
- 细胞分裂
- DNA 合成
- 红细胞生成
- 预防胎儿神经管缺陷

**推荐量：** 400μg DFE/天
**备孕/孕早期：** 600μg DFE/天

**食物来源：**

**补充建议：**
- 备孕女性必须补充（孕前 3 个月开始）
- 补充剂剂量：400-800μg/天
- 选择甲基叶酸（5-MTHF）吸收更好

---

**作用：**
- 激素调节
- 蛋白质代谢
- 神经递质合成

## 七、特殊时期营养

### 备孕期

**营养重点：**
- 叶酸：600μg DFE/天（孕前 3 个月开始）
- 铁：18mg/天
- 碘：120μg/天
- 维生素 D：1000-2000 IU/天
- 均衡饮食，达到健康体重

**避免：**
- 酒精
- 生食（生鱼片、生鸡蛋）
- 高汞鱼类（鲨鱼、旗鱼）
- 过量咖啡因（<200mg/天）

---

### 孕期

**热量需求：**
- 孕早期：无需额外热量
- 孕中期：+300kcal/天
- 孕晚期：+450kcal/天

**营养重点：**
- 蛋白质：1.1g/kg（孕前体重）
- 叶酸：600μg DFE/天
- 铁：24mg/天
- 钙：1000mg/天
- 维生素 D：600IU/天
- DHA：200-300mg/天

**避免：**
- 酒精
- 生食
- 高汞鱼类
- 过量咖啡因

---

### 哺乳期

**热量需求：**
- +500kcal/天（前 6 个月）
- +400kcal/天（6 个月后）

**营养重点：**
- 蛋白质：1.3g/kg（孕前体重）
- 钙：1000mg/天
- 维生素 D：600IU/天
- DHA：300mg/天
- 充足水分（+1000mL/天）

---

### 围绝经期（45-55 岁）

**营养重点：**
- 钙：1000mg/天
- 维生素 D：1000-2000 IU/天
- 蛋白质：1.2-1.5g/kg（预防肌肉流失）
- 植物雌激素：大豆异黄酮
- Omega-3：心血管健康

**缓解更年期症状：**
- 大豆制品（豆腐、豆浆）
- 亚麻籽
- 避免辛辣、咖啡因（潮热）
- 规律运动

---

### 绝经后（55 岁+）

**营养重点：**
- 钙：1000-1200mg/天
- 维生素 D：1000-2000 IU/天
- 蛋白质：1.2-1.5g/kg
- 维生素 B12：吸收下降，可能需要补充
- 膳食纤维：预防便秘

**预防：**
- 骨质疏松
- 心血管疾病
- 认知功能下降

## 九、常见问题

### Q1：女性健身会变"金刚芭比"吗？

**答：** 不会。

- 女性睾酮水平只有男性的 1/10-1/20
- 自然训练很难练出大块肌肉
- 女性健身会变得更紧致、有线条
- "金刚芭比"照片多是职业选手 + 极端饮食 + 可能用药

---

### Q2：经期可以运动吗？

**答：** 可以，但需调整。

**经期（第 1-5 天）：**
- 轻度运动：散步、瑜伽、拉伸
- 避免高强度、倒立动作
- 听从身体感受

**经后（第 6 天起）：**
- 逐渐恢复训练
- 卵泡期状态最佳

---

### Q3：减脂期月经停了怎么办？

**答：** 立即增加热量和脂肪！

**原因：**
- 热量过低
- 脂肪摄入过低
- 体脂率过低（<17%）
- 压力过大

**解决方案：**
1. 增加热量 300-500kcal/天
2. 增加脂肪至 1.0-1.2g/kg
3. 减少训练量
4. 如 3 个月未恢复，就医

**警告：** 长期闭经会导致骨质疏松！

---

### Q4：孕期可以运动吗？

**答：** 可以，但需调整。

**安全运动：**
- 散步
- 游泳
- 孕妇瑜伽
- 轻度力量训练

**避免：**
- 接触性运动
- 高冲击运动
- 仰卧动作（孕中期后）
- 高温环境运动

**建议：** 运动前咨询医生

---

### Q5：素食女性如何获取足够营养？

**答：** 可以，但需要规划。

**重点关注：**
- 蛋白质：豆制品、豆类、藜麦、坚果
- 铁：菠菜、黑木耳、红豆 + 维生素 C
- 钙：豆腐、绿叶蔬菜、芝麻酱
- 维生素 B12：必须补充（植物性食物几乎不含）
- 锌：南瓜子、芝麻、豆类

**建议：**
- 蛋白质需求略高（吸收率低）
- 补充维生素 B12
- 多样化食物

---

### Q6：如何缓解经前综合征（PMS）？

**答：** 营养和生活方式调整有帮助。

**营养策略：**
- 维生素 B6：50-100mg/天
- 镁：200-400mg/天
- 钙：1000mg/天
- Omega-3：1-2g/天
- 减少盐分（缓解水肿）
- 减少咖啡因（缓解情绪波动）

**生活方式：**
- 规律运动
- 充足睡眠
- 压力管理
- 瑜伽、冥想

---

### Q7：产后如何恢复身材？

**答：** 耐心 + 科学方法。

**时间线：**
- 产后 6 周内：恢复为主，轻度活动
- 产后 6 周 -6 个月：逐渐恢复训练
- 产后 6 个月 +：正常训练

**营养：**
- 哺乳期：+500kcal/天
- 高蛋白：1.3g/kg
- 充足水分

**运动：**
- 先恢复核心和盆底肌
- 逐渐增加强度
- 不要急于减重

**警告：** 产后抑郁很常见，如有情绪问题及时就医。
