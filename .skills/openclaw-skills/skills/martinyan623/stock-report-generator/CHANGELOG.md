## [1.0.0] - 2026-03-22

### 🎉 初始版本发布

A股股票复盘报告自动生成工具正式上线，支持任意A股股票分析！

### ✨ 新增功能

#### 报告生成
- **午间复盘报告** — 支持11:30后快速生成，包含实时行情、技术面、基本面速览
- **全天复盘报告** — 支持15:00后深度分析，包含8大完整模块
- **自动文件命名** — 按照 `{股票名}_{午间复盘|全天复盘}_{YYYYMMDD}.md` 规范
- **支持任意A股** — 通过参数传入任意股票名称和代码

#### 多维度评分体系
- **技术面评分** (30分) — 趋势判断、均线位置、量价配合
- **估值水平评分** (25分) — PE/PB历史分位、股息率
- **基本面评分** (25分) — 业绩增速、ROE、财务健康
- **资金面评分** (20分) — 资金流向、市场情绪
- **综合评级** — A/B/C/D 四级评级标准

#### 可视化功能
- **四维评分雷达图** — 自动生成 matplotlib 雷达图
- **良好线/及格线标注** — 80分良好线、60分及格线参考
- **中文标签支持** — 支持 WenQuanYi Zen Hei 字体

#### 趋势分析
- **评分趋势对比** — 与昨日/上周评分自动对比
- **评级变化预警** — 升级/降级/持平标识
- **变化趋势可视化** — 表格展示变化幅度

#### 报告模块

**午间复盘（8大模块）**：
1. 实时行情 — 当前价、涨跌幅、成交量、量比、市盈率
2. 技术面分析 — K线形态、均线系统、支撑压力、量能分析
3. 基本面速览 — 估值水平、业绩观点、行业动态
4. 市场环境 — 所属板块表现、龙头联动
5. 大盘与资金面 — 上证指数、主力资金流向
6. 午盘操作策略 — 持仓者建议、空仓者建议、短线/中线策略
7. 关键价位提醒 — 支撑位、压力位、止损位
8. 风险提示

**全天复盘（+4个深度模块）**：
9. 多维度评分体系 + 雷达图可视化
10. 全天行情回顾 — 核心数据、日内走势、量能分析
11. 技术面深度分析 — 日K线形态、均线系统、趋势判断
12. 基本面跟踪 — 估值水平、业绩回顾、机构持仓
13. 市场环境研判 — 板块表现、资金流向、行业新闻
14. 次日及本周策略 — 情景分析、持仓建议、重点观察指标
15. 风险提示 — 下行风险因素、关键观察节点
16. 总结 — 今日点评、核心观点、策略建议

### 📦 技术实现

- **生成脚本** — `scripts/generate_report.py` 完整实现，支持命令行参数
- **评分引擎** — 基于A股标准化评分逻辑
- **雷达图生成** — matplotlib 可视化模块
- **通用模板** — 支持任意A股股票，通过参数传入

- **SKILL.md** — 完整工作流程和报告模板
- **README.md** — 项目介绍和使用指南

### 🛠️ 工具依赖

- Python 3.10+
- kimi_finance — 股票实时行情
- matplotlib — 雷达图生成
- memory_search — 历史评分查询（可选）

### 💡 使用示例

```bash
# 命令行使用
python generate_report.py 贵州茅台 600519.SH 全天复盘 2026-03-21
python generate_report.py 宁德时代 300750.SZ 午间复盘 2026-03-21

# 对话使用
"生成贵州茅台 2026-03-21 全天复盘报告"
"生成宁德时代 2026-03-21 午间复盘报告"
```

### 📝 已知限制

- 当前使用示例数据，实际部署需接入 kimi_finance 获取实时数据
- 雷达图生成需要安装 WenQuanYi Zen Hei 中文字体
- 历史评分对比需配合 memory_search 使用
