# whoislookup 🔍

Domain WHOIS lookup tool. Query registrar, creation/expiry dates, nameservers, and status for any domain. Supports 17+ TLDs with auto-detection. Zero dependencies — pure Python stdlib sockets.

## Usage
```bash
python3 whoislookup.py github.com example.org
python3 whoislookup.py --raw example.com
python3 whoislookup.py --json openclaw.ai
```
