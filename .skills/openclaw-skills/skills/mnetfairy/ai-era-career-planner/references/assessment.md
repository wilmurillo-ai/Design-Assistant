# 职业测评框架

> 本文件提供标准职业兴趣测评工具，供职业规划对话中使用。

---

## 测评一：霍兰德职业兴趣六型（RIASEC）

通过对话引导用户选择，获取其霍兰德代码。

### 六大类型简述

| 类型 | 代码 | 关键词 | 典型职业 |
|------|------|--------|---------|
| 现实型 | R | 动手、操作、机械 | 工程师、厨师、机械师 |
| 研究型 | I | 研究、探索、分析 | 科学家、医生、分析师 |
| 艺术型 | A | 创意、表达、美感 | 设计师、作家、音乐家 |
| 社会型 | S | 助人、沟通、教育 | 教师、心理咨询师、护士 |
| 企业型 | E | 领导、说服、管理 | 企业家、销售、管理者 |
| 常规型 | C | 规整、组织、执行 | 会计、行政、审计 |

### 快速判断对话

请用户回答三个问题：
1. "你更享受一个人安静做事，还是和很多人一起？"
2. "你更喜欢具体的事实和数据，还是抽象的概念和创意？"
3. "你更在意做事的规则和流程，还是做事的自由和变化？"

组合判断：
- 安静+事实+规则 → R（现实型）
- 安静+概念+自由 → I（研究型）或 A（艺术型）
- 一起+概念+自由 → S（社会型）
- 一起+事实+说服 → E（企业型）
- 一起+事实+规则 → C（常规型）

## 测评二：职业价值观五项

问用户："以下三项你最看重哪一项？"

1. **成就感** — 工作能让我有成果、被认可
2. **人际关系** — 工作环境和谐，有人关心我
3. **自主性** — 我能自己安排时间，做自己想做的事
4. **稳定性** — 这份工作稳定，不会突然消失
5. **成长性** — 这份工作能让我学到东西，不断进步

## 测评三：个人优势识别

问用户："回想过去，让你感觉最自豪的一件事情是什么？用一句话描述你在这件事里扮演的角色。"

分析关键词：
- "我亲自做成了…" → 执行力/动手能力强（R型）
- "我帮到了…" → 同理心/助人能力强（S型）
- "我想到了一个新方法…" → 创新能力（I/A型）
- "我让一群人一起行动了…" → 领导力（E型）
- "我把复杂的事情整理清楚了…" → 分析/条理能力（C型）

---

## 测评四：抗压与工作风格

问用户："如果遇到一件很难的工作，你会怎么做？"

1. 先仔细分析，搞清楚再动手
2. 先做起来，边做边调整
3. 找人商量，请教别人
4. 想有没有新的方法或工具
5. 按照规则和流程一步步处理
