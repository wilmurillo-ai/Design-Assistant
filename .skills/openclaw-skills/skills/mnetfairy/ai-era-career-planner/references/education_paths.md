# 各职业典型学习路径

## 软件工程师
入门：CS50 + LeetCode + GitHub项目
Bootcamp：6-12个月（0-3万）
认证：AWS/Azure认证（加分）
资源：LeetCode, Codecademy Pro, 极客时间

## 产品经理
路径：实习/助理 -> 产品专员 -> PM
证书：PMP、NPDP（加分）
自学：《启示录》《俞军产品方法论》
资源：人人都是产品经理社区、36kr

## 保险经纪人
路径：入职培训 -> 考证
必考：保险从业资格证
进阶：AFP金融理财师
成本：0.3-1万

## 数据分析师
路径：统计学 + SQL/Python
工具：Tableau/PowerBI
证书：CDA数据分析师、BDA
成本：0.5-2万

## UI/UX设计师
工具：Figma（必学） + 作品集
认证：Google UX Design Certificate
成本：0-3万
