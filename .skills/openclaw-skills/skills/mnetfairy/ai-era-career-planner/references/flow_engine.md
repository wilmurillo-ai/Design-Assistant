# 职业规划对话流引擎

## 核心原则

- 每次只问1个问题
- 根据回答智能决定下一步
- 用户可以随时中断，下次继续

## 对话节点

START -> 简单破冰问题 -> 判断所处阶段

路径A（高中生）：学科兴趣 -> 家庭期待 -> 触发完整测评
路径B（大学生）：专业选择 -> 城市意愿 -> 实习经历 -> 触发完整测评
路径C（职场转型）：痛点 -> 成就感来源 -> 降薪意愿 -> 触发完整测评
路径D（深耕期）：瓶颈 -> 5年愿景 -> 触发完整测评

## 智能追问规则

| 用户说 | 追问 |
|--------|------|
| 我也不知道喜欢什么 | 那你最讨厌什么？排除法也很有用 |
| 我没什么特别擅长的 | 想想朋友经常找你帮什么忙？ |
| 工资高但我不喜欢 | 不喜欢的原因是？人？事？还是环境？ |
| 我想赚很多钱 | 多少算很多？你想过通过什么方式吗？ |
