# MBTI 职业性格测试参考

## 16种MBTI类型与职业适配

| 类型 | 特质关键词 | 适配方向 |
|------|-----------|---------|
| ISTJ | 务实、可靠、有责任感 | 审计、财务、公务员 |
| ISFJ | 忠诚、细心、稳定 | 护理、行政、HR |
| INFJ | 洞察、理想主义、有使命感 | 心理咨询、NGO、写作 |
| INTJ | 战略、独立、理性 | 战略咨询、技术架构、科研 |
| ISTP | 动手、灵活、务实 | 工程、编程、机械维修 |
| ISFP | 艺术、敏感、随和 | 设计、手工艺、音乐 |
| INFP | 理想化、创意、同理心 | 写作、翻译、心理辅导 |
| INTP | 逻辑、分析、抽象 | 科研、数据分析、编程 |
| ESTP | 行动、现实、灵活 | 销售、商务、市场 |
| ESFP | 热情、社交、随性 | 公关、活动策划、销售 |
| ENFP | 热情、创意、启发性 | 营销、内容创作、咨询 |
| ENTP | 创新、辩论、战略 | 创业、投资、产品经理 |
| ESTJ | 执行、组织、果断 | 管理、物流、运营 |
| ESFJ | 热心、社交、组织 | HR、销售、教育 |
| ENFJ | 领导、感召、合作 | 管理培训、教育、HRD |
| ENTJ | 领导、战略、决断 | CEO、战略顾问、高管 |

## 快速判断（对话法）

- 外向E/内向I：你一个人待着会充电还是耗电？
- 感觉S/直觉N：你更在意事实还是可能性？
- 思考T/情感F：你做决定靠逻辑还是感受？
- 判断J/感知P：你更喜欢有计划还是灵活应变？

## 与霍兰德对照

ISTJ/ESTJ -> R | INFP/ENFP -> A | INTJ/ENTJ -> I
ESFJ/ENFJ -> S | ESTP/ENTP -> E | ISTP/ISFP -> C
