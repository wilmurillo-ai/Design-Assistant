# 职业锚测试参考

## 八大职业锚

| 锚型 | 核心驱动力 | 典型职业 |
|------|-----------|---------|
| 技术/职能型 | 专业技术能力，不愿做管理 | 工程师、医生、设计师 |
| 管理型 | 承担整合责任，渴望晋升 | GM、总监、创业者 |
| 自主/独立型 | 自己作主，不受约束 | 自由职业、顾问、投资人 |
| 安全/稳定型 | 可预期、风险低 | 公务员、财务、HR |
| 创业型 | 创造属于自己的东西 | 创始人、发明家、自媒体 |
| 服务/奉献型 | 改变他人命运 | 医生、教师、NGO |
| 纯粹挑战型 | 解决难题，击败强者 | 咨询、竞技、销售 |
| 生活方式型 | 工作服务于生活，不是相反 | 设计师、作家、健身教练 |

## 快速判断三问

1. 给你500万，你还做现在的工作吗？ -> 安全锚
2. 如果必须选：带团队 or 做专家？ -> 管理锚 vs 技术锚
3. 你愿意为创业赌上所有储蓄吗？ -> 创业锚
