# 行业研究执行框架详细版

本文档为 SKILL.md 六大维度的详细执行指南，包含每个步骤的具体操作方法和检查清单。

## 第一步：行业定义与概览 — 详细执行

### 1.1 行业边界划定
- 明确行业的标准分类（GICS / 申万 / 中信行业分类）
- 划清相邻行业边界（如"AI芯片"是半导体子类还是AI基础设施？取决于分析角度）
- 注明分析口径：全球 / 中国 / 特定区域

### 1.2 市场规模测算
- **自上而下**：从大行业规模 × 渗透率/占比 → 细分市场规模
- **自下而上**：主要玩家营收加总 × 调整系数 → 市场规模
- 两种方法互相验证，偏差>20%需要解释原因
- 标注数据来源（如"IDC 2024年报告"、"公司年报加总"）

### 1.3 增速判断
- 历史5年CAGR
- 未来3-5年预测CAGR及依据
- 识别增速拐点（加速/减速的驱动因素）

### 1.4 行业生命周期
| 阶段 | 特征 | 投资策略 |
|------|------|----------|
| 萌芽期 | 技术不成熟、市场小、增速高但不稳定 | 高风险高回报，选技术领先者 |
| 成长期 | 渗透率快速提升、玩家增多、盈利能力改善 | 黄金投资期，选龙头 |
| 成熟期 | 增速放缓、集中度提升、现金流稳定 | 选有护城河和分红的龙头 |
| 衰退期 | 市场萎缩、技术替代 | 回避或只做空 |

## 第二步：产业链深度拆解 — 详细执行

### 2.1 绘制产业链图谱
用以下格式描述产业链：

```
上游 → 中游 → 下游
[具体环节A] → [具体环节B] → [具体环节C]
  代表公司          代表公司          代表公司
  毛利率XX%        毛利率XX%        毛利率XX%
```

### 2.2 价值链分析
对每个环节评估：
- **营收规模**：该环节的市场规模
- **毛利率**：哪个环节最赚钱
- **进入壁垒**：技术/资金/时间/关系
- **集中度**：该环节是分散还是集中
- **议价能力**：上下游谁更有话语权

### 2.3 关键瓶颈
- 哪个环节是"卡脖子"的？
- 供不应求还是供过于求？
- 国产替代进展如何？

### 风险等级定义
- 🔴 **红色**：高概率 + 高影响，需要重点关注和规避
- 🟠 **橙色**：中等概率或中等影响，需要持续跟踪
- 🟡 **黄色**：低概率或可控影响，了解即可

## 第六步：投资建议 — 详细执行

### 6.1 行业评级标准

| 评级 | 含义 | 对应行动 |
|------|------|----------|
| ⭐⭐⭐⭐⭐ | 强烈看好，高确定性成长 | 积极配置 |
| ⭐⭐⭐⭐ | 看好，较好的风险回报比 | 标配或超配 |
| ⭐⭐⭐ | 中性，机会和风险并存 | 标配，精选个股 |
| ⭐⭐ | 偏谨慎，风险大于机会 | 低配或回避 |
| ⭐ | 看空，重大风险 | 回避 |

### 6.2 核心标的筛选标准
- 行业龙头或细分赛道冠军
- 有明确的竞争优势/护城河
- 财务健康（资产负债率合理、现金流充裕）
- 估值合理（相对历史和同业）
- 有近期催化剂

## 行业特殊模板

### AI / 科技行业额外关注
- 大模型能力进展和成本下降曲线
- 算力供需关系和资本开支周期
- 应用层落地进展和商业化验证
- 开源 vs 闭源生态演进
- 中美科技竞争和出口管制影响

### 新能源行业额外关注
- 装机量和渗透率数据
- 原材料价格走势（锂、硅、钴等）
- 补贴政策变化
- 技术路线竞争（如钠电 vs 锂电）
- 产能过剩风险

### 消费行业额外关注
- 消费者信心指数和支出意愿
- 渠道结构变化（线上渗透率、直播电商）
- 品牌力和提价能力
- 人口结构变化（老龄化、Z世代、下沉市场）
- 国货替代趋势

### 医疗健康行业额外关注
- 研发管线进展和FDA/NMPA审批
- 集采/DRG政策影响
- 创新药 vs 仿制药格局
- 医保支付结构变化
- 人口老龄化红利
