# Plant Care Templates (Top 20 Houseplants)

1. **Monstera Deliciosa (Swiss Cheese Plant)**
   - Base Water: 10 days
   - Light: Bright indirect
   - Toxic: Yes
   
2. **Epipremnum aureum (Golden Pothos)**
   - Base Water: 7-10 days
   - Light: Low to bright indirect
   - Toxic: Yes

3. **Zamioculcas zamiifolia (ZZ Plant)**
   - Base Water: 21-30 days
   - Light: Low to bright indirect
   - Toxic: Yes

4. **Sansevieria trifasciata (Snake Plant)**
   - Base Water: 21-30 days
   - Light: Low to direct
   - Toxic: Yes

5. **Ficus lyrata (Fiddle Leaf Fig)**
   - Base Water: 7-10 days
   - Light: Bright indirect to direct
   - Toxic: Yes

6. **Chlorophytum comosum (Spider Plant)**
   - Base Water: 7-10 days
   - Light: Bright indirect
   - Toxic: No (Pet Safe!)

7. **Spathiphyllum (Peace Lily)**
   - Base Water: 7 days (will dramatically droop when thirsty)
   - Light: Low to medium indirect
   - Toxic: Yes (Highly toxic to cats)

8. **Calathea orbifolia**
   - Base Water: 5-7 days (needs high humidity)
   - Light: Medium indirect
   - Toxic: No (Pet Safe!)

9. **Maranta leuconeura (Prayer Plant)**
   - Base Water: 5-7 days
   - Light: Medium indirect
   - Toxic: No (Pet Safe!)

10. **Pilea peperomioides (Chinese Money Plant)**
    - Base Water: 7-10 days
    - Light: Bright indirect
    - Toxic: No (Pet Safe!)

*(Agent note: Use these baseline frequencies, modified by pot size and season, to generate care schedules).*
