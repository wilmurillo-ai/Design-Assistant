# DocuScan
The Smartest Scanner in Your Pocket. 

DocuScan turns messy photos of receipts, contracts, whiteboards, and handwritten notes into perfect, searchable PDFs—instantly.

## Features
* **Perfect Document Reconstruction:** Reads layout, headers, paragraphs, and tables.
* **Handwriting to Text:** We transcribe messy notes for you.
* **Smart Auto-Naming:** Automatically names your PDFs based on content.
* **Multi-Page Magic:** Send multiple photos and get one perfectly combined PDF.
* **Data Extraction & Tables:** Turns spreadsheets and receipts into digital tables.
* **Format Flexibility:** Get Word docs, text files, or markdown instead of a PDF.

## Requirements
* Python 3.x
* Playwright (`pip install playwright && playwright install chromium`)

## Installation
1. Run the setup prompt found in `SETUP-PROMPT.md`.
2. Install Playwright if not already installed:
 ```bash
 pip install playwright
 playwright install chromium
 ```

## Trust & Security
🛡 **Codex Security Verified** 
Your documents are processed securely using your existing AI connection. Our code never phones home, never saves your documents to external servers, and never trains models on your data. What happens in your chat stays in your chat.
