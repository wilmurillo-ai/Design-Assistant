# InvoiceGen

**Stop Paying SaaS Subscriptions Just to Bill Your Clients.**

Say "Bill Acme Corp for 10 hours of design work" and get a perfectly styled, branded PDF invoice in seconds. No forms, no clunky UI, and absolutely no monthly fees.

![Codex Security Verified](https://img.shields.io/badge/Codex-Security_Verified-brightgreen)

## Features
- **Conversational Invoicing:** "Send a $500 invoice to Jane for the logo project, net 30."
- **Client Memory:** Remembers client details for future use.
- **Auto-Numbering:** Sequential numbering handled automatically.
- **Beautiful PDFs:** Professional templates via Playwright.
- **Smart Drafts:** Auto-generates the email copy to send to your client.
- **100% Local & Private:** Your financial data never leaves your machine.

## Quick Start
1. Ensure you have Python and Playwright installed:
 ```bash
 pip3 install playwright
 playwright install chromium
 ```
2. Run the setup prompt with your AI assistant:
 "Initialize InvoiceGen using the SETUP-PROMPT.md file."
3. Start invoicing! "Draft an invoice for TechCorp, 5 hours consulting at $100/hr."
