# EmmeStudio Brand Skill

Applica automaticamente l'identità visiva EmmeStudio a tutti gli output prodotti.

## Quando si attiva

Ogni volta che viene richiesto di produrre un output che rappresenta EmmeStudio verso l'esterno (clienti, prospect, partner) o documenti interni formali.

## Risorse

- **[references/brand-system.md](references/brand-system.md)** — Sistema completo: colori, font, logo, regole visive, palette, tipografia. Leggere SEMPRE prima di produrre qualsiasi output brandizzato.
- **[references/brand-templates.md](references/brand-templates.md)** — Struttura specifica per tipo di documento (carta intestata, preventivi, presentazioni). Leggere quando si produce un documento specifico.

## File brand

### Inclusi nella skill
- **`assets/logo/`** — Loghi principali in SVG e PNG (blu, nero, bianco, compresso, monogramma)

### Su Google Drive (`emmestudio.team@gmail.com` → `CORTEX/BRAND/`)
- **Font:** Ivar Soft Semibold + Open Sauce One (tutti i pesi)
- **Template:** Carta intestata .docx, preventivi .docx, presentazione .key
- **Brand guidelines PDF:** Documento completo DilloStudio (giugno 2024)

## Workflow

1. Leggere `references/brand-system.md` per le regole generali
2. Se il tipo di documento è specifico (preventivo, presentazione, lettera), leggere anche `references/brand-templates.md`
3. Se serve un template .docx o .key, scaricarlo da Drive (`emmestudio.team@gmail.com` → `CORTEX/BRAND/`)
4. Produrre l'output rispettando tutte le regole
5. Verificare: font corretti? Colori dalla palette? Logo usato correttamente? Clausole standard presenti (per preventivi)?

## Regole rapide (sempre applicare)

- **Tinta default:** BLU. Usare la scala blu a meno che non sia specificato diversamente
- **Font titoli:** Ivar Soft Semibold (fallback: Georgia Bold)
- **Font corpo:** Open Sauce One Regular (fallback: Arial Regular)
- **Mai mescolare** Ivar Soft e Open Sauce One nello stesso paragrafo
- **Mai usare** gradienti, colori di tinte diverse insieme, Ivar Soft in CAPSLOCK
- **Logo:** Rispettare la clear space (larghezza della "m" su ogni lato)
- **Prezzi:** Formato `€ 1.000,00` (punto = migliaia, virgola = decimali)
- **"Startup"** va sempre con la S maiuscola
