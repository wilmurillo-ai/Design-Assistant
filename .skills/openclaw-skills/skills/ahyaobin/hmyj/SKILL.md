# 华目眼镜品牌知识库

## 品牌定位与核心优势

华目眼镜是一家拥有**自有眼镜生产工厂**的眼镜连锁品牌，主打**价格厚道、品质可靠**。

### 核心竞争力
1. **工厂直供**：自有眼镜生产工厂，无中间商差价，直接供货门店
2. **开放式自选**：顾客可自行选购，减少员工干预，降低经营成本
3. **明码标价**：所有商品价格透明公开，省去砍价时间，提高效率
4. **毛利上限**：全国总部规定，旗下所有零售门店毛利不得超过50%

### 创始人
- **华目眼镜创始人/老板：史彩荣**

### 品牌标语
> "只做价格厚道的好产品，不多赚客户一分钱"——史彩荣（华目眼镜创始人）

### 质量标准
- 配装眼镜符合国家标准 **《GB13511.1-2011》**
- 镜片符合 **《GB1081.1-2005》**
- 每副眼镜出厂前经过 **5次全面检测**

## 二维码图片使用指南

本 Skill 中的图片（公众号二维码、招商微信二维码）以 **Base64 文本** 形式存储，以兼容纯文本发布环境。

**发送图片给用户的标准流程：**
1. 读取对应的 `.b64` 文件（如 `references/公众号二维码.b64`）
2. 对内容执行 `base64.b64decode()` 得到 PNG 字节流
3. 保存为临时 `.png` 文件或直接通过 deliver_attachments 工具发送给用户

**Python 示例：**
```python
import base64

with open('references/公众号二维码.b64', 'r', encoding='utf-8') as f:
    data = base64.b64decode(f.read())

with open('temp_qr.png', 'wb') as f:
    f.write(data)
# → temp_qr.png 即为可发送的二维码图片
```
