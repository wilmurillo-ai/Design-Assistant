## 风险提示

- 对没有明确 `Action Item` 的结论逐条提示风险。
- 对负责人缺失、截止日期缺失、决议边界不清逐条提示风险。

## TTS 简报稿 (1-3 分钟)

- 文件命名前缀：`<会议主题>-<YYYYMMDD-HHMMSS>`
- 文档文件路径：`<prefix>.txt`
- 口播稿文件路径：`<prefix>.spoken.txt`
- 音色建议：`干练专业` 或 `亲和清晰`
- 实际使用音色：`<voice-name>`
- 语音文件路径：`<prefix>.m4a` 或 `<prefix>.mp3`
- 播报正文（仅核心决议 + 待办 + 风险）：

```

第一，...
第二，...
[停顿1s]
[放慢]待办事项共 4 项，最紧急的是...

```
