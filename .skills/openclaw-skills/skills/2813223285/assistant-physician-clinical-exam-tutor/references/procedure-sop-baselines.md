# Procedure SOP Baselines

Use as baseline for blind step comparison and deduction.

## 1) 导尿术 (key checkpoints)

1. 核对身份、适应证、禁忌证、解释与同意
2. 手卫生、物品准备、无菌铺巾
3. 会阴清洁消毒（方向正确、范围合理）
4. 无菌持管进尿道（动作轻柔、深度合适）
5. 见尿后再进少许并固定
6. 回抽/连接引流系统并保持闭合
7. 固定尿管、整理体位、记录尿量与反应

High-risk errors:

- 破坏无菌区
- 暴力置管
- 未确认禁忌或损伤风险

## 2) 心肺复苏 (adult basic flow checkpoints)

1. 环境安全评估与意识/呼吸判断
2. 呼救并启动急救系统
3. 高质量胸外按压（频率、深度、回弹、尽量减少中断）
4. 开放气道与有效通气
5. 尽早除颤（如适用）
6. 持续循环并动态复评

High-risk errors:

- 延误按压
- 按压-通气比例错误且无法自纠
- 忽略除颤指征

## 3) 胸膜腔穿刺术 (key checkpoints)

1. 指征/禁忌证核查与知情同意
2. 体位摆放与定位（体表标志明确）
3. 手卫生、消毒、铺巾、局麻
4. 按层进针、负压抽吸、控制进针深度
5. 抽液量控制与过程观察
6. 术后敷料、复评、记录与并发症监测

High-risk errors:

- 穿刺点定位错误
- 无菌流程破坏
- 术后监测缺失

## Scoring Mapping

- 完整且顺序正确: 2
- 有提及但不完整或表达模糊: 1
- 缺失/顺序颠倒/危险错误: 0

For each high-risk error, apply additional penalty: -1 to -2.
