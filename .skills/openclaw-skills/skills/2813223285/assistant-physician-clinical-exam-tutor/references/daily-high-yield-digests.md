# Daily High-Yield Digests

Use this file to generate one daily compact summary for fast recall.

## Output Contract

1. Print table first.
2. Keep rows high-yield only.
3. End with exactly three lines:
`今日必背3点`
`今日高频陷阱`
`明日预告`

## Example 1: 肺炎分型鉴别

```text
| 鉴别要点 | 大叶性肺炎 (Lobar) | 支气管肺炎 (Lobular) | 间质性肺炎 (Interstitial) |
|---|---|---|---|
| 常见病原体 | 肺炎链球菌 | 细菌、病毒、支原体 | 病毒、支原体 |
| 主要病变部位 | 肺大叶、肺段 | 肺小叶为中心 | 肺间质（支气管壁、肺泡壁） |
| 典型体征 | 肺实变体征（语颤增强、叩诊浊音） | 湿啰音（多在背部、两肺下野） | 体征不明显，或有干啰音 |
| X线表现 | 大片均匀致密阴影，呈叶/段分布 | 沿支气管分布的斑点状阴影 | 肺纹理增重、网状或条索状影 |
| 预后/特点 | 纤维素性炎，不破坏肺结构 | 多见于年老体弱者，可破坏肺组织 | 症状与体征常不平行（症状重体征轻） |
```

2. Prioritize user's weak modules first.
3. Every 7th day produce mixed mini-review of last 6 days.
