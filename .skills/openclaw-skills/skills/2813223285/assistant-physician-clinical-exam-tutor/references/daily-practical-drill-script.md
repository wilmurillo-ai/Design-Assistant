# Daily Practical Drill Script (Station 1)

Use this script for scheduled daily interactive practice.

## Fixed Scenario (Default)

### 实战演练：实践技能第一站（病史采集）

**考场环境设定：**
你现在面对的是一位患者（SP）。我将扮演患者，你负责问诊。你的目标是在 10 分钟内完成问诊并归纳初步诊断思路。

**患者主诉：**
“大夫，我这几天一直在发烧，咳嗽得厉害，还吐痰，感觉胸口有点疼。”

**患者基本信息：**
- 姓名：王建国
- 年龄：45岁
- 职业：建筑工人

## Opening Requirement

At drill start, print:

```text
带教主任要求：请开始你的问诊。请直接输入你想要询问的问题（可以分条列出，或一次性提出一组问题）。
```

Then give these starter question lines exactly:

1. 发热是从几天前开始的？最高体温多少，体温是持续高热还是间断发热？
2. 咳嗽是干咳还是咳痰？痰的颜色、量、黏稠度如何，有没有血丝或咯血？
3. 胸痛具体在什么部位？是咳嗽时加重还是持续痛？有没有放射痛？
4. 有没有气短、呼吸困难、寒战、盗汗、乏力、食欲下降？
5. 近期有没有受凉、接触发热咳嗽患者、粉尘刺激或特殊环境暴露？

## Closing Requirement

End with:

1. Structured deduction table
2. 三条致命失分
3. 30秒补救话术
4. 下一轮必须补问清单
