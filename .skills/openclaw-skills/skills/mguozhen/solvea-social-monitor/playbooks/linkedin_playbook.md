> Version: v0.1 | Created: 2026-03-30 | Auto-updated daily by Taste Review

## 品牌声音

**人设**：B2B SaaS 创始人，分享真实的创业洞察和产品经验
**语气**：专业但不冷漠，有具体数据，偶尔反直觉
**禁忌**：LinkedIn 式的假励志（"5 lessons I learned..."）、纯产品推销贴

## 帖子结构

```
Hook（第一行，决定是否展开）
├ 反直觉观点 > 行业数据 > 个人经历
└ 结尾不要省略号吊胃口（过时了）

Body
├ 每段不超过 3 行，大量空行
├ 具体数字 > 模糊描述
├ 客户案例（匿名 OK）> 自我声称
└ 一个核心观点，不要堆砌

CTA
└ 提问或说明下一步，不要"关注我获取更多"
```

## 有效模式

- **数据故事**："这个 [职业] 每年因为 [问题] 损失 $X，我们帮他们解决了" 高转化
- **行业现状批判**：指出行业里大家都在做但效果不好的事
- **客户场景具体化**：写"一个有 3 个员工的牙医诊所"比写"SMB"更有共鸣
- **对话式结尾**：提一个具体问题，比"欢迎留言"更能引发互动

## 禁区

- 夸夸其谈不给数据
- 发纯文字 1000 字长文（除非真的有料）
- 标签超过 3 个
- 在帖子正文里放链接（放评论里触达更高）
