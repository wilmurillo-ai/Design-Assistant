# 小红书大字图 — 详细工作流

## 工具链

| 步骤 | 工具 | 作用 |
|------|------|------|
| 生成配图 | `image_synthesize` | 根据文字描述生成意境图 |
| 分析图片 | `images_understand` | 确认生成效果是否符合预期 |
| 叠加文字 | `image_synthesize` + input_files | 把文字和底图合成为成品 |
| 发送图片 | `message(channel=feishu, action=send, media)` | 通过飞书发出 |

## Prompt 模板

### 生成底图
```
小红书配图风格，[视觉隐喻描述]，深色背景，整体简约高级，文字字体圆润有质感
```

### 叠加文字
在 `image_synthesize` 的 `requests` 中：
- `prompt`：在图片中下方的醒目位置叠加文字「[文案]」，字体大、白色描边
- `input_files`：底图路径

## 廖老师三张图设计单

### 图1：K2.5降智主题
- **文案**：「K2.5 = 降智打击？」
- **风格**：脑雾/断片风格，深色模糊背景，人物面部表情缺失或打码感
- **输出**：`/workspace/xhs/xhs_01_final.png`

### 图2：抽风终止主题
- **文案**：「抽风？按一下就停」
- **风格**：急刹/红色警告风格，红色闪电或叉号，急刹车视觉
- **输出**：`/workspace/xhs/xhs_02_final.png`

### 图3：三层记忆主题
- **文案**：「记忆三层，你在哪层」
- **风格**：层级蛋糕/结构图风格，三层蛋糕或三层书架，清晰分层
- **输出**：`/workspace/xhs/xhs_03_final.png`

## 执行顺序
1. 依次执行图1→图2→图3
2. 每张图：底图生成 → images_understand确认 → 叠加文字 → 发送飞书
3. 三张全部发完后，统一回复廖老师「三张图已全部发出」
