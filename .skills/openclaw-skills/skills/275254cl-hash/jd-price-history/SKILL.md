# 京东价格历史技能

查询京东商品价格历史，识别真实低价，避开"先涨后降"促销套路。

## 功能

1. **价格历史**: 查看商品 90 天/180 天/1 年价格走势
2. **低价提醒**: 设置目标价格，降价时通知
3. **促销分析**: 识别真实优惠还是先涨后降
4. **比价推荐**: 推荐同款更低价商品
5. **涨价预警**: 监控商品涨价趋势
6. **历史最低价**: 显示商品历史最低价格和日期

## 使用方式

```
查询这个商品的价格历史 https://item.jd.com/12345678.html
```

```
这款商品的历史最低价是多少
```

```
设置降价提醒，低于¥500 时通知我
```

```
现在是不是入手的好时机
```
