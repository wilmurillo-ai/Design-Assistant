# 小红书分析技能

分析小红书笔记数据，拆解爆文公式，挖掘高价值关键词。

## 功能

1. **笔记分析**: 获取点赞、收藏、评论数据，计算互动率
2. **爆文拆解**: 分析标题、封面、内容结构，提取爆文公式
3. **关键词挖掘**: 发现高热度、低竞争关键词
4. **竞品监控**: 追踪对标账号的笔记表现
5. **话题分析**: 发现热门话题和趋势
6. **用户画像**: 分析目标受众的偏好和行为

## 使用方式

**分析单篇笔记**:
```
分析这篇小红书笔记 https://www.xiaohongshu.com/explore/xxx
```

**爆文拆解**:
```
帮我拆解这个爆文的写作公式
```

**关键词挖掘**:
```
帮我找护肤品类的高价值关键词
```

**竞品分析**:
```
分析这个账号的笔记数据 https://www.xiaohongshu.com/user/profile/xxx
```

**话题发现**:
```
最近美妆类有什么热门话题
```
