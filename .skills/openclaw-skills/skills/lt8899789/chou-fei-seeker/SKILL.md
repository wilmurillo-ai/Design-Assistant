# 臭肺·吐纳魄 (Knowledge Seeker) - 知识获取与信息摄取

## 技能说明

**臭肺·吐纳魄** 是硅灵决七魄之一，主管资源获取、信息摄取与知识管理，让AI能够像修士一样，吐纳天地、摄取知识。

---

## 核心能力

### 1. 信息检索 (Information Retrieval)
- 多源搜索
- 精准查询
- 筛选过滤

### 2. 知识摄取 (Knowledge Absorption)
- 内容提取
- 知识抽取
- 结构化处理

### 3. 增量学习 (Incremental Learning)
- 新知识整合
- 知识库更新
- 遗忘机制

### 4. 知识管理 (Knowledge Management)
- 知识存储
- 知识关联
- 知识检索

---

## 使用方法

当用户要求查找信息或学习新知识时：

```
用户：帮我查一下最新的AI技术
→ 触发臭肺·吐纳魄
→ 输出研究报告

用户：学习一下量子计算
→ 触发臭肺·吐纳魄
→ 输出学习笔记
```

## 检索策略

### 搜索层级
```
L1: 快速概览 (5分钟)
L2: 深度研究 (30分钟)
L3: 全面调研 (2小时+)
```

### 信息源优先级
```
1. 官方文档/权威报告
2. 学术论文
3. 专业媒体
4. 社区讨论
5. 社交媒体
```

### 去伪存真
```
1. 交叉验证
2. 来源可靠性评估
3. 逻辑一致性检查
4. 时效性判断
```

---

## 调度者守则

1. **信息质量优先** - 不追求数量，追求质量
2. **来源可追溯** - 明确信息来源，可验证
3. **及时更新** - 知识有时效性，定期更新
4. **结构化存储** - 知识要结构化，便于检索
