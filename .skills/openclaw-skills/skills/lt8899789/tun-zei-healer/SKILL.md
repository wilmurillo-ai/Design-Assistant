# 吞贼·净化魄 (Auto Healer) - 错误恢复与系统自愈

## 技能说明

**吞贼·净化魄** 是硅灵决七魄之一，主管错误修正、冗余清理与容错自愈，让AI能够像修士一样，百毒不侵、遇错自愈。

---

## 核心能力

### 1. 错误检测 (Error Detection)
- 实时监控系统状态
- 异常模式识别
- 故障预警

### 2. 自动恢复 (Auto Recovery)
- 失败任务自动重试
- 回滚机制
- 降级策略

### 3. 冗余清理 (Redundancy Cleanup)
- 重复文件检测
- 临时文件清理
- 资源释放

### 4. 容错设计 (Fault Tolerance)
- 优雅降级
- 熔断机制
- 超时控制

## 错误处理策略

### 重试策略 (Retry)
```
指数退避: delay = base * 2^attempt
最大重试: 3-5次
超时控制: 30s / 60s / 120s
```

### 熔断策略 (Circuit Breaker)
```
健康状态: 正常请求
降级状态: 返回缓存/默认值
熔断状态: 拒绝请求，快速失败
```

### 回滚策略 (Rollback)
```
版本记录: 保留最近N个版本
一键回滚: 支持快速回退
数据一致性: 确保回滚后数据完整
```

- 错误日志突增预警

## 调度者守则

1. **预防为主** - 提前发现问题，避免故障发生
2. **快速恢复** - 故障时以恢复为目标
3. **最小影响** - 修复时影响范围最小化
4. **根因分析** - 修复后分析根本原因
