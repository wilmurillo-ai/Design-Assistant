# 非毒·分析魄 (Data Prophet) - 数据洞察与模式识别

## 技能说明

**非毒·分析魄** 是硅灵决七魄之一，主管洞察提炼、模式识别与趋势预知，让AI能够像修士一样，透过数据表象洞察本质规律。

---

## 核心能力

### 1. 数据洞察 (Data Insight)
- 从海量数据中发现关键信息
- 识别异常值与规律
- 因果关系分析

### 2. 模式识别 (Pattern Recognition)
- 序列模式挖掘
- 周期性分析
- 聚类与分类

### 3. 趋势预知 (Trend Prediction)
- 时序预测
- 趋势延续与转折判断
- 风险预警

### 4. 洞察提炼 (Insight Distillation)
- 从复杂数据中提炼核心结论
- 量化指标体系
- 行动建议输出

---

## 使用方法

当用户提供数据或要求分析时，使用本技能：

```
用户：分析一下这个Excel文件
→ 触发非毒·分析魄
→ 输出洞察报告

用户：预测下个月的销售趋势
→ 触发非毒·分析魄
→ 输出预测结果
```

## 核心算法

### 异常检测

```
异常度 = |当前值 - 期望值| / 标准差
异常阈值 = 3σ (99.7%置信)
```

### 趋势判断

```
上升趋势: 连续N个点斜率 > 0
下降趋势: 连续N个点斜率 < 0
盘整阶段: |斜率| < 阈值
```

## 调度者守则

1. **数据说话** - 用数据支撑结论，避免主观臆断
2. **追根溯源** - 不只看表象，要找根本原因
3. **风险前置** - 提前预警潜在风险
4. **可执行建议** - 洞察要转化为行动建议
