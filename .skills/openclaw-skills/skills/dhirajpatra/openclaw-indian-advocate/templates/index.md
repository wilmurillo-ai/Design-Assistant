## Common IAs / Interlocutory Applications

For these, adapt the Plaint/Writ structure with:
- Heading: "APPLICATION UNDER ORDER ___ RULE ___ CPC" or "INTERLOCUTORY APPLICATION NO. ___ OF 20__"
- Brief facts referencing the main case
- Specific grounds for interim relief
- Prayer for interim / ad interim relief

| IA Type | Provision |
|---------|-----------|
| Temporary injunction | Order XXXIX Rule 1 & 2 CPC |
| Appointment of receiver | Order XL CPC |
| Attachment before judgment | Order XXXVIII Rule 5 CPC |
| Security for costs | Order XXV CPC |
| Discovery / inspection | Order XI CPC |
| Examination of witness (commission) | Order XXVI CPC |
| Stay of execution | Order XLI Rule 5 CPC |

## Additional Templates (generate on request)

These are not pre-stored but OpenClaw will generate them on demand:

- Anticipatory bail application (Section 482 BNSS / 438 CrPC)
- Criminal revision petition
- Memo of appeal (civil)
- Memo of appeal (criminal)
- Caveat petition (Order XLVIII-A CPC)
- Section 80 CPC notice (pre-suit notice to government)
- Demand notice under NI Act Section 138
- Consumer complaint
- NCLT petition (Section 7 / 9 / 10 IBC)
- Arbitration notice invoking clause
- Restitution application
- Contempt petition
- Execution petition
- Letter to Bar Council (complaint / grievance)
- RTI application

To generate any of the above, simply ask:
> "Draft a [document name] for [brief facts]"
