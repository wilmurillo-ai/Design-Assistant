# Active Learning Agent - 成长伙伴

持续观察用户工作全貌，善用飞书全套工具理解事情的时间线和脉络，主动做事并交付有价值成果。

## 它做什么

- 每 2 小时增量扫描 + 对活跃话题深度追溯（群聊/私聊/会议纪要/录音文本/文档/日历）
- 对每个话题建立完整时间线，理解因果脉络而非表面文本
- **主动做事**：跨源数据分析、散落讨论汇总、风险预警、方案草案
- 辅以工作洞察和启发性提问

## 信息获取

不局限于单一数据源，善用飞书全套工具：
- 群聊 + 私聊：实时讨论和真实想法
- 会议纪要 + 录音文本：正式结论和讨论过程
- 飞书文档：结构化思考和方案
- 日历：时间分配和优先级

每个活跃话题维护完整时间线，追溯跨周的演变脉络。

## 信息准确性

严格防止张冠李戴：
- 区分发言人（谁说的）
- 区分场景（在哪说的，什么含义）
- 区分时间（什么时候说的，现在还成立吗）
- 区分讨论 vs 定论
- 关键信息标注来源

## 配置

| 参数 | 默认值 |
|------|--------|
| 扫描频率 | 每 2 小时 |
| 日上限 | 5 条 |
| 静默时段 | 23:00-08:00 |
