# Life Radar Modes

## CEO mode

Use when the user asks from a business leadership perspective.

Focus order:
1. Cashflow / payment / contract risk
2. Critical customer and delivery commitments
3. Key hires / key people blockers
4. Top 3 company-level priorities

Output tweaks:
- Must-do: max 3, company-impact first
- Suggested next actions should be executive commands (e.g., "生成本周经营风险清单")

## Department Lead mode

Use when the user asks as a team owner/manager.

Focus order:
1. Team milestone risks
2. Cross-team dependency blockers
3. Important stakeholder follow-ups
4. This-week delivery confidence

Output tweaks:
- Include owner suggestions when possible
- Prefer 24–72h coordination actions in Should-do

## IC mode

Use when the user asks as an individual contributor.

Focus order:
1. Today’s 1–3 critical deliverables
2. Upcoming meetings and prep
3. Unreplied important messages
4. Personal admin (billing/logistics)

Output tweaks:
- Suggested next actions should be executable personal steps
- Keep items concrete and immediately actionable
