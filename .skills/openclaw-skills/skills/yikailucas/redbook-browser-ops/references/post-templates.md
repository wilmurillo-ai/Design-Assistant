# Post Templates

## 1) 无图短文（效率类）

标题：今天开始把生活调成「轻量模式」

正文：
最近在练一个小习惯：
把每天要做的事，压缩成最重要的 3 件。
不追求完美，先把最关键的做完。

结果意外地轻松很多：
- 焦虑少了
- 执行更稳了
- 晚上也不那么内耗了

如果你也经常被待办压得喘不过气，
试试「3件事原则」：先完成，再优化。

## 2) 产品分享（通用）

标题：这周我反复在用的一个工具

正文：
这周高频在用一个工具，最有感的是这三点：
1. 上手快，10 分钟内能跑通
2. 能直接接到我的真实工作流
3. 出错成本低，回滚方便

适合：想提效但不想折腾太久的人。
不适合：追求一步到位、全自动的人。

## 3) 复盘类

标题：这周复盘：我删掉了 3 个低效习惯

正文：
这周做了一件事：不加任务，只删低效动作。

我删掉了：
- 无目的刷信息流
- 同一任务反复切换工具
- 没有截止时间的“伪待办”

下周只保留两条原则：
- 每天先做最难的一件事
- 所有任务都带截止时间
