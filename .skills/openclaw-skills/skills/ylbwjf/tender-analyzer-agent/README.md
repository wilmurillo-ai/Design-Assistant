# 标书智能分析助手

自动分析招标文件和投标文件，提取关键条款，识别风险点，评估中标概率，生成结构化分析报告。

## 核心功能

### 📊 10大分析维度

1. **投标须知分析** - 资格要求、时间节点、形式要求
2. **采购内容分析** - 需求理解、技术规格、验收标准
3. **评标办法分析** - 评分标准、分值权重、价格计算
4. **加分项识别** - 可得分项、得分策略
5. **扣分项/废标项** - 风险点、规避措施
6. **付款方式分析** - 付款节奏、资金占用成本
7. **违约条款分析** - 违约金、单方解除权
8. **服务/维保分析** - 质保期、响应时间、维保成本
9. **敞口风险分析** - 变更、索赔、知识产权
10. **倾向性分析** - 技术参数、业绩要求、资质要求

### 🎯 输出成果

- 风险提示总览（⭐评级）
- 详细分析（10个维度）
- 资金占用成本估算
- 违约风险量化
- 维保成本估算
- 投标建议（中标概率、策略建议）

## 使用方式

### 方式1: 直接发送标书内容

将标书内容复制发送给我，我会自动分析：

```
小色，帮我分析这份标书：
[粘贴标书内容]
```

### 方式2: 分析特定维度

```
分析评标办法：
[粘贴评标办法内容]
```

```
分析付款条款：
[粘贴付款条款内容]
```

```
分析合同条款：
[粘贴合同条款内容]
```

### 方式3: 生成投标策略

```
生成投标策略：
- 项目类型: 政府采购IT项目
- 预算: 1000万
- 评标方法: 综合评分法
- 技术分: 40%，价格分: 40%
```
