```

控制价: 2850万元
```

### 6. 违约条款分析

#### 延期交付违约金
- 延期每天: 合同额的0.5‰
- 最高限额: 合同额的10%
- 延期超过30天: 甲方有权解除合同

**风险量化**:
- 每天违约金: 1.3万元
- 最高违约金: 260万元
- 占合同额比例: 10%

#### 质量不合格责任
- 返工/修复费用由乙方承担
- 质量违约金: 合同额的5%
- 严重不合格: 甲方有权解除合同

#### 单方解除权
**甲方解除条件**:
- 延期超过30天
- 质量严重不合格
- 乙方破产或丧失资质

**解除后果**:
- 乙方退还已付款项
- 乙方赔偿甲方损失
- 乙方承担违约责任

#### 风险评级
⭐⭐⭐ (中等风险)

**风险点**:
- 延期违约金比例偏高(0.5‰/天)
- 单方解除权偏向甲方
- 质量责任较重

**建议**:
- 合理评估工期，避免延期
- 争取降低违约金比例至0.3‰
- 明确质量不合格认定标准
