# 标书分析助手 - 持续学习框架

## 学习目标

通过主动学习，持续丰富标书分析维度，提升分析深度和准确性。

## 学习方向

### 1. 行业维度扩展
- 政府采购（已覆盖）
- 工程建设（房建/市政/交通）
- IT信息化（已覆盖）
- 金融服务（银行/保险/证券）
- 运营商（移动/联通/电信）
- 能源电力（电网/石油/新能源）
- 医疗卫生（医院/医保/公卫）
- 教育科研（高校/科研院所）
- 军工航天（涉密/特种）

### 2. 分析维度深化

#### 新增维度
- **11. 竞争对手分析** - 预判对手策略、优劣势对比
- **12. 历史中标分析** - 历史价格、中标规律
- **13. 政策法规分析** - 最新政策影响、合规要求
- **14. 技术趋势分析** - 技术路线、创新方向
- **15. 供应链分析** - 上下游风险、成本结构
- **16. 地域因素分析** - 地方保护、本地优先
- **17. 季节因素分析** - 招标淡旺季、资金节奏
- **18. 客户画像分析** - 决策风格、偏好特征

#### 现有维度深化
- **评标办法** → 细分到各行业评标特点
- **付款方式** → PPP/EPC/代建等特殊模式
- **违约条款** → 不可抗力、情势变更
- **敞口风险** → 汇率风险、原材料涨价

### 3. 分析方法升级

#### 量化分析
- 中标概率预测模型
- 报价最优区间计算
- 资金成本精算模型
- 风险评估矩阵

#### 智能分析
- 标书倾向性AI识别
- 竞争对手画像生成
- 历史数据趋势分析
- 行业对标分析

## 学习来源

### 官方渠道
- 中国政府采购网政策解读
- 国家发改委招投标政策
- 各行业招投标管理办法
- 财政部/住建部等部委文件

### 专业平台
- 中国招标投标公共服务平台
- 千里马、剑鱼标讯等
- 招投标培训机构
- 行业协会指南

### 案例库
- 中标公告分析
- 废标案例研究
- 质疑投诉案例
- 合同纠纷案例

## 学习机制

### 每日学习（自动）
- 搜索最新招投标政策
- 学习1-2个行业案例
- 积累新的风险识别点

### 每周总结（自动）
- 整理本周学习成果
- 更新分析维度
- 完善知识库

### 每月迭代（手动+自动）
- 评估分析框架有效性
- 增加新的分析维度
- 优化分析方法

## 汇报内容

每次学习后汇报：
1. 学习内容概览
2. 新增分析维度/要点
3. 典型案例分析
4. 知识框架更新
5. 下次学习方向
