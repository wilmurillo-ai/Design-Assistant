# 行业知识: 政府采购

## 行业特点

### 法规体系
- 《中华人民共和国政府采购法》
- 《中华人民共和国招标投标法》
- 《政府采购货物和服务招标投标管理办法》
- 《政府采购非招标采购方式管理办法》

### 采购方式
1. **公开招标** - 主要方式，竞争充分
2. **邀请招标** - 邀请3家以上
3. **竞争性谈判** - 技术复杂、不能确定详细规格
4. **竞争性磋商** - 服务类项目常用
5. **单一来源** - 唯一供应商
6. **询价** - 货物规格标准统一

### 特殊要求
- 国产化要求（优先采购国产）
- 节能环保要求
- 中小企业扶持政策
- 监狱企业、残疾人福利企业政策
- 贫困地区农副产品采购

## 评标特点

### 评标委员会
- 5人以上单数
- 专家不少于总成员2/3
- 专家随机抽取

### 评标方法
- **综合评分法** - 服务类、技术复杂项目
- **最低评标价法** - 货物类、标准产品

### 价格分计算
```
政府采购价格分计算公式:
价格分 = (评标基准价 / 投标报价) × 价格权重

评标基准价:
- 有效投标报价的算术平均价
- 或最低投标报价
- 或去掉最高最低后的平均价
```

## 风险识别

### 政策风险
- 政策变化导致项目取消
- 预算调整导致规模缩减
- 审计严格导致验收困难

### 合规风险
- 围标串标风险
- 弄虚作假风险
- 违规分包风险

### 资金风险
- 财政拨款延迟
- 付款周期长
- 质保金比例高

## 应对策略

### 投标策略
- 充分研究政策要求
- 确保资质合规
- 突出国产化、节能环保
- 关注中小企业政策

### 报价策略
- 合理低价，避免恶性竞争
- 考虑资金成本
- 预留政策风险成本

### 履约策略
- 严格按合同执行
- 做好过程文档
- 配合审计检查

## 典型案例

### 案例1: 某政府采购信息化项目
**背景**: 预算500万，公开招标
**特点**: 要求国产化率80%，节能环保认证
**策略**: 突出国产方案，提供节能证明
**结果**: 中标

### 案例2: 某政府采购服务项目
**背景**: 预算300万，竞争性磋商
**特点**: 技术评分权重高
**策略**: 重点投入技术方案
**结果**: 技术分第一，中标
