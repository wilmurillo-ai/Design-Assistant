# 分析维度12: 历史中标分析

## 分析目的

通过历史中标数据分析，发现中标规律，指导投标决策和报价策略。

## 分析内容

### 1. 历史价格分析

#### 价格趋势
```
近3年同类项目中标价格:
- 2023年: 平均折扣率 85%
- 2024年: 平均折扣率 82%
- 2025年: 平均折扣率 80%

趋势: 价格竞争加剧，折扣率逐年下降
```

#### 价格区间
| 项目规模 | 预算(万) | 中标价区间(万) | 折扣率 |
|----------|----------|----------------|--------|
| 小型 | <500 | 400-450 | 80-90% |
| 中型 | 500-2000 | 1200-1600 | 75-85% |
| 大型 | >2000 | 按项目评估 | 70-80% |

#### 价格影响因素
- 竞争程度（竞争对手数量）
- 项目紧急程度
- 技术复杂度
- 客户预算充裕度
- 关系深浅

### 2. 中标规律分析

#### 中标企业特征
- 资质等级分布
- 企业规模分布
- 地域分布（本地/外地）
- 行业经验分布

#### 中标价格特征
- 平均折扣率
- 价格离散程度
- 最低价中标比例
- 合理价中标比例

#### 时间规律
- 招标淡旺季
- 年底冲刺现象
- 政策影响时点

### 3. 客户偏好分析

#### 评标偏好
- 技术导向型 - 重方案、轻价格
- 价格导向型 - 最低价优先
- 平衡型 - 综合考虑
- 关系导向型 - 重关系、轻竞争

#### 历史合作情况
- 长期合作供应商
- 轮换机制
- 新供应商机会

### 4. 废标/流标分析

#### 废标原因统计
| 废标原因 | 占比 | 典型案例 |
|----------|------|----------|
| 资格不符 | 30% | 资质过期、业绩不足 |
| 形式错误 | 25% | 签字盖章、密封问题 |
| 报价异常 | 20% | 超预算、低于成本 |
| 实质性偏离 | 15% | 技术参数不符 |
| 其他 | 10% | 串通、弄虚作假 |

#### 流标原因统计
- 投标人不足3家
- 所有报价超预算
- 技术方案均不满足
- 招标文件问题

### 5. 质疑投诉分析

#### 质疑热点
- 评分不公
- 倾向性条款
- 程序违规
- 结果异常

#### 投诉结果
- 投诉成功率
- 常见胜诉理由
- 败诉原因分析

## 分析方法

### 数据收集
1. 收集近3年同类项目中标公告
2. 整理中标价格、中标企业信息
3. 统计废标/流标/质疑案例

### 数据分析
1. 计算平均折扣率、价格区间
2. 分析中标企业特征
3. 识别中标规律
4. 发现异常值

### 趋势预测
1. 价格趋势预测
2. 竞争态势预测
3. 中标概率预测

## 分析输出

```markdown

### 价格分析

- 本项目预估中标价: XX万 (预算的80-85%)
- 价格趋势: 竞争加剧，价格下行

### 中标规律
- 中标企业特征: [描述]

### 客户偏好

### 风险提示

```

## 案例：某政务云项目历史分析

### 数据收集
收集近3年10个同类项目中标数据。

### 分析结果
- 平均折扣率: 82%
- 中标企业: 本地企业占60%
- 价格趋势: 逐年下降
- 客户偏好: 技术导向型

### 策略建议
- 报价策略: 预算的80-83%
- 技术策略: 突出创新方案
- 关系策略: 加强本地合作

### 结果
我方以预算81%的价格中标。
