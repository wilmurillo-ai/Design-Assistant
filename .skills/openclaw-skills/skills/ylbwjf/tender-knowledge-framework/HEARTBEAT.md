# 标书分析助手 - 定时学习任务

## 汇报时间
每天 8:00、12:00、16:00、20:00（每4小时一次）

## 学习内容

### 每次学习任务
1. **搜索最新标书知识文章**（3-5篇）
2. **学习招投标技巧视频/图文**（2-3个）
3. **积累行业案例分析**（1-2个）
4. **更新知识框架**

### 学习来源轮换
- **周一**: 政府采购 + 政策法规
- **周二**: 金融行业 + 银行招标
- **周三**: 运营商 + 通信行业
- **周四**: 交通基建 + 工程建设
- **周五**: 央企国企 + 能源行业
- **周六**: 综合技巧 + 投标策略
- **周日**: 案例复盘 + 框架整理

## 汇报内容模板

1. 学习概览（本次学习了多少内容）
2. 核心知识点（3-5个重点）
3. 实用技巧（可直接用的方法）
4. 案例分享（成功/失败案例）
5. 知识框架更新（新增/完善）
6. 下一步计划

## 输出文件
- `memory/tender-learning/YYYY-MM-DD.md` - 每日学习日志
- `tender-knowledge-framework/knowledge/` - 分类知识库
- `tender-knowledge-framework/cases/` - 案例库
