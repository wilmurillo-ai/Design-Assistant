# 招标文件解读技巧

## 一、招标文件结构速览

### 标准招标文件组成
1. **招标公告** - 项目基本信息、资格要求
2. **投标人须知** - 投标规则、注意事项
3. **评标办法** - 评分标准、权重分配
4. **合同条款** - 商务条件、法律约束
5. **技术规范书** - 技术要求、验收标准
6. **投标文件格式** - 编制要求、提交规范

### 快速阅读技巧
1. **先读公告** - 判断是否符合资格条件
2. **再看评分** - 了解竞争重点
3. **细读合同** - 识别风险条款
4. **核对格式** - 避免形式错误

## 三、评分标准拆解方法

### 评分标准类型
1. **综合评分法** - 技术+商务+价格，最常见
2. **最低评标价法** - 价格优先，技术合格即可
3. **性价比法** - 性能/价格比值

### 综合评分法拆解
```
总分 = 技术分(40%) + 商务分(20%) + 价格分(40%)

技术分细项：
- 技术方案 (15分)
- 项目经验 (10分)
- 团队配置 (10分)
- 实施方案 (5分)

商务分细项：
- 企业资质 (10分)
- 财务状况 (5分)
- 信誉评价 (5分)

价格分计算：
- 基准价 = 有效投标平均价
- 价格分 = (基准价/投标价) × 价格权重
```

### 评分策略制定
- **技术分高**的项目 - 突出方案优势、案例匹配
- **价格分高**的项目 - 精准报价、控制成本
- **商务分高**的项目 - 强调资质、信誉、财务

## 四、废标条款识别

### 常见废标情形
1. **资格不符** - 资质等级、业绩要求不满足
2. **形式错误** - 签字盖章、密封、份数
3. **实质性偏离** - 技术参数、商务条款不满足
4. **报价异常** - 低于成本、高于预算
5. **串通投标** - 文件雷同、报价规律性

### 废标风险防范
- 制作**废标条款检查清单**
- 投标前**交叉复核**
- 预留充足时间**形式审查**

## 五、合同条款关键识别

### 高风险条款
| 条款类型 | 风险点 | 应对策略 |
|----------|--------|----------|
| 付款条款 | 预付款低、进度款滞后 | 评估资金压力 |
| 违约责任 | 违约金过高、单方解除权 | 争取合理条款 |
| 知识产权 | 成果归属不清 | 明确权属边界 |
| 争议解决 | 仲裁地不利、适用法律 | 争取公平条款 |
| 变更条款 | 变更权过大、补偿不足 | 限定变更范围 |

## 七、常见错误

1. **只看价格** - 忽视技术分和商务分
2. **资格漏看** - 某一项不满足导致废标
3. **评分误读** - 理解错评分细则
4. **合同轻视** - 中标后才发现条款不利
5. **时间误判** - 准备时间不足
