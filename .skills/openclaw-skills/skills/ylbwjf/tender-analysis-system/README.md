# 标书分析助手

自动化抓取金融、政府、运营商、交通、大型央企等领域的招标信息，智能分析标书内容，按维度生成结构化分析报告。

## 功能特性

### 📊 监控范围
- **政府采购** - 中国政府采购网、各省市政府采购网
- **金融招标** - 各大银行、保险、证券采购平台
- **运营商** - 移动、联通、电信采购与招标网
- **交通基建** - 交通运输厅、铁路局、机场集团
- **大型央企** - 国家电网、中石油、中石化等

### 🔍 分析维度
1. **合同维度** - 合同类型、期限、违约责任、知识产权、保密条款
2. **付款方式** - 付款节奏、预付款比例、质保金、资金占用成本
3. **技术参数** - 核心要求、性能指标、合规要求（等保/密评）
4. **评标办法** - 评标方法、分值权重、资格门槛、废标风险
5. **其他** - 交付周期、售后服务、培训要求、资质门槛

### ⏰ 汇报机制
- **每小时** - 自动抓取更新
- **每2小时** - 重点汇报（高价值项目、紧急截止提醒）
- **实时推送** - 金额>1000万项目立即推送

## 快速开始

### 1. 安装依赖
```bash
cd tender-analysis-system
pip install pyyaml requests
```

### 2. 配置数据源
编辑 `config.yaml`，配置需要监控的网站和推送方式（飞书/邮件）。

### 3. 运行测试
```bash
./run.sh
```

### 4. 设置定时任务
```bash
./cron_setup.sh
```

## 配置说明

### 数据源配置
```yaml
data_sources:
  - name: "中国政府采购网"
    url: "http://www.ccgp.gov.cn/"
    type: "rss"  # rss/api/web
    category: "政府"
    keywords: ["信息化", "数字化", "系统集成"]
    min_amount: 1000000  # 最低金额（元）
    enabled: true
```

### 推送配置
```yaml
notifications:
  feishu:
    enabled: true
    webhook: "https://open.feishu.cn/..."

  email:
    enabled: true
    smtp_server: "smtp.126.com"
    username: "your_email@126.com"
    to: "recipient@example.com"
```

## 注意事项

1. **合规性** - 遵守各网站的robots协议和使用条款
2. **频率控制** - 避免过高频率导致IP被封

4. **时效性** - 招标信息变化快，及时关注更新
