# Podman Desktop

Podman Desktop is the best free and open source tool to work with Containers and Kubernetes for developers. Get an intuitive and user-friendly interface to easily build, manage, and deploy containers and Kubernetes — all from your desktop. ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from container-desktop/container-desktop

## Usage

Run any command: `container-desktop <command> [args]`
