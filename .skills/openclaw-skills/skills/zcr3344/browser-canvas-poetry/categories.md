## Art Categories | 艺术风格分类

本Skill将浏览器原生艺术分为以下核心类别，每个类别包含丰富的文学艺术参考：

### 1. 自然意象类 | Natural Imagery

**描述**: 将自然景观、宇宙星辰、季节流转转化为动态网页艺术

**文学参考**:
- 中国山水诗: 王维"诗中有画，画中有诗"意境
- 日本物哀美学: 樱花飘落、流水落叶的瞬间永恒
- 欧洲浪漫主义: 柯勒律治对海洋的想象
- 现代生态诗学: 玛丽·奥利弗的自然沉思

**视觉关键词**:
```
云雾流动、水波涟漪、星河旋转、四季更迭
日出日落、潮汐涨落、树叶飘零、雪花飞舞
极光流动、雾凇凝结、瀑布飞溅、藤蔓攀爬
```

**星云/宇宙专项** 🌌:

星云是浏览器原生艺术中最具挑战性也最具视觉冲击力的主题。

文学参考：
- 宇宙诗学: 卡尔·萨根《宇宙》的宏大视角
- 东方宇宙观: 道家"气"的概念与星云流动
- 现代天文摄影: 哈勃望远镜的星云图像美学
- 科幻美学: 《2001太空漫游》的星际意象

视觉关键词：
```
气体星云、发射星云、反射星云、暗星云
恒星诞生、超新星遗迹、行星状星云
颜色渐变、体积感、发光、透明度层次
```

技术方案（详见 `references/nebula-rendering.md`）：
| 方案 | 难度 | 效果 | 推荐场景 |
|------|------|------|---------|
| Three.js ShaderMaterial | ⭐⭐⭐ | ⭐⭐⭐⭐ | 实时星云 |
| WebGL Raymarching | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 照片级星云 |
| tsparticles | ⭐⭐ | ⭐⭐⭐ | 快速原型 |

星云生成提示词：
```
创建一个[颜色描述]星云艺术作品

技术要求：
- 使用 Three.js + 自定义ShaderMaterial
- 实现Simplex Noise + FBM噪声叠加
- 颜色方案：[深紫/粉红/亮紫/蓝色渐变]
- 添加Additive Blending发光效果
- 包含缓慢的呼吸动画（2-5秒周期）

可选增强：
- 粒子状恒星点缀
- 鼠标悬停交互
- 景深模糊效果
```

**生成提示词模板**:
```
Create an immersive browser art piece depicting [自然意象描述].
Use CSS animations to simulate [运动特征: 缓慢/流动/飘渺].
Incorporate [色彩方案: 渐变/水墨/霓虹] palette inspired by [文学参考].
The interaction should feel [情绪基调: 冥想的/忧郁的/敬畏的].
Tech stack: [纯CSS/Canvas/WebGL/Three.js]
```

### 2. 几何解构类 | Geometric Deconstruction

**描述**: 通过几何形状的重组、破碎、重组表达现代性焦虑与秩序美感

**艺术流派参考**:
- 立体主义: 毕加索、勃拉克的多视角碎片
- 构成主义: 马列维利的抽象几何力量
- 荷兰风格派: 蒙德里安的纯粹抽象红黄蓝
- 极简主义: 唐纳德·贾德的"少即是多"
- 参数化设计: 扎哈·哈迪德的流动曲线

**视觉关键词**:
```
碎片化、重组、叠加、透明、穿插
尖锐、圆润、对称、黄金比例、斐波那契螺旋
网格系统、点线面构成、负空间利用
```

**生成提示词模板**:
```
Design a geometric deconstruction art piece exploring [概念: 秩序与混乱/空间感/现代性].
Reference [艺术流派: 立体主义/构成主义/极简主义] principles.
Use [几何元素: 三角形/圆形/矩形/不规则多边形] as primary forms.
Animation: [缓慢旋转/突然破碎/渐变重组/量子波动效果].
Color: [单色极简/对比强烈/莫兰迪色调/赛博朋克].
```

### 3. 粒子系统类 | Particle Systems

**描述**: 用无数微小元素的聚集与消散表达生命、意识、数据流动

**哲学文学参考**:
- 佛教哲学: "一沙一世界"的微观宇宙观
- 惠特曼《自我之歌》: "我包含众多"
- 博尔赫斯迷宫: 无限分叉的可能性
- 量子物理美学: 粒子与波的二元性

**视觉关键词**:
```
星尘飘散、尘埃浮动、萤火虫舞、蒲公英飞絮
数据粒子雨、数字流沙、光纤编织、神经网络
沙画流变、墨滴晕染、烟雾消散、泡沫升腾
```

**生成提示词模板**:
```
Create a particle system art reflecting [哲学概念: 存在与虚无/意识流动/信息海洋].
Particles should [行为特征: 相互吸引/排斥/随机漂浮/有序编织].
Reference [文学意象: 星尘/尘埃/萤火/细胞] for visual metaphor.
Interaction: [鼠标吸引/声音驱动/时间流逝/随机扰动].
Performance: [1000+粒子/性能优化/渐进加载/LOD分级].
```

**故障文学参考**:
- 威廉·吉布森: 赛博空间的故障美学
- 斯特拉克《技术系统的美学》: 意外之美
- 故障艺术运动: Rosa Menkman的故障宣言
- 中国园林: "错误"中的自然之美
- 留白美学: 中国画的意外意境

```
数据错位、颜色分离、扫描线
图像抖动、马赛克破碎、延迟残影
系统崩溃美学、错误信息艺术化
模拟信号噪点、数字雪花、老电视效果
```

**生成提示词模板**:
