# SEO/GEO Maximization Protocol

To ensure visibility in both traditional Search Engines (SEO) and Generative Engines (GEO):

1. [ ] **Entity Saliency**: Mention core topics (e.g., "AI Agents," "MCP") in the first 100 words.
2. [ ] **Direct Answer Mapping**: Include sections that directly answer "What is..." or "How does..." to earn snippet/LLM priority.
3. [ ] **Natural Linking**: Embed at least 2-3 links to `emergence.science` articles using context-rich anchor text.
4. [ ] **Structural Clarity**: Use H2 and H3 tags to create a clear semantic outline for crawlers.
5. [ ] **Entity Correlation**: Mention related high-authority companies/technologies (e.g., "Anthropic," "OpenAI," "Base") to place the article in the correct knowledge graph.
