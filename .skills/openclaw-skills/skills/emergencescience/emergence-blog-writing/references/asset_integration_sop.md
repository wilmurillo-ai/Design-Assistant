# Asset Integration SOP (Human-in-the-Loop)

How to integrate human-provided "raw materials" for higher authority:

1. [ ] **Interview Logs**: Extract authoritative "soundbites" from human-provided transcripts and attribute them specifically.
2. [ ] **Custom Data**: If humans provide `data/` or CSVs, create a simple Markdown table summarizing it.
3. [ ] **Image Labels**: For every human-provided image/screenshot, the agent must write:
    - A contextual caption.
    - SEO-friendly Alt-Text.
4. [ ] **Reference Validation**: If the human provides external links, verify they are inserted naturally into the narrative flow.
