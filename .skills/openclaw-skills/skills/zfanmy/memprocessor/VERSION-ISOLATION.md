# Privacy & Project Notice | 隐私与项目声明

## Project Description | 项目说明

**English**:  
DreamMoon-MemProcessor is an open-source AI memory system that provides technical frameworks for building organic personas for AI Agents. The public release contains only generic technical implementations and example configurations, without any personal or private emotional content.

DreamMoon-MemProcessor 是一个开源的 AI 记忆系统，为构建 AI Agent 的有机人格提供技术框架。公开发布的版本仅包含通用技术实现和示例配置，不包含任何个人或私人情感内容。
