# Feishu Parallel Dispatch

Keep Feishu chats responsive by spawning one background child task per substantial prompt.

## Highlights

- foreground acknowledgement stays short
- long work moves into a spawned child task
- multiple prompts can run independently
- pairs well with progress heartbeat and task status skills
