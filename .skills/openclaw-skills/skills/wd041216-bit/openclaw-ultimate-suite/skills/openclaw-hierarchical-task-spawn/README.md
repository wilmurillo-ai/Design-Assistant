# OpenClaw Hierarchical Task Spawn

Split substantial OpenClaw work into a shallow task tree instead of one oversized child task.

## What It Adds

- simple/medium/complex task classification
- medium tasks split into fine leaf tasks
- complex tasks split into medium workstreams first
- progressive reporting without waiting for every branch to finish

## Good Fits

- research + report + table
- PM plan + deliverable generation
- multi-deliverable Feishu tasks
