# 🏢 办公效率示例 - 会议准备

> 使用晴晴自动激活 + 日程管理 + 文档创建

---

## 🎯 场景

你需要准备下周的会议：
- 查看日程安排
- 创建会议笔记模板
- 润色发言稿
- 创建准备任务清单
- 发送会议邀请

---

## 🚀 晴晴自动激活流程

### 步骤 1：一句话启动

```bash
# 直接说需求
"帮我准备下周的会议"

# 晴晴自动识别：办公效率场景
# 自动激活技能组合:
# 1. calendar (日程管理)
# 2. note (笔记管理)
# 3. writing-assistant (写作辅助)
# 4. todolist (待办管理)
# 5. office (文档处理)
```

---

### 步骤 2：日程检查 (自动)

```
calendar 自动检查:
├─ 下周会议列表 (时间、主题、参会人)
├─ 会议冲突检测 (时间重叠提醒)
├─ 会议室预订 (地点、设备)
└─ 日历同步 (飞书/Google/Outlook)

检查范围：下周一到周五
```

---

### 步骤 3：笔记模板创建 (自动)

```
note 自动创建模板:
├─ 会议主题 (清晰明确)
├─ 议程列表 (时间分配)
├─ 参会人名单 (角色、职责)
├─ 讨论要点 (核心问题)
├─ 决策记录 (结论、行动项)
└─ 后续跟进 (负责人、截止日期)

模板格式：Markdown (可导出 Word/PDF)
```

---

### 步骤 4：发言稿润色 (自动)

```
writing-assistant 自动润色:
├─ 语法检查 (错别字、病句)
├─ 逻辑优化 (结构清晰)
├─ 语气调整 (正式/简洁/亲和)
├─ 关键词强调 (核心信息)
└─ 时长控制 (5-10 分钟)

发言稿类型：开场白/主题演讲/总结
```

---

### 步骤 5：任务清单创建 (自动)

```
todolist 自动创建:
├─ 会议材料准备 (PPT、文档)
├─ 设备测试 (投影、音响)
├─ 参会人确认 (邮件/消息)
├─ 会议室布置 (桌椅、茶水)
└─ 会议记录安排 (拍照、录音)

优先级：高/中/低 + 截止日期
```

---

### 步骤 6：会议邀请发送 (自动)

```
office + feishu-im 自动发送:
├─ 邀请邮件 (时间、地点、议程)
├─ 飞书消息 (提醒、确认)
├─ 日历邀请 (自动添加到日历)
└─ 参会确认 (RSVP 追踪)

发送时间：提前 3-5 天
```

---

## 📋 完整命令示例

### 自动激活 (推荐)

```bash
# 一句话启动全流程
"帮我准备下周的产品评审会议，主题是 Q2 产品规划，参会人包括产品、技术、设计团队"

# 晴晴自动执行:
# 1. 识别：办公效率场景
# 2. 激活：calendar + note + writing-assistant + todolist
# 3. 检查：日程安排 (时间、地点、参会人)
# 4. 创建：笔记模板 (议程、讨论要点)
# 5. 润色：发言稿 (开场白、主题演讲)
# 6. 管理：任务清单 (材料、设备、确认)
# 7. 发送：会议邀请 (邮件、飞书)
```

### 手动调用 (可选)

```bash
# 日程检查
/calendar "查看下周的会议安排"

# 笔记模板
/note "创建产品评审会议笔记模板，包括议程、讨论要点、决策记录"

# 发言稿润色
/writing-assistant "润色产品评审发言稿，主题 Q2 规划，时长 10 分钟，正式风格"

# 任务管理
/todolist "创建会议准备任务清单，包括材料、设备、参会人确认"

# 文档处理
/office "创建会议邀请邮件，包括时间、地点、议程"
```
