# 📱 社交媒体运营示例 - 小红书发布

> 使用晴晴自动激活 + 内容创作 + 排期管理

---

## 🎯 场景

你想发布小红书笔记：
- 产品测评笔记
- 内容优化
- 排期管理
- 竞品数据采集
- 营销策略制定

---

## 🚀 晴晴自动激活流程

### 步骤 1：一句话启动

```bash
# 直接说需求
"我想发布一篇小红书笔记"

# 晴晴自动识别：社交媒体场景
# 自动激活技能组合:
# 1. xiaohongshu-mcp (小红书自动化)
# 2. social-media-scheduler (排期管理)
# 3. content-creator (内容创作)
# 4. tiktok-crawling (竞品数据采集)
# 5. marketing (营销策略)
```

---

### 步骤 2：内容创作 (自动)

```
content-creator 自动创作:
├─ 标题优化 (吸引点击，包含关键词)
├─ 正文结构 (痛点 → 解决方案 → 产品)
├─ 标签选择 (热门 + 精准)
├─ 配图建议 (3-5 张，高清)
└─ 互动引导 (点赞、收藏、评论)

内容风格：专业/幽默/温馨 (可指定)
```

---

### 步骤 3：竞品数据采集 (自动)

```
tiktok-crawling + multi-search-engine 采集:
├─ 竞品笔记：标题、内容、标签
├─ 热门笔记：点赞、收藏、评论
├─ 用户反馈：评论区的痛点、需求
└─ 趋势分析：热门话题、标签

采集范围：最近 7 天、同类产品 Top 20
```

---

### 步骤 4：内容优化 (自动)

```
基于竞品数据，content-creator 优化:
├─ 标题 A/B 测试 (3 个版本)
├─ 正文优化 (痛点更精准)
├─ 标签优化 (热门 + 长尾)
└─ 配图优化 (建议具体场景)
```

---

### 步骤 5：排期管理 (自动)

```
social-media-scheduler 自动排期:
├─ 最佳发布时间 (根据受众活跃时间)
├─ 发布频率 (每周 3-5 篇)
├─ 内容日历 (主题规划)
└─ 互动提醒 (回复评论、私信)

平台：小红书 (可扩展抖音、微博)
```

---

### 步骤 6：发布执行 (自动)

```
xiaohongshu-mcp 自动发布:
├─ 登录账号 (Cookie/OAuth)
├─ 上传内容 (标题、正文、图片)
├─ 添加标签 (自动填充)
├─ 发布确认 (人工审核)
└─ 数据追踪 (点赞、收藏、评论)
```

---

## 📋 完整命令示例

### 自动激活 (推荐)

```bash
# 一句话启动全流程
"我想发布一篇小红书笔记，产品是智能水杯，目标用户是白领，风格专业，预算 1000 元推广"

# 晴晴自动执行:
# 1. 识别：社交媒体场景
# 2. 激活：xiaohongshu-mcp + content-creator + scheduler
# 3. 创作：内容优化 (标题 + 正文 + 标签)
# 4. 采集：竞品数据 (Top 20 笔记)
# 5. 排期：最佳发布时间
# 6. 发布：自动上传 + 人工确认
```

### 手动调用 (可选)

```bash
# 内容创作
/content-creator "创作智能水杯小红书笔记，目标白领，专业风格"

# 竞品采集
/tiktok-crawling "采集智能水杯竞品笔记，最近 7 天 Top 20"

# 排期管理
/social-media-scheduler "规划下周小红书发布排期，每周 3 篇"

# 发布执行
/xiaohongshu-mcp "发布笔记：标题 + 正文 + 图片"

# 营销策略
/marketing "制定智能水杯小红书推广策略，预算 1000 元"
```
