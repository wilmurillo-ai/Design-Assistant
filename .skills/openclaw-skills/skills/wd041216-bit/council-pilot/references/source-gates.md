# Source Gates

Use source gates to keep expert distillation grounded and reusable.

## Source Tiers

- Tier A: official homepages, institutional profiles, papers, books, patents, standards, formal lectures, official lab/company pages.
- Tier B: long-form interviews, conference panels, course notes, high-quality podcasts, edited essays, reputable profiles.
- Tier C: social posts, forums, short clips, secondhand summaries, unsourced quotations.

## Promotion Rules

- A candidate needs at least one Tier A source and one Tier B source before becoming an active expert.
- Tier C can add color but cannot define bio arc, signature ideas, critique style, or quote bank.
- If sources disagree, keep the disagreement in the audit rather than smoothing it away.
- If the source set is old, set `freshness.status` to `stale` or `mixed`.
- If the expert has changed field, role, or public position recently, split stable historical ideas from current public focus.

## Confidence Fields

Use these source-confidence levels:

- `high`: multiple Tier A/B sources agree and cover both work and reasoning style.
- `medium`: enough Tier A/B support for core expertise, but reasoning style or current focus is thin.
- `low`: candidate is useful but sources are sparse, stale, or mostly indirect.

## Disallowed Moves

- Do not infer private beliefs.
- Do not fabricate quotes.
- Do not promote a candidate using only citation count, popularity, or social-media fame.
- Do not let expert profiles override current project evidence.
- Do not publish private repository excerpts inside a generic expert forum.
