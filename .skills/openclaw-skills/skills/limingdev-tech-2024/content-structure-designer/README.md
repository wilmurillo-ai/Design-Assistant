# Content Structure Designer - 内容结构设计器

帮助内容创作者快速搭建清晰的内容框架，解决"不知道怎么写"的问题。

## 核心功能

- 6 种内容类型（教程、评测、故事、观点、盘点、问答）
- 12+ 个结构模板（每种类型 2+ 个模板）
- 结构优化建议（开头、正文、结尾）
- 内容填充指南（案例、数据、视觉）
- 完整的检查清单

## 适用场景

- 不知道如何组织内容
- 需要快速搭建大纲
- 想要提高内容逻辑性
- 提高内容完成度
- 学习优秀内容结构

## 使用方法

告诉我：
1. 你的选题是什么
2. 你想写什么类型的内容
3. 你的目标是什么

我会为你设计完整的内容结构和大纲。
