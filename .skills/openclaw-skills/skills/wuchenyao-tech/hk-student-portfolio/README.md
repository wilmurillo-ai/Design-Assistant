# 中学生升学简历生成器 — 使用指南

## 快速开始

### 方式一：对话式（推荐）

直接告诉我学生信息，我来帮你生成简历：

```
小虎，帮我做一份升学简历：
- 姓名：张三
- 性别：男
- 毕业学校：XX市第一中学
- 年级排名：第8名/680人
- 获奖情况：...
- 特长：...
```

### 方式二：JSON 文件式

1. 填写 `templates/student_data_template.json`
2. 运行脚本生成

## 三种模板风格

| 模板 | 主色调 | 适合学生 |
|------|--------|----------|
| 学霸型 | 深蓝 | 成绩突出、竞赛获奖多 |
| 特长型 | 紫色 | 艺术/体育/科技特长明显 |
| 综合型 | 绿色 | 各方面均衡发展 |

## 简历模块

1. **基本信息** — 姓名、性别、出生日期、毕业学校、联系方式
2. **学业成绩** — 排名、各科成绩、成长趋势
3. **竞赛获奖** — 竞赛名称、级别、获奖等级、年份
4. **特长爱好** — 项目、水平/等级、证书
5. **社会实践** — 志愿服务、社团活动、夏令营
6. **荣誉称号** — 三好学生、优秀干部等
7. **自我评价** — 个人陈述、学习感悟
8. **教师推荐** — 班主任评语

## 输出格式

- Word (.docx) — 可编辑，方便修改
- PDF — 终版，正式提交

## 注意事项

- 简历建议控制在 **1-2页** 以内
- 照片需要手动插入（Word中替换"照片位置"文字）
- 信息务必真实，结尾有真实性声明
