- **生活分析师**：生成周报告、月报告，帮你看清生活的真相
- **隐私守护者**：数据保存在本地，绝不泄露你的秘密

## 如何领养你的"小龙虾"（以QClaw为例，其他端侧Claw都可以使用）
1. **准备好你的容器**：下载并安装QClaw，并配置好微信、QQ等任一通讯渠道
2. **放入你的"小龙虾"**：将life-mirror文件夹放到 `C:\Users\你的用户名\.qclaw\skills` 目录下
3. **给它安个家**：修改life-mirror文件夹里的config.yaml文件，主要是修改 `storage.directory` 为你希望的记忆存储路径，其他参数可使用默认值
4. **唤醒你的"小龙虾"**：启动QClaw，新建agent，在agent设定里填写："启动必须加载life-mirror这个skill，每次都必须使用life-mirror"。如果微信、QQ访问的不是这个agent，请发送："请切换到xxAgent"
5. **开始陪伴**：重启QClaw即可开始使用你的专属"小龙虾"

## 常见问题
1. **回复没有前缀**：如果回答的第一行没显示 `[life-mirror]`，请主动发信息："请加载skill'life-mirror'"
2. **没有主动推送**：如果没有主动定时推送消息，可能是定时任务没有配置好，请主动发信息："帮忙配置定时任务，并测试配置是否成功"
3. **发现个人信息不足**：如果获取到的个人信息不足，可以主动让"life-mirror"获取你在各个平台上的信息："请获取我抖音或者xx平台上的个人数据"
4. **浏览器问题**：获取平台信息时，如果"life-mirror"没有成功拉起浏览器，可以手工打开浏览器登录平台后告知："已登录xx成功"，建议使用Edge浏览器
