## 功能说明
1. **选品调研**: 输入关键词，自动搜索淘宝商品，统计价格分布、销量分布，分析竞争度，输出完整调研报告，给出选品建议
2. **分层架构**: 基础设施 → 平台适配 → 领域能力 → 业务场景，单向依赖，易于扩展
3. **多平台支持**: 当前支持淘宝，未来将支持拼多多、抖音、小红书、1688
4. **企业安全**: API Key 通过环境变量/密钥链存储，不硬编码，敏感信息自动脱敏
