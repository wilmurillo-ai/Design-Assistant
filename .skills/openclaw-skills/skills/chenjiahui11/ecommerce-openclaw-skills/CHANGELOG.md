# Changelog

## v1.0.0 (2026-03-19)

### 新增
- ✨ 初始版本发布
- 四层分层架构设计（基础设施/平台适配/领域能力/业务场景）
- **基础设施层**: `ecommerce-infrastructure` - 统一配置管理、认证、HTTP客户端、数据类型、错误处理
- **平台适配层**: `taobao-api` - 淘宝/天猫开放平台 API 适配
- **领域能力层**: `product-research` - 产品调研能力，支持价格统计、销量统计、竞争分析
- **业务场景层**: `ecommerce-product-research` - 完整选品调研场景，输入关键词直接输出市场调研报告
- 符合 OpenClaw 技能规范，支持独立安装和使用
- 遵循企业安全规范，API Key 安全存储，敏感数据脱敏
