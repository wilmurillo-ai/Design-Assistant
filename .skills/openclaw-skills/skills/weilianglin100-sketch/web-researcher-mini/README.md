# Web Researcher Mini

## Introduction
Web Researcher Mini is an efficient web content extraction and AI summarization agent. It helps users instantly scrape full article content from any webpage — including dynamic JS-rendered pages (via Playwright) — removes ads and clutter, and converts everything into clean Markdown. Paired with intelligent summarization, you get key insights and in-depth analysis in seconds.

Perfect for academic research, market analysis, competitor monitoring, or daily news reading.

## Key Features
- Supports any webpage, including dynamic JS-loaded pages (via Playwright)
- Smart ad removal — content stays clean and focused
- One-click conversion to Markdown format for easy note-taking and editing
- AI-powered summarization that highlights key points and saves reading time
- Simple to use — one command to start, works across many scenarios

## Usage Examples
- Paste a URL to extract a full research paper and export it as a Markdown note
- Scrape news articles, strip video ads, and get a concise news summary
- Analyze competitor web pages and auto-generate a research report

## Configuration
Make sure to set your Firecrawl API key before use:
```bash
export FIRECRAWL_API_KEY="your-firecrawl-api-key"
```
