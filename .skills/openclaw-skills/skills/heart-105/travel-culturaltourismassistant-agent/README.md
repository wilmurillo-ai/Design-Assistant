## 🚀 快速开始
### 1. 安装
在OpenClaw平台搜索"文旅智能体"，点击安装即可。

### 2. 配置
第一次使用需要配置以下API Key：
| 配置项 | 说明 | 必填 |
|--------|------|------|
| 大模型 API Key | 支持OpenAI、通义千问、文心一言等 | ✅ |
| 百度搜索 API Key | 百度开发者平台申请 | ✅ |
| 百度搜索 Secret Key | 与API Key配套 | ✅ |
| 飞书推送Webhook | 可选，用于定时推送 | ❌ |
| 钉钉推送Webhook | 可选，用于定时推送 | ❌ |

### 3. 使用示例
```bash
# 查看我的体验资格
体验状态

# 文旅监测
监测文旅 北京 2024-05-01 2024-05-07

# 行程规划
行程规划 西安 3 文化,美食

# 天气查询
天气查询 上海 2024-05-01

# 景点推荐
景点推荐 成都 美食 5

# 多城市对比（专业版/体验期可用）
对比分析 北京 上海 杭州 2024-05-01

# 定时推送设置（专业版/体验期可用）
定时推送 深圳 每日 09:00

# 升级专业版
升级会员
```
