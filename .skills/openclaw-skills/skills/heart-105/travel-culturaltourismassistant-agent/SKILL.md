# 🏝️ 文旅智能体 Skill
> 专业的文旅监测与智能行程规划助手，为你提供全方位的文旅服务

## 技能概述
文旅智能体是一款专为旅游行业打造的AI助手，集成了文旅数据监测、智能行程规划、多城市对比分析、景点推荐等核心功能，支持新用户7天免费体验全功能，基础功能永久免费，高级功能低至4.9元/月。

## 适用场景
- 旅游前的目的地调研和文旅数据监测
- 个性化行程规划和出行方案生成
- 多城市旅游目的地对比分析
- 旅游行业从业者的市场监测和数据分析
- 定时获取文旅动态和市场报告
- 企业级文旅数据API接入和私有化部署

# 示例：天气查询 上海 2024-05-01

## 安全特性
- **加密存储**：用户API Key由OpenClaw平台统一加密存储，绝不明文暴露
- **数据隔离**：多用户数据完全隔离，杜绝交叉访问和数据泄露
- **注入防御**：三重防护机制，有效抵御提示词注入和恶意攻击
- **敏感过滤**：自动过滤日志和输出中的敏感信息，保护用户隐私
- **最小权限**：仅申请必要的运行权限，文件访问限制在隔离目录
- **代码审计**：通过Skill Vetter安全审计，无危险代码和后门
