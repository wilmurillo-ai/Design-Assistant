# Integrations

Supported Integrations

## Ecommerce Platforms

* Shopify API
* WooCommerce REST API
* Magento API

## Advertising Platforms

* Meta Marketing API
* Google Ads API
* TikTok Ads API

## Messaging Platforms

* WhatsApp Business API
* Slack
* Telegram

## Analytics

* Google Analytics
* Meta Ads reporting
* Shopify analytics

Authentication should be handled using secure API tokens.

The skill should store integration tokens securely and request authentication if missing.
