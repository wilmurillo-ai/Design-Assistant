## 6-Frame Story Template (Copy-Paste Ready)

Replace [ANIMAL], [OUTFIT], [GOAL], [OBSTACLE], [TWIST]:

```
Frame 1 (HOOK):
[ANIMAL] wearing [OUTFIT], standing confidently with arms crossed, 
determination in eyes, [SETTING backdrop], dramatic lighting, 
photorealistic, 9:16 vertical

Frame 2 (SETUP):
Same [ANIMAL] in [OUTFIT], [FIRST ACTION toward GOAL], 
environment [SETTING DETAIL], photorealistic, 9:16 vertical

Frame 3 (CONFLICT):
Same [ANIMAL] in [OUTFIT], encountering [OBSTACLE], 
overwhelmed/surprised expression, photorealistic, 9:16 vertical

Frame 4 (PEAK):
Same [ANIMAL] in [OUTFIT], [STRUGGLE with OBSTACLE], 
sweating heavily, [DANGER ELEMENT visible in background], 
photorealistic, high tension, 9:16 vertical

Frame 5 (TWIST):
Same [ANIMAL], [UNEXPECTED OUTCOME - opposite of goal], 
[EMOTION: defeated but slightly proud / shocked / laughing], 
photorealistic, 9:16 vertical

Frame 6 (CTA):
Same [ANIMAL], giving thumbs up / waving at camera, 
speech bubble "[FUNNY QUOTE]", bright colors, photorealistic, 9:16 vertical
```

## Style Modifiers by Content Type

**Comedy/Cute:** `soft warm lighting, pastel background, rounded shapes, big expressive eyes`

**Epic/Dramatic:** `cinematic lighting, dark background, dramatic shadows, heroic pose`

**Realistic:** `natural photography, real environment, documentary style, candid moment`

**Cartoon-ish:** `3D render, Pixar style, vibrant colors, exaggerated expressions`
