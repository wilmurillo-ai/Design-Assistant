# TikTok Channel Setup Guide

## Account Setup
- Username: use character name (e.g., @hammy.adventures, @fathamsterofficial)
- Profile photo: AI-generated portrait of your character (consistent with video style)
- Bio:
  Line 1: Character name + emoji
  Line 2: "New video daily 🎬"
  Line 3: CTA link (Linktree or direct product)
- Category: Entertainment

## Profile Photo Prompt
```
[CHARACTER] portrait, face close-up, friendly expression,
clean white/gradient background, photorealistic,
square format 1:1, high resolution
```

## First 3 Videos Strategy
- Video 1: Character introduction ("Meet [Character]!")
- Video 2: Character's first mission (simple, charming)
- Video 3: First failure (establishes the "always fails but tries again" pattern)

Post all 3 on Day 1 — TikTok gives new accounts a "new account boost"

## Consistency Rules
- Same posting time every day (trains algorithm + audience)
- Same intro music or sound (brand recognition)
- Same text font/color in CapCut (visual brand)
- Same character in EVERY video (no one-offs)

## TikTok New Account Boost
New accounts get extra distribution for first 7 days.
Post best content in first week — this sets your baseline performance.
Post minimum 14 videos in first 7 days (2/day).

## Analytics to Track (Weekly)
| Metric | Goal Week 1 | Goal Week 4 |
|---|---|---|
| Average watch time | >50% | >65% |
| 3-second view rate | >40% | >55% |
| Profile visits/video | >2% | >5% |
| Follower conversion | >1% | >3% |

## When to Pivot
If after 30 videos (2 weeks), average views < 500:
1. Change hook style (keep same character)
2. Change music (trending sound vs original)
3. Change video length (try shorter: 10-15s)
4. Keep character — don't change the character
