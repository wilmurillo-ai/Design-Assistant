# Character Bible — TikTok Channel Identity

## Why Character Consistency Matters

Viewers follow CHARACTERS, not videos.
- Consistent character = viewers recognize instantly in FYP
- Character arc = reason to keep watching (what happens next?)
- Personality = emotional connection = shares + comments + follows

The most successful AI comedy channels (like the cat/lion video) have ONE recurring character viewers root for.

---

## Character Creation Framework

Before making any video, define your character ONCE and stick to it.

### Core Identity Sheet

```
CHARACTER NAME: [e.g., "Hammy" the hamster]
SPECIES/APPEARANCE:
  - Base animal: [hamster / cat / penguin / bear / etc.]
  - Signature color: [orange fur / grey / black and white]
  - Signature outfit: [always wears X — e.g., red bow tie, tiny hat]
  - Size: [always tiny relative to world]
  - Face: [round eyes, chubby cheeks — specific features]

PERSONALITY:
  - Core trait: [overconfident / timid-but-brave / endlessly optimistic]
  - Fatal flaw: [always overestimates abilities / too trusting / too greedy]
  - Quirk: [always has a "plan" that fails / never learns from mistakes]
  - Emotion range: [determined → panicked → defeated → immediately tries again]

MOTIVATION:
  - What does character always WANT? [respect / food / adventure / love]
  - What always GETS IN THE WAY? [own incompetence / bad luck / big world]

CATCHPHRASE (optional):
  - Text that appears when character is confident: [e.g., "Kế hoạch hoàn hảo 😤"]
  - Text that appears when character fails: [e.g., "Lần sau chắc được 😤"]
```

## The Cute Factor — Why It Matters

**Core principle:** Cuteness = instant emotional investment before the story even starts.

The cuter the character, the faster viewers bond → the more they feel tension, laugh at failures, and want to see what happens next.

### Cuteness Multipliers (stack these for maximum effect)
| Factor | Effect | Example |
|---|---|---|
| **Chubby/round body** | Triggers caregiving instinct | Fat Orange Cat, chubby hamster |
| **Big round eyes** | Infant schema = automatic empathy | Most anime-style animals |
| **Small vs big world** | Vulnerability = audience wants to protect | Tiny hamster in giant bank |
| **Familiar species** | Pre-existing emotional bond | Cats (internet mascots), dogs |
| **Funny outfit** | Contrast = cute + absurd | Cherry-print shorts, ski mask |
| **Determined expression** | Cute + ambitious = rooting for them | Hamster with plan |

### Why Cats Work Best
- Cats = the internet's mascot — billions already have emotional bond
- Cats have **independent personality** (unlike dogs) → believable doing "bad" things
- "Cat does something bold/naughty" = totally in-character → no suspension of disbelief needed
- **Chubby orange cat** specifically = most viral cat type on TikTok (search "fat orange cat")
- Cat can be both **victim AND villain** in the same video

### Cute Character Do's and Don'ts
✅ Keep character chubby/round at all times (never slim or realistic-proportioned)
✅ Big expressive eyes — show all emotions through eyes
✅ Signature cute accessory (bow, tiny hat, funny shorts)
✅ Small relative to environment — always emphasize size difference
✅ Let character show vulnerability even when being bold (nervous sweat = cute)

❌ Never make cute character look aggressive or scary
❌ Never remove signature outfit (breaks character recognition)
❌ Never make character too competent (underdog status = cuteness amplifier)

## Proven Character Archetypes

### Archetype 1: The Overconfident Underdog ⭐ (Highest Engagement)
- Small animal, big ambitions
- Always has a "perfect plan"
- Plan always fails spectacularly
- Immediately plans next attempt (never gives up)
- **Why it works**: Viewers root for them + laugh at failures = perfect loop
- **Example**: Hammy the hamster trying to rob a bank

**Visual consistency rules:**
- Always same proportions (tiny vs giant world)
- Always same signature item (ski mask, backpack, bow tie)
- Same "determined" face for planning, same "panicked" face for failure

### Archetype 2: The Dignified Aristocrat
- Elegant animal (lion, cat, peacock) in refined setting
- Tries to maintain dignity in absurd situations
- World keeps ruining their composure
- **Why it works**: Contrast between elegance and chaos = comedy gold
- **Example**: Lion in silk robe getting pranked by tiny cat

### Archetype 3: The Innocent Explorer
- Wide-eyed animal discovering human world for first time
- Everything is magical and confusing
- Misunderstands everything in endearing ways
- **Why it works**: Wholesome + curious = share to friends/family
- **Example**: Penguin visiting a city for the first time

### Archetype 4: The Unlikely Professional
- Animal in a job/role completely wrong for their species
- Takes job very seriously, world doesn't cooperate
- **Why it works**: Absurdity + relatability (we've all felt incompetent at work)
- **Example**: Tiny hamster as CEO of giant corporation

## Story Arc Framework (For Series)

Single videos get views. **Series** get followers.

### The 3-Act Mini-Series (3 videos)
```
Video 1 — THE PLAN
  Hook: "Hammy có kế hoạch mới 👀"
  Content: Character announces ambitious goal, prepares confidently
  End: "Ngày mai bắt đầu... 😤"
  CTA: "Theo dõi để xem kết quả 👇"

Video 2 — THE ATTEMPT
  Hook: "Ngày trọng đại của Hammy 🏆"
  Content: Execution of plan, things start going wrong
  End: Cliffhanger — disaster about to happen
  CTA: "Part 3 ngày mai 😱"

Video 3 — THE RESULT
  Hook: "Chuyện gì xảy ra với Hammy... 😭"
  Content: Spectacular failure OR surprise success
  End: Character immediately planning next attempt
  CTA: "Hammy sẽ thử gì tiếp theo? Comment nhé 👇"
```

### The Running Gag (Infinite Series)
Same setup, different scenario each video:
- "Hammy tries to [X]" — new X every video
- Viewers anticipate the failure = rewatch + share
- Create 20+ videos, channel identity = "the Hammy channel"

## Visual Consistency Checklist

Before generating each AI image, verify:

- [ ] Same animal species as established
- [ ] Same color/markings (describe exactly in prompt)
- [ ] Same signature outfit/accessory
- [ ] Same relative size to environment
- [ ] Same art style (photorealistic / cartoon / etc.)
- [ ] Same aspect ratio (9:16 vertical)

**Consistency prompt prefix** (add to every frame):
```
"[CHARACTER NAME], a [SPECIES] with [EXACT DESCRIPTION: color, size, signature item], 
in a photorealistic style consistent with previous frames..."
```

**Example:**
```
"Hammy, a tiny chubby orange hamster with round black eyes wearing a tiny black ski mask 
and miniature black backpack, photorealistic style, in a [SCENE]..."
```

## Emotional Journey Per Video

Every video must take viewers through this arc:

```
😐 Neutral scroll
↓
😮 Hook catches attention (curiosity/surprise)
↓  
😄 Amusement builds (setup payoff)
↓
😂 Peak laugh moment (twist/failure)
↓
🥺 Empathy for character (they tried so hard)
↓
❤️ Affection (want to see character again → FOLLOW)
```

If your video doesn't hit ALL 6 emotional beats, revise before posting.

## Quality Checklist Before Every Post

### Script Quality
- [ ] Hook is under 10 words, creates immediate curiosity
- [ ] Conflict is relatable (viewer has felt this way)
- [ ] Twist is unexpected but makes sense in hindsight
- [ ] CTA asks a specific question (not just "comment below")
- [ ] Emotional arc: curiosity → amusement → surprise → affection

### Visual Quality
- [ ] Character looks exactly the same as previous videos
- [ ] Text overlay: max 6 words, readable font, contrasting color
- [ ] Each frame tells a clear moment of the story
- [ ] 9:16 vertical format
- [ ] No watermarks on AI images

### Audio Quality
- [ ] Music matches emotional tone of each scene
- [ ] Sound effect at twist/surprise moment
- [ ] Volume: music not louder than any speech/text reading time
- [ ] Trending sound used (if applicable)

### Platform Optimization
- [ ] Video length: 15-30 seconds
- [ ] Caption: character name + situation + 1 question + hashtags
- [ ] Hashtags: 3-5 max (not #fyp)
- [ ] Posted at optimal time (7-9 AM / 12-1 PM / 8-10 PM VN time)
- [ ] First comment ready to post immediately after upload

### Engagement Setup
- [ ] First comment written: something funny/relatable to pin
- [ ] Reply template ready for common responses
- [ ] "Part 2?" teased if video allows continuation
