# Script Templates for TikTok Content Factory

## Template A: AI Comedy Story (Core Format)
Best for: AI image series, animal characters, absurd humor

```
HOOK (0-3s):
Visual: [Most dramatic/funny frame from the story]
Text: "[Character] vs [Challenge] 👀"
Sound: [Dramatic music sting]

SETUP (3-8s):
Visual: Character confidently preparing
Text: "Kế hoạch hoàn hảo... 😤"
Sound: [Sneaky/planning music]

CONFLICT (8-18s):
Visual: First obstacle encountered
Text: "Bước [N]: [Action]..."
Sound: [Music continues, slight tension]

PEAK (18-25s):
Visual: Things going wrong
Text: "Ủa... 👁️👁️"
Sound: [Record scratch or dramatic chipmunk]

TWIST (25-30s):
Visual: Unexpected outcome
Text: "[Funny consequence] 😭"
Sound: [Sad trombone or fail sound]

CTA (30-35s):
Visual: Character thumbs up / wave at camera
Text: "Tag [type of person] 👇"
Sound: [Upbeat outro]
```

## Text Overlay Best Practices

### Formatting Rules
- **Max 6 words** per text frame
- **Font size**: Large enough to read without zooming
- **Color**: White text + black stroke OR black text + white stroke
- **Position**: Top 1/3 or bottom 1/3 (avoid center — algorithm crops for thumbnails)
- **Duration**: Each text shown minimum 2 seconds

### High-Engagement Text Patterns
```
Questions: "Chuyện gì xảy ra tiếp theo? 👀"
Cliffhangers: "Chờ đến cuối video..."
Relatability: "Ai cũng từng như này 😭"
Surprise setup: "Không ai ngờ được..."
Identity tags: "POV: Bạn là [character]"
```

### Emoji Usage
- Use emoji to **replace words** (saves space, more visual)
- ✅ "Kế hoạch hoàn hảo 😤" 
- ❌ "Kế hoạch hoàn hảo của tôi thật sự rất tốt"
- Max 2 emoji per text frame

## Sound Strategy

### Music Selection by Scene
| Scene | Music Type | CapCut Search |
|---|---|---|
| Hook | Attention grab | "dramatic sting" "record scratch" |
| Setup | Sneaky/Planning | "sneaky music" "spy theme" |
| Conflict | Building tension | "suspense buildup" |
| Peak | Chaos | "dramatic chipmunk" "fails music" |
| Twist | Surprise | "sad trombone" "vine boom" |
| CTA | Upbeat/Fun | "happy ending" "success music" |

### Trending Sounds
Check TikTok's "Trending" sounds weekly and create 1-2 videos using current top sounds.
Trending sounds get 3x more reach on the For You page.
