# 股市晨報 固定模板格式

## ⚠️ 執行前必讀
每次生成晨報前，必須先閱讀本模板，不得主觀臆斷任何格式。

---

## ⚠️ 大標題日期原則（2026-04-19 確認）
**大標題(x.xx) = 報告生成日期（自然日），不等於交易日期**

示例：
- 4月19日（周日）生成報告，交易日期是4月18日（周五）→ 大標題：`(4.19)`，第一部分：`📍一、上週五股巿回顧`
- 4月21日（周一）生成報告，交易日期是4月20日（周日）→ 大標題：`(4.21)`，第一部分：`📍一、昨日股巿回顧`

---

```
其中：凈買入xxx；

凈賣出xxx。

📍三、熱點資訊

▶️ xxx

xxx

▶️ xxx

xxx

▶️ xxx

📍四、熱門港股

▶️ [公司名稱+代碼，例如：小米集團(01810.HK)]

xxx

資料來源: xxx

「只供內部參考，不可外傳」
```

---

## 固定規則

1. 大標題含 🔴 和 🔵
2. 📍一、X股巿回顧：X = 昨日 / 週一~週五 / 上週一~上週五
3. 📍二：南下資金總數 + 箭頭方向（📈=凈買入，📉=凈賣出）由實際數據決定
4. 數字後帶（港元）：**只有小標題第一行**帶「（港元）」，📈📉兩行及摘要**不重複**标注
5. ▶️ 標題和摘要之間**必須空一行**
6. 第四部分只寫一檔股票
7. 📍後無空格，直接接字
8. 不確定的數據寫「🚨待更新」，嚴禁捏造數字

---

## 日期判斷邏輯

| 條件 | 格式 |
|------|------|
| 上一個交易日是昨天，且今天和昨天之間無中斷 | 📍一、昨日股巿回顧 |
| 上一個交易日是本週，但中間有中斷（假期） | 📍一、週X股巿回顧 |
| 上一個交易日是上週 | 📍一、上週X股巿回顧 |

---

## 常見錯誤（必須避免）

| 錯誤 | 正確 |
|------|------|
| ▶️ 標題和摘要直接連在一起 | ▶️ 標題**必須空一行**摘要 |
| 📈📉兩行後面帶（港元）| 只小標題第一行帶（港元）|
| 📍一、 後有空格 | 📍一、直接接字 |
| 數字估算或捏造 | 寫「🚨待更新」 |
| 第四部分寫多檔股票 | 只寫一檔 |
| 大標題隨便寫 | 大標題=第三部分三個▶️標題分號分隔 |
