# 股市晨報 常見錯誤對照

## 錯誤9（最高發-必須強制記憶）：▶️ 標題和摘要之間必須空一行

- **錯誤**：`▶️ 標題\n摘要`（直接連在一起）
- **正確**：`▶️ 標題\n\n摘要`（**空一行**再寫摘要）
- **高發原因**：2026-04-10、04-11、04-12 連續三次都漏
- **⚠️ 強制約束**：任何情況下不得省略，即使趕時間也必須嚴格空行

---

## 錯誤1（原則性）：數據編造

- **錯誤**：搜索結果沒有依據，仍寫了具體結論
- **正確**：搜到什麼寫什麼，搜不到寫「🚨待更新」，嚴禁捏造

## 錯誤2：大標題用錯了

- **錯誤**：標題隨便寫
- **正確**：標題 = 第三部分「熱點資訊」的三條新聞標題，分號分隔

## 錯誤3：📍 header多了空格

- **錯誤**：`📍 一、`（有空格）
- **正確**：`📍一、`（直接接字）

## 錯誤4：南下資金箭頭方向寫反

- **正確**：📈 = 凈買入，📉 = 凈賣出

## 錯誤5：個股資金流向格式多了括號

- **錯誤**：`藥明生物(02269) 6.88億`
- **正確**：`藥明生物6.88億`（無括號）

## 錯誤6：統計數字多寫「港元」

- **錯誤**：`共計逾30億港元`
- **正確**：`共計逾30億`（第二部分只有小標題帶「（港元）」）

## 錯誤7：▶️ 標題加了粗體

- **正確**：`▶️ 美伊停火談判...`（無粗體無括號）

## 錯誤8：第四部分數量過多

- **正確**：第四部分只寫**一檔**股票

## 錯誤9：▶️ 標題和摘要之間必須空一行

（同上，最高發錯誤）

---

## 錯誤10（2026-04-20 新增）：南下資金📈📉行不確定時不得刪除，應標「待更新」

- **錯誤**：港股通(滬)/港股通(深)細分數字不確定時，直接刪除📈📉兩行
- **正確**：保留📈📉兩行，直接填入「待更新」
  ```
  📍二、南下資金凈買入170.10億（港元）
  📈 港股通(滬): 待更新
  📈 港股通(深): 待更新
  ```
- **原因**：總數字（170.10億）和細分數字（滬/深）是不同口徑；總數字確定時仍需保留格式，用戶自行填入細分

## 核查清單（發送前必讀）

1. [ ] 大標題 = 第三部分三個▶️標題分號分隔
2. [ ] 📈📉箭頭方向正確（📈=凈買入，📉=凈賣出）
3. [ ] 第二部分只有小標題帶（港元），📈📉兩行不帶
4. [ ] ▶️標題和摘要之間有空行
5. [ ] 📍後無空格
6. [ ] 第四部分只有一檔股票
7. [ ] 所有數字有搜尋依據，無法確認的寫「🚨待更新」
8. [ ] 📈📉細分數字不確定時，保留行並寫「待更新」，不得刪除
