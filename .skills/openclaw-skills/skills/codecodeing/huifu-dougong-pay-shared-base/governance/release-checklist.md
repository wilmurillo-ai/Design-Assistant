# 发布检查清单

发版前建议逐项检查。

## 结构检查

- 新增的文档是否已经放到正确目录
- 共享资料是否被 skill 正确引用
- 新增 skill 是否已经进入 README

## 内容检查

- 正文里的运行时说明是否和 metadata 一致
- 环境变量声明是否和正文一致
- 是否出现真实密钥、真实签名、真实下载地址
- Java 特有内容是否误放到协议层

## 兼容检查

- 现有 10 个 skill 的 slug 是否没变
- 旧链接是否还能打开
- README 路径是否都能跳转

## 发布说明检查

- 文档版本号是否更新
- 修订说明是否更新
- 官方 SDK 版本是否更新
- 新增能力是否写入版本说明
