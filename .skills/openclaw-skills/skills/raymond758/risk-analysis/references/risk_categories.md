# 风险分类标准与预警等级说明

## 一、风险类型分类

### 1. 跌倒风险（fall）
**识别场景**：
- 人员突然倒地且长时间未起身
- 身体倾斜角度超过阈值
- 运动轨迹突然变化

**风险等级判定**：
- 高风险（≥0.9）：明确倒地且无动作
- 中风险（0.7-0.9）：疑似跌倒，姿态异常
- 低风险（<0.7）：短暂弯腰或坐下

### 2. 健康风险（health）
#### 2.1 心梗前兆
- 表情痛苦，手捂胸口
- 出汗过多，面色苍白
- 突然站立不稳

#### 2.2 脑梗前兆
- 口角歪斜，流口水
- 单侧肢体无力
- 言语不清，表情僵硬

#### 2.3 突发疾病
- 突然倒地抽搐
- 长时间静止不动
- 异常痛苦表情

### 3. 异常行为（behavior）
- **剧烈运动**：快速奔跑、打斗等
- **长时间静止**：同一姿势保持超过30分钟
- **区域闯入**：进入禁止区域
- **物品遗留**：异常物品出现在监测区域
- **人员聚集**：超过5人聚集超过10分钟

## 二、预警等级标准

| 等级 | 颜色 | 置信度范围 | 处理策略 |
|------|------|-----------|---------|
| 高风险 | 红色 | ≥0.9 | 立即触发多渠道报警，通知相关人员 |
| 中风险 | 黄色 | 0.7-0.9 | 记录事件，自动抓拍图片，通知管理员关注 |
| 低风险 | 蓝色 | 0.5-0.7 | 仅记录日志，不主动通知 |

## 三、预警内容模板
```
🚨 高风险事件预警
⏰ 时间: 2024-01-01 12:00:00
⚠️ 类型: 跌倒风险
📊 置信度: 0.95
📝 描述: 检测到人员倒地，已超过10秒未起身
💡 建议: 立即查看现场情况，必要时拨打急救电话
📍 位置: 客厅摄像头
```

## 四、处理建议库

### 跌倒事件
1. 立即查看现场，确认人员意识状态
2. 如有受伤，不要随意移动，拨打120
3. 检查周边环境，消除安全隐患
4. 必要时联系家属

### 心梗/脑梗前兆
1. 立即拨打120急救电话
2. 让患者保持平躺，不要移动
3. 如有病史，服用对应急救药物
4. 密切观察生命体征

### 异常闯入
1. 查看监控确认人员身份
2. 必要时启动安防报警
3. 通知安保人员前往处理
4. 保存监控录像作为证据
