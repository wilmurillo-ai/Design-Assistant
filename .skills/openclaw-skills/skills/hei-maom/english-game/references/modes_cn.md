# 模式设计

## 1. vocab

定位：Wordle 风格 + 英语学习收尾。

推荐规则：
- 5 到 11 个字母优先
- 最多 6 到 9 次机会
- 单词必须是规范英文词
- 词库难度按 cet4 / cet6 / kaoyan / ielts / toefl 控制

提示层级：
1. 中文释义
2. 首字母
3. 词根词缀
4. 同义替换

## 2. guess

定位：会话式猜词。

第一版优先只做 definition：
- 给英文释义
- 用户直接发单词
- 猜中就结束

后续可扩展：
- example
- synonym
- antonym
- cloze

## 3. speaking

定位：轻量英语口语互动。

第一版推荐三类任务：
- repeat：复述一句话
- answer：回答一个简短问题
- polish：用户自由说一句，机器人帮润色

注意：
- 识别结果可用于表达反馈
- 不要把它包装成精细发音测评
- 重点放在表达是否清楚、是否自然、有没有更好的说法
