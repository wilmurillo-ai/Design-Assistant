# 时间解析规则（中文）

## 支持的表达

- `今天 5 点前`
- `今天下午三点`
- `今晚 8 点`
- `明天上午`
- `后天中午`
- `周五前`
- `下周一 10 点`
- `3月20日 15:00`
- `2026-03-20 15:00`

## 默认补全

### 只有日期，没有时间

默认补全为：`18:00`

例：
- `周五前交` → 默认截止时间 `周五 18:00`

### 只有时段，没有具体小时

- `上午` → `10:00`
- `中午` → `12:00`
- `下午` → `15:00`
- `晚上 / 今晚` → `20:00`
- `凌晨` → `01:00`

### 只有纯时间

- 若今天该时刻还没过，用今天
- 若今天该时刻已过，自动顺延到明天

## 提醒时间

不是固定 30 分钟，而是根据时间远近、重要性关键词和是否显式要求“提前多久提醒”自动判断

例如：

- `今天 17:00 前给我` → 可能提醒在 `16:20` 或 `15:30`，取决于任务紧急程度
- `下周一开会别忘了` → 可能更早预留数小时甚至前一天提醒

## 不建立提醒的典型情况

- 没有时间
- 没有待办/提醒语义
- 解析出的提醒时间已经早于当前时间
