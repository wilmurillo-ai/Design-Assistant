# 数据出境检查清单

**适用场景**: OpenClaw 服务器位于中国境外，或服务中国用户

## 快速判断

### 是否需要申报数据出境安全评估？

**满足任一条件即需申报**:

```
□ 向境外提供重要数据
□ 关键信息基础设施运营者向境外提供个人信息
□ 处理 100 万人以上个人信息的数据处理者向境外提供个人信息
□ 自上年 1 月 1 日起累计向境外提供 10 万人个人信息
□ 自上年 1 月 1 日起累计向境外提供 1 万人敏感个人信息
```

**OpenClaw 常见场景判断**:

| 场景 | 是否需要申报 | 说明 |
|------|--------------|------|
| 个人使用，服务器在海外 | ⚠️ 需评估 | 如服务中国用户，可能触发 |
| 企业部署，服务器在国内 | ✅ 无需申报 | 数据未出境 |
| 使用境外 API（如 OpenAI） | ⚠️ 需评估 | 对话数据可能传输到境外 |
| 仅本地处理，无网络传输 | ✅ 无需申报 | 数据未出境 |

## 数据分类

### 重要数据识别

**OpenClaw 可能涉及的重要数据**:

| 数据类型 | 是否重要数据 | 判断依据 |
|----------|--------------|----------|
| 用户对话记录 | 视情况 | 如含敏感信息则属于 |
| API 密钥和配置 | 是 | 影响系统安全 |
| 用户身份信息 | 是 | 个人信息保护范畴 |
| 系统日志 | 否 | 一般数据 |
| 公开配置 | 否 | 非敏感 |

### 个人信息分类

| 级别 | 示例 | 出境要求 |
|------|------|----------|
| 一般个人信息 | 用户名、邮箱 | 标准合同/认证 |
| 敏感个人信息 | 生物识别、金融账户、行踪轨迹 | 单独同意 + 安全评估 |
| 未成年人信息 | 14 岁以下儿童信息 | 监护人同意 + 严格保护 |

## 合规路径选择

### 路径一：安全评估（强制情形）

**适用**: 满足上述申报条件之一

**流程**:
```
1. 自评估 → 2. 提交材料 → 3. 省级网信办初审 → 4. 国家网信办评估 → 5. 结果通知
```

**时间**: 45 个工作日（复杂情况可延长 15 个工作日）

**材料清单**:
- [ ] 申报书
- [ ] 数据出境风险自评估报告
- [ ] 与境外接收方拟订立的法律文件
- [ ] 其他相关材料

### 路径二：标准合同（推荐个人/小企业）

**适用**: 不满足强制评估条件

**流程**:
```
1. 签署标准合同 → 2. 备案（省级网信办）→ 3. 生效
```

**时间**: 合同生效后 10 个工作日内备案

**模板下载**: http://www.cac.gov.cn/2023-02/24/c_1678396071696263.htm

### 路径三：保护认证

**适用**: 跨国公司、集团内部数据传输

**认证机构**: 中国网络安全审查技术与认证中心（CCRC）

**时间**: 约 3-6 个月

## OpenClaw 数据出境风险评估

### 数据流向分析

```bash
# 检查配置的 API 端点
grep -r "api\." ~/.openclaw/config.json 2>/dev/null

# 检查网络请求目标
# 常见境外服务：
# - api.openai.com (美国)
# - api.anthropic.com (美国)
# - api.google.com (美国)
# - dashscope.aliyuncs.com (中国境内 ✅)
```

### 风险等级评估

| 配置 | 风险等级 | 建议 |
|------|----------|------|
| 仅使用境内 API（如通义千问） | 🟢 低 | 数据不出境 |
| 使用境外 API + 境内服务器 | 🟡 中 | 需评估 + 标准合同 |
| 使用境外 API + 境外服务器 | 🔴 高 | 强制安全评估 |
| 本地部署 + 无外部 API | 🟢 低 | 数据不出境 |

## 检查清单

### 部署前

- [ ] 确认服务器位置（境内/境外）
- [ ] 识别所有数据出境场景
- [ ] 评估是否触发安全评估申报
- [ ] 选择合适的合规路径（评估/合同/认证）
- [ ] 准备相关法律文件

### 部署中

- [ ] 启用数据传输加密（HTTPS/TLS）
- [ ] 配置数据加密存储
- [ ] 设置访问控制和审计日志
- [ ] 实施数据最小化原则

### 部署后

- [ ] 定期（每年）重新评估
- [ ] 监控数据出境活动
- [ ] 及时更新法律文件
- [ ] 保存评估记录（至少 3 年）

## 违规处罚

| 违法行为 | 处罚 | 法律依据 |
|----------|------|----------|
| 未经评估向境外提供数据 | 责令改正 + 警告；情节严重的处 100-500 万罚款，可责令暂停业务 | 数据安全法 46 条 |
| 未按要求备案标准合同 | 责令改正 + 警告 | 个人信息保护法 66 条 |
| 向境外提供重要数据 | 责令改正 + 警告；情节严重的处 500-2000 万罚款，可吊销执照 | 数据安全法 46 条 |

## 实用工具

### 服务器位置检测

```bash
# 检测当前服务器位置
curl -s ipinfo.io

# 检测 API 端点位置
curl -s -o /dev/null -w "%{remote_ip}" https://api.openai.com
curl -s -o /dev/null -w "%{remote_ip}" https://dashscope.aliyuncs.com
```

### 数据流向监控

```bash
# 监控出站连接
sudo tcpdump -i any -n port 443 | grep -E "ESTABLISHED"

# 分析网络连接
netstat -antp | grep ESTABLISHED | awk '{print $5}' | cut -d: -f1 | sort | uniq -c
```
