# 中国法规合规检查清单

**适用场景**: 在中国境内运营或服务中国用户的 OpenClaw 部署

## 核心法规

### 1. 《网络安全法》（2017）

**关键要求**:
- [ ] 网络运营者需履行网络安全保护义务
- [ ] 采取数据分类、重要数据备份和加密措施
- [ ] 记录并留存相关网络日志（不少于 6 个月）
- [ ] 制定网络安全事件应急预案

**OpenClaw 检查项**:
```bash
# 检查日志留存
ls -la ~/.openclaw/logs/
# 确认日志保留策略（建议配置轮转，保留 180 天+）

# 检查加密措施
grep -r "https\|ssl\|tls" ~/.openclaw/config.json
```

### 2. 《数据安全法》（2021）

**关键要求**:
- [ ] 建立数据分类分级管理制度
- [ ] 重要数据需明确保护责任部门
- [ ] 数据出境需进行安全评估
- [ ] 定期开展数据安全风险评估

**数据分类参考**:
| 级别 | 示例 | 保护要求 |
|------|------|----------|
| 一般数据 | 公开信息、非敏感配置 | 基础访问控制 |
| 重要数据 | 用户信息、API 密钥 | 加密存储 + 访问审计 |
| 核心数据 | 生物识别、金融信息 | 境内存储 + 专项审批 |

### 3. 《个人信息保护法》（2021）

**关键要求**:
- [ ] 处理个人信息需取得用户同意
- [ ] 明示处理目的、方式和范围
- [ ] 提供个人信息查询、更正、删除渠道
- [ ] 个人信息出境需通过安全评估或认证

**OpenClaw 隐私政策必备条款**:
```markdown
1. 收集的信息类型（如：对话记录、配置数据）
2. 收集目的（如：提供 AI 助手服务）
3. 存储位置和期限（如：本地存储，删除即销毁）
4. 共享情况（如：不与第三方共享）
5. 用户权利（如：查询、导出、删除数据）
6. 联系方式（如：隐私问题联系邮箱）
```

### 4. 《数据出境安全评估办法》（2022）

**触发条件**（满足任一即需申报）:
- [ ] 数据处理者向境外提供重要数据
- [ ] 关键信息基础设施运营者向境外提供个人信息
- [ ] 处理 100 万人以上个人信息的数据处理者向境外提供个人信息
- [ ] 自上年 1 月 1 日起累计向境外提供 10 万人个人信息或 1 万人敏感个人信息

**OpenClaw 评估要点**:
```bash
# 检查服务器位置
curl -s ipinfo.io/country

# 检查是否使用境外服务
grep -r "api.openai.com\|api.anthropic.com\|api.google.com" ~/.openclaw/
```

**申报流程**:
1. 通过所在地省级网信办提交申报材料
2. 材料包括：数据出境风险自评估报告、相关法律文件等
3. 评估周期：45 个工作日（复杂情况可延长）

## 合规检查清单

### 基础合规（必须完成）

- [ ] **隐私政策**: 创建 `PRIVACY.md` 文档，包含必备条款
- [ ] **用户同意**: 首次使用时明确告知数据收集范围
- [ ] **数据加密**: 敏感数据（API 密钥、用户信息）加密存储
- [ ] **日志留存**: 配置日志轮转，保留不少于 6 个月
- [ ] **访问控制**: 启用身份验证，限制未授权访问

### 数据出境评估（如适用）

- [ ] **服务器位置确认**: 确认 OpenClaw 服务器所在国家/地区
- [ ] **数据流向分析**: 识别所有数据传输到境外的场景
- [ ] **影响评估**: 评估是否触发安全评估申报条件
- [ ] **申报准备**: 如需申报，准备相关材料

### 安全保护措施

- [ ] **防火墙**: 启用并配置防火墙规则
- [ ] **入侵检测**: 部署基础入侵检测系统（如 fail2ban）
- [ ] **备份策略**: 定期备份配置和数据
- [ ] **应急响应**: 制定安全事件应急预案

## 违规处罚参考

| 违法行为 | 处罚 | 法律依据 |
|----------|------|----------|
| 未履行网络安全保护义务 | 责令改正 + 警告；拒不改正处 1-10 万罚款 | 网络安全法 59 条 |
| 非法向境外提供数据 | 责令改正 + 警告；情节严重的处 100-500 万罚款 | 数据安全法 46 条 |
| 违法处理个人信息 | 责令改正 + 没收违法所得；情节严重的处 5000 万以下或营收 5% 罚款 | 个人信息保护法 66 条 |

## 合规文档模板

### 隐私政策模板 (`PRIVACY.md`)

```markdown
# OpenClaw 隐私政策

## 1. 收集的信息

我们收集以下信息以提供服务：
- 对话记录（用于 AI 上下文理解）
- 配置数据（用于个性化设置）
- 使用日志（用于故障排查）

## 2. 收集目的

- 提供 AI 助手服务
- 改善用户体验
- 保障系统安全

## 3. 存储和保护

- 所有数据存储于 [中国境内/具体位置]
- 敏感数据采用 AES-256 加密
- 访问权限严格控制

## 4. 数据共享

- 不与任何第三方共享用户数据
- 法律法规要求的除外

## 5. 用户权利

您有权：
- 查询个人数据
- 更正错误信息
- 删除个人数据
- 导出个人数据

## 6. 联系我们

隐私问题请联系：[邮箱地址]
```
