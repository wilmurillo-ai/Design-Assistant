# Changelog

## v1.0.0 (2026-03-02)

### ✨ 新特性

- 🎯 **完整的微信公众号文章抓取功能**：支持所有常见格式的微信文章
- 📝 **多格式输出**：HTML、Markdown、纯文本三种格式任选
- 🧹 **智能内容清洗**：自动去除广告、赞赏、阅读数等无关内容
- 🖼️ **图片提取**：自动提取文章中的所有图片URL
- 📊 **数据分析**：自动计算字数和预计阅读时间
- 🔄 **重试机制**：网络波动时自动重试
- 🛡️ **异常处理**：完善的错误处理和日志系统

### 🚀 性能优化

- ⚡ **高性能抓取**：直接HTTP请求HTML，无需浏览器渲染
- 📦 **轻量级**：仅依赖3个核心Python库
- 🎯 **精准解析**：支持多种HTML结构的文章
- 🚄 **快速响应**：平均处理时间<10秒
