# Dialectic Overlay

## Use when
- open-ended ideation
- strategic reframing
- two strong positions may yield a higher-order synthesis

## Do not use when
- a simple decision fork is enough
- the task is mainly verification
- the problem is still too vague to frame

## Pair with
- synthesis-memo
- decision-memo
- minority-report

## Expected artifact
- synthesis memo or explicit deadlock report
