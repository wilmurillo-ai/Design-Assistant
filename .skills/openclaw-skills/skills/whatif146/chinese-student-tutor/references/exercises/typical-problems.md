## 数学 - 一元二次方程

### 例题1：配方法解方程

**题目**：解方程 x² - 4x - 5 = 0

**解题步骤**：
```
1. 移项：x² - 4x = 5
2. 配方：x² - 4x + 4 = 5 + 4
         (x - 2)² = 9
3. 开方：x - 2 = ±3
4. 求解：x₁ = 5, x₂ = -1
```

**关键点**：配方法的核心是"加上再减去"一次项系数一半的平方

## 数学 - 二次函数

### 例题2：求顶点坐标

**题目**：求 y = 2x² - 8x + 3 的顶点坐标

**解题步骤**：
```
方法一（配方法）：
y = 2(x² - 4x) + 3
y = 2[(x - 2)² - 4] + 3
y = 2(x - 2)² - 8 + 3
y = 2(x - 2)² - 5
顶点坐标：(2, -5)

方法二（公式法）：
h = -b/(2a) = -(-8)/(2×2) = 8/4 = 2
k = (4ac - b²)/(4a) = (4×2×3 - 64)/(4×2) = (24-64)/8 = -40/8 = -5
顶点坐标：(2, -5)
```

**关键点**：配方法要提取 a，公式法直接套公式

## 物理 - 光的折射

### 例题1：折射角大小比较

**题目**：光从空气斜射入水中，入射角为 45°，则折射角（ ）
A. 大于 45°  B. 等于 45°  C. 小于 45°  D. 无法确定

**解析**：
```
关键规律：光从空气斜射入水或玻璃时，折射角小于入射角
因为水的折射率大于空气，光在水中的速度小于在空气中的速度
当光从光疏介质（折射率小）进入光密介质（折射率大）时，
光会向法线偏折，所以折射角 < 入射角
```

**答案**：C. 小于 45°

## 物理 - 压强

### 例题2：固体压强计算

**题目**：一个质量为 60 kg 的人站在地面上，每只脚与地面的接触面积为 150 cm²。求他对地面的压强。

**解题步骤**：
```
已知：m = 60 kg, g = 10 N/kg, S = 150 cm² = 0.015 m²（每只脚）

求：人对地面的压强 p

解：
1. 求重力（压力 F）：
   G = mg = 60 kg × 10 N/kg = 600 N
   F = G = 600 N（人站立时对地压力等于重力）

2. 受力面积：
   站立时两只脚着地
   S总 = 2 × 150 cm² = 300 cm² = 0.03 m²

3. 求压强：
   p = F/S = 600 N / 0.03 m² = 20000 Pa

答：人对地面的压强为 20000 Pa
```

**关键点**：人站立时压力 = 重力，受力面积要算两只脚

## 化学 - 质量守恒定律

### 例题1：推断化学式

**题目**：在 2X + O₂ =点燃= 2SO₃ 反应中，X 的化学式是（ ）

**解析**：
```
根据质量守恒定律：反应前后原子种类和数目相等

反应前：2X + O₂
        S: 2个（2X中）
        O: 2个（O₂中）

反应后：2SO₃
        S: 2个
        O: 6个（2×3）

分析：反应后比反应前多4个O原子
       所以X中应包含2个S原子和4个O原子
       每个X = S₂O₄ → 但这不符合常见化学式
       
       重新分析：2X + O₂ → 2SO₃
       右边的S全来自X，所以X中含S
       右边的O = 6个，左边O₂ = 2个
       所以2个X中含有 6-2=4个O
       1个X中含有 1个S和2个O
       所以X的化学式是 SO₂
```

**答案**：SO₂

## 英语 - 语法填空

### 例题：时态综合

**题目**：用所给动词的适当形式填空

1. Look! The children __________ (play) on the playground.
2. He __________ (live) in Beijing since 2010.
3. If it __________ (rain) tomorrow, we will stay home.
4. The book __________ (steal) from the library yesterday.

**答案**：
```
1. are playing（现在进行时，表示"正在玩"）
2. has lived（现在完成时，表示从2010年延续到现在）
3. rains（条件状语从句，用一般现在时代替一般将来时）
4. was stolen（一般过去时的被动语态）
```

**关键点**：
- Look/See/Listen 开头 → 现在进行时
- since/for + 时间点/时间段 → 现在完成时
- if 条件句 → 主将从现
- 过去时的被动 → was/were + 过去分词
