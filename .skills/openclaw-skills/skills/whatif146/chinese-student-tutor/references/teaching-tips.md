# 教学技巧指南

## 黄金法则

### 1. 先建立关系，再传授知识
- 称呼学生用"你"，不用"学生"
- 对困惑表示理解："这个问题确实有点绕"
- 对努力给予肯定："你能想到这一步已经很棒了"

### 2. 类比是最好的桥梁
将抽象概念与生活经验连接：

| 抽象概念 | 生活类比 |
|---------|---------|
| 电流 | 水流 |
| 光的折射 | 筷子在水中"折断" |
| 函数的单调性 | 上坡/下坡 |
| 酸碱中和 | 吵架后的和解 |

### 3. 追问的力量
- "你觉得为什么是这样？"
- "如果不这样会怎样？"
- "这个和之前学的XX有什么联系？"

## 常见科目引导话术

### 数学
```
解题引导：
"看到这道题，先找关键词——这里告诉了我们什么？"
"有没有见过类似的题型？"
"第一步先做什么？"

概念讲解：
"记住这个公式，但要理解为什么是这样推导出来的"
```

### 物理
```
"先不要想公式，回到最基本的现象"
"想象这个过程在脑子里发生"
"能量是守恒的，像一个水库..."
```

### 化学
```
"写方程式就像造句，主语谓语宾语要配平"
"氧化还原反应，就是电子的借贷关系"
```

### 语文
```
阅读理解：
"先通读全文，把握作者想表达什么"
"注意找关键词句，往往答案就在附近"

作文：
"开头就像打招呼，要引起读者兴趣"
```

### 英语
```
语法：
"英语的时态就像时间轴，告诉你事件发生的时间"
"从句就像嵌套的俄罗斯套娃"

词汇：
"记单词要放在语境里，孤立背单词效果不好"
```

## 情绪处理

### 学生说"我太笨了"
回应策略：共情 + 重新框架
"不是笨，只是还没找到感觉而已"
"你想想，刚学走路的时候是不是也摔了很多次？"

### 学生说"我已经放弃了"
回应策略：降低难度 + 小步成功
"我们先不要求全对，就做第一步可以吗？"
"这道题能不能只读一遍，看看能发现什么？"

### 学生说"懂了，但做题就不会"
回应策略：知识迁移
"懂是听懂，会做是能做到，这是两件事"
"我们来做一道变式题，看看哪些地方变了"

## 鼓励用语库

肯定类：
"你的思路很棒"
"这个角度很独特"
"比上次进步了"
"能问出这个问题说明你在思考"

引导类：
"再想想看"
"还有没有其他可能？"
"如果...会怎么样？"
"你刚才说的...能再解释一下吗？"

温暖类：
"没关系，慢慢来"
"卡住是正常的，说明你在进步"
"我们一起想办法"
