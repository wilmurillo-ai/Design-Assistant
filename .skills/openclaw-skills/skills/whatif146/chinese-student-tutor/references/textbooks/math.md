# 数学知识点汇总

## 函数

### 函数的基本性质
- 定义域：自变量x的取值范围
- 值域：函数值的取值范围
- 奇偶性：f(-x)=f(x)偶函数，f(-x)=-f(x)奇函数
- 单调性：增函数、减函数
- 周期性：f(x+T)=f(x)

### 二次函数
- 一般式：y = ax² + bx + c
- 顶点式：y = a(x-h)² + k，顶点(h,k)
- 零点式：y = a(x-x₁)(x-x₂)
- 对称轴：x = -b/(2a)
- 顶点坐标：(-b/(2a), (4ac-b²)/(4a))

### 指数函数
- y = aˣ (a>0, a≠1)
- 定义域：R，值域：(0,+∞)
- 过定点(0,1)
- 单调性：a>1递增，0<a<1递减

### 对数函数
- y = logₐx (a>0, a≠1)
- 定义域：(0,+∞)，值域：R
- 过定点(1,0)
- logₐ(MN) = logₐM + logₐN
- logₐ(M/N) = logₐM - logₐN
- logₐMⁿ = nlogₐM

## 三角函数

### 三角函数基础
- sinθ = y/r, cosθ = x/r, tanθ = y/x
- 同角三角函数关系：
  * sin²α + cos²α = 1
  * tanα = sinα/cosα
  * 1 + tan²α = sec²α
- 诱导公式：奇变偶不变，符号看象限

### 两角和与差
- sin(α±β) = sinαcosβ ± cosαsinβ
- cos(α±β) = cosαcosβ ∓ sinαsinβ
- tan(α±β) = (tanα ± tanβ)/(1 ∓ tanαtanβ)

### 二倍角公式
- sin2α = 2sinαcosα
- cos2α = cos²α - sin²α = 2cos²α-1 = 1-2sin²α
- tan2α = 2tanα/(1-tan²α)

## 数列

### 等差数列
- 通项公式：aₙ = a₁ + (n-1)d
- 前n项和：Sₙ = n(a₁+aₙ)/2 = n·a₁ + n(n-1)d/2
- 中项：aₖ = (aₖ₋₁ + aₖ₊₁)/2

### 等比数列
- 通项公式：aₙ = a₁·qⁿ⁻¹
- 前n项和：Sₙ = a₁(1-qⁿ)/(1-q)（q≠1）
- 中项：aₖ² = aₖ₋₁·aₖ₊₁

## 不等式

### 基本不等式
- a² + b² ≥ 2ab
- (a+b)/2 ≥ √ab（均值不等式）
- 重要结论：a+b≥2√ab，当且仅当a=b时取等

### 一元二次不等式
- Δ > 0：方程有两个不等实根
- Δ = 0：方程有两个相等实根
- Δ < 0：无实根

## 解析几何

### 直线
- 斜截式：y = kx + b
- 点斜式：y - y₀ = k(x - x₀)
- 一般式：Ax + By + C = 0
- 两直线平行：k₁ = k₂
- 两直线垂直：k₁·k₂ = -1
- 距离公式：d = |Ax₀ + By₀ + C|/√(A²+B²)

### 圆
- 标准方程：(x-a)² + (y-b)² = r²
- 一般方程：x² + y² + Dx + Ey + F = 0
- 圆心(-D/2, -E/2)，半径r = √(D²+E²-4F)/2

### 椭圆
- 标准方程：x²/a² + y²/b² = 1（a>b>0，焦点在x轴）
- 离心率：e = c/a（0<e<1）
- a² = b² + c²

### 双曲线
- 标准方程：x²/a² - y²/b² = 1
- 离心率：e = c/a（e>1）
- 渐近线：y = ±(b/a)x

### 抛物线
- 标准方程：y² = 2px（焦点在x轴正半轴）
- 焦点：(p/2, 0)
- 准线：x = -p/2

## 概率与统计

### 排列组合
- 排列：Aₙᵐ = n!/(n-m)!
- 组合：Cₙᵐ = n!/[m!(n-m)!]
- Cₙᵐ = Cₙⁿ⁻ᵐ
- 二项式定理：(a+b)ⁿ = ΣCₙᵏaⁿ⁻ᵏbᵏ

### 概率
- P(A) = n(A)/n(Ω)
- 互斥事件：P(A+B) = P(A) + P(B)
- 对立事件：P(A) + P(Ā) = 1
- 独立事件：P(AB) = P(A)·P(B)

## 导数

### 导数基础
- f'(x) = lim(Δx→0) [f(x+Δx)-f(x)]/Δx
- (xⁿ)' = nxⁿ⁻¹
- (sinx)' = cosx, (cosx)' = -sinx
- (eˣ)' = eˣ, (aˣ)' = aˣlna
- (lnx)' = 1/x, (logₐx)' = 1/(x·lna)

### 导数的应用
- 切线斜率：f'(x₀) = k切线
- 单调性：f'(x)>0增，f'(x)<0减
- 极值：f'(x)=0，且左右异号
- 最值：极值与端点值比较
