# 物理知识点汇总

## 力学

### 运动的描述
- 质点：理想化模型，有质量但无大小形状
- 参考系：描述物体运动时选作标准的物体
- 位移：初位置到末位置的有向线段（矢量）
- 速度：位移与时间的比值，v = Δx/Δt
- 加速度：速度变化量与时间的比值，a = Δv/Δt

### 匀变速直线运动
- 速度公式：v = v₀ + at
- 位移公式：x = v₀t + ½at²
- 速度-位移公式：v² - v₀² = 2ax
- 自由落体：g = 9.8m/s²（取10m/s²）

### 相互作用
- 重力：G = mg，方向竖直向下
- 弹力：F = kx，胡克定律
- 摩擦力：f = μN（滑动摩擦）
- 力的合成与分解：平行四边形法则

### 牛顿运动定律
- 牛顿第一定律：惯性定律，一切物体保持匀速或静止
- 牛顿第二定律：F = ma，加速度与合外力成正比
- 牛顿第三定律：作用力与反作用力，大小相等方向相反
- 超重与失重：加速向上超重，加速向下载重

### 曲线运动
- 曲线运动条件：合外力方向与速度方向不共线
- 平抛运动：水平匀速+竖直自由落体
- 圆周运动：v = ωr = 2πr/T

### 万有引力
- 万有引力定律：F = GMm/r²
- 万有引力常量：G = 6.67×10⁻¹¹ N·m²/kg²
- 三种宇宙速度：
  * 第一宇宙速度：7.9km/s（环绕速度）
  * 第二宇宙速度：11.2km/s（脱离速度）
  * 第三宇宙速度：16.7km/s（逃逸速度）

### 机械能
- 功：W = Fs cosθ
- 功率：P = W/t = Fv
- 动能：Ek = ½mv²
- 重力势能：Ep = mgh
- 机械能守恒定律：只有重力（或弹力）做功时，机械能守恒

### 动量
- 动量：p = mv（矢量）
- 动量定理：I = Δp = FΔt
- 动量守恒：系统合外力为零时，总动量不变

## 热学

### 分子动理论
- 分子大小：10⁻¹⁰ m数量级
- 阿伏加德罗常数：NA = 6.02×10²³ mol⁻¹
- 分子热运动：温度越高，热运动越剧烈
- 分子间作用力：引力和斥力同时存在

### 气体实验定律
- 等温变化：PV = 常数（玻意耳定律）
- 等容变化：P/T = 常数（查理定律）
- 等压变化：V/T = 常数（盖·吕萨克定律）
- 理想气体状态方程：PV = nRT

## 电学

### 电场
- 库仑定律：F = kQ₁Q₂/r²
- 电场强度：E = F/q = kQ/r²
- 电势能：Ep = qφ
- 电势差：U = W/q = Ed

### 电路
- 欧姆定律：I = U/R
- 电阻定律：R = ρL/S
- 串并联电路特点
- 电功率：P = UI = I²R = U²/R

### 磁场
- 磁感应强度：B = F/(IL)
- 左手定则：判断通电导线在磁场中的受力
- 右手定则：判断感应电流方向
- 洛伦兹力：F = qvB（v⊥B时）

### 电磁感应
- 楞次定律：感应电流的磁场阻碍原磁场变化
- 法拉第电磁感应定律：E = nΔΦ/Δt
- 感生电动势：E = BLv（平动切割）

## 光学

### 光的折射
- 折射定律：n₁sinθ₁ = n₂sinθ₂
- 折射率：n = c/v = sinθ₁/sinθ₂
- 全反射：sinθc = 1/n
- 光疏介质：折射率较小的介质
- 光密介质：折射率较大的介质

### 光的干涉与衍射
- 干涉条件：频率相同、相位差恒定
- 明条纹条件：Δx = kλ（k=0,±1,±2...）
- 暗条纹条件：Δx = (2k+1)λ/2
- 衍射：光绕过障碍物偏离直线传播的现象

## 原子物理

### 原子结构
- 汤姆孙模型：枣糕模型
-卢瑟福模型：核式结构（行星模型）
- 玻尔模型：能量量子化，能级跃迁
- 能级跃迁：hν = E初 - E终

### 原子核
- 原子核组成：质子和中子（统称核子）
- 核力：强相互作用，短程力
- 质能方程：E = mc²
- 结合能：核子结合成原子核时释放的能量
