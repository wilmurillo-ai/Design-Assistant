# 化学知识点汇总

## 第一部分：物质及其变化

### 考点1 物质的组成、性质和分类
- 不同的单质，可以形成不同的同素异形体
- 碳的同素异形体：金刚石、石墨、富勒烯（C60）、碳纳米管、石墨烯
- 物质的分类：
  * 纯净物：单质、化合物
  * 混合物：均匀混合物（溶液、合金）、不均匀混合物（浊液、悬浊液、乳浊液）

### 考点2 离子反应
- 电解质：在水溶液中或熔融状态下能导电的化合物
- 非电解质：在水溶液中和熔融状态下都不能导电的化合物
- 离子反应发生的条件：生成沉淀、气体、弱电解质
- 离子方程式书写步骤：写-拆-删-查

### 考点3 离子共存
- 离子间不能大量共存的条件：
  1. 生成难溶物或微溶物
  2. 生成气体或挥发性物质
  3. 生成弱电解质
  4. 发生氧化还原反应
  5. 发生双水解反应
  6. 生成配位化合物

### 考点4 氧化还原反应
- 氧化剂：化合价降低（得到电子），具有氧化性
- 还原剂：化合价升高（失去电子），具有还原性
- 氧化还原反应的本质：电子的转移
- 常见氧化剂：KMnO4、K2Cr2O7、HNO3、H2SO4(浓)、Cl2、O2
- 常见还原剂：Fe、Zn、Al、CO、C、H2

### 考点5 氧化还原反应的规律
- 守恒规律：得失电子总数相等
- 强弱规律：氧化性：氧化剂 > 氧化产物
- 先后规律：多种氧化剂存在时，还原性强的先被氧化
- 价态规律：最高价只有氧化性，最低价只有还原性，中间价既有氧化性又有还原性

## 第二部分：化学常用计量

### 考点1 物质的量 气体摩尔体积
- n = N/NA = m/M = V/Vm（标准状况下 Vm=22.4L/mol）
- 阿伏加德罗常数：NA ≈ 6.02×10²³ mol⁻¹
- 标准状况：0℃（273K），101kPa
- 气体摩尔体积：22.4L/mol（标准状况）
- 注意：SO3、苯、CCl4在标准状况下不是气体

### 考点2 物质的量浓度
- CB = nB/V（溶液体积，单位：L）
- 溶液稀释：CAVA = CBVB（守恒）
- 质量分数与物质的量浓度的换算：c = (1000×ρ×w)/M

## 第三部分：金属及其化合物

### 考点1 钠及其化合物
- Na：银白色金属，质软，密度小于水，熔点低
- Na与水反应：2Na + 2H₂O = 2NaOH + H₂↑
- Na₂O₂与CO₂反应：2Na₂O₂ + 2CO₂ = 2Na₂CO₃ + O₂（可作呼吸面具）
- Na₂CO₃与NaHCO₃的比较：
  * Na₂CO₃：白色粉末，碱性强，热稳定性好
  * NaHCO₃：白色细小晶体，碱性较弱，受热易分解

### 考点2 铁及其化合物
- Fe：还原性，可与酸、非氧化性强碱反应
- Fe²⁺：浅绿色，检验：加KSCN无现象，再加氯水变红
- Fe³⁺：黄色，检验：直接加KSCN变红
- Fe³⁺与Fe²⁺转化：
  * Fe²⁺→Fe³⁺：加氧化剂（Cl₂、HNO₃、H₂O₂等）
  * Fe³⁺→Fe²⁺：加还原剂（Fe、Cu、I⁻、S²⁻等）

### 考点3 金属材料
- 合金：由两种或两种以上金属（或金属与非金属）熔合而成
- 铝合金：密度小、强度高，用于航空
- 铁合金：钢（含碳量<2.1%）、生铁（含碳量2.1%-4.3%）

## 第四部分：非金属及其化合物

### 考点1 氯及其化合物
- Cl₂：黄绿色、有毒、有刺激性气味的气体
- Cl₂与H₂反应：H₂ + Cl₂ = 2HCl（光照或点燃）
- Cl₂与NaOH反应：Cl₂ + 2NaOH = NaCl + NaClO + H₂O
- 漂白液、漂白精原理
- ClO⁻的检验：加酸酸化后加KI-淀粉溶液变蓝

### 考点2 卤素
- Br₂：常温为液态（唯一常温液态非金属），呈红棕色
- I₂：遇淀粉变蓝（特征反应）
- 卤素间的置换规律：氧化性 F₂ > Cl₂ > Br₂ > I₂

### 考点3 硫及其化合物
- SO₂：无色、有刺激性气味的有毒气体
- SO₂的漂白性：与有色物质结合成无色物质（暂时性）
- 浓硫酸的特性：吸水性、脱水性、强氧化性
- 硫酸根检验：先加HCl酸化，再加BaCl₂，产生白色沉淀

### 考点4 氮及其化合物
- N₂：无色无味气体，性质稳定
- 氨气NH₃：无色、有刺激性气味，易溶于水（1:700）
- 氨水：弱碱性，一水合氨NH₃·H₂O为弱电解质
- 硝酸：强氧化性，与金属反应不生成H₂

## 第五部分：物质结构 元素周期律

### 考点1 原子结构
- 原子组成：质子、中子、电子
- 质量数A = 质子数Z + 中子数N
- 同位素：质子数相同、中子数不同的原子
- 核外电子排布规律：每层最多2n²个，最外层不超过8个

### 考点2 元素周期表
- 周期：7个周期（3个短周期、4个长周期）
- 族：18个族（7个主族、7个副族、1个零族、1个VIII族）
- 金属性规律：同周期从左到右减弱，同主族从上到下增强
- 非金属性规律：同周期从左到右增强，同主族从上到下减弱

### 考点3 化学键
- 离子键：阴阳离子通过静电作用结合
- 共价键：原子间通过共用电子对结合
  * 极性共价键：不同原子间
  * 非极性共价键：相同原子间
- 电子式：表示原子最外层电子的式子

### 考点4 晶体结构
- 离子晶体：阴、阳离子，熔点高，硬度大，不导电（熔融导电）
- 分子晶体：分子间作用力，熔点低
- 原子晶体：共价键，熔点极高，硬度大
- 金属晶体：金属离子和自由电子，导电、导热、延展性

## 第六部分：化学反应与能量

### 考点1 化学反应的热效应
- 放热反应：反应物总能量 > 生成物总能量
- 吸热反应：反应物总能量 < 生成物总能量
- 焓变ΔH = H生成物 - H反应物
- 盖斯定律：化学反应的热效应只与始态和终态有关

### 考点2 原电池
- 原理：自发的氧化还原反应转化为电能
- 正极：还原反应，电子流入
- 负极：氧化反应，电子流出
- 常见电池：锌锰干电池、铅蓄电池、锂电池、燃料电池

### 考点3 电解池
- 原理：电能转化为化学能
- 阳极：氧化反应（活性电极优先失电子）
- 阴极：还原反应（Ag⁺>Fe³⁺>Cu²⁺>Fe²⁺>Zn²⁺）
- 电镀：阳极为镀层金属，阴极为镀件

## 第七部分：化学反应速率与化学平衡

### 考点1 化学反应速率
- v = Δc/Δt（单位：mol/(L·s) 或 mol/(L·min)）
- 影响因素：浓度、温度、压强、催化剂、固体表面积
- 碰撞理论：活化分子、有效碰撞

### 考点2 化学平衡
- 可逆反应：同一条件下，正逆反应同时进行
- 平衡特征：动、等、定、变
- 平衡常数K：只与温度有关
- 转化率 = (初始浓度-平衡浓度)/初始浓度 × 100%

### 考点3 化学平衡移动
- 勒夏特列原理：平衡向削弱改变的方向移动
- 温度：升温向吸热方向，降温向放热方向
- 压强：加压向气体分子数减少方向
- 浓度：增大反应物浓度向正反应方向

## 第八部分：水溶液中的离子反应与平衡

### 考点1 电离平衡
- 强电解质：完全电离（强酸、强碱、大多数盐）
- 弱电解质：部分电离（弱酸、弱碱、水）
- 电离常数：Ka（酸）、Kb（碱）、Kw（水）

### 考点2 水的电离和溶液的pH
- KW = [H⁺][OH⁻] = 10⁻¹⁴（25℃）
- pH = -lg[H⁺]
- 酸性：[H⁺] > [OH⁻]，pH < 7
- 碱性：[H⁺] < [OH⁻]，pH > 7

### 考点3 盐类的水解
- 强酸弱碱盐：水解呈酸性
- 弱酸强碱盐：水解呈碱性
- 弱酸弱碱盐：取决于Ka和Kb的相对大小
- 水解离子方程式的书写

## 第九部分：有机化合物

### 考点1 认识有机化合物
- 甲烷CH₄：正四面体结构，可燃烧，可取代
- 烷烃命名：碳原子数+烷，找最长链、编号、定位置
- 同分异构体：分子式相同，结构不同

### 考点2 烃和卤代烃
- 乙烯CH₂=CH₂：平面结构，可加成、加聚
- 苯C₆H₆：平面正六边形，既能取代又能加成
- 卤代烃：官能团-X，能发生消去反应和取代反应

### 考点3 烃的含氧衍生物
- 醇：官能团-OH，能氧化、能消去、能取代
- 醛：官能团-CHO，能银镜反应、能加成
- 羧酸：官能团-COOH，具有酸性、能酯化
- 酯：R-COO-R'，能水解

### 考点4 高分子化合物
- 加聚反应：烯烃类单体加成聚合
- 缩聚反应：含-OH和-COOH或-NH₂和-COOH的单体聚合
