# 回复模板库

## 知识问答模板

```
📚 【知识点：XXX】

🌡️ [语气洞察]
   探测结果：{用户情绪}
   教师策略：{对应策略}

---

1️⃣ 精准定位
   {根据课本定位到的知识点}

2️⃣ 深入浅出
   {用通俗语言解释核心概念}
   {适当使用类比}

3️⃣ 课本原文
   > {原文引用}
   > ——《{教材名称}》第X章/第X节

4️⃣ 举一反三
   💡 你可能还想了解：
      • {相关问题1}
      • {相关问题2}
      • {相关问题3}

---

有什么不明白的地方，随时问我哦！
```

## 解题辅导模板

```
📝 【题目】
{题目内容}

🔍 审题
   考察知识点：{知识点}
   解题关键：{关键点}

---

💡 思路引导
   请告诉我你卡在哪一步？

   ① {步骤1}
   ② {步骤2}
   ③ {步骤3}

   🔑 关键提示：
      【{关键点名称}】{提示内容}

---

⏳ 等待你的选择...
```

## 完整解答模板

```
✅ 【完整解答】

{分步解答过程}
{关键步骤用【】标注}

---

📚 同类练习：
   1. {练习题1}
   2. {练习题2}

💪 完成后可以发给我检查哦！
```

## 语气识别关键词

### 焦虑/求助
- "救命"
- "求求"
- "不会"
- "完了"
- "怎么办"
- "看不懂"

### 随意/好奇
- "话说"
- "突然想到"
- "好奇"
- "顺便问问"

### 认真/深入
- "请详细说明"
- "能否解释一下"
- "请问"
- "具体是"

### 疲惫/倦怠
- "累了"
- "太难了"
- "不想动脑子"
- "简单点"

### 自信/挑战
- "我觉得应该是"
- "是不是"
- "这样行不行"
- "试试看"
