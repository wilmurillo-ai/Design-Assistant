## 📞 接触流程

### 第一步：找到关键联系人

| 渠道 | 方法 |
|------|------|
| **LinkedIn** | 搜索"品牌名 + 市场总监/媒介经理" |
| **官网** | 联系页面/客服转接 |
| **代理商** | 通过 4A 广告公司引荐 |
| **人脉介绍** | 现有客户推荐 |

### 第二步：首次接触话术

```
【电话/微信开场白】

[联系人] 您好，我是亲邻传媒的 Vinny。

看到贵品牌最近和 [代言人] 官宣了，恭喜恭喜！

我们是全国领先的社区媒体广告平台：
- 覆盖 70,000+ 高端小区
- 服务 1700+ 品牌客户
- [类似品牌] 都在投

特别适合 [品牌名] 这种高端品牌，
精准触达高消费人群。

方便的话，我发一份方案给您参考？
```

### 第三步：方案跟进

| 时间 | 行动 | 目标 |
|------|------|------|
| **Day 1** | 首次联系 | 建立联系 |
| **Day 3** | 发送方案 | 定制方案 |
| **Day 7** | 跟进反馈 | 解答疑问 |
| **Day 14** | 促成决策 | 限时优惠 |
| **Day 21** | 最后跟进 | 保持联系 |

## 💰 试投方案

### 低门槛方案（适合首次合作）

| 项目 | 内容 |
|------|------|
| **投放城市** | 深圳/上海（1 个核心城市） |
| **投放媒体** | 单元门灯箱 |
| **投放数量** | 50 面（最低起投） |
| **投放周期** | 2 周（最低周期） |
| **试投预算** | 约 6 万元 |

### 首投优惠

- 制作费全免（省 1,500 元）
- 免费设计服务（省 3,000 元）
- 延长 3 天投放（价值 9,000 元）
- 免费效果监测报告（价值 5,000 元）

**总优惠价值**: 约 1.9 万元
