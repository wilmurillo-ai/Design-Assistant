## 💬 常见异议处理

### 异议 1: "预算有限"

```
理解您的考虑。其实社区媒体投放的门槛很低：

1. **灵活起投** - 单元门 50 面起投，广告门双周起刊
2. **精准投放** - 只投目标小区，避免浪费
3. **效果可衡量** - LBS 数据监测，ROI 清晰

我们可以先从小规模试点开始，
看到效果后再扩大投放范围。

您觉得怎么样？
```

### 异议 2: "已经有合作的媒体公司了"

```
完全理解！多一个备选方案也没有损失嘛 😊

我们亲邻传媒的优势是：
1. **差异化媒体** - 单元门/广告门，必经之路触达
2. **精准 LBS** - 按小区定向，避免泛投放
3. **规模化** - 70,000+ 小区，全国覆盖

不妨把我们当作补充渠道，
在特定区域或特定时期做增量投放。

我可以先发一份方案给您参考，不成交也没关系。
```

### 异议 3: "需要内部讨论"

```
完全理解！这么大的决策确实需要团队讨论。

我这边准备一些材料，方便您内部沟通：
1. 详细的投放方案
2. 类似客户的案例数据
3. ROI 测算表

您看什么时候方便，我可以过来拜访，
当面给您的团队做一个简短的汇报？

或者我们先约个电话会议也可以。
```

### 异议 4: "效果怎么保证"

```
效果保证是我们最重视的。我们有三个保障：

1. **数据监测** - LBS 精准监测，曝光量可量化
2. **A/B 测试** - 不同小区、不同媒体组合测试
3. **优化调整** - 根据数据实时优化投放策略

而且，我们服务了 1700+ 品牌客户，
复投率超过 70%，这就是效果的最好证明。

您有什么具体的效果指标吗？
我们可以针对性地设计监测方案。
```

## 🎯 促成交易话术

### 限时优惠

```
[客户名称] 您好，

跟您同步一个好消息：

本月签约的客户，可以享受：
1. **制作费减免** - 原价 [金额]，现价 [金额]
2. **免费设计** - 价值 [金额] 的设计服务
3. **延长投放** - 买 2 周送 3 天

优惠名额有限，目前只剩 [数字] 个了。

您看这周能定下来吗？
我可以帮您预留这个优惠名额。
```

### 案例促成

```
[客户名称] 您好，

跟您分享一个刚完成的案例：

[类似品牌] 上周刚在我们这边投放，
选了 [数量] 个小区，投放 [周数] 周，
结果 [具体效果数据]。

他们决定下周追加投放，
扩大到 [数量] 个小区。

您的情况和他们很像，
要不要也先小规模试一下？
```

### 稀缺性促成

```
[客户名称] 您好，

跟您同步一下排期情况：

您看中的 [城市/区域]，
下个月的档期已经定了 [百分比]% 了，
特别是 [核心小区]，只剩 [数量] 个点位了。

如果您确定要投的话，
建议尽快锁定这些点位，
不然可能被其他客户订走了。

您看这周能定下来吗？
```
