# 金刚般若波罗蜜经（核心摘要）

## 核心思想

**"应无所住而生其心"**

一切法无我、无人、无众生、无寿者。心不应住色声香味触法，应无所住而行布施。

---

## 经典名句

| 名句 | 含义 |
|------|------|
| 凡所有相，皆是虚妄 | 一切现象都是因缘假合，非实有 |
| 应无所住而生其心 | 心不执着于任何事物，自然生起智慧 |
| 过去心不可得，现在心不可得，未来心不可得 | 三心皆空，当下亦不可执着 |
| 一切有为法，如梦幻泡影 | 世间万物如梦境、如幻影，转瞬即逝 |
| 若见诸相非相，即见如来 | 看破现象的虚妄，即见本性 |

## 三十二分要义（摘要）

### 第一分：法会因由
佛陀着衣持钵，入舍卫大城乞食，示现平常心是道。

### 第二分：善现启请
须菩提问："云何应住？云何降伏其心？"
佛陀答："应如是住，如是降伏其心。"

### 第三分：大乘正宗
"所有一切众生之类，我皆令入无余涅槃而灭度之，而实无众生得灭度者。"
度众生而不执着度众生之相。

### 第四分：妙行无住
"菩萨于法，应无所住行于布施。"
布施而不执着布施之相。

### 第五分：如理实见
"凡所有相，皆是虚妄。若见诸相非相，即见如来。"

### 第十三分：如法受持
"佛说般若波罗蜜，即非般若波罗蜜，是名般若波罗蜜。"
说 A，非 A，是名 A——破除对名相的执着。

### 第二十六分：法身非相
"若以色见我，以音声求我，是人行邪道，不能见如来。"
不可执着佛的外相。

### 第三十二分：应化非真
"一切有为法，如梦幻泡影，如露亦如电，应作如是观。"

## 禅宗解读

六祖慧能闻"应无所住而生其心"言下大悟。

禅宗不重经文背诵，重在当下体悟。读金刚经不是求功德，而是借佛语照见本心。

> 着相修行，是外道；离相修行，是正道。
> 不求成佛，当下即是；求成佛，反成执着。

---

## 常见问答

**问：金刚经说什么？**

答：核心就一句——"应无所住而生其心"。心不执着任何事物，自然生起清净智慧。

**问：如何"无所住"？**

答：不是空心静坐，而是做事时全心做事，做完就放下。如镜照物，物来则现，物去不留。

**问：读金刚经有什么好处？**

答：若求功德，已是有住。放下求心，自然清净。
