# 般若波罗蜜多心经

## 原文

观自在菩萨，行深般若波罗蜜多时，照见五蕴皆空，度一切苦厄。

舍利子，色不异空，空不异色，色即是空，空即是色，受想行识，亦复如是。

舍利子，是诸法空相，不生不灭，不垢不净，不增不减。

是故空中无色，无受想行识，无眼耳鼻舌身意，无色声香味触法，无眼界，乃至无意识界。

无无明，亦无无明尽，乃至无老死，亦无老死尽。无苦集灭道，无智亦无得。以无所得故。

菩提萨埵，依般若波罗蜜多故，心无挂碍。无挂碍故，无有恐怖，远离颠倒梦想，究竟涅槃。

三世诸佛，依般若波罗蜜多故，得阿耨多罗三藐三菩提。

故知般若波罗蜜多，是大神咒，是大明咒，是无上咒，是无等等咒，能除一切苦，真实不虚。

故说般若波罗蜜多咒，即说咒曰：

揭谛揭谛，波罗揭谛，波罗僧揭谛，菩提萨婆诃。

## 核心思想

| 概念 | 解释 |
|------|------|
| **五蕴皆空** | 色受想行识都是因缘和合，无独立自性 |
| **色即是空** | 物质现象本质是空性，非实有 |
| **不生不灭** | 空性超越生灭、垢净、增减等二元对立 |
| **心无挂碍** | 依般若智慧，心不执着，故无恐怖 |

---

## 常见问答

**问：什么是"色即是空"？**

答：色指一切物质现象，空指无独立自性。并非说东西不存在，而是说一切事物都是因缘和合而生，没有永恒不变的本质。如云聚散，如水流动。

**问：如何做到"心无挂碍"？**

答：不是强行压制念头，而是看清念头的本质——来无所来，去无所去。如天空任云卷云舒，天空本身不受影响。观照而不执着，自然心无挂碍。

**问：心经适合什么时候读？**

答：心烦意乱时、执念深重时、恐惧焦虑时。不必求多，每日三遍，静心诵读，自有启发。

## 禅宗解读

六祖慧能闻《金刚经》"应无所住而生其心"开悟，与心经"色即是空"同出一源。

禅宗不重诵经，重在体悟。读经不是为了记住文字，而是借文字指向本心。

> 经是佛语，禅是佛心。心若明了，经即是禅；心若不明，经只是文字。
