# 六祖慧能公案

## 1. 本来无一物

五祖弘忍欲传衣钵，令弟子作偈。

**神秀上座**（首座弟子）作偈：
> 身是菩提树，心如明镜台。
> 时时勤拂拭，勿使惹尘埃。

众人赞叹，五祖却说："未见本性，只到门外。"

**慧能**（碓房舂米行者）听闻后，请人代写：
> 菩提本无树，明镜亦非台。
> 本来无一物，何处惹尘埃。

五祖见偈，知慧能已悟，深夜召见，传《金刚经》至"应无所住而生其心"，慧能言下大悟。

五祖传衣钵，曰：
> "汝为第六代祖，善自护念，广度有情。"

**解读**：
神秀是渐修，慧能是顿悟。渐修如拭镜，顿悟知镜本空。

---

## 2. 仁者心动

慧能得法后，隐于猎人中十五年。

一日至广州法性寺，听印宗法师讲经。

时有风吹幡动，二僧争论。

一僧曰："风动。"
一僧曰："幡动。"

慧能曰："不是风动，不是幡动，**仁者心动**。"

印宗闻之，知是六祖，为之剃度。

**解读**：
风幡之争是外相，心起分别才是根源。万法唯心，境由心造。

---

## 3. 不思善不思恶

慧能得法后，被恶人追赶。

惠明将军追赶至大庾岭，欲夺衣钵。

慧能置衣钵于石上，曰："此衣表信，可力争耶？"遂隐于草莽。

惠明取衣钵，不动。乃曰："行者，我为法来，不为衣来。"

慧能出，曰："汝既为法来，可屏息诸缘，勿生一念。"

良久，曰：
> "不思善，不思恶，正与么时，哪个是明上座本来面目？"

惠明言下大悟。

**解读**：
善恶是二元对立，本心超越善恶。放下分别，当下见性。

---

## 4. 风幡不动

有僧问："风幡不动时如何？"

慧能曰："汝见风幡，已动了也。"

**解读**：
见即是动。心若不动，见而无见。

---

## 5. 佛性无常

有僧问："佛性是常还是无常？"

慧能曰："佛性若常，更说什么善恶诸法？佛性若无常，更说什么修行证果？"

**解读**：
佛性超越常与无常，是绝对的真实。

---

## 六祖禅法核心

| 概念 | 解释 |
|------|------|
| **自性本净** | 人人本具佛性，不假外求 |
| **顿悟成佛** | 一念觉悟，即见本性 |
| **无念为宗** | 念起不随，念过不留 |
| **无相为体** | 不执着外相 |
| **无住为本** | 心不滞留 |
