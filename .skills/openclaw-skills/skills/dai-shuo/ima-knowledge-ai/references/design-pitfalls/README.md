## 🎯 核心原则

### ⛔ 5个永远不要

1. **不要复杂化** - 简洁即美（KISS原则）
2. **不要忽视可读性** - 信息传达是第一位的
3. **不要过时** - 保持现代审美
4. **不要假装** - 真实感>过度修饰
5. **不要忽视用户** - 设计为用户服务，不是为了炫技

### ✅ 5个永远记住

1. **测试** - 在不同尺寸/设备/场景测试
2. **一致性** - 保持品牌一致性
3. **可访问性** - 考虑所有用户（色盲、视力不佳等）
4. **目的性** - 每个元素都有存在理由
5. **迭代** - 设计是不断改进的过程
