## 💼 行业应用建议

### 科技行业
**推荐色：**
1. Transformative Teal（创新感）
2. Cloud Dancer + Cobalt Blue（信任+现代）
3. 避免过于鲜艳的粉色、红色

### 时尚行业
**推荐色：**
1. Violet、Dark Purple（高级感）
2. Clashing Shades（前卫）
3. Cloud Dancer作为背景

### 食品饮料
**推荐色：**
1. Bright Pink（甜点、饮品）
2. Green Shades（有机、健康）
3. 避免Cloud Dancer为主色（抑制食欲）

### 美妆行业
**推荐色：**
1. Bright Pink、Violet（传统强项）
2. Cloud Dancer包装（高级感）
3. 金属色调（秋冬）

### 家居设计
**推荐色：**
1. Transformative Teal（WGSN推荐）
2. Cloud Dancer + 大地色
3. "栖迟"主题色（中国市场）
