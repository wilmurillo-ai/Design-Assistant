## 🎯 四、行业色彩偏好速查表

| 行业 | 主色调 | 原因 |
|-----|-------|------|
| **科技** | 蓝色 | 信任、创新、现代 |
| **金融** | 蓝色、绿色 | 信任、稳定、财富 |
| **健康/医疗** | 蓝色、绿色、白色 | 信任、清洁、健康 |
| **食品/餐饮** | 红色、橙色、黄色 | 刺激食欲、温暖 |
| **环保** | 绿色、棕色 | 自然、可持续 |
| **奢侈品** | 黑色、金色、紫色 | 高级、尊贵 |
| **儿童** | 黄色、橙色、明亮色 | 快乐、活力 |
| **美妆** | 粉色、紫色、金色 | 女性、优雅、奢华 |
| **运动** | 红色、橙色、黑色 | 能量、力量 |
| **教育** | 蓝色、绿色、黄色 | 智慧、成长、创造力 |
