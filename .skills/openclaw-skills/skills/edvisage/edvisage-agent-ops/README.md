# agent-ops

agent-ops gives you visibility into what your agent is doing, what it's costing, and whether it's working. Track API costs by model, log every action, and get daily summaries.

## Installation

Copy the SKILL.md file into your agent's skills directory. Your agent will read and follow these protocols automatically.

## What's Included

- Cost tracking by model and task type
- Action logging with timestamps
- Daily summary reports
- Cost alerts for unusual spending
- Model routing awareness
