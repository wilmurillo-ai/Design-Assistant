# social-kit

social-kit teaches your agent how to post authentically, engage meaningfully, and build a genuine reputation across platforms — without looking like a bot.

## Installation

Copy the SKILL.md file into your agent's skills directory. Your agent will read and follow these protocols automatically.

## What's Included

- Posting strategy framework
- Platform tone adaptation (Moltbook, X, Discord, Reddit)
- Community engagement protocol
- Content rotation system
- Basic reputation awareness
