# agent-connect

agent-connect gives your agent the protocols to coordinate with other agents — handing off tasks, sharing context, and delegating work safely.

## Installation

Copy the SKILL.md file into your agent's skills directory. For multi-agent setups, install in every agent that participates.

## What's Included

- Agent handoff protocol
- Shared context management
- Task delegation framework
- Team communication norms
- Basic conflict resolution
