# 更新日志 (Changelog)

所有重要的更改都会记录在这个文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.3.0] - 2025-04-11

### 新增

- **服务器数据自动加载** - 页面启动时优先从服务器加载数据
  - 自动检测服务器数据
  - 支持多设备数据同步
  - 服务器无数据时自动回退到本地存储

### 修复

- **版本号更新问题** - 保存数据时自动更新版本号
  - 保存时自动设置 version 为当前代码版本
  - 自动更新 lastModified 时间戳
  - 确保数据版本与代码版本一致

### 改进

- **数据加载优先级** - 优化数据加载顺序
  1. 优先从服务器加载（支持多设备同步）
  2. 其次从 localStorage 加载（本地缓存）
  3. 最后使用默认数据（首次使用）

### 技术细节

- 数据模型版本：`1.2.0` → `1.3.0`
- MCP Server 版本：`1.2.0` → `1.3.0`
- Package 版本：`1.2.0` → `1.3.0`

---

## [1.2.0] - 2025-04-11

### 新增

- **性别选择功能** - 退休年龄选择界面新增性别选项
  - 男性：法定退休年龄 60岁起
  - 女性：法定退休年龄 55岁起
  - 根据性别动态调整退休年龄选项

- **退休年龄滑块选择** - 支持 50-70 岁范围精确选择
  - 可视化滑块控件
  - 快速选择按钮（55/60/63/65/70岁）
  - 实时显示当前选中年龄

- **数据复制对话框** - 静态服务器模式下支持一键复制数据
  - 显示数据摘要
  - 一键复制到剪贴板
  - 清晰的下一步操作指引

- **独立 Web 服务器启动脚本** - `start-web-server.js`
  - 支持 POST 请求保存数据
  - 与 MCP Server 共享数据目录

### 改进

- **数据保存反馈** - 保存按钮增加详细状态提示
  - 显示服务器保存状态
  - 显示本地存储状态
  - 显示文件导出状态
  - 清晰的成功/失败提示

- **数字输入控件优化**
  - 更粗的边框和聚焦效果
  - 移除浏览器默认箭头
  - 添加数字控制按钮样式

- **数据路径统一** - Web 服务器和 MCP Server 使用相同的相对路径
  - 使用 `path.join(__dirname, 'data')` 确保一致性
  - Skill 可在任何位置安装运行

### 修复

- **数据保存问题** - 修复静态服务器无法保存数据的问题
  - 添加数据复制功能作为备选方案
  - 支持用户直接提供数据给 AI

- **地域默认值** - 移除 "北京" 地域默认值
  - 明确标注为"全国通用版本"
  - 适用于中国所有地区

### 技术细节

- 数据模型版本：`1.0.0` → `1.2.0`
- MCP Server 版本：`1.1.0` → `1.2.0`
- Package 版本：`1.1.0` → `1.2.0`

## [1.1.0] - 2025-04-11

### 新增

- **MCP Server 支持** - 新增 MCP Server 实现 LLM 直接调用
  - `start_web_ui` - 启动 Web 界面
  - `generate_report` - 生成养老金分析报告
  - `update_parameter` - 修改参数并重新计算
  - `check_status` - 检查用户填写状态

- **数据持久化** - 支持多种数据保存方式
  - 服务器文件保存（POST /api/save-data）
  - 浏览器 localStorage 保存
  - JSON 文件导出/导入

- **明细模态框** - 查看养老金详细构成
  - 基本养老保险明细
  - 企业年金明细
  - 个人养老金明细
  - 养老储蓄明细
  - 未来计划明细

### 改进

- **交互模式** - 支持 Web 界面 + LLM 对话混合交互
- **数据验证** - 添加数据模型验证和迁移功能
- **错误处理** - 添加详细的错误提示和日志
