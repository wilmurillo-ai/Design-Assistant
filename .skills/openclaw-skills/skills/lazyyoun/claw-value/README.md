# 🦞 ClawValue - OpenClaw 龙虾值评估系统

量化你在 OpenClaw 上的 AI 自动化能力，生成趣味化的"龙虾能力估值"评估报告。

## ✨ 功能特点

- 📊 **使用深度评估**：五级技术指标体系（入门/初级/中级/高级/大师）
- 💰 **价值估算**：基于真实数据的价值量化，根据金额区间显示不同颜色和趣味文案
- 🦞 **龙虾锐评**：趣味调侃话术系统，支持 URL 参数传入自定义锐评
- 🎨 **16 套主题风格**：新野兽派（默认）、深海龙虾、赛博霓虹、极简水墨等
- 🏆 **成就系统**：24 个成就徽章，解锁你的专属称号
- 🖼️ **AI 图片生成**：生成专属龙虾形象，保存到本地
- 📸 **截图导出**：一键导出页面截图
- ⌨️ **快捷键支持**：R/T/M/? 等快捷操作

![示意图](./clawvalueshowpic.png)

## 🚀 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 启动服务

```bash
python3 scripts/server.py --port 5002
```

访问：http://localhost:5002

### 命令行数据采集

```bash
python3 scripts/collector.py --output json
```

## 📡 API 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/` | GET | 仪表盘主数据接口 |

| `/api/skills` | GET | 获取技能列表 |
| `/api/sessions` | GET | 获取会话历史 |
| `/api/evaluation` | GET | 获取评估结果 |
| `/api/refresh` | POST | 刷新数据 |
| `/api/generate-image` | POST | 生成龙虾图片（保存到本地） |
| `/api/health` | GET | 健康检查 |

    ├── models.py           # 数据模型
    ├── collector.py        # 数据采集模块
    ├── evaluation.py       # 评估引擎
    ├── image_generator.py  # 图片生成模块

    ├── constants.py        # 常量定义

## 🔧 技术栈

- **后端**: Python 3 + Flask
- **数据库**: SQLite
- **前端**: HTML/CSS/JavaScript
- **图片生成**: 阿里云万象 API
- **截图导出**: html2canvas
