# Full ASO Strategy Report Template

Use this template when the user wants a comprehensive ASO strategy document covering all areas. Read the other reference files before generating this report — this template is the output structure, not the methodology.

## When to Generate a Full Report

Generate this when the user:
- Is launching a new app
- Wants a complete overhaul of their ASO strategy
- Needs a document to share with their team or stakeholders
- Explicitly asks for a "full strategy" or "complete ASO plan"

For focused requests (just keywords, just screenshots, etc.), don't use this template — use the targeted reference files instead.
