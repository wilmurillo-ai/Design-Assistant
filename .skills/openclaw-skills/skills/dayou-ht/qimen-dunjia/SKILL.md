# 奇门遁甲排盘系统

中国传统时空易理预测学，基于阴阳五行、九宫八卦、八门九星、八神等元素进行时空推演。

## 使用场景

✅ **使用此 skill 当：**

- 需要择时（选择吉时吉日）
- 问事占卜（事业、财运、感情、人际）
- 今日运势参考
- 决策时机分析
- 需要吉方吉色建议

## 使用方法

### 直接运行（当前时间）

```bash
node ~/.openclaw/workspace/skills/qimen-dunjia/qimen.js
```

### 指定时间

```bash
node ~/.openclaw/workspace/skills/qimen-dunjia/qimen.js "2026-03-15 14:30"
```

## 输出说明

| 项目 | 说明 |
|------|------|
| **四柱** | 年月日时干支 |
| **节气/局制** | 当前节气、阴阳遁、局数 |
| **财运指数** | 生门落宫+五行生克+神星权重 |
| **问事成功率** | 值符+九星+八门综合评分 |
| **人际/感情** | 六合神+阴阳二气+落宫状态 |
| **结果可能性** | 全局吉门配置+空亡影响 |
| **吉方/吉色** | 值符宫位对应的方位和颜色 |
| **秘法推语** | AI 生成的运势解读 |

## 特殊符号说明

- **驿马（马）**：动迁、变化、驿动之象
- **旬空（空）**：空亡、无力、时机未到
- **刑**：相刑、冲突、阻碍
- **迫**：门迫、压力大、受制
- **墓**：入墓、收藏、隐藏
