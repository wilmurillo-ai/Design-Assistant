## 🔍 已发现的关键信息

### 登录页面
- **正确 URL**: `https://v.flomoapp.com/login` （不是 `flomoapp.com`）
- **输入框类型**: 手机号/邮箱（`input[type="text"]`）
- **登录方式**: 点击"登录"按钮后跳转到 `/mine`

### 导出流程
```
点击用户名
  ↓
菜单弹出（包含：账号详情、会员中心、偏好设置、导出/导入笔记等）
  ↓
悬停或点击"导出/导入笔记"
  ↓
子菜单出现：导出笔记 / 导入笔记
  ↓
点击"导出笔记"
  ↓
弹窗出现：标题"导出笔记"，说明文字，"导出"按钮
  ↓
点击"导出"按钮
  ↓
进度提示：正在生成导出文件（x%）
  ↓
导出完成，自动下载 ZIP 文件
```

### 元素选择器
- 用户名：`text=/Ryan\\.B/` 或通过文本查找
- 导出/导入笔记：`get_by_text('导出/导入笔记')`
- 导出笔记子菜单：`get_by_text('导出笔记', exact=True)`
- 导出按钮：`get_by_role('button', name='导出')`

## 🚀 使用方式

### 方式1：手动转换（推荐，最稳定）⭐⭐⭐⭐⭐

**适用场景**：首次导入、定期更新

**步骤**：
```bash
# 1. 从 flomo 网页版手动导出数据
# 2. 运行转换脚本
cd <skill-directory>

python scripts/convert_v2.py \
  --input "~/Downloads/flomo@YourName-日期/YourName的笔记.html" \
  --output "~/Documents/Obsidian/flomo" \
  --mode by-date \
  --tag-prefix "flomo/"
```

**优点**：
- ✅ 100% 可靠
- ✅ 完整支持所有功能
- ✅ 不需要保存密码
- ✅ 可以随时运行

**缺点**：
- ⏰ 需要手动操作（约 1 分钟）
