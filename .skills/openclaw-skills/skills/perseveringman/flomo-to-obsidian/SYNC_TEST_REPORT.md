## 🐛 发现并修复的问题

### 问题 1：解压后找不到 HTML 文件
**发现时间**：2026-03-11 17:16:02

**现象**：
```
ERROR - 解压后找不到 HTML 文件
```

**原因**：
- 解压后的 HTML 文件在子目录中：`flomo@YourName-20260311/YourName的笔记.html`
- 脚本只在顶层目录查找：`*.html`

**解决方案**：
```python
# 修改前
html_files = list(extract_dir.glob('*.html'))

# 修改后（递归查找）
html_files = list(extract_dir.glob('**/*.html'))
```

**状态**：✅ 已修复

### 问题 3：后台模式登录状态检测
**发现时间**：2026-03-11 17:17:08

**现象**：
- 已经登录但仍然进入登录分支
- URL 检测不准确

**原因**：
- 页面加载完成后才能获取正确的 URL
- `page.url` 在跳转过程中可能不准确

**解决方案**：
```python
# 等待页面完全加载
page.goto('https://v.flomoapp.com/login', wait_until='networkidle')
time.sleep(2)

# 然后检查 URL
current_url = page.url
if '/mine' in current_url:
    logger.info("✅ 检测到已登录状态")
```

**状态**：✅ 已修复

### 问题 4：转换器参数错误
**发现时间**：2026-03-11 17:24:29

**现象**：
```
TypeError: ObsidianConverter.__init__() got an unexpected keyword argument 'memos'
```

**原因**：
- `ObsidianConverter` 的初始化参数与调用不匹配
- 应该传 `flomo_html_dir` 而不是 `memos`

**解决方案**：
```python
# 修改前
converter = ObsidianConverter(
    memos=memos,
    source_dir=str(html_file.parent)
)

# 修改后
converter = ObsidianConverter(
    output_dir=str(self.output_dir),
    flomo_html_dir=str(html_file.parent),
    mode='by-date',
    tag_prefix=self.tag_prefix
)
result = converter.convert(notes)
```

**状态**：✅ 已修复

### 问题 5：点击用户名失败
**发现时间**：2026-03-11 17:24:17

**现象**：
- 通过坐标点击用户名不稳定
- 有时点不到正确位置

**原因**：
- 页面布局可能变化
- 坐标点击不可靠

**解决方案**：
```python
# 优先使用文本匹配
username_selectors = [
    'text=YourName',  # 直接匹配用户名
    '[class*="user"]',
    '[class*="avatar"]'
]

for selector in username_selectors:
    locator = page.locator(selector).first
    if locator.count() > 0:
        locator.click()
        break
```

**状态**：✅ 已修复
