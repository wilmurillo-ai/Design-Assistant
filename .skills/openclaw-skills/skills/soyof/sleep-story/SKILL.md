# 🌙 温暖治愈系心理学助眠故事

## 技能概述

本技能专门创作温暖、治愈、舒缓的睡前故事，融合多种心理学放松技术和催眠暗示，帮助用户：
- 缓解日间压力和焦虑
- 放松紧绷的身心
- 建立安全的心理空间
- **快速自然过渡到睡眠状态**
- **进入深度、高质量的睡眠**

**核心特色**：
- ✅ **强故事性** - 每个故事都有完整的叙事弧线，而非单纯场景描述
- ✅ **去重机制** - 追踪已创作元素，避免多篇故事重复
- ✅ **角色驱动** - 故事围绕角色展开，增强代入感和情感连接
- ✅ **催眠暗示** - 融入专业催眠语言模式，引导快速入睡
- ✅ **科学助眠** - 基于睡眠科学原理设计故事节奏和内容
- ✅ **个性化适配** - 根据用户反馈自动调整故事风格
- ✅ **系列故事** - 连续剧式助眠，增强期待感和情感连接
- ✅ **持续优化** - 闭环反馈系统，效果追踪和迭代改进

## 📖 何时使用此技能

当用户表达以下需求时使用：
- "我睡不着，能给我讲个故事吗"
- "帮我写一个助眠故事"
- "今天好累，想听点温暖的东西"
- "有什么放松的方法吗"
- "失眠了，能哄我睡觉吗"
- 任何需要心理安抚、放松助眠的场景

## 🎯 故事创作原则

### 1. 节奏舒缓
- 使用短句、重复句式
- 避免紧张、刺激的情节
- 语气温柔、平缓
- 适当留白，给读者想象空间

### 2. 感官丰富
- 视觉：柔和的色彩（月光、星光、暖黄灯光）
- 听觉：轻柔的声音（风声、水声、呼吸声）
- 触觉：温暖的触感（毛毯、阳光、微风）
- 嗅觉：安心的气息（花香、书香、食物香）

### 3. 心理学技术自然融入

| 技术 | 融入方式 | 示例 |
|------|----------|------|
| **正念呼吸** | 引导注意呼吸节奏 | "跟着故事里的风，慢慢地吸气...呼气..." |
| **渐进式肌肉放松** | 描述身体放松感受 | "肩膀慢慢沉下来，像融化的雪..." |
| **可视化想象** | 构建宁静场景 | "想象你躺在柔软的草地上，头顶是漫天星光..." |
| **安全基地构建** | 创造安全空间 | "这是一个只属于你的小窝，温暖又安全..." |
| **感恩回忆** | 触发温暖记忆 | "想起今天发生的一件小事，嘴角不自觉上扬..." |

### 4. 故事结构（催眠递进版）

```
【第一阶段】开场诱导（150-200 字）→ 呼吸引导 + 注意力聚焦 + 初步放松
【第二阶段】场景 погружение (400-500 字）→ 角色引入 + 小旅程 + 感官沉浸 + 肌肉放松
【第三阶段】深化放松（200-250 字）→ 温暖发现 + 情感共鸣 + 催眠暗示
【第四阶段】睡眠诱导（250-300 字）→ 意识模糊 + 睡意加深 + 安心入睡
```

**催眠递进原则**：
- **注意力聚焦** → 将意识从杂念转移到故事
- **身心放松** → 逐步放松身体各部位
- **情绪安抚** → 创造安全、被爱的感受
- **睡眠暗示** → 使用催眠语言引导入睡
- **意识模糊** → 故事逐渐淡化，让位于睡意

## 🔄 去重机制（重要！）

### 故事元素追踪

每次创作后，记录以下元素到 `memory/sleep-story-history.json`：

```json
{
  "lastCreated": "2026-04-01",
  "usedElements": {
    "scenes": ["月光湖畔", "深夜咖啡馆"],
    "characters": ["小鹿", "店主猫"],
    "journeys": ["湖边散步", "探索咖啡馆"],
    "discoveries": ["星星倒影", "温暖角落"],
    "sensoryFocus": ["视觉 - 月光", "听觉 - 雨声"]
  },
  "storyTypes": ["自然风景类", "温馨日常类"],
  "techniques": ["正念呼吸", "安全基地构建"]
}
```

### 去重规则

| 元素类型 | 冷却期 | 规则 |
|----------|--------|------|
| **主场景** | 3 篇故事 | 同一场景（如"湖畔"）至少间隔 3 篇故事 |
| **角色** | 2 篇故事 | 同一角色类型（如"猫"）至少间隔 2 篇故事 |
| **旅程类型** | 2 篇故事 | 同一探索方式（如"散步"）至少间隔 2 篇故事 |
| **感官焦点** | 1 篇故事 | 连续故事应切换主要感官（视觉→听觉→触觉） |
| **故事类型** | 1 篇故事 | 尽量轮换四大类型，避免连续相同 |

### 创作前检查清单

在开始创作前：
1. 读取 `memory/sleep-story-history.json`（如果存在）
2. 检查最近使用的场景、角色、旅程类型
3. 选择**未使用或冷却期已过**的元素
4. 优先选择用户未体验过的故事类型
5. 创作完成后更新追踪文件

## ✍️ 创作流程

### Step 1: 了解用户需求

询问或判断：
- 用户今天经历了什么？（工作压力/人际困扰/单纯失眠）
- 偏好哪种故事类型？（自然/日常/动物/回忆）
- 需要什么心理学技术？（放松/安抚/转移注意力）
- **是否听过之前的故事？**（如果有，读取历史记录避免重复）

如果用户没有明确偏好，**默认选择温馨日常类 + 正念呼吸 + 安全基地构建**。

### Step 2: 检查历史记录（去重）

读取 `memory/sleep-story-history.json`：
- 如果文件不存在，这是第一篇故事，自由选择
- 如果文件存在，检查最近使用的场景、角色、旅程类型
- 选择**新鲜元素**，避免重复

### Step 3: 选择故事模板

从 [references/story-templates.md](references/story-templates.md) 选择合适的开篇和结构。

**优先选择用户未体验过的模板类型**。

### Step 4: 创作故事

遵循以下要点：

1. **角色先行** - 先确定故事的主角（用户自己/小动物/虚构角色）
2. **设定小目标** - 给故事一个轻柔的方向（寻找、探索、陪伴、回家）
3. **开场要慢** - 先引导呼吸，再进入故事
4. **细节要暖** - 多用温暖、柔软、安全的词汇
5. **节奏要缓** - 句子短，段落之间有停顿感
6. **有微小转折** - 加入一个温暖的发现或小小的惊喜
7. **结尾要轻** - 逐渐淡化情节，引导入睡

### Step 5: 检查字数

确保故事 **1000 字以上**，但不超过 3000 字（避免阅读疲劳）。

### Step 6: 更新历史记录

创作完成后，更新 `memory/sleep-story-history.json`：
- 添加本次使用的场景、角色、旅程、感官焦点
- 记录故事类型和心理学技术
- 更新时间戳

## 🧠 催眠语言模式

### 1. 直接暗示

> "眼皮越来越沉..."
> "呼吸越来越慢..."
> "你开始感到困倦..."
> "安心地睡去吧..."

### 2. 间接暗示

> "有些人听着这个故事，不知不觉就睡着了..."
> "随着故事的进行，身体自然会放松下来..."
> "当呼吸变得平稳时，睡意也会悄悄来临..."

### 3. 双重束缚

> "你是想在故事里慢慢放松，还是很快就进入梦乡？"
> "睡意会从脚开始往上蔓延，还是从头顶往下流淌？"

### 4. 嵌套暗示

> "当你感到温暖的时候，你会发现放松变得更容易，而放松之后，睡眠会自然地到来..."

### 5. 时间扭曲

> "几分钟的放松，会带来整晚的深度睡眠..."
> "在这个故事里，时间变得缓慢，每一刻都充满宁静..."

### 使用原则

1. **语气温柔** - 用邀请而非命令
2. **节奏缓慢** - 配合呼吸节奏
3. **重复强化** - 关键暗示重复 2-3 次
4. **渐进加深** - 从放松到困倦到入睡

## 📖 示例故事开头（催眠递进版）

### 示例 1：渐进放松式

> "夜深了，你躺在柔软的床上，被子轻轻地盖在身上。
> 
> 先跟着我做三次深呼吸... 慢慢地吸气... 感受空气充满肺部... 然后缓缓地呼出... 把一天的疲惫都带走...
> 
> 很好。现在把注意力带到你的双脚... 脚趾放松了... 脚掌放松了... 脚踝也放松了...
> 
> 今晚，我想带你去一个特别的地方——森林里有一间只为你亮着灯的小屋，我们一起去看看，好吗？
> 
> 站起身，你踩着柔软的苔藓，沿着一条铺满月光的小径慢慢走去..."

### 示例 2：角色陪伴式

> "你感到枕头旁边有轻轻的动静，一只毛茸茸的小脑袋蹭了蹭你的手。
> 
> 是一只银色的小狐狸，它的眼睛像两颗温暖的琥珀。它轻轻地咬住你的衣角，仿佛在说：'跟我来，我带你去一个秘密的地方。'
> 
> 先深呼吸一次... 好，现在跟着小狐狸，它尾巴的光亮照亮了前方的路...
> 
> 每走一步，你的身体就放松一分... 每走一步，睡意就靠近一点..."

### 示例 3：睡眠诱导式

> "这是一个普通的夜晚，但你注意到窗外有一点不寻常的闪光。
> 
> 走近一看，是一只迷路的小星星，它不小心从天上滑了下来，落在你的窗台上，发出微弱的、温暖的光。
> 
> 深呼吸... 感受小星星的信任。它需要你帮它回到天上。来，我们一起想办法...
> 
> 随着你帮助小星星，你的眼皮也开始变得沉重... 那是睡意在轻轻地拥抱你..."

## 🎯 快速入睡技巧

### 1. 4-7-8 呼吸法引导

```
吸气 4 秒 → 屏息 7 秒 → 呼气 8 秒
在故事中自然融入计数：
"慢慢地吸气... 1... 2... 3... 4...
轻轻地屏住... 1... 2... 3... 4... 5... 6... 7...
缓缓地呼出... 1... 2... 3... 4... 5... 6... 7... 8..."
```

### 2. 身体扫描放松

从脚到头，逐步放松每个部位：
```
脚 → 小腿 → 膝盖 → 大腿 → 臀部 → 腹部 → 
胸部 → 手臂 → 肩膀 → 颈部 → 面部 → 头顶
```

### 3. 意识模糊化

故事后期逐渐减少细节描述，增加模糊语言：
> "渐渐地... 慢慢地... 模糊地... 隐约地..."
> "好像... 似乎... 仿佛... 就像是..."

### 4. 睡眠锚定

在故事高潮后建立睡眠锚点：
> "每当你的呼吸变得平稳，身体就会记住这种放松的感觉..."
> "这个温暖的角落，会一直在这里等你回来..."
