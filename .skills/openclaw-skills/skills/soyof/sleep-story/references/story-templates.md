# 故事模板库（催眠递进版）

## 🌙 催眠故事结构原则

每个助眠故事都遵循**催眠递进结构**，从清醒状态逐步引导至深度睡眠：

```
【第一阶段】注意力聚焦 (150-200 字)
  目标：将意识从杂念转移到故事
  技术：呼吸引导 + 感官聚焦
  语言：清晰、具体、有画面感

【第二阶段】身心放松 (400-500 字)
  目标：放松身体，降低生理唤醒
  技术：渐进式肌肉放松 + 呼吸绑定
  语言：温暖、柔软、节奏缓慢

【第三阶段】情绪安抚 (200-250 字)
  目标：创造安全感，释放心理防御
  技术：安全基地 + 感恩回忆 + 温暖发现
  语言：被爱、被接纳、被保护

【第四阶段】意识模糊 (150-200 字)
  目标：减少认知加工，进入半睡半醒
  技术：模糊语言 + 间接暗示
  语言：渐渐地、好像、仿佛、隐约

【第五阶段】睡眠诱导 (250-300 字)
  目标：自然过渡到睡眠
  技术：直接暗示 + 睡眠锚定
  语言：眼皮沉、呼吸慢、安心睡
```

## 模板使用说明

1. **不要照搬模板** - 模板是骨架，需要填充细节和个性化元素
2. **保持递进节奏** - 确保从清晰→放松→模糊→睡眠的过渡
3. **检查催眠元素** - 每个阶段都要有对应的技术融入
4. **去重检查** - 使用前检查历史记录，避免重复场景和角色

## 模板 1：自然风景类 - 《月光引路》

```
【开场 - 角色引入 + 呼吸引导】(150 字)
夜深了，先做三次深呼吸...
你注意到窗外有一道特别的月光，比平时更亮、更温柔...
那道月光像是在邀请你，跟着它走...

【场景引入 + 小目标】(200 字)
推开窗，月光铺成了一条银色的小径，延伸到远处的森林...
你踏上这条光之路，脚下是柔软的草地...
月光似乎在带你去某个地方，你好奇地跟着...

【主体故事 - 旅程】(500 字)
走进森林，树木轻轻地为你让开道路...
月光小径带你来到一片开阔的湖边，湖面像镜子一样平静...
湖中央有一座小岛，岛上有一棵发光的树...
你找到一艘小船，慢慢地划向小岛...
水波轻轻地摇晃着船，像摇篮一样...
靠近小岛，你看到树下有一个小小的身影在等你...

【放松融入】(200 字)
那是一只银色的鹿，它的角上挂着星星...
它不怕你，静静地走近，用温暖的鼻子碰了碰你的手...
那一刻，你感到一种深深的平静...
肩膀慢慢放松，像融化的雪...
呼吸和湖水一起，轻轻地起伏...

【温暖发现】(150 字)
银色鹿带你走到发光树下，树上挂满了小小的光球...
每一个光球里，都装着一个美好的记忆...
你伸手触碰其中一个，温暖的光流进心里...
那是你今天最平静的时刻，被永远地保存了...

【睡眠引导】(150 字)
躺在树下，银鹿静静地陪在你身边...
头顶的星星在闪烁，像是在说晚安...
眼皮越来越沉，呼吸越来越慢...
湖水轻轻摇晃着整个岛屿，像摇篮一样...
在月光的守护下，安心睡去吧...
晚安，好梦...
```

## 模板 2：温馨日常类 - 《深夜书店的秘密》

```
【开场 - 呼吸引导】(150 字)
先让呼吸慢下来，跟着故事的节奏...
吸气... 呼气... 把注意力带到这里...
你走在一条安静的街道上，已经是深夜了...

【场景引入 + 小目标】(200 字)
街角有一家书店，还亮着暖黄色的灯...
你推开门，门铃轻轻地响了一声...
店主是一只戴着眼镜的老猫，它点点头，继续看书...
你想找一本能让心静下来的书，开始在书架间慢慢逛...

【主体故事 - 探索】(500 字)
书架很高，散发着旧书的香气...
你走过一排排书架，手指轻轻地滑过书脊...
在最里面的角落，你发现了一本没有名字的书...
翻开第一页，里面是空白的，但当你盯着看时，文字慢慢地浮现...
那是专属于你的故事，写着你今天经历的小事...
每一件小事都被温柔地记录着，即使是普通的瞬间...
你继续翻页，看到明天的篇章也开始浮现...
那是充满希望的、平静的明天...

【放松融入】(200 字)
双手捧着书，温暖从掌心传到全身...
你找到一个靠窗的位置坐下，窗外下着小雨...
身体陷进柔软的沙发里，越来越沉...
老猫跳到你腿上，找到一个舒服的位置，开始打呼噜...
那呼噜声像一个小马达，有规律地震动着...

【温暖发现】(150 字)
你突然明白，这本书会一直在这里等你...
任何时候，只要你需要，就可以回来翻阅...
你被这个世界温柔地记着，从来不是一个人...
壁炉里的火轻轻地跳动着，发出噼啪声...
你感到一种久违的平静，像被世界温柔地接纳...

【睡眠引导】(150 字)
老猫睡着了，身体轻轻地起伏...
雨声越来越轻，灯光越来越柔和...
书还捧在手里，但眼睛慢慢地闭上...
在书香和雨声里，进入温暖的梦乡...
晚安，明天见...
```

## 模板 3：动物伙伴类 - 《迷路的小云鲸》

```
【开场 - 呼吸引导】(150 字)
慢慢地吸气... 呼气... 把注意力带到这里...
你躺在草地上，看着天空中的云朵...
有一朵云的形状很特别，像一只小小的鲸鱼...

【场景引入 + 小任务】(200 字)
突然，那朵云鲸从天空中慢慢地飘下来，落在你身边...
它很小，只有你的手臂那么长，发出柔和的蓝光...
它看起来有点害怕，轻轻地颤抖着...
你明白了，它迷路了，需要帮助回到天上...
你轻声说："别怕，我帮你。"

【主体故事 - 陪伴与帮助】(500 字)
小云鲸信任地蹭了蹭你的手，冰凉但柔软...
你把它抱起来，它发出轻轻的、像风铃一样的声音...
你开始想办法送它回家...
站起来，你走到一个高一点的地方，举起小云鲸...
但它还是够不到天空...
小云鲸发出担忧的声音，你安慰它："没关系，我们还有其他办法。"
你想到，也许它需要的是风...
你闭上眼睛，感受风的方向...
轻轻地，你对着小云鲸吹了一口气，带着你的祝福...

【放松融入】(200 字)
小云鲸开始发光，越来越亮...
它慢慢地飘起来，离你远了一点，但还在你身边盘旋...
你跟着它的节奏，慢慢地呼吸...
吸气... 呼气... 小云鲸也一起一伏...
你的呼吸变得和小云鲸一样，轻柔而规律...
身体随着呼吸，越来越放松...

【温暖发现】(150 字)
小云鲸突然明白了，它自己可以飞...
它绕着你飞了一圈，像是在告别...
然后它越飞越高，回到了天空中...
在天上，它变成了一朵真正的云，和其他云一起漂浮...
但它还记得你，在天空中轻轻地闪烁着蓝光...
那一刻，你感到被无条件地信任着，你的善意帮助了一个小生命...

【睡眠引导】(150 字)
躺在草地上，看着天空中的小云鲸...
它还在闪烁，像是在说谢谢...
眼皮沉了，呼吸慢了，星星也模糊了...
在星空的守护下，安心入睡...
晚安，小云鲸... 晚安...
```

## 模板 4：童年回忆类 - 《外婆的阁楼》

```
【开场 - 呼吸引导 + 回忆触发】(150 字)
先让心静下来，跟着我回到一个熟悉的地方...
深呼吸... 想起一个让你感到安全的人...
也许是外婆，也许是爷爷，也许是某个照顾过你的人...

【场景引入】(200 字)
你回到了小时候的那栋老房子，推开木门...
熟悉的饭菜香扑面而来，院子里的摇椅在轻轻地摇晃...
你走上楼梯，来到阁楼...
这是你的秘密基地，只有你知道的地方...

【主体故事 - 探索与发现】(500 字)
阁楼里堆满了旧物，阳光从圆形的小窗照进来...
你慢慢地走着，脚下是老旧的木地板，发出轻微的吱呀声...
角落里有一个木箱子，你记得里面装着宝贝...
打开箱子，里面是：
- 一个褪色的布娃娃，脸上还有你小时候画的笑脸
- 一罐玻璃弹珠，在阳光下闪闪发光
- 一本相册，里面是你长大的每一个瞬间
- 一封泛黄的信，是外婆写给你的，虽然你不记得什么时候收到的...

你拿起那封信，轻轻地读：
"亲爱的宝贝，无论你长多大，飞多远，这里永远是你的家。
累了就回来，外婆的阁楼永远为你留着。"

【放松融入】(200 字)
眼泪轻轻地流下来，但不是悲伤，是温暖...
你把信贴在胸口，感受那份跨越时间的爱...
身体随着呼吸，慢慢地放松...
肩膀沉下来，像卸下了所有的负担...
你想起，你一直被爱着，从来都是...

【温暖发现】(150 字)
外婆从楼梯走上来，手里端着一盘刚烤好的饼干...
她还是记忆中的样子，微笑着说："饿了吧？"
你接过饼干，咬了一口，还是那个熟悉的味道...
外婆坐在你身边，你们没有说话，只是静静地坐着...
那种被照顾、被疼爱的感觉，暖到心里...

【睡眠引导】(150 字)
外婆轻轻地拍着你的背，像小时候一样...
摇椅还在摇晃，但你已经睁不开眼睛了...
在熟悉的节奏里，安心地睡去吧...
外婆的爱会一直陪着你，在梦里，在心里...
晚安，宝贝... 外婆爱你...
```

## 模板 5：综合类 - 《星光列车》

```
【开场 - 呼吸引导 + 肌肉放松】(150 字)
深呼吸... 从脚到头，慢慢放松...
脚趾放松... 小腿放松... 膝盖放松...
大腿放松... 身体越来越沉...

【场景引入 + 小旅程】(200 字)
你来到一个小小的火车站，已经是深夜了...
站台上只有一盏灯，发出暖黄色的光...
一列小小的列车慢慢地进站，没有声音，像飘进来一样...
列车是银色的，车窗里透出星星的光...
你走上列车，找到一个舒服的座位坐下...
列车轻轻地启动，你开始了今晚的旅程...

【主体故事 - 旅程中的风景】(400 字)
列车越开越高，离开了地面，飞向天空...
窗外的风景开始变化：
- 你飞过安静的城市，灯光像地上的星星
- 你飞过森林，树木在月光下像剪影
- 你飞过海洋，海面反射着银河的光
- 你飞过云层，云朵像棉花糖一样柔软...

列车里还有其他乘客：
- 一只抱着枕头的小熊，已经睡着了
- 一只兔子，在轻轻地织着什么东西
- 一个和你一样的人，对你微笑点头...

你们都不说话，但感到一种陪伴的温暖...

【安全基地 + 感恩】(200 字)
列车长是一只慈祥的猫头鹰，它走过来，递给你一杯热茶...
"这是今晚的特别饮品，喝了它，你会记得所有美好的事。"
你接过茶杯，温热的水汽升腾...
喝了一口，你想起了今天的一件小事...
也许是陌生人的一个微笑，也许是一顿好吃的饭，也许是一片美丽的云...
那件小事现在变得特别珍贵，像一颗星星，被你收藏在心里...

【温暖发现】(150 字)
列车慢慢地减速，你看到目的地了...
是一片开满花的草地，天空中挂满了星星...
星星们都在闪烁，像是在欢迎你...
你走下列车，脚踩在柔软的草地上...
星星们慢慢地靠近你，围绕着你，形成一个温暖的光圈...
你被星光包围着，被整个宇宙温柔地抱着...

【睡眠引导】(150 字)
躺在草地上，星星还在头顶闪烁...
列车轻轻地开走了，但你知道它明晚还会来接你...
眼皮沉了，呼吸慢了，星星也模糊了...
在星空的守护下，安心入睡...
晚安，世界... 晚安，旅人...
```

## 模板 6：治愈类 - 《修补星星的人》

```
【开场 - 呼吸引导】(150 字)
慢慢地吸气... 呼气...
今晚，我要带你去见一个特别的人...
他住在天上，做着一份特别的工作...

【场景引入】(200 字)
你沿着一条云梯慢慢地往上爬，越来越高...
终于，你来到一座小小的房子，门口挂着一个牌子：
"星星修补铺"
你推开门，一位白发老爷爷正在工作台上忙碌...
他抬起头，微笑着说："来啦？坐吧。"

【主体故事 - 观看与学习】(500 字)
工作台上摆满了各种各样的星星：
- 有的星星暗了，需要重新点亮
- 有的星星裂了，需要用月光胶水修补
- 有的星星蒙尘了，需要用云朵擦拭...

老爷爷拿起一颗暗了的星星，轻轻地吹了一口气...
星星开始发光，越来越亮...
他拿起一颗裂了的星星，涂上月光胶水，小心翼翼地粘合...
裂缝消失了，星星完好如初...
他递给你一颗蒙尘的星星："你来试试。"

你接过星星，用柔软的云朵轻轻地擦...
擦着擦着，星星开始发光，照亮了你的脸...
老爷爷点点头："做得很好。每一颗星星都值得被温柔对待。"

【放松融入】(200 字)
你继续擦着星星，动作越来越慢，越来越轻柔...
你的呼吸也变得轻柔，和擦星星的节奏一样...
身体随着动作，慢慢地放松...
手臂放松... 肩膀放松... 心也放松...
老爷爷在旁边轻轻地说："你也在被修补着，你知道吗？"

【温暖发现】(150 字)
你突然明白，今天所有的疲惫、所有的不开心...
都像星星上的灰尘，可以被轻轻地擦掉...
你还是你，只是需要一点温柔的照顾...
老爷爷递给你一颗小小的星星："这是给你的。"
"当你感到累的时候，看看它，记得有人在乎你。"

【睡眠引导】(150 字)
握着那颗星星，它发出温暖的光...
你沿着云梯慢慢地往下走，回到床上...
星星放在枕边，还在轻轻地发光...
眼皮沉了，呼吸慢了，光也柔和了...
在星星的守护下，安心入睡...
晚安，被爱着的人...
```

## 模板 7：四季类 - 《冬夜的小暖炉》

```
【开场 - 呼吸引导】(150 字)
外面下着雪，你待在温暖的房间里...
先做三次深呼吸... 感受室内的温暖...
呼气，把寒冷都呼出去...

【场景引入】(200 字)
房间里只有一个老式的暖炉，里面的火轻轻地跳动着...
你坐在暖炉前的地毯上，怀里抱着一杯热可可...
窗外，雪花一片片地落下，安静又美丽...
这是属于你的冬夜，整个世界都安静了...

【主体故事 - 冬夜的访客】(500 字)
突然，你听到轻轻的敲门声...
打开门，外面站着一只小刺猬，它冻得发抖...
"可以让我进来取暖吗？"它小声问...
你把它抱进来，放在暖炉边的地毯上...
小刺猬慢慢地舒展开，不再蜷缩...

接着，又来了一只小鸟，它的翅膀上沾着雪花...
你又让它进来，它站在椅子背上，轻轻地抖着羽毛...
然后是一只小狐狸，它不好意思地站在门口...
"进来吧，"你说，"这里很温暖。"

小暖炉前，现在有了三个小伙伴...
你们都不说话，只是静静地享受着温暖...
小刺猬发出满足的叹息，小鸟轻轻地唱歌，小狐狸闭着眼睛...

【放松融入】(200 字)
你捧着热可可，温暖从掌心传到全身...
身体陷进地毯里，越来越沉...
暖炉里的火发出噼啪声，像催眠曲...
你的呼吸变得缓慢，和小鸟的歌声一样轻柔...
肩膀放松，背放松，整个人都放松...

【温暖发现】(150 字)
你突然感到，温暖是可以分享的...
你给了它们温暖，它们也给了你陪伴...
这个冬夜，因为分享，变得特别美好...
小刺猬睡着了，发出轻轻的呼噜声...
小鸟把头埋进翅膀，也睡着了...
小狐狸靠在你脚边，温暖地依偎着...
```

## 使用指南

### 选择模板

1. **根据用户状态选择**：
   - 压力大 → 模板 1（月光引路）或 模板 5（星光列车）
   - 孤独 → 模板 2（书店）或 模板 7（冬夜暖炉）
   - 焦虑 → 模板 3（云鲸）或 模板 6（修补星星）
   - 怀念 → 模板 4（外婆阁楼）
   - 失眠 → 模板 8（深海灯笼鱼）

2. **根据历史记录去重**：
   - 检查 `memory/sleep-story-history.json`
   - 避免使用最近 2-3 篇用过的模板

3. **个性化调整**：
   - 加入用户提到的元素（喜欢的动物、颜色、场景）
   - 调整故事长度（用户偏好）
   - 调整感官重点（视觉/听觉/触觉）

### 创作要点

1. **不要照搬模板** - 模板是骨架，需要填充细节
2. **保持叙事弧线** - 确保有目标、进展、发现、收尾
3. **融入心理学技术** - 自然地加入呼吸、放松、感恩
4. **避免重复** - 检查历史记录，使用新鲜元素
