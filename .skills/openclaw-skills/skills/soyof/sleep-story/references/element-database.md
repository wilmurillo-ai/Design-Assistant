# 故事元素数据库

## 使用说明

创作故事前，从各类别中选择元素组合。遵循**去重规则**：
- 主场景：至少间隔 3 篇故事重复
- 角色：至少间隔 2 篇故事重复
- 旅程类型：至少间隔 2 篇故事重复
- 感官焦点：每篇切换
