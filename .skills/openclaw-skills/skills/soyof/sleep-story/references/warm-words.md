# 温暖词汇库

## 使用原则

1. **多感官组合** - 同时调动视觉、听觉、触觉、嗅觉
2. **避免刺激** - 不用鲜艳色彩、尖锐声音、强烈气味
3. **重复强化** - 关键词可以重复出现，强化印象
4. **个人化调整** - 根据用户偏好选择词汇
5. **渐进变化** - 从清晰具体→模糊抽象，配合意识状态

## 模糊词汇（意识过渡用）

**用于故事后期，帮助意识从清晰转向模糊**

| 类型 | 词汇 | 示例 |
|------|------|------|
| **时间模糊** | 渐渐地、慢慢地、不知不觉、不知不觉中 | "渐渐地，周围的一切都模糊了..." |
| **空间模糊** | 某处、某个地方、不远处、 somewhere | "好像在某处，有一个温暖的地方..." |
| **感知模糊** | 好像、似乎、仿佛、隐约、依稀 | "仿佛听到远处传来的声音..." |
| **程度模糊** | 有些、一点、些许、微微、稍稍 | "有些困倦了..." |
| **存在模糊** | 也许、可能、或许、像是 | "也许是梦开始了..." |

**使用原则**：
- 故事早期：使用清晰具体的词汇
- 故事中期：开始混入模糊词汇
- 故事后期：以模糊词汇为主，让意识自然飘散

## 词汇搭配建议

### 经典组合

| 组合 | 效果 | 示例 |
|------|------|------|
| **柔软 + 温暖** | 基础放松 | "柔软而温暖的被子" |
| **慢慢 + 轻轻** | 节奏放缓 | "慢慢地、轻轻地" |
| **朦胧 + 模糊** | 意识过渡 | "朦胧中，一切都模糊了" |
| **安心 + 安全** | 心理安抚 | "安心又安全的感觉" |
| **渐渐 + 沉睡** | 入睡引导 | "渐渐地进入沉睡" |

### 避免组合

| 避免 | 原因 |
|------|------|
| 明亮 + 兴奋 | 激活清醒状态 |
| 快速 + 突然 | 打破放松节奏 |
| 尖锐 + 刺耳 | 触发警觉反应 |
| 紧张 + 焦虑 | 负面情绪联想 |
