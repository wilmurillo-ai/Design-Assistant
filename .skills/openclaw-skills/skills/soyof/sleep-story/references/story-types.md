# 故事类型库（叙事驱动版）

## 1. 自然风景类 🌲

**核心意象**：森林、海洋、星空、山川、湖泊、草原

**适用场景**：
- 用户喜欢大自然
- 需要开阔心境
- 想要逃离城市喧嚣
- 感到被束缚、需要自由感

**典型元素**：
- 月光洒在湖面上
- 微风吹过草地
- 星星在夜空闪烁
- 远处传来海浪声
- 森林里的小径

**叙事模式**：
- **探索型**：沿着小径/河流/光线探索未知
- **相遇型**：在自然中遇见神秘的动物或存在
- **发现型**：找到一个隐藏的自然奇观

**放松技术**：可视化想象、正念呼吸

**故事目标示例**：
- 寻找月光尽头的小岛
- 跟随一只神秘的动物深入森林
- 发现一个只有你能看到的自然奇观

**示例开篇**：
> "想象你站在一片宁静的湖边，月光像银色的纱，轻轻地铺在水面上。湖中央有一座小岛，岛上有一棵发光的树。你找到一艘小船，慢慢地划向那里..."

---

## 2. 温馨日常类 🏠

**核心意象**：小屋、咖啡馆、图书馆、厨房、壁炉、阳台、书店

**适用场景**：
- 用户感到孤独
- 需要归属感
- 喜欢生活气息
- 渴望被接纳、被理解

**典型元素**：
- 暖黄色的灯光
- 热饮的香气
- 柔软的沙发和毛毯
- 窗外的雨声
- 书页翻动的声音

**叙事模式**：
- **庇护型**：找到一个只属于你的安全空间
- **任务型**：完成一件简单温暖的小事（找书、煮茶、整理）
- **陪伴型**：与某个温暖的存在共度时光

**放松技术**：安全基地构建、感恩回忆

**故事目标示例**：
- 在深夜书店找到一本专属之书
- 在咖啡馆里完成一次温暖的对话
- 在小屋里布置一个舒适的角落

**示例开篇**：
> "街角有一家咖啡馆，只有你还醒着。推开门，暖黄色的灯光迎接你。店主是一只慵懒的猫，它点点头，继续打盹。你想找一个靠窗的位置坐下..."

## 3. 动物伙伴类 🐱

**核心意象**：小猫、兔子、鲸鱼、鸟类、狐狸、鹿、刺猬

**适用场景**：
- 用户喜欢动物
- 需要陪伴感
- 想要被无条件接纳
- 感到孤独、需要情感连接

**典型元素**：
- 毛茸茸的触感
- 动物温暖的体温
- 轻轻的呼噜声
- 信任的眼神
- 一起度过的安静时光

**叙事模式**：
- **帮助型**：帮助一只遇到困难的小动物
- **陪伴型**：与动物一起完成某件事
- **引导型**：动物带你去一个特别的地方

**放松技术**：安全基地构建、触觉想象

**故事目标示例**：
- 送迷路的小动物回家
- 照顾一只受伤的小动物
- 跟随动物探索一个秘密基地

**示例开篇**：
> "一只柔软的小猫轻轻地蹭着你的脚踝，它抬起头，用信任的眼神看着你。它咬住你的衣角，仿佛在说：'跟我来，我带你去一个特别的地方。'你跟着它，走进夜色中..."

## 4. 童年回忆类 🧸

**核心意象**：外婆家、老街道、学校、游戏、秘密基地、老树、阁楼

**适用场景**：
- 用户怀念过去
- 需要安全感
- 想要回到简单时光
- 感到疲惫、渴望被照顾

**典型元素**：
- 熟悉的气味（饭菜香、阳光味）
- 老物件的触感
- 童年的声音（蝉鸣、摇椅声）
- 被保护的感觉
- 无忧无虑的时光

**叙事模式**：
- **回归型**：回到一个熟悉的童年场景
- **发现型**：在旧物中发现被遗忘的温暖
- **重逢型**：与记忆中的人或物重逢

**放松技术**：感恩回忆、安全基地构建

**故事目标示例**：
- 在外婆的阁楼发现一个旧箱子
- 回到老街道，发现一切都还在
- 与记忆中的亲人共度一个下午

**示例开篇**：
> "你回到了小时候的外婆家，推开木门，熟悉的饭菜香扑面而来。外婆从厨房探出头来，笑着说：'回来啦？'院子里的摇椅还在轻轻地摇晃，像是在等你..."

## 5. 奇幻治愈类 ✨

**核心意象**：星星、月亮、云朵、光、魔法、梦境

**适用场景**：
- 用户需要希望感
- 喜欢想象力丰富的故事
- 想要逃离现实
- 感到迷茫、需要方向

**典型元素**：
- 会发光的物体
- 有魔法的存在
- 超现实的场景
- 温柔的奇迹

**叙事模式**：
- **任务型**：完成一个温柔的魔法任务
- **学习 型**：从一个智慧存在那里学习
- **转变型**：经历一个温柔的内在转变

**放松技术**：可视化想象、积极暗示

**故事目标示例**：
- 帮助修补一颗暗了的星星
- 学习如何点亮自己的光
- 在梦境中找到一个答案

**示例开篇**：
> "你沿着一条云梯慢慢地往上爬，来到一座小小的房子，门口挂着'星星修补铺'的牌子。一位白发老爷爷正在工作，他抬起头：'来啦？坐吧。'"

## 6. 季节限定类 🍂

**核心意象**：春樱、夏夜、秋月、冬雪

**适用场景**：
- 用户注意到季节变化
- 想要应景的故事
- 喜欢自然节律

**典型元素**：
- 春天的花开、新芽
- 夏天的星空、萤火虫
- 秋天的落叶、丰收
- 冬天的雪、暖炉

**叙事模式**：
- **体验型**：体验一个季节特有的美好
- **过渡型**：从一个季节过渡到另一个
- **庆祝型**：庆祝季节带来的礼物

**放松技术**：正念觉察、感恩练习

**故事目标示例**：
- 在冬夜里与小动物分享温暖
- 在夏夜追逐萤火虫
- 在秋天收集落叶做纪念品

**示例开篇**：
> "外面下着雪，你待在温暖的房间里。暖炉里的火轻轻地跳动着，你怀里抱着一杯热可可。突然，你听到轻轻的敲门声..."

## 混合类型

可以根据用户需求混合多种类型，例如：

| 混合类型 | 示例 | 适用场景 |
|----------|------|----------|
| **自然 + 动物** | 森林里的小动物之家 | 喜欢自然又需要陪伴 |
| **日常 + 童年** | 小时候的温馨厨房 | 怀念家庭温暖 |
| **动物 + 奇幻** | 会魔法的小猫 | 喜欢动物又爱想象 |
| **自然 + 季节** | 冬夜的森林小屋 | 应景的自然故事 |
| **童年 + 奇幻** | 外婆的魔法阁楼 | 怀念中带点神奇 |

## 选择建议

| 用户状态 | 推荐类型 | 叙事模式 |
|----------|----------|----------|
| 工作压力大 | 自然风景类 | 探索型（开阔心境） |
| 感到孤独 | 温馨日常类 / 动物伙伴类 | 陪伴型 |
| 焦虑不安 | 温馨日常类 | 庇护型（安全基地） |
| 怀念过去 | 童年回忆类 | 回归型 |
| 需要希望 | 奇幻治愈类 | 转变型 |
| 失眠严重 | 动物伙伴类 | 引导型（跟随动物） |
| 无法选择 | 温馨日常类 | 任务型（最通用） |

## 故事性检查清单

创作完成后，检查故事是否有足够的叙事性：

- [ ] **有角色** - 用户或某个动物/存在作为主角
- [ ] **有目标** - 一个轻柔的方向（寻找、探索、帮助、回家）
- [ ] **有进展** - 故事在向前推进，不是静态描述
- [ ] **有发现** - 一个温暖的转折或惊喜
- [ ] **有情感** - 能引发某种温暖的情感（安心、被爱、平静）
- [ ] **有收尾** - 自然地过渡到睡眠

如果以上有任何一项不满足，考虑增加叙事元素。
