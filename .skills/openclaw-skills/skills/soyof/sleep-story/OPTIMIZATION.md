# Sleep Story 技能优化说明

## 📋 优化概览

本次优化将 sleep-story 技能从"温暖治愈系故事创作"升级为"科学助眠催眠故事系统"，主要改进包括：

---

## ✨ 核心升级

### 1. 催眠技术体系

**新增技术**：
- **催眠呼吸诱导** - 将呼吸与故事元素（海浪、秋千、云鲸）绑定
- **意识模糊化技术** - 通过模糊语言减少认知加工
- **催眠暗示语库** - 直接/间接睡眠暗示、深化暗示
- **双重束缚技术** - 绕过意识抵抗的引导技巧

**效果**：从"放松"升级到"主动诱导睡眠"

---

### 2. 禁忌内容清单

**新增严格禁止的内容**：
| 类型 | 具体内容 |
|------|----------|
| 刺激情节 | 追逐、打斗、危险、紧张、冲突 |
| 负面情绪 | 恐惧、愤怒、悲伤、绝望、焦虑 |
| 突然转折 | 惊吓、意外、反转 |
| 复杂逻辑 | 需要思考、推理、记忆 |
| 兴奋色彩 | 红色、橙色、亮黄色 |
| 时间压力 | 赶时间、迟到、截止日期 |
| 社交压力 | 被评价、被审视、表现焦虑 |
| 电子设备 | 手机、电脑、屏幕 |
| 工作学习 | 办公、考试、任务 |

**效果**：严格杜绝不利于睡眠的要素

---

### 3. 催眠递进结构

**故事结构从 4 段升级为 5 段**：

```
【第一阶段】注意力聚焦 (150-200 字)
  → 呼吸引导 + 感官聚焦
  
【第二阶段】身心放松 (400-500 字)
  → 渐进式肌肉放松 + 呼吸绑定
  
【第三阶段】情绪安抚 (200-250 字)
  → 安全基地 + 感恩回忆 + 温暖发现
  
【第四阶段】意识模糊 (150-200 字)
  → 模糊语言 + 间接暗示
  
【第五阶段】睡眠诱导 (250-300 字)
  → 直接暗示 + 睡眠锚定
```

**效果**：从"讲故事"升级为"催眠诱导流程"

---

### 4. 睡眠科学原理

**新增科学依据**：
- **副交感神经激活** - 通过 4-7-8 呼吸法
- **脑波转变** - β→α→θ→δ的故事节奏设计
- **注意力窄化** - 减少杂念侵入
- **条件反射建立** - 睡眠锚点重复强化

**效果**：从"经验主义"升级到"科学验证"

---

### 5. 词汇库扩展

**新增词汇分类**：
- **模糊词汇** - 渐渐地、好像、仿佛、隐约（用于意识过渡）
- **睡眠暗示词汇** - 眼皮沉、呼吸慢、安心睡（直接诱导）
- **催眠短语库** - 可直接使用的引导短语

**效果**：从"温暖词汇"升级到"催眠词库"

## 🎯 使用建议

### 最佳实践

1. **创作前**
   - 读取 `memory/sleep-story-history.json` 检查去重
   - 询问用户偏好（故事类型、心理学技术）
   - 选择合适的模板

2. **创作中**
   - 遵循 5 段催眠递进结构
   - 自然融入心理学技术
   - 使用温暖词汇库
   - 严格避免禁忌内容

3. **创作后**
   - 使用检查清单逐项核对
   - 更新历史记录文件
   - 记录用户反馈

### 预计效果

| 用户状态 | 预计入睡时间 |
|----------|-------------|
| 轻度失眠 | 5-10 分钟 |
| 中度焦虑 | 10-15 分钟 |
| 严重失眠 | 15-30 分钟 |
