# 抖音爆款拆解分析技能

**一句话介绍：** 发抖音链接，自动分析视频为何火爆、触达人群、行为心理。

---

## 功能

- ✅ 自动解析抖音分享链接
- ✅ 提取视频元数据（标题、作者、数据指标）
- ✅ **下载视频并提取帧，真正分析画面内容**
- ✅ 用 image 工具分析画面构图、色调、类型
- ✅ 六步分析框架（基础信息→数据表现→火爆原因→人群画像→行为心理→可复用规律）

---

## 使用方式

直接发送抖音分享链接给机器人：

```
https://v.douyin.com/xxx
```

或：

```
https://www.douyin.com/video/xxx
```

机器人会自动：
1. 解析链接，提取视频信息
2. 下载视频，提取关键帧
3. 分析画面内容
4. 输出完整分析报告

---

## 分析框架

### 六步分析法

| 步骤 | 内容 |
|------|------|
| 1. 基础信息 | 视频/作者/时长/话题/数据 |
| 2. 数据表现 | 点赞/评论/转发/收藏/互动率 + 行业对比 |
| 3. 火爆原因 | 选题/开头钩子/情绪触发 |
| 4. 人群画像 | 核心人群/痛点/占比 |
| 5. 行为心理 | 为什么点赞/评论/转发/收藏 |
| 6. 可复用规律 | 选题/开头/情绪公式 |

### 关键指标

| 指标 | 低 | 中 | 高 | 极高 |
|------|----|----|-----|------|
| 评论率 | <3% | 3-5% | 5-10% | >10% |
| 转发率 | <5% | 5-8% | 8-15% | >15% |
| 互动率 | <10% | 10-20% | 20-30% | >30% |
| 收藏率 | <15% | 15-30% | 30-50% | >50% |

## 技术实现

### 画面分析流程

```
抖音链接 → 提取视频ID → 请求分享页 → 解析_ROUTER_DATA → 
获取视频URL → 下载视频 → ffmpeg提取帧 → image工具分析
```

### 视频数据获取

1. 请求抖音分享页面
2. 从 HTML 中提取 `_ROUTER_DATA` JSON
3. 解析获取：标题、作者、数据、视频URL

### 帧提取

```bash
ffmpeg -i video.mp4 -ss 00:00:05 -vframes 1 frame.jpg
```

### 画面分析

使用 image 工具分析帧图片：
- 画面主体
- 构图特点
- 色调氛围
- 内容类型（实拍/CGI/动画/录屏）
