## 使用方法

### 列出所有基准测试
```bash
python scripts/benchmark_db.py --action list
```

### 添加新的基准测试
```bash
python scripts/benchmark_db.py \
  --action add \
  --category tool-type \
  --test-name "新测试" \
  --input "测试输入" \
  --expected-output "预期输出"
```

### 与基准对比
```bash
python scripts/benchmark_db.py \
  --action compare \
  --skill-path /path/to/skill \
  --category tool-type
```

### 获取排行榜
```bash
python scripts/benchmark_db.py \
  --action leaderboard \
  --category tool-type \
  --limit 10
```

## 自定义基准

### 添加领域特定基准

```yaml
# 添加到 references/test-cases.yaml
- name: "领域特定测试"
  category: "your-category"
  input: "领域特定输入"
  expected: "领域特定预期"
  metrics:
    accuracy: 0.90
    cost: 0.50
```

### 导入外部基准

```bash
python scripts/benchmark_db.py \
  --action import \
  --from-file external-benchmarks.yaml \
  --category your-category
```
