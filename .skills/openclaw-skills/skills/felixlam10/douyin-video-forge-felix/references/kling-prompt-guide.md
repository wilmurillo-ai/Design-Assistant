# Kling 3.0 (可灵) Prompt 编写指南

> 本文档供 LLM Agent 在生成 Kling API 视频 prompt 时参考。所有建议基于 Kling 3.0 模型特性。

---

## 1. Prompt 结构

### 1.1 推荐组织结构

按以下顺序组织 prompt，层次清晰，模型理解效果最佳：

```
主体 → 动作 → 场景 → 氛围/光线 → 镜头语言 → 画质要求
```

**示例：**
```
一位穿着白色连衣裙的年轻女性，在樱花树下缓缓转身微笑，
春日午后的公园，阳光透过花瓣洒下斑驳光影，
温暖柔和的自然光，浅景深，电影质感，
镜头缓慢推近至面部特写，4K高清画质
```

### 1.2 中文 vs 英文 prompt

| 维度 | 中文 prompt | 英文 prompt |
|------|------------|------------|
| 人物理解 | 对中国人面部特征、服饰描述更精准 | 适合欧美面孔、西式场景 |
| 场景描述 | 中式建筑、传统文化场景效果更好 | 现代都市、科幻场景略有优势 |
| 镜头语言 | 支持良好，但部分专业术语识别不如英文 | 镜头运动指令识别更稳定 |
| 推荐策略 | **抖音短视频场景首选中文** | 需要特定风格时可混合使用 |

### 1.3 Prompt 长度建议

- **推荐长度**：100-200 字中文（或 50-100 英文单词）
- **过短**（<50字）：画面细节不足，模型自由发挥空间大，结果不可控
- **过长**（>300字）：模型可能忽略后半段描述，关键信息被稀释
- **核心原则**：每个关键要素用一句话描述，避免重复和冗余修饰

### 1.4 负面 prompt（negative_prompt）

适用场景：
- 避免人物畸形：`模糊, 变形, 多余手指, 扭曲面部, 低质量`
- 避免画面瑕疵：`噪点, 水印, 文字, 黑边, 过曝, 欠曝`
- 避免风格偏差：`卡通, 动画, 素描, 油画`（当需要真实感时）

**常用负面 prompt 模板：**
```
模糊, 变形, 畸形, 低质量, 噪点, 水印, 过曝, 抖动, 多余肢体
```

## 2. 画面关键词库

### 2.1 人物描述词

| 类别 | 关键词示例 |
|------|-----------|
| 年龄 | 婴儿、幼童、少年、青年女性、中年男性、老年人、20岁左右 |
| 性别/外貌 | 长发女性、短发男性、精致五官、清秀面容、轮廓分明 |
| 表情 | 微笑、大笑、沉思、惊讶、专注、平静、自信 |
| 服饰 | 白色T恤、西装革履、汉服、职业装、运动服、休闲穿搭 |
| 动作 | 行走、奔跑、转身、点头、挥手、品尝美食、翻阅书籍 |

### 2.2 场景描述词

| 类别 | 关键词示例 |
|------|-----------|
| 室内 | 咖啡厅、办公室、卧室、厨房、书房、健身房、直播间 |
| 室外 | 城市街道、海边沙滩、山顶、草地、古镇小巷、高楼天台 |
| 时间 | 清晨、正午、黄昏、傍晚、深夜、日出时分、金色夕阳 |
| 天气 | 晴天、阴天、小雨、大雪、雾气弥漫、彩虹 |
| 光线 | 自然光、逆光、侧光、顶光、霓虹灯光、烛光、柔和灯光 |

### 2.3 画面风格词

- **真实感**：真实, 写实, 纪实风格, 自然真实
- **电影感**：电影质感, 胶片感, 宽银幕, 35mm镜头, 浅景深, 电影调色
- **广告质感**：商业广告质感, 高端, 精致, 品牌感, 杂志封面风格
- **色调控制**：暖色调, 冷色调, 高对比度, 低饱和度, 莫兰迪色系, 赛博朋克色调
- **特殊风格**：延时摄影, 慢动作, 航拍视角, 微距摄影

### 2.4 画面质量词

```
4K, 高清, 超高清, 专业摄影, 精细画质, 高分辨率, 细节丰富, 画面清晰锐利
```

> 提示：质量词放在 prompt 末尾，不宜堆砌过多，选 2-3 个即可。

## 3. 镜头运动指令

### 3.1 支持的运动类型

| 运动类型 | 中文描述 | 适用场景 |
|---------|---------|---------|
| 推镜（Zoom In） | 镜头缓慢推近 | 强调细节、人物特写、产品展示 |
| 拉镜（Zoom Out） | 镜头缓慢拉远 | 揭示全景、场景转换、结尾收束 |
| 平移（Pan） | 镜头水平横移 | 展示宽阔场景、多人物、产品陈列 |
| 跟随（Follow） | 镜头跟随主体移动 | 人物行走、车辆行驶、运动场景 |
| 环绕（Orbit） | 镜头围绕主体旋转 | 产品360度展示、人物全方位展现 |
| 俯拍（Tilt Down） | 镜头从高处向下 | 俯瞰全景、食物展示、桌面场景 |
| 仰拍（Tilt Up） | 镜头从低处向上 | 建筑仰望、人物气势、天空过渡 |
| 固定机位（Static） | 镜头不动 | 对话场景、静物展示、稳定叙事 |
| 手持晃动（Handheld） | 轻微手持晃动 | 纪实感、Vlog风格、增加真实感 |

### 3.2 运动幅度控制

在 prompt 中通过修饰词控制运动速度和幅度：

- **缓慢**：`镜头缓慢推近` — 适合情绪渲染、产品展示
- **匀速**：`镜头平稳横移` — 适合场景扫描、平稳叙事
- **快速**：`镜头快速拉远` — 适合转场、冲击感
- **微幅**：`镜头轻微晃动` — 适合增加真实感

> 注意：单个 5 秒片段建议只用 1 种镜头运动，避免多种运动叠加导致画面混乱。

## 4. 角色一致性（kling_elements 参数）

### 4.1 参考图片要求

- **分辨率**：建议不低于 512x512，最佳 1024x1024
- **角度**：正面或3/4侧面照，五官清晰可见，无遮挡
- **数量**：支持 2-50 张参考图，建议 5-10 张覆盖不同角度和表情
- **背景**：纯色或简洁背景效果更好，避免复杂环境干扰

### 4.2 最佳实践

1. **多段落角色一致**：在视频脚本的不同段落中使用同一组 `kling_elements` 参考图，确保同一角色外观不变
2. **参考图多样性**：包含正面、侧面、全身、半身等多角度照片，帮助模型建立完整的角色模型
3. **避免极端造型变化**：同一角色在不同段落中避免大幅更换发型、服饰，模型难以同时兼顾一致性和造型变化

### 4.3 配合 prompt 的技巧

- prompt 中对角色的描述应与参考图保持一致，不要与参考图特征矛盾
- 可用 prompt 指定参考图未体现的信息（如动作、表情变化）
- 示例：参考图是黑长发女性 → prompt 中写"一位黑色长发的年轻女性在微笑"，不要写"金色短发"

## 5. 音频生成

### 5.1 motion_has_audio 参数

- 设为 `true` 时，模型会根据画面内容自动生成匹配的环境音效
- **效果**：脚步声、风声、水声、城市噪音等环境音较自然
- **局限**：人声/对话质量不稳定，不建议依赖此功能生成对话

### 5.2 中文语音注意事项

- 模型生成的中文语音可能存在发音不准、语调不自然的问题
- **推荐方案**：使用独立 TTS 服务生成旁白/对话，再与视频合成
- 环境音可用原生 audio，人声建议后期叠加

### 5.3 BGM 与原生音频配合

- 原生音频主要用于环境音效，不适合作为 BGM
- **推荐工作流**：
  1. 视频生成时开启 `motion_has_audio: true` 获取环境音
  2. 后期叠加 BGM 音轨，环境音降低至 20-30% 音量
  3. 如需纯净 BGM，则关闭原生音频，完全使用外部音轨

## 6. 常见问题与解决方案

### 6.1 人物变形/畸形

| 问题 | 解决方案 |
|------|---------|
| 多余手指/肢体 | 负面 prompt 加入"多余手指, 多余肢体, 畸形"；避免手部特写 |
| 面部扭曲 | 使用参考图(kling_elements)锚定面部特征；避免极端角度 |
| 身体比例失调 | prompt 中明确描述人物体态，如"身材匀称的成年女性" |
| 多人场景混乱 | 控制画面人数不超过 3 人；明确每个人物的位置关系 |

### 6.2 画面抖动/闪烁

- **使用固定机位**：`固定镜头` 或 `三脚架拍摄` 减少不必要的运动
- **避免过快运动**：复杂场景中避免"快速"修饰词
- **场景简化**：减少画面中的运动元素数量，聚焦主体动作
- **负面 prompt**：加入 `画面抖动, 闪烁, 不稳定`

### 6.3 文字渲染局限

- Kling 3.0 **不擅长**生成精确的中文/英文文字
- 避免在 prompt 中要求画面出现具体文字内容
- **替代方案**：后期使用视频编辑工具叠加文字图层
- 如必须包含文字，使用简短英文单词（1-3个字母）成功率较高

### 6.4 复杂动作的分步描述策略

单个 5 秒片段难以承载复杂连续动作，推荐拆分：

**错误示范**（一个片段中描述过多动作）：
```
一个人从椅子上站起来，走到窗边，打开窗户，深呼吸，然后转身坐回去
```

**正确做法**（拆分为多个片段）：
```
片段1：一位男性从椅子上缓缓站起，目光看向窗户方向
片段2：男性走到窗前，伸手推开窗户，微风吹动发丝
片段3：男性站在窗边深呼吸，表情放松，阳光照在脸上
```

**原则：**
- 每个 5 秒片段只包含 1-2 个核心动作
- 片段之间通过相似的结束/开始姿态保持连贯性
- 配合 kling_elements 确保人物一致性

## 附：抖音短视频 Prompt 模板

### 产品展示类
```
[产品名称]特写，放置在[场景]上，[光线描述]，
镜头缓慢环绕展示产品细节，商业广告质感，4K高清
```

### 人物 Vlog 类
```
一位[年龄性别描述]的[外貌特征]，在[场景]中[动作描述]，
[时间/光线]，自然真实的纪实风格，镜头[运动描述]，高清画质
```

### 美食类
```
[食物名称]特写，[烹饪状态描述]，热气腾腾，
暖色调，微距摄影，镜头缓慢推近，食物广告质感，细节丰富
```

### 风景/旅行类
```
[地点]的[场景描述]，[时间]，[天气/光线]，
壮丽自然风光，镜头缓慢平移/航拍视角，电影质感，超高清
```
