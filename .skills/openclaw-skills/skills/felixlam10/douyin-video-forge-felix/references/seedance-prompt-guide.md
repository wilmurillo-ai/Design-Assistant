# Seedance 2.0 Prompt 编写指南

> 本文档供 LLM Agent 在生成 Seedance 优化版脚本时参考。

---

## 一、Seedance 与可灵（Kling）的核心差异

### 1.1 使用方式差异

| 维度 | 可灵 (Kling) | Seedance 2.0 |
|------|-------------|--------------|
| 调用方式 | API 接口调用 | 手动输入（网页/客户端） |
| 脚本要求 | 结构化 JSON 字段 | 便于人工复制粘贴的纯文本 |
| 输出格式 | 程序直接消费 | 需要人一眼看懂、直接粘贴 |

**关键影响**：生成的 Seedance prompt 必须是可直接复制的完整段落，不要包含 JSON 结构、字段名、或需要人工拼装的碎片。每个镜头的 prompt 独立成段，段与段之间用空行分隔即可。

### 1.2 画面风格偏好

- **可灵**：擅长电影感画面、精确的镜头运动控制、风格化视觉效果。
- **Seedance**：擅长自然真实的画面质感、人物动作连贯性、日常场景还原。Seedance 在人物面部表情和肢体动作方面表现更优。

### 1.3 输入格式差异

- **可灵**：接受关键词堆叠 + 参数式描述（如 `camera: dolly in, style: cinematic`）。
- **Seedance**：偏好**自然语言叙事**，用完整句子描述你想看到的画面。关键词堆砌反而会降低生成质量。

### 1.4 各自擅长的视频类型

- **可灵更强**：特效类、风格化转场、精确镜头运动、抽象艺术风格。
- **Seedance 更强**：产品展示（真实质感）、人物表演、日常生活场景、动作连贯的叙事片段。

## 二、自然语言描述最佳实践

### 2.1 用完整句子，不要关键词堆砌

```
-- 不推荐（关键词式）：
女性，25岁，咖啡馆，微笑，暖色调，特写，柔光

-- 推荐（自然语言式）：
一位年轻女性坐在咖啡馆靠窗的位置，阳光透过玻璃洒在她脸上，她端起咖啡杯轻轻抿了一口，露出满足的微笑。
```

### 2.2 按时间顺序描述动作

动作描述应遵循"先发生 -> 后发生"的时间线，让模型理解动作的因果关系：

```
她缓缓打开礼盒的盖子，看到里面的项链后眼睛一亮，小心翼翼地拿起来放在锁骨前比了比，转头看向镜子。
```

### 2.3 情感和氛围用具体细节传达

不要写抽象形容词，用可视化的细节暗示情绪：

```
-- 不推荐：氛围温馨浪漫
-- 推荐：桌上的蜡烛微微摇晃，暖黄色的光映在两个人的脸上，窗外的雨点沿着玻璃缓缓滑下。
```

### 2.4 避免过于技术性的摄影术语

Seedance 不像可灵那样精确响应专业摄影指令。用日常描述替代：

| 摄影术语 | Seedance 友好描述 |
|---------|-----------------|
| dolly in | 镜头缓缓推近 |
| crane shot | 镜头从高处缓缓下降 |
| rack focus | 焦点从前景转移到背景 |
| dutch angle | 画面略微倾斜 |
| bokeh | 背景模糊柔和，突出前景主体 |
| golden hour | 傍晚柔和的暖色阳光 |

## 三、Prompt 结构建议

每段 prompt 遵循三段式：**场景设定 -> 动作描述 -> 情感氛围**。

### 3.1 结构模板

```
[场景设定] 在哪里、谁、什么时间/光线条件。
[动作描述] 发生了什么、人物怎么动、物体怎么变化。
[情感氛围] 整体给人什么感觉，用 1-2 个具体视觉细节收尾。
```

### 3.2 示例：30 秒产品展示（护肤品）

**镜头 1（0-8秒）：**
清晨的浴室里，柔和的自然光从窗户照进来。一瓶精华液放在大理石台面上，瓶身上的水珠在光线下闪烁。一只手缓缓拿起瓶子，按压出透明的精华液滴在指尖。

**镜头 2（8-18秒）：**
一位女性对着镜子，将精华液轻轻拍在脸颊上，指尖沿着脸部轮廓向上提拉。镜头靠近她的面部，可以看到皮肤逐渐呈现出光泽感。她的表情放松而愉悦。

**镜头 3（18-30秒）：**
她站在窗边，侧脸沐浴在阳光中，皮肤看起来通透有光泽。镜头缓缓拉远，她转过头来对着镜头微微一笑。产品瓶身出现在画面前景，背景柔和虚化。

### 3.3 示例：15 秒场景演绎（咖啡品牌）

**镜头 1（0-7秒）：**
雨天的街角咖啡馆，玻璃窗上布满雨滴。一位穿着米色风衣的年轻人推门走进来，带着一身微微的雨水气息。他在吧台前坐下，搓了搓有些冷的手。

**镜头 2（7-15秒）：**
咖啡师将一杯刚做好的拿铁推到他面前，热气缓缓升腾。他双手捧起杯子，靠近闻了闻咖啡的香气，然后喝了一口，整个人的肩膀明显放松下来。窗外的雨还在下，但店内温暖而安静。

## 四、从可灵 Prompt 转换为 Seedance Prompt

### 4.1 关键词式 -> 叙事式

**转换原则**：将并列的关键词扩展为有逻辑关系的句子。

```
可灵原文：
young woman, silk dress, garden, golden hour, walking slowly,
flowers blooming, gentle breeze, hair flowing, soft smile

Seedance 转换后：
傍晚时分，一位穿着丝质裙子的年轻女性在花园小径上缓步走着。
夕阳的暖光照在她身上，微风吹起她的头发和裙角。
路两旁的花朵正在盛开，她低头看着花丛，嘴角带着淡淡的笑意。
```

### 4.2 镜头术语 -> 自然描述

将可灵 prompt 中的摄影指令转化为画面描述：

```
可灵：camera slowly dollies in from medium shot to close-up on the face
Seedance：镜头从远处缓缓靠近，最终停留在她的面部特写。

可灵：low angle shot, dramatic lighting, lens flare
Seedance：从略低的角度看过去，光线从人物身后照过来，边缘泛着一层光晕。

可灵：tracking shot following subject from left to right
Seedance：镜头跟随她从左向右移动，保持在她的侧面。
```

### 4.3 保留核心创意元素

转换时必须保留的要素：
- **主体**：人物/产品的外观、姿态、状态不变。
- **动作**：核心动作和动作顺序不变。
- **情绪基调**：温暖/冷静/紧张等情绪方向不变。
- **光线条件**：用自然描述替代术语，但明暗关系保持一致。

可以适当省略的：
- 精确的镜头运动参数（Seedance 无法精确控制）。
- 后期特效描述（如 color grading 细节）。
- 多余的风格标签（cinematic, editorial 等）。

## 五、注意事项

### 5.1 Prompt 长度建议

- 单段 prompt 推荐 **50-150 字**（中文字符）。
- 低于 30 字：信息不足，生成结果随机性大。
- 超过 200 字：模型可能忽略后半部分描述，重点被稀释。
- 最核心的画面元素放在前 2 句话中。

### 5.2 分段生成的衔接

Seedance 每次生成一个片段，片段之间的衔接需注意：

- **保持主体一致性**：每段 prompt 都要重新描述主体外观（发型、服装颜色等），不能假设模型记得上一段。
- **场景过渡**：相邻片段的环境/光线应有逻辑延续，避免突变。
- **动作承接**：下一段的起始姿态应与上一段的结束姿态一致。例如上一段以"她转身面向窗户"结尾，下一段应以"她面向窗户站着"开头。
- **情绪连贯**：避免相邻片段的情绪基调剧烈跳跃。

### 5.3 生成后的常见调整方向

生成结果不理想时，按以下方向调整 prompt：

| 问题 | 调整方法 |
|------|---------|
| 动作不够明显 | 加强动作幅度描述，使用更具体的动词 |
| 画面太静/太空 | 增加环境动态细节（风吹、光影变化） |
| 人物表情不对 | 用具体的面部动作替代情绪词（"眉头微皱"而非"忧虑"） |
| 光线不符预期 | 明确描述光源方向和强度 |
| 产品不够突出 | 强调产品与环境的对比，描述产品的材质和反光 |
| 画面风格偏差 | 增加色调描述（"整体画面偏暖的米黄色调"） |

## 六、Agent 生成 Seedance Prompt 的检查清单

生成 Seedance 版 prompt 后，逐项确认：

- [ ] 是纯文本段落，可直接复制粘贴
- [ ] 使用自然语言叙事，无关键词堆砌
- [ ] 无专业摄影术语（已转换为自然描述）
- [ ] 每段在 50-150 字之间
- [ ] 动作按时间顺序排列
- [ ] 包含场景设定、动作描述、氛围细节三要素
- [ ] 相邻片段有主体描述重复以保持一致性
- [ ] 核心创意元素（动作、情绪、产品特征）从可灵版本完整保留
