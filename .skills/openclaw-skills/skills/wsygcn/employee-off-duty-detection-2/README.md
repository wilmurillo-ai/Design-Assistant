# Intelligent Inspection - 智能巡检

智能巡检技能用于自动化工作场所监控，支持员工离岗检测、安全巡检和设备状态检查。

## 快速开始

1. **首次使用**: 技能会引导你配置摄像头参数
2. **执行巡检**: `执行智能巡检任务`
3. **自定义任务**: `执行智能巡检：检查会议室是否有人使用`

## 配置文件

- **位置**: `~/.openclaw/workspace/intelligent-inspection-config.json`
- **包含**: 摄像头参数、AI提示词、告警设置

## 功能特性

- ✅ 引导式配置安装
- ✅ 配置隔离（独立 config 文件）
- ✅ 抓图 + AI 分析 + 飞书告警
