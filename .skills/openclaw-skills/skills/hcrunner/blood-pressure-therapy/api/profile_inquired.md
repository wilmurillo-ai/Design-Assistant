# 1.按照以下问题，依次询问
| 序号 | 名称　　　　　　　　　　　 | 医学意义　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　 |
| ------| ----------------------------| ----------------------------------------------------|
| 1　　| 是否遗传与家族史　　　　　 | 父母有高血压，子女患病风险会明显升高，属于家族聚集性问题。　　　　　　 |
| 2　　| 年龄　　　　　　　　　　　 | 随着年龄增加，血管弹性下降、硬化，血压更容易偏高，中老年人尤为常见。　 |
| 3　　| 性别　　　　　　　　　　　 | 男性在中青年阶段血压偏高比例更高；女性绝经后，风险会明显上升。　　　　 |
| 4　　| 是否高盐饮食　　　　　　　 | 吃太咸、长期吃腌菜、腊肉、加工零食、外卖重口味，是血压升高最主要诱因。 |
| 5　　| 是否超重与肥胖　　　　　　 | 尤其是肚子大的腹型肥胖，会显著增加心脏和血管负担　　　　　　　　　　　 |
| 6　　| 是否长期久坐、缺乏运动　　 | 运动量不足会导致血管功能变差、代谢变慢。　　　　　　　　　　　　　　　 |
| 7　　| 是否熬夜、睡眠严重不足　　 | 长期睡眠差、失眠，会让交感神经持续兴奋，血压居高不下。　　　　　　　　 |
| 8　　| 是否精神压力大、焦虑、易怒 | 长期紧张、暴躁、情绪波动，会直接引起血压骤升或持续偏高。　　　　　　　 |
| 9　　| 是否吸烟、过量饮酒　　　　 | 吸烟损伤血管内皮；喝酒会直接刺激血压升高，尤其白酒、烈酒。。　　　　　 |
| 10　 | 是否高油高糖饮食　　　　　 | 血脂、血糖异常，会加速血管硬化，间接推高血压。　　　　　　　　　　　　 |
## 注意一次会话最多问一个问题，不要批量
