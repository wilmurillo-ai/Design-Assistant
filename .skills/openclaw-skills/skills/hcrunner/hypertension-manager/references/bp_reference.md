# 血压参考标准

## 成人血压分类标准（中国高血压防治指南 2024修订版）

| 分类 | 收缩压 (mmHg) | 舒张压 (mmHg) |
|------|:---:|:---:|
| 理想血压 | < 120 | < 80 |
| 正常血压 | 120-129 | 80-84 |
| 正常高值 | 130-139 | 85-89 |
| 1级高血压（轻度） | 140-159 | 90-99 |
| 2级高血压（中度） | 160-179 | 100-109 |
| 3级高血压（重度） | ≥ 180 | ≥ 110 |

> 当收缩压和舒张压分属不同级别时，以较高的级别为准。

## 特殊人群参考

### 60岁以上老年人
- 目标血压：< 150/90 mmHg（如耐受可降至 < 140/90）
- 正常高值上限可适当放宽至 150/95

### 已服药患者
- 评估时应考虑药物效果
- 如果服药后仍 ≥ 140/90，提示可能需要调整用药

### 糖尿病/肾病患者
- 目标血压更严格：< 130/80 mmHg
- 需要更频繁的监测

## 血压测量注意事项

1. 测量前30分钟不吸烟、不饮咖啡、不运动
2. 静坐休息5分钟后测量
3. 每次测量2-3遍，取平均值
4. 建议每天早晚各测一次
5. 早晨测量在服药和早餐前
6. 晚上测量在睡前

## 家庭血压监测目标

| 人群 | 目标血压 |
|------|----------|
| 一般高血压患者 | < 135/85 mmHg |
| 老年人（≥65岁） | < 145/90 mmHg |
| 糖尿病/肾病患者 | < 130/80 mmHg |

## 低血压参考

| 分类 | 收缩压 | 舒张压 |
|------|:---:|:---:|
| 正常低值 | 90-119 | 60-79 |
| 低血压 | < 90 | < 60 |
| 严重低血压 | < 80 | < 50 |

> 低血压伴随头晕、乏力、冷汗等症状时需要就医。
