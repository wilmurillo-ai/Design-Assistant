# 通用办公写作领域（common）

## 识别特征
问题涉及：报告、邮件、会议纪要、方案文档、总结汇报等职场写作，但不属于专业垂直领域。

## 任务类型
- writing：通用写作
- analysis：分析总结
- planning：方案规划
- summarization：摘要整理

## 必要槽位
| 槽位 | 说明 |
|------|------|
| target | 任务目标 |
| context | 场景背景 |
| audience | 输出受众 |
| output_format | 输出形式 |

## 常见缺失项
- 最终给谁看（领导/客户/团队）
- 篇幅/深度要求
- 语气/风格要求

## 追问模板
1. "这份内容最后是给谁看的？"
2. "你更想突出成果、问题分析，还是下一步计划？"
3. "希望简洁提纲还是完整文档？"

## Rewrite 模板
请以资深职场写作顾问视角，围绕【{target}】，面向【{audience}】，结合【{context}】，输出【{output_format}】。请确保结构清晰、重点突出、结论先行。
