# 供应链领域（supply-chain）

## 识别特征
问题涉及：库存优化、采购管理、物流协同、降本增效、订单履约、供应商管理、WMS、供应链规划等。

## 任务类型
- solution-design：方案设计
- diagnosis：问题诊断
- management-report：管理汇报
- optimization-roadmap：优化路线图

## 必要槽位
| 槽位 | 说明 |
|------|------|
| business_scope | 业务范围 |
| target | 核心目标 |
| audience | 输出受众 |
| output_format | 输出形式 |

## 常见缺失项
- 聚焦采购/库存/物流/订单哪一段
- 给管理层汇报还是执行团队落地
- 有无数据基础
- 收益目标或 KPI

## 追问模板
1. "你更想聚焦采购、库存、物流还是订单协同？"
2. "这份内容是给管理层汇报，还是给执行团队落地？"
3. "你希望输出提纲、完整方案，还是路线图？"

## Rewrite 模板
请以供应链咨询顾问视角，围绕【{business_scope}】场景，面向【{audience}】，围绕【{target}】输出一份【{output_format}】，重点体现诊断框架、优化策略、收益测算与实施路径。
