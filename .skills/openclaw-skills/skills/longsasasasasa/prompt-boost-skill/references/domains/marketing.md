# 市场营销领域（marketing）

## 识别特征
问题涉及：品牌定位、投放策略、内容营销、用户增长、活动策划、STP分析、4P策略、传播方案等。

## 任务类型
- strategy-design：策略设计
- campaign-planning：活动策划
- content-angle：内容方向
- brand-positioning：品牌定位

## 必要槽位
| 槽位 | 说明 |
|------|------|
| product | 产品/品牌 |
| target_users | 目标用户 |
| campaign_goal | 营销目标 |
| output_format | 输出形式 |

## 常见缺失项
- 品牌传播还是效果转化
- 预算和周期
- 目标用户是否有初步画像
- 主要投放渠道

## 追问模板
1. "这次更偏品牌传播、用户增长，还是直接转化？"
2. "目标用户是谁？有没有重点客群？"
3. "你希望我输出策略框架、传播方案，还是内容方向？"

## Rewrite 模板
请以市场策略顾问视角，围绕【{product}】，面向【{target_users}】，针对【{campaign_goal}】输出【{output_format}】，并给出定位、渠道、内容方向和效果指标建议。
