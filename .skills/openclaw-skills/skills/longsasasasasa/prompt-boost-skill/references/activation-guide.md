# Activation Guide

## When to Activate

激活本技能的条件（满足任一即可）：
- 用户问题过短、过泛、过口语化
- 用户知道想做什么，但表达不够专业
- 问题属于数据分析、技术排障、产品方案、市场营销、供应链等专业场景
- 下游业务 Agent 需要结构化输入
- 用户明确要求"优化提问"、"帮我把问题说专业一点"

## When NOT to Activate

以下情况通常不需要激活：
- 纯闲聊
- 直接事实问答（用户已精确描述）
- 问题已经高度结构化完整
- 用户明确要求跳过澄清直接执行

## Mode Selection

| 输入特征 | 推荐模式 |
|---------|---------|
| 一句话、口语化 | quick |
| 有一定背景，但不够完整 | professional |
| 专业领域、复杂任务、高要求 | expert |
