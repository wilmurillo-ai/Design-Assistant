# Anima-AIOS 使用示例

**版本：** v6.0.0

---

## 基础功能

### 写记忆（自动同步）

```
你：记住：今天完成了 Anima v6.0 的五层记忆架构开发
Agent：✅ 记忆已保存，+2 EXP（semantic）
```

背后自动发生：
1. OpenClaw 写入 memory/2026-03-24.md
2. memory_watcher 检测到变化 → 同步到 L2
3. EXP +1（自动）
4. 每日任务"写一条记忆"自动完成

### 搜索记忆

```
你：搜索记忆"架构设计"
Agent：找到 3 条相关记忆：
  1. [L3] v6 选择 Python 统一技术栈（强度: 0.95）
  2. [L3] 五层记忆模型设计（强度: 0.87）
  3. [L2] Phase 3 宫殿开发记录（强度: 0.72）
```

搜索结果按 `相关性 × 记忆强度` 排序。
