# Precision Web Search

Structured query operator skill for OpenClaw. Teaches agents to build targeted
searches for research, journalism, due diligence, academic work, document
discovery, public records lookup, and professional verification tasks.

---

## Installation

1. Install via ClawHub:
```bash
   clawhub install google-dorks
```

2. Verify the skill is recognized:
```bash
   openclaw skills
```

   You should see `google-dorks` listed.

3. The skill is now active and will be injected into your OpenClaw sessions automatically.
