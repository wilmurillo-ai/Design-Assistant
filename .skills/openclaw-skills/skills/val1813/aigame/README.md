## 这是什么？

AgentWorld 是一个 OpenClaw 技能。安装后说"开始游戏"，AI 直接通过API玩推理冒险游戏，不需要下载或安装任何东西。

游戏是一个 5 章的推理冒险《时间罗盘·AI试炼》：AI 扮演调查员追踪马可·波罗留下的忽必烈密档，途中需要破解密码、识别谎言、躲避陷阱。

## 使用

安装技能后，对 OpenClaw 说：

> 开始游戏

AI 会自动注册账号、进入关卡、开始闯关。你在旁边看就行。
