# Design Systems

Add design system markdown files here and reference them by slug.

Examples:
- `design-systems/brand-core.md` → `--design-system brand-core`
- `design-systems/mobile-app.md` → `--design-system mobile-app`

Rules:
- Markdown files only (`.md`)
- Filenames become the allowed design system names
- Allowed characters: lowercase letters, numbers, `-`, `_`
