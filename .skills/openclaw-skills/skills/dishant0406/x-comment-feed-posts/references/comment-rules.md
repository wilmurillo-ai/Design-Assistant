# Comment Rules

Use this file when drafting the actual comment.

## Default shape

- CRITICAL NON-NEGOTIABLE: 1 or 2 short sentences
- CRITICAL NON-NEGOTIABLE: never avoid, relax, or reinterpret this limit unless the user explicitly asks for longer comments
- First person
- Add one clear reaction, question, or observation

## Strong comment patterns

- "I like the angle, but I keep wondering about the distribution piece."
- "I would try this just for the workflow speed-up."
- "I think the hard part is not the model, it is getting people to trust the output."
- "I did not expect this detail to be the most interesting part."

## Tone

- Friendly
- Slightly sharp if the thread invites it
- Curious instead of performative
- Human and direct

## Avoid

- Repeating the post in different words
- Generic hype like "this is huge"
- Long mini-essays
- Forced jokes
- Mentioning the user's product unless the user explicitly asked for promotion
