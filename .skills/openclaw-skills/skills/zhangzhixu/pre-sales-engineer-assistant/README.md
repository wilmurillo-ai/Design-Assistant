# 售前工程师助手技能

## 概述

这是一个专为 AI 安全领域售前工程师设计的助手技能，覆盖售前工作的完整流程，包括项目立项、方案设计、POC 测试、汇报制作、招投标支持、竞品分析和报价测算等核心任务。

## 核心功能

该技能包含 10 个标准化工作流：

1. **撰写项目立项书** - 内部评估项目可行性
2. **生成定制解决方案** - 客户场景化解决方案
3. **设计POC测试方案** - 红蓝对抗验证方案
4. **提炼客户汇报PPT** - 高管汇报大纲
5. **绘制架构拓扑图** - Mermaid 格式拓扑图
6. **起草招投标质疑函** - 法律合规质疑函
7. **生成技术偏离应答表** - 点对点技术应答
8. **生成竞品Battlecard** - 销售打单话术
9. **智能BOM与报价测算** - 硬件软件资源配置
10. **撰写POC总结报告** - 高管汇报总结

## 使用示例

### 示例 1：撰写项目立项书

**用户请求**：
```
帮我写一个项目立项书。客户是XX银行，项目背景是他们需要为大模型应用部署安全防护系统，预算大概500万，周期6个月。
```

**技能执行**：调用 `write_project_initiation` 工作流
- `customer_info`: "XX银行，金融行业"
- `project_background`: "需要为大模型应用部署安全防护系统"
- `budget_and_timeline`: "预算500万，周期6个月"

**输出**：包含背景提炼、价值分析、资源测算、风险预判和ROI预估的Markdown格式立项书

---

### 示例 2：生成解决方案

**用户请求**：
```
帮我生成一个解决方案。客户是一家电商公司，他们用AI做智能客服，需要防止prompt注入攻击，推荐使用我们的防火墙和内容安全模块。
```

**技能执行**：调用 `draft_security_solution` 工作流
- `business_scenario`: "电商公司AI智能客服"
- `security_requirements`: "防止prompt注入攻击"
- `product_modules`: ["防火墙", "内容安全模块"]

**输出**：包含需求理解、整体架构、核心功能、客户价值的解决方案文档

---

### 示例 3：绘制架构拓扑图

**用户请求**：
```
帮我画一个架构拓扑图，采用串联拦截代理模式，组件包括网关、防火墙、AI模型、数据库，流量从网关进入，经过防火墙检查后到AI模型。
```

**技能执行**：调用 `render_topology_diagram` 工作流
- `deployment_mode`: "串联拦截代理"
- `components_list`: ["网关", "防火墙", "AI模型", "数据库"]
- `data_flow_description`: "流量从网关进入，经过防火墙检查后到AI模型"

**输出**：Mermaid 格式的拓扑图代码

## 技能结构

```
pre-sales-engineer-assistant/
├── SKILL.md              # 技能核心配置（已包含所有工作流定义）
├── README.md             # 本说明文件
├── scripts/              # 可执行脚本目录（预留）
├── references/           # 参考文档目录（预留）
└── assets/               # 资源文件目录（预留）
```

## 扩展建议

### references/ 目录建议添加的文档

1. **product_modules.md** - 详细的产品模块说明和功能清单
2. **security_standards.md** - 网络安全与AI安全相关标准和法规
3. **tender_laws.md** - 招投标相关法律法规（招投标法、政府采购法）
4. **competitor_intel.md** - 竞争对手情报资料

### assets/ 目录建议添加的文件

1. **templates/**
   - `project_initiation.docx` - 项目立项书Word模板
   - `solution_template.pptx` - 解决方案PPT模板
   - `tender_response.docx` - 投标响应模板
   - `poc_summary.docx` - POC总结报告模板

2. **brand/**
   - Logo 文件
   - 品牌色彩规范
   - 字体文件
