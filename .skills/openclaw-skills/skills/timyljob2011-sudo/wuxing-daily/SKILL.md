# 老黄历Daily (Laohuangli Daily)

## Overview

本 Skill 提供基于中国传统命理学的五行命理分析和每日运势提醒功能。

通过用户提供的出生年月日时（生辰八字），可以：
1. **计算个人五行属性**（金木水火土）
2. **分析命格特征与性格倾向**
3. **生成每日运势报告**
4. **设置每日五行运势提醒**（通过 cron 任务）

## 核心概念

### 五行基础

**五行相生相克：**
- 相生：木生火 → 火生土 → 土生金 → 金生水 → 水生木
- 相克：木克土 → 土克水 → 水克火 → 火克金 → 金克木

**天干五行对应：**
- 甲乙（木）、丙丁（火）、戊己（土）、庚辛（金）、壬癸（水）

**地支五行对应：**
- 寅卯（木）、巳午（火）、申酉（金）、亥子（水）、辰戌丑未（土）

## 使用流程

### 1. 用户输入生辰八字

获取用户出生信息：
- 出生年份（公历或农历）
- 出生月份
- 出生日期
- 出生时辰（可选，精确到小时）

**示例用户输入：**
> "我是农历1989年十一月十五出生的"
> "帮我算一下1990年5月20日的八字"

### 2. 计算五行命理

**执行脚本：**
```bash
python3 scripts/calculate_wuxing.py <年> <月> <日> <时>
```

**步骤：**
1. 将农历转换为公历（如果需要）
2. 根据年柱确定纳音五行
3. 排出完整八字（年柱、月柱、日柱、时柱）
4. 分析八字中的五行分布
5. 判断五行旺衰与缺失
6. 推算喜用神

**输出示例：**
```
========================================
五行命理分析报告
========================================

【八字排盘】
  年柱：己巳 (土火)
  月柱：丙子 (火水)
  日柱：丙午 (火火)
  时柱：甲午 (木火)

【本命纳音】大林木

【日主】丙 (火)

【五行分布】
  金：░░░░ (0个) - 缺失
  木：█░░░ (1个) - 偏弱
  水：█░░░ (1个) - 偏弱
  火：████ (4个) - 偏旺
  土：█░░░ (1个) - 偏弱

【五行分析】
  偏旺：火
  偏弱：木, 水, 土
  缺失：金

【喜用神】土, 金, 水

【性格特征】
  丙火日主：热情开朗，积极向上，有领导力
  但火旺易冲动，需注意情绪管理

========================================
```

### 3. 每日运势分析

**执行脚本：**
```bash
python3 scripts/daily_fortune.py <日主天干>
```

**分析逻辑：**
- 获取当日天干地支
- 对比个人日主与当日天干五行关系
- 分析生克影响，给出运势评级

**运势评级标准：**

| 当日天干与个人日主关系 | 运势评级 | 建议 |
|---------------------|---------|-----|
| 被生（如日主甲木，当日癸水） | ★★★★★ 大吉 | 签约、求财、出行、社交 |
| 同五行（如日主甲木，当日乙木） | ★★★★ 吉 | 顺势而为，日常事务 |
| 我生（如日主甲木，当日丙火） | ★★★ 平 | 付出努力，会有回报 |
| 克我（如日主甲木，当日庚金） | ★★ 需注意 | 谨慎行事，避免决策 |
| 我克（如日主甲木，当日戊土） | ★★★ 平 | 需要努力，可以成功 |

**输出示例：**
```
🌟 今日五行运势 🌟

📅 今日干支：壬子
   五行属性：水 水

【今日运势】★★☆☆☆ 需谨慎

💡 今日五行克制日主，宜守不宜攻

【宜】
  ✅ 谨慎
  ✅ 内省
  ✅ 整理
  ✅ 休息

【忌】
  ❌ 重大决策
  ❌ 投资
  ❌ 争吵
  ❌ 高风险活动

【幸运元素】
  🎨 幸运色：黑色、蓝色
  🧭 幸运方位：北方
  🔢 幸运数字：2
```

### 4. 农历转换

**执行脚本：**
```bash
# 公历转农历
python3 scripts/lunar_convert.py <年> <月> <日>

# 显示今日农历
python3 scripts/lunar_convert.py
```

**输出示例：**
```
2026年3月19日
农历：2026年二月初一 丙午年 [马]
```

### 5. 设置每日提醒

使用 cron 创建每日运势提醒：

```bash
# 每天早上 8:17 发送运势提醒（避开整点高峰）
cron add --name "daily-wuxing-reminder" \
  --schedule "17 8 * * *" \
  --command "python3 /path/to/daily_fortune.py <日主>"
```

## 注意事项

1. **娱乐性质**：本 Skill 提供的命理分析仅供娱乐参考，不应作为人生重大决策的唯一依据
2. **文化尊重**：紫微斗数和五行命理是中国传统文化的一部分，使用时请保持尊重
3. **时间精度**：出生时辰越精确，分析结果越准确（至少精确到时辰，最好到小时）
4. **农历转换**：自动处理公历转农历，但遇到闰月需要用户确认
5. **日柱计算**：基于万年历数据校准，以1989年12月12日（丙午日）为基准计算
