# 老黄历Daily (Laohuangli Daily)

基于中国传统命理学的五行命理分析和每日运势提醒工具。

## 功能特点

- 🔮 **八字排盘**：根据出生年月日时排出完整八字
- 🌟 **五行分析**：分析金木水火土五行分布与旺衰
- 📊 **命格解读**：推算纳音五行、喜用神、性格特征
- 📅 **每日运势**：根据当日干支分析个人运势
- 🔔 **定时提醒**：可设置每日运势自动推送

## 快速开始

### 计算五行命理

```bash
python3 scripts/calculate_wuxing.py 1989 12 12 12
```

### 查看今日运势

```bash
python3 scripts/daily_fortune.py 丙
```

### 农历转换

```bash
python3 scripts/lunar_convert.py 2026 3 19
```

## 使用示例

**输入生日信息：**
> "我是农历1989年十一月十五出生的"

**获取分析报告：**
- 八字：己巳年 丙子月 丙午日
- 纳音：大林木
- 日主：丙火
- 五行：火旺（4个），缺金
- 喜用神：土、金、水

**查看今日运势：**
- 今日壬子日（水水）
- 运势：★★☆☆☆ 需谨慎
- 宜：保守、休息、整理
- 忌：重大决策、投资
