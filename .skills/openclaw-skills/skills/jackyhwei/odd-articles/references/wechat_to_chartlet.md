# 微信文章转公众号贴图

一键把微信公众号文章转成公众号贴图 + 发布文案。

---

## 功能

输入微信文章链接，自动生成：
1. **9张公众号贴图轮播图**（纯信息图设计，540×720px）
2. **公众号贴图发布文案**（真人分享风格，300字左右）

---

## 使用方法

### 触发词

| 触发词 | 说明 |
|-------|------|
| `/chartlet` + 微信链接 | 生成公众号贴图 |
| "转公众号贴图" + 微信链接 | 同上 |
| "做成公众号贴图" + 微信链接 | 同上 |

### 示例

```
/chartlet https://mp.weixin.qq.com/s/xxxxx
```

或直接说：
```
把这篇文章转成公众号贴图：https://mp.weixin.qq.com/s/xxxxx
```

## 输出内容

### 1. 公众号贴图轮播图（HTML）

**9张卡片结构：**
1. **封面** - 吸睛标题 + 核心亮点
2. **核心价值** - 信息图展示核心能力
3. **痛点/背景** - 为什么需要这个东西
4. **案例/成果1** - 具体展示
5. **使用步骤** - 4步流程图
6. **案例/成果2** - 更多展示
7. **案例/成果3** - 补充展示
8. **方法论/金句** - 提炼核心理念
9. **行动召唤** - 链接 + 微信号

**设计规范：**
- 纯信息图设计，不放原文截图
- oddmeta品牌色（墨绿 #1A3328 + 鱼红 #C44536）
- 暗底和浅底交替，视觉舒适
- 每张卡片都有品牌角标和页码

**下载方式：**
- 点击"📥 下载全部图片(ZIP)"一键打包下载
- 或"📸 逐张下载"单张保存

### 2. 公众号贴图发布文案

**文案风格：**
- 第一人称真实分享，不是产品介绍
- 有使用场景和个人感受
- 口语化表达，像朋友聊天
- 300字左右，8-12个标签

**结构：**
```
[真实场景/痛点开头]

[发现这个东西的过程]

[实际使用体验]

[核心价值提炼]

[行动召唤]

.
#标签1 #标签2 ...
```

## 工作流程

**Step 1: 抓取文章**
```bash
python3 "${SKILL_DIR}/scripts/wechat_download.py" "<URL>" --json
```
- 超时30秒
- 返回标题、正文、图片列表

**Step 2: 分析文章**
- 提取核心概念、关键数据、亮点特色
- 识别痛点、解决方案、方法论

**Step 3: 设计卡片**
- 规划9张卡片的内容和视觉
- 用纯CSS设计信息图，不放原文图片
- 确保信息清晰、层次分明

**Step 4: 生成HTML**
- 输出路径：`/tmp/[主题]-chartlet.html`
- 自动在浏览器打开预览
- 包含ZIP打包下载功能

**Step 5: 生成文案**
- 参考 `references/generate_chartlet.md` 的规范
- 真人分享风格，不要营销感
- 300字左右 + 8-12个标签

## 设计原则

### 信息图 vs 截图

❌ **不要放原文截图**
- 显得杂乱、不专业
- 图片模糊、文字看不清
- 没有设计感

✅ **用信息图表达**
- 清晰简洁，重点突出
- 视觉统一，品牌感强
- 专业美观，易于阅读

### 卡片设计要点

**核心原则：**
1. **一页一个点** - 每张卡片只讲一件事
2. **视觉替代文字** - 用图标、流程图、卡片布局代替大段文字
3. **留白充足** - 不要塞满，给眼睛喘息空间
4. **色彩克制** - 墨绿85% + 鱼红5% + 其余10%

**常用视觉元素：**
- 流程图（A → B → C）
- 编号列表（1️⃣2️⃣3️⃣）
- 卡片网格（2×3、1×4等）
- 左边框强调框
- 大号数字/图标
- 引号金句

## 文案原则

### 真人感 vs 营销感

❌ **营销感太重**
```
发现一个神器，能让Claude Code秒变领域专家。

这就是pdf2skills要解决的问题——把任何专业书的知识，
变成Claude Code的技能包。

核心价值：买书容易读书难...
```
太像产品介绍稿，不像真人分享。

✅ **真人分享感**
```
最近玩疯了一个东西，能给Claude Code"装技能"。

我书架上有本《云彩收集者手册》，46种云的分类，
但一直没看。然后发现pdf2skills这个工具...

用法超简单，整个过程几分钟搞定。
```
像朋友聊天，有真实场景和感受。

### 写作技巧

1. **用真实场景开头**
   - "我书架上有本..."
   - "之前遇到个问题..."
   - "最近在做xxx的时候..."

2. **有个人情绪**
   - "太好玩了" "上头了" "停不下来"
   - "贼简单" "爽爆了" "有点像xxx"

3. **用口语化表达**
   - "然后我就..." "结果发现..."
   - "感觉像..." "好像能..."

4. **留悬念/引导互动**
   - "感兴趣可以试试"
   - "链接评论区见"
   - "私信我xxx"

## 品牌元素

### oddmeta 品牌色

| 名称 | 色值 | 用途 |
|------|------|------|
| 墨绿主色 | `#1A3328` | 暗底背景、标题、边框 |
| 鱼红点缀 | `#C44536` | 强调色、数字、重点 |
| 宣纸底 | `#F2EDE3` | 浅底背景 |
| 深墨 | `#0F1F18` | 更深背景 |
| 苔灰 | `#7A8C80` | 次要文字 |

### 品牌角标

每张卡片左上角：
```
○ ODDMETA
```
- 暗底卡片：白色/半透明
- 浅底卡片：墨绿/半透明

### 页码

右下角：`1/9`
- 暗底：`rgba(255,255,255,0.3)`
- 浅底：`rgba(26,51,40,0.2)`

## 故障处理

| 问题 | 处理 |
|------|------|
| 微信文章抓取失败 | 提示用户手动复制正文 |
| 文章太短（<500字） | 压缩为6-7张卡片 |
| 文章太长（>5000字） | 精选核心，控制9张以内 |
| 浏览器不自动打开 | 告知用户手动打开HTML路径 |

- 每次只处理一篇文章
