# 平台文案规范

本文档定义了各社交媒体平台的文案规范。

## 微贴图

### 封面文案

- **字数**：20 字以内
- **风格**：口语化、有冲击力
- **结构**：关键词 + 痛点/效果

### 正文文案

- **字数**：300-350 字
- **结构**：
  1. 开头：Hook（痛点/悬念）
  2. 中间：核心内容（3-4 个要点）
  3. 结尾：CTA + 标签

### 标签

- 8-12 个标签
- 包含领域关键词
- 热门标签穿插

### 示例

```
开头：
兄弟们！我终于搞定了这个难题！

中间：
1. 第一点...
2. 第二点...
3. 第三点...

结尾：
收藏起来慢慢看～ 👇
#标签1 #标签2 #标签3
```

## 微博/博客园/朋友圈

### 风格特点

- **简洁**：每条不超过 280 字
- **真实**：第一人称日常感
- **互动**：留问题引导评论

### 结构

```
[正文内容]
↓

[互动问题]
```

### 圈子标签

- 选择 1-2 个相关圈子
- 圈子名用 # 包围

---

## 播客

### 脚本结构

1. **开场**（1-2 分钟）
   - 打招呼
   - 本期主题介绍
   - 预告亮点

2. **正文**（15-30 分钟）
   - 逻辑清晰
   - 适当留白
   - 口语化表达

3. **结尾**（1-2 分钟）
   - 总结要点
   - CTA（关注、评论）
   - 下期预告

### 语音要求

- 语速：每分钟 150-180 字
- 停顿：关键点停顿
- 情感：自然起伏

- 简短精炼
- 适度 emoji

---

## 微博

### 风格特点
- **简洁**：每条不超过 2000 字（普通用户），长微博可达万字
- **话题**：带 #话题标签
- **互动**：转发、评论、点赞

### 结构
```
[正文内容]

#话题1 #话题2 #话题3
```

### 注意事项
- 微博内容审核较严，避免敏感词
- 热门话题更容易获得曝光
- 图片/视频可增加互动

## 核心原则

1. **适配平台**：不是照搬，要转换
2. **真实感**：不做作，有个人特色
3. **有价值**：要么有用，要么有趣
4. **有行动**：明确 CTA
