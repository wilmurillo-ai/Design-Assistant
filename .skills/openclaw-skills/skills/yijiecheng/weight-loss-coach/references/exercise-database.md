# 运动 MET 值与热量消耗表

## 计算公式
```
消耗热量(kcal) = MET × 体重(kg) × 时间(小时)
```
示例：60kg 的人慢跑30分钟 = 7.0 × 60 × 0.5 = 210 kcal

## 运动建议逻辑

### 根据目标热量缺口推荐运动：
- 缺口 < 200 kcal：快走30分钟即可
- 缺口 200-400 kcal：慢跑30分钟 或 骑车45分钟
- 缺口 400-600 kcal：跑步45分钟 或 HIIT 30分钟
- 缺口 > 600 kcal：提示用户不建议完全依靠运动弥补，应结合饮食调整

### 运动频率建议（基于目标）：
- 纯减脂：每周4-5次有氧 + 2-3次力量
- 维持体型：每周3-4次混合训练
- 增肌减脂：每周3-4次力量 + 2-3次有氧

### 新手保护：
- 前2周不建议超过每周3次
- 每次不超过45分钟
- 两次力量训练之间需要休息24-48小时
