# Save/Load Context Plugin for OpenClaw

Save and restore conversation context with `/save` and `/load` commands.

## Quick Start

```bash
# Save current conversation
/save my-session

# List saved contexts  
/load

# Load a saved context
/load my-session

# Delete a saved context
/load --delete my-session
```
