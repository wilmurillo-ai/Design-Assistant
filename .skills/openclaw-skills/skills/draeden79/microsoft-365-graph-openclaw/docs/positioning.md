# Positioning

## What this project is

- Push-based Microsoft Graph integration for OpenClaw.
- A self-hosted, operationally auditable path for event-driven mail triggers.
- A cost-aware alternative to recurring polling wake-ups.

## What this project is not

- Not a "zero cost" automation layer.
- Not a generic replacement for all Microsoft 365 integrations.
- Not a managed cloud SaaS; this is self-hosted/operator-owned.

## Allowed claims

- "Push-based Microsoft Graph for OpenClaw."
- "Wake the agent only when work actually happens."
- "Replace recurring inbox polling with Graph change notifications."

## Prohibited claims

- "First" or "only" in category without hard evidence.
- "Zero cost."
- "Fully autonomous."
- "Production-ready" without explicit operational context and constraints.
