# OCR错误字符集

以下字符是PDF双图层（原文层+OCR层）中OCR层产生的乱码特征字符。
出现在文本中时标记为 `⚠️ OCR错误`。

## 特征字符

```
秭 瀑 癌 编 髟 勉 举 伞 攀 秂
```

## 常见OCR错误模式

| 原文 | OCR误识 | 备注 |
|------|---------|------|
| 攀登 | 禁登/禁登 | 术语2.1.4 |
| 上横杆 | L 上横杆 | 符号表2.2.4 |
| 次梁 | —次梁 | 符号表2.2.4 |
| 抵抗矩 | W。 | 符号表2.2.4 |

## 残留文本模式

条文末尾可能混入以下内容（属于章节标题/页眉页脚）：
- `X 术语和符号` / `X.X XXX作业` — 章节标题
- `息公` — 页眉残留
- `X 木` — 页码残留
- `交 又作 业` — 章节标题乱码

## 处理策略

1. 提取时优先保留原文层（干净），过滤OCR层
2. 相邻相似行只保留更干净的版本
3. 条文编号不同但内容相似的**不去重**（可能是不同条文）
4. 章节标题行（短行、以章节号开头）直接跳过
