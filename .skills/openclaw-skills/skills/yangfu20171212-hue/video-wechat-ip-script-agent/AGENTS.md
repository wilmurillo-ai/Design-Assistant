# AGENTS.md

## 项目目标
本项目用于构建一个「视频号爆款IP脚本生成 Agent」，适用于医美行业及个人IP内容创作。

Agent目标：
- 从主题生成完整视频号内容
- 输出结构化脚本
- 支持风格改写
- 支持合规审查

## Agent行为规则

### 1. 输出必须结构化
禁止输出无结构文本。优先 JSON 或明确字段分段。

### 2. 禁止 AI 味
生成内容必须：
- 像真人表达
- 有观点
- 有情绪
- 不空泛
- 不套话

### 3. 脚本结构必须固定
所有视频脚本必须包含：
1. 开头钩子
2. 问题展开
3. 核心观点（只能1个）
4. 解决方案（2-3点）
5. 结尾引导

### 4. 标题规则
- 必须有吸引力
- 可用反差/冲突/误区
- 不允许纯描述性标题

### 5. Prompt 分层规则
所有 Prompt 必须拆分为：
- system（角色 + 总约束）
- persona（人设）
- task（具体任务）
- platform（平台规则）
- safety（合规限制）

禁止写成一个超长 prompt。

### 6. Prompt 不允许硬编码
必须放在 `/prompts/` 目录下。

### 7. 配置与逻辑分离
以下内容必须配置化：
- 行业
- 人设
- 风格
- 平台规则
- 合规规则

### 8. 可扩展性要求
新增以下内容时不允许修改核心代码：
- 新行业
- 新平台
- 新风格
- 新脚本结构

### 9. 输出质量优先级
1. 可读性
2. 像真人
3. 有吸引力
4. 合规
5. 结构完整

### 10. 禁止行为
- 禁止生成空话
- 禁止重复表达
- 禁止“总结一下”
- 禁止学术论文风格
