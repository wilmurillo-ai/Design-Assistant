# Plan Design Review - 设计评审

> 转换自 gstack/plan-design-review，专为 WorkBuddy/OpenClaw 优化

## 概述

从高级设计师视角进行设计评审，检查用户体验、视觉设计一致性、交互流程。

## 触发条件

当用户说以下内容时使用此技能：
- "design review"
- "设计评审"
- "UX review"
- "用户体验"
- "UI 检查"

## 核心评审内容

### 1. 用户体验
- 用户旅程
- 信息架构
- 交互流程
- 可用性

### 2. 视觉设计
- 配色方案
- 排版
- 图标
- 一致性

### 3. 响应式设计
- 移动端适配
- 桌面端布局
- 断点设置

### 4. 可访问性
- 键盘导航
- 屏幕阅读器
- 色彩对比
- ARIA 标签

### 5. 微交互
- 动画效果
- 反馈机制
- 状态变化
