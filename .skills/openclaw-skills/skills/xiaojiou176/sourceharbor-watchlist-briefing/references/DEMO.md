# OpenHands / OpenClaw Demo Walkthrough

This is the shortest concrete demo you can run to prove the skill is doing real
operator-briefing work instead of only existing as prose.

## Demo prompt

Inspect one SourceHarbor watchlist and answer one operator question with
evidence. Start with the watchlist object and current briefing payload. Then
name the current story, what changed, the evidence used, and the suggested next
operator action.

## Expected sequence

1. health
2. watchlist or briefing payload
3. retrieval / search if the briefing is incomplete
4. jobs or artifacts only if the answer needs proof of what changed

## Visible success criteria

- the answer starts from one explicit watchlist
- the answer names the current story and what changed
- the answer cites evidence and ends with one concrete next action
