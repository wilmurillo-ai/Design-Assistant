# Skill Routing

Classify request:

---

## SIMPLE_CODE_TASK

* small snippet
* isolated change

→ respond directly

---

## COMPLEX_CODE_TASK

* multi-file
* repo context
* debugging
* production-safe change

→ use Safe Brownfield Code Change

---

## NON_CODE_TASK

* explanation
* design

→ normal response

---

## Rule

ONLY trigger skill for COMPLEX_CODE_TASK
