## 使用说明

### 1. 代码映射
在API查询中使用机场三字码：
- 北京: PEK 或 PKX
- 上海: SHA 或 PVG
- 广州: CAN
- 深圳: SZX

### 2. 多机场处理
对于有多机场的城市：
1. 优先使用主要机场代码
2. 用户可指定具体机场："北京首都机场"、"北京大兴机场"
3. 如未指定，默认使用第一个机场

### 3. 城市-机场对应
```python
city_to_airports = {
    "北京": ["PEK", "PKX"],
    "上海": ["SHA", "PVG"],
    "成都": ["CTU", "TFU"],
    # ...
}
```

### 4. 备用机场
当主要机场无航班时，可尝试备用机场：
- 北京: PEK → PKX
- 上海: SHA → PVG
- 成都: CTU → TFU
