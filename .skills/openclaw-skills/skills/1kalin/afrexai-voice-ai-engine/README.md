# Voice AI Agent Engineering 🎙️

Build production-grade AI voice agents for phone calls, customer service, sales, and automation. Platform-agnostic methodology — works with Vapi, Retell, Bland AI, Vocode, LiveKit, Twilio, or any custom stack.

## What This Skill Does

Gives your AI agent a complete voice AI engineering methodology:

- **Use case scoring** — Know if voice AI is right before building
- **Platform selection** — Compare 7 platforms with decision tree
- **Conversation design** — Turn-by-turn flow architecture with YAML templates
- **Voice UX rules** — Phone conversations ≠ chatbots. Different rules apply
- **System prompts** — Optimized for voice (short, conversational, latency-aware)
- **Latency engineering** — Hit <1.5s response time across the full pipeline
- **Compliance** — TCPA, GDPR, PCI-DSS, HIPAA checklists by jurisdiction
- **Testing** — Voice-specific QA including phone line, noise, and accent testing
- **Monitoring** — Real-time dashboard with alert rules and weekly review
- **Cost calculator** — Per-call ROI vs human agents
