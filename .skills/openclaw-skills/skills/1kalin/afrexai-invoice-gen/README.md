# Invoice Generator

Creates professional invoices in markdown or HTML. Just tell it who to bill and for what.

## What It Does

- Generates clean, professional invoices
- Supports markdown and printable HTML output
- Handles tax calculations and multiple line items
- Tracks invoice history and outstanding payments
- Supports recurring invoices for repeat clients

## Usage

"Invoice Acme Corp $5,000 for website redesign, net 30."
"Create this month's invoice for my retainer client."
