# RFP & Proposal Responder

Stop spending 20+ hours on proposal responses. This skill turns RFPs into structured, competitive bids in minutes.

## Features

- Executive summary generation
- Requirements compliance matrix
- Good/Better/Best pricing structures
- Risk mitigation sections
- Realistic timeline with dependencies
- Competitive differentiator framing

## Install

```bash
clawhub install afrexai-rfp-responder
```

## Quick Start

Paste your RFP requirements and company details. The agent builds a submission-ready response draft.
