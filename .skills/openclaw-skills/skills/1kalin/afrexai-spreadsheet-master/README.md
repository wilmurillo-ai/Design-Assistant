# Spreadsheet & Data Wrangling Master

> Complete spreadsheet methodology — data cleanup, transformation, analysis, dashboards, automation, and reporting. Works with CSV, Excel, Google Sheets, or any tabular data.

## Install

```bash
clawhub install afrexai-spreadsheet-master
```

## What This Skill Does

Turns your AI agent into a spreadsheet power user:

- **Data Cleaning** — Decision tree for handling duplicates, missing values, outliers, type mismatches, and structural issues
- **Transformation** — 12 essential operations (filter, sort, group, pivot, join, derive, etc.) with cross-platform formula reference
- **Analysis** — Descriptive, diagnostic, predictive, and prescriptive analysis frameworks with the Insight Formula
- **Dashboards** — Executive summary templates, KPI formatting rules, chart selection guide
- **Automation** — Recurring report templates with quality checks and error handling
- **5 Ready-to-Use Patterns** — Financial model, CRM tracker, inventory, project tracker, budget vs actual
- **Data Quality** — Validation checklist, quality scoring (0-100), cleaning log template
- **Format Conversion** — CSV ↔ Excel ↔ JSON ↔ Google Sheets with encoding and gotcha guides
- **Scale** — Size thresholds and tool recommendations from 10 rows to 10M+

## Quick Start

Tell your agent:
- "Clean this CSV" — runs the full cleaning protocol
- "Analyze this data" — assessment → analysis → insights
- "Build a dashboard" — populates an executive summary template
- "What's wrong with this data?" — health check + validation

## Who It's For

Anyone working with spreadsheets: analysts, founders, ops teams, accountants, marketers, sales reps. If you touch tabular data, this skill makes your agent 10X better at it.
