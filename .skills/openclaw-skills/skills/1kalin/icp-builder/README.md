# ICP Builder

Builds Ideal Customer Profiles from your actual customer data. Defines who to target, how to score them, and who to avoid.

## What It Does

- Creates company-level and buyer-level profiles
- Builds a weighted scoring model for prospect prioritization
- Defines trigger events that signal buying intent
- Includes anti-ICP (disqualifiers) so you don't waste time

## Usage

"Build an ICP for our project management tool. Our best customers are 50-200 person tech companies."
