# 高级砍价策略

## 心理学技巧

### 1. 锚定效应

先给一个较低的价格锚定，即使被拒绝，后续出价也会显得更合理。

```
开口：500
被拒后：那 580 行吗？（卖家更容易接受）
```

### 2. 互惠原则

提供对卖家有利的真实条件。

```
"我可以同城自提省运费"
"我可以马上付款，不用你等"
```

> ⚠️ 只承诺用户真正愿意做的事，不要代用户做承诺（如好评晒图等）。

### 3. 表达购买诚意

表达真实的购买意愿。

```
"价格合适的话可以马上付款"
"我可以同城自提省运费"
```

> ⚠️ 不要替用户做虚假承诺（如"今天就要用"、"给你好评晒图"等），只表达真实意愿。

### 4. 社会认同

引用市场价格作为参考。

```
"我看了十几个同款，大多在 XXX 价位"
"咸鱼上这个成色一般都是 XXX 价"
```

## 场景化话术

### 数码产品砍价

```
你好，这个还在吗？
请问是什么时候买的？有盒子和配件吗？
电池健康度多少？有没有维修过？
我看同款 XX 成色的大概在 [价格] 左右，
方便 [目标价] 出吗？可以今天拍~
```

### 服饰砍价

```
亲，这个还有吗？
请问是正品吗？有购买记录吗？
穿过几次呀？有没有色差或瑕疵？
方便看下实物图和吊牌吗？
[看图后] 挺喜欢的，[目标价] 能出吗？
```

### 家电砍价

```
你好，这个还在出吗？
用了多久了？有什么问题吗？
方便同城自提看下实物吗？
自提的话能便宜点吗？[目标价] 可以吗？
```

### 大件/贵重物品

```
你好，这个很感兴趣~
方便看看更多实拍细节图吗？
可以同城当面交易验货吗？
当面的话 [目标价] 可以接受吗？
```

> ⚠️ 始终在闲鱼平台内交易，不要引导到微信等站外渠道。走平台担保交易保护双方权益。

## 应对卖家套路

### 卖家说"已经最低了"

```
理解的，那我再加 XX 块，[价格] 怎么样？
真的很喜欢这个，这个价格我马上拍~
```

### 卖家说"有人出更高"

```
如果有更高的你直接卖就行，不用等我~
我这边就是这个预算，合适的话随时拍
```

### 卖家不回复

等待 4-6 小时后：
```
你好，还在吗？我出的价格可以考虑下~
可以的话麻烦回复我，谢谢
```

### 卖家反复加价

```
我们之前不是说好 XXX 吗？
麻烦按说好的价格改下吧，不然我就看别家了
```

## 防骗指南

### 常见骗术

1. **低价诱骗**：价格远低于市场价 → 可能是假货或骗子
2. **引导私聊**：要求加微信/QQ 交易 → 失去平台保护
3. **闪照陷阱**：只发阅后即焚图片 → 无法留证
4. **假冒客服**：冒充官方要求操作 → 永远不要相信
5. **到付骗局**：要求先付款再发货 → 坚持用平台支付

### 安全交易原则

- 永远走平台交易，不要私下转账
- 贵重物品坚持验货担保
- 截图保存所有聊天记录
- 收货后仔细验货再确认
- 发现问题及时申请退款
