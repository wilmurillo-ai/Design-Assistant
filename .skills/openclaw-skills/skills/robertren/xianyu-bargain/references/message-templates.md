
## 场景化话术

### 回应卖家询问

**问同城**
- 友好：可以同城自提的！这样也省运费～
- 专业：支持同城当面交易，方便验货。
- 随性：同城没问题，你定时间地点
- 幽默：同城的！咱俩说不定是邻居呢哈哈

**问什么时候要**
- 友好：不是特别急，但价格合适今天就可以拍～
- 专业：时间上比较灵活，价格合适即可交易。
- 随性：看价格呗，合适就现在
- 幽默：钱包说合适就现在，不合适就明天再问一遍😂

**问还要吗**
- 友好：要的要的！很喜欢，就等您松口了～
- 专业：确定要的，请问价格方面可以再商量吗？
- 随性：要啊！就看价格了
- 幽默：当然要！诚意满满的！就等老板给个好价了

## 超时跟进话术

**第一次跟进**
```
你好，还在吗？之前问的{商品}，{当前价}能出吗？我挺喜欢的～
```
```
老板～我又来了，{商品}考虑得怎么样？{当前价}可以的话随时拍哦
```

**第二次跟进**
```
嗨～最后问一次，{商品}{当前价}可以吗？可以的话我马上拍，不行也没关系～
```
```
老板在吗？还是想要这个{商品}，{当前价}是我的诚意价了，考虑下？
```

## 随机选择逻辑

LLM 在生成回复时，应从同一风格和阶段的多个模板中随机选择一个作为参考，然后结合上下文自然改写，避免每次回复完全相同。
