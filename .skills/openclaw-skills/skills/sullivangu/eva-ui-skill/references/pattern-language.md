# Pattern Language

## Core Motifs

### Stripe Bars

Use for alert emphasis, directional flow, and high-energy separators.

### Chevron Ladders

Use for escalation rails, lock-state reinforcement, and side columns.

### Tick Fields

Use for charts, scopes, and any area that needs measured tension.

### Grid and Mesh

Use for tactical maps, field simulation, and targeting surfaces.

### Dots and Dither

Use for low-contrast texture and density without large empty surfaces.

### Brackets and Reticles

Use for target framing, scoped focal points, and corner emphasis.

### Threshold Bands

Use for progress, pressure, sync, threat, and state-range communication.

## Composition Rules

- Each major panel should combine at least one data element and one structural motif.
- Pattern primitives should carry meaning: urgency, direction, state separation, or measurement.
- Stripe, chevron, and hatch motifs should not appear randomly.
