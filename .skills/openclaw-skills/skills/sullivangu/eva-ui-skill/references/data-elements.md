# Data Elements

## Readout Elements

- numeric counters
- tabular columns
- timestamps
- coordinate pairs
- serial identifiers
- short state strings

## Measurement Elements

- axis ticks
- calibration rulers
- min/max labels
- threshold markers
- segmented bars
- vertical level meters
- waveform traces
- ring scales for scope views

## Mapping Elements

- contour meshes
- grid coordinates
- target boxes
- crosshair centers
- sweep arcs
- direction arrows

## Annotation Elements

- leader lines
- side-callout labels
- caption clusters
- dense legend blocks
- reference IDs

## Easy To Miss But Important

- tiny calibration ticks that make a panel feel measured
- repeated mini-labels that create machine density
- threshold color zones behind values
- divider rhythms that separate blocks without cards
- side indexes and reference IDs that imply system depth
