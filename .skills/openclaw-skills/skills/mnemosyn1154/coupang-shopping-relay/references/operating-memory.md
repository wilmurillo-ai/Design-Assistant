# Operating Memory

이 문서는 쿠팡 relay 스킬이 실전에서 더 잘 동작하도록 성공/실패 패턴을 누적하는 방법을 정의합니다.

## 목적

쿠팡은 동적 UI, 탭 전환, 지연 로딩 때문에 같은 행동도 세션마다 안정성이 달라질 수 있습니다. 따라서 다음 정보를 누적해야 합니다:
- 잘 되는 읽기 패턴
- 흔들리는 쓰기 패턴
- 실패 증상
- 실제로 먹힌 fallback
- 특정 상품군에 대한 사용자 선호

## 어디에 저장할까

권장 순서:
1. 사용자가 지정한 작업 노트나 운영 문서
2. 현재 작업용 저장소 안의 별도 메모 파일
3. 파일 생성이 애매하면 현재 대화에 세션 요약을 남기고, 다음에 메모 파일 생성을 제안

주의:
- ClawHub로 설치된 스킬 디렉토리 자체를 운영 중에 수정하는 방식은 권장하지 않습니다.
- 설치된 스킬은 규칙 모음으로 보고, 실제 메모리는 사용자가 쓰는 워크스페이스에 둡니다.

## 절대 저장하지 말 것

- 쿠팡 비밀번호
- 카드번호
- 인증번호
- 전체 주소 전문
- 전화번호 전문
- 개인통관고유부호
- 결제 비밀번호

## 세션이 끝날 때 최소한 남길 항목

1. 오늘 안정적으로 된 작업
2. 오늘 실패하거나 흔들린 작업
3. 실패 증상
4. 먹힌 fallback
5. 다음 세션에서 바로 적용할 규칙

## 현재까지의 시드 메모

아래는 실제 테스트에서 이미 확인된 유용한 시드 규칙입니다.

상세한 실전 로그는 [improvement-log.md](improvement-log.md)에 따로 보관합니다. 이 문서는 그 로그에서 반복적으로 재사용할 운영 규칙만 추려 쌓는 용도입니다.

### Stable
- 브라우저 실행과 연결
- 쿠팡 접속
- 로그인 상태 확인
- 검색창 사용
- 검색 결과 읽기
- 장바구니 기존 품목 읽기
- 장바구니 일부 품목 삭제
- 화면 스크린샷 캡처

### Unstable
- 검색 결과 클릭 후 상세 페이지 이동 안정성
- 상세 페이지 장바구니 담기 후 반영 확인
- 장바구니 신규 품목 즉시 확인
- 동적 ref 변경
- 다중 탭 포커스 꼬임
- 장바구니 페이지 스냅샷 불완전 수집

### Recommended Operating Loop — 단건 모드
1. 로그인 상태 확보
2. 검색 결과에서 후보 탐색
3. 후보를 1~3개로 축소
4. 사용자에게 후보, 가격, 용량 제시
5. 사용자 승인 후 한 품목만 장바구니 담기
6. 장바구니 화면 캡처 및 재검수
7. 필요하면 다음 품목으로 반복
8. 결제 직전에는 대상 품목과 총액만 따로 정리

### Recommended Operating Loop — 배치 모드
사용자가 여러 재료/상품을 한 번에 요청한 경우 자동 전환.
1. 로그인 상태 확보
2. 재료 목록 파싱
3. 각 재료마다:
   a. 상세 페이지 탭 모두 닫고 쿠팡 검색 페이지에서 새로 시작
   b. 검색 실행 → wait → snapshot
   c. 자동 상품 선택 (로켓배송 우선, 광고 건너뛰기, 리뷰 많은 순, 옵션 없는 상품 우선)
   d. 상세 페이지 이동 → wait → snapshot → 장바구니 담기
   e. 1~2초 대기 → 성공 판정 (장바구니 가기 버튼/레이어, 아이콘 숫자 증가, 토스트 중 하나)
   f. 실패 시 기록하고 다음 재료로 진행
4. 전체 배치 완료 후 장바구니 이동 → wait → snapshot → 일괄 검증
5. 최종 결과 보고 (성공 품목, 실패 품목, 합계)
- 중간 보고 금지, 사용자에게 중간에 묻지 않음
- 사용자가 결과 보고 후 교체 요청 시에만 후보를 보여줌
- wait 없이 찍은 snapshot은 신뢰하지 않음

### Current Example Preference Rules
- 장바구니에는 식단 관련 품목과 비식단 관련 품목이 섞일 수 있으므로 분리 표시
- 닭가슴살 탐색 시 `생`, `냉장`, `400~600g`을 우선
- `수비드`, `훈제`, `실온보관`, `5kg 이상 대용량`, `100g x N 완제품 묶음`은 기본 후순위
- 장바구니 담기 직후에는 실패로 단정하기 전에 팝업, 혜택 레이어, `장바구니 가기` 버튼을 먼저 확인

### Current Known Cart Data
- 대추방울토마토 500g
- 국내산 브로콜리 2개입
- 더여백26 모델링팩 도구 3종 세트
- 더여백26 푸딩이 판테할마이드 5% 모델링팩 300g
- 하림 닭가슴살 1kg는 삭제된 것으로 확인

## 메모를 언제 갱신할까

반드시 갱신하는 경우:
- 장바구니 추가/삭제/수량 변경을 실제로 실행했을 때
- 탭 포커스 문제나 ref 문제를 겪었을 때
- 결제 직전 검수 화면에서 새 구조를 발견했을 때
- 특정 상품군에 대한 선호 규칙이 생겼을 때

읽기 전용 탐색만 했다면 짧은 요약만 남겨도 충분합니다.
