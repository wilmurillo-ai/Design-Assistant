# Coupang Relay Memory

이 파일은 쿠팡 Chrome relay 세션에서 성공/실패 패턴을 누적하기 위한 작업 메모 템플릿입니다.

민감정보는 적지 마세요:
- 비밀번호
- 카드번호
- 인증번호
- 전체 주소 전문
- 전화번호 전문
- 개인통관고유부호

## Stable
- 브라우저 실행과 연결
- 쿠팡 접속
- 로그인 상태 확인
- 검색창 사용
- 검색 결과 읽기
- 장바구니 기존 품목 읽기
- 장바구니 일부 품목 삭제
- 스크린샷 캡처

## Unstable
- 검색 결과 클릭 후 상세 페이지 이동 안정성
- 상세 페이지 장바구니 담기 후 반영 확인
- 장바구니 신규 품목 즉시 확인
- 동적 ref 변경
- 다중 탭 포커스 꼬임
- 장바구니 페이지 스냅샷 불완전 수집

## Working Fallbacks
- 스냅샷 재수집
- 탭 포커스 재정렬
- 장바구니 직접 이동
- 장바구니 스크린샷 첨부 후 사용자 재검수
- 장바구니 담기 직후 1~2초 성공 UI 확인 후 `장바구니 가기` 버튼 우선 사용

## Product Preferences
- 장바구니에는 식단 관련 품목과 비식단 관련 품목이 섞일 수 있으므로 분리 표시
- 닭가슴살 탐색 시 `생`, `냉장`, `400~600g`을 우선
- `수비드`, `훈제`, `실온보관`, `5kg 이상 대용량`, `100g x N 완제품 묶음`은 기본 후순위

## Current Shopping Rules
- 기본 루프는 `읽기 -> 후보 축소 -> 사용자 확인 -> 단일 액션 -> 다시 읽기`
- 한 번에 한 품목만 장바구니 쓰기 액션 수행
- 장바구니 담기 후에는 반드시 장바구니 페이지에서 재확인
- 결제 직전에는 이번 주문 대상 품목만 따로 정리
- 총액, 배송 시점, 결제수단 라벨을 사용자에게 요약
