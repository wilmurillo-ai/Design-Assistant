# SBTI

一键完成 SBTI 人格测评，生成专属报告，助力自我认知与职业规划。

## 技能说明

### 🔍 技能核心价值

SBTI（Sociability, Boldness, Thinking, Independence）人格测试，聚焦**社交性**、**果敢度**、**思维模式**、**独立性**四大核心维度，精准刻画人格特质。

本技能将专业测评体系 AI 化，无需复杂操作，即可快速完成测试并生成结构化报告，帮你深度认识自我、优化人际互动、明确职业方向，解决自我探索、职场适配、人际沟通等高频需求。

### 🛠️ 功能亮点

- **专业测评体系**：基于心理学理论设计 SBTI 标准题库，覆盖四大维度、16 种人格类型，测评结果科学可靠。

- **一键式操作**：用户仅需按提示完成选择题，无需额外资料，3-5 分钟即可完成全流程。

- **结构化报告输出**：自动生成包含维度得分解析、人格类型定义、优势短板、人际相处建议、职业适配方向的完整报告，可直接阅读、保存复用。

- **高复用性**：适用于个人自我探索、职场新人职业规划、团队成员性格适配分析等多场景，可反复使用。

- **稳定可交付**：输入清晰（仅需完成测评答题），输出格式固定，结果一致，无需反复追问用户信息。

### 📝 使用步骤

1. 安装本技能后，在对话中触发「SBTI 人格测试」

2. 按系统提示依次完成所有测评题目，凭第一直觉选择即可

3. 提交答案后，系统自动计算维度得分，10 秒内生成专属 SBTI 人格报告

4. 可直接查看报告全文，也可复制保存，用于后续自我复盘或分享参考

### 🎯 适用人群

- **普通用户**：渴望深度自我认知、探索个人特质

- **职场人士**：用于职业规划、岗位适配、团队协作优化

- **学生群体**：辅助专业选择、未来发展方向探索

- **人力资源从业者**：用于团队性格分析、人才适配参考

### ⚠️ 重要说明

- 本测试为心理学辅助测评工具，结果仅供自我参考与成长指引，不构成专业心理诊断或职业决策唯一依据。

- 测评结果基于用户答题选择，建议凭真实感受作答，以保证结果准确性。

- 技能持续迭代，后续将新增报告对比、人格类型深度解读、场景化建议等功能。

### ✅ 交付成果

- 完整 SBTI 人格测评报告（文本格式，含维度解析、类型定义、优势短板、人际/职业建议）

- 可复用的个人人格特质总结，支持多次测评对比分析

#### 📝 类型描述

**SOLO 孤儿**：自我价值感低、防御性强

你是一个内心矛盾的人。在人际关系中，你既有强烈的边界感（So2=H），渴望保持独立空间，但内心又缺乏自信（S1=L）。你不喜欢被规则束缚（A2=L），追求自由和舒适，但做事时又比较谨慎，倾向于避免风险（Ac1=L）。你的执行力偏弱，常常需要外部压力才能推动完成任务。在社交方面，你处于一种弹性状态，既不完全封闭，也不过分主动。
