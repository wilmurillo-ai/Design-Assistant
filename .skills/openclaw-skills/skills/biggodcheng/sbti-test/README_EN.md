# SBTI

Complete SBTI personality assessment with one click and generate an exclusive report to support self-awareness and career planning.

## Skill Description

### 🔍 Core Value

SBTI (Sociability, Boldness, Thinking, Independence) personality test focuses on four core dimensions: sociability, boldness, thinking patterns, and independence, precisely characterizing personality traits. This skill transforms the professional assessment system into an AI-powered solution—no complex operations required. Quickly complete the test and generate a structured report to help you deeply understand yourself, optimize interpersonal interactions, clarify career directions, and address high-frequency needs such as self-exploration, workplace adaptation, and interpersonal communication.

### 🛠️ Feature Highlights

- **Professional Assessment System**: Based on psychological theory, the SBTI standard question bank covers four dimensions and 16 personality types, delivering scientifically reliable results.

- **One-Click Operation**: Users only need to complete multiple-choice questions as prompted. No additional materials required—complete the full process in 3-5 minutes.

- **Structured Report Output**: Automatically generates a complete report including dimension score analysis, personality type definition, strengths & weaknesses, interpersonal relationship suggestions, and career compatibility directions. Ready to read, save, and reuse.

- **High Reusability**: Suitable for multiple scenarios including personal self-exploration, career planning for workplace newcomers, and team member personality compatibility analysis. Can be used repeatedly.

- **Stable & Deliverable**: Clear input (only requires completing assessment questions), fixed output format, consistent results, no need to repeatedly ask users for information.

### 📝 Usage Steps

1. After installing this skill, trigger "SBTI Personality Test" in Coze conversation.

2. Complete all assessment questions sequentially as prompted by the system. Trust your first intuition when selecting answers.

3. After submitting answers, the system automatically calculates dimension scores and generates your exclusive SBTI personality report within 10 seconds.

4. You can view the full report directly, or copy and save it for future self-reflection or sharing reference.

### 🎯 Target Audience

- General users who desire deep self-awareness and exploration of personal traits
- Workplace professionals: for career planning, role adaptation, and team collaboration optimization
- Students: assisting in major selection and future development direction exploration
- HR practitioners: for team personality analysis and talent compatibility reference

### ⚠️ Important Notes

- This test is a psychological auxiliary assessment tool. Results are for self-reference and growth guidance only, and do not constitute professional psychological diagnosis or the sole basis for career decisions.

- Assessment results are based on user's answer choices. It is recommended to answer based on true feelings to ensure result accuracy.

- The skill is continuously iterating. Future updates will include report comparison, in-depth personality type interpretation, scenario-based recommendations, and more features.

### ✅ Deliverables

- Complete SBTI personality assessment report (text format, including dimension analysis, type definition, strengths & weaknesses, interpersonal/career recommendations)
- Reusable personal personality trait summary, supporting multiple assessment comparisons
