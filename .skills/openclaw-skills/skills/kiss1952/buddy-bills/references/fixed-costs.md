# 固定支出配置规则

## 配置文件路径
`finance-records/固定支出配置.yaml`

## 自动录入规则

### 每月15号（工资发放日）自动录入
- 家庭转账（金额见配置）
- 医保（金额见配置）

### 每月29号自动录入（当月无29号则28号）
- 房贷（金额见配置）

### 年付项目（用户告知后手动录入）
- 物业费
- 车位费
- 到期前1个月提醒用户

## 配置文件格式

```yaml
固定支出列表:
  - 名称: 房贷
    金额: 0.00
    周期: 每月
    自动录入规则: 每月29号，无则28号
    账户: 银行自动扣款

  - 名称: 家庭开支转账
    金额: 0.00
    周期: 每月
    自动录入规则: 工资发放日（每月15号）
    收款人: 待填写

  - 名称: 医保
    金额: 0.00
    周期: 每月
    自动录入规则: 工资发放日（每月15号）

  - 名称: 物业费
    金额: 0.00
    周期: 年付
    录入规则: 用户告知后手动录入
    上次缴费: YYYY-MM-DD
    下次到期: YYYY年MM月

每月固定支出合计: 0.00
年度一次性固定支出合计: 0.00
```

## 收入配置文件路径
`finance-records/收入/收入配置.yaml`

```yaml
工资发放规则:
  发放日: 每月15号
  金额: 0.00
  年度预计总收入: 0.00
```
