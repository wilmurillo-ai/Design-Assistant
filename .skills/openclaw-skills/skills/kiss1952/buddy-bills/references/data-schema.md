# 数据目录结构与文件格式规范

## 目录结构

```
finance-records/
├── index.yaml                    # 总索引（月度汇总数字，快速统计用）
├── 固定支出配置.yaml              # 固定支出自动录入规则
├── summary/                      # 月度汇总文件
│   └── YYYY/
│       └── MM.yaml               # 每月最后一天生成
├── 餐饮/
│   ├── index.yaml                # 年度索引
│   └── YYYY/
│       └── MM.yaml               # 月度明细
├── 生活购物/
├── 医疗/
├── 通讯/
├── 房贷/
├── 物业费/
├── 车位费/
├── 家庭转账/
├── 医保/
└── 收入/
    ├── 收入配置.yaml
    └── YYYY/
        └── MM.yaml
```

## 总索引格式（index.yaml）

```yaml
YYYY:
  MM:
    总支出: 0.00
    房贷: 0.00
    家庭转账: 0.00
    餐饮: 0.00
    收入: 0.00
    结余: 0.00
```

## 月度汇总格式（summary/YYYY/MM.yaml）

```yaml
# 月度汇总 - YYYY年MM月

## 一、收支总览
收入:
  工资: 0.00
  总收入: 0.00
支出:
  房贷: 0.00
  总支出: 0.00
结余: 0.00
储蓄率: 0%

## 二、每周总结
### 第N周（MM月DD日 - MM月DD日）
  支出: 0.00
  收入: 0.00
  明细:
    - YYYY-MM-DD | 商户 | 分类 | 金额

## 三、支出明细汇总
### 分类名（金额）
  占总支出: 0%
  明细:
    - YYYY-MM-DD | 商户 | 金额

## 四、统计分析
支出排行:
  1. 分类: 金额（占比%）
最大单笔: 金额（商户）
固定支出: 金额
浮动支出: 金额

## 五、环比对比
  上月结余: 0.00

## 六、备注
```

## 分类明细格式（分类/YYYY/MM.yaml）

```yaml
# 分类名明细 - YYYY年MM月

月度汇总:
  总计: 0.00

明细:
  - 日期: YYYY-MM-DD
    商户: 名称
    子分类: 子类
    金额: 0.00
    支付方式: 待补充
    备注: 备注
```
