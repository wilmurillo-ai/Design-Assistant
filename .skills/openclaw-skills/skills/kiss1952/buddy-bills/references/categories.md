# 支出分类体系

## 一级分类及子分类

| 一级分类 | 子分类 | 示例 |
|----------|--------|------|
| 餐饮 | 早餐/午餐/晚餐/咖啡/外卖/聚餐 | 麦当劳、瑞幸、美团外卖 |
| 生活购物 | 超市/日用品/服装/家居 | 小象超市、淘宝、京东 |
| 医疗 | 药品/疫苗/挂号/体检 | 药店、医院、疫苗接种 |
| 通讯 | 话费/宽带/流量 | 话费充值、宽带续费 |
| 交通 | 加油/停车/打车/公交 | 加油站、滴滴、地铁 |
| 娱乐 | 电影/游戏/旅行/健身 | 电影票、KTV、健身房 |
| 教育 | 课程/书籍/培训 | 网课、书籍、培训班 |
| 房贷 | — | 银行房贷还款 |
| 物业费 | — | 年付物业管理费 |
| 车位费 | — | 年付停车位费用 |
| 家庭转账 | — | 家庭日常开支转账 |
| 医保 | — | 个人/家庭医疗保险 |
| 收入 | 工资/奖金/兼职/其他 | 月薪、年终奖、副业 |

## 智能分类规则

- 关键词"咖啡/奶茶/星巴克/瑞幸" → 餐饮-咖啡
- 关键词"外卖/美团/饿了么" → 餐饮-外卖
- 关键词"超市/便利店/菜市场" → 生活购物-超市
- 关键词"药/疫苗/医院/诊所" → 医疗
- 关键词"话费/流量/宽带" → 通讯
- 关键词"加油/停车/滴滴/打车" → 交通
- 关键词"工资/薪资/发工资" → 收入-工资
- 关键词"房贷/还贷/月供" → 房贷
