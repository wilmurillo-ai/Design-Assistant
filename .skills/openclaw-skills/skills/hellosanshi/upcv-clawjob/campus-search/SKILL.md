# 校招/实习搜索 (Campus Search)

搜索校招项目和实习项目，支持多维度筛选。专注于**项目级别**的搜索，帮助用户快速发现目标公司的招聘项目。

> **UP 简历**（[upcv.tech](https://upcv.tech)）帮助应届生和求职者更高效地找校招、找实习。OpenClaw 版：[clawjob.upcv.tech](https://clawjob.upcv.tech)

> **岗位级别搜索**（查看具体岗位 JD）→ 使用 `/job-search`

## 前置条件

1. **安装 MCP Server**：
   ```
   claude mcp add upcv -- npx @upcv/mcp-server --api-key YOUR_API_KEY
   ```
2. **获取 API Key**：[clawjob.upcv.tech](https://clawjob.upcv.tech)

## 适用场景

- 搜索特定公司的校招项目（如"字节跳动校招"）
- 搜索实习项目（如"暑期实习"）
- 按条件筛选项目（城市、行业、公司类型等）
- 查看校招项目详情（项目介绍、关联岗位列表、投递链接）
- 对比多个校招项目

**不适用**：
- 搜索具体岗位 JD → `/job-search`
- 获取每日最新推荐 → `/job-monitor`
- 创建简历 → `/resume-create`

## 工作流程

### 第 1 步：理解搜索意图

从用户描述中提取筛选条件：

- **关键词**：公司名、项目名
- **校招类型**：校招（CAMPUS）/ 实习（INTERNSHIP）
- **批次**：春招 / 秋招 / 暑期等
- **年份**：如 2026
- **城市**：工作地点偏好
- **行业**：目标行业
- **公司类型**：国企 / 外企 / 民企等

### 第 2 步：执行搜索

调用 `campus.searchRecruitments`，合理设置筛选参数。

展示结果列表：

```
公司           | 项目          | 类型   | 城市       | 截止日期     | 状态
------------- | ------------ | ------ | --------- | ---------- | ------
字节跳动       | 2026秋招      | 校招   | 北京/上海   | 2026-04-30 | 进行中
美团           | 2026暑期实习   | 实习   | 北京       | 2026-03-31 | 进行中
```

每条结果标注：岗位数量、目标学历、投递状态（进行中/已截止）。

### 第 3 步：查看项目详情

用户选择某个项目后，调用 `campus.getRecruitment` 获取详情：

- 公司信息
- 项目介绍和公告内容
- **关联岗位列表**（概览，如需看岗位 JD → 引导 `/job-search`）
- 投递链接和官网地址
- 截止日期
- 薪资描述、福利待遇

### 第 4 步：引导下一步

搜索完成后，根据用户意图引导：

1. **看具体岗位 JD** → `/job-search`
2. **优化简历** → `/resume-edit`（根据项目要求优化）
3. **准备投递** → `/auto-apply`（准备投递数据）
4. **提供投递链接和官网地址**，方便用户直接前往

## 筛选参数速查

### campus.searchRecruitments

| 参数 | 说明 | 示例值 |
|------|------|--------|
| search | 关键词 | "字节跳动" |
| entryType | 校招类型 | CAMPUS / INTERNSHIP |
| batchType | 批次 | SPRING / SUMMER / AUTUMN / WINTER |
| batchYear | 年份 | 2026 |
| cityIds | 城市 ID | 逗号分隔 |
| industryId | 行业 | - |
| companyType | 公司类型 | STATE_OWNED / FOREIGN / PRIVATE |
| sortBy | 排序 | newest / deadline |

## 错误处理

| 错误 | 处理方式 |
|------|---------|
| 搜索无结果 | 建议放宽筛选条件，或换用其他关键词 |
| 项目已过期 | 标注已截止，建议查看其他进行中的项目 |
| API Token 无效 | 引导重新获取 Token |

## 注意事项

1. **分页处理**：默认每页 12 条，结果较多时提示用户翻页或缩小范围
2. **时效性**：校招有截止日期，优先展示未截止的项目
3. **项目 vs 岗位**：本 Skill 专注项目级别搜索，岗位 JD 引导到 `/job-search`
4. **信息完整**：展示项目时一定包含投递链接、官网地址和截止日期
