# Changelog

## [1.0.0] - 2026-03-25

### Initial Release

- **Renamed from kie-image to kay-image**
- Clear dependency declaration in metadata
- Explicit credential requirements (KIE_API_KEY)
- Simplified documentation structure
- Added required env var declaration in SKILL.md frontmatter

### Features
- Text-to-image generation
- Image-to-image editing
- Image understanding (GPT-5, Gemini)
- Video understanding (Gemini)
- Multiple resolutions: 1K/2K/4K
- Multiple aspect ratios: 1:1, 3:4, 4:3, 9:16, 16:9, 21:9
