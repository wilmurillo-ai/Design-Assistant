## 情绪曲线控制

### 从丧到治愈（冷→暖）
```
整体色调从第一格到第四格渐变：
- 第一格：灰蓝、深褐，低饱和冷色
- 第二格：开始转暖，出现暖色点缀
- 第三格：蜜黄、暖橙，阳光照射效果
- 第四格：明亮温暖，蓝天白云
```

### 搞笑节奏（亮→更亮）
```
整体保持明亮活泼的色调，
通过表情和动作递进搞笑程度：
- 第一格：正常表情
- 第二格：疑惑/惊讶
- 第三格：尴尬/无奈
- 第四格：夸张搞笑/反转
```

## 快速生成公式

### 漫画 = 布局 + 分镜 + 角色 + 色调
```
[布局：四格漫画，2x2网格]
+
[分镜：每格画面+文字描述]
+
[角色：统一角色特征描述]
+
[色调：整体色调控制]
```

### 封面 = 主体 + 场景 + 情绪 + 色调
```
[主体：主要人物/元素]
+
[场景：背景环境]
+
[情绪：氛围描述]
+
[色调：色彩方案]
```
