## 📖 使用方式对比

### 旧方式（串行）- 不推荐

```bash
# 生成 4 张图，每张等完成后再生成下一张
# 总耗时：~8 分钟

kie-image -p "第一格..." -o comic-01.png --ar 3:4 -r 2K
# 等待 2 分钟...

kie-image -p "第二格..." -o comic-02.png --ar 3:4 -r 2K
# 等待 2 分钟...

kie-image -p "第三格..." -o comic-03.png --ar 3:4 -r 2K
# 等待 2 分钟...

kie-image -p "第四格..." -o comic-04.png --ar 3:4 -r 2K
# 等待 2 分钟...
```

### 新方式（批量）- 推荐 ⭐

```bash
# 1. 创建批量任务配置
cat > comic-tasks.json << 'EOF'
[
  {
    "prompt": "第一格...",
    "output": "comic-01.png",
    "ar": "3:4"
  },
  {
    "prompt": "第二格...",
    "output": "comic-02.png",
    "ar": "3:4"
  },
  {
    "prompt": "第三格...",
    "output": "comic-03.png",
    "ar": "3:4"
  },
  {
    "prompt": "第四格...",
    "output": "comic-04.png",
    "ar": "3:4"
  }
]
EOF

# 2. 批量生成（总耗时：~3 分钟）
kie-image-batch -b comic-tasks.json -o ./output

# 执行流程：
# 1. 批量提交 4 个任务（不等待）
# 2. 统一轮询（每 20s 检查一次，最多 10 次）
# 3. 全部完成后并行下载
# 4. 删除json文件
```
