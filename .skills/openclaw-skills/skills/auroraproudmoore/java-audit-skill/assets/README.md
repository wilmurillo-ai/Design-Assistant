# Assets Directory

This directory contains visual assets for the Java Audit Skill project:

- **Diagrams**: Architecture diagrams, flowcharts
- **Images**: Screenshots, example outputs
- **Charts**: Statistics, comparison charts

## Contributing

When adding assets, please:
1. Use descriptive filenames (e.g., `phase2-flowchart.png`)
2. Optimize image sizes for web viewing
3. Add alt text descriptions in documentation when referencing images
