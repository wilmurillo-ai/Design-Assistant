# 第1层：情绪入口

> 情绪是冰山的顶端，是进入潜意识的入口

---

## 任务

识别情绪 + 触发事件，为深度挖掘奠定基础

---

## 分析技巧

### 1. 情绪识别

**基础情绪**：
- 愤怒：边界被侵犯
- 焦虑：未来不确定性
- 羞耻：自我评价过低
- 抑郁：能量耗竭
- 孤独：连接缺失

**复合情绪**：
- 委屈 = 愤怒 + 羞耻
- 内疚 = 羞耻 + 责任感
- 嫉妒 = 羞耻 + 愤怒
- 恐惧 = 焦虑 + 威胁感

### 2. 触发事件分析

**5W1H法**：
- Who：谁触发了这个情绪？
- What：具体发生了什么？
- When：什么时候发生的？
- Where：在什么场景下？
- Why：为什么这个事件触发了你？
- How：你当时是怎么反应的？

### 3. 情绪强度评估

**1-10分量表**：
- 1-3：轻微，可忽略
- 4-6：中等，需要注意
- 7-8：强烈，需要处理
- 9-10：极端，可能需要专业帮助

---

## 常见模式

| 情绪 | 常见触发 | 可能的需求 |
|------|----------|------------|
| 愤怒 | 被批评、被忽视 | 尊重、认可 |
| 焦虑 | 不确定性、失控 | 安全、控制 |
| 羞耻 | 失败、被拒绝 | 接纳、归属 |
| 抑郁 | 长期压力、无望 | 意义、能量 |
| 孤独 | 缺乏连接、被误解 | 理解、陪伴 |

```markdown
- **触发事件**：[5W1H描述]

```
