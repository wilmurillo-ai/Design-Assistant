# 第3层：思维模式

> 恐惧是模式的燃料，场景是模式的触发器

---

## 任务

识别反复出现的恐惧和触发场景，揭示思维模式

---

## 核心理论

**思维模式**：在特定场景下自动激活的认知习惯

**形成机制**：
```
触发场景 → 核心恐惧 → 自动思维 → 情绪反应 → 行为反应
```

**模式特征**：
- 反复出现（跨时间、跨场景）
- 自动激活（无意识）
- 难以改变（固化）

---

## 5次追问（中间2次）

### 追问3：如果不被满足，你会怎么样？

**常见恐惧**：
- 被抛弃 → 孤独终老
- 被否定 → 一无是处
- 失控 → 灾难发生
- 失败 → 没有价值

**引导语**：
> "如果你[需求]不被满足，最坏的结果是什么？你最害怕的是什么？"

### 追问4：这个恐惧在哪个场景反复出现？

**常见场景**：
- 权威场景：面对领导、老师、长辈
- 亲密场景：恋爱、婚姻、家庭
- 社交场景：聚会、演讲、陌生环境
- 工作场景：考核、竞争、合作

**引导语**：
> "这种恐惧是不是在其他场景也出现过？能不能举出3个例子？"

## 常见思维模式

### 1. 灾难化

**特征**：把小事放大成灾难

**典型思维**：
- "如果失败，我的人生就完了"
- "如果不焦虑，坏事就会发生"

**触发场景**：不确定性、失控

### 2. 非黑即白

**特征**：只有极端，没有中间

**典型思维**：
- "要么完美，要么失败"
- "要么被爱，要么被抛弃"

**触发场景**：评价、比较

### 3. 过度负责

**特征**：把别人的情绪归咎于自己

**典型思维**：
- "他不开心一定是因为我"
- "我要为所有人的感受负责"

**触发场景**：冲突、批评

### 4. 受害者心态

**特征**：自己是被动的、无力的

**典型思维**：
- "都是他们的错"
- "我没办法改变"

**触发场景**：挫折、不公

## 分析技巧

### 1. 恐惧识别

**问自己**：
- 我最害怕的结果是什么？
- 这个恐惧是真实的还是想象的？
- 如果恐惧成真，真的有那么可怕吗？

### 2. 场景追踪

**问自己**：
- 这个模式第一次出现在什么时候？
- 最近一次出现在什么场景？
- 有没有哪个场景这个模式不会出现？

### 3. 模式命名

**给模式起个名字**：
- "我的批评过敏症"
- "我的完美主义陷阱"
- "我的受害者剧本"
