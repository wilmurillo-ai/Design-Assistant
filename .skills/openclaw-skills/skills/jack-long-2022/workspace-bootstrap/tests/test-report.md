## 🐛 发现并修复的问题

### 问题1：wizard.sh 目录结构未创建

- **现象**：`wizard.sh` 第77行报错 `No such file or directory`
- **原因**：wizard.sh 在创建 SOUL.md 前没有先创建目录结构
- **方案**：在 wizard.sh 中调用 bootstrap.sh 创建目录结构
- **影响场景**：场景1、场景3
- **状态**：✅ 已修复

### 问题2：SOUL.md 缺少小龙虾名称

- **现象**：测试验证失败，SOUL.md 不包含小龙虾名称
- **原因**：wizard.sh 生成的 SOUL.md 没有在身份部分包含 $AGENT_NAME
- **方案**：在 SOUL.md 的身份部分添加小龙虾名称
- **影响场景**：场景1
- **状态**：✅ 已修复

### 问题3：bc 命令缺失

- **现象**：`bc: command not found`
- **原因**：系统未安装 bc 命令
- **方案**：使用 awk 替代 bc 进行浮点数计算
- **影响场景**：场景2、test-all.sh
- **状态**：✅ 已修复
