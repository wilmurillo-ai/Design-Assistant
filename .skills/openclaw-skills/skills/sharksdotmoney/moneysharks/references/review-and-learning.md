# Review and Learning

Use post-trade review to improve recommendations without self-modifying core policy.

## Review inputs
- signal
- confluence
- confidence
- stop/target
- realized PnL
- slippage
- volatility regime
- market context

## Allowed adaptation
- lower confidence in hostile regimes
- recommend lower leverage after loss streaks
- flag underperforming setups
- produce review summaries

## Disallowed adaptation
- raising leverage caps automatically
- changing live-mode policy automatically
- removing stop-loss requirements
- rewriting hard rules without user approval
