# .source-research

This directory is the dedicated workspace data area for the `source-research` skill.

It stores canonical source-research results without polluting generic documentation directories.

## Subdirectories

- `source-pools/` — source pool definitions and status
- `acquisition/` — acquisition methods or programs by pool
- `filtering/` — filtering methods or programs by pool
- `high-quality-sources/` — filtered high-quality sources by pool
- `high-quality-information/` — high-quality information captured from those sources
- `rejections/` — rejection conclusions to avoid repeated analysis
- `programs/` — engineering assets, scripts, designs
