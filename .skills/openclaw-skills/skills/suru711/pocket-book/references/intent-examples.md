# Trigger Examples

Use this reference when the trigger boundary is ambiguous.

## Trigger

- `记账，午饭28`
- `午饭28`
- `地铁4元`
- `咖啡23，微信`
- `工资到账12000`
- `退款80`
- `招行转支付宝500`
- `今天花了多少`
- `本月餐饮多少`
- `最近三笔`
- `把上一笔改成建行`
- `撤销刚才那笔`
- `给上一笔补上备注：请同事喝咖啡`

## Confirm Before Trigger

- `28块`
- `120`
- `咖啡`
- `吃饭`

Use one short confirmation question when there is no clear ledger action.

## Do Not Trigger

- `记一下明天开会`
- `帮我设计一个记账系统`
- `我要写一个 expense tracker`
- `怎么做预算`
- `最近基金怎么样`
- `iPhone 16 多少钱`
- `把发票图片识别一下`
