# AutoDock VINA 使用手册参考
## 常用参数说明
| 参数 | 含义 | 默认值 | 推荐取值 |
|------|------|--------|----------|
| exhaustiveness | 搜索穷尽度，数值越大搜索越全面，耗时越长 | 8 | 一般任务8-16，高精度任务32 |
| num_modes | 输出的对接构象数目 | 9 | 1-10，按需设置 |
| energy_range | 输出构象的最大能量差（kcal/mol） | 3 | 2-5 |
| cpu | 使用的CPU核心数 | 全部可用核心 | 按需调整 |

## 结果解读
输出的打分是结合亲和力（Affinity），单位kcal/mol，数值越负表示结合能力越强：
- < -7 kcal/mol: 较强结合活性
- -5 ~ -7 kcal/mol: 中等结合活性
- > -5 kcal/mol: 结合活性较弱

## 注意事项
1. 蛋白输入必须是去水、去配体的PDB格式，若存在共晶配体建议提前移除
2. 口袋尺寸建议比实际结合口袋大2-5Å，避免截断配体构象
3. 对于含金属离子的蛋白，需要额外处理金属配位键，默认流程不支持
4. SMILES如果包含手性中心，需要明确指定手性，否则生成的构象可能不符合预期
