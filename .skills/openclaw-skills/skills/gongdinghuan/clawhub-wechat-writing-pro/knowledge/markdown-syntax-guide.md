## 一、基本语法（全平台兼容）

### 1. 标题

```markdown
# 一级标题 → <h1>
## 二级标题 → <h2>
### 三级标题 → <h3>
#### 四级标题 → <h4>
##### 五级标题 → <h5>
###### 六级标题 → <h6>
```

- 备用语法：`===`（一级）、`---`（二级）
- **最佳实践**：`#` 后加空格，标题前后加空行
- **公众号注意**：发布时已填标题，文内不重复 `#` 一级标题

### 2. 段落与换行

- 段落间用**空行**分隔
- 换行：行末两个以上空格 + 回车，或 `<br>`
- **不要**用空格/制表符缩进段落

### 3. 粗体与斜体

| 样式 | 语法 | 最佳实践 |
|:-----|:-----|:---------|
| **粗体** | `**文字**` | 用星号，不用下划线 |
| *斜体* | `*文字*` | 用星号 |
| ***粗斜体*** | `***文字***` | 用星号 |
| ~~删除线~~ | `~~文字~~` | 扩展语法 |

- 单词中间加粗用星号：`Love**is**bold`
- 中文加粗前后加空格

### 4. 引用 (Blockquotes)

```markdown
> 第一层引用
>> 第二层引用
>>> 第三层引用
```

- 引用内可包含标题、列表、代码块等任意元素
- 引用前后加空行

### 5. 列表 (Lists)

- 有序：`1.` 开始（数字不必连续，但建议从1开始）
- 无序：全用 `-`，**不混用** `- * +`
- 嵌套：缩进4空格
- 数字开头转义：`- 1968\.`
- 列表中添加段落/引用/代码块需缩进

### 6. 代码 (Code)

- 行内：`` `code` ``
- 含反引号：`` ``包含`反引号` `` ``（注意首尾空格）
- 代码块围栏：` ```python ` 或 `~~~html`
- 支持40+语言高亮
- **行内代码中反斜杠不转义**

### 7. 分割线

- 三个以上 `***`、`---`、`___`
- **上方必须空一行**，否则会被解析为标题

### 8. 链接 (Links)

```markdown
<!-- 行内链接 -->
[文字](URL "可选标题")

<!-- 自动链接 -->
<https://www.example.com>
<test@example.com>

<!-- 引用式链接（提升可读性） -->
[文字][标签]
[标签]: URL "标题"

<!-- 格式化链接 -->
**[粗体链接](url)**
*[*斜体链接](url)*
[`代码链接`](url)
```

- 标签不区分大小写
- URL 含空格用 `%20`，含括号用 `%28` `%29`
- **公众号**：仅 `mp.weixin.qq.com` 域名链接可直接跳转

### 9. 图片 (Images)

```markdown
![描述](image.jpg "可选标题")

<!-- 链接图片 -->
[![描述](image.jpg)](https://example.com)

<!-- 引用式图片 -->
![描述][标签]
[标签]: image.jpg "标题"
```

### 10. 转义字符

可转义：`\ ` * _ { } [ ] ( ) # + - . ! |

使用 `\` 转义，如 `\*` 显示为 `*`

### 11. HTML 元素

- 块级元素（div, table, pre, p）前后加空行
- 不要缩进 HTML 标签
- 块级 HTML 内不能使用 Markdown 语法

## 二、扩展语法（部分平台支持）

### 1. 删除线
`~~删除线~~` → <del>删除线</del>

### 2. 任务列表
```markdown
- [x] 已完成
- [ ] 未完成
```

### 3. 表情符号
- 直接复制粘贴 emoji
- 使用简码：`:joy:` → 😂（不同平台简码可能不同）

### 4. 上标与下标
- 上标：`X^2^` → X²（或 `<sup>2</sup>`）
- 下标：`H~2~O` → H₂O（或 `<sub>2</sub>`）

### 5. 自动 URL 链接
- URL 自动转为链接
- 反引号包裹可禁用：`` `http://example.com` ``

### 6. 表格
```markdown
| 左对齐 | 居中 | 右对齐 |
| :--- | :---: | ---: |
| 内容 | 内容 | 内容 |
```

- 每行开头结尾加 `|` 提升兼容性
- 单元格宽度不影响渲染
- 表格内不能用标题、引用、列表、图片
- 单元格内换行用 `<br>`
- 表格内列表用 `<ul><li>`
- 表格内竖线转义用 `&#124;`

### 7. 脚注
```markdown
正文内容[^1]

[^1]: 脚注说明
```
- 脚注以参考资料形式列在文末

### 8. 标题 ID
```markdown
### 自定义标题 {#custom-id}
```
- 用于目录跳转

### 9. 定义列表
```markdown
术语1
: 定义1

术语2
: 定义2
```

### 10. 高亮
```markdown
==高亮文本==
```

## 三、高级技巧

### 1. 下划线
```html
<ins>下划线文字</ins>
```

### 2. 缩进
```html
&nbsp;&nbsp;&nbsp;&nbsp;缩进内容
```

### 3. 居中
```html
<center>居中文本</center>
<p style="text-align:center">居中文本</p>
```

### 4. 颜色
```html
<font color="red">红色文字</font>
<p style="color:blue">蓝色文本</p>
```

### 5. 注释
```markdown
[注释]: # "这是隐藏注释"
```

### 6. 提示框（Admonitions）
```markdown
> :warning: **警告:** 这是警告提示
> :memo: **注意:** 这是注意提示
> :bulb: **技巧:** 这是技巧提示
```

### 7. 图片大小
```html
<img src="image.png" width="200" height="100">
```

### 8. 图片标题
```html
<figure>
  <img src="image.jpg" alt="描述" />
  <figcaption>图片标题</figcaption>
</figure>
```

### 9. 链接新窗口
```html
<a href="URL" target="_blank">新窗口打开</a>
```

### 10. 常用符号

| 代码 | 符号 |
|:-----|:-----|
| `&copy;` | © |
| `&reg;` | ® |
| `&trade;` | ™ |
| `&euro;` | € |
| `&larr;` | ← |
| `&uarr;` | ↑ |
| `&rarr;` | → |
| `&darr;` | ↓ |
| `&#176;` | ° |
| `&#960;` | π |

### 11. 表格格式化
- 换行：`<br>`
- 列表：`<ul><li>项目</li></ul>`

### 12. 目录（手动）
```markdown
- [章节1](#heading-id-1)
- [章节2](#heading-id-2)
```

### 13. 视频嵌入
```markdown
[![视频标题](封面图.jpg)](视频链接)
```

## 四、CommonMark 规范

- Markdown 的标准化规范
- 确保不同平台渲染一致
- 建议遵循 CommonMark 以保证兼容性
