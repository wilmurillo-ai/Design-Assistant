# Chinese audience guide (zh-CN / zh-HK)

## CLI language

Use `--lang zh-CN` or `--lang zh-HK` with `scripts/analyze.py` to translate **signals** and append a **short disclaimer** in Chinese. Deeper line-by-line translation can be extended via [`setup/config/i18n_glossary_zh.json`](../../config/i18n_glossary_zh.json).

## Terminology profiles

Persona YAML may set `terminology_profile` (`mixed`, `a_share`, `hk_listed`) for documentation and future template expansion. Numeric rows in reports remain English labels by default for scan-ability; extend the glossary for full UI parity.
