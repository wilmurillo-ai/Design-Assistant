# Xiaohongshu Viral Copy Generator

Generate engaging, viral-ready Xiaohongshu copy using proven templates and hooks.

## Quick Start

```
User: 帮我写一个职场种草文案
→ Generate: hook + body + CTA + hashtags
```

## Copy Structure

Each viral post follows this structure:

1. **Hook (开头钩子)** - 1-2句抓住注意力
2. **Body (正文)** - 3-4段分享价值
3. **CTA (互动引导)** - 1句引导评论
4. **Hashtags (话题)** - 5-8个相关话题

## Template Types

### 1. 种草安利型
- Hook: "姐妹们！XXX也太绝了吧！"
- Body: 分享使用体验、对比图、真实感受
- CTA: "你们用过的觉得怎么样？"

### 2. 避坑指南型  
- Hook: "千万别XXX！血泪教训！"
- Body: 踩坑经历 + 正确方法
- CTA: "你们有踩过类似的坑吗？"

### 3. 教程攻略型
- Hook: "3分钟学会XXX！"
- Body: 步骤1、2、3 + 要点总结
- CTA: "还想看什么教程？"

### 4. 情感共鸣型
- Hook: "XX的痛谁懂？"
- Body: 引发共鸣的故事 + 解决方法
- CTA: "评论区说说你的故事"

## Style Guidelines

- **Tone**: 口语化、亲切、真实
- **Emoji**: 适量使用，增强视觉
- **Length**: 300-500字最佳
- **Hashtags**: 核心词 + 流量词
