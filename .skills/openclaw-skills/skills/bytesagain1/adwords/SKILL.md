# Copywriter ✍️

营销文案全能助手，从标题到落地页一站搞定。

## 使用方式

| 需求 | 命令 | 说明 |
| Command | Description |
|---------|-------------|
| `headline` | 标题公式库(100个)+变体生成 |
| `body` | 正文撰写(多种风格) |
| `aida` | AIDA框架文案生成 |
| `pains` | 用户痛点挖掘+文案转化 |
| `testimonial` | 客户证言/评价文案 |
| `landing` | 完整落地页文案 |


## 运行

```bash
bash scripts/copy.sh <command> [args...]
```
