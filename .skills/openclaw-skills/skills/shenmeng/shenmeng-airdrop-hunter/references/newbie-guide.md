## ⚠️ 关键提醒

### 安全第一
1. **助记词** = 你的全部资产，绝对不能泄露
2. 只在官方渠道操作，警惕钓鱼网站
3. 不点击不明链接，不随意授权
4. 大额资金不放在撸毛钱包

### 防女巫（Anti-Sybil）
1. 同一个项目，不要多个账号互转
2. 交互时间要分散，不要集中操作
3. 使用不同的资金来源（如果需要主网交互）
4. 模拟真实用户行为

### 成本控制
- 测试网：零成本，只需要时间
- 主网交互：准备好 ETH 做 Gas 费
- 时间成本：每周5-10小时
