# 风险评估指南

## 反撸（Anti-Sybil）机制解析

### 什么是反撸

项目方通过技术手段识别并过滤"女巫攻击"（一人多号），确保空投分给真实用户。

### 常见检测手段

**1. 链上分析**
- 资金来源追踪（同一地址给多个地址充值）
- 交互模式分析（时间、金额、路径雷同）
- 合约调用相似度
- Gas设置模式

**2. 设备指纹**
- IP地址
- 浏览器指纹（Canvas, WebGL, Fonts）
- 硬件信息
- 时区、语言

**3. 社交验证**
- Twitter账号质量（注册时间、粉丝、活跃度）
- Discord行为模式
- 邮箱域名
- 手机号关联

**4. 行为分析**
- 操作速度（过快=机器人）
- 鼠标轨迹
- 页面停留时间
- 操作顺序

### 高风险信号

| 信号 | 风险等级 |
|------|----------|
| 同一交易所地址充值多个钱包 | 🔴 极高 |
| 交易时间高度集中 | 🔴 极高 |
| 交易金额完全相同 | 🔴 极高 |
| 交互路径完全一致 | 🔴 极高 |
| 使用相同IP/指纹 | 🟠 高 |
| 钱包间相互转账 | 🟠 高 |
| 社交账号新注册 | 🟡 中 |
| 操作过于规律 | 🟡 中 |

## 项目风险评估框架

### 评估维度

**1. 反撸严格度（40%）**
```
评分标准：
- 明确声明反撸：+2分（风险↑）
- 有KYC要求：+3分（风险↑↑）
- 社交账号验证：+1分
- 行为分析系统：+2分
- 历史反撸案例：+2分
```

**2. 空投确定性（30%）**
```
评分标准：
- 官方明确空投：-3分（风险↓）
- 社区猜测：+1分
- 无明确信号：+2分
- 已否认空投：+3分（但可能撒谎）
```

**3. 投入成本（20%）**
```
评分标准：
- 测试网零成本：-2分
- 需要少量Gas：+1分
- 需要大额资金：+2分
- 需要KYC：+3分
```

**4. 时间成本（10%）**
```
评分标准：
- 任务简单快速：-1分
- 需要持续数月：+1分
- 任务复杂耗时：+2分
```

## 具体项目风险分析

### Azuro - 反撸典型案例

**项目情况**：
- 预测市场基础设施
- 融资$25M
- 已发币（$AZUR）

**反撸措施**：
- 严格的多账号检测
- 积分系统频繁变更规则
- 大量账号被封禁

**结果**：
- 代币价格从$45跌至$6（-86%）
- 许多羊毛党颗粒无收
- 反撸成本极高

**教训**：
- 明确有反撸机制的项目，不适合批量
- 积分价值可能大幅波动
- 需要评估时间成本和机会成本

### LayerZero - 模糊空投

**项目情况**：
- 跨链互操作协议
- 融资$293M
- 估值$30亿

**争议**：
- 长期未明确空投计划
- 社区猜测不断
- 反女巫严格

**策略建议**：
- 少量精品号参与
- 不要All in
- 关注官方公告

## 风险控制策略

### 1. 分散原则

**项目分散**：
- 不要All in一个项目
- 同时关注5-10个项目
- 不同赛道配置

**账号分散**：
- 精品号、批量号分离
- 不同项目用不同组合
- 避免资金关联

### 2. 止损原则

**时间止损**：
- 设定最大投入时间
- 超过则放弃或减量

**资金止损**：
- 设定最大投入金额
- 严格遵守预算

**机会成本**：
- 定期评估是否值得继续
- 及时调整策略

### 3. 验证原则

**信息验证**：
- 只信官方渠道
- 验证链接真实性
- 警惕钓鱼网站

**项目验证**：
- 查证融资信息
- 确认团队背景
- 查看代码审计

### 4. 安全原则

**资金安全**：
- 使用硬件钱包（大额）
- 小额资金用于交互
- 定期归集/分散

**账号安全**：
- 不同项目不同密码
- 启用2FA
- 定期更换代理IP

## 常见陷阱识别

### 诈骗类型

**1. 钓鱼网站**
- 特征：URL相似但不同
- 防范：书签访问，仔细核对

**2. 假空投**
- 特征：要求先转账
- 防范：正规空投不需要你付钱

**3. 假客服**
- 特征：私信要求助记词
- 防范：官方不会索要私钥

**4. 假APP**
- 特征：非官方渠道下载
- 防范：官网下载，验证签名

### 心理压力

**FOMO（害怕错过）**：
- 症状：看到别人赚钱就冲动入场
- 对策：制定计划，严格执行

**沉没成本谬误**：
- 症状：已经投入了，就要继续投
- 对策：定期评估，及时止损

**过度自信**：
- 症状：认为自己不会被套
- 对策：保持谦逊，敬畏市场
