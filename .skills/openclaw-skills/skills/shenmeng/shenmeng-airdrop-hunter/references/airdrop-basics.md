# 空投基础知识

## 什么是空投（Airdrop）

空投是加密货币项目向用户免费分发代币的行为，目的包括：
- **冷启动**：快速获取用户和流动性
- **去中心化**：分散代币持有，避免中心化风险
- **奖励**：回馈早期支持者、测试网参与者
- **营销**：通过口碑传播获取新用户

## 空投类型

### 1. 追溯空投（Retroactive）
**特点**：奖励过去使用产品的用户
- 例子：Uniswap (2020)、1inch、dYdX
- 条件：在快照日期前使用过协议
- 策略：关注新协议，尽早使用

### 2. 任务空投（Task-based）
**特点**：完成特定任务获得资格
- 例子：LayerZero、zkSync
- 任务类型：交易、质押、桥接、社交任务
- 策略：按任务清单执行，注意时间成本

### 3. 测试网空投（Testnet）
**特点**：参与测试网获得主网代币
- 例子：Celestia、Aptos
- 优势：零成本或低成本
- 风险：不确定是否会发币

### 4. 持有空投（Holder）
**特点**：持有特定代币获得空投
- 例子：Ethereum Name Service (ENS)
- 条件：持有母链代币或特定NFT
- 策略：关注快照日期

### 5. 质押空投（Staking）
**特点**：质押代币获得新代币
- 例子：Cosmos生态空投
- 条件：质押ATOM、OSMO等
- 策略：定期查询质押者空投

## 空投时间线

```
项目启动 → 测试网 → 主网上线 → 快照 → TGE（代币生成）→ 空投发放
   ↑         ↑         ↑         ↑          ↑            ↑
  融资公布   零成本    需资金    最后机会   代币流通    变现时机
```

### 关键节点
- **融资公布**：开始关注
- **测试网上线**：开始交互
- **主网上线**：增加交互频率
- **快照公告**：最后冲刺
- **TGE（Token Generation Event）**：代币上线
- **Claim**：领取空投
- **Vesting**：解锁期（可能有锁仓）

## 常见空投标准

### 1. 交易量标准
- 达到一定交易量（如$10K+）
- 交易笔数（如5笔以上）
- 交易频次（如持续3个月）

### 2. 资金量标准
- TVL（锁仓量）
- 质押金额
- 流动性提供金额

### 3. 活跃标准
- 登录频次
- 交互的合约数量
- 使用的功能数量

### 4. 时间标准
- 早期用户（Early Adopter）
- 长期持有者（HODL）
- 持续活跃用户

### 5. 社交标准
- 关注社交媒体
- 加入Discord/Telegram
- 完成Galxe/Crew3任务

## 空投领取流程

### 1. 检查资格
- 访问项目官网或DappRadar
- 连接钱包查询
- 关注官方社交媒体公告

### 2. Claim代币
- 访问官方Claim页面
- 注意钓鱼网站（验证URL）
- 支付Gas费领取

### 3. 税务考虑
- 美国：空投视为收入，按收到时价值计税
- 其他国家：政策不同，需咨询专业人士
- 中国大陆：虚拟货币相关税务灰色地带

### 4. 变现策略
- **开盘即卖**：适合大多数项目
- **分批卖出**：25%→30%→45%
- **长期持有**：看好项目长期价值

## 多账号策略基础

### 账号矩阵
| 类型 | 数量 | 投入 | 策略 |
|------|------|------|------|
| 精品号 | 1-3个 | 高 | 深度交互，每个项目都做到位 |
| 批量号 | 10-50个 | 中 | 主要项目做，求覆盖率 |
| 羊毛号 | 50+个 | 低 | 只做零成本测试网 |

### 防关联要点
- **IP隔离**：不同IP，使用代理/VPN
- **浏览器指纹**：使用指纹浏览器（AdsPower, MoreLogin）
- **资金隔离**：不同钱包，资金不互转
- **行为差异化**：交互时间、金额、路径不雷同
- **社交隔离**：不同邮箱、Discord账号

## 常见术语

| 术语 | 含义 |
|------|------|
| **Sybil** | 女巫攻击，指一人多号刷空投 |
| **Anti-Sybil** | 反女巫检测，项目方识别多号的机制 |
| **Snapshot** | 快照，记录用户数据的时间点 |
| **TGE** | Token Generation Event，代币生成事件 |
| **Vesting** | 锁仓，代币逐步解锁的机制 |
| **Cliff** | 锁仓期，解锁前的等待期 |
| **Fair Launch** | 公平启动，无预售无VC的发行方式 |
| **追溯空投** | 奖励历史用户的空投 |
