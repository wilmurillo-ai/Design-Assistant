# 技术分析完整指南

## 核心指标详解

### 1. 移动平均线 (MA)

#### 简单移动平均线 (SMA)
```
SMA = (P1 + P2 + ... + Pn) / n
```
常用周期: 7日, 20日, 50日, 200日

#### 指数移动平均线 (EMA)
```
EMA = Price × k + EMA_昨日 × (1-k)
其中 k = 2 / (n+1)
```
**特点**: 对近期价格更敏感

#### 黄金交叉与死亡交叉
- **黄金交叉**: 短期 MA 上穿长期 MA → 看涨信号
- **死亡交叉**: 短期 MA 下穿长期 MA → 看跌信号

### 2. 相对强弱指数 (RSI)

```
RSI = 100 - (100 / (1 + RS))
RS = 平均涨幅 / 平均跌幅
```

**解读**:
- RSI > 70: 超买区域，可能回调
- RSI < 30: 超卖区域，可能反弹
- RSI 50: 多空分界线

**进阶用法**:
- 底背离: 价格新低但 RSI 未创新低 → 看涨
- 顶背离: 价格新高但 RSI 未创新高 → 看跌

### 3. MACD (移动平均收敛发散)

```
MACD 线 = 12日 EMA - 26日 EMA
信号线 = MACD 线的 9日 EMA
柱状图 = MACD 线 - 信号线
```

**信号**:
- MACD 上穿信号线 → 买入
- MACD 下穿信号线 → 卖出
- 柱状图由负转正 → 动能转强

### 4. 布林带 (Bollinger Bands)

```
中轨 = 20日 SMA
上轨 = 中轨 + 2 × 20日标准差
下轨 = 中轨 - 2 × 20日标准差
```

**应用**:
- 价格触及上轨 → 可能超买
- 价格触及下轨 → 可能超卖
- 带宽收窄 → 即将出现大行情（ squeeze ）
- 带宽扩大 → 波动加剧

### 5. 成交量指标

#### OBV (能量潮)
```
今日 OBV = 昨日 OBV + 今日成交量 (如果收盘上涨)
         = 昨日 OBV - 今日成交量 (如果收盘下跌)
```

#### 量价关系法则
1. **价涨量增** - 趋势健康，可继续持有
2. **价涨量缩** - 上涨乏力，警惕回调
3. **价跌量增** - 恐慌抛售，可能见底
4. **价跌量缩** - 无人接盘，继续观望

## 支撑阻力位计算

### 经典方法

#### 前高低点法
- **支撑位**: 前期低点
- **阻力位**: 前期高点
- **突破后**: 支撑变阻力，阻力变支撑

#### 斐波那契回撤
```
0%   - 波段高点
23.6%
38.2% - 弱支撑/阻力
50%   - 心理关口
61.8% - 黄金分割，强支撑/阻力
78.6%
100%  - 波段低点
```

#### 枢轴点 (Pivot Points)
```
枢轴点 (PP) = (前高 + 前低 + 前收盘) / 3

阻力位:
R1 = 2×PP - 前低
R2 = PP + (前高 - 前低)
R3 = 前高 + 2×(PP - 前低)

支撑位:
S1 = 2×PP - 前高
S2 = PP - (前高 - 前低)
S3 = 前低 - 2×(前高 - PP)
```

## 趋势判断

### 趋势线
```
上升趋势: 连接两个以上低点，向上倾斜
下降趋势: 连接两个以上高点，向下倾斜
```

### 趋势强度
- **ADX > 25**: 趋势较强
- **ADX > 50**: 趋势很强
- **ADX < 20**: 趋势较弱，可能震荡

### 多重时间框架分析
```
月线/周线 - 判断大趋势方向
日线     - 确定主要趋势
4小时    - 寻找入场点
1小时    - 精确入场时机
```

**规则**: 顺势交易，小周期服从大周期

## 实战策略

### 策略1: 趋势跟踪
```
买入条件:
1. 价格在 50MA 之上
2. 50MA 向上倾斜
3. RSI > 50
4. MACD 金叉
5. 成交量放大

卖出条件:
1. 价格跌破 20MA
2. RSI 顶背离
3. MACD 死叉
```

### 策略2: 均值回归
```
买入条件:
1. 价格触及布林带下轨
2. RSI < 30
3. 出现看涨K线形态
4. 成交量温和放大

卖出条件:
1. 价格触及布林带上轨
2. RSI > 70
3. 出现看跌K线形态
```

### 策略3: 突破交易
```
买入条件:
1. 价格突破关键阻力位
2. 突破时成交量放大 2 倍以上
3. 收盘站稳阻力位上方
4. RSI 在 50-70 之间（不超买）

止损设置: 突破K线低点下方 3%
目标位: 下一个阻力位或 1:2 盈亏比
```
