# Pump.fun 与新币发射分析指南

## Pump.fun 机制详解

### Bonding Curve 曲线机制

Pump.fun 使用 Bonding Curve（联合曲线）定价模型：

```
价格随供应量增加而指数上涨
- 早期买家价格低
- 后期买家价格高
- 流动性随着买入自动增加
```

**关键参数**:
- 创建费用: 0.02 SOL (~$3-5)
- 毕业阈值: $69,000 市值
- 毕业流动性: $12,000 自动存入 Raydium
- 流动性处理: 永久销毁（防止撤池子）

### 发射流程

```
1. 创建代币 (0.02 SOL)
   ↓
2. Bonding Curve 阶段
   - 任何人可以买入/卖出
   - 价格随供应量自动调整
   ↓
3. 达到 $69k 市值
   - 自动"毕业"到 Raydium
   - 创建永久流动性池
   - 开启更广泛交易
```

## 新币评估清单

- [ ] 合约地址是否可验证

- [ ] 是否有官方网站/社交媒体

- [ ] 总供应量是否合理
- [ ] 团队分配比例（<20% 为佳）
- [ ] 是否有增发/冻结权限
- [ ] 流动性是否锁定

### 社区热度检查 🔥
- [ ] Twitter/X 粉丝数量和活跃度
- [ ] Discord/Telegram 社区规模
- [ ] KOL 是否关注和提及
- [ ] 社交媒体讨论热度

### 交易数据分析 📊
- [ ] 交易量是否健康（> $100k/24h）
- [ ] 买卖比率是否平衡
- [ ] 是否有大额异常交易

## 风险信号 🚨

### 极高风险信号
1. **团队持仓 > 80%** - 随时可能砸盘
2. **合约未验证** - 可能存在后门/漏洞
3. **无社交媒体** - 无法追踪项目动态
4. **年龄 < 24小时** - 无历史数据可参考
5. **交易量异常低** - 缺乏真实买盘支撑

### 中等风险信号
1. **开发者钱包频繁操作** - 可能在做市或出货
2. **持有者高度集中** - 前10地址 > 60%
3. **价格波动极不稳定** - 可能是洗盘或操纵
4. **社交媒体突然静默** - 团队可能已放弃

## 关键指标解释

### 市值 (Market Cap)
```
市值 = 当前价格 × 流通供应量

解读:
- < $100k: 极高风险，可能归零
- $100k-$1M: 早期阶段，高波动
- $1M-$10M: 成长期，开始建立社区
- $10M+: 相对成熟，需要更强叙事支撑
```

### 流动性 (Liquidity)
```
流动性 = DEX 池中锁定的资金

健康标准:
- 流动性/市值 比率 > 0.1 为佳
- 流动性过低容易被操纵价格
- 毕业项目应有 $12k+ 锁定流动性
```

### 持有者分布
```
理想分布:
- 前10持有者 < 30%
- 前50持有者 < 60%
- 持有者数量持续增长

危险信号:
- 单个地址 > 20%
- 前10地址合计 > 70%
- 持有者数量停滞或下降
```

### 交易数据
```
关键指标:
- 24h 交易量 vs 市值比率（>0.1 为佳）
- 买卖比率（接近 1:1 健康）
- 大额交易占比（过高可能是操纵）
- 交易笔数（反映真实用户活跃度）
```

### BONK 模式
- 社区空投 Solana 生态用户
- 长期持续建设 DeFi 集成
- 从 Meme 转型为实用代币

### WIF 模式
- 简单但易记的 IP（戴帽子的狗）
- 持续社区活动和 Meme 传播
- 保持去中心化，无团队控盘

### POPCAT 模式
- 病毒式社交媒体传播
- 游戏化和互动体验
- 快速抓住热点时机

## 失败案例教训

### 常见死亡模式
1. **Rug Pull** - 团队撤走流动性
2. **Slow Rug** - 团队持续小额抛售
3. ** abandonment** - 团队放弃项目
4. ** Copycat** - 无创新的仿盘
5. ** Hype & Dump** - 仅炒作无实质

### 预警指标
- 团队钱包开始频繁转出
- 社交媒体更新频率下降
- 交易量持续萎缩
- 持有者数量持续下降
- 负面消息和社区 FUD
