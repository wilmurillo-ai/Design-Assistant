# Odds Movement Monitor 📊

盘口变化监控专家 - 追踪体育赛事亚盘、欧赔、大小球的实时变动。

## 核心能力

1. **实时监控** — 多平台盘口数据采集与对比
2. **变化检测** — 盘口升降、水位变动、异常波动识别
3. **趋势分析** — 历史数据追踪、变化轨迹可视化
4. **预警通知** — 关键变化自动提醒
5. **信号识别** — 大额注单信号、诱盘陷阱检测

## 适用场景

| 场景 | 说明 |
|------|------|
| **赛前监控** | 监控今晚比赛的盘口变化 |
| **异常检测** | 检测突然降盘的场次 |
| **对比分析** | 对比多个平台的盘口差异 |
| **趋势追踪** | 查看盘口变化历史 |
| **机会发现** | 发现盘口错位的比赛 |
| **风险控制** | 监控已投注比赛的盘口异动 |

## 监控维度

### 亚盘监控 (Asian Handicap)
- 盘口变化（-0.5 → -0.75）
- 水位波动（0.95 → 0.85）
- 临场变盘（开场前1小时）

### 欧赔监控 (European Odds)
- 胜平负赔率变化
- 凯利指数变动
- 机构分歧度

### 大小球监控 (Over/Under)
- 盘口升降（2.5 → 2.75）
- 水位变化趋势
- 临场升盘/降盘

## 使用方法

### 监控单场比赛
```
监控 [球队A] vs [球队B] 的盘口变化
```

### 批量监控
```
监控今晚所有英超比赛的盘口
```

### 异常检测
```
检测盘口异常变动的比赛
```

## 数据来源

- OddsAPI - 全球赔率聚合
- Bet365 - 主流平台数据
- Pinnacle - 专业玩家平台
- Smarkets - 交易所数据

## 输出格式

监控报告包含：
1. 初始盘口 vs 当前盘口
2. 变化幅度和时间线
3. 异常信号标记
4. 机构态度解读
5. 行动建议（观望/跟进/回避）
