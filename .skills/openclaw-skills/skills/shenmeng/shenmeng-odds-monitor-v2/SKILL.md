# 📊 Odds Movement Monitor

体育赛事赔率变动追踪助手 — 捕捉每一个赔率异动信号

## 核心能力

### 1. 赔率实时监控
- **亚盘追踪** - 实时监控让球盘变化（初盘→即时盘）
- **欧赔监控** - 追踪胜平负赔率变动趋势
- **大小球监控** - 监测大小球盘口和水位变化

### 2. 异动识别算法
- **显著变动** - 盘口升降超过0.25球
- **水位异常** - 赔率水位突然大幅调整
- **逆向变动** - 与市场预期相反的赔率调整

### 3. 联赛覆盖
- **主流联赛** - 英超、西甲、德甲、意甲、法甲
- **国际赛事** - 欧冠、欧联、世界杯、欧洲杯
- **亚洲赛事** - 中超、J联赛、K联赛、亚冠

### 4. 预警机制
- **即时推送** - 赔率异动实时通知
- **分级预警** - 按变动幅度分级（关注/重要/紧急）
- **自定义规则** - 设置个性化的预警条件

### 5. 数据分析
- **历史回查** - 查看任意比赛的赔率变化历史
- **机构对比** - 对比不同机构的盘口差异
- **趋势识别** - 识别赔率变化的趋势和模式

## 使用工作流

### 场景1: 关注特定比赛
当你想密切关注某场比赛的赔率变化时：
1. 输入比赛信息（联赛、对阵双方）
2. 选择监控的盘口类型（亚盘/欧赔/大小球）
3. 设置预警阈值
4. 接收实时赔率变动通知

### 场景2: 寻找价值投注机会
当你想发现赔率异动中的机会时：
1. 监控多场比赛的赔率变化
2. 识别异常变动（逆向变盘、水位异常）
3. 结合基本面分析判断机构意图
4. 捕捉最佳投注时机

### 场景3: 赛前赔率趋势分析
当你想分析赔率变化趋势时：
1. 查看比赛的赔率变化历史
2. 分析变盘原因（阵容变化、资金流向等）
3. 预测最终盘口走向

## 数据源

- **主流机构数据** - 实时采集多家主流机构赔率数据
- **覆盖范围** - 全球主要足球联赛和赛事
- **更新频率** - 数据实时更新，异动即时推送
