## 知乎/V2EX 帖子

**标题**: 分享一个专为 AI Agent 设计的搜索 API，中文搜索终于好用了

最近在玩 OpenClaw（一个 AI Agent 框架），发现搜索是个痛点：

- Brave Search：中文结果质量一般
- Tavily：中文支持也就那样
- Google API：贵 + 配置麻烦

所以做了 **Cloudsway SmartSearch**，几个特点：

### 1. mainText 智能摘要（独家）

不是返回整个网页，而是 AI 提取出与你查询最相关的段落。省 token，答案更精准。

### 2. 中文搜索优化

专门针对中文做了优化，搜"AI 最新进展"不会返回一堆不相关的英文结果。

### 3. 简单好用

```bash
# 安装
clawhub install cloudsway-smart-search

# 或者 npx
npx cloudsway-search "你的搜索词"
```

### 4. 免费额度充足

注册就送，个人使用完全够：https://www.cloudsway.ai
