# GB/T 9704-2012 党政机关公文格式

## 概述

GB/T 9704-2012 是中国国家标准《党政机关公文格式》，规定了党政机关公文（包括命令、决定、公告、通告、通知、通报、报告、请示、批复、意见、函、纪要等）的纸张要求、印刷装订要求、公文版式要求、公文各要素编排规则等。

## 适用范围

- 各级党的机关、政府机关
- 人大、政协、法院、检察院
- 人民团体、企事业单位

## 主要技术指标

### 纸张要求

| 项目 | 要求 |
|------|------|
| 纸张类型 | 纸白度为80%~90%，耐折度≥15次 |
| 纸张定量 | 60g/m²~80g/m² |
| 纸张规格 | A4型纸：210mm×297mm |

### 页边距要求

| 边距 | 要求 |
|------|------|
| 上白边 | 37mm |
| 下白边 | 35mm |
| 左白边 | 28mm |
| 右白边 | 26mm |

### 版心尺寸

- 版心尺寸：156mm×225mm（宽×高）

## 公文各要素及编排规则

### 公文标题

- **字体**：小标宋体或黑体
- **字号**：二号（约22磅）
- **编排**：居中排列，编排于红色分隔线下
- **多行标题**：行间距36磅，各行文字居中

### 公文正文

- **字体**：仿宋体
- **字号**：三号（约16磅）
- **每页行数**：22-24行
- **行间距**：28磅
- **每行字数**：28-30字

### 标题层次序号

| 层次 | 序号格式 | 字体 | 字号 |
|------|----------|------|------|
| 第一层 | 一、二、三、 | 黑体 | 三号 |
| 第二层 | （一）（二）（三） | 楷体 | 三号 |
| 第三层 | 1. 2. 3. | 仿宋 | 三号 |
| 第四层 | （1）（2）（3） | 仿宋 | 三号 |

### 附件

- 在正文下空一行，左空二字编排"附件"二字
- 附件序号用阿拉伯数字（如"1."）
- 附件标题居左空二字编排

### 落款（签发人/发文机关）

- 发文机关署名居右编排
- 印章顶端应上距正文或附件末行一行空
- 日期右空四字编排

## 公文特定格式

### 会议纪要

- 不显示红色分隔线
- 需标注"第X期"
- 发送范围标注在主题词下

### 命令（令）

- 令号："第X号"
- 正文不需标注主送机关
- 签发人职务标注在签发人姓名右下方

## 常见错误

1. **标题加粗** - 公文标题不应加粗
2. **页边距错误** - 未按标准设置页边距
3. **行距过密** - 正文行距应保持28磅
4. **序号混乱** - 三级标题使用"1、"格式（正确应为"1."）
5. **字体错误** - 使用宋体代替仿宋体
