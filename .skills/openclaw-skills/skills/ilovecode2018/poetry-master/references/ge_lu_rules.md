### 校验流程

1. **用户已给出词牌/体裁**：直接在搜韵网对应页面输入校验，选择正确的词牌和体式
2. **用户未给出词牌**：根据字数、句式、韵脚特征在搜韵网搜索匹配词牌，再校验
3. **校验结果优先**：搜韵网的校验结果为最终依据，本文档仅作辅助参考
4. **出错处理**：若脚本自动化失败，手动在浏览器中打开搜韵网页面进行校验

### 词谱多体问题

**这是最容易出错的地方！必须特别注意：**

同一词牌名下往往有**多种体式**（变体），例如：

| 词牌 | 代表体式 | 特殊差异 |
|------|---------|---------|
| 卜算子 | 苏轼体（钦谱正体） | 上片第三句末字可仄可平，不同体式有不同要求 |
| | 张先体 | 与苏轼体有细微平仄差异 |
| | 黄公度体 | 句式略有不同 |
| 行香子 | 晁补之体（钦谱正体） | 部分句末为仄声韵 |
| | 苏轼体（龙谱正体） | 上片五平韵、下片四平韵 |
| | 秦观体 | 前后段各五平韵 |
| 菩萨蛮 | 李白体（正体） | 上下阕各换韵（先仄后平） |
| | 温庭筠体 | 与李白体有平仄差异 |

**校验时必须：**
1. 询问或推测用户按哪家词谱/哪位词人的体式填词
2. 在搜韵网选择对应的体式进行校验
3. 若无法确定，以《钦定词谱》正体为默认参照，但需标注"按某某体校验"
4. **不确定时不轻易判定"出韵"或"格律错误"**

## 一、近体诗基本规则

### 1.1 四声与平仄

现代汉语四声对应古音关系：

| 现代声调 | 第一声 | 第二声 | 第三声 | 第四声 |
|---------|--------|--------|--------|--------|
| 调类 | 阴平 | 阳平 | 上声 | 去声 |
| 平仄 | 平 | 平 | 仄 | 仄 |

**入声字**（古音中归仄声，现代普通话已分化到各声调）需特别记忆。常见入声字举例：

- **现代读平声的入声字**：一、七、八、十、百、白、黑、独、读、福、服、竹、菊、足、局、国、德、得、合、达、别、决、觉、学、实、石、食、直、值、职、急、吉、级、及、集、即、七、疾、漆、习、席、袭、狄、笛、敌、跌、叠、蝶、帖、贴、铁、烈、劣、列、猎、裂、略、掠、录、六、陆、绿、鹿、目、木、牧、穆、谷、骨、哭、忽、窟、出、述、术、入、日、肉、热、柔、若、弱、局、菊、橘、曲、屈、戌、蓄、旭、育、欲、狱、浴、域、役、疫、译、驿、益、易、液、腋、业、叶、页

### 1.2 押韵规则

近体诗押韵规则严格：
- **偶数句必须押韵**（首句可押可不押）
- **一韵到底**，中间不得换韵
- **押平声韵**为主（仄韵诗较少见）
- 以《平水韵》为标准，共 106 韵

**平水韵常用韵部（30个平声韵）**：

|------|---------|

| 上平十一真 | 真人春神新尘身臣民宾滨频 |
| 上平十四寒 | 寒难丹单安欢端看宽酸团 |

### 1.3 对仗规则

律诗（八句）中：
- **颔联（3、4句）必须对仗**
- **颈联（5、6句）必须对仗**
- 首联（1、2句）和尾联（7、8句）一般不对仗

对仗要求：
- **词性相同**：名词对名词，动词对动词，形容词对形容词
- **句法结构相同**：主谓对主谓，动宾对动宾，偏正对偏正
- **意义关联**：同类事物相对，避免"合掌"（意思完全相同）

## 二、基本句式（四种平仄格式）

### 2.1 五言绝句四种格式

**格式一：仄起首句不入韵**
```
仄仄平平仄，平平仄仄平。（韵）
平平平仄仄，仄仄仄平平。（韵）
```

**格式二：仄起首句入韵**
```
仄仄仄平平。（韵）平平仄仄平。（韵）
平平平仄仄，仄仄仄平平。（韵）
```

**格式三：平起首句不入韵**
```
平平平仄仄，仄仄仄平平。（韵）
仄仄平平仄，平平仄仄平。（韵）
```

**格式四：平起首句入韵**
```
平平仄仄平。（韵）仄仄仄平平。（韵）
仄仄平平仄，平平仄仄平。（韵）
```

### 2.2 七言绝句四种格式

在五言基础上，句首加两字（与第一字平仄相反）：

**格式一：平起首句入韵**
```
平平仄仄仄平平。（韵）仄仄平平仄仄平。（韵）
仄仄平平平仄仄，平平仄仄仄平平。（韵）
```

**格式二：仄起首句入韵**
```
仄仄平平仄仄平。（韵）平平仄仄仄平平。（韵）
平平仄仄平平仄，仄仄平平仄仄平。（韵）
```

### 2.3 律诗（五律、七律）

律诗 = 两首绝句叠加，共八句四联。前四句与后四句分别遵循绝句格律，但中间两联（颔联、颈联）必须对仗。

七律平起首句入韵格式：
```
平平仄仄仄平平。（韵）仄仄平平仄仄平。（韵）
仄仄平平平仄仄，平平仄仄仄平平。（韵）  ← 颔联对仗
平平仄仄平平仄，仄仄平平仄仄平。（韵）  ← 颈联对仗
仄仄平平平仄仄，平平仄仄仄平平。（韵）
```

### 2.4 "一三五不论，二四六分明"

- 五言诗：第一、三字平仄可灵活，第二、四字必须严格
- 七言诗：第一、三、五字平仄可灵活，第二、四、六字必须严格
- **例外**：避免"孤平"（除韵脚外，整句只剩一个平声字），避免"三平调"（句末连续三个平声字）

## 三、古体诗（古风）

古体诗相对自由：
- 不拘平仄，不要求对仗
- 可换韵，可押仄声韵
- 句式灵活，可杂言（三、五、七言混用）
- 篇幅不限

常见形式：
- **五古**：每句五字，不拘平仄
- **七古**：每句七字，不拘平仄
- **乐府**：原为乐府机关所采集民歌，后文人仿作
- **歌行**：篇幅较长，句式灵活，常用"兮"字

## 四、宋词格律

### 4.1 词的基本概念

- **词牌名**：每首词的曲调名称，决定字数、句式、平仄、押韵
- **阕**：词的分段，上阕（上片）和下阕（下片）
- **小令**：58字以内
- **中调**：59-90字
- **长调**：91字以上

### 4.2 重要：词谱来源与多体问题

**⚠️ 词谱有多个权威来源，同一词牌在不同来源中的格律可能不同：**

| 词谱来源 | 全称 | 特点 |
|---------|------|------|
| **钦谱** | 《钦定词谱》 | 清康熙敕编，收 826 词牌 2306 体，最权威、收录最全 |
| **龙谱** | 龙榆生《唐宋词格律》 | 现代通行，选取常用体式，简洁实用 |
| **白谱** | 白香词谱 | 清代舒梦兰编，选 100 调，入门常用 |

**三者差异举例**：
- 《行香子》：龙谱以苏轼体为正格（上片五平韵），钦谱以晁补之体为正格（部分句末仄声韵）
- 《卜算子》：钦谱收多体（苏轼、张先、黄公度等），龙谱仅列一体
- 《菩萨蛮》：钦谱收数十体，不同体式的换韵位置和句式均有差异

**校验原则**：
1. **用户给出了词牌**：以搜韵网校验为准，自动选择正确的词谱和体式
2. **用户未给出词牌**：根据字数、句式特征搜索匹配，再用搜韵网校验
3. **不确定时不轻判**：若无法确定具体体式，标注"格律待考"而非直接判错

### 4.3 常用词牌格律（仅作参考，以搜韵网校验为准）

> ⚠️ 以下词谱仅列最常用的一种体式，不代表唯一正确格式。同一词牌可能有多种合法体式。

#### 忆江南（27字，单调）
```

中仄中平平仄仄，中平中仄仄平平。（韵）
中仄仄平平。（韵）
```
*示例：白居易《忆江南》*

#### 如梦令（33字，单调）
```

中仄仄平平，中仄中平平仄。（韵）
平仄，（叠）平仄，（叠）中仄中平平仄。（韵）
```
*示例：李清照《如梦令·常记溪亭日暮》*

#### 卜算子（44字，双调）

#### 行香子（66字，双调）

### 4.4 词韵

词韵以《词林正韵》为标准，将平水韵合并为 19 部：
- 第一部至第十四部为平声韵（部分韵部包含仄声）
- 第十五部至第十九部为入声韵

词的押韵比诗灵活：
- 可平韵到底、可仄韵到底
- 可换韵（如菩萨蛮、清平乐）
- 入声韵在词中大量使用（如满江红、念奴娇）

### 4.5 未标注词牌时的自动识别

当用户未给出词牌名时，可通过以下特征辅助识别：

1. **数总字数**：统计全词字数（不含标点）
2. **查句式**：记录每句的字数序列，如卜算子为 [5,5,7,5,5,5,7,5]
3. **看韵脚**：分析韵脚位置和平仄，区分平韵体和仄韵体
4. **在搜韵网搜索**：使用搜韵网的"词谱查询"功能，按字数和句式匹配

常见词牌字数速查：

## 五、常见格律错误类型

### 5.1 诗病

| 诗病 | 说明 | 示例 |
|------|------|------|
| **孤平** | 除韵脚外，整句仅一个平声字（主要针对仄收句） | 仄仄**平**仄仄（应为仄仄平平仄） |
| **三平调** | 句末连续三个平声字 | 仄仄**平平平**（应避免） |
| **失粘** | 相邻两联之间，第二句与第三句第二字平仄相同（应相反） | 第二句"平平仄仄平"，第三句"平平平仄仄" |
| **失对** | 一联之内，出句与对句第二字平仄相同（应相反） | 出句"平平平仄仄"，对句"平平仄仄平" |
| **重韵** | 同一韵脚字重复出现 | — |
| **凑韵** | 为押韵而勉强用不恰当的字 | — |
| **合掌** | 对仗句意思完全重复 | "春风"对"东风" |
| **出韵** | 押了非本韵部的字 | 东韵中混入冬韵字 |

### 5.2 点评重点关注项

1. **体裁判断**：先判断是古体诗还是近体诗，再判断具体体裁
2. **格律检验**：近体诗需逐字检查平仄、押韵、对仗
3. **古体宽容**：古体诗不对平仄对仗作严格要求，但韵脚仍需合理
4. **词谱对照**：宋词需对照具体词牌格律检查，**必须先确认体式**
5. **搜韵网校验**：对于词作，建议在搜韵网进行精确校验，避免单一词谱误判
