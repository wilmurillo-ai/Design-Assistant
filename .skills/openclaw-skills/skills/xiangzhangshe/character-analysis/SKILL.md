# Character Analysis - 形象性格分析（升级版）

通过面部/手部特征 + 行为科学，分析性格倾向，给出实用建议。

## 核心特色（升级版）

- 🔮 **传统分析**：面相/手相特征解读
- 📊 **行为经济学**：风险偏好、决策风格、认知偏误
- 🧠 **行为心理学**：大五人格、行为模式、动机分析
- 💡 **实用建议**：工作、人际、理财、个人成长

---

## 免责声明

⚠️ **本分析仅供娱乐参考，不代表科学结论。不涉及健康、命运预测。如有重要决策需求，请咨询专业人士。**

## 核心工作流

**步骤1：接收信息**
- 用户上传面部照片或手部照片
- 询问性别（确定手相：男左女右）
- 说明分析性质：趣味参考，非专业预测

**步骤2：分析流程**
```
1. 传统特征分析（面相/手相）
   ↓
2. 行为经济学分析
   - 风险偏好评估
   - 决策风格识别
   - 认知偏误诊断
   ↓
3. 行为心理学分析
   - 大五人格评分
   - 行为模式识别
   - 动机分析
   ↓
4. 综合建议生成
   - 工作职业建议
   - 人际关系建议
   - 理财投资建议
   - 个人成长建议
```

## 注意事项

1. **安全第一**：每次输出必须有免责声明
2. **科学态度**：行为经济学/心理学内容需注明"基于行为科学研究"
3. **不说绝对话**：用"可能""倾向""通常"，不用"一定""注定"
4. **正向引导**：即使看到"负面"特征，也转化为成长建议
5. **保护隐私**：照片分析完不保存，不用于其他用途
6. **拒绝迷信**：不预测具体事件（婚姻次数、死亡时间、具体收益等）
7. **适度专业**：行为经济学内容要有深度，但保持易懂
