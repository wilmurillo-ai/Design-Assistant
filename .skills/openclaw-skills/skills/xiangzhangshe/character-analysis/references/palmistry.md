# 手相特征库

基于传统手相学的特征分析，转化为性格倾向描述。

---

## 一、手型

| 手型 | 特征 | 性格倾向 |
|------|------|----------|
| **方形手** | 手指方正，指节分明 | 务实、理性、注重实际、缺乏浪漫 |
| **圆形手** | 手指圆润，手掌柔软 | 感性、善于社交、适应力强、容易被影响 |
| **长形手** | 手指修长，手掌窄 | 艺术气质、思考型、敏感、追求完美 |
| **锥形手** | 手指尖细，手掌宽 | 创意丰富、行动力强、善于表达 |
| **混合型** | 各特征混合 | 性格多元，适应能力强 |

## 二、掌丘

| 掌丘 | 特征 | 性格倾向 |
|------|------|----------|
| **金星丘**（拇指根部） | 厚实 | 活力充沛、热情、爱冒险 |
| **木星丘**（食指根部） | 厚实 | 有野心、领导力、权力欲 |
| **水星丘**（小指根部） | 厚实 | 财运好、善于理财、商业头脑 |
| **太阳丘**（无名指根部） | 厚实 | 艺术天赋、追求荣誉、爱美 |
| **月亮丘**（小指下方） | 厚实 | 想象力丰富、直觉强、情绪化 |
| **土星丘**（中指根部） | 厚实 | 责任感强、保守、稳定 |
| **火星丘**（拇指与食指之间） | 厚实 | 行动力强、勇气、有魄力 |

## 三、三大主线

### 生命线

| 特征 | 性格倾向 |
|------|----------|
| **深长清晰** | 生命力旺盛、健康意识强、意志坚定 |
| **浅淡** | 活力不足、容易疲劳、动力不够 |
| **有岛纹** | 健康有波折、需注意身体 |
| **呈弧形** | 性格开朗、善于社交 |
| **呈直线** | 务实、踏实 |
| **断裂** | 人生有重大转折或健康危机 |

### 智慧线

| 特征 | 性格倾向 |
|------|----------|
| **深长** | 思考深入、理性、有分析能力 |
| **浅短** | 感性、直觉强、行动力强但思考浅 |
| **延伸向月丘** | 想象力丰富、艺术气质、情绪化 |
| **延伸向食指丘** | 有野心、功利、理性 |
| **呈波浪** | 思想活跃、创意多、容易分心 |
| **呈直线** | 务实、实际、专注 |

### 感情线

| 特征 | 性格倾向 |
|------|----------|
| **深长清晰** | 重视感情、专一、善于表达 |
| **浅淡** | 感情淡薄、理性、独立 |
| **有岛纹** | 感情有创伤、容易受伤 |
| **呈链状** | 感情用事、情绪波动大 |
| **延伸到食指** | 追求精神共鸣 |
| **延伸向水星丘** | 物质与情感并重 |

## 四、手指

| 手指 | 特征 | 性格倾向 |
|------|------|----------|
| **拇指** | 较长 | 意志力强、有主见、自律 |
| **拇指** | 较短 | 随和、灵活、缺乏毅力 |
| **食指** | 较长 | 有野心、权力欲、领导力 |
| **中指** | 较长 | 有责任感、追求成就、自律 |
| **无名指** | 较长 | 艺术天赋、追求美感、爱美 |
| **小指** | 较长 | 沟通能力强、商业头脑、财运 |
| **手指并拢** | 缝隙小 | 谨慎、守秘密、理性 |
| **手指分开** | 缝隙大 | 开放、创意、灵活 |

---

## 使用说明

1. **男左女右**：男性分析左手，女性分析右手
2. **综合判断**：多条线结合判断，不只看一条
3. **正向解读**：负面特征转化为成长建议
4. **避免预测**：不预测具体事件（寿命、婚姻次数等）
