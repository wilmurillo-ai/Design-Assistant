# 面相特征库

基于传统面相学的特征分析，转化为性格倾向描述。

---

## 一、整体面部

### 脸型

| 脸型 | 特征 | 性格倾向 |
|------|------|----------|
| **圆脸** | 丰满圆润 | 为人亲切、善于社交、容易相处，但可能缺乏决断力 |
| **方脸** | 棱角分明 | 意志坚定、原则性强、有领导力，但可能过于固执 |
| **长脸** | 瘦长 | 思考型、善于分析、专注力强，但可能过于严肃 |
| **椭圆脸** | 标准 | 性格平衡、人缘好、适应力强 |

### 额头

| 特征 | 性格倾向 |
|------|----------|
| **宽广** | 思维活跃、创意丰富、志向远大 |
| **狭窄** | 务实、专注、眼光可能不够长远 |
| **高** | 智慧、抽象思维能力强 |
| **低** | 接地气、注重现实 |
| **饱满** | 精力充沛、积极向上 |
| **有皱纹** | 经验丰富、思考深入 |

## 二、眉毛

| 特征 | 性格倾向 |
|------|----------|
| **浓密** | 意志力强、有主见、可能固执 |
| **稀疏** | 随和、灵活、但可能缺乏自信 |
| **长** | 友情为重、善于维护关系 |
| **短** | 行动派、急性子、缺乏耐心 |
| **粗** | 果断、魄力、领导力 |
| **细** | 细腻、敏感、艺术气质 |
| **眉尾上扬** | 积极向上、不服输 |
| **眉尾下垂** | 悲观、消极、容易放弃 |
| **眉毛压眼** | 自我要求高、追求完美 |

| 特征 | 性格倾向 |

| **大眼** | 好奇心强、善于观察、桃花运好 |
| **小眼** | 专注、务实、占有欲强 |

## 使用说明

1. **综合判断**：单一特征不能代表全部，需要综合多个特征
2. **正向转化**：即使特征看起来"负面"，也要转化为成长建议
3. **避免绝对**：用"倾向""可能"等词
4. **结合现代**：把传统术语转化为现代人容易理解的语言
