# Quality Checklist — Chapter Review & Scoring

Run this checklist on every chapter before presenting for approval.

---

## Language Quality
- [ ] Zero forbidden AI phrases found
- [ ] Em-dashes: max 1 per 1,000 words
- [ ] No repetitive paragraph openings (check first words)
- [ ] No triple-adjective stacks
- [ ] No "In summary/conclusion" endings
- [ ] Contractions used naturally in dialogue
- [ ] Active voice dominant (passive <15% of sentences)
- [ ] Adverbs (-ly words) minimized
- [ ] Specific details over generic descriptions
- [ ] No repeated words within same paragraph (except articles/pronouns)
- [ ] No overuse of "suddenly," "just," "really," "very"

## Continuity
- [ ] Opens with forward momentum (no chapter recap)
- [ ] Builds on previous chapter events/arguments
- [ ] Characters show development from earlier chapters
- [ ] No facts/explanations repeated from prior chapters
- [ ] Running themes/metaphors maintained
- [ ] Unresolved threads from earlier chapters acknowledged
- [ ] Timeline consistency with prior chapters

## Genre Compliance
- [ ] Age-appropriate language (children's books)
- [ ] Medical claims sourced (health non-fiction)
- [ ] Scripture properly integrated (faith books)
- [ ] Technical instructions verified (AI/tech books)
- [ ] Tone matches target audience throughout
- [ ] Genre conventions honored (romance beats, mystery clues, etc.)

## Engagement
- [ ] Chapter ending creates forward pull (not a neat conclusion)
- [ ] At least one surprising/delightful moment
- [ ] Pacing varies appropriately (tension > release > tension)
- [ ] Dialogue feels natural and distinct per character
- [ ] Would pass the "read aloud" test
- [ ] First paragraph hooks immediately

## Structure
- [ ] Word count within target range for this chapter
- [ ] Follows approved outline for this chapter
- [ ] No tangents or deviations from blueprint
- [ ] Proper scene breaks where needed
- [ ] Front matter references correct (if applicable)

## Final Check
- [ ] Read one random paragraph — does it sound human?
- [ ] Read the opening — would you keep reading?
- [ ] Read the closing — do you want the next chapter?

---

## Scoring System

When reviewing a chapter, score on these 6 dimensions (1-10):

| Dimension | Weight | What It Measures |
|-----------|--------|-----------------|
| Humanization | 25% | Absence of AI tells, natural voice, authentic feel |
| Continuity | 20% | Builds on previous chapters, no repetition, forward momentum |
| Genre Fit | 15% | Matches target audience and genre conventions |
| Craft | 15% | Sentence variety, imagery, pacing, dialogue quality |
| Content Accuracy | 15% | Factual correctness, proper sourcing, no misleading claims |
| Engagement | 10% | Would a reader want to keep going? |

## Verdicts

- **8-10 average**: APPROVED — proceed to approval gate
- **6-7 average**: REVISE — specific notes back to writer, one rewrite attempt
- **Below 6**: REWRITE — writer starts the chapter fresh with new instructions

**Minimum requirement: 7 average with no single dimension below 5.**

## Review Report Format

```
# CHAPTER [X] REVIEW — [Chapter Title]

## Verdict: [APPROVED / REVISE / REWRITE]

## Scores
- Humanization: X/10
- Continuity: X/10
- Genre Fit: X/10
- Craft: X/10
- Content Accuracy: X/10
- Engagement: X/10
- **Weighted Average: X.X/10**

## What Works
[2-3 specific strengths]

## What Needs Improvement
[Specific, actionable notes — not vague feedback]

## Specific Rewrite Instructions (if REVISE/REWRITE)
[Exact passages, sections, or patterns to fix]
```
