# meeting-note — 会议/谈话纪要（Zettelkasten 友好）

探讨/决策型会议与谈话纪要：结论、共识/分歧、决策轨迹、隐含假设、风险与机会、行动项，并接入 Zettelkasten 知识网络。适用于整理会议纪要、复盘讨论、把多方探讨变成可执行资产。信息/学习型谈话（培训、分享会、单向汇报）请改用 deep-learning。

## When to Use

- 多方**探讨、决策**的会议与讨论（需结论、共识/分歧、决策轨迹、行动项）
- 关键词：会议纪要、会议、谈话、讨论、meeting notes

## Workflow at a Glance

| Step | 内容 |
|------|------|
| 0 | 执行协议：TODO 拆解 → 逐项执行 |
| 1 | 会议基础信息与重要度（L3/L2/L1） |
| 2 | 多轮阅读：议题切分 → 主讲人/关键决策者专项 → 深层倾听与权力动态 |
| 3 | 逐议题结构化（结论/共识/非共识/过程/隐藏/风险/待讨论） |
| 4 | 关键信息原子化（决策/洞察/假设·风险原子）+ Zettelkasten 连接 |
| 5 | 行动项与跟进（🔴🟡🟢，可衡量+责任人+截止时间） |

输出模板见 `references/meeting_note_output_template.md`。
