## 快速选择指南

**按意图快速选择：**

- **新功能** → `:sparkles:`
- **修 Bug** → `:bug:` 或 `:ambulance:`（紧急）
- **改文档** → `:memo:`
- **重构** → `:recycle:`
- **删代码** → `:fire:`
- **改 UI** → `:lipstick:` 或 `:art:`（结构）
- **上线** → `:rocket:`
- **测试** → `:white_check_mark:` 或 `:test_tube:`（失败测试）
- **改配置** → `:wrench:`
- **依赖** → `:arrow_up:` / `:arrow_down:` / `:heavy_plus_sign:` / `:heavy_minus_sign:`
- **CI/CD** → `:construction_worker:` 或 `:green_heart:`（修复）
- **安全** → `:lock:`
- **回滚** → `:rewind:`
- **破坏性变更** → `:boom:`
