# 经济数据类型详解

## 高重要性数据

### 非农就业数据 (NFP)
- **发布时间**：每月第一个周五 21:30（北京时间）
- **重要性**：🔴 高
- **利多特征**：实际值 > 预期
- **影响**：直接影响市场流动性和风险偏好
- **单位**：+200K, +250K

### CPI 消费者价格指数
- **发布时间**：每月中旬 21:30（北京时间）
- **重要性**：🔴 高
- **利多特征**：实际值 < 预期（通胀压力缓解）
- **影响**：直接影响美联储政策预期
- **单位**：2.3%, 2.4%

### FOMC 利率决议
- **发布时间**：每季度（周三）02:00（北京时间）
- **重要性**：🔴 高
- **利多特征**：维持不变或降息
- **影响**：决定基准利率，影响全球市场
- **单位**：5.25-5.50%

### GDP 季度报告
- **发布时间**：每季度末 21:30（北京时间）
- **重要性**：🔴 高
- **利多特征**：实际值 > 预期
- **影响**：经济增长核心指标
- **单位**：2.0%, 1.8%

### 失业率
- **发布时间**：每月第一个周五 21:30（北京时间，与非农同日）
- **重要性**：🔴 高
- **利多特征**：实际值 < 预期（就业市场改善）
- **影响**：就业市场健康状况指标
- **单位**：3.8%, 4.0%

## 中等重要性数据

- **重要性**：🟡 中
- **利多特征**：实际值 > 预期
- **影响**：非农先行指标
- **单位**：+180K, +195K

- **单位**：50.5, 52.0

### 服务业 PMI

- **影响**：服务业景气度

### PPI 生产者价格指数
- **发布时间**：每月中旬 21:30（北京时间）

- **影响**：通胀先行指标

## 利多利空判断逻辑

### 利多型指标（数值越大越利多）
```
实际值 > 预期值 (+2% 以上) → 利多 🟢 (强)
实际值 > 预期值 (0-2%) → 利多 🟢 (中等)
实际值 ≈ 预期值 (±2%) → 中性 🟡
实际值 < 预期值 (-2% 以上) → 利空 🔴 (强)
```

### 利空型指标（数值越小越利多）
```
实际值 < 预期值 (-2% 以上) → 利多 🟢 (强)
实际值 < 预期值 (0-2%) → 利多 🟢 (中等)
实际值 ≈ 预期值 (±2%) → 中性 🟡
实际值 > 预期值 (+2% 以上) → 利空 🔴 (强)
```

## 市场影响强度

- **强** 💪 - 偏差 > 2%，影响显著，可能引发大波动
- **中等** ⚖️ - 偏差 0-2%，有一定影响，市场有一定反应
- **弱** 📊 - 偏差 ≈ 0%，影响有限，市场反应不大

## 对加密市场的影响

### 利多数据的影响
- 市场情绪转向积极
- 风险偏好上升
- 可能推动加密货币价格上涨
- 交易量可能增加

### 利空数据的影响
- 市场情绪转为谨慎
- 风险偏好下降
- 可能导致加密货币价格下跌
- 可能引发避险情绪

### 中性数据的影响
- 市场反应有限
- 价格可能小幅波动
- 技术面分析更重要
- 关注其他因素

## 使用建议

1. **关注发布时间** - 提前做好仓位管理
2. **实际值 vs 预期值** - 偏差越大影响越显著
3. **结合前值** - 了解数据趋势
4. **综合判断** - 不要只看单一数据
5. **风险管理** - 数据发布前后谨慎操作
