# EvoMemory Core - Evolutionary Local Memory System
Full-stack evolutionary local memory system for OpenClaw AI Agents.

## What is EvoMemory?
EvoMemory is not just a vector database plugin, it's a complete, production-ready local memory system designed for AI Agents. It's built on top of our mature ChromaDB plugin, with extended capabilities for hybrid retrieval, data management, and Chinese semantic optimization.

## Why choose EvoMemory?
- **Complete solution**: No need to install multiple plugins, get all memory capabilities out of the box
- **Dual backend**: Switch between ChromaDB and LanceDB freely according to your needs
- **Chinese optimized**: Native support for BGE-M3 Chinese semantic vector model
- **100% offline**: No external API calls, all data stays on your local machine
- **Backward compatible**: Works with existing OpenClaw memory interface, zero migration cost

## Quick Start
1. Run `install.bat` (Windows) or `./install.sh` (Linux/macOS)
2. Run `evomemory test` to verify installation
3. Done! Your memory system is ready to use
