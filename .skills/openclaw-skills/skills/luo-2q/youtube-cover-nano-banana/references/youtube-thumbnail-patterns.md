# YouTube Thumbnail Patterns

## Use This Reference

Use this file when the user's text is long, abstract, or weakly visual and you need help turning it into a thumbnail concept.

## Prompt Scaffold

Use this base structure and adapt it:

```text
Create a high-conversion YouTube thumbnail in 16:9 format. Show [main subject] as the dominant focal point, [emotion / action], framed in a [camera framing] shot. Use a [style] visual treatment with [lighting], [color palette], and a clean [background] that supports the subject without clutter. Leave clear negative space for large headline text on [left/right/top/bottom]. The composition should feel bold, simple, highly clickable, and readable on mobile. Emphasize strong contrast, clear separation, and a single obvious story beat.
```

## Strategy Selection

Choose the angle that makes the user click fastest:

- `Shock`: mistakes, danger, loss, warning, exposed truth
- `Desire`: improvement, money, body, status, speed, convenience
- `Curiosity`: hidden mechanism, unexpected comparison, withheld reveal
- `Authority`: proof, data, expertise, premium result
- `Transformation`: before/after, weak/strong, messy/clean, old/new

If several apply, choose one primary angle and one supporting cue.

## Overlay Text Rules

- Prefer 2 to 4 words
- Use verbs, numbers, or emotionally loaded nouns
- Avoid filler words
- Avoid repeating the full title
- Avoid multiple lines unless the user explicitly wants text-heavy composition

Examples:

- `Stop Doing This`
- `AI Replaced It`
- `10X Faster Setup`
- `Why It Fails`
- `Real Profit Math`

## Domain Cues

### Finance

- Use sharp lighting, clean typography space, premium dark or metallic accents
- Prefer charts, money, laptops, trading screens, documents, or confident expert framing

### Tech / AI

- Use futuristic screens, device close-ups, blue/cyan accents, controlled glow
- Prefer hands-on interaction, dashboards, automation, or model outputs

### Fitness

- Use strong body language, sweat, effort, muscular tension, before/after framing
- Prefer high-energy lighting and warm highlights

### Education

- Use clear authority, explanation posture, whiteboard/screen/book cues
- Prefer less chaos and more clarity

### Business / Marketing

- Use confident presenter framing, dashboard elements, growth cues, high-contrast text zone
- Prefer money, conversion, audience, or strategy symbolism without clutter

## Bad Prompt Smells

Revise the prompt if it:

- Describes more than two major subjects
- Requires the model to render long text exactly
- Uses generic phrases like "make it eye-catching" without visual detail
- Reads like a movie poster or website hero image instead of a thumbnail
- Contains too many small props
- Has no emotional signal

## Example Transformations

### Example 1

Input copy:

`90% of creators waste time on thumbnails and still get low CTR`

Output angle:

`Expose a painful thumbnail mistake with a frustrated creator and a failing performance cue`

Overlay text:

`Low CTR?`

Prompt:

```text
Create a high-conversion YouTube thumbnail in 16:9 format. Show a frustrated content creator at a desk staring at a disappointing analytics screen with a large red downward CTR indicator as the dominant visual story. Medium close-up framing, expressive face, one hand on forehead, dramatic contrast lighting, modern creator-economy style, dark desk setup with bright screen glow, red and yellow accents for urgency, clean background with minimal studio elements, and clear negative space on the left for large headline text. The image should feel bold, simple, emotional, and instantly readable on mobile.
```

### Example 2

Input copy:

`This AI workflow cuts my video production time from 8 hours to 45 minutes`

Output angle:

`Show speed and leverage through an AI-powered setup and a dramatic time comparison`

Overlay text:

`8h -> 45m`

Prompt:

```text
Create a high-conversion YouTube thumbnail in 16:9 format. Show a confident creator using an AI-powered desktop workflow as the main subject, with a dramatic visual time comparison from 8 hours to 45 minutes represented by large simple interface cues, medium shot, energetic pose, sleek modern tech aesthetic, bright screen glow, cyan and orange accent colors, cinematic studio lighting, clean workstation background, and clear empty space on the right for large headline text. Keep the composition simple, high contrast, highly clickable, and optimized for mobile readability.
```
