# User Growth Coach v2.2

> 三层反馈复盘系统：即时反馈 → 历史模式识别 → 日常上下文注入 → 深层行为洞察

## 概述

不只是记录复盘，而是一个**连接你所有输入的智能反馈系统**。通过三层分析，自动发现你没意识到的行为模式，并给出基于心理学的机制解释和具体行为处方。

## 🚀 Quick Start

安装后，直接在聊天中发送：

```
复盘
```

然后输入你想记录的内容即可。Skill 会自动记录并给出即时反馈。

**更多触发词：**
- `复盘 深度` — 全量三层分析
- `周盘` — 自动加载 7 天上下文
- `情绪` — 情绪拆解
- `成长汇总` — 洞察版五段式汇总
- `帮助` — 显示所有可用命令

**推荐配置（可选）：**
设置 Cron 任务，每天 22:30 自动生成 Daily Digest，23:00 发送复盘提醒。详见 `references/reminders.md`。

## ✨ 核心特性

### 三层反馈架构

```
L1 ──── 当前输入 ──── 即时反馈（所有模式）
         │
L2 ──── 输入间关联 ── 模式识别、承诺追踪、情绪趋势、深层模式洞察（标准+深度）
         │
L3 ──── 日常上下文 ── 当天对话摘要注入，关联你的实际行动（仅深度模式）
```

### 🆕 深层模式洞察

当同一行为模式在 7 天内出现 ≥ 2 次时，自动触发深层洞察，每个洞察包含三层解释：

```
💡「预设困难逃避」
是什么：还没动手就在脑子里预演各种障碍，用"太难了"来合理化不行动。
为什么：大脑倾向高估未知任务的难度来保护你不去冒险——损失厌恶的变体。
怎么办：默认先动手试 30 秒。大多数预设的困难在动手瞬间就消失了。
```

- 洞察数量不限制，有多少给多少
- 同一模式不重复解释，只追踪行为变化
- 用户可说"展开讲讲"进入完整深度解释模式

## 反馈模式

| 模式 | 层级 | 适用场景 |
|------|------|---------|
| `快速` | L1 | 快速记一笔，不想被打断 |
| `标准`（默认）| L1 + L2 | 日常复盘，带历史关联和模式洞察 |
| `深度` | L1 + L2 + L3 | 全量分析，关联当天所有交互上下文 |

## L2 模式检测能力

| 检测项 | 说明 |
|--------|------|
| **重复主题** | 同一标签 7 天内出现 ≥ 2 次 |
| **承诺追踪** | 自动提取承诺，后续复盘追问是否兑现 |
| **情绪趋势** | 连续多天的情绪变化方向 |
| **维度失衡** | 工作/健康/社交等维度的投入比例偏差 |
| **矛盾检测** | 前后复盘内容的语义矛盾 |
| **深层模式洞察** | 重复行为的心理机制解释（是什么/为什么/怎么办） |

## 洞察版汇总（五段式）

触发 `成长汇总` 或 `复盘汇总` 后，输出：

1. **今日关键洞察** — 跨天模式识别（模式级）
2. **上下文关联** — 日常行动与复盘的深度关联
3. **承诺追踪** — ✅已兑现 / ⚠️未兑现 / 🔄进行中
4. **To-Do** — 1-3 条，含场景 + 动作 + 完成定义
5. **明日关注点** — 单指标，可观测

## 配置

| 变量 | 默认值 | 说明 |
|------|--------|------|
| `OPENCLAW_SESSIONS_DIR` | `~/.openclaw/agents/main/sessions` | Session transcript 目录 |

## 设计理念

### 为什么不只是记录？

传统的复盘工具只做"记录"——你写下来就完了。但人类最大的盲区不是"忘了记"，而是**"看不到自己的模式"**。

这个 skill 的核心价值是：
- **L1**：确保你每次复盘都有即时反馈，不是自说自话
- **L2**：自动发现你 7-30 天内的重复行为、未兑现承诺、情绪波动
- **L3**：把你当天的实际行动（做了什么、搜了什么、和谁聊了什么）和复盘内容交叉对比

### 深层模式洞察的价值

人对自己的行为模式通常是"知道但做不到"。深层洞察不只是指出模式，还解释**为什么你的大脑会这么干**（心理机制），并给出一条**具体到场景的制动规则**。
