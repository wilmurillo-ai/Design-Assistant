# Reminders (建议配置)

## Default Schedule

> 以下为建议时间，可根据个人作息调整。

| 任务 | 建议时间 | 说明 |
|------|---------|------|
| Daily Digest 提取 | 每天 22:30 | 为 L3 准备当日上下文数据 |
| 日复盘提醒 | 每天 23:00 | 提醒用户开始复盘 |
| 周复盘提醒 | 每周日 20:30 | 周盘 |
| 月复盘提醒 | 每月最后一天 21:00 | 月盘 |

## Reminder Message Style
- 简短、无说教
- 示例：
  - `今天收尾了吗？复盘。`
  - `周盘时间到：进步/失误/下周聚焦。`
  - `月盘：给四维打分并定下月主线。`

## Channel
优先当前聊天渠道。

## Execution Note
定时任务建议由 OpenClaw Cron 配置执行，消息内容只发触发词（如"复盘"），由 Skill 自动展开流程。
