## 📋 标准化操作流程

### 步骤1：任务分析
```
输入：任务描述
输出：是否记忆相关操作
检查点：
1. 任务是否包含"记忆"、"总结"、"错误"、"经验"等关键词？
2. 是否需要记录或检索信息？
```

### 步骤2：系统检查
```
输入：任务描述
工具：python memory_system_enforcer.py "任务描述"
输出：检查结果
检查点：
1. 是否发现禁止操作？
2. 是否正确使用update_memory工具？
```

### 步骤3：正确操作
根据任务类型选择正确操作：

#### 类型A：记录错误教训
```
正确操作：使用update_memory工具
模板：
- 标题：错误类型 + 简要描述
- 内容：错误详情 + 原因分析 + 正确做法 + 检查清单
```

#### 类型B：记录成功经验
```
正确操作：使用update_memory工具
模板：
- 标题：成功类型 + 简要描述
- 内容：成功详情 + 关键因素 + 可复制经验 + 应用场景
```

#### 类型C：检索相关记忆
```
正确操作：使用start_work.py脚本
命令：python start_work.py "任务描述"
输出：相关记忆报告
```

### 步骤4：验证执行
```
检查点：
1. 是否使用了update_memory工具？
2. 是否创建了md文件？（必须为否）
3. 是否100%执行了主人指令？
```
