# 市场罗盘分析报告模板

---

## 报告格式规范

### 禁止事项
- 禁止出现任何字段名（cs_temp、above_ma60_ratio、cs_hy_dispersion 等），一律使用中文名
- 禁止使用"一定"、"必然"、"肯定"等绝对化表述
- 禁止对未来走势做确定性预测

### 必须遵守
- 结论先行：第一部分必须是核心结论，通俗直白，不超过 3 句话
- 指标现状表：用表格呈现所有指标当前值和状态
- 详细分析：每个维度说清楚"数值是多少 → 说明什么 → 背后的市场演绎是什么"
- 操作建议：明确、可执行，不模糊

## 报告质量检查清单

生成报告后，逐项检查：

- [ ] 核心结论是否通俗直白，普通人能看懂
- [ ] 指标现状表是否完整，所有字段都有填写
- [ ] 是否有任何字段名出现在报告正文中（cs_temp 等）
- [ ] 趋势结构分析是否说清楚了"数值→含义→市场演绎"三个层次
- [ ] 是否识别并标注了指数与广度的背离（如有）
- [ ] 是否识别并标注了单日极端跳升（如有）
- [ ] 估值层是否说明了对趋势结论的修正作用
- [ ] 情绪层是否描述了情绪的演绎过程而不只是当前值
- [ ] 操作建议是否明确可执行
- [ ] 是否包含免责声明
