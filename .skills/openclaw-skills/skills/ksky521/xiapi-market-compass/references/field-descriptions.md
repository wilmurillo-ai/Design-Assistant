# 市场罗盘字段说明

## 数据命令

```bash
daxiapi market compass
```

返回最近 20 个交易日的罗盘数据，每行一个交易日。

---

## 字段对照表

| 返回字段名 | 中文名 | 类型 | 值域 | 说明 |
|---|---|---|---|---|
| 日期 | 日期 | date | YYYY-MM-DD | 交易日期 |
| 估值温度 | 估值温度 | float | 0–100 | 全市场PB中位数的历史百分位，反映当前估值在历史上的高低位置 |
| 恐贪指数 | 恐贪指数 | float | 0–100 | 市场情绪综合指数，0为极度恐慌，100为极度贪婪 |
| 趋势温度(60日) | 中期趋势广度 | float | 0–100 | 全市场收盘价在MA60之上的股票占比（%），反映中期趋势的健康程度 |
| 趋势指标(20日) | 短期趋势广度 | float | 0–100 | 全市场收盘价在MA20之上的股票占比（%），反映短期趋势的扩散程度 |
| 趋势指标(200日) | 长期趋势广度 | float | 0–100 | 全市场收盘价在MA200之上的股票占比（%），反映长期牛熊结构 |
| 动量温度 | 动量温度 | float | 0–100 | 全市场CS中位数的历史百分位，反映当前市场动量在历史上的高低位置 |
| CS中位数 | CS中位数 | float | 约-20至+20 | 全市场股票收盘价相对EMA20的对数乖离率中位数，公式：(ln(close)-ln(EMA20))×100，正值表示股价在均线上方，负值表示在下方 |
| CS离散度 | 行业分化度 | float | 0–5+ | 各行业CS均值的标准差，数值越大说明行业间涨跌分化越严重 |
| 全A指数 | 全A指数 | float | - | 万得全A指数点位，作为市场整体价格锚点 |

---

## 指标分层归属

### 趋势结构层（权重最高，决定仓位方向）

**广度组**：
- 长期趋势广度（MA200占比）：判断长期牛熊结构，变化极慢，是底层状态判断
- 中期趋势广度（MA60占比）：核心指标，判断中期趋势是否健康，是仓位决策的主要依据
- 短期趋势广度（MA20占比）：判断短期趋势扩散程度，变化快，需结合中期趋势广度看差值

**弹性组**：
- 动量温度：判断当前市场动量在历史上的位置，是超买还是超卖
- CS中位数：判断全市场股票相对均线的偏离程度，正值过大说明短期过热
- 行业分化度：判断市场热度是均匀分布还是集中在少数行业

### 估值层（修正趋势结论的力度）

- 估值温度：判断当前市场贵不贵，修正趋势结论的激进程度

### 情绪层（判断操作时机）

- 恐贪指数：判断投资者情绪状态，识别情绪拐点

### 价格锚点

- 全A指数：所有指标的价格参照系，用于识别指数与内部结构的背离

---

## 重要注意事项

**CS中位数与动量温度的关系**：两者都基于CS（收盘价相对EMA20的乖离率）计算，但含义不同。CS中位数是当天的绝对值，反映当前位置；动量温度是CS中位数的历史百分位，反映当前位置在历史上的高低。两者结合使用：CS中位数告诉你"现在在哪"，动量温度告诉你"这个位置历史上有多极端"。

**行业分化度的方向性**：行业分化度本身没有方向性，高分化不等于风险，也不等于机会。必须结合中期趋势广度来判断：中期趋势广度健康时的高分化是结构性机会，中期趋势广度破坏时的高分化是局部反弹，操作策略完全相反。

**三层广度的层次关系**：正常市场应满足长期趋势广度 ≥ 中期趋势广度 ≥ 短期趋势广度。出现倒挂（尤其是短期趋势广度远高于中期趋势广度）时，说明短期反弹力度超过中期结构修复速度，需要在报告中特别标注。
