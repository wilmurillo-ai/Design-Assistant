# CS 指标说明

## 什么是 CS 指标

CS（EMA20乖离率）是收盘价与 EMA20（20日指数移动平均线）的乖离率，用于衡量板块短期强度。EMA20 对近期价格赋予更高权重，是众多交易者关注的重要趋势参考线，突破 EMA20 对于板块而言是判断短期趋势的重要信号。

- CS > 0：价格在 EMA20 上方，短期偏强，多头力量占优
- CS < 0：价格在 EMA20 下方，短期偏弱，空头力量占优
- CS 绝对值越大：短期偏离程度越高，需警惕过热或超卖

## CS 阈值解读

| CS 范围 | 状态 | 操作建议 |
|---|---|---|
| > 15 | 过热 | 不追高，等待回调 |
| 5–15 | 领涨 | 可考虑入场 |
| 0–5 | 正常 | 关注 |
| -5–0 | 偏弱 | 观望 |
| < -5 | 深度偏弱 | 建议离场或观望 |

## 板块分类标准

### 领涨板块

- CS > 5，且 `sector top` 验证领涨股数量 ≥ 3 只
- 若 CS > 5 但领涨股 < 3 只，属于个股行情带动，不称为板块行情
- CS > 15 标注过热，不建议追高

### 上升板块

- 出现在 `cs_crossover_ma20_names` 中（今日 CS 刚突破 CS_MA20）
- 或在 `csHeatmap` 历史序列中近2日以上 CS 方向连续为 ↑
- 优先使用预处理字段，直接可信

### 反转板块

- CS < 0 且近2日 CS 连续 ↑ 且当日涨跌幅 > 0
- 需在 `csHeatmap` 中确认 CS 从较深负值开始回升，而非在 -1 附近小幅震荡

## 使用注意事项

1. **滞后性**：CS 指标具有滞后性，可能错过最佳买卖时机
2. **极端行情**：在极端行情下，CS 可能持续高位或低位
3. **结合趋势**：建议结合历史序列方向（↑↓）综合判断，不单独看当日绝对值
4. **板块验证**：领涨板块必须用领涨股数量验证，区分板块行情与个股行情
