# 字段说明和名词解释

本文档解释盘前简报 Skill 中使用的字段和名词。

## 市场温度指标

### 估值温度 (PE Temperature)

**定义**：市盈率历史百分位

**含义**：衡量当前估值在历史中的相对位置

**取值范围**：0-100

**解读**：

| 数值范围  | 状态   | 投资启示               |
| --------- | ------ | ---------------------- |
| 0-20      | 低估   | 具备长期投资价值       |
| 20-70     | 中性   | 估值合理，中性看待     |
| 70-100    | 高估   | 注意估值风险           |

**时间维度**：长期（5-10 年视角）

---

### 恐贪指数 (Fear & Greed Index)

**定义**：市场贪婪恐惧程度

**含义**：衡量短期市场情绪极端值

**取值范围**：0-100

**解读**：

| 数值范围 | 状态     | 投资启示               |
| -------- | -------- | ---------------------- |
| 0-10     | 极度恐惧 | 可能接近短期底部       |
| 10-30    | 恐惧     | 市场情绪悲观           |
| 30-70    | 中性     | 情绪平稳               |
| 70-90    | 贪婪     | 市场情绪乐观           |
| 90-100   | 极度贪婪 | 可能接近短期顶部       |

**时间维度**：短期（1-2 周视角）

**买入信号**：< 30 时可能出现买入机会

---

### 趋势温度 (Trend Temperature)

**定义**：股价高于 MA60 的股票占比

**含义**：衡量中期趋势强度

**取值范围**：0-100

**解读**：

| 数值范围 | 状态   | 投资启示           |
| -------- | ------ | ------------------ |
| 0-20     | 低迷   | 市场处于弱势       |
| 20-40    | 偏弱   | 趋势尚未确立       |
| 40-60    | 震荡   | 多空平衡           |
| 60-80    | 偏强   | 上升趋势确立       |
| 80-100   | 过热   | 注意回调风险       |

**时间维度**：中期（1-3 个月视角）

**买入信号**：< 40 时可能出现中期买点

---

### 动量温度 (Momentum Temperature)

**定义**：动量值中位数百分位

**含义**：衡量市场整体动能

**取值范围**：0-100

**解读**：

| 数值范围 | 状态   | 投资启示           |
| -------- | ------ | ------------------ |
| 0-20     | 偏弱   | 市场动能不足       |
| 20-40    | 偏弱   | 动能较弱           |
| 40-60    | 正常   | 动能正常           |
| 60-80    | 偏强   | 动能较强           |
| 80-100   | 强势   | 动能强劲           |

**时间维度**：中短期（1-4 周视角）

**买入信号**：< 40 时可能出现超跌反弹机会

---

## 板块热力图指标

### 板块强度 (QD)

**定义**：板块综合强度评分

**含义**：衡量板块整体强弱程度

**取值范围**：0-100

**解读**：数值越高，板块越强势

---

### 涨跌幅 (ZDF)

**定义**：价格变动百分比

**类型**：
- `zdf`：当日涨跌幅
- `zdf5`：5日累计涨跌幅
- `zdf10`：10日累计涨跌幅
- `zdf20`：20日累计涨跌幅

**单位**：%

---

## 涨跌停指标

### 涨停数量 (Up Count)

**定义**：当日涨停股票数量

**含义**：衡量市场活跃度和题材热度

**解读**：

| 数值范围 | 活跃度 | 市场状态       |
| -------- | ------ | -------------- |
| < 30     | 低     | 市场低迷       |
| 30-60    | 中     | 市场正常       |
| 60-100   | 高     | 市场活跃       |
| > 100    | 极高   | 市场过热       |

---

### 跌停数量 (Down Count)

**定义**：当日跌停股票数量

**含义**：衡量市场恐慌程度

**解读**：

| 数值范围 | 恐慌度 | 市场状态       |
| -------- | ------ | -------------- |
| < 10     | 低     | 市场情绪稳定   |
| 10-30    | 中     | 出现恐慌情绪   |
| > 30     | 高     | 市场恐慌       |

---

### 连板数量

**定义**：连续涨停天数

**类型**：
- `up_2b_count`：2连板数量
- `up_3b_count`：3连板及以上数量

**含义**：衡量题材持续性

---

## 北向资金指标

### 北向净流入 (North Net)

**定义**：沪股通 + 深股通净流入总额

**单位**：亿元

**含义**：衡量外资对A股的态度

**解读**：

| 数值范围      | 流向   | 外资态度       |
| ------------- | ------ | -------------- |
| > 50          | 大幅流入 | 外资积极看好 |
| 20-50         | 流入   | 外资偏乐观     |
| -20 到 20     | 持平   | 外资观望       |
| -50 到 -20    | 流出   | 外资偏谨慎     |
| < -50         | 大幅流出 | 外资看空     |

---

## 市场宽度指标

### 上涨家数 (Up Count)

**定义**：当日上涨股票数量

**含义**：衡量市场广度

---

### 下跌家数 (Down Count)

**定义**：当日下跌股票数量

**含义**：衡量市场广度

---

### 涨跌比

**计算**：上涨家数 / 下跌家数

**解读**：

| 数值范围 | 市场状态       |
| -------- | -------------- |
| > 2      | 普涨，强势市场 |
| 1-2      | 偏强           |
| 0.5-1    | 偏弱           |
| < 0.5    | 普跌，弱势市场 |

---

## 温度指标对比

| 指标       | 时间维度 | 更新频率       | 主要用途           |
| ---------- | -------- | -------------- | ------------------ |
| 估值温度   | 长期     | 交易日晚8点    | 长期投资决策       |
| 恐贪指数   | 短期     | 交易日晚8点    | 短期情绪判断       |
| 趋势温度   | 中期     | 交易日收盘后   | 中期趋势判断       |
| 动量温度   | 中短期   | 交易日收盘后   | 动能状态判断       |
