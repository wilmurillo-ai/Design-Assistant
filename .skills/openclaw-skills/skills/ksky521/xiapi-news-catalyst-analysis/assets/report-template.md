# 个股消息面分析报告模板

---

## 报告规范

**必须遵守**：

- 结论先行：第一部分是核心结论，综合评分 + 一句话判断，普通人能看懂
- 噪音说明：必须在数据概况中说明过滤前后条数，体现信噪比
- 公告分级：必须用 columns 字段对公告做 ⭐⭐⭐/⭐⭐/⭐ 分级，不能只列标题
- 研报量化：必须提取 rating 和 EPS 字段，分析评级变化和 EPS 趋势
- 评分可追溯：每条高影响事件的方向分/信源分/时效分必须可查
- 禁止绝对化表述："一定涨"改为"可能"、"大概率"
- 免责声明：所有报告必须包含

## 报告质量检查清单

生成报告后逐项检查：

- [ ] 核心结论是否包含综合评分（0-100）和一句话判断
- [ ] 数据概况是否说明了噪音过滤前后条数和信噪比
- [ ] 研报部分是否提取了 rating 字段，列出了评级分布
- [ ] 研报部分是否分析了评级变化（同一机构前后对比）
- [ ] 研报部分是否提取了 EPS 字段，分析了预测趋势
- [ ] 公告部分是否使用了 columns 字段做分级，而非只看标题
- [ ] ⭐⭐⭐ 级公告是否有关键内容摘要
- [ ] 有效舆情是否已过滤三类噪音
- [ ] 每条有效舆情事件是否有方向分/信源分/时效分
- [ ] 综合评分计算过程是否可追溯
- [ ] 是否包含后续跟踪建议
- [ ] 是否包含风险提示和免责声明
