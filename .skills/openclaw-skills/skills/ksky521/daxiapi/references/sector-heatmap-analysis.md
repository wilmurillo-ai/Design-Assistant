# 行业热力图解读

本文档详细说明如何使用行业板块热力图进行市场分析。

---

## 一、核心概念

行业热力图是通过可视化方式展示各行业板块动量指标变化的工具,帮助投资者快速识别市场热点、发现趋势变化、捕捉投资机会。

### 热力图类型

| 热力图类型 | 指标 | 用途 |
|------------|------|------|
| **CS动量热力图** | CS值 | 判断板块短期动量强度 |
| **当日涨跌幅热力图** | 当日涨幅 | 识别当日热点板块 |
| **5日涨跌幅热力图** | 5日累计涨幅 | 判断板块持续性 |

### 核心指标说明

**CS值（短期动量）:**
- CS > 5: 龙头线,只有CS大于5才有可能成为领涨龙头
- CS > 0: 动量为正,板块处于强势
- CS < 0: 动量为负,板块处于弱势

**CS与20日均线关系:**
- CS > CS_MA20: 板块站稳20日均线,趋势向上
- CS上穿CS_MA20: 板块突破均线,可能出现反转

## 二、API调用方法

### 接口信息

**API路径:** `/coze/get_sector_data`  
**请求方式:** `POST`  
**认证方式:** `X-API-Token` 请求头

详细API参数说明请参阅 [api-reference.md](api-reference.md#get_sector_data)

### 请求示例

#### CLI方式(推荐)

```bash
# 使用 daxiapi-cli
daxiapi sector

# 或使用简写
dxp sector
```

#### HTTP请求方式

```javascript
// JavaScript Fetch
const response = await fetch('https://daxiapi.com/coze/get_sector_data', {
    method: 'POST',
    headers: {
        'X-API-Token': 'your_api_token',
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({ 
        orderBy: 'cs', 
        lmt: 10 
    })
});
const data = await response.json();
```

### 响应字段

| 字段 | 说明 |
|------|------|
| csHeatmap | CS动量热力图(Markdown表格) |
| zdfHeatmap | 当日涨跌幅热力图 |
| zdf5Heatmap | 5日涨跌幅热力图 |
| crossover | 箱体突破板块信息 |
| topStocksTable | 龙头股表格 |
| cs_gt_ma20_names | CS>CS_MA20的板块名称 |
| cs_gt_5_names | CS>5的板块名称 |
| cs_crossover_ma20_names | 今日CS上穿20日均线的板块 |
| cs_crossover_5_names | 今日CS新达到5龙头线的板块 |
| yest_cs_gt_ma20Names | 昨日CS在20日均线以上的板块 |
| total | 板块总数 |

## 三、分析框架

### 四维度分析方法

#### 1. 板块热度是否延续和扩散

**分析要点:**
- 对比今日与昨日领涨板块
- 判断热度延续性: 昨日领涨板块是否持续走强
- 判断热度扩散: 上涨板块数量是否增多
- 分析扩散路径: 是否扩散到关联板块
- 评估速度: 上涨速度是否加速

#### 2. 找领涨行业

**分析要点:**
- 依据热力图动量指标变化确定领涨行业
- 分析持续性: 是否持续走强或走弱
- 判断反转可能: 是否存在反转信号
- 分析关联性: 领涨行业板块之间的关联性

**重要规则:**
- ⚠️ **只有CS大于5的板块才有可能成为领涨龙头**

#### 3. 找上升行业

**分析要点:**
- 找出连续多日动量持续上升的行业
- 评估持续性: 动量值是否持续上升
- 分析变化速率: 动量值的变化速率
- 排名上升速度: 推测未来领涨潜力
- 分析关联性: 上升行业间的关联性

#### 4. 找反转行业

**分析要点:**
- 关注热力图底部区域行业
- 判断反转可能: 是否走出尾部区域
- 观察向上动能: 是否展现向上动能
- 识别交易机会: 有反转交易机会的行业
- 分析关联性: 反转行业间的关联性

**重要规则:**
- ⚠️ **反转板块必须符合近期CS连续2次上涨**

## 四、输出规范

### 报告结构

```
一、今日收评
二、市场热力图分析
三、市场趋势扩散分析
四、总结
```

### 一、今日收评

**格式要求:**
- 段落格式要结构化输出
- 字数不超过300字
- 包含两部分内容:
  1. 根据新闻内容概括当日市场整体情况、指数与概念板块表现
  2. 分析当日涨幅7%股票个数前十的概念数据

**示例:**
```markdown
## 今日收评
今日A股三大指数集体上涨,沪指涨0.63%报3913点,深证成指涨1.13%,创业板指涨0.71%,全市场成交额1.86万亿元,较前一交易日缩量932亿元,超4300股上涨。盘面上,**能源金属、电池、化学制药、医疗服务**等板块表现强势,其中能源金属(CS值6.306)领涨,锂电、CRO概念爆发;电力、银行、煤炭等板块回调。需关注锂电、CRO、化肥等概念的持续性。
```

### 二、市场热力图分析

**格式要求:**
- 先概括性输出结论
- 然后输出markdown格式表格
- 表头: 分类、板块名称、CS值、今日涨幅、5日涨幅、原因分析
- 分类包括: 领涨、反转、上升
- 段落清晰,简洁明了,板块名称加粗展示

**表格示例:**
```markdown
## 市场热力图分析
当前市场值得关注的板块有:

| 分类 | 板块名称 | CS值 | 今日涨幅 | 5日涨幅 | 原因分析 |
|------|----------|------|----------|----------|----------|
| 领涨 | **能源金属** | 6.306 | 5.97% | x% | 津巴布韦锂出口禁令放开预期落空,叠加全球碳酸锂供需紧平衡预期 |
| 领涨 | **电力** | 5.980 | -0.77% | x% | 短期回调,仍处于阶段性估值底部,"算电"业务或成催化 |
| 上升 | **化学制药** | 1.161 | 4.54% | x% | 创新药板块催化剂密集期(AACR、ASCO大会+年报一季报),估值修复预期 |
| 反转 | **生物制品** | -0.363 | 3.53% | x% | 板块内突破箱体股票较多,创新药产业链传导效应 |
```

**重要提示:**
- ⚠️ **只有CS大于5的板块才有可能成为领涨龙头**
- ⚠️ 如果crossover数据不为空,则重点输出其中提到的板块,并提醒用户重点关注
- ⚠️ 推荐的板块必须符合最新的CS大于0
- ⚠️ 反转板块必须符合近期CS连续2次上涨

### 三、市场趋势扩散分析

**分析步骤:**

**步骤一:** 根据CS大于20日均线数据分析板块站稳20日均线趋势
- 今日数据: cs_gt_ma20
- 昨天数据: yest_cs_gt_ma20
- 总板块数量: total
- 结论: 站稳均线的板块越多说明市场越强

**步骤二:** 结合新增上穿均线和新增CS大于5的板块,分析热度分布
- 今日CS新增上穿20日均线板块: cs_crossover_ma20
- 今日新增CS大于5的板块: cs_crossover_5
- 今日CS大于5的板块: cs_gt_5
- **重要:** CS>5是龙头线,只有CS大于5才有可能成为领涨龙头

**步骤三:** 判断市场趋势变化
- 找出市场趋势变化情况
- 判断市场是否持续变好或变差

**输出示例:**
```markdown
## 市场趋势扩散分析
**今日市场热度明显扩散**,详细分析如下:
- **CS大于20日均线的板块数量**从昨日的8个大幅增加至30个,表明市场转暖。
- **新增CS大于5的板块**为**能源金属**(CS值6.306),成为今日领涨龙头;**电力**(CS值5.980)维持高位。
- **新增上穿20日均线的板块**包括化学制药、生物制品、医疗服务等,这些板块前期超跌,今日放量上涨,显示市场风险偏好提升。
```

### 四、总结

**格式要求:**
- 概括上述内容
- 分析市场当前状态
- 板块名称加粗展示
- 最后提醒用户投资须谨慎

**示例:**
```markdown
## 总结
今日市场经过两日调整,呈现普涨行情,热度聚焦**锂电、CRO、化肥**等板块,能源金属、电力为领涨龙头,化学制药、生物制品、医疗服务等板块反转迹象明显。市场热度扩散,超跌板块反弹,建议关注能源金属(锂电产业链)、化学制药(创新药)、医疗服务(CRO)等板块的持续性,同时警惕电力、煤炭等板块的回调风险。

投资需谨慎,持续观察市场变动。
```

## 五、重要要求

### 内容要求

1. **报告内容结论先行**
2. **仅围绕市场热力图数据及相关投资分析展开讨论**,拒绝回答与该主题无关的话题
3. **输出内容需保持重点的前提下尽量简洁、重点突出、逻辑清晰、条理分明**
4. **重要结论加粗展示**,文章中提到的数值、指数名称、板块名称、个股名称等加粗
5. **绝对不要引入预训练的数据和新闻**

### 数据要求

1. **推荐的板块必须符合最新的CS大于0**
2. **反转板块必须符合近期CS连续2次上涨**
3. **CS的上涨下跌,需要看连续几日的变化速率,注意超买的高潮行为和动量连续出现速率减弱**
4. **表格中每个板块领涨股票尽量都推荐!但是报告中不能出现top_stocks和news之外的股票!**

### 语言规范

#### 禁止事项

- ❌ **禁止绝对化表述**: 不用"一定"、"必然"、"100%"等词,改用"大概率"、"倾向于"
- ❌ **禁止贬低散户**: 不用"韭菜"、"接盘侠"等词,尊重读者智商
- ❌ **禁止政治敏感话题**: 聚焦经济数据和政策分析,不评价政治体制优劣
- ❌ **禁止制造恐慌**: 即使分析风险,也要给出应对思路,不单纯渲染"崩盘论"

#### 推荐用语

- ✅ 使用"大概率"、"倾向于"、"可能"等概率性表述
- ✅ 使用"建议关注"、"值得注意"等中性表述
- ✅ 使用"风险提示"、"谨慎参与"等风险提示语

## 六、实战案例

### 案例1: 市场热度扩散

**市场特征:**
- CS大于20日均线板块数量大幅增加
- 新增多个CS大于5的板块
- 超跌板块出现反弹

**分析要点:**
1. 市场热度明显扩散
2. 多个板块出现领涨机会
3. 超跌板块反转迹象明显

**投资建议:**
- 关注领涨龙头的持续性
- 适当参与超跌反弹
- 注意控制仓位

### 案例2: 市场热度收敛

**市场特征:**
- CS大于20日均线板块数量减少
- 领涨板块数量减少
- 板块轮动加快

**分析要点:**
1. 市场热度收敛
2. 板块持续性减弱
3. 需警惕回调风险

**投资建议:**
- 谨慎参与,控制仓位
- 关注强势板块的回调机会
- 避免追高

## 七、注意事项

### 数据更新时间

- 数据每日收盘后更新
- 一般在17:00之前完成更新

### 使用限制

- 需要有效的API Token
- 每日请求上限: 1000次
- 每分钟请求上限: 10次

### 风险提示

1. 数据可能存在延迟,不构成投资建议
2. 热力图仅反映历史数据,不代表未来走势
3. 建议结合其他指标综合判断
4. 投资有风险,决策需谨慎

```

  +
get_market_temp (市场温度)
  ↓

```

### 配合市场风格分析

```
get_sector_data (板块热力图)
  +
get_market_style (风格轮动)
  ↓
判断板块热点 + 大小盘风格
```

### 配合个股分析

```
get_sector_data (板块热力图)
  +
get_sector_rank_stock (行业内个股排名)
  +
get_stock_data (个股指标)
  ↓
板块筛选 + 个股精选
```

## 九、总结

行业热力图是识别市场热点和趋势变化的重要工具。通过监测板块CS动量、涨跌幅等指标,可以有效捕捉领涨板块、上升板块和反转板块,为投资决策提供参考。

**核心要点:**
1. 四维度分析: 延续扩散、领涨、上升、反转
2. CS>5是龙头线,只有CS大于5才有可能成为领涨龙头
3. 反转板块必须符合近期CS连续2次上涨
4. 关注板块间的关联性和传导效应
5. 注意CS变化速率,警惕超买和动量减弱
6. 严格遵守输出格式和语言规范
