# 市场温度解读

本文档详细说明如何使用市场温度指标进行市场情绪和趋势分析。

---

## 一、核心概念

市场温度是一组反映市场整体情绪、估值水平和趋势强度的综合指标体系。通过监测这些指标,可以有效判断市场的冷热程度,为投资决策提供参考。

### 四大核心指标

| 指标 | 定义 | 时间维度 | 主要用途 |
|------|------|----------|----------|
| **估值温度** | 市场整体市盈率的历史百分位 | 长期 | 判断市场估值水平 |
| **恐贪指数** | 市场短期贪婪和恐惧程度 | 短期 | 捕捉市场情绪极端值 |
| **趋势温度** | 股价高于60日均线的占比 | 中期 | 判断市场趋势强度 |
| **动量温度** | 股票动量值中位数的历史百分位 | 中短期 | 评估市场动量状态 |

---

## 二、估值温度

### 定义

市场整体市盈率的历史百分位,取值范围 **0-100**。

**举例:**
- 20 代表估值低于历史上20%的时间
- 80 代表估值超过历史上80%的时间

### 解读方式

| 温度范围 | 估值状态 | 市场特征 | 投资建议 |
|----------|----------|----------|----------|
| **< 20** | 低估 | 市场估值处于历史低位 | 或许是入场的好时机,可考虑定投 |
| **20-70** | 中性 | 估值处于合理区间 | 正常投资 |
| **> 70** | 高估 | 市场可能存在泡沫 | 投资需谨慎,可考虑分批卖出 |

### 特别注意

⚠️ **估值温度是长期指标**,对短期操作没有太大指导意义。

- 低估时可以考虑定投
- 高估时可以考虑分批卖出
- 不要试图精确抄底逃顶

---

## 三、恐贪指数

### 定义

衡量市场短期贪婪和恐惧程度的指标,取值范围 **0-100**。

### 解读方式

| 指数范围 | 状态 | 市场特征 | 投资建议 |
|----------|------|----------|----------|
| **0-10** | 极度恐惧 | 市场情绪极度悲观,往往暗示可能是潜在的阶段性底部区域 | 可考虑分批买入或定投 |
| **10-30** | 恐惧 | 市场情绪低落 | 可作为观察或小额试探性买入的时机 |
| **30-70** | 中性 | 市场情绪平稳,无明显恐惧或贪婪倾向 | 正常投资,保持策略 |
| **70-90** | 贪婪 | 市场情绪乐观,投资者积极性高,但此时要警惕市场过热 | 谨慎投资,控制仓位 |
| **90-100** | 极度贪婪 | 市场情绪亢奋,可能是潜在的阶段性顶部区域 | 适合分批卖出或止盈 |

### 投资启示

- **逆向思维**: 在极度恐惧时贪婪,在极度贪婪时恐惧
- **分批操作**: 不要一次性买入或卖出
- **结合趋势**: 恐贪指数需结合趋势指标综合判断

---

## 四、趋势温度

### 定义

市场中股价高于60日均线的占比,反映当前市场的趋势强度,取值范围 **0-100**。

**举例:** 20 代表市场中股价高于60日均线占比20%

### 解读方式

| 数值范围 | 名称 | 市场特征 | 投资策略 |
|----------|------|----------|----------|
| **0-20** | 极度低迷区 | 仅少数股票站在MA60上方,市场整体处于中期下跌趋势,空头主导 | 风险大于机会,谨慎观望 |
| **20-40** | 弱势震荡区 | 部分股票开始尝试突破MA60,但尚未形成趋势合力,市场处于空头向多头过渡的初期 | 结构性机会出现,但中线趋势未确认,需警惕反复 |
| **40-60** | 中性平衡区 | 多空力量相对均衡,市场处于中期震荡或趋势转换期,方向需结合其他信号判断 | 指数可能窄幅波动,板块轮动加快,适合高抛低吸,避免追涨杀跌 |
| **60-80** | 强势觉醒区 | 多数股票进入中期上升趋势,多头力量占据主导,市场进入中线上涨周期 | 主线板块明确,资金跟风意愿强,适合顺势参与主流题材 |
| **80-100** | 过热泡沫区 | 几乎所有股票均突破MA60,市场情绪极度亢奋,中期趋势过度延伸 | 风险加速积累,注意控制仓位 |

### 趋势判断要点

1. **趋势确认**: 40以上可确认中期上升趋势
2. **风险预警**: 80以上需警惕过热风险
3. **底部信号**: 20以下可能接近阶段性底部

---

## 五、动量温度

### 定义

市场股票动量值中位数的历史百分位,动量值代表个股动量强弱,取值范围 **0-100**。

### 解读方式

| 动量范围 | 动量状态 | 市场特征 | 投资建议 |
|----------|----------|----------|----------|
| **< 20** | 动量太弱 | 股票超跌 | 可关注超跌反弹机会 |
| **20-40** | 偏弱 | 动量不足 | 谨慎观望 |
| **40-60** | 正常 | 属于正常范围 | 正常交易 |
| **60-80** | 偏强 | 动量较强 | 可适当参与 |
| **> 80** | 动量强势 | 可能离市场高潮不远 | 注意风险,考虑减仓 |

### 动量分析要点

- 动量温度反映市场整体动能
- 配合趋势温度使用效果更佳
- 动量过高往往预示短期调整

## 六、API调用方法

### 接口信息

**API路径:** `/coze/get_market_temp`  
**请求方式:** `POST`  
**认证方式:** `X-API-Token` 请求头

### 请求示例

#### CLI方式(推荐)

```bash
# 使用 daxiapi-cli
daxiapi market-temp

# 或使用简写
dxp market-temp
```

#### HTTP请求方式

```javascript
// JavaScript Fetch
const response = await fetch('https://daxiapi.com/coze/get_market_temp', {
    method: 'POST',
    headers: {
        'X-API-Token': 'your_api_token',
        'Content-Type': 'application/json'
    }
});
const data = await response.json();
```

```bash
# cURL
curl -X POST \
     -H "X-API-Token: your_api_token" \
     -H "Content-Type: application/json" \
     https://daxiapi.com/coze/get_market_temp
```

### 响应字段

| 字段 | 类型 | 说明 |
|------|------|------|
| date | string | 交易日期 |
| valuation_temp | number | 估值温度 |
| fear_greed_index | number | 恐贪指数 |
| trend_temp | number | 趋势温度 |
| momentum_temp | number | 动量温度 |

## 七、分析框架

### 完整分析流程

```
步骤1: 获取最新数据
  ↓
调用 get_market_temp 获取四个温度指标
  ↓
步骤2: 输出最新数据
  ↓
用markdown表格展示各指标最新值和结论
  ↓
步骤3: 分析变化趋势
  ↓
分析恐贪、趋势、动量三个指标最近20天变动情况
  ↓
步骤4: 综合判断与建议
  ↓
根据信号数量给出投资建议
```

## 八、输出规范

### 步骤1: 输出最新数据

**格式要求:**

```markdown

| 指标名 | 最新值 | 结论 |
|--------|--------|------|
| 估值温度 | XX | 根据阈值判断结论 |
| 恐贪指数 | XX | 根据阈值判断结论 |
| 趋势温度 | XX | 根据阈值判断结论 |
| 动量温度 | XX | 根据阈值判断结论 |
```

**示例:**

```markdown
## 市场的最新数据

| 指标名 | 最新值 | 结论 |
|--------|--------|------|
| 估值温度 | 45 | 中性,估值处于合理区间 |
| 恐贪指数 | 25 | 恐惧,可作为观察或小额试探性买入的时机 |
| 趋势温度 | 35 | 弱势震荡区,结构性机会出现但需警惕反复 |
| 动量温度 | 30 | 偏弱,动量不足,谨慎观望 |
```

### 步骤2: 分析变化趋势

**格式要求:**

```markdown
## 变化分析

| 指标名 | 分析 |
|--------|------|
| 恐贪指数 | 最近20天的变化趋势描述 |
| 趋势温度 | 最近20天的变化趋势描述 |
| 动量温度 | 最近20天的变化趋势描述 |
```

**分析要点:**
- 观察指标是上升、下降还是震荡
- 判断变化速度(快速/缓慢)
- 识别关键转折点
- 评估当前所处位置

### 步骤3: 综合结论

**信号判断规则:**

| 买入信号条件 | 说明 |
|--------------|------|
| 恐贪指数 < 30 | 市场情绪恐惧,可能接近底部 |
| 趋势温度 < 40 | 市场处于弱势区,可能接近底部 |
| 动量温度 < 40 | 动量偏弱,超跌反弹机会 |

**结论输出规则:**

| 信号数量 | 投资建议 |
|----------|----------|
| **1个指标** | 提示不要跟风抛售 |
| **2个指标** | 提示可以寻找已经超跌反弹机会 |
| **3个指标** | 强烈建议买入 |

**格式要求:**

```markdown
## 综合结论

一句话总结上面指标的趋势和结论。

例如: 当前市场处于[状态],[X]个指标出现买入信号,[建议内容]。
```

## 九、实战案例

### 案例1: 市场底部特征

**数据:**
- 估值温度: 15 (低估)
- 恐贪指数: 8 (极度恐惧)
- 趋势温度: 18 (极度低迷区)
- 动量温度: 12 (动量太弱)

**分析:**
1. 四个指标均处于低位
2. 恐贪、趋势、动量三个指标都出现买入信号
3. 市场情绪极度悲观

**结论:**
市场处于极度低迷状态,3个指标出现买入信号,强烈建议分批买入或定投。

### 案例2: 市场过热特征

**数据:**
- 估值温度: 75 (高估)
- 恐贪指数: 92 (极度贪婪)
- 趋势温度: 88 (过热泡沫区)
- 动量温度: 85 (动量强势)

**分析:**
1. 四个指标均处于高位
2. 市场情绪极度亢奋
3. 中期趋势过度延伸

**结论:**
市场处于过热状态,风险加速积累,建议分批卖出或止盈。

### 案例3: 震荡市场特征

**数据:**
- 估值温度: 50 (中性)
- 恐贪指数: 45 (中性)
- 趋势温度: 52 (中性平衡区)
- 动量温度: 48 (正常)

**分析:**
1. 四个指标均处于中性区间
2. 多空力量相对均衡
3. 市场方向不明确

**结论:**
市场处于震荡状态,无明显买入或卖出信号,建议保持策略,高抛低吸。

## 十、注意事项

### 数据更新时间

- 数据每日收盘后更新
- 一般在17:00之前完成更新

### 使用限制

- 需要有效的API Token
- 每日请求上限: 1000次
- 每分钟请求上限: 10次

### 风险提示

1. **指标滞后性**: 温度指标存在一定滞后,不构成投资建议
2. **极端情况**: 市场可能出现极端情况,指标可能持续处于极端值
3. **综合判断**: 建议结合其他指标和市场环境综合判断
4. **分批操作**: 不要一次性买入或卖出,建议分批操作
5. **长期视角**: 估值温度是长期指标,短期波动属正常现象

## 十二、总结

市场温度指标是判断市场冷热程度的重要工具。通过监测估值、恐贪、趋势、动量四个维度的温度值,可以有效识别市场的极端状态,为投资决策提供参考。

**核心要点:**
1. 估值温度是长期指标,用于判断市场估值水平
2. 恐贪指数捕捉市场情绪极端值,逆向操作
3. 趋势温度判断市场中期趋势强度
4. 动量温度评估市场整体动能
5. 多指标交叉验证,避免单一指标决策
6. 分批操作,不要试图精确抄底逃顶
