# 市场风格分析

本文档详细说明如何使用大小盘波动差值指标进行市场风格分析。

---

## 一、核心概念

### 什么是大小盘风格轮动

A股市场存在显著的大小盘风格轮动特征,市场资金会在大盘股和小盘股之间周期性切换。精准捕捉这一规律对于投资决策至关重要。

### 监测体系构建

我们构建了双维度监测体系:

- **沪深300指数** - 大盘股代表
- **中证2000指数** - 小盘股标杆

通过计算二者在n日周期内的涨跌幅差值,构建直观的风格轮动观测指标。

---

## 二、大小盘波动差值指标

### 计算公式

```
大小盘波动差值 = 中证2000涨幅 - 沪深300涨幅
```

### 差值解读

| 差值范围   | 含义                       | 市场判断                        |
| ---------- | -------------------------- | ------------------------------- |
| **> 0**    | 中证2000涨幅 > 沪深300涨幅 | 小盘股强于大盘股,小盘股占优     |
| **< 0**    | 中证2000涨幅 < 沪深300涨幅 | 小盘股弱于大盘股,大盘股占优     |
| **≥ 10%**  | 小盘股大幅领先             | 强烈看空小盘股,注意均值回归风险 |
| **≤ -10%** | 大盘股大幅领先             | 强烈看空大盘股,注意均值回归风险 |

---

## 三、均值回归信号

### 核心原理

当差值出现 **±10%** 左右时,可能出现均值回归,即市场会切换风格。

### 具体案例

| 差值达到 | 含义                | 预期走势                  |
| -------- | ------------------- | ------------------------- |
| **-10%** | 小盘股弱于大盘股10% | 可能出现反转,小盘股将走强 |
| **+10%** | 小盘股强于大盘股10% | 可能出现反转,大盘股将走强 |

### 投资启示

- 差值达到 **-10%** → 强烈看空大盘股,关注小盘股机会
- 差值达到 **+10%** → 强烈看空小盘股,关注大盘股机会

---

## 四、运动方向判断

### 差值趋势分析

| 差值方向     | 含义                         | 投资策略       |
| ------------ | ---------------------------- | -------------- |
| **向下运动** | 大盘股在未来一段时间表现更好 | 侧重大盘股配置 |
| **向上运动** | 小盘股在未来一段时间表现更好 | 侧重小盘股配置 |

### 分析要点

1. 观察差值的连续变化趋势
2. 判断运动速度和加速度
3. 结合绝对值位置综合判断

---

## 五、机构vs游资活跃度

### 市场参与者特征

| 风格占优       | 主要推动力量     | 市场特征             |
| -------------- | ---------------- | -------------------- |
| **大盘股占优** | 机构资金更加积极 | 价值投资、蓝筹股行情 |
| **小盘股占优** | 游资更加积极     | 题材炒作、成长股行情 |

### 投资启示

- 大盘股占优时,关注机构偏好标的(白马股、蓝筹股)
- 小盘股占优时,关注游资活跃标的(题材股、成长股)

## 六、API调用方法

### 接口信息

**API路径:** `/coze/get_market_style`  
**请求方式:** `GET`  
**认证方式:** `X-API-Token` 请求头

### 请求示例

#### CLI方式(推荐)

```bash
# 使用 daxiapi-cli
daxiapi market-style

# 或使用简写
dxp market-style
```

#### HTTP请求方式

```javascript
// JavaScript Fetch
const response = await fetch('https://daxiapi.com/coze/get_market_style', {
    method: 'GET',
    headers: {'X-API-Token': 'your_api_token'}
});
const data = await response.json();
```

```bash
# cURL
curl -H "X-API-Token: your_api_token" \
     https://daxiapi.com/coze/get_market_style
```

### 响应字段

| 字段 | 类型   | 说明                   |
| ---- | ------ | ---------------------- |
| date | string | 交易日期               |
| diff | number | 大小盘波动差值(百分比) |

## 七、分析框架

### 完整分析流程

```
步骤1: 获取数据
  ↓
调用 get_market_style 获取大小盘波动差值
  ↓
步骤2: 差值绝对值判断
  ↓
判断是否接近 ±10% 均值回归阈值
  ↓
步骤3: 差值方向判断
  ↓
分析差值运动趋势(向上/向下)
  ↓
步骤4: 市场参与者判断
  ↓
判断机构/游资活跃度
  ↓
步骤5: 综合投资建议
  ↓
结合数值、趋势、市场特征给出建议
```

### 多维度交叉验证

1. **数值维度** - 差值绝对值大小
2. **趋势维度** - 差值运动方向
3. **速度维度** - 差值变化速率
4. **市场维度** - 机构/游资活跃度

## 八、投资建议规范

### 建议原则

1. **量化判断** - 根据差值偏离均值的程度量化判断风格切换可能性
2. **趋势评估** - 依据差值变动速率评估短期和中期走向
3. **综合分析** - 多指标交叉验证,避免单一指标决策
4. **模糊正确** - 不给具体仓位,推荐方向要精准

### 典型场景建议

#### 场景1: 差值接近 -10%

```
当前差值: -9.5%
趋势: 向下运动趋缓
判断: 小盘股极度弱势,接近均值回归阈值
建议: 关注小盘股反转机会,大盘股风险增加
```

#### 场景2: 差值接近 +10%

```
当前差值: +9.2%
趋势: 向上运动加速
判断: 小盘股极度强势,接近均值回归阈值
建议: 警惕小盘股回调风险,关注大盘股机会
```

#### 场景3: 差值在正常区间

```
当前差值: +3.5%
趋势: 向上运动
判断: 小盘股温和占优
建议: 适度配置小盘股,关注趋势变化
```

## 九、注意事项

### 数据更新时间

- 数据每日收盘后更新
- 一般在17:00之前完成更新

### 使用限制

- 需要有效的API Token
- 每日请求上限: 1000次
- 每分钟请求上限: 10次

### 风险提示

1. 数据可能存在延迟,不构成投资建议
2. 均值回归信号仅供参考,市场可能出现极端情况
3. 建议结合其他指标综合判断
4. 投资有风险,决策需谨慎

## 十、与其他指标配合使用

### 配合市场温度指标

```
get_market_style (风格轮动)
  +
get_market_temp (恐贪指数)
  +
get_market_degree (市场温度)
  ↓
综合判断市场整体环境
```

### 配合板块热力图

```
get_market_style (风格轮动)
  +
get_sector_data (板块热力图)
  ↓
判断大小盘风格 + 板块轮动
```

### 配合个股分析

```
get_market_style (风格轮动)
  +
get_stock_data (个股指标)
  ↓
根据风格偏好筛选个股
```

## 十一、实战案例

### 案例1: 2024年初市场风格切换

**背景:**

- 差值持续在 -8% 到 -12% 之间波动
- 大盘股持续占优

**分析:**

1. 差值接近 -10% 均值回归阈值
2. 差值运动方向开始趋缓
3. 机构资金活跃度下降

**建议:**

- 关注小盘股反转机会
- 减少大盘股配置

### 案例2: 题材股行情判断

**背景:**

- 差值从 0 上升到 +6%
- 向上运动趋势明显

**分析:**

1. 小盘股开始占优
2. 游资活跃度提升
3. 题材股行情启动

**建议:**

- 适度参与题材股
- 关注游资活跃标的
- 注意控制仓位

## 十二、总结

大小盘波动差值是捕捉A股市场风格轮动的重要工具。通过监测差值的绝对值、运动方向、变化速率等维度,可以有效判断市场风格切换时机,为投资决策提供参考。

**核心要点:**

1. 差值 ±10% 是重要阈值,注意均值回归
2. 差值方向预示未来风格走向
3. 结合机构/游资活跃度判断市场特征
4. 多指标交叉验证,避免单一指标决策
