# 热门股票分析报告模板

---

## 报告规范

**必须遵守**：

- 结论先行：第一部分是核心结论，一句话，普通人能看懂
- 市场性质明确：报告开头必须说明今日是上涨日/回调日/分化日
- 概念频次统计是主线识别的量化基础，不能靠感觉归纳
- 主线必须分级：区分 A/B/C 级，不把所有强势股平铺罗列
- 截断数据必须说明：所有结论注明"基于截断展示数据"，并引导用户去 daxiapi.com 查看完整数据
- 操作参考必须分三档：按追高风险分高/中/低，不笼统说"可关注"
- 禁止绝对化表述："一定涨停"改为"可能"、"大概率"
- 免责声明：所有报告必须包含

## 报告质量检查清单

生成报告后逐项检查：

- [ ] 报告开头是否注明截断数据说明，并附 daxiapi.com 链接
- [ ] 是否完成了市场性质判断（上涨日/回调日/分化日），并在报告中明确说明
- [ ] 核心结论是否一句话说清楚今日主线和市场性质
- [ ] 是否完成概念频次统计（Top 15），并以此作为主线识别的量化依据
- [ ] 主线是否区分了 A/B/C 级，每条 A/B 级主线是否包含广度、跨板块性、新鲜度三个维度
- [ ] A 级主线是否有热力图数据印证
- [ ] 代表股票表格是否包含 5日涨幅和 20日涨幅
- [ ] C 级跟风是否注明了热度低的具体原因（CS 值或概念板块涨跌幅）
- [ ] 操作参考是否按🔴/🟡/🟢三档输出，每档是否有具体理由
- [ ] 操作参考是否附 daxiapi.com 链接
- [ ] 风险提示是否包含 daxiapi.com 完整数据链接
- [ ] 是否包含免责声明
