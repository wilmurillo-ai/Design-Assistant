# 字段说明和名词解释

本文档详细说明大小盘风格轮动分析中涉及的术语和字段。

## 核心术语

### 大小盘风格

**定义**：指市场中大盘股和小盘股的相对表现差异。

**大盘股**：
- 代表指数：沪深300（代码：1.000300）
- 特点：市值大、流动性好、机构持股比例高
- 行业分布：金融、地产、能源、消费等传统行业居多

**小盘股**：
- 代表指数：中证2000（代码：1.932000）
- 特点：市值小、弹性大、游资活跃
- 行业分布：科技、医药、新能源等新兴行业居多

### 风格轮动

**定义**：市场资金在大盘股和小盘股之间切换的现象。

**典型周期**：
- 小盘强周期：市场情绪活跃，游资主导，题材炒作盛行
- 大盘强周期：市场情绪谨慎，机构主导，价值投资主导

### 波动差值

**定义**：中证2000涨跌幅与沪深300涨跌幅的差值。

**计算公式**：
```
波动差值 = 中证2000涨跌幅 - 沪深300涨跌幅
```

**数值解读**：

| 差值范围 | 含义                       | 资金特征         |
| -------- | -------------------------- | ---------------- |
| > 5%     | 小盘股明显强于大盘股       | 游资高度活跃     |
| 2%~5%    | 小盘股较强                 | 游资相对积极     |
| 0%~2%    | 小盘股略强                 | 均衡偏小盘       |
| -2%~0%   | 大盘股略强                 | 均衡偏大盘       |
| -5%~-2%  | 大盘股较强                 | 机构相对积极     |
| < -5%    | 大盘股明显强于小盘股       | 机构高度活跃     |

## 字段说明

### spread（波动差值）

**类型**：number

**单位**：百分比（%）

**说明**：当日中证2000涨跌幅减去沪深300涨跌幅的结果

**示例**：
- `spread: 2.5` 表示小盘股比大盘股强 2.5%
- `spread: -1.8` 表示大盘股比小盘股强 1.8%

### trend（趋势方向）

**类型**：string

**可选值**：
- `上升`：最近20日差值总体上升，小盘风格增强
- `下降`：最近20日差值总体下降，大盘风格增强
- `震荡`：最近20日差值波动无明显方向

**判断标准**：
- 连续5日以上差值同向变化 → 明确趋势
- 差值波动在±1%内 → 震荡

### csi2000（中证2000数据）

**类型**：object

**子字段**：

| 字段   | 类型   | 说明               |
| ------ | ------ | ------------------ |
| code   | string | 指数代码（1.932000）|
| name   | string | 指数名称           |
| change | number | 当日涨跌幅（%）    |

### csi300（沪深300数据）

**类型**：object

**子字段**：

| 字段   | 类型   | 说明               |
| ------ | ------ | ------------------ |
| code   | string | 指数代码（1.000300）|
| name   | string | 指数名称           |
| change | number | 当日涨跌幅（%）    |

### data（历史数据）

**类型**：array

**说明**：最近20日的差值历史数据

**元素格式**：

```json
{
  "date": "2025-01-15",
  "spread": 2.5
}
```

## 均值回归

**定义**：当差值偏离正常范围后，倾向于回归平均值的现象。

**应用场景**：

| 差值范围        | 均值回归可能性 | 投资含义               |
| --------------- | -------------- | ---------------------- |
| 差值 ≈ ±10%     | 较高           | 风格可能切换           |
| 差值 ≈ ±5%      | 中等           | 关注切换信号           |
| 差值 ≈ 0%       | 低             | 风格均衡               |

**⚠️ 注意**：
- 均值回归不保证一定发生
- 极端情况下风格可能持续分化
- 需结合市场环境综合判断

## 风格切换信号

**典型特征**：

1. **差值极端化**：
   - 差值接近 ±10% 历史极端值
   - 持续时间较长（5日以上）

2. **趋势反转**：
   - 差值变化方向反转
   - 成交量异常放大

3. **市场环境变化**：
   - 政策面重大变化
   - 流动性环境改变
   - 风险偏好转变

## 风险提示

1. **滞后性**：差值数据为收盘后更新，盘中无法实时判断
2. **不准确性**：风格轮动受多种因素影响，单一指标存在局限
3. **市场风险**：历史规律可能失效，投资需谨慎

## 相关概念

### 市场宽度

指市场中上涨股票数量与下跌股票数量的比例，反映市场整体健康程度。

### 资金流向

指市场资金在不同板块、不同风格之间的流动方向。

### 风险偏好

指投资者对风险的接受程度，影响资金在大盘股和小盘股之间的配置。
