# CLI 命令参考

本文档详细介绍市场温度分析相关的 CLI 命令。

## 核心命令

### market temp — 市场温度四大指标

```bash
npx daxiapi-cli@latest market temp
```

**功能说明**：获取市场温度四大指标数据，返回最近 20 个交易日，最新日期在前。

**返回字段**：

| 字段     | 说明       | 取值范围   |
| -------- | ---------- | ---------- |
| 日期     | 交易日期   | YYYY-MM-DD |
| 估值温度 | 市盈率历史百分位 | 0-100 |
| 恐贪指数 | 市场贪婪恐惧程度 | 0-100，当日晚 9 点前可能为 0（数据未更新） |
| 趋势温度 | 股价高于 MA60 的占比 | 0-100 |
| 动量温度 | CS 中位数历史百分位 | 0-100 |

**返回示例**：

```
[20]{"日期","估值温度","恐贪指数","趋势温度","动量温度"}:
  2026-04-09,49.61,0,25.61,35.59
  2026-04-08,50.48,44.48,28.09,61.02
  ...
```

> ⚠️ 恐贪指数在交易日晚 9 点前可能显示为 0，属于数据未更新，**不代表极度恐惧**，请勿误读。

---

### market compass — 市场三维结构（推荐）

```bash
npx daxiapi-cli@latest market compass
```

**功能说明**：获取比 `market temp` 更完整的市场结构数据，在四大温度指标基础上额外返回趋势指标(20日/200日)、CS 原始中位数、CS 离散度、全A指数点位，返回最近 20 个交易日。

**返回字段**：

| 字段             | 说明                                   | 取值范围 / 单位 |
| ---------------- | -------------------------------------- | --------------- |
| 日期             | 交易日期                               | YYYY-MM-DD      |
| 估值温度         | 市盈率历史百分位                       | 0-100           |
| 恐贪指数         | 市场贪婪恐惧程度                       | 0-100，晚 9 点前可能为 0 |
| 趋势温度(60日)   | 股价高于 MA60 的占比（中期趋势强度）   | 0-100           |
| 趋势指标(20日)   | 股价高于 MA20 的占比（短期趋势强度）   | 0-100           |
| 趋势指标(200日)  | 股价高于 MA200 的占比（长期趋势强度）  | 0-100           |
| 动量温度(CS中位数百分位) | CS 中位数的历史百分位          | 0-100           |
| CS中位数         | 全市场 CS 的原始中位数值               | 通常 -15 到 +15 |
| CS离散度         | 全市场 CS 的标准差，反映市场分化程度   | 正数，越大越分化 |
| 全A指数          | 全市场综合指数点位                     | 点              |

**返回示例**：

```
[20]{"日期","估值温度","恐贪指数","趋势温度(60日)","趋势指标(20日)","趋势指标(200日)","动量温度(CS中位数百分位)","CS中位数","CS离散度","全A指数"}:
  2026-04-09,49.61,0,25.61,37.38,45.02,35.59,-1.26,1.97,6020.05
  2026-04-08,50.48,44.48,28.09,48.51,48.69,61.02,0.23,1.85,6059.75
  ...
```

> 💡 **推荐优先使用 `market compass`**，数据更完整，可替代 `market temp`。

### market style — 大小盘风格轮动

```bash
npx daxiapi-cli@latest market style
```

**功能说明**：获取大小盘风格数据，计算中证2000（小盘）与沪深300（大盘）的涨跌幅差值，返回最近 30 个交易日，并给出当前风格判断和历史百分位。

**返回字段**：

| 字段         | 说明                                         |
| ------------ | -------------------------------------------- |
| 日期         | 交易日期                                     |
| 大小盘波动差值 | 中证2000涨幅 - 沪深300涨幅，正值小盘强，负值大盘强 |

**底部汇总信息**：

- `当前风格`：如"均衡偏大盘（历史P28）"
- `当前差值`：最新一日差值
- `历史百分位`：当前差值在历史中的位置

**解读规则**：

| 差值范围 | 风格判断 | 说明 |
| -------- | -------- | ---- |
| > +5     | 明显小盘风格 | 小盘股显著强于大盘 |
| +1 ~ +5  | 偏小盘   | 小盘略强 |
| -1 ~ +1  | 均衡     | 大小盘相当 |
| -5 ~ -1  | 偏大盘   | 大盘略强 |
| < -5     | 明显大盘风格 | 大盘股显著强于小盘 |

**返回示例**：

```
[30]{"日期","大小盘波动差值"}:
  2026-04-09,-2.82
  2026-04-08,-2.34
  ...
当前风格: 均衡偏大盘（历史P28）,
当前差值: -2.82,
历史百分位: P28
```

### market value — 指数估值数据

```bash
npx daxiapi-cli@latest market value
```

**功能说明**：获取 A 股主要指数的估值数据，包含 PE/PB/温度/百分位，返回多个指数的最新估值。

**返回字段**：

| 字段      | 说明                         |
| --------- | ---------------------------- |
| date      | 数据日期                     |
| 温度      | 综合估值温度（0-100）        |
| PB        | 市净率                       |
| PE        | 市盈率（TTM）                |
| PE百分位  | PE 在历史中的百分位（0-100） |
| PB百分位  | PB 在历史中的百分位（0-100） |

> ⚠️ **重要陷阱**：`market value` 的输出**不包含指数名称**，每行只有数值，无法直接判断对应哪个指数。如需按指数分析估值，建议结合大虾皮网站或其他来源确认指数对应关系。

**返回示例**：

```
items[36]{date,"温度",PB,PE,"PE百分位","PB百分位"}:
  2026-04-08,"56.32","1.21","11.24","74.12","38.52"
  2026-04-08,"58.02","1.41","13.90","76.71","39.32"
  ...
```

## 配置命令

### 检查 Token 配置

```bash
npx daxiapi-cli@latest config get token
```

### 配置 Token

```bash
npx daxiapi-cli@latest config set token YOUR_TOKEN_FROM_DAXIAPI
```

## 命令选项

```bash
npx daxiapi-cli@latest market --help
```

## 错误处理

### 常见错误码

| 错误码 | 说明         | 解决方案                   |
| ------ | ------------ | -------------------------- |
| 401    | 认证失败     | 检查 Token 是否正确配置    |
| 404    | API 不存在   | 检查命令拼写               |
| 429    | 请求频率超限 | 等待 30-60 秒后重试        |
| 500    | 服务器错误   | 稍后重试                   |

## 最佳实践

### DO ✅

1. 优先使用 `market compass`，数据比 `market temp` 更完整
2. 使用最新版本的 CLI：`npx daxiapi-cli@latest`
3. 数据更新时间为每日收盘后，恐贪指数晚 9 点后更新

### DON'T ❌

1. 不要把恐贪指数为 0 误读为"极度恐惧"，可能是数据未更新
2. 不要在盘中查询（数据未更新）
3. 不要频繁调用 API（避免触发限流）
