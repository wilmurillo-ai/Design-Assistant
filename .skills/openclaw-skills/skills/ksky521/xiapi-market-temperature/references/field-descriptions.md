# 字段说明

本文档详细说明市场温度分析相关的字段定义和名词解释。

## 核心字段

### 估值温度（valuation_temp）

**定义**：市场整体市盈率的历史百分位，取值范围 0-100。

**计算逻辑**：当前市盈率在历史数据中的位置

**数值含义**：

- 20 代表估值低于历史上 20% 的时间
- 80 代表估值超过历史上 80% 的时间

**解读方式**：

| 温度范围  | 估值状态 | 市场特征             | 投资建议                       |
| --------- | -------- | -------------------- | ------------------------------ |
| < 20      | 低估     | 市场估值处于历史低位 | 或许是入场的好时机，可考虑定投 |
| 20-70     | 中性     | 估值处于合理区间     | 正常投资                       |
| > 70      | 高估     | 市场可能存在泡沫     | 投资需谨慎，可考虑分批卖出     |

**时间维度**：长期指标

**特别说明**：

- 估值温度是长期指标，对短期操作没有太大指导意义
- 低估时可以考虑定投
- 高估时可以考虑分批卖出
- 不要试图精确抄底逃顶

---

### 恐贪指数（fear_greed_index）

**定义**：衡量市场短期贪婪和恐惧程度的指标，取值范围 0-100。

**计算逻辑**：综合市场波动、市场动能、社交媒体情绪等多维度数据

**解读方式**：

| 指数范围   | 状态     | 市场特征                                            | 投资建议                         |
| ---------- | -------- | --------------------------------------------------- | -------------------------------- |
| 0-10       | 极度恐惧 | 市场情绪极度悲观，往往暗示可能是潜在的阶段性底部区域 | 可考虑分批买入或定投             |
| 10-30      | 恐惧     | 市场情绪低落                                        | 可作为观察或小额试探性买入的时机 |
| 30-70      | 中性     | 市场情绪平稳，无明显恐惧或贪婪倾向                  | 正常投资，保持策略                |
| 70-90      | 贪婪     | 市场情绪乐观，投资者积极性高，但此时要警惕市场过热   | 谨慎投资，控制仓位                |
| 90-100     | 极度贪婪 | 市场情绪亢奋，可能是潜在的阶段性顶部区域            | 适合分批卖出或止盈               |

**时间维度**：短期指标

**投资启示**：

- **逆向思维**：在极度恐惧时贪婪，在极度贪婪时恐惧
- **分批操作**：不要一次性买入或卖出
- **结合趋势**：恐贪指数需结合趋势指标综合判断

---

### 趋势温度（trend_temp）

**定义**：市场中股价高于 60 日均线的占比，反映当前市场的趋势强度，取值范围 0-100。

**计算逻辑**：(股价 > MA60 的股票数量 / 总股票数量) × 100

**举例**：20 代表市场中股价高于 60 日均线占比 20%

**解读方式**：

| 数值范围   | 名称       | 市场特征                                                                 | 投资策略                                                |
| ---------- | ---------- | ------------------------------------------------------------------------ | ------------------------------------------------------- |
| 0-20       | 极度低迷区 | 仅少数股票站在 MA60 上方，市场整体处于中期下跌趋势，空头主导                 | 风险大于机会，谨慎观望                                   |
| 20-40      | 弱势震荡区 | 部分股票开始尝试突破 MA60，但尚未形成趋势合力，市场处于空头向多头过渡的初期 | 结构性机会出现，但中线趋势未确认，需警惕反复              |
| 40-60      | 中性平衡区 | 多空力量相对均衡，市场处于中期震荡或趋势转换期，方向需结合其他信号判断     | 指数可能窄幅波动，板块轮动加快，适合高抛低吸，避免追涨杀跌 |
| 60-80      | 强势觉醒区 | 多数股票进入中期上升趋势，多头力量占据主导，市场进入中线上涨周期           | 主线板块明确，资金跟风意愿强，适合顺势参与主流题材        |
| 80-100     | 过热泡沫区 | 几乎所有股票均突破 MA60，市场情绪极度亢奋，中期趋势过度延伸                 | 风险加速积累，注意控制仓位                               |

**时间维度**：中期指标

**趋势判断要点**：

1. **趋势确认**：40 以上可确认中期上升趋势
2. **风险预警**：80 以上需警惕过热风险
3. **底部信号**：20 以下可能接近阶段性底部

---

### 动量温度（momentum_temp）

**定义**：市场股票动量值中位数的历史百分位，动量值代表个股动量强弱，取值范围 0-100。

**计算逻辑**：计算所有股票的动量值，取中位数，然后计算其在历史数据中的百分位

**解读方式**：

| 动量范围  | 动量状态 | 市场特征           | 投资建议           |
| --------- | -------- | ------------------ | ------------------ |
| < 20      | 动量太弱 | 股票超跌           | 可关注超跌反弹机会 |
| 20-40     | 偏弱     | 动量不足           | 谨慎观望           |
| 40-60     | 正常     | 属于正常范围       | 正常交易           |
| 60-80     | 偏强     | 动量较强           | 可适当参与         |
| > 80      | 动量强势 | 可能离市场高潮不远 | 注意风险，考虑减仓  |

**时间维度**：中短期指标

**动量分析要点**：

- 动量温度反映市场整体动能
- 配合趋势温度使用效果更佳
- 动量过高往往预示短期调整

## compass 专属字段

以下字段仅在 `market compass` 命令中返回，`market temp` 不包含。

### 趋势指标(20日)

**定义**：市场中股价高于 20 日均线（MA20）的股票占比，反映短期趋势强度，取值范围 0-100。

**与趋势温度(60日)的区别**：

| 指标 | 均线 | 时间维度 | 用途 |
| ---- | ---- | -------- | ---- |
| 趋势温度(60日) | MA60 | 中期 | 判断中期趋势方向 |
| 趋势指标(20日) | MA20 | 短期 | 判断短期趋势强度，更灵敏 |
| 趋势指标(200日) | MA200 | 长期 | 判断长期牛熊格局 |

**解读**：数值越高，短期趋势越强；数值越低，短期趋势越弱。

---

### 趋势指标(200日)

**定义**：市场中股价高于 200 日均线（MA200）的股票占比，反映长期趋势格局，取值范围 0-100。

**解读方式**：

| 数值范围 | 市场格局 | 说明 |
| -------- | -------- | ---- |
| > 60     | 长期牛市格局 | 多数股票处于长期上升趋势 |
| 40-60    | 长期中性格局 | 多空力量均衡 |
| < 40     | 长期熊市格局 | 多数股票处于长期下降趋势 |

**特别说明**：MA200 是判断牛熊市的重要参考，变化缓慢，适合长期视角分析。

---

### CS中位数

**定义**：全市场所有股票 CS（Close-to-EMA Strength，收盘价相对均线强度）的原始中位数值，反映市场整体动能的绝对水平。

**与动量温度的区别**：

- **动量温度**：CS 中位数的历史百分位（0-100），反映相对历史的高低
- **CS中位数**：CS 的原始值，反映当前市场动能的绝对强弱

**典型取值范围**：通常在 -15 到 +15 之间

**解读**：

- 正值：市场整体动能偏强，多数股票收盘价高于均线
- 负值：市场整体动能偏弱，多数股票收盘价低于均线
- 接近 0：市场动能中性

---

### CS离散度

**定义**：全市场所有股票 CS 值的标准差，反映市场内部分化程度。

**解读方式**：

| 离散度水平 | 市场特征 | 含义 |
| ---------- | -------- | ---- |
| 低（< 1.5） | 市场高度一致 | 多数股票同涨同跌，系统性行情 |
| 中（1.5-2.5） | 正常分化 | 板块轮动正常，结构性机会存在 |
| 高（> 2.5） | 市场高度分化 | 强弱分化明显，选股难度加大 |

**用途**：离散度高时，个股差异大，需要更精准的选股；离散度低时，市场整体性强，指数ETF更有效。

---

### 全A指数

**定义**：全市场综合指数点位，反映 A 股整体市场的价格水平。

**用途**：

- 提供市场整体价格参考
- 配合温度指标判断市场所处位置
- 观察指数与温度指标的背离情况

---

## 名词解释

### MA60（60 日移动平均线）

**定义**：过去 60 个交易日收盘价的平均值连成的曲线

**用途**：判断中期趋势的重要参考线

**解读**：

- 价格在 MA60 之上：中期趋势向上
- 价格在 MA60 之下：中期趋势向下

### 市盈率（PE Ratio）

**定义**：股价与每股收益的比率

**计算公式**：市盈率 = 股价 / 每股收益

**用途**：判断股票估值水平的重要指标

**解读**：

- 市盈率较低：可能被低估
- 市盈率较高：可能被高估

### 动量（Momentum）

**定义**：股票价格变化的速度和幅度

**用途**：衡量股价变化的强度

**解读**：

- 动量较高：股价变化较快，趋势较强
- 动量较低：股价变化较慢，趋势较弱

### 百分位（Percentile）

**定义**：数据在历史数据中的位置

**举例**：百分位为 80 表示当前值超过历史上 80% 的时间

**用途**：判断当前值的相对高低

## 买入信号判断

**信号条件**：

| 指标         | 买入信号条件 | 说明                         |
| ------------ | ------------ | ---------------------------- |
| 恐贪指数     | < 30         | 市场情绪恐惧，可能接近底部   |
| 趋势温度     | < 40         | 市场处于弱势区，可能接近底部 |
| 动量温度     | < 40         | 动量偏弱，超跌反弹机会       |

**信号数量与建议**：

| 信号数量   | 投资建议                       |
| ---------- | ------------------------------ |
| 1 个指标   | 提示不要跟风抛售               |
| 2 个指标   | 可以寻找超跌反弹机会           |
| 3 个指标   | 强烈建议分批买入或定投         |

## 使用注意事项

### 双维度分析框架

本 Skill 采用"长期估值 + 短期情绪"双维度分析框架：

| 维度 | 核心指标 | 时间周期 | 主要用途 | 操作频率 | 持有周期 |
|------|---------|---------|---------|---------|---------|
| 长期估值维度 | 估值温度 | 长期（月度/季度） | 判断市场制度、资产配置 | 低频 | 月度到年度 |
| 短期情绪维度 | 恐贪/趋势/动量 | 短期（周度） | 捕捉交易时机、仓位调整 | 中频 | 天到周 |

**交叉验证**：趋势温度和动量温度同属短期情绪维度，可相互验证：
- 趋势温度反映 MA60 之上股票占比（中期趋势强度）
- 动量温度反映 CS（20日EMA乖离率）市场中位数（短期动能）
- 两者同向变化时信号更可靠

### 估值底部特殊场景

当估值温度 < 20 时，进入"估值底部"场景：
- **忽略情绪指标**：此时情绪指标可能极度悲观，但不应影响决策
- **越跌越买**：加大定投力度，分批建仓
- **等待估值修复**：持有周期月度到年度
- **核心逻辑**：估值底部是长期布局的最佳时机

### 情绪短期操作要点

**收益率来源**：情绪指标的收益率来自情绪恐慌之后几天到2周以内的情绪反弹

**操作要点**：
- **建仓时机**：在情绪最低谷和拐点出现时建仓
- **持有周期**：天到周（情绪反弹通常在几天到2周内完成）
- **止盈策略**：情绪恢复到中性区域后考虑止盈
- **止损策略**：设置紧止损，防止情绪继续恶化

**情绪拐点识别**：
- 恐贪指数 < 10（极度恐惧）+ 开始回升 → 情绪拐点
- 趋势温度 < 20（极度低迷）+ 开始回升 → 趋势拐点
- 动量温度 < 20（动量太弱）+ 开始回升 → 动量拐点
- 多个情绪指标同时出现拐点时，信号更可靠

### 综合判断原则

1. **优先判断估值底部**：如果估值温度 < 20，直接进入估值底部场景
2. **多指标交叉验证**：不要仅根据单一指标做决策
3. **结合双维度框架**：区分长期估值和短期情绪的作用
4. **关注趋势变化**：不仅看当前值，还要看变化趋势
5. **承认不确定性**：指标只是参考，不构成投资建议
