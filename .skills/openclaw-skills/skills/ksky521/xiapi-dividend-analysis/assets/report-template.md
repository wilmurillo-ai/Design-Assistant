# 红利指数分析报告模板

---

## 报告规范

**必须遵守**：
- 结论先行：第一句话说清楚当前状态和操作方向，普通人能看懂
- 推导链完整：数据 → 判断依据 → 结论，不能只报分数
- 行情结构分析：必须覆盖高位、下行、低位、回升四个阶段
- 策略差异化：针对用户策略给出不同建议
- 数据可靠性诚实：历史数据不足时主动说明

---

### 3.4 当前回升状态

- 结论：[回升信号可信，可以介入 / 回升信号尚弱，建议等待确认]

## 七、风险提示

- 打分指标基于均值回归假设，如指数进入长期下行通道，低分不代表安全
- 历史数据量不足时，"历史低位"的参考意义有限
- 打分指标具有滞后性，可能错过最佳买卖时机
- 本分析仅供参考，不构成投资建议

## 报告质量检查清单

生成报告后逐项检查：

- [ ] 核心结论是否一句话说清楚当前状态和操作方向
- [ ] 是否分析了完整行情结构（高位→下行→低位→回升四个阶段）
- [ ] 每个阶段是否有"数据 → 推导过程 → 结论"三层结构
- [ ] 是否标注了顶部、底部、反转确认三个关键转折点
- [ ] 是否区分了急跌和慢跌，并说明操作逻辑差异
- [ ] 是否说明了底部形态（V形 vs 震荡）
- [ ] 是否针对用户策略给出了差异化建议
- [ ] 是否说明了历史数据量是否充足
- [ ] 是否包含风险提示和免责声明
