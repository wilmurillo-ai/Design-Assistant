# 字段说明

## 打分数据字段

| 字段名 | 类型   | 说明 |
|--------|--------|------|
| date   | string | 数据日期，格式 YYYY-MM-DD |
| score  | number | 综合打分，范围 0-100，**分数 0 是最强超卖信号，不代表无效** |
| cs     | number | CS 值，正值表示价格偏强，负值表示价格偏弱 |
| rsi    | number | RSI 值，范围 0-100 |

## 分数区间含义

| 分数范围 | 状态     | 含义 |
|----------|----------|------|
| 0–10     | 极度超卖 | 历史极值区间，难得的低位窗口 |
| 10–20    | 超卖     | 低位区间，值得重点关注，可开始分批布局 |
| 20–35    | 偏低     | 偏低但未到极值，可开始少量布局 |
| 35–65    | 正常     | 正常波动区间，无明显信号 |
| 65–80    | 偏高     | 偏高，持仓者注意控制仓位 |
| 80–90    | 超买     | 高位区间，逢高减仓 |
| 90–100   | 极度超买 | 历史极值区间，大幅减仓或清仓 |

## CS 值解读

CS 值衡量价格相对短期均线的偏离程度。

| CS 值范围 | 含义 |
|-----------|------|
| > 3       | 短期明显偏强，注意过热风险 |
| 0–3       | 短期偏强 |
| -2–0      | 短期偏弱 |
| < -2      | 短期明显偏弱，可能存在超卖 |

CS 值从低点回升趋近 0 轴，是底部回升的早期信号。

## RSI 值解读

RSI 衡量价格动量的相对强弱。

| RSI 范围 | 含义 |
|----------|------|
| < 45     | 动量偏弱 |
| 45–65    | 正常区间 |
| > 65     | 动量偏强 |

## 数据可靠性说明

打分系统基于历史数据做归一化计算。如果指数历史数据较短，参考系偏短，"历史低位"的参考意义需打折扣。建议结合指数净值走势和当前股息率水平做交叉验证。
