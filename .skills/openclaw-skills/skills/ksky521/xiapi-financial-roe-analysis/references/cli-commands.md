# CLI 命令与字段说明

## report finance 命令

### 命令格式

```bash
npx daxiapi-cli@latest report finance <code>
# 或本地安装后
daxiapi report finance <code>
```

### 参数说明

| 参数 | 说明 | 示例 |
|------|------|------|
| `<code>` | 股票代码（6位数字） | `600519`（贵州茅台）、`000858`（五粮液） |

### 返回字段清单

命令返回多个报告期（季报/年报）的财务数据，每期包含以下字段：

| 字段名 | 含义 | 单位 | 备注 |
|--------|------|------|------|
| 科目\时间 | 报告期 | - | 格式：YYYY-MM-DD，如 2024-12-31 |
| 净利润 | 归母净利润 | 亿元 | 核心盈利指标 |
| 净利润同比增长率 | 归母净利润同比增速 | % | |
| 扣非净利润 | 扣除非经常性损益后归母净利润 | 亿元 | 衡量主营盈利质量 |
| 扣非净利润同比增长率 | 扣非净利润同比增速 | % | |
| 营业总收入 | 营业总收入 | 亿元 | |
| 营业总收入同比增长率 | 营收同比增速 | % | |
| 基本每股收益 | EPS | 元/股 | |
| 每股净资产 | BPS | 元/股 | |
| 每股资本公积金 | - | 元/股 | |
| 每股未分配利润 | - | 元/股 | |
| 每股经营现金流 | 每股经营活动现金流净额 | 元/股 | 可用于估算经营现金流总额 |
| 销售净利率 | 归母净利润 / 营业收入 | % | 杜邦因子之一 |
| 销售毛利率 | 毛利润 / 营业收入 | % | |
| 净资产收益率 | 加权ROE | % | 核心指标 |
| 净资产收益率-摊薄 | 摊薄ROE | % | |
| 营业周期 | 存货周转天数 + 应收账款周转天数 | 天 | |
| 存货周转率 | 营业成本 / 平均存货 | 次/期 | |
| 存货周转天数 | 报告期天数 / 存货周转率 | 天 | |
| 应收账款周转天数 | 应收账款 / 日均营收 | 天 | |
| 流动比率 | 流动资产 / 流动负债 | 倍 | |
| 速动比率 | (流动资产-存货) / 流动负债 | 倍 | |
| 保守速动比率 | (货币资金+短期投资) / 流动负债 | 倍 | |
| 产权比率 | 总负债 / 归母净资产 | 倍 | |
| 资产负债率 | 总负债 / 总资产 | % | |

### 数据覆盖范围

- **历史区间**：通常覆盖近 6-7 年的季报和年报（约 25 个报告期）
- **报告期类型**：季报（Q1/Q2/Q3）+ 年报（Q4/全年），均为累计口径
- **数据口径**：合并报表口径，归母净利润/净资产

### 字段缺失说明

以下字段**不在**命令返回范围内，分析时需明确标注"数据不足"：

| 缺失字段 | 影响的分析模块 | 处理方式 |
|--------|------------|--------|
| 总资产（绝对值） | 总资产周转率精确计算 | 可用每股净资产×股本估算净资产，但总资产需降级处理 |
| 经营活动现金流净额（绝对值） | 盈利含金量校验 | 可用每股经营现金流×股本估算 |
| 资本开支 | 自由现金流计算 | 无法精确计算，需降级说明 |
| WACC相关参数（Beta、无风险利率、债务成本） | EVA/WACC分析 | 跳过该模块，标注"数据不足" |
| 分业务收入/毛利率 | 分业务ROE拆解 | 跳过或降级为定性分析 |
| 母公司报表数据 | 合并vs母公司ROE对比 | 跳过该对比 |
| 同行竞争对手数据 | 行业对标 | 跳过或标注"待补充" |

### 使用示例

```bash
# 获取贵州茅台财报数据
npx daxiapi-cli@latest report finance 600519

# 获取五粮液财报数据
npx daxiapi-cli@latest report finance 000858

# 获取招商银行财报数据（金融行业，注意使用金融行业框架）
npx daxiapi-cli@latest report finance 600036
```

### 常见错误

| 错误 | 原因 | 解决方式 |
|------|------|--------|
| 401 Unauthorized | Token 未配置或已过期 | 执行 `daxiapi config set token <YOUR_TOKEN>` |
| 429 Too Many Requests | 请求频率超限（每分钟10次） | 等待1分钟后重试 |
| 返回空数据 | 股票代码错误或该股票无财报数据 | 确认股票代码，或使用 `daxiapi search <公司名>` 查询 |
