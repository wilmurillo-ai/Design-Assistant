# 财务深度分析报告输出模板

## 报告格式规范

- 所有核心时间序列数据、对标数据必须以表格形式列示
- 所有核心结论、异常风险、核心驱动因素必须**加粗标注**
- 数据来源必须标注，并至少包含 `daxiapi report finance <code>` 与巨潮资讯财报 PDF 公告
- 严禁提供股票投资建议、股价涨跌预测
- 报告开头必须优先输出核心结论摘要

---

## 深度分析版报告结构

```markdown
# [公司名称]（[股票代码]）财务深度分析报告

**分析日期：** XXXX年XX月XX日
**数据截止：** [YYYY-MM-DD] 收盘（必填具体交易日）
**数据来源：** `daxiapi report finance <code>` + 巨潮资讯财报 PDF（公告标题/期次）
**最新报告期：** [如：2024年报 / 2024Q3 / 2024半年报]（必填，说明财务数据截止到哪一期）
**分析版本：** 深度分析版
**免责声明：** 本报告仅为财务与业务解读，不构成任何投资建议。
```
