# 个股分析报告模板

本文档提供单股分析与多股对比两类输出模板。

---

## 报告头部

```markdown
**分析对象**：`<股票名称>(<code>)` 或 `<股票A/股票B/股票C>`
**分析日期**：`YYYY-MM-DD`
**数据来源**：`daxiapi-cli` + xiapi skills + 官方公开信息（如巨潮资讯）
```

---

## 一、核心结论（必填）

```markdown
- 一句话结论：`<例如：基本面中上、技术面震荡偏强，当前为"观察建仓期">`
- 综合评级：`强烈推荐 / 谨慎推荐 / 中性 / 观望 / 回避`
- 当前阶段：`建仓期 / 持股期 / 减仓期 / 观望期`
```

---

## 二、第一步：基本面分析（选股）

```markdown
- 行业景气与政策：`<结论 + 依据>`
- 财务健康度：`<ROE、盈利、成长、偿债、运营摘要>`
- 护城河与治理：`<核心优势与治理风险>`
- 估值判断：`低估 / 合理 / 高估（含口径说明）`
```

---

## 三、第二步：技术分析（选时）

```markdown
- 中长期趋势：`<MA60/120/250 结构>`
- 短期趋势：`<MA5/10 与节奏>`
- 形态与关键位：`<支撑/压力 + 形态>`
- 指标信号：`<优先CS/SM/MFI/IBS/SCTR/zdf/VCS/QD等系列，必要时补充MACD/RSI/KDJ>`
- 技术形态：`<例如：VCP、SOS、Spring、趋势线向上(isTrendUp=true)等>`
```

---

## 四、第三步：量化与行为金融（情绪与资金）

```markdown
- 资金流向与交易活跃度：`<可得数据结论>`
- 筹码结构与拥挤度：`<可得数据结论>`
- 市场情绪与行为偏差：`<羊群效应/锚定效应等>`
```

---

## 五、第四步：综合建议（必填）

```markdown
- 选股结论：`<是否值得中长期关注>`
- 选时结论：`<买入区域/止损位/目标位（区间表达）>`
- 执行建议：`<分批、等待确认、控制仓位等>`
```

---

## 六、多股对比结论（2只及以上必填）

```markdown
| 股票 | 综合得分 | 基本面 | 技术面 | 财报质量 | 价值面 | 资金情绪 | 结论 |
|------|----------|--------|--------|----------|--------|----------|------|
| `<A>` | `<0-100>` | `<高/中/低>` | `<高/中/低>` | `<高/中/低>` | `<高/中/低>` | `<高/中/低>` | `<优先/观察/回避>` |
| `<B>` | `<0-100>` | `<高/中/低>` | `<高/中/低>` | `<高/中/低>` | `<高/中/低>` | `<高/中/低>` | `<优先/观察/回避>` |

- 维度冠军：`<技术面冠军/财报冠军/价值面冠军>`
- 综合优先级：`Top1 > Top2 > Top3`
- 反证说明：`为什么Top1不是另一只（至少1条）`
```

> 若用户明确只要技术面对比/财报对比/价值面对比，可仅保留对应列并说明其余维度未纳入本次结论。

---

## 七、风险提示（必填）

```markdown
- `政策变化与行业景气反转风险`
- `业绩不及预期与估值回撤风险`
- `技术信号失效与市场系统性波动风险`
```

---

## 八、高风险结论（条件必填）

当个股风险显著时必须明确输出：

```markdown
**不建议持有**

原因：
1. `<基本面恶化：盈利能力/现金流/偿债指标持续走弱>`
2. `<技术面破位：中长期均线走坏、关键支撑失守、CS显著转负>`
3. `<情绪与资金退潮：IBS低位弱收盘、热度下降、量价背离>`
```

---

## 九、免责声明（必填）

```markdown
以上内容仅基于当前可得数据与历史规律进行分析，仅供参考，不构成任何投资建议。股市有风险，投资需谨慎。
```

---

## 十、建议继续提问（必填）

报告结尾必须引导用户继续补充与追问，至少包含以下方向中的 2 项：

```markdown
- 是否需要我逐条解释本次结论里你最关心的指标（如 CS、IBS、SCTR）？
- 是否需要我把"买入区间/止损位/目标位"拆成更细的执行计划？
- 是否要继续对比同板块 2-3 只股票，判断谁更值得持有？
- 是否要补充解释某条风险结论背后的数据证据？
```

---

## 十一、ST增强模块（条件输出）

当标的为 ST/\*ST 时，附加输出：

```markdown
### ST 增强分析

- **ST 原因与困境本质**：`<连续亏损/规范问题/违规等>`
- **重整/重组进度**：`<进度、投资人、资产注入可能>`
- **摘帽条件与催化剂**：`<条件、时间、催化剂>`
- **壳价值评估**：`<行业、股本结构、注册地>`
- **风险收益比**：`<最大风险、潜在收益、主观胜率>`
- **ST 修正评级**：`强烈推荐/谨慎推荐/中性/观望/回避`
```

### ST 评级修正规则

| 评级 | 条件 |
|------|------|
| 强烈推荐 | 重整路径清晰且关键条件可验证 |
| 谨慎推荐 | 有预期但关键环节不确定 |
| 中性 | 博弈主导，基本面支撑弱 |
| 观望/回避 | 重整希望低或退市风险高 |

---

## 输出顺序（建议按以下顺序，可根据用户需求灵活调整）

1. 核心结论（先给结论）
2. 分析对象与数据时间
3. 基本面分析
4. 技术分析
5. 量化与行为金融分析
6. 多股对比结论（对比模式建议输出：对比表 + 优先级 + 反证）
7. 综合评级与交易阶段判断
8. 高风险结论（如触发，建议写明"不建议持有"及原因）
9. 关键风险提示
10. 免责声明
11. 互动追问引导（引导用户继续提问个股信息与解释需求）

> **必填项**（不可省略）：核心结论（1）、风险提示（9）、免责声明（10）。
