# 微信文章自动保存 — 使用指南

## 工作原理

当你在微信发送微信公众号文章链接时，我会自动：

1. **检测** — 识别链接是否是微信文章（`https://mp.weixin.qq.com/s/...`）
2. **抓取** — 获取文章标题、正文、链接
3. **摘要** — 智能生成要点总结
4. **保存** — 写入 Markdown 到 Miaoyan/待学习
5. **回复** — 微信确认保存完成

## 使用方式

### 方式 1：直接发链接（推荐）

在微信里直接发送文章链接：

```
https://mp.weixin.qq.com/s/j_P3vf_zCDF6GpAZ_bM-0w
```

我会自动检测并保存。

### 方式 2：带文字发送

```
保存文章 https://mp.weixin.qq.com/s/j_P3vf_zCDF6GpAZ_bM-0w
```

或

```
https://mp.weixin.qq.com/s/j_P3vf_zCDF6GpAZ_bM-0w
保存这篇
```

都可以自动触发。

### 方式 3：手动调用

```bash
python3 auto_save_article.py "https://mp.weixin.qq.com/s/..."
```

## 保存位置

所有文章都保存到：

```
~/Library/Mobile Documents/iCloud~com~tw93~miaoyan/Documents/待学习/
```

文件名格式：`[文章标题].md`

## 文件内容

每个保存的文件包含：

```markdown
# 📌 要点

[自动生成的摘要，提取关键句]

---

# 正文

[完整文章内容]

---

# 源链接

[原文链接]

保存时间: 2026-03-23
```

## 自动触发规则

我会在以下情况自动保存：

- ✅ 你发送 `https://mp.weixin.qq.com/s/...` 链接
- ✅ 链接在消息中间或末尾
- ✅ 链接前后有其他文字也没关系
- ✅ 一条消息中有多个链接会全部保存

## 不会触发的情况

- ❌ 其他网站的链接（只识别微信公众号）
- ❌ 已损坏或过期的链接
- ❌ 需要登录的文章

## 常见问题

### Q: 保存失败了怎么办？

A: 我会回复错误信息。常见原因：
- 网络超时（15秒内无响应）
- 文章已删除或链接过期
- Miaoyan 文件夹不存在

### Q: 可以改变保存位置吗？

A: 可以。编辑 `save_article_to_miaoyan.py` 中的 `MIAOYAN_DIR` 变量。

### Q: 摘要生成不好怎么办？

A: 编辑 `generate_summary()` 函数，调整关键词识别规则。

### Q: 支持其他笔记应用吗？

A: 目前只支持 Miaoyan。可以修改脚本支持其他应用（如 Obsidian、Notion）。

## 技术细节

- **抓取方式** — curl + 正则表达式（绕过 SSRF）
- **超时设置** — 15 秒
- **编码** — UTF-8
- **并发** — 单线程顺序处理

## 隐私说明

- 文章内容仅保存到本地 iCloud
- 不上传到任何云服务
- 不记录或分析文章内容
