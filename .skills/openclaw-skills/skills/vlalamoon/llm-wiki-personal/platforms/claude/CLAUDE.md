# Claude Code 入口

这是 Claude Code 的薄入口文件。共享说明看 [../../README.md](../../README.md)，核心能力看 [../../SKILL.md](../../SKILL.md)。

## Claude 应该怎么装

优先执行：

```bash
bash install.sh --platform claude
```

默认安装位置：`~/.claude/skills/llm-wiki`

## 兼容入口

老用户仍然可以继续执行：

```bash
bash setup.sh
```

它现在会走同一套安装流程，不再单独维护另一份逻辑。
