# Meta-Router: Automated Cognitive Gateway ⚡

> **The Problem:** The more skills you install, the slower and "dumber" your Agent becomes because it tries to memorize every manual simultaneously.
> **The Solution:** Meta-Router builds an automated index for your OpenClaw environment. It reads the "Table of Contents" first, and only opens the "Chapter" you need.
> **The Goal:** Zero token waste. Instant response.

---

### 🚀 Key Features

* **Auto-Triggered Scanning**: Meta-Router detects environment changes. Every time you install a new skill, it automatically updates its internal index—no manual config required.
* **Lazy Loading**: Skills are only loaded into the context window when they are actually needed.
* **Instant Routing**: Supports shortcuts for power users to bypass the routing logic entirely.
