# 飞书机器人创建指南

## 步骤 1：创建应用

1. 访问 [飞书开放平台](https://open.feishu.cn/)
2. 点击「创建应用」
3. 填写应用名称和描述
4. 点击「创建」

## 步骤 2：添加机器人能力

1. 进入应用详情页
2. 左侧菜单点击「添加能力」
3. 选择「机器人」
4. 点击「开通」

## 步骤 3：获取凭证

1. 左侧菜单点击「凭证与权限」
2. 在「应用凭证」区域可以找到：
   - **App ID** - 格式 `cli_xxxxx`
   - **App Secret** - 点击「获取」按钮查看

## 步骤 4：配置事件订阅（可选）

如果需要接收消息：

1. 左侧菜单点击「事件订阅」
2. 点击「添加事件」
3. 添加以下事件：
   - `im.message.receive_as_bot` - 接收消息
   - `im.message.send_as_bot` - 发送消息
4. 点击「发布」

## 步骤 5：添加机器人到群聊

1. 在飞书中打开目标群聊
2. 点击群设置 → 添加群机器人
3. 搜索并添加你的应用

## 常见问题

### Q: App Secret 在哪里看？
A: 应用详情页 → 凭证与权限 → 应用凭证 → App Secret（需要先创建应用）

### Q: 机器人收不到消息怎么办？
A: 检查事件订阅是否添加了 `im.message.receive_as_bot` 并发布

### Q: 如何测试机器人？
A: 在飞书中 @机器人 发送消息，看是否收到回复
