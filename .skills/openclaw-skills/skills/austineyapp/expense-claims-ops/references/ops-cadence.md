# Ops Cadence (Lightweight)

## Weekly cadence
- Monday: collect previous week receipts
- Wednesday: first-pass categorization + block detection
- Friday: finalize claim-ready pack + submit

## Fast daily routine (5-10 min)
1. Add new receipts
2. Tag category + purpose
3. Resolve one blocked item
4. Keep blocked_count trending down

## Escalation rule
If blocked_count > 5 for 3+ days, trigger a focused 20-min cleanup session.
