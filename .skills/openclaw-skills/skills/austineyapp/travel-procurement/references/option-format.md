# Option Format

Per option:
- Option: A/B/C
- Provider + mode
- Departure → arrival
- Duration
- Price (all-in)
- Inclusions/exclusions
- Cancellation terms
- Capacity fit (pax/luggage)
- Risk note

Recommendation:
- Best value
- Best convenience
- Best fallback
- Decision ask: “Reply A/B/C to proceed.”
