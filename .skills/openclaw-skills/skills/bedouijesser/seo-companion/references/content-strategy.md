# Content Strategy & Keyword Reference

Use this file for keyword targeting, content planning, cannibalization analysis, and rewrite strategy.

## Principles

- One page should usually target one primary intent/topic cluster.
- Supporting variations belong on the same page when they share intent.
- Separate pages are justified when intents materially differ.
- Content quality is not just length; it is usefulness, specificity, and fit to the SERP.

## Keyword selection

Evaluate keywords with:
- intent
- business relevance
- achievable competition
- SERP quality/type
- existing site authority

### Avoid
- chasing only vanity high-volume terms
- treating KD as an objective universal truth
- creating multiple pages for slight phrasing variants with identical intent

## Clustering

Group variants together when they share:
- the same likely SERP
- the same solution intent
- overlapping subtopics

Split them when users clearly want different outcomes.

## Cannibalization

Suspect cannibalization when:
- multiple pages target near-identical intent
- pages swap rankings for the same query set
- internal anchors and titles overlap heavily

### Typical remedies
- merge weaker page into stronger page
- reposition one page for a distinct intent
- improve internal linking / canonical clarity

## Content quality review

Assess:
- whether the page answers the user’s real question
- whether it adds original clarity or evidence
- whether headings cover the core subtopics
- whether examples, proof, screenshots, or unique insight exist
- whether the page is mostly filler

## Thin content

Treat low word count as a clue, not a verdict.

### Pages often okay to be brief
- contact pages
- simple conversion pages
- utility pages
- thank-you pages

### Pages usually needing more depth
- service pages expected to rank
- category pages targeting search demand
- educational blog posts
- comparison pages

## Recommended deliverables

### Keyword opportunity brief
Include:
- primary topic
- supporting variants
- intent class
- reason to target now
- page type recommendation
- suggested outline

### Content rewrite brief
Include:
- current page problem summary
- missing subtopics
- stronger title / H1 direction
- heading outline
- internal links to add
- evidence/proof to include
