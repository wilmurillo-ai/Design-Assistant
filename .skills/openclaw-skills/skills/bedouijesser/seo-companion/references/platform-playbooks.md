# Platform Playbooks

This file contains platform-specific notes that are useful but perishable.

## Use this file only when needed

Read this file when the user specifically asks about:
- Google Search Console setup or reports
- GA4 setup or report navigation
- Bing Webmaster Tools setup
- Looker Studio dashboard wiring
- tool-specific workflows in SEMrush / Ahrefs / Ubersuggest / BrightLocal

## Guidance

- Treat menu paths and pricing as time-sensitive.
- Prefer describing goals and outputs over click-by-click UI narration unless the user needs exact setup help.
- If a UI path may have changed, say so.

## Tool notes

### Google Search Console
Use for:
- query/click/impression validation
- coverage/indexation clues
- sitemap submission
- page performance checks

### GA4
Use for:
- landing page performance
- organic traffic patterns
- engagement diagnostics

### Bing Webmaster Tools
Useful for:
- sitemap submission
- additional query ideas
- secondary search engine visibility

### Third-party SEO suites
Useful for:
- keyword discovery
- backlink snapshots
- site crawls
- competitor gap analysis

But do not treat tool estimates as ground truth when first-party data exists.
