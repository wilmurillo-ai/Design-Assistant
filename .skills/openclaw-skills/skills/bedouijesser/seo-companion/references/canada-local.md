# Canada Local Notes

Use this only for Canada-specific local SEO work.

## Notes

- Directory opportunities and citation sources vary by province/city and change over time.
- Prioritize major authoritative platforms and sector-specific directories over dumping a business into every possible directory.
- Review each directory for legitimacy and live traffic before recommending it.

## Useful categories to check

- Google Business Profile
- Apple Maps
- Bing Places
- Facebook
- Yelp
- Yellow Pages
- niche/trade-specific directories
- chamber of commerce / local associations

## Rule

Do not recommend mass low-quality directory submissions just because a list exists.
