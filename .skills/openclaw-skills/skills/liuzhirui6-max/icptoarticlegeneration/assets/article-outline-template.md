## 质量检查项

### ICP匹配度
- [ ] 标题明确目标受众
- [ ] 痛点描述符合ICP
- [ ] 语言风格适配ICP偏好
- [ ] 解决方案针对ICP需求

### 内容价值
- [ ] 有独特观点或方法
- [ ] 提供可操作的建议
- [ ] 有充分的数据/案例支撑
- [ ] 信息密度适中，无废话

### 可读性
- [ ] 结构清晰，层次分明
- [ ] 小标题准确概括内容
- [ ] 段落长度适中（不超过200字）
- [ ] 使用列表、加粗等格式优化阅读体验

### 行动召唤
- [ ] CTA明确具体
- [ ] 说明了行动收益
- [ ] 降低了行动门槛
- [ ] CTA与文章价值主张一致
