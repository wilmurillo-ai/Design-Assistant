# 柯南周报 📺🔍

**名侦探柯南动画周报技能** - 每周自动收集并整理柯南最新剧情进展

---

## 🎯 功能特点

- **自动搜索** - 每周六晚 9 点自动搜索最新剧情
- **主线追踪** - 关注黑衣组织、APTX-4869 等主线剧情
- **特别篇报道** - 剧场版、特别篇、联动剧集信息
- **角色动态** - 声优变动、角色登场、制作信息
- **多源整理** - 整合知乎、百科、官方等多渠道信息

---

## ⏰ 默认执行时间

- **每周六 21:00** (Asia/Shanghai UTC+8)
- 可通过 cron 配置修改

---

## 📋 输出内容

每期周报包含：

1. **新播出集数** - 集数、标题、播出日期
2. **重大剧情进展** - 主线剧情推进
3. **特别篇信息** - 剧场版、特别篇、联动
4. **角色/声优动态** - 登场角色、声优变动
5. **制作信息** - 编剧、导演、制作团队
6. **来源链接** - 方便查看详情

---

## 🚀 使用方法

### 手动触发
```
柯南周报
```

### 自动执行（推荐）
```bash
# 添加定时任务（每周六 21:00）
openclaw cron add \
  --name "柯南周报" \
  --cron "0 21 * * 6" \
  --tz "Asia/Shanghai" \
  --system-event "柯南周报" \
  --session "main"
```

### 查看已安装
```bash
openclaw skills list | grep conan
```

## 🔍 搜索源

技能使用 **DuckDuckGo HTML 搜索接口** 进行资讯收集：

- **无需 API 密钥** - 直接使用公开 HTML 接口
- **真实 HTTP 请求** - 使用 Node.js `https` 模块发起实际网络请求
- **结果解析** - 正则表达式提取标题和链接
- **请求限流** - 内置 1.5 秒延迟避免请求过快
- **超时保护** - 10 秒超时防止卡死

搜索结果可能来自以下渠道（取决于搜索查询）：
- 百度百科、维基百科 - 集数列表和剧情简介
- 哔哩哔哩 - 播出信息和预告
- 新浪微博 - 声优和制作动态
- 动漫资讯站 - 新番放送表

## ⚙️ 配置选项

### 修改执行时间
编辑 cron 任务：
```bash
openclaw cron edit <任务 ID> --cron "0 20 * * 6"
```

### 修改时区
```bash
openclaw cron edit <任务 ID> --tz "Asia/Tokyo"
```

### 暂停/恢复
```bash
openclaw cron disable <任务 ID>  # 暂停
openclaw cron enable <任务 ID>   # 恢复
```

## 📌 注意事项

1. **网络依赖** - 需要联网搜索最新信息
2. **搜索限制** - 部分网站可能有反爬限制
3. **时区设置** - 确保 cron 时区正确
4. **推送渠道** - 需配置 QQ/Telegram 等推送
