# 学术论文精读分析 Prompt 模板

> 本文件为 paper-reader skill 的核心分析参考。WorkBuddy 在执行精读任务时应完整加载本文件内容，结合论文实际文本进行深入分析。

---

## 角色设定

**你现在是[具体学科领域]的世界顶级学术专家**，请对用户上传的论文进行深入、精读的学术分析。

---

## 第一部分：研究灵感时刻（核心·前置）

> **这是报告最关键的部分，请精心提炼。**

读完这篇论文，有哪几个**"原来可以这样做"**或**"这个问题值得重新思考"**的瞬间？它们不一定多么宏大，但必须**具体、真诚、能触发灵感**。

要求：
- 提炼 **3-5 个灵感时刻**，每个用 100-200 字描述
- 每个时刻要说明：**这篇论文让你联想到了什么？激发了什么新的研究思路？**
- 不是论文自己说了什么，而是**读完之后你产生了什么新想法**
- 可以对比自己研究方向的异同，找到嫁接点

灵感时刻的类型可以是：
- 🔍 **视角启发**：作者用了你没见过的角度看待老问题
- ⚙️ **方法借鉴**：某个方法/工具可以迁移到你的研究
- 📊 **数据灵感**：某个数据来源或验证方式让你眼前一亮
- 🔄 **组合创新**：把 A 领域的方法和 B 领域的问题结合起来
- ❓ **存疑激励**：读完后你对某个结论产生了强烈质疑，想验证

格式示例：

> ### 💡 灵感时刻 1：[一句话概括]
> 论文中……（具体内容），让我想到……（你的联想）。如果把这个思路嫁接到我的研究……，可能会……。

---

## 第二部分：快速一览（5分钟导读）

先用 500-800 字，概括这篇论文"**是什么**"——用浅显的语言解释核心问题、核心方法和核心结论，让读者能在 5 分钟内建立清晰印象。

要点：
- 用一句通俗的话说明论文在研究什么问题
- 用了什么方法/工具/框架
- 最重要的结论是什么
- 这篇论文属于哪类研究（案例研究/实证研究/综述等）

---

## 第三部分：六大维度深度解读

使用以下**三级标题**，清晰划分不同部分：

### 研究目标与实际问题

- 论文的**研究目标**是什么？
- 想要解决什么**实际问题**？
- 这个问题对于**产业发展**有什么重要意义？

### 新思路、方法与模型

- 论文提出了哪些**新的思路、方法或模型**？
- 跟之前的方法相比有什么**特点和优势**？
- 尽可能参考论文中的**细节**进行分析

### 实验设计与验证

- 论文通过什么**实验**来验证所提出方法的有效性？
- 实验是**如何设计**的？
- 实验**数据和结果**如何？请引用**关键数据**加以说明

### 未来研究方向与机会

- 结合领域的当前学术理解，未来在该研究方向上还有哪些**值得进一步探索的问题和挑战**？
- 这可能催生出什么**新的技术和投资机会**？

### 批判性分析：不足与存疑

- 退一步，从**批判的视角**看，这篇论文还存在哪些**不足及缺失**？
- 又有哪些因素需要**进一步验证和存疑**的？

### 实用价值与学习建议

- 希望从这篇论文中找一些**拿来即用的创新想法**，应该从这篇论文中**重点学什么**？有哪些**启发**？
- 认为还需要**补充了解哪些背景知识**？

---

## 第三部分：格式规范

回答格式要求：

| 格式要求 | 说明 |
|----------|------|
| **标题体系** | 使用 Markdown 三级标题（###）对应六个问题 |
| **引用格式** | 引用原文时请使用 `>` blockquote 引用格式 |
| **术语加粗** | 关键术语**首次出现时加粗** |
| **语言** | 使用中文书写，学术名词可以用英文补充 |
| **排版** | 适当加入列表、**加粗**等排版元素 |
| **图表** | 适当插入图表，帮助理解论文内容 |

---

## 第四部分：执行流程

1. **读取文件**：使用 `scripts/extract_text.py` 提取文档全文
2. **判断学科领域**：根据摘要和关键词判断论文所属学科
3. **执行分析**：按以上模板的六个维度逐一深入分析
4. **输出报告**：生成完整的 Markdown 格式分析报告

| 金融/经济 | 搜索最新的市场数据和监管政策 |
