# 高中地理提分技能 | Geography Score Improvement

🔥 专为高中地理学习打造的专业提分技能，基于1425份权威地理资料（课标、教材、真题、讲义）训练，内容精准权威，杜绝通用大模型的超纲/错误内容，覆盖高中地理全学习场景。

## ✨ 核心功能

### 🆓 永久免费功能（无次数/范围限制）
1. **多模态基础精准答疑**
   - 支持文字、拍照、语音三种提问方式
   - 精准解释地理概念、原理、课标要求，标注对应教材章节
   - 基础题目解析：提供客观题答案、主观题核心要点、考点定位、基础解题步骤

2. **教材&大纲同步查询**
   - 全国各版本高中地理教材（人教版/湘教版/鲁教版等）章节框架、知识点目录、学习目标
   - 教育部课标要求、高考必考/选考范围、考点层级划分

3. **地理基础工具包**
   - 高考必备公式大全（时间计算、正午太阳高度、比例尺等）
   - 核心图表判读规则（等值线、气候图、洋流图、人口金字塔图等）
   - 速记口诀、核心分界线汇总、高频名词解释

4. **基础错题收录**
   - 一键收录提问过的题目，手动添加错题
   - 自动标注对应知识点，支持按知识点/时间分类查看

### 🎁 限量体验功能
1. **高考真题体验库**：近3年全国卷真题选择题基础解析，每个模块限免5道经典真题
2. **提分技巧体验包**：通用答题规范、入门级答题技巧，每个题型限免1道例题演示
3. **高频易错点科普**：高考高频易错点基础清单，标注易错概念、避坑提示

## 🎯 适用场景
- 📚 日常学习：知识点答疑、教材查询、预习复习辅助
- ✍️ 做题练习：题目解析、考点定位、错题整理
- 📝 考前复习：公式查询、速记口诀、易错点回顾
- 🎒 随身工具：随时随地查询地理知识点、图表判读规则

## 🚀 安装方法
### 方法1：ClawHub商店安装（推荐）
1. 打开OpenClaw技能商店，搜索「地理提分」
2. 点击安装即可直接使用

### 方法2：本地导入
1. 下载 `geography-score-improvement.skill` 文件
2. 在OpenClaw管理后台选择「导入本地技能」，选择下载的文件
3. 导入完成后即可使用

### 方法3：源码部署
1. 克隆本仓库到OpenClaw技能目录
2. 重启OpenClaw服务，技能会自动加载
