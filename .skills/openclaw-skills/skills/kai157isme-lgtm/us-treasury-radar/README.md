## 运行方式

```bash
python3 ~/workspace/agent/skills/us-treasury-radar/radar.py
```

## 核心指标说明

| 指标 | 含义 |
|------|------|
| Δ (Delta) | 每月新增债务规模 |
| Γ (Gamma) | 债务增长速度的变化率 |
| MoM | 环比变化 |
| YoY | 同比变化 |

## 信号判定

- **戴维期双杀**：Γ > 0 + 外资 MoM < 0
- **空头警告**：发债速度非线形扩张
- **去美元化**：外资持续减持
