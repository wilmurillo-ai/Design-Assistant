### 测试 5：跨渠道同步 ✅

**测试代码**:
```python
from memory_plus import MemoryPlus

mp = MemoryPlus()
mp.connect()

# 飞书消息
feishu_messages = [
    {
        'content': '老板：今天下午 3 点开会',
        'sender': '李威',
        'timestamp': '2026-04-07 14:30:00',
        'message_id': 'msg_001'
    }
]

mp.sync_from_channel('feishu', feishu_messages)

# 微信消息
wechat_messages = [
    {
        'content': '小花朵：妈妈，老师表扬我了',
        'sender': '小花朵',
        'timestamp': '2026-04-07 16:00:00',
        'message_id': 'wx_001'
    }
]

mp.sync_from_channel('wechat', wechat_messages)

stats = mp.get_stats()
print(f"总记忆块：{stats['total_chunks']}")
print(f"本次写入：{stats['session_writes']}")

mp.close()
```

**结果**:
