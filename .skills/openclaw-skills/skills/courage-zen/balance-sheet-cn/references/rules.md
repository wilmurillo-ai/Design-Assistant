# 资产负债表生成规则详解

## 规则执行顺序

**重要**: 规则必须按 1→2→3→...→11 的顺序执行，顺序错误会导致数据不正确。

---

## 规则 1：保留格式

**内容**: 采用文件中的资产负债表的格式。表格中非数字的单元格中的文字保留。表格中的 EXCEL 公式也保留。

**操作**:
- 复制所有单元格的样式（字体、边框、填充、对齐方式）
- 复制列宽和行高
- 复制合并单元格
- 保留所有公式（不计算，保持公式形式）
- 保留所有文字内容（如标题、科目名称、备注等）

**例外**: 数字单元格按后续规则处理。

---

## 规则 2：累计金额→上月余额

**内容**: 把累计金额一列的数值覆盖到上月余额中。

**操作**:
1. 读取 D 列（累计金额）的**计算后数值**（不是公式）
2. 将数值复制到 B 列（上月余额）对应行

**示例**:
- D7 = 258498.64（数值）→ B7 = 258498.64
- D4 = D5+D6+D9（公式）→ 计算后为 1961051.19 → B4 = 1961051.19

**注意**: 此规则在规则 4-5 **之前**执行，使用的是源文件 D 列的原始值。

---

## 规则 3：固定资产合计

**内容**: "固定资产合计"行的"本月发生额"项填数值 0。

**操作**: C5 = 0

---

## 规则 4：招商银行

**内容**: "招商银行"行的"累计金额"项填写的数值为：本文件明细表中 M 列"招商银行"行对应的"余额"项数值的总计。

**操作**:
1. 遍历明细表所有行
2. 查找 M 列包含"招商银行"的行
3. 累加这些行对应的 A 列（余额）数值
4. 将结果填入 D7

**示例**:
- Row 59: M=招商银行，A=27780.68
- Row 64: M=招商银行，A=201062.35
- D7 = 27780.68 + 201062.35 = 228843.03

---

## 规则 5：交通银行

**内容**: "交通银行（公）"行的"累计金额"项填写的数值为：本文件明细表中 M 列"交通银行"行对应的"余额"项数值的总计。

**操作**:
1. 遍历明细表所有行
2. 查找 M 列包含"交通银行"的行
3. 累加这些行对应的 A 列（余额）数值
4. 将结果填入 D8

**示例**:
- Row 68: M=交通银行，A=78128.7
- D8 = 78128.7

---

## 规则 6：应收款合计

**内容**: "应收款合计"行的"本月发生额"项填写的数值为：本文件明细表中 C 列"应收"行对应的"发生金额"项数值的总计。

**操作**:
1. 遍历明细表所有行
2. 查找 C 列等于"应收"的行
3. 累加这些行对应的 H 列（发生金额）数值
4. 将结果填入 C9

**示例**:
- Row 29: C=应收，H=200000
- C9 = 200000

---

## 规则 7：预收款

**内容**: "预收款"行的"本月发生额"项填写的数值为：本文件明细表中 C 列"预收"行对应的"发生金额"项数值的总计。

**操作**:
1. 遍历明细表所有行
2. 查找 C 列等于"预收"的行
3. 累加这些行对应的 H 列（发生金额）数值
4. 将结果填入 C11

**示例**:
- 多行 C=预收，H 值分别为 4900, 4900, 4900, 2450, 29800...
- C11 = 所有 H 值之和 = 201736

---

## 规则 8：应付款

**内容**: "应付款"行的"本月发生额"项填写的数值为：本文件明细表中 C 列"应付"行对应的"发生金额"项数值的总计。

**操作**:
1. 遍历明细表所有行
2. 查找 C 列等于"应付"的行
3. 累加这些行对应的 H 列（发生金额）数值
4. 将结果填入 C12

**示例**:
- 如果没有 C=应付 的行
- C12 = 0

---

## 规则 9：当年利润

**内容**: "当年利润"行的"本月发生额"项填写的数值为：本文件利润表中 B 列"经营利润"行对应的 E 列数值。

**操作**:
1. 在利润表中查找 B 列包含"经营利润"的行
2. 读取该行 E 列的数值（计算后的值）
3. 将结果填入 C14

**示例**:
- 利润表 Row 26: B=经营利润，E=29583.54
- C14 = 29583.54

---

## 规则 10：未分配利润

**内容**: "未分配利润"行的"本月发生额"项填写数值 0。

**操作**: C15 = 0

---

## 规则 11：公式计算

**内容**: 有计算公式的单元格采用公式计算。

**操作**:
- 保留所有原有公式
- C 列公式恢复：
  - C4 = D4-B4
  - C6 = C7+C8
  - C7 = D7-B7
  - C8 = D8-B8
  - C10 = C11+C12
  - C13 = C14+C15
  - C16 = D16-B16

**注意**: 公式中的单元格引用会自动使用新值计算。

---

## 规则 1 清理说明

规则 1 提到"其他除了【美金账户 (美金)】所在行的数字、'累计金额'一列的数字，其他数字都删除"。

**实际操作**:
- **保留**: D 列所有值（累计金额）
- **保留**: 第 16 行所有值（美金账户）
- **保留**: B 列中由规则 2 复制的数值
- **保留**: C 列中规则 3,6-10 设置的值和公式
- **删除**: C 列中原有的其他数值（但保留公式）

---

## 执行流程总结

```
1. 复制源文件格式和所有内容
2. 规则 2: D 列原始数值 → B 列
3. 规则 3: C5 = 0
4. 规则 4-5: 修改 D7, D8（银行余额）
5. 规则 6-10: 设置 C9, C11, C12, C14, C15
6. 规则 1&11: 恢复 C 列公式
7. 保存文件
```
