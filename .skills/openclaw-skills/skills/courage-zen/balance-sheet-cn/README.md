# 资产负债表生成技能

## 快速开始

### 方法 1: 命令行

```bash
cd ~/.openclaw/workspace/skills/资产负债表
python3 scripts/generate_balance_sheet.py <源文件路径>
```

### 方法 2: Python 调用

```python
from scripts.balance_sheet_generator import BalanceSheetGenerator

generator = BalanceSheetGenerator('财务报表 202511-t.xlsx')
generator.generate('资产负债表.xlsx')
```

### 方法 3: OpenClaw 对话

上传财务报表文件后，说："生成资产负债表"

## 11 条规则

1. **保留格式** - 格式、公式、文字保留
2. **累计金额→上月余额** - D 列数值复制到 B 列
3. **固定资产合计** - C5=0
4. **招商银行** - D7=明细表 M 列招商银行余额总计
5. **交通银行** - D8=明细表 M 列交通银行余额总计
6. **应收款合计** - C9=明细表 C 列应收 H 列发生金额总计
7. **预收款** - C11=明细表 C 列预收 H 列发生金额总计
8. **应付款** - C12=明细表 C 列应付 H 列发生金额总计
9. **当年利润** - C14=利润表经营利润 E 列数值
10. **未分配利润** - C15=0
11. **公式计算** - 保留所有公式

## 输出示例

| 科目 | 上月余额 | 本月发生额 | 累计金额 |
|------|---------|-----------|---------|
| 招商银行 | 258,498.64 | =D7-B7 | 228,843.03 |
| 交通银行（公） | 17,153.55 | =D8-B8 | 78,128.70 |
| 应收款合计 | 1,685,000 | 200,000 | =B9+C9 |
| 预收款 | 689,872.40 | 201,736 | =B11+C11 |
| 当年利润 | 557,069.52 | 29,583.54 | =B14+C14 |

## 注意事项

- 规则必须按顺序执行（1→2→3→...→11）
- 规则 2 在规则 4-5 之前执行
- M 列银行数据可能有多个记录，需要求和
- 金额保留 2 位小数
