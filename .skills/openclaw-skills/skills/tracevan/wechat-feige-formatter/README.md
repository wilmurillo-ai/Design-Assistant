## 🔍 优化规则详解

AI 会按照以下规则优化你的文档：

### 1. 识别流程 → 转换为 STEP 框
- 关键词：`第X步`、`步骤 X`、`首先`、`然后`、`接着`
- 转换为：`> STEP 01 标题` 格式

### 2. 为标题生成解释 → 添加 📖 说明
- 每个 `## ` 二级标题下添加一行解释
- 格式：`> 📖 这是对标题的简短说明`

### 3. 标记关键词 → `==关键词==` 高亮
- 识别专业术语、新概念、重要术语
- 用 `==词==` 包围

### 4. 识别总结 → 转换为 Insight 框
- 关键词：`最重要的`、`核心要点`、`记住`
- 转换为：`> ✨ 内容` 或 `> 💡 内容`

### 5. 识别指令 → 转换为 Prompt 框
- 包含：`应该`、`需要`、`必须`、代码示例
- 转换为：`> 🤖 PROMPT: 标题`

👉 详细规则见：`references/排版优化规则.md`

| **分隔线** | 桃色渐变 | 段落分割 |

## ⚠️ 复制粘贴到公众号

### 正确方式
1. 用浏览器打开生成的 HTML
2. **Ctrl+A** 全选
3. **Ctrl+C** 复制
4. 在微信公众号编辑器中 **普通粘贴**（Ctrl+V）

### 常见问题

**Q: 排版乱了？**
A: 用普通粘贴，不要用"粘贴并匹配格式"

**Q: 能改颜色吗？**
A: 可以。上传文件时跟 OpenClaw 说"用紫色排版"之类的

**Q: 本地图片怎么办？**
A: HTML 中用 `![](path)` 标记，输出后在公众号编辑器重新上传图片即可
