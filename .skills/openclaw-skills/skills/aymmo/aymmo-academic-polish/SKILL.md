# 学术写作润色体系

## 核心能力

本 Skill 提供完整的学术写作风格化工具集，基于学术文学性风格建立，融合戴锦华批判性与汪民安理论阐述两种方法论。

## 使用流程

### 1. 文本润色（去AIGC化）

**快速检查清单**：
- □ 打破对仗结构（减少"一方面...另一方面"）
- □ 变化连接词（避免"因此""从而""进而"过度重复）
- □ 长短句交替（制造呼吸感）
- □ 加入限定词（"某种程度上""一定程度上"）
- □ 使用隐喻点缀（"逐鹿""蛰伏""游离""缝合"）
- □ 分段处理（每段一个核心论点）

**高频替换词库**：
| AI痕迹词 | 替换选项 |
|---------|---------|
| 因此 | 由此观之、基于此、这意味着 |
| 从而 | 进而、继而、由此 |
| 但是 | 然而、反观、值得注意的是 |
| 通过...方式 | 借由...、以...为路径 |
| 已经呈现出 | 早已溢出、正在重塑 |

### 2. 追问库使用

**核心追问框架**：
1. **主体追问**：谁是"X"？谁被排除在外？
2. **概念谱系**：理论从哪里来？经历了怎样的改写？
3. **话语建构**：谁在定义？遮蔽了什么？
4. **形式分析**：表面与实质的错位是什么？
5. **他者追问**：生产"X"的人在哪里？

### 3. 反方观点处理

**五步回应法**：
1. **吸收**：承认对方立场的合理性
2. **转化**：指出其内在悖论或盲点
3. **超越**：揭示双方共享的未经审视的前提
4. **历史化**：将争论置入具体历史语境
5. **重构**：提出新的问题框架

### 4. 概念对子处理

**拒绝虚假二分，展示概念流动**：
- 生产/消费 → 消费是否已成为生产的一种形式？
- 精英/大众 → "大众"是被建构的客体还是真正的主体？
- 抵抗/收编 → 抵抗的形式是否已被资本预设？
- 真实/想象 → 想象是否构成另一种真实？

### 5. 抵抗的考古学

**五维度分析框架**：
1. **时间分层**：从越轨到正统的历史地层
2. **话语转型**：关键概念的语义漂移
3. **空间迁移**：从地下到中心的地理轨迹
4. **主体形成**：身份认同的建构与消解
5. **收编机制**：抵抗如何被转化为商品

## 句式模板速查

### 引入悖论
"如果说...那么...却又..."
> 如果说批判者看到了A，那么他们却又忽略了B

### 谱系展开
"从...到...这一...脉络"
> 从30年代左翼文化到社会主义文化这一主流脉络

### 揭示建构性
"与其说是在...不如说是在..."
> 与其说满足需求，不如说规范群体

### 追问逻辑
"如果...那么..."
> 如果中产阶级占80%，那么剩下20%如何生活？

### 历史积淀
"这无疑出自...事实上，它也是..."
> 这无疑出自记忆，事实上，它也是积淀

### 表面/实质
"尽管从表面上看来...但事实上..."
> 尽管表面政治掌控，但事实上资本注入

### 错位揭示
"...被成功地组织在...之中"
> 下岗被组织在经典社会主义叙事之中

### 数据震撼
"...的一个爆炸性新闻是..."
> 1996年的一个爆炸性新闻是...

```bash
node scripts/polish-checklist.js "待检查文本"
```

或手动检查：
- [ ] 无第一人称

- [ ] 长短句交错
- [ ] 有思维痕迹（"值得追问的是"）

## 核心写作原则（关键洞见）

> **从"反驳对手的封闭姿态"转向"揭示机制的开放分析"**
> 
> 关键学术洞见：能看出"微妙却致命的逻辑缺陷"的人不多，而这些缺陷正是人类学术的深层结构性通病。

### 三大逻辑缺陷审查

#### 1. 逻辑踏空（过于想当然）
**本质**：认知吝啬的学术体现，用"因此"掩盖论证断层

**错误示范**：
```
"这种'中心位置'因此强化了对女性情感劳动的规范性要求。
因此，中国的发展路径可以被理解为一种'伦理化的后女性主义怪物女性气质'。"
```

**问题**：从分析到结论之间缺乏充分的论证链条，"因此"一词掩盖了逻辑断层

**修正**：
```
"这种'中心位置'生产的是一种高度规范性的主体位置——狐妖的能动性
只能通过'关系性劳动'来呈现...这种'中心位置'因此强化而非松动了
对女性情感劳动的规范性要求。" [补充中间论证]
"基于上述分析，中国的发展路径可以被理解为..." [结论建立在完整论证之上]
```

#### 2. 结论过重（内省低能的体现）
**本质**：元认知失败，用"显然""毫无疑问"关闭自我质疑

**错误示范**：
```
"将中国狐妖形象的'柔化'理解为对性别规训的摆脱显然是一种理论上的误导。"
```

**问题**：结论过于绝对，表现出过度自信的封闭姿态，关闭了进一步讨论的空间

**修正**：
```
"基于上述分析，中国狐妖形象的'柔化'所呈现的并非一种简单的线性进步叙事...
而是揭示了一种更为复杂的权力运作机制。"
```

#### 3. 对手预设（投射机制）
**本质**：将自身狭隘投射到不存在的客体上，树立稻草人

**错误示范**：
```
"将亚文化的'出圈'理解为抵抗的胜利显然是一种误导。恰恰相反，
这实际上是一种更精致的收编。"
```

**问题**：预设了一个"将出圈理解为抵抗胜利"的对手立场，但这个对手可能并不存在

**修正**：
```
"亚文化的'出圈'所呈现的并非一种简单的线性进步叙事。当边缘性被编码为
可流通的符号，这种'出圈'可能同时包含着扩展与收编的复杂张力。"
```

### 开放分析的写作原则

**封闭姿态**（差）：
- 寻找对手 → 树立靶子 → 猛烈批判 → 证明自我正确

**开放分析**（好）：
- 观察现象 → 识别复杂性 → 揭示内在机制 → 提出解释框架

**核心口诀**：
```
不树靶子，直接分析；
不跳逻辑，步步为营；
不轻下结论，保持开放；
揭示机制，而非反驳对手。
```

**详细指南**：参考 `references/logic-review-guide.md`

## 英文写作润色（English Academic Writing）

对于英文论文润色，参考 `references/english-academic-guide.md`：

### 核心原则
- **Hedging（模糊限定）**：避免绝对化表述，使用"may suggest", "appears to", "tends to"
- **Nominalization（名词化）**：动词→名词，提升学术正式度
- **Passive Voice（被动语态）**：强调研究对象而非研究者
- **Impersonal Style（去人称化）**：避免"I think", "we believe"

### 常见AI痕迹（英文）
- "very", "really", "basically", "actually" → 删除或替换
- "there is/are" 开头 → 改为具体主语
- "thing", "stuff", "something" → 具体化
- "and so on", "etc." → 删除或完整列举

### 学科适配
- **Cultural Studies**: 批判性词汇、理论术语密度
- **Media Studies**: 技术词汇、平台特定术语
- **Sociology**: 方法论精确性、变量描述

**详细指南**：见 `references/english-academic-guide.md`
