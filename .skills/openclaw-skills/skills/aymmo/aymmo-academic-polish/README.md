# 学术写作润色 (Academic Polish)

一个专业的学术写作辅助技能，融合戴锦华批判写作与汪民安理论阐述方法论。

## 功能特性

### 中文润色
- 去除AI痕迹：打破对称结构、替换高频连接词
- 逻辑缺陷审查：识别逻辑踏空、结论过重、对手预设
- 戴锦华式批判：追问框架、概念谱系、话语建构
- 汪民安式理论：悖论引入、空间分析、感官政治

### 英文润色
- 地道学术英语表达
- Hedging策略（模糊限定）
- IMRAD结构优化
- 期刊风格适配

### 核心能力
1. **五级写作体系**：去AI味 → 学术规范 → 去酸腐 → 智识诚实 → 学术文学性
2. **三大逻辑缺陷审查**：逻辑踏空、结论过重、对手预设
3. **追问库**：主体追问、概念谱系、话语建构、形式分析、他者追问
4. **反方观点处理**：五步回应法（吸收→转化→超越→历史化→重构）

## 核心写作原则

> **从"反驳对手的封闭姿态"转向"揭示机制的开放分析"**

**口诀**:
```
不树靶子，直接分析；
不跳逻辑，步步为营；
不轻下结论，保持开放；
揭示机制，而非反驳对手。
```
