# 追问库模板

> 基于戴锦华三重追问法（谁是主体/谁在命名/遮蔽了什么）
> 针对"想象力消费"研究的追问设计框架

---

## 第一组：主体的追问（对应"何谓大众"）

### 核心问题：谁是"想象力消费者"？谁被排除？

| 追问 | 分析目标 | 戴锦华式表述 |
|------|---------|-------------|
| 谁是"想象力消费者"？ | 揭露主体的建构性 | 那些被纳入"想象"版图的人 |
| 谁被排除在这"想象"之外？ | 发掘被压抑的他者 | 那些无法进入影院、无力支付会员的人 |
| "沉默的大多数"是谁？ | 重新定义概念 | 真正的"大众"是热衷谈论"大众"的人不关注的对象 |

### 具体追问模板

**追问1：准入门槛**
```
如果说，"想象力消费"标示着一种新型的文化参与方式，那么，在今日中国
方兴未艾的"想象力消费"，显然并未给无力支付票价的人群留有空间。或许
没有人会怀疑他们正是当代中国社会语境中的"大众"，而且毫无疑问，是
"想象力"之话题的人们毫不关注的对象。
```

**追问2：空间排斥**
```
电影票价格、平台会员费、周边消费，构成了怎样的准入门槛？
当"想象力"成为可消费的对象时，消费行为本身是否已经成为一种生产？
```

**追问3：生产者的隐身**
```
那些生产"想象力"的人（网文写手、游戏美工、剧组场工）如何从话语中消失？
如果说，他们才是"想象力"的真正生产者，那么，"想象力消费"话语
显然并未给此类"生产者"留有空间。
```

## 第二组：概念谱系的追问（对应"文化研究及其陷阱"）

### 核心问题：这个理论从哪里来？经历了怎样的改写？

| 追问 | 分析目标 | 戴锦华式表述 |
|------|---------|-------------|
| 理论的旅行路线是什么？ | 谱系学分析 | 从西方理论到中文语境的改写 |
| 是否创造了新的全能命名？ | 警惕万能框架 | "想象力消费"是否正在成为可以套用一切的框架？ |
| 对事实差异性的关注是否被抹除？ | 防止过度简化 | 描述电影、游戏、网文时是否忽视了本质差异？ |

### 具体追问模板

**追问1：理论旅行**
```
从某种意义上说，"想象力消费"这一概念在中文学界的浮现，并非一次
纯粹的理论命名行为，而是90年代以来中国消费社会成型过程中，知识界
试图以理论话语捕捉文化现实的一次重要尝试。然而，这一尝试本身却
蕴含着某种深刻的悖论：它既要借助西方理论资源来诊断本土文化症候，
又必须在这借助过程中避免沦为西方理论的注脚。
```

**追问2：概念边界**
```
当我们用"想象力消费"描述电影、游戏、网文时，是否忽视了它们之间的
本质差异？这一概念是否正在成为可以套用一切的万能框架？
```

**追问3：命名政治**
```
谁有权定义什么是"好的"想象力？
知识分子的批判是否本身就是一种阶级区隔的表演？
```

## 第三组：话语建构的追问（对应"格调还是阶级"）

### 核心问题：谁在定义"想象力"？这个话语遮蔽了什么？

| 追问 | 分析目标 | 戴锦华式表述 |
|------|---------|-------------|
| 谁在生产"大众文化"？ | 政治经济学分析 | 谁在定义"想象力"的边界？ |
| "中产阶级"是如何被建构的？ | 揭露阶级话语 | "想象力消费者"是如何被建构为理想主体的？ |
| 概念的政治经济学背景是什么？ | 历史化分析 | 与电影产业化、平台资本运作有何关联？ |

### 具体追问模板

**追问1：主体建构**
```
定位于中产阶级的"想象力消费"话语，与其说是在描述一种既存的文化
现实，不如说是在规范着一个理想的消费者群体。说得直白些，"想象力消费"
的自我定位，其功能之一，便是为"下一个世纪的中国"，呼唤和构造着
具有特定消费能力的中产阶级社群。
```

**追问2：政治经济学**
```
"想象力消费"的兴起与电影产业化、平台资本运作有何关联？
当平台声称"为用户推荐感兴趣的内容"时，这种"个性化"是否恰恰是对
用户可能性的限制？
```

**追问3：全球化/本土**
```
当中国奇幻电影使用好莱坞的视觉语法讲述"东方"故事时，这是"全球化"
还是"本土化"？或者，这一二分本身是否需要被质疑？
```

## 第四组：遮蔽与隐显的追问（对应"现实与话语逻辑"）

### 核心问题：这个话语让什么不可见？

| 追问 | 分析目标 | 戴锦华式表述 |
|------|---------|-------------|
| 下岗工人如何从传媒中消失？ | 追踪隐显机制 | 体力劳动、底层经验如何从视野中消失？ |
| "常识"系统如何被复活？ | 意识形态分析 | "想象力"如何取代"现实"成为新的合法性来源？ |
| 阶级话语如何成为禁忌？ | 揭示压抑机制 | 当"想象力"被神化时，什么被压抑了？ |

### 具体追问模板

**追问1：现实的改写**
```
如果说，对"想象力"的推崇间或包含着对80年代"枯燥现实主义"的
反拨，那么，对"想象力消费"的轻率拥抱，却又无疑与中国知识界对
社会民主理想的追求，发生了深刻而内在的结构性冲突。这间或是造成
当代知识群体面对"想象力消费"的现实，多少陷于自相矛盾的原因之一。
```

**追问2：阶级隐形**
```
当"想象力"成为可消费的商品时，"现实"是否也被重构为一种特定的
"想象"？（如对"中产生活"的想象性描绘）
创作劳动的剥削如何被转译为"梦想""热爱""IP孵化"？
```

**追问3：形式的追问**
```
当平台声称"为用户推荐感兴趣的内容"时，这种"个性化"是否恰恰是对
用户可能性的限制？
"完读率""留存率"如何规范创作者的生产？
```

## 第五组：形式的追问（对应"权力·媒介"）

### 核心问题：媒介技术如何重构"想象力"的生产与流通？

| 追问 | 分析目标 | 戴锦华式表述 |
|------|---------|-------------|
| 传媒系统的资本化如何发生？ | 政治经济学 | 平台经济如何重构生产与流通？ |
| 表面与实质的错位是什么？ | 意识形态分析 | 表面上的解放，实质上是什么的强化？ |
| 数据的意识形态意义是什么？ | 批判数据主义 | 票房、播放量如何成为新的"真理"标准？ |

### 具体追问模板

**追问1：资本化机制**
```
尽管从表面上看来，"想象力消费"标示着一种文化民主化的趋势，但
事实上，伴随着流媒体平台的不断扩张、算法的不断精密，是资本的
大量注入与内容生产机制的日渐商业化。如果说，90年代中国最重要的
社会变革，便是昔日的经典权力开始转化为企业资本，那么，类似情形
在"想象力"的生产领域之中尤为剧烈而清晰。
```

**追问2：数据意识形态**
```
票房数据、播放量、热度指数，如何成为新的"真理"标准？
当"想象力"的生产被"完读率""留存率"等KPI规范时，"原创"是否
已经成为对成功套路的精明复制？
```

**追问3：算法权力**
```
算法推荐机制不仅回应用户的"兴趣"，更在塑造用户的"兴趣"。
因此，"自由"与"必然"的边界，在平台经济的逻辑中变得日渐模糊。
```

## 第六组：被遮蔽的他者（对应"'大众'主体"）

### 核心问题：真正的"大众"在哪里？

| 追问 | 分析目标 | 戴锦华式表述 |
|------|---------|-------------|
| 农民工的"无名无语"意味着什么？ | 阶级分析 | 生产"想象力"的人如何从话语中消失？ |
| "沉默的大多数"如何被重新定义？ | 重新定义 | "大众"是消费者还是生产者？ |
| 阶级压迫如何被转译？ | 意识形态分析 | 剥削如何被转译为"梦想""热爱"？ |

### 具体追问模板

**追问1：生产者的隐身**
```
但较之于热衷于讨论"想象力消费"的知识分子，在当代中国"创意产业"
的勃兴面前，同样遭到重压，却完全处于无名无语状态中的，是充满了
各大城市的网文写手、游戏美工、影视剧组临时工；而正是他们明确地
成为"想象力经济"得以"软着陆"的对象。如果说，他们才是"想象力"
的真正生产者，那么，在今日中国方兴未艾的"想象力消费"话语，显然
并未给此类"生产者"留有空间。
```

**追问2：劳动剥削**
```
"创意劳动"的新型剥削机制是什么？
当"想象力"在私密场景（手机、耳机）中被消费时，这种"个体性"是否
已经被平台的"集体数据"所捕获和重构？
```

**追问3：身份转换**
```
从"粉丝"到"作者"到"打工人"的身份轨迹，标示着什么样的主体性消
解过程？
```

## 戴锦华式综合表述

```
有趣的是，无论是"想象力消费"的捍卫者还是批判者，似乎都共享
着一系列未经审视的前提："想象力"与"消费"之间存在某种本质的、
不可调和的张力。前者试图证明这种张力可以被克服（通过"好的"
消费），后者则坚持这种张力的不可克服性（因而消费即堕落）。
然而，这一共享前提本身，或许正是需要被追问的："想象力"与
"消费"的二分，是否本身就是一种特定的历史建构？
```
