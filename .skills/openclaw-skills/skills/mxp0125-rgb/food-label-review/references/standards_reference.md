## 一、GB 7718-2011 核心条款（2011版，仍广泛使用）

### 4.1.2.2.1 & 4.1.2.2.2 — 产品名称字号/颜色规范（★★★★★案例5新增）

**4.1.2.2.1（易误解名称的强制处理）**
> 当"新创名称""奇特名称""音译名称""牌号名称""地区俚语名称"或"商标名称"含有易使人误解食品属性的文字或术语（词语）时，应在所示名称的同一展示版面的临近位置使用**不大于属性名称的字高**且与属性名称**相同的字体、颜色**标示食品真实属性的专用名称。

**4.1.2.2.2（属性名本身字号误导的处理）**
> 当食品真实属性的专用名称因字号或字体颜色不同易使人误解食品属性时，也应使用**同一字号**及**同一字体颜色**标示食品真实属性的专用名称。

**实操判断：两条款的区别**
- 4.1.2.2.1：产品名称本身含易误解文字（如"水果棒"≠"水果汁"）
- 4.1.2.2.2：真实属性名称字号太小/颜色太淡，导致被误认

---

### 4.1.3.1.3 — 复合配料展开规则（★★★★案例4+10新增）

> 如果某种配料是由两种或两种以上的其他配料构成的复合配料（不包括复合食品添加剂），应在配料表中标示复合配料的名称，随后将复合配料的原始配料按加入量的递减顺序一一排列。

**展开豁免条件（须同时满足）：**
1. 该复合配料已有**国家标准、行业标准或地方标准**
2. 该复合配料的**加入量小于食品总量的25%**

**不满足任一条件 → 须展开全部原始配料**

**案例记忆：**
- "盐渍话梅汁"（案例4/8）：若为复合配料且添加量>25%，须展开原始配料或标属性名称
- "食用香精"（案例10）：不建议展开原始配料，避免误导消费者认为香精起其他作用

---

### 4.1.6.1.2 — 产地标注地市级（★★★★案例7新增）

> 不能依法独立承担法律责任的集团公司的分公司或集团公司的生产基地，应标示集团公司和分公司的名称、地址；**或仅标示集团公司的名称、地址及产地**，产地应当按照行政区划标注到**地市级地域**。

**正确示例：** "四川省成都市" / "广东省深圳市" / "浙江省杭州市"

---

### 3.8.2 — 外文与中文对应（★★★★案例6新增）

> 进口食品须同时使用中文，外文应与中文一一对应。

**实操重点：**
- 所有外文标示内容均须有对应中文
- 中文字号不得小于外文字号（案例8/9累计确认）
- 不得仅标注外文而无中文

---

### 3.6 — 品种/地区俚语名称规范（★★★★案例4新增）

> 食品名称若使用地区俚语名称或音译名称，须同时标示真实属性名称。

**案例：** "青芒"须标示为"青芒（芒果）原汁"（案例4/8）

---

### 4.1.10 — 执行标准代号（★★★★案例7新增）

> 在国内生产并在国内销售的预包装食品（不包括进口预包装食品）应标示产品所执行的**标准代号和顺序号**。

**建议删除年代号的原因：** 若标注"GB 7718-2011"，标准修订后包装须重新印刷；标注"GB 7718"则自动对接最新版本。

## 二、GB 7718-2025 新版标准核心变化（★★★★★案例9首次出现）

> 实施日期：2026年3月16日（保质期6个月以上的食品可延至2027年3月16日）

### 主要变化对比

| 项目 | GB 7718-2011 | GB 7718-2025 |
|------|-------------|--------------|
| 保质期标示 | "保质期"（自行计算） | "**保质期到期日**"（直接标注到期日） |
| 日期顺序 | 未强制统一 | 统一按**年-月-日**顺序 |
| 日期分隔符 | 未规定 | 可用空格/斜线/连字符 |
| 预包装定义 | 较大 | 扩大：增加计量销售食品+长度标识食品 |
| 主要展示面 | 有定义 | **删除**主要展示面定义（由《食品标识监督管理办法》规定） |
| 字体字号 | 正文有具体规定 | **删除**具体规定，由《食品标识监督管理办法》规定 |
| 净含量 | 正文有具体规定 | **删除**具体规定，由《食品标识监督管理办法》规定 |
| 配料表 | — | 复合配料须强制标示名称；复配添加剂须展开 |
| 数字标签 | 无 | 新增数字标签定义（GB/T 37101） |

### 4.2 — 真实属性名称（新版，案例9新增）

> 预包装食品应在食品名称的临近位置标示反映食品真实属性的专用名称。

**关键：** 字号、字体、颜色须与食品名称**一致**。

**案例：** "云雾皇帝柑气泡果汁饮料"（案例9）— 须将真实属性名称"复合果汁饮料"以同一字号标示在食品名称临近位置。

### 4.4 — 配料强调与定量标示（新版，案例9新增）

> 如果在食品标签中对某种配料进行了强调，应标明该配料的含量或其在终产品中的含量。

**案例：** "岭南风物"（案例9）— 须明确哪个配料是"岭南风物"，不得模糊强调。

### 4.7 — 日期标示（新版）

> 预包装食品应统一按照**年、月、日**的顺序标示生产日期和**保质期到期日**。

**可仅标"保质期到期日"的情况：**
- 保质期6个月及以上
- 保质期到期日可从生产日期推算

**"保质期"改为"保质期至"：** 须按新标表述（案例9不符合项）

## 三、GB 28050-2011 vs 2025 核心变化

### 2011版强制标示："1+4"
> 能量 + 蛋白质 + 脂肪 + 碳水化合物 + 钠

### 2025版强制标示："1+6"（★★★★★案例10新增）
> 能量 + 蛋白质 + 脂肪 + **饱和脂肪（酸）** + 碳水化合物 + **糖** + 钠

**新增强制提示语：**
> 营养成分表下方须标注：**"儿童青少年应避免过量摄入盐油糖"**

**豁免：** 包装饮用水等豁免营养标签的食品，同步豁免此提示语。

---

### 附录B.2.6 — 营养声称字号规范（★★★★案例4新增）

> 营养声称的字号不得大于食品名称及商标的字号。

**案例：** "高膳食纤维"字号>产品名称字号 → 不符合项（案例4/8）

**格式要求：** 含量声称须同时标示含量及NRV%，字号不得大于食品名称。

---

### 能量计算公式

```
能量（kJ/100g）= 蛋白质×17 + 脂肪×37 + 碳水化合物×17 + 膳食纤维×8
```

**华测实际操作：** 直接用公式计算，与标签标示值比对；允许误差须客户提供书面说明。

## 四、GB 13432-2013 特殊膳食用食品标签（★★★★案例10新增）

**适用范围：** 营养强化食品、特殊医学用途配方食品、婴幼儿配方食品等

**与GB 7718-2011的关系：** GB 7718-2011的相应条款均须同时满足

### 4.3 能量和营养成分

> 应以"方框表"形式标示能量、蛋白质、脂肪、碳水化合物、钠，以及相应产品标准中要求的其他营养成分及其含量。

**案例：** 钙营养饮须在营养成分表中增加标示"肽类"含量（案例10建议项）

### 4.4 食用方法和适宜人群

> 应标示食用方法、每日或每餐食用量，以指导消费者合理使用。

**特殊医学用途食品：** 可标示"每日或每餐食用量参照医生或者临床营养师的指导"。

### 儿童/婴幼儿图案误导（★★★★案例10不符合项）

> 特殊膳食用食品标签不得有误导消费者认为产品为儿童/婴幼儿食用的图案。

**案例：** 钙营养饮因含儿童图案被发不符合项（案例10）

## 五、GB/T 31121-2014 果蔬汁类及其饮料（★★★★案例3新增）

> 适用于果蔬汁（浆）、浓缩果蔬汁（浆）、果蔬汁（浆）类饮料

### 关键条款：b）果蔬汁（浆）类饮料

> 应**显著标明(原)果汁(浆)总含量**或(原)蔬菜汁(浆)总含量

**标示位置（二选一）：**
1. "营养成分表"**附近位置**
2. **与产品名称在包装物或容器的同一展示版面**

**实操要求：** 须确认实际包装时两者能在同一版面呈现（案例3建议确认项）

### "100%果汁"标示规则

> 只有符合"声称100%"要求的产品，才可以在标签的任意部位标示"100%"；否则只能在营养成分表附近位置标示"果蔬汁含量：100%"。

## 六、JJF 1070-2023 定量包装商品净含量允差（★★★★案例8+9新增）

### 净含量允差表

| 净含量Q（g或mL） | 允差 |
|----------------|------|
| Q ≤ 50 | 9% |
| 50 ＜ Q ≤ 100 | 4.5% |
| 100 ＜ Q ≤ 200 | 4.5% |
| 200 ＜ Q ≤ 300 | 3% |
| 300 ＜ Q ≤ 500 | 3% |
| 500 ＜ Q ≤ 1000 | 1.5% |
| Q ＞ 1000 | 1% |

### 字符高度要求（GB 7718-2011 表3）

| 净含量 | 最小字符高度 |
|--------|------------|
| ≤ 50g / ≤ 50mL | **2.0mm** |
| 50g ~ 200g / 50mL ~ 200mL | **3.0mm** |
| 200g ~ 1000g / 200mL ~ 1000mL | **4.0mm** |
| ＞ 1000g / ＞ 1000mL | **5.0mm** |

**版面其他文字字符高度：** 须不小于**1.8mm**（案例7+8+9+10累计确认项）

## 七、GB/T 191-2008 包装储运图示标志（★★★★案例8新增）

| 标志名称 | 含义 | 适用场景 |
|---------|------|---------|
| 🌧 防雨 | 怕雨淋 | 须防雨的食品 |
| ☀ 防日光 | 怕日晒 | 须避光保存的食品 |
| ⬆ 易碎品 | 小心轻放 | 玻璃包装食品 |
| ↑ 向上 | 须垂直放置 | 须保持向上的食品 |
| 📦 堆码上限 | 最大堆码层数 | 仓储运输标识 |

**实操：** 须确认标签非储运包装；若为储运包装，须按GB/T 191标示（案例8确认项）

> 产品或者其包装上的标识必须真实，并符合下列要求：
> （一）有产品质量检验**合格证明**

**合格证明样式：** "检验合格"印章 / 合格证标签 / 官方认证标志

## 九、核心审核逻辑图

```
收到标签
  ↓
判断产品类型
  ├── 普通预包装食品 → GB 7718-2011 + GB 28050-2011/2025
  ├── 特殊膳食用食品 → + GB 13432-2013
  ├── 果蔬汁类饮料   → + GB/T 31121-2014
  └── 定量包装商品   → + JJF 1070-2023
  ↓
第一步：核查产品名称（3.3/3.4/3.6/4.1.2.2.1/4.1.2.2.2）
  ↓
第二步：核查配料表（4.1.3 → 复合配料展开4.1.3.1.3）
  ↓
第三步：核查营养标签（GB 28050强制1+6 + 附录B.2.6）
  ↓
第四步：核查日期/规格/生产者信息
  ↓
第五步：核查共性确认项（合格证/字符高度/商标/条码）
```
