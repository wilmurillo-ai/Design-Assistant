## ⚠️ 审核边界说明（华测检测客户须知 — 重要！）

> 正式审核前须向客户/报告使用方说明以下审核范围边界：

| 序号 | 边界说明 | 处理方式 |
|---|---|---|
| B01 | 审核基于送审标签及客户提供信息，**不负责核实其真实性** | 要求客户提供书面承诺或证明材料 |
| B02 | " **"表示不适用于该产品标签，或该产品标签无此条款要求** | — |
| B03 | GB 28050-2011 4.1：仅审核**营养标签的格式及NRV%值**，不审核能量及营养素含量真实性 | 以检测报告为准 |
| B04 | **不审核**配料排列顺序及食品添加剂是否超范围添加；只审核引导词是否正确及食品添加剂名称是否规范 | — |
| B05 | 若样品为**电子版标签**：不审核强制标示内容的**字符高度**及**生产日期格式** | 以实物印刷为准 |
| B06 | 若样品为**样版标签**：不审核**生产日期格式** | 以实物印刷为准 |
| B07 | **不审核**：非食用物质标识内容 | 须另行专项审核 |
| B08 | **进口预包装食品**：仅审核中文标示内容，**不审核**外文、中外文对应情况 | — |
| B09 | 营养成分表**默认**以100g为单位；企业可更改为每100mL或每份 | 须标注清楚 |
| B10 | **不审核**：二维码、商品条码、生产企业生产资质 | 须在报告中注明 |

## 审核结论类型定义

| 类型 | 含义 | 示例 |
|------|------|------|
| **确认项** | 须客户提供真实性声明或证明材料的合规项 | "请确认'山野杨梅'真实有依据，无虚假误导" |
| **建议项** | 非强制，但推荐改进的合规项 | 建议添加致敏物质提示 |
| **声明项** | 客户须书面声明承担责任的项 | 商标授权使用声明 |
| **不符合项** | 明确违反强制标准的项 | SC编号缺失、营养标签格式错误 |
| **华测复审意见** | 初审后客户修改，华测二次确认的意见 | — |
