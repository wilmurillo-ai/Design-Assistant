# 全球RoHS/WEEE法规详细对比（2024-2026最新）

---

## 一、中国 China

| 项目 | 法规 | 状态 |
|------|------|------|
| **有害物质** | GB 26572-2025 **强制性** | ⚠️ 2027-08-01实施 |
| **旧RoHS** | GB/T 26572-2011（推荐性） | → 2027年被取代 |
| **检测方法** | GB/T 39560系列（2024-03-01起强制） | ✅ 现行 |
| **6项物质** | 铅/汞/镉/六价铬/PBB/PBDE | 统一限量：0.1%/0.01% |
| **产品分类** | Ⅰ类（纳入达标目录）/ Ⅱ类（其他产品） | 新增 |
| **E-waste** | GB/T 42581-2023（WEEE标准） | 发展中 |
| **合格评定** | 企业自我声明 + 平台备案 | ✅ 现行 |

## 二、欧盟 EU

| 项目 | 法规 | 状态 |
|------|------|------|
| **有害物质** | RoHS 2011/65/EU（Directive）→ 正在修订RoHS 3 | ⚠️ 修订中 |
| **现有限制** | 10项物质（新增DEHP/BBP/DBP/DIBP） | ✅ 现行 |
| **豁免** | 附件III/IV豁免清单（定期评审） | 持续更新 |
| **WEEE** | WEEE 2012/19/EU（Directive） | ✅ 现行 |
| **回收率** | 收集率45%/65%/85%（分阶段目标） | ✅ 强制 |
| **注册** | 生产商须在各国主管机构注册 | ✅ 强制 |
| **REACH** | EC 1907/2006（化学品注册） | ✅ 现行 |
| **电池指令** | Battery Directive 2006/66/EC → 新电池法规（2023） | ⚠️ 过渡期 |

## 三、英国 UK

| 项目 | 法规 | 状态 |
