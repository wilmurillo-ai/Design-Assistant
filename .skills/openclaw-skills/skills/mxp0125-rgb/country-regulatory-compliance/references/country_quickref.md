# 主要市场合规要求速查表（2024-2026最新）

---

## 一、中国（China · CCC）

| 项目 | 要求 |
|------|------|
| **主管机构** | 国家认监委（CNCA）/ 市场监管总局 |
| **认证** | CCC强制性产品认证（19种家用设备） |
| **安全标准** | GB/T 4706.1-2024（2026-08-01，2027-07-31过渡截止） |
| **EMC标准** | GB 4343.1-2024（2026-06-01，2027-07-31过渡截止） |
| **RoHS** | GB 26572-2025（2027-08-01，首个强制性RoHS国标） |
| **插头** | GB 1002/2099（三眼扁插/两眼扁插） |
| **电网** | AC 220V / 50Hz |
| **CB接受** | ✅ 接受（CNCA认可CB机构） |
| **能效** | GB 12021系列强制 + 2025厨房电器规则 |
| **食品接触** | GB 4806.1-2024（电热水壶/豆浆机等须认证） |

## 二、欧盟（EU · CE）

| 项目 | 要求 |
|------|------|
| **主管机构** | 欧盟委员会 + 成员国市场监督局 |
| **安全** | 低压指令 LVD 2014/35/EU（50-1000V AC） |
| **EMC** | EMC指令 2014/30/EU |
| **RED** | 无线电设备指令 2014/53/EU（2025-08-01起网络安全强制） |
| **生态设计** | ErP指令 2009/125/EC（能效要求） |
| **RoHS** | RoHS指令 2011/65/EU（EU RoHS） |
| **WEEE** | WEEE指令 2012/19/EU |
| **GPSR** | 一般产品安全法规 EU 2023/988（2024-12-13起） |
| **插头** | 各国不同：Type F（德）/ Type L（意）/ Type G（英）等 |
| **电网** | AC 230V / 50Hz（统一） |
| **CB接受** | ✅ 高度接受 |

## 三、英国（UK · UKCA/CE）

| 项目 | 要求 |
|------|------|
| **主管机构** | 商业贸易部（DBT） |
| **UKCA标志** | 可选（目前非强制，大多数产品仍可使用CE） |
| **CE认可** | ✅ 延长接受（至进一步通知） |
| **产品安全** | Product Safety and Metrology Regs 2019 |
| **EMC** | EMC Regulations 2016 |
| **RoHS** | RoHS Regulations 2012（UK版） |
| **插头** | BS 1363（Type G，英式三脚方插） |
| **电网** | AC 230V / 50Hz |
| **北爱尔兰** | 仍执行CE标记要求 |

## 四、印度（India · BIS CRS）

| 项目 | 要求 |
|------|------|
| **主管机构** | 印度标准局（BIS） |
| **认证** | CRS强制注册（Scheme II） |
| **安全标准** | IS 302(Part 1):2024（2025-08-28强制） |
| **新法规** | Machinery and Electrical Equipment Safety (Omnibus Technical Regulation) Order 2024（2025-08-28） |
| **E-waste** | E-waste (Management) Rules 2022 |
| **插头** | IS 1293（印度三眼圆脚插） |
| **电网** | AC 230V / 50Hz |
| **CB接受** | ⚠️ 接受CB报告，但须BIS认可实验室验证 |

## 五、日本（Japan · PSE）

| 项目 | 要求 |
|------|------|

## 六、韩国（Korea · KC）

| 项目 | 要求 |
|------|------|
| **主管机构** | 韩国技术标准院（KATS） |
| **KC安全认证** | 强制性（型式试验+工厂检查） |
| **KC SDoC** | 低风险产品自我声明 |

| **技术标准** | K标准（基于IEC） |
| **插头** | KS C 8301（韩国三眼圆脚） |
| **电网** | AC 220V / 60Hz |
| **CB接受** | ✅ 接受CB报告作为技术依据 |

## 九、海湾（SASO · Saudi Arabia）

| 项目 | 要求 |
|------|------|
| **主管机构** | SASO（沙特标准、计量与质量组织） |
| **SASO IECEE** | 电子电气产品（管制清单内须） |
| **SASO LVE** | 低压电气设备安全 |
| **能效（SEEP）** | 空调/冰箱/洗衣机须加贴能效标签 |
| **技术标准** | SASO标准（等同IEC） |
| **插头** | SASO批准标准插头（类似BS 1363） |
| **电网** | AC 220V / 60Hz |
| **CB接受** | ✅ 基于IECEE CB Scheme，高度接受 |
