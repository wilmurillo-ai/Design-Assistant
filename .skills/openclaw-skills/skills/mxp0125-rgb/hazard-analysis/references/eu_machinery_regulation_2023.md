# EU Machinery Regulation 2023/1230 详解（★★★★重点更新）

> 正式生效：2023年7月13日
> 强制执行：2027年1月20日
> 取代：Directive 2006/42/EC（机械指令）

---

## 核心变化（与2006/42/EC对比）

### 1. AI安全功能 → 强制第三方认证

| 情况 | 要求 |
|------|------|
| 无AI安全功能的普通机械 | 仍可自我声明（Module A） |
| AI用于安全相关功能 | **须Notified Body（公告机构）认证** |

**高风险AI场景（举例）：**
- 基于视觉识别的协作机器人安全监控系统
- 自动避障系统（AMR/自动导引车）
- 预测性维护系统（涉及安全功能）
- 自适应压力控制系统

### 2. 网络安全（Cybersecurity）成为强制性安全要求

> "The machine must also be protected against cybersecurity attacks" — CRA & Machinery Regulation

**具体要求：**
- 防止未经授权的远程访问/控制
- 防止恶意软件篡改安全功能
- 防止黑客利用数字接口入侵
- 须在技术文档中记录网络安全措施

### 3. 数字文档（Digital Documentation）正式合法化

| 文档类型 | 2006/42/EC | 2023/1230 |
|---------|-----------|----------|
| 使用说明书 | 须纸质（或电子版须应消费者要求提供） | **电子版正式合法**；除非消费者明确要求纸质 |
| 技术文档 | 纸质/电子 | 电子版（e-TF）正式认可 |
| 符合性声明（DoC） | 纸质 | **数字DoC（e-DoC）正式认可** |

### 4. 实质性改装 → 改装者成为新制造商

> "Substantial modifications to a machine make you the manufacturer"

**什么算"实质性改装"？**
- 改变机器的安全特性或功能
- 影响安全功能的软件修改
- 结构改变导致新的危险
- 增加新的操作模式

**改装者须承担全部责任：**
- 重新进行风险评估（覆盖全生命周期）
- 编制新技术文档
- 签署新的DoC
- 重新加贴CE标志

### 5. 全生命周期风险评估

> "The entire lifecycle of the machine (installation, maintenance, decommissioning, foreseeable misuse) must be considered"

| 阶段 | 风险评估覆盖要求 |
|------|--------------|
| 设计（A） | 从设计源头消除危险（本质安全设计） |
| 安装（D） | 安装过程危险、安装人员培训 |
| 运行（E） | 操作规程、误用场景、维护期间危险 |
| 报废（F/G） | 能量隔离程序、有害物质处置 |

## 过渡时间轴

```
2023-06-14  →  Regulation (EU) 2023/1230正式发布
2023-07-13  →  正式生效（Directive 2006/42/EC仍适用）
2024-01-20  →  可以开始按新Regulation申请认证
2027-01-20  →  全面强制执行（42个月过渡期结束）
2027年后    →  所有新上市机械须符合EU 2023/1230
```

## 协调标准体系变化

| 项目 | 内容 |
|------|------|
| 现行协调标准 | 依据2006/42/EC的协调标准清单 |
| 新协调标准 | 将逐步更新为符合2023/1230的新标准 |
| 协调标准网站 | europa.eu（EU Official Journal） |

**重要：** 2027年后，即使产品符合旧协调标准，若不符合新Regulation要求，仍须整改。

---

## CE标志新要求

| 项目 | 要求（2023/1230） |
|------|----------------|
| 标志高度 | ≥**5mm**（不变） |
| 可见性 | 须可见、清晰、不可磨灭（不变） |
| 附加信息 | **须同时标注**：制造商名称/地址、型号、批号/序列号、制造年份 |
| 数字CE | 数字文档时代替纸质粘贴（须可读取） |

## 中国机械出口欧盟注意事项

| 时间节点 | 要求 |
|---------|------|
| **现在（~2027）** | 按2006/42/EC申请CE；可提前按2023/1230做规划 |
| **2027年后** | 按2023/1230全面合规；重点关注AI功能认证和网络安全 |
| **AI功能** | 须提前准备Notified Body认证（约6~12个月） |
| **数字文档** | 建立e-TF技术文档管理系统 |
