## 第二部分：2018年机构改革回顾（PPTX文件解析）

### 2.1 自然资源部（MNR）★ 新设

**整合来源：**
- 国土资源部（MLR）→ 主体并入
- 国家海洋局（SOA）→ 主体并入
- 国家测绘地理信息局（隶属MLR）→ 并入
- 国家林业局（部分）→ 撤销，成立国家林业和草原局（归MNR）

**核心职责：**
- 自然资源调查评价/确权登记
- 建立空间规划体系（主体功能区规划→多规合一）
- 国有自然资源所有者权益
- 统一调查评价和确权登记
- 资源补偿机制建立

---

### 2.2 生态环境部（MEE）★ 由环保部升级

**整合来源：**
- 环境保护部（MEP）→ 撤销，升级为生态环境部
- 气候变化应对（从NDRC并入）
- 地下水污染监测（从MLR并入）
- 水功能区划/排口设置/流域保护（从MWR并入）
- 农村面源污染治理（从MoA并入）
- 海洋环境保护（从SOA并入）

**核心职责：**
- 起草生态环境法律法规和标准
- 统一生态环境监测和执法
- 污染防治攻坚战
- 核与辐射安全
- 中央生态环保督察

---

### 2.3 农业农村部（MARA）★ 新设

**整合来源：**
- 农业部 → 撤销，并入
- 国家发展和改革委农业投资项目管理 → 并入
- 国土资源部农田治理职责 → 并入
- 水利部农田灌溉管理职责 → 并入

**核心职责：**
- 统筹乡村振兴战略
- 农业投资项目管理
- 农田治理和高标准农田建设
- 畜牧/兽医/农药/农机管理

---

### 2.4 文化和旅游部（MCT）★ 合并

**整合来源：**
- 文化部 → 撤销
- 国家旅游局 → 撤销

**核心职责：**
- 统筹文化事业/文化产业/旅游业发展
- 文物考古与保护
- 文化市场执法
- 国际文化交流

### 2.5 退役军人事务部（MVA）★ 新设

**整合来源：**
- 民政部优抚安置局 → 剥离并入

### 2.6 应急管理部（MEM）★ 新设

**整合来源：**
- 国家安全生产监督管理总局（SAWS）→ 撤销，整体并入

### 2.7 国家市场监督管理总局（GAMR/SAMR）★ 整合（原PPTX称GAMR）

**整合来源：**
- 国家工商行政管理总局（SAIC）→ 整体并入
- 国家质量监督检验检疫总局（AQSIQ）→ 整体并入（含CNCA/SAC）
- 国家食品药品监督管理总局（CFDA）→ 整体并入
- 发改委价格监督与反垄断执法 → 剥离并入
- 商务部反垄断执法（国务院反垄断委员会办公室）→ 剥离并入

**核心职责：**
- 市场综合监督管理
- 反垄断执法（统一执法权）
- 食品安全监管（统筹协调，药品/医疗器械由新药监局负责）
- 特种设备安全
- 计量标准认证认可

**下设机构：**
- 国家药品监督管理局（NMPA）：药品/医疗器械/特殊食品监管
- 国家知识产权局（SIPO）：专利/商标/地理标志（由SAMR管理）

## 第三部分：2023年最新机构改革（深化改革）

### 3.1 国家金融监督管理总局（NFRA）★ 2023年新设

**成立时间：** 2023年5月18日（正式挂牌）
**整合来源：**
- 中国银行保险监督管理委员会（CBRC）→ 撤销
- 中国保险监督管理委员会（CIRC）→ 撤销
- 央行对金融控股公司的日常监管职责 → 并入
- 央行金融消费者保护职责 → 并入
- 证监会投资者保护职责 → 并入

**监管范围：**
- 银行、保险、信托、金融资产管理公司、金融租赁、汽车金融、消费金融等
- **排除：** 证券业（由证监会监管）
- **宏观审慎框架：** 央行保留制定权

**监管格局变化：**
```
旧格局（一行两会）：  央行 + 银保监会 + 证监会
新格局（一行一局一会）：央行 + 金融监管总局 + 证监会
```

**地方金融监管：**
- 设立31个省级监管局 + 5个计划单列市/副省级城市派出机构
- 地方金融监管局（金融办）剥离监管职能，保留发展职能

---

### 3.2 国家数据局（NDA）★ 2023年新设

**成立时间：** 2023年10月（正式挂牌）
**定位：** 国务院发改委管理的国家局

**主要职责：**
- 统筹数据基础制度建设
- 数字中国建设规划
- 数据要素市场化改革
- 公共数据开放共享
- 数据安全和个人信息保护

**下设机构：** 国家数据产业发展司、数据资源司等

**企业影响：**
- 数据资产入表/数据交易规则
- 企业数据合规（个人信息保护/数据安全法）
- 数据跨境流动（数据出境安全评估）

---

### 3.3 国家疾病预防控制局（NDCB）★ 2021年成立

**成立时间：** 2021年4月（2021年5月13日揭牌）
**定位：** 国家卫健委管理的国家局（副部级）

**整合来源：**
- 卫健委疾病预防控制相关职能 → 整合升级
- 疾控体系改革试点经验 → 全面推广

**核心职责：**
- 传染病及突发公共卫生事件监测预警
- 疾病预防控制体系现代化建设
- 公共卫生应急指挥协调
- 疾控网络建设和实验室管理

**下设机构：**
- 中国疾病预防控制中心（CDC，技术支撑单位）

## 第六部分：中央与地方监管对应关系

### 6.1 央地监管分工原则

```
党中央（决策层）
    ↓ 政策/法律/标准
国务院（执行层）
    ↓ 授权/监督
国务院组成部门（中央主管部门）
    ↓ 法律法规授权 + 省级政府执行
省级政府/省级主管部门
    ↓ 属地管理
地市级/区县级政府及主管部门
```
