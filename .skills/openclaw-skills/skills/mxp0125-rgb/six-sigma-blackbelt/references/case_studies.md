## 案例一：某家电企业注塑不良率改善

### 项目基本信息
- **行业**：家电制造（塑料件）
- **项目类型**：DMAIC
- **实施时间**：2025.01 - 2025.05（5个月）
- **黑带角色**：项目负责人

### Define（界定）

**问题陈述：**
注塑车间不良率长期维持在 3.8%（目标≤1.0%），年损失约 ¥380万元

**VOC客户之声：**
- 客户A（组装线）：塑料件缺料/飞边导致停线，每停线1次损失 ¥2000
- 客户B（品质部）：外观不良率高导致客户投诉月均3次

**Y定义：**
- Y₁：注塑不良率（%）
- Y₂：停线次数（次/月）

**基线数据：**
- 不良率：均值3.8%，σ=0.62%（12个月数据）
- Cpk=0.67（目标≥1.33）
- 主要不良模式：飞边（48%）、缺料（29%）、缩水（15%）、其他（8%）

**项目章程关键要素：**
| 项目 | 内容 |
|------|------|
| 项目名称 | 注塑车间不良率改善 |
| 团队构成 | 黑带1人+绿带2人+注塑工程师3人+品质工程师1人 |
| 范围 | 注塑车间全部机型（12台机） |
| 改善目标 | 不良率≤1.0%，Cpk≥1.33 |
| 财务收益 | ≥¥200万元/年 |
| 完成日期 | 2025.05.31 |

### Measure（测量）

**MSA分析：**
- 测量系统：卡尺（精度0.01mm）
- GRR结果：%GRR=8.2%（<10%），NDC=5（≥4）
- 结论：测量系统可接受

**过程能力分析：**
- Cpk=0.67，Ppk=0.71（不稳定过程）
- 主要变异来源：多变量分析显示（机台间差异43%，模具间差异31%，操作员间差异26%）

**不良模式Pareto分析：**
飞边48%+缺料29%=77%，优先改善飞边和缺料

### Analyze（分析）

**假设检验（不良率与机台）：**
- H₀：各机台不良率无显著差异
- H₁：各机台不良率有显著差异
- 检验方法：卡方检验（离散数据）
- 结果：P值<0.001，拒绝H₀，结论：机台间存在显著差异

**DOE试验（优化注塑参数）：**

| 因子 | 水平1（低）| 水平2（高）|
|------|-----------|-----------|
| 料温 | 210℃ | 230℃ |
| 保压压力 | 60 bar | 80 bar |
| 冷却时间 | 8s | 12s |

- 设计：L9（3³）正交表，9次试验
- 响应：不良率（%）
- 最优组合：料温220℃+保压压力70bar+冷却时间10s

**回归分析（缺料不良与料温/压力）：**
- 方程：不良率 = 12.3 - 0.056×料温 - 0.038×压力
- R²=0.89（解释89%的变异）
- 料温和压力均显著（P<0.05）

### Improve（改进）

**改善措施：**
1. 料温统一调整为220℃（±2℃控制）
2. 保压压力标准化为70bar（±2bar）
3. 模具点检周期从月度→周度
4. 操作员培训：标准化操作SOP

**改善结果：**
| 指标 | 改善前 | 改善后 | 改善幅度 |
|------|--------|--------|---------|
| 不良率 | 3.8% | 0.72% | 降低81% |
| Cpk | 0.67 | 1.48 | +0.81 |
| 停线次数 | 月均18次 | 月均3次 | 降低83% |

### Control（控制）

**SPC控制图：**
- I-MR图监控日不良率
- 控制限：UCL=1.5%，LCL=0%，CL=0.9%
- 更新控制图频率：每日

**控制计划：**
- 料温/压力点检：每班2次
- 模具清洁：每周1次
- 不良率报表：每日发给生产主管和质量经理
- 应急响应：不良率超1.5%立即停产审查

## 案例二：某手机制造商焊接不良改善

### 项目基本信息
- **行业**：消费电子（SMT焊接）
- **项目类型**：DMAIC
- **实施时间**：4个月
- **财务收益**：¥156万元/年

### 核心分析结果

**MSA分析（测量系统）：**
- X射线检测仪 %GRR=6.1%（<10%优秀）
- NDC=5（满足区分度要求）

**假设检验结果：**
- 两组焊料（A厂家/B厂家）不良率比较
- 两样本比例检验：P值<0.01 → B厂家的焊料不良率显著更低
- 决策：切换到B厂家焊料

**DOE结果（焊接参数优化）：**
- 因子：回流温度/焊接时间/助焊剂用量
- 最优参数：回流温度235℃+焊接时间45s+助焊剂量减少20%
- 改善后不良率：1.2%（目标达成）

## 案例三：某食品企业生产周期时间改善（精益黑带）

### 项目基本信息
- **行业**：食品加工
- **项目类型**：DMAIC + Lean
- **实施时间**：3个月

### 精益分析（Value Stream Map）

**现状VSM关键数据：**
| 工序 | 增值时间（VT）| 等待时间（WT）| 总周期时间 |
|------|------------|------------|---------|
| 原料检验 | 20min | 120min | 140min |
| 混合 | 45min | 30min | 75min |
| 包装 | 60min | 240min | 300min |
| 入库 | 10min | 180min | 190min |
| **总计** | **135min** | **570min** | **705min** |

**增值比（VT ratio）：** 135/705 = 19.1%（目标≥30%）

### 改善措施

**快速换模（SMED）：**
- 将原料检验和入库等待时间从批次结束前→批次开始前（并行处理）
- 包装等待时间减少60%（通过批次计划优化）

**5S改善：**
- 混合工序：工具定置管理，查找时间减少70%
- 包装工序：模具快速切换，切换时间减少55%

**改善结果：**
- 总周期时间：705min → 380min（改善46%）
- 增值比：19.1% → 35.5%（超过30%目标）
- 年化节省等待成本：¥89万元

## 案例四：某汽车零部件企业尺寸超差改善

### 项目基本信息
- **行业**：汽车零部件（冲压件）
- **问题**：关键尺寸公差超差率 2.3%
- **财务收益**：¥210万元/年

### DMAIC应用

**MSA分析（CMM测量系统）：**
- %GRR=12.3%（10-30%→可接受但需改进）
- NDC=4（接近4，需要改进）

**分析阶段关键发现：**
- 多变量分析：原材料批次间差异是主要变异来源（贡献率62%）
- 供应商审计：发现来料检验抽样方案不合理（AQL取样而非计量型）

**改善措施：**
1. 供应商质量改善协议（来料不良率要求≤0.5%）
2. 来料检验从AQL计数型→计量型（CMM测量）
3. 模具维护周期调整（从5000件→3500件）

**改善结果：**
- 尺寸超差率：2.3% → 0.28%
- Cpk：0.83 → 1.52
- 年化节约：¥210万元（退货/返工/停产损失减少）

## 案例五：DFSS新产品开发（手机摄像头密封设计）

### DFSS项目流程

**Define（界定）:**
- 客户需求：防水等级IP68（30分钟浸水1.5米）
- CTQ：密封圈压缩量（12-18%）、材质耐温性（-40℃~85℃）

**Measure（测量）:**
- 现有设计能力评估：IP67（不满足IP68）
- 测量系统：气密测试仪，GRR=5.2%

**Analyze（分析）:**
- FMEA识别：密封圈压缩量不足（第1风险，S=9）
- DOE优化密封圈参数：L18正交表
- 蒙特卡罗模拟：参数不确定性量化→DPMO预测

**Design（设计）:**
- 最优参数组合：压缩量15%（中值）、密封圈邵氏硬度58A
- 密封结构优化（倒角设计减少应力集中）

**Verify（验证）:**
- IP68防水测试：50台样机，0台渗漏
- 加速老化测试（85℃/85%RH，1000小时）：0不良
- 量产可行性：模具调整成本¥3万元，可接受

## 黑带项目常见失败原因与预防措施

### 失败模式一：数据基础薄弱

**症状：** MSA未通过，分析结论不可靠
**预防：**
- M阶段第一件事：做MSA（%GRR<10%再开始分析）
- 测量系统不合格必须先改善测量

### 失败模式二：项目范围过大

**症状：** 6个月无法完成，项目搁置
**预防：**
- 黑带项目范围：改善幅度≥50%，周期≤6个月
- 建议选择3个月能完成的项目先练手

### 失败模式三：改善措施未固化

**症状：** 项目结束后改善效果消失
**预防：**
- C阶段必须建立SPC控制图（不是报表）
- 控制计划必须嵌入生产流程，不是挂在墙上
- 交接培训：让操作员理解为什么要这样做

### 失败模式四：财务收益无法量化

**症状：** 财务部门不认可收益
**预防：**
- 项目开始前与财务确认COPQ基线计算方法
- 改善后3个月内财务确认，避免跨期影响
- 保留原始数据备查
