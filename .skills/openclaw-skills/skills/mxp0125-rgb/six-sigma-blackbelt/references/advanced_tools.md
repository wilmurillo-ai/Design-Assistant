## 一、FMEA失效模式与影响分析

### 1.1 DFMEA vs PFMEA核心差异

| 维度 | DFMEA（设计FMEA） | PFMEA（过程FMEA） |
|------|------------------|------------------|
| 启动时间 | 设计概念阶段（APQP第一阶段） | 过程设计阶段（APQP第三阶段） |
| 分析对象 | 产品设计特性 | 过程工序特性 |
| 责任方 | 设计工程师团队 | 工艺/质量工程师 |
| 失效模式 | 设计缺陷导致的潜在失效 | 制造/装配过程中的潜在失效 |
| 严重度S评分 | 基于设计缺陷对客户的影响 | 基于产品缺陷对客户的影响 |
| 发生度O评分 | 设计缺陷发生概率 | 过程失效发生概率 |
| 探测度D评分 | 现有设计验证方法探测能力 | 现有过程控制探测能力 |

### 1.2 严重度S评分标准（1-10分）

| 评分 | 客户影响 | 举例 |
|------|---------|------|
| 10 | 安全/法规违规，无预警 | 安全带失效/刹车失灵 |
| 9 | 无预警的重大产品失效 | 发动机抱死 |
| 8 | 重大产品失效（可预警）| 空调完全不制冷 |
| 7 | 中等失效，影响使用 | 收音机无声音 |
| 6 | 次要失效，轻微不便 | 点烟器不工作 |
| 5 | 舒适性问题，可察觉 | 收音机有杂音 |
| 4 | 轻微不便 | 仪表灯略暗 |
| 3 | 客户注意到瑕疵 | 表面有轻微划痕 |
| 2 | 客户几乎未注意 | 毫米级外观缺陷 |
| 1 | 无实际影响 | 无关紧要的外观差异 |

### 1.3 RPN决策矩阵

| RPN范围 | 风险等级 | 行动优先级 | 典型响应 |
|---------|---------|-----------|---------|
| **1-50** | 低风险 | 可接受 | 保持现有控制，不需额外措施 |
| **51-100** | 中等风险 | 中优先级 | 改进现有设计/工艺控制 |
| **101-200** | 高风险 | 高优先级 | 必须制定改善行动计划，明确责任人 |
| **> 200** | 极高风险 | 紧急处理 | 立即采取行动停止生产/发运，重新设计/工艺 |

**重要原则：** RPN不能代替S/O/D单项评估。例如高S值（9+）即使RPN不高也必须立即改善。

## 二、QFD质量机能展开

### 2.1 QFD四阶段展开

```
第一阶段：产品规划（House of Quality）
  客户需求（Whats）→ 技术特性（Hows）→ 关系矩阵 → 目标值

第二阶段：零部件展开
  技术特性 → 零部件特性 → 关系矩阵 → 零部件目标值

第三阶段：工艺规划
  零部件特性 → 工艺参数 → 关系矩阵 → 工艺目标值

第四阶段：工艺控制
  工艺参数 → 控制计划 → 关键控制点 → 检验规范
```

### 2.2 质量屋（House of Quality）核心要素

| 要素 | 说明 |
|------|------|
| 左墙 | 客户需求（Customer Requirements）|
| 天花板 | 技术特性/工程措施（Engineering Characteristics）|
| 屋顶 | 自相关矩阵（Relationships between Hows）|
| 房间 | 关系矩阵（How much each How addresses each What）|
| 右墙 | 重要度权重（Importance Ratings）|
| 地下室 | 目标值（Targets）|

### 2.3 QFD关系强度符号

| 符号 | 含义 | 分值 |
|------|------|------|
| ● | 强相关（Strong relationship）| 9 |
| ○ | 中等相关（Moderate relationship）| 3 |
| △ | 弱相关（Weak relationship）| 1 |
| （留空）| 无相关（No relationship）| 0 |

## 三、TRIZ发明问题解决理论

**39个工程系统参数（精选常用）：**

| 序号 | 参数 | 示例 |
|------|------|------|
| 1 | 运动物体重量 | 移动部件质量 |
| 2 | 静止物体重量 | 固定部件质量 |
| 3 | 运动物体长度 | 行程/尺寸 |
| 9 | 运动物体体积 | 容器容量 |
| 10 | 静止物体体积 | 设备体积 |
| 12 | 运动物体形状 | 外形设计 |
| 13 | 组成结构稳定性 | 结构完整性 |
| 14 | 强度 | 材料强度 |
| 17 | 运动物体温度 | 加热/冷却 |
| 21 | 运动物体速度 | 生产节拍 |
| 22 | 能量损耗 | 效率损失 |
| 23 | 物质损耗 | 材料浪费 |
| 26 | 物质便利性 | 加工难度 |
| 27 | 可靠性 | 失效率 |
| 32 | 制造精确性 | 加工精度 |
| 35 | 适应性/通用性 | 多品种切换 |

**40条发明原理（精选最常用）：**

| 原理 | 描述 | 应用场景 |
|------|------|---------|
| 1 | 分割 | 将对象分成独立部分 → 模块化设计 |
| 2 | 抽取 | 将噪声/有害部分分离 → 吸尘器的分离尘袋 |
| 3 | 局部质量 | 改变局部特性 → 刀具镀层 |
| 4 | 不对称 | 引入不对称 → 偏心轴承 |
| 5 | 组合 | 并列/序列组合 → 多功能合一 |
| 6 | 多功能性 | 一物多用 → 带写字功能的手机 |
| 8 | 配重 | 增加平衡质量 → 平衡轮 |
| 10 | 预加作用 | 预先执行必要改变 → 预紧螺栓 |
| 13 | 反向 | 反向思维 → 洗碗机洗蔬菜 |
| 15 | 动态化 | 刚体→可调节 → 万向节 |
| 17 | 一维变多维 | 3D取代2D → 立体仓库 |
| 18 | 振动 | 引入振动 → 超声波清洗 |
| 19 | 周期动作 | 连续→周期 → 脉冲加工 |
| 20 | 连续动作 | 周期→连续 → 连续焊接 |
| 22 | 有益作用放大 | 放大有益效果 → 高压水刀切割 |
| 23 | 反馈 | 引入闭环 → PID温控系统 |
| 24 | 中介物 | 引入中介 → 导轨定位 |
| 25 | 自服务 | 自我服务 → 自清洁涂层 |
| 28 | 机械系统替代 | 气动→电动 → 电动阀门 |
| 30 | 柔性壳体/薄膜 | 刚性→柔性 → 充气密封 |
| 33 | 多孔材料 | 引入多孔 → 蜂窝结构 |
| 35 | 参数改变 | 改变物态/浓度 → 冷却液浓度调节 |
| 39 | 惰性环境 | 充氮保护 → 防氧化存储 |

### 3.2 TRIZ应用步骤

```
Step 1: 识别问题（Define the problem clearly）
         ↓
Step 2: 转换为TRIZ问题（Map to a TRIZ problem）
         - 技术矛盾：两个参数冲突
         - 物理矛盾：同一参数有相反要求
         ↓
Step 3: 查表找原理（Apply TRIZ tools）
         - 技术矛盾：查39参数×40原理矩阵
         - 物理矛盾：分离原理（时间/空间/条件/系统）
         ↓
Step 4: 评估方案（Evaluate solutions）
         ↓
Step 5: 实施最优方案（Implement the best solution）
```

## 四、田口稳健设计（Taguchi Method）

### 4.1 正交表速查表

| 正交表 | 类型 | 因子数 | 水平数 | 试验次数 | 适用场景 |
|--------|------|--------|--------|---------|---------|
| L4 | 2³ | 3 | 2 | 4 | 筛选3因子 |
| L8 | 2⁷ | 7 | 2 | 8 | 筛选7因子 |
| L9 | 3⁴ | 4 | 3 | 9 | 4因子3水平 |
| L12 | 2¹¹ | 11 | 2 | 12 | 11因子筛选 |
| L16 | 2⁵×4¹ | 5个2水平+1个4水平 | 混合 | 16 | 混合水平 |
| L18 | 2¹×3⁷ | 1个2水平+7个3水平 | 混合 | 18 | 混水平优化 |
| L25 | 5⁶ | 6 | 5 | 25 | 5水平全面试验 |

### 4.2 信噪比（SN比）计算公式

| 类型 | 公式 | 适用场景 |
|------|------|---------|
| 望目（NB）| SN = -10×log₁₀(σ²/μ²) | 目标值+追求变异最小 |
| 望小（SB）| SN = -10×log₁₀(∑y²/n) | 越小越好（缺陷数/磨损量）|
| 望大（LB）| SN = -10×log₁₀(∑(1/y²)/n) | 越大越好（强度/寿命）|

### 4.3 田口DOE与常规DOE对比

| 维度 | 田口DOE | 常规DOE |
|------|---------|--------|
| 目标 | 稳健性（抗噪声干扰）| 效应显著性（统计分析）|
| 正交表 | 正交表（牺牲部分交互作用）| 完整因子设计或部分因子设计 |
| 控制因子 | 目标因子（可调控）| 主效应因子 |
| 噪声因子 | 明确标识（温度/湿度波动）| 不纳入设计 |
| 评价指标 | SN比（信噪比）| F值/P值 |
| 优化逻辑 | 优化SN比最大的参数组合 | 找统计显著因子 |

### 4.4 田口设计四步骤

1. **设定特性**：望目/望小/望大
2. **选择正交表**：根据因子数和水平数选择L
3. **安排试验**：将因子分配到正交表各列
4. **分析SN比**：计算各试验SN比 → 方差分析 → 确定最优组合

## 五、集成应用框架（QFD+TRIZ+田口）

### 5.1 DFSS集成设计路径（跨国研发中心标准流程）

```
阶段一：VOC + QFD（需求展开）
客户需求 → 质量屋 → 技术特性优先级排序
    ↓
阶段二：TRIZ（创新概念生成）
技术矛盾/物理矛盾 → 40条发明原理 → 创新概念方案
    ↓
阶段三：田口DOE（稳健参数设计）
正交表设计 → SN比分析 → 最优参数组合
    ↓
阶段四：FMEA（风险评估）
潜在失效模式 → S/O/D评分 → RPN改善优先序
    ↓
阶段五：蒙特卡罗模拟（量化风险）
参数分布 → 随机模拟 → DPMO预测
    ↓
阶段六：设计验证（DFM/DFA）
可制造性/装配性/测试性/维修性
```

### 5.2 蒙特卡罗模拟（Monte Carlo Simulation）

**原理：** 用随机抽样模拟参数不确定性，计算最终质量特性分布

**步骤：**
1. 确定输入参数的分布（正态/均匀/对数正态等）
2. 根据分布类型生成随机数（通常n≥10000次）
3. 代入Y = f(X)计算输出Y值分布
4. 统计Y的均值/标准差/DPMO

**Minitab路径：** Calc → Random Data → 选择分布 → 模拟n次 → Calc → Column Statistics
