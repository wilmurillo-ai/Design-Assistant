# 效果归因模型说明

本文档定义常用的效果归因模型及其适用场景。

---

## 一、归因模型概述

归因模型用于确定各渠道/触点对最终转化的贡献程度。

---

## 二、常用归因模型

### 1. 首次触点归因 (First Touch)

**定义**：用户首次接触的渠道获得 100% 归因

**适用场景**：
- 品牌认知类活动
- 新用户获取分析
- 品牌投放效果评估

**优缺点**：
| 优点 | 缺点 |
|------|------|
| 简单直观 | 忽略后续触点影响 |
| 适合品牌类评估 | 低估转化类渠道 |

**示例**：
```
用户路径：搜索广告 → 社群推送 → 下单
归因结果：搜索广告 100%
```

---

### 2. 末次触点归因 (Last Touch)

**定义**：用户转化前最后一次接触的渠道获得 100% 归因

**适用场景**：
- 促销转化类活动
- 直接转化分析
- 效果投放评估

**优缺点**：
| 优点 | 缺点 |
|------|------|
| 突出临门一脚 | 低估品牌/认知渠道 |
| 适合促销活动 | 忽略前置触点价值 |

**示例**：
```
用户路径：搜索广告 → 社群推送 → 下单
归因结果：社群推送 100%
```

---

### 3. 线性归因 (Linear)

**定义**：用户路径上所有触点平均分配归因

**适用场景**：
- 多渠道协同类活动
- 整体营销效果评估
- 中立分析场景

**优缺点**：
| 优点 | 缺点 |
|------|------|
| 考虑所有触点 | 忽略触点重要性差异 |
| 相对公平 | 可能高估低价值触点 |

**示例**：
```
用户路径：搜索广告 → 社群推送 → App推送 → 下单
归因结果：每个触点 33.3%
```

---

### 4. 时间衰减归因 (Time Decay) ⭐ 推荐

**定义**：越接近转化的触点获得越高权重

**计算公式**：
```
权重 = 0.5^(n-1)
n = 距离转化的天数
```

**适用场景**：
- 电商大促活动
- 限时促销活动
- 短周期营销活动

**优缺点**：
| 优点 | 缺点 |
|------|------|
| 符合用户决策逻辑 | 需要完整用户路径数据 |
| 平衡前后触点价值 | 计算相对复杂 |

**示例**：
```
用户路径（转化当天）：
- 3天前：搜索广告   权重 = 0.5^2 = 0.25
- 1天前：社群推送   权重 = 0.5^0 = 1.00
- 当天：App推送     权重 = 0.5^0 = 1.00

归因结果：
- 搜索广告：0.25 / (0.25+1+1) = 11.1%
- 社群推送：1.00 / 2.25 = 44.4%
- App推送：1.00 / 2.25 = 44.4%
```

---

### 5. 位置归因 (Position Based)

**定义**：首次和末次触点各占 40%，中间触点平分 20%

**适用场景**：
- 品牌类 + 转化类混合活动
- 全链路营销分析

**示例**：
```
用户路径：搜索广告 → 社群推送 → App推送 → 下单
归因结果：
- 搜索广告（首次）：40%
- App推送（末次）：40%
- 社群推送（中间）：20%
```

---

## 三、模型选择指南

| 活动类型 | 推荐模型 | 原因 |
|----------|----------|------|
| 品牌投放 | 首次触点 | 关注用户首次认知 |
| 促销转化 | 末次触点 | 关注临门一脚 |
| 大促活动 | 时间衰减 | 符合决策路径 |
| 全链路营销 | 线性/位置 | 考虑所有触点 |

---

## 四、归因配置

```yaml
attribution:
  # 归因模型
  model: "time_decay"
  
  # 回溯窗口（天）
  lookback_window: 7
  
  # 衰减系数
  decay_factor: 0.5
  
  # 渠道权重（可选）
  channel_weights:
    - name: "搜索广告"
      weight: 1.0
    - name: "社群推送"
      weight: 1.0
    - name: "App推送"
      weight: 1.2  # 可以手动调整
```

---

## 五、争议处理规则

1. **数据为准**：以埋点数据为唯一依据
2. **模型优先**：使用时间衰减模型作为默认
3. **口径一致**：全周期使用同一归因模型
4. **透明公开**：归因逻辑向所有干系人公开
