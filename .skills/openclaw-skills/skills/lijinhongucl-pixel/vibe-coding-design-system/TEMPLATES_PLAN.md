## 📐 每个模板的完整内容

每个模板都必须包含完整的 **9 个部分**：

1. **Visual Theme & Atmosphere** (500+ 行)
   - 设计哲学、氛围、密度、设计语言
   - 目标用户、品牌个性
   - 设计原则（3-5条）
   - 视觉隐喻（3-5个）

2. **Color Palette & Roles** (200+ 行)
   - 主色调（4-6个，含RGB）
   - 辅助色（4-6个）
   - 语义色（成功/警告/错误/信息）
   - 渐变（2-3个示例）
   - 无障碍访问标准

3. **Typography Rules** (300+ 行)
   - 字体家族（中文/英文/代码）
   - 字体层级表（h1-h6, body, caption）
   - 中英文混排规范
   - 特殊排版（代码块、引用、渐变文字）
   - 数字字体

4. **Component Stylings** (400+ 行)
   - 按钮（primary/secondary/ghost，所有状态）
   - 输入框（default/focus/error/disabled）
   - 卡片（default/interactive/media）
   - 导航（header/sidebar/breadcrumbs）
   - 反馈（alerts/toasts/modals/tooltips）
   - 完整CSS代码示例

5. **Layout Principles** (200+ 行)
   - 间距系统（xs-3xl，含用法）
   - 网格系统（列数、间距、最大宽度）
   - 断点定义（mobile/tablet/desktop）
   - 布局模式（5种典型布局）
   - 留白哲学

6. **Depth & Elevation** (150+ 行)
   - 阴影系统（sm/md/lg/xl/2xl）
   - z-index层级（base到toast）
   - 层级原则
   - 玻璃态效果

7. **Do's and Don'ts** (200+ 行)
   - ✅ 正确做法（8-10条）
   - ❌ 错误做法（8-10条）
   - 常见错误（5个）
   - 中文排版注意事项（6条）

8. **Responsive Behavior** (150+ 行)
   - 断点系统表
   - 触摸目标尺寸
   - 响应式模式（导航/网格）
   - 折叠策略

9. **Agent Prompt Guide** (300+ 行)
   - 快速参考（CSS变量）
   - 常用提示词（3-5个）
   - 代码模板（React组件 + CSS Module）
   - AI Agent指令（10条）
   - 组件生成清单（10项）
