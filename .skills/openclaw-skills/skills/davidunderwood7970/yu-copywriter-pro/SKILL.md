# Copywriter ✍️

营销文案全能助手，从标题到落地页一站搞定。

## 使用方式

| 需求 | 命令 | 说明 |
|------|------|------|
| 标题生成 | `headline` | 100个标题公式+变体生成 |
| 正文撰写 | `body` | 不同风格正文写作 |
| AIDA框架 | `aida` | AIDA框架文案生成 |
| 痛点挖掘 | `pains` | 用户痛点分析+文案转化 |
| 客户证言 | `testimonial` | 客户评价/案例文案 |
| 落地页 | `landing` | 完整落地页文案结构 |

## 运行

```bash
bash scripts/copy.sh <command> [args...]
```

- `headline` — 标题公式库(100个)+变体生成

- `pains` — 用户痛点挖掘+文案转化
- `testimonial` — 客户证言/评价文案
