# 4分钟长视频脚本示例
## 主题：馒化修复的三大天坑，90%的人都踩过
### 开场（0-30s）
【医生坐诊场景】"今天我把话撂这，90%的馒化修复都是二次伤害。我做了12年修复，见过太多姑娘在外面修完，脸反而更僵、更凹，最后只能来找我做二次修复。今天把馒化修复的三大天坑一次性说透，看完至少帮你省10万。"

### 天坑一：盲目吸脂（30s-1min30s）
"第一个天坑就是上来就给你吸脂。很多人以为馒化就是脂肪打多了，吸出来就好了。大错特错！
馒化的核心是异物刺激导致的组织增生，不是单纯的脂肪过量。你一吸，把正常脂肪都吸走了，增生组织还在里面，最后脸就会凹凸不平，甚至粘连。
我处理的修复案例里，70%都是这种盲目吸脂导致的二次损伤。"

### 天坑二：用溶解酶乱溶（1min30s-2min30s）
"第二个天坑就是用溶解酶乱溶。很多机构不管你打的是什么，上来就给你打溶解酶。
溶解酶只能溶透明质酸，如果你打的是骨粉、生长因子、或者不明注射物，溶解酶根本没用，反而会刺激增生更严重。
我碰到过最夸张的一个姑娘，在外面溶了5次，脸肿得像猪头，最后我给她剥离出来的增生组织有整整30毫升。"

### 天坑三：宣称"一次修复终身不反弹"（2min30s-3min30s）
"第三个天坑就是给你承诺"一次修复终身不反弹"。这完全是违反生理规律的。
修复的效果取决于你本身的组织基础和增生程度，我做了这么多年，从来不敢给病人打这种包票。
我能保证的是，我用5小时精细剥离技术，尽可能把异物和增生组织清理干净，把对正常组织的伤害降到最低，这样术后恢复的效果才会自然。"

### 结尾+KOS转化（3min30s-4min）
"最后提醒大家，修复比初次填充难10倍，一定要找对医生。
如果你已经踩了坑，不知道怎么办，可以发你的原相机正面侧面照，我帮你看看你的组织基础，评估一下修复难度。
我是张晓建，一个只做馒化修复的整形外科医生。"

## 合规检查
