# medical-ip-director 医疗IP操盘手技能

## 技能简介
这是一个专为医疗医美行业打造的AI内容生产工具，集成了十年医疗IP运营经验，能够帮助医生和运营团队快速产出合规、高转化的内容，搭建完整的内容矩阵体系。

## 核心功能
1. **语料智能处理**：从医生口语化访谈中自动提炼核心观点和技术壁垒
2. **合规自动审核**：内置医疗广告法违禁词库，自动过滤违规内容
3. **多格式内容生成**：支持30秒短视频、4-5分钟长视频脚本生成
4. **KOS转化植入**：自动在内容中植入无痕转化引导，提升留资率
5. **内容矩阵管理**：自动生成和维护医生IP超级内容矩阵库
6. **热点快速响应**：能够结合热点事件产出专业内容，借势引流

## 适用人群
- 医疗美容机构运营团队
- 医生IP运营操盘手
- 内容创作团队
- 想要打造个人IP的医生

## 使用方法

### 基础使用
1. **语料分析**：`/analyze [医生访谈语料]`
   - 自动提取核心观点、技术壁垒
   - 完成违禁词审查
   - 自动归类到内容矩阵

2. **脚本生成**：`/script [时长/对标链接]`
   - 支持指定时长：30s/1min/5min
   - 支持对标爆款视频，保留节奏替换内容
   - 自动附加KOS转化结尾

3. **热点内容**：`/hot_topic [热点事件/明星生图]`
   - 用医生专业观点拆解热点
   - 实现热点引流+专业转化

4. **矩阵存档**：`/save_matrix`
   - 导出完整的医生IP内容矩阵库
   - 格式为Markdown，方便后续使用

### 示例
```
/analyze 我做馒化修复已经12年了，核心技术是5小时精细剥离，不会伤害正常组织，市面上很多机构都做不到这么精细，因为他们没有解剖学基础。

/script 30s 对标这个爆款视频的节奏：https://v.douyin.com/xxxxxx

/hot_topic 某明星馒化脸修复事件
```

## 合规说明
本技能严格遵守《医疗广告管理办法》及平台内容规范，所有输出内容均经过违禁词过滤，但建议最终发布前仍需人工审核，确保符合当地监管要求。
