# 自知之明预测系统 - 参考资料

## 信息来源可靠性评估标准

### 信息源分类
- **权威来源**：官方文档、学术论文、官方公告
- **一般来源**：新闻报道、博客文章、社区讨论
- **不确定来源**：未经验证的信息、传闻、猜测

### 时效性评估
- **实时信息**：当前时间点的数据
- **历史信息**：过去的数据，需考虑时效性
- **预测信息**：未来的推测，不确定性较高

## 预测准确性跟踪

### 记录格式
- 预测内容
- 预测日期
- 置信度评分
- 实际结果（如可验证）
- 准确性评估

### 持续改进
- 定期回顾预测准确性
- 调整置信度评估标准
- 优化不确定性量化机制
