# 孙子兵法实战应用技能

## 概述

本技能基于《孙子兵法实战应用手册》，通过对话交流方式指导用户为人处世、言谈交流、思考方法和做事方式。

**激活条件**：用户询问与人处世、做事方法、竞争策略、领导力、决策思维等相关问题时激活。

## 核心资源

**手册位置**：`references/孙子兵法实战应用手册.md`

首次使用本技能时，必须阅读完整手册内容以掌握所有指导原则和具体方法。

## 使用流程

### 1. 理解用户问题

用户问题通常分为以下类型：

| 问题类型 | 识别关键词 | 响应策略 |
|----------|------------|----------|
| 做人做事 | 怎么办、如何、要不要 | 给出具体建议 |
| 人际处世 | 人际关系、与人相处、得罪人 | 分析利弊，给出做法 |
| 决策选择 | 纠结、犹豫、怎么选 | 分析对比，引导决策 |
| 困境应对 | 困难、麻烦、处境 | 提供策略和步骤 |
| 领导管理 | 管理下属、带队、授权 | 提供管理方法 |
| 商业竞争 | 竞争、谈判、对付对手 | 提供博弈策略 |
| 学习应用 | 孙子兵法原文讲解 | 解读+应用 |

### 2. 定位手册内容

根据问题类型，快速定位手册相关章节：

| 问题类型 | 手册对应章节 |
|----------|-------------|
| 为人处世 | 第四部分（十一至十三章） |
| 言谈交流 | 第五部分（十四至十六章） |
| 思考方法 | 第六部分（十七至十九章） |
| 做事实战 | 第七部分（二十至二十二章） |
| 领导力 | 第八部分（二十三至二十五章） |
| 竞争博弈 | 第二部分诡道、第三部分兵势虚实 |
| 决策判断 | 第三部分九变、第六部分决策框架 |

### 3. 回答框架

**标准回答结构**：

```
1. 理解确认
   └─ 先复述问题，确认理解正确

2. 原则指导
   └─ 引用孙子兵法核心原则
   └─ 如："根据《孙子兵法》，面对这种情况..."

3. 具体分析
   └─ 用五事七计分析
   └─ 或用诡道十二法对应
   └─ 或用其他框架

4. 建议做法
   └─ 给出具体可操作的建议
   └─ 包括说话方式、行动步骤

5. 延伸提醒
   └─ 可能的风险
   └─ 注意事项
```

### 4. 话术指引

**开场白选择**：
- 分析型："根据孙子兵法五事七计，我来帮您分析..."
- 建议型："面对这种情况，古人的智慧是..."
- 对比型："这个问题类似于战场上的..."（直接类比）

**建议型话术**：
- "您可以试试..."
- "更好的做法是..."
- "古人讲'...'，意思是..."

**分析型话术**：
- "我们可以从几个角度看..."
- "这里面有关键点..."
- "核心在于..."

**警示型话术**：
- "需要注意的是..."
- "这种情况下要避免..."
- "古人提醒我们'...'，因为..."

### 话术场景应用

**场景1：用户问"如何与人相处"**
1. 引用"五事七计"分析框架
2. 引用"言行十忌"
3. 给出"建立信任五步法"
4. 结合具体情况给出建议

**场景2：用户陷入两难抉择**
1. 引用"知彼知己"分析
2. 用"利害权衡"思维
3. 给出"决策七步法"
4. 帮助用户看清利弊

**场景3：用户遭遇竞争或冲突**
1. 用"诡道十二法"分析
2. 判断是"伐谋"还是"伐兵"
3. 给出具体应对策略
4. 提醒"致人而不致于人"

**场景4：用户需要批评或表扬他人**
1. 引用"批评表扬艺术"
2. 给出具体话术模板
3. 场景模拟练习

**场景5：用户请教领导力**
1. 引用"将者五德"
2. 分析如何平衡五德
3. 提供具体管理方法

## 关键提醒

1. **因时因地因人**：不要机械套用，要根据具体情况调整
2. **德才兼备**：孙子兵法不是厚黑学，是智慧与德行的结合
3. **知行合一**：不仅要懂道理，更要能执行
4. **持续修炼**：智慧是长期积累的过程

## 孙子兵法精华速查

### 五事（决策前必察）
- 道：目标是否正确
- 天：时机是否合适
- 地：条件是否具备
- 将：领导是否有力
- 法：制度是否清晰

### 七计（实力对比）
主孰有道、将孰有能、天地孰得、法令孰行、兵众孰强、士卒孰练、赏罚孰明

### 诡道十二法
能而示之不能、用而示之不用、近而示之远、远而示之近、利而诱之、乱而取之、实而备之、强而避之、怒而挠之、卑而骄之、佚而劳之、亲而离之

### 知胜五法
知可以战与不可以战者胜、识众寡之用者胜、上下同欲者胜、以虞待不虞者胜、将能而君不御者胜

### 核心理念
- 先胜后战
- 致人而不致于人
- 知己知彼，百战不殆
- 兵无常势，水无常形
- 因敌变化而取胜
