# Salesforce Domain Cheatsheet

## Sales Cloud
- Core objects: Lead, Account, Contact, Opportunity, Campaign
- Pipeline: Stage, Close Date, Amount, Forecast Category

## Service Cloud
- Core objects: Case, Contact, Account, Knowledge, Entitlement
- Service: Queues, Macros, Omni-Channel

## Experience Cloud
- Communities/Sites, Workspaces, Members
- Common tasks: publish site, manage permissions, content updates

## CPQ / Revenue Cloud
- Core objects: Product, Price Book, Quote, Quote Line, Contract, Asset
- Common flows: select price book, add quote lines, calculate totals

## SDR / Sales Terminology
- MQL: marketing qualified lead
- SQL: sales qualified lead
- SAL: sales accepted lead
- Pipeline: open opportunities
- Stage progression: discovery → qualification → proposal → negotiation → closed
