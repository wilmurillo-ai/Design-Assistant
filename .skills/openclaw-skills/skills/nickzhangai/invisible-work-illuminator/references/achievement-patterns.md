# Achievement Patterns

## 1. Weekly update

### Template

> 本周我围绕……做了系统梳理，重点完成了……，并识别出……，为后续……提供了基础。

## 2. Performance review

### Template

> 我持续承担了若干隐性但关键的工作，包括……、……、……。这些工作提升了……，并帮助团队在……上更快推进。

## 3. Project recap

### Template

> 这项工作的价值不只是产出了一份材料，更重要的是把原本分散、难以行动的信息收束成了可判断、可讨论、可推进的问题框架。

## 4. Resume bullet

### Template

> Synthesized fragmented cross-functional feedback into actionable priorities, improving decision clarity and reducing coordination overhead.

## 5. No-metrics but still honest

When numbers are unavailable, describe one of these:

- ambiguity reduced
- blockers surfaced
- alignment improved
- decision speed improved
- repeated effort avoided
