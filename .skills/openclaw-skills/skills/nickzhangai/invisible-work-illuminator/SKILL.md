# Invisible Work Illuminator

Make hidden work visible without faking impact.

## Core Job

Translate work like:

- collecting scattered feedback
- aligning people
- reducing confusion
- clarifying priorities
- diagnosing causes
- preventing issues
- cleaning up messy processes

into outcome language that others can recognize.

## Workflow

1. Extract the hidden work.
   - What did the user actually do?
   - What complexity did they absorb?
   - What confusion or risk did they reduce?
2. Identify the value created.
   - clearer decisions
   - lower risk
   - saved time
   - better prioritization
   - smoother collaboration
3. Convert effort into achievement statements.
4. Match the output to the use case:
   - weekly update
   - manager sync
   - performance review
   - project recap
   - resume bullet

## Output Pattern

Default to:

1. **你实际做了什么**
2. **这件事为什么有价值**
3. **可以怎么写成果**
4. **短版 / 正式版 / 汇报版**

## Writing Rules

- Do not fabricate metrics.
- If hard numbers are unavailable, describe leverage:
  - reduced ambiguity
  - clarified priorities
  - surfaced blockers
  - converted noise into action
- Prefer verbs like:
  - 梳理
  - 提炼
  - 对齐
  - 识别
  - 归因
  - 推进
  - 沉淀
- Show transformation, not busyness.

## Good Achievement Frames

### A. From effort to outcome

```text
我将分散反馈整理为结构化问题清单，并识别出当前最关键的阻塞项。
```

### B. From coordination to leverage

```text
我把多方零散信息压缩成可执行判断，帮助团队更快确定优先级。
```

### C. From cleanup to impact

```text
我不是只做了记录，而是把模糊问题转成了后续可推进的动作。
```
