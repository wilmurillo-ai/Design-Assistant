## 速览

- 平台：智谱开放平台 / BigModel
- 生态：GLM / ChatGLM 系列
- 支持方式：HTTP API、官方 SDK、OpenAI SDK 兼容
- 常见能力：文本、多模态、智能体、精调、评测

## 文档中可见的关键信息

- 官方文档明确列出多种接入路径：
  - HTTP API 调用
  - 官方 Python SDK
  - 官方 Java SDK
  - OpenAI SDK 兼容
  - LangChain 集成
- 平台强调模型概览、上下文长度、价格等都在文档内可查

## 使用建议

1. 官方文档入口
2. 让用户确认要走哪条接入路线：
