## 📊 核心筛选逻辑

基于对用户实际筛选行为（120+ 条数据）的分析，筛选逻辑遵循**三层判断**：

> **「设计相关性 × 实用性 × 创新性」三维评分**

| 维度 | 评分标准 | 权重 |
|------|---------|------|
| **设计相关性** | 是否直接影响设计工作？ | 40% |
| **实用性** | 能否直接用/参考？ | 35% |
| **创新性** | 是否有新交互/新洞察？ | 25% |

**综合得分≥70%** → 采用  
**综合得分<70%** → 排除

## 📊 动态分布策略（v2.1.5 新增）

**核心理念**：根据内容质量动态调整，不强制平衡比例。

**原则**：
1. **质量优先** — 优质内容多的领域多输出，优质内容少的领域少输出
2. **不设上限** — 某领域今天特别重要，可以占 50%+
3. **不设下限** — 某领域今天没新闻，可以只占 5% 或更少
4. **总量控制** — 每日输出 35-50 条，避免信息过载

**示例**：
- AI 领域今天有重大事件（如 Anthropic 胜诉）→ AI 领域可占 40-50%
- 手机领域今天没大新闻 → 手机领域可只占 5-10%
- 设计领域有重要周刊发布 → 设计领域可占 30-35%

**优势**：
- ✅ 反映真实新闻分布
- ✅ 不错过重要事件
- ✅ 避免为了平衡而降低标准

## 🎯 边界情况判断

### 判断流程

```
1. 是否是专利/交互细节？
   ├─ 是 + 交互创新 → ✅ 采用（100%）
   ├─ 是 + 常规专利 → ❌ 排除
   └─ 否 → 继续

2. 是否是设计工具/资源？
   ├─ 是 → ✅ 采用（85%）
   └─ 否 → 继续

3. 是否是深度文章/分析？
   ├─ 是 → ✅ 采用（85%）
   └─ 否 → 继续

4. 是否是 AI+ 设计工作流？
   ├─ 是 → ✅ 采用（80%）
   └─ 否 → 继续

5. 是否是大厂 AI/设计战略？
   ├─ 是 → ✅ 采用（60%）
   └─ 否 → 继续

6. 是否是设计作品/审美参考？
   ├─ 是 + 知名设计师/高讨论度/奖项 → ✅ 采用（85%）
   ├─ 是 + 优质创意 → ✅ 采用（70%）
   ├─ 是 + 常规展示 → ❌ 排除
   └─ 否 → 继续

7. 是否是纯资讯/公告？
   ├─ 是 + 设计影响 → ✅ 采用
   ├─ 是 + 无设计影响 → ❌ 排除
   └─ 否 → 继续

8. 是否是社区八卦/教程/营销？
   ├─ 是 → ❌ 排除
   └─ 否 → 边界判断
```

## 📝 输出格式要求

### 情报卡片格式

```markdown
### [排名]. [公司/事件]

**标题**: [情报标题](链接)
**维度**: [1][3] | **分级**: S
**来源**: 来源名称
**时间**: 发布时间
**摘要**: 1-2 句核心信息

**设计师意义**: 
- 对设计工作的直接影响
- 可能带来的设计机会
- 需要警惕的风险
```

### 趋势洞察格式

```markdown
## 💡 趋势洞察：[趋势名称]

### 👁️ 观察（事实层）
- 客观描述现象
- 关键信号 1、2、3

### 🧠 分析（解读层）
- 为什么现在发生？
- 设计师视角的解读

### 🔮 预测（预判层）
- 短期影响（1-3 个月）
- 中期影响（3-12 个月）
```

## 📰 周刊深度抓取规范（v1.6.0 新增）

### 适用场景

**日报/周报中出现周刊源时**（如 Moonvy 设计素材周刊、优设网体验碎周报），必须执行深度抓取：

| 周刊名称 | RSS 摘要局限 | 深度抓取要求 |
|----------|-------------|-------------|
| Moonvy 设计素材周刊 | 仅显示简介，无具体产品/工具链接 | 进入当期详情页，抓取完整内容 |
| 优设网体验碎周报 | 仅显示固定介绍语 | 进入当期详情页，抓取完整内容 |

### 执行流程

```
1. 检测到周刊源（Moonvy/优设网体验碎周报）
   ↓
2. 提取期数，构建详情页 URL
   ↓
3. 使用 web_fetch 抓取详情页（extractMode=markdown）
   ↓
4. 结构化整理内容（按栏目分类 + 保留所有链接）
   ↓
5. 输出到日报/周报的「🎨 设计领域」或独立「📎 设计资源」板块
```

### 输出要求

**日报/周报中的周刊内容必须包含**:

- ✅ **完整栏目结构** — 按原周刊的栏目分类呈现
- ✅ **所有产品/工具链接** — 方便用户直接访问
- ✅ **结构化表格** — 产品推荐/工具资源使用表格呈现
- ✅ **核心亮点摘要** — 1-2 句说明产品/工具的价值

**禁止行为**:

- ❌ 仅输出 RSS 摘要（如"设计素材周刊 206 期"无具体内容）
- ❌ 省略产品/工具链接
- ❌ 不分类别混合呈现

### 示例对比

**❌ 错误（仅 RSS 摘要）**:
```
### 🎨 设计领域
- Moonvy 设计素材周刊 206 期
- 优设网体验碎周报第 273 期
```

**✅ 正确（深度抓取后）**:
```
### 🎨 设计领域
**Moonvy 设计素材周刊 206 期**:
- Figma AI 教育版开放使用（每月 3000 积分）
- 谷歌 Stitch 推出「氛围设计」
- 7 款产品推荐：Motion Core 动画库、字体预览工具...

**优设网体验碎周报 273 期**:
- 5 个大产品小细节：Petcube 切焦效果、OPPO 折叠计数...
- 10 款产品发现：个人资产管理、AI 写作识别、短剧 Agent...
```
