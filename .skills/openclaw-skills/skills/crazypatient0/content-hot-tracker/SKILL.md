## What This Skill Does

1. **热点追踪** - 实时监控微博、知乎、抖音、小红书、B站 等平台的热搜/热门话题
2. **趋势分析** - 分析话题热度趋势，识别上升期话题
3. **选题建议** - 根据热点生成内容选题方向
4. **竞品分析** - 分析同领域创作者的热门内容
5. **标题建议** - 提供爆款标题模板和写作方向

## When to Use

- 日常内容选题枯竭时
- 追热点不知道如何切入时
- 想了解行业最新动态时
- 不知道什么内容能火时

## How It Works

### 1. 热点追踪

使用 web_search 工具搜索各平台实时热点：

```
搜索格式: "{平台} 热搜榜 {日期}"
例如: "微博热搜 2026-03-13" / "抖音热门话题 今天"
```

### 2. 趋势分析

分析话题热度：
- 上升期话题：优先追，流量红利期
- 平稳期话题：可追，需差异化角度
- 下降期话题：避免，已过热度高峰

### 3. 选题生成

根据热点提供 3-5 个具体选题方向：
- 观点类：针对热点事件的观点输出
- 知识类：热点相关的知识科普
- 故事类：热点相关的真实经历/案例
- 盘点类：热点事件的综合整理
- 预测类：热点事件的未来展望

### 4. 竞品分析

搜索同领域热门内容：
- "关键词 + 爆款 / 10万+ / 点赞过万"
- 分析高赞内容的共同特点

### 5. 标题建议

提供 5-8 个备选标题：
- 数字开头型
- 悬念型
- 对话型
- 情绪型

## Configuration

### 偏好设置

可以自定义以下设置：
- 偏好平台：微博/抖音/小红书/知乎/B站
- 内容领域：科技/美食/职场/情感/娱乐
- 选题风格：深度/轻松/干货/故事

### 设置命令

```
# 设置偏好领域
设置内容领域为[科技]

# 设置偏好平台
主要追踪[微博]和[抖音]
```

## Limitations

- 热点数据来自搜索，可能不完全实时
- 某些平台可能需要 VPN
- 趋势分析基于公开数据，可能有偏差
