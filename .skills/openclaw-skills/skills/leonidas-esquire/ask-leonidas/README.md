# Ask Leonidas — LEONIDAS Prompt Generator for OpenClaw

**Turn any professional pain point into a structured, quality-scored AI prompt in seconds.**

Ask Leonidas is a prompt engineering API built on the LEONIDAS framework — a structured approach to prompt design that automatically infers your role, industry, and desired outcome, then returns a ready-to-use prompt scored for quality and classified by type.

---

## Why this skill exists

The OpenClaw community keeps running into the same wall: **the agent is installed, the skills are loaded, and it still feels generic.** The root cause is almost always the same — the prompts driving the agent's behavior were written quickly, without structure, and without a clear framework.
