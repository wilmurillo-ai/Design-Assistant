# AI News Collector - OpenClaw Skill

AI news collection and curation assistant

## Installation

### Via ClawHub
```bash
clawhub install modesty-ai-news-collector
```

### Manual Installation
```bash
git clone https://github.com/ModestyRichards/ai-news-collector.git
cp -r ai-news-collector ~/.openclaw/skills/ai-news-collector
```
