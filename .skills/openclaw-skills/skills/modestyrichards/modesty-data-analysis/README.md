# data-analysis - OpenClaw Skill

## Installation

```bash
openclaw skill install data-analysis
```

> 📖 **Complete setup guide**: https://skillboss.co/skill.md

### Via ClawHub
```bash
clawhub install modesty-data-analysis
```

### Manual Installation
```bash
git clone https://github.com/ModestyRichards/data-analysis.git
cp -r data-analysis ~/.openclaw/skills/data-analysis
```
