# AI Daily Briefing - OpenClaw Skill

AI-powered daily briefing and news aggregation assistant

## Installation

### Via ClawHub
```bash
clawhub install modesty-ai-daily-briefing
```

### Manual Installation
```bash
git clone https://github.com/ModestyRichards/ai-daily-briefing.git
cp -r ai-daily-briefing ~/.openclaw/skills/ai-daily-briefing
```
