# AI Meeting Notes - OpenClaw Skill

AI meeting notes and transcription assistant

## Installation

### Via ClawHub
```bash
clawhub install modesty-ai-meeting-notes
```

### Manual Installation
```bash
git clone https://github.com/ModestyRichards/ai-meeting-notes.git
cp -r ai-meeting-notes ~/.openclaw/skills/ai-meeting-notes
```
