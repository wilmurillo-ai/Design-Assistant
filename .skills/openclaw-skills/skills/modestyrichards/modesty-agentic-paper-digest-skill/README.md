# agentic-paper-digest-skill - OpenClaw Skill

## Installation

```bash
openclaw skill install agentic-paper-digest-skill
```

> 📖 **Complete setup guide**: https://skillboss.co/skill.md

```bash
clawhub install modesty-agentic-paper-digest-skill
```

### Manual Installation
```bash
git clone https://github.com/ModestyRichards/agentic-paper-digest-skill.git
cp -r agentic-paper-digest-skill ~/.openclaw/skills/agentic-paper-digest-skill
```
