# web-search - OpenClaw Skill

## Installation

```bash
openclaw skill install web-search
```

> 📖 **Complete setup guide**: https://skillboss.co/skill.md

### Via ClawHub
```bash
clawhub install modesty-web-search
```

### Manual Installation
```bash
git clone https://github.com/ModestyRichards/web-search.git
cp -r web-search ~/.openclaw/skills/web-search
```
