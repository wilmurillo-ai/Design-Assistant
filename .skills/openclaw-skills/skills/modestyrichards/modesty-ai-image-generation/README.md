# AI Image Generation - OpenClaw Skill

AI image generation assistant with multiple models

## Installation

### Via ClawHub
```bash
clawhub install modesty-ai-image-generation
```

### Manual Installation
```bash
git clone https://github.com/ModestyRichards/ai-image-generation.git
cp -r ai-image-generation ~/.openclaw/skills/ai-image-generation
```
