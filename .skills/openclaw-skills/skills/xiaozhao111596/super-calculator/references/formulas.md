## 金融计算公式

### 复利（Compound Interest）
```
F = P × (1 + r/n)^(n×t)
```
- F = 终值
- P = 本金（Principal）
- r = 年利率（Decimal，如 5% → 0.05）
- n = 年复利次数（按月复利=12，按季=4，按年=1）
- t = 期限（年）

**逆运算（求利率）：** r = n × ( (F/P)^(1/(n×t)) - 1 )

### 单利（Simple Interest）
```
I = P × r × t
F = P + I = P × (1 + r×t)
```

### 等额本息贷款月供
```
M = P × [r(1+r)^n] / [(1+r)^n - 1]
```
- M = 每月还款额
- P = 贷款本金
- r = 月利率（年利率 ÷ 12）
- n = 总月数（年限 × 12）

**总利息 = M × n - P**

### 等额本金月供（递减）
```
每月本金 = P / n
每月利息 = 剩余本金 × 月利率
每月还款 = 每月本金 + 每月利息
```

### 年化收益率（IRR近似）
```
Annual Return = (Final Value / Initial Value)^(1/years) - 1
```

### 汇率换算
```
目标货币 = 源货币 × 汇率
```

## 统计计算公式

### 均值（Mean）
```
x̄ = Σx / n
```

### 方差（Variance）
```
σ² = Σ(x - x̄)² / n      （总体方差）
s² = Σ(x - x̄)² / (n-1)   （样本方差）
```

### 标准差（Standard Deviation）
```
σ = √[Σ(x - x̄)² / n]
```

### 中位数（Median）
- 奇数个数据：中间那个
- 偶数个数据：中间两个的平均值

### 百分位数
```
第p百分位数：排序后，第 n×p/100 个值（线性插值处理）
```

## 健康指标公式

### BMI（体质指数）
```
BMI = 体重(kg) / 身高(m)²
```
| BMI范围 | 分类（中国标准） |
|---------|----------------|
| < 18.5 | 偏瘦 |
| 18.5–24 | 正常 |
| 24–28 | 偏胖 |
| ≥ 28 | 肥胖 |

### BMR（基础代谢率）

**男性：**
```
BMR = 13.7×体重(kg) + 5×身高(cm) - 6.8×年龄 + 66
```

**女性：**
```
BMR = 9.5×体重(kg) + 1.9×身高(cm) - 4.7×年龄 + 655
```

### 每日所需热量（TDEE）
```
TDEE = BMR × 活动系数
```
| 活动水平 | 系数 |
|---------|------|
| 久坐（几乎不运动） | 1.2 |
| 轻度（每周1-3天运动） | 1.375 |
| 中度（每周3-5天运动） | 1.55 |
| 重度（每周6-7天运动） | 1.725 |
| 运动员 | 1.9 |

## 方程求解公式

### 一元一次方程 ax + b = 0
```
x = -b / a   （当 a ≠ 0）
```

### 一元二次方程 ax² + bx + c = 0
```
Δ = b² - 4ac  （判别式）

Δ > 0：x = (-b ± √Δ) / 2a
Δ = 0：x = -b / 2a（重根）
Δ < 0：无实数解，有两个共轭复根
```

### 二元一次方程组
```
a1x + b1y = c1
a2x + b2y = c2

D = a1·b2 - a2·b1
Dx = c1·b2 - c2·b1
Dy = a1·c2 - a2·c1

x = Dx / D
y = Dy / D
（D ≠ 0 时有唯一解）
```
