# 情节结构与爽文套路参考

无论是文学创作还是网文创作，成熟的情节结构都能保证故事的吸引力。

## 经典文学结构

### 三幕式结构（Three-Act Structure）
最经典的故事结构，适用于大多数短篇/中篇小说。
- **第一幕：设置 (25%)**：介绍现状 → 激励事件打破现状 → 主角决定行动
- **第二幕：对抗 (50%)**：进入新世界 → 遇到盟友/敌人 → 重大转折(中点) → 困难升级
- **第三幕：结局 (25%)**：最低点(看似失败) → 最终高潮对决 → 展现新常态

### 英雄之旅（Hero's Journey）
适合冒险/奇幻题材。
1. 平凡世界 → 2. 冒险召唤 → 3. 拒绝召唤 → 4. 遇见导师 → 5. 跨越门槛
6. 考验/盟友/敌人 → 7. 接近最深洞穴 → 8. 苦难煎熬(接近死亡) → 9. 获得奖赏
10. 返回之路 → 11. 复活(最终考验) → 12. 满载而归

### 悬疑结构
1. **谜题出现**：发现尸体/事件 → 侦探接手 → 初步调查线索指向某人
2. **调查深入**：审讯嫌疑人 → 新线索推翻假设 → 第二起事件(模式浮现) → 侦探陷入危险
3. **真相揭示**：重大突破/反转 → 最终对决 → 案件解决

## 爽文情节结构与套路

### 爽文节奏公式
- **微观单章**：铺垫(前20%) → 冲突升级(中间50%) → 爆发/反转(后20%) → 钩子(最后10%)
- **中观卷级**：困境建立+获得机缘 → 小试牛刀+初次打脸 → 引来更强对手 → 连番挑战 → 卷级BOSS战
- **宏观全书**：开篇卷(废柴逆袭) → 发展卷(扬名立万) → 转折卷(绝地反击) → 高潮卷(巅峰对决)

### 十大经典爽文套路
1. **扮猪吃虎**：伪装弱者 → 被人看不起 → 关键时刻展示实力 → 众人震惊（每3-5章一次）
2. **退婚打脸**：被抛弃/羞辱 → 获得机缘实力飞涨 → 前任后悔莫及（通常作开篇引擎）
3. **拍卖会装逼**：竞价被嘲笑没钱 → 一掷千金碾压 → 众人震惊身份（每10-15章一次）
4. **越级挑战**：面对高等级必败对手 → 出人意料获胜 → 声名大噪（每5-8章一次）
5. **身份揭秘**：普通身份行事被刁难 → 真实身份曝光 → 所有人跪了（每15-20章一次）
6. **绝境突破**：陷入绝境 → 生死之间感悟突破 → 绝地反杀（每8-12章一次）
7. **打脸势利眼**：狗眼看人低 → 展示实力/财力 → 势利眼跪舔（日常穿插）
8. **故人重逢**：早期看不起主角的人 → 多年后发现主角飞黄腾达 → 震惊后悔
9. **逆转局势**：局势一边倒不利 → 主角使出底牌一己扭转 → 全场欢呼（卷终高潮必用）
10. **连环打脸**：打A → A叫来更强的B → 打B → 叫C → 大BOSS出面终极碾压（大高潮）

### 情绪曲线设计
- **低开高走**：开局低谷 → 渐入佳境 → 爽点爆发（最常用）
- **过山车式**：多个小高低谷交替 → 终极高潮（高潮章节）
- **反转式**：看似绝望 → 跌入最低点(转折点) → 绝地反击（转折章节）
