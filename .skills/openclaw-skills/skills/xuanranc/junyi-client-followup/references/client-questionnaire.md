# Client Questionnaire — 月度复盘问卷模板（客户版）

## 0-3 岁问卷

```
🍼 {孩子姓名} 月度成长小记（{YYYY年M月}）

Hi {家长称呼}，又到了每月的成长小记时间 ☺️
随便聊几句就好，不用写很多：

1️⃣ 上个月建议的活动/游戏，你们试了哪些？宝宝什么反应？
2️⃣ 宝宝这个月有什么新变化？（新动作、新词汇、新习惯都算）
3️⃣ 吃饭/睡觉/情绪方面有什么变化吗？
4️⃣ 有没有遇到什么让你困惑或担心的事？
5️⃣ 你觉得目前的节奏合适吗？太松/太紧/刚好？

💡 想到什么写什么，你的观察就是最好的记录。
```

## 3-8 岁问卷

```
🧒 {孩子姓名} 月度成长复盘（{YYYY年M月}）

Hi {家长称呼}，又到了每月的成长小复盘时间 ☺️
回复以下几个问题就好，不用写很多，几句话即可：

1️⃣ 上个月的学习/活动建议，你们试了哪些？孩子的反应怎么样？
2️⃣ 孩子这个月有什么新变化？（学习、社交、兴趣、习惯都算）
3️⃣ 学校最近有什么反馈吗？（老师评语、成绩、课堂表现等）
4️⃣ 有没有遇到什么困难或让你纠结的事？
5️⃣ 你觉得目前的节奏合适吗？太松/太紧/刚好？

💡 想到什么写什么，不用面面俱到。你的观察就是最好的数据。
```
