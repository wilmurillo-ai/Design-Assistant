# Thread Assets

Starter code and templates for threading/concurrency.

| File | Use |
|------|-----|
| `python-thread-starter.py` | Minimal Python threading + lock for shared counter. |
| `python-thread-pool.py` | Python `ThreadPoolExecutor` example (submit + as_completed). |

Add templates for other languages (Java, C#, Go, etc.) here when the user asks for them.
