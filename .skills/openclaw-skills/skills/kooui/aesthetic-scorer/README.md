# Aesthetic Scorer Skill / 照片美学评分 Skill

[![Skill Version](https://img.shields.io/badge/version-1.7.0-blue.svg)](https://clawhub.ai/skill/aesthetic-scorer)

A comprehensive image aesthetic evaluation skill that combines two powerful local models with AI-powered analysis. Perfect for photographers, content creators, and anyone looking to improve their visual content.

一款全面的图片美学评估 Skill,结合了两个强大的本地模型和 AI 驱动的分析。非常适合摄影师、内容创作者以及任何希望提升视觉内容质量的人士。

## 🔒 Privacy & Security / 隐私与安全

- **100% Local Processing**: All image processing happens on your machine / **100% 本地处理**: 所有图片处理都在您的机器上进行
- **No Data Transmission**: No images or scores are sent to external servers / **无数据传输**: 不会将图片或评分发送到外部服务器
- **No API Keys Required**: Works completely offline / **无需 API 密钥**: 完全离线工作
- **Privacy First**: Your photos never leave your device / **隐私优先**: 您的照片永远不会离开您的设备
- **Open Source Models**: Uses well-known, audited models (Improved Aesthetic Predictor & NIMA) / **开源模型**: 使用知名且经过审核的模型

## Features / 功能特性

### 🔥 Fast & Accurate Scoring (Local Models) / 快速准确的评分 (本地模型)
- **Improved Aesthetic Predictor** (dynamic 45%-70%): Ultra-fast response (<1 second) / 超快响应 (<1秒)
  - CLIP ViT-L/14 + MLP for content-level aesthetic scoring
  - Deep learning-based aesthetic scoring / 基于深度学习的美学评分
  - Real-time evaluation capability / 实时评估能力
- **NIMA - Neural Image Assessment** (dynamic 30%-55%): Detailed quality distribution analysis / 详细的质量分布分析
  - MobileNet backbone with probability distribution output
  - Provides standard deviation for dynamic weight calculation
  - Perceptual quality assessment / 感知质量评估
  - Comprehensive quality breakdown / 全面的质量分解
- **Dynamic Weight System**: Weights adjust based on NIMA's score consensus
  - High controversy (high std) → more weight to IAP (content)
  - Low controversy (low std) → balanced weights
- **Completely offline**: No network required, full privacy / 完全离线:无需网络,完全隐私
- **No API limits**: Evaluate unlimited photos / 无 API 限制:可评估无限照片

### 🎨 AI-Powered Professional Analysis / AI 驱动的专业分析
- **Detailed evaluation text**: Generated by AI based on: / 详细评价文本:基于以下内容生成
  - Weighted scores from both models / 两个模型的加权分数
  - Visual understanding of photo content / 对照片内容的视觉理解
  - Professional photography knowledge / 专业摄影知识
- **Comprehensive analysis**: / 全面分析:
  - Composition evaluation / 构图评价
  - Color analysis / 色彩分析
  - Lighting assessment / 光线评估
  - Technical quality review / 技术质量审查
- **Actionable suggestions**: / 可操作的建议:
  - Shooting techniques / 拍摄技巧
  - Post-processing tips / 后期处理建议
  - Composition optimization / 构图优化

## Score Interpretation Guide / 评分解读指南

### "从夯到拉" Rating System / "从夯到拉" 评分系统 ⭐

| Score Range | 等级 / Level | Description / 描述 |
|-------------|-------------|-------------------|
| 9.0-10.0 | **夯 (Hāng)** | 好到没话说,顶级水平 / Exceptional, top-tier, perfect |
| 8.0-8.9 | **顶级** | 极好,专业水准 / Excellent, professional level |
| 7.0-7.9 | **人上人** | 很好,超越常人 / Very good, above average, outstanding |
| 6.0-6.9 | **NPC** | 不起眼,普普通通 / Average, unremarkable, plain |
| 0.0-5.9 | **拉完了** | 差到没法再差 / Terrible, needs major improvement |

**等级说明**:
- **夯**: 好到没话说,完美之作
- **顶级**: 极好,专业水准
- **人上人**: 很好,超越常人
- **NPC**: 不起眼,普普通通 (源自游戏术语,指不受玩家控制的角色)
- **拉完了**: 差到没法再差,需要大幅改进

### Traditional Rating / 传统评分

| Score Range | Level | Description / 描述 |
|-------------|-------|-------------|
| 9.0-10.0 | 优秀 | Exceptional quality, professional level / 卓越质量,专业水平 |
| 8.0-8.9 | 很好 | High quality with minor improvements needed / 高质量,仅需小幅改进 |
| 7.0-7.9 | 良好 | Solid quality, above average / 扎实质量,高于平均水平 |
| 6.0-6.9 | 一般 | Average quality, noticeable room for improvement / 平均质量,有明显改进空间 |
| 4.0-5.9 | 较差 | Below average, significant improvements needed / 低于平均水平,需要大幅改进 |
| 0.0-3.9 | 很差 | Poor quality, substantial improvements needed / 质量差,需要大幅改进 |

## Advanced Usage / 高级用法

### Batch Processing / 批量处理

```python
import os
from scripts.score_improved_predictor import predict_aesthetic

images_dir = "path/to/images"  # 图片路径 / Image directory
for filename in os.listdir(images_dir):
    if filename.endswith(('.jpg', '.png')):
        image_path = os.path.join(images_dir, filename)
        result = predict_aesthetic(image_path)
        print(f"{filename}: {result['score']:.2f}/10")
```

class AestheticWorkflow:

        pass
