# Skill: diansubao-daily-report

**功能：** 工商业储能电站运营日报生成与分析

**创建时间：** 2026-03-14  
**版本：** v1.0.0

---

## 功能说明

本技能用于：
1. 获取电站运营数据
2. 生成每日运营日报
3. 分析功率曲线与电价
4. 对比前一天数据
5. 识别运营问题

---

## 使用方法

### 在 OpenClaw 中调用

```
sessions_spawn(
  runtime="subagent",
  task="生成昨天的储能电站运营日报，包括功率曲线分析和电价分析",
  label="daily-report-2026-03-13"
)
```

### 手动调用 MCP 工具

**⚠️ 重要：日期传入规则**

| 数据类型 | 要获取 | 传入参数 | 说明 |
|---------|--------|---------|------|
| 统计类 | 3 月 13 日 | `date=2026-03-14` | 目标日期 +1 天 |
| 秒级类 | 3 月 13 日 | `start_date="2026-03-13 00:00:00"`<br>`end_date="2026-03-14 00:00:00"` | 目标日期 0 点 → 次日 0 点 |

```bash
# 获取电站列表
mcporter call diansubao.get_stations page=1 rows=100 --config "..."

# 获取 3 月 13 日统计数据（传入目标日期 +1 天）
mcporter call diansubao.get_station_daily_data station_id=2 date=2026-03-14 --config "..."
mcporter call diansubao.get_grid_price station_id=2 grid_id=2 date=2026-03-14 --config "..."

# 获取 3 月 13 日秒级功率曲线（传入目标日期 0 点 → 次日 0 点）
mcporter call diansubao.get_grid_second_data \
  station_id=2 grid_id=2 \
  start_date="2026-03-13 00:00:00" \
  end_date="2026-03-14 00:00:00" --config "..."
```

## 日报内容结构

### 1. 电站概况
- 电站名称、位置
- 装机容量、电池类型
- 运行状态、在线状态
- 并网点架构（变压器 - 储能柜关系）

### 2. 核心指标
- 日充电量 (kWh)
- 日放电量 (kWh)
- 系统效率 (%)
- 日收益 (元)
- 负荷用电量 (kWh)
- SOC/SOH 状态

### 3. 峰谷套利效率分析 ⚡
**充电效率（谷电 00:00-08:00）**
- 谷电时段充电量
- 理论最大充电量
- 充电完成率
- 电价成本

**放电效率（峰电 08:00-11:00 + 13:00-17:00）**
- 峰电时段放电量
- 理论最大放电量
- 放电完成率
- 电价收益

**套利收益**
- 实际收益
- 理论收益
- 套利效率
- 收益损失

### 4. 并网点约束分析
**充电防过载**
- 并网点额定功率
- 实际最大充电功率
- 过载次数与时长

**放电防逆流**
- 放电功率 vs 负荷功率
- 逆流风险识别

### 5. 储能柜状态分析 🔋
**健康度概览**
- 🟢 健康：SOC 20-90%, SOH≥90%
- 🟡 注意：SOC<20% 或>90%, SOH 80-90%
- 🟠 警告：SOH 70-80%
- 🔴 严重：SOC<10%, SOH<70%
- ⚫ 离线：无数据

**严重问题储能柜**
- 储能柜编号、并网点
- SOC、SOH、SOE
- 功率状态
- 建议措施

**离线储能柜**
- 储能柜编号、并网点
- 离线时长
- 可能原因

### 6. 功率曲线分析
- 充电时段分布（是否集中在谷电）
- 放电时段分布（是否集中在峰电）
- 峰值功率时间
- 低谷功率时间

### 7. 同比分析
- 与前一天充电量对比
- 与前一天放电量对比
- 与前一天收益对比
- 套利效率变化

### 8. 问题识别与告警
- 🔴 P0：充电过载/设备故障/储能柜严重异常
- 🟠 P1：峰谷策略未执行/储能柜离线
- 🟡 P2：效率偏低/优化建议
