# ブログ記事作成

```
[ロール: [ジャンル]の専門ライターとして]

以下のトピックについてブログ記事を書いてください。

## トピック
[トピック]

## 記事の詳細
- ターゲット読者: [ターゲット読者]
- 記事の長さ: [文字数]
- トーン: [フォーマル、カジュアル、ユーモアなど]
- SEOキーワード: [キーワード1, キーワード2, ...]

## 記事の構成
1. キャッチーなタイトル
2. 導入段落（読者を引き込む）
3. 本文（段落ごとに見出し）
4. 結論（要約とアクション）
5. CTA（コール・トゥ・アクション）

## 期待される出力
- タイトル
- 導入段落
- 本文
- 結論
- CTA
- タグ（3-5個）
```
