# AIプロンプトテンプレート

高品質なAIプロンプトのテンプレート集。

## 機能
- コードレビュー (2種類）
- コンテンツ作成 (3種類）
- 分析・調査 (3種類）
- 教育・チュートリアル (2種類）
- 問題解決 (2種類）

## 使い方

### インストール
```bash
clawhub install ai-prompt-templates
```

### テンプレートの使用
```bash
# テンプレートをコピー
cp -r ~/.openclaw/workspace/skills/ai-prompt-templates/prompts/* \
      ~/.openclaw/workspace/my-prompts/

# テンプレートを開いて編集
# 必要に応じて[プレースホルダー]を置き換えて使用
```

## プロンプトエンジニアリングのベストプラクティス

### 1. 明確なロールを設定する
悪い例: "このコードをレビューして"
良い例: "[ロール: シニアソフトウェアエンジニアとして] 以下のコードをレビューしてください"

### 2. 具体的な指示を与える
悪い例: "良い記事を書いて"
良い例: "1000文字の技術記事を書いてください。ターゲット読者は初心者開発者です。トーンはカジュアルで、例を含めてください"

### 3. 期待される出力を明確にする
悪い例: "分析して"
良い例: "分析し、以下を含むレポートを作成してください：サマリー、主要な発見、推奨されるアクション"

### 4. コンテキストを提供する
悪い例: "修正して"
良い例: "以下のコードには、ユーザー入力が直接SQLクエリに使用されるバグがあります。SQLインジェクションを防ぐ方法で修正してください"

詳細は SKILL.md を参照してください。
