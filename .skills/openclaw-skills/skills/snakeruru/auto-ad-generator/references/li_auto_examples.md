# Li Auto (理想汽车) Advertisement Analysis

## Brand Visual Identity

### Color System
Li Auto uses sophisticated gradient palettes that evoke premium quality:

**Primary Palette**: Deep Blue → Purple
- Hex: `#1a237e` → `#7c4dff`
- Mood: Premium, tech-forward, trustworthy
- Usage: Main product launches, flagship models

**Secondary Palette**: Warm Orange → Pink
- Hex: `#ff6b35` → `#f7931e`
- Mood: Energetic, youthful, aspirational
- Usage: Lifestyle-focused campaigns

**Accent Palette**: Teal → Cyan
- Hex: `#00897b` → `#00bcd4`
- Mood: Clean, modern, sustainable
- Usage: EV/technology messaging

### Typography Patterns

**Headline Treatment**:
- Font: Bold, geometric sans-serif (Chinese: 思源黑体 Bold)
- Size: Large, dominant (40-60% of top third)
- Color: White or very light against dark gradients
- Style: Clean, no effects, relies on size for impact

**Subtitle Treatment**:
- Font: Light weight, same family
- Size: 40-50% of headline
- Tracking: Generous letter-spacing
- Position: Directly below headline

**Slogan Placement**:
- Font: Medium weight, often italicized
- Size: Between headline and subtitle
- Position: Bottom third, above brand elements

### Composition Principles

**Rule of Thirds Application**:
```
┌─────────────────┐ ← Headline (top third)
│    理想i6       │
│  新形态纯电     │
├─────────────────┤
│                 │ ← Car (middle third, lower)
│    [CAR]        │
│                 │
├─────────────────┤
│  代言人/Logo    │ ← Supporting elements (bottom)
│   品牌口号      │
└─────────────────┘
```

**Car Placement**:
- Angle: 3/4 front view (most common)
- Position: Centered, slightly low (60-70% down frame)
- Scale: Occupies 50-70% of horizontal space
- Lighting: Soft key light from top-left, rim light on edges

**Celebrity Integration**:
- Position: Overlapping car or offset to side
- Scale: 40-60% of car height
- Treatment: Natural lighting to match car
- Purpose: Human connection, aspirational lifestyle

### Key Visual Elements

**Background Effects**:
- Smooth gradient (no banding)
- Subtle light rays from top
- Atmospheric haze/depth
- No texture or patterns

**Car Treatment**:
- Clean, showroom quality
- Reflections enhanced but natural
- Shadows grounded and soft
- Color accuracy maintained

**Logo Placement**:
- Bottom left or bottom center
- 100-150px width
- White or single color
- Subtle, doesn't compete with car

## Adaptation Guidelines

### For Different Car Types

**Sedan/Sports Car**:
- Use warm gradients (orange, red)
- Dynamic angles (3/4 rear)
- Bold, condensed typography
- Speed/performance messaging

**SUV/Crossover**:
- Use premium gradients (blue, purple)
- Front 3/4 view
- Bold but approachable typography
- Capability/lifestyle messaging

**MPV/Minivan**:
- Use clean gradients (teal, white)
- Side profile or interior focus
- Light, friendly typography
- Space/family messaging

### For Different Price Points

**Luxury (40w+)**:
- Dark, rich gradients
- Minimal text, large negative space
- Subtle celebrity or no person
- Focus on craftsmanship details

**Premium (20-40w)**:
- Balanced gradients
- Clear value proposition text
- Relatable celebrity or lifestyle scene
- Feature highlights

**Mass Market (<20w)**:
- Bright, energetic gradients
- Promotional text, clear pricing
- Diverse, approachable talent
- Practical benefits
