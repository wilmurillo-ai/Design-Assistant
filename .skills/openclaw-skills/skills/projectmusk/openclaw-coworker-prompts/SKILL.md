# OpenClaw AI Coworker Prompt Pack

A collection of 100+ precise prompts for building productive AI coworkers with OpenClaw.

## Quick Install

1. In OpenClaw: `clawhub install openclaw-coworker-prompts`
2. Create a new agent using one of the prompts from `prompts/` directory.
3. Assign relevant tools (web, code, exec) for the agent's role.

## Contents

- `prompts_full.json` — All 100 prompts with categories
- `samples/` — Ready-to-use agent definitions (YAML/JSON)
- `README.md` — Detailed usage guide

## Sample Agent

Create an agent with this prompt:

```
You are a senior software engineer. Review pull requests for bugs, security issues, and best practices. Comment line-by-line with constructive feedback.
```

Enable tools: `exec`, `web_search`, `read`, `edit`.
