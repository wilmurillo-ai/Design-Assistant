# Handoff Template / 交接模板

```md
# Task
<short task name>

## Goal
- What outcome is required?
- 这次交接的目标是什么？

## Done
- What has already been completed?
- 已经完成了什么？

## Key context
- Critical facts, decisions, constraints
- Links or paths to source material
- 关键事实、决策、限制条件
- 相关资料的链接或路径

## Remaining work
- [ ] Next task 1
- [ ] Next task 2
- [ ] 下一步待办 1
- [ ] 下一步待办 2

## Risks / caveats
- What could go wrong?
- What is uncertain?
- 有哪些风险？
- 哪些地方还不确定？

## Next agent
- Agent: <name>
- Expected output: <deliverable>
- 接手 agent：<名称>
- 预期产出：<交付物>

## Files to read
- path/to/file1
- path/to/file2
- 需要优先阅读的文件路径
```

## Usage notes / 使用建议

Keep this short enough to scan in under a minute.
If there is a large artifact, save the artifact separately and summarize it here.
Use bullet points instead of long paragraphs where possible.

尽量控制在 1 分钟内能扫读完。
如果有较大的产物，单独存文件，并在这里做摘要。
尽量使用项目符号，少写大段长文。
