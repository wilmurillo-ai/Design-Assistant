# Sendivent Skills

Agent skills for the [Sendivent](https://sendivent.com) multi-channel notification API.

Gives your AI coding agent full knowledge of the Sendivent API — authentication, endpoints, code patterns, and best practices — so it can write production-ready integration code without you explaining anything.

## Install

```bash
npx skills add appitudeio/sendivent-skills
```

## What's included

| Skill | Description |
|-------|-------------|
| **Sendivent API** | REST endpoints, authentication, error codes, channels, template variables, webhooks |

### Reference docs

| File | Description |
|------|-------------|
| `references/sdk-guide.md` | Node.js SDK examples with `@appitude/sendivent` |
| `references/api-guide.md` | Endpoint reference with request/response shapes |

## Supported agents

Works with Claude Code, Cursor, Copilot, Windsurf, Codex, Replit, and [40+ more](https://github.com/vercel-labs/skills).
