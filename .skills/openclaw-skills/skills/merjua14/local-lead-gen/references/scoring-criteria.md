# Website Scoring Criteria (0-100)

## Score Components

### SSL Certificate (0-20 points)
- HTTPS enabled with valid cert: 20
- HTTPS with expired/self-signed cert: 10
- No HTTPS (HTTP only): 0

### Mobile Responsiveness (0-20 points)
- Has viewport meta tag + responsive design: 20
- Has viewport meta but poor mobile layout: 10
- No viewport meta tag: 0

### Page Load Speed (0-20 points)
- Loads in <3 seconds: 20
- Loads in 3-6 seconds: 10
- Loads in >6 seconds or times out: 0

### Modern Design Signals (0-20 points)
- Modern CSS framework (Tailwind, Bootstrap 5+): 20
- Older but functional design: 10
- Table-based layout, Flash, or pre-2015 design patterns: 0
- Indicators: check for viewport, flexbox/grid, modern font loading

### Content & Functionality (0-20 points)
- Working links, images load, contact info visible: 20
- Some broken elements but mostly functional: 10
- Multiple broken links, missing images, dead forms: 0

## Interpretation
- **80-100**: Good site — skip
- **50-79**: Mediocre — optional outreach
- **20-49**: Bad — strong outreach candidate
- **0-19**: Terrible — highest priority outreach

## Red Flags (auto-qualify for outreach)
- Site returns 404/500 errors
- "Under construction" or parked domain
- No contact information anywhere
- Copyright date 3+ years old
- Flash or Java applets detected
