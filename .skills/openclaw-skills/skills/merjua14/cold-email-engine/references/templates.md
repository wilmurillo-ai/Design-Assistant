# Proven Cold Email Templates

## Template 1: The Website Critique (Web Design / Marketing)
```
Subject: Quick question about {company}'s website

Hi {first_name},

I was looking at {company}'s website and noticed {pain_point}.

I help businesses in {city} fix exactly this. Usually takes about a week and the results speak for themselves.

Worth a quick 10-min chat?

{sender_name}
```

## Template 2: The Competitor Angle (Any B2B)
```
Subject: How {competitor} is getting ahead

Hi {first_name},

I noticed {competitor} recently {specific_action}. Companies in {industry} that don't adapt tend to fall behind fast.

We help businesses like {company} stay competitive with {service}. Happy to share what we're seeing work.

Open to a quick call this week?

{sender_name}
```

## Template 3: The Data Hook (Analytics / SaaS)
```
Subject: {company} is leaving money on the table

Hi {first_name},

I ran a quick analysis on {company}'s {website/social/presence} and found 3 things costing you customers:

1. {issue_1}
2. {issue_2}
3. {issue_3}

I put together a free report with fixes. Want me to send it over?

{sender_name}
```

## Template 4: The Referral Ask (Warm-ish)
```
Subject: {mutual_connection} suggested I reach out

Hi {first_name},

{mutual_connection} mentioned you might be looking for help with {pain_point}. We just helped {similar_company} solve the same problem and they saw {result}.

Would love to share how we did it. Free to chat Thursday or Friday?

{sender_name}
```

## Template 5: The Break-Up (Last Follow-Up)
```
Subject: Closing the loop

Hi {first_name},

I've reached out a couple times about helping {company} with {service}. I know you're busy, so I won't keep following up.

If it ever becomes a priority, I'm here. Just reply to this email.

Best,
{sender_name}
```

## Subject Line Formulas (Ranked by Open Rate)
1. "{company} — quick question" (45-55%)
2. "Noticed something about {company}" (40-50%)
3. "{first_name}, quick thought" (35-45%)
4. "Idea for {company}" (30-40%)
5. "{mutual_connection} suggested I reach out" (50-60% when genuine)
