# Proven Cold Email Templates

## Template 1: Bad Website Pitch (35-45% open rate)
```
Subject: Quick question about {business_name}'s website

Hi {first_name},

I was looking at {business_name}'s website and noticed {specific_issue}.

I help local businesses in {city} fix exactly this — usually takes about a week and costs less than you'd think.

Would you be open to a quick 10-minute call to see if it's worth fixing?

Best,
{sender_name}
```

## Template 2: Automation Services (30-40% open rate)
```
Subject: Saving {business_name} 10+ hours per week

Hi {first_name},

I noticed {business_name} might benefit from some automation — specifically around {specific_issue}.

I build custom AI-powered workflows that handle repetitive tasks: lead generation, email follow-ups, data entry, client onboarding.

Most of my clients save 10-15 hours per week. Happy to show you how in a quick call.

{sender_name}
```

## Template 3: Follow-Up #1 (Day 3)
```
Subject: Re: Quick question about {business_name}'s website

Hi {first_name},

Just following up on my note from a few days ago. I found one more thing that might be worth looking at — {specific_issue}.

Happy to put together a free 2-minute audit if you're interested. No strings attached.

Best,
{sender_name}
```

## Template 4: Break-Up Email (Day 7)
```
Subject: Last note from me

Hi {first_name},

I don't want to be a pest — this will be my last email.

If you ever want to chat about modernizing {business_name}'s online presence, I'm here. Just reply to this email.

Wishing you the best,
{sender_name}
```

## Subject Line Performance
| Subject | Open Rate |
|---------|-----------|
| "Quick question about {business_name}" | 35-45% |
| "Noticed something on your website" | 30-40% |
| "Saving {business_name} 10+ hours/week" | 25-35% |
| "{first_name}, your website might be losing customers" | 20-30% |
| "Last note from me" | 40-50% (break-up emails perform well) |
