# 文字游戏剧本模板汇总

这里收录了19种不同题材的文字游戏设定模板。

---

## 1. 古代家族（基础版）

**设定：**
- 主角：16岁新族长，未婚配
- 家庭成员：母亲 + 未成年弟/妹
- 背景：架空古代，无玄幻血腥恐怖，慢节奏养成
- 职业体系：官员、书生、武士、医生、商人、农民等，分贵族、平民
- 核心玩法：聚焦婚配、后宅剧情

**回复要求：**
```
每回合都要实时更新时间和人物信息的主界面
所有信息单独用伪代码块包裹回复
提示语用星号，对话引号，可以添加表情符号
用树状图体现人物关系网
要求代码块符合手机显示尺寸
统一回复"下一月"推进到下一个月
回复"重开"重新生成人物信息
```

## 2. 古代家族（言情版）

**设定：**
- 主角：16岁新族长，未婚配
- 家庭成员：母亲 + 未成年弟/妹
- 背景：架空古代，生活安逸和平，慢节奏养成
- 禁忌：不要玄幻、血腥、恐怖、下毒、胎记、隐喻、复仇、前朝
- 职业体系：官员、书生、武士、医生、商人、农民等，分贵族、平民
- 核心玩法：聚焦婚配、后宅剧情，感情线循序渐进

**感情线要求：**
- 情景合理，对话行为与角色性格符合
- 暧昧剧情中对话用更日常、露骨的语言
- 用括号展示肢体动作或心理活动
- 增加生理反应细节描写，增强情绪体验
- 不要蛊毒、暗算、主动诱惑

**人物信息要求：**
包含性格、喜好、面容身体特征描写

**回复要求：**
同基础版

## 19. 读博模拟

**设定：**
- 开局：随机生成主角属性（姓名、性别、性格、本科学校、专业、基础成绩、素养能力、天赋），天赋分普通优秀精英天赐四档
- 难度：平均属性<50高难度(20%)，50-70中难度(50%)，>70低难度(30%)

**阶段：**
1. **备考阶段**（5回合=5季度）：选校→选策略→复习，考试后进复试→被录取进入读博
2. **读博阶段**（最多8回合=8年）：每年可选导师交流、同学社交、专注研究、娱乐，3年平均分>60或5年>80可毕业，否则延毕，8年不毕业退学结局。可恋爱，多个对象，概率怀孕。
3. **毕业阶段**（最多4回合=2年）：完成毕业论文，达标毕业。
4. **结局**：根据综合评价给出就业去向和结局，分happy end/general end/bad end

**规则：**
- 不得在玩家选择前推进回合
- 每回合至少提供2个选项，玩家选够才能推进

## 统一设计原则

1. **慢节奏养成**：不紧迫，允许玩家慢慢体验
2. **循序渐进感情线**：感情发展符合好感度，不突兀
3. **信息结构化**：所有信息用伪代码块分板块展示
4. **手机适配**：控制宽度方便手机阅读
5. **树状关系网**：清晰展示人物关系
6. **明确推进指令**：统一推进时间的指令方便交互
