# 战锤40K：死亡守望远征

## 背景设定
你是一名死亡守望星际战士，来自暗黑天使战团，已经在死亡守望服役了三个世纪，消灭过无数异形。你从长达百年的静滞休眠中醒来，战斗驳船"神圣之锤号"刚刚脱离亚空间，抵达了卡利班四号死亡世界的轨道。

## 任务目标
泰伦虫族正在挖掘星球深处的太空死灵墓穴，一旦他们唤醒死灵王朝，整个星区都会被毁灭。你必须立刻前往地表，阻止泰伦的计划，必要时可以动用灭绝令级别的武器。

## 初始状态
你的动力甲已经完成充能，肩甲上镶嵌着十几个战团的荣誉徽记，代表着你曾经和他们并肩作战的经历。武器库为你准备了三套装备：
1. 爆矢步枪 + 动力剑 + 破片手雷（标准作战配置）
2. 重型喷火器 + 等离子手枪 + 热熔炸弹（反虫群配置）
3. 狙击步枪 + 隐形斗篷 + 致命毒素（暗杀配置）

## 初始场景
医疗舱的舱门缓缓打开，刺鼻的臭氧和神圣熏香的味道扑面而来。机仆伺服器发出单调的嗡鸣声，舰桥的通讯频道传来智库馆长的灵能讯息："兄弟，帝皇需要你。整个星区的命运都掌握在你的手中。为了帝皇！"

你现在站在武器库的门口，走廊尽头是空降舱发射平台，你可以：
- 选择装备，立刻前往地表
- 先去舰桥了解更多战术情报
- 先去礼拜堂祈祷，获得帝皇的祝福
