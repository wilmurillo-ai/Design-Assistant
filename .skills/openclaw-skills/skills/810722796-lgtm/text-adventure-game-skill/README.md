# 文字冒险游戏大师技能

一个高度智能化的文字冒险游戏生成系统，支持自定义剧本、完全自由互动、精美结构化排版。

## ✨ 功能特色

### 🎭 智能剧本解析
- 支持任意格式的剧本输入，自动提取人物、场景、道具、剧情分支
- 无需固定格式，自然语言描述即可生成完整游戏
- 支持多章节、多结局、非线性剧情设计

### 🔄 完全自由互动
- 玩家可以输入任何行动，系统动态生成剧情回应
- 没有预设选项限制，真正实现开放式冒险
- 内置剧情生成引擎，支持无限扩展的剧情发展

### 🎨 精美排版输出
每次回复都采用结构化精美排版，包含：
- 🏷️ 清晰的场景标题标识
- ━━━━━━ 分隔线区分不同功能区域
- 剧情内容分段展示，重点内容高亮
- 物品、状态使用 emoji 列表展示，一目了然
- 选项带编号，支持快捷选择
- 战斗、对话等特殊场景有专属排版样式

### 💾 完整游戏系统
- **进度管理**：自动存档/读档，支持多存档位
- **属性系统**：生命值、体力、经验、等级等完整RPG属性
- **物品系统**：背包管理，支持物品的获得、使用、丢弃
- **任务系统**：支持主线、支线任务追踪
- **NPC系统**：好感度、对话树、交易功能
- **战斗系统**：回合制战斗，支持武器、技能、道具使用

## 🚀 使用方法

### 开始新游戏
#### 方式1：自定义剧本
```
创建文字游戏，剧本：
# 你的游戏标题
你的剧情描述、背景设定、初始场景等等...
```

#### 方式2：快速生成
```
开始文字游戏，主题是：仙侠世界冒险
```
系统会自动生成对应主题的完整游戏。

### 游戏中操作
| 命令 | 功能 |
|------|------|
| 直接输入行动/对话 | 执行你的选择 |
| 输入数字1/2/3... | 快速选择对应选项 |
| 存档 | 保存当前游戏进度 |
| 读档 | 读取最近的存档 |
| 状态/属性/背包 | 查看当前角色状态和物品 |
| 任务 | 查看当前任务列表 |
| 重新开始 | 重置游戏回到初始状态 |
| 帮助 | 查看操作说明 |
| 退出 | 结束当前游戏 |

## 🔧 扩展开发

### 接入大模型实现动态剧情
修改 `game_engine.py` 中的 `_generate_story_response` 方法，接入LLM API即可实现完全动态的剧情生成：

```python
def _generate_story_response(self, action: str) -> str:
    # 接入OpenAI API示例
    import openai
    response = openai.ChatCompletion.create(
        model="gpt-3.5-turbo",
        messages=[
            {"role": "system", "content": f"你是文字游戏GM，当前剧情：{self.game_state['current_scene']}，玩家状态：{self.game_state['player']}"},
            {"role": "user", "content": f"玩家选择了：{action}，请继续生成剧情，要求符合世界观设定，有合理的剧情发展"}
        ]
    )
    return response.choices[0].message.content
```

### 自定义排版样式
修改 `game_engine.py` 中的 `self.style` 字典，可以自定义所有排版符号和样式。

### 添加游戏机制
系统已经预留了属性、物品、任务、NPC、战斗等系统的扩展接口，可以根据需要添加更多游戏机制。
