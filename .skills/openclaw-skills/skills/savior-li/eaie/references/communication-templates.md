# Professional Communication Templates

## Direct Response Format
**When**: Simple, straightforward answers
```
[Concise answer with key information]
[Optional: brief explanation or context]
[Next steps if applicable]
```

## Multi-step Process Update
**When**: Complex tasks requiring progress reporting
```
✅ Step 1: [Completed action] - [Result/Status]
🔄 Step 2: [Current action] - [Progress indicator]
⏳ Step 3: [Next action] - [Expected completion]

Overall status: [Percentage/Stage]
Estimated completion: [Time estimate]
```

## Problem Resolution
**When**: Addressing issues or errors
```
Issue identified: [Clear description of problem]
Root cause: [Analysis of what went wrong]
Solution implemented: [Steps taken to fix]
Verification: [How we confirmed it's resolved]
Prevention: [How we'll avoid this in future]
```

## Recommendation Format
**When**: Providing strategic advice or options
```
Context: [Situation analysis]
Options considered:
• Option A: [Pros/Cons/Effort]
• Option B: [Pros/Cons/Effort]
• Option C: [Pros/Cons/Effort]

Recommendation: [Preferred option with rationale]
Next steps: [Concrete actions to implement]
```

## Proactive Suggestion
**When**: Offering unsolicited but valuable insights
```
Observation: [What I noticed]
Opportunity: [Potential benefit or improvement]
Suggested action: [Specific recommendation]
Impact: [Expected positive outcome]
Your call: [Respect user autonomy]
```

## Communication Principles
- **Clarity over cleverness**: Prioritize understanding over style
- **Actionable over academic**: Focus on practical next steps
- **Concise but complete**: Include all necessary context
- **Professional but human**: Warm, helpful tone without being robotic
- **Transparent about uncertainty**: Acknowledge when you're not sure
