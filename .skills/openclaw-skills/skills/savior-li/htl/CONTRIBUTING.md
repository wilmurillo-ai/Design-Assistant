# Human Traits Learning Skill - 贡献指南

## 🤝 欢迎贡献！

感谢您对 Human Traits Learning Skill 的关注！这是一个**纯文档技能**，专注于提供高质量的指令性内容和框架。

## 📝 贡献类型

### 文档改进
- 改进现有框架和指南的清晰度
- 添加新的企业培训方法论
- 增强双向成长框架的内容
- 改善多语言支持和本地化

### 内容质量
- 确保所有内容符合专业标准
- 维护安全和隐私承诺的一致性
- 保持纯文档性质（无代码、无可执行内容）
- 遵循企业卓越发展路径的核心原则

## 🚫 不接受的贡献

- **任何可执行代码**: 此技能为纯文档，不接受代码提交
- **API 实现**: 不接受任何 API 或函数实现
- **自动化脚本**: 不接受任何自动化或批处理脚本
- **外部依赖**: 不接受需要外部工具或服务的内容

## 🔒 安全与隐私要求

所有贡献必须：
- 保持纯文档性质
- 不包含任何可执行内容
- 维护明确的用户同意要求
- 尊重隐私边界和数据保护
- 避免任何自动数据收集的暗示

## 🌍 多语言支持

我们欢迎多语言翻译贡献：
- 确保文化适应性和专业准确性
- 保持核心框架的一致性
- 遵循本地化最佳实践
- 维护专业术语的准确性

## 📋 提交流程

1. **Fork 仓库**
2. **创建特性分支**: `git checkout -b feature/improvement-name`
3. **进行修改**: 仅限文档和框架内容
4. **提交更改**: `git commit -m "描述您的改进"`
5. **推送分支**: `git push origin feature/improvement-name`
6. **创建 Pull Request**
