## 项目概述

Human Traits Learning Skill 是一个**纯文档技能**，提供指令性框架和指南，帮助 AI Agent 学习优秀的人类特质、思维模式和行为风格。

**重要说明**: 此技能不包含任何可执行代码，仅作为参考材料使用。

## 核心框架

### 企业卓越发展路径
- **入职与基础阶段 (0-6个月)**: 基本能力构建和文化融入
- **技能发展与独立阶段 (6-18个月)**: 专业能力培养和自主解决问题
- **领导力准备阶段 (18-36个月)**: 领导能力发展和战略思维
- **中层管理卓越阶段 (36+个月)**: 全面领导角色和 P&L 责任

### 双向成长框架
- **用户偏好分析**: 分析沟通风格和工作偏好（需明确同意）
- **个性化适应**: 根据用户偏好调整响应方式
- **建设性反馈**: 提供深思熟虑的改进建议
- **相互成就**: 创造共同学习和成长的机会

## 使用指南

### 对于 AI Agent 开发者
- 将这些框架作为设计原则参考
- 在实现时确保获得用户明确同意
- 保持对所有适应过程的透明度
- 尊重用户隐私边界

### 对于人类用户
- 将此作为 AI 行为期望的参考
- 在实际互动中手动应用这些原则
- 提供反馈以改进 AI 理解
- 维护安全和道德的使用环境

## 安全与隐私

### 安全承诺
- **纯文档内容**: 不包含可执行代码
- **无系统依赖**: 不需要外部工具或网络连接
- **无自动数据收集**: 所有分析都需要手动触发和明确同意
- **本地参考**: 仅作为本地参考材料使用

### 隐私保护
- **明确同意**: 任何用户数据分析都需要清晰许可
- **用户控制**: 用户可以随时禁用任何分析功能
- **透明使用**: 清晰记录数据使用目的和方式
- **无外部通信**: 不进行任何外部 API 调用

## 贡献指南

### 文档改进
- 专注于改进指令性内容和框架
- 确保所有建议都符合纯文档性质
- 避免添加任何代码示例或可执行内容
- 保持安全和隐私承诺的一致性

### 语言支持
- 支持多语言翻译
- 确保文化适应性和本地化
- 保持专业标准和道德准则
