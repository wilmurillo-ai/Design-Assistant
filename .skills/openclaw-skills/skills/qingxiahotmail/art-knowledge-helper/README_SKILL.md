## 这是什么

一个专为**艺术学习者**打造的书籍知识库管理助手技能。
帮你：整理、分类、归档、同步你的艺术类电子书收藏。

## 核心能力

- 📚 **书籍扫描**：自动识别 Downloads 目录中的艺术类书籍（PDF/EPUB）
- 🔍 **缺口分析**：对照现有库存，找出缺失的门类
- 📦 **智能归档**：自动归类到正确的分类目录
- 🔄 **云盘同步**：保持本地与百度网盘完全一致
- 📊 **库存统计**：实时了解知识库规模和结构

## 使用方式

### 日常归档（推荐流程）

**第一步：扫描下载目录**
```
python scripts/scan_downloads.py
```
找出最近 30 天下载的艺术类书籍，对照已有库存标记"新增候选"。

**第二步：归档新书**
```
python scripts/archive_books.py
```
自动将新书复制到对应的分类目录。

**第三步：同步到云盘**
```
python scripts/sync_to_baidu.py
```
保持本地与百度网盘同步。

**第四步：验证同步**
```
python scripts/verify_sync.py
```
确认两边完全一致。

### 查看库存统计
```
python scripts/kb_stats.py
```

## 分类原则

一本书只存在于**一个分类目录**中。
如果难以判断，按以下优先级：

1. **技法优先**：同类书中，以技法教程为主
2. **作者优先**：同一艺术家，优先放在"各国艺术"
3. **时代优先**：艺术史以时代为子分类（印象派、现代艺术等）

## 云盘使用说明

朋友拿到知识库后，推荐这样用：

1. **下载到本地**：将分享的文件夹下载到 `E:\艺术知识库\`
2. **建立自己的同步备份**：建议开启百度网盘同步到自己的云盘
3. **保持定期归档**：每次下载新书，运行一次归档脚本

## 疑难解答

**Q：书籍找不到对应分类怎么办？**
A：放在最接近的分类即可，知识库没有"错误分类"，只有"暂时分类"。

**Q：百度网盘路径不同怎么办？**
A：修改 scripts 中的 BD 变量为你的实际路径。

**Q：脚本报编码错误怎么办？**
A：在 PowerShell 中运行 `chcp 65001` 切换 UTF-8 模式后再执行脚本。
