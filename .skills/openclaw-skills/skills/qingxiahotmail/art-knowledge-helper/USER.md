## 知识库路径

> ⚠️ **路径统一在 `config.json` 中配置，不要在这里改。**
>
> 首次使用请打开 `config.json`，按以下格式填写：
>
> ```json
> {
>   "knowledge_base_path": "D:\\艺术知识库",
>   "baidu_path": "D:\\BaiduSyncdisk\\小艺\\艺术知识库",
>   "downloads_path": "C:\\Users\\你的用户名\\Downloads"
> }
> ```
>
> 三个路径均可自由填写，所有脚本自动读取，无需再改其他地方。
