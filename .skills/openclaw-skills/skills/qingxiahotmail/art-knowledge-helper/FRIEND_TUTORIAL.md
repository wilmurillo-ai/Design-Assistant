# 🎨 小艺艺术知识库助手 · 朋友使用教程

> _「愿学习之路，文艺相随。」_
> 版本：v1.0 | 2026-04-22

---

## 这是什么？

这是一套帮你**整理艺术类电子书**的工具。

想象一下：你下载了几百本 PDF，乱七八糟堆在 Downloads 里，找书全靠记忆。
这个小助手可以帮你：

- ✅ 自动找出哪些是艺术类书籍
- ✅ 自动分类到对应的文件夹
- ✅ 同步到百度网盘备份

---

## 第一步：准备工作

### 1.1 你需要一台 Windows 电脑

推荐 Windows 10 或 Windows 11。

### 1.2 安装 Python

打开浏览器，访问：

```
https://www.python.org/downloads/
```

点击 **Download Python 3.12**（或最新版）

下载完成后运行安装程序，**一定要勾选**：

> ✅ Add Python to PATH

然后点击 **Install Now**。

### 1.3 下载书籍文件夹

朋友发给你的百度网盘链接，下载整个 `艺术知识库` 文件夹到你的电脑。

建议放在：**`E:\艺术知识库\`**

（如果 E 盘不存在，放到任何你喜欢的位置，只要记住路径就好）

### 1.4 准备好下载文件夹

通常你的下载文件夹在这里：

```
C:\Users\你的用户名\Downloads
```

以后你下载的新书都会先到这里。

---

## 第二步：配置路径

用记事本打开工具包里的脚本文件，找到顶部的路径配置，修改为你的实际路径：

```
# 把这些改成你的实际路径
KB = Path(r'E:\艺术知识库')                      # 你的知识库在哪里
DOWNLOADS = Path(r'C:\Users\你的用户名\Downloads')  # 你的下载文件夹在哪里
```

保存文件即可。

---

## 第三步：开始使用

打开 **PowerShell**（Windows 按 `Win + X`，选"终端"或"Windows PowerShell"）

先切换到工具目录：

```powershell
cd C:\Users\你的用户名\Downloads\art-knowledge-helper-skill\scripts
```

---

### 🔍 场景一：整理新下载的书籍

每次你下载了新书，先运行这个：

```powershell
python scan_downloads.py
```

它会扫描你最近 30 天下载的文件，找出艺术类的书，显示给你看。

---

### 📦 场景二：把新书归档到知识库

```powershell
python archive_books.py
```

它会自动判断每本书应该放进哪个分类，然后帮你复制进去。

---

### 📊 场景三：查看知识库统计

```powershell
python kb_stats.py
```

看看你现在有多少书，哪个分类最丰富。

---

### ☁️ 场景四：备份到百度网盘

```powershell
python sync_to_baidu.py
```

把本地知识库同步到百度网盘，记得先在百度网盘客户端设置好同步文件夹。

---

### ✅ 场景五：检查同步状态

```powershell
python verify_sync.py
```

确认本地和百度盘是否完全一致。

## 常见问题

**Q：运行脚本时报错怎么办？**
A：确保已经安装了 Python，并且路径配置正确。

**Q：书籍分类不对怎么办？**
A：没关系！手动移动一下就好，知识库没有"错误分类"。

**Q：百度网盘同步不工作？**
A：先确保百度网盘客户端已经启动，并且设置了 `艺术知识库` 文件夹的同步。

## 推荐使用节奏

| 时机 | 操作 |
|------|------|
| 每周一次 | 运行 `scan_downloads.py` + `archive_books.py` 整理新书 |
| 每次整理后 | 运行 `sync_to_baidu.py` 备份 |
| 每月一次 | 运行 `verify_sync.py` 检查同步状态 |
| 随时 | 运行 `kb_stats.py` 看看知识库的成长 |
