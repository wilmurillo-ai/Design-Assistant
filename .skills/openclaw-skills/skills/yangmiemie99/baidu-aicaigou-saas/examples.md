# 百度爱采购 SaaS 使用示例

## 示例 1：素材库上传（子能力 A）

**用户输入：**

> 帮我上传这些图片到百度爱采购素材库：
> - /path/to/image1.jpg
> - /path/to/image2.png

**路由判断：** 关键词"上传图片""素材库" → 路由到子能力 A（素材库上传）

**执行过程：**

1. 选择浏览器自动化方式（检测原生工具 → Playwright CLI）
2. 打开有头浏览器，访问 `https://b2bwork.baidu.com/shop/material/index`
3. 检查登录态，必要时等待用户登录
4. 获取页面快照，定位"本地上传"按钮
5. 通过 `input[type="file"]` 设置文件路径列表
6. 等待上传完成，确认素材出现在列表中

**预期输出：**

```
素材上传完成：
- image1.jpg ✓ 上传成功
- image2.png ✓ 上传成功
共 2 张素材已添加到素材库。
```

## 示例 2：商品维护（子能力 B）

**用户输入：**

> 帮我创建一个商品
> - /path/to/image1.jpg
> - /path/to/image2.png
> - /path/to/image3.jpg
> - /path/to/image4.png> 
> - /path/to/image5.png

**路由判断：** 关键词"爱采购后台""创建商品" → 路由到子能力 B（商品维护）

**执行过程：**

1. 初始化浏览器，进入商品管理列表页
2. 处理登录（如需要）
3. 搜索或定位目标商品
4. 进入编辑页面，修改标题字段为"工业级不锈钢管件"
5. 保存并等待页面反馈

**预期输出：**

```
商品标题已更新：
- 原标题：xxx
- 新标题：工业级不锈钢管件
- 状态：保存成功
```
