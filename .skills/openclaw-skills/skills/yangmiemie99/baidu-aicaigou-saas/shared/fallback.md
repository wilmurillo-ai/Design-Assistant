# 失败降级与兜底策略

当自动化执行不可用或中途失败时，按以下策略逐步降级。

## 降级流程

```
检测环境 → 尝试安装 → 重试一次 → 人工引导
```

### 第 1 步：检测当前环境

确认失败原因（工具缺失、权限不足、页面异常等）。

### 第 2 步：尝试安装缺失工具

优先尝试安装 `playwright-cli`（见 [browser-framework.md](browser-framework.md)），安装后重试。

### 第 3 步：重试一次

安装成功后自动重试当前失败步骤，仅重试一次。

### 第 4 步：切换人工引导模式

重试仍失败时，进入人工引导：

1. 明确告知用户当前缺失的能力或失败原因
2. 给出**最短路径点击步骤**（具体菜单路径和按钮名称）
3. 等待用户回传结果（截图或成功确认）
4. 收到用户回传后，继续执行后续步骤

## 人工引导输出格式

```
当前自动化不可用，请手动操作：
1. 打开 [具体URL]
2. 点击 [菜单/按钮名称]
3. 执行 [具体操作]
4. 完成后请告知结果或发送截图
```
