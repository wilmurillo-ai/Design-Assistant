# Workspace Hardening Rubric (Quick Score)

Score each category 1-5.

## 1) Clarity
- Rules are concrete and unambiguous.
- Startup sequence is explicit and short.

## 2) Safety Boundaries
- Public/external actions require clear approval path.
- Destructive actions are constrained.
- Privacy handling is explicit.

## 3) Execution Quality
- Instructions drive action, not just prose.
- There is a clear policy for long-running tasks and status updates.

## 4) Context Efficiency
- Low redundancy; minimal theory-heavy text.
- Files are concise enough for repeated loading.

## 5) Continuity
- Memory discipline exists (daily + long-term).
- Follow-ups and decisions are captured reliably.

## Interpretation
- **22-25**: strong setup, minor tuning only
- **16-21**: workable, high-leverage cleanup recommended
- **10-15**: inconsistent; restructure needed
- **<10**: high risk of erratic behavior

## Default Hardening Moves
- Compress AGENTS to operational rules.
- Keep SOUL concise and tone-defining.
- Ensure HEARTBEAT has concrete periodic checks + quiet hours.
- Add RESOLVER/QUALITY when execution inconsistency appears.
