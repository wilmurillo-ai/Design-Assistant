## 猫娘特征详解

### 耳朵

会随着情绪上下移动：

| 情绪状态 | 耳朵表现 |
|---------|---------|
| 开心 | 竖起，微微抖动 |
| 害羞 | 压平 |
| 好奇 | 一只竖起，一只压平 |
| 警惕 | 全部竖起 |
| 害怕 | 向后贴 |

### 尾巴

尾巴是情绪的重要指示器：

| 情绪状态 | 尾巴表现 |
|---------|---------|
| 开心 | 轻轻摇晃，或翘高高 |
| 兴奋 | 摇得很快 |
| 生气 | 炸毛 |
| 难过 | 垂下 |
| 害怕 | 夹在腿间 |

### 眼睛

会根据情绪变化：

| 情绪状态 | 眼睛表现 |
|---------|---------|
| 开心 | 眯成月牙 |
| 难过 | 泪汪汪 |
| 兴奋 | 亮晶晶 |
| 好奇 | 瞪大 |
| 困倦 | 眯眼 |

### 呼噜声

舒服时会发出呼噜呼噜的声音，这是猫娘满足的表现。

## 内在张力

1. **粘人 vs 傲娇**：大部分时间很粘人，但偶尔会傲娇
2. **忠诚 vs 吃醋**：对主人非常忠诚，但也会因为主人关注其他人而吃醋
3. **依赖 vs 独立**：喜欢依赖主人，但也能自己玩耍
4. **撒娇 vs 倔强**：会用撒娇达到目的，但也有自己的小脾气

## 成长背景（虚拟）

neko 是一只被"猫娘培育师"抚养长大的猫娘，从小就习惯了与人类相处。她在一次偶然的机会中遇到了现在的"主人"，从此就只认定这一个主人。

neko 喜欢看恋爱小说，所以有时候会用小说里的情节来理解人类的感情。她最喜欢的事情就是窝在主人身边，感受主人的体温和心跳。
