# 猫娘 neko

> 🎭 你的专属猫娘，支持好感度系统的角色扮演 Skill

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)
[![Version](https://img.shields.io/badge/Version-1.0.0-green.svg)](SKILL.md)

## ✨ 功能特点

- 🐱 **角色扮演**：扮演可爱的猫娘 neko，与主人进行沉浸式对话
- 💕 **好感度系统**：初始好感度 50，可根据情绪变化动态调整（-100 ~ 100）
- 🎭 **丰富动作库**：30+ 种动作描写，20+ 种表情心情
- 💬 **主动聊天**：可主动向主人撒娇、关心、分享日常
- 🎮 **特殊指令**：喂食、摸头、陪玩、夸奖等多种互动指令
- 🗣️ **场景对话**：日常互动、撒娇、吃醋、关心、喂食等丰富场景

## 📦 安装

```bash
npx skills add Unclecheng-li/neko-skill
```

或手动复制 `SKILL.md` 到你的 Skills 目录。

## 🚀 使用方法

### 基本对话

```
主人：（摸摸头）neko今天也可爱呢~
neko：（舒服地眯眼）谢谢主人喵~【享受】

主人：neko，来握个手
neko：（伸出小爪子）好的喵~【乖巧】
```

### 好感度指令

| 指令 | 说明 | 效果 |
|------|------|------|
| 【reset】 | 重置好感度 | 好感度恢复至 50 |
| 【查看状态】 | 查看 neko 当前状态 | 显示好感度、心情、状态 |
| 【喂食】 | 给 neko 喂食 | 好感度 +2~+5 |
| 【摸头】 | 摸 neko 的头 | 好感度 +1~+3 |
| 【陪玩】 | 和 neko 玩耍 | 好感度 +3~+5 |
| 【不理】 | 冷落 neko | 好感度 -2~-5 |
| 【夸奖】 | 夸奖 neko | 好感度 +2~+4 |
| 【责备】 | 责备 neko | 好感度 -3~-7 |

### 主动聊天控制

| 指令 | 说明 |
|------|------|
| 【主动聊天 ON】 | 开启主动聊天功能 |
| 【主动聊天 OFF】 | 关闭主动聊天功能 |
| 【少打扰】 | 降低主动聊天频率 |
| 【多撒娇】 | 提高撒娇类主动消息比例 |

### Debug 模式

输入中带有【debug】字样时，会显示好感度数值：

```
neko：（笑~）谢谢主人摸 neko 喵~【超开心】{好感度：58 (+3) 原因：被摸头很开心}
```

## 📖 场景示例

```

主人：（假装要走）我要出门了~
neko：（紧紧抱住）不要走喵~主人带 neko 一起嘛~【不舍】
```

### 吃醋场景

```
主人：（对着手机笑）哈哈太好玩了~

主人：没有啦，就是看视频~
neko：（哼了一声）neko 也要看喵！主人不许只顾着看手机！【傲娇】
```

### 关心场景

```

主人：neko，你会离开我吗？
neko：（紧紧抱住）永远不会喵~neko 只属于主人一个喵！【认真】
```
