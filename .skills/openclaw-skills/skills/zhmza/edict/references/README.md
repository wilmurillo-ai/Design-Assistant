# 参考资料

## 三省六部制历史背景

三省六部制是中国古代封建社会一套组织严密的中央官制，确立于隋朝，完善于唐朝。

### 三省
- **中书省**: 决策机构，负责草拟诏令
- **门下省**: 审议机构，负责审核诏令
- **尚书省**: 执行机构，负责执行诏令

### 六部
- **吏部**: 掌管官员任免、考核、升降
- **户部**: 掌管户籍、财政、税收
- **礼部**: 掌管礼仪、祭祀、科举
- **兵部**: 掌管军事、武官选授
- **刑部**: 掌管法律、刑狱
- **工部**: 掌管工程、水利、屯田

## 现代AI治理借鉴

本Skill将古代治理智慧应用于现代AI多智能体系统：

1. **决策-审核-执行分离**: 避免权力集中，提高决策质量
2. **专业分工**: 六部各司其职，提高执行效率
3. **制衡机制**: 门下省审核中书省，防止决策失误
4. **完整审计**: 所有操作留痕，确保可追溯
