# Prompt 优化指南

## 基础结构

```
[主体描述] + [动作] + [场景/背景] + [镜头语言] + [光影/色调] + [风格]
```

## 主体描述

### 人物

```
✓ A young Asian woman in her 20s with short black hair, wearing a white blouse
✗ 一个女生
```

关键要素：
- 年龄/性别
- 外貌特征
- 服装
- 表情/姿态

### 物体

```
✓ A sleek silver MacBook Pro on a minimalist white desk
✗ 一台电脑
```

## 动作描述

### 具体动词

| 模糊 | 具体 |
|------|------|
| 看 | gazing at, glancing at, staring at |
| 走 | walking slowly, striding confidently |
| 用 | typing on, scrolling through, tapping on |

### 动作节奏

- `slowly` / `gradually` - 慢动作
- `quickly` / `suddenly` - 快动作
- `smoothly` / `seamlessly` - 流畅

## 场景/背景

### 室内

```
modern minimalist office with floor-to-ceiling windows
cozy home office with warm lighting and plants
sleek tech startup workspace with exposed brick walls
```

### 室外

```
busy city street at sunset with golden hour lighting
serene park bench surrounded by autumn leaves
rooftop terrace overlooking city skyline at dusk
```

## 镜头语言

### 景别

| 术语 | 效果 |
|------|------|
| extreme close-up | 超特写，眼睛/嘴唇 |
| close-up | 特写，脸部 |
| medium close-up | 近景，胸部以上 |
| medium shot | 中景，腰部以上 |
| medium long shot | 中远景，膝盖以上 |
| long shot / wide shot | 全景 |
| extreme long shot | 大远景 |

### 镜头运动

| 术语 | 效果 |
|------|------|
| static shot | 固定镜头 |
| pan left/right | 横摇 |
| tilt up/down | 纵摇 |
| dolly in/out | 推/拉 |
| tracking shot | 跟拍 |
| crane shot | 升降镜头 |
| handheld | 手持晃动 |

### 景深

```
shallow depth of field, blurred background (背景虚化)
deep focus, everything in sharp detail (全景深)
bokeh effect (光斑效果)
```

## 光影/色调

### 光源

```
natural soft lighting from large window
dramatic side lighting with strong shadows
warm golden hour sunlight
cool blue ambient light
```

### 色调

```
warm color palette with orange and yellow tones
cool color grading with blue and teal
high contrast with deep blacks
soft pastel colors
cinematic color grading
```

## 风格关键词

### 真实感

```
photorealistic, hyperrealistic, 8K quality
shot on Sony A7III, 35mm lens
RAW photo, unedited
```

### 电影感

```
cinematic, movie still, film grain
anamorphic lens flare
directed by [导演名]
```

### 风格化

```
anime style, Studio Ghibli
3D render, Pixar style
minimalist, flat design
vintage 80s aesthetic
```

## 负面提示 (Negative Prompt)

常用排除项：

```
blurry, low quality, distorted, deformed
watermark, text, logo
extra limbs, bad anatomy
cartoon (如果要真实感)
```

## 分镜类型示例

### 痛点展示

```
Frustrated office worker staring at cluttered computer screen 
with multiple error messages, dim lighting, tired expression, 
medium close-up shot, shallow depth of field, 
desaturated color grading
```

### 解决方案

```
Confident professional smiling while using sleek app interface,
clean modern workspace, bright natural lighting,
medium shot, warm color palette,
smooth camera push-in
```

### 效果展示

```
Happy user celebrating with fist pump gesture,
laptop showing success screen,
bright cheerful lighting, confetti particles,
medium shot, slightly slow motion,
vibrant saturated colors
```

### CTA

```
Product logo elegantly appearing on gradient background,
smooth particle effects, professional motion graphics,
centered composition, clean typography,
brand colors
```
