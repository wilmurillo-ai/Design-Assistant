# 📋 Meeting Prep Skill for WorkBuddy

> 为重要会议快速生成结构化、专业美观的会前材料（HTML 网页 + 腾讯文档）。

## ✨ 功能

- **一句话触发**：说"帮我准备 XX 会议的会前材料"即可
- **自动调研**：从网络搜索、PPT/PDF 附件中提取关键信息
- **精美 HTML 输出**：深蓝渐变 Header、白色卡片、彩色标签、分级 Checklist
- **7 大内容模块**：背景速览 → 核心议题 → 合作机会 → 选题建议 → 切入思路 → 待确认清单 → 沟通策略
- **腾讯文档联动**（可选）：一键同步到腾讯文档，方便会中记录

## 📦 安装

### 方式一：WorkBuddy 内安装（推荐）

在 WorkBuddy 对话中输入：

```
帮我安装这个 skill：https://github.com/你的用户名/meeting-prep-skill
```

### 方式二：手动安装

```bash
git clone https://github.com/你的用户名/meeting-prep-skill.git ~/.workbuddy/skills/meeting-prep
```

## 🚀 使用

### 基础用法

```
帮我准备明天和 XX 的合作会议的会前材料
```

### 进阶用法（带附件）

```
帮我准备 4月17日 CACC 算法大会分会场报告的会前材料。
对方是 CCF 组委会，这是他们发的邀请函 @邀请函.pdf
我方是腾讯游戏学堂，这里有我们的介绍 @介绍PPT.pptx
```

### Prompt 模板

```
帮我准备 [会议主题] 的会前材料。

会议信息：
- 时间：[YYYY年MM月DD日 HH:MM]
- 对方：[机构名/人名]
- 我方：[部门/团队]
- 会议目的：[简述]

已有资料：
- [拖入文件或描述]

重点关注：
- [需要特别调研的方向]
- [需要准备的问题]
```

## 📖 输出示例

生成的会前材料包含：

| 模块 | 说明 |
|------|------|
| 对方背景速览 | 机构基本面、关键人物、已有合作基础 |
| 核心议题 | 会议议程、重要环节、时间线 |
| 合作机会分析 | 多个机会并列对比，评估价值和投入 |
| 选题建议 | 可选方向对比，含切入点建议 |
| 合作切入思路 | 具体参与方式、投入评估 |
| 待确认事项清单 | 分级 Checklist（🔴 紧急 / 🟢 探讨 / 🔵 后续） |
| 沟通策略备忘 | 谈判要点、注意事项、双赢锚点 |
