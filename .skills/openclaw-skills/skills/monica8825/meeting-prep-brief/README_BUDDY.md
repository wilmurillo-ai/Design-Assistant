# 会前材料制作 Skill

为重要会议快速生成结构化、专业美观的会前材料。

## 使用方法

直接告诉 Claw 你要准备什么会议，他会自动调用这个 Skill：

### 基础用法

```
帮我准备明天和 XX 的合作会议的会前材料
```

### 进阶用法（带附件）

```
帮我准备 4月17日 CACC算法大会分会场报告的会前材料。
对方是 CCF 组委会，这是他们发的邀请函 @邀请函.pdf
我方是腾讯游戏学堂，这里有我们的介绍 @介绍PPT.pptx
```

### Prompt 模板

```
帮我准备 [会议主题] 的会前材料。

会议信息：
- 时间：[YYYY年MM月DD日 HH:MM]
- 对方：[机构名/人名]
- 我方：[部门/团队]
- 会议目的：[简述]

已有资料：
- [拖入文件或描述]

重点关注：
- [需要特别调研的方向]
- [需要准备的问题]
```

## 输出物

1. **HTML 网页**（本地可预览） — 精美的会前材料页面
2. **腾讯文档**（可选） — 方便会中实时记录

## 内容模块

根据会议类型自动选配：

| 模块 | 说明 |
|------|------|
| 对方背景速览 | 机构基本面、关键人物、已有合作基础 |
| 核心议题/关键信息 | 会议议程、重要环节、时间线 |
| 合作机会分析 | 多个机会并列对比，评估价值和投入 |
| 选题/方案建议 | 可选方向对比，含切入点建议 |
| 合作切入思路 | 具体参与方式、投入评估 |
| 待确认事项清单 | 分级 Checklist（紧急/探讨/后续） |
| 沟通策略备忘 | 谈判要点、注意事项、双赢锚点 |

## 小贴士

- 提供的资料越多（邮件、PPT、官网链接等），材料质量越高
- 如果需要同步到腾讯文档，确保已配置腾讯文档 MCP Token
- 生成后可以随时要求修改某个模块的内容
- 会后也可以让 Claw 根据会议结果更新材料、生成 follow-up 待办
