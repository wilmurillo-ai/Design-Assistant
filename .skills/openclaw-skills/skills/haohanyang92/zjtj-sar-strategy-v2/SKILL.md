# ZJTJ+SAR 四维选股策略 V2

> 本策略为完全重新设计的版本，结合主力动向、趋势判断、资金验证和市场情绪四个维度，适合A股短线波段操作。

---

## 一、策略介绍

### 1.1 核心理念

**"顺势而为，乘轿而上"** —— 本策略的核心思想是在市场环境配合的前提下，找到主力已经介入、趋势开始启动的强势股。

通过四个维度的交叉验证：
- **ZJTJ（庄家抬轿）**：识别主力控盘状态，确认是否有"庄家"在拉升股价
- **SAR（抛物线指标）**：判断趋势方向，在趋势启动初期介入
- **成交量**：验证价格突破的真实性，避免"假突破"陷阱
- **市场情绪**：判断大盘环境，确保在安全的市场氛围下操作

### 1.2 策略优势

| 优势 | 说明 |
|------|------|
| 多维验证 | 四个维度互相验证，减少假信号 |
| 趋势跟随 | SAR指标天然具有趋势跟随特性 |
| 主力识别 | ZJTJ帮助识别主力资金动向 |
| 环境判断 | 结合市场情绪，避免逆势操作 |

### 1.3 策略局限

- 适用于**短线波段**操作，不适合长线价值投资
- 在**震荡市**中可能产生较多无效信号
- 需要**每日复盘**跟踪持仓股动态
- 对**流动性较差的股票**信号可能失真

## 二、四大维度详解

### 2.1 第一维：ZJTJ（庄家抬轿）—— 判断主力控盘状态

#### 指标原理

ZJTJ（庄家抬轿）是一个判断主力资金动向的指标系列，通过分析资金的流入流出、控盘程度来识别主力是否在运作股票。

#### 核心信号

| 信号 | 含义 | 强度 |
|------|------|------|
| **ZJTJ红柱** | 主力资金净流入 | ⭐⭐⭐ |
| **ZJTJ黄柱** | 主力高度控盘 | ⭐⭐⭐⭐ |
| **ZJTJ紫柱** | 主力拉升阶段 | ⭐⭐⭐⭐⭐ |
| **ZJTJ绿柱** | 主力出货 | ⭐⭐ |

#### 实战用法

1. **关注红柱/黄柱出现的连续性**：连续3天以上红柱，说明主力持续买入
2. **配合股价位置**：在低位出现ZJTJ信号更可靠，高位可能是诱多
3. **看ZJTJ的斜率**：斜率越陡，说明主力拉升意愿越强

> **注意**：ZJTJ指标需要用**同花顺/通达信/东方财富**等软件的ZJTJ指标（不同软件算法可能略有差异），或者使用类似的的主力资金流向指标替代。

#### 如何查看ZJTJ指标

**同花顺**：
1. 打开股票详情页（输入股票代码）
2. 点击「**副图指标**」
3. 在指标列表中找到「**ZJTJ**」或「**庄家抬轿**」
4. 点击添加即可显示

**通达信**：
1. 打开股票详情页
2. 点击「**指标**」或「**副图**」
3. 在自编指标或特色指标中找到「**ZJTJ**」
4. 添加到副图显示

**通达信快捷键**：输入「**ZJTJ**」或「**.ZJTJ**」可直接调出指标

> 如果软件没有ZJTJ指标，可用「**主力资金**」「**主力净流入**」等指标替代，关键是要看到主力资金的流入状态。

### 2.2 第二维：SAR（抛物线指标）—— 判断趋势方向

#### 指标原理

SAR（Stop and Reverse，抛物线指标）是一种**趋势跟踪**指标，由J. Welles Wilder Jr.发明。指标特性：
- 当股价在SAR之上时，说明处于**上升趋势**
- 当股价在SAR之下时，说明处于**下降趋势**
- SAR的点位会**跟随股价移动**，形成类似抛物线的轨迹

#### 核心信号

| 信号 | 含义 | 强度 |
|------|------|------|
| **股价上穿SAR** | 趋势转多，可能开始上涨 | ⭐⭐⭐⭐ |
| **股价下穿SAR** | 趋势转空，可能开始下跌 | ⭐⭐⭐⭐ |
| **SAR连续上升** | 上升趋势确认，持股待涨 | ⭐⭐⭐ |
| **SAR连续下降** | 下降趋势确认，观望为主 | ⭐⭐⭐ |

#### 实战用法

1. **首次买入信号**：股价从SAR下方**首次上穿**SAR，是较为可靠的买入信号
2. **趋势确认**：股价持续在SAR上方运行，说明趋势健康
3. **止损位**：SAR点位可以作为**动态止损位**，股价跌破SAR出场

### 2.3 第三维：成交量 —— 验证真假突破

#### 指标原理

量在价先，成交量是价格的**先行指标**。健康的上涨需要成交量的配合，无量的上涨往往是"虚涨"。

#### 核心信号

| 信号 | 含义 | 强度 |
|------|------|------|
| **放量上涨** | 量价配合，上涨可靠 | ⭐⭐⭐⭐⭐ |
| **缩量回调** | 回调缩量，说明抛压小 | ⭐⭐⭐⭐ |
| **放量突破** | 突破时放量，真突破概率大 | ⭐⭐⭐⭐⭐ |
| **缩量上涨** | 量价背离，可能是诱多 | ⭐⭐ |
| **放量下跌** | 恐慌抛售，可能接近底部 | ⭐⭐⭐ |

#### 实战用法

1. **突破买入时**：必须配合放量（量能≥5日均量的1.5倍）
2. **回调时观察**：缩量回调是好事，说明惜售
3. **异常放量**：量能突然放大3倍以上要警惕，可能是主力对倒出货

---

### 2.4 第四维：市场情绪 —— 判断大盘环境

#### 指标原理

**"覆巢之下无完卵"** —— 即便是个股技术面完美，如果大盘处于下跌趋势，成功的概率也会大幅下降。市场情绪是**系统性风险**的过滤器。

#### 核心信号

| 市场状态 | 特征 | 操作建议 |
|----------|------|----------|
| **亢奋期** | 涨停潮、成交量放大、全线上涨 | 积极操作，保持仓位 |
| **分歧期** | 板块轮动、涨跌互现 | 精选个股，控制仓位 |
| **冰点期** | 跌多涨少、情绪低迷 | 轻仓或空仓观望 |
| **修复期** | 止跌企稳、热点出现 | 试探性建仓 |

#### 实战用法

1. **每日复盘**：查看上证指数、创业板指、涨跌家数
2. **关注情绪指标**：涨跌停数量、炸板率、接力情绪
3. **仓位调整**：
   - 亢奋期：仓位可提升至80%
   - 分歧期：仓位控制在50%
   - 冰点期：仓位降至20%以下
   - 修复期：仓位30%

## 三、评分标准（满分100分）

### 3.1 分值分配

| 维度 | 指标 | 满分 | 说明 |
|------|------|------|------|
| **ZJTJ（庄家抬轿）** | 主力资金状态 | 30分 | 判断主力是否在运作 |
| **SAR（抛物线指标）** | 趋势方向 | 25分 | 判断多空趋势 |
| **成交量** | 量价配合 | 20分 | 验证真假突破 |
| **市场情绪** | 大盘环境 | 25分 | 系统性风险过滤 |

### 3.2 ZJTJ评分细则（30分）

| 状态 | 得分 | 说明 |
|------|------|------|
| 紫柱（拉升期） | 30分 | 主力正在拉升，强烈看多 |
| 黄柱（控盘期） | 25分 | 主力高度控盘，稳步上涨 |
| 红柱（流入期） | 20分 | 主力资金净流入，趋势向好 |
| 红转黄 | 22分 | 流入转控盘，强势信号 |
| 黄转紫 | 28分 | 控盘转拉升，加速上涨 |
| 绿柱（出货期） | 0分 | 主力出货，坚决回避 |
| 无信号/平盘 | 5分 | 无主力迹象 |

### 3.3 SAR评分细则（25分）

| 状态 | 得分 | 说明 |
|------|------|------|
| 股价上穿SAR首日 | 25分 | 趋势反转，强烈买入信号 |
| 股价持续在SAR上方（3日内） | 20分 | 上升趋势健康（与上穿SAR不重复计分，取高） |
| SAR连续上升（5日内） | 15分 | 上升趋势确认（与上穿SAR不重复计分，取高） |
| 股价在SAR上方但偏离>10% | 12分 | 短期超买，有回调风险 |
| 股价在SAR下方 | 0分 | 下降趋势，不买入 |
| 股价下穿SAR | 0分 | 趋势破位，卖出信号 |

> **注**：SAR评分采用**取高原则**，同一时间只取最高分，不重复计分。

### 3.4 成交量评分细则（20分）

| 状态 | 得分 | 说明 |
|------|------|------|
| 放量突破（量能≥1.5倍5日均量，且股价创20日内新高） | 20分 | 量价配合，真突破概率大 |
| 温和放量上涨（1.2-1.5倍5日均量） | 17分 | 健康上涨 |
| 缩量回调（量能<0.8倍5日均量） | 15分 | 回调缩量，抛压小 |
| 平量上涨（0.8-1.2倍5日均量） | 12分 | 量价正常 |
| 缩量上涨（<0.8倍5日均量） | 5分 | 量价背离，可能是诱多 |
| 放量下跌（>1.5倍5日均量） | 5分 | 恐慌抛售，观望 |
| 量能异常放大（>3倍5日均量） | 8分 | 警惕对倒，谨慎 |

> **放量标准**：当日成交量 ≥ 1.5倍5日成交量均线（MA5）

### 3.5 市场情绪评分细则（25分）

| 市场状态 | 得分 | 说明 |
|----------|------|------|
| **亢奋期** | 25分 | 市场活跃，积极做多 |
| **修复期** | 20分 | 企稳回升，可以操作 |
| **分歧期** | 12分 | 轮动分化，精选个股 |
| **冰点期** | 0分 | 情绪低迷，不操作 |
| 大盘在5日均线上方 | +5分 | 额外加分 |
| 大盘在20日均线上方 | +3分 | 额外加分 |
| 大盘MACD金叉 | +3分 | 额外加分 |

> **注1**：情绪评分采用**取低原则**，如果处于冰点期，即使其他指标好也不得分。  
> **注2**：额外加分项加分后不超过25分（即情绪总分最高25分）。  
> **注3**：**总分上限100分**，各项加分后若超过100分，按100分计算。

### 3.6 综合评分等级

| 总分 | 等级 | 操作建议 |
|------|------|----------|
| **90-100分** | 完美 | 强烈买入，重仓操作 |
| **75-89分** | 优秀 | 积极买入，仓位适中 |
| **60-74分** | 良好 | 可以买入，轻仓试错 |
| **45-59分** | 一般 | 观望为主，不建议买 |
| **45分以下** | 较差 | 坚决不买，等待机会 |

> **注**：总分上限为**100分**，各项加分后若超过100分，按100分计算。

## 四、买入条件

### 4.1 基础买入条件（必须同时满足）

1. **ZJTJ评分 ≥ 15分**（有主力运作）
2. **SAR评分 > 0分**（股价在SAR上方或刚上穿）
3. **成交量评分 ≥ 12分**（量价配合正常）
4. **市场情绪评分 ≥ 12分**（非冰点期）

### 4.2 进阶买入条件（满足基础+任一进阶）

- **✅ 突破买入**：股价放量突破近期高点 + ZJTJ红/黄/紫柱
- **✅ 回踩买入**：股价回踩SAR企稳 + 缩量回调 + ZJTJ维持红柱
- **✅ 二次买点**：首次上穿SAR后回调不破SAR + 再次上穿

### 4.3 买入流程

```
1. 筛选：每日收盘后复盘，找出ZJTJ出现信号的股票
2. 验证：检查SAR是否在股价下方，成交量是否配合
3. 评分：按评分标准打分，总分≥60分可考虑买入
4. 试仓：首次买入不超过总仓位的1/3
5. 确认：买入后次日如果继续走强，可加仓
```

## 五、卖出条件（止盈止损）

### 5.1 止损规则

| 止损条件 | 执行动作 |
|----------|----------|
| **股价跌破SAR** | 立即止损，无条件卖出 |
| **收盘价<买入价-7%** | 次日开盘卖出 |
| **ZJTJ出现绿柱** | 主力可能出货，次日卖出 |
| **放量下跌（量能>1.5倍）** | 可能是破位，次日卖出 |

> **硬性纪律**：单只股票亏损超过**7%**必须止损！

### 5.2 止盈规则

| 止盈条件 | 执行动作 |
|----------|----------|
| **涨幅≥15%** | 卖出1/3，锁住部分利润 |
| **涨幅≥25%** | 再卖出1/3，保留1/3博取更大涨幅 |
| **涨幅≥35%** | 全部卖出，锁定利润 |
| **SAR拐头向下** | 趋势可能结束，卖出1/2 |
| **高位放量滞涨** | 主力可能出货，清仓 |

### 5.3 动态止盈策略

采用**移动止损**保护利润：

```
买入后：
- 涨幅<10%：止损位设为买入价
- 涨幅10-20%：止损位上移至买入价
- 涨幅20-30%：止损位上移至涨幅10%位置
- 涨幅>30%：止损位上移至涨幅20%位置
```

### 5.4 卖出流程

```
1. 触发止盈：按上述规则分批卖出
2. 触发止损：立即卖出，不犹豫
3. 大盘风险：市场进入冰点期，可先卖出避险
4. 换股操作：如果发现更好的标的，可卖出持有的股票
```

## 六、风险提示

### 6.1 系统性风险

| 风险 | 描述 | 应对 |
|------|------|------|
| **大盘暴跌** | 系统性下跌，个股难以幸免 | 市场情绪冰点期降仓或空仓 |
| **政策风险** | 突发的政策利空 | 关注政策面，保持警惕 |
| **板块轮动** | 热点切换导致踏空 | 不追涨，等回调再买 |

### 6.2 个股风险

| 风险 | 描述 | 应对 |
|------|------|------|
| **主力出货** | ZJTJ突然变绿 | 设立止损位，快进快出 |
| **假突破** | 放量突破后回落 | 止损保护，跌破SAR必须出 |
| **业绩暴雷** | 突发的业绩利空 | 分散持仓，避免单票重仓 |
| **流动性风险** | 小盘股买卖困难 | 优先选择日成交>1亿的股票 |

### 6.3 交易心理风险

| 风险 | 描述 | 应对 |
|------|------|------|
| **踏空焦虑** | 怕错过上涨而追高 | 等待回调，不追涨 |
| **亏损恐慌** | 亏损后急于扳本 | 严格止损，不扛单 |
| **盈利回吐** | 赚钱后又亏回去 | 移动止盈，保护利润 |

### 6.4 风险控制铁律

> ⚠️ **必须遵守**
> 
> 1. 单只股票仓位不超过总资金的20%
> 2. 单只股票亏损7%必须止损
> 3. 市场冰点期仓位不超过30%
> 4. 不买ST股、亏损股、问题股
> 5. 永远不要满仓操作
> 6. 连续3只股票止损后暂停操作

### 6.5 不同市场环境的仓位建议

| 市场环境 | 总仓位建议 | 说明 |
|----------|------------|------|
| **牛市** | 80%-100% | 趋势向上，积极做多，可适当重仓 |
| **震荡市** | 40%-60% | 区间震荡，控制仓位，高抛低吸 |
| **熊市** | 0%-20% | 趋势向下，轻仓或空仓观望为主 |
| **亢奋期** | 80% | 市场活跃，可提升仓位 |
| **分歧期** | 50% | 轮动分化，保持中性仓位 |
| **冰点期** | ≤20% | 情绪低迷，严格控制仓位 |
| **修复期** | 30% | 试探性建仓，边走边看 |

> **注**：仓位建议为参考值，需根据个人风险承受能力和实际情况调整。新手建议在此基础上再降一档。

## 七、适用场景

### 7.1 最佳使用场景

| 场景 | 描述 | 策略表现 |
|------|------|----------|
| **牛市主升浪** | 大盘趋势向上，热点活跃 | ⭐⭐⭐⭐⭐ 表现最佳 |
| **反弹行情** | 大盘超跌后企稳回升 | ⭐⭐⭐⭐ 表现较好 |
| **震荡上行** | 大盘区间震荡，个股分化 | ⭐⭐⭐ 表现一般 |
| **板块轮动** | 热点切换快，持续性差 | ⭐⭐ 需精选个股 |

### 7.2 不适用场景

| 场景 | 描述 | 风险 |
|------|------|------|
| **熊市下跌** | 大盘单边下跌 | ⭐⭐⭐⭐⭐ 高风险 |
| **市场冰点** | 情绪极度低迷 | ⭐⭐⭐⭐ 高风险 |
| **震荡筑底** | 方向不明 | ⭐⭐⭐ 容易反复打脸 |
| **政策突变** | 突发重大政策 | ⭐⭐⭐⭐ 不可预测 |

### 7.3 适用投资者类型

- **短线交易者**：能每日复盘，关注短期波动
- **波段操作者**：能持股1-4周，追求波段收益
- **有一定经验**：了解技术分析，会看成交量
- **纪律性强**：能严格止损，不情绪化交易

> **不适合**：没有时间看盘、无法执行纪律、追求长线价值投资的投资者

## 八、策略使用流程

### 8.1 每日复盘流程

```
📅 每日收盘后（16:00-18:00）
│
├─ Step 1: 看大盘（10分钟）
│  ├─ 上证指数趋势位置
│  ├─ 涨跌家数对比
│  └─ 判断市场情绪阶段
│
├─ Step 2: 筛选股票（20分钟）
```
