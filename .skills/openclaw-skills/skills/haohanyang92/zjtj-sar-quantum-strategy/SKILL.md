# ZJTJ + SAR + 成交量 + 市场情绪 四维选股策略

## 四大维度

| 维度 | 指标 | 作用 |
|-----|------|------|
| 1️⃣ 主力动向 | ZJTJ | 判断庄家是否进场、是否出货 |
| 2️⃣ 趋势判断 | SAR | 判断方向 + 动态止损 |
| 3️⃣ 资金验证 | 成交量 | 验证真假突破 |
| 4️⃣ 大盘环境 | 市场情绪 | 避免在冰点/亢奋期操作 |

---

## 🟢 买入条件（四维同时满足）

### 维度1：ZJTJ（主力信号）

| 信号 | 状态 | 操作 |
|-----|------|------|
| 🟡 黄色箭头 | 开始控盘 | ✅ 关注 |
| 🔴 红色 | 有庄控盘 | ✅ 买入 |
| 🟣 紫色 | 高度控盘 | ⚠️ 谨慎追 |

### 维度2：SAR（趋势信号）

| 信号 | 状态 | 操作 |
|-----|------|------|
| 点在K线下方 | 上涨趋势 | ✅ 持有 |
| 点从上方翻到下方 | 买入信号 | ✅ 买入时机 |

### 维度3：成交量（资金验证）

| 情况 | 信号 | 操作 |
|-----|------|------|
| 涨停时放量（较前一日+30%以上） | 真突破 | ✅ 跟进 |
| 涨停时缩量 | 假突破 | ❌ 不追 |
| 上涨缩量 | 量价背离 | ⚠️ 警惕 |

### 维度4：市场情绪（环境判断）

| 情绪阶段 | 仓位建议 |
|---------|---------|
| 🧊 冰点（涨停<20家） | 轻仓/空仓，不做 |
| ⚖️ 分歧（涨停20-50家） | 半仓操作 |
| 🌟 正常（涨停50-80家） | 正常仓位 |
| 🔥 亢奋（涨停>80家） | 重仓干龙头 |

---

## 🔴 卖出条件

### 止盈（分批卖出）

| 涨幅 | 操作 |
|-----|------|
| +10% | 卖1/3 |
| +20% | 卖1/3 |
| +30% | 剩1/3（博更大） |

### 止损（保护本金）

| 条件 | 操作 |
|-----|------|
| SAR点翻到K线上方 | 卖出（趋势结束） |
| 下跌 > 7% | 止损 |
| ZJTJ变绿色（出货） | 清仓 |
| ZJTJ变白色（无庄） | 清仓 |

### 大盘风险信号

| 信号 | 操作 |
|-----|------|
| 涨停家数 < 15家 | 停止买入 |
| 跌停家数 > 30家 | 清仓观望 |

---

## ⚠️ 策略禁区（绝对不做）

1. **冰点 + ZJTJ无信号** → 不做
2. **ZJTJ白色 + SAR下降** → 不做
3. **缩量涨停** → 不追
4. **大盘跌停 > 30家** → 停止所有操作

---

## 📊 选股公式（通达信）

```
ZJTJ > 0 AND SAR < C AND V > REF(V,1)*1.3
```

解释：ZJTJ控盘 + SAR在K线上方 + 成交量放量30%

## 🔄 每日复盘流程

```
收盘后（16:00-17:00）：

1️⃣ 看大盘情绪
   - 涨停 < 20家 → 今天不做
   
2️⃣ 筛选ZJTJ
   - 出现黄色/红色信号的股票
   
3️⃣ 筛选SAR
   - 点在K线下方
   
4️⃣ 验证成交量
   - 放量上涨的保留
   
5️⃣ 选出标的
   - 符合4个条件 → 第二天开盘买
   
6️⃣ 制定买卖计划
   - 涨到哪卖，跌破哪止损
```
