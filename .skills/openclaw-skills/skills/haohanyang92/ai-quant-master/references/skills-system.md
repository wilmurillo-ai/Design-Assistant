# Skills技能系统

## 简介

OpenClaw的Skills系统是其核心亮点，支持技能安装、组合、延伸。技能本质上是包含SKILL.md说明文档的项目包，可通过自然语言调用和组合。本指南基于视频第5集内容整理。

---

## 技能市场

**网站**：https://clawhub.com

- 每周新增大量技能
- 覆盖金融、编程、生活等各领域
- 可搜索、评分、评论

### 安装技能

```bash
# 方式一：CLI安装
openclaw skills install <skill-name>

# 方式二：手动下载
# 1. 从clawhub.com下载技能包
# 2. 解压到 ~/.openclaw/workspace/skills/ 目录
```

---

## 技能结构

每个技能包含：

```
skill-name/
├── SKILL.md          # 技能说明文档（必需）
├── references/       # 参考资料目录
│   ├── guide1.md
│   └── guide2.md
├── examples/         # 代码示例
│   ├── example1.py
│   └── example2.py
└── scripts/          # 脚本目录（如有）
    └── run.sh
```

### SKILL.md结构

```markdown
# 技能名称

## 简介
简要说明技能功能

## 适用场景
列出技能适用的场景

## 使用方法
- 触发词1：执行xxx
- 触发词2：执行yyy

## 配置要求
- API Key需求
- 环境依赖

## 注意事项
使用注意点
```

## 自定义技能开发

### 开发流程

1. 在 `~/.openclaw/workspace/skills/` 下创建技能目录
2. 编写 SKILL.md 说明文档
3. 编写参考文档和示例代码
4. 通过OpenClaw加载并测试

### 示例：股票分析技能

```markdown
# Stock Analysis Skill

## 功能
基于AKShare获取A股数据，进行技术分析和基本面评分

## 触发词
- "分析股票"
- "查询行情"
- "基本面分析"

## 核心能力
- 实时行情获取
- 财务数据查询
- 技术指标计算
- 综合评分输出

## API依赖
- AKShare（免费）
- pandas/numpy
```

## 技能组合与延伸

### 组合多个技能

OpenClaw可以自动分析需求，组合多个现有技能生成新技能：

```markdown
# 基金投研团队技能
- 组合了：AKShare数据分析 + 财务分析 + 行业研究
- 自动生成了新的复合技能
```

### 技能延伸案例

```
原始技能：
- 股票深度分析
- 股票监控预警

组合后生成：
- 基金投研团队（多智能体协作框架）
```
