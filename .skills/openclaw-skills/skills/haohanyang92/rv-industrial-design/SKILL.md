# 房车工业设计（RV Industrial Design）

## 核心能力

本技能可输出专业房车设计文档，包括：
- 设计概念与风格定义
- 空间布局规划方案
- CMF设计规范（色彩/材料/工艺）
- 水电系统设计指导
- 人机工程与舒适度标准
- HMI/智能控制功能规划
- 竞品分析与品牌定位
- 用户场景与旅程地图

**注意**：本技能输出设计文档和规格参数，实际渲染图/效果图需调用画师（nano-banana-pro）配合。

## 7大设计维度

### 维度1：空间布局规划

**核心原则**：紧凑、多功能、可变形、安全优先

**布局类型**：
| 类型 | 特点 | 适用场景 |
|------|------|---------|
| B型（Vanlife） | 原厂车身，空间紧凑 | 2人以下，短途旅行 |
| C型（Cab-over） | 额头床，空间大 | 家庭，长途旅居 |
| 拖挂式 | 车厢与车头分离 | 长期定居基地 |
| 升顶帐篷式 | 顶部可变床铺 | 轻量化，季节性使用 |

**功能区划分**：
- 驾驶区：主驾+副驾座椅（可旋转）
- 起居区：卡座沙发+餐桌（可转换为床）
- 厨卫区：紧凑厨房+干湿分离卫浴
- 睡眠区：固定床/升降床/额头床
- 储物区：顶柜+底柜+外置储物舱

**人流动线**：
- 通道宽度 ≥ 500mm（单人通过）
- 主要通道 ≥ 600mm（双向通过）
- 车门开口 ≥ 700mm（紧急疏散）

---

### 维度2：外观造型设计

**设计原则**：功能性 + 品牌识别 + 空气动力学

**车身结构**：
| 结构 | 特点 | 成本 |
|------|------|------|
| 承载式车身 | 低重心，适合B型 | 中 |
| 非承载式车身 | 大梁结构，适合C型 | 高 |
| 拖挂底盘 | 独立车厢 | 低 |

**外观设计要素**：
- **额头设计**：C型额头顶部空间 + 空气导流
- **箱体结构**：保温板拼接 + 铝合金骨架
- **外置配置**：遮阳棚支架、自行车架、行李架、太阳能板
- **灯具设计**：LED灯带（夜间氛围）+ 越野探照灯
- **空气动力学**：额头圆角优化 ≥ 15%风阻降低

**设计语言建立**：
1. 定义品牌DNA（线条特征、符号元素）
2. 建立家族化前脸/尾部特征
3. 保持一定程度的零部件通用性

---

### 维度3：CMF设计

**C - 色彩（Color）**：
- 外观色：白色/银灰为主（防晒+耐脏+通用性）
- 内饰色：浅木色+米白+局部跳色（活力感）
- 功能色：警示橙、安全黄用于安全标识

**M - 材料（Material）**：
| 部位 | 推荐材料 | 特性 |
|------|---------|------|
| 车厢外板 | 铝合金复合板 / FRP | 防腐、保温 |
| 车厢内板 | PVC发泡板 / 生态皮革 | 防水、易清洁 |
| 地板 | 防水复合木地板 / 柚木地板 | 耐磨、防滑 |
| 家具 | E0级复合板材 / 铝合金框架 | 轻量化 |
| 保温层 | XPS/EPS泡沫 + 无纺布 | 隔热性能 |
| 密封胶 | 改性硅烷胶 | 耐候、弹性 |

**F - 工艺（Finishing）**：
- 车厢拼接：型材卡扣 + 螺栓预紧
- 防水处理：接缝处双重密封胶条
- 防腐处理：底盘装甲喷涂 + 防锈蜡
- 表面处理：磨砂阳极氧化（铝合金件）

### 维度4：水电系统设计

**水系统**：
```
进水 → 净水箱（80-200L）→ 过滤器 → 水泵 → 热水锅炉 → 龙头/花洒
                                              ↓
                    灰水箱（废水）← 洗手盆/淋浴
                                              ↓
                    黑水箱（粪便）← 马桶
```

**关键参数**：
| 参数 | 建议值 |
|------|--------|
| 净水箱 | 100-150L（满足3-5天） |
| 灰水箱 | 50-80L |
| 黑水箱 | 50-60L |
| 水泵 | 12V/5-10L/min |
| 热水 | 12V热水器 / 燃气热水器 |
| 水管 | 食品级PE软管（内径≥20mm） |

**电系统**：
```
市电/发电机 → 充电机 → 锂电池（100-400Ah）→ 逆变器（3000W）→ 交流插座
                                                      ↓
太阳能板（400-1000W）→ 太阳能控制器 → 锂电池
                                                      ↓
12V直流负载：灯光、冰箱、水泵、排风扇
```

**关键参数**：
| 参数 | 建议值 |
|------|--------|
| 锂电池 | 磷酸铁锂（LiFePO4）≥ 200Ah |
| 太阳能 | 400-800W（视使用场景） |
| 逆变器 | 正弦波，≥ 3000W |
| 市电充电 | ≥ 30A |
| 底盘取电 | ≥ 30A |

### 维度5：人机工程与舒适度

**座椅安全标准**：
- 行车状态：座椅须正向安装，安全带固定
- 行驶中禁止站立、躺卧
- 座椅须通过ECE R44/UN R129碰撞测试

**睡眠舒适**：
- 固定床：长 ≥ 1900mm，宽 ≥ 1200mm（双人）
- 床垫：记忆棉/乳胶，厚度 ≥ 100mm
- 额头床：注意头部碰撞风险（须有缓冲）

**厨卫标准**：
- 厨房操作台高度 850-950mm
- 冰箱容积 ≥ 50L（旅途需求）
- 卫浴间：干湿分离，防水地面 + 排风扇

**噪音控制**：
- 墙体隔音量 ≥ 30dB
- 底盘隔振垫 + 发动机舱隔音棉
- 驻车空调噪音 ≤ 55dB

**温度控制**：
- 保温层厚度 ≥ 50mm（四季适用）
- 驻车暖风 ≥ 2kW（柴油暖风/冷暖空调）
- 合理通风：顶置换气扇 + 侧窗

### 维度6：HMI/智能控制

**中控系统功能规划**：

| 功能模块 | 详细内容 |
|---------|---------|
| 电力监控 | 电池电量、剩余使用时间、用电负载、充电状态 |
| 水箱监控 | 净水/灰水/黑水液位、剩余天数提醒 |
| 温控管理 | 车内温度、湿度、暖风/冷风控制 |
| 灯光控制 | 氛围灯（多模式）、阅读灯、示廓灯 |
| 安防监控 | 车门状态、烟雾报警、燃气泄漏报警 |
| 设备联动 | 欢迎回家模式（灯光+空调+窗帘自动开启） |
| 远程控制 | APP远程开关设备、查看状态 |
| 用电管理 | 优先级的设置（市电/太阳能/电池） |

**HMI设计原则**：
- 信息层级清晰：一级（电力+水箱），二级（详细设置）
- 一键场景模式：露营模式、睡眠模式、出行模式
- 容错设计：重要操作需二次确认
- 低功耗待机：熄屏待机功耗 ≤ 5W

### 维度7：品牌与用户体验

**设计语言建立流程**：
1. 品牌定位 → 2. 用户研究 → 3. 设计DNA提取 → 4. 设计规范输出

**竞品分析维度**：
| 品牌 | 布局特点 | CMF | 智能化 | 价格区间 |
|------|---------|-----|-------|---------|
| 大通RV80 | C型，实用为主 | 商务灰 | 基础 | 30-50万 |
| 宇通T510 | 家庭向，空间大 | 温馨木纹 | 中等 | 40-60万 |
| 长城炮房车 | B型，轻量化 | 科技感 | 基础 | 25-40万 |
| 湃力特 | 欧洲设计，精致 | 高端 | 丰富 | 50-80万 |

**用户场景分析**：

| 场景 | 用户需求 | 设计要点 |
|------|---------|---------|
| 长途旅行 | 续航、舒适 | 大水箱、大电量、舒适床铺 |
| 周末露营 | 便捷、快速 | B型、快速展开 |
| 亲子家庭 | 安全、娱乐 | 儿童座椅接口、娱乐系统 |
| 退休旅居 | 长期居住 | 稳定性、医疗设备兼容 |
| 野外作业 | 通过性、续航 | 四驱、越野配置 |

**用户体验旅程地图**：
```
选车 → 试驾 → 提车 → 第一次旅行 → 日常维护 → 升级改装 → 推荐他人
```
