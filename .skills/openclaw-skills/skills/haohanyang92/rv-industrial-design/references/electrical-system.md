# 水电系统设计参考

## 水系统设计

### 系统拓扑
```
进水口（重力加油）→ 滤网 → 水泵（12V） → 净水箱 → 热水器 → 各用水点
                                    ↓
                        一体化水表（流量监测）
                                    ↓
各用水点排水 → 灰水箱 → 电动排污阀
                 ↓
马桶排水 → 黑水箱 → 电动排污阀
```

### 水箱容量规划
| 使用场景 | 净水箱 | 灰水箱 | 黑水箱 |
|---------|-------|-------|-------|
| 周末露营（2人） | 80L | 40L | 30L |
| 长途旅行（4人） | 150L | 80L | 60L |
| 长期旅居（2人） | 200L | 100L | 60L |

### 水泵选型
| 类型 | 规格 | 适用场景 |
|------|------|---------|
| 隔膜泵 | 12V/5L/min/0.3MPa | 小型B型 |
| 离心泵 | 12V/10L/min/0.4MPa | C型标准配置 |
| 变频泵 | 12V/变频/静音 | 高端车型 |

### 热水方案
| 方案 | 功率 | 容积 | 特点 |
|------|------|------|------|
| 电热水器 | 500-1500W | 20-40L | 需接市电 |
| 燃气热水器 | 10kW | 即热式 | 需配燃气罐 |
| 柴油暖风热水 | 5-10kW | 8-15L | 四季通用 |

## 电系统设计

### 系统拓扑
```
                    ┌─ 市电充电（220V）
                    │
                    ↓
充电来源 → 充电机 → 锂电池组（LiFePO4）→ 逆变器 → 220V交流负载
                    ↑
太阳能板 → 太阳能控制器 ─┘
                    ↑
              底盘取电（12V）

直流负载：12V灯光、水泵、冰箱、排风扇
```

### 锂电池选型
| 容量 | 续航能力 | 适用场景 | 价格区间 |
|------|---------|---------|---------|
| 100Ah/12V | 1.2kWh | 周末轻使用 | 3000-5000元 |
| 200Ah/12V | 2.4kWh | 长途旅行 | 6000-10000元 |
| 400Ah/12V | 4.8kWh | 长期旅居 | 12000-20000元 |

**注意**：必须使用磷酸铁锂（LiFePO4），具有BMS保护板

### 太阳能系统配置
| 太阳能功率 | 日发电量 | 适用场景 |
|-----------|---------|---------|
| 200W | 0.8-1kWh | 补电、维持 |
| 400W | 1.6-2kWh | 短途旅行 |
| 600W | 2.4-3kWh | 长途旅行 |
| 800W+ | 3.2kWh+ | 长期旅居/无市电 |

**组件选型**：
- 太阳能板：单晶硅，效率 ≥ 20%
- 控制器：MPPT类型，效率 ≥ 95%
- 安装：硬板（屋顶固定）/ 软板（折叠便携）

### 逆变器选型
| 功率 | 适用负载 | 注意事项 |
|------|---------|---------|
| 1500W | 电视、小家电、笔记本 | 不能带大功率感性负载 |
| 3000W | 微波炉、电饭煲、空调 | 标准配置 |
| 5000W | 大功率设备 | 需配合大容量电池 |

**推荐**：正弦波逆变器，功率因数0.8-1.0

### 底盘取电
| 方式 | 电流 | 适用场景 |
|------|------|---------|
| 点烟器 | 10A/120W | 临时补电 |
| 专用充电接口 | 30A/360W | 标准配置 |
| 太阳能互补 | 30A+200W | 弱底盘充电 |

## 安全保护
| 保护类型 | 规格 | 说明 |
|---------|------|------|
| 短路保护 | 直流空开 | 12V/30A |
| 过载保护 | 逆变器内置 | 自动降载 |
| 过放保护 | BMS管理 | 锂电池保护 |
| 漏电保护 | 漏电保护器 | 220V侧 |
| 接地保护 | 接地极+接地线 | 220V侧 |
