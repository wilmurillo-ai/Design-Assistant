# 空间布局详细规范

## 布局类型详解

### B型房车（Vanlife）
- 基于轻客/Van车型改装
- 典型长度：5-5.5米
- 空间利用率：50-60%
- 布局：驾驶室+后部生活区
- 优势：通过性好，可进地下车库
- 劣势：空间受限，无独立卫浴

**典型布局**：
```
[驾驶座] → [会客区(可变床)] → [厨卫区]
```

### C型房车（Cab-over）
- 重型载货车底盘+上装厢体
- 典型长度：6-8米
- 空间利用率：75-85%
- 布局：额头床+中置客厅+后置厨卫/床铺
- 优势：空间大，设施齐全
- 劣势：车身较高，通过性受限

**典型布局**：
```
[额头床] → [会客区] → [厨卫区] → [后床]
```

### 拖挂式房车（Trailer）
- 独立车厢，由SUV/皮卡牵引
- 典型长度：4-8米
- 空间利用率：80-90%（无驾驶舱占用）
- 优势：牵引车可单独使用
- 劣势：倒车入库需技巧

## 功能区尺寸标准

### 通道宽度
| 场景 | 最小宽度 | 推荐宽度 |
|------|---------|---------|
| 单人侧身通过 | 400mm | 500mm |
| 双人会车 | 600mm | 700mm |
| 轮椅通行 | 800mm | 900mm |

## 可转换家具设计原则

### 座椅变床（会客区）
```
白天：4人对坐卡座 + 升降桌
夜间：卡座拼床 + 桌板移除 → 1400×2000mm双人床
```

**转换步骤**：
1. 移除升降桌面
2. 放倒卡座靠背
3. 铺设床板（滑轨式）
4. 铺床垫 + 床品

### 驾驶室旋转座椅
- 驻车后前排座椅旋转270°
- 与后舱形成会客区
- 注意旋转路径不与方向盘干涉
