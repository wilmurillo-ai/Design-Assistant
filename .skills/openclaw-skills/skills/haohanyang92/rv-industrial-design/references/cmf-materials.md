## 材料选择

### 外墙板材料
| 材料 | 优点 | 缺点 | 价格区间 |
|------|------|------|---------|
| 铝合金复合板 | 防腐、轻量化 | 成本高 | 高 |
| FRP（玻璃钢） | 整体性好 | 重量大 | 中 |

## 工艺标准

### 车厢密封工艺
```
1. 板材拼接：型材卡槽 + 自攻螺钉（间距≤150mm）
2. 密封处理：接缝处填充改性硅烷胶（宽度≥15mm）
3. 防水卷材：整车底部 + 轮弧内侧
4. 排水设计：车身底部设排水孔，坡度≥3°
```

### 保温工艺
```
结构层（由内到外）：
1. 内饰板（8-10mm）
2. 防潮层（铝箔贴面）
3. 保温层（XPS 30-50mm）
4. 结构层（铝合金骨架/木板）
5. 防水层（防雨布/TPO）
6. 外蒙皮（FRP/铝合金板）
```

### 防腐工艺
| 部位 | 处理方式 |
|------|---------|
| 底盘 | 防锈涂层 + 装甲喷涂 + PVC护板 |
| 骨架 | 热镀锌/铝合金骨架 |
| 连接件 | 不锈钢螺栓 + 防腐垫片 |
| 线束过墙 | 硅胶防水套 + 端面防腐胶 |
