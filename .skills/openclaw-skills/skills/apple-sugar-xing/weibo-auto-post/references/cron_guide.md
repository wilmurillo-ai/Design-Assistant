# 微博定时任务配置指南

## 创建微博定时任务的 Cron 表达式

| 任务类型 | Cron 表达式 | 说明 |
|---------|------------|------|
| 早间热点速评 | `0 8 * * *` | 每天 8:00 |
| AI×生活 | `0 11 * * *` | 每天 11:00 |
| 身份标签型 | `0 14 * * *` | 每天 14:00 |
| AI工具实战 | `30 15 * * *` | 每天 15:30 |
| 争议辩论型 | `0 17 * * *` | 每天 17:00 |
| 深夜互动 | `0 21 * * *` | 每天 21:00 |

## Cron Job 创建命令

使用 `cron(action="add", ...)` 创建定时任务，示例：

```python
cron(
    action="add",
    job={
        "name": "微博V3-早间热点速评",
        "schedule": {"kind": "cron", "expr": "0 8 * * *", "tz": "Asia/Shanghai"},
        "sessionTarget": "isolated",
        "payload": {
            "kind": "agentTurn",
            "message": "【早间AI热点速评 · V3】发布微博。\n\n步骤：\n1. 搜索微博热搜榜+百度热搜，找AI/科技相关热点\n2. 查看 D:\\QClaw_workspace\\weibo_plan_v2.md 的执行记录，排除已发布话题\n3. 创作一条简短犀利的热点评微博（100-150字），必须有个人立场和观点\n4. 配图：用 Python PIL 制作科技感卡片图\n5. 三层话题标签：常驻 + 流量热搜 + 互动\n6. 浏览器自动化发布，链接记录到 D:\\QClaw_workspace\\weibo_plan_v2.md\n7. 清理临时文件"
        },
        "delivery": {"mode": "none"}
    }
)
```

## 定时任务执行要求

每次 cron 触发时，AI 需要：

1. **搜索热点**：确定当天具体话题
2. **查执行记录**：`D:\QClaw_workspace\weibo_plan_v2.md` 排除已发布
3. **创作内容**：按内容类型指南写作
4. **生成配图**：调用 `make_card.py`
5. **浏览器发布**：调用 `publish.py`
6. **记录执行**：写入执行记录
7. **清理文件**：删除 `C:\Users\13113\Pictures\weibo_*.png`

## 注意事项

- **坐标偏移**：微博页面窗口位置因人而异，首次使用后如有偏移，需手动校准 `publish.py` 中的坐标参数
- **账号风控**：如账号被限制，cron 任务会报错，需解除限制后重试
- **执行记录**：必须写入记录文件，否则同一话题会被重复发布
