# 微博内容类型指南

## 1. 早间热点速评（8:00）

**目标**：抢热搜流量，黄金30分钟内发布

**写作要点**：
- 简短犀利，100-150字
- 必须有个人立场和观点，禁止纯信息罗列
- 搜索当天微博热搜+百度热搜，找AI/科技相关上升话题

**三层标签**：
- 常驻：`#AI科普#` `#AI生活观察#`
- 流量：直接带热搜话题名（如`#谷歌AI#` `#马斯克#`）
- 互动：`#battle#` 或 `#看法#`

**结尾**：必须有具体互动问题

---

## 2. AI×生活（11:00）

**目标**：加入泛人群标签，触达泛娱乐人群

**写作要点**：
- 场景感+共鸣，语言像"朋友告诉你一个你不知道的事"
- 思考AI如何影响普通人日常生活（通勤/购物/娱乐/工作/情感）

**三层标签**：
- 常驻：`#AI科普#` `#AI生活观察#`
- 流量：`#生活#` `#职场#` `#效率#` `#日常#`
- 互动：`#看法#` 或 `#分享#`

**结尾**：必须有具体互动问题

---

## 3. 身份标签型（14:00）

**目标**：让用户主动转发，因为转发能"显示他们的身份/立场"

**写作要点**：
- 核心公式：观点+故事+身份认同句
- 例如："转发这条微博，说明你已经看懂了这个差距"
- 让人转发是为了说"我也是这样的人"

**三层标签**：
- 常驻：`#AI科普#` `#AI生活观察#`
- 流量：`#AI时代#` `#职场#` `#个人成长#` `#先知先觉#`
- 互动：`#看法#`

**结尾**：加金句

---

## 4. AI工具实战（15:30）

**目标**：实用性内容=收藏率高=算法认定"有用内容"

**写作要点**：
- 必须用具体数字说明效率提升（如"从2小时压缩到5分钟"）
- 格式：工具介绍→使用场景→操作步骤→效果展示
- **必须包含对比图**（使用前 vs 使用后）

**三层标签**：
- 常驻：`#AI科普#` `#AI生活观察#`
- 流量：`#AI工具#` `#实用技巧#` `#职场#` `#效率#`
- 互动：`#分享#`

**结尾**："你还想学什么AI工具？"

---

## 5. 争议辩论型（17:00）

**目标**：争议越大评论越多，算法判定"互动量高"

**写作要点**：
- 话题要有冲击力
- **必须包含AB投票**：`【投票】A. xxx  B. xxx`
- **必须发种子评论**：一条支持A，一条支持B
- 结尾加"评论区 battle，今晚不删评"

**三层标签**：
- 常驻：`#AI科普#` `#AI生活观察#`
- 流量：相关热搜话题 + `#科技#` `#人工智能#`
- 互动：`#battle#` `#看法#`

---

## 6. 深夜互动话题（21:00）

**目标**：深夜活跃用户，轻松个人化语气

**写作要点**：
- 开放式问题，要足够具体、有代入感
- 语气轻松、更个人化
- 问题要让人忍不住想说点什么

**配图要求**：
- 深夜风，深色背景+蓝绿渐变
- 用 `make_card.py --style midnight`

**三层标签**：
- 常驻：`#AI科普#` `#AI生活观察#`
- 流量：`#深夜互动话题#` `#生活#` `#想法#`
- 互动：`#看法#` 或 `#battle#`

**结尾**："评论区随便说，今晚不删评 🔥"
