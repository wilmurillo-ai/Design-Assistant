# Model Context Windows

已知模型的上下文窗口大小（token 数）。用于判断是否需要压缩记忆。

## 常见模型

| 模型 | 最大上下文 | 建议压缩阈值 (70%) |
|------|-----------|-------------------|
| qwen3.6-plus | 200,000 | 140,000 |
| qwen3.5-plus | 200,000 | 140,000 |
| qwen-plus | 32,000 | 22,400 |
| qwen-turbo | 8,000 | 5,600 |
| gpt-4o | 128,000 | 89,600 |
| gpt-4o-mini | 128,000 | 89,600 |
| claude-sonnet-4 | 200,000 | 140,000 |
| claude-haiku | 200,000 | 140,000 |

## 获取模型上下文

在 agent 环境中，通过 `session_status` 工具查看：
```
📚 Context: 45k/200k (22.5%)
```
分母即为该模型的最大上下文窗口。

## 阈值计算

```
警告线    = max_context × 0.50
压缩线    = max_context × 0.70
紧急线    = max_context × 0.85
```

## 注意事项

- 不同 provider 对同一模型可能有不同的上下文限制
- 实际可用上下文可能略低于标称值（系统消息占用）
- 图片、文件附件也会占用上下文空间
