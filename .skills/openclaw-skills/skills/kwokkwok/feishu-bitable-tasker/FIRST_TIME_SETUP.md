# 飞书应用配置指南

本文档供 AI 引导用户完成飞书应用的创建和权限配置。

## 创建飞书应用

1. 访问 https://open.feishu.cn/
2. 点击「创建应用」→「创建企业自建应用」
3. 填写应用名称和描述
4. 在应用详情页记下 `App ID` 和 `App Secret`

## 配置权限

在应用管理页面 →「权限管理」→「批量导入权限」，粘贴：

```json
{
  "scopes": {
    "tenant": [
      "bitable:app",
      "wiki:wiki",
      "docx:document"
    ]
  }
}
```

- `bitable:app` — 多维表格读写
- `wiki:wiki` — 知识库读写
- `docx:document` — 文档块读写（用于子页面列表）

## 创建知识库多维表格

**重要：** 必须在**知识库**中创建多维表格，不能使用独立的多维表格。

1. 打开飞书知识库
2. 在知识库中创建多维表格
3. 配置表格结构和字段

## 添加应用到多维表格

飞书应用需要在多维表格上手动授权，否则会报 `permission denied`。

1. 打开知识库中的多维表格
2. 点击右上角 **"..."** → **"更多"** → **"添加文档应用"**
3. 在弹出的对话框中**搜索你的应用名称**，选择并添加
4. **将权限设置为"可管理"**（默认为"可编辑"，权限不足）

> ⚠️ **关于权限级别：** 默认添加应用时权限为"可编辑"，但这不足以完成创建子节点等操作。必须手动将权限改为**"可管理"**。

> 注意：是在**多维表格节点级别**添加应用，不是在知识库空间设置中添加成员。

## 文档结构

配置完成后，创建文档时会自动生成如下层级：

```
知识库
  └── 多维表格 (你创建)
        └── "{表名}-任务文档" (自动创建)
              └── 2026-03           ← 月文档（自动创建/复用）
                    └── 2026-03-21    ← 日文档（自动创建/复用）
                          └── 任务标题   ← 任务文档
```

任务文档根节点（"{表名}-任务文档"）会在首次配置时自动创建，所有任务文档都在这个节点下按日期组织。
