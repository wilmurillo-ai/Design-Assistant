# 股票 API 参考

## 新浪财经 API

### 接口地址

```
http://hq.sinajs.cn/list=<symbol>
```

### 支持的市场

| 市场 | 前缀/后缀 | 示例 |
|------|----------|------|
| A 股 (上交所) | .ss | 600519.ss |
| A 股 (深交所) | .sz | 000001.sz |
| 港股 | rt_hk + 代码 | rt_hk00700 |
| 美股 | gb_ + 小写代码 | gb_baba |

### 返回数据格式

#### A 股

```
var hq_str_ss600519="贵州茅台，1750.00,1740.00,1755.00,1760.00,1745.00,..."
```

字段说明：
- 字段 0: 股票名称
- 字段 1: 今开
- 字段 2: 昨收
- 字段 3: 当前价
- 字段 4: 最高
- 字段 5: 最低
- 字段 8: 成交量 (股)
- 字段 9: 成交额 (元)

#### 港股

```
var hq_str_rt_hk00700="腾讯控股，350.00,5.00,1.45,..."
```

字段说明：
- 字段 1: 股票名称
- 字段 2: 当前价
- 字段 3: 涨跌额
- 字段 4: 涨跌幅%

#### 美股

```
var hq_str_gb_baba="Alibaba Group Holding Limited，85.50,2.30,2.77,..."
```

### 限制

- 无需 API key
- 无官方调用频率限制
- 建议添加随机数防缓存：`?rn=<timestamp>`
