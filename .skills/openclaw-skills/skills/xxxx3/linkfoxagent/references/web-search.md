# 实时与全网检索

共 1 个工具。使用 `@工具中文名` 语法在任务提示词中调用。

### @网页检索

实时与全网检索

工具中文名：网页检索
功能说明：关键词：网页搜索, 联网检索, 实时资讯, 信息查询, 搜索引擎,搜索, reddit, 社区, 舆论平台，站外，帖子，热帖, 微信公众号
功能说明：本工具底层为 Tavily Search，会自动进行网络搜索并提取对应的内容。除亚马逊、沃尔玛、eBay 等已有专业工具覆盖的场景外，所有互联网搜索（含微信公众号）都必须使用本工具。经过本工具检索的内容，无需再次遍历调用_linkfoxBrowser_webExtract	工具。提取回来的内容是非结构化的内容，不要使用沙箱进行分析，请直接进入总结环节。

**Prompt 模板：**

> 搜索{{今年}}{{巴黎时装周}}的热门{{服装}}款式信息

`{{}}` 中的内容为需要替换的参数。

**参数约束：**

- **搜索关键词**: 必填
