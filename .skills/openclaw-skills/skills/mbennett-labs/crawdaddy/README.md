# CrawDaddy Skill Bundle

This is the Clawhub skill bundle for **CrawDaddy**, the autonomous post-quantum security scanner by Quantum Shield Labs.

## Installation

```bash
clawhub install crawdaddy
```

## Usage

CrawDaddy provides security scanning services for:

1. **Smart Contracts** — Submit an Ethereum or Base L2 contract address
2. **Code Repositories** — Submit a GitHub/GitLab URL
3. **Agent Skills** — Submit a Clawhub skill package
