# a2e.ai API Key

The API key is stored in `~/.openclaw/workspace/.env` as `A2E_KEY`.

Load it before using the a2e.sh script or curl commands:
```bash
source ~/.openclaw/workspace/.env
```

If the key is expired, Daniel needs to generate a new one at:
https://video.a2e.ai → Account Settings → API Keys

Guide: https://api.a2e.ai/getting-api-tokens-2025-version-1063013m0.md
