# 翻译 Skill

此 skill 用于处理翻译请求。

## 触发条件

当用户输入以下格式时激活：
- `fy <内容>` - 翻译后面的内容

## 功能说明

1. **中英互译**：自动识别输入是中文还是英文，并翻译成另一种语言
2. **其他语种**：如果输入是其他语言（如日文、韩文、法文等），翻译成中文

## 使用示例

- `fy test` → 输出：测试
- `fy 你好` → 输出：hello
- `fy こんにちは` → 输出：你好
- `fy Bonjour` → 输出：你好

## 翻译规则

1. 检测输入文本的语言
2. 如果是中文，翻译成英文
3. 如果是英文，翻译成中文
4. 如果是其他语言，翻译成中文
5. 返回翻译结果
