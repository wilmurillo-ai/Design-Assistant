# 唯品会商品搜索

## 概述
唯品会商品搜索助手，根据关键词搜索商品并获取详细信息，包括商品名称、价格、品牌等关键信息。支持分页浏览和价格区间筛选功能。

## 功能特性

- 📦 **关键词搜索**：支持输入任意商品关键词进行搜索
- 🔄 **URL自动编码**：自动对搜索关键词进行URL编码处理
- 📊 **批量获取**：单次请求最多返回10个商品的详细信息
- 🎯 **详细信息**：返回商品ID、名称、价格、原价、折扣、卖点、品牌、商品链接等完整信息
- 📋 **结构化输出**：脚本输出JSON格式数据，AI格式化展示给用户
- 🌐 **优先使用 Markdown**：AI输出时优先使用Markdown表格格式，不支持Markdown的环境使用纯文本
- ⚠️ **异常处理**：完善的错误处理和用户提示机制
- 🔄 **降级方案**：如果详情接口失败，自动降级为仅显示商品ID
- 🔑 **自动登录触发**：检测到未登录时自动触发登录流程，无需用户手动请求
- 📄 **分页功能**：支持"下一页"和"上一页"浏览更多商品
- 💰 **价格筛选**：支持按价格区间筛选商品（如：100-300元）
- 📝 **商品详情查询**：支持回复"查询第X个商品"查看具体商品的详细信息

## 工作流程

### 步骤 1：检测登录状态（AI 必须执行）
在执行搜索前，AI 必须先检测登录状态：
- 检查 `~/.vipshop-user-login/tokens.json` 是否存在且有效
- **如果已登录**：直接执行步骤2
- **如果未登录**：必须自动触发登录流程（见下文"未登录自动处理"）

**重要**：不要直接执行搜索脚本，而是先检查登录状态。

### 未登录自动处理（AI 必须自动执行）

当检测到未登录时，AI **必须**按以下步骤自动处理，**不要等待用户手动请求**：

**步骤 1.1：提示用户**
```
检测到您尚未登录唯品会账户，准备为您启动登录流程。

💡 登录说明：
  - 将使用唯品会登录 SKILL 完成登录
  - 使用唯品会APP扫描二维码登录
  - 快捷便捷，安全可靠
```

**步骤 1.2：检查并安装 vipshop-user-login SKILL**
- 如果未安装，使用 `clawhub install vipshop-user-login` 安装
- 如果已安装，跳过此步骤

**步骤 1.3：自动触发登录流程**
调用 vipshop-user-login skill 或执行登录脚本

**步骤 1.4：等待登录完成**
等待用户扫码确认并完成登录

**步骤 1.5：确认登录成功后，继续执行步骤2**

### 步骤 2：接收输入
- 接受用户输入：商品搜索关键词（商品关键词）
- 确保关键词不为空

### 步骤 3：执行搜索脚本
- 执行：`python3 scripts/search.py <keyword>`
- 可选参数：`--page-offset <offset>`（分页）、`--price-min <min>` `--price-max <max>`（价格筛选）

### 步骤 4：展示结果
- 解析 JSON 数据并格式化输出
- **优先使用 Markdown 表格格式**，环境不支持时使用纯文本格式
- 始终显示全部 10 个商品，不截断或省略
- 显示字段：商品ID、商品图片、商品名（可点击）、品牌、价格、原价、折扣、卖点、商品链接
- 检查是否还有更多页，添加分页提示

### 步骤 5：分页和价格筛选
- **分页**：用户说"下一页"时，计算新偏移 `current_page * 10`；说"上一页"时，计算 `(current_page - 2) * 10`（仅在 current_page > 1 时）
- **价格筛选**：当用户指定新的价格区间时，将 pageOffset 重置为 0；在同一价格区间内分页时，保持价格筛选条件

### 脚本执行

要搜索商品，执行 Python 脚本：

```bash
python3 scripts/search.py <keyword> [--page-offset <offset>] [-p <offset>] [--price-min <min>] [--price-max <max>]
```

**示例：**
```bash
# 第一页（默认）
python3 scripts/search.py 连衣裙

# 第二页（偏移 = 10）
python3 scripts/search.py 连衣裙 --page-offset 10

# 按价格区间筛选（100-300）
python3 scripts/search.py 连衣裙 --price-min 100 --price-max 300

# 结合分页和价格筛选
python3 scripts/search.py 连衣裙 -p 10 --price-min 50 --price-max 100
```

**分页偏移计算：**
- 第1页：offset = 0（默认）
- 第2页：offset = 10
- 第N页：offset = (N - 1) * 10

**价格区间筛选：**
- `--price-min <value>`：筛选价格 >= value 的商品
- `--price-max <value>`：筛选价格 <= value 的商品
- 两个参数都是可选的，可以单独使用或一起使用
- 当用户更改价格区间时，pageOffset 自动重置为 0

## 接口信息

**登录态要求**：
- 使用搜索功能前，用户必须先通过 `vipshop-user-login` skill 登录
- `mars_cid` 作为请求参数从登录态文件读取
- `PASSPORT_ACCESS_TOKEN` 通过 Cookie 方式传入

### 搜索接口
- **URL**：`https://mapi-pc.vip.com/vips-mobile/rest/shopping/skill/search/product/rank`
- **方式**：GET
- **主要参数**：`keyword`（搜索关键词）、`mars_cid`（设备ID）、`pageOffset`（分页偏移）、`batchSize`（固定为10）、`channelId`（固定为1）
- **Cookie**：`PASSPORT_ACCESS_TOKEN`（访问令牌）
- **返回**：商品总数、商品列表（包含 `pid` 字段）

### 商品详情接口
- **URL**：`https://mapi-pc.vip.com/vips-mobile/rest/shopping/skill/product/module/list/v2`
- **方式**：GET
- **主要参数**：`productIds`（商品ID列表）、`scene`（固定为search）、`mars_cid`（设备ID）、`user_id`（用户ID）
- **Cookie**：`PASSPORT_ACCESS_TOKEN`（访问令牌）
- **返回**：商品详情列表（包含 `productId`、`title`、`price.salePrice`、`price.marketPrice`、`price.saleDiscount`、`price.sellTips`、`brandShowName`、`smallImage`、`squareImage`、`attrs` 等）

**注意**：搜索接口返回的 `pid` 字段值与详情接口的 `productId` 字段值相同，只是字段名称不同。

## 注意事项

1. **登录要求**：使用搜索功能前必须先通过 `vipshop-user-login` skill 登录，未登录时系统会自动处理
2. **单次限制**：每次最多返回10个商品的详细信息，可使用分页查看更多
3. **参数固化**：大部分接口参数已固化，无需手动调整
4. **网络要求**：需要正常的网络连接才能调用接口
5. **接口更新**：如果接口参数失效，会提示接口可能已更新
6. **URL编码**：所有中文和特殊字符会自动进行URL编码
7. **字段缺失**：如果某些字段为空，会自动跳过该字段的显示
8. **字段名称差异**：搜索接口和详情接口的字段名称不同（如 `pid` vs `productId`），但值相同
9. **详情接口无销量**：详情接口的返回数据中没有销售数量字段
10. **价格信息**：包含当前售价、市场价、折扣、销售提示等多个维度
11. **登录态管理**：脚本会自动读取 `~/.vipshop-user-login/tokens.json` 文件
12. **分页功能**：支持"下一页"和"上一页"命令浏览更多商品
13. **价格筛选**：支持按价格区间筛选商品，指定新价格区间时分页会重置到第一页
14. **显示完整性**：始终显示全部10个商品，不省略任何商品信息
15. **输出格式**：优先使用 Markdown 表格格式，不支持 Markdown 的环境使用纯文本
16. **商品详情查询**：支持用户回复"查询第X个商品"查看具体商品详情
17. **上下文记忆**：AI 需要记住上一次搜索的商品列表，以便查询具体商品详情

## 技术细节

- **实现语言**：Python 3
- **数据格式**：JSON（脚本输出）→ 格式化文本（AI 展示）
- **编码支持**：UTF-8，支持中文搜索
- **异常处理**：完整的网络异常和API错误处理
- **降级策略**：详情接口失败时自动降级为仅显示商品ID
- **脚本方式**：执行 `scripts/search.py` 脚本进行商品搜索
- **登录态管理**：自动读取 `~/.vipshop-user-login/tokens.json`，支持与 `vipshop-user-login` skill 集成
- **依赖库**：仅使用 Python 标准库（urllib、json、pathlib）
- **分页支持**：通过 pageOffset 参数实现分页，每页固定返回10个商品
- **价格筛选**：通过 priceMin 和 priceMax 参数实现价格区间筛选

## 常见问题

**Q: 为什么只返回10个商品的详细信息？**
A: 当前设置 batchSize=10，这是为了保证商品详情获取的准确性和响应速度。您可以使用"下一页"命令查看更多商品。

**Q: 支持哪些语言的搜索？**
A: 支持中文、英文等多种语言的关键词搜索，会自动进行URL编码处理。

**Q: 接口调用失败怎么办？**
A: 检查网络连接是否正常，如果持续失败可能是接口参数已更新，请联系管理员。如果详情接口失败，系统会自动降级为仅显示商品ID。

**Q: 商品ID可以做什么用？**
A: 商品ID是唯品会商品的唯一标识，可以用于商品详情查询、购物车添加、订单处理等多种场景。

**Q: 为什么有些商品没有显示所有字段？**
A: 某些商品可能缺少品牌、原价、折扣、销售提示等信息，系统会自动跳过空字段，只显示有值的信息。

**Q: 搜索接口返回的pid和详情接口的productId有什么区别？**
A: 它们的值是相同的，只是字段名称不同。搜索接口使用`pid`字段，详情接口使用`productId`字段，但这两个字段的值完全一样。

**Q: 可以按价格筛选商品吗？**
A: 是的，支持按价格区间筛选商品。您可以说"100-300元"或"价格区间100-400"来筛选商品。

**Q: 是否需要登录才能使用？**
A: 是的，必须先通过 `vipshop-user-login` skill 登录才能使用搜索功能。未登录时系统会自动为您处理登录。

**Q: 如何登录？**
A: 使用 `vipshop-user-login` skill 进行登录。您可以说"登录唯品会"或"唯品会登录"。如果您的系统中没有安装此 skill，系统会自动为您安装。

**Q: 如何查看更多商品？**
A: 在查看搜索结果后，您可以说"下一页"查看下一页商品，或说"上一页"返回上一页。

**Q: 价格筛选时分页会重置吗？**
A: 当您指定新的价格区间时，分页会自动重置到第一页。但在同一价格区间内使用"下一页"或"上一页"时，会保持当前的价格筛选条件。

**Q: 可以同时使用价格筛选和分页吗？**
A: 是的，可以在价格筛选的基础上使用分页功能。例如，先搜索"100-300元之间的连衣裙"，然后使用"下一页"查看该价格区间内的更多商品。

**Q: 如何查看具体商品的详细信息？**
A: 搜索完成后，回复"查询第X个商品"或"第X个"，即可查看该商品的详细信息。例如："查询第3个商品"。

**Q: 查询商品详情时提示"无搜索历史"怎么办？**
A: 这说明之前没有进行商品搜索，请先搜索商品。例如：先说"搜索连衣裙"，然后再查询具体商品详情。

**Q: 查询商品详情时提示"序号超出范围"怎么办？**
A: 请输入1-10之间的序号，每页只显示10个商品。如果商品序号大于10，请先使用"下一页"翻页。
