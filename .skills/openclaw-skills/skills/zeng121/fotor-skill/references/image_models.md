## Resolution Support

- `1k` -- Standard (all models)
- `2k` -- High (most models)
- `4k` -- Ultra (most models)
- `auto` -- Use model default

## Aspect Ratio Support

`1:1`, `16:9`, `9:16`, `4:3`, `3:4`, `3:2`, `2:3`, `21:9`, `auto`
