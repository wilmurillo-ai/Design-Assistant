# 推荐指数评分标准

## 评分维度（满分 10 分）

### 1. 网络热度（30%）
- 多平台提及频率
- 搜索指数趋势
- 话题标签热度

**评分标准：**
- 9-10分：全网爆款，现象级热度
- 7-8分：热门推荐，频繁刷屏
- 5-6分：小有名气，稳定讨论
- 3-4分：冷门小众，提及较少
- 1-2分：几乎无讨论

### 2. 用户口碑（40%）
- 好评率
- 差评集中点
- 体验满意度

**评分标准：**
- 9-10分：好评如潮，强烈推荐
- 7-8分：整体不错，略有瑕疵
- 5-6分：褒贬不一，评价分化
- 3-4分：差评较多，问题明显
- 1-2分：普遍踩雷，不推荐

### 3. 独特性（20%）
- 当地特色
- 不可替代性
- 体验独特度

**评分标准：**
- 9-10分：独一无二，必打卡
- 7-8分：特色鲜明，值得一去
- 5-6分：有一定特色，但可替代
- 3-4分：普通常见，缺乏亮点
- 1-2分：毫无特色，随处可见

### 4. 性价比（10%）
- 门票/消费价格
- 体验价值
- 时间成本

**评分标准：**
- 9-10分：物超所值，性价比高
- 7-8分：价格合理，体验匹配
- 5-6分：略贵但可接受
- 3-4分：价格偏高，体验一般
- 1-2分：价格虚高，严重不值

## 综合评分计算

总分 = 热度×0.3 + 口碑×0.4 + 独特性×0.2 + 性价比×0.1

## 评级标准

| 总分 | 评级 | 推荐度 |
|------|------|--------|
| 9.0-10 | S级 | 必去，强烈推荐 |
| 8.0-8.9 | A级 | 推荐，值得一去 |
| 7.0-7.9 | B级 | 可选，有时间去 |
| 6.0-6.9 | C级 | 一般，可跳过 |
| < 6.0 | D级 | 不推荐，避雷 |
