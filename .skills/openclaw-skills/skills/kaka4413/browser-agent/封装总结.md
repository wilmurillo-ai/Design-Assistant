### 学到的技术内容

#### 1️⃣ Chrome Agent 协议 / CDP 浏览器控制
**来源:** @xiangxiang103 - "测试了一下 Chrome 新版原生支持的 Agent 协议"

**关键知识点:**
- Chrome 新版原生支持 Agent 协议，比无头浏览器更方便
- 官方包有安全提示弹窗问题，会卡住无法使用
- WebSocket 直连是更靠谱的平替方案
- Windows 平台需要特殊兼容处理
- 应用场景：Notion 同步飞书、整理 GitHub star、查 Analytics 数据、删 Twitter 帖子

**封装到技能:**
- `scripts/browser_agent.py` - 完整的 CDP 客户端实现
- `scripts/session_manager.py` - WebSocket 会话保持机制
- `SKILL.md` - 故障排查章节（安全提示弹窗解决方案）

- 操作步骤：开启远程调试 → 连接 CDP → 执行命令

### 核心功能

| 功能 | 实现方式 | 对应学习点 |
|------|---------|-----------|
| CDP 直连 | WebSocket 连接，绕过安全弹窗 | @xiangxiang103 的 WebSocket 方案 |
| 会话保持 | 心跳机制，5 分钟超时自动重连 | CDP 超时问题解决方案 |
| Windows 兼容 | 固定用户数据目录，避免会话丢失 | Windows 平台兼容分支 |
| 多动作支持 | navigate/screenshot/click/type/eval | 应用场景多样化 |
| OpenClaw 集成 | 映射到 browser 工具 | 原生支持 |

### 使用方法

**命令行:**
```bash
# 检查连接
python skills/browser-agent/scripts/browser_agent.py --check

# 截图
python skills/browser-agent/scripts/browser_agent.py --url "https://example.com" --action screenshot

# 点击
python skills/browser-agent/scripts/browser_agent.py --url "https://example.com" --action click --selector "#button"
```

**OpenClaw 工具:**
```python
browser(action="start")
browser(action="navigate", url="https://x.com")
browser(action="snapshot", refs="aria")
browser(action="act", kind="click", ref="e123")
```
