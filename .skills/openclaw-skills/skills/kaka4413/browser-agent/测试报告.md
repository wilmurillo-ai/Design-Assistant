## 发现的问题

### 问题 1: Chrome WebSocket 403 错误

**现象:** 直接使用 `browser_agent.py` 脚本连接时出现 403 Forbidden

**错误信息:**
```
WebSocketBadStatusException: Handshake status 403 Forbidden
Rejected an incoming WebSocket connection from the http://127.0.0.1:18800 origin.
Use the command line flag --remote-allow-origins=http://127.0.0.1:18800
```

**原因:** Chrome 90+ 版本默认拒绝 WebSocket 连接，需要 `--remote-allow-origins` 标志

**解决方案:**
1. **推荐:** 使用 OpenClaw 内置 browser 工具（已正确配置）
2. **自行启动 Chrome:** 添加 `--remote-allow-origins=*` 标志
