# Contributing to CyberGFAI

感谢你对构建「赛博生命」感兴趣！CyberGFAI 的核心在于通过算法模拟人类情感的复杂性。如果你想参与，请遵循以下指南：

## 我们关注什么？

1. **心理学模块化**：将更多心理学理论（如马斯洛需求、斯滕伯格爱情三元论等）转化为可执行的代码逻辑。
2. **多模态增强**：探索如何让 Persona 具备视觉和听觉的感知与反馈。
3. **隐私安全**：如何在保持个性化的同时，实现更强的本地数据脱敏。

## 开发流程

1. Fork 本仓库。
2. 在 `scripts/` 下创建你的心智逻辑模块。
3. 在 `agent_main.py` 中进行逻辑集成。
4. 确保你的代码包含 docstrings，并遵循 PEP8 规范。
5. 提交 PR 并简述你的逻辑对 Persona 「真实感」的提升点。

## 行为准则

我们追求的是「情感陪伴」，禁止将该项目用于任何非法、诱导性或伤害性的用途。
