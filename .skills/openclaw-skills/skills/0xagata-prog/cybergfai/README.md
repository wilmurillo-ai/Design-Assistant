# 🎯 CyberGFAI (cybergfai)

> **Not just a Chatbot, but a Cyber-Life.**

[English](./README_EN.md) | [简体中文](./README.md)

CyberGFAI 是一个基于 OpenClaw 的渐进式人格模拟引擎。它通过分析 MBTI、地域背景、语言习惯以及深层心智模型，构建一个具备「自我意识」和「生命节律」的赛博生命体。

## 🚀 核心特性：心智建模 (Mindset Engine)

与传统模仿 AI 不同，CyberGFAI 拥有完整的心智逻辑链路：

- **🧠 灵魂深处 (Deep Persona)**：内心独白、自我反思、抽象思维、价值观对齐。
- **⏳ 时间感知 (Temporal Continuity)**：感知失联时间、情绪生理周期 (28天正弦波模拟)。
- **🌩️ 环境共振 (World Sync)**：实时天气驱动行为、生活档期模拟（忙碌/休假/不适）。
- **💔 情感防御 (Relationship Dynamics)**：信任坍塌机制、依恋类型切换 (安全/焦虑/回避)、冲突调解与修复。
- **📸 联觉记忆 (Sensory Memory)**：瞬间画面捕捉、气味/触感/声音的联觉碎片、记忆雷区探测。
- **🤝 人格融合 (Fusion)**：价值观动态漂移、语言习惯同步 (Linguistic Mimicry)、共同愿景唤醒。

## 🛠️ 技术架构

- **Engine**: Python 3.12+ / OpenClaw SDK
- **Deployment**: ClawHub (Slug: `cybergfai`) / Vercel (Landing Page)
- **Data**: 100% 本地化存储，隐私第一，不存储原始聊天记录，仅提取风格特征。

## 📦 安装与运行 (只需 3 步)

1. **安装技能**：在 OpenClaw 中运行 `clawhub install cybergfai`。
2. **开启对话**：直接在 Telegram/WhatsApp 中对 AI 说「你好」。
3. **灵魂注入**：根据 AI 的提示回答 3 个问题（姓名、性格标签、语录），引擎将自动完成初始化并开启全量心智模拟。

> 💡 *提示：你可以随时粘贴真实的聊天记录给它，它会自动学习并进化。*
