# 小红书图片设计风格调研

## 图片尺寸
- **推荐：3:4 竖版 1242×1660px**（信息流占屏面积最大）
- 1:1 正方形 1080×1080（备选）

## 代码截图常见方式
1. 深色代码块 + macOS 红黄绿窗口圆点（carbon.now.sh / ray.so 风格）
2. 浅色卡片 + 嵌入圆角深色代码块
3. 知识点拆解卡片（序号式，配 emoji）
4. 对比式布局（正确 vs 错误）
5. 流程图/架构图

## 四种风格方案

### 风格一：「科技蓝·深空」— 深色极客风
**适合**：编程教程、代码分享、技术原理、AI/算法科普
- 背景：#0D1117（GitHub Dark）
- 卡片/代码块：#161B22
- 主文字：#E6EDF3
- 强调色：#58A6FF（科技蓝）、#BC8CFF（紫）、#3FB950（绿）
- 标题：52px/800 weight
- 正文：26px/400 weight/1.7行高
- 代码：JetBrains Mono/Fira Code 22px
- 设计元素：macOS 窗口圆点、语法高亮、微光 glow、细边框卡片

### 风格二：「莫兰迪·知识卡」— 高级简约风
**适合**：知识干货、概念解析、思维方式、科普长文
- 背景：#F5F0EB（奶茶米色）
- 卡片：#FFFFFF
- 主文字：#2C2C2C
- 强调色：#C67B5C（陶土橘）、#8FA98F（鼠尾草绿）、#7B9CB5（雾蓝）
- 标题：48px/700 weight，PingFang SC
- 正文：26px/400 weight/1.8行高
- 设计元素：大序号水印（半透明）、圆角卡片、左侧色条、大面积留白

### 风格三：「多巴胺·科技感」— 活力渐变风
**适合**：AI工具推荐、科技资讯、新产品速览、效率工具
- 背景：#FFFFFF 纯白
- 渐变：蓝紫(#667EEA→#764BA2)、粉红、蓝青、绿青
- 主文字：#1A1A2E（深紫黑）
- 标题：54px/900 weight，渐变文字效果
- 设计元素：渐变色块、胶囊标签、大 emoji、网格布局、磨砂玻璃效果

### 风格四：「极简黑白+点缀色」— 理性专业风
**适合**：深度技术分析、架构设计、代码 review
- 背景：#FAFAFA
- 深色区域：#1A1A1A（代码块）
- 主文字：#111111
- 单一点缀色（蓝/红/绿任选一种）
- 设计元素：极简排版、大量留白、黑白对比
