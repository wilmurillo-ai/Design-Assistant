# 错题本管理指南

## 错题归因矩阵

每道错题按两个维度归类：

### 维度一：知识点
具体到知识点，而不是「数学错了」。
例：`数学 > 概率 > 条件概率计算`

### 维度二：能力层次
| 层次 | 说明 | 解决方案 |
|------|------|---------|
| 记忆 | 知识点没背住、公式忘了 | 重新记忆，加强复现频率 |
| 理解 | 背了但不懂为什么 | 重新理解推导过程 |
| 应用 | 懂了但不会用 | 多做同类型练习题 |
| 分析 | 复杂题不会拆解 | 练习解题框架，看解析 |

---

## 艾宾浩斯复习计划

错题按以下间隔复习，提高记忆留存率：

```
第1次复习：错题后第1天
第2次复习：第3天
第3次复习：第7天
第4次复习：第15天
第5次复习：第30天
→ 完成5次复习后标记为「已掌握」
```

## 错题本命令

| 命令 | 说明 |
|------|------|
| `查看错题本` | 列出所有错题 |
| `查看[科目]错题` | 按科目筛选 |
| `今日复习` | 推送今天到期的复习题 |
| `错题统计` | 薄弱知识点排行榜 |
| `删除错题 [编号]` | 标记为已掌握 |
| `清空错题本` | 清除所有记录（需确认） |

---

## 错题记录格式

记录时收集以下信息：
1. 科目（自动检测）
2. 知识点（自动标注，可修改）
3. 错误原因（从选项中选）：
   - 粗心计算错
   - 知识点忘了
   - 看错题目条件
   - 方法不对
   - 完全不会
