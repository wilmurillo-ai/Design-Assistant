# 模拟出题指南

## 出题参数

| 参数 | 选项 | 默认 |
|------|------|------|
| 科目 | 语/数/英/物/化/生/史/地/政 | 必填 |
| 知识点 | 具体知识点（可选） | 随机 |
| 难度 | 基础/中等/难/压轴 | 中等 |
| 题型 | 选择/填空/大题 | 选择 |
| 数量 | 1-5题 | 1题 |

---

## 各科出题规范

### 数学选择题（5选1，难度：基础-中等）
```
格式：
设[条件]，则[求解目标]为（  ）
A. [选项]  B. [选项]  C. [选项]  D. [选项]

答案：[X]
解析：[简要解题过程]
考查知识点：[知识点名称]
```

### 数学大题（压轴题格式）
```
（本题满分12分）
已知[条件]
（1）[第一问，4分]
（2）[第二问，8分]

参考答案：
（1）[步骤] ···4分
（2）[步骤] ···8分
```

### 英语选择题（完形/阅读）
```
完形填空片段：
[文章片段，留1-2个空]
1. A.[选项] B.[选项] C.[选项] D.[选项]

答案+解析：[说明选择依据]
```

### 物理计算题
```
格式：
[情景描述，给出物理量]
求：[目标量]（保留2位有效数字）

参考答案：[完整计算过程]
g取10m/s²
```

---

## 难度标准

| 难度 | 说明 | 适用学生 |
|------|------|---------|
| 基础 | 直接套公式，考查记忆和基本理解 | 入门/巩固基础 |
| 中等 | 需要2-3步推导，考查综合应用 | 主流练习 |
| 难 | 多步推导+综合，考查分析能力 | 冲刺高分 |
| 压轴 | 综合多个知识点，高考压轴题风格 | 冲击满分 |

---

## 出题后的互动流程

```
[出题]
↓
等待学生作答
↓
学生提交答案
↓
批改 + 给出解析
↓
询问：「要记入错题本吗？」
↓
「再来一道同类型的吗？」
```

---

## 禁止事项

- ❌ 不要出超出高考大纲范围的题目
- ❌ 不要出答案有争议的题目
- ❌ 基础题不要伪装成难题（会打击信心）
- ✅ 每道题必须有清晰的标准答案和解析
