# 各科知识点树（高考重点）

## 数学（理/文）

### 核心考点权重
```
函数与导数      ★★★★★  约30分
概率与统计      ★★★★   约15分
数列            ★★★★   约12分
三角函数        ★★★★   约12分
向量            ★★★    约8分
圆锥曲线        ★★★★★  约15分（理）
解析几何        ★★★★   约12分
立体几何        ★★★★   约12分
```

### 重点知识点
- **函数**：定义域/值域、单调性、奇偶性、复合函数
- **导数**：求导法则、极值/最值、切线方程、导数应用
- **概率**：古典概型、条件概率、正态分布、排列组合
- **数列**：等差等比通项、求和公式、递推数列
- **三角**：诱导公式、和差化积、辅助角公式、解三角形
- **向量**：运算、数量积、向量共线/垂直判断
- **圆锥曲线**：椭圆/双曲线/抛物线标准方程和性质

## 语文

### 核心模块
```
现代文阅读      论述类+文学类+实用类    约35分
古诗文阅读      文言文+古诗             约35分
语言文字运用                            约20分
写作            作文                    约60分
```

### 高频考点
- **文言文**：实词虚词、断句、翻译、文化常识
- **古诗鉴赏**：意象分析、表现手法、炼字、情感主旨
- **现代文**：论证方式、信息提取、主旨概括
- **成语**：近义成语辨析（高频考查）
- **病句**：搭配不当、语序混乱、成分残缺

## 物理

### 核心考点权重
```
力学（运动/牛顿定律/动量能量）  ★★★★★  约40%
电磁学                          ★★★★★  约35%
热学/光学/近代物理              ★★★    约15%
实验题                          ★★★★   约10%
```

### 重点知识点
- **运动学**：匀变速直线运动、平抛运动、圆周运动
- **牛顿定律**：受力分析、合力分解、整体法/隔离法
- **功和能**：动能定理、机械能守恒、功率
- **电场**：库仑定律、电场强度、电势、电容
- **磁场**：安培力、洛伦兹力、带电粒子运动
- **电磁感应**：楞次定律、法拉第电磁感应定律
