# 志愿填报参考指南

## 填报策略框架

### 冲稳保原则
- **冲刺**：历年分数线比你高5-15分的院校（30%录取概率）
- **稳妥**：历年分数线与你持平或低5分以内（70%录取概率）
- **保底**：历年分数线比你低10-20分的院校（>95%录取概率）

建议比例：冲2-3所，稳3-4所，保2-3所

## 选校维度

除分数外，重要考量因素：

| 维度 | 说明 |
|------|------|
| 城市 | 一线城市资源多，但竞争压力大 |
| 院校类型 | 985/211/双一流/普通本科 |
| 专业实力 | 院校排名≠专业排名，查专业教育排行 |
| 就业率 | 重点看专业对口就业率 |
| 保研率 | 有读研意向的要看 |
| 学费住宿 | 部分院校费用差异大 |

## 志愿填报时间节点

```
3-4月：了解目标院校和专业，收集信息
5月：关注招生政策变化
6月初：高考结束，估分定位
6月中：查看成绩和一分一段表
6月下：填报志愿（各省时间略有差异）
7-8月：录取结果公布
```

---

## 重要提醒

⚠️ **本数据仅供参考：**
- 院校录取分数每年波动
- 新高考省份实行平行志愿，规则与老高考不同
- 专业调剂要谨慎，尽量服从调剂以防落档
- 最终以当年官方发布的招生简章为准

**建议：** 结合省级招生考试院官网、阳光高考平台查询准确数据。
