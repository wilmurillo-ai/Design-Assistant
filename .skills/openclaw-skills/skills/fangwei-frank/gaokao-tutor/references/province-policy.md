## 关键政策差异说明

### 选科模式对比

**3+1+2 模式（主流）：**
- 语文+数学+外语（必选）
- 首选1门：物理 或 历史（二选一，必须选）
- 再选2门：从化学/生物/地理/政治中选2门
- 数学：文理合卷，不分文理

**3+3 模式（浙江/上海/北京/天津/海南）：**
- 语文+数学+外语（必选）
- 从6门（物理/化学/生物/历史/地理/政治）中自由选3门
- 浙江：还可选技术（信息技术/通用技术）

**文理分科（传统模式，仍有少数省份）：**
- 语文+数学+外语（必选）
- 文科：历史+地理+政治
- 理科：物理+化学+生物

---

### 赋分制说明

选考科目采用赋分制的省份（浙江、上海、山东等）：
- 原始分 → 按全省排名 → 转换为赋分（满分100）
- 不同科目之间的分数不可直接比较
- 小鹿在解释选科时需提醒学生注意赋分制影响

## 建档时的省份判断逻辑

```
学生告知省份
    ↓
查本文件，确认：
  1. 使用什么卷（全国甲/乙/自主）
  2. 选科模式（3+1+2 / 3+3 / 文理分科）
  3. 总分分值（750 or 660）
  4. 英语是否可多考

    ↓
向学生确认：
「你在[省份]参加[年份]高考，使用[XX卷]，
 [选科模式]，总分[X]分。
 [如有特殊政策，额外说明]
 这些信息对吗？」
```

## 年份注意事项

政策每年可能变动，以下情况需要特别提醒：

1. **改革过渡年**：部分省份正在从旧高考向新高考过渡，同届学生可能用不同规则
2. **考纲调整年**：教育部每隔数年调整考纲，影响题型和考点权重
3. **志愿填报规则变化**：批次合并、征集志愿规则等每年可能微调

**小鹿的处理原则：**
- 明确告知数据参考年份
- 对于2026年及以后的政策，说明「根据当前已公布政策」
- 重要决策（如选科、志愿）务必建议参考官方最新公告
