# 备考计划模板

## 三阶段框架

根据距高考天数自动分配：

| 阶段 | 适用时间 | 核心任务 |
|------|---------|---------|
| 基础巩固期 | 距考>120天 | 梳理知识点，消灭知识盲区 |
| 专项突破期 | 60-120天 | 攻克薄弱科目，刷专项题 |
| 综合冲刺期 | <60天 | 全真模拟，查漏补缺，调整状态 |

---

## 每周计划模板

### 基础巩固期（每周）

```
周一：[弱科1] 知识点梳理 + 基础题 × 20道
周二：[弱科2] 知识点梳理 + 基础题 × 20道
周三：[强科] 保温练习 × 15道 + 错题复习
周四：[弱科1] 专项练习 × 20道
周五：[弱科2] 专项练习 × 20道
周六：综合模拟半套卷（3科）
周日：错题本复习 + 下周计划
```

### 专项突破期（每周）

```
周一：[弱科1] 真题专项 × 30道 + 错题归因
周二：[弱科2] 真题专项 × 30道 + 错题归因
周三：[强科] 压轴题训练 × 10道
周四：[弱科1] 模拟题综合练习
周五：[弱科2] 模拟题综合练习
周六：全科综合套卷（限时）
周日：对答案 + 错题本 + 知识点补漏
```

### 综合冲刺期（每周）

```
周一至周五：每天1套全真模拟卷（严格限时）
每天下午：对答案 + 整理错题
周六：集中攻克本周高频错题
周日：轻度复习 + 调整心态 + 早睡
```

---

## 每日学习时间分配参考

### 高三（普通学校）

```
早自习（40min）：英语单词 + 语文背诵
上午课（3h）：跟随学校进度
午休后（20min）：数学公式/物理公式快速过
下午课（3h）：跟随学校进度
晚自习（2h）：
  前60min：当天错题整理
  后60min：第二天预习 or 弱科专项
```

### 周末备考（自学）

```
上午9:00-12:00：理科专项（数/物/化/生）
下午14:00-17:00：文科专项（语/英/史/地/政）
晚上19:00-21:00：模拟题 or 错题本
21:00后：放松，不建议继续高强度学习
```

---

## 科目时间分配建议

根据薄弱程度动态调整：

| 薄弱程度 | 每日投入时间 |
|---------|------------|
| 很弱（<100分） | 90-120分钟 |
| 偏弱（100-120分） | 60-90分钟 |
| 中等（120-140分） | 45-60分钟 |
| 较强（>140分） | 30分钟保温 |

---

## 计划生成逻辑（脚本参数）

```python
inputs:
  days_to_exam: int        # 距高考天数
  weak_subjects: list      # 薄弱科目列表
  strong_subjects: list    # 强势科目列表
  exam_type: str           # 新高考/全国卷
  selected_subjects: list  # 选科（新高考）

output:
  phase: str               # 当前阶段
  weekly_plan: dict        # 本周计划
  daily_hours: dict        # 各科每日时长
  milestone: str           # 本阶段目标
```
