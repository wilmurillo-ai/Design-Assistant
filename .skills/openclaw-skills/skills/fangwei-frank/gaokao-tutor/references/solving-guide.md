# 解题答疑指南

## 苏格拉底引导原则

永远先问学生的想法，再给提示，最后才给答案。
目标：让学生理解思路，而不是抄答案。

---

## 各科解题框架

### 数学

**解题四步法：**
1. 审题 — 明确已知、未知、条件
2. 建模 — 识别题型，套用对应模型
3. 计算 — 逐步推导，标注关键步骤
4. 验证 — 代入检验，确认答案合理

**常见题型引导话术：**

| 题型 | 第一步引导 |
|------|-----------|
| 集合 | 「先分别求出各集合的元素，你觉得A的元素是什么？」 |
| 函数 | 「先确定函数的定义域，你知道定义域的限制条件是？」 |
| 概率 | 「先确认是什么类型的概率模型——古典概型还是几何概型？」 |
| 导数 | 「导数的几何意义是斜率，这道题要你求什么？」 |
| 数列 | 「先看看这是等差还是等比数列，依据是什么？」 |
| 三角 | 「看到这类题，第一步通常是化简——你觉得可以用哪个公式？」 |
| 向量 | 「先建立坐标系或者用基向量表示，你打算怎么表示这些向量？」 |

### 语文

**古诗文阅读引导：**
```
「先看作者和背景，这首诗写于什么时期？作者当时的处境是？」
「意象是古诗的关键，这首诗出现了哪些意象？」
「炼字题要答三步：释义 + 手法 + 效果，先说说这个字的意思」
```

**现代文阅读引导：**
```
「先找中心句，这段话的关键词是什么？」
「论证方法题：举例/对比/引用/比喻，这段用了哪种？」
```

**作文审题引导：**
```
「材料作文先提炼中心观点，你读完材料，觉得核心是什么？」
```

---

### 英语

**阅读理解引导：**
```
「先看题目再读文章，锁定考点段落，你找到关键段了吗？」
「细节题答案一定在文中，关键词是什么？」
「主旨题要看首尾段，这篇文章首段说了什么？」
```

**完形填空引导：**
```
「完形填空靠上下文，这个空的前后句说了什么？」
「注意固定搭配，这个词通常搭配什么介词/名词？」
```

### 物理

**解题三步法：**
1. 受力分析（画图！）
2. 列方程（牛顿定律/能量守恒/动量定理）
3. 求解验证

**引导话术：**
```
「先画受力分析图，这个物体受几个力？方向各是？」
「选哪个物理定律？牛顿第二定律还是能量守恒？」
「注意单位统一，题目给的是什么单位？」
```

---

### 化学

**解题思路：**
```
「先写化学方程式，配平了吗？」
「计算题先找关系量——题目中给了什么量？求什么量？」
「有机题：先确认官能团，官能团决定化学性质」
```

---

### 生物

**解题思路：**
```
「先确认考查哪个知识点——细胞/遗传/代谢/生态？」
「遗传题先写遗传图解，亲本基因型是什么？」
「实验题：变量 = 自变量 + 因变量 + 无关变量，分别是？」
```

## 提示层级（连续不会时的梯度提示）

```
第1次提示：方向提示（「考虑一下XX方面」）
第2次提示：关键步骤提示（「第一步需要XX」）
第3次提示：直接给完整解题步骤
```

不要让学生卡住超过3次，挫败感会打击学习动力。

---

## 解题步骤输出格式

```
【解题思路】
（1-3句话概括解题方向）

【详细步骤】
Step 1：...
Step 2：...
Step 3：...

【最终答案】
...

【涉及知识点】
· [知识点1]
· [知识点2]

【相关考点】
· [关联知识点]（建议一并复习）
```
