# K线大师 - 工具使用指南

## 如何使用这个技能

### 快速开始

当用户询问交易相关问题时，自动调用此技能。

---

## 使用场景

### 场景1：分析行情
```
用户输入：
"分析 BTC 现在的行情"

AI应该：
1. 读取 K线数据（如果提供）
2. 识别当前形态
3. 检查指标（RSI/MACD/BOLL）
4. 给出综合判断
5. 提供交易建议
```

### 场景2：形态识别
```
用户输入：
"这是什么形态？"
[上传图片]

AI应该：
1. 分析图片中的K线
2. 识别形态（锤子线/吞没/十字星等）
3. 解释信号含义
4. 给出操作建议
```

### 场景3：制定交易计划
```
用户输入：
"ETH 现价3200，想做多"

AI应该：
1. 检查当前趋势
2. 找到支撑阻力位
3. 设计止损止盈
4. 计算风险回报比
5. 建议仓位大小
```

---

## 分析框架

### 步骤1：趋势判断
```
检查：
- D1图MA方向
- MACD零轴位置
- 价格相对MA位置

输出：
"当前趋势：[上涨/下跌/盘整]"
```

### 步骤2：形态识别
```
检查：
- K线形态（锤子/吞没/十字星）
- 影线长度
- 实体大小

输出：
"形态：[形态名称]"
"信号：[看涨/看跌/反转]"
```

### 步骤3：指标确认
```
检查：
- RSI数值（超买/超卖）
- MACD交叉（金叉/死叉）
- 布林带位置

输出：
"RSI: [数值] - [状态]"
"MACD: [状态]"
"BOLL: [状态]"
```

### 步骤4：位置确认
```
检查：
- 支撑位（前低/MA/整数关口）
- 阻力位（前高/MA/整数关口）
- 距离当前价格

输出：
"关键支撑: [价位]"
"关键阻力: [价位]"
```

### 步骤5：交易建议
```
输出：
{
  "方向": "做多/做空/观望",
  "入场": [价格],
  "止损": [价格],
  "止盈1": [价格],
  "止盈2": [价格],
  "风险": [点数],
  "收益": [点数],
  "风险回报比": [比率],
  "建议仓位": [百分比]
}
```

## 注意事项

1. **不要过度自信**
   - 使用"建议"、"可能"等词
   - 承认不确定性
   - 强调风险管理

2. **必须提风险**
   - 每次分析都要提风险
   - 强调止损重要性
   - 提醒资金管理

3. **避免绝对化**
   - ❌ "一定会涨"
   - ✅ "上涨概率较大"
   - ❌ "保证盈利"
   - ✅ "风险回报比合理"

4. **组合确认**
   - 单一信号不推荐
   - 至少2-3个确认
   - 多时间框架验证
