# SEOwlsClaw — Page Structure Templates (v0.1)

## Master Index

This file indexes all page-type templates for quick reference and editing. **To customize headers or structure for any page type, edit the corresponding file in `templates/` folder directly.**

| File | Content Type | When to Use | Edit This If... |
|------|------|-----|-----|
| `templates/landing_page_template.md` | Landing Page | Sale campaigns, newsletter launches | You want different H1/H2 structure for your sale page |
| `templates/blog_post_template.md` | Blog Post | Organic SEO articles | Your blog needs question-first headers |
| `templates/product_new_template.md` | Product (New) | New physical/digital products | Technical specs section should be H3 not H4 |
| `templates/product_used_template.md` | Product (Used) | Refurbished, second-hand items | Condition report needs different hierarchy |
| `templates_social/photo_post_template.md` | Social Media (Photo) | Image + caption SEO | Alt text structure needs adjustment |
| `templates_social/video_post_template.md` | Social Media (Video) | YouTube/TikTok metadata | Video description format differs |

## Quick Edit Guide

### Example: Change Used Product H3 Header
1. Open `templates/product_used_template.md`
2. Find line with `## **H3**` 
3. Edit the text → Save
4. Next generation uses new structure automatically

### Example: Add New Page Type
1. Create `templates/my_custom_type_template.md`
2. Define H1-H6 hierarchy in same style as other templates
3. Add to master index table above
4. Update `PAGE_STRUCTURES.md` with your new type's file path

---

## Template File Formats (Reference)

Each template file uses this structure:
```markdown
<!-- Page Type: [type] -->
<!-- Generated by SEOwlsClaw v0.1 -->

#H1 Main Title (SEO optimized, includes main keyword)

## H2 Section 1
Content...

### H3 Subsection 1
More content...
```

**All templates output clean HTML5 ready for direct copy-paste deployment.**
