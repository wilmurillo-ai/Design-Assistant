# MGTV Skill - 智能链接推送功能

## 核心逻辑

**一句话说明**：人类用户无法正常观看视频时，自动推送播放链接。

## 环境检测

脚本会自动检测运行环境，判断是否能够打开浏览器：

### 无头环境（直接推送链接）

检测到以下情况时，**不会尝试打开浏览器**，直接推送链接：

- ✅ `HEADLESS=true` 环境变量
- ✅ `CI=true` 环境变量（CI/CD 环境）
- ✅ Linux 无图形界面（`DISPLAY` 未设置）
- ✅ Electron 环境

### 正常环境（尝试打开浏览器）

- ✅ macOS：有图形界面
- ✅ Windows：有图形界面
- ✅ Linux：有图形界面（`DISPLAY` 已设置）

## 使用场景

### 1. 本地开发（正常环境）
```bash
node scripts/search-mgtv.js --query "乘风破浪的姐姐"
# → 自动打开浏览器播放
```

### 2. 服务器环境（无头）
```bash
HEADLESS=true node scripts/search-mgtv.js --query "乘风破浪的姐姐"
# → 推送链接，手动复制打开
```

### 3. CI/CD 环境
```bash
CI=true node scripts/search-mgtv.js --query "乘风破浪的姐姐"
# → 推送链接，手动复制打开
```

### 4. Docker 容器
```bash
docker run --rm my-image node scripts/search-mgtv.js --query "乘风破浪的姐姐"
# → 推送链接，手动复制打开
```

## 技术实现

### 环境检测函数

```javascript
function isHeadlessEnvironment() {
  if (process.env.HEADLESS === 'true') return true;
  if (process.env.CI === 'true') return true;
  if (process.env.DISPLAY === undefined && process.platform === 'linux') return true;
  if (process.env.ELECTRON_RUN_AS_NODE === '1') return true;
  return false;
}
```

## 优势

1. **智能判断**：自动识别环境，无需手动配置
2. **用户体验**：能打开浏览器时自动打开，不能时推送链接
3. **清晰提示**：无头环境下输出格式化的链接和操作提示
4. **广泛适用**：支持本地、服务器、CI/CD、Docker 等多种场景
