# 微信公众号标题公式库

## ⚠️ 硬性限制
- 标题严格 ≤ 32 字（含标点符号）
- 用「·」或「｜」分隔多个亮点
- 禁止"震惊体"（震惊/厉害了/解密）→ 会被判标题党限流

## 六大标题公式

### 1. 关键词法（推荐·利于搜索推荐）
- 带核心关键词，命中用户搜索意图
- 示例：`AI日报3.15｜百万上下文GA·世界模型融10亿·谷歌30%代码AI写`

### 2. 数据法（推荐·吸引眼球）
- 用具体数字制造信息密度感
- 示例：`财经周报3.15｜油价破百·IEA史上最大释储·下周超级事件周`

### 3. 反差法
- 制造认知冲突
- 示例：`做了10年公众号，小号终于迎来第一篇10万+`

### 4. 问号式
- 激发好奇心
- 示例：`为什么谷歌30%的代码已经由AI生成？`

### 5. 感叹号式（慎用）
- 制造紧迫感，但不宜过度
- 示例：`油价破百！全球能源危机来了`

### 6. 情绪点法
- 一眼看上去就有情绪共鸣
- 示例：`70位硅谷程序员的集体焦虑：我们几乎不写代码了`

## 日报/周报标题模板

```
[类型][日期]｜[亮点1]·[亮点2]·[亮点3]
```

示例：
- `AI日报3.15｜百万上下文GA·世界模型融10亿·谷歌30%代码AI写`
- `财经周报3.15｜油价破百·IEA史上最大释储·下周超级事件周`
- `AI周刊3.15｜Claude可视化·Hume零幻觉TTS·CodeSpeak规格编程`

## 字数检查技巧

写完标题后用 python 检查：
```python
len("你的标题") <= 32  # 必须为 True
```
