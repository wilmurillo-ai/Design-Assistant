## 🔍 关键发现

### 1. 作品描述框逻辑

**发现：**
- 快手使用单个文本框填写「作品描述」
- 标题和关键词写在同一个框中
- 关键词格式：`#关键词`，空格分隔
- 最多支持 4 个关键词

**正确格式示例：**
```
明天更好 #美好的 #欢快 #一天的
```

**错误格式（会导致关键词识别失败）：**
```
明天更好 美好的，欢快，一天的  # 逗号分隔❌
明天更好 #美好的，#欢快的，#一天的  # 逗号分隔❌
```

### 2. 封面设置流程

**完整流程：**
1. 点击「封面设置」按钮 → 打开封面弹窗
2. 切换到「上传封面」标签（默认是「封面截取」）
3. 点击「上传图片」按钮 → 触发文件选择
4. 选择图片文件 → 等待上传完成
5. 点击「确认」按钮 → 应用封面

**注意事项：**
- 封面弹窗有两个标签：「封面截取」和「上传封面」
- 必须切换到「上传封面」标签才能看到上传按钮
- 上传完成后必须点击「确认」，否则封面不会应用

### 3. 页面元素识别

**关键选择器（2026-04-03 实测）：**

| 元素 | 选择器 | 说明 |
|------|--------|------|
| 上传视频按钮 | `button:has-text("上传视频")` | 触发文件选择 |
| 作品描述框 | `textarea[placeholder*="作品描述"]` | 标题 + 关键词 |
| 封面设置按钮 | `button:has-text("封面设置")` | 打开封面弹窗 |
| 上传封面标签 | `div:has-text("上传封面")` | 弹窗内标签页 |
| 上传图片按钮 | `button:has-text("上传图片")` | 触发文件选择 |
| 确认按钮 | `button:has-text("确认")` | 应用封面 |
| 发布按钮 | `button:has-text("发布")` | 最终发布 |

## ⚠️ 注意事项

### 关键词限制
- 最多 4 个关键词
- 每个关键词前加 `#`
- 关键词之间用空格分隔
- 关键词后不需要额外符号

### 封面图片要求
- 格式：PNG/JPG
- 建议尺寸：1080x1920 (9:16) 或 1080x1080 (1:1)
- 文件大小：建议 < 5MB

### 视频上传超时
- 默认超时：300 秒（5 分钟）
- 大文件可通过 `--wait-timeout` 参数调整
- 示例：`--wait-timeout 600`（10 分钟）
