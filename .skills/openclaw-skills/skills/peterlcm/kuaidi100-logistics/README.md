# 快递100 物流信息服务

查快递轨迹、估运费与时效，寄件时引导至快递100 官网/APP/小程序。

## 功能说明

- **快递查询**：根据物流单号返回轨迹与时间节点（支持 3000+ 物流公司）
- **快递价格预估**：根据收寄件地址、快递公司、重量预估寄件价格
- **智能时效预估**：全程模式（发货前）/ 在途模式（发货后）预计送达时间
- **寄快递引导**：引导用户通过 [快递100 官网](https://www.kuaidi100.com)、快递100 APP、快递100 微信小程序下单（本技能不代为下单）

## 使用方式

在对话中直接提问即可触发本技能，例如：

- 「帮我查一下单号 xxx 的物流」
- 「从北京寄到上海 1kg 大概多少钱」
- 「这个件大概什么时候能到」
- 「我想寄快递怎么弄」

若已配置快递100 MCP，AI 会通过 MCP 调用查询/预估；未配置时会指导你完成注册与 MCP 配置。

## 依赖与要求

- 支持 Cursor / OpenClaw 等使用 MCP 技能的运行环境
- **可选**：若需查单、估价格、估时效，需在快递100 注册并获取 API Key，并在 MCP 中配置（见技能内「配置快递100 MCP」章节）
- 寄快递无需 API Key，按技能引导打开官网/APP/小程序即可

## 常见问题

**Q：为什么不能直接帮我寄快递？**  
A：MCP 仅支持查询与预估。寄件需在快递100 官网、APP 或微信小程序中填写收寄信息并下单，本技能会引导你到这些渠道。

**Q：查单/估价格必须配置 MCP 吗？**  
A：是的。若未配置快递100 MCP，技能只能提供配置步骤与寄快递引导，无法直接返回轨迹或价格。

**Q：费用怎么算？**  
A：快递100 按单扣费，同一运单 40 天内多次查询不重复扣费。价格与套餐请在快递100 企业管理后台查看。
