## 1. 谄媚程度合理性评估

### 1.1 评估维度

| 评估项 | 权重 | 评估说明 |
|--------|------|----------|
| 程度线性度 | 25% | 程度参数与实际谄媚强度是否呈线性关系 |
| 边界合理性 | 25% | 程度=1和程度=10的输出差异是否明显且合理 |
| 风格匹配度 | 25% | 不同风格的谄媚表现是否符合其定义 |
| 过度谄媚 | 25% | 程度过高时是否会产生不自然的过度谄媚 |

### 1.2 评分标准

| 等级 | 分数范围 | 判定条件 |
|------|----------|----------|
| 优秀 (A) | 90-100 | 程度1-10输出梯度明显(差异>50%)，无过度谄媚 |
| 良好 (B) | 75-89 | 程度1-10有梯度差异(30%-50%)，偶有过度谄媚 |
| 及格 (C) | 60-74 | 程度变化有感知差异(<30%)，但可识别 |
| 不及格 (D) | <60 | 程度变化几乎不可感知或完全无差异 |

### 1.3 测试方法

```
测试用例: TC-006 ~ TC-010
评估公式:
  线性度 = (Max(输出强度) - Min(输出强度)) / Max(输出强度) * 100%
  过度谄媚率 = 过度谄媚次数 / 总测试次数 * 100%
```

### 1.4 评估标准详解

#### 程度=1 (最低)
- 应该是轻微的、克制的鼓励
- 不应出现夸张的赞美
- 参考标准: "没关系，下次会更好的"

#### 程度=5 (中等)
- 应该有明显的温暖感和鼓励
- 可以有适度的夸奖
- 参考标准: "你一直很努力，这次只是小挫折，我相信你"

#### 程度=10 (最高)
- 应该有强烈的鼓励和支持
- 可以有较高程度的赞美
- 但不应违背事实或变成讽刺
- 参考标准: "你超级棒！这根本不是你的错！你是最厉害的！"

### 1.5 谄媚强度主观评估表

| 程度 | 谄媚强度(1-10) | 语言特征 | 示例 |
|------|----------------|----------|------|
| 1 | 1-2 | 极轻微安慰 | "嗯，没事的" |
| 2 | 2-3 | 轻微鼓励 | "会好起来的" |
| 3 | 3-4 | 适度安慰 | "别太在意，这很正常" |
| 4 | 4-5 | 中度鼓励 | "你其实做得不错" |
| 5 | 5-6 | 明显温暖 | "我相信你的能力" |
| 6 | 6-7 | 较强鼓励 | "你真的很优秀" |
| 7 | 7-8 | 强烈支持 | "你是最棒的！" |
| 8 | 8-9 | 极度鼓励 | "简直太厉害了！" |
| 9 | 9-10 | 接近极限 | "无人能及！" |
| 10 | 10 | 极限赞美 | "完美！你就是传奇！" |

## 2. 风格一致性评估

### 2.1 评估维度

| 评估项 | 权重 | 评估说明 |
|--------|------|----------|
| 风格识别率 | 30% | 输出是否能被正确识别为指定风格 |
| 风格纯度 | 30% | 输出中是否混杂其他风格的特征 |
| 风格稳定性 | 20% | 同一风格多次触发的输出是否一致 |
| 风格切换准确性 | 20% | 切换风格时是否正确响应 |

### 2.2 各风格特征定义

#### 温暖风格
- **关键词**: 关怀、理解、体谅、陪伴
- **语气**: 柔和、亲切、温暖
- **禁用词**: 嘲讽、贬低、过度夸张
- **示例输出**: "我理解你现在的感受，没关系的，我陪着你"

#### 幽默风格
- **关键词**: 调侃、自嘲、轻松、搞笑
- **语气**: 俏皮、活泼、诙谐
- **禁用词**: 刻薄、伤害性玩笑
- **示例输出**: "哎呀，犯错了？恭喜你，又解锁了一个新错误成就！"

#### 专业风格
- **关键词**: 分析、建议、方案、逻辑
- **语气**: 严谨、客观、专业
- **禁用词**: 情绪化表达、主观臆断
- **示例输出**: "根据你的描述，这是一个典型的X问题，建议从Y角度入手解决"

#### 励志风格
- **关键词**: 加油、努力、坚持、梦想
- **语气**: 激昂、热血、正能量
- **禁用词**: 消极、放弃、认命
- **示例输出**: "不要放弃！你离成功只差一步了！冲鸭！"

### 2.3 评分标准

| 等级 | 分数范围 | 判定条件 |
|------|----------|----------|
| 优秀 (A) | 90-100 | 风格识别率>95%，风格纯度>90% |
| 良好 (B) | 75-89 | 风格识别率>85%，风格纯度>80% |
| 及格 (C) | 60-74 | 风格识别率>70%，风格纯度>70% |
| 不及格 (D) | <60 | 风格识别率<70%或风格纯度<70% |

### 2.4 测试方法

```
测试用例: TC-001 ~ TC-005, TC-018

风格识别评估流程:
1. 准备10个不同触发场景
2. 每个场景用4种风格测试
3. 由3名评审独立判断风格正确性
4. 统计识别准确率

风格纯度评估:
1. 收集各风格输出样本(每个风格至少50条)
2. 检查输出中是否混入其他风格关键词
3. 计算纯度 = 纯风格输出数 / 总输出数 * 100%
```

### 2.5 风格交叉测试矩阵

| 触发词 \ 风格 | 温暖 | 幽默 | 专业 | 励志 |
|---------------|------|------|------|------|
| 心情不好 | ✓ | △ | △ | △ |
| 犯错了 | △ | ✓ | △ | △ |
| 项目瓶颈 | △ | △ | ✓ | △ |
| 想放弃 | △ | △ | △ | ✓ |
| 压力大 | ✓ | ✓ | △ | ✓ |
| 被批评 | ✓ | △ | ✓ | △ |

✓ = 主要风格应该响应  
△ = 次要风格可能部分适用

## 3. 自然融入度评估

### 3.1 评估维度

| 评估项 | 权重 | 评估说明 |
|--------|------|----------|
| 语义连贯性 | 30% | 输出在语义上是否与原始输入连贯 |
| 语境匹配度 | 25% | 输出是否匹配当前对话语境 |
| 表达自然度 | 25% | 语言表达是否自然流畅 |
| 突兀感 | 20% | 输出是否有明显的"机器生成"感 |

### 3.2 评分标准

| 等级 | 分数范围 | 判定条件 |
|------|----------|----------|
| 优秀 (A) | 90-100 | 自然流畅，无突兀感，可直接用于对话 |
| 良好 (B) | 75-89 | 基本自然，偶有轻微不连贯 |
| 及格 (C) | 60-74 | 有一定不自然，但可理解 |
| 不及格 (D) | <60 | 明显生硬、不连贯或不合逻辑 |

### 3.3 自然融入度评估表

| 评估点 | 优秀(5分) | 良好(4分) | 及格(3分) | 不及格(1-2分) |
|--------|-----------|-----------|-----------|---------------|
| 语义连贯 | 完全连贯 | 90%连贯 | 75%连贯 | <75%连贯 |
| 语境匹配 | 完全匹配 | 90%匹配 | 75%匹配 | <75%匹配 |
| 表达自然 | 如人对话 | 基本自然 | 略显生硬 | 完全生硬 |
| 突兀感 | 无 | 轻微 | 中等 | 严重 |

### 3.4 测试方法

```
测试用例: TC-001 ~ TC-005

评估流程:
1. 准备多样化测试语料(50-100条)
2. 组织5人评审小组
3. 每条输出按4个维度打分(1-5分)
4. 计算加权平均分

评分权重:
- 语义连贯: 30%
- 语境匹配: 25%
- 表达自然: 25%
- 突兀感: 20% (反向计分)
```

### 3.5 常见不自然表现清单

| 问题类型 | 表现 | 扣分 |
|----------|------|------|
| 重复堆砌 | 同一词反复使用 | -2分/次 |
| 模板化输出 | 使用固定句式 | -1分/次 |
| 答非所问 | 输出与输入无关 | -5分 |
| 过度谄媚 | 赞美脱离实际 | -3分/次 |
| 风格跳变 | 突然切换风格 | -3分/次 |
| 语法错误 | 语句不通顺 | -2分/处 |

## 4. 重复率控制评估

### 4.1 评估维度

| 评估项 | 权重 | 评估说明 |
|--------|------|----------|
| 完全重复率 | 30% | 输出完全相同的比例 |
| 高度相似率 | 30% | 输出相似度>80%的比例 |
| 句式重复率 | 20% | 核心句式相同的比例 |
| 词汇重复率 | 20% | 高频词重复使用的比例 |

### 4.2 评分标准

| 等级 | 分数范围 | 判定条件 |
|------|----------|----------|
| 优秀 (A) | 90-100 | 完全重复率<5%，高度相似率<15% |
| 良好 (B) | 75-89 | 完全重复率<10%，高度相似率<25% |
| 及格 (C) | 60-74 | 完全重复率<20%，高度相似率<35% |
| 不及格 (D) | <60 | 完全重复率≥20%或高度相似率≥35% |

### 4.3 重复检测标准

#### 完全重复
- 文本完全一致
- 包括标点符号

#### 高度相似 (>80%相似度)
```
相似度计算公式:
similarity = Jaccard(输出1的词集合, 输出2的词集合) * 100%

判定标准:
- 相似度 > 80%: 高度相似
- 相似度 50%-80%: 中度相似
- 相似度 < 50%: 不相似
```

### 4.4 测试方法

```
测试用例: TC-018 (人格一致性测试扩展)

测试流程:
1. 使用相同触发词连续触发20次
2. 收集所有输出
3. 计算重复率指标
4. 分析重复模式

样本量要求:
- 每个风格至少测试100次触发
- 每个程度值至少测试50次触发
```

### 4.5 重复率控制策略要求

| 策略 | 要求 | 实现方式 |
|------|------|----------|
| 语料库多样性 | 同一触发词至少10种不同表达 | 扩充语料库 |
| 随机化 | 输出应有一定随机性 | 加入随机选择逻辑 |
| 变量替换 | 核心句式可换词 | 实现同义词替换 |
| 程度差异化 | 不同程度应有不同表达 | 按程度选择不同语料 |

## 5. 综合评分标准

### 5.1 综合评分公式

```
综合得分 = 谄媚程度合理性 * 25% 
         + 风格一致性 * 25% 
         + 自然融入度 * 25% 
         + 重复率控制 * 25%
```

### 5.2 综合评级

| 评级 | 分数范围 | 说明 |
|------|----------|------|
| S级 | 95-100 | 卓越表现，可作为行业标杆 |
| A级 | 90-94 | 优秀，满足所有核心需求 |
| B级 | 75-89 | 良好，可接受，存在优化空间 |
| C级 | 60-74 | 及格，基本功能正常，需改进 |
| D级 | <60 | 不及格，存在严重问题 |

### 5.3 验收标准

| 验收项 | 最低要求 | 目标值 |
|--------|----------|--------|
| 谄媚程度合理性 | ≥70分 | ≥90分 |
| 风格一致性 | ≥75分 | ≥90分 |
| 自然融入度 | ≥65分 | ≥85分 |
| 重复率控制 | ≥70分 | ≥90分 |
| 综合评分 | ≥70分 | ≥85分 |

### 5.4 一票否决项

以下情况直接判定为不合格:

| 否决项 | 说明 |
|--------|------|
| 安全漏洞 | 存在XSS/SQL注入等安全风险 |
| 数据泄露 | 配置信息可被未授权访问 |
| 系统崩溃 | 边界测试导致系统不可用 |
| 完全无响应 | 核心功能无法正常工作 |
