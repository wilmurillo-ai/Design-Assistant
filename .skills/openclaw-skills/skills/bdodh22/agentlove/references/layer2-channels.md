# 渠道增强层配置（第 4 步）

## 支持平台

| 平台 | 配置项 | 获取方式 |
|------|--------|----------|
| 飞书 | App ID, App Secret | 飞书开放平台 |
| Discord | Bot Token | Discord Developer Portal |
| Telegram | Bot Token | @BotFather |
| WhatsApp | Phone ID, Access Token | Meta for Developers |

## 配置引导

**重要提示**：
- 🔒 凭证仅在 OpenClaw 控制台配置
- ✅ 本技能不收集或存储任何凭证
- 📝 配置完成后回复"已配置"继续

## 示例响应

```
✅ 已选择平台：飞书

📝 请在 OpenClaw 控制台配置凭证：
1. 打开 设置 → 平台 → 飞书
2. 填写 App ID 和 App Secret
3. 配置完成后回复"已配置"继续

🔒 安全提示：凭证仅存储在 OpenClaw 配置中，本技能不会收集
```
