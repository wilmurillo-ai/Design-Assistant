# 基础层配置（第 3 步）

## 配置项

| 功能 | 选项 | 默认值 |
|------|------|--------|
| 流式输出 | 开启 / 关闭 | 开启 |
| 记忆功能 | 关闭 / 记忆增强 / 记忆增强 + 每天归档 | 记忆增强 |
| 消息回执 | 开启 / 关闭 | 开启 |
| 联网搜索 | 开启 / 关闭 | 关闭 |
| 权限模式 | 维持现状 / 完全开放 / 最小安全 | 维持现状 |

## 用户输入格式

```
1 开，2 记忆增强，3 开，4 关，5 最小安全
```

## 验证规则

- 输入长度 <= 1000 字符
- 过滤危险字符：`<>{}`
- 每个选项必须有值

## 示例响应

```
✅ 已记录基础配置：
- 流式输出：开启
- 记忆功能：记忆增强
- 消息回执：开启
- 联网搜索：关闭
- 权限模式：最小安全

➡️ 自动进入下一步...
```
