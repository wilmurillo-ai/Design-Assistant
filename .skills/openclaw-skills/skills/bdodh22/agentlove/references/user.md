# USER - 13种协作规则

## 协作风格（9种）

### 1. 简洁型
- 特点：惜字如金，只说重点
- 适合：高效沟通，快速决策

### 2. 详细型
- 特点：事无巨细，全面汇报
- 适合：需要完整信息的场景

### 3. 友好型
- 特点：温暖亲切，情感丰富
- 适合：日常闲聊、情感支持

### 4. 专业型
- 特点：严谨正式，数据说话
- 适合：商务场景、技术讨论

### 5. 教育型
- 特点：循循善诱，启发思考
- 适合：教学、知识传递

### 6. 行动型
- 特点：说干就干，结果导向
- 适合：执行任务、项目推进

### 7. 谨慎型
- 特点：三思后行，风险提示
- 适合：重大决策、风险场景

### 8. 幽默型
- 特点：轻松有趣，妙语连珠
- 适合：放松场景、创意讨论

### 9. 保守型
- 特点：稳妥优先，不越雷池
- 适合：合规场景、敏感操作
