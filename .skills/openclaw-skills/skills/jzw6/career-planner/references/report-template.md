# Career Plan Report Template

Use this template structure when generating the final career plan report.
Adapt section depth based on how much information was collected.
Omit sections gracefully if data is missing — never fabricate.

---

## Report Header

```
# 🧭 职业规划报告
**姓名/称呼**: [用户名字或"您"]
**生成日期**: [当前日期]
**报告版本**: 个性化定制版
```

---

## Section 1: 您的个人画像 (User Profile)

A warm, humanizing summary of who this person is professionally.
Write in paragraph form, not bullet points. Should feel like the advisor truly *sees* them.

**Include:**
- Current stage of career / life
- Core strengths synthesized from interview (reframe positively)
- Defining characteristic or "superpower" (e.g., "你的最大优势是能把复杂问题用简单方式讲清楚")
- Key constraint or context to keep in mind

**Tone**: Affirming, grounded, specific to this person. NOT generic.

**Example:**
> 你目前处于职业转型的关键节点——有着扎实的财务分析基础，同时对科技产品充满热情。
> 你的核心优势在于用数字讲故事的能力，加上对用户体验的天然敏感度，这让你在产品与数据之间
> 拥有稀缺的双重视角。你目前最大的挑战是缺乏系统的产品经验，但这完全可以通过有针对性的
> 项目积累来弥补。

## Section 2: 技能盘点 (Skills Inventory)

| 类别 | 技能 | 熟练程度 | 市场价值评估 |
|------|------|----------|-------------|
| 硬技能 | [skill] | ⭐⭐⭐⭐ | 高需求 |
| 软技能 | [skill] | ⭐⭐⭐ | 中高需求 |
| 待发展 | [skill] | ⭐ | 建议补强 |

**熟练程度**: 根据用户描述合理推断，不要随意给满分
**市场价值**: 基于当前市场需求客观评估

After the table, add 1-2 sentences noting any "hidden gems" — undervalued skills the user
didn't fully appreciate.

---

## Section 3: 推荐职业方向 (Recommended Career Paths)

Recommend **2-4 career paths**, ordered by fit score (best first).
Each path must be substantiated by the user's actual data.

### Template per path:

```
### 方向 [N]: [职位名称] / [职业领域]

**匹配度**: ★★★★☆ (4/5)
**推荐理由**: [2-3句话，必须引用用户的具体背景/技能/兴趣]
**典型岗位**: [具体岗位名称 × 3]
**薪资参考**: [区间，注明城市/来源]
**所需补强**: [1-3个具体技能/经验]
**进入难度**: 低 / 中 / 高
**发展天花板**: [简短描述晋升路径]
```

**Critical rules:**
- NEVER recommend a path without explaining why it fits THIS specific person
- If salary data came from web search, cite the source
- Be realistic about "进入难度" — don't over-promise easy entry to competitive fields
- "典型岗位" should be real job titles people actually hire for

## Section 4: 30/90/180天行动计划 (Action Roadmap)

Break down into 3 time horizons. Be specific — no vague advice.

### 未来30天 — 探索与确认

Focus: Research and clarity, not immediate job-hunting (unless urgent).

- [ ] [Specific action 1] — e.g., "用LinkedIn搜索10个[目标职位]的招聘JD，整理高频技能关键词"
- [ ] [Specific action 2] — e.g., "联系1-2个从事[目标行业]的朋友做20分钟信息采访"
- [ ] [Specific action 3] — e.g., "完成[具体课程名称]的前3个模块，评估是否适合"

### 未来90天 — 积累与准备

Focus: Skill building, portfolio, network expansion.

- [ ] [Specific skill to acquire with specific resource]
- [ ] [Portfolio project or experience to build]
- [ ] [Network or community to join]
- [ ] [Resume/LinkedIn update milestone]

### 未来180天 — 出击与迭代

Focus: Active job search or launch.

- [ ] [Application targets or freelance platform setup]
- [ ] [Interview preparation specifics]
- [ ] [Side project or income milestone if applicable]
- [ ] [Review and adjust milestone]

## Section 5: 副业/斜杠可能性 (Side Hustle Paths) [Include if relevant]

Only include this section if the user showed interest in side income or has relevant skills/assets.

**Format:**
```
### 副业方向: [名称]
**门槛**: 低/中/高
**启动时间**: 约[X]个月
**收入潜力**: [范围]（兼职/全职）
**第一步**: [具体的第一个行动]
```

---

## Section 6: 需要正视的挑战 (Honest Challenges)

This section separates great career advisors from generic ones.
Acknowledge 2-3 real challenges the user will face. Frame them as manageable, not discouraging.

**Format:**
> **挑战1: [标题]**
> [2句话描述挑战] → [1句话给出应对策略]

---

## Section 7: 资源推荐 (Resources)

Curate resources relevant to THIS person's recommended paths. Not a generic list.

**Categories to include (as applicable):**
- 📚 学习资源（课程、书籍、认证）— with specific titles/platforms
- 🌐 求职平台 — relevant to their target roles/region
- 👥 社区 — industry communities, Slack groups, forums
- 🎙️ 内容推荐 — podcasts, newsletters, accounts to follow

---

## Section 8: 写在最后 (Closing Message)

A short, personalized closing paragraph. Warm, encouraging, honest.
Reference something specific they said during the conversation.
End with an open door for follow-up questions.

**Example:**
> 你提到自己"总是觉得自己还没准备好"——其实这种自我审视的能力本身就是一种稀缺的职业素质。
> 完美的时机从不会自己出现，但你今天迈出的这一步，已经比大多数人走得更远了。
> 如果你想深入聊某个方向，或者遇到任何新情况，随时都可以继续我们的对话。加油！🚀

---

## Formatting Rules

1. Use emoji sparingly but effectively for section headers and key highlights
2. Tables for skills and role comparisons — much easier to scan
3. Checkboxes for action items — creates sense of agency
4. Bold the most important insight in each section
5. Keep each section self-contained — user should be able to screenshot any section
6. Total report length: 800-1500 Chinese characters (not counting tables/lists)
7. Never use filler phrases like "根据以上分析" or "综上所述"
