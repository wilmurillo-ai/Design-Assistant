# Skill-to-Role Mapping Reference

Quick reference matrix for matching skill clusters to career paths.
Use alongside the user's interests and values — skills alone don't determine fit.

---

## How to Use This File

1. Identify the user's top 3-5 skills from the interview
2. Find matching skill clusters below
3. Cross-reference with their interests (Area 3) and lifestyle preferences (Area 5)
4. Shortlist 2-4 roles for the report

## Cross-Skill Bonus Combos (高价值技能组合)

These combinations command premium in the market:

| 技能组合 | 推荐方向 | 溢价说明 |
|---------|---------|---------|
| 技术 + 产品思维 | 技术产品经理 | 稀缺，薪资高30-50% |

| 财务 + 数字工具 | 财务BP / FP&A | 企业需求稳定 |

## Emerging/High-Growth Roles (2024-2025)

Roles to mention when user has adjacent skills and growth mindset:

- **AI产品经理** — 需要: 产品基础 + AI工具理解（高需求，早期红利期）
- **提示词工程师/AI训练师** — 需要: 写作能力 + 逻辑思维（新兴，远程友好）
- **出海运营** — 需要: 英语 + 跨文化理解 + 运营经验（高薪，需求旺盛）
- **私域运营** — 需要: 社群运营 + 用户心理（低门槛，副业可做）
- **DTC品牌运营** — 需要: 营销 + 供应链理解（适合有副业基础者）
- **知识付费/IP运营** — 需要: 专业知识 + 表达能力（副业首选）
- **ESG/可持续发展顾问** — 需要: 政策理解 + 商业分析（新兴，高端市场）

## Role Difficulty Reference

Use to calibrate "进入难度" in report:

| 难度 | 特征 | 典型例子 |
|------|------|---------|
| 低 | 需求量大，技能可自学，有初级岗位 | 内容运营、客服、初级数据 |
| 中 | 有门槛但可跨行，需3-6个月准备 | UI设计、项目管理、销售BD |
| 高 | 专业背景或多年经验要求，竞争激烈 | 战略咨询、投资分析、算法工程师 |
| 极高 | 头部机会，需要品牌背书 | 顶级VC、国际咨询MBB、大厂管培 |
