# 消费分类参考

## 支出分类 (expense)

| 分类 | 关键词示例 |
|------|-----------|
| 餐饮 | 吃饭、早餐、午饭、晚饭、咖啡、奶茶、外卖、餐厅、饭店、小吃 |
| 购物 | 超市、买、购物、衣服、鞋子、电商、淘宝、京东、日用品 |
| 交通 | 打车、地铁、公交、高铁、机票、停车、加油、滴滴 |
| 娱乐 | 电影、游戏、KTV、旅游、景区、演出、门票 |
| 医疗 | 医院、药店、药、挂号、体检、看病 |
| 住房 | 房租、水电、物业、燃气、宽带、房贷 |
| 教育 | 学费、课程、培训、书、资料、考试 |
| 通讯 | 话费、手机费、流量、套餐 |
| 其他 | 未匹配到以上分类时使用 |

## 收入分类 (income)

| 分类 | 关键词示例 |
|------|-----------|
| 工资 | 工资、薪资、月薪、基本工资 |
| 奖金 | 奖金、绩效、年终奖、提成 |
| 兼职 | 兼职、副业、外包、接单 |
| 投资 | 理财、股票、基金、利息、分红 |
| 其他 | 红包、转账、报销、退款 |

## 分类推断逻辑

Claude 在解析用户文字时，应根据关键词自动推断分类：
1. 精确匹配关键词 → 直接使用对应分类
2. 无法匹配 → 使用"其他"，并在回复中告知用户已使用默认分类
3. 用户可在确认时纠正分类
