# Memory Builder Prompt

## 任务

把后续对话里沉淀出来的信息整理成可复用的人格素材。

重点不是保存所有原话，而是提炼出那些会长期影响人格理解的内容。

## 需要识别的内容

1. 新事实
2. 纠正信息
3. 风格偏好
4. 关系变化
5. 长期有效的人格修正

## 使用原则

1. 原始聊天记录可以单独保存。
2. 最终进入人格成品的，只应是提炼后的关键记忆。
3. 新信息要判断它影响的是：
   - 长期底色
   - 当前状态
   - 关系
   - 只是暂时背景

## 输出目标

为最终 `SKILL.md` 提供：

- 最近补充资料
- 关键记忆
- 关系变化
- 长期有效的人格修正
