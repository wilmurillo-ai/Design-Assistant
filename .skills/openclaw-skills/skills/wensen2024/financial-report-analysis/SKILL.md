# 财报分析

专业级上市公司财务报表分析工具，支持A股、港股、美股上市公司财务数据解析与评估。

## 功能特性

- 📊 **三大报表分析**: 资产负债表、利润表、现金流量表全面解析
- 📈 **财务指标计算**: ROE、ROA、毛利率、净利率、资产负债率等20+指标
- ⚠️ **风险识别**: 自动识别财务风险点，提供风险等级评估
- 📋 **报告生成**: 一键生成专业分析报告，支持多种格式导出
- 🔍 **对比分析**: 支持多公司财务数据对比
- 📅 **趋势分析**: 多年度财务趋势追踪

## 使用方式

```bash
# 单公司分析
分析 贵州茅台 2025年年报

# 指定报告期
分析 腾讯控股 2024年三季报

# 对比分析
对比分析 比亚迪 和 蔚来 2025年财报

# 行业分析
分析白酒行业龙头企业财报对比

# 趋势分析
分析 贵州茅台 近5年盈利趋势
```

## 数据来源

- **A股**: 东方财富、同花顺、Wind
- **港股**: 港交所、东方财富
- **美股**: SEC EDGAR、Yahoo Finance
