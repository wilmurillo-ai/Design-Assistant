# 合同审查

专业级合同智能审查工具，帮助企业和个人识别合同风险，保障合法权益。

## 功能特性

- ⚠️ **风险识别**: 不公平条款、违法条款、权责不对等条款
- 📋 **条款分析**: 关键条款提取、合规性检查、完整性检查
- ➕ **缺失补充**: 必备条款提醒、建议补充条款、条款模板推荐
- 🔄 **版本对比**: 合同版本差异对比、条款变更追踪
- 📊 **风险评估**: 综合风险评分、分项风险分析
- 📚 **法律依据**: 相关法律条文引用、司法案例参考

## 使用方式

```bash
# 合同审查
审查这份合同 并指出风险点

# 条款分析
分析这份合同的违约责任条款

# 合同对比
对比这两份合同的差异 contract_v1.pdf contract_v2.pdf

# 条款补充
检查这份服务合同缺失哪些必备条款

# 风险评估
评估这份投资协议的风险等级
```

## 审查维度

### 合法性
- 是否违反法律强制性规定
- 是否违反公序良俗
- 是否存在无效条款

### 公平性
- 权利义务是否对等
- 违约责任是否合理
- 解除条件是否公平

### 完整性
- 必备条款是否齐全
- 约定是否明确具体
- 是否存在遗漏事项

### 可操作性
- 履行方式是否明确
- 时间节点是否清晰
- 违约责任是否可执行

## 注意事项

1. 审查建议仅供参考，不构成法律意见
2. 重大合同建议咨询专业律师
3. 不同地区法律可能存在差异
4. 合同签订前请仔细阅读所有条款
