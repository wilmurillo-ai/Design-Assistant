# A股持仓监控脚本

监控A股持仓的止损、支撑、目标价和异常波动。

## 功能

- 实时获取持仓股票现价
- 检测是否触及止损/支撑/目标位
- 计算浮盈/浮亏
- 乖离率超5%警告
- 当日跌幅超5%异动告警

## 触发条件

| 条件 | 标记 | 操作建议 |
|------|------|---------|
| 触及止损价 | 🔴 | 建议止损 |
| 跌破支撑位 | 🟠 | 关注是否企稳 |
| 当日跌幅>5% | 🔴 | 异动告警 |
| 乖离率>+5% | ⚠️ | 短线过热 |

## 文件

- `holding_monitor.py` - 主程序

## 安装

1. 下载脚本
2. 修改脚本顶部的 `HOLDINGS` 列表，填入你的持仓
3. 设置执行权限：`chmod +x holding_monitor.py`

## 配置

编辑 `HOLDINGS` 列表：

```python
HOLDINGS = [
    {
        "name": "贵州茅台",
        "code": "sh600519",  # sh=上证 sz=深证
        "cost": 1800.0,
        "qty": 10,
        "stop_loss": 1700.0,
        "support": 1750.0,
        "target": "2000-2100",
        "optimistic": 2200,
    },
]
```

## 运行方式

**手动运行：**
```bash
python3 holding_monitor.py
```

**定时检查（每天9:30-15:00每30分钟一次）：**
```bash
# 添加到crontab
crontab -e

# 写入：
*/30 9-15 * * 1-5 /path/to/holding_monitor.py >> /path/to/holding_monitor.log 2>&1
```

## 注意事项

- 需要网络连接获取实时行情
- 数据源：新浪财经API
- 止损和支撑位需要根据个人风险承受能力设置
- 建议配合飞书/微信机器人实现推送通知
