# 工程量计算公式

## 土方工程

### 基坑开挖
```
V = (A + 2C + KH) × (B + 2C + KH) × H + K²H³/3

其中:
V = 土方体积 (m³)
A = 基础底面长度 (m)
B = 基础底面宽度 (m)
C = 工作面宽度 (m)
K = 放坡系数
H = 开挖深度 (m)
```

### 放坡系数参考
| 土壤类别 | 放坡起点 (m) | 人工挖土 | 机械挖土 |
|---------|------------|---------|---------|
| 一、二类土 | 1.20 | 1:0.50 | 1:0.33 |
| 三类土 | 1.50 | 1:0.33 | 1:0.25 |
| 四类土 | 2.00 | 1:0.25 | 1:0.10 |

## 混凝土工程

### 基础混凝土
```
V = 基础底面积 × 基础高度 - 预留孔洞体积

条形基础：V = 断面面积 × 长度
独立基础：V = 长 × 宽 × 高
筏板基础：V = 板面积 × 板厚
```

### 梁混凝土
```
V = 梁断面面积 × 梁长 - 板厚所占体积

矩形梁：V = 梁宽 × 梁高 × 梁长
圈梁：V = 断面面积 × (外墙中心线 + 内墙净长线)
```

### 板混凝土
```
V = 板面积 × 板厚 - 柱、梁所占体积

有梁板：V = 板体积 + 梁体积
无梁板：V = 板体积 + 柱帽体积
```

### 柱混凝土
```
V = 柱断面面积 × 柱高

矩形柱：V = 柱宽 × 柱厚 × 柱高
圆形柱：V = π × r² × 柱高
构造柱：V = 断面面积 × (层高 + 马牙槎增加体积)
```

## 钢筋工程

### 钢筋重量计算
```
G = L × W

其中:
G = 钢筋重量 (kg)
L = 钢筋长度 (m)
W = 单位长度重量 (kg/m)

常用钢筋单位重量:
Φ6  = 0.222 kg/m
Φ8  = 0.395 kg/m
Φ10 = 0.617 kg/m
Φ12 = 0.888 kg/m
Φ14 = 1.21  kg/m
Φ16 = 1.58  kg/m
Φ18 = 2.00  kg/m
Φ20 = 2.47  kg/m
Φ22 = 2.98  kg/m
Φ25 = 3.85  kg/m
Φ28 = 4.83  kg/m
Φ32 = 6.31  kg/m
```

### 钢筋搭接长度
```
搭接长度 = 锚固长度 × 修正系数

修正系数:
- 绑扎搭接：1.2 ~ 1.6
- 机械连接：按接头类型
- 焊接连接：按焊接工艺
```

## 模板工程

### 接触面积计算
```
基础模板：S = 基础周长 × 基础高度
柱模板：S = 柱周长 × 柱高
梁模板：S = (梁宽 + 2 × 梁高) × 梁长
板模板：S = 板面积

扣除规则:
- 梁柱交接处不重复计算
- 板与梁交接处扣除梁侧面积
- 单个孔洞面积>0.3m²时扣除
```

---

## 砌体工程

### 墙体体积
```
V = (L × H - 门窗洞口面积) × 墙厚 - 嵌入构件体积

其中:
L = 墙体长度
  - 外墙：按中心线计算
  - 内墙：按净长线计算
H = 墙体高度
  - 有梁：算至梁底
  - 无梁：算至板顶
```

### 标准砖用量
```
砖数 = 墙体体积 × 512 块/m³

标准砖规格：240×115×53 mm
灰缝按 10mm 计算
```

## 装饰装修

### 抹灰面积
```
S = 墙面面积 + 顶棚面积 - 门窗洞口面积

墙面：S = 墙长 × 墙高
顶棚：S = 房间净面积

洞口扣除:
- 门窗洞口侧壁不增加
- 踢脚线不扣除
- 装饰线不展开
```

### 地面面积
```
S = 房间净面积

规则:
- 扣除凸出地面的构筑物
- 不扣除间壁墙
- 门洞开口部分不增加
```

## 计算工具脚本

使用 `scripts/calculate_quantity.py` 进行自动计算：

```bash
# 计算钢筋重量
python3 scripts/calculate_quantity.py rebar --diameter 25 --length 100

# 计算混凝土体积
python3 scripts/calculate_quantity.py concrete --type beam --width 0.3 --height 0.6 --length 10

# 计算模板面积
python3 scripts/calculate_quantity.py formwork --type column --width 0.5 --height 3.5
```
