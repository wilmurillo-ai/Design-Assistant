# 安全检查清单

## 日常检查（每日必检）

### 1. 临边防护
- [ ] 基坑周边防护栏杆（高度≥1.2m）
- [ ] 楼层临边防护
- [ ] 楼梯口防护
- [ ] 电梯井口防护（定型化防护门）
- [ ] 预留洞口盖板或防护栏杆

### 2. 临时用电
- [ ] 三级配电两级保护
- [ ] 一机一闸一漏一箱
- [ ] 电缆线无破损、老化
- [ ] 配电箱接地良好
- [ ] 电工持证上岗

### 3. 脚手架
- [ ] 立杆基础坚实
- [ ] 连墙件设置符合规范
- [ ] 剪刀撑连续设置
- [ ] 脚手板满铺、固定
- [ ] 安全网张挂严密

### 4. 起重机械
- [ ] 塔吊限位装置有效
- [ ] 吊索具无破损
- [ ] 信号工持证上岗
- [ ] 起重作业区域警戒
- [ ] 设备验收合格标识

### 5. 消防安全
- [ ] 灭火器配置到位
- [ ] 消防通道畅通
- [ ] 动火作业审批
- [ ] 易燃物品隔离存放
- [ ] 吸烟区设置

## 隐患整改流程

```
1. 发现隐患 → 拍照记录
2. 下发整改通知单 → 明确责任人和期限
3. 整改实施 → 责任班组整改
4. 复查验收 → 安全员复查
5. 闭环销项 → 归档记录
```

---

## 常见隐患及处理

| 隐患类型 | 典型问题 | 整改措施 | 整改期限 |
|---------|---------|---------|---------|
| 临边防护 | 防护栏杆缺失 | 立即设置定型化防护 | 立即 |
| 临时用电 | 私拉乱接 | 拆除违规线路，规范布线 | 当日 |
| 脚手架 | 连墙件缺失 | 按规范补设 | 24 小时 |
| 消防 | 灭火器失效 | 更换合格灭火器 | 当日 |
| 机械 | 限位失灵 | 停机维修，验收后使用 | 立即停机 |
