# 第一阶段：规划与构思（世界构建）

## 步骤

### 第一步：确定体裁与方向

与用户确认以下内容（如未指定）：

- 科幻子类型（硬科幻/太空歌剧/赛博朋克/反乌托邦/生物朋克/后末日/时间旅行科幻等）

- 目标读者及篇幅（短篇小说 5,000 字以上/中篇小说 20,000 字以上/长篇小说 80,000 字以上）

- 基调（严肃/娱乐/文学/幽默）

- 您想参考的任何作者或作品

### 第二步：设定小说核心要素（科幻核心元素）

定义 1-3 个推动故事发展的科幻小说要素：

```

核心要素示例：

- 科技：意识转移技术、光速导航、通用人工智能超级智能

- 社会：一个以情感为货币、共享记忆、死亡由个人选择的社会

- 生物系统：人类的趋异进化、共生外星生命形式、合成生态系统

``` 针对每一种新科技，进行三阶段分析，探讨“它存在后，世界会发生哪些变化？”：

1. 直接影响（科技本身的影响）

2. 社会影响（制度、文化和伦理的变革）

3. 个人影响（对人类生活方式和身份认同的影响）

### 第三步：系统化世界观

根据以下模板构建设定素材，并将其记录在 `settings.md` 文件中：

```markdown
# 世界设定
## 基本信息

- 时代：

- 设定：

- 技术水平：

## 核心科技/新科技

- [名称]：[概述]

- 原理：

- 限制/局限性：

- 社会影响：

## 社会结构

- 政治制度：

- 经济系统：

- 阶级/不平等：- 文化与价值观：

## 历史（导致当今的重要事件）

- [时代]：[事件]

## 日常生活

- 衣着、食物和住所的特点：

- 通讯方式：

- 娱乐：

## 词汇表

- [术语]：[定义]

```

### 第四步：明确主题

用一句话定义故事的核心问题（主题）：

```

主题示例：

- “当意识可以被复制时，‘自我’是什么？”

- “在一个完全安全得到保障的社会里，自由是必要的吗？”

- “即使人类穿越星辰，孤独也能被克服吗？”

```

将主题与故事的所有元素（情节、人物冲突、世界观）联系起来。

### 第五步：输出

将完成的设置文档保存为 `settings.md`，并将其展示给用户。

收到反馈并进行修改后，建议进入第二阶段。
