## NL to canonical scene name hints (examples)

These pairs support **generalized** name resolution when **no full catalog match** exists; combine with **similarity** ranking (e.g. "dusk" and "sunset" ~95%) per **[Scene name matching (shared)](../scene-manage.md#scene-name-matching-shared)**.

- Dreamy moonlight, quiet ambiance, sleep aid - Moonlit night scene  
- A touch of dusk feeling - Sunset scene  
- Need to focus / time for yoga - Mindfulness scene  
- Guests are coming - Guest reception scene  
- Time for tea - Tea time scene
