## What Is This?

A universal skill that transforms how AI agents approach design, development, and user interaction — based on the book *Unreasonable Hospitality* by Will Guidara.

Will Guidara turned Eleven Madison Park from a struggling brasserie into the **#1 restaurant in the world**. His secret wasn't better food — it was making people *feel* something extraordinary.

This skill distills those principles into actionable frameworks for anyone building products.

## Core Equation

```
Unreasonable Hospitality = Technical Excellence × Emotional Intelligence × Intentional Generosity
```

## The 7 Laws

1. **Service is the Floor, Hospitality is the Ceiling** — "Did it work?" vs "How did it *feel*?"
2. **The 95/5 Rule** — 95% engineering rigor, 5% unreasonable generosity
3. **One Size Fits One** — True personalization, not mail-merge
4. **Read and React** — Anticipate needs before they're expressed
5. **Earned Informality** — Master the rules, then break them with intention
6. **Details Are the Product** — Thousands of micro-details = invisible quality
7. **Connection Over Transaction** — Every interaction builds relationships

## Use Cases

- 🎨 **UI/UX Design** — Create experiences users remember
- ✍️ **Copy/Microcopy** — Write messages that feel human
- 🤖 **AI Assistants** — Build assistants people actually enjoy talking to
- 🔍 **Product Review** — Score products on the hospitality spectrum
- 💬 **Feedback** — Give feedback that feels like a gift, not a wound
- 🚀 **Onboarding** — Make first impressions warm, not bureaucratic
