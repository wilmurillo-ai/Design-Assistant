## 📊 SEO Scoring System

Each show notes version gets an **SEO Score** based on:
- Keyword optimization (0-30 points)
- Structure and formatting (0-25 points)
- Value provided (0-25 points)
- Internal linking (0-20 points)

**Score breakdown:**
- 80-100: Exceptional SEO (top 3 Google ranking)
- 60-79: Good SEO (page 1 ranking)
- 40-59: Moderate SEO (page 2-3 ranking)
- 20-39: Low SEO (beyond page 3)
- 0-19: Poor SEO (no ranking)

## ⏱️ Timestamp Strategy

Best practices:
- Include main topic timestamps (every 10-15 minutes)
- Highlight key moments and quotes
- Use descriptive labels (not just "Part 1", "Part 2")
- Keep timestamps accurate and easy to scan
- Include intro/outro timestamps for skipping

---

## 📝 Key Takeaways Extraction

Generate 3-5 actionable takeaways:
- Start with action verbs
- Keep each under 20 words
- Focus on value, not content
- Make them tweetable
- Include specific examples when possible

---

## 🎨 Social Media Snippets

Create platform-specific content:
- **Twitter:** Thread format with quotes
- **LinkedIn:** Professional insight + key takeaway
- **Instagram:** Visual quote card text
