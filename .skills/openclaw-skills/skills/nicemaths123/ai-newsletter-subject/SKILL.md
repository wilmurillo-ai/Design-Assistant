## 🎯 Subject Line Formulas

### Formula 1: The Curiosity Hook
```
"The one thing about [topic] nobody tells you"
"What I learned from [experience]"
"Why [common belief] is wrong"
```

### Formula 2: The Benefit Promise
```
"How to [benefit] in [timeframe]"
"Get [result] without [pain]"
"[Number] ways to [benefit]"
```

### Formula 3: The Number-Driven
```
"7 [things] you need to know about [topic]"
"3 mistakes you're making with [topic]"
"5 secrets to [benefit]"
```

### Formula 4: The Personalized
```
"[Name], here's what you missed"
"Your [resource] is ready"
"[Name], I have something for you"
```

### Formula 5: The Urgency-Based
```
"Last chance: [offer] ends tonight"
"24 hours left to [benefit]"
"Don't miss this: [topic]"
```

## 📊 Open Rate Scoring System

Each subject line gets an **Open Rate Score** based on:
- Curiosity factor (0-30 points)
- Personalization (0-25 points)
- Length optimization (0-25 points)
- Mobile preview (0-20 points)

**Score breakdown:**
- 80-100: Exceptional (40%+ open rate)
- 60-79: High (30-40% open rate)
- 40-59: Moderate (20-30% open rate)
- 20-39: Low (10-20% open rate)
- 0-19: Poor (<10% open rate)

## 📱 Mobile Optimization

Best practices:
- Keep under 40 characters for full mobile preview
- Put important words first
- Test on both iOS and Android
- Avoid truncation of key info
- Use preview text effectively

## 🧪 A/B Testing Strategy

Test these elements:
- Length (short vs long)
- Personalization (name vs no name)
- Numbers (specific vs general)
- Emojis (with vs without)
- Question vs statement
