# Templates Index

Use templates as blueprint files for generation. Each template defines:

- when to use the pattern
- required and optional sections
- component and data interactions
- scaffold skeleton with placeholders
- validation rules

## Authoring Rule

Templates must not contain prefilled fake business stories as the default
output. Use placeholders and structural guidance so generated pages start as
scaffolds and can be filled with user-provided content.
