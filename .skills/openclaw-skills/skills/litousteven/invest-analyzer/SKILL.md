# Invest Analyzer - 个股分析工具

> 基于四流派投资体系的个股深度分析工具

## ⚠️ 依赖要求（重点）

Invest Analyzer 的核心在于：**先拿到“可信的年报/三表数据”，再做初筛/FCF/决策**。

### A. 搜索/抓数能力（必选其一）

- ✅ **首选：baidu-search**（中文年报/公告最稳）
  - 用于搜索：年报 PDF、公告、财务指标页面
  - Key/鉴权的配置与维护 **交由 baidu-search skill 自身处理**（按其文档配置即可）
  - 示例：
    ```bash
    python3 skills/baidu-search/scripts/search.py '{
      "query":"中国卫星 600118 2024 年年度报告 PDF site:cninfo.com.cn",
      "search_recency_filter":"year"
    }'
    ```

- ✅ 备选：**agent-browser**（网页抓取/可视化点选）
  - 当目标站点反爬、API 不稳定、或需要登录/翻页时用

- ✅ 备选：**直接用财报/公告直链（PDF）**
  - 例如 `static.cninfo.com.cn/finalpage/...PDF`
  - 这种方式最“确定性”，适合流水线化

> 不推荐依赖 `web_search(Brave)` 作为中文财报来源：需要额外 key，且在国内环境经常不可用。

### B. PDF 解析能力（强烈推荐，至少一个）

- ✅ **pymupdf-pdf-parser-clawdbot-skill**（快，适合“先定位页码/再抽表”）
- ✅ OpenClaw 内置 `pdf` 工具（稳，适合“按页提取数值”）

> 实战建议：先用 PyMuPDF 把 PDF 转 `output.json`（每页 text），用关键词定位页码；再按页抽取三表数据。

## 实战参考流程（PDF→三表→FCF，强烈推荐）

> 我们在“中国卫星（600118）”实战中验证过的一条龙流程，可复用、可自动化。

### 0) 先判定 PDF 类型（避免扫描版浪费时间）

- **文本版（可复制/可抽取连续文本）** → 继续下面步骤
- **扫描版/图片版（几乎无文本层）** → **直接放弃解析**，转去官方渠道下载“标准年报 PDF（含文本层）”再继续

判断方法（快速）：抽取前几页/随机几页，如果无法抽取连续长文本，基本就是扫描版。

1) **搜索年报 PDF**（baidu-search）
2) **下载到本地**（建议 `/tmp/openclaw/` 统一管理）
3) **关键词定位页码**（合并资产负债表/利润表/现金流量表；短期借款/租赁负债/CAPEX 等）
4) **抽取三表关键数值**（按页提取，避免“摘要缺表/解析器丢字段”）
5) **计算**：
   - `FCF ≈ 经营活动现金流净额 - 购建固定资产/无形资产等支付的现金`
   - `净现金 = 现金及等价物 - 有息负债`
6) **输出 invest-analyzer 报告**（按本 skill 的“决策输出格式”）

## 简介

本技能整合自 [investTemplate V5.5.6](https://github.com/sunheyi6/investTemplate)，提供标准化的个股分析流程。

## 核心能力

- ✅ **5分钟快速初筛**：排除高风险标的
- ✅ **四流派判定**：纯硬收息 / 价值发现 / 烟蒂股 / 关联方资源
- ✅ **FCF估值计算**：剔除净现金FCF倍数核心指标
- ✅ **决策建议**：买入/观望/卖出 + 仓位建议
- ✅ **持仓管理**：状态标签、组合韧性评估

## 分析流程（对齐 investTemplate V5.5.6）

关键变化：
- **动态安全边际**：牛市/熊市阈值不同（核心 EV/FCF）
- **强制计算 Owner Earnings**：避免PE陷阱/减值扭曲
- **错误作废机制**：核心数据错误→全文作废重来
- **央国企门槛更硬**：民营标的原则上不进核心仓位


### 第一步：快速初筛（5分钟）

| 检查项 | 排除标准 |
|--------|---------|
| 地缘政治 | 非洲/冲突区资产>10% ❌ |
| 价格位置 | 距52周高点<30% ❌ |
| 负债扫描 | 财务费用>5亿（非周期股）❌ |
| 现金覆盖 | 现金<市值20% ❌ |

### 第二步：流派判定

| 流派 | 特征 | 适用标的 |
|------|------|---------|
| 🛡️ 纯硬收息 | 股息率>5%，连续5年派息 | 电信/公用事业 |
| 🔍 价值发现 | PE<10，FCF/市值>10% | 消费/制造 |
| 🚬 烟蒂股 | 净现金>市值，PB<0.3 | 港股小盘 |
| 🔗 关联方 | 国企背景、母公司支持 | 物业/基建 |

### 第三步：FCF估值（核心）

```
FCF = 经营现金流 - 维持性CAPEX
（注意：股息不扣除！）

剔除净现金FCF倍数 = (市值 - 净现金) / FCF

估值参考：
- 3-5倍：金龟级（推荐建仓）
- 5-8倍：合理偏低
- 8-10倍：估值合理
- >10倍：高估
```

### 第四步：决策输出

```
输出格式：
- 标的：[名称/代码]
- 流派：[判定结果]
- 初筛结果：[通过/未通过]
- 核心指标：[FCF/FCF倍数/股息率等]
- 估值结论：[低估/合理/高估]
- 建议：[买入/观望/卖出]
- 仓位建议：[试探/建仓/加仓/清仓]
- 风险提示：[关键风险点]
```

## 数据来源（建议分层使用）

- **S/A级（可复核，优先）**：交易所/披露平台公告（深交所/上交所/巨潮资讯等）、公司 IR（投资者关系）官网、（如有）HKEX/SEC。
- **B级（辅助/快速）**：东方财富/同花顺/雪球/腾讯财经等的行情与估值数据（用于市值、PE/PB、历史分位、对标横比），但关键结论需回公告口径校验。
- **B~A-级（高频经营数据，行业相关）**：销量/上险/出口/装机等（乘联会/中汽协/海关及第三方汇总），用于判断景气、份额与价格压力。
- **A-~B+（信用与债务细节）**：债券募集说明书、评级报告、银行间披露（用于拆有息负债结构、担保、或有风险）。

> 原则：**年报/公告定“真相”，第三方定“速度”，高频定“趋势”。**
> invest-analyzer 输出中应标注：关键数字来自哪类来源（S/A/B），避免混口径。
