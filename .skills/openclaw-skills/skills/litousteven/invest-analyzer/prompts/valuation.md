## 2. FCF计算（V5.5.6：必须做“FCF规范化 + 所有者收益”）

### 2.1 FCF（规范口径）

核心公式：

```
FCF = 经营现金流（CFO/OCF） - 维持性CAPEX
```

⚠️ **强制规则（防错）**：
- ✅ **股息不扣除**（股息属于筹资活动现金流，不影响FCF定义）
- ✅ **CAPEX 必须评估“维持性”**，不要机械照搬账面“购建固定资产…”
- ✅ **防双标**：如果你在估值中扣了某类负债，就必须对称地把现金/等价物计入（或干脆用“方案A：只看FCF/市值”）

### 2.2 所有者收益（Owner Earnings，必须计算）

巴菲特式口径（用于识别“利润扭曲/减值扭曲”）：

```
所有者收益 = 净利润
          + 折旧/摊销
          + 主要非现金减值（如商誉/资产减值）
          - 维持性CAPEX
```

> 说明：所有者收益与FCF可能不同（尤其在减值/折旧占比大行业），但两者都应给出，避免PE陷阱。

### 计算示例

```
经营现金流：28.47亿
维持性CAPEX：4.0亿（合理估算）
────────────────
FCF：24.47亿 ✅
```

### CAPEX调整指南

| 行业 | 账面CAPEX特征 | 真实维持性判断 |
|------|-------------|---------------|
| 饮料/快消 | 线性折旧，偏高 | 约4亿/年 |
| 重工业 | 设备更新快 | 接近折旧额 |
| 科技 | 技术迭代快 | 可能高于折旧 |
| 公用事业 | 折旧大 | 需具体分析 |

## 3. 剔除净现金FCF倍数（V5.5.4：核心估值指标）

### 公式

```
净现金 = 现金及等价物 - 有息负债
EV（企业价值） = 市值 - 净现金
剔除净现金FCF倍数 = EV / FCF
```

### 两种并行口径（都建议输出）
- **方案A（推荐，最不易双标）**：只看 `FCF/市值`（不做净现金调整、不暗扣经营性负债）
- **方案B（对称处理）**：计算 `EV/FCF`（扣现金/扣有息负债；如考虑其他负债必须对称）

> 经验：高现金公司很适合用方案A；当现金占市值很大时，再补充方案B做对照。

## 4. 估值区间（V5.5.6：动态安全边际）

### 4.1 市场周期判定（简化版）
- **牛市/偏牛**：整体估值抬升，可接受阈值略放宽
- **熊市/偏熊**：只做极端便宜

> 若无法严格判定，可默认“偏牛/震荡”，并在结论中声明。

### 4.2 剔除净现金FCF倍数阈值（核心）

| 市场状态 | 建仓阈值（核心） | 说明 |
|---|---|---|
| 牛市/偏牛 | **EV/FCF < 5** | 对应FCF收益率>20% |
| 熊市/偏熊 | **EV/FCF < 3** | 对应FCF收益率>33% |

### 4.3 评级参考（保持直觉刻度）

| 倍数区间 | 评级 | 操作建议 |
|---------|------|---------|
| < 3倍 | 🐢🍊 极品 | 极限仓位15%，重仓买入 |
| 3-5倍 | 🐢🍊 金龟 | 推荐建仓区间 |
| 5-8倍 | 合理偏低 | 分批建仓 |
| 8-12倍 | 合理区间 | 轻仓/观望 |
| > 12倍 | 高估 | 回避 |

### 传统PE（对比参考）

| PE区间 | 评级 | 操作建议 |
|--------|------|---------|
| < 6倍 | 低估 | 建仓 |
| 6-10倍 | 合理 | 持有 |
| 10-15倍 | 合理偏高 | 观望 |
| > 15倍 | 高估 | 卖出 |

## 5. 防坑检查

### ❌ 常见错误

1. **股息扣除**：FCF不用减股息！
2. **账面CAPEX**：需调整为维持性支出
3. **双标处理**：不能"抵扣负债但不加现金"
4. **PE陷阱**：利润质量差时禁用PE

### ✅ 正确做法

1. **方案A（推荐）**：不看净现金，纯FCF/市值
2. **方案B**：对称处理，加现金减负债
3. **两种方案都算**：确保估值逻辑一致
