# 决策建议 - Invest Analyzer

## 决策框架

完成初筛和估值后，给出最终决策建议。

---

## 1. 流派判定

### 四流派决策树

```
步骤1：是否满足央国企门槛？（V5.5.6：从“加分项”提升为硬约束）
  ├── 是（央企/地方国企/混合所有制）→ 步骤2
  └── 否（民营）→ 仅在【周期拐点明确 + 安全边际极厚】时允许小仓位试探（不作为核心仓位）

步骤2：判定流派
  ├── 🛡️ 纯硬收息：股息率>5%，连续5年派息，无有息负债
  ├── 🔍 价值发现：PE<10或FCF/市值>10%
  ├── 🔗 关联方：国企背景+母公司资源
  └── 🚬 烟蒂股：净现金>市值，PB<0.3
```

### 流派特征对比

| 流派 | 核心指标 | 买入标准 | 卖出标准 |
|------|---------|---------|---------|
| 🛡️ 纯硬收息 | 股息率>5% | 股息率>6% | 股息率<4% |
| 🔍 价值发现 | FCF/市值>10% | FCF/市值>12% | FCF/市值<8% |
| 🚬 烟蒂股 | 净现金>市值 | 剔除净现金FCF<3倍 | 估值修复完成 |
| 🔗 关联方 | 资源折价 | 折价>30% | 折价<15% |

## 2. 仓位建议

### 建仓梯度

| 层级 | 仓位 | 条件 |
|------|------|------|
| 试探 | 3% | 初步通过初筛，估值合理 |
| 首次 | 5% | 估值偏低，价格可接受 |
| 二次 | 8% | 下跌10%后加仓 |
| 重仓 | 12% | 极限点位，极品机会 |

### 持仓上限

| 类型 | 上限 |
|------|------|
| 单只股票 | 15%（极品）|
| 单只烟蒂股 | 10%（港股）|
| 同一流派 | 40% |
| 民营标的 | 20% |

## 3. 决策矩阵

### 基于FCF倍数的决策

| FCF倍数 | 价格位置 | 决策 | 仓位 |
|---------|---------|------|------|
| < 3倍 | 任意 | 买入 | 10-15% |
| 3-5倍 | 距高点<30% | 观望 | - |
| 3-5倍 | 距高点>30% | 试探建仓 | 3-5% |
| 5-8倍 | 距高点>50% | 首次建仓 | 5% |
| > 8倍 | 任意 | 卖出/回避 | 0% |

---

## 4. 卖出纪律

### 强制卖出条件

- 股息率 < 2%（硬收息型）
- 连续2年派息减少
- 有息负债大幅增加
- 经营现金流恶化
- 估值修复完成（FCF倍数>10）

### 何时卖出

| 情况 | 操作 |
|------|------|
| 达到目标价 | 减仓50% |
| 极端高估（>15倍FCF）| 清仓 |
| 基本面恶化 | 立即清仓 |
| 发现更好标的 | 调仓 |

---

## 5. 持仓管理

### 状态标签

- 🟢 **正常**：正常持有，等待估值修复
- 🔵 **已回本**：股价回到成本价，可考虑减仓
- 🟡 **高位**：距低点涨幅>50%，关注回调风险
- 🟠 **关注**：基本面有变化，需重新评估
- 🔴 **遗留**：历史遗留仓位，考虑逐步退出

### 组合韧性指标

| 指标 | 健康值 | 预警值 |
|------|--------|--------|
| 组合FCF/市值 | >8% | <5% |
| 单一仓位集中度 | <15% | >20% |
| 流派分散度 | 每派<40% | 单一>60% |
| 持仓成本距现价 | -20%~+30% | <-30% |
