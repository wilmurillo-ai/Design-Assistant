# 后期调色入门指南

面向摄影新手的后期调色教学内容，Agent 可根据此内容向用户解释调色思路和参数。

---

## 曝光三角复习

拍照时三个参数互相影响，像水龙头的开关：

- **光圈**（水龙头口径大小）：控制进光量的大小。口径大（数字小如 f/1.8）→ 进光多 → 背景模糊。口径小（数字大如 f/11）→ 进光少 → 全景清晰。
- **快门速度**（开水龙头的时间长短）：控制进光的时间。时间短（如 1/1000s）→ 冻结瞬间。时间长（如 2s）→ 拍出运动轨迹。
- **ISO**（水压大小）：控制传感器对光的敏感度。ISO 低（100）→ 画质干净。ISO 高（3200）→ 画面亮但噪点多。

**核心规则**：调大一个参数，就需要调小另一个来保持同样的亮度。

## 调色基础概念

### 色温 (Temperature)
让画面偏黄（暖）还是偏蓝（冷）。阴天拍的照片偏蓝，调暖一些更舒服；日落照片本身就很暖，可以适当降温保持自然。

### 色调 (Tint)
在暖冷之外的第二条轴：偏绿还是偏洋红。通常微调即可，用于修正肤色或制造特殊氛围。

### 饱和度 (Saturation)
画面中颜色的鲜艳程度。新手常见错误是拉太高，看起来不自然。建议小幅调整，或只调整「自然饱和度」（自动保护已有色彩）。

### 明度 (Luminance)
某种颜色的明暗程度。比如「天空蓝的明度降低」能让天空更深邃，不改变整体曝光。

### 对比度 (Contrast)
亮部和暗部的差距。高对比度 → 画面硬朗有冲击力。低对比度 → 画面柔和朦胧（日系风格常用）。

### 高光 / 阴影
分别控制画面中亮的部分和暗的部分的亮度。压低高光可以找回天空细节，提亮阴影可以恢复暗部内容。

## 新手调色工作流

按以下顺序调色，效率最高：

1. **曝光**：先让画面亮度正确（不过亮不过暗）
2. **白平衡**（色温 + 色调）：让颜色看起来自然，或故意偏色制造氛围
3. **对比度**：决定画面的整体力度
4. **高光 / 阴影**：找回亮部和暗部的细节
5. **饱和度 / 自然饱和度**：调整颜色鲜艳程度
6. **色调曲线**：精细调整亮暗分布（进阶）
7. **HSL 面板**：单独调整某种颜色（如让天空更蓝、草地更绿）
8. **锐化 + 降噪**：最后处理细节

## 常见调色风格参数参考

### 日系清新风格

| 参数 | 建议值 | 说明 |
|------|--------|------|
| 曝光 | +0.3 ~ +0.7 | 略微过曝，画面明亮 |
| 对比度 | -10 ~ -20 | 降低对比度，画面柔和 |
| 色温 | 偏冷（向蓝移动） | 蓝青色调是日系标志 |
| 高光 | -30 ~ -50 | 找回天空细节 |
| 阴影 | +20 ~ +40 | 提亮暗部 |
| 饱和度 | -10 ~ -20 | 低饱和更清新 |
| 色调曲线 | 暗部端点向上抬起 | 营造空气感（褪色效果） |
| HSL - 蓝色色相 | 偏青 | 天空偏青蓝色 |

### 复古胶片风格

| 参数 | 建议值 | 说明 |
|------|--------|------|
| 曝光 | 0 ~ -0.3 | 略暗一些有味道 |
| 对比度 | -5 ~ +5 | 保持适中 |
| 色温 | 偏暖（向黄移动） | 胶片感标志 |
| 高光 | -20 ~ -40 | 压暗高光 |
| 阴影 | +10 ~ +30 | 适度提亮暗部 |
| 饱和度 | -15 ~ -25 | 降低饱和度 |
| 色调曲线 | S 型曲线，暗部偏绿 | 暗部带绿是经典胶片特征 |
| 颗粒 | 中等 | 模拟胶片颗粒感 |
| 暗角 | 中等 | 画面边缘变暗 |

### 电影感风格

| 参数 | 建议值 | 说明 |
|------|--------|------|
| 曝光 | 0 ~ -0.5 | 电影通常偏暗 |
| 对比度 | +10 ~ +20 | 较高对比度 |
| 色温 | 偏冷或偏暖（取决于场景） | 蓝色偏科技感，橙色偏温暖 |
| 高光 | -30 ~ -50 | 压暗高光 |
| 阴影 | -10 ~ +10 | 保持暗部深沉 |
| 饱和度 | -10 ~ +10 | 适中或略降 |
| 色调曲线 | 强 S 型曲线 | 增加电影感对比 |
| HSL - 橙色明度 | 降低 | 让肤色有质感 |
| HSL - 蓝色饱和度 | +20 ~ +30 | 增强蓝色存在感 |

### HDR 风格

| 参数 | 建议值 | 说明 |
|------|--------|------|
| 曝光 | 0 | 从正常曝光开始 |
| 对比度 | +15 ~ +25 | 增强对比 |
| 高光 | -50 ~ -80 | 大幅压低高光 |
| 阴影 | +50 ~ +80 | 大幅提亮阴影 |
| 清晰度 | +30 ~ +50 | 增强细节质感 |
| 饱和度 | +10 ~ +20 | 色彩更鲜艳 |
| 锐化 | 较高 | 边缘更清晰 |
