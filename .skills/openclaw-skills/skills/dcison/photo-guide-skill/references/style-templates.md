# 摄影风格参数模板

Agent 通过匹配最接近的风格模板来交叉验证推断的拍摄参数。

## 使用方法

1. 观察照片特征（景深、色调、运动感等）
2. 从下方找到最匹配的风格
3. 用模板中的参数范围验证推断结果

---

## 人像糖水片

**特征描述**：背景虚化柔美，人物突出，肤色白皙通透，整体明亮柔和。

| 参数 | 典型范围 | 说明 |
|------|----------|------|
| 光圈 | f/1.4 - f/2.8 | 大光圈制造强虚化 |
| 快门 | 1/125s - 1/250s | 保证人物清晰 |
| ISO | 100 - 200 | 低 ISO 保画质 |
| 焦段 | 50mm - 85mm | 中长焦压缩背景 |
| 角度 | 平视或微俯 | 自然亲切 |
| 光线 | 逆光或散射光 | 肤色柔和 |

**后期方向**：提亮肤色、降低对比度、暖色调偏移、适度磨皮

---

## 日系清新

**特征描述**：画面偏亮、低对比度、色调偏冷或淡蓝绿色，有空气感，常见于日常街景和生活照。

| 参数 | 典型范围 | 说明 |
|------|----------|------|
| 光圈 | f/2.8 - f/4.0 | 适度虚化 |
| 快门 | 1/250s - 1/500s | 户外光线充足 |
| ISO | 100 - 200 | 干净画质 |
| 焦段 | 35mm - 50mm | 标准视角有生活感 |
| 角度 | 平视 | 日常视角 |
| 光线 | 散射光或柔和顺光 | 柔和不硬 |

**后期方向**：提高曝光、降低对比度、色温偏冷、高光加蓝、暗部加青绿

---

## 城市街拍

**特征描述**：画面真实、有故事感，人物和环境都清晰，色调自然或偏暖，有现场氛围。

| 参数 | 典型范围 | 说明 |
|------|----------|------|
| 光圈 | f/4.0 - f/5.6 | 保证前后景都清晰 |
| 快门 | 1/250s - 1/500s | 抓拍定格 |
| ISO | 200 - 800 | 适应多变光线 |
| 焦段 | 24mm - 35mm | 广角包含环境信息 |
| 角度 | 平视或低角度 | 有冲击力 |
| 光线 | 自然光，侧光为主 | 增加层次 |

**后期方向**：自然色彩、适度提高清晰度、可偏暖复古色调

---

## 夜景城市

**特征描述**：深色背景上有灯光、车轨、霓虹等光源，画面华丽有氛围感。

| 参数 | 典型范围 | 说明 |
|------|----------|------|
| 光圈 | f/8.0 - f/11 | 小光圈让灯光星芒化 |
| 快门 | 2s - 10s | 长曝光拍车轨和灯光 |
| ISO | 100 | 低 ISO 保证暗部干净 |
| 焦段 | 14mm - 35mm | 广角拍出城市壮阔感 |
| 角度 | 俯视或平视 | 取决于构图 |
| 光线 | 环境光源 | 需要三脚架 |

**后期方向**：提升暗部细节、增强灯光色彩、可加蓝紫色调、提高清晰度

---

## 美食静物

**特征描述**：食物色彩鲜艳诱人，背景简洁，光线柔和有质感，常见俯拍或 45 度角。

| 参数 | 典型范围 | 说明 |
|------|----------|------|
| 光圈 | f/2.8 - f/4.0 | 适度虚化背景 |
| 快门 | 1/60s - 1/125s | 静物不需要高速快门 |
| ISO | 200 - 400 | 室内光线适中 |
| 焦段 | 50mm（微距更佳） | 突出食物细节 |
| 角度 | 45 度或俯拍 | 最常见的美食角度 |
| 光线 | 侧光或逆光 | 突出食物质感和光泽 |

**后期方向**：提高饱和度和鲜艳度、暖色调、提升清晰度、可加轻微暗角

---

## 风光建筑

**特征描述**：全景清晰，远近都锐利，画面大气壮观，色彩饱和度高。

| 参数 | 典型范围 | 说明 |
|------|----------|------|
| 光圈 | f/8.0 - f/11 | 全景清晰 |
| 快门 | 1/125s - 1/250s（手持）或三脚架 | 保证画面锐利 |
| ISO | 100 - 200 | 最佳画质 |
| 焦段 | 14mm - 35mm | 广角拍出空间感 |
| 角度 | 低角度或平视 | 突出建筑气势 |
| 光线 | 黄金时段（日出日落） | 色彩最丰富 |

**后期方向**：提高清晰度和细节、增强天空色彩、HDR 效果、矫正透视变形

---

## 运动抓拍

**特征描述**：主体动作被凝固在瞬间，画面动感强烈，背景可能有虚化。

| 参数 | 典型范围 | 说明 |
|------|----------|------|
| 光圈 | f/2.8 - f/4.0 | 大光圈提高快门速度 |
| 快门 | 1/500s - 1/2000s | 凝固高速运动 |
| ISO | 400 - 1600 | 保证足够快门速度 |
| 焦段 | 70mm - 200mm | 长焦拉近拍摄 |
| 角度 | 平视 | 主体最有冲击力的角度 |
| 光线 | 自然光为主 | 户外光线充足 |

**后期方向**：提高清晰度和锐度、增强对比度、裁剪突出主体

---

## 宠物摄影

**特征描述**：宠物可爱生动，眼睛锐利有神，毛发展现细节，背景简洁。

| 参数 | 典型范围 | 说明 |
|------|----------|------|
| 光圈 | f/2.0 - f/4.0 | 虚化背景突出宠物 |
| 快门 | 1/250s - 1/500s | 宠物好动需要高速快门 |
| ISO | 200 - 800 | 室内室外兼顾 |
| 焦段 | 50mm - 85mm | 中长焦不影响宠物 |
| 角度 | 与宠物眼睛齐平 | 最亲切自然 |
| 光线 | 自然光或柔光 | 避免闪光灯吓到宠物 |

**后期方向**：锐化毛发细节、提亮眼神光、自然色彩、适度提高饱和度

---

## 旅行纪实

**特征描述**：画面包含人物和当地环境，有故事感和地域特色，色调真实自然。

| 参数 | 典型范围 | 说明 |
|------|----------|------|
| 光圈 | f/4.0 - f/8.0 | 保证人物和环境都清晰 |
| 快门 | 1/125s - 1/500s | 旅行中灵活抓拍 |
| ISO | 100 - 400 | 大部分场景够用 |
| 焦段 | 24mm - 50mm | 包含人文环境信息 |
| 角度 | 平视 | 尊重和融入感 |
| 光线 | 利用现场光线 | 纪实感强 |

**后期方向**：保持自然色彩、适度调整曝光、可加轻微胶片质感

---

## 复古胶片

**特征描述**：画面有颗粒感，色彩偏暖或偏绿，对比度偏低，有怀旧情绪和独特氛围。

| 参数 | 典型范围 | 说明 |
|------|----------|------|
| 光圈 | f/2.0 - f/4.0 | 模拟老镜头质感 |
| 快门 | 1/60s - 1/250s | 日常场景 |
| ISO | 400 - 800 | 模拟胶片颗粒感 |
| 焦段 | 35mm - 50mm | 经典胶片焦段 |
| 角度 | 自由 | 不拘泥于角度 |
| 光线 | 自然光 | 柔和自然 |

**后期方向**：降低饱和度、加颗粒噪点、色温偏暖、暗角效果、色彩偏移（暗部偏绿/青）
