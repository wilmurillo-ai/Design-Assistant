## 문제

성격 테스트도 해봤고, 1월 1일에 목표도 세워봤고, 장단점 리스트도 만들어봤습니다. 그런데 정작 중요한 순간에는 — 멈춰버립니다.

**"이 직장으로 옮겨야 할까?"**
**"이 관계가 맞는 걸까?"**
**"나는 제대로 된 방향으로 가고 있을까?"**

진짜 답은 이미 당신 안에 있습니다. 수백 개의 작은 결정들 — 당신을 흥분시킨 것, 지치게 한 것, 계속 돌아온 것, 조용히 피한 것 — 속에 흩어져 있을 뿐이죠.

**아무도 그 패턴을 추적하지 않았습니다. 지금까지는.**

## Identity Compass가 하는 일

당신의 대화를 관찰합니다. 판단하려는 게 아니라 — **듣기** 위해서.

선호를 표현하거나, 무언가를 거부하거나, 아이디어에 눈이 반짝이거나, 갈림길에서 망설일 때마다 — 시스템이 작은 나침반 바늘 하나를 포착합니다. 시간이 지나면 바늘들이 정렬됩니다. 방향이 드러납니다.

그래서 실제 결정을 마주했을 때, 추측 대신 이렇게 물을 수 있습니다:

> *"이 선택이 내가 지금까지 가던 방향과 같은 곳을 가리키고 있나?"*

### 실제 예시

| 당신이 말하면... | 시스템이 보는 것... |
|----------------|-------------------|
| "남의 플레이북 따르느니 내가 만드는 게 낫지" | 강한 자율성 시그널 |
| "연봉은 좋은데 그냥 시키는 거 하는 거잖아" | 안티패턴: 주인의식 없는 실행 |
| "사이드 프로젝트 하다가 시간 가는 줄 몰랐어" | 몰입 상태 → 핵심 가치 지표 |
| "솔직히 그 사람 사는 거 보면 부러워" | 열망 시그널 → 방향 후보 |

양식을 채우지 않습니다. 설문에 답하지 않습니다. 그냥... 대화합니다. 나침반이 나머지를 합니다.

## 무엇을 얻나요

### 🎯 당신의 방향 (H 벡터)
당신이 누구인지 — 누구여야 한다고 생각하는지가 아니라 — 를 포착하는 한 줄.

> *"구조보다 자율을 택하고, 연구를 통해 깊이를 쌓고, 편안함보다 혁신을 추구하는 사람."*

이건 별자리 운세가 아닙니다. 당신의 실제 결정에서 계산된 겁니다.

### 📊 정렬도 점수 (M)
최근 선택들이 진짜 방향과 얼마나 정렬되어 있는지 보여주는 0~1 사이의 숫자.

- **0.8+** → 완전히 잠겼습니다. 결정이 일관됩니다.
- **0.5-0.7** → 대체로 맞는 방향, 약간의 노이즈.
- **0.3 미만** → 여러 방향을 동시에 당기고 있습니다. 성찰할 때.

### ⚖️ 의사결정 시뮬레이션
갈림길에 서 있나요? 각 선택지를 시뮬레이션합니다:

```
선택지 A: 스타트업 오퍼 수락
  → 정렬도 0.68 → 0.72로 변화 ✅

선택지 B: 현 직장 유지
  → 정렬도 0.68 → 0.61로 변화 ⚠️
```

"선택지 A가 더 낫다"가 아닙니다. **"선택지 A가 더 '당신답다'."**

### 🗺️ 의사결정 지도
모든 과거 결정이 구슬인 인터랙티브 시각화 — 크기는 중요도, 색상은 정렬도. 패턴이 실시간으로 떠오릅니다.

## 이런 분을 위한 겁니다

- **이직 고민 중** — "떠나야 하는 건 아는데, 어디로?"
- **창업자** — "내가 정말 원하는 걸 만들고 있나?"
- **갈림길에 선 모든 사람** — "맞는 것 같은데 설명을 못 하겠어"
- **뻔한 조언에 지친 사람** — 이건 템플릿이 아니라 *당신의* 데이터로 만들어집니다

## 이것은 이런 게 아닙니다

- ❌ 성격 테스트 아닙니다 (자기보고 없음)
- ❌ 목표 설정 앱 아닙니다 (목표는 변하고, 방향은 지속됩니다)
- ❌ 대신 결정해주는 AI 아닙니다 (데이터를 주지, 결정을 내리지 않습니다)
- ❌ 클라우드 기반 아닙니다 (모든 건 당신 기기에 남습니다)

## 빠른 시작

### 1. 스킬 설치

```bash
# 방법 A: ClawHub (추천)
npx clawhub@latest install identity-compass

# 방법 B: 수동
git clone https://github.com/ico1036/identity-compass.git
cp -r identity-compass/ ~/.openclaw/workspace/skills/identity-compass/
```

### 2. Obsidian vault 설정

나침반은 의사결정 벡터를 [Obsidian](https://obsidian.md) 호환 vault에 저장합니다. 폴더만 만들면 됩니다:

```bash
mkdir -p ~/.openclaw/workspace/obsidian-vault/compass/{vectors,clusters,signals,prior}
```

```
~/.openclaw/workspace/obsidian-vault/compass/
├── vectors/          # 개별 결정 노트 (에이전트가 자동 생성)
├── clusters/         # 그룹화된 패턴 (자동 생성)
├── signals/          # 대화 신호 로그 (자동 생성)
└── prior/            # Phase 1 변증법 결과 (자동 생성)
```

### 3. 대화 시작

끝입니다. [OpenClaw](https://openclaw.ai) 에이전트와 평소처럼 대화하세요. 나침반이 의사결정 시그널을 감지하면 자동 활성화됩니다:

- 커리어 토론, 회사 비교
- "A 할까 B 할까?"
- 선호, 거부, 흥분
- 인생 방향 질문

에이전트가 백그라운드에서 벡터를 추출합니다. 양식도, 설문도 없습니다.

### 4. 지도 보기

```bash
cd ~/.openclaw/workspace/skills/identity-compass/scripts
python3 -m http.server 8742
# http://localhost:8742/visualize_2d.html 열기
```

시각화는 `compass_data.json`에서 로드됩니다 (에이전트가 자동 생성).

## 데이터 파일

`scripts/` 디렉토리의 JSON 파일들 — 모두 에이전트가 자동 관리하지만 직접 확인/수정 가능:

| 파일 | 용도 | 자동 생성? |
|------|------|:---:|
| `compass_data.json` | 시각화 데이터 (구슬 + H + 클러스터) | ✅ |
| `vectors.json` | 원시 의사결정 벡터 | ✅ |
| `magnetization.json` | H 방향 + M 점수 | ✅ |
| `sample_data.json` | 데모 데이터 | 포함됨 |

### compass_data.json 수동 작성

에이전트가 충분한 데이터를 모으기 전에 시각화를 미리 보고 싶다면:

```bash
cp scripts/sample_data.json scripts/compass_data.json
# compass_data.json을 본인 데이터로 수정
```

**필드 가이드:**

| 필드 | 설명 |
|------|------|
| `dir` | 3D 벡터: `[자율성, 깊이, 혁신]`. 범위: -1 ~ 1 |
| `w` | 가중치 (1-10). 재무적 영향, 시간 투자, 비가역성, 감정 강도 기준 |
| `cl` | 색상: `identity`, `hot`, `active`, `warm`, `tension`, `cool`, `avoid` |
| `match` | H 벡터와의 코사인 유사도 (0-1). 기회(회사)에만 사용 |
| `H.mag` | 자화도 (0-1). 당신의 결정들이 얼마나 정렬되어 있는지 |

> **참고:** 모든 개인 데이터 파일은 `.gitignore`에 포함 — 절대 기기 밖으로 나가지 않습니다.

### 물리 모델

Identity Compass는 **통계역학**에서 차용합니다. 각 결정은 방향과 강도를 가진 자기 스핀입니다.

- **H (자기장)** = 핵심 방향, 변증법 대화로 추출
- **M (자화도)** = 모든 스핀 벡터의 H 정렬도
- **감쇄** = 오래된 결정은 `0.95^(days/30)`로 페이드 — 지금의 당신이 더 중요

### 3축

| 축 | (+) | (−) |
|----|-----|-----|
| X | 자율성 | 구조 |
| Y | 깊이 | 폭 |
| Z | 혁신 | 안정 |

### Phase 1: 변증법적 추출

"목표가 뭐예요?"라고 묻지 않습니다. 네 가지 대화 패턴을 사용합니다:
1. **딜레마** — 강제 트레이드오프로 숨겨진 우선순위 드러냄
2. **시점 이동** — "5년 전의 당신이 지금 당신을 본다면?"
3. **모순 짚기** — "X라고 했는데 Y를 했잖아요..."
4. **완료형** — "사실 이미 알고 있지 않나요?"

### Phase 2: 베이지안 벡터 수집

각 결정에서 추출:
- 3D 방향 벡터
- 가중치 (1-10): 재무적 영향, 시간 투자, 비가역성, 감정 강도, 언급 빈도
- 축별 Beta(α,β) 사후확률 업데이트

### Phase 3: 가상 시뮬레이션

새 선택지 → 가상 구슬 → ΔM 계산 → 정렬도 비교.
