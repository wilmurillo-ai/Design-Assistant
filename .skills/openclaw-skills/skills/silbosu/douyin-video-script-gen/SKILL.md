# 爆款短视频逐字稿生成器

## 使用方法

当用户需要"生成短视频逐字稿"、"帮我写视频脚本"、"做一条爆款视频文案"时使用此技能。

## 工作流程

### 第一步：确认领域和主题

询问用户：
- 要做什么领域的视频？（如：AI、职场、育儿、商业等）
- 具体主题是什么？

### 第二步：选题（关键！）

**搜索入口**：今日头条搜索 https://www.toutiao.com/

**搜索技巧**：
1. 选择**"综合"**板块（不是视频！），筛选**"资讯"**
2. 筛选时间：近7天或近30天
3. 搜索关键词如"AI副业"、"AI趋势"等
4. 找阅读量高的爆款文章
5. 可以**整合多篇相关爆款**提炼一个主题

**找选题原则**：
- 优先选择近期的（7-30天内）
- 阅读量高的
- 可以从4-5篇相关文章中提取共同点，整合成一个主题

### 第三步：写开头（重要！）

开头公式（任选其一）：
1. "你能想象吗？...（描述一个惊人的场景）"
2. "信不信我能...（展示一个惊人的能力或结果）"
3. "我突然意识到...（分享一个颠覆认知的洞察）"

**注意**：开头是到**句号为止**的第一句完整话，不是只抄前三秒！

### 第四步：列观点

用三种方式组合观点：
1. 原文章观点（最多用40%）
2. 同类选题文章观点
3. 评论区高赞评论观点（点赞5000+）

每个观点要**详细展开**，包含：
- 观点核心句
- 解释说明
- 具体案例或数据
- 口语化表达

### 第五步：填内容（关键！）

**口语化原则**：
- 想象自己在说话，不是写文章
- 每15秒一个情绪钩子
- 内容要丰富，每个观点150-300字

**字数参考**：
- 4分钟 ≈ 960-1200字
- 5分钟 ≈ 1200-1500字
- 6-7分钟 ≈ 1500-2000字

### 第六步：检查

- 第一遍：通读检查流畅度
- 第二遍：检查情绪钩子位置
- 第三遍：确保口语化、接地气

## 输出格式

生成如下格式的逐字稿：

```
【开头】（约50-80字，用开头公式）
...

【观点1】（约200-300字，详细展开）
...

【观点2】（约200-300字，详细展开）
...

【观点3】（约200-300字，详细展开）
...

【观点4】（约200-300字，详细展开）
...

【结尾】（约30-50字）
...
```

## 关键提醒

1. **搜索选综合板块**→资讯，不是视频！
2. **选题必须"确定性"**——选已验证能爆的
3. **开头用公式**：你能想象吗？/ 信不信我能？/ 我突然意识到？
4. **开头到句号为止**，不是只抄前三秒
5. **口语化！口语化！口语化！**
6. **内容要丰富**：每个观点都要详细展开，有解释、有案例
7. **可整合多篇**：从4-5篇相关爆款提炼一个主题
8. 每15秒设置情绪钩子
