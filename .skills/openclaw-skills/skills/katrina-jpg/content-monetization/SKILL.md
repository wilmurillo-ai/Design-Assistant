# Content Monetization Playbook

幫你將內容變現既Skill - Twitter, YouTube, Newsletter, Products

## 功能
1. **Content Strategy** - 制定內容策略同niche定位
2. **Twitter Growth** - 幫你grow Twitter following
3. **Newsletter** - 整Ghost/Substack newsletter
4. **Digital Products** - 賣PDF, Notion templates, prompt packs
5. **Affiliate** -  setup affiliate links

## 安裝
```bash
npx clawhub@latest install content-monetization
```

## 使用
當用戶話「我想靠內容賺錢」既時候：
1. 了解佢既興趣同專長
2. 建議適合既platform同format
3. 提供具體既变现路徑

## 範例
```
User: 我想寫AI既blog賺錢
AI: 你想免費定收費？建議...
```
