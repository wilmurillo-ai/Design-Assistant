# 肉鸽地牢游戏 - 规格说明书

## 项目概述
- **项目名**: Rogue Dungeon
- **类型**: 经典肉鸽回合制地牢探索游戏
- **核心玩法**: 玩家在地牢中探索、战斗、收集装备，死了重来
- **目标用户**: 休闲玩家

## 视觉与渲染
- 终端/控制台风格界面
- ASCII 字符渲染地图
- 颜色区分：玩家(绿色)、敌人(红色)、物品(黄色)、墙壁(#)、地面(.)

## 游戏机制

### 角色属性
- **HP**: 生命值 (初始 100)
- **Attack**: 攻击力 (初始 10)
- **Defense**: 防御力 (初始 0)
- **Gold**: 金币 (初始 0)
- **Level**: 等级 (初始 1)
- **Exp**: 经验值 (初始 0)

### 地图系统
- 房间+走廊结构
- 房间数量: 5-8个
- 每层地牢 10x20 格子
- 随机生成墙壁、地面、敌人、物品

### 战斗系统
- 回合制：玩家先手
- 攻击 = 攻击力 - 敌人防御 (最小1)
- 敌人攻击 = 敌人攻击力 - 玩家防御 (最小1)
- 敌人 AI: 简单追击玩家

### 物品系统
- **武器**: +Attack
- **防具**: +Defense  
- **药水**: 恢复 HP
- **金币**: 收集用于购买

### 敌人类型
1. **史莱姆**: HP 20, ATK 5, DEF 0
2. **哥布林**: HP 30, ATK 8, DEF 2
3. **骷髅**: HP 40, ATK 12, DEF 5
4. **龙**: HP 100, ATK 20, DEF 10 (BOSS)

## 控制
- **W/A/S/D** 或 **方向键**: 移动
- **1-3**: 使用背包物品
- **R**: 重新开始

## 胜利条件
- 击败 BOSS 龙，到达下一层
- 无限层数

## 失败条件
- HP ≤ 0
