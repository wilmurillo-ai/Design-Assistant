### 1. 爆款标题创作

**标题公式库：**

| 类型 | 公式 | 例子 |
|------|------|------|
| 数字法 | 数字+痛点/好处 | "5年经验总结：面试官最讨厌的3种回答" |
| 悬念法 | 省略关键信息 | "那个月薪3000的女生，现在..." |
| 对比法 | 反差+冲击力 | "985学生和专科生，差距到底在哪？" |
| 疑问法 | 戳中好奇心 | "为什么你敷面膜没用？" |
| 命令法 | 强动词+结果 | "立刻停止！正在毁掉你皮肤的5个习惯" |
| 合集法 | 限定词+清单 | "2026年最值得买的10款粉底液" |
| 蹭热度 | 热点事件+关联 | "《哪吒2》票房破百亿，我也来蹭一波" |
| 身份法 | 特定人群+共鸣 | "月薪5000的女生，都吃什么早餐？" |

**标题禁区：**
- ❌ 不要标题党（标题与内容严重不符）
- ❌ 不要过于夸大（"震惊！""绝了！""太牛了！"已用烂）
- ❌ 不要中英混杂（显得不专业）
- ❌ 不要太长（超过20字容易被截断）

### 2. 正文写作技巧

**开头写法（黄金3秒法则）：**

```
✅ 好的开头：
- "我以为...结果..."（反差型）
- "姐妹们！这件事我忍不了了！"（情绪型）
- "本人亲自测试了30天，结果..."（信任型）
- "真的！这件事越早知道越好！"（紧迫型）
- "左滑看对比👉"（视觉引导型）

❌ 差的开头：
- "今天给大家分享一下..."（太平淡）
- "随着时代的发展..."（太官方）
- "首先...其次...最后..."（太死板）
```

**正文结构：**

```
【痛点引入】先戳中用户痛点/共鸣点
    ↓
【解决方案】给出你的方法/经验/产品
    ↓
【个人经历】用故事增加真实感
    ↓
【价值总结】强调能给读者带来什么
    ↓
【互动引导】结尾提问，提高评论率
```

**正文 emoji 使用规范：**

| 场景 | 推荐emoji | 数量 |
|------|----------|------|
| 标题 | 🔥💰✨📌⭐❗💡🎯 | 1-2个 |
| 正文分段 | 📍✨💡⚠️🎁🔥❗ | 每段1-2个 |
| 强调重点 | ❗❓💯👍🙌 | 适量 |
| 列表 | 1️⃣2️⃣3️⃣ / ✅❌⭐ | 每项1个 |

**emoji 使用禁忌：**
- ❌ 不要用老式符号（用的人太多了）
- ❌ 不要在同一段落堆超过5个emoji
- ❌ 不要用 Windows 不兼容的特殊符号
- ❌ 不要用 emoji 替代文字（它是辅助，不是主角）

### 3. 话题标签策略

**标签数量公式：**
- 粉丝 < 1000：每日9个标签（最大化曝光）
- 粉丝 1000-10000：6个标签（精准引流）
- 粉丝 > 10000：3-5个标签（维护调性）

**标签组合策略：**

```
【1-2个泛流量标签】
#小红书 #涨粉 #干货 #分享 #日常

【2-3个垂直领域标签】
#护肤 #穿搭 #职场 #美妆 #健身

【1-2个细分场景标签】
#学生党 #上班族 #新手妈妈 #30+姐姐
```

**标签选择原则：**
- ✅ 选有热度但不是过亿顶流的（竞争太激烈）
- ✅ 选跟自己账号定位匹配的
- ✅ 选最近流行的新兴话题（有流量红利）
- ❌ 不要选与内容无关的泛流量标签

### 🔥 功能一：爆款仿写器（内测中）

#### 什么是爆款仿写？

输入任意小红书爆款笔记，AI自动拆解其成功公式，然后套用结构生成新内容。

#### 使用方法

```
【爆款链接】：https://www.xiaohongshu.com/explore/xxxxx
【新品类/主题】：你想写的新主题
【目标人群】：新内容的目标读者
【核心卖点】：你想传达的核心信息
```

#### 仿写流程

```
Step 1：结构分析
├── 标题公式识别
├── 开头钩子类型
├── 正文结构拆解
├── emoji使用模式
└── 标签组合策略

Step 2：公式提炼
├── 爆款元素提取
├── 可复用模板
└── 变体方向建议

Step 3：内容生成
├── 套用标题公式
├── 保持开头钩子类型
├── 沿用正文结构
├── 替换内容主题
└── 生成3个标题备选
```

#### 仿写示例

**输入爆款结构分析：**
```
原笔记主题：护肤品空瓶记
标题公式：数字法 + 痛点
开头类型：个人经历共鸣型
正文结构：清单式 + 个人心得
emoji模式：每段1-2个，重点处强调
标签：垂直类 + 使用场景
```

**生成新内容（咖啡测评）：**
```
标题选项：
A. 空瓶了5罐！咖啡星人真实测评
B. 喝了30款咖啡，总结出这3个必买
C. 咖啡新手必看！从入门到入坑全记录

正文结构：
1. 开头：个人经历（从什么时候开始喝咖啡）
2. 中间：清单式测评（每款咖啡的亮点+槽点）
3. 结尾：个人推荐 + 互动引导

标签组合：
#咖啡 #咖啡测评 #咖啡推荐 #黑咖啡
#减脂期咖啡 #提神醒脑 #咖啡新手 #咖啡星人
```

#### 核心逻辑

#### 平台特性对比

| 平台 | 风格 | 标题特点 | 正文字数 | 标签 | 特殊要求 |
