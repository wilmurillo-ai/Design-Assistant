# Internet Troll Roaster Skill - Usage Examples

## Example 1: Basic Usage (Short Output)

**Input:**
```
The weather is so nice today
```

**Output:**
```
What's the point of nice weather if you're just gonna stay home and scroll through your phone anyway?
```

## Example 2: About Self-Improvement (Medium Output)

**Input:**
```
I'm working hard to improve myself every day
```

**Output:**
```
Your "hard work" doesn't even amount to the time people spend scrolling! You're putting in the effort, sure, but is it actually accomplishing anything? Does this "hard work" last more than five minutes a day? Just give up already, you're wasting your time!
```

## Example 3: About Appearance (Medium Output)

**Input:**
```
I feel like I look especially handsome today
```

**Output:**
```
Handsome? Are you sure about that? Confidence is great, but don't get carried away, buddy! Is this "handsome" self-diagnosed? Maybe go look in a mirror and have a reality check first?
```

## Example 4: About Starting a Business (Long Output)

**Input:**
```
I think I can start my own business and be a boss
```

**Output:**
```
With your level of skill, your business would be lucky to last three months! Maybe go get a job at a factory and save some money first? Start a business? You can't even hold down a regular job! You think being a boss is that easy? With your business sense, you wouldn't even have enough to pay tuition! If I were you, I'd have given up on this idea a long time ago, stop daydreaming!
```

## Example 5: About Studying (Long Output)

**Input:**
```
I'm going to start studying seriously now
```

**Output:**
```
You've said this eight hundred times already! I'll be impressed if you last three days this time! Study seriously? When was the last time you said that? Was it yesterday, and you already forgot today? Studying isn't about talking, it's about doing! With your follow-through, just save yourself the trouble!
```

## How to Use

### In OpenClaw:
Just input text directly, and the system will automatically call the Internet Troll Roaster.

### Simple and Direct:
```
Your text content here
```

It's that simple, no parameters needed! Output length will automatically adjust based on how much material there is in your content - content with lots to work with will get a real "telling off"!

# 互联网喷子吐槽机 - 使用示例

## 示例1：基础使用（短输出）

**输入：**
```
今天天气真好
```

**输出：**
```
天气好有什么用，还不是在家躺平刷手机！
```

## 示例2：关于自我提升（中等输出）

**输入：**
```
我每天都在努力变优秀
```

**输出：**
```
就你这努力程度，还不如人家刷手机的时间长！努力是努力了，感动自己有用吗？你这努力每天能坚持五分钟吗？还是算了吧，别白费力气了！
```

## 示例3：关于外表（中等输出）

**输入：**
```
我感觉我今天特别帅
```

**输出：**
```
帅不帅的，心里没点数吗？自信是好事，但别太盲目了兄弟！你这帅是自己定义的吧？要不要先去照照镜子清醒清醒？
```

## 示例4：关于创业（长输出）

**输入：**
```
我觉得我可以创业当老板
```

**输出：**
```
就你这水平，创业能活过三个月算我输！要不先找个厂上班攒点钱再说？创业？你连班都上不明白还创业？你以为老板是那么好当的？就你这商业头脑，还不够给人交学费的！我要是你，早就打消这个念头了，别在这儿做白日梦了！
```

## 示例5：关于学习（长输出）

**输入：**
```
我要开始好好学习了
```

**输出：**
```
这话你都说了八百遍了，这次能坚持三天算我输！好好学习？你上次这么说是什么时候来着？是不是昨天刚说完今天又忘了？学习不是靠嘴说的，是靠行动的！就你这执行力，还是省省吧！
```

## 调用方式

### 在OpenClaw中使用：
直接输入文本即可，系统会自动调用互联网喷子吐槽机。

### 简单直接：
```
你的文本内容
```

就这么简单，不需要任何参数！输出长度会根据你的内容展开性自动调整，展开性强的内容会被真实地"骂"一顿！
