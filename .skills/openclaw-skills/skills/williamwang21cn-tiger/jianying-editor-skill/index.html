常见问题 (FAQ)

看不到新生成的草稿？

剪映软件不会实时刷新列表。生成草稿后，请重启剪映，或者随便点击一个旧草稿再退出来，就能看到新的了。

自动导出失败？

自动导出脚本模拟了鼠标键盘操作。运行导出时，请不要动鼠标和键盘。仅支持 剪映 5.9 或早版。
