## 使用说明

### 如何选择 Category

根据用户问题中的关键词，匹配对应的信息类型：

**示例匹配规则：**
- "基本信息"、"注册信息"、"工商信息" → 01
- "股东"、"出资"、"持股" → 04
- "高管"、"管理人员"、"董事"、"监事" → 03
- "变更记录"、"变更历史" → 05
- "对外投资"、"投资的公司" → 08
- "股权冻结" → 09
- "股权出质"、"质押" → 12/13/14
- "动产抵押" → 15-21
- "行政处罚" → 22
- "年报"、"年度报告" → 23-31
- "历史"相关 + 具体信息类型 → 32-34

**智能体使用方式：**
1. 分析用户问题，提取企业名称作为 `keyword`
2. 识别用户关注的信息类型关键词
3. 在本映射表中查找对应的 `category` 编号
4. 调用脚本：`python scripts/query.py --keyword "企业名称" --category "01"`

### 示例

**用户问题：**"帮我查询南京水利公司的基本信息"
- keyword: "南京水利公司"
- category: "01"（企业基本信息）
- 命令：`python scripts/query.py --keyword "南京水利公司" --category "01"`

**用户问题：**"查一下腾讯的股东信息"
- keyword: "腾讯"
- category: "04"（企业股东及出资信息）
- 命令：`python scripts/query.py --keyword "腾讯" --category "04"`

**用户问题：**"查看阿里的高管都有谁"
- keyword: "阿里"
- category: "03"（企业主要管理人员信息）
- 命令：`python scripts/query.py --keyword "阿里" --category "03"`
