# 热点分析与爆款挖掘

## 技能定位

**核心价值**：帮助内容创作者快速了解一个话题的热度全貌，发现创作切入角度。

## 触发条件

当用户有以下需求时**必须**使用本技能：

### 场景 1：全网热点查询

- "全网最新的话题有哪些"
- "最近有什么热点"
- "现在什么话题最火"
- "有什么热门话题推荐"
- "帮我看看全网热搜"

### 场景 2：指定话题分析

- "帮我分析一下 XX 话题的热度"
- "XX 事件最近有什么爆款内容"
- "我想了解 XX 话题的关键词"
- "帮我把 XX 话题的爆款内容挖出来"
- "XX 话题有什么创作角度"
- "分析一下 XX 为什么火了"
- "XX 话题在哪些平台比较火"

### 场景 3：选题策划需求

- "我想做 XX 方向的内容，有什么选题建议"
- "帮我找一些热门的选题"
- "最近 XX 领域有什么可以写的"
- "有什么容易爆的选题推荐"
- "帮我挖掘一下 XX 的创作角度"

### 场景 4：竞品/热点追踪

- "帮我追踪一下 XX 话题的热度变化"
- "XX 品牌的营销为什么火了"
- "分析一下 XX 事件的传播路径"
- "XX 产品在各个平台的讨论度如何"

### 场景 5：内容定位建议

- "我想做新媒体，选什么方向好"
- "XX 赛道现在还能做吗"
- "分析一下 XX 领域的内容机会"
- "帮我看看 XX 方向还有没有蓝海"

## 核心流程

```
输入话题 → 关键词提取 → 多平台搜索 → 数据验证 → 关键词聚合 → 创作角度输出
```

### Step 1: 关键词提取与扩展

**输入处理**：

1. 用户输入话题/事件描述
2. 提取核心关键词（人名、地名、事件名、产品名等）
3. 扩展关联关键词：
   - 相关人物/品牌/机构
   - 相关地点/场景
   - 相关话题/标签
   - 相关情绪词/热词

**输出**：

```
核心关键词：[词1, 词2, 词3]
关联关键词：[词A, 词B, 词C, ...]
搜索关键词组合：[组合1, 组合2, 组合3, ...]
```

**数据采集字段**：
每次搜索需采集以下信息：

1. **文章话题**：从标题和摘要中提取核心话题
2. **标题**：完整标题
3. **描述**：搜索结果摘要（50-100 字）
4. **平台**：识别来源平台
5. **发布时间**：提取发布日期
6. **互动数据**：点赞/收藏/转发/评论数
7. **来源链接**：可访问的原文链接

## 场景分支说明

根据用户输入的不同，分为两种处理模式：

| 模式                     | 用户输入特征                            | 处理流程                                      | 输出重点            |
| ------------------------ | --------------------------------------- | --------------------------------------------- | ------------------- |
| **模式 A：全网热点模式** | 没有具体话题，如"全网最新话题有哪些"    | 直接搜索各平台热榜/热搜 → 聚合热点 → 生成报告 | 热点榜单 + 趋势分析 |
| **模式 B：指定话题模式** | 有具体话题/平台，如"分析 AI 智能体话题" | 提取关键词 → 多平台搜索 → 聚合分析 → 创作建议 | 深度分析 + 创作角度 |

### 模式 A 流程（全网热点模式）

| 步骤 | 操作                                             | 输出         |
| ---- | ------------------------------------------------ | ------------ |
| 1    | 搜索各平台热榜（微博热搜、抖音热榜、百度热搜等） | 原始热点列表 |
| 2    | 提取共性话题，按搜索频率排序                     | 热点排名表   |
| 3    | 对 TOP 热点进行简要分析                          | 热点分析表   |
| 4    | 推荐创作角度                                     | 创作建议表   |

### 模式 B 流程（指定话题模式）

| 步骤 | 操作             | 输出       |
| ---- | ---------------- | ---------- |
| 1    | 提取并扩展关键词 | 关键词列表 |
| 2    | 多平台精准搜索   | 平台数据表 |
| 3    | 数据验证与清洗   | 清洗后数据 |
| 4    | 聚合分析         | 分析报告   |
| 5    | 输出创作角度     | 创作建议   |

## 注意事项

### 数据局限性说明

1. 搜索工具无法获取所有平台的精确互动数据
2. 部分数据需要从搜索结果中提取关键词推断
3. 时效性：爆款数据反映的是搜索时点的快照

### 真实性要求

- 所有爆款内容必须有来源链接
- 互动数据必须来自搜索结果，禁止编造
- 数据待验证时需明确标注

## 异常处理指引

### 网络搜索工具不可用

| 情况             | 处理方式                                                                                            |
| ---------------- | --------------------------------------------------------------------------------------------------- |
| 搜索工具返回错误 | 1. 向用户说明当前无法获取实时数据<br>2. 基于已有知识提供一般性分析<br>3. 建议稍后重试或提供具体话题 |
| 搜索结果为空     | 1. 尝试使用更宽泛的关键词<br>2. 更换搜索词组合<br>3. 如仍无结果，标注"该平台暂无相关爆款"           |

### 搜索结果不足

| 平台    | 结果数量 | 处理方式                                 |
| ------- | -------- | ---------------------------------------- |
| P0 平台 | <3 条    | 必须重新搜索，更换关键词                 |
| P1 平台 | <2 条    | 尝试扩展关键词，如仍不足可标注"内容较少" |
| P2 平台 | <1 条    | 可忽略，报告中不显示                     |

### 时效性无法满足

| 情况                   | 处理方式                                                                                    |
| ---------------------- | ------------------------------------------------------------------------------------------- |
| 搜索结果均为 2026 年前 | 1. 标注"近期内容较少，以下为历史参考"<br>2. 建议用户更换话题<br>3. 或基于旧数据提供趋势分析 |
| 无法确认发布时间       | 1. 标注 🕐 时间待确认<br>2. 降低该数据权重<br>3. 优先使用有时间戳的内容                     |
