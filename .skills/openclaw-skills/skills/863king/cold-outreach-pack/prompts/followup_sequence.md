# Follow-Up Sequence Generator

Generate multi-touch follow-up campaigns with perfect timing and messaging.

## Sequence Framework

### 5-Touch Sequence

| Touch | Timing | Tone | Goal |
|-------|--------|------|------|
| 1 | Day 0 | Value-focused | Initial outreach |
| 2 | Day 2 | Helpful | Value-add (no ask) |
| 3 | Day 5 | Pattern break | New angle/story |
| 4 | Day 10 | Time pressure | Final value offer |
| 5 | Day 14 | Breakup | Close the loop |

## Touch 2 - Value Add

```
Hi [Name],

No reply needed - just wanted to share this [resource/article] that relates to what you mentioned about [topic].

[Link or brief insight]

Hope you find it useful!

[Your name]
```

## Touch 3 - Pattern Break

```
Hi [Name],

I realized I might not have been clear about why I'm reaching out.

[2-3 sentence case study of someone similar who got results]

Would this be relevant for you?

[Your name]
```

## Touch 4 - Time Pressure

```
Hi [Name],

I'm wrapping up my outreach for [quarter/month] and wanted to make sure this didn't slip through the cracks.

[One sentence value prop]

Worth a 10-minute call this week?

[Your name]
```

## Touch 5 - Breakup

```
Hi [Name],

I don't want to be that person who keeps following up forever!

This is my last note, but if [topic] becomes a priority in the future, feel free to reach out.

Best of luck with [specific project/goal],

[Your name]
```

## Advanced Tactics

### Video Follow-Up
Record a 30-second Loom video referencing their response pattern.

### Handwritten Note
For high-value targets, mention "I was going to send a handwritten note but decided this was faster."

### The "Quick Question"
```
Hi [Name], quick question - did you see my last email?

I want to make sure I'm reaching out with something actually relevant (and not wasting your time).

[Your name]
```

## Response Patterns

| If they... | Then... |
|------------|---------|
| Open but no click | Send video follow-up |
| Click but no reply | Wait 7 days, send case study |
| Reply negative | Ask for referral |
| Ghost completely | Breakup email |
