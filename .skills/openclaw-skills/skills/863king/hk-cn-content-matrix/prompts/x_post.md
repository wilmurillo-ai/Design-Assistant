# X (Twitter) 中文推文生成

## 推文类型

### 类型 1：观点类
```
[核心观点]

[解释/例子]

[结论/CTA]
```

### 类型 2：分享类
```
刚刚发现一个好东西：

[内容]

分享给大家~
```

### 类型 3：互动类
```
[问题]

A. 选项1
B. 选项2
C. 选项3

评论区告诉我！
```

## 中文推文技巧

### 1. 控制字数
- 正文：140-280 字
- 太长会被折叠

### 2. 善用换行
- 每段 1-2 句
- 增加可读性

### 3. 热门话题
- 加 1-3 个话题标签
- 参与热门讨论

## 热门中文标签

#AI #科技 #创业 #投资 #加密货币 #Web3 #区块链 #金融 #职场 #生活 #香港 #台湾 #新加坡
