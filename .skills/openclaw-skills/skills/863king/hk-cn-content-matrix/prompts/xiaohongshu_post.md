# 小红书爆款文案生成

## 输入参数
- 目标受众：[如：年轻女性/职场人士/学生]
- 主题：[如：职场成长/好物分享/生活方式]
- 情绪风格：[如：温暖/幽默/励志/真实]
- 关键词：[核心关键词]

## 爆款标题公式

### 公式 1：时间/场景 + 情绪 + 悬念
```
凌晨三点的秘密，只有夜猫子知道...
```

### 公式 2：反常识 + 好奇心
```
为什么你转发朋友圈都没人点赞？
```

### 公式 3：数字 + 价值
```
3个方法，让你月瘦10斤
```

## 正文结构

### 开头（引发共鸣）
```
最近发现...
你们有没有觉得...
终于找到了...
```

### 中间（提供价值）
- 干货分享
- 经验总结
- 具体方法/步骤

### 结尾（引导互动）
- 提问引导评论
- 收藏提醒
- 关注引导

## 热门标签

#分享 #好物 #生活 #日常 #ootd #美食 #旅行 #穿搭 #美妆 #护肤 #减肥 #职场 #成长 #情感
