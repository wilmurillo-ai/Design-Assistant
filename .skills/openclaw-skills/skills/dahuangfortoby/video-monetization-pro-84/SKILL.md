## 🚀 快速开始

### 模式 1：全自动 MV 创作（推荐）

```
输入："创作一个 MV" 或 "今日热点 MV"
输出：完整 MV + 4 平台发布 + 收益监控
```

**执行流程**：
1. 搜索当日热点（web_search，消耗 Google Custom Search 配额）
2. 生成 MV 主题和歌词（ai-video-script）
3. 法律合规审查（规则 13/15）
4. 生成 Suno 提示词（烟嗓女主捌十肆）
5. 创作分镜脚本（video-analyzer 格式）
6. 生成音乐（Suno API）
7. 生成 MV（TapNow.ai / 快手可灵）
8. 发布到 4 平台（bilibili-upload / youtube-publisher）
9. 设置收益监控（每日 09:00 自动报告）

---

### 模式 2：热点分析 + MV 主题

```
输入："分析今日热点" 或 "推荐 MV 主题"
输出：3-5 个可执行的 MV 主题方案
```

**输出内容**：
- 热点事件摘要（来源 + 热度数据）
- MV 主题建议（3-5 个）
- 爆款概率预测（基于历史数据）
- 预计收益范围（保守/乐观）
- 执行难度评估（低/中/高）

---

### 模式 3：歌词创作 + 法律审查

```
输入："写一首关于 [主题] 的歌词"
输出：完整歌词 + 法律审查报告
```

**审查维度**：
- 🔴 高风险：敏感单位/政府/色情暴力/侵权 → 删除
- 🟡 中风险：医疗/金融建议/夸大宣传 → 修改
- 🟢 低风险：基本合规 → 通过

**输出格式**：
```markdown
## 歌词
[完整歌词]

## 法律审查
- 宪法合规：✅
- 网络安全法：✅
- 广告法：⚠️ 第 3 段"最赚钱"改为"高收益"
- 著作权法：✅ 原创

## 风险等级
🟢 低风险（可发布）
```

---

### 模式 4：收益监控报告

```
输入："收益报告" 或 "监控账户"
输出：各平台收益数据 + 趋势分析
```

**监控平台**：
- 抖音（播放/点赞/涨粉/分成）
- B 站（播放/互动/花火商单）
- YouTube（播放/AdSense 收益）
