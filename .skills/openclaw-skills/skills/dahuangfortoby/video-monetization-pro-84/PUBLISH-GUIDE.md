## 发布流程

### 步骤 1：安装 ClawHub CLI

```bash
npm install -g clawhub
# 或
pnpm add -g clawhub
```

### 步骤 2：登录 ClawHub

```bash
clawhub login
# 使用 GitHub 账号登录（需注册至少 7 天）
```

### 步骤 3：配置付费技能

编辑 `SKILL.md`，确认 metadata 包含：

```yaml
metadata:
  clawdbot:
    paid: true
    price: 199
    currency: CNY
```

### 步骤 4：发布技能

```bash
cd /Users/huang/.openclaw/workspace/skills/video-monetization-pro

# 发布到 ClawHub
clawhub publish --paid --price 199 --currency CNY
```

### 步骤 5：验证发布

```bash
# 搜索技能
openclaw skills search "video-monetization-pro"

# 查看技能信息
openclaw skills info video-monetization-pro
```
