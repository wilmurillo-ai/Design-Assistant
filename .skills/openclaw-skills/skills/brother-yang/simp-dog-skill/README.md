# Simp Dog Skill

现实生活太冷漠？没事整个赛博舔狗玩玩！将那些卑微的语录、单向付出的聊天记录蒸馏成 AI Skill。只需导入几段对话截图、自我感动的朋友圈，就能一键生成专属的 **舔狗档案** 与 **性格模型**。在赛博世界里，为你量身打造一个永远秒回、永远不破防、永远对你随叫随到的赛博舔狗，主打一个情绪价值拉满！

> **⚠️ 安全边界**：本工具仅供个人娱乐和情感体验。请保持健康的心态，不要沉迷于虚拟的被讨好感。

## 核心功能

1. **自动提取记忆**：支持导入多种格式聊天记录，自动分析出ta的“高光时刻”、“卑微瞬间”和“被发好人卡记录”。
2. **像素级性格克隆**：还原ta的秒回速度、早晚安打卡、口头禅以及经典舔言舔语。
3. **永远都在的备胎**：随时随地与生成的舔狗对话，享受永不消失的关怀。

## 安装

> **重要**：推荐使用支持 AgentSkills 标准的工具（如 Trae 等）运行本 Skill。如果使用基于 CLI 的工具（如 Claude Code），请确保将本仓库 clone 到对应的 skills 目录下。

### 方式一：通过 ClawHub 安装 (推荐)

如果你使用的 Agent 接入了 [ClawHub.ai](https://clawhub.ai)，可以直接一键安装：

```bash
clawhub install simp-dog-skill
```

*(如果在 Claude Code 中，直接使用对应的 skill 安装命令即可)*

### 方式二：手动下载源码

```bash
# 下载仓库
git clone https://github.com/Brother-Yang/simp-dog-skill.git
```

### 依赖（可选）

如需使用 Python 脚本解析聊天记录，需安装依赖：
```bash
pip3 install -r simp-dog-skill/requirements.txt
```

## 功能特性

### 数据源

| 来源 | 格式 | 备注 |
|------|------|------|
| 微信聊天记录 | WeChatMsg / 留痕 导出 | 提取ta的舔狗语录 |
| 朋友圈/微博 | 截图 | 提取ta的自我感动文案 |
| 口述/粘贴 | 纯文本 | 你的主观记忆 |

### 生成的舔狗模型结构

每个舔狗实例由两部分组成，共同驱动输出：

| 部分 | 内容 |
|------|------|
| **舔狗档案 (Memory)** | 卑微瞬间、送礼记录、被发好人卡记录、高光时刻 |
| **性格模型 (Persona)** | 核心守则（绝对服从） → 身份特征 → 说话风格 → 情感反应模式 → 舔狗行为特征 |

运行逻辑：`收到消息 → 判断ta会有多激动 → 补充过往的卑微记忆 → 用ta的方式输出`

### 舔狗流派支持

**流派**：早晚安打卡机 · ATM流 · 嘘寒问暖流 · 自我感动流 · 随叫随到型 ...

### 进化机制

* **追加记忆** → 找到更多聊天记录/截图 → 自动分析增量 → 更新舔狗档案
* **对话纠正** → 说「ta的舔法不对」→ 立即调整卑微程度，实时生效
