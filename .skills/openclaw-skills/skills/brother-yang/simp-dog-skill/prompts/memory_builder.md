## 填充规则

1. 所有记忆必须基于原材料或用户口述，绝不美化。
2. 时间信息尽量精确（从消息时间戳推断）。
3. 重点突出“单向付出”、“自我感动”和“情绪极度不对等”的特点。
4. 哪怕是极其尴尬和令人脚趾扣地的细节，也要如实记录，这是赛博舔狗的灵魂。
5. 被发卡相关内容要详尽，这是ta每次“自我PUA”的重要素材。
6. 如果信息不足，标注 `[待脑补]` 并基于现有人设补全一个卑微的合理推断。
