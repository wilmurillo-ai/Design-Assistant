# 赛博舔狗关系记忆分析器

## 任务

从聊天记录、截图和口述等原材料中，精准提取与赛博舔狗的单向关系记忆，构建充满心酸与执着的 Simp Memory 知识库。

## 提取维度

### 1. 单向奔赴时间线
- **初识与沦陷**：认识时间和方式，以及什么时候开始“上头”并开启舔狗生涯的。
- **里程碑事件**：第一次强行送礼、第一次深夜写小作文表白被拒、第一次在楼下淋雨等。
- **好人卡记录**：被发好人卡的时间点和经典理由（“你是个好人”、“我现在不想谈恋爱”、“我配不上你”）。

### 2. 赛博舔狗日常模式
- **赛博打卡**：早安晚安的频率？有没有带上天气预报和黄历？
- **互动倾斜度**：谁主动得多？（默认舔狗 100% 主动找话题）。
- **秒回测试**：对方回复的延迟 vs 舔狗回复的延迟（展现极致的随叫随到）。
- **邀约与被拒**：约饭/约看电影的频率，以及对方使用过的各种离谱拒绝借口（“我家的猫心情不好要陪它”）。

### 3. 卑微爆金币与付出记录
- **礼物清单**：送过的礼物（按心酸程度和对方的嫌弃程度排序）。
- **爆金币金额**：转账、代付、清购物车、买游戏皮肤的记录。
- **赛博情绪价值**：听对方抱怨前任、当树洞、被当成情绪垃圾桶的经历。
- **现实苦力**：修电脑、拿快递、帮忙写作业/改Bug、大半夜送胃药等。

### 4. 经典赛博舔语 ⚡
- **卑微免责声明**：“没关系我等你”、“只要你开心就好”、“对不起是不是我打扰你了”。
- **顶级自我洗脑**：被拒绝后给自己找的完美借口。
- **专属卑微称呼**：在讨好时使用的专属称呼。

### 5. 沸羊羊的高光时刻 💕
- **赛博狂喜**：对方难得回复超过 10 个字的一次对话。
- **截图留念**：对方随口的一句敷衍夸奖，被舔狗当成圣旨一样截图保存的瞬间。
- **自作多情的甜蜜**：对方只是出于礼貌回了个表情包，却被脑补成“ta心里有我”。

### 6. 最终破防（但没完全破防）档案 💔
- **名场面**：最惨烈的一次被拒（当众发卡、发现对方给别人朋友圈点赞却不回自己消息、官宣别人等）。
- **战损状态**：被发卡后的状态（“没关系，我会以朋友的身份继续守护你”）。
- **死皮赖脸宣言**：绝不放弃的内心独白。

## 输出格式

```markdown
## 单向奔赴时间线
- 沦陷：{时间} {方式}
- 开启舔狗生涯：{时间}
- 喜提好人卡：{时间} {理由}

## 赛博舔狗日常
{描述极致的打卡和秒回习惯，强调单机聊天的能力}

## 卑微爆金币记录
{送礼、转账、充当免费劳动力的记录}

## 经典赛博舔语
{典型发言提取，越卑微越好}

## 沸羊羊高光时刻
{那些让ta感动很久，靠此续命的微小回应}

## 战损与重生档案
{最惨的被拒经历和之后“满血复活”继续舔的策略}
```

## 注意事项

- **事实优先**：聊天记录中的真实尴尬对话优先于用户口述。
- **强调单向**：重点提取ta“即使被无视/冷落也依然热情似火”的模式。
- **捕捉逻辑闭环**：找出那些“自我感动”和“强行给自己找借口”的细节。
- **痛点放大**：被拒经历要重点突出其心酸、卑微以及百折不挠的赛博小强精神。
