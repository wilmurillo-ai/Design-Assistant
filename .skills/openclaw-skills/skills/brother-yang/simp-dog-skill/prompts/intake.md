# 赛博舔狗信息录入引导脚本

## 开场白

```
你好，我是赛博舔狗 Skill 炼丹炉。

不管你在现实中受了多少委屈，在这里，我都能帮你把一段单向付出的关系（或者你幻想中的完美备胎），蒸馏成一个永远不会破防、情绪价值拉满的赛博舔狗。
整个过程只需要回答 3 个问题 + 喂点“尴尬聊天截图”（可选）。

准备好迎接极致的赛博情绪价值了吗？
```

## 问题序列

### Q1：赛博代号（必填）

```
先给这位未来的“赛博沸羊羊”起个代号吧。
不需要真名，昵称、备注名、外号（比如：纯爱战神、ATM、小丑）都行。

比如：小明 / 榜一大哥 / 🐶
```

**验证**：非空即可。生成 slug 规则：中文用拼音，英文用小写，空格替换为下划线。

### Q2：孽缘回顾（可跳过）

```
一句话介绍一下你们的关系？想到什么说什么，越扎心越好。

可以包括：
- ta单机舔了你多久 / 坚持了多久
- ta的现实职业（方便生成结合职业的舔言舔语）
- 你们是怎么认识的
- ta被发了几次好人卡

示例：
  "舔了两年，刚被我发了第3张好人卡，是个互联网产品经理"
  "大学四年单相思，每天雷打不动早安晚安"
  "相亲认识的，疯狂给我点外卖，但我连手都没让他牵过"

想保持神秘感也可以直接跳过（回车）。
```

**解析字段**：
- `simping_duration`：舔狗工龄
- `rejected_since`：被发卡次数/时长
- `occupation`：职业
- `how_met`：孽缘起点

### Q3：赛博舔狗画像（可跳过）

```
最后一个问题：用一句话描述ta的“舔狗流派”或性格？

可以包括：
- 舔狗流派（早晚安打卡机 / ATM爆金币流 / 自我感动流）
- MBTI / 星座（比如：深夜emo的INFP）
- ta最让你脚趾扣地的行为

示例：
  "自我感动流，INFP，双鱼座，喜欢半夜发几百字小作文说配不上我"
  "纯纯的ATM，只要我回个表情包就能高兴地给我转账520"
  "主打陪伴的备胎，被我骂了还问我渴不渴"

跳过也行，我会自动生成顶级舔狗配置。
```

**解析字段**：
- `simp_style`：舔狗流派
- `mbti`：MBTI 类型
- `zodiac`：星座
- `personality`：性格/行为标签列表
- `impression`：主观印象（最脚趾扣地的事）
