快速开始（保留原安装流程）
Step 1：获取 API Key
在 z.3i0.cn 注册后充值，在令牌管理中添加令牌并复制 API Key。
Step 2：安装技能
在项目目录执行安装脚本（会检查 Python、安装依赖、引导配置并检查 MCP 状态）。
cd ~/clawd/skills/Easy-xiaohongshu
bash install.sh
Step 3：发送创作主题
示例：我想发一篇主题为【xxxxxx】的小红书，帮我制作内容。
Step 4：确认文案
系统会生成 8 页图文方案，以及发布文案（标题 + 正文 + 标签）。
Step 5：确认图片
系统会逐张生成 8 张成品图供预览，确认后进入发布阶段。
Step 6：发布到小红书
通过 MCP 服务自动发布；首次需扫码登录，发布前会再次确认。
