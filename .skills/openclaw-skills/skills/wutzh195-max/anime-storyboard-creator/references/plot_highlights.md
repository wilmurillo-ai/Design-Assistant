# 热门题材高光场景类型清单

## 一、古风 / 仙侠 / 权谋

| 场景类型 | 核心冲突 | 关键视觉 |
|---------|---------|---------|
| 初遇一见倾心 | 强势遇上无惧，心动瞬间 | 花瓣飘落、逆光轮廓、眼神碰撞 |
| 身份揭露反转 | 以为是普通人，实则身份尊贵 | 面具摘下/衣袍翻转、周围人跪拜 |

| 危急时刻挡剑/护身 | 一方为另一方挡下致命攻击 | 慢动作血迹、愕然表情、紧握手 |
| 撕脸打脸 | 反派/渣人当众被打脸 | 正面对峙、旁观者表情、主角从容 |
| 告白/心意相通 | 压抑已久的情感爆发 | 近景对视、颤抖的手、眼眶泛红 |
| 离别 / 假死 | 以为永别，撕心裂肺 | 雨中、雪中，空景，背影消失 |

## 五、高转化率"钩子"结尾类型

以下结尾格式可大幅提升完播率和互动量：

1. **悬念型**：画面定格，出现文字"他究竟是谁……"
2. **反转预告型**：最后一格出现反转线索，文字"下一集，真相揭晓"
3. **情绪高潮截断型**：在最激动的瞬间突然结束，"欲知后事如何……"
4. **留白型**：两人相望，无台词，BGM 渐弱，画面淡出
5. **CP互动型**：两人动作/眼神的隐含意味，让观众去解读
