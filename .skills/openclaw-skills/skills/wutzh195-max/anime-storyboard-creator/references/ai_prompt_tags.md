# AI 绘图风格标签库（动漫短视频专用）

## 画面风格标签

### 古风/仙侠
```
ancient chinese style, xianxia, ink painting, traditional chinese clothing, hanfu,
flowing robes, jade accessories, misty mountains, palace architecture, lanterns,
cherry blossom, lotus pond, moonlight, scroll painting aesthetic
```

### 现代都市
```
modern cityscape, contemporary fashion, urban setting, neon lights, coffee shop,
office building, luxury interior, street photography style, bokeh background,
cinematic lighting, metropolitan atmosphere
```

### 玄幻/奇幻
```
fantasy world, magical atmosphere, cultivation novel style, divine light,
spirit energy effects, mystical runes, floating islands, dragon elements,
ancient temple ruins, ethereal glow, qi aura effects
```

### 甜宠/恋爱
```
romantic atmosphere, soft lighting, pastel colors, warm tone, sparkle effects,
flower petals falling, golden hour lighting, intimate distance, sweet expression,
shoujo manga style, dreamlike background, heart bokeh
```

### 悬疑/反转
```
dramatic lighting, high contrast, shadow play, cinematic noir,
mysterious atmosphere, cold blue tone, tense expression, rain effects,
dark background, spotlight focus, psychological thriller aesthetic
```

### 逆袭/爽文
```
power aura, golden light effect, triumphant expression, crowd background,
epic scale, rising sun, dramatic angle low shot, victorious pose,
transformation scene, glowing eyes, confident posture
```

## 景别对应构图建议

| 景别 | AI Prompt 构图关键词 |
|------|---------------------|
| 远景 | `wide shot, establishing shot, full environment visible, tiny figure` |
| 全景 | `full body shot, character and environment balanced, landscape visible` |
| 中景 | `medium shot, waist up, two characters in frame, background visible` |
| 近景 | `medium close-up, chest up, facial expression visible, shallow depth` |
| 特写 | `close-up shot, face focus, emotional expression, background blurred` |
| 大特写 | `extreme close-up, eye detail, lips or hand detail, macro texture` |

## 角色一致性描述模板

每格【画面】中必须重复以下角色固定描述：

**示例：古风男主**
```
[character: tall male, black long hair tied up partially, ice-cold expression,
black and dark gold hanfu robes, jade pendant, sharp jawline, intense dark eyes]
```

**示例：古风女主**
```
[character: slender female, black hair in elegant updo with hairpin, 
light blue silk hanfu, gentle smile, bright almond eyes, delicate features]
```

**示例：现代男主**
```
[character: tall male, short neat black hair, sharp features, dark business suit,
cold expression, broad shoulders, confident posture]
```
