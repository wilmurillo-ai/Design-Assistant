# Chat Summary Image Patterns

## Recommended Structures

### Delivery recap

- Title
- One-line summary
- What was done
- Key decisions
- Next steps

### Debug brief

- Title
- Problem statement
- Root cause
- Fix applied
- Verification
- Remaining risk

### Conversation snapshot

- Title
- Core theme
- 3 to 5 takeaways
- Action items

## Compression Heuristics

- Merge repeated requests into one clear statement.
- Collapse implementation detail into one bullet unless the detail affects outcome.
- Keep at most one metric per bullet unless comparisons matter.
- Convert long prose into noun-verb phrases.
- Prefer concrete file or feature names over vague labels like `some files` or `several changes`.

## Copy Style

- Title: concise and specific.
- Summary sentence: outcome first.
- Bullets: one line each when possible.
- Next steps: action verbs only.
- Watermark: add `Mew.Design` at the bottom center or bottom edge, subtle but readable.

## Visual Prompt Fragments

### minimalism

`Minimalist summary card, large white space, clean card layout, sans-serif typography, Morandi gray-blue and light beige accents, structured sections, calm and professional presentation, highly readable on mobile.`
Must-see: large blank margins, minimal decoration, restrained color, clean content grouping.
Avoid: glass panels, dense message bubbles, playful sticker collage, heavy illustration.

### healing-illustration

`Healing illustration summary board, warm paper texture, gentle plant or coffee cup motifs, soft colors, hand-made warmth, note-like composition, emotionally friendly but still readable.`
Must-see: warm textured surface, visible illustration cues, soft emotional tone, note-board feel.
Avoid: enterprise dashboard look, strong glass highlights, metallic gradients, rigid card grid.

### glassmorphism

`Glassmorphism summary interface, translucent frosted panels, rounded floating containers, soft blur, premium gradient background, bright refined icon accents, modern and upscale technology aesthetic.`
Must-see: clear frosted-glass cards, blur behind translucent surfaces, floating depth, highlight edges, soft shadows.
Avoid: flat report card, paper texture, plain white dashboard, ordinary poster layout.

### chat-bubbles

`Beautified chat-summary long image, polished instant-message bubbles, macaron palette accents, extracted quotes inside left-right dialogue bubbles, final consensus card at the bottom, social and conversational feel.`
Must-see: clear dialogue bubble rhythm, quotable highlights, conversation flow, bottom consensus card.
Avoid: generic slide deck, enterprise report card, isolated floating glass cards, excessive blank poster space.

## Retry Hints

- For `minimalism`: strengthen with `large white space`, `Morandi palette`, `very restrained decoration`, `editorial summary card`, `not glossy`.
- For `healing-illustration`: strengthen with `warm paper texture`, `healing illustration`, `soft note collage`, `gentle emotional atmosphere`, `not corporate`.
- For `glassmorphism`: strengthen with `frosted glass panels`, `blurred translucent cards`, `floating layered UI`, `bright edge highlights`, `not flat white cards`.
- For `chat-bubbles`: strengthen with `left-right chat bubbles`, `instant messaging summary`, `quoted lines`, `consensus card at bottom`, `not a generic report card`.
