## 当前状态

- 优先级：高 / 中 / 低

## 关键入口
- 状态文件：`STATUS.md`
- 待办列表：`TODO.md`
- 决策记录：`DECISIONS.md`
- 日志记录：`LOG.md`
- 参考资料：`REFERENCES.md`
- 交接文件：`HANDOFF.md`

## 恢复项目时怎么开始
1. 先读 `STATUS.md`
2. 再读 `TODO.md`
3. 如需背景，再看 `DECISIONS.md`、`LOG.md`、`REFERENCES.md`
