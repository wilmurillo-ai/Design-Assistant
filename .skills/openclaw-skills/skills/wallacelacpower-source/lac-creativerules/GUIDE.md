# LAC POWER 品牌设计指南 v2.0

## 品牌概述

LAC POWER 是一家专注于清洁能源解决方案的企业，致力于通过可再生能源照亮世界的每一个角落。

| 项目 | 内容 |
|------|------|
| 品牌名 | LAC POWER |
| Slogan | Renewable energy, light every corner of the world. |
| 核心价值 | 清洁、可持续、全球视野、科技创新 |
| 目标市场 | B2B客户 + B2C终端用户（非洲及全球新兴市场） |

## Logo使用规范

### Logo变体

| 类型 | 文件 | 使用场景 |
|------|------|----------|
| 完整文字Logo | LPW.jpg | 正式商务场合、文件、报告 |
| 图形化Logo | LPI.png | 简化场景、图标、角落标识、社交媒体 |

### 使用规则

**位置规则：**
- 默认位置：左上角
- 特殊情况：底部居中（仅当用户明确要求时）

**尺寸规则：**
- 最小宽度：30px（确保清晰可辨）
- 建议占比：画面5-10%
- Logo周围留白：至少Logo高度的1/2

**禁止事项：**
- 禁止拉伸或扭曲
- 禁止改变Logo颜色
- 禁止添加阴影或描边效果
- 禁止放置在视觉冲突区域

## 设计风格指南

### 1. 写实风格（产品展示）

- 使用真实的光照和阴影
- 材质纹理清晰可辨
- 避免过度抽象或扁平化
- 产品渲染图需专业级质量

### 2. 商业质感（B2B物料）

- 干净整洁的版面
- 专业的排版间距
- 高端但不过度奢华
- 数据可视化清晰
- 参考Dyness品牌调性

### 3. 活力亲和（B2C物料）

- 明亮但不过于花哨
- 情感连接元素
- 生活化场景融入
- 参考Felicity/iTel品牌调性

### 4. 科技感

- 现代几何元素
- 发光效果和光晕（适度）
- 线条简洁有力
- 融入能量流动、电路等符号

## 受众风格指南

### B2C受众：活力亲切

**用户画像：**
- 家庭用户、户外爱好者、旅行者
- 关注产品实用性、生活品质
- 追求性价比和便捷性

**文案风格：**
- 口语化、亲切
- 强调生活场景
- 情感共鸣
- 行动号召直接

**视觉风格：**
- 明亮色调
- 情感连接元素
- 生活化场景
- 参考Felicity/iTel

### B2B受众：专业严谨

**用户画像：**
- 经销商、工程商、企业采购
- 关注产品可靠性、技术参数
- 决策周期长、理性决策

**文案风格：**
- 技术术语准确
- 数据导向
- 专业可信
- 强调解决方案

**视觉风格：**
- 专业感
- 数据可视化
- 技术表达
- 参考Dyness
