# 五行在六爻卦象分析中的作用

## 🔥 五行核心概念

五行（木、火、土、金、水）是能量相位，而非物质本身：
- **木**：生长、创新、开始
- **火**：热烈、显明、扩散
- **土**：稳定、承载、转化
- **金**：刚硬、收敛、决断
- **水**：流动、隐藏、滋养

### 相生相克关系

```
相生：木 → 火 → 土 → 金 → 水 → 木
相克：木 → 土 → 水 → 火 → 金 → 木
```

## 🎯 六爻中的五行应用

### 1. 每爻配五行

六爻中每爻不仅有阴阳属性，还配五行：
- 爻的五行由其在地支中的位置决定
- 同一卦中不同爻可能有不同五行

### 2. 六亲体系

| 六亲 | 代表 | 生克关系 |
|------|------|----------|
| 父母 | 长辈、保护 | 生我者为父母 |
| 兄弟 | 同辈、竞争 | 与我同类 |
| 子孙 | 晚辈、释放 | 我生者子孙 |
| 妻财 | 妻子、财富 | 我克者妻财 |
| 官鬼 | 权威、小人 | 克我者官鬼 |

### 3. 用神（Significator）

用神是代表问事者或核心问题的爻。

**选择用神**：
- 问父母事 → 父母爻为用神
- 问财运 → 妻财爻为用神
- 问官运 → 官鬼爻为用神

**分析用神**：
- 被生 → 有利、有人帮助
- 被克 → 不利、有人阻碍
- 旺相 → 能量充足
- 休囚 → 能量不足

## 🏛️ 宫位五行系统

### 八卦宫位

| 宫 | 五行 | 首卦 |
|----|------|------|
| 乾宫 | 金 | 乾为天 |
| 兑宫 | 金 | 泽天夬 |
| 离宫 | 火 | 离为火 |
| 震宫 | 木 | 震为雷 |
| 巽宫 | 木 | 风天大畜 |
| 坎宫 | 水 | 坎为水 |
| 艮宫 | 土 | 艮为山 |
| 坤宫 | 土 | 坤为地 |

### 宫位的作用

1. **确定世应**：每宫卦的世爻和应爻位置固定
2. **六亲归属**：根据宫五行确定各爻六亲
3. **变卦参考**：变卦仍在同一宫系统中分析

## 🔮 五行生克在解卦中的应用

### 基本判断

| 关系 | 对用神影响 | 含义 |
|------|------------|------|
| 生 | ✅ 有利 | 有人帮助、支持 |
| 克 | ❌ 不利 | 有人阻碍、小人 |
| 同 | ➡️ 中性 | 同类竞争或合作 |
| 被生 | 📈 付出 | 能量输出、帮助他人 |
| 被克 | ⚠️ 受制 | 被动接受、压力大 |

### 旺相休囚

| 状态 | 含义 | 力量 |
|------|------|------|
| 旺 | 当令 | 最强 |
| 相 | 我令 | 强 |
| 休 | 退让 | 中 |
| 囚 | 衰败 | 弱 |
| 死 | 失令 | 最弱 |

## 📝 分析示例

> **问：求职能否成功？**
> - 用神：官鬼爻（代表工作机会）
> - 官鬼爻五行为金，得土爻相生
> - 无火爻克制
> - **结论**：用神旺相且得生，求职有利

---

## 🌟 五行与动爻

动爻是变化的关键：
1. **动爻的五行属性**决定变化方向
2. **动爻生克**用神，影响事态发展
3. **变卦后的五行**分析新的能量格局

---

## ⚠️ 注意事项

1. 五行是能量分析工具，不是绝对预测
2. 需要结合卦象整体结构判断
3. 用神选择是分析的第一步
4. 生克关系要综合考虑，不能只看一点
