# 六爻案例记录模板

## 记录说明

记录每次占卜的过程和结果，便于：
- 追踪问事发展
- 验证卦象准确度
- 总结解卦经验
- 形成个人占卜档案

## 使用示例

```
【基本信息】
日期：2026年4月20日 10:30
问事：长子升学考试结果
占问者：王先生

【起卦信息】
第1次（初爻）：两正一背（—少阳）
第2次（第2爻）：三枚背面（X老阴）
第3次（第3爻）：两背一正（--少阴）
第4次（第4爻）：三枚正面（O老阳）
第5次（第5爻）：两正一背（—少阳）
第6次（上爻）：两背一正（--少阴）

主卦：山水蒙
变卦：风山渐

【用神分析】
用神：官鬼爻（二爻）
旺衰：官鬼爻得日建相生
动爻：二爻动（老阴变阳）、四爻动（老阳变阴）

【解卦判断】
吉凶：吉
原因：用神旺相，动爻生助
应期：三个月内见分晓

【后续追踪】
记录时间：2026年4月20日
事情进展：孩子顺利通过考试
验证结果：☑ 准确

【备注】
考试结果比预期好
```
