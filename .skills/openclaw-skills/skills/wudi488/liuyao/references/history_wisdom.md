## 二、孙子兵法策略库

### 2.1 卦象对应策略

| 卦象状态 | 孙子兵法 | 应对策略 |
|----------|----------|----------|
| **用神旺相，机遇来临** | "不动如山" | 按兵不动，等待时机 |
| **用神休囚，困难重重** | "敌进我退" | 以退为进，保存实力 |
| **动爻相生，有人相助** | "天时不如地利" | 借势而为，顺势而上 |
| **动爻相克，小人作祟** | "知己知彼" | 谨慎行事，防人之心 |
| **变卦有利，趋势向好** | "侵掠如火" | 乘胜追击，把握机会 |
| **变卦不利，趋势向下** | "疾如风" | 快速撤退，避免损失 |

### 2.2 决策三十六计

| 原文 | 卦象解读 | 应用场景 |
|------|----------|----------|
| **瞒天过海** | 暗度陈仓 | 表面顺从，暗中准备 |
| **以逸待劳** | 静待时机 | 养精蓄锐，后发制人 |
| **趁火打劫** | 趁势而起 | 抓住对方弱点 |
| **借刀杀人** | 借力打力 | 利用他人之力 |
| **趁虚而入** | 乘隙捣虚 | 抓住对方空虚之时 |
| **金蝉脱壳** | 全身而退 | 化解危机，保全实力 |

### 2.3 知己知彼篇

```
孙子曰：知彼知己，百战不贻

六爻应用：
├── 知彼 → 卦中对手/小人状态
│   └── 看官鬼爻、动爻克我者
└── 知己 → 卦中自我/用神状态
    └── 看用神旺衰、生助克害
```

## 四、历史智慧应用示例

### 案例：问能否跳槽

```
传统解读：
- 用神官鬼爻旺相 → 有机会
- 动爻相生 → 有人推荐

➕ 历史智慧：
- 参照"韩信跳槽"：从项羽→刘邦，终成大器
- 时机判断：需卦变益，需耐心等待
- 策略："敌进我退"——现在不宜冲动

综合建议：
✅ 有机会，但需等待合适时机
✅ 参照韩信，先积累后行动
⚠️ 切忌学项羽负气出走
```

### 案例：问合伙生意

```
传统解读：
- 妻财爻旺 → 财运不错
- 兄弟爻动 → 有竞争

➕ 历史智慧：
- 参照"桃园结义"vs"曹操孙权联盟"
- 合伙要看"义"——志同道合
- 警惕"孙刘联盟"的内部矛盾

综合建议：
⚠️ 合伙可行，但需谨慎选择合伙人
✅ 参照刘备，合伙要有共同目标
⚠️ 制定规则在前，避免日后纷争
```
