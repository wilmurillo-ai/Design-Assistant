## 六、案例学习要点

### 总结规律

| 情况 | 判断依据 |
|------|----------|
| 官鬼爻旺相 | 有利变动，如升职、考试 |
| 妻财爻旺相 | 财运亨通 |
| 官鬼爻休囚 | 不利变动 |
| 妻财爻受克 | 破财、亏损 |
| 动爻生用神 | 吉事可成 |
| 动爻克用神 | 凶事需防 |

### 常见应期

| 情况 | 通常应期 |
|------|----------|
| 旺相发动 | 当日或当月 |
| 休囚发动 | 数月后 |
| 冲合应期 | 逢冲或逢合之日 |
| 墓库 | 冲墓之日 |
