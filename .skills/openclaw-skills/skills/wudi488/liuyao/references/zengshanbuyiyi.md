# 《增删卜易》精华摘要

> 清·野鹤老人著，被公认为六爻入门第一书

---

## 核心思想

### 一、实证主义原则
- **只保留经验证的理论**
- 反对牵强附会的解释
- 强调"实践是检验真理的唯一标准"

### 二、解卦三层次
```
第一层：判断吉凶（核心）
第二层：确定应期（时间）
第三层：分析细节（辅助）
```

---

## 精髓章节

### 1. 用神章（最重要）

**用神选取总则**：
| 问事 | 用神 | 附注 |
|------|------|------|
| 问财 | 妻财爻 | - |
| 问官 | 官鬼爻 | - |
| 问事 | 父母爻 | 文书、合同、房屋 |
| 问出行 | 父母爻 | 车船、交通 |
| 问疾病 | 官鬼爻 | - |
| 问诉讼 | 官鬼爻 | 对手方 |
| 问行人 | 用神爻 | 妻财爻代表妻子 |

**特殊用神**：
- 问家宅：父母爻
- 问仓库：妻财爻
- 问手艺：子孙爻

---

### 2. 动变章

**动爻判断**：
- 动则变，变则有事
- 动爻为事情发生的征兆
- 越旺越易发生

**生克规则**：
```
用神旺相 + 动爻生扶 → 吉
用神休囚 + 动爻克冲 → 凶
动爻化回头克 → 凶
动爻化回头生 → 吉
```

---

### 3. 旺衰章

**月建（当令者旺）**：
| 月份 | 旺的五行 |
|------|----------|
| 寅卯月 | 木 |
| 巳午月 | 火 |
| 申酉月 | 金 |
| 亥子月 | 水 |
| 辰戌丑未 | 土 |

**日建**：
- 日建为当天的主宰
- 与月建同功，能生克冲用神
- 旺相休囚的参考

---

### 4. 吉凶章

**判断总纲**：
> "吉凶大爻为主，衰旺扶抑为辅"

**步骤**：
1. 先看用神旺衰
2. 兼看动爻变化
3. 参合日建月建
4. 综合判断吉凶

---

### 5. 应期章

**应期推断**：
- 远应年月，近应日时
- 以生旺墓绝为应期
- 以冲合为应期

**常见应期**：
| 情况 | 应期 |
|------|------|
| 用神动而生之 | 当日或当月 |
| 用神旺而入墓 | 冲墓之日 |
| 用神被冲 | 逢合之日 |
| 忌神动克 | 逢生之日 |

## 经典案例（简略）

### 案例一：求财
```
占得：妻财爻旺相发动
解：财爻旺动，财之将得
断：近日可得
验：果然三日内进财
```

### 案例二：求官
```
占得：官鬼爻临日建
解：官旺扶身，仕途有望
断：近期有升迁
验：一月后升迁
```

### 案例三：疾病
```
占得：官鬼爻休囚被克
解：官鬼受制，病将痊愈
断：调理后可康
验：两月后康复
```

3. **融会旺衰章** - 判断旺衰标准
4. **实践验证** - 每日占卜验证

1. 从《火珠林》理解原理
2. 从《黄金策》深化理论
3. 从《易隐》提升象数
