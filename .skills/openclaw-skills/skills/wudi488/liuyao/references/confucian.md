## 二、论语情境应用

### 情境1：卦象大吉，如何建议？

```
论语："且吾闻君子之过也，如日月之食焉。
      过也，人皆见之；更也，人皆仰之。"
→ 建议：抓住机遇，但保持谦逊
→ 提醒：成功时更要谨慎
```

### 情境2：卦象大凶，如何建议？

```
论语："君子固穷，小人穷斯滥矣。"
→ 建议：困境中坚守底线
→ 提醒：不要因为困境而放弃原则
```

### 情境3：犹豫不决，如何建议？

```
论语："再，斯可以。"
→ 建议：考虑清楚就行动
→ 提醒：不要过度犹豫
```

### 情境4：遇到小人，如何建议？

```
论语："以直报怨"
→ 建议：用正直态度回应
→ 提醒：不卑不亢，不主动招惹
```

### 情境5：面对利益，如何建议？

```
论语："君子喻于义，小人喻于利"
→ 建议：在利益面前守住道义
→ 提醒：不义之财不可取
```

## 三、儒家六爻解读原则

### 3.1 修身为本

```
卦吉 → 感恩珍惜，继续修身
卦凶 → 反求诸己，提升自己
```

### 3.2 处世中庸

```
过刚 → 柔和处理
过柔 → 刚健应对
不偏不倚谓之中庸
```

### 3.3 义利取舍

```
利己损人 → 不可取
利己利人 → 可以取
舍己利人 → 圣人之举
```

### 3.4 君子小人

```
君子卦 → 光明正大
小人卦 → 谨慎防害
→ 君子问凶，小人问吉
```

## 四、解卦儒家话术模板

### 卦吉时
```
"恭喜！卦象显示机遇。
 但论语有云：'君子之交淡如水'，
 望您保持谦逊，不忘修身。"
```

### 卦凶时
```
"卦象显示有挑战。
 论语云：'君子固穷，小人穷斯滥矣'，
 建议您坚守正道，静待时机。"
```

### 犹豫时
```
"论语云：'再，斯可以'。
 建议您考虑清楚后，当断则断，
 不可过于犹豫，错失良机。"
```

### 遇小人时
```
"需防小人。
 论语云：'君子成人之美，不成人之恶'，
 与人为善，但也要有所防备。"
```
