## 第二课：认识六爻

### 2.1 什么是六爻

六爻是用三枚硬币掷6次，从下往上排列得到的卦象。

### 2.2 爻的符号

| 掷硬币结果 | 符号 | 名称 | 含义 |
|------------|------|------|------|
| 三枚正面 | O | 老阳 | 阳爻，动则变阴 |
| 三枚背面 | X | 老阴 | 阴爻，动则变阳 |
| 两正一背 | — | 少阳 | 阳爻，不变 |
| 两背一正 | -- | 少阴 | 阴爻，不变 |

### 2.3 爻位排列

```
第6次（上爻）  ←  最上面
第5次（五爻）
第4次（四爻）
第3次（三爻）
第2次（二爻）
第1次（初爻）  ←  最下面，从这里开始记录
```

### 2.4 记忆练习

```
问题：三枚硬币都是正面，是什么？
答案：老阳 O（动爻，会变成阴）

问题：两背一正是什么？
答案：少阴 --（阴爻，不变）
```

## 第三课：用神选取

### 3.1 什么是用神

用神是卦中代表所问之事的核心爻。找准用神是解卦的第一步。

### 3.2 用神选取表

| 你问的事 | 用神是 | 记忆口诀 |
|----------|--------|----------|
| 问财运、生意 | 妻财爻 | 财爻管钱 |
| 问事业、工作 | 官鬼爻 | 官鬼管工作 |
| 问考试、学业 | 官鬼爻 | 考试也是"官" |
| 问文书、合同 | 父母爻 | 文书靠父母 |
| 问疾病 | 官鬼爻 | 疾病克我者 |
| 问婚姻、感情 | 妻财爻/官鬼爻 | 男问妻，女问夫 |

### 3.3 记忆练习

```
问题：问"能不能升职"，用什么神？
答案：官鬼爻（事业、职位）

问题：问"这笔生意能不能赚钱"，用什么神？
答案：妻财爻（财运、钱）
```

## 第四课：旺衰判断

### 4.1 什么是旺衰

旺衰表示用神的强弱状态，决定吉凶基础。

### 4.2 旺衰五状态

| 状态 | 含义 | 吉凶 |
|------|------|------|
| 旺 | 当令得地 | 🟢 大吉 |
| 相 | 受月建生助 | 🟢 次吉 |
| 休 | 生我者已衰 | 🟡 平常 |
| 囚 | 被月建克制 | 🔴 平常偏凶 |
| 死 | 休极无气 | 🔴 大凶 |

### 4.3 五行旺相月令

| 月份 | 旺的五行 |
|------|----------|
| 寅、卯月 | 木 |
| 巳、午月 | 火 |
| 申、酉月 | 金 |
| 亥、子月 | 水 |
| 辰、戌、丑、未月 | 土 |

### 4.4 记忆练习

```
问题：现在是农历三月，用神是木，旺衰如何？
答案：旺（木在春月当令）

问题：用神属火，在冬天状态如何？
答案：死（火在冬月失令）
```

## 第五课：动变分析

### 5.1 什么是动变

动爻是卦中变化的爻，它的变化会影响整个卦的吉凶。

### 5.2 动变规则

| 动爻类型 | 变化 | 说明 |
|----------|------|------|
| 老阳 O | 变阴 -- | 阳极转阴 |
| 老阴 X | 变阳 — | 阴极转阳 |
| 少阳 — | 不变 | 稳定 |
| 少阴 -- | 不变 | 稳定 |

### 5.3 生克关系

| 关系 | 对用神的影响 | 吉凶 |
|------|--------------|------|
| 动爻生用神 | 帮助 | 🟢 吉 |
| 动爻克用神 | 损害 | 🔴 凶 |
| 动爻化回头克 | 被反制 | 🔴 大凶 |

### 5.4 记忆练习

```
问题：老阳 O 动后会变成什么？
答案：阴爻 --

问题：用神被动爻相生，说明什么？
答案：有人帮助，吉利
```

## 第六课：综合解卦

### 6.1 解卦四步骤

```
第一步：列卦象（排出六爻）
    ↓
第二步：找用神（根据问事选取）
    ↓
第三步：看旺衰（日月生克判断）
    ↓
第四步：看动变（动爻影响）
    ↓
综合判断吉凶
```

### 6.2 吉凶判断原则

| 情况 | 判断 |
|------|------|
| 用神旺相 + 动爻生扶 | 🟢 大吉 |
| 用神旺相 + 动爻克冲 | 🔴 先好后坏 |
| 用神休囚 + 动爻生扶 | 🟡 先难后易 |
| 用神休囚 + 动爻克冲 | 🔴 大凶 |

### 6.3 记忆练习

```
问题：用神旺相，卦中动爻生它，吉凶？
答案：大吉

问题：用神休囚，卦中动爻克它，吉凶？
答案：大凶
```
