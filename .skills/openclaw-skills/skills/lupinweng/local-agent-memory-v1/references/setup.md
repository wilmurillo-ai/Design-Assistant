# Setup and Topic Layout

## Minimum structure

```text
MEMORY.md
HEARTBEAT.md
memory/
  YYYY-MM-DD.md
  semantic/
  procedural/
```

## Recommended semantic topics

- user preferences and context
- assistant identity and style
- environment and platform facts
- memory governance
- operating constitution

## Recommended procedural topics

- local memory workflow
- dream maintenance workflow
- deep dream pass workflow

## Setup rule

Start small.
Do not try to build perfect taxonomy immediately.
Only split topics when real repeated patterns appear.
