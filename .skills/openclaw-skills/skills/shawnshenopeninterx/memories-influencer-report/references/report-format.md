# Influencer Vetting Report Format

## Template

```markdown
# Influencer Vetting Report

**Influencer**: @handle
**Platform**: Instagram/TikTok/YouTube/Twitter
**Date**: YYYY-MM-DD
**Videos Analyzed**: N

## Profile Overview
- **Followers**: XXK
- **Content Type**: Lifestyle/Tech/Beauty/etc.
- **Posting Frequency**: X/week
- **Profile URL**: https://...

## Video Analysis

### Video 1: [title/link]
| Criteria | Score | Notes |
|----------|-------|-------|
| Camera-facing | ✅/❌ | Creator visible and addressing camera |
| Audio quality | ✅/❌ | Clear speech, no background noise |
| Lighting | ✅/❌ | Well-lit, professional look |
| Content relevance | ✅/❌ | On-brand, engaging content |
| Brand safety | ✅/❌ | No controversial/offensive content |

**AI Transcript Summary**: Brief summary of what was said...
**Visual Analysis**: Description of visual elements, setting, presentation style...

### Video 2: [title/link]
(same format)

## Overall Assessment
- **Score**: XX/100
- **Recommendation**: ✅ APPROVED / ❌ REJECTED
- **Key Strengths**: ...
- **Concerns**: ...
- **Best For**: Types of brand partnerships this creator suits
```

## Scoring Guide

| Score Range | Recommendation |
|------------|----------------|
| 80-100 | ✅ APPROVED — Excellent fit |
| 60-79 | ⚠️ CONDITIONAL — Review concerns |
| 0-59 | ❌ REJECTED — Significant issues |

### Criteria Weights
- Camera-facing presence: 20%
- Audio quality: 15%
- Lighting/production: 15%
- Content relevance: 25%
- Brand safety: 25%
