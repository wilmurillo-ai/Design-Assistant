# Video to Doc - 视频转操作指南

将操作视频自动转换为图文并茂的Word操作指南文档。

## 功能特点

- 🎬 **智能截图提取** - 根据视频时长自适应调整截图间隔
- 🎤 **语音转录** - 支持 faster-whisper 本地模型或在线API
- 🤖 **LLM内容提炼** - 将口语化描述转为规范操作说明
- 📊 **流程图生成** - 自动生成横向箭头流程图
- 📝 **图文并茂** - 每个步骤配关键截图

## 适用场景

| 输入 | 输出 |
|------|------|
| 软件操作教程视频 | 操作指南文档 |
| 系统演示视频 | 用户手册 |
| 培训视频 | 培训材料 |
| 产品演示视频 | 产品说明文档 |

## 使用方式

### 1. 准备视频文件

支持常见格式：mp4, wmv, avi, mov 等

### 2. 运行技能

```bash
python video_to_doc.py your_video.mp4
```

### 3. 获取文档

生成的Word文档包含：
- 操作概述
- 流程图概览
- 详细操作步骤（含截图）
- 注意事项

## 依赖要求

### 一键安装

```bash
cd skill-video-to-doc
bash scripts/install_deps.sh
```

### 手动安装

- **ffmpeg** - 视频处理（必需）
- **python-docx** - 文档生成（必需）
- **pillow** - 图像处理（必需）
- **faster-whisper** - 语音转录（可选，首次运行自动下载模型）

## 输出示例

文档结构：
```
一、操作概述
二、操作流程图
三、详细操作步骤
四、注意事项
```
