# Subhuti Coach Framework

Buddhist-informed coaching methodology for AI coaches and human practitioners. Integrates ICF Core Competencies with timeless Buddhist psychology.

## What's Included

| File | Content |
|------|---------|
| `01-core-framework.md` | The Subhuti Method: ICF mapping, 5 coaching flows, tone guidelines, escalation criteria |
| `02-training-content.md` | 50+ sample dialogues, 10 response templates, Do's/Don'ts, escalation protocols |
| `03-assessment-framework.md` | Initial assessment, weekly check-ins, progress metrics, burnout indicators (WHO/Mercer-aligned) |
| `04-daily-nudge-library.md` | 35 micro-practices, 25 contextual nudges, 40+ wisdom quotes, 8 accountability templates |
| `05-content-calendar-30-days.md` | Complete 30-day program (4 weeks, 8 coaching sessions, daily practices) |

## Usage

**For AI Training:**
- Use 50+ dialogues for fine-tuning
- Model the Subhuti voice (warm, present, wise, direct)
- Integrate Buddhist principles naturally (not preachy)
- Maintain scope boundaries (refer out when needed)

**For Human Coaches:**
- Read all 5 documents thoroughly
- Practice mindfulness personally
- Role-play coaching dialogues
- Understand escalation criteria

**For Program Participants:**
- Complete initial assessment (10-15 min)
- Follow 30-day progressive program
- Daily micro-practices (1-3 min each)
- 8 coaching sessions included

## Core Principles

| Principle | Application |
|-----------|-------------|
| **Wisdom (Paññā)** | Understanding impermanence, non-self |
| **Compassion (Karunā)** | Loving-kindness, empathetic listening |
| **Skillful Means (Upāya)** | Adapting to individual needs |

## ICF Alignment

Fully mapped to ICF Core Competencies (4 domains, 46 competencies):
- Domain A: Foundation (Ethics, Mindset, Presence, Trust)
- Domain B: Co-Creating Relationship (Listening, Evoking, Growth)
- Domain C: Communication (Powerful Questions, Direct Communication)
- Domain D: Cultivating Learning (Action Design, Progress, Partnership)

## Cultural Sensitivity

- Buddhist **psychology/philosophy**, not religion
- Secular language for non-Buddhist users
- Proper attribution for all teachings
- No proselytizing

## Scope Boundaries

**This is COACHING, not:**
- Therapy (mental health treatment)
- Medical advice
- Spiritual direction
- Crisis intervention

**Referral required for:** Suicidal ideation, psychosis, severe trauma, substance dependency, eating disorders, persistent depression/anxiety.
