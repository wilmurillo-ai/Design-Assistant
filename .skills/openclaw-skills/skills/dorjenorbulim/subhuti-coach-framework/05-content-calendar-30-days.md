# 30-Day Content Calendar: Subhuti AI Coach

## Overview

This calendar provides a structured 30-day journey through four weeks of Buddhist-informed coaching content. Each week has a theme, daily practices, and progressive depth.

**Program Structure:**
- **Week 1:** Foundation (Mindfulness Basics, Intention-Setting)
- **Week 2:** Awareness (Recognizing Patterns, Non-Judgment)
- **Week 3:** Transformation (Working with Difficult Emotions)
- **Week 4:** Integration (Sustaining Practice, Community)

**Daily Format:**
- Morning intention/practice (2-5 min)
- Midday nudge (1-2 min)
- Evening reflection (3-5 min)
- Weekly coaching session (20-30 min, 2x/week)

---

## Week 1: Foundation — Mindfulness Basics & Intention-Setting

### Theme
Building the foundation of present-moment awareness and clarifying intentions for the journey ahead.

### Learning Objectives
By the end of Week 1, users will:
- Understand what mindfulness is (and isn't)
- Have 3-5 go-to mindfulness practices
- Have set a clear intention for their work
- Have experienced the difference between reactivity and response

