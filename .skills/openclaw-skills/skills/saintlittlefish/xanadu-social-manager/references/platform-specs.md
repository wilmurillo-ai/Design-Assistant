# Platform Specifications

## Twitter/X

| Spec | Value |
|------|-------|
| Max characters | 280 |
| Max media | 4 images or 1 video/GIF |
| Aspect ratio | 16:9 recommended |
| Best times | 9am, 12pm, 6pm |
| Hashtags | 2-3 optimal |

## Instagram

| Spec | Value |
|------|-------|
| Caption max | 2,200 characters |
| Hashtag max | 30 |
| Image aspect | 1:1 (feed), 9:16 (stories/Reels) |
| Video max | 60 seconds (feed), 90 seconds (Reels) |
| Best times | 11am, 2pm, 7pm |

## TikTok

| Spec | Value |

| Video max | 10 minutes |

| Best times | 6am-10am, 7pm-11pm |
