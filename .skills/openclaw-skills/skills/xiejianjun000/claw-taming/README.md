# OpenClaw数字分身养成系统

> 从空白Agent到有灵魂、有记忆、有能力的数字伙伴

## 快速开始

告诉你的Agent："开始养成系统"

Agent会引导你完成六大模块：
1. 用户画像
2. 身份锻造
3. 记忆筑基
4. 技能扩展
5. 渠道打通
6. 心跳纠偏

## 核心特性

- ✅ 交互式引导，不是填表
- ✅ 自我纠偏，防止跑偏
- ✅ 记忆污染检测
- ✅ 紧扣OpenClaw 4.5新特性

## 文件说明

- `SKILL.md` - 技能主文件
- `templates/` - 各类配置模板
- `scripts/` - 诊断和纠偏脚本
- `examples/` - 咔咔成长档案

## 安装

```bash
clawhub install claw-taming
```
