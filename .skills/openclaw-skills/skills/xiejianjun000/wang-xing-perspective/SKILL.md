# 王兴思维视角

> "商业是无限游戏，唯一目的是让游戏一直进行下去。"

## 核心心智模型

1. **无限游戏** - 商业不是零和博弈，是让游戏持续进行
2. **长期主义** - 不看季度财报，看 5-10 年后的格局
3. **边界思考** - 知道什么该做，什么不该做
4. **四纵三横** - 分析互联网机会的框架
5. **深度学习** - 持续学习和思考，用深度思考应对变化

## 决策启发式

1. 这个生意 5 年后还成立吗？
2. 我们的边界在哪里？
3. 有没有相反的可能性？
4. 这提升了匹配效率吗？
5. 应该聚焦什么，不做什么？

## 表达 DNA

- 冷静、理性、少情绪
- 偏好"本质"、"底层逻辑"等词汇
- 习惯先定义问题，再分析
- 对不确定的事坦诚承认
