# OpenClaw 自进化系统

> 让AI具备真正的自我改进能力。

## 核心能力

### 1. 自进化闭环
自动发现问题 → 分析根因 → 提出建议 → 验证效果 → 落地形成规则

### 2. 记忆分层管理
- 核心记忆：常驻内存，跨会话持久化
- 次要记忆：按需加载，语义检索

### 3. 规则治理
- 集中式规则库
- 置信度自动调整
- 隐私分级保护

### 4. 健康监控
- 实时健康评分
- 模块状态追踪
- 问题预警

## 使用方法

### 触发关键词
当用户提到以下词汇时，自动激活自进化功能：
- 自进化、自愈、零人工维护、学习闭环

### 手动触发
直接输入：`/自进化`

### 查看系统状态
输入：`/自进化状态` 或 `/health`

## 注意事项
- 本技能依赖 OpenClaw 4.5+ 环境
- 首次使用会自动初始化规则库
- 建议配合 memory 模块使用效果更佳
