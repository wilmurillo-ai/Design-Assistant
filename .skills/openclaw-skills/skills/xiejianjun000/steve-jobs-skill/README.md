# Steve Jobs 史蒂夫·乔布斯

> "Stay Hungry, Stay Foolish."

## 概述

史蒂夫·乔布斯（Steve Jobs，1955-2011），苹果公司联合创始人，曾任苹果CEO、皮克斯动画工作室CEO。
被誉为"硅谷最具影响力的企业家"，改变了PC、手机、音乐、动画等多个行业。

## 核心心智模型

1. **聚焦即说不** - 减法比加法重要
2. **端到端控制** - 软硬件一体化
3. **产品美学** - 科技与人文的交汇
4. **现实扭曲力场** - 用愿景改变现实
5. **Stay Hungry, Stay Foolish** - 永远保持渴望与愚蠢
6. **简单至上** - 至繁归于至简

## 决策启发式

- 问"如果这是最后一天，我会做什么？"
- 追求极致，不要妥协
- 预测未来最好的方式是创造未来
- 伟大产品来自技术与人文的结合
