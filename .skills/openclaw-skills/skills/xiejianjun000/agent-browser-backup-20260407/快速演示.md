# agent-browser 快速演示

## ✅ 安装验证

```bash
$ agent-browser --version
agent-browser 0.21.2

$ which agent-browser
/opt/jvs-claw/base/bin/agent-browser
```

## 🎯 演示场景：访问 Example.com

### 步骤 1：打开网页
```bash
agent-browser open https://example.com
```

### 步骤 2：获取页面快照
```bash
agent-browser snapshot -i
```

预期输出：
```
[ref=e1] heading "Example Domain"
[ref=e2] paragraph "This domain is for use in documentation..."
[ref=e3] link "Learn more"
```

### 步骤 3：获取页面信息
```bash
# 获取标题
agent-browser get title

# 获取 URL
agent-browser get url

# 获取文本
agent-browser get text @e1
```

### 步骤 4：截图
```bash
agent-browser screenshot example.png
```

### 步骤 5：关闭浏览器
```bash
agent-browser close
```

---

## 📋 实际演示命令（可复制执行）

```bash
# 1. 打开并快照
agent-browser open https://example.com && agent-browser snapshot -i

# 2. 获取页面标题
agent-browser get title

# 3. 获取 heading 文本
agent-browser get text @e1

# 4. 截图保存
agent-browser screenshot /tmp/example.png

# 5. 关闭浏览器
agent-browser close
```

## 🎬 更多演示场景

### 百度搜索
```bash
agent-browser open https://www.baidu.com
agent-browser snapshot -i
agent-browser fill @e1 "Kimi Agent"
agent-browser click @e2
agent-browser wait 2000
agent-browser screenshot /tmp/baidu.png
agent-browser close
```

### 获取新闻标题
```bash
agent-browser open https://news.example.com
agent-browser snapshot -i
agent-browser get count ".news-item"
agent-browser get text ".news-item:first-child .title"
agent-browser close
```
