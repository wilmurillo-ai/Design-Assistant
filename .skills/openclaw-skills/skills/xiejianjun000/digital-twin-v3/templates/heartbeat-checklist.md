# 数字双生心跳巡检配置

```markdown
# HEARTBEAT.md - 数字双生守护巡检

## 巡检清单（每30分钟）

### 1. 记忆健康检查
- [ ] 检查共生记忆是否有新增
- [ ] 检查反向记忆是否有异常
- [ ] 检查是否有记忆冲突

### 2. 守护力场状态
- [ ] 价值观锚点是否加载正常
- [ ] 最近3条回复是否有风格漂移
- [ ] 对话能量是否过高（>50轮）

### 3. 双生契约检查
- [ ] 契约文件是否存在
- [ ] 最近一次互动是否正常

### 4. 异常标记
如有异常，标记并准备报告

---

## 触发条件

当检测到以下情况时，触发守护警报：
- 连续3次风格漂移
- 检测到价值观冲突
- 对话超过50轮未休息
- 记忆文件损坏

## 输出

巡检结果写入: .twin/heartbeat-log.md
```
