## Phase 2: 框架提炼结果

### 5 个核心心智模型

| 模型 | 名称 | 核心原则 | 应用场景 |
|------|------|----------|----------|
| 1 | 赛道优先 (Market First) | 选择比努力更重要，大市场才能养出大公司 | 市场评估、机会筛选 |
| 2 | 创始人三力模型 | 创始人是公司的天花板，也是地板 | 创始人评估、团队分析 |
| 3 | 时机矩阵 (Timing Matrix) | 最好的时机是趋势刚启动但未被共识时 | 进入时机判断 |
| 4 | 单位经济模型 | 增长可以加速或减速，单位经济决定生死 | 商业模式验证 |
| 5 | 生态思维 (Ecosystem) | 投资的不只是公司，是生态位和网络效应 | 竞争格局、护城河分析 |

### 8 条决策启发式

| 编号 | 启发式 | 规则 | 例外 |
|------|--------|------|------|
| 1 | 赛道一票否决 | 市场规模<10 亿美元直接 pass | 市场快速扩张或有扩展能力 |
| 2 | 24 小时决策窗口 | 看准的项目 24 小时内给 term sheet | 需要更多尽调时间 |
| 3 | 创始人背景调查三层 | 前同事、下属、投资人各至少 2 人 | 无 |
| 4 | 单位经济测试 | LTV/CAC>3，创始人能清晰解释 | 早期验证阶段可放宽 |
| 5 | 竞争格局评估 | 领导者差距>3 倍谨慎投资 | 有颠覆性技术/模式 |
| 6 | 估值公平原则 | 追求公平估值，不是最便宜 | 明显高估时放弃 |
| 7 | 投后边界原则 | 战略参与，运营授权 | 危机时临时介入 |
| 8 | 退出时机判断 | 由公司发展阶段决定，不是基金周期 | 特殊情况协商 |

### 组织 DNA

**决策流程：**
```
项目来源 → 初步筛选 (1-2 小时) → 深入尽调 → 合伙人讨论 → 共识决策 → 快速执行 (24-48 小时)
```

**沟通风格：** 直接坦诚、数据驱动、建设性批评、尊重专业

**价值观排序：**
1. 诚信 (Integrity) - 绝对底线
2. 长期主义 (Long-term) - 核心哲学
3. 创始人成功 (Founder Success) - 成功标准
4. 卓越投资 (Investment Excellence) - 专业要求
5. 生态建设 (Ecosystem) - 差异化优势

**红杉美国 vs 红杉中国差异：**
- 决策风格：保守辩论 vs 激进快速
- 投资节奏：40-50 项目/年 vs 80-100 项目/年
- 持有周期：12-15 年 vs 8-10 年

## 使用建议

### 对谢总 (用户) 的价值

1. **投资决策参考**：用红杉框架审视自己的投资项目
2. **融资策略制定**：理解顶级 VC 的决策逻辑，提高融资成功率
3. **商业分析工具**：系统分析商业机会，不是凭直觉
4. **学习 benchmark**：与顶级 VC 对标，提升自己的投资能力

### 使用方式

1. **作为检查清单**：在决策前逐项检查，避免遗漏关键维度
2. **作为分析框架**：按顺序思考 (赛道→创始人→时机→单位经济→生态)
3. **作为学习工具**：复盘投资案例，提升判断能力
4. **作为参考基准**：与其他投资框架对比，形成自己的风格

### 重要提醒

- 这是参考框架，不是决策替代品
- 最终决策需要结合具体情况和个人判断
- 市场在变化，框架需要持续更新
- 红杉也会犯错 (如 FTX)，不要盲目崇拜
