# 高瓴资本经典投资案例

## 案例一：腾讯（2005 年）

### 投资背景
- **时间**：2005 年
- **阶段**：上市后不久
- **投资金额**：约 2000 万美元（初期）
- **背景**：腾讯刚上市，市值不高，业务以 QQ 为主

### 决策逻辑
1. **创始人评估**：马化腾产品思维强，专注用户体验
2. **商业模式**：社交网络 + 增值服务，现金流好
3. **行业趋势**：互联网在中国刚起步，空间巨大
4. **护城河**：网络效应，用户粘性强

### 投后管理
- 长期持有，不干预经营
- 在腾讯发展过程中持续支持
- 帮助对接资源

### 投资结果
- **回报**：数百倍回报
- **持有时间**：15 年+
- **意义**：高瓴成名作，验证价值投资理念

### 关键启示
- 早期识别平台型公司
- 长期持有的力量
- 不干预优秀管理层

## 案例二：京东（2010 年）

### 投资背景
- **时间**：2010 年
- **阶段**：C 轮
- **投资金额**：3 亿美元（领投）
- **背景**：京东从线下转型线上，自建物流争议大

### 决策逻辑
1. **创始人评估**：刘强东执行力强，有远见
2. **商业模式**：B2C 电商，自营 + 物流
3. **差异化**：自建物流是核心竞争力
4. **市场空间**：中国电商渗透率低，增长快

### 关键决策点
**张磊说服刘强东**：
- 刘强东原计划融资 7500 万美元
- 张磊说："要么不投，要投就投 3 亿"
- "物流必须自建，这是护城河"
- "钱不够可以再加码"

### 投后管理
- 帮助引入人才（CFO 等）
- 对接腾讯等资源
- 持续加注（多轮跟投）
- 在亏损时坚定支持

### 投资结果
- **回报**：数十倍回报
- **持有时间**：10 年+
- **意义**：验证了赋能式投资和长期主义

### 关键启示
- 创始人格局决定企业天花板
- 护城河需要长期投入
- 在困难时坚定支持

2. **商业模式**：本地生活服务平台
3. **竞争格局**：千团大战，看好美团的执行力
4. **市场空间**：本地生活数字化空间巨大

### 投后管理
- 在竞争最激烈时持续支持
- 帮助对接资源
- 战略建议（如并购大众点评）

### 投资结果
- **回报**：百倍以上
- **持有时间**：10 年+
- **意义**：验证了对创始人判断的准确性

### 关键启示
- 创始人学习能力比当前业务更重要
- 在竞争激烈时保持定力
- 本地生活是长坡厚雪

- **投资金额**：约 400 亿元（15% 股权）
- **背景**：格力混改，高瓴成为大股东

4. **估值**：合理，有安全边际

- 推动数字化转型
- 引入互联网思维
- 促进与腾讯等企业合作
- 协助渠道改革

### 投资结果
- **回报**：波动中（受市场影响）
- **持有时间**：长期
- **意义**：传统企业转型的尝试

### 关键启示
- 传统企业数字化是趋势
- 混改是投资机会

## 案例总结

### 共同特点

1. **创始人优秀**：都有格局、执行力、学习能力
2. **长期持有**：大多持有 5-10 年以上
3. **深度赋能**：不只是资金，还有资源和方法
4. **逆向投资**：在别人不看好的时候进入
5. **集中持仓**：核心项目投资金额大

### 决策启发式

1. **投人优先**：创始人品质是第一位
2. **长期视角**：不看短期波动
3. **深度研究**：理解生意本质
4. **敢于下注**：看准了重注
5. **持续支持**：困难时不撤退
