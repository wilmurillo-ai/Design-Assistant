# 高瓴资本投资方法论

## 核心投资哲学

### 1. 价值投资的中国实践

高瓴资本创始人张磊将价值投资理念与中国市场特点相结合，形成了独特的投资方法论：

**核心信念**：
- "投资就是投人"——创始人和团队的品质是首要考量
- "做时间的朋友"——长期主义，陪伴企业成长
- "守正用奇"——坚守正道，出奇制胜
- "弱水三千，只取一瓢"——专注能力圈，深度研究

### 2. 第一性原理思维

张磊强调从商业本质出发思考：
- 回归基本面，分析行业的底层逻辑
- 不追逐短期风口，关注长期价值创造
- 用常识判断，避免复杂模型的过度依赖

### 3. 深度研究驱动

**研究方法论**：
- 全产业链研究：从上游到下游，理解整个生态
- 全球化视角：对标国际最佳实践
- 跨学科思维：融合经济学、心理学、科技趋势
- 实地调研：深入一线，与创始人、员工、客户对话

### 4. 长期主义

**时间维度**：
- 投资周期：5-10 年甚至更长
- 不追求短期退出，关注企业长期价值
- "做企业的超长期合伙人"

### 5. 赋能式投资

高瓴不只是财务投资者，更是价值创造者：
- 数字化赋能：帮助传统企业数字化转型
- 战略赋能：引入行业资源，协助战略规划
- 运营赋能：分享最佳实践，提升运营效率
- 人才赋能：协助招募关键人才

### 6. 生态化布局

**投资逻辑**：
- 构建产业生态，而非孤立投资
- 寻找协同效应，促进被投企业合作
- 全产业链布局，把握产业演进趋势

### 7. 风险控制

**风控原则**：
- 先赔率后概率：先看下行风险，再看上行空间
- 安全边际：价格低于内在价值时投资
- 分散与集中的平衡：核心持仓集中，整体适度分散
- 持续跟踪：投后持续监控，及时应对变化

## 关键研究框架

### 行业分析框架

1. **市场规模与增长**：TAM/SAM/SOM 分析
2. **竞争格局**：波特五力 + 生态位分析
3. **商业模式**：盈利逻辑、现金流特征
4. **进入壁垒**：技术、品牌、网络效应、规模
5. **监管环境**：政策风险与机遇

### 公司分析框架

1. **创始人特质**：格局、执行力、学习能力
2. **团队质量**：核心团队的互补性与稳定性
3. **产品/服务**：差异化、用户价值、复购率
4. **财务健康**：现金流、ROE、资本效率
5. **成长动力**：短期、中期、长期增长引擎

### 估值框架

1. **DCF 模型**：基于长期现金流的内在价值
2. **可比公司**：行业对标，相对估值
3. **先赔率后概率**：风险收益比优先
4. **情景分析**：乐观、中性、悲观三种情景

## 投资决策流程

```
深度研究 → 初步判断 → 尽职调查 → 投资决策 → 投后赋能 → 持续跟踪
```

**关键决策点**：
- 是否理解这个生意的本质？
- 创始人是否值得长期信任？
- 是否有足够的护城河？
- 价格是否提供了安全边际？
- 高瓴能否提供独特价值？
