# 高瓴资本投资框架 Skill

## 项目概述

基于女娲蒸馏方法论，对高瓴资本的投资框架进行深度调研和提炼，形成可复用的投资决策思维框架 Skill。

## 为什么选择高瓴资本

- **中国顶级 VC/PE**：管理规模超过 1000 亿美元
- **成熟投资方法论**：价值投资的中国实践
- **与红杉差异化**：价值投资 vs 成长投资
- **公开信息丰富**：张磊《价值》、投资案例、研究文章
- **对谢总投资决策有直接参考价值**

## 核心产出

### 5 个心智模型

| 模型 | 核心思想 | 应用场景 |
|------|---------|---------|
| **第一性原理思维** | 从商业本质出发，回归基本面 | 分析生意本质、判断长期趋势 |
| **创始人优先模型** | 投资就是投人，三维度评估 | 创始人评估、团队判断 |
| **护城河动态观** | 唯一护城河是持续创造价值 | 竞争优势分析、可持续性判断 |
| **时间复利思维** | 做时间的朋友，长期持有 | 投资决策、持有周期判断 |
| **生态协同思维** | 构建产业生态，正和游戏 | 生态布局、资源协同 |

### 8 条决策启发式

| 启发式 | 含义 | 应用 |
|--------|------|------|
| **先赔率后概率** | 先看下行风险，再看上行空间 | 风险收益评估 |
| **不懂不投** | 只投资真正理解的生意 | 能力圈边界判断 |
| **选择比努力更重要** | 选择好的赛道 | 行业选择 |
| **弱水三千，只取一瓢** | 专注能力圈，看准了重注 | 持仓集中度 |
| **守正用奇** | 坚守正道，出奇制胜 | 原则与灵活平衡 |
| **做企业的超长期合伙人** | 长期陪伴，深度赋能 | 投后管理 |
| **数字化是生存题** | 传统企业必须数字化 | 转型判断 |
| **疯狂的理想主义** | 创始人需要有改变世界的理想 | 创始人评估 |

## 如何使用

### 激活方式

当提到以下内容时自动激活：
- 「用高瓴的视角」「高瓴会怎么看」「张磊会怎么想」
- 「hillhouse perspective」「高瓴模式」「张磊模式」
- 「价值投资」「长期主义」「做时间的朋友」
- 「这个创始人怎么样」「护城河在哪里」「先赔率后概率」
- 「赋能式投资」「深度研究」「不懂不投」

### 使用场景

✅ **适合使用**：
- 评估长期投资价值
- 分析创始人品质
- 判断护城河可持续性
- 决定是否长期持有
- 传统企业数字化转型
- 一级市场投资决策
- 审视已有投资决策

❌ **不适合**：
- 短期交易
- 投机性投资
- 看不懂的行业
- 没有安全边际的高估值

## 与红杉框架的差异

| 维度 | 高瓴 | 红杉 |
|------|------|------|
| **投资哲学** | 价值投资（巴菲特式） | 成长投资（VC 式） |
| **持有周期** | 5-10 年+ | 3-7 年 |
| **决策风格** | 深度研究、集中决策 | 快速决策、分散组合 |
| **投后管理** | 深度赋能 | 适度参与 |
| **估值方法** | DCF、安全边际 | 市场法、成长倍数 |

详细内容见 `redwood-comparison.md`

## 验证结果

基于公开信息和投资案例，高瓴框架的有效性得到验证：
- 创始人评估：成功案例的创始人都有格局、执行力、学习能力
- 长期主义：长期持有的项目回报更高
- 赋能式投资：深度赋能的项目表现更好
- 深度研究：研究深入的项目决策更准确
