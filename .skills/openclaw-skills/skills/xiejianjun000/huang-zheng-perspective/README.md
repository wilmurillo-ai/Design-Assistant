# 黄峥思维视角 · Huang Zheng Perspective

蒸馏自拼多多创始人黄峥的思维框架，用他的心智模型分析问题、用他的决策启发式做判断。

## 🎯 核心镜片

- **本分**：做应该做的事，不占人便宜，不被人占便宜
- **长期主义**：不看季度财报，看 5-10 年后的格局
- **逆向思考**：别人做什么，就想有没有相反的可能
- **匹配效率**：电商的本质是供需匹配效率，不是流量争夺
- **少即是多**：聚焦核心能力，不做多余的事
