# 认知 Skill 工厂 v2.0

> 用顶级思维做决策

## 🎯 产品介绍

认知 Skill 工厂包含**17 个顶级认知 Skill**，涵盖中国企业家、投资人和国际大师的思维框架。

每个 Skill 都包含：
- 7-10 个核心心智模型
- 8-10 条决策启发式
- 完整的表达 DNA
- 诚实边界说明

## 💡 使用示例

### 用张一鸣思维分析创业方向

```
用户：用张一鸣的视角分析，现在该不该 All in AI 创业？

张一鸣思维视角：
这个问题，我从几个维度看。

先说结论：可以 All in，但有前提条件。

第一性原理分析：
回归本质，AI 创业的核心痛点是什么？
不是缺技术，是缺应用场景和商业化路径。

算法思维：
这个业务的输入 - 处理 - 输出效率如何？
如果效率提升<10 倍，不值得做。

延迟满足：
短期利益 vs 长期价值，哪个更重要？
AI 基础设施是长期价值，应用层是短期利益。

全球化思维：
市场边界在哪里？
只做中国市场天花板太低，必须从第一天就设计全球能力。

综合判断：
如果满足以下条件，可以 All in：
1. 效率提升>10 倍
2. 有明确商业化路径
3. 从第一天就设计全球能力
4. 团队有 AI 基础设施经验

否则，建议先小范围验证。
```

### 用红杉资本框架评估项目

```
用户：用红杉的视角评估这个 AI 项目

红杉资本投资框架：
我从 5 个维度分析。

1. 赛道优先：
   - 市场规模：>1000 亿
   - 增长率：>50%/年
   - 竞争格局：分散，有机会

2. 创始人三力：
   - 洞察力：⭐⭐⭐⭐⭐
   - 执行力：⭐⭐⭐⭐
   - 领导力：⭐⭐⭐⭐

3. 时机矩阵：
   - 技术成熟度：成熟
   - 市场接受度：早期
   - 竞争强度：中等

4. 单位经济：
   - LTV/CAC：>3 ✅
   - 毛利率：>70% ✅
   - 复购率：>80% ✅

5. 生态思维：
   - 网络效应：有
   - 规模效应：有
   - 品牌效应：早期

综合评分：85/100
建议：可以投资，但需要验证市场接受度。
```
