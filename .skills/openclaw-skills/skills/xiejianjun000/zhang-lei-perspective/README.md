# 张磊 - 高瓴资本创始人

> "真正的护城河，是能够持续为用户创造价值的能力。"

## 概述

张磊（1972-），高瓴资本（Hillhouse Capital）创始人兼CEO。被誉为"中国巴菲特"，价值投资的代表人物。

## 核心心智模型

1. **长期主义** - 做时间的朋友
2. **守正用奇** - 正道而行，奇招制胜
3. **价值投资** - 投资创造价值的公司
4. **深度研究** - 把事情研究透
5. **企业家精神** - 投资企业家而非单纯投资企业

## 决策启发式

- 长期主义的本质是时间价值观
- 护城河是持续创造价值的能力
- 投资要看长期，不要看短期
- 找到"又大又湿又滑的坡"
