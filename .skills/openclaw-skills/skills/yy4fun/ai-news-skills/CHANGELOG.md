# Changelog

## v1.3.0 (2026-03-22)

### 新功能
- **深入分析框架**：新增 `analysis_framework.md`，系统化深度解读方法（玩家拆解 → 竞争本质 → 终局判断 → 多视角扫描 → 自我追问）
- **框架自我迭代**：群聊反馈自动记录到反馈日志，每次周报生成后触发自我复盘，框架持续进化，无需人工介入
- **图片采集**：深度解读自动从原文提取数据图表、架构图、产品截图（每篇最多 2 张，总计最多 3 张）
- **观察链集成**：深度解读结束后自动检查 `watch_chains.md`，将分析结论转化为跟踪项

### 修复
- **飞书 wiki 发布**：修复 `space_id is not int` 错误，新增 `wiki_target.json` 配置文件，明确 space_id（整数）和 parent_node_token 的正确传递方式
- **关注清单写入**：修复 wiki 内嵌多维表格的权限问题（91403），明确需先通过 `get_node` 获取 app_token
- **cron 精简**：cron 任务消息不再重复 skill 内部逻辑，只传参数和防护栏，避免 cron 指令与 skill 流程不一致
