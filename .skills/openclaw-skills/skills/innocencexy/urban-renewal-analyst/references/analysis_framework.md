# 城市更新改造现场分析框架

## 一、六维分析体系

### 维度 1：整体环境（Weight: 20%）

#### 核心指标

| 指标 | 说明 | 评分影响 |
|------|------|---------|
| 空间格局 | 开敞/压抑/适宜，D/H 比值判断 | 高 |
| 动线可达性 | 主次入口清晰度，步行连续性 | 高 |
| 街道界面连续性 | 沿街建筑面完整比例 | 中 |
| 视线廊道 | 有无遮挡，对景质量 | 中 |
| 微环境质量 | 光照、通风、噪声推断 | 中 |
| 区位关系 | 与城市公共空间、绿地、交通的衔接 | 低 |

#### 评分细则（1-5分）

- **5分**：D/H比1.5-3，界面连续率≥80%，动线清晰，视野开阔，微环境良好
- **4分**：整体良好，1-2项指标轻微不足，不影响基本使用
- **3分**：空间格局或动线存在明显问题，影响体验，需局部调整
- **2分**：多项指标欠佳，空间压抑或界面破碎，需系统性改造
- **1分**：环境恶劣，动线混乱，无法满足基本使用需求

---

### 维度 2：结构 / 建筑形态（Weight: 20%）

#### 核心指标

| 指标 | 说明 | 评分影响 |
|------|------|---------|
| 结构体系 | 框架/砖混/钢结构，体系合理性 | 高 |
| 结构病害 | 裂缝、沉降、倾斜、构件缺失 | 最高 |
| 比例关系 | 层高/开间/立面比例协调性 | 中 |
| 界面完整性 | 违章搭建、拆改痕迹 | 中 |
| 消防疏散 | 通道宽度、安全出口 | 高 |

#### 结构病害判断标准

| 病害类型 | 轻度 | 中度 | 重度 |
|---------|------|------|------|
| 裂缝 | <0.3mm，表面裂缝 | 0.3-1mm，斜裂缝 | >1mm，贯穿裂缝 |
| 外墙饰面 | 局部空鼓 | 大面积剥落 | 基层酥碱 |
| 结构构件 | 表面碳化 | 钢筋外露 | 截面损失 |

---

### 维度 3：外立面（Weight: 20%）

#### 主要材质老化特征

| 材质类型 | 常见病害 | 改造建议 |
|---------|---------|---------|
| 普通涂料 | 褪色、起皮、霉斑 | 重新涂刷弹性涂料或质感涂料 |
| 陶瓷面砖 | 空鼓、脱落、污染 | 修补/翻贴/干挂新材料 |
| 天然石材 | 污染、锈斑、风化 | 清洗、防水处理、局部更换 |
| 金属板 | 锈蚀、变形、焊缝开裂 | 防腐处理、局部更换 |
| 混凝土裸露 | 碳化、钢筋锈胀 | 防碳化涂层、结构修复 |

#### 色彩分析参照

- **和谐原则**：主色+辅色+点缀色控制在 3 色以内
- **地域适应**：参考当地历史建筑色谱（如江南白墙黛瓦、闽南红砖）
- **退让原则**：改造后立面色彩应比周边环境略退一步，不抢眼

---

### 维度 4：硬质装修（Weight: 15%）

#### 常见问题清单

- [ ] 人行铺装破损率（>20%即为"存在明显破损"）
- [ ] 无障碍坡道缺失（高差>2cm需设坡道）
- [ ] 盲道中断（连续性中断即为缺陷）
- [ ] 雨水口堵塞或缺失
- [ ] 护栏高度不足（<0.9m为安全隐患）
- [ ] 停车位划线混乱
- [ ] 垃圾收集点位置不合理

#### 铺装材质推荐（城市更新改造）

| 场景 | 推荐材质 | 说明 |
|------|---------|------|
| 商业街道 | 花岗岩、透水砖 | 耐磨、易清洁 |
| 居住小区 | 透水混凝土、彩色沥青 | 经济实用 |
| 历史街区 | 青砖、石板 | 延续历史肌理 |
| 公共广场 | 花岗岩、石材 | 大尺度、耐久 |

---

### 维度 5：软装 / 配景家具（Weight: 10%）

#### 评估维度

**标识系统（3项检查）**：
1. 导向牌设置间距（建议 50m 以内一处）
2. 文字规范性（中英文、字号）
3. 色彩与整体风格协调性

**街道家具（4项检查）**：
1. 座椅密度（每 30-50m 一处为宜）
2. 自行车停车架配置
3. 垃圾分类投放设施
4. 灯具类型与高度（路灯 5-8m，景观灯 1-3m）

---

### 维度 6：绿化景观（Weight: 15%）

#### 绿化覆盖率参照标准

| 场景类型 | 建议绿化率 | 来源 |
|---------|---------|------|
| 住宅小区 | ≥30% | 《城市居住区规划设计标准》 |
| 商业街道 | ≥15%（行道树为主） | 《城市绿化条例》 |
| 公共广场 | ≥20% | 《城市公园配套建设标准》 |
| 历史街区 | 依历史格局，增补庭院绿化 | 地方保护条例 |

#### 植物配置建议层次

```
第一层：乔木（高度8m+）—— 遮阴、构建空间骨架
第二层：中层乔木/小乔木（3-8m）—— 过渡、季相变化
第三层：灌木（0.5-3m）—— 界定边界、装饰
第四层：地被/草坪（0-0.5m）—— 覆盖地面、生态基底
第五层：花卉/时令植物 —— 色彩点缀、节日氛围
```

## 二、改造级别判断矩阵

### 自动判定规则

```
综合得分（满分30分）= Σ（各维度评分）

维护保养级（25-30分）：
  → 仅需清洁、修补、补植，无需设计介入

微改造级（20-24分）：
  → 重点解决外立面老化、硬质破损、绿化养护问题
  → 保留全部建筑结构，局部整治

中度更新级（13-19分）：
  → 结构检测（需委托专业机构），骨架通常可保留
  → 外立面翻新、景观重设、设施系统更新
  → 需编制更新方案

全面更新级（7-12分）：
  → 结构安全评估，部分可能需加固
  → 全面翻新，系统性重新设计
  → 需编制详细城市设计方案

拆除重建级（6分以下）：
  → 场地价值需重新评估
  → 建议委托专业机构开展可行性研究
```

## 三、改造优先级矩阵

| 问题类型 | 优先级 | 理由 |
|---------|-------|------|
| 结构安全隐患 | P0 紧急 | 直接影响人身安全 |
| 消防疏散障碍 | P0 紧急 | 法律强制要求 |
| 无障碍设施缺失 | P1 高 | 法规要求，影响特殊群体 |
| 外立面严重老化/脱落 | P1 高 | 安全隐患+形象问题 |
| 铺装严重破损 | P1 高 | 安全隐患 |
| 绿化严重缺失 | P2 中 | 影响微气候和品质 |
| 设施老旧/缺失 | P2 中 | 影响使用体验 |
| 色彩不协调 | P3 低 | 美观问题，可统筹推进 |
| 标识系统混乱 | P3 低 | 使用便利性问题 |
