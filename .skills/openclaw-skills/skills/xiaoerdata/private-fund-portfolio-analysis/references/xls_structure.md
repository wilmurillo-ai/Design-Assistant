# 估值表 XLS 结构详解

## 文件来源

私募基金估值表（托管估值表），格式为 `.xls`（Excel 97-2003），通过 xlrd 库读取。

---

## 工作表结构

| 索引 | 内容 |
|------|------|
| sheet 0 | 估值表主数据（持仓、账户、汇总） |
| sheet 1+ | 佣金计算、分账明细等（通常不需要） |

---

## 关键列（0-indexed，共14列 A-N）

| 列索引 | 列名（示例） | 说明 |
|--------|------------|------|
| 0 | 科目代码 | 账户标识，如 600036、3102.03.01.IC2606 |
| 1 | 科目名称 | 股票名称或合约名称 |
| 2 | 币种 | CNY |
| 3 | 数量 | 股份数或份额数 |
| 4 | 结算价/单位 | 股票=股份数；期货=手数 |
| 5 | 昨日收盘/单位净值 | 昨日结算价（期货）或昨收（股票） |
| 6 | 成本 | 累计成本（股票）或昨结算价×乘数×手数（期货） |
| 7 | 成本占比 | 成本占总成本比例 |
| 8 | 市值/单位净值 | 市值（股票）或昨结算价×乘数×手数（期货） |
| 9 | 市值占比 | 市值占总市值比例 |
| 10 | 应收/应付 | 期货=PnL 浮动盈亏 |
| 11 | 估值增值/保证金率 | 股票=估值增值；期货=保证金率（注意不是保证金！） |
| 12 | 保证金占用 | 期货保证金余额 |
| 13 | 备注 | 备注信息 |

> **注意**：列索引因估值表模板不同可能有所差异，建议在脚本中用 `print(row[:12])` 打印前几行验证列位置。

## 关键行

### 股票持仓行

**识别方式**：`科目代码`为6位数字，以6/5/0/3/8/4开头
- `col4` = 持股数量
- `col6` = 累计成本（元）
- `col8` = 当前市值（元）
- `col9` = 市值占比
- `col10` = 今日盈亏（元）
- `col11` = 估值增值

```python
if acc and acc[0].isdigit() and len(acc) == 6 and acc[0] in "6508324":
    mv = float(row[8])  # 市值在 col8
    cost = float(row[6])  # 成本在 col6
```

### 期货持仓行

**识别方式**：`科目代码`以 `3102.03.01.` 开头

以 IC2606（沪深300股指期货）为例：

```
Row 662: IC2606
  col0 = "3102.03.01.IC2606 CFX"  # 账户代码
  col1 = "IC2606"                   # 合约名称
  col2 = "CNY"                      # 币种
  col3 = 1                          # 份额
  col4 = 12                         # ★ 手数
  col5 = -8280.63                   # ★ 昨日结算价
  col6 = -19,873,500.26             # ★ 昨结算价×乘数×手数 = -8280.63×200×12
  col7 = -0.028094                  # 比率（≠保证金率，用处见下方）
  col8 = -0.623357                  # 汇总占比
  col9 = 7369.8                      # ★ 今日结算价
  col10 = -17,687,520.00            # ★ 今结算价×乘数×手数 = -7369.8×200×12
  col11 = -0.025004                  # 另一个比率
  col12 = 2,185,980.26             # 浮动盈亏
  col13 = "正常交易"
```

#### 合约乘数（标准）

| 合约 | 乘数 |
|------|------|
| IF（沪深300股指） | 300 元/点 |
| IH（上证50股指） | 300 元/点 |
| IC（中证500股指） | 200 元/点 |
| IM（中证1000股指） | 200 元/点 |

#### 关键公式

```python
# 名义本金（当前市值）
notional = abs(今日结算价) * 乘数 * 手数
# 例如 IC2606: 7369.8 × 200 × 12 = 17,687,520

# 昨结算价市值（成本）
prev_notional = abs(昨日结算价) * 乘数 * 手数
# 例如 IC2606: 8280.63 × 200 × 12 = 19,873,512

# 浮动盈亏
pnl = -(notional - prev_notional)
# 例如 IC2606: -(17,687,520 - 19,873,512) = +2,185,992 ≈ col12
```

#### 期货账户结构

```
3102.03    中金所_投机_卖方_股指         [汇总]
3102.03.01 中金所_投机_卖方_股指_成本    [多头/空头成本，行662-668]
3102.03.02 中金所_投机_卖方_股指_冲抵成本 [对冲抵消，行669-676]
           行669 = -行661, 行670 = -行662, ...
```

**如何获取总名义本金：**
- 方法1（推荐）：各合约今结算价×乘数×手数求和 = col9 × 乘数 × col4
- 方法2：|row661-668 的 col6| 之和 = 昨结算价体系的总合约价值

**4.4亿 vs 4.1亿的区别：**
- 4.4亿 = |row661-668 col6 之和| = 昨结算价体系（成本/面值）
- 4.1亿 = |row661-668 col10 之和| = 今结算价体系（市值/当前价值）
- 两者之差 ≈ 期货浮动盈亏

## Header 信息行

| 识别关键词 | 位置 | 含义 |
|-----------|------|------|
| 基金名称 | 行首列 | 基金名称 |
| NAV/单位净值 | 行名含"单位净值" | 基金单位净值 |
| 估值日期 | 行名含"估值" | 估值日期 |
| 份额 | 汇总行 | 基金总份额 |

```python
# 典型 Header 解析
for r in range(10):  # 前10行通常是表头
    row = [ws.cell_value(r, c) for c in range(ws.ncols)]
    if "单位净值" in str(row):
        hdr["nav"] = float(row[9])   # 单位净值在 col9
        hdr["date"] = row[1]         # 估值日期
    if "份额" in str(row[0]) or "总份额" in str(row[1]):
        hdr["units"] = float(row[3])  # 总份额在 col3
```

## 常见问题

**Q：XLS 列位置不固定怎么办？**
> 在解析函数开头遍历前50行打印所有非空行，通过科目名称关键词（如"IC2606"、"股票投资"）定位正确列位。

**Q：期货冲抵账户（3102.03.02）如何处理？**
> 忽略不做分析用，3102.03.01 已是完整敞口。冲抵账户用于估值表内部对冲。

**Q：XLS 是 `.xlsb` 格式（Excel二进制）？**
> 用 `pyxlsb` 库读取，或让用户提供 `.xls`/`.xlsx` 格式。
