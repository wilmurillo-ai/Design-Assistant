# Excel AI Analyzer 📊

智能 Excel 数据分析技能 - 让数据说话，3 秒生成专业分析报告！

## ✨ 功能亮点

- 🚀 **一键分析**：上传 Excel，自动完成全维度分析
- 📈 **统计分析**：均值、方差、分布、趋势，应有尽有
- 🔍 **异常检测**：自动识别离群值，避免数据陷阱
- 💡 **智能洞察**：基于数据特征给出专业建议
- 📝 **报告生成**：Markdown 格式，可直接用于汇报

## 🎯 适用场景

- 销售数据分析
- 财务报表审查
- 运营数据监控
- 用户行为分析
- 实验数据处理
- 任何需要快速理解 Excel 数据的场景

## 📦 安装

```bash
# 从 ClawHub 安装
npx clawhub@latest install excel-ai-analyzer

# 或从本地安装
npx clawhub@latest install ./skills/excel-ai-analyzer
```

## 💬 使用示例

### 基础用法
```
帮我分析这个 Excel 文件
[上传 data.xlsx]
```

### 指定分析维度
```
分析 sales.xlsx 的销售额数据，重点关注异常值
```

### 生成报告
```
分析 report.xlsx 并生成 Markdown 报告
```

## 📊 输出示例

分析报告包含：

1. **数据概览** - 行数、列数、数据类型
2. **统计分析** - 每列的详细统计指标
3. **异常检测** - 标记离群值和异常数据
4. **数据洞察** - 专业分析建议
