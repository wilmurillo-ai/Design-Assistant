## §15 AIGC Advanced Custom Element Parameters (vod_create_aigc_advanced_custom_element.py)


Based on the `CreateAigcAdvancedCustomElement` API, supports creating custom elements via video or multiple images.

### Common Parameters (`create` subcommand)

| Parameter | Type | Description |
|------|------|------|
| `--sub-app-id` | int | VOD application ID (**required**, can also be set via environment variable `TENCENTCLOUD_VOD_SUB_APP_ID`) |
| `--region` | string | Region, default `ap-guangzhou` |
| `--json` | flag | JSON format output |
| `--dry-run` | flag | Preview request parameters without calling the API |

> ⚠️ `--sub-app-id`, `--region`, `--dry-run`, `--no-wait`, `--max-wait` are only supported by the `create` subcommand; `--json` is supported in both `create` and `list` subcommands.

### create Parameters (Create Element)

#### Required Parameters

| Parameter | Type | Required | Description |
|------|------|------|------|
| `--element-name` | string | ✅ | Element name, must not exceed **20 characters**. Use a descriptive name for easy lookup later. |
| `--element-description` | string | ✅ | Element description, must not exceed **100 characters**. Provide a clear functional description. |
| `--reference-type` | enum | ✅ | Element reference type: `video_refer` (video character element) / `image_refer` (multi-image element) |

#### Optional Parameters

| Parameter | Type | Required | Description |
|------|------|------|------|
| `--element-voice-id` | string | - | Element voice ID, binds an existing voice from the voice library. Leave empty to skip voice binding. **Only supported for `video_refer` type.** |
| `--element-video-list` | JSON string | Required for video_refer | Element reference video list (JSON string). See format description below. |
| `--element-image-list` | JSON string | Required for image_refer | Element reference image list (JSON string). See format description below. |
| `--tag-list` | JSON string | - | Element tag list (JSON string). See format description below. |
| `--session-id` | string | - | Deduplication identifier (max 50 characters; duplicate within 3 days returns an error) |
| `--session-context` | string | - | Source context for passing through user request information (max 1000 characters) |
| `--tasks-priority` | int | - | Task priority (-10 to 10, higher value means higher priority, default 0) |
| `--interactive` / `-i` | flag | - | Enable interactive guided mode with step-by-step parameter prompts (**recommended for new users**) |
| `--no-wait` | flag | - | Submit task only without waiting for result (default waits automatically) |
| `--max-wait` | int | - | Maximum wait time (seconds, default 600) |

### Reference Type Description

| Reference Type | Description | Supports Voice Binding | Parameter Used |
|---------------|------|------------|-----------|
| `video_refer` | Video character element, defines appearance via reference video | ✅ Supported | `--element-video-list` |
| `image_refer` | Multi-image element, defines appearance via multiple images | ❌ Not supported | `--element-image-list` |

### Element Video List Format

Used only for `video_refer` type. Video format supports MP4/MOV only, duration 3s–8s, aspect ratio 16:9 or 9:16, 1080P, max 1 clip, size no more than 200MB.

```json
{"refer_videos":[{"video_url":"https://example.com/ref.mp4"}]}
```

| Field | Type | Description |
|------|------|------|
| `refer_videos` | array | Reference video list, max 1 clip |
| `refer_videos[].video_url` | string | Video URL (required, must not be empty) |

### Element Image List Format

Used only for `image_refer` type. Must include at least 1 frontal reference image and 1–3 reference images from other angles.

```json
{
  "frontal_image": "https://example.com/front.jpg",
  "refer_images": [
    {"image_url": "https://example.com/side.jpg"},
    {"image_url": "https://example.com/back.jpg"}
  ]
}
```

| Field | Type | Required | Description |
|------|------|------|------|
| `frontal_image` | string | ✅ | Frontal reference image URL (must include exactly 1) |
| `refer_images` | array | ✅ | Reference images from other angles or close-ups (1–3 images, must differ from the frontal image) |
| `refer_images[].image_url` | string | ✅ | Reference image URL |

### Tag List Format

```json
[{"tag_id": "o_101"}, {"tag_id": "o_102"}]
```

| Field | Type | Description |
|------|------|------|
| `tag_id` | string | Tag ID, format examples: `o_101`, `o_102` |

### Error Code Reference

| Error Type | Cause | Resolution |
|---------|------|---------|
| Missing Parameter | Missing required parameter | Check that Element Name, Element Description, and Reference Type are provided |
| Invalid Parameter Value.Element Name | Element name exceeds 20 characters | Shorten the element name |
| Invalid Parameter Value.Element Description | Element description exceeds 100 characters | Shorten the element description |
| Invalid Parameter Value.Reference Type | Reference type not supported | Use `video_refer` or `image_refer` |
| Invalid Parameter Value.Element Video List | Video format or parameters do not meet requirements | Check video format (MP4/MOV), duration (3s–8s), resolution (1080P) |
| Invalid Parameter Value.Element Image List | Image parameters do not meet requirements | Ensure frontal_image and at least 1 refer_images entry are included |
| Invalid Parameter Value.Element VoiceId | Voice ID does not exist or type not supported | Confirm the voice exists in the voice library; image_refer type does not support voice binding |
| Sub AppId required | Sub-application ID not specified | Add `--sub-app-id` parameter or set environment variable `TENCENTCLOUD_VOD_SUB_APP_ID` |
| Limit Exceeded | Rate limit exceeded | Default 20 requests/second |

## Usage Examples
## §15 AIGC Advanced Custom Element (vod_create_aigc_advanced_custom_element.py)


### Basic Usage

#### Interactive Guided Creation (Recommended)
```bash
python scripts/vod_create_aigc_advanced_custom_element.py create --interactive
```

The interactive guide will prompt you step by step for:
1. **Element name** (≤20 characters)
2. **Element description** (≤100 characters)
3. **Reference type** (video_refer / image_refer)
4. **Voice ID** (video_refer only, can be skipped)
5. **Reference video/image URL**
6. **Tag list** (can be skipped)
7. **Sub-application ID**

#### Create Multi-Image Element (image_refer)
```bash
python scripts/vod_create_aigc_advanced_custom_element.py create \
    --sub-app-id 1500046725 \
    --element-name "Product Display Element" \
    --element-description "Multi-image custom element for e-commerce product display, supports outfit changes and product image generation" \
    --reference-type image_refer \
    --element-image-list '{"frontal_image":"https://example.com/front.jpg","refer_images":[{"image_url":"https://example.com/side.jpg"},{"image_url":"https://example.com/back.jpg"}]}' \
    --tag-list '[{"tag_id":"o_101"},{"tag_id":"o_102"}]'
```

#### Create Video Character Element (video_refer, bind voice)
```bash
python scripts/vod_create_aigc_advanced_custom_element.py create \
    --sub-app-id 1500046725 \
    --element-name "Video Character A" \
    --element-description "Character element customized from reference video, bound to a dedicated voice, for short video generation" \
    --reference-type video_refer \
    --element-video-list '{"refer_videos":[{"video_url":"https://example.com/ref_video.mp4"}]}' \
    --element-voice-id "123333" \
    --tag-list '[{"tag_id":"o_201"}]'
```

#### Create Video Character Element (without voice binding)
```bash
python scripts/vod_create_aigc_advanced_custom_element.py create \
    --sub-app-id 1500046725 \
    --element-name "Character B" \
    --element-description "Visual-only character element, no voice binding" \
    --reference-type video_refer \
    --element-video-list '{"refer_videos":[{"video_url":"https://example.com/actor.mp4"}]}'
```

#### Preview Request Parameters (without executing)
```bash
python scripts/vod_create_aigc_advanced_custom_element.py create \

    --element-name "Test Element" \
```
