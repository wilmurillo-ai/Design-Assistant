### ⚠️ Common Parameter Mistakes

| Incorrect Usage | Correct Usage | Description |
|---------|---------|------|

### API Conflict Decision Guide

| Scenario | Recommended Script | Description |
|------|---------|------|
| Known FileId, query details | `vod_describe_media.py` | Describe Media Infos for exact query |
| Fuzzy search by name/tag, etc. | `vod_search_media.py` | Search Media fuzzy search |
| Known FileId, but need search-style query | `vod_search_media.py` | Search Media.File Ids parameter |
| Specify application by app name | `vod_search_media.py` | First use Describe Sub App Ids to match application |

> ⚠️ **Mandatory requirement**: At least one search condition must be specified (`--names`/`--file-ids`/`--tags`/`--categories`, etc.), otherwise the script will exit with an error.
