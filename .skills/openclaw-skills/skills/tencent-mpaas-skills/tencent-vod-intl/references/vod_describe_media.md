### ⚠️ Common Parameter Mistakes

| Incorrect Usage | Correct Usage | Description |
|---------|---------|------|
| `vod_describe_media.py 5145403721233902989` | `vod_describe_media.py --file-id 5145403721233902989` | FileId **must be passed via the `--file-id` parameter**; positional parameters are not supported |
| `vod_describe_media.py --file-id xxx` (without `--json`) | `vod_describe_media.py --file-id xxx --json` | **When the user requests JSON-formatted output, or needs to extract fields such as playback URL, subtitles, or cover image, `--json` must be added**, then use `jq` to process |
| `--filters meta` | `--filters metaData` | Filter values are case-sensitive and must be spelled in full |

> 🚨 **Mandatory Rule**: When the user says "extract playback URL", "extract subtitle URL", "extract cover URL", "JSON-formatted output", or any scenario that requires parsing response fields, **the `--json` parameter must be added**, then use `jq` to extract the desired fields. Without `--json`, the output is in human-readable format and cannot be parsed by `jq`.

## §3 Media Information Query Parameters (vod_describe_media.py)


### General Parameters

| Parameter | Type | Required | Description |
|------|------|------|------|
| `--file-id` | string list | ✅ | Media file ID (supports multiple, up to 20) |
| `--filters` | string list | - | Specifies the types of information to return (if not specified, all are returned; supports the following 14 types) |
| `--region` | string | - | Region, default `ap-guangzhou` |
| `--sub-app-id` | int | - | VOD sub-application ID (required for accounts created after 2023-12-25; can also be set via environment variable `TENCENTCLOUD_VOD_SUB_APP_ID`) |
| `--json` | flag | - | JSON-formatted output |
| `--dry-run` | flag | - | Preview parameters without calling the API |

### filters Available Values

| Filter Value | Description | Included Content |
|-----------|------|---------|
| `basicInfo` | Basic information | Name, description, category, type, creation/update time, playback URL, cover image, storage info, source info (live stream / TRTC / web recording, etc.) |
| `metaData` | Metadata | Size, container format, bitrate, resolution, duration, rotation angle, MD5, video stream info (codec, frame rate), audio stream info (sample rate) |
| `transcodeInfo` | Transcode result info | Transcode outputs per specification (URL, container, size, duration, resolution, bitrate, MD5, digital watermark, video/audio stream info) |
| `animatedGraphicsInfo` | Animated image result info | Animated image output (URL, format, size, resolution, bitrate, start/end time) |
| `imageSpriteInfo` | Image sprite info | Thumbnail width/height, count per image, total large image count, large image URLs, Web Vtt URL |
| `snapshotByTimeOffsetInfo` | Time-offset snapshot info | Snapshot outputs per template (time offset, URL, watermark template) |
| `sampleSnapshotInfo` | Sampled snapshot info | Sample type, sample interval, snapshot count, snapshot URLs, watermark template |
| `keyFrameDescInfo` | Keyframe timestamp info | Time offsets and content descriptions |
| `adaptiveDynamicStreamingInfo` | Adaptive bitrate streaming info | Playback URL, packaging format, encryption type, size, digital watermark, subtitle info, sub-stream info |
| `miniProgramReviewInfo` | Mini Program review info | Template ID, review result, playback URL, review elements, metadata |
| `subtitleInfo` | Subtitle info | Subtitle ID, name, language, format, source, URL |
| `reviewInfo` | Review info | Media review info (template ID, review suggestion, review time, violation types), cover review info |
| `mpsAiMediaInfo` | MPS AI media info | AI analysis task list (task type, template ID, output files, output text) |
| `imageUnderstandingInfo` | Image understanding info | Image understanding result list (template ID, output files) |

### §3.6 Advanced Usage

#### Batch query multiple media files and save results
```bash
python scripts/vod_describe_media.py \
    --file-id 5285485487985271487 5285485487985271488 \
    --json > media_infos.json
```

#### Query and extract playback URL
```bash
python scripts/vod_describe_media.py \
    --file-id 5285485487985271487 \
    --filters basicInfo --json | jq -r '.MediaInfoSet[0].BasicInfo.MediaUrl'
```

#### Query and extract transcode output URL list
```bash
python scripts/vod_describe_media.py \
    --file-id 5285485487985271487 \
    --filters transcodeInfo --json | jq -r '.MediaInfoSet[0].TranscodeInfo.TranscodeSet[].Url'
```

#### Query and check whether media exists
```bash
RESULT=$(python scripts/vod_describe_media.py \
    --file-id 5285485487985271487 \
    --json)
NOT_EXIST=$(echo "$RESULT" | jq -r '.Not Exist FileId Set | length')
if [ "$NOT_EXIST" -eq 0 ]; then
    echo "Media exists"
else
    echo "Media does not exist"
fi
```
