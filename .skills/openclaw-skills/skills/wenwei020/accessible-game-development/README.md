# 无障碍HTML游戏开发技能

## 技能说明
按照《无障碍HTML游戏开发规范》开发适配屏幕阅读器的HTML游戏，支持国内争渡读屏，兼顾全盲/低视力用户。

## 使用方式
```
开发无障碍游戏，按照 D:\无障碍游戏\无障碍HTML游戏开发规范.md 编写
```

## 规范文件
`D:\无障碍游戏\无障碍HTML游戏开发规范.md` - 完整开发规范

## 要点
- 全盲用户体验优先，兼顾低视力和明眼陪伴
- 标准ARIA语义，争渡/NVDA/JAWS完美兼容
- 每个可交互元素必须完整aria-label，动态更新必须同步
- 快捷键支持，常用操作一键到达
- 多种音效/素材方案供选择
