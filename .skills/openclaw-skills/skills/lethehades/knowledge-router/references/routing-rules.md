# routing rules

## rule intent
Primary: stable-rule
Secondary: improvement, method

## fact intent
Primary: daily-fact
Secondary: evidence

## method intent
Primary: method
Secondary: stable-rule

## evidence intent
Primary: evidence
Secondary: daily-fact

## improvement intent
Primary: improvement
Secondary: stable-rule, method

## ranking principles
- Prefer the most direct source type for the intent.
- Prefer durable sources over noisy sources when both can answer the same question.
- Use audit as proof, not as the first source for broad conceptual guidance.
