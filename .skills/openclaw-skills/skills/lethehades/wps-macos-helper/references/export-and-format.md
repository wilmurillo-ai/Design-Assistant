# export-and-format

## macOS + WPS export guidance
- Export to PDF only after checking page size, margins, fonts, and page breaks.
- Check at least the first page, a middle page, and the last page before final export.
- If layout is unstable, simplify styles before exporting.

## When to use each format
- Use Markdown when the content still needs heavy AI restructuring.
- Use docx when the user wants editable office output.
- Use PDF for final sharing, printing, or locked review.

## Formatting checks before export
- Title and heading hierarchy
- Page margins
- Headers and footers
- Page numbers
- Table widths and row breaks
- Image alignment and captions
- A4 / Letter consistency
