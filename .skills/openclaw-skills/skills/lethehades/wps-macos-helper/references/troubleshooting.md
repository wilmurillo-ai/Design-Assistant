# troubleshooting

## Typical issues

### Document opens with messy layout
- Check page size first.
- Check font substitution.
- Replace direct formatting with styles where practical.
- Inspect tables and floating images.

### PDF export looks wrong
- Re-check margins, page breaks, and section breaks.
- Ensure images are anchored safely.
- Simplify overly complex layout before export.

### Table breaks across pages badly
- Reduce width pressure.
- Check row break settings.
- Consider rebuilding the table after text cleanup.

### Page numbers or headers are wrong
- Check section breaks.
- Verify odd/even or first-page header settings.
- Re-apply numbering after layout stabilization.

## Escalation strategy
1. Simplify formatting.
2. Rebuild structure in Markdown or clean docx.
3. Re-open in WPS.
4. Export again only after visual confirmation.
