# target types

## skill
Expected core items:
- `SKILL.md`
- optional `references/`
- optional `scripts/`
- optional `README.md` and `CHANGELOG.md`

Prefer a minimal, self-contained surface.

## repo
Expected core properties:
- some understandable project entry or explanation
- no obvious private workspace directories mixed into the public root

## bundle
Expected core properties:
- clear purpose
- only files meant to be shared
- avoid dragging the whole working directory into the bundle
