# Professional 模式 - Phase 6：钩子包装

## 6.1 章节标题优化

### 爆款标题公式

**都市爽文**：`[身份] + [反差] + [爽点]`
示例：上门女婿是神医 / 兵王回归，发现女儿住狗窝

**玄幻修仙**：`[境界] + [突破] + [爽点]`
示例：废物逆袭成武帝 / 丹田破碎后，我觉醒混沌体

**穿越重生**：`[身份] + [逆袭] + [爽点]`
示例：重生1998，商界教父 / 穿越成赘婿，我靠系统躺赢

### 标题检查

- [ ] 包含核心爽点？
- [ ] 有身份反差/悬念？
- [ ] 不超过20字？

---

## 6.2 章节简介优化

### 简介公式

`[本章核心爽点] + [悬念钩子]`

### 示例

```
他当着全场的面，轻轻一弹。
金丹期护体灵气，碎了。
未婚妻脸色惨白——
而这，才刚刚开始。
```

### 检查

- [ ] 50字以内？
- [ ] 包含本章最大爽点？
- [ ] 留下悬念？

## 6.3 连载钩子规划

### 钩子类型

| 类型 | 效果 | 示例 |
|------|------|------|
| **信息炸弹** | 揭示惊人信息 | "爸，你还活着？" |
| **冲突升级** | 矛盾即将爆发 | "三辆奔驰拦住去路" |
| **悬念收尾** | 留下未解之谜 | "手机响了，显示：爸爸" |

### 钩子分布

```
第1章结尾：最强爽点预告
第2-3章结尾：日常钩子
第4章结尾：大钩子（名场面后）
第5章结尾：悬念钩子
循环...
```
