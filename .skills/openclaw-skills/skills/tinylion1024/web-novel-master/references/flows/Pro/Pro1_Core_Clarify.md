# Professional 模式 - Phase 1：核心明确

**模式**: Professional 专业模式（作者・完整但不繁琐）
**Phase数**: 2/8

---

## Q1: 网文类型

```
Question: 你想要创作什么类型的爆款网文？
Options:
- 【都市爽文】神医/兵王/总裁/保镖/上门女婿
- 【玄幻修仙】废材逆袭/系统觉醒/宗门崛起/炼丹
- 【穿越重生】历史穿越/末世囤货/仙帝重生/校园重生
- 【甜宠言情】霸道总裁/校园甜文/病娇/马甲掉落
- 【悬疑惊悚】盗墓探险/灵异/刑侦/直播算命
- 【脑洞文】系统流/直播流/反派流/无敌流
- ⭐自由描述
```

**记录**：`genre`、`genreSubtype`

## Q2: 金手指设定

```
Question: 主角的金手指是什么？
Options:
- 【系统流】任务系统/签到系统/合成系统
- 【重生记忆】末世重生/仙帝重生/赌石重生
- 【传承获得】上古功法/失传技能/衣钵传承
- 【异能觉醒】透视/读心/预知/召唤
- 【随身空间】储物/种植/养殖空间
- 【血脉体质】龙族血脉/神农血脉
- ⭐自由描述
```

**记录**：`goldenFingerType`、`goldenFingerName`

---

## Q3: 核心爽点

```
Question: 这本书的核心爽点是什么？（可多选，最多2个）
Options:
- 【打脸爽】退婚打脸/身份打脸/实力打脸
- 【升级爽】突破境界/越阶杀敌
- 【装逼爽】低调被逼/身份反差
- 【甜宠爽】吃醋/壁咚/护短/撒糖
- 【复仇爽】重生复仇/隐忍清算
- 【燃点爽】热血战斗/守护家人
- ⭐自由描述
```

**记录**：`coreTropes`

```
Question: 主角是？

- ⭐自由描述
```
