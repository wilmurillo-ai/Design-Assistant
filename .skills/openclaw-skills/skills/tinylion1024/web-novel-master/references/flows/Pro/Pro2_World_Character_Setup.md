## Q5: 世界观

```
Question: 故事发生在什么样的世界？
Options:
- 【都市背景】现代都市/当代中国/都市异能
- 【古代背景】架空历史/王朝/民国
- 【玄幻世界】修仙大陆/异界/魔法世界
- 【末世背景】丧尸爆发/灾难末世/废土
- 【仙界/神界】仙界/神界/诸天万界
- ⭐自由描述
```

**追问**：世界独特规则
**记录**：`worldSetting`、`worldRules`

## Q6: 情感线

```
Question: 主角有几条情感线？
Options:
- 【单线】1V1，专一
- 【多线】1V2或1V3
- 【无CP】无感情戏
```

**追问 — 女主类型**（如有）：
软萌型 / 高冷型 / 飒爽型 / 病娇型 / 马甲型 / 傻白甜型

**追问 — 情感节奏**：
快热型 / 慢热型 / 曲折型 / 甜宠型

**记录**：`romanceLines`、`femaleLeadType`、`romancePacing`

---

## Q7: 反派设计

```
Question: 反派的类型是？
Options:
- 【单一强敌】从头到尾一个主要反派
- 【阶段BOSS】每个阶段一个主要敌人
- 【内部背叛】身边人/亲人/兄弟反目
- 【势力BOSS】家族/宗门/公司对抗
- 【幕后黑手】最终BOSS隐藏很深
- ⭐自由描述
```

**追问 — 反派特点**：
仗势欺人 / 看不起主角 / 抢资源 / 背后捅刀 / 自以为是

**记录**：`antagonistType`、`antagonistTraits`

---

## Q8: 名场面规划

```
Question: 名场面密度？
Options:
- 【少量】每10章1个
- 【标准】每5章1个 ⭐
- 【密集】每3章1个
- 【轰炸】每2章1个
```

**追问 — 类型偏好**：
打脸型 / 装逼型 / 甜宠型 / 燃点型 / 情感型

**追问 — 第1章开场**：
退婚打脸 / 系统激活 / 重生归来 / 当众打脸 / 金手指觉醒

**记录**：`famousSceneDensity`、`famousSceneTypes`、`chapter1Scene`
