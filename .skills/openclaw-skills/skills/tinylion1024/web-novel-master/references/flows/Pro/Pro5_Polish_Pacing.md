## 5.1 AI味清除

### 高频词（出现即改）

| AI词汇 | 替换 |
|--------|------|
| "值得注意的是" | 删除或改写 |
| "不禁""油然而生" | 具体情绪 |
| "仿佛""宛如" | 真实比喻 |
| "彰显""诠释""赋能" | 具体动作 |

### 句式检查

- [ ] 避免连续相同句式
- [ ] 每句话不超过2个"的"
- [ ] 禁止"的的"

## 5.2 语言质量

### 白描

用动作代替形容：
```
反例：他很愤怒
正例：他握紧拳头，指节发白
```

### 留白

关键场景用沉默制造张力：
```
他跪了。
膝盖碰地的声音，很响。
没人说话。
```

### 情绪外放

网文情绪要直白：
```
反例：她心中有些难过
正例：眼眶红了，她狠狠吸了口气
```
