# Fast 模式 - Phase 4：最终校验

**模式**: Fast 快速模式（爱好者・极简）
**Phase数**: 5/5

---

## 4.1 逐章校验

对每一章检查：

1. **文件存在性** — 检查章节文件是否存在
2. **字数检查** — `python scripts/check_chapter_wordcount.py <文件>`
3. **爽点检查** — 本章是否有核心爽点？
4. **金句检查** — 是否有让读者想截图的句子？

---

## 4.2 不合格处理

| 问题 | 处理 |
|------|------|
| 字数不足2000 | 扩充内容 |
| 缺少爽点 | 增加爽点描写 |
| 缺少金句 | 强化结尾 |

**最多重试2次**
