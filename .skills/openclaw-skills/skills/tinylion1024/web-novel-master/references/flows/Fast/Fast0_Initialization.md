# Fast 模式 - Phase 0：初始化

**模式**: Fast 快速模式（爱好者・极简）
**Phase数**: 1/5

---

## 0.1 偏好加载

检查 `user-preferences.json` 是否存在：
- 存在 → 加载偏好数据
- 不存在 → 使用默认值，首次交互后自动创建

---

## 0.2 中断续写检测

扫描 `./web-novels/` 目录，查找含 `03-写作计划.json` 且 `status` 为 `"in_progress"` 的项目。

**如有未完成项目**：
- 展示：小说名称，完成进度（X/Y 章已完成）
- 提供选项：
  - `继续上次创作` → 从中断章节继续
  - `开始新作品` → 进入 Phase 1
