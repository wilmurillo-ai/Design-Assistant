## 1.3 趋势分析

### 当前趋势

```
【都市】身份反差流 → 神医/兵王/总裁复合身份
【玄幻】系统流 → 任务系统向合成/直播系统演进
【穿越】历史流 → 民国/清穿/历史架空增多
【甜宠】马甲流 → 女主多重身份揭露
```

### 读者偏好变化

| 偏好 | 趋势 |
|------|------|
| 节奏 | 越来越快，前3章必须出爽点 |
| 情感 | 甜宠为主流，虐文需控制篇幅 |
| 长度 | 中短篇（30-50章）完本率高 |
| 题材 | 融合流增多，都市+异能/穿越+系统 |
