# Industrial 模式 - Phase 3：世界规则

**模式**: Industrial 工业模式（团队・最完整・可流水线）
**Phase数**: 4/10

---

## 3.1 世界观设定

```
Question: 世界背景？
Options:
- 【都市背景】现代都市/当代中国/都市异能
- 【古代背景】架空历史/王朝/民国/古代都市
- 【玄幻世界】修仙大陆/异界/魔法世界
- 【末世背景】丧尸爆发/灾难末世/废土
- 【仙界/神界】仙界/神界/诸天万界
```

**追问** — 世界独特规则：
```
- 灵气复苏/规则改变
- 末世降临/灾难频发
- 都市异能觉醒
- 传统与现代融合
```

**记录**：`worldSetting`、`worldRules`
