# Industrial 模式 - Phase 2：核心定位

**模式**: Industrial 工业模式（团队・最完整・可流水线）
**Phase数**: 3/10

---

## 2.1 类型定位

```
Question: 选择作品赛道？
Options:
- 【都市爽文】神医/兵王/总裁/保镖/上门女婿
- 【玄幻修仙】废材逆袭/系统觉醒/宗门崛起/炼丹
- 【穿越重生】历史穿越/末世囤货/仙帝重生/校园重生
- 【甜宠言情】霸道总裁/校园甜文/病娇/马甲掉落
- 【悬疑惊悚】盗墓探险/灵异/刑侦/直播算命
- 【脑洞文】系统流/直播流/反派流/无敌流
```

**记录**：`genre`、`genreSubtype`

## 2.3 核心爽点

```
Question: 核心爽点？（最多2个）
Options:
- 【打脸爽】退婚打脸/身份打脸/实力打脸/当众碾压
- 【升级爽】突破境界/越阶杀敌/等级碾压
- 【装逼爽】低调被逼/身份反差/随手王炸
- 【甜宠爽】吃醋/壁咚/护短/马甲掉落/撒糖
- 【复仇爽】重生复仇/隐忍清算/大仇得报
- 【燃点爽】热血战斗/守护家人/绝境翻盘
```

**记录**：`coreTropes`

## 2.4 主角设定

### 数量与视角

```
Question: 主角设定？
Options:
- 男性主角（单男主）
- 女性主角（单女主）
- 双主角（男女主双线）
```

### 性格与身份

```
性格：扮猪吃虎 / 杀伐果断 / 温情守护 / 幽默风趣 / 高冷无敌
身份：废物流 / 普通人 / 隐藏身份 / 强者回归
```

**记录**：`protagonistCount`、`protagonistPersonality`、`protagonistOrigin`

## 2.5 读者定位

```
Question: 目标读者群？
Options:
- 【小白读者】喜欢直白爽文，不需要烧脑
- 【老白读者】喜欢有深度/有脑子的爽文
- 【女性读者】偏甜宠/感情线为主
- 【男性读者】偏打脸/升级/战斗
- 【大众读者】男女通吃，老少皆宜
```

**记录**：`targetAudience`
