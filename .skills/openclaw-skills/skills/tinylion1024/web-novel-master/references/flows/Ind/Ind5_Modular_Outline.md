# Industrial 模式 - Phase 5：模块化大纲

**模式**: Industrial 工业模式（团队・最完整・可流水线）
**Phase数**: 6/10

---

## 5.1 章节数量规划

```
Question: 计划创作多少章？
Options:
- 20章（短篇，约5-6万字）
- 30章（中篇，约7-10万字）⭐
- 50章（中长篇，约12-18万字）
- 100章+（长篇，20万字+）

```

## 5.4 伏笔系统

### 伏笔记录表

```markdown
## 伏笔记录表

| 伏笔ID | 章节 | 描述 | 回收章节 | 状态 |
|-------|------|------|---------|------|
| F001 | 第2章 | 提到某人身份 | 第10章 | 埋设 |
| F002 | 第3章 | 神秘物品出现 | 第15章 | 埋设 |
```

### 伏笔密度

- 每5章至少埋设1个伏笔
- 每10章至少回收1个伏笔
- 伏笔回收后标记状态为"已回收"

**记录**：`foreshadowSystem`
