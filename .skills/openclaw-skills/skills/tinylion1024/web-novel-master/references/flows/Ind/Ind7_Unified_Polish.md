# Industrial 模式 - Phase 7：统一润色

**模式**: Industrial 工业模式（团队・最完整・可流水线）
**Phase数**: 8/10

---

## 7.1 统一风格检查

### 文风标准

| 检查项 | 标准 |
|--------|------|
| 叙事人称 | 一致（第一人称/第三人称） |
| 场景描写 | 简洁，不超过段落20% |
| 对话比例 | 全文30-50% |
| 形容词使用 | 能删就删，用动词和名词 |

### 语言规范

**AI味清除清单**：

| 禁用词 | 替换方案 |
|--------|---------|
| "值得注意的是" | 删除或改写 |
| "不禁""油然而生" | 具体情绪 |
| "仿佛""宛如" | 真实比喻或删除 |
| "这一刻" | 删除或具体化 |
| "彰显""诠释""赋能" | 具体动作 |
| "实际上""本质上" | 删除 |

**句式规范**：
- 每句话不超过 2 个"的"
- 禁止连续相同句式（超过3句）
- 禁止连续形容词堆砌

---

## 7.2 逐章润色流程

对每一章执行：

### 润色步骤

1. **AI味清除** — 逐段检查禁用词、句式
2. **白描强化** — 关键场景用动作代替形容
3. **留白检查** — 高潮/转折处是否有留白
4. **情绪外放** — 网文情绪是否直白到位
5. **金句检查** — 是否有让人想截图的句子
6. **节奏检查** — 每500字是否有推进
7. **人物语气** — 对话是否符合人物性格
8. **伏笔确认** — 伏笔是否埋设/回收到位

### 润色记录

```markdown

| 检查项 | 状态 | 修改内容 |

| 情绪 | ✅ | - |

```

## 7.3 节奏调整

### 全局节奏检查

- [ ] 每阶段是否有明确的目标？
- [ ] 爽点分布是否均匀？
- [ ] 憋→爽节奏是否合理？

### 憋爽比例

```
轻松期（日常/甜蜜）：爽7 : 憋3
压迫期（被压制/困境）：憋6 : 爽4
爆发期（逆袭/打脸/高潮）：爽8 : 憋2
```

### 节奏问题修复

| 问题 | 修复方案 |
|------|---------|
| 全程太紧 | 增加小爽点过渡 |
| 全程太松 | 压缩铺垫，加快节奏 |
| 高潮太近 | 增加过渡章 |
| 高潮太远 | 提前引爆 |
