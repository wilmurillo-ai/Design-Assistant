# 模式选择器

**触发时机**：Phase 0 初始化完成后，用户选择创作模式

---

## 模式选择

```
🎯 爆款网文创作模式

请选择创作模式：

【Fast 快速模式】5 个 Phase
爱好者・极简｜快速开始｜快开写
适合：已有明确想法，想立刻动笔的爱好者

【Professional 专业模式】8 个 Phase
作者・完整但不繁琐｜质量优先
适合：认真创作，追求质量的作者

【Industrial 工业模式】10 个 Phase
团队・最完整・可流水线
适合：长篇创作，团队协作，流水线生产
```

---

## 三模式对比

| 维度 | Fast | Professional | Industrial |
|------|------|--------------|------------|
| **Phase数** | 5 | 8 | 10 |
| **市场调研** | ❌ | ❌ | ✅ |
| **问答深度** | 极简 | 适中 | 完整 |
| **大纲精细度** | 简略 | 完整 | 模块化 |
| **写作模式** | 串行 | 串行+并行 | 串行+并行+Teams |
| **润色深度** | 基础 | 完整 | 工业化QC |
| **包装发布** | 简单 | 完整 | 运营规划 |
| **适用场景** | 快速动笔 | 单人创作 | 团队协作 |

## 模式详细说明

### Fast 快速模式（5 Phase）

| Phase | 文件 | 职责 |
|-------|------|------|
| 0 | Fast/Fast0_Initialization.md | 初始化 |
| 1 | Fast/Fast1_Idea_Clarify.md | 想法明确（极简问答） |

| 4 | Fast/Fast4_Final_Validation.md | 最终校验（字数+爽点） |
