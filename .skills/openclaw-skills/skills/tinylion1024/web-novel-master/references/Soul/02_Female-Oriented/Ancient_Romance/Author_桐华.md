# 桐华 - 文风分析

## 作家简介

知名作家，代表作《步步惊心》《三生三世十里桃花》《那片星空那片海》《长相思》。以文笔优美、情感虐心、结局虐心著称。

## 核心文风特征

### 1. 文笔优美
- 语言诗意盎然
- 意境描写出色
- 情感细腻动人

### 2. 情感虐心
- 虐恋是核心
- 结局多为悲剧
- 让人意难平

### 3. 人物塑造
- 女主坚韧深情
- 男主类型多样
- 配角同样出彩

### 4. 叙事风格
- 节奏舒缓
- 注重内心描写
- 情感层层递进

### 5. 金句风格
```
「三生三世，十里桃花。」
「情深不寿，强极则辱。」
「这世间最远的距离，不是生与死，而是我在你面前，你却不知道我爱你。」
```

## 经典模板

### 穿越虐恋开篇
```
【穿越异世】来到陌生时空
【身份特殊】不受宠的嫡女/庶女
【卷入纷争】被家族/皇子利用
【虐恋情深】爱而不得/误会重重
```

### 仙侠虐恋模板
```
【命定相遇】前世今生的羁绊
【身份阻隔】仙魔两立/正邪不两立
【误解重重】虐点密集
【生离死别】悲剧结局
```

## 适用题材

- 古言虐恋
- 仙侠虐恋
- 穿越重生
- 三生三世

## 避坑提示

- 虐心要适度，太过则读者弃文
- 悲剧结局要铺垫自然
- 感情线要细腻才能动人
