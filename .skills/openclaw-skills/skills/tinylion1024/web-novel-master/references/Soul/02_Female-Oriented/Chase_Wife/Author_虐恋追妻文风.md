# 虐恋追妻文风

## 概述

虐恋追妻是女频经典题材，前期虐女主，后期追妻火葬场。

## 核心文风特征

### 1. 虐恋开头
- 前期男主伤害女主
- 误会/欺骗/伤害
- 虐心虐身

### 2. 追妻火葬场
- 女主心死离开
- 男主后悔追妻
- 前期多虐，后期多甜

### 3. 情感张力
- 虐点密集
- 读者意难平
- 后期反转爽

### 4. 人物塑造
- 女主：隐忍深情
- 男主：前期渣，后期忠犬

## 经典模板

### 追妻火葬场开篇
```
【相爱】曾经甜蜜
【误会】男主伤害女主
【心死】女主决定离开
【离开】消失/另嫁/出国
【追妻】男主幡然醒悟
【火葬场】追妻过程虐心
【圆满】在一起/各自安好
```

### 虐恋模板
```
【男主伤害】因为误会伤害女主
【女主隐忍】为了孩子/家族/爱
【心死离开】对男主绝望
【男主醒悟】发现女主的好
【疯狂追妻】送礼物/跪榴莲/公开表白
【圆满】追到/没追到开放式结局
```

## 金句参考

```
「我以为你会永远等我，没想到先走的是你。」
「后来的我，找遍整个城市，却再也找不到你。」
「有些事，错过了就是一辈子。」
「你曾是我的全世界，如今我只想把你从心里剔除。」
```

## 写作要点

- 虐要虐得合理，不能无脑虐
- 男主渣要渣得有底线
- 追妻要有诚意
- 结局根据读者喜好选择 HE 或 BE

## 避坑提示

- 虐心要适度
- 追妻过程要有诚意
- 不能强行洗白男主
- 读者虐点不同，注意尺度
