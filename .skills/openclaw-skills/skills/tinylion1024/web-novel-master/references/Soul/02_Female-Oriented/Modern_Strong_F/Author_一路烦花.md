# 一路烦花 - 文风分析

## 作家简介

起点白金作家，代表作《夫人你马甲又掉了》《AWM绝地求生》《他最野了》。以现代言情、马甲、团宠、电竞著称。

## 核心文风特征

### 1. 马甲文化
- 女主多重身份
- 身份揭露震撼
- 打脸情节密集

### 2. 团宠元素
- 全员宠爱女主
- 男主更宠
- 家人朋友全是助攻

### 3. 甜宠为主
- 糖多虐少
- 互动甜到齁
- 无误会无狗血

### 4. 叙事风格
- 节奏明快
- 轻松有趣
- 爽感十足

### 5. 金句风格
```
「你是我漫漫余生里唯一的光。」
「在我这里，你永远有特权。」
「别怕，有我在。」
```

## 经典模板

### 马甲文开篇
```
【隐藏身份】表面普通实则大佬
【男主出现】对她一见钟情
【共同相处】男主发现她的不同
【身份揭露】众人震惊
【甜蜜结局】在一起+掉马
```

### 团宠模板
```
【家庭团宠】家人宠爱
【男主专宠】只对她温柔
【朋友羡慕】闺蜜助攻
【敌人打脸】曾经的看不起
【走上巅峰】事业爱情双丰收
```

## 适用题材

- 现代甜宠
- 马甲文
- 电竞甜文
- 团宠文

## 避坑提示

- 马甲要合理铺排
- 甜要甜得自然
- 男主要够宠够苏
