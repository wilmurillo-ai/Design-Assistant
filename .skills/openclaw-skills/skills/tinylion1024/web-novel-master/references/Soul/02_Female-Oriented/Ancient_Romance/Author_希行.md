# 希行 - 文风分析

## 作家简介

起点白金作家，代表作《娇娘医经》《君九龄》《第一侯》《大帝姬》。以女主强大、腹黑、深沉著称。

## 核心文风特征

### 1. 女主塑造
- 智商情商双高
- 沉稳内敛有城府
- 不圣母不白莲花

### 2. 复仇主线
- 复仇是核心驱动力
- 步步为营的布局
- 快意恩仇的结局

### 3. 权谋元素
- 宅斗宫斗精彩
- 势力角逐激烈
- 人物关系复杂

### 4. 叙事风格
- 节奏沉稳
- 伏笔密布
- 逻辑严密

### 5. 金句风格
```
「人不犯我，我不犯人；人若犯我，百倍还之。」
「欠债还钱，天经地义。」
「我不是君子，所以不用以德报怨。」
```

## 经典模板

### 女主复仇开篇
```
【前世悲剧】被害/家族覆灭
【重生复仇】带着仇恨归来
【步步为营】暗中积蓄力量
【逐个清算】仇人一个个付出代价
【收获爱情】复仇路上遇见真爱
```

### 女主翻身模板
```
【隐忍蛰伏】初期示弱
【暗中布局】发展势力
【时机成熟】开始反击
【碾压对手】让所有人震惊
【站上巅峰】复仇成功+爱情圆满
```

## 适用题材

- 重生复仇
- 宅斗宫斗
- 女强权谋
- 古言医女

## 避坑提示

- 复仇要有理有据，不能无脑
- 女主智商要始终在线
- 感情线要水到渠成
