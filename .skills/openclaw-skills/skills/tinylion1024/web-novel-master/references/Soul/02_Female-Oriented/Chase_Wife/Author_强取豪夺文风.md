# 强取豪夺文风

## 概述

强取豪夺是女频经典题材，男主强势追求/占有，女主从抗拒到接受。

## 核心文风特征

### 1. 强取豪夺
- 男主强势霸道
- 不按常理出牌
- 占有欲强

### 2. 情感拉扯
- 男追女逃
- 欲拒还迎
- 暧昧不断

### 3. 甜虐交织
- 有甜有虐
- 玻璃渣中找糖
- 虐后更甜

### 4. 人物塑造
- 男主：霸道总裁/腹黑/占有欲强
- 女主：倔强/独立/有底线

## 经典模板

### 强取豪夺开篇
```
【相遇】命中注定的相遇
【追求】男主强势追求
【抗拒】女主不屈服
【纠缠】纠缠中产生感情
【接受】被男主打动
【圆满】在一起
```

### 经典桥段
```
【壁咚】经典壁咚场面
【醉酒】酒后吐真言/霸王硬上弓
【吃醋】男主为女主吃醋
【占有】男主强势宣布主权
【逃逃逃】女主试图逃离
【追追追】男主追回
【宠溺】在一起后的甜蜜
```

## 写作要点

- 男主要苏要帅，不能油腻
- 甜要甜得自然
- 女主要有底线有骨气
- 强取豪夺要有底线

## 避坑提示

- 男主不能太渣无底线
- 强取豪夺要适度
- 女主不能太傻白甜
- 尊重是底线
