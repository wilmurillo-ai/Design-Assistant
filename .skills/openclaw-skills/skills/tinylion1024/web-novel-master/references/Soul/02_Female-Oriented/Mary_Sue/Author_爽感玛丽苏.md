# 爽感玛丽苏

## 概述

爽感玛丽苏是女频爆款题材，女主万人迷，全方位碾压，人生赢家。

## 核心文风特征

### 1. 万人迷人设
- 所有男人都爱她
- 女人都嫉妒她
- 魅力无法挡

### 2. 全方位碾压
- 智商碾压
- 情商碾压
- 颜值碾压

### 3. 人生赢家
- 事业成功
- 爱情美满
- 友情忠诚

### 4. 打脸密集
- 绿茶打脸
- 渣男打脸
- 嫉妒者打脸

## 经典模板

### 爽感玛丽苏开篇
```
【女主降生】天选之人
【成长惊人】从小就很优秀
【男人都爱】各路男神倾心
【女人嫉妒】各种使绊子
【逐个打脸】让敌人付出代价
【登上巅峰】事业爱情双丰收
```

### 万人迷模板
```
【魅力无穷】所有人被她吸引
【男神环绕】霸道总裁/温柔竹马/高冷学长
【绿茶针对】白莲花女配
【打脸绿茶】让她原形毕露
【男主独宠】只爱她一人
【走上巅峰】成为人生赢家
```

## 写作要点

- 女主要有魅力但不能绿茶
- 打脸要爽但不能无脑
- 男主要专一
- 爽感要密度适中

## 避坑提示

- 女主不能太白莲
- 打脸要有理有据
- 避免过于YY
- 人物要有底线
