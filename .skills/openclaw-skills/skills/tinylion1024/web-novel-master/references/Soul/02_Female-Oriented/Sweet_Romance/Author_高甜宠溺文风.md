# 高甜宠溺文风

## 概述

高甜宠溺是女频最受欢迎的风格之一，全程无虐，糖分超标。

## 核心文风特征

### 1. 甜到齁
- 全程高糖
- 互动甜到爆
- 无虐或微虐

### 2. 宠溺无上限
- 男主宠妻无极限
- 各种宠妻操作
- 读者全程姨母笑

### 3. 轻松好读
- 不费脑子
- 放松心情
- 爽感直接

### 4. 人物塑造
- 男主：完美老公模板
- 女主：可爱/软糯/聪明

## 经典模板

### 高甜开篇
```
【相遇】甜蜜偶遇
【相知】暧昧升温
【相爱】甜蜜恋爱
【结婚】婚后更甜
【生子】一家三口/四口甜蜜
```

### 宠妻操作
```
【随时随地宠】任何时候都宠
【宠溺称呼】宝宝/老婆/宝贝
【物质宠溺】送房/送车/送公司
【精神宠溺】无条件的爱和支持
【细节宠溺】记住每一个小细节
```

## 金句参考

```
「我会宠你一辈子。」
「在我这里，你永远可以任性。」
「你的每一个小脾气，在我眼里都是可爱的。」
「我对你，没有底线，只有更宠。」
```

## 写作要点

- 甜要甜得自然
- 男主要苏要帅
- 互动要可爱
- 避免油腻

## 避坑提示

- 甜不能过于油腻
- 女主不能太傻白甜
- 甜文也需要有剧情
- 男主要符合人设逻辑
