# 万能女主文风

## 概述

万能女主/女强马甲文是女频爆款题材，女主全能人设，多重马甲。

## 核心文风特征

### 1. 全能人设
- 女主无所不能
- 多重身份
- 智商情商双高

### 2. 马甲密集
- 隐藏身份众多
- 掉马震撼
- 打脸爽点

### 3. 团宠元素
- 全员宠爱
- 男主更宠
- 打脸嫉妒者

### 4. 爽感爆棚
- 打脸情节密集
- 身份揭露高潮
- 全员震惊

## 经典模板

### 万能女主开篇
```
【隐藏身份】表面普通实则全能
【男主出现】一见钟情/青梅竹马
【共同经历】相处中发现她的不凡
【掉马时刻】身份被揭露
【震惊全场】所有人刮目相看
【甜蜜结局】幸福在一起
```

### 马甲掉落模板
```
【多个马甲】总裁/神医/黑客/团宠
【隐藏实力】故意示弱
【有人挑衅】看不起女主
【实力打脸】揭露真实身份
【众人震惊】原来她这么厉害
【男主骄傲】我的女人就是棒
```

## 金句参考

「你们不知道的，还多着呢。」

## 写作要点

- 马甲要层层递进
- 掉马要有仪式感
- 打脸要爽
- 男主要配得上女主

## 避坑提示

- 马甲要合理铺排
- 不能太过无脑
- 爽要爽得自然
- 女主要有人性弱点
