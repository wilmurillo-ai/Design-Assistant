# 叶非夜 - 文风分析

## 作家简介

起点白金作家，代表作《隔墙有男神》《国民老公带回家》《好想住你隔壁》《许你万丈光芒》。以现代言情、甜宠、总裁、电竞著称。

## 核心文风特征

### 1. 甜宠为王
- 糖分超标
- 互动甜到爆
- 无虐或微虐

### 2. 总裁男主
- 多金帅气
- 专一深情
- 宠妻无上限

### 3. 女主塑造
- 独立自主
- 有自己的事业
- 不依附男人

### 4. 叙事风格
- 节奏明快
- 轻松好读
- 更新稳定

### 5. 金句风格
```
「你是我的，谁都抢不走。」
「我等你，等了十几年。」
「遇见你，是我这辈子最幸运的事。」
```

## 经典模板

### 总裁甜宠开篇
```
【相遇重逢】小时候的羁绊/偶遇
【暧昧升温】相处中渐生情愫
【确认关系】甜蜜交往
【经历考验】误会/情敌/家族
【圆满结局】结婚/生子/幸福
```

### 甜宠互动模板
```
【日常撒糖】甜蜜互动
【壁咚公主抱】经典甜文桥段
【吃醋撒娇】女主视角的可爱
【宠溺无限】男主视角的宠
【双向奔赴】彼此深爱
```

## 适用题材

- 现代甜宠
- 总裁豪门
- 电竞恋爱
- 明星网红

## 避坑提示

- 甜要甜得自然不油腻
- 霸道总裁不能油腻
- 女主不能傻白甜
