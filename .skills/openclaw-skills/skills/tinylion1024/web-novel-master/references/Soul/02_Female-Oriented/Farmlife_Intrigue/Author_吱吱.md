# 吱吱 - 文风分析

## 作家简介

起点白金作家，代表作《庶女攻略》《花开锦绣》《金陵春》《慕金丝》。以穿越宅斗、种田经商、细腻描写著称。

## 核心文风特征

### 1. 宅斗精彩
- 家族内部斗争写实
- 人物心思细腻
- 手段描写高明

### 2. 种田元素
- 发家致富是主线
- 经商头脑展现
- 财富积累爽感

### 3. 女主塑造
- 聪慧隐忍
- 有心机有手段
- 独立自主

### 4. 叙事风格
- 节奏舒缓
- 注重日常
- 细腻入微

### 5. 金句风格
```
「人敬我一尺，我敬人一丈。」
「靠人不如靠己。」
「这世上，没有无缘无故的好意。」
```

## 经典模板

### 穿越种田开篇
```
【穿越异世】来到古代
【身份低微】庶女/不受宠
【独立自主】发展自己的事业
【嫁入豪门】高嫁/宅斗
【站稳脚跟】成为当家主母
```

### 经商模板
```
【发现商机】独特眼光
【小本起家】稳步发展
【做大做强】商业帝国
【宅斗辅助】内宅稳定
【幸福结局】事业爱情双收
```

## 适用题材

- 穿越种田
- 宅斗宫斗
- 经商致富
- 古言言情

## 避坑提示

- 种田节奏要稳，不能急
- 宅斗逻辑要严密
- 女主智商要始终在线
