# 治愈系甜宠

## 概述

治愈系甜宠以温暖、治愈、甜为主，适合喜欢温馨向的读者。

## 核心文风特征

### 1. 温暖治愈
- 文风温暖
- 情感细腻
- 治愈人心

### 2. 甜宠为主
- 糖多无虐
- 日常温馨
- 细水长流

### 3. 人物塑造
- 男女主都温暖
- 互相治愈
- 共同成长

### 4. 叙事风格
- 节奏舒缓
- 注重日常
- 细节动人

## 经典模板

### 治愈甜宠开篇
```
【相遇】温暖的相遇
【相处】日常甜蜜
【相知】互相了解
【相爱】自然在一起
【相伴】细水长流的幸福
```

### 治愈元素
```
【日常甜蜜】一起吃饭/散步/旅行
【互相治愈】过去的伤痛被对方治愈
【共同成长】一起变得更好
【家庭温暖】结婚后的温馨日常
【细水长流】平淡中见真情
```

## 金句参考

```
「你是我生命中的光，温暖了我所有的时光。」
「和你在一起的每一天，都是治愈。」
「我不需要轰轰烈烈，只想和你细水长流。」
「你的存在，就是我最大的治愈。」
```

## 写作要点

- 温暖是基调
- 细节要打动人
- 情感要细腻
- 甜要甜得自然不刻意

## 避坑提示

- 不能太平淡无聊
- 需要有打动人心的点
- 治愈系也需要有高潮
- 情感铺垫要到位
