# 苏闲佞 - 文风分析

## 作家简介

起点作家，代表作《重生之将门毒后》《沈姑娘的马甲掉了》《恶毒女配她母神》。以重生复仇、马甲、打脸著称。

## 核心文风特征

### 1. 重生复仇
- 女主带着仇恨重生
- 步步为营的复仇
- 快意恩仇

### 2. 马甲元素
- 多重身份隐藏
- 身份揭露打脸
- 爽感密集

### 3. 打脸虐渣
- 曾经欺负她的人付出代价
- 打脸情节密集
- 读者代入感强

### 4. 人物塑造
- 女主腹黑有手段
- 不圣母不白莲
- 智商情商双高

### 5. 金句风格
```
「欠我的，我会一笔一笔讨回来。」
「人不犯我，我不犯人；人若犯我，我必百倍还之。」
「这一世，我只为自己而活。」
```

## 经典模板

### 重生复仇开篇
```
【前世悲剧】被害/家破人亡
【重生回到过去】带着记忆
【暗中发展】积蓄力量
【逐个清算】让仇人付出代价
【收获幸福】复仇成功+真爱
```

### 打脸模板
```
【曾经羞辱】上辈子看不起她的人
【今生打脸】让她/他付出代价
【众人震惊】原来她这么厉害
【男主欣慰】我的女人真优秀
【圆满结局】复仇+幸福
```

## 适用题材

- 重生复仇
- 马甲打脸
- 古言言情
- 宫斗宅斗

## 避坑提示

- 复仇要有理有据
- 打脸要爽但不能无脑
- 感情线要细腻
