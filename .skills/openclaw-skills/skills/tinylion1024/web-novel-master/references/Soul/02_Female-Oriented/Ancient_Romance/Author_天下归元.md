# 天下归元 - 文风分析

## 作家简介

起点白金作家，代表作《扶摇》《凰权》《天盛长歌》《千金笑》。以大女主、权谋斗争、女强男强著称。

## 核心文风特征

### 1. 大女主
- 女主智商在线
- 独立自主不依赖
- 有野心有手段

### 2. 权谋深刻
- 朝堂斗争精彩
- 人物心思复杂
- 布局缜密

### 3. 女强男强
- 男女主势均力敌
- 并肩作战
- 灵魂伴侣

### 4. 叙事风格
- 格局宏大
- 节奏稳健
- 伏笔千里

### 5. 金句风格
```
「我不要嫁入豪门，我自己就是豪门。」
「这天下，我与你共分。」
「若为自由故，一切皆可抛。」
```

## 经典模板

### 女强权谋开篇
```
【女主重生】带着前世记忆
【身份改变】从废物到强者
【朝堂风云】卷入皇权争斗
【男主出现】势均力敌的对手
【并肩作战】共同应对危机
```

### 女强模板
```
【女主崛起】从弱到强
【能力展示】让所有人刮目相看
【打脸虐渣】曾经的欺辱者付出代价
【收获爱情】真爱而非工具人男主
```

## 适用题材

- 古言权谋
- 女强宠文
- 宫斗宅斗
- 女强男强

## 避坑提示

- 女主要真正强大，不能伪女强
- 权谋要经得起推敲
- 男主要配得上女主
