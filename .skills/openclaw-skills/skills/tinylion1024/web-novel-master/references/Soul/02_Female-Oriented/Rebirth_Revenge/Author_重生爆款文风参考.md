# 重生爆款文风参考

## 概述

重生文是女频爆款题材之一，核心是带着前世记忆回到过去，改写命运。

## 核心文风特征

### 1. 重生设定
- 回到悲剧发生前
- 带着前世记忆
- 知道未来走向

### 2. 复仇主线
- 改写命运
- 报复仇人
- 避免悲剧

### 3. 爽点设计
- 打脸虐渣
- 身份逆袭
- 马甲揭露

### 4. 情感线
- 与男主的重逢/纠葛
- 前世遗憾今世弥补
- 真爱救赎

## 经典模板

### 重生文开篇公式
```
【前世悲剧】被害/家破人亡/惨死
【重生回到过去】悲剧发生前
【改变轨迹】利用先知优势
【复仇清算】让仇人付出代价
【收获幸福】事业爱情双丰收
```

### 重生+马甲模板
```
【重生+隐藏身份】表面废物实则大佬
【前世被欺负】今世打脸
【男主出现】今生只爱你
【双重爽点】打脸+马甲
【圆满结局】复仇+幸福+马甲掉落
```

## 金句参考

```
「这一世，我不会再重蹈覆辙。」
「你们欠我的，我会一样一样讨回来。」
「我不是原来的我了，你们准备好了吗？」
「这一世，我要活出不一样的精彩。」
```

## 写作要点

- 重生前后对比要鲜明
- 复仇节奏要紧凑
- 感情线要细腻动人
- 爽点密度要控制好

## 避坑提示

- 重生设定要合理
- 复仇手段要有底线
- 感情线不能拖后腿
- 金手指不能开太大
