## 使用方式

1. **创作前**：根据目标赛道选择对应的文风参考
2. **人设时**：参考作者风格确定主角性格和行为模式
3. **写情节**：参考爽点设计安排打脸、升级、感情线节奏
4. **润色时**：参考金句风格强化语言感染力

## 文风选择指南

- 想写**快节奏爽文** → 参考天蚕土豆、风凌天下
- 想写**凡人流写实** → 参考忘语、耳根
- 想写**都市热销** → 参考老鹰吃小鸡、打眼
- 想写**历史权谋** → 参考猫腻、愤怒的香蕉
- 想写**女强宠文** → 参考天下归元、一路烦花
- 想写**重生复仇** → 参考希行、苏闲佞
- 想写**甜宠治愈** → 参考叶非夜、治愈系甜宠
