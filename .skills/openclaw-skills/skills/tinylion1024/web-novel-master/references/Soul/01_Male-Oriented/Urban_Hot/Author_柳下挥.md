# 柳下挥 - 文风分析

## 作家简介

起点白金作家，代表作《近身保镖》《终极教师》《猎赝大师》。以都市热血、保镖、教师、赝品鉴定著称。

## 核心文风特征

### 1. 都市热血
- 主角身手不凡
- 保镖/教师/鉴定师身份
- 热血流+专业流结合

### 2. 人物塑造
- 主角亦正亦邪
- 有原则但不死板
- 女人缘好但专情

### 3. 爽点节奏
- 打斗场面精彩
- 装逼打脸节奏快
- 暧昧情感拿捏准

### 4. 叙事风格
- 节奏明快
- 代入感强
- 配角出彩

### 5. 金句风格
```
「我不喜欢麻烦，但麻烦来了，我也不怕。」
「做人要有底线，没底线的人不配活着。」
「有些事，明知不可为而为之，是因为心里有不能放下的东西。」
```

## 经典模板

### 近身保镖流开篇
```
【高手归来】隐世高手出山
【身份隐藏】表面身份普通
【卷入纷争】保护目标卷入势力
【强势碾压】展现真实实力
```

### 暧昧情感模板
```
【命中注定】与女主相遇
【欢喜冤家】相互看不顺眼
【患难与共】共同经历危险
【日久生情】感情逐渐升温
```

## 适用题材

- 都市保镖
- 热血教师
- 古玩赝品
- 暧昧言情

## 避坑提示

- 热血与暧昧比例要平衡
- 主角人设要讨喜
- 打斗场面要精彩
