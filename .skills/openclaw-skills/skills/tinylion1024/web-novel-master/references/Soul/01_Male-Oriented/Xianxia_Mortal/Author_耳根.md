# 耳根 - 文风分析

## 作家简介

起点白金作家，代表作《仙逆》《求魔》《我欲封天》《三寸人间》。以想象力奇特、文笔细腻、虐心著称。

## 核心文风特征

### 1. 想象力奇特
- 世界观不走寻常路
- 力量体系另辟蹊径
- 开创独特的流派风格

### 2. 虐心大师
- 主角命运多舛
- 情绪波动极大
- 虐后必爽，但过程极虐

### 3. 文笔细腻
- 心理描写深入
- 意境营造出色
- 战斗描写偏写意

### 4. 人物塑造
- 主角意志坚定指数极高
- 为达目的可以牺牲一切
- 孤独中前行，少有同行者

### 5. 金句风格
```
「我若要有，天不可无；我若要无，天不可有！」
「逆天之路，由我开辟！」
「既然无用，何必存在！」
```

## 经典模板

### 仙逆流开篇
```
【悲剧开局】失去最重要的人/物
【执念驱动】复仇/找回/改变
【意志碾压】精神力量超越一切
【孤独求道】无人理解，独行
```

### 绝境翻盘模板
```
【极度压抑】被逼至绝境
【执念爆发】特殊信念觉醒
【意志压制】精神层面碾压
【行动逆转】物理层面跟进
```

## 适用题材

- 仙侠虐心
- 意志流修仙
- 复仇升级
- 独特世界观

## 避坑提示

- 虐心要控制度，太过则读者流失
- 节奏过慢是常见问题
- 女主戏份极少，不适合言情
