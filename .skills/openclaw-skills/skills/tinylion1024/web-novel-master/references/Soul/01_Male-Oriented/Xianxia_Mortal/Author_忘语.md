# 忘语 - 文风分析

## 作家简介

起点白金作家，代表作《凡人修仙传》《魔天记》《玄界之门》。开创凡人流流派，以严谨设定和低调主角著称。

## 核心文风特征

### 1. 凡人流开创者
- 主角非天选之子，无逆天气运
- 资源靠争取，等级提升步步为营
- 低开高走，靠智商和心机生存

### 2. 升级体系严谨
- 境界分明：炼气→筑基→金丹→元婴→化神→大乘→真仙
- 每个境界都有明确瓶颈和克服方法
- 丹药、法器、阵法体系完整自洽

### 3. 人物塑造
- 主角韩跑跑：谨慎、隐忍、算计
- 不主动惹事，但被逼必反击
- 红颜知己不多，感情戏淡

### 4. 战斗描写
- 智斗多于力斗
- 符箓、法器、阵法配合
- 以弱胜强靠的是计谋而非爆发

### 5. 金句风格
```
「修仙路上，每一步都是生死抉择。」
「小心驶得万年船。」
「不惹事，但绝不怕事。」
```

## 经典模板

### 凡人流开篇
```
【资质平庸】没有特殊体质/血脉
【资源匮乏】一切靠自己去挣
【心机深沉】隐忍低调，暗中发展
【稳扎稳打】不冒进，每步都算计
```

### 越级战斗模板
```
【情报先行】了解敌人弱点
【准备充分】符箓/法器/阵法齐全
【留有后手】永远有保命手段
【一击脱离】不恋战，达到目的就走
```

## 适用题材

- 传统仙侠
- 凡人流修仙
- 门派崛起
- 资源争夺型

## 避坑提示

- 节奏偏慢，不适合快节奏读者
- 感情戏弱，不适合言情向
- 主角过于谨慎可能显得憋屈
