# 老鹰吃小鸡 - 文风分析

## 作家简介

起点白金作家，代表作《万族之劫》《全球高武》《星门》。都市高武流开创者，更新速度惊人，日更两万字起步。

## 核心文风特征

### 1. 日更狂魔
- 日更量极大（2-3万字/天）
- 节奏极快，不水剧情
- 战斗描写密集

### 2. 高武设定
- 都市背景+高武世界观
- 万族林立，种族战争
- 主角崛起于微末

### 3. 爽点密集
- 战斗爽点极多
- 装逼打脸节奏快
- 升级速度极快

### 4. 人物塑造
- 主角枭雄型：有野心、有手段
- 杀伐果断，不拖泥带水
- 兄弟情谊热血

### 5. 金句风格
```
「实力至上，万族平等。」
「杀该杀之人，无愧于心。」
「我只是想活下去，有错吗？」
```

## 经典模板

### 高武都市开篇
```
【现代都市】熟悉的日常背景
【万族觉醒】异界生物降临/复苏
【实力至上】武力决定一切
【主角崛起】平民逆袭/特殊体质
```

### 战斗模板
```
【外敌入侵】万族挑战
【主角应战】越级击杀
【震慑全场】实力证明
【收获丰厚】资源/地位提升
```

## 适用题材

- 都市高武
- 万族争霸
- 异界入侵
- 热血战斗

## 避坑提示

- 日更压力大，需要存稿
- 力量体系容易崩坏
- 感情戏较弱
