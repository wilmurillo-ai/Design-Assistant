# 梦入神机 - 文风分析

## 作家简介

起点白金作家，代表作《阳神》《永生》《圣王》《星河大帝》。以想象力磅礴、战斗描写宏大、设定创新著称。

## 核心文风特征

### 1. 想象力磅礴
- 力量体系创新：神魂、肉身、气运
- 世界观宏大：诸天万界
- 战斗体系自成一派

### 2. 战斗描写
- 大场面描写精彩
- 招式名华丽有气势
- 越级战斗是标配

### 3. 爽点设计
- 装逼打脸节奏稳定
- 逆伐越级杀敌
- 势力碾压爽快

### 4. 人物塑造
- 主角腹黑有算计
- 不圣母，但有原则
- 红颜知己配置合理

### 5. 金句风格
```
「天要灭我，我便逆天！」
「武道通神，万劫不灭！」
「天地不仁，以万物为刍狗！」
```

## 经典模板

### 阳神流开篇
```
【废物开局】资质低下但心气高
【意外觉醒】获得特殊传承
【快速崛起】修炼速度惊人
【势力碾压】建立自己势力
```

### 战斗模板
```
【气势先行】先声夺人
【实力碾压】绝对力量
【收服人心】胜利后招揽
【势力扩张】滚雪球发展
```

## 适用题材

- 玄幻热血
- 势力争斗
- 武道修仙
- 诸天万界

## 避坑提示

- 设定不要过于复杂
- 势力体系要有主次
- 战斗描写要有张有弛
