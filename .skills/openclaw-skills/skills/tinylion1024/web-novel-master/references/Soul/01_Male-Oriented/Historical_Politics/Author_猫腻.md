# 猫腻 - 文风分析

## 作家简介

起点白金作家，代表作《庆余年》《间客》《将夜》《择天记》《道君》。以文笔华丽、权谋深刻、人物立体著称。

## 核心文风特征

### 1. 文笔华丽
- 语言优美凝练
- 意境营造出色
- 对话机锋处处

### 2. 权谋深刻
- 势力斗争复杂
- 人物立场多元
- 没有绝对正邪

### 3. 人物塑造
- 主角智慧深沉
- 配角同样出彩
- 人物行为动因清晰

### 4. 世界观构建
- 设定新颖独特
- 力量体系自洽
- 地域风貌各异

### 5. 金句风格
```
「腹有诗书气自华。」
「人生如棋，我为棋子，亦可为棋手。」
「有些事，不是因为看到希望才坚持，而是坚持了才看到希望。」
```

## 经典模板

### 权谋流开篇
```
【身世成谜】主角身份扑朔迷离
【暗中布局】各方势力角逐
【智慧博弈】不是打打杀杀，是人情世故
【破局崛起】在夹缝中成长
```

### 人物群像模板
```
【主角】算无遗策
【对手】势均力敌
【盟友】各有所图
【红颜】灵魂伴侣
```

## 适用题材

- 古典仙侠
- 都市修真
- 权谋斗争
- 科幻玄幻

## 避坑提示

- 文笔要求高，不适合新手
- 节奏偏慢，要有耐心
- 权谋描写需要严密逻辑
