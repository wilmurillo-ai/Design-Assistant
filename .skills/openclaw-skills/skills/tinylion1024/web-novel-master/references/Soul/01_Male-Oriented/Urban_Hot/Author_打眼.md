# 打眼 - 文风分析

## 作家简介

起点白金作家，代表作《黄金瞳》《天才相师》《神藏》《捡漏》。以都市异能、捡漏赌石、古玩鉴定著称。

## 核心文风特征

### 1. 都市异能流
- 赌石捡漏是核心元素
- 异能/鉴定眼是金手指
- 暴富之路令人向往

### 2. 专业性强
- 古玩知识丰富
- 赌石技巧专业
- 行业描写真实

### 3. 爽点设计
- 捡漏暴富爽
- 打脸虐渣爽
- 鉴别真伪爽

### 4. 人物塑造
- 主角有学识、有眼力
- 不张扬但有实力
- 做事有底线

### 5. 金句风格
```
「眼力决定一切。」
「捡漏靠的是本事，不是运气。」
「真的假不了，假的真不了。」
```

## 经典模板

### 捡漏流开篇
```
【意外获得】异能觉醒/传承
【眼力非凡】看穿真假
【捡漏暴富】低价买真品
【打脸虐渣】专家/对手打脸
```

### 行业晋升模板
```
【入行】从小圈子进入
【积累】经验和财富积累
【扬名】鉴定界崭露头角
【巅峰】成为行业标杆
```

## 适用题材

- 都市异能
- 赌石捡漏
- 古玩鉴定
- 暴富逆袭

## 避坑提示

- 专业知识要准确
- 赌石描写不能过于夸张
- 升级节奏要稳健
