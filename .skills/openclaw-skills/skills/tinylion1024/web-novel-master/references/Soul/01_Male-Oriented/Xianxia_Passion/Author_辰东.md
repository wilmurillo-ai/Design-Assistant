# 辰东 - 文风分析

## 作家简介

起点白金作家，代表作《神墓》《遮天》《完美世界》《圣墟》。以宏大世界观和想象力著称。

## 核心文风特征

### 1. 世界观构建
- **宏大叙事**：诸天万界、仙域战场、纪元轮回
- 地图层层展开：北斗星域→仙域→界海→堤坝后方
- 势力格局极大：禁区、至尊、仙王

### 2. 爽点设计
- **独狼式主角**：孤独中前行，强势碾压
- 爽点不在打脸，而在绝境翻盘
- 逆伐越级杀敌是标配

### 3. 战斗描写
- 大场面描写堪称网文天花板
- 千军万马、诸天对决、诸神黄昏
- 战斗与剧情深度绑定，不是为爽而爽

### 4. 悬念设置
- 草蛇灰线，伏笔千里
- 身世之谜是核心悬念
- 世界观层层剥开，吸引追读

### 5. 金句风格
```
「谁在称无敌，哪个敢言不败？」
「独断万古！」
「仙路尽头谁为峰，一见无始道成空！」
```

## 经典模板

### 开篇公式
```
【谜团】主角身世成谜，或记忆缺失
【世界观】宏大背景徐徐展开
【暗线】有一条伏笔贯穿始终
【钩子】第一个敌人 or 第一个谜团
```

### 逆伐模板
```
【绝境】面对远超自己的敌人
【爆发】特殊体质/血脉/记忆觉醒
【碾压】不是简单打脸，是层次的碾压
【余韵】留下更大悬念
```

## 适用题材

- 玄幻宏大叙事
- 诸天万界
- 战族/仙族设定
- 史诗感强烈

## 避坑提示

- 世界观铺太大容易崩
- 更新不稳定容易断节奏
- 伏笔太多需要完美回收
- 独狼人设不利于情感线
