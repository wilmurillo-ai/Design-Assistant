# 天蚕土豆 - 文风分析

## 作家简介

起点白金作家，代表作《斗破苍穹》《大主宰》《元尊》。玄幻热血流派标杆人物。

## 核心文风特征

### 1. 爽点节奏
- **退婚流开创者**：「莫欺少年穷」成为网文标志性金句
- 憋→爽节奏极其精准：通常3章铺垫，第4章必释放
- 打脸公式：羞辱→沉默→金句反击→碾压

### 2. 升级体系
- 等级分明：斗者→斗师→大斗师→斗灵→斗王→斗皇→斗宗→斗尊→斗圣→斗帝
- 每级都有标志性异象描写
- 突破必有天地异变，仪式感拉满

### 3. 战斗描写
- 招式名华丽：佛怒火莲、无尽剑域
- 视觉冲击强：金光万丈、雷霆万钧
- 心理描写简洁：冷静、果断、杀伐

### 4. 金句风格
```
「三十年河东，三十年河西，莫欺少年穷！」
「我若成帝，天下再无抗手！」
「既然你们想死，我便成全你们！」
```

### 5. 情感线
-女主描写偏被动，人设不如男主出彩
- 情感戏是弱点，通常点到为止
- 单女主或无女主专注升级

## 经典模板

### 开篇公式
```
【困境】废物开局，被退婚/被嘲笑
【爆发】金手指觉醒 or 意外传承
【宣言】「从今日起...」
【钩子】第一场战斗 or 第一个敌人
```

### 打脸模板
```
反派：「废物也配...」
主角：（不说话，微微一笑）
金句：「三年后，让你高攀不起」
结果：碾压，全场震惊
```

## 适用题材

- 玄幻热血
- 废材逆袭
- 宗门崛起
- 异火/丹药体系

## 避坑提示

- 情感线不要深入，容易崩
- 女主不要抢戏
- 升级节奏要稳，不能断层
- 反派要够强，打脸才爽
