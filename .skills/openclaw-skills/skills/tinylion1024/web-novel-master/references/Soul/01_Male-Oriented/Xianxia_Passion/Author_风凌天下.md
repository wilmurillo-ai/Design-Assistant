# 风凌天下 - 文风分析

## 作家简介

代表作品《凌天传说》《异世邪君》《傲世九重天》。以节奏明快、爽感密集著称。

## 核心文风特征

### 1. 快节奏
- 开篇即冲突，无废话
- 每章必有爽点或钩子
- 章节短小精悍，信息密度高

### 2. 爽点密集
- 都市流写法嫁接玄幻
- 装逼打脸节奏极快
- 多女主，后宫属性明显

### 3. 人物塑造
- 主角枭雄式人格：有手段、有心机
- 不圣母，但有底线
- 女人缘极好，女性角色出彩

### 4. 战斗描写
- 简洁凌厉，不拖泥带水
- 胜利描写快意恩仇
- 嘴炮与实力并存

### 5. 金句风格
```
「我若九天神皇，纵死也无憾！」
「敢犯强汉者，虽远必诛！」
「我的拳头，就是道理！」
```

## 经典模板

### 都市玄幻混搭
```
【都市身份】隐藏豪门/少爷/王者归来
【金手指】武力碾压 or 财富滔天
【情感】多女主，各有特色
【节奏】装逼打脸循环
```

## 适用题材

- 都市玄幻混搭
- 王朝/帝国设定
- 快节奏升级
- 多女主

## 避坑提示

- 多女主需要驾驭能力
- 都市背景不能太悬浮
- 打脸节奏要控制好密度
