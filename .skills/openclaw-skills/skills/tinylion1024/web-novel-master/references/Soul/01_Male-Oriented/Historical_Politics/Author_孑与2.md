# 孑与2 - 文风分析

## 作家简介

起点白金作家，代表作《汉乡》《明天下》《唐人的餐桌》。以历史功底扎实、人物鲜活、故事性强著称。

## 核心文风特征

### 1. 历史功底扎实
- 历史细节真实
- 时代氛围浓厚
- 典章制度熟悉

### 2. 人物塑造
- 配角群像鲜活
- 历史名人新解
- 人物行为逻辑自洽

### 3. 故事性强
- 主线清晰
- 支线丰富
- 人物命运与历史交织

### 4. 叙事风格
- 幽默与厚重并存
- 温情与热血交织
- 格局大而不空

### 5. 金句风格
```
「历史是由人写的，人心才是历史的真正走向。」
「活人不能让尿憋死，办法总比困难多。」
「家国天下，先有家，再有国。」
```

## 经典模板

### 历史穿越流开篇
```
【穿越到古代】熟悉的历史节点
【身份低微】庶子/赘婿/平民
【改变命运】利用现代知识
【融入历史】在历史洪流中崛起
```

### 群像模板
```
【主角视角】贯穿主线
【历史名人】赋予新灵魂
【小人物命运】折射大时代
【势力角逐】推动剧情发展
```

## 适用题材

- 历史穿越
- 古代种田
- 家族崛起
- 军事政治

## 避坑提示

- 历史知识要准确
- 穿越设定要合理
- 现代知识运用不能过于开挂
