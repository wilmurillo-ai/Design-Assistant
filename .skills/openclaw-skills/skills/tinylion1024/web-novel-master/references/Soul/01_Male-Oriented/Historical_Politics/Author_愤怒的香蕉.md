# 愤怒的香蕉 - 文风分析

## 作家简介

起点白金作家，代表作《赘婿》《蝉动》。以文笔细腻、节奏沉稳、权谋深刻著称。

## 核心文风特征

### 1. 文笔细腻
- 描写细致入微
- 心理刻画深刻
- 叙事节奏沉稳

### 2. 权谋深沉
- 布局精妙
- 人物心思复杂
- 政治斗争写实

### 3. 情感描写
- 感情含蓄而深沉
- 不同于一般爽文
- 更接近传统文学

### 4. 叙事风格
- 慢热但后劲大
- 不追求即时爽点
- 注重人物成长

### 5. 金句风格
```
「天下之事，匹夫有责。」
「读书人的脊梁，有时候比刀剑更硬。」
「乱世之中，能活下来的，都不是简单人。」
```

## 经典模板

### 权谋流开篇
```
【身份低下】赘婿/庶子/底层
【隐忍蛰伏】暗中积蓄力量
【时机成熟】开始展露锋芒
【步步为营】权谋布局
```

### 政治斗争模板
```
【棋局】各方势力角逐
【落子】主角布局
【收网】胜利果实
【后遗症】新的危机
```

## 适用题材

- 历史权谋
- 穿越架空
- 赘婿逆袭
- 政治斗争

## 避坑提示

- 节奏偏慢，不适合快节奏读者
- 爽点不如小白文直接
- 感情线发展较慢
