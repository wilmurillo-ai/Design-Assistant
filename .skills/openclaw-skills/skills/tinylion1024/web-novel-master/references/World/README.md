# 世界观目录

世界观设定目录 - 力量体系、势力分布、世界规则。

## 目录结构

```
World/
├── Power-Systems/               # 力量体系
│   ├── Cultivation.md           # 修仙力量体系
│   ├── Martial-Arts.md           # 武侠力量体系
│   ├── Modern-Ability.md        # 现代异能体系
│   └── Sci-Fi-Tech.md          # 科幻科技体系
├── Factions/                    # 势力设定
│   ├── Sects.md                 # 宗门势力
│   ├── Corporations.md           # 企业势力
│   └── Families.md              # 家族势力
├── Rules/                       # 世界规则
│   ├── Magic-Rules.md           # 魔法规则
│   ├── System-Rules.md         # 系统规则
│   └── Social-Rules.md         # 社会规则
└── Geography/                   # 地理设定
    ├── Continents.md           # 大陆设定
    └── Cities.md               # 城市设定
```

## 目录说明

### Power-Systems 力量体系

| 文件 | 内容 | 适用题材 |
|------|------|---------|
| Cultivation.md | 修仙境界、资源、战斗、升级节奏 | 玄幻仙侠 |
| Martial-Arts.md | 武功境界、门派、江湖恩怨 | 武侠小说 |
| Modern-Ability.md | 异能分类、等级、都市设定 | 都市异能 |
| Sci-Fi-Tech.md | 科技等级、星际势力、赛博朋克 | 科幻小说 |

### Factions 势力设定

| 文件 | 内容 | 适用题材 |
|------|------|---------|
| Sects.md | 宗门等级、组织架构、资源争夺 | 玄幻仙侠 |
| Corporations.md | 企业等级、商战、职场斗争 | 都市商战 |
| Families.md | 家族等级、嫡庶之争、豪门恩怨 | 都市豪门、古言 |

### Rules 世界规则

| 文件 | 内容 | 适用题材 |
|------|------|---------|
| Magic-Rules.md | 魔法等级、元素、施法规则 | 西幻、异世界 |
| System-Rules.md | 系统流规则、任务、商城 | 系统文、系统流 |
| Social-Rules.md | 社会阶层、货币、势力关系 | 所有题材 |

### Geography 地理设定

| 文件 | 内容 | 适用题材 |
|------|------|---------|
| Continents.md | 大陆格局、区域特色、禁地秘境 | 玄幻仙侠 |
| Cities.md | 城市分级、功能区、势力分布、打脸场景 | 都市小说 |

## 使用方式

1. **创作前**：根据目标题材选择对应的力量体系和势力设定
2. **构建世界观**：参考势力关系、货币体系、社会规则
3. **设计场景**：参考城市设定和地图设计安排剧情场景
4. **设计爽点**：参考打脸场景模板安排名场面
