# 城市设定

## 概述

城市是都市网文的核心场景，是势力交锋、身份冲突、装逼打脸的主要舞台。一个设计合理的城市体系能让读者快速代入，让爽点场景更加丰富。

---

## 一、城市等级体系

### 都市城市分级

| 等级 | 特征 | 人口规模 | 代表城市 | 势力密度 |
|------|------|---------|---------|---------|
| **一线超一线** | 国家经济命脉，顶级势力汇聚 | 3000万+ | 京城魔都 | 超高 |
| **一线** | 区域核心，豪门世家 | 1000-3000万 | 省会城市 | 高 |
| **强二线** | 经济发展快，新贵崛起 | 500-1000万 | 沿海发达 | 较高 |
| **二线** | 平衡发展，机会与风险并存 | 300-500万 | 普通省会 | 中等 |
| **三线** | 发展中城市，人情社会 | 100-300万 | 地级市 | 较低 |
| **四线** | 小城市，关系社会 | 50-100万 | 县城 | 低 |
| **农村/小镇** | 熟人社会，背景简单 | 10万以下 | 乡村 | 极低 |

### 修仙城市分级

| 等级 | 名称 | 灵气浓度 | 势力等级 | 交易规模 |
|------|------|---------|---------|---------|
| **王城** | 都城/皇都 | 极高 | 皇室+顶级宗门 | 国家级 |
| **大城** | 州城/府城 | 高 | 一流宗门 | 州级 |
| **中城** | 郡城/重镇 | 中等 | 二流势力 | 郡级 |
| **小城** | 县城/边城 | 稀薄 | 三流势力 | 县级 |
| **集镇** | 集市/坊市 | 无 | 散修聚集 | 交易点 |

## 二、典型城市设定

### 一线城市（京城/魔都）

```
【特征】
├─ 顶级豪门世家聚集
├─ 政商权力中心
├─ 机会最多但竞争最激烈
├─ 潜规则最多
└─ 主角崛起的主战场

【经典场景】
├─ 顶级宴会（被嘲笑 → 打脸）
├─ 豪门相亲（被看不起 → 揭露身份）
├─ 商战博弈（被算计 → 反杀）
└─ 权力斗争（站队 → 押注正确 → 加官进爵）

【标配元素】
├─ 会所/高端场所
├─ 豪门家族聚会
├─ 顶级拍卖行
└─ 政要名流社交圈
```

### 二线城市（省会/强市）

```
【特征】
├─ 当地豪门的地盘
├─ 关系社会明显
├─ 机会中等
└─ 主角早期成长的舞台

【经典场景】
├─ 本地龙头企业
├─ 官场关系网
├─ 家族企业争斗
└─ 拆迁/土地纠纷

【标配元素】
├─ 五星级酒店
├─ 商业中心
├─ 高尔夫球场
└─ 家族祠堂
```

### 三四线城市（县城/小镇）

```
【特征】
├─ 熟人社会，关系重要
├─ 本地家族控制一切
├─ 故事开头的地方
└─ 小打小闹，小打脸场景

【经典场景】
├─ 被本地家族欺负
├─ 父母被羞辱
├─ 女朋友被相亲
└─ 主角决定去大城市发展

【标配元素】
├─ 县衙门/镇政府
├─ 本地龙头企业（其实很小）
├─ 高中/大学同学会
└─ 老家祠堂
```

## 三、城市功能区设定

### 都市区域划分

```
【核心商务区 CBD】
├─ 顶级写字楼聚集
├─ 金融中心
├─ 白领聚集地
└─ 装逼打脸高发区

【高端住宅区】
├─ 别墅区/高端小区
├─ 富人区
└─ 豪门恩怨发生地

【老城区】
├─ 旧房子/老弄堂
├─ 背景复杂
└─ 可能有隐藏高手

【开发区/新区】
├─ 新兴企业
├─ 暴发户
└─ 城中村/待拆迁

【郊区/工业区】
├─ 工厂/仓库
├─ 灰色交易
└─ 暗杀/绑架场景
```

### 修仙城市区域

```
【坊市】
├─ 修士交易场所
├─ 丹药/法器/功法买卖
└─ 散修聚集地

【宗门驻地】
├─ 宗门在城市内的分支
├─ 招收弟子
└─ 处理俗事

【世家府邸】
├─ 大家族的宅院
├─ 占地极大
└─ 禁制阵法保护

【任务大殿】
├─ 发布任务
├─ 接受委托
└─ 修士接活的地方

【拍卖行】
├─ 顶级资源交易
├─ 捡漏机会
└─ 势力博弈
```

## 四、城市势力分布

### 一线城市势力格局

```
【顶级势力】（轻易不出现）
├─ 顶级豪门（皇室相关/开国元勋）
├─ 隐藏世家（传承千年）
└─ 神秘组织（情报/杀手/商业）

【核心势力】（日常冲突对象）
├─ 一线企业/商业巨头
├─ 政界大家族
└─ 军界实力派

【中层势力】（给主角练手）
├─ 外资企业
├─ 发展中家族
└─ 新兴势力

【底层势力】（早期打脸对象）
├─ 本地小老板
├─ 小官小吏
└─ 暴发户
```

### 势力关系图谱

```
【京城顶级圈子】
┌─────────────────────────────────────┐
│         核心权力圈（5-10个顶级家族）     │
│  （制定规则，决定国家大事）               │
└──────────────────┬──────────────────┘
                   │依附/联姻
┌──────────────────┴──────────────────┐
│          二级权力圈（20-30个一流家族）   │
│  （政/商/军各界核心）                   │
└──────────────────┬──────────────────┘
                   │合作/竞争
┌──────────────────┴──────────────────┐
│          三级圈子（100+二三流家族）        │
│  （中层管理者，本地豪强）                │
└─────────────────────────────────────┘
```

## 五、城市爽点场景

### 经典都市场景

| 场景 | 设定 | 爽点 |
|------|------|------|
| **会所聚会** | 被邀请参加高端聚会 | 身份揭露，众人震惊 |
| **豪车场景** | 停车场被嘲讽保安/司机 | 实际是车主/大少 |
| **同学会** | 当年被嘲笑的同学 | 现在身份反转 |
| **相亲局** | 被相亲对象嫌弃 | 揭露真实身份 |
| **商业谈判** | 被大公司看不起 | 反收购/反杀 |
| **家族聚会** | 家族内部被欺压 | 关键时刻翻盘 |
| **医院场景** | 被医生护士怠慢 | 院长来道歉 |

### 打脸场景模板

```
【场景一：会所打脸】
被挡在会所门外 → 嘲笑是穷鬼
→ 亮出VIP身份 → 服务员震惊
→ 对方被赶出去

【场景二：豪车打脸】
女朋友被豪车车主撩 → 被嘲笑骑电动车
→ 主角开的是全球限量版 → 对方灰溜溜走
→ 女朋友后悔

【场景三：同学会打脸】
当年风云人物 → 现在混得最好来炫耀
→ 主角低调出现 → 各种瞧不起
→ 主角亮出身份 → 当年风云人物变跟班

【场景四：商务谈判打脸】
被大公司代表轻视 → 以为是小公司
→ 揭露真正实力 → 对方公司需要主角救命
→ 反客为主
```

## 六、城市背景设计

### 城市经济生态

```
【一线城市生态】
├─ 顶层：资本大鳄、权力核心
├─ 中层：金领、高管、专业人士
├─ 底层：白领、打工者
└─ 边缘：外卖、快递、底层服务业

【主角切入方式】
├─ 创业：从小做起，步步为营
├─ 就业：进入顶级公司，职场斗争
├─ 继承：隐藏身份，进入核心
└─ 异能：利用异能获得优势
```

### 城市文化特点

| 城市类型 | 文化特点 | 适合题材 |
|---------|---------|---------|
| 京城 | 政治中心，权力为王 | 官场、权谋 |
| 魔都 | 商业发达，资本博弈 | 商战、职场 |
| 沿海 | 开放包容，机会多 | 创业、贸易 |
| 内陆 | 人情社会，关系复杂 | 家族、拆迁 |
| 特区 | 创新活力，新贵崛起 | 创业、科技 |

### 城市与剧情联动

```
【第一卷：起点】
小城市/县城 → 被欺负 → 去大城市
（建立代入感，小打小闹）

【第二卷：崛起】
二线城市 → 进入核心圈子 → 小有名气
（积累资源，稳步上升）

【第三卷：巅峰】
一线城市 → 与顶级势力交锋 → 成为核心
（终极目标，装最大逼）

【番外：回归】
衣锦还乡 → 曾经看不起的人 → 打脸
（终极爽点）
```
