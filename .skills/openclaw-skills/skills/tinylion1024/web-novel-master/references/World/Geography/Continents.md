# 大陆设定

## 概述

大陆/世界设定是网文世界观的物理骨架，决定了地图的大小、势力分布、资源地点和剧情发展空间。一个好的世界设定应该层次分明，让主角的探索有方向感，让读者有代入感。

---

## 一、地图层级设计

### 玄幻世界层级

```
【诸天万界】
├─ 仙域（仙人居住，超脱一切）
├─ 圣域（大能聚集）
├─ 神州大陆（主角所在，主要舞台）
│   ├─ 中州（中央区域，顶级势力汇聚）
│   ├─ 东域（沿海发达地区）
│   ├─ 西域（沙漠/高原，苦寒之地）
│   ├─ 南域（热带雨林，妖兽横行）
│   └─ 北域（冰雪世界，极北之地）
└─ 附属小世界（秘境/遗迹/次元空间）

【修仙界层级】
├─ 下界（凡人界，故事主要发生地）
├─ 灵界（灵气充沛，修士聚集）
└─ 仙界（仙人居住，飞升目的地）
```

### 都市世界层级

```
【全球格局】
├─ 发达国家/地区（主要舞台）
│   ├─ 一线城市（顶级势力聚集）
│   ├─ 二线城市（中型势力）
│   └─ 三四线城市（基层舞台）
└─ 发展中国家/地区（机遇之地）
```

## 二、区域设计模板

### 中州设定（中央区域）

| 设定 | 内容 |
|------|------|
| **势力** | 顶级宗门3-5个、古老世家5-8个、皇室 |
| **资源** | 顶级灵脉、多处秘境、传承丰富 |
| **竞争** | 最激烈，决定天下格局 |
| **主角机会** | 最多但也最危险 |

### 东域设定（沿海区域）

| 设定 | 内容 |
|------|------|
| **势力** | 商会发达、海运繁荣、相对开放 |
| **资源** | 海底灵脉、海族宝物、海外传承 |
| **特点** | 商业氛围浓厚、机会与风险并存 |
| **主角机会** | 商业崛起、海洋传承 |

### 西域设定（沙漠/高原）

| 设定 | 内容 |
|------|------|
| **势力** | 佛教/喇嘛教、游牧民族、古老王国 |
| **资源** | 矿产丰富、古城遗迹、特殊矿物 |
| **特点** | 民风彪悍、宗教色彩浓厚 |
| **主角机会** | 佛道传承、古老宝藏 |

### 南域设定（热带雨林）

| 设定 | 内容 |
|------|------|
| **势力** | 妖族聚集、蛊虫修炼、瘴气毒物 |
| **资源** | 灵药丰富、妖兽内丹、蛊术传承 |
| **特点** | 危险重重，机缘也大 |
| **主角机会** | 毒医传承、妖兽伙伴 |

### 北域设定（冰雪世界）

| 设定 | 内容 |
|------|------|
| **势力** | 冰系宗门、雪族、冰凤凰后裔 |
| **资源** | 冰魄晶、寒冰泉水、上古遗迹 |
| **特点** | 环境恶劣、修炼条件苛刻 |
| **主角机会** | 冰系传承、寒冰体质 |

## 三、关键地点设定

### 势力核心地

| 地点 | 所属 | 特点 | 剧情价值 |
|------|------|------|---------|
| 万剑山 | 万剑宗 | 剑修圣地，万剑齐飞 | 剑道传承 |
| 丹道谷 | 丹道圣地 | 终年丹香，丹药交易中心 | 炼丹主线 |
| 天机城 | 天机阁 | 情报中心，消息最灵通 | 获取情报 |
| 轮回殿 | 神秘势力 | 掌管轮回转世 | 身世之谜 |
| 万宝阁 | 商会联盟 | 拍卖行，顶级交易 | 淘宝捡漏 |

### 危险禁地

| 禁地 | 危险等级 | 特点 | 可能的机缘 |
|------|---------|------|----------|
| 万古禁地 | 极高 | 封印远古存在，进入者必死 | 远古传承 |
| 妖兽森林 | 高 | 高级妖兽聚集 | 驯兽/内丹 |
| 深渊魔域 | 极高 | 魔气侵蚀，心智受损 | 魔族传承 |
| 天火遗迹 | 高 | 终年天火，无法接近 | 火焰传承 |
| 荒古废墟 | 中 | 古城遗迹，上古文明 | 宝藏/传承 |

### 秘境设定

```
【秘境类型】
├─ 宗门秘境 → 宗门弟子试炼用
├─ 遗迹秘境 → 远古大能洞府
├─ 天然秘境 → 天地灵气汇聚形成
├─ 特殊秘境 → 轮回/时间/空间相关

【秘境规则】
├─ 进入条件：实力/资质/钥匙
├─ 时间限制：在秘境停留时间有限
├─ 产出限制：资源有限，先到先得
└─ 危险程度：与收益成正比
```

## 四、世界观构建模板

### 地图结构模板

```
【世界地图】
┌─────────────────────────────────────────────┐
│                  中州（中央）                   │
│  ┌──────────────────────────────────────┐   │
│  │  万剑宗 │ 太虚宫 │ 皇室 │ 古老世家群  │   │
│  └──────────────────────────────────────┘   │
│      ↑               ↑              ↑       │
│    东域            南域           西域      │
│  沿海商城        妖兽森林       沙漠高原    │
│                                             │
│              北域（冰雪）                    │
│            冰宫/雪族圣地                     │
└─────────────────────────────────────────────┘

【经典剧情线】
├─ 从东域小城出发 → 进入中州 → 成为顶级势力
├─ 穿越到西域 → 获得佛道传承 → 返回中州
└─ 在南域历险 → 收服妖兽 → 北域冰宫结因果
```

### 世界格局模板

```
【天下格局】
├─ 顶级势力（2-3个）→ 决定天下大事
├─ 一流势力（10-20个）→ 区域霸主
├─ 二流势力（100+）→ 基础格局
└─ 三流/散修 → 芸芸众生

【势力分布原则】
├─ 顶级势力分散各地，避免直接冲突
├─ 同一区域一流势力不超过3个
├─ 有公共资源（秘境/灵脉）引发争夺
└─ 交通要道是势力交界处，冲突频繁
```

## 五、地图设计原则

### 核心原则

| 原则 | 说明 | 应用 |
|------|------|------|
| **层次分明** | 不同区域有不同特色 | 主角探索有方向感 |
| **危险与机遇并存** | 越危险地方资源越丰富 | 推动剧情发展 |
| **势力平衡** | 没有绝对霸主 | 主角有崛起空间 |
| **留白原则** | 不要把所有设定都写死 | 给读者想象空间 |

### 地图扩展节奏

```
【第一卷】东域小城/中州边缘
├─ 主角起点
├─ 1-2个势力冲突
└─ 小试牛刀

【第二卷】中州核心
├─ 进入大势力
├─ 4-5个势力交织
└─ 名声初显

【第三卷】天下格局
├─ 跨区域冒险
├─ 接触隐秘势力
└─ 成为一方霸主

【后续卷】诸天万界/仙域
├─ 世界观扩大
├─ 更多位面
└─ 最终飞升
```

## 六、地域特色设计

### 地域性格特点

| 地域 | 居民性格 | 文化特点 | 适合主角背景 |
|------|---------|---------|------------|
| 中州 | 精明、势力 | 礼仪之邦，繁文缛节 | 大家族、皇子 |
| 东域 | 开放、重商 | 商业气息浓厚 | 商人家庭、穿越者 |
| 西域 | 彪悍、虔诚 | 宗教色彩浓重 | 苦行者、佛道 |
| 南域 | 狡黠、神秘 | 蛊术巫毒盛行 | 毒医、妖兽伙伴 |
| 北域 | 坚韧、冷傲 | 冰雪文化 | 冰属性体质 |

### 势力分布图

```
                    ┌─── 皇室（皇权）───┐
                    │                  │
        ┌───────┐  ┌─┴──────┐  ┌───────┐
        │万剑宗 │  │ 太虚宫 │  │ 古老  │
        │(剑道) │──┤(综合) │──┤ 世家群 │
        └───────┘  └────────┘  └───────┘
              ↑                    ↑
        ┌─────┴──────┐     ┌────┴────┐
        │ 商会联盟    │     │ 天机阁  │
        │ (东域代表) │     │ (情报)  │
        └────────────┘     └─────────┘
```

### 地图与剧情联动

```
【地图设计 × 剧情节奏】

起点（边缘小城）
→ 成长（进入核心区域）
→ 冲突（卷入大势力斗争）
→ 历险（探索危险秘境）
→ 崛起（成为一方霸主）
→ 终极（影响天下格局）
→ 飞升（离开这个世界）
```
