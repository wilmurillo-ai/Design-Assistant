# 宗门势力

## 概述

宗门势力是玄幻修仙小说的核心势力体系，决定了主角的出身、成长路径和社会地位。一个设计合理的宗门体系能让升级更具仪式感，让战斗更有层次感。

---

## 一、宗门等级体系

### 势力等级划分

| 等级 | 名称 | 底蕴 | 掌门境界 | 威慑范围 |
|------|------|------|---------|---------|
| **超一流** | 圣地、古教 | 传承万年+，仙人坐镇 | 大乘/渡劫 | 整个大陆 |
| **一流** | 顶级宗门 | 传承千年+，底蕴深厚 | 化神/大乘 | 数州之地 |
| **二流** | 高级宗门 | 传承数百年 | 元婴/化神 | 一州之地 |
| **三流** | 普通宗门 | 传承百年左右 | 金丹/元婴 | 区域范围 |
| **不入流** | 小门派 | 数十年底蕴 | 筑基/金丹 | 一城之地 |

### 典型势力格局

```
【超一流势力】（泰山北斗，轻易不出的老怪物）
├─ 天机阁（情报天下第一）
├─ 万剑宗（剑道第一）
├─ 丹道圣地（丹药垄断）
└─ 远古世家（血脉传承）

【一流势力】（格局主导者）
├─ 各洲顶级宗门（每洲3-5个）
├─ 古老家族（世代传承）
└─ 帝国皇室（凡间统治者）

【二流势力】（中坚力量）
├─ 各郡王牌宗门
└─ 大家族分支

【底层势力】（日常冲突对象）
├─ 小门派
├─ 家族旁支
└─ 散修联盟
```

## 二、宗门组织架构

### 典型架构

```
【宗主】— 最高权力，决策宗门大事
    ↓
【副宗主/长老团】— 辅助决策，管理宗门
    ↓
【峰主/堂主】— 管理具体部门
    ↓
【核心弟子】— 宗门未来，各峰精英
    ↓
【普通弟子】— 宗门基石，外门弟子
    ↓
【杂役/记名】— 最底层，做杂务
```

### 宗门职位明细

| 职位 | 人数 | 权力 | 待遇 |
|------|------|------|------|
| 宗主 | 1人 | 最高决策权 | 宗门供奉+传承优先 |
| 太上长老 | 1-3人 | 监督权，重大时刻出手 | 不参与管理 |
| 长老 | 10-20人 | 各峰/堂管理 | 资源分配优先 |
| 首席弟子 | 每峰1人 | 弟子中地位最高 | 获得掌门亲自指导 |
| 核心弟子 | 数十人 | 宗门核心力量 | 资源充足 |
| 内门弟子 | 数百人 | 执行任务 | 稳定资源 |
| 外门弟子 | 数千人 | 基础劳作 | 基本保障 |

## 三、宗门资源体系

### 核心资源

| 资源 | 作用 | 获取难度 | 分配规则 |
|------|------|---------|---------|
| **灵脉** | 宗门立身之本，灵气核心 | 极难获得 | 宗主掌控 |
| **传承殿** | 功法传承，决定宗门未来 | 需要底蕴 | 长老团决定 |
| **秘境入口** | 试炼/资源产出 | 看守严密 | 分配给弟子 |
| **丹药储备** | 培养弟子 | 炼药师炼制 | 按贡献分配 |
| **法器阁** | 武器装备 | 锻造师打造 | 贡献点兑换 |

### 贡献点制度

```
【贡献点获取】
完成任务 → 获得贡献点
↓积累
【贡献点兑换】
100点 → 炼气期功法
500点 → 筑基丹一颗
1000点 → 中品法器一件
5000点 → 核心弟子选拔资格
10000点 → 进入秘境机会

【贡献点扣除】
违反门规 → 扣除贡献点
任务失败 → 扣除部分贡献点
背叛宗门 → 废除修为，逐出宗门
```

## 四、宗门关系设定

### 势力关系图

```
【同盟关系】
同一等级宗门结盟 → 资源共享 → 联合对外
    ↓
【依附关系】
小宗门依附大宗门 → 提供资源/人手 → 获得庇护
    ↓
【竞争关系】
同等宗门争夺 → 秘境入口/灵脉/弟子
    ↓
【敌对关系】
正邪不两立 → 宗门大战 → 灭门惨案
```

### 冲突类型

| 冲突类型 | 起因 | 典型场景 |
|---------|------|---------|
| **秘境争夺** | 发现新秘境 | 数十宗门汇聚 |
| **灵脉开采** | 灵脉枯竭/新发现 | 生死大战 |
| **弟子争夺** | 天才弟子被挖角 | 两派恩怨 |
| **正邪大战** | 理念不合 | 全面战争 |
| **传承之争** | 远古遗迹开启 | 群雄争霸 |

## 五、宗门模板

### 模板一：剑修宗门

```
【特点】专注剑道，攻击力最强，性格大多孤傲
【等级】一流/超一流
【代表】万剑宗、剑冢、剑谷
【核心资源】剑冢（历代剑修陨落之地）
【修炼路径】养剑→炼剑→人剑合一→以身化剑
【宗门规矩】剑道至上，弱者无人权
【典型冲突】剑冢试炼、剑道大比、剑魔出世
```

### 模板二：炼丹宗门

```
【特点】掌控丹药命脉，富得流油，但战斗力一般
【等级】一流
【代表】丹道圣地、百草堂
【核心资源】丹方、古丹遗迹
【修炼路径】识药→炼丹→丹道宗师→以丹入道
【宗门规矩】炼丹师地位最高，非炼丹师是二等公民
【典型冲突】丹方争夺、师徒反目、商业竞争
```

### 模板三：体修宗门

```
【特点】肉身成圣，近战无敌，但修炼痛苦
【等级】二流（顶级体修宗门可到一流）
【代表】蛮神殿、不灭宗
【核心资源】炼体秘境、肉身传承
【修炼路径】淬体→锻骨→炼脏→金身→不灭
【宗门规矩】强者为尊，弱者服从
【典型冲突】肉身试炼、境界压制、肉身成圣
```

### 模板四：综合型宗门

```
【特点】各脉齐全，实力均衡，但无特别突出
【等级】超一流/一流
【代表】太虚宫、天玄宗
【核心资源】综合传承，万法兼容
【修炼路径】博采众长→自成一脉→万法归一
【宗门规矩】以和为贵，但有核心利益时也会出手
【典型冲突】内部分歧、外部挑战、主脉争夺
```

## 六、宗门爽点设计

### 主角在宗门中的爽点

| 爽点 | 设计 | 效果 |
|------|------|------|
| **入门考核** | 废柴身份→震惊四座→被顶级长老收徒 | 身份逆转 |
| **核心弟子选拔** | 越级战胜→震惊全场→地位飙升 | 实力证明 |
| **宗门大比** | 从不被看好→一路碾压→获得传承 | 名利双收 |
| **秘境探险** | 捡漏远古传承→获得逆天机缘 | 资源暴涨 |
| **叛徒清算** | 曾经的欺辱者→一一清算 | 快意恩仇 |
| **外敌入侵** | 关键时刻力挽狂澜→成为宗门英雄 | 地位巩固 |
| **传承争夺** | 击败所有竞争者→获得掌门之位 | 权力巅峰 |

### 宗门冲突模板

```
【弱小时期】
被欺辱 → 忍辱负重 → 暗中积蓄力量
    ↓
【成长时期】
展露天赋 → 引起注意 → 获得机遇
    ↓
【崛起时期】
连续打脸 → 震动宗门 → 成为核心
    ↓
【巅峰时期】
带领宗门 → 走向辉煌 → 成为传奇
```
