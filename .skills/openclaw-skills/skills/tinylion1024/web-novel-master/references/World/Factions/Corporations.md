# 企业势力

## 概述

现代都市/科幻背景下的企业势力设定，是都市异能、商战、职场类网文的核心势力体系。企业之间的竞争、职场潜规则、商业异能斗争，为主角提供了丰富的升级打脸场景。

---

## 一、企业等级体系

### 企业规模划分

| 等级 | 规模 | 市值/营收 | 掌控资源 | 代表 |
|------|------|----------|---------|------|
| **巨头** |跨国集团 | 万亿级以上 | 行业垄断、国家经济命脉 | 托尼家族、罗斯柴尔德 |
| **一线** | 大型集团 | 千亿级 | 城市经济支柱 | 上市公司、百年企业 |
| **二线** | 中型企业 | 百亿级 | 细分领域领先 | 行业龙头 |
| **三线** | 小型企业 | 十亿级 | 地区知名 | 创业公司 |
| **不入流** | 小微企业 | 亿级以下 | 勉强生存 | 初创公司 |

### 都市商战格局

```
【顶级财团】（掌控国家经济命脉）
├─ 金融财团（银行、保险、投资）
├─ 实业财团（能源、矿产、重工）
├─ 科技财团（互联网、AI、生物）
└─ 军火财团（地下势力、灰色收入）

【一线企业】（各行业龙头）
├─ 互联网巨头
├─ 房地产大亨
├─ 连锁商业帝国
└─ 医药集团

【二线企业】（地方豪强）
├─ 地方龙头企业
└─ 家族企业

【中小企业】（普通人舞台）
└─ 职场小白起点
```

## 二、企业组织架构

### 典型架构

```
【董事会】— 最高决策层
    ↓
【CEO/总裁】— 日常最高管理者
    ↓
【VP/副总裁】— 各板块负责人
    ↓
【总监】— 部门负责人
    ↓
【经理】— 项目/团队负责人
    ↓
【主管】— 小组负责人
    ↓
【员工】— 普通员工
```

### 职位级别与权力

| 职位 | 权力范围 | 典型行为 |
|------|---------|---------|
| CEO | 公司整体决策 | 战略方向、人事任免 |
| 总裁 | 某一集团板块 | 资源调配、业务整合 |
| VP | 多个部门 | 跨部门协调 |
| 总监 | 单个部门 | 部门预算、人事权 |
| 经理 | 项目/团队 | 任务分配、绩效评定 |
| 主管 | 小组 | 日常管理、汇报 |
| 员工 | 本职工作 | 执行任务 |

## 三、商业异能体系

### 商业异能类型

| 类型 | 能力 | 商战应用 | 稀有度 |
|------|------|---------|--------|
| **读心异能** | 读取他人想法 | 谈判、面试、投资 | 稀有 |
| **预知异能** | 短暂看到未来 | 投资、赌博、商业预判 | 传说 |
| **魅惑异能** | 影响他人决策 | 谈判、客户关系 | 稀有 |
| **时间异能** | 短暂加速/减速 | 紧急决策、危机处理 | 传说 |
| **空间异能** | 瞬移、储物 | 物流、谈判筹码展示 | 稀有 |
| **召唤异能** | 召唤生物/物品 | 展示实力、谈判威慑 | 稀有 |

### 企业内斗模式

```
【派系斗争】
├─ 太子派（继承人相关）
├─ 职业经理人派
├─ 创始人派
└─ 外部资本派

【典型内斗场景】
方案被打压 → 被调离核心岗位 → 忍辱负重
→ 关键项目出问题 → 关键时刻逆转 → 打脸对方
→ 获得更高层支持 → 彻底翻盘
```

## 四、职场打脸模板

### 经典职场爽点

| 场景 | 设定 | 爽点 |
|------|------|------|
| **入职即被看不起** | 空降/新人/被嘲笑背景 | 实际身份揭露，震惊全场 |
| **方案被打压** | 好方案被否决/窃取 | 事实说话，对方被打脸 |
| **竞标翻盘** | 大项目被截胡 | 绝境逆转，最终拿下 |
| **对手公司挖人** | 被高薪诱惑 | 坚守忠诚，获得更大回报 |
| **商业间谍** | 被陷害/被栽赃 | 揭露真相，反将一军 |
| **收购/并购** | 被收购方 → 反收购 | 以弱胜强，蛇吞象 |

### 都市商战经典模板

```
【废柴入职】
普通大学毕业 → 入职被嘲笑 → 展现真实实力
→ 打脸领导 → 获得赏识

【空降高管】
临危受命 → 被老员工架空 → 暗中调查
→ 掌握证据 → 雷厉风行 → 权力巩固

【创业逆袭】
被原公司背叛 → 创业被嘲笑 → 做出成绩
→ 原公司危机 → 关键时刻收购 → 成为行业巨头
```

## 五、家族企业设定

### 家族企业特点

```
【优势】
├─ 资源集中，决策快速
├─ 忠诚度高，向心力强
└─ 品牌积累，用户信任

【劣势】
├─ 任人唯亲，人才流失
├─ 创新不足，守成心态
└─ 内部派系，权力斗争
```

### 继承者设定模板

```
【废柴继承者】
被嘲笑 → 隐藏身份 → 关键时刻证明
→ 获得家族认可 → 成为真正继承人

【私生子回归】
被排挤 → 母亲去世 → 回归家族
→ 被陷害 → 反击 → 夺回属于自己的一切

【上门女婿】
被看不起 → 妻子家道中落 → 关键时刻出手
→ 逆转局势 → 成为家族实际掌控者
```

## 六、商业版图设定

### 典型行业格局

```
【互联网行业】
阿里系 vs 腾讯系 vs 字节系 → 三足鼎立

【房地产行业】
万科系 vs 恒大系 vs 碧桂园系 → 群雄逐鹿

【金融行业】
国有四大行 vs 股份制银行 vs 民间资本

【医药行业】
跨国药企 vs 国企药企 vs 创新药企
```

### 势力关系模板

```
【合作】 → 资源互换、联合竞标
【竞争】 → 价格战、人才争夺
【收购】 → 大鱼吃小鱼
【并购】 → 强强联合
【打压】 → 垄断市场、排挤对手
【联盟】 → 中小企业的生存之道
```
