# 武侠力量体系

## 概述

武侠体系以冷兵器战斗和内力为核心，融合江湖恩怨、门派争斗、秘籍传承等元素。核心爽点在于武功的高低、江湖地位的变迁、以及侠客的快意恩仇。

---

## 一、武功境界

### 传统武侠境界

| 境界 | 别称 | 特征 | 代表人物 |
|------|------|------|---------|
| **三流** | 初入门径 | 粗浅武功，普通武者 | 江湖小卒 |
| **二流** | 小有所成 | 内力初成，可开碑裂石 | 门派弟子 |
| **一流** | 一方高手 | 内力深厚，独当一面 | 门派长老 |
| **绝顶** | 江湖名家 | 顶尖高手，少有对手 | 掌门/家主 |
| **超一流** | 宗师之姿 | 接近宗师，开宗立派 | 隐世高人 |
| **宗师** | 一代宗师 | 内力化境，心意相通 | 传说人物 |
| **先天** | 先天之境 | 返璞归真，天人合一 | 武林神话 |
| **天人** | 破碎虚空 | 超脱凡俗，飞升而去 | 书中仙人 |

### 内力等级

```
【初期】疏通经脉、凝聚内力 → 打通任督二脉
【中期】内力深厚、可外放伤敌 → 隔空点穴
【后期】内力化劲、可凝气为形 → 剑气刀气
【圆满】收发随心、无坚不摧 → 摘花飞叶皆可伤人
```

## 三、门派势力

### 江湖格局

```
【顶级门派】（泰山北斗）
少林、武当、明教、丐帮（正邪各一）
↓超然
【一流门派】（底蕴深厚）
峨眉、昆仑、崆峒、华山、嵩山
↓竞争
【二流门派】（地方豪强）
各地方门派，争夺资源
↓淘汰
【三流门派/不入流】（底层江湖）
小门派、山寨、镖局
```

### 势力关系图

```
正派联盟 ←→ 魔教邪派
  ↑           ↓
少林 ←→ 武当 ←→ 明教
  ↓           ↓
其他正派    邪派联盟
```

## 五、战斗场景模板

### 江湖对决模板

```
【比武招亲】废柴被嘲笑 → 关键时刻出手 → 震惊四座
→ 抱得美人归

【门派大比】小门派弟子 → 越级挑战 → 一战成名
→ 地位飙升

【正邪大战】主角夹缝中生存 → 关键时刻力挽狂澜
→ 奠定江湖地位

【秘籍争夺】多方势力争夺 → 主角渔翁得利
→ 获得传承，实力暴涨
```

### 越级战斗条件

| 条件 | 说明 | 举例 |
|------|------|------|
| 顶级武功 | 上乘内功 + 精妙招式 | 九阴真经 + 降龙十八掌 |
| 丰富实战 | 经验碾压 | 老江湖 vs 新秀 |
| 底牌 | 暗器/毒/秘术 | 小李飞刀、化功大法 |
| 意志 | 置之死地而后生 | 绝境爆发 |
