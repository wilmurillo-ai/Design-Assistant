# 科幻科技体系

## 概述

科幻体系以科技力量为核心，融合星际探索、机甲战斗、基因进化、赛博朋克等元素。核心爽点在于科技碾压、以弱胜强、以及人类极限的突破。

---

## 一、科技等级

### 文明等级（卡尔达肖夫指数）

| 等级 | 特征 | 能力 | 代表设定 |
|------|------|------|---------|
| **0型** | 行星文明 | 利用本土资源 | 现代地球 |
| **0.7型** | 初级星际 | 太空殖民开始 | 星际探索时代 |
| **1型** | 恒星文明 | 掌控行星能源 | 地球联邦 |
| **1.5型** | 行星级帝国 | 跨星系航行 | 银河帝国 |
| **2型** | 恒星文明 | 利用恒星能源 | 戴森球文明 |
| **3型** | 星系文明 | 掌控整个星系 | 宇宙帝国 |

### 科技树分支

```
【能源科技】→ 核聚变 → 反物质 → 真空能 → 零点能
【材料科技】→ 纳米材料 → 超钛合金 → 能量护盾 → 规则武器
【生命科技】→ 基因优化 → 机械义体 → 意识上传 → 数字生命
【信息科技】→ 量子计算 → 强AI → 心灵链接 → 全知系统
【空间科技】→ 曲率引擎 → 空间折叠 → 虫洞 → 维度跳跃
```

## 三、星际势力设定

### 典型势力格局

```
【银河联邦】（秩序阵营）
统一货币与法律，维护银河和平
↓对抗
【星际帝国】（侵略阵营】
军事化管理，扩张欲望强烈
↓对抗
【星际联盟】（中立阵营】
多星球联合，利益至上
↓合作
【星际商会】（商业阵营】
掌控星际贸易，科技领先
↓冲突
【星际海盗】（混乱阵营】
烧杀抢掠，无恶不作
```

### 势力关系设计

```
核心爽点：主角从弱小势力崛起，逐步对抗/加入/颠覆各大势力

【典型模板】
废物流落 → 加入小型势力 → 展现天赋 → 获得核心科技
→ 挑战大势力 → 以弱胜强 → 建立自己的势力
→ 对抗星际帝国/拯救银河联邦
```

## 四、赛博朋克设定

### 社会结构

```
【顶层：AI管理者/超级企业CEO】
掌控核心AI，决策整个城市
    ↓
【中层：企业高管/义体改造者】
拥有高级义体，社会精英
    ↓
【底层：普通人/数据乞丐】
生活在阴影中，被数据淹没
    ↓
【地下：黑客/改造人/变异者】
游走于法律之外，灰色地带
```

### 义体改造等级

| 等级 | 改造程度 | 能力 | 社会地位 |
|------|---------|------|---------|
| **一级** | 辅助型 | 增强感官/记忆 | 普通市民 |
| **二级** | 功能型 | 战斗义肢/加速 | 工人/士兵 |
| **三级** | 深度型 | 超高速/高输出 | 精英士兵 |
| **四级** | 完全型 | 意识上传/机械身体 | 企业核心 |
| **五级** | 升华型 | 超越人类极限 | 传说存在 |

## 五、科幻爽点设计

### 核心爽点

| 爽点类型 | 设计方法 | 例子 |
|---------|---------|------|
| **科技碾压** | 以弱胜强，用智慧/创意击败强敌 | 黑客入侵AI、改造人打败机甲 |
| **身份逆转** | 废柴其实是超级天才/隐藏血脉 | 流浪汉是失落的皇子/超级AI |
| **捡漏暴富** | 废墟中捡到远古科技/失落传承 | 捡到一艘外星战舰 |
| **科技突破** | 研究出改变世界的发明 | 无限能源、治愈癌症 |
| **机甲觉醒** | 普通人驾驶传说机甲大杀四方 | 草根驾驶员击败王牌 |

### 经典场景模板

```
【星际捡漏】
流落废墟 → 发现古代遗迹 → 获得失落科技
→ 实力暴涨 → 震惊所有势力

【机甲觉醒】
被认为废柴 → 机甲意外认主 → 战斗力飙升
→ 在大赛/战争中大放异彩

【黑客逆袭】
被嘲笑技术不行 → 关键时刻入侵核心系统
→ 以一人之力扭转战局

【星际霸主】
从一颗殖民星起步 → 整合资源 → 建立势力
→ 对抗星际帝国 → 成为银河霸主
```
