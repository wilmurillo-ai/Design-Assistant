# 系统规则

## 概述

系统流是网文中最核心的金手指类型之一。系统作为主角的外挂，提供任务、奖励、技能，简化了传统修炼的复杂性，让升级之路清晰可见。系统流的核心爽点在于：任务奖励的期待感、数值碾压的快感、系统的神秘感。

---

## 一、系统类型

### 主流系统类型

| 系统类型 | 核心功能 | 爽点设计 | 代表作品 |
|---------|---------|---------|---------|
| **任务系统** | 发布任务→完成→奖励 | 任务奖励的期待感 | 大部分系统文 |
| **选择系统** | 面临选择→不同结果 | 命运掌控感 | 设定系 |
| **合成系统** | 低级物品→高级物品 | 无限可能的期待 | 炼器/炼丹流 |
| **召唤系统** | 召唤历史人物/生物 | 名将名人助战 | 召唤流 |
| **签到系统** | 每日签到→累计奖励 | 日常小惊喜 | 日常+系统 |
| **属性系统** | 透明化属性面板 | 数值碾压的爽感 | 数据流 |
| **直播系统** | 观众打赏→获得力量 | 社交互动爽感 | 直播流 |
| **宠物系统** | 捕捉/培养宠物 | 养成+收集 | 宝可梦流 |

### 系统能力等级

| 系统等级 | 解锁功能 | 外观表现 | 稀有度 |
|---------|---------|---------|--------|
| **Lv.1** | 基础面板、任务 | 普通文字提示 | 常见 |
| **Lv.2** | 属性加成、技能 | 声音提示 | 稀有 |
| **Lv.3** | 空间背包、合成 | 虚拟投影 | 珍贵 |
| **Lv.4** | 智能辅助、商城 | 全息投影 | 传说 |
| **Lv.5** | 系统升级、自我意识 | 人形系统精灵 | 神话 |

## 二、系统核心机制

### 任务系统

```
【任务类型】
├─ 主线任务 → 推进剧情，必做
├─ 支线任务 → 额外奖励，可选
├─ 日常任务 → 每日刷新，积累资源
├─ 隐藏任务 → 触发条件不明，高奖励
└─ 限时任务 → 限时完成，超额奖励

【奖励类型】
├─ 经验值 → 升级
├─ 技能点 → 解锁技能
├─ 道具 → 装备/消耗品
├─ 货币 → 系统商城购买
└─ 特殊 → 称号/血统/身份

【惩罚机制】（可选）
├─ 任务失败 → 扣除经验/道具
├─ 拒绝任务 → 降低系统好感度
└─ 放弃任务 → 触发负面事件
```

### 商城系统

```
【系统商城】
├─ 技能商店 → 购买技能书/技能升级
├─ 装备商店 → 购买武器/防具
├─ 道具商店 → 消耗品/材料
├─ 血统商店 → 购买血脉/体质
└─ 外观商店 → 皮肤/特效（不增加战力）

【货币类型】
├─ 任务币 → 完成日常/主线获得
├─ 荣誉点 → PVP/特殊成就获得
└─ 充值 → 小说中一般没有，保持平衡
```

### 合成系统

```
【合成规则】
1级材料 × 10 → 2级材料
2级材料 × 10 → 3级材料
...

【装备合成】
武器碎片 × 100 → 完整武器
防具碎片 × 100 → 完整防具

【丹药合成】
低级药材 × 特定比例 → 高级丹药

【合成暴击】（爽点）
└─ 10%概率触发 → 产出翻倍
```

## 三、系统与剧情融合

### 剧情触发模式

```
【危机触发】主角濒死 → 系统激活
【继承触发】获得遗物 → 系统转移
【契约触发】签订特殊契约 → 系统绑定
【重生触发】重生到过去 → 系统随行
【穿越触发】穿越异世界 → 系统激活
```

### 系统升级条件

```
【升级条件】
├─ 完成主线任务 → 系统升级
├─ 累计签到天数 → 系统升级
├─ 达成特定成就 → 解锁功能
└─ 宿主实力提升 → 系统解锁

【升级效果】
├─ 解锁新功能模块
├─ 任务奖励提升
├─ 商城折扣增加
└─ 系统外观升级
```

## 四、系统爽点设计

### 核心爽点

| 爽点 | 设计方法 | 效果 |
|------|---------|------|
| **任务奖励** | 高风险任务→高额奖励 | 期待感 |
| **属性碾压** | 系统面板透明→数值优势 | 优越感 |
| **技能多样** | 多样技能组合→战术变化 | 丰富感 |
| **隐藏成就** | 意外触发隐藏成就→逆天奖励 | 惊喜感 |
| **系统升级** | 完成里程碑→功能质变 | 成长感 |
| **称号获取** | 达成成就→获得拉风称号 | 荣誉感 |

### 系统打脸模板

```
【正常流程】
系统激活 → 获得第一个任务
→ 完成新手任务 → 获得奖励
→ 实力提升 → 打脸之前嘲笑的人

【装逼打脸】
遇到强敌 → 对方嘲讽
→ 系统提示新任务（击败对手）
→ 接受任务 → 击败对手 → 获得奖励
→ 对方震惊

【越级挑战】
遇到高境界敌人
→ 系统发布越级挑战任务
→ 完成任务获得越级手段
→ 成功越级 → 震惊全场
```

## 五、系统规则设定

### 平衡性原则

```
【不能无敌】
├─ 系统有使用次数/冷却限制
├─ 高级技能需要完成任务解锁
├─ 系统货币获取需要努力
└─ 避免主角太强导致没有紧张感

【代价机制】
├─ 某些能力需要消耗寿命
├─ 使用禁忌技能有反噬
├─ 系统升级需要完成任务
└─ 维持系统需要资源
```

### 系统与世界观融合

```
【融入方式】
├─ 系统是远古大能留下的传承
├─ 系统是异世界意志的碎片
├─ 系统是主角前世的记忆
└─ 系统是某个实验的产物

【悬念设计】
├─ 系统的真正目的是什么？
├─ 系统背后有什么存在？
├─ 系统的创造者是谁？
└─ 为什么选择主角？
```

## 六、系统文模板

### 典型开局模板

```
【濒死激活】
主角被围殴 → 濒死之际
→ "叮，系统激活中..."
→ 获得新手礼包
→ 反杀敌人 → 打脸完成

【废物觉醒】
被认为是废物 → 被退婚/嘲讽
→ 意外获得系统
→ 完成新手任务 → 实力提升
→ 打脸之前看不起自己的人

【穿越携系统】
穿越到异世界 → 系统随身
→ 第一个任务：生存
→ 完成 → 获得奖励
→ 在异世界崛起
```

### 系统节奏模板

```
第一章：系统激活
第三章：第一个任务完成，获得奖励
第五章：第一场战斗，系统辅助
第十章：小有名气，打脸之前的敌人
第二十章：系统升级，解锁新功能
第五十章：成为一方霸主
第一百章：系统秘密开始揭露
...
```
