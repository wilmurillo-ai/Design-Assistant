# 社会规则

## 概述

社会规则是网文世界观的骨架，决定了人物行为的底层逻辑。一个设计合理的社会体系让升级更有意义，让打脸更有爽感，让读者更容易代入。

---

## 一、社会阶层体系

### 修仙世界阶层

```
【顶层：仙人/远古存在】
掌控规则，不死不灭
  ↓
【上层：大乘/渡劫期/圣地】
决定天下格局，轻易不出
  ↓
【中层：化神/元婴期/一流宗门】
各方霸主，决定区域格局
  ↓
【下层：金丹/筑基期/普通修士】
宗门弟子、散修，实力说话
  ↓
【底层：凡人/炼气期】
蝼蚁般的存在，炮灰
```

### 都市世界阶层

```
【顶层：顶级权贵/古老家族】
掌控国家经济命脉，权势滔天
  ↓
【上层：商界巨头/政界高层】
社会精英，呼风唤雨
  ↓
【中层：中产阶层/小老板】
普通人天花板，为生活奔波
  ↓
【下层：普通员工/白领】
996打工人，社会大多数
  ↓
【底层：失业者/流浪汉】
被社会遗忘的存在
```

## 二、社会规则设定

### 底层规则

| 规则 | 说明 | 网文应用 |
|------|------|---------|
| **实力为尊** | 实力决定地位，弱肉强食 | 升级是唯一出路 |
| **资源有限** | 高阶资源稀缺，必须争夺 | 冲突的根本原因 |
| **强者制定规则** | 顶层决定秩序 | 主角成强者后可改变规则 |
| **因果报应** | 杀强者有天谴/业火 | 控制战斗结果 |
| **传承为王** | 有传承才能走得更远 | 争夺传承是核心爽点 |

### 社会运转规则

```
【资源分配规则】
├─ 宗门：贡献点制度
├─ 家族：血脉/嫡庶决定资源
├─ 散修：弱肉强食，自由竞争
└─ 国家：律法制约，秩序为王

【晋升规则】
├─ 宗门：实力/贡献决定地位
├─ 家族：继承权/能力决定地位
├─ 国家：战功/政绩决定官位
└─ 散修：实力为尊，打出来的地位

【冲突解决规则】
├─ 宗门：宗门规矩，长老裁决
├─ 家族：家族会议，家主裁决
├─ 国家：法律/强者裁决
└─ 散修：拳头说话，死伤自负
```

## 三、货币与经济体系

### 修仙世界货币

| 货币 | 等级 | 换算 | 购买力 |
|------|------|------|--------|
| 铜板 | 最低 | 100铜=1两银子 | 普通凡人一天开销 |
| 白银 | 中 | 10两=1锭 | 普通人家一年开销 |
| 黄金 | 高 | 1锭黄金=100两白银 | 修士入门用 |
| 灵石 | 通用 | 下品→中品→上品→极品 | 修士主要货币 |
| 仙石 | 最高 | 1仙石=100极品灵石 | 仙界通用 |

### 都市经济体系

```
【货币层级】
├─ 普通人：人民币/美元
├─ 富裕阶层：房产/股票
├─ 富豪阶层：股权/公司
└─ 顶级家族：国家经济命脉

【资源体系】
├─ 生存资源：食物、水、住房
├─ 发展资源：教育、医疗、人脉
├─ 权力资源：信息、渠道、背景
└─ 核心资源：传承、灵气、特殊物品
```

## 四、势力关系法则

### 势力博弈规则

```
【合作法则】
├─ 利益交换：互利共赢
├─ 强弱联合：强者保护弱者，弱者依附
├─ 联姻绑定：婚姻是势力联合的纽带
└─ 盟约约束：违约有代价

【竞争法则】
├─ 资源争夺：零和游戏，有限资源
├─ 人才争夺：天才决定势力未来
├─ 话语权争夺：制定规则的权利
└─ 势力范围：地盘意识，此消彼长

【压制法则】
├─ 上位者制定规则
├─ 下位者必须服从
├─ 越级挑战代价巨大
└─ 但天才可以打破常规
```

### 势力冲突类型

| 冲突类型 | 起因 | 解决方式 | 常见场景 |
|---------|------|---------|---------|
| **资源争夺** | 灵脉/秘境/宝物 | 实力说话 | 战争/比武 |
| **继承之争** | 掌门/家主之位 | 能力/手段 | 家族内斗 |
| **复仇清算** | 祖先恩怨/个人仇恨 | 斩草除根 | 灭门惨案 |
| **正邪对立** | 理念不合 | 你死我活 | 大战 |
| **收编/碾压** | 看上对方资源 | 强者通吃 | 灭门/招安 |

## 五、社会规则与爽点

### 规则打破爽点

```
【以弱胜强】
规则：实力为尊，弱者应该服从
打破：主角以弱胜强
爽点：震惊众人，打破既定认知

【废柴逆袭】
规则：血脉/天赋决定命运
打破：废柴获得传承/系统
爽点：身份逆转，啪啪打脸

【改写规则】
规则：强者制定规则
打破：主角成为强者，改写规则
爽点：终极爽点，站在顶峰

【打破阶级】
规则：阶层固化，难以跨越
打破：主角从底层一路崛起
爽点：草根逆袭，励志
```

### 社会潜规则

```
【潜规则一：低调行事】
├─ 初出茅庐不要太高调
├─ 容易招来强者注意
└─ 主角往往打破这条

【潜规则二：站队要准】
├─ 选择错误的靠山会导致灭门
├─ 主角往往在关键时刻选择正确
└─ 这是政治智慧

【潜规则三：斩草除根】
├─ 得罪人要彻底消灭
├─ 留下后患是愚蠢行为
└─ 主角因为各种原因留下敌人，最终反杀

【潜规则四：利益至上】
├─ 没有永远的朋友，只有永远的利益
├─ 亲人/朋友都可能背叛
└─ 主角拥有永远不会背叛的伙伴
```

## 六、社会体系设计模板

### 修仙社会体系模板

```
【秩序井然型】
├─ 国家管理修士，禁止私斗
├─ 宗门受国家管辖
├─ 有修士律法约束
└─ 爽点：权贵犯法与庶民同罪

【弱肉强食型】
├─ 实力为尊，没有秩序
├─ 散修生存艰难
├─ 宗门占据资源
└─ 爽点：从蝼蚁崛起为巨擘

【多方角力型】
├─ 宗门/家族/帝国三足鼎立
├─ 相互制衡
├─ 主角在夹缝中崛起
└─ 爽点：成为第三极，改变格局
```

### 都市社会体系模板

```
【阶层分明型】
├─ 大家族掌控一切
├─ 普通人难以跨越
├─ 婚姻是跨越阶层的途径
└─ 爽点：废柴赘婿逆袭

【资本为王型】
├─ 商界巨头操控社会
├─ 职场斗争激烈
├─ 金钱可以买通一切
└─ 爽点：以智慧/异能战胜资本

【权力为王型】
├─ 政界家族掌控权力
├─ 权力斗争残酷
├─ 一人得道鸡犬升天
└─ 爽点：从政界小白到权力巅峰
```
