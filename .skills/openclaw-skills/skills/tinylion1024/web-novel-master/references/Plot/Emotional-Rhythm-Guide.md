# 情绪节奏指南

网络小说的本质是情绪消费。读者追更是为了情绪波动——爽、甜、燃、虐、哭、笑。

---

## 1. 情绪节奏的本质

**好节奏 = 张弛有度 = 压抑与满足的循环**

| 状态 | 情绪 | 读者感受 |
|-------|---------|----------------|
| 张（憋） | 压抑/紧张/危机 | 屏住呼吸，等待爆发 |
| 弛（放） | 释放/甜蜜/满足 | 舒服/爽/傻笑 |

```
憋 → 小爽 → 憋 → 中爽 → 大憋 → 大爽
 ↓      ↓      ↓       ↓       ↓        ↓
压抑  小反击  加重   中反击   困境    反击
        压迫          打脸
```

---

## 2. 情绪曲线类型

### 1. 先压后放型（最常见）

```
低点 ────────────────── 高点
   ↓                         ↑
压抑/困境                爆发/反击/打脸
   ↓                         ↑
   └────── 蓄力 ─────────┘
```

**适用：** 废物流逆袭、退婚打脸、装逼打脸

**节奏：**
```
第1-3章：压抑（憋）
第4-5章：小反抗（小爽）
第6-7章：更大压迫（更憋）
第8章：反击打脸（大爽）
```

### 2. 连续上升型

```
低 ─────────────────── 高
   ↑       ↑       ↑
 小爽   中爽   大爽
```

**适用：** 甜宠文、升级流、系统流

**节奏：**
```
每3-5章：一个小型高潮
每10章：一个大型高潮
连续上升，爽感叠加
```

### 3. 波动型

```
高 ─┬── 高 ─┬── 高
      ↓      ↓      ↓
低 ─┴── 低 ─┴── 低
  ↑       ↑       ↑
虐     甜     虐
```

**适用：** 言情、虐恋

**节奏：**
```
虐3章 → 甜1章 → 虐3章 → 更甜
            ↑                        ↑
         小甜                    大甜（在一起/结婚）
```

### 4. 波浪型

```
高 ─── 高峰 ─── 低 ─── 高峰 ─── 低
  ↑           ↓      ↑        ↓
燃         缓冲    虐      燃      结尾
```

**适用：** 热血、战斗

**节奏：**
```
战斗 ─── 胜利 ─── 创伤 ─── 恢复 ─── 更强战斗
   ↓         ↓         ↓         ↓            ↓
  燃         爽        虐        期待          更燃
```

## 3. 情绪节点设计

### 关键情绪节点

| 节点 | 位置 | 情绪类型 | 强度 |
|------|----------|-------------|-----------|
| **觉醒点** | 第1章 | 爽/燃 | ★★★★★ |
| **第一次打脸** | 第3章 | 爽 | ★★★★ |
| **情感羁绊** | 第5章 | 甜/期待 | ★★★ |
| **第一个大高潮** | 第10章 | 燃/爽 | ★★★★★ |
| **虐点** | 第15章 | 虐/哭 | ★★★★ |
| **身份揭露** | 第20章 | 震惊/爽 | ★★★★★ |
| **终极高潮** | 大结局 | 燃/爽/甜 | ★★★★★ |

### 每章情绪结构

```
章开头：承接上一章情绪（或转折）
        ↓
情绪铺垫：压抑/甜蜜/紧张
        ↓
情绪蓄力：越来越强
        ↓
情绪爆发：爽/甜/虐/燃
        ↓
章结尾：钩子 + 余韵
```

## 4. 压抑与释放比例

### 压抑阈值

| 压抑等级 | 最大连续章数 | 超过风险 |
|------------------|------------------------|------------------|
| 轻度压抑 | 3-5章 | 读者流失 |
| 中度压抑 | 2-3章 | 读者焦虑 |
| 重度压抑 | 1-2章 | 读者弃书 |

### 各类型压抑-满足比例

| 类型 | 憋:爽 | 描述 |
|-------|-------|-------------|
| 都市爽文 | 3:7 | 快节奏，连续爽点 |
| 玄幻仙侠 | 4:6 | 压抑稍长，爽感更强 |
| 甜宠文 | 2:8 | 甜蜜为主，偶尔小虐 |
| 虐文 | 7:3 | 虐为主，大甜结尾 |
| 悬疑 | 5:5 | 紧张与解开交替 |

### 压抑释放技巧

**技巧1：小爽过渡**
```
压抑3章后，释放一个小爽
→ 让读者喘口气
→ 保持期待
```

**技巧2：信息透露**
```
压抑过程中，透露一点好消息
→ 读者知道苦难快结束了
→ 但不知道具体怎么反转
→ 期待更强
```

**技巧3：双重蓄力**
```
压抑时越压抑
→ 释放时爽感越强
→ 但压抑时间不能超过阈值
```

## 5. 情绪节奏工具

### 1. 情绪气压表

```
□ 低气压（压抑）：虐/危机/憋
□ 中气压（平稳）：日常/进展/蓄力
□ 高气压（紧张）：冲突/对峙/危机
□ 爆发气压（释放）：爽/燃/打脸/高潮
```

**每章结尾检查：**
- 当前气压是什么？
- 下一章应该往哪个方向走？
- 是否有连续3章同一气压？

### 2. 情绪日志

记录每章情绪状态，便于整体把控：

```
第1章：爆发气压（觉醒爽点）
第2章：中气压（日常蓄力）
第3章：高气压（压抑）
第4章：爆发气压（第一次打脸）
第5章：中气压（甜蜜互动）
...
```

## 6. 各类型情绪节奏

### 都市爽文

```
节奏：快
憋：1-2章内释放
爽：打脸要密集

典型节奏：
打脸 ─── 小爽 ─── 压抑 ─── 大爽 ─── 小日常
   ↓           ↓           ↓           ↓           ↓
第1章       第2章       第3章       第4章       第5章
```

### 玄幻仙侠

```
节奏：中
憋：可以稍长，但不超过5章
爽：升级感要足够

典型节奏：
困境 ─── 机缘 ─── 修炼 ─── 突破 ─── 验证
    ↓            ↓             ↓            ↓             ↓
   憋1         期待          小爽         大爽         打脸
```

### 甜宠文

```
节奏：慢
憋：最多1章
爽：连续甜蜜

典型节奏：
撒糖 ─── 吃醋 ─── 化解 ─── 更甜 ─── 误会 ─── 大甜
  ↓       ↓          ↓          ↓            ↓               ↓
 甜1     酸1        甜2        爆发         虐1           结婚
```

### 悬疑

```
节奏：紧张为主
憋：持续紧张
爽：解开悬念

典型节奏：
悬念 ─── 线索 ─── 迷雾 ─── 误导 ─── 真相 ─── 新悬念
   ↓         ↓       ↓        ↓         ↓           ↓
 好奇      紧张     更迷     误导     爽          更大好奇
```

## 7. 情绪节奏检查清单

### 每章检查

- [ ] 这章的情绪曲线形状是什么？
- [ ] 开头的情绪是否承接上一章？
- [ ] 压抑是否超过3章？
- [ ] 爽点是否在情绪高峰？
- [ ] 结尾钩子够不够强？

### 每10章检查

- [ ] 是否至少有1个大爽点？
- [ ] 压抑-满足比例是否合适？
- [ ] 情绪是否持续上升/波动？
- [ ] 读者追更时会不会觉得"不够爽"？

---

## 8. 情绪与名场面配合

名场面必须在情绪高峰。

### 情绪到位时

```
情绪高峰 + 名场面 = 爆炸爽
```

### 情绪不到位时

```
平淡情绪 + 名场面 = 效果减半
→ 需要先蓄力
→ 再释放名场面
```

### 名场面情绪公式

```
【1】情绪蓄力（前20%）
   压抑/被嘲笑/痛苦

【2】情绪积累（中50%）
   越来越压抑/越来越期待

【3】情绪爆发（后30%）
   名场面出现
   爽/燃/甜/打脸
   金句轰炸

【4】情绪余韵（结尾钩子）
   还没结束...
   后面还有更爆炸的...
```
