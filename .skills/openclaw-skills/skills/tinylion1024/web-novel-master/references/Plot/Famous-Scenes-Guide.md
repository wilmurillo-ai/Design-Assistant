# 名场面设计指南

名场面是爆款网络小说的核心竞争力——让读者欲罢不能、忍不住分享的场景。

---

## 1. 名场面的定义

**名场面 = 情绪爆发点 × 戏剧张力 × 记忆锚点**

| 元素 | 描述 |
|---------|-------------|
| **情绪爆发点** | 读者情绪被充分调动（爽/哭/怒/笑/燃） |
| **戏剧张力** | 冲突、反转、出人意料 |
| **记忆锚点** | 金句/名台词/名画面，读者能复述 |

---

## 2. 名场面类型系统

### A. 爽感名场面（让读者"爽"）

| 类型 | 核心公式 | 典型场景 |
|------|-------------|---------------|
| **打脸** | 羞辱 → 逆转 → 碾压 | 废物当众出丑 → 揭露真实身份 → 巴掌响彻全场 |
| **验证** | 质疑 → 展示 → 震惊 | 众人质疑 → 主角出手 → 全场震惊 |
| **装逼** | 低调 → 被迫 → 不情愿展示 | 主角不想出手 → 被逼 → 实力碾压 |
| **复仇** | 隐忍 → 时机成熟 → 算账 | 多年隐忍 → 敌人落难 → 亲手复仇 |
| **反击** | 被逼绝境 → 金手指 → 逆转 | 被逼到极限 → 系统激活 → 绝地翻盘 |

### B. 情感名场面（让读者"哭/甜/虐"）

| 类型 | 核心公式 | 典型场景 |
|------|-------------|---------------|
| **牺牲** | 危机 → 选择 → 牺牲 | 为救XX，主角选择牺牲自己 |
| **表白** | 误会 → 真相 → 表白 | 发现对方一直默默付出 → 真情表白 |
| **离别** | 必须离开 → 依依不舍 → 承诺 | 必须离开 → 含泪告别 → 下次相见 |
| **重逢** | 多年分离 → 物是人非 → 珍惜 | 多年后重逢 → 一切都变了 → 珍惜当下 |
| **误会** | 看到表面 → 真相 → 心痛 | 看到他和其他女人在一起 → 其实是妹妹 → 后悔 |

### C. 热血名场面（让读者"燃"）

| 类型 | 核心公式 | 典型场景 |
|------|-------------|---------------|
| **英雄** | 不可能 → 拼死一搏 → 奇迹 | 所有人都说不可能 → 主角冒死 → 竟然成功了 |
| **守护** | 家人被威胁 → 爆发 → 碾压 | 家人被害 → 主角怒发冲冠 → 灭敌 |
| **团战** | 队友一个个倒下 → 最终逆转 → 团队胜利 | 队友一个个倒下 → 最后翻盘 → 团队胜利 |
| **觉醒** | 被压制 → 极限 → 觉醒 | 一直被殴打 → 触发条件 → 实力暴涨 |

### D. 悬疑/反转名场面（让读者"懵/惊"）

| 类型 | 核心公式 | 典型场景 |
|------|-------------|---------------|
| **身份反转** | 以为是好人 → 揭露 → 震惊 | 信任的人 → 竟是幕后黑手 |
| **实力反转** | 以为是废物 → 真相 → 打脸 | 以为主角是废物 → 实际最强 |
| **关系反转** | 以为是敌人 → 真相 → 懵了 | 以为是竞争对手 → 实际是亲兄弟 |
| **阴谋揭露** | 表面平静 → 暴露 → 混乱 | 看似和平 → 揭露惊人阴谋 |

## 3. 名场面时间线规划

### 20章名场面节奏

```
第1章：【觉醒名场面】系统激活/重生回归/退婚打脸
         → 立刻抓住读者

第3章：【第一次打脸名场面】小试牛刀，初步验证爽感
         → 建立"这个故事很爽"的期待

第5章：【情感名场面】第一次心动/误会/离别
         → 建立情感羁绊

第7章：【装逼名场面】当众打脸/展示实力
         → 积累"等着看主角碾压"的情绪

第10章：【第一个大高潮名场面】副本通关/复仇第一步
         → 第一个情绪高峰

第12章：【牺牲/虐名场面】重要角色牺牲/离别
         → 制造情绪低谷，增强后续爽感

第15章：【身份反转名场面】隐藏身份揭露
         → 重大悬念释放

第18章：【团战/热血名场面】守护XX/团队战
         → 再次拉高情绪

第20章：【大结局名场面】最终对决/最终复仇/甜蜜结局
         → 高潮结尾
```

### 节奏原则

- 每3-5章必须有1个中型名场面
- 每10章必须有1个大型名场面（全书高潮锚点）
- 名场面之间必须有"情绪缓冲"（日常/甜蜜/蓄力）

## 4. 名场面设计三要素

### 1. 【蓄力】名场面之前必须有铺垫

羞辱要够狠（不够狠打脸不够爽）
误会要够深（不够深解开心甜不够甜）
压迫要够强（不够强燃放不够燃）

**反面教材：**
> 主角毫无铺垫突然变强

**正面教材：**
> 所有人都嘲笑他是废物，未婚妻退婚，小姨子被欺负...第3章他出手时，读者已经憋着一股气等着发泄

### 2. 【爆发】情绪到达顶点时，要写透

动作/语言/心理描写要到位
最高潮处不要停笔
金句/名台词必须出现

**反面教材：**
> 他展示了实力，众人都很震惊。

**正面教材：**
> 他轻轻抬起手。
>
> "等一下。"
>
> 声音不大，却如石子投入静水，整个大殿鸦雀无声。
>
> 笼罩在他身上的封印，碎裂。
>
> "你说什么？"大长老的声音微微颤抖。
>
> "我说，"他抬起头，眼中燃烧着三年来压抑的怒火，"你们欠我的，是时候讨回来了。"

### 3. 【余韵】名场面之后必须有情绪延续

描写周围人的反应
描写主角的心理状态
为下一个名场面埋下种子

## 5. 名场面公式模板

### 打脸名场面公式

```
【蓄力】XX羞辱主角，说"你这辈子估计都..."
【爆发】主角淡淡一笑，揭露/说出/做出XX
【结果】众人震惊，XX脸色发白
【金句】"你说的'这辈子'...好像有点短？"
```

**完整示例：**
> "废物就是废物，"王少把茶泼到他脸上，"你这辈子只配捡我的剩饭吃。"
>
> 林墨没动。他只是笑了笑。
>
> 门外，十几辆黑色迈巴赫缓缓停下。一位白发苍苍的老人下车，快步走进酒店。
>
> "主人！"老人单膝跪地，"属下接驾来迟！"
>
> 王少的脸，绿了。

### 装逼名场面公式

```
【背景】众人鄙视，主角被无视
【挑衅】有人故意刁难/羞辱
【不情愿】主角叹气："既然你们都想看..."
【展示】主角展示实力/揭露真相
【金句】"不是我想装逼，是你们逼我的。"
```

### 热血名场面公式

```
【绝境】队友倒下，敌人逼近，资源耗尽
【触发】XX（家人/爱人/信念）被触碰
【爆发】主角觉醒/突破/爆发
【碾压】敌人从嘲讽变为恐惧
【金句】"谁也不能动我在乎的人。"
```

### 甜宠名场面公式

```
【吃醋】看到他/她和别人在一起
【行动】直接拉过/壁咚/宣示主权
【金句】"以后离他远点，听到没？"
【反应】对方心跳加速/脸红/挣扎
```

## 7. 金句库

### 打脸金句

- "你说的'这辈子'...好像有点短？"
- "我不是针对谁，你们各位...都是垃圾。"
- "谁刚才叫我是废物来着？"
- "有些东西，你这辈子都够不着。"

### 装逼金句

- "不是我想装逼，是你们逼我的。"
- "随便玩玩而已，没想到你们当真了。"
- "这玩意儿？我小孩的玩具。"

### 热血金句

- "谁也不能动我在乎的人。"
- "此战，为兄弟！为宗门！"
- "燃尽一切，又怎样！"

### 甜蜜金句

- "以后离他远点，听到没？"
- "只有我能欺负我的人。"
- "想我了？嗯？"

## 8. 名场面密度检查

### 每2000字必须包含

| 类型 | 数量 | 描述 |
|------|----------|-------------|
| 小爽 | 1-2 | 轻度打脸/小惊喜/暧昧互动 |
| 中爽 | 1 | 明显打脸/情感升温/热血场面 |
| 钩子 | 1-2 | 信息炸弹/悬念/下章预告 |

### 检查清单

- [ ] 这章有爽点吗？（没有，读者觉得"水"）
- [ ] 爽点前有蓄力吗？（没有蓄力打脸不够爽）
- [ ] 有金句吗？（没有金句名场面不"名"）
- [ ] 结尾钩子够不够强？（读者想不想点下一章？）
