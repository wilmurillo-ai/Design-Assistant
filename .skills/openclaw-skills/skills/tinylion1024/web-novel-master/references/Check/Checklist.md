# 网文写作全流程自检清单

> **适用场景**：Fast / Professional / Industrial 三模式通用
> **适用题材**：男频 · 女频 · 爽文 · 甜宠 · 重生 · 追妻 · 玄幻 · 都市

---

## 一、开篇自检（黄金前三章）

- [ ] 第一章内出现：主角 + 金手指 + 核心冲突
- [ ] 500 字内抓住眼球，无冗长背景铺垫
- [ ] 主线目标清晰：复仇 / 变强 / 求生 / 逆袭 / 恋爱 / 搞钱
- [ ] 无多余配角抢戏，世界观不堆砌
- [ ] 第一章结尾带钩子（悬念 / 危机 / 反转）
- [ ] 书名、简介、开篇风格统一
- [ ] 节奏干净，不水字数、不废话

---

## 二、人设自检

- [ ] 主角性格统一，不忽强忽弱、不前后矛盾
- [ ] 主角动机明确，行为有逻辑
- [ ] 反派不降智，压迫感合理
- [ ] 配角有记忆点，不全员工具人
- [ ] 男女主人设标签清晰：软萌 / 腹黑 / 霸总 / 杀伐果断 / 舔狗 / 追妻 等
- [ ] 人设不混乱：不一边恋爱脑一边极端冷静
- [ ] 符合赛道调性：女频不强行男频思维，男频不矫情

---

## 三、金手指自检

- [ ] 金手指前三章出现，作用清晰
- [ ] 能力不崩坏、不无限逆天
- [ ] 有规则 / 限制 / 冷却 / 代价，避免无敌崩盘
- [ ] 与剧情绑定，不是摆设
- [ ] 能直接制造爽点 / 甜点 / 转机
- [ ] 规则不频繁修改
- [ ] 与题材匹配：重生→先知；系统→任务奖励；修仙→功法 / 体质

---

## 四、剧情节奏自检

- [ ] 每 3 章一个小高潮 / 小爽点 / 小反转
- [ ] 每 10 章一个大高潮
- [ ] 不水日常、不水对话、不原地打转
- [ ] 主线清晰，支线不干扰主线
- [ ] 剧情推进快，信息密度足够
- [ ] 每章结尾留钩子，提升追读
- [ ] 节奏稳定，不忽快忽慢

---

## 五、爽点 & 甜宠 & 情绪自检

### 男频爽文
- [ ] 打脸及时、不憋屈
- [ ] 升级 / 夺宝 / 装逼 / 复仇密度足够
- [ ] 势力碾压、等级对比清晰
- [ ] 反派快速退场，不长期恶心读者

### 女频甜宠 / 追妻 / 重生
- [ ] 甜而不腻，不工业糖精
- [ ] 互动自然，情绪拉扯到位
- [ ] 追妻火葬场：前期虐 → 后期追 → 女主态度明确
- [ ] 重生复仇：打脸快、反派报应快
- [ ] 情绪稳定：不忽甜忽虐崩人设

---

## 六、逻辑漏洞自检

- [ ] 人物行为符合身份、性格、处境
- [ ] 世界规则统一，不前后矛盾
- [ ] 时间线、年龄、地图不乱
- [ ] 金手指不突然超纲、不凭空加强
- [ ] 势力格局合理，不忽强忽弱
- [ ] 剧情因果完整：有因必有果
- [ ] 不强行巧合、不强行降智

---

## 七、章节结构自检

- [ ] 章节标题有吸引力、带信息点
- [ ] 单章字数稳定（主流 1800–2200 字）
- [ ] 开头承接上一章，不跳戏
- [ ] 中间有推进：冲突 / 互动 / 反转
- [ ] 结尾留钩子：悬念 / 危机 / 惊喜 / 表白 / 打脸
- [ ] 短句多、段落短，阅读流畅
- [ ] 无大段设定、大段心理、大段回忆插叙

---

## 八、文笔 & 对话自检

- [ ] 对话符合人物身份、年龄、性格
- [ ] 不全员一个说话语气
- [ ] 对话不水字数，有信息 / 有情绪 / 有推进
- [ ] 动作 + 神态 + 对话搭配，不单调
- [ ] 文风统一：不古言不现言乱跳
- [ ] 无错字、病句、标点错误
- [ ] 通俗流畅，不晦涩、不强行装文笔

---

## 九、违禁词 & 平台规范自检

- [ ] 无政治敏感、历史歪曲、违规地名
- [ ] 无色情擦边、低俗描写
- [ ] 无过度暴力、血腥、恐怖
- [ ] 无人身歧视、性别歧视、地域歧视
- [ ] 不美化违法犯罪、不宣扬不良价值观
- [ ] 不出现平台违禁词、敏感词
- [ ] 合规可过审，不踩红线

---

## 十、最终发布自检

- [ ] 全文通读一遍，无错漏、乱码、重复
- [ ] 章节顺序正确，无断章、漏章
- [ ] 标签、分类、简介准确
- [ ] 前三章最强，留住第一波读者
- [ ] 存稿充足，更新节奏稳定
- [ ] 文风统一、情绪统一、节奏统一
- [ ] 可直接发布，无风险、无硬伤

---

> **提示**：建议每章写完后立即自检，全文完成后通检一遍发现问题及时修改。
