# 审核目录

审核清单目录 - 自检清单、避坑指南、质量审核。

## 目录结构

```
Check/
└── Checklist.md           # 网文写作全流程自检清单
```

## 内容概览

| 文件 | 内容 | 适用场景 |
|------|------|---------|
| **Checklist.md** | 十大自检维度：开篇、人设、金手指、节奏、爽点、逻辑、章节结构、文笔、违禁词、发布 | Fast / Professional / Industrial 三模式通用 |

## 十大自检维度

```
一、开篇自检     → 黄金前三章，主角+金手指+核心冲突
二、人设自检     → 性格统一、动机明确、反派不降智
三、金手指自检   → 前三章出现、有规则限制、与剧情绑定
四、剧情节奏自检 → 每3章小高潮、每10章大高潮
五、爽点甜宠自检 → 男频打脸及时、女频甜而不腻
六、逻辑漏洞自检 → 行为合理、规则统一、无强行巧合
七、章节结构自检 → 标题吸引、字数稳定、结尾留钩子
八、文笔对话自检 → 语气区分、不水字数、文风统一
九、违禁词自检   → 无政治敏感、无低俗擦边、合规可过审
十、发布自检     → 通读无误、章节完整、存稿充足
```
