## 组件说明

### 1. SkillLibrary（技能库）

负责技能的存储和检索：
- **持久化**：YAML/JSON 文件 + ChromaDB 向量存储
- **检索**：语义相似度搜索
- **防重**：余弦相似度 > 0.9 时触发合并

### 2. TaskPlanner（任务规划器）

决定如何处理用户任务：
- `use_skill`：复用现有技能
- `generate_skill`：生成新技能
- `composite`：组合多个技能

### 3. SkillGenerator（技能生成器）

使用 LLM 自动生成技能代码：
- 强制继承 BaseSkill 基类
- 强制实现三个方法
- 模板限制，避免代码失控

### 4. SkillTester（技能测试器）

在安全沙盒中测试技能：
- **SimpleTester**：本地执行（开发用）
- **DockerTester**：容器隔离（生产用）
- 超时、内存、CPU 限制

### 5. SkillEvaluator（技能评估器）

量化评分系统：
```
Score = 0.4×SR + 0.2×Sp + 0.2×R + 0.2×Q
```
- SR: 成功率
- Sp: 速度
- R: 鲁棒性
- Q: 代码质量

### 6. SkillOptimizer（技能优化器）

双轨优化：
- **改写法**：保留逻辑，修复错误
- **压缩法**：简化代码，提升效率

### 7. SkillComposer（技能组合器）

复杂任务分解：
- DAG 有向无环图
- 拓扑排序执行
- 自动传递输入输出

### 8. AutoRefactor（自动重构）

定时任务：
- 分析使用模式
- 发现高频技能组合
- 自动生成 meta-skill

## 数据流

```
用户请求 → Planner 路由 → 
    │
    ├─→ 已有技能 → Tester → Evaluator → Library
    │
    └─→ 新技能 → Generator → Tester → Evaluator →
        │                              │
        └────── 未通过 ───────────────→ Optimizer → Tester
                                           │
                                      通过 ↓
                                    → Library
```
