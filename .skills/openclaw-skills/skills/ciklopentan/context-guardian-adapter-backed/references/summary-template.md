# Summary Template

Use this markdown shape for durable recovery summaries.

```markdown
# Context Guardian Summary

## Goal
<goal>

## Current Phase
<current phase>

## Completed
- <completed step>

## Decisions
- Decision: <decision>
  Reason: <reason>

## Constraints
- <constraint>

## Artifacts
- path: <path>
  role: <kind>

## Open Issues
- <issue>

## Last Successful Action
<action>

## Next Action
<next action>

## Recovery Notes
- Resume from the latest durable state.
- Re-check important files before editing.
- Do not redo completed work without validation.
```
