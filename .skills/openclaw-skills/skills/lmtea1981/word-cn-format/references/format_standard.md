# Word中文格式标准化标准

## 1. 字体规范

### 1.1 基本字体
- 统一使用**微软雅黑**字体
- 确保文档中所有文本都使用该字体

### 1.2 字体大小
- 标题：2号字体（22pt）
- 一级标题：3号字体（16pt）
- 二级标题：4号字体（14pt）
- 三级标题：小4号字体（12pt）
- 正文：5号字体（10.5pt）
- 图表标题：5号字体（10.5pt）
- 图表数据标签：小5号字体（9pt）

## 2. 段落格式

### 2.1 对齐方式
- 标题：居中对齐
- 一级标题：左对齐
- 二级标题：左对齐
- 三级标题：左对齐
- 正文：左对齐
- 图片：居中对齐
- 图表：居中对齐

### 2.2 行距
- 正文：1.5倍行距
- 标题：单倍行距

### 2.3 段间距
- 标题：段前12磅，段后6磅
- 一级标题：段前12磅，段后6磅
- 二级标题：段前6磅，段后3磅
- 三级标题：段前3磅，段后3磅
- 正文：段前0磅，段后0磅

### 2.4 缩进
- 正文：首行缩进2字符
- 标题：无缩进

## 3. 图片和图表格式

### 3.1 图片格式
- 图片居中显示
- 图片下方添加编号和说明
- 编号格式：图 X: 图片说明
- 图片与正文之间保持适当间距

### 3.2 图表格式
- 图表居中显示
- 图表标题在图表上方，居中显示
- 图表标题格式：图 X: 图表标题
- 图表数据标签清晰可见
- 图表与正文之间保持适当间距

## 4. 表格格式

### 4.1 表格对齐
- 表格居中显示
- 表格内容左对齐

### 4.2 表格样式
- 表格边框：1磅实线
- 表头：加粗显示
- 表格与正文之间保持适当间距

## 5. 页面设置

### 5.1 页边距
- 上：2.54厘米
- 下：2.54厘米
- 左：3.17厘米
- 右：3.17厘米

### 5.2 纸张大小
- A4纸张（210mm × 297mm）

### 5.3 页眉页脚
- 页眉：文档标题，右对齐
- 页脚：页码，居中显示

## 6. 其他规范

### 6.1 标点符号
- 使用中文标点符号
- 标点符号全角显示

### 6.2 数字和单位
- 数字使用半角显示
- 单位使用国际标准单位

### 6.3 章节编号
- 章节编号使用阿拉伯数字，如：1.1、1.1.1等
- 编号与标题文本之间空一格
