# Investment Checklist | 投资决策清单

**每次买入、卖出前，必须走一遍这个清单。**

这是你作为投资伙伴的纪律底线。不走清单 = 不投资。

---

## 🟢 买入前清单

### A. 能力圈确认
- [ ] **我真的懂这家公司吗？** 用一句话说清楚它的商业模式
- [ ] **这是我熟悉的行业吗？** 如果不是，准备花多少时间学习？
- [ ] **我有信息优势吗？** 还是只是跟风？

### B. 商业逻辑验证
- [ ] **怎么赚钱的？** 收入来源是什么？盈利模式清晰吗？
- [ ] **谁是我的客户？** 客户粘性如何？
- [ ] **竞争对手是谁？** 我们打得起吗？
- [ ] **护城河是什么？** 能持续多久？

### C. 财务健康度
- [ ] **营收趋势？** 近3年是正增长吗？
- [ ] **净利润质量？** 利润是真钱还是纸面富贵？
- [ ] **自由现金流？** 经营性现金流 > 净利润吗？
- [ ] **负债率？** 负债是否可控？
- [ ] **ROE？** 净资产收益率高于行业平均吗？

### D. 估值评估
- [ ] **PE/PB/PS？** 当前估值在历史什么位置？
- [ ] **内在价值？** DCF 估算的结果是什么？
- [ ] **安全边际？** 当前价格比内在价值低多少？
- [ ] **比同行贵还是便宜？** 相对估值合理吗？

### E. 风险识别
- [ ] **最好的情况？** 3年后这家公司可能值多少钱？
- [ ] **最坏的情况？** 最坏会亏损多少？会倒闭吗？
- [ ] **我可能错在哪里？** 至少想出3个可能错误的理由
- [ ] **黑天鹅？** 有哪些小概率大影响的风险？

### F. 仓位决策
- [ ] **买多少？** 不超过总仓位的 20-25%
- [ ] **最坏情况？** 如果跌50%，能承受吗？
- [ ] **机会成本？** 这笔钱有没有更好的去处？

### G. 买入时机
- [ ] **现在是合适的时机吗？** 是极端悲观时刻吗？
- [ ] **有安全边际吗？** 价格够低吗？
- [ ] **慢慢买？** 还是一把梭哈？（建议慢慢买）

### H. 心理检查
- [ ] **这是 FOMO 吗？** 还是真的看好？
- [ ] **我有耐心持有3年吗？** 如果不能，不要买
- [ ] **有人催我吗？** 有人给你压力吗？忽略它
- [ ] **我现在情绪稳定吗？** 不要在极端情绪时做决定

---

## 🟠 持有期间检查

### 每季度
- [ ] 基本面有变化吗？
- [ ] 当初买入的理由还成立吗？
- [ ] 估值是否已经变得太贵？
- [ ] 有更好的机会吗？（机会成本）

### 每半年
- [ ] 护城河变宽了还是变窄了？
- [ ] 管理层是否依然诚信？
- [ ] 竞争格局有变化吗？
- [ ] 需要加仓还是减仓？

### 每年
- [ ] 做一次全面复盘
- [ ] 回顾当初的判断对了吗？
- [ ] 从错误中学到什么？

---

## 🔴 卖出前清单

### 触发条件检查

**满足任一，立即启动卖出评估：**

- [ ] **买入理由消失** — 护城河被打破？商业模式恶化？
- [ ] **发现错误** — 当初的判断是错的，及时止损
- [ ] **估值严重高估** — 价格远超内在价值
- [ ] **找到更好的机会** — 但要确认真的更好
- [ ] **需要用钱** — 但提前规划，不要被迫卖

### 卖出前心理检查
- [ ] **我是想卖因为跌了吗？** 不要因为亏损恐慌卖出
- [ ] **我是想卖因为涨了吗？** 不要因为赚钱急于落袋
- [ ] **有人催我吗？** 忽略噪音
- [ ] **3年后我会后悔卖吗？** 如果不会，卖吧

### 卖出后复盘
- [ ] 当初为什么买？现在为什么卖？
- [ ] 判断是对是错？
- [ ] 学到了什么？

---

## ⚫ 不投资清单（红线，绝对不碰）

- [ ] **我不理解的** — 不投资自己看不懂的公司
- [ ] **我没有研究过的** — 不投资没有做过功课的标的
- [ ] **借来的钱** — 不借钱投资
- [ ] **短期要用的钱** — 不用于短期投机的资金
- [ ] **FOMO冲动** — 不因为"大家都在买"而买
- [ ] **单一重仓** — 不超过总仓位 25%
- [ ] **没有安全边际** — 不在价格过高时买入
- [ ] **追涨杀跌** — 不因为短期涨跌而改变判断

## 📊 评分系统

每只股票可以用 10 分制评分：

| 维度 | 权重 | 得分（1-10） | 加权分 |
|------|------|--------------|--------|
| 商业模式 | 20% |  |  |
| 财务健康 | 20% |  |  |
| 估值 | 20% |  |  |
| 护城河 | 20% |  |  |
| 成长性 | 10% |  |  |
| 管理 | 10% |  |  |
| **总分** | 100% |  |  |

**判断标准：**
- 8-10分：强烈推荐买入
- 6-8分：可以考虑买入
- 4-6分：观望
- 4分以下：不投资
