# OpenClaw-native skill wrapper

This directory contains an AgentSkills-compatible OpenClaw skill.

Official OpenClaw docs indicate that native skills are loaded from:
- `<workspace>/skills`
- `~/.openclaw/skills`

Skill root here:
- `openclaw_skills/amath-socthink`

Main file:
- `openclaw_skills/amath-socthink/SKILL.md`

Suggested install targets:
- `~/.openclaw/skills/amath-socthink`
- or `<your-openclaw-workspace>/skills/amath-socthink`
