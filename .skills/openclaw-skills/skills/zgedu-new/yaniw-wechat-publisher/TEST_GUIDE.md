# 技能包测试流程

本文档提供完整的测试流程，确保技能包功能正常后再进行打包上架。

---

## 🧪 测试环境准备

### 1. 环境检查

```bash
# 检查Python版本
python --version
# 应该显示 Python 3.7+

# 检查Playwright安装
python -c "import playwright; print('Playwright已安装')"

# 如果未安装，执行：
pip install playwright
playwright install chromium
```

### 2. 配置文件准备

```bash
# 进入技能包目录
cd /Users/zgedu/WorkBuddy/Claw/.codebuddy/skills/yaniw-wechat-publisher/

# 创建测试配置文件
cp references/config.template.json references/my_accounts.json

# 编辑配置文件，填写真实的AppID/AppSecret
# ⚠️ 注意：这是测试配置，测试完成后要删除！
```

### 3. IP白名单设置

**必须完成这一步，否则无法发布文章！**

```bash
# 查看当前IP
curl ifconfig.me

# 记下IP地址，然后在微信公众平台添加到白名单
```

## 📋 功能测试清单

### 测试1：配置加载测试

**测试目的**：验证配置文件能正确加载

**测试步骤**：

1. 在 WorkBuddy 中说："列出所有公众号"

**预期结果**：

```
📋 已配置的公众号列表：

1. 我与AI那些事
   - 状态：✅ 已启用
   - 作者：我与AI那些事
   - 累计发文：0篇

当前激活：我与AI那些事
```

**如果失败**：
- 检查 `my_accounts.json` 格式是否正确
- 检查文件路径是否正确

---

### 测试2：文章生成测试

**测试目的**：验证文章生成功能正常

**测试步骤**：

1. 在 WorkBuddy 中说："今天给公众号写一篇关于AI写作的文章"
2. 等待AI生成文章
3. 检查文章内容

**预期结果**：

```
📄 文章已生成：

标题：AI写作如何提升公众号运营效率

摘要：#AI写作工具 公众号运营 效率提升 WorkBuddy使用技巧

正文：
[文章内容...]

文末话题：AI写作 公众号运营 效率提升 自动化工具 内容创作...

⚠️ 请确认文章内容是否满意？
```

**检查项目**：

- [ ] 标题没有表情符号
- [ ] 摘要以#开头，空格结束
- [ ] 正文内容完整
- [ ] 文末有10个话题关键字
- [ ] 正文结尾没有转载声明和更新时间

### 测试6：IP白名单错误测试

**测试目的**：验证IP白名单错误提示正确

**测试步骤**：

1. 在微信公众平台暂时删除IP白名单
2. 尝试发布文章

**预期结果**：

```
❌ 发布失败：IP白名单未设置

检测到您的当前IP：202.96.128.86

请按以下步骤设置IP白名单：

1. 登录微信公众平台：https://mp.weixin.qq.com
2. 进入【开发 - 基本配置 - IP白名单】
3. 点击【修改】
4. 输入IP地址：202.96.128.86
5. 点击【确认】
6. 等待5-10分钟后重试
```

**测试完成后**：
- 记得重新添加IP白名单

---

### 测试7：错误处理测试

**测试目的**：验证各种错误情况的处理

```
⚠️ 未找到配置文件

请先创建配置文件：
1. 复制 references/config.template.json
2. 重命名为 references/my_accounts.json
3. 填写你的公众号信息

详见：CONFIGURATION.md
```
