# yaniw-wechat-publisher v1.1.0 更新说明

## 版本信息

- **版本号**: v1.1.0
- **发布日期**: 2026-03-12
- **更新类型**: 安全优化版本

## 更新内容

### 🛡️ 安全增强

#### 1. 支持环境变量配置（推荐）

**新增文件**: `.env.example`

现在推荐使用环境变量存储敏感凭证，而非配置文件：

```bash
# 复制模板文件
cp .env.example .env

# 编辑 .env 文件
WECHAT_APP_ID_1=你的AppID
WECHAT_APP_SECRET_1=你的AppSecret
```

**优势**:
- 更安全，避免配置文件误提交
- 支持多环境配置
- 符合 12-Factor App 最佳实践

#### 2. 凭证管理器

**新增文件**: `scripts/credential_manager.py`

统一凭证管理模块，支持多源凭证获取：

- 优先级 1: 环境变量（推荐）
- 优先级 2: 配置文件（传统方式）

#### 3. 错误信息脱敏

**优化文件**: `scripts/publish_to_wechat.py`

API 错误信息自动脱敏处理：

- 不再直接显示 AppID/AppSecret
- 错误信息更友好、更安全
- 降低敏感信息泄露风险

### 📦 依赖更新

新增依赖：

## 升级指南

### 对于新用户

推荐使用环境变量配置：

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 配置环境变量
cp .env.example .env
# 编辑 .env 文件，填写 AppID/AppSecret

# 3. 设置 IP 白名单
# 详见 CONFIGURATION.md
```

### 对于 v1.0.0 用户

两种方式：

**方式一：继续使用配置文件**（无需改动）

现有配置文件 `my_accounts.json` 仍然可用，无需任何修改。

**方式二：迁移到环境变量**（推荐）

```bash
# 1. 从配置文件中复制 AppID/AppSecret
cat references/my_accounts.json

# 2. 创建 .env 文件
cp .env.example .env

# 3. 填写凭证
WECHAT_APP_ID_1=你的AppID
WECHAT_APP_SECRET_1=你的AppSecret

# 4. （可选）删除配置文件
rm references/my_accounts.json
```
