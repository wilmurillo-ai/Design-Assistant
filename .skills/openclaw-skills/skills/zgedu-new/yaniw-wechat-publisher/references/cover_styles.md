# 封面图风格说明

## 5种封面风格

### 风格1：紫色科技风

**特点：**
- 渐变紫色调
- 科技感强烈
- 适合科技、AI、工具类文章

**颜色：**
- 主色：#667eea
- 辅色：#764ba2
- 渐变：linear-gradient(135deg, #667eea 0%, #764ba2 100%)

**适用场景：**
- AI工具介绍
- 科技产品评测
- 工具使用教程

---

### 风格2：蓝色科技风

**特点：**
- 深蓝色调
- 专业稳重

**颜色：**
- 主色：#1e3c72
- 辅色：#2a5298、#7e8ba3
- 渐变：linear-gradient(135deg, #1e3c72 0%, #2a5298 50%, #7e8ba3 100%)

**适用场景：**
- 技术教程

- 专业分享

## 封面图尺寸

- 宽度：900像素
- 高度：383像素
- 格式：PNG
- 文件大小：< 1MB

## 设计原则

1. **标题清晰**：文章标题要清晰可见
2. **对比度高**：文字和背景对比度要高
3. **简洁大方**：不要过于复杂
4. **主题相关**：颜色风格与文章主题相关

## 使用建议

| 公众号类型 | 推荐风格 |
|-----------|---------|
| 科技/AI类 | 风格1、风格2 |
| 生活/情感类 | 风格3、风格5 |
| 活动/新闻类 | 风格4 |
| 综合/多主题 | 根据文章内容选择 |

## 自动选择规则

如果用户没有指定风格，根据文章关键词自动选择：

- 包含"AI"、"科技"、"工具" → 风格1或风格2
- 包含"生活"、"宠物"、"情感" → 风格3或风格5
- 包含"活动"、"新闻"、"通知" → 风格4
- 默认 → 使用公众号的 default_cover_style
