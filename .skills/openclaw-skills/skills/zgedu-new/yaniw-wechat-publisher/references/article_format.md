# 文章格式规范

## 文章结构

### 1. 标题

**要求：**
- 不要使用表情符号
- 简洁明了，突出主题
- 长度建议：15-30字

**示例：**
```
✅ 正确：如何用AI提高公众号运营效率
❌ 错误：如何用AI提高公众号运营效率🎉
```

### 2. 摘要关键字

**要求：**
- 40字左右
- 以#开头，空格结束
- 包含核心关键词
- 用空格分隔

**示例：**
```
#AI写作工具 公众号运营 效率提升 WorkBuddy使用技巧 
```

### 3. 正文内容

**要求：**
- 符合公众号风格
- 段落清晰，易于阅读
- 可适当使用表情符号（正文内）
- 突出重点内容

**格式建议：**

#### 段落
- 每段3-5行
- 段落之间空一行
- 使用小标题分隔章节

#### 强调
- 重要内容使用**粗体**
- 关键词可以标注
- 适当使用列表

#### 示例：

```
今天要给大家介绍一款超实用的AI工具——WorkBuddy。

## 为什么选择WorkBuddy？

**WorkBuddy** 是一款专为公众号运营者设计的AI助手，它能帮你：

1. 自动生成文章内容
2. 设计精美的封面图
3. 一键发布到公众号

### 核心优势

- 🚀 效率提升：原来2小时的工作，现在10分钟搞定
- 💰 成本降低：无需雇佣小编，AI帮你搞定
- 📈 质量保证：专业的内容生成，提升文章质量
```

### 4. 文末话题关键字

**要求：**
- 10个相关话题
- 直接写在正文结尾
- 用空格分隔
- 不要换行

**示例：**
```
AI写作 公众号运营 效率提升 自动化工具 内容创作 微信公众号 新媒体运营 工作效率 写作工具 AI应用
```

## 完整示例

```
标题：如何用AI提高公众号运营效率

摘要：#AI写作工具 公众号运营 效率提升 WorkBuddy使用技巧 

正文：

今天要给大家介绍一款超实用的AI工具——WorkBuddy。

## 为什么选择WorkBuddy？

**WorkBuddy** 是一款专为公众号运营者设计的AI助手，它能帮你自动生成文章、设计封面、一键发布。

### 核心功能

1. **语音创作**：口述就能发文
2. **自动封面**：5种风格任你选
3. **智能发布**：一键发到公众号

### 使用体验

我自己用了3个月，每天发文时间从2小时缩短到10分钟，效率提升了10倍！

文末话题：AI写作 公众号运营 效率提升 自动化工具 内容创作 微信公众号 新媒体运营 工作效率 写作工具 AI应用
```

## 禁止内容

以下内容不要出现在文章中：

1. ❌ 正文结尾的转载声明
2. ❌ 正文结尾的更新时间
3. ❌ 标题中的表情符号
4. ❌ 过长的段落（超过8行）
5. ❌ 过多的表情符号（影响专业度）

## 质量检查清单

发布前检查：

- [ ] 标题没有表情符号
- [ ] 摘要以#开头，空格结束
- [ ] 正文段落清晰
- [ ] 文末有10个话题关键字
- [ ] 正文结尾没有转载声明
- [ ] 正文结尾没有更新时间
