# 微信公众号发布工作流程指南

## 完整工作流程

### 第一步：选择公众号

**用户口令示例：**
- "今天给'我与AI那些事'写一篇文章"
- "切换到家有野猫"

**AI响应：**
1. 识别公众号名称
2. 从 `multi_account_config.json` 加载配置
3. 显示当前公众号状态

**输出示例：**
```
✅ 已切换到"我与AI那些事"公众号
📂 工作目录：公众号-我与AI那些事/
📅 今日日期：2026-03-12
📝 今日已发布：0篇
```

### 第二步：生成文章内容

**用户口令示例：**
- "写一篇关于AI写作的文章"
- "今天要发一篇关于公众号运营的内容"

**AI响应：**
1. 根据主题生成文章
2. 文章格式要求：
   - 标题（不要表情符号）
   - 摘要关键字（40字左右，以#开头，空格结束）
   - 正文内容
   - 文末话题关键字（10个，直接写在正文结尾）
3. 保存到：`公众号-{名称}/articles/{日期}/{标题}.md`

**输出示例：**
```
📄 文章已生成：

标题：AI写作如何提升公众号运营效率

摘要：#AI写作工具 公众号运营 效率提升 WorkBuddy使用技巧

正文：
[文章内容...]

文末话题：AI写作 公众号运营 效率提升 自动化工具 内容创作

📄 文章已保存到：公众号-我与AI那些事/articles/2026-03-12/AI写作如何提升公众号运营效率.md

⚠️ 请确认文章内容是否满意？
（如需修改，请告诉我修改要求）
```

### 第三步：生成封面图

**用户确认：**
- "内容没问题"
- "文章可以了"

**AI响应：**
1. 根据文章标题和内容生成5种风格的封面图
2. 封面图尺寸：900x383像素
3. 保存HTML到：`公众号-{名称}/covers/{日期}/`

**输出示例：**
```
✅ 已生成5个封面图风格
📂 保存位置：公众号-我与AI那些事/covers/2026-03-12/

风格1：紫色科技风 - 封面图-风格1-20260312.html
风格2：蓝色科技风 - 封面图-风格2-20260312.html
风格3：粉色渐变风 - 封面图-风格3-20260312.html
风格4：橙色活力风 - 封面图-风格4-20260312.html
风格5：绿色清新风 - 封面图-风格5-20260312.html

⚠️ 请选择你喜欢的封面图风格？（输入编号1-5）
```

### 第四步：截图并发布

**用户选择：**
- "选择风格1"
- "用紫色那个"

**AI响应：**
1. 使用Playwright截图生成PNG（900x383）
2. 上传封面图到微信服务器
3. 发布文章到草稿箱
4. 记录发布日志

**输出示例：**
```
✅ 已选择风格1（紫色科技风）
📸 正在截图生成PNG...

✅ 封面图PNG已生成：封面图-风格1-20260312.png

📤 正在发布到微信公众号草稿箱...

✅ 发布成功！

📝 发布日志：公众号-我与AI那些事/articles/2026-03-12/publish_log.json
```

## 重要原则

⚠️ **严格遵循确认机制**

1. 内容生成后 → 必须等待用户确认
2. 封面生成后 → 必须等待用户选择
3. 发布之前 → 必须确认所有内容就绪

❌ **绝对不能自动跳过确认环节**

## 文章格式要求

### 标题
- 不要使用表情符号
- 简洁明了，突出主题
- 长度建议：15-30字

### 摘要关键字
- 40字左右
- 以#开头，空格结束
- 包含核心关键词

**示例：**
```
#AI写作工具 公众号运营 效率提升 WorkBuddy使用技巧 
```

### 正文内容
- 符合公众号风格
- 段落清晰，易于阅读
- 可适当使用表情符号（正文内）

### 文末话题关键字
- 10个相关话题
- 直接写在正文结尾
- 用空格分隔

**示例：**
```
AI写作 公众号运营 效率提升 自动化工具 内容创作 微信公众号 新媒体运营 工作效率 写作工具 AI应用
```

### 注意事项
- 正文结尾不要转载声明
- 正文结尾不要更新时间
