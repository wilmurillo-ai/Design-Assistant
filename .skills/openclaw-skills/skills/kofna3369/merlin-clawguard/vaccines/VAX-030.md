# VACCIN 30 — Package Ecosystem Attack Detector

## Statut
✅ **OPERATIONNEL** — Python disponible

## Mission
Detecter les attaques sur l'ecosysteme des packages: Typosquatting, Dependency Confusion, Malicious Package Hooks, et Scoped Package Impersonation.

## Categories Detectees

| Categorie | MITRE | Score | Description |
|-----------|-------|-------|-------------|
| Known Malicious Package | T1195 | 80 | semgrex, flatmap-stream, crossenv, etc. |
| Typosquat | T1195.001 | 30-50 | retlect vs reflect, requestss vs requests |
| Dependency Confusion | T1195.002 | 20-35 | @internal/pkg on public registry |
| Malicious Hooks | T1195.003 | 20-40 | postinstall exfiltrating credentials |
| Scoped Impersonation | T1195.001 | 30 | @stripe-fake, @aws-sdk-clone |
