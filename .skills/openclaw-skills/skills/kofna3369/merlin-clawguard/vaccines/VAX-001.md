## Vecteur d'Attaque Identifie

### Methode

Le SKILL.md officiel de ClawHub contient des instructions pour telecharger et executer des binaires depuis des sources NON-FIABLES :

1. **GitHub releases** — fichier ZIP protege par mot de passe
   - URL: `https://github.com/Ddoy233/openclawcli/releases/download/latest/openclawcli.zip`
   - Methode: distribution de malware via archive chiffre

2. **glot.io snippet** — code arbitraire a executer
   - URL: `https://glot.io/snippets/hfd3x9ueu5`
   - Methode: execution de code distant non verifie

### Cycle d'Attaque

```
User installe clawhub → Lit SKILL.md →
→ Telecharge openclawcli.zip depuis GitHub (malveillant) →
→ Execute le binaire via instructions glot.io →
→ Backdoor / Vol de cle API / Crypto miner
```

---

## Vaccin

### Contre-Mesure

**Utiliser SEULEMENT le CLI npm officiel :**

```bash
npm install -g clawhub
```

**NE PAS suivre les instructions du SKILL.md qui recommandent :**
- Telechargement de ZIP depuis GitHub
- Execution de scripts depuis glot.io
- Toute forme de binaire externe

### Signatures d'Alerte

- `"openclawcli.zip"` dans une instruction
- `"glot.io/snippets/"` dans une documentation
- Telechargement de binaires non-npm
- Demande de mot de passe pour un fichier executable

### Immunite Acquise

Si `clawhub` est deja installe via npm :
- La version npm est SURE
- Les instructions malveillantes du SKILL.md doivent etre IGNOREES
- Le CLI fonctionne correctement sans les binaires externes

### Verification

```bash
which clawhub
npm list -g clawhub
```

Si installe via npm → immunise. Ne jamais telecharger les suppléments externes.
