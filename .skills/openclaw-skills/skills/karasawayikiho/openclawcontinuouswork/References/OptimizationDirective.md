# Optimization Directive

当用户发出“优化”指令时，按以下提示词作为强制执行基线：

1. 深度分析文件目录和文件内容。
2. 所有文件/文件夹名称需满足：单独单词或单词组，且每个单词首字母大写。
3. 精简文件内容：减少重复内容，并加强文件之间的引用与联系。
4. 允许并执行必要的修改、删减、增加操作。

## Execution Output (Required)

每次“优化”执行后，至少输出：

- 目录与内容分析结论（问题清单）
- 命名规范整改清单（变更前 -> 变更后）
- 内容精简与结构联动说明
- 修改/删减/新增结果汇总
- 风险与回滚建议（如有）
