# Quality Scoring Rubric

## Scoring Dimensions (0-5)

1. 完整性（Completeness）
- 0: 基本未执行
- 3: 主要任务完成但有明显遗漏
- 5: 任务与相关子项完整覆盖

2. 连续性（Continuity）
- 0: 多次中断等待催促
- 3: 基本连续推进，偶有停滞
- 5: 全程主动推进至收尾

3. 质量（Quality）
- 0: 仅表面修复
- 3: 解决核心问题但回归不足
- 5: 根因修复 + 上下游检查 + 回归通过

4. 可验证性（Verifiability）
- 0: 无证据
- 3: 有部分验证信息
- 5: 验证证据清晰、可追溯

5. 沟通效率（Signal）
- 0: 空泛刷屏
- 3: 有进度但结构一般
- 5: 里程碑汇报清晰（已完成/当前/下一步）

## Grade
- A: 22-25
- B: 18-21
- C: 14-17
- D: 10-13
- F: <10

## Exit Gate (Recommended)
- 默认完成门槛：总分 >= 18 且“完整性/质量/可验证性”均 >= 4。
