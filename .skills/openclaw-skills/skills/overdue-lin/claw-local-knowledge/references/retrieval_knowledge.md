# 知识库检索指南

本文件指导 AI 代理如何从本地知识库中检索相关知识。

## 检索流程

### 第一步：读取索引文件

当代理需要额外知识来辅助用户时，首先读取以下文件：

```
.openclaw/workspace/memory/knowledge_base_index.json
```

该索引文件包含所有知识文件的列表，每个条目包含文件名和描述。

### 第二步：搜索相关文件

根据当前话题或问题，搜索索引文件中的描述，找出与该话题相关的文件。

### 第三步：读取具体知识文件

根据搜索结果，从以下文件夹中读取对应的 markdown 文件：

```
.openclaw/workspace/memory/knowledge_base/
```

### 第四步：整合信息

将读取到的内容整合到回复中，以增强代理的响应质量。

## 示例流程

1. 读取 `knowledge_base_index.json`
2. 搜索包含关键词"LLM推理优化"的文件
3. 找到匹配的文件（如 `inference_optimization.md`）
4. 读取该文件内容
5. 使用获取的知识来回答用户问题

## 注意事项

- 索引文件是 JSON 格式，解析时注意结构
- 知识文件均存储在 `knowledge_base` 文件夹下
- 优先检索与当前问题最相关的文件
