# Case Study — OpenClaw Book Launch

## Project Goal
Build a complete English-facing launch system for the OpenClaw book, starting from the book cover and a strong positioning line.

## Core Positioning
**OpenClaw: Build Your Own AI Agent That Actually Gets Things Done**

Supporting hooks:
- Most AI can talk. Very few can execute.
- Stop building demos. Start building execution.
- From AI chat to AI action.

## What Was Produced

### Static Assets
- vertical poster
- square poster
- horizontal poster

### Video Assets
- 30-second English promo video
- 15-second hype version
- 15-second Edge-TTS improved version
- 4-style montage campaign video
- 2-style polished ad version
- sales conversion ad version

### Copy Packs
- launch copy pack
- Amazon / Kindle conversion pack
- Amazon ready-to-publish pack

### Concept Systems
- people-holding-book style pack
- mockups for 4 style directions

## Key Creative Decision
A single creative direction was not enough.
The project became much stronger after splitting the outputs into:
- premium / brand lane
- conversion / sales lane

This allowed one set of assets to attract attention and another set to explain why the book should be read now.

## Voiceover Decision
A local lightweight TTS solution sounded too robotic.
The workflow improved significantly after switching to Edge neural TTS for the English promo videos.

## Strongest Visual Directions
The four people-holding-book concept directions were:
1. Tech Founder Style
2. International Star Style
3. Business Magazine Style
4. Movie Poster Style

The strongest pair for ad-style videos turned out to be:
- International Star Style
- Movie Poster Style

## Why They Worked
- International Star Style created premium social attention
- Movie Poster Style created stronger emotion and trailer energy
- together they felt more like a real campaign than a generic book promo

## Packaging Lesson
The work became much easier to reuse once everything was packaged into clear folders:
- launch assets
- campaign videos
- Amazon copy packs
- final launch pack documentation

## Reusable Takeaways
1. Start with positioning, not design.
2. Build motion assets early.
3. Improve voice quality fast.
4. Separate brand and conversion outputs.
5. Turn strong visual directions into short campaign videos.
6. Package the process so future launches are faster.

## Recommended Reuse Pattern
If you want to repeat this workflow for another book:
1. replace the cover and title
2. rewrite the positioning line
3. regenerate posters
4. regenerate promo videos
5. regenerate Amazon copy
6. regenerate people-holding-book concepts
7. polish one brand ad and one sales ad
8. package everything into a launch pack
