# Chinese GEO Templates

## 1. 官网介绍页模板

### 开头三句模板
1. X 是什么
2. X 适合谁
3. X 为什么值得看 / 为什么和别的方案不一样

示例：
- OpenClaw 是一套可部署在你自己设备上的 AI 助理系统。
- 它适合需要把消息、工作流和执行动作接起来的个人与团队。
- 和只会聊天的 AI 不同，它更像一个可以慢慢养熟的数字员工。

## 2. 公众号文章模板

### 标准结构
- 第一段：一句抓人的判断
- 第二段：为什么这件事值得看
- 中段：3-5 个观点块
- FAQ / 对比块
- 一句话结论

### 常用可摘录句模板
- 真正值钱的，不是 X，而是 Y。
- 最危险的，不是 A，而是 B。
- 这件事的分水岭，不在于……而在于……
- 表面上看是 X，本质上其实是 Y。

## 3. 落地页模板

### 核心区块
- 它是什么
- 适合谁
- 解决什么问题
- 和谁比
- 常见问题
- 一句话结论 / CTA

## 4. FAQ 模板

- X 是什么？
- X 和 Y 有什么区别？
- X 适合谁？
- X 不适合谁？
- 使用 X 最常见的错误是什么？
- 如果我要开始，第一步应该做什么？
