# Rewrite Playbook

## Article rewrites

When rewriting an article for GEO:
1. Replace long intros with direct setup
2. Add a definition block in the first screenful
3. Add query-matching H2s
4. Add at least one comparison table or list
5. Add an FAQ near the end
6. Add a short summary / verdict section

## Landing page rewrites

Landing pages often fail GEO because they sound branded but say little.

Fixes:
- Say exactly what the product is in one line
- Say who it is for in one line
- Say what job it helps complete in one line
- Add “How it works” and “Use cases” sections
- Add “X vs alternatives” section if relevant

## Docs / help center rewrites

Docs can perform well in answer engines if they are explicit.

Fixes:
- Add short answers before detailed steps
- Use clean headings matching support queries
- Add troubleshooting blocks
- Add examples with exact terms users search for

## Good sentence shapes

Use sentence shapes like:
- “X is a …”
- “X helps [role] do [job] by [mechanism].”
- “X is best for …”
- “Compared with Y, X is stronger at … but weaker at …”
- “The most common mistake is …”
