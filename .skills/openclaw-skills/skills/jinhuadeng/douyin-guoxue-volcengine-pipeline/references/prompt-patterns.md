# Prompt Patterns / 提示词模式

## EN

Use a consistent 3-shot structure for guoxue short videos.

### Shot 1 — atmosphere / opening world
- establish heaven-earth relation
- broad landscape
- moving clouds / river / mist
- restrained color palette

### Shot 2 — human meaning / character
- scholar / gentleman / wise woman / bridge / mountain path
- robe motion, grass motion, controlled camera move
- emotional center of the clip

### Shot 3 — resolution / opening / ascent
- light opening up
- wider spatial release
- symbolic upward momentum or grounded stillness

### Style anchors
- Song-dynasty aesthetics
- eastern negative space
- black / ink-cyan / warm gold or earth tones
- cinematic lighting
- realistic + ink-wash fusion
- no cartoon, no anime, no neon cyberpunk

## 中文

国学短视频建议固定 3 镜头结构。

### 镜头1：起势
- 先立天地气象
- 山川、云海、河流、晨雾
- 负责把观众拉进意境

### 镜头2：立人
- 人物 / 桥 / 山路 / 草木 / 舟
- 把卦象变成“人该怎么做”的画面隐喻
- 这是情绪中心镜头

### 镜头3：收束
- 天光打开 / 空间展开 / 大地承载 / 气机流转
- 负责把主题落到一句金句上

## Example theme angles

### 乾卦
- upward force
- discipline
- self-strengthening
- rising sky / sun / mountain summit

### 坤卦
- grounding
- bearing / containing
- thick virtue
- land / river / field / soft wind

### 泰卦
- circulation
- openness
- things beginning to move smoothly
- heaven-earth exchange / bridge / open river / brightened sky
