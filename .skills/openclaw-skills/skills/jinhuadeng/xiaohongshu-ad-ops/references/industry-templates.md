# Industry Templates

## 1. AI 安装 / 培训 / 顾问服务

### 核心卖点
- 不是讲概念，而是帮你装好、跑起来、接业务
- 适合老板 / 团队 / 销售 / 内容运营

### 最适合角度
- 反常识：真正值钱的不是模型，而是工作流
- 结果型：装完之后哪个环节最先省事
- 案例型：某客户/某团队怎么接住了业务

## 2. 本地商家

### 核心卖点
- 到店 / 留资 / 私信咨询
- 结果感和信任感比功能感更重要

### 最适合角度
- 前后对比
- 到店结果
- 用户真实体验

## 3. 教育培训

### 核心卖点
- 学完能干什么
- 多久能上手
- 是否适合小白

### 最适合角度
- 筛选型
- 案例型
- 常见误区型

## 4. 高客单顾问 / 服务型产品

### 核心卖点
- 信任比点击重要
- 先筛选，再成交

### 最适合角度
- 适合谁 / 不适合谁
- 常见误区
- 为什么很多人一开始就做错了
