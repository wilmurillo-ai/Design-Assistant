# Note Templates

## 1. 痛点型

### 标题模板
- 做了很久 X，最崩溃的不是 Y，而是 Z
- 如果你也在做 X，这个坑大概率你也踩过
- 很多人以为问题在 A，其实真正卡住的是 B

### 正文结构
1. 直接抛痛点场景
2. 说清旧做法为什么不行
3. 给出新解法
4. 用一个结果/案例增强信任
5. 低门槛 CTA

## 2. 结果型

### 标题模板
- 我把 X 从 A 做到 B，最关键的不是努力，而是……
- 7 天后最明显的变化，不是 C，而是 D
- 做完这一步，X 的转化终于开始上来了

## 3. 对比型

### 标题模板
- 自己做 vs 找专业团队，差别到底在哪？
- 传统做法和 AI 做法，真正的差别不是快，而是……
- 便宜方案和高客单方案，真正差在哪？

## 4. 筛选型

### 标题模板
- 这件事适合谁，不适合谁，我直接说清楚
- 如果你现在处于这 3 种情况，再来了解 X
- 不是所有人都适合做 X，这篇先帮你筛一下

## CTA 模板
- 想看完整案例，先私信我
- 评论区留“想看”，我发你方案
- 如果你想知道适不适合自己，先来问我
