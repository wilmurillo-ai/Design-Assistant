# Actions

## morning_question

Builds message as:

```text
今天抛个问题：

<question text>
```

## afternoon_followup

Builds message as:

```text
下午接着聊一句：

<followup text>
```

## evening_case

Builds message as:

```text
今晚丢个案例给大家。

【<title>】
<case text>
```

## Dry run

Use `--dry-run` to preview what would be posted without sending.
