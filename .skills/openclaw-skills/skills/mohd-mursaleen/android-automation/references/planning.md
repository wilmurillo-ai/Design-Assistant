# Planning Reference

Use this before launching automation to decide what details to gather from the user.

## What To Ask

Only ask for details that materially affect the flow. Do not ask for things the user already provided.

## Domain Checklists

### Movie Tickets

Usually ask for:

- city or cinema area
- date
- showtime or time window
- number of tickets

Ask only if relevant:

- language
- format such as IMAX or 3D
- seat preference or budget

### Ride Booking

Usually ask for:

- pickup location
- destination
- now vs scheduled

Ask only if relevant:

- ride type
- payment method

### Food or Grocery

Usually ask for:

- exact items
- quantity
- delivery address if not obvious

Ask only if relevant:

- substitutions
- budget
- payment rule

### Shopping

Usually ask for:

- exact product
- size, color, or variant
- quantity

Ask only if relevant:

- shipping address
- payment method
