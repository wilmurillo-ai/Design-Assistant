# 超短线炒股 Skill

> 新手教学 + 选股策略 + 实战技巧

---

## 一、什么是超短线？

**超短线**指持股周期在 1-3 天内的交易模式，甚至日内 T+0。
追求的是**资金利用率**和**快速获利**。

### 特点

- 持股时间短
- 资金周转快
- 收益波动大
- 需要盯盘

---

## 二、核心术语

| 术语 | 含义 |
|------|------|
| **打板** | 在股票涨停瞬间买入 |
| **首板** | 股票的第一个涨停 |
| **连板** | 连续涨停 |
| **龙头** | 板块涨幅第一、带动板块的股票 |
| **跟风** | 跟随龙头上涨的股票 |
| **炸板** | 涨停被打开 |
| **核按钮** | 跌停价挂单卖出 |
| **反包** | 前一天炸板/跌停，第二天涨停 |
| **量能** | 成交量大小 |
| **换手** | 成交率 |

## 三、常用技术指标

### 1. MACD

- **金叉**：DIF 上穿 DEA → 买入信号
- **死叉**：DIF 下穿 DEA → 卖出信号
- **0 轴**：0 轴上方多头，下方空头

### 2. KDJ

- **J 值 > 80**：超买，可能回调
- **J 值 < 20**：超卖，可能反弹
- **金叉买入**：K 上穿 D

### 3. 成交量

- 放量上涨：资金入场，看多
- 缩量下跌：惜售，可能见底
- 天量：可能见顶

### 4. 均线

- 5 日均线：短期生命线
- 10 日均线：中期趋势
- 突破均线：看多
- 跌破均线：看空

## 四、选股策略

### 策略一：首板打板

**核心逻辑**：做板块第一个涨停的股票

**选股条件**：

1. 板块内涨幅第一
2. 首次涨停（非二板三板）
3. 成交量放大（量能充足）
4. 封板时间早（越早越强）
5. 盘口买单大（涨停板牢固）

**仓位**：单只不超过 10% 总资金

### 策略二：龙头战法

**识别龙头股**：

1. **超强人气**：市场关注度高
2. **主动性**：自己涨，带动板块
3. **带动性**：板块内其他股票跟随
4. **涨幅领先**：板块涨幅第一
5. **换手充分**：充分换手后涨停更稳

**龙头特征**：

- 换手 5%-15%
- 连续涨停 3 板以上
- 大资金持续买入
- 板块内带动性强

### 策略三：反包战法

**适用场景**：前一天炸板或跌停，第二天低开高走涨停

**条件**：

1. 前一天套牢盘多
2. 当天低开高走
3. 成交量放大
4. 题材有延续性

### 策略四：尾盘选股

**优点**：规避日内风险，第二天可走

**选股方法**：

1. 涨幅 3%-7%
2. 成交量放大
3. 均线多头排列
4. 板块效应

## 五、情绪周期

### 周期四阶段

1. **启动期**：冰点转暖，少量涨停
2. **高潮期**：批量涨停，情绪亢奋
3. **退潮期**：炸板增多，亏钱效应
4. **冰点期**：涨停极少，情绪低迷

### 实战应用

- **启动期/高潮期**：积极做多
- **退潮期/冰点期**：控仓或休息

## 六、仓位管理

### 铁律

| 规则 | 说明 |
|------|------|
| 单票不超过 10% | 分散风险 |
| 单日累计不超过 30% | 防止过度交易 |
| 止损线 -7% | 到了必须割 |
| 止盈线 +15% | 落袋为安 |

### 仓位公式

```
仓位 = 账户总资金 × 风险系数 / 单股最大亏损
```

## 七、止盈止损

### 止损（必须执行）

- **固定止损**：-7% 必须割
- **破位止损**：跌破关键均线走人
- **题材走弱**：板块不行了果断走

### 止盈

- **分批止盈**：+7% 卖一半，+15% 再卖一半
- **尾盘不板走**：涨停没封住必须走
- **情绪高潮走**：别人贪婪我恐惧

## 八、新手避坑指南

### ❌ 常见错误

1. **追高**：涨幅 8% 了还追
2. **YY**（意淫）：觉得自己买的是龙头
3. **不止损**：套住了就装死
4. **满仓干**：一把梭哈
5. **频繁交易**：每天都在买卖

### ✅ 正确心态

1. **只做确定性**：没有把握就空仓
2. **错了就认**：止损要果断
3. **让利润奔跑**：不要涨一点就卖
4. **尊重市场**：市场永远是对的
5. **复盘总结**：每天写交易笔记

## 常见问题

### Q1: 新手适合做超短线吗？

不建议。新手建议先从波段/中线开始，熟悉市场后再尝试超短线。

### Q2: 没有消息怎么选股？

看涨停板。涨停板是资金投票的结果，代表市场认可。

### Q3: 打板什么时候卖？

- 第二天不涨停就走
- 炸板立即走
- +15% 可考虑分批卖

### Q4: 怎么判断龙头？

看涨幅、换手、带动性。龙头是市场选出来的，不是YY出来的。

### Q5: 天天亏钱还要继续吗？

停下来复盘。亏钱说明方法有问题，先学习再交易。
