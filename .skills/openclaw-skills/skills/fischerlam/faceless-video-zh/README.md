# faceless-video-zh

[![ClawHub Skill](https://img.shields.io/badge/ClawHub-Skill-blueviolet)](https://clawhub.io)
[![Version](https://img.shields.io/badge/version-1.0.12-blue)](SKILL.md)

> **把内容做成不露脸视频，更适合无真人出镜的讲解和自动化内容形式。**

## 适合什么场景
适合不露脸内容、自动化内容、旁白型讲解视频和 faceless 表达。

## 一条命令直接开始

```bash
sparki run --file ./video.mp4 --mode prompt-driven --prompt "做成一个不露脸的讲解型视频" --aspect-ratio 9:16
```

## 推荐提示词
- 做成一个不露脸的讲解型视频。
- 把它变成更像 faceless explainer 的形式。
- 做成 automation-style 的内容。

## 相关技能
- `AI 解说`
- `AI 字幕`
- `TikTok 爆款剪辑`
