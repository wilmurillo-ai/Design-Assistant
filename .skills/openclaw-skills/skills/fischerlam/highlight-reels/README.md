[![ClawHub Skill](https://img.shields.io/badge/ClawHub-Skill-blueviolet)](https://clawhub.io)
[![Version](https://img.shields.io/badge/version-1.0.12-blue)](SKILL.md)

> **Extract the best moments and turn them into a fast, punchy highlight reel.**

## Best for
Sports, events, speeches, recaps, and any footage where only the strongest moments should remain.

## Quick start

```bash
sparki run --file ./video.mp4 --mode prompt-driven --prompt "extract the best moments and turn it into a highlight reel" --aspect-ratio 9:16
```

## Prompt templates
- Extract the best moments and turn it into a highlight reel.
- Pull out the strongest moments and make the recap punchy.
- Cut the highlights from this footage and keep only the most exciting clips.
