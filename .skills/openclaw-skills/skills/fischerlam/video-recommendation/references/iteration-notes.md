# Iteration Notes

Use this file to record what the skill gets wrong in real usage.

## Things worth logging

- user request
- what was recommended
- what worked
- what felt off
- whether the recommendations were too obvious, too niche, too shallow, or too random
- any source-quality issue

## Common refinement directions

### If recommendations are too generic
- tighten context extraction
- increase weight on recent chat
- prune obvious mainstream defaults

### If recommendations are too random
- reduce recommendation angles from 4 to 2-3
- improve thematic coherence

### If recommendations are too verbose
- shorten rationale
- switch to title + link format more often

### If recommendations are too dry
- add delight as a stronger ranking factor
- allow more surprising picks when relevance remains high

### If recommendations feel like search results
- increase pruning
- require a reason for every recommended video unless links-only mode is requested
