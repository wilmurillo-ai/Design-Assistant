# Personalization

## Build a recommendation profile from context

Look for:
- current project or obsession
- recurring themes in recent chat
- language preference
- preferred density: light, deep, tactical, philosophical
- desired emotional effect: fun, awe, stimulation, clarity, inspiration

## Example signals

### Project-linked
If the user is building a product, bias toward:
- founder interviews
- product breakdowns
- technical showcases
- adjacent market insight

### Mission-linked
If the user often talks about AI, humanity, the future, or meaning, bias toward:
- long-form AI conversations
- deep explainers
- documentaries about systems change

### Creative-linked
If the user is working on video, media, or storytelling, bias toward:
- AI short films
- filmmaking breakdowns
- creator workflows
- standout ad or music video experiments

## Practical rule

Do not recommend based on a generic demographic stereotype.
Recommend based on live context and accumulated taste.
