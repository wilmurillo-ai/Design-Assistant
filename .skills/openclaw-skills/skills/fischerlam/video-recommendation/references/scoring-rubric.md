# Scoring Rubric

Use this rubric to rank candidate videos before recommending them.

## Score each candidate from 1-5

### 1. Relevance
How tightly does this match the user's recent chat, current project, or active curiosity?

### 2. Signal density
How much real value is there per minute?
Down-rank fluff, repetition, and empty hype.

### 3. Delight / aliveness
Is this likely to feel alive, surprising, or energizing?
Not every video must be fun, but dead videos lose.

### 4. Taste fit
Does this feel like the kind of recommendation that earns trust?
Would a sharp user think: "yes, this was chosen, not sprayed"?

### 5. Actionability
Does it serve the user's current purpose?
Examples:
- immediate watch
- inspiration for a product
- understanding a topic
- getting unstuck creatively

## Ranking guidance

- Default threshold: recommend only candidates averaging 4+
- One exceptional 5 in relevance can outweigh moderate novelty
- Do not include filler just to reach a round number
- 5 strong picks beat 10 diluted ones
