# Output Patterns

## 1. Links only

Use when the user says things like:
- just give me links
- no explanation
- send 10 videos

Format:
- Title
- Link

## 2. Short curated list

Format:
- Title
- Link
- Why this fits

Keep rationale to one line.

## 3. Grouped recommendations

Format:
- Theme heading
- 2-4 videos per theme
- 1 line per video

Useful when the user's interest has multiple edges.

## 4. Top 3 now

Format:
- Ranked 1-3
- One sharp reason each

Use when the user wants something immediate.

## Tone

- direct
- tasteful
- low filler
- not algorithmic
- not over-explaining
