[![ClawHub Skill](https://img.shields.io/badge/ClawHub-Skill-blueviolet)](https://clawhub.io)
[![Version](https://img.shields.io/badge/version-1.0.12-blue)](SKILL.md)

> **Turn ideas or footage into faceless-style videos without needing a person on camera.**

## Best for
No-on-camera content, automation-style explainers, faceless educational videos, and narration-led outputs.

## Quick start

```bash
sparki run --file ./video.mp4 --mode prompt-driven --prompt "make this into a faceless explainer" --aspect-ratio 9:16
```

## Prompt templates
- Make this into a faceless explainer.
- Turn this into no-on-camera content with a clear narrated structure.
- Create a faceless automation-style video from this material.
