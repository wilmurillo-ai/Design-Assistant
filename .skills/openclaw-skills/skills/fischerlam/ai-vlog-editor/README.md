> **Turn raw footage into a clean, creator-style vlog with better pacing and stronger story flow.**

## Best for
Daily-life videos, travel footage, creator-style edits, and lifestyle vlogs.

## Quick start

```bash
sparki run --file ./video.mp4 --mode prompt-driven --prompt "edit this into a chill daily vlog" --aspect-ratio 9:16
```

## Prompt templates
- Edit this into a chill daily vlog.
- Turn this travel footage into a creator-style vlog with clean pacing.
- Make this feel like an upbeat YouTube vlog with smoother flow.
