# ai-vlog-editor-zh

[![ClawHub Skill](https://img.shields.io/badge/ClawHub-Skill-blueviolet)](https://clawhub.io)
[![Version](https://img.shields.io/badge/version-1.0.12-blue)](SKILL.md)

> **把原始素材剪成更有节奏、更像创作者成片的 Vlog。**

## 适合什么场景
适合日常记录、旅行素材、生活方式内容和创作者风格 Vlog。

## 一条命令直接开始

```bash
sparki run --file ./video.mp4 --mode prompt-driven --prompt "把这段素材剪成一个松弛一点的日常 vlog" --aspect-ratio 9:16
```

## 推荐提示词
- 把这段素材剪成一个松弛一点的日常 vlog。
- 把这组旅行素材做成更有创作者感的 vlog。
- 让这条视频更像 YouTube 风格的生活记录。

## 相关技能
- `长视频转短视频`
- `AI 字幕`
- `视频尺寸调整`
