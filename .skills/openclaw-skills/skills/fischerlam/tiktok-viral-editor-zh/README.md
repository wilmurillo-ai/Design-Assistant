[![Version](https://img.shields.io/badge/version-1.0.12-blue)](SKILL.md)

> **把素材剪成更像 TikTok 的短视频：开头更强、节奏更快、注意力更集中。**

## 适合什么场景
适合想要更强 hook、更快节奏、更有“爆款感”的短视频内容。

## 一条命令直接开始

```bash
sparki run --file ./video.mp4 --mode prompt-driven --prompt "让这条视频更像 TikTok，开头更强，节奏更快" --aspect-ratio 9:16
```

## 推荐提示词
- 让这条视频更像 TikTok，开头更强，节奏更快。
- 把这条视频剪得更有爆款感，更抓人。
- 做成更像 TikTok 原生内容的短视频。

## 相关技能
- `长视频转短视频`
- `AI 字幕`
- `视频尺寸调整`
