# long-to-short

[![ClawHub Skill](https://img.shields.io/badge/ClawHub-Skill-blueviolet)](https://clawhub.io)
[![Version](https://img.shields.io/badge/version-1.0.12-blue)](SKILL.md)

> **Turn long videos into short clips, reels, and shorts with stronger hooks and cleaner cutdowns.**

## Best for
Podcasts, interviews, talks, streams, and long-form footage that need to become short-form clips.

## Quick start

```bash
sparki run --file ./video.mp4 --mode prompt-driven --prompt "turn this long video into short clips with strong hooks" --aspect-ratio 9:16
```

## Prompt templates
- Turn this long video into short clips with strong hooks.
- Cut this podcast into short-form clips that start with the strongest moments.
- Extract the best short-form moments from this interview and make them more watchable.
