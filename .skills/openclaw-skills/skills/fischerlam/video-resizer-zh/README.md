> **把同一条视频快速适配成 9:16、1:1、16:9 等不同平台比例。**

## 适合什么场景
适合 Reels、Shorts、TikTok、Instagram、YouTube 等平台适配和多版本分发。

## 一条命令直接开始

```bash
sparki run --file ./video.mp4 --mode prompt-driven --prompt "改成适合竖屏观看，并尽量保持主体居中" --aspect-ratio 9:16
```

## 推荐提示词
- 改成适合竖屏观看，并尽量保持主体居中。
- 把这条视频适配成 Reels / Shorts 的比例。
- 改成 Instagram 方屏版本。
