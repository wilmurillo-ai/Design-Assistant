# 哪个 Sparki Skill 适合我？

如果你不确定该装哪个 Sparki skill，就先看这个。

## 从这里开始
- 想先装一个总入口？→ `AI Video Editor`
- 想剪 vlog？→ `ai-vlog-editor`
- 想把长视频切成短视频？→ `long-to-short`
- 想做字幕优先的视频？→ `ai-caption`
- 想做解说型内容？→ `ai-commentary`
- 想改视频比例适配平台？→ `video-resizer`
- 想提炼高光？→ `highlight-reels`
- 想做不露脸内容？→ `faceless-video`
- 想做更像 TikTok 的短视频？→ `tiktok-viral-editor`
- 想剪口播 / talking-head 视频？→ `talking-head-editor`

## 说明
这是一个导航型 skill，用来帮用户先选对 skill，再开始剪视频。
