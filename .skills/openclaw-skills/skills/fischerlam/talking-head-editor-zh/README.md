# talking-head-editor-zh

[![ClawHub Skill](https://img.shields.io/badge/ClawHub-Skill-blueviolet)](https://clawhub.io)
[![Version](https://img.shields.io/badge/version-1.0.12-blue)](SKILL.md)

> **把口播和镜头前讲话的内容剪得更清楚、更紧凑、更好看。**

## 适合什么场景
适合口播、教程、采访、镜头前讲话、presenter 视频和讲解类内容。

## 一条命令直接开始

```bash
sparki run --file ./video.mp4 --mode prompt-driven --prompt "把这段口播剪得更紧一点，让表达更清楚" --aspect-ratio 9:16
```

## 推荐提示词
- 把这段口播剪得更紧一点，让表达更清楚。
- 让这个 presenter 视频更干净、更利落。
- 把这段采访做成更好的讲解视频。

## 相关技能
- `AI 解说`
- `AI 字幕`
- `视频尺寸调整`
