# long-to-short-zh

[![ClawHub Skill](https://img.shields.io/badge/ClawHub-Skill-blueviolet)](https://clawhub.io)
[![Version](https://img.shields.io/badge/version-1.0.12-blue)](SKILL.md)

> **把长视频快速拆成短视频片段，提炼更适合传播的重点内容。**

## 适合什么场景
适合播客、采访、直播、演讲和长内容切条分发。

## 一条命令直接开始

```bash
sparki run --file ./video.mp4 --mode prompt-driven --prompt "把长视频切成几个有强 hook 的短视频" --aspect-ratio 9:16
```

## 推荐提示词
- 把长视频切成几个有强 hook 的短视频。
- 从这段播客里提炼几个最适合传播的片段。
- 把这条长访谈剪成适合 Shorts 的短视频。
