> **Turn footage into a faster, hook-driven, TikTok-native cut with stronger retention and a more viral feel.**

## Best for
Short-form videos that need a stronger hook, faster pacing, and a more attention-grabbing TikTok feel.

## Quick start

```bash
sparki run --file ./video.mp4 --mode prompt-driven --prompt "make this feel more TikTok-native with a stronger hook and faster pacing" --aspect-ratio 9:16
```

## Prompt templates
- Make this feel more TikTok-native with a stronger hook and faster pacing.
- Edit this for TikTok with a punchier opening and tighter cuts.
- Turn this into a more viral short with stronger attention-grabbing beats.
