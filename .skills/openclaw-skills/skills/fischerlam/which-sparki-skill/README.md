# Which Sparki Skill Should I Use?

Use this guide when you are not sure which Sparki skill best fits your workflow.

## Start here
- Want one broad default entry point? → `AI Video Editor`
- Want vlogs? → `ai-vlog-editor`
- Want long videos turned into short clips? → `long-to-short`
- Want caption-first output? → `ai-caption`
- Want commentary-style edits? → `ai-commentary`
- Want platform resizing? → `video-resizer`
- Want highlight reels? → `highlight-reels`
- Want faceless content? → `faceless-video`
- Want TikTok-style viral edits? → `tiktok-viral-editor`
- Want talking-head cleanup? → `talking-head-editor`

## Notes
This is a routing / discovery skill. It helps the user choose the right Sparki skill before they start editing.
