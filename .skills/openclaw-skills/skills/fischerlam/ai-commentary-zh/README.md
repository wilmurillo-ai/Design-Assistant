> **把素材做得更像解说视频：结构更清楚，叙述感更强，观点表达更完整。**

## 适合什么场景
适合讲解视频、解说内容、reaction 风格内容和需要更强叙述性的表达。

## 一条命令直接开始

```bash
sparki run --file ./video.mp4 --mode prompt-driven --prompt "把它做成解说风格，结构更清晰" --aspect-ratio 9:16
```

## 推荐提示词
- 把它做成解说风格，结构更清晰。
- 让这条视频更有讲解感和旁白感。
- 把这条内容做成更像 commentary 的版本。

## 相关技能
- `AI 字幕`
- `口播视频剪辑`
- `TikTok 爆款剪辑`
