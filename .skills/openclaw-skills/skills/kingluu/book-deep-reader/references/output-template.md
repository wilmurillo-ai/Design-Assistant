## 使用指南

### 什么时候用什么部分

| 读者需求 | 重点部分 |
|---------|---------|
| 快速了解全书 | 第〇、一章 |
| 深入学习某章 | 第二章对应章节 |
| 理解核心理念 | 第三章 |
| 拿来就用 | 第四章 |
| 要给别人讲 | 第五章 |

### 长度参考

- 概览型笔记：3000-5000字
- 标准深度笔记：8000-15000字
- 可用于教学的完整笔记：15000-25000字
- **包含批判性评估的完整笔记：20000-30000字**

根据用户需求和书籍复杂度调整。

### 新增批判性评估部分使用说明

**与原有功能的区别：**

| 维度 | 原有功能（教学导向） | 新增功能（批判+行动导向） |
|------|---------------------|---------------------------|
| **视角** | 如何理解和传授这本书 | 如何评价这本书的价值并应用 |
| **深度** | 书籍内部逻辑分析 | 书籍在学术脉络中的定位 |
| **实用性** | 知识理解与教学 | 行动指导与跨界应用 |
| **完整性** | 书籍内容提取 | 书籍与外部知识的关联 |

**何时使用批判性评估部分：**

| 读者需求 | 重点章节 |
|----------|----------|
| 快速判断这本书是否值得读 | 6.1-6.3 |
| 了解学术争议和局限性 | 6.5 |
| 寻找跨界应用灵感 | 6.7 |
| 制定读书后的行动计划 | 6.9 |
| 与同类书籍对比选择 | 6.4 |
