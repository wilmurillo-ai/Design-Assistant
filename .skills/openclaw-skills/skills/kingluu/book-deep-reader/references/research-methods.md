# Research Methods Reference

Detailed strategies for gathering authoritative book content.

## Source Hierarchy (trust from high to low)

| Level | Source Type | Example | Trust |
|-------|------------|---------|-------|
| 1 | 原著原文 / Original text | Book itself, PDF, Kindle highlights | ★★★★★ |
| 2 | 官方Wiki/社区 / Official wiki | reinventingorganizationswiki.com | ★★★★★ |
| 3 | 作者访谈/演讲 / Author talks | YouTube keynotes, podcast transcripts | ★★★★☆ |
| 4 | 出版社页面 / Publisher pages | Douban, publisher's own site | ★★★★☆ |
| 5 | 深度书评 / In-depth reviews | Douban reviews > 1000字, academic reviews | ★★★☆☆ |
| 6 | 读书笔记/思维导图 / Reader notes | Blog posts, mind maps, study groups | ★★★☆☆ |
| 7 | 简介/短评 / Brief descriptions | Amazon, Goodreads short reviews | ★★☆☆☆ |

## Search Strategies by Book Type

### 管理/商业书籍 (Management/Business)

```
搜索1: "<书名>" 目录 章节 内容
搜索2: "<书名>" 书评 深度 读书笔记
搜索3: "<English title>" wiki summary
搜索4: "author name" "<key concept>" explanation
搜索5: "<书名>" 思维导图 脑图
```

### 学术/理论书籍 (Academic/Theory)

```
搜索1: "<book title>" chapter summary key arguments
搜索2: "author name" theory framework explained
搜索3: site:philpapers.org OR site:jstor.org "<book title>"
搜索4: "<书名>" 读后感 书评 豆瓣
```

### 实践/技能书籍 (Practical/Skills)

```
搜索1: "<书名>" 方法 步骤 工具
搜索2: "<book title>" practical guide steps
搜索3: "<书名>" 实践案例 成功经验
```

## Fetching Strategy

For each book, aim to fetch:

1. **Douban book page** (via ISBN) — structure, TOC, description
2. **2-3 detailed Douban reviews** — substantive analysis
3. **Official wiki/community** (if exists) — authoritative details
4. **1-2 external analysis articles** — alternative perspectives
5. **Publisher page** — official description, author bio

Total: 5-8 sources minimum for a thorough analysis.

## Cross-Reference Rules

1. **Chapter structure**: Must match publisher/Douban listing
2. **Key concepts**: Must appear in at least 2 independent sources
3. **Case study data**: Must cite original source (book or wiki)
4. **Author quotes**: Must be verifiable — if uncertain, mark as paraphrase
5. **Controversial claims**: Must include counter-evidence if it exists

## Red Flags (Do Not Trust)

- "According to the book..." without specific chapter/page reference
- Contradictory information across sources without resolution
- Case study data that seems too round or too perfect
- Claims about the book's content from people who clearly haven't read it
- Machine-generated summaries that miss nuance
