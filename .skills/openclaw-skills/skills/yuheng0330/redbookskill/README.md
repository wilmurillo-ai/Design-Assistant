# 小红书饮品热点追踪 Skill

🍹 专为小红书饮品赛道女性博主打造的自动化热点追踪工具

## 功能特点

- ✅ **无品牌导向** - 双重过滤排除所有品牌内容，只保留普通人可复制选题
- ✅ **多赛道覆盖** - 果茶 / 咖啡 / 健康饮品全覆盖
- ✅ **每日自动运行** - 默认每天 9 点准时推送
- ✅ **手机友好 HTML 报告** - 马卡龙配色，完美适配移动端
- ✅ **飞书机器人推送** - 正文预览 + 完整报告附件
- ✅ **合规低成本** - 官方数据源，新用户免费额度可用半年

## 快速开始

### 1. 前置准备

- 火山引擎账号并开通 ArkClaw 自动化编排
- 飞书自定义机器人 Webhook

### 2. 配置环境变量

```bash
export FEISHU_WEBHOOK=https://open.feishu.cn/open-apis/bot/v2/hook/xxx
```

### 3. 安装 Skill

在 ArkClaw 控制台 → Skill 管理 → 上传 ZIP 包安装

## 编排流程

完整可视化编排流程：

1. **定时触发** - Cron 表达式 `0 9 * * *`，时区 Asia/Shanghai
2. **小红书热点采集** - 多关键词搜索，排除品牌，筛选低粉爆款
3. **AI 选题拆解** - 豆包 4o 计算热度，输出结构化选题
4. **HTML 报告生成** - 模板渲染生成手机友好报告
5. **飞书推送** - 富文本预览 + HTML 附件推送

详细配置说明参见 [SKILL.md](./SKILL.md)

## 配置参数

| 参数名 | 默认值 | 说明 |
|--------|--------|------|
| FEISHU_WEBHOOK | - | 飞书机器人 Webhook（必填）|
| exclude_brands | `["喜茶","瑞幸","奈雪","星巴克","蜜雪冰城","茶百道","古茗","CoCo"]` | 排除品牌列表 |
| search_keywords | `["果茶","咖啡","健康饮品","自制果茶","低卡咖啡","无糖饮品","居家饮品","宿舍饮品"]` | 搜索关键词 |
| cron_expression | `0 9 * * *` | 定时触发规则 |
| max_follower_count | 100000 | 最大作者粉丝量 |
| min_interaction | 300 | 最低总互动量 |
| top_n_hotspots | 10 | 输出热点数量 |
