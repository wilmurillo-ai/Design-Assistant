## Common Web3 PM Role Types

| Role | Real Hiring Model |
|---|---|
| Wallet PM | Account, signing, assets, transaction path, safety, retention |
| DeFi PM | Protocol mechanics, yield, risk, TVL, position UX |
| DEX / Swap PM | Routing, slippage, liquidity, MEV, conversion, failure handling |
| On-chain Data PM | Data parsing, protocol modeling, metrics, data quality, SLA |
| Growth PM | Incentives, attribution, activation, retention, trading conversion |
| AI Wallet PM | Agent permission, execution, trust, risk, policy, auditability |
| Product Lead | Priority, roadmap, cross-team execution, metrics |
| Product Director | Strategy, organization, business model, risk, resource allocation |

## Four Translation Moves

1. Translate JD into role reality.
2. Translate product surface into business model.
3. Translate company public signals into current priorities.
4. Translate candidate experience into role proof.
