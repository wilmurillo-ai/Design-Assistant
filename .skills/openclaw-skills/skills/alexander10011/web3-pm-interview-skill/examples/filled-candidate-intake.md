# Filled Candidate Intake Example

```yaml
candidate:
  alias: Candidate A
  current_role: Senior Product Manager at a Web2 fintech company
  years_experience: 7
  target_level: Senior PM
  target_company: Major Web3 wallet / exchange
  target_role: Wallet Senior Product Manager
  target_jd: Wallet product role covering swap, bridge, asset management, and user growth
  interview_stage: Hiring Manager
  time_left: 7 days
  top_3_projects:
    - name: Cross-border payment product
      role: Product owner
      scope: Designed onboarding, KYC, transaction flow, and risk review
      result: Increased activated users by 35% and reduced failed transfers by 18%
      relevance_to_target_role: Transaction flow, risk, user activation, financial UX
    - name: Internal data dashboard
      role: Product lead
      scope: Built metrics dashboard for transaction success and user funnel
      result: Helped operations reduce support tickets by 22%
      relevance_to_target_role: Data product mindset and metric ownership
    - name: Partner API integration
      role: PM
      scope: Integrated payment partner APIs and defined SLA
      result: Reduced integration time from 6 weeks to 3 weeks
      relevance_to_target_role: Protocol/API integration and cross-team execution
  web3_experience: User
  technical_depth: Medium
  management_experience: Small Team
  founder_experience: None
  english_level: Fluent
  strongest_edge: Fintech transaction product experience and data-driven execution
  biggest_risk: Limited hands-on DeFi and wallet protocol depth
  specific_questions:
    - How do I translate Web2 fintech experience into wallet PM relevance?
    - What DeFi and wallet basics must I learn before the hiring manager round?
```

## Expected Skill Diagnosis

- Fit level: medium-high if the role values transaction flow, risk, and growth.
- Biggest translation: payment transaction flow -> wallet transaction path.
- Biggest gap: DeFi, signing, approval, bridge, and on-chain data literacy.
- Prep priority: wallet basics, DeFi basics, product experience review, and one strong wallet growth idea.
