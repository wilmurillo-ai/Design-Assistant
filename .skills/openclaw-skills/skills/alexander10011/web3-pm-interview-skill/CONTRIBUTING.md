# Contributing

Thanks for considering a contribution.

This project is designed to help Web3 PM candidates prepare for interviews while preserving privacy and avoiding company-sensitive information.

## What To Contribute

Good contributions include:

- New anonymized interview patterns
- Better templates
- New role playbooks, such as NFT marketplace PM, security PM, exchange growth PM
- More domain references
- Better mock interview scoring criteria
- Bilingual copy improvements

## Privacy Rules

Do not contribute:

- Raw interview recordings
- Private recruiter messages
- Compensation details
- Non-public company information
- Named interviewer analysis
- Internal metrics
- Any personal data without permission

Use role labels instead:

- Hiring manager
- Wallet lead
- Cross-functional interviewer
- Bar raiser
- HRBP

## Contribution Format

For a new interview pattern, use:

```md
# Anonymous Case: [Role Type]

## Candidate Type

## Target Role

## Role Reality

## Interview Loop

## High-Probability Questions

## Winning Mainline

## Risks And De-risking

## Prep Plan
```

## Review Standard

A contribution should be:

- Practical
- Role-specific
- Privacy-safe
- Useful for an AI agent
- Useful for a real candidate under time pressure
