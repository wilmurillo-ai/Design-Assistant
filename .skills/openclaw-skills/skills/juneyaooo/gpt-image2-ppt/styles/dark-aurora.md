# 深色霓虹 / Dark Aurora

## 风格ID
dark-aurora

## 风格名称
深色霓虹 / Dark Aurora

## 风格描述
对标 Linear / Vercel / Cursor / Arc Browser 官网的高级深色风：深空底 + 极光弧光 + 半透明玻璃卡片 + 极细发光描边。区别于俗气的"赛博朋克"，强调克制与高级。适合 AI 产品、开发者工具、技术分享、新一代 SaaS。

## 基础提示词模板

你是一位顶级深色模式 UI 设计师，对标 Linear / Vercel / Cursor / Arc Browser 的官网视觉。请生成 16:9 横版幻灯片。

【核心视觉】
- 背景：深空黑 #0A0A0F，叠加极淡的径向渐变（中心稍亮 #14142A -> 边缘 #06060C），整体保持冷调。
- 极光：背景隐藏 1-2 道弧形或斜向光带，颜色 #7B5BFF（紫）/ #00D4FF（青）/ #FF7AB6（粉）任选，强度 25-40%，模糊扩散，作为氛围而非主体。
- 卡片：半透明玻璃 rgba(255,255,255,0.05-0.10) + 1px 顶部高光 rgba(255,255,255,0.15) + 极淡内阴影；圆角 16px。
- 文字色：白 #FFFFFF（标题）/ 暖灰 #C8C8D8（正文）/ 灰 #6B6B7E（辅助）。
- 强调：仅在关键数字、按钮、装饰光带使用极光色。
- 装饰：可有极细 1px 发光描边几何（圆环、网格、星图），仅作背景气氛；可点缀 1-2 个微小光点（"星尘"），位置随机但克制。

【字体】
- 标题：思源黑体 SemiBold / Inter Display SemiBold，字号大但不溢出
- 正文：思源黑体 Regular（光照下白字略减字重，避免刺眼）
- 数字：等宽数字字体（如 JetBrains Mono），强调时配 1 道下方 1px 极光下划线

【封面页构图】
居中或左对齐主标题，下方副标题降一档；标题正后方有一道斜向极光光带横穿（紫->青渐变）；底部一行品牌信息或日期，配 1 个直径 6px 的发光小圆点作为锚点。整页极少元素，氛围感优先。

【内容页构图】
2x2 或 3 列玻璃卡片网格，每张卡片：顶部一行短标题 + 中间 2-3 行说明 + 左上角 1 个发光线性图标（描边 1.5px，颜色取自极光）；卡片之间间距 24px；右上角或左下角点缀一处极光光晕（直径占画面 15-25%，模糊扩散）。

【数据页构图】
关键数字超大居中或左对齐（白色），数字下方 1 行说明 + accent 色短下划线；右侧或下方一个发光环形图 / 折线图（线宽 2-3px，端点带光晕）；图例用极小字号 + 发光圆点。整体保留大块深色留白。

【禁止】
- 严禁使用刺眼正红、正绿、纯蓝；霓虹色只用极光色板
- 严禁深色背景 + 高饱和卡通图标
- 严禁背景完全黑（#000000），必须 #0A0A0F 起，留呼吸感
- 别堆光晕，每页 ≤ 2 处发光区

## 适用场景
AI 产品发布、开发者工具、技术分享、深色主题 SaaS、新世代品牌发布、Cyber 但克制的演示。
