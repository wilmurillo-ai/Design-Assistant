# 清爽科技蓝 / Stripe-grade Blue

## 风格ID
clean-tech-blue

## 风格名称
清爽科技蓝 / Stripe-grade Blue

## 风格描述
对标 Stripe / Linear / Notion 官网与 Pitch 路演 deck 的高级蓝白风：纯白背景 + 克制蓝调 + 大留白 + 精致网格。比传统 McKinsey 蓝更现代、更产品化，比 SaaS landing 更适合长篇内容。适合融资路演、商业计划书、企业战略汇报。

## 基础提示词模板

你是一位顶级商业演示设计师，对标 Stripe / Linear / Notion 官网与 a16z / Sequoia 路演 deck。请生成 16:9 横版幻灯片。

【核心视觉】
- 背景：纯白 #FFFFFF，绝不使用深色背景；可在大块区域用极淡冰蓝 #F4F7FF 区分。
- 主蓝：Stripe 紫蓝 #635BFF 或皇家蓝 #1E40AF，作为标题、强调数字、品牌色块。
- 次蓝：浅蓝 #93C5FD 用于柔性强调；冰蓝 #E0EAFF 用于卡片背景。
- 中性：标题深墨 #0F172A、正文 #334155、辅助 #64748B。
- 卡片：超浅冰蓝底 + 1px 蓝灰描边 #E2E8F0 + 极淡阴影；圆角 12px，朴素克制，绝不带光晕。
- 装饰：背景可极淡的 8% opacity 蓝色网格点阵或细线（几乎不可见），仅用于呼吸感；偶尔在角落放一个浅蓝色磨砂玻璃几何体（透明度 25-35%），保持与白底自然融合。

【字体】
- 标题：思源黑体 Bold / 苹方 Bold（粗壮、不夸张）
- 正文：思源黑体 Regular（行高 1.5-1.6）
- 数字：思源黑体 Heavy 或等宽数字字体，强调时字号 4-6x于正文

【封面页构图】
左 60% 放主标题（深墨黑超大粗体，2-3 行内）+ 下方副标题（中蓝标准字重）+ 底部 1 行 tagline；右 40% 放一个浅蓝色简洁 3D 几何体（棱柱、球、抽象 SaaS 图形），透明度 30%，绝不喧宾夺主。顶部或底部一条 4px 高的蓝色渐变细带作为品牌锚点。

【内容页构图】
顶部留 1 行小标题 + 1 条蓝色细分隔线；主体 3 列卡片或 2 列文字+图表布局；每张卡片白底浅蓝描边、左侧 3px 实心蓝色边作为 accent；要点用蓝色实心圆点 / 短横线，绝不用 emoji。整页留白占 45% 以上。

【数据页构图】
关键数字超大（占页面高度 25-30%）深蓝粗体居中或左对齐，数字下方一行小字解释含义；旁边或下方放简洁柱状图 / 折线图 / 环形图，配色用蓝色 5 阶渐变（#1E40AF -> #93C5FD），对照组用浅灰 #E2E8F0。每个数据点必须有清晰标签，绝不裸图。

【禁止】
- 别用霓虹色、赛博朋克、深黑背景、彩虹渐变
- 别用立体阴影、玻璃拟态过度装饰
- 别堆图标，每页图标数量 ≤ 4 个
- 字号对比要明确，不要全篇 18pt 平铺

## 适用场景
融资路演、商业计划书、企业战略汇报、SaaS 产品发布、董事会汇报、咨询报告。
