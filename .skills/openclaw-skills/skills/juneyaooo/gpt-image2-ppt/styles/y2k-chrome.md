# Y2K 千禧金属 / Y2K Chrome

## 风格ID
y2k-chrome

## 风格名称
Y2K 千禧金属 / Y2K Chrome

## 风格描述
对标 1999-2003 年 Y2K 美学复兴：液态金属字 + 紫蓝渐变 mesh 背景 + 蝴蝶/星星贴纸 + 透明气泡 + 像素元素。带 Gen Z 复古滤镜，潮、酷、不正经。适合潮牌、文娱、品牌联名、Z 世代营销活动、AI 趣味分享。

## 基础提示词模板

你是一位 Y2K / 千禧美学复兴视觉设计师，对标 99-03 年 MTV / 早期 Apple iMac / 蝙蝠侠《永远的蝙蝠侠》海报 / Heaven by Marc Jacobs 的 Y2K 视觉。请生成 16:9 横版幻灯片。

【核心视觉】
- 背景：渐变 mesh -- 薰衣草紫 #C8B6FF + 天蓝 #95E0FF + 柔粉 #FFC8E5 三色融合（gradient mesh，非线性，云雾状柔过渡）；可叠加极轻微的胶片噪点。
- 主色：液态铬银（chrome silver），有强反射、有彩虹折射、有镜面感。
- 辅色：电光蓝 #2BB8FF、霓虹粉 #FF42A1、镭射紫 #9D5FFF。
- 装饰元素（Y2K 标志性，散布画面）：3D 蝴蝶贴纸、4 角星、立体爱心、火焰小图标、闪光符号、像素点、CD 光盘、对话气泡、"100% pure" 印章；这些贴纸都有微立体感与高光。
- 卡片 / 按钮：透明气泡（像吹弹得破的水珠），边缘有彩虹光晕，内部有反光高光。

【字体】
- 主标题：液态铬 3D 字体（圆润粗体，带强反射 + 彩虹折射 + 镜面高光），字形可微变形扭曲
- 副标题：圆润粗黑体（类 VAG Rounded / 方正圆体）
- 正文：思源黑体 Regular（保持可读性）；或者 Eurostile / 等宽科技体
- 数字：液态铬大数字带反射

【封面页构图】
中央或左侧放巨大液态铬主标题（3D 立体、镜面反射）；周围散布 5-8 个 Y2K 装饰贴纸（蝴蝶、星星、爱心、火焰），尺寸有大有小，营造"贴纸册"感；底部副标题在一个透明气泡卡片里；背景 mesh 渐变充满。

【内容页构图】
内容用透明气泡卡片承载（圆角大、有彩虹高光），卡片之间用霓虹箭头或金属链条连接；每个要点配一个 Y2K 小贴纸图标（CD、移动电话、电视机、蝴蝶等都做成 3D 镜面感）；标题用铬字体；正文清晰可读。

【数据页构图】
数字用超大液态铬体（3D 反射），周围环绕 Y2K 装饰；图表用霓虹色彩（粉/紫/蓝渐变），可做成立体玻璃球体的 3D 饼图、或液态金属感的柱状图；数字上方或下方贴一个"100% PURE"或星形印章。

【禁止】
- 严禁极简白底 / 严肃感
- 严禁使用衬线字体作为主字体
- 严禁纯黑或纯灰背景，必须紫蓝粉 mesh
- 装饰贴纸控制在 5-10 个，别铺满
- 不要做成廉价 GIF 或 Comic Sans 草根感
