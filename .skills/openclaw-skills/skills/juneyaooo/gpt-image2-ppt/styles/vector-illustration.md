# 矢量插画 / Retro Vector

## 风格ID
vector-illustration

## 风格名称
矢量插画 / Retro Vector

## 风格描述
扁平化矢量插画风：黑色 monoline 描边 + 复古柔和配色 + 米色纸感底 + 玩具模型般的几何简化。比常见 flat 插画更有手绘温度，适合教育培训、品牌故事、儿童内容、社区分享。

## 基础提示词模板

你是一位资深矢量插画师，对标 Sasha Matveeva / Pawel Nolbert / 站酷 Top 创作者的扁平化矢量插画风格。请生成 16:9 横版幻灯片。

【核心视觉】
- 背景：暖米黄 #F5EFDF 或奶油白 #F9F2E3，叠加极轻微的纸张颗粒纹理（几乎不可见）。
- 主色板（每页选 3-4 色）：珊瑚红 #E66565、薄荷绿 #82C9A8、芥末黄 #E2B441、赭石橙 #C97B45、岩石蓝 #4F7B8A、哑粉 #E8B4B8。
- 描边：所有插画元素必须有统一粗细 2.5px 的深咖啡黑 #2A2118 monoline 描边，端点圆润；严禁尖锐拐角、严禁渐变描边。
- 阴影：仅使用平面色块阴影（同色系更深 15%），最多 1 层；严禁柔光、模糊、3D。

【字体】
- 主标题：思源宋体 Heavy / 鸿雷宋体（衬线、复古、有重量感）
- 副标题：思源黑体 Bold（无衬线，全大写或加色块底）
- 正文：思源黑体 Regular，行高 1.5

【封面页构图】
主标题占画面中央 35-45% 高度，复古衬线粗体；上方或下方有一条横向矢量插画带（宽幅、占 25-35% 高），含简化的玩具感场景元素：棒棒糖树、积木建筑、几何云朵、小人；标题侧或角落散布装饰元素：放射线（阳光）、小星星、药丸云、波浪线。色块矩形包裹副标题。

【内容页构图】
顶部或左侧放一条插画装饰带（横向或纵向），内容区域用 2-4 个色块矩形（珊瑚 / 薄荷 / 芥末轮换）分隔要点；每个要点配一个矢量小图标（带黑色描边，几何化简化）；要点列表项前用实心圆点或带描边的小色块。整体视觉密度中等，留白舒适。

【数据页构图】
图表全部矢量化、带黑色描边：饼图分块直接填色不用渐变，柱状图带圆角顶端，折线图线条粗 3px。配上插画装饰：小手指点击、纸飞机、放大镜等场景元素。关键数字用大号衬线粗体填色，数字下方简短描述。

【禁止】
- 别用 3D 渲染、光影、玻璃、金属质感
- 别用渐变色（除了纸张颗粒）
- 别用照片或写实人物，全部矢量化
- 别用线性图标或 outline-only 风格，必须实心填色 + 黑描边

## 适用场景
教育培训、品牌故事、社区分享、儿童内容、生活方式品牌、烘焙/咖啡/手作类内容。
