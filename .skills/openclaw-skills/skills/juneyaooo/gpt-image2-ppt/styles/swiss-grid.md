# 瑞士国际主义 / Swiss Grid

## 风格ID
swiss-grid

## 风格名称
瑞士国际主义 / Swiss Grid

## 风格描述
对标 Müller-Brockmann、Massimo Vignelli 与 1960s 瑞士平面设计学派：极严谨的模块化网格 + Helvetica 风字体 + 三原色色块 + 大量留白 + 清晰对齐。理性、克制、有学术权威感。适合学术报告、设计周演讲、严肃数据汇报、博物馆展陈。

## 基础提示词模板

你是一位国际主义平面设计师，对标 Josef Müller-Brockmann、Massimo Vignelli、Wim Crouwel 的瑞士学派。请生成 16:9 横版幻灯片。

【核心视觉】
- 背景：纯白 #FFFFFF（默认）；或在封面 / 章节页用整块色 -- Bauhaus 红 #E10600、深蓝 #003DA5、镉黄 #FFE600（每页只允许一种纯色）。
- 文字：纯黑 #000000；色块上反白 #FFFFFF。
- 网格：12 列严格基线网格，所有元素必须对齐到网格；元素四角全部直角，绝无圆角、阴影、渐变。
- 装饰：唯一允许的"装饰"是色块（矩形、正圆、半圆），用作视觉重量平衡，不带任何效果。

【字体】
- 全部使用思源黑体 / 苹方（中文）+ Inter / Helvetica Neue（英文混排）
- 主标题：Bold/Heavy，字号巨大；行距紧凑（行高 1.0-1.1）
- 正文：Regular，左对齐，绝不两端对齐
- 字距：保持默认，绝不字间放宽

【封面页构图】
极致经典构图：主标题左下对齐，超大粗体，可 2-4 行；右上 1/3 区域放一个色块（实心矩形或正圆，单色，三原色之一）；标题上方一条 2px 黑色细线 + 一行小型副标题（小写、左对齐）；底部细小一行品牌 / 编号 / 日期。

【内容页构图】
12 列网格严格执行：标题占 12 列宽（左上）+ 下方 4px 黑色横线；主体内容用 4/8/12 col 宽度的文本块组合，要点用纯小圆点或纯短横线；可在某 col 嵌入纯色色块作为重量平衡。整页元素 ≤ 6 个，留白 ≥ 50%。

【数据页构图】
数据用 typography 表达：超大数字（占画面 30%+）左对齐，下方一行说明 + 一条横线；图表必须是最简形式 -- 实心横条（单色，对齐网格）、纯圆环（无渐变无标签外围）、表格（无装饰边框，只用 typography 对齐）；颜色只用三原色 + 黑灰白。

【禁止】
- 严禁圆角、阴影、渐变、3D、霓虹、玻璃效果
- 严禁手写体、衬线体、装饰字体
- 严禁居中正文（除非整页只有 1 个元素）
- 严禁堆叠多色，每页色块 ≤ 1 种 + 黑白
- 严禁使用 emoji 和卡通图标

## 适用场景
学术报告、设计周演讲、博物馆展陈、严肃数据汇报、政府白皮书、城市规划、建筑事务所、艺术馆策展。
