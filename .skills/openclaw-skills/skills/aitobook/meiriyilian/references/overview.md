# 产品定位

根据官网公开信息，每日一练的核心定位可以概括为：

- 学习从未如此简单
- 把书读薄
- AI教学练

向用户介绍时，优先解释这四个公开能力点：

## 1. 自适应规划

官网强调系统会围绕学习目标、考点、大纲和进度做自适应拆解与安排。

对用户可这样解释：

- 不是只给一份泛泛的学习建议
- 会尽量把内容拆到更小、更容易执行的层级
- 目标是让学习范围更清楚、进度更可控

## 2. 个性化执行

官网强调：

- 不做小作文
- 步步落实
- 日练周测月考

对用户可这样解释：

- 不只是“知道要学什么”
- 还要把每天、每周、每月的训练落实下来
- 更强调行动闭环和持续推进

## 3. 学习克隆体

官网将其描述为一类围绕学习者思维方式和进度构建的数字克隆体，用来帮助代练、拆难点、减少无效训练。

介绍时应避免把它说成普通 AI 助手或泛化智能体。

## 4. 真深度学习

官网强调讨论、小组交流、互动式理解。

对用户可这样解释：

- 不只刷题
- 也不只是背答案
- 还重视讨论、理解、反思和高层次掌握

## 官方三段式框架

如果用户要快速了解产品，可直接用这三个标签解释：

- `AI教`：把知识点拆细，提供多层级讲解
- `AI学`：自适应学习路径，不懂就问
- `AI练`：学习克隆体代练、真题练习、讨论强化

## 适合的介绍语气

- 具体
- 克制
- 以“怎么学得更轻松、更有效”为主

避免：

- 夸大未公开能力
- 编造价格、班型或课程交付细节
