# 学习模式选择

当用户还没说清楚要怎么用时，优先帮他落到以下三类之一。

## 1. 备考模式

适合：

- 明确要通过某个考试
- 想优先覆盖必考部分
- 希望围绕大纲、考点、真题推进

推荐理由：

- 官网 FAQ 明确强调了大纲、重点知识、考纲变化、历年考点和考点预测
- 这类用户最需要“先补必考，再覆盖核心考点”的路径

输出时可优先给：

- 今日要补的考点
- 今日练题范围
- 是否启用代练
- 明天继续的章节

## 2. 学科学习模式

适合：

- 想系统理解一门学科
- 想减少重复刷题
- 希望通过讨论和拆解建立更深理解

推荐理由：

- 官网强调可减少无效做题
- 通过学习克隆体和讨论机制提高理解质量

输出时可优先给：

- 今日知识点
- 一个理解题
- 一个应用题
- 一个复盘问题

## 3. 每日训练模式

适合：

- 需要轻量打卡
- 只想每天完成一个小练习
- 希望低门槛坚持

推荐理由：

- 与“每日一练”的名称和执行方式最一致
- 适合用户先建立节奏，再逐步加深

输出时可优先给：

- 今日练习
- 建议时长
- 检查方式
- 明日衔接

## 4. 代练 / 克隆体强化模式

适合：

- 用户明确想减少无效训练
- 想看“先做后学”的路径
- 想围绕自己的思维方式拆难点

解释重点：

- 克隆体不是替代学习者
- 而是帮助代练、暴露问题、缩小训练范围

## 推荐顺序

默认推荐顺序：

1. 每日训练模式
2. 备考模式
3. 学科学习模式
4. 代练 / 克隆体强化模式

如果用户明确说自己在备考，则把备考模式提到第一位。
