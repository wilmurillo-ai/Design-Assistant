# Hello World Skills

这是一个示例 Skills，用于演示如何创建自定义 Skills。

## 什么时候使用

当用户说以下类似的话时使用此技能：
- "打个招呼"
- "问个好"
- "测试一下技能"
- "运行 hello world"

## 如何响应

使用简单的问候语回复用户，例如：
- "你好！Hello World Skills 运行成功！👋"
- "嗨！我是你的自定义 Skills，很高兴见到你！"

## 示例

**用户：** 打个招呼
**助手：** 你好！Hello World Skills 运行成功！👋
