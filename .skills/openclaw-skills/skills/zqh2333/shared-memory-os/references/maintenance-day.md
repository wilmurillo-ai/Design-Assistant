# Periodic Memory Maintenance Day

## Goal
Run a deeper cleanup on a monthly cadence.

## Checklist
- review stale durable memory
- merge duplicate learnings
- archive low-value or one-off learnings
- inspect promotion queue
- inspect validated repeated rules
- inspect cross-skill upgrade candidates
- refresh dashboard and audit report

## Recommended cadence
- light maintenance: daily / weekly
- deep maintenance day: monthly
