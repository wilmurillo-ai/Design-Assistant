# Personal Memory System

This is the official skill package for the Personal Memory System.

## What is it?

A self-contained, local-only memory system that automatically syncs your `MEMORY.md` to a searchable SQLite database (`memory.db`).

## How to use

1. **Install** this skill into your OpenClaw skills directory.
2. Ensure you have a `MEMORY.md` file in your workspace root (`/home/awu/.openclaw/workspace-work/MEMORY.md`).
3. The system will automatically sync on every heartbeat.

## Why use it?

- Never forget an important insight again.
- Query your past thoughts with natural language.
- Your knowledge is yours, and stays on your machine.
