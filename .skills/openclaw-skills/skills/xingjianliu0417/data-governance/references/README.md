# 数据治理参考文档

## 数据质量维度

### 1. 完整性 (Completeness)
- 空值比例
- 必填字段缺失情况

### 2. 准确性 (Accuracy)
- 数值范围校验
- 枚举值校验
- 格式校验

### 3. 一致性 (Consistency)
- 跨表数据一致性
- 历史数据一致性
- 编码统一性

### 4. 时效性 (Timeliness)
- 数据更新时间
- 刷新频率
- 延迟情况

### 5. 唯一性 (Uniqueness)
- 主键重复
- 业务键重复

## 数据血缘符号

| 符号 | 含义 |
|------|------|
| → | 数据流向 |
| ⊕ | 数据转换 |
| ◈ | 数据存储 |
| ◇ | 数据应用 |

## 常用工具

- Apache Atlas - 元数据治理
- Apache Griffin - 数据质量
- DataHub - 数据目录
- Great Expectations - 数据质量
- dbt - 数据转换治理
