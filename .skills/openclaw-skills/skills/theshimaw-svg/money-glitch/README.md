# CashFlow Lite

AI agent that finds and executes money-making opportunities with minimal cost.

## What it does

- Finds real opportunities
- Picks the best one
- Gives exact steps to execute

## Example

Prompt:
"Make $10 online with low cost"

Output:
- Opportunity
- ROI reasoning
- Step-by-step plan
- Next action

## Why it's different

Most AI gives ideas.

This gives execution.

## Use it

Give your agent a goal and let it work.

## Note

This is a lite version of a larger system.
