# AweSeed SD-WAN Skill

This skill describes how to set up AweSeed SD-WAN with the desktop client and use the same Oray account to build automatic cross-site networking between multiple devices.

## Overview

AweSeed SD-WAN helps users connect devices across different locations without requiring a public IP. It is suitable for personal remote access, enterprise networking, industrial connectivity, and remote device management.

## Skill File

- `aweseed-sdwan.md`

## What This Skill Covers

- Downloading and installing the correct AweSeed client from the official download page
- Registering an Oray account and signing in through the client
- Completing automatic cross-site networking by signing in on at least two devices with the same account
- Understanding key product features and common usage scenarios

## Installation

1. Open `https://pgy.oray.com/download`.
2. Download the client that matches your operating system.
3. Install the client on each device that should join the network.

## Account Setup

1. Launch the AweSeed client.
2. Register an Oray account if needed.
3. Sign in with the same account on all target devices.

## Networking Setup

When AweSeed is installed on at least two devices and those devices are signed in with the same Oray account, cross-site networking is completed automatically.

## Core Features

- Cross-site networking without a public IP
- Global intelligent routing on a high-speed backbone network
- Layer 2 networking for industrial remote debugging and data collection
- Remote device management with cloud-based monitoring and alerts
- Secure Wi-Fi intranet access for employees and web authentication for visitors
- Application proxy access with reverse proxy protection and granular authorization
- Support for domestic innovation-oriented technology systems
- IoT SIM services for enterprise traffic usage
- Operations channel services for device discovery and PLC maintenance management
- SDK and API embedding for lightweight integration
