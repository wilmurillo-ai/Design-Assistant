# 🌐 Stealth-Browser Skill

**Die professionelle Browser-Automation für OpenClaw mit automatischem Google-Login.**

---

## 🎯 Was macht dieser Skill?

**Stellen Sie sich vor:**
- Sie loggen sich **einmal** bei Google ein
- Der Skill speichert die Session
- Danach können Sie **alle Google-Dienste vollautomatisch nutzen**
- Ohne erneuten Login
- Ohne Bot-Erkennung
- Ohne manuelles Eingreifen

## 🚀 So einfach ist es

### 1. Installieren

```bash
clawhub install stealth-browser
```

### 2. Einmalig einloggen

```bash
stealth-browser login google
```

### 3. Fertig!

```bash
stealth-browser open "https://mail.google.com"
```

→ Sie sind automatisch eingeloggt!
