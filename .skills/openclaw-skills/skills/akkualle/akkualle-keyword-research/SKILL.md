# 🔑 Keyword Research Tool

**Finde die besten Keywords für deinen Content.**

## ✨ Features

| Funktion | Beschreibung |
|----------|--------------|
| **Keyword Discovery** | High-Value Keywords finden |
| **Search Intent** | Intent-Analyse (Informational, Commercial, Transactional) |
| **Difficulty Score** | Wie schwer ist das Keyword zu ranken? |
| **Volume Check** | Monatliches Suchvolumen |
| **Opportunity Mapping** | Content-Ideen generieren |

## 🚀 Verwendung

```bash
# Keyword recherchieren
keyword-research "e scooter kaufen"

# Bulk-Analyse
keyword-research --file keywords.txt

# Mit Difficulty
keyword-research "keyword" --difficulty
```
