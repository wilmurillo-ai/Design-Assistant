# xinwenlianbo-translator

> 把《新闻联播》的官方话语翻译成正常人能读懂的中文。

---

## 这个技能是什么

《新闻联播》每天 19:00 在央视播出，30 分钟，是中国政府向全国传递政策信号的核心渠道。但它有一套自成体系的话语系统——"强调""指出""要求""作出重要指示"背后各有不同分量；头条的位置、用词的选择、措辞的等级，都是读懂这档节目的关键。

**xinwenlianbo-translator** 这个技能做的事情只有一件：把《新闻联播》的官方语言，翻译成受过高等教育但对政务话语不熟悉的普通人能看懂的中文，并解读出其中的政策信号和实际影响。

---

## 触发方式

在对话中提到以下任意一项，技能会自动启动：

- 「今天的新闻联播说了什么」
- 「帮我解读一下联播」
- 「xwlb / xinwenlianbo」
- 「XX 日的新闻联播」
- 「最近有什么政策动向」（结合联播语境）

---

## 技能会做什么

### 第一步：抓取文字稿

从 [tv.cctv.com](https://tv.cctv.com/lm/xwlb/index.shtml) 实时获取当日（或指定日期）的《新闻联播》文字稿全文，支持指定历史日期。

### 第二步：识别新闻类型

将每条新闻归类：领导人活动、政策发布、经济数据、外交活动、突发事件、典型报道、国际新闻——不同类型的解读策略不同。

### 第三步：话术解码 + 结构解析

- 查询内置话术词典，把官方表述翻译成通俗语言
- 分析新闻在节目中的编排位置（头条 vs 中段 vs 尾段），判断信号强度
- 识别用词等级（「强调」≠「指出」≠「要求」）

### 第四步：输出结构化解读

每条新闻输出：

```
一句话解读 / 关键信息（是什么、意味着什么、对你有什么影响）/ 话术对照表
```

全期结束后附上「今日要点」和「信号雷达」。

## 设计原则

1. **忠实原文，不增不减** — 只翻译，不评论，不添加主观判断
2. **通俗但不降智** — 目标读者是受过高等教育的成年人，不做过度简化
3. **区分事实与解读** — 清楚标注哪些是新闻事实，哪些是分析推断
4. **注重信号，不陷琐碎** — 重点解读政策走向，跳过无意义的例行报道
5. **保持中立** — 不做价值判断，只做语言和结构层面的解码
6. **标注不确定性** — 存在多种解读可能时，如实说明

---

## 局限性

- 文字稿来源为 CCTV 官网公开内容，网络问题可能导致抓取失败
- 历史内容依赖 CCTV 网站保留情况，极早期内容可能无法获取
- 解读基于公开话语体系，无法获知内部背景信息
- 政策信号分析属于推断，不代表官方立场
