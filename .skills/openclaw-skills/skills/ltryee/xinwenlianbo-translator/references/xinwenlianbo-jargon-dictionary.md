# 《新闻联播》话术词典

本词典收录《新闻联播》及中国政府公文中高频出现的官方表述，并提供对应的通俗翻译。翻译时先在本词典中查找匹配项；如无直接匹配，根据下方分类规则推导。

| 官方表述 | 通俗翻译 | 信号强度 |
|----------|----------|----------|
| 习近平总书记 | 最高领导人，国家最高决策者 | 核心信号 |
| 出席 | 去了，但没讲话 | 低 |

| 主持召开会议 | 亲自开会讨论，该议题优先级高 | 高 |
| 强调 | 领导人认为这件事非常重要，必须执行 | 高 |
| 指出 | 领导人提出了一个观点或方向 | 中 |
| 要求 | 领导人下达了具体指令，需要落实 | 高 |
| 作出重要指示 | 领导人直接过问并给出方向 | 很高 |
| 同意/批准 | 顶格背书，方案可以推进 | 很高 |
| 签署命令/主席令 | 法律层面的最终确认，即将生效 | 很高 |
| 致信/贺信 | 远程表达关注和支持，通常不亲自出席 | 低-中 |
| 批示 | 书面指示，由下属执行 | 高 |
| 讲话/发表讲话 | 正式场合的系统论述，代表官方立场 | 高 |
