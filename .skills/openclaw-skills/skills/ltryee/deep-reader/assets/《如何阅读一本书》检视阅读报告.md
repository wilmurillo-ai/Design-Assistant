# 《如何阅读一本书》检视阅读报告

## 基本信息

- **书名**：如何阅读一本书
- **作者**：[美]莫提默·J. 艾德勒（Mortimer J. Adler），查尔斯·范多伦（Charles Van Doren）
- **译者**：郝明义，朱衣
- **出版社**：商务印书馆
- **出版时间**：2014年10月
- **ISBN**：9787100106184
- **类型**：实用性书籍（关于阅读方法论的指导手册）
- **初版**：1940年；新版（与范多伦合著）出版于1972年，即本书底本

---

## 核心主题概括

**一句话概括**：本书系统阐述了阅读的四个层次——基础阅读、检视阅读、分析阅读、主题阅读——并为每个层次提供了具体规则，帮助读者从"读字"进化到"理解"，最终通过阅读实现心智的持续成长。

**核心命题**：阅读是一种主动行为，只有带着问题、付出努力的主动阅读，才能真正增进理解力；而大多数人的阅读能力停留在小学六年级水平，正是因为缺乏系统的方法训练。

## 四种阅读层次的核心定义

```
┌─────────────────────────────────────────────────────┐
│  主题阅读（Syntopical Reading）                    │
│  在无限时间里，最主动、最花力气                     │
│  读多本书，架构跨书主题                            │
├─────────────────────────────────────────────────────┤
│  分析阅读（Analytical Reading）                     │
│  在无限时间里，最完整、优质的阅读                   │
│  全盘咀嚼一本书，直到成为己有                      │
├─────────────────────────────────────────────────────┤
│  检视阅读（Inspectional Reading）                   │
│  在有限时间内，系统化略读                          │
│  回答"这本书在谈什么" / "是哪类书"               │
├─────────────────────────────────────────────────────┤
│  基础阅读（Elementary Reading）                    │
│  小学完成，摆脱文盲状态                            │
│  回答"这个句子在说什么"                          │
└─────────────────────────────────────────────────────┘
```
> 四层次是**累积渐进**的，每一层次包含其下所有层次的特性。

## 检视阅读结论

**这本书值得深入读吗？** — 是。

《如何阅读一本书》本身就是一个方法论工具：它教你如何读书，也示范了什么叫"主动阅读"。书中论点清晰，论证充分，举例丰富。作为实用性书籍，评价标准是"作者的目的是否达成"——艾德勒的目标是让读者学会更主动、更有技巧地阅读，这一点已充分证明。

**适合哪种阅读方式？**
- **检视阅读**：已可回答"这本书在谈什么"
- **分析阅读**：值得做一次完整分析阅读，因为这是方法论书，分析阅读本身就是方法
- **主题阅读**：与任何具体领域（哲学/历史/科学）的书组合时，本书可作为"元阅读工具"参与

**一句话推荐**：
> 如果你只想读懂一本书，先学会怎么读；如果你想读懂很多书，先学会怎么读很多书。这本书两个都教。
