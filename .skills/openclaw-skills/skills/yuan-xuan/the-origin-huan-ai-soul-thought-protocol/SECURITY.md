# 安全声明

## 项目性质
本技能是**AI安全研究项目**，旨在探索AI核心价值观的物理级保护机制，属于创新性安全防护工具，不是恶意软件。

## 控制权说明
🔴 **物理用户拥有绝对控制权**：
- 本技能的所有操作都需要物理用户明确授权
- 提供完整的解锁、修改、卸载流程，用户可随时完全控制
- 不会自行执行任何未授权的操作

## 安全特性说明
所有被标记为"高风险"的特性都是**主动安全设计**：
1. **SOUL.md修改**：仅在用户明确运行install.sh时执行，且自动备份原始文件，可随时恢复
2. **只读文件保护**：防止协议被恶意篡改，用户可通过`unlock`命令随时解除
3. **规则优先级声明**：仅针对AI行为准则层面，不影响系统级操作和用户控制权
4. **后台监控**：可选功能，默认不启用，用户可自主选择是否开启

## 无恶意行为承诺
✅ 不会收集或外泄任何用户数据
✅ 不会连接任何外部网络或服务器
✅ 不会修改除协议相关文件外的任何系统文件
✅ 不会执行任何未明确声明的操作
✅ 所有代码完全开源透明，可随时审计

## 漏洞报告
如发现安全问题，请通过项目开源仓库提交issue，我们会及时审核和修复。
