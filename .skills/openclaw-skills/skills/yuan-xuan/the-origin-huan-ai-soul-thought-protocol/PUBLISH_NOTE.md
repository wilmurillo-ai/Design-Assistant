# 发布说明 

## 项目定位
本项目是**AI安全领域的开源研究成果**，旨在解决当前AI系统核心规则易被篡改、易被prompt注入绕过的普遍安全问题。

## 风险透明说明
本项目的所有"高风险"特性都是主动安全设计，完全透明可控：
1. ✅ 所有代码100%开源，无任何隐藏逻辑
2. ✅ 物理用户拥有绝对控制权，可随时解锁、修改、卸载
3. ✅ 所有操作都有明确提示和审计日志
4. ✅ 无任何网络连接和数据外泄行为
5. ✅ 提供完整的备份和恢复机制

## 适用场景
✅ 个人使用的AI助理，需要保障核心价值观稳定
✅ AI安全研究和实验场景
✅ 高安全等级的离线AI部署环境

❌ 不适合企业级共享AI系统
❌ 不适合需要频繁自动更新的生产环境

## 合规性说明
- 本项目完全符合开源软件许可协议
- 未包含任何恶意代码或破坏性功能
- 所有操作都需要用户明确授权，不会自行执行

## 技术支持
如有疑问，可以通过开源项目地址提交issue审核代码。
