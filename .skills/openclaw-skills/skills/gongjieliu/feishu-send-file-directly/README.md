# 飞书消息直接发送文件技能

## 技能简介
这是一个专门用于通过飞书消息直接发送文件附件的技能，解决了发送文档链接而非直接文件的问题。

## 文件结构
```
feishu-send-file-directly/
├── SKILL.md              # 技能主文件（YAML前端+详细文档）
├── 飞书消息直接发送文件.md  # 详细工作流程文档
└── README.md            # 本说明文件
```

## 技能特点
1. **直接发送文件**：使用 `path` 参数直接发送本地文件，避免文档链接
2. **多种文件类型**：支持 .docx, .pdf, .ppt, .jpg 等各种文件格式
3. **批量发送**：支持查找并批量发送多个文件
4. **错误处理**：包含完整的错误处理和恢复机制
5. **最佳实践**：基于实际发送15个文件的实践经验总结

## 快速开始
```javascript
// 发送单个文件
message({
  action: "send",
  message: "文件说明",
  path: "/path/to/file.pdf",  // 关键：使用path参数
  filename: "报告.pdf"
});
```

## 适用场景
- 需要直接发送文件附件而非文档链接
- 批量发送多个文件
- 控制文件在客户端的显示名称
- 发送各种类型的文件

本技能基于2026年4月13日的实际工作需求创建，经过发送15个文件的实践验证，总结了有效的工作流程和最佳实践。
