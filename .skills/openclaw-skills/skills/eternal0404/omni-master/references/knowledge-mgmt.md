# Knowledge Management

## Tools Available
- **notion** — Pages, databases, blocks (structured knowledge)
- **obsidian** — Markdown vault (linked knowledge graph)
- **apple-notes** — Quick capture, searchable
- **bear-notes** — Tagged, markdown notes
- **blogwatcher** — RSS/Atom feed monitoring

## Knowledge Patterns

### PARA Method
- **Projects** — Active, with deadline
- **Areas** — Ongoing responsibility
- **Resources** — Reference material
- **Archives** — Completed/inactive

### Zettelkasten
- Atomic notes (one idea each)
- Unique IDs for linking
- Bidirectional links
- Index notes for navigation

### Second Brain
- Capture → Organize → Distill → Express
- Progressive summarization
- Save only what resonates

## Note-Taking Workflow
1. **Capture** — Quick entry (don't organize yet)
2. **Process** — Clarify and categorize
3. **Organize** — File in appropriate place
4. **Review** — Regular cleanup and linking
5. **Retrieve** — Search when needed
