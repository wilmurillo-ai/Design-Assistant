# Project Management

## Tools Available
- **trello** — Kanban boards (lists, cards, labels)
- **notion** — All-in-one workspace (pages, databases)
- **things-mac** — macOS task manager
- **apple-reminders** — Simple task lists
- **github** — Issue tracking, milestones, projects

## Methodologies

### Kanban
- Columns: To Do → In Progress → Review → Done
- WIP limits to prevent overload
- Visualize flow, identify bottlenecks

### Agile/Scrum
- Sprints (1-4 weeks)
- Daily standups
- Sprint planning, review, retrospective
- User stories with acceptance criteria

### GTD (Getting Things Done)
- Capture → Clarify → Organize → Review → Engage
- Inbox zero mentality
- Next actions, waiting for, someday/maybe

## Task Management Patterns

### Task Properties
- Title, description, priority, due date
- Assignee, tags, project, status
- Dependencies (blocks / blocked by)

### Priority Levels
1. **Critical** — Do now, blocks everything
2. **High** — Do today
3. **Medium** — Do this week
4. **Low** — When available
5. **Someday** — Future consideration

### Milestones

- Burndown chart generation
- Cross-platform sync (GitHub ↔ Trello ↔ Notion)

- Burndown/burnup charts
- Velocity tracking
