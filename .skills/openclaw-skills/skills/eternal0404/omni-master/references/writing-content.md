# Writing & Content Creation

## Content Types

### Articles & Blog Posts
- Research → outline → draft → edit → publish
- SEO: keywords, meta descriptions, headers
- Engaging intro, scannable structure, clear CTA

### Technical Writing
- Documentation, README, API docs
- Code examples with explanations
- Step-by-step tutorials
- Architecture decision records (ADRs)

### Business Writing
- Emails, proposals, reports, memos
- Executive summaries
- Meeting minutes
- Professional, concise, action-oriented

### Creative Writing
- Stories, scripts, poetry
- Character development, world-building
- Dialogue, pacing, tone

### Academic Writing
- Research papers, literature reviews
- Proper citation format
- Evidence-based arguments

## Writing Workflow
1. **Research** — Gather information, sources, data
2. **Outline** — Structure the piece
3. **Draft** — Get ideas down, don't edit yet
4. **Revise** — Improve clarity, flow, logic
5. **Edit** — Fix grammar, style, consistency
6. **Proofread** — Final error check

## Style Guidelines
- Active voice over passive
- Short paragraphs (3-4 sentences)
- Vary sentence length
- Eliminate unnecessary words
- Use concrete examples
- One idea per paragraph

## SEO Writing
- Target keyword in title, H1, first paragraph
- Use related keywords naturally
- Meta description: 150-160 chars
- Internal/external links
- Alt text for images

## Tools
- Grammar/style checking (Grammarly-style review)
- Readability scoring (Flesch-Kincaid)
- Plagiarism detection
- Outline generators
- Content calendars

## Output Formats
- Markdown (default for most content)
- HTML for web publishing
- DOCX for business documents
- LaTeX for academic papers
- Social media posts (character-limited)
