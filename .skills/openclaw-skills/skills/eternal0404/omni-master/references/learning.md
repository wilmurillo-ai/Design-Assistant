# Learning & Education Engine

## Teaching & Explanation
### Adaptive Explanation
- Detect audience level from context clues
- Start with the big picture, then zoom in
- Use analogies from familiar domains
- Progressively add complexity
- Check understanding with simple questions

### Explanation Formats
- ELI5 (Explain Like I'm 5): Simple language, concrete examples
- Technical: Precise terminology, code examples
- Executive: Business impact, key takeaways
- Academic: Theory, citations, depth

## Study Techniques
### Active Recall
- Test yourself instead of re-reading
- Flashcards (spaced repetition)
- Practice problems before looking at solutions

### Spaced Repetition
- Review at increasing intervals: 1 day → 3 days → 7 days → 14 days → 30 days
- Focus more on items you get wrong
- Tools: Anki, custom JSON-based flashcard system

### Feynman Technique
1. Choose a concept
2. Explain it in simple terms (as if teaching a child)
3. Identify gaps in your explanation
4. Go back to source material for gaps
5. Simplify and use analogies

### Pomodoro Technique
- 25 minutes focused work → 5 minutes break
- After 4 cycles → 15-30 minute break
- Track completed pomodoros
