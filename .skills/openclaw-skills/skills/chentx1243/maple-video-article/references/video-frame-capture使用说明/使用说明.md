## 概述

该工具用于按固定时间间隔从本地视频中截取画面，并将图片保存到指定目录。适合做视频内容预览、关键帧采样、后续图像识别输入等场景。

## 适用场景

- 需要快速了解视频整体内容
- 需要从长视频中按时间间隔提取截图
- 需要跳过重复度较高的相似画面
- 需要把截帧结果继续交给后续脚本处理

## 快速开始

下面命令表示每隔 10 秒截取一帧：

```powershell
python scripts/video_frame_capture.py --input "D:\videos\meeting.mp4" --output-dir "D:\frames\meeting" --interval-seconds 10
```

## 参数说明

- `--input`：本地视频文件路径，必填
- `--output-dir`：输出目录，必填
- `--interval-seconds`：截帧间隔，单位秒，必填，且必须大于 0
- `--skip-similar-frames`：启用相似帧跳过逻辑
- `--similarity-threshold`：相似度阈值，范围 `0` 到 `1`，默认值为 `0.70`
- `--image-extension`：输出图片格式，可选 `jpg` 或 `png`，默认值为 `jpg`

## 输出文件命名规则

截取的图片文件名格式如下：

```text
视频文件名_00h00m00s_0001.jpg
```

示例：

```text
meeting_00h01m30s_0003.jpg
```

命名组成说明：

- 第一段：原视频文件名，不含扩展名
- 第二段：该帧在视频中的时间点，格式为 `hhmmss`
- 第三段：本次抓取尝试对应的四位序号

## 相似帧跳过机制

启用 `--skip-similar-frames` 后，脚本会将当前候选帧与上一张已保存图片进行比较：

- 相似度高于阈值：跳过当前帧，不保存图片
- 相似度低于或等于阈值：保存当前帧，并将该帧作为新的比较基准

## 新增结果清单输出

脚本处理完成后，除了输出截帧图片，还会在 `--output-dir` 目录下额外生成一个 `result.json` 文件，用于记录本次实际保存成功的图片结果列表。

### result.json 规则

- 文件名固定为 `result.json`
- 保存位置为 `--output-dir/result.json`
- 内容为 JSON 数组
- 数组中的每一项对应一个实际保存成功的图片文件
- 每一项包含以下字段：
- `file_index`：图片保存序号，从 `1` 开始递增
- `file_path`：图片完整路径，包含文件名

### result.json 示例

```json
[
  {
    "file_index": 1,
    "file_path": "D:\\frames\\meeting\\meeting_00h00m00s_0001.jpg"
  },
  {
    "file_index": 2,
    "file_path": "D:\\frames\\meeting\\meeting_00h00m10s_0002.jpg"
  }
]
```

## 终端输出说明

脚本运行时会输出以下几类日志：

- 输入视频路径
- 输出目录路径
- 截帧间隔
- 是否启用相似帧跳过
- 视频总时长
- 当前进度和抓取位置
- 单张图片保存结果
- 最终统计信息
- `result.json` 的保存路径

## 使用示例

### 基础截帧

```powershell
python scripts/video_frame_capture.py --input "D:\videos\meeting.mp4" --output-dir "D:\frames\meeting" --interval-seconds 10
```

### 跳过相似帧

```powershell
python scripts/video_frame_capture.py --input "D:\videos\meeting.mp4" --output-dir "D:\frames\meeting" --interval-seconds 10 --skip-similar-frames
```

### 自定义相似度阈值

```powershell
python scripts/video_frame_capture.py --input "D:\videos\meeting.mp4" --output-dir "D:\frames\meeting" --interval-seconds 10 --skip-similar-frames --similarity-threshold 0.80
```
