# 支持的视频格式

## 支持的输入格式

视频帧截取工具支持以下主流视频格式，覆盖日常办公、媒体制作、网页传输等各类场景，兼容性适配绝大多数本地视频文件：

- **MP4**（.mp4）—— 目前最通用、兼容性最强的视频格式，适用于绝大多数设备与平台
- **MKV**（.mkv）——  Matroska开源视频容器格式，支持多音轨、多字幕和高清视频封装
- **MOV**（.mov）—— QuickTime专属视频格式，常用于苹果设备、专业视频剪辑场景
- **AVI**（.avi）—— 音视频交错格式，经典老牌视频格式，适配老旧设备与传统剪辑工程
- **FLV**（.flv）—— Flash视频格式，早年网页流媒体常用格式，部分存量视频仍为此格式
- **WMV**（.wmv）—— Windows媒体视频格式，微软平台专属，适配Windows系统原生播放
- **WebM**（.webm）—— 开源网页专用视频格式，体积小巧，适配网页端快速加载与播放
- **M4V**（.m4v）—— MPEG-4标准衍生视频格式，常用于苹果设备视频存储与传输

## 技术细节说明

### 帧提取原理

工具脚本依托OpenCV库的VideoCapture模块实现精准帧提取，全程按照指定时间间隔完成操作，核心执行流程如下：

1. 通过`cv2.VideoCapture`方法打开本地视频文件，完成视频流初始化
2. 根据视频总帧数和帧率（FPS）计算视频总时长，确定截取时间范围
3. 通过`CAP_PROP_POS_MSEC`参数精准定位到指定毫秒级时间戳位置
4. 按设定间隔完成帧捕获，并按照指定格式保存至目标目录

### 相似帧检测逻辑

启用`--skip-similar-frames`跳过相似帧参数后，脚本会通过以下步骤完成帧相似度比对，高效剔除重复冗余画面：

1. 将待比对帧统一缩放至320x180分辨率，降低计算量，提升比对效率
2. 将彩色帧转换为灰度图像，剔除色彩干扰，聚焦画面内容差异
3. 计算两帧之间的像素绝对差值，量化画面差异程度
4. 通过公式**1.0 - 平均像素差值/255.0**计算相似度得分，分值越接近1代表画面越相似
5. 相似度得分高于设定阈值的帧直接跳过，低于或等于阈值的帧则保存并更新基准对比帧

### 输出文件命名规范

所有截取的帧文件统一遵循标准化命名规则，方便后续查找、整理和溯源，格式如下：

```
视频文件名_时分秒时间戳_截取序号.jpg
```

示例：`meeting_00h01m30s_0003.jpg`

- **视频文件名**：原始视频文件去除后缀后的名称
- **时分秒时间戳**：帧在视频中对应的时间位置，格式为小时+分钟+秒
- **截取序号**：帧的截取顺序编号，固定4位数字，不足位数用0补齐

## 依赖环境

工具运行需安装以下Python第三方库，核心依赖为OpenCV相关包，配套库会自动关联安装：

- `opencv-python-headless`—— 核心视频处理与帧提取库，无GUI界面版本，适配服务器与轻量化环境
- `numpy`—— 数组运算与数值计算库，随OpenCV自动安装，用于帧数据处理与相似度计算

一键安装所有依赖命令：`pip install -r requirements.txt`
