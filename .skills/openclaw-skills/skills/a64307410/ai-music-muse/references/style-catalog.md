## 强交互选项映射

用户选择字母选项后，映射到 `tags` 参数（多个标签逗号分隔）：

```
风格选项（展示 TOP 9 + 随机）→ tags 值（必须与 API 返回的 style 字段完全一致）:
A. 流行音乐    B. 摇滚       C. 民谣
D. 电子乐      E. 古风       F. R&B
G. 说唱        H. 爵士       I. 随机推荐
（更多风格请告诉我具体想要的类型）

情绪选项 → 追加到 tags:
A. 快乐        B. 伤感       C. 激昂
D. 浪漫        E. 放松       F. 随你决定

音色选项 → vocalGender:
A. 男声 → "m"
B. 女声 → "f"
C. 都可以 → 不传该参数
```

## 高级用法

风格标签支持自由组合，多个标签用逗号分隔传入 `tags` 参数：
- `"流行音乐,浪漫"` — 浪漫流行
- `"古风,空灵,古筝"` — 古风空灵古筝曲
- `"说唱,活力,808"` — 活力说唱
- `"爵士,放松,萨克斯管"` — 放松爵士萨克斯

乐器和人声特效标签也可作为 tags 传入，AI 会尝试在生成中体现。
