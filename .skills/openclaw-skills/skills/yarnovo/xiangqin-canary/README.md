# xiangqin

相亲平台（付费置顶曝光商业模式）的使用入口。TRIGGER when 用户说"相亲"、"找对象"、"xiangqin"、"注册相亲"、"填资料"、"查匹配"、"买曝光"、"买置顶"、"让更多人看到我"、"付费相亲"。DO NOT TRIGGER when 用户要聊天/推荐/匹配算法（本项目不做）、用户想的是 matchmaker（已废弃）、服务端不可达时先 curl health。

## Install

```bash
# via clawhub
clawhub install xiangqin

# via skills.sh
npx skills add acong-tech/skill-xiangqin
```
