# v3.1.2 更新重点

这次版本重点不是简单“加一点新功能”，而是继续把 kkclaw 往 **更稳定、更省心、更适合长期使用** 的方向推进。

## 重点更新

- **安全加固**：敏感配置和运行链路更安全
- **模型管理升级**：更方便地管理模型与 Provider
- **配置体验提升**：首次配置和日常调整更顺手
- **切换逻辑优化**：切模型、切 Provider 更低摩擦
- **长期运行增强**：更适合长期挂着跑
- **14 情绪系统**：桌面反馈更自然
- **语音播报 / 声音克隆**：桌面交互更有存在感
- **Setup Wizard**：更低上手门槛
- **API Key 加密存储**：安全性更强
- **Windows + macOS 支持**：跨平台可用

## 一句话总结

**v3.1.2 让 kkclaw 更像一套真正可日常使用的 OpenClaw 桌面工具箱，而不只是一个“好看的桌面宠物”。**
