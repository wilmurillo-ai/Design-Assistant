# Source Priority

GlobalWire does not need to crawl everything, but it should know which sources deserve more trust when patching or cleaning up an answer.

## Priority order

### Tier 1: strongest checks

- official statements and primary documents
- Reuters
- Associated Press
- BBC
- Bloomberg
- Financial Times
- Wall Street Journal

### Tier 2: useful support

- Guardian
- New York Times
- Al Jazeera
- NPR
- Nikkei

### Tier 3: supplementary Chinese sources

- Caixin
- Xinhua
- The Paper
- CCTV News

### Tier 4: weak signals

- X / Twitter posts
- Weibo
- Telegram
- prediction markets

## Practical rule

When the current answer already looks plausible:

- keep editing light
- only patch obvious gaps
- prefer adding one or two stronger sources over rebuilding the entire answer

## Daily brief rule

For a "what happened today" briefing:

- prioritize events with clear cross-border spillover
- prefer geopolitics, war, sanctions, macro, energy, and globally material AI/technology shifts
- avoid promoting local accidents, entertainment news, or niche domestic political items into the top card list unless they clearly became global stories
- symbolic or ceremonial political/religious events should usually stay out of the main card list unless they changed power, law, conflict, or global diplomacy in a concrete way

## For major conflict timelines

- international and official sources should shape the main arc
- Chinese media can supplement but should not become the main spine
