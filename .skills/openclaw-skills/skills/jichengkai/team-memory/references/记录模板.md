### 记录块格式

#### 观察记录 [OBS-YYYYMMDD-001]
```markdown

**追踪项**: 
- [ ] {高/中/低优先级} {事项} (来源: OBS-YYYYMMDD-001)
```

## 标签系统

### 记录类型标签
- `#对话` - 对话记录
- `#管理动作` - 管理决定
- `#OKR沟通` - OKR相关沟通

### 内容标签
- `#承诺` - 对成员的承诺
- `#我的承诺` - 我需要完成的事
- `#O1` `#O2` `#O3` `#O4` - OKR板块关联

### 反思标签
- `#反思` - 需要反思的内容
- `#进化` - 管理者自身的成长
- `#学习` - 学到的新方法

---

## 关键词映射

### 时间词
昨天、今天、前天、本周、上周、最近、月初、月中、{具体日期}

### 类型词
- 观察：观察、发现、看到、技术突破、项目交付、帮助同事
- 对话：1:1、谈话、沟通、表扬、鼓励、提醒、批评、承诺
- 管理：晋升、调岗、分配任务、年终评价
- OKR：季度、年度、进度、KR、完成情况

### 评级词
- 优秀/超预期 → ⭐⭐⭐⭐⭐
- 良好/符合预期 → ⭐⭐⭐⭐
- 一般/正常 → ⭐⭐⭐
- 需关注/低于预期 → ⚠️

### 优先级词
- 紧急/重要 → 高
- 一般/需关注 → 中
- 建议/长期 → 低
