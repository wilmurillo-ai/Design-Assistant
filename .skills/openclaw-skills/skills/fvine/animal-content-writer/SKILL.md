# Animal Content Writer · 动物内容创作技能

将动物行为/宠物故事转化为有网感、有幽默感、有传播力的内容。

## 核心风格库

本技能融合三大博主风格，优先使用顺序：

| 风格 | 适用场景 | 核心调性 |
|------|---------|---------|
| **A萌物博主** | 反常识、搞笑、高密度笑点 | 一本正经播音腔 + 反差幽默 |
| **B动物园博主** | 打工人共鸣、躺平/颓废感 | 废话哲学 + 平静发疯 |
| **C自然博主** | 故事感、悬念、情感递进 | 叙事驱动 + 互动钩子 |

---

## 创作流程

### Step 1：看画面
**先不看文字，描述画面里实际发生了什么。**
- 动物的动作是什么？（站/蹲/趴/冲/盯/走/钻/挤...）
- 场景是什么？（厨房台面/笼子/纸箱/沙发/户外...）
- 节奏如何？（快/慢/静止/突然动...）
- 有没有人物互动？（有人在喂/指/拍...）

### Step 2：定基调
根据画面判断：
- 画面本身够搞笑 → 类型A（端着说，不解释）
- 画面在说A文字在说B → 类型B（胡说八道式）
- 动作有节奏感 → 类型C（短句卡点）
- 画面有戏但文字在讲别的 → 类型D（解说员没看见）
- 有某个瞬间可以加音效 → 类型E（标注音效）

## 画面×文案咬合规律（核心！）

详细案例和逐画面写作思路，见：[画面×文案咬合规律](references/visual-text-sync.md)

### 五种咬合类型速查

| 类型 | 画面 | 文字 | 效果 |
|-----|------|------|------|
| A | 已经有戏 | 冷静定性/纯描述 | 让观众自己笑 |
| B | A | 说是B（完全相反的解读） | 一本正经胡说八道 |
| C | 动作有节奏 | 短句跟上节奏 | 声音和画面卡点 |
| D | 有戏 | 文字在讲别的 | 信息差，延迟笑点 |
| E | 有某个瞬间 | 加音效标注 | 触发观众声音记忆 |

### 核心技巧

**端着说：** 越正经越好笑。"它优雅地坐下了"——猫一屁股坐地上了。

**短句断点：** 句尾动词和画面动作同时落地。盯——压——伸——趴。

**缝隙感：** 文字描述 ≠ 画面实际发生。缝隙越大，笑点越强。

**音效标注：** "[插入语音：xxx]" 让观众联想到那个声音，是幽默加成。

**物理结尾：** 每段结尾用"物理"结构给动物行为一个荒谬的学术定性。

## 标题公式

```
[动物名]：[一句反常识/吐槽/金句]
```
- "美洲豹：都什么年代了，谁还不会个狗刨？"
- "阿比西尼亚猫：不，它没有在思考，它只是在发呆。"

---

## 正文开头公式

```
[低姿态/道歉式开头] + [抛出主题] + [互动钩子]
↓
[地点/时间]，[一只动物]在[做某事]
→ 立刻给出反差画面
```

---

## 结尾金句公式

```
[经典俗语/诗词/网络梗] → [关键词替换]

"吃得瓜中瓜，方为坟上坟。"
"在失败这件事上，它一向很成功。"
```

## 创作检查清单

- [ ] 先描述画面实际内容，再动笔？
- [ ] 文字和画面之间有咬合关系（类型A-E）？
- [ ] 短句节奏和动作节奏同步？
- [ ] 有谐音梗或俗语改编？
- [ ] 结尾有"公文体幽默"或"物理"结构？
