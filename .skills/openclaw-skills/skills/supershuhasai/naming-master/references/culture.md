# 诗词典故与历史文化取名参考

## 一、《诗经》取名

《诗经》是中国最早的诗歌总集，历来是取名的重要来源。孔子曰："不学诗，无以言。"

### 《诗经·国风》精选

#### 《周南》
- "关关雎鸠，在河之洲。窈窕淑女，君子好逑。" → **洲、淑、窈、窕、逑**
- "南有乔木，不可休思。" → **乔、木、休、思**
- "桃之夭夭，灼灼其华。" → **夭、灼、华**（华：光彩、繁华）

#### 《召南》
- "维鹊有巢，维鸠居之。" → **维、居**
- "甘棠之木，蔽芾其叶。" → **棠、叶**
- "野有死麕，白茅包之。" → **白、茅**

#### 《秦风》
- "蒹葭苍苍，白露为霜。所谓伊人，在水一方。" → **苍、露、霜、伊**（经典！"伊"字常用于女孩名）
- "岂曰无衣？与子同袍。" → **同、袍**
- "车邻驷铁，辚辚阙阙。" → **铁**

#### 《卫风》
- "淇水滺滺，桧楫松舟。" → **淇、滺、楫、舟**
- "瞻彼淇奥，绿竹猗猗。有匪君子，如切如磋，如琢如磨。" → **淇、绿、竹、猗、匪、切、磋、琢、磨**（"如琢如磨"比喻修养品德，极为高雅）

#### 《王风》
- "彼黍离离，彼稷之苗。" → **黍、稷、苗**
- "知我者谓我心忧，不知我者谓我何求。" → **知、忧、求**

#### 《郑风》
- "风雨如晦，鸡鸣不已。" → **晦**（虽为黑暗，但常引申为在困境中不放弃）
- "青青子衿，悠悠我心。" → **青、衿、悠**（"悠悠"极美，可用于女孩名）

#### 《小雅》
- "昔我往矣，杨柳依依。今我来思，雨雪霏霏。" → **依、思、霏**（"依依"和"霏霏"皆为极美之名）
- "高山仰止，景行行止。" → **高、景**（"景行"意为高尚的德行）
- "如月之恒，如日之升。" → **恒、升**（永恒、上升，男女皆宜）
- "秩秩斯干，幽幽南山。" → **干、幽、南**

#### 《大雅》
- "文王在上，於昭于天。" → **昭、天**
- "既明且哲，以保其身。" → **明、哲**
- "上天之载，无声无臭。" → **天、声**

#### 《颂》
- "日就月将，学有缉熙于光明。" → **就、将、缉、熙、明**（"缉熙"意为光明，极为雅致）
- "穆穆文王，於缉熙敬止。" → **穆、缉、熙、敬**
- "维天之命，於穆不已。" → **维、穆**

## 二、《楚辞》取名

《楚辞》以屈原为代表，文辞瑰丽，意境深远，是取名的重要来源。"男楚辞，女诗经"之说流传甚广。

### 《离骚》精选
- "名余曰正则兮，字余曰灵均。" → **正则、灵均**（屈原自己的名字）
- "日月忽其不淹兮，春与秋其代序。" → **序**（秩序、规律）
- "路漫漫其修远兮，吾将上下而求索。" → **修、远**（"修远"极好！）
- "朝饮木兰之坠露兮，夕餐秋菊之落英。" → **兰、露、菊、英**
- "芳与泽其杂糅兮，唯昭质其犹未亏。" → **芳、泽、昭**（"昭质"指光明纯洁的品质）
- "乘骐骥以驰骋兮，来吾道夫先路。" → **骐、骥、驰、骋**
- "长太息以掩涕兮，哀民生之多艰。" → **太息**（过于悲伤，不宜取名）

### 《九歌》精选
- "悲莫悲兮生别离，乐莫乐兮新相知。" → **知**（新知、相知）
- "秋兰兮青青，绿叶兮紫茎。" → **兰、青**（"秋兰"极为雅致）
- "若有人兮山之阿，被薜荔兮带女萝。" → **若、萝**（"若"字极美）
- "搴汀洲兮杜若，将以遗兮远者。" → **汀、杜、若、远**
- "沅有芷兮澧有兰，思公子兮未敢言。" → **芷、兰、思**（"芷兰"极雅）

### 《九章》精选
- "浩浩沅湘，分流汩兮。" → **浩、湘**
- "愿岁并谢，与长友兮。" → **岁、友**

## 三、唐诗宋词取名

### 唐诗精选

#### 李白
- "长风破浪会有时，直挂云帆济沧海。" → **云帆、济沧、沧海**
- "天生我材必有用，千金散尽还复来。" → **材、复**
- "举杯邀明月，对影成三人。" → **明、月**
- "飞流直下三千尺，疑是银河落九天。" → **银河、九天**
- "大鹏一日同风起，扶摇直上九万里。" → **鹏、扶、摇**

#### 杜甫
- "会当凌绝顶，一览众山小。" → **凌、绝**
- "随风潜入夜，润物细无声。" → **润、声**
- "读书破万卷，下笔如有神。" → **读、神**
- "两个黄鹂鸣翠柳，一行白鹭上青天。" → **青、鸣**

#### 王维
- "行到水穷处，坐看云起时。" → **云起**（极为雅致，寓意绝处逢生）
- "明月松间照，清泉石上流。" → **明、松、清、泉**
- "空山新雨后，天气晚来秋。" → **新雨**
- "大漠孤烟直，长河落日圆。" → **河、圆**

#### 苏轼
- "但愿人长久，千里共婵娟。" → **长久、婵娟**
- "不识庐山真面目，只缘身在此山中。" → **真**
- "竹杖芒鞋轻胜马，谁怕？一蓑烟雨任平生。" → **平生**
- "人生到处知何似，应似飞鸿踏雪泥。" → **飞鸿**

#### 李清照
- "知否，知否，应是绿肥红瘦。" → **知**
- "寻寻觅觅，冷冷清清，凄凄惨惨戚戚。" → **清**
- "莫道不销魂，帘卷西风，人比黄花瘦。" → **黄花**

## 七、取名避讳与禁忌

### 现代取名应避免
1. **生僻字**：如"龘、翾、龂"等，影响日常生活
2. **谐音不雅**：如"史珍香"（屎真香）、"杜子腾"（肚子疼）
3. **过于常见**：如"伟、敏、静、丽、强"等，重名率极高
4. **消极含义**：如"衰、败、苦、难"等
5. **过度洋化**：完全英文名作中文名，如"杰克、玛丽"
6. **与直系亲属重名**：传统避讳
7. **多音字过多**：造成读音混乱
8. **笔画过于复杂**：如"鑫、淼、龘"组合

### 取名用字比例建议
- 名字用字应能被98%以上的人正确认读
- 名字总笔画建议控制在10-30画之间
- 名字的声母韵母应与姓氏有良好区分
