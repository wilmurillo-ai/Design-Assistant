# 八卦与周易起名智慧

## 一、八卦基础

### 先天八卦（伏羲八卦）
先天八卦反映宇宙的本体结构，体现阴阳对立统一。

```
        乾（天）☰
   巽（风）☴      坎（水）☵
        离（火）☲
   震（雷）☳      艮（山）☶
        坤（地）☷
        兑（泽）☱
```

方位对应：乾南、坤北、离东、坎西、兑东南、巽西南、震东北、艮西北

### 后天八卦（文王八卦）
后天八卦反映万物发展的规律，用于实际应用。

方位对应：离南、坎北、震东、兑西、巽东南、坤西南、艮东北、乾西北

## 二、出生时间与卦象

### 起卦方法（简易梅花易数）

#### 方法一：数理起卦
取出生年月日时的数字进行运算：
- 上卦 = (年 + 月 + 日) % 8，对应：1乾 2兑 3离 4震 5巽 6坎 7艮 0坤
- 下卦 = (年 + 月 + 日 + 时) % 8，同上对应
- 动爻 = (年 + 月 + 日 + 时) % 6，对应：1~6爻

#### 方法二：年支起卦
根据出生年地支直接对应卦象（简化法）：
- 子午卯酉 → 坎离震兑（四正卦）
- 寅申巳亥 → 乾坤巽艮（四维卦）
- 辰戌丑未 → 坤坤坤坤（四库，统归坤）

#### 方法三：时辰与卦象
根据出生时辰对应后天八卦方位：
- 子时（北）→ 坎卦 ☵ 水
- 丑时（东北偏）→ 艮卦 ☶ 山
- 寅时（东北）→ 艮卦 ☶ 山
- 卯时（东）→ 震卦 ☳ 雷
- 辰时（东南偏）→ 巽卦 ☴ 风
- 巳时（东南）→ 巽卦 ☴ 风
- 午时（南）→ 离卦 ☲ 火
- 未时（西南偏）→ 坤卦 ☷ 地
- 申时（西南）→ 坤卦 ☷ 地
- 酉时（西）→ 兑卦 ☱ 泽
- 戌时（西北偏）→ 乾卦 ☰ 天
- 亥时（西北）→ 乾卦 ☰ 天

- 卦辞：元亨利贞
- 起名启示：取名宜取刚健、正直、进取之意。如"健"、"正"、"元"、"亨"。
- 品德关联：仁、义、礼、智、信五常俱全。

- 卦辞：元亨，利牝马之贞

**15. 谦卦（地中有山）** — 谦虚受益，卑以自牧
- 卦辞：亨，君子有终

## 四、周易起名核心原则

### 1. 取象法
从卦象的自然意象中提取字词：
- 乾（天）：天、宇、穹、昊、宸、寰
- 坤（地）：坤、垚、壤、垣、域、疆
- 震（雷）：震、霆、雳、雷、轰
- 巽（风）：风、岚、飔、飒、飘
- 坎（水）：水、渊、泉、浩、澜、淼
- 离（火）：火、炎、烨、灿、煜、晖
- 艮（山）：山、峰、岳、峻、巍、崴
- 兑（泽）：泽、润、沛、滋、淼

### 2. 取意法
从卦辞、爻辞的哲学含义中提取：
- "自强不息" → 息、强、行、健
- "厚德载物" → 厚、德、载、物
- "谦受益" → 谦、益、受
- "含章可贞" → 章、贞、含

### 3. 取辞法
直接引用卦辞、爻辞中的优美文辞：
- "君子以自强不息" → 天行
- "含章可贞，或从王事" → 含章
- "文明以健，中正而应" → 文健
- "大亨以正，天之道也" → 大正
