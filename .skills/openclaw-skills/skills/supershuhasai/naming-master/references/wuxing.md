# 五行理论与八字推算

## 一、五行基础

五行：金、木、水、火、土，是中国古代哲学的核心理论体系。

### 五行特性
- **木**：生长、升发、条达、舒畅。东方，春季，青色，酸味。主仁。
- **火**：温热、升腾、明亮。南方，夏季，赤色，苦味。主礼。
- **土**：承载、生化、受纳。中央，长夏，黄色，甘味。主信。
- **金**：清洁、肃降、收敛。西方，秋季，白色，辛味。主义。
- **水**：寒凉、滋润、下行。北方，冬季，黑色，咸味。主智。

### 五行生克关系
- **相生**：木→火→土→金→水→木（母生子，子养母）
- **相克**：木→土→水→火→金→木（制约平衡）

### 五行旺衰判断
根据季节判断五行旺衰：
- 春季（寅卯月）：木旺，火相，水休，金囚，土死
- 夏季（巳午月）：火旺，土相，木休，水囚，金死
- 秋季（申酉月）：金旺，水相，土休，火囚，木死
- 冬季（亥子月）：水旺，木相，金休，土囚，火死
- 四季末（辰戌丑未月）：土旺，金相，火休，木囚，水死

## 三、年柱推算法

### 天干计算（公历年份）
年天干 = (公历年份 - 3) % 10，对应：1甲 2乙 3丙 4丁 5戊 6己 7庚 8辛 9壬 0癸

示例：2024年 → (2024-3)%10 = 1 → 甲

### 地支计算（公历年份）
年地支 = (公历年份 - 3) % 12，对应：1子 2丑 3寅 4卯 5辰 6巳 7午 8未 9申 10酉 11戌 0亥

示例：2024年 → (2024-3)%12 = 3 → 辰

### 注意事项
- 立春（约2月4日）前出生者，年柱按上一年计算
- 年柱以立春为界，非以农历正月初一为界

### 年上起月法（五虎遁）
甲己之年丙作首（正月起丙寅）
乙庚之年戊为头（正月起戊寅）
丙辛之岁寻庚上（正月起庚寅）
丁壬壬寅顺水流（正月起壬寅）
若问戊癸何处起（正月起甲寅）
甲寅之上好追求

即：年干为甲/己 → 月干从丙起；年干为乙/庚 → 月干从戊起；以此类推。

## 七、五行统计与平衡分析

### 八字五行统计
1. 排出八字四柱（年月日时）
2. 将每个天干地支转化为五行
3. 统计各五行出现次数
4. 考虑地支藏干的影响（本气权重最大）
5. 结合月令（出生月地支）判断日主旺衰

### 五行评分标准
- 某五行出现3次以上为"旺"
- 某五行出现1-2次为"平"
- 某五行未出现为"缺"
- 缺失的五行就是名字需要补益的方向

### 五行起名补益原则
1. **缺什么补什么**：最基本的原则，缺木补木、缺水补水
2. **喜用神原则**：需综合考虑日主旺衰，有时缺的未必喜用
3. **平衡为上**：不追求五种元素完全平均，但尽量不出现严重失衡
4. **生克协调**：补益的五行需与日主形成良好的生克关系

### 常见组合建议
- 日主为木且弱：喜水（生木）、木（比助）
- 日主为火且弱：喜木（生火）、火（比助）
- 日主为土且弱：喜火（生土）、土（比助）
- 日主为金且弱：喜土（生金）、金（比助）
- 日主为水且弱：喜金（生水）、水（比助）
