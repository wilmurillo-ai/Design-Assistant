## ✨ 核心功能

### 🎯 格式化输入系统
- **输入格式**: 时间 + 环境 + 互动对象 + 做了什么
- **示例**: "晚上，家里，一条蛇，在地板上爬行"

### ❓ 交互式 3 问引导
- 智能提问补充梦境细节
- 提高解读准确性

### 📊 双版本解读系统
- **传统版**: 基于周公解梦理论，含吉凶评级
- **科学版**: 基于现代心理学，含情绪分析

### 🔍 元素分析系统
- 1个主要元素 + 2个次要元素分析
- 数据库包含9个核心元素（蛇、房子、汽车、飞行、水、火、光、人等）

### 🍀 智能吉凶建议
- **大吉/吉**: 锦上添花建议
- **平/中吉**: 转吉之法 + 防止化凶
- **凶/厄**: 破解之法
