### 🇨🇳🇬🇧 中英双语支持
- 中文输入 → 中文解释
- English input → English interpretation
- 自动识别语言 (auto-detect)

- 传统周公解梦释义
- 现代心理学解读
- 吉凶预示（大吉/吉/中吉/平/凶/大凶）

- 如何把握机会
- 如何增强正面能量
- 风水布局建议

- 化解方法（饰品/风水/行为）
- 心理调适建议
- 实用可行的破解方案

- "我梦见蛇是什么意思？"
- "昨晚梦见掉牙齿"

## 调用脚本

```bash
# 基础解梦
python scripts/zhougong_dream.py --dream "蛇" --lang zh
python scripts/zhougong_dream.py --dream "water" --lang en
python scripts/zhougong_dream.py --dream "被追赶" --lang auto

# 带结局分析
python scripts/zhougong_dream.py --dream "飞行" --ending "飞得很高" --lang zh
python scripts/zhougong_dream.py --dream "falling" --ending "被接住" --lang en

# 批量查询
python scripts/zhougong_dream.py --batch dreams_list.txt --output report.md
```

## 吉凶等级说明

| 等级 | 英文 | 说明 | 建议 |
|------|------|------|------|
| 大吉 | Very Auspicious | 非常好的预兆 | 把握机会，大胆行动 |
| 吉 | Good | 好的预兆 | 积极推进，注意细节 |
| 中吉 | Medium Good | 中等偏上 | 谨慎乐观，适当行动 |
| 平 | Neutral | 平稳无波 | 保持现状，不宜大动 |
| 凶 | Bad | 需注意 | 小心谨慎，做好防范 |
| 大凶 | Very Bad | 严重警告 | 避免冒险，寻求化解 |
