# Stock Watcher Pro - 股票盯盯智能监控系统

股票盯盯是一款专业的单支股票深度监控工具，专注于长期跟踪单支股票的走势和操盘手行为模式。通过持续监控，系统会逐渐了解该股票的特性，包括操盘手习惯、支撑点、压力位等关键信息。

## 🚀 快速安装

```bash
# 使用 ClawHub CLI 安装
clawhub install stock-watcher-cn

# 或在 OpenClaw 中加载技能
openclaw skills load stock-watcher-cn
```

## 🔥 核心功能

### 🎯 七大预警规则
1. **成本百分比预警**: 基于持仓成本的盈亏提醒
2. **单日涨跌幅预警**: 个股/ETF/黄金差异化阈值
3. **成交量异动预警**: 基于5日均量的放量/缩量检测
4. **均线金叉死叉预警**: MA5/MA10/MA20趋势判断
5. **RSI超买超卖预警**: 70/30阈值判断
6. **跳空缺口预警**: 向上/向下跳空检测
7. **动态止盈止损预警**: 移动止损保护利润

### 🧠 操盘手习惯分析
- **时间偏好分析**: 识别操盘手在一天中的活跃时段
- **价格行为模式**: 学习操盘手的价格操作习惯
- **成交量行为**: 分析操盘手的成交量操作模式
- **支撑阻力位学习**: 自动识别和学习关键价位
- **个性化预警**: 基于操盘手习惯的智能预警

### 📱 分级预警系统
- **🚨 紧急级**: 多条件共振 (如: 放量+均线金叉+突破成本)
- **⚠️ 警告级**: 2个条件触发 (如: RSI超卖+放量)
- **📢 提醒级**: 单一条件触发

### ⏰ 智能频率控制
- **交易时间**: 每5分钟监控
- **午休时间**: 每10分钟监控
- **收盘后**: 每30分钟监控
- **凌晨**: 每小时监控（仅伦敦金）

## 📦 安装使用

### 依赖安装
```bash
pip install requests
```

### 配置监控列表
编辑 `scripts/monitor.py` 中的 `WATCHLIST` 数组，配置你的关注股票：

```python
WATCHLIST = [
    {
        "code": "300383",           # 股票代码
        "name": "光环新网",         # 股票名称
        "market": "sz",             # 市场 (sh/sz/fx)
        "type": "individual",       # 类型 (individual/etf/gold)
        "cost": 17.00,              # 持仓成本
        "focus": True,              # 重点关注
        "alerts": {
            "cost_pct_2above": 15.0,   # 盈利15%提醒
            "cost_pct_below": -12.0,   # 亏损12%止损
            "change_pct_above": 4.0,   # 日内大涨4%预警
            "change_pct_below": -4.0,  # 日内大跌4%预警
            "volume_surge": 3.0        # 3倍放量预警
        }
    }
]
```

### 飞书推送配置
编辑 `scripts/monitor.py` 中的 `feishu_target`：
```python
self.feishu_target = "your_feishu_user_id"  # 替换为你的飞书用户 ID
```

### 启动监控
```bash
cd scripts
python3 monitor.py
```

### 后台运行
```bash
./control.sh start     # 启动
./control.sh status    # 查看状态
./control.sh log       # 查看日志
./control.sh stop      # 停止
```

## 📈 标的类型差异化

| 类型 | 日内异动阈值 | 成交量阈值 | 适用标的 |
|------|-------------|-----------|----------|
| individual (个股) | ±4% | 2倍 | 江西铜业、中国平安 |
| etf (ETF) | ±2% | 1.8倍 | 恒生医疗、创50等 |
| gold (黄金) | ±2.5% | 无 | 伦敦金 |

## 🎨 特色亮点

- **符合中国习惯**: 红色表示上涨/盈利，绿色表示下跌/亏损
- **智能频率**: 根据交易时段自动调整监控频率
- **多源数据**: 东方财富、新浪财经 API
- **飞书集成**: 通过 OpenClaw message 工具推送预警
- **操盘手学习**: 智能学习股票交易习惯
- **分级预警**: 避免过度打扰，重要信号优先推送

## ⚠️ 注意事项

1. **技术指标有滞后性**: 均线、MACD等都是滞后指标，用于确认趋势而非预测
2. **避免过度交易**: 预警只是参考，不要每个信号都操作
3. **多条件共振更可靠**: 单一指标容易假信号，多条件共振更准确
4. **动态止盈要灵活**: 回撤5%减仓、10%清仓是建议，根据市场灵活调整

**核心原则**: 
> 预警系统目标是"不错过大机会，不犯大错误"，不是"抓住每一个波动"。
