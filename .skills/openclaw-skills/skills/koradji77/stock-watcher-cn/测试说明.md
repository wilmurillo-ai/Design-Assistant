## 🚀 快速开始

### 1. 安装到 OpenClaw

```bash
# 将技能目录复制到 OpenClaw 工作区
xcopy "E:\股票盯盯-v4.0.0" "C:\Users\Administrator\.openclaw\workspace\skills\stock-watcher" /E /Y

# 或在 OpenClaw 中加载技能
openclaw skills load stock-watcher
```

### 2. 配置关注股票

编辑 `scripts/monitor.py`，修改 `WATCHLIST` 配置：

```python
WATCHLIST = [
    {
        "code": "300383",           # 股票代码
        "name": "光环新网",         # 股票名称
        "market": "sz",             # 市场 (sh/sz/fx)
        "type": "individual",       # 类型 (individual/etf/gold)
        "cost": None,               # 持仓成本（可选，null 表示不监控盈亏）
        "focus": True,              # 是否重点关注（true=预警，false=不预警）
        "alerts": {
            "cost_pct_above": 15.0,   # 盈利 15% 提醒
            "cost_pct_below": -12.0,  # 亏损 12% 止损
            "change_pct_above": 4.0,  # 日内异动 ±4%
            "change_pct_below": -4.0,
            "volume_surge": 3.0       # 成交量 3 倍放量预警
        }
    }
]
```

### 3. 配置飞书推送（可选）

编辑 `scripts/monitor.py`，修改 `feishu_target`：

```python
self.feishu_target = "ou_xxxxx"  # 飞书用户 ID
```

## ⏰ Cron 定时任务配置

### 已有任务列表

| 任务名称 | Cron 表达式 | 执行时间 | 推送 |
|---------|------------|---------|------|
| 股票盯盘 - 上午 | `0,30 9-11 * * 1-5` | 9:00, 9:30, 10:00, 10:30, 11:00 | ✅ |
| 股票盯盘 - 上午总结 | `35 11 * * 1-5` | 11:35 | ✅ |
| 股票盯盘 - 午后 | `30 13 * * 1-5` | 13:30 | ✅ |
| 股票盯盘 - 下午 | `0 14 * * 1-5` | 14:00 | ✅ |
| 股票盯盘 - 午后第二次 | `30 14 * * 1-5` | 14:30 | ✅ |
| 股票盯盘 - 收盘总结 | `50 14 * * 1-5` | 14:50 | ❌（只缓存） |
| 股票盯盘 - 收盘总结推送 | `0 15 * * 1-5` | 15:00 | ✅ |

### 添加 Cron 任务

```bash
# 盘中监控（每 30 分钟）
openclaw cron add \
  --name "股票盯盘 - 盘中监控" \
  --cron "0,30 9-14 * * 1-5" \
  --message "cd C:\\Users\\Administrator\\.openclaw\\workspace\\skills\\stock-monitor-skill\\scripts && python monitor.py" \
  --tz "Asia/Shanghai"

# 14:50 预收集
openclaw cron add \
  --name "股票盯盘 - 收盘预收集" \
  --cron "50 14 * * 1-5" \
  --message "cd C:\\Users\\Administrator\\.openclaw\\workspace\\skills\\stock-monitor-skill\\scripts && python close_summary.py pre_close" \
  --no-deliver \
  --tz "Asia/Shanghai"

# 15:00 收盘总结推送
openclaw cron add \
  --name "股票盯盘 - 收盘总结推送" \
  --cron "0 15 * * 1-5" \
  --message "cd C:\\Users\\Administrator\\.openclaw\\workspace\\skills\\stock-monitor-skill\\scripts && python close_summary.py close_summary" \
  --tz "Asia/Shanghai"
```

## 📊 功能特性

### 1. 七大数据预警

| 预警类型 | 说明 |
|---------|------|
| 💰 成本百分比 | 盈利/亏损达到设定百分比 |
| 📈 单日涨跌幅 | 个股±4%、ETF±2%、黄金±3% |
| 📊 成交量异动 | 放量/缩量达到 5 日均量的 N 倍 |
| 🌟 均线金叉/死叉 | MA5/MA10/MA20 趋势判断 |
| 🔥 RSI 超买/超卖 | 70/30 阈值判断 |
| ⬆️ 跳空缺口 | 向上/向下跳空检测 |
| 📉 动态止盈止损 | 移动止损保护利润 |

### 2. 操盘手习惯分析

- ⏰ 时间偏好分析（识别活跃时段）
- 📊 成交量行为模式
- 📈 价格行为习惯
- 🎯 支撑阻力位学习

### 3. 分级预警系统

| 级别 | 触发条件 | 推送 |
|------|---------|------|
| 🚨 紧急 | ≥3 个预警条件 | ✅ |
| ⚠️ 警告 | 2 个预警条件 | ✅ |
| 📢 提醒 | 1 个预警条件 | ✅ |

### 4. 两阶段收盘总结

- **14:50** 预收集数据（缓存，不推送）
- **15:00** 生成总结并推送（最终版）

---

## ⚠️ 注意事项

1. **时区设置**: 所有 Cron 任务使用 `Asia/Shanghai` 时区
2. **关注列表**: 只有 `focus: True` 的股票才会被监控
3. **推送频率**: 同类预警 30 分钟内只发一次
4. **周末模式**: 周六日只监控伦敦金（A 股休市）
5. **13:00 不推送**: 午后开盘不推送（避免骚扰）
