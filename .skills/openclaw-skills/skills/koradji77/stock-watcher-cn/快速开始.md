# 股票盯盯技能快速开始指南

## 安装依赖
```bash
pip install requests
```

## 配置你的关注列表
编辑 `scripts/monitor.py` 中的 `WATCHLIST` 部分：
```python
WATCHLIST = [
    {
        "code": "300383",           # 股票代码
        "name": "光环新网",         # 股票名称
        "market": "sz",             # 市场 (sh/sz/fx)
        "type": "individual",       # 类型 (individual/etf/gold)
        "cost": 17.00,              # 你的持仓成本（可选）
        "focus": True,              # 重点关注（true=预警，false=不预警）
        "alerts": {
            "cost_pct_above": 15.0,   # 盈利15%提醒
            "cost2_pct_below": -12.0,  # 亏损12%止损
            "change_pct_above": 4.0,  # 日内大涨4%预警
            "change_pct_below": -4.0, # 日内大跌4%预警
            "volume_surge": 3.0       # 3倍放量预警（操盘手异动）
        }
    }
]
```

## 配置飞书推送目标
编辑 `scripts/monitor.py` 中的 `feishu_target`：
```python
self.feishu_target = "your_feishu_user_id"  # 替换为你的飞书用户 ID
```

## 运行测试
```bash
cd scripts
python3 monitor.py --test
```

## 后台运行
```bash
./control.sh start     # 启动
./control.sh status    # 查看状态
./control.sh log       # 查看日志
./control.sh stop      # 停止
```

## 定时任务设置（建议）
使用 OpenClaw cron 功能设置定时监控：
```bash
openclaw cron add \
  --name "股票盯盘 - 盘中监控" \
  --cron "0,30 9-14 * * 1-5" \
  --message "cd /root/.openclaw/workspace/skills/stock-watcher/scripts && python3 monitor.py" \
  --tz "Asia/Shanghai"
```

## 重要提醒
1. **股票代码需要正确**：A股代码（如300383），市场对应正确（sh/sz）
2. **持仓成本很重要**：如果你设置了成本价，系统会计算盈亏百分比
3. **飞书推送需要配置**：确保你的飞书 ID 正确
4. **时间设置**：系统基于北京时间，无需调整
