# 股票盯盯 (Stock Watcher) - 详细使用说明

## 📖 简介

股票盯盯是一款专业的股票监控预警系统，专门为长期跟踪单支股票走势和操盘手行为模式设计。它通过持续监控，逐渐了解股票的特性，包括操盘手习惯、支撑点、压力位等关键信息。

## 🚀 快速安装

### 方法一：使用 ClawHub CLI（推荐）
```bash
# 安装技能
clawhub install stock-watcher-cn

# 进入技能目录
cd ~/.openclaw/workspace/skills/stock-watcher-cn
```

### 方法二：手动安装
1. 下载技能包或从 Git 仓库克隆
2. 将 `stock-watcher` 文件夹复制到 OpenClaw 工作区的 `skills` 目录
3. 安装 Python 依赖：
   ```bash
   pip install requests
   ```

## 📝 配置指南

### 第一步：修改监控列表
复制 `config-example.py` 到 `scripts/config.py`，然后编辑：

```python
WATCHLIST = [
    {
        "code": "600362",           # 股票代码（必填）
        "name": "江西铜业",         # 股票名称（必填）
        "market": "sh",             # 市场：上证(sh) / 深证(sz) / 外汇(fx)
        "type": "individual",       # 类型：个股(individual) / ETF(etf) / 黄金(gold)
        "cost": 57.00,              # 持仓成本价（元，可选）
        "focus": True,              # 重点关注（True=监控，False=忽略）
        "alerts": {
            "cost_pct_above": 15.0,   # 盈利百分比提醒（例：盈利15%）
            "cost_pct_below": -12.0,  # 亏损百分比止损（例：亏损12%）
            "change_pct_above": 4.0,  # 日内大涨阈值（例：4%）
            "change_pct_below": -4.0, # 日内大跌阈值（例：-4%）
            "volume_surge": 2.0,      # 成交量异动倍数（例：2倍）
            "ma_monitor": True,       # 均线金叉死叉预警（默认True）
            "rsi_monitor": True,      # RSI超买超卖预警（默认True）
            "gap_monitor": True,      # 跳空缺口预警（默认True）
            "trailing_stop": True    # 动态止盈预警（默认True）
        }
    }
]
```

### 第二步：配置飞书推送
```python
# 在 scripts/monitor.py 中找到 feishu_target
self.feishu_target = "your_feishu_user_id"  # 替换为你的飞书用户ID
```

**获取飞书用户ID方法**：
1. 打开飞书开发者工具或API文档
2. 查找你的用户ID（格式如：ou_78f9e41a60baa905e5c2fe8178216ee6）

### 第三步：配置智能频率（可选）
```python
SMART_SCHEDULE = {
```

## 🔔 预警级别说明

### 🚨 紧急级（红色）
**触发条件：** 3个或更多预警条件同时触发
**示例：**
```
🚨【紧急】🔴 江西铜业 (600362)
━━━━━━━━━━━━━━━━━━━━
💰 当前价格：¥65.50 (+15.0%)
📊 持仓成本：¥57.00 | 盈亏：🔴+14.9%

🎯 触发预警 (3项):
  • 🎯 盈利 15% (目标价 ¥65.55)
  • 🌟 均线金叉 (MA5¥63.2上穿MA10¥62.8)
  • 📊 放量 2.5倍 (5日均量)
```

### ⚠️ 警告级（橙色）
**触发条件：** 2个预警条件触发
**示例：**
```
⚠️【警告】🟢 恒生医疗 (159892)
━━━━━━━━━━━━━━━━━━━━
💰 当前价格：¥0.72 (-10.0%)
📊 持仓成本：¥0.80 | 盈亏：🟢-10.0%

🎯 触发预警 (2项):
  • 📉 日内大跌 -10.0%
  • ❄️ RSI超卖 (28.5)，可能反弹
```

### 📢 提醒级（蓝色）
**触发条件：** 单一条件触发
**示例：**
```
📢【提醒】🔴 中国平安 (601318)
━━━━━━━━━━━━━━━━━━━━
💰 当前价格：¥70.50 (+6.8%)
📊 持仓成本：¥66.00 | 盈亏：🔴+6.8%

🎯 触发预警：
  • 📉 利润回撤 5.2%，建议减仓保护利润
  (最高盈利12%已回撤)
```

## ⏰ 智能频率控制

### 交易时段安排
| 时间段 | 监控频率 | 监控标的 |
|--------|----------|----------|
| 交易时间 (9:30-11:30, 13:00-15:00) | 每5分钟 | 全部+技术指标 |
| 午休时间 (11:30-13:00) | 每10分钟 | 全部 |
| 收盘后 (15:00-24:00) | 每30分钟 | 全部 (日线数据) |
| 凌晨 (0:00-9:30) | 每1小时 | 仅伦敦金 |
| 周末 | 每1小时 | 仅伦敦金 |

### 防骚扰机制
- **同类预警30分钟内只发一次**：避免重复提醒
- **收盘后简化监控**：只检查日线数据
- **周末模式**：A股休市，只监控伦敦金

## 🔧 高级配置

### 操盘手习惯分析
系统会自动学习操盘手的行为模式：
- **时间偏好**：识别活跃时段
- **价格行为**：学习价格操作习惯
- **成交量行为**：分析成交量操作模式
- **支撑阻力位**：自动识别关键价位

### 自定义预警阈值
```python
"alerts": {
    "cost_pct_above": 20.0,     # 盈利20%提醒（激进型）
    "cost_pct_below": -15.0,    # 亏损15%止损（保守型）
    "change_pct_above": 5.0,    # 日内异动±5%（敏感型）
    "volume_surge": 3.0,       # 放量3倍提醒（灵敏度高）
    "ma_monitor": False,      # 关闭均线预警
    "rsi_monitor": False,     # 关闭RSI预警
    "gap_monitor": True,      # 开启跳空缺口预警
    "trailing_stop": True     # 开启动态止盈
}
```

## 💡 使用技巧

### 1. 成本价设置技巧
- **真实持仓**：输入你的实际持仓成本价
- **模拟持仓**：如果你想关注某支股票但未持仓，设置 `cost: None`
- **分批建仓**：如果分批买入，可以用平均成本

### 2. 预警阈值优化
- **新手投资者**：阈值设置宽一些（±6%~±8%）
- **经验投资者**：阈值设置窄一些（±3%~±4%）
- **日内交易者**：重点关注成交量异动和日内涨跌幅

### 3. 多股票监控策略

## 🛠️ 故障排查

### 1. 无法获取股票数据
**解决方案：**
- 检查股票代码是否正确
- 确认市场代码匹配（上证：sh，深证：sz）
- 确认网络连接正常
- 股票代码示例：
  - 上证：600362（江西铜业）
  - 深证：300383（光环新网）
  - ETF：159892（恒生医疗）
  - 黄金：hf_XAU（伦敦金）

### 2. 飞书推送不生效
**解决方案：**
- 检查 `feishu_target` 是否正确
- 确认 OpenClaw 已配置飞书插件
- 检查飞书用户 ID 格式（ou_xxxxxx）
- 测试飞书推送功能：
  ```python
  # 在 scripts/monitor.py 中添加测试代码
  self.send_feishu_alert("测试消息", "这是一个测试")
  ```

### 3. 预警频率过高
**解决方案：**
- 调整 SMART_SCHEDULE 中的间隔时间
- 提高预警阈值
- 关闭部分预警规则（如 `ma_monitor: False`）
- 设置 `focus: False` 忽略某些股票

### 4. 技术指标不更新
**解决方案：**
- 确保在交易时间运行（技术指标需要实时数据）
- 收盘后只提供日线数据，技术指标不更新
- 周末技术指标暂停更新

## 📈 最佳实践

### 1. 监控策略
- **关注3-5支核心股票**：不要贪多，专注核心标的
- **设置合理的预警阈值**：根据个人风险偏好调整
- **结合其他分析工具**：股票盯盯是预警工具，不是决策工具

### 2. 投资建议
- **多条件共振更可靠**：多个指标同时触发时信号更强
- **不要过度交易**：预警只是参考，不要每个信号都操作
- **动态止盈要灵活**：回撤5%减仓、10%清仓是建议，根据市场灵活调整

### 3. 组合使用
**股票盯盯 + 股票筛选器**：
- **筛选器**：帮你找到值得关注的股票
- **盯盯**：长期跟踪你选定的股票
这是一个完整的投资工具组合。
