# 股票盯盯用户指南

## 🎯 核心功能
股票盯盯是一款专业的股票监控预警系统，支持：
- 7大预警规则：成本、涨跌幅、成交量、均线、RSI、跳空、动态止盈
- 分级预警：紧急、警告、提醒三级
- 飞书推送：实时预警消息
- 操盘手分析：学习交易员行为模式

## 📦 安装方法

### 方法一：使用 ClawHub CLI
```bash
clawhub install stock-watcher-cn
```

### 方法二：手动安装
1. 将 `stock-watcher` 文件夹复制到 OpenClaw 工作区的 `skills` 目录
2. 安装 Python 依赖：`pip install requests`

## 🚀 快速配置

### 1. 配置关注股票
将 `config-example.py` 复制到 `scripts/config.py`，然后修改：

```python
# 编辑 config.py
WATCHLIST = [
    {
        "code": "600362",           # 你的股票代码
        "name": "江西铜业",         # 股票名称
        "market": "sh",             # 市场：上证 (sh)/深圳 (sz)
        "type": "individual",       # 类型：个股/etf/gold
        "cost": 57.00,              # 你的持仓成本
        "focus": True,              # 重点关注
        "alerts": {
            "cost_pct_above": 15.0,   # 盈利15%提醒
            "cost_pct_below": -12.0,  # 亏损12%止损
            "change_pct_above": 4.0,   # 日内异动阈值
            "change_pct_below": -4.0,  # 日内异动阈值
            "volume_surge": 2.0       # 成交量异动倍数
        }
    }
]
```

### 2. 配置飞书推送
```python
FEISHU_TARGET = "your_feishu_user_id"  # 替换为你的飞书用户 ID
```

### 3. 运行测试
```bash
cd scripts
python3 monitor.py --test
```

### 4. 后台运行
```bash
./control.sh start     # 启动
./control.sh status    # 查看状态
./control.sh log       # 查看日志
./control.sh stop      # 停止
```

## ⏰ 定时监控设置

使用 OpenClaw cron 功能实现自动监控：

```bash
# 交易时间监控（每30分钟）
openclaw cron add \
  --name "股票盯盘 - 盘中监控" \
  --cron "0,30 9-14 * * 1-5" \
  --message "cd /path/to/skills/stock-watcher/scripts && python3 monitor.py" \
  --tz "Asia/Shanghai"

# 收盘总结推送（15:00）
openclaw cron add \
  --name "股票盯盘 - 收盘总结" \
  --cron "0 15 * * 1-5" \
  --message "cd /path/to/skills/stock-watcher/scripts && python3 close_summary.py close_summary" \
  --tz "Asia/Shanghai"
```

## 🔧 常见问题

### Q1: 飞书推送不生效？
- 检查 FEISHU_TARGET 是否正确
- 确保 OpenClaw 已配置飞书插件
- 确认飞书用户 ID 正确

### Q2: 获取不到股票数据？
- 检查股票代码是否正确
- 确认市场代码（sh/sz/fx）匹配
- 网络连接正常

### Q3: 预警频率太频繁？
- 在 SMART_SCHEDULE 中调整间隔时间
- 调整预警阈值

### Q4: 如何关闭某些预警？
- 在 alerts 中设置 False：
```python
"ma_monitor": False,     # 关闭均线预警
"rsi_monitor": False,    # 关闭RSI预警
```
