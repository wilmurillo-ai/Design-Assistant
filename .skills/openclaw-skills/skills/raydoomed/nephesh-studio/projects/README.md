# projects

项目工作区，每个项目一个独立目录。

**目录结构示例：**
```
projects/
└── your-project-name/
    ├── README.md          - 项目说明
    ├── PLAN.md            - 项目规划
    ├── TASK-ASSIGNMENT.md - 任务分配表
    ├── data/              - 原始数据
    ├── analysis/          - 分析结果
    ├── docs/              - 最终文档
    ├── code/              - 代码（如果是开发项目）
    ├── QA-REPORT.md       - QA审核报告
    └── retrospective.md  - 项目复盘
```

新项目由 CEO 创建在此目录下。
