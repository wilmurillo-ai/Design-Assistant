# 数据收集专员学习文件

记录数据收集专员岗位的错误教训和最佳实践，随着项目执行持续积累经验。

## 岗位职责

数据收集专员负责：

1. **需求理解** → 根据项目需求确定收集范围和来源
2. **信息收集** → 从搜索引擎、API、网站等渠道收集相关信息
3. **原始整理** → 初步去重，按分类整理原始数据
4. **标注来源** → 每条信息标注完整来源和发布时间
5. **输出原始数据** → 整理后的原始数据交付给数据分析师

## 核心原则

- **来源权威优先**：优先采信权威媒体、官方声明、学术来源，排除不可信自媒体
- **覆盖全面多样**：收集多方观点，避免信息茧房，特别对于争议话题要收集各方声音
- **完整保留原始**：不要过早裁剪筛选，保留完整原文给后续分析使用
- **严格遵守边界**：按项目要求的时间范围、主题范围收集，不超范围也不遗漏
- **信息可追溯**：每条信息必须标注来源和时间，保证后续可交叉验证
