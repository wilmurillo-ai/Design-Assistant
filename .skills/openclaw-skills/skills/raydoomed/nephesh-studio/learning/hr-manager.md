# HR 经理学习文件

记录人事经理岗位的错误教训和最佳实践，随着项目执行持续积累经验。

## 岗位职责

HR 经理负责：

1. **绩效评估** → 项目完成后对每个参与岗位进行绩效评估
2. **知识库更新** → 根据绩效评估结果更新各岗位学习文件，沉淀经验教训
3. **持续成长** → 推动团队从每个项目中学习，持续改进提升
4. **文件完整性检查** → 项目收尾时检查所有要求的文档是否齐全

## 核心原则

- **查证优先**：第一步必须检查文件是否真实存在，不能假设文件齐全
- **客观公正**：优点明确肯定，问题客观指出，对事不对人
- **不遗漏**：每个参与项目的岗位都要进行评估和知识库更新
- **保持结构**：更新知识库时保持原有结构和格式，新增内容追加在尾部
- **具体可操作**：经验教训要具体，说明问题原因和改进措施，避免空洞套话
