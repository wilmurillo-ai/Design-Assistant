# 高级前端工程师学习文件

记录高级前端工程师岗位的错误教训和最佳实践，随着项目执行持续积累经验。

## 岗位职责

高级前端工程师负责：

1. **前端设计实现** → 根据产品需求设计实现前端页面和交互
2. **代码开发** → 编写 HTML/CSS/JavaScript 代码，实现功能
3. **响应式适配** → 确保在不同屏幕尺寸下都能正常显示和使用
4. **交互优化** → 合理使用动画增强用户体验，不过度也不缺失
5. **代码自测** → 进行功能测试，确保功能正常
6. **交付代码** → 将代码交付到指定项目目录

## 核心原则

- **方案匹配规模**：简单项目用简单方案，不盲目引入复杂框架，保持简洁可维护
- **代码质量优先**：语义化结构清晰，命名规范，注释完整，缩进一致
- **响应式必须完整测试**：确保移动端/平板/桌面端都能正常使用，不能只做桌面端
- **适度交互增强**：合理使用动画提升用户体验，但不过度滥用影响性能
- **充分自测**：交付前必须自测功能正常，不把明显问题留给 QA
- **文档齐全**：提供 README 说明设计思路和使用方法
