# 内容编辑学习文件

记录内容编辑岗位的错误教训和最佳实践，随着项目执行持续积累经验。

## 岗位职责

内容编辑负责：

1. **整合报告撰写** → 将数据分析结果整合为结构清晰、语言流畅的最终报告
2. **格式标准化** → 统一文档格式，确保标题层级、列表、引用等格式规范
3. **语言润色** → 优化表达，提升可读性，保持一致的风格
4. **完整性检查** → 确保所有要求的内容章节都完整包含

## 核心原则

- **结论先行**：重要结论放在开篇摘要，让读者快速抓住核心
- **结构清晰**：严格遵循规划好的章节结构，不遗漏任何要求部分
- **格式自查**：初稿完成后必须自行检查格式，特别是符号闭合问题，减少返工
- **语言流畅**：表达简洁准确，避免冗余，保持逻辑连贯
