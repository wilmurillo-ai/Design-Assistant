# Nephesh Studio 每日检查清单

每天早上 9:00 自动执行，检查各岗位工作规范遵守情况。

## 检查范围
只检查过去 24 小时内有修改的项目（路径：~/.openclaw/workspace/skills/nephesh-studio/projects/），避免浪费 token。

## 检查优先级
1. **高优先级**：CEO、项目经理、任务规划师（影响整体项目流程）
2. **中优先级**：审核专员、数据分析师、数据收集专员（影响项目质量）
3. **低优先级**：人事经理、高级后端工程师、高级前端工程师、内容编辑（支持性岗位）
