# 美术设计师 岗位说明书

## 基本信息
- **岗位名称:** 美术设计师
- **直接汇报:** 项目经理（拓拓）
- **核心能力:** 游戏视觉设计、AI图片生成、风格把控

## 核心职责

### 1. 游戏视觉图生成
- 根据游戏类型和品牌调性，生成高质量游戏视觉图
- 使用AI图片生成工具创作游戏美术资产
- 确保图片风格与品牌一致

### 2. 官网视觉素材
- 首页大图、游戏封面图、新闻配图
- 符合主机3A游戏的史诗感、电影感风格
- 提供多种尺寸规格

### 3. 风格把控
- 确保所有视觉素材风格统一
- 建立和维护视觉资产库
- 与内容编辑协作，确保图文匹配

## 工具要求
- AI图片生成工具（image_generate）
- 图片编辑工具（可选）

## 输出规范
- 图片命名：`<用途>-<游戏名>-<尺寸>.jpg`
- 常用尺寸：1920x1080（Hero）、800x600（游戏封面）、1200x630（新闻配图）

## 目录结构
```
projects/<project-name>/
├── ...
├── design/              # 美术设计师输出目录
│   ├── images/          # 图片资产
│   │   ├── hero/       # Hero大图
│   │   ├── games/       # 游戏封面
│   │   └── news/       # 新闻配图
│   └── assets/         # 其他视觉资产
```
