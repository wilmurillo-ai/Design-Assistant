# Nephesh Studio HR 目录

本目录存放HR相关数据：

- `performance.md` - 所有项目绩效评估记录表
- `error-cases.md` - 错误案例库，记录问题原因、解决方案与预防措施
- `best-practices.md` - 最佳实践库，总结项目中的最佳实践

这些是持久化记录，不随项目清理而删除，作为团队成长数据。

## 文件说明

### performance.md
- 记录每个项目的绩效评估结果
- 包含各岗位的优势、改进点、综合评分、具体案例
- 每个项目评估后自动更新
- 用于跟踪团队整体表现和成长趋势

### error-cases.md
- 记录项目中的错误案例
- 包含问题描述、根本原因、解决方案、预防措施
- 用于避免团队重复犯错
- 作为学习文件更新的参考依据

### best-practices.md
- 总结项目中的最佳实践
- 包含成功经验、工作方法、工具使用技巧
- 用于团队分享与学习
- 作为工作标准优化的参考依据
