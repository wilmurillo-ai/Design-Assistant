# Story Relay - 故事接龙

多人协作创作故事，AI智能续写，支持图文生成。让创作变得有趣！

## 功能特性

| 功能 | 说明 |
|------|------|
| ✍️ **多人协作** | 支持多人共同创作一个故事 |
| 🤖 **AI续写** | AI自动续写，保持故事连贯性 |
| 🎨 **图文生成** | 支持生成插图，增强视觉效果 |
| 📚 **故事管理** | 支持多个故事并行创作 |
| 💾 **自动保存** | 实时保存，随时继续 |
| 🎯 **主题设定** | 可设定故事主题、风格、角色 |

## 快速开始

### 1. 开始新故事

```
开始故事：科幻小说，主题是AI觉醒
```

AI会生成故事开头，你可以续写或让AI继续。

### 2. 续写故事

```
续写：主角突然发现了一个隐藏文件
```

AI会将你的内容融入故事，并保持连贯性。

### 3. AI自动续写

```
继续故事
```

AI会自动续写下一章。

### 4. 查看故事

```
查看故事
```

显示当前故事进度。

### 5. 列出所有故事

```
故事列表
```

显示所有正在创作的故事。

### 6. 完成故事

```
完成故事
```

故事标记为完成，生成最终版本。

## 最佳实践

1. **设定清晰主题** - 开始前明确故事类型和风格
2. **保持连贯性** - 续写时注意前后逻辑
3. **善用AI续写** - 卡壳时让AI帮忙
4. **定期保存** - 虽然自动保存，但重要节点可主动查看
5. **多人协作** - 可以邀请朋友一起续写同一个故事

## 常见问题

**Q: 可以同时创作多个故事吗？**

A: 可以！使用"故事列表"查看所有故事，用"切换故事：ID"切换。

**Q: AI续写会改变我的设定吗？**

A: 不会。AI会严格遵守你设定的主题、风格、角色、场景。

**Q: 如何删除故事？**

A: 使用"删除故事：ID"命令。

**Q: 支持导出故事吗？**

A: 目前支持JSON格式，未来会支持PDF/EPUB。

**Q: 可以邀请朋友一起创作吗？**

A: 可以！把故事ID分享给朋友，他们也可以续写同一故事。
