# Story Relay - 故事接龙

多人协作创作故事，AI智能续写，支持图文生成。

## 功能亮点

✅ **多人协作** - 支持多人共同创作一个故事
✅ **AI续写** - AI自动续写，保持故事连贯性
✅ **图文生成** - 支持生成插图，增强视觉效果
✅ **故事管理** - 支持多个故事并行创作
✅ **自动保存** - 实时保存，随时继续
✅ **主题设定** - 可设定故事主题、风格、角色

## 快速开始

### 1. 开始新故事

```bash
cd /root/.openclaw/workspace/skills/story-relay
node index.js start 科幻小说，AI觉醒
```

AI会生成故事开头。

### 2. 续写故事

```bash
node index.js write 主角突然发现了一个隐藏文件
```

AI会将你的内容融入故事。

### 3. AI自动续写

```bash
node index.js continue
```

AI会自动续写下一章。

### 4. 查看故事

```bash
node index.js view
```

显示当前故事进度。

### 5. 列出所有故事

```bash
node index.js list
```

显示所有正在创作的故事。

## 命令列表

| 命令 | 说明 | 示例 |
|------|------|------|
| `start <主题>` | 开始新故事 | `start 科幻冒险` |
| `continue` | AI续写 | `continue` |
| `write <内容>` | 续写内容 | `write 主角发现秘密` |
| `view` | 查看当前故事 | `view` |
| `list` | 列出所有故事 | `list` |
| `switch <ID>` | 切换故事 | `switch STORY-001` |
| `complete` | 完成故事 | `complete` |
| `delete <ID>` | 删除故事 | `delete STORY-001` |
| `set-style <风格>` | 设置默认风格 | `set-style 悬疑` |

## 最佳实践

1. **设定清晰主题** - 开始前明确故事类型和风格
2. **保持连贯性** - 续写时注意前后逻辑
3. **善用AI续写** - 卡壳时让AI帮忙
4. **定期保存** - 虽然自动保存，但重要节点可主动查看
5. **多人协作** - 可以邀请朋友一起续写同一个故事
