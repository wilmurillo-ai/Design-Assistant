# Personal RPG - 个人RPG系统

将你的日常变成游戏，完成任务获得经验值！

## 功能亮点

✅ **游戏化** - 日常任务转化为RPG任务
✅ **经验系统** - 完成任务获得XP
✅ **等级系统** - XP积累升级，解锁新能力
✅ **成就系统** - 完成特定目标解锁成就
✅ **属性系统** - 智力、力量、敏捷、创造力
✅ **数据统计** - 追踪任务完成情况

## 快速开始

### 1. 查看角色

```bash
cd /root/.openclaw/workspace/skills/personal-rpg
node index.js view 角色
```

### 2. 添加任务

```bash
node index.js add 完成作业,中等,50
```

### 3. 完成任务

```bash
node index.js complete TASK-001
```

### 4. 查看升级进度

```bash
node index.js upgrade
```

## 命令列表

| 命令 | 说明 | 示例 |
|------|------|------|
| `view [角色|任务|成就]` | 查看信息 | `view 角色` |
| `add <描述>,<难度>,<XP>` | 添加任务 | `add 完成作业,中等,50` |
| `complete <ID>` | 完成任务 | `complete TASK-001` |
| `list` | 任务列表 | `list` |
| `delete <ID>` | 删除任务 | `delete TASK-001` |
| `stats` | 数据统计 | `stats` |
| `upgrade` | 升级进度 | `upgrade` |

## 最佳实践

2. **保持连续性** - 连续完成任务获得额外奖励
3. **多样化任务** - 平衡学习、运动、日常任务
4. **定期查看** - 查看升级进度和统计
5. **追求成就** - 完成成就获得额外XP
