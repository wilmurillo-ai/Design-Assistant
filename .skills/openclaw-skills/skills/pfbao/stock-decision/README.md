# Stock Decision Skill

**股票买入决策技能 - 技术分析 + 宏观环境综合判断**

---

## 功能概述

该技能为用户提供全面的股票买入决策建议,结合技术分析和宏观环境判断,帮助用户做出更明智的投资决策。

**核心功能**:
- ✅ 技术面分析: 基于改进策略一(多指标共振 + 高位预警)
- ✅ 宏观环境判断: 行业周期、公司治理、宏观经济(自动化查询)
- ✅ 综合评分: 技术面 + 基本面综合评级
- ✅ 止盈止损: 基于技术指标和风险等级给出具体价位
- ✅ 历史回测: 验证策略有效性,计算胜率和收益率
- ✅ 风险提示: 明确标注风险等级和注意事项

---

## 使用方法

### 触发方式

用户可以通过以下方式触发此技能:

1. **公司名称**: "腾讯控股可以买入吗?"
2. **股票代码**: "00700适合买入吗?"
3. **一般询问**: "帮我判断这只股票是否适合买入"
4. **买入建议**: "XX股票买入建议"

### 输入示例

```
用户: "腾讯控股可以买入吗?"
用户: "00700今天适合买吗?"
用户: "帮我分析一下康方生物的买入点"
用户: "XX股票现在能买吗?"
```

## 技术架构

### 数据流程

```
用户输入股票名称/代码
    ↓
westock-data 获取股票数据
    ↓
计算技术指标 (改进策略一)
    ↓
web_search 查询宏观环境
    ↓
综合评分计算
    ↓
生成决策报告
```

### 评分逻辑

```
技术评分 = (满足条件数 × 10) - (排除条件数 × 5) + 50
范围: 0-100分

综合评分 = 技术评分 × 行业系数 × 治理系数 × 宏观系数
范围: 0-100分
```

### 风险调整系数

| 风险类型 | 正常 | 轻微 | 中等 | 严重 | 极严重 |
|---------|------|------|------|------|--------|
| 行业周期 | 1.0 | - | 0.6 | 0.4 | - |
| 公司治理 | 1.0 | 0.85 | 0.6 | 0.2 | - |
| 宏观环境 | 1.0 | 0.85 | - | 0.5 | - |

## 注意事项

1. **数据真实性**: 必须使用westock-data获取真实数据
2. **时效性**: 查询最新数据,确保实时性
3. **风险提示**: 明确标注风险,不保证收益
4. **免责声明**: 仅供参考,不构成投资建议
5. **综合判断**: 技术面 + 基本面,不单一依赖
