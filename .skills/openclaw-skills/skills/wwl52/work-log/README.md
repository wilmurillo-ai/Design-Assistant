# 📝 工作日志技能

自动化记录工作日志，每日/每周定时生成专业日报/周报。

## 功能特性

- ✅ **自由记录** - 随时添加/删除当天工作日志
- ⏰ **定时日报** - 每天 18:00 自动发送日报
- 📅 **定时周报** - 每周一 09:00 自动发送周报
- 📊 **双模式输出** - 表格模式（清晰展示）+ 文本模式（直接交领导）

## 日报/周报格式

### 表格模式
| 时间 | 任务内容 | 进度 | 产出/成果 |
|------|----------|------|----------|
| 10:00 | 完成任务A | 100% | 完成交付 |

### 文本模式
```
今日工作内容
1. 任务名称 - 进度：100% | 备注：已完成

今日风险/阻碍
• 无

明日计划
• 计划事项1
• 计划事项2
```

## 安装

```bash
claw skill install work-log
```

## 使用方法

### 添加日志
```
用户：记一下，今天完成了XXX功能开发
系统：✅ 已记录到今日工作日志 [10:20]
```

### 查询日志
```
用户：看看今天的日志
系统：展示当日所有日志条目
```

### 生成日报
```
用户：生成日报
系统：展示表格模式 + 文本模式日报
```

### 生成周报
```
用户：生成周报
系统：展示表格模式 + 文本模式周报
```

## 配置

日志存储位置：`~/.workbuddy/work-log/logs.json`

## 触发词

工作日志、日志记录、记日志、写日志、日志总结、下班总结、周总结、工作记录
