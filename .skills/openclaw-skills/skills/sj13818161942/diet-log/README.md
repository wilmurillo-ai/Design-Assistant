# 🍽️ diet-log — OpenClaw 饮食记录与营养分析 Skill

> 接入真实食物营养数据库的 OpenClaw 智能饮食助手。记录每餐饮食、自动分析 30+ 种营养素、支持阶段性统计报告。

[English](README_EN.md) | 简体中文

---

## ✨ 功能特性

- **🔍 真实营养数据**：内置 1643 条食物营养成分数据，覆盖谷/肉/蛋/菜/果/豆等 16 大类食物
- **📊 全营养分析**：热量、蛋白质、脂肪、碳水化合物、膳食纤维、7 种脂肪酸、10 种矿物质、13 种维生素
- **🧠 智能匹配**：精确匹配 → 同类参考 → 主动提问，三级策略应对数据库无法精确识别的情况
- **📅 阶段统计**：支持按日/周/月统计，输出每日明细 + 周期均值 + 与参考标准对比
- **💾 本地存储**：所有记录保存在本地 JSON 文件，隐私安全、无需联网

## 📦 安装

### 方式一：ClawHub 安装（推荐）

```bash
clawhub install diet-log
```

### 方式二：手动安装

1. 下载或克隆本仓库
2. 将 `diet-log` 目录放入你的 OpenClaw workspace skills 目录：
   ```
   ~/.openclaw/workspace/skills/diet-log/
   ```
3. 重启 OpenClaw

## 🚀 使用方法

安装后在 OpenClaw 中直接对话即可：

### 记录饮食 & 营养分析

```
我今天早餐吃了：一碗面条、一个鸡蛋、一份青菜
```

OpenClaw 会自动：
1. 识别食物列表并查询营养数据
2. 计算本餐全部营养素摄入
3. 输出完整营养报告
4. 自动保存到本地记录

### 阶段性统计

```
统计最近7天的饮食情况
```

输出最近 7 天每日明细、周期总摄入、日均摄入，并与参考标准对比。

### 补充说明食物信息

如果数据库无法精确匹配某食材，可以补充说明：

```
那杯饮品是鲜榨橙汁，大约300ml
```

## 🔬 营养分析维度

### 宏量营养素
能量(kcal)、蛋白质(g)、脂肪(g)、碳水化合物(g)、膳食纤维(g)

### 脂肪酸
饱和脂肪酸、单不饱和脂肪酸、多不饱和脂肪酸、反式脂肪酸

### 矿物质
钙、铁、锌、钾、钠、镁、磷、铜、锰、硒

### 维生素
维A、维C、维D、维E、维K、维B1、维B2、维B3、维B5、维B6、维B7、叶酸、维B12

---

## 📊 食物匹配策略

当数据库无法精确匹配某食物时，按以下策略处理：

1. **精确匹配**：查找名字/别名包含关键词的食物
2. **同类参考**：在同一类别（畜肉/禽肉/蔬菜/豆制品等）中寻找最接近的食物，并在记录中标注「参考值」
3. **主动提问**：无法判断时，向用户询问食物类别后再做替代

## 📚 数据来源与许可

食物营养数据来自 [LuckyHookin/foodwake](https://github.com/LuckyHookin/foodwake)（[Apache-2.0](https://www.apache.org/licenses/LICENSE-2.0) 协议）。

> ⚠️ 数据为 foodwake.com 的 2020 年快照，可能与中国最新食物成分表存在差异。营养计算为估算值，实际摄入因烹饪方式、食材产地等因素可能有偏差。本工具仅供一般营养参考，不作为医学诊断或治疗依据。
