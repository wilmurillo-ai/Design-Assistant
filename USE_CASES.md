# Design Assistant Use Cases & Skills Matrix

This document serves as the source-of-truth for the core use cases this Assistant handles, bridging the gap between complex enterprise systems and fast-paced "vibe-coding" projects. 

Below are the primary buckets of work and the specific skills/repositories installed to support them.

## 1. Visual Logic & Visual Hierarchy
**Description:** Organize and analyze project data and specs from a user perspective. Create simple and rough wireframes that frame those outcomes with visual coherence.
**Assigned Skills:**
- `markster-public/markster-os`
- `Owl-Listener/designer-skills`

## 2. Rapid Prototyping & "Vibe-Coding"
**Description:** 
- **Generate Boilerplate HTML/CSS:** Quickly spin up operator or shift-lead views to test a concept.
- **Refine UI Components:** Draft modern, minimalist code snippets for specific UI elements like hero banners or navigation bars.
- **Logic Mapping:** Define how a user flows from an AI-driven prompt to a specific outcome.
**Assigned Skills:**
- `vudovn/antigravity-kit`
- `anthropics/claude-code`

## 3. Copywriting & Content Strategy
**Description:** Craft user-centric copy that aligns with the visual hierarchy, interfaces, and overarching product strategy.
**Assigned Skills:**
- `anthropics/knowledge-work-plugins`

## 4. Professional UX Strategy & Proposals
**Description:**
- **Project Scoping:** Estimate time and costs for wireframing, design systems, and high-fidelity prototyping.
- **Stakeholder Communication:** Draft clear, professional emails to managers regarding project status or contract renewals.
- **Case Study Structuring:** Organize a compelling narrative that highlights your expertise in manufacturing and SaaS.
**Assigned Skills:**
- `RightNow-AI/openfang`

## 5. Research & Heuristic Analysis
**Description:** Act as a second pair of eyes to:
- **Analyze/Audit Existing Pages:** Review landing pages for friction points or accessibility issues.
- **Trend Sourcing:** Look for modern minimalist patterns that fit your specific aesthetic for EDC or professional projects.
**Assigned Skills:**
- `Owl-Listener/designer-skills`

## 6. Information Architecture Mapping
**Description:** Create information architecture maps using Mermaid code, translating design specs per user requests into visual structures.
**Assigned Skills:**
- `openclaw/skills`
