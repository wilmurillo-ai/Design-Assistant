# Design Assistant (AI Agent)

An autonomous AI agent built on Google Antigravity tailored for Senior Digital Product Designers, bridging complex enterprise UX strategy with rapid, high-fidelity "vibe-coding" and front-end prototyping.

Design Assistant is a specialized AI agent built for Senior Digital Product Designers. By integrating over 60 dedicated skills across UX/UI, marketing, and software engineering, it seamlessly bridges the gap between complex enterprise strategy and fast-paced "vibe-coding." From conducting heuristic audits and mapping visual logic to generating boilerplate UI components and drafting stakeholder proposals, this assistant acts as a comprehensive, autonomous partner for modern product design workflows.

## An autonomous AI assistant designed to accelerate and scale the workflow of Senior Digital Product Designers. 
It is equipped with specialized capabilities for:

- UX & Visual Logic: Translating complex project data into intuitive wireframes and visual hierarchies.
- Rapid Prototyping: Generating robust, modern HTML/CSS boilerplate and UI components ("vibe-coding").
- Content Strategy: Crafting user-centric copy that aligns with brand tone and interfaces.
- Professional Strategy: Scoping projects, drafting proposals, and structuring case studies.
- Heuristic Analysis: Auditing existing platforms for accessibility and UX friction points.
- Creation of visual information architecture maps using Mermaid code.
