# Design Assistant (AI Agent)

An autonomous AI agent built on Google Antigravity tailored for Senior Digital Product Designers, bridging complex enterprise UX strategy with rapid, high-fidelity "vibe-coding" and front-end prototyping.

Design Assistant is a specialized AI agent built for Senior Digital Product Designers. By integrating over 60 dedicated skills across UX/UI, marketing, and software engineering, it seamlessly bridges the gap between complex enterprise strategy and fast-paced "vibe-coding." From conducting heuristic audits and mapping visual logic to generating boilerplate UI components and drafting stakeholder proposals, this assistant acts as a comprehensive, autonomous partner for modern product design workflows.

## An autonomous AI assistant designed to accelerate and scale the workflow of Senior Digital Product Designers. 
It is equipped with specialized capabilities for:

- UX & Visual Logic: Translating complex project data into intuitive wireframes and visual hierarchies.
- Rapid Prototyping: Generating robust, modern HTML/CSS boilerplate and UI components ("vibe-coding").
- Content Strategy: Crafting user-centric copy that aligns with brand tone and interfaces.
- Professional Strategy: Scoping projects, drafting proposals, and structuring case studies.
- Heuristic Analysis: Auditing existing platforms for accessibility and UX friction points.
- Creation of visual information architecture maps using Mermaid code.

## Use Cases in Action

**1. UX & Visual Logic**
*Example:* You're tasked with designing a new onboarding flow for a complex enterprise SaaS product. You simply provide the raw feature requirements and user personas to the Design Assistant. It will instantly analyze the data to propose a cohesive visual hierarchy, generating a rough structural wireframe that guarantees intuitive navigation before you even open Figma.

**2. Rapid Prototyping ("Vibe-Coding")**
*Example:* You need to quickly pitch a new dark-mode dashboard concept to a client but don't have time to build high-fidelity mockups. You ask the assistant to "vibe-code an MVP sales dashboard." It will autonomously spin up clean, modern HTML and CSS boilerplate featuring glassmorphic panels, CSS grid layouts, and the 60-30-10 color rule—giving you a stunning, interactive prototype in seconds.

**3. Content Strategy**
*Example:* Your interface feels empty without realistic copy, and placeholder "Lorem Ipsum" isn't selling the design. You prompt the assistant to draft microcopy for an industrial manufacturing app. The agent adapts its tone instantly, producing concise, actionable, and friction-free UI text—like precise error messages and tooltips—that align perfectly with the target demographic.

**4. Professional Strategy**
*Example:* A stakeholder asks for a project scope and timeline for a new design system initiative. Instead of spending hours drafting, you ask the Design Assistant to scope the project. It generates a professional, systematized proposal detailing low-fidelity vs. high-fidelity tasks, complete with accurate time estimates and a polished executive summary ready to be emailed to management.

**5. Heuristic Analysis**
*Example:* You've just finished a landing page design, but you want an objective second opinion. You feed the live URL or HTML code to the assistant for an audit. Acting as a seasoned UX researcher, it scans the page, identifies accessibility violations, points out conversion friction points, and recommends modern, minimalist design trends to resolve them.

**6. Information Architecture Mapping**
*Example:* You are planning a massive structural overhaul of an e-commerce platform and need to visualize the user journey. You provide the assistant with your rough design specs, and it immediately outputs a syntactically perfect Mermaid.js flowchart. This gives you a clear, easily editable visual map of the entire state machine and site architecture directly in your markdown file.
